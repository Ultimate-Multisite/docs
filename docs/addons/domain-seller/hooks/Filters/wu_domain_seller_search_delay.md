# Filter: wu_domain_seller_search_delay

Filters the domain search debounce delay in milliseconds.

Increase this value to reduce API calls on slow connections.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $delay | `int` | Debounce delay in milliseconds. Default 500. |
