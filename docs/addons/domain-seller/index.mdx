## Installation

1. Ensure **Multisite Ultimate v2.0.0+** is installed and active
2. Upload the plugin files to `/wp-content/plugins/ultimate-multisite-domain-seller/`
3. Activate the plugin through the 'Plugins' menu in WordPress Network Admin
4. Navigate to **Ultimate Multisite > Settings > Domain Seller** to configure providers
5. Create domain products in **Ultimate Multisite > Products**

## Frequently Asked Questions

## What version of Multisite Ultimate is required?

This addon requires Multisite Ultimate version 2.0.0 or higher. It is not compatible with Ultimate Multisite v1.x.

## Which domain providers are supported?

Currently OpenSRS is fully supported. The addon is architected to easily support additional providers - more will be added in future releases.

## Can I set different pricing for different TLDs?

Yes! You can create separate domain products for different TLD groups, each with their own pricing markup and configuration.

## Does it support automatic renewals?

Yes, domains can be set to auto-renew and integrate with customer memberships for seamless billing.

## Can customers manage their domains?

Yes, customers get a domain management interface in their panel to view registrations, renewal dates, and basic management functions.
