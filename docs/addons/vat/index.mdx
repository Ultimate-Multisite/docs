# VAT

Multisite Ultimate: European VAT

Adds additional features and tools to Multisite Ultimate's default tax handling to better support European VAT.
Automatically adds EU VAT rates to the Tax Rates page from

## Installation

1. Upload 'ultimate-multisite-vat' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Got to Ultimate Multisite > Settings > Taxes and Enable EU Vat.
4. Go Click on Tax Rates and Update EU VAT Rates at the bottom and Save Tax Rates.
5. Done!
