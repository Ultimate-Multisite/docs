# Ultimate Multisite Addons

Addons extend Ultimate Multisite with additional features and integrations. Most addons are paid and can be purchased from [ultimatemultisite.com](https://ultimatemultisite.com).

## Installation

Addons can be installed in two ways:

1. **Via the Addon Page** -- Navigate to your site's addon management page in the WordPress admin
2. **Manual Download** -- Download from [your account](https://ultimatemultisite.com/my-account/downloads/) and upload through Plugins > Add New
