# Filter: networks_user_is_network_admin

Filters the networks a user is the administrator of.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $ | `array\|bool` | List of network IDs or false if no networks for the user. |
| $ | `int` | User ID for which the networks should be returned. |
