# Material WP

WordPress admin interface has been the same for ages. But have you ever wondered how a Material Design version of the admin interface would look like? 

Material WP is an admin theme developed to give you that look and feel, helping you to provide your clients with a more customized Dashboard.

## Features

- Custom CSS support
- Easy styling settings
- Menu reordering
- Easy import/export

## Documentation

**Customizing your Admin Panel with Material WP**

## Installation

1. Upload 'material-wp' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

## Usage Guide

WordPress admin interface has been the same for ages. But have you ever wondered how a Material Design version of the admin interface would look like?

Material WP is an admin theme developed to give you that look and feeling, helping you to provide your clients with a more customized Dashboard. Material WP is also part of Ultimate Multisite’s premium plugins core.

After **installing it**, go to your _Plugins_ page and click to activate it on your Network.
