# Site Exporter

## Content

We understand the need of being able to move subsites within your network going to a single site installation and vice versa should your customer decide to migrate their website. The Site exporter and importer addon makes this task easier. This tool export and import sites using zip files which can also serve as a backup.

## Features

* Export and Import Sites

* Easy-to-use interface

* Create site backup

## Documentation
**How to Export and Import a Site**

## Installation

1. Upload **ultimate-multisite-site-exporter** to the **/wp-content/plugins/** directory
2. Activate the plugin through the **Plugins** menu in WordPress
3. Follow the step by step Wizard to set the plugin up

## Usage Guide

With Ultimate Multisite you can export a site from your network or import a site to your multisite or single-site installation.

To start the process, go to our add-ons page and install the Ultimate Multisite: Site Exporter & Importer add-on.![Add-ons page](/img/admin/addons-page.png)

![Add-ons page with install option](/img/admin/addons-page.png)

Go to your Plugins page and click to **Network Activate** the add-on you have just installed.<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Site Exporter addon -->

The **Export & Import** option can be found on the Ultimate Multisite menu, on the left.

<!-- Screenshot unavailable: Ultimate Multisite menu showing Export and Import option -->

## How to export a site

To export a site from your network, go to **Export & Import** on the Ultimate Multisite menu and, on the **Export Sites** tab, click to **Generate new Site Export**.

<!-- Screenshot unavailable: Export Sites tab with Generate new Site Export button -->

Search for the site you want to export and select the items you want to include in the exported file. You can include themes, plugins, and uploads. If your site is too big, you can choose to run the export process in the background.<!-- Screenshot unavailable: Site export options with site search, include themes/plugins/uploads checkboxes -->

Click the **Export Site** button. Your exported site will be available on the **Existing Exports** list. Click the **Download ZIP** button to download the file.<!-- Screenshot unavailable: Existing Exports list with Download ZIP button -->

## How to import a site

To import a site, go to the **Import a Site** tab and upload the .zip file of the site you want to import. You can alter the URL to a new one or keep the old URL by leaving the space blank. Also, you can check the box to delete the .zip file from your installation after the import. Then click to **Begin Import**.

<!-- Screenshot unavailable: Import a Site tab with zip upload field and Begin Import button -->

The process can take a little while. You will see a pending import in the list below the importer.

<!-- Screenshot unavailable: Pending import in the list below the importer -->

You can stop the process by clicking **Cancel Import**.

<!-- Screenshot unavailable: Cancel Import button to stop the import process -->

After the process is complete, your site is already available on your network.

## How to import a site on a single install

To import a site to your single install first install and activate the importer on your site. You can find the link to download it on the exporter tab.

<!-- Screenshot unavailable: Exporter tab with link to download single-site importer -->

You can find the importer on the **Tools** menu, on the left, on the subitem **Export & Import**.

<!-- Screenshot unavailable: Single-site Tools menu showing Export and Import submenu -->

Upload the .zip file of the site you want to import. You can alter the URL to a new one or keep the old URL by leaving the space blank. Also, you can check the box to delete the .zip file from your installation after the import. Then click to **Begin Import**.

<!-- Screenshot unavailable: Single-site import form with zip upload and Begin Import button -->

The process can take a little while. You will see a pending import in the list below the importer.

<!-- Screenshot unavailable: Single-site pending import in the list -->

You can stop the process by clicking **Cancel Import**.

<!-- Screenshot unavailable: Cancel Import button to stop the import process -->

After the process is finished, your site will be replaced by the site you have just imported.
