# Content Sync

Multisite Ultimate: Content Sync

This addon for Multisite Ultimate allows network administrators to synchronize content from any site to other sites in the network. It supports all post types including posts, pages, custom post types, and Elementor templates. The addon includes a multi-step wizard interface for selecting content and target sites, with batch processing and real-time progress feedback.

## Features

* Content Selection: Visual card grid with thumbnails and post type filtering
* All Post Types: Sync posts, pages, products, Elementor templates, and any custom post type
* Site Selection: Select which sites to sync content to, with source site exclusion and child site pre-selection
* Batch Processing: Syncs to 10 sites at a time with progress updates
* Media Sync: Automatically copies and remaps media files (images, videos) from post content to target sites
* Taxonomy Sync: Preserves categories, tags, and custom taxonomies
* Elementor Support: Full support for Elementor templates with CSS regeneration and template ID remapping
* Conflict Resolution: Option to replace existing content or skip it
* Cache Management: Automatically clears Elementor cache on target sites after sync

## Requirements

* WordPress 5.3 or higher
* PHP 7.4 or higher
* Multisite Ultimate plugin (active)
* Elementor plugin (optional, required for Elementor template sync)

## Installation

1. Upload 'ultimate-multisite-content-sync' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in your WordPress Network Admin
3. Configure settings in Ultimate Multisite → Settings → Content Sync
4. Navigate to Ultimate Multisite → Content Sync to start syncing
