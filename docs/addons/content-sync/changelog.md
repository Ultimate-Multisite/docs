# Content Sync Changelog

### Version 1.1.0 - Released on 2025-12-31
* New: Support for syncing all post types (posts, pages, custom post types)
* New: Post type filtering in content selection step
* New: Source site selection step - sync from any site, not just the main site
* New: Child site pre-selection for template sites
* New: Media extraction from post_content for all post types
* New: Taxonomy syncing (categories, tags, custom taxonomies)
* Changed: Renamed from "Elementor Template Sync" to "Content Sync"
* Changed: "Select Templates" step renamed to "Select Data"
* Improved: Elementor-specific code now only runs for elementor_library post type

### Version 1.0.0 - Released on 2025-12-19
* Initial release
* Multi-step wizard for template and site selection
* Batch processing with real-time progress feedback
* Full media sync with automatic ID remapping
* Settings integration with Ultimate Multisite
* Site exclusion support via network settings
* Elementor cache clearing after sync
* Template taxonomy preservation
