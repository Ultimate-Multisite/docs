# Chuck Norris Facts

Add a bit of humor to your daily WordPress administration tasks with the **Ultimate Multisite - Chuck Norris Facts** addon. When activated, you'll see a random Chuck Norris fact displayed in the upper right corner of every admin page.

Perfect for bringing smiles to your team, lightening the mood during long admin sessions, or just celebrating the legendary awesomeness that is Chuck Norris.

## Why Chuck Norris Facts?

* **Boost Team Morale**: Add a fun element to your network administration
* **Break the Ice**: Great conversation starter for new team members
* **Productivity Boost**: A quick laugh can improve focus and productivity
* **Timeless Humor**: Chuck Norris facts never get old

**110+ Legendary Facts**
A comprehensive collection of classic Chuck Norris jokes and facts that celebrate the legend himself.

**Non-Intrusive Display**
Facts appear in the admin bar in an unobtrusive way that doesn't interfere with your workflow.

**Translation Ready**
All facts are wrapped in translation functions, making it easy to localize for your language or create custom facts for your team.

**Fully Responsive**
Facts display properly on desktop, tablet, and mobile screens - Chuck Norris is awesome on all devices.

**RTL Support**
Full support for right-to-left languages because Chuck Norris transcends language barriers.

**Accessibility Focused**

## Installation

1. Ensure Ultimate Multisite plugin is installed and activated on your WordPress network
2. Upload the addon files to your `/wp-content/plugins/` directory
3. Network activate the plugin through the 'Plugins' menu in WordPress
4. Enjoy random Chuck Norris facts on every admin page!

No configuration needed - just activate and smile!

## Frequently Asked Questions

## Does this require Ultimate Multisite?

While designed as an Ultimate Multisite addon, the plugin works independently and can run on any WordPress installation.

## Can I add my own facts?

Yes! The facts are translation-ready. You can create your own `.po` file with custom facts and place it in the `/lang` directory. Or you can filter the facts array programmatically.

## Does this slow down my admin?

Not at all! The plugin is extremely lightweight and has no impact on performance. It only displays text - no external API calls or heavy processing.

## Can I disable it on certain pages?

The facts are hidden automatically on pages where they might interfere (like the block editor). You can also use WordPress hooks to customize where facts appear.

## Are the facts appropriate for all audiences?

Yes! All facts are clean, family-friendly humor celebrating Chuck Norris's legendary status.

## Can I translate the facts to another language?

Absolutely! All facts use WordPress translation functions. Create a translation file for your language and place it in the `/lang` directory. Text domain: `ultimate-multisite-chuck-norris-facts`

## Does it work with multisite?

Yes! It's designed specifically for multisite networks and can be network-activated.
