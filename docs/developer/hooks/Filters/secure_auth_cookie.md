# Filter: secure_auth_cookie

Filters whether the auth cookie should only be sent over HTTPS.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $secure | `bool` | Whether the cookie should only be sent over HTTPS. |
| $user_id | `int` | User ID. |
