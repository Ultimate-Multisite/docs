# Filter: wu_add_new_domain_modal_fields

Filters the fields for the add new domain modal.

Allows addons (e.g. Domain Seller) to modify or replace the domain registration fields.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $fields | `array` | The form fields. |
