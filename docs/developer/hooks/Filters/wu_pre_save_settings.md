# Filter: wu_pre_save_settings

Allow developers to filter settings before save by Ultimate Multisite.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $settings | `array` | The settings to be saved. |
| $settings_to_save | `array` | The new settings to add. |
| $saved_settings | `array` | The current settings saved. |
