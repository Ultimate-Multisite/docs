# Filter: wu_plan_product_types

Filter the product types that should be treated as plans.

This filter allows addons to register additional product types that should be recognized as plans in validation and segregation.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | Array of product types to treat as plans. |
