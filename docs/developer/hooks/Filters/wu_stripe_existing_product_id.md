# Filter: wu_stripe_existing_product_id

Filters the ID of the product to check for. If this exists, the new subscription will use this product.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $product_id | `string` | ID of the Stripe product to check for. |
| $name | `string` | Product name. |
