# Filter: wu_stripe_existing_plan_id

Filters the ID of the plan to check for. If this exists, the new subscription will use this plan.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $plan_id | `string` | ID of the Stripe plan to check for. |
| $membership_level | `object` | Membership level object. |
