# Filter: get_site_url_for_previewer

Allow plugin developers to filter the URL used in the previewer

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $domain | `string` | Default domain being used right now, useful for manipulations |
| $domain_options | `array` | List of all the domain options entered in the Ultimate Multisite Settings -&gt; Network Settings -&gt; Domain Options |
