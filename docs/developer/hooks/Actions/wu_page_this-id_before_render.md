# Action: wu_page_\{$this->id\}_before_render

Allow plugin developers to add additional content before we print the page.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $page_id | `string` | The id of this page. |
| $page | `object` | The page object. |
