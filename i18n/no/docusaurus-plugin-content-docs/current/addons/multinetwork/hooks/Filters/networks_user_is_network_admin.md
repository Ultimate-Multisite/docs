# Filter: networks_user_is_network_admin {#filter-networksuserisnetworkadmin}

Filtrerer nettverkene en bruker er administrator av.

| Navn | Type | Beskrivelse |
|------|------|-------------|
| $ | `array\|bool` | Liste over nettverks-IDer, eller `false` hvis det ikke finnes nettverk for brukeren. |
| $ | `int` | Bruker-IDen for hvilke nettverkene skal returneres. |
