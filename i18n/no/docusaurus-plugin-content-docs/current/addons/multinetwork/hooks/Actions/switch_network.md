# Action: switch_network

Utløses når det nåværende nettverkskonteksten byttes.

## Parametere {#parameters}

| Navn | Type | Beskrivelse |
|------|------|-------------|
| $new_network_id | `int` | ID-en til nettverket som byttes til. |
| $old_network_id | `int` | ID-en til det tidligere aktive nettverket. |
