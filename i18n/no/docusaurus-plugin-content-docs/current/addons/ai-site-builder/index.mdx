# AI Site Builder

ultimate-multisite-ai-site-builder

## Krav

- WordPress 5.3 eller høyere
- PHP 7.4 eller høyere
- Multisite Ultimate plugin (aktiv)

## Installasjon

1. Last opp addon-filene til katalogen din `/wp-content/plugins/`
2. Aktiver pluginet via menyen 'Plugins' i WordPress
3. Konfigurer addon-innstillingene i nettverksadministratoren

## Utvikling

### Oppsett

```bash
# Installer avhengigheter
composer install
npm install

# Kjør tester
npm run test

# Kjør sjekk av kodekoder
vendor/bin/phpcs
vendor/bin/phpstan

# Rett kodeformatfeil
vendor/bin/phpcbf

# Bygg for produksjon
npm run build
```

### Prosjektstruktur

```
ultimate-multisite-ai-site-builder/
├── inc/                          # PHP klasser og inkluderinger
│   ├── class-multisite-ultimate-updater.php  # Oppdateringshåndterer
│   └── class-ai_site_builder-main.php             # Hovedfunksjonalitet
├── tests/                        # Enhetstester
│   ├── bootstrap.php            # Test-bootstrap
│   ├── class-*-base.php         # Basis testklasse
│   └── class-*-test.php         # Testtilfeller
├── views/                       # Malfiler
├── assets/                      # Statiske ressurser
│   ├── css/                     # Stilark
│   ├── js/                      # JavaScript-filer
│   └── images/                  # Bilder
├── lang/                        # Språkfiler
├── composer.json                # PHP avhengigheter
├── package.json                 # Node.js avhengigheter
├── phpunit.xml.dist             # PHPUnit-konfigurasjon
├── .phpcs.xml.dist              # Kodekoder-konfigurasjon
└── rector.php                   # Regler for kodeoppgradering
```

### Testing

Addonet inkluderer et omfattende testrammeverk:

- **Enhetstester (Unit Tests)**: PHPUnit-baserte tester med WordPress test suite
- **Basis Testklasse (Base Test Class)**: Tilbyr hjelpefunksjoner for vanlige testscenarioer
- **Testdekning (Test Coverage)**: Konfigurert for å spore kodekledningen
- **Multisite Testing**: Testene kjøres i et multisite-miljø

### Kodekoder

- **WordPress Coding Standards**: Håndhevet via PHPCS
- **PHP 7.4+ Kompatibilitet**: Støtter moderne PHP-funksjoner
- **Statisk Analyse**: PHPStan-integrasjon for typesjekking
- **Kodeoppgradering**: Rector-regler for PHP-oppgraderinger

### Byggsystem

- **Prosesserings av ressurser (Asset Processing)**: Minifisering av CSS/JS-filer
- **Oversettelse (Translation)**: Generering av POT-filer
- **Pakking (Packaging)**: Opprettelse av arkiver for distribusjon
- **Utviklingsmodus (Development Mode)**: Ikke-minifiserte ressurser for feilsøking

## Konfigurasjon

Addonet støtter ulike konfigurasjonsalternativer som er tilgjengelige via nettverksadministratorgrensesnittet.

## Hooks og Filtre

### Handlinger (Actions)

- `ultimate-multisite-ai-site-builder_init` - Utløses etter addon-initialisering
- `ultimate-multisite-ai-site-builder_loaded` - Utløses etter at alle addon-filer er lastet

### Filtre (Filters)

- `ultimate-multisite-ai-site-builder_settings` - Endre addon-innstillingene
- `ultimate-multisite-ai-site-builder_enabled` - Overstyre addonens aktiverings-/deaktiveringsstatus
