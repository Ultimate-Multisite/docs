# PayFast {#payfast}

Ultimate Multisite: Payfast Gateway

Legger til Payfast – en rask og sikker betalingsgateway på nettet i Sør-Afrika – som en betalingsmulighet i Ultimate Multisite.

## Installasjon {#installation}

1. Last opp 'ultimate-multisite-payfast' til mappen '/wp-content/plugins/'
2. Aktiver pluginet via menyen 'Plugins' i WordPress
3. Følg den trinnvise guiden for å sette opp pluginet

## Brukerveiledning {#usage-guide}

PayFast er en ledende betalingsbehandler for bedrifter i Sør-Afrika. De er dedikert til å hjelpe nettforhandlere med å vokse ved å gjøre transaksjoner sømløse og sikre. Etter en forespørsel fra Ultimate Multisite-brukerne, bestemte vi oss for å lage et tillegg (addon) som gjør integrasjon med PayFast mulig. Denne artikkelen viser hvordan du setter opp PayFast som betalingsgateway for å motta betaling i ditt Ultimate Multisite-nettverk.

Det første steget er å installere tillegget.

# Hvordan installere Payfast add-on: {#how-to-install-the-payfast-add-on}

Gå til add-ons-siden og velg Ultimate Multisite: Payfast Gateway add-on.

![Add-ons page](/img/admin/addons-page.png)

Add-on-vinduet vil dukke opp. Klikk på Install Now.

![Add-ons page with install option](/img/admin/addons-page.png)

Etter at installasjonen er fullført, blir du omdirigert til plugins-siden. Her klikker du bare på Network Activate, og _Payfast_ add-on vil bli aktivert på nettverket ditt.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# Oppsett av PayFast {#setting-up-payfast}

Etter at du har installert add-on-et, kan du nå gå til Ultimate Multisite-innstillingene og begynne å konfigurere PayFast.

På siden **Ultimate Multisite settings** finner du fanen "Payments". Under denne fanen skal du se "Payfast" sammen med en bryter (toggle button) for å aktivere eller deaktivere denne betalingsgatewayen.

![Payment gateway settings](/img/config/settings-payment-gateways.png)Klikk på bryteren for å aktivere den. Dette vil deretter utvide boksen og vise forskjellige felt du må fylle ut, for eksempel **Merchant ID, Merchant Key og PassPhrase.** Du kan få denne informasjonen fra din **Payfast**-konto

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Neste steg er å logge inn på PayFast-kontoen din for å få den informasjonen du trenger.

Når du har logget inn, kan du umiddelbart se legitimasjonene du trenger fra **Payfast**-kontoadministrasjonen (dashboard).

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->Du kan enkelt kopiere og lime inn legitimasjonene og fylle ut detaljene i Ultimate Multisite-innstillingene dine. Deretter klikker du på knappen **"Save settings** " for å lagre endringene.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Det var det, PayFast-betalingsintegrasjonen din er fullført!
