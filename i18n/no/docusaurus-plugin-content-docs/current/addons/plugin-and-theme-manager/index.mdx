# Plugin & Theme Manager

Multisite Ultimate: Plugin and Theme Manager

Rediger eller skjule metadata (tittel, forfatter, miniatyrbilde og beskrivelse) for installerte plugins og temaer. Opprett kategorier for plugins og temaer slik at brukerne dine kan filtrere dem, og legg til en vakker, tilpasset Plugins-side i brukerpanelet ditt.

For å bruke pluginet, kan du navigere til Plugins- eller Themes-sidene i nettverksadministratoren din og klikke på "Edit Plugin/Theme info".

## Installasjon {#installation}

1. Last opp 'ultimate-multisite-plugin-and-theme-manager' til katalogen '/wp-content/plugins/'
2. Aktiver pluginet via menyen 'Plugins' i WordPress Network Admin

## Brukerveiledning {#usage-guide}

Ultimate Multisite: Plugin and Theme Manager er et kraftig **premium** Ultimate Multisite add-on som gir deg full kontroll over hvordan nettverksbrukerne dine ser og samhandler med plugin- og temaalternativene på nettverket ditt.

Dette add-onet tjener tre formål:

  1. Det lar deg endre metadataene til både plugins og temaer fullstendig. Dette inkluderer navn, beskrivelse, forfatter og til og med miniatyrbilder!

  2. Det lar deg kategorisere plugins og temaer, slik at brukerne dine enkelt kan sortere dem.

  3. Det gir deg muligheten til å erstatte den gamle, kjedelige plugins-siden med en ny, tema-lignende side med sortering etter kategorier.

Etter at du har lastet det ned og installert det, vil en ny fane – _Add-on Settings_ – vises på Ultimate Multisite Settings-siden.
