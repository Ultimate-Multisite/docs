# Fluent Forms

Dette tilbehøret gjør det mulig å legge til grenser for Fluent Forms i Ultimate Multisite i produkter, medlemskap eller nettsteder.
Fluent Forms bruker egendefinerte tabeller i stedet for egendefinerte posttyper, så de vanlige begrensningene ikke fungerer med Fluent Forms.

## Funksjoner {#features}

- Legg til grenser for Fluent Forms eller Conversational Fluent Forms
- Kunder får en vennlig melding som oppfordrer dem til å oppgradere planen sin når de når grensen
