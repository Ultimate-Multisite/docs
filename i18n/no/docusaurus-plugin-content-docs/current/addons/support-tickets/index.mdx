# Support Tickets

**Support Tickets** er et kraftig tillegg (addon) for Ultimate Multisite som legger til et omfattende system for support-billetter i ditt WordPress-nettverk. Det er perfekt for SaaS-plattformer, WaaS (Website as a Service)-leverandører og multisite-nettverk som trenger å tilby kundestøtte både på nettverksnivå og på enkeltnettsidenivå.

## Hvorfor velge Support Tickets?

* **Støtte på flere nivåer**: Nettverksadministratorer håndterer billetter for hele nettverket, mens nettsideeiere administrerer sin egen kundestøtte.
* **Profesjonell arbeidsflyt**: Fullstendig styring av billettlivssyklusen med prioriteringer, statusser og tildelinger.
* **Brukeropplevelse**: Et vakkert og responsivt grensesnitt som gjør det enkelt for kunder å få hjelp.
* **Sikkert og pålitelig**: Bygget etter WordPress' beste sikkerhetspraksis og optimalisert databaseytelse.
* **Sømløs integrasjon**: Fungerer perfekt med Ultimate Multisites kunde- og nettsideadministrasjon.

## Nøkkelfunksjoner

**Fullstendig billettadministrasjon**

* Unikt bilonnenummereringssystem for enkel referanse
* Konfigurerbare prioritetstrinn (Lav, Normal, Høy, Kritisk)
* Omfattende statusarbeidsflyt (Åpen, Pågår, Venter, Løst, Lukket)
* Kategorisering av billetter (Generelt, Teknisk, Fakturering, Funksjonsforespørsel, Feilrapport)
* Tildel billetter til supportagenter med valgfri auto-tildeling

**Trådet samtaleveksling**

* Full samtale-tråding for billett-svar
* Distinkt utforming for svar fra ansatte og kunder
* Private notater fra ansatte som kundene ikke kan se
* Sanntidsgrensesnitt drevet av AJAX
* Moderne teksteditor med formateringsalternativer

**Filvedlegg**

* Sikker filopplasting med støtte for dra-og-slipp
* Konfigurerbare begrensninger for filtype og størrelse
* Avansert sikkerhet med deteksjon av skadelige filer
* Beskyttede nedlastinger med tilgangskontroller

**Smarte varsler**

* Automatisk e-postvarsling for nye billetter, svar og statusendringer
* Tilpassbare e-postmaler
* Rollebaserte varsler for ansatte og kunder
* Dynamiske malvariabler for personlige meldinger

**Kraftig administrasjonsgrensesnitt**

* Avansert bilistliste med sortering og filtrering
* Rask redigering av status, prioritet og tildelinger
* Massehandlinger for effektiv administrasjon
* Statistikk-dashboard med ytelsesmålinger
* Omfattende innstillingspanel

**Vakkert kundeportal**

* Rent, responsivt grensesnitt på alle enheter
* Designet med mobil i tankene (Mobile-first)
* Avansert søk og filtrering
* Moderne editor med formateringsverktøylinje
* Enkle å bruke skjemaer for innsending av billetter

**Fleksibel integrasjon**

* Bruk shortcodes eller Ultimate Multisite UI-elementer
* Vis billett-skjemaer og lister hvor som helst på nettstedet ditt
* Fungerer sømløst med multisite-nettverksstrukturen
* Utvidbart med hooks og filtre

## Perfekt for

* SaaS-plattformer som trenger kundestøtte
* WaaS-leverandører som administrerer flere klientnettsteder
* Medlemsnettsteder med trinnvis støtte
* Nettverksadministratorer som administrerer flere merkevarer
* Nettsideeiere som tilbyr kundeservice

## Installasjon

1. Forsikre deg om at Ultimate Multisite-plugin er installert og aktivert på ditt WordPress-nettverk.
2. Last opp addon-filene til mappen `/wp-content/plugins/`.
3. Aktiver pluginet på nettverksnivå via 'Plugins'-menyen i WordPress.
4. Naviger til Network Admin > Support Tickets for å konfigurere innstillingene dine.

## Ofte stilte spørsmål

## Krever dette Ultimate Multisite?

Ja, dette tillegget krever at Ultimate Multisite er installert og aktivt. Det integreres dypt med Ultimate Multisites kunde- og nettsideadministrasjonsfunksjoner.

## Kan nettsideeiere håndtere sin egen kundestøtte?

Absolutt! Nettsideeiere kan bruke billettsystemet til å gi støtte til sine egne kunder, mens nettverksadministratorer kan håndtere billetter for hele nettverket.

## Hvordan sender kunder inn billetter?

Du kan legge til skjemaer for innsending av billetter hvor som helst på nettstedet ditt ved hjelp av shortcodes eller Ultimate Multisite UI-elementer. Kundene fyller bare ut skjemaet og sender inn forespørselen sin.

## Kan jeg tilpasse billettstatusser og prioriteringer?

Ja! Du kan fullstendig tilpasse prioritetstrinn, statusser og billetttyper i nettverksadministrasjonens innstillinger for å matche din støttearbeidsflyt.

## Støtter det filvedlegg?

Ja, både kunder og ansatte kan legge ved filer til billetter og svar. Du har full kontroll over tillatte filtyper og størrelsesgrenser for sikkerhet.

## Hvordan håndteres varsler?

Systemet sender automatisk e-postvarsler for nye billetter, svar og statusendringer. Du kan tilpasse alle e-postmaler i innstillingene.

## Kan jeg tildele billetter til spesifikke supportagenter?

Ja, du kan manuelt tildele billetter til agenter eller aktivere auto-tildeling for å distribuere billetter automatisk.

## Finnes det private notater for ansatte?

Ja, ansatte kan legge til private svar som kun er synlige for andre ansatte, ikke for kundene.

## Er det mobilvennlig?

Absolutt! Hele grensesnittet er bygget med en mobil-først-tilnærming og fungerer vakkert på alle enheter.

## Kan jeg vise bilistlister og skjemaer ved hjelp av shortcodes?

Ja! Tilgjengelige shortcodes:
* `[wu_support_tickets]` - Viser kundens bilistliste
* `[wu_support_ticket_form]` - Viser skjema for ny billettinnsending
* `[wu_support_ticket_view ticket_id="123"]` - Viser spesifikk billett

## Hvilke filtyper kan lastes opp?

Du kan konfigurere tillatte filtyper i innstillingene. Systemet inkluderer avansert sikkerhetsskanning for å beskytte mot skadelige filer.

## Kan jeg eksportere billettdata?

Ja, addon-et inkluderer funksjonalitet for å eksportere billettdata og statistikk.
