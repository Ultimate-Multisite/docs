# Innholds-synkronisering {#content-sync}

Multisite Ultimate: Innholds-synkronisering

Dette tillegget for Multisite Ultimate lar nettverksadministratorer synkronisere innhold fra ethvert nettsted til andre nettsteder i nettverket. Det støtter alle posttyper, inkludert innlegg, sider, tilpassede posttyper og Elementor-maler. Tillegget inkluderer et flertrinns-wizard-grensesnitt for å velge innhold og målnettsteder, med stikkprøvebehandling og sanntids fremdriftsfeedback.

## Funksjoner {#features}

* **Innholdsvalg:** Visuell kortrutemaskin med miniatyrbilder og filtrering etter posttype
* **Alle posttyper:** Synkroniserer innlegg, sider, produkter, Elementor-maler og alle tilpassede posttyper
* **Nettstedsvalg:** Velg hvilke nettsteder du vil synkronisere innhold til, med eksklusjon av kilde-nettstedet og forhåndsvalg av barnesteder
* **Stikkprøvebehandling:** Synkroniserer til 10 nettsteder om gangen med oppdateringer av fremdrift
* **Mediesynkronisering:** Kopierer og omdirigerer automatisk mediefiler (bilder, videoer) fra postinnhold til målnettstedene
* **Taksonomisynkronisering:** Bevarer kategorier, tagger og tilpassede taksonomier
* **Elementor-støtte:** Full støtte for Elementor-maler med CSS-regenerering og omdirigering av mal-ID-er
* **Konfliktløsning:** Mulighet til å erstatte eksisterende innhold eller hoppe over det
* **Cache-håndtering:** Tømmer automatisk Elementor-cachen på målnettstedene etter synkronisering

## Krav {#requirements}

* WordPress 5.3 eller høyere
* PHP 7.4 eller høyere
* Multisite Ultimate plugin (aktiv)
* Elementor plugin (valgfritt, men nødvendig for synkronisering av Elementor-maler)

## Installasjon {#installation}

1. Last opp 'ultimate-multisite-content-sync' til katalogen '/wp-content/plugins/'
2. Aktiver pluginet via menyen 'Plugins' i WordPress Network Admin
3. Konfigurer innstillingene under Ultimate Multisite → Settings → Content Sync
4. Naviger til Ultimate Multisite → Content Sync for å starte synkroniseringen
