# Support Agents

Å administrere et helt nettverk alene er ikke en enkel oppgave. Etter hvert som nettverket ditt begynner å vokse, vil du merke en økende etterspørsel etter support. Det er en god idé å få med seg folk som kan hjelpe deg med å lette arbeidsmengden. Men å gi full tilgang/tillatelse er risikabelt og kan kompromittere nettverket ditt. Det er her Support Agent add-on kommer inn.

Denne add-on-en lar deg opprette en ny type WordPress-bruker kalt Support Agent, og gir deg full kontroll over hvilke tillatelser denne brukeren har. Å gi tillatelse er like enkelt som å slå på en bryter (toggle button), og du kan slå den av hvis du bestemmer deg for å fjerne den.

## Funksjoner {#features}

- Detaljert liste over WordPress- og Ultimate Multisite-tillatelser
- Aktiver og deaktiver tilgang enkelt
- Opprett et ubegrenset antall Agent-kontoer

## Installasjon {#installation}

1. Last opp 'ultimate-multisite-support-agents' til katalogen '/wp-content/plugins/'
2. Aktiver plugin-en via menyen 'Plugins' i WordPress
3. Følg den trinnvise Wizarden for å sette opp plugin-en
