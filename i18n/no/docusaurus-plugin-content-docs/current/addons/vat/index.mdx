# VAT

Multisite Ultimate: Europeisk MVA

Tilfører ekstra funksjoner og verktøy til Multisite Ultimate sin standard skattehåndtering for bedre støtte av europeisk MVA. Legger automatisk til EU-MVA-satser på siden for skattesatser fra

## Installasjon

1. Last opp 'ultimate-multisite-vat' til mappen '/wp-content/plugins/'
2. Aktiver pluginet via menyen 'Plugins' i WordPress
3. Gå til Ultimate Multisite > Innstillinger > Skatter og aktiver EU Vat.
4. Gå til Skattesatser og oppdater EU MVA-satser nederst, og lagre skattesatsene.
5. Ferdig!
