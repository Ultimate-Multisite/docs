# Loco Translate

Ultimate Multisite: Loco Translate Integration addon gir nettstedsspesifikke oversettelsesmuligheter til WordPress multisite-nettverket ditt. Hvis du noen gang har trengt at ulike nettsteder i nettverket ditt skal ha ulike oversettelser av samme plugin eller theme, er dette addon den perfekte løsningen.

**Hva dette addon gjør**

Dette addon integrerer Ultimate Multisite sømløst med den populære Loco Translate plugin, slik at hvert undernettsted i nettverket ditt kan vedlikeholde sine egne tilpassede oversettelser. Enten du driver en multiregional plattform, betjener ulike språkfellesskap eller bare trenger oversettelsesfleksibilitet på tvers av nettverket ditt, gjør dette addon det enkelt.

**Perfekt for:**

* Multiregionale bedrifter med ulike lokaliseringsbehov
* Utdanningsinstitusjoner som betjener mangfoldige fellesskap
* SaaS-plattformer som tilbyr flerspråklig støtte
* Nettbyråer som administrerer flere klientnettsteder
* Ethvert multisite-nettverk som krever oversettelsesfleksibilitet

## Installasjon {#installation}

1. Sørg for at Ultimate Multisite er installert og nettverksaktivert
2. Installer og aktiver Loco Translate plugin
3. Last opp dette addon til plugins-katalogen din
4. Nettverksaktiver Loco Translate Integration addon
5. Begynn å opprette nettstedsspesifikke oversettelser gjennom Loco Translate på et hvilket som helst undernettsted

Det var det! Integrasjonen fungerer automatisk uten ekstra konfigurasjon.

## Ofte stilte spørsmål {#frequently-asked-questions}

## Må jeg konfigurere noe etter installasjon? {#do-i-need-to-configure-anything-after-installation}

Nei! Integrasjonen er automatisk. Når både addon og Loco Translate er aktive, kan undernettsteder umiddelbart begynne å opprette sine egne tilpassede oversettelser.

## Hva skjer hvis et undernettsted ikke oppretter tilpassede oversettelser? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

Nettstedet vil automatisk bruke nettverksomfattende oversettelser som reserve. Ingenting bryter sammen, og alt fungerer sømløst.

## Kan undernettsteder få tilgang til nettverksomfattende oversettelser? {#can-subsites-access-network-wide-translations}

Vanlige nettstedsadministratorer ser og redigerer bare oversettelsene for sitt eget nettsted. Superadministratorer kan få tilgang til både nettverksomfattende og nettstedsspesifikke oversettelser, avhengig av hvilket nettsted de viser.

## Fungerer dette med både plugins og themes? {#does-this-work-with-both-plugins-and-themes}

Ja! Nettstedsspesifikke oversettelser fungerer for både WordPress plugins og themes.

## Vil dette gjøre multisite-nettverket mitt tregere? {#will-this-slow-down-my-multisite-network}

Nei. Dette addon bruker effektiv filinnlasting som bare sjekker for nettstedsspesifikke oversettelser når det trengs. Det er ingen merkbar ytelsespåvirkning.

## Kan jeg kopiere nettverksoversettelser til et bestemt nettsted som utgangspunkt? {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

Ja! Når du redigerer oversettelser i Loco Translate, kan du enkelt starte med nettverksoversettelser og tilpasse dem for hvert nettsted.

## Fungerer dette med WooCommerce og andre plugins? {#does-this-work-with-woocommerce-and-other-plugins}

Absolutt! Enhver plugin eller theme som bruker standard WordPress oversettelsesfunksjoner, vil fungere med nettstedsspesifikke oversettelser.
