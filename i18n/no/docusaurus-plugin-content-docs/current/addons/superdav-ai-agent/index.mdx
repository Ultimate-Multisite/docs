# Superdav AI Agent

Superdav AI Agent er en avansert WordPress AI-assistent som utvider Gratis AI Agent med temabevisste ferdigheter, forbedrede internettsøkfunksjoner, nettstedbyggerfunksjoner og valgfrie automatiseringsintegrasjoner. Versjon 1.18.0 legger til leseverktøy for Google Calendar, kontakt- og deltakertilordning, godkjenningsporter for mennesker, deduplisering av påminnelser, TextBee SMS-varsler, administrert klargjøring av Superdav-tjeneste og en separat distribuert Advanced-følgepakke for arbeidsflyter for betrodde utviklere.

## Nøkkelfunksjoner

- **Temabevisste ferdigheter** — tilpasser automatisk veiledning og funksjoner basert på ditt aktive WordPress-tema og installerte plugins
- **Flere leverandører av internettsøk** — velg mellom Brave Search og Tavily for rikere søkeresultater og avanserte forskningsmuligheter
- **Kontaktskjemaer for nettstedbygger** — opprett fullt funksjonelle kontaktskjemaer direkte fra chat-grensesnittet ved hjelp av din installerte skjemaplugin
- **Forbedret nettstedbygger** — forbedrede flerfasede planer for nettstedbygging med bedre plugin-oppdagelse og feilretting
- **Innebygde temaferdigheter** — spesialisert veiledning for Block Themes, Classic Themes, Kadence Blocks og Kadence Theme
- **Forbedret WooCommerce-integrasjon** — omarbeidet til å bruke native WooCommerce API-er for bedre pålitelighet og kompatibilitet
- **Automatisk leverandørgjenkjenning** — oppdager og oppdaterer automatisk tilgjengelige plugins og temaer når de aktiveres eller deaktiveres
- **Generering av tilpasset logo** — generer og bygg inn tilpassede SVG-logoer med automatisk navneromssikker sanitering
- **Validering av palettkontrast** — sjekk fargepar for samsvar med WCAG-tilgjengelighet før de brukes på temaet ditt
- **Fotoforbedret oppdagelse** — last opp bilder under Theme Builder-oppdagelsesintervjuet for å påvirke designretningen
- **Serveringsmenyer** — generer strukturerte sider for mat- og drikkemenyer for restauranter, kafeer og serveringsbedrifter
- **Forhåndsvisninger på desktop og mobil** — forhåndsvis designet ditt på desktop og mobile enheter under valg av designretning
- **Tier 1-nettstedsskraping** — sd-ai-agent/site-scrape er nå tilgjengelig som standard for å analysere konkurrent- og inspirasjonsnettsteder
- **Leseverktøy for Google Calendar** — les konfigurerte kalendere og hendelser slik at automatiseringer kan resonnere om kommende tidsplankontekst
- **Kontakt- og deltakertilordning** — match kalenderdeltakere med kjente nettstedsbrukere og kontakter før oppfølgingsarbeid genereres
- **Godkjenningsporter for mennesker** — sett sensitive automatiseringer på pause for gjennomgang før meldinger sendes, arbeid gjenopptas eller endringer gjøres
- **Påminnelsesoppføringer og deduplisering** — spor påminnelser som allerede er sendt, slik at gjentatte automatiseringskjøringer ikke varsler samme mottaker to ganger
- **TextBee SMS-varsler** — send konfigurerte tekstmeldingsvarsler via TextBee når en SMS-leverandør er aktivert
- **Administrert klargjøring av Superdav-tjeneste** — koble støttede nettsteder til hostede Superdav-endepunkter med automatisk oppsett i stedet for manuell endepunktkonfigurasjon
- **Advanced-følgepakke** — separat distribuerte verktøy for betrodde utviklere for filsystem, database, WP-CLI, REST-dispatcher, plugin-bygger, git-øyeblikksbilde, brukeradministrasjon og benchmark-arbeidsflyter

## Krav

- WordPress 6.0 eller nyere
- PHP 8.1 eller nyere
- WordPress Multisite (nettverksaktivert) eller enkeltnettstedsinstallasjon
- En aktiv AI-leverandørkoblingsplugin (Anthropic, OpenAI eller kompatibel)
- Gratis AI Agent v1.9.0 eller nyere (Superdav AI Agent utvider Gratis AI Agent)

## Installasjon

1. Last opp plugin-filene til `/wp-content/plugins/superdav-ai-agent`
2. Aktiver pluginen via **Plugins**-skjermen i WordPress
3. Sørg for at Gratis AI Agent også er aktivert
4. Gå til **Gratis AI Agent → Settings** og konfigurer leverandøren din for internettsøk, administrert Superdav-tjenestetilkobling, Google Calendar eller TextBee-legitimasjon etter behov

WordPress.org-pakken inneholder kjernefunksjonene i Superdav AI Agent. Advanced-følgepakken bygges og distribueres separat for miljøer der verktøy for betrodde utviklere er eksplisitt installert og autorisert.

## Temabevisste ferdigheter

Superdav AI Agent oppdager automatisk ditt aktive tema og installerte plugins, og laster den passende temabevisste ferdigheten inn i assistentens kontekst. Tilgjengelige ferdigheter inkluderer:

| Ferdighet | Gjelder for |
|---|---|
| **Block Themes** | Temaer som bruker WordPress sin blokkbaserte arkitektur (Full Site Editing) |
| **Classic Themes** | Tradisjonelle WordPress-temaer som bruker PHP-maler og den klassiske editoren |
| **Kadence Blocks** | Nettsteder med Kadence Blocks-pluginen aktiv |
| **Kadence Theme** | Nettsteder med Kadence-temaet som aktivt tema |

Se [Temabevisste ferdigheter](../../user-guide/configuration/theme-aware-skills) for detaljert veiledning om hver ferdighet.

## Leverandører av internettsøk

Superdav AI Agent v1.10.0 legger til **Tavily** som et alternativ til Brave Search for internettsøkspørringer. Begge leverandører tilbyr søkeresultater i sanntid og kan konfigureres i **Gratis AI Agent → Settings → Advanced**.

| Leverandør | Funksjoner | Oppsett |
|---|---|---|
| **Brave Search** | Raske, personvernfokuserte søkeresultater | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Rikere søkeresultater, avanserte forskningsmuligheter | [Tavily API](https://tavily.com/) |

Se [Internett-søk](../../user-guide/configuration/internet-search) for konfigurasjonsinstruksjoner.

## Kalender-, godkjennings- og varslingsarbeidsflyter

Superdav AI Agent v1.18.0 utvider automatiseringsarbeidsflyter utover innholds- og nettstedbyggingsoppgaver:

- **Google Calendar-verktøy** lar agenten lese konfigurerte kalendere og hendelsesdetaljer når kalenderlegitimasjon er tilgjengelig.
- **Kontakt- og deltakerhjelpere** kobler hendelsesdeltakere til WordPress-brukere eller kjente kontakter før påminnelser eller oppfølgingsoppgaver forberedes.
- **Menneskelige godkjenningsporter** lar en automatisering settes på pause til en autorisert bruker vurderer den foreslåtte handlingen.
- **Påminnelsesoppføringer** hindrer dupliserte kalender- eller varslingspåminnelser når en automatisering prøves på nytt eller planlegges gjentatte ganger.
- **TextBee SMS-støtte** aktiverer tekstmeldingsvarsler når TextBee-leverandøren er konfigurert.

Bruk disse verktøyene til tidsplanbevisst hjelp, deltakeroppfølging og påminnelsesarbeidsflyter som fortsatt krever menneskelig vurdering før sensitive meldinger sendes.

Se [Kontaktsskjemaer for nettstedbyggeren](../../user-guide/administration/site-builder-contact-forms) for detaljert veiledning.

## Konfigurasjon

Alle innstillinger er under **Gratis AI Agent → Innstillinger** i WordPress-administrasjonen. Superdav AI Agent legger til følgende konfigurasjonsalternativer:

| Innstilling | Beskrivelse |
|---|---|
| **Internet Search Provider** | Velg mellom Brave Search og Tavily (valgfritt) |
| **Tavily API Key** | API-nøkkel for Tavily-internettsøk (hvis Tavily er valgt) |
| **Managed Superdav Service** | Koble støttede nettsteder til den driftede Superdav-tjenesten med automatisk endpoint-klargjøring |
| **Google Calendar Credentials** | Konfigurer kalendertilgang slik at agenten kan lese kalendere og hendelser |
| **TextBee SMS Provider** | Konfigurer TextBee-legitimasjon for godkjente SMS-varslingsarbeidsflyter |

## Avansert følgepakke

Superdav AI Agent v1.18.0 legger til en separat distribuert avansert følgepakke for pålitelige utviklerarbeidsflyter. Disse funksjonene er ikke samlet i WordPress.org-kjernepakken. Installer følgepakken bare på nettsteder der administratorer uttrykkelig stoler på at agenten får tilgang til utviklerverktøy.

Avanserte verktøy inkluderer filsystemoperasjoner, databasetilgang, WP-CLI, REST-dispatcherarbeidsflyter, plugin-byggerverktøy, git-øyeblikksbilder, brukeradministrasjonshjelpere og benchmark-verktøy. Begrens disse verktøyene til pålitelige administratorer og gjennomgå konfigurasjonen av godkjenningsporter før du aktiverer dem på produksjonsnettsteder.

## Forbedringer i v1.18.0

- **Google Calendar-støtte** — les konfigurerte kalendere og hendelser for tidsplanbevisste automatiseringer
- **Kontakt- og deltakerkobling** — match kalenderdeltakere med nettstedbrukere og kontakter
- **Menneskelige godkjenningsporter** — sett sensitive automatiseringer på pause for vurdering før de fortsetter
- **Påminnelsesdeduplisering** — registrer påminnelser for å unngå dupliserte varsler ved nye forsøk eller gjentatte kjøringer
- **TextBee SMS-leverandør** — send konfigurerte tekstmeldingsvarsler etter nødvendige godkjenninger
- **Oppsett av administrert Superdav-tjeneste** — klargjør driftede tjeneste-endpoints automatisk for støttede nettsteder
- **Avansert følgedistribusjon** — publiser verktøy for pålitelige utviklere separat fra WordPress.org-kjernebygget
- **Pålitelighetsrettelser** — forbedre modellvalg, tidsavbrudd for forespørsler, håndtering av standard oppsettmodell, håndtering av resonneringstekst, veiledning for nye forsøk, herding av kalendertoken, herding av påminnelser, onboarding-lanseringer, gjenopptak av bekreftede evner og gjennomgangsproblemer med WordPress.org-pakking

## Forbedringer i v1.16.0

- **AI Client cache** — nå støttet av transients for vedvarende lagring på tvers av forespørsler, noe som forhindrer tap av data under langvarige agentoppgaver
- **Tier 1 site-scrape-verktøy** — sd-ai-agent/site-scrape er nå tilgjengelig som standard i Theme Builder for å analysere konkurrent- og inspirasjonsnettsteder
- **Plugin-radhandlingslenker** — korrigert og omdøpt for tydelighet

## Forbedringer i v1.10.0

- **WooCommerce-integrasjon refaktorert** — bruker nå native WooCommerce API-er for bedre pålitelighet og kompatibilitet
- **Automatisk oppdatering av leverandørliste** — oppdager automatisk når plugins aktiveres eller deaktiveres, og oppdaterer listen over tilgjengelige leverandører
- **Forbedret nettstedsbygger** — forbedrede flerfasede byggeplaner for nettsteder med bedre feilgjenoppretting
- **Tema-bevisste ferdigheter** — fire nye innebygde ferdigheter som automatisk tilpasser seg ditt aktive tema

## Feilrettinger i v1.10.0

- Rettet `navigate-to`-funksjon som forårsaket uendelige omlastingssløyfer på enkelte admin-sider
- Rettet `list-posts`-funksjon som ikke riktig løste kategori- og etikett-navn til slugs
- Rettet WP-CLI-kommandoer som manglet namespace-aliaser etter en tidligere refaktorering
- Rettet hendelsesautomatisering slik at den håndterer nettsteder der automatiseringstabeller ennå ikke er opprettet, på en ryddig måte
- Rettet `memory-save`-funksjon slik at den bruker korrekt namespace-prefiks i systeminstruksjonsbyggeren
- Rettet at skalare verktøyresultater pakkes korrekt inn før de returneres til AI-en
- Rettet at bruksstatistikk håndterer det eldre ability-nøkkelformatet korrekt ved oppgradering fra eldre versjoner
