# Admin Page Creator {#admin-page-creator}

## Innhold {#content}
Admin Page Creator lar deg enkelt legge til nye admin-sider og under-sider i WordPress-administrasjonspanelet (dashboard). Det kan brukes til å lage en velkomstside eller en opplæringsguide som fungerer som en landingsside for nye brukere. Du kan bruke sidebyggere som Beaver Builder, Elementor, Brizy og Oxygen for å designe sidene uten problemer.

Det tilbyr også støtte for tilpasset HTML, CSS og JavaScript, i tillegg til tilgangskontroll basert på plan og brukerrolle!

## Funksjoner {#features}
* Støtte for tilpasset HTML og CSS
* JavaScript-støtte
* PHP Content Mode
* Tillatelser basert på brukerrolle
* Støtte for sidebyggere

## Dokumentasjon {#documentation}

## Installasjon {#installation}

1. Last opp **ultimate-multisite-admin-page-creator** til katalogen **/wp-content/plugins/**
2. Aktiver pluginet via **Plugins**-menyen i ditt WordPress Network Admin

## Brukerveiledning {#usage-guide}

Mens WordPress gjør det veldig enkelt å legge til innhold på nettsidens forside, krever det å lage sider og under-sider på admin-siden noe grunnleggende koding. Ultimate Multisite: Admin Page Creator har som mål å bringe den samme brukervennligheten fra forsiden til bakenden av nettstedene i nettverket ditt.

## Hvorfor bruke Admin Page Creator? {#why-use-admin-page-creator}

Kanskje du ønsker å tilby en tilpasset side på kundenes admin-panel med opplæringsmateriell, eller kanskje du vil legge til en tilpasset side med FAQ og en kontakthåndteringswidget. Mulighetene er grenseløse med Admin Page Creator, og det er det vi skal utforske i dagens veiledning.

## Opprette din første tilpassede admin-side {#creating-your-first-custom-admin-page}

Etter at du har installert og aktivert Ultimate Multisite: Admin Page Creator, naviger til nettverksadmin-menyen og gå til Admin Pages.

![Custom Admin Pages list on Network Admin](/img/admin/admin-pages-list.png)

_Liste over tilpassede admin-sider i Network Admin_

Du vil se en liste over alle admin-sidene du har opprettet så langt (siden dette er ditt første besøk, vil listen være tom). Klikk på **Add new Admin Page**. Du blir omdirigert til redigereren for Admin Page.

### Page Title {#page-title}

Dette er tittelen som vises øverst på siden, ikke navnet i admin-menyen.

## Menyelementer {#menu-options}

På siden, i General Options meta-boksen, kan du konfigurere menytype, etikett og mye mer.

### Menu Types {#menu-types}

Din nye tilpassede admin-side kan ha to forskjellige menytyper: toppnivåmeny og undermeny.

En toppnivåmeny-side, som navnet antyder, vil legge til den nylig opprettede siden din i admin-menyen sammen med standard WordPress-sider.

Hvis du velger toppnivå-alternativet, kan du også velge menyeorden og et menyeikon.

Hvis du ønsker å gjøre denne siden til en undermeny, kan du velge en foreldre-side for å koble denne siden til. Du kan legge til din tilpassede side i standard WordPress-sider eller i dine toppnivå-tilpassede admin-sider.

## Avanserte alternativer {#advanced-options}

Du bør kunne legge til alle slags innhold på dine tilpassede admin-sider. Dette inkluderer CSS-regler og filer, JavaScript og eksterne biblioteker. Du kan gjøre dette ved hjelp av Advanced Options Tab.

### Custom CSS {#custom-css}

På CSS-fanen kan du legge til din egen tilpassede CSS-kode, samt importere CSS-filer fra eksterne steder (som CDN-er, Google Fonts, osv.).

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_Tilpasset CSS støttes også. Du kan også inkludere eksterne stilfiler!_

### Custom JavaScript {#custom-javascript}

Det samme gjelder for JavaScript. Du kan legge til eksterne biblioteker som skal brukes i din tilpassede JavaScript-kode.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Legg til din egen JavaScript-kode!_

## Tillatelser {#permissions}

Også i Advanced Options kan du velge hvilke under-sider som skal ha tilgang til denne siden. Dette er nyttig hvis du planlegger å vise forskjellig eksklusivt innhold til ulike plannivåer, for eksempel.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Du har full kontroll over hvor denne siden skal vises_

**Viktig**: Begge betingelsene må være oppfylt for at siden skal vises. Det betyr at hvis brukeren er en kunde av en gitt plan, men ikke har en av de valgte rollene (eller omvendt), vil siden ikke vises på panelet deres.

## Aktiv {#active}

Hvis du vil deaktivere en tilpasset admin-side uten å slette den eller rote med tillatelsene, bare fjern krysset fra Active-alternativet.

Å deaktivere en toppnivåside vil automatisk deaktivere alle under-sider som er knyttet til den foreldre-siden.
