# Ultimate Multisite Dokumentasjon {#ultimate-multisite-documentation}

Velkommen til Ultimate Multisite-dokumentasjonen. Ultimate Multisite transformerer WordPress Multisite til en fullverdig Website as a Service (WaaS) plattform.

## Dokumentasjonsseksjoner {#documentation-sections}

### [User Guide](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

Alt du trenger for å sette opp, konfigurere og administrere WaaS-plattformen din. Dekker installasjon, betalingsløsninger (payment gateways), domenemapping, klientadministrasjon og mer.

### [Developer Documentation](/developer/hooks) {#developer-documentation}

Teknisk referanse for utviklere som bygger på Ultimate Multisite. Inkluderer referanse for hooks, REST API-dokumentasjon og veiledninger for utvikling av tilleggsfunksjoner (addons).

### [Addons](/addons) {#addons}

Dokumentasjon for betalte tilleggs-plugins (addon plugins) som utvider Ultimate Multisite med ekstra funksjoner som WooCommerce-integrasjon, analyse (analytics), salg av domener og mer.
