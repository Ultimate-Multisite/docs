# Plassering av annonser på nettverket ditt med Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector er et **gratis tillegg** som gir deg verktøyene til å plassere annonser på planene dine direkte fra redigeringssiden.

Etter at du har installert Ultimate Multisite: Ad Injector, går du til _Plugins_-siden for å aktivere det.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Klikk på Network Activate-knappen_

Når tillegget er aktivt, vil metaboksen _Advanced Options_ på redigeringssiden for planen ha et nytt alternativ som heter _Ad Settings_.

![Annonseinnstillinger på redigeringssiden for planen](/img/config/product-edit-full.png)

_Nå kan du redigere annonseinnstillinger for hver av planene dine_

I denne fanen kan du legge til annonsekoder for innlegg og sider på forsiden, før og etter innholdet.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Etter at du har satt inn annonsekoden, vil den vises på innleggene og sidene for den spesifikke planen._

Du kan også ha annonser inkludert i toppteksten på administrasjonssider for brukere av den planen.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Gå til Admin Panel Ad Code-alternativet hvis du vil vise annonser i brukerens kontrollpanel_

### Mer enn bare annonsering

Du kan også bruke Ultimate Multisite: Ad Injector til å tilby support til høyere planer.

Bare gå til _Admin Panel Ads_-alternativet og legg til HTML-koden for supportverktøyet ditt.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Legg til kundestøtte for spesifikke planer_

På denne måten kan folk nå deg direkte fra kontrollpanelet sitt!
