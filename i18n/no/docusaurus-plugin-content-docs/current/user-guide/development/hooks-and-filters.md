# Ultimate Multisite Hooks og filtere

Ultimate Multisite består av en hel rekke funksjoner og databasespørringer. Det har hooks og filtere som lar deg endre eller legge til ny kode i eksisterende funksjoner.

Vi har laget en liste over disse hookene og filtrene som en referanse for å hjelpe avanserte brukere med å tilpasse nettverket sitt ytterligere. Du kan få tilgang til listen fra Github ved å klikke på lenken nedenfor.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
