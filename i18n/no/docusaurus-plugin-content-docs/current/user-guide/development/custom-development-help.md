# Hvordan få hjelp til tilpasset utvikling fra Codeable?

Noen ganger trenger vi virkelig litt kul tilpasning for å få nettverksflytene våre til å fungere akkurat slik vi ønsker. Hvis det er det du leter etter, anbefaler vi [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), et sted hvor du kan finne erfarne utviklere med ekspertise på Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) er en plattform som kobler kunder med kvalifiserte WordPress-frilansutviklere som kan jobbe med prosjekter i alle størrelser. De har en grundig søknadsprosess for å sikre at du blir matchet med eksperter som er spesifikt kvalifisert for oppgaven du har i tankene.

### Hvordan oppretter du et prosjekt i Codeable?

Først går du til [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) og klikker på **Start a Project**-knappen.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Nå, for å sende inn et prosjekt, klikk til venstre på **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

Under «I need help with» velger du **Customizing**:

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

Under «My» velger du **Plugin**:

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Deretter klikker du på **Continue**-knappen.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Nå er det på tide å beskrive prosjektet og behovene dine så godt du kan. Prøv å gi så mye informasjon som mulig!

<!-- Screenshot unavailable: Codeable project description form -->

Svar på følgende spørsmål for å hjelpe med prisestimering.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Etter det, hvis du ikke har en [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE)-konto ennå, oppretter du en og klikker på **Publish My Project**-knappen.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

Når du har publisert [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE)-prosjektet ditt, vil plattformen koble deg med utviklere som matcher ferdighetene som kreves for prosjektet ditt.

Denne prosessen vil hjelpe deg med å finne riktig person for jobben, som har erfaring med WordPress og Ultimate Multisite!

Codeables minimumspris er 70–120 USD per time. Estimater vil variere avhengig av arbeidets omfang, kompleksitet og hvor raskt prosjektet må ferdigstilles.

Når du får estimatet ditt, vil du kunne ansette utvikleren som passer best til dine behov og starte prosjektet sammen!
