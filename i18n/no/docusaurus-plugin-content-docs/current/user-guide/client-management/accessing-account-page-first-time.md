# Hvordan får jeg tilgang til kontosiden min for første gang?

Velkommen til Ultimate Multisite!

Du får tilgang til kontosiden din via lenken du mottok på e-post etter at du kjøpte plugin-en vår:

<!-- Screenshot unavailable: Account page email with access link -->

Bruk e-postadressen du oppga under kjøpet, og passordet du mottok på e-post.

<!-- Screenshot unavailable: Login page with email and password fields -->

Siden dette er første gang du logger inn på denne kontoen, blir du bedt om å endre passordet ditt. Endre det og klikk på _Lagre_-knappen.

<!-- Screenshot unavailable: Change password form with Save button -->

På kontosiden din kan du administrere alle kontodetaljene dine, som fakturering, fornyelser, kanselleringer og mye annet.
