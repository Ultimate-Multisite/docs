# Hvordan få hjelp

Du har fått din gyldige lisensnøkkel, og med den følger tilgang til teamet vårt som kan hjelpe deg med problemer du måtte støte på mens du setter opp Ultimate Multisite.

Denne leksjonen tar for seg de tilgjengelige supportkanalene for lisensinnehavere og hvilken informasjon du bør ha klar når du kontakter supportteamet vårt.

## Hjelpesiden

Du kan finne ulike artikler og veiledninger på hjelpesiden vår via lenken nedenfor.

[)

Vi oppfordrer brukerne våre til å gjøre seg kjent med de grunnleggende funksjonene i Ultimate Multisite. Det finnes «Kom i gang»-artikler på hjelpesiden vår, og du finner også svar på vanlige spørsmål der, som hvordan du installerer tillegg, hvordan du integrerer betalingsløsninger, og lignende.

## Kontakt support

Hvis du står fast og ikke finner svaret på hjelpesiden vår, kan du kontakte supportteamet vårt enten ved å klikke på snakkebobleikonet nederst til høyre på nettsiden vår ELLER ved å sende en e-post direkte til support@ultimatemultisite.com.

Når du kontakter support om tekniske problemer, sørg for å gi en detaljert beskrivelse av problemet sammen med skjermbilder, feilmeldinger eller annen informasjon som gjør det enklere for supportteamet å forstå og vurdere saken din. Du må også oppgi lisensnøkkelen din. Kun aktive lisensinnehavere har rett på support.

## Brukerforum

[Brukerforumet vårt](https://community.ultimatemultisite.com/) er et sted hvor Ultimate Multisite-brukere kan samhandle med hverandre. Det fungerer som en informasjonsplattform der medlemmer kan legge ut innlegg, kommentere diskusjoner, gi råd, samarbeide eller søke hjelp fra fellesskapet. Du må oppgi Ultimate Multisite-lisensnøkkelen din for å få tilgang til lukkede områder/sider.
