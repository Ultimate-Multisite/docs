# Administrere medlemskap

I Ultimate Multisite kan du administrere medlemskap og få tilgang til alle abonnementene i nettverket ditt. Se gjennom og rediger abonnementsdetaljer som planer og produkter, antall nettsteder, fakturabeløp og -frekvens, betalingshistorikk og tidsstempler.

For å få tilgang til denne siden, gå til fanen **Memberships** i Ultimate Multisite-menyen.

![Medlemskapsliste](/img/admin/memberships-list.png)

På Memberships-siden kan du se en liste over kundene dine og produktene knyttet til kontoene deres. Du kan filtrere medlemskap etter status: aktiv, ventende, på vent, utløpt eller kansellert.

Klikk på et medlemskap for å få tilgang til siden der du kan redigere medlemskapsdetaljer — faktureringsinformasjon, nettsted- eller besøksgrenser og mer.

Du kan også legge til nye medlemskap knyttet til brukere som allerede er registrert i nettverket ditt. Klikk på **Add Membership** og konfigurer detaljene.

![Legg til medlemskap-dialog](/img/admin/membership-add-modal.png)

Klikk deretter på **Create Membership**. Det nye medlemskapet vil vises i medlemskapslisten.
