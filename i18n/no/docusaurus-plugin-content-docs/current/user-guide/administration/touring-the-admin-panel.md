# En omvisning i Admin-panelet

Ultimate Multisite er laget for å være så enkelt og selvforklarende som mulig, men med et så kraftig verktøy kan det være nyttig med en guidet omvisning. La oss gå gjennom admin-sidene så du blir kjent med oppsettet.

## Dashboard

Ultimate Multisite sitt **Dashboard** viser grunnleggende rapporter og analyser, inkludert inntekter, nettstedsaktivitet, medlemsvekst, besøkstall og geografiske data.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

## Checkout Forms

**Checkout Forms** gir deg fleksibilitet til å lage tilpassede registreringssider. Du kan tilpasse felter, legge til egne klasser og skript, og begrense tilgang basert på besøkendes land eller plassering.

![Checkout Forms-liste](/img/admin/checkout-forms-list.png)

## Products

I **Products**-seksjonen oppretter du ulike planer, pakker og tjenester for nettverket ditt. Definer priser, faktureringsfrekvenser og begrensninger og kvoter på produktnivå.

![Produktliste](/img/admin/products-list.png)

## Memberships

**Memberships**-siden viser alle abonnementer i nettverket ditt. Se gjennom og rediger abonnementsdetaljer som planer, produkter, fakturabeløp og -frekvenser, betalingshistorikk og tidsstempler.

![Medlemskapsliste](/img/admin/memberships-list.png)

## Payments

**Payments**-siden gir deg en rask oversikt over betalingshistorikken i hele nettverket, med detaljert transaksjonsinformasjon inkludert spesifikke produkter og beløp.

![Betalingsliste](/img/admin/payments-list.png)

## Customers

**Customers**-siden viser alle medlemmer som er registrert i nettverket ditt, med grunnleggende brukerinformasjon som navn, e-postadresse, siste innlogging og en «bytt til»-lenke for å logge inn på deres underside.

![Kundeliste](/img/admin/customers-list.png)

## Sites

**Sites**-siden lister opp alle undersider i nettverket ditt, inkludert nettstedsmaler og kunders egne sider. Administrer enkelt tilknyttede domener, begrensninger og kvoter, plugins og temaer, samt miniatyrbilder for nettsteder.

![Nettstedsliste](/img/admin/sites-list.png)

## Domains

**Domains**-siden er dedikert til egendefinerte domener som er tilknyttet undersider. Som superadmin kan du manuelt legge til eller tilknytte egendefinerte domenenavn og se DNS-poster og logger.

![Domeneliste](/img/admin/domains-list.png)

## Discount Codes

Med **Discount Codes** kan du opprette kupongkoder for å tilby rabatter til sluttbrukerne dine. Angi rabattverdien og målrett mot spesifikke brukere eller grupper.

![Rabattkode-liste](/img/admin/discount-codes-list.png)

## Broadcasts

**Broadcasts** er et verktøy for å sende ut kunngjøringer eller private meldinger til brukerne på undersidene dine. Målrett mot spesifikke brukergrupper basert på planen eller produktet de abonnerer på.

![Kringkastingsliste](/img/admin/broadcasts-list.png)

## Settings

På **Settings**-siden konfigurerer du Ultimate Multisite — registreringsinnstillinger, betalinger, API og webhooks, domenetilknytning og andre integrasjoner.

![Innstillingssiden](/img/admin/settings-general.png)

## Events

**Events**-siden holder oversikt over alle hendelser og logger i nettverket ditt. Den sporer aktiviteter som planendringer, registreringer og andre bevegelser — nyttig for å overvåke multisite-nettverket ditt.

![Hendelsesliste](/img/admin/events-list.png)

## Webhooks

Med **Webhooks** kan du levere data til en annen applikasjon. Nyttig for å sende data fra Ultimate Multisite til plattformer som Zapier.

![Webhook-liste](/img/admin/webhooks-list.png)
