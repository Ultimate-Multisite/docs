# Filter: wu_plan_product_types

Filtrerer produktkategoriene som skal behandles som planer.

Dette filteret lar tilleggspakker (addons) registrere ekstra produktkategorier som skal gjenkjennes som planer under validering og separasjon.

## Parametere

| Navn | Type | Beskrivelse |
|------|------|-------------|
| $plan_types | `array` | Array med produktkategorier som skal behandles som planer. |
