# Filter: wu_independent_billing_cycle_product_types

Filtrerer produktkategorier som har uavhengige faktureringssykluser.

Produkter med uavhengige faktureringssykluser tvinges ikke til å matche planens faktureringsperiode i handlekurven. Dette er nyttig for produkter som domeneregistreringer, som alltid faktureres årlig uavhengig av om planen er månedlig eller årlig.

## Parametere {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | Array av produktkategorier med uavhengige faktureringssykluser. |
