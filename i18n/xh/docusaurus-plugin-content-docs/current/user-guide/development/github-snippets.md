# Indlela yokusebenzisa ii-snippets ze-Ultimate Multisite kwi-repository yethu ye-GitHub {#how-to-use-ultimate-multisite-snippets-on-our-github-repository}

Kukho ii-snippets zekhowudi ezifumanekayo kwi-repository ye-GitHub ezisoloko zicelwa ngabasebenzisi be-Ultimate Multisite abafuna ukongeza imisebenzi emincinci njengokongeza iskripthi se-Google Analytics kumaphepha okubhalisa okanye ukufihla i-meta box kwi-admin dashboard.

Eli nqaku liza kukubonisa indlela yokusebenzisa, okanye ngokuthe ngqo apho kufuneka ubeke khona ezi khowudi.

Ungazifumana ii-snippets kwikhonkco elingezantsi.

https://github.com/next-press/wp-ultimo-snippets/

Kukho iindlela ezi-2 zokongeza ikhowudi

  1. Kwifayile ye-functions.php ye-theme yakho.

  2. Must-Use Plugins (mu-plugins)

# Indlela yokongeza i-snippet kwifayile ye-functions.php ye-theme yakho. {#how-to-add-the-snippet-on-your-themes-functionsphp-file}

  1. Ngena kwi-WordPress Network admin dashboard yakho uze uye ku-Themes >Theme Editor (Jonga umfanekiso wesikrini ongezantsi).

  2. Kwiphepha elithi "Edit Themes", qinisekisa ukuba ukhethe i-theme yakho esebenzayo kwindawo yokukhetha eyehlayo ekwicala eliphezulu lasekunene lesikrini sakho (#3 kumfanekiso wesikrini ongezantsi).

  3. Cofa ifayile ye-functions.php phantsi kwecandelo elithi "Theme Files" ukuze ulayishe ifayile. Skrolela ezantsi uze uncamathisele i-snippet ye-Ultimate Multisite oyifumene kwi-repository ye-GitHub.

# Indlela yokudala ii-Must-Use Plugins (mu-plugins) {#how-to-create-must-use-plugins-mu-plugins}

I-WordPress inophawu olukuvumela ukuba ulayishe umsebenzi owenzelwe wena obizwa ngokuba yi-"Must-Use Plugins", okanye "mu-plugins" ngokufutshane.

Ezi mu-plugins zikhethekileyo zilayishwa phambi kwazo zonke ezinye ii-plugins eziqhelekileyo, kwaye azinakucinywa ukusebenza. Kwi-multisite network, ikhowudi kwezi mu-plugins iza kulayishwa kuzo zonke iisayithi kufakelo lwakho.

1\. Sebenzisa i-FTP okanye i-SSH ukufikelela kwi-filesystem yofakelo lwakho lwe-WordPress.

2\. Ngaphakathi kwe-directory ye-wp-content yofakelo lwakho lwe-WordPress, yenza i-directory entsha ebizwa ngokuba: mu-plugins.

<!-- Screenshot unavailable: File manager showing wp-content directory with mu-plugins folder -->

3\. Yenza ifayile entsha ye-PHP kwikhompyutha yakho ebizwa wu-snippet.php usebenzisa i-Notepad okanye nawuphi na umhleli wekhowudi.

4\. Beka i-snippet yekhowudi ye-Ultimate Multisite oyifumene kwi-repository ye-GitHub efayilini uze uyigcine. Ungongeza nale khowudi ngaphezulu kwe-snippet yekhowudi ukuze ubhale ileyibhile ye-MU plugin yakho.
