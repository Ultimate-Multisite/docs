# Ultimate Multisite iihuku neefilitha {#ultimate-multisite-hooks-and-filters}

Ultimate Multisite yenziwe ngumthwalo omkhulu wemisebenzi nemibuzo yedathabheyisi. Ineehuku neefilitha eziza kukuvumela ukuba uguqule okanye wongeze ikhowudi entsha kwimisebenzi esele ikhona.

Senze uluhlu lwezi hoku neefilitha njengereferensi ukunceda abasebenzisi abaphambili ukuba baqhubeke belungiselela inethiwekhi yabo. Ungafikelela kuluhlu oluvela ku-Github ngokucofa ikhonkco elingezantsi.
