# Ukurhoxisa Ubulungu (v2) {#canceling-a-membership-v2}

_**INQAKU ELIBALULEKILEYO: Eli nqaku libhekisa ku-Ultimate Multisite version 2.x.**_

Ukurhoxisa ubulungu kwicala labathengi bakho kulula kakhulu.

Ukuba umsebenzisi wakho ufuna ukurhoxisa ubulungu bakhe, kufuneka aqale angene kwiphepha lakhe le-****Account****.

Apho, kufuneka aye ku-**Indawo Yengozi** , aze acofe ku-**Cima Account**.

![Iphepha le-Account elineNdawo Yengozi kunye neqhosha le-Cima Account](/img/account-page/danger-zone.png)

Imodyuli iya kuvuleka icela isiqinisekiso sakhe. Kufuneka avule ukhetho lwe-**Qinisekisa Ukucinywa kwe-Account** aze acofe ku-**Cima Account**.

![Ingxoxo ethi Qinisekisa Ukucinywa kwe-Account](/img/account-page/delete-account-confirm.png)

Oku kuya kumkhupha kwisayithi kuze kurhoxise ubulungu bakhe kwinethiwekhi yakho, kwakunye nokucima isayithi yakhe kwinethiwekhi yakho.

![Ukhutshiwe emva kokucinywa kwe-account](/img/account-page/logged-out-after-delete.png)

![Isiqinisekiso sokurhoxiswa kobulungu](/img/account-page/membership-canceled.png)

**Kubalulekile:** Qaphela ukuba eli nyathelo aliyi kumsusa umthengi kwinethiwekhi yakho, kuba umthengi angaba nobulungu obungaphezu kobunye okanye angabi nabulungu buqhagamshelwe kuye.

![Umthengi usekho kwinethiwekhi emva kokurhoxisa](/img/admin/customer-after-cancel.png)

![Umthengi ongenabo ubulungu obusebenzayo](/img/admin/memberships-customer-empty.png)
