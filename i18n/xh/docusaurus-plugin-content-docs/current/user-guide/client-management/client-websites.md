# Iiwebhusayithi zoMthengi wakho (v2)

Emva kokubhalisa kuthungelwano lwakho, abathengi bakho banokufikelela kwi-Dashboard yabo.

Ngaphandle kwephepha le-**Account**, apho banokubona ulwazi olubalulekileyo malunga neentlawulo zabo, ubulungu, iidomeyini, imida yeplan kunye nokunye, bakwanokufikelela kwiphepha le-**IiSayithi Zam**, apho banokubona baze balawule zonke iiwebhusayithi zabo. Into ekufuneka bayenze kukucofa ku-IiSayithi Zam kwi-Dashboard yabo.

![Ikhonkco le-IiSayithi Zam kwi-Dashboard yomthengi](/img/account-page/my-sites-link.png)

Baza kuboniswa zonke iiwebhusayithi zabo, ngeqhosha elithi **Lawula** kanye ngezantsi kwenye nganye kuzo. Ukuba bacofa kulo, baza kuthunyelwa kwakhona kwi-**Dashboard yale webhusayithi ethile**.

Banokubona nokuba yeyiphi kwiwebhusayithi zabo eyi-**primary** kwiiplan ezibavumela ukuba babe newebhusayithi engaphezu kwenye.

![Iphepha le-IiSayithi Zam elibonisa iiwebhusayithi zomthengi kunye namaqhosha okuLawula](/img/account-page/my-sites-page.png)

Nanku umbono opheleleyo wonxibelelwano lokuhlela isayithi:

![Unxibelelwano olupheleleyo lokuhlela isayithi](/img/admin/site-edit-full.png)

Ukuba bafuna ukutshintsha iwebhusayithi yabo eyi-primary, kufuneka baye kwiphepha le-Account ngokucofa kwimenyu ye-**Account**.

![Ikhonkco lemenyu ye-Account kwi-Dashboard yomthengi](/img/account-page/account-menu.png)

Unokujonga kwaye uhlele iinkcukacha zomthengi:

![Unxibelelwano lokuhlela umthengi](/img/admin/customer-edit.png)

Emva koko, kufuneka bacofe ku-**Tshintsha iSayithi eMiselweyo**.

![Iqhosha lokutshintsha iSayithi eMiselweyo kwiphepha le-account](/img/account-page/change-default-site.png)

Ifestile ephumayo iza kuvela. Cofa kuluhlu olwehlayo uze ukhethe ukuba yeyiphi iwebhusayithi ofuna ukuyenza iwebhusayithi eyi-primary.

![Ukhetho loluhlu olwehlayo lokutshintsha iSayithi eMiselweyo](/img/account-page/change-default-site-dropdown.png)

Emva koko, cofa nje ku-**Tshintsha iSayithi eMiselweyo** kwaye ugqibile!

![Ukuqinisekisa utshintsho lwesayithi emiselweyo](/img/account-page/change-default-site-confirm.png)
