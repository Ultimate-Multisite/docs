# Ukwehlisela isicwangciso kwinqanaba elisezantsi (v2)

_**INQAKU ELIBALULEKILEYO: Eli nqaku libhekisa kwi-Ultimate Multisite version 2.x.**_

Ukwehlisela isicwangciso okanye umrhumo kwinqanaba elisezantsi sisenzo esiqhelekileyo abathengi bakho abanokusenza ukuba banohlahlo-lwabiwo-mali olulinganiselweyo okanye bagqibe kwelokuba abazukufuna izixhobo ezininzi ukuqhuba isayithi yabo engaphantsi.

## Indlela yokwehlisela isicwangciso kwinqanaba elisezantsi {#how-to-downgrade-a-plan}

Abathengi bakho bangakwehlisela isicwangciso sabo kwinqanaba elisezantsi nangaliphi na ixesha ngokungena kwi-Dashboard yolawulo yesayithi yabo engaphantsi baze bacofe **Tshintsha** phantsi kwephepha labo le-Account.

![Iphepha le-Account lomthengi elinekhadi elithi Ubulungu Bakho neqhosha elithi Tshintsha](/img/account-page/membership-change-button.png)

Emva kokucofa iqhosha elithi **Tshintsha**, umsebenzisi/umthengi uza kuthunyelwa kwiphepha lokuphuma apho anokukhetha isicwangciso afuna ukutshintsha umrhumo wakhe uye kuso.

![Iphepha leenketho zokwehlisela isicwangciso kwicala lomthengi](/img/account-page/downgrade-picker.png)

Kulo mzekelo, sehlisela isicwangciso ukusuka ku-**Premium** ukuya ku-**Free**.

Ukuqhubeka, umsebenzisi kufuneka acofe nje iqhosha elithi **Gqibezela Ukuphuma**. Emva koko liza kumbuyisela kwiphepha le-Account elibonisa umyalezo malunga notshintsho olusalindileyo lobulungu. Utshintsho luza kuqala ukusebenza **kumjikelo olandelayo wokuhlawuliswa** womthengi.

![Iphepha le-Account elibonisa ibhena yotshintsho lobulungu olusalindileyo](/img/account-page/pending-change.png)

### Kwenzeka ntoni xa umsebenzisi ehlisela isicwangciso sakhe kwinqanaba elisezantsi {#what-happens-when-a-user-downgrades-their-plan}

Kubalulekile ukuqaphela ukuba ukwehlisela isicwangciso kwinqanaba elisezantsi akutshintshi uqwalaselo olukhoyo kwisayithi engaphantsi yomsebenzisi.

Akutshintshi ngokuzenzekelayo itemplate yesayithi kuba ukutshintsha itemplate yesayithi kuya kucima ngokupheleleyo kuze kusete ngokutsha isayithi engaphantsi. Oku kwenzelwa ukuthintela ukulahleka kwedatha okungafunekiyo. Ngoko indawo yediski, imixholo, izandiso njl. ziya kuhlala zinjalo ngaphandle kwezithuba.

Siyaqonda ukuba eyona nkxalabo yakho inokuba yimida nezabelo ozimise phantsi kwesicwangciso ngasinye, kodwa kufuneka siqwalasele umonakalo onokwenziwa kwisayithi engaphantsi yomsebenzisi ukuba sicima okanye sitshintsha naluphi na uqwalaselo lwayo.

Kwizithuba ezigqitha umda omiselwe kwisicwangciso, uneenketho ezi-3 ezahlukeneyo: **Gcina izithuba njengoko zinjalo** *,* **Hambisa izithuba emgqomeni** *,* okanye **Hambisa izithuba kuyilo olungapapashwanga** *.* Ungakuqwalasela oku phantsi kweesethingi ze-Ultimate Multisite.

![Iphepha leNetwork Admin Settings Sites elibonisa iinketho zokuziphatha xa umda wezithuba ugqithiwe](/img/account-page/settings-sites-post-limit.png)

### Kwenzeka ntoni kwintlawulo {#what-happens-to-the-payment}

Kwi-version 2.0, akusafuneki naluphi na uhlengahlengiso kwintlawulo ngokwe-proration.

Oku kungenxa yokuba inkqubo iza kulinda ubulungu obukhoyo ukuba **bugqibe umjikelo wabo wokuhlawuliswa ngaphambi kokuba** isicwangciso/ubulungu obutsha buqale ukusebenza. Isixa esitsha sokuhlawulisa sobulungu obutsha siya kusetyenziswa ngokuzenzekelayo size sihlawuliswe kumjikelo olandelayo wokuhlawuliswa.
