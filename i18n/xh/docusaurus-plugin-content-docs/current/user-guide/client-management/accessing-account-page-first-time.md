# Ndingalifikelela njani iphepha le-Account yam okokuqala? {#how-can-i-access-my-account-page-for-the-first-time}

Wamkelekile kwi-Ultimate Multisite!

Fikelela kwiphepha lakho le-Account ngekhonkco olifumene kwi-imeyile yakho emva kokuthenga iplagi yethu:

<!-- Umfanekiso wesikrini awufumaneki: I-imeyile yephepha le-Account enekhonkco lokufikelela -->

Sebenzisa i-imeyile oyinikezileyo ngexesha lokuthenga kunye negama lokugqitha olifumene kwi-imeyile yakho.

<!-- Umfanekiso wesikrini awufumaneki: Iphepha lokungena elinemimandla ye-imeyile negama lokugqitha -->

Kuba eli lixesha lakho lokuqala ukungena kule Account, uya kucelwa ukuba utshintshe igama lakho lokugqitha. Litshintshe uze ucofe iqhosha elithi _Gcina_.

<!-- Umfanekiso wesikrini awufumaneki: Ifom yokutshintsha igama lokugqitha eneqhosha elithi Gcina -->

Kwiphepha lakho le-Account, ungalawula zonke iinkcukacha ze-Account yakho ezifana neentlawulo, ukuhlaziywa, ukurhoxiswa, kunye nezinye izinto ezininzi.
