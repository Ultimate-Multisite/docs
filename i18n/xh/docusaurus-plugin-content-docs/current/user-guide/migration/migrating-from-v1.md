# Ukufuduka ukusuka ku-V1

## Ultimate Multisite itshintshile ukusuka kusapho lwayo lokuqala lokukhutshwa lwe-1.x ukuya kusapho lokukhutshwa lwe-2.x.

Ultimate Multisite inguqulelo 2.0 nangaphezulu ibhalwe ngokutsha ngokupheleleyo i-codebase, nto leyo ethetha ukuba kuncinci kakhulu okwabelwana ngako phakathi kwenguqulelo endala nentsha. Ngenxa yoko, xa uphucula ukusuka ku-1.x ukuya ku-2.x, idatha yakho kuya kufuneka ifuduselwe kwifomathi ezinokuqondwa ziinguqulelo ezintsha.

Ngethamsanqa, Ultimate Multisite 2.0+ **iza nomfudusi** owakhelwe kwi-core okwazi ukubhaqa idatha esuka kwinguqulelo endala aze ayiguqulele kwifomathi entsha. Oku kufuduka kwenzeka ngexesha le-**Setup Wizard** yenguqulelo 2.0+.

Esi sifundo sigubungela indlela asebenza ngayo umfudusi, into omawuyenze kwiimeko zokusilela, kunye nendlela yokusombulula iingxaki ezinokuvela ngexesha lale nkqubo.

_**KUBALULEKILE: Phambi kokuba uqale ukuphucula ukusuka kwinguqulelo 1.x ukuya kwinguqulelo 2.0 nceda uqinisekise ukuba wenza ikopi yokhuseleko yedatabase yesayithi yakho**_

## Amanyathelo okuqala

Inyathelo lokuqala kukukhuphela ifayile ye-plugin .zip uze ufake inguqulelo 2.0 kwi-network admin dashboard yakho.

Emva kokuba [ufake waza wavula inguqulelo 2.0](../getting-started/installing-ultimate-multisite.md), inkqubo iya kubhaqa ngokuzenzekelayo ukuba i-Multisite yakho isebenza kwinguqulelo endala kwaye uya kubona lo myalezo phezulu kwiphepha le-plugin.

_**QAPHELA:** Ukuba une-Ultimate Multisite 1.x efakiweyo kwi-Multisite yakho, uya kuba nenketho yokutshintsha i-plugin ngenguqulelo osandul’ ukuyikhuphela. Nceda, qhubeka ucofe ku-**Replace current with uploaded**._

<!-- Screenshot unavailable: Plugin page showing option to replace v1 with uploaded v2 version -->

Iphepha elilandelayo liya kukwazisa ukuba zeziphi ii-add-ons zelifa ozifakileyo kunye nenguqulelo 1.x. Liya kuba nemiyalelo yokuba inguqulelo oyisebenzisayo iyahambelana na nenguqulelo 2.0 okanye kufuneka ufake inguqulelo ephuculiweyo ye-add-on emva kokufuduka.

<!-- Screenshot unavailable: Message on the top of the plugins page about updating to v2.0, with list of add-ons that need updating -->

Nje ukuba ukulungele ukuqhubeka, ungacofa iqhosha elithi **Visit the Installer to finish the upgrade**.

<!-- Screenshot unavailable: Button saying Visit the Installer to finish the upgrade -->

Emva koko iya kukuzisa kwiphepha le-installation wizard elinemiyalezo embalwa yokwamkela. Kufuneka ucofe nje **Get Started** ukuze uye kwiphepha elilandelayo.

<!-- Screenshot unavailable: Setup Wizard welcoming page with Get Started button -->

Emva kokucofa **Get Started** , iya kukuthumela kwakhona kwi-Pre-install Checks_._ Oku kuya kukubonisa i-System Information yakho kunye nokufakwa kwe-WordPress kwaye kukuxelele ukuba kuyahlangabezana na **neemfuno ze-Ultimate Multisite**.

<!-- Screenshot unavailable: Pre-install Checks page showing system requirements confirmation -->

Inyathelo elilandelayo kukufaka i-license key yakho ye-Ultimate Multisite kwaye uvule i-plugin. Oku kuya kuqinisekisa ukuba zonke iifitsha, kuquka ii-add-ons, ziya kufumaneka kwisayithi yakho.

<!-- Screenshot unavailable: License activation page with license key input and Agree and Activate button -->

Emva kokufaka isitshixo sakho, cofa **Agree & Activate**.

Emva kokuvula ilayisensi, ungaqala ukufakwa kokwenyani ngokucofa **Install** kwiphepha elilandelayo. Oku kuya kudala ngokuzenzekelayo iifayile eziyimfuneko kunye nedatabase efunekayo ukuze inguqulelo 2.0 isebenze.

<!-- Screenshot unavailable: Installation page showing what will be updated with Install button -->

## Ngoku, ukufuduka

Umfudusi unefitsha yokhuseleko eyakhelweyo apho iya kujonga yonke i-multisite yakho ukuqinisekisa ukuba yonke idatha yakho ye-Ultimate Multisite inokufuduswa ngaphandle kweengxaki. Cofa iqhosha elithi **Run Check** ukuqalisa inkqubo.

<!-- Screenshot unavailable: Migration page with Run Check button to verify data can be converted -->

Emva kokusebenzisa uvavanyo, unezinto ezimbini ezinokwenzeka: isiphumo sinokuba **sinempazamo** okanye **singenampazamo**.

### Kunempazamo

Ukuba ufumana umyalezo wempazamo, kuya kufuneka unxibelelane neqela lethu lenkxaso ukuze likuncede ukulungisa impazamo. Qinisekisa ukuba **unikezela nge-error log** xa usenza itikiti. Ungakhuphela i-log okanye ucofe ikhonkco elithi nxibelelana neqela lethu lenkxaso. Iya kuvula i-widget yoncedo kwicala lasekunene lephepha lakho enamasimi agcwaliswe kwangaphambili kuwe aquka ii-error logs phantsi kwenkcazo.

_**Kuba inkqubo ifumene impazamo, awuyi kukwazi ukuqhubeka nokufudukela kwinguqulelo 2.0. Ungabuyela kwinguqulelo 1.x ukuze uqhubeke nokuqhuba inethiwekhi yakho de impazamo ilungiswe.**_

### Akukho mpazamo

Ukuba inkqubo ayifumani mpazamo, uya kubona umyalezo wempumelelo kunye neqhosha elithi **Migrate** ezantsi eliya kukuvumela ukuba uqhubeke nokufuduka. Kweli phepha, uya kukhunjuzwa ukuba wenze ikopi yokhuseleko yedatabase yakho phambi kokuba uqhubeke, nto leyo esiyincomayo kakhulu. Cofa **Migrate** ukuba sele unayo ikopi yokhuseleko.

Kwaye yiyo yonke into efunekayo!

Ungaqhubeka nokuqhuba ukuseta kwe-Wizard ukuze uhlaziye ilogo yakho nezinye izinto kwinethiwekhi yakho okanye uqale ukujonga imenyu ye-Ultimate Multisite inguqulelo 2.0 kunye nojongano lwayo olutsha. Qhubeka wonwabe.
