# Isiganeko: Bhalisa i-akhawunti nge-Zapier {#event-register-an-account-via-zapier}

Kwinqaku [Ukudibanisa Ultimate Multisite ne-Zapier](zapier.md), sixoxe ngendlela yokusebenzisa i-Zapier ukwenza izenzo ezahlukeneyo ngaphakathi kwe-Ultimate Multisite ngokusekelwe kwizivuseleli neziganeko. Kweli nqaku, siza kubonisa indlela onokudibanisa ngayo ii-aplikeshini zomntu wesithathu. Siza kusebenzisa i-Google Sheets njengomthombo wedatha size sithumele ulwazi kwi-Ultimate Multisite ukuze kubhaliswe i-akhawunti.

Ngaphambi kokudibanisa i-Zapier, yiya ku-**Ultimate Multisite > Settings > API & Webhooks** uze uqinisekise ukuba i-API ivuliwe. Khuphela i-API Key kunye ne-API Secret kwesi sikrini xa i-Zapier icela iinkcukacha ze-akhawunti ye-Ultimate Multisite.

![Iisetingi ze-API kunye nee-Webhooks ezine-API Key, API Secret, kunye nokhetho lwe-Enable API](/img/admin/settings-api-webhooks.png)

Okokuqala, kufuneka wenze **Google Sheet** phantsi kwe-Google Drive yakho. Qinisekisa ukuba uchaza ngokufanelekileyo ikholamu nganye ukuze ukwazi ukuthelekisa idatha ngokulula kamva.

Emva kokudala i-Google sheet, ungangena kwi-akhawunti yakho ye-Zapier uze uqale ukwenza i-zap.

Phantsi kwendawo yokukhangela ye-**"App event"** khetha **"Google Sheets"**


Emva koko kwindawo ethi "**Event** " khetha "**New spreadsheet row** " uze ucofe "**Continue** "

Inyathelo elilandelayo liza kukucela ukuba ukhethe **Google Account** apho i-**Google Sheet** igcinwe khona. Ngoko qinisekisa nje ukuba i-akhawunti ka-google echanekileyo ichaziwe.


Phantsi kwe-**"Set up trigger** ", kuya kufuneka ukhethe kwaye uchaze i-google spreadsheet kunye ne-worksheet oza kuyisebenzisa apho idatha iza kuvela khona. Qhubeka nje uzigcwalise ezo zinto uze ucofe "**Continue** "

Okulandelayo kukuba "**test your trigger** " ukuze uqinisekise ukuba i-google sheet yakho idityaniswe ngokufanelekileyo.

Ukuba uvavanyo lwakho luphumelele, kufuneka ubone isiphumo esibonisa amaxabiso athile avela kwii-spreadsheets zakho. Cofa "**Continue** " ukuze uqhubeke.

Inyathelo elilandelayo kukuseta isenzo sesibini esiza kudala okanye sibhalise i-akhawunti kwi-Ultimate Multisite. Kwibala lokukhangela khetha "**Ultimate Multisite(2.0.2)** "


Phantsi kwendawo ethi "**Event** ", khetha "**Register an Account in Ultimate Multisite** " emva koko ucofe iqhosha elithi "**Continue** ".

Phantsi kwe-"**Set up an action** ", uza kubona amabala ahlukeneyo afumanekayo kwidatha yomthengi, ubulungu, iimveliso, njl. Ungathelekisa amaxabiso aphantsi kwe-google sheet yakho uze uwabele kwibala elifanelekileyo apho kufuneka azaliswe khona njengoko kubonisiwe kumfanekiso-skrini ongezantsi.


Emva kokuthelekisa amaxabiso, ungavavanya isenzo.
