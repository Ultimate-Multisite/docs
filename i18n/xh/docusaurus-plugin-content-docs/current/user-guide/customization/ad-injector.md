# Ukubeka iintengiso kuthungelwano lwakho nge-Ultimate Multisite: Ad Injector {#placing-ads-on-your-network-with-ultimate-multisite-ad-injector}

Ultimate Multisite: Ad Injector yi-**add-on yasimahla** ekunika izixhobo zokubeka iintengiso kwiiplani zakho ngqo kwiphepha lazo lokuhlela.

Emva kokufaka i-Ultimate Multisite: Ad Injector, yiya kwiphepha lakho le-_Plugins_ ukuze uyivule.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Cofa iqhosha le-Network Activate_

Nje ukuba i-add-on isebenze, ibhokisi ye-meta ye-_Advanced Options_ kwisikrini sokuhlela iplani iya kuba nokhetho olutsha olubizwa ngokuba yi-_Ad Settings_.

![Ad Settings kwisikrini sokuhlela iplani](/img/config/product-edit-full.png)

_Ngoku ungahlela izicwangciso zeentengiso kwiplani nganye kwezakho_

Kule tab, uya kukwazi ukongeza iikhowudi zeentengiso kwizithuba nakumaphepha kwi-front-end, ngaphambi nasemva komxholo.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Emva kokufaka ikhowudi yentengiso yakho, iya kubonakala kwizithuba nakumaphepha ale plani ikhethekileyo._

Ungakwazi kananjalo ukubandakanya iintengiso kwi-header yamaPhepha oLawulo kubasebenzisi baloo plani.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Yiya kukhetho lwe-Admin Panel Ad Code ukuba ufuna ukubonisa iintengiso kwi-back-end yomsebenzisi wakho_

### Ngaphaya kweNtengiso {#beyond-advertising}

Ungasebenzisa kananjalo i-Ultimate Multisite: Ad Injector ukunikezela ngenkxaso kwiiplani ezikumgangatho ophezulu.

Yiya nje kukhetho lwe-_Admin Panel Ads_ uze wongeze ikhowudi ye-HTML yesixhobo sakho senkxaso.
