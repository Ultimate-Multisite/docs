# Indlela yokufumana uncedo

Ufumene i-License key yakho esemthethweni, kwaye enxulumene nayo kukho ukufikelela kwiqela lethu ukuze likuncede ukusombulula iingxaki onokudibana nazo ngexesha lakho lokuseta Ultimate Multisite.

Esi sifundo sigubungela amajelo enkxaso akhoyo kubanini beelayisensi kunye nohlobo lolwazi oya kudinga ukuba ube nalo ukuze uqhagamshelane neqela lethu lenkxaso.

## Isayithi yoncedo

Ungafikelela kumanqaku ahlukeneyo kunye nezifundo ezisuka kwisayithi yethu yoncedo onokuyifumana kwikhonkco elingezantsi.

[)

Sikhuthaza abasebenzisi bethu ukuba baziqhelanise neempawu ezisisiseko kunye nemisebenzi ye-Ultimate Multisite. Kukho amanqaku athi "Getting Started" afumanekayo kwisayithi yethu yoncedo, kwaye iimpendulo kwimibuzo eqhelekileyo zinokufumaneka apho nazo, njengendlela yokufaka ii-add-ons, indlela yokudibanisa payment gateways, njl.

## Qhagamshelana nenkxaso

Ukuba uyabambeka kwaye awukwazi ukufumana impendulo kwisayithi yethu yoncedo, ungaqhagamshelana neqela lethu lenkxaso nokuba ngokucofa uphawu lwebhaluni yencoko olukwicala lasekunene elisezantsi lewebhusayithi yethu OKANYE ngokuthumela i-imeyile ngqo ku-support@ultimatemultisite.com.

Xa uqhagamshelana nenkxaso ngemiba yobugcisa, nceda uqinisekise ukuba unika inkcazo eneenkcukacha yengxaki yakho kunye ne-screenshot, imiyalezo yeempazamo, okanye naluphi na ulwazi oluya kuba luncedo kubantu bethu benkxaso ukuze baqonde kwaye bavavanye ingxaki yakho ngokulula. Kufuneka unike ne-license key yakho. Ngabanini beelayisensi ezisebenzayo kuphela abanelungelo lenkxaso.

## Isayithi yoluntu

[Isayithi yethu yoluntu](https://community.ultimatemultisite.com/) yindawo apho abasebenzisi be-Ultimate Multisite banokusebenzisana khona. Isebenza njengenkqubo yolwazi apho amalungu anokupapasha, aphawule kwiingxoxo, anike iingcebiso, asebenzisane, okanye afumane izimvo nolwazi kuluntu. Kuya kufuneka unike i-Ultimate Multisite license key yakho ukuze uvule iindawo/amaphepha abucala.
