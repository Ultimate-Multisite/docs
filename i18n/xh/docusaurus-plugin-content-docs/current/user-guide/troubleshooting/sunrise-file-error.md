# Impazamo xa kufakwa ifayile ye-Sunrise

Ifayile ye-sunrise.php yifayile ekhethekileyo i-WordPress eyikhangelayo ngoxa iziqalisa. Ukuze i-WordPress ikwazi ukubona ifayile ye-sunrise.php, kufuneka ibekwe ngaphakathi **kwefolda ye-wp-content**.

Xa uvula i-Ultimate Multisite kwaye udlula kwi-wizard yokuseta efana nale onayo kumfanekiso-skrini, i-Ultimate Multisite izama ukukopa ifayile yethu ye-sunrise.php iye kwifolda ye-wp-content.

<!-- Screenshot unavailable: Iphepha le-wizard yokuseta elibonisa inyathelo lokufakwa kwe-sunrise.php -->

Amaxesha amaninzi, siyakwazi ukukopa ifayile ngempumelelo kwaye yonke into isebenze. Nangona kunjalo, ukuba kukho into engasetywanga kakuhle (iimvume zefolda, umzekelo), ungadibana nemeko apho i-Ultimate Multisite ingakwazi ukukopa ifayile.

Ukuba ufunda umyalezo wempazamo i-Ultimo ekunika wona, uya kubona ukuba kanye yile nto yenzekileyo apha: **Ukukopa i-Sunrise kusilele**.

<!-- Screenshot unavailable: Umyalezo wempazamo obonisa ukuba ukukopa i-Sunrise kusilele -->

Ukulungisa oko, ungakopa nje ifayile ye-sunrise.php ngaphakathi kwefolda ye-plugin ye-wp-ultimo uze uyincamathisele kwifolda yakho ye-wp-content. Emva kokuba wenze oko, layisha kwakhona iphepha le-wizard kwaye iitshekhi kufuneka ziphumelele.

<!-- Screenshot unavailable: Umphathi weefayile obonisa i-sunrise.php ngaphakathi kwefolda ye-plugin ye-wp-ultimo --> Nakweyiphi na imeko, oku kunokufuna uhlolo ngokubanzi lweemvume zeefolda zakho ukuze uphephe ukuba neengxaki kwixesha elizayo (hayi kuphela nge-Ultimate Multisite kodwa nangezinye ii-plugin kunye nemixholo ngokunjalo).

**Isixhobo se-Health Check** esiyinxalenye ye-WordPress (ungasifikelela nge-**paneli yolawulo > Izixhobo > Health Check** yesayithi yakho ephambili) siyakwazi ukukwazisa ukuba iimvume zeefolda zakho zisetyelwe kumaxabiso anokubangela iingxaki nge-WordPress.
