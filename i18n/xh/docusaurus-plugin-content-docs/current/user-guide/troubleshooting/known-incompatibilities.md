# Ukungahambelani okwaziwayo nezinye iiPlugins

Emva kokufaka nokuvula i-Ultimate Multisite, unokudibana nemiba ngenxa yokungahambelani okungakalungiswa nezinye ii-plugin.

Sisebenza ngenkuthalo ukunciphisa olu luhlu ngenguqulelo entsha nganye, kodwa ukuba i-plugin idweliswe apha, kusenokufuneka uyicime ukuze ube ne-Ultimate Multisite Network esebenza ngokupheleleyo.

Nalu uluhlu lwazo kunye nento ekufuneka uyenze kwezi meko:

**Ezinye ii-plugin ze-WPMU DEV** Ezinye zee-plugin _ezingasagcinwanga_ yi-WPMU DEV ziyaziwa ngokubangela ukuba inkqubo yobhaliso ye-Ultimate Multisite ime kwinyathelo lokugqibela (umsebenzisi akakwazi ukwenza Account). Nalu uluhlu lwee-plugin ezibangela ezo ngxaki:

  * Signup Language: _Ultimate Multisite ine-add-on enika umsebenzi ofanayo, ebizwa ngokuba yi-Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Oku kwakhelwe kwi-Ultimate Multisite, ngoko akukho sidingo se-plugin eyongezelelweyo._

  * New Blog Templates: _Oku kwakhelwe kwi-Ultimate Multisite, ngoko akukho sidingo se-plugin eyongezelelweyo._

  * Set Password on Multisite Blog Creation: _Oku kwakhelwe kwi-Ultimate Multisite, ngoko akukho sidingo se-plugin eyongezelelweyo._

  * Signup TOS: _Oku kwakhelwe kwi-Ultimate Multisite, ngoko akukho sidingo se-plugin eyongezelelweyo._

  * Anti-Splog: Nangona inamandla amancinci, nayo _yakhelwe kwi-Ultimate Multisite._

**Qaphela** : Zonke ezinye ii-plugin ze-WPMU Dev zinokusetyenziswa ngokwesiqhelo kunye ne-Ultimate Multisite. Imizekelo iquka _Smush_ , _Forminator_ , _Defender,_ njl.
