# Ukulawula Amakhonkco Anokwabelwana Ngawo Ezicwangciso {#managing-shareable-links-for-plans}

Ultimate Multisite ikunika ukhetho lokwabelana ngamakhonkco esicwangciso ngqo nabathengi bakho. Ukuze wenze oku, yiya nakweliphi na iphepha lokuhlela lezicwangciso zakho uze ucofe iqhosha elithi _Cofa ukukopa iKhonkco elinokwabelwana ngalo_.

![Iphepha lokuhlela imveliso elineqhosha lekhonkco elinokwabelwana ngalo](/img/config/product-edit-full.png)

_KwiPhepha lokuHlela leSicwangciso, ungafumana iqhosha lokudala amakhonkco anokwabelwana ngawo_

Qaphela ukuba le ndlela isebenzisa ubuninzi bentlawulo obungagqibekanga ukwakha ikhonkco. Oku kuthetha ukuba ukuba intlawulo yakho engagqibekanga yeyanyanga, umzekelo, ekugqibeleni, uya kuba nekhonkco elinje: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Ukuba ufuna ikhonkco elinokwabelwana ngalo libe yeyekota ngokungagqibekanga, kuya kufuneka wongeze u-3 ekugqibeleni

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Ukukhetha isicwangciso _premium_ kwishedyuli yonyaka, wongeza u-1 emva koko igama:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Khumbula ukuba awunakuba nekhonkco elinye kuphela lobuninzi bentlawulo obahlukeneyo. Kufuneka wabelane ngekhonkco elithile ngalinye kubuninzi bentlawulo yakho nganye.

Nje ukuba wabelane ngekhonkco nabathengi bakho, liya kubathumela kwinyathelo lokuqala lenkqubo yakho yokubhalisa.
