# Ukukhangela kwi-Intanethi

Gratis AI Agent v1.5.0 yongeza isakhono se-**Ukukhangela kwi-Intanethi** esivumela umncedisi we-AI ukuba afumane ulwazi oluhlaziyiweyo kwiwebhu ngexesha lencoko. Oku kuxhaswa yi-[Brave Search API](https://brave.com/search/api/) okanye yi-[Tavily API](https://tavily.com/).

## Indlela Esebenza Ngayo {#how-it-works}

Xa ukukhangela kwi-intanethi kuvuliwe, umncedisi unokubuza ngokuzenzekelayo umboneleli wokukhangela ommisileyo xa egqiba ukuba umbuzo ufuna ulwazi lwangoku okanye lwangaphandle — umzekelo, iindaba zamva nje, amaxabiso aphilayo, okanye uxwebhu olunokuba lutshintshile ukusukela kumhla wokuqeqeshwa kwemodeli.

Iziphumo zifunyanwa ngexesha lokwenyani kwaye zifakwa kumxholo womncedisi ngaphambi kokuba avelise impendulo. Umncedisi uyabonisa xa esebenzise iziphumo zokukhangela ukuphendula umbuzo.

## Ukuvula Ukukhangela kwi-Intanethi {#enabling-internet-search}

Ukukhangela kwi-intanethi kufuna isitshixo se-API esivela kumboneleli wokukhangela omkhethileyo. Ukukulungiselela:

1. Yiya ku-**Gratis AI Agent → Settings → Advanced**.
2. Fumana uluhlu lokwehla lwe-**Internet Search Provider** uze ukhethe nokuba yi-**Brave Search** okanye yi-**Tavily**.
3. Faka isitshixo sakho se-API kwindawo ehambelanayo. Amakhonkco okubhalisa aboniswa ecaleni kwendawo nganye.
4. Cofa **Save Settings**.

Nje ukuba isitshixo sigcinwe, isakhono se-Internet Search siyafumaneka ngokuzenzekelayo kumncedisi.

## Brave Search {#brave-search}

### Ukufumana Brave Search API Key {#obtaining-a-brave-search-api-key}

1. Ndwendwela [iphepha le-Brave Search API](https://brave.com/search/api/).
2. Bhalisela isicwangciso. Inqanaba lasimahla liyafumaneka elinomda wezicelo zenyanga.
3. Khuphela isitshixo sakho se-API kwi-dashboard yabaphuhlisi ye-Brave Search.
4. Sincamathisele kwindawo ye-**Brave Search API Key** kwiisethingi ze-Gratis AI Agent.

### Imida Yokusetyenziswa {#usage-limits}

Ukusetyenziswa kubizwa yi-Brave Search ngokusekelwe kwinani lemibuzo eyenziweyo. Impendulo nganye ye-AI eqalisa ukukhangela ibalwa njengombuzo omnye. Beka esweni ukusetyenziswa kwakho kwi-[dashboard yabaphuhlisi ye-Brave Search](https://brave.com/search/api/) ukuze uphephe iindleko ezingalindelekanga.

## Tavily {#tavily}

Superdav AI Agent v1.10.0 yongeza **Tavily** njengomnye umboneleli wokukhangela kwi-intanethi, inikezela ngeziphumo zokukhangela ezityebileyo kunye nezakhono zophando eziphambili.

### Ukufumana isitshixo se-Tavily API {#obtaining-a-tavily-api-key}

1. Ndwendwela [iphepha le-Tavily API](https://tavily.com/).
2. Bhalisela Account. Inqanaba lasimahla liyafumaneka elinomda wezicelo zenyanga.
3. Khuphela isitshixo sakho se-API kwi-dashboard ye-Tavily.
4. Sincamathisele kwindawo ye-**Tavily API Key** kwiisethingi ze-Gratis AI Agent.

### Imida Yokusetyenziswa {#usage-limits-1}

Ukusetyenziswa kubizwa yi-Tavily ngokusekelwe kwinani leefowuni ze-API ezenziweyo. Impendulo nganye ye-AI eqalisa ukukhangela ibalwa njengefowuni enye. Beka esweni ukusetyenziswa kwakho kwi-[dashboard ye-Tavily](https://tavily.com/) ukuze uphephe iindleko ezingalindelekanga.

## Ukucima Ukukhangela kwi-Intanethi {#disabling-internet-search}

Susa isitshixo se-API kwindawo yomboneleli wokukhangela osebenzayo uze ugcine. Isakhono se-Internet Search asisayi kuphinda sinikwe umncedisi.

:::note
Ukukhangela kwi-intanethi kongeza ukulibaziseka kwiimpendulo kuba umncedisi kufuneka alinde iziphumo zokukhangela ngaphambi kokuba avelise impendulo. Kwiimeko zokusetyenziswa ezixhomekeke kwixesha, cinga ukuba ukukhangela ngexesha lokwenyani kuyimfuneko na okanye ulwazi olwakhelwe ngaphakathi lomncedisi lwanele.
:::
