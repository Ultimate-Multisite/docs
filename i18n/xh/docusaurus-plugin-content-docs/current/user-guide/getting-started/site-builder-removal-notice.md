# Isaziso sokususwa kweMowudi yoMakhi weSayithi

**Imowudi yoMakhi weSayithi isusiwe kwi-Superdav AI Agent v1.12.0.** Ukuba ubusebenzisa imowudi yoMakhi weSayithi, kufuneka ufudukele kwi-**Setup Assistant agent** ukuze wenze umxholo kwaye usete isayithi.

## Kwenzeke Ntoni? {#what-happened}

### Imowudi yoMakhi weSayithi (Endala) {#site-builder-mode-legacy}

Imowudi yoMakhi weSayithi yayilujongano olusekelwe kwiwizard loku:

- Ukwenza iisayithi usebenzisa iitemplate
- Ukuqwalasela iisetingi ezisisiseko
- Ukukhetha umxholo
- Ukuseta umxholo wokuqala

### Yintoni Ethathe Indawo Yayo? {#what-replaced-it}

I-**Setup Assistant agent** ngoku iphatha yonke imisebenzi yoMakhi weSayithi nge:

- Useto olubhetyebhetye ngakumbi, olukhokelwa yi-arhente
- Iinketho ezingcono zokwenza umxholo ube ngokwezifiso
- Udityaniso nenkqubo yokuqalisa ye-Theme Builder
- Inkumbulo ye-site_brief eqhubekayo yeeseshoni zexesha elizayo

## Ukuba Ubusebenzisa Imowudi yoMakhi weSayithi {#if-you-were-using-site-builder-mode}

### Iisayithi Zakho Zikhuselekile {#your-sites-are-safe}

- Iisayithi ezikhoyo ezenziwe ngemowudi yoMakhi weSayithi ziyaqhubeka zisebenza
- Akukho kulahleka kwedatha okanye ukuphazamiseka kwesayithi
- Ungaqhubeka ulawula iisayithi zakho njengesiqhelo

### Fudukela kwi-Setup Assistant Agent {#migrate-to-setup-assistant-agent}

Kuseto lwesayithi entsha okanye utshintsho lomxholo, sebenzisa i-Setup Assistant agent:

```
"Ndincede ndisete isayithi entsha"
```

okanye

```
"Qalisa inkqubo yokuqalisa ye-Theme Builder"
```

I-Setup Assistant agent inika ukusebenza okufanayo ngobhetyebhetye obungakumbi.

## Uthelekiso: Umakhi weSayithi vs. Setup Assistant {#comparison-site-builder-vs-setup-assistant}

| Uphawu | Umakhi weSayithi (Ususiwe) | Setup Assistant (Entsha) |
|---------|----------------------|----------------------|
| Indlela yokuseta | Ifom yewizard | Incoko ne-arhente |
| Ukukhetha umxholo | Iitemplate ezichazwe kwangaphambili | Ukuveliswa ngokwezifiso |
| Ukwenza ngokwezifiso | Iinketho ezilinganiselweyo | Inkqubo yoyilo epheleleyo |
| Isishwankathelo sesayithi | Asigcinwanga | Inkumbulo eqhubekayo |
| Iiseshoni zexesha elizayo | Phinda useto | Sebenzisa i-site_brief egciniweyo |
| Ubhetyebhetye | Ukuhamba komsebenzi okumiselweyo | Incoko eguqukayo |

## Ukufudukela kwi-Setup Assistant Agent {#migrating-to-setup-assistant-agent}

### Kwiisayithi Ezintsha {#for-new-sites}

Endaweni yokusebenzisa imowudi yoMakhi weSayithi:

1. Cela: "Ndincede ndisete isayithi entsha"
2. I-Setup Assistant agent iya kukukhokela kwezi zinto:
   - Injongo neenjongo zesayithi
   - Abaphulaphuli ojolise kubo
   - Isazisi sebrand
   - Ukuveliswa komxholo
   - Uqwalaselo lokuqala

### Kwiisayithi Ezikhoyo {#for-existing-sites}

Ukuba unesayithi ekhoyo evela kwimowudi yoMakhi weSayithi:

1. Ungaqhubeka uyisebenzisa njengoko injalo
2. Ukuhlaziya umxholo, cela: "Yila kwakhona isayithi yam"
3. I-Setup Assistant agent iya kukunceda wenze umxholo omtsha
4. Idatha yesayithi yakho ihlala ingatshintshanga

### Kutshintsho Lomxholo {#for-theme-changes}

Endaweni yokukhetha umxholo kwimowudi yoMakhi weSayithi:

1. Cela: "Tshintsha umxholo wam"
2. I-Setup Assistant agent iya:
   - Kubuza ngezinto ozikhethayo kuyilo
   - Kuvelisa umxholo owenzelwe wena
   - Kuwuvula kwisayithi yakho

## Umahluko Ophambili {#key-differences}

### Imowudi yoMakhi weSayithi {#site-builder-mode}

```
1. Khetha itemplate
2. Khetha umxholo
3. Qwalasela iisetingi ezisisiseko
4. Kugqityiwe
```

### Setup Assistant Agent {#setup-assistant-agent}

```
1. Chaza injongo yesayithi yakho
2. Chaza abaphulaphuli ojolise kubo
3. Khetha izinto ozikhethayo kuyilo
4. I-arhente ivelisa umxholo owenzelwe wena
5. I-arhente ivula umxholo
6. Isishwankathelo sesayithi sigcinwa seeseshoni zexesha elizayo
```

## Izibonelelo ze-Setup Assistant Agent {#benefits-of-setup-assistant-agent}

### Ibhetyebhetye Ngakumbi {#more-flexible}

- Chaza isayithi yakho ngolwimi lwendalo
- Fumana iingcebiso ezenzelwe wena
- Ziqhelanise neemfuno zakho ezithile

### Ukwenza Ngokwezifiso Okungcono {#better-customization}

- Ukuveliswa komxholo owenzelwe wena
- Izigqibo zenkqubo yoyilo
- Iitokeni zoyilo eziqhubekayo

### Inkumbulo Eqhubekayo {#persistent-memory}

- I-site_brief yakho iyagcinwa
- Iiarhente zexesha elizayo ziyayiqonda isayithi yakho
- Akukho mfuneko yokuphinda ulwazi loseto

### Ukuhamba Komsebenzi Okudityanisiweyo {#integrated-workflow}

- Inkqubo yokuqalisa ye-Theme Builder
- Isakhono soBuhle beNkqubo yoYilo
- Izilawuli zokubonakala kwezakhono
- Zonke zisebenza kunye ngaphandle kwengxaki

## Ukusombulula Iingxaki {#troubleshooting}

### Andiyifumani imowudi yoMakhi weSayithi {#i-cant-find-site-builder-mode}

Imowudi yoMakhi weSayithi isusiwe. Sebenzisa i-Setup Assistant agent endaweni yayo:

```
"Ndincede ndisete isayithi entsha"
```

### Ndifuna ukuphinda ndenze isayithi evela kuMakhi weSayithi {#i-want-to-recreate-a-site-from-site-builder}

Ungayiphinda uyenzile nge-Setup Assistant agent:

1. Cela: "Ndincede ndisete isayithi entsha"
2. Chaza injongo noyilo lwesayithi yakho yoqobo
3. I-arhente iya kuvelisa umxholo ofanayo
4. I-site_brief yakho iya kugcinwa ukuze ibhekiselwe kwixesha elizayo

### Isayithi yam ekhoyo yoMakhi weSayithi ayisebenzi {#my-existing-site-builder-site-isnt-working}

Iisayithi ezikhoyo ezenziwe ngemowudi yoMakhi weSayithi ziyaqhubeka zisebenza. Ukuba uhlangabezana neengxaki:

1. Jonga ukuba umxholo wakho usasebenza
2. Qinisekisa ukuba ii-plugin zakho zenziwe zasebenza
3. Jonga iilogi zeempazamo ze-WordPress
4. Qhagamshelana nenkxaso ukuba iingxaki ziyaqhubeka

### Ndisenokusebenzisa iitemplate zam ezindala zoMakhi weSayithi? {#can-i-still-use-my-old-site-builder-templates}

Iitemplate zoMakhi weSayithi azisafumaneki. Nangona kunjalo:

- Iisayithi zakho ezikhoyo ziyaqhubeka zisebenza
- Ungaphinda wenze iisayithi ezifanayo nge-Setup Assistant agent
- I-Setup Assistant agent inika iinketho ezininzi zokwenza ngokwezifiso

## Amanyathelo Alandelayo {#next-steps}

1. **Kwiisayithi ezintsha**: Sebenzisa i-Setup Assistant agent
2. **Kwiisayithi ezikhoyo**: Qhubeka uzisebenzisa njengoko zinjalo
3. **Kutshintsho lomxholo**: Cela uncedo kwi-Setup Assistant agent
4. **Ukucokisa uyilo**: Sebenzisa isakhono soBuhle beNkqubo yoYilo
