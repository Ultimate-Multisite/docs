# Ushwankathelo lwe-Dashboard {#dashboard-overview}

Ultimate Multisite ine-**Dashboard** ezinikeleyo phantsi kwemenyu ye-Ultimate Multisite, kwaye yongeza neewijethi kwi-Dashboard yeNethiwekhi. Eli nqaku lichaza into oza kuyibona kwezi ndawo zombini.

## I-Dashboard yeNethiwekhi {#the-network-dashboard}

Ultimate Multisite ifaka iiwijethi eziluncedo kwi-dashboard yenqanaba lenethiwekhi efumaneka ku-**Iisayithi Zam → Umlawuli weNethiwekhi → Dashboard**.

![I-Dashboard yeNethiwekhi enee-widgets ze-Ultimate Multisite](/img/admin/network-dashboard-full.png)

Nantsi into ebonelelwa yiwijethi nganye:

**Amanyathelo Okuqala:** Ivela kufakelo olutsha kwaye ikukhumbuza ukugqiba imisebenzi yokuseta efana nokuqwalasela indlela yokuhlawula kunye nokudala Account yovavanyo.

![Iwijethi yaManyathelo Okuqala](/img/admin/network-dashboard-first-steps.png)

**Isishwankathelo:** Inika ingxelo ngenani lobhaliso kunye nengeniso yosuku. I-MRR (Ingeniso Ephindaphindekayo Yenyanga) ibonisa uqikelelo lwesixa esipheleleyo kubathengi abanobulungu obuvelisa ingeniso.

![Iwijethi yeSishwankathelo](/img/admin/network-dashboard-summary.png)

**Umsinga Womsebenzi:** Inika ingxelo ngeziganeko kulo lonke i-Ultimate Multisite — ubhaliso, ukurhoxiswa, ukudalwa kweesayithi, ukucinywa, kunye neminye imisebenzi.

![Iwijethi yoMsinga Womsebenzi](/img/admin/network-dashboard-activity-stream.png)

**Ngoku Kanye:** Ibonisa isishwankathelo esifutshane sabasebenzisi kunye neesayithi kwinethiwekhi, inezifinyezo zokudala iisayithi ezintsha okanye abasebenzisi abatsha.

![Iwijethi kaNgoku Kanye](/img/admin/network-dashboard-right-now.png)

**Iindaba & Iingxoxo:** Ikhupha ize ibonise iindaba zamva nje ze-Ultimate Multisite kuquka uhlaziyo, iimpazamo ezinzima, kunye nezilungiso zokhuseleko.

![Iwijethi yeendaba](/img/admin/network-dashboard-news.png)

## I-Dashboard ye-Ultimate Multisite {#the-ultimate-multisite-dashboard}

Ngelixa i-dashboard yenethiwekhi ibonisa ulwazi olupheleleyo malunga nenethiwekhi, i-dashboard ye-Ultimate Multisite (kwinqanaba eliphezulu lemenyu ye-Ultimate Multisite) ibonisa ulwazi malunga nenkonzo.

![I-Dashboard ye-Ultimate Multisite](/img/admin/um-dashboard-top.png)

**Ukukhula kweNgeniso Ephindaphindekayo Yenyanga:** Ibonisa ulwahlulo lwenyanga lwengeniso kwisithuba sekhalenda seenyanga ezili-12. Ingeniso entsha iboniswa ngombala oluhlaza okwesibhakabhaka ngelixa ukurhoxiswa kubonakala ngombala obomvu.

![Itshathi yokukhula kwe-MRR](/img/admin/um-dashboard-mrr.png)

**Inkxaso Yoluhlu Lwemihla:** Hluza idatha ye-dashboard ngokwexesha ukuze ugxile kulwazi oludingayo. Uluhlu lwemihla lulawula okuboniswa kwiiwijethi ezingezantsi.

![Uluhlu lwemihla kunye neenkcukacha-manani ze-dashboard](/img/admin/um-dashboard-date-range.png)

Iiwijethi ze-dashboard ziquka:

- **Ingeniso** — Ingeniso iyonke yexesha kunye ne-MRR kunye nolwahlulo ngemveliso nganye
- **Ubulungu Obutsha** — Bangaphi ubulungu obutsha obudalwe kwixesha elikhethiweyo
- **Ubhaliso ngokwamazwe** — Ulwahlulo lobhaliso ngokwejografi
- **Ubhaliso ngeeFomu** — Bangaphi abathengi ababhalise ngefomu nganye ye-checkout xa kuthelekiswa nabo bongezwe ngabaphathi
- **Iisayithi Ezindwendwelwe Kakhulu** — Iisayithi ezindwendwelwe kakhulu kwinethiwekhi yakho (kufuna ukuvumela ukubalwa kotyelelo ku-**Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**)

![Iinkcukacha-manani ze-Dashboard](/img/admin/um-dashboard-stats.png)
