# Ukulawula Ubulungu

Kwi-Ultimate Multisite ungalawula ubulungu kwaye ufikelele kuyo yonke imirhumo kwinethiwekhi yakho. Hlola uze uhlele iinkcukacha zomrhumo ezifana nezicwangciso neemveliso, inani leewebhusayithi, isixa sentlawulo nokuphindaphinda kwayo, imbali yeentlawulo, kunye nezitampu zexesha.

Ukufikelela kweli phepha, yiya kwithebhu ethi **Ubulungu** kwimenyu ye-Ultimate Multisite.

![Iphepha loluhlu lobulungu](/img/admin/memberships-list.png)

Kwiphepha loBulungu ungabona uluhlu lwabathengi bakho kunye neemveliso ezinxulumene neeakhawunti zabo. Ungahluza ubulungu ngokwemeko: busebenzayo, bulindile, bumisiwe okwethutyana, buphelelwe lixesha, okanye burhoxisiwe.

Ubulungu basimahla buphathwa njengobulungu **bobomi bonke**. Abufumani mhla wokuphelelwa lixesha kwaye abuyi kuphelelwa lixesha ngokuzenzekelayo. Ukuba isicwangciso sasimahla kufuneka siphele emva kwexesha lovavanyo okanye ixesha elimiselweyo, sebenzisa imveliso ehlawulwayo enexesha lovavanyo, rhoxisa ubulungu ngesandla, okanye uhambise umthengi kwesinye isicwangciso xa ukufikelela kwakhe kufuneka kutshintshe.

Cofa kubulungu ukuze ufikelele kwiphepha apho unokuhlela iinkcukacha zobulungu — ulwazi lokuhlawulisa, imida yeewebhusayithi okanye yotyelelo, nokunye.

![Ujongano lokuhlela ubulungu](/img/admin/membership-edit.png)

Nanku umbono opheleleyo wephepha lokuhlela ubulungu:

![Ujongano olupheleleyo lokuhlela ubulungu](/img/admin/membership-edit-full.png)

Unokongeza nobulungu obutsha obunxulumene nabasebenzisi ababhaliswe ngaphambili kwinethiwekhi yakho. Cofa **Yongeza Ubulungu** uze uqwalasele iinkcukacha.

![Imodal yokongeza Ubulungu](/img/admin/membership-add-modal.png)

Emva koko cofa **Yenza Ubulungu**. Ubulungu obutsha buza kuvela kuluhlu lobulungu.
