# Ukuqonda i-Ultimate Multisite Dashboard

Ngaphambi kokuba singene nzulu kwiinketho zoqwalaselo eziphambili ngakumbi kunye nemiba yokusebenza ye-Ultimate Multisite, masijonge kwi-dashboard.

Njengee-plugin ezininzi ze-WordPress, Ultimate Multisite yenza ungeniso lwemenyu kwinqanaba lenethiwekhi olunesihloko esithi **Ultimate Multisite** kunye nenani lamangeniso emenyu encinane ngaphantsi.

## Ii-Widgets ze-Network Dashboard

Ultimate Multisite ifaka ii-widget ezimbalwa eziluncedo kwi-dashboard yenqanaba lenethiwekhi. Le dashboard ifumaneka ngokuhamba uye ku-**My Sites → Network Admin → Dashboard**. Ii-widget ziyilelwe ukunika umlawuli wenethiwekhi ukufikelela lula kulwazi nakwimisebenzi eqhelekileyo.

![Network Dashboard enee-widget ze-Ultimate Multisite](/img/admin/network-dashboard-full.png)

### Amanyathelo Okuqala

![I-widget yaManyathelo Okuqala](/img/admin/network-dashboard-first-steps.png)

Le widget ibonakala kufakelo olutsha lwe-Ultimate Multisite kwaye ikhumbuza umlawuli wenethiwekhi ukuba agqibe imisebenzi eyongezelelweyo efana nokuqwalasela indlela yokuhlawula nokudala i-akhawunti yovavanyo.

### Isishwankathelo

![I-widget yeSishwankathelo](/img/admin/network-dashboard-summary.png)

I-widget yeSishwankathelo inika ingxelo ngenani lobhaliso kunye nengeniso yalo yosuku. I-MRR (Monthly Recurring Revenue) ibonisa isixa esiqikelelweyo kubathengi abanamalungu avelisa ingeniso.

### Umsinga Womsebenzi

![I-widget yoMsinga Womsebenzi](/img/admin/network-dashboard-activity-stream.png)

Le widget inika ingxelo ngeziganeko kwi-Ultimate Multisite. Iziganeko ziquka ubhaliso, ukurhoxiswa, ukudalwa kweesayithi, ukucinywa, kunye neminye imisebenzi.

Nceda ubhekiselele kwicandelo leZiganeko kuxwebhu ukuze ufumane inkcazelo eneenkcukacha ngakumbi.

### Ngoku Kanye

![I-widget ethi Ngoku Kanye](/img/admin/network-dashboard-right-now.png)

Le widget yesixhobo ibonisa isishwankathelo esifutshane sabasebenzisi kunye neesayithi kwinethiwekhi. Iindlela ezimfutshane eziluncedo phezulu zivumela ukufikelela ngonqakrazo olunye ukudala iisayithi ezintsha okanye abasebenzisi.

### Iindaba neengxoxo

![I-widget yeeNdaba neengxoxo](/img/admin/network-dashboard-news.png)

Le widget ilanda kwaye ibonise iindaba zamva nje ze-Ultimate Multisite. Hlala uyijongile le widget ukuze ufumane ulwazi ngohlaziyo, iimpazamo ezibalulekileyo, kunye nezilungiso zokhuseleko.

## Ultimate Multisite Dashboard

Apho i-dashboard yenqanaba lenethiwekhi ibonisa ulwazi olupheleleyo malunga nenethiwekhi, i-Ultimate Multisite dashboard (ekwizinga eliphezulu lemenyu ye-Ultimate Multisite) ibonisa ulwazi malunga nenkonzo.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

### Ukukhula kweMonthly Recurring Revenue

Itshathi ye-MRR (Monthly Recurring Revenue) ibonisa ukwahlulwa kwengeniso ngenyanga kwisithuba sekhalenda seenyanga ezili-12.

![Itshathi yokukhula kwe-MRR](/img/admin/um-dashboard-mrr.png)

Ingeniso entsha ilandelwa ngombala oluhlaza okwesibhakabhaka ngelixa ukurhoxiswa kuboniswa ngombala obomvu.

### Inkxaso yoluhlu lwemihla

Kuxhomekeke kwinani labathengi kunye nomsebenzi, izibalo zinokuba ninzi kakhulu. Inkxaso yoluhlu lwemihla eyakhelwe kwi-dashboard ivumela umlawuli wenethiwekhi ukuba agxile kuphela kulwazi kunye nexesha elifunekayo.

![Isikhethi soluhlu lwemihla kunye nezibalo](/img/admin/um-dashboard-date-range.png)

Uluhlu lwemihla lulawula ixesha lolwazi oluboniswa zii-widget ezithe ngqo ngezantsi, kuquka ingeniso, amalungu amatsha, ubhaliso ngokwelizwe, ubhaliso ngokwefom, kunye neesayithi ezityelelwe kakhulu.

![Ii-widget zezibalo ze-Dashboard](/img/admin/um-dashboard-stats.png)

:::tip
Ukuze i-widget ethi **Most Visited Sites** isebenze, yenza ukubalwa kotyelelo kusebenze ku-**Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**.
:::
