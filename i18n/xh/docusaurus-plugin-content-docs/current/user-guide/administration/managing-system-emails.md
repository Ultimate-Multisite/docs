# Ukulawula Ii-imeyili Zenkqubo

Ultimate Multisite ithumela ii-imeyili zenkqubo ezahlukeneyo kwiziganeko ezifana nokuqinisekiswa kokubhalisa, iirisithi zentlawulo, kunye notshintsho lobulungu. Ungazilungiselela ezi templates zee-imeyili ukuze zihambelane nebrand yakho.

## Ukujonga Ii-imeyili Zenkqubo

Yiya ku-**Ultimate Multisite → Settings** uze ucofe ithebhu ethi **Emails** ukuze ubone iisethingi ezinxulumene nee-imeyili.

![Settings - Emails tab](/img/admin/settings-emails.png)

Ukusuka apho, ungafikelela kwiphepha le-**System Emails** ukuze ubone zonke ii-templates zee-imeyili.

![Uluhlu lwe-System Emails](/img/admin/system-emails-list.png)

Ungahlela kwakhona ii-templates zee-imeyili nganye usebenzisa umhleli we-imeyile yenkqubo:

![Umhleli we-imeyile yenkqubo](/img/admin/system-email-editor.png)

## Ukungenisa Ii-templates Ezintsha Zee-imeyile

Iinguqulelo ezintsha ze-Ultimate Multisite kunye nezongezo zinokubhalisa ii-templates ezintsha zee-imeyile. Ukuthintela ukungqubana, ii-templates ezintsha azongezwa ngokuzenzekelayo — kufuneka uzingenise ngesandla.

Ukuze ungenise ii-templates ezintsha zee-imeyile:

1. Yiya kwiphepha le-**System Emails**
2. Cofa iqhosha elithi **Reset & Import** phezulu ephepheni
3. Tshintsha ukhetho lwe-**Import Emails** ukuze ubone ii-templates ezikhoyo
4. Khetha ii-templates ofuna ukuzingenisa uze uqinisekise

## Ukuseta Ngokutsha Ii-templates Zee-imeyile

Ukuba uyilungiselele i-template ye-imeyile kwaye ufuna ukubuyela kumxholo osisiseko, uneendlela ezimbini:

1. **Cima uze ungenise kwakhona** — Cima i-imeyile yenkqubo uze uyingenise kwakhona (oku kucima iimetriki zokuthumela)
2. **Seta ngokutsha ngesixhobo** — Sebenzisa isixhobo se-**Reset & Import**, tshintsha ukhetho lwe-**Reset**, uze ukhethe ii-imeyile ofuna ukuziseta ngokutsha

Ukhetho lwe-Reset luyacetyiswa kuba lugcina iimetriki zokuthumela kunye nezinye iinkcukacha zemetadata ngelixa lubuyisela umxholo osisiseko.
