# Ukulawula Abathengi

Iphepha le-**Customers** libonisa uluhlu lwamalungu abhaliswe kwinethiwekhi yakho. Eli phepha libonisa ulwazi olusisiseko lomsebenzisi, kuquka igama, idilesi ye-imeyile, ukungena kokugqibela, kunye nekhonkco elithi **switch to** elikuvumela ukuba ungene kwisiza esingaphantsi somsebenzisi njengomlawuli wesiza.

![Iphepha loluhlu lwabathengi](/img/admin/customers-list.png)

Cofa **Manage** ngezantsi kwekhadi lomthengi ukuze ubone ulwazi oluneenkcukacha ngalo mthengi.

Kwiphepha leenkcukacha zomthengi, ibhloko ethi **At a Glance** ibonisa i-ID yomthengi, ixesha lokungena kokugqibela, kunye nesixa esipheleleyo esichithwe ngulo mthengi.

Ibhhloko ethi **Customer Options** ikuvumela ukuba usete lo mthengi njenge-**VIP** (ithegi yokwenza kube lula ukumfumana). Ithebhu ethi **Billing Info** ibonisa ulwazi lokubhatala olufana ne-imeyile, idilesi, kunye nelizwe.

Ibhhloko ethi **Fields** ibonisa igama lomsebenzisi lomthengi, i-imeyile, kwaye ikuvumela ukuba utshintshe ubume be-**Email Verification**.

Ngezantsi kwebhloko ye-Customer Options uza kufumana iibhloko ezine ezongezelelweyo:

- **Memberships** — Imirhumo yangoku umthengi abhalise kuyo
- **Payments** — Imbali yeentlawulo zomthengi
- **Sites** — Iziza ezizezalo mthengi
- **Events** — Iziganeko ezinxulumene nalo mthengi (ukuqinisekiswa kwe-imeyile, utshintsho lokubhatala, njl.)

Cofa i-ayikhoni yotolo (→) ecaleni kwento nayiphi na ukuze uye kwiphepha leenkcukacha zayo. Umzekelo, ukucofa utolo ecaleni kwesiza kukusa kwiphepha lokuhlela isiza apho unokutshintsha unxulumano lobulungu, uhlobo lwesiza, kunye nobume bokusebenza.

Kufana nangezobulungu kunye neentlawulo — nganye ikhokelela kumaphepha eenkcukacha apho unokuhlela ubume bobulungu, umthengi onxulumene noko, iimveliso, useto lwesango lentlawulo, kwanokuqhuba iimbuyiselo.

Kwicala lasekunene lephepha lomthengi, uza kufumana:

- **Last Login & IPs** — Ixesha lokungena kokugqibela, idilesi ye-IP, kunye nelizwe
- **Timestamps** — Ixesha i-akhawunti yomthengi eyadalwa ngalo kunye nokuguqulwa kokugqibela
- **Delete** — Ukhetho lokucima umthengi ngokusisigxina

:::warning
Ukucima umthengi kusisigxina kwaye akunakubuyiselwa umva.
:::
