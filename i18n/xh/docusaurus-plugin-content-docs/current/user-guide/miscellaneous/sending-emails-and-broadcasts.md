# Ukuthumela ii-Emails kunye nee-Broadcasts (v2)

_**INQAKU ELIBALULEKILEYO: Eli nqaku libhekisa ku-Ultimate Multisite version 2.x.**_

Ultimate Multisite iza nophawu oluza kukuvumela ukuba unxibelelane nabathengi bakho ngokuthumela i-email kumsebenzisi ojoliswe kuye okanye kwiqela labasebenzisi, kwakunye nokuthumela izaziso kwi-admin dashboard yabo ukuze usasaze izibhengezo

## Yongeza izaziso ze-admin kwi-dashboard yabathengi bakho ngee-Broadcasts

Usebenzisa uphawu lwe-broadcast lwe-Ultimate Multisite, ungongeza **izaziso ze-admin** kwi-subsite admin dashboard yomsebenzisi wakho.

Oku kuluncedo kakhulu ukuba kufuneka wenze isibhengezo esifana nokugcinwa kwenkqubo okanye ukunikezela ngeemveliso okanye iinkonzo ezintsha kubasebenzisi bakho abakhoyo. Nantsi indlela isaziso se-admin esiza kubonakala ngayo kwi-dashboard yomsebenzisi wakho.

<!-- Screenshot unavailable: Admin notice broadcast shown on a customer's subsite dashboard -->

Ukuqala isaziso se-admin, yiya kwi-network admin dashboard yakho kwaye phantsi kwemenyu ye-**Ultimate Multisite**, uya kufumana ukhetho lwe-**Broadcasts**.

![Iphepha loluhlu lwee-Broadcasts kwi-admin ye-Ultimate Multisite](/img/admin/broadcasts-list.png)

Unokuhlela nee-broadcasts ezikhoyo:

![Ujongano lokuhlela i-broadcast](/img/admin/broadcast-edit.png)

Kweli phepha, cofa iqhosha elithi **Yongeza i-Broadcast** phezulu.

Oku kuza kuvula ifestile ye-modal yokongeza i-broadcast apho ungakhetha khona uhlobo lwe-broadcast ofuna ukuyithumela.

Qhubeka ukhethe **Umyalezo** emva koko ucofe iqhosha elithi **Inyathelo Elilandelayo**.

![I-modal yokongeza i-broadcast enokhetho loMyalezo olukhethiweyo](/img/admin/broadcast-add-message.png)

Ifestile elandelayo iza kukucela nokuba yi-**Umthengi ojoliswe kuye** okanye **Imveliso ejoliswe kuyo**. Qaphela ukuba ungakhetha abasebenzisi abangaphezu komnye okanye imveliso engaphezu kwenye.

Ukukhangela nokuba yi-akhawunti yomsebenzisi okanye imveliso kufuneka uqale ngokuchwetheza igama eliphambili ngaphakathi kwendawo.

Phantsi kwendawo ethi **Uhlobo lomyalezo**, ungakhetha umbala wesaziso. Oku kuza kugxininisa ukungxamiseka komyalezo wakho.

Emva koko ungacofa **Inyathelo Elilandelayo**.

![Iindawo zabathengi abajoliswe kubo, imveliso ejoliswe kuyo kunye nohlobo lomyalezo ze-broadcast yoMyalezo](/img/admin/broadcast-message-targets.png)

Ifestile elandelayo yindawo apho ungaqala khona ukubhala umyalezo wakho ngokufaka isihloko kunye nomxholo/umyalezo ofuna ukuwusasaza kubasebenzisi.

![Isihloko somyalezo we-broadcast kunye nomhleli womxholo kwinyathelo lokubhala](/img/admin/broadcast-edit.png)

Emva kokudala umyalezo wakho, ungacofa iqhosha elithi **Thumela**.

Yiloo nto ke. Isaziso se-admin kufuneka sibonakale ngoko nangoko kwi-dashboard yomsebenzisi wakho.

## Thumela ii-emails kubathengi bakho

Usebenzisa uphawu lwe-broadcast lwe-Ultimate Multisite, ungathumela i-email kubasebenzisi bakho. Unokhetho lokuthumela i-email kuphela kubasebenzisi abathile okanye ukujolisa kwiqela elithile labasebenzisi ngokusekelwe kwimveliso okanye isicwangciso ababhalise phantsi kwaso.

Ukuqala i-email broadcast, yiya kwi-network admin dashboard yakho kwaye phantsi kwemenyu ye-Ultimate Multisite, uya kufumana ukhetho lwe-Broadcast.

![Iphepha loluhlu lwee-Broadcasts elisetyenziswa njengendawo yokuqala ye-email broadcast](/img/admin/broadcasts-list.png)

Kweli phepha, cofa iqhosha elithi **Yongeza i-broadcast** phezulu.

Oku kuza kuvula ifestile ye-modal yokongeza i-broadcast apho ungakhetha khona uhlobo lwe-broadcast ofuna ukuyithumela. Qhubeka ukhethe **Email** emva koko ucofe iqhosha elithi **Inyathelo Elilandelayo**.

![I-modal yokongeza i-broadcast enokhetho lwe-Email olukhethiweyo](/img/admin/broadcast-add-email.png)

Ifestile elandelayo iza kukucela nokuba yi-**Umthengi ojoliswe kuye** okanye **Imveliso ejoliswe kuy** o. Qaphela ukuba ungakhetha abasebenzisi abangaphezu komnye okanye imveliso engaphezu kwenye.

Ukukhangela nokuba yi-akhawunti yomsebenzisi okanye imveliso kufuneka uqale ngokuchwetheza igama eliphambili ngaphakathi kwendawo.

Nje ukuba abaphulaphuli ojolise kubo bakhethwe, ungacofa **Inyathelo Elilandelayo**.

![Ukukhethwa kwabathengi abajoliswe kubo kunye nemveliso ejoliswe kuyo ye-Email broadcast](/img/admin/broadcast-email-targets.png)

Ifestile elandelayo yindawo apho ungaqala khona ukubhala i-email yakho ngokufaka isihloko kunye nomxholo/umyalezo ofuna ukuwuthumela kubasebenzisi.

<!-- Screenshot unavailable: Email broadcast subject and content editor on the compose step -->

Emva kokudala umyalezo wakho, ungacofa iqhosha elithi **Thumela**.

Yiloo ndlela kulula ngayo ukuthumela i-email kubasebenzisi bakho bokugqibela usebenzisa uphawu lwe-broadcast.

## Ii-emails zenkqubo

Ii-emails zenkqubo kwi-Ultimate Multisite zezo **zaziso ezizenzekelayo** ezithunyelwa yinkqubo emva kwezenzo ezithile ezifana nobhaliso, intlawulo, ukuma kwedomain, njalo njalo. Ezi emails zinokuhlelwa okanye zitshintshwe kwiisethingi ze-Ultimate Multisite. Ikwaza nophawu oluza kukuvumela ukuba usete ngokutsha kwaye ungenise iisethingi ezikhoyo ukusuka kolunye ufakelo lwe-Ultimate Multisite.

### Ukuseta ngokutsha & Ukungenisa

Iinguqulelo ezintsha ze-Ultimate Multisite, kwakunye nee-add-ons, zinako kwaye ziya kubhalisa ii-emails ezintsha ngamaxesha athile.

Ukuthintela ungquzulwano kunye neminye imiba, **asiyi kongeza ii-template ezintsha ze-email njengee-System Emails kufakelo lwakho ngokuzenzekelayo** , ngaphandle kokuba zibalulekile ekusebenzeni ngokuchanekileyo kophawu oluthile.

Nangona kunjalo, ii-super admins kunye nee-agents zinokungenisa ezi emails zisandula ukubhaliswa ngesixhobo sokungenisa. Loo nkqubo iza kudala i-email entsha yenkqubo enomxholo kunye noqwalaselo lwe-template entsha ye-email, ivumele i-super admin ukuba yenze naluphi na utshintsho elufunayo okanye izigcine zinjalo.

#### Indlela yokungenisa ii-emails zenkqubo

Yiya kwiphepha lakho lee-Settings ze-Ultimate Multisite uze uye kwi-tab ethi **Emails**.

![I-tab ye-Emails kwiisethingi ze-Ultimate Multisite ebonisa icandelo lee-System Emails](/img/config/settings-emails-tab.png)

Emva koko, kwi-sidebar, cofa iqhosha elithi **Lungiselela ii-System Emails**.

<!-- Screenshot unavailable: Customize System Emails button on the System Emails sidebar panel -->

Kwiphepha le-System Emails, uza kubona iqhosha lesenzo elithi **Seta ngokutsha & Ngenisa** phezulu. Ukucofa elo qhosha kufuneka kuvule ifestile ye-modal yokungenisa nokuseta ngokutsha.

![Iqhosha lesenzo lokuseta ngokutsha okanye lokungenisa kwiphepha le-admin lee-System Emails](/img/admin/system-emails-reset-import.png)

Emva koko, ungatshintsha uvule okanye uvale iinketho ze-Import Emails ukuze ubone ukuba zeziphi ii-emails zenkqubo ezikhoyo ezinokungeniswa.

#### Ukuseta kwakhona ii-imeyile zenkqubo

Ngamanye amaxesha, uya kuqaphela ukuba utshintsho olwenzileyo kwitemplate ethile ye-imeyile alusakusebenzeli, kwaye ungathanda ukuyiseta kwakhona ibuyele **kwimeko yayo engagqibekanga**.

Kwiimeko ezinjalo, uneenketho ezimbini: unokucima nje imeyile yenkqubo uze uyingenise kwakhona (usebenzisa imiyalelo engentla) - nto leyo eza kucima iimetrikhi zokuthumela nezinye izinto, nto leyo eyenza le ndlela ingakhethwa kakhulu.

Okanye ungasebenzisa **isixhobo sokuSeta kwakhona & Ngenisa** ukuseta kwakhona loo template ye-imeyile.

Ukuseta kwakhona itemplate ye-imeyile, unokulandela amanyathelo angentla de ufike kwisixhobo sokuSeta kwakhona & Ngenisa, emva koko, uvule inketho ethi **Seta kwakhona** uze ukhethe ii-imeyile ofuna ukuziseta kwakhona zibuyela kumxholo wazo ongagqibekanga.
