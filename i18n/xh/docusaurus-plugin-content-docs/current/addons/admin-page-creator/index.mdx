# Umyili Wamaphepha Olawulo {#admin-page-creator}

## Umxholo {#content}
Umyili wamaphepha olawulo uza kukuvumela ukuba wongeze ngokulula amaphepha amatsha olawulo namaphepha angaphantsi ngaphakathi kwe-WordPress admin dashboard. Inokusetyenziswa ukudala iphepha lokwamkela okanye iphepha lesifundo elinokusebenza njengephepha lokufika kubasebenzisi abatsha. Ungasebenzisa umakhi wephepha onjenge-Beaver Builder, Elementor, Brizy, kunye ne-Oxygen ukuyila amaphepha ngaphandle kobunzima.

Ikwabonelela ngenkxaso ye-HTML, CSS ne-JavaScript yesiko, kwakunye nolawulo lokufikelela olusekelwe kwi-plan nakwindima yomsebenzisi!

## Iimpawu {#features}
* Inkxaso ye-HTML ne-CSS yesiko
* Inkxaso ye-JavaScript
* Imo yoMxholo we-PHP
* Imvume esekelwe kwindima yoMsebenzisi
* Inkxaso kubakhi bamaphepha

## Ufakelo {#installation}

1. Layisha **ultimate-multisite-admin-page-creator** kulawulo lwe-**/wp-content/plugins/**
2. Vula i-plugin ngemenyu ye-**Plugins** kwi-WordPress Network Admin yakho

## Isikhokelo sokuSebenzisa {#usage-guide}

Nangona i-WordPress isenza kube lula ngokwenene ukongeza umxholo kwicala langaphambili le-site, ukudala amaphepha namaphepha angaphantsi kwicala lolawulo kufuna izakhono ezisisiseko zokubhala ikhowudi. Ultimate Multisite: Umyili Wamaphepha Olawulo ujolise ekuziseni olo lula lokusebenzisa olufanayo lwecala langaphambili kwicala elingasemva lee-site zenethiwekhi yakho.

## Iinketho zeMenyu {#menu-options}

Kwicala, kwi-meta-box yeeNketho eziQhelekileyo, uza kukwazi ukuqwalasela uhlobo lwemenyu, ileyibhile, nokunye okuninzi.

### Iintlobo zeMenyu {#menu-types}

Iphepha lakho elitsha lolawulo lesiko linokuba neentlobo ezimbini ezahlukeneyo zemenyu: imenyu yenqanaba eliphezulu kunye nemenyu engaphantsi.

Iphepha lemenyu yenqanaba eliphezulu, njengoko igama lisitsho, liza kongeza iphepha lakho elitsha elidalwe kwimenyu yolawulo ecaleni kwamaphepha e-WordPress angagqibekanga.

Ukuba ukhetha inketho yenqanaba eliphezulu, uza kukwazi ukukhetha ulandelelwano lwemenyu kunye ne-icon yemenyu nayo.

Ukuba ufuna ukwenza eli phepha libe liphepha lemenyu engaphantsi, uza kukwazi ukukhetha iphepha elingumzali oza kuncamathisela eli phepha kulo. Ungongeza iphepha lakho lesiko kumaphepha e-WordPress angagqibekanga okanye kumaphepha akho olawulo esiko akwinqanaba eliphezulu.

## Iinketho eziPhambili {#advanced-options}

Kufuneka ukwazi ukongeza zonke iintlobo zomxholo kumaphepha akho olawulo esiko. Oko kuquka imithetho neefayile ze-CSS, i-JavaScript, kunye neelayibrari zangaphandle nazo. Ungakwenza oko usebenzisa i-Tab yeeNketho eziPhambili.

### I-CSS yesiko {#custom-css}

Kwi-tab ye-CSS, ungongeza ikhowudi yakho ye-CSS yesiko, kwakunye nokungenisa iifayile ze-CSS ezivela kwii-site ezikude (njengee-CDN, Google Fonts, njl.).

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_I-CSS yesiko nayo iyaxhaswa. Ungabandakanya neefayile zesimbo zangaphandle!_

### I-JavaScript yesiko {#custom-javascript}

Okufanayo kuyasebenza nakwi-JavaScript. Ungongeza ielayibrari zangaphandle oza kuzisebenzisa kwikhowudi yakho ye-JavaScript yesiko.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Yongeza ikhowudi yakho ye-JavaScript!_

## Iimvume {#permissions}

Kwakhona kwiiNketho eziPhambili, uza kukwazi ukukhetha ukuba zeziphi ii-sub-site eziza kuba nokufikelela kweli phepha. Oku kuluncedo ukuba uceba ukunika umxholo okhethekileyo owahlukileyo kumanqanaba ahlukeneyo e-plan, umzekelo.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Unolawulo olupheleleyo lokuba eli phepha liza kuvela phi_

**Kubalulekile** : Zombini iimeko kufuneka zifezekiswe ukuze iphepha liboniswe. Oko kuthetha ukuba ukuba umsebenzisi ngumthengi we-plan ethile, kodwa akanayo enye yeendima ezikhethiweyo (okanye ngokuphambeneyo) iphepha aliyi kuvela kwiphaneli yakhe.

## Iyasebenza {#active}

Ukuba ufuna ukukhubaza iphepha lolawulo lesiko ngaphandle kokulicima okanye ukuphazamisa iimvume, susa nje uphawu kwinketho ethi Active.

Ukuyekisa ukusebenza kwephepha lenqanaba eliphezulu kuza kuyekisa ngokuzenzekelayo ukusebenza kwawo onke amaphepha angaphantsi ancanyathiselwe kwelo phepha lingumzali.
