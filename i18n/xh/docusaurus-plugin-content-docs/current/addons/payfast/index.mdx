# PayFast

Multisite Ultimate: Isango le-Payfast

Yongeza i-Payfast - isango lentlawulo elikhawulezayo nelikhuselekileyo kwi-intanethi eMzantsi Afrika - njengokhetho lwentlawulo lwe-Multisite Ultimate.

## Ufakelo {#installation}

1. Layisha 'ultimate-multisite-payfast' kulawulo lwe-'/wp-content/plugins/'
2. Vula i-plugin ngemenyu ethi 'Plugins' kwi-WordPress
3. Landela i-Wizard inyathelo ngenyathelo ukuze usete i-plugin

## Isikhokelo sokusebenzisa {#usage-guide}

I-PayFast ngumqhubekekisi ophambili wentlawulo kwi-intanethi kumashishini aseMzantsi Afrika. Izinikele ekuncedeni abarhwebi be-intanethi bakhule ngokwenza iintengiselwano zibe lula kwaye zikhuseleke. Ngokwesicelo somsebenzisi we-Ultimate Multisite, sigqibe ekubeni senze isongezo esiza kwenza ukudityaniswa ne-PayFast kwenzeke. Eli nqaku liza kubonisa indlela yokuseta i-PayFast njengesango lakho lentlawulo ukuze ufumane intlawulo kuthungelwano lwakho lwe-Ultimate Multisite.

Inyathelo lokuqala kukufaka isongezo.

# Indlela yokufaka isongezo se-Payfast:

Yiya kwiphepha lezongezo uze ukhethe isongezo se-Ultimate Multisite: Payfast Gateway.

![Iphepha lezongezo](/img/admin/addons-page.png)

Ifestile yesongezo iza kuvela. Cofa ku-Install Now.

![Iphepha lezongezo elinokhetho lokufaka](/img/admin/addons-page.png)

Emva kokuba ufakelo lugqityiwe, uza kuthunyelwa kwiphepha lee-plugin. Apha, cofa nje ku-Network Activate kwaye isongezo se-_Payfast_ siya kuvulwa kuthungelwano lwakho.

# Ukuseta i-PayFast

Emva kokufaka isongezo, ngoku ungaya kwiisetingi ze-Ultimate Multisite uze uqale ukuqwalasela i-PayFast.

Phantsi kwephepha le-**Ultimate Multisite settings**, ungafumana ithebhu ethi "Payments". Phantsi kwale thebhu, kufuneka ubone "**Payfast** " kunye neqhosha lokutshintsha ukuze uvule okanye uvale eli sango lentlawulo.

![Iisetingi zesango lentlawulo](/img/config/settings-payment-gateways.png)Qhubeka ucofe iqhosha lokutshintsha ukuze ulivule. Emva koko liza kwandisa ibhokisi ukubonisa iindawo ezahlukeneyo ekufuneka uzigcwalise ezifana ne-**Mechant ID, Merchant Key and PassPhrase.** Ungalufumana olu lwazi kwi-Account yakho ye-**Payfast**

<!-- Umfanekiso-skrini awufumaneki: Iisetingi ezandisiweyo ze-PayFast ezibonisa iindawo ze-Merchant ID, Merchant Key, kunye ne-PassPhrase -->

Inyathelo elilandelayo kukungena kwi-Account yakho ye-PayFast ukuze ufumane ulwazi oludingayo.

Xa ungenile, ungabona ngoko nangoko iinkcukacha zokungena ozidingayo kwi-dashboard ye-Account yakho ye-**Payfast**.

<!-- Umfanekiso-skrini awufumaneki: I-dashboard ye-Account ye-PayFast ebonisa iinkcukacha ze-Merchant ID kunye ne-Merchant Key -->Ungazikopa ngokulula uze uzincamathisele iinkcukacha zokungena, kwaye ugcwalise iinkcukacha kwiisetingi zakho ze-Ultimate Multisite. Emva koko cofa iqhosha elithi **"Save settings** " ukuze ugcine utshintsho.

<!-- Umfanekiso-skrini awufumaneki: Iisetingi ze-Ultimate Multisite PayFast ezineenkcukacha zokungena ezigcwalisiweyo kunye neqhosha lokugcina -->

Kuphelele apho, ukudityaniswa kwentlawulo yakho ye-PayFast kugqityiwe!
