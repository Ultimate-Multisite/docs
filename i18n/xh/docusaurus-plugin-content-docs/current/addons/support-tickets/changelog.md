# Ilogi yotshintsho yamatikiti enkxaso

### 1.0.4 - 2026-05-05
* Kuphuculwe: Kususiwe uvimba weefayile vendor/ ekulandelweni yi-Git (sele ugutyungelwe yi-.gitignore), kuncitshiswa ubungakanani bendawo yokugcina
* Kuphuculwe: Kuvavanyiwe ukuya kutsho kwi-WordPress 7.0

### 1.0.3 - 2026-05-01
* Ulungiso: Yamkela amaxabiso e-null kwiiseta zemodeli yetikiti ezivumela null
* Ulungiso: Gcina onke amatikiti kwiitafile zehlabathi kuyo yonke inethiwekhi ukuze kubekho inkxaso efanelekileyo ye-multisite
* Ulungiso: Fihla iifildi zabasebenzi kwaye ulungise ukuhanjiswa kwakhona kwephepha elingenanto kwifomu yomthengi yetikiti elitsha
* Ulungiso: Buyisela iminxeba ye-add_meta() engachazwanga nge-update_meta() echanekileyo yokugcina metadata
* Ulungiso: Buyisela uhlolo lwesakhono esingabhaliswanga nge-wu_view_all_support_tickets echanekileyo
* Ulungiso: Yongeza abaphathi be-AJAX abalahlekileyo besimo setikiti, ukwabela, kunye nezenzo zokuhlela ngokukhawuleza
* Ulungiso: Lungisa igama lendlela yokubona impendulo yabasebenzi kwiimbono zetikiti
* Ulungiso: Dibanisa abaphathi beempendulo abaphindwe kabini kwaye ulungelelanise amagama ezenzo ze-nonce
* Ulungiso: Yongeza imbono ye-frontend elahlekileyo ye-[wu_submit_ticket] shortcode
* Ulungiso: Yongeza ikholam ye-user_id elahlekileyo kwaye ulungise indlela ye-Support_Ticket::get_user_id()
* Ulungiso: Susa uphawu locaphulo oluphindwe kabini olongezelelweyo kwithegi yokhetho lwesihluzi sokubaluleka
* Ulungiso: Yongeza ipaneli yolawulo lwamatikiti kumphathi wenethiwekhi yee-super admins
* Kuphuculwe: Dibanisa i-CSS yomphathi kwisitayile esinye sangaphandle
* Kuphuculwe: Susa imenyu engaphantsi ye-Settings kwimenyu yomphathi wesayithi engaphantsi
* Kuphuculwe: Layisha ii-asethi ze-frontend ngokwemiqathango kuphela kumaphepha amatikiti enkxaso
* Kuphuculwe: Tsiba i-autoloader yeplagi xa i-autoloader yengcambu ye-Bedrock sele ilayishe izixhomekeki

### 1.0.2 - 2025-12-11
* Kongezwe: Ukusebenza okupheleleyo kokujonga itikiti kumphathi nakwi-frontend
* Kongezwe: Umphathi we-AJAX wokungeniswa kweempendulo zetikiti
* Kongezwe: Inkxaso yokusebenza kwempendulo yetikiti ngokuphathwa kwefomu okufanelekileyo
* Kongezwe: Ukuboniswa kwesaziso esifanelekileyo sokungeniswa kwamatikiti kunye neempendulo kumphathi
* Kongezwe: Ukudityaniswa ngokuzenzekelayo kwetikiti nomsebenzisi wangoku ekuzingeniseleni kwabathengi
* Kongezwe: Uphuculo lokhuseleko lokuthintela abathengi ekugqithiseni ubunini betikiti
* Kongezwe: Imisebenzi yoncedo elahlekileyo (wu_format_date, wu_user_can_view_ticket, njl.)
* Kongezwe: Ukukhutshelwa kunye nokuphathwa okufanelekileyo kweefayile eziqhotyoshelweyo
* Kongezwe: Inkqubo yezaziso ze-imeyile yeempendulo zetikiti kunye notshintsho lwesimo
* Kulungisiwe: Ifomu yempendulo yetikiti ngoku iquka i-ID yetikiti efunekayo
* Kulungisiwe: Imiba yokubonakala kwezaziso kwiimbono zamatikiti kumphathi
* Kulungisiwe: Iimpazamo zesintaksi kwimisebenzi yamatikiti enkxaso
* Kuphuculwe: Ulwakhiwo lwedathabheyisi ngeeklasi ezifanelekileyo zemibuzo kunye ne-schema zeempendulo kunye nezincamatheliso
* Kuphuculwe: Ukufuduselwa kwiipropati zokwenene endaweni yee-attributes zeemodeli zetikiti
