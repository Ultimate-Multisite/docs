# IiArhente zeNkxaso {#support-agents}

Ukulawula inethiwekhi yonke wedwa ayingomsebenzi olula. Njengoko inethiwekhi yakho iqala ukukhula, uya kuqalisa ukubona ukwanda kwemfuno yenkxaso. Kuya kuba licebo elihle ukuzisa abantu abaza kukunceda ukunciphisa umthwalo wakho womsebenzi. Nangona kunjalo, ukunika ukufikelela/iimvume ezipheleleyo kuyingozi kwaye kunokubeka inethiwekhi yakho emngciphekweni. Kulapho esi songezo seArhente yeNkxaso singena khona.

Esi songezo sikuvumela ukuba udale uhlobo olutsha lomsebenzisi weWordPress obizwa ngokuba yiArhente yeNkxaso, kwaye sikunika ulawulo olupheleleyo kwiimvume analo loo msebenzisi. Ukunika imvume kulula njengokuvula iqhosha lokutshintsha, kwaye ulivale xa ugqibe ekubeni uyisuse.

## Iimpawu {#features}

- Uluhlu olwahluliweyo lweemvume zeWordPress & Ultimate Multisite
- Vumela & khubaza ukufikelela ngokulula
- Dala inani elingenamda leeAccount zeArhente

## Ufakelo {#installation}

1. Layisha 'ultimate-multisite-support-agents' kwi-directory ethi '/wp-content/plugins/'
2. Vula isongezo ngemenyu ethi 'Izongezo' kwiWordPress
3. Landela iWizard inyathelo ngenyathelo ukuseta isongezo
