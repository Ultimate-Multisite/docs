# Fluent Forms {#fluent-forms}

Esi songezelelo senza kube nokwenzeka ukongeza imida kwi-Fluent Forms ku-Multisite Ultimate kwiimveliso, ubulungu okanye iisayithi.
Fluent Forms isebenzisa iitafile ezilungiselelweyo endaweni yeentlobo zezithuba ezilungiselelweyo, ngoko iinketho eziqhelekileyo zokumisela imida azisebenzi ne-Fluent Forms.

## Iimpawu {#features}

- Yongeza imida ye-Fluent Forms okanye i-Conversational Fluent Forms
- Abathengi bafumana umyalezo onobubele obabakhuthaza ukuba baphucule isicwangciso sabo xa befikelela kumda
