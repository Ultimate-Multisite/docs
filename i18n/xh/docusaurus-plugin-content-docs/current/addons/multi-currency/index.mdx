# Multi-Currency {#multi-currency}

Ultimate Multisite: Multi-Currency yenza inethiwekhi yakho yamkele iintlawulo ngeemali ezininzi. Iguqula amaxabiso ngokuzenzekelayo isebenzisa amazinga otshintshiselwano aphilayo okanye esandla, ikuvumela usete amaxabiso achanekileyo ngemveliso nganye kuyo nayiphi na imali, kwaye ibhaqe imali ekhethwa lundwendwe ngalunye nge-geolocation — konke kulawulwa kwi-Dashboard enye yolawulo.

## Iimpawu Ezibalulekileyo {#key-features}

- **Uguqulo oluzenzekelayo lwamazinga otshintshiselwano** — amaxabiso aguqulwa ukusuka kwimali yakho esisiseko kusetyenziswa amazinga afunyenwe kumnikezeli omkhethileyo
- **Ukudlula amaxabiso ngemveliso nganye** — seta amaxabiso achanekileyo ajolise kwimali ethile kuyo nayiphi na imveliso ukuze udlule uguqulo oluzenzekelayo
- **Indawo yokukhetha imali kubhaliso** — ivumela abathengi bakhethe imali abayithandayo ngexesha le-checkout/ubhaliso
- **Abanikezeli abaninzi bamazinga otshintshiselwano** — ECB, Frankfurter, Stripe, okanye amazinga esandla ngokupheleleyo
- **Ukubhaqwa kwemali okusekelwe kwi-geolocation** — kukhetha kwangaphambili ngokuzenzekelayo imali ehambelana nelizwe londwendwe
- **Dashboard yolawulo** — lawula iimali ezisebenzayo, amazinga otshintshiselwano, kunye nezicwangciso zomnikezeli kwindawo enye

## Iimfuno {#requirements}

- Ultimate Multisite 2.0.0 okanye ngaphezulu
- Ufakelo lwe-WordPress Multisite
- PHP 7.4 okanye ngaphezulu

## Ufakelo {#installation}

1. Layisha iifayile ze-plugin ku-`/wp-content/plugins/ultimate-multisite-multi-currency`
2. Vula i-plugin ngesikrini se-**Plugins** kwi-WordPress (Network Activate)
3. Yiya ku-**Ultimate Multisite → Multi-Currency** ukuze uvule i-Dashboard yolawulo
4. Yongeza iimali ofuna ukuzamkela uze uqwalasele umnikezeli wakho wamazinga otshintshiselwano

## Uqwalaselo {#configuration}

### Dashboard Yolawulo {#admin-dashboard}

I-Dashboard yolawulo ye-Multi-Currency (**Ultimate Multisite → Multi-Currency**) yindawo esembindini yoku:

- Yongeza, vula, okanye uvale iimali
- Khetha umnikezeli wamazinga otshintshiselwano kwaye uhlaziye amazinga xa efuneka
- Phonononga amazinga angoku azo zonke iimali ezisebenzayo
- Seta amazinga esandla xa abanikezeli abazenzekelayo bengafanelekanga

### Abanikezeli Bamazinga Otshintshiselwano {#exchange-rate-providers}

| Umnikezeli | Inkcazelo |
|---|---|
| **ECB** | European Central Bank — amazinga asimahla emihla ngemihla eenethiwekhi ezisekelwe kwi-EUR |
| **Frankfurter** | Open-source ECB mirror API — isimahla, akukho API key ifunekayo |
| **Stripe** | Isebenzisa amazinga aphilayo e-FX e-Stripe — ifuna udibaniso lwe-Stripe olusebenzayo |
| **Manual** | Ufaka amazinga ngokuthe ngqo; akukho minxeba ye-API yangaphandle |

Khetha umnikezeli wakho phantsi kwe-**Ultimate Multisite → Multi-Currency → Settings → Exchange Rate Provider**. Amazinga anokuhlaziywa ngesandla nangaliphi na ixesha okanye acwangciselwe ukuhlaziywa ngokuzenzekelayo.

### Ukudlula Amaxabiso Ngemveliso Nganye {#per-product-pricing-overrides}

Xa uguqulo oluzenzekelayo lungachanekanga ngokwaneleyo, unokuseta ixabiso elichanekileyo lemali ethile kuyo nayiphi na imveliso:

1. Vula umhleli wemveliso (**Ultimate Multisite → Products → Edit**)
2. Skrolela kwicandelo le-**Currency Pricing**
3. Faka ixabiso elichanekileyo lemali nganye ofuna ukuyidlula
4. Gcina imveliso

Iimveliso ezingenalo udlulo zisebenzisa ixabiso eliguqulweyo ukusuka kwimali yakho esisiseko.

### Indawo Yokukhetha Imali Kubhaliso {#currency-selector-signup-field}

Indawo yokukhetha imali inokongezwa kwinkqubo yakho yobhaliso/checkout ukuze abathengi bakhethe imali abayithandayo xa bebhalisa:

1. Yiya ku-**Ultimate Multisite → Multi-Currency → Settings**
2. Vula **Show currency selector on signup**
3. Ngokuzikhethela seta imali engagqibekanga okanye uvumele i-geolocation imisele ixabiso elikhethwe kwangaphambili

Imali ekhethiweyo igcinwa kunye nomrhumo kwaye isetyenziselwa yonke intlawulo elandelayo.

### Ukubhaqwa Kwemali Okusekelwe Kwi-Geolocation {#geolocation-based-currency-detection}

Xa ukubhaqwa kwe-geolocation kuvuliwe, i-addon idibanisa ilizwe londwendwe nemali kwaye iyikhetha kwangaphambili kwindawo yokukhetha imali. Ukuvula:

1. Yiya ku-**Ultimate Multisite → Multi-Currency → Settings**
2. Vula **Detect currency from visitor location**
3. Dibanisa amazwe neemali kwitheyibhile ye-**Geolocation Mapping**

Ukuba akukho kudibanisa kukhoyo kwilizwe londwendwe, imali esisiseko yenethiwekhi isetyenziswa njengento yokubuyela kuyo.

## Imibuzo Ebuzwa Rhoqo {#frequently-asked-questions}

### Kwenzeka ntoni ukuba umnikezeli wamazinga otshintshiselwano akafumaneki? {#what-happens-if-an-exchange-rate-provider-is-unavailable}

I-addon ibuyela kumazinga agqibele ukufunyanwa ngempumelelo. Ukuba akukho mazinga akhe afunyanwa, ibuyela kumazinga esandla owaqwalaseleyo. Kuyacetyiswa ukuqwalasela amazinga esandla njengomnatha wokhuseleko.

### Ndingasebenzisa abanikezeli abahlukileyo kwiimali ezahlukileyo? {#can-i-use-different-providers-for-different-currencies}

Hayi — umnikezeli omnye uyasebenza ngexesha elinye kuzo zonke iimali. Sebenzisa **Manual** ukuba ufuna ulawulo olupheleleyo kumazinga ngamanye.

### Ngaba i-geolocation ifuna i-plugin eyahlukileyo? {#does-geolocation-require-a-separate-plugin}

Akukho plugin yangaphandle ye-geolocation ifunekayo. I-addon isebenzisa ukubhaqwa kwe-IP-to-country okwakhelwe ngaphakathi kwi-WordPress. Ukuchaneka okuphezulu, inkonzo ye-geolocation ezinikeleyo inokudityaniswa ngee-filter hooks ezinikiweyo.

### Ngaba ukudlula ngemveliso nganye kusetyenziswa ngaphambi okanye emva kwerhafu? {#are-per-product-overrides-applied-before-or-after-tax}

Ukudlula ngemveliso nganye kuseta ixabiso langaphambi kwerhafu kwimali ekujoliswe kuyo. Irhafu ibalwa ngaphezulu kwexabiso lodlulo kusetyenziswa uqwalaselo lwakho lwerhafu olukhoyo.
