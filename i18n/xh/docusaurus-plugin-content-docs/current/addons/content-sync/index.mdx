# Ungqamaniso Lomxholo {#content-sync}

Multisite Ultimate: Ungqamaniso Lomxholo

Esi songezelelo se-Multisite Ultimate sivumela abalawuli benethiwekhi ukuba bangqamanise umxholo ukusuka kuyo nayiphi na isayithi ukuya kwezinye iisayithi kwinethiwekhi. Sixhasa zonke iintlobo zeposti kuquka iiposti, amaphepha, iintlobo zeposti ezenziwe ngokwezifiso, kunye neetemplate ze-Elementor. Esi songezelelo siquka ujongano lomncedisi onamanyathelo amaninzi lokukhetha umxholo kunye neesayithi ekujoliswe kuzo, ngokusetyenzwa ngeebhetshi kunye nengxelo yenkqubela yexesha lokwenyani.

## Iimpawu {#features}

* Ukukhethwa Komxholo: Igridi yamakhadi abonakalayo eneethumbnails kunye nokuhluza ngohlobo lweposti
* Zonke Iintlobo Zeposti: Ngqamanisa iiposti, amaphepha, iimveliso, iitemplate ze-Elementor, kunye naluphi na uhlobo lweposti olwenziwe ngokwezifiso
* Ukukhethwa Kwesayithi: Khetha ukuba zeziphi iisayithi ekufuneka ungqamanise umxholo kuzo, ngokukhutshelwa ngaphandle kwesayithi yomthombo kunye nokukhethwa kwangaphambili kwesayithi engumntwana
* Ukusetyenzwa Ngeebhetshi: Ingqamanisa ukuya kwiisayithi ezili-10 ngexesha kunye nohlaziyo lwenkqubela
* Ungqamaniso Lwemidiya: Ikopa ngokuzenzekelayo kwaye iphinde imape iifayile zemidiya (imifanekiso, iividiyo) ukusuka kumxholo weposti ukuya kwiisayithi ekujoliswe kuzo
* Ungqamaniso Lweetaxonomy: Igcina iindidi, iithegi, kunye neetaxonomy ezenziwe ngokwezifiso
* Inkxaso ye-Elementor: Inkxaso epheleleyo yeetemplate ze-Elementor ngokuphinda kuveliswe i-CSS kunye nokuphinda kumapwe ii-ID zeetemplate
* Ukusombulula Ungquzulwano: Ukhetho lokutshintsha umxholo okhoyo okanye ukuwutsiba
* Ulawulo Lwecache: Icoca ngokuzenzekelayo i-cache ye-Elementor kwiisayithi ekujoliswe kuzo emva kongqamaniso

## Iimfuno {#requirements}

* WordPress 5.3 okanye ngaphezulu
* PHP 7.4 okanye ngaphezulu
* Iplugin ye-Multisite Ultimate (iyasebenza)
* Iplugin ye-Elementor (uyazikhethela, iyafuneka kungqamaniso lweetemplate ze-Elementor)

## Ufakelo {#installation}

1. Layisha 'ultimate-multisite-content-sync' kulawulo lwe-'/wp-content/plugins/'
2. Sebenzisa iplugin ngokusebenzisa imenyu ethi 'Iiplagin' kwi-WordPress Network Admin yakho
3. Qwalasela iisetingi ku-Ultimate Multisite → Settings → Ungqamaniso Lomxholo
4. Yiya ku-Ultimate Multisite → Ungqamaniso Lomxholo ukuze uqalise ukungqamanisa
