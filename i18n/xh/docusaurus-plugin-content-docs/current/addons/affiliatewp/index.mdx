# AffiliateWP {#affiliatewp}

Ultimate Multisite: Uhlanganiso lwe-AffiliateWP

Sebenzisa i-AffiliateWP enamandla ukukhulisa isiseko sabathengi se-Ultimo Network yakho!

## Ufakelo {#installation}

1. Layisha 'ultimate-multisite-git ' kwidirektori ethi '/wp-content/plugins/'
2. Vula i-plugin ngemenyu ethi 'Iiplugin' kwi-WordPress
3. Landela i-Wizard inyathelo ngenyathelo ukuze usete i-plugin

## Isikhokelo sokuSebenzisa {#usage-guide}

[AffiliateWP](https://affiliatewp.com/) iye yaba sisisombululo esithandwayo xa kusakhiwa inkqubo yamaqabane entengiso kwishishini lakho le-intanethi. Ultimate Multisite: AffiliateWP yibhulorho eya kukuvumela ukuba wakhe into efanayo kwinethiwekhi yakho ye-premium.

## Ukuqalisa {#getting-started}

Okokuqala, **kufuneka ube** ne-[**AffiliateWP**](https://affiliatewp.com/) **isebenza**! Awuyi kukwazi ukusebenzisa isongezo sethu ukuba awunayo **i-plugin edityaniswa nayo**. Ukuba awucwangcisi ukuvumela abathengi bakho bayisebenzise nakwiisayithi zabo, i-AffiliateWP kufuneka isebenze kuphela kwisayithi yakho ephambili.

Kwakhona, kuya kufuneka zombini **Ultimate Multisite** kunye **Ultimate Multisite: AffiliateWP** **zivulwe** **kwinethiwekhi yonke**.

## Ukongeza i-Ultimate Multisite njengoHlanganiso lwe-AffiliateWP {#adding-ultimate-multisite-as-an-affiliatewp-integration}

Kwisayithi yakho ephambili, yiya kwiiSetingi zamaQabane entengiso, kwaye kwithebhu yoHlanganiso, phawula i-Ultimate Multisite. Oku kuya kuxelela [AffiliateWP](https://affiliatewp.com/) ukuba iqale ukulandelela utyelelo kunye neenguquko ezivela kwindlela yokubhalisa ye-Ultimate Multisite.

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_Phawula ibhokisi ye-Ultimate Multisite ezantsi uze ugcine_

Yiyo phantse yonke into efunekayo. Utyelelo oluvela kwiilinki zokuthunyelwa luya kulandelwa kwaye xa loo msebenzisi kamva esenza intlawulo yakhe yokuqala, inguquko iya kwenziwa kwaye ikhomishini iya kubekelwa bucala iqabane lentengiso elinoxanduva lwenguquko.

Kuphela yintlawulo yokuqala efunyenwe kumsebenzisi omtsha owenziweyo eya kulandelwa, nangona kunjalo. Ukuvumela iintlawulo zokuthunyelwa eziphindaphindayo, kuya kufuneka ufake [i-AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## Ukuvumela ukuThunyelwa okuPhindaphindayo {#enabling-recurring-referrals}

Emva kokufaka [i-AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/), yiya kwiithebhu zeeSetingi zamaQabane entengiso zokuThunyelwa okuPhindaphindayo, uze uvumele ukusetyenziswa kokuthunyelwa okuphindaphindayo.

<!-- Screenshot unavailable: AffiliateWP Recurring Referrals settings with Enable Recurring Referrals toggle -->

_Kulula kangangoko: Vumela ukuThunyelwa okuPhindaphindayo kwaye sele ulungile!_

Kwakhona, yiyo yonke into efunekayo!

Ngoku, ngalo lonke ixesha intlawulo entsha ifunyenwa kwi-akhawunti ethunyelweyo, iqabane lentengiso elinoxanduva liya kufumana isabelo salo ngokuzenzekelayo!
