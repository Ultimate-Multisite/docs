# AI Umakhi Wesayithi {#ai-site-builder}

ultimate-multisite-ai-site-builder

## Iimfuno {#requirements}

- WordPress 5.3 okanye ngaphezulu
- PHP 7.4 okanye ngaphezulu
- i-plugin ye-Multisite Ultimate (iyasebenza)

## Ufakelo {#installation}

1. Layisha iifayile ze-addon kwidirectory yakho `/wp-content/plugins/`
2. Yenza i-plugin isebenze nge-menu ethi 'Plugins' kwi-WordPress
3. Qwalasela iisetingi ze-addon kulawulo lwenethiwekhi

## Uphuhliso {#development}

### Useto {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Uvavanyo {#testing}

Le addon ibandakanya isakhelo sovavanyo esibanzi:

- **Iimvavanyo zeYunithi**: Iimvavanyo ezisekelwe kwi-PHPUnit kunye neseti yovavanyo ye-WordPress
- **Iklasi yoVavanyo Esisiseko**: Inika iindlela zoncedo kwiimeko zovavanyo eziqhelekileyo
- **Ugxininiso loVavanyo**: Luqwalaselwe ukulandela ugxininiso lwekhowudi
- **Uvavanyo lweMultisite**: Iimvavanyo ziqhutywa kwimeko ye-multisite

### Imigangatho yeKhowudi {#code-standards}

- **Imigangatho yokuKhowuda ye-WordPress**: Inyanzeliswa nge-PHPCS
- **Ukuhambelana ne-PHP 7.4+**: Iimpawu zale mihla ze-PHP ziyaxhaswa
- **Uhlalutyo Olungatshintshiyo**: Udibaniso lwe-PHPStan lokujonga uhlobo
- **Ukwenziwa kweKhowudi ibe yeyanamhlanje**: Imithetho ye-Rector yophuculo lwe-PHP

### Inkqubo yoKwakha {#build-system}

- **Ukuqhutywa kwee-Asethi**: Ukuncitshiswa kweefayile ze-CSS/JS
- **Uguqulelo**: Ukuveliswa kwefayile ye-POT
- **Ukupakisha**: Ukwenziwa kwe-archive yokusasazwa
- **Imowudi yoPhuhliso**: Ii-asethi ezingancitshiswanga zokulungisa iimpazamo

## Uqwalaselo {#configuration}

Le addon ixhasa iinketho zoqwalaselo ezahlukeneyo ezifikeleleka ngojongano lolawulo lwenethiwekhi.

## Hooks kunye neeFilters {#hooks-and-filters}

### Izenzo {#actions}

- `ultimate-multisite-ai-site-builder_init` - Iqaliswa emva kokuqaliswa kwe-addon
- `ultimate-multisite-ai-site-builder_loaded` - Iqaliswa emva kokuba zonke iifayile ze-addon zilayishiwe

### IiFilters {#filters}

- `ultimate-multisite-ai-site-builder_settings` - Guqula iisetingi ze-addon
- `ultimate-multisite-ai-site-builder_enabled` - Bhala ngaphezulu imeko yokwenza i-addon isebenze/ingasebenzi
