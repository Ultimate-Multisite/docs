# Isikhethi Solwimi {#language-selector}

Multisite Ultimate: Isikhethi Solwimi ekubhaliseni

Yongeza indawo yesikhethi solwimi kwinkqubo yokubhalisa.

## Ufakelo {#installation}

1. Layisha 'ultimate-multisite-language-selector' kwidirectory ethi '/wp-content/plugins/'
2. Vuselela iplagi ngemenyu ethi 'Iiplagi' kwi-WordPress Network Admin yakho

## Isikhokelo Sokusetyenziswa {#usage-guide}

Ngale songezelelo sasimahla ungavumela abasebenzisi bakho bakhethe ukuba loluphi ulwimi abangathanda ukulubona kwinkqubo yokubhalisa nakwiphepha lokungena.

Emva [kokuvuselela ilayisensi yakho](../../user-guide/getting-started/installing-ultimate-multisite.md) uya kukwazi ukufaka izongezelelo ze-Ultimate Multisite ilayisensi yakho ekunika ukufikelela kuzo.

Yiya kwiphepha lezicwangciso ze-Ultimate Multisite yakho uze, kwimenyu esekhohlo, ucofe ileyibhile ethi _Izongezelelo_.

![Iphepha lezongezelelo](/img/admin/addons-page.png)

_Yiya kwiphepha lezongezelelo ukuze uqalise ukufaka Isikhethi Solwimi_

Apho, uya kufumana zonke izongezelelo esizinikezelayo.

Izongezelelo zasimahla ziyafumaneka kubo bonke abathengi bethu, nokuba sesiphi isicwangciso sobhaliso ababhalise kuso.

Cofa iqhosha elithi _Yifake Ngoku!_ ngaphantsi komfanekiso omncinci wesongezelelo. Emva koko yivuselele kwiphepha leePlagi.

![Iphepha lezongezelelo elineqhosha lokufaka](/img/admin/addons-page.png)

_Cofa iqhosha ukufaka isongezelelo_

<!-- Umfanekiso-skrini awufumaneki: iphepha leePlagi ze-WP libonisa iqhosha le-Network Activate lesongezelelo se-Language Selector -->

_Yivuselele kwiphepha lakho leePlagi_

Ngoku, yiya kwimenyu yeZicwangciso ze-Ultimate Multisite uze ungene kwithebhu ethi _Izicwangciso zeNethiwekhi_. Apho, ezantsi kwephepha, uya kufumana icandelo elitsha elibizwa _Iilwimi_.

![Iphepha lezicwangciso](/img/config/settings-general.png)

_Icandelo elitsha kraca liyavela!_

Ungayishiya ingenanto ukuze ubonise zonke iilwimi njengeenketho okanye ukhethe ezo ufuna ukuzenza zifumaneke. Iilwimi zezo uzifakileyo kulungiselelo lwakho lwe-WordPress.

<!-- Umfanekiso-skrini awufumaneki: imenyu eyehlayo yokukhetha ulwimi ibonisa iilwimi ezifumanekayo kufakelo lwe-WP -->

_Uluhlu lweelwimi ezifumanekayo kolu fakelo lwe-WP_

Emva kokugcina utshintsho olwenzileyo, imenyu yokukhetha eneelwimi ozikhethileyo iya kuboniswa kwiphepha lokuNgena nakwiNkqubo yokuBhalisa.

## Ukuseta Ulwimi nge-URL {#setting-the-language-via-the-url}

Ungalutshintsha kwakhona ulwimi lwephepha ngokugqithisa iparamitha ye-locale kwi-URL. Ukuba ufuna ukubhekisa kwiphepha lokubhalisa ngesiTaliyane, umzekelo, yongeza ?locale=it_IT kwi-URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Ulwimi kufuneka lufakwe kwinethiwekhi yakho ye-WordPress ukuze oku kusebenze.
