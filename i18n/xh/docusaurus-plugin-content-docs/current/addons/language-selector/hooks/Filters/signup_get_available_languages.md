# Isihluzi: signup_get_available_languages

Sihluza uluhlu lweelwimi ezikhoyo kubhaliso lweesayithi ze-front-end.

Ukudlulisa uluhlu olungenanto kule hook kuya kuvala ukuboniswa kwesetingi kwifomu yobhaliso, kwaye ulwimi olungagqibekanga luya kusetyenziswa xa kusenziwa isayithi. Iilwimi ezingekafakelwa ziya kususwa.

## Iiparamitha {#parameters}

|------|------|-------------|
| $available_languages | `array` | Iilwimi ezikhoyo. |
