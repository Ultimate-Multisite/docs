# Superdav AI Agent {#superdav-ai-agent}

Superdav AI Agent ngumncedisi we-WordPress AI ophambili owandisa Gratis AI Agent ngezakhono eziqonda theme, amandla okuphendla i-intanethi aphuculiweyo, iimpawu zesakhi sesayithi, kunye nodibaniso oluzenzekelayo olukhethwayo. Uguqulelo 1.18.0 longeza izixhobo zokufunda ze-Google Calendar, ukumatanisa abafowunelwa nabazimasi, amasango okuvunywa ngabantu, ukususa ukuphindaphindwa kwezikhumbuzi, izaziso ze-SMS ze-TextBee, ukulungiselela inkonzo ye-Superdav elawulwayo, kunye nephakheji ye-Advanced ehamba nayo esasazwa ngokwahlukeneyo ukwenzela iinkqubo zomsebenzi zabaphuhlisi abathembekileyo.

## Iimpawu Ezibalulekileyo {#key-features}

- **Izakhono eziqonda theme** — zilungelelanisa ngokuzenzekelayo isikhokelo namandla ngokusekelwe kwi-theme yakho ye-WordPress esebenzayo nakwiiplugin ezifakiweyo
- **Ababoneleli abaninzi bokuphendla i-intanethi** — khetha phakathi kwe-Brave Search ne-Tavily ukuze ufumane iziphumo zokuphendla ezityebileyo namandla ophando aphambili
- **Iifomu zoqhagamshelwano zesakhi sesayithi** — yenza iifomu zoqhagamshelwano ezisebenza ngokupheleleyo ngqo kwindawo yencoko usebenzisa iplugin yeefomu oyifakileyo
- **Isakhi sesayithi esiphuculiweyo** — izicwangciso zokwakha isayithi ezinamanyathelo amaninzi eziphuculweyo, ezinokufumanisa ngcono iiplugin nokubuyisela kwiimpazamo
- **Izakhono ze-theme ezakhelwe ngaphakathi** — isikhokelo esikhethekileyo se-Block Themes, Classic Themes, Kadence Blocks, kunye ne-Kadence Theme
- **Udibaniso lwe-WooCommerce oluphuculiweyo** — luhlengahlengisiwe ukuze lusebenzise ii-API zemveli ze-WooCommerce ukuze kuthembeke ngakumbi kwaye kuhambelane ngcono
- **Ukufumanisa ababoneleli ngokuzenzekelayo** — kufumanisa ngokuzenzekelayo kwaye kuhlaziya iiplugin nee-theme ezikhoyo xa zivulwa okanye zicinywa
- **Ukuveliswa kwe-logo yesiko** — velisa kwaye ufake ii-logo ze-SVG zesiko ngokucocwa okukhuselekileyo kwe-namespace ngokuzenzekelayo
- **Ukuqinisekiswa komahluko we-palette** — jonga izibini zemibala ukuze zihambelane nokufikeleleka kwe-WCAG ngaphambi kokuzisebenzisa kwi-theme yakho
- **Ukufumanisa okuphuculwe ngeefoto** — layisha iifoto ngexesha lodliwano-ndlebe lokufumanisa lwe-Theme Builder ukuze ziphembelele indlela yoyilo
- **Iimenyu zoshishino lokwamkela iindwendwe** — velisa amaphepha eemenu zokutya neziselo acwangcisiweyo kwiindawo zokutyela, iikhefi, namashishini okwamkela iindwendwe
- **Imiboniso yangaphambili yedesktop neyeselula** — jonga uyilo lwakho kwangaphambili kwizixhobo zedesktop nezeselula ngexesha lokukhetha indlela yoyilo
- **Ukukhuhla isayithi kwe-Tier 1** — sd-ai-agent/site-scrape ngoku iyafumaneka ngokungagqibekanga ukuhlalutya iiwebhusayithi zokhuphiswano nezokuphefumlelwa
- **Izixhobo zokufunda ze-Google Calendar** — funda iikhalenda neziganeko ezilungiselelweyo ukuze i-automation ikwazi ukuqonda umxholo weshedyuli ezayo
- **Ukumatanisa abafowunelwa nabazimasi** — thelekisa abathathi-nxaxheba bekhalenda nabasebenzisi besayithi nabafowunelwa abaziwayo ngaphambi kokuvelisa umsebenzi wokulandelela
- **Amasango okuvunywa ngabantu** — nqumamisa ii-automation ezibuthathaka ukuze ziphononongwe ngaphambi kokuthumela imiyalezo, ukuqhubeka nomsebenzi, okanye ukwenza utshintsho
- **Iirekhodi zezikhumbuzi nokususa ukuphindaphindwa** — landela izikhumbuzi esele zithunyelwe ukuze ukuqhutywa kwe-automation okuphindaphindiweyo kungamazisi umamkeli ofanayo kabini
- **Izaziso ze-SMS ze-TextBee** — thumela izaziso zemiyalezo ebhaliweyo ezilungiselelweyo nge-TextBee xa umboneleli we-SMS evuliwe
- **Ukulungiselela inkonzo ye-Superdav elawulwayo** — qhagamshela iisayithi ezixhaswayo kwii-endpoint ze-Superdav ezisingathiweyo ngokuseta okuzenzekelayo endaweni yokulungiselela i-endpoint ngesandla
- **Iphakheji ye-Advanced ehamba nayo** — izixhobo zabaphuhlisi abathembekileyo ezisasazwa ngokwahlukeneyo zeenkqubo zomsebenzi ze-filesystem, database, WP-CLI, REST dispatcher, isakhi se-plugin, git snapshot, ulawulo lwabasebenzisi, kunye ne-benchmark

## Iimfuno {#requirements}

- WordPress 6.0 okanye ngaphezulu
- PHP 8.1 okanye ngaphezulu
- WordPress Multisite (evulwe kwinethiwekhi) okanye ufakelo lwesayithi enye
- Iplugin esebenzayo yomdibanisi womboneleli we-AI (Anthropic, OpenAI, okanye ehambelanayo)
- Gratis AI Agent v1.9.0 okanye ngaphezulu (Superdav AI Agent yandisa Gratis AI Agent)

## Ufakelo {#installation}

1. Layisha iifayile ze-plugin ku-`/wp-content/plugins/superdav-ai-agent`
2. Vula iplugin ngesikrini se-**Plugins** kwi-WordPress
3. Qinisekisa ukuba Gratis AI Agent nayo ivuliwe
4. Yiya ku-**Gratis AI Agent → Settings** kwaye ulungiselele umboneleli wakho wokuphendla i-intanethi, uqhagamshelo lwenkonzo ye-Superdav elawulwayo, Google Calendar, okanye iziqinisekiso ze-TextBee njengoko kufuneka

Iphakheji ye-WordPress.org iqulethe iimpawu ezisisiseko ze-Superdav AI Agent. Iphakheji ye-Advanced ehamba nayo yakhiwa kwaye isasazwa ngokwahlukeneyo kwiindawo apho izixhobo zabaphuhlisi abathembekileyo zifakwa kwaye zigunyaziswa ngokucacileyo.

## Izakhono Eziqonda Theme {#theme-aware-skills}

Superdav AI Agent ifumanisa ngokuzenzekelayo i-theme yakho esebenzayo kunye neeplugin ezifakiweyo, ilayisha isakhono esifanelekileyo esiqonda theme kumxholo womncedisi. Izakhono ezikhoyo ziquka:

| Isakhono | Sisebenza Kwi |
|---|---|
| **Block Themes** | Ii-theme ezisebenzisa ulwakhiwo lwe-WordPress olusekelwe kwiibhloko (Full Site Editing) |
| **Classic Themes** | Ii-theme zemveli ze-WordPress ezisebenzisa iitemplate ze-PHP nomhleli wakudala |
| **Kadence Blocks** | Iisayithi ezineplugin ye-Kadence Blocks esebenzayo |
| **Kadence Theme** | Iisayithi ezine-theme ye-Kadence njenge-theme esebenzayo |

Jonga [Izakhono Eziqonda Theme](../../user-guide/configuration/theme-aware-skills) ukuze ufumane isikhokelo esineenkcukacha kwisakhono ngasinye.

## Ababoneleli Bokuphendla I-intanethi {#internet-search-providers}

Superdav AI Agent v1.10.0 yongeza **Tavily** njengenye indlela ye-Brave Search kwimibuzo yokuphendla i-intanethi. Bobabini ababoneleli banikezela ngeziphumo zokuphendla zexesha lokwenyani kwaye banokulungiselelwa ku-**Gratis AI Agent → Settings → Advanced**.

| Umboneleli | Iimpawu | Ukuseta |
|---|---|---|
| **Brave Search** | Iziphumo zokuphendla ezikhawulezayo, ezigxile kubumfihlo | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Iziphumo zokuphendla ezityebileyo, amandla ophando aphambili | [Tavily API](https://tavily.com/) |

Jonga [Ukuphendla I-intanethi](../../user-guide/configuration/internet-search) ukuze ufumane imiyalelo yolungiselelo.

## Iikhalenda, Ukuvunywa, Neenkqubo Zomsebenzi Zezaziso {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 yandisa iinkqubo zomsebenzi ezizenzekelayo ngaphaya kwemisebenzi yomxholo nokwakhiwa kwesayithi:

- **Izixhobo ze-Google Calendar** zivumela i-agent ukuba ifunde iikhalenda ezilungiselelweyo kunye neenkcukacha zeziganeko xa iziqinisekiso zekhalenda zikhona.
- **Abancedisi babafowunelwa nabazimasi** bamatanisa abathathi-nxaxheba beziganeko nabasebenzisi be-WordPress okanye abafowunelwa abaziwayo ngaphambi kokulungiselela izikhumbuzi okanye imisebenzi yokulandelela.
- **Amasango okuvunywa ngabantu** avumela i-automation ukuba inqumame de umsebenzisi ogunyazisiweyo aphonononge isenzo esicetywayo.
- **Iirekhodi zezikhumbuzi** zithintela izikhumbuzi zekhalenda okanye zezaziso eziphindaphindiweyo xa i-automation izanywa kwakhona okanye icwangciswe ngokuphindaphindiweyo.
- **Inkxaso ye-SMS ye-TextBee** yenza izaziso zemiyalezo ebhaliweyo xa umboneleli we-TextBee elungiselelwe.

Sebenzisa ezi zixhobo kuncedo oluqonda ishedyuli, ukulandelela abazimasi, kunye neenkqubo zomsebenzi zezikhumbuzi ezisafuna uphononongo lomntu ngaphambi kokuba imiyalezo ebuthathaka ithunyelwe.

## Uqwalaselo {#configuration}

Zonke iisetingi ziphantsi kwe-**Gratis AI Agent → Settings** kulawulo lwe-WordPress. Superdav AI Agent yongeza ezi ndlela zilandelayo zolungiselelo:

| Isetingi | Inkcazelo |
|---|---|
| **Internet Search Provider** | Khetha phakathi kwe-Brave Search ne-Tavily (ukhetho) |
| **Tavily API Key** | Isitshixo se-API sokuphendla i-intanethi nge-Tavily (ukuba i-Tavily ikhethiwe) |
| **Managed Superdav Service** | Qhagamshela iisayithi ezixhaswayo kwinkonzo ye-Superdav esingathiweyo ngokulungiselela i-endpoint ngokuzenzekelayo |
| **Google Calendar Credentials** | Lungiselela ukufikelela kwikhalenda ukuze i-agent ikwazi ukufunda iikhalenda neziganeko |
| **TextBee SMS Provider** | Lungiselela iziqinisekiso ze-TextBee zeenkqubo zomsebenzi zezaziso ze-SMS ezivunyiweyo |

## Iphakheji ye-Advanced Ehamba Nayo {#advanced-companion-package}

Superdav AI Agent v1.18.0 yongeza iphakeji ye-Advanced ehamba nayo esasazwa ngokwahlukeneyo ukwenzela iinkqubo zomsebenzi zabaphuhlisi abathembekileyo. La mandla awabandakanywanga kwiphakheji engundoqo ye-WordPress.org. Faka iphakeji ehamba nayo kuphela kwiisayithi apho abalawuli bethemba ngokucacileyo i-agent ukuba ifikelele kwizixhobo zophuhliso.

Izixhobo ze-Advanced ziquka imisebenzi ye-filesystem, ukufikelela kwi-database, WP-CLI, iinkqubo zomsebenzi ze-REST dispatcher, izixhobo zesakhi se-plugin, ii-git snapshots, abancedisi bolawulo lwabasebenzisi, kunye nezixhobo ze-benchmark. Gcina ezi zixhobo zinqunyelwe kubalawuli abathembekileyo kwaye uphonononge uqwalaselo lwe-approval-gate ngaphambi kokuzivula kwiisayithi zemveliso.
