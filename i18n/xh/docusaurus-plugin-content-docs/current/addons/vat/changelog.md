# Ingxelo yotshintsho ye-VAT {#vat-changelog}

Inguqulelo 1.0.7 - Ikhutshwe nge-2026-02-03

* Ulungiso: Iinombolo ze-VAT zaseJamani (DE) bezisilela ekuqinisekisweni ngenxa yeempazamo ze-VIES SOAP service MS_MAX_CONCURRENT_REQ. Kutshintshwe ukusuka kwi-SOAP API yakudala kwaya kwi-EU VIES REST API esemthethweni ukuze kuqinisekiswe ngokuthembeka ngakumbi.
* Ulungiso: Iinombolo ze-VAT ngoku zamkelwa xa i-VIES service ingafumaneki okwethutyana, endaweni yokwaliwa ngokungachanekanga. Ungeno lwelog luyadalwa ukuze luphononongwe kamva.
* Ulungiso: Ifildi ye-VAT/Tax ID ngoku iyabonakala kuwo onke amazwe, hayi kumalungu e-EU kuphela. Ii-Tax ID ezingezizo ze-EU (umz. iinombolo zaseSwitzerland ze-CHE) zigcinwa ukuze ziboniswe kwi-invoice ngaphandle kokuqinisekiswa kwe-VIES.
* Ulungiso: Ukukhangelwa kwe-prefix ye-VAT yaseGrisi (GR), Monaco (MC), ne-Isle of Man (IM) bekonakele ngenxa yelogic engachanekanga yokukhangela kwi-array.
* Uphuculo: Kususiwe uxhomekeko lwe-dev olungafunekiyo lwe-wp-cli-bundle olunokubangela iimpazamo ezibulalayo xa lusetyenziswa kunye nesongezo se-Site Exporter.
* Uphuculo: Kongezwe iseti ebanzi yovavanyo lwe-unit (iimvavanyo ezingama-53).

Inguqulelo 1.0.6 - Ikhutshwe nge-2026-01-25

* Ulungiso: Iintengiselwano zasekhaya ze-B2B ngoku zihlawulisa i-VAT ngokuchanekileyo. Ngokwemithetho ye-EU VAT, i-reverse charge isebenza kuphela kwiintengiselwano ze-B2B ezinqumla imida, hayi xa ilizwe lomthengi lifana nelizwe lenkampani.

Inguqulelo 1.0.5 - Ikhutshwe nge-2026-01-22

* Ulungiso: Isongezo besingalayishi okanye singasebenzi kakuhle.
* Utshintsho: Kutshintshwe umthombo wedatha yereyithi ye-VAT ukusuka ku-euvatrates.com kwaya kwirepository ye-ibericode/vat-rates ukuze kufumaneke idatha ethembeke ngakumbi kwaye egcinwa ngenkuthalo.
* Ulungiso: Kulungiswe impazamo yokuchwetheza kwixabiso lokhetho lwe-super_reduced_rates.
* Uphuculo: Kongezwe iitshekhi ze-null ekuphathweni kwekhowudi yelizwe ukuze kuthintelwe iimpazamo.
* Utshintsho: Kususiwe iifayile zoguqulelo ezibundelweyo endaweni yohlaziyo oluzenzekelayo nge-Traduttore.
