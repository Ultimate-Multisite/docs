# VAT

Multisite Ultimate: VAT yaseYurophu

Yongeza iimpawu nezixhobo ezongezelelweyo kulawulo lwerhafu olusisiseko lwe-Multisite Ultimate ukuze ixhase ngcono i-VAT yaseYurophu.
Yongeza ngokuzenzekelayo amaxabiso e-EU VAT kwiphepha lamaXabiso eRhafu ukusuka

## Ufakelo

1. Layisha 'ultimate-multisite-vat' kulawulo lwe-'/wp-content/plugins/'
2. Vula iplugin ngemenyu ethi 'IiPlugin' kwi-WordPress
3. Yiya ku-Ultimate Multisite > Iisetingi > Iirhafu uze uvule i-EU VAT.
4. Cofa ku-Maxabiso eRhafu uze uhlaziye amaXabiso e-EU VAT ezantsi, emva koko ugcine amaXabiso eRhafu.
5. Kugqityiwe!
