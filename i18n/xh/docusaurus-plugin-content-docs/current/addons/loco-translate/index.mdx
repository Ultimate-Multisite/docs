# Loco Translate

Isongezo se-Ultimate Multisite: Loco Translate Integration sizisa amandla enguqulelo akhethekileyo kwisayithi nganye kwinethiwekhi yakho ye-multisite ye-WordPress. Ukuba wakhe wafuna iisayithi ezahlukeneyo kwinethiwekhi yakho zibe neenguqulelo ezahlukeneyo zeplagin okanye umxholo ofanayo, esi songezo sisisombululo esigqibeleleyo.

**Okwenziwa Sesi Songezo**

Esi songezo sidibanisa ngokungenamthungo i-Ultimate Multisite neplagin edumileyo ye-Loco Translate, sivumela isayithi nganye engaphantsi kwinethiwekhi yakho ukuba igcine iinguqulelo zayo ezenzelwe yona. Nokuba uqhuba iqonga lemimandla emininzi, ukhonza uluntu oluthetha iilwimi ezahlukeneyo, okanye ufuna nje ukuguquguquka kweenguqulelo kuyo yonke inethiwekhi yakho, esi songezo sikwenza kube lula.

**Silungele:**

* Amashishini emimandla emininzi aneemfuno ezahlukeneyo zokulungelelanisa ulwimi nendawo
* Amaziko emfundo akhonza uluntu olwahlukeneyo
* Amaqonga e-SaaS anika inkxaso yeelwimi ezininzi
* Ii-arhente zewebhu ezilawula iisayithi ezininzi zabathengi
* Nayiphi na inethiwekhi ye-multisite efuna ukuguquguquka kweenguqulelo

## Ufakelo {#installation}

1. Qinisekisa ukuba i-Ultimate Multisite ifakiwe kwaye yenziwe yasebenza kwinethiwekhi
2. Faka kwaye wenze isebenze iplagin ye-Loco Translate
3. Layisha esi songezo kulawulo lwakho lweeplagin
4. Yenza sisebenze kwinethiwekhi isongezo se-Loco Translate Integration
5. Qalisa ukwenza iinguqulelo ezikhethekileyo kwisayithi nge-Loco Translate kuyo nayiphi na isayithi engaphantsi

Kuphelele apho! Udibaniso lusebenza ngokuzenzekelayo ngaphandle kwalo naluphi na uqwalaselo olongezelelweyo.

## Imibuzo Ebuzwa Rhoqo {#frequently-asked-questions}

## Ngaba kufuneka ndiqwalasele nantoni na emva kofakelo? {#do-i-need-to-configure-anything-after-installation}

Hayi! Udibaniso luzenzekela. Nje ukuba zombini isongezo ne-Loco Translate zisebenze, iisayithi ezingaphantsi zinokuqalisa kwangoko ukwenza iinguqulelo zazo ezenzelwe zona.

## Kwenzeka ntoni ukuba isayithi engaphantsi ayenzi iinguqulelo ezenzelwe yona? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

Isayithi iya kusebenzisa ngokuzenzekelayo iinguqulelo zenethiwekhi yonke njengento yokubuyela kuyo. Akukho nto yonakalayo, kwaye yonke into isebenza ngokungenamthungo.

## Ngaba iisayithi ezingaphantsi zingafikelela kwiinguqulelo zenethiwekhi yonke? {#can-subsites-access-network-wide-translations}

Abalawuli besayithi abaqhelekileyo babona kwaye bahlele kuphela iinguqulelo zesayithi yabo. Abalawuli abakhulu banokufikelela kuzo zombini iinguqulelo zenethiwekhi yonke nezikhethekileyo kwisayithi, kuxhomekeke ekubeni bajonge eyiphi isayithi.

## Ngaba oku kusebenza kuzo zombini iiplagin nemixholo? {#does-this-work-with-both-plugins-and-themes}

Ewe! Iinguqulelo ezikhethekileyo kwisayithi zisebenza kuzo zombini iiplagin nemixholo ye-WordPress.

## Ngaba oku kuya kuyicothisa inethiwekhi yam ye-multisite? {#will-this-slow-down-my-multisite-network}

Hayi. Isongezo sisebenzisa ukulayishwa kweefayile okusebenzayo okujonga kuphela iinguqulelo ezikhethekileyo kwisayithi xa zifuneka. Akukho mpembelelo ibonakalayo ekusebenzeni.

## Ndingazikopa iinguqulelo zenethiwekhi kwisayithi ethile njengesiqalo? {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

Ewe! Ngelixa uhlela iinguqulelo kwi-Loco Translate, ungaqala ngokulula ngeenguqulelo zenethiwekhi uze uzenze zilungele isayithi nganye.

## Ngaba oku kusebenza ne-WooCommerce kunye nezinye iiplagin? {#does-this-work-with-woocommerce-and-other-plugins}

Ngokuqinisekileyo! Nayiphi na iplagin okanye umxholo osebenzisa imisebenzi esemgangathweni yeenguqulelo ze-WordPress uya kusebenza ngeenguqulelo ezikhethekileyo kwisayithi.
