# Material WP

Ujongano lolawulo lwe-WordPress belusoloko lufana iminyaka emininzi. Kodwa wakha wazibuza ukuba inguqulelo ye-Material Design yojongano lolawulo ingabonakala njani? 

Material WP ngumxholo wolawulo ophuhliselwe ukukunika loo nkangeleko nemvakalelo, ukukunceda unike abathengi bakho i-Dashboard eyenziwe ngakumbi ngokwezifiso.

## Iimpawu

- Inkxaso ye-Custom CSS
- Iisetingi zesimbo ezilula
- Ukucwangcisa kwakhona imenyu
- Ukungenisa/ukukhupha okulula

## Amaxwebhu

**Ukwenza i-Admin Panel yakho ibe ngokwezifiso nge-Material WP**

## Ukufakwa

1. Layisha 'material-wp' kulawulo lwe-'/wp-content/plugins/'
2. Vula iplagi ngeemenyu ye-'Plugins' kwi-WordPress
