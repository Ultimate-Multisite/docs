# Umphathi Wezongezo Nemixholo

Multisite Ultimate: Umphathi Wezongezo Nemixholo

Hlela okanye ufihle ulwazi lwe-meta (isihloko, umbhali, umfanekiso omncinci nenkcazelo) lwezongezo nemixholo efakiweyo. Yenza iindidi zezongezo nemixholo ukuze uvumele abasebenzisi bakho bazihluze kwaye wongeze iphepha elihle leZongezo elilungiselelweyo kwiphaneli yabasebenzisi bakho.

Ukuze usebenzise isongezo, ungaya nje kumaphepha akho eZongezo okanye eMixholo kumlawuli wenethiwekhi yakho uze ucofe ku-"Hlela ulwazi lweSongezo/Umxholo".

## Ufakelo {#installation}

1. Layisha 'ultimate-multisite-plugin-and-theme-manager' kulawulo lwe-'/wp-content/plugins/'
2. Vula isongezo usebenzisa imenyu ethi 'Izongezo' kwi-WordPress Network Admin yakho

## Isikhokelo Sokusetyenziswa {#usage-guide}

Ultimate Multisite: Umphathi Wezongezo Nemixholo sisongezo esinamandla **esihlawulelwayo** se-Ultimate Multisite esikunika ulawulo olupheleleyo kwindlela abasebenzisi benethiwekhi yakho ababona ngayo baze basebenzisane ngayo neenketho zezongezo nemixholo kwinethiwekhi yakho.

Esi songezo sisebenza ngeenjongo ezintathu:

  1. Sikuvumela ukuba utshintshe ngokupheleleyo idatha ye-meta yezongezo kunye nemixholo. Oko kuquka igama, inkcazelo, igama lombhali, kwanemifanekiso emincinci!

  2. Sikuvumela ukuba uhlele izongezo nemixholo ngokweendidi, ukuze uvumele abasebenzisi bakho bazilungelelanise ngokulula.

  3. Sikunika ukhetho lokutshintsha iphepha elidala nelidikayo lezongezo ngephepha elitsha elifana neleemixholo elinokuhlelwa ngokweendidi.

Emva kokuyikhuphela nokuyifaka, ithebhu entsha - _Iisetingi Zesongezo_ \- iya kubonakala kwiphepha le-Ultimate Multisite Settings.
