# Gratis AI Agent

Gratis AI Agent ngumncedisi we-AI ozimeleyo we-WordPress onokucwangcisa, akhe, aze alawule iisayithi ze-WordPress ezipheleleyo ngemiyalelo yolwimi lwendalo. Ibonisa iseti ecwangcisiweyo **yezakhono** — izenzo ezahlukileyo, ezinokuvavanywa iarhente enokuzibiza — ezigubungela yonke into ukusuka kulawulo lomxholo ukuya kulungiselelo lwenkqubo yoyilo. Uguqulelo 1.9.0 longeza izakhono zokudala umxholo, ukuphononongwa okubonakalayo kwephepha ngeesikrini ezithathwe kwicala lomxhasi, iiprofayili ezintlanu zearhente ezakhelweyo, kunye neeflegi zeempawu zolawulo lofikelelo kunye nokufakela uphawu.

## Iimpawu Ezibalulekileyo {#key-features}

- **Inkqubo yezakhono** — ikhathalogu yeemodyuli, enokwandiswa yezenzo iarhente enokuzibiza kufakelo lwakho lwe-WordPress
- **Ulawulo Lomxholo** — dala izithuba namaphepha (kuquka ukwabela iitemplate zamaphepha), dala izithuba ezininzi ngexesha elinye kubizo olunye, seta imifanekiso ephambili, uze wakhe iifom zoqhagamshelwano
- **Uphononongo Olubonakalayo** — thatha iiscreenshot zamaphepha aphilayo, thelekisa iimeko zangaphambi/zasemva, uze ufumane uphononongo loyilo oluveliswe yi-AI olugubungela ubeko, uhlobo lombhalo, umbala, kunye nokufikeleleka
- **Iiarhente ezintlanu ezakhelweyo** — Umbhali Womxholo, Umakhi Wesayithi, Isitudiyo Soyilo, Umphathi wePlugin, kunye noMncedisi Wenkxaso; zitshintshwa ngeSikhethi seArhente kwisihloko sengxoxo
- **Iiflegi zeempawu** — iitogoli zolawulo lofikelelo kunye nokufakela uphawu ku-Iisetingi → Iiflegi Zeempawu kwizithintelo zendima, ukubizwa ngegama elingenaphawu, kunye nokwenza ngokwezifiso iwijethi
- **Ulawulo lweentlobo zezithuba ezilungiselelweyo** — bhalisa, dwelisa, uze ucime iintlobo zezithuba ezilungiselelweyo ezinokugcinwa ngokuqhubekayo nasemva kokuqalisa kwakhona
- **Ulawulo lweetaksonomi ezilungiselelweyo** — bhalisa, dwelisa, uze ucime iitaksonomi ezilungiselelweyo ngokugcinwa ngokuqhubekayo
- **Izakhono zeNkqubo Yoyilo** — faka i-CSS elungiselelweyo, lawula iipatheni zeebhloko ezikhethiweyo, seta ilogo yesayithi, uze usebenzise iipresethi ze-theme.json
- **Ulawulo lweZimbo zeHlabathi** — funda uze ubhale amaxabiso e-theme.json eepalethi zemibala, uhlobo lombhalo, izithuba, nokunye
- **Ulawulo lweMenyu Yokukhangela** — dala uze uhlaziye iimenyu zokukhangela nezinto zazo ngokwenkqubo
- **Ulawulo Lweenketho** — funda uze ubhale iinketho ze-WordPress ngoluhlu lokuthintela lokhuseleko olwakhelweyo ukukhusela iisetingi ezibalulekileyo
- **Irejistri Yezakhono Ezifakelwayo** — fumana uze uvule iipakethi zezakhono ezongezelelweyo ngaphandle kokutshintsha iifayile ezingundoqo zeplugin
- **Ulungelelaniso loMakhi Wesayithi v2** — izicwangciso zokwakha isayithi ezinamanyathelo amaninzi ngokufunyanwa kweplugin, ukulandelwa kwenkqubela, kunye nokubuyisela kwiimpazamo
- **Umyalelo wovavanyo lwe-WP-CLI** — sebenzisa isuti yovavanyo lokusebenza kwezakhono zearhente kumgca wemiyalelo ye-CI kunye novavanyo lwemodeli

## Iimfuno {#requirements}

- WordPress 6.0 okanye ngaphezulu
- PHP 8.1 okanye ngaphezulu
- WordPress Multisite (evulwe kwinethiwekhi) okanye ufakelo lwesayithi enye
- Iplugin yesiqhagamsheli somboneleli we-AI esebenzayo (Anthropic, OpenAI, okanye ehambelanayo)

## Ufakelo {#installation}

1. Layisha iifayile zeplugin ku-`/wp-content/plugins/gratis-ai-agent`
2. Vula iplugin ngesikrini se-**Plugins** kwi-WordPress
3. Yiya ku-**Gratis AI Agent → Iisetingi** uze uqhagamshele umboneleli wakho we-AI
4. (Ngokuzithandela) Jonga **Irejistri Yezakhono** ukuze ufake iipakethi zezakhono ezongezelelweyo

## Ukuqalisa Ngokukhawuleza {#quick-start}

Nje ukuba ivulwe yaza yaqhagamshelwa kumboneleli we-AI, vula ipaneli ye-**Gratis AI Agent** kwibha esecaleni yolawulo lwe-WordPress. Chwetheza umyalelo wolwimi lwendalo kwaye iarhente iya kucwangcisa ize iphumeze amanyathelo afunekayo isebenzisa izakhono zayo ezikhoyo.

**Imiyalelo yomzekelo:**

```
Dala uhlobo lwesithuba olulungiselelweyo lwe-"Portfolio" ngetaksonomi ye-"Project Category".
```

```
Faka ipalethi yemibala yemowudi emnyama kwi-theme uze usete umbala ophambili ku-#1a1a2e.
```

```
Yakha iwebhusayithi yerestyu enemenyu ekwi-intanethi, ifom yokubhukisha, kunye nephepha loqhagamshelwano.
```

## Isishwankathelo Sezakhono {#abilities-overview}

Izakhono zizenzo ezisisiseko iarhente enokuzenza. Isakhono ngasinye yiklasi ye-PHP ebhalisiweyo ebonisa iskimu (iiparamitha kunye neentlobo zokubuyisa) iarhente ezifundayo ngexesha lokusebenza. Izakhono ziqokelelwe kwiindawo zokusebenza:

| Indawo | Izakhono |
|---|---|
| **Ulawulo Lomxholo** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **Uphononongo Olubonakalayo** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **Iintlobo Zezithuba Ezilungiselelweyo** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **Iitaksonomi Ezilungiselelweyo** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **Inkqubo Yoyilo** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **Izimbo Zehlabathi** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **Iimenyu Zokukhangela** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **Ulawulo Lweenketho** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **Izakhono Ezifakelwayo** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **Umakhi Wesayithi** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

Bona [Isalathiso Sezakhono](./abilities) ukuze ufumane uxwebhu olupheleleyo lweeparamitha kunye nemizekelo yokusetyenziswa.

## WP-CLI {#wp-cli}

Gratis AI Agent iza nosapho lwemiyalelo ye-`wp gratis-ai-agent` yokusebenzisa iimvavanyo zokusebenza nokulawula izakhono kumgca wemiyalelo. Bona [Isalathiso se-WP-CLI](./wp-cli) ukuze ufumane uluhlu olupheleleyo lwemiyalelo.

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## Uqwalaselo {#configuration}

Zonke iisetingi ziphantsi kwe-**Gratis AI Agent → Iisetingi** kulawulo lwe-WordPress:

| Isetingi | Inkcazelo |
|---|---|
| **Umboneleli we-AI** | Khetha uze uqwalasele isiqhagamsheli somboneleli we-AI esisebenzayo |
| **Imodeli** | Khetha imodeli yolwimi eza kusetyenziswa kwiiseshoni zearhente |
| **IiToken Ezininzi** | Uhlahlo lwabiwo-mali lweetoken kwisicelo ngasinye (okungagqibekanga 4096) |
| **Ixesha Lokuphela Lesakhono** | Imizuzwana ephezulu isakhono ngasinye esinokusebenza ngayo (okungagqibekanga 30) |
| **Ukuloga KweDebug** | Loga ubizo lwezakhono kunye nokuqiqa kwearhente kwi-log ye-debug ye-WordPress |

## Uluhlu Lokuthintela Lokhuseleko Lolawulo Lweenketho {#options-management-safety-blocklist}

Isakhono soLawulo Lweenketho siquka uluhlu lokuthintela olwakhelweyo oluthintela iarhente ekufundeni okanye ekubhaleni iinketho ze-WordPress ezibuthathaka. Uluhlu lokuthintela olungagqibekanga lugubungela:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (iinketho ze-URL — sebenzisa isixhobo sokufudusa esizinikeleyo kwezi)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (izitshixo ezifihlakeleyo)
- `active_plugins`, `template` (ukuvulwa kweplugin/theme — sebenzisa izakhono ezizinikeleyo)
- Iinkcukacha zokuqinisekisa zamasango entlawulo kunye nezitshixo ze-API ezibhaliswe ziiplugin ezifakiweyo

Amangeniso ongezelelweyo anokongezwa ngesihluzi se-`gratis_ai_agent_options_blocklist`.

## IiHook Nezihluzi {#hooks-and-filters}

### Izenzo {#actions}

- `gratis_ai_agent_ability_registered` — ivuswa emva kokuba isakhono sibhalisiwe, ifumana igama leklasi yesakhono
- `gratis_ai_agent_plan_started` — ivuswa xa isicwangciso somakhi wesayithi siqala ukuphunyezwa
- `gratis_ai_agent_plan_completed` — ivuswa xa isicwangciso sigqitywa ngempumelelo
- `gratis_ai_agent_plan_failed` — ivuswa xa isicwangciso sidibana nempazamo engenakubuyiselwa

### Izihluzi {#filters}

- `gratis_ai_agent_abilities` — lungisa uluhlu lwezakhono ezibhalisiweyo ngaphambi kokuba iarhente ilulayishe
- `gratis_ai_agent_options_blocklist` — yandisa okanye unciphise uluhlu lokuthintela lokhuseleko lweenketho
- `gratis_ai_agent_benchmark_questions` — yongeza imibuzo yovavanyo lokusebenza elungiselelweyo kwisuti yovavanyo
- `gratis_ai_agent_plan_plugins` — lungisa uluhlu lokufunyanwa kweplugin ngexesha lokuveliswa kwesicwangciso sesayithi
