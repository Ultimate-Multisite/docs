Version 1.3.0 - Ikhutshwe nge-2026-06-02
- Entsha: Kongezwe isilumkiso somlawuli-wenethiwekhi xa ibhalansi yomthengisi kwakhona we-HostAfrica isihla kakhulu
- Entsha: Kongezwe ukudibanisa ngokuzenzekelayo iidomeyini ezisandul’ ukubhaliswa kwisiza senethiwekhi
- Lungisa: Kusetyenziswe iimfuno zeenkalo zomregistrant kuphela xa kubhaliswa idomeyini entsha
- Lungisa: Izaziso zebhalansi yokubeka esweni zenziwe zakwazi ukucinywa
- Lungisa: Kuqinisekisiwe ukuba iinkcukacha zokuhlawula zomregistrant we-WooCommerce ziyagcinwa
- Lungisa: Kunyanzeliswe iimfuno zoqhagamshelwano zomregistrant ngexesha lobhaliso
- Lungisa: Kuthintelwe iimveliso zobhaliso lweedomeyini ukuba zingadalwa nge-0% markup
- Lungisa: Kugcinwe ukhetho lweedomeyini kunye namaxabiso kuyo yonke inkqubo yeseshoni yokuhlawula
- Lungisa: Kuphuculwe ukuboniswa kwemali yamaxabiso eedomeyini ze-HostAfrica
- Lungisa: Kuphuculwe ukuziphatha kwe-form-action yokuhlawula ukuthintela ukungahambelani kwe-query-var ye-WP-core
- Kuphuculwe: Kudityaniswe amaxwebhu oqwalaselo lomthengisi kwakhona we-HostAfrica kwisikhokelo sokuseta

Version 1.2.0 - Ikhutshwe nge-2026-05-25
- Entsha: Kongezwe i-HostAfrica njengodibaniso lokuthengisa iidomeyini olunenkxaso yokuhlawula, iwizadi yokuseta, ukukhangela, TLD/amaxabiso, ubhaliso, uhlaziyo, ukudlulisela, nameserver, DNS, EPP code, registrar lock, kunye nokhuseleko lwe-ID

- Kuphuculwe: Kuhlaziywe imetadata yokuhambelana kwe-plugin ukuya ku-WordPress 7.0 kwi-readme ye-addon
- Kuphuculwe: Kuhlaziywe iitemplate zocwangciso ezisetyenziselwa ulungelelwaniso lokukhutshwa okuzayo
