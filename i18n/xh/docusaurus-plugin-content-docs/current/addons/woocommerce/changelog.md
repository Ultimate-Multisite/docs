# Ingxelo yotshintsho yoDityaniso lwe-WooCommerce

Inguqulelo 2.2.0 - Ikhutshwe ngo-2026-07-01
* Entsha: Izixa zerhafu ze-Ultimate Multisite ngoku zivela njengemigca eyahlukeneyo yeentlawulo ze-WooCommerce kwi-checkout, nto leyo eyenza iitotali zerhafu zicace ngakumbi phambi kwentlawulo.
* Entsha: Kongezwe useto lokuzikhethela "Misa Ubulungu Ngoko Nangoko xa Uhlaziyo Lusilele" kunye nesihluzi `wu_woo_suspend_on_payment_failure` kwiisayithi ezifuna ukumiswa ngenxa yokusilela kohlaziyo ngexesha lefestile yokuzama kwakhona ye-WooCommerce Subscriptions.
* Ulungiso: Kulungelelaniswe iimeko ze-WooCommerce Subscription ezinokuhlala zingahambelani nobulungu be-Ultimate Multisite emva kohlaziyo olusileleyo okanye olubuyiselweyo.
* Ulungiso: Kongezwe imali esetyenziswa yivenkile ye-WooCommerce kuluhlu lwemali lwe-Ultimate Multisite xa ibingekho.
* Ulungiso: Kugcinwe iinkcukacha zokubhatala zomthengi xa kuthunyelwa ababhalisi kwi-checkout ye-WooCommerce.
* Kuphuculwe: Kongezwe ukuhambelana ne-Jetpack Autoloader 5.
* Kuphuculwe: Kucocwe ukuveliswa kwephakheji yokukhutshwa ukuze ii-zip ze-GitHub kunye neemarike ziphephe iifolda zangaphakathi zokulungiselela kunye neefayile zophuhliso.
