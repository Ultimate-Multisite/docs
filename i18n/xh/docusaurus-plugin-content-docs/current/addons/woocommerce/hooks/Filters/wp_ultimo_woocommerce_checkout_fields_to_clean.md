# Isihluzi: wp_ultimo_woocommerce_checkout_fields_to_clean {#filter-wpultimowoocommercecheckoutfieldstoclean}

Ivumela abaphuhlisi ukuba bahluze iindawo zedilesi yokubhatala ezisisiseko zeWooCommerce.

Kuba Ultimate Multisite ineendawo zedilesi yokubhatala, kunokuba nengqiqo ukususa iindawo ezisisiseko eziboniswa nguWooCommerce kwifomu ye-checkout. Ngokungagqibekanga, akukho xabiso lidluliselwa kolu luhlu, ngoko zonke iindawo eziqwalaselwe kwiWooCommerce ukuba ziboniswe ziyagcinwa.

- 2.0.0
