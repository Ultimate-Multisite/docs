# Mailster {#mailster}

Guqula inethiwekhi yakho eneesayithi ezininzi ibe ngumatshini onamandla wokwakha uluhlu lwe-imeyile ngesongezo soDibaniso lwe-Ultimate Multisite Mailster. Yongeza ngokuzenzekelayo abathengi abatsha kuluhlu lwakho lwe-imeyile lwe-Mailster xa bebhalisa okanye bethenga, udale unxibelelwano olungenamthungo phakathi kweqonga lakho lobulungu nesicwangciso sakho sentengiso nge-imeyile.

## Iimpawu Ezibalulekileyo {#key-features}

**Ungqamaniso Oluzenzekelayo Lwababhalisi**
* Yongeza ngokuzenzekelayo abathengi kuluhlu lwe-Mailster xa bebhalisa
* Khetha ukungqamanisa kwangoko ekubhaliseni okanye ulinde ukuqinisekiswa kwentlawulo
* Gcina uluhlu lwakho lwe-imeyile lutsha kwaye luhlaziyiwe ngaphandle kongenelelo ngesandla

**Ulawulo Loluhlu Oluguquguqukayo**
* Seta uluhlu olungagqibekanga jikelele kubo bonke abathengi abatsha
* Gqitha okungagqibekanga ngoluhlu olukhethekileyo kwimveliso kumaphulo ajoliswe kubantu abathile
* Lawula ngokulula ukunikezelwa koluhlu ngojongano oluqhelekileyo lwe-Mailster

**Iinketho Zokuvuma Ukungena Ezihambelana ne-GDPR**
* Khetha phakathi kobhaliso oluzenzekelayo okanye imvume esekelwe kwibhokisi yokukhangela
* Yenza i-double opt-in isebenze kubhaliso oluqinisekisiweyo
* Bonisa/fihla ibhokisi yokukhangela ye-opt-in ngokusekelwe kwiimfuno zakho zokuthobela

**Ukumataniswa Kwedatha Okukrelekrele**
* Ngokuzenzekelayo ngqamanisa amagama abathengi nolwazi loqhagamshelwano
* Matanisa iidilesi zentlawulo neendawo zababhalisi
* Landelela imihla yokubhalisa kumaphulo ajoliswe kubantu abathile

**Ulawulo Kwinqanaba Lemveliso**
* Nika iimveliso ezahlukeneyo uluhlu olwahlukeneyo
* Yenza amaphulo ajoliswe kubantu abathile ngokusekelwe koko kuthengwa ngabathengi
* Khubaza i-Mailster kwiimveliso ezithile xa kufuneka

**Amava Obungcali**
* Udibaniso lwendalo lwe-Mailster UI lokukhetha uluhlu
* Ukuloga okuneenkcukacha ukusombulula iingxaki nokuqinisekisa
* Uhlaziyo olungenamthungo lwababhalisi abasele bekhona

## Ilungele {#perfect-for}

* Abanini besayithi yobulungu abakha uluhlu lwe-imeyile
* Amaqonga e-SaaS afuna ukukhulisa ubudlelwane nabathengi abatsha
* Abadali beekhosi abahlula abafundi ngokobhaliso
* Nabani na osebenzisa Ultimate Multisite kunye ne-Mailster

## Isebenza Njani {#how-it-works}

1. Faka kwaye wenze isongezo sisebenze kwinethiwekhi yakho ye-Ultimate Multisite
2. Qwalasela uluhlu olungagqibekanga jikelele kwiisetingi ze-Ultimate Multisite
3. Ngokuzikhethela gqitha ngoluhlu olukhethekileyo kwimveliso
4. Khetha imo yakho ye-opt-in nexesha lobhaliso
5. Bukela uluhlu lwakho lwe-Mailster lukhula ngokuzenzekelayo!

## Ufakelo {#installation}

1. Layisha iifayile ze-plugin kulawulo lwezongezo ze-Ultimate Multisite
2. Qinisekisa ukuba i-Mailster ifakiwe kwaye iyasebenza kwisayithi yakho ephambili
3. Yenza isongezo sisebenze ngephepha lezongezo le-Ultimate Multisite
4. Yiya ku-Settings > Mailster Integration ukuze uqwalasele
5. Khetha uluhlu lwakho olungagqibekanga kwaye uqwalasele ukhetho lwe-opt-in

## Imibuzo Ebuzwa Rhoqo {#frequently-asked-questions}

## Ngaba esi songezo sisebenza ngaphandle kwe-Ultimate Multisite? {#does-this-addon-work-without-ultimate-multisite}

Hayi, esi songezo sifuna Ultimate Multisite ukuze sisebenze. Siyilelwe ngokukodwa ukudibanisa i-Mailster nenkqubo yabathengi neemveliso ye-Ultimate Multisite.

## Ngaba kufuneka i-Mailster ifakwe kuzo zonke iisayithi? {#do-i-need-mailster-installed-on-all-sites}

Hayi, i-Mailster kufuneka ifakwe kwaye yenziwe isebenze kuphela kwisayithi yakho ephambili yenethiwekhi. Isongezo siya kuqhagamshela kuyo ngokuzenzekelayo kuzo zonke iisayithi ezingaphantsi.

## Ndinganika iimveliso ezahlukeneyo uluhlu olwahlukeneyo? {#can-i-assign-different-lists-to-different-products}

Ewe! Ungaseta uluhlu olungagqibekanga jikelele, emva koko ulugqithe kwisiseko semveliso nganye. Oku kukuvumela ukuba wenze amaphulo e-imeyile ajoliswe kakhulu ngokusekelwe koko kuthengwa ngabathengi.

## Kwenzeka ntoni ukuba umthengi sele ekhona kwi-Mailster? {#what-happens-if-a-customer-already-exists-in-mailster}

Ngokungagqibekanga, isongezo siya kuhlaziya ababhalisi abasele bekhona size sibongeze kuluhlu olutsha. Ungakukhubaza oku kwiisetingi ukuba ukhetha ukutsiba ababhalisi abasele bekhona.

## Ngaba oku kuyahambelana ne-GDPR? {#is-this-gdpr-compliant}

Ewe! Isongezo siquka imo ye-opt-in esekelwe kwibhokisi yokukhangela apho abathengi kufuneka bavume ngokucacileyo ukufumana ii-imeyile. Oku kuyahlangabezana neemfuno ze-GDPR xa kudityaniswe nemigaqo-nkqubo yabucala efanelekileyo.

## Ndingasebenzisa i-double opt-in? {#can-i-use-double-opt-in}

Ngokuqinisekileyo! Yenza isetingi ye-double opt-in isebenze kwaye i-Mailster iya kuthumela ii-imeyile zokuqinisekisa phambi kokwenza ubhaliso lusebenze.
