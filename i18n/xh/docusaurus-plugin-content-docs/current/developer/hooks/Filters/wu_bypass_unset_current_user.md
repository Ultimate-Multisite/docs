# Isihluzi: wu_bypass_unset_current_user

Vumela abaphuhlisi ukuba badlule kwikhowudi yokungaseti umsebenzisi wangoku.

Ukubuyisa nantoni na engeyiyo null kuya kudlula ukungaseti komsebenzisi wangoku ongene ngemvume. Oku kunokuba luncedo kwezinye iimeko, umzekelo, xa ujongene neesayithi ezingaphantsi ezisetyenziswa njengeepaneli zolawulo.

## Iiparamitha

| Igama | Uhlobo | Inkcazelo |
|------|------|-------------|
| $null_or_bypass | `mixed` | Null ukuze kuqhutyekwe, nantoni na enye ukuze kudlule kuyo. |
| $current_user | `false\|\WP_User` | I-objekthi yomsebenzisi wangoku. |
