# Isihluzi: wu_magic_link_enforce_ip

Hluza ukuba kuqinisekiswe na idilesi ye-IP.

Seta ku-false ukuze uvumele iithokheni zisebenze kwiinetwekhi ezahlukeneyo. Oku kunciphisa ukhuseleko kodwa kwandisa ukusetyenziswa lula (umzekelo, kubasebenzisi beefowuni eziphathwayo abatshintsha iinetwekhi).

## Iiparamitha {#parameters}

| Igama | Uhlobo | Inkcazelo |
|------|------|-------------|
| $enforce | `bool` | Ukuba kuqinisekiswe ukuhambelana kwedilesi ye-IP. |
