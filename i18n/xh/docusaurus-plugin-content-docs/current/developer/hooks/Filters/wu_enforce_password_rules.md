# Isihluzi: wu_enforce_password_rules

Sihluza ukuba kunyanzeliswe imithetho eyongezelelweyo yephasiwedi.

Xa kuyinyani, kunyanzelisa ubude obuncinane neemfuno zoonobumba. Kwenziwa ngokuzenzekelayo kwisethingi ethi "Super Strong" okanye xa uphawu lwe-Strong Password lwe-Defender Pro lusebenza.

## Iiparamitha

| Igama | Uhlobo | Inkcazelo |
|------|------|-------------|
| $enforce_rules | `bool` | Ukuba kunyanzeliswe imithetho eyongezelelweyo. |
| $strength_setting | `string` | Ixabiso lesethingi yomlawuli. |
| $defender_active | `bool` | Ukuba i-Defender Pro Strong Password iyasebenza. |
