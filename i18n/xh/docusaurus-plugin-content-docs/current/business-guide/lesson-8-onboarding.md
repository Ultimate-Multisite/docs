# Isifundo 8: Ukwamkelwa nokuqaliswa kwabathengi {#lesson-8-customer-onboarding}

Ukwenza umthengi abhalise sisiqingatha kuphela somsebenzi. Ukuba bangena, bazive begqithiselwe zizinto ezininzi, baze bangaphindi babuye, sele ubalahlekile. Esi sifundo siyila amava aguqula ukubhalisa okutsha kube ngumthengi osebenzayo nonomdla.

## Apho Besishiyeke Khona {#where-we-left-off}

IFitSite sele inophawu olupheleleyo kunye nokuhamba kwe-checkout okusebenzayo. Ngoku sigxila koko kwenzekayo emva kokuba umnini westudiyo somzimba egqibe ukubhalisa aze afike kwisiza sakhe esitsha okokuqala.

## Kutheni Ukwamkelwa nokuqaliswa Kubalulekile {#why-onboarding-matters}

Imizuzu yokuqala engama-30 emva kokubhalisa imisela ukuba umthengi uya kuhlala okanye ahambe. Umnini westudiyo somzimba othi:

- Angene aze abone isiza esele sibukeka njengewebhusayithi yezomzimba → uyahlala
- Azi ngokucacileyo into ekufuneka ayenze elandelayo → uyahlala
- Azive elahlekile kwi-WordPress dashboard eqhelekileyo → uyahamba

Iitemplate zakho ze-niche (Isifundo 4) zisingatha inqaku lokuqala. Esi sifundo sisingatha elesibini.

## Amava Okungena Kokuqala {#the-first-login-experience}

### Iwijethi yeDashboard Yokwamkela {#welcome-dashboard-widget}

Yenza iwijethi ye-dashboard eyenziwe ngokwezifiso ebulisa abathengi abatsha kwaye ibakhokele kuseto. Oku kufuneka kubonakale kakhulu xa bengena okokuqala.

**Ukuqala ngokukhawuleza kweFitSite:**

1. **Yongeza igama nestudiyo sakho kunye nelogo** -- Ikhonkco kwiCustomizer okanye kuseto lweSite Identity
2. **Hlaziya ishedyuli yeeklasi zakho** -- Ikhonkco ngokuthe ngqo kumhleli wephepha leClasses
3. **Yongeza abaqeqeshi bakho** -- Ikhonkco kumhleli wephepha leTrainers
4. **Seta iinkcukacha zakho zoqhagamshelwano** -- Ikhonkco kumhleli wephepha leContact
5. **Jonga kuqala isiza sakho** -- Ikhonkco kwicala langaphambili

Inyathelo ngalinye linxibelela ngokuthe ngqo kwiphepha okanye useto olufanelekileyo. Akukho kukhangela kwiimenyu.

### Yenza iDashboard Ibelula {#simplify-the-dashboard}

Abathengi abatsha akufuneki babone yonke into yemenyu yeWordPress. Cinga ngoku:

- Ukufihla izinto zemenyu ezingahambelani nolawulo lwesiza sezomzimba (umz., Comments ukuba azisetyenziswa)
- Ukuhlela kwakhona imenyu ukuze ubeke izinto ezisetyenziswa kakhulu kuqala
- Ukongeza iilebhile zemenyu ezenziwe ngokwezifiso ezivakalayo kwi-niche ("Istudiya Sakho" endaweni ka-"Appearance")

I-addon ye-[Plugin & Theme Manager](/addons/plugin-and-theme-manager) inokunceda ukulawula oko abathengi bakubonayo.

## Ulandelelwano Lwee-imeyile Zokwamkela {#welcome-email-sequence}

I-imeyile enye yokwamkela ayonelanga. Seta ulandelelwano olukhokelela abathengi kwiveki yabo yokuqala:

### I-imeyile 1: Wamkelekile (Ngoko nangoko emva kokubhalisa) {#email-1-welcome-immediately-after-signup}

- Umxholo: "Wamkelekile kwiFitSite -- iwebhusayithi yakho yestudiyo isebenza"
- Umxholo wangaphakathi: Ikhonkco lokungena, amanyathelo okuqala ngokukhawuleza, ikhonkco kwizixhobo zoncedo
- Isimbo: Sinomdla, siyakhuthaza, sigxile kwezomzimba

### I-imeyile 2: Iimpumelelo Ezikhawulezayo (Usuku 1) {#email-2-quick-wins-day-1}

- Umxholo: "Izinto ezi-3 zokwenza kuqala kwiFitSite yakho"
- Umxholo wangaphakathi: Yongeza ilogo yakho, hlaziya umfanekiso we-hero wekhasi lasekhaya, yongeza ishedyuli yeeklasi zakho
- Bandakanya ii-screenshot ezibonisa kanye apho kufuneka ucofe khona

### I-imeyile 3: Yenze Ibe Yeyakho (Usuku 3) {#email-3-make-it-yours-day-3}

- Umxholo: "Yenza isiza sakho sezomzimba sivelele"
- Umxholo wangaphakathi: Lungisa imibala, yongeza iifoto zabaqeqeshi, bhala ibali lestudiyo sakho
- Ikhonkco kwimizekelo yeziza ezinkulu zezomzimba eqongeni

### I-imeyile 4: Yiya Bukhoma (Usuku 7) {#email-4-go-live-day-7}

- Umxholo: "Ukulungele ukwabelana ngeFitSite yakho nehlabathi?"
- Umxholo wangaphakathi: Uluhlu lokujonga into ekufuneka uqinisekise ngaphambi kokwabelana, indlela yokudibanisa idomain eyenziwe ngokwezifiso (ukuba uku-Growth/Pro), iingcebiso zokwabelana kumajelo onxibelelwano

:::tip Email Automation
Sebenzisa [Webhooks](/user-guide/integrations/webhooks) okanye [Zapier](/user-guide/integrations/zapier) ukuqalisa ezi imeyile ngeqonga lakho lentengiso ye-imeyile. Oku kukunika ulawulo olungakumbi kwixesha kwaye kukuvumela ukuba ulandele umdla.
:::

## Izixhobo Zoncedo {#help-resources}

Yenza umxholo woncedo ogxile kwi-niche ophendula imibuzo ebuza ngokwenene ngabanini beestudiyo zomzimba:

### Amanqaku eKnowledge Base {#knowledge-base-articles}

- "Indlela yokuhlaziya ishedyuli yeeklasi zakho"
- "Ukongeza nokuhlela iiprofayile zabaqeqeshi"
- "Ukutshintsha ilogo nemibala yestudiyo sakho"
- "Ukudibanisa igama ledomain lakho" (kubathengi beGrowth/Pro)
- "Ukongeza iwijethi yokubhukisha kwisiza sakho"

Bhalele abasebenzisi abangengobuchwephesha. Sebenzisa ii-screenshot. Kuphephe ulwimi lobuchwephesha lweWordPress.

### Iividiyo Zokuhamba Ngamanyathelo {#video-walkthroughs}

Iirekhodi zesikrini ezimfutshane (imizuzu emi-2-3) ezibonisa:

- Ukungena kokuqala kunye nokuqhelaniswa
- Ukuhlela ikhasi lasekhaya
- Ukuhlaziya ishedyuli yeeklasi
- Ukongeza umqeqeshi omtsha

Ezi akufuneki zibe yimveliso ezigqibeleleyo. Okucacileyo, okuluncedo, nokugxile kwi-niche yeyona nto ibalulekileyo.

## Iphepha leAccount {#the-account-page}

Ultimate Multisite iquka [Iphepha leAccount](/user-guide/client-management/account-page) elijonge kubathengi apho abathengi balawula umrhumo wabo. Yilungise ngokwezifiso ukuze:

- Ibonise isicwangciso sabo sangoku seFitSite
- Ibonise iinketho zokunyusela phezulu kunye neenzuzo ezijolise kwezomzimba
- Ibonelele ngembali yentlawulo kunye nokukhutshelwa kwee-invoice
- Inxibelele kwizixhobo zoncedo

## Ukulinganisa Impumelelo Yokwamkelwa nokuqaliswa {#measuring-onboarding-success}

Landela ezi zalathisi ukuze wazi ukuba ukwamkelwa nokuqaliswa kwakho kuyasebenza na:

- **Izinga lokuvuselela**: Yeyiphi ipesenti yababhalisileyo abayilungisa ngokwenene isiza sabo kwiveki yokuqala?
- **Ukungena kwiveki yokuqala**: Kangaphi umthengi omtsha engena ngexesha leveki yakhe yokuqala?
- **Amatikiti enkxaso avela kubathengi abatsha**: Umthamo ophezulu uthetha ukuba ukwamkelwa nokuqaliswa kwakho kunezikhewu
- **Uguqulo ukusuka kuvavanyo ukuya kokuhlawulwayo**: Ukuba unikezela ngovavanyo, yeyiphi ipesenti eguqukayo?

```
Uthungelwano lweFitSite
├── WordPress Multisite (imowudi ye-subdomain)
├── Ultimate Multisite (ilungiselelwe + inophawu)
├── IDomain yeQonga (fitsite.com + wildcard SSL)
├── Iitemplate zeSiza (Studio Essential, Gym Pro, Fitness Chain)
├── Iimveliso (Starter, Growth, Pro + Order Bumps)
├── Ukuhamba kweCheckout (kugxile kwi-niche, kuvavanyiwe)
├── Uphawu (ukungena, dashboard, ii-imeyile, ii-invoice, isiza sentengiso)
├── Ukwamkelwa nokuqaliswa kwabathengi
│   ├── Iwijethi ye-dashboard yokuQalisa ngokukhawuleza
│   ├── Dashboard eyenziwe yalula yolawulo lwesiza sezomzimba
│   ├── Ulandelelwano lokwamkela lwee-imeyile ezi-4
│   ├── Knowledge base egxile kwi-niche
│   ├── Iividiyo zokuhamba ngamanyathelo
│   └── Iphepha le-account elenziwe ngokwezifiso
└── Ilungele isicwangciso samaxabiso (isifundo esilandelayo)
```

## Esikwakhileyo Kwesi Sifundo {#what-we-built-this-lesson}

- **Amava okungena okokuqala akhokelwayo** ngewijethi yokuQalisa ngokukhawuleza
- **Dashboard eyenziwe yalula** egxile kwimisebenzi yolawulo lwesiza sezomzimba
- **Ulandelelwano lwee-imeyile zokwamkela** olukhokelela abathengi kwiveki yabo yokuqala
- **Izixhobo zoncedo ezigxile kwi-niche** ezibhalelwe abanini beestudiyo zomzimba abangengobuchwephesha
- **Iimetriki zokwamkelwa nokuqaliswa** zokulandela nokuphucula amava
