# ਅਕਸਰ ਪੁੱਛੇ ਜਾਣ ਵਾਲੇ ਸਵਾਲ

ਇੱਥੇ ਕੁਝ **ਅਕਸਰ ਪੁੱਛੇ ਜਾਣ ਵਾਲੇ ਸਵਾਲ** ਹਨ ਜੋ ਸਹਾਇਤਾ ਬੇਨਤੀਆਂ ਦੌਰਾਨ ਅਕਸਰ ਸਾਹਮਣੇ ਆਉਂਦੇ ਹਨ।

  * **Ultimate Multisite ਕੀ ਹੈ?**

Ultimate Multisite ਇੱਕ WordPress plugin ਹੈ, ਜੋ WordPress multisite ਇੰਸਟਾਲ ਲਈ ਬਣਾਈ ਗਈ ਹੈ, ਜੋ ਤੁਹਾਡੀ WordPress ਇੰਸਟਾਲ ਨੂੰ ਸਾਈਟਾਂ ਦੇ ਇੱਕ ਪ੍ਰੀਮੀਅਮ ਨੈੱਟਵਰਕ ਵਿੱਚ ਬਦਲ ਦਿੰਦੀ ਹੈ – ਜਿਵੇਂ [WordPress.com](https://WordPress.com) – ਜਿਸ ਨਾਲ ਗਾਹਕ ਮਹੀਨਾਵਾਰ, ਤਿਮਾਹੀ, ਜਾਂ ਸਾਲਾਨਾ ਫੀਸਾਂ ਰਾਹੀਂ ਸਾਈਟਾਂ ਬਣਾ ਸਕਦੇ ਹਨ (ਤੁਸੀਂ ਮੁਫ਼ਤ ਪਲਾਨ ਵੀ ਬਣਾ ਸਕਦੇ ਹੋ)।

  * **ਕੀ Ultimate Multisite ਬਿਨਾਂ Multisite ਦੇ ਕੰਮ ਕਰੇਗਾ?**

ਨਹੀਂ। Ultimate Multisite ਲਈ WordPress multisite, subdirectory, ਜਾਂ subdomain ਜ਼ਰੂਰੀ ਹੈ।

  * **Ultimate Multisite ਲਈ ਕੀ ਲੋੜਾਂ ਹਨ?**

Ultimate Multisite ਲਈ ਲੋੜਾਂ:

**WordPress ਵਰਜ਼ਨ** : v5.3+ (ਸਿਫਾਰਸ਼ੀ: ਆਖਰੀ ਸਥਿਰ ਵਰਜ਼ਨ)  
**PHP ਵਰਜ਼ਨ** : 7.4.x (> 8.0 ਲਈ ਸਹਾਇਤਾ ਜਲਦੀ ਆ ਰਹੀ ਹੈ)  
**MySQL ਵਰਜ਼ਨ** : v5+ (ਸਿਫਾਰਸ਼ੀ: ਵਰਜ਼ਨ 5.6, ਜੇਕਰ ਤੁਹਾਡੇ ਹੋਸਟਿੰਗ ਪ੍ਰਦਾਤਾ 'ਤੇ 8.0 ਉਪਲਬਧ ਨਹੀਂ ਹੈ)

_Multisite (subdomain ਜਾਂ subdirectory) ਵੀ ਐਕਟੀਵੇਟ ਹੋਣਾ ਚਾਹੀਦਾ ਹੈ।_ [WordPress Multisite ਕਿਵੇਂ ਇੰਸਟਾਲ ਕਰੀਏ](/user-guide/getting-started/how-to-install-wordpress-multisite)।

  * **ਕੀ Ultimate Multisite Subdirectories ਵਾਲੇ Multisite ਨਾਲ ਕੰਮ ਕਰੇਗਾ?**

ਹਾਂ। Ultimate Multisite subdirectory ਇੰਸਟਾਲ ਅਤੇ subdirectory ਇੰਸਟਾਲ ਦੋਵਾਂ ਦਾ ਸਮਰਥਨ ਕਰਦਾ ਹੈ।

  * **ਕੀ Ultimate Multisite Subdomains ਵਾਲੇ Multisite ਨਾਲ ਕੰਮ ਕਰੇਗਾ?**

ਹਾਂ। Ultimate Multisite subdirectory ਇੰਸਟਾਲ ਅਤੇ subdomain ਇੰਸਟਾਲ ਦੋਵਾਂ ਦਾ ਸਮਰਥਨ ਕਰਦਾ ਹੈ।

  * **ਕੀ ਮੈਂ Ultimate Multisite ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਇਹ ਸੀਮਤ ਕਰ ਸਕਦਾ ਹਾਂ ਕਿ ਯੂਜ਼ਰ ਕਿਹੜੇ Plugins ਅਤੇ Themes ਵਰਤ ਸਕਦੇ ਹਨ?**

ਹਾਂ। Ultimate Multisite ਨਾਲ ਤੁਸੀਂ ਚੁਣ ਸਕਦੇ ਹੋ ਕਿ ਤੁਹਾਡੇ ਨੈੱਟਵਰਕ 'ਤੇ ਹਰੇਕ ਪਲਾਨ ਲਈ ਕਿਹੜੇ plugins ਅਤੇ themes ਉਪਲਬਧ ਹਨ।

  * **ਕੀ ਮੈਂ Ultimate Multisite ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਯੂਜ਼ਰ ਆਪਣੀ ਸਾਈਟ 'ਤੇ ਕਿੰਨੀਆਂ ਪੋਸਟਾਂ ਬਣਾ ਸਕਦਾ ਹੈ ਇਹ ਸੀਮਤ ਕਰ ਸਕਦਾ ਹਾਂ?**

ਹਾਂ। Ultimate Multisite ਨਾਲ ਤੁਸੀਂ ਆਪਣੇ ਨੈੱਟਵਰਕ 'ਤੇ **ਹਰ ਉਪਲਬਧ ਪੋਸਟ ਟਾਈਪ** ਲਈ ਪੋਸਟ ਕੋਟੇ ਸੈੱਟ ਕਰ ਸਕਦੇ ਹੋ। ਤੁਸੀਂ ਐਡਮਿਨ ਕਿੰਨੇ ਯੂਜ਼ਰਾਂ ਨੂੰ ਸੱਦਾ ਦੇ ਸਕਦਾ ਹੈ, **ਡਿਸਕ ਸਪੇਸ ਦੀ ਮਾਤਰਾ,** ਅਤੇ ਹੋਰ ਬਹੁਤ ਕੁਝ ਵੀ ਸੀਮਤ ਕਰ ਸਕਦੇ ਹੋ।

  * **ਕੀ Ultimate Multisite Site Templates ਦਾ ਸਮਰਥਨ ਕਰਦਾ ਹੈ?**

ਹਾਂ। Ultimate Multisite ਨਾਲ ਤੁਸੀਂ ਜਿੰਨੇ ਚਾਹੋ Site Templates ਬਣਾ ਸਕਦੇ ਹੋ। Site Templates themes ਨਹੀਂ ਹਨ, ਇਹ ਤੁਹਾਡੇ ਨੈੱਟਵਰਕ 'ਤੇ ਆਮ ਸਾਈਟਾਂ ਹਨ। ਤੁਸੀਂ plugins, themes ਐਕਟੀਵੇਟ ਕਰ ਸਕਦੇ ਹੋ, ਅਤੇ ਉਸ Site Template ਨੂੰ ਜਿਸ ਤਰੀਕੇ ਨਾਲ ਚਾਹੋ ਕਸਟਮਾਈਜ਼ ਕਰ ਸਕਦੇ ਹੋ। ਜੇਕਰ ਕੋਈ ਯੂਜ਼ਰ ਉਹ template ਚੁਣਦਾ ਹੈ, ਤਾਂ ਉਸਦੀ ਸਾਈਟ ਉਸ template ਦੀ ਕਾਪੀ ਵਜੋਂ ਬਣਾਈ ਜਾਵੇਗੀ, ਸਾਰੀਆਂ ਕਸਟਮਾਈਜ਼ੇਸ਼ਨਾਂ ਸਮੇਤ।

ਤੁਸੀਂ ਕਿਸੇ ਪਲਾਨ ਲਈ ਕੋਈ ਖਾਸ template ਵੀ ਸੈੱਟ ਕਰ ਸਕਦੇ ਹੋ। ਅਜਿਹਾ ਕਰਨ ਨਾਲ, ਉਸ ਪਲਾਨ 'ਤੇ ਸਾਈਨ ਅੱਪ ਕਰਨ ਵਾਲੇ ਹਰ ਯੂਜ਼ਰ ਦੀ ਸਾਈਟ ਉਸ Site Template ਦੀ ਡੁਪਲੀਕੇਟ ਵਜੋਂ ਬਣਾਈ ਜਾਵੇਗੀ।

  * **ਕੀ Ultimate Multisite WooCommerce ਨਾਲ ਇੰਟੀਗ੍ਰੇਟ ਕਰਦਾ ਹੈ?**

ਹਾਂ!

ਜੇਕਰ ਤੁਸੀਂ ਚਾਹੁੰਦੇ ਹੋ ਕਿ ਤੁਹਾਡੇ ਨੈੱਟਵਰਕ ਯੂਜ਼ਰਾਂ ਲਈ WooCommerce ਉਪਲਬਧ ਹੋਵੇ, ਜਿਸ ਨਾਲ ਉਹ ਸਟੋਰ ਸੈੱਟ ਕਰ ਸਕਣ, ਤਾਂ Ultimate Multisite ਨਾਲ ਇਹ ਸੰਭਵ ਹੈ। ਤੁਸੀਂ ਇਹ ਵੀ ਸੀਮਤ ਕਰ ਸਕਦੇ ਹੋ ਕਿ ਤੁਹਾਡੇ ਗਾਹਕ ਹਰੇਕ ਪਲਾਨ ਟੀਅਰ 'ਤੇ ਕਿੰਨੇ ਉਤਪਾਦ ਬਣਾ ਸਕਦੇ ਹਨ।

Ultimate Multisite ਲਈ ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਭੁਗਤਾਨਾਂ ਦੇ ਮਾਮਲੇ ਵਿੱਚ, ਤੁਸੀਂ [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) ਦੀ ਵਰਤੋਂ ਕਰ ਸਕਦੇ ਹੋ ਤਾਂ ਜੋ ਤੁਹਾਡੇ Ultimate Multisite ਗਾਹਕ ਤੁਹਾਡੀ ਮੁੱਖ ਸਾਈਟ 'ਤੇ WooCommerce gateways ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਭੁਗਤਾਨ ਕਰ ਸਕਣ। ਇਸ ਤਰੀਕੇ ਨਾਲ, ਤੁਸੀਂ WooCommerce ਲਈ ਬਣਾਏ ਗਏ payment gateways ਦੀ ਵਿਸ਼ਾਲ ਲਾਇਬ੍ਰੇਰੀ ਦਾ ਫਾਇਦਾ ਉਠਾ ਸਕਦੇ ਹੋ ਅਤੇ ਉਨ੍ਹਾਂ ਦੀ ਵਰਤੋਂ ਆਪਣੇ Ultimate Multisite ਗਾਹਕਾਂ ਨੂੰ ਬਿੱਲ ਕਰਨ ਲਈ ਕਰ ਸਕਦੇ ਹੋ।

  * **ਕੀ WPMUDEV Pro Sites ਤੋਂ ਸਬਸਕ੍ਰਿਪਸ਼ਨਾਂ ਨੂੰ ਮਾਈਗ੍ਰੇਟ ਕਰਨ ਦਾ ਕੋਈ ਤਰੀਕਾ ਹੈ?**

ਹਾਂ, Ultimate Multisite ਹੁਣ ਤੁਹਾਨੂੰ [ਤੁਹਾਡੇ Pro Sites ਨੈੱਟਵਰਕ ਨੂੰ ਮਾਈਗ੍ਰੇਟ ਕਰਨ ਲਈ ਇੱਕ ਮਦਦਗਾਰ ਟੂਲ](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) ਪੇਸ਼ ਕਰਦਾ ਹੈ।

  * **ਮੇਰੇ Plan Edit Page 'ਤੇ "Allowed Plugins" ਟੈਬ ਖਾਲੀ ਕਿਉਂ ਹੈ?**

Ultimate Multisite ਤੁਹਾਨੂੰ ਸਿਰਫ਼ ਉਨ੍ਹਾਂ plugins ਤੱਕ ਪਹੁੰਚ ਦੀ ਇਜਾਜ਼ਤ ਜਾਂ ਇਨਕਾਰ ਕਰਨ ਦਿੰਦਾ ਹੈ ਜੋ ਤੁਹਾਡੇ ਨੈੱਟਵਰਕ 'ਤੇ network-activated ਨਹੀਂ ਹਨ। ਜਦੋਂ ਕੋਈ plugin network-activated ਹੁੰਦੀ ਹੈ, ਤਾਂ ਆਪਣੀ ਪ੍ਰਕਿਰਤੀ ਮੁਤਾਬਕ, ਇਹ ਤੁਹਾਡੇ ਨੈੱਟਵਰਕ ਦੀ ਹਰ ਸਾਈਟ 'ਤੇ ਡਿਫਾਲਟ ਤੌਰ 'ਤੇ ਲੋਡ ਹੋਵੇਗੀ।

ਜੇਕਰ ਤੁਹਾਨੂੰ "Allowed Plugins" ਟੈਬ 'ਤੇ ਕੋਈ plugins ਨਹੀਂ ਦਿਖਾਈ ਦਿੰਦੀਆਂ, ਤਾਂ ਚੈੱਕ ਕਰੋ ਕਿ ਲੋੜੀਂਦੀ plugin network-active ਹੈ ਜਾਂ ਨਹੀਂ।

  * **ਕੀ ਮੈਂ ਆਪਣੀ Pricing Table ਦਾ ਡਿਜ਼ਾਈਨ ਵਰਤ ਸਕਦਾ ਹਾਂ?**

ਹਾਂ। ਵਰਜ਼ਨ 1.4.0 ਤੋਂ, Ultimate Multisite template overrides ਦਾ ਸਮਰਥਨ ਕਰਦਾ ਹੈ।

  * **ਕੀ Ultimate Multisite WP Engine ਨਾਲ ਕੰਮ ਕਰਦਾ ਹੈ?**

ਹਾਂ। Ultimate Multisite ਨੂੰ WP Engine ਨੇ ਹਮੇਸ਼ਾ ਸਮਰਥਨ ਦਿੱਤਾ ਹੈ, ਪਰ domain mapping ਨੂੰ ਕੰਮ ਕਰਵਾਉਣ ਲਈ ਐਡਮਿਨ ਤੋਂ ਵਾਧੂ ਮੈਨੂਅਲ ਕਦਮਾਂ ਦੀ ਲੋੜ ਸੀ (WP Engine ਪੈਨਲ ਵਿੱਚ domain ਜੋੜਨਾ)। **ਵਰਜ਼ਨ 1.6.0 ਤੋਂ ਹੁਣ ਅਜਿਹਾ ਨਹੀਂ ਹੈ।** Ultimate Multisite ਸਿੱਧੇ WP Engine API ਨਾਲ ਇੰਟੀਗ੍ਰੇਟ ਕਰਦਾ ਹੈ ਅਤੇ mapping ਤੋਂ ਬਾਅਦ domains ਆਪਣੇ ਆਪ ਪੈਨਲ ਵਿੱਚ ਜੁੜ ਜਾਂਦੇ ਹਨ।

  * **ਤੁਸੀਂ ਕਿਹੜੇ ਹੋਸਟਿੰਗ ਪ੍ਰਦਾਤਾ ਦੀ ਸਿਫਾਰਸ਼ ਕਰਦੇ ਹੋ?**

Ultimate Multisite ਕਿਸੇ ਵੀ ਹੋਸਟਿੰਗ ਪ੍ਰਦਾਤਾ ਨਾਲ ਕੰਮ ਕਰਨਾ ਚਾਹੀਦਾ ਹੈ। ਫਰਕ ਸਿਰਫ਼ ਇਹ ਹੈ ਕਿ ਕੁਝ ਹੋਸਟਿੰਗ ਕੰਪਨੀਆਂ ਜਿਵੇਂ Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane ਅਤੇ [Runcloud.io](http://Runcloud.io) ਕੋਲ domain mapping ਸਮਰੱਥਾਵਾਂ ਲਈ ਵਧੇਰੇ ਮਜ਼ਬੂਤ ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਹੈ ਇਸ ਲਈ ਉਹ ਤੁਰੰਤ ਕੰਮ ਕਰਦੇ ਹਨ। ਹੋਰ ਹੋਸਟਿੰਗਾਂ ਲਈ, ਤੁਹਾਨੂੰ domain mapping ਮੈਨੂਅਲੀ ਸੈੱਟ ਕਰਨ ਦੀ ਲੋੜ ਹੋ ਸਕਦੀ ਹੈ।
