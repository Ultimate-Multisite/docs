# Ultimate Multisite: Ad Injector ਨਾਲ ਆਪਣੇ ਨੈੱਟਵਰਕ 'ਤੇ ਵਿਗਿਆਪਨ ਲਗਾਉਣਾ

Ultimate Multisite: Ad Injector ਇੱਕ **ਮੁਫ਼ਤ add-on** ਹੈ ਜੋ ਤੁਹਾਨੂੰ ਆਪਣੇ ਪਲਾਨਾਂ 'ਤੇ ਵਿਗਿਆਪਨ ਲਗਾਉਣ ਦੇ ਸਾਧਨ ਦਿੰਦਾ ਹੈ, ਸਿੱਧੇ ਉਹਨਾਂ ਦੇ ਸੋਧ ਪੰਨੇ ਤੋਂ।

Ultimate Multisite: Ad Injector ਇੰਸਟਾਲ ਕਰਨ ਤੋਂ ਬਾਅਦ, ਇਸਨੂੰ ਐਕਟੀਵੇਟ ਕਰਨ ਲਈ ਆਪਣੇ _Plugins_ ਪੰਨੇ 'ਤੇ ਜਾਓ।

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Network Activate ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ_

ਜਦੋਂ add-on ਐਕਟਿਵ ਹੋ ਜਾਵੇ, ਤਾਂ ਪਲਾਨ ਸੋਧ ਸਕ੍ਰੀਨ 'ਤੇ _Advanced Options_ ਮੈਟਾ ਬਾਕਸ ਵਿੱਚ _Ad Settings_ ਨਾਂ ਦਾ ਇੱਕ ਨਵਾਂ ਵਿਕਲਪ ਹੋਵੇਗਾ।

![ਪਲਾਨ ਸੋਧ ਸਕ੍ਰੀਨ 'ਤੇ Ad Settings](/img/config/product-edit-full.png)

_ਹੁਣ ਤੁਸੀਂ ਆਪਣੇ ਹਰੇਕ ਪਲਾਨ ਲਈ ਵਿਗਿਆਪਨ ਸੈਟਿੰਗਾਂ ਨੂੰ ਸੋਧ ਸਕਦੇ ਹੋ_

ਉਸ ਟੈਬ 'ਤੇ, ਤੁਸੀਂ ਫਰੰਟ-ਐਂਡ ਵਿੱਚ ਪੋਸਟਾਂ ਅਤੇ ਪੰਨਿਆਂ ਲਈ, ਸਮੱਗਰੀ ਤੋਂ ਪਹਿਲਾਂ ਅਤੇ ਬਾਅਦ ਵਿੱਚ ਵਿਗਿਆਪਨ ਕੋਡ ਜੋੜ ਸਕੋਗੇ।
