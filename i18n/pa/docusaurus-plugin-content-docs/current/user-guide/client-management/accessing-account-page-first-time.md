# ਮੈਂ ਪਹਿਲੀ ਵਾਰ ਆਪਣੇ ਖਾਤੇ ਦੇ ਪੇਜ ਨੂੰ ਕਿਵੇਂ ਐਕਸੈਸ ਕਰ ਸਕਦਾ/ਸਕਦੀ ਹਾਂ?

Ultimate Multisite ਵਿੱਚ ਤੁਹਾਡਾ ਸੁਆਗਤ ਹੈ!

ਸਾਡਾ plugin ਖਰੀਦਣ ਤੋਂ ਬਾਅਦ ਤੁਹਾਨੂੰ ਈਮੇਲ ਵਿੱਚ ਜੋ ਲਿੰਕ ਮਿਲਿਆ ਸੀ, ਉਸ ਰਾਹੀਂ ਆਪਣੇ ਖਾਤੇ ਦੇ ਪੇਜ ਤੱਕ ਪਹੁੰਚੋ:

<!-- Screenshot unavailable: Account page email with access link -->

ਖਰੀਦਾਰੀ ਸਮੇਂ ਦਿੱਤੀ ਗਈ ਈਮੇਲ ਅਤੇ ਈਮੇਲ ਵਿੱਚ ਮਿਲਿਆ ਪਾਸਵਰਡ ਵਰਤੋ।

<!-- Screenshot unavailable: Login page with email and password fields -->

ਕਿਉਂਕਿ ਤੁਸੀਂ ਇਸ ਖਾਤੇ ਵਿੱਚ ਪਹਿਲੀ ਵਾਰ ਲੌਗਇਨ ਕਰ ਰਹੇ ਹੋ, ਤੁਹਾਨੂੰ ਆਪਣਾ ਪਾਸਵਰਡ ਬਦਲਣ ਲਈ ਕਿਹਾ ਜਾਵੇਗਾ। ਇਸਨੂੰ ਬਦਲੋ ਅਤੇ _Save_ ਬਟਨ ਤੇ ਕਲਿੱਕ ਕਰੋ।

<!-- Screenshot unavailable: Change password form with Save button -->

ਆਪਣੇ ਖਾਤੇ ਦੇ ਪੇਜ ਤੇ, ਤੁਸੀਂ ਆਪਣੇ ਖਾਤੇ ਦੀਆਂ ਸਾਰੀਆਂ ਜਾਣਕਾਰੀਆਂ ਦਾ ਪ੍ਰਬੰਧਨ ਕਰ ਸਕਦੇ ਹੋ ਜਿਵੇਂ ਕਿ ਬਿਲਿੰਗ, ਰੀਨਿਊਅਲ, ਕੈਂਸਲੇਸ਼ਨ, ਅਤੇ ਹੋਰ ਬਹੁਤ ਕੁਝ।
