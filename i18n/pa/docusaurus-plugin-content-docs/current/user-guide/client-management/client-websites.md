# ਤੁਹਾਡੇ ਗਾਹਕਾਂ ਦੀਆਂ ਵੈੱਬਸਾਈਟਾਂ (v2)

ਤੁਹਾਡੇ ਨੈੱਟਵਰਕ 'ਤੇ ਰਜਿਸਟਰ ਹੋਣ ਤੋਂ ਬਾਅਦ, ਤੁਹਾਡੇ ਗਾਹਕ ਆਪਣੇ dashboard ਤੱਕ ਪਹੁੰਚ ਕਰ ਸਕਦੇ ਹਨ।

**Account** ਪੇਜ ਤੋਂ ਇਲਾਵਾ, ਜਿੱਥੇ ਉਹ ਆਪਣੀਆਂ ਪੇਮੈਂਟਾਂ, ਮੈਂਬਰਸ਼ਿਪਾਂ, ਡੋਮੇਨਾਂ, ਪਲਾਨ ਦੀਆਂ ਸੀਮਾਵਾਂ ਅਤੇ ਹੋਰ ਬਹੁਤ ਕੁਝ ਬਾਰੇ ਮਹੱਤਵਪੂਰਨ ਜਾਣਕਾਰੀ ਦੇਖ ਸਕਦੇ ਹਨ, ਉਹਨਾਂ ਕੋਲ **My Sites** ਪੇਜ ਤੱਕ ਵੀ ਪਹੁੰਚ ਹੈ, ਜਿੱਥੇ ਉਹ ਆਪਣੀਆਂ ਸਾਰੀਆਂ ਵੈੱਬਸਾਈਟਾਂ ਦੇਖ ਅਤੇ ਸੰਭਾਲ ਸਕਦੇ ਹਨ। ਉਹਨਾਂ ਨੂੰ ਬੱਸ ਆਪਣੇ dashboard 'ਤੇ My Sites 'ਤੇ ਕਲਿੱਕ ਕਰਨਾ ਹੈ।

![ਗਾਹਕ dashboard ਵਿੱਚ My Sites ਲਿੰਕ](/img/admin/customers-list.png)

ਉਹਨਾਂ ਨੂੰ ਆਪਣੀਆਂ ਸਾਰੀਆਂ ਵੈੱਬਸਾਈਟਾਂ ਦਿਖਾਈਆਂ ਜਾਣਗੀਆਂ, ਹਰ ਇੱਕ ਦੇ ਹੇਠਾਂ **Manage** ਬਟਨ ਹੋਵੇਗਾ। ਜੇਕਰ ਉਹ ਇਸ 'ਤੇ ਕਲਿੱਕ ਕਰਦੇ ਹਨ, ਤਾਂ ਉਹਨਾਂ ਨੂੰ **ਉਸ ਖਾਸ ਵੈੱਬਸਾਈਟ ਦੇ dashboard** 'ਤੇ ਭੇਜ ਦਿੱਤਾ ਜਾਵੇਗਾ।

ਉਹ ਇਹ ਵੀ ਦੇਖ ਸਕਦੇ ਹਨ ਕਿ ਉਹਨਾਂ ਦੀਆਂ ਵੈੱਬਸਾਈਟਾਂ ਵਿੱਚੋਂ ਕਿਹੜੀ **ਮੁੱਖ** ਹੈ, ਉਹਨਾਂ ਪਲਾਨਾਂ ਲਈ ਜੋ ਉਹਨਾਂ ਨੂੰ ਇੱਕ ਤੋਂ ਵੱਧ ਵੈੱਬਸਾਈਟਾਂ ਰੱਖਣ ਦੀ ਇਜਾਜ਼ਤ ਦਿੰਦੇ ਹਨ।

![My Sites ਪੇਜ ਜੋ ਗਾਹਕ ਦੀਆਂ ਵੈੱਬਸਾਈਟਾਂ Manage ਬਟਨਾਂ ਨਾਲ ਦਿਖਾਉਂਦਾ ਹੈ](/img/admin/customers-list.png)

ਜੇਕਰ ਉਹ ਆਪਣੀ ਮੁੱਖ ਵੈੱਬਸਾਈਟ ਬਦਲਣਾ ਚਾਹੁੰਦੇ ਹਨ, ਤਾਂ ਉਹਨਾਂ ਨੂੰ **Account** ਮੀਨੂ 'ਤੇ ਕਲਿੱਕ ਕਰਕੇ Account ਪੇਜ 'ਤੇ ਜਾਣਾ ਹੋਵੇਗਾ।

![ਗਾਹਕ dashboard ਵਿੱਚ Account ਮੀਨੂ ਲਿੰਕ](/img/admin/customers-list.png)

ਫਿਰ, ਉਹਨਾਂ ਨੂੰ **Change Default Site** 'ਤੇ ਕਲਿੱਕ ਕਰਨਾ ਚਾਹੀਦਾ ਹੈ।

![Account ਪੇਜ 'ਤੇ Change Default Site ਬਟਨ](/img/admin/customers-list.png)

ਇੱਕ ਪੌਪ-ਅੱਪ ਵਿੰਡੋ ਦਿਖਾਈ ਦੇਵੇਗੀ। ਡ੍ਰੌਪ-ਡਾਊਨ ਲਿਸਟ 'ਤੇ ਕਲਿੱਕ ਕਰੋ ਅਤੇ ਚੁਣੋ ਕਿ ਤੁਸੀਂ ਕਿਹੜੀ ਵੈੱਬਸਾਈਟ ਨੂੰ ਮੁੱਖ ਵੈੱਬਸਾਈਟ ਬਣਾਉਣਾ ਚਾਹੁੰਦੇ ਹੋ।

![Change Default Site ਡ੍ਰੌਪਡਾਊਨ ਚੋਣ](/img/admin/customers-list.png)

ਫਿਰ, ਬੱਸ **Change Default Site** 'ਤੇ ਕਲਿੱਕ ਕਰੋ ਅਤੇ ਹੋ ਗਿਆ!

![ਮੁੱਖ ਸਾਈਟ ਬਦਲਣ ਦੀ ਪੁਸ਼ਟੀ](/img/admin/customers-list.png)
