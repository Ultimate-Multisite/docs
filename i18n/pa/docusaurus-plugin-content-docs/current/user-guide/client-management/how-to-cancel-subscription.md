# ਮੈਂ ਆਪਣੀ ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਕਿਵੇਂ ਰੱਦ ਕਰ ਸਕਦਾ/ਸਕਦੀ ਹਾਂ?

ਜੇਕਰ ਤੁਹਾਡੇ ਕੋਲ Ultimate Multisite ਲਾਇਸੈਂਸ ਹੈ, ਤਾਂ ਤੁਸੀਂ ਕਿਸੇ ਵੀ ਸਮੇਂ ਇਸਦੀ ਰੀਨਿਊਅਲ ਰੱਦ ਕਰ ਸਕਦੇ ਹੋ। ਬੱਸ ਹੇਠਾਂ ਦਿੱਤੇ ਕਦਮਾਂ ਦੀ ਪਾਲਣਾ ਕਰੋ:


<!-- Screenshot unavailable: Account page email with access link -->

ਖਰੀਦਦਾਰੀ ਸਮੇਂ ਦਿੱਤੀ ਗਈ ਈਮੇਲ ਅਤੇ ਆਪਣਾ ਪਾਸਵਰਡ ਵਰਤੋ।

<!-- Screenshot unavailable: Login page with email and password fields -->

ਇਹ ਤੁਹਾਡੇ Account Page ਦੀ ਸ਼ੁਰੂਆਤੀ ਸਕ੍ਰੀਨ ਹੈ:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

ਆਪਣੀ ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਰੱਦ ਕਰਨ ਲਈ, ਖੱਬੇ ਪਾਸੇ ਦੇ ਮੀਨੂ ਵਿੱਚ _Renewing & Billing_ 'ਤੇ ਜਾਓ।

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

ਇੱਕ ਸਾਈਡ ਵਿੰਡੋ ਖੋਲ੍ਹਣ ਲਈ ਸੱਜੇ ਪਾਸੇ ਦੇ ਤੀਰ 'ਤੇ ਕਲਿੱਕ ਕਰੋ। ਫਿਰ, ਤੁਹਾਨੂੰ ਉਹ ਵਿਕਲਪ ਚੁਣਨਾ ਚਾਹੀਦਾ ਹੈ ਜੋ **auto-renew** ਨੂੰ ਰੱਦ ਕਰੇਗਾ।

ਸਿਸਟਮ ਤੁਹਾਨੂੰ ਇੱਕ ਪੁਸ਼ਟੀ ਸੁਨੇਹਾ ਦਿਖਾਏਗਾ।

ਜੇਕਰ ਤੁਸੀਂ ਯਕੀਨੀ ਹੋ ਕਿ ਤੁਸੀਂ ਆਪਣੀ ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਰੱਦ ਕਰਨਾ ਚਾਹੁੰਦੇ ਹੋ, ਤਾਂ ਬੱਸ _**Cancel Renewals**_ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ। ਇਸ ਕਾਰਵਾਈ ਤੋਂ ਬਾਅਦ, ਤੁਹਾਨੂੰ ਇੱਕ ਛੋਟਾ ਜਿਹਾ ਸਰਵੇਖਣ ਭਰਨ ਲਈ ਕਿਹਾ ਜਾਵੇਗਾ।

ਹੋ ਗਿਆ! ਤੁਹਾਡੀ ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਆਪਣੇ ਆਪ ਰੀਨਿਊ ਨਹੀਂ ਹੋਵੇਗੀ।

**ਤੁਹਾਡੇ ਕੋਲ ਤੁਹਾਡੀ ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਖਤਮ ਹੋਣ ਤੱਕ ਇੱਕ ਵੈਧ ਕੀ ਹੋਵੇਗੀ**। ਜੇਕਰ ਤੁਸੀਂ ਆਪਣੀ ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਦੁਬਾਰਾ ਸ਼ੁਰੂ ਕਰਨਾ ਚਾਹੁੰਦੇ ਹੋ, ਤਾਂ ਤੁਹਾਨੂੰ ਇਹ ਹੱਥੀਂ ਕਰਨਾ ਪਵੇਗਾ।
