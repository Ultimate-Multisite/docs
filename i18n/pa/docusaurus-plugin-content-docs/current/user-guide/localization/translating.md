# Ultimate Multisite ਦਾ ਅਨੁਵਾਦ ਕਰਨਾ

Ultimate Multisite ਅੰਗਰੇਜ਼ੀ ਵਿੱਚ ਬਣਾਈ ਗਈ ਹੈ, ਪਰ ਇਸ plugin ਦਾ ਪੂਰੀ ਤਰ੍ਹਾਂ ਅਨੁਵਾਦ ਕੀਤਾ ਜਾ ਸਕਦਾ ਹੈ ਅਤੇ ਅਸੀਂ ਅਨੁਵਾਦ ਯੋਗਦਾਨਾਂ ਦਾ ਦਿਲੋਂ ਸਵਾਗਤ ਕਰਦੇ ਹਾਂ।

## ਮੁੱਖ Plugin ਅਨੁਵਾਦ

ਮੁੱਖ Ultimate Multisite plugin ਦਾ ਅਨੁਵਾਦ ਅਧਿਕਾਰਤ **WordPress.org ਅਨੁਵਾਦ ਪਲੇਟਫਾਰਮ** ਰਾਹੀਂ ਕੀਤਾ ਜਾਂਦਾ ਹੈ:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

ਇਹ ਮਿਆਰੀ WordPress ਕਮਿਊਨਿਟੀ ਅਨੁਵਾਦ ਸਿਸਟਮ ਹੈ। ਜਦੋਂ ਅਨੁਵਾਦ ਲੋੜੀਂਦੀ ਮਨਜ਼ੂਰੀ ਸੀਮਾ ਤੱਕ ਪਹੁੰਚ ਜਾਂਦੇ ਹਨ, ਤਾਂ ਉਹ WordPress plugin ਅੱਪਡੇਟਾਂ ਰਾਹੀਂ ਸਾਰੇ Ultimate Multisite ਵਰਤੋਂਕਾਰਾਂ ਨੂੰ ਆਪਣੇ ਆਪ ਭੇਜ ਦਿੱਤੇ ਜਾਂਦੇ ਹਨ।

**ਸਾਨੂੰ ਤੁਹਾਡੀ ਮਦਦ ਦੀ ਲੋੜ ਹੈ!** ਜੇ ਤੁਸੀਂ ਅੰਗਰੇਜ਼ੀ ਤੋਂ ਇਲਾਵਾ ਕੋਈ ਹੋਰ ਭਾਸ਼ਾ ਬੋਲਦੇ ਹੋ, ਤਾਂ ਕਿਰਪਾ ਕਰਕੇ ਅਨੁਵਾਦ ਵਿੱਚ ਯੋਗਦਾਨ ਪਾਉਣ ਬਾਰੇ ਸੋਚੋ। ਕੁਝ strings ਦਾ ਅਨੁਵਾਦ ਕਰਨਾ ਵੀ ਮਦਦਗਾਰ ਹੈ। ਸ਼ੁਰੂ ਕਰਨ ਲਈ:

1. [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/) 'ਤੇ ਜਾਓ
2. ਆਪਣੀ ਭਾਸ਼ਾ ਚੁਣੋ
3. "Contribute Translation" 'ਤੇ ਕਲਿੱਕ ਕਰੋ
4. ਤੁਹਾਨੂੰ ਮੁਫ਼ਤ WordPress.org ਖਾਤੇ ਦੀ ਲੋੜ ਹੋਵੇਗੀ

ਜੇ ਤੁਸੀਂ ਆਪਣੀ ਭਾਸ਼ਾ ਲਈ ਅਨੁਵਾਦ ਸੰਪਾਦਕ ਬਣਨਾ ਚਾਹੁੰਦੇ ਹੋ (ਅਨੁਵਾਦਾਂ ਨੂੰ ਮਨਜ਼ੂਰ ਕਰਨ ਦੇ ਯੋਗ), ਤਾਂ [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) ਰਾਹੀਂ ਸਾਡੇ ਨਾਲ ਸੰਪਰਕ ਕਰੋ।

## Addon ਅਨੁਵਾਦ

Addon plugins ਦਾ ਵੱਖਰੇ ਤੌਰ 'ਤੇ ਇੱਥੇ ਅਨੁਵਾਦ ਕੀਤਾ ਜਾਂਦਾ ਹੈ:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Addon ਅਨੁਵਾਦ ਇਸ ਵੇਲੇ ਸ਼ੁਰੂਆਤੀ ਬਿੰਦੂ ਵਜੋਂ AI ਦੁਆਰਾ ਤਿਆਰ ਕੀਤੇ ਗਏ ਹਨ। ਇਹ ਸਾਰੀਆਂ ਸਹਾਇਕ ਭਾਸ਼ਾਵਾਂ ਨੂੰ ਕਵਰ ਕਰਦੇ ਹਨ ਪਰ ਇਹਨਾਂ ਵਿੱਚ ਅਜੀਬ ਵਾਕ ਬਣਤਰ ਜਾਂ ਗਲਤੀਆਂ ਹੋ ਸਕਦੀਆਂ ਹਨ। ਅਸੀਂ ਮਨੁੱਖੀ ਅਨੁਵਾਦਕਾਂ ਦਾ ਸਵਾਗਤ ਕਰਦੇ ਹਾਂ ਜੋ ਇਹਨਾਂ ਨੂੰ ਹੋਰ ਕੁਦਰਤੀ ਅਤੇ ਸਹੀ ਬਣਾਉਣ ਲਈ ਸਮੀਖਿਆ ਕਰਨ ਅਤੇ ਸੁਧਾਰ ਸੁਝਾਉਣ।

## ਸਥਾਨਕ ਅਨੁਵਾਦ

ਜੇ ਤੁਸੀਂ ਮੁੱਖ ਪ੍ਰੋਜੈਕਟ ਵਿੱਚ ਯੋਗਦਾਨ ਪਾਉਣ ਦੀ ਬਜਾਏ ਆਪਣੇ ਪ੍ਰੋਜੈਕਟ ਲਈ ਸਥਾਨਕ ਤੌਰ 'ਤੇ ਅਨੁਵਾਦ ਬਣਾਉਣਾ ਚਾਹੁੰਦੇ ਹੋ, ਤਾਂ ਦੋ ਵਿਕਲਪ ਹਨ:

### Poedit ਦੀ ਵਰਤੋਂ ਕਰਨਾ

ਤੁਸੀਂ `.pot` ਫਾਈਲ ਦਾ ਅਨੁਵਾਦ ਕਰਨ ਲਈ ਡੈਸਕਟਾਪ ਐਪਲੀਕੇਸ਼ਨ **[Poedit](https://poedit.net)** ਦੀ ਵਰਤੋਂ ਕਰ ਸਕਦੇ ਹੋ:

1. [poedit.net](https://poedit.net) ਤੋਂ Poedit ਡਾਊਨਲੋਡ ਕਰੋ
2. `wp-content/plugins/ultimate-multisite/lang/` ਤੋਂ `.pot` ਫਾਈਲ ਖੋਲ੍ਹੋ
3. Poedit ਇੰਟਰਫੇਸ ਵਿੱਚ strings ਦਾ ਅਨੁਵਾਦ ਕਰੋ
4. ਸੇਵ ਕਰੋ -- Poedit `.po` ਅਤੇ `.mo` ਫਾਈਲਾਂ ਬਣਾਉਂਦਾ ਹੈ
5. ਆਉਟਪੁੱਟ ਫਾਈਲਾਂ ਨੂੰ `lang` ਫੋਲਡਰ ਵਿੱਚ ਰੱਖੋ

### Loco Translate ਦੀ ਵਰਤੋਂ ਕਰਨਾ

**[Loco Translate](https://wordpress.org/plugins/loco-translate/)** WordPress plugin ਤੁਹਾਨੂੰ ਬਿਨਾਂ ਕਿਸੇ ਡੈਸਕਟਾਪ ਟੂਲ ਦੇ ਸਿੱਧੇ admin panel ਤੋਂ ਅਨੁਵਾਦ ਕਰਨ ਦਿੰਦਾ ਹੈ। ਇਹ ਤੁਹਾਡੇ ਲਈ ਅਨੁਵਾਦ ਫਾਈਲਾਂ ਬਣਾਉਣ ਅਤੇ ਕੰਪਾਇਲ ਕਰਨ ਦਾ ਕੰਮ ਸੰਭਾਲਦਾ ਹੈ।
