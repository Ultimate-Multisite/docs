# ਹੋਰ Plugins ਨਾਲ ਜਾਣੀਆਂ-ਪਛਾਣੀਆਂ ਅਸੰਗਤਤਾਵਾਂ

Ultimate Multisite ਨੂੰ ਇੰਸਟਾਲ ਅਤੇ ਐਕਟੀਵੇਟ ਕਰਨ ਤੋਂ ਬਾਅਦ, ਤੁਹਾਨੂੰ ਹੋਰ plugins ਨਾਲ ਅਣਸੁਲਝੀਆਂ ਅਸੰਗਤਤਾਵਾਂ ਕਾਰਨ ਸਮੱਸਿਆਵਾਂ ਆ ਸਕਦੀਆਂ ਹਨ।

ਅਸੀਂ ਹਰ ਨਵੀਂ ਰਿਲੀਜ਼ ਨਾਲ ਇਸ ਸੂਚੀ ਨੂੰ ਘਟਾਉਣ ਲਈ ਸਰਗਰਮੀ ਨਾਲ ਕੰਮ ਕਰ ਰਹੇ ਹਾਂ, ਪਰ ਜੇਕਰ ਕੋਈ plugin ਇੱਥੇ ਸੂਚੀਬੱਧ ਹੈ, ਤਾਂ ਤੁਹਾਨੂੰ ਪੂਰੀ ਤਰ੍ਹਾਂ ਕੰਮ ਕਰਨ ਵਾਲਾ Ultimate Multisite Network ਪ੍ਰਾਪਤ ਕਰਨ ਲਈ ਇਸਨੂੰ ਡੀਐਕਟੀਵੇਟ ਕਰਨ ਦੀ ਲੋੜ ਹੋ ਸਕਦੀ ਹੈ।

ਇੱਥੇ ਉਹਨਾਂ ਦੀ ਸੂਚੀ ਹੈ ਅਤੇ ਅਜਿਹੇ ਮਾਮਲਿਆਂ ਵਿੱਚ ਤੁਹਾਨੂੰ ਕੀ ਕਰਨਾ ਚਾਹੀਦਾ ਹੈ:

**ਕੁਝ WPMU DEV plugins** WPMU DEV ਦੁਆਰਾ _ਹੁਣ ਸਾਂਭ-ਸੰਭਾਲ ਨਾ ਕੀਤੇ ਜਾਣ ਵਾਲੇ_ ਕੁਝ plugins Ultimate Multisite ਦੀ ਰਜਿਸਟ੍ਰੇਸ਼ਨ ਪ੍ਰਕਿਰਿਆ ਨੂੰ ਆਖਰੀ ਕਦਮ 'ਤੇ ਰੋਕ ਦਿੰਦੇ ਹਨ (ਯੂਜ਼ਰ ਅਕਾਊਂਟ ਨਹੀਂ ਬਣਾ ਸਕਦਾ)। ਇੱਥੇ ਉਹ plugins ਹਨ ਜੋ ਇਹ ਸਮੱਸਿਆਵਾਂ ਪੈਦਾ ਕਰਦੇ ਹਨ:

  * Signup Language: _Ultimate Multisite ਵਿੱਚ ਇੱਕ add-on ਹੈ ਜੋ ਇਹੀ ਕਾਰਜਸ਼ੀਲਤਾ ਪੇਸ਼ ਕਰਦਾ ਹੈ, ਜਿਸਦਾ ਨਾਮ ਹੈ Ultimate Multisite: Language Switcher।_

  * Set Blog Description on Blog Creation: _ਇਹ Ultimate Multisite ਵਿੱਚ ਪਹਿਲਾਂ ਤੋਂ ਹੀ ਮੌਜੂਦ ਹੈ, ਇਸ ਲਈ ਵੱਖਰੇ plugin ਦੀ ਲੋੜ ਨਹੀਂ ਹੈ।_

  * New Blog Templates: _ਇਹ Ultimate Multisite ਵਿੱਚ ਪਹਿਲਾਂ ਤੋਂ ਹੀ ਮੌਜੂਦ ਹੈ, ਇਸ ਲਈ ਵੱਖਰੇ plugin ਦੀ ਲੋੜ ਨਹੀਂ ਹੈ।_

  * Set Password on Multisite Blog Creation: _ਇਹ Ultimate Multisite ਵਿੱਚ ਪਹਿਲਾਂ ਤੋਂ ਹੀ ਮੌਜੂਦ ਹੈ, ਇਸ ਲਈ ਵੱਖਰੇ plugin ਦੀ ਲੋੜ ਨਹੀਂ ਹੈ।_

  * Signup TOS: _ਇਹ Ultimate Multisite ਵਿੱਚ ਪਹਿਲਾਂ ਤੋਂ ਹੀ ਮੌਜੂਦ ਹੈ, ਇਸ ਲਈ ਵੱਖਰੇ plugin ਦੀ ਲੋੜ ਨਹੀਂ ਹੈ।_

  * Anti-Splog: ਭਾਵੇਂ ਘੱਟ ਸ਼ਕਤੀਸ਼ਾਲੀ ਹੈ, ਪਰ ਇਹ ਵੀ Ultimate Multisite ਵਿੱਚ _ਪਹਿਲਾਂ ਤੋਂ ਹੀ ਮੌਜੂਦ ਹੈ।_

**ਨੋਟ** : ਬਾਕੀ ਸਾਰੇ WPMU Dev plugins ਨੂੰ Ultimate Multisite ਦੇ ਨਾਲ ਆਮ ਤੌਰ 'ਤੇ ਵਰਤਿਆ ਜਾ ਸਕਦਾ ਹੈ। ਉਦਾਹਰਨਾਂ ਵਿੱਚ _Smush_ , _Forminator_ , _Defender,_ ਆਦਿ ਸ਼ਾਮਲ ਹਨ।
