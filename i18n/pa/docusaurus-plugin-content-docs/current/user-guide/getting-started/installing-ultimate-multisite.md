# Ultimate Multisite ਇੰਸਟਾਲ ਕਰਨਾ

:::note
ਇਹ ਟਿਊਟੋਰੀਅਲ ਮੰਨਦਾ ਹੈ ਕਿ ਤੁਸੀਂ ਪਹਿਲਾਂ ਹੀ WordPress Multisite ਇੰਸਟਾਲ ਅਤੇ ਕੌਂਫਿਗਰ ਕਰ ਚੁੱਕੇ ਹੋ। ਇਹ ਕਿਵੇਂ ਕਰਨਾ ਹੈ, ਇਹ ਸਿੱਖਣ ਲਈ WP Beginner ਦਾ [ਇਹ ਟਿਊਟੋਰੀਅਲ](https://www.wpbeginner.com/glossary/multisite/) ਦੇਖੋ।
:::

## Plugin ਇੰਸਟਾਲ ਕਰਨਾ

Ultimate Multisite [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) 'ਤੇ ਮੁਫ਼ਤ ਉਪਲਬਧ ਹੈ।

ਆਪਣੇ **Network Admin Dashboard** ਤੋਂ, **Plugins → Add New Plugin** 'ਤੇ ਜਾਓ।

![Network Admin Add New Plugin ਪੇਜ](/img/installation/add-new-plugin.png)

**"Ultimate Multisite"** ਖੋਜੋ (ਸਹੀ ਮੈਚ ਲਈ ਕੋਟੇਸ਼ਨ ਮਾਰਕਸ ਨਾਲ) ਅਤੇ ਇਹ ਪਹਿਲੇ ਨਤੀਜੇ ਵਜੋਂ ਦਿਖਾਈ ਦੇਵੇਗਾ। **Install Now** 'ਤੇ ਕਲਿੱਕ ਕਰੋ।

![Ultimate Multisite ਦਿਖਾਉਂਦੇ ਖੋਜ ਨਤੀਜੇ](/img/installation/search-ultimate-multisite.png)

ਇੰਸਟਾਲ ਹੋਣ ਤੋਂ ਬਾਅਦ, ਆਪਣੇ ਪੂਰੇ ਨੈੱਟਵਰਕ 'ਤੇ plugin ਨੂੰ ਐਕਟੀਵੇਟ ਕਰਨ ਲਈ **Network Activate** 'ਤੇ ਕਲਿੱਕ ਕਰੋ।

![Plugin ਇੰਸਟਾਲ ਹੋਇਆ Network Activate ਬਟਨ ਨਾਲ](/img/installation/plugin-installed.png)

ਐਕਟੀਵੇਸ਼ਨ ਤੋਂ ਬਾਅਦ, ਤੁਸੀਂ ਆਪਣੇ ਆਪ Setup Wizard 'ਤੇ ਰੀਡਾਇਰੈਕਟ ਹੋ ਜਾਓਗੇ।

![Plugin ਐਕਟੀਵੇਟ ਹੋਇਆ ਅਤੇ wizard 'ਤੇ ਰੀਡਾਇਰੈਕਟ](/img/installation/plugin-activated.png)

## Setup Wizard

Setup Wizard ਤੁਹਾਨੂੰ ਲਗਭਗ 10 ਮਿੰਟਾਂ ਵਿੱਚ Ultimate Multisite ਕੌਂਫਿਗਰ ਕਰਨ ਵਿੱਚ ਮਦਦ ਕਰੇਗਾ।

### ਸਵਾਗਤ

ਸ਼ੁਰੂ ਕਰਨ ਲਈ **Get Started** 'ਤੇ ਕਲਿੱਕ ਕਰੋ।

![Setup Wizard ਸਵਾਗਤ ਸਕ੍ਰੀਨ](/img/installation/wizard-welcome.png)

### ਪ੍ਰੀ-ਇੰਸਟਾਲ ਜਾਂਚਾਂ

ਇਹ ਕਦਮ ਤੁਹਾਡੇ ਸਿਸਟਮ ਦੀ ਜਾਣਕਾਰੀ ਅਤੇ WordPress ਇੰਸਟਾਲੇਸ਼ਨ ਦੀ ਜਾਂਚ ਕਰਦਾ ਹੈ ਤਾਂ ਜੋ ਯਕੀਨੀ ਬਣਾਇਆ ਜਾ ਸਕੇ ਕਿ ਇਹ Ultimate Multisite ਦੀਆਂ ਲੋੜਾਂ ਪੂਰੀਆਂ ਕਰਦਾ ਹੈ। ਜੇ ਸਭ ਕੁਝ ਠੀਕ ਲੱਗਦਾ ਹੈ, ਤਾਂ **Go to the Next Step** 'ਤੇ ਕਲਿੱਕ ਕਰੋ।

![ਸਿਸਟਮ ਲੋੜਾਂ ਦਿਖਾਉਂਦੀਆਂ ਪ੍ਰੀ-ਇੰਸਟਾਲ ਜਾਂਚਾਂ](/img/installation/wizard-pre-install-checks.png)

### ਇੰਸਟਾਲੇਸ਼ਨ

ਇੰਸਟਾਲਰ ਲੋੜੀਂਦੀਆਂ ਡੇਟਾਬੇਸ ਟੇਬਲਾਂ ਬਣਾਏਗਾ ਅਤੇ `sunrise.php` ਫਾਈਲ ਇੰਸਟਾਲ ਕਰੇਗਾ ਜੋ Ultimate Multisite ਨੂੰ ਕੰਮ ਕਰਨ ਲਈ ਚਾਹੀਦੀ ਹੈ। ਅੱਗੇ ਵਧਣ ਲਈ **Install** 'ਤੇ ਕਲਿੱਕ ਕਰੋ।

![ਡੇਟਾਬੇਸ ਟੇਬਲਾਂ ਅਤੇ sunrise.php ਦਿਖਾਉਂਦਾ ਇੰਸਟਾਲੇਸ਼ਨ ਕਦਮ](/img/installation/wizard-installation.png)

### ਤੁਹਾਡੀ ਕੰਪਨੀ

ਆਪਣੀ ਕੰਪਨੀ ਦੀ ਜਾਣਕਾਰੀ ਭਰੋ ਅਤੇ ਆਪਣੀ ਡਿਫਾਲਟ ਮੁਦਰਾ ਸੈੱਟ ਕਰੋ। ਇਹ ਜਾਣਕਾਰੀ ਤੁਹਾਡੇ WaaS ਪਲੇਟਫਾਰਮ ਵਿੱਚ ਹਰ ਥਾਂ ਵਰਤੀ ਜਾਵੇਗੀ। ਪੂਰਾ ਹੋਣ 'ਤੇ **Continue** 'ਤੇ ਕਲਿੱਕ ਕਰੋ।

![ਤੁਹਾਡੀ ਕੰਪਨੀ ਕੌਂਫਿਗਰੇਸ਼ਨ ਕਦਮ](/img/installation/wizard-your-company.png)

### ਡਿਫਾਲਟ ਸਮੱਗਰੀ

ਇਹ ਕਦਮ ਤੁਹਾਨੂੰ ਪਹਿਲਾਂ ਤੋਂ ਬਣੇ ਟੈਂਪਲੇਟ, ਪ੍ਰੋਡਕਟ, ਅਤੇ ਹੋਰ ਸ਼ੁਰੂਆਤੀ ਸਮੱਗਰੀ ਇੰਸਟਾਲ ਕਰਨ ਦਿੰਦਾ ਹੈ। Ultimate Multisite ਦੀਆਂ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ ਨੂੰ ਸਮਝਣ ਦਾ ਇਹ ਵਧੀਆ ਤਰੀਕਾ ਹੈ। ਡਿਫਾਲਟ ਸਮੱਗਰੀ ਜੋੜਨ ਲਈ **Install** 'ਤੇ ਕਲਿੱਕ ਕਰੋ, ਜਾਂ ਜੇ ਤੁਸੀਂ ਸ਼ੁਰੂ ਤੋਂ ਸ਼ੁਰੂ ਕਰਨਾ ਚਾਹੁੰਦੇ ਹੋ ਤਾਂ ਇਸ ਕਦਮ ਨੂੰ ਛੱਡ ਦਿਓ।

![ਡਿਫਾਲਟ ਸਮੱਗਰੀ ਇੰਸਟਾਲੇਸ਼ਨ ਕਦਮ](/img/installation/wizard-default-content.png)

### ਸਿਫਾਰਸ਼ੀ Plugins

ਵਿਕਲਪਿਕ ਤੌਰ 'ਤੇ ਸਿਫਾਰਸ਼ੀ ਸਾਥੀ plugins ਇੰਸਟਾਲ ਕਰੋ। ਇਹਨਾਂ ਨੂੰ ਜੋੜਨ ਲਈ **Install** 'ਤੇ ਕਲਿੱਕ ਕਰੋ ਜਾਂ ਅੱਗੇ ਜਾਣ ਲਈ ਛੱਡ ਦਿਓ।

![ਸਿਫਾਰਸ਼ੀ plugins ਕਦਮ](/img/installation/wizard-recommended-plugins.png)

### ਤਿਆਰ!

ਬੱਸ ਇਹੀ ਹੈ! ਤੁਹਾਡੀ Ultimate Multisite ਇੰਸਟਾਲੇਸ਼ਨ ਪੂਰੀ ਹੋ ਗਈ ਹੈ। ਤੁਸੀਂ ਹੁਣ **Network Admin Dashboard** ਤੋਂ ਆਪਣਾ Website as a Service ਪਲੇਟਫਾਰਮ ਬਣਾਉਣਾ ਸ਼ੁਰੂ ਕਰ ਸਕਦੇ ਹੋ।

![ਸੈੱਟਅੱਪ ਪੂਰਾ - ਤਿਆਰ ਸਕ੍ਰੀਨ](/img/installation/wizard-ready.png)

![Ultimate Multisite ਐਕਟਿਵ ਨਾਲ Network Admin Dashboard](/img/installation/network-dashboard.png)

ਅੱਗੇ ਵਧੋ ਅਤੇ ਮਜ਼ੇ ਕਰੋ!
