# Ultimate Multisite ਲੋੜਾਂ

Ultimate Multisite ਖਰੀਦਣ ਤੋਂ ਪਹਿਲਾਂ ਤੁਹਾਡੇ ਮਨ ਵਿੱਚ ਪਹਿਲਾ ਸਵਾਲ ਇਹ ਆ ਸਕਦਾ ਹੈ ਕਿ ਸਾਡੇ plugin ਨੂੰ ਚਲਾਉਣ ਲਈ ਕੀ ਲੋੜਾਂ ਹਨ। ਖੁਸ਼ੀ ਦੀ ਗੱਲ ਹੈ ਕਿ Ultimate Multisite ਚਲਾਉਣ ਲਈ ਸਿਰਫ਼ ਕੁਝ ਬੁਨਿਆਦੀ ਲੋੜਾਂ ਹਨ: ਇੱਕ **domain name**, ਇੱਕ ਵਧੀਆ **hosting provider** ਅਤੇ ਇੱਕ WordPress **Multisite** installation। ਬੱਸ ਇਹੀ ਚਾਹੀਦਾ ਹੈ!

## Domain Name

ਪਹਿਲਾਂ, ਆਓ domain name ਬਾਰੇ ਗੱਲ ਕਰੀਏ।

Domain name ਚੁਣਦੇ ਸਮੇਂ, ਇਹ ਧਿਆਨ ਵਿੱਚ ਰੱਖੋ ਕਿ ਤੁਹਾਡਾ ਖਾਸ niche ਕੀ ਹੈ ਅਤੇ ਤੁਸੀਂ Ultimate Multisite ਨਾਲ ਕੀ ਬਣਾਉਣਾ ਚਾਹੁੰਦੇ ਹੋ: ਕੀ ਤੁਸੀਂ ਕੋਈ agency ਬਣਾਉਣਾ ਚਾਹੁੰਦੇ ਹੋ? ਦੁਕਾਨ ਵਰਗੀਆਂ ਵੈੱਬਸਾਈਟਾਂ ਵੇਚਣਾ ਚਾਹੁੰਦੇ ਹੋ? ਜਾਂ ਸ਼ਾਇਦ WordPress web hosting? Ultimate Multisite ਨਾਲ ਸਭ ਕੁਝ ਸੰਭਵ ਹੈ।

ਸਾਡਾ plugin subscription ਸੇਵਾਵਾਂ, payment ਇਕੱਠਾ ਕਰਨਾ, checkout forms, discount vouchers ਅਤੇ ਗਾਹਕ ਸੰਚਾਰ ਦੀ ਸਹੂਲਤ ਦਿੰਦਾ ਹੈ। WordPress Multisite ਨੂੰ install, configure ਅਤੇ maintain ਕਰਨ ਲਈ ਲੋੜੀਂਦੇ ਬਹੁਤ ਸਾਰੇ ਕੰਮ Ultimate Multisite ਦੁਆਰਾ ਆਸਾਨ ਬਣਾ ਦਿੱਤੇ ਜਾਂਦੇ ਹਨ।

ਤੁਸੀਂ ਜੋ ਵੀ niche ਚੁਣੋ, ਉਸ ਨੂੰ ਸਾਡੇ plugin ਨਾਲ ਸੰਭਾਲਿਆ ਜਾ ਸਕਦਾ ਹੈ ਜਦੋਂ ਤੱਕ ਤੁਸੀਂ ਵੈੱਬਸਾਈਟਾਂ ਨੂੰ ਸੇਵਾ ਵਜੋਂ ਵੇਚ ਰਹੇ ਹੋ! ਯਕੀਨੀ ਬਣਾਓ ਕਿ ਤੁਸੀਂ ਅਜਿਹਾ domain name ਚੁਣੋ ਜੋ ਤੁਹਾਡੀ ਕਾਰੋਬਾਰੀ ਯੋਜਨਾ ਨਾਲ ਮੇਲ ਖਾਂਦਾ ਹੋਵੇ।

## Hosting Provider

ਹੁਣ ਜਦੋਂ ਤੁਹਾਡੇ ਕੋਲ ਤੁਹਾਡੇ niche ਅਤੇ ਕਾਰੋਬਾਰ ਨੂੰ ਦਰਸਾਉਣ ਵਾਲਾ ਆਕਰਸ਼ਕ domain name ਹੈ, ਤੁਹਾਨੂੰ Ultimate Multisite ਨਾਲ ਆਪਣੇ network ਨੂੰ host ਕਰਨ ਲਈ ਇੱਕ ਵਧੀਆ web hosting ਕੰਪਨੀ ਚੁਣਨੀ ਪਵੇਗੀ। ਬਾਜ਼ਾਰ ਵਿੱਚ ਕਈ ਵਧੀਆ ਵਿਕਲਪ ਹਨ। ਕੁਝ hosting providers Ultimate Multisite ਨਾਲ ਚੰਗੀ ਤਰ੍ਹਾਂ integrate ਹੁੰਦੇ ਹਨ, ਜੋ ਕਿ ਤੁਹਾਡੀ ਯੋਜਨਾ ਦੇ ਮੁਤਾਬਕ ਇੱਕ ਫੈਸਲਾਕੁਨ ਕਾਰਕ ਹੋ ਸਕਦਾ ਹੈ।

Ultimate Multisite ਕਿਸੇ ਵੀ hosting provider ਵਿੱਚ ਕੰਮ ਕਰਨਾ ਚਾਹੀਦਾ ਹੈ। ਫਰਕ ਸਿਰਫ਼ ਇਹ ਹੈ ਕਿ ਕੁਝ hosting ਕੰਪਨੀਆਂ ਜਿਵੇਂ _WPMU DEV Hosting_, _WP Engine_, _Closte_, _Cloudways_, _Gridpane_ ਅਤੇ _Runcloud_ ਕੋਲ domain mapping ਲਈ ਵਧੇਰੇ ਨੇੜਲਾ integration ਹੈ। ਹੋਰ hosting ਲਈ, ਤੁਹਾਨੂੰ custom domains ਨੂੰ ਖੁਦ map ਕਰਨਾ ਪੈ ਸਕਦਾ ਹੈ। AutoSSL ਦਾ ਸਮਰਥਨ ਕਰਨ ਵਾਲਾ hosting provider ਲੈਣਾ ਵੀ ਚੰਗਾ ਵਿਚਾਰ ਹੈ। _Closte_, _Runcloud_ ਅਤੇ _cPanel_ ਇਸਦਾ ਸਮਰਥਨ ਕਰਦੇ ਹਨ।

Server specifications ਦੀ ਗੱਲ ਕਰੀਏ ਤਾਂ, ਕੋਈ ਖਾਸ ਫਾਰਮੂਲਾ ਨਹੀਂ ਹੈ ਕਿਉਂਕਿ ਇਹ ਤੁਹਾਡੇ ਦੁਆਰਾ ਪੇਸ਼ ਕੀਤੀਆਂ ਜਾਣ ਵਾਲੀਆਂ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ ਅਤੇ ਤੁਹਾਡੇ network ਦੇ ਆਕਾਰ 'ਤੇ ਨਿਰਭਰ ਕਰੇਗਾ। Shared hosting ਇੱਕ ਵਧੀਆ ਸ਼ੁਰੂਆਤ ਹੋ ਸਕਦੀ ਹੈ, ਪਰ ਜਿਵੇਂ ਹੀ ਤੁਹਾਡਾ network ਵਧਣਾ ਸ਼ੁਰੂ ਹੋਵੇ, ਤੁਹਾਨੂੰ ਸ਼ਾਇਦ ਇੱਕ dedicated VPS ਦੀ ਲੋੜ ਪਵੇਗੀ।

## WordPress Multisite

ਹੁਣ, ਆਓ WordPress Multisite installation ਬਾਰੇ ਥੋੜੀ ਗੱਲ ਕਰੀਏ।

WordPress Multisite ਤੁਹਾਨੂੰ ਇੱਕ ਹੀ installation 'ਤੇ ਸਾਈਟਾਂ ਦਾ network ਰੱਖਣ ਦੀ ਸਹੂਲਤ ਦਿੰਦਾ ਹੈ। ਇਹ ਇੱਕ built-in ਵਿਸ਼ੇਸ਼ਤਾ ਹੈ, ਪਰ ਇਹ ਮੂਲ ਰੂਪ ਵਿੱਚ ਚਾਲੂ ਨਹੀਂ ਹੁੰਦੀ। ਤੁਹਾਨੂੰ ਇਸ ਵਿਸ਼ੇਸ਼ਤਾ ਨੂੰ ਆਪਣੀ WordPress installation ਅਤੇ web hosting control panel ਵਿੱਚ ਚਾਲੂ ਕਰਨ ਦੀ ਲੋੜ ਹੈ।

Multisite ਚਾਲੂ ਕੀਤੇ WordPress ਦਾ ਹੋਣਾ **Ultimate Multisite ਦੀ ਬੁਨਿਆਦ ਹੈ**। ਕਿਉਂਕਿ Ultimate Multisite ਦੀ ਮੁੱਖ ਵਰਤੋਂ ਉਪਭੋਗਤਾਵਾਂ ਨੂੰ ਵੈੱਬਸਾਈਟਾਂ ਨੂੰ ਸੇਵਾ ਵਜੋਂ ਸੰਭਾਲਣ ਅਤੇ ਵੇਚਣ ਵਿੱਚ ਮਦਦ ਕਰਨਾ ਹੈ, ਇਹ Multisite ਚਾਲੂ ਕੀਤੇ WordPress 'ਤੇ ਕਰਨਾ ਜ਼ਰੂਰੀ ਹੈ।

ਸਾਡੇ ਕੋਲ **ਇੱਕ ਵਿਸਤ੍ਰਿਤ ਲੇਖ** ਹੈ ਜੋ ਕਦਮ-ਦਰ-ਕਦਮ ਸਿਖਾਉਂਦਾ ਹੈ ਕਿ ਤੁਹਾਡੀ WordPress installation 'ਤੇ Multisite ਵਿਸ਼ੇਸ਼ਤਾ ਨੂੰ ਚਾਲੂ ਕਰਨ ਲਈ ਕੀ ਕਰਨਾ ਚਾਹੀਦਾ ਹੈ।

WordPress 'ਤੇ Multisite ਚਾਲੂ ਕਰਨ ਤੋਂ ਬਾਅਦ, Ultimate Multisite install ਕਰਨ ਦਾ ਸਮਾਂ ਹੈ। Ultimate Multisite ਨੂੰ ਸਹੀ ਤਰ੍ਹਾਂ install ਕਰਨ ਲਈ, ਤੁਹਾਡੇ ਕੋਲ ਹੋਣਾ ਚਾਹੀਦਾ ਹੈ:

  * WordPress: v5.3+ (ਸਿਫਾਰਸ਼ੀ: ਆਖਰੀ stable version)

  * PHP: 7.4.x (> 8.0 ਦਾ ਸਮਰਥਨ ਜਲਦੀ ਆ ਰਿਹਾ ਹੈ)

  * MySQL: v5+ (ਸਿਫਾਰਸ਼ੀ: 5.6, ਜੇਕਰ ਤੁਹਾਡੇ hosting provider 'ਤੇ 8.0 ਉਪਲਬਧ ਨਹੀਂ ਹੈ)

Ultimate Multisite ਲਈ ਇਹ ਹੀ software ਲੋੜਾਂ ਹਨ।

ਧਿਆਨ ਰੱਖੋ ਕਿ Ultimate Multisite PHP 8.0 ਨਾਲ ਕੰਮ ਕਰ ਸਕਦਾ ਹੈ ਪਰ ਅਸੀਂ ਇਸਨੂੰ PHP 7.4.x ਨਾਲ ਚਲਾਉਣ ਦੀ ਸਿਫਾਰਸ਼ ਕਰਦੇ ਹਾਂ।

ਨਾਲ ਹੀ, ਅਸੀਂ ਇਸਨੂੰ **main domains** 'ਤੇ install ਕਰਨ ਦੀ ਸਿਫਾਰਸ਼ ਕਰਦੇ ਹਾਂ, subdomains ਜਾਂ subdirectories 'ਤੇ ਨਹੀਂ। ਇਹ subdomains/subdirectories 'ਤੇ ਕੰਮ ਕਰ ਸਕਦਾ ਹੈ, ਪਰ ਕੁਝ ਗਲਤੀਆਂ ਆ ਸਕਦੀਆਂ ਹਨ।

Ultimate Multisite ਨੂੰ install ਕਰਨ ਦੀ ਵਿਸਤ੍ਰਿਤ ਗਾਈਡ ਲਈ, **ਇੱਥੇ ਕਲਿੱਕ ਕਰੋ**।
