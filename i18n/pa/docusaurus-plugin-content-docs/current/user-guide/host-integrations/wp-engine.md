# WP Engine ਏਕੀਕਰਨ

## ਸੰਖੇਪ ਜਾਣਕਾਰੀ
WP Engine ਇੱਕ ਪ੍ਰੀਮੀਅਮ ਮੈਨੇਜਡ WordPress ਹੋਸਟਿੰਗ ਪਲੇਟਫਾਰਮ ਹੈ ਜੋ WordPress ਸਾਈਟਾਂ ਲਈ ਬਿਹਤਰ ਪ੍ਰਦਰਸ਼ਨ, ਸੁਰੱਖਿਆ ਅਤੇ ਸਕੇਲੇਬਿਲਟੀ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ। ਇਹ ਏਕੀਕਰਨ Ultimate Multisite ਅਤੇ WP Engine ਵਿਚਕਾਰ ਆਟੋਮੈਟਿਕ ਡੋਮੇਨ ਸਿੰਕਿੰਗ ਨੂੰ ਸਮਰੱਥ ਬਣਾਉਂਦਾ ਹੈ।

## ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ
- ਆਟੋਮੈਟਿਕ ਡੋਮੇਨ ਸਿੰਕਿੰਗ
- ਮਲਟੀਸਾਈਟ ਇੰਸਟਾਲੇਸ਼ਨਾਂ ਲਈ ਸਬਡੋਮੇਨ ਸਹਾਇਤਾ
- WP Engine ਦੇ ਮੌਜੂਦਾ ਸਿਸਟਮਾਂ ਨਾਲ ਸਹਿਜ ਏਕੀਕਰਨ

## ਲੋੜਾਂ
ਇਹ ਏਕੀਕਰਨ ਆਟੋਮੈਟਿਕ ਤੌਰ 'ਤੇ ਪਤਾ ਲਗਾ ਲੈਂਦਾ ਹੈ ਕਿ ਤੁਸੀਂ WP Engine 'ਤੇ ਹੋਸਟ ਕਰ ਰਹੇ ਹੋ ਅਤੇ ਬਿਲਟ-ਇਨ WP Engine API ਦੀ ਵਰਤੋਂ ਕਰਦਾ ਹੈ। ਜੇਕਰ WP Engine plugin ਐਕਟਿਵ ਹੈ ਅਤੇ ਸਹੀ ਢੰਗ ਨਾਲ ਕੌਂਫਿਗਰ ਹੈ, ਤਾਂ ਕਿਸੇ ਵਾਧੂ ਸੈਟਿੰਗ ਦੀ ਲੋੜ ਨਹੀਂ ਹੈ।

ਹਾਲਾਂਕਿ, ਜੇਕਰ ਤੁਹਾਨੂੰ ਏਕੀਕਰਨ ਨੂੰ ਮੈਨੂਅਲੀ ਕੌਂਫਿਗਰ ਕਰਨ ਦੀ ਲੋੜ ਹੈ, ਤਾਂ ਤੁਸੀਂ ਆਪਣੀ `wp-config.php` ਫਾਈਲ ਵਿੱਚ ਇਹਨਾਂ ਵਿੱਚੋਂ ਕੋਈ ਇੱਕ constant ਡਿਫਾਈਨ ਕਰ ਸਕਦੇ ਹੋ:

```php
define('WPE_APIKEY', 'your_api_key'); // ਤਰਜੀਹੀ ਤਰੀਕਾ
// ਜਾਂ
define('WPE_API', 'your_api_key'); // ਵਿਕਲਪਕ ਤਰੀਕਾ
```

## ਸੈੱਟਅੱਪ ਹਿਦਾਇਤਾਂ

### 1. WP Engine Plugin ਦੀ ਪੁਸ਼ਟੀ ਕਰੋ

ਜੇਕਰ ਤੁਸੀਂ WP Engine 'ਤੇ ਹੋਸਟ ਕਰ ਰਹੇ ਹੋ, ਤਾਂ WP Engine plugin ਪਹਿਲਾਂ ਹੀ ਇੰਸਟਾਲ ਅਤੇ ਐਕਟੀਵੇਟ ਹੋਣਾ ਚਾਹੀਦਾ ਹੈ। ਯਕੀਨੀ ਬਣਾਓ ਕਿ:

1. WP Engine plugin ਐਕਟਿਵ ਹੈ
2. ਫਾਈਲ `wp-content/mu-plugins/wpengine-common/class-wpeapi.php` ਮੌਜੂਦ ਹੈ

### 2. ਏਕੀਕਰਨ ਨੂੰ ਸਮਰੱਥ ਕਰੋ

1. ਆਪਣੇ WordPress admin ਵਿੱਚ, Ultimate Multisite > Settings 'ਤੇ ਜਾਓ
2. "Domain Mapping" ਟੈਬ 'ਤੇ ਜਾਓ
3. "Host Integrations" ਤੱਕ ਸਕ੍ਰੋਲ ਕਰੋ
4. WP Engine ਏਕੀਕਰਨ ਨੂੰ ਸਮਰੱਥ ਕਰੋ
5. "Save Changes" 'ਤੇ ਕਲਿੱਕ ਕਰੋ

## ਇਹ ਕਿਵੇਂ ਕੰਮ ਕਰਦਾ ਹੈ

### ਡੋਮੇਨ ਸਿੰਕਿੰਗ

ਜਦੋਂ Ultimate Multisite ਵਿੱਚ ਕੋਈ ਡੋਮੇਨ ਮੈਪ ਕੀਤਾ ਜਾਂਦਾ ਹੈ:

1. ਏਕੀਕਰਨ WP Engine API ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਤੁਹਾਡੀ WP Engine ਇੰਸਟਾਲੇਸ਼ਨ ਵਿੱਚ ਡੋਮੇਨ ਜੋੜਦਾ ਹੈ
2. WP Engine ਡੋਮੇਨ ਕੌਂਫਿਗਰੇਸ਼ਨ ਅਤੇ SSL certificate ਜਾਰੀ ਕਰਨ ਦਾ ਕੰਮ ਸੰਭਾਲਦਾ ਹੈ
3. ਜਦੋਂ ਕੋਈ ਡੋਮੇਨ ਮੈਪਿੰਗ ਹਟਾਈ ਜਾਂਦੀ ਹੈ, ਤਾਂ ਏਕੀਕਰਨ ਡੋਮੇਨ ਨੂੰ WP Engine ਤੋਂ ਹਟਾ ਦੇਵੇਗਾ

### ਸਬਡੋਮੇਨ ਸਹਾਇਤਾ

ਸਬਡੋਮੇਨ ਮਲਟੀਸਾਈਟ ਇੰਸਟਾਲੇਸ਼ਨਾਂ ਲਈ:

1. ਜਦੋਂ ਕੋਈ ਨਵੀਂ ਸਾਈਟ ਬਣਾਈ ਜਾਂਦੀ ਹੈ ਤਾਂ ਏਕੀਕਰਨ ਹਰੇਕ ਸਬਡੋਮੇਨ ਨੂੰ WP Engine ਵਿੱਚ ਜੋੜਦਾ ਹੈ
2. WP Engine ਸਬਡੋਮੇਨ ਕੌਂਫਿਗਰੇਸ਼ਨ ਨੂੰ ਸੰਭਾਲਦਾ ਹੈ
3. ਜਦੋਂ ਕੋਈ ਸਾਈਟ ਮਿਟਾਈ ਜਾਂਦੀ ਹੈ, ਤਾਂ ਏਕੀਕਰਨ ਸਬਡੋਮੇਨ ਨੂੰ WP Engine ਤੋਂ ਹਟਾ ਦੇਵੇਗਾ

## ਮਹੱਤਵਪੂਰਨ ਨੋਟਸ

### Wildcard ਡੋਮੇਨ

ਸਬਡੋਮੇਨ ਮਲਟੀਸਾਈਟ ਇੰਸਟਾਲੇਸ਼ਨਾਂ ਲਈ, WP Engine ਸਹਾਇਤਾ ਨਾਲ ਸੰਪਰਕ ਕਰਕੇ wildcard ਡੋਮੇਨ ਕੌਂਫਿਗਰੇਸ਼ਨ ਦੀ ਬੇਨਤੀ ਕਰਨ ਦੀ ਸਿਫਾਰਸ਼ ਕੀਤੀ ਜਾਂਦੀ ਹੈ। ਇਸ ਨਾਲ ਸਾਰੇ ਸਬਡੋਮੇਨ ਹਰ ਇੱਕ ਨੂੰ ਵੱਖਰੇ ਤੌਰ 'ਤੇ ਜੋੜੇ ਬਿਨਾਂ ਆਟੋਮੈਟਿਕ ਕੰਮ ਕਰਨਗੇ।

### SSL Certificates

WP Engine ਇਸ ਏਕੀਕਰਨ ਰਾਹੀਂ ਜੋੜੇ ਗਏ ਸਾਰੇ ਡੋਮੇਨਾਂ ਲਈ SSL certificate ਜਾਰੀ ਕਰਨ ਅਤੇ ਰੀਨਿਊ ਕਰਨ ਨੂੰ ਆਟੋਮੈਟਿਕ ਸੰਭਾਲਦਾ ਹੈ। ਕਿਸੇ ਵਾਧੂ ਕੌਂਫਿਗਰੇਸ਼ਨ ਦੀ ਲੋੜ ਨਹੀਂ ਹੈ।

## ਸਮੱਸਿਆ ਨਿਵਾਰਨ

### API ਕਨੈਕਸ਼ਨ ਸਮੱਸਿਆਵਾਂ
- ਯਕੀਨੀ ਬਣਾਓ ਕਿ WP Engine plugin ਐਕਟਿਵ ਅਤੇ ਸਹੀ ਢੰਗ ਨਾਲ ਕੌਂਫਿਗਰ ਹੈ
- ਜੇਕਰ ਤੁਸੀਂ API key ਮੈਨੂਅਲੀ ਡਿਫਾਈਨ ਕੀਤੀ ਹੈ, ਤਾਂ ਚੈੱਕ ਕਰੋ ਕਿ ਇਹ ਸਹੀ ਹੈ
- ਜੇਕਰ API ਨਾਲ ਸਮੱਸਿਆ ਆ ਰਹੀ ਹੈ ਤਾਂ WP Engine ਸਹਾਇਤਾ ਨਾਲ ਸੰਪਰਕ ਕਰੋ

### ਡੋਮੇਨ ਨਹੀਂ ਜੋੜਿਆ ਗਿਆ
- ਕਿਸੇ ਵੀ ਗਲਤੀ ਸੁਨੇਹਿਆਂ ਲਈ Ultimate Multisite ਲੌਗਸ ਚੈੱਕ ਕਰੋ
- ਯਕੀਨੀ ਬਣਾਓ ਕਿ ਡੋਮੇਨ ਪਹਿਲਾਂ ਤੋਂ WP Engine ਵਿੱਚ ਜੋੜਿਆ ਨਹੀਂ ਗਿਆ ਹੈ
- ਯਕੀਨੀ ਬਣਾਓ ਕਿ ਤੁਹਾਡਾ WP Engine ਪਲਾਨ ਤੁਹਾਡੇ ਵੱਲੋਂ ਜੋੜੇ ਜਾ ਰਹੇ ਡੋਮੇਨਾਂ ਦੀ ਗਿਣਤੀ ਨੂੰ ਸਹਾਇਤਾ ਕਰਦਾ ਹੈ

### ਸਬਡੋਮੇਨ ਸਮੱਸਿਆਵਾਂ
- ਜੇਕਰ ਸਬਡੋਮੇਨ ਕੰਮ ਨਹੀਂ ਕਰ ਰਹੇ, ਤਾਂ WP Engine ਸਹਾਇਤਾ ਨਾਲ ਸੰਪਰਕ ਕਰਕੇ wildcard ਡੋਮੇਨ ਕੌਂਫਿਗਰੇਸ਼ਨ ਦੀ ਬੇਨਤੀ ਕਰੋ
- ਯਕੀਨੀ ਬਣਾਓ ਕਿ ਤੁਹਾਡੀਆਂ DNS ਸੈਟਿੰਗਾਂ ਮੁੱਖ ਡੋਮੇਨ ਅਤੇ ਸਬਡੋਮੇਨਾਂ ਲਈ ਸਹੀ ਢੰਗ ਨਾਲ ਕੌਂਫਿਗਰ ਹਨ
