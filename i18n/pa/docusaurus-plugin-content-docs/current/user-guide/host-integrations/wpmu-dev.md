# WPMU DEV ਇੰਟੀਗ੍ਰੇਸ਼ਨ

## ਸੰਖੇਪ ਜਾਣਕਾਰੀ
WPMU DEV ਇੱਕ ਸੰਪੂਰਨ WordPress ਪਲੇਟਫਾਰਮ ਹੈ ਜੋ WordPress ਸਾਈਟਾਂ ਲਈ ਹੋਸਟਿੰਗ, plugins, ਅਤੇ ਸੇਵਾਵਾਂ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ। ਇਹ ਇੰਟੀਗ੍ਰੇਸ਼ਨ Ultimate Multisite ਅਤੇ WPMU DEV ਹੋਸਟਿੰਗ ਵਿਚਕਾਰ ਆਟੋਮੈਟਿਕ ਡੋਮੇਨ ਸਿੰਕਿੰਗ ਅਤੇ SSL ਸਰਟੀਫਿਕੇਟ ਪ੍ਰਬੰਧਨ ਨੂੰ ਸਮਰੱਥ ਬਣਾਉਂਦਾ ਹੈ।

## ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ
- ਆਟੋਮੈਟਿਕ ਡੋਮੇਨ ਸਿੰਕਿੰਗ
- SSL ਸਰਟੀਫਿਕੇਟ ਪ੍ਰਬੰਧਨ
- SSL ਸਰਟੀਫਿਕੇਟ ਪੁਸ਼ਟੀ ਲਈ ਵਧੇਰੇ ਕੋਸ਼ਿਸ਼ਾਂ

## ਲੋੜਾਂ
ਇਹ ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਆਪਣੇ ਆਪ ਪਤਾ ਲਗਾ ਲੈਂਦਾ ਹੈ ਕਿ ਤੁਸੀਂ WPMU DEV 'ਤੇ ਹੋਸਟਿੰਗ ਕਰ ਰਹੇ ਹੋ ਅਤੇ ਬਿਲਟ-ਇਨ API ਦੀ ਵਰਤੋਂ ਕਰਦਾ ਹੈ। ਜੇਕਰ ਤੁਸੀਂ WPMU DEV 'ਤੇ ਹੋਸਟਿੰਗ ਕਰ ਰਹੇ ਹੋ ਤਾਂ ਕਿਸੇ ਵਾਧੂ ਸੈੱਟਅੱਪ ਦੀ ਲੋੜ ਨਹੀਂ ਹੈ।

ਇਹ ਇੰਟੀਗ੍ਰੇਸ਼ਨ `WPMUDEV_HOSTING_SITE_ID` constant ਦੀ ਮੌਜੂਦਗੀ ਦੀ ਜਾਂਚ ਕਰਦਾ ਹੈ, ਜੋ WPMU DEV 'ਤੇ ਹੋਸਟਿੰਗ ਕਰਨ ਵੇਲੇ ਆਪਣੇ ਆਪ ਪਰਿਭਾਸ਼ਿਤ ਹੋ ਜਾਂਦਾ ਹੈ।

## ਸੈੱਟਅੱਪ ਨਿਰਦੇਸ਼

### 1. WPMU DEV ਹੋਸਟਿੰਗ ਦੀ ਪੁਸ਼ਟੀ ਕਰੋ

ਜੇਕਰ ਤੁਸੀਂ WPMU DEV 'ਤੇ ਹੋਸਟਿੰਗ ਕਰ ਰਹੇ ਹੋ, ਤਾਂ ਲੋੜੀਂਦੇ constants ਪਹਿਲਾਂ ਹੀ ਪਰਿਭਾਸ਼ਿਤ ਹੋਣੇ ਚਾਹੀਦੇ ਹਨ। ਯਕੀਨੀ ਬਣਾਓ ਕਿ:

1. `WPMUDEV_HOSTING_SITE_ID` constant ਤੁਹਾਡੇ environment ਵਿੱਚ ਪਰਿਭਾਸ਼ਿਤ ਹੈ
2. ਤੁਹਾਡੇ ਕੋਲ API ਪਹੁੰਚ ਵਾਲੀ ਇੱਕ ਸਰਗਰਮ WPMU DEV ਮੈਂਬਰਸ਼ਿਪ ਹੈ

### 2. ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਨੂੰ ਸਮਰੱਥ ਕਰੋ

1. ਆਪਣੇ WordPress admin ਵਿੱਚ, Ultimate Multisite > Settings 'ਤੇ ਜਾਓ
2. "Domain Mapping" ਟੈਬ 'ਤੇ ਜਾਓ
3. "Host Integrations" ਤੱਕ ਹੇਠਾਂ ਸਕ੍ਰੋਲ ਕਰੋ
4. WPMU DEV ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਨੂੰ ਸਮਰੱਥ ਕਰੋ
5. "Save Changes" 'ਤੇ ਕਲਿੱਕ ਕਰੋ

## ਇਹ ਕਿਵੇਂ ਕੰਮ ਕਰਦਾ ਹੈ

### ਡੋਮੇਨ ਸਿੰਕਿੰਗ

ਜਦੋਂ Ultimate Multisite ਵਿੱਚ ਕੋਈ ਡੋਮੇਨ ਮੈਪ ਕੀਤਾ ਜਾਂਦਾ ਹੈ:

1. ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਤੁਹਾਡੇ ਹੋਸਟਿੰਗ ਖਾਤੇ ਵਿੱਚ ਡੋਮੇਨ ਜੋੜਨ ਲਈ WPMU DEV API ਦੀ ਵਰਤੋਂ ਕਰਦਾ ਹੈ
2. ਇਹ ਆਪਣੇ ਆਪ ਡੋਮੇਨ ਦਾ www ਵਰਜ਼ਨ ਵੀ ਜੋੜ ਦਿੰਦਾ ਹੈ
3. WPMU DEV ਡੋਮੇਨ ਸੰਰਚਨਾ ਅਤੇ SSL ਸਰਟੀਫਿਕੇਟ ਜਾਰੀ ਕਰਨ ਦਾ ਕੰਮ ਸੰਭਾਲਦਾ ਹੈ

### SSL ਸਰਟੀਫਿਕੇਟ ਪ੍ਰਬੰਧਨ

ਇਹ ਇੰਟੀਗ੍ਰੇਸ਼ਨ WPMU DEV ਹੋਸਟਿੰਗ ਲਈ SSL ਸਰਟੀਫਿਕੇਟ ਪੁਸ਼ਟੀ ਕੋਸ਼ਿਸ਼ਾਂ ਦੀ ਗਿਣਤੀ ਵਧਾਉਣ ਲਈ ਸੈੱਟ ਕੀਤਾ ਗਿਆ ਹੈ, ਕਿਉਂਕਿ SSL ਸਰਟੀਫਿਕੇਟ ਜਾਰੀ ਕਰਨ ਅਤੇ ਇੰਸਟਾਲ ਕਰਨ ਵਿੱਚ ਕੁਝ ਸਮਾਂ ਲੱਗ ਸਕਦਾ ਹੈ। ਮੂਲ ਰੂਪ ਵਿੱਚ, ਇਹ SSL ਸਰਟੀਫਿਕੇਟ ਪੁਸ਼ਟੀ ਲਈ 10 ਵਾਰ ਤੱਕ ਕੋਸ਼ਿਸ਼ ਕਰੇਗਾ, ਜਦੋਂ ਕਿ ਮਿਆਰੀ 5 ਕੋਸ਼ਿਸ਼ਾਂ ਹੁੰਦੀਆਂ ਹਨ।

## ਮਹੱਤਵਪੂਰਨ ਨੋਟਸ

### ਡੋਮੇਨ ਹਟਾਉਣਾ

ਇਸ ਵੇਲੇ, WPMU DEV API ਡੋਮੇਨ ਹਟਾਉਣ ਦਾ ਤਰੀਕਾ ਪ੍ਰਦਾਨ ਨਹੀਂ ਕਰਦਾ। ਜਦੋਂ Ultimate Multisite ਵਿੱਚ ਕੋਈ ਡੋਮੇਨ ਮੈਪਿੰਗ ਹਟਾਈ ਜਾਂਦੀ ਹੈ, ਤਾਂ ਡੋਮੇਨ ਤੁਹਾਡੇ WPMU DEV ਹੋਸਟਿੰਗ ਖਾਤੇ ਵਿੱਚ ਰਹੇਗਾ। ਜੇਕਰ ਲੋੜ ਹੋਵੇ ਤਾਂ ਤੁਹਾਨੂੰ ਇਸਨੂੰ WPMU DEV ਹੋਸਟਿੰਗ dashboard ਤੋਂ ਖੁਦ ਹਟਾਉਣਾ ਪਵੇਗਾ।

### API ਪ੍ਰਮਾਣਿਕਤਾ

ਇਹ ਇੰਟੀਗ੍ਰੇਸ਼ਨ WPMU DEV API key ਦੀ ਵਰਤੋਂ ਕਰਦਾ ਹੈ ਜੋ ਤੁਹਾਡੇ WordPress database ਵਿੱਚ `wpmudev_apikey` option ਵਜੋਂ ਸਟੋਰ ਹੁੰਦੀ ਹੈ। ਜਦੋਂ ਤੁਸੀਂ ਆਪਣੀ ਸਾਈਟ ਨੂੰ WPMU DEV ਨਾਲ ਕਨੈਕਟ ਕਰਦੇ ਹੋ ਤਾਂ ਇਹ ਆਪਣੇ ਆਪ ਸੈੱਟ ਹੋ ਜਾਂਦੀ ਹੈ।

## ਸਮੱਸਿਆ ਨਿਵਾਰਨ

### API ਕਨੈਕਸ਼ਨ ਸਮੱਸਿਆਵਾਂ
- ਯਕੀਨੀ ਬਣਾਓ ਕਿ ਤੁਹਾਡੀ ਸਾਈਟ WPMU DEV ਨਾਲ ਸਹੀ ਤਰ੍ਹਾਂ ਜੁੜੀ ਹੋਈ ਹੈ
- ਜਾਂਚ ਕਰੋ ਕਿ `wpmudev_apikey` option ਤੁਹਾਡੇ WordPress database ਵਿੱਚ ਸੈੱਟ ਹੈ
- ਯਕੀਨੀ ਬਣਾਓ ਕਿ ਤੁਹਾਡੀ WPMU DEV ਮੈਂਬਰਸ਼ਿਪ ਸਰਗਰਮ ਹੈ

### SSL ਸਰਟੀਫਿਕੇਟ ਸਮੱਸਿਆਵਾਂ
- WPMU DEV ਨੂੰ SSL ਸਰਟੀਫਿਕੇਟ ਜਾਰੀ ਕਰਨ ਵਿੱਚ ਕੁਝ ਸਮਾਂ ਲੱਗ ਸਕਦਾ ਹੈ (ਆਮ ਤੌਰ 'ਤੇ 5-15 ਮਿੰਟ)
- ਇਹ ਇੰਟੀਗ੍ਰੇਸ਼ਨ SSL ਸਰਟੀਫਿਕੇਟਾਂ ਲਈ 10 ਵਾਰ ਤੱਕ ਜਾਂਚ ਕਰਨ ਲਈ ਸੈੱਟ ਕੀਤਾ ਗਿਆ ਹੈ
- ਜੇਕਰ ਕਈ ਕੋਸ਼ਿਸ਼ਾਂ ਤੋਂ ਬਾਅਦ ਵੀ SSL ਸਰਟੀਫਿਕੇਟ ਜਾਰੀ ਨਹੀਂ ਹੋ ਰਹੇ, ਤਾਂ WPMU DEV ਸਪੋਰਟ ਨਾਲ ਸੰਪਰਕ ਕਰੋ

### ਡੋਮੇਨ ਨਹੀਂ ਜੋੜਿਆ ਗਿਆ
- ਕਿਸੇ ਵੀ ਗਲਤੀ ਸੁਨੇਹਿਆਂ ਲਈ Ultimate Multisite logs ਦੀ ਜਾਂਚ ਕਰੋ
- ਯਕੀਨੀ ਬਣਾਓ ਕਿ ਡੋਮੇਨ ਪਹਿਲਾਂ ਹੀ WPMU DEV ਵਿੱਚ ਨਹੀਂ ਜੋੜਿਆ ਗਿਆ
- ਯਕੀਨੀ ਬਣਾਓ ਕਿ ਤੁਹਾਡਾ WPMU DEV ਹੋਸਟਿੰਗ ਪਲਾਨ ਤੁਹਾਡੇ ਵੱਲੋਂ ਜੋੜੇ ਜਾ ਰਹੇ ਡੋਮੇਨਾਂ ਦੀ ਗਿਣਤੀ ਦਾ ਸਮਰਥਨ ਕਰਦਾ ਹੈ
