# ਚੈੱਕਆਊਟ ਫਾਰਮ

ਚੈੱਕਆਊਟ ਫਾਰਮ ਨਵੇਂ ਗਾਹਕਾਂ ਨੂੰ ਜੋੜਨ ਲਈ ਵੱਖ-ਵੱਖ ਤਰੀਕਿਆਂ ਨਾਲ ਪ੍ਰਯੋਗ ਕਰਨ ਦਾ ਇੱਕ ਸੌਖਾ ਅਤੇ ਲਚਕਦਾਰ ਤਰੀਕਾ ਹੈ।

Ultimate Multisite 2.0 ਵਿੱਚ ਇੱਕ ਚੈੱਕਆਊਟ ਫਾਰਮ ਐਡੀਟਰ ਹੈ ਜੋ ਤੁਹਾਨੂੰ ਜਿੰਨੇ ਚਾਹੋ ਫਾਰਮ ਬਣਾਉਣ ਦਿੰਦਾ ਹੈ, ਵੱਖ-ਵੱਖ ਫੀਲਡਾਂ, ਪੇਸ਼ਕਸ਼ ਵਾਲੇ ਉਤਪਾਦਾਂ, ਆਦਿ ਨਾਲ।

ਇਸ ਫੀਚਰ ਤੱਕ ਪਹੁੰਚਣ ਲਈ, ਖੱਬੇ ਸਾਈਡ-ਬਾਰ 'ਤੇ Checkout Forms ਮੀਨੂ 'ਤੇ ਜਾਓ।

![ਚੈੱਕਆਊਟ ਫਾਰਮਾਂ ਦੀ ਸੂਚੀ](/img/config/checkout-forms-list.png)

ਇਸ ਪੰਨੇ 'ਤੇ, ਤੁਸੀਂ ਆਪਣੇ ਸਾਰੇ ਚੈੱਕਆਊਟ ਫਾਰਮ ਦੇਖ ਸਕਦੇ ਹੋ।

ਜੇ ਤੁਸੀਂ ਨਵਾਂ ਬਣਾਉਣਾ ਚਾਹੁੰਦੇ ਹੋ, ਤਾਂ ਪੰਨੇ ਦੇ ਉੱਪਰ Add Checkout Form 'ਤੇ ਕਲਿੱਕ ਕਰੋ।

ਤੁਸੀਂ ਸ਼ੁਰੂਆਤੀ ਬਿੰਦੂ ਵਜੋਂ ਇਹਨਾਂ ਤਿੰਨ ਵਿਕਲਪਾਂ ਵਿੱਚੋਂ ਕੋਈ ਇੱਕ ਚੁਣ ਸਕਦੇ ਹੋ: single step, multi-step ਜਾਂ blank। ਫਿਰ, ਐਡੀਟਰ 'ਤੇ ਜਾਣ ਲਈ ਕਲਿੱਕ ਕਰੋ।

![ਚੈੱਕਆਊਟ ਫਾਰਮ ਐਡੀਟਰ](/img/config/checkout-form-editor.png)

ਵਿਕਲਪਕ ਤੌਰ 'ਤੇ, ਤੁਸੀਂ ਆਪਣੇ ਮੌਜੂਦਾ ਫਾਰਮਾਂ ਨੂੰ ਉਹਨਾਂ ਦੇ ਨਾਮ ਹੇਠਾਂ ਦਿੱਤੇ ਵਿਕਲਪਾਂ 'ਤੇ ਕਲਿੱਕ ਕਰਕੇ ਸੋਧ ਜਾਂ ਡੁਪਲੀਕੇਟ ਕਰ ਸਕਦੇ ਹੋ। ਉੱਥੇ, ਤੁਹਾਨੂੰ ਫਾਰਮ ਦਾ shortcode ਕਾਪੀ ਕਰਨ ਜਾਂ ਫਾਰਮ ਮਿਟਾਉਣ ਦੇ ਵਿਕਲਪ ਵੀ ਮਿਲਣਗੇ।

![ਚੈੱਕਆਊਟ ਫਾਰਮ ਹੋਵਰ ਐਕਸ਼ਨ](/img/config/checkout-form-hover-actions.png)

### ਚੈੱਕਆਊਟ ਫਾਰਮ ਸੋਧਣਾ

ਤੁਸੀਂ ਵੱਖ-ਵੱਖ ਮਕਸਦਾਂ ਲਈ ਚੈੱਕਆਊਟ ਫਾਰਮ ਬਣਾ ਸਕਦੇ ਹੋ। ਇਸ ਉਦਾਹਰਨ ਵਿੱਚ ਅਸੀਂ ਇੱਕ ਰਜਿਸਟ੍ਰੇਸ਼ਨ ਫਾਰਮ 'ਤੇ ਕੰਮ ਕਰਾਂਗੇ।

ਚੈੱਕਆਊਟ ਫਾਰਮ ਐਡੀਟਰ 'ਤੇ ਜਾਣ ਤੋਂ ਬਾਅਦ, ਆਪਣੇ ਫਾਰਮ ਨੂੰ ਇੱਕ ਨਾਮ ਦਿਓ (ਜੋ ਸਿਰਫ਼ ਅੰਦਰੂਨੀ ਹਵਾਲੇ ਲਈ ਵਰਤਿਆ ਜਾਵੇਗਾ) ਅਤੇ ਇੱਕ slug (ਜੋ shortcodes ਬਣਾਉਣ ਲਈ ਵਰਤਿਆ ਜਾਂਦਾ ਹੈ)।

![ਚੈੱਕਆਊਟ ਫਾਰਮ ਐਡੀਟਰ](/img/config/checkout-form-editor.png)

ਫਾਰਮ ਸਟੈਪਾਂ ਅਤੇ ਫੀਲਡਾਂ ਨਾਲ ਬਣੇ ਹੁੰਦੇ ਹਨ। ਤੁਸੀਂ Add New Checkout Step 'ਤੇ ਕਲਿੱਕ ਕਰਕੇ ਨਵਾਂ ਸਟੈਪ ਜੋੜ ਸਕਦੇ ਹੋ।

![ਨਵਾਂ ਚੈੱਕਆਊਟ ਸਟੈਪ ਜੋੜੋ](/img/config/checkout-form-add-step.png)

ਮੋਡਲ ਵਿੰਡੋ ਦੀ ਪਹਿਲੀ ਟੈਬ 'ਤੇ, ਆਪਣੇ ਫਾਰਮ ਦੇ ਸਟੈਪ ਦੀ ਸਮੱਗਰੀ ਭਰੋ। ਇਸਨੂੰ ਇੱਕ ID, ਇੱਕ ਨਾਮ ਅਤੇ ਇੱਕ ਵਰਣਨ ਦਿਓ। ਇਹ ਚੀਜ਼ਾਂ ਜ਼ਿਆਦਾਤਰ ਅੰਦਰੂਨੀ ਤੌਰ 'ਤੇ ਵਰਤੀਆਂ ਜਾਂਦੀਆਂ ਹਨ।

![ਚੈੱਕਆਊਟ ਫਾਰਮ ਸਟੈਪ](/img/config/checkout-form-step.png)

ਅੱਗੇ, ਸਟੈਪ ਦੀ ਦਿੱਖ ਸੈੱਟ ਕਰੋ। ਤੁਸੀਂ "Always show", "Only show for logged in users" ਜਾਂ "Only show for guests" ਵਿੱਚੋਂ ਚੁਣ ਸਕਦੇ ਹੋ।

![ਚੈੱਕਆਊਟ ਫਾਰਮ ਸਟੈਪ](/img/config/checkout-form-step.png)

ਅੰਤ ਵਿੱਚ, ਸਟੈਪ ਦੀ ਸਟਾਈਲ ਸੈੱਟ ਕਰੋ। ਇਹ ਵਿਕਲਪਿਕ ਫੀਲਡਾਂ ਹਨ।

![ਚੈੱਕਆਊਟ ਫਾਰਮ ਸਟੈਪ](/img/config/checkout-form-step.png)

ਹੁਣ, ਸਾਡੇ ਪਹਿਲੇ ਸਟੈਪ ਵਿੱਚ ਫੀਲਡਾਂ ਜੋੜਨ ਦਾ ਸਮਾਂ ਹੈ। ਬੱਸ Add New Field 'ਤੇ ਕਲਿੱਕ ਕਰੋ ਅਤੇ ਉਹ ਸੈਕਸ਼ਨ ਦੀ ਕਿਸਮ ਚੁਣੋ ਜੋ ਤੁਸੀਂ ਚਾਹੁੰਦੇ ਹੋ।

![ਫੀਲਡਾਂ ਨਾਲ ਚੈੱਕਆਊਟ ਫਾਰਮ ਸਟੈਪ](/img/config/checkout-form-step.png)

ਹਰ ਫੀਲਡ ਦੇ ਵੱਖ-ਵੱਖ ਪੈਰਾਮੀਟਰ ਹੁੰਦੇ ਹਨ ਜੋ ਭਰਨੇ ਪੈਂਦੇ ਹਨ। ਇਸ ਪਹਿਲੀ ਐਂਟਰੀ ਲਈ, ਅਸੀਂ "Username" ਫੀਲਡ ਚੁਣਾਂਗੇ।

![ਚੈੱਕਆਊਟ ਫਾਰਮ ਸਟੈਪ](/img/config/checkout-form-step.png)

![ਚੈੱਕਆਊਟ ਫਾਰਮ ਸਟੈਪ](/img/config/checkout-form-step.png)

![ਚੈੱਕਆਊਟ ਫਾਰਮ ਸਟੈਪ](/img/config/checkout-form-step.png)

ਤੁਸੀਂ ਜਿੰਨੇ ਚਾਹੋ ਸਟੈਪ ਅਤੇ ਫੀਲਡਾਂ ਜੋੜ ਸਕਦੇ ਹੋ। ਆਪਣੇ ਗਾਹਕਾਂ ਨੂੰ ਚੁਣਨ ਲਈ ਆਪਣੇ ਉਤਪਾਦ ਦਿਖਾਉਣ ਲਈ, Pricing Table ਫੀਲਡ ਵਰਤੋ। ਜੇ ਤੁਸੀਂ ਆਪਣੇ ਗਾਹਕਾਂ ਨੂੰ ਟੈਮਪਲੇਟ ਚੁਣਨ ਦੇਣਾ ਚਾਹੁੰਦੇ ਹੋ, ਤਾਂ Template Selection ਫੀਲਡ ਜੋੜੋ। ਅਤੇ ਇਸੇ ਤਰ੍ਹਾਂ।

_**ਨੋਟ:** ਚੈੱਕਆਊਟ ਫਾਰਮ ਬਣਾਉਣ ਲਈ username, email, password, site title, site URL, order summary, payment, ਅਤੇ submit button ਲਾਜ਼ਮੀ ਫੀਲਡਾਂ ਹਨ।_

ਜਦੋਂ ਤੁਸੀਂ ਆਪਣੇ ਚੈੱਕਆਊਟ ਫਾਰਮ 'ਤੇ ਕੰਮ ਕਰ ਰਹੇ ਹੋ, ਤੁਸੀਂ ਹਮੇਸ਼ਾ Preview ਬਟਨ ਵਰਤ ਕੇ ਦੇਖ ਸਕਦੇ ਹੋ ਕਿ ਤੁਹਾਡੇ ਗਾਹਕ ਫਾਰਮ ਨੂੰ ਕਿਵੇਂ ਦੇਖਣਗੇ। ਤੁਸੀਂ ਮੌਜੂਦਾ ਯੂਜ਼ਰ ਜਾਂ ਵਿਜ਼ਿਟਰ ਵਜੋਂ ਦੇਖਣ ਵਿਚਕਾਰ ਵੀ ਬਦਲ ਸਕਦੇ ਹੋ।

![ਚੈੱਕਆਊਟ ਫਾਰਮ ਸੇਵ](/img/config/checkout-form-save.png)

![ਚੈੱਕਆਊਟ ਫਾਰਮ ਐਡੀਟਰ](/img/config/checkout-form-editor.png)

ਅੰਤ ਵਿੱਚ, Advanced Options ਵਿੱਚ ਤੁਸੀਂ "Thank You" ਪੰਨੇ ਲਈ ਸੁਨੇਹਾ ਸੈੱਟ ਕਰ ਸਕਦੇ ਹੋ, ਕਨਵਰਜ਼ਨ ਟ੍ਰੈਕ ਕਰਨ ਲਈ snippets ਜੋੜ ਸਕਦੇ ਹੋ, ਆਪਣੇ ਚੈੱਕਆਊਟ ਫਾਰਮ ਵਿੱਚ ਕਸਟਮ CSS ਜੋੜ ਸਕਦੇ ਹੋ ਜਾਂ ਇਸਨੂੰ ਕੁਝ ਖਾਸ ਦੇਸ਼ਾਂ ਤੱਕ ਸੀਮਤ ਕਰ ਸਕਦੇ ਹੋ।

![ਐਡਵਾਂਸਡ ਵਿਕਲਪ](/img/config/checkout-form-advanced.png)

ਤੁਸੀਂ ਸੱਜੇ ਕਾਲਮ ਵਿੱਚ ਇਸ ਵਿਕਲਪ ਨੂੰ ਟੌਗਲ ਕਰਕੇ ਆਪਣੇ ਚੈੱਕਆਊਟ ਫਾਰਮ ਨੂੰ ਹੱਥੀਂ ਚਾਲੂ ਜਾਂ ਬੰਦ ਵੀ ਕਰ ਸਕਦੇ ਹੋ, ਜਾਂ ਫਾਰਮ ਨੂੰ ਪੱਕੇ ਤੌਰ 'ਤੇ ਮਿਟਾ ਸਕਦੇ ਹੋ।

![ਐਕਟਿਵ ਟੌਗਲ](/img/config/checkout-form-active.png)

ਆਪਣਾ ਚੈੱਕਆਊਟ ਫਾਰਮ ਸੇਵ ਕਰਨਾ ਨਾ ਭੁੱਲੋ!

![ਸੇਵ ਬਟਨ](/img/config/checkout-form-save.png)

ਆਪਣੇ ਫਾਰਮ ਦਾ shortcode ਲੈਣ ਲਈ Generate Shortcode 'ਤੇ ਕਲਿੱਕ ਕਰੋ ਅਤੇ ਮੋਡਲ ਵਿੰਡੋ ਵਿੱਚ ਦਿਖਾਈ ਦੇਣ ਵਾਲੇ ਨਤੀਜੇ ਨੂੰ ਕਾਪੀ ਕਰੋ।

![shortcode ਨਾਲ ਸੇਵ ਬਟਨ](/img/config/checkout-form-save.png)
