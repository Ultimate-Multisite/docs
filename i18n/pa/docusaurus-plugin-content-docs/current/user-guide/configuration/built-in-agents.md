# ਮੌਜੂਦ 에이ਜెంట్ (Built-in Agents)

Gratis AI Agent v1.9.0 ਵਿੱਚ ਪੰਜ ਮੌਜੂਦ 에이ਜెంట్ ਸ਼ਾਮਲ ਹਨ। ਹਰੇਕ 에이ਜెంట్ ਨੂੰ ਇੱਕ ਖਾਸ ਤਸਵੀਰ (focused set of tools), ਇੱਕ ਤਿਆਰ ਸਿਸਟਮ ਪ੍ਰੌਮਟ (tailored system prompt), ਅਤੇ ਉਸ ਖੇਤਰ ਨਾਲ ਮਿਲਦੇ-ਜੁਲਦੇ ਸ਼ੁਰੂਆਤੀ ਸੁਝਾਅ (starter suggestions) ਨਾਲ ਪਹਿਲਾਂ ਹੀ ਕਨਫਿਗਰ ਕੀਤਾ ਗਿਆ ਹੈ। 에이ਜెంట్ ਬਦਲਣ ਨਾਲ ਅਸਿਸਟੈਂਟ ਕੀ ਕਰ ਸਕਦਾ ਹੈ ਅਤੇ ਕਿਵੇਂ ਜਵਾਬ ਦੇਵੇਗਾ, ਇਹ ਬਦਲ ਜਾਂਦਾ ਹੈ — ਅਤੇ ਇਹ ਸਭ ਤੁਹਾਡੇ ਵੱਲੋਂ ਕੋਈ ਕਨਫਿਗਰੇਸ਼ਨ ਕੀਤੇ ਬਿਨਾਂ।

## 에이ਜెంట్ ਕੀ ਹੈ? (What Is an Agent?)

ਹਰੇਕ 에이ਜెంట్ ਇੱਕ ਨਾਮੀ ਕਨਫਿਗਰੇਸ਼ਨ ਪ੍ਰੋਫਾਈਲ ਹੁੰਦੀ ਹੈ ਜੋ ਇਹ ਚੀਜ਼ਾਂ ਜੋੜਦੀ ਹੈ:

- **ਟੂਲਸ (Tools)** — ਉਹ ਯੋਗਤਾਵਾਂ (abilities) ਜੋ 에이ਜెంట్ ਨੂੰ ਵਰਤਣ ਦੀ ਇਜਾਜ਼ਤ ਹੈ (ਉਦਾਹਰਨ ਲਈ, ਇੱਕ Content Writer ਕੋਲ ਪੋਸਟ ਬਣਾਉਣ ਦੀ ਯੋਗਤਾ ਹੁੰਦੀ ਹੈ; ਇੱਕ Design Studio ਕੋਲ CSS ਅਤੇ theme.json ਦੀਆਂ ਯੋਗਤਾਵਾਂ ਹੁੰਦੀਆਂ ਹਨ)
- **ਸਿਸਟਮ ਪ੍ਰੌਮਟ (System prompt)** — ਨਿਰਦੇਸ਼ ਜੋ 에이ਜੈਂਟ ਦੇ ਟੋਨ, ਤਰਜੀਹਾਂ ਅਤੇ ਸੀਮਾਵਾਂ ਨੂੰ ਸੈੱਟ ਕਰਦੇ ਹਨ
- **ਸੁਝਾਅ (Suggestions)** — ਚੈਟ ਇੰਟਰਫੇਸ ਵਿੱਚ ਦਿਖਾਏ ਗਏ ਪਹਿਲਾਂ ਤੋਂ ਲਿਖੇ ਹੋਏ ਪ੍ਰੌਮਟ, ਜੋ ਤੁਹਾਨੂੰ ਜਲਦੀ ਸ਼ੁਰੂਆਤ ਕਰਨ ਵਿੱਚ ਮਦਦ ਕਰਦੇ ਹਨ

## 에이ਜెంట్ ਪਿਕਰ ਤੱਕ ਪਹੁੰਚ ਕਰਨਾ (Accessing the Agent Picker)

1. WordPress ਐਡਮਿਨ ਸਾਈਡਬਾਰ ਵਿੱਚ **Gratis AI Agent** ਪੈਨਲ ਖੋਲ੍ਹੋ।
2. ਚੈਟ ਹੈਡਰ ਦੇ ਉੱਪਰ-ਖੱਡੇ ਕੋਨੇ ਵਿੱਚ **에이ਜెంట్ ਆਈਕਨ** 'ਤੇ ਕਲਿੱਕ ਕਰੋ (ਇਹ ਆਈਕਨ ਸਰਗਰਮ 에이ਜੈਂਟ ਨੂੰ ਦਰਸਾਉਣ ਲਈ ਬਦਲ ਜਾਂਦਾ ਹੈ)।
3. **에이ਜెంట్ ਪਿਕਰ (Agent Picker)** ਇੱਕ ਫਾਰਮ-ਟੇਬਲ ਓਵਰਲੇ ਵਜੋਂ ਖੁੱਲ੍ਹਦਾ ਹੈ। ਹਰੇਕ 에이ਜੈਂਟ ਨੂੰ ਉਸਦੇ ਆਈਕਨ, ਨਾਮ ਅਤੇ ਇੱਕ ਲਾਈਨ ਦੇ ਵੇਰਵੇ ਨਾਲ ਸੂਚੀਬੱਧ ਕੀਤਾ ਗਿਆ ਹੈ।
4. ਇਸਨੂੰ ਐਕਟੀਵੇਟ ਕਰਨ ਲਈ ਕਿਸੇ 에이ਜੈਂਟ ਰੋ (row) 'ਤੇ ਕਲਿੱਕ ਕਰੋ। ਚੈਟ ਹੈਡਰ ਤੁਰੰਤ ਅਪਡੇਟ ਹੋ ਜਾਂਦਾ ਹੈ।

ਤੁਸੀਂ ਗੱਲਬਾਤ ਦੌਰਾਨ ਵੀ 에이ਜੈਂਟ ਬਦਲ ਸਕਦੇ ਹੋ — ਨਵੇਂ 에이ਜੈਂਟ ਦਾ ਸਿਸਟਮ ਪ੍ਰੌਮਟ ਅਗਲੇ ਸੁਨੇਹੇ ਤੋਂ ਲਾਗੂ ਹੋ ਜਾਂਦਾ ਹੈ।

## ਪੰਜ ਮੌਜੂਦ 에이ਜెంట్ (The Five Built-in Agents)

### Content Writer

**ਮੁੱਖ ਫੋਕਸ:** ਪੋਸਟਾਂ, ਪੇਜਾਂ ਅਤੇ ਸੰਪਰਕ ਫਾਰਮਾਂ ਨੂੰ ਬਣਾਉਣਾ ਅਤੇ ਸੰਪਾਦਿਤ ਕਰਨਾ।

**ਉਪਲਬਧ ਟੂਲਸ:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`

**ਇਹ ਕੀ ਚੰਗੀ ਤਰ੍ਹਾਂ ਕਰਦਾ ਹੈ:**
- ਇੱਕ ਸੰਖੇਪ ਰੂਪਰੇਖਾ ਜਾਂ ਆਊਟਲਾਈਨ ਤੋਂ ਬਲੌਗ ਪੋਸਟ ਡਰਾਫਟ ਕਰਨਾ ਅਤੇ ਪਬਲਿਸ਼ ਕਰਨਾ
- ਇੱਕ ਨਵੀਂ ਸਾਈਟ ਲਈ ਲੈਂਡਿੰਗ ਪੇਜਾਂ ਦੇ ਬੈਚ ਬਣਾਉਣਾ
- ਸੰਪਰਕ ਅਤੇ ਪੁੱਛਗਿੱਛ ਫਾਰਮ ਬਣਾਉਣਾ
- URL ਜਾਂ ਸਰਚ ਤੋਂ ਪੋਸਟਾਂ 'ਤੇ ਫੀਚਰਡ ਇਮੇਜ ਸੈੱਟ ਕਰਨਾ

**ਸ਼ੁਰੂਆਤੀ ਸੁਝਾਅ:**
- *WordPress multisite ਦੇ ਲਾਭਾਂ ਬਾਰੇ 500-ਸ਼ਬਦ ਦਾ ਬਲੌਗ ਪੋਸਟ ਲਿਖੋ।*
- *ਇੱਕ About, Services, ਅਤੇ Contact ਪੇਜ ਬਣਾਓ ਅਤੇ ਪਬਲਿਸ਼ ਕਰੋ।*
- *Contact ਪੇਜ 'ਤੇ ਇੱਕ ਬੁਕਿੰਗ ਪੁੱਛਗਿੱਛ ਫਾਰਮ ਜੋੜੋ।*

---

### Site Builder

**ਮੁੱਖ ਫੋਕਸ:** ਇੱਕ ਸਿੰਗਲ ਪ੍ਰੌਮਟ ਤੋਂ ਸ਼ੁਰੂ ਕਰਕੇ ਪੂਰੀ ਵੈੱਬਸਾਈਟ ਬਣਾਉਣਾ।

**ਉਪਲਬਧ ਟੂਲਸ:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`

**ਇਹ ਕੀ ਚੰਗੀ ਤਰ੍ਹਾਂ ਕਰਦਾ ਹੈ:**
- ਵਰਣਨ ਕੀਤੇ ਗਏ ਕਾਰੋਬਾਰੀ ਕਿਸਮ ਲਈ ਇੱਕ ਬਹੁ-ਪੜਾਅ ਵਾਲੀ ਸਾਈਟ ਬਿਲਡ ਪਲਾਨ ਤਿਆਰ ਕਰਨਾ
- ਹਰੇਕ ਪੜਾਅ ਨੂੰ ਆਟੋਨੋਮਸ ਤੌਰ 'ਤੇ ਕਾਰਜ ਕਰਨਾ — cấu trúc, ਸਮੱਗਰੀ, ਨੈਵੀਗੇਸ਼ਨ, ਡਿਜ਼ਾਈਨ
- ਮਨੁੱਖੀ ਹਮਲ干ਗਤ ਦਖਲਅੰਦੇਸ਼ ਤੋਂ ਬਿਨਾਂ ਪਲਾਨ ਦੌਰਾਨ ਗਲਤੀਆਂ ਤੋਂ ਸੁਧਾਰ ਕਰਨਾ
- ਬਿਲਡ ਦੇ ਹਿੱਸੇ ਵਜੋਂ ਸਿਫਾਰਸ਼ ਕੀਤੇ ਪਲੱਗਇਨ ਸਥਾਪਤ ਕਰਨਾ
- ਚੈਟ ਇੰਟਰਫੇਸ ਤੋਂ ਸਿੱਧੇ ਤੌਰ 'ਤੇ ਸੰਪਰਕ ਫਾਰਮ ਬਣਾਉਣਾ (Superdav AI Agent v1.10.0+)

**ਸ਼ੁਰੂਆਤੀ ਸੁਝਾਅ:**
- *ਇੱਕ ਗੈਲਰੀ ਪੋਸਟ ਟਾਈਪ, ਇੱਕ ਬੁਕਿੰਗ ਪੇਜ, ਅਤੇ ਇੱਕ ਸੰਪਰਕ ਫਾਰਮ ਨਾਲ ਫੋਟੋਗ੍ਰਾਫੀ ਪੋਰਟਫੋਲੀਓ ਸਾਈਟ ਬਣਾਓ।*
- *ਇੱਕ ਆਨਲਾਈਨ ਮੀਨੂ, ਖੁੱਲ੍ਹਣ ਦੇ ਘੰਟੇ, ਅਤੇ ਟੇਬਲ-ਬੁਕਿੰਗ ਪੁੱਛਗਿੱਛ ਫਾਰਮ ਵਾਲੀ ਰੈਸਟੋਰੈਂਟ ਵੈੱਬਸਾਈਟ ਬਣਾਓ।*
- *ਸਰਵਿਸ ਪੇਜਾਂ, ਇੱਕ ਪੋਰਟਫੋਲੀਓ ਸੈਕਸ਼ਨ, ਅਤੇ ਇੱਕ ਬਲੌਗ ਨਾਲ ਇੱਕ ਫ੍ਰੀਲੈਂਸ ਕੰਸਲਟਿੰਗ ਸਾਈਟ ਸੈੱਟ ਕਰੋ।*
- *ਸਾਈਟ ਬਿਲਡਰ ਦੀ ਵਰਤੋਂ ਕਰਕੇ Contact ਪੇਜ 'ਤੇ ਇੱਕ ਸੰਪਰਕ ਫਾਰਮ ਜੋੜੋ।*

### Design Studio

**ਮੁੱਖ ਫੋਕਸ:** ਵਿਜ਼ੂਅਲ ਕਸਟਮਾਈਜ਼ੇਸ਼ਨ — ਰੰਗ, ਟਾਈਪੋਗ੍ਰਾਫੀ, CSS, ਅਤੇ ਬਲਾਕ ਪੈਟਰਨ।

**ਉਪਲਬਧ ਟੂਲਸ:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**ਇਹ ਕੀ ਚੰਗੀ ਤਰ੍ਹਾਂ ਕਰਦਾ ਹੈ:**
- ਨਾਮਾਂ ਵਾਲੇ تھیਮ ਪ੍ਰੀਸੈਟ ਲਾਗੂ ਕਰਨਾ (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- theme.json ਰਾਹੀਂ ਗਲੋਬਲ ਟਾਈਪੋਗ੍ਰਾਫੀ ਅਤੇ ਰੰਗ ਪੈਲੇਟ ਨੂੰ ਫਾਈਨ-ਟਿਊਨ ਕਰਨਾ
- ਬ੍ਰਾਂਡ-ਵਿਸ਼ੇਸ਼ ਓਵਰਰਾਈਡ ਲਈ ਕਸਟਮ CSS ਇੰਜੈਕਟ ਕਰਨਾ
- ਇੱਕ ਪੇਜ ਦਾ ਸਕ੍ਰੀਨਸ਼ਾਟ ਲੈਣਾ ਅਤੇ ਡਿਜ਼ਾਈਨ ਦੀਆਂ ਸਮੱਸਿਆਵਾਂ ਲਈ ਇਸਦੀ ਸਮੀਖਿਆ ਕਰਨਾ

**ਸ਼ੁਰੂਆਤੀ ਸੁਝਾਅ:**
- *warm-editorial ਪ੍ਰੀਸੈਟ ਲਾਗੂ ਕਰੋ ਅਤੇ ਫਿਰ ਪ੍ਰਾਇਮਰੀ ਰੰਗ ਨੂੰ #2d6a4f ਸੈੱਟ ਕਰੋ।*
- *ਹੋਮਪੇਜ ਦਾ ਸਕ੍ਰੀਨਸ਼ਾਟ ਲਓ ਅਤੇ ਮੈਨੂੰ ਦੱਸੋ ਕਿ ਤੁਸੀਂ ਕੀ ਸੁਧਾਰ ਕਰੋਗੇ।*
- *ਇੱਕ ਫੁੱਲ-ਵਾਈਡਥ ব্যাকਗ੍ਰਾਉਂਡ ਇਮੇਜ ਅਤੇ ਸੈਂਟਰਡ ਹੈਡਿੰਗ ਨਾਲ ਇੱਕ ਦੁਬਾਰਾ ਵਰਤਣ ਯੋਗ ਹੀਰੋ ਬਲਾਕ ਪੈਟਰਨ ਬਣਾਓ।*

### Plugin Manager

**ਮੁੱਖ ਫੋਕਸ:** WordPress ਪਲੱਗਇਨ ਲੱਭਣਾ, ਸਥਾਪਤ ਕਰਨਾ ਅਤੇ ਪ੍ਰਬੰਧਿਤ ਕਰਨਾ।

**ਉਪਲਬਧ ਟੂਲਸ:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**ਇਹ ਕੀ ਚੰਗੀ ਤਰ੍ਹਾਂ ਕਰਦਾ ਹੈ:**
- ਵਰਣਨ ਕੀਤੇ ਗਏ ਵਰਤੋਂ ਕੇਸ ਲਈ ਸਭ ਤੋਂ ਵਧੀਆ ਪਲੱਗਇਨ ਸਿਫਾਰਸ਼ ਕਰਨਾ
- ਰਜਿਸਟਰੀ ਤੋਂ ਯੋਗਤਾ ਪੈਕ ਸਥਾਪਤ ਕਰਨਾ
- ਸ਼੍ਰੇਣੀ ਅਨੁਸਾਰ ਉਪਲਬਧ ਯੋਗਤਾ ਕੈਟਾਲਾਗ ਨੂੰ ਵੇਖਣਾ

**ਸ਼ੁਰੂਆਤੀ ਸੁਝਾਅ:**
- *ਮੈਂਬਰਸ਼ਿਪ ਡਾਇਰੈਕਟਰੀ ਲਈ ਸਭ ਤੋਂ ਵਧੀਆ ਪਲੱਗਇਨ ਕਿਹੜਾ ਹੈ?*
- *WooCommerce ਯੋਗਤਾ ਪੈਕ ਸਥਾਪਤ ਕਰੋ।*
- *ਮੈਨੂੰ ਸਾਰੇ ਉਪਲਬਧ e-commerce ਯੋਗਤਾ ਪੈਕ ਦਿਖਾਓ।*

### Support Assistant

**ਮੁੱਖ ਫੋਕਸ:** ਸਾਈਟ ਦੀ ਸਮੱਗਰੀ, ਸੈਟਿੰਗਾਂ, ਅਤੇ WordPress ਕਨਫਿਗਰੇਸ਼ਨ ਬਾਰੇ ਸਵਾਲਾਂ ਦੇ ਜਵਾਬ ਦੇਣਾ।

**ਉਪਲਬਧ ਟੂਲਸ:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**ਇਹ ਕੀ ਚੰਗੀ ਤਰ੍ਹਾਂ ਕਰਦਾ ਹੈ:**
- ਮੌਜੂਦਾ ਸਾਈਟ ਸੈਟਿੰਗਾਂ ਅਤੇ ਵਿਕਲਪਾਂ ਨੂੰ ਦੇਖਣਾ
- ਇਹ ਸਮਝਾਉਣਾ ਕਿ ਸਾਈਟ 'ਤੇ ਕਿਹੜੇ ਪੋਸਟ ਟਾਈਪ, ਟੈਕਸੋਨੋਮੀ, ਅਤੇ ਮੀਨੂ ਕਨਫਿਗਰ ਕੀਤੇ ਗਏ ਹਨ
- ਲਾਈਵ ਵੈਲਿਊ ਪੜ੍ਹ ਕੇ "ਇਹ ਸੈਟਿੰਗ ਕੀ ਕਰਦੀ ਹੈ?" ਸਵਾਲਾਂ ਦੇ ਜਵਾਬ ਦੇਣਾ
- ਕੋਈ ਬਦਲਾਅ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ ਇੱਕ ਰੀਡ-ਓਨਲੀ ਡਾਇਗਨੋਸਟਿਕ ਲੇਅਰ ਵਜੋਂ ਕੰਮ ਕਰਨਾ

**ਸ਼ੁਰੂਆਤੀ ਸੁਝਾਅ:**
- *ਇਸ ਸਾਈਟ 'ਤੇ ਇਸ ਵੇਲੇ ਕਿਹੜੇ ਪਲੱਗਇਨ ਅਤੇ ਸੈਟਿੰਗਾਂ ਐਕਟਿਵ ਹਨ?*
- *ਇਸ ਸਾਈਟ 'ਤੇ ਰਜਿਸਟਰ ਕੀਤੇ ਸਾਰੇ ਕਸਟਮ ਪੋਸਟ ਟਾਈਪ ਸੂਚੀਬੱਧ ਕਰੋ।*
- *ਕਿਹੜੇ ਨੈਵੀਗੇਸ਼ਨ ਮੀਨੂ ਮੌਜੂਦ ਹਨ ਅਤੇ ਉਹ ਕਿੱਥੇ ਐਸਾਈਨ ਕੀਤੇ ਗਏ ਹਨ?*

## 에이ਜెంట్ ਨੂੰ ਕਸਟਮਾਈਜ਼ ਕਰਨਾ (Customising Agents)

ਹਰੇਕ ਮੌਜੂਦ 에이ਜੈਂਟ ਨੂੰ `gratis_ai_agent_agents` ਫਿਲਟਰ ਰਾਹੀਂ ਵਧਾਇਆ ਜਾਂ ਬਦਲਿਆ ਜਾ ਸਕਦਾ ਹੈ।

### ਇੱਕ ਮੌਜੂਦ 에이ਜੈਂਟ ਵਿੱਚ ਕਸਟਮ ਸਿਸਟਮ ਪ੍ਰੌਮਟ ਜੋੜਨਾ

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### ਇੱਕ ਨਵਾਂ 에이ਜੈਂਟ ਰਜਿਸਟਰ ਕਰਨਾ

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
```
