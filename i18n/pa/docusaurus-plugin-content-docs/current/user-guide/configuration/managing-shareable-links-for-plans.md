# ਪਲਾਨਾਂ ਲਈ ਸ਼ੇਅਰ ਕਰਨਯੋਗ ਲਿੰਕਾਂ ਦਾ ਪ੍ਰਬੰਧਨ

Ultimate Multisite ਤੁਹਾਨੂੰ ਆਪਣੇ ਗਾਹਕਾਂ ਨਾਲ ਪਲਾਨ ਲਿੰਕ ਸਿੱਧੇ ਸ਼ੇਅਰ ਕਰਨ ਦਾ ਵਿਕਲਪ ਦਿੰਦਾ ਹੈ। ਇਹ ਕਰਨ ਲਈ, ਆਪਣੇ ਕਿਸੇ ਵੀ ਪਲਾਨ ਦੇ ਐਡਿਟ ਪੇਜ 'ਤੇ ਜਾਓ ਅਤੇ _Click to copy the Shareable Link_ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।

![ਸ਼ੇਅਰ ਕਰਨਯੋਗ ਲਿੰਕ ਬਟਨ ਵਾਲਾ ਪ੍ਰੋਡਕਟ ਐਡਿਟ ਪੇਜ](/img/config/product-edit-full.png)

_ਪਲਾਨ ਦੇ ਐਡਿਟ ਪੇਜ 'ਤੇ, ਤੁਸੀਂ ਸ਼ੇਅਰ ਕਰਨਯੋਗ ਲਿੰਕ ਬਣਾਉਣ ਲਈ ਬਟਨ ਲੱਭ ਸਕਦੇ ਹੋ_

ਧਿਆਨ ਦਿਓ ਕਿ ਇਹ ਵਿਧੀ ਲਿੰਕ ਬਣਾਉਣ ਲਈ ਡਿਫਾਲਟ ਬਿਲਿੰਗ ਫ੍ਰੀਕੁਐਂਸੀ ਵਰਤਦੀ ਹੈ। ਇਸਦਾ ਮਤਲਬ ਹੈ ਕਿ ਜੇਕਰ ਤੁਹਾਡੀ ਡਿਫਾਲਟ ਬਿਲਿੰਗ ਮਹੀਨਾਵਾਰ ਹੈ, ਉਦਾਹਰਣ ਵਜੋਂ, ਅੰਤ ਵਿੱਚ ਤੁਹਾਨੂੰ ਇਸ ਤਰ੍ਹਾਂ ਦਾ ਲਿੰਕ ਮਿਲੇਗਾ: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

ਜੇਕਰ ਤੁਸੀਂ ਚਾਹੁੰਦੇ ਹੋ ਕਿ ਸ਼ੇਅਰ ਕਰਨਯੋਗ ਲਿੰਕ ਤਿਮਾਹੀ 'ਤੇ ਡਿਫਾਲਟ ਹੋਵੇ, ਤਾਂ ਤੁਹਾਨੂੰ ਅੰਤ ਵਿੱਚ 3 ਜੋੜਨਾ ਪਵੇਗਾ

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

_premium_ ਪਲਾਨ ਨੂੰ ਸਾਲਾਨਾ ਅਨੁਸੂਚੀ 'ਤੇ ਚੁਣਨ ਲਈ, ਤੁਸੀਂ 1 ਜੋੜੋਗੇ ਅਤੇ ਫਿਰ ਸ਼ਬਦ:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

ਯਾਦ ਰੱਖੋ ਕਿ ਤੁਸੀਂ ਵੱਖ-ਵੱਖ ਬਿਲਿੰਗ ਫ੍ਰੀਕੁਐਂਸੀਆਂ ਲਈ ਸਿਰਫ਼ ਇੱਕ ਲਿੰਕ ਨਹੀਂ ਰੱਖ ਸਕਦੇ। ਤੁਹਾਨੂੰ ਆਪਣੀ ਹਰੇਕ ਬਿਲਿੰਗ ਫ੍ਰੀਕੁਐਂਸੀ ਲਈ ਇੱਕ ਖਾਸ ਲਿੰਕ ਸ਼ੇਅਰ ਕਰਨਾ ਚਾਹੀਦਾ ਹੈ।

ਜਦੋਂ ਤੁਸੀਂ ਆਪਣੇ ਗਾਹਕਾਂ ਨਾਲ ਲਿੰਕ ਸ਼ੇਅਰ ਕਰਦੇ ਹੋ, ਇਹ ਉਨ੍ਹਾਂ ਨੂੰ ਤੁਹਾਡੇ ਸਾਈਨਅੱਪ ਫਲੋ ਦੇ ਪਹਿਲੇ ਕਦਮ 'ਤੇ ਰੀਡਾਇਰੈਕਟ ਕਰ ਦੇਵੇਗਾ।
