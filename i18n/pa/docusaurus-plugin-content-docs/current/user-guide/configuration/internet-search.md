# ਇੰਟਰਨੈਟ ਸਰਚ

Gratis AI Agent v1.5.0 ਇੱਕ **Internet Search** ਯੋਗਤਾ ਜੋੜਦਾ ਹੈ, ਜਿਸ ਨਾਲ AI ਅਸਿਸਟੈਂਟ ਗੱਲਬਾਤ ਦੌਰਾਨ ਵੈੱਬ ਤੋਂ ਤਾਜ਼ਾ ਜਾਣਕਾਰੀ ਕੱਢ ਸਕਦਾ ਹੈ। ਇਹ [Brave Search API](https://brave.com/search/api/) ਜਾਂ [Tavily API](https://tavily.com/) ਦੁਆਰਾ ਚਲਾਇਆ ਜਾਂਦਾ ਹੈ।

## ਇਹ ਕਿਵੇਂ ਕੰਮ ਕਰਦਾ ਹੈ {#how-it-works}

ਜਦੋਂ ਇੰਟਰਨੈਟ ਸਰਚ ਸਥਾਪਿਤ (enabled) ਹੁੰਦਾ ਹੈ, ਤਾਂ ਅਸਿਸਟੈਂਟ ਆਪਣੇ ਸੈਟ ਕੀਤੇ ਸਰਚ ਪ੍ਰੋਵਾਈਡਰ ਨੂੰ ਆਪਣੇ ਆਪ ਕੁਐਰੀ ਕਰ ਸਕਦਾ ਹੈ ਜਦੋਂ ਉਸਨੂੰ ਲੱਗਦਾ ਹੈ ਕਿ ਕਿਸੇ ਸਵਾਲ ਲਈ ਮੌਜੂਦਾ ਜਾਂ ਬਾਹਰੀ ਜਾਣਕਾਰੀ ਦੀ ਲੋੜ ਹੈ — ਜਿਵੇਂ ਕਿ ਹਾਲੀਆ ਖ਼ਬਰਾਂ, ਲਾਈਵ ਕੀਮਤਾਂ, ਜਾਂ ਦਸਤਾਵੇਜ਼ ਜੋ ਮਾਡਲ ਦੀ ਸਿਖਲਾਈ ਦੀ ਕਟਆਊਟ ਤੋਂ ਬਾਅਦ ਬਦਲ ਸਕਦੇ ਹਨ।

ਨਤੀਜੇ (Results) ਰੀਅਲ ਟਾਈਮ ਵਿੱਚ ਕੱਢੇ ਜਾਂਦੇ ਹਨ ਅਤੇ ਜਵਾਬ ਬਣਾਉਣ ਤੋਂ ਪਹਿਲਾਂ ਅਸਿਸਟੈਂਟ ਦੇ ਕੰਟੈਕਸਟ ਵਿੱਚ ਪਾ ਦਿੱਤੇ ਜਾਂਦੇ ਹਨ। ਅਸਿਸਟੈਂਟ ਦੱਸਦਾ ਹੈ ਕਿ ਉਸਨੇ ਸਵਾਲ ਦਾ ਜਵਾਬ ਦੇਣ ਲਈ ਸਰਚ ਨਤੀਜਿਆਂ ਦੀ ਵਰਤੋਂ ਕੀਤੀ ਹੈ।

## ਇੰਟਰਨੈਟ ਸਰਚ ਸਥਾਪਿਤ ਕਰਨਾ {#enabling-internet-search}

ਇੰਟਰਨੈਟ ਸਰਚ ਲਈ ਤੁਹਾਡੇ ਚੁਣੇ ਹੋਏ ਸਰਚ ਪ੍ਰੋਵਾਈਡਰ ਤੋਂ ਇੱਕ API key ਦੀ ਲੋੜ ਹੁੰਦੀ ਹੈ। ਇਸਨੂੰ ਸੈਟ ਕਰਨ ਲਈ:

1. **Gratis AI Agent → Settings → Advanced** 'ਤੇ ਜਾਓ।
2. **Internet Search Provider** ਡ੍ਰੌਪਡਾਊਨ ਲੱਭੋ ਅਤੇ ਜਾਂ ਤਾਂ **Brave Search** ਜਾਂ **Tavily** ਚੁਣੋ।
3. ਆਪਣੇ API key ਨੂੰ ਸਬੰਧਤ ਫੀਲਡ ਵਿੱਚ ਪਾਓ। ਹਰੇਕ ਫੀਲਡ ਦੇ ਨਾਲ ਸਾਈਨ-ਅੱਪ ਲਿੰਕ ਦਿਖਾਏ ਗਏ ਹਨ।
4. **Save Settings** 'ਤੇ ਕਲਿੱਕ ਕਰੋ।

ਇੱਕ ਵਾਰ key ਸੇਵ ਹੋ ਜਾਣ ਤੋਂ ਬਾਅਦ, ਇੰਟਰਨੈਟ ਸਰਚ ਯੋਗਤਾ ਆਪਣੇ ਆਪ ਅਸਿਸਟੈਂਟ ਲਈ ਉਪਲਬਧ ਹੋ ਜਾਂਦੀ ਹੈ।

## Brave Search {#brave-search}

### Brave Search API Key ਪ੍ਰਾਪਤ ਕਰਨਾ {#obtaining-a-brave-search-api-key}

1. [Brave Search API page](https://brave.com/search/api/) 'ਤੇ ਜਾਓ।
2. ਇੱਕ ਪਲਾਨ ਲਈ ਸਾਈਨ-ਅੱਪ ਕਰੋ। ਮਹੀਨਾਵਾਰ ਰੀਕਵੈਸਟ ਸੀਮਾ ਨਾਲ ਇੱਕ ਫ੍ਰੀ ਟਾਇਰ ਉਪਲਬਧ ਹੈ।
3. Brave Search developer dashboard ਤੋਂ ਆਪਣੀ API key ਕਾਪੀ ਕਰੋ।
4. ਇਸਨੂੰ Gratis AI Agent settings ਵਿੱਚ **Brave Search API Key** ਫੀਲਡ ਵਿੱਚ ਪੇਸਟ ਕਰੋ।

### ਵਰਤੋਂ ਦੀਆਂ ਸੀਮਾਵਾਂ {#usage-limits}

ਵਰਤੋਂ ਦਾ ਬਿੱਲ Brave Search ਦੁਆਰਾ ਕੀਤੇ ਗਏ ਕੁਐਰੀਆਂ ਦੀ ਗਿਣਤੀ ਦੇ ਆਧਾਰ 'ਤੇ ਕੀਤਾ ਜਾਂਦਾ ਹੈ। ਹਰ AI ਜਵਾਬ ਜੋ ਸਰਚ ਨੂੰ ਟ੍ਰਿਗਰ ਕਰਦਾ ਹੈ, ਉਸਨੂੰ ਇੱਕ ਕੁਐਰੀ ਮੰਨਿਆ ਜਾਂਦਾ ਹੈ। ਅਚਾਨਕ ਖਰਚੇ ਤੋਂ ਬਚਣ ਲਈ ਆਪਣੀ ਵਰਤੋਂ [Brave Search developer dashboard](https://brave.com/search/api/) ਵਿੱਚ ਨਿਗਰਾਨੀ ਕਰੋ।

## Tavily {#tavily}

Superdav AI Agent v1.10.0 ਇੱਕ ਬਦਲਵੇਂ ਇੰਟਰਨੈਟ ਸਰਚ ਪ੍ਰੋਵਾਈਡਰ ਵਜੋਂ **Tavily** ਜੋੜਦਾ ਹੈ, ਜੋ ਹੋਰ ਅਮੀਰ ਸਰਚ ਨਤੀਜੇ ਅਤੇ ਐਡਵਾਂਸ ਰਿਸਰਚ ਯੋਗਤਾਵਾਂ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ।

### Tavily API Key ਪ੍ਰਾਪਤ ਕਰਨਾ {#obtaining-a-tavily-api-key}

1. [Tavily API page](https://tavily.com/) 'ਤੇ ਜਾਓ।
2. ਇੱਕ account ਲਈ ਸਾਈਨ-ਅੱਪ ਕਰੋ। ਮਹੀਨਾਵਾਰ ਰੀਕਵੈਸਟ ਸੀਮਾ ਨਾਲ ਇੱਕ ਫ੍ਰੀ ਟਾਇਰ ਉਪਲਬਧ ਹੈ।
3. Tavily dashboard ਤੋਂ ਆਪਣੀ API key ਕਾਪੀ ਕਰੋ।
4. ਇਸਨੂੰ Gratis AI Agent settings ਵਿੱਚ **Tavily API Key** ਫੀਲਡ ਵਿੱਚ ਪੇਸਟ ਕਰੋ।

### ਵਰਤੋਂ ਦੀਆਂ ਸੀਮਾਵਾਂ {#usage-limits-1}

ਵਰਤੋਂ ਦਾ ਬਿੱਲ Tavily ਦੁਆਰਾ ਕੀਤੇ ਗਏ API ਕਾਲਾਂ ਦੀ ਗਿਣਤੀ ਦੇ ਆਧਾਰ 'ਤੇ ਕੀਤਾ ਜਾਂਦਾ ਹੈ। ਹਰ AI ਜਵਾਬ ਜੋ ਸਰਚ ਨੂੰ ਟ੍ਰਿਗਰ ਕਰਦਾ ਹੈ, ਉਸਨੂੰ ਇੱਕ ਕਾਲ ਮੰਨਿਆ ਜਾਂਦਾ ਹੈ। ਅਚਾਨਕ ਖਰਚੇ ਤੋਂ ਬਚਣ ਲਈ ਆਪਣੀ ਵਰਤੋਂ [Tavily dashboard](https://tavily.com/) ਵਿੱਚ ਨਿਗਰਾਨੀ ਕਰੋ।

## ਇੰਟਰਨੈਟ ਸਰਚ ਬੰਦ ਕਰਨਾ {#disabling-internet-search}

API key ਨੂੰ ਸਰਗਰਮ ਸਰਚ ਪ੍ਰੋਵਾਈਡਰ ਫੀਲਡ ਵਿੱਚੋਂ ਹਟਾ ਦਿਓ ਅਤੇ ਸੇਵ ਕਰੋ। ਇੰਟਰਨੈਟ ਸਰਚ ਯੋਗਤਾ ਹੁਣ ਅਸਿਸਟੈਂਟ ਨੂੰ ਨਹੀਂ ਦਿੱਤੀ ਜਾਵੇਗੀ।

:::note
ਇੰਟਰਨੈਟ ਸਰਚ ਜਵਾਬਾਂ ਵਿੱਚ ਲੇਟੈਂਸੀ (latency) ਜੋੜਦਾ ਹੈ ਕਿਉਂਕਿ ਅਸਿਸਟੈਂਟ ਨੂੰ ਜਵਾਬ ਬਣਾਉਣ ਤੋਂ ਪਹਿਲਾਂ ਸਰਚ ਨਤੀਜਿਆਂ ਦਾ ਇੰਤਜ਼ਾਰ ਕਰਨਾ ਪੈਂਦਾ ਹੈ। ਸਮੇਂ-ਸੰਵੇਦਨਸ਼ੀਲ ਵਰਤੋਂ ਦੇ ਮਾਮਲਿਆਂ ਲਈ, ਵਿਚਾਰ ਕਰੋ ਕਿ ਕੀ ਰੀਅਲ-ਟਾਈਮ ਸਰਚ ਜ਼ਰੂਰੀ ਹੈ ਜਾਂ ਕੀ ਅਸਿਸਟੈਂਟ ਦਾ 内-ਬਿਲਟ ਗਿਆਨ ਕਾਫ਼ੀ ਹੈ।
:::
