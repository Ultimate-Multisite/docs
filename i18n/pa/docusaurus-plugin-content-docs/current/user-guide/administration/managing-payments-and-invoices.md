# ਭੁਗਤਾਨਾਂ ਅਤੇ ਇਨਵੌਇਸਾਂ ਦਾ ਪ੍ਰਬੰਧਨ

## ਭੁਗਤਾਨ ਸੈਟਿੰਗਾਂ

ਭੁਗਤਾਨ ਪ੍ਰਾਪਤ ਕਰਨਾ ਸ਼ੁਰੂ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ, ਤੁਹਾਨੂੰ ਭੁਗਤਾਨ ਨਾਲ ਸੰਬੰਧਿਤ ਸੈਟਿੰਗਾਂ ਨੂੰ ਕੌਂਫਿਗਰ ਕਰਨਾ ਪਵੇਗਾ। **Ultimate Multisite → Settings** 'ਤੇ ਜਾਓ ਅਤੇ **Payment** ਟੈਬ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।

![ਭੁਗਤਾਨ ਸੈਟਿੰਗਾਂ ਟੈਬ](/img/admin/settings-payments-top.png)

### ਆਮ ਭੁਗਤਾਨ ਵਿਕਲਪ

ਆਮ ਸੈਟਿੰਗਾਂ ਵਿੱਚ ਤੁਸੀਂ ਇਹ ਕੌਂਫਿਗਰ ਕਰ ਸਕਦੇ ਹੋ:

- **Currency** — ਲੈਣ-ਦੇਣ ਲਈ ਵਰਤੀ ਜਾਣ ਵਾਲੀ ਡਿਫਾਲਟ ਮੁਦਰਾ
- **Currency Position** — ਮੁਦਰਾ ਚਿੰਨ੍ਹ ਕਿੱਥੇ ਦਿਖਾਈ ਦੇਵੇ (ਰਕਮ ਤੋਂ ਪਹਿਲਾਂ/ਬਾਅਦ)

![ਭੁਗਤਾਨ ਗੇਟਵੇ ਸੈਟਿੰਗਾਂ](/img/admin/settings-payments-options.png)

### Payment Gateways

Ultimate Multisite ਕਈ payment gateways ਦਾ ਸਮਰਥਨ ਕਰਦਾ ਹੈ। ਤੁਸੀਂ Payment ਸੈਟਿੰਗਾਂ ਟੈਬ ਤੋਂ ਹਰੇਕ ਨੂੰ ਚਾਲੂ ਅਤੇ ਕੌਂਫਿਗਰ ਕਰ ਸਕਦੇ ਹੋ।

![Payment gateway ਕੌਂਫਿਗਰੇਸ਼ਨ](/img/admin/settings-payments-gateways.png)

ਉਪਲਬਧ gateways ਵਿੱਚ ਸ਼ਾਮਲ ਹਨ:

- **Stripe** — Stripe ਰਾਹੀਂ ਕ੍ਰੈਡਿਟ ਕਾਰਡ ਭੁਗਤਾਨ
- **PayPal** — PayPal ਭੁਗਤਾਨ
- **Manual** — ਔਫਲਾਈਨ ਜਾਂ ਕਸਟਮ ਭੁਗਤਾਨ ਪ੍ਰੋਸੈਸਿੰਗ ਲਈ

ਹਰੇਕ gateway ਦਾ ਆਪਣਾ ਕੌਂਫਿਗਰੇਸ਼ਨ ਸੈਕਸ਼ਨ ਹੈ ਜਿੱਥੇ ਤੁਸੀਂ API keys ਅਤੇ ਹੋਰ ਸੈਟਿੰਗਾਂ ਦਾਖਲ ਕਰਦੇ ਹੋ।

![ਵਾਧੂ gateway ਸੈਟਿੰਗਾਂ](/img/admin/settings-payments-gateways-2.png)

### Sandbox Mode

ਤੁਸੀਂ ਲਾਈਵ ਜਾਣ ਤੋਂ ਪਹਿਲਾਂ ਆਪਣੇ ਭੁਗਤਾਨ ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਦੀ ਜਾਂਚ ਕਰਨ ਲਈ **Sandbox Mode** ਚਾਲੂ ਕਰ ਸਕਦੇ ਹੋ। ਜਦੋਂ sandbox mode ਚਾਲੂ ਹੁੰਦਾ ਹੈ, ਤਾਂ ਕੋਈ ਅਸਲ ਚਾਰਜ ਨਹੀਂ ਲੱਗੇਗਾ।

## ਭੁਗਤਾਨ ਦੇਖਣਾ

ਆਪਣੇ ਨੈੱਟਵਰਕ ਦੇ ਸਾਰੇ ਲੈਣ-ਦੇਣ ਦੇਖਣ ਲਈ Ultimate Multisite ਦੇ ਅਧੀਨ **Payments** ਪੇਜ 'ਤੇ ਜਾਓ।

![ਭੁਗਤਾਨਾਂ ਦੀ ਸੂਚੀ](/img/admin/payments-list.png)

ਤੁਸੀਂ ਸਥਿਤੀ (ਪੂਰਾ ਹੋਇਆ, ਬਕਾਇਆ, ਅਸਫਲ, ਵਾਪਸ ਕੀਤਾ) ਦੇ ਆਧਾਰ 'ਤੇ ਭੁਗਤਾਨਾਂ ਨੂੰ ਫਿਲਟਰ ਕਰ ਸਕਦੇ ਹੋ ਅਤੇ ਖਾਸ ਲੈਣ-ਦੇਣ ਲੱਭ ਸਕਦੇ ਹੋ।

ਕਿਸੇ ਭੁਗਤਾਨ 'ਤੇ ਕਲਿੱਕ ਕਰਕੇ ਇਸਦੇ ਪੂਰੇ ਵੇਰਵੇ ਦੇਖੋ ਜਿਸ ਵਿੱਚ ਲਾਈਨ ਆਈਟਮਾਂ, ਸੰਬੰਧਿਤ ਮੈਂਬਰਸ਼ਿਪ, ਗਾਹਕ ਜਾਣਕਾਰੀ, ਅਤੇ payment gateway ਡੇਟਾ ਸ਼ਾਮਲ ਹਨ।

## ਇਨਵੌਇਸ

Ultimate Multisite ਭੁਗਤਾਨਾਂ ਲਈ ਆਪਣੇ ਆਪ ਇਨਵੌਇਸ ਬਣਾ ਸਕਦਾ ਹੈ। ਤੁਸੀਂ Payment ਸੈਟਿੰਗਾਂ ਤੋਂ ਇਨਵੌਇਸ ਟੈਂਪਲੇਟ ਅਤੇ ਨੰਬਰਿੰਗ ਫਾਰਮੈਟ ਨੂੰ ਕਸਟਮਾਈਜ਼ ਕਰ ਸਕਦੇ ਹੋ।

ਇਨਵੌਇਸ ਕਸਟਮਾਈਜ਼ੇਸ਼ਨ ਵਿਕਲਪਾਂ ਵਿੱਚ ਸ਼ਾਮਲ ਹਨ:

- ਇਨਵੌਇਸਾਂ 'ਤੇ ਦਿਖਾਈ ਦੇਣ ਵਾਲਾ **ਕੰਪਨੀ ਦਾ ਨਾਮ ਅਤੇ ਪਤਾ**
- **ਇਨਵੌਇਸ ਨੰਬਰਿੰਗ** ਫਾਰਮੈਟ ਅਤੇ ਕ੍ਰਮ
- ਇਨਵੌਇਸ ਹੈਡਰ 'ਤੇ ਦਿਖਾਇਆ ਜਾਣ ਵਾਲਾ **ਲੋਗੋ**
- ਸ਼ਰਤਾਂ, ਨੋਟਸ, ਜਾਂ ਕਾਨੂੰਨੀ ਜਾਣਕਾਰੀ ਲਈ **ਕਸਟਮ ਫੁੱਟਰ ਟੈਕਸਟ**

ਇਨਵੌਇਸ ਟੈਂਪਲੇਟ ਕਸਟਮਾਈਜ਼ ਕਰਨ ਲਈ, **Ultimate Multisite → Settings → Payment** 'ਤੇ ਜਾਓ ਅਤੇ ਇਨਵੌਇਸ ਨਾਲ ਸੰਬੰਧਿਤ ਸੈਟਿੰਗਾਂ ਲੱਭੋ।
