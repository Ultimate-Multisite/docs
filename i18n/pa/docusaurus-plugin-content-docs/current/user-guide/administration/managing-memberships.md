# ਮੈਂਬਰਸ਼ਿਪਾਂ ਦਾ ਪ੍ਰਬੰਧਨ

Ultimate Multisite ਵਿੱਚ ਤੁਸੀਂ ਮੈਂਬਰਸ਼ਿਪਾਂ ਦਾ ਪ੍ਰਬੰਧਨ ਕਰ ਸਕਦੇ ਹੋ ਅਤੇ ਆਪਣੇ ਨੈੱਟਵਰਕ ਦੀਆਂ ਸਾਰੀਆਂ ਸਬਸਕ੍ਰਿਪਸ਼ਨਾਂ ਤੱਕ ਪਹੁੰਚ ਕਰ ਸਕਦੇ ਹੋ। ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਦੇ ਵੇਰਵਿਆਂ ਦੀ ਸਮੀਖਿਆ ਅਤੇ ਸੋਧ ਕਰੋ ਜਿਵੇਂ ਕਿ ਪਲਾਨ ਅਤੇ ਉਤਪਾਦ, ਸਾਈਟਾਂ ਦੀ ਗਿਣਤੀ, ਬਿਲਿੰਗ ਰਕਮ ਅਤੇ ਵਾਰਵਾਰਤਾ, ਭੁਗਤਾਨ ਇਤਿਹਾਸ, ਅਤੇ ਸਮਾਂ-ਮੋਹਰਾਂ।

ਇਸ ਪੰਨੇ ਤੱਕ ਪਹੁੰਚਣ ਲਈ, Ultimate Multisite ਮੀਨੂ ਵਿੱਚ **Memberships** ਟੈਬ 'ਤੇ ਜਾਓ।

![ਮੈਂਬਰਸ਼ਿਪ ਸੂਚੀ ਪੰਨਾ](/img/admin/memberships-list.png)

Memberships ਪੰਨੇ 'ਤੇ ਤੁਸੀਂ ਆਪਣੇ ਗਾਹਕਾਂ ਦੀ ਸੂਚੀ ਅਤੇ ਉਨ੍ਹਾਂ ਦੇ ਖਾਤਿਆਂ ਨਾਲ ਜੁੜੇ ਉਤਪਾਦ ਦੇਖ ਸਕਦੇ ਹੋ। ਤੁਸੀਂ ਮੈਂਬਰਸ਼ਿਪਾਂ ਨੂੰ ਸਥਿਤੀ ਅਨੁਸਾਰ ਫਿਲਟਰ ਕਰ ਸਕਦੇ ਹੋ: ਸਰਗਰਮ, ਬਕਾਇਆ, ਰੋਕੀ ਹੋਈ, ਮਿਆਦ ਪੁੱਗੀ, ਜਾਂ ਰੱਦ ਕੀਤੀ।

ਮੈਂਬਰਸ਼ਿਪ ਦੇ ਵੇਰਵਿਆਂ ਨੂੰ ਸੋਧਣ ਲਈ ਕਿਸੇ ਮੈਂਬਰਸ਼ਿਪ 'ਤੇ ਕਲਿੱਕ ਕਰੋ — ਬਿਲਿੰਗ ਜਾਣਕਾਰੀ, ਸਾਈਟ ਜਾਂ ਵਿਜ਼ਿਟ ਸੀਮਾਵਾਂ, ਅਤੇ ਹੋਰ ਬਹੁਤ ਕੁਝ।

ਤੁਸੀਂ ਆਪਣੇ ਨੈੱਟਵਰਕ 'ਤੇ ਪਹਿਲਾਂ ਤੋਂ ਰਜਿਸਟਰਡ ਉਪਭੋਗਤਾਵਾਂ ਨਾਲ ਜੁੜੀਆਂ ਨਵੀਆਂ ਮੈਂਬਰਸ਼ਿਪਾਂ ਵੀ ਜੋੜ ਸਕਦੇ ਹੋ। **Add Membership** 'ਤੇ ਕਲਿੱਕ ਕਰੋ ਅਤੇ ਵੇਰਵੇ ਸੈੱਟ ਕਰੋ।

![ਮੈਂਬਰਸ਼ਿਪ ਜੋੜੋ ਮੋਡਲ](/img/admin/membership-add-modal.png)

ਫਿਰ **Create Membership** 'ਤੇ ਕਲਿੱਕ ਕਰੋ। ਨਵੀਂ ਮੈਂਬਰਸ਼ਿਪ ਮੈਂਬਰਸ਼ਿਪ ਸੂਚੀ ਵਿੱਚ ਦਿਖਾਈ ਦੇਵੇਗੀ।
