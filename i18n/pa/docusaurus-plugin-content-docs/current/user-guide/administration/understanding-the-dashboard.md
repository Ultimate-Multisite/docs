# Ultimate Multisite Dashboard ਨੂੰ ਸਮਝਣਾ

Ultimate Multisite ਦੀਆਂ ਹੋਰ ਐਡਵਾਂਸਡ ਸੈਟਿੰਗਾਂ ਅਤੇ ਕਾਰਜਸ਼ੀਲ ਪਹਿਲੂਆਂ ਵਿੱਚ ਡੂੰਘਾਈ ਨਾਲ ਜਾਣ ਤੋਂ ਪਹਿਲਾਂ, ਆਓ dashboard 'ਤੇ ਇੱਕ ਨਜ਼ਰ ਮਾਰੀਏ।

ਬਹੁਤ ਸਾਰੇ WordPress plugins ਵਾਂਗ, Ultimate Multisite ਵੀ network-level 'ਤੇ **Ultimate Multisite** ਨਾਂ ਦੀ ਇੱਕ ਮੀਨੂ ਐਂਟਰੀ ਬਣਾਉਂਦਾ ਹੈ ਜਿਸ ਦੇ ਹੇਠਾਂ ਕਈ sub-menu ਐਂਟਰੀਆਂ ਹੁੰਦੀਆਂ ਹਨ।

## Network Dashboard Widgets

Ultimate Multisite network-level dashboard ਵਿੱਚ ਕੁਝ ਉਪਯੋਗੀ widgets ਜੋੜਦਾ ਹੈ। ਇਹ dashboard **My Sites → Network Admin → Dashboard** 'ਤੇ ਜਾ ਕੇ ਮਿਲਦਾ ਹੈ। ਇਹ widgets network administrator ਨੂੰ ਜਾਣਕਾਰੀ ਅਤੇ ਆਮ ਕੰਮਾਂ ਤੱਕ ਆਸਾਨ ਪਹੁੰਚ ਦੇਣ ਲਈ ਬਣਾਏ ਗਏ ਹਨ।

![Ultimate Multisite widgets ਨਾਲ Network Dashboard](/img/admin/network-dashboard-full.png)

### ਪਹਿਲੇ ਕਦਮ

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

ਇਹ widget Ultimate Multisite ਦੀ ਨਵੀਂ installation 'ਤੇ ਦਿਖਾਈ ਦਿੰਦਾ ਹੈ ਅਤੇ network administrator ਨੂੰ ਵਾਧੂ ਕੰਮ ਪੂਰੇ ਕਰਨ ਦੀ ਯਾਦ ਦਿਵਾਉਂਦਾ ਹੈ ਜਿਵੇਂ ਕਿ ਭੁਗਤਾਨ ਦਾ ਤਰੀਕਾ ਸੈੱਟ ਕਰਨਾ ਅਤੇ ਟੈਸਟ ਖਾਤਾ ਬਣਾਉਣਾ।

### ਸੰਖੇਪ

![Summary widget](/img/admin/network-dashboard-summary.png)

Summary widget ਦਿਨ ਦੀਆਂ signups ਅਤੇ ਉਨ੍ਹਾਂ ਤੋਂ ਹੋਈ ਕਮਾਈ ਦੀ ਰਿਪੋਰਟ ਦਿੰਦਾ ਹੈ। MRR (Monthly Recurring Revenue) ਕਮਾਈ ਵਾਲੀਆਂ memberships ਵਾਲੇ ਗਾਹਕਾਂ ਦਾ ਅਨੁਮਾਨਿਤ ਕੁੱਲ ਦਰਸਾਉਂਦਾ ਹੈ।

### ਗਤੀਵਿਧੀ ਸਟ੍ਰੀਮ

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

ਇਹ widget Ultimate Multisite ਵਿੱਚ ਹੋਣ ਵਾਲੀਆਂ ਘਟਨਾਵਾਂ ਦੀ ਰਿਪੋਰਟ ਦਿੰਦਾ ਹੈ। ਘਟਨਾਵਾਂ ਵਿੱਚ signups, ਰੱਦ ਕਰਨਾ, ਸਾਈਟ ਬਣਾਉਣਾ, ਮਿਟਾਉਣਾ, ਅਤੇ ਹੋਰ ਗਤੀਵਿਧੀਆਂ ਸ਼ਾਮਲ ਹਨ।

ਵਧੇਰੇ ਵਿਸਤਾਰਪੂਰਵਕ ਜਾਣਕਾਰੀ ਲਈ ਦਸਤਾਵੇਜ਼ਾਂ ਦੇ Events ਸੈਕਸ਼ਨ ਨੂੰ ਦੇਖੋ।

### ਹੁਣੇ

![Right Now widget](/img/admin/network-dashboard-right-now.png)

ਇਹ ਉਪਯੋਗੀ widget network ਵਿੱਚ ਯੂਜ਼ਰਾਂ ਅਤੇ ਸਾਈਟਾਂ ਦਾ ਇੱਕ ਛੋਟਾ ਸਾਰ ਦਿਖਾਉਂਦਾ ਹੈ। ਉੱਪਰ ਦਿੱਤੇ ਸੁਵਿਧਾਜਨਕ shortcuts ਨਾਲ ਤੁਸੀਂ ਇੱਕ ਕਲਿੱਕ 'ਤੇ ਨਵੀਆਂ ਸਾਈਟਾਂ ਜਾਂ ਯੂਜ਼ਰ ਬਣਾ ਸਕਦੇ ਹੋ।

### ਖ਼ਬਰਾਂ ਅਤੇ ਚਰਚਾਵਾਂ

![News and Discussions widget](/img/admin/network-dashboard-news.png)

ਇਹ widget Ultimate Multisite ਦੀਆਂ ਤਾਜ਼ਾ ਖ਼ਬਰਾਂ ਲਿਆਉਂਦਾ ਅਤੇ ਦਿਖਾਉਂਦਾ ਹੈ। updates, ਮਹੱਤਵਪੂਰਨ bugs, ਅਤੇ security fixes ਬਾਰੇ ਜਾਣਕਾਰੀ ਲਈ ਇਸ widget 'ਤੇ ਨਜ਼ਰ ਰੱਖੋ।

## Ultimate Multisite Dashboard

ਜਿੱਥੇ network-level dashboard ਪੂਰੇ network ਬਾਰੇ ਸੰਪੂਰਨ ਜਾਣਕਾਰੀ ਦਿੰਦਾ ਹੈ, ਉੱਥੇ Ultimate Multisite dashboard (Ultimate Multisite ਮੀਨੂ ਦੇ ਉੱਪਰਲੇ ਪੱਧਰ 'ਤੇ ਮਿਲਦਾ ਹੈ) ਸੇਵਾ ਬਾਰੇ ਜਾਣਕਾਰੀ ਦਿੰਦਾ ਹੈ।

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

### ਮਹੀਨਾਵਾਰ ਆਵਰਤੀ ਆਮਦਨ ਵਾਧਾ

MRR (Monthly Recurring Revenue) ਚਾਰਟ 12 ਮਹੀਨਿਆਂ ਦੇ ਕੈਲੰਡਰ ਸਮੇਂ ਦੌਰਾਨ ਮਹੀਨਾ-ਦਰ-ਮਹੀਨਾ ਆਮਦਨ ਦਿਖਾਉਂਦਾ ਹੈ।

![MRR Growth ਚਾਰਟ](/img/admin/um-dashboard-mrr.png)

ਨਵੀਂ ਆਮਦਨ ਨੀਲੇ ਰੰਗ ਵਿੱਚ ਦਿਖਾਈ ਜਾਂਦੀ ਹੈ ਜਦੋਂ ਕਿ ਰੱਦ ਕੀਤੀਆਂ ਲਾਲ ਰੰਗ ਵਿੱਚ।

### ਮਿਤੀ-ਸੀਮਾ ਸਹਾਇਤਾ

ਗਾਹਕਾਂ ਅਤੇ ਗਤੀਵਿਧੀ ਦੀ ਗਿਣਤੀ ਦੇ ਆਧਾਰ 'ਤੇ, ਅੰਕੜੇ ਭਾਰੂ ਹੋ ਸਕਦੇ ਹਨ। Dashboard ਵਿੱਚ ਬਣੀ ਮਿਤੀ-ਸੀਮਾ ਸਹਾਇਤਾ network administrator ਨੂੰ ਸਿਰਫ਼ ਲੋੜੀਂਦੀ ਜਾਣਕਾਰੀ ਅਤੇ ਸਮੇਂ 'ਤੇ ਧਿਆਨ ਕੇਂਦਰਿਤ ਕਰਨ ਦਿੰਦੀ ਹੈ।

![ਮਿਤੀ ਸੀਮਾ ਚੋਣਕਾਰ ਅਤੇ ਅੰਕੜੇ](/img/admin/um-dashboard-date-range.png)

ਮਿਤੀ ਸੀਮਾ ਹੇਠਾਂ ਦਿੱਤੇ widgets ਦੁਆਰਾ ਦਿਖਾਈ ਜਾਣਕਾਰੀ ਦੀ ਮਿਆਦ ਨਿਰਧਾਰਿਤ ਕਰਦੀ ਹੈ, ਜਿਸ ਵਿੱਚ ਆਮਦਨ, ਨਵੀਆਂ memberships, ਦੇਸ਼ ਅਨੁਸਾਰ signups, ਫਾਰਮ ਅਨੁਸਾਰ signups, ਅਤੇ ਸਭ ਤੋਂ ਵੱਧ ਦੇਖੀਆਂ ਗਈਆਂ ਸਾਈਟਾਂ ਸ਼ਾਮਲ ਹਨ।

![Dashboard ਅੰਕੜੇ widgets](/img/admin/um-dashboard-stats.png)

:::tip
**Most Visited Sites** widget ਨੂੰ ਕੰਮ ਕਰਾਉਣ ਲਈ, **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** 'ਤੇ ਵਿਜ਼ਿਟ ਗਿਣਤੀ ਚਾਲੂ ਕਰੋ।
:::
