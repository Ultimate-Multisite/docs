# Action: wu_mcp_adapter_initialized

ਇਹ ਐਕਸ਼ਨ (Action) ਉਦੋਂ ਚੱਲਦਾ ਹੈ ਜਦੋਂ MCP ਐਡਾਪਟਰ (adapter) ਇਨੀਸ਼ੀਅਲਾਈਜ਼ ਹੋ ਜਾਂਦਾ ਹੈ।

ਇਹ ਹੋਰ ਪਲੱਗਇਨਾਂ ਅਤੇ ਥੀਮਾਂ ਨੂੰ ਆਪਣੀਆਂ ਯੋਗਤਾਵਾਂ (abilities) ਰਜਿਸਟਰ ਕਰਨ ਦੀ ਇਜਾਜ਼ਤ ਦਿੰਦਾ ਹੈ।

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | MCP ਐਡਾਪਟਰ ਦਾ ਈਨਸਟੈਂਸ (instance)। |
