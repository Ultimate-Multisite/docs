# Action: wu_register_integrations {#action-wuregisterintegrations}

ਇਹ ਕਾਰਵਾਈ ਇੰਟੀਗ੍ਰੇਸ਼ਨਾਂ ਨੂੰ ਆਪਣੇ ਆਪ ਰਜਿਸਟਰ ਕਰਨ ਦੀ ਇਜਾਜ਼ਤ ਦਿੰਦੀ ਹੈ।

ਕੋਰ (Core) ਇੱਥੇ ਆਪਣੀਆਂ ਇੰਟੀਗ੍ਰੇਸ਼ਨਾਂ ਰਜਿਸਟਰ ਕਰਦਾ ਹੈ।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $registry | `\Integration_Registry` | ਰਜਿਸਟਰੀ ਦਾ ਇੰਸਟੈਂਸ। |
