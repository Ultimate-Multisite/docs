# Action: wu_after_switch_template

ਇਹ ਪਲੱਗਇਨ ਡਿਵੈਲਪਰਾਂ ਨੂੰ ਇਹ ਮੌਕਾ ਦਿੰਦਾ ਹੈ ਕਿ ਉਹ ਕੋਈ ਫੰਕਸ਼ਨ ਹੁੱਕ ਕਰ ਸਕਣ ਜਦੋਂ ਕੋਈ ਯੂਜ਼ਰ ਜਾਂ ਸੁਪਰ ਐਡਮਿਨ ਸਾਈਟ ਦਾ ਟੈਂਪਲੇਟ ਬਦਲਦਾ ਹੈ।
