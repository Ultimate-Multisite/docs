# Action: wu_page_before_render {#action-wupagebeforerender}

ਇਹ ਪਲੱਗਇਨ ਡਿਵੈਲਪਰਾਂ ਨੂੰ ਪੇਜ ਨੂੰ ਪ੍ਰਿੰਟ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ ਵਾਧੂ ਕੰਟੈਂਟ ਜੋੜਨ ਦੀ ਇਜਾਜ਼ਤ ਦਿੰਦਾ ਹੈ।

## ਪੈਰਾਮੀਟਰਾਂ {#parameters}

| ਨਾਮ | ਕਿਸਮ | ਵਰਣਨ |
|------|------|-------------|
| $page_id | `string` | ਇਸ ਪੇਜ ਦੀ ID। |
| $page | `object` | ਪੇਜ ਦਾ ਆਬਜੈਕਟ। |
