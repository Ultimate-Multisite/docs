# Action: wu_mcp_abilities_registered

ਇਹ ਕੰਮ ਉਦੋਂ ਚੱਲਦਾ ਹੈ ਜਦੋਂ ਕਿਸੇ entity ਲਈ MCP abilities register ਹੋ ਜਾਂਦੀਆਂ ਹਨ।

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $ability_prefix | `string` | ਯੋਗਤਾ ਦਾ ਪੂਰਵ缀 (ability prefix)। |
| $model_name | `string` | ਮਾਡਲ ਦਾ ਨਾਮ (model name)। |
| $this | `object` | ਮੈਨੇਜਰ ਦਾ ਇੰਸਟੈਂਸ (manager instance)। |
