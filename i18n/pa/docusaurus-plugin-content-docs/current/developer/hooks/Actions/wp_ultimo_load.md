# Action: wp_ultimo_load

ਇਹ ਟ੍ਰਿਗਰ ਉਦੋਂ ਚੱਲਦਾ ਹੈ ਜਦੋਂ ਸਾਰੀਆਂ ਜ਼ਰੂਰੀ (dependencies) ਚੀਜ਼ਾਂ ਲੋਡ ਹੋ ਚੁੱਕੀਆਂ ਹੁੰਦੀਆਂ ਹਨ।

ਇਹ plugin developers ਨੂੰ ਨਵੀਆਂ ਕਾਰਜਕਮਤਾਵਾਂ (functionality) ਜੋੜਨ ਦੀ ਇਜਾਜ਼ਤ ਦਿੰਦਾ ਹੈ। ਜਿਵੇਂ ਕਿ, ਨਵੇਂ Hosting providers ਲਈ ਸਪੋਰਟ ਜੋੜਨਾ, ਆਦਿ।
