# Action: wu_site_created

ਇਹ ਐਕਸ਼ਨ ਉਦੋਂ ਚੱਲਦਾ ਹੈ ਜਦੋਂ ਕੋਈ ਸਾਈਟ ਪਹਿਲੀ ਵਾਰ ਬਣਾਈ ਜਾਂਦੀ ਹੈ।

ਜੇਕਰ ਸਾਈਟ ਕਿਸੇ ਟੈਂਪਲੇਟ ਤੋਂ ਡੁਪਲੀਕੇਟ ਕੀਤੀ ਜਾਂਦੀ ਹੈ, ਤਾਂ ਇਹ ਐਕਸ਼ਨ ਨਹੀਂ ਚੱਲੇਗਾ।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $data | `array` | ਉਹ ਆਬਜੈਕਟ ਡਾਟਾ ਜੋ ਸਟੋਰ ਕੀਤਾ ਜਾਵੇਗਾ। |
| $site | `\Site` | ਆਬਜੈਕਟ ਦਾ ਇੰਸਟੈਂਸ। |
