# Action: wu_register_capabilities

ਇਹ ਐਕਸ਼ਨ ਐਡਆਨਜ਼ (addons) ਨੂੰ ਇੰਟੀਗ੍ਰੇਸ਼ਨਾਂ (integrations) ਨਾਲ ਕੈਪੇਬਿਲਿਟੀ ਮਾਡੂਲ ਜੋੜਨ ਦੀ ਇਜਾਜ਼ਤ ਦੇਣ ਲਈ ਚੱਲਦਾ ਹੈ।

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $registry | `\Integration_Registry` | ਰਜਿਸਟਰੀ ਦਾ ਇੰਸਟੈਂਸ। |
