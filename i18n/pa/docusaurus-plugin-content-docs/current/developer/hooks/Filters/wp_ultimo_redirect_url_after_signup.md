# Filter: wp_ultimo_redirect_url_after_signup {#filter-wpultimoredirecturlaftersignup}

Redirect URL ਸੈੱਟ ਕਰੋ।

ਇਹ ਇੱਕ ਪੁਰਾਣਾ (legacy) filter ਹੈ। ਇਸ ਵਿੱਚ ਪਾਸ ਹੋਣ ਵਾਲੇ ਕੁਝ parameters ਉਪਲਬਧ ਨਹੀਂ ਹੋ ਸਕਦੇ, ਜਿਵੇਂ ਕਿ $site_id।

### Since {#since}

- 1.1.3: ਡਿਵੈਲਪਰਾਂ ਨੂੰ redirect URL ਨੂੰ filter ਕਰਨ ਦੀ ਇਜਾਜ਼ਤ ਦਿੱਤੀ ਗਈ।
