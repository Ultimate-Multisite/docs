# Filter: wu_admin_notices {#filter-wuadminnotices}

ਇਹ ਡਿਵੈਲਪਰਾਂ ਨੂੰ Ultimate Multisite ਦੁਆਰਾ ਜੋੜੇ ਗਏ ਐਡਮਿਨ ਨੋਟਿਸ (admin notices) ਨੂੰ ਫਿਲਟਰ ਕਰਨ ਦੀ ਇਜਾਜ਼ਤ ਦਿੰਦਾ ਹੈ।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $notices | `array` | ਉਸ ਖਾਸ ਪੈਨਲ ਲਈ ਨੋਟਿਸਾਂ ਦੀ ਸੂਚੀ। |
| $all_notices | `array` | ਜੋੜੇ ਗਏ ਨੋਟਿਸਾਂ ਦੀ ਸੂਚੀ, ਜੋ ਪੈਨਲ ਅਨੁਸਾਰ ਵੰਡੀ ਹੋਈ ਹੈ। |
| $panel | `string` | ਜਿਸ ਪੈਨਲ ਤੋਂ ਨੋਟਿਸਾਂ ਲੈਣੀਆਂ ਹਨ। |
| $filter | `string` | ਜੇਕਰ ਹਟਾਏ ਜਾਣ ਵਾਲੇ ਨੋਟਿਸਾਂ ਨੂੰ ਫਿਲਟਰ ਕਰ ਦਿੱਤਾ ਗਿਆ ਹੈ। |
| $dismissed_messages | `array` | ਹਟਾਏ ਗਏ ਨੋਟਿਸ ਕੀਜ਼ (keys) ਦੀ ਸੂਚੀ। |

### Since {#since}

- 2.0.0
