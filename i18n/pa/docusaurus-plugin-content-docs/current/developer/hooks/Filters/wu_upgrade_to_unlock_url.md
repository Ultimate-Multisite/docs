# Filter: wu_upgrade_to_unlock_url

ਡਿਵੈਲਪਰਾਂ ਨੂੰ ਅਪਗ੍ਰੇਡ ਟੂ ਅਨਲੌਕ URL ਬਦਲਣ ਦੀ ਇਜਾਜ਼ਤ ਦੇਣਾ।

## ਪੈਰਾਮੀਟਰ {#parameters}

| Name | Type | ਵਰਣਨ |
|------|------|-------------|
| $url | `string` | ਅਪਗ੍ਰੇਡ ਕਰਨ ਵਾਲਾ URL। |
| $args | `array` | ਉਹ ਮਾਡਿਊਲ ਅਤੇ ਲਿਮਿਟ ਦਾ ਕਿਸਮ ਜਿਸਨੂੰ ਅਪਗ੍ਰੇਡ ਕਰਨ ਦੀ ਲੋੜ ਹੈ। |

### ਸਰੋਤ {#source}

ਲਾਈਨ 177 'ਤੇ [`inc/functions/limitations.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/limitations.php#L177) ਵਿੱਚ ਪਰਿਭਾਸ਼ਿਤ ਹੈ।
