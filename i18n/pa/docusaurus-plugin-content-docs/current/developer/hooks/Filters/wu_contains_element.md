# Filter: wu_contains_element {#filter-wucontainselement}

Developers ਨੂੰ ਸ਼ੁਰੂਆਤੀ ਖੋਜ ਦੇ ਨਤੀਜਿਆਂ ਨੂੰ ਬਦਲਣ ਦੀ ਇਜਾਜ਼ਤ ਦਿੰਦਾ ਹੈ।

ਇਹ ਤੀਜੀ ਧਿਰ ਦੇ ਬਿਲਡਰਾਂ (third-party builders) ਅਤੇ ਇਸ ਤਰ੍ਹਾਂ ਦੀਆਂ ਚੀਜ਼ਾਂ ਲਈ ਕੰਮ ਆਉਂਦਾ ਹੈ।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $contains_elements | `bool` | ਜੇਕਰ ਐਲੀਮੈਂਟ ਕੰਟੈਂਟ ਵਿੱਚ ਮੌਜੂਦ ਹੈ। |
| $content | `string` | ਜਿਸ ਕੰਟੈਂਟ ਦੀ ਜਾਂਚ ਹੋ ਰਹੀ ਹੈ। |
| $element | `self` | ਮੌਜੂਦਾ ਐਲੀਮੈਂਟ। |
| $post | `null\|\WP_Post` | ਜਾਂਚ ਕਰਨ ਲਈ ਪੋਸਟ। |
