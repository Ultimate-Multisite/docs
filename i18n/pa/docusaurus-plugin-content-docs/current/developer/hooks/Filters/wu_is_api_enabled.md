# Filter: wu_is_api_enabled

ਪਲੱਗਇਨ ਡਿਵੈਲਪਰਜ਼ ਨੂੰ API ਲਈ ਇੱਕ ਖਾਸ ਸਥਿਤੀ (state) ਲਾਗੂ ਕਰਨ ਦੀ ਇਜਾਜ਼ਤ ਦਿੰਦਾ ਹੈ।

### Since {#since}

- 1.7.4
### Source {#source}

[`inc/class-api.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-api.php#L435) ਵਿੱਚ ਲਾਈਨ 435 'ਤੇ ਪਰਿਭਾਸ਼ਿਤ ਕੀਤਾ ਗਿਆ ਹੈ।


## Returns {#returns}
(ਇਹ ਫਿਲਟਰ ਇੱਕ ਬੂਲੀਅਨ (boolean) মান ਵਾਪਸ ਕਰਦਾ ਹੈ, ਜੋ ਇਹ ਦੱਸਦਾ ਹੈ ਕਿ ਕੀ API ਸੇਵਾ ਚਾਲੂ ਹੈ ਜਾਂ ਨਹੀਂ।)
