# Filter: wu_pre_save_settings {#filter-wupresavesettings}

Ultimate Multisite ਦੁਆਰਾ ਸੈਟਿੰਗਾਂ ਨੂੰ ਸੇਵ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ ਫਿਲਟਰ ਕਰਨ ਦੀ ਇਜਾਜ਼ਤ ਡਿਵੈਲਪਰਾਂ ਨੂੰ ਦਿੰਦਾ ਹੈ।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $settings | `array` | ਉਹ ਸੈਟਿੰਗਾਂ ਜੋ ਸੇਵ ਕੀਤੀਆਂ ਜਾਣਗੀਆਂ। |
| $settings_to_save | `array` | ਜੋ ਨਵੀਆਂ ਸੈਟਿੰਗਾਂ ਜੋੜਨ ਹਨ। |
| $saved_settings | `array` | ਮੌਜੂਦਾ ਸੇਵ ਕੀਤੀਆਂ ਸੈਟਿੰਗਾਂ। |
