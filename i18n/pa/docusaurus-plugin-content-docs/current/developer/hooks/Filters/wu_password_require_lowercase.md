# Filter: wu_password_require_lowercase

ਪਾਸਵਰਡਾਂ ਵਿੱਚ ਛੋਟੇ ਅੱਖਰ (lowercase letters) ਦੀ ਲੋੜ ਹੋਵੇ ਜਾਂ ਨਾ, ਇਹ ਫਿਲਟਰ ਕਰਦਾ ਹੈ।

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | ਕਿ ਛੋਟੇ ਅੱਖਰ ਲਾਜ਼ਮੀ ਹਨ ਜਾਂ ਨਹੀਂ। ਜਦੋਂ ਨਿਯਮ ਲਾਗੂ ਹੁੰਦੇ ਹਨ ਤਾਂ ਡਿਫਾਲਟ ਤੌਰ 'ਤੇ ਸਹੀ (true) ਹੁੰਦਾ ਹੈ। |
| $defender_active | `bool` | ਕਿ ਕੀ Defender Pro Strong Password ਸਰਗਰਮ (active) ਹੈ। |
