# Filter: wu_current_set_customer

Developers ਨੂੰ ਡਿਫਾਲਟ (default) ਕੰਮ ਕਰਨ ਦੇ ਤਰੀਕੇ ਨੂੰ ਬਦਲਣ ਅਤੇ ਮੌਜੂਦਾ ਗਾਹਕ (current customer) ਨੂੰ ਵੱਖਰੇ ਤਰੀਕੇ ਨਾਲ ਸੈੱਟ ਕਰਨ ਦੀ ਇਜਾਜ਼ਤ ਦਿੰਦਾ ਹੈ।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $customer | `\WP_Ultimo\Models\Customer` | ਉਹ ਮੌਜੂਦਾ ਗਾਹਕ ਜਿਸਨੂੰ ਸੈੱਟ ਕਰਨਾ ਹੈ। |
| $ | `self` | ਵਰਤਮਾਨ class ਦਾ instance। |
