# Filter: wu_maybe_redirect_to_admin_panel_exclusion_list {#filter-wumayberedirecttoadminpanelexclusionlist}

ਇੱਕ ਬਾਹਰ ਰੱਖਣ ਵਾਲੀ ਸੂਚੀ (exclusion list) ਬਣਾਉਣਾ ਜਿਸ ਨਾਲ ਆਟੋ-ਰੀਡਾਇਰੈਕਟ ਰੁਕ ਜਾਵੇ।

ਇਹ ਜ਼ਰੂਰੀ ਹੈ ਕਿਉਂਕਿ ਨਹੀਂ ਤਾਂ ਪੇਜ ਬਿਲਡਰ ਲੌਗਇਨ ਪੇਜ ਨੂੰ ਐਡਿਟ ਨਹੀਂ ਕਰ ਪਾਏਗਾ ਜਦੋਂ ਕਿ ਉਹ ਪਰਿਭਾਸ਼ਿਤ ਹੋਵੇਗਾ।
