# Filter: wu_plan_product_types

ਉਹ Product types ਨੂੰ Filter ਕਰੋ ਜਿਨ੍ਹਾਂ ਨੂੰ Plans ਵਜੋਂ ਮੰਨਿਆ ਜਾਣਾ ਚਾਹੀਦਾ ਹੈ।

ਇਹ Filter add-ons ਨੂੰ ਵਾલિਡੇਸ਼ਨ (validation) ਅਤੇ ਸੈਗregation (segregation) ਦੌਰਾਨ Plans ਵਜੋਂ ਪਛਾਣੇ ਜਾਣ ਵਾਲੇ ਵਾਧੂ Product types ਰਜਿਸਟਰ ਕਰਨ ਦੀ ਇਜਾਜ਼ਤ ਦਿੰਦਾ ਹੈ।

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | ਉਹ Product types ਦਾ Array ਜਿਨ੍ਹਾਂ ਨੂੰ Plans ਵਜੋਂ ਮੰਨਿਆ ਜਾਣਾ ਹੈ। |
