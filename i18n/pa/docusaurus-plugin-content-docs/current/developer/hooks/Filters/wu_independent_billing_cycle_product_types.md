# Filter: wu_independent_billing_cycle_product_types

ਉਹਨਾਂ ਪ੍ਰੋਡਕਟ ਟਾਈਪਸ ਨੂੰ ਫਿਲਟਰ ਕਰੋ ਜਿਨ੍ਹਾਂ ਦੇ ਆਪਣੇ (independent) ਬਿਲਿੰਗ ਸਾਈਕਲ ਹੁੰਦੇ ਹਨ।

ਜਿਨ੍ਹਾਂ ਪ੍ਰੋਡਕਟਸ ਦੇ ਆਪੇ ਬਿਲਿੰਗ ਸਾਈਕਲ ਹੁੰਦੇ ਹਨ, ਉਹਨਾਂ ਨੂੰ ਕਾਰਟ ਵਿੱਚ ਪਲਾਨ ਦੀ ਬਿਲਿੰਗ ਪੀਰੀਅਡ ਨਾਲ ਮਿਲਣ ਲਈ ਮਜਬੂਰ ਨਹੀਂ ਕੀਤਾ ਜਾਂਦਾ। ਇਹ ਉਹਨਾਂ ਪ੍ਰੋਡਕਟਸ ਲਈ ਕੰਮ ਆਉਂਦਾ ਹੈ ਜਿਵੇਂ ਕਿ ਡੋਮੇਨ ਰਜਿਸਟ੍ਰੇਸ਼ਨ, ਜੋ ਪਲਾਨ ਮਹੀਨਾਵਾਰ ਹੋਵੇ ਜਾਂ ਸਾਲਾਨਾ, ਹਮੇਸ਼ਾ ਸਾਲਾਨਾ ਬਿੱਲ ਕਰਦੇ ਹਨ।

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | ਉਹਨਾਂ ਪ੍ਰੋਡਕਟ ਟਾਈਪਸ ਦਾ ਐਰੇ ਜਿਨ੍ਹਾਂ ਦੇ ਆਪੇ ਬਿਲਿੰਗ ਸਾਈਕਲ ਹੁੰਦੇ ਹਨ। |
