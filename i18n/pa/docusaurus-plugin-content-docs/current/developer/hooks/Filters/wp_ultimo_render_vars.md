# Filter: wp_ultimo_render_vars

प्लगइन डेवलपर्स को रेंडर कॉन्टेक्स्ट में अतिरिक्त वेरिएबल ग्लोबली जोड़ने की अनुमति देता है।

## पैरामीटर्स

| Name | Type | Description |
|------|------|-------------|
| $args | `array` | ऐरे जिसमें रेंडर कॉल द्वारा पास किए गए वेरिएबल होते हैं। |
| $view | `string` | उस व्यू का नाम जिसे रेंडर किया जाना है। |
| $default_view | `string` | फॉलबैक व्यू का नाम। |

### Since

- 2.0.0
