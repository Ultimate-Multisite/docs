# Ultimate Multisite ਦੀ ਡਾਕੂਮੈਂਟੇਸ਼ਨ {#ultimate-multisite-documentation}

Ultimate Multisite ਦੀ ਡਾਕੂਮੈਂਟੇਸ਼ਨ ਵਿੱਚ ਤੁਹਾਡਾ ਸਵਾਗਤ ਹੈ। Ultimate Multisite WordPress Multisite ਨੂੰ ਇੱਕ ਪੂਰੇ Website as a Service (WaaS) ਪਲੇਟਫਾਰਮ ਵਿੱਚ ਬਦਲ ਦਿੰਦਾ ਹੈ।

## ਡਾਕੂਮੈਂਟੇਸ਼ਨ ਸੈਕਸ਼ਨ {#documentation-sections}

### [User Guide](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

ਆਪਣੇ WaaS ਪਲੇਟਫਾਰਮ ਨੂੰ ਸੈੱਟਅੱਪ ਕਰਨ, ਕੰフィਗਰ ਕਰਨ ਅਤੇ ਮੈਨੇਜ ਕਰਨ ਲਈ ਜੋ ਕੁਝ ਵੀ ਚਾਹੀਦਾ ਹੈ। ਇਸ ਵਿੱਚ ਇੰਸਟਾਲੇਸ਼ਨ, ਪੇਮੈਂਟ ਗੇਟਵੇ, ਡੋਮੇਨ ਮੈਪਿੰਗ, ਕਲਾਇੰਟ ਮੈਨੇਜਮੈਂਟ, ਅਤੇ ਹੋਰ ਚੀਜ਼ਾਂ ਸ਼ਾਮਲ ਹਨ।

### [Developer Documentation](/developer/hooks) {#developer-documentation}

Ultimate Multisite 'ਤੇ ਕੰਮ ਕਰਨ ਵਾਲੇ ਡਿਵੈਲਪਰਾਂ ਲਈ ਤਕਨੀਕੀ ਹਵਾਲਾ। ਇਸ ਵਿੱਚ hooks reference, REST API ਡਾਕੂਮੈਂਟੇਸ਼ਨ, ਅਤੇ addon ਡਿਵੈਲਪਮੈਂਟ ਗਾਈਡ ਸ਼ਾਮਲ ਹਨ।

### [Addons](/addons) {#addons}

ਪੇਡ addon plugins ਦੀ ਡਾਕੂਮੈਂਟੇਸ਼ਨ ਜੋ Ultimate Multisite ਨੂੰ ਵਾਧੂ ਫੀਚਰ ਨਾਲ ਜੋੜਦੇ ਹਨ, ਜਿਵੇਂ ਕਿ WooCommerce ਇੰਟੀਗ੍ਰੇਸ਼ਨ, ਐਨਾਲਿਟਿਕਸ, ਡੋਮੇਨ ਸੇਲਿੰਗ, ਅਤੇ ਹੋਰ।
