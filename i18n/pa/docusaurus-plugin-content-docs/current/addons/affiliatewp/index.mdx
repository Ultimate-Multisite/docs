# AffiliateWP

Multisite Ultimate: AffiliateWP Integration

ਆਪਣੇ Ultimo Network ਦੇ client base ਨੂੰ ਵਧਾਉਣ ਲਈ ਸ਼ਕਤੀਸ਼ਾਲੀ AffiliateWP ਦੀ ਵਰਤੋਂ ਕਰੋ!

## Installation

1. 'ultimate-multisite-git ' ਨੂੰ '/wp-content/plugins/' directory ਵਿੱਚ ਅਪਲੋਡ ਕਰੋ।
2. WordPress ਵਿੱਚ 'Plugins' ਮੀਨੂ ਰਾਹੀਂ plugin ਨੂੰ ਐਕਟਿਵੇਟ ਕਰੋ।
3. plugin ਸੈਟਅਪ ਕਰਨ ਲਈ ਕਦਮ-ਦਰ-ਕਦਮ Wizard ਦੀ ਪਾਲਣਾ ਕਰੋ।

## Usage Guide

[AffiliateWP](https://affiliatewp.com/) ਤੁਹਾਡੇ ਆਨਲਾਈਨ ਕਾਰੋਬਾਰ ਲਈ ਇੱਕ affiliate program ਬਣਾਉਣ ਵੇਲੇ ਸਭ ਤੋਂ ਵਧੀਆ ਹੱਲ ਬਣ ਗਿਆ ਹੈ। Ultimate Multisite: AffiliateWP ਉਹ ਪੁਲ ਹੈ ਜੋ ਤੁਹਾਨੂੰ ਆਪਣੇ ਪ੍ਰੀਮੀਅਮ ਨੈਟਵਰਕ ਲਈ ਵੀ ਇਹੀ ਕੁਝ ਬਣਾਉਣ ਦੇਵੇਗਾ।

## Getting Started

ਸਭ ਤੋਂ ਪਹਿਲਾਂ, ਤੁਹਾਡੇ ਕੋਲ [**AffiliateWP**](https://affiliatewp.com/) **ਐਕਟਿਵ** ਹੋਣਾ **ਜ਼ਰੂਰੀ** ਹੈ! ਜੇ ਤੁਹਾਡੇ ਕੋਲ **ਇਸ ਨਾਲ ਇੰਟੀਗ੍ਰੇਟ ਹੋਣ ਵਾਲਾ plugin** ਨਹੀਂ ਹੈ, ਤਾਂ ਤੁਸੀਂ ਸਾਡੇ add-on ਦੀ ਵਰਤੋਂ ਨਹੀਂ ਕਰ ਸਕੋਗੇ। ਜੇ ਤੁਸੀਂ ਆਪਣੇ clients ਨੂੰ ਆਪਣੀਆਂ ਸਾਈਟਾਂ 'ਤੇ ਵੀ ਇਸਦੀ ਵਰਤੋਂ ਕਰਨ ਦੀ ਯੋਜਨਾ ਨਹੀਂ ਬਣਾ ਰਹੇ, ਤਾਂ AffiliateWP ਸਿਰਫ਼ ਤੁਹਾਡੀ ਮੁੱਖ ਸਾਈਟ 'ਤੇ ਐਕਟਿਵ ਹੋਣਾ ਚਾਹੀਦਾ ਹੈ।

ਨਾਲ ਹੀ, ਤੁਹਾਨੂੰ **Ultimate Multisite** ਅਤੇ **Ultimate Multisite: AffiliateWP** ਦੋਵਾਂ ਨੂੰ **network-wide** **ਐਕਟਿਵੇਟ** ਕਰਨ ਦੀ ਲੋੜ ਹੋਵੇਗੀ।

## Adding Ultimate Multisite as an AffiliateWP Integration

ਆਪਣੀ ਮੁੱਖ ਸਾਈਟ 'ਤੇ, Affiliates Settings 'ਤੇ ਜਾਓ, ਅਤੇ Integrations tab 'ਤੇ, Ultimate Multisite ਨੂੰ ਚੈੱਕ ਕਰੋ। ਇਹ [AffiliateWP](https://affiliatewp.com/) ਨੂੰ Ultimate Multisite sign-up flow ਤੋਂ ਵਿਜ਼ਿਟਾਂ ਅਤੇ conversions ਨੂੰ track ਕਰਨਾ ਸ਼ੁਰੂ ਕਰਨ ਲਈ ਕਹੇਗਾ।

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_ਹੇਠਾਂ Ultimate Multisite ਬਾਕਸ ਨੂੰ ਚੈੱਕ ਕਰੋ ਅਤੇ ਸੇਵ ਕਰੋ_

ਬੱਸ ਇੰਨਾ ਹੀ ਕਰਨਾ ਪੈਂਦਾ ਹੈ। referral links ਤੋਂ ਆਉਣ ਵਾਲੀਆਂ ਵਿਜ਼ਿਟਾਂ ਨੂੰ track ਕੀਤਾ ਜਾਵੇਗਾ ਅਤੇ ਜਦੋਂ ਉਹ ਯੂਜ਼ਰ ਬਾਅਦ ਵਿੱਚ ਆਪਣਾ ਪਹਿਲਾ ਭੁਗਤਾਨ ਕਰਦਾ ਹੈ, ਤਾਂ conversion ਬਣਾ ਦਿੱਤਾ ਜਾਵੇਗਾ ਅਤੇ commission ਉਸ affiliate ਨੂੰ ਦਿੱਤਾ ਜਾਵੇਗਾ ਜੋ conversion ਲਈ ਜ਼ਿੰਮੇਵਾਰ ਹੈ।

ਹਾਲਾਂਕਿ, ਸਿਰਫ਼ ਨਵੇਂ ਬਣਾਏ ਗਏ ਯੂਜ਼ਰ ਤੋਂ ਪ੍ਰਾਪਤ ਹੋਇਆ ਪਹਿਲਾ ਭੁਗਤਾਨ ਹੀ track ਕੀਤਾ ਜਾਵੇਗਾ। recurring referral payments ਨੂੰ ਐਨੇਬਲ ਕਰਨ ਲਈ, ਤੁਹਾਨੂੰ [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) ਦੀ ਲੋੜ ਹੋਵੇਗੀ।

## Enabling Recurring Referrals

[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) ਨੂੰ ਇੰਸਟਾਲ ਕਰਨ ਤੋਂ ਬਾਅਦ, Affiliates Settings Recurring Referrals tab 'ਤੇ ਜਾਓ, ਅਤੇ recurring referrals ਦੀ ਵਰਤੋਂ ਨੂੰ ਐਨੇਬਲ ਕਰੋ।

<!-- Screenshot unavailable: AffiliateWP Recurring Referrals settings with Enable Recurring Referrals toggle -->

_ਇਸ ਤੋਂ ਵੀ ਸੌਖਾ: Recurring Referrals ਨੂੰ ਐਨੇਬਲ ਕਰੋ ਅਤੇ ਤੁਸੀਂ ਤਿਆਰ ਹੋ!_

ਦੁਬਾਰਾ, ਬੱਸ ਇੰਨਾ ਹੀ ਕਰਨਾ ਪੈਂਦਾ ਹੈ!

ਹੁਣ, ਜਦੋਂ ਵੀ ਕਿਸੇ refer किए गए account 'ਤੇ ਨਵਾਂ ਭੁਗਤਾਨ ਪ੍ਰਾਪਤ ਹੁੰਦਾ ਹੈ, ਤਾਂ ਜ਼ਿੰਮੇਵਾਰ affiliate ਨੂੰ ਆਪਣੇ ਆਪ ਹੀ ਆਪਣਾ ਹਿੱਸਾ ਮਿਲ ਜਾਵੇਗਾ!
