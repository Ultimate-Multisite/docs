# Action: wp_ultimo_affiliatewp_load {#action-wpultimoaffiliatewpload}

ਜਦੋਂ ਸਾਰੀਆਂ ਨਿਰਭਰਤਾਵਾਂ (dependencies) ਲੋਡ ਹੋ ਜਾਂਦੀਆਂ ਹਨ, ਤਾਂ ਇਹ ਟ੍ਰਿਗਰ ਹੁੰਦਾ ਹੈ।

ਇਹ ਪਲੱਗਇਨ ਡਿਵੈਲਪਰਾਂ ਨੂੰ ਨਵੀਂ ਕਾਰਜਕੁਸ਼ਲਤਾ (functionality) ਜੋੜਨ ਦੀ ਇਜਾਜ਼ਤ ਦਿੰਦਾ ਹੈ। ਜਿਵੇਂ ਕਿ, ਨਵੇਂ ਹੋਸਟਿੰਗ ਪ੍ਰੋਵਾਈਡਰਾਂ ਲਈ ਸਪੋਰਟ ਜੋੜਨਾ, ਆਦਿ।
