# ਮਲਟੀ-ਨੈੱਟਵਰਕ ਚੇਂਜਲੌਗ

### 1.3.0
* ਨਵਾਂ: ਨੈੱਟਵਰਕ ਟੈਂਪਲੇਟ ਪ੍ਰੀਵਿਊਅਰ — ਖਰੀਦਣ ਤੋਂ ਪਹਿਲਾਂ ਲਾਈਵ ਪੈਨਲ ਨਾਲ ਨੈੱਟਵਰਕ ਟੈਂਪਲੇਟਾਂ ਨੂੰ ਬ੍ਰਾਊਜ਼ ਅਤੇ ਪ੍ਰੀਵਿਊ ਕਰੋ।
* ਨਵਾਂ: ਚੈੱਕਆਉਟ 'ਤੇ ਨੈੱਟਵਰਕ ਟੈਂਪਲੇਟ ਪਿਕਰ — ਗਾਹਕ ਸਾਈਨਅੱਪ ਚੈੱਕਆਉਟ ਫਲੋ ਦੌਰਾਨ ਨੈੱਟਵਰਕ ਟੈਂਪਲੇਟ ਚੁਣ ਸਕਦੇ ਹਨ।
* ਨਵਾਂ: ਕ੍ਰਾਸ-ਨੈੱਟਵਰਕ ਮੈਜਿਕ ਲਿੰਕ SSO — ਗਾਹਕ ਮੈਜਿਕ-ਲਿੰਕ অথেন্টিਕੇਸ਼ਨ ਰਾਹੀਂ ਸਬ-ਨੈੱਟਵਰਕ ਤੱਕ ਆਸਾਨੀ ਨਾਲ ਪਹੁੰਚ ਕਰ ਸਕਦੇ ਹਨ।
* ਨਵਾਂ: ਨੈੱਟਵਰਕ ਮੀਡੀਆ ਕਲੋਨਿੰਗ — ਟੈਂਪਲੇਟ ਤੋਂ ਨੈੱਟਵਰਕ ਕਲੋਨ ਕਰਦੇ ਸਮੇਂ ਸਾਈਟ ਮੀਡੀਆ ਸਹੀ ਢੰਗ ਨਾਲ ਡੁਪਲੀਕੇਟ ਹੋ ਜਾਂਦਾ ਹੈ।
* ਫਿਕਸ: ਨੈੱਟਵਰਕ ਕਲੋਨਿੰਗ ਹੁਣ ਮਲਟੀ-ਟੈਨੈਂਸੀ ਐਡਨ ਨਾਲ ਕੰਮ ਕਰਦਾ ਹੈ; ਸੂਪਰ ਐਡਮਿਨ ਪ੍ਰਵੀਲੇਜ ਕਲੋਨ ਤੋਂ ਬਾਅਦ ਮੁੜ ਦਿੱਤੇ ਜਾਂਦੇ ਹਨ।
* ਫਿਕਸ: ਨੈੱਟਵਰਕ ਬਣਾਉਣਾ ਹੁਣ WooCommerce ਅਤੇ ਸਾਰੇ ਪੇਮੈਂਟ ਗੇਟਵੇ ਨਾਲ compatible ਹੈ।
* ਫਿਕਸ: ਫਾਲਬੈਕ ਡੋਮੇਨ ਜਨਰੇਸ਼ਨ ਵਿੱਚ ਹੁਣ ਪੋਰਟ ਨੰਬਰ ਸ਼ਾਮਲ ਹੈ।
* ਫਿਕਸ: ਟੈਂਪਲੇਟ ਪਿਕਰ CSS ਨੂੰ ਸਾਰੇ تھیਮਾਂ 'ਤੇ ਸਹੀ ਰੈਂਡਰਿੰਗ ਲਈ ਇੱਕ enqueued stylesheet ਵਿੱਚ ਤਬਦੀਲ ਕੀਤਾ ਗਿਆ।
* ਫਿਕਸ: ਸਾਈਟ ਕੁਇਰੀ ਸਕੋਪ ਵਿੱਚ undefined `network_id` key ਦੇ ਖ਼ਤਰੇ ਤੋਂ ਬਚਾਅ।
* ਫਿਕਸ: ਕੰਟੈਕਸਟ ਸਵਿਚਿੰਗ ਤੋਂ ਪਹਿਲਾਂ ਨੈੱਟਵਰਕ ਪਰਮਿਸ਼ਨ ਚੈੱਕ `wpdb` ਟੇਬਲ ਪ੍ਰੋਪਰਟੀ 에ਰਰ ਨੂੰ ਰੋਕਦਾ ਹੈ।
