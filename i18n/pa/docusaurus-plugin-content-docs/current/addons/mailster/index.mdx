## ਮੁੱਖ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ (Key Features) {#key-features}

**Automatic Subscriber Sync (ਆਟੋਮੈਟਿਕ ਸਬਸਕਰਾਈਬਰ ਸਿੰਕ)**
* ਸਾਈਨ ਅਪ ਕਰਨ 'ਤੇ ਗਾਹਕਾਂ ਨੂੰ ਆਪਣੇ ਆਪ Mailster lists ਵਿੱਚ ਸ਼ਾਮਲ ਕਰੋ
* ਸਾਈਨ ਅਪ 'ਤੇ ਤੁਰੰਤ ਸਿੰਕ ਕਰਨਾ ਚੁਣੋ ਜਾਂ ਭੁਗਤਾਨ ਦੀ ਪੁਸ਼ਟੀ ਦਾ ਇੰਤਜ਼ਾਰ ਕਰੋ

**GDPR-Compliant Opt-In Options (GDPR-ਪਾਲਣਯੋਗ Opt-In ਵਿਕਲਪ)**

**Smart Data Mapping (ਸਮਾਰਟ ਡਾਟਾ ਮੈਪਿੰਗ)**

## ਇਹ ਕਿਵੇਂ ਕੰਮ ਕਰਦਾ ਹੈ (How It Works) {#how-it-works}

1. ਆਪਣੇ Ultimate Multisite network 'ਤੇ addon ਨੂੰ install ਅਤੇ activate ਕਰੋ
2. Ultimate Multisite settings ਵਿੱਚ ਗਲੋਬਲ ਡਿਫਾਲਟ lists configure ਕਰੋ
3. ਵਿਕਲਪਿਕ ਤੌਰ 'ਤੇ ਪ੍ਰੋਡਕਟ-ਵਿਸ਼ੇਸ਼ lists ਨਾਲ override ਕਰੋ
4. ਆਪਣਾ opt-in mode ਅਤੇ ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਦਾ ਸਮਾਂ ਨਿਰਧਾਰਤ ਕਰੋ
5. ਆਪਣੇ Mailster lists ਨੂੰ ਆਟੋਮੈਟਿਕ ਵਧਦੇ ਦੇਖੋ!

## Installation (ਇੰਸਟਾਲੇਸ਼ਨ) {#installation}

1. plugin files ਨੂੰ ਆਪਣੇ Ultimate Multisite addons directory ਵਿੱਚ upload ਕਰੋ
2. ਯਕੀਨੀ ਬਣਾਓ ਕਿ Mailster ਤੁਹਾਡੀ ਮੁੱਖ ਸਾਈਟ 'ਤੇ install ਅਤੇ active ਹੈ

5. ਆਪਣੀਆਂ ਡਿਫਾਲਟ lists ਚੁਣੋ ਅਤੇ opt-in preferences configure ਕਰੋ
