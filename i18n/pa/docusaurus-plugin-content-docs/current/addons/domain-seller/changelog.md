- ਸੁਧਾਰ: WP-core query-var ਅਣਮੈਲਾਂ ਨੂੰ ਰੋਕਣ ਲਈ ਚੈਕਆਉਟ ਫਾਰਮ-ਐਕਸ਼ਨ ਵਿਹਾਰ ਬਿਹਤਰ ਕੀਤਾ
