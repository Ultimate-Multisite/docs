# Fluent Forms {#fluent-forms}

ਇਹ ਐਡ-ਆਨ ਤੁਹਾਨੂੰ Ultimate Multisite ਵਿੱਚ products, memberships, ਜਾਂ sites 'ਤੇ Fluent Forms ਲਈ ਲਿਮਿਟ (limits) ਜੋੜਨ ਦੇ ਯੋਗ ਬਣਾਉਂਦਾ ਹੈ।
Fluent Forms custom post types ਦੀ ਬਜਾਏ custom tables ਦੀ ਵਰਤੋਂ ਕਰਦਾ ਹੈ, ਇਸ ਲਈ ਆਮ ਲਿਮਿਟਿੰਗ ਵਿਕਲਪ Fluent Forms ਨਾਲ ਕੰਮ ਨਹੀਂ ਕਰਦੇ।

## Features {#features}

- Fluent Forms ਜਾਂ Conversational Fluent Forms ਲਈ ਲਿਮਿਟ ਜੋੜੋ
- ਜਦੋਂ ਗਾਹਕ ਲਿਮਿਟ ਪਹੁੰਚ ਜਾਂਦੇ ਹਨ ਤਾਂ ਉਨ੍ਹਾਂ ਨੂੰ ਇੱਕ ਦੋਸਤਾਨਾ ਸੁਨੇਹਾ ਮਿਲਦਾ ਹੈ ਜੋ ਉਨ੍ਹਾਂ ਨੂੰ ਆਪਣਾ ਪਲਾਨ ਅਪਗ੍ਰੇਡ ਕਰਨ ਲਈ ਕਹਿੰਦਾ ਹੈ
