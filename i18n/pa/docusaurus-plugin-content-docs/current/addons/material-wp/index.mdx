# Material WP

WordPress ਦਾ ਐਡਮਿਨ ਇੰਟਰਫੇਸ ਬਹੁਤ ਸਮੇਂ ਤੋਂ ਇੱਕੋ ਜਿਹਾ ਰਿਹਾ ਹੈ। ਪਰ ਕੀ ਤੁਸੀਂ ਕਦੇ ਸੋਚਿਆ ਹੈ ਕਿ ਐਡਮਿਨ ਇੰਟਰਫੇਸ ਦਾ Material Design ਵਰਜ਼ਨ ਕਿਹੋ ਜਿਹਾ ਲੱਗੇਗਾ?

Material WP ਇੱਕ ਐਡਮਿਨ تھیਮ ਹੈ ਜੋ ਤੁਹਾਨੂੰ ਉਹ ਲੁੱਕ ਅਤੇ ਫੀਲ (look and feel) ਦੇਣ ਲਈ ਤਿਆਰ ਕੀਤਾ ਗਿਆ ਹੈ, ਜਿਸ ਨਾਲ ਤੁਸੀਂ ਆਪਣੇ ਕਲਾਇੰਟਾਂ ਨੂੰ ਇੱਕ ਹੋਰ ਕਸਟਮਾਈਜ਼ਡ ਡੈਸ਼ਬੋਰਡ (Dashboard) ਦੇ ਸਕਦੇ ਹੋ।

## ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ (Features) {#features}

- ਕਸਟਮ CSS ਸਪੋਰਟ
- ਆਸਾਨ ਸਟਾਈਲਿੰਗ ਸੈਟਿੰਗਾਂ
- ਮੀਨੂ ਨੂੰ ਦੁਬਾਰਾ ਸੈਟ ਕਰਨਾ (Menu reordering)
- ਆਸਾਨ ਇੰਪੋਰਟ/ਐਕਸਪੋਰਟ

## ਡਾਕੂਮੈਂਟੇਸ਼ਨ (Documentation) {#documentation}

**Material WP ਨਾਲ ਆਪਣੇ ਐਡਮਿਨ ਪੈਨਲ ਨੂੰ ਕਸਟਮਾਈਜ਼ ਕਰਨਾ**

## ਇੰਸਟਾਲੇਸ਼ਨ (Installation) {#installation}

1. 'material-wp' ਨੂੰ '/wp-content/plugins/' ਡਾਇਰੈਕਟਰੀ ਵਿੱਚ ਅਪਲੋਡ ਕਰੋ
2. WordPress ਵਿੱਚ 'Plugins' ਮੀਨੂ ਰਾਹੀਂ ਪਲੱਗਇਨ ਐਕਟੀਵੇਟ ਕਰੋ

## ਵਰਤੋਂ ਗਾਈਡ (Usage Guide) {#usage-guide}

WordPress ਦਾ ਐਡਮਿਨ ਇੰਟਰਫੇਸ ਬਹੁਤ ਸਮੇਂ ਤੋਂ ਇੱਕੋ ਜਿਹਾ ਰਿਹਾ ਹੈ। ਪਰ ਕੀ ਤੁਸੀਂ ਕਦੇ ਸੋਚਿਆ ਹੈ ਕਿ ਐਡਮਿਨ ਇੰਟਰਫੇਸ ਦਾ Material Design ਵਰਜ਼ਨ ਕਿਹੋ ਜਿਹਾ ਲੱਗੇਗਾ?

Material WP ਇੱਕ ਐਡਮਿਨ تھیਮ ਹੈ ਜੋ ਤੁਹਾਨੂੰ ਉਹ ਲੁੱਕ ਅਤੇ ਫੀਲ (look and feeling) ਦੇਣ ਲਈ ਤਿਆਰ ਕੀਤਾ ਗਿਆ ਹੈ, ਜਿਸ ਨਾਲ ਤੁਸੀਂ ਆਪਣੇ ਕਲਾਇੰਟਾਂ ਨੂੰ ਇੱਕ ਹੋਰ ਕਸਟਮਾਈਜ਼ਡ ਡੈਸ਼ਬੋਰਡ (Dashboard) ਦੇ ਸਕਦੇ ਹੋ। Material WP, Ultimate Multisite ਦੇ ਪ੍ਰੀਮੀਅਮ ਪਲੱਗਇਨ ਕੋਰ ਦਾ ਵੀ ਹਿੱਸਾ ਹੈ।

ਇਸ ਨੂੰ **ਇੰਸਟਾਲ ਕਰਨ** ਤੋਂ ਬਾਅਦ, ਆਪਣੇ _Plugins_ ਪੇਜ 'ਤੇ ਜਾਓ ਅਤੇ ਆਪਣੇ ਨੈੱਟਵਰਕ (Network) 'ਤੇ ਇਸਨੂੰ ਐਕਟੀਵੇਟ ਕਰਨ ਲਈ ਕਲਿੱਕ ਕਰੋ।

_ਆਪਣੇ Plugins ਪੇਜ 'ਤੇ ਨੈੱਟਵਰਕ-ਵਾਈਜ਼ ਐਕਟੀਵੇਟ ਕਰੋ_

_**ਨੋਟ:**_ _ਜੇ ਤੁਸੀਂ ਸਿਰਫ਼ ਇਸਨੂੰ ਐਕਟੀਵੇਟ ਕਰਦੇ ਹੋ, ਤਾਂ ਇਹ ਤੁਹਾਡੇ ਕਲਾਇੰਟ ਦੇ ਸਾਹਮਣੇ ਇੱਕ ਵਿਕਲਪ ਵਜੋਂ ਦਿਖਾਈ ਦੇਵੇਗਾ ਜਿਸਨੂੰ ਸਿਰਫ਼ ਉਸ ਯੂਜ਼ਰ ਲਈ تھیਮ ਜੋੜਨ ਲਈ ਨਿਸ਼ਾਨ ਲਗਾ ਸਕਦਾ ਹੈ।_

ਹੁਣ, ਤੁਹਾਡਾ ਐਡਮਿਨ ਇੰਟਰਫੇਸ ਪੂਰੀ ਤਰ੍ਹਾਂ ਨਾਲ ਰੀਡਿਜ਼ਾਈਨ ਹੋ ਗਿਆ ਹੈ! ਤੁਹਾਨੂੰ ਖੱਬੇ ਕਾਲਮ ਵਿੱਚ Material WP ਨਾਮ ਦਾ ਇੱਕ ਨਵਾਂ ਮੀਨੂ ਵੀ ਮਿਲਦਾ ਹੈ
