# Content Sync

Multisite Ultimate: Content Sync

Multisite Ultimate ਲਈ ਇਹ addon network administrators ਨੂੰ network ਦੇ ਕਿਸੇ ਵੀ site ਤੋਂ content ਨੂੰ network ਦੇ ਹੋਰ sites 'ਤੇ sync (samānjit) ਕਰਨ ਦੀ ਇਜਾਜ਼ਤ ਦਿੰਦਾ ਹੈ। ਇਹ posts, pages, custom post types, ਅਤੇ Elementor templates ਸਮੇਤ ਸਾਰੇ post types ਨੂੰ support ਕਰਦਾ ਹੈ। ਇਸ addon ਵਿੱਚ content ਚੁਣਨ ਅਤੇ target sites ਦੀ ਚੋਣ ਕਰਨ ਲਈ ਇੱਕ multi-step wizard interface ਸ਼ਾਮਲ ਹੈ, ਜਿਸ ਵਿੱਚ batch processing ਅਤੇ real-time progress feedback ਮਿਲਦਾ ਹੈ।

## Features {#features}

* Content Selection: Thumbnails ਅਤੇ post type filtering ਵਾਲਾ visual card grid
* All Post Types: posts, pages, products, Elementor templates, ਅਤੇ ਕੋਈ ਵੀ custom post type sync ਕਰੋ
* Site Selection: ਚੁਣੋ ਕਿ content ਕਿਹੜੇ sites 'ਤੇ sync ਕਰਨਾ ਹੈ, ਜਿਸ ਵਿੱਚ source site ਨੂੰ exclude ਕਰਨਾ ਅਤੇ child site ਨੂੰ pre-select ਕਰਨਾ ਸ਼ਾਮਲ ਹੈ
* Batch Processing: ਇੱਕ ਵਾਰ ਵਿੱਚ 10 sites 'ਤੇ sync ਕਰਦਾ ਹੈ, ਜਿਸ ਨਾਲ progress updates ਮਿਲਦੇ ਹਨ
* Media Sync: post content ਵਿੱਚੋਂ media files (images, videos) ਨੂੰ target sites 'ਤੇ ਆਪਣੇ ਆਪ copy ਕਰਦਾ ਹੈ ਅਤੇ unmap ਕਰਦਾ ਹੈ
* Taxonomy Sync: categories, tags, ਅਤੇ custom taxonomies ਨੂੰ preserve ਕਰਦਾ ਹੈ
* Elementor Support: CSS regeneration ਅਤੇ template ID remapping ਨਾਲ Elementor templates ਲਈ ਪੂਰਾ support
* Conflict Resolution: ਮੌਜੂਦਾ content ਨੂੰ replace ਕਰਨ ਜਾਂ skip ਕਰਨ ਦਾ option
* Cache Management: sync ਤੋਂ ਬਾਅਦ target sites 'ਤੇ Elementor cache ਨੂੰ ਆਪਣੇ ਆਪ clear ਕਰਦਾ ਹੈ

## Requirements {#requirements}

* WordPress 5.3 ਜਾਂ ਇਸ ਤੋਂ ਵੱਧ
* PHP 7.4 ਜਾਂ ਇਸ ਤੋਂ ਵੱਧ
* Multisite Ultimate plugin (active)
* Elementor plugin (optional, Elementor template sync ਲਈ required)

## Installation {#installation}

1. 'ultimate-multisite-content-sync' ਨੂੰ '/wp-content/plugins/' directory ਵਿੱਚ upload ਕਰੋ
2. ਆਪਣੇ WordPress Network Admin ਵਿੱਚ 'Plugins' menu ਰਾਹੀਂ plugin activate ਕਰੋ
3. Ultimate Multisite → Settings → Content Sync ਵਿੱਚ settings configure ਕਰੋ
4. Syncing ਸ਼ੁਰੂ ਕਰਨ ਲਈ Ultimate Multisite → Content Sync 'ਤੇ ਜਾਓ
