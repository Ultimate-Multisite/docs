# Support Tickets {#support-tickets}

**Support Tickets** Ultimate Multisite ਲਈ ਇੱਕ ਸ਼ਕਤੀਸ਼ਾਲੀ addon ਹੈ ਜੋ ਤੁਹਾਡੇ WordPress network ਵਿੱਚ ਇੱਕ ਪੂਰਾ ਸਪੋਰਟ ਟਿਕਟ ਸਿਸਟਮ ਜੋੜਦਾ ਹੈ। ਇਹ SaaS platforms, WaaS (Website as a Service) providers, ਅਤੇ multisite networks ਲਈ ਬਿਲਕੁਲ ਸਹੀ ਹੈ ਜਿਨ੍ਹਾਂ ਨੂੰ network ਅਤੇ ਵਿਅਕਤੀਗਤ site ਦੋਵਾਂ ਪੱਧਰਾਂ 'ਤੇ ਗਾਹਕ ਸਪੋਰਟ ਪ੍ਰਦਾਨ ਕਰਨ ਦੀ ਲੋੜ ਹੁੰਦੀ ਹੈ।

## Support Tickets ਕਿਉਂ ਚੁਣें? {#why-choose-support-tickets}

* **ਕਈ ਪੱਧਰਾਂ ਦਾ ਸਪੋਰਟ (Multi-Level Support)**: Network admins network-ਵਿਆਪੀ ਟਿਕਟਾਂ ਨੂੰ ਸੰਭਾਲਦੇ ਹਨ, ਜਦੋਂ ਕਿ site owners ਆਪਣੇ ਗਾਹਕ ਸਪੋਰਟ ਦਾ ਪ੍ਰਬੰਧਨ ਕਰਦੇ ਹਨ।
* **ਪੇਸ਼ੇਵਰ ਕੰਮਕਾਚ (Professional Workflow)**: ਤਰਜੀਹਾਂ (priorities), ਸਥਿਤੀਆਂ (statuses), ਅਤੇ ਅਸਾਈਨਮੈਂਟਾਂ ਨਾਲ ਪੂਰੇ ਟਿਕਟ life cycle ਦਾ ਪ੍ਰਬੰਧਨ।
* **ਗਾਹਕ ਅਨੁਭਵ (Customer Experience)**: ਇੱਕ ਸੁੰਦਰ, responsive interface ਜੋ ਗਾਹਕਾਂ ਲਈ ਮਦਦ ਲੈਣਾ ਆਸਾਨ ਬਣਾਉਂਦਾ ਹੈ।
* **ਸੁਰੱਖਿਅਤ ਅਤੇ ਭਰੋਸੇਮੰਦ (Secure & Reliable)**: WordPress security best practices ਨਾਲ ਬਣਾਇਆ ਗਿਆ ਅਤੇ optimized database performance ਵਾਲਾ।
* **ਸਮੂਲੀਕ ਏਕੀਕਰਨ (Seamless Integration)**: Ultimate Multisite ਦੇ ਗਾਹਕ ਅਤੇ site management ਨਾਲ ਬਿਲਕੁਲ ਚੰਗੀ ਤਰ੍ਹਾਂ ਕੰਮ ਕਰਦਾ ਹੈ।

## ਮੁੱਖ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ (Key Features) {#key-features}

**ਪੂਰਾ ਟਿਕਟ ਪ੍ਰਬੰਧਨ (Complete Ticket Management)**

* ਆਸਾਨ ਹਵਾਲੇ ਲਈ ਵਿਲੱਖਣ ਟਿਕਟ ਨੰਬਰਿੰਗ ਸਿਸਟਮ
* ਕਨਫਿਗਰੇਬਲ ਤਰਜੀਹ ਪੱਧਰ (Low, Normal, High, Critical)
* ਵਿਆਪਕ ਸਥਿਤੀ ਕੰਮਕਾਚ (Open, In Progress, Waiting, Resolved, Closed)
* ਟਿਕਟ ਵਰਗੀਕਰਨ (General, Technical, Billing, Feature Request, Bug Report)
* ਟਿਕਟਾਂ ਨੂੰ ਸਪੋਰਟ agents ਨੂੰ ਅਸਾਈਨ ਕਰਨਾ, ਜਿਸ ਵਿੱਚ ঐচ্ছਿਕ auto-assignment ਸ਼ਾਮਲ ਹੈ।

**ਥ੍ਰੀਡਡ ਗੱਲਬਾਤ (Threaded Conversations)**

* ਟਿਕਟ ਜਵਾਬਾਂ ਲਈ ਪੂਰੀ ਗੱਲਬਾਤ ਥ੍ਰੀਡਿੰਗ।
* ਸਟਾਫ ਅਤੇ ਗਾਹਕ ਜਵਾਬਾਂ ਲਈ ਵੱਖਰਾ ਸਟਾਈਲਿੰਗ।
* ਗੋਪਨੀਯ ਸਟਾਫ ਨੋਟਸ ਜੋ ਗਾਹਕ ਨਹੀਂ ਦੇਖ ਸਕਦੇ।
* real-time AJAX-powered interface।
* ਫਾਰਮੈਟਿੰਗ ਵਿਕਲਪਾਂ ਵਾਲਾ ਆਧੁਨਿਕ text editor।

**ਫਾਈਲ ਅਟੈਚਮੈਂਟਸ (File Attachments)**

* ਡ੍ਰੈਗ-ਐਂਡ-ਡ੍ਰੌਪ ਸਪੋਰਟ ਨਾਲ ਸੁਰੱਖਿਅਤ ਫਾਈਲ ਅਪਲੋਡ।
* ਕਨਫਿਗਰੇਬਲ ਫਾਈਲ ਕਿਸਮ ਅਤੇ ਆਕਾਰ ਦੀਆਂ ਪਾਬੰਦੀਆਂ।
* ਮਾਲੀਸ਼ੀਅਸ ਫਾਈਲ ਪਛਾਣ ਨਾਲ ਉੱਨਤ ਸੁਰੱਖਿਆ।
* ਐਕਸੈਸ ਕੰਟਰੋਲ ਨਾਲ ਸੁਰੱਖਿਅਤ ਡਾਊਨਲੋਡ।

**ਸਮਾਰਟ ਨੋਟੀਫਿਕੇਸ਼ਨ (Smart Notifications)**

* ਨਵੇਂ ਟਿਕਟਾਂ, ਜਵਾਬਾਂ, ਅਤੇ ਸਥਿਤੀ ਬਦਲਾਅ ਲਈ ਆਟੋਮੈਟਿਕ email alerts।
* ਕਸਟਮਾਈਜ਼ੇਬਲ email templates।
* ਸਟਾਫ ਅਤੇ ਗਾਹਕਾਂ ਲਈ role-based notifications।
* ਵਿਅਕਤੀਗਤ ਸੰਦੇਸ਼ਾਂ ਲਈ dynamic template variables।

**ਸ਼ਕਤੀਸ਼ਾਲੀ Admin Interface**

* ਸਾਰਟਿੰਗ ਅਤੇ ਫਿਲਟਰਿੰਗ ਵਾਲੀ ਉੱਨਤ ਟਿਕਟ ਸੂਚੀ।
* ਸਥਿਤੀ, ਤਰਜੀਹ, ਅਤੇ ਅਸਾਈਨਮੈਂਟ ਲਈ ਛੇਤੀ ਸੰਪਾਦਨ (Quick edit)।
* ਕੁਸ਼ਲ ਪ੍ਰਬੰਧਨ ਲਈ bulk actions।
* performance metrics ਵਾਲਾ statistics dashboard।
* ਵਿਆਪਕ ਸੈਟਿੰਗਾਂ ਪੈਨਲ।

**ਸੁੰਦਰ ਗਾਹਕ ਪੋਰਟਲ (Beautiful Customer Portal)**

* ਸਾਰੇ devices 'ਤੇ ਸਾਫ਼, responsive interface।
* Mobile-first design।
* ਉੱਨਤ ਖੋਜ ਅਤੇ ਫਿਲਟਰਿੰਗ।
* ਫਾਰਮੈਟਿੰਗ ਟੂਲਬਾਰ ਵਾਲਾ ਆਧੁਨਿਕ editor।
* ਵਰਤਣ ਵਿੱਚ ਆਸਾਨ ਟਿਕਟ ਜਮ੍ਹਾਂ ਕਰਨ ਦੇ ਫਾਰਮ।

**ਲਚਕੀਲਾ ਏਕੀਕਰਨ (Flexible Integration)**

* shortcodes ਜਾਂ Ultimate Multisite UI elements ਦੀ ਵਰਤੋਂ ਕਰੋ।
* ਆਪਣੇ site ਦੇ ਕਿਸੇ ਵੀ ਹਿੱਸੇ 'ਤੇ ਟਿਕਟ ਫਾਰਮ ਅਤੇ ਸੂਚੀ ਪ੍ਰਦਰਸ਼ਿਤ ਕਰੋ।
* multisite network structure ਨਾਲ ਬਿਲਕੁਲ ਚੰਗੀ ਤਰ੍ਹਾਂ ਕੰਮ ਕਰਦਾ ਹੈ।
* hooks ਅਤੇ filters ਨਾਲ ਵਧਾਇਆ ਜਾ ਸਕਦਾ ਹੈ।

## ਕਿਸ ਲਈ ਸਹੀ (Perfect For) {#perfect-for}

* ਗਾਹਕ ਸਪੋਰਟ ਦੀ ਲੋੜ ਵਾਲੇ SaaS platforms
* ਕਈ client sites ਦਾ ਪ੍ਰਬੰਧਨ ਕਰਨ ਵਾਲੇ WaaS providers
* ਪੱਧਰੀ ਸਪੋਰਟ ਵਾਲੇ Membership sites
* ਕਈ brand ਦਾ ਪ੍ਰਬੰਧਨ ਕਰਨ ਵਾਲੇ Network administrators
* ਗਾਹਕ ਸੇਵਾ ਪ੍ਰਦਾਨ ਕਰਨ ਵਾਲੇ Site owners

## ਸਥਾਪਨਾ (Installation) {#installation}

1. ਯਕੀਨੀ ਬਣਾਓ ਕਿ Ultimate Multisite plugin ਤੁਹਾਡੇ WordPress network 'ਤੇ ਸਥਾਪਿਤ ਅਤੇ activate ਹੈ।
2. addon files ਨੂੰ ਆਪਣੇ `/wp-content/plugins/` directory ਵਿੱਚ upload ਕਰੋ।
3. WordPress ਵਿੱਚ 'Plugins' menu ਰਾਹੀਂ network 'ਤੇ plugin activate ਕਰੋ।
4. ਆਪਣੀਆਂ ਸੈਟਿੰਗਾਂ ਨੂੰ configure ਕਰਨ ਲਈ Network Admin > Support Tickets 'ਤੇ ਜਾਓ।

## ਅਕਸਰ ਪੁੱਛੇ ਜਾਣ ਵਾਲੇ ਸਵਾਲ (Frequently Asked Questions) {#frequently-asked-questions}

## ਕੀ ਇਸ ਲਈ Ultimate Multisite ਦੀ ਲੋੜ ਹੈ? {#does-this-require-ultimate-multisite}

ਹਾਂ, ਇਸ addon ਲਈ Ultimate Multisite ਦਾ ਸਥਾਪਿਤ ਅਤੇ active ਹੋਣਾ ਜ਼ਰੂਰੀ ਹੈ। ਇਹ Ultimate Multisite ਦੀਆਂ ਗਾਹਕ ਅਤੇ site management features ਨਾਲ ਡੂੰਘਾ ਏਕੀਕਰਨ ਕਰਦਾ ਹੈ।

## ਕੀ site owners ਆਪਣੇ ਗਾਹਕ ਸਪੋਰਟ ਦਾ ਪ੍ਰਬੰਧਨ ਕਰ ਸਕਦੇ ਹਨ? {#can-site-owners-handle-their-own-customer-support}

ਬਿਲਕੁਲ! site owners ਆਪਣੇ ਗਾਹਕਾਂ ਨੂੰ ਸਪੋਰਟ ਪ੍ਰਦਾਨ ਕਰਨ ਲਈ ਟਿਕਟ ਸਿਸਟਮ ਦੀ ਵਰਤੋਂ ਕਰ ਸਕਦੇ ਹਨ, ਜਦੋਂ ਕਿ network admins network-ਵਿਆਪੀ ਸਪੋਰਟ ਟਿਕਟਾਂ ਨੂੰ ਸੰਭਾਲ ਸਕਦੇ ਹਨ।

## ਗਾਹਕ ਟਿਕਟਾਂ ਕਿਵੇਂ ਜਮ੍ਹਾਂ ਕਰਦੇ ਹਨ? {#how-do-customers-submit-tickets}

ਤੁਸੀਂ shortcodes ਜਾਂ Ultimate Multisite UI elements ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਆਪਣੇ site ਦੇ ਕਿਸੇ ਵੀ ਹਿੱਸੇ 'ਤੇ ਟਿਕਟ ਜਮ੍ਹਾਂ ਕਰਨ ਦੇ ਫਾਰਮ ਜੋੜ ਸਕਦੇ ਹੋ। ਗਾਹਕ ਸਿਰਫ਼ ਫਾਰਮ ਭਰਦੇ ਹਨ ਅਤੇ ਆਪਣਾ ਬੇਨਤੀ ਜਮ੍ਹਾਂ ਕਰਦੇ ਹਨ।

## ਕੀ ਮੈਂ ਟਿਕਟ ਸਥਿਤੀਆਂ ਅਤੇ ਤਰਜੀਹਾਂ ਨੂੰ ਕਸਟਮਾਈਜ਼ ਕਰ ਸਕਦਾ ਹਾਂ? {#can-i-customize-the-ticket-statuses-and-priorities}

ਹਾਂ! ਤੁਸੀਂ ਆਪਣੇ ਸਪੋਰਟ ਕੰਮਕਾਚ ਨਾਲ ਮੇਲ ਖਾਂਣ ਲਈ network admin settings ਵਿੱਚ ਤਰਜੀਹ ਪੱਧਰਾਂ, ਸਥਿਤੀਆਂ, ਅਤੇ ਟਿਕਟ ਕਿਸਮਾਂ ਨੂੰ ਪੂਰੀ ਤਰ੍ਹਾਂ ਕਸਟਮਾਈਜ਼ ਕਰ ਸਕਦੇ ਹੋ।

## ਕੀ ਇਹ ਫਾਈਲ ਅਟੈਚਮੈਂਟਸ ਸਪੋਰਟ ਕਰਦਾ ਹੈ? {#does-it-support-file-attachments}

ਹਾਂ, ਗਾਹਕ ਅਤੇ ਸਟਾਫ ਦੋਵੇਂ ਟਿਕਟਾਂ ਅਤੇ ਜਵਾਬਾਂ ਨਾਲ ਫਾਈਲਾਂ ਜੋੜ ਸਕਦੇ ਹਨ। ਤੁਹਾਡੇ ਕੋਲ ਸੁਰੱਖਿਆ ਲਈ允许 ਫਾਈਲ ਕਿਸਮਾਂ ਅਤੇ ਆਕਾਰ ਸੀਮਾਵਾਂ 'ਤੇ ਪੂਰਾ ਕੰਟਰੋਲ ਹੈ।

## ਨੋਟੀਫਿਕੇਸ਼ਨਾਂ ਦਾ ਪ੍ਰਬੰਧਨ ਕਿਵੇਂ ਹੁੰਦਾ ਹੈ? {#how-are-notifications-handled}

ਸਿਸਟਮ ਆਪਣੇ ਆਪ ਨਵੇਂ ਟਿਕਟਾਂ, ਜਵਾਬਾਂ, ਅਤੇ ਸਥਿਤੀ ਬਦਲਾਅ ਲਈ email notifications ਭੇਜਦਾ ਹੈ। ਤੁਸੀਂ ਸੈਟਿੰਗਾਂ ਵਿੱਚ ਸਾਰੇ email templates ਨੂੰ ਕਸਟਮਾਈਜ਼ ਕਰ ਸਕਦੇ ਹੋ।

## ਕੀ ਮੈਂ ਟਿਕਟਾਂ ਨੂੰ ਖਾਸ ਸਪੋਰਟ agents ਨੂੰ ਅਸਾਈਨ ਕਰ ਸਕਦਾ ਹਾਂ? {#can-i-assign-tickets-to-specific-support-agents}

ਹਾਂ, ਤੁਸੀਂ ਟਿਕਟਾਂ ਨੂੰ agents ਨੂੰ ਹੱਥੀਂ ਅਸਾਈਨ ਕਰ ਸਕਦੇ ਹੋ ਜਾਂ ਟਿਕਟਾਂ ਨੂੰ ਆਟੋਮੈਟਿਕ ਤੌਰ 'ਤੇ ਵੰਡਣ ਲਈ auto-assignment ਨੂੰ enable ਕਰ ਸਕਦੇ ਹੋ।

## ਕੀ ਸਟਾਫ ਲਈ ਗੋਪਨੀਯ ਨੋਟਸ ਹੁੰਦੇ ਹਨ? {#are-there-private-notes-for-staff}

ਹਾਂ, ਸਟਾਫ ਮੈਂਬਰ ਗੋਪਨੀਯ ਜਵਾਬ ਜੋੜ ਸਕਦੇ ਹਨ ਜੋ ਸਿਰਫ਼ ਹੋਰ ਸਟਾਫ ਮੈਂਬਰਾਂ ਨੂੰ ਦਿਖਾਈ ਦਿੰਦੇ ਹਨ, ਗਾਹਕਾਂ ਨੂੰ ਨਹੀਂ।

## ਕੀ ਇਹ mobile-friendly ਹੈ? {#is-it-mobile-friendly}

ਬਿਲਕੁਲ! ਪੂਰਾ interface mobile-first approach ਨਾਲ ਬਣਾਇਆ ਗਿਆ ਹੈ ਅਤੇ ਸਾਰੇ devices 'ਤੇ ਖੂਬਸੂਰਤੀ ਨਾਲ ਕੰਮ ਕਰਦਾ ਹੈ।

## ਕੀ ਮੈਂ shortcodes ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਟਿਕਟ ਸੂਚੀਆਂ ਅਤੇ ਫਾਰਮ ਪ੍ਰਦਰਸ਼ਿਤ ਕਰ ਸਕਦਾ ਹਾਂ? {#can-i-display-ticket-lists-and-forms-using-shortcodes}

ਹਾਂ! ਉਪਲਬਧ shortcodes:
* `[wu_support_tickets]` - ਗਾਹਕ ਟਿਕਟ ਸੂਚੀ ਪ੍ਰਦਰਸ਼ਿਤ ਕਰੋ
* `[wu_support_ticket_form]` - ਨਵਾਂ ਟਿਕਟ ਜਮ੍ਹਾਂ ਕਰਨ ਦਾ ਫਾਰਮ ਪ੍ਰਦਰਸ਼ਿਤ ਕਰੋ
* `[wu_support_ticket_view ticket_id="123"]` - ਖਾਸ ਟਿਕਟ ਪ੍ਰਦਰਸ਼ਿਤ ਕਰੋ

## ਕਿਹੜੀਆਂ ਫਾਈਲ ਕਿਸਮਾਂ upload ਕੀਤੀਆਂ ਜਾ ਸਕਦੀਆਂ ਹਨ? {#what-file-types-can-be-uploaded}

ਤੁਸੀਂ ਸੈਟਿੰਗਾਂ ਵਿੱਚ允许 ਫਾਈਲ ਕਿਸਮਾਂ ਨੂੰ configure ਕਰ ਸਕਦੇ ਹੋ। ਸਿਸਟਮ ਮਾਲੀਸ਼ੀਅਸ ਫਾਈਲਾਂ ਤੋਂ ਬਚਾਅ ਲਈ ਉੱਨਤ ਸੁਰੱਖਿਆ ਸਕੈਨਿੰਗ ਸ਼ਾਮਲ ਕਰਦਾ ਹੈ।

## ਕੀ ਮੈਂ ਟਿਕਟ ਡਾਟਾ export ਕਰ ਸਕਦਾ ਹਾਂ? {#can-i-export-ticket-data}

ਹਾਂ, addon ਵਿੱਚ ਟਿਕਟ ਡਾਟਾ ਅਤੇ ਸਟੈਟਿਸਟਿਕਸ ਲਈ export ਕਰਨ ਦੀ ਸਮਰੱਥਾ ਸ਼ਾਮਲ ਹੈ।
