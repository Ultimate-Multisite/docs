# Loco Translate {#loco-translate}

Ultimate Multisite: Loco Translate Integration addon ਤੁਹਾਡੇ WordPress multisite network ਨੂੰ site-specific translation capabilities ਨਾਲ ਜੋੜਦਾ ਹੈ। ਜੇਕਰ ਤੁਹਾਨੂੰ ਕਦੇ ਇਹ ਲੋੜ ਪਈ ਹੈ ਕਿ ਤੁਹਾਡੇ network ਦੇ ਵੱਖ-ਵੱਖ sites ਨੂੰ ਇੱਕੋ plugin ਜਾਂ theme ਦੇ ਵੱਖ-ਵੱਖ translations ਚਾਹੀਦੇ ਹਨ, ਤਾਂ ਇਹ addon ਸਹੀ ਹੱਲ ਹੈ।

**ਇਹ Addon ਕੀ ਕਰਦਾ ਹੈ**

ਇਹ addon Ultimate Multisite ਨੂੰ ਪ੍ਰਸਿੱਧ Loco Translate plugin ਨਾਲ ਬਿਨਾਂ ਕਿਸੇ ਰੁਕਾਵਟ ਦੇ ਜੋੜਦਾ ਹੈ, ਜਿਸ ਨਾਲ ਤੁਹਾਡੇ network ਦੇ ਹਰੇਕ subsite ਨੂੰ ਆਪਣੇ ਖੁਦ ਦੇ custom translations ਰੱਖਣ ਦੇ ਯੋਗ ਬਣਾਉਂਦਾ ਹੈ। ਭਾਵੇਂ ਤੁਸੀਂ ਇੱਕ multi-regional platform ਚਲਾ ਰਹੇ ਹੋ, ਵੱਖ-ਵੱਖ ਭਾਸ਼ਾ ਦੇ 커뮤ਨਿਟੀ ਨੂੰ ਸੇਵਾ ਦੇ ਰਹੇ ਹੋ, ਜਾਂ ਸਿਰਫ਼ ਆਪਣੇ network ਵਿੱਚ translation flexibility ਚਾਹੀਦੀ ਹੈ, ਇਹ addon ਇਸਨੂੰ ਬਹੁਤ ਆਸਾਨ ਬਣਾ ਦਿੰਦਾ ਹੈ।

**ਇਸ ਲਈ ਸਹੀ:**

* ਵੱਖ-ਵੱਖ localization ਲੋੜਾਂ ਵਾਲੇ multi-regional business
* ਵੱਖ-ਵੱਖ 커뮤니ਟੀ ਨੂੰ ਸੇਵਾ ਦੇਣ ਵਾਲੇ educational institutions
* multilingual support ਦੇਣ ਵਾਲੇ SaaS platforms
* ਕਈ client sites ਨੂੰ manage ਕਰਨ ਵਾਲੀਆਂ web agencies
* ਕਿਸੇ ਵੀ multisite network ਨੂੰ ਜਿਸਨੂੰ translation flexibility ਦੀ ਲੋੜ ਹੈ

## Installation {#installation}

1. ਯਕੀਨੀ ਬਣਾਓ ਕਿ Ultimate Multisite install ਹੋਇਆ ਹੈ ਅਤੇ network activate ਹੈ
2. Loco Translate plugin install ਅਤੇ activate ਕਰੋ
3. ਇਸ addon ਨੂੰ ਆਪਣੇ plugins directory ਵਿੱਚ upload ਕਰੋ
4. Loco Translate Integration addon ਨੂੰ network level 'activate' ਕਰੋ
5. ਕਿਸੇ ਵੀ subsite 'ਤੇ Loco Translate ਰਾਹੀਂ site-specific translations ਬਣਾਉਣਾ ਸ਼ੁਰੂ ਕਰੋ

ਇਹੀ! integration ਕਿਸੇ ਵਾਧੂ configuration ਤੋਂ ਬਿਨਾਂ ਆਪਣੇ ਆਪ ਕੰਮ ਕਰਦਾ ਹੈ।

## Frequently Asked Questions {#frequently-asked-questions}

## ਕੀ installation ਤੋਂ ਬਾਅਦ ਮੈਨੂੰ ਕੁਝ configure ਕਰਨ ਦੀ ਲੋੜ ਹੈ? {#do-i-need-to-configure-anything-after-installation}

ਨਹੀਂ! integration ਆਪਣੇ ਆਪ ਹੋ ਜਾਂਦਾ ਹੈ। ਇੱਕ ਵਾਰ ਜਦੋਂ addon ਅਤੇ Loco Translate ਦੋਵੇਂ active ਹੋ ਜਾਂਦੇ ਹਨ, ਤਾਂ subsites ਤੁਰੰਤ ਆਪਣੇ ਖੁਦ ਦੇ custom translations ਬਣਾਉਣਾ ਸ਼ੁਰੂ ਕਰ ਸਕਦੇ ਹਨ।

## ਕੀ ਹੋਵੇਗਾ ਜੇਕਰ ਕੋਈ subsite custom translations ਨਹੀਂ ਬਣਾਉਂਦਾ? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

site ਆਪਣੇ ਆਪ network-wide translations ਨੂੰ fallback ਵਜੋਂ ਵਰਤ ਲਵੇਗਾ। ਕੁਝ ਵੀ ਟੁੱਟਦਾ ਨਹੀਂ, ਅਤੇ ਸਭ ਕੁਝ ਬਿਨਾਂ ਕਿਸੇ ਰੁਕਾਵਟ ਦੇ ਕੰਮ ਕਰਦਾ ਰਹਿੰਦਾ ਹੈ।

## ਕੀ subsites network-wide translations ਤੱਕ ਪਹੁੰਚ ਕਰ ਸਕਦੇ ਹਨ? {#can-subsites-access-network-wide-translations}

ਆਮ site admins ਸਿਰਫ਼ ਆਪਣੇ site ਦੇ translations ਦੇਖ ਅਤੇ edit ਕਰ ਸਕਦੇ ਹਨ। Super admins, ਇਸ ਗੱਲ 'ਤੇ ਨਿਰਭਰ ਕਰਦੇ ਹੋਏ ਕਿ ਉਹ ਕਿਹੜਾ site ਦੇਖ ਰਹੇ ਹਨ, network-wide ਅਤੇ site-specific dono translations ਤੱਕ ਪਹੁੰਚ ਕਰ ਸਕਦੇ ਹਨ।

## ਕੀ ਇਹ plugins ਅਤੇ themes ਦੋਵਾਂ ਨਾਲ ਕੰਮ ਕਰਦਾ ਹੈ? {#does-this-work-with-both-plugins-and-themes}
