# Mailchimp

Multisite Ultimate: MailChimp Integration

ਆਪਣੇ Multisite Ultimate ਨੈੱਟਵਰਕ ਨੂੰ ਆਪਣੇ Mailchimp ਨਾਲ ਜੋੜੋ। ਤੁਸੀਂ ਕੁਝ ਪਲਾਨਾਂ ਨੂੰ MailChimp Lists ਨਾਲ ਲਿੰਕ ਕਰ ਸਕਦੇ ਹੋ ਅਤੇ ਸਾਈਨਅੱਪ (signup) 'ਤੇ ਆਟੋਮੈਟਿਕ ਤੌਰ 'ਤੇ ਸਬਸਕ੍ਰਾਈਬਰਾਂ ਨੂੰ ਉਨ੍ਹਾਂ ਲਿਸਟਾਂ ਵਿੱਚ ਜੋੜ ਸਕਦੇ ਹੋ।

## Installation {#installation}

1. 'ultimate-multisite-mailchimp' ਨੂੰ '/wp-content/plugins/' ਡਾਇਰੈਕਟਰੀ ਵਿੱਚ ਅਪਲੋਡ ਕਰੋ
2. ਆਪਣੇ WordPress Network Admin ਵਿੱਚ 'Plugins' ਮੀਨੂ ਰਾਹੀਂ ਪਲੱਗਇਨ ਐਕਟੀਵੇਟ ਕਰੋ

## Usage Guide {#usage-guide}

ਸਬਸਕ੍ਰਾਈਬਰਾਂ ਦੇ ਈਮੇਲਾਂ ਨੂੰ ਆਪਣੀਆਂ MailChimp ਲਿਸਟਾਂ ਨਾਲ ਸਿੰਕ ਕਰਨਾ ਸਾਡੇ Ultimate Multisite: MailChimp Integration Add-on ਨਾਲ ਬਹੁਤ ਆਸਾਨ ਹੈ। ਇਹ ਟਿਊਟোরিয়াল ਤੁਹਾਡੀ ਸਾਰੀ ਸੈਟਅਪ ਕਰਨ ਵਿੱਚ ਮਦਦ ਕਰਨ ਲਈ ਤਿਆਰ ਕੀਤਾ ਗਿਆ ਹੈ!

## Getting an API key {#getting-an-api-key}

ਇਸ ਤੋਂ ਪਹਿਲਾਂ ਕਿ ਅਸੀਂ ਵੱਖ-ਵੱਖ ਉਪਲਬਧ ਸੈਟਿੰਗਾਂ ਵਿੱਚ ਦਾਖਲ ਹੋਈਏ, ਸਾਨੂੰ ਇੱਕ MailChimp API key ਦੀ ਲੋੜ ਹੈ। ਇਹ Ultimate Multisite ਨੂੰ MailChimp API ਨਾਲ ਗੱਲ ਕਰਨ ਦੇਵੇਗਾ ਤਾਂ ਜੋ ਉਹ ਲਿਸਟਾਂ, ਗਰੁੱਪਾਂ ਨੂੰ ਪ੍ਰਾਪਤ ਕਰ ਸਕੇ ਅਤੇ ਉਪਭੋਗਤਾ ਸਾਈਨਅੱਪ ਕਰਨ 'ਤੇ ਈਮੇਲ ਪਤੇ ਉਨ੍ਹਾਂ ਲਿਸਟਾਂ ਅਤੇ ਗਰੁੱਪਾਂ ਵਿੱਚ ਜੋੜ ਸਕੇ।

ਆਪਣੀ MailChimp API key ਲੈਣ ਲਈ ਇਸ ਟਿਊਟোরিয়াল 'ਤੇ ਪਾਲਣਾ ਕਰੋ: [About API Keys | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

<!-- Screenshot unavailable: MailChimp API key settings in Ultimate Multisite Add-on Settings -->

_ਆਪਣੀ MailChimp API key ਨੂੰ_ _**Ultimate Multisite Settings - > Add-on Settings -> MailChimp**_ _ਤੇ ਜੋੜੋ_

## Integration Mode {#integration-mode}

MailChimp Integration add-on ਦੋ "integration modes" (ਇਨਟੀਗ੍ਰੇਸ਼ਨ ਮੋਡ) ਪੇਸ਼ ਕਰਦਾ ਹੈ, ਜਿਸਦਾ ਮਤਲਬ ਹੈ ਕਿ ਈਮੇਲ ਪਤੇ ਸਿੰਕ ਕਰਨ ਲਈ ਤੁਸੀਂ ਇਸ add-on ਦੀ ਵਰਤੋਂ ਕਰਨ ਦੇ ਦੋ ਵੱਖ-ਵੱਖ ਤਰੀਕੇ ਹਨ।

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_ਜ਼ਿਆਦਾ ਲਚਕਤਾ ਲਈ ਦੋ ਇਨਟੀਗ੍ਰੇਸ਼ਨ ਮੋਡ!_

### Mode 1: Multiple Lists {#mode-1-multiple-lists}

ਡਿਫਾਲਟ ਮੋਡ **Multiple Lists** ਦੀ ਵਰਤੋਂ ਕਰਦਾ ਹੈ। ਇਹ ਮੋਡ ਤੁਹਾਨੂੰ ਹਰੇਕ ਪਲਾਨ ਲਈ ਕਈ MailChimp ਲਿਸਟਾਂ ਚੁਣਨ ਦੇਵੇਗਾ। ਜੇ ਤੁਸੀਂ ਆਪਣੇ ਉਪਭੋਗਤਾਵਾਂ ਨੂੰ ਵੱਖ-ਵੱਖ ਲਿਸਟਾਂ ਵਿੱਚ ਵੰਡਦੇ ਹੋ (segment) ਤਾਂ ਇਹ ਸਭ ਤੋਂ ਵਧੀਆ ਵਿਕਲਪ ਹੈ।

ਉਦਾਹਰਨ ਲਈ, ਮੰਨ ਲਓ ਕਿ ਤੁਹਾਡੇ MailChimp account 'ਤੇ 3 ਵੱਖ-ਵੱਖ ਲਿਸਟਾਂ ਹਨ: **Newsletter** , **Plan A** , ਅਤੇ **Plan B**। ਤੁਸੀਂ ਚਾਹੁੰਦੇ ਹੋ ਕਿ ਸਾਰੇ ਉਪਭੋਗਤਾ, ਭਾਵੇਂ ਉਹ ਕਿਸੇ ਵੀ ਪਲਾਨ ਦੇ ਹੋਣ, Newsletter ਲਿਸਟ ਵਿੱਚ ਵੀ ਸ਼ਾਮਲ ਹੋਣ, ਤਾਂ ਜੋ ਉਹ ਤੁਹਾਡੇ ਅਪਡੇਟਸ ਤੋਂ ਨਾ ਗਵਾਉਣ।

ਇਹ ਪ੍ਰਾਪਤ ਕਰਨ ਲਈ, ਤੁਸੀਂ Multiple Lists ਮੋਡ ਦੀ ਵਰਤੋਂ ਕਰਕੇ Plan A MailChimp Settings ਵਿੱਚ **Newsletter** ਅਤੇ **Plan A** ਦੋਵੇਂ ਲਿਸਟਾਂ ਜੋੜ ਸਕਦੇ ਹੋ, ਅਤੇ Plan B MailChimp Settings ਵਿੱਚ **Newsletter** ਅਤੇ **Plan B** ਜੋੜ ਸਕਦੇ ਹੋ।

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_ਨਵੇਂ ਖਾਤਿਆਂ ਅਤੇ ਰੱਦ ਕੀਤੇ ਖਾਤਿਆਂ ਲਈ ਡਿਫਾਲਟ ਲਿਸਟਾਂ ਚੁਣੋ_

### Mode 2: Single List + Multiple Groups {#mode-2-single-list--multiple-groups}

MailChimp ਤੁਹਾਨੂੰ ਆਪਣੀ ਲਿਸਟ ਨੂੰ **Groups** ਨਾਮ ਦੇ ਉਪਭੋਗਤਾਵਾਂ ਦੇ ਛੋਟੇ ਗਰੁੱਪਾਂ ਵਿੱਚ ਵੰਡਣ (segment) ਦੀ ਇਜਾਜ਼ਤ ਦਿੰਦਾ ਹੈ। ਇਹ ਤੁਹਾਨੂੰ ਜ਼ਿਆਦਾ ਕੰਟਰੋਲ ਦਿੰਦਾ ਹੈ ਕਿਉਂਕਿ ਤੁਸੀਂ MailChimp 'ਤੇ ਇੱਕ ਖਾਸ ਸਬਸਕ੍ਰਾਈਬਰਾਂ ਦੇ ਹਿੱਸੇ (subset) ਨੂੰ ਕੈਂਪੇਨ ਭੇਜ ਸਕਦੇ ਹੋ, ਬਿਨਾਂ ਇਹ ਕਹਿਣ ਦੀ ਲੋੜ ਪੈਣ ਕਿ ਤੁਸੀਂ ਉਸੇ ਕੈਂਪੇਨ ਦੀ ਕਈ ਕਾਪੀਆਂ ਬਣਾਓ (ਜੋ ਬਾਅਦ ਵਿੱਚ ਉਸ ਕੈਂਪੇਨ ਦੇ ਮੈਟ੍ਰਿਕਸ ਦਾ ਵਿਸ਼ਲੇਸ਼ਣ ਕਰਨਾ ਮੁਸ਼ਕਲ ਬਣਾਉਂਦਾ ਹੈ)।

ਉਦਾਹਰਨ ਲਈ, ਤੁਹਾਡੇ ਕੋਲ **My Network Subscribers** ਨਾਮ ਦੀ ਇੱਕ ਸਿੰਗਲ ਲਿਸਟ ਹੋ ਸਕਦੀ ਹੈ, ਅਤੇ ਇਸ ਦੇ ਅੰਦਰ **Plan A** ਅਤੇ **Plan B** ਲਈ ਗਰੁੱਪ ਹੋ ਸਕਦੇ ਹਨ।

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_ਡਿਫਾਲਟ ਲਿਸਟਾਂ ਅਤੇ ਗਰੁੱਪਾਂ ਨੂੰ ਚੁਣੋ, ਜੇਕਰ ਤੁਸੀਂ “single list + multiple groups” ਇਨਟੀਗ੍ਰੇਸ਼ਨ ਮੋਡ ਦੀ ਵਰਤੋਂ ਕਰ ਰਹੇ ਹੋ_

## Default Lists and Groups {#default-lists-and-groups}

ਇੱਕ ਵਾਰ ਜਦੋਂ ਤੁਸੀਂ ਇਹ ਚੁਣ ਲੈਂਦੇ ਹੋ ਕਿ ਤੁਹਾਡੇ ਖਾਸ ਵਰਤੋਂ ਦੇ ਕੇਸ ਲਈ ਕਿਹੜਾ ਇਨਟੀਗ੍ਰੇਸ਼ਨ ਮੋਡ ਵਧੇਰੇ ਸਹੀ ਹੈ, ਤਾਂ ਇਹ ਡਿਫਾਲਟ ਲਿਸਟਾਂ (ਅਤੇ ਮੋਡ ਦੇ ਆਧਾਰ 'ਤੇ ਗਰੁੱਪਾਂ) ਨੂੰ ਚੁਣਨ ਦਾ ਸਮਾਂ ਹੈ। ਡਿਫਾਲਟ ਲਿਸਟ ਦੀ ਵਰਤੋਂ ਉਦੋਂ ਕੀਤੀ ਜਾਵੇਗੀ ਜਦੋਂ ਕਿਸੇ ਖਾਸ ਪਲਾਨ ਲਈ ਕੋਈ ਲਿਸਟ ਨਹੀਂ ਚੁਣੀ ਗਈ ਹੋਵੇ।

ਤੁਸੀਂ ਰੱਦ ਕੀਤੇ ਖਾਤਿਆਂ ਲਈ ਵੀ ਡਿਫਾਲਟ ਲਿਸਟ ਚੁਣ ਸਕਦੇ ਹੋ। ਜਦੋਂ ਵੀ ਕੋਈ ਉਪਭੋਗਤਾ ਆਪਣੀ ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਖਤਮ ਕਰਦਾ ਹੈ ਅਤੇ ਆਪਣੇ ਖਾਤੇ ਨੂੰ ਤੁਹਾਡੇ ਨੈੱਟਵਰਕ ਤੋਂ ਹਟਾਉਂਦਾ ਹੈ, ਤਾਂ ਉਸਦਾ ਈਮੇਲ ਪਤਾ ਰੱਦ ਕੀਤੇ ਖਾਤਿਆਂ ਲਈ ਸੈੱਟ ਕੀਤੇ ਲਿਸਟ ਵਿੱਚ ਤਬਦੀਲ ਹੋ ਜਾਵੇਗਾ।

## Enabling Double Subscription {#enabling-double-subscription}

ਉਸੇ ਸੈਟਿੰਗ ਪੇਜ 'ਤੇ, ਤੁਸੀਂ **Double Subscription** ਨੂੰ ਐਨੇਬਲ ਕਰ ਸਕੋਗੇ। ਇਹ ਵਿਕਲਪ ਉਪਭੋਗਤਾ ਨੂੰ ਡਿਫਾਲਟ ਚੁਣੀ ਗਈ ਲਿਸਟ ਅਤੇ ਉਸ ਉਪਭੋਗਤਾ ਦੇ ਪਲਾਨ ਲਈ ਚੁਣੀ ਗਈ ਲਿਸਟ ਦੋਵਾਂ ਵਿੱਚ ਜੋੜ ਦੇਵੇਗਾ। ਜੇ ਇਹ ਵਿਕਲਪ ਐਨੇਬਲ ਨਹੀਂ ਕੀਤਾ ਗਿਆ ਹੈ, ਤਾਂ ਉਪਭੋਗਤਾ ਨੂੰ ਸਿਰਫ਼ ਉਸ ਉਪਭੋਗਤਾ ਦੇ ਪਲਾਨ ਲਈ ਚੁਣੀ ਗਈ ਲਿਸਟ ਵਿੱਚ ਜੋੜਿਆ ਜਾਵੇਗਾ। ਜੇਕਰ ਉਸ ਖਾਸ ਪਲਾਨ ਲਈ ਕੁਝ ਵੀ ਸੈੱਟ ਨਹੀਂ ਕੀਤਾ ਗਿਆ ਹੈ, ਤਾਂ ਉਪਭੋਗਤਾ ਨੂੰ ਚੁਣੀ ਗਈ ਡਿਫਾਲਟ ਲਿਸਟ ਵਿੱਚ ਜੋੜਿਆ ਜਾਵੇਗਾ।

_ਸਾਈਨ-ਅੱਪ 'ਤੇ ਤੁਹਾਡੇ ਉਪਭੋਗਤਾਵਾਂ ਨੂੰ ਡਿਫਾਲਟ ਲਿਸਟ ਅਤੇ ਪਲਾਨ ਦੀ ਲਿਸਟ ਦੋਵਾਂ ਵਿੱਚ ਸਬਸਕ੍ਰਾਈਬ ਕਰਵਾ ਸਕਦੇ ਹੋ_

## Selecting Lists and Groups for each of the Plans {#selecting-lists-and-groups-for-each-of-the-plans}

ਆਪਣੇ **Plans** 'ਤੇ ਜਾਓ ਅਤੇ ਉਸ ਪਲਾਨ ਨੂੰ ਚੁਣੋ ਜਿਸਨੂੰ ਤੁਸੀਂ ਐਡਿਟ ਕਰਨਾ ਚਾਹੁੰਦੇ ਹੋ। ਨੋਟ ਕਰੋ ਕਿ **Advanced Options** meta-box 'ਤੇ, MailChimp ਨਾਮ ਦਾ ਇੱਕ ਨਵਾਂ ਟੈਬ ਉਪਲਬਧ ਹੋਵੇਗਾ।

ਉਸ ਟੈਬ 'ਤੇ, ਤੁਸੀਂ ਉਸ ਪਲਾਨ ਲਈ ਕਈ ਲਿਸਟਾਂ ਜਾਂ ਇੱਕ ਸਿੰਗਲ ਲਿਸਟ ਅਤੇ ਕਈ ਗਰੁੱਪ (ਇਨਟੀਗ੍ਰੇਸ਼ਨ ਮੋਡ ਦੇ ਆਧਾਰ 'ਤੇ) ਚੁਣ ਸਕਦੇ ਹੋ।

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_ਆਪਣੇ ਹਰੇਕ ਪਲਾਨ ਲਈ ਵਰਤਣ ਵਾਲੀਆਂ ਲਿਸਟਾਂ ਅਤੇ ਗਰੁੱਪਾਂ ਨੂੰ ਚੁਣੋ_

ਜਦੋਂ ਕੋਈ ਨਵਾਂ ਉਪਭੋਗਤਾ ਉਸ ਪਲਾਨ ਲਈ ਸਾਈਨ-ਅੱਪ ਕਰਦਾ ਹੈ, ਤਾਂ ਉਸਦਾ ਈਮੇਲ ਪਤਾ ਚੁਣੀ ਗਈ ਲਿਸਟਾਂ ਅਤੇ ਗਰੁੱਪਾਂ ਵਿੱਚ ਜੋੜ ਦਿੱਤਾ ਜਾਵੇਗਾ!
