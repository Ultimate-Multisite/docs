# ਬਦਲਾਅ ਲਾਗ

## 1.18.0 — 2026-06-29 ਨੂੰ ਜਾਰੀ ਕੀਤਾ ਗਿਆ {#1180--released-on-2026-06-29}

### ਨਵਾਂ {#new}

- **Google Calendar ਟੂਲ** — ਸਮਾਂ-ਸੂਚੀ-ਜਾਗਰੂਕ ਆਟੋਮੇਸ਼ਨਾਂ ਲਈ ਸੰਰਚਿਤ ਕੈਲੰਡਰ ਅਤੇ ਇਵੈਂਟ ਪੜ੍ਹੋ
- **ਸੰਪਰਕ ਮੈਪਿੰਗ ਅਤੇ ਹਾਜ਼ਰ ਸਹਾਇਕ** — ਕੈਲੰਡਰ ਭਾਗੀਦਾਰਾਂ ਨੂੰ ਸਾਈਟ ਵਰਤੋਂਕਾਰਾਂ ਅਤੇ ਸੰਪਰਕਾਂ ਨਾਲ ਮਿਲਾਓ
- **ਮਨੁੱਖੀ ਮਨਜ਼ੂਰੀ ਗੇਟ ਅਤੇ ਯਾਦ-ਦਿਹਾਨੀ ਰਿਕਾਰਡ** — ਸਮੀਖਿਆ ਲਈ ਆਟੋਮੇਸ਼ਨਾਂ ਨੂੰ ਰੋਕੋ ਅਤੇ ਦੁਹਰਾਈਆਂ ਸੂਚਨਾਵਾਂ ਤੋਂ ਬਚੋ
- **TextBee SMS ਪ੍ਰਦਾਤਾ** — TextBee ਰਾਹੀਂ ਸੰਰਚਿਤ ਟੈਕਸਟ-ਸੁਨੇਹਾ ਸੂਚਨਾਵਾਂ ਭੇਜੋ
- **ਉੱਨਤ ਸਾਥੀ ਪੈਕੇਜ** — ਭਰੋਸੇਯੋਗ-ਵਿਕਾਸਕਾਰ ਫਾਈਲਸਿਸਟਮ, ਡਾਟਾਬੇਸ, WP-CLI, REST ਡਿਸਪੈਚਰ, ਪਲੱਗਇਨ ਬਿਲਡਰ, git ਸਨੈਪਸ਼ਾਟ, ਵਰਤੋਂਕਾਰ-ਪ੍ਰਬੰਧਨ, ਅਤੇ ਬੈਂਚਮਾਰਕ ਟੂਲ ਸ਼ਾਮਲ ਕਰੋ ਜੋ WordPress.org ਬਿਲਡ ਤੋਂ ਵੱਖਰੇ ਤੌਰ ਤੇ ਵੰਡੇ ਜਾਂਦੇ ਹਨ

### ਸੁਧਾਰਿਆ ਗਿਆ {#improved}

- **ਪ੍ਰਬੰਧਿਤ Superdav ਸੇਵਾ ਸੈਟਅੱਪ** — ਸਮਰਥਿਤ ਸਾਈਟਾਂ ਲਈ ਹੋਸਟ ਕੀਤੇ ਸੇਵਾ ਅੰਤ-ਬਿੰਦੂ ਅਤੇ ਆਟੋਮੈਟਿਕ ਕਨੈਕਸ਼ਨ ਪ੍ਰੋਵਿਜ਼ਨਿੰਗ ਸ਼ਾਮਲ ਕਰੋ
- **ਰਿਲੀਜ਼ ਪੈਕੇਜਿੰਗ** — ਵੱਖਰੇ ਮੁੱਖ ਅਤੇ ਉੱਨਤ ZIP ਬਣਾਓ, ਦੋਵੇਂ GitHub ਤੇ ਪ੍ਰਕਾਸ਼ਿਤ ਕਰੋ, ਅਤੇ WordPress.org ਨੂੰ ਸਿਰਫ਼ ਮੁੱਖ ਪੈਕੇਜ ਭੇਜੋ

### ਠੀਕ ਕੀਤਾ ਗਿਆ {#fixed}

- **AI ਬੇਨਤੀ ਭਰੋਸੇਯੋਗਤਾ** — ਮਾਡਲ ਚੋਣ, ਬੇਨਤੀ ਟਾਈਮਆਉਟ, ਡਿਫਾਲਟ ਸੈਟਅੱਪ ਮਾਡਲ, ਤਰਕ-ਟੈਕਸਟ ਸੰਭਾਲ, ਅਤੇ ਅਵੈਧ ਟੂਲ ਕਾਲਾਂ ਲਈ ਮੁੜ-ਕੋਸ਼ਿਸ਼ ਮਾਰਗਦਰਸ਼ਨ ਸੁਧਾਰੋ
- **ਕੈਲੰਡਰ ਅਤੇ ਯਾਦ-ਦਿਹਾਨੀ ਮਜ਼ਬੂਤੀ** — Google Calendar ਟੋਕਨਾਂ ਅਤੇ ਯਾਦ-ਦਿਹਾਨੀ ਡੀਡੁਪਲੀਕੇਸ਼ਨ ਨੂੰ ਮਜ਼ਬੂਤ ਕਰੋ
- **ਆਨਬੋਰਡਿੰਗ ਅਤੇ ਮਨਜ਼ੂਰੀ ਮੁੜ-ਸ਼ੁਰੂਆਤਾਂ** — ਫਰੰਟਐਂਡ ਆਨਬੋਰਡਿੰਗ ਲਾਂਚ ਅਤੇ ਪੁਸ਼ਟੀ ਕੀਤੀ ਸਮਰੱਥਾ ਮੁੜ-ਸ਼ੁਰੂਆਤਾਂ ਨੂੰ ਠੀਕ ਕਰੋ
- **WordPress.org ਪੈਕੇਜਿੰਗ ਸਮੀਖਿਆ ਮੁੱਦੇ** — ਮੁੱਖ ਰਿਲੀਜ਼ ਲਈ ਪੈਕੇਜਿੰਗ ਸਮੀਖਿਆ ਫੀਡਬੈਕ ਨੂੰ ਹੱਲ ਕਰੋ

## 1.16.0 — 2026-05-20 ਨੂੰ ਜਾਰੀ ਕੀਤਾ ਗਿਆ {#1160--released-on-2026-05-20}

### ਨਵਾਂ {#new-1}

- **Logo SVG ਬਣਾਉਣ ਦੀ ਸਮਰੱਥਾ** — Theme Builder ਹੁਣ namespace-ਸੁਰੱਖਿਅਤ ਸੈਨਿਟਾਈਜ਼ੇਸ਼ਨ ਨਾਲ ਕਸਟਮ ਲੋਗੋ SVG ਤਿਆਰ ਅਤੇ ਐਂਬੈਡ ਕਰ ਸਕਦਾ ਹੈ
- **ਖੋਜ ਇੰਟਰਵਿਊ ਵਿੱਚ ਫੋਟੋ ਅੱਪਲੋਡ** — Theme Builder ਖੋਜ ਇੰਟਰਵਿਊ ਵਿੱਚ ਹੁਣ ਹੋਰ ਧਨਾਢ ਡਿਜ਼ਾਈਨ ਸੰਦਰਭ ਲਈ ਫੋਟੋ ਅੱਪਲੋਡ ਕਦਮ ਸ਼ਾਮਲ ਹੈ
- **ਪੈਲੇਟ ਕੰਟਰਾਸਟ ਪ੍ਰਮਾਣਿਤ ਕਰਨ ਦੀ ਸਮਰੱਥਾ** — ਕਿਸੇ ਥੀਮ ਤੇ ਲਾਗੂ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ ਰੰਗ ਜੋੜਿਆਂ ਦੀ WCAG ਅਨੁਕੂਲਤਾ ਲਈ ਜਾਂਚ ਕਰੋ
- **ਹਾਸਪਿਟੈਲਿਟੀ ਮੇਨੂ** — Theme Builder ਹੁਣ ਹਾਸਪਿਟੈਲਿਟੀ ਕਾਰੋਬਾਰਾਂ ਲਈ ਸੰਰਚਿਤ ਭੋਜਨ ਅਤੇ ਪੇਅ ਮੇਨੂ ਪੰਨੇ ਤਿਆਰ ਕਰ ਸਕਦਾ ਹੈ
- **ਡੈਸਕਟਾਪ ਅਤੇ ਮੋਬਾਈਲ ਪ੍ਰੀਵਿਊ ਰੈਂਡਰਿੰਗ** — ਡਿਜ਼ਾਈਨ-ਦਿਸ਼ਾ ਚੋਣ ਦੌਰਾਨ ਡੈਸਕਟਾਪ ਅਤੇ ਮੋਬਾਈਲ ਡਿਵਾਈਸਾਂ ਤੇ ਆਪਣੇ ਡਿਜ਼ਾਈਨ ਦਾ ਪ੍ਰੀਵਿਊ ਕਰੋ
- **ਨੈਵੀਗੇਸ਼ਨ ਲੇਬਲ ਪੈਰਾਮੀਟਰ** — Create Menu ਸਮਰੱਥਾ ਹੁਣ ਪੰਨੇ ਦੇ ਸਿਰਲੇਖ ਤੋਂ ਵੱਖਰੇ `navigation_label` ਦਾ ਸਮਰਥਨ ਕਰਦੀ ਹੈ
- **Tier 1 ਟੂਲ ਉਪਲਬਧਤਾ** — sd-ai-agent/site-scrape ਹੁਣ Theme Builder ਵਿੱਚ ਡਿਫਾਲਟ ਤੌਰ ਤੇ ਉਪਲਬਧ Tier 1 ਟੂਲ ਹੈ

### ਠੀਕ ਕੀਤਾ ਗਿਆ {#fixed-1}

- **AI Client ਕੈਸ਼** — ਹੁਣ ਕ੍ਰਾਸ-ਬੇਨਤੀ ਸਥਿਰਤਾ ਲਈ ਟ੍ਰਾਂਜ਼ਿਅੰਟਾਂ ਦੁਆਰਾ ਸਮਰਥਿਤ ਹੈ, ਜਿਸ ਨਾਲ ਲੰਬੇ ਸਮੇਂ ਚੱਲਣ ਵਾਲੇ ਏਜੰਟ ਕੰਮਾਂ ਤੇ ਡਾਟਾ ਨੁਕਸਾਨ ਰੋਕਿਆ ਜਾਂਦਾ ਹੈ
- **ਪਲੱਗਇਨ ਕਤਾਰ ਕਾਰਵਾਈ ਲਿੰਕ** — ਸਪਸ਼ਟਤਾ ਲਈ ਠੀਕ ਕੀਤੇ ਅਤੇ ਨਾਮ ਬਦਲੇ ਗਏ

## 1.10.0 — 2026-05-05 ਨੂੰ ਜਾਰੀ ਕੀਤਾ ਗਿਆ {#1100--released-on-2026-05-05}

- **ਪ੍ਰਦਾਤਾ ਸੂਚੀ ਆਟੋਮੈਟਿਕ ਤੌਰ ਤੇ ਤਾਜ਼ਾ ਹੁੰਦੀ ਹੈ** — ਜਦੋਂ ਕੋਈ ਵੀ ਪਲੱਗਇਨ ਸਰਗਰਮ ਜਾਂ ਗੈਰ-ਸਰਗਰਮ ਕੀਤਾ ਜਾਂਦਾ ਹੈ

- **ਇਵੈਂਟ ਆਟੋਮੇਸ਼ਨ** — ਉਹਨਾਂ ਸਾਈਟਾਂ ਨੂੰ ਸੁਚੱਜੇ ਤਰੀਕੇ ਨਾਲ ਸੰਭਾਲਦਾ ਹੈ ਜਿੱਥੇ ਆਟੋਮੇਸ਼ਨ ਟੇਬਲ ਹਾਲੇ ਨਹੀਂ ਬਣੀਆਂ
- **memory-save ਸਮਰੱਥਾ** — ਹੁਣ ਸਿਸਟਮ ਨਿਰਦੇਸ਼ ਬਿਲਡਰ ਵਿੱਚ ਠੀਕ namespace ਪ੍ਰੀਫਿਕਸ ਵਰਤਦੀ ਹੈ
- **ਸਕੇਲਰ ਟੂਲ ਨਤੀਜੇ** — ਹੁਣ AI ਨੂੰ ਵਾਪਸ ਭੇਜਣ ਤੋਂ ਪਹਿਲਾਂ ਠੀਕ ਤਰ੍ਹਾਂ ਰੈਪ ਕੀਤੇ ਜਾਂਦੇ ਹਨ
- **ਵਰਤੋਂ ਅੰਕੜੇ** — ਹੁਣ ਪੁਰਾਣੇ ਸੰਸਕਰਣਾਂ ਤੋਂ ਅੱਪਗ੍ਰੇਡ ਤੇ ਲੈਗੇਸੀ ਸਮਰੱਥਾ ਕੁੰਜੀ ਫਾਰਮੈਟ ਨੂੰ ਠੀਕ ਤਰ੍ਹਾਂ ਸੰਭਾਲਦੇ ਹਨ
