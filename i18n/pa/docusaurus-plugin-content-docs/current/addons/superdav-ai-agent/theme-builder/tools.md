# ਸਾਧਨ ਅਤੇ ਸੈਟਿੰਗਾਂ

Theme Builder ਤੁਹਾਡੀ WordPress ਸਾਈਟ ਦਾ ਵਿਸ਼ਲੇਸ਼ਣ ਕਰਨ, ਡਿਜ਼ਾਈਨ ਕਰਨ ਅਤੇ ਬਣਾਉਣ ਲਈ ਸਾਧਨਾਂ (tools) ਦਾ ਇੱਕ ਸੈੱਟ ਵਰਤਦਾ ਹੈ। ਵਰਜ਼ਨ 1.16.0 ਵਿੱਚ, **sd-ai-agent/site-scrape ਹੁਣ ਇੱਕ Tier 1 tool ਹੈ**, ਜਿਸ ਕਾਰਨ ਇਹ Theme Builder ਵਿੱਚ ਡਿਫਾਲਟ ਤੌਰ 'ਤੇ ਉਪਲਬਧ ਹੈ।

## ਉਪਲਬਧ ਸਾਧਨ (Available Tools)

### Tier 1 ਸਾਧਨ (ਹਮੇਸ਼ਾ ਉਪਲਬਧ)

Tier 1 ਸਾਧਨ Theme Builder ਵਿੱਚ ਕੋਈ ਵਾਧੂ ਸੈਟਿੰਗ ਕੀਤੇ ਬਿਨਾਂ ਹੀ ਡਿਫਾਲਟ ਤੌਰ 'ਤੇ ਉਪਲਬਧ ਹੁੰਦੇ ਹਨ।

#### sd-ai-agent/site-scrape

**ਮਕਸਦ:** ਡਿਜ਼ਾਈਨਰ ਇਨਸਪੀਰੇਸ਼ਨ, ਕੰਟੈਂਟ ਸਟਰਕਚਰ, ਅਤੇ ਕਾਰਜਕੁਸ਼ਲਤਾ (functionality) ਨੂੰ ਕੱਢਣ ਲਈ ਮੌਜੂਦਾ ਵੈੱਬਸਾਈਟਾਂ ਦਾ ਵਿਸ਼ਲੇਸ਼ਣ ਕਰਨਾ।

**ਸਮਰੱਥਾਵਾਂ (Capabilities):**

- **ਵੈੱਬਸਾਈਟ ਵਿਸ਼ਲੇਸ਼ਣ** — ਮੁਕਾਬਲੇਬਾਜ਼ ਜਾਂ ਇਨਸਪੀਰੇਸ਼ਨ ਵਾਲੀਆਂ ਵੈੱਬਸਾਈਟਾਂ ਨੂੰ scrape ਕਰਨਾ ਅਤੇ ਵਿਸ਼ਲੇਸ਼ਣ ਕਰਨਾ
- **ਡਿਜ਼ਾਈਨ ਕੱਢਣਾ** — ਰੰਗ, ਫੌਂਟ, ਅਤੇ ਲੇਆਉਟ ਪੈਟਰਨਾਂ ਦੀ ਪਛਾਣ ਕਰਨਾ
- **ਕੰਟੈਂਟ ਸਟਰਕਚਰ** — ਪੇਜ ਦੇ ਆਰਗੈਨੀਜ਼ੇਸ਼ਨ ਅਤੇ ਹਾਇਰਾਰਕੀ ਨੂੰ ਸਮਝਣਾ
- **ఫీਚਰ ਪਛਾਣ** — ਪਲੱਗਇਨ ਅਤੇ ਕਾਰਜਕੁਸ਼ਲਤਾ ਦੀ ਪਛਾਣ ਕਰਨਾ
- **ਪਰਫਾਰਮੈਂਸ ਵਿਸ਼ਲੇਸ਼ਣ** — ਪੇਜ ਦੀ ਗਤੀ (speed) ਅਤੇ ਆਪਟੀਮਾਈਜ਼ੇਸ਼ਨ ਦੀ ਜਾਂਚ ਕਰਨਾ
- **SEO ਵਿਸ਼ਲੇਸ਼ਣ** — ਮੀਟਾ ਟੈਗਾਂ ਅਤੇ ਸਟਰਕਚਰਡ ਡੇਟਾ ਦੀ ਸਮੀਖਿਆ ਕਰਨਾ

**ਵਰਤੋਂ (Usage):**

```
Analyze the design of example.com and use it as inspiration for my site.
```

**ਇਹ ਕੀ ਕੱਢਦਾ ਹੈ (What It Extracts):**

- ਰੰਗ ਪੈਲੇਟ ਅਤੇ ਰੰਗ ਦੀ ਵਰਤੋਂ
- ਟਾਈਪੋਗ੍ਰਾਫੀ (ਫੌਂਟ ਅਤੇ ਆਕਾਰ)
- ਲੇਆਉਟ ਸਟਰਕਚਰ ਅਤੇ ਸਪੇਸਿੰਗ
- ਨੈਵੀਗੇਸ਼ਨ ਪੈਟਰਨ
- ਕੰਟੈਂਟ ਦਾ ਆਰਗੈਨੀਜ਼ੇਸ਼ਨ
- ਤਸਵੀਰਾਂ ਅਤੇ ਮੀਡੀਆ ਦੀ ਵਰਤੋਂ
- ਇੰਟਰਐਕਟਿਵ ਤੱਤ
- ਮੋਬਾਈਲ ਰਿਸਪਾਂਸਿਵਨੈਸ

**ਸੀਮਾਵਾਂ (Limitations):**

- ਪਾਸਵਰਡ ਨਾਲ ਸੁਰੱਖਿਅਤ ਸਾਈਟਾਂ ਨੂੰ scrape ਨਹੀਂ ਕਰ ਸਕਦਾ
- robots.txt ਅਤੇ ਸਾਈਟ ਨੀਤੀਆਂ ਦਾ ਸਤਿਕਾਰ ਕਰਦਾ ਹੈ
- ਡਾਇਨਾਮਿਕ ਕੰਟੈਂਟ ਨੂੰ ਕੈਪਚਰ ਨਹੀਂ ਕਰ ਸਕਦਾ
- JavaScript-ਭਾਰੀ ਸਾਈਟਾਂ ਦਾ ਵਿਸ਼ਲੇਸ਼ਣ ਸੀਮਤ ਹੋ ਸਕਦਾ ਹੈ
- ਵੱਡੀਆਂ ਸਾਈਟਾਂ ਦਾ ਵਿਸ਼ਲੇਸ਼ਣ ਕਰਨ ਵਿੱਚ ਜ਼ਿਆਦਾ ਸਮਾਂ ਲੱਗ ਸਕਦਾ ਹੈ

### Tier 2 ਸਾਧਨ (ਵਿਕਲਪਿਕ)

Tier 2 ਸਾਧਨ Theme Builder ਸੈਟਿੰਗਾਂ ਵਿੱਚ ਖੁੱਲ੍ਹ ਕੇ (explicitly) ਸੈੱਟ ਕੀਤੇ ਜਾਣ 'ਤੇ ਉਪਲਬਧ ਹੁੰਦੇ ਹਨ।

#### Advanced Analytics

ਵਿਸਤ੍ਰਿਤ ਪ੍ਰਦਰਸ਼ਨ ਮਾਪਦੰਡ (detailed performance metrics) ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ:

- ਪੇਜ ਲੋਡ ਦਾ ਸਮਾਂ
- Core Web Vitals
- SEO ਸਕੋਰ
- Accessibility ਸਕੋਰ
- Best practices ਸਕੋਰ

#### Content Optimizer

ਇਨ੍ਹਾਂ ਲਈ ਵਿਸ਼ਲੇਸ਼ਣ ਕਰਦਾ ਹੈ ਅਤੇ ਸੁਧਾਰ ਸੁਝਾਅ ਦਿੰਦਾ ਹੈ:

- ਪੜ੍ਹਨਯੋਗਤਾ (Readability)
- SEO ਆਪਟੀਮਾਈਜ਼ੇਸ਼ਨ
- ਕੀਵਰਡ ਦੀ ਵਰਤੋਂ
- ਕੰਟੈਂਟ ਸਟਰਕਚਰ
- ਕਾਲ-ਟੂ-ਐਕਸ਼ਨ (Call-to-action) ਦੀ ਜਗ੍ਹਾ

### Tier 3 ਸਾਧਨ (ਪ੍ਰੀਮੀਅਮ)

Tier 3 ਸਾਧਨ ਲਈ ਵਾਧੂ ਸੈਟਿੰਗ ਜਾਂ ਪ੍ਰੀਮੀਅਮ ਫੀਚਰਾਂ ਦੀ ਲੋੜ ਹੁੰਦੀ ਹੈ।

#### AI Content Generator

ਇਨ੍ਹਾਂ ਲਈ ਕੰਟੈਂਟ ਤਿਆਰ ਕਰਦਾ ਹੈ:

- ਪ੍ਰੋਡਕਟ ਡਿਸਕ੍ਰਿਪਸ਼ਨ
- ਸਰਵਿਸ ਪੇਜ
- ਬਲੌਗ ਪੋਸਟ
- ਮੀਟਾ ਡਿਸਕ੍ਰਿਪਸ਼ਨ
- ਕਾਲ-ਟੂ-ਐਕਸ਼ਨ ਕਾਪੀ

#### Advanced Design Tools

ਵਧੇਰੇ ਡਿਜ਼ਾਈਨ ਸਮਰੱਥਾਵਾਂ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ:

- Custom CSS generation
- Animation creation
- ਇੰਟਰਐਕਟਿਵ ਤੱਤ ਡਿਜ਼ਾਈਨ
- Advanced color theory
- Typography optimization

## Tool ਸੈਟਿੰਗ (Tool Configuration)

### ਸਾਧਨ ਸੈੱਟ ਕਰਨਾ (Enabling Tools)

Theme Builder ਵਿੱਚ ਵਾਧੂ ਸਾਧਨ ਸੈੱਟ ਕਰਨ ਲਈ:

1. **Gratis AI Agent → Settings** 'ਤੇ ਜਾਓ
2. **Theme Builder → Tools** 'ਤੇ ਜਾਓ
3. ਲੋੜ ਅਨੁਸਾਰ ਟੂਲ ਓਨ/ਆਫ ਕਰੋ
4. ਸੈਟਿੰਗਾਂ ਸੇਵ ਕਰੋ

### Tool ਪਰਮਿਸ਼ਨ (Tool Permissions)

ਕੁਝ ਸਾਧਨਾਂ ਨੂੰ ਪਰਮਿਸ਼ਨਾਂ ਦੀ ਲੋੜ ਹੁੰਦੀ ਹੈ:

- **Site scraping** — ਇੰਟਰਨੈਟ ਐਕਸੈਸ ਦੀ ਲੋੜ
- **Analytics** — Google Analytics integration ਦੀ ਲੋੜ
- **Content generation** — API keys ਦੀ ਲੋੜ
- **Advanced features** — ਪ੍ਰੀਮੀਅਮ ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਦੀ ਲੋੜ ਹੋ ਸਕਦੀ ਹੈ

### API Keys ਅਤੇ Credentials

ਉਨ੍ਹਾਂ ਸਾਧਨਾਂ ਲਈ API keys ਕਨਫਿਗਰ ਕਰੋ ਜਿਨ੍ਹਾਂ ਦੀ ਲੋੜ ਹੈ:

1. **Gratis AI Agent → Settings → API Keys** 'ਤੇ ਜਾਓ
2. ਹਰੇਕ ਸਾਧਨ ਲਈ credentials ਭਰੋ
3. ਕਨੈਕਸ਼ਨ ਟੈਸਟ ਕਰੋ
4. ਸੁਰੱਖਿਅਤ ਢੰਗ ਨਾਲ ਸੇਵ ਕਰੋ

## sd-ai-agent/site-scrape ਦੀ ਵਰਤੋਂ ਕਰਨਾ

### ਬੁਨਿਆਦੀ ਵਰਤੋਂ (Basic Usage)

Theme Builder ਨੂੰ ਇੱਕ ਵੈੱਬਸਾਈਟ ਦਾ ਵਿਸ਼ਲੇਸ਼ਣ ਕਰਨ ਲਈ ਕਹੋ:

```
Analyze the design of my competitor's site at competitor.com
and suggest design improvements for my site.
```

### ਖਾਸ ਵਿਸ਼ਲੇਸ਼ਣ (Specific Analysis)

ਵਿਸ਼ਲੇਸ਼ਣ ਦੀਆਂ ਖਾਸ ਕਿਸਮਾਂ ਦੀ ਮੰਗ ਕਰੋ:

```
Extract the color palette from example.com and use it as inspiration.
```

```
Analyze the navigation structure of example.com and apply similar
organization to my site.
```

```
Check the mobile responsiveness of example.com and ensure my site
is equally responsive.
```

### ਤੁਲਨਾਤਮਕ ਵਿਸ਼ਲੇਸ਼ਣ (Comparison Analysis)

ਕਈ ਸਾਈਟਾਂ ਦੀ ਤੁਲਨਾ ਕਰੋ:

```
Compare the designs of site1.com and site2.com and create a design
that combines the best elements of both.
```

## Tool ਦੀਆਂ ਸੀਮਾਵਾਂ ਅਤੇ ਵਿਚਾਰ (Tool Limitations and Considerations)

### Rate Limiting

- ਸਰਵਰ 'ਤੇ ਜ਼ਿਆਦਾ ਬੋਝ ਪਾਉਣ ਤੋਂ ਰੋਕਣ ਲਈ scraping rate-limited ਹੁੰਦਾ ਹੈ
- ਇੱਕੋ ਸਾਈਟ 'ਤੇ ਕਈ ਬੇਨਤੀਆਂ (requests) ਨੂੰ ਘਟਾਇਆ ਜਾ ਸਕਦਾ ਹੈ (throttled)
- ਵੱਡੀਆਂ ਸਾਈਟਾਂ ਦਾ ਵਿਸ਼ਲੇਸ਼ਣ ਕਰਨ ਵਿੱਚ ਜ਼ਿਆਦਾ ਸਮਾਂ ਲੱਗ ਸਕਦਾ ਹੈ

### ਕੰਟੈਂਟ ਦੀ ਸਹੀਤਾ (Content Accuracy)

- ਡਾਇਨਾਮਿਕ ਕੰਟੈਂਟ ਪੂਰੀ ਤਰ੍ਹਾਂ ਕੈਪਚਰ ਨਹੀਂ ਹੋ ਸਕਦਾ
- JavaScript-render ਕੀਤੇ ਕੰਟੈਂਟ ਅਧੂਰਾ ਹੋ ਸਕਦਾ ਹੈ
- ਰੀਅਲ-ਟਾਈਮ ਡਾਟਾ ਪ੍ਰਤੀਬਿੰਬਿਤ ਨਹੀਂ ਹੋ ਸਕਦਾ
- ਕੁਝ ਕੰਟੈਂਟ ਪੇ-ਵਾਲਾਂ (paywalls) ਪਿੱਛੇ ਹੋ ਸਕਦਾ ਹੈ

### ਕਾਨੂੰਨੀ ਅਤੇ ਨੈਤਿਕ ਵਿਚਾਰ (Legal and Ethical Considerations)

- robots.txt ਅਤੇ ਸਾਈਟ ਨੀਤੀਆਂ ਦਾ ਸਤਿਕਾਰ ਕਰੋ
- ਨਕਲ ਕਰਨ ਲਈ ਕਾਪੀਰਾਈਟਡ ਕੰਟੈਂਟ scrape ਨਾ ਕਰੋ
- ਵਿਸ਼ਲੇਸ਼ਣ ਦੀ ਵਰਤੋਂ ਇਨਸਪੀਰੇਸ਼ਨ ਲਈ ਕਰੋ, ਨਕਲ ਕਰਨ ਲਈ ਨਹੀਂ
- ਇਹ ਯਕੀਨੀ ਬਣਾਓ ਕਿ ਤੁਹਾਡੇ ਕੋਲ ਕੱਢੇ ਗਏ ਕੰਟੈਂਟ ਦੀ ਵਰਤੋਂ ਕਰਨ ਦੇ ਅਧਿਕਾਰ ਹਨ
- ਵਿਸ਼ਲੇਸ਼ਣ ਕੀਤੀਆਂ ਸਾਈਟਾਂ ਦੀਆਂ ਸੇਵਾ ਦੀਆਂ ਸ਼ਰਤਾਂ (terms of service) ਦੀ ਪਾਲਣਾ ਕਰੋ

### ਪ੍ਰਦਰਸ਼ਨ ਪ੍ਰਭਾਵ (Performance Impact)

- ਵੱਡੀਆਂ ਸਾਈਟਾਂ ਨੂੰ scrape ਕਰਨ ਵਿੱਚ ਸਮਾਂ ਲੱਗ ਸਕਦਾ ਹੈ
- ਇੱਕੋ ਸਮੇਂ ਕਈ scrapes ਕਰਨ ਨਾਲ ਧੀਮਾ ਹੋ ਸਕਦਾ ਹੈ
- ਨੈਟਵਰਕ ਕਨੈਕਟੀਵਿਟੀ ਗਤੀ ਨੂੰ ਪ੍ਰਭਾਵਿਤ ਕਰਦੀ ਹੈ
- ਵੱਡੇ ਡਾਟਾ ਸੈੱਟਾਂ ਨੂੰ ਹੋਰ ਪ੍ਰੋਸੈਸਿੰਗ ਦੀ ਲੋੜ ਹੋ ਸਕਦੀ ਹੈ

## ਵਧੀਆ ਅਭਿਆਸ (Best Practices)

### ਸਾਈਟ ਵਿਸ਼ਲੇਸ਼ਣ ਦੀ ਵਰਤੋਂ ਕਰਨਾ

1. **ਕਈ ਸਾਈਟਾਂ ਦਾ ਵਿਸ਼ਲੇਸ਼ਣ ਕਰੋ** — ਕਈ ਸਰੋਤਾਂ ਤੋਂ ਇਨਸਪੀਰੇਸ਼ਨ ਇਕੱਠੀ ਕਰੋ
2. **ਸਟਰਕਚਰ 'ਤੇ ਧਿਆਨ ਦਿਓ** — ਲੇਆਉਟ ਅਤੇ ਆਰਗੈਨੀਜ਼ੇਸ਼ਨ ਪੈਟਰਨ ਸਿੱਖੋ
3. **ਰੰਗ ਕੱਢੋ** — ਰੰਗ ਪੈਲੇਟਾਂ ਨੂੰ ਸ਼ੁਰੂਆਤੀ ਬਿੰਦੂ ਵਜੋਂ ਵਰਤੋ
4. **ਟਾਈਪੋਗ੍ਰਾਫੀ ਦਾ ਅਧਿਐਨ ਕਰੋ** — ਉਹ ਫੌਂਟ ਕ组合 ਪਛਾਣੋ ਜੋ ਤੁਹਾਨੂੰ ਪਸੰਦ ਹਨ
5. **ਨੈਵੀਗੇਸ਼ਨ ਦੀ ਸਮੀਖਿਆ ਕਰੋ** — ਮੀਨੂ ਦੇ ਆਰਗੈਨੀਜ਼ੇਸ਼ਨ ਨੂੰ ਸਮਝੋ

### ਨੈਤਿਕ Scraping

1. **robots.txt ਦਾ ਸਤਿਕਾਰ ਕਰੋ** — ਸਾਈਟ ਗਾਈਡਲਾਈਨਾਂ ਦੀ ਪਾਲਣਾ ਕਰੋ
2. **ਕੰਟੈਂਟ ਦੀ ਨਕਲ ਨਾ ਕਰੋ** — ਵਿਸ਼ਲੇਸ਼ਣ ਦੀ ਵਰਤੋਂ ਸਿਰਫ਼ ਇਨਸਪੀਰੇਸ਼ਨ ਲਈ ਕਰੋ
3. **ਅਧਿਕਾਰਾਂ ਦੀ ਪੁਸ਼ਟੀ ਕਰੋ** — ਯਕੀਨੀ ਬਣਾਓ ਕਿ ਤੁਸੀਂ ਕੱਢੇ ਗਏ ਕੰਟੈਂਟ ਦੀ ਵਰਤੋਂ ਕਰ ਸਕਦੇ ਹੋ
4. **ਸਰੋਤਾਂ ਨੂੰ ਗ੍ਰੈਟਿਊਟ ਕਰੋ** — ਇਨਸਪੀਰੇਸ਼ਨ ਦੇ ਸਰੋਤਾਂ ਨੂੰ ਕ੍ਰੈਡਿਟ ਦਿਓ
5. **ਸ਼ਰਤਾਂ ਦੀ ਪਾਲਣਾ ਕਰੋ** — ਸੇਵਾ ਦੀਆਂ ਸ਼ਰਤਾਂ (terms of service) ਦੀ ਪਾਲਣਾ ਕਰੋ

### Tool ਦੀ ਪ੍ਰਭਾਵਸ਼ਾਲੀਤਾ ਨੂੰ ਵੱਧ ਕਰਨਾ

1. **ਖਾਸ ਹੋਵੋ** — ਵਿਸ਼ਲੇਸ਼ਣ ਦੀਆਂ ਖਾਸ ਕਿਸਮਾਂ ਦੀ ਬੇਨਤੀ ਕਰੋ
2. **ਸੰਦਰਭ ਪ੍ਰਦਾਨ ਕਰੋ** — ਆਪਣੀ ਸਾਈਟ ਦਾ ਮਕਸਦ ਸਮਝਾਓ
3. **ਉਮੀਦਾਂ ਸੈੱਟ ਕਰੋ** — ਆਪਣੇ ਡਿਜ਼ਾਈਨ ਦੇ ਟੀਚਿਆਂ ਦਾ ਵਰਣਨ ਕਰੋ
4. **ਇਟਰੇਟ ਕਰੋ** — ਨਤੀਜਿਆਂ ਦੇ ਆਧਾਰ 'ਤੇ ਸੁਧਾਰ ਕਰੋ
5. **ਸਾਧਨਾਂ ਨੂੰ ਜੋੜੋ** — ਵਿਆਪਕ ਵਿਸ਼ਲੇਸ਼ਣ ਲਈ ਕਈ ਸਾਧਨਾਂ ਦੀ ਵਰਤੋਂ ਕਰੋ

## ਤ러ਬਲਸ਼ੂਟਿੰਗ (Troubleshooting)

### ਸਾਈਟ scrape ਨਹੀਂ ਹੋ ਰਹੀ

- ਜਾਂਚ ਕਰੋ ਕਿ ਸਾਈਟ ਜਨਤਕ ਤੌਰ 'ਤੇ ਪਹੁੰਚਯੋਗ ਹੈ ਜਾਂ ਨਹੀਂ
- ਪੁਸ਼ਟੀ ਕਰੋ ਕਿ robots.txt scraping ਦੀ ਇਜਾਜ਼ਤ ਦਿੰਦਾ ਹੈ
- ਕੋਈ ਹੋਰ ਸਾਈਟ ਕੋਸ਼ਿਸ਼ ਕਰੋ
- ਇੰਟਰਨੈਟ ਕਨੈਕਸ਼ਨ ਦੀ ਜਾਂਚ ਕਰੋ
- ਜੇ ਸਮੱਸਿਆ ਜਾਰੀ ਰਹਿੰਦੀ ਹੈ ਤਾਂ ਸਪੋਰਟ ਨਾਲ ਸੰਪਰਕ ਕਰੋ

### ਅਧੂਰਾ ਵਿਸ਼ਲੇਸ਼ਣ (Incomplete Analysis)

- ਸਾਈਟ ਵਿੱਚ ਡਾਇਨਾਮਿਕ ਕੰਟੈਂਟ ਹੋ ਸਕਦਾ ਹੈ
- JavaScript ਪੂਰੀ ਤਰ੍ਹਾਂ render ਨਹੀਂ ਹੋ ਸਕਦਾ
- ਵੱਡੀਆਂ ਸਾਈਟਾਂ ਲਈ timeout ਹੋ ਸਕਦਾ ਹੈ
- ਇਸ ਦੀ ਬਜਾਏ ਖਾਸ ਪੇਜਾਂ ਦਾ ਵਿਸ਼ਲੇਸ਼ਣ ਕਰਨ ਦੀ ਕੋਸ਼ਿਸ਼ ਕਰੋ
- ਖਾਸ ਵਿਸ਼ਲੇਸ਼ਣ ਕਿਸਮਾਂ ਦੀ ਬੇਨਤੀ ਕਰੋ

### ਧੀਮਾ ਪ੍ਰਦਰਸ਼ਨ (Slow Performance)

- ਵੱਡੀਆਂ ਸਾਈਟਾਂ ਦਾ ਵਿਸ਼ਲੇਸ਼ਣ ਕਰਨ ਵਿੱਚ ਜ਼ਿਆਦਾ ਸਮਾਂ ਲੱਗਦਾ ਹੈ
- ਇੱਕੋ ਸਮੇਂ ਕਈ scrapes ਕਰਨ ਨਾਲ ਧੀਮਾ ਹੋ ਜਾਂਦਾ ਹੈ
- ਨੈਟਵਰਕ ਕਨੈਕਟੀਵਿਟੀ ਗਤੀ ਨੂੰ ਪ੍ਰਭਾਵਿਤ ਕਰਦੀ ਹੈ
- ਪੀਕ ਆਵਰਾਂ (peak hours) ਦੌਰਾਨ ਦੀ ਬਜਾਏ ਕੋਸ਼ਿਸ਼ ਕਰੋ
- ਪਹਿਲਾਂ ਛੋਟੇ ਹਿੱਸਿਆਂ ਦਾ ਵਿਸ਼ਲੇਸ਼ਣ ਕਰੋ
