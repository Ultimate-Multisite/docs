# Logo SVG ਬਣਾਓ

**Logo SVG ਬਣਾਉਣ** ਦੀ ਸਮਰੱਥਾ Theme Builder ਨੂੰ ਤੁਹਾਡੀ WordPress ਸਾਈਟ ਵਿੱਚ ਕਸਟਮ ਲੋਗੋ SVG ਬਣਾਉਣ ਅਤੇ ਐਮਬੈਡ ਕਰਨ ਦੀ ਇਜਾਜ਼ਤ ਦਿੰਦੀ ਹੈ, ਜਿਸ ਵਿੱਚ ਆਟੋਮੈਟਿਕ ਨਾਮ-ਸਪੇਸ ਸੁਰੱਖਿਅਤ ਸੈਨੀਟਾਈਜ਼ੇਸ਼ਨ ਸ਼ਾਮਲ ਹੈ।

## ਸੰਖੇਪ ਜਾਣਕਾਰੀ (Overview) {#overview}

ਇਹ ਸਮਰੱਥਾ ਤੁਹਾਡੀ ਸਾਈਟ ਦੀ ਬ੍ਰਾਂਡਿੰਗ ਦਿਸ਼ਾ ਅਤੇ ਡਿਜ਼ਾਈਨ ਪਸੰਦਾਂ ਦੇ ਆਧਾਰ 'ਤੇ ਸਕੇਲੇਬਲ ਵੈਕਟਰ ਗ੍ਰਾਫਿਕਸ (SVG) ਲੋਗੋ ਬਣਾਉਂਦੀ ਹੈ। ਬਣਾਏ ਗਏ SVG ਨੂੰ ਆਟੋਮੈਟਿਕ ਤੌਰ 'ਤੇ ਸੈਨੀਟਾਈਜ਼ ਕੀਤਾ ਜਾਂਦਾ ਹੈ ਤਾਂ ਜੋ ਇਹ ਯਕੀਨੀ ਬਣਾਇਆ ਜਾ ਸਕੇ ਕਿ ਇਹ WordPress ਵਿੱਚ ਵਰਤਣ ਲਈ ਸੁਰੱਖਿਅਤ ਹਨ, ਨਾਲ ਹੀ ਵਿਜ਼ੂਅਲ ਇੰਟੈਗ੍ਰਿਟੀ ਵੀ ਬਣੀ ਰਹੇ।

## ਪੈਰਾਮੀਟਰ (Parameters) {#parameters}

| ਪੈਰਾਮੀਟਰ | ਟਾਈਪ | ਜ਼ਰੂਰੀ | ਵੇਰਵਾ |
|-----------|------|----------|-------------|
| `site_name` | string | Yes | ਸਾਈਟ ਦਾ ਨਾਮ ਜਿਸ ਲਈ ਲੋਗੋ ਬਣਾਉਣਾ ਹੈ |
| `style` | string | Yes | ਡਿਜ਼ਾਈਨ ਸ਼ੈਲੀ (ਉਦਾਹਰਨ ਲਈ, "modern", "classic", "minimalist", "playful") |
| `colors` | array | No | ਲੋਗੋ ਵਿੱਚ ਵਰਤਣ ਲਈ ਹੈਕਸ ਰੰਗ ਕੋਡਾਂ ਦਾ ਐਰੇ (ਉਦਾਹਰਨ ਲਈ, `["#678233", "#ffffff"]`) |
| `width` | number | No | ਪਿਕਸਲਾਂ ਵਿੱਚ SVG ਦੀ ਚੌੜਾਈ (ਡਿਫਾਲਟ: 200) |
| `height` | number | No | ਪਿਕਸਲਾਂ ਵਿੱਚ SVG ਦੀ ਉਚਾਈ (ਡਿਫਾਲਟ: 200) |
| `include_text` | boolean | No | ਕੀ ਲੋਗੋ ਵਿੱਚ ਸਾਈਟ ਦਾ ਨਾਮ ਟੈਕਸਟ ਵਜੋਂ ਸ਼ਾਮਲ ਕਰਨਾ ਹੈ (ਡਿਫਾਲਟ: true) |

## ਆਉਟਪੁੱਟ ਫਾਰਮੈਟ (Output Format) {#output-format}

ਇਹ ਸਮਰੱਥਾ ਹੇਠ ਲਿਖੀ ਢਾਂਚਾ ਵਾਲੀ SVG ਸਟ੍ਰਿੰਗ ਵਾਪਸ ਕਰਦੀ ਹੈ:

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## SVG ਸੈਨੀਟਾਈਜ਼ੇਸ਼ਨ ਵਿਵਹਾਰ (SVG Sanitisation Behaviour) {#svg-sanitisation-behaviour}

ਬਣਾਏ ਗਏ SVG ਆਟੋਮੈਟਿਕ ਨਾਮ-ਸਪੇਸ ਸੁਰੱਖਿਅਤ ਸੈਨੀਟਾਈਜ਼ੇਸ਼ਨ ਤੋਂ ਗੁਜ਼ਰਦੇ ਹਨ ਤਾਂ ਜੋ:

- **ਖ਼ਤਰਨਾਕ ਐਟ੍ਰੀਬਿਊਟਸ ਹਟਾਏ ਜਾਣ** — ਇਹ ਘਟਨਾ ਹੈਂਡਲਰਾਂ (event handlers), ਸਕ੍ਰਿਪਟਾਂ, ਅਤੇ ਸੰਭਾਵਤ ਤੌਰ 'ਤੇ ਖ਼ਤਰਨਾਕ ਐਟ੍ਰੀਬਿਊਟਸ ਨੂੰ ਹਟਾ ਦਿੰਦਾ ਹੈ।
- **ਨੇਮਸਪੇਸ ਸੁਰੱਖਿਅਤ ਰਹਿਣ** — ਸਹੀ ਰੈਂਡਰਿੰਗ ਲਈ SVG ਨੇਮਸਪੇਸ (xmlns, xlink) ਨੂੰ ਬਣਾਈ ਰੱਖਦਾ ਹੈ।
- **ਢਾਂਚਾ ਦੀ ਤਸਦੀਕ** — ਇਹ ਯਕੀਨੀ ਬਣਾਉਂਦਾ ਹੈ ਕਿ SVG W3C ਮਾਪਦੰਡਾਂ (standards) ਦੀ ਪਾਲਣਾ ਕਰਦਾ ਹੈ।
- **ਐਂਟਿਟੀ ਕੋਡਿੰਗ** — ਟੈਕਸਟ ਸਮੱਗਰੀ ਵਿੱਚ ਖਾਸ ਅੱਖਰਾਂ ਨੂੰ ਸਹੀ ਢੰਗ ਨਾਲ ਐਸਕੇਪ ਕਰਦਾ ਹੈ।
- **ਬਾਹਰੀ ਰੈਫਰੈਂਸ ਹਟਾਏ ਜਾਣ** — ਬਾਹਰੀ ਸ਼ੀਟਸਟਾਈਲ ਅਤੇ ਤਸਵੀਰ ਰੈਫਰੈਂਸ ਨੂੰ ਹਟਾ ਦਿੰਦਾ ਹੈ।

ਇਹ ਯਕੀਨੀ ਬਣਾਉਂਦਾ ਹੈ ਕਿ SVG ਨੂੰ WordPress ਵਿੱਚ ਸਿੱਧਾ ਐਮਬੈਡ ਕਰਨਾ ਸੁਰੱਖਿਅਤ ਹੈ, ਜਿਸ ਲਈ ਵਾਧੂ ਸੈਨੀਟਾਈਜ਼ੇਸ਼ਨ ਦੀ ਲੋੜ ਨਹੀਂ ਹੈ।

## ਵਰਤੋਂ ਦਾ ਉਦਾਹਰਨ (Usage Example) {#usage-example}

**ਪ੍ਰੋਮਪਟ (Prompt):**
```
Generate a modern logo for my tech startup called "CloudSync" using blue and white colors.
```

**ਨਤੀਜਾ (Result):**
ਇਹ ਸਮਰੱਥਾ ਇੱਕ SVG ਲੋਗੋ ਬਣਾਉਂਦੀ ਹੈ ਜੋ:
- ਸਾਈਟ ਦਾ ਨਾਮ "CloudSync" ਸ਼ਾਮਲ ਕਰਦਾ ਹੈ
- ਨਿਰਧਾਰਤ ਨੀਲੇ ਅਤੇ ਸਫੈਦ ਰੰਗ ਸਕੀਮ ਦੀ ਵਰਤੋਂ ਕਰਦਾ ਹੈ
- ਆਧੁਨਿਕ ਡਿਜ਼ਾਈਨ ਸਿਧਾਂਤਾਂ ਦੀ ਪਾਲਣਾ ਕਰਦਾ ਹੈ
- ਆਟੋਮੈਟਿਕ ਤੌਰ 'ਤੇ ਸੈਨੀਟਾਈਜ਼ ਹੋ ਕੇ ਵਰਤਣ ਲਈ ਤਿਆਰ ਹੈ

## Theme Builder ਨਾਲ ਏਕੀਕਰਨ (Integration with Theme Builder) {#integration-with-theme-builder}

ਜਦੋਂ Theme Builder ਦੀ ਡਿਜ਼ਾਈਨ- ਦਿਸ਼ਾ ਚੋਣ ਦੀ ਵਰਤੋਂ ਕੀਤੀ ਜਾਂਦੀ ਹੈ, ਤਾਂ Generate Logo SVG ਸਮਰੱਥਾ:

1. ਤੁਹਾਡੀ ਡਿਜ਼ਾਈਨ ਦਿਸ਼ਾ ਅਤੇ ਰੰਗ ਪੈਲੇਟ ਦਾ ਵਿਸ਼ਲੇਸ਼ਣ ਕਰਦੀ ਹੈ
2. ਤੁਹਾਡੀਆਂ ਪਸੰਦਾਂ ਨਾਲ ਮਿਲਦਾ ਇੱਕ ਕਸਟਮ SVG ਲੋਗੋ ਬਣਾਉਂਦੀ ਹੈ
3. ਲੋਗੋ ਨੂੰ ਆਟੋਮੈਟਿਕ ਤੌਰ 'ਤੇ ਤੁਹਾਡੀ ਸਾਈਟ ਦੇ ਹੈਡਰ/ਬ੍ਰਾਂਡਿੰਗ ਖੇਤਰ ਵਿੱਚ ਐਮਬੈਡ ਕਰਦੀ ਹੈ
4. SVG ਨੂੰ WordPress ਮੀਡੀਆ ਵਿੱਚ ਇੱਕ ਕਸਟਮ ਲੋਗੋ ਵਜੋਂ ਸਟੋਰ ਕਰਦੀ ਹੈ

## ਵਧੀਆ ਅਭਿਆਸ (Best Practices) {#best-practices}

- **ਸਪਸ਼ਟ ਸ਼ੈਲੀ ਪਸੰਦਾਂ ਪ੍ਰਦਾਨ ਕਰੋ** — ਤੁਸੀਂ ਕਿਸ ਤਰ੍ਹਾਂ ਦੀ ਡਿਜ਼ਾਈਨ ਸ਼ੈਲੀ ਚਾਹੁੰਦੇ ਹਨ (ਆਧੁਨਿਕ, ਕਲਾਸਿਕ, ਖੇਡਣ ਵਾਲੀ, ਆਦਿ) ਦਾ ਵਰਣਨ ਕਰੋ।
- **ਰੰਗ ਨਿਰਧਾਰਤ ਕਰੋ** — ਇਕਸਾਰਤਾ ਲਈ ਆਪਣੇ ਬ੍ਰਾਂਡ ਰੰਗ ਸ਼ਾਮਲ ਕਰੋ।
- **ਰੈਂਡਰਿੰਗ ਦਾ ਟੈਸਟ ਕਰੋ** — ਪੁਸ਼ਟੀ ਕਰੋ ਕਿ ਲੋਗੋ ਵੱਖ-ਵੱਖ ਸਕ੍ਰੀਨ ਸਾਈਜ਼ਾਂ 'ਤੇ ਸਹੀ ਢੰਗ ਨਾਲ ਦਿਖਾਈ ਦਿੰਦਾ ਹੈ।
- **ਹੋਰ ਕਸਟਮਾਈਜ਼ ਕਰੋ** — ਆਕਾਰ ਅਤੇ ਸਥਾਨ ਨੂੰ 调整 ਕਰਨ ਲਈ WordPress ਦੇ ਲੋਗੋ ਕਸਟਮਾਈਜ਼ੇਸ਼ਨ ਟੂਲਸ ਦੀ ਵਰਤੋਂ ਕਰੋ।

## ਸੀਮਾਵਾਂ (Limitations) {#limitations}

- SVG ਲੋਗੋ ਸਟੈਟਿਕ ਗ੍ਰਾਫਿਕਸ ਵਜੋਂ ਬਣਾਏ ਜਾਂਦੇ ਹਨ (ਐਨੀਮੇਟਿਡ ਨਹੀਂ)।
- ਕਈ ਤੱਤਾਂ ਵਾਲੇ ਗੁੰਝਲਦਾਰ ਲੋਗੋ ਲਈ ਮਨੁੱਖੀ ਸੁਧਾਰ ਦੀ ਲੋੜ ਹੋ ਸਕਦੀ ਹੈ।
- ਕਸਟਮ ਫੌਂਟ ਸਪੋਰਟ ਨਹੀਂ ਕੀਤੇ ਜਾਂਦੇ; ਟੈਕਸਟ ਸਿਸਟਮ ਫੌਂਟਾਂ ਦੀ ਵਰਤੋਂ ਕਰਦਾ ਹੈ।
- ਬਹੁਤ ਵੱਡੇ ਜਾਂ ਬਹੁਤ ਛੋਟੇ ਆਕਾਰ ਗੁਣਵੱਤਾ ਨੂੰ ਪ੍ਰਭਾਵਿਤ ਕਰ ਸਕਦੇ ਹਨ।
