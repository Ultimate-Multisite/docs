# Superdav AI Agent

Superdav AI Agent ਇੱਕ ਉੱਨਤ WordPress AI ਸਹਾਇਕ ਹੈ ਜੋ Gratis AI Agent ਨੂੰ theme-ਸਚੇਤ ਹੁਨਰਾਂ, ਵਧੀਆਂ ਇੰਟਰਨੈੱਟ ਖੋਜ ਸਮਰੱਥਾਵਾਂ, ਸਾਈਟ ਨਿਰਮਾਤਾ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ, ਅਤੇ ਵਿਕਲਪਿਕ ਆਟੋਮੇਸ਼ਨ ਇੰਟੀਗ੍ਰੇਸ਼ਨਾਂ ਨਾਲ ਵਧਾਉਂਦਾ ਹੈ। ਸੰਸਕਰਣ 1.18.0 Google Calendar ਪੜ੍ਹਨ ਵਾਲੇ ਟੂਲ, ਸੰਪਰਕ ਅਤੇ ਹਾਜ਼ਰ ਵਿਅਕਤੀ ਮੈਪਿੰਗ, ਮਨੁੱਖੀ ਮਨਜ਼ੂਰੀ ਗੇਟ, ਰਿਮਾਈਂਡਰ ਡਿਡੂਪਲੀਕੇਸ਼ਨ, TextBee SMS notifications, ਪ੍ਰਬੰਧਿਤ Superdav ਸੇਵਾ ਪ੍ਰੋਵਿਜ਼ਨਿੰਗ, ਅਤੇ ਭਰੋਸੇਯੋਗ ਡਿਵੈਲਪਰ ਵਰਕਫ਼ਲੋਜ਼ ਲਈ ਵੱਖਰੇ ਤੌਰ 'ਤੇ ਵੰਡਿਆ ਗਿਆ Advanced ਸਾਥੀ ਪੈਕੇਜ ਸ਼ਾਮਲ ਕਰਦਾ ਹੈ।

## ਮੁੱਖ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ

- **Theme-ਸਚੇਤ ਹੁਨਰ** — ਤੁਹਾਡੇ ਸਰਗਰਮ WordPress theme ਅਤੇ ਇੰਸਟਾਲ ਕੀਤੇ plugins ਦੇ ਆਧਾਰ 'ਤੇ ਮਾਰਗਦਰਸ਼ਨ ਅਤੇ ਸਮਰੱਥਾਵਾਂ ਨੂੰ ਆਪਣੇ ਆਪ ਅਨੁਕੂਲ ਕਰਦਾ ਹੈ
- **ਕਈ ਇੰਟਰਨੈੱਟ ਖੋਜ ਪ੍ਰਦਾਤਾ** — ਹੋਰ ਸਮ੍ਰਿੱਧ ਖੋਜ ਨਤੀਜਿਆਂ ਅਤੇ ਉੱਨਤ ਰਿਸਰਚ ਸਮਰੱਥਾਵਾਂ ਲਈ Brave Search ਅਤੇ Tavily ਵਿੱਚੋਂ ਚੁਣੋ
- **ਸਾਈਟ ਨਿਰਮਾਤਾ ਸੰਪਰਕ ਫਾਰਮ** — ਆਪਣੇ ਇੰਸਟਾਲ ਕੀਤੇ ਫਾਰਮ plugin ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਚੈਟ ਇੰਟਰਫੇਸ ਤੋਂ ਸਿੱਧੇ ਪੂਰੀ ਤਰ੍ਹਾਂ ਕਾਰਗਰ ਸੰਪਰਕ ਫਾਰਮ ਬਣਾਓ
- **ਵਧਾਇਆ ਗਿਆ ਸਾਈਟ ਨਿਰਮਾਤਾ** — ਬਿਹਤਰ plugin ਖੋਜ ਅਤੇ ਗਲਤੀ ਰਿਕਵਰੀ ਨਾਲ ਸੁਧਾਰੇ ਹੋਏ ਬਹੁ-ਪੜਾਅ ਸਾਈਟ ਨਿਰਮਾਣ ਪਲਾਨ
- **ਬਿਲਟ-ਇਨ theme ਹੁਨਰ** — Block Themes, Classic Themes, Kadence Blocks, ਅਤੇ Kadence Theme ਲਈ ਵਿਸ਼ੇਸ਼ ਮਾਰਗਦਰਸ਼ਨ
- **ਸੁਧਾਰੀ ਹੋਈ WooCommerce ਇੰਟੀਗ੍ਰੇਸ਼ਨ** — ਬਿਹਤਰ ਭਰੋਸੇਯੋਗਤਾ ਅਤੇ ਅਨੁਕੂਲਤਾ ਲਈ ਮੂਲ WooCommerce APIs ਦੀ ਵਰਤੋਂ ਕਰਨ ਵਾਸਤੇ ਰੀਫੈਕਟਰ ਕੀਤਾ ਗਿਆ
- **ਆਟੋਮੈਟਿਕ ਪ੍ਰਦਾਤਾ ਪਛਾਣ** — ਉਪਲਬਧ plugins ਅਤੇ themes ਨੂੰ ਸਰਗਰਮ ਜਾਂ ਨਿਸ਼ਕ੍ਰਿਆ ਕੀਤੇ ਜਾਣ 'ਤੇ ਆਪਣੇ ਆਪ ਪਛਾਣਦਾ ਅਤੇ ਰਿਫ੍ਰੈਸ਼ ਕਰਦਾ ਹੈ
- **ਕਸਟਮ ਲੋਗੋ ਜਨਰੇਸ਼ਨ** — ਆਟੋਮੈਟਿਕ namespace-ਸੁਰੱਖਿਅਤ ਸੈਨੀਟਾਈਜ਼ੇਸ਼ਨ ਨਾਲ ਕਸਟਮ SVG ਲੋਗੋ ਤਿਆਰ ਅਤੇ ਐਂਬੈਡ ਕਰੋ
- **ਪੈਲੇਟ ਕਾਂਟ੍ਰਾਸਟ ਪ੍ਰਮਾਣਿਕਤਾ** — ਆਪਣੇ theme 'ਤੇ ਲਾਗੂ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ WCAG ਐਕਸੈਸਿਬਿਲਟੀ ਅਨੁਕੂਲਤਾ ਲਈ ਰੰਗ ਜੋੜਿਆਂ ਦੀ ਜਾਂਚ ਕਰੋ
- **ਫੋਟੋ-ਵਧਾਈ ਖੋਜ** — ਡਿਜ਼ਾਈਨ ਦਿਸ਼ਾ ਨੂੰ ਪ੍ਰਭਾਵਿਤ ਕਰਨ ਲਈ Theme Builder ਖੋਜ ਇੰਟਰਵਿਊ ਦੌਰਾਨ ਫੋਟੋਆਂ ਅੱਪਲੋਡ ਕਰੋ
- **ਮਹਿਮਾਨਨਿਵਾਜ਼ੀ ਮੀਨੂ** — ਰੈਸਟੋਰੈਂਟਾਂ, ਕੈਫੇਆਂ, ਅਤੇ ਮਹਿਮਾਨਨਿਵਾਜ਼ੀ ਕਾਰੋਬਾਰਾਂ ਲਈ ਸੰਰਚਿਤ ਭੋਜਨ ਅਤੇ ਪੇਅ ਮੀਨੂ ਪੰਨੇ ਬਣਾਓ
- **ਡੈਸਕਟਾਪ ਅਤੇ ਮੋਬਾਈਲ ਪੂਰਵਦਰਸ਼ਨ** — ਡਿਜ਼ਾਈਨ-ਦਿਸ਼ਾ ਚੋਣ ਦੌਰਾਨ ਆਪਣੇ ਡਿਜ਼ਾਈਨ ਦਾ ਡੈਸਕਟਾਪ ਅਤੇ ਮੋਬਾਈਲ ਡਿਵਾਈਸਾਂ 'ਤੇ ਪੂਰਵਦਰਸ਼ਨ ਕਰੋ
- **ਟੀਅਰ 1 ਸਾਈਟ ਸਕ੍ਰੈਪਿੰਗ** — sd-ai-agent/site-scrape ਹੁਣ ਮੁਕਾਬਲਤੀ ਅਤੇ ਪ੍ਰੇਰਣਾ ਵੈੱਬਸਾਈਟਾਂ ਦਾ ਵਿਸ਼ਲੇਸ਼ਣ ਕਰਨ ਲਈ ਮੂਲ ਰੂਪ ਵਿੱਚ ਉਪਲਬਧ ਹੈ
- **Google Calendar ਪੜ੍ਹਨ ਵਾਲੇ ਟੂਲ** — ਕਨਫਿਗਰ ਕੀਤੇ ਕੈਲੰਡਰ ਅਤੇ ਇਵੈਂਟ ਪੜ੍ਹੋ ਤਾਂ ਜੋ ਆਟੋਮੇਸ਼ਨ ਆਉਣ ਵਾਲੇ ਸ਼ਡਿਊਲ ਸੰਦਰਭ ਬਾਰੇ ਤਰਕ ਕਰ ਸਕਣ
- **ਸੰਪਰਕ ਅਤੇ ਹਾਜ਼ਰ ਵਿਅਕਤੀ ਮੈਪਿੰਗ** — ਫਾਲੋ-ਅੱਪ ਕੰਮ ਬਣਾਉਣ ਤੋਂ ਪਹਿਲਾਂ ਕੈਲੰਡਰ ਭਾਗੀਦਾਰਾਂ ਨੂੰ ਜਾਣੇ-ਪਛਾਣੇ ਸਾਈਟ ਯੂਜ਼ਰਾਂ ਅਤੇ ਸੰਪਰਕਾਂ ਨਾਲ ਮਿਲਾਓ
- **ਮਨੁੱਖੀ ਮਨਜ਼ੂਰੀ ਗੇਟ** — ਸੁਨੇਹੇ ਭੇਜਣ, ਕੰਮ ਮੁੜ ਸ਼ੁਰੂ ਕਰਨ, ਜਾਂ ਬਦਲਾਅ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ ਸਮੀਖਿਆ ਲਈ ਸੰਵੇਦਨਸ਼ੀਲ ਆਟੋਮੇਸ਼ਨਾਂ ਨੂੰ ਰੋਕੋ
- **ਰਿਮਾਈਂਡਰ ਰਿਕਾਰਡ ਅਤੇ ਡਿਡੂਪਲੀਕੇਸ਼ਨ** — ਪਹਿਲਾਂ ਹੀ ਭੇਜੇ ਜਾ ਚੁੱਕੇ ਰਿਮਾਈਂਡਰਾਂ ਨੂੰ ਟ੍ਰੈਕ ਕਰੋ ਤਾਂ ਜੋ ਦੁਹਰਾਏ ਜਾਣ ਵਾਲੇ ਆਟੋਮੇਸ਼ਨ ਰਨ ਉਸੇ ਪ੍ਰਾਪਤਕਰਤਾ ਨੂੰ ਦੋ ਵਾਰ ਸੂਚਿਤ ਨਾ ਕਰਨ
- **TextBee SMS notifications** — ਜਦੋਂ ਕੋਈ SMS ਪ੍ਰਦਾਤਾ ਸਮਰੱਥ ਹੋਵੇ ਤਾਂ TextBee ਰਾਹੀਂ ਕਨਫਿਗਰ ਕੀਤੀਆਂ ਟੈਕਸਟ-ਸੁਨੇਹਾ ਸੂਚਨਾਵਾਂ ਭੇਜੋ
- **ਪ੍ਰਬੰਧਿਤ Superdav ਸੇਵਾ ਪ੍ਰੋਵਿਜ਼ਨਿੰਗ** — ਸਮਰਥਿਤ ਸਾਈਟਾਂ ਨੂੰ ਮੈਨੂਅਲ endpoint ਕਨਫਿਗਰੇਸ਼ਨ ਦੀ ਬਜਾਏ ਆਟੋਮੈਟਿਕ ਸੈੱਟਅਪ ਨਾਲ ਹੋਸਟ ਕੀਤੇ Superdav endpoints ਨਾਲ ਜੋੜੋ
- **Advanced ਸਾਥੀ ਪੈਕੇਜ** — ਫ਼ਾਈਲਸਿਸਟਮ, ਡਾਟਾਬੇਸ, WP-CLI, REST ਡਿਸਪੈਚਰ, plugin ਨਿਰਮਾਤਾ, git ਸਨੈਪਸ਼ਾਟ, ਯੂਜ਼ਰ-ਪ੍ਰਬੰਧਨ, ਅਤੇ ਬੈਂਚਮਾਰਕ ਵਰਕਫ਼ਲੋਜ਼ ਲਈ ਵੱਖਰੇ ਤੌਰ 'ਤੇ ਵੰਡੇ ਗਏ ਭਰੋਸੇਯੋਗ-ਡਿਵੈਲਪਰ ਟੂਲ

## ਲੋੜਾਂ

- WordPress 6.0 ਜਾਂ ਇਸ ਤੋਂ ਉੱਚਾ
- PHP 8.1 ਜਾਂ ਇਸ ਤੋਂ ਉੱਚਾ
- WordPress Multisite (ਨੈੱਟਵਰਕ-ਸਰਗਰਮ) ਜਾਂ ਸਿੰਗਲ-ਸਾਈਟ ਇੰਸਟਾਲੇਸ਼ਨ
- ਇੱਕ ਸਰਗਰਮ AI ਪ੍ਰਦਾਤਾ ਕਨੈਕਟਰ plugin (Anthropic, OpenAI, ਜਾਂ ਅਨੁਕੂਲ)
- Gratis AI Agent v1.9.0 ਜਾਂ ਇਸ ਤੋਂ ਉੱਚਾ (Superdav AI Agent, Gratis AI Agent ਨੂੰ ਵਧਾਉਂਦਾ ਹੈ)

## ਇੰਸਟਾਲੇਸ਼ਨ

1. plugin ਫ਼ਾਈਲਾਂ ਨੂੰ `/wp-content/plugins/superdav-ai-agent` 'ਤੇ ਅੱਪਲੋਡ ਕਰੋ
2. WordPress ਵਿੱਚ **Plugins** ਸਕ੍ਰੀਨ ਰਾਹੀਂ plugin ਨੂੰ ਸਰਗਰਮ ਕਰੋ
3. ਯਕੀਨੀ ਬਣਾਓ ਕਿ Gratis AI Agent ਵੀ ਸਰਗਰਮ ਹੈ
4. **Gratis AI Agent → Settings** 'ਤੇ ਜਾਓ ਅਤੇ ਲੋੜ ਅਨੁਸਾਰ ਆਪਣਾ ਇੰਟਰਨੈੱਟ ਖੋਜ ਪ੍ਰਦਾਤਾ, ਪ੍ਰਬੰਧਿਤ Superdav ਸੇਵਾ ਕਨੈਕਸ਼ਨ, Google Calendar, ਜਾਂ TextBee ਕ੍ਰੈਡੈਂਸ਼ਲ ਕਨਫਿਗਰ ਕਰੋ

WordPress.org ਪੈਕੇਜ ਵਿੱਚ ਮੁੱਖ Superdav AI Agent ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ ਸ਼ਾਮਲ ਹਨ। Advanced ਸਾਥੀ ਪੈਕੇਜ ਉਹਨਾਂ ਵਾਤਾਵਰਣਾਂ ਲਈ ਵੱਖਰੇ ਤੌਰ 'ਤੇ ਬਣਾਇਆ ਅਤੇ ਵੰਡਿਆ ਜਾਂਦਾ ਹੈ ਜਿੱਥੇ ਭਰੋਸੇਯੋਗ ਡਿਵੈਲਪਰ ਟੂਲ ਸਪਸ਼ਟ ਤੌਰ 'ਤੇ ਇੰਸਟਾਲ ਅਤੇ ਅਧਿਕ੍ਰਿਤ ਕੀਤੇ ਜਾਂਦੇ ਹਨ।

## ਤੁਰੰਤ ਸ਼ੁਰੂਆਤ

ਸਰਗਰਮ ਹੋਣ ਤੋਂ ਬਾਅਦ, Superdav AI Agent ਤੁਹਾਡੀ Gratis AI Agent ਇੰਸਟਾਲੇਸ਼ਨ ਨੂੰ theme-ਸਚੇਤ ਹੁਨਰਾਂ ਅਤੇ ਵਾਧੂ ਸਮਰੱਥਾਵਾਂ ਨਾਲ ਆਪਣੇ ਆਪ ਵਧਾਉਂਦਾ ਹੈ। WordPress ਐਡਮਿਨ ਸਾਈਡਬਾਰ ਵਿੱਚ **Gratis AI Agent** ਪੈਨਲ ਖੋਲ੍ਹੋ ਅਤੇ ਨਵੀਆਂ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ ਦੀ ਵਰਤੋਂ ਸ਼ੁਰੂ ਕਰੋ।

**ਉਦਾਹਰਨ ਨਿਰਦੇਸ਼:**

```
ਸੰਪਰਕ ਪੰਨੇ ਵਿੱਚ ਇੱਕ ਸੰਪਰਕ ਫਾਰਮ ਸ਼ਾਮਲ ਕਰੋ।
```

```
ਨਵੀਨਤਮ WordPress ਸੁਰੱਖਿਆ ਸਰਵੋਤਮ ਅਭਿਆਸਾਂ ਲਈ ਵੈੱਬ 'ਤੇ ਖੋਜ ਕਰੋ।
```

```
ਸਰਗਰਮ theme ਦੇ ਬਲਾਕ ਪੈਟਰਨਾਂ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਇੱਕ ਹੀਰੋ ਸੈਕਸ਼ਨ ਬਣਾਓ।
```

```
ਕੱਲ੍ਹ ਦੇ ਕਨਫਿਗਰ ਕੀਤੇ Google Calendar ਇਵੈਂਟਾਂ ਦੀ ਸਮੀਖਿਆ ਕਰੋ ਅਤੇ ਹਾਜ਼ਰ ਵਿਅਕਤੀਆਂ ਲਈ ਰਿਮਾਈਂਡਰਾਂ ਦਾ ਮਸੌਦਾ ਬਣਾਓ।
```

```
ਮੇਰੇ ਵੱਲੋਂ ਸੁਨੇਹਾ ਮਨਜ਼ੂਰ ਕਰਨ ਤੋਂ ਬਾਅਦ ਇੱਕ SMS ਰਿਮਾਈਂਡਰ ਭੇਜੋ।
```

## Theme-ਸਚੇਤ ਹੁਨਰ

Superdav AI Agent ਆਪਣੇ ਆਪ ਤੁਹਾਡੇ ਸਰਗਰਮ theme ਅਤੇ ਇੰਸਟਾਲ ਕੀਤੇ plugins ਨੂੰ ਪਛਾਣਦਾ ਹੈ, ਅਤੇ ਸਹਾਇਕ ਦੇ ਸੰਦਰਭ ਵਿੱਚ ਉਚਿਤ theme-ਸਚੇਤ ਹੁਨਰ ਲੋਡ ਕਰਦਾ ਹੈ। ਉਪਲਬਧ ਹੁਨਰਾਂ ਵਿੱਚ ਸ਼ਾਮਲ ਹਨ:

| ਹੁਨਰ | ਇਨ੍ਹਾਂ 'ਤੇ ਲਾਗੂ |
|---|---|
| **Block Themes** | WordPress ਬਲਾਕ-ਆਧਾਰਿਤ (ਪੂਰੀ ਸਾਈਟ ਸੰਪਾਦਨ) ਆਰਕੀਟੈਕਚਰ ਵਰਤਣ ਵਾਲੇ themes |
| **Classic Themes** | PHP ਟੈਂਪਲੇਟਾਂ ਅਤੇ ਕਲਾਸਿਕ ਐਡੀਟਰ ਵਰਤਣ ਵਾਲੇ ਰਵਾਇਤੀ WordPress themes |
| **Kadence Blocks** | ਸਰਗਰਮ Kadence Blocks plugin ਵਾਲੀਆਂ ਸਾਈਟਾਂ |
| **Kadence Theme** | ਸਰਗਰਮ theme ਵਜੋਂ Kadence theme ਵਾਲੀਆਂ ਸਾਈਟਾਂ |

ਹਰੇਕ ਹੁਨਰ ਬਾਰੇ ਵਿਸਤ੍ਰਿਤ ਮਾਰਗਦਰਸ਼ਨ ਲਈ [Theme-ਸਚੇਤ ਹੁਨਰ](../../user-guide/configuration/theme-aware-skills) ਵੇਖੋ।

## ਇੰਟਰਨੈੱਟ ਖੋਜ ਪ੍ਰਦਾਤਾ

Superdav AI Agent v1.10.0 ਇੰਟਰਨੈੱਟ ਖੋਜ ਕੁਏਰੀਆਂ ਲਈ Brave Search ਦੇ ਵਿਕਲਪ ਵਜੋਂ **Tavily** ਸ਼ਾਮਲ ਕਰਦਾ ਹੈ। ਦੋਵੇਂ ਪ੍ਰਦਾਤਾ ਰੀਅਲ-ਟਾਈਮ ਖੋਜ ਨਤੀਜੇ ਦਿੰਦੇ ਹਨ ਅਤੇ **Gratis AI Agent → Settings → Advanced** ਵਿੱਚ ਕਨਫਿਗਰ ਕੀਤੇ ਜਾ ਸਕਦੇ ਹਨ।

| ਪ੍ਰਦਾਤਾ | ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ | ਸੈਟਅੱਪ |
|---|---|---|
| **Brave Search** | ਤੇਜ਼, ਪਰਦੇਦਾਰੀ-ਕੇਂਦ੍ਰਿਤ ਖੋਜ ਨਤੀਜੇ | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | ਹੋਰ ਸੰਮ੍ਰਿਧ ਖੋਜ ਨਤੀਜੇ, ਉੱਨਤ ਖੋਜ ਸਮਰੱਥਾਵਾਂ | [Tavily API](https://tavily.com/) |

ਕਨਫਿਗਰੇਸ਼ਨ ਹਦਾਇਤਾਂ ਲਈ [Internet Search](../../user-guide/configuration/internet-search) ਵੇਖੋ।

## Theme Builder ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ

Superdav AI Agent v1.16.0 ਵਿੱਚ Theme Builder ਸ਼ਕਤੀਸ਼ਾਲੀ ਨਵੀਆਂ ਸਮਰੱਥਾਵਾਂ ਸ਼ਾਮਲ ਕਰਦਾ ਹੈ:

### ਫੋਟੋ ਅੱਪਲੋਡ ਨਾਲ ਡਿਸਕਵਰੀ ਇੰਟਰਵਿਊ

Theme Builder ਡਿਸਕਵਰੀ ਇੰਟਰਵਿਊ ਵਿੱਚ ਹੁਣ ਇੱਕ ਫੋਟੋ ਅੱਪਲੋਡ ਕਦਮ ਸ਼ਾਮਲ ਹੈ ਜਿੱਥੇ ਤੁਸੀਂ ਦ੍ਰਿਸ਼ਟੀਗਤ ਪ੍ਰੇਰਣਾ ਦੇ ਸਕਦੇ ਹੋ। Theme Builder ਨੂੰ ਤੁਹਾਡੀ ਡਿਜ਼ਾਈਨ ਦਿਸ਼ਾ ਸਮਝਣ ਵਿੱਚ ਮਦਦ ਲਈ ਆਪਣੇ ਬ੍ਰਾਂਡ, ਮੁਕਾਬਲੇਦਾਰ ਸਾਈਟਾਂ, ਜਾਂ ਡਿਜ਼ਾਈਨ ਪ੍ਰੇਰਣਾ ਦੀਆਂ ਫੋਟੋਆਂ ਅੱਪਲੋਡ ਕਰੋ।

ਵਿਸਤ੍ਰਿਤ ਰਹਿਨੁਮਾਈ ਲਈ [ਡਿਸਕਵਰੀ ਇੰਟਰਵਿਊ](./theme-builder/discovery-interview.md) ਵੇਖੋ।

### ਡੈਸਕਟਾਪ ਅਤੇ ਮੋਬਾਈਲ ਪੂਰਵਦਰਸ਼ਨਾਂ ਨਾਲ ਡਿਜ਼ਾਈਨ ਦਿਸ਼ਾ

Theme Builder ਵੱਲੋਂ ਤੁਹਾਡਾ ਪੂਰਾ theme ਤਿਆਰ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ ਆਪਣੀ ਡਿਜ਼ਾਈਨ ਨੂੰ ਡੈਸਕਟਾਪ ਅਤੇ ਮੋਬਾਈਲ ਦੋਹਾਂ ਡਿਵਾਈਸਾਂ ਤੇ ਸਮੀਖਿਆ ਕਰੋ। ਕਈ ਜਵਾਬਦੇਹ ਬ੍ਰੇਕਪੋਇੰਟਾਂ ਤੇ ਰੀਅਲ-ਟਾਈਮ ਪੂਰਵਦਰਸ਼ਨ ਵੇਖੋ।

ਵਿਸਤ੍ਰਿਤ ਰਹਿਨੁਮਾਈ ਲਈ [ਡਿਜ਼ਾਈਨ ਦਿਸ਼ਾ](./theme-builder/design-direction.md) ਵੇਖੋ।

Theme Builder ਹੁਣ ਰੈਸਟੋਰੈਂਟਾਂ, ਕੈਫੇਆਂ, ਬਾਰਾਂ, ਅਤੇ ਹੋਰ ਹਾਸਪਿਟਾਲਿਟੀ ਕਾਰੋਬਾਰਾਂ ਲਈ ਢਾਂਚਾਬੱਧ ਭੋਜਨ ਅਤੇ ਪੇਅ ਮੀਨੂ ਸਫ਼ੇ ਤਿਆਰ ਕਰ ਸਕਦਾ ਹੈ। ਮੀਨੂਆਂ ਵਿੱਚ ਕੀਮਤਾਂ, ਵੇਰਵੇ, ਆਹਾਰ ਸੰਬੰਧੀ ਜਾਣਕਾਰੀ, ਅਤੇ ਐਲਰਜਨ ਚੇਤਾਵਨੀਆਂ ਸ਼ਾਮਲ ਹੁੰਦੀਆਂ ਹਨ।

ਵਿਸਤ੍ਰਿਤ ਰਹਿਨੁਮਾਈ ਲਈ [ਹਾਸਪਿਟਾਲਿਟੀ ਮੀਨੂ](./theme-builder/hospitality-menus.md) ਵੇਖੋ।

| **Google Calendar Credentials** | ਕੈਲੰਡਰ ਪਹੁੰਚ ਕਨਫਿਗਰ ਕਰੋ ਤਾਂ ਜੋ ਏਜੰਟ ਕੈਲੰਡਰ ਅਤੇ ਇਵੈਂਟ ਪੜ੍ਹ ਸਕੇ |

- **ਪਲੱਗਇਨ ਕਤਾਰ ਕਾਰਵਾਈ ਲਿੰਕ** — ਸਪਸ਼ਟਤਾ ਲਈ ਠੀਕ ਕੀਤੇ ਅਤੇ ਨਾਮ ਬਦਲੇ ਗਏ
