# Superdav AI Agent

Superdav AI Agent ਇੱਕ ਉੱਨਤ WordPress AI ਸਹਾਇਕ ਹੈ ਜੋ Gratis AI Agent ਨੂੰ تھیਮ-ਜਾਗਰੂਕ ਹੁਨਰ (theme-aware skills), ਸੁਧਰੀਆਂ ਹੋਈਆਂ ਇੰਟਰਨੈਟ ਸਰਚ ਸਮਰੱਥਾਵਾਂ, ਅਤੇ ਬਿਹਤਰ ਸਾਈਟ ਬਿਲਡਰ ਫੀਚਰਾਂ ਨਾਲ ਵਧਾਉਂਦਾ ਹੈ। Version 1.16.0 ਵਿੱਚ ਕਸਟਮ ਲੋਗੋ ਜਨਰੇਸ਼ਨ, ਪੈਲੇਟ ਕੰਟ੍ਰਾਸਟ ਵੈਲੀਡੇਸ਼ਨ, ਡਿਸਕਵਰੀ ਇੰਟਰਵਿਊ ਵਿੱਚ ਫੋਟੋ ਅਪਲੋਡ, ਸਟ੍ਰਕਚਰਡ ਹੋਸਪਿਟੈਲਿਟੀ ਮੀਨੂ ਜਨਰੇਸ਼ਨ, ਅਤੇ Theme Builder ਵਿੱਚ ਡੈਸਕਟਾਪ/ਮੋਬਾਈਲ ਪ੍ਰੀਵਿਊ ਰੈਂਡਰਿੰਗ ਸ਼ਾਮਲ ਕੀਤੀਆਂ ਗਈਆਂ ਹਨ।

## ਮੁੱਖ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ (Key Features)

- **تھیਮ-ਜਾਗਰੂਕ ਹੁਨਰ (Theme-aware skills)** — ਤੁਹਾਡੇ ਸਰਗਰਮ WordPress تھیਮ ਅਤੇ ਸਥਾਪਿਤ ਪਲੱਗਇਨਾਂ ਦੇ ਆਧਾਰ 'ਤੇ ਸਹਾਇਤਾ ਅਤੇ ਸਮਰੱਥਾਵਾਂ ਨੂੰ ਆਪਣੇ ਆਪ ਅਨੁਕੂਲ ਬਣਾਉਂਦਾ ਹੈ।
- **ਬਹੁਤ ਗਿਣਤੀ ਇੰਟਰਨੈਟ ਸਰਚ ਪ੍ਰੋਵਾਈਡਰ** — ਰੰਗ੍ਰਿਸ਼ ਸਰਚ ਰਿਜ਼ਲਟਸ ਅਤੇ ਉੱਨਤ ਖੋਜ ਸਮਰੱਥਾਵਾਂ ਲਈ Brave Search ਅਤੇ Tavily ਵਿੱਚੋਂ ਚੋਣ ਕਰੋ।
- **ਸਾਈਟ ਬਿਲਡਰ ਕੰਟੈਕਟ ਫਾਰਮ** — ਆਪਣੇ ਸਥਾਪਿਤ ਫਾਰਮ ਪਲੱਗਇਨ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਚੈਟ ਇੰਟਰਫੇਸ ਤੋਂ ਸਿੱਧੇ ਤੌਰ 'ਤੇ ਪੂਰੀ ਤਰ੍ਹਾਂ ਕੰਮ ਕਰਨ ਵਾਲੇ ਕੰਟੈਕਟ ਫਾਰਮ ਬਣਾਓ।
- **ਸੁਧਰੀ ਸਾਈਟ ਬਿਲਡਰ** — ਬਿਹਤਰ ਪਲੱਗਇਨ ਡਿਸਕਵਰੀ ਅਤੇ ਐਰਰ ਰికਵਰੀ ਨਾਲ ਸੁਧਰੀਆਂ ਬਹੁ-ਪੜਾਅ ਵਾਲੀਆਂ ਸਾਈਟ ਬਿਲਡ ਪਲਾਨਾਂ।
- **ਇੰਬिल्ਟ تھیਮ ਹੁਨਰ** — Block Themes, Classic Themes, Kadence Blocks, ਅਤੇ Kadence Theme ਲਈ ਵਿਸ਼ੇਸ਼ ਸਹਾਇਤਾ।
- **ਸੁਧਰੀ WooCommerce ਇੰਟੀਗ੍ਰੇਸ਼ਨ** — ਬਿਹਤਰ ਭਰੋਸੇਯੋਗਤਾ ਅਤੇ ਅਨੁਕੂਲਤਾ ਲਈ native WooCommerce APIs ਦੀ ਵਰਤੋਂ ਕਰਨ ਲਈ ਸੁਧਾਰਿਆ ਗਿਆ।
- **ਆਟੋਮੈਟਿਕ ਪ੍ਰੋਵਾਈਡਰ ਡਿਟੈਕਸ਼ਨ** — ਜਦੋਂ ਪਲੱਗਇਨਾਂ ਜਾਂ تھیਮਾਂ ਨੂੰ ਐਕਟੀਵੇਟ ਜਾਂ ਡੀਐਕਟੀਵੇਟ ਕੀਤਾ ਜਾਂਦਾ ਹੈ ਤਾਂ ਉਪਲਬਧ ਪਲੱਗਇਨਾਂ ਅਤੇ تھیਮਾਂ ਦਾ ਆਪਣੇ ਆਪ ਨਿਰੀਖਣ ਅਤੇ ਤਾਜ਼ਾ ਕਰਦਾ ਹੈ।
- **ਕਸਟਮ ਲੋਗੋ ਜਨਰੇਸ਼ਨ** — ਆਟੋਮੈਟਿਕ ਨਾਮ-ਸਪੇਸ ਸੇਫ ਸੈਨੀਟਾਈਜ਼ੇਸ਼ਨ ਨਾਲ ਕਸਟਮ SVG ਲੋਗੋ ਬਣਾਓ ਅਤੇ ਐਮਬੈਡ ਕਰੋ।
- **ਪੈਲੇਟ ਕੰਟ੍ਰਾਸਟ ਵੈਲੀਡੇਸ਼ਨ** — ਆਪਣੇ تھیਮ 'ਤੇ ਲਾਗੂ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ WCAG ਪਹੁੰਚਯੋਗਤਾ ਪਾਲਣਾ ਲਈ ਰੰਗ ਜੋੜਿਆਂ ਦੀ ਜਾਂਚ ਕਰੋ।
- **ਫੋਟੋ-ਐਨਹਾਂਸਡ ਡਿਸਕਵਰੀ** — ਡਿਜ਼ਾਈਨ ਦਿਸ਼ਾ ਨੂੰ ਪ੍ਰਭਾਵਿਤ ਕਰਨ ਲਈ Theme Builder ਡਿਸਕਵਰੀ ਇੰਟਰਵਿਊ ਦੌਰਾਨ ਫੋਟੋਆਂ ਅਪਲੋਡ ਕਰੋ।
- **ਹੋਸਪਿਟੈਲਿਟੀ ਮੀਨੂ** — ਰੈਸਟੋਰੈਂਟਾਂ, ਕੈਫੇ, ਅਤੇ ਹੋਸਪਿਟੈਲਿਟੀ ਕਾਰੋਬਾਰਾਂ ਲਈ ਸਟ੍ਰਕਚਰਡ ਫੂਡ ਅਤੇ ਬੇਵਰੇਜ ਮੀਨੂ ਪੇਜ ਬਣਾਓ।
- **ਡੈਸਕਟਾਪ ਅਤੇ ਮੋਬਾਈਲ ਪ੍ਰੀਵਿਊ** — ਡਿਜ਼ਾਈਨ- ਦਿਸ਼ਾ ਚੋਣ ਦੌਰਾਨ ਡੈਸਕਟਾਪ ਅਤੇ ਮੋਬਾਈਲ ਡਿਵਾਈਸਾਂ 'ਤੇ ਆਪਣੇ ਡਿਜ਼ਾਈਨ ਦਾ ਪ੍ਰੀਵਿਊ ਕਰੋ।
- **Tier 1 ਸਾਈਟ ਸਕਰੇਪਿੰਗ** — sd-ai-agent/site-scrape ਹੁਣ ਡਿਫਾਲਟ ਤੌਰ 'ਤੇ ਮੁਕਾਬਲੇਬਾਜ਼ ਅਤੇ ਪ੍ਰੇਰਨਾ ਦੇ ਵੈੱਬਸਾਈਟਾਂ ਦਾ ਵਿਸ਼ਲੇਸ਼ਣ ਕਰਨ ਲਈ ਉਪਲਬਧ ਹੈ।

## ਲੋੜਾਂ (Requirements)

- WordPress 6.0 ਜਾਂ ਇਸ ਤੋਂ ਵੱਧ
- PHP 8.1 ਜਾਂ ਇਸ ਤੋਂ ਵੱਧ
- WordPress Multisite (network-activated) ਜਾਂ single-site installation
- ਇੱਕ ਸਰਗਰਮ AI ਪ੍ਰੋਵਾਈਡਰ ਕਨੈਕਟਰ ਪਲੱਗਇਨ (Anthropic, OpenAI, ਜਾਂ ਅਨੁਕੂਲ)
- Gratis AI Agent v1.9.0 ਜਾਂ ਇਸ ਤੋਂ ਵੱਧ (Superdav AI Agent, Gratis AI Agent ਦਾ ਵਿਸਤਾਰ ਕਰਦਾ ਹੈ)

## ਸਥਾਪਨਾ (Installation)

1. ਪਲੱਗਇਨ ਫਾਈਲਾਂ ਨੂੰ `/wp-content/plugins/superdav-ai-agent` 'ਤੇ ਅਪਲੋਡ ਕਰੋ।
2. WordPress ਵਿੱਚ **Plugins** ਸਕ੍ਰੀਨ ਰਾਹੀਂ ਪਲੱਗਇਨ ਨੂੰ ਐਕਟੀਵੇਟ ਕਰੋ।
3. ਯਕੀਨੀ ਬਣਾਓ ਕਿ Gratis AI Agent ਵੀ ਐਕਟੀਵੇਟ ਹੈ।
4. **Gratis AI Agent → Settings** 'ਤੇ ਜਾਓ ਅਤੇ ਆਪਣੇ ਇੰਟਰਨੈਟ ਸਰਚ ਪ੍ਰੋਵਾਈਡਰ ਨੂੰ ਕੌਂਫਿਗਰ ਕਰੋ (ਵਿਕਲਪਿਕ)।

## ਜਲਦੀ ਸ਼ੁਰੂਆਤ (Quick Start)

ਐਕਟੀਵੇਟ ਹੋਣ ਤੋਂ ਬਾਅਦ, Superdav AI Agent ਆਪਣੇ ਆਪ ਤੁਹਾਡੀ Gratis AI Agent installation ਨੂੰ تھیਮ-ਜਾਗਰੂਕ ਹੁਨਰ ਅਤੇ ਵਾਧੂ ਸਮਰੱਥਾਵਾਂ ਨਾਲ ਸੁਧਾਰਦਾ ਹੈ। WordPress ਐਡਮਿਨ ਸਾਈਬਾਰ ਵਿੱਚ **Gratis AI Agent** ਪੈਨਲ ਖੋਲ੍ਹੋ ਅਤੇ ਨਵੀਆਂ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ ਦੀ ਵਰਤੋਂ ਕਰਨਾ ਸ਼ੁਰੂ ਕਰੋ।

**ਉਦਾਹਰਨ ਨਿਰਦੇਸ਼:**

```
Add a contact form to the Contact page.
```

```
Search the web for the latest WordPress security best practices.
```

```
Create a hero section using the active theme's block patterns.
```

## تھیਮ-ਜਾਗਰੂਕ ਹੁਨਰ (Theme-Aware Skills)

Superdav AI Agent ਆਪਣੇ ਸਰਗਰਮ تھیਮ ਅਤੇ ਸਥਾਪਿਤ ਪਲੱਗਇਨਾਂ ਦਾ ਆਪਣੇ ਆਪ ਨਿਰੀਖਣ ਕਰਦਾ ਹੈ, ਅਤੇ ਸਹਾਇਕ ਦੇ context ਵਿੱਚ ਸਹੀ تھیਮ-ਜਾਗਰੂਕ ਹੁਨਰ ਲੋਡ ਕਰਦਾ ਹੈ। ਉਪਲਬਧ ਹੁਨਰਾਂ ਵਿੱਚ ਸ਼ਾਮਲ ਹਨ:

| Skill | Applies To |
|---|---|
| **Block Themes** | Themes using the WordPress block-based (Full Site Editing) architecture |
| **Classic Themes** | Traditional WordPress themes using PHP templates and the classic editor |
| **Kadence Blocks** | Sites with the Kadence Blocks plugin active |
| **Kadence Theme** | Sites with the Kadence theme as the active theme |

ਹਰੇਕ ਹੁਨਰ 'ਤੇ ਵਿਸਤ੍ਰਿਤ ਗਾਈਡੈਂਸ ਲਈ [Theme-Aware Skills](../../user-guide/configuration/theme-aware-skills) ਵੇਖੋ।

## ਇੰਟਰਨੈਟ ਸਰਚ ਪ੍ਰੋਵਾਈਡਰ (Internet Search Providers)

Superdav AI Agent v1.10.0 ਨੇ ਇੰਟਰਨੈਟ ਸਰਚ ਕਵਰੀਆਂ ਲਈ Brave Search ਦੇ ਬਦਲੇ **Tavily** ਨੂੰ ਸ਼ਾਮਲ ਕੀਤਾ ਹੈ। ਦੋਵੇਂ ਪ੍ਰੋਵਾਈਡਰ ਰੀਅਲ-ਟਾਈਮ ਸਰਚ ਰਿਜ਼ਲਟਸ ਪੇਸ਼ ਕਰਦੇ ਹਨ ਅਤੇ ਇਹਨਾਂ ਨੂੰ **Gratis AI Agent → Settings → Advanced** ਵਿੱਚ ਕੌਂਫਿਗਰ ਕੀਤਾ ਜਾ ਸਕਦਾ ਹੈ।

| Provider | Features | Setup |
|---|---|---|
| **Brave Search** | ਤੇਜ਼, ਗੋਪਨੀਯਤਾ-ਕੇਂਦਰਿਤ ਸਰਚ ਰਿਜ਼ਲਟਸ | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | ਰੰਗ੍ਰਿਸ਼ ਸਰਚ ਰਿਜ਼ਲਟਸ, ਉੱਨਤ ਖੋਜ ਸਮਰੱਥਾਵਾਂ | [Tavily API](https://tavily.com/) |

ਕੌਂਫਿਗਰੇਸ਼ਨ ਨਿਰਦੇਸ਼ਾਂ ਲਈ [Internet Search](../../user-guide/configuration/internet-search) ਵੇਖੋ।

## ਸਾਈਟ ਬਿਲਡਰ ਕੰਟੈਕਟ ਫਾਰਮ (Site Builder Contact Forms)

Site Builder agent ਹੁਣ ਚੈਟ ਇੰਟਰਫੇਸ ਤੋਂ ਸਿੱਧੇ ਕੰਟੈਕਟ ਫਾਰਮ ਬਣਾ ਸਕਦਾ ਹੈ। ਸਿਰਫ਼ Site Builder ਨੂੰ ਕੰਟੈਕਟ ਫਾਰਮ ਬਣਾਉਣ ਲਈ ਕਹੋ, ਅਤੇ ਇਹ ਤੁਹਾਡੇ ਸਥਾਪਿਤ ਫਾਰਮ ਪਲੱਗਇਨ (Contact Form 7, WPForms, Fluent Forms, ਜਾਂ Gravity Forms) ਦਾ ਆਪਣੇ ਆਪ ਨਿਰੀਖਣ ਕਰਕੇ ਵਰਤਣ ਲਈ ਤਿਆਰ ਫਾਰਮ ਬਣਾ ਦੇਵੇਗਾ।

ਵਿਸਤ੍ਰਿਤ ਗਾਈਡੈਂਸ ਲਈ [Site Builder Contact Forms](../../user-guide/administration/site-builder-contact-forms) ਵੇਖੋ।

## تھیਮ ਬਿਲਡਰ ਫੀਚਰਾਂ (Theme Builder Features)

Superdav AI Agent v1.16.0 ਵਿੱਚ Theme Builder ਸ਼ਕਤੀਸ਼ਾਲੀ ਨਵੀਆਂ ਸਮਰੱਥਾਵਾਂ ਸ਼ਾਮਲ ਹਨ:

### ਫੋਟੋ ਅਪਲੋਡ ਨਾਲ ਡਿਸਕਵਰੀ ਇੰਟਰਵਿਊ (Discovery Interview with Photo Upload)

Theme Builder ਡਿਸਕਵਰੀ ਇੰਟਰਵਿਊ ਵਿੱਚ ਹੁਣ ਫੋਟੋ ਅਪਲੋਡ ਕਦਮ ਸ਼ਾਮਲ ਹੈ ਜਿੱਥੇ ਤੁਸੀਂ ਵਿਜ਼ੂਅਲ ਇਨਸਪੀਰੇਸ਼ਨ ਪ੍ਰਦਾਨ ਕਰ ਸਕਦੇ ਹੋ। Theme Builder ਨੂੰ ਤੁਹਾਡੀ ਡਿਜ਼ਾਈਨ ਦਿਸ਼ਾ ਨੂੰ ਸਮਝਣ ਵਿੱਚ ਮਦਦ ਕਰਨ ਲਈ ਆਪਣੇ ਬ੍ਰਾਂਡ, ਮੁਕਾਬਲੇਬਾਜ਼ ਸਾਈਟਾਂ, ਜਾਂ ਡਿਜ਼ਾਈਨ ਇਨਸਪੀਰੇਸ਼ਨ ਦੀਆਂ ਫੋਟੋਆਂ ਅਪਲੋਡ ਕਰੋ।

ਵਿਸਤ੍ਰਿਤ ਗਾਈਡੈਂਸ ਲਈ [Discovery Interview](./theme-builder/discovery-interview.md) ਵੇਖੋ।

### ਡੈਸਕਟਾਪ ਅਤੇ ਮੋਬਾਈਲ ਪ੍ਰੀਵਿਊ ਨਾਲ ਡਿਜ਼ਾਈਨ ਦਿਸ਼ਾ (Design Direction with Desktop and Mobile Previews)

Theme Builder ਤੁਹਾਡਾ ਪੂਰਾ تھیਮ ਬਣਾਉਣ ਤੋਂ ਪਹਿਲਾਂ ਡੈਸਕਟਾਪ ਅਤੇ ਮੋਬਾਈਲ ਦੋਵਾਂ ਡਿਵਾਈਸਾਂ 'ਤੇ ਆਪਣੇ ਡਿਜ਼ਾਈਨ ਦੀ ਸਮੀਖਿਆ ਕਰੋ। ਕਈ ਰਿਸਪਾਂਸਿਵ ਬ੍ਰੇਕਪੁਆਇੰਟਾਂ 'ਤੇ ਰੀਅਲ-ਟਾਈਮ ਪ੍ਰੀਵਿਊ ਵੇਖੋ।

ਵਿਸਤ੍ਰਿਤ ਗਾਈਡੈਂਸ ਲਈ [Design Direction](./theme-builder/design-direction.md) ਵੇਖੋ।

### ਹੋਸਪਿਟੈਲਿਟੀ ਮੀਨੂ (Hospitality Menus)

Theme Builder ਹੁਣ ਰੈਸਟੋਰੈਂਟਾਂ, ਕੈਫੇ, ਬਾਰਾਂ, ਅਤੇ ਹੋਰ ਹੋਸਪਿਟੈਲਿਟੀ ਕਾਰੋਬਾਰਾਂ ਲਈ ਸਟ੍ਰਕਚਰਡ ਫੂਡ ਅਤੇ ਬੇਵਰੇਜ ਮੀਨੂ ਪੇਜ ਬਣਾ ਸਕਦਾ ਹੈ। ਮੀਨੂ ਵਿੱਚ ਕੀਮਤਾਂ, ਵੇਰਵੇ, ਖੁਰਾਕ ਸੰਬੰਧੀ ਜਾਣਕਾਰੀ, ਅਤੇ ਐਲਰਜਨ ਚੇਤਾਵਨੀਆਂ ਸ਼ਾਮਲ ਹੁੰਦੀਆਂ ਹਨ।

ਵਿਸਤ੍ਰਿਤ ਗਾਈਡੈਂਸ ਲਈ [Hospitality Menus](./theme-builder/hospitality-menus.md) ਵੇਖੋ।

### ਨਵੀਆਂ ਸਮਰੱਥਾਵਾਂ (New Abilities)

- **[Generate Logo SVG](./abilities/generate-logo-svg.md)** — ਆਟੋਮੈਟਿਕ ਸੈਨੀਟਾਈਜ਼ੇਸ਼ਨ ਨਾਲ ਕਸਟਮ SVG ਲੋਗੋ ਬਣਾਓ।
- **[Validate Palette Contrast](./abilities/validate-palette-contrast.md)** — WCAG ਪਾਲਣਾ ਲਈ ਰੰਗ ਜੋੜਿਆਂ ਦੀ ਜਾਂਚ ਕਰੋ।
- **[Create Menu](./abilities/create-menu.md)** — ਵੱਖ-ਵੱਖ ਨੈਵੀਗੇਸ਼ਨ ਲੇਬਲਾਂ ਨਾਲ ਨੈਵੀਗੇਸ਼ਨ ਮੀਨੂ ਬਣਾਓ।

ਉਪਲਬਧ ਟੂਲ ਅਤੇ Tier 1 ਸਾਈਟ-ਸਕਰੇਪ ਟੂਲ ਬਾਰੇ ਜਾਣਕਾਰੀ ਲਈ [Tools and Configuration](./theme-builder/tools.md) ਵੇਖੋ।

## ਕੌਂਫਿਗਰੇਸ਼ਨ (Configuration)

ਸਾਰੀਆਂ ਸੈਟਿੰਗਾਂ WordPress ਐਡਮਿਨ ਵਿੱਚ **Gratis AI Agent → Settings** ਹੇਠਾਂ ਹਨ। Superdav AI Agent ਹੇਠ ਲਿਖੇ ਕੌਂਫਿਗਰੇਸ਼ਨ ਵਿਕਲਪ ਜੋੜਦਾ ਹੈ:
