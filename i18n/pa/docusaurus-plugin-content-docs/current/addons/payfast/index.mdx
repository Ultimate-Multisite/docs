# PayFast

Multisite Ultimate: Payfast Gateway

ਇਹ Payfast - ਜੋ ਕਿ South Africa ਵਿੱਚ ਇੱਕ ਤੇਜ਼ ਅਤੇ ਸੁਰੱਖਿਅਤ ਆਨਲਾਈਨ ਭੁਗਤਾਨ ਗੇਟਵੇ ਹੈ - ਨੂੰ Multisite Ultimate ਦੇ ਭੁਗਤਾਨ ਵਿਕਲਪ ਵਜੋਂ ਜੋੜਦਾ ਹੈ।

## ਇੰਸਟਾਲੇਸ਼ਨ

1. 'ultimate-multisite-payfast' ਨੂੰ '/wp-content/plugins/' ਡਾਇਰੈਕਟਰੀ ਵਿੱਚ ਅਪਲੋਡ ਕਰੋ।
2. WordPress ਵਿੱਚ 'Plugins' ਮੀਨੂ ਰਾਹੀਂ ਪਲੱਗਇਨ ਐਕਟਿਵੇਟ ਕਰੋ।
3. ਪਲੱਗਇਨ ਸੈਟਅੱਪ ਕਰਨ ਲਈ ਕਦਮ-ਦਰ-ਕਦਮ ਵਿਜ਼ਾਰਡ (Wizard) ਦੀ ਪਾਲਣਾ ਕਰੋ।

## ਵਰਤੋਂ ਗਾਈਡ

PayFast South Africa ਵਿੱਚ ਕਾਰੋਬਾਰਾਂ ਲਈ ਇੱਕ ਪ੍ਰਮੁੱਖ ਆਨਲਾਈਨ ਭੁਗਤਾਨ ਪ੍ਰੋਸੈਸਰ ਹੈ। ਇਹ ਆਨਲਾਈਨ ਵਪਾਰੀਆਂ ਨੂੰ ਲੈਣ-ਦੇਣ ਨੂੰ ਸੁਚਾਰੂ ਅਤੇ ਸੁਰੱਖਿਅਤ ਬਣਾ ਕੇ ਵਧਣ ਵਿੱਚ ਮਦਦ ਕਰਨ ਲਈ ਸਮਰਪਿਤ ਹੈ। Ultimate Multisite ਯੂਜ਼ਰ ਦੀ ਮੰਗ ਅਨੁਸਾਰ, ਅਸੀਂ ਇੱਕ ਐਡ-ਆਨ ਬਣਾਉਣ ਦਾ ਫੈਸਲਾ ਕੀਤਾ ਹੈ ਜੋ PayFast ਨਾਲ ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਨੂੰ ਸੰਭਵ ਬਣਾਏਗਾ। ਇਹ ਲੇਖ ਦਿਖਾਉਂਦਾ ਹੈ ਕਿ ਆਪਣੇ Ultimate Multisite ਨੈਟਵਰਕ ਵਿੱਚ ਭੁਗਤਾਨ ਪ੍ਰਾਪਤ ਕਰਨ ਲਈ PayFast ਨੂੰ ਆਪਣੇ ਭੁਗਤਾਨ ਗੇਟਵੇ ਵਜੋਂ ਕਿਵੇਂ ਸੈਟਅੱਪ ਕਰੀਏ।

ਪਹਿਲਾ ਕਦਮ ਐਡ-ਆਨ ਇੰਸਟਾਲ ਕਰਨਾ ਹੈ।

# Payfast ਐਡ-ਆਨ ਕਿਵੇਂ ਇੰਸਟਾਲ ਕਰੀਏ:

ਐਡ-ਆਨਜ਼ ਪੇਜ 'ਤੇ ਜਾਓ ਅਤੇ Ultimate Multisite: Payfast Gateway ਐਡ-ਆਨ ਚੁਣੋ।

![Add-ons page](/img/admin/addons-page.png)

ਐਡ-ਆਨ ਵਿੰਡੋ ਖੁੱਲ੍ਹ ਜਾਵੇਗੀ। 'Install Now' 'ਤੇ ਕਲਿੱਕ ਕਰੋ।

![Add-ons page with install option](/img/admin/addons-page.png)

ਇੰਸਟਾਲੇਸ਼ਨ ਹੋਣ ਤੋਂ ਬਾਅਦ, ਤੁਹਾਨੂੰ ਪਲੱਗਇਨਜ਼ ਪੇਜ 'ਤੇ ਰੀਡਾਇਰੈਕਟ ਕੀਤਾ ਜਾਵੇਗਾ। ਇੱਥੇ, ਸਿਰਫ਼ 'Network Activate' 'ਤੇ ਕਲਿੱਕ ਕਰੋ ਅਤੇ _Payfast_ ਐਡ-ਆਨ ਤੁਹਾਡੇ ਨੈਟਵਰਕ 'ਤੇ ਐਕਟਿਵੇਟ ਹੋ ਜਾਵੇਗਾ।

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# PayFast ਸੈਟਅੱਪ ਕਰਨਾ

ਐਡ-ਆਨ ਇੰਸਟਾਲ ਕਰਨ ਤੋਂ ਬਾਅਦ, ਤੁਸੀਂ ਹੁਣ Ultimate Multisite ਸੈਟਿੰਗਜ਼ 'ਤੇ ਜਾ ਕੇ PayFast ਨੂੰ ਕੰフィਗਰ ਕਰਨਾ ਸ਼ੁਰੂ ਕਰ ਸਕਦੇ ਹੋ।

**Ultimate Multisite settings** ਪੇਜ ਦੇ ਹੇਠਾਂ, ਤੁਹਾਨੂੰ "Payments" ਟੈਬ ਮਿਲੇਗਾ। ਇਸ ਟੈਬ ਦੇ ਹੇਠਾਂ, ਤੁਹਾਨੂੰ ਇਸ ਭੁਗਤਾਨ ਗੇਟਵੇ ਨੂੰ ਐਨੇਬਲ ਜਾਂ ਡਿਸੇਬਲ ਕਰਨ ਲਈ ਇੱਕ ਟੋਗਲ ਬਟਨ ਦੇ ਨਾਲ "**Payfast** " ਮਿਲੇਗਾ।

![Payment gateway settings](/img/config/settings-payment-gateways.png)ਅੱਗੇ ਵਧੋ ਅਤੇ ਇਸਨੂੰ ਐਨੇਬਲ ਕਰਨ ਲਈ ਟੋਗਲ ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ। ਫਿਰ ਇਹ ਬਾਕਸ ਖੁੱਲ੍ਹ ਜਾਵੇਗਾ ਅਤੇ ਵੱਖ-ਵੱਖ ਫੀਲਡ ਦਿਖਾਏਗਾ ਜਿਨ੍ਹਾਂ ਨੂੰ ਤੁਹਾਨੂੰ ਭਰਨਾ ਪਵੇਗਾ, ਜਿਵੇਂ ਕਿ **Mechant ID, Merchant Key ਅਤੇ PassPhrase।** ਤੁਸੀਂ ਇਹ ਜਾਣਕਾਰੀ ਆਪਣੇ **Payfast** account ਤੋਂ ਲੈ ਸਕਦੇ ਹੋ।

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

ਅਗਲਾ ਕਦਮ ਉਹ ਜਾਣਕਾਰੀ ਪ੍ਰਾਪਤ ਕਰਨ ਲਈ ਆਪਣੇ PayFast account ਵਿੱਚ ਲੌਗਇਨ ਕਰਨਾ ਹੈ।

ਲੌਗਇਨ ਕਰਨ 'ਤੇ, ਤੁਸੀਂ ਤੁਰੰਤ ਆਪਣੇ **Payfast** account dashboard ਤੋਂ ਲੋੜੀਂਦੇ ਕ੍ਰੈਡੈਂਸ਼ੀਅਲ ਦੇਖ ਸਕਦੇ ਹੋ।

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->ਤੁਸੀਂ ਸਿਰਫ਼ ਕ੍ਰੈਡੈਂਸ਼ੀਅਲ ਨੂੰ ਕਾਪੀ ਅਤੇ ਪੇਸਟ ਕਰਕੇ ਆਪਣੀਆਂ Ultimate Multisite ਸੈਟਿੰਗਜ਼ ਵਿੱਚ ਵੇਰਵੇ ਭਰ ਸਕਦੇ ਹੋ। ਫਿਰ ਬਦਲਾਅ ਸੇਵ ਕਰਨ ਲਈ **"Save settings** " ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

ਬੱਸ, ਤੁਹਾਡਾ PayFast ਭੁਗਤਾਨ ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਪੂਰਾ ਹੋ ਗਿਆ ਹੈ!
