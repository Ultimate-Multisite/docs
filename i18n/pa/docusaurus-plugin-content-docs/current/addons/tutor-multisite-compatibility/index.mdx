# Tutor Multisite Compatibility {#tutor-multisite-compatibility}

ਇੱਕ WordPress plugin ਜੋ Tutor LMS ਲਈ multisite environments ਵਿੱਚ compatibility fixes ਜੋੜਦਾ ਹੈ।

## Description {#description}

ਇਹ plugin WordPress multisite networks 'ਤੇ Tutor LMS ਚਲਾਉਣ ਲਈ ਜ਼ਰੂਰੀ compatibility fixes ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ। ਇਹ user enrollment, instructor management, course limitations, ਅਤੇ ਹੋਰ multisite-specific functionality ਨੂੰ ਸੰਭਾਲਦਾ ਹੈ।

## Features {#features}

- **Database Table Management**: Multisite foreign key constraints ਕਾਰਨ ਗਵਾਹੀਆਂ (missing) Tutor LMS tables ਨੂੰ ਆਪਣੇ ਆਪ ਲੱਭਦਾ ਅਤੇ ਬਣਾਉਂਦਾ ਹੈ।
- **Multisite User Enrollment**: network sites 'ਤੇ user enrollment manage ਕਰਨ ਲਈ custom AJAX handlers।
- **Instructor Role Management**: multisite environments ਵਿੱਚ automatic instructor role assignment ਅਤੇ management।
- **Course Limitations**: site plans ਦੇ ਆਧਾਰ 'ਤੇ course ਅਤੇ funnel creation limits manage ਕਰਨ ਲਈ Ultimate Multisite ਨਾਲ integration।
- **Custom Admin Views**: multisite-aware instructor list ਅਤੇ management interface।
- **Admin Tools**: network ਅਤੇ site administrators ਲਈ database management interface।

## Requirements {#requirements}

- WordPress 5.0+
- PHP 7.4+
- Tutor LMS plugin (automatically enforced via `Requires Plugins` header)
- WordPress Multisite installation
- Ultimate Multisite plugin (course limitations functionality ਲਈ)

## Installation {#installation}

1. plugin files ਨੂੰ `/wp-content/plugins/tutor-multisite-compatibility/` 'ਤੇ upload ਕਰੋ।
2. ਯਕੀਨੀ ਬਣਾਓ ਕਿ Tutor LMS install ਅਤੇ activate ਹੈ।
3. WordPress ਵਿੱਚ 'Plugins' menu ਰਾਹੀਂ plugin activate ਕਰੋ।
4. plugin ਆਪਣੇ ਆਪ Tutor LMS ਨਾਲ integrate ਹੋ ਜਾਵੇਗਾ।

- Course product ID display

## Database Management {#database-management}

### The Problem {#the-problem}

WordPress multisite environments ਵਿੱਚ, Tutor LMS table creation, users table ਨਾਲ foreign key constraints ਕਾਰਨ ਫੇਲ੍ਹ ਹੋ ਜਾਂਦੀ ਹੈ। ਇਹ ਇਸ ਲਈ ਹੁੰਦਾ ਹੈ ਕਿਉਂਕਿ:

- Site-specific tables `wp_2_tutor_carts` ਵਰਗੇ prefixes ਦੀ ਵਰਤੋਂ ਕਰਦੇ ਹਨ।
- Foreign keys `wp_2_users` ਨੂੰ reference ਕਰਦੇ ਹਨ (ਜੋ ਮੌਜੂਦ ਨਹੀਂ ਹੈ)।
- ਅਸਲ users table `wp_users` ਹੈ (ਜੋ ਸਾਰੇ network ਵਿੱਚ ਸਾਂਝਾ ਹੈ)।

### The Solution {#the-solution}

ਇਹ plugin ਆਪਣੇ ਆਪ:

1. **Missing Tables Detect ਕਰਦਾ ਹੈ**: foreign key constraints ਕਾਰਨ ਨਾ ਬਣ ਸਕਣ ਵਾਲੀਆਂ tables ਦੀ ਜਾਂਚ ਕਰਦਾ ਹੈ।
2. **Correct References ਨਾਲ Tables ਬਣਾਉਂਦਾ ਹੈ**: ਸਹੀ user table name (`wp_users` in multisite) ਦੀ ਵਰਤੋਂ ਕਰਦਾ ਹੈ।
3. **Multiple Scenarios ਨੂੰ Handle ਕਰਦਾ ਹੈ**:
   - Plugin activation/reactivation
   - Network ਵਿੱਚ ਨਵਾਂ site creation
   - Admin interface ਰਾਹੀਂ manual table checking

### Affected Tables {#affected-tables}

- `tutor_coupon_usages` - users table ਨੂੰ foreign key ਨਾਲ ਰੱਖਦਾ ਹੈ
- `tutor_carts` - users table ਨੂੰ foreign key ਨਾਲ ਰੱਖਦਾ ਹੈ
- `tutor_notification_preferences` - users table ਨੂੰ foreign key ਨਾਲ ਰੱਖਣ ਵਾਲੀ TutorPro table

### Admin Interface {#admin-interface}

database management tools ਤੱਕ ਪਹੁੰਚ ਕਰੋ:

- **Network Admin**: Settings → Tutor Multisite DB
- **Site Admin**: Tools → Tutor Multisite DB

ਇਹ interface ਤੁਹਾਨੂੰ ਇਹ ਕਰਨ ਦੀ ਇਜਾਜ਼ਤ ਦਿੰਦਾ ਹੈ:
- ਸਾਰੇ sites 'ਤੇ missing tables ਦੇਖਣਾ
- table creation ਨੂੰ manually trigger ਕਰਨਾ
- database configuration ਦੀ ਜਾਂਚ ਕਰਨਾ
- detailed status information ਦੇਖਣਾ
