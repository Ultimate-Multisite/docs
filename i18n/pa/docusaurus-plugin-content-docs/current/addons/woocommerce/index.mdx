# WooCommerce Integration

WooCommerce Integration ਬਿਲਕੁਲ ਉਸੇ ਤਰੀਕੇ ਨਾਲ ਕੰਮ ਕਰਦਾ ਹੈ ਜਿਵੇਂ ਕਿ ਸਾਡਾ Manual Gateway ਕਰਦਾ ਹੈ। ਇਹ ਤੁਹਾਡੇ ਉਪਭੋਗਤਾਵਾਂ (users) ਨੂੰ ਬਹੁਤ ਸਾਰੀ ਲਚਕਤਾ (flexibility) ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ, ਜਿਸ ਨਾਲ ਉਹ ਤੁਹਾਡੇ WooCommerce ਇੰਸਟਾਲ 'ਤੇ ਮੌਜੂਦ ਕਿਸੇ ਵੀ ਭੁਗਤਾਨ ਫਾਰਮ ਨੂੰ ਚੁਣ ਸਕਦੇ ਹਨ। ਅਤੇ ਇਹ ਬਹੁਤ ਸੌਖਾ ਹੈ: ਜਦੋਂ ਉਪਭੋਗਤਾ ਦਾ ਬਿਲਿੰਗ ਕਾਲ (billing period) ਖਤਮ ਹੋ ਜਾਂਦਾ ਹੈ, ਤਾਂ ਉਪਭੋਗਤਾ ਦੀ ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਰੋਕ (put on hold) ਦਿੱਤੀ ਜਾਂਦੀ ਹੈ। ਇਸੇ ਸਮੇਂ, ਨੈਟਵਰਕ ਦੀ ਮੁੱਖ ਸਾਈਟ ਦੇ WooCommerce ਇੰਸਟਾਲ 'ਤੇ ਇੱਕ ਆਰਡਰ ਬਣਾਇਆ ਜਾਵੇਗਾ। ਫਿਰ ਉਪਭੋਗਤਾ ਨੂੰ ਉਨ੍ਹਾਂ ਦੇ ਈਮੇਲ 'ਤੇ ਇੱਕ ਇਨਵੌਇਸ ਮਿਲਦਾ ਹੈ, ਜਿਸ ਵਿੱਚ ਭੁਗਤਾਨ ਕਰਨ ਲਈ ਇੱਕ ਲਿੰਕ ਹੁੰਦਾ ਹੈ। ਇੱਕ ਵਾਰ ਭੁਗਤਾਨ ਪ੍ਰਾਪਤ ਹੋਣ 'ਤੇ, ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਨਵਿਆਰਣ (renewed) ਹੋ ਜਾਂਦੀ ਹੈ।

## Features

- Ultimate Multisite ਅਤੇ WooCommerce ਵਿਚਕਾਰ ਆਸਾਨ ਇੰਟੀਗ੍ਰੇਸ਼ਨ
- ਕਈ ਤਰ੍ਹਾਂ ਦੇ ਭੁਗਤਾਨ ਤਰੀਕੇ ਉਪਲਬਧ
- WooCommerce Subscriptions ਨਾਲ ਪੂਰੀ ਤਰ੍ਹਾਂ ਆਟੋਮੇਟਡ ਪ੍ਰੋਸੈਸ

## Documentation

**WooCommerce Integration ਸੈੱਟ ਕਰਨਾ**

**Ultimate Multisite ਨਾਲ ਆਪਣੇ ਭੁਗਤਾਨ ਵਿਕਲਪਾਂ ਦਾ ਵਿਸਤਾਰ: WooCommerce Integration**

## Installation

1. 'ultimate-multisite-woocommerce' ਨੂੰ '/wp-content/plugins/' ਡਾਇਰੈਕਟਰੀ ਵਿੱਚ ਅਪਲੋਡ ਕਰੋ
2. WordPress ਵਿੱਚ 'Plugins' ਮੀਨੂ ਰਾਹੀਂ ਪਲੱਗਇਨ ਨੂੰ ਐਕਟਿਵੇਟ ਕਰੋ
3. ਪਲੱਗਇਨ ਸੈੱਟ ਕਰਨ ਲਈ ਕਦਮ-ਦਰ-ਕਦਮ ਵਿਜ਼ਾਰਡ (Wizard) ਦੀ ਪਾਲਣਾ ਕਰੋ

## Usage Guide

_**ਮਹੱਤਵਪੂਰਨ ਨੋਟ: ਇਹ ਲੇਖ Ultimate Multisite version 2.x ਦਾ ਹਵਾਲਾ ਦਿੰਦਾ ਹੈ।**_

_**ਧਿਆਨ ਦਿਓ:** Ultimate Multisite: WooCommerce Integration ਲਈ ਤੁਹਾਡੀ ਮੁੱਖ ਸਾਈਟ 'ਤੇ ਘੱਟੋ ਘੱਟ WooCommerce ਦਾ ਐਕਟਿਵੇਟ ਹੋਣਾ ਜ਼ਰੂਰੀ ਹੈ।_

ਅਸੀਂ ਸਮਝਦੇ ਹਾਂ ਕਿ ਕੁਝ ਦੇਸ਼ਾਂ ਵਿੱਚ _Stripe_ ਅਤੇ _PayPal_ ਉਪਲਬਧ ਨਹੀਂ ਹਨ, ਜਿਸ ਕਾਰਨ Ultimate Multisite ਉਪਭੋਗਤਾਵਾਂ ਨੂੰ ਸਾਡੇ ਪਲੱਗਇਨ ਦੀ ਪ੍ਰਭਾਵਸ਼ਾਲੀ ਤਰ੍ਹਾਂ ਵਰਤੋਂ ਕਰਨ ਵਿੱਚ ਸੀਮਤਤਾ ਆਉਂਦੀ ਹੈ। ਇਸ ਲਈ ਅਸੀਂ _WooCommerce_ ਨੂੰ ਇੰਟੀਗ੍ਰੇਟ ਕਰਨ ਲਈ ਇੱਕ **add-on** ਬਣਾਇਆ ਹੈ।

_WooCommerce_ ਇੱਕ ਬਹੁਤ ਮਸ਼ਹੂਰ eCommerce ਪਲੱਗਇਨ ਹੈ। ਦੁਨੀਆ ਭਰ ਦੇ ਡਿਵੈਲਪਰਾਂ ਨੇ ਇਸ ਵਿੱਚ ਵੱਖ-ਵੱਖ ਭੁਗਤਾਨ ਗੇਟਵੇਅ (payment gateways) ਨੂੰ ਇੰਟੀਗ੍ਰੇਟ ਕਰਨ ਲਈ add-ons ਬਣਾਏ ਹਨ। ਅਸੀਂ ਇਸ ਦਾ ਫਾਇਦਾ ਉਠਾ ਕੇ Ultimate Multisite ਬਿਲਿੰਗ ਸਿਸਟਮ ਵਿੱਚ ਵਰਤੇ ਜਾ ਸਕਣ ਵਾਲੇ ਭੁਗਤਾਨ ਗੇਟਵੇਅ ਦਾ ਵਿਸਤਾਰ ਕੀਤਾ ਹੈ।

## Setting It Up

ਪਹਿਲਾਂ, ਤੁਹਾਨੂੰ WooCommerce Integration add-on ਸਥਾਪਿਤ ਕਰਨ ਦੀ ਲੋੜ ਹੈ ਜੋ **ਲਾਈਸੈਂਸਡ ਉਪਭੋਗਤਾਵਾਂ ਲਈ ਮੁਫ਼ਤ** ਹੈ। ਇਸਨੂੰ ਸਥਾਪਿਤ ਕਰਨ ਲਈ, ਆਪਣੇ ਸੁਪਰ ਐਡਮਿਨ ਡੈਸ਼ਬੋਰਡ 'ਤੇ Ultimate Multisite 'ਤੇ ਕਲਿੱਕ ਕਰੋ ਅਤੇ **Add-ons** ਚੁਣੋ।

![Add-ons page](/img/admin/addons-page.png)

ਇੱਥੇ ਤੁਹਾਨੂੰ ਸਾਰੇ Ultimate Multisite add-ons ਮਿਲਣਗੇ। **Ultimate Multisite: WooCommerce Integration** add-on 'ਤੇ ਕਲਿੱਕ ਕਰੋ।

![Add-ons page showing WooCommerce Integration](/img/admin/addons-page.png)

add-on ਦੇ ਵੇਰਵੇ ਨਾਲ ਇੱਕ ਵਿੰਡੋ ਖੁੱਲ੍ਹੇਗੀ। ਸਿਰਫ਼ **Install Now** 'ਤੇ ਕਲਿੱਕ ਕਰੋ *.*

<!-- Screenshot unavailable: WooCommerce Integration addon install popup with Install Now button -->

ਸਥਾਪਨਾ ਹੋਣ ਤੋਂ ਬਾਅਦ, ਤੁਹਾਨੂੰ ਪਲੱਗਇਨਾਂ ਦੇ ਪੇਜ 'ਤੇ ਰੀਡਾਇਰੈਕਟ ਕੀਤਾ ਜਾਵੇਗਾ। ਇੱਥੇ, ਸਿਰਫ਼ **Network Activate** 'ਤੇ ਕਲਿੱਕ ਕਰੋ ਅਤੇ _WooCommerce_ add-on ਤੁਹਾਡੇ ਨੈਟਵਰਕ 'ਤੇ ਐਕਟਿਵੇਟ ਹੋ ਜਾਵੇਗਾ।

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for WooCommerce Integration -->

ਇਹ ਐਕਟਿਵੇਟ ਕਰਨ ਤੋਂ ਬਾਅਦ, ਜੇਕਰ ਤੁਹਾਡੀ ਵੈੱਬਸਾਈਟ 'ਤੇ ਅਜੇ ਵੀ WooCommerce ਪਲੱਗਇਨ ਸਥਾਪਿਤ ਅਤੇ ਐਕਟਿਵੇਟ ਨਹੀਂ ਹੈ, **ਤਾਂ ਤੁਹਾਨੂੰ ਇੱਕ ਯਾਦ ਦਿਵਾਉਣ ਵਾਲਾ ਨੋਟ (reminder) ਮਿਲੇਗਾ**।

<!-- Screenshot unavailable: Admin notice reminding to install and activate WooCommerce plugin -->

ਇੰਸਟਾਲ ਹੋਣ 'ਤੇ, ਤੁਹਾਨੂੰ Ultimate Multisite payments settings ਦੇ ਹੇਠਾਂ ਇੱਕ ਵਾਧੂ ਵਿਕਲਪ ਦਿਖਾਈ ਦੇਵੇਗਾ। ਫਿਰ ਤੁਸੀਂ ਉੱਥੋਂ _WooCommerce_ ਵਿਕਲਪ ਨੂੰ **enable** ਕਰ ਸਕਦੇ ਹੋ।

![Payment gateway settings](/img/config/settings-payment-gateways.png)

ਤੁਸੀਂ ਕੁਝ ਬੁਨਿਆਦੀ ਸੈਟਿੰਗਾਂ ਕੰフィਗਰ ਕਰ ਸਕਦੇ ਹੋ, ਜਿਵੇਂ ਕਿ ਉਹ ਡਿਸਪਲੇ ਨਾਮ ਜੋ ਫਰੰਟ ਐਂਡ 'ਤੇ ਦਿਖਾਈ ਦੇਵੇਗਾ ਅਤੇ ਤੁਹਾਡੇ ਉਪਭੋਗਤਾਵਾਂ ਨੂੰ ਉਨ੍ਹਾਂ ਦੇ ਚੈੱਕਆਉਟ ਫਲੋ ਵਿੱਚ ਗਾਈਡ ਕਰਨ ਲਈ ਨਿਰਦੇਸ਼। ਫਿਰ ਤੁਸੀਂ ਆਪਣੀ **ਮੁੱਖ ਸਾਈਟ ਦੀ _WooCommerce_ ਸੈਟਿੰਗਾਂ** 'ਤੇ ਜਾ ਕੇ ਉਹ ਭੁਗਤਾਨ ਗੇਟਵੇਅ ਐਕਟਿਵੇਟ ਅਤੇ ਕੰフィਗਰ ਕਰ ਸਕਦੇ ਹੋ ਜੋ ਤੁਸੀਂ ਵਰਤਣਾ ਪਸੰਦ ਕਰਦੇ ਹੋ।

## How to manage the payment

WooCommerce Integration ਬਿਲਕੁਲ ਉਸੇ ਤਰੀਕੇ ਨਾਲ ਕੰਮ ਕਰਦਾ ਹੈ ਜਿਵੇਂ ਕਿ ਸਾਡਾ **Manual Gateway** ਕਰਦਾ ਹੈ। ਨੁਕਸਾਨ ਇਹ ਹੈ ਕਿ ਹਰ ਭੁਗਤਾਨ ਬਿਲਿੰਗ ਕਾਲ ਦੇ ਅੰਤ ਵਿੱਚ ਹੱਥੀਂ (manually) ਕਰਨਾ ਪੈਂਦਾ ਹੈ। ਇਹ ਪ੍ਰਵਾਹ (flow) ਇਸ ਤਰ੍ਹਾਂ ਕੰਮ ਕਰਦਾ ਹੈ:

  * ਇੱਕ ਵਾਰ ਉਪਭੋਗਤਾ ਦਾ ਬਿਲਿੰਗ ਚੱਕਰ ਖਤਮ ਹੋ ਜਾਂਦਾ ਹੈ, ਤਾਂ ਮੈਂਬਰਸ਼ਿਪ **ਰੋਕ (on hold)** ਦਿੱਤੀ ਜਾਂਦੀ ਹੈ। ਇਸੇ ਸਮੇਂ, ਤੁਹਾਡੇ ਨੈਟਵਰਕ ਦੀ ਮੁੱਖ ਸਾਈਟ 'ਤੇ _WooCommerce_ ਇੰਸਟਾਲ 'ਤੇ ਇੱਕ ਆਰਡਰ ਬਣਾਇਆ ਜਾਵੇਗਾ।

  * ਉਪਭੋਗਤਾਵਾਂ ਨੂੰ ਫਿਰ ਉਨ੍ਹਾਂ ਦੇ ਈਮੇਲ 'ਤੇ ਇੱਕ ਇਨਵੌਇਸ ਮਿਲਦਾ ਹੈ, ਜਿਸ ਵਿੱਚ ਭੁਗਤਾਨ ਕਰਨ ਲਈ ਇੱਕ ਲਿੰਕ ਹੁੰਦਾ ਹੈ (ਇਹ ਲਿੰਕ ਉਨ੍ਹਾਂ ਦੇ ਸਬਸਾਈਟ ਡੈਸ਼ਬੋਰਡ ਤੋਂ ਉਨ੍ਹਾਂ ਦੇ ਬਿਲਿੰਗ ਹਿਸਟਰੀ 'ਤੇ ਵੀ ਦਿਖਾਈ ਦਿੰਦਾ ਹੈ)। ਇਹ ਲਿੰਕ ਉਪਭੋਗਤਾ ਨੂੰ _**WooCommerce**_ **checkout** 'ਤੇ ਰੀਡਾਇਰੈਕਟ ਕਰਦਾ ਹੈ, ਜਿੱਥੇ ਉਪਭੋਗਤਾ ਆਪਣੇ ਭੁਗਤਾਨ ਨੂੰ ਪ੍ਰੋਸੈਸ ਕਰਨ ਲਈ **ਉਪਲਬਧ _WooCommerce_ ਗੇਟਵੇਅ** ਵਿੱਚੋਂ ਇੱਕ ਚੁਣ ਸਕਦੇ ਹਨ।

  * ਇੱਕ ਵਾਰ ਭੁਗਤਾਨ ਪ੍ਰਾਪਤ ਹੋਣ 'ਤੇ, ਮੈਂਬਰਸ਼ਿਪ **ਨਵਿਆਰਣ (renewed)** ਹੋ ਜਾਂਦੀ ਹੈ, ਭੁਗਤਾਨ Ultimate Multisite 'ਤੇ ਲੌਗ ਕੀਤਾ ਜਾਂਦਾ ਹੈ ਅਤੇ ਆਰਡਰ ਨੂੰ ਪੂਰਾ (complete) ਸੈੱਟ ਕਰ ਦਿੱਤਾ ਜਾਂਦਾ ਹੈ।

### Some Important Notes:

  * Ultimate Multisite ਦੀ ਡਿਫਾਲਟ ਕਾਰਜਕੁਸ਼ਲਤਾ (default functionality) ਇਸ ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਫਾਰਮ ਨਾਲ ਕੰਮ ਕਰਦੀ ਰਹਿੰਦੀ ਹੈ। ਉਦਾਹਰਨ ਲਈ, **Subscription Management** ਸਕ੍ਰੀਨ ਤੋਂ **refunds ਜਾਰੀ ਕਰਨਾ** WooCommerce ਨਾਲ ਸੰਚਾਰ ਕਰੇਗਾ ਤਾਂ ਜੋ ਚੁਣੇ ਹੋਏ ਭੁਗਤਾਨ ਫਾਰਮ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਰਿਫੰਡ ਜਾਰੀ ਕੀਤਾ ਜਾ ਸਕੇ।

  * Ultimate Multisite: WooCommerce Integration ਨੂੰ ਕੰਮ ਕਰਨ ਲਈ WP-Cron 'ਤੇ ਨਿਰਭਰਤਾ (relies) ਹੈ। ਇਹ ਜ਼ਿਆਦਾਤਰ ਸਮੇਂ ਬਹੁਤ ਵਧੀਆ ਕੰਮ ਕਰਦਾ ਹੈ, ਪਰ ਜੇਕਰ ਸਾਈਟ 'ਤੇ ਟ੍ਰੈਫਿਕ ਘੱਟ ਹੈ ਤਾਂ ਕ੍ਰੋਨ (cron) ਨੂੰ ਨਿਯਤ ਇਵੈਂਟਾਂ ਦੀ ਜਾਂਚ ਕਰਨ ਲਈ ਪ੍ਰੇਰਿਤ ਨਹੀਂ ਕੀਤਾ ਜਾਂਦਾ ਅਤੇ ਇਵੈਂਟ ਗਾਇਬ ਹੋ ਜਾਂਦਾ ਹੈ, ਜਿਸ ਨਾਲ ਕਾਰਜਕਾਰੀ ਵਿੱਚ ਦੇਰੀ ਹੋ ਸਕਦੀ ਹੈ।

## Woocommerce Subscriptions Plugin

ਹਰ ਮਹੀਨੇ ਹੱਥੀਂ ਭੁਗਤਾਨ ਕਰਨਾ ਅੰਤ-ਉਪਭੋਗਤਾਵਾਂ (end-users) ਲਈ ਆਦਰਸ਼ ਨਹੀਂ ਹੈ ਕਿਉਂਕਿ ਇੱਕ ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਨੂੰ ਹਰ ਬਿਲਿੰਗ ਚੱਕਰ ਵਿੱਚ ਆਪਣੇ ਆਪ ਵਾਰ-ਵਾਰ ਹੋਣਾ ਸਪੱਕਟ ਹੁੰਦਾ ਹੈ। ਇਸ ਲਈ ਅਸੀਂ WooCommerce Integration add-on ਨੂੰ **WooCommerce Subscriptions plugin** ਨੂੰ ਸਪੋਰਟ ਕਰਨ ਲਈ ਡਿਜ਼ਾਈਨ ਕੀਤਾ ਹੈ। ਇਹ ਤੁਹਾਡੇ ਅੰਤ-ਉਪਭੋਗਤਾਵਾਂ ਲਈ ਲੈਣ-ਦੇਣ ਨੂੰ ਆਸਾਨ ਬਣਾਉਣ ਲਈ ਭੁਗਤਾਨ ਨੂੰ ਆਟੋਮੇਟ ਕਰ ਦੇਵੇਗਾ।

ਤੁਹਾਨੂੰ ਸਿਰਫ਼ [Woocommerce Subscriptions plugin](https://woocommerce.com/products/woocommerce-subscriptions/) **ਸਥਾਪਿਤ** ਕਰਨ ਅਤੇ ਇਸਨੂੰ **ਆਪਣੀ ਮੁੱਖ ਸਾਈਟ 'ਤੇ ਐਕਟਿਵੇਟ** ਰੱਖਣ ਦੀ ਲੋੜ ਹੈ। ਫਿਰ Ultimate Multisite payments settings 'ਤੇ ਵਾਪਸ ਜਾਓ ਅਤੇ **integration ਨੂੰ enable** ਕਰੋ।

ਅਤੇ ਬੱਸ ਇੰਨਾ। ਤੁਹਾਡੇ ਅੰਤ-ਉਪਭੋਗਤਾਵਾਂ ਨੂੰ ਹੁਣ ਆਪਣੇ ਬਿਲਿੰਗ ਚੱਕਰ ਦੇ ਅੰਤ ਵਿੱਚ ਹੱਥੀਂ ਭੁਗਤਾਨ ਕਰਨ ਦੀ ਲੋੜ ਨਹੀਂ ਹੋਵੇਗੀ।

ਇਸ ਤੋਂ ਇਲਾਵਾ ਕਿ ਤੁਹਾਡੀ ਮੁੱਖ ਸਾਈਟ 'ਤੇ _WooCommerce_ ਦੇ ਹੇਠਾਂ ਇੱਕ ਆਰਡਰ ਬਣੇਗਾ, ਇਹ ਹਰੇਕ account ਲਈ ਇੱਕ ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਵੀ ਬਣਾਏਗਾ।
