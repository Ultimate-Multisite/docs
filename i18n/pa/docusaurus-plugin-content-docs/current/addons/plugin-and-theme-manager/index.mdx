# Plugin & Theme Manager {#plugin--theme-manager}

Multisite Ultimate: Plugin and Theme Manager

Install ਕੀਤੇ Plugins ਅਤੇ Themes ਦੀ meta information (title, author, thumbnail ਅਤੇ description) ਨੂੰ Edit ਜਾਂ Hide ਕਰੋ। Plugins ਅਤੇ Themes ਲਈ categories ਬਣਾਓ ਤਾਂ ਜੋ ਤੁਹਾਡੇ ਉਪਭੋਗਤਾ (users) ਉਨ੍ਹਾਂ ਨੂੰ filter ਕਰ ਸਕਣ ਅਤੇ ਆਪਣੇ panel ਵਿੱਚ ਇੱਕ ਸੁੰਦਰ custom Plugins page ਜੋੜ ਸਕਣ।

Plugin ਦੀ ਵਰਤੋਂ ਕਰਨ ਲਈ, ਤੁਸੀਂ ਆਪਣੇ network admin 'Plugins' ਜਾਂ 'Themes' pages 'ਤੇ ਜਾ ਕੇ "Edit Plugin/Theme info" 'ਤੇ ਕਲਿੱਕ ਕਰ ਸਕਦੇ ਹੋ।

## Installation {#installation}

1. 'ultimate-multisite-plugin-and-theme-manager' ਨੂੰ '/wp-content/plugins/' directory ਵਿੱਚ Upload ਕਰੋ
2. ਆਪਣੇ WordPress Network Admin ਵਿੱਚ 'Plugins' menu ਰਾਹੀਂ plugin ਨੂੰ Activate ਕਰੋ

## Usage Guide {#usage-guide}

Ultimate Multisite: Plugin and Theme Manager ਇੱਕ ਸ਼ਕਤੀਸ਼ਾਲੀ **premium** Ultimate Multisite add-on ਹੈ ਜੋ ਤੁਹਾਨੂੰ ਇਹ ਪੂਰਾ ਕੰਟਰੋਲ ਦਿੰਦਾ ਹੈ ਕਿ ਤੁਹਾਡੇ network ਦੇ ਉਪਭੋਗਤਾ (users) ਤੁਹਾਡੇ network 'ਤੇ plugin ਅਤੇ theme options ਨੂੰ ਕਿਵੇਂ ਦੇਖਦੇ ਅਤੇ ਨਾਲ ਕਿਵੇਂ ਗੱਲਬਾਤ ਕਰਦੇ ਹਨ।

ਇਹ add-on ਤਿੰਨ ਕੰਮ ਕਰਦਾ ਹੈ:

  1. ਇਹ ਤੁਹਾਨੂੰ Plugins ਅਤੇ Themes ਦੋਵਾਂ ਦੀ metadata ਨੂੰ ਪੂਰੀ ਤਰ੍ਹਾਂ ਬਦਲਣ ਦਿੰਦਾ ਹੈ। ਇਸ ਵਿੱਚ ਨਾਮ (name), description, author ਦਾ ਨਾਮ, ਅਤੇ ਇੱਥੋਂ ਤੱਕ ਕਿ thumbnails ਵੀ ਸ਼ਾਮਲ ਹਨ!

  2. ਇਹ ਤੁਹਾਨੂੰ Plugins ਅਤੇ Themes ਨੂੰ categorize ਕਰਨ ਦਿੰਦਾ ਹੈ, ਤਾਂ ਜੋ ਤੁਹਾਡੇ ਉਪਭੋਗਤਾ ਉਨ੍ਹਾਂ ਨੂੰ ਆਸਾਨੀ ਨਾਲ sort ਕਰ ਸਕਣ।

  3. ਇਹ ਤੁਹਾਨੂੰ ਪੁਰਾਣੇ, ਬੋਰਿੰਗ plugins page ਨੂੰ categories ਨਾਲ sort ਹੋਣ ਵਾਲੇ ਇੱਕ ਨਵੇਂ themes-like page ਨਾਲ ਬਦਲਣ ਦਾ option ਦਿੰਦਾ ਹੈ।

ਇਸ ਨੂੰ download ਅਤੇ install ਕਰਨ ਤੋਂ ਬਾਅਦ, Ultimate Multisite Settings page 'ਤੇ ਇੱਕ ਨਾਵਾਂ tab - _Add-on Settings_ - ਦਿਖਾਈ ਦੇਵੇਗਾ।
