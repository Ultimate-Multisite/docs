# Support Agents

ਪੂਰੇ ਨੈਟਵਰਕ ਨੂੰ ਸਿਰਫ਼ ਆਪਣੇ ਆਪ ਸੰਭਾਲਣਾ ਕੋਈ ਆਸਾਨ ਕੰਮ ਨਹੀਂ ਹੈ। ਜਿਵੇਂ-ਜਿਵੇਂ ਤੁਹਾਡਾ ਨੈਟਵਰਕ ਵੱਧਦਾ ਜਾਵੇਗਾ, ਤੁਹਾਨੂੰ ਸਪੋਰਟ ਦੀ ਵਧਦੀ ਮੰਗ ਮਹਿਸੂਸ ਹੋਵੇਗੀ। ਆਪਣੇ ਕੰਮ ਦਾ ਬੋਝ ਹਲਕਾ ਕਰਨ ਲਈ ਲੋਕਾਂ ਦੀ ਮਦਦ ਲੈਣਾ ਇੱਕ ਵਧੀਆ ਵਿਚਾਰ ਹੋਵੇਗਾ। ਹਾਲਾਂਕਿ, ਪੂਰਾ ਐਕਸੈਸ/ਪਰਮਿਸ਼ਨ ਦੇਣਾ ਖ਼ਤਰਨਾਕ ਹੋ ਸਕਦਾ ਹੈ ਅਤੇ ਤੁਹਾਡੇ ਨੈਟਵਰਕ ਨਾਲ ਸਮਝੌਤਾ ਕਰ ਸਕਦਾ ਹੈ। ਇੱਥੇ ਹੀ ਇਹ Support Agent add-on ਕੰਮ ਆਉਂਦਾ ਹੈ।

ਇਹ add-on ਤੁਹਾਨੂੰ Support Agent ਨਾਮਕ ਇੱਕ ਨਵਾਂ ਤਰ੍ਹਾਂ ਦਾ WordPress user ਬਣਾਉਣ ਦੀ ਇਜਾਜ਼ਤ ਦਿੰਦਾ ਹੈ, ਅਤੇ ਇਹ ਤੁਹਾਨੂੰ ਪੂਰਾ ਕੰਟਰੋਲ ਦਿੰਦਾ ਹੈ ਕਿ ਉਸ user ਕੋਲ ਕਿਹੜੀਆਂ ਪਰਮਿਸ਼ਨਾਂ ਹੋਣਗੀਆਂ। ਪਰਮਿਸ਼ਨ ਦੇਣਾ ਇੱਕ ਟੋਗਲ ਬਟਨ ਚਾਲੂ ਕਰਨ ਜਿੰਨਾ ਆਸਾਨ ਹੈ, ਅਤੇ ਜੇ ਤੁਸੀਂ ਇਸਨੂੰ ਹਟਾਉਣਾ ਚਾਹੋਗੇ ਤਾਂ ਇਸਨੂੰ ਬੰਦ ਕਰਨਾ ਵੀ اتنا ਹੀ ਆਸਾਨ ਹੈ।

## ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ

*   WordPress ਅਤੇ Ultimate Multisite ਦੀਆਂ ਪਰਮਿਸ਼ਨਾਂ ਦੀ ਵਿਸਤ੍ਰਿਤ ਸੂਚੀ
*   ਆਸਾਨੀ ਨਾਲ ਐਕਸੈਸ ਨੂੰ ਚਾਲੂ ਅਤੇ ਬੰਦ ਕਰਨਾ
*   ਬੇਅੰਤ ਗਿਣਤੀ ਵਿੱਚ Agent accounts ਬਣਾਉਣਾ

## ਇੰਸਟਾਲੇਸ਼ਨ

1.  'ultimate-multisite-support-agents' ਨੂੰ '/wp-content/plugins/' ਡਾਇਰੈਕਟਰੀ ਵਿੱਚ ਅਪਲੋਡ ਕਰੋ।
2.  WordPress ਵਿੱਚ 'Plugins' ਮੀਨੂ ਰਾਹੀਂ plugin ਨੂੰ ਐਕਟਿਵੇਟ ਕਰੋ।
3.  plugin ਸੈਟਅੱਪ ਕਰਨ ਲਈ ਕਦਮ-ਦਰ-ਕਦਮ Wizard ਦੀ ਪਾਲਣਾ ਕਰੋ।
