# Filter: wu_cap_captcha_config

Cap Captcha ਦੀ ਕਨਫਿਗਰੇਸ਼ਨ ਨੂੰ ਫਿਲਟਰ ਕਰਦਾ ਹੈ।

ਇਹ ਡਿਵੈਲਪਰਾਂ ਨੂੰ Cap Captcha ਸੈਟਿੰਗਜ਼ ਨੂੰ ਕਸਟਮਾਈਜ਼ ਕਰਨ ਦੀ ਇਜਾਜ਼ਤ ਦਿੰਦਾ ਹੈ।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $config | `array` | Cap ਕਨਫਿਗਰੇਸ਼ਨ ਵਾਲਾ array। |
| $level | `string` | ਸੁਰੱਖਿਆ ਦਾ ਪੱਧਰ (fast, medium, max)। |
