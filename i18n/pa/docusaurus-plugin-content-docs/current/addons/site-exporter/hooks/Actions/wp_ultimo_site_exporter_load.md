# Action: wp_ultimo_site_exporter_load

ਇਹ ਉਦੋਂ ਟ੍ਰਿਗਰ ਹੁੰਦਾ ਹੈ ਜਦੋਂ ਸਾਰੀਆਂ ਡਿਪੈਂਡੈਂਸੀਜ਼ (dependencies) ਲੋਡ ਹੋ ਜਾਂਦੀਆਂ ਹਨ।

ਇਹ ਪਲੱਗਇਨ ਡਿਵੈਲਪਰਾਂ ਨੂੰ ਨਵੀਆਂ ਕਾਰਜਕਮਤਾਵਾਂ (functionality) ਜੋੜਨ ਦੀ ਇਜਾਜ਼ਤ ਦਿੰਦਾ ਹੈ। ਜਿਵੇਂ ਕਿ, ਨਵੇਂ ਹੋਸਟਿੰਗ ਪ੍ਰੋਵਾਇਡਰਾਂ ਲਈ ਸਪੋਰਟ, ਆਦਿ।
