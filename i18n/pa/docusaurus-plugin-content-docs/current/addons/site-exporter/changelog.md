# ਸਾਈਟ ਐਕਸਪੋਰਟਰ ਚੇਂਜਲੌਗ {#site-exporter-changelog}

Version: 1.0.6 - ਜਾਰੀ ਹੋਇਆ 2026-05-11
* ਨਵਾਂ: ਸਾਈਟ ਐਕਸਪੋਰਟ ਹੁਣ ਇੱਕ ਆਪ-ਸ਼ੁਰੂ ਹੋਣ ਵਾਲਾ index.php ਨਾਲ ਪਾਉਂਦੇ ਹਨ, ਤਾਂ ਜੋ ZIP ਨੂੰ ਕਿਸੇ ਨਵੇਂ ਹੋਸਟ 'ਤੇ ਵੱਖਰੇ plugin ਇੰਸਟਾਲ ਕੀਤੇ ਬਿਨਾਂ ਲਗਾ ਸਕਿਆ ਜਾ ਸਕੇ।
* ਨਵਾਂ: ਨੈੱਟਵਰਕ ਐਕਸਪੋਰਟ ਐਡਮਿਨਰੇਟਰਾਂ ਨੂੰ ਸਾਈਟ ਐਕਸਪੋਰਟ ਐਡਮਿਨ ਪੇਜ ਤੋਂ ਸਾਰੇ ਸਬਸਾਈਟਾਂ ਨੂੰ ਇੱਕੋ ਆਰਕਾਈਵ ਵਿੱਚ ਐਕਸਪੋਰਟ ਕਰਨ ਦਿੰਦਾ ਹੈ।

Version: 1.0.5 - ਜਾਰੀ ਹੋਇਆ 2026-05-05
* ਸੰchein: `get_config` ਵਰਗੇ ਅਣ-ਮੌਕਡ (unmocked) methods ਨੂੰ ਸੰਭਾਲਣ ਲਈ WP_CLI polyfill stub ਵਿੱਚ `__callStatic` ਜੋੜ ਦਿੱਤਾ ਗਿਆ ਹੈ, ਜਿਸ ਨਾਲ ਟੈਸਟ ਬੂਟਸਟ੍ਰੈਪ ਫੇਲ੍ਹ ਹੋਣ ਤੋਂ ਰੋਕਿਆ ਜਾ ਸਕੇ।

Version: 1.0.4 - ਜਾਰੀ ਹੋਇਆ 2025-11-25

* Ultimate Multisite 2.4 ਨਾਲ ਅਨੁਕੂਲਤਾ (compatibility) ਲਈ ਸੰchein ਕੀਤਾ ਗਿਆ।

Version: 1.0.2 - ਜਾਰੀ ਹੋਇਆ 2025-09-28

* ਪ੍ਰੀਫਿਕਸ ਨੂੰ ultimate-multisite ਵਿੱਚ ਬਦਲਿਆ; ਟੈਕਸਟ ਡੋਮੇਨ ਅਪਡੇਟ ਕੀਤਾ; ਅਤੇ version ਵਧਾਇਆ।

### Version 1.0.1 - ਜਾਰੀ ਹੋਇਆ 2023-08-09 {#version-101---released-on-2023-08-09}

* ਸੁਧਾਰਿਆ ਗਿਆ: ਹੁਣ ਇਹ ਯਕੀਨੀ ਬਣਾਉਂਦਾ ਹੈ ਕਿ ਇਮਪੋਰਟਰ ਫਾਈਲ ਦਾ ਸਹੀ ਫਾਈਲ ਪਾਥ ਮਿਲ ਸਕੇ।
* ਸੰchein ਕੀਤਾ ਗਿਆ: ਇੰਪੋਰਟ ਦੌਰਾਨ ਗਲਤੀਆਂ ਰੋਕਣ ਲਈ ਆਬਜੈਕਟ ਕੈਸ਼ (object cache) ਨੂੰ ਡਿਸੇਬਲ ਕਰ ਦਿੱਤਾ ਗਿਆ।
* ਸੰchein ਕੀਤਾ ਗਿਆ: ਇਮਪੋਰਟਰ plugin ਦੇ ਡਾਊਨਲੋਡ ਲਿੰਕ ਦੇ ਕੰਮ ਨਾ ਕਰਨ ਵਾਲੇ ਮੁੱਦੇ ਨੂੰ ਠੀਕ ਕੀਤਾ।
