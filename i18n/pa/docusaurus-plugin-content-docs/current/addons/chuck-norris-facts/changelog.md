### 1.0.1 - 2025-11-03 {#101---2025-11-03}
* ਜੋੜਿਆ: Ultimate Multisite update server ਰਾਹੀਂ ਆਟੋਮੈਟਿਕ ਅਪਡੇਟ ਕਾਰਜਕੁਸ਼ਲਤਾ।
* ਅਪਡੇਟ ਕੀਤਾ: WordPress 6.6 ਤੱਕ ਟੈਸਟ ਕੀਤਾ ਗਿਆ।
* ਅਪਡੇਟ ਕੀਤਾ: composer autoloader ਨਾਲ plugin ਦਾ ਢਾਂਚਾ ਸੁਧਾਰਿਆ ਗਿਆ।
* ਅਪਡੇਟ ਕੀਤਾ: ਆਧੁਨਿਕ WordPress ਇੰਸਟਾਲੇਸ਼ਨਾਂ ਨਾਲ ਬਿਹਤਰ ਅਨੁਕੂਲਤਾ।
