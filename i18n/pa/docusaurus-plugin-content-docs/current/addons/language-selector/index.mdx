# Language Selector

Multisite Ultimate: Sign-up 'ਤੇ Language Selector

Sign-up flow ਵਿੱਚ ਇੱਕ language selector field ਜੋੜੋ।

## Installation

1. 'ultimate-multisite-language-selector' ਨੂੰ '/wp-content/plugins/' directory ਵਿੱਚ upload ਕਰੋ
2. ਆਪਣੇ WordPress Network Admin ਵਿੱਚ 'Plugins' menu ਰਾਹੀਂ plugin activate ਕਰੋ

## Usage Guide

ਇਸ ਮੁਫ਼ਤ add-on ਨਾਲ ਤੁਸੀਂ ਆਪਣੇ users ਨੂੰ ਇਹ ਚੁਣਨ ਦੇ ਸਕਦੇ ਹੋ ਕਿ ਉਹ sign-up flow ਅਤੇ login page 'ਤੇ ਕਿਹੜੀ language ਦੇਖਣਾ ਚਾਹੁੰਦੇ ਹਨ।

[ਆਪਣਾ license activate ਕਰਨ ਤੋਂ ਬਾਅਦ](../../user-guide/getting-started/installing-ultimate-multisite.md) ਤੁਸੀਂ Ultimate Multisite add-ons install ਕਰ ਸਕੋਗੇ ਜਿਨ੍ਹਾਂ ਤੱਕ ਤੁਹਾਡੇ license ਨਾਲ ਪਹੁੰਚ ਮਿਲਦੀ ਹੈ।

ਆਪਣੇ Ultimate Multisite settings page 'ਤੇ ਜਾਓ ਅਤੇ ਫਿਰ, ਸੱਜੇ ਪਾਸੇ (left) menu 'ਤੇ, _Add-ons_ label 'ਤੇ ਕਲਿੱਕ ਕਰੋ।

![Add-ons page](/img/admin/addons-page.png)

_Language Selector install ਕਰਨਾ ਸ਼ੁਰੂ ਕਰਨ ਲਈ Add-ons page 'ਤੇ ਜਾਓ_

ਉੱਥੇ, ਤੁਹਾਨੂੰ ਸਾਰੇ add-ons ਮਿਲ ਜਾਣਗੇ ਜੋ ਅਸੀਂ ਪੇਸ਼ ਕਰਦੇ ਹਾਂ।

ਮੁਫ਼ਤ add-ons ਸਾਡੇ ਸਾਰੇ customers ਲਈ ਉਪਲਬਧ ਹਨ, ਭਾਵੇਂ ਉਨ੍ਹਾਂ ਨੇ ਕਿਹੜਾ subscription plan ਲਿਆ ਹੋਵੇ।

add-on ਦੇ thumbnail ਦੇ ਹੇਠਾਂ ਦਿੱਤੇ _Install it Now!_ button 'ਤੇ ਕਲਿੱਕ ਕਰੋ। ਫਿਰ ਇਸਨੂੰ Plugins page 'ਤੇ activate ਕਰੋ।

![Add-ons page with install button](/img/admin/addons-page.png)

_add-on install ਕਰਨ ਲਈ button 'ਤੇ ਕਲਿੱਕ ਕਰੋ_

![Settings page](/img/config/settings-general.png)

## URL ਰਾਹੀਂ Language ਸੈੱਟ ਕਰਨਾ

ਤੁਸੀਂ locale parameter ਨੂੰ URL 'ਤੇ bypass ਕਰਕੇ ਵੀ page ਦੀ language ਬਦਲ ਸਕਦੇ ਹੋ। ਜੇ ਤੁਸੀਂ, ਉਦਾਹਰਨ ਲਈ, Italian ਵਿੱਚ register page 'ਤੇ redirect ਕਰਨਾ ਚਾਹੁੰਦੇ ਹੋ, ਤਾਂ URL ਵਿੱਚ ?locale=it_IT ਜੋੜੋ।

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

ਇਸ ਨੂੰ ਕੰਮ ਕਰਨ ਲਈ language ਤੁਹਾਡੇ WordPress network 'ਤੇ install ਹੋਣੀ ਚਾਹੀਦੀ ਹੈ।
