# ਭਾਸ਼ਾ ਸੇਲੈਕਟਰ ਚੇਂਜਲੌਗ {#language-selector-changelog}

ਸੰਸਕਰਣ: 1.1.3 - 2026-05-05 ਨੂੰ ਰਿਲੀਜ਼ ਹੋਇਆ
* ਸੁਧਾਰ: ਇਨਪੁਟ ਵੇਰੀਏਬਲਸ ਹੁਣ ਸੈਨੀਟਾਈਜ਼ (sanitized) ਕੀਤੇ ਜਾਂਦੇ ਹਨ ਅਤੇ ਕਸਟਮ ਹੁੱਕ ਨਾਮਾਂ ਨੂੰ ਪੂਰਵਕ (prefixed) ਕੀਤਾ ਗਿਆ ਹੈ ਤਾਂ ਜੋ ਤੀਜੀ-ਧਿਰ ਦੇ ਪਲੱਗਇਨ ਨਾਲ ਕੋਈ ਟਕਰਾਅ ਨਾ ਹੋਵੇ।

ਸੰਸਕਰਣ: 1.1.2 - 2026-01-01 ਨੂੰ ਰਿਲੀਜ਼ ਹੋਇਆ
* ਸੁਧਾਰ: ਕੰਪੈਟਿਬਿਲਟੀ ਅਪਡੇਟਸ ਅਤੇ ਕੋਡ ਦੀ ਗੁਣਵੱਤਾ ਵਿੱਚ ਸੁਧਾਰ ਕੀਤਾ ਗਿਆ।

ਸੰਸਕਰਣ: 1.1.0 - 2025-09-28 ਨੂੰ ਰਿਲੀਜ਼ ਹੋਇਆ
* ਪ੍ਰੀਫਿਕਸ ਦਾ ਨਾਮ ultimate-multisite ਕੀਤਾ ਗਿਆ; ਟੈਕਸਟ ਡੋਮੇਨ ਅਪਡੇਟ ਕੀਤਾ; ਅਤੇ ਸੰਸਕਰਣ ਵਧਾਇਆ।
