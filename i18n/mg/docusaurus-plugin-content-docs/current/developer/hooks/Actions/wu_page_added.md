# Action: wu_page_added

Avelao ny mpamorona plugin hampandeha zavatra fanampiny rehefa voasoratra ny pejy.

Tsy toy ny wu_page_load, izay mandeha ihany rehefa jerena ny pejy manokana iray, ity hook ity dia mandeha amin'ny fisoratana anarana ho an'ny pejy admin rehetra ampiana amin'ny fampiasana kaody Ultimate Multisite.

## Parametera

| Anarana | Karazana | Famaritana |
|------|------|-------------|
| $page_id | `string` | Ny ID an'ity pejy ity. |
| $page_hook | `string` | Ny anaran'ny hook an'ity pejy ity. |

- 2.0.0
