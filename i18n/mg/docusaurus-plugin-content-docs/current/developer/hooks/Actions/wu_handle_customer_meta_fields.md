# Hetsika: wu_handle_customer_meta_fields

Mamela ny mpamorona plugin hitahiry angona meta amin'ny fomba samihafa raha ilain'izy ireo izany.

## Parametra {#parameters}

| Anarana | Karazana | Famaritana |
|------|------|-------------|
| $meta_repository | `array` | Ny lisitry ny saha meta, voarafitra ho key =&gt; value. |
| $customer | `\Customer` | Ny zavatra customer an'ny Ultimate Multisite. |
| $checkout | `\Checkout` | Ny kilasin'ny famaranana fividianana. |

### Hatramin'ny {#since}

- 2.0.0
