# Sivana: wu_should_create_domain_record_for_site

Manivana raha tokony hamorona firaketana domain ho an’ny tranonkala vao noforonina ny Ultimate Multisite.

Ampiasao ity sivana ity mba hanakanana na hanemorana ny famoronana mandeha ho azy ny firaketana domain ho an’ireo tranonkala mampiasa domain fototra iombonana amin’ny checkout-form, mpampiantrano anatiny, na domain izay hotantanan’ny fampidirana hafa misaraka.

## Masontsivana {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $create | `bool` | Raha tokony hoforonina ny firaketana domain. |
| $site | `WP_Site` | Ilay zavatra tranonkala vao noforonina. |

### Hatramin’ny {#since}

- 2.13.0

### Loharano {#source}

Voafaritra ao amin’ny `inc/functions/domain.php`.


## Mamerina {#returns}

Boolean manondro raha hamorona ny firaketana domain.
