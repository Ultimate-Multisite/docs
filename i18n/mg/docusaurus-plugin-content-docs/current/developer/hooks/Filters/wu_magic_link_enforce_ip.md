# Sivana: wu_magic_link_enforce_ip

Sivanina raha hampihatra fanamarinana adiresy IP.

Apetraka ho false mba hamelana ny tokens hiasa avy amin'ny tambajotra samihafa. Mampihena ny fiarovana izany saingy mampitombo ny fahamoran'ny fampiasana (ohatra, ho an'ny mpampiasa finday mifamadika tambajotra).

## Masontsivana

| Anarana | Type | Famaritana |
|------|------|-------------|
| $enforce | `bool` | Raha hampihatra ny fampitoviana adiresy IP. |
