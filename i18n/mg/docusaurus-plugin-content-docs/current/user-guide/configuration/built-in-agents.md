# Agents anatiny

Gratis AI Agent v1.9.0 dia mitondra agents anatiny dimy, samy efa voaomana mialoha miaraka amin’ny fitambaran-fitaovana mifantoka, system prompt namboarina manokana, ary soso-kevitra fanombohana mifanaraka amin’ny asa mahazatra amin’io sehatra io. Ny fifindrana eo amin’ireo agents dia manova izay azon’ilay assistant atao sy ny fomba famaliany — tsy mila fikirakirana avy aminao. Superdav AI Agent v1.18.0 dia afaka manampy fitaovana mahafantatra fandaharam-potoana, firaketana fampahatsiahivana, vavahadin-fankatoavana, ary fampandrenesana SMS amin’ireo workflows ireo rehefa voakirakira ny integrations mifandraika.

## Inona no atao hoe Agent?

Ny agent tsirairay dia mombamomba fikirakirana manana anarana izay mampifangaro:

- **Fitaovana** — ireo fahaiza-manao azon’ilay agent antsoina (oh: ny Content Writer dia manana fidirana amin’ny fahaiza-mamorona lahatsoratra; ny Design Studio dia manana fidirana amin’ny fahaiza-manao CSS sy theme.json)
- **System prompt** — toromarika mametraka ny feo, laharam-pahamehana, ary fetra ho an’ilay agent
- **Soso-kevitra** — prompts efa voasoratra mialoha aseho ao amin’ny endrika chat mba hanampy anao hanomboka haingana

## Fidirana amin’ny Agent Picker

1. Sokafy ny tontonana **Gratis AI Agent** ao amin’ny sisin’ny admin WordPress.
2. Tsindrio ny **kisary agent** eo amin’ny zorony ambony havia amin’ny lohatenin’ny chat (miova ilay kisary mba hanehoana ny agent mavitrika).
3. Misokatra ho toy ny rakotra form-table ny **Agent Picker**. Voatanisa miaraka amin’ny kisary, anarana, ary famaritana andalana iray ny agent tsirairay.
4. Tsindrio ny laharan’agent iray hampavitrika azy. Havaozina avy hatrany ny lohatenin’ny chat.

Azonao atao koa ny mifindra agents eo afovoan-dresaka — manomboka mihatra amin’ny hafatra manaraka ny system prompt an’ilay agent vaovao.

## Ireo Agents anatiny dimy

### Content Writer

**Fifantohana:** Mamorona sy manitsy lahatsoratra, pejy, ary taratasy fifandraisana.

**Fitaovana misy:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`. Raha alefa ny integrations Superdav AI Agent v1.18.0, dia mety ho azo ampiasaina amin’ny workflows ankatoavina koa ny tontolon-kalandrie voakirakira, vavahadin-fankatoavana, fampahatsiahivana, ary fitaovana fampandrenesana SMS.

**Zavatra hainy tsara:**
- Mandrafitra sy mamoaka lahatsoratra bilaogy avy amin’ny brief na drafitra
- Mamorona andiana landing pages ho an’ny site vaovao
- Manangana taratasy fifandraisana sy fanontaniana
- Mametraka featured images amin’ny lahatsoratra avy amin’ny URL na fikarohana
- Mandrafitra hafatra fanarahana hetsika avy amin’ny tontolon’ny Google Calendar voakirakira, avy eo miato ho amin’ny fankatoavana alohan’ny handefasana fampandrenesana

**Soso-kevitra fanombohana:**
- *Manorata lahatsoratra bilaogy 500 teny momba ny tombontsoan’ny WordPress multisite.*
- *Mamoròna pejy About, Services, ary Contact ary avoahy izy ireo.*
- *Ampio taratasy fanontaniana famandrihana ao amin’ny pejy Contact.*
- *Mandrafeta fampahatsiahivana ho an’ireo mpandray anjara amin’ny hetsika kalandrie voakirakira rahampitso ary miandrasa fankatoavana alohan’ny handefasana azy.*

### Site Builder

**Fifantohana:** Famoronana tranonkala hatramin’ny voalohany ka hatramin’ny farany avy amin’ny prompt tokana.

**Fitaovana misy:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`. Miaraka amin’ny Superdav AI Agent v1.18.0, dia mety ho azo ampiasaina ireo fitaovana managed-service, fankatoavana, fampahatsiahivana, kalandrie, ary SMS voakirakira amin’izay alefan’ny mpitantana azy.

**Zavatra hainy tsara:**
- Mamorona drafitra fananganana site amin’ny dingana maro ho an’ny karazam-barotra voalaza
- Manatanteraka dingana tsirairay amin’ny fomba mizaka tena — rafitra, atiny, fitetezana, famolavolana
- Miverina amin’ny laoniny avy amin’ny hadisoana eo afovoan-drafitra nefa tsy mila fidirana an-tanana
- Mametraka plugins atolotra ho ampahany amin’ny fananganana
- Mamorona taratasy fifandraisana mivantana avy amin’ny endrika chat (Superdav AI Agent v1.10.0+)
- Mandrindra fampahatsiahivana fanombohana na fanarahana mpandray anjara tsy misy fampandrenesana miverimberina rehefa alefa ny vavahadin-fankatoavana sy ny firaketana fampahatsiahivana

**Soso-kevitra fanombohana:**
- *Manangana site portfolio momba ny fakana sary miaraka amin’ny karazana lahatsoratra gallery, pejy famandrihana, ary taratasy fifandraisana.*
- *Mamoròna tranonkala trano fisakafoanana miaraka amin’ny menu an-tserasera, ora fisokafana, ary taratasy fanontaniana famandrihana latabatra.*
- *Amboary site consulting freelance miaraka amin’ny pejin’ny tolotra, fizarana portfolio, ary bilaogy.*
- *Ampio taratasy fifandraisana amin’ny pejy Contact amin’ny alalan’ny site builder.*
- *Rehefa ankatoavina ny lisitra fanamarinana fanombohana site, alefaso fampahatsiahivana SMS amin’ilay fifandraisan’ny stakeholder voakirakira.*

### Design Studio

**Fifantohana:** Fanamboarana endrika hita maso — loko, typography, CSS, ary block patterns.

**Fitaovana misy:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Zavatra hainy tsara:**
- Mampihatra presets theme manana anarana (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- Manatsara typography maneran-tany sy palettes loko amin’ny alalan’ny theme.json
- Manindrona CSS manokana ho an’ny fanitsiana mifanaraka amin’ny marika
- Maka screenshot amin’ny pejy iray ary mandinika azy amin’ny olan’ny famolavolana

**Soso-kevitra fanombohana:**
- *Ampiharo ny preset warm-editorial ary avy eo apetraho ho #2d6a4f ny loko fototra.*
- *Makà screenshot amin’ny pejy fandraisana ary lazao ahy izay hatsarainao.*
- *Mamoròna hero block pattern azo ampiasaina indray miaraka amin’ny sary ambadika feno sakany sy lohateny afovoany.*

### Plugin Manager

**Fifantohana:** Mahita, mametraka, ary mitantana plugins WordPress.

**Fitaovana misy:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Zavatra hainy tsara:**
- Manolotra ny plugin tsara indrindra ho an’ny tranga fampiasana voalaza
- Mametraka ability packs avy amin’ny registry
- Mijery ny katalaogin’ny abilities misy araka ny sokajy

**Soso-kevitra fanombohana:**
- *Inona no plugin tsara indrindra ho an’ny membership directory?*
- *Apetraho ny WooCommerce abilities pack.*
- *Asehoy ahy ny ecommerce ability packs rehetra misy.*

### Support Assistant

**Fifantohana:** Mamaly fanontaniana momba ny atiny, ny fikirakirana, ary ny fikirakirana WordPress amin’ny site.

**Fitaovana misy:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Zavatra hainy atao tsara:**
- Mitady ny firafitra sy safidy ankehitriny amin’ny tranonkala
- Manazava hoe inona ny karazan-dahatsoratra, fanakilasiana, ary menio voarindra ao amin’ny tranonkala
- Mamaly fanontaniana hoe "inona no ataon’ity firafitra ity?" amin’ny famakiana sanda mivantana
- Miasa ho sosona fitiliana vakiana fotsiny alohan’ny hanaovana fanovana

**Soso-kevitra fanombohana:**
- *Inona avy ireo plugin sy firafitra miasa amin’izao fotoana izao amin’ity tranonkala ity?*
- *Tanisao avokoa ny karazan-dahatsoratra namboarina manokana voasoratra anarana amin’ity tranonkala ity.*
- *Inona avy ireo menio fitetezana misy ary aiza no nanendrena azy ireo?*

## Fampidirana automatique Superdav

Rehefa voarindra ny fampidirana Superdav AI Agent v1.18.0, afaka mandray anjara amin’ny fizotry ny asa automatique azo antoka kokoa sy mahafantatra fandaharam-potoana ireo agent tafiditra:

- **Fitaovana famakiana Google Calendar** mamela ny agent hijery ireo kalandrie sy hetsika voarindra alohan’ny hanoratana asa fanarahana.
- **Fampifandraisana contact sy mpanatrika** manampy hampifanaraka ny mpandray anjara amin’ny hetsika amin’ireo mpampiasa WordPress na contact fantatra.
- **Vavahadin’ny fankatoavan’olombelona** mampiato asa saro-pady mandra-pijeren’ny mpampiasa nahazo alalana sy hanamafisany azy.
- **Firaketana fampahatsiahivana** misoroka fampandrenesana miverimberina rehefa manandrana indray na miverina ny asa voalahatra.
- **Fampandrenesana SMS TextBee** mandefa hafatra an-tsoratra voarindra ihany rehefa alefa ny mari-pamantarana SMS sy ny fahazoan-dalana amin’ny fizotry ny asa.

Fizotry ny asa atolotra: angataho ny agent hanomana ny hafatra na asa, jereo ny fangatahana fankatoavana, avy eo avelao hitohy ilay asa nankatoavina. Ho an’ny fampahatsiahivana miverimberina, avelao ho mandeha ny fanesorana fampahatsiahivana dika mitovy mba tsy hampandrenesina imbetsaka ilay hetsika na contact iray ihany.

## Fanamboarana manokana ny Agents

Ny agent tafiditra tsirairay dia azo itarina na soloina amin’ny alalan’ny sivana `gratis_ai_agent_agents`.

### Manampy prompt system namboarina manokana amin’ny agent efa misy

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### Manoratra anarana agent vaovao

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

Miseho ao amin’ny Mpifidy Agent avy hatrany ilay agent vaovao rehefa mandeha ilay sivana.

### Manala agent tafiditra

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
