# Famoronana Kodina (v2)

_**FANAM-BAHANA MANAN-TENA: Io lahatsoratra ity dia manondro ny Ultimate Multisite version 2.x.**_

Amin'ny alalan'ny Ultimate Multisite, afaka mamorona code fanaovana resaka (discount codes) ianao mba hanomezana tombontsoa ho an'ny mpanjifa amin'ny fandraisana (subscriptions). Ary ny famoronana azy dia mora!

## Famoronana sy Fanovana Code Fanaovana Resaka

Mba hamoronana na hanova code fanaovana resaka, mandehana any **Ultimate Multisite > Discount Codes**.

![Listan'ny code fanaovana resaka — toerana tsy misy alohan'ny famoronana code rehetra](/img/config/discount-codes-empty.png)

Eo ianao dia hahita lisitra ireo code fanaovana resaka efa namoronanao.

Afaka tsindrio ny **Add Discount Code** mba hamoronana coupon vaovao na afaka manova ireo efa misy ianao amin'ny alalan'ny fipetraka eo aminy sy tsindriana ny **Edit**.

![Listan'ny code fanaovana resaka miaraka amin'ny fampiasana hover mampiseho ny rohy Edit sy Delete](/img/config/discount-codes-list-hover.png)

![Fampidirana bokotra Add Discount Code ao anatin'ny lohatenin'ny pejy](/img/config/discount-codes-add-button.png)

Ho rahitra ianao amin'ny pejy izay hamoronana na hanovana ny coupon code anao. Amin'ity ohatra ity dia hamorona vaovao isika.

![Pejy fanovana code fanaovana resaka miaraka amin'ny faritra rehetra hita](/img/config/discount-code-edit.png)

Andao hijery ireo fepetra azo atao eto:

**Enter Discount Code:** Io dia anaran'ilay code fanaovana resaka anao ihany. Tsy io no code izay ilain'ny mpanjifanao hampiasaina amin'ny forms fandraisana (checkout form).

**Description:** Eto, afaka manazava fohy ny antony nanaovana ity coupon ity ianao.

![Faritra anaran'ny code fanaovana resaka sy famaritana eo ambony pejy fanovana](/img/config/discount-code-description.png)

Afaka jerena koa ny code fanaovana resaka ho mavitrika (active) na tsy mavitrika (inactive):

![Toetran'ny code fanaovana resaka mavitrika](/img/config/discount-code-active.png)

**Coupon code:** Ity no toerana izay mamaritra ny code izay ilain'ny mpanjifanao hampidirina mandritra ny fandraisana (checkout).

![Coupon code field where customers enter the code at checkout](/img/config/discount-code-coupon-field.png)

**Rabona (Discount):** Eto, afaka ny **isany (percentage)** na **volana voafetra (fixed amount)** ho an'ny coupon code anao iantohana.

![Discount amount setting with percentage or fixed-amount dropdown](/img/config/discount-code-amount.png)

**Apetra amin'ny fanavaozana (Apply to renewals):** Raha tsy voafidy ity safidy ity dia ho azo ampiasaina ao anatin'ny **fandoavam-bolana voalohany** ihany ity coupon code ity. Ny fandoavam-bola hafa rehetra dia tsy hisy rabona. Raha voafidy ianao, dia ho mandeha ny coupon code amin'ny fandoavam-bola rehetra ho avy.

**Rabona ho an'ny fandoavana (Setup fee discount):** Raha tsy voafidy ity safidy ity dia **tsy hanome rabona na ho an'ny fandoavam-bolana (setup fee)** ny coupon code. Raha voafidy ianao, dia afaka mametraka ny rabona (isany na volana voafetra) izay hataon'ity coupon code ity amin'ny fandoavam-bola (setup fee) anao.

![Apply to renewals and setup fee discount toggle options](/img/config/discount-code-renewals.png)

**Mavitrika (Active):** Apetra na avela ho tsy mavitrika manokana ity coupon code ity.

![Active toggle to manually enable or disable the discount code](/img/config/discount-code-active.png)

Eo amin'ny **Advanced Options**, dia misy ireto fepetra ireto:

![Discount code advanced options](/img/config/discount-code-advanced.png)

**Fetran'ny fampiasana (Limit uses):**

  * **Fampiasana (Uses):** Eto, afaka jerenao hoe firy no nampiasaina ilay coupon code.

  * **Fetran'ny fampiasana (Max uses):** Ity dia hanome fetra ny isan'ny fotoana azo ampiasain'ny mpampiasa ity coupon code ity. Ohatra, raha mametraka ianao hoe 10 eto, dia azo ampiasaina ilay coupon code mandritra ny 10 fotoana ihany. Aorian'izay fetra io, tsy azo ampiasaina intsony ilay coupon code.

![Limit uses setting with current uses count and max uses field](/img/config/discount-code-limit-uses.png)

**Daty fanombohana sy fandroahana:** Eo no ianao afaka manampy daty fanombohana sy/na daty fandroahana ho an'ny coupon anao.

![Start and expiration date fields for scheduling the discount code](/img/config/discount-code-dates.png)

**Fametrahana vokatra:** Raha mandoro ny **Select products** ianao, dia hasehon'ny anao avokoa ny vokatra rehetra. Afaka misafidy manokana (amin'ny alalan'ny fanindriana na fanapahana azy) izay vokatra afaka mandray ity coupon code ity ianao. Ireo vokatra tsy voafidy eto dia tsy hisy fiovana raha toa ka miezaka mampiasa ity coupon code ity ny mpanjara anao.

![Limit products section with per-product toggle switches](/img/config/discount-code-limit-products.png)

Rehefa vita ny fametrahana ireo safidy rehetra ireo, tsindrio ny **Save Discount Code** mba hitahirizana ny coupon anao ary vita!

![Save Discount Code button at the bottom of the edit page](/img/config/discount-code-save.png)

Ny coupon dia eo amin'ny lisitra anao ary avy eo, afaka tsindrio ianao mba **hanovana na hanesorana** azy.

![Discount code row in the list with Edit and Delete hover actions](/img/config/discount-codes-list-hover.png)

### Fampiasana URL Parameters:

Raha tianao ny manova ny tabilao vidiny na manangana pejy coupon code tsara ho an'ny tranonkalanao ary te hampiasa coupon code amin'ny checkout form anao aho, dia afaka manao izany amin'ny alalan'ny URL parameters ianao.

Voalohany, mila mahazo ny link azo zaraina (shareable link) ho an'ny plan anao ianao. Mba hanaovana izany, mandehin'ny **Ultimate Multisite > Products** ary safidio ny plan iray.

Tsindrio ny bokotra **Click to Copy Shareable Link**. Izany dia hanome anao ny link azo zaraina ho an'ity plan ity. Amin'izay tranga nataontsika, ny link azo zaraina nomena dia [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/)_.

![Product page with shareable link button](/img/config/products-list.png)

Mba hampiasana ny kody fanaovana resaka (discount code) amin'ity plan ity, dia ampiana ny parameter **?discount_code=XXX** ao anatin'ny URL ianao. Izay **XXX** no kody fanaovana resaka (coupon code).

Ao anatin'ity ohatra ity, hampiasaina ny kody fanaovana resaka **50OFF** amin'ity vokatra manokana ity isika.

Ny URL ho an'ity plan manokana sy rehefa ampiasana ny kody fanaovana resaka 50OFF dia hitovy toy izao: [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/) _**?discount_code=50OFF**_.
