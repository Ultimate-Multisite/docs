# Fahasami ny fametrahana ny Sunrise File

Ny rakitra `sunrise.php` dia rakitra manokana izay mitady azy ny WordPress rehefa mandeha (bootstraps) izy. Mba hahafahana mahita ilay `sunrise.php` an'ny WordPress, tsy maintsy eo ao anatin'ny **wp-content folder** izy.

Rehefa mampiasa Ultimate Multisite ianao ary mandalo amin'ilay setup wizard toy ny voalaza ao amin'ilay sary, dia miezaka atao avy amin'i Ultimate Multisite ny kopiana ilay rakitra `sunrise.php` any anatin'ny wp-content folder.

<!-- Sary tsy azo jerena: Pejy setup wizard mampiseho ny dingana fametrahana sunrise.php -->

Amin'ny ankapobeny, mahavita kopiana ilay rakitra sy mandeha tsara isika. Na izany aza, raha misy zavatra tsy voarafitra tsara (ohatra, ny alalan'ny permissions an'ny folder), dia mety ho lasa toe-javatra izay tsy afaka manao kopiana ilay rakitra i Ultimate Multisite.

Raha vakina ny hafatra fahadisoana omen'i Ultimo anao, dia hahita fa izany no tena mitranga eto: **Sunrise copy failed** (Tsy nahomby ny kopiana Sunrise).

<!-- Sary tsy azo jerena: Hafatra fahadisoana mampiseho ny Sunrise copy failed -->

Mba hanamboarana izany, afaka manao fotsiny ianao hoe kopiana ilay rakitra `sunrise.php` ao anatin'ny folder plugin wp-ultimo ary apetaho ao anaty wp-content folder anao. Aorian'izay, avereno ny pejy wizard ary tokony ho tafiditra tsara ny fanamarinana rehetra.

<!-- Sary tsy azo jerena: File manager mampiseho ny sunrise.php ao anatin'ny folder plugin wp-ultimo --> Na inona na inona izany, mety mila manao fanamarinana ankapobeny momba ny permissions an'ny folder ianao mba hisorohana olana ho avy (tsy fotsiny amin'i Ultimate Multisite fa amin'ny plugin sy theme hafa koa).

Ny **Health Check tool** izay anisan'ny WordPress (afaka miditra azy ianao amin'ny alalan'ny **admin panel > Tools > Health Check** ao amin'ny site lehiben'ny anao) dia afaka manampy anao hahalala raha toa ka voarafitra tsara ny permissions an'ny folder izay mety hiteraka olana ho an'ny WordPress.
