# Fandrim-pahalalina amin'ny Plugin hafa fantatra

Aorian'ny fametrahana sy fanatanterahana ny Ultimate Multisite, mety hahatsirana olana ianao noho tsy voavaha ireo tsy fampifanarahana (incompatibilities) amin'ny plugin hafa.

Miasa mavitrika izahay mba hampihenana ity lisitra ity isaky ny famoahana vaovao, fa raha misy plugin voasoratra eto dia mety mila manapaka azy ianao mba hahatonga ny Ultimate Multisite Network ho mandeha tanteraka.

Ity misy lisitry ny ireo ary inona no tokony hataonao amin'izany:

**Ny plugin WPMU DEV sasany** Ny plugin sasany izay tsy voatantana intsony (no longer maintained) dia fantatra fa mahatonga ny fomba fametrahana Ultimate Multisite ho mijanona eo amin'ny dingana farany (tsy afaka mamorona kaonty ny mpampiasa). Ireto misy lisitry ny plugin izay miteraka ireo olana ireo:

  * Signup Language: _Misy fanampiny ao amin'ny Ultimate Multisite manome ny fomba fiasa mitovy, antsoina hoe Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Mba voatanteraka ao anatin'ny Ultimate Multisite izany, ka tsy mila plugin hafa intsony ianao._

  * New Blog Templates: _Mba voatanteraka ao anatin'ny Ultimate Multisite izany, ka tsy mila plugin hafa intsony ianao._

  * Set Password on Multisite Blog Creation: _Mba voatanteraka ao anatin'ny Ultimate Multisite izany, ka tsy mila plugin hafa intsony ianao._

  * Signup TOS: _Mba voatanteraka ao anatin'ny Ultimate Multisite izany, ka tsy mila plugin hafa intsony ianao._

  * Anti-Splog: Na dia tsy mahery loatra aza, dia _voatanteraka ao anatin'ny Ultimate Multisite koa_.

**Fanamarihana** : Ny plugin WPMU Dev hafa rehetra dia azo ampiasaina tsara miaraka amin'ny Ultimate Multisite. Ohatra, _Smush_ , _Forminator_ , _Defender,_ sns.
