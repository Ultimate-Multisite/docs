# Ahoana ny fampiasana Ultimate Multisite snippets ao amin'ny GitHub repository anay

Misy code snippets azo alaina ao amin'ny GitHub repository izay tena tadiavina matetika avy amin'ireo mpampiasa Ultimate Multisite izay te-hiresaka asa kely toy ny famafana script Google Analytics amin'ny pejy fandraisana (sign-up pages) na ny fanakiana meta box avy ao amin'ny admin dashboard.

Ity lahatsoratra ity dia hampiseho anareo ny fomba fampiasana azy na toerana tokony hapetraka ireo code ireo.

Afaka mahita ireo snippets ianao amin'ity rohy etsy ambany.

https://github.com/next-press/wp-ultimo-snippets/

Misy fomba 2 ahafahanao manampy ny code:

  1. Ao anatin'ny fichier functions.php an'ny theme-nao.

  2. Must-Use Plugins (mu-plugins)

# Ahoana no fanampiana ny snippet ao amin'ny fichier functions.php an'ny theme-nao.

  1. Midira ao amin'ny WordPress Network admin dashboard anao ary mandehana any Themes > Theme Editor (Jereo ny sary eto ambany).

  2. Eo amin'ny pejy "Edit Themes", dia ataovy azo antoka fa voafidy ny theme mampiasa anareo eo amin'ilay dropdown field izay eo an-davarana ambony ankavanana an'ny efijera anareo (#3 amin'ilay sary eto ambany).

  3. Kitiro ny fichier functions.php eo ambanin'ny fizarana "Theme Files" mba handefasana ilay fichier. Mandroso any ambany ary apetraho ao anatin'ilay snippet Ultimate Multisite azo avy amin'ny GitHub repository.

# Ahoana no famoronana Must-Use Plugins (mu-plugins)

Misy fahaiza-manao ao amin'ny WordPress ahafahanao mitondra asa manokana antsoina hoe "Must-Use Plugins", na "mu-plugins" raha fohy ny anarana.

Ireo mu-plugins manokana ireo dia apetraka alohan'ny plugin rehetra ara-dalàna, ary tsy azo ahena (deactivated) izy ireo. Ao amin'ny multisite network iray, ny code ao anatin'ireo mu-plugins ireo dia ho voaova amin'ny toerana rehetra ao amin'ny installation anareo.

1. Ampiasao FTP na SSH mba hidirana amin'ny rafitry ny fichier (filesystem) an'ny WordPress anareo.

2. WordPress install-nao, wp-content dikan'ny, dia ataovy folder vaovao antso hoe: mu-plugins.

<!-- Screenshot tsy misy: File manager mampiseho ny wp-content directory miaraka amin'ny folder mu-plugins -->

3. Ataovy file PHP vaovao eo amin'ny solosainao antso hoe wu-snippet.php, dia ampiasao Notepad na code editor anao.

4. Apetraka ao anatin'io file io ny Ultimate Multisite code snippet izay azo avy amin'ny GitHub repository ianao ary ataovy save. Afaka manampy ity code ity eo ambony ny code snippet koa ianao mba hanamarina hoe mu plugin no izany.
