# Ny fanamboarana ny Admin Panel-nao amin'ny PRO Theme

Matetika rehefa manao asa ho mpanjifa isika dia mila manome traikefa manokana antsika amin'ny dashboard an'ny WordPress ho an'ireo mpanjifantsika. Raha mitady endrika sy fihetseham-po matihanina kokoa ianao, ny PRO Theme dia afaka manampy anao.

Ny PRO Theme dia admin theme natao mba hanampiana anao rehefa manova ny Dashboard an'ireo mpanjifantsika. Ary izy koa dia ampahany amin'ny core plugin premium an'ny Ultimate Multisite.

Aorian'ny fametrahana azy, mandehin'ny pejy _Plugins_ ary tsindrio mba hampandehanana azy eo amin'ny Network-nao.

<!-- Screenshot tsy azo jerena: Pejy Plugins an'i WP mampiseho ny bokotra Activate ho an'ny PRO Theme ho an'ny Network -->

_Ampandeha azy manerana ny Network-nao eo amin'ny pejy Plugins_

_**FANAMARIHANA:**_ _Raha vao mametraka azy ianao, dia haseho ao amin'ny mpanjifanao ho safidy azo atao mba hametrahana ilay theme ho an'io mpampiasa io ihany._

Ity dia ny interface admin-nao izay novana tanteraka! Manana menu vaovao eo an-dakiana (left column) antsoina hoe PRO Theme ianao.
