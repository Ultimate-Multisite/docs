# Fampanana Rakibolana Fandaharam-pifandra an'ny Fandaharam-pifandra amin'ny tambajotra misy anareo amin'ny Ultimate Multisite: Ad Injector

Ny Ultimate Multisite: Ad Injector dia **plugin maimaim-poana** izay manome anao ny fitaovana mba hampidirana (place) fampahafantarana (ads) ao amin'ireo plan-nao mivantana avy amin'ny pejy fanovana azy.

Aorian'ny fametrahana ny Ultimate Multisite: Ad Injector, mandehana any amin'ny pejy _Plugins_ anao mba hampandehanana azy.

<!-- Screenshot tsy azo jerena: Pejy WP Plugins mampiseho ny bokotra Network Activate ho an'ny addon Ad Injector -->

_Tsindrio ny bokotra Network Activate_

Rehefa mandeha ny addon, ny _Advanced Options_ meta box eo amin'ny pejy fanovana plan dia hisy safidy vaovao antsoina hoe _Ad Settings_.

![Ad Settings eo amin'ny pejy fanovana plan](/img/config/product-edit-full.png)

_Ity dia afaka manova ny fepetra fampahafantarana ho an'ny plan tsirairay anareo ianao._

Amin'io tab io, afaka manampy kody fampahafantarana (ad codes) ho an'ny posts sy pages ao amin'ny front-end ianao, alohan'ny sy aorian'ny votoaty.

<!-- Screenshot tsy azo jerena: Feno kody fampahafantarana ho an'ny posts sy pages alohan'ny votoaty -->

<!-- Screenshot tsy azo jerena: Ohatra ny fisehoan'ny fampahafantarana amin'ny pejy post ho plan manokana -->

_Aorian'ny fametrahana ny kody fampahafantarana anao, dia hiseho eo amin'ny posts sy pages an'io plan io._

Afaka misy fampahafantarana (ads) ianao ao amin'ny lohateny (header) an'ny Admin Pages ho an'ny mpampiasa an'io plan io koa.

<!-- Screenshot tsy azo jerena: Safidy Ad Code ao amin'ny plan settings eo amin'ny Admin Panel -->

<!-- Screenshot tsy azo jerena: Ohatra ny fisehoan'ny fampahafantarana ao amin'ny admin panel an'ny mpampiasa -->

_Mandehana any amin'ny safidy Ad Code ao amin'ny Admin Panel raha te-hampiseho fampahafantarana eo amin'ny back-end an'ny mpampiasa ianao._

### Lehibe kokoa noho ny Fampahafantarana (Beyond Advertising)

Afaka mampiasa ny Ultimate Multisite: Ad Injector koa ianao mba hanomezana fanohanana ho an'ireo plan ambony indrindra.

Mandehana fotsiny any amin'ny safidy _Admin Panel Ads_ ary ampiana ny HTML code ho an'ny tool fanohananao.

Amin'izay fomba ity, afaka miantso anareo avy ao amin'ny Admin Panel izy ireo!
