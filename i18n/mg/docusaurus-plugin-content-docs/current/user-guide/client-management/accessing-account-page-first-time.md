# Ahoana no afaka miditra amin'ny pejy kaontinao ho an'ny voalohany? {#how-can-i-access-my-account-page-for-the-first-time}

Faly anareo ao amin'Ultimate Multisite!

Afaka miditra amin'ny pejy kaontinao amin'ny alalan'ilay link tonga tao amin'ny mailaka avy namidy ny plugin nataonay:

<!-- Screenshot unavailable: Account page email with access link -->

Ampiasao ilay mailaka izay nomenao nandritra ny fividianana sy ny teny (password) izay nomenao ao anatin'ilay mailaka.

<!-- Screenshot unavailable: Login page with email and password fields -->

Satria izao no voalohany ianao mandeha amin'ity kaontina ity, dia ho fantatra fa hovakiana ny fiovana ny teny (password) anao. Ovay ary tsindrio ny _Save_ button.

<!-- Screenshot unavailable: Change password form with Save button -->

Ao amin'ny pejy kaontinao, afaka mitantana ny antsipiriany rehetra momba ny kaontinao ianao toy ny fandoavam-bola (billings), fanavaozana (renewals), fandavana (cancellations), ary zavatra maro hafa.
