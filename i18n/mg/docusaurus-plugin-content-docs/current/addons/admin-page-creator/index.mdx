# Admin Page Creator

## Votoaty {#content}
Admin Page Creator dia hamela anao hanampy mora foana pejy admin vaovao sy zana-pejy ao anatin'ny WordPress admin dashboard. Azo ampiasaina hamoronana pejy fandraisana na pejy torolalana izay afaka miasa ho pejy fipetrahana ho an'ny mpampiasa vaovao izy io. Afaka mampiasa page builder toy ny Beaver Builder, Elementor, Brizy, ary Oxygen ianao handrafetana ireo pejy tsy misy fahasahiranana.

Manolotra fanohanana HTML, CSS ary JavaScript namboarina manokana ihany koa izy io, ary koa fanaraha-maso fidirana mifototra amin'ny plan sy ny anjara asan'ny mpampiasa!

## Endri-javatra {#features}
* Manohana HTML sy CSS namboarina manokana
* Fanohanana JavaScript
* Fomba Votoaty PHP
* Fahazoan-dalana mifototra amin'ny anjara asan'ny mpampiasa
* Fanohanana ho an'ny page builders

## Antontan-taratasy {#documentation}
[Famoronana Pejy Admin sy Zana-pejy Namboarina Manokana amin'ny fampiasana Multisite Ultimate: Admin Page Creator](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## Fametrahana {#installation}

1. Ampidiro ao amin'ny lahatahiry **/wp-content/plugins/** ny **ultimate-multisite-admin-page-creator**
2. Alefaso ny plugin amin'ny alalan'ny menio **Plugins** ao amin'ny WordPress Network Admin-nao

## Torolàlana Fampiasana {#usage-guide}

Na dia ataon'ny WordPress tena mora aza ny manampy votoaty amin'ny lafiny hita ivelany amin'ny sites, ny famoronana pejy sy zana-pejy amin'ny lafiny admin dia mitaky fahaiza-manao fototra amin'ny fanoratana code. Ultimate Multisite: Admin Page Creator dia mikendry ny hitondra io fahatsoran'ny fampiasana amin'ny lafiny hita ivelany io ho amin'ny lafiny admin amin'ny network sites-nao.

## Nahoana no mampiasa Admin Page Creator? {#why-use-admin-page-creator}

Angamba ianao te hanolotra pejy namboarina manokana ao amin'ny tontonana admin an'ny clients-nao misy torolalana, angamba ianao te hanampy pejy namboarina manokana misy FAQ sy contact widget. Tsy misy fetra ny zavatra azo atao amin'ny Admin Page Creator, ary izany no hojerentsika amin'ny torolalana androany.

## Mamorona ny Pejy Admin Namboarina Manokana voalohany {#creating-your-first-custom-admin-page}

Rehefa avy mametraka sy mampandeha ny Ultimate Multisite: Admin Page Creator, mandehana any amin'ny menio network admin-nao ary mankanesa amin'ny Admin Pages.

![Lisitry ny Pejy Admin Namboarina Manokana ao amin'ny Network Admin](/img/admin/admin-pages-list.png)

_Lisitry ny Pejy Admin Namboarina Manokana ao amin'ny Network Admin_

Ho hitanao ny lisitry ny pejy admin rehetra noforoninao hatreto (satria fitsidihanao voalohany ity, ho foana ny lisitra). Mandrosoa ary tsindrio **Add new Admin Page**. Hafindra any amin'ny mpamoaka Admin Page ianao.

## Safidin'ny Menio {#menu-options}

Eo amin'ny sisiny, ao amin'ny meta-box General Options, ho afaka hanamboatra ny karazana menio, mari-panondroana, ary maro hafa ianao.

### Karazana Menio {#menu-types}

Ny pejy admin namboarinao manokana vaovao dia afaka manana karazana menio roa samihafa: menio ambaratonga ambony sy zana-menio.

Ny pejy menio ambaratonga ambony, araka ny anarany, dia hanampy ny pejinao vao noforonina ao amin'ny menio admin miaraka amin'ireo pejy WordPress mahazatra.

Raha misafidy ny safidy ambaratonga ambony ianao, ho afaka hisafidy ny filaharan'ny menio sy kisary menio ihany koa.

Raha tianao hatao pejy zana-menio ity pejy ity, ho afaka hisafidy pejy ray aman-dreny hampifandraisana ity pejy ity aminy ianao. Afaka manampy ny pejinao namboarina manokana amin'ny pejy WordPress mahazatra na amin'ireo pejy admin namboarina manokana ambaratonga ambony anao ianao.

<!-- Pikantsary tsy misy: Safidy karazana menio Admin Page Creator miaraka amin'ny firafitry ny mari-panondroana, filaharana, ary kisary -->

_Amboary tanteraka ny singa menio ho an'ity Pejy Namboarina Manokana ity_

_Afaka manampy ity pejy ity ho zana-menio an'ny pejy menio hafa ihany koa ianao, ao anatin'izany ireo ambaratonga ambony namboarinao manokana._

## Safidy Mandroso {#advanced-options}

Tokony ho afaka hanampy karazana votoaty rehetra amin'ireo pejy admin namboarinao manokana ianao. Tafiditra amin'izany ny fitsipika sy rakitra CSS, JavaScript, ary koa tranomboky ivelany. Afaka manao izany ianao amin'ny fampiasana ny Takelaka Advanced Options.

### CSS Namboarina Manokana {#custom-css}

Ao amin'ny takelaka CSS, afaka manampy ny code CSS namboarinao manokana ianao, ary koa manafatra rakitra CSS avy amin'ny sites lavitra (toy ny CDNs, Google Fonts, sns).

<!-- Pikantsary tsy misy: Mpamoaka CSS namboarina manokana Admin Page Creator miaraka amin'ny fanafarana stylesheet ivelany -->

_Tohanana ihany koa ny CSS namboarina manokana. Afaka mampiditra rakitra endrika ivelany ihany koa ianao!_

### JavaScript Namboarina Manokana {#custom-javascript}

Toy izany koa no manan-kery amin'ny JavaScript. Afaka manampy tranomboky ivelany hampiasaina amin'ny code JavaScript namboarinao manokana ianao.

<!-- Pikantsary tsy misy: Mpamoaka JavaScript namboarina manokana Admin Page Creator miaraka amin'ny fanafarana tranomboky ivelany -->

_Ampio ny code JavaScript-nao manokana!_

## Fahazoan-dalana {#permissions}

Ao amin'ny Advanced Options ihany koa, ho afaka hisafidy izay sub-sites hanana fidirana amin'ity pejy ity ianao. Mahasoa izany raha mikasa hanolotra votoaty manokana samihafa ho an'ny ambaratonga plan samihafa ianao, ohatra.

<!-- Pikantsary tsy misy: Firafitry ny fahazoan-dalana Admin Page Creator ho an'ny fanaraha-maso fidirana mifototra amin'ny plan sy anjara asa -->

_Manana fifehezana feno amin'izay toerana hisehoan'ity pejy ity ianao_

**Zava-dehibe** : Tsy maintsy feno ireo fepetra roa ireo vao aseho ilay pejy. Midika izany fa raha client amin’ny plan iray ny mpampiasa, nefa tsy manana iray amin’ireo anjara voafantina (na ny mifamadika amin’izany), dia tsy hiseho eo amin’ny panely-ny ilay pejy.

## Mavitrika {#active}

Raha te-hanafoana pejy admin namboarina manokana ianao nefa tsy mamafa azy na mikitikitika ny fahazoan-dalana, dia esory fotsiny ny marika amin’ny safidy Mavitrika.

Ny fanafoanana pejy ambaratonga ambony dia hanafoana ho azy ny zana-pejy rehetra mifandray amin’io pejy reny io.
