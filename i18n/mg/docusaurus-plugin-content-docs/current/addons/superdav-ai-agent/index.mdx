# Superdav AI Agent

Superdav AI Agent dia mpanampy AI WordPress mandroso izay manitatra ny Gratis AI Agent amin’ny fahaiza-manao mahafantatra lohahevitra, fahaiza-mikaroka amin’ny aterineto nohatsaraina, endri-pananganana tranonkala, ary fampidirana automatique tsy voatery atao. Ny kinova 1.18.0 dia manampy fitaovana famakiana Google Calendar, fampifandraisana contact sy mpanatrika, vavahadin’ny fankatoavan’olombelona, fanesorana fampahatsiahivana miverimberina, fampahafantarana SMS TextBee, famatsiana serivisy Superdav tantanana, ary fonosana mpiara-dia Advanced zaraina misaraka ho an’ny fizotran’asa mpamorona atokisana.

## Endri-javatra fototra {#key-features}

- **Fahaiza-manao mahafantatra lohahevitra** — mampifanaraka ho azy ny torolalana sy ny fahaiza-manao arakaraka ny lohahevitra WordPress mavitrika sy ny fanitarana napetrakao
- **Mpanome fikarohana aterineto maro** — misafidiana eo amin’ny Brave Search sy Tavily hahazoana valim-pikarohana manankarena kokoa sy fahaiza-manao fikarohana mandroso
- **Taratasy contact ho an’ny mpanangana tranonkala** — mamoròna taratasy contact miasa tanteraka mivantana avy amin’ny interface chat amin’ny alalan’ny fanitarana taratasy napetrakao
- **Mpanangana tranonkala nohatsaraina** — drafitra fananganana tranonkala misy dingana maro nohatsaraina miaraka amin’ny fahitana fanitarana tsara kokoa sy fanarenana hadisoana
- **Fahaiza-manao lohahevitra anatiny** — torolalana manokana ho an’ny Lohahevitra Block, Lohahevitra Klasika, Kadence Blocks, ary Kadence Theme
- **Fampidirana WooCommerce nohatsaraina** — naverina nalamina hampiasa API WooCommerce natoraly ho an’ny fahatokisana sy fifanarahana tsara kokoa
- **Fahitana mpanome automatique** — mahita sy manavao ho azy ireo fanitarana sy lohahevitra misy rehefa ahetsika na ajanona izy ireo
- **Famoronana logo manokana** — mamoròna sy ampidiro logo SVG manokana miaraka amin’ny fanadiovana automatique azo antoka amin’ny namespace
- **Fanamarinana fifanoheran’ny palette** — jereo ny tsiroaroa loko amin’ny fanarahana ny fepetra fidirana WCAG alohan’ny hampiharana azy amin’ny lohahevitrao
- **Fahitana nohatsaraina amin’ny sary** — mampakara sary mandritra ny dinidinika fikarohana Theme Builder mba hisy fiantraikany amin’ny lalana famolavolana
- **Menio ho an’ny fandraisam-bahiny** — mamoròna pejy menio sakafo sy zava-pisotro voarafitra ho an’ny trano fisakafoanana, kafe, ary orinasa fandraisam-bahiny
- **Topi-maso desktop sy finday** — jereo mialoha ny famolavolanao amin’ny fitaovana desktop sy finday mandritra ny fifantenana lalana famolavolana
- **Fakàna votoatin-tranonkala Tier 1** — sd-ai-agent/site-scrape dia misy amin’ny alàlan’ny default izao ho an’ny famakafakana tranonkalan’ny mpifaninana sy tranonkala aingam-panahy
- **Fitaovana famakiana Google Calendar** — mamaky kalandrie sy hetsika voarindra mba hahafahan’ny automatique mandinika ny tontolon’ny fandaharam-potoana ho avy
- **Fampifandraisana contact sy mpanatrika** — mampifanaraka ireo mpandray anjara amin’ny kalandrie amin’ireo mpampiasa sy contact fantatra ao amin’ny tranonkala alohan’ny hamoronana asa fanarahana
- **Vavahadin’ny fankatoavan’olombelona** — mampiato automatique saro-pady hojerena alohan’ny handefasana hafatra, hanohizana asa, na hanaovana fanovana
- **Rakitra fampahatsiahivana sy fanesorana dika mitovy** — manara-maso ireo fampahatsiahivana efa nalefa mba tsy hampandrenesana indroa ilay mpandray iray ihany rehefa mandeha miverina ny automatique
- **Fampahafantarana SMS TextBee** — mandefa fampahafantarana hafatra an-tsoratra voarindra amin’ny alalan’ny TextBee rehefa alefa ny mpanome SMS
- **Famatsiana serivisy Superdav tantanana** — mampifandray tranonkala tohanana amin’ny endpoint Superdav ampiantranoina miaraka amin’ny fametrahana automatique fa tsy fanamboarana endpoint an-tanana
- **Fonosana mpiara-dia Advanced** — fitaovana mpamorona atokisana zaraina misaraka ho an’ny rafitra rakitra, tahiry angona, WP-CLI, REST dispatcher, mpanangana fanitarana, git snapshot, fitantanana mpampiasa, ary fizotran’asa benchmark

## Fepetra takiana {#requirements}

- WordPress 6.0 na avo kokoa
- PHP 8.1 na avo kokoa
- WordPress Multisite (ahetsika amin’ny tambajotra) na fametrahana tranonkala tokana
- Fanitarana mpampifandray mpanome AI mavitrika (Anthropic, OpenAI, na mifanaraka)
- Gratis AI Agent v1.9.0 na avo kokoa (Superdav AI Agent manitatra ny Gratis AI Agent)

## Fametrahana {#installation}

1. Ampakaro any amin’ny `/wp-content/plugins/superdav-ai-agent` ireo rakitra fanitarana
2. Alefaso ny fanitarana amin’ny alalan’ny efijery **Fanitarana** ao amin’ny WordPress
3. Ataovy azo antoka fa alefa ihany koa ny Gratis AI Agent
4. Mankanesa any amin’ny **Gratis AI Agent → Fanamboarana** ary amboary araka izay ilaina ny mpanome fikarohana aterineto, fifandraisana serivisy Superdav tantanana, Google Calendar, na mari-pamantarana TextBee

Ny fonosana WordPress.org dia ahitana ireo endri-javatra fototra Superdav AI Agent. Ny fonosana mpiara-dia Advanced dia amboarina sy zaraina misaraka ho an’ny tontolo izay ametrahana sy anomezan-dalana mazava ireo fitaovana mpamorona atokisana.

Rehefa alefa, Superdav AI Agent dia manatsara ho azy ny fametrahana Gratis AI Agent anao amin’ny fahaiza-manao mahafantatra lohahevitra sy fahaiza-manao fanampiny. Sokafy ny tontonana **Gratis AI Agent** ao amin’ny sisiny admin WordPress ary atombohy ampiasaina ireo endri-javatra vaovao.

**Ohatra toromarika:**

```
Manampia taratasy contact ao amin’ny pejy Contact.
```

```
Karohy amin’ny tranonkala ireo fomba fanao tsara indrindra farany momba ny fiarovana WordPress.
```

```
Mamoròna fizarana hero amin’ny alalan’ny lamina block an’ny lohahevitra mavitrika.
```

```
Avereno jerena ireo hetsika Google Calendar voarindra rahampitso ary manorata fampahatsiahivana ho an’ny mpanatrika.
```

```
Mandefasa fampahatsiahivana SMS rehefa ankatoaviko ny hafatra.
```

## Mpanome fikarohana aterineto {#internet-search-providers}

Superdav AI Agent v1.10.0 dia manampy **Tavily** ho safidy hafa ankoatra ny Brave Search ho an’ny fangatahana fikarohana amin’ny aterineto. Samy manolotra valim-pikarohana amin’ny fotoana tena izy ireo mpanome roa ireo ary azo amboarina ao amin’ny **Gratis AI Agent → Fanamboarana → Mandroso**.

| Mpanome | Endri-javatra | Fanamboarana |
|---|---|---|
| **Brave Search** | Valin-karoka haingana, mifantoka amin’ny fiainana manokana | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Valin-karoka manankarena kokoa, fahaiza-manao fikarohana mandroso | [Tavily API](https://tavily.com/) |

Jereo ny [Fikarohana amin’ny Internet](../../user-guide/configuration/internet-search) ho an’ny torolalana fanamboarana.

## Fikorianan’asa Kalandrie, Fankatoavana, ary Fampahafantarana {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 dia manitatra ny fikorianan’asa automatique mihoatra ny asa votoaty sy fananganana site:

- **Fitaovana Google Calendar** dia mamela ny agent hamaky kalandrie voarindra sy antsipirian-javatra momba ny hetsika rehefa misy ny mari-pamantarana kalandrie.
- **Mpanampy fifandraisana sy mpanatrika** dia mampifandray ny mpandray anjara amin’ny hetsika amin’ny mpampiasa WordPress na fifandraisana fantatra alohan’ny hanomanana fampahatsiahivana na asa fanarahana.
- **Vavahady fankatoavan’olombelona** dia mamela automatique iray hiato mandra-pandinihan’ny mpampiasa nahazo alalana ilay hetsika naroso.
- **Rakitsoratra fampahatsiahivana** dia misoroka fampahatsiahivana kalandrie na fampahafantarana miverimberina rehefa averina andramana na voalahatra miverimberina ny automatique.
- **Fanohanana TextBee SMS** dia mamela fampahafantarana amin’ny hafatra an-tsoratra rehefa voarindra ny mpanome TextBee.

Ampiasao ireo fitaovana ireo ho an’ny fanampiana mahafantatra fandaharam-potoana, fanarahana mpanatrika, ary fikorianan’asa fampahatsiahivana izay mbola mitaky fandinihan’olombelona alohan’ny handefasana hafatra saro-pady.

## Fanamboarana {#configuration}

Ao ambanin’ny **Gratis AI Agent → Settings** avokoa ny fikirana rehetra ao amin’ny admin WordPress. Superdav AI Agent dia manampy ireto safidy fanamboarana manaraka ireto:

| Fikirana | Famaritana |
|---|---|
| **Mpanome Fikarohana amin’ny Internet** | Misafidiana eo amin’ny Brave Search sy Tavily (tsy voatery) |
| **Tavily API Key** | API key ho an’ny fikarohana amin’ny Internet Tavily (raha Tavily no voafidy) |
| **Serivisy Superdav Tantanana** | Ampifandraiso amin’ny serivisy Superdav ampiantranoina ny site tohanana miaraka amin’ny fanomezana endpoint automatique |
| **Mari-pamantarana Google Calendar** | Amboary ny fidirana kalandrie mba hahafahan’ny agent mamaky kalandrie sy hetsika |
| **Mpanome TextBee SMS** | Amboary ny mari-pamantarana TextBee ho an’ny fikorianan’asa fampahafantarana SMS nankatoavina |

## Fonosana Mpiara-miasa Mandroso {#advanced-companion-package}

Superdav AI Agent v1.18.0 dia manampy fonosana mpiara-miasa Advanced zaraina misaraka ho an’ny fikorianan’asan’ny developer azo itokisana. Ireo fahaiza-manao ireo dia tsy tafiditra ao amin’ny fonosana fototra WordPress.org. Apetraho amin’ny site izay itokisan’ny administratera mazava tsara ny agent hiditra amin’ny fitaovan’ny developer ihany ny fonosana mpiara-miasa.

Fehezo ho an’ny administratera azo itokisana ireo fitaovana ireo ary diniho ny fanamboarana approval-gate alohan’ny hamelana azy amin’ny site production.
