# AI Site Builder

ultimate-multisite-ai-site-builder

## Fepetra takiana

- WordPress 5.3 na ambony
- PHP 7.4 na ambony
- Plugin Multisite Ultimate (mavitrika)

## Fametrahana

1. Ampakaro ao amin’ny lahatahiry `/wp-content/plugins/` ny rakitra addon
2. Alefaso amin’ny alalan’ny menio 'Plugins' ao amin’ny WordPress ilay plugin
3. Amboary ao amin’ny admin tambajotra ny fikirakiran’ilay addon

## Fampivoarana

### Fanamboarana

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Rafitry ny tetikasa

```
ultimate-multisite-ai-site-builder/
├── inc/                          # Kilasy PHP sy rakitra ampidirina
│   ├── class-multisite-ultimate-updater.php  # Mpitantana fanavaozana
│   └── class-ai_site_builder-main.php             # Fiasa fototra
├── tests/                        # Fitsapana singa
│   ├── bootstrap.php            # Fanombohana fitsapana
│   ├── class-*-base.php         # Kilasy fitsapana fototra
│   └── class-*-test.php         # Tranga fitsapana
├── views/                       # Rakitra môdely
├── assets/                      # Loharano raikitra
│   ├── css/                     # Stylesheets
│   ├── js/                      # Rakitra JavaScript
│   └── images/                  # Sary
├── lang/                        # Rakitra fiteny
├── composer.json                # Fiankinan-doha PHP
├── package.json                 # Fiankinan-doha Node.js
├── phpunit.xml.dist             # Fikirakirana PHPUnit
├── .phpcs.xml.dist              # Fikirakirana fenitry ny code
└── rector.php                   # Fitsipika fanavaozana code
```

### Fitsapana

Ahitana rafitra fitsapana feno ilay addon:

- **Fitsapana singa**: Fitsapana mifototra amin’ny PHPUnit miaraka amin’ny WordPress test suite
- **Kilasy fitsapana fototra**: Manome fomba mpanampy ho an’ny toe-javatra fitsapana mahazatra
- **Fandrakofana fitsapana**: Voakirakira hanara-maso ny fandrakofana code
- **Fitsapana Multisite**: Mihazakazaka ao amin’ny tontolo multisite ny fitsapana

### Fenitry ny code

- **WordPress Coding Standards**: Ampiharina amin’ny alalan’ny PHPCS
- **Fifanarahana amin’ny PHP 7.4+**: Tohanana ny endri-javatra PHP maoderina
- **Fanadihadiana statika**: Fampidirana PHPStan ho an’ny fanamarinana karazana
- **Fanavaozana code**: Fitsipika Rector ho an’ny fanavaozana PHP

### Rafitra fananganana

- **Fanodinana assets**: Fampihenana haben’ny rakitra CSS/JS
- **Fandikan-teny**: Famoronana rakitra POT
- **Fonosana**: Famoronana arisiva hozaraina
- **Fomba fampivoarana**: Assets tsy nohafohezina ho an’ny debugging

## Fikirakirana

Manohana safidy fikirakirana isan-karazany azo idirana amin’ny alalan’ny interface admin tambajotra ilay addon.

## Hooks sy Filters

### Actions

- `ultimate-multisite-ai-site-builder_init` - Alefa aorian’ny fanombohana ilay addon
- `ultimate-multisite-ai-site-builder_loaded` - Alefa rehefa voapetraka avokoa ny rakitra addon rehetra

### Filters

- `ultimate-multisite-ai-site-builder_settings` - Ovay ny fikirakiran’ilay addon
- `ultimate-multisite-ai-site-builder_enabled` - Soloy ny satan’ny fampandehanana/fampitsaharana ilay addon
