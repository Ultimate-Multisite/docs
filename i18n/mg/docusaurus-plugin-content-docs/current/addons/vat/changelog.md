# Tantan’ny fanovana VAT {#vat-changelog}

Version 1.0.7 - Navoaka tamin’ny 2026-02-03

* Fanamboarana: Tsy nahomby tamin’ny fanamarinana ny laharana VAT Alemana (DE) noho ny lesoka MS_MAX_CONCURRENT_REQ an’ny serivisy VIES SOAP. Nafindra avy amin’ny SOAP API taloha ho amin’ny EU VIES REST API ofisialy mba hahazoana fanamarinana azo antoka kokoa.
* Fanamboarana: Ekena ankehitriny ny laharana VAT rehefa tsy misy vonjimaika ny serivisy VIES, fa tsy lavina diso. Foronina ny firaketana log hojerena aoriana.
* Fanamboarana: Hita ankehitriny ho an’ny firenena rehetra ny saha VAT/Tax ID, fa tsy ho an’ny mpikambana ao amin’ny EU ihany. Tehirizina ny tax ID tsy EU (oh: laharana CHE Soisa) mba haseho amin’ny faktiora tsy misy fanamarinana VIES.
* Fanamboarana: Tapaka ny fikarohana prefix VAT ho an’i Gresy (GR), Monaco (MC), ary Isle of Man (IM) noho ny lojikan’ny fikarohana array tsy marina.
* Fanatsarana: Nesorina ny dev dependency wp-cli-bundle tsy ilaina izay mety hiteraka lesoka mahafaty rehefa ampiasaina miaraka amin’ny addon Site Exporter.
* Fanatsarana: Nampiana andiana unit test feno (fitsapana 53).

Version 1.0.6 - Navoaka tamin’ny 2026-01-25

* Fanamboarana: Mampandoa VAT araka ny tokony ho izy ankehitriny ny fifanakalozana B2B an-toerana. Araka ny fitsipika EU VAT, mihatra amin’ny fifanakalozana B2B miampita sisintany ihany ny reverse charge, fa tsy rehefa mitovy amin’ny firenen’ny orinasa ny firenen’ny mpanjifa.

* Fanamboarana: Addon tsy miakatra na tsy miasa araka ny tokony ho izy.
* Fanovana: Nafindra ny loharanon’ny angona tahan’ny VAT avy amin’ny euvatrates.com ho amin’ny tahiry ibericode/vat-rates mba hahazoana angona azo antoka kokoa sy tazomina mavitrika.
* Fanamboarana: Nahitsy ny diso fanoratana ny sandan’ny safidy super_reduced_rates.
* Fanatsarana: Nampiana fanamarinana null ho an’ny fikirakirana country code mba hisorohana lesoka.
* Fanovana: Nesorina ny rakitra fandikan-teny nampidirina niaraka, ka nosoloina fanavaozana mandeha ho azy amin’ny alalan’ny Traduttore.
