# Mpifantina Fiteny

Ultimate Multisite: Mpifantina Fiteny amin'ny fisoratana anarana

Ampio saha mpifantina fiteny amin'ny fizotry ny fisoratana anarana.

## Fametrahana

1. Ampidiro ny 'ultimate-multisite-language-selector' ao amin'ny lahatahiry '/wp-content/plugins/'
2. Alefaso ny plugin amin'ny alalan'ny menio 'Plugins' ao amin'ny WordPress Network Admin anao

## Torolàlana fampiasana

Amin'ity add-on maimaim-poana ity dia azonao avela hifidy ny fiteny tiany ho hita amin'ny fizotry ny fisoratana anarana sy amin'ny pejin'ny fidirana ny mpampiasa anao.

Rehefa avy [mampihetsika ny license-nao](../../user-guide/getting-started/installing-ultimate-multisite.md) ianao dia ho afaka hametraka add-ons Ultimate Multisite izay omen'ny license-nao fidirana.

Mankanesa any amin'ny pejin'ny fikirana Ultimate Multisite anao ary avy eo, ao amin'ny menio eo ankavia, tsindrio ny marika _Add-ons_.

![Pejy Add-ons](/img/admin/addons-page.png)

_Mandehana any amin'ny pejy Add-ons hanombohana mametraka Mpifantina Fiteny_

Ao, ho hitanao ny add-ons rehetra atolotray.

Ny add-ons maimaim-poana dia azon'ny mpanjifanay rehetra ampiasaina, na inona na inona plan subscription nisoratany.

Tsindrio ny bokotra _Install it Now!_ eo ambanin'ny sary madinin'ilay add-on. Avy eo ampandehano izany ao amin'ny pejy Plugins.

![Pejy Add-ons misy bokotra fametrahana](/img/admin/addons-page.png)

_Tsindrio ny bokotra hametrahana ilay add-on_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_Ampandehano izany ao amin'ny pejy Plugins anao_

Ankehitriny, mandehana any amin'ny menio Settings an'ny Ultimate Multisite ary midira ao amin'ny kiheba _Network Settings_. Ao, eo amin'ny faran'ny pejy, hahita fizarana vaovao antsoina hoe _Languages_ ianao.

![Pejy Settings](/img/config/settings-general.png)

_Miseho ny fizarana vaovao tanteraka!_

Azonao avela ho banga izany mba hampisehoana ny fiteny rehetra ho safidy, na mifidy izay tianao hatao ho azo ampiasaina. Ireo fiteny ireo dia ireo napetrakao ao amin'ny WordPress set up anao.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_Lisitry ny fiteny azo ampiasaina amin'ity WP install ity_

Rehefa avy mitahiry ny fanovana nataonao ianao, dia haseho ao amin'ny pejy Login ary koa ao amin'ny Sign-up Flow ny menio fifantenana misy ireo fiteny nofidinao.

![Fikirana Login sy Fisoratana anarana](/img/config/settings-login-registration.png)

_Mpifantina Fiteny ao amin'ny Pejy Login_

![Pejy fisoratana anarana](/img/config/registration-page.png)

_Mpifantina Fiteny ao amin'ny Sign-up Flow_

## Fametrahana ny fiteny amin'ny alalan'ny URL

Azonao ovaina ihany koa ny fitenin'ny pejy iray amin'ny fampandalovana ny masontsivana locale mankany amin'ny URL. Raha te hampitodika any amin'ny pejy fisoratana anarana amin'ny teny italianina ianao, ohatra, ampio ?locale=it_IT amin'ny URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Tsy maintsy apetraka ao amin'ny WordPress network anao ilay fiteny mba hiasa izany.
