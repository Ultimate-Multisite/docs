# Content Sync {#content-sync}

Multisite Ultimate: Content Sync

Ity addon ho an'ny Multisite Ultimate ity dia mamela ny mpitantana tambajotra hampifandrindra votoaty avy amin'izay site rehetra mankany amin'ny site hafa ao amin'ny tambajotra. Manohana ny karazana lahatsoratra rehetra izy, anisan'izany ny lahatsoratra, pejy, custom post types, ary Elementor templates. Ny addon dia ahitana interface wizard misy dingana maro hisafidianana votoaty sy site kendrena, miaraka amin'ny batch processing sy real-time progress feedback.

## Endri-javatra {#features}

* Fifantenana votoaty: Grid karatra hita maso misy sary madinika sy sivana karazana lahatsoratra
* Karazana lahatsoratra rehetra: Mampifandrindra lahatsoratra, pejy, vokatra, Elementor templates, ary izay custom post type rehetra
* Fifantenana site: Safidio izay site hampifandrindrana votoaty, miaraka amin'ny fanilihana site loharano sy fisafidianana mialoha ny site zanaka
* Batch Processing: Mampifandrindra amin'ny site 10 isaky ny mandeha miaraka amin'ny fanavaozam-pandrosoana
* Fampifandrindrana media: Mandika sy manamboatra rohy media files (sary, horonan-tsary) avy amin'ny votoatin'ny lahatsoratra mankany amin'ny site kendrena ho azy
* Fampifandrindrana taxonomy: Mitahiry sokajy, tags, ary custom taxonomies
* Fanohanana Elementor: Fanohanana feno ho an'ny Elementor templates miaraka amin'ny famoronana indray CSS sy fanamboarana mapping ny template ID
* Famahana fifandirana: Safidy hanolo votoaty efa misy na hitsipaka azy
* Fitantanana cache: Manadio ho azy ny cache Elementor amin'ny site kendrena aorian'ny sync

## Fepetra takiana {#requirements}

* WordPress 5.3 na ambony
* PHP 7.4 na ambony
* Multisite Ultimate plugin (mavitrika)
* Elementor plugin (tsy voatery, ilaina amin'ny fampifandrindrana Elementor template)

## Fametrahana {#installation}

1. Ampidiro ny 'ultimate-multisite-content-sync' ao amin'ny lahatahiry '/wp-content/plugins/'
2. Ampandehano ny plugin amin'ny alalan'ny menu 'Plugins' ao amin'ny WordPress Network Admin-nao
3. Amboary ny fikirana ao amin'ny Ultimate Multisite → Settings → Content Sync
4. Mandehana any amin'ny Ultimate Multisite → Content Sync hanombohana fampifandrindrana
