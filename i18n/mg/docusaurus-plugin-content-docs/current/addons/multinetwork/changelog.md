# Diarin’ny fiovan’ny Multi-Network {#multi-network-changelog}

### 1.3.0 {#130}
* Vaovao: Mpijery mialoha lasitra tambajotra — zahao sy jereo mialoha ireo lasitra tambajotra miaraka amin’ny tontonana mivantana alohan’ny hividianana.
* Vaovao: Mpifidy lasitra tambajotra amin’ny checkout — afaka misafidy lasitra tambajotra ny mpanjifa mandritra ny fizotran’ny checkout fisoratana anarana.
* Vaovao: SSO magic link miampita tambajotra — afaka miditra amin’ny tambajotra madinika tsy misy sakana ny mpanjifa amin’ny alalan’ny fanamarinana magic-link.
* Vaovao: Fandikana media tambajotra — adika ara-dalàna ny media an’ny site rehefa mandika tambajotra avy amin’ny lasitra.
* Fanitsiana: Miasa izao ny fandikana tambajotra na tsy misy ny addon multi-tenancy; omena indray ny tombontsoan’ny super admin aorian’ny clone.
* Fanitsiana: Mifanaraka amin’ny WooCommerce sy ny vavahadin-karama rehetra izao ny famoronana tambajotra.
* Fanitsiana: Ampidirina izao ny laharan’ny port amin’ny famoronana domain fallback.
* Fanitsiana: Nafindra ho stylesheet enqueued ny CSS an’ny mpifidy lasitra mba hiseho tsara amin’ny theme rehetra.
* Fanitsiana: Fiarovana amin’ny key `network_id` tsy voafaritra ao amin’ny sehatry ny query an’ny site.
* Fanitsiana: Ny fanamarinana fahazoan-dàlana amin’ny tambajotra alohan’ny fanovana contexte dia misoroka hadisoana amin’ny property tabilao `wpdb`.
