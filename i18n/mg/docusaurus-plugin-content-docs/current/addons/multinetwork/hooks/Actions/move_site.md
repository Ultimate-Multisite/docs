# Action: move_site

Mipoitra rehefa nafindra ho amin’ny tambajotra vaovao ny site iray.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $site_id | `int` | ID-n’ilay site nafindra. |
| $old_network_id | `int` | ID-n’ilay tambajotra niavian’ny site. |
| $new_network_id | `int` | ID-n’ilay tambajotra namindrana ny site. |
