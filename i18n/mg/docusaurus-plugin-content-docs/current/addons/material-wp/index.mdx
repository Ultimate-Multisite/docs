# Material WP

Efa ela be no tsy niova ny seha-pitantanana WordPress. Saingy efa mba nieritreritra ve ianao hoe hanao ahoana ny endriky ny kinova Material Design an’ilay seha-pitantanana? 

Material WP dia lohahevitra admin novolavolaina hanome anao io endrika sy fahatsapana io, manampy anao hanolotra Dashboard namboarina manokana kokoa ho an’ny mpanjifanao.

## Toetoetra

- Fanohanana CSS namboarina manokana
- Fikirakirana endrika mora
- Fandaminana indray ny menio
- Fanafarana/fanondranana mora

## Antontan-taratasy

**Fanamboarana manokana ny Panel Admin-nao amin’ny Material WP**

## Fametrahana

1. Ampidiro ao amin’ny lahatahiry '/wp-content/plugins/' ny 'material-wp'
2. Ampandehano amin’ny alalan’ny menio 'Plugins' ao amin’ny WordPress ilay plugin
