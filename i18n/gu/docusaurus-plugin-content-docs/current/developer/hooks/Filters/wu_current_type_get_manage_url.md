# Filter: wu_current_\{$type\}_get_manage_url

આ ફિલ્ટર ડેવલપર્સને મેનેજ સાઇટ URL ના પેરામીટર્સમાં ફેરફાર કરવાની મંજૂરી આપે છે.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $manage_site_url | `string` | મેનેજ સાઇટ URL. |
| $id | `int` | સાઇટ ID. |
| $site_hash | `string` | સાઇટ હેશ. |

### Since

- 2.0.9
### Source

[`inc/class-current.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-current.php#L214) માં લાઇન 214 પર વ્યાખ્યાયિત છે.

## Returns
ફેરફાર કરેલ મેનેજ URL.
