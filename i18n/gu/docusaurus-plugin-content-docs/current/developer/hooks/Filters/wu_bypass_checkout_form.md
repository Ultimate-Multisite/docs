# Filter: wu_bypass_checkout_form

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $bypass | `string\|bool` | જો આપણે ચેકઆઉટ ફોર્મને બાયપાસ કરવો છે, અથવા ફોર્મની જગ્યાએ પાછું આપવા માટેની સ્ટ્રિંગ. |
| $atts | `array` | ચેકઆઉટ બ્લોક/શોર્ટકોડના પેરામીટર્સ. |
