# ફિલ્ટર: wu_sso_url

ક્રોસ-ડોમેન ગ્રાહક ક્રિયાઓ માટે પાછા મોકલવામાં આવે તે પહેલાં જનરેટ થયેલા SSO URL ને ફિલ્ટર કરે છે.

જ્યારે કોઈ ઇન્ટિગ્રેશનને sovereign-tenant SSO લિંકમાં વિશ્વસનીય સંદર્ભ ઉમેરવાની જરૂર હોય અથવા Ultimate Multisiteની token માન્યતા જાળવી રાખીને broker URL બદલવાની જરૂર હોય ત્યારે આ ફિલ્ટરનો ઉપયોગ કરો.

## પેરામીટર્સ

| નામ | પ્રકાર | વર્ણન |
|------|------|-------------|
| $sso_url | `string` | જનરેટ થયેલ SSO URL. |
| $user | `WP_User` | SSO મુલાકાત દ્વારા પ્રમાણિત થનારો વપરાશકર્તા. |
| $site_id | `int` | મુલાકાત માટે લક્ષ્ય site ID. |
| $redirect_to | `string` | સફળ SSO માન્યતા પછીનું ગંતવ્ય URL. |

### ત્યારથી

- 2.13.0

### સ્રોત

`inc/sso/class-sso.php` માં વ્યાખ્યાયિત.


## રિટર્ન્સ

ફિલ્ટર થયેલ SSO URL.
