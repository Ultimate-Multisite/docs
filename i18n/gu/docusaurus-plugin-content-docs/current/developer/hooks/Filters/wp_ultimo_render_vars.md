# Filter: wp_ultimo_render_vars {#filter-wpultimorendervars}

પ્લગઇન ડેવલપર્સને રેન્ડર કન્ટેક્સ્ટમાં વૈશ્વિક સ્તરે વધારાના વેરીએબલ ઉમેરવાની મંજૂરી આપે છે.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $args | `array` | રેન્ડર કોલ દ્વારા પસાર કરાયેલા વેરીએબલ ધરાવતું એરે. |
| $view | `string` | જે view ને રેન્ડર કરવાનો છે તેનું નામ. |
| $default_view | `string` | fallback_view નું નામ. |
