# Filter: wu_is_mcp_enabled

પ્લગઇન ડેવલપર્સને MCP એડેપ્ટર માટે ચોક્કસ સ્થિતિ (state) સેટ કરવાની મંજૂરી આપે છે.

## પેરામીટર્સ

| નામ | પ્રકાર | વર્ણન |
|------|------|-------------|
| $enabled | `bool` | શું MCP એડેપ્ટર સક્ષમ (enabled) છે કે નહીં. |

### ક્યારથી

- 2.5.0
### સ્ત્રોત

તે [`inc/class-mcp-adapter.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-mcp-adapter.php#L250) માં લાઇન 250 પર વ્યાખ્યાયિત થયેલ છે.
