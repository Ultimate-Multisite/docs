# Filter: wp_ultimo_skip_network_active_check

ડેવલપર્સને આ ચેકને બાયપાસ કરવાની મંજૂરી આપે છે.

આ ત્યારે ઉપયોગી છે જ્યારે તમે કમ્પોઝર-આધારિત અને અન્ય કસ્ટમ સેટઅપ્સ (જેમ કે Bedrock) નો ઉપયોગ કરી રહ્યા હોવ, જ્યાં mu-plugins તરીકે પ્લગઇન્સનો ઉપયોગ કરવો સામાન્ય હોય.
