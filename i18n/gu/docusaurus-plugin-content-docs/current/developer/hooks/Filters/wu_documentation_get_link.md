# Filter: wu_documentation_get_link

પ્લગઇન ડેવલપર્સને લિંક્સને ફિલ્ટર કરવાની મંજૂરી આપે છે.

મને ખાતરી નથી કે આ કેટલું ઉપયોગી થશે, પણ તેને રાખવામાં કોઈ નુકસાન નથી.

## પેરામીટર્સ {#parameters}

| Name | Type | વર્ણન |
|------|------|-------------|
| $link | `string` | રજીસ્ટર કરેલી લિંક |
| $slug | `string` | લિંક મેળવવા માટે વપરાયેલ સ્લગ |
| $default_link | `string` | રજીસ્ટર કરેલી ડિફોલ્ટ લિંક |
