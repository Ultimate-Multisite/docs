# Filter: wu_independent_billing_cycle_product_types

જે પ્રોડક્ટ પ્રકારોના સ્વતંત્ર બિલિંગ ચક્ર હોય છે, તેને ફિલ્ટર કરો.

સ્વતંત્ર બિલિંગ ચક્ર ધરાવતા પ્રોડક્ટ્સને કાર્ટમાં પ્લાનના બિલિંગ સમયગાળા સાથે મેળવવા માટે મજબૂર કરવામાં આવતા નથી. આ એવા પ્રોડક્ટ્સ માટે ઉપયોગી છે, જેમ કે ડોમેન રજીસ્ટ્રેશન, જે પ્લાન માસિક છે કે વાર્ષિક, તેની પરવા કર્યા વિના હંમેશા વાર્ષિક બિલિંગ કરે છે.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | સ્વતંત્ર બિલિંગ ચક્ર ધરાવતા પ્રોડક્ટ પ્રકારોની એરે (Array). |
