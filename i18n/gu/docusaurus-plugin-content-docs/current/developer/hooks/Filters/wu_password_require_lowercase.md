# Filter: wu_password_require_lowercase {#filter-wupasswordrequirelowercase}

પાસવર્ડ્સમાં નાના અક્ષરો (lowercase letters) ની જરૂરિયાત છે કે નહીં તે ફિલ્ટર કરે છે.

## પેરામીટર્સ {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | પાસવર્ડમાં નાના અક્ષરો જરૂરી છે કે નહીં. નિયમો લાગુ કરવા પર ડિફોલ્ટ રૂ (Default) `true` હોય છે. |
| $defender_active | `bool` | શું Defender Pro Strong Password સક્રિય (active) છે કે નહીં. |
