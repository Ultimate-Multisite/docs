# Filter: wu_pre_save_settings {#filter-wupresavesettings}

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $settings | `array` | જે સેટિંગ્સ સેવ કરવાની છે. |
| $settings_to_save | `array` | ઉમેરવાના નવા સેટિંગ્સ. |
| $saved_settings | `array` | હાલમાં સેવ કરેલા સેટિંગ્સ. |
