# Filter: wu_limits_is_post_above_limit

આ તપાસે છે કે આપેલ પોસ્ટ પ્રકાર (post type) આ પ્લાન પર ઉપયોગમાં લઈ શકાય છે કે નહીં. પ્લગઇન ડેવલપર્સને રીટર્ન વેલ્યુને ફિલ્ટર કરવાની મંજૂરી આપે છે.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $value | `bool` | જો પોસ્ટ પ્રકાર (post type) બંધ (disabled) છે કે નહીં |
