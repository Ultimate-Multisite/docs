# ફિલ્ટર: wp_ultimo_redirect_url_after_signup {#filter-wpultimoredirecturlaftersignup}

રીડાયરેક્ટ URL સેટ કરો.

આ એક જૂનું (legacy) ફિલ્ટર છે. અહીં જે કેટલાક પેરામીટર્સ આપવામાં આવે છે તે ઉપલબ્ધ નથી, જેમ કે $site_id.

### ક્યારથી {#since}

- 1.1.3: ડેવલપર્સને રીડાયરેક્ટ URL ફિલ્ટર કરવાની મંજૂરી આપે છે.
