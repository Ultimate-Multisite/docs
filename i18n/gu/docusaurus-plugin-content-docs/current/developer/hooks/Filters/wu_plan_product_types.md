# Filter: wu_plan_product_types

એ પ્રોડક્ટ ટાઇપ્સને ફિલ્ટર કરો જેને પ્લાન તરીકે ગણવામાં આવવા જોઈએ.

આ ફિલ્ટર એડઓન્સને વધારાના પ્રોડક્ટ ટાઇપ્સ રજીસ્ટર કરવાની મંજૂરી આપે છે, જેને વેલિડેશન (validation) અને અલગ પાડવામાં (segregation) પ્લાન તરીકે ઓળખવામાં આવવા જોઈએ.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | પ્લાન તરીકે ગણવા માટેના પ્રોડક્ટ ટાઇપ્સની એરે. |
