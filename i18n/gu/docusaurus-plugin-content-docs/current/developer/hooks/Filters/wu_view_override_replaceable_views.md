# Filter: wu_view_override_replaceable_views {#filter-wuviewoverridereplaceableviews}

ડેવલપર્સને બદલી શકાય તેવી (replaceable) યાદીમાં વધારાના ફોલ્ડર્સ ઉમેરવાની મંજૂરી આપે છે.

સાવચેત રહો, કારણ કે વધારાના ફોલ્ડર્સની મંજૂરી આપવાથી Ultimate Multisite ના વર્ઝનને બદલે જૂની નકલો (out-of-date copies) લોડ થઈ શકે છે.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $replaceable_views | `array` | મંજૂર ફોલ્ડર્સની યાદી. |
