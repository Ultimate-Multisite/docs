# Filter: wu_password_require_special {#filter-wupasswordrequirespecial}

પાસવર્ડમાં સ્પેશિયલ કેરેક્ટર્સની જરૂર છે કે નહીં તે ફિલ્ટર કરે છે.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | શું સ્પેશિયલ કેરેક્ટર્સની જરૂર છે. જ્યારે નિયમો લાગુ કરવામાં આવે ત્યારે ડિફોલ્ટ રૂ (true) હોય છે. |
| $defender_active | `bool` | શું Defender Pro Strong Password સક્રિય છે. |
