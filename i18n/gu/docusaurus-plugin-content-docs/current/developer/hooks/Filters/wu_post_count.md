# Filter: wu_post_count

પ્લગઇન ડેવલપર્સને કાઉન્ટના કુલ આંકડામાં ફેરફાર કરવાની મંજૂરી આપે છે

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $count | `int` | કુલ પોસ્ટની સંખ્યા |
| $post_counts | `object` | `wp_count_posts` ફંક્શન દ્વારા પરત કરાયેલ WordPress ઓબ્જેક્ટ |
| $post_type | `string` | પોસ્ટ પ્રકારનો સ્લગ (post type slug) |

### Since

- 1.9.1
### Source

[`inc/limitations/class-limit-post-types.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/limitations/class-limit-post-types.php#L136) માં લાઇન 136 પર વ્યાખ્યાયિત છે


## Returns
નવો કુલ આંકડો
