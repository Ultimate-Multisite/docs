# Filter: wu_membership_renewal_expiration_date {#filter-wumembershiprenewalexpirationdate}

| Name | Type | Description |
|------|------|-------------|
| $expiration | `string` | ગણતરી કરેલી સમાપ્તિ તારીખ. |
| $plan | `\Product` | સભ્યપદ સ્તરની વસ્તુ (Membership level object). |
| $membership_id | `int` | સભ્યપદની ID. |
| $membership | `\Membership` | સભ્યપદની વસ્તુ (Membership object). |
