# Filter: wu_admin_notices

Ultimate Multisite દ્વારા ઉમેરવામાં આવેલા એડમિન નોટિફિકેશન્સને ફિલ્ટર કરવા માટે ડેવલપર્સને મંજૂરી આપે છે.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $notices | `array` | તે ચોક્કસ પેનલ માટે નોટિફિકેશન્સની યાદી. |
| $all_notices | `array` | ઉમેરવામાં આવેલા નોટિફિકેશન્સની યાદી, જે પેનલ પ્રમાણે અલગ કરેલા હોય. |
| $panel | `string` | જે પેનલમાંથી નોટિફિકેશન્સ મેળવવા છે. |
| $filter | `string` | જો ડિસ્મિસ કરી શકાય તેવા નોટિફિકેશન્સને ફિલ્ટર કરવામાં આવ્યા હોય. |
| $dismissed_messages | `array` | ડિસ્મિસ કરેલા નોટિસ કીઝની યાદી. |
