# Filter: wu_hosting_integration_wizard_sections

હોસ્ટિંગ ઇન્ટિગ્રેશન સેટઅપ માટેના વિઝાર્ડ વિભાગોને ફિલ્ટર કરે છે.

આ એડઓન્સને વિઝાર્ડ વિભાગો ઉમેરવા, દૂર કરવા અથવા તેમાં ફેરફાર કરવાની મંજૂરી આપે છે.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $sections | `array` | વિઝાર્ડ વિભાગો. |
| $integration | `\WP_Ultimo\Integrations\Host_Providers\Base_Host_Provider\|\WP_Ultimo\Integrations\Integration` | જે ઇન્ટિગ્રેશનને કન્ફિગર કરવામાં આવી રહ્યું છે. |
| $page | `\Hosting_Integration_Wizard_Admin_Page` | વિઝાર્ડ પેજની ઇન્સ્ટન્સ. |
