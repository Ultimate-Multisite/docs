# Filter: wu_password_require_number

પાસવર્ડમાં નંબર હોવા જરૂરી છે કે નહીં તે ફિલ્ટર કરે છે.

## પેરામીટર્સ

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | પાસવર્ડમાં નંબર હોવા જરૂરી છે કે નહીં. જ્યારે નિયમો લાગુ કરવામાં આવે છે, ત્યારે ડિફોલ્ટ રૂ (true) હોય છે. |
| $defender_active | `bool` | શું Defender Pro Strong Password સક્રિય છે કે નહીં. |
