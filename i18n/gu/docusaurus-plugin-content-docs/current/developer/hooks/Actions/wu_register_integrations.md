# Action: wu_register_integrations

ઇન્ટિગ્રેશન્સને પોતાને રજીસ્ટર કરવાની મંજૂરી આપવા માટે આ એક્શન ટ્રિગર થાય છે.

કોર (Core) અહીં તેના ઇન્ટિગ્રેશન્સ રજીસ્ટર કરે છે.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $registry | `\Integration_Registry` | રજીસ્ટ્રીનું ઇન્સ્ટન્સ (The registry instance). |
