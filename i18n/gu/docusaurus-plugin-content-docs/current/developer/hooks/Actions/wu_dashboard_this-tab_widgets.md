# Action: wu_dashboard_\{$this->tab\}_widgets

પ્લગઇન ડેવલપર્સને નેટવર્ક ડેશબોર્ડ પેનલ પર વિજેટ્સ (widgets) ઉમેરવાની મંજૂરી આપે છે.

## પેરામીટર્સ

| Name | Type | Description |
|------|------|-------------|
| $tab | `string` | વર્તમાન ટેબ (current tab). |
| $screen | `\WP_Screen` | સ્ક્રીન ઓબ્જેક્ટ (screen object). |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Ultimate Multisite એડમિન પેજ ઇન્સ્ટન્સ (admin page instance). |
