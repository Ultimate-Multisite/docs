# Action: wu_mcp_adapter_initialized

જ્યારે MCP એડેપ્ટર શરૂ (initialize) થઈ જાય, ત્યારબાદ આ એક્શન ફાયર થાય છે.

આ અન્ય પ્લગઇન્સ અને થીમ્સને તેમની પોતાની ક્ષમતાઓ (abilities) રજીસ્ટર કરવાની મંજૂરી આપે છે.

## પેરામીટર્સ {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | MCP એડેપ્ટરની ઇન્સ્ટન્સ. |

### ક્યારથી {#since}

- 2.5.0
