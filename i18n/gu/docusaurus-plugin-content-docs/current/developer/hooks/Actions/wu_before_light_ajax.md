# Action: wu_before_light_ajax

અમુક કિસ્સાઓમાં, ક્રિયાઓ (actions) ને હેન્ડલ કરવા માટે અમારે વધારાના સંસાધનો (resources) લોડ કરવાની જરૂર પડશે.

આ action નો ઉપયોગ જરૂર પડ્યે વધારાની dependencies લોડ કરવા માટે થઈ શકે છે.
