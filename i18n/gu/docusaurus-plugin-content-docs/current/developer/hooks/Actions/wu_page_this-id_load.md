# Action: wu_page_{$this->id}_load {#action-wupagethis-idload}

પ્લગઇન ડેવલપર્સને અમારા પેજીસમાં વધારાના હૂક્સ (hooks) ઉમેરવાની મંજૂરી આપે છે.

## પેરામીટર્સ {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $id | `string` | આ પેજની ID. |
| $page_hook | `string` | આ પેજનો પેજ હૂક. |
| $admin_page | `self` | પેજનું ઇન્સ્ટન્સ. |

### ક્યારથી {#since}

- 1.8.2
- 2.0.4: ત્રીજો પેરામીટર ઉમેરવામાં આવ્યો: પેજનું ઇન્સ્ટન્સ.
