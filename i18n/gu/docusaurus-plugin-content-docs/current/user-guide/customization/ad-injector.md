# Ultimate Multisite: Ad Injector વડે તમારા નેટવર્ક પર જાહેરાતો મૂકવી

Ultimate Multisite: Ad Injector એક **મફત add-on** છે જે તમને તમારા plans પર જાહેરાતો મૂકવાનાં સાધનો આપે છે, સીધા તેમના edit page પરથી.

Ultimate Multisite: Ad Injector ઇન્સ્ટોલ કર્યા પછી, તેને activate કરવા માટે તમારા _Plugins_ page પર જાઓ.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Network Activate બટન પર ક્લિક કરો_

એકવાર add-on active થઈ જાય પછી, plan edit screen પર _Advanced Options_ meta box માં _Ad Settings_ નામનો એક નવો વિકલ્પ આવશે.

![Plan edit screen પર Ad Settings](/img/config/product-edit-full.png)

_હવે તમે તમારા દરેક plan માટે ad settings edit કરી શકો છો_

આ tab પર, તમે front-end માં posts અને pages માટે, content ની પહેલાં અને પછી, ad codes ઉમેરી શકશો.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_તમારી જાહેરાતનો code દાખલ કર્યા પછી, તે એ ચોક્કસ plan ના posts અને pages પર દેખાશે._

તમે એ plan ના users માટે Admin Pages ના header માં પણ જાહેરાતો ઉમેરી શકો છો.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_જો તમે તમારા user ના back-end પર જાહેરાતો બતાવવા માંગતા હો, તો Admin Panel Ad Code વિકલ્પ પર જાઓ_

### જાહેરાતોથી આગળ

તમે Ultimate Multisite: Ad Injector નો ઉપયોગ high-tier plans ને support આપવા માટે પણ કરી શકો છો.

બસ _Admin Panel Ads_ વિકલ્પ પર જાઓ અને તમારા support tool માટે HTML code ઉમેરો.

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_ચોક્કસ plans માં customer support ઉમેરો_
