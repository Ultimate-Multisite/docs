# PRO Theme વડે તમારા Admin Panel ને કસ્ટમાઇઝ કરવું

ઘણીવાર ક્લાયન્ટ વર્ક કરતી વખતે આપણને WordPress ના admin dashboard માં અમારા ક્લાયન્ટ્સને કસ્ટમ અનુભવ આપવાની જરૂર પડે છે. જો તમે વધુ પ્રોફેશનલ દેખાવ અને અનુભવ શોધી રહ્યા છો, તો PRO Theme તમને મદદ કરી શકે છે.

PRO Theme એ એક admin theme છે જે તમારા ક્લાયન્ટ્સના Dashboard ને કસ્ટમાઇઝ કરવામાં તમને મદદ કરવા માટે વિકસાવવામાં આવી છે. અને તે Ultimate Multisite ના premium plugins core નો પણ ભાગ છે.

તેને ઇન્સ્ટોલ કર્યા પછી, તમારા _Plugins_ પેજ પર જાઓ અને તમારા Network પર તેને activate કરવા માટે ક્લિક કરો.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_તમારા Plugins પેજ પર તેને network-wide activate કરો_

_**નોંધ:**_ _જો તમે ફક્ત તેને ઇન્સ્ટોલ કરો છો, તો તે તમારા ક્લાયન્ટ્સને એક વિકલ્પ તરીકે દેખાશે જે ફક્ત તે યુઝર માટે theme ઉમેરવા માટે માર્ક કરી શકાય છે._

હવે, તમારું admin interface સંપૂર્ણપણે નવી ડિઝાઇન સાથે તૈયાર છે! તમને ડાબી કોલમમાં PRO Theme નામનું એક નવું menu પણ મળશે.
