# અન્ય Plugins સાથે જાણીતી અસંગતતાઓ

Ultimate Multisite ઇન્સ્ટોલ અને એક્ટિવેટ કર્યા પછી, અન્ય plugins સાથેની અસંગતતાઓને કારણે તમને સમસ્યાઓ આવી શકે છે.

અમે દરેક નવા રિલીઝ સાથે આ યાદી ઘટાડવા માટે સક્રિય રીતે કામ કરી રહ્યા છીએ, પરંતુ જો કોઈ plugin અહીં સૂચિબદ્ધ છે, તો તમારે Ultimate Multisite Network ને સંપૂર્ણ રીતે કાર્યરત રાખવા માટે તેને ડિએક્ટિવેટ કરવું પડી શકે છે.

અહીં તેમની યાદી છે અને આ કિસ્સાઓમાં તમારે શું કરવું જોઈએ:

**કેટલાક WPMU DEV plugins** WPMU DEV દ્વારા _હવે જાળવવામાં નહીં આવતા_ કેટલાક plugins Ultimate Multisite રજિસ્ટ્રેશન ફ્લોને છેલ્લા સ્ટેપ પર અટકાવી દે છે (યુઝર એકાઉન્ટ બનાવી શકતો નથી). અહીં તે સમસ્યાઓ ઊભી કરતા plugins ની યાદી છે:

  * Signup Language: _Ultimate Multisite પાસે એક add-on છે જે સમાન કાર્યક્ષમતા આપે છે, જેને Ultimate Multisite: Language Switcher કહેવાય છે._

  * Set Blog Description on Blog Creation: _આ Ultimate Multisite માં બિલ્ટ-ઇન છે, તેથી વધારાના plugin ની જરૂર નથી._

  * New Blog Templates: _આ Ultimate Multisite માં બિલ્ટ-ઇન છે, તેથી વધારાના plugin ની જરૂર નથી._

  * Set Password on Multisite Blog Creation: _આ Ultimate Multisite માં બિલ્ટ-ઇન છે, તેથી વધારાના plugin ની જરૂર નથી._

  * Signup TOS: _આ Ultimate Multisite માં બિલ્ટ-ઇન છે, તેથી વધારાના plugin ની જરૂર નથી._

  * Anti-Splog: ઓછું શક્તિશાળી હોવા છતાં, આ પણ _Ultimate Multisite માં બિલ્ટ-ઇન છે._

**નોંધ** : અન્ય તમામ WPMU Dev plugins Ultimate Multisite સાથે સામાન્ય રીતે વાપરી શકાય છે. ઉદાહરણો માં _Smush_ , _Forminator_ , _Defender,_ વગેરેનો સમાવેશ થાય છે.
