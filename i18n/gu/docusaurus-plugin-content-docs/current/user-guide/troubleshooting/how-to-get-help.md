# મદદ કેવી રીતે મેળવવી

તમને તમારી માન્ય License key મળી ગઈ છે, અને તેની સાથે Ultimate Multisite સેટ કરતી વખતે તમને આવતી સમસ્યાઓ ઉકેલવામાં મદદ કરવા માટે અમારી ટીમનો સંપર્ક કરવાની સુવિધા પણ મળે છે.

આ પાઠમાં license ધારકો માટે ઉપલબ્ધ support channels અને અમારી support ટીમનો સંપર્ક કરવા માટે તમારે કઈ માહિતી તૈયાર રાખવી જોઈએ તે વિશે જણાવ્યું છે.

## Help Site

તમે અમારી help site પરથી વિવિધ લેખો અને tutorials મેળવી શકો છો, જેની link નીચે આપેલી છે.

[)

અમે અમારા users ને Ultimate Multisite ની મૂળભૂત features અને functions થી પરિચિત થવા પ્રોત્સાહિત કરીએ છીએ. અમારી help site પર "Getting Started" લેખો ઉપલબ્ધ છે અને સામાન્ય પ્રશ્નોના જવાબો પણ ત્યાં મળી શકે છે, જેમ કે add-ons કેવી રીતે install કરવા, payment gateways કેવી રીતે integrate કરવા, વગેરે.

## Support નો સંપર્ક કરો

જો તમે અટકી જાઓ અને અમારી help site પરથી જવાબ ન મળે, તો તમે અમારી support ટીમનો સંપર્ક કરી શકો છો - કાં તો અમારી website ના નીચે જમણી બાજુએ આવેલા speech bubble icon પર click કરીને અથવા સીધો support@ultimatemultisite.com પર email મોકલીને.

ટેકનિકલ સમસ્યાઓ માટે support નો સંપર્ક કરતી વખતે, કૃપા કરીને તમારી સમસ્યાનું વિગતવાર વર્ણન, screenshot, error messages, અથવા કોઈપણ માહિતી આપો જે અમારી support ટીમને તમારી સમસ્યા સરળતાથી સમજવામાં અને મૂલ્યાંકન કરવામાં મદદરૂપ થાય. તમારે તમારી license key પણ આપવી પડશે. ફક્ત active license ધારકોને જ support મળવાનો અધિકાર છે.

## Community Site

અમારી [community site](https://community.ultimatemultisite.com/) Ultimate Multisite users માટે એકબીજા સાથે વાતચીત કરવાનું સ્થળ છે. તે એક માહિતી પ્રણાલી તરીકે કામ કરે છે જ્યાં સભ્યો પોસ્ટ કરી શકે છે, ચર્ચાઓ પર ટિપ્પણી કરી શકે છે, સલાહ આપી શકે છે, સહયોગ કરી શકે છે, અથવા crowdsource કરી શકે છે. Private spaces/pages unlock કરવા માટે તમારે તમારી Ultimate Multisite license key આપવી જરૂરી રહેશે.
