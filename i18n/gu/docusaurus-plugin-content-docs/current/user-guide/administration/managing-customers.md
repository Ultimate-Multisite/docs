# ગ્રાહકોનું સંચાલન

**Customers** પેજ તમારા નેટવર્કમાં રજીસ્ટર થયેલા સભ્યોની યાદી દર્શાવે છે. આ પેજ પર યુઝરની મૂળભૂત માહિતી જોવા મળે છે જેમ કે નામ, ઈમેલ એડ્રેસ, છેલ્લું લોગિન, અને **switch to** લિંક જે તમને યુઝરની subsite માં site administrator તરીકે લોગિન કરવા દે છે.

![Customers list page](/img/admin/customers-list.png)

ગ્રાહકના કાર્ડ નીચે **Manage** પર ક્લિક કરો જેથી તે ગ્રાહક વિશે વિગતવાર માહિતી જોઈ શકાય.

ગ્રાહકના detail પેજ પર, **At a Glance** બ્લોક ગ્રાહક ID, છેલ્લા લોગિનનો સમય, અને આ ગ્રાહકે કુલ કેટલી રકમ ખર્ચી છે તે દર્શાવે છે.

**Customer Options** બ્લોકમાં તમે આ ગ્રાહકને **VIP** તરીકે સેટ કરી શકો છો (એક ટેગ જેથી તેમને શોધવામાં સરળતા રહે). **Billing Info** ટેબ billing માહિતી દર્શાવે છે જેમ કે ઈમેલ, સરનામું, અને દેશ.

**Fields** બ્લોક ગ્રાહકનું username, ઈમેલ દર્શાવે છે, અને તમને **Email Verification** સ્ટેટસ બદલવાની મંજૂરી આપે છે.

Customer Options બ્લોકની નીચે તમને ચાર વધારાના બ્લોક મળશે:

- **Memberships** — ગ્રાહક હાલમાં જે subscriptions માં સબસ્ક્રાઇબ છે તે
- **Payments** — ગ્રાહકનો પેમેન્ટ ઇતિહાસ
- **Sites** — આ ગ્રાહકની માલિકીની sites
- **Events** — આ ગ્રાહક સાથે સંબંધિત ઇવેન્ટ્સ (email verification, billing ફેરફારો, વગેરે)

કોઈપણ આઇટમની બાજુમાં arrow icon (→) પર ક્લિક કરો જેથી તેના detail પેજ પર જઈ શકાય. ઉદાહરણ તરીકે, site ની બાજુમાં arrow પર ક્લિક કરવાથી તમે site edit પેજ પર પહોંચો છો જ્યાં તમે membership association, site type, અને active status બદલી શકો છો.

આ જ રીતે memberships અને payments માટે પણ — દરેક detail પેજ સાથે લિંક થયેલું છે જ્યાં તમે membership status, સંકળાયેલ ગ્રાહક, products, payment gateway settings એડિટ કરી શકો છો, અને refunds પણ પ્રોસેસ કરી શકો છો.

ગ્રાહક પેજની જમણી બાજુએ, તમને મળશે:

- **Last Login & IPs** — છેલ્લા લોગિનનો સમય, IP address, અને દેશ
- **Timestamps** — ગ્રાહક એકાઉન્ટ ક્યારે બનાવવામાં આવ્યું અને છેલ્લે ક્યારે સુધારવામાં આવ્યું
- **Delete** — ગ્રાહકને કાયમી ધોરણે ડિલીટ કરવાનો વિકલ્પ

:::warning
ગ્રાહકને ડિલીટ કરવું કાયમી છે અને તેને પાછું લાવી શકાતું નથી.
:::
