# સિસ્ટમ ઇમેઇલ્સનું સંચાલન

Ultimate Multisite વિવિધ સિસ્ટમ ઇમેઇલ્સ મોકલે છે જેમ કે સાઇનઅપ કન્ફર્મેશન, પેમેન્ટ રિસિપ્ટ્સ અને મેમ્બરશિપ ફેરફારો. તમે આ ઇમેઇલ ટેમ્પલેટ્સને તમારી બ્રાન્ડ અનુસાર કસ્ટમાઇઝ કરી શકો છો.

## સિસ્ટમ ઇમેઇલ્સ જોવી

**Ultimate Multisite → Settings** પર જાઓ અને ઇમેઇલ સંબંધિત સેટિંગ્સ જોવા માટે **Emails** ટેબ પર ક્લિક કરો.

![Settings - Emails tab](/img/admin/settings-emails.png)

ત્યાંથી, તમે બધી ઇમેઇલ ટેમ્પલેટ્સ જોવા માટે **System Emails** પેજ પર જઈ શકો છો.

![System Emails list](/img/admin/system-emails-list.png)

## નવી ઇમેઇલ ટેમ્પલેટ્સ ઇમ્પોર્ટ કરવી

Ultimate Multisite ના નવા વર્ઝન અને add-ons નવી ઇમેઇલ ટેમ્પલેટ્સ રજિસ્ટર કરી શકે છે. કોન્ફ્લિક્ટ્સ ટાળવા માટે, નવી ટેમ્પલેટ્સ આપોઆપ ઉમેરાતી નથી — તમારે તેમને મેન્યુઅલી ઇમ્પોર્ટ કરવી પડશે.

નવી ઇમેઇલ ટેમ્પલેટ્સ ઇમ્પોર્ટ કરવા માટે:

1. **System Emails** પેજ પર જાઓ
2. પેજની ટોચ પર **Reset & Import** બટન પર ક્લિક કરો
3. ઉપલબ્ધ ટેમ્પલેટ્સ જોવા માટે **Import Emails** ઓપ્શન ટોગલ કરો
4. તમે જે ટેમ્પલેટ્સ ઇમ્પોર્ટ કરવા માંગો છો તે પસંદ કરો અને કન્ફર્મ કરો

## ઇમેઇલ ટેમ્પલેટ્સ રીસેટ કરવી

જો તમે કોઈ ઇમેઇલ ટેમ્પલેટ કસ્ટમાઇઝ કરી હોય અને ડિફોલ્ટ કન્ટેન્ટ પર પાછા જવા માંગતા હો, તો તમારી પાસે બે વિકલ્પો છે:

1. **ડિલીટ કરો અને ફરીથી ઇમ્પોર્ટ કરો** — સિસ્ટમ ઇમેઇલ ડિલીટ કરો અને તેને ફરીથી ઇમ્પોર્ટ કરો (આનાથી સેન્ડ મેટ્રિક્સ ભૂંસાઈ જશે)
2. **ટૂલ દ્વારા રીસેટ કરો** — **Reset & Import** ટૂલનો ઉપયોગ કરો, **Reset** ઓપ્શન ટોગલ કરો, અને તમે જે ઇમેઇલ્સ રીસેટ કરવા માંગો છો તે પસંદ કરો

Reset ઓપ્શન વધુ સારો છે કારણ કે તે ડિફોલ્ટ કન્ટેન્ટ રિસ્ટોર કરતી વખતે સેન્ડ મેટ્રિક્સ અને અન્ય મેટાડેટા સાચવી રાખે છે.
