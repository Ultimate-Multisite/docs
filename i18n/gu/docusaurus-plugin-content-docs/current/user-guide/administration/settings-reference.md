# સેટિંગ્સ સંદર્ભ

આ પેજ એવા સેટિંગ્સની માહિતી આપે છે જે Ultimate Multisite માં રોજિંદા એડમિનિસ્ટ્રેશન અને તાજેતરના વર્તણૂક ફેરફારોને અસર કરે છે.

## અન્ય વિકલ્પો (Other Options)

**Other Options** વિસ્તાર **Ultimate Multisite > Settings > Login & Registration** હેઠળ દેખાય છે.

| Setting | Description |
|---|---|
| **Enable Jumper** | આ એડમિન વિસ્તારમાં Jumper ક્વિક નેવિગેશન ટૂલ બતાવે છે. તેનો ઉપયોગ Ultimate Multisite સ્ક્રીન, નેટવર્ક ઓબ્જેક્ટ્સ અને સપોર્ટેડ એડમિન ડેસ્ટિનેશન્સ પર સીધા જ જવા માટે કરો. જો તમે આ શોર્ટકટ દેખાવા ન આપો તો તેને ડિસેબલ કરો. |

## એરર રિપોર્ટિંગ અને ટેલિમેટ્રી (Error reporting and telemetry)

અગાઉનું એરર-રિપોર્ટિંગ ઓપ્ટ-ઇન સેટિંગ સેટિંગ્સ પેજમાંથી દૂર કરવામાં આવ્યું છે. ઑનનિમાસ ટેલિમેટ્રી (Anonymous telemetry) ડિસેબલ છે અને તેને એનેબલ કરવા માટે કોઈ UI ટૉગલ નથી.

જો તમે સેટિંગ્સ પેજ માટે આંતરિક રનબુક્સ (runbooks) અથવા સ્ક્રીનશોટ જાળવી રાખો છો, તો જૂના એરર-રિપોર્ટિંગ ઓપ્ટ-ઇન ફીલ્ડનો સંદર્ભ દૂર કરો જેથી એડમિનિસ્ટ્રેટર્સ કોઈ એવું સેટિંગ શોધી ન કરે જે હવે હાજર નથી.

## ઇમ્પોર્ટ/એક્સપોર્ટ સેટિંગ્સ (Import/Export settings)

**Import/Export** સેટિંગ્સ ટેબ વર્ણવે છે કે તે કયા સેટિંગ્સને નિયંત્રિત કરે છે અને સાઇટ અને નેટવર્ક આર્કાઇવ્સ માટે સીધું **Ultimate Multisite > Site Export** સાથે લિંક કરે છે. ઇમ્પોર્ટ/એક્સપોર્ટ કન્ફિગરેશન માટે સેટિંગ્સ ટેબનો ઉપયોગ કરો, single-site એક્સપોર્ટ/ઇમ્પોર્ટ વર્કફ્લો માટે **Tools > Export & Import** નો ઉપયોગ કરો, અને જ્યારે તમને સંપૂર્ણ નેટવર્ક એક્સપોર્ટ આર્કાઇવની જરૂર હોય ત્યારે Site Export ટૂલનો ઉપયોગ કરો.
