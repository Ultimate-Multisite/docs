# સાઇટ એક્સપોર્ટ {#site-export}

**સાઇટ એક્સપોર્ટ** એડમિન પેજ નેટવર્ક એડમિનિસ્ટ્રેટર્સને એક સાઇટ, અથવા આખું નેટવર્ક, ને ડાઉનલોડ કરી શકાય તેવા આર્કાઇવમાં પેક કરવા દે છે. આ આર્કાઇવ માઇગ્રેશન, બેકઅપ અથવા કામ સોંપવા (hand-off) માટે ઉપયોગી છે.

## એક સાઇટ એક્સપોર્ટ કરવી {#exporting-one-site}

**Ultimate Multisite > Site Export** પર જાઓ અને **Generate new Site Export** પસંદ કરો. જે સબસાઇટ તમારે એક્સપોર્ટ કરવી છે તેને પસંદ કરો, અને પછી પસંદ કરો કે આર્કાઇવમાં uploads, plugins, અને themes શામેલ કરવા છે કે નહીં.

એક્સપોર્ટ પૂર્ણ થયા પછી, **Existing Exports** લિસ્ટમાંથી ZIP ડાઉનલોડ કરો. એક્સપોર્ટ ZIP માં હવે એક self-booting `index.php` અને એક `readme.txt` શામેલ હોય છે, જેથી આ આર્કાઇવને નવા હોસ્ટ પર અપલોડ કરીને, અલગથી કોઈ ઇમ્પોર્ટર plugin ઇન્સ્ટોલ કર્યા વિના શરૂ કરી શકાય.

## આખું નેટવર્ક એક્સપોર્ટ કરવું {#exporting-the-whole-network}

જ્યારે તમને નેટવર્કના બધા સબસાઇટ્સને સમાવતો એક જ આર્કાઇવ જોઈતો હોય, ત્યારે સાઇટ એક્સપોર્ટ પેજ પર **Network Export** નો ઉપયોગ કરો. આ ખાસ કરીને હોસ્ટ માઇગ્રેશન, ડિઝાસ્ટર-રિકવરી ડ્રિલ્સ, અથવા સ્ટેજિંગ રીબિલ્ડ્સ પહેલા ઉપયોગી છે, જ્યાં દરેક સબસાઇટને સાથે જ જવું પડે.

કારણ કે નેટવર્ક એક્સપોર્ટ એક સાઇટ એક્સપોર્ટ કરતાં ઘણો મોટો હોઈ શકે છે, તેથી તેને ઓછા ટ્રાફિકના સમયે ચલાવો અને ખાતરી કરો કે લક્ષ્ય સ્ટોરેજમાં uploads, plugins, themes, અને જનરેટ થયેલા ZIP ફાઇલો માટે પૂરતી ખાલી જગ્યા છે.

### નેટવર્ક ઇમ્પોર્ટ બંડલ્સ {#network-import-bundles}

Ultimate Multisite 2.12.0 થી શરૂ કરીને, Site Exporter **network import bundles** જનરેટ કરી શકે છે — જે નેટવર્ક-વ્યાપી સાઇટ રિસ્ટોરેશન માટે ખાસ ડિઝાઇન કરેલા આર્કાઇવ છે. એક નેટવર્ક ઇમ્પોર્ટ બંડલમાં નવા નેટવર્ક ઇન્સ્ટોલેશન પર અનેક સાઇટ્સને રિસ્ટોર કરવા માટે જરૂરી તમામ ફાઇલો અને મેટાડેટા હોય છે.

#### નેટવર્ક ઇમ્પોર્ટ બંડલ જનરેટ કરવો {#generating-a-network-import-bundle}

1. **Ultimate Multisite > Site Export** પર જાઓ
2. **Generate new Network Export** પર ક્લિક કરો
3. એક્સપોર્ટ પ્રકાર તરીકે **Network Import Bundle** પસંદ કરો
4. uploads, plugins, અને themes શામેલ કરવા છે કે નહીં તે પસંદ કરો
5. **Generate** પર ક્લિક કરો
6. **Existing Exports** લિસ્ટમાંથી બંડલ ZIP ડાઉનલોડ કરો

#### નેટવર્ક ઇમ્પોર્ટ બંડલમાંથી રિસ્ટોર કરવું {#restoring-from-a-network-import-bundle}

સાઇટ્સને નેટવર્ક ઇમ્પોર્ટ બંડલમાંથી રિસ્ટોર કરવા માટે:

1. તમારા લક્ષ્ય હોસ્ટ પર Ultimate Multisite ઇન્સ્ટોલ કરો
2. મલ્ટિસાઇટ સેટઅપ વિઝાર્ડ પૂર્ણ કરો
3. નવા નેટવર્ક પર **Ultimate Multisite > Site Export** પર જાઓ
4. **Import Network Bundle** પર ક્લિક કરો
5. નેટવર્ક ઇમ્પોર્ટ બંડલ ZIP ફાઇલ અપલોડ કરો
6. સ્ક્રીન પર આપેલા ઇમ્પોર્ટ સૂચનોને અનુસરો
7. ઇમ્પોર્ટ પ્રક્રિયા તમામ સાઇટ્સ, તેમનો કન્ટેન્ટ અને કોન્ફિગરેશન્સને રિસ્ટોર કરશે

નેટવર્ક ઇમ્પોર્ટ બંડલ્સ નીચેની વસ્તુઓ જાળવી રાખે છે:
- સાઇટ કન્ટેન્ટ (posts, pages, custom post types)
- સાઇટ સેટિંગ્સ અને ઓપ્શન્સ
- યુઝર રોલ્સ અને પરમિશન
- plugins અને themes (જો બંડલમાં શામેલ હોય)
- મીડિયા uploads (જો બંડલમાં શામેલ હોય)
- કસ્ટમ ડેટાબેઝ ટેબલ્સ અને ડેટા

## self-booting export ZIP ઇન્સ્ટોલ કરવો {#installing-a-self-booting-export-zip}

નવા હોસ્ટ પર self-booting ZIP ને રિસ્ટોર કરવા માટે:

1. એક્સપોર્ટ કરેલા ZIP ની સામગ્રીને લક્ષ્ય web root પર અપલોડ કરો.
2. બ્રાઉઝરમાં અપલોડ કરેલ `index.php` ખોલો.
3. બંડલ્ડ એક્સપોર્ટ પેકેજમાંથી સ્ક્રીન પર આપેલા ઇન્સ્ટોલર સૂચનોને અનુસરો.
4. કામચલાઉ ફાઇલો દૂર કરતા પહેલા, બંડલ્ડ `readme.txt` માં એક્સપોર્ટ-વિશિષ્ટ નોંધોની સમીક્ષા કરો.

addon-વિશિષ્ટ ઇન્સ્ટોલેશન અને ઇમ્પોર્ટ વિગતો માટે, [Site Exporter addon documentation](/addons/site-exporter) જુઓ.

Ultimate Multisite 2.9.0 માં ઉમેરવામાં આવેલા single-site tool માટે, [Export & Import](/user-guide/administration/export-import) જુઓ.
