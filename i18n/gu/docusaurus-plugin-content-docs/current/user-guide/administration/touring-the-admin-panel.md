# Admin Panelની મુલાકાત {#touring-the-admin-panel}

Ultimate Multisite શક્ય તેટલું સરળ અને સ્વયં-સ્પષ્ટ બનવાનો ઉદ્દેશ રાખે છે, પરંતુ આટલું શક્તિશાળી સાધન હોય ત્યારે માર્ગદર્શિત મુલાકાત મદદરૂપ થઈ શકે છે. ચાલો તમને પરિચિત કરવામાં મદદ કરવા માટે admin પૃષ્ઠો પર નજર કરીએ.

## Dashboard {#dashboard}

Ultimate Multisite **Dashboard** આવક, સાઇટ પ્રવૃત્તિઓ, સભ્યપદ વૃદ્ધિ, મુલાકાતી સંખ્યા અને ભૌગોલિક ડેટા સહિતના મૂળભૂત અહેવાલો અને વિશ્લેષણ બતાવે છે.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

અહીં dashboardનો સંપૂર્ણ દૃશ્ય છે:

![Ultimate Multisite Dashboard સંપૂર્ણ પૃષ્ઠ](/img/admin/um-dashboard-full.png)

અને dashboardનો નીચેનો વિભાગ:

![Ultimate Multisite Dashboard નીચેનો વિભાગ](/img/admin/um-dashboard-bottom.png)

અહીં નેટવર્ક dashboardનો પણ સંપૂર્ણ દૃશ્ય છે:

![Network Dashboard સંપૂર્ણ પૃષ્ઠ](/img/admin/network-dashboard-full-page.png)

## Checkout ફોર્મ્સ {#checkout-forms}

**Checkout ફોર્મ્સ** તમને કસ્ટમ નોંધણી પૃષ્ઠો બનાવવાની લવચીકતા આપે છે. તમે ક્ષેત્રો કસ્ટમાઇઝ કરી શકો છો, કસ્ટમ ક્લાસ અને સ્ક્રિપ્ટ્સ દાખલ કરી શકો છો, અને મુલાકાતીના દેશ અથવા સ્થાનના આધારે પ્રવેશ મર્યાદિત કરી શકો છો.

![Checkout ફોર્મ્સની સૂચિ](/img/admin/checkout-forms-list.png)

## ઉત્પાદનો {#products}

**ઉત્પાદનો** વિભાગ એ જગ્યા છે જ્યાં તમે તમારા નેટવર્ક માટે વિવિધ પ્લાન, પેકેજ અને સેવાઓ બનાવો છો. ઉત્પાદન સ્તરે કિંમતો, બિલિંગ આવર્તનો અને મર્યાદાઓ તથા ક્વોટા વ્યાખ્યાયિત કરો.

![ઉત્પાદનોની સૂચિ](/img/admin/products-list.png)

## સભ્યપદો {#memberships}

**સભ્યપદો** પૃષ્ઠ તમારા નેટવર્કમાં તમામ સબ્સ્ક્રિપ્શન બતાવે છે. પ્લાન, ઉત્પાદનો, બિલિંગ રકમો અને આવર્તનો, ચુકવણી ઇતિહાસ અને સમયમુદ્રાઓ સહિત સબ્સ્ક્રિપ્શન વિગતોની સમીક્ષા કરો અને ફેરફાર કરો.

![સભ્યપદોની સૂચિ](/img/admin/memberships-list.png)

## ચુકવણીઓ {#payments}

**ચુકવણીઓ** પૃષ્ઠ તમારા સમગ્ર નેટવર્કમાં ચુકવણી ઇતિહાસનો ઝડપી દૃશ્ય આપે છે, જેમાં ચોક્કસ ઉત્પાદનો અને રકમો સહિત વિગતવાર વ્યવહાર માહિતી હોય છે.

![ચુકવણીઓની સૂચિ](/img/admin/payments-list.png)

## ગ્રાહકો {#customers}

**ગ્રાહકો** પૃષ્ઠ તમારા નેટવર્કમાં નોંધાયેલા તમામ સભ્યોને નામ, ઇમેઇલ સરનામું, છેલ્લું લોગિન અને તેમના subsiteમાં લોગિન કરવા માટેની "switch to" લિંક સહિત મૂળભૂત વપરાશકર્તા માહિતી સાથે બતાવે છે.

![ગ્રાહકોની સૂચિ](/img/admin/customers-list.png)

## સાઇટ્સ {#sites}

**સાઇટ્સ** પૃષ્ઠ તમારા નેટવર્કમાં સાઇટ ટેમ્પ્લેટ્સ અને ગ્રાહક-માલિકીની સાઇટ્સ સહિત તમામ subsitesની સૂચિ આપે છે. મેપ કરેલા ડોમેન, મર્યાદાઓ અને ક્વોટા, plugins અને themes, અને સાઇટ થંબનેલ્સ સરળતાથી મેનેજ કરો.

![સાઇટ્સની સૂચિ](/img/admin/sites-list.png)

## ડોમેન {#domains}

**ડોમેન** પૃષ્ઠ subsites પર મેપ કરેલા કસ્ટમ ડોમેન માટે સમર્પિત છે. super admin તરીકે, તમે કસ્ટમ ડોમેન નામો મેન્યુઅલી ઉમેરી અથવા મેપ કરી શકો છો અને DNS રેકોર્ડ્સ અને લોગ્સ જોઈ શકો છો.

![ડોમેનની સૂચિ](/img/admin/domains-list.png)

## ડિસ્કાઉન્ટ કોડ્સ {#discount-codes}

**ડિસ્કાઉન્ટ કોડ્સ** તમને તમારા અંતિમ વપરાશકર્તાઓને ડિસ્કાઉન્ટ આપવા માટે કૂપન કોડ્સ બનાવવાની મંજૂરી આપે છે. ડિસ્કાઉન્ટ મૂલ્ય નિર્દિષ્ટ કરો અને ચોક્કસ વપરાશકર્તાઓ અથવા જૂથોને લક્ષ્ય બનાવો.

![ડિસ્કાઉન્ટ કોડ્સની સૂચિ](/img/admin/discount-codes-list.png)

## પ્રસારણો {#broadcasts}

**પ્રસારણો** એ જાહેરાતો રોલ આઉટ કરવા અથવા તમારા subsite વપરાશકર્તાઓને ખાનગી સંદેશા મોકલવા માટેનું સાધન છે. તેઓ સબ્સ્ક્રાઇબ કરેલા પ્લાન અથવા ઉત્પાદનના આધારે ચોક્કસ વપરાશકર્તા જૂથોને લક્ષ્ય બનાવો.

![પ્રસારણોની સૂચિ](/img/admin/broadcasts-list.png)

## Settings {#settings}

**Settings** પૃષ્ઠ એ છે જ્યાં તમે Ultimate Multisite કન્ફિગર કરો છો — નોંધણી સેટિંગ્સ, ચુકવણીઓ, API અને webhooks, ડોમેન મેપિંગ અને અન્ય ઇન્ટિગ્રેશન્સ.

AI connector સેટિંગ્સ હાલમાં સમર્થિત OAuth provider poolsની સૂચિ આપે છે: Anthropic Max, OpenAI ChatGPT/Codex અને Google AI Pro. દરેક provider કાર્ડ super adminsને accounts જોડવા, સચવાયેલા accounts રીફ્રેશ કરવા, ઇમેઇલ દ્વારા accounts દૂર કરવા અને sandboxed પર્યાવરણ browser redirectને અવરોધે ત્યારે manual OAuth fallback વાપરવાની મંજૂરી આપે છે. Cursor Pro setup વિકલ્પો admin panelમાંથી દૂર કરવામાં આવ્યા છે.

ChatGPT/Codex accounts connector-backed tool useને સમર્થન આપે છે જ્યાં operation toolsને મંજૂરી આપે છે, તેથી connector-supported operations પર આધારિત admin workflows OpenAI account જોડાયા પછી Codex tool વર્તનનો ઉપયોગ કરી શકે છે.

## ઇવેન્ટ્સ {#events}

**ઇવેન્ટ્સ** પૃષ્ઠ તમારા નેટવર્કમાં તમામ ઇવેન્ટ્સ અને લોગ્સનો રેકોર્ડ રાખે છે. તે પ્લાન ફેરફારો, સાઇનઅપ્સ અને અન્ય હલચલ જેવી પ્રવૃત્તિઓને ટ્રેક કરે છે — તમારા multisiteનું નિરીક્ષણ કરવા માટે ઉપયોગી.

![ઇવેન્ટ્સની સૂચિ](/img/admin/events-list.png)

## Webhooks {#webhooks}

**Webhooks** તમને અન્ય એપ્લિકેશનને ડેટા પહોંચાડવાની મંજૂરી આપે છે. Ultimate Multisiteમાંથી Zapier જેવા પ્લેટફોર્મ્સ પર ડેટા મોકલવા માટે ઉપયોગી.

![Webhooksની સૂચિ](/img/admin/webhooks-list.png)
