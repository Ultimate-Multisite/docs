# સાઇટ્સનું સંચાલન

સાઇટ્સ (અથવા સબસાઇટ્સ) તમારા WaaS બિઝનેસનો મુખ્ય ભાગ છે. Ultimate Multisite માં 3 પ્રકારની સાઇટ્સ છે:

- **ગ્રાહક-માલિકીની** — ચોક્કસ ગ્રાહકોની સાઇટ્સ
- **સાઇટ ટેમ્પલેટ્સ** — પૂર્વ-નિર્ધારિત સાઇટ્સ જે ગ્રાહકો શરૂઆતના બિંદુ તરીકે પસંદ કરી શકે છે
- **મુખ્ય સાઇટ** — તમારી પ્રાથમિક નેટવર્ક સાઇટ

## સાઇટ્સ જોવી

તમારા નેટવર્કની બધી સબસાઇટ્સ જોવા માટે **Ultimate Multisite → Sites** પર જાઓ. દરેક સાઇટ પ્રકાર પ્રમાણે લેબલ થયેલી હોય છે અને તમે All Sites, Customer-Owned, Templates, અથવા Pending દ્વારા ફિલ્ટર કરી શકો છો.

![Sites list page](/img/admin/sites-list.png)

## નવી સાઇટ ઉમેરવી

નવી સાઇટ બનાવવા માટે **Add Site** બટન પર ક્લિક કરો. તમારે આ માહિતી આપવી પડશે:

- **Site Title** — નવી સાઇટનું નામ
- **Site URL** — સાઇટ માટે domain/path
- **Site Type** — તે ગ્રાહક સાઇટ છે, ટેમ્પલેટ છે, કે સામાન્ય સાઇટ છે

**Copy Site** વિકલ્પ તમને હાલના સાઇટ ટેમ્પલેટના આધારે નવી સાઇટ બનાવવા દે છે. જ્યારે enabled હોય, ત્યારે તમે શરૂઆતના બિંદુ તરીકે કયું ટેમ્પલેટ વાપરવું તે પસંદ કરી શકો છો. મીડિયા ફાઇલ્સ સામેલ કરવા માટે **Copy Media on Duplication** enabled છે તેની ખાતરી કરો.

## હાલની સાઇટનું સંચાલન

**Edit Site** પેજ ખોલવા માટે કોઈપણ સાઇટ પર **Manage** ક્લિક કરો. અહીં તમને મળશે:

### મૂળભૂત માહિતી

સાઇટનું નામ, પ્રકાર, site ID, અને વર્ણન. તમે mapped domain, સંકળાયેલ membership, અને સાઇટની માલિકી ધરાવતું ગ્રાહક ખાતું પણ જોઈ શકશો.

### સાઇટ વિકલ્પો

સાઇટની ક્ષમતાઓ અને મર્યાદાઓ configure કરો:

- **Visit limits** — સાઇટ મુલાકાતોની મહત્તમ સંખ્યા
- **User account limits** — દરેક user role દીઠ મર્યાદાઓ
- **Disk space** — સાઇટ માટે સ્ટોરેજ ક્વોટા
- **Custom domain** — આ સાઇટ માટે domain mapping enable કરો
- **Plugin and theme visibility** — કયા plugins અને themes દેખાય, છુપાયેલા હોય, અથવા પૂર્વ-સક્રિય હોય તે નિયંત્રિત કરો

ડિફોલ્ટ રૂપે, સાઇટ્સ membership level પર સેટ કરેલી મર્યાદાઓને અનુસરે છે. સાઇટ level પર મર્યાદાઓ સેટ કરવાથી membership settings override થશે.

### સંકળાણો

સાઇટ વિકલ્પોની નીચે તમને આ વિશે માહિતી મળશે:

- સાઇટ સાથે સંકળાયેલા **Mapped domains**
- સાઇટ જે **Membership** હેઠળ છે
- સાઇટ સાથે જોડાયેલું **Customer account**

### જમણી Sidebar

જમણી બાજુએ તમે આ કરી શકો છો:

- Toggle button વડે **સાઇટ enable/disable કરો**
- **સાઇટનો પ્રકાર બદલો** અથવા માલિકી ફરીથી સોંપો
- **સાઇટ image/thumbnail સેટ કરો** (સાઇટ ટેમ્પલેટ્સ માટે frontend પર દેખાય છે)
- **સાઇટ કાયમ માટે delete કરો**

:::warning
સાઇટ delete કરવું ઉલટાવી શકાતું નથી. સાઇટ અને તેની બધી સામગ્રી કાયમ માટે દૂર થઈ જશે.
:::
