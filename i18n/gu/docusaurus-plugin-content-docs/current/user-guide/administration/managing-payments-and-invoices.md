# ચુકવણીઓ અને ઇન્વૉઇસનું સંચાલન

## ચુકવણી સેટિંગ્સ

તમે ચુકવણીઓ મેળવવાનું શરૂ કરો તે પહેલાં, તમારે ચુકવણી-સંબંધિત સેટિંગ્સ ગોઠવવાની જરૂર પડશે. **Ultimate Multisite → Settings** પર જાઓ અને **Payment** ટૅબ પર ક્લિક કરો.

![ચુકવણી સેટિંગ્સ ટૅબ](/img/admin/settings-payments-top.png)

અહીં ચુકવણી સેટિંગ્સ પેજનું સંપૂર્ણ દૃશ્ય છે:

![ચુકવણી સેટિંગ્સ સંપૂર્ણ પેજ](/img/admin/settings-payments-full.png)

### સામાન્ય ચુકવણી વિકલ્પો

સામાન્ય સેટિંગ્સમાં તમે ગોઠવી શકો છો:

- **ચલણ** — વ્યવહારો માટે વપરાતું ડિફૉલ્ટ ચલણ
- **ચલણ સ્થાન** — ચલણનું પ્રતીક ક્યાં દેખાય છે (રકમ પહેલાં/પછી)

![ચુકવણી ગેટવે સેટિંગ્સ](/img/admin/settings-payments-options.png)

### ચુકવણી ગેટવે

Ultimate Multisite અનેક ચુકવણી ગેટવેને સપોર્ટ કરે છે. તમે Payment સેટિંગ્સ ટૅબમાંથી દરેકને સક્ષમ અને ગોઠવી શકો છો.

![ચુકવણી ગેટવે ગોઠવણી](/img/admin/settings-payments-gateways.png)

ઉપલબ્ધ ગેટવેમાં શામેલ છે:

- **Stripe** — Stripe મારફતે ક્રેડિટ કાર્ડ ચુકવણીઓ
- **PayPal** — PayPal ચુકવણીઓ
- **Manual** — ઑફલાઇન અથવા કસ્ટમ ચુકવણી પ્રક્રિયા માટે

દરેક ગેટવેનો પોતાનો ગોઠવણી વિભાગ હોય છે જ્યાં તમે API કીઓ અને અન્ય સેટિંગ્સ દાખલ કરો છો.

![વધારાના ગેટવે સેટિંગ્સ](/img/admin/settings-payments-gateways-2.png)

### Sandbox મોડ

લાઇવ જતા પહેલાં તમારી ચુકવણી ઇન્ટિગ્રેશનનું પરીક્ષણ કરવા માટે તમે **Sandbox Mode** સક્ષમ કરી શકો છો. જ્યારે sandbox mode સક્રિય હોય, ત્યારે કોઈ વાસ્તવિક ચાર્જ લેવામાં આવશે નહીં.

## ચુકવણીઓ જોવી

તમારા નેટવર્કમાંના બધા વ્યવહારો જોવા માટે Ultimate Multisite હેઠળના **Payments** પેજ પર જાઓ.

![ચુકવણીઓની યાદી](/img/admin/payments-list.png)

તમે ચુકવણીઓને સ્થિતિ (પૂર્ણ, બાકી, નિષ્ફળ, રિફંડ થયેલી) પ્રમાણે ફિલ્ટર કરી શકો છો અને ચોક્કસ વ્યવહારો માટે શોધી શકો છો.

ચુકવણીની સંપૂર્ણ વિગતો જોવા માટે તેના પર ક્લિક કરો, જેમાં લાઇન આઇટમ્સ, સંકળાયેલી સભ્યતા, ગ્રાહક માહિતી અને ચુકવણી ગેટવે ડેટાનો સમાવેશ થાય છે.

## ઇન્વૉઇસ

Ultimate Multisite ચુકવણીઓ માટે આપમેળે ઇન્વૉઇસ જનરેટ કરી શકે છે. તમે Payment સેટિંગ્સમાંથી ઇન્વૉઇસ ટેમ્પલેટ અને નંબરિંગ ફૉર્મેટ કસ્ટમાઇઝ કરી શકો છો.

ઇન્વૉઇસ કસ્ટમાઇઝેશન વિકલ્પોમાં શામેલ છે:

- **કંપનીનું નામ અને સરનામું** ઇન્વૉઇસ પર દર્શાવાયેલ
- **ઇન્વૉઇસ નંબરિંગ** ફૉર્મેટ અને ક્રમ
- **લોગો** ઇન્વૉઇસ હેડર પર દર્શાવાયેલ
- **કસ્ટમ ફૂટર ટેક્સ્ટ** નિયમો, નોંધો અથવા કાનૂની માહિતી માટે

ઇન્વૉઇસ ટેમ્પલેટ કસ્ટમાઇઝ કરવા માટે, **Ultimate Multisite → Settings → Payment** પર જાઓ અને ઇન્વૉઇસ-સંબંધિત સેટિંગ્સ શોધો.
