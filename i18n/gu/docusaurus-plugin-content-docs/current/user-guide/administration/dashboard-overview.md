# Dashboard ની ઝાંખી

Ultimate Multisite પાસે Ultimate Multisite મેનૂ હેઠળ એક સમર્પિત **Dashboard** છે, અને તે Network Dashboard માં widgets પણ ઉમેરે છે. આ લેખમાં તમે બંને જગ્યાએ શું જોશો તે વિશે માહિતી આપેલ છે.

## Network Dashboard

Ultimate Multisite **My Sites → Network Admin → Dashboard** પર મળતા network-level dashboard માં ઉપયોગી widgets ઉમેરે છે.

![Ultimate Multisite widgets સાથે Network Dashboard](/img/admin/network-dashboard-full.png)

દરેક widget શું પ્રદાન કરે છે તે અહીં છે:

**First Steps:** નવા installation પર દેખાય છે અને તમને payment method configure કરવા અને test account બનાવવા જેવા setup કાર્યો પૂર્ણ કરવાની યાદ અપાવે છે.

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Summary:** દિવસના signups અને revenue ની સંખ્યા દર્શાવે છે. MRR (Monthly Recurring Revenue) revenue-generating memberships ધરાવતા ગ્રાહકો પરથી અંદાજિત કુલ રકમ દર્શાવે છે.

![Summary widget](/img/admin/network-dashboard-summary.png)

**Activity Stream:** Ultimate Multisite પરની પ્રવૃત્તિઓ જેમ કે signups, cancellations, site creations, deletions અને અન્ય activities વિશે માહિતી આપે છે.

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** Network માં users અને sites નો ટૂંકો સારાંશ દર્શાવે છે અને નવી sites કે users બનાવવા માટે shortcuts આપે છે.

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** Ultimate Multisite ના નવીનતમ સમાચારો જેમ કે updates, critical bugs અને security fixes મેળવે છે અને દર્શાવે છે.

![News widget](/img/admin/network-dashboard-news.png)

## Ultimate Multisite Dashboard

જ્યારે network dashboard network વિશે સમગ્ર માહિતી રજૂ કરે છે, ત્યારે Ultimate Multisite dashboard (Ultimate Multisite મેનૂના ટોચના સ્તરે) service વિશે માહિતી રજૂ કરે છે.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Monthly Recurring Revenue Growth:** 12 મહિનાના calendar period દરમિયાન revenue નું monthly breakdown દર્શાવે છે. નવો revenue વાદળી રંગમાં બતાવવામાં આવે છે જ્યારે cancellations લાલ રંગમાં દેખાય છે.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**Date-range Support:** તમને જરૂરી માહિતી પર ધ્યાન કેન્દ્રિત કરવા માટે સમયગાળા પ્રમાણે dashboard data filter કરો. Date range નીચેના widgets માં શું દર્શાવાય છે તે નિયંત્રિત કરે છે.

![Date range અને dashboard stats](/img/admin/um-dashboard-date-range.png)

Dashboard widgets માં સમાવેશ થાય છે:

- **Revenue** — MRR અને per-product breakdowns સાથે સમયગાળા માટેનો કુલ revenue
- **New Memberships** — પસંદ કરેલા સમયગાળામાં કેટલી નવી memberships બનાવવામાં આવી
- **Signups by Countries** — Signups નું ભૌગોલિક breakdown
- **Signups by Forms** — કેટલા ગ્રાહકોએ દરેક checkout form દ્વારા signup કર્યું vs admins દ્વારા ઉમેરાયા
- **Most Visited Sites** — તમારા network પરની સૌથી વધુ મુલાકાત લેવાયેલી sites (આ માટે **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** પર visit counting enable કરવું જરૂરી છે)

![Dashboard statistics](/img/admin/um-dashboard-stats.png)
