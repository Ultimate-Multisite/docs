# હું મારું સબ્સ્ક્રિપ્શન કેવી રીતે રદ કરી શકું?

જો તમારી પાસે Ultimate Multisite લાઇસન્સ છે, તો તમે ગમે ત્યારે તેનું રિન્યુઅલ રદ કરી શકો છો. બસ નીચેના સ્ટેપ્સ ફોલો કરો:


<!-- Screenshot unavailable: Account page email with access link -->

ખરીદી દરમિયાન તમે આપેલ ઇમેઇલ અને તમારો પાસવર્ડ વાપરો.

<!-- Screenshot unavailable: Login page with email and password fields -->

આ તમારા Account Page ની શરૂઆતની સ્ક્રીન છે:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

તમારું સબ્સ્ક્રિપ્શન રદ કરવા માટે, ડાબી બાજુના મેનૂમાં _Renewing & Billing_ પર જાઓ.

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

સાઇડ વિન્ડો ખોલવા માટે જમણી બાજુના એરો પર ક્લિક કરો. પછી, **auto-renew** રદ કરવાનો વિકલ્પ પસંદ કરો.

સિસ્ટમ તમને એક કન્ફર્મેશન મેસેજ બતાવશે.

જો તમને ખાતરી છે કે તમે તમારું સબ્સ્ક્રિપ્શન રદ કરવા માંગો છો, તો બસ _**Cancel Renewals**_ બટન પર ક્લિક કરો. આ પછી, તમને એક ટૂંકા સર્વેનો જવાબ આપવાનું કહેવામાં આવશે.

થઈ ગયું! તમારું સબ્સ્ક્રિપ્શન આપોઆપ રિન્યુ નહીં થાય.

**તમારું સબ્સ્ક્રિપ્શન સમાપ્ત થાય ત્યાં સુધી તમારી પાસે માન્ય key રહેશે**. જો તમે તમારું સબ્સ્ક્રિપ્શન ફરીથી ચાલુ કરવા માંગો છો, તો તમારે તે મેન્યુઅલી કરવું પડશે.
