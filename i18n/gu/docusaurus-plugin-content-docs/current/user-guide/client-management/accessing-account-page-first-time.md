# હું મારા એકાઉન્ટ પેજને પ્રથમ વખત કેવી રીતે એક્સેસ કરી શકું?

Ultimate Multisite માં આપનું સ્વાગત છે!

અમારું plugin ખરીદ્યા પછી તમને ઈમેઈલમાં મળેલી લિંક દ્વારા તમારા એકાઉન્ટ પેજને એક્સેસ કરો:

<!-- Screenshot unavailable: Account page email with access link -->

ખરીદી દરમિયાન તમે આપેલ ઈમેઈલ અને તમારા ઈમેઈલમાં મળેલ પાસવર્ડનો ઉપયોગ કરો.

<!-- Screenshot unavailable: Login page with email and password fields -->

આ તમારું આ એકાઉન્ટમાં પ્રથમ વખત લૉગિન હોવાથી, તમને પાસવર્ડ બદલવાનું કહેવામાં આવશે. તેને બદલો અને _Save_ બટન પર ક્લિક કરો.

<!-- Screenshot unavailable: Change password form with Save button -->

તમારા એકાઉન્ટ પેજ પર, તમે બિલિંગ, રિન્યુઅલ્સ, કેન્સલેશન્સ અને બીજી ઘણી બાબતો જેવી તમારી બધી એકાઉન્ટ વિગતોનું સંચાલન કરી શકો છો.
