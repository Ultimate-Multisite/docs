# Ultimate Multisite ઇન્સ્ટોલ કરવું

:::note
આ ટ્યુટોરિયલ ધારે છે કે તમારી પાસે પહેલેથી WordPress Multisite ઇન્સ્ટોલ અને કોન્ફિગર થયેલું છે. કેવી રીતે કરવું તે જાણવા માટે, WP Beginner નું [આ ટ્યુટોરિયલ](https://www.wpbeginner.com/glossary/multisite/) જુઓ.
:::

## Plugin ઇન્સ્ટોલ કરવું

Ultimate Multisite [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) પર મફતમાં ઉપલબ્ધ છે.

તમારા **Network Admin Dashboard** માંથી, **Plugins → Add New Plugin** પર જાઓ.

![Network Admin Add New Plugin પેજ](/img/installation/add-new-plugin.png)

**"Ultimate Multisite"** (ચોક્કસ મેચ માટે અવતરણ ચિહ્નો સાથે) શોધો અને તે પહેલા પરિણામ તરીકે દેખાશે. **Install Now** પર ક્લિક કરો.

![Ultimate Multisite બતાવતા શોધ પરિણામો](/img/installation/search-ultimate-multisite.png)

ઇન્સ્ટોલ થયા પછી, તમારા સમગ્ર નેટવર્કમાં plugin ને સક્રિય કરવા માટે **Network Activate** પર ક્લિક કરો.

![Network Activate બટન સાથે ઇન્સ્ટોલ થયેલ Plugin](/img/installation/plugin-installed.png)

સક્રિયકરણ પછી, તમે આપમેળે Setup Wizard પર રીડાયરેક્ટ થશો.

![Plugin સક્રિય અને wizard પર રીડાયરેક્ટ](/img/installation/plugin-activated.png)

## Setup Wizard

Setup Wizard તમને લગભગ 10 મિનિટમાં Ultimate Multisite કોન્ફિગર કરવામાં માર્ગદર્શન આપશે.

### સ્વાગત

શરૂ કરવા માટે **Get Started** પર ક્લિક કરો.

![Setup Wizard સ્વાગત સ્ક્રીન](/img/installation/wizard-welcome.png)

### પ્રી-ઇન્સ્ટોલ તપાસ

આ પગલું તમારી સિસ્ટમ માહિતી અને WordPress ઇન્સ્ટોલેશન તપાસે છે જેથી ખાતરી થાય કે તે Ultimate Multisite ની જરૂરિયાતો પૂરી કરે છે. જો બધું બરાબર લાગે, તો **Go to the Next Step** પર ક્લિક કરો.

![સિસ્ટમ જરૂરિયાતો બતાવતી પ્રી-ઇન્સ્ટોલ તપાસ](/img/installation/wizard-pre-install-checks.png)

### ઇન્સ્ટોલેશન

ઇન્સ્ટોલર જરૂરી database tables બનાવશે અને `sunrise.php` ફાઇલ ઇન્સ્ટોલ કરશે જે Ultimate Multisite ને કામ કરવા માટે જરૂરી છે. આગળ વધવા માટે **Install** પર ક્લિક કરો.

![Database tables અને sunrise.php બતાવતું ઇન્સ્ટોલેશન પગલું](/img/installation/wizard-installation.png)

### તમારી કંપની

તમારી કંપનીની માહિતી ભરો અને તમારી ડિફોલ્ટ ચલણ સેટ કરો. આ માહિતી તમારા WaaS પ્લેટફોર્મમાં વિવિધ જગ્યાએ વપરાશે. પૂર્ણ થયા પછી **Continue** પર ક્લિક કરો.

![તમારી કંપની કોન્ફિગરેશન પગલું](/img/installation/wizard-your-company.png)

### ડિફોલ્ટ કન્ટેન્ટ

આ પગલું તમને પૂર્વનિર્ધારિત templates, products, અને અન્ય સ્ટાર્ટર કન્ટેન્ટ ઇન્સ્ટોલ કરવા દે છે. Ultimate Multisite ની સુવિધાઓથી પરિચિત થવાની આ એક ઉત્તમ રીત છે. ડિફોલ્ટ કન્ટેન્ટ ઉમેરવા માટે **Install** પર ક્લિક કરો, અથવા જો તમે શરૂઆતથી શરૂ કરવા માંગતા હો તો આ પગલું છોડી દો.

![ડિફોલ્ટ કન્ટેન્ટ ઇન્સ્ટોલેશન પગલું](/img/installation/wizard-default-content.png)

### ભલામણ કરેલ Plugins

વૈકલ્પિક રીતે ભલામણ કરેલ સાથી plugins ઇન્સ્ટોલ કરો. તેમને ઉમેરવા માટે **Install** પર ક્લિક કરો અથવા ચાલુ રાખવા માટે છોડી દો.

![ભલામણ કરેલ plugins પગલું](/img/installation/wizard-recommended-plugins.png)

### તૈયાર!

બસ આટલું જ! તમારું Ultimate Multisite ઇન્સ્ટોલેશન પૂર્ણ થયું. હવે તમે **Network Admin Dashboard** માંથી તમારું Website as a Service પ્લેટફોર્મ બનાવવાનું શરૂ કરી શકો છો.

![Setup પૂર્ણ - તૈયાર સ્ક્રીન](/img/installation/wizard-ready.png)

![Ultimate Multisite સક્રિય સાથે Network Admin Dashboard](/img/installation/network-dashboard.png)

હવે આગળ વધો અને મજા કરો!
