# Ultimate Multisite જરૂરિયાતો

Ultimate Multisite ખરીદતા પહેલા તમે પહેલો પ્રશ્ન પૂછી શકો છો કે અમારું plugin ચલાવવા માટે શું જરૂરિયાતો છે. સારા સમાચાર એ છે કે Ultimate Multisite ચલાવવા માટે માત્ર થોડી મૂળભૂત જરૂરિયાતો છે: એક **ડોમેન નામ**, એક સારી **hosting કંપની** અને WordPress **Multisite** installation. બસ, આટલું જ તમારે જોઈએ!

## ડોમેન નામ

સૌપ્રથમ, ચાલો ડોમેન નામ વિશે વાત કરીએ.

ડોમેન નામ પસંદ કરતી વખતે, ધ્યાનમાં રાખો કે તમારી ચોક્કસ niche શું છે અને Ultimate Multisite વડે તમે શું બનાવશો: શું તમે એજન્સી બનાવવા માંગો છો? દુકાન જેવી વેબસાઇટ્સ વેચવા માંગો છો? કે પછી WordPress web hosting? Ultimate Multisite સાથે બધું શક્ય છે.

અમારું plugin subscription સેવાઓ, payment collection, checkout forms, discount vouchers અને ગ્રાહક સંદેશાવ્યવહાર પૂરું પાડે છે. WordPress Multisite install કરવા, configure કરવા અને maintain કરવા માટે જરૂરી મોટાભાગનું કામ Ultimate Multisite સરળ બનાવે છે.

જ્યાં સુધી તમે સેવા તરીકે વેબસાઇટ્સ વેચો છો, ત્યાં સુધી તમે કોઈપણ niche પસંદ કરો તેને અમારા plugin વડે મેનેજ કરી શકાય છે! ખાતરી કરો કે તમારા business plan સાથે મેળ ખાતું ડોમેન નામ પસંદ કરો.

## Hosting કંપની

હવે તમારી પાસે એક આકર્ષક ડોમેન નામ છે જે તમારી niche અને business રજૂ કરે છે, હવે તમારે Ultimate Multisite સાથે તમારા network ને host કરવા માટે એક સારી web hosting કંપની પસંદ કરવાની જરૂર છે. બજારમાં કેટલાક સારા વિકલ્પો છે. કેટલીક hosting કંપનીઓ Ultimate Multisite સાથે સારી રીતે integrate થાય છે, જે તમે શું બનાવવાનું plan કરો છો તેના આધારે નિર્ણાયક પરિબળ હોઈ શકે છે.

Ultimate Multisite કોઈપણ hosting કંપની પર કામ કરવું જોઈએ. ફરક માત્ર એટલો છે કે _WPMU DEV Hosting_, _WP Engine_, _Closte_, _Cloudways_, _Gridpane_ અને _Runcloud_ જેવી કેટલીક hosting કંપનીઓમાં domain mapping માટે વધુ સારું integration છે. અન્ય hosting માટે, તમારે custom domains manually map કરવાની જરૂર પડી શકે છે. AutoSSL support કરતી hosting કંપની મેળવવી પણ સારો વિચાર છે. _Closte_, _Runcloud_ અને _cPanel_ તેને support કરે છે.

Server specifications ની વાત કરીએ તો, કોઈ ચોક્કસ formula નથી કારણ કે તે તમે કઈ features અને functions ઓફર કરશો અને તમારા network ના કદ પર આધાર રાખે છે. Shared hosting શરૂઆત માટે સારી હોઈ શકે છે, પણ જેમ જેમ તમારું network વધવા લાગે, તેમ તમને કદાચ dedicated VPS ની જરૂર પડશે.

## WordPress Multisite

હવે, ચાલો WordPress Multisite installation વિશે થોડી વાત કરીએ.

WordPress Multisite તમને એક જ installation પર sites નું network રાખવાની મંજૂરી આપે છે. આ built-in feature છે, પણ તે default રીતે active નથી. તમારે આ feature તમારા WordPress installation અને web hosting control panel માં enable કરવાની જરૂર છે.

Multisite enabled WordPress **Ultimate Multisite નો આધાર છે**. Ultimate Multisite નો મુખ્ય ઉપયોગ users ને સેવા તરીકે websites મેનેજ કરવામાં અને વેચવામાં મદદ કરવાનો હોવાથી, આ Multisite enabled WordPress પર થવું જરૂરી છે.

અમારી પાસે **એક વિગતવાર article** છે જે step by step શીખવાડે છે કે તમારા WordPress installation પર Multisite feature enable કરવા માટે શું કરવાની જરૂર છે.

તમારા WordPress પર Multisite enable થયા પછી, હવે Ultimate Multisite install કરવાનો સમય છે. Ultimate Multisite યોગ્ય રીતે install કરવા માટે, તમારી પાસે આ હોવું જોઈએ:

  * WordPress: v5.3+ (ભલામણ: છેલ્લું stable version)

  * PHP: 7.4.x (> 8.0 માટે support ટૂંક સમયમાં આવશે)

  * MySQL: v5+ (ભલામણ: 5.6, જો તમારી hosting કંપની પર 8.0 ઉપલબ્ધ ન હોય તો)

Ultimate Multisite માટે આ જ software જરૂરિયાતો છે.

ધ્યાનમાં રાખો કે Ultimate Multisite PHP 8.0 સાથે કામ કરી શકે છે પણ અમે તેને PHP 7.4.x સાથે ચલાવવાની ભલામણ કરીએ છીએ.

ઉપરાંત, અમે તેને **main domains** પર install કરવાની ભલામણ કરીએ છીએ, subdomains અથવા subdirectories પર નહીં. તે subdomains/subdirectories પર કામ કરી શકે છે, પણ કેટલીક errors આવી શકે છે.

Ultimate Multisite કેવી રીતે install કરવું તેની વિગતવાર માર્ગદર્શિકા માટે, **અહીં ક્લિક કરો**.
