# Ultimate Multisite નું ભાષાંતર

Ultimate Multisite અંગ્રેજીમાં વિકસાવવામાં આવ્યું છે, પરંતુ આ plugin સંપૂર્ણપણે અનુવાદ કરી શકાય તેવું છે અને અમે ભાષાંતર યોગદાનનું હંમેશા સ્વાગત કરીએ છીએ.

## મુખ્ય Plugin ના ભાષાંતર

મુખ્ય Ultimate Multisite plugin નું ભાષાંતર સત્તાવાર **WordPress.org ભાષાંતર પ્લેટફોર્મ** દ્વારા કરવામાં આવે છે:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

આ WordPress સમુદાયની માનક ભાષાંતર પ્રણાલી છે. જ્યારે ભાષાંતર પૂરતી મંજૂરીની સપાટીએ પહોંચે છે, ત્યારે તે WordPress plugin અપડેટ્સ દ્વારા તમામ Ultimate Multisite વપરાશકર્તાઓને આપમેળે વિતરિત થાય છે.

**અમને તમારી મદદની જરૂર છે!** જો તમે અંગ્રેજી સિવાયની કોઈ ભાષા બોલો છો, તો કૃપા કરીને ભાષાંતરમાં યોગદાન આપવાનું વિચારો. થોડીક strings નું ભાષાંતર કરવાથી પણ મદદ મળે છે. શરૂ કરવા માટે:

1. [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/) ની મુલાકાત લો
2. તમારી ભાષા પસંદ કરો
3. "Contribute Translation" પર ક્લિક કરો
4. તમારે મફત WordPress.org એકાઉન્ટની જરૂર પડશે

જો તમે તમારી ભાષા માટે ભાષાંતર editor બનવા માંગો છો (ભાષાંતરને મંજૂર કરી શકે તેવા), તો [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) દ્વારા સંપર્ક કરો.

## Addon ના ભાષાંતર

Addon plugins નું ભાષાંતર અલગથી આ જગ્યાએ કરવામાં આવે છે:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Addon ના ભાષાંતર હાલમાં શરૂઆતના બિંદુ તરીકે AI દ્વારા બનાવવામાં આવ્યા છે. તે તમામ સમર્થિત ભાષાઓને આવરી લે છે પરંતુ તેમાં અણઘડ શબ્દરચના અથવા અચોક્કસતા હોઈ શકે છે. અમે માનવ અનુવાદકોને સમીક્ષા કરવા અને તેમને વધુ કુદરતી અને સચોટ બનાવવા માટે સૂચનો આપવા માટે આવકારીએ છીએ.

## સ્થાનિક ભાષાંતર

જો તમે upstream યોગદાન આપવાને બદલે તમારા પોતાના પ્રોજેક્ટ માટે સ્થાનિક રીતે ભાષાંતર બનાવવાનું પસંદ કરો છો, તો બે વિકલ્પો છે:

### Poedit નો ઉપયોગ

તમે `.pot` ફાઇલનું ભાષાંતર કરવા માટે ડેસ્કટોપ એપ્લિકેશન **[Poedit](https://poedit.net)** નો ઉપયોગ કરી શકો છો:

1. [poedit.net](https://poedit.net) પરથી Poedit ડાઉનલોડ કરો
2. `wp-content/plugins/ultimate-multisite/lang/` માંથી `.pot` ફાઇલ ખોલો
3. Poedit ઇન્ટરફેસમાં strings નું ભાષાંતર કરો
4. સેવ કરો -- Poedit `.po` અને `.mo` ફાઇલો બનાવે છે
5. આઉટપુટ ફાઇલોને `lang` ફોલ્ડરમાં મૂકો

### Loco Translate નો ઉપયોગ

**[Loco Translate](https://wordpress.org/plugins/loco-translate/)** WordPress plugin તમને કોઈપણ ડેસ્કટોપ ટૂલ્સ વિના admin panel માંથી સીધું ભાષાંતર કરવા દે છે. તે તમારા માટે ભાષાંતર ફાઇલો બનાવવા અને કમ્પાઇલ કરવાનું સંભાળે છે.
