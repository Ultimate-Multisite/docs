# Ultimate Multisite Hooks અને filters

Ultimate Multisite ઘણા બધા functions અને database queries થી બનેલું છે. તેમાં hooks અને filters છે જે તમને હાલના functions માં ફેરફાર કરવા અથવા નવો code ઉમેરવાની સુવિધા આપે છે.

અમે advanced users ને તેમના network ને વધુ customize કરવામાં મદદ કરવા માટે આ hooks અને filters ની એક સંદર્ભ યાદી બનાવી છે. નીચેની link પર click કરીને તમે Github પરથી આ યાદી જોઈ શકો છો.
