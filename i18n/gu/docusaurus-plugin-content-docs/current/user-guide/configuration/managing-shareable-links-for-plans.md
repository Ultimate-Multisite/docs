# પ્લાન માટે શેર કરી શકાય તેવી લિંક્સનું સંચાલન

Ultimate Multisite તમને તમારા ક્લાયન્ટ્સ સાથે સીધી પ્લાન લિંક્સ શેર કરવાનો વિકલ્પ આપે છે. આ કરવા માટે, તમારા કોઈપણ પ્લાનના edit પેજ પર જાઓ અને _Click to copy the Shareable Link_ બટન પર ક્લિક કરો.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_પ્લાનના Edit પેજ પર, તમને શેર કરી શકાય તેવી લિંક્સ બનાવવાનું બટન મળશે_

ધ્યાન રાખો કે આ પદ્ધતિ લિંક બનાવવા માટે default billing frequency નો ઉપયોગ કરે છે. આનો અર્થ એ છે કે જો તમારી default billing માસિક છે, ઉદાહરણ તરીકે, તો અંતે તમને આવી લિંક મળશે: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

જો તમે ઈચ્છો છો કે શેર કરી શકાય તેવી લિંક ત્રિમાસિક પર default હોય, તો તમારે અંતમાં 3 ઉમેરવું પડશે

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

_premium_ પ્લાનને વાર્ષિક શેડ્યૂલ પર પસંદ કરવા માટે, તમે 1 ઉમેરો અને પછી શબ્દ:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

યાદ રાખો કે તમે અલગ-અલગ billing frequencies માટે માત્ર એક જ લિંક રાખી શકતા નથી. તમારે તમારી દરેક billing frequency માટે એક ચોક્કસ લિંક શેર કરવી જોઈએ.

એકવાર તમે તમારા ગ્રાહકો સાથે લિંક શેર કરો, તે તેમને તમારા signup flow ના પ્રથમ પગલા પર રીડાયરેક્ટ કરશે.
