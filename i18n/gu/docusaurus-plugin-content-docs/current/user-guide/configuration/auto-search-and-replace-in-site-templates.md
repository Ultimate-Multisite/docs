# સાઇટ ટેમ્પ્લેટ્સમાં ઑટો સર્ચ અને રિપ્લેસ (v2)

_**આ ટ્યુટોરિયલ માટે WP UItimo version 2.x જરૂરી છે.**_

Ultimate Multisite ની સૌથી શક્તિશાળી સુવિધાઓમાંની એક છે રજિસ્ટ્રેશન ફોર્મમાં ટેક્સ્ટ, કલર અને સિલેક્ટ ફિલ્ડ્સ ઉમેરવાની ક્ષમતા. એકવાર આપણે આ ડેટા મેળવી લઈએ, પછી આપણે તેનો ઉપયોગ પસંદ કરેલા સાઇટ ટેમ્પ્લેટના અમુક ભાગોમાં કન્ટેન્ટ પહેલેથી ભરવા માટે કરી શકીએ છીએ. પછી, જ્યારે નવી સાઇટ પબ્લિશ થાય છે, ત્યારે Ultimate Multisite રજિસ્ટ્રેશન દરમિયાન દાખલ કરેલી ખરેખર માહિતી સાથે placeholders ને બદલી નાખે છે.

ઉદાહરણ તરીકે, તમે તમારી ટેમ્પ્લેટ સાઇટ્સ placeholders સાથે બનાવી શકો છો. Placeholders ડબલ curly braces વડે ઘેરાયેલા હોવા જોઈએ - {{placeholder_name}}.

પછી, તમે આ ડેટા મેળવવા માટે મેચિંગ રજિસ્ટ્રેશન ફિલ્ડ ઉમેરી શકો છો.

તમારા ગ્રાહક રજિસ્ટ્રેશન દરમિયાન આ ફિલ્ડ ભરી શકશે.

Ultimate Multisite પછી ગ્રાહક દ્વારા આપેલા ડેટા સાથે placeholders ને આપમેળે બદલી નાખશે.

## **"placeholders થી ભરેલું ટેમ્પ્લેટ" સમસ્યાનું સમાધાન**

આ બધું સરસ છે, પણ આપણને એક ખરાબ સમસ્યા આવે છે: હવે આપણી સાઇટ ટેમ્પ્લેટ્સ - જે આપણા ગ્રાહકો જોઈ શકે છે - બેડોળ placeholders થી ભરેલી છે જે કંઈ ખાસ જણાવતા નથી.

આનું સમાધાન કરવા માટે, અમે placeholders માટે નકલી મૂલ્યો સેટ કરવાનો વિકલ્પ આપીએ છીએ, અને જ્યારે તમારા ગ્રાહકો ટેમ્પ્લેટ સાઇટ્સ જુએ છે ત્યારે અમે આ મૂલ્યોનો ઉપયોગ સર્ચ અને રિપ્લેસ માટે કરીએ છીએ.

ટેમ્પ્લેટ placeholders એડિટર ઍક્સેસ કરવા માટે **Ultimate Multisite > Settings > Sites** પર જાઓ, અને પછી, સાઇડબારમાં Edit Placeholders લિંક પર ક્લિક કરો.

![Sites settings સાઇડબારમાં Edit Placeholders લિંક](/img/config/settings-sites.png)

આ તમને placeholders કન્ટેન્ટ એડિટર પર લઈ જશે, જ્યાં તમે placeholders અને તેમના સંબંધિત કન્ટેન્ટ ઉમેરી શકો છો.

![Placeholder નામો અને મૂલ્યો સાથે Placeholders કન્ટેન્ટ એડિટર](/img/config/settings-sites.png)
