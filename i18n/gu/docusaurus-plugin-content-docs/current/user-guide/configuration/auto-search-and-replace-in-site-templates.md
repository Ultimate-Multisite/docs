# સાઇટ ટેમ્પ્લેટ્સમાં આપમેળે શોધો અને બદલો (v2)

_**આ ટ્યુટોરિયલ માટે WP UItimo version 2.x જરૂરી છે.**_

Ultimate Multisite ની સૌથી શક્તિશાળી સુવિધાઓમાંની એક નોંધણી ફોર્મમાં મનગમતું લખાણ, રંગ અને પસંદગી ક્ષેત્રો ઉમેરવાની ક્ષમતા છે. એકવાર આપણે તે ડેટા કૅપ્ચર કરી લઈએ, પછી પસંદ કરેલા સાઇટ ટેમ્પ્લેટના ચોક્કસ ભાગોમાં સામગ્રીને પહેલેથી ભરવા માટે તેનો ઉપયોગ કરી શકીએ છીએ. પછી, જ્યારે નવી સાઇટ પ્રકાશિત થાય છે, ત્યારે Ultimate Multisite નોંધણી દરમિયાન દાખલ કરેલી વાસ્તવિક માહિતીથી પ્લેસહોલ્ડર્સને બદલશે.

ઉદાહરણ તરીકે, તમે તમારી ટેમ્પ્લેટ સાઇટ્સ પ્લેસહોલ્ડર્સ સાથે બનાવી શકો છો. પ્લેસહોલ્ડર્સ ડબલ કર્લી બ્રેસિસથી ઘેરાયેલા ઉમેરવા જોઈએ - {{placeholder_name}}.

પછી, તમે તે ડેટા કૅપ્ચર કરવા માટે સરળતાથી મેળ ખાતું નોંધણી ક્ષેત્ર ઉમેરી શકો છો

તમારો ગ્રાહક પછી નોંધણી દરમિયાન તે ક્ષેત્ર ભરી શકશે.

Ultimate Multisite પછી ગ્રાહક દ્વારા આપવામાં આવેલા ડેટાથી પ્લેસહોલ્ડર્સને આપમેળે બદલી દેશે.

## **"પ્લેસહોલ્ડર્સથી ભરેલા ટેમ્પ્લેટ"ની સમસ્યાનું સમાધાન** {#solving-the-template-full-of-placeholders-problem}

આ બધું ઉત્તમ છે, પરંતુ અમને એક અણગમતી સમસ્યા આવે છે: હવે અમારી સાઇટ ટેમ્પ્લેટ્સ - જેને અમારા ગ્રાહકો મુલાકાત લઈ શકે છે - અણગમતા પ્લેસહોલ્ડર્સથી ભરેલી હોય છે, જે બહુ કંઈ જણાવતા નથી.

તેનું સમાધાન કરવા માટે, અમે પ્લેસહોલ્ડર્સ માટે નકલી મૂલ્યો સેટ કરવાનો વિકલ્પ પ્રદાન કરીએ છીએ, અને તમારા ગ્રાહકો મુલાકાત લઈ રહ્યા હોય ત્યારે ટેમ્પ્લેટ સાઇટ્સ પર તેમની સામગ્રી શોધવા અને બદલવા માટે અમે તે મૂલ્યોનો ઉપયોગ કરીએ છીએ.

તમે **Ultimate Multisite > સેટિંગ્સ > સાઇટ્સ** પર જઈને, સાઇટ ટેમ્પ્લેટ વિકલ્પો વિસ્તાર સુધી સ્ક્રોલ કરીને, અને પછી **પ્લેસહોલ્ડર્સ સંપાદિત કરો** લિંક પર ક્લિક કરીને ટેમ્પ્લેટ પ્લેસહોલ્ડર્સ એડિટર ઍક્સેસ કરી શકો છો.

![સાઇટ્સ સેટિંગ્સ પેજમાં સાઇટ ટેમ્પ્લેટ વિકલ્પો વિસ્તાર](/img/config/settings-sites-templates-section.png)

તે તમને પ્લેસહોલ્ડર્સના સામગ્રી એડિટર પર લઈ જશે, જ્યાં તમે પ્લેસહોલ્ડર્સ અને તેમની સંબંધિત સામગ્રી ઉમેરી શકો છો.
