# વારંવાર પૂછાતા પ્રશ્નો

અહીં કેટલાક **વારંવાર પૂછાતા પ્રશ્નો** છે જે support requests દરમિયાન વારંવાર આવે છે.

  * **Ultimate Multisite શું છે?**

Ultimate Multisite એક WordPress plugin છે, જે WordPress multisite installs માટે બનાવવામાં આવ્યું છે, જે તમારા WordPress install ને premium sites ના network માં રૂપાંતરિત કરે છે – [WordPress.com](https://WordPress.com) જેવું – જે clients ને માસિક, ત્રિમાસિક, અથવા વાર્ષિક ફી દ્વારા sites બનાવવાની મંજૂરી આપે છે (તમે Free plans પણ બનાવી શકો છો).

  * **શું Ultimate Multisite Multisite વિના કામ કરશે?**

ના. Ultimate Multisite ને WordPress multisite, subdirectory, અથવા subdomain ની જરૂર છે.

  * **Ultimate Multisite માટે શું requirements છે?**

Ultimate Multisite ને જરૂર છે:

**WordPress Version** : v5.3+ (ભલામણ: છેલ્લું Stable Version)  
**PHP Version** : 7.4.x (> 8.0 નો Support ટૂંક સમયમાં આવશે)  
**MySQL Version** : v5+ (ભલામણ: Version 5.6, જો તમારા hosting provider પર 8.0 ઉપલબ્ધ ન હોય તો)

_Multisite (subdomain અથવા subdirectory) પણ activate કરવું જરૂરી છે._ [WordPress Multisite કેવી રીતે Install કરવું](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **શું Ultimate Multisite Subdirectories સાથે Multisite સાથે કામ કરશે?**

હા. Ultimate Multisite subdirectory installs અને subdirectory installs બંનેને support કરે છે.

  * **શું Ultimate Multisite Subdomains સાથે Multisite સાથે કામ કરશે?**

હા. Ultimate Multisite subdirectory installs અને subdomain installs બંનેને support કરે છે.

  * **શું હું Ultimate Multisite નો ઉપયોગ કરીને users કયા Plugins અને Themes વાપરી શકે તે મર્યાદિત કરી શકું?**

હા. Ultimate Multisite સાથે તમે પસંદ કરી શકો છો કે તમારા network પર દરેક plan માટે કયા plugins અને themes ઉપલબ્ધ છે.

  * **શું હું Ultimate Multisite નો ઉપયોગ કરીને user તેની site પર કેટલા posts બનાવી શકે તે મર્યાદિત કરી શકું?**

હા. Ultimate Multisite સાથે તમે તમારા network પર **દરેક ઉપલબ્ધ post type** માટે post quotas સ્થાપિત કરી શકો છો. તમે admin કેટલા users ને invite કરી શકે, **disk space ની માત્રા,** અને ઘણું બધું પણ મર્યાદિત કરી શકો છો.

  * **શું Ultimate Multisite Site Templates ને support કરે છે?**

હા. Ultimate Multisite સાથે તમે જેટલા Site Templates જોઈએ તેટલા બનાવી શકો છો. Site Templates themes નથી, તે તમારા network પર સામાન્ય sites છે. તમે plugins, themes activate કરી શકો છો, અને તે Site Template ને તમે જે રીતે ઈચ્છો તે રીતે customize કરી શકો છો. જો user તે template પસંદ કરે છે, તો તેની site તે template ની copy તરીકે બનાવવામાં આવશે, બધા customizations સાથે.

તમે કોઈ ચોક્કસ template ને plan માટે પણ set કરી શકો છો. આમ કરવાથી, તે plan પર sign up કરનાર દરેક user ની site તે Site Template ની duplicate તરીકે બનાવવામાં આવશે.

  * **શું Ultimate Multisite WooCommerce સાથે integrate થાય છે?**

હા!

જો તમે ઈચ્છો છો કે તમારા network users ને WooCommerce ઉપલબ્ધ હોય, જેથી તેઓ stores set up કરી શકે, તો તે Ultimate Multisite નો ઉપયોગ કરીને શક્ય છે. તમે દરેક plan tier પર તમારા clients કેટલા products બનાવી શકશે તે પણ મર્યાદિત કરી શકો છો.

Ultimate Multisite માટે subscription payments ના સંદર્ભમાં, તમે [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) નો ઉપયોગ કરી શકો છો જેથી તમારા Ultimate Multisite clients તમારી main site પર WooCommerce gateways નો ઉપયોગ કરીને pay કરી શકે. આ રીતે, તમે WooCommerce માટે develop થયેલ payment gateways ની વિશાળ library નો ઉપયોગ કરી શકો છો અને તેનો ઉપયોગ તમારા Ultimate Multisite clients ને bill કરવા માટે કરી શકો છો.

  * **શું WPMUDEV Pro Sites માંથી subscriptions migrate કરવાનો કોઈ રસ્તો છે?**

હા, Ultimate Multisite હવે તમને [migrate કરવામાં મદદ કરવા માટે એક helpful tool](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) આપે છે તમારા Pro Sites network ને.

  * **મારા Plan Edit Page પર "Allowed Plugins" tab ખાલી કેમ છે?**

Ultimate Multisite ફક્ત તમને તમારા network પર network-activated ન હોય તેવા plugins ને access આપવા અથવા deny કરવાની મંજૂરી આપે છે. જ્યારે કોઈ plugin network-activated હોય છે, ત્યારે તેની પ્રકૃતિ મુજબ, તે તમારા network ની દરેક site પર default રીતે load થશે.

જો તમે "Allowed Plugins" tab પર કોઈ plugins listed જોતા નથી, તો ચકાસો કે desired plugin network-active છે કે નહીં.

  * **શું હું મારું પોતાનું Pricing Table design વાપરી શકું?**

હા. version 1.4.0 થી, Ultimate Multisite template overrides ને support કરે છે.

  * **શું Ultimate Multisite WP Engine સાથે કામ કરે છે?**

હા. Ultimate Multisite હંમેશા WP Engine દ્વારા supported હતું, પરંતુ domain mapping કામ કરાવવા માટે admin તરફથી manual extra steps ની જરૂર હતી (WP Engine panel માં domain add કરવું). **version 1.6.0 થી આ હવે જરૂરી નથી.** Ultimate Multisite WP Engine API સાથે directly integrate થાય છે અને domains mapping પછી automatically panel માં add થાય છે.

  * **તમે કયા hosting provider ની ભલામણ કરો છો?**

Ultimate Multisite કોઈપણ hosting provider સાથે કામ કરવું જોઈએ. ફક્ત તફાવત એ છે કે Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane અને [Runcloud.io](http://Runcloud.io) જેવી કેટલીક hosting company domain mapping capabilities માટે tighter integration ધરાવે છે જેથી તેઓ out of the box કામ કરે છે. અન્ય hostings માટે, તમારે domain mapping manually set up કરવું પડી શકે છે.
