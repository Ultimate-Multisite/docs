# AffiliateWP

Multisite Ultimate: AffiliateWP Integration

તમારા Ultimo Network નો ક્લાયન્ટ બેઝ વધારવા માટે શક્તિશાળી AffiliateWP નો ઉપયોગ કરો!

## Installation {#installation}

1. 'ultimate-multisite-git ' ને '/wp-content/plugins/' ડિરેક્ટરીમાં અપલોડ કરો.
2. WordPress માં 'Plugins' મેનુ દ્વારા પ્લગઇન એક્ટિવેટ કરો.
3. પ્લગઇનને સેટ કરવા માટે સ્ટેપ-બાય-સ્ટેપ વિઝાર્ડને અનુસરો.

## Usage Guide {#usage-guide}

[AffiliateWP](https://affiliatewp.com/) તમારા ઓનલાઈન બિઝનેસ માટે એફિલિએટ પ્રોગ્રામ બનાવતી વખતે એક મુખ્ય સોલ્યુશન બની ગયું છે. Ultimate Multisite: AffiliateWP એ એક સેતુ છે જે તમને તમારા પ્રીમિયમ નેટવર્ક માટે પણ આવું જ કંઈક બનાવવામાં મદદ કરશે.

## Getting Started {#getting-started}

સૌથી પહેલા, તમારે [**AffiliateWP**](https://affiliatewp.com/) **એક્ટિવ** હોવું **જરૂરી છે**! જો તમારી પાસે **તે પ્લગઇન ન હોય જે તેની સાથે ઇન્ટિગ્રેટ થાય**, તો તમે અમારું એડ-ઓન વાપરી શકશો નહીં. જો તમે તમારા ક્લાયન્ટ્સને તેમની સાઇટ્સ પર પણ તેનો ઉપયોગ કરવા દેવાની યોજના નથી બનાવી રહ્યા, તો AffiliateWP ફક્ત તમારી મુખ્ય સાઇટ પર જ એક્ટિવ હોવું જોઈએ.

આ ઉપરાંત, તમારે **Ultimate Multisite** અને **Ultimate Multisite: AffiliateWP** બંને **network-wide** **એક્ટિવેટ** કરવાની જરૂર પડશે.

## Adding Ultimate Multisite as an AffiliateWP Integration {#adding-ultimate-multisite-as-an-affiliatewp-integration}

તમારી મુખ્ય સાઇટ પર, Affiliates Settings પર જાઓ, અને Integrations ટેબ પર, Ultimate Multisite ચેક કરો. આ [AffiliateWP](https://affiliatewp.com/) ને Ultimate Multisite sign-up flow માંથી વિઝિટ્સ અને કન્વર્શન્સ ટ્રેક કરવાનું શરૂ કરવા કહેશે.

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_નીચે Ultimate Multisite બોક્સ ચેક કરો અને સેવ કરો_

આનાથી લગભગ બધું પૂરું થઈ ગયું. રેફરલ લિંક્સમાંથી આવતી વિઝિટ્સ ટ્રેક કરવામાં આવશે અને જ્યારે તે યુઝર પાછળથી પોતાનો પહેલો પેમેન્ટ કરશે, ત્યારે કન્વર્ઝન બનાવવામાં આવશે અને કમિશન તે એફિલિએટને આપેલું કરવામાં આવશે જે આ કન્વર્ઝન માટે જવાબદાર છે.

જોકે, ફક્ત નવા બનાવેલા યુઝર પાસેથી મળતો પહેલો પેમેન્ટ જ ટ્રેક થશે. પુનરાવર્તિત (recurring) રેફરલ પેમેન્ટ્સને સક્ષમ કરવા માટે, તમારે [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) ની જરૂર પડશે.

## Enabling Recurring Referrals {#enabling-recurring-referrals}

[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) ઇન્સ્ટોલ કર્યા પછી, Affiliates Settings ના Recurring Referrals ટેબ પર જાઓ અને recurring referrals નો ઉપયોગ સક્ષમ કરો.

<!-- Screenshot unavailable: AffiliateWP Recurring Referrals settings with Enable Recurring Referrals toggle -->

_જેટલું સરળ હોય તેટલું: Recurring Referrals ને એનેબલ કરો અને તમે તૈયાર છો!_

ફરીથી, આટલું જ પૂરતું છે!

હવે, જ્યારે પણ કોઈ રેફર કરેલા એકાઉન્ટ પર નવો પેમેન્ટ પ્રાપ્ત થશે, ત્યારે જવાબદાર એફિલિએટને આપમેળે તેનો પોતાનો હિસ્સો મળી જશે!
