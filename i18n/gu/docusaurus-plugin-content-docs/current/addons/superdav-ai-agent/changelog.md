# ફેરફાર ઇતિહાસ {#changelog}

## 1.18.0 — 2026-06-29ના રોજ રિલીઝ થયું {#1180--released-on-2026-06-29}

### નવું {#new}

- **Google Calendar સાધનો** — સમયપત્રક-જાગૃત ઓટોમેશન માટે કૉન્ફિગર કરેલા કેલેન્ડર અને ઇવેન્ટ્સ વાંચો
- **સંપર્ક મેપિંગ અને હાજરી સહાયકો** — કેલેન્ડર ભાગ લેનારાઓને સાઇટ વપરાશકર્તાઓ અને સંપર્કો સાથે મેળવો
- **માનવીય મંજૂરી ગેટ્સ અને રિમાઇન્ડર રેકોર્ડ્સ** — સમીક્ષા માટે ઓટોમેશન થોભાવો અને ડુપ્લિકેટ સૂચનાઓ ટાળો
- **TextBee SMS પ્રદાતા** — TextBee મારફતે કૉન્ફિગર કરેલી ટેક્સ્ટ-સંદેશ સૂચનાઓ મોકલો
- **અદ્યતન સાથી પેકેજ** — WordPress.org બિલ્ડથી અલગ રીતે વિતરિત વિશ્વસનીય-ડેવલપર ફાઇલસિસ્ટમ, ડેટાબેઝ, WP-CLI, REST ડિસ્પેચર, plugin બિલ્ડર, git સ્નેપશોટ, વપરાશકર્તા-મેનેજમેન્ટ, અને બેન્ચમાર્ક સાધનો ઉમેરો

### સુધારેલું {#improved}

- **મેનેજ્ડ Superdav સેવા સેટઅપ** — સમર્થિત સાઇટ્સ માટે હોસ્ટેડ સેવા endpoints અને આપમેળે કનેક્શન પ્રોવિઝનિંગ ઉમેરો
- **રિલીઝ પેકેજિંગ** — અલગ core અને Advanced ZIPs બનાવો, બંનેને GitHub પર પ્રકાશિત કરો, અને WordPress.org પર માત્ર core પેકેજ મોકલો

### સુધારેલું {#fixed}

- **AI વિનંતી વિશ્વસનીયતા** — model પસંદગી, વિનંતી સમયસમાપ્તિઓ, ડિફૉલ્ટ સેટઅપ model, reasoning-text હેન્ડલિંગ, અને અમાન્ય સાધન કૉલ્સ માટે પુનઃપ્રયાસ માર્ગદર્શન સુધારો
- **કેલેન્ડર અને રિમાઇન્ડર મજબૂતીકરણ** — Google Calendar ટોકન્સ અને રિમાઇન્ડર ડીડુપ્લિકેશન મજબૂત બનાવો
- **ઓનબોર્ડિંગ અને મંજૂરી રિઝ્યુમ્સ** — frontend ઓનબોર્ડિંગ લોન્ચ અને પુષ્ટિ થયેલી ક્ષમતા રિઝ્યુમ્સ સુધારો
- **WordPress.org પેકેજિંગ સમીક્ષા સમસ્યાઓ** — core રિલીઝ માટે પેકેજિંગ સમીક્ષા પ્રતિસાદનું નિરાકરણ કરો

## 1.16.0 — 2026-05-20ના રોજ રિલીઝ થયું {#1160--released-on-2026-05-20}

### નવું {#new-1}

- **Generate Logo SVG ક્ષમતા** — Theme Builder હવે namespace-safe sanitisation સાથે કસ્ટમ logo SVGs જનરેટ અને એમ્બેડ કરી શકે છે
- **ડિસ્કવરી ઇન્ટરવ્યુમાં ફોટો અપલોડ** — Theme Builder ડિસ્કવરી ઇન્ટરવ્યુમાં હવે વધુ સમૃદ્ધ ડિઝાઇન સંદર્ભ માટે ફોટો અપલોડ પગલું સામેલ છે
- **Validate Palette Contrast ક્ષમતા** — theme પર લાગુ કરતાં પહેલાં WCAG પાલન માટે રંગ જોડીઓ તપાસો
- **હૉસ્પિટાલિટી મેનૂઝ** — Theme Builder હવે હૉસ્પિટાલિટી વ્યવસાયો માટે માળખાબદ્ધ ખાદ્ય અને પીણા મેનૂ પૃષ્ઠો જનરેટ કરી શકે છે
- **ડેસ્કટૉપ અને મોબાઇલ પૂર્વાવલોકન રેન્ડરિંગ** — ડિઝાઇન-દિશા પસંદગી દરમિયાન તમારી ડિઝાઇનનું ડેસ્કટૉપ અને મોબાઇલ ઉપકરણો પર પૂર્વાવલોકન કરો
- **નેવિગેશન લેબલ પેરામીટર** — Create Menu ક્ષમતા હવે પૃષ્ઠ શીર્ષકથી અલગ એક વિશિષ્ટ `navigation_label`ને સમર્થન આપે છે
- **Tier 1 સાધન ઉપલબ્ધતા** — sd-ai-agent/site-scrape હવે Theme Builderમાં ડિફૉલ્ટ રૂપે ઉપલબ્ધ Tier 1 સાધન છે

### સુધારેલું {#fixed-1}

- **AI Client કૅશ** — હવે ક્રોસ-વિનંતી સ્થાયિત્વ માટે transients દ્વારા બૅક થયેલું છે, જે લાંબા સમય ચાલતા agent કાર્યોમાં ડેટા ગુમાવવાનું અટકાવે છે
- **Plugin પંક્તિ ક્રિયા લિંક્સ** — સ્પષ્ટતા માટે સુધારવામાં અને ફરી નામ આપવામાં આવ્યા

## 1.10.0 — 2026-05-05ના રોજ રિલીઝ થયું {#1100--released-on-2026-05-05}

### નવું {#new-2}

- **Tavily ઇન્ટરનેટ શોધ** — Brave Search સાથે વધુ સમૃદ્ધ ઇન્ટરનેટ શોધ પરિણામો માટે Tavilyને શોધ પ્રદાતા તરીકે ઉમેરો
- **Theme-જાગૃત બિલ્ટ-ઇન કુશળતાઓ** — Block Themes, Classic Themes, Kadence Blocks, અને Kadence Theme કુશળતા માર્ગદર્શિકાઓ હવે plugin સાથે આવે છે
- **સાઇટ બિલ્ડર સંપર્ક ફોર્મ ક્ષમતા** — chat ઇન્ટરફેસમાંથી સીધા કોઈપણ પૃષ્ઠમાં સંપર્ક ફોર્મ ઉમેરો

### સુધારેલું {#improved-1}

- **WooCommerce ઇન્ટિગ્રેશન રિફેક્ટર કર્યું** — હવે વધુ સારી વિશ્વસનીયતા અને સુસંગતતા માટે મૂળ WooCommerce APIsનો ઉપયોગ કરે છે
- **પ્રદાતા યાદી આપમેળે રિફ્રેશ થાય છે** — જ્યારે કોઈપણ plugin સક્રિય અથવા નિષ્ક્રિય થાય છે

### સુધારેલું {#fixed-2}

- **navigate-to ક્ષમતા** — કેટલાક admin પૃષ્ઠો પર અનંત રિલોડ લૂપ સુધાર્યો
- **list-posts ક્ષમતા** — હવે કેટેગરી અને ટૅગ નામોને slugsમાં યોગ્ય રીતે રિઝોલ્વ કરે છે
- **WP-CLI કમાન્ડ્સ** — અગાઉના રિફેક્ટર પછી ગુમ થયેલા namespace aliases પુનઃસ્થાપિત કર્યા
- **ઇવેન્ટ ઓટોમેશન** — જ્યાં automation કોષ્ટકો હજી બનાવવામાં આવ્યા નથી એવી સાઇટ્સને સરળતાથી હેન્ડલ કરે છે
- **memory-save ક્ષમતા** — હવે system instruction builderમાં યોગ્ય namespace prefixનો ઉપયોગ કરે છે
- **Scalar સાધન પરિણામો** — હવે AIને પરત કરતાં પહેલાં યોગ્ય રીતે આવરિત થાય છે
- **ઉપયોગ આંકડા** — હવે જૂના સંસ્કરણોમાંથી અપગ્રેડ કરતી વખતે legacy ક્ષમતા key formatને યોગ્ય રીતે હેન્ડલ કરે છે
