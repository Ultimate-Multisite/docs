# Superdav AI Agent

Superdav AI Agent એ અદ્યતન WordPress AI સહાયક છે, જે Gratis AI Agent ને થીમ-જાગૃત કુશળતાઓ, સુધારેલી ઇન્ટરનેટ શોધ ક્ષમતાઓ, સાઇટ બિલ્ડર સુવિધાઓ અને વૈકલ્પિક ઓટોમેશન ઇન્ટિગ્રેશન્સ સાથે વિસ્તારે છે. આવૃત્તિ 1.18.0 Google Calendar વાંચન સાધનો, સંપર્ક અને હાજરી આપનારાઓનું મેપિંગ, માનવીય મંજૂરી ગેટ્સ, રિમાઇન્ડર ડીડુપ્લિકેશન, TextBee SMS સૂચનાઓ, સંચાલિત Superdav સેવા પ્રોવિઝનિંગ અને વિશ્વસનીય ડેવલપર વર્કફ્લોઝ માટે અલગથી વિતરિત અદ્યતન સહાયક પેકેજ ઉમેરે છે.

## મુખ્ય સુવિધાઓ

- **થીમ-જાગૃત કુશળતાઓ** — તમારી સક્રિય WordPress થીમ અને ઇન્સ્ટોલ કરેલા પ્લગિન્સના આધારે માર્ગદર્શન અને ક્ષમતાઓને આપમેળે અનુકૂળ બનાવે છે
- **બહુવિધ ઇન્ટરનેટ શોધ પ્રદાતાઓ** — વધુ સમૃદ્ધ શોધ પરિણામો અને અદ્યતન સંશોધન ક્ષમતાઓ માટે Brave Search અને Tavily માંથી પસંદ કરો
- **સાઇટ બિલ્ડર સંપર્ક ફોર્મ્સ** — તમારા ઇન્સ્ટોલ કરેલા ફોર્મ પ્લગિનનો ઉપયોગ કરીને ચેટ ઇન્ટરફેસમાંથી સીધા સંપૂર્ણ કાર્યક્ષમ સંપર્ક ફોર્મ્સ બનાવો
- **સુધારેલ સાઇટ બિલ્ડર** — વધુ સારી પ્લગિન શોધ અને ભૂલ પુનઃપ્રાપ્તિ સાથે સુધારેલી બહુ-તબક્કાવાળી સાઇટ બિલ્ડ યોજનાઓ
- **બિલ્ટ-ઇન થીમ કુશળતાઓ** — Block Themes, Classic Themes, Kadence Blocks અને Kadence Theme માટે વિશિષ્ટ માર્ગદર્શન
- **સુધારેલ WooCommerce એકીકરણ** — વધુ સારી વિશ્વસનીયતા અને સુસંગતતા માટે મૂળ WooCommerce APIs નો ઉપયોગ કરવા માટે રિફેક્ટર કર્યું છે
- **આપમેળે પ્રદાતા શોધ** — પ્લગિન્સ અને થીમ્સ સક્રિય અથવા નિષ્ક્રિય થાય ત્યારે ઉપલબ્ધ પ્લગિન્સ અને થીમ્સને આપમેળે શોધે અને રિફ્રેશ કરે છે
- **કસ્ટમ લોગો જનરેશન** — આપમેળે નેમસ્પેસ-સુરક્ષિત સેનિટાઇઝેશન સાથે કસ્ટમ SVG લોગો બનાવો અને એમ્બેડ કરો
- **પેલેટ કોન્ટ્રાસ્ટ માન્યતા** — તમારી થીમ પર લાગુ કરતા પહેલાં WCAG ઍક્સેસિબિલિટી અનુપાલન માટે રંગ જોડીઓને ચકાસો
- **ફોટો-વર્ધિત ડિસ્કવરી** — ડિઝાઇન દિશાને પ્રભાવિત કરવા માટે Theme Builder ડિસ્કવરી ઇન્ટરવ્યુ દરમિયાન ફોટા અપલોડ કરો
- **હોસ્પિટાલિટી મેનૂઝ** — રેસ્ટોરાં, કેફે અને હોસ્પિટાલિટી વ્યવસાયો માટે સંરચિત ખોરાક અને પીણાં મેનૂ પૃષ્ઠો બનાવો
- **ડેસ્કટોપ અને મોબાઇલ પૂર્વાવલોકનો** — ડિઝાઇન-દિશા પસંદગી દરમિયાન ડેસ્કટોપ અને મોબાઇલ ઉપકરણો પર તમારી ડિઝાઇનનું પૂર્વાવલોકન કરો
- **સ્તર 1 સાઇટ સ્ક્રેપિંગ** — સ્પર્ધક અને પ્રેરણાદાયક વેબસાઇટ્સનું વિશ્લેષણ કરવા માટે sd-ai-agent/site-scrape હવે મૂળભૂત રીતે ઉપલબ્ધ છે
- **Google Calendar વાંચન સાધનો** — રૂપરેખાંકિત કૅલેન્ડર્સ અને ઇવેન્ટ્સ વાંચો જેથી ઓટોમેશન્સ આગામી સમયપત્રક સંદર્ભ વિશે વિચાર કરી શકે
- **સંપર્ક અને હાજરી આપનારાઓનું મેપિંગ** — અનુગામી કાર્ય જનરેટ કરતા પહેલાં કૅલેન્ડર ભાગીદારોને જાણીતા સાઇટ વપરાશકર્તાઓ અને સંપર્કો સાથે મેળવો
- **માનવીય મંજૂરી ગેટ્સ** — સંદેશા મોકલવા, કાર્ય ફરી શરૂ કરવા અથવા ફેરફારો કરવા પહેલાં સમીક્ષા માટે સંવેદનશીલ ઓટોમેશન્સને થોભાવો
- **રિમાઇન્ડર રેકોર્ડ્સ અને ડીડુપ્લિકેશન** — પહેલેથી મોકલાયેલા રિમાઇન્ડર્સને ટ્રૅક કરો જેથી પુનરાવર્તિત ઓટોમેશન રન્સ એ જ પ્રાપ્તકર્તાને બે વાર સૂચિત ન કરે
- **TextBee SMS સૂચનાઓ** — SMS પ્રદાતા સક્ષમ હોય ત્યારે TextBee મારફતે રૂપરેખાંકિત ટેક્સ્ટ-સંદેશ સૂચનાઓ મોકલો
- **Managed Superdav service provisioning** — મેન્યુઅલ એન્ડપોઇન્ટ રૂપરેખાંકનની જગ્યાએ આપમેળે સેટઅપ સાથે સમર્થિત સાઇટ્સને હોસ્ટેડ Superdav એન્ડપોઇન્ટ્સ સાથે જોડો
- **અદ્યતન સહાયક પેકેજ** — ફાઇલસિસ્ટમ, ડેટાબેઝ, WP-CLI, REST ડિસ્પેચર, પ્લગિન બિલ્ડર, git સ્નૅપશોટ, વપરાશકર્તા-વ્યવસ્થાપન અને બેન્ચમાર્ક વર્કફ્લોઝ માટે અલગથી વિતરિત વિશ્વસનીય-ડેવલપર સાધનો

## આવશ્યકતાઓ

- WordPress 6.0 અથવા તેથી ઊંચું
- PHP 8.1 અથવા તેથી ઊંચું
- WordPress Multisite (નેટવર્ક-સક્રિય) અથવા સિંગલ-સાઇટ ઇન્સ્ટોલેશન
- સક્રિય AI પ્રદાતા કનેક્ટર પ્લગિન (Anthropic, OpenAI અથવા સુસંગત)
- Gratis AI Agent v1.9.0 અથવા તેથી ઊંચું (Superdav AI Agent, Gratis AI Agent ને વિસ્તારે છે)

## ઇન્સ્ટોલેશન

1. પ્લગિન ફાઇલોને `/wp-content/plugins/superdav-ai-agent` પર અપલોડ કરો
2. WordPress માં **Plugins** સ્ક્રીન મારફતે પ્લગિન સક્રિય કરો
3. Gratis AI Agent પણ સક્રિય છે તેની ખાતરી કરો
4. **Gratis AI Agent → Settings** પર જાઓ અને જરૂર મુજબ તમારા ઇન્ટરનેટ શોધ પ્રદાતા, સંચાલિત Superdav સેવા કનેક્શન, Google Calendar અથવા TextBee ઓળખપત્રો રૂપરેખાંકિત કરો

WordPress.org પેકેજમાં મુખ્ય Superdav AI Agent સુવિધાઓ સામેલ છે. અદ્યતન સહાયક પેકેજ એવા પર્યાવરણો માટે અલગથી બનાવવામાં અને વિતરિત કરવામાં આવે છે જ્યાં વિશ્વસનીય ડેવલપર સાધનો સ્પષ્ટ રીતે ઇન્સ્ટોલ અને અધિકૃત કરવામાં આવ્યા હોય.

## ઝડપી શરૂઆત

સક્રિય થયા પછી, Superdav AI Agent તમારી Gratis AI Agent ઇન્સ્ટોલેશનને થીમ-જાગૃત કુશળતાઓ અને વધારાની ક્ષમતાઓ સાથે આપમેળે સુધારે છે. WordPress એડમિન સાઇડબારમાં **Gratis AI Agent** પેનલ ખોલો અને નવી સુવિધાઓનો ઉપયોગ શરૂ કરો.

**ઉદાહરણ સૂચનાઓ:**

```
સંપર્ક પૃષ્ઠમાં સંપર્ક ફોર્મ ઉમેરો.
```

```
નવીનતમ WordPress સુરક્ષા શ્રેષ્ઠ પદ્ધતિઓ માટે વેબ શોધો.
```

```
સક્રિય થીમના બ્લોક પેટર્ન્સનો ઉપયોગ કરીને હીરો વિભાગ બનાવો.
```

```
આવતીકાલની રૂપરેખાંકિત Google Calendar ઇવેન્ટ્સની સમીક્ષા કરો અને હાજરી આપનારાઓ માટે રિમાઇન્ડર્સનો ડ્રાફ્ટ બનાવો.
```

```
હું સંદેશને મંજૂરી આપ્યા પછી SMS રિમાઇન્ડર મોકલો.
```

## થીમ-જાગૃત કુશળતાઓ

Superdav AI Agent તમારી સક્રિય થીમ અને ઇન્સ્ટોલ કરેલા પ્લગિન્સને આપમેળે શોધે છે, અને સહાયકના સંદર્ભમાં યોગ્ય થીમ-જાગૃત કુશળતા લોડ કરે છે. ઉપલબ્ધ કુશળતાઓમાં સમાવેશ થાય છે:

| કુશળતા | ક્યા પર લાગુ પડે છે |
|---|---|
| **Block Themes** | WordPress બ્લોક-આધારિત (સંપૂર્ણ સાઇટ સંપાદન) આર્કિટેક્ચરનો ઉપયોગ કરતી થીમ્સ |
| **Classic Themes** | PHP ટેમ્પલેટ્સ અને ક્લાસિક એડિટરનો ઉપયોગ કરતી પરંપરાગત WordPress થીમ્સ |
| **Kadence Blocks** | Kadence Blocks પ્લગિન સક્રિય હોય તેવી સાઇટ્સ |
| **Kadence Theme** | સક્રિય થીમ તરીકે Kadence થીમ ધરાવતી સાઇટ્સ |

દરેક કુશળતા પર વિગતવાર માર્ગદર્શન માટે [થીમ-જાગૃત કુશળતાઓ](../../user-guide/configuration/theme-aware-skills) જુઓ.

## ઇન્ટરનેટ શોધ પ્રદાતાઓ

Superdav AI Agent v1.10.0 ઇન્ટરનેટ શોધ ક્વેરીઝ માટે Brave Search ના વિકલ્પ તરીકે **Tavily** ઉમેરે છે. બંને પ્રદાતાઓ રીઅલ-ટાઇમ શોધ પરિણામો આપે છે અને **Gratis AI Agent → Settings → Advanced** માં રૂપરેખાંકિત કરી શકાય છે.

| પ્રદાતા | સુવિધાઓ | સેટઅપ |
|---|---|---|
| **Brave Search** | ઝડપી, ગોપનીયતા-કેન્દ્રિત શોધ પરિણામો | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | વધુ સમૃદ્ધ શોધ પરિણામો, અદ્યતન સંશોધન ક્ષમતાઓ | [Tavily API](https://tavily.com/) |

રૂપરેખાંકન સૂચનાઓ માટે [ઇન્ટરનેટ શોધ](../../user-guide/configuration/internet-search) જુઓ.

## કૅલેન્ડર, મંજૂરી અને સૂચના વર્કફ્લોઝ

Superdav AI Agent v1.18.0 સામગ્રી અને સાઇટ-બિલ્ડિંગ કાર્યોની બહાર ઓટોમેશન વર્કફ્લોઝને વિસ્તારે છે:

- **Google Calendar સાધનો** જ્યારે કૅલેન્ડર ઓળખપત્રો ઉપલબ્ધ હોય ત્યારે એજન્ટને રૂપરેખાંકિત કૅલેન્ડરો અને ઇવેન્ટની વિગતો વાંચવા દે છે.
- **સંપર્ક અને હાજરી સહાયકો** રિમાઇન્ડર અથવા અનુગામી કાર્યો તૈયાર કરતા પહેલાં ઇવેન્ટ સહભાગીઓને WordPress વપરાશકર્તાઓ અથવા જાણીતા સંપર્કો સાથે મેપ કરે છે.
- **માનવીય મંજૂરી ગેટ્સ** પ્રસ્તાવિત ક્રિયાની અધિકૃત વપરાશકર્તા સમીક્ષા કરે ત્યાં સુધી ઓટોમેશનને થોભવાની મંજૂરી આપે છે.
- **રિમાઇન્ડર રેકોર્ડ્સ** જ્યારે ઓટોમેશન ફરીથી અજમાવવામાં આવે અથવા વારંવાર શેડ્યૂલ કરવામાં આવે ત્યારે ડુપ્લિકેટ કૅલેન્ડર અથવા સૂચના રિમાઇન્ડર્સ અટકાવે છે.
- **TextBee SMS સપોર્ટ** જ્યારે TextBee પ્રદાતા રૂપરેખાંકિત હોય ત્યારે ટેક્સ્ટ-સંદેશ સૂચનાઓ સક્ષમ કરે છે.

આ સાધનોનો ઉપયોગ શેડ્યૂલ-જાગૃત સહાય, હાજરી અનુસરણ અને રિમાઇન્ડર વર્કફ્લોઝ માટે કરો, જેમાં સંવેદનશીલ સંદેશા મોકલાય તે પહેલાં હજી પણ માનવીય સમીક્ષાની જરૂર હોય છે.

## સાઇટ બિલ્ડર સંપર્ક ફોર્મ્સ

સાઇટ બિલ્ડર એજન્ટ હવે ચેટ ઇન્ટરફેસમાંથી સીધા સંપર્ક ફોર્મ્સ બનાવી શકે છે. સાઇટ બિલ્ડરને સંપર્ક ફોર્મ બનાવવા માટે ફક્ત કહો, અને તે આપમેળે તમારું ઇન્સ્ટોલ કરેલું ફોર્મ plugin (Contact Form 7, WPForms, Fluent Forms, અથવા Gravity Forms) શોધી કાઢશે અને ઉપયોગ માટે તૈયાર ફોર્મ જનરેટ કરશે.

વિગતવાર માર્ગદર્શન માટે [સાઇટ બિલ્ડર સંપર્ક ફોર્મ્સ](../../user-guide/administration/site-builder-contact-forms) જુઓ.

## Theme Builder સુવિધાઓ

Superdav AI Agent v1.16.0 માં Theme Builder શક્તિશાળી નવી ક્ષમતાઓનો સમાવેશ કરે છે:

### ફોટો અપલોડ સાથે ડિસ્કવરી ઇન્ટરવ્યૂ

Theme Builder ડિસ્કવરી ઇન્ટરવ્યૂમાં હવે ફોટો અપલોડ પગલું સામેલ છે જ્યાં તમે દૃશ્ય પ્રેરણા આપી શકો છો. Theme Builder ને તમારી ડિઝાઇન દિશા સમજવામાં મદદ કરવા માટે તમારા બ્રાન્ડ, સ્પર્ધક સાઇટ્સ અથવા ડિઝાઇન પ્રેરણાના ફોટા અપલોડ કરો.

વિગતવાર માર્ગદર્શન માટે [ડિસ્કવરી ઇન્ટરવ્યૂ](./theme-builder/discovery-interview.md) જુઓ.

### ડેસ્કટોપ અને મોબાઇલ પ્રિવ્યૂ સાથે ડિઝાઇન દિશા

Theme Builder તમારી સંપૂર્ણ theme જનરેટ કરે તે પહેલાં ડેસ્કટોપ અને મોબાઇલ બંને ઉપકરણો પર તમારી ડિઝાઇનની સમીક્ષા કરો. બહુવિધ પ્રતિભાવશીલ બ્રેકપોઇન્ટ્સ પર રીઅલ-ટાઇમ પ્રિવ્યૂ જુઓ.

વિગતવાર માર્ગદર્શન માટે [ડિઝાઇન દિશા](./theme-builder/design-direction.md) જુઓ.

### હોસ્પિટાલિટી મેનુઓ

Theme Builder હવે રેસ્ટોરાં, કેફે, બાર અને અન્ય હોસ્પિટાલિટી વ્યવસાયો માટે રચિત ખાદ્ય અને પીણાં મેનુ પૃષ્ઠો જનરેટ કરી શકે છે. મેનુઓમાં કિંમતો, વર્ણનો, આહાર સંબંધિત માહિતી અને એલર્જન ચેતવણીઓ સામેલ છે.

વિગતવાર માર્ગદર્શન માટે [હોસ્પિટાલિટી મેનુઓ](./theme-builder/hospitality-menus.md) જુઓ.

### નવી ક્ષમતાઓ

- **[લોગો SVG જનરેટ કરો](./abilities/generate-logo-svg.md)** — આપમેળે સેનિટાઇઝેશન સાથે કસ્ટમ SVG લોગો બનાવો
- **[પેલેટ કોન્ટ્રાસ્ટ માન્ય કરો](./abilities/validate-palette-contrast.md)** — WCAG અનુપાલન માટે રંગ જોડીઓ તપાસો
- **[મેનુ બનાવો](./abilities/create-menu.md)** — અલગ નેવિગેશન લેબલ્સ સાથે નેવિગેશન મેનુઓ બનાવો

ઉપલબ્ધ સાધનો અને Tier 1 સાઇટ-સ્ક્રેપ સાધન વિશેની માહિતી માટે [સાધનો અને રૂપરેખાંકન](./theme-builder/tools.md) જુઓ.

## રૂપરેખાંકન

બધી સેટિંગ્સ WordPress એડમિનમાં **Gratis AI Agent → Settings** હેઠળ છે. Superdav AI Agent નીચેના રૂપરેખાંકન વિકલ્પો ઉમેરે છે:

| સેટિંગ | વર્ણન |
|---|---|
| **ઇન્ટરનેટ શોધ પ્રદાતા** | Brave Search અને Tavily વચ્ચે પસંદ કરો (વૈકલ્પિક) |
| **Tavily API Key** | Tavily ઇન્ટરનેટ શોધ માટે API key (જો Tavily પસંદ કર્યું હોય) |
| **મેનેજ્ડ Superdav સેવા** | સમર્થિત સાઇટ્સને આપમેળે endpoint પ્રોવિઝનિંગ સાથે હોસ્ટેડ Superdav સેવા સાથે જોડો |
| **Google Calendar ઓળખપત્રો** | કૅલેન્ડર ઍક્સેસ રૂપરેખાંકિત કરો જેથી એજન્ટ કૅલેન્ડરો અને ઇવેન્ટ્સ વાંચી શકે |
| **TextBee SMS પ્રદાતા** | મંજૂર SMS સૂચના વર્કફ્લોઝ માટે TextBee ઓળખપત્રો રૂપરેખાંકિત કરો |

## અદ્યતન સાથી પેકેજ

Superdav AI Agent v1.18.0 વિશ્વસનીય ડેવલપર વર્કફ્લોઝ માટે અલગથી વિતરિત અદ્યતન સાથી પેકેજ ઉમેરે છે. આ ક્ષમતાઓ WordPress.org કોર પેકેજમાં બંડલ નથી. સાથી પેકેજ ફક્ત એવી સાઇટ્સ પર ઇન્સ્ટોલ કરો જ્યાં એડમિનિસ્ટ્રેટર્સ એજન્ટને ડેવલપર ટૂલિંગ ઍક્સેસ કરવા માટે સ્પષ્ટપણે વિશ્વાસ કરે છે.

અદ્યતન સાધનોમાં ફાઇલસિસ્ટમ ઓપરેશન્સ, ડેટાબેઝ ઍક્સેસ, WP-CLI, REST ડિસ્પેચર વર્કફ્લોઝ, plugin બિલ્ડર યુટિલિટીઝ, git સ્નૅપશોટ્સ, વપરાશકર્તા-વ્યવસ્થાપન સહાયકો અને બેન્ચમાર્ક ટૂલિંગ સામેલ છે. આ સાધનોને વિશ્વસનીય એડમિનિસ્ટ્રેટર્સ સુધી મર્યાદિત રાખો અને પ્રોડક્શન સાઇટ્સ પર તેમને સક્ષમ કરતા પહેલાં મંજૂરી-ગેટ રૂપરેખાંકનની સમીક્ષા કરો.
