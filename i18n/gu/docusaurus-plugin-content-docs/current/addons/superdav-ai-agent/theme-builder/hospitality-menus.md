# હોસ્પિટાલિટી મેનુઝ

**હોસ્પિટાલિટી મેનુઝ** ફીચર Theme Builder ને તમારા WordPress સાઇટમાં સીધા જ સંરચિત (structured) ફૂડ અને બેવરેજ મેનુ પેજીસ બનાવવાની અને એમ્બેડ કરવાની મંજૂરી આપે છે.

## ઝાંખી (Overview)

Theme Builder હવે હોસ્પિટાલિટી વ્યવસાયો, જેમાં રેસ્ટોરન્ટ્સ, કેફે, બાર અને કેટરિંગ સેવાઓનો સમાવેશ થાય છે, તેમના માટે વ્યાવસાયિક, સંરચિત મેનુ પેજીસ બનાવી શકે છે. આ મેનુ તમારા સાઇટના ડિઝાઇનમાં સંપૂર્ણ રીતે સંકલિત (integrated) હોય છે અને તેને સરળતાથી અપડેટ અને મેનેજ કરી શકાય છે.

## સપોર્ટેડ હોસ્પિટાલિટી પ્રકારો (Supported Hospitality Types)

- **રેસ્ટોરન્ટ્સ** — સંપૂર્ણ સેવા આપતા ડાઇનિંગ મેનુ
- **કેફે** — કોફી અને હળવા નાસ્તાના મેનુ
- **બાર અને લાઉન્જ** — પીણાં અને એપેટાઇઝર મેનુ
- **બેકરીઝ** — પેસ્ટ્રી અને બ્રેડ મેનુ
- **કેટરિંગ સેવાઓ** — ઇવેન્ટ મેનુ વિકલ્પો
- **ફૂડ ટ્રક્સ** — મોબાઇલ ફૂડ સર્વિસ મેનુ
- **બ્રુઅરીઝ અને વાઇનરીઝ** — વર્ણન સાથેના પીણાંના મેનુ

## મેનુનું માળખું (Menu Structure)

### મેનુ કેટેગરીઝ (Menu Categories)

મેનુને કેટેગરીમાં ગોઠવવામાં આવે છે:

- **એપેટાઇઝર્સ** — સ્ટાર્ટર્સ અને નાની વાનગીઓ
- **એન્ટ્રીઝ** — મુખ્ય કોર્સ
- **સાઇડ્સ** — સાથમાં લેવાતી વસ્તુઓ અને શાકભાજી
- **ડેઝર્ટ્સ** — મીઠાઈઓ
- **બેવરેજીસ** — પીણાં (આલ્કોહોલિક અને નોન-આલ્કોહોલિક)
- **સ્પેશિયલ્સ** — દૈનિક અથવા સિઝનલ સ્પેશિયલ્સ

### મેનુ આઇટમ ફોર્મેટ (Menu Item Format)

દરેક મેનુ આઇટમમાં નીચેની વિગતો શામેલ હોય છે:

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### આઇટમ ફીલ્ડ્સ (Item Fields)

| Field | Type | Description |
|-------|------|-------------|
| `name` | string | વાનગી અથવા પીણાંનું નામ |
| `description` | string | આઇટમનું વિગતવાર વર્ણન |
| `price` | string | કિંમત (કરન્સી સાથે ફોર્મેટ કરેલ) |
| `dietary_info` | array | આહાર સંબંધિત ગુણધર્મો (Vegan, Gluten-Free, વગેરે) |
| `allergens` | array | સામાન્ય એલર્જન (Nuts, Shellfish, વગેરે) |
| `availability` | string | ક્યારે ઉપલબ્ધ છે (Daily, Seasonal, વગેરે) |

## હોસ્પિટાલિટી મેનુ બનાવવું (Creating Hospitality Menus)

### પગલું ૧: મેનુની માહિતી આપો (Step 1: Provide Menu Information)

Theme Builder ને તમારા મેનુ વિશે જણાવો:

```
Create a restaurant menu for my Italian restaurant with appetizers,
pasta dishes, main courses, and desserts. Include prices and descriptions.
```

### પગલું ૨: Theme Builder મેનુ બનાવે છે (Step 2: Theme Builder Generates Menu)

Theme Builder:

૧. એક સંરચિત મેનુ પેજ બનાવે છે
૨. તેને તમારા સાઇટના થીમ સાથે મેચ કરવા માટે ડિઝાઇન કરે છે
૩. આઇટમ્સને કેટેગરીમાં ગોઠવે છે
૪. કિંમતો અને વર્ણનો ફોર્મેટ કરે છે
૫. આહાર અને એલર્જનની માહિતી ઉમેરે છે

### પગલું ૩: સમીક્ષા કરો અને કસ્ટમાઇઝ કરો (Step 3: Review and Customize)

તમે કરી શકો છો:

૧. મેનુ આઇટમ્સ અને કિંમતોમાં એડિટ કરો
૨. કેટેગરી ઉમેરો અથવા દૂર કરો
૩. કેટેગરીમાં આઇટમ્સનો ક્રમ બદલો
૪. વર્ણનો અને આહારની માહિતી અપડેટ કરો
૫. સ્ટાઇલિંગ અને લેઆઉટ એડજસ્ટ કરો

## મેનુ ડિસ્પ્લે વિકલ્પો (Menu Display Options)

### સંપૂર્ણ મેનુ પેજ (Full Menu Page)

તમારા સંપૂર્ણ મેનુને દર્શાવતું એક સમર્પિત પેજ:

- કેટેગરી દ્વારા ગોઠવાયેલું
- સર્ચ કરી શકાય તેવું અને ફિલ્ટર કરી શકાય તેવું
- પ્રિન્ટ-ફ્રેન્ડલી લેઆઉટ
- મોબાઇલ-રેસ્પોન્સિવ ડિઝાઇન

### મેનુ વિજેટ (Menu Widget)

અન્ય પેજીસ પર મેનુ સેક્શન્સ એમ્બેડ કરો:

- હોમપેજ પર ફીચર કરેલ આઇટમ્સ
- સાઇડબાર પર દૈનિક સ્પેશિયલ્સ
- બાર પેજ પર બેવરેજ મેનુ
- ફૂટર પર ડેઝર્ટ શોકેસ

### મેનુ PDF (Menu PDF)

ડાઉનલોડ કરી શકાય તેવું PDF મેનુ જનરેટ કરો:

- વ્યાવસાયિક ફોર્મેટિંગ
- પ્રિન્ટ કરવા માટે તૈયાર ગુણવત્તા
- છબીઓ અને વર્ણનો શામેલ છે
- શેર કરવા અને ઈમેલ કરવા માટે સરળ

## આહાર અને એલર્જનની માહિતી (Dietary and Allergen Information)

### આહાર ગુણધર્મો (Dietary Attributes)

આઇટમ્સને આહારની માહિતી સાથે માર્ક કરો:

- **Vegan** — કોઈ પ્રાણી ઉત્પાદનો નહીં
- **Vegetarian** — માંસ નહીં
- **Gluten-Free** — સેલિયાક રોગ માટે સુરક્ષિત
- **Dairy-Free** — દૂધના ઉત્પાદનો નહીં
- **Nut-Free** — કોઈ વૃક્ષના બદામ કે મગફળી નહીં
- **Low-Carb** — ઓછા કાર્બોહાઇડ્રેટ્સ
- **High-Protein** — પ્રોટીનથી ભરપૂર

### એલર્જન ચેતવણીઓ (Allergen Warnings)

સામાન્ય એલર્જનનો સમાવેશ કરો:

- **Nuts** — વૃક્ષના બદામ અને મગફળી
- **Shellfish** — ક્રસ્ટેશિયન્સ અને મોલસ્ક
- **Fish** — તમામ માછલીની પ્રજાતિઓ
- **Dairy** — દૂધ અને દૂધના ઉત્પાદનો
- **Eggs** — મરઘનાના ઈંડા
- **Soy** — સોયા ઉત્પાદનો
- **Gluten** — ઘઉં અને ગ્લુટેન ધરાવતા અનાજ
- **Sesame** — તલના બીજ અને તેલ

## મેનુ મેનેજમેન્ટ (Menu Management)

### કિંમતો અપડેટ કરવી (Updating Prices)

મેનુ કિંમતો સરળતાથી અપડેટ કરો:

૧. મેનુ પેજ પર જાઓ
૨. "Edit Menu" પર ક્લિક કરો
૩. આઇટમ્સ માટે કિંમતો અપડેટ કરો
૪. ફેરફારો સેવ કરો
૫. ફેરફારો તરત જ તમારી સાઇટ પર દેખાશે

### સિઝનલ આઇટમ્સ ઉમેરવી (Adding Seasonal Items)

સિઝનલ મેનુ વિવિધતા બનાવો:

૧. એક નવું મેનુ વર્ઝન બનાવો
૨. સિઝનલ આઇટમ્સ ઉમેરો
૩. આઇટમ્સને "Seasonal" તરીકે માર્ક કરો
૪. ઉપલબ્ધતાની તારીખો શેડ્યૂલ કરો
૫. સિઝન દરમિયાન આપમેળે પ્રદર્શિત કરો

### સ્પેશિયલ્સનું સંચાલન કરવું (Managing Specials)

દૈનિક અથવા સાપ્તાહિક સ્પેશિયલ્સ પ્રદર્શિત કરો:

૧. એક "Specials" કેટેગરી બનાવો
૨. ઉપલબ્ધતાની તારીખો સાથે આઇટમ્સ ઉમેરો
૩. હોમપેજ પર સ્પેશિયલ્સને હાઇલાઇટ કરો
૪. દૈનિક અથવા સાપ્તાહિક અપડેટ કરો
૫. જૂના સ્પેશિયલ્સને આર્કાઇવ કરો

## Theme Builder સાથે સંકલન (Integration with Theme Builder)

જ્યારે હોસ્પિટાલિટી સાઇટ્સ માટે Theme Builder નો ઉપયોગ કરો છો:

૧. **ઓટોમેટિક મેનુ ડિટેક્શન** — ઓળખે છે કે તમે ફૂડ/બેવરેજ વ્યવસાય છો કે નહીં
૨. **મેનુ પેજ ક્રિએશન** — વ્યાવસાયિક મેનુ પેજીસ જનરેટ કરે છે
૩. **ડિઝાઇન મેચિંગ** — મેનુ તમારા સાઇટના થીમ સાથે મેચ થાય છે
૪. **મોબાઇલ ઓપ્ટિમાઇઝેશન** — મેનુ ફોન પર સુંદર રીતે દેખાય છે
૫. **SEO ઓપ્ટિમાઇઝેશન** — મેનુ સર્ચ એન્જિન માટે અનુકૂળ છે

## શ્રેષ્ઠ પદ્ધતિઓ (Best Practices)

### મેનુ ડિઝાઇન (Menu Design)

- **સ્પષ્ટ સંરચના** — તાર્કિક કેટેગરી માળખું
- **વાંચી શકાય તેવા વર્ણનો** — ભૂખ લગાડનારા અને માહિતીપ્રદ
- **સુસંગત કિંમત નિર્ધારણ** — સ્પષ્ટ કરન્સી અને ફોર્મેટિંગ
- **વ્યાવસાયિક ફોટા** — ઉચ્ચ-ગુણવત્તાવાળા ફૂડ ઈમેજીસ
- **Whitespace** — પેજને ભીડવા ન દો

### કન્ટેન્ટ (Content)

- **સચોટ વર્ણનો** — આઇટમ્સનું સચોટ વર્ણન કરો
- **સ્પેશિયલ્સ હાઇલાઇટ કરો** — સ્પેશિયલ્સને અલગ પાડો
- **એલર્જનનો સમાવેશ કરો** — હંમેશા એલર્જનની યાદી બનાવો
- **નિયમિત અપડેટ કરો** — કિંમતો અને આઇટમ્સને વર્તમાન રાખો
- **ભૂખ લગાડતી ભાષાનો ઉપયોગ કરો** — આઇટમ્સને સ્વાદિષ્ટ લાગે તે રીતે બનાવો

### એક્સેસિબિલિટી (Accessibility)

- **વાંચી શકાય તેવા ફોન્ટ્સ** — સ્પષ્ટ, વાંચી શકાય તેવા ટાઇપોગ્રાફીનો ઉપયોગ કરો
- **પૂરતો કોન્ટ્રાસ્ટ** — ખાતરી કરો કે લખાણ વાંચી શકાય છે
- **આહાર લેબલ્સ** — આહારના વિકલ્પોને સ્પષ્ટ રીતે માર્ક કરો
- **એલર્જન ચેતવણીઓ** — એલર્જનને મુખ્ય રીતે પ્રદર્શિત કરો
- **મોબાઇલ-ફ્રેન્ડલી** — તમામ ઉપકરણો પર ટેસ્ટ કરો
