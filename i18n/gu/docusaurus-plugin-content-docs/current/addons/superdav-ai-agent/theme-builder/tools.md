# ટૂલ્સ અને કન્ફિગરેશન

Theme Builder તમારા WordPress સાઇટનું વિશ્લેષણ કરવા, ડિઝાઇન કરવા અને બનાવવામાં મદદ કરવા માટે ટૂલ્સનો સમૂહ વાપરે છે. વર્ઝન 1.16.0 માં, **sd-ai-agent/site-scrape હવે Tier 1 ટૂલ છે**, જે તેને Theme Builder માં ડિફોલ્ટ રૂપે ઉપલબ્ધ બનાવે છે.

## ઉપલબ્ધ ટૂલ્સ

### Tier 1 ટૂલ્સ (હંમેશા ઉપલબ્ધ)

Tier 1 ટૂલ્સ કોઈપણ વધારાના કન્ફિગરેશન વિના Theme Builder માં ડિફોલ્ટ રૂપે ઉપલબ્ધ હોય છે.

#### sd-ai-agent/site-scrape

**હેતુ:** ડિઝાઇનનો પ્રેરણા સ્ત્રોત, કન્ટેન્ટ સ્ટ્રક્ચર અને કાર્યક્ષમતા બહાર કાઢવા માટે હાલની વેબસાઇટ્સનું વિશ્લેષણ કરવું.

**ક્ષમતાઓ:**

- **વેબસાઇટ વિશ્લેષણ** — સ્પર્ધક અથવા પ્રેરણા સ્ત્રોત વેબસાઇટ્સને સ્ક્રૅપ અને વિશ્લેષણ કરવું
- **ડિઝાઇન નિષ્કર્ષણ** — રંગો, ફોન્ટ્સ અને લેઆઉટ પેટર્ન ઓળખવી
- **કન્ટેન્ટ સ્ટ્રક્ચર** — પેજનું આયોજન અને વંશવેલો સમજવો
- **ફીચર ડિટેક્શન** — પ્લગઇન્સ અને કાર્યક્ષમતા ઓળખવી
- **પરફોર્મન્સ વિશ્લેષણ** — પેજની સ્પીડ અને ઓપ્ટિમાઇઝેશન તપાસવું
- **SEO વિશ્લેષણ** — મેટા ટેગ્સ અને સ્ટ્રક્ચર્ડ ડેટાની સમીક્ષા કરવી

**ઉપયોગ:**

```
Analyze the design of example.com and use it as inspiration for my site.
```

**શું બહાર કાઢે છે:**

- કલર પેલેટ અને રંગનો ઉપયોગ
- ટાઇપોગ્રાફી (ફોન્ટ્સ અને કદ)
- લેઆઉટ સ્ટ્રક્ચર અને જગ્યા (spacing)
- નેવિગેશન પેટર્ન
- કન્ટેન્ટનું આયોજન
- ઇમેજ અને મીડિયાનો ઉપયોગ
- ઇન્ટરેક્ટિવ તત્વો
- મોબાઇલ રિસ્પોન્સિવનેસ

**મર્યાદાઓ:**

- પાસવર્ડથી સુરક્ષિત સાઇટ્સને સ્ક્રૅપ કરી શકતું નથી
- robots.txt અને સાઇટ નીતિઓનું પાલન કરે છે
- ડાયનેમિક કન્ટેન્ટ કેપ્ચર ન કરી શકે
- JavaScript-heavy સાઇટ્સમાં વિશ્લેષણ મર્યાદિત હોઈ શકે છે
- મોટી સાઇટ્સનું વિશ્લેષણ કરવામાં વધુ સમય લાગી શકે છે

### Tier 2 ટૂલ્સ (વૈકલ્પિક)

Tier 2 ટૂલ્સ Theme Builder સેટિંગ્સમાં સ્પષ્ટપણે એનેબલ કરવા પર ઉપલબ્ધ હોય છે.

#### Advanced Analytics

વિગતવાર પરફોર્મન્સ મેટ્રિક્સ પ્રદાન કરે છે:

- પેજ લોડ ટાઇમ્સ
- Core Web Vitals
- SEO સ્કોર
- Accessibility સ્કોર
- Best practices સ્કોર

#### Content Optimizer

નીચેના માટે વિશ્લેષણ કરે છે અને સુધારા સૂચવે છે:

- વાંચનક્ષમતા (Readability)
- SEO ઓપ્ટિમાઇઝેશન
- કીવર્ડનો ઉપયોગ
- કન્ટેન્ટ સ્ટ્રક્ચર
- Call-to-action નું સ્થાન

### Tier 3 ટૂલ્સ (પ્રીમિયમ)

Tier 3 ટૂલ્સ માટે વધારાના કન્ફિગરેશન અથવા પ્રીમિયમ ફીચર્સની જરૂર છે.

#### AI Content Generator

નીચેના માટે કન્ટેન્ટ જનરેટ કરે છે:

- પ્રોડક્ટ ડિસ્ક્રિપ્શન્સ
- સર્વિસ પેજીસ
- બ્લોગ પોસ્ટ્સ
- મેટા ડિસ્ક્રિપ્શન્સ
- Call-to-action કોપી

#### Advanced Design Tools

વધુ અદ્યતન ડિઝાઇન ક્ષમતાઓ પ્રદાન કરે છે:

- કસ્ટમ CSS જનરેશન
- એનિમેશન બનાવવું
- ઇન્ટરેક્ટિવ તત્વ ડિઝાઇન
- એડવાન્સ્ડ કલર થિયરી
- ટાઇપોગ્રાફી ઓપ્ટિમાઇઝેશન

## ટૂલ કન્ફિગરેશન

### ટૂલ્સને એનેબલ કરવા

Theme Builder માં વધારાના ટૂલ્સને એનેબલ કરવા માટે:

1. **Gratis AI Agent → Settings** પર જાઓ
2. **Theme Builder → Tools** પર જાઓ
3. જરૂર મુજબ ટૂલ્સને ઓન/ઓફ કરો
4. સેટિંગ્સ સેવ કરો

### ટૂલ પરમિશન

કેટલાક ટૂલ્સને પરમિશનની જરૂર હોય છે:

- **Site scraping** — ઇન્ટરનેટ એક્સેસની જરૂર છે
- **Analytics** — Google Analytics ઇન્ટિગ્રેશનની જરૂર છે
- **Content generation** — API keys ની જરૂર છે
- **Advanced features** — પ્રીમિયમ સબ્સ્ક્રિપ્શનની જરૂર પડી શકે છે

### API Keys અને Credentials

જે ટૂલ્સને તેની જરૂર હોય તેમના માટે API keys કન્ફિગર કરો:

1. **Gratis AI Agent → Settings → API Keys** પર જાઓ
2. દરેક ટૂલ માટે credentials દાખલ કરો
3. કનેક્શનનું પરીક્ષણ કરો
4. સુરક્ષિત રીતે સેવ કરો

## sd-ai-agent/site-scrape નો ઉપયોગ કરવો

### મૂળભૂત ઉપયોગ

Theme Builder ને વેબસાઇટનું વિશ્લેષણ કરવા કહો:

```
Analyze the design of my competitor's site at competitor.com
and suggest design improvements for my site.
```

### ચોક્કસ વિશ્લેષણ

વિશ્લેષણના ચોક્કસ પ્રકારોની વિનંતી કરો:

```
Extract the color palette from example.com and use it as inspiration.
```

```
Analyze the navigation structure of example.com and apply similar
organization to my site.
```

```
Check the mobile responsiveness of example.com and ensure my site
is equally responsive.
```

### સરખામણી વિશ્લેષણ

બહુવિધ સાઇટ્સની સરખામણી કરો:

```
Compare the designs of site1.com and site2.com and create a design
that combines the best elements of both.
```

## ટૂલ મર્યાદાઓ અને વિચારણાઓ

### Rate Limiting

- સર્વર ઓવરલોડ અટકાવવા માટે સ્ક્રૅપિંગ rate-limited છે
- એક જ સાઇટ પર બહુવિધ વિનંતીઓ થ્રોટલ થઈ શકે છે
- મોટી સાઇટ્સનું વિશ્લેષણ કરવામાં વધુ સમય લાગી શકે છે

### કન્ટેન્ટની ચોકસાઈ

- ડાયનેમિક કન્ટેન્ટ સંપૂર્ણપણે કેપ્ચર ન થઈ શકે
- JavaScript-rendered કન્ટેન્ટ અધૂરું હોઈ શકે છે
- રિયલ-ટાઇમ ડેટા પ્રતિબિંબિત ન થઈ શકે
- કેટલાક કન્ટેન્ટ પે-વૉલ્સ પાછળ હોઈ શકે છે

### કાનૂની અને નૈતિક વિચારણાઓ

- robots.txt અને સાઇટ નીતિઓનું પાલન કરો
- પુનરાવર્તન માટે કોપીરાઇટ કરેલ કન્ટેન્ટ સ્ક્રૅપ ન કરો
- પ્રેરણા માટે વિશ્લેષણનો ઉપયોગ કરો, ન કે નકલ કરવા માટે
- ખાતરી કરો કે તમારી પાસે કાઢેલા કન્ટેન્ટનો ઉપયોગ કરવાનો અધિકાર છે
- વિશ્લેષિત સાઇટ્સની સેવા શરતોનું પાલન કરો

### પરફોર્મન્સ અસર

- મોટી સાઇટ્સને સ્ક્રૅપ કરવામાં સમય લાગી શકે છે
- બહુવિધ એકસાથે સ્ક્રૅપ કરવાથી ધીમું થઈ શકે છે
- નેટવર્ક કનેક્ટિવિટી ઝડપને અસર કરે છે
- મોટા ડેટાસેટ્સને વધુ પ્રોસેસિંગની જરૂર પડી શકે છે

## શ્રેષ્ઠ પદ્ધતિઓ (Best Practices)

### સાઇટ વિશ્લેષણનો ઉપયોગ કરવો

1. **બહુવિધ સાઇટ્સનું વિશ્લેષણ કરો** — ઘણા સ્ત્રોતોમાંથી પ્રેરણા એકત્રિત કરો
2. **સ્ટ્રક્ચર પર ધ્યાન કેન્દ્રિત કરો** — લેઆઉટ અને આયોજન પેટર્ન શીખો
3. **રંગો બહાર કાઢો** — કલર પેલેટ્સનો ઉપયોગ શરૂઆતનું બિંદુ તરીકે કરો
4. **ટાઇપોગ્રાફીનો અભ્યાસ કરો** — તમને ગમતા ફોન્ટના સંયોજનો ઓળખો
5. **નેવિગેશન સમીક્ષા કરો** — મેનુના આયોજનને સમજો

### નૈતિક સ્ક્રૅપિંગ

1. **robots.txt નું પાલન કરો** — સાઇટ માર્ગદર્શિકાઓનું પાલન કરો
2. **કન્ટેન્ટની નકલ ન કરો** — માત્ર પ્રેરણા માટે વિશ્લેષણનો ઉપયોગ કરો
3. **અધિકારની ચકાસણી કરો** — ખાતરી કરો કે તમે કાઢેલા કન્ટેન્ટનો ઉપયોગ કરી શકો છો
4. **સ્રોતોને એટ્રિબ્યુટ કરો** — પ્રેરણાના સ્ત્રોતોને ક્રેડિટ આપો
5. **શરતોનું પાલન કરો** — સાઇટની સેવા શરતોનું પાલન કરો

### ટૂલની અસરકારકતાને મહત્તમ કરવી

1. **ચોક્કસ રહો** — વિશ્લેષણના ચોક્કસ પ્રકારોની વિનંતી કરો
2. **સંદર્ભ પ્રદાન કરો** — તમારી સાઇટનો હેતુ સમજાવો
3. **અપેક્ષાઓ સેટ કરો** — તમારા ડિઝાઇન લક્ષ્યોનું વર્ણન કરો
4. **પુનરાવર્તન કરો** — પરિણામોના આધારે સુધારો કરો
5. **ટૂલ્સને સંયોજિત કરો** — વ્યાપક વિશ્લેષણ માટે બહુવિધ ટૂલ્સનો ઉપયોગ કરો

## ટ્રબલશૂટિંગ

### સાઇટ સ્ક્રૅપ ન થાય

- તપાસો કે સાઇટ જાહેર રીતે ઉપલબ્ધ છે કે નહીં
- ચકાસો કે robots.txt સ્ક્રૅપિંગની મંજૂરી આપે છે કે નહીં
- અન્ય સાઇટનો પ્રયાસ કરો
- ઇન્ટરનેટ કનેક્શન તપાસો
- જો સમસ્યા ચાલુ રહે તો સપોર્ટનો સંપર્ક કરો

### અધૂરું વિશ્લેષણ

- સાઇટમાં ડાયનેમિક કન્ટેન્ટ હોઈ શકે છે
- JavaScript સંપૂર્ણપણે રેન્ડર ન થઈ શકે
- મોટી સાઇટ્સ ટાઇમઆઉટ થઈ શકે છે
- તેના બદલે ચોક્કસ પેજીસનું વિશ્લેષણ કરવાનો પ્રયાસ કરો
- ચોક્કસ વિશ્લેષણ પ્રકારોની વિનંતી કરો

### ધીમું પરફોર્મન્સ

- મોટી સાઇટ્સનું વિશ્લેષણ કરવામાં વધુ સમય લાગે છે
- બહુવિધ એકસાથે સ્ક્રૅપ કરવાથી ધીમું પડે છે
- નેટવર્ક કનેક્ટિવિટી ઝડપને અસર કરે છે
- પીક અવર્સ દરમિયાન ટાળવાનો પ્રયાસ કરો
