# Domain Seller ફેરફારની નોંધ

Version 1.1.0 - 2026-05-08 ના રોજ રજૂ કરવામાં આવ્યું

* **સુધારેલ:** સેટિંગ્સ પેજમાંથી ગ્લોબલ માર્કઅપ સેટિંગ્સ દૂર કરવામાં આવ્યા — કિંમત હવે ફક્ત પ્રતિ-ઉત્પાદન (per-product) છે
* **સુધારેલ:** ઝડપી નેવિગેશન માટે સેટિંગ્સ પેજ પર "Manage Domain Products" લિંક ઉમેરવામાં આવી
* **સુધારેલ:** ડોમેન પ્રોડક્ટ સેટિંગ્સ માટે વધુ સ્પષ્ટ ફીલ્ડ વર્ણન અને ટૂલટિપ્સ (catch-all vs TLD-specific, માર્કઅપ પ્રકારો, પ્રારંભિક કિંમત)
* **સુધારેલ:** સેટિંગ્સ પેજમાં વધુ સારા વર્ણનો (TLDs શોધવા, રિન્યુઅલ, DNS, નોટિફિકેશન)

Version 1.0.1 - 2026-02-27 ના રોજ રજૂ કરવામાં આવ્યું

* **નવું:** બલ્ક કિંમત વ્યવસ્થાપન માટે TLD ઇમ્પોર્ટ ટૂલ
* **નવું:** ડોમેન પ્રોડક્ટ્સ માટે પ્રારંભિક કિંમત સપોર્ટ
* **નવું:** Cypress સાથે E2E ટેસ્ટ સ્યુટ
* **નવું:** ડોમેન જીવનચક્ર નોટિફિકેશન માટે ઇમેઇલ ટેમ્પલેટ્સ
* **નવું:** એડમિન ડોમેન રજીસ્ટ્રેશન મોડલ પર રજિસ્ટ્રન્ટ એડ્રેસ ફીલ્ડ, સેટિંગ્સમાંથી પ્રી-પૉપ્યુલેટેડ
* **નવું:** એડ, એડિટ અને ડિલીટ રેકોર્ડ સપોર્ટ સાથે ગ્રાહક DNS મેનેજમેન્ટ ઇન્ટરફેસ
* **નવું:** ઓટોમેટિક ડોમેન મેપિંગ સાથે "Bring your own domain" ચેકઆઉટ વિકલ્પ
* **નવું:** ચેકઆઉટ દરમિયાન ડોમેન નામમાંથી ઓટો-જનરેટ સાઇટ URL
* **નવું:** સેટિંગ્સમાં ડિફોલ્ટ નેમસર્વર અને DNS રેકોર્ડ કન્ફિગરેશન
* **નવું:** કોર ડોમેન એડિટ પેજ પર ડોમેન રજીસ્ટ્રેશન વિગતો અને DNS મેનેજમેન્ટ
* **નવું:** સેટઅપ વિઝાર્ડ સેન્સિબલ ડિફોલ્ટ સાથે ડિફોલ્ટ ડોમેન પ્રોડક્ટ આપોઆપ બનાવે છે
* **નવું:** બધા કન્ફિગર કરેલ પ્રદાતાઓ પર cron દ્વારા દૈનિક ઓટોમેટિક TLD sync
* **નવું:** પ્રતિ-ઉત્પાદન કન્ફિગરેશન સાથે WHOIS પ્રાઇવસી પ્રોટેક્શન (હંમેશા ચાલુ, ગ્રાહક પસંદગી, અથવા નિષ્ક્રિય)
* **નવું:** કિંમત પ્રદર્શન અને ડાર્ક મોડ સપોર્ટ સાથે WHOIS પ્રાઇવસી ચેકઆઉટ ચેકબોક્સ
* **નવું:** મેન્યુઅલ ડોમેન રજીસ્ટ્રેશન માટે Register Domain એડમિન પેજ
* **નવું:** Ultimate Multisite update server દ્વારા ઓટોમેટિક પ્લગઇન અપડેટ્સ
* **નવું:** પ્રોડક્ટ લિસ્ટ ટેબલમાં ડોમેન પ્રોડક્ટ પ્રકાર ફિલ્ટર ટેબ પર્પલ બેજ સ્ટાઇલિંગ સાથે
* **નવું:** ડોમેન ચેકઆઉટ ફોર્મ પર રજિસ્ટ્રન્ટ કોન્ટેક્ટ ફીલ્ડ્સ (નામ, સરનામું, શહેર, રાજ્ય, પિન કોડ, દેશ, ફોન)
* **નવું:** રજિસ્ટ્રાર API ને કૉલ કરતા પહેલા સ્પષ્ટ ભૂલ સંદેશાઓ સાથે રજિસ્ટ્રન્ટ ફીલ્ડ વેલિડેશન
* **નવું:** ડોમેન રજીસ્ટ્રેશન ઇવેન્ટ્સ માટે પ્રદાતા-વિશિષ્ટ લોગ ચેનલો (દા.ત. domain-seller-namecheap.log)
* **નવું:** મુખ્ય રજીસ્ટ્રેશન/સાઇનઅપ ચેકઆઉટ ફોર્મ પર રજિસ્ટ્રન્ટ કોન્ટેક્ટ ફીલ્ડ્સ (ડોમેન રજીસ્ટર કરતી વખતે બતાવવામાં આવે છે)
* **સુધારેલ:** ડોમેન સર્ચ ચેકઆઉટ ફીલ્ડને યુનિફાઇડ ડોમેન સિલેક્શન ફીલ્ડથી બદલવામાં આવ્યું છે જે સબડોમેન, રજીસ્ટર અને હાલના ડોમેન ટેબને સપોર્ટ કરે છે
* **સુધારેલ:** ડોમેન પ્રોડક્ટ સેટિંગ્સ કોર વિજેટ સિસ્ટમ દ્વારા પ્રોડક્ટ એડિટ પેજ પર ઇનલાઇન રેન્ડર થાય છે
* **સુધારેલ:** ગ્રાહક ડોમેન માહિતી કોર ડોમેન મેપિંગ વિજેટમાં સંકલિત થાય છે, ન કે અલગ મેટાબોક્સમાં
* **સુધારેલ:** TLD ઇમ્પોર્ટ વિઝાર્ડ બધા પ્રદાતાઓમાંથી એક-ક્લિક sync માં સરળ બનાવવામાં આવ્યું છે
* **સુધારેલ:** Namecheap ડોમેન ઉપલબ્ધતા માટે ઝડપી શોધ માટે બેચ API કૉલનો ઉપયોગ કરે છે
* **સુધારેલ:** Namecheap કિંમત API યોગ્ય પેરામીટર્સ અને પ્રતિસાદ પાર્સિંગનો ઉપયોગ કરે છે
* **સુધારેલ:** એક જ નેટવર્ક વિકલ્પમાં કેન્દ્રિત TLD સ્ટોરેજ
* **સુધારેલ:** DNS ફેરફારો, ટ્રાન્સફર અને કન્ફિગરેશન એપ્લિકેશન માટે ડોમેન એક્ટિવિટી લોગિંગ
* **સુધારેલ:** બેચ વેલિડેશન સાથે IANA માસ્ટર લિસ્ટનો ઉપયોગ કરીને OpenSRS માટે સંપૂર્ણ TLD sync
* **સુધારેલ:** પેજિનેટેડ API વિનંતીઓ સાથે Namecheap માટે સંપૂર્ણ TLD sync
* **સુધારેલ:** લેગસી પ્રદાતા વર્ગોને Integration Registry pattern સાથે બદલવામાં આવ્યા
* **સુધારેલ:** DNS અને ટ્રાન્સફર કન્ફિગરેશન સાથે સેટિંગ્સ પેનલ
* **સુધારેલ:** ફોન નંબરો આપમેળે +CC.NNN રજિસ્ટ્રાર ફોર્મેટમાં ફોર્મેટ થાય છે
* **સુધારેલ:** ફોન ફીલ્ડ વેલિડેશન સબમિટ કરતા પહેલા ફોર્મેટિંગ અક્ષરો દૂર કરે છે
