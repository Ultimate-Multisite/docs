# Domain Seller

**Multisite Ultimate: Domain Seller** તમારા WordPress Multisite નેટવર્કને એક શક્તિશાળી ડોમેન વેચાણ પ્લેટફોર્મમાં પરિવર્તિત કરે છે. આનાથી ગ્રાહકો સાઇન-અપ દરમિયાન અથવા તેમના ડેશબોર્ડમાંથી સીધા કસ્ટમ ડોમેન રજીસ્ટર કરી શકે છે, જેમાં રીઅલ-ટાઇમ ઉપલબ્ધતા તપાસ અને પારદર્શક કિંમત નિર્ધારણની સુવિધા હોય છે.

### Domain Seller શા માટે?

કોઈપણ WaaS (Website as a Service) પ્લેટફોર્મ માટે ડોમેન રજીસ્ટ્રેશન એ સૌથી વધુ માંગવાળી વધારાની સેવાઓ (upsells) માંથી એક છે. તમારા ગ્રાહકોને ત્રીજા પક્ષના રજિસ્ટ્રાર પાસે મોકલવાને બદલે, તમે સીધા ડોમેન ઓફર કરી શકો છો — જેનાથી તમારા નેટવર્કમાં વધુ આવક જળવાશે અને ગ્રાહકોને એક સરળ અનુભવ મળશે.

### મુખ્ય વિશેષતાઓ (Key Features)

#### બહુવિધ ડોમેન રજિસ્ટ્રાર સપોર્ટ (Multiple Domain Registrar Support)
* **OpenSRS** — સંપૂર્ણ રજીસ્ટ્રેશન, રિન્યુઅલ અને મેનેજમેન્ટ
* **Namecheap** — WhoisGuard અને સેન્ડબોક્સ ટેસ્ટિંગ સાથે સંપૂર્ણ એકીકરણ (Complete integration)
* **વિસ્તૃત આર્કિટેક્ચર (Extensible Architecture)** — ઇન્ટિગ્રેશન રજિસ્ટ્રી પેટર્ન પર બનેલું, જે વધુ પ્રદાતાઓ (providers) ઉમેરવાનું સરળ બનાવે છે
* **બિલ્ટ-ઇન કનેક્શન ટેસ્ટિંગ (Built-in Connection Testing)** — લાઇવ થતા પહેલા પ્રદાતાની ઓળખ (credentials) અને કનેક્ટિવિટીની માન્યતા (validate)

#### માર્ગદર્શિત સેટઅપ વિઝાર્ડ (Guided Setup Wizard)
* **વન-ક્લિક ઓનબોર્ડિંગ (One-click onboarding)** — પ્રદાતા કન્ફિગરેશન, TLD ઇમ્પોર્ટ અને ડોમેન પ્રોડક્ટ ક્રિએશન દ્વારા માર્ગદર્શન આપે છે
* **ઓટોમેટિક TLD sync** — તમારા કનેક્ટેડ પ્રદાતાઓ પાસેથી ઉપલબ્ધ તમામ TLDs અને હોલસેલ કિંમતો (wholesale pricing) ઇમ્પોર્ટ કરે છે
* **ડિફોલ્ટ પ્રોડક્ટ ક્રિએશન** — સમજદાર માર્કઅપ ડિફોલ્ટ્સ સાથે ઉપયોગ માટે તૈયાર ડોમેન પ્રોડક્ટ બનાવે છે

#### લવચીક ડોમેન કિંમત નિર્ધારણ (Flexible Domain Pricing)
* **ત્રણ માર્કઅપ મોડ્સ (Three markup modes)** — ટકાવારી માર્કઅપ, નિશ્ચિત માર્કઅપ, અથવા TLD દીઠ નિશ્ચિત કિંમત
* **TLD-વિશિષ્ટ પ્રોડક્ટ્સ (TLD-specific products)** — વિવિધ TLD ગ્રુપ્સ (.com/.net, .io, country codes, વગેરે) માટે અલગ ડોમેન પ્રોડક્ટ બનાવો
* **પ્રારંભિક કિંમત (Introductory pricing)** — નવા રજીસ્ટ્રેશનોને આકર્ષવા માટે પ્રથમ વર્ષની ડિસ્કાઉન્ટેડ કિંમત ઓફર કરો
* **રીઅલ-ટાઇમ કિંમત (Real-time pricing)** — ઇન્ટેલિજન્ટ કેશિંગ સાથે પ્રદાતાઓ તરફથી લાઇવ હોલસેલ કિંમતો
* **કૂપન સપોર્ટ (Coupon support)** — Multisite Ultimate ડિસ્કાઉન્ટ કોડ સાથે સંપૂર્ણ સુસંગતતા

#### WHOIS પ્રાઇવસી પ્રોટેક્શન (WHOIS Privacy Protection)
* **પ્રોડક્ટ દીઠ કન્ફિગરેશન (Per-product configuration)** — WHOIS પ્રાઇવસીને Disabled, Always Included, અથવા Customer Choice પર સેટ કરો
* **ગ્રાહક ઓપ્ટ-ઇન (Customer opt-in)** — જ્યારે Customer Choice પર સેટ કરવામાં આવે છે, ત્યારે ચેકઆઉટ દરમિયાન ગ્રાહકોને પ્રાઇવસી પ્રોટેક્શન ઉમેરવાની ચેકબોક્સ મળે છે
* **કન્ફિગરેબલ કિંમત (Configurable pricing)** — પ્રાઇવસી એડ-ઓન માટે પ્રતિ વર્ષ કિંમત સેટ કરો, અથવા તે મફત ઓફર કરો
* **રજિસ્ટ્રાર એકીકરણ (Registrar integration)** — રજીસ્ટ્રેશન સમયે આપમેળે WhoisGuard (Namecheap) અથવા WHOIS પ્રાઇવસી (OpenSRS) સક્ષમ કરે છે

#### એકીકૃત ચેકઆઉટ અનુભવ (Unified Checkout Experience)
* **ત્રણ ડોમેન મોડ્સ (Three domain modes)** — ગ્રાહકો ઉપડોમેન (free), નવો ડોમેન રજીસ્ટર કરવા, અથવા તેઓ જે ડોમેન પહેલેથી માલિક છે તેને મેપ કરવા વચ્ચે પસંદગી કરી શકે છે
* **રીઅલ-ટાઇમ ડોમેન સર્ચ (Real-time domain search)** — પેજ રીલોડ વિના AJAX-સંચાલિત ઉપલબ્ધતા તપાસ
* **સ્માર્ટ સૂચનો (Smart suggestions)** — જ્યારે પસંદ કરેલો ડોમેન ઉપલબ્ધ ન હોય ત્યારે વૈકલ્પિક TLD સૂચનો
* **પારદર્શક કિંમત (Transparent pricing)** — રજીસ્ટ્રેશન કિંમત, રિન્યુઅલ કિંમત અને વૈકલ્પિક WHOIS પ્રાઇવસી ફીનું સ્પષ્ટ પ્રદર્શન
* **રજિસ્ટ્રન્ટ સંપર્ક ફીલ્ડ્સ (Registrant contact fields)** — ચેકઆઉટ દરમિયાન જરૂરી રજિસ્ટ્રન્ટ માહિતી (નામ, સરનામું, ફોન) ઇનલાઇન એકત્રિત કરે છે
* **ઓટો-જનરેટેડ સાઇટ URL (Auto-generated site URL)** — સાઇટનું સરનામું પસંદ કરેલા ડોમેનમાંથી આપમેળે મેળવવામાં આવે છે

#### ઓટો-રિન્યુઅલ સિસ્ટમ (Auto-Renewal System)
* **સભ્યપદ-સંલગ્ન રિન્યુઅલ (Membership-tied renewals)** — ગ્રાહકનું સભ્યપદ સક્રિય હોય ત્યાં સુધી ડોમેન આપમેળે રિન્યુ થાય છે
* **સ્માર્ટ શેડ્યુલિંગ (Smart scheduling)** — ડોમેન સમાપ્તિ તારીખોના આધારે રિન્યુઅલ જોબ્સ કતારમાં મૂકવામાં આવે છે
* **રીટ્રાય લોજિક (Retry logic)** — નિષ્ફળ રિન્યુઅલ પ્રયાસો માટે ઇન્ટેલિજન્ટ રીટ્રાય્સ
* **ઈમેલ નોટિફિકેશન્સ (Email notifications)** — આવનારા રિન્યુઅલ, સફળ રિન્યુઅલ, નિષ્ફળતા અને સમાપ્તિ માટે કન્ફિગરેબલ એલર્ટ્સ

#### ગ્રાહક DNS મેનેજમેન્ટ (Customer DNS Management)
* **સેલ્ફ-સર્વિસ DNS (Self-service DNS)** — ગ્રાહકો તેમના ડેશબોર્ડમાંથી DNS રેકોર્ડ્સ (A, AAAA, CNAME, MX, TXT) ઉમેરી, સંપાદિત અને કાઢી શકે છે
* **ડોમેન સ્ટેટસ (Domain status)** — રીઅલ-ટાઇમ રજીસ્ટ્રેશન સ્ટેટસ, સમાપ્તિ તારીખો અને નેમસર્વર માહિતી
* **ઓટોમેટિક ડોમેન મેપિંગ (Automatic domain mapping)** — રજીસ્ટર કરેલા અથવા હાલના ડોમેન આપમેળે ગ્રાહકના સાઇટ સાથે મેપ થઈ જાય છે

#### એડમિન ટૂલ્સ (Admin Tools)
* **મેન્યુઅલ ડોમેન રજીસ્ટ્રેશન (Manual domain registration)** — એડમિન પેનલમાંથી ગ્રાહકો વતી ડોમેન રજીસ્ટર કરો
* **ડોમેન મોનિટરિંગ (Domain monitoring)** — એક જ પેજમાંથી તમામ ગ્રાહક ડોમેન, સ્ટેટસ અને સમાપ્તિ તારીખો જુઓ
* **પ્રદાતા-વિશિષ્ટ લોગિંગ (Provider-specific logging)** — દરેક રજિસ્ટ્રાર માટે અલગ લોગ ચેનલો (દા.ત., domain-seller-namecheap.log)
* **રજીસ્ટ્રેશન વિગતો (Registration details)** — કોર ડોમેન એડિટ પેજ પર રજિસ્ટ્રન્ટની માહિતી અને રજીસ્ટ્રેશન ડેટા જુઓ

### સપોર્ટેડ પ્રદાતાઓ (Supported Providers)

* **OpenSRS** — રજીસ્ટ્રેશન, રિન્યુઅલ, DNS મેનેજમેન્ટ અને WHOIS પ્રાઇવસી સાથે સંપૂર્ણ એકીકરણ
* **Namecheap** — રજીસ્ટ્રેશન, રિન્યુઅલ, બેચ ઉપલબ્ધતા તપાસ અને મફત WhoisGuard સાથે સંપૂર્ણ એકીકરણ
* **GoDaddy** — રજીસ્ટ્રેશન અને રિન્યુઅલ
* **ResellerClub** — રજીસ્ટ્રેશન, રિન્યુઅલ અને DNS મેનેજમેન્ટ
* **NameSilo** — રજીસ્ટ્રેશન અને રિન્યુઅલ
* **Enom** — રજીસ્ટ્રેશન, રિન્યુઅલ અને DNS મેનેજમેન્ટ

### તકનીકી વિગતો (Technical Details)

* **જરૂરી છે (Requires):** PHP 7.4+, Multisite Ultimate 2.4.12+
* **આર્કિટેક્ચર (Architecture):** Integration Registry pattern with Domain_Selling_Capability interface
* **ઓટોલોડિંગ (Autoloading):** Jetpack autoloader with PSR-4 class mapping
* **સ્ટાન્ડર્ડ્સ (Standards):** WordPress coding standards, strict typing, prepared statements

## ઇન્સ્ટોલેશન (Installation)

1. ખાતરી કરો કે **Multisite Ultimate v2.4.12+** ઇન્સ્ટોલ અને નેટવર્ક-એક્ટિવેટેડ છે
2. પ્લગઇનને `/wp-content/plugins/ultimate-multisite-domain-seller/` પર અપલોડ કરો
3. WordPress Network Admin માં **Plugins** માંથી પ્લગઇનને નેટવર્ક-એક્ટિવેટ કરો
4. પ્રથમ એક્ટિવેશન પર **Setup Wizard** આપમેળે શરૂ થાય છે — નીચેના પગલાં અનુસરો:
   * તમારા ડોમેન રજિસ્ટ્રારને કનેક્ટ કરો (OpenSRS અથવા Namecheap)
   * કનેક્શનનું પરીક્ષણ કરો
   * ઉપલબ્ધ TLDs અને હોલસેલ કિંમતો ઇમ્પોર્ટ કરો
   * કિંમત માર્કઅપ સાથે તમારી પ્રથમ ડોમેન પ્રોડક્ટ બનાવો
5. **Ultimate Multisite > Checkout Forms** માં તમારા ચેકઆઉટ ફોર્મમાં **Domain Selection** ફીલ્ડ ઉમેરો

## વારંવાર પૂછાતા પ્રશ્નો (Frequently Asked Questions)

### Multisite Ultimate નું કયું વર્ઝન જરૂરી છે?

વર્ઝન 2.4.12 અથવા તેથી વધુ. આ એડ-ઓન WP Ultimo v1.x અથવા 2.4.12 કરતા ઓછા Ultimate Multisite વર્ઝન સાથે સુસંગત નથી.

### કયા ડોમેન પ્રદાતાઓ સપોર્ટેડ છે?

છ રજિસ્ટ્રાર સંપૂર્ણપણે સપોર્ટેડ છે: **OpenSRS**, **Namecheap**, **GoDaddy**, **ResellerClub**, **NameSilo**, અને **Enom**. આ એડ-ઓનનું ઇન્ટિગ્રેશન રજિસ્ટ્રી આર્કિટેક્ચર કસ્ટમ પ્રદાતાઓ ઉમેરવાનું સરળ બનાવે છે — [developer guide](/developer/integration-guide/custom-registrar) જુઓ.

### શું હું WHOIS પ્રાઇવસી પ્રોટેક્શન ઓફર કરી શકું છું?

હા. દરેક ડોમેન પ્રોડક્ટમાં WHOIS પ્રાઇવસી સેટિંગ હોય છે જેમાં ત્રણ મોડ્સ હોય છે: Disabled, Always Included, અથવા Customer Choice. જ્યારે Customer Choice પર સેટ કરવામાં આવે છે, ત્યારે ગ્રાહકોને ચેકઆઉટ દરમિયાન કિંમત સાથે (અથવા Namecheap ના WhoisGuard માટે "Free") ચેકબોક્સ દેખાય છે. પ્રાઇવસી ફ્લેગ રજીસ્ટ્રેશન સમયે આપમેળે રજિસ્ટ્રારને પાસ થઈ જાય છે.

### શું હું વિવિધ TLDs માટે અલગ કિંમત સેટ કરી શકું છું?

હા. વિવિધ TLD ગ્રુપ્સ માટે અલગ ડોમેન પ્રોડક્ટ બનાવો. દરેક પ્રોડક્ટનો પોતાનો માર્કઅપ પ્રકાર (ટકાવારી, નિશ્ચિત માર્કઅપ, અથવા નિશ્ચિત કિંમત), સપોર્ટેડ TLDs અને પ્રારંભિક કિંમત હોઈ શકે છે.

### ગ્રાહકોને કયા ચેકઆઉટ વિકલ્પો દેખાય છે?

ડોમેન સિલેક્શન ફીલ્ડ ગ્રાહકોને ત્રણ ટેબ આપે છે: **Subdomain** (તમારા નેટવર્ક પર મફત ઉપડોમેનનો ઉપયોગ કરો), **Register New Domain** (નવો ડોમેન સર્ચ અને રજીસ્ટર કરો), અને **Existing Domain** (તેઓ જે ડોમેન પહેલેથી માલિક છે તેને મેપ કરો). ફક્ત તમે જે મોડ્સ સક્ષમ કરો તે જ દેખાશે.

### શું તે ઓટોમેટિક રિન્યુઅલને સપોર્ટ કરે છે?

હા. ડોમેન ઓટો-રિન્યુ થાય તે રીતે સેટ કરી શકાય છે અને તે ગ્રાહકના સભ્યપદ સ્ટેટસ સાથે જોડાયેલા હોય છે. રિન્યુઅલ પેમેન્ટ ગ્રાહકના હાલના પેમેન્ટ ગેટવે દ્વારા પ્રોસેસ થાય છે.

### શું ગ્રાહકો તેમના DNS રેકોર્ડ્સ મેનેજ કરી શકે છે?

હા. ગ્રાહકોને તેમના ડેશબોર્ડમાં A, AAAA, CNAME, MX, અને TXT રેકોર્ડ્સ ઉમેરવા, સંપાદિત કરવા અને કાઢી નાખવા માટે DNS મેનેજમેન્ટ ઇન્ટરફેસ મળે છે. DNS મેનેજમેન્ટ OpenSRS, ResellerClub, અને Enom સાથે ઉપલબ્ધ છે.

### શું ત્યાં કોઈ સેટઅપ વિઝાર્ડ છે?

હા. સેટઅપ વિઝાર્ડ પ્રથમ એક્ટિવેશન પર આપમેળે ચાલે છે અને તમને રજિસ્ટ્રાર કનેક્ટ કરવા, TLDs ઇમ્પોર્ટ કરવા અને તમારી પ્રથમ ડોમેન પ્રોડક્ટ બનાવવામાં માર્ગદર્શન આપે છે. તમે એડ-ઓન સેટિંગ્સમાંથી વિઝાર્ડને ફરીથી ચલાવી પણ શકો છો.
