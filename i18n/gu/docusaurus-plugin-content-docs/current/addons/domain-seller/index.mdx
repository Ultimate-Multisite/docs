# Domain Seller

**Multisite Ultimate: Domain Seller** તમારા WordPress Multisite નેટવર્કને શક્તિશાળી ડોમેન વેચાણ પ્લેટફોર્મમાં રૂપાંતરિત કરે છે. ગ્રાહકોને સાઇન-અપ દરમિયાન અથવા તેમના Dashboardમાંથી સીધા કસ્ટમ ડોમેન નોંધાવવા દો, રિયલ-ટાઇમ ઉપલબ્ધતા તપાસ અને પારદર્શક કિંમત સાથે.

### Domain Seller શા માટે?

ડોમેન નોંધણી કોઈપણ WaaS (સેવા તરીકે વેબસાઇટ) પ્લેટફોર્મ માટે સૌથી વધુ માંગવામાં આવતી વધારાની વેચાણ ઓફરોમાંની એક છે. તમારા ગ્રાહકોને તૃતીય-પક્ષ રજિસ્ટ્રાર પાસે મોકલવાને બદલે, તમે સીધા ડોમેન્સ ઓફર કરી શકો છો — ગ્રાહકોને અવરોધરહિત અનુભવ આપતા આપતા તમારા નેટવર્કમાં વધુ આવક રાખી શકો છો.

### મુખ્ય સુવિધાઓ

#### બહુવિધ ડોમેન રજિસ્ટ્રાર સપોર્ટ
* **OpenSRS** — સંપૂર્ણ નોંધણી, નવીકરણ અને વ્યવસ્થાપન
* **Namecheap** — WhoisGuard અને sandbox પરીક્ષણ સાથે સંપૂર્ણ એકીકરણ
* **HostAfrica** — Domains Reseller API દ્વારા ડોમેન શોધ, કિંમત, નોંધણી, ટ્રાન્સફર, DNS અને ID સુરક્ષા
* **Openprovider** — રિસેલર કિંમત, નોંધણી, નવીકરણ, ટ્રાન્સફર, DNS, WHOIS ગોપનીયતા અને TLD sync
* **Hostinger** — શેર કરેલા Hostinger એકીકરણ tokenનો ઉપયોગ કરીને ડોમેન શોધ અને નોંધણી
* **વિસ્તૃત કરી શકાય તેવી આર્કિટેક્ચર** — Integration Registry pattern પર બનેલું, જેથી વધુ પ્રદાતાઓ ઉમેરવાનું સરળ બને છે
* **બિલ્ટ-ઇન કનેક્શન પરીક્ષણ** — live કરતા પહેલાં પ્રદાતા credentials અને connectivity માન્ય કરો

#### માર્ગદર્શિત Setup વિઝાર્ડ
* **એક-ક્લિક onboarding** — પ્રદાતા રૂપરેખાંકન, TLD import અને ડોમેન product નિર્માણમાં તમને માર્ગદર્શન આપે છે
* **સ્વચાલિત TLD sync** — તમારા જોડાયેલા પ્રદાતાઓમાંથી બધા ઉપલબ્ધ TLDs અને wholesale pricing import કરે છે
* **ડિફોલ્ટ product નિર્માણ** — સમજદાર markup defaults સાથે વાપરવા તૈયાર ડોમેન product બનાવે છે

#### લવચીક ડોમેન કિંમત
* **ત્રણ markup modes** — ટકાવારી markup, નિશ્ચિત markup અથવા દરેક TLD માટે નિશ્ચિત કિંમત
* **TLD-વિશિષ્ટ products** — અલગ-અલગ TLD જૂથો (.com/.net, .io, દેશ કોડ્સ, વગેરે) માટે અલગ ડોમેન products બનાવો
* **પ્રારંભિક કિંમત** — નવી નોંધણીઓને આકર્ષવા માટે ડિસ્કાઉન્ટેડ પ્રથમ વર્ષની કિંમત ઓફર કરો
* **રિયલ-ટાઇમ કિંમત** — બુદ્ધિશાળી caching સાથે પ્રદાતાઓ પાસેથી live wholesale pricing
* **Coupon સપોર્ટ** — Multisite Ultimate discount codes સાથે સંપૂર્ણ સુસંગતતા

#### WHOIS ગોપનીયતા સુરક્ષા
* **દરેક product દીઠ રૂપરેખાંકન** — WHOIS ગોપનીયતાને Disabled, Always Included અથવા Customer Choice પર સેટ કરો
* **ગ્રાહક opt-in** — Customer Choice પર સેટ હોય ત્યારે, checkout checkbox ગ્રાહકોને ગોપનીયતા સુરક્ષા ઉમેરવા દે છે
* **રૂપરેખાંકિત કરી શકાય તેવી કિંમત** — privacy add-on માટે દર વર્ષની કિંમત સેટ કરો, અથવા તેને મફત ઓફર કરો
* **રજિસ્ટ્રાર એકીકરણ** — નોંધણી સમયે WhoisGuard (Namecheap) અથવા WHOIS ગોપનીયતા (OpenSRS) આપમેળે સક્ષમ કરે છે

#### એકીકૃત Checkout અનુભવ
* **ત્રણ ડોમેન modes** — ગ્રાહકો subdomain (મફત), નવો ડોમેન નોંધાવવો અથવા તેમની પાસે પહેલેથી જ હોય તેવા હાલના ડોમેનને mapping કરવો તેમાંથી પસંદ કરે છે
* **રિયલ-ટાઇમ ડોમેન શોધ** — પેજ reload વિના AJAX-powered ઉપલબ્ધતા તપાસ
* **સ્માર્ટ સૂચનો** — પસંદ કરેલો ડોમેન લેવાયેલો હોય ત્યારે વૈકલ્પિક TLD સૂચનો
* **પારદર્શક કિંમત** — નોંધણી કિંમત, નવીકરણ કિંમત અને વૈકલ્પિક WHOIS ગોપનીયતા ફીનું સ્પષ્ટ પ્રદર્શન
* **રજિસ્ટ્રન્ટ સંપર્ક fields** — checkout દરમિયાન inline જરૂરી રજિસ્ટ્રન્ટ માહિતી (નામ, સરનામું, ફોન) એકત્ર કરે છે
* **સ્વયં-ઉત્પન્ન સાઇટ URL** — સાઇટ સરનામું પસંદ કરેલા ડોમેન પરથી આપમેળે બનાવવામાં આવે છે

#### સ્વચાલિત નવીકરણ સિસ્ટમ
* **Membership-સાથે જોડાયેલા નવીકરણો** — ગ્રાહકનું membership સક્રિય હોય ત્યાં સુધી ડોમેન્સ આપમેળે renew થાય છે
* **સ્માર્ટ scheduling** — નવીકરણ jobs ડોમેન expiry datesના આધારે queue કરવામાં આવે છે
* **Retry logic** — નિષ્ફળ નવીકરણ પ્રયાસો માટે બુદ્ધિશાળી retries
* **Email notifications** — આવનારા નવીકરણો, સફળ નવીકરણો, નિષ્ફળતાઓ અને expirations માટે રૂપરેખાંકિત કરી શકાય તેવી alerts

#### ગ્રાહક DNS વ્યવસ્થાપન
* **Self-service DNS** — ગ્રાહકો તેમના Dashboardમાંથી DNS records (A, AAAA, CNAME, MX, TXT) ઉમેરી, સંપાદિત કરી અને કાઢી શકે છે
* **ડોમેન સ્થિતિ** — રિયલ-ટાઇમ નોંધણી સ્થિતિ, expiry dates અને nameserver માહિતી
* **સ્વચાલિત ડોમેન mapping** — નોંધાયેલા અથવા હાલના ડોમેન્સ ગ્રાહકની સાઇટ પર આપમેળે mapped થાય છે

#### Admin Tools
* **હાથથી ડોમેન નોંધણી** — admin panelમાંથી ગ્રાહકોના વતી ડોમેન્સ નોંધાવો
* **ડોમેન monitoring** — એક પેજ પરથી બધા ગ્રાહક ડોમેન્સ, સ્થિતિઓ અને expiry dates જુઓ
* **પ્રદાતા-વિશિષ્ટ logging** — દરેક રજિસ્ટ્રાર માટે અલગ log channels (દા.ત., domain-seller-namecheap.log)
* **નોંધણી વિગતો** — core domain edit page પર રજિસ્ટ્રન્ટ માહિતી અને નોંધણી data જુઓ

### સપોર્ટેડ પ્રદાતાઓ

* **OpenSRS** — નોંધણી, નવીકરણ, DNS વ્યવસ્થાપન અને WHOIS ગોપનીયતા સાથે સંપૂર્ણ એકીકરણ
* **Namecheap** — નોંધણી, નવીકરણ, batch availability checking અને મફત WhoisGuard સાથે સંપૂર્ણ એકીકરણ
* **HostAfrica** — નોંધણી, નવીકરણ, ટ્રાન્સફર, DNS વ્યવસ્થાપન, EPP code retrieval, registrar lock અને ID સુરક્ષા
* **Openprovider** — નોંધણી, નવીકરણ, ટ્રાન્સફર, DNS વ્યવસ્થાપન, EPP code retrieval, registrar lock, WHOIS ગોપનીયતા અને reseller TLD sync
* **Hostinger** — નોંધણી, ડોમેન lookup, nameserver updates, registrar lock અને WHOIS ગોપનીયતા; નવીકરણો Hostinger auto-renew દ્વારા સંભાળવામાં આવે છે
* **GoDaddy** — નોંધણી અને નવીકરણ
* **ResellerClub** — નોંધણી, નવીકરણ અને DNS વ્યવસ્થાપન
* **NameSilo** — નોંધણી અને નવીકરણ
* **Enom** — નોંધણી, નવીકરણ અને DNS વ્યવસ્થાપન

### તકનીકી વિગતો

* **જરૂરી છે:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **આર્કિટેક્ચર:** Domain_Selling_Capability interface સાથે Integration Registry pattern
* **Autoloading:** PSR-4 class mapping સાથે Jetpack autoloader
* **ધોરણો:** WordPress coding standards, strict typing, prepared statements

## સ્થાપન

1. ખાતરી કરો કે **Multisite Ultimate v2.4.12+** ઇન્સ્ટોલ થયેલ છે અને નેટવર્ક-સક્રિય છે
2. plugin ને `/wp-content/plugins/ultimate-multisite-domain-seller/` પર અપલોડ કરો
3. WordPress Network Admin માં **Plugins**માંથી plugin ને નેટવર્ક-સક્રિય કરો
4. પ્રથમ સક્રિયકરણ પર **Setup Wizard** આપમેળે શરૂ થાય છે — નીચેના પગલાં અનુસરો:
   * તમારા domain registrar ને જોડો
   * જોડાણનું પરીક્ષણ કરો
   * ઉપલબ્ધ TLDs અને wholesale pricing આયાત કરો
   * pricing markup સાથે તમારું પ્રથમ domain product બનાવો
5. **Ultimate Multisite > Checkout Forms** માં તમારા checkout form માં **Domain Selection** field ઉમેરો

## વારંવાર પૂછાતા પ્રશ્નો

### Multisite Ultimate નું કયું વર્ઝન જરૂરી છે?

વર્ઝન 2.4.12 અથવા તેથી વધુ. addon WP Ultimo v1.x અથવા 2.4.12 થી નીચેના Ultimate Multisite વર્ઝનો સાથે સુસંગત નથી.

### કયા domain providers સમર્થિત છે?

નવ registrars સમર્થિત છે: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo**, અને **Enom**. registrar APIs અલગ-અલગ કામગીરીઓ પ્રગટ કરતી હોવાથી ક્ષમતાઓ provider મુજબ બદલાય છે; provider-દર-provider અવલોકન માટે setup guide જુઓ. addon ની Integration Registry architecture custom providers ઉમેરવાનું સરળ બનાવે છે — [developer guide](/developer/integration-guide/custom-registrar) જુઓ.

### શું હું WHOIS privacy protection ઓફર કરી શકું?

હા. દરેક domain product માં ત્રણ modes સાથે WHOIS Privacy setting હોય છે: Disabled, Always Included, અથવા Customer Choice. Customer Choice પર સેટ કરેલ હોય ત્યારે, customers checkout દરમિયાન price (અથવા Namecheap ના WhoisGuard માટે "Free") સાથે checkbox જુએ છે. privacy flag registration સમયે આપમેળે registrar ને મોકલવામાં આવે છે.

### શું હું અલગ TLDs માટે અલગ pricing સેટ કરી શકું?

હા. અલગ TLD groups માટે જુદા domain products બનાવો. દરેક product પાસે પોતાની markup type (percentage, fixed markup, અથવા fixed price), supported TLDs, અને introductory pricing હોઈ શકે છે.

### customers કયા checkout options જુએ છે?

Domain Selection field customers ને ત્રણ tabs આપે છે: **Subdomain** (તમારા network પર free subdomain વાપરો), **Register New Domain** (નવો domain શોધો અને register કરો), અને **Existing Domain** (તેઓ પહેલેથી ધરાવતા domain ને map કરો). ફક્ત તમે enable કરેલા modes જ દેખાશે.

### શું તે automatic renewals ને support કરે છે?

હા. Domains auto-renew પર સેટ કરી શકાય છે અને customer's membership status સાથે જોડાયેલા હોય છે. Renewal payments customer's existing payment gateway દ્વારા processed થાય છે.

### શું customers તેમના DNS records manage કરી શકે છે?

હા. Customers ને તેમના dashboard માં A, AAAA, CNAME, MX, અને TXT records ઉમેરવા, edit કરવા, અને delete કરવા માટે DNS management interface મળે છે. DNS management OpenSRS, ResellerClub, Enom, HostAfrica, અને Openprovider સાથે ઉપલબ્ધ છે. hosted domains માટે Hostinger DNS core Hostinger domain-mapping integration દ્વારા handled થાય છે.

### શું setup wizard છે?

હા. setup wizard પ્રથમ activation પર આપમેળે ચાલે છે અને તમને registrar જોડવા, TLDs આયાત કરવા, અને તમારું પ્રથમ domain product બનાવવા માટે માર્ગદર્શન આપે છે. તમે addon settings માંથી wizard ફરીથી પણ ચલાવી શકો છો.
