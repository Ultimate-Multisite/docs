# PayFast {#payfast}

Multisite Ultimate: Payfast Gateway

આમાં Payfast - જે દક્ષિણ આફ્રિકામાં એક ઝડપી અને સુરક્ષિત ઓનલાઈન પેમેન્ટ ગેટવે છે - ને Multisite Ultimate ના પેમેન્ટ વિકલ્પ તરીકે ઉમેરવામાં આવે છે.

## ઇન્સ્ટોલેશન {#installation}

1. 'ultimate-multisite-payfast' ને '/wp-content/plugins/' ડિરેક્ટરીમાં અપલોડ કરો
2. WordPress માં 'Plugins' મેનુ દ્વારા પ્લગઇન એક્ટિવેટ કરો
3. પ્લગઇનને સેટ કરવા માટે સ્ટેપ-બાય-સ્ટેપ વિઝાર્ડને અનુસરો

## ઉપયોગ માર્ગદર્શિકા {#usage-guide}

PayFast દક્ષિણ આફ્રિકામાં વ્યવસાયો માટે એક મુખ્ય ઓનલાઈન પેમેન્ટ પ્રોસેસર છે. તે વ્યવહારોને સરળ અને સુરક્ષિત બનાવીને ઓનલાઈન વેપારીઓને વિકાસ કરવામાં મદદ કરવા માટે સમર્પિત છે. Ultimate Multisite યુઝરની વિનંતી મુજબ, અમે એક એડ-ઓન બનાવવાનું નક્કી કર્યું છે જે PayFast સાથે ઇન્ટિગ્રેશન શક્ય બનાવશે. આ લેખમાં બતાવવામાં આવ્યું છે કે તમારા Ultimate Multisite નેટવર્કમાં પેમેન્ટ મેળવવા માટે PayFast ને તમારા પેમેન્ટ ગેટવે તરીકે કેવી રીતે સેટ કરવું.

પહેલું પગલું એડ-ઓન ઇન્સ્ટોલ કરવાનું છે.

# Payfast એડ-ઓન કેવી રીતે ઇન્સ્ટોલ કરવું: {#how-to-install-the-payfast-add-on}

એડ-ઓન્સ પેજ પર જાઓ અને Ultimate Multisite: Payfast Gateway એડ-ઓન પસંદ કરો.

![Add-ons page](/img/admin/addons-page.png)

એડ-ઓન વિન્ડો પોપ-અપ થશે. 'Install Now' પર ક્લિક કરો.

![Add-ons page with install option](/img/admin/addons-page.png)

ઇન્સ્ટોલેશન પૂર્ણ થયા પછી, તમને પ્લગઇન્સ પેજ પર રીડાયરેક્ટ કરવામાં આવશે. અહીં, ફક્ત 'Network Activate' પર ક્લિક કરો અને _Payfast_ એડ-ઓન તમારા નેટવર્ક પર એક્ટિવેટ થઈ જશે.

# PayFast સેટઅપ કરવું {#setting-up-payfast}

એડ-ઓન ઇન્સ્ટોલ કર્યા પછી, તમે હવે Ultimate Multisite સેટિંગ્સ પર જઈને PayFast કન્ફિગર કરવાનું શરૂ કરી શકો છો.

**Ultimate Multisite settings** પેજ હેઠળ, તમને "Payments" ટેબ મળી શકે છે. આ ટેબ હેઠળ, તમને આ પેમેન્ટ ગેટવેને enable અથવા disable કરવા માટે એક ટૉગલ બટન સાથે "**Payfast** " મળશે.

![Payment gateway settings](/img/config/settings-payment-gateways.png)આગળ વધીને તેને enable કરવા માટે ટૉગલ બટન પર ક્લિક કરો. પછી તે બોક્સ વિસ્તરીને વિવિધ ફીલ્ડ્સ બતાવશે જેને તમારે ભરી શલબલ છે, જેમ કે **Mechant ID, Merchant Key અને PassPhrase.** તમે આ માહિતી તમારા **Payfast** એકાઉન્ટમાંથી મેળવી શકો છો

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

આગળનું પગલું જરૂરી માહિતી મેળવવા માટે તમારા PayFast એકાઉન્ટમાં લોગિન કરવાનું છે.

Upon logging in, you can immediately see the credentials you need from your **Payfast** account dashboard.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->તમે ફક્ત ક્રેડેન્શિયલને કોપી અને પેસ્ટ કરીને તમારા Ultimate Multisite સેટિંગ્સમાં વિગતો ભરી શકો છો. પછી ફેરફારો સેવ કરવા માટે **"Save settings** " બટન પર ક્લિક કરો.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

બસ, તમારું PayFast પેમેન્ટ ઇન્ટિગ્રેશન પૂર્ણ થઈ ગયું છે!
