# Filter: wp_ultimo_skip_network_active_check {#filter-wpultimoskipnetworkactivecheck}

ડેવલપર્સને આ ચેકને અવગણવાની (short-circuit) મંજૂરી આપે છે.

આ ત્યારે ખૂબ ઉપયોગી છે જ્યારે તમે composer-આધારિત અને અન્ય કસ્ટમ સેટઅપ્સ, જેમ કે Bedrock, નો ઉપયોગ કરી રહ્યા હોવ, જ્યાં mu-plugins તરીકે પ્લગઇન્સનો ઉપયોગ સામાન્ય હોય.

- 2.0.0
