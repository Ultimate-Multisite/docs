# ભાષા પસંદગીકાર (Language Selector)

Multisite Ultimate: Sign-up પર ભાષા પસંદગીકાર

સાઇન-અપ પ્રક્રિયામાં ભાષા પસંદગીકાર ફીલ્ડ ઉમેરો.

## સ્થાપન (Installation)

1. 'ultimate-multisite-language-selector' ને '/wp-content/plugins/' ડિરેક્ટરીમાં અપલોડ કરો
2. તમારા WordPress Network Admin માં 'Plugins' મેનૂ દ્વારા પ્લગઇન એક્ટિવેટ કરો

## ઉપયોગ માર્ગદર્શિકા (Usage Guide)

આ મફત એડ-ઓન સાથે, તમે તમારા વપરાશકર્તાઓને સાઇન-અપ પ્રક્રિયા અને લોગિન પેજ પર કઈ ભાષા જોવા માંગે છે તે પસંદ કરવા દેવા શકો છો.

[તમારી લાયસન્સ એક્ટિવેટ કર્યા પછી](../../user-guide/getting-started/installing-ultimate-multisite.md) તમે તે Ultimate Multisite એડ-ઓન્સ ઇન્સ્ટોલ કરી શકશો જે તમારી લાયસન્સ દ્વારા તમને એક્સેસ મળે છે.

તમારી Ultimate Multisite સેટિંગ્સ પેજ પર જાઓ અને પછી, ડાબી બાજુના મેનૂમાં, _Add-ons_ લેબલ પર ક્લિક કરો.

![Add-ons page](/img/admin/addons-page.png)

_ભાષા પસંદગીકાર ઇન્સ્ટોલ કરવાનું શરૂ કરવા માટે Add-ons પેજ પર જાઓ_

ત્યાં, તમને બધા એડ-ઓન્સ મળશે જે અમે ઓફર કરીએ છીએ.

મફત એડ-ઓન્સ અમારા તમામ ગ્રાહકો માટે ઉપલબ્ધ છે, ભલે તેમણે કયા સબ્સ્ક્રિપ્શન પ્લાન માટે સાઇન અપ કર્યું હોય.

એડ-ઓનના થંબનેલની નીચે આપેલ _Install it Now!_ બટન પર ક્લિક કરો. પછી તેને Plugins પેજ પર એક્ટિવેટ કરો.

![Add-ons page with install button](/img/admin/addons-page.png)

_એડ-ઓન ઇન્સ્ટોલ કરવા માટે બટન પર ક્લિક કરો_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_તેને તમારા Plugins પેજ પર એક્ટિવેટ કરો_

હવે, Ultimate Multisite ના Settings મેનૂ પર જાઓ અને _Network Settings_ ટેબમાં પ્રવેશ કરો. ત્યાં, પેજની નીચે, તમને _Languages_ નામનો એક નવો વિભાગ મળશે.

![Settings page](/img/config/settings-general.png)

_એક નવો વિભાગ દેખાય છે!_

તમે તેને ખાલી રાખીને બધા ભાષાઓને વિકલ્પો તરીકે બતાવી શકો છો અથવા તમે કઈ ભાષાઓ ઉપલબ્ધ કરાવવા માંગો છો તે પસંદ કરી શકો છો. ભાષાઓ એ છે જે તમે તમારા WordPress સેટઅપમાં ઇન્સ્ટોલ કરી છે.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_આ WP ઇન્સ્ટોલેશન પર ઉપલબ્ધ ભાષાઓની યાદી_

તમે કરેલા ફેરફારોને સેવ કર્યા પછી, તમે પસંદ કરેલી ભાષાઓ સાથેનું એક પસંદગી મેનુ લોગિન પેજ તેમજ સાઇન-અપ ફ્લો પર પણ પ્રદર્શિત થશે.

![Login and Registration settings](/img/config/settings-login-registration.png)

_લોગિન પેજ પર ભાષા પસંદગીકાર_

![Registration page](/img/config/registration-page.png)

_સાઇન-અપ ફ્લો પર ભાષા પસંદગીકાર_

## URL દ્વારા ભાષા સેટ કરવી

તમે લોકેલ પેરામીટરને બાયપાસ કરીને URL માં પણ પેજની ભાષા બદલી શકો છો. ઉદાહરણ તરીકે, જો તમે ઇટાલિયનમાં રજીસ્ટર પેજ પર રીડાયરેક્ટ કરવા માંગતા હો, તો URL માં ?locale=it_IT ઉમેરો.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

આ કામ કરવા માટે ભાષા તમારા WordPress નેટવર્કમાં ઇન્સ્ટોલ હોવી જરૂરી છે.
