# લેંગ્વેજ સિલેક્ટર ચેન્જલોગ

સંસ્કરણ: 1.1.3 - 2026-05-05 ના રોજ રજૂ કરવામાં આવ્યું
* સુધારો: ઇનપુટ વેરીએબલ્સ હવે સાફ (sanitized) કરવામાં આવે છે અને તૃતીય-પક્ષ પ્લગઇન્સ સાથેના ટકરાવને રોકવા માટે કસ્ટમ હૂક નામોને પ્રીફિક્સ આપવામાં આવ્યું છે.

સંસ્કરણ: 1.1.2 - 2026-01-01 ના રોજ રજૂ કરવામાં આવ્યું
* સુધારેલ: સુસંગતતા અપડેટ્સ અને કોડની ગુણવત્તામાં સુધારો.

સંસ્કરણ: 1.1.0 - 2025-09-28 ના રોજ રજૂ કરવામાં આવ્યું
* પ્રીફિક્સનું નામ ultimate-multisite માં બદલ્યું; ટેક્સ્ટ ડોમેન અપડેટ કર્યો; સંસ્કરણ વધારો.
