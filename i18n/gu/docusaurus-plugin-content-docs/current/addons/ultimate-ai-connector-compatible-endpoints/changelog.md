# Ultimate AI Connector for Compatible Endpoints નું Changelog {#ultimate-ai-connector-for-compatible-endpoints-changelog}

## Version 2.0.0 — 2026-04-24 ના રોજ રજૂ કરવામાં આવ્યું {#version-200--released-on-2026-04-24}

* નવું: બહુવિધ પ્રદાતા સપોર્ટ — એકસાથે ઘણા AI endpoints ગોઠવી શકાય છે અને પ્રદાતાઓમાં આપમેળે fallback સાથે requests નું માર્ગદર્શન કરી શકાય છે.
* સુધારો: દરેક પ્રદાતા માટે સાચા પ્રદાતા IDs, રજીસ્ટ્રેશન URLs, અને HTTP filter scoping સાથે Multi-provider SDK નું એકીકરણ.
* સુધારો: નવું પ્રદાતા કાર્ડ હવે ઉમેરતા જ આપમેળે expand થાય છે; plugin update પર script cache busting.
* સુધારો: eval() માટે dynamic provider class namespace અને નવા ઉમેરાયેલા પ્રદાતાઓ માટે auto-expand behaviour.
* સુધારો: WordPress 6.9+ સુસંગતતા માટે સ્થિર Card/CardBody/CardHeader/CardDivider components નો ઉપયોગ કર્યો (હવે પ્રાયોગિક નથી).
* સુધારો: પ્રદાતાને drag-to-reorder કરવા માટે undefined DragHandle ને unicode grip icon સાથે બદલવામાં આવ્યું.
* સુધારો: દરેક page load પર થતી blocking HTTP request ને દૂર કરવામાં આવી.
* સુધારેલ: GitHub Actions workflows ને Node.js 24 માં અપગ્રેડ કરવામાં આવ્યા.
