# VAT ફેરફારની નોંધ {#vat-changelog}

Version 1.0.7 - 2026-02-03 ના રોજ રિલીઝ

* સુધારો: VIES SOAP service MS_MAX_CONCURRENT_REQ ભૂલોને કારણે જર્મન (DE) VAT નંબરોની માન્યતા ચકાસણીમાં સમસ્યા આવતી હતી. વધુ વિશ્વસનીય માન્યતા માટે જૂના SOAP API ને બદલે હવે સત્તાવાર EU VIES REST API નો ઉપયોગ કરવામાં આવ્યો છે.
* સુધારો: હવે VIES સેવા કામચલાઉ રીતે અનુપલબ્ધ હોય તો પણ VAT નંબરો સ્વીકારવામાં આવે છે, અને ભૂલથી નકારી કાઢવામાં આવતા નથી. સમીક્ષા માટે એક લોગ એન્ટ્રી બનાવવામાં આવે છે.
* સુધારો: VAT/Tax ID ફીલ્ડ હવે માત્ર EU સભ્યો માટે જ નહીં, પરંતુ તમામ દેશો માટે દેખાય છે. નોન-EU ટેક્સ ID (જેમ કે Swiss CHE નંબરો) VIES માન્યતા વિના ઇન્વૉઇસ પ્રદર્શન માટે સ્ટોર કરવામાં આવે છે.
* સુધારો: ગ્રીસ (GR), મોનાકો (MC), અને આઇલ ઓફ મેન (IM) માટે VAT પ્રિફિક્સ લુકઅપ ખોટો હતો કારણ કે એરે લુકઅપ લોજિકમાં ભૂલ હતી.
* સુગમતા: એક બિનજરૂરી wp-cli-bundle dev dependency દૂર કરવામાં આવી છે, જે Site Exporter addon સાથે ઉપયોગમાં લેવાતી વખતે fatal errors નું કારણ બની શકે છે.
* સુગમતા: વ્યાપક યુનિટ ટેસ્ટ સ્યુટ (53 ટેસ્ટ) ઉમેરવામાં આવ્યું છે.

Version 1.0.6 - 2026-01-25 ના રોજ રિલીઝ

* સુધારો: હવે ઘરેલું B2B વ્યવહારો પર યોગ્ય રીતે VAT લેવામાં આવે છે. EU VAT નિયમો મુજબ, રિવર્સ ચાર્જ ફક્ત સરહદ પાર B2B વ્યવહારો પર લાગુ પડે છે, જ્યારે ગ્રાહકનો દેશ કંપનીના દેશ સાથે મેળ ખાય છે ત્યારે નહીં.

Version 1.0.5 - 2026-01-22 ના રોજ રિલીઝ

* સુધારો: Addon લોડ થતો ન હતો અથવા યોગ્ય રીતે કામ કરતો ન હતો.
* ફેરફાર: વધુ વિશ્વસનીય અને સક્રિય રીતે જાળવવામાં આવતા ડેટા માટે VAT રેટ ડેટા સ્ત્રોત euvatrates.com માંથી ibericode/vat-rates રિપોઝિટરીમાં બદલવામાં આવ્યો છે.
* સુધારો: super_reduced_rates ઓપ્શન વેલ્યુમાં ટાઇપો સુધારવામાં આવ્યો છે.
* સુગમતા: ભૂલો અટકાવવા માટે country code હેન્ડલિંગ માટે null checks ઉમેરવામાં આવ્યા છે.
* ફેરફાર: Traduttore દ્વારા સ્વચાલિત અપડેટ્સને પ્રાધાન્ય આપવાને કારણે બંડલ્ડ ટ્રાન્સલેશન ફાઇલો દૂર કરવામાં આવી છે.
