# Action: wp_ultimo_vat_load

જ્યારે બધા નિર્ભરતાઓ (dependencies) લોડ થઈ જાય ત્યારે ટ્રિગર થાય છે.

આ પ્લગઇન ડેવલપર્સને નવી કાર્યક્ષમતા ઉમેરવાની મંજૂરી આપે છે. ઉદાહરણ તરીકે, નવા હોસ્ટિંગ પ્રોવાઇડર્સને સપોર્ટ ઉમેરવો વગેરે.
