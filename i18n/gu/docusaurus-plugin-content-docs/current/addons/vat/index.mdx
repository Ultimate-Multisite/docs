# VAT

Multisite Ultimate: European VAT

Multisite Ultimateના ડિફોલ્ટ ટેક્સ હેન્ડલિંગમાં વધારાની સુવિધાઓ અને સાધનો ઉમેરે છે, જેથી યુરોપિયન VAT ને વધુ સારી રીતે સપોર્ટ કરી શકાય.
આ પ્લગઇન Tax Rates પેજમાં આપમેળે EU VAT દરો ઉમેરે છે.

## Installation {#installation}

1. 'ultimate-multisite-vat' ને '/wp-content/plugins/' ડિરેક્ટરીમાં અપલોડ કરો.
2. WordPress માં 'Plugins' મેનૂ દ્વારા પ્લગઇનને એક્ટિવેટ કરો.
3. Ultimate Multisite > Settings > Taxes પર જાઓ અને EU Vat ને Enable કરો.
4. Tax Rates પર ક્લિક કરો અને નીચે આપેલા EU VAT દરો અપડેટ કરો અને Tax Rates સેવ કરો.
5. તૈયાર!
