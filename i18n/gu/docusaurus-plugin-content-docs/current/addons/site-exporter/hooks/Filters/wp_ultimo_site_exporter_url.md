# Filter: wp_ultimo_site_exporter_url

ડેવલપર્સને પ્લગઇનના મૂળ (plugin root) સુધીના સાપેક્ષ માર્ગ (relative path) નું સંચાલન કરવાની મંજૂરી આપે છે.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $dir | `string` | પ્લગઇનના મૂળથી જે માર્ગ (path) તમારે એક્સેસ કરવો છે. |

### Since

- 0.0.1
### Source

[`inc/class-helper.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-site-exporter/blob/main/inc/class-helper.php#L52) માં લાઇન 52 પર વ્યાખ્યાયિત છે.

## Returns
પ્લગઇનના મૂળથી સાપેક્ષ માર્ગ.
