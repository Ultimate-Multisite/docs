# Action: wp_ultimo_site_exporter_load {#action-wpultimositeexporterload}

જ્યારે તમામ dependencies લોડ થઈ જાય ત્યારે આ એક્શન ટ્રિગર થાય છે.

આ પ્લગઇન ડેવલપર્સને નવી કાર્યક્ષમતા (functionality) ઉમેરવાની મંજૂરી આપે છે. ઉદાહરણ તરીકે, નવા હોસ્ટિંગ પ્રોવાઇડર્સને સપોર્ટ કરવો, વગેરે.
