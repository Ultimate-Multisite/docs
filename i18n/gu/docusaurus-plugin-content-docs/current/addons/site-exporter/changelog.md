# સાઇટ એક્સપોર્ટર ફેરફારની નોંધ {#site-exporter-changelog}

Version: 1.0.6 - Released on 2026-05-11
* નવું: સાઇટ એક્સપોર્ટ્સ હવે એક સ્વયં-બૂટિંગ index.php સાથે પેક થાય છે, જેથી ZIP ને અલગ પ્લગઇન ઇન્સ્ટોલ કર્યા વિના નવા હોસ્ટ પર ઇન્સ્ટોલ કરી શકાય.
* નવું: નેટવર્ક એક્સપોર્ટ એડમિનિસ્ટ્રેટર્સને સાઇટ એક્સપોર્ટ એડમિન પેજ પરથી તમામ સબસાઇટ્સને એક જ આર્કાઇવમાં એક્સપોર્ટ કરવાની મંજૂરી આપે છે.

Version: 1.0.5 - Released on 2026-05-05
* સુધારો: `get_config` જેવા અનમોક કરેલા મેથડ્સને હેન્ડલ કરવા માટે WP_CLI polyfill stub માં `__callStatic` ઉમેર્યું, જેનાથી ટેસ્ટ બૂટસ્ટ્રેપ નિષ્ફળતા અટકે છે.

Version: 1.0.4 - Released on 2025-11-25

* Ultimate Multisite 2.4 સાથે સુસંગતતા (compatibility) સુધારેલ છે.

Version: 1.0.2 - Released on 2025-09-28

* પ્રીફિક્સને ultimate-multisite માં નામ બદલ્યું; ટેક્સ્ટ ડોમેન અપડેટ કર્યો; વર્ઝન વધાર્યું.

### Version 1.0.1 - Released on 2023-08-09 {#version-101---released-on-2023-08-09}

* સુધારેલ: હવે ખાતરી કરે છે કે ઇમ્પોર્ટર ફાઇલનો સાચો ફાઇલ પાથ મળે.
* સુધારો: ઇમ્પોર્ટ દરમિયાન ભૂલો અટકાવવા માટે ઓબ્જેક્ટ કેશ (object cache) બંધ કર્યો.
* સુધારો: ઇમ્પોર્ટર પ્લગઇન ડાઉનલોડ લિંક કામ ન કરવાના મુદ્દાને ઠીક કર્યો.

### Version 1.0.0 - Released on 2022-12-23 {#version-100---released-on-2022-12-23}

* સુધારો: એક્સપોર્ટ ફોર્મ પોપઅપ લોડ ન થવાનો મુદ્દો;
* સુધારો: એક્સપોર્ટ પેજ પર Javascript ફાઇલો લોડ ન થવાનો મુદ્દો;
* આંતરિક: નવો બિલ્ડ પ્રોસેસ;

### Version 1.0.0-beta.4 - Released on 2022-01-13 {#version-100-beta4---released-on-2022-01-13}

* આંતરિક: હૂક્સ અને ફિલ્ટર્સ જનરેટર ઉમેર્યું;
* આંતરિક: ડેવલપરની સુવિધા (quality of life) માટે WP Ultimo સ્ટબ્સ ઉમેર્યા;
