# Loco Translate

Ultimate Multisite: Loco Translate Integration addon તમારા WordPress multisite network માં site-specific translation capabilities લાવે છે. જો તમને ક્યારેય જરૂર પડી હોય કે તમારા network માં અલગ-અલગ સાઇટ્સને એક જ plugin અથવા theme ના અલગ-અલગ translations જોઈતા હોય, તો આ addon શ્રેષ્ઠ ઉકેલ છે.

**આ addon શું કરે છે**

આ addon Ultimate Multisite ને લોકપ્રિય Loco Translate plugin સાથે સરળતાથી જોડે છે, જેનાથી તમારા network માં દરેક subsite ને પોતાના કસ્ટમ translations જાળવી રાખવાની મંજૂરી મળે છે. ભલે તમે કોઈ multi-regional platform ચલાવી રહ્યા હોવ, અલગ ભાષાના સમુદાયોને સેવા આપી રહ્યા હોવ, અથવા ફક્ત તમારા network માં translation ની લવચીકતા (flexibility) જોઈતી હોય, આ addon તેને ખૂબ સરળ બનાવી દે છે.

**આ કોના માટે શ્રેષ્ઠ છે:**

* વિવિધ સ્થાનિક જરૂરિયાતોવાળા multi-regional વ્યવસાયો
* વિવિધ સમુદાયોને સેવા આપતા શૈક્ષણિક સંસ્થાઓ
* બહુભાષી (multilingual) સપોર્ટ આપતા SaaS પ્લેટફોર્મ્સ
* અનેક client સાઇટ્સનું સંચાલન કરતી વેબ એજન્સીઓ
* translation ની લવચીકતાની જરૂર હોય તેવું કોઈપણ multisite network

## ઇન્સ્ટોલેશન

1. ખાતરી કરો કે Ultimate Multisite ઇન્સ્ટોલ અને network એક્ટિવેટ થયેલ છે
2. Loco Translate plugin ઇન્સ્ટોલ અને એક્ટિવેટ કરો
3. આ addon ને તમારા plugins directory માં અપલોડ કરો
4. Loco Translate Integration addon ને network સ્તરે એક્ટિવેટ કરો
5. કોઈપણ subsite પર Loco Translate દ્વારા site-specific translations બનાવવાનું શરૂ કરો

બસ આટલું! કોઈ વધારાના configuration વગર આ integration આપોઆપ કામ કરે છે.

## વારંવાર પૂછાતા પ્રશ્નો

## શું ઇન્સ્ટોલેશન પછી મારે કંઈ configure કરવાની જરૂર છે?

ના! આ integration આપોઆપ છે. એકવાર addon અને Loco Translate બંને active થઈ જાય, પછી subsites તરત જ પોતાના કસ્ટમ translations બનાવવાનું શરૂ કરી શકે છે.

## જો કોઈ subsite કસ્ટમ translations ન બનાવે તો શું થશે?

સાઇટ આપમેળે fallback તરીકે network-wide translations નો ઉપયોગ કરશે. કંઈ તૂટતું નથી, અને બધું સરળતાથી કામ કરે છે.

## શું subsites network-wide translations એક્સેસ કરી શકે છે?

સામાન્ય site admins ફક્ત તેમની પોતાની સાઇટના translations જોઈ અને એડિટ કરી શકે છે. Super admins જે સાઇટ જોઈ રહ્યા છે તેના આધારે network-wide અને site-specific બંને translations એક્સેસ કરી શકે છે.

## શું આ plugins અને themes બંને સાથે કામ કરે છે?

હા! Site-specific translations WordPress plugins અને themes બંને માટે કામ કરે છે.

## શું આનાથી મારું multisite network ધીમું પડશે?

ના. આ addon કાર્યક્ષમ file loading નો ઉપયોગ કરે છે જે ફક્ત જરૂર પડ્યે જ site-specific translations ની તપાસ કરે છે. કોઈ નોંધપાત્ર performance impact નથી.

## શું હું network translations ને કોઈ ચોક્કસ સાઇટ માટે શરૂઆત તરીકે કોપી કરી શકું?

હા! Loco Translate માં translations એડિટ કરતી વખતે, તમે સરળતાથી network translations થી શરૂઆત કરી શકો છો અને દરેક સાઇટ માટે તેને કસ્ટમાઇઝ કરી શકો છો.

## શું આ WooCommerce અને અન્ય plugins સાથે કામ કરે છે?

ચોક્કસ! કોઈપણ plugin અથવા theme જે standard WordPress translation functions નો ઉપયોગ કરે છે, તે site-specific translations સાથે કામ કરશે.
