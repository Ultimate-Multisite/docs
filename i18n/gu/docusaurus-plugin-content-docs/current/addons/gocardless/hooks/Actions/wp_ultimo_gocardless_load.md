# Action: wp_ultimo_gocardless_load

જ્યારે બધા dependencies load થઈ જાય ત્યારે આ action trigger થાય છે.

આ plugin developers ને નવી functionality ઉમેરવા દે છે. ઉદાહરણ તરીકે, નવા Hosting providers માટે સપોર્ટ ઉમેરવો, વગેરે.
