# GoCardless ફેરફારની નોંધ

Version: 1.0.3 - રજૂ કરવામાં આવ્યું 2026-05-05
* સુધારો: જ્યારે GoCardless સભ્યપદ પેન્ડિંગ અથવા હોલ્ડ સ્થિતિમાં રદ કરવામાં આવે છે, ત્યારે છોડી દેવાયેલી પેન્ડિંગ સાઇટ્સ હવે સ્વચાલિત રીતે સાફ થઈ જાય છે.
* સુધારેલ: જ્યારે Bedrock root autoloader એ નિર્ભરતાઓ (dependencies) પહેલેથી લોડ કરી લીધી હોય, ત્યારે પ્લગઇન ઓટોલોડરને છોડી દેવામાં આવે છે.

Version: 1.0.1 - રજૂ કરવામાં આવ્યું 2025-09-28

* પ્રીફિક્સને ultimate-multisite માં નામ બદલ્યું; ટેક્સ્ટ ડોમેન અપડેટ કર્યું; અને સંસ્કરણમાં વધારો કર્યો.

2021-09-24 - Version 1.0.0-beta.4

* ઉમેર્યું: mu-plugins આધારિત સેટઅપ્સ માટે filter wp_ultimo_skip_network_active_check;

2021-06-16 - Version 1.0.0-beta.3

* સુધારો: વેબહૂક હેન્ડલર્સને સહીઓ (signatures)ની પુષ્ટિ કરવામાં નિષ્ફળતા મળી હતી.
