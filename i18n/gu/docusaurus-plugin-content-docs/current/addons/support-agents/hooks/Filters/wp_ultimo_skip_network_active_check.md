# Filter: wp_ultimo_skip_network_active_check

ડેવલપર્સને આ ચેકને બાયપાસ (short-circuit) કરવાની મંજૂરી આપે છે.

આ ત્યારે ઉપયોગી છે જ્યારે તમે composer-આધારિત અને અન્ય કસ્ટમ સેટઅપ્સનો ઉપયોગ કરી રહ્યા હોવ, જેમ કે Bedrock. આ પ્રકારના સેટઅપ્સમાં, mu-plugins તરીકે પ્લગઇન્સનો ઉપયોગ સામાન્ય હોય છે.
