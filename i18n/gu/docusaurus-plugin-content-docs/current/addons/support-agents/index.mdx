# Support Agents

એક આખું નેટવર્ક જાતે મેનેજ કરવું એ બહુ સરળ કામ નથી. જેમ જેમ તમારું નેટવર્ક વધે છે, તેમ તેમ સપોર્ટની માંગ પણ વધે છે. તમારું કામ હળવું કરવા માટે લોકોને મદદ લેવી સારી વાત છે. જોકે, તેમને સંપૂર્ણ એક્સેસ/પરવાનગી આપવી જોખમી છે અને તે તમારા નેટવર્કને નુકસાન પહોંચાડી શકે છે. અહીં જ આ Support Agent add-on કામમાં આવે છે.

આ add-on તમને Support Agent નામના એક નવા પ્રકારના WordPress યુઝર બનાવવાની મંજૂરી આપે છે, અને તે યુઝર પાસે કઈ પરવાનગીઓ (permissions) છે તેના પર તમને સંપૂર્ણ નિયંત્રણ આપે છે. પરવાનગી આપવી એટલી સરળ છે જેટલી કોઈ ટૉગલ બટન (toggle button) ચાલુ કરવી, અને જો તમારે તે દૂર કરવી હોય તો તેને બંધ કરી શકાય છે.

## Features

* WordPress અને Ultimate Multisite ની પરવાનગીઓની વિગતવાર યાદી
* સરળતાથી એક્સેસને ચાલુ અને બંધ કરો
* અમર્યાદિત એજન્ટ એકાઉન્ટ્સ બનાવો

## Installation

1. 'ultimate-multisite-support-agents' ને '/wp-content/plugins/' ડિરેક્ટરીમાં અપલોડ કરો
2. WordPress માં 'Plugins' મેનૂ દ્વારા પ્લગઇન એક્ટિવેટ કરો
3. પ્લગઇન સેટ કરવા માટે સ્ટેપ-બાય-સ્ટેપ વિઝાર્ડ (Wizard) ને અનુસરો
