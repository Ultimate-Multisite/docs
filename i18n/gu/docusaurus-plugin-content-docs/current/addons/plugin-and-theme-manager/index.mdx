# પ્લગઇન અને થીમ મેનેજર

Multisite Ultimate: Plugin and Theme Manager

ઇન્સ્ટોલ કરેલા પ્લગઇન અને થીમ્સની મેટા માહિતી (શીર્ષક, લેખક, થંબનેલ અને વર્ણન)ને એડિટ કરો અથવા છુપાવો. તમારા વપરાશકર્તાઓને ફિલ્ટર કરવા માટે પ્લગઇન અને થીમ્સ માટે કેટેગરી બનાવો અને તમારા વપરાશકર્તાઓના પેનલ પર એક સુંદર કસ્ટમ પ્લગઇન્સ પેજ ઉમેરો.

આ પ્લગઇનનો ઉપયોગ કરવા માટે, તમે ફક્ત તમારા નેટવર્ક એડમિન પરના તમારા પ્લગઇન્સ અથવા થીમ્સ પેજીસ પર જઈને "Edit Plugin/Theme info" પર ક્લિક કરી શકો છો.

## ઇન્સ્ટોલેશન {#installation}

1. 'ultimate-multisite-plugin-and-theme-manager' ને '/wp-content/plugins/' ડિરેક્ટરીમાં અપલોડ કરો
2. તમારા WordPress Network Admin માં 'Plugins' મેનૂ દ્વારા પ્લગઇનને એક્ટિવેટ કરો

## ઉપયોગ માર્ગદર્શિકા {#usage-guide}

Ultimate Multisite: Plugin and Theme Manager એક શક્તિશાળી **પ્રીમિયમ** Ultimate Multisite એડ-ઓન છે જે તમને તમારા નેટવર્ક પર તમારા નેટવર્ક વપરાશકર્તાઓ પ્લગઇન અને થીમ વિકલ્પોને કેવી રીતે જુએ છે અને તેની સાથે ક્રિયાપ્રતિક્રિયા કરે છે તેના પર સંપૂર્ણ નિયંત્રણ આપે છે.

આ એડ-ઓન ત્રણ હેતુઓ પૂરા પાડે છે:

  1. તે તમને પ્લગઇન અને થીમ્સ બંનેની મેટા માહિતીને સંપૂર્ણપણે બદલવાની મંજૂરી આપે છે. આમાં નામ, વર્ણન, લેખકનું નામ અને થંબનેલ પણ શામેલ છે!

  2. તે તમને પ્લગઇન અને થીમ્સને કેટેગરાઇઝ કરવાની મંજૂરી આપે છે, જેથી તમારા વપરાશકર્તાઓ તેમને સરળતાથી સૉર્ટ કરી શકે.

  3. તે તમને જૂના, કંટાળાજનક પ્લગઇન્સ પેજને કેટેગરી દ્વારા સૉર્ટ થતા નવા થીમ્સ જેવા પેજથી બદલવાનો વિકલ્પ આપે છે.

તેને ડાઉનલોડ અને ઇન્સ્ટોલ કર્યા પછી, Ultimate Multisite Settings પેજ પર એક નવો ટેબ - _Add-on Settings_ - દેખાશે.
