# Filter: wp_ultimo_skip_network_active_check

ડેવલપર્સને આ ચેકને ટૂંકમાં (short-circuit) કરવાની મંજૂરી આપે છે.

આ ત્યારે ઉપયોગી છે જ્યારે તમે composer-based અને અન્ય કસ્ટમ સેટઅપ્સ (custom setups) નો ઉપયોગ કરી રહ્યા હોવ, જેમ કે Bedrock. આ પરિસ્થિતિમાં, mu-plugins તરીકે પ્લગઇન્સનો ઉપયોગ કરવો સામાન્ય છે.
