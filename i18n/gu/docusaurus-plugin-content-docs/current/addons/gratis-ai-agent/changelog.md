# ચેન્જલોગ

## ૧.૯.૦ — ૨૦૨૬-૦૪-૨૮ ના રોજ રિલીઝ

### નવી સુવિધાઓ (New)

- **`create_contact_form` ક્ષમતા** — આ સક્રિય ફોર્મ પ્લગિન (Contact Form 7, WPForms, Fluent Forms, અથવા Gravity Forms) નો ઉપયોગ કરીને કોન્ટેક્ટ ફોર્મ બનાવે છે અને તેને કોઈપણ પોસ્ટ અથવા પેજમાં એમ્બેડ કરવા માટે તૈયાર શોર્ટકોડ (shortcode) પાછો આપે છે.
- **`set_featured_image` ક્ષમતા** — હાલના મીડિયા લાઇબ્રેરી अटैચમેન્ટ ID અથવા રિમોટ URL માંથી પોસ્ટ અથવા પેજ માટે મુખ્ય છબી (featured image) સોંપે છે; જો URL આપવામાં આવે તો આપમેળે છબી ઇમ્પોર્ટ કરે છે.
- **`batch_create_posts` ક્ષમતા** — એક જ ક્ષમતા કૉલ (ability call) માં બહુવિધ પોસ્ટ્સ બનાવે છે. તે `create_post` જેવી જ પેરામીટર્સને સપોર્ટ કરે છે, દરેક પોસ્ટની સફળતા/નિષ્ફળતાની જાણ કરે છે, અને તેમાં વૈકલ્પિક `stop_on_error` મોડ પણ છે.
- **`page_template` પેરામીટર** — `create_post` અને `update_post` માં ઉમેરવામાં આવ્યું છે. આ કૉલ દરમિયાન PHP પેજ ટેમ્પલેટ ફાઇલ (દા.ત., `page-full-width.php`) સોંપે છે. `update_post` માં થીમ ડિફોલ્ટ પર પાછા જવા માટે ખાલી સ્ટ્રિંગ (empty string) પાસ કરો.
- **ક્લાયન્ટ-સાઇડ સ્ક્રીનશોટ ક્ષમતાઓ** — `capture_screenshot`, `compare_screenshots`, અને `review_page_design`. સર્વર-સાઇડ હેડલેસ બ્રાઉઝર દ્વારા લાઇવ પેજની સંપૂર્ણ અથવા Viewport સ્ક્રીનશોટ લે છે, બે સ્ક્રીનશોટની તુલના (diff) કરે છે, અને લેઆઉટ, ટાઇપોગ્રાફી, કલર અને એક્સેસિબિલિટીને આવરી લેતો AI-જનરેટેડ ડિઝાઇન રિવ્યૂ આપે છે.
- **પાંચ બિલ્ટ-ઇન એજન્ટ્સ** — કન્ટેન્ટ રાઇટર (Content Writer), સાઇટ બિલ્ડર (Site Builder), ડિઝાઇન સ્ટુડિયો (Design Studio), પ્લગિન મેનેજર (Plugin Manager), અને સપોર્ટ આસિસ્ટન્ટ (Support Assistant). દરેક એજન્ટ પાસે ટૂલ્સનો એક સમર્પિત સેટ, એક કસ્ટમાઇઝ્ડ સિસ્ટમ પ્રોમ્પ્ટ અને સ્ટાર્ટર સૂચનો હોય છે. ચેટ હેડરમાં નવા **Agent Picker** દ્વારા સ્વિચ કરી શકાય છે. [Built-in Agents](../../user-guide/configuration/built-in-agents) જુઓ.
- **ફીચર ફ્લેગ્સ (Feature flags)** — એક્સેસ-કંટ્રોલ ટોગલ્સ (એડમિનિસ્ટ્રેટર્સ સુધી મર્યાદિત કરો, નેટવર્ક એડમિનિસ્ટ્રેટર્સ સુધી મર્યાદિત કરો, સબ્સ્ક્રાઇબર એક્સેસ, સભ્યો ન હોય તો ડિસેબલ કરો) અને બ્રાન્ડિંગ વિકલ્પો (ફૂટર એટ્રિબ્યુશન છુપાવો, કસ્ટમ એજન્ટ નામ, એજન્ટ પિકર છુપાવો, ચેટ અવતાર તરીકે સાઇટ આઇકનનો ઉપયોગ કરો) સાથે નવો **Settings → Feature Flags** ટેબ. [Gratis AI Agent Settings](../../user-guide/administration/gratis-ai-agent-settings) જુઓ.
- **છેલ્લું સત્ર પુનઃસ્થાપિત કરવું (Restore last session)** — ચેટ પેનલ હવે પેજ લોડ પર અને વિજેટ ખોલવા પર સૌથી તાજેતરની વાતચીત આપમેળે રિલોડ કરે છે, જેથી પેજ નેવિગેશન દરમિયાન કન્ટેક્સ્ટ ક્યારેય ખોવાતો નથી.
- **પ્લગિન એક્શન લિંક્સ (Plugin action links)** — હવે WordPress ના **Plugins → Installed Plugins** સ્ક્રીન પર પ્લગિન વર્ણન નીચે સેટિંગ્સ અને એબિલિટીઝ રજિસ્ટ્રી (Abilities Registry) માટે ઝડપી લિંક્સ દેખાય છે.

### સુધારાઓ (Improved)

- **છબી સ્ત્રોત પુનરાવર્તન (Image source retry)** — ડાઉનલોડ નિષ્ફળ જાય ત્યારે એજન્ટ હવે AI-જનરેટેડ છબી પર પાછા ફરતા પહેલા તમામ કન્ફિગર કરેલા મફત છબી સ્ત્રોતોને ફરીથી પ્રયાસ કરે છે.
- **મોડેલ માહિતી પેનલ (Model info panel)** — ચેટ હેડરમાં હંમેશા દેખાય છે; હવે પ્રથમ સંદેશ પછી છુપાવવામાં આવતી નથી.
- **ઓટો-સ્ક્રોલ વર્તન (Auto-scroll behaviour)** — વાંચવા માટે વપરાશકર્તા ઉપર સ્ક્રોલ કરે ત્યારે ઓટો-સ્ક્રોલ થોભી જાય છે; એક ફ્લોટિંગ **Scroll to bottom** બટન દેખાય છે અને વપરાશકર્તા સૌથી તાજેતરના સંદેશ સુધી પહોંચે ત્યારે આપમેળે બંધ થઈ જાય છે.
- **એજન્ટ પિકર UI (Agent Picker UI)** — તેને ફોર્મ-ટેબલ ઓવરલે તરીકે ફરીથી ડિઝાઇન કરવામાં આવ્યું છે જેમાં દરેક એજન્ટના આઇકોન હોય છે, જેનાથી એક નજરમાં એજન્ટ્સને ઓળખવા અને સ્વિચ કરવા સરળ બને છે.
- **લેઝી-લોડેડ JS chunks** — ચેટ વિજેટના પ્રારંભિક JavaScript બંડલને હવે લેઝી-લોડેડ chunks માં વિભાજિત કરવામાં આવ્યું છે, જેનાથી પ્રારંભિક બંડલના કદમાં ૭૫–૯૦% ઘટાડો થયો છે.
- **ચેટ વિજેટ રિડિઝાઇન (Chat widget redesign)** — યુનિફાઇડ AI આઇકન અગાઉના કસ્ટમ અવતારને બદલે વપરાયું છે; તે બિલ્ટ-ઇન એજન્ટ સિસ્ટમ સાથે સુસંગત છે.
- **URL લિંકિફિકેશન (URL linkification)** — સિસ્ટમ સંદેશાઓ અને એરર મેસેજ બબલ્સમાં દેખાતા URL ને હવે ક્લિક કરી શકાય તેવી લિંક્સ તરીકે રેન્ડર કરવામાં આવે છે.

### સુધારા (Fixed)

- **ક્ષમતાની શોધક્ષમતા (Ability discoverability)** — વર્ણનો, સિસ્ટમ પ્રોમ્પ્ટ રેફરન્સ અને નેમસ્પેસ એલાઈનમેન્ટ સુધારવામાં આવ્યા છે જેથી તમામ ક્ષમતાઓ એજન્ટની ટૂલ લિસ્ટમાં વિશ્વસનીય રીતે દેખાય.
- **પ્રોવાઇડર્સ કેશ (Providers cache)** — પ્રોવાઇડર્સને હવે વર્ઝન કાઉન્ટર દ્વારા સાઇટ-વાઇડ કેશ કરવામાં આવે છે, જે મલ્ટિસાઇટ નેટવર્ક્સ પર જૂના પ્રોવાઇડરના મુદ્દાઓને અટકાવે છે.
- **`ability_invalid_output`** — ૧૨ એબિલિટી હેન્ડલર્સમાં ઠીક કરવામાં આવ્યું છે; બધા યોગ્ય રીતે સ્ટ્રક્ચર્ડ JSON પ્રતિસાદ આપે છે.
- **`pending_client_tool_calls` પાઇપલાઇન** — એન્ડ-ટુ-એન્ડ વાયર કરવામાં આવી છે જેથી ક્લાયન્ટ-સાઇડ ટૂલ કૉલ્સ યોગ્ય રીતે પૂર્ણ થાય અને પરિણામો મોડેલને પાછા મળે.
- **હિસ્ટરી ડ્રોઅર (History drawer)** — રિવર્ટ ન કરી શકાય તેવા ફેરફારોને રિવર્ટ લિસ્ટમાંથી બાકાત રાખવામાં આવ્યા છે; **View full history** લિંક હવે યોગ્ય રીતે કામ કરે છે.
- **ફેરફાર/રિવર્ટ સિસ્ટમ (Changes/revert system)** — પાંચ અલગ-અલગ bugs ઠીક કરવામાં આવ્યા અને નવા એડમિન ઇન્ટરફેસ હેઠળ એકીકૃત કરવામાં આવ્યા.
- **સેવ સેટિંગ્સ ટોસ્ટ (Save Settings toast)** — **Save Settings** પર ક્લિક કર્યા પછી સ્નેકબાર નોટિફિકેશન હવે વિશ્વસનીય રીતે દેખાય છે.
- **ટ્રેશ કન્ટેક્સ્ટ મેનૂ (Trash context menu)** — **Delete Permanently** વિકલ્પ ઉમેરવામાં આવ્યો જેથી વસ્તુઓને ટ્રેશ વ્યૂ છોડ્યા વિના હાર્ડ-ડીલીટ કરી શકાય.
- **Edit & resend** — એડિટ આઇકન પર ક્લિક કરવાથી હવે માત્ર ક્લિક કરેલા સંદેશ માટે એડિટ મોડમાં પ્રવેશ થાય છે, સમગ્ર થ્રેડના સંદેશાઓ માટે નહીં.
- **ચેટ લેઆઉટ ઊંચાઈ (Chat layout height)** — જ્યારે પ્લગિન-ઇન્જેક્ટેડ કન્ટેન્ટ (એડમિન નોટિસીસ, બેનર્સ) પેજની ઉપર દેખાય છે, ત્યારે ચેટ પેનલ તેની ઊંચાઈને એડજસ્ટ કરે છે, જેનાથી ઇનપુટ એરિયા સ્ક્રીનથી નીચે ન ધકેલાય.

## ૧.૪.૦ — ૨૦૨૬-૦૪-૦૯ ના રોજ રિલીઝ

### નવી સુવિધાઓ (New)

- **WP-CLI બેન્ચમાર્ક કમાન્ડ** (`wp gratis-ai-agent benchmark`) — CI પાઇપલાઇન્સ અને મોડેલ મૂલ્યાંકન વર્કફ્લો માટે કમાન્ડ લાઇનમાંથી એજન્ટ ક્ષમતાઓ v1 બેન્ચમાર્ક સ્યુટ ચલાવે છે. પ્રતિ પ્રશ્ન ચલાવવું, પ્રોવાઇડર/મોડેલ ઓવરરાઇડ્સ અને JSON/CSV આઉટપુટને સપોર્ટ કરે છે.
- **એજન્ટ ક્ષમતાઓ v1 બેન્ચમાર્ક સ્યુટ (Agent Capabilities v1 benchmark suite)** — એક струкચર્ડ સેટ ઓફ complex, multi-step prompts જે સંપૂર્ણ ક્ષમતાની સપાટી (ability surface) નો ઉપયોગ કરે છે, જેમાં સ્કોરિંગ અને ટોકન/સમયની જાણકારી હોય છે.
- **કસ્ટમ પોસ્ટ ટાઇપ ક્ષમતાઓ (Custom Post Type abilities)** — `register_post_type`, `list_post_types`, `delete_post_type`. રજીસ્ટ્રેશન્સ ઓપ્શન્સ ટેબલ દ્વારા રીસ્ટાર્ટ દરમિયાન ટકી રહે છે.
- **કસ્ટમ ટેક્સોનોમી ક્ષમતાઓ (Custom Taxonomy abilities)** — `register_taxonomy`, `list_taxonomies`, `delete_taxonomy`. વૈધાનિક (hierarchical) અને ફ્લેટ ટેક્સોનોમીને વૈકલ્પિક રીરાઇટ સ્લગ્સ સાથે સપોર્ટ કરે છે.
- **ડિઝાઇન સિસ્ટમ ક્ષમતાઓ (Design System abilities)** — `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset`. પાંચ ક્યુરેટેડ પ્રિસેટ્સનો સમાવેશ થાય છે: `minimal-dark`, `warm-editorial`, `corporate-blue`, `vibrant-startup`, `classic-blog`.
- **グローバル સ્ટાઇલ ક્ષમતાઓ (Global Styles abilities)** — `get_global_styles`, `set_global_styles`, `reset_global_styles`. વર્ડપ્રેસ ગ્લોબલ સ્ટાઇલ API દ્વારા theme.json મૂલ્યો વાંચે છે અને લખે છે.
- **નેવિગેશન મેનુ મેનેજમેન્ટ ક્ષમતા (Navigation Menu management ability)** — `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus`. નેસ્ટેડ મેનુ અને થીમ લોકેશન એસાઇનમેન્ટને સપોર્ટ કરે છે.
- **ઓપ્શન્સ મેનેજમેન્ટ ક્ષમતા (Options Management ability)** — `get_option`, `set_option`, `delete_option`, `list_options`. критиકલ વર્ડપ્રેસ ઓપ્શન્સને ફેરફારથી બચાવતું બિલ્ટ-ઇન સેફ્ટી બ્લોકલિસ્ટ (safety blocklist) શામેલ છે.
- **ઇન્સ્ટોલેબલ એબિલિટીઝ રજિસ્ટ્રી (Installable Abilities Registry)** — `list_available_abilities`, `install_ability`, `recommend_plugin`. વર્ડપ્રેસ પ્લગિન્સ તરીકે વિતરિત એબિલિટી પેક્સ શોધો અને એક્ટિવેટ કરો.
- **સાઇટ બિલ્ડર ઓર્કેસ્ટ્રેશન v2 (Site Builder Orchestration v2)** — `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`. પ્લગિન ડિસ્કવરી, સ્ટેપ આઉટપુટ રેફરન્સ, પ્રગતિ ટ્રેકિંગ અને સ્વયંસંચાલિત એરર રિકવરી સાથે બહુ-તબક્કાના સાઇટ બિલ્ડ પ્લાન.
- **રેસ્ટોરન્ટ વેબસાઇટ બેન્ચમાર્ક પ્રશ્ન** (`q-restaurant-website`) — CPT રજીસ્ટ્રેશન, ડિઝાઇન સિસ્ટમ, નેવિગેશન અને પ્લગિન ડિસ્કવરીને આવરી લેતો એન્ડ-ટુ-એન્ડ ટેસ્ટ.
- **AI પ્રોવાઇડર કનેક્ટર પ્લગિન્સ** ઝડપી સ્થાનિક વિકાસ સેટઅપ માટે WordPress Playground blueprints માં ઉમેરવામાં આવ્યા.
