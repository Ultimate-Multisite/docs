# Gratis AI Agent

Gratis AI Agent એક સ્વયંસંચાલિત WordPress AI સહાયક છે જે કુદરતી ભાષાના સૂચનો દ્વારા સંપૂર્ણ WordPress સાઇટ્સનું આયોજન કરી શકે છે, બનાવી શકે છે અને સંચાલન કરી શકે છે. તે **ક્ષમતાઓ (abilities)** નો એક માળખાગત સમૂહ પ્રદર્શિત કરે છે — જે વિશિષ્ટ, પરીક્ષણ કરી શકાય તેવી ક્રિયાઓ છે જે એજન્ટને કૉલ કરી શકે છે — જે કન્ટેન્ટ મેનેજમેન્ટથી લઈને ડિઝાઇન સિસ્ટમ કસ્ટમાઇઝેશન સુધીની દરેક વસ્તુને આવરી લે છે. વર્ઝન 1.9.0 માં કન્ટેન્ટ બનાવવાની ક્ષમતાઓ, ક્લાયન્ટ-સાઇડ સ્ક્રીનશોટ દ્વારા વિઝ્યુઅલ પેજ રિવ્યૂ, પાંચ બિલ્ટ-ઇન એજન્ટ પ્રોફાઇલ્સ, અને એક્સેસ કંટ્રોલ અને બ્રાન્ડિંગ માટે ફીચર ફ્લેગ્સ ઉમેરવામાં આવ્યા છે.

## મુખ્ય વિશેષતાઓ (Key Features)

- **Abilities system** — એક મોડ્યુલર, વિસ્તૃત ક્રિયાઓનો કેટલોગ જે એજન્ટ તમારી WordPress ઇન્સ્ટોલેશન પર કૉલ કરી શકે છે.
- **કન્ટેન્ટ મેનેજમેન્ટ (Content Management)** — પોસ્ટ અને પેજીસ બનાવવી (પેજ ટેમ્પલેટ્સ સોંપવા સહિત), એક જ કૉલમાં બહુવિધ પોસ્ટ્સનું બેચ-ક્રિએશન કરવું, ફીચરલ ઇમેજીસ સેટ કરવી અને કોન્ટેક્ટ ફોર્મ બનાવવું.
- **વિઝ્યુઅલ રિવ્યૂ (Visual Review)** — લાઇવ પેજીસના સ્ક્રીનશોટ કેપ્ચર કરવા, પહેલા/પછીની સ્થિતિની તુલના કરવી, અને લેઆઉટ, ટાઇપોગ્રાફી, કલર અને એક્સેસિબિલિટીને આવરી લેતો AI-જનરેટેડ ડિઝાઇન રિવ્યૂ મેળવવો.
- **પાંચ બિલ્ટ-ઇન એજન્ટ્સ (Five built-in agents)** — કન્ટેન્ટ રાઇટર, સાઇટ બિલ્ડર, ડિઝાઇન સ્ટુડિયો, પ્લગઇન મેનેજર અને સપોર્ટ આસિસ્ટન્ટ; જે ચેટ હેડરના એજન્ટ પિકર દ્વારા બદલી શકાય છે.
- **ફીચર ફ્લેગ્સ (Feature flags)** — રોલ રેસ્ટ્રિક્શન, વ્હાઇટ-લેબલ નામકરણ અને વિજેટ કસ્ટમાઇઝેશન માટે Settings → Feature Flags માં એક્સેસ-કંટ્રોલ અને બ્રાન્ડિંગ ટૉગલ્સ.
- **કસ્ટમ પોસ્ટ ટાઇપ મેનેજમેન્ટ (Custom Post Type management)** — કસ્ટમ પોસ્ટ ટાઇપ્સ રજીસ્ટર કરવા, લિસ્ટ કરવા અને ડિલીટ કરવા, જે રીસ્ટાર્ટ દરમિયાન પણ ટકી રહે છે.
- **કસ્ટમ ટેક્સોનોમી મેનેજમેન્ટ (Custom Taxonomy management)** — કસ્ટમ ટેક્સોનોમીઝ રજીસ્ટર કરવા, લિસ્ટ કરવા અને ડિલીટ કરવા, જે રીસ્ટાર્ટ દરમિયાન પણ ટકી રહે છે.
- **ડિઝાઇન સિસ્ટમ ક્ષમતાઓ (Design System abilities)** — કસ્ટમ CSS ઇન્જેક્ટ કરવું, ક્યુરેટેડ બ્લોક પેટર્નનું સંચાલન કરવું, સાઇટ લોગો સેટ કરવો અને theme.json પ્રિસેટ્સ લાગુ કરવા.
- **ગ્લોબલ સ્ટાઇલ મેનેજમેન્ટ (Global Styles management)** — કલર પેલેટ્સ, ટાઇપોગ્રાફી, સ્પેસિંગ અને વધુ માટે theme.json મૂલ્યો વાંચવા અને લખવા.
- **નેવિગેશન મેનુ મેનેજમેન્ટ (Navigation Menu management)** — નેવિગેશન મેનુ અને તેના આઇટમ્સ પ્રોગ્રામમેટિકલી બનાવવું અને અપડેટ કરવું.
- **ઓપ્શન્સ મેનેજમેન્ટ (Options Management)** — critical સેટિંગ્સનું રક્ષણ કરવા માટે બિલ્ટ-ઇન સેફ્ટી બ્લોકલિસ્ટ સાથે WordPress ઓપ્શન્સ વાંચવા અને લખવા.
- **ઇન્સ્ટોલેબલ ક્ષમતાઓ રજીસ્ટ્રી (Installable Abilities Registry)** — કોર પ્લગઇન ફાઇલોમાં ફેરફાર કર્યા વિના વધારાના ક્ષમતા પેક્સ શોધવા અને એક્ટિવેટ કરવા.
- **સાઇટ બિલ્ડર ઓર્કેસ્ટ્રેશન v2 (Site Builder Orchestration v2)** — પ્લગઇન ડિસ્કવરી, પ્રગતિ ટ્રેકિંગ અને એરર રિકવરી સાથે બહુ-પગલાંવાળા સાઇટ બિલ્ડ પ્લાન.
- **WP-CLI બેન્ચમાર્ક કમાન્ડ (WP-CLI benchmark command)** — CI અને મોડેલ મૂલ્યાંકન માટે કમાન્ડ લાઇનમાંથી એજન્ટ ક્ષમતા બેન્ચમાર્ક સ્યુટ ચલાવવો.

## જરૂરિયાતો (Requirements)

- WordPress 6.0 અથવા તેથી વધુ
- PHP 8.1 અથવા તેથી વધુ
- WordPress Multisite (network-activated) અથવા single-site ઇન્સ્ટોલેશન
- એક સક્રિય AI પ્રદાતા કનેક્ટર પ્લગઇન (Anthropic, OpenAI, અથવા સુસંગત)

## ઇન્સ્ટોલેશન (Installation)

1. પ્લગઇન ફાઇલોને `/wp-content/plugins/gratis-ai-agent` પર અપલોડ કરો.
2. WordPress માં **Plugins** સ્ક્રીન દ્વારા પ્લગઇન એક્ટિવેટ કરો.
3. **Gratis AI Agent → Settings** પર જાઓ અને તમારા AI પ્રદાતાને કનેક્ટ કરો.
4. (વૈકલ્પિક) વધારાના ક્ષમતા પેક્સ ઇન્સ્ટોલ કરવા માટે **Abilities Registry** બ્રાઉઝ કરો.

## ઝડપી શરૂઆત (Quick Start)

એક્ટિવેટ થયા પછી અને AI પ્રદાતા સાથે કનેક્ટ થયા પછી, WordPress એડમિન સાઇબારમાં **Gratis AI Agent** પેનલ ખોલો. કુદરતી ભાષાના સૂચન લખો અને એજન્ટ ઉપલબ્ધ ક્ષમતાઓનો ઉપયોગ કરીને જરૂરી પગલાંનું આયોજન કરશે અને અમલ કરશે.

**ઉદાહરણ સૂચનો:**

```
Create a "Portfolio" custom post type with a "Project Category" taxonomy.
```

```
Inject a dark-mode colour palette into the theme and set the primary colour to #1a1a2e.
```

```
Build a restaurant website with an online menu, booking form, and contact page.
```

## ક્ષમતાઓનો અવલોકન (Abilities Overview)

ક્ષમતાઓ એ પરમાણુ ક્રિયાઓ છે જે એજન્ટ કરી શકે છે. દરેક ક્ષમતા એક રજીસ્ટર્ડ PHP class છે જે એક સ્કીમા (પેરામીટર્સ અને રીટર્ન પ્રકારો) પ્રદર્શિત કરે છે જે એજન્ટ રનટાઇમ પર વાંચે છે. ક્ષમતાઓને કાર્યાત્મક ક્ષેત્રોમાં જૂથબદ્ધ કરવામાં આવી છે:

| Area | Abilities |
|---|---|
| **કન્ટેન્ટ મેનેજમેન્ટ (Content Management)** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **વિઝ્યુઅલ રિવ્યૂ (Visual Review)** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **કસ્ટમ પોસ્ટ ટાઇપ્સ (Custom Post Types)** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **કસ્ટમ ટેક્સોનોમીઝ (Custom Taxonomies)** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **ડિઝાઇન સિસ્ટમ (Design System)** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **ગ્લોબલ સ્ટાઇલ (Global Styles)** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **નેવિગેશન મેનુઝ (Navigation Menus)** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **ઓપ્શન્સ મેનેજમેન્ટ (Options Management)** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **ઇન્સ્ટોલેબલ ક્ષમતાઓ (Installable Abilities)** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **સાઇટ બિલ્ડર (Site Builder)** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

સંપૂર્ણ પેરામીટર ડોક્યુમેન્ટેશન અને ઉપયોગના ઉદાહરણો માટે [Abilities Reference](./abilities) જુઓ.

## WP-CLI

Gratis AI Agent કમાન્ડ લાઇનમાંથી બેન્ચમાર્ક્સ ચલાવવા અને ક્ષમતાનું સંચાલન કરવા માટે `wp gratis-ai-agent` કમાન્ડ ફેમિલી સાથે આવે છે. સંપૂર્ણ કમાન્ડ લિસ્ટ માટે [WP-CLI Reference](./wp-cli) જુઓ.

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## કન્ફિગરેશન (Configuration)

તમામ સેટિંગ્સ WordPress એડમિનમાં **Gratis AI Agent → Settings** હેઠળ છે:

| Setting | Description |
|---|---|
| **AI Provider** | સક્રિય AI પ્રદાતા કનેક્ટર પસંદ કરો અને કન્ફિગર કરો. |
| **Model** | એજન્ટ સેશન્સ માટે ઉપયોગમાં લેવાતી ભાષા મોડેલ પસંદ કરો. |
| **Max Tokens** | પ્રતિ-વિનંતી ટોકન બજેટ (ડિફોલ્ટ 4096). |
| **Ability Timeout** | એક વ્યક્તિગત ક્ષમતા મહત્તમ કેટલા સેકન્ડ સુધી ચાલી શકે છે (ડિફોલ્ટ 30). |
| **Debug Logging** | WordPress debug log માં ક્ષમતા કૉલ્સ અને એજન્ટ તર્ક (reasoning) લોગ કરો. |

## ઓપ્શન્સ મેનેજમેન્ટ સેફ્ટી બ્લોકલિસ્ટ (Options Management Safety Blocklist)

ઓપ્શન્સ મેનેજમેન્ટ ક્ષમતામાં એક બિલ્ટ-ઇન બ્લોકલિસ્ટ શામેલ છે જે એજન્ટને સંવેદનશીલ WordPress ઓપ્શન્સ વાંચવાથી અથવા લખવાથી અટકાવે છે. ડિફોલ્ટ બ્લોકલિસ્ટમાં નીચેનાનો સમાવેશ થાય છે:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (URL ઓપ્શન્સ — આ માટે એક સમર્પિત માઇગ્રેશન ટૂલનો ઉપયોગ કરો)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (ગુપ્ત કીઝ)
- `active_plugins`, `template` (પ્લગઇન/થીમ એક્ટિવેશન — સમર્પિત ક્ષમતાઓનો ઉપયોગ કરો)
- ઇન્સ્ટોલ કરેલા પ્લગઇન દ્વારા રજીસ્ટર કરાયેલ પેમેન્ટ ગેટવે ક્રેડેન્શિયલ્સ અને API કીઝ.

વધારાની એન્ટ્રીઓ `gratis_ai_agent_options_blocklist` ફિલ્ટર દ્વારા ઉમેરી શકાય છે.

## Hooks અને Filters

### Actions

- `gratis_ai_agent_ability_registered` — જ્યારે કોઈ ક્ષમતા રજીસ્ટર થાય ત્યારે ટ્રિગર થાય છે, ક્ષમતા ક્લાસનું નામ પ્રાપ્ત કરે છે.
- `gratis_ai_agent_plan_started` — જ્યારે સાઇટ બિલ્ડર પ્લાન એક્ઝેક્યુશન શરૂ કરે ત્યારે ટ્રિગર થાય છે.
- `gratis_ai_agent_plan_completed` — જ્યારે પ્લાન સફળતાપૂર્વક પૂર્ણ થાય ત્યારે ટ્રિગર થાય છે.
- `gratis_ai_agent_plan_failed` — જ્યારે પ્લાન પુનઃપ્રાપ્ત ન કરી શકાય તેવી ભૂલનો સામનો કરે ત્યારે ટ્રિગર થાય છે.

### Filters

- `gratis_ai_agent_abilities` — એજન્ટ તેને લોડ કરતા પહેલા રજીસ્ટર્ડ ક્ષમતાની યાદીને સુધારે છે.
- `gratis_ai_agent_options_blocklist` — ઓપ્શન્સ સેફ્ટી બ્લોકલિસ્ટને વિસ્તૃત અથવા ઘટાડે છે.
- `gratis_ai_agent_benchmark_questions` — મૂલ્યાંકન સ્યુટમાં કસ્ટમ બેન્ચમાર્ક પ્રશ્નો ઉમેરે છે.
- `gratis_ai_agent_plan_plugins` — સાઇટ પ્લાન જનરેશન દરમિયાન પ્લગઇન ડિસ્કવરી લિસ્ટને સુધારે છે.
