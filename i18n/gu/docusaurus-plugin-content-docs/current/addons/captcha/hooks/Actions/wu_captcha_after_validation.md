# Action: wu_captcha_after_validation

કેપ્ચા માન્યતા (captcha validation) પૂર્ણ થયા પછી આ એક્શન ટ્રિગર થાય છે.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $result | `array` | માન્યતાનું પરિણામ (validation result). |
| $token | `string` | જે ટોકનનું માન્યતા કરવામાં આવ્યું હતું (validated token). |
