# Action: wu_captcha_integrations_init {#action-wucaptchaintegrationsinit}

જ્યારે બધા ઇન્ટિગ્રેશન્સ (integrations) શરૂ થઈ જાય પછી આ એક્શન ટ્રિગર થાય છે.

તૃતીય-પક્ષ (third-party) ઇન્ટિગ્રેશન્સ અહીં હૂક થઈ શકે છે જેથી તેઓ પોતાનું ઉમેરી શકે.
