# Filter: wu_cap_captcha_config {#filter-wucapcaptchaconfig}

Cap Captcha કન્ફિગરેશન ફિલ્ટર કરો.

આ ડેવલપર્સને Cap Captcha ની સેટિંગ્સને કસ્ટમાઇઝ કરવાની મંજૂરી આપે છે.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $config | `array` | Cap કન્ફિગરેશન એરે. |
| $level | `string` | સુરક્ષા સ્તર (fast, medium, max). |
