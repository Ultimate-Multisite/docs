# સપોર્ટ ટિકિટ્સ ફેરફારની નોંધ

### 1.0.4 - 2026-05-05
* સુધારો: `vendor/` ડિરેક્ટરીને Git ટ્રેકિંગમાંથી દૂર કરવામાં આવી (જે પહેલેથી જ .gitignore દ્વારા કવર છે), જેનાથી રિપોઝિટરીનું કદ ઘટ્યું.
* સુધારો: WordPress 7.0 સુધીનું પરીક્ષણ કરવામાં આવ્યું.

### 1.0.3 - 2026-05-01
* સુધારો: નલ (null) મૂલ્યોને નલ કરી શકાય તેવા ટિકિટ મોડેલ સેટર્સમાં સ્વીકાર કરવામાં આવ્યા.
* સુધારો: યોગ્ય મલ્ટીસાઇટ સપોર્ટ માટે તમામ ટિકિટ્સને ગ્લોબલ નેટવર્ક-વ્યાપી ટેબલમાં સ્ટોર કરવામાં આવી.
* સુધારો: સ્ટાફ ફીલ્ડ્સ છુપાવવામાં આવ્યા અને ગ્રાહક નવા ટિકિટ ફોર્મ પર ખાલી પેજ રીડાયરેક્ટની ભૂલ સુધારી.
* સુધારો: મેટાડેટા સ્ટોરેજ માટે `undefined add_meta()` કોલ્સને યોગ્ય `update_meta()` થી બદલવામાં આવ્યા.
* સુધારો: અનરજીસ્ટર્ડ કેપેબિલિટી ચેકને યોગ્ય `wu_view_all_support_tickets` થી બદલવામાં આવ્યો.
* સુધારો: ટિકિટ સ્ટેટસ, એસાઇનમેન્ટ અને ક્વિક-એડિટ એક્શન્સ માટે ખૂટતા AJAX હેન્ડલર્સ ઉમેરવામાં આવ્યા.
* સુધારો: ટિકિટ વ્યૂમાં સ્ટાફ રિસ્પોન્સ ડિટેક્શન માટે મેથડનું નામ સુધારવામાં આવ્યું.
* સુધારો: ડુપ્લિકેટ રિપ્લાય હેન્ડલર્સને એકીકૃત કરવામાં આવ્યા અને નોન્સ એક્શન નામોને સંરેખિત કરવામાં આવ્યા.
* સુધારો: `[wu_submit_ticket]` શોર્ટકોડ માટે ખૂતું ફ્રન્ટએન્ડ વ્યૂ ઉમેરવામાં આવ્યું.
* સુધારો: ખૂલતું `user_id` કૉલમ ઉમેરવામાં આવ્યું અને `Support_Ticket::get_user_id()` મેથડને સુધારવામાં આવી.
* સુધારો: પ્રાયોરિટી ફિલ્ટર ઓપ્શન ટેગમાં વધારાના ડબલ-કોલૂટ કૉમા દૂર કરવામાં આવ્યા.
* સુધારો: સુપર એડમિન માટે નેટવર્ક એડમિન ટિકિટ મેનેજમેન્ટ પેનલ ઉમેરવામાં આવી.
* સુધારો: એડમિન CSS ને એક જ બાહ્ય સ્ટાઇલશીટમાં એકીકૃત કરવામાં આવી.
* સુધારો: સબસાઇટ એડમિન મેનૂમાંથી સેટિંગ્સ સબમેનૂ દૂર કરવામાં આવ્યું.
* સુધારો: ફ્રન્ટએન્ડ એસેટ્સને માત્ર સપોર્ટ ટિકિટ પેજીસ પર કન્ડિશનલી લોડ કરવામાં આવે છે.
* સુધારો: જ્યારે Bedrock root autoloader પહેલેથી જ dependencies લોડ કરી ચૂક્યો હોય ત્યારે પ્લગઇન ઓટોલોડરને સ્કીપ કરવામાં આવે છે.

### 1.0.2 - 2025-12-11
* ઉમેર્યું: એડમિન અને ફ્રન્ટએન્ડ બંને માટે સંપૂર્ણ ટિકિટ જોવાની કાર્યક્ષમતા.
* ઉમેર્યું: ટિકિટ રિપ્લાય સબમિશન માટે AJAX હેન્ડલર.
* ઉમેર્યું: યોગ્ય ફોર્મ હેન્ડલિંગ સાથે ટિકિટ રિપ્લાય કાર્યક્ષમતા માટે સપોર્ટ.
* ઉમેર્યું: એડમિનમાં ટિકિટ સબમિશન અને રિપ્લાય માટે યોગ્ય નોટિસ પ્રદર્શન.
* ઉમેર્યું: ગ્રાહક દ્વારા સ્વ-જમા કરવામાં આવેલી ટિકિટ સાથે આપમેળે વર્તમાન વપરાશકર્તાનું જોડાણ.
* ઉમેર્યું: ગ્રાહકોને ટિકિટ માલિકીને ઓવરરાઇડ કરતા અટકાવવા માટે સુરક્ષા enhancement.
* ઉમેર્યું: ખૂટતા હેલ્પર ફંક્શન્સ (wu_format_date, wu_user_can_view_ticket, વગેરે).
* ઉમેર્યું: યોગ્ય ફાઇલ એટેચમેન્ટ ડાઉનલોડ અને હેન્ડલિંગ.
* ઉમેર્યું: ટિકિટ રિપ્લાય અને સ્ટેટસ ફેરફારો માટે ઇમેઇલ નોટિફિકેશન સિસ્ટમ.
* સુધારેલ: ટિકિટ રિપ્લાય ફોર્મમાં હવે જરૂરી ટિકિટ ID શામેલ છે.
* સુધારેલ: એડમિન ટિકિટ વ્યૂમાં નોટિસની વિઝિબિલિટીની સમસ્યાઓ.
* સુધારેલ: સપોર્ટ ટિકિટ ફંક્શન્સમાં Syntax errors.
* સુધારો: રિસ્પોન્સ અને એટેચમેન્ટ્સ માટે યોગ્ય ક્વેરી અને સ્કીમા ક્લાસ સાથે ડેટાબેઝ સ્ટ્રક્ચર સુધારવામાં આવ્યું.
* સુધારો: ટિકિટ મોડેલો માટે એટ્રિબ્યુટ્સને બદલે વાસ્તવિક પ્રોપર્ટીઝમાં માઇગ્રેશન.
