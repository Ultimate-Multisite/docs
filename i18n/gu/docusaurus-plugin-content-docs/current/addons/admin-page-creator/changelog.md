# Admin Page Creator Changelog

### Version 1.9.0 - Released on 2026-01-18

* સુધારેલ: એક્ટિવેટ (activating) કરતી વખતે થતી ગંભીર ભૂલો (Fatal errors).
* રિફૅક્ટર કરેલ: કન્ટેન્ટ સ્ત્રોત ક્લાસને યોગ્ય પેરેન્ટ કન્સ્ટ્રક્ટર ઇનિશિયલાઇઝેશન સાથે Singleton trait નો ઉપયોગ કરવા માટે.
* સુધારેલ: Vue.js template compilation errors જે કન્ટેન્ટ પ્રકાર પસંદગીકાર (content type selector) ને રેન્ડર થતા અટકાવતા હતા.
* સુધારેલ: ખૂટતી Vue data properties (template_id, external_link_url_embedable, external_link_url_checking).
* સુધારેલ: પેજ લોડ પર TinyMCE એડિટર યોગ્ય રીતે ઇનિશિયલાઇઝ ન થતો હતો.
* સુધારેલ: Divi support file Composer દ્વારા ઓટોલોડ ન થતી હતી.
* સુધારેલ: Divi support class માં PHP સિન્ટેક્સ ભ્રષ્ટ (corrupted) હતી.
* સુધારો: JavaScript asset નું સંગઠન (organization) અને minification સુધારેલ.
* સુધારો: લેટેસ્ટ WordPress અને page builder વર્ઝન સાથે સુસંગતતા (compatibility) સુધારેલ.
