# Action: switch_network {#action-switchnetwork}

જ્યારે વર્તમાન નેટવર્ક સંદર્ભ (context) બદલાય છે ત્યારે આ એક્શન ચાલે છે.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $new_network_id | `int` | જે નેટવર્ક પર સ્વિચ કરવામાં આવી રહ્યું છે તેનો ID. |
| $old_network_id | `int` | પહેલા જે નેટવર્ક હતું તેનો ID. |
