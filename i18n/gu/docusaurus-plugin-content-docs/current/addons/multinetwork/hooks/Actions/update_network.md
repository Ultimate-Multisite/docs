# Action: update_network {#action-updatenetwork}

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $network_id | `int` | ઉમેરેલા નેટવર્કની ID. |
| $args | `array` | નેટવર્ક આર્ગ્યુમેન્ટ્સનો એસોસિએટીવ એરે. |
