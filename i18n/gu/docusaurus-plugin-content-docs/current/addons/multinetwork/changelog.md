# મલ્ટી-નેટવર્ક ફેરફારની નોંધ {#multi-network-changelog}

### 1.3.0 {#130}
* નવું: નેટવર્ક ટેમ્પલેટ પ્રિવ્યૂઅર — ખરીદી કરતા પહેલા લાઇવ પેનલ સાથે નેટવર્ક ટેમ્પલેટ્સ બ્રાઉઝ કરો અને પ્રિવ્યૂ કરો.
* નવું: ચેકઆઉટ પર નેટવર્ક ટેમ્પલેટ પિકર — ગ્રાહકો સાઇનઅપ ચેકઆઉટ ફ્લો દરમિયાન નેટવર્ક ટેમ્પલેટ પસંદ કરી શકે છે.
* નવું: ક્રોસ-નેટવર્ક મેજિક લિંક SSO — ગ્રાહકો મેજિક-લિંક ઓથેન્ટિકેશન દ્વારા સબ-નેટવર્ક્સને સરળતાથી એક્સેસ કરી શકે છે.
* નવું: નેટવર્ક મીડિયા ક્લોનિંગ — ટેમ્પલેટમાંથી નેટવર્ક્સને ક્લોન કરતી વખતે સાઇટ મીડિયા યોગ્ય રીતે ડુપ્લિકેટ થાય છે.
* સુધારો: મલ્ટી-ટેનન્સી એડઓન વિના પણ નેટવર્ક ક્લોનિંગ હવે કામ કરે છે; ક્લોન કર્યા પછી સુપર એડમિન વિશેષાધિકાર ફરીથી આપવામાં આવે છે.
* સુધારો: નેટવર્ક બનાવવું હવે WooCommerce અને તમામ પેમેન્ટ ગેટવે સાથે સુસંગત છે.
* સુધારો: ફોલબેક ડોમેન જનરેશનમાં હવે પોર્ટ નંબરનો સમાવેશ થાય છે.
* સુધારો: ટેમ્પલેટ પિકર CSS ને બધા થીમ્સ પર યોગ્ય રેન્ડરિંગ માટે એન્ક્યુડ સ્ટાઇલશીટમાં ખસેડવામાં આવ્યું છે.
* સુધારો: સાઇટ ક્વેરી સ્કોપમાં અવ્યાખ્યાયિત `network_id` કી સામે રક્ષણ પૂરું પાડવામાં આવ્યું છે.
* સુધારો: કન્ટેક્સ્ટ સ્વિચિંગ પહેલા નેટવર્ક પરમિશન ચેક કરવાથી `wpdb` ટેબલ પ્રોપર્ટીની ભૂલો અટકાવે છે.
