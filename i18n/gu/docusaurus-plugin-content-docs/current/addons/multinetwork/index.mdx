# Multi-Network {#multi-network}

Ultimate Multisite: Multinetwork એડઓન તમારા Ultimate Multisite પ્લેટફોર્મને શક્તિશાળી મલ્ટિનેટવર્ક કાર્યક્ષમતા સાથે વિસ્તારે છે. આ એડઓન તમારા ગ્રાહકોને માત્ર વ્યક્તિગત સાઇટ્સ જ નહીં, પરંતુ સંપૂર્ણ WordPress મલ્ટિસાઇટ નેટવર્ક્સ ખરીદવા અને સંચાલિત કરવાની મંજૂરી આપે છે.

## Key Features {#key-features}

* **Network Provisioning**: તમારા ગ્રાહકો માટે સંપૂર્ણ WordPress મલ્ટિસાઇટ નેટવર્ક્સ આપમેળે બનાવવું અને ગોઠવવું.
* **Network Templates**: ગ્રાહકોને સાઇનઅપ કરતા પહેલા તૈયાર નેટવર્ક ટેમ્પલેટ બ્રાઉઝ કરવા, પ્રિવ્યૂ કરવા અને પસંદ કરવા દો.
* **Network Management**: મલ્ટિનેટવર્ક એન્વાયર્નમેન્ટ્સ માટે સંપૂર્ણ સંચાલન ક્ષમતાઓ.
* **Isolated Networks**: દરેક ગ્રાહક નેટવર્ક તેના પોતાના ડેટાબેઝ ટેબલ્સ અને કન્ફિગરેશન સાથે સ્વતંત્ર રીતે કાર્ય કરે છે.
* **Cross-Network SSO**: મેજિક-લિંક ઓથેન્ટિકેશન સાથે ગ્રાહકોને તેમના સબ-નેટવર્ક્સની સીમલેસ ઍક્સેસ આપો.
* **Scalable Architecture**: શ્રેષ્ઠ સંસાધન ફાળવણી સાથે અનેક નેટવર્ક્સને અસરકારક રીતે સંભાળો.
* **Integration**: Ultimate Multisite ની સબ્સ્ક્રિપ્શન અને બિલિંગ સિસ્ટમ્સ સાથે સીમલેસ રીતે સંકલિત થાય છે.

## Use Cases {#use-cases}

* **White Label Hosting**: ગ્રાહકોને તેમના પોતાના બ્રાન્ડેડ મલ્ટિસાઇટ નેટવર્ક્સ પ્રદાન કરો.
* **Agency Solutions**: એજન્સીઓને એક જ પ્લેટફોર્મમાંથી અનેક ક્લાયન્ટ નેટવર્ક્સનું સંચાલન કરવાની મંજૂરી આપો.
* **Enterprise Deployments**: જટિલ મલ્ટી-નેટવર્ક જરૂરિયાતોવાળી મોટી સંસ્થાઓને ટેકો આપો.
* **Education Platforms**: વિવિધ શાળાઓ અથવા વિભાગો માટે અલગ નેટવર્ક્સ બનાવો.

## Requirements {#requirements}

* Ultimate Multisite 2.0.0 or higher
* WordPress Multisite installation
* PHP 7.4 or higher
* MySQL 5.6 or higher

## Installation {#installation}

1. Upload the plugin files to the `/wp-content/plugins/ultimate-multisite-multinetwork` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure multinetwork settings through Ultimate Multisite → Settings

## Network Templates {#network-templates}

વર્ઝન 1.3.0 નવા નેટવર્ક્સ ખરીદતા ગ્રાહકો માટે ટેમ્પલેટ ડિસ્કવરી અને સિલેક્શન ફ્લો ઉમેરે છે. જ્યારે તમે ઈચ્છો છો કે દરેક નવા ગ્રાહક નેટવર્ક કોઈ તૈયાર માળખું (structure), ડિઝાઇન, કન્ટેન્ટ લાઇબ્રેરી, અથવા વર્ટિકલ-વિશિષ્ટ સેટઅપમાંથી શરૂ થાય, ત્યારે નેટવર્ક ટેમ્પલેટ્સનો ઉપયોગ કરો.

### Template previewer {#template-previewer}

નેટવર્ક ટેમ્પલેટ પ્રિવ્યૂઅર ગ્રાહકોને ચેકઆઉટ કરતા પહેલા ઉપલબ્ધ ટેમ્પલેટ્સ બ્રાઉઝ કરવાની મંજૂરી આપે છે. દરેક ટેમ્પલેટને એક લાઇવ પ્રિવ્યૂ પેનલમાં ખોલી શકાય છે, જેથી ગ્રાહકો નેટવર્ક ખરીદતા પહેલા લેઆઉટ અને કન્ટેન્ટની તુલના કરી શકે.

ગ્રાહકોને મદદ કરવા માટે પ્રિવ્યૂઅરનો ઉપયોગ કરો:

* Review the look and structure of each available network template
* Compare multiple templates without leaving the purchase flow
* Choose the template that best matches their use case before signup

ટેમ્પલેટના નામ અને વર્ણન ગ્રાહક-મૈત્રીપૂર્ણ રાખો. પ્રિવ્યૂઅર ત્યારે સૌથી વધુ ઉપયોગી છે જ્યારે ગ્રાહકો હેતુપૂર્વક ઉપયોગના કેસ (use case)ને સમજી શકે, જેમ કે એજન્સી નેટવર્ક્સ, શાળા નેટવર્ક્સ, ફ્રેન્ચાઇઝ નેટવર્ક્સ, અથવા વ્હાઇટ-લેબલ હોસ્ટિંગ બંડલ્સ.

### Template picker at checkout {#template-picker-at-checkout}

ટેમ્પલેટ પિકકરને સાઇનઅપ ચેકઆઉટ ફ્લોમાં એક ફીલ્ડ તરીકે ઉમેરી શકાય છે. જ્યારે તેને કન્ફિગર કરવામાં આવે છે, ત્યારે ગ્રાહકો ચેકઆઉટ દરમિયાન જે નેટવર્ક ટેમ્પલેટ જોઈએ છે તે પસંદ કરે છે, અને જ્યારે તેમનું નવું નેટવર્ક પ્રોવિઝન કરવામાં આવે છે ત્યારે પસંદ કરેલ ટેમ્પલેટનો ઉપયોગ થાય છે.

ચેકઆઉટ ફ્લોમાં પિકકરનો ઉપયોગ કરવા માટે:

1. Prepare the networks you want to offer as templates.
2. Enable the Multinetwork template selection field in the relevant signup or checkout form.
3. Choose which templates should be available for that checkout flow.
4. Publish the checkout flow and test a signup to confirm the selected template is applied to the new network.

જો કોઈ ચેકઆઉટ ફ્લો માત્ર એક નેટવર્ક પ્રોડક્ટ ઓફર કરે છે, તો પણ પિકકર એક માર્ગદર્શિત ઓનબોર્ડિંગ સ્ટેપ તરીકે ઉપયોગી થઈ શકે છે. મલ્ટી-પ્લાન ફનલ્સ માટે, ફક્ત તે જ ટેમ્પલેટ્સ પ્રદર્શિત કરો જે પસંદ કરેલ પ્રોડક્ટ અથવા પ્લાન સાથે સુસંગત હોય.

## Cross-Network Magic Link SSO {#cross-network-magic-link-sso}

ક્રોસ-નેટવર્ક મેજિક લિંક SSO ગ્રાહકોને તેમના પ્રાથમિક એકાઉન્ટ એરિયા અને સબ-નેટવર્ક્સ વચ્ચે મેન્યુઅલી ફરીથી સાઇન ઇન કર્યા વિના移動 કરવાની મંજૂરી આપે છે. જ્યારે કોઈ ગ્રાહક ક્રોસ-નેટવર્ક એક્સેસ લિંકને અનુસરે છે, ત્યારે Ultimate Multisite એક સમય-મર્યાદિત મેજિક લિંક જનરેટ કરે છે અને ગ્રાહકને ગંતવ્ય નેટવર્કમાં ઓથેન્ટિકેટ કરે છે.

આ એવા ગ્રાહકો માટે અનુભવ સુધારે છે જે એક કરતાં વધુ નેટવર્ક્સનું સંચાલન કરે છે અથવા જેમને ખરીદી કર્યા પછી તરત જ નવા ક્લોન કરેલા ટેમ્પલેટ નેટવર્ક્સની ઍક્સેસની જરૂર હોય છે.

Admin checklist:

* Confirm the Multinetwork addon is active across the WordPress multisite installation.
* Make sure customers have the correct network access before they are offered cross-network links.
* Keep network domains and mapped domains correctly configured so magic links resolve to the intended destination network.
* Test the customer dashboard and any post-checkout access buttons after changing network, domain, or checkout settings.

મેજિક લિંકને લોગિન લિંક તરીકે ગણવી જોઈએ. તે સીમલેસ ગ્રાહક ઍક્સેસ માટે ડિઝાઇન કરવામાં આવી છે અને તેનો ઉપયોગ અથવા તેને ફોરવર્ડ કરવો જોઈએ નહીં જે ઇરાદાપૂર્વના સત્રની બહાર હોય.

## Frequently Asked Questions {#frequently-asked-questions}

### What is multinetwork? {#what-is-multinetwork}

મલ્ટિનેટવર્ક એ એક WordPress ફીચર છે જે તમને એક જ WordPress ઇન્સ્ટોલેશન પર અનેક સ્વતંત્ર મલ્ટિસાઇટ નેટવર્ક્સ ચલાવવાની મંજૂરી આપે છે.

### How is this different from regular multisite? {#how-is-this-different-from-regular-multisite}

નિયમિત મલ્ટિસાઇટ એક નેટવર્કની અંદર અનેક સાઇટ્સની મંજૂરી આપે છે. મલ્ટિનેટવર્ક એક જ WordPress ઇન્સ્ટોલેશનની અંદર, પોતાના સાઇટ્સ સાથે, અનેક નેટવર્ક્સની મંજૂરી આપે છે.

### Does this require special server configuration? {#does-this-require-special-server-configuration}

હા, મલ્ટિનેટવર્ક માટે ચોક્કસ સર્વર સેટઅપ અને કન્ફિગરેશનની જરૂર છે. વિગતવાર સેટઅપ સૂચનાઓ માટે કૃપા કરીને દસ્તાવેજીકરણનો સંદર્ભ લો.
