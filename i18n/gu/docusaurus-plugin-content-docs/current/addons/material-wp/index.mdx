# Material WP

WordPressનું એડમિન ઇન્ટરફેસ ઘણા વર્ષોથી એક જ રહ્યું છે. પણ શું તમે ક્યારેય વિચાર્યું છે કે Material Design વર્ઝનનું એડમિન ઇન્ટરફેસ કેવું દેખાશે?

Material WP એ એક એડમિન થીમ છે જે તમને તે લુક અને ફીલ આપવા માટે વિકસાવવામાં આવી છે, જે તમને તમારા ક્લાયન્ટ્સને વધુ કસ્ટમાઇઝ્ડ Dashboard પ્રદાન કરવામાં મદદ કરે છે.

## વિશેષતાઓ {#features}

- કસ્ટમ CSS સપોર્ટ
- સરળ સ્ટાઇલિંગ સેટિંગ્સ
- મેનુને ફરીથી ગોઠવવાની સુવિધા
- સરળ ઇમ્પોર્ટ/એક્સપોર્ટ

## દસ્તાવેજીકરણ {#documentation}

**Material WP સાથે તમારા એડમિન પેનલને કસ્ટમાઇઝ કરો**

## ઇન્સ્ટોલેશન {#installation}

૧. 'material-wp' ને '/wp-content/plugins/' ડિરેક્ટરીમાં અપલોડ કરો
૨. WordPress માં 'Plugins' મેનુ દ્વારા પ્લગઇનને એક્ટિવેટ કરો

## ઉપયોગ માર્ગદર્શિકા {#usage-guide}

WordPressનું એડમિન ઇન્ટરફેસ ઘણા વર્ષોથી એક જ રહ્યું છે. પણ શું તમે ક્યારેય વિચાર્યું છે કે Material Design વર્ઝનનું એડમિન ઇન્ટરફેસ કેવું દેખાશે?

Material WP એ એક એડમિન થીમ છે જે તમને તે લુક અને ફીલ આપવા માટે વિકસાવવામાં આવી છે, જે તમને તમારા ક્લાયન્ટ્સને વધુ કસ્ટમાઇઝ્ડ Dashboard પ્રદાન કરવામાં મદદ કરે છે. Material WP, Ultimate Multisite ના પ્રીમિયમ પ્લગઇન્સ કોરનો પણ એક ભાગ છે.

**ઇન્સ્ટોલ કર્યા પછી**, તમારા _Plugins_ પેજ પર જાઓ અને તમારા નેટવર્ક પર તેને એક્ટિવેટ કરવા માટે ક્લિક કરો.
