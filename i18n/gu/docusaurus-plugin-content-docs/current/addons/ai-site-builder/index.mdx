# AI Site Builder

ultimate-multisite-ai-site-builder

## જરૂરીયાતો (Requirements) {#requirements}

- WordPress 5.3 અથવા તેનાથી ઉપરનું
- PHP 7.4 અથવા તેનાથી ઉપરનું
- Multisite Ultimate plugin (active)

## ઇન્સ્ટોલેશન (Installation) {#installation}

1. એડન ફાઇલોને તમારા `/wp-content/plugins/` ડિરેક્ટરીમાં અપલોડ કરો.
2. WordPress માં 'Plugins' મેનુ દ્વારા પ્લગઇનને એક્ટિવેટ કરો.
3. નેટવર્ક એડમિનમાં એડન સેટિંગ્સ કન્ફિગર કરો.

## ડેવલપમેન્ટ (Development) {#development}

### સેટઅપ (Setup) {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```


### ટેસ્ટિંગ (Testing) {#testing}

આ એડન એક વ્યાપક ટેસ્ટિંગ ફ્રેમવર્ક સાથે આવે છે:

- **Unit Tests**: PHPUnit-આધારિત ટેસ્ટ્સ સાથે WordPress ટેસ્ટ સ્યુટ.
- **Base Test Class**: સામાન્ય ટેસ્ટ દૃશ્યો માટે મદદરૂપ પદ્ધતિઓ (helper methods) પૂરી પાડે છે.
- **Test Coverage**: કોડ કવરેજ ટ્રેક કરવા માટે કન્ફિગર કરેલ છે.
- **Multisite Testing**: ટેસ્ટ્સ મલ્ટિસાઇટ એન્વાયર્નમેન્ટમાં ચાલે છે.

### કોડ સ્ટાન્ડર્ડ્સ (Code Standards) {#code-standards}

- **WordPress Coding Standards**: PHPCS દ્વારા લાગુ કરવામાં આવે છે.
- **PHP 7.4+ Compatibility**: આધુનિક PHP ફીચર્સને સપોર્ટ કરે છે.
- **Static Analysis**: ટાઇપ ચેકિંગ માટે PHPStan ઇન્ટિગ્રેશન.
- **Code Modernization**: PHP અપગ્રેડ માટે Rector નિયમો.

### બિલ્ડ સિસ્ટમ (Build System) {#build-system}

- **Asset Processing**: CSS/JS ફાઇલો માટે મિનિફિકેશન (Minification).
- **Translation**: POT ફાઇલ જનરેશન.
- **Packaging**: વિતરણ માટે આર્કાઇવ બનાવવું.
- **Development Mode**: ડિબગિંગ માટે અનમિનિફાઇડ એસેટ્સ.

## કન્ફિગરેશન (Configuration) {#configuration}

આ એડન નેટવર્ક એડમિન ઇન્ટરફેસ દ્વારા ઉપલબ્ધ વિવિધ કન્ફિગરેશન વિકલ્પોને સપોર્ટ કરે છે.

## હૂક્સ અને ફિલ્ટર્સ (Hooks and Filters) {#hooks-and-filters}

### એક્શન્સ (Actions) {#actions}

- `ultimate-multisite-ai-site-builder_init` - એડન શરૂ થયા પછી ટ્રિગર થાય છે.
- `ultimate-multisite-ai-site-builder_loaded` - બધા એડન ફાઇલો લોડ થયા પછી ટ્રિગર થાય છે.

### ફિલ્ટર્સ (Filters) {#filters}

- `ultimate-multisite-ai-site-builder_settings` - એડન સેટિંગ્સને મોડિફાય કરવા માટે.
- `ultimate-multisite-ai-site-builder_enabled` - એડનને enable/disable સ્ટેટસને ઓવરરાઇડ કરવા માટે.
