# Content Sync

Multisite Ultimate: Content Sync

Multisite Ultimate માટે આ એડઓન નેટવર્ક એડમિનિસ્ટ્રેટર્સને નેટવર્કના કોઈપણ સાઇટમાંથી અન્ય સાઇટ્સમાં કન્ટેન્ટ sync કરવાની મંજૂરી આપે છે. તે posts, pages, custom post types, અને Elementor templates સહિત તમામ post types ને સપોર્ટ કરે છે. આ એડઓનમાં કન્ટેન્ટ અને ટાર્ગેટ સાઇટ્સ પસંદ કરવા માટે એક multi-step wizard interface છે, જેમાં batch processing અને real-time progress feedback મળે છે.

## વિશેષતાઓ (Features)

* કન્ટેન્ટ સિલેક્શન: થંબનેલ્સ અને post type filtering સાથે વિઝ્યુઅલ કાર્ડ ગ્રીડ
* બધા post types: posts, pages, products, Elementor templates, અને કોઈપણ custom post type sync કરો
* સાઇટ સિલેક્શન: કયા સાઇટ્સમાં કન્ટેન્ટ sync કરવું તે પસંદ કરો, જેમાં source site exclusion અને child site pre-selection નો સમાવેશ થાય છે
* બેચ પ્રોસેસિંગ: progress updates સાથે એક સમયે 10 સાઇટ્સમાં sync કરે છે
* મીડિયા sync: post content માંથી મીડિયા ફાઇલો (images, videos) આપમેળે copy અને remap કરે છે લક્ષ્ય સાઇટ્સ પર
* ટેક્સોનોમી sync: categories, tags, અને custom taxonomies ને સાચવે છે
* Elementor સપોર્ટ: CSS regeneration અને template ID remapping સાથે Elementor templates માટે સંપૂર્ણ સપોર્ટ
* conflict resolution: હાલના કન્ટેન્ટને replace કરવાનો અથવા તેને skip કરવાનો વિકલ્પ
* કેશ મેનેજમેન્ટ: sync પછી લક્ષ્ય સાઇટ્સ પર Elementor cache આપમેળે clear કરે છે

## જરૂરિયાતો (Requirements)

* WordPress 5.3 or higher
* PHP 7.4 or higher
* Multisite Ultimate plugin (active)
* Elementor plugin (optional, required for Elementor template sync)

## ઇન્સ્ટોલેશન (Installation)

1. 'ultimate-multisite-content-sync' ને '/wp-content/plugins/' ડિરેક્ટરીમાં upload કરો.
2. તમારા WordPress Network Admin માં 'Plugins' મેનુ દ્વારા plugin activate કરો.
3. Ultimate Multisite → Settings → Content Sync માં સેટિંગ્સ configure કરો.
4. sync શરૂ કરવા માટે Ultimate Multisite → Content Sync પર જાઓ.
