# મીટર આધારિત પ્લાનનો ફેરફારનો લોગ

Version 1.1.0 - રજૂ કરવામાં આવ્યું: 2026-05-05
- નવું: મલ્ટિસાઇટ સબસાઇટ્સ માટે AI ટોકન બિલિંગ — કસ્ટમર સાઇટ્સ પર AI ટોકનનો ઉપયોગ ટ્રેક કરો અને બિલ કરો, જેમાં કન્ફિગરેબલ પ્રતિ-ટોકન રેટ્સ ઉપલબ્ધ છે.
- નવું: કનેક્ટર અમલ (Connector enforcement) ને ડાયનેમિક લિમિટ ડિસ્કવરી અને રાઇટ-થ્રુ સાથે ફરીથી બનાવવામાં આવ્યું છે, જે તમામ કનેક્ટર્સ પર રીઅલ-ટાઇમ ચોકસાઈ સુનિશ્ચિત કરે છે.
- સુધારો: ડેટાબેઝ ટેબલ અપગ્રેડ્સ હવે BerlinDB સ્કીમા વ્યાખ્યાઓ સાથે યોગ્ય રીતે સંરેખિત છે, જે નવા ઇન્સ્ટોલેશન પર અપગ્રેડ નિષ્ફળતા અટકાવે છે.
- સુધારો: ડેટાબેઝ અપગ્રેડ કોલબેકને યોગ્ય ફોર્મેટમાં રૂપાંતરિત કરવામાં આવ્યા છે, જેનાથી silent અપગ્રેડ નિષ્ફળતાનું નિરાકરણ આવ્યું છે.
- સુધારો: AI Usage Overage Markup ઇનપુટ ફીલ્ડમાં હવે અપૂર્ણાંક મૂલ્યો (Fractional values) સ્વીકારવામાં આવે છે.
- સુધારો: પ્લગઇન લોડ પરના ગંભીર ભૂલો (fatal errors) અને ડબલ-ઇનિશિયલાઇઝેશન સમસ્યાઓનું નિરાકરણ કર્યું.
- સુધારેલ: `check-env` npm સ્ક્રિપ્ટ ઉમેરવામાં આવી છે જેથી ડેવલપર એન્વાયર્નમેન્ટ્સ પ્રથમ રન પર સ્વયં-સંરચિત (self-configure) થઈ શકે.
