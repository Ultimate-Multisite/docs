# Ultimate Multisite દસ્તાવેજીકરણ

Ultimate Multisite documentation માં આપનું સ્વાગત છે. Ultimate Multisite, WordPress Multisite ને એક સંપૂર્ણ વેબસાઇટ એઝ અ સર્વિસ (WaaS) પ્લેટફોર્મમાં પરિવર્તિત કરે છે.

## દસ્તાવેજીકરણ વિભાગો {#documentation-sections}

### [વપરાશકર્તા માર્ગદર્શિકા](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

તમારા WaaS પ્લેટફોર્મને સેટઅપ કરવા, configure કરવા અને manage કરવા માટે તમને જે કંઈની જરૂર છે તે અહીં છે. આમાં ઇન્સ્ટોલેશન, પેમેન્ટ ગેટવેઝ, ડોમેન મેપિંગ, ક્લાયન્ટ મેનેજમેન્ટ અને અન્ય વિષયો આવરી લેવામાં આવ્યા છે.

### [વિકાસકર્તા દસ્તાવેજીકરણ](/developer/hooks) {#developer-documentation}

Ultimate Multisite પર કામ કરતા developers માટેનો તકનીકી સંદર્ભ (Technical reference). આમાં hooks reference, REST API documentation અને addon development guides શામેલ છે.

### [એડઓન્સ](/addons) {#addons}

આ દસ્તાવેજીકરણ એવા પેઇડ addon plugins માટે છે જે Ultimate Multisite ને વધારાની સુવિધાઓ જેમ કે WooCommerce integration, analytics, domain selling, અને અન્ય વસ્તુઓ સાથે વિસ્તૃત કરે છે.

- [GitHub Issues](https://github.com/Ultimate-Multisite/ultimate-multisite/issues) -- બગ્સની જાણ કરવા માટે
