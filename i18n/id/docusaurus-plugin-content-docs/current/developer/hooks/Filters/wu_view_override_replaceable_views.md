# Filter: wu_view_override_replaceable_views {#filter-wuviewoverridereplaceableviews}

Memungkinkan pengembang untuk menambahkan folder tambahan ke daftar yang dapat diganti.

Hati-hati, karena mengizinkan folder tambahan mungkin menyebabkan salinan yang sudah usang dimuat alih-alih versi Ultimate Multisite.

## Parameter {#parameters}

| Nama | Tipe | Deskripsi |
|------|------|-------------|
| $replaceable_views | `array` | Daftar folder yang diizinkan. |

### Sejak {#since}

- 2.0.0
### Sumber {#source}

Didefinisikan di [`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L50) pada baris 50
