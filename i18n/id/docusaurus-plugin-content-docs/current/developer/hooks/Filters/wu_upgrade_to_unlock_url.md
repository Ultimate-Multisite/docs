# Filter: wu_upgrade_to_unlock_url {#filter-wuupgradetounlockurl}

Memungkinkan pengembang untuk mengubah URL yang digunakan untuk membuka kunci setelah melakukan peningkatan (upgrade).

## Parameter {#parameters}

| Nama | Tipe | Deskripsi |
|------|------|-------------|
| $url | `string` | URL peningkatan (upgrade). |
| $args | `array` | Modul dan jenis batasan (limit) yang perlu ditingkatkan (upgrade). |

### Sumber {#source}

Didefinisikan di [`inc/functions/limitations.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/limitations.php#L177) pada baris 177.
