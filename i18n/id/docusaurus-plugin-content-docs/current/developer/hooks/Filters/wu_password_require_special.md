# Filter: wu_password_require_special {#filter-wupasswordrequirespecial}

Filter untuk menentukan apakah karakter khusus harus diwajibkan dalam kata sandi.

## Parameter {#parameters}

| Nama | Tipe | Deskripsi |
|------|------|-------------|
| $require | `bool` | Apakah harus mewajibkan karakter khusus. Default true ketika aturan diberlakukan. |
| $defender_active | `bool` | Apakah Defender Pro Strong Password aktif. |

### Sejak {#since}

- 2.4.0
