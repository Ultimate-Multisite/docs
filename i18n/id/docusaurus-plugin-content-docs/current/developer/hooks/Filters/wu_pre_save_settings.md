# Filter: wu_pre_save_settings {#filter-wupresavesettings}

Memungkinkan pengembang untuk memfilter pengaturan sebelum disimpan oleh Ultimate Multisite.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $settings | `array` | Pengaturan yang akan disimpan. |
| $settings_to_save | `array` | Pengaturan baru yang akan ditambahkan. |
| $saved_settings | `array` | Pengaturan yang saat ini tersimpan. |

### Since {#since}

- 2.0.18
