# Filter: wu_password_require_uppercase

Memfilter apakah kata sandi harus menyertakan huruf kapital.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | Apakah perlu huruf kapital. Default `true` saat aturan diberlakukan. |
| $defender_active | `bool` | Apakah Defender Pro Strong Password aktif. |
