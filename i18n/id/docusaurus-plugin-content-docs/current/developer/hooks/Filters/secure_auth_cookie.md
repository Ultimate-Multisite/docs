# Filter: secure_auth_cookie

Memfilter apakah cookie otentikasi harus hanya dikirim melalui HTTPS.

## Parameter {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $secure | `bool` | Apakah cookie harus hanya dikirim melalui HTTPS. |
| $user_id | `int` | ID Pengguna. |

### Sejak {#since}

- 3.1.0
