# Filter: wu_should_redirect_to_primary_domain

Memungkinkan pengembang untuk membatalkan (short-circuit) pengalihan, sehingga mencegah pengalihan tersebut terjadi.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $should_redirect | `bool` | Menentukan apakah pengalihan harus dilakukan atau tidak. |
