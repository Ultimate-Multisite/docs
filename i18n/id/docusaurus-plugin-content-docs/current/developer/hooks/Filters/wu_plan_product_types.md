# Filter: wu_plan_product_types

Memfilter tipe produk yang harus diperlakukan sebagai paket (plans).

Filter ini memungkinkan *addon* untuk mendaftarkan tipe produk tambahan yang harus dikenali sebagai paket dalam proses validasi dan pemisahan data.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | Array dari tipe produk yang akan diperlakukan sebagai paket. |

### Since {#since}

- 2.3.0
