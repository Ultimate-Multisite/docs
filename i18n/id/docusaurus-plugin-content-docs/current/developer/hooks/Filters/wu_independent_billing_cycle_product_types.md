# Filter: wu_independent_billing_cycle_product_types

Memfilter tipe produk yang memiliki siklus penagihan independen.

Produk dengan siklus penagihan independen tidak dipaksa harus sesuai dengan periode penagihan paket di keranjang belanja. Ini berguna untuk produk seperti registrasi domain yang selalu ditagih tahunan, terlepas dari apakah paketnya bulanan atau tahunan.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | Array tipe produk yang memiliki siklus penagihan independen. |
