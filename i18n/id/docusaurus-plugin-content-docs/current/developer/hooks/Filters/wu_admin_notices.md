# Filter: wu_admin_notices

Memungkinkan pengembang untuk memfilter pemberitahuan admin yang ditambahkan oleh Ultimate Multisite.

## Parameter

| Nama | Tipe | Deskripsi |
|------|------|-------------|
| $notices | `array` | Daftar pemberitahuan untuk panel tertentu itu. |
| $all_notices | `array` | Daftar pemberitahuan yang ditambahkan, dipisahkan berdasarkan panel. |
| $panel | `string` | Panel tempat mengambil pemberitahuan. |
| $filter | `string` | Jika pemberitahuan yang bisa dihilangkan sudah difilter. |
| $dismissed_messages | `array` | Daftar kunci pemberitahuan yang sudah dihilangkan. |

### Sejak

- 2.0.0
### Sumber

Didefinisikan di [`inc/class-admin-notices.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-admin-notices.php#L121) pada baris 121
