# Filter: secure_logged_in_cookie

Mengatur apakah *cookie* login hanya boleh dikirim melalui HTTPS.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $secure_logged_in_cookie | `bool` | Apakah *cookie* login hanya boleh dikirim melalui HTTPS. |
| $user_id | `int` | ID Pengguna. |
| $secure | `bool` | Apakah *cookie* otentikasi hanya boleh dikirim melalui HTTPS. |

### Since {#since}

- 3.1.0
