# Filter: wu_username_from_email

Menyaring nama pengguna pelanggan baru.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $username | `string` | Nama pengguna pelanggan. |
| $email | `string` | Alamat email pelanggan baru. |
| $new_user_args | `array` | Array argumen pengguna baru, mungkin termasuk nama depan dan nama belakang. |
| $suffix | `string` | String tambahan yang ditambahkan ke nama pengguna agar unik. |
