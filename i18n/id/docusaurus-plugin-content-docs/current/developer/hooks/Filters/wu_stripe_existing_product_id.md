# Filter: wu_stripe_existing_product_id

Berfungsi menyaring ID produk yang akan diperiksa. Jika ID ini ditemukan, langganan baru akan menggunakan produk tersebut.

## Parameter

| Name | Type | Description |
|------|------|-------------|
| $product_id | `string` | ID produk Stripe yang akan diperiksa. |
| $name | `string` | Nama produk. |
