# Filter: wu_\{$id\}_form_fields

Memfilter bidang-bidang pada sebuah formulir. Formulir tersebut diidentifikasi berdasarkan ID yang ada di nama filter.

## Parameter

| Name | Type | Description |
|------|------|-------------|
| $fields | `array` | Daftar bidang-bidang dari formulir. |
