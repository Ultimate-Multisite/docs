# Filter: secure_auth_redirect {#filter-secureauthredirect}

Menentukan apakah akan menggunakan pengalihan (redirect) otentikasi yang aman.

## Parameter {#parameters}

| Name | Type | Deskripsi |
|------|------|-------------|
| $secure | `bool` | Menentukan apakah akan menggunakan pengalihan (redirect) otentikasi yang aman. Default false. |

### Sejak {#since}

- 3.1.0
