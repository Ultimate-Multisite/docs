# Action: wu_page_{$this->id}_register_widgets {#action-wupagethis-idregisterwidgets}

Dipicu setelah widget terdaftar untuk halaman ini.

Bagian dinamis dari nama hook, `$this->id`, merujuk pada ID halaman.

## Parameter {#parameters}

| Nama | Tipe | Deskripsi |
|------|------|-------------|
| $id | `string` | ID halaman. |
| $page_hook | `string` | Hook halaman. |
| $page | `object` | Objek halaman. |
