# Aksi: wu_page_{$this->id}_before_render

Memungkinkan pengembang plugin untuk menambahkan konten tambahan sebelum halaman dicetak.

## Parameter

| Nama | Tipe | Deskripsi |
|------|------|-------------|
| $page_id | `string` | ID halaman ini. |
| $page | `object` | Objek halaman. |

### Sejak

- 1.8.2
