# Action: wu_dashboard_{$this->tab}_widgets

Memungkinkan pengembang plugin untuk menambahkan widget ke Panel Dashboard Jaringan.

## Parameter {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $tab | `string` | Tab saat ini. |
| $screen | `\WP_Screen` | Objek layar. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Instance halaman admin Ultimate Multisite. |
