# Action: wu_site_created {#action-wusitecreated}

Tidak dipicu jika diduplikasi dari template.

## Parameter {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $data | `array` | Data objek yang akan disimpan. |
| $site | `\Site` | Instansi objek. |
