# Action: wu_page_before_render

Memungkinkan pengembang plugin untuk menambahkan konten tambahan sebelum halaman ditampilkan.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $page_id | `string` | ID halaman ini. |
| $page | `object` | Objek halaman. |
