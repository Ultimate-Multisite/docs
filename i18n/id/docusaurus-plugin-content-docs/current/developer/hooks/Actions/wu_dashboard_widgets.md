# Action: wu_dashboard_widgets {#action-wudashboardwidgets}

Memungkinkan pengembang plugin untuk menambahkan widget ke Panel Dashboard Jaringan (Network Dashboard Panel).

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $tab | `string` | Tab yang sedang aktif. |
| $screen | `\WP_Screen` | Objek layar (screen object). |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Instance halaman admin Ultimate Multisite. |
