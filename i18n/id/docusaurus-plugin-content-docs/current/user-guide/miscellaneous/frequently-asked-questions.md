# Pertanyaan yang Sering Diajukan

Berikut adalah beberapa **Pertanyaan yang Sering Diajukan** yang kerap muncul dalam permintaan dukungan.

  * **Apa itu Ultimate Multisite?**

Ultimate Multisite adalah plugin WordPress yang dibuat untuk instalasi WordPress multisite, yang mengubah instalasi WordPress Anda menjadi jaringan situs premium – seperti [WordPress.com](https://WordPress.com) – memungkinkan klien membuat situs melalui biaya bulanan, triwulanan, atau tahunan (Anda juga dapat membuat paket Gratis).

  * **Apakah Ultimate Multisite bisa berfungsi tanpa Multisite?**

Tidak. Ultimate Multisite memerlukan WordPress multisite, subdirektori, atau subdomain.

  * **Apa saja persyaratan untuk Ultimate Multisite?**

Ultimate Multisite memerlukan:

**Versi WordPress** : v5.3+ (Disarankan: Versi Stabil Terbaru)  
**Versi PHP** : 7.4.x (Dukungan untuk > 8.0 segera hadir)  
**Versi MySQL** : v5+ (Disarankan: Versi 5.6, jika hosting Anda belum menyediakan versi 8.0)

_Multisite (subdomain atau subdirektori) juga perlu diaktifkan._ [Cara Menginstal WordPress Multisite](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **Apakah Ultimate Multisite bisa berfungsi dengan Multisite menggunakan Subdirektori?**

Ya. Ultimate Multisite mendukung instalasi subdirektori maupun instalasi subdirektori.

  * **Apakah Ultimate Multisite bisa berfungsi dengan Multisite menggunakan Subdomain?**

Ya. Ultimate Multisite mendukung instalasi subdirektori maupun instalasi subdomain.

  * **Bisakah saya menggunakan Ultimate Multisite untuk membatasi Plugin dan Tema yang dapat digunakan pengguna?**

Ya. Dengan Ultimate Multisite Anda dapat memilih plugin dan tema mana saja yang tersedia untuk setiap paket di jaringan Anda.

  * **Bisakah saya menggunakan Ultimate Multisite untuk membatasi jumlah post yang dapat dibuat pengguna di situsnya?**

Ya. Dengan Ultimate Multisite Anda dapat menetapkan kuota post untuk **setiap jenis post yang tersedia** di jaringan Anda. Anda juga dapat membatasi jumlah pengguna yang dapat diundang oleh admin, **jumlah ruang disk,** dan masih banyak lagi.

  * **Apakah Ultimate Multisite mendukung Site Template?**

Ya. Dengan Ultimate Multisite Anda dapat membuat Site Template sebanyak yang Anda inginkan. Site Template bukanlah tema, melainkan situs biasa di jaringan Anda. Anda dapat mengaktifkan plugin, tema, dan menyesuaikan Site Template tersebut sesuai keinginan. Jika pengguna memilih template tersebut, situsnya akan dibuat sebagai salinan dari template itu, dengan semua penyesuaian yang sudah ada.

Anda juga dapat menetapkan template tertentu untuk suatu paket. Dengan begitu, setiap pengguna yang mendaftar di paket tersebut akan memiliki situs yang dibuat sebagai duplikat dari Site Template tersebut.

  * **Apakah Ultimate Multisite terintegrasi dengan WooCommerce?**

Ya!

Jika Anda ingin pengguna jaringan Anda memiliki akses ke WooCommerce, sehingga mereka dapat membuat toko, hal itu bisa dilakukan menggunakan Ultimate Multisite. Anda bahkan dapat membatasi jumlah produk yang dapat dibuat klien Anda di setiap tingkatan paket.

Untuk pembayaran langganan Ultimate Multisite, Anda dapat menggunakan [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) agar klien Ultimate Multisite Anda dapat membayar menggunakan gateway WooCommerce di situs utama Anda. Dengan cara ini, Anda dapat memanfaatkan perpustakaan gateway pembayaran yang luas yang dikembangkan untuk WooCommerce dan menggunakannya untuk menagih klien Ultimate Multisite Anda.

  * **Apakah ada cara untuk memigrasikan langganan dari WPMUDEV Pro Sites?**

Ya, Ultimate Multisite kini menawarkan [alat bantu untuk memigrasikan](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) jaringan Pro Sites Anda.

  * **Mengapa tab "Allowed Plugins" kosong di Halaman Edit Paket saya?**

Ultimate Multisite hanya memungkinkan Anda mengizinkan atau menolak akses ke plugin yang tidak diaktifkan secara network-activated di jaringan Anda. Ketika sebuah plugin diaktifkan secara network-activated, secara alami plugin tersebut akan dimuat secara default di setiap situs di jaringan Anda.

Jika Anda tidak melihat plugin yang terdaftar di tab "Allowed Plugins", periksa apakah plugin yang diinginkan sudah network-active atau belum.

  * **Bisakah saya menggunakan desain Tabel Harga sendiri?**

Ya. Sejak versi 1.4.0, Ultimate Multisite mendukung template override.

  * **Apakah Ultimate Multisite berfungsi dengan WP Engine?**

Ya. Ultimate Multisite selalu didukung oleh WP Engine, tetapi untuk membuat domain mapping berfungsi sebelumnya memerlukan langkah manual tambahan dari admin (menambahkan domain ke panel WP Engine). **Hal ini tidak lagi diperlukan sejak versi 1.6.0.** Ultimate Multisite terintegrasi langsung dengan API WP Engine dan domain akan ditambahkan ke panel secara otomatis setelah pemetaan.

  * **Penyedia hosting mana yang Anda rekomendasikan?**

Ultimate Multisite seharusnya berfungsi dengan penyedia hosting mana pun. Perbedaannya hanya beberapa perusahaan hosting seperti Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane dan [Runcloud.io](http://Runcloud.io) memiliki integrasi yang lebih erat untuk kemampuan domain mapping sehingga langsung berfungsi tanpa konfigurasi tambahan. Untuk hosting lainnya, Anda mungkin perlu mengatur domain mapping secara manual.
