# Bagaimana Cara Mendapatkan Bantuan Pengembangan Kustom dari Codeable?

Terkadang, kita benar-benar membutuhkan kustomisasi yang keren agar alur kerja network kita sesuai dengan keinginan. Jika itu yang kamu cari, kami merekomendasikan [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), tempat di mana kamu bisa menemukan developer berpengalaman yang ahli dalam Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) adalah platform yang menghubungkan pelanggan dengan developer WordPress freelance berkualitas, yang mampu mengerjakan proyek dalam skala apa pun. Mereka memiliki proses seleksi yang ketat untuk memastikan kamu dicocokkan dengan ahli yang benar-benar sesuai dengan tugas yang kamu butuhkan.

### Bagaimana cara membuat proyek di Codeable?

Pertama, kunjungi [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) dan klik tombol **Start a Project**.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Sekarang, untuk mengajukan proyek, klik **Start a Project** di sebelah kiri.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

Di bagian "I need help with", pilih **Customizing**:

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

Di bagian "My", pilih **Plugin**:

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Kemudian klik tombol **Continue**.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Sekarang, saatnya menjelaskan proyek dan kebutuhanmu sebaik mungkin. Usahakan memberikan informasi selengkap mungkin!

<!-- Screenshot unavailable: Codeable project description form -->

Jawab pertanyaan-pertanyaan berikut untuk membantu penentuan harga.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Setelah itu, jika kamu belum punya akun [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), buat akun terlebih dahulu lalu klik tombol **Publish My Project**.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

Setelah kamu mempublikasikan proyek [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), platform akan menghubungkanmu dengan developer yang sesuai dengan keahlian yang dibutuhkan proyekmu.

Proses ini akan membantu menemukan orang yang tepat untuk pekerjaanmu, yang berpengalaman dengan WordPress dan Ultimate Multisite!
