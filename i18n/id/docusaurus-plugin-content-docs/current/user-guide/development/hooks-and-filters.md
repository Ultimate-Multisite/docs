# Hook dan Filter Ultimate Multisite

Ultimate Multisite terdiri dari berbagai fungsi dan kueri database. Plugin ini memiliki hook dan filter yang memungkinkan Anda memodifikasi atau menambahkan kode baru ke fungsi yang sudah ada.

Kami telah membuat daftar hook dan filter ini sebagai referensi untuk membantu pengguna tingkat lanjut menyesuaikan jaringan mereka lebih jauh. Anda dapat mengakses daftar tersebut dari Github dengan mengklik tautan di bawah ini.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
