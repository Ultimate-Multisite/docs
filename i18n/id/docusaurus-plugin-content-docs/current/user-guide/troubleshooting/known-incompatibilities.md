# Ketidakcocokan yang Diketahui dengan Plugin Lain

Setelah menginstal dan mengaktifkan Ultimate Multisite, Anda mungkin menemui masalah akibat ketidakcocokan dengan plugin lain yang belum teratasi.

Kami terus berupaya mengurangi daftar ini di setiap rilis baru, tetapi jika sebuah plugin tercantum di sini, Anda mungkin perlu menonaktifkannya agar Ultimate Multisite Network dapat berfungsi dengan baik.

Berikut adalah daftar plugin tersebut beserta langkah yang perlu Anda lakukan:

**Beberapa plugin WPMU DEV** Beberapa plugin yang _sudah tidak_ dikelola oleh WPMU DEV diketahui menyebabkan proses pendaftaran Ultimate Multisite terhenti di langkah terakhir (pengguna tidak dapat membuat akun). Berikut daftar plugin yang menyebabkan masalah tersebut:

  * Signup Language: _Ultimate Multisite memiliki add-on dengan fungsi yang sama, yaitu Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Fitur ini sudah tersedia di Ultimate Multisite, jadi tidak perlu plugin tambahan._

  * New Blog Templates: _Fitur ini sudah tersedia di Ultimate Multisite, jadi tidak perlu plugin tambahan._

  * Set Password on Multisite Blog Creation: _Fitur ini sudah tersedia di Ultimate Multisite, jadi tidak perlu plugin tambahan._

  * Signup TOS: _Fitur ini sudah tersedia di Ultimate Multisite, jadi tidak perlu plugin tambahan._

  * Anti-Splog: Meskipun tidak selengkap plugin aslinya, fitur ini juga _sudah tersedia di Ultimate Multisite._

**Catatan**: Semua plugin WPMU Dev lainnya dapat digunakan secara normal bersama Ultimate Multisite. Contohnya termasuk _Smush_, _Forminator_, _Defender,_ dan lain-lain.
