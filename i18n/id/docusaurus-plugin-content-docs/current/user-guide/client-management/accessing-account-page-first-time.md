# Bagaimana cara mengakses halaman akun saya untuk pertama kalinya?

Selamat datang di Ultimate Multisite!

Akses halaman akun Anda melalui link yang dikirimkan ke e-mail setelah membeli plugin kami:

<!-- Screenshot unavailable: Account page email with access link -->

Gunakan email yang Anda berikan saat pembelian dan password yang dikirimkan ke email Anda.

<!-- Screenshot unavailable: Login page with email and password fields -->

Karena ini pertama kalinya Anda login ke akun ini, Anda akan diminta untuk mengganti password. Ganti password Anda dan klik tombol _Save_.

<!-- Screenshot unavailable: Change password form with Save button -->

Di halaman akun, Anda dapat mengelola semua detail akun seperti tagihan, perpanjangan, pembatalan, dan berbagai hal lainnya.
