# Website Klien Anda (v2)

Setelah mendaftar di jaringan Anda, pelanggan dapat mengakses dashboard mereka.

Selain halaman **Akun**, di mana mereka bisa melihat informasi penting tentang pembayaran, keanggotaan, domain, batasan paket, dan lainnya, mereka juga memiliki akses ke halaman **Situs Saya**, tempat mereka bisa melihat dan mengelola semua website mereka. Yang perlu mereka lakukan hanyalah mengklik Situs Saya di dashboard mereka.

![Tautan Situs Saya di dashboard pelanggan](/img/admin/customers-list.png)

Mereka akan melihat semua website mereka, dengan tombol **Kelola** tepat di bawah masing-masing website. Jika mereka mengkliknya, mereka akan diarahkan ke **dashboard website tersebut**.

Mereka juga bisa melihat website mana yang menjadi website **utama** untuk paket yang memungkinkan mereka memiliki lebih dari satu website.

![Halaman Situs Saya menampilkan website pelanggan dengan tombol Kelola](/img/admin/customers-list.png)

Jika mereka ingin mengubah website utama, mereka perlu membuka halaman Akun dengan mengklik menu **Akun**.

![Tautan menu Akun di dashboard pelanggan](/img/admin/customers-list.png)

Kemudian, mereka harus mengklik **Ubah Situs Default**.

![Tombol Ubah Situs Default di halaman akun](/img/admin/customers-list.png)

Akan muncul jendela pop-up. Klik pada daftar drop-down dan pilih website mana yang ingin Anda jadikan website utama.

![Pilihan dropdown Ubah Situs Default](/img/admin/customers-list.png)

Lalu, cukup klik **Ubah Situs Default** dan selesai!

![Mengonfirmasi perubahan situs default](/img/admin/customers-list.png)
