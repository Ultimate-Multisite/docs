# Situs Web Klien Anda (v2) {#your-clients-websites-v2}

Setelah mendaftar di jaringan Anda, pelanggan Anda dapat mengakses Dashboard mereka.

Selain halaman **Account**, tempat mereka dapat melihat informasi penting mengenai pembayaran, keanggotaan, domain, batasan paket, dan lainnya, mereka juga memiliki akses ke halaman **Situs Saya**, tempat mereka dapat melihat dan mengelola semua situs web mereka. Yang perlu mereka lakukan hanyalah mengeklik Situs Saya di Dashboard mereka.

![Tautan Situs Saya di Dashboard pelanggan](/img/account-page/my-sites-link.png)

Mereka akan ditampilkan semua situs web mereka, dengan tombol **Kelola** tepat di bawah masing-masing. Jika mereka mengekliknya, mereka akan diarahkan ke **Dashboard situs web spesifik ini**.

Mereka juga dapat melihat situs web mana yang merupakan situs web **utama** mereka untuk paket yang memungkinkan mereka memiliki lebih dari satu situs web.

![Halaman Situs Saya yang menampilkan situs web pelanggan dengan tombol Kelola](/img/account-page/my-sites-page.png)

Berikut tampilan lengkap antarmuka pengeditan situs:

![Antarmuka lengkap pengeditan situs](/img/admin/site-edit-full.png)

Jika mereka ingin mengubah situs web utama mereka, mereka perlu membuka halaman Account dengan mengeklik menu **Account**.

![Tautan menu Account di Dashboard pelanggan](/img/account-page/account-menu.png)

Anda juga dapat melihat dan mengedit detail pelanggan:

![Antarmuka pengeditan pelanggan](/img/admin/customer-edit.png)

Lalu, mereka harus mengeklik **Ubah Situs Default**.

![Tombol Ubah Situs Default di halaman Account](/img/account-page/change-default-site.png)

Jendela pop-up akan muncul. Klik daftar drop-down dan pilih situs web mana yang ingin Anda jadikan situs web utama.

![Pilihan drop-down Ubah Situs Default](/img/account-page/change-default-site-dropdown.png)

Lalu, cukup klik **Ubah Situs Default** dan selesai!

![Mengonfirmasi perubahan situs default](/img/account-page/change-default-site-confirm.png)
