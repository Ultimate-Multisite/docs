# Bagaimana cara membatalkan langganan saya?

Jika Anda memiliki lisensi Ultimate Multisite, Anda dapat membatalkan perpanjangannya kapan saja. Ikuti langkah-langkah berikut:


<!-- Screenshot unavailable: Account page email with access link -->

Gunakan email yang Anda gunakan saat pembelian beserta kata sandi Anda.

<!-- Screenshot unavailable: Login page with email and password fields -->

Ini adalah tampilan awal Halaman Akun Anda:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

Untuk membatalkan langganan, pada menu di sebelah kiri, pilih _Renewing & Billing._

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

Klik tanda panah di sebelah kanan untuk membuka jendela samping. Kemudian, pilih opsi yang akan membatalkan **perpanjangan otomatis**.

Sistem akan menampilkan pesan konfirmasi.

Jika Anda yakin ingin membatalkan langganan, cukup klik tombol _**Cancel Renewals**_. Setelah itu, Anda akan diminta untuk mengisi survei singkat.

Selesai! Langganan Anda tidak akan diperpanjang secara otomatis.

**Anda akan tetap memiliki kunci yang valid sampai masa langganan Anda berakhir**. Jika ingin mengaktifkan kembali langganan, Anda perlu melakukannya secara manual.
