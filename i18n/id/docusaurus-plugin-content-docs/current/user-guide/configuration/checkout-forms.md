# Formulir Checkout

Formulir Checkout adalah cara yang mudah dan fleksibel untuk bereksperimen dengan berbagai pendekatan saat mencoba mengonversi pelanggan baru.

Ultimate Multisite 2.0 menyediakan editor Formulir Checkout yang memungkinkan Anda membuat formulir sebanyak yang Anda inginkan, dengan field yang berbeda, produk yang ditawarkan berbeda, dan sebagainya.

Untuk mengakses fitur ini, buka menu Checkout Forms di sidebar kiri.

![Daftar Formulir Checkout](/img/config/checkout-forms-list.png)

Di halaman ini, Anda dapat melihat semua formulir checkout yang Anda miliki.

Jika Anda ingin membuat formulir baru, cukup klik Add Checkout Form di bagian atas halaman.

Anda dapat memilih salah satu dari tiga opsi berikut sebagai titik awal: single step, multi-step, atau blank. Kemudian, klik Go to the Editor.

![Editor Formulir Checkout](/img/config/checkout-form-editor.png)

Selain itu, Anda dapat mengedit atau menduplikasi formulir yang sudah ada dengan mengklik opsi di bawah namanya. Di sana, Anda juga akan menemukan opsi untuk menyalin shortcode formulir atau menghapus formulir.

![Aksi hover formulir checkout](/img/config/checkout-form-hover-actions.png)

### Mengedit Formulir Checkout

Anda dapat membuat formulir checkout untuk berbagai keperluan. Dalam contoh ini, kita akan membuat formulir pendaftaran.

Setelah masuk ke editor formulir checkout, beri nama formulir Anda (yang hanya akan digunakan untuk referensi internal) dan slug (digunakan untuk membuat shortcode, misalnya).

![Editor Formulir Checkout](/img/config/checkout-form-editor.png)

Formulir terdiri dari step dan field. Anda dapat menambahkan step baru dengan mengklik Add New Checkout Step.

![Tambah Checkout Step Baru](/img/config/checkout-form-add-step.png)

Pada tab pertama jendela modal, isi konten step formulir Anda. Berikan ID, nama, dan deskripsi. Item-item ini sebagian besar digunakan untuk keperluan internal.

![Step formulir checkout](/img/config/checkout-form-step.png)

Selanjutnya, atur visibilitas step. Anda dapat memilih antara "Always show", "Only show for logged in users", atau "Only show for guests".

![Step formulir checkout](/img/config/checkout-form-step.png)

Terakhir, konfigurasikan gaya step. Field-field ini bersifat opsional.

![Step formulir checkout](/img/config/checkout-form-step.png)

Sekarang, saatnya menambahkan field ke step pertama kita. Cukup klik Add New Field dan pilih jenis bagian yang Anda inginkan.

![Step formulir checkout dengan field](/img/config/checkout-form-step.png)

Setiap field memiliki parameter yang berbeda untuk diisi. Untuk entri pertama ini, kita akan memilih field "Username".

![Step formulir checkout](/img/config/checkout-form-step.png)

![Step formulir checkout](/img/config/checkout-form-step.png)

![Step formulir checkout](/img/config/checkout-form-step.png)

Anda dapat menambahkan step dan field sebanyak yang Anda butuhkan. Untuk menampilkan produk Anda agar pelanggan dapat memilih, gunakan field Pricing Table. Jika Anda ingin membiarkan klien memilih template, tambahkan field Template Selection. Dan seterusnya.

_**Catatan:** username, email, password, site title, site URL, order summary, payment, dan tombol submit adalah field wajib untuk membuat formulir checkout._

Saat Anda mengerjakan formulir checkout, Anda selalu dapat menggunakan tombol Preview untuk melihat bagaimana klien Anda akan melihat formulir tersebut. Anda juga dapat beralih antara tampilan sebagai pengguna yang sudah terdaftar atau pengunjung.

![Simpan Formulir Checkout](/img/config/checkout-form-save.png)

![Editor Formulir Checkout](/img/config/checkout-form-editor.png)

Terakhir, di Advanced Options Anda dapat mengonfigurasi pesan untuk halaman "Terima Kasih", menambahkan snippet untuk melacak konversi, menambahkan CSS kustom ke formulir checkout Anda, atau membatasinya untuk negara tertentu.

![Opsi Lanjutan](/img/config/checkout-form-advanced.png)

Anda juga dapat mengaktifkan atau menonaktifkan formulir checkout secara manual dengan mengubah toggle opsi ini di kolom kanan, atau menghapus formulir secara permanen.

![Toggle aktif](/img/config/checkout-form-active.png)

Jangan lupa untuk menyimpan formulir checkout Anda!

![Tombol simpan](/img/config/checkout-form-save.png)

Untuk mendapatkan shortcode formulir Anda, klik Generate Shortcode dan salin hasil yang ditampilkan di jendela modal.

![Tombol simpan dengan shortcode](/img/config/checkout-form-save.png)
