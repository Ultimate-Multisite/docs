# Mengelola Tautan yang Dapat Dibagikan untuk Paket

Ultimate Multisite menyediakan opsi untuk membagikan tautan paket langsung kepada klien Anda. Untuk melakukannya, buka halaman edit salah satu paket Anda dan klik tombol _Klik untuk menyalin Tautan yang Dapat Dibagikan_.

![Halaman edit produk dengan tombol tautan yang dapat dibagikan](/img/config/product-edit-full.png)

_Di Halaman Edit Paket, Anda dapat menemukan tombol untuk membuat tautan yang dapat dibagikan_

Perlu diperhatikan bahwa mekanisme ini menggunakan frekuensi penagihan default untuk membuat tautan. Artinya, jika penagihan default Anda adalah bulanan, misalnya, pada akhirnya Anda akan mendapatkan tautan seperti ini: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Jika Anda ingin tautan yang dapat dibagikan menggunakan frekuensi triwulanan secara default, Anda perlu menambahkan angka 3 di akhir

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Untuk memilih paket _premium_ dengan jadwal tahunan, tambahkan angka 1 lalu kata:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Perlu diingat bahwa Anda tidak bisa memiliki satu tautan saja untuk berbagai frekuensi penagihan. Anda harus membagikan satu tautan khusus untuk setiap frekuensi penagihan yang Anda miliki.

Setelah Anda membagikan tautan kepada pelanggan, mereka akan diarahkan ke langkah pertama dari alur pendaftaran Anda.
