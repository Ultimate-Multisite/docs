# Mengelola Membership

Di Ultimate Multisite, Anda dapat mengelola membership dan mengakses semua langganan di jaringan Anda. Tinjau dan edit detail langganan seperti paket dan produk, jumlah situs, nominal dan frekuensi tagihan, riwayat pembayaran, dan cap waktu.

Untuk mengakses halaman ini, buka tab **Memberships** di menu Ultimate Multisite.

![Halaman daftar membership](/img/admin/memberships-list.png)

Di halaman Memberships, Anda dapat melihat daftar pelanggan beserta produk yang terkait dengan akun mereka. Anda dapat memfilter membership berdasarkan status: aktif, tertunda, ditahan, kedaluwarsa, atau dibatalkan.

Klik pada sebuah membership untuk mengakses halaman tempat Anda dapat mengedit detail membership â€” info tagihan, batas situs atau kunjungan, dan lainnya.

Anda juga dapat menambahkan membership baru yang dikaitkan dengan pengguna yang sudah terdaftar di jaringan Anda. Klik **Add Membership** dan atur detailnya.

![Modal Tambah Membership](/img/admin/membership-add-modal.png)

Kemudian klik **Create Membership**. Membership baru akan muncul di daftar membership.
