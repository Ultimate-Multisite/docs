# Mengelola Membership {#managing-memberships}

Di Ultimate Multisite Anda dapat mengelola membership dan mengakses semua subscription di jaringan Anda. Tinjau dan edit detail subscription seperti plan dan produk, jumlah situs, jumlah dan frekuensi penagihan, riwayat pembayaran, dan timestamp.

Untuk mengakses halaman ini, buka tab **Memberships** di menu Ultimate Multisite.

![Halaman daftar Membership](/img/admin/memberships-list.png)

Di halaman Memberships Anda dapat melihat daftar pelanggan Anda dan produk yang terkait dengan Account mereka. Anda dapat memfilter membership berdasarkan status: aktif, tertunda, ditahan, kedaluwarsa, atau dibatalkan.

Membership gratis diperlakukan sebagai membership **seumur hidup**. Membership ini tidak menerima tanggal kedaluwarsa dan tidak akan kedaluwarsa secara otomatis. Jika plan gratis harus berakhir setelah masa percobaan atau periode tetap, gunakan produk berbayar dengan masa percobaan, batalkan membership secara manual, atau pindahkan pelanggan ke plan lain ketika akses mereka harus berubah.

Klik sebuah membership untuk mengakses halaman tempat Anda dapat mengedit detail membership — info penagihan, batas situs atau kunjungan, dan lainnya.

![Antarmuka edit Membership](/img/admin/membership-edit.png)

Berikut tampilan lengkap halaman edit membership:

![Antarmuka lengkap edit Membership](/img/admin/membership-edit-full.png)

Anda juga dapat menambahkan membership baru yang terkait dengan pengguna yang sebelumnya terdaftar di jaringan Anda. Klik **Add Membership** dan konfigurasikan detailnya.

![Modal Add Membership](/img/admin/membership-add-modal.png)

Lalu klik **Create Membership**. Membership baru akan muncul di daftar membership.
