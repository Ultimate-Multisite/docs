# Gambaran Umum Dashboard

Ultimate Multisite memiliki **Dashboard** khusus di bawah menu Ultimate Multisite, dan juga menambahkan widget ke Network Dashboard. Artikel ini membahas apa yang akan Anda lihat di kedua tempat tersebut.

## Network Dashboard

Ultimate Multisite menyisipkan widget-widget berguna ke dashboard tingkat jaringan yang terdapat di **My Sites → Network Admin → Dashboard**.

![Network Dashboard dengan widget Ultimate Multisite](/img/admin/network-dashboard-full.png)

Berikut fungsi masing-masing widget:

**First Steps:** Muncul pada instalasi baru dan mengingatkan Anda untuk menyelesaikan tugas pengaturan seperti mengonfigurasi metode pembayaran dan membuat akun percobaan.

![Widget First Steps](/img/admin/network-dashboard-first-steps.png)

**Summary:** Melaporkan jumlah pendaftaran dan pendapatan hari itu. MRR (Monthly Recurring Revenue) menunjukkan proyeksi total dari pelanggan dengan keanggotaan yang menghasilkan pendapatan.

![Widget Summary](/img/admin/network-dashboard-summary.png)

**Activity Stream:** Melaporkan berbagai aktivitas di Ultimate Multisite — pendaftaran, pembatalan, pembuatan situs, penghapusan, dan aktivitas lainnya.

![Widget Activity Stream](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** Menampilkan ringkasan singkat pengguna dan situs di jaringan beserta pintasan untuk membuat situs atau pengguna baru.

![Widget Right Now](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** Mengambil dan menampilkan berita terbaru Ultimate Multisite termasuk pembaruan, bug kritis, dan perbaikan keamanan.

![Widget News](/img/admin/network-dashboard-news.png)

## Dashboard Ultimate Multisite

Sementara network dashboard menyajikan informasi menyeluruh tentang jaringan, dashboard Ultimate Multisite (di tingkat teratas menu Ultimate Multisite) menyajikan informasi tentang layanan.

![Dashboard Ultimate Multisite](/img/admin/um-dashboard-top.png)

**Monthly Recurring Revenue Growth:** Menampilkan rincian pendapatan bulanan selama periode 12 bulan kalender. Pendapatan baru ditampilkan dalam warna biru sedangkan pembatalan ditampilkan dalam warna merah.

![Grafik MRR Growth](/img/admin/um-dashboard-mrr.png)

**Date-range Support:** Filter data dashboard berdasarkan periode waktu untuk fokus pada informasi yang Anda butuhkan. Rentang tanggal mengontrol apa yang ditampilkan di widget-widget di bawahnya.

![Rentang tanggal dan statistik dashboard](/img/admin/um-dashboard-date-range.png)

Widget-widget dashboard meliputi:

- **Revenue** — Total pendapatan untuk periode tersebut dengan MRR dan rincian per produk
- **New Memberships** — Berapa banyak keanggotaan baru yang dibuat dalam periode yang dipilih
- **Signups by Countries** — Rincian pendaftaran berdasarkan geografis
- **Signups by Forms** — Berapa banyak pelanggan yang mendaftar melalui masing-masing formulir checkout vs. ditambahkan oleh admin
- **Most Visited Sites** — Situs-situs yang paling banyak dikunjungi di jaringan Anda (memerlukan pengaktifan penghitungan kunjungan di **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**)

![Statistik dashboard](/img/admin/um-dashboard-stats.png)
