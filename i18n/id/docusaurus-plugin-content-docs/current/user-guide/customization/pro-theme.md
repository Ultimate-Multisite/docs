# Menyesuaikan Panel Admin dengan PRO Theme

Saat mengerjakan proyek untuk klien, kita sering kali perlu memberikan pengalaman khusus di dashboard admin WordPress. Jika Anda mencari tampilan yang lebih profesional, PRO Theme bisa menjadi solusinya.

PRO Theme adalah tema admin yang dirancang untuk membantu Anda menyesuaikan Dashboard klien. Plugin ini juga merupakan bagian dari plugin premium inti Ultimate Multisite.

Setelah menginstalnya, buka halaman _Plugins_ dan klik untuk mengaktifkannya di Network Anda.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Aktifkan secara network-wide di halaman Plugins Anda_

_**CATATAN:**_ _Jika Anda hanya menginstalnya tanpa mengaktifkan di network, plugin ini akan muncul sebagai opsi bagi klien Anda yang bisa mereka centang untuk menambahkan tema hanya pada akun pengguna mereka._

Sekarang, tampilan admin Anda sudah sepenuhnya didesain ulang! Anda juga akan melihat menu baru di kolom kiri bernama PRO Theme.
