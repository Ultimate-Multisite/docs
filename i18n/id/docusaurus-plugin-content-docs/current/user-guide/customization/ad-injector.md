# Memasang Iklan di Jaringan Anda dengan Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector adalah **add-on gratis** yang memberikan Anda alat untuk memasang iklan pada paket langganan langsung dari halaman editnya.

Setelah menginstal Ultimate Multisite: Ad Injector, buka halaman _Plugins_ untuk mengaktifkannya.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Klik tombol Network Activate_

Setelah add-on aktif, kotak meta _Advanced Options_ pada halaman edit paket langganan akan memiliki opsi baru bernama _Ad Settings_.

![Ad Settings pada halaman edit paket langganan](/img/config/product-edit-full.png)

_Sekarang Anda dapat mengedit pengaturan iklan untuk setiap paket langganan Anda_

Pada tab tersebut, Anda dapat menambahkan kode iklan untuk pos dan halaman di front-end, sebelum dan sesudah konten.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Setelah memasukkan kode iklan, iklan akan muncul di pos dan halaman dari paket langganan tersebut._

Anda juga dapat menampilkan iklan di header Halaman Admin untuk pengguna paket langganan tersebut.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Buka opsi Admin Panel Ad Code jika Anda ingin menampilkan iklan di back-end pengguna_

### Lebih dari Sekadar Iklan

Anda juga dapat menggunakan Ultimate Multisite: Ad Injector untuk menawarkan dukungan kepada paket langganan tingkat tinggi.

Cukup buka opsi _Admin Panel Ads_ dan tambahkan kode HTML untuk alat dukungan Anda.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Tambahkan dukungan pelanggan ke paket langganan tertentu_

Dengan cara ini, pengguna dapat menghubungi Anda langsung dari Admin Panel mereka!
