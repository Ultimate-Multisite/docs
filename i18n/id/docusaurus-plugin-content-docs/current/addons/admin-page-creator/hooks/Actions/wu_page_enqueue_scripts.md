# Action: wu_page_enqueue_scripts {#action-wupageenqueuescripts}

Menambahkan aksi kustom ini agar kita memiliki tempat untuk memuat (enqueue) skrip dan gaya sesuai kebutuhan, serta memastikan skrip tersebut hanya dimuat di halaman yang tepat.

## Parameter {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $ | `\WU_Page` | Halaman admin saat ini |

### Since {#since}

- 1.3.0
