# Filter: wu_apc_should_display_admin_menu

Memungkinkan admin untuk menyembunyikan halaman menu admin, jika mereka sudah menyelesaikan proses kustomisasi.

## Parameter

| Name | Type | Description |
|------|------|-------------|
| $should_display_admin_menu | `bool` | Jika kita perlu menampilkan halaman tersebut. |
| $network_active | `bool` | Memeriksa apakah plugin aktif di tingkat admin jaringan. |
