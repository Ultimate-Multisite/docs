# Material WP

Antarmuka admin WordPress sudah sangat lama tidak berubah. Tapi, pernahkah Anda bertanya-tanya seperti apa tampilan antarmuka admin dengan desain Material Design?

Material WP adalah tema admin yang dikembangkan untuk memberikan tampilan dan nuansa tersebut, membantu Anda memberikan *Dashboard* yang lebih disesuaikan kepada klien Anda.

## Fitur {#features}

- Dukungan CSS kustom
- Pengaturan gaya yang mudah
- Pengurutan ulang menu
- Impor/ekspor yang mudah

## Dokumentasi {#documentation}

**Menyesuaikan Panel Admin Anda dengan Material WP**

## Instalasi {#installation}

1. Unggah 'material-wp' ke direktori '/wp-content/plugins/'
2. Aktifkan plugin melalui menu 'Plugins' di WordPress

## Panduan Penggunaan {#usage-guide}

Antarmuka admin WordPress sudah sangat lama tidak berubah. Tapi, pernahkah Anda bertanya-tanya seperti apa tampilan antarmuka admin dengan desain Material Design?

Material WP adalah tema admin yang dikembangkan untuk memberikan tampilan dan nuansa tersebut, membantu Anda memberikan *Dashboard* yang lebih disesuaikan kepada klien Anda. Material WP juga merupakan bagian dari inti plugin premium Ultimate Multisite.

Setelah **menginstalnya**, buka halaman _Plugins_ Anda dan klik untuk mengaktifkannya di seluruh Jaringan (Network) Anda.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Material WP -->

_Aktifkan di seluruh jaringan pada halaman Plugins Anda_

_**CATATAN:**_ _Jika Anda hanya mengaktifkannya, itu akan muncul di klien Anda sebagai opsi yang dapat ditandai untuk menambahkan tema hanya ke pengguna tersebut._
