# VAT

Multisite Ultimate: PPN Eropa

Menambahkan fitur dan alat tambahan pada penanganan pajak bawaan Multisite Ultimate untuk mendukung PPN Eropa dengan lebih baik. Secara otomatis menambahkan tarif PPN Uni Eropa ke halaman Tarif Pajak dari

## Instalasi

1. Unggah 'ultimate-multisite-vat' ke direktori '/wp-content/plugins/'
2. Aktifkan plugin melalui menu 'Plugins' di WordPress
3. Buka Ultimate Multisite > Settings > Taxes dan Aktifkan EU Vat.
4. Klik pada Tax Rates dan Perbarui EU VAT Rates di bagian bawah, lalu Simpan Tax Rates.
5. Selesai!
