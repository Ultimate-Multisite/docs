# Pemilih Bahasa

Ultimate Multisite: Pemilih Bahasa di Halaman Daftar

Tambahkan kolom pemilih bahasa ke alur pendaftaran.

## Instalasi {#installation}

1. Unggah 'ultimate-multisite-language-selector' ke direktori '/wp-content/plugins/'
2. Aktifkan plugin melalui menu 'Plugins' di Admin Jaringan WordPress Anda

## Panduan Penggunaan {#usage-guide}

Dengan add-on gratis ini, Anda dapat membiarkan pengguna Anda memilih bahasa apa yang ingin mereka lihat di alur pendaftaran dan di halaman login.

Setelah [mengaktifkan lisensi Anda](../../user-guide/getting-started/installing-ultimate-multisite.md), Anda akan dapat menginstal add-on Ultimate Multisite yang diizinkan oleh lisensi Anda.

Buka halaman pengaturan Ultimate Multisite Anda, lalu di menu sebelah kiri, klik label _Add-ons_.

![Halaman Add-ons](/img/admin/addons-page.png)

_Buka halaman Add-ons untuk mulai menginstal Language Selector_

Di sana, Anda akan menemukan semua add-on yang kami sediakan.

Add-on gratis tersedia untuk semua pelanggan kami, terlepas dari paket langganan yang mereka daftarkan.

Klik tombol _Install it Now!_ di bawah thumbnail add-on. Kemudian aktifkan di halaman Plugins.

![Halaman Add-ons dengan tombol instal](/img/admin/addons-page.png)

_Klik tombol untuk menginstal add-on_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_Aktifkan di halaman Plugins Anda_

Sekarang, buka menu Pengaturan Ultimate Multisite dan masuk ke tab _Network Settings_. Di sana, di bagian bawah halaman, Anda akan menemukan bagian baru bernama _Languages_.

![Halaman Pengaturan](/img/config/settings-general.png)

_Bagian baru muncul!_

Anda dapat membiarkannya kosong untuk menampilkan semua bahasa sebagai pilihan, atau memilih bahasa mana yang ingin Anda sediakan. Bahasa-bahasa tersebut adalah bahasa yang sudah Anda instal dalam pengaturan WordPress Anda.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_Daftar bahasa yang tersedia di instalasi WP ini_

Setelah menyimpan perubahan yang Anda buat, menu pilihan dengan bahasa yang Anda pilih akan ditampilkan di halaman Login maupun di Alur Pendaftaran.

![Pengaturan Login dan Pendaftaran](/img/config/settings-login-registration.png)

_Pemilih Bahasa di Halaman Login_

![Halaman Pendaftaran](/img/config/registration-page.png)

_Pemilih Bahasa di Alur Pendaftaran_

## Mengatur Bahasa Melalui URL {#setting-the-language-via-the-url}

Anda juga dapat mengubah bahasa sebuah halaman dengan melewati parameter locale di URL. Misalnya, jika Anda ingin mengarahkan ke halaman pendaftaran dalam bahasa Italia, tambahkan ?locale=it_IT ke URL tersebut.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Bahasa tersebut harus terinstal di jaringan WordPress Anda agar ini berfungsi.
