# AI Site Builder

ultimate-multisite-ai-site-builder

## Persyaratan {#requirements}

- WordPress 5.3 atau lebih tinggi
- PHP 7.4 atau lebih tinggi
- Plugin Ultimate Multisite (aktif)

## Instalasi {#installation}

1. Unggah file addon ke direktori `/wp-content/plugins/` Anda
2. Aktifkan plugin melalui menu 'Plugins' di WordPress
3. Konfigurasi pengaturan addon di admin jaringan

## Pengembangan {#development}

### Pengaturan Awal {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

```
ultimate-multisite-ai-site-builder/
```

### Pengujian {#testing}

Addon ini mencakup kerangka kerja pengujian yang komprehensif:

- **Unit Tests**: Pengujian berbasis PHPUnit dengan *test suite* WordPress
- **Base Test Class**: Menyediakan metode bantu untuk skenario pengujian umum
- **Test Coverage**: Dikonfigurasi untuk melacak cakupan kode (*code coverage*)
- **Multisite Testing**: Pengujian dijalankan dalam lingkungan multisite

### Standar Kode {#code-standards}

- **WordPress Coding Standards**: Diterapkan melalui PHPCS
- **Kompatibilitas PHP 7.4+**: Mendukung fitur PHP modern
- **Analisis Statis**: Integrasi PHPStan untuk pemeriksaan tipe data (*type checking*)
- **Modernisasi Kode**: Aturan Rector untuk peningkatan PHP

### Sistem Build {#build-system}

- **Pemrosesan Aset**: Minifikasi untuk file CSS/JS
- **Terjemahan**: Pembuatan file POT
- **Pengemasan**: Pembuatan arsip untuk distribusi
- **Mode Pengembangan**: Aset yang belum diminifikasi untuk *debugging*

## Konfigurasi {#configuration}

Addon ini mendukung berbagai opsi konfigurasi yang dapat diakses melalui antarmuka admin jaringan.

## Hooks dan Filter {#hooks-and-filters}

### Actions (Aksi) {#actions}

- `ultimate-multisite-ai-site-builder_init` - Dipicu setelah inisialisasi addon
- `ultimate-multisite-ai-site-builder_loaded` - Dipicu setelah semua file addon dimuat

### Filters (Filter) {#filters}

- `ultimate-multisite-ai-site-builder_settings` - Memodifikasi pengaturan addon
- `ultimate-multisite-ai-site-builder_enabled` - Menimpa status aktif/nonaktif addon
