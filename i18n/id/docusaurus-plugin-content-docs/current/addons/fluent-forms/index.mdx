# Fluent Forms

Add-on ini memungkinkan Anda menambahkan batas (limits) pada Fluent Forms di Multisite Ultimate, baik untuk produk, keanggotaan (memberships), maupun situs.
Fluent Forms menggunakan tabel kustom (custom tables) alih-alih custom post types, sehingga opsi pembatasan (limiting options) biasa tidak berfungsi dengan Fluent Forms.

## Fitur

- Menambahkan batas untuk Fluent Forms atau Conversational Fluent Forms.
- Pelanggan akan menerima pesan ramah yang mendorong mereka untuk meningkatkan paket (plan) mereka ketika mereka mencapai batas.
