# Catatan Perubahan Mailster

### 1.0.0 {#100}
* Rilis awal
* Sinkronisasi pelanggan otomatis saat pendaftaran atau pembayaran
* Penugasan daftar yang spesifik untuk produk
* Opt-in berbasis kotak centang untuk kepatuhan GDPR
* Dukungan *double opt-in*
* Pemetaan bidang yang komprehensif
* Pencatatan (logging) yang detail
