# Agen Dukungan

Mengelola seluruh jaringan sendirian bukanlah tugas yang mudah. Seiring jaringan Anda mulai berkembang, Anda akan melihat peningkatan permintaan dukungan. Akan sangat baik jika Anda mengajak orang-orang yang dapat membantu meringankan beban kerja Anda. Namun, memberikan akses/izin penuh sangat berisiko dan dapat membahayakan jaringan Anda. Di sinilah add-on Agen Dukungan ini berperan.

Add-on ini memungkinkan Anda membuat jenis pengguna WordPress baru yang disebut Agen Dukungan (Support Agent), dan memberi Anda kendali penuh atas izin apa yang dimiliki pengguna tersebut. Memberikan izin semudah menyalakan tombol *toggle*, dan Anda cukup mematikannya jika Anda memutuskan untuk menghapusnya.

## Fitur

- Daftar izin WordPress & Ultimate Multisite yang terperinci
- Mengaktifkan & menonaktifkan akses dengan mudah
- Membuat jumlah akun Agen yang tidak terbatas

## Instalasi

1. Unggah 'ultimate-multisite-support-agents' ke direktori '/wp-content/plugins/'
2. Aktifkan plugin melalui menu 'Plugins' di WordPress
3. Ikuti Wizard langkah demi langkah untuk mengatur plugin
