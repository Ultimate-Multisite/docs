# Filter: networks_user_is_network_admin

Memfilter jaringan tempat pengguna bertindak sebagai administrator.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $ | `array\|bool` | Daftar ID jaringan atau `false` jika pengguna tidak memiliki jaringan. |
| $ | `int` | ID pengguna untuk jaringan yang harus dikembalikan. |
