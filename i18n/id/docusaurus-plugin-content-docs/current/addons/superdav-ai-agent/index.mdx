# Superdav AI Agent

Superdav AI Agent adalah asisten AI WordPress lanjutan yang memperluas Gratis AI Agent dengan kemampuan yang sadar tema, kapabilitas pencarian internet yang ditingkatkan, fitur pembuat situs, dan integrasi otomatisasi opsional. Versi 1.18.0 menambahkan alat baca Google Calendar, pemetaan kontak dan peserta, gerbang persetujuan manusia, deduplikasi pengingat, notifikasi SMS TextBee, penyediaan layanan Superdav terkelola, dan paket pendamping Advanced yang didistribusikan secara terpisah untuk alur kerja pengembang tepercaya.

## Fitur Utama

- **Kemampuan sadar tema** — secara otomatis menyesuaikan panduan dan kapabilitas berdasarkan tema WordPress aktif Anda dan plugin yang terpasang
- **Beberapa penyedia pencarian internet** — pilih antara Brave Search dan Tavily untuk hasil pencarian yang lebih kaya dan kapabilitas riset lanjutan
- **Formulir kontak pembuat situs** — buat formulir kontak yang berfungsi penuh langsung dari antarmuka chat menggunakan plugin formulir yang terpasang
- **Pembuat situs yang ditingkatkan** — rencana pembuatan situs multi-fase yang ditingkatkan dengan penemuan plugin dan pemulihan kesalahan yang lebih baik
- **Kemampuan tema bawaan** — panduan khusus untuk Tema Block, Tema Klasik, Kadence Blocks, dan Kadence Theme
- **Integrasi WooCommerce yang ditingkatkan** — direfaktor untuk menggunakan API native WooCommerce demi keandalan dan kompatibilitas yang lebih baik
- **Deteksi penyedia otomatis** — secara otomatis mendeteksi dan menyegarkan plugin serta tema yang tersedia saat diaktifkan atau dinonaktifkan
- **Pembuatan logo kustom** — hasilkan dan sematkan logo SVG kustom dengan sanitasi otomatis yang aman untuk namespace
- **Validasi kontras palet** — periksa pasangan warna untuk kepatuhan aksesibilitas WCAG sebelum diterapkan ke tema Anda
- **Penemuan berbasis foto** — unggah foto selama wawancara penemuan Theme Builder untuk memengaruhi arah desain
- **Menu hospitality** — hasilkan halaman menu makanan dan minuman terstruktur untuk restoran, kafe, dan bisnis hospitality
- **Pratinjau desktop dan mobile** — pratinjau desain Anda di perangkat desktop dan mobile selama pemilihan arah desain
- **Scraping situs Tingkat 1** — sd-ai-agent/site-scrape kini tersedia secara default untuk menganalisis situs web pesaing dan inspirasi
- **Alat baca Google Calendar** — baca kalender dan acara yang dikonfigurasi agar otomatisasi dapat menalar konteks jadwal mendatang
- **Pemetaan kontak dan peserta** — cocokkan peserta kalender dengan pengguna situs dan kontak yang dikenal sebelum menghasilkan pekerjaan tindak lanjut
- **Gerbang persetujuan manusia** — jeda otomatisasi sensitif untuk ditinjau sebelum mengirim pesan, melanjutkan pekerjaan, atau melakukan perubahan
- **Catatan pengingat dan deduplikasi** — lacak pengingat yang sudah dikirim sehingga eksekusi otomatisasi berulang tidak memberi tahu penerima yang sama dua kali
- **Notifikasi SMS TextBee** — kirim notifikasi pesan teks yang dikonfigurasi melalui TextBee saat penyedia SMS diaktifkan
- **Penyediaan layanan Superdav terkelola** — hubungkan situs yang didukung ke endpoint Superdav yang di-host dengan penyiapan otomatis alih-alih konfigurasi endpoint manual
- **Paket pendamping Advanced** — alat pengembang tepercaya yang didistribusikan secara terpisah untuk alur kerja sistem file, database, WP-CLI, dispatcher REST, pembuat plugin, snapshot git, manajemen pengguna, dan benchmark

## Persyaratan

- WordPress 6.0 atau lebih tinggi
- PHP 8.1 atau lebih tinggi
- WordPress Multisite (diaktifkan di jaringan) atau instalasi situs tunggal
- Plugin konektor penyedia AI aktif (Anthropic, OpenAI, atau kompatibel)
- Gratis AI Agent v1.9.0 atau lebih tinggi (Superdav AI Agent memperluas Gratis AI Agent)

## Instalasi

1. Unggah file plugin ke `/wp-content/plugins/superdav-ai-agent`
2. Aktifkan plugin melalui layar **Plugin** di WordPress
3. Pastikan Gratis AI Agent juga diaktifkan
4. Buka **Gratis AI Agent → Settings** dan konfigurasikan penyedia pencarian internet, koneksi layanan Superdav terkelola, Google Calendar, atau kredensial TextBee sesuai kebutuhan

Paket WordPress.org berisi fitur inti Superdav AI Agent. Paket pendamping Advanced dibangun dan didistribusikan secara terpisah untuk lingkungan tempat alat pengembang tepercaya dipasang dan diotorisasi secara eksplisit.

## Mulai Cepat

Setelah diaktifkan, Superdav AI Agent secara otomatis menyempurnakan instalasi Gratis AI Agent Anda dengan kemampuan sadar tema dan kapabilitas tambahan. Buka panel **Gratis AI Agent** di sidebar admin WordPress dan mulai gunakan fitur baru.

**Contoh instruksi:**

```
Tambahkan formulir kontak ke halaman Kontak.
```

```
Cari web untuk praktik terbaik keamanan WordPress terbaru.
```

```
Buat bagian hero menggunakan pola block dari tema aktif.
```

```
Tinjau acara Google Calendar yang dikonfigurasi untuk besok dan susun draf pengingat untuk peserta.
```

```
Kirim pengingat SMS setelah saya menyetujui pesan.
```

## Kemampuan Sadar Tema

Superdav AI Agent secara otomatis mendeteksi tema aktif dan plugin yang terpasang, lalu memuat kemampuan sadar tema yang sesuai ke dalam konteks asisten. Kemampuan yang tersedia meliputi:

| Kemampuan | Berlaku Untuk |
|---|---|
| **Tema Block** | Tema yang menggunakan arsitektur berbasis block WordPress (Full Site Editing) |
| **Tema Klasik** | Tema WordPress tradisional yang menggunakan template PHP dan editor klasik |
| **Kadence Blocks** | Situs dengan plugin Kadence Blocks aktif |
| **Kadence Theme** | Situs dengan tema Kadence sebagai tema aktif |

Lihat [Kemampuan Sadar Tema](../../user-guide/configuration/theme-aware-skills) untuk panduan terperinci tentang setiap kemampuan.

## Penyedia Pencarian Internet

Superdav AI Agent v1.10.0 menambahkan **Tavily** sebagai alternatif untuk Brave Search bagi kueri pencarian internet. Kedua penyedia menawarkan hasil pencarian real-time dan dapat dikonfigurasi di **Gratis AI Agent → Settings → Advanced**.

| Penyedia | Fitur | Penyiapan |
|---|---|---|
| **Brave Search** | Hasil pencarian cepat yang berfokus pada privasi | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Hasil pencarian yang lebih kaya, kemampuan riset lanjutan | [Tavily API](https://tavily.com/) |

Lihat [Pencarian Internet](../../user-guide/configuration/internet-search) untuk instruksi konfigurasi.

## Alur Kerja Kalender, Persetujuan, dan Notifikasi

Superdav AI Agent v1.18.0 memperluas alur kerja otomatisasi melampaui tugas konten dan pembuatan situs:

- **Alat Google Calendar** memungkinkan agen membaca kalender yang dikonfigurasi dan detail acara saat kredensial kalender tersedia.
- **Pembantu kontak dan peserta** memetakan peserta acara ke pengguna WordPress atau kontak yang dikenal sebelum menyiapkan pengingat atau tugas tindak lanjut.
- **Gerbang persetujuan manusia** memungkinkan otomatisasi dijeda hingga pengguna berwenang meninjau tindakan yang diusulkan.
- **Catatan pengingat** mencegah pengingat kalender atau notifikasi duplikat saat otomatisasi dicoba ulang atau dijadwalkan berulang kali.
- **Dukungan SMS TextBee** memungkinkan notifikasi pesan teks saat penyedia TextBee dikonfigurasi.

Gunakan alat ini untuk bantuan yang sadar jadwal, tindak lanjut peserta, dan alur kerja pengingat yang tetap memerlukan tinjauan manusia sebelum pesan sensitif dikirim.

## Konfigurasi

Semua pengaturan berada di bawah **Gratis AI Agent → Settings** di admin WordPress. Superdav AI Agent menambahkan opsi konfigurasi berikut:

| Pengaturan | Deskripsi |
|---|---|
| **Internet Search Provider** | Pilih antara Brave Search dan Tavily (opsional) |
| **Tavily API Key** | API key untuk pencarian internet Tavily (jika Tavily dipilih) |
| **Managed Superdav Service** | Hubungkan situs yang didukung ke layanan Superdav yang di-host dengan penyediaan endpoint otomatis |
| **Google Calendar Credentials** | Konfigurasikan akses kalender agar agen dapat membaca kalender dan acara |
| **TextBee SMS Provider** | Konfigurasikan kredensial TextBee untuk alur kerja notifikasi SMS yang disetujui |

## Paket Pendamping Advanced

Superdav AI Agent v1.18.0 menambahkan paket pendamping Advanced yang didistribusikan secara terpisah untuk alur kerja developer tepercaya. Kemampuan ini tidak dibundel dalam paket inti WordPress.org. Pasang paket pendamping hanya pada situs tempat administrator secara eksplisit memercayai agen untuk mengakses tooling developer.

Alat Advanced mencakup operasi filesystem, akses database, WP-CLI, alur kerja dispatcher REST, utilitas pembuat plugin, snapshot git, pembantu manajemen pengguna, dan tooling benchmark. Batasi alat ini hanya untuk administrator tepercaya dan tinjau konfigurasi gerbang persetujuan sebelum mengaktifkannya di situs produksi.
