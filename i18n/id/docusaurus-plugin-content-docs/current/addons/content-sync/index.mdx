# Sinkronisasi Konten {#content-sync}

Multisite Ultimate: Content Sync

Addon untuk Multisite Ultimate ini memungkinkan administrator jaringan untuk menyinkronkan konten dari situs mana pun ke situs lain dalam jaringan. Addon ini mendukung semua jenis postingan, termasuk postingan, halaman, custom post types, dan template Elementor. Addon ini mencakup antarmuka wizard multi-langkah untuk memilih konten dan situs target, dengan pemrosesan batch dan umpan balik progres waktu nyata.

## Fitur {#features}

* Pemilihan Konten: Grid kartu visual dengan thumbnail dan filter jenis postingan
* Semua Jenis Postingan: Sinkronisasi postingan, halaman, produk, template Elementor, dan jenis postingan kustom apa pun
* Pemilihan Situs: Pilih situs mana yang akan disinkronkan kontennya, dengan pengecualian situs sumber dan pra-seleksi situs anak
* Pemrosesan Batch: Menyinkronkan ke 10 situs sekaligus dengan pembaruan progres
* Sinkronisasi Media: Secara otomatis menyalin dan memetakan ulang file media (gambar, video) dari konten postingan ke situs target
* Sinkronisasi Taksonomi: Mempertahankan kategori, tag, dan taksonomi kustom
* Dukungan Elementor: Dukungan penuh untuk template Elementor dengan regenerasi CSS dan pemetaan ulang ID template
* Penyelesaian Konflik: Opsi untuk mengganti konten yang sudah ada atau melewatinya
* Manajemen Cache: Secara otomatis menghapus cache Elementor di situs target setelah sinkronisasi

## Persyaratan {#requirements}

* WordPress 5.3 atau lebih tinggi
* PHP 7.4 atau lebih tinggi
* Plugin Multisite Ultimate (aktif)
* Plugin Elementor (opsional, wajib untuk sinkronisasi template Elementor)

## Instalasi {#installation}

1. Unggah 'ultimate-multisite-content-sync' ke direktori '/wp-content/plugins/'
2. Aktifkan plugin melalui menu 'Plugins' di Admin Jaringan WordPress Anda
3. Konfigurasikan pengaturan di Ultimate Multisite → Settings → Content Sync
4. Navigasi ke Ultimate Multisite → Content Sync untuk memulai sinkronisasi
