# Plugin & Theme Manager

Multisite Ultimate: Plugin dan Theme Manager

Edit atau sembunyikan informasi meta (judul, penulis, thumbnail, dan deskripsi) dari plugin dan tema yang terpasang. Buat kategori untuk plugin dan tema agar pengguna Anda dapat memfilternya dan menambahkan halaman Plugins kustom yang indah di panel pengguna Anda.

Untuk menggunakan plugin ini, Anda cukup masuk ke halaman Plugins atau Themes di Admin Jaringan Anda dan klik pada "Edit Plugin/Theme info".

## Instalasi {#installation}

1. Unggah 'ultimate-multisite-plugin-and-theme-manager' ke direktori '/wp-content/plugins/'
2. Aktifkan plugin melalui menu 'Plugins' di Admin Jaringan WordPress Anda

## Panduan Penggunaan {#usage-guide}

Ultimate Multisite: Plugin dan Theme Manager adalah add-on Ultimate Multisite **premium** yang kuat yang memberi Anda kendali penuh atas cara pengguna jaringan Anda melihat dan berinteraksi dengan opsi plugin dan tema di jaringan Anda.

Add-on ini memiliki tiga fungsi:

  1. Memungkinkan Anda mengubah metadata plugin dan tema secara keseluruhan. Ini termasuk nama, deskripsi, nama penulis, dan bahkan thumbnail!

  2. Memungkinkan Anda mengkategorikan plugin dan tema, agar pengguna Anda dapat menyortirnya dengan mudah.

  3. Memberikan opsi untuk mengganti halaman plugins lama yang membosankan dengan halaman seperti tema yang baru dengan penyortiran berdasarkan kategori.

Setelah mengunduh dan menginstalnya, tab baru - _Add-on Settings_ - akan muncul di halaman Pengaturan Ultimate Multisite.
