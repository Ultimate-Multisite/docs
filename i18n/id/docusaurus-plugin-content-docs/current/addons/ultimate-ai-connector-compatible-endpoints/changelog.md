# Catatan Perubahan Ultimate AI Connector untuk Endpoint yang Kompatibel

## Versi 2.0.0 — Dirilis pada 2026-04-24

* Baru: Dukungan multi-penyedia — konfigurasikan banyak endpoint AI dan arahkan permintaan dengan pemulihan otomatis di seluruh penyedia.
* Perbaikan: Integrasi SDK multi-penyedia dengan ID penyedia, URL pendaftaran, dan cakupan filter HTTP yang benar per penyedia.
* Perbaikan: Kartu penyedia baru kini mengembang otomatis saat ditambahkan; pemecahan cache skrip saat pembaruan plugin.
* Perbaikan: Ruang nama kelas penyedia dinamis untuk eval() dan perilaku pengembangan otomatis untuk penyedia yang baru ditambahkan.
* Perbaikan: Menggunakan komponen Card/CardBody/CardHeader/CardDivider yang stabil (tidak lagi eksperimental) untuk kompatibilitas WordPress 6.9+.
* Perbaikan: Mengganti DragHandle yang tidak terdefinisi dengan ikon pegangan unicode untuk penyusunan ulang penyedia dengan menyeret.
* Perbaikan: Menghilangkan permintaan HTTP yang memblokir yang terpicu pada setiap pemuatan halaman.
* Peningkatan: Alur kerja GitHub Actions ditingkatkan ke Node.js 24.
