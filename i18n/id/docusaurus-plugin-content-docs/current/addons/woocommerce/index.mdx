# Integrasi WooCommerce

Integrasi WooCommerce berfungsi dengan cara yang sangat mirip dengan cara *Manual Gateway* kami. Integrasi ini menawarkan fleksibilitas yang sangat besar bagi pengguna Anda, memungkinkan mereka memilih bentuk pembayaran apa pun yang tersedia di instalasi WooCommerce Anda. Dan ini sederhana: setelah masa penagihan pengguna berakhir, langganan pengguna akan ditangguhkan (*put on hold*). Pada saat yang sama, pesanan akan dibuat di instalasi WooCommerce pada situs utama jaringan (*network*). Pengguna kemudian menerima faktur di email mereka, lengkap dengan tautan untuk melakukan pembayaran. Setelah pembayaran diterima, langganan akan diperbarui (*renewed*).

## Fitur

- Integrasi yang mudah antara Ultimate Multisite dan WooCommerce
- Beberapa metode pembayaran tersedia
- Proses yang sepenuhnya otomatis dengan WooCommerce Subscriptions

## Dokumentasi

**Pengaturan Integrasi WooCommerce**

**Memperluas Opsi Pembayaran Anda dengan Ultimate Multisite: Integrasi WooCommerce**

## Instalasi

1. Unggah 'ultimate-multisite-woocommerce' ke direktori '/wp-content/plugins/'
2. Aktifkan plugin melalui menu 'Plugins' di WordPress
3. Ikuti Wizard langkah demi langkah untuk mengatur plugin

## Panduan Penggunaan

_**CATATAN PENTING: Artikel ini merujuk pada Ultimate Multisite versi 2.x.**_

_**PERHATIAN:** Ultimate Multisite: WooCommerce Integration memerlukan WooCommerce untuk diaktifkan setidaknya di situs utama Anda._

Kami memahami bahwa _Stripe_ dan _PayPal_ tidak tersedia di beberapa negara, yang membatasi atau menghambat pengguna Ultimate Multisite untuk menggunakan plugin kami secara efektif. Jadi, kami membuat **add-on untuk mengintegrasikan _WooCommerce_**.

_WooCommerce_ adalah plugin *eCommerce* yang sangat populer. Pengembang di seluruh dunia telah membuat add-on untuk mengintegrasikan berbagai *payment gateway* ke dalamnya. Kami memanfaatkan ini untuk memperluas *payment gateway* yang dapat Anda gunakan dalam sistem penagihan Ultimate Multisite.

## Pengaturan

Pertama, Anda perlu menginstal add-on Integrasi WooCommerce yang **gratis untuk pengguna berlisensi**. Untuk menginstalnya, klik Ultimate Multisite di *dashboard* super admin Anda dan pilih **Add-ons**.

![Add-ons page](/img/admin/addons-page.png)

Di sini Anda dapat menemukan semua add-on Ultimate Multisite. Klik pada add-on **Ultimate Multisite: WooCommerce Integration**.

![Add-ons page showing WooCommerce Integration](/img/admin/addons-page.png)

Sebuah jendela akan muncul dengan detail add-on. Cukup klik **Install Now** *.*

<!-- Screenshot unavailable: WooCommerce Integration addon install popup with Install Now button -->

Setelah instalasi selesai, Anda akan diarahkan ke halaman *plugins*. Di sini, cukup klik **Network Activate** dan add-on _WooCommerce_ akan diaktifkan di jaringan Anda.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for WooCommerce Integration -->

Setelah diaktifkan, jika Anda masih belum menginstal dan mengaktifkan plugin WooCommerce di situs web Anda, **Anda akan menerima pengingat**.

<!-- Screenshot unavailable: Admin notice reminding to install and activate WooCommerce plugin -->

Setelah terinstal, Anda akan melihat opsi tambahan di pengaturan pembayaran Ultimate Multisite. Anda kemudian dapat **mengaktifkan** opsi _WooCommerce_ dari sana.

![Payment gateway settings](/img/config/settings-payment-gateways.png)

Anda dapat mengonfigurasi beberapa pengaturan dasar seperti nama tampilan yang akan muncul di *front end* dan petunjuk untuk memandu pengguna Anda dalam alur *checkout* mereka. Kemudian Anda dapat pergi ke **pengaturan _WooCommerce_ Situs Utama** Anda untuk mengaktifkan dan mengonfigurasi *payment gateway* yang Anda sukai.

## Cara Mengelola Pembayaran

Integrasi WooCommerce berfungsi dengan cara yang sangat mirip dengan cara **Manual Gateway** kami. Kekurangannya adalah setiap pembayaran harus dibayar secara manual di akhir periode penagihan. Berikut adalah cara kerjanya:

  * Setelah siklus penagihan pengguna berakhir, keanggotaan akan **ditangguhkan** (*put on hold*). Pada saat yang sama, pesanan akan dibuat di instalasi _WooCommerce_ pada situs utama jaringan Anda.

  * Pengguna kemudian menerima faktur di email mereka, lengkap dengan tautan untuk melakukan pembayaran (tautan tersebut juga muncul di riwayat penagihan mereka dari *dashboard* subsite). Tautan tersebut mengarahkan pengguna ke **checkout _WooCommerce_**, di mana pengguna akan dapat memilih salah satu **gateway _WooCommerce_ yang tersedia** untuk memproses pembayaran mereka.

  * Setelah pembayaran diterima, keanggotaan akan **diperbarui** (*renewed*), pembayaran dicatat di Ultimate Multisite, dan pesanan diatur untuk selesai.

### Beberapa Catatan Penting:

  * Fungsionalitas *default* Ultimate Multisite masih berfungsi dengan formulir integrasi ini. Misalnya, **menerbitkan pengembalian dana** dari layar **Manajemen Langganan** akan berkomunikasi dengan WooCommerce untuk menerbitkan pengembalian dana menggunakan *payment form* yang dipilih.

  * Ultimate Multisite: WooCommerce Integration bergantung pada WP-Cron untuk bekerja. Ini berfungsi dengan sangat baik sebagian besar waktu, tetapi jika situs memiliki lalu lintas rendah, maka *cron* tidak dipicu untuk memeriksa acara terjadwal dan acara tersebut terlewat, yang dapat menyebabkan penundaan eksekusi.

## Plugin Woocommerce Subscriptions

Memulai pembayaran secara manual setiap bulan tidak ideal bagi pengguna akhir karena langganan diharapkan untuk diperbarui secara otomatis setiap siklus penagihan. Jadi, kami merancang add-on Integrasi WooCommerce untuk **mendukung plugin WooCommerce Subscriptions**. Ini akan mengotomatiskan pembayaran sehingga transaksi menjadi lebih mudah bagi pengguna akhir Anda.

Yang perlu Anda lakukan hanyalah **menginstal** [plugin Woocommerce Subscriptions](https://woocommerce.com/products/woocommerce-subscriptions/) dan **mengaktifkannya di situs utama Anda**. Kemudian kembali ke pengaturan pembayaran Ultimate Multisite dan **mengaktifkan integrasi**.

Dan itu saja. Pengguna akhir Anda tidak perlu lagi melakukan pembayaran secara manual di akhir siklus penagihan mereka.

Selain pesanan yang dibuat di situs utama Anda di bawah _WooCommerce_, ini juga akan membuat langganan untuk setiap akun.
