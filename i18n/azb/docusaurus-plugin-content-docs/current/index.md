# Ultimate Multisite Documentation

Ultimate Multisite documentation-ağına hoş geldiniz. Ultimate Multisite, WordPress Multisite'ı tam teşekküllü bir Website as a Service (WaaS) platformuna dönüştürür.

## Documentation Sections

### [User Guide](/user-guide/getting-started/ultimate-multisite-101)

WaaS platformunuzu kurmanız, yapılandırmanız ve yönetmeniz için ihtiyacınız olan her şey burada. Kurulum, ödeme ağ geçitleri, domain eşleştirme, müşteri yönetimi ve daha fazlasını kapsar.

### [Developer Documentation](/developer/hooks)

Ultimate Multisite üzerinde geliştirme yapacak geliştiriciler için teknik referans. Hook referansı, REST API dokümantasyonu ve addon geliştirme rehberleri içerir.

### [Addons](/addons)

Ultimate Multisite'ı WooCommerce entegrasyonu, analizler, domain satışı gibi ek özelliklerle genişleten ücretli addon plugin'leri için dokümantasyon.
