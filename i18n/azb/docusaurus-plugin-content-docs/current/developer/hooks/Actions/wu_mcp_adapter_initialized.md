# Action: wu_mcp_adapter_initialized

Nagti-trigger ito pagkatapos ma-initialize ang MCP adapter.

Pinapayagan nito ang ibang plugins at themes na magrehistro ng sarili nilang kakayahan.

## Mga Parameter

| Name | Type | Description |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | Ang instance ng MCP adapter. |

### Mula pa noong

- 2.5.0
### Pinagmulan

Nakatakda sa [`inc/class-mcp-adapter.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-mcp-adapter.php#L98) sa linya 98
