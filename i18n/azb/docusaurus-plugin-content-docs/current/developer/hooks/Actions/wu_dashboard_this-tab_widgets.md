| Name | Type | Description |
|------|------|-------------|

| $screen | `\WP_Screen` | Ekran nesnesi. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Ultimate Multisite admin sayfa örneği. |
