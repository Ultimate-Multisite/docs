# Action: wu_dashboard_widgets

Plugin geliştiricilerinin Network Dashboard Panel'e widget eklemesine izin verir.

## Parametreler

| Ad | Tip | Açıklama |
|------|------|-------------|
| $tab | `string` | Şu anki sekme. |
| $screen | `\WP_Screen` | Ekran nesnesi. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Ultimate Multisite admin sayfası örneği. |

### Versiyon

- 2.0.0
### Kaynak

[`inc/admin-pages/class-dashboard-admin-page.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/admin-pages/class-dashboard-admin-page.php#L225) dosyasında 225. satırda tanımlanmıştır.
