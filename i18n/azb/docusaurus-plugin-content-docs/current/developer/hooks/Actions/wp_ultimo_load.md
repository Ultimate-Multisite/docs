# Aksiyon: wp_ultimo_load

Tüm bağımlılıklar yüklendiğinde tetiklenir.

Eklenti geliştiricilerinin yeni işlevsellikler eklemesine olanak tanır. Örneğin, yeni hosting sağlayıcıları için destek eklemek gibi.
