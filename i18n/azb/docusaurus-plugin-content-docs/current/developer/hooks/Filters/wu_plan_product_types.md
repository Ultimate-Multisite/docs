# Filter: wu_plan_product_types

Bu filter, hangi ürün tiplerinin plan olarak kabul edilmesi gerektiğini belirler.

Bu filter, eklentilerin (addons) doğrulama (validation) ve ayırma (segregation) işlemlerinde plan olarak tanınması gereken ek ürün tipleri kaydetmesine olanak tanır.

## Parametreler {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | Plan olarak kabul edilecek ürün tiplerinin dizisi (array). |

### Since {#since}

- 2.3.0

[`inc/functions/product.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/product.php#L196) dosyasında 196. satırda tanımlanmıştır.
