# Filter: wu_contains_element

Developerlere ilk arama sonuçlarını değiştirme imkanı tanır.

Bu, özellikle üçüncü taraf yapıcılar (third-party builders) gibi durumlarda işe yarar.

## Parametreler

| Name | Type | Description |
|------|------|-------------|
| $contains_elements | `bool` | Element içeriğinde yer alıyor mu? |
| $content | `string` | İncelenen içerik. |
| $element | `self` | Mevcut element. |
| $post | `null\|\WP_Post` | Kontrol edilecek gönderi. |
