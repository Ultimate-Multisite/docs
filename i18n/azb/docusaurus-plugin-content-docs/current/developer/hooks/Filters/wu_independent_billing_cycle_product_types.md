# Filter: wu_independent_billing_cycle_product_types

Onay mga product type na may sariling billing cycle.

Ang mga produkto na may independent billing cycles ay hindi napipilitang sumunod sa billing period ng plan sa cart. Nakakatulong ito para sa mga produkto tulad ng domain registration na palaging taunang singil kahit na ang plan ay buwanan o taunan.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | Array ng mga product type na may independent billing cycles. |

- 2.5.0
