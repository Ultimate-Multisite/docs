# Filter: wu_admin_notices

Ultimate Multisite tarafından eklenen admin bildirimlerini (notices) filtrelemenize olanak tanır.

## Parametreler {#parameters}

| Name | Type | Açıklama |
|------|------|-------------|
| $notices | `array` | O panel için bildirim listesi. |
| $all_notices | `array` | Eklenen, panel bazında ayrılmış bildirim listesi. |
| $panel | `string` | Bildirimlerin alınacağı panel. |
| $filter | `string` | Kaldırılabilir bildirimler filtrelenmişse. |
| $dismissed_messages | `array` | Kaldırılmış bildirim anahtarlarının listesi. |

### Kullanılmaya Başladığı {#since}

- 2.0.0
