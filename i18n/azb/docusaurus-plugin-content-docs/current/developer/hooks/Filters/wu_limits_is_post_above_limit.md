# Filter: wu_limits_is_post_above_limit

Bu fonksiyon, belirli bir post tipinin bu plan kapsamında izin verilip verilmediğini kontrol eder. Plugin geliştiricilerinin dönüş değerini filtrelemesine olanak tanır.

## Parametreler

| Ad | Tip | Açıklama |
|------|------|-------------|
| $value | `bool` | Post tipi devre dışıysa veya değilse. |
