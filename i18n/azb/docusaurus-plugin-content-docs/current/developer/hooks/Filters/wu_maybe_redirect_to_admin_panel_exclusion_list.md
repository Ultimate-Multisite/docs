# Filter: wu_maybe_redirect_to_admin_panel_exclusion_list

Bu filtre, otomatik yönlendirmeyi engelleyen parametrelerden oluşan bir hariç tutma listesi oluşturur.

Bunun gerekli olması, aksi takdirde sayfa oluşturucunun (page builder) giriş sayfasını tanımladıktan sonra düzenleyemeyeceği anlamına geliyor.
