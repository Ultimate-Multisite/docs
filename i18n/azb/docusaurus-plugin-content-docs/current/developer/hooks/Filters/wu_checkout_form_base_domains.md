# فیلتر: wu_checkout_form_base_domains

اؤده‌مه فورمو اوچون مشترک اساس دامنه‌لری فیلترله‌ییر کی، نگاشت‌اولونموش دامنه قئیدلرینه چئوریلمسین.

بو فیلتره او واخت ایشلدین کی، بیر اینتگراسییا اؤده‌مه فورمو **سایت URL** ساحه‌لری اوچون علاوه اساس دامنه‌لر وئریر. بو فیلترله قایتاریلان دامنه‌لر هر سایتین اؤزل دامنه‌سی یئرینه، مشترک قئیدیات میزبانلاری کیمی قبول اولونور.

## پارامترلر

| آد | نوع | آچیقلاما |
|------|------|-------------|
| $domains | `array` | اؤده‌مه فورمو قورولوشوندان توپلانمیش مشترک اساس دامنه‌لر. |

### بو نسخه‌دن باشلایاراق

- 2.13.0

### قایناق

`inc/functions/domain.php` ایچینده تعریف اولونوب.


## قایتاریر

نورماللاشدیریلمیش اؤده‌مه فورمو اساس دامنه‌لرینین آرایه‌سی.
