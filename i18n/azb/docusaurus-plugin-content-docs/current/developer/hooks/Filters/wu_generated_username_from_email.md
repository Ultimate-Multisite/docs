# Filter: wu_generated_username_from_email

Oluşturulan müşteri kullanıcı adını filtreler.

## Parametreler

| Adı | Türü | Açıklama |
|------|------|-------------|
| $username | `string` | Oluşturulan kullanıcı adı. |
| $email | `string` | Yeni müşteri e-posta adresi. |
| $new_user_args | `array` | Yeni kullanıcı argümanlarının dizisi, belki ilk ve soyadları da içerebilir. |
| $suffix | `string` | Kullanıcı adını benzersiz yapmak için eklenecek metin. |

### Versiyon

- 3.7.0
