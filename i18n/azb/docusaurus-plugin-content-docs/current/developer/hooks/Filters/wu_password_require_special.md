# Filter: wu_password_require_special {#filter-wupasswordrequirespecial}

Bu filter, şifrelerde özel karakter zorunlu olup olmadığını kontrol etmenizi sağlar.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | Özel karakter zorunlu olup olmadığını belirtir. Kurallar etkinleştirildiğinde varsayılan değeri `true`'dur. |
| $defender_active | `bool` | Defender Pro Strong Password eklentisinin aktif olup olmadığını kontrol eder. |
