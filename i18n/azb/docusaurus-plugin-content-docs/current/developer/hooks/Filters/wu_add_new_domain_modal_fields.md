# Filtre: wu_add_new_domain_modal_fields

Alan ekleme penceresindeki (modal) alanları filtreler.

Bu filtre, eklentilerin (örneğin, Domain Seller) alan adı kayıt alanlarını değiştirmesine veya tamamen farklı alanlarla değiştirmesine olanak tanır.

## Parametreler {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $fields | `array` | Form alanları. |

### Versiyon {#since}

- 2.1.0
