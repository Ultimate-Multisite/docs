# Filter: wu_documentation_get_link {#filter-wudocumentationgetlink}

Plugin geliştiricilerinin linkleri filtrelemesine izin verir.

Bu işe yarayıp yaramayacağından emin değilim, ama olması zarar etmez.

## Parametreler {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $link | `string` | Kaydedilen link |
| $slug | `string` | Linki almak için kullanılan slug |
| $default_link | `string` | Kaydedilen varsayılan link |

- 1.7.0
