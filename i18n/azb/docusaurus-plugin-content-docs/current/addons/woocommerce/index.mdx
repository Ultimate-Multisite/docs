# WooCommerce بیرلشدیرمه‌سی

WooCommerce بیرلشدیرمه‌سی بیزیم Manual Gateway ایشله‌ییشینه چوخ بنزه‌ر بیر یوللا ایشله‌ییر. بو، ایشلدنلره چوخ اسنکلیک وئریر و اونلارا WooCommerce قورولوشونوزدا وار اولان هر هانسی اؤده‌مه فورمون سئچمه‌یه ایمکان یارادیر. ساده‌دیر: ایشلدننین حساب‌کشی دؤوره‌سی بیتن کیمی، ایشلدنن آبونمانی ساخلامایا قویولور. ائینی زاماندا، شبکه‌نین اصلی سایتینین WooCommerce قورولوشوندا بیر سیفارش یارادیلار. سونرا ایشلدنن اؤز ایمیلینه اؤده‌مه ائتمک اوچون بیر باغلانتی ایله فاکتور آلیر. اؤده‌مه آلیناندان سونرا، آبونمان یئنیلنیر.

## اؤزللیک‌لر {#features}

- Ultimate Multisite ایله WooCommerce آراسیندا آسان بیرلشدیرمه
- بیر نئچه اؤده‌مه یؤنتَمی
- بیرلیکده گلن آبونمانلار اؤزه‌یی ایله خودکار تکرارلانان حساب‌کشی، ایسته‌یه باغلی WooCommerce Subscriptions پلاگین آرتیرمالاری ایله
- اؤده‌مه مرحله‌سی توپلاملاری، Ultimate Multisite وئرگی‌لرینی آیری، وئرگی‌لنمه‌ین WooCommerce خَرج سطیرلری کیمی گؤستریر تا مشتری‌لر اؤده‌مه‌دن قاباق وئرگی میقدارلارینی گؤزدن کئچیره بیلسینلر

## قورولوم {#installation}

1. 'ultimate-multisite-woocommerce' فایلینی '/wp-content/plugins/' دایرکتوری‌سینه یوکله‌یین
2. پلاگینی WordPress ایچینده 'Plugins' منوسو ایله فعال ائدین
3. پلاگینی قورماق اوچون آددیم-آددیم Wizard-ی ایزله‌یین

## ایشلتمه بِلگه‌سی {#usage-guide}

_**اؤنه‌ملی قئید: بو مقاله Ultimate Multisite 2.x نسخه‌سینه عایددیر.**_

_**دیقت:** Ultimate Multisite: WooCommerce بیرلشدیرمه‌سی اوچون WooCommerce حداقل اصلی سایتینیزده فعال اولمالیدیر._

بیز باشا دوشوروک کی، _Stripe_ و _PayPal_ بعضی اؤلکه‌لرده یوخدور و بو، Ultimate Multisite ایشلدنلرینین بیزیم پلاگینی اتکیلی ایشلتمه‌سینی محدودلاشدیرار یا چتینلشدیرر. بونا گؤره _WooCommerce_ **بیرلشدیرمه‌سی اوچون بیر افزونه** یاراتدیق.

_WooCommerce_ چوخ تانینمیش بیر eCommerce پلاگینیدیر. دونیانین هر یئرینده دولوپرلر، موختلیف اؤده‌مه گئچیدلرینی اونا بیرلشدیرمک اوچون افزونه‌لر یارادیبلار. بیز بوندان یارارلاندیق تا Ultimate Multisite حساب‌کشی سیستمینده ایشلده بیله‌جه‌یینیز اؤده‌مه گئچیدلرینی گئنیشلندیره‌ک.

### اؤده‌مه مرحله‌سی وئرگی خَرج سطیرلری {#checkout-tax-fee-lines}

بیر Ultimate Multisite اؤده‌مه مرحله‌سینده وئرگی اولاندا، بیرلشدیرمه هر Ultimate Multisite وئرگی میقدارینی WooCommerce اؤده‌مه مرحله‌سینه آیری، وئرگی‌لنمه‌ین خَرج سطیری کیمی اضافه ائدیر. بو، اؤده‌مه‌دن قاباق وئرگی توپلامینی گؤرونن ساخلايیر و ائینی حالدا WooCommerce-ین وئرگی خَرجینی یئنه وئرگی‌له‌مه‌سینین قاباغینی آلیر.

## اؤده‌مه‌نی نئجه ایداره ائتمک {#how-to-manage-the-payment}

WooCommerce بیرلشدیرمه‌سی بیزیم **Manual Gateway** ایشله‌ییشینه چوخ بنزه‌ر یوللا ایشله‌ییر. منفی طرفی بودور کی، هر اؤده‌مه حساب‌کشی دؤوره‌سینین سونوندا الیله اؤده‌نیلمالیدیر. آخش بئله ایشله‌ییر:

  * ایشلدننین حساب‌کشی دؤوره‌سی بیتن کیمی، عضویت **ساخلامایا** قویولور. ائینی زاماندا، شبکه‌نیزین اصلی سایتینین _WooCommerce_ قورولوشوندا بیر سیفارش یارادیلار.

  * سونرا ایشلدنلر ایمیل‌لرینده اؤده‌مه ائتمک اوچون بیر باغلانتی ایله فاکتور آلیرلار (بو باغلانتی اونلارین آلت‌سایت Dashboard-ینده حساب‌کشی تاریخچه‌سینده ده گؤرونور). او باغلانتی ایشلدننی _**WooCommerce**_ **اؤده‌مه مرحله‌سینه** یؤنله‌ندیرر؛ اورادا ایشلدنلر اؤده‌مه‌لرینی ایشله‌مک اوچون **موجود _WooCommerce_ گئچیدلریندن** بیرینی سئچه بیله‌جه‌کلر.

  * اؤده‌مه آلیناندان سونرا، عضویت **یئنیله‌نیر**، اؤده‌مه Ultimate Multisite-ده قئید اولونور و سیفارش تاماملانمیش وعضیته قویولور.

### بیر نئچه اؤنه‌ملی قئید: {#some-important-notes}

  * Ultimate Multisite-ین وارسایلان ایشله‌ییشی بو بیرلشدیرمه فورمو ایله ده ایشله‌ییر. مسلن، **Subscription Management** صفحه‌سیندن **پول قایترما چیخارماق**، سئچیلن اؤده‌مه فورمو ایله پول قایترمانی ائتمک اوچون WooCommerce ایله ایلتیباط قوراجاق.

  * Ultimate Multisite: WooCommerce بیرلشدیرمه‌سی ایشله‌مک اوچون WP-Cron-ا دایانیر. بو چوخ وقت چوخ یاخشی ایشله‌ییر، اما اگر سایتین ترافیکی آز اولسا، cron زمانلانمیش رویدادلارینی یوخلاماغا تحریک ائدیلمیر و رویداد قاچیر؛ بو دا ایجرا گئجیکمه‌لرینه سبب اولا بیلر.

## WooCommerce Subscriptions پلاگینی {#woocommerce-subscriptions-plugin}

هر آی یئنی اؤده‌مه‌نی ال ایله باشلاتماق سون ایشلدنلر اوچون ایده‌آل دئییل، چونکی آبونمان‌لارین هر حساب‌دؤوره‌سینده اؤزبه‌اؤز تکرار اولماسی گؤزلنیر. WooCommerce Integration افزودنی‌سی ایندی اساس آبونمان کودونو اؤز ایچینده گتیرir، اونا گؤره تکرار اولان محصول‌لار تکرارلی حساب‌لاما اوچون همیشه آبونمان‌لاردان ایشلدیر.

بو Integration ایله تکرار اولان محصول‌لار ساتماق اوچون آیریجا [WooCommerce Subscriptions plugin](https://woocommerce.com/products/woocommerce-subscriptions/) آلماق یا قورماق لازم دئییل. اگر او plugin آلینیب، قورولوب و اصلی سایتینیزده موجود اولسا، Ultimate Multisite اونو ایشلده‌جک و WooCommerce آرتیق آبونمان ایداره‌سی ویژگی‌لرینی گؤستر‌جک. او موجود اولمایاندا، ایچه‌ریده گتیرلن اساس آبونمان کودو یئنه ده تکرار اولان محصول‌لاری ایداره ائدیر.

![WooCommerce آبونمان integration تنظیمات‌لاری](/img/addons/woocommerce-subscription-integration.png)

بو قدر. سون ایشلدنلرینیز آرتیق تکرار اولان محصول‌لار اوچون حساب‌دؤوره‌سینین سونوندا ال ایله اؤده‌مه ائتمه‌یه احتیاج دویمایاجاقلار.

اصلی سایتینیزده _WooCommerce_ آلتین‌دا بیر سفارش یارادیلماسیندان علاوه، هر Account اوچون ده بیر آبونمان یاراداجاق.

### اوغورسوز یئنیله‌مه‌ده آسقییا آلما داورانی‌شی {#failed-renewal-suspension-behavior}

Varsayılan اولاراق، WooCommerce Subscriptions اوغورسوز اؤده‌مه‌نین تکرار زمان‌بندی‌سینی ایشله‌دیکجه، عضولوق‌لار فعال قالیر. بو، عادی WooCommerce Subscriptions مهلت پنجره‌سینی ساخلاماقلا، برپا اولان بیر یئنیله‌مه‌نین آبونمان و عضولوق‌و تئزده‌ن سینک حالینا گتیرمه‌سینه ایمکان وئریر، گؤریش حاققینی واختیندان قاباق کسمه‌دن.

داها سخت گؤریش کنترولو ایسته‌ین ایداره‌چی‌لر WooCommerce Integration تنظیمات‌لارین‌دا **Suspend Memberships Immediately on Failed Renewals** فعال ائده بیلرلر. بو اختیارلی تنظیم فعال اولاندا، Ultimate Multisite یئنیله‌مه اؤده‌مه‌سی اوغورسوز اولان کیمی، WooCommerce Subscriptions تکرار ایداره‌سینین بیت‌مه‌سینی گؤزله‌مه‌دن باغلی عضولوق‌و آسقییا آلیر.

توسعه‌چی‌لر همین قراری `wu_woo_suspend_on_payment_failure` فیلتر ایله دَییشدیره بیلرلر. Hook مرجع صفحه‌لری افزودنی منبعیندن یارادیلیر، اونا گؤره بو اؤزت فقط فیلتر آدینی مخصوص آسقی سیاست‌لری اوچون گؤسترir.
