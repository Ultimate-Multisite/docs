# Aksiyon: switch_network

Mevcut ağ bağlamı değiştirildiğinde tetiklenir.

## Parametreler

| Name | Type | Description |
|------|------|-------------|
| $new_network_id | `int` | Geçilen ağın kimliği (ID). |
| $old_network_id | `int` | Daha önce aktif olan ağın kimliği (ID). |

### Versiyon

- 1.3.0
