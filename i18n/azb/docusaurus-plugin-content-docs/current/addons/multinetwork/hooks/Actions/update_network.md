# Action: update_network {#action-updatenetwork}

Bu hook, var olan bir network güncellendikten sonra tetiklenir.

## Parametreler {#parameters}

| Ad | Tip | Açıklama |
|------|------|-------------|
| $network_id | `int` | Eklenen network'ün ID'si. |
| $args | `array` | Network argümanlarının bir sözlük (associative) dizisi. |
