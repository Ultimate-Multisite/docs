# Support Agents

Bir network'i tek başına yönetmek kolay bir iş değil. Network büyümeye başladıkça, destek talebinin arttığını göreceksiniz. İş yükünüzü hafifletmenize yardımcı olacak insanlarla çalışmanız iyi olur. Ancak, tam erişim/izin vermek risklidir ve network'inizi tehlikeye atabilir. İşte bu noktada Support Agent eklentisi devreye giriyor.

Bu eklenti, Support Agent adında yeni bir WordPress kullanıcı türü oluşturmanıza olanak tanır ve bu kullanıcının hangi izinlere sahip olacağı üzerinde size tam kontrol sağlar. İzin vermek, bir anahtarı açma düğmesine basmak kadar kolaydır; kaldırmaya karar verirseniz, kapatmak da aynı derecede kolaydır.

## Özellikler {#features}

* WordPress ve Ultimate Multisite izinlerinin detaylı listesi
* Erişimi kolayca açma ve kapatma
* Sınırsız sayıda Agent hesabı oluşturma

## Kurulum {#installation}

1. 'ultimate-multisite-support-agents' dosyasını '/wp-content/plugins/' dizinine yükleyin
2. WordPress'teki 'Eklentiler' menüsünden eklentiyi etkinleştirin
3. Eklentiyi kurmak için adım adım Sihirbaz'ı takip edin
