# Admin Page Creator

## İçerik {#content}
Admin Page Creator, WordPress dashboard'unuzda yeni yönetici sayfaları ve alt sayfaları eklemenizi kolaylaştıracak. Bu özellik, yeni kullanıcılar için bir açılış sayfası veya eğitim sayfası görevi görecek bir karşılama sayfası oluşturmak için kullanılabilir. Sayfaları zahmetsizce tasarlamak için Beaver Builder, Elementor, Brizy ve Oxygen gibi sayfa oluşturucuları kullanabilirsiniz.

Ayrıca özel HTML, CSS ve JavaScript desteği, plan ve kullanıcı rolüne dayalı erişim kontrolü sunar!

## Özellikler {#features}
* Özel HTML ve CSS Desteği
* JavaScript Desteği
* PHP İçerik Modu
* Kullanıcı rolüne dayalı izinler
* Sayfa oluşturuculara destek

## Kurulum {#installation}

1. **ultimate-multisite-admin-page-creator** dosyasını **/wp-content/plugins/** dizinine yükleyin
2. Eklentiyi WordPress Network Admin'deki **Plugins** menüsünden etkinleştirin

## Kullanım Kılavuzu {#usage-guide}

WordPress, sitelerin ön yüzüne içerik eklemeyi çok kolay hale getirirken, yönetici tarafında sayfa ve alt sayfa oluşturmak biraz temel kodlama bilgisi gerektirir. Ultimate Multisite: Admin Page Creator, bu kolaylığı ağ sitelerinizin arka yüzüne taşımayı amaçlar.

## Admin Page Creator'ı Neden Kullanmalısınız? {#why-use-admin-page-creator}

Belki müşterilerinizin admin paneline eğitimler içeren özel bir sayfa sunmak istersiniz, belki de SSS'ler ve bir iletişim widget'ı içeren özel bir sayfa eklemek istersiniz. Admin Page Creator ile olasılıklar sonsuz ve bugün yapacağımız eğitimde bunları keşfedeceğiz.

## İlk Özel Yönetici Sayfanızı Oluşturma {#creating-your-first-custom-admin-page}

Ultimate Multisite: Admin Page Creator'ı kurup etkinleştirdikten sonra, ağ yöneticisi menünüze gidin ve Admin Pages'a tıklayın.

![Custom Admin Pages list on Network Admin](/img/admin/admin-pages-list.png)

_Network Admin'deki Özel Yönetici Sayfaları listesi_

Şimdiye kadar oluşturduğunuz tüm yönetici sayfalarının bir listesini göreceksiniz (ilk ziyaretiniz olduğu için liste boş olacaktır). **Add new Admin Page**'a tıklayın. Bu sizi Admin Page düzenleyicisine yönlendirecektir.

## Özel Yönetici Sayfasının Öğeleri {#the-elements-of-a-custom-admin-page}

Admin Page düzenleyicisinde, sayfanızı özelleştirmek için birçok farklı seçeneğiniz olacak.

Bu, sayfanın en üstünde görünecek olan başlıktır, yönetici menü etiketi değil.

### İçerik Türü Seçici (Content Type Selector) {#content-type-selector}

Ultimate Multisite: Admin Page Creator, sayfanın içeriği konusunda size iki seçenek sunar: ya varsayılan WordPress WYSIWYG düzenleyicisini kullanabilir ya da HTML düzenleyicisine geçebilirsiniz.

HTML konusunda deneyimli değilseniz, varsayılan WordPress düzenleyicisinde kalın. Sayfanın içeriğini, normal bir Gönderi veya Sayfa'da yapacağınız gibi düzenleyebileceksiniz.

## Menü Seçenekleri {#menu-options}

Yan tarafta, General Options meta kutusunda, menü türünü, etiketini ve çok daha fazlasını yapılandırabileceksiniz.

### Menü Türleri (Menu Types) {#menu-types}

Yeni özel yönetici sayfanız iki farklı menü türüne sahip olabilir: ana menü (top-level menu) ve alt menü (submenu).

Adından da anlaşılacağı gibi, ana menü sayfası, yeni oluşturduğunuz sayfayı varsayılan WordPress sayfalarının yanına yönetici menüsüne ekleyecektir.

Ana menü seçeneğini seçerseniz, menü sırasını ve bir menü simgesini de seçebilirsiniz.

Bu sayfayı bir alt menü sayfası yapmak isterseniz, bu sayfayı bağlayacağınız bir üst sayfa seçebilirsiniz. Özel sayfanızı varsayılan WordPress sayfalarına veya ana özel yönetici sayfalarınıza ekleyebilirsiniz.

_Bu sayfayı, kendi özel ana sayfalarınız dahil olmak üzere, diğer menü sayfalarının alt menüsü olarak da ekleyebilirsiniz._

## Gelişmiş Seçenekler (Advanced Options) {#advanced-options}

Özel yönetici sayfalarınıza her türlü içeriği ekleyebilmeniz gerekir. Buna CSS kuralları ve dosyaları, JavaScript ve harici kütüphaneler de dahildir. Bunu Gelişmiş Seçenekler sekmesini kullanarak yapabilirsiniz.

### Özel CSS (Custom CSS) {#custom-css}

CSS sekmesinde kendi özel CSS kodunuzu ekleyebilir ve uzaktaki sitelerden (CDN'ler, Google Fonts vb.) CSS dosyaları içe aktarabilirsiniz.

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

### Özel JavaScript (Custom JavaScript) {#custom-javascript}

JavaScript için de durum aynıdır. Özel JavaScript kodunuzda kullanmak üzere harici kütüphaneler ekleyebilirsiniz.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Kendi JavaScript kodunuzu ekleyin!_

## İzinler (Permissions) {#permissions}

Ayrıca Gelişmiş Seçenekler'de, hangi alt sitelerin bu sayfaya erişimine sahip olacağını seçebilirsiniz. Bu, örneğin farklı plan katmanlarına farklı özel içerikler sunmayı planlıyorsanız kullanışlıdır.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Bu sayfanın nerede görüneceği üzerinde tam kontrolünüz var_

**Önemli**: Sayfanın gösterilmesi için her iki koşulun da karşılanması gerekir. Bu şu anlama gelir: kullanıcı belirli bir plana sahip bir müşteri olsa bile, seçilen rollerden birine sahip değilse (veya tam tersi), sayfa panelinde görünmeyecektir.

## Aktif (Active) {#active}

Bir özel yönetici sayfasına silmeden veya izinlerle uğraşmadan devre dışı bırakmak isterseniz, sadece Active seçeneğindeki kutucuğu kaldırın.

Ana bir sayfayı devre dışı bırakmak, o üst sayfaya bağlı tüm alt sayfaları otomatik olarak devre dışı bırakacaktır.
