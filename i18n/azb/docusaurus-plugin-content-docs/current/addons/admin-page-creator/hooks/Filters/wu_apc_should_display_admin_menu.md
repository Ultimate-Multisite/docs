# Filter: wu_apc_should_display_admin_menu {#filter-wuapcshoulddisplayadminmenu}

Yöneticilerin, özelleştirme sürecini tamamladıysa, yönetici menüsü sayfalarını gizlemesine izin verir.

## Parametreler {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $should_display_admin_menu | `bool` | Sayfaların gösterilip gösterilmeyeceğini belirtir. |
| $network_active | `bool` | Eklentinin ağ yöneticisi (network admin) tarafından etkinleştirilip etkinleştirilmediğini kontrol eder. |
