# Журнал изменений для создателя страниц админки

### Версия 1.9.0 - Выпущено 2026-01-18 {#version-190---released-on-2026-01-18}

* Исправлено: Критические ошибки при активации.
* Переработан: Классы источников контента теперь используют трейт Singleton с правильной инициализацией родительского конструктора.
* Исправлено: Ошибки компиляции шаблонов Vue.js, из-за которых не отображался селектор типа контента.
* Исправлено: Отсутствующие свойства данных Vue (template_id, external_link_url_embedable, external_link_url_checking).
* Исправлено: Редактор TinyMCE не инициализировался должным образом при загрузке страницы.
* Исправлено: Файл поддержки Divi не загружался автоматически через Composer.
* Исправлено: Поврежденный синтаксис PHP в классе поддержки Divi.
* Улучшено: Организация и минификация JavaScript-ресурсов.
* Улучшено: Совместимость с последними версиями WordPress и page builder.
