# چوخ‌ایجاره‌چی‌لیک {#multi-tenancy}

آلت‌سایت‌لاری پرفورمانس و گؤوه‌نلیک اوچون سرورلر و دیتابیس‌لر آراسیندا پایلایین

## Version 1.2.0 بوراخیلیشینین اؤنه چیخانلاری {#version-120-release-highlights}

Multi-Tenancy 1.2.0 تولید عایرما ایش‌آخیشلاری اوچون مستقل ایجاره‌چی تؤملینی آرتیریر:

- قیسا‌اؤمورلو، تکرار حمله‌سیندن قورونان SSO گؤروش‌لری ایله حال‌سیز ایجاره‌چی اوتوماتیک گیریشی.
- مستقل ایجاره‌چیلر اوچون هر آلت‌سایتا آیری دیتابیس و فایل‌سیستم عایرماسی.
- میزبان‌دان آگاه ایجاره‌چی یاراتماق اوچون Hostinger چوخ‌ایجاره‌چی‌لیک قابلیتی.
- کؤهنه داده یوخلامالاری و async گؤندریش ایشلنمه‌سی اوچون مستقل کؤچورمه دوغرولاما قاپیلاری.
- Bedrock، FrankenPHP و localhost قورولوشلاری اوچون تنظیم‌اولا بیلن عینی‌ماشین دیتابیس میزبان باغلانماسی.
- ایجاره‌چی باشلاتما، دیتابیس یازان، شِما باشلاتما و ایشلدن یاراتما یاخشیلاشمالاری.
- ایجاره‌چی کیملیک‌لرینی تمیزله‌ین داها گؤوه‌نلی مستقل سیلمه.

[مستقل کؤچورمه ایش‌آخیشی](/user-guide/getting-started/sovereign-migration-workflow)، [چوخ‌ایجاره‌چی‌لیک عایرماسی](/user-guide/configuration/multi-tenancy-isolation)، و [مستقل ایجاره‌چی عملیاتی](/user-guide/administration/sovereign-tenant-operations) اوچون ایشلدن راهنما‌لارینا باخین.

## گره‌کلر {#requirements}

- WordPress 5.3 یا داها یوخاری
- PHP 7.4 یا داها یوخاری
- Multisite Ultimate افزونه‌سی (فعال)

## قوراشدیرما {#installation}

1. افزونه فایل‌لارینی `/wp-content/plugins/` دایرکتوری‌سینه یوکله‌یین
2. افزونه‌نی WordPress ایچینده 'Plugins' منوسی ایله فعال ائدین
3. افزونه تنظیم‌لرینی شبکه ایداره‌چی‌سینده قورون

## گلیشدیرمه {#development}

### قورولوش {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

```

├── inc/                          # PHP کلاس‌لاری و شامل‌لر

├── tests/                        # واحد تست‌لری
│   ├── bootstrap.php            # تست باشلاتماسی
│   ├── class-*-base.php         # تؤمل تست کلاسی
│   └── class-*-test.php         # تست موردلری
├── views/                       # قالب فایل‌لاری
├── assets/                      # ساکین دارلیقلار
│   ├── css/                     # استایل‌شیط‌لر
│   ├── js/                      # JavaScript فایل‌لاری
│   └── images/                  # تصویرلر
├── lang/                        # دیل فایل‌لاری
├── composer.json                # PHP باغلیلیق‌لاری

├── phpunit.xml.dist             # PHPUnit قورولوشو
├── .phpcs.xml.dist              # کد استانداردلاری قورولوشو

```

### تست {#testing}

بو افزونه گئنیش بیر تست چارچیوَه‌سی داخیل ائدیر:

- **واحد تست‌لری**: WordPress تست مجموعه‌سی ایله PHPUnit اساسلی تست‌لر
- **تؤمل تست کلاسی**: عمومی تست سناریولاری اوچون یاردیمچی روش‌لار وئریر
- **تست اؤرتویو**: کد اؤرتویونو ایزله‌مک اوچون قورولوب
- **چوخ‌سایت تستی**: تست‌لر چوخ‌سایت محیطینده ایشله‌ییر

### کد استانداردلاری {#code-standards}

- **WordPress کدلاشتیرما استانداردلاری**: PHPCS ایله اجرا اولونور
- **PHP 7.4+ اویغونلوغو**: مدرن PHP اؤزللیک‌لری دستکلنیر
- **ساکین تحلیل**: نوع یوخلاماسی اوچون PHPStan بیرلشمه‌سی
- **کدو مدرن‌لشدیرمه**: PHP یوکسه‌لتمه‌لری اوچون Rector قایدالاری

### قورما سیستمی {#build-system}

- **دارلیق ایشله‌مه**: CSS/JS فایل‌لاری اوچون کیچیلتمه
- **ترجمه**: POT فایل یاراتماسی
- **باغلاما**: پایلاماق اوچون آرشیو یاراتماسی
- **گلیشدیرمه حالتی**: عیب‌تاپما اوچون کیچیلدیلمه‌میش دارلیقلار

## قورولوش {#configuration}

بو افزونه شبکه ایداره‌چی آرا‌یوزو ایله چاتیلان چئشیدلی قورولوش سئچیم‌لرینی دستکله‌ییر.

### مستقل ایجاره‌چی قورولوشو {#sovereign-tenant-configuration}

مستقل ایجاره‌چیلر اوچون، ایجاره‌چی دیتابیسینی، فایل‌سیستم کؤکونو، دیتابیس میزبان باغلانماسینی و میزبان قابلیتی‌نی، ترافیکی ایجاره‌چی‌یه یؤنلتمه‌دن قاباق قورون. دیتابیس کیملیک‌لری، میزبان باغلانماسی، یؤنلندیرمه یا ایشلدن یاراتمایا تأثیر ائدن هر دَییشیک‌دن سونرا کؤچورمه دوغرولاماسینی ایشله‌دین.

`localhost`-و یالنیز ایجاره‌چی اجرا محیطی و دیتابیس ایجازه‌لری یئرلی سوکت یا عینی‌ماشین گئریشینی گؤزله‌دییی زامان ایشلدین. تأمین‌ایدن‌نیزین قوردوغو ایجازه میزبان او اولا، `127.0.0.1` یا بیر خیدمت میزبان آدینی ایشلدین.

### Hostinger قابلیتی {#hostinger-capability}

Hostinger چوخ‌ایجاره‌چی‌لیک قابلیتی پایلاشیلمیش Hostinger بیرلشمه token-ی ایله ایشله‌ییر. دوغرولامادان قاباق دامینین hPanel-ده وار اولدوغونو، DNS-ین Hostinger Account-ونا اشاره ائتدییینی و ایجاره‌چی دیتابیس تنظیم‌لرینین hPanel ایله اویغون اولدوغونو دوغرولایین.

## Hook-لار و Filter-لر {#hooks-and-filters}

### Action-لار {#actions}

- `ultimate-multisite-addon-template_init` - افزونه باشلاتماسیندان سونرا چاغیریلیر
- `ultimate-multisite-addon-template_loaded` - بوتون افزونه فایل‌لاری یوکلندن سونرا چاغیریلیر

### Filter-لر {#filters}

- `ultimate-multisite-addon-template_settings` - افزونه تنظیم‌لرینی دَییشدیرین
- `ultimate-multisite-addon-template_enabled` - افزونه‌نین فعال/غیرفعال حالتی‌نی اؤزونه‌گؤره دَییشدیرین
