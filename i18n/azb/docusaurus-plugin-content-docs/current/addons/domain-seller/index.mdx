# Domain Seller

**Multisite Ultimate: Domain Seller** سیزین WordPress Multisite شبکه‌نیزی گوجلو دومین ساتیش پلاتفورماسینا چئویرر. موشتریلر قئیدیات زامانی یا اؤز Dashboard-لاریندان اؤزل دومین‌لری بیر باشا قئید ائده بیلرلر؛ حقیقی زاماندا موجودلوغ یوخلاماسی و شفاف قیمتله.

### نییه Domain Seller؟ {#why-domain-seller}

دومین قئیدی هر WaaS (وب‌سایت بیر خیدمت کیمی) پلاتفورماسی اوچون ان چوخ ایسته‌نیلن ارتیریم ساتیشلاردان بیریدیر. موشتریلرینیزی اوچونجو طرف registrar-ا گؤندرمک یئرینه، دومین‌لری بیر باشا سیز تکلیف ائده بیلرسینیز — موشتریلره مانعئسیز تجربه وئررکن، گلیرین داها چوخونو اؤز شبکه‌نیزده ساخلیارسینیز.

### اساس اؤزلیک‌لر {#key-features}

#### چوخلو دومین Registrar دسته‌یی {#multiple-domain-registrar-support}
* **OpenSRS** — تام قئیدیات، تجدید و ایداره‌اتمه
* **Namecheap** — WhoisGuard و sandbox سینیغی ایله کامل بیرلشمه
* **HostAfrica** — Domains Reseller API واسطه‌سیله دومین آختاریشی، قیمت، قئیدیات، کؤچورمه، DNS و ID قورونماسی
* **Openprovider** — Reseller قیمت‌لری، قئیدیات، تجدید، کؤچورمه‌لر، DNS، WHOIS محرملییی و TLD سینک
* **Hostinger** — مشترک Hostinger بیرلشمه token-ی ایله دومین آختاریشی و قئیدیات
* **گئنیشلندیریله بیلن معماری** — Integration Registry قالیبی اوستونده قورولوب، داها چوخ provider آرتیرماغی آسانلاشدیریر
* **داخیلی باغلانتی سینیغی** — یایینا چیخماقدان قاباق provider کیملیک بیلگی‌لرینی و باغلانتی‌نی یوخلایین

#### یؤنلندیریلن قورولوش Wizard-ی {#guided-setup-wizard}
* **بیر کلیکله باشلاتما** — سیزی provider قورولوشو، TLD ایمپورتی و دومین محصولو یاراتماسیندان کئچیریر
* **اوتوماتیک TLD سینک** — باغلی provider-لارینیزدان بوتون موجود TLD-لری و توپدان قیمت‌لری ایمپورت ائدیر
* **واریلان محصول یاراتماسی** — عاقلانه markup واریلانلاری ایله ایشلتمه‌یه حاضیر دومین محصولو یارادیر

#### اسنک دومین قیمت‌لندیرمه‌سی {#flexible-domain-pricing}
* **اوچ markup حالتی** — یوزده markup، ثابت markup، یا هر TLD اوچون ثابت قیمت
* **TLD-یه اؤزل محصوللار** — فرقلی TLD قروپ‌لاری اوچون آییری دومین محصوللاری یارادین (.com/.net، .io، اؤلکه کودلاری و س.)
* **تانیتیم قیمت‌لندیرمه‌سی** — یئنی قئیدیاتلاری جلب ائتمک اوچون بیرینجی ایل اوچون ایندیریملی قیمت تکلیف ائدین
* **حقیقی زامان قیمت‌لندیرمه‌سی** — آغیللی cache ایله provider-لاردان جانلی توپدان قیمت‌لر
* **Coupon دسته‌یی** — Multisite Ultimate ایندیریم کودلاری ایله تام اویغونلوق

#### WHOIS محرملیق قورونماسی {#whois-privacy-protection}
* **هر محصول اوچون قورولوش** — WHOIS محرملییینی Disabled، Always Included، یا Customer Choice کیمی تنظیم ائدین
* **موشتری‌نین قبول ائتمه‌سی** — Customer Choice سئچیلنده، checkout چئک‌باکسی موشتریلره محرملیق قورونماسی آرتیرماغا ایمکان وئرر
* **قورولا بیلن قیمت** — محرملیق add-on-و اوچون هر ایللیک قیمت تعیین ائدین، یا اونو پولسوز تکلیف ائدین
* **Registrar بیرلشمه‌سی** — قئیدیات زامانی اوتوماتیک WhoisGuard (Namecheap) یا WHOIS محرملییینی (OpenSRS) فعال ائدیر

#### بیرلشیک Checkout تجربه‌سی {#unified-checkout-experience}
* **اوچ دومین حالتی** — موشتریلر subdomain (پولسوز)، یئنی دومین قئید ائتدیرمک، یا اؤزلرینه آید موجود دومینی خریطه‌لندیرمک آراسیندا سئچیرلر
* **حقیقی زامان دومین آختاریشی** — صفحه یئنیلنمه‌دن AJAX ایله موجودلوغ یوخلاماسی
* **آغیللی تکلیف‌لر** — سئچیلن دومین گؤتورولموش اولدوقدا بدیل TLD تکلیف‌لری
* **شفاف قیمت‌لندیرمه** — قئیدیات قیمتی، تجدید قیمتی و اختیاری WHOIS محرملیق حاققی آیدین گؤستریلیری
* **قئید ائدن شخصین تماس ساحه‌لری** — checkout زامانی لازیم اولان قئید ائدن بیلگی‌لرینی (آد، آدرس، تلفن) ائله هم اورادا توپلار
* **اوتوماتیک یارادیلان سایت URL-ی** — سایت آدرسی سئچیلن دومین‌دن اوتوماتیک چیخاریلیر

#### اوتوماتیک تجدید سیستمی {#auto-renewal-system}
* **عضویت‌ه باغلی تجدیدلر** — موشتری‌نین عضویت‌ی فعال اولدوغو موددتجه دومین‌لر اوتوماتیک تجدید اولونور
* **آغیللی زمانلاما** — تجدید ایشلری دومینین بیتیش تاریخ‌لرینه گؤره نؤوبه‌یه قویولور
* **یئنی‌دن سیناما منطقی** — اوغورسوز تجدید چالیشمالاری اوچون آغیللی یئنی‌دن سینامالار
* **ایمیل بیلدیریشلری** — یاخینلاشان تجدیدلر، اوغورلو تجدیدلر، اوغورسوزلوقلار و بیتملر اوچون قورولا بیلن خبرلر

#### موشتری DNS ایداره‌اتمه‌سی {#customer-dns-management}
* **اؤزون‌خیدمت DNS** — موشتریلر اؤز Dashboard-لاریندان DNS قئیدلرینی (A, AAAA, CNAME, MX, TXT) آرتیرا، دَییشدیره و سیله بیلرلر
* **دومین دورومو** — حقیقی زامان قئیدیات دورومو، بیتیش تاریخ‌لری و nameserver بیلگی‌لری
* **اوتوماتیک دومین خریطه‌لندیرمه‌سی** — قئید ائدیلمیش یا موجود دومین‌لر اوتوماتیک موشتری‌نین سایتینا خریطه‌لندیریلیر

#### Admin آلت‌لری {#admin-tools}
* **ال ایله دومین قئیدیاتى** — admin panel-دن موشتریلر آدینا دومین قئید ائدین
* **دومین ایزلمه‌سی** — بوتون موشتری دومین‌لرینی، دوروملاری و بیتیش تاریخ‌لرینی بیر صفحه‌دن گؤرون
* **Provider-ه اؤزل log-لاما** — هر registrar اوچون آییری log کاناللاری (اؤرنک: domain-seller-namecheap.log)
* **قئیدیات دتایل‌لاری** — اساس دومین دَییشدیرمه صفحه‌سینده قئید ائدن بیلگی‌لرینی و قئیدیات داده‌لرینی گؤرون

### دَسته‌کلنَن Provider-لار {#supported-providers}

* **OpenSRS** — قئیدیات، تجدید، DNS ایداره‌اتمه‌سی و WHOIS محرملییی ایله تام بیرلشمه
* **Namecheap** — قئیدیات، تجدید، توپلو موجودلوغ یوخلاماسی و پولسوز WhoisGuard ایله تام بیرلشمه
* **HostAfrica** — قئیدیات، تجدید، کؤچورمه‌لر، DNS ایداره‌اتمه‌سی، EPP کودو آلما، registrar قفلی و ID قورونماسی
* **Openprovider** — قئیدیات، تجدید، کؤچورمه‌لر، DNS ایداره‌اتمه‌سی، EPP کودو آلما، registrar قفلی، WHOIS محرملییی و reseller TLD سینک
* **Hostinger** — قئیدیات، دومین آختاریشی، nameserver گؤنجه‌للمه‌لری، registrar قفلی و WHOIS محرملییی؛ تجدیدلر Hostinger auto-renew طرفیندن ایداره اولونور
* **GoDaddy** — قئیدیات و تجدید
* **ResellerClub** — قئیدیات، تجدید و DNS ایداره‌اتمه‌سی
* **NameSilo** — قئیدیات و تجدید
* **Enom** — قئیدیات، تجدید و DNS ایداره‌اتمه‌سی

### تکنیکی دتایل‌لار {#technical-details}

* **لازیمدیر:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **معماری:** Domain_Selling_Capability interface ایله Integration Registry قالیبی
* **Autoloading:** PSR-4 class mapping ایله Jetpack autoloader
* **استانداردلار:** WordPress کودلاما استانداردلاری، strict typing، prepared statements

## قوراشدیرما {#installation}

1. **Multisite Ultimate v2.4.12+** قورولدوغونا و شبکه‌ده فعال ائدیلدیینه امین اولون
2. افزونه‌نی `/wp-content/plugins/ultimate-multisite-domain-seller/` ایچینه یوکلین
3. افزونه‌نی WordPress Network Admin ایچین‌ده **افزونه‌لر** بؤلمه‌سیندن شبکه‌ده فعال ائدین
4. **قورولوش سئحیرلی‌سی** ایلک فعال‌لاشما‌دا اؤز-اؤزونه آچیلیر — بو آددیملاری ایزلین:
   * دامنه ثبت‌چی‌نیزله باغلانین
   * باغلانتی‌نی یوخلایین
   * موجود TLDلری و توپدان قییمت‌لری وارد ائدین
   * ایلک دامنه محصولونوزو قییمت آرتیمی ایله یارادین
5. **Ultimate Multisite > اؤدنیشر فورملاری** ایچین‌ده اؤدنیشر فورمونوزا **دامنه سئچیمی** ساحه‌سینی آرتیرین

## تئز-تئز سورولان سواللار {#frequently-asked-questions}

### Multisite Ultimate نین هانسی نسخه‌سی لازیم‌دیر؟ {#what-version-of-multisite-ultimate-is-required}

نسخه 2.4.12 یا داها یوخاری. بو addon، WP Ultimo v1.x یا Ultimate Multisite نین 2.4.12-دن آشاغی نسخه‌لری ایله اویغون دئییل.

### هانسی دامنه تأمین‌چی‌لری دَسته‌کلنیر؟ {#which-domain-providers-are-supported}

دوققوز ثبت‌چی دَسته‌کلنیر: **OpenSRS**، **Namecheap**، **HostAfrica**، **Openprovider**، **Hostinger**، **GoDaddy**، **ResellerClub**، **NameSilo** و **Enom**. قابلیت‌لر تأمین‌چی‌یه گؤره فرق ائدیر، چونکی ثبت‌چی APIلری فرقلی عمل‌لری گؤستریر؛ هر تأمین‌چی اوزرینه اومومی باخیش اوچون قورولوش بئلگه‌سینه باخین. addon-ون Integration Registry معماری‌سی، اؤزل تأمین‌چی‌لر آرتیرماغی آسان ائدیر — [گلیشدیرن بئلگه‌سینه](/developer/integration-guide/custom-registrar) باخین.

### WHOIS محرملیق قورونماسی تکلیف ائده بیلرم؟ {#can-i-offer-whois-privacy-protection}

بلی. هر دامنه محصولونون اوچ حالتلی WHOIS محرملیق تنظیمی وار: سوندورولموش، همیشه داخیل، یا موشتری سئچیمی. موشتری سئچیمی سئچیله‌نده، موشتری‌لر اؤدنیشر زامانی قییمتله بیر checkbox گؤرورلر (یا Namecheapین WhoisGuard-ی اوچون "پولسوز"). محرملیق بایراغی ثبت زامانی اؤز-اؤزونه ثبت‌چی‌یه گؤندریلیر.

### فرقلی TLDلر اوچون فرقلی قییمت قویابیله‌رم؟ {#can-i-set-different-pricing-for-different-tlds}

بلی. فرقلی TLD قروپ‌لاری اوچون آیری دامنه محصول‌لاری یارادین. هر محصولون اؤز آرتیم نوعو (فایز، ثابت آرتیم، یا ثابت قییمت)، دَسته‌کلنن TLDلری و تانیتیم قییمتی اولا بیلر.

### موشتری‌لر هانسی اؤدنیشر سئچیم‌لرینی گؤرور؟ {#what-checkout-options-do-customers-see}

دامنه سئچیمی ساحه‌سی موشتری‌لره اوچ تب وئریر: **آلت‌دامنه** (شبکه‌نیزده پولسوز آلت‌دامنه ایشلدین)، **یئنی دامنه ثبت ائت** (یئنی دامنه آختار و ثبت ائت)، و **موجود دامنه** (اؤزلرینده اولان دامنه‌نی باغلا). یالنیز فعال ائتدیینیز حالت‌لر گؤرونه‌جک.

### خودکار یئنی‌لنمه‌لری دَسته‌کلیر؟ {#does-it-support-automatic-renewals}

بلی. دامنه‌لر auto-renew اوچون تنظیم ائدیله بیلر و موشتری‌نین عضویت دورومونا باغلانیر. یئنی‌لنمه اؤدنیشرلری موشتری‌نین موجود اؤدنیشر دروازه‌سی ایله ایشلنیر.

### موشتری‌لر DNS قئیدلرینی ایداره ائده بیلر؟ {#can-customers-manage-their-dns-records}

بلی. موشتری‌لر A، AAAA، CNAME، MX و TXT قئیدلرینی آرتیرماق، دَییشدیرمک و سیلمک اوچون اؤز Dashboardلارىندا DNS ایداره آرایوزو آلارلار. DNS ایداره‌سی OpenSRS، ResellerClub، Enom، HostAfrica و Openprovider ایله موجوددور. میزبانی ائدیلن دامنه‌لر اوچون Hostinger DNS، اصلی Hostinger دامنه باغلاما اینتگراسیاسی ایله ایداره ائدیلیر.

### قورولوش سئحیرلی‌سی وار؟ {#is-there-a-setup-wizard}

بلی. قورولوش سئحیرلی‌سی ایلک فعال‌لاشما‌دا خودکار ایشلیر و سیزی ثبت‌چی‌یه باغلانماق، TLDلری وارد ائتمک و ایلک دامنه محصولونوزو یاراتماق آددیملاریندان کئچیر. سئحیرلی‌نی addon تنظیم‌لریندن یئنی‌دن ده ایشلده بیلرسینیز.
