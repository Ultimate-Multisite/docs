# Action: wp_ultimo_ptm_page_edit_after_title

## Parametreler

| Ad | Tip | Açıklama |
|------|------|-------------|
| $ | `object` | Bilgileri tutan nesne. |
| $ | `\WU_Page` | Multisite Ultimate Sayfa örneği. |

### Kullanılabilir

- 1.8.2

[`views/base/edit.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-plugin-and-theme-manager/blob/main/views/base/edit.php#L48) dosyasında 48. satırda tanımlanmıştır.
