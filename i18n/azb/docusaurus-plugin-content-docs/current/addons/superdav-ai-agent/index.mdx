# Superdav AI Agent

Superdav AI Agent قیراغین بیر WordPress AI یاردیمچیسیدیر کی Gratis AI Agent-ی تم‌دن آگاه باجاریقلار، گؤجلندیریلمیش اینترنت آختاریش قابلیتلری، سایت قوروجو اوزللیکلری و ایسته‌یه باغلی اؤز-اؤزونه ایشله‌مه اینتگراسیالاری ایله گئنیشلدیر. 1.18.0 سورومی Google Calendar اوخوما آلتلری، علاقه و قاتیلان اویغونلاشدیرماسی، انسان تائیدی قاپیلاری، یادآوری تکرارسیزلاشدیرماسی، TextBee SMS بیلدیریشلری، ایداره اولونان Superdav خدمتی قورولوشو، و اینانیلان گلیشدیرن ایش آقیشلاری اوچون آیری یاییلان ایرلی یانداش پاکتی آرتیریر.

## اساس اوزللیکلر

- **تم‌دن آگاه باجاریقلار** — یؤنلندیرمه و قابلیتلری سیزین فعال WordPress تمینیزه و قورولموش افزونه‌لره گؤره اؤز-اؤزونه اویغونلاشدیرار
- **چوخلو اینترنت آختاریش وئرنلری** — داها زنگین آختاریش نَتیجه‌لری و ایرلی آراشدیرما قابلیتلری اوچون Brave Search و Tavily آراسیندان سئچین
- **سایت قوروجو علاقه فرمالاری** — قورولموش فرم افزونه‌نیز ایله چت آرا اوزوندن بیر‌باشا تام ایشله‌ین علاقه فرمالاری یارادین
- **گؤجلندیریلمیش سایت قوروجو** — داها یاخشی افزونه تاپماسی و خطا برپاسی ایله یاخشیلاشدیریلمیش چوخ‌مرحله‌لی سایت قورما پلانلاری
- **داخیلی تم باجاریقلاری** — Block Themes، Classic Themes، Kadence Blocks و Kadence Theme اوچون اؤزل یؤنلندیرمه
- **یاخشیلاشدیریلمیش WooCommerce اینتگراسیاسی** — داها چوخ اینانیلیق و اویغونلوق اوچون دوغما WooCommerce API-لریندن ایستیفاده ائتمگه یئنی‌دن دوزلدیلیب
- **اؤز-اؤزونه وئرن تاپیلماسی** — افزونه‌لر و تم‌لر فعال یا دا غیرفعال اولاندا موجود اولانلاری اؤز-اؤزونه تاپیر و یئنیلَییر
- **اؤزل لوگو یارادیلماسی** — اؤز-اؤزونه namespace-ایمن تمیزلمه ایله اؤزل SVG لوگولار یارادین و یئرلشدیرین
- **پالتا کنتراست یوخلاماسی** — تمینیزه اویغولامازدان قاباق رنگ جوتلرینی WCAG اریشیله‌بیلرلیک اویغونلوغونا گؤره یوخلایین
- **عکس‌له گؤجلندیریلمیش کشف** — Theme Builder کشف مصاحبه‌سی زامانی عکس یوکلَیین کی دیزاین یؤنونو اتکیله‌سین
- **مهمان‌داری منولاری** — رستورانلار، کافه‌لر و مهمان‌داری ایشلری اوچون قورولوشلو یئمک و ایچکی منو صحیفه‌لری یارادین
- **دسکتاپ و موبایل اؤن‌باکیشلاری** — دیزاین یؤنو سئچیلن زامان دیزاینینیزی دسکتاپ و موبایل دستگاهلاردا اؤن‌جَه گؤرون
- **1-جی سطح سایت قاشیماسی** — sd-ai-agent/site-scrape ایندی رقابَتچی و الهام وئره‌ن وب‌سایتلری تحلیل ائتمک اوچون پیش‌فرض اولاراق موجوددور
- **Google Calendar اوخوما آلتلری** — قورولموش تقویم‌لری و اولایلاری اوخویور کی اؤز-اؤزونه ایشله‌مه‌لر یاخین برنامه باغلامینی آنلایا بیلسین
- **علاقه و قاتیلان اویغونلاشدیرماسی** — ایزلَمه ایشی یاراتمازدان قاباق تقویم قاتیلانلارینی تانینان سایت ایستیفاده‌چیلری و علاقه‌لرله اویغونلاشدیرار
- **انسان تائیدی قاپیلاری** — پیام گؤندرمکدن، ایشی دوام ائتدیرمکدن یا دَییشیکلیک ائتمکدن قاباق حساس اؤز-اؤزونه ایشله‌مه‌لری باخیش اوچون ساخلا
- **یادآوری قئیدلری و تکرارسیزلاشدیرما** — آرتیق گؤندریلمیش یادآوریلاری ایزله کی تکرار اؤز-اؤزونه ایشله‌مه‌لر همی گیرَنه ایکی دفعه بیلدیریش گؤندرمسین
- **TextBee SMS بیلدیریشلری** — SMS وئرنی فعال اولاندا TextBee یولو ایله قورولموش متن پیام بیلدیریشلری گؤندَر
- **ایداره اولونان Superdav خدمتی قورولوشو** — ال ایله endpoint قورولوشو یئرینه، دستکلنن سایتلری اؤز-اؤزونه قورولوشلا میزبانلی Superdav endpoint-لرینه باغلا
- **ایرلی یانداش پاکت** — فایل سیستمی، داده‌بازاسی، WP-CLI، REST dispatcher، افزونه قوروجو، git snapshot، ایستیفاده‌چی ایداره‌سی و benchmark ایش آقیشلاری اوچون آیری یاییلان اینانیلان-گلیشدیرن آلتلری

## گرکلر

- WordPress 6.0 یا دا داها یوخاری
- PHP 8.1 یا دا داها یوخاری
- WordPress Multisite (شبکه‌دن فعال) یا دا تک-سایت قورولوشو
- فعال بیر AI وئرن باغلایان افزونه‌سی (Anthropic، OpenAI، یا اویغون)
- Gratis AI Agent v1.9.0 یا دا داها یوخاری (Superdav AI Agent، Gratis AI Agent-ی گئنیشلدیر)

## قوراشدیرما

1. افزونه فایللارینی `/wp-content/plugins/superdav-ai-agent` یولونا یوکلَیین
2. WordPress ایچینده **افزونه‌لر** صفحه‌سیندن افزونه‌نی فعال ائدین
3. Gratis AI Agent-ین ده فعال اولدوغونا امین اولون
4. **Gratis AI Agent → Settings** بؤلمه‌سینه گئدین و احتیاجا گؤره اینترنت آختاریش وئرنینیزی، ایداره اولونان Superdav خدمتی باغلانتیسینی، Google Calendar-ی یا TextBee کیملیک بیلگیلرینی قورون

WordPress.org پاکتی اساس Superdav AI Agent اوزللیکلرینی ایچریر. ایرلی یانداش پاکت، اینانیلان گلیشدیرن آلتلرینین آچیقجا قورولوب ایجازه وئریلدیگی محیط‌لر اوچون آیری قورولور و یاییلیر.

## تئز باشلانغیج

فعال اولاندان سونرا، Superdav AI Agent سیزین Gratis AI Agent قورولوشونوزو تم‌دن آگاه باجاریقلار و آرتیق قابلیتلرله اؤز-اؤزونه گؤجلندیرر. WordPress ایداره یان چوبوغوندا **Gratis AI Agent** پنلینی آچین و یئنی اوزللیکلردن ایستیفاده ائتمگه باشلایین.

**اؤرنک دستورلار:**

```
علاقه صفحه‌سینه بیر علاقه فرماسی آرتیر.
```

```
سون WordPress گوونلیک ان یاخشی عمللری اوچون وب‌ده آختار.
```

```
فعال تمین بلوک pattern-لریندن ایستیفاده ائدَرک بیر hero بؤلمه‌سی یارات.
```

```
صاباحکی قورولموش Google Calendar اولایلارینی یوخلا و قاتیلانلار اوچون یادآوری متنلری حاضیرلا.
```

```
من پیامى تائید ائتدیکدن سونرا بیر SMS یادآوریسی گؤندَر.
```

## تم‌دن آگاه باجاریقلار

Superdav AI Agent فعال تمینیزی و قورولموش افزونه‌لری اؤز-اؤزونه تاپیر، اویغون تم‌دن آگاه باجارغی یاردیمچی باغلامینا یوکلَییر. موجود باجاریقلار بونلاریدیر:

| باجاریق | بونا اویغولانار |
|---|---|
| **Block Themes** | WordPress بلوک-اساسلی (Full Site Editing) معماریسیندان ایستیفاده ائدن تم‌لر |
| **Classic Themes** | PHP قالب‌لاری و کلاسیک ویرایشچی‌دن ایستیفاده ائدن عنعنه‌وی WordPress تم‌لری |
| **Kadence Blocks** | Kadence Blocks افزونه‌سی فعال اولان سایتلر |
| **Kadence Theme** | فعال تمی Kadence اولان سایتلر |

هر باجاریق اوستونه آیرینتیلی یؤنلندیرمه اوچون [تم‌دن آگاه باجاریقلار](../../user-guide/configuration/theme-aware-skills) بؤلمه‌سینه باخین.

## اینترنت آختاریش وئرنلری

Superdav AI Agent v1.10.0 اینترنت آختاریش سورغولاری اوچون Brave Search-ه بدل اولاراق **Tavily** آرتیریر. هر ایکی وئرن حقیقی-زامانلی آختاریش نَتیجه‌لری وئریر و **Gratis AI Agent → Settings → Advanced** ایچینده قورولا بیلر.

| تأمین‌چی | اوزللیکلر | قورولوش |
|---|---|---|
| **Brave Search** | سورعتلی، گیزلیلک‌اوداقلی آختاریش نتیجه‌لری | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | داها زنگین آختاریش نتیجه‌لری، ایره‌لی آراشدیرما قابلیّت‌لری | [Tavily API](https://tavily.com/) |

قورولوش گؤستریشلری اوچون [اینتئرنت آختاریشی](../../user-guide/configuration/internet-search) بؤلومونه باخین.

## تقویم، اونای و بیلدیریش ایش آقیشلاری

Superdav AI Agent v1.18.0 اوتوماسیون ایش آقیشلارینی محتوا و site قورما ایشلریندن اوتاری گئنیشله‌دیر:

- **Google Calendar ابزارلاری** تقویم کیملیک بیلگی‌لری موجود اولاندا agent-ه قورولموش تقویم‌لری و اؤیونتی آیریتی‌لارینی اوخوماق ایمکانی وئریر.
- **ایلتیباط و قاتیلان کؤمکچی‌لری** یادآوری‌لار یا ایزلمه ایشلری حاضیرلامازدان قاباق اؤیونتی قاتیلانلارینی WordPress ایشلدن‌لرینه یا تانینان ایلتیباطلارا اویغونلاشدیرار.
- **ایnsan اونای قاپیلاری** بیر اوتوماسیونون، صلاحیّتلی ایشلدن تکلیف اولونان عمله باخانادک دایانماسینا ایمکان وئریر.
- **یادآوری قئیدلری** اوتوماسیون یئنیدن سیناناندا یا تکرارلی برنامه‌له‌نینجه تکراری تقویم یا بیلدیریش یادآوری‌لارینین قارشی‌سینی آلیر.
- **TextBee SMS دَسته‌یی** TextBee provider قورولاندا متن پیام بیلدیریش‌لرینی ایشله‌دیر.

بو ابزارلاری زمان‌بندی‌دن خبرلی کؤمک، قاتیلان ایزلمه‌سی و حسّاس پیام‌لار گؤندریلمزده‌ن قاباق هله‌ده ایnsan باخیشی ایسته‌یه‌ن یادآوری ایش آقیشلاری اوچون ایشلدین.

## Site Builder ایلتیباط فورملاری

Site Builder agent ایندی چت آراوزوندن بیر‌باشا ایلتیباط فورملاری یارادا بیلر. ساده‌جه Site Builder-دن بیر ایلتیباط فورمو یاراتماسینی ایسته‌یین؛ او، قورولموش form plugin-ینیزی (Contact Form 7، WPForms، Fluent Forms، یا Gravity Forms) اؤز-اؤزونه تانیش و ایستیفاده‌یه حاضیر فورم یاراداجاق.

آیریتی‌لی راهنما اوچون [Site Builder ایلتیباط فورملاری](../../user-guide/administration/site-builder-contact-forms) بؤلومونه باخین.

## Theme Builder اوزللیکلری

Superdav AI Agent v1.16.0-دا Theme Builder گوجلو یئنی قابلیّت‌لرله گلیر:

### عکس یوکله‌مه‌لی کشف مصاحیبه‌سی

Theme Builder کشف مصاحیبه‌سی ایندی عکس یوکله‌مه آدیمی‌نی ده ایچینه آلیر؛ بورادا گؤرسَل الهام وئره بیلرسیز. Theme Builder-ین دیزاین یؤنونوزو باشا دوشمه‌سینه کؤمک اوچون برندینیزین، رقیب site-لارین یا دیزاین الهامی عکس‌لارینی یوکله‌یین.

آیریتی‌لی راهنما اوچون [کشف مصاحیبه‌سی](./theme-builder/discovery-interview.md) بؤلومونه باخین.

### Desktop و Mobile اؤن‌باخیشلارلا دیزاین یؤنو

Theme Builder تام theme-ینیزی یاراتمازدان قاباق دیزاینینیزی هم desktop، هم ده mobile قورغولاردا گؤزدن کئچیرین. بیر نئچه responsive قیریلما نوقته‌سینده گئرچک‌زامان اؤن‌باخیشلارا باخین.

آیریتی‌لی راهنما اوچون [دیزاین یؤنو](./theme-builder/design-direction.md) بؤلومونه باخین.

### قوناق‌داری منولاری

Theme Builder ایندی رستورانلار، کافه‌لر، بارلار و باشقا قوناق‌داری ایشلری اوچون قورولوشلو یئمک و ایچکی menu صفحه‌لری یارادا بیلر. Menu-لارا قیمت‌لر، آچیقلامالار، پئرهیز بیلگی‌سی و آللئرگئن اویاريلاری داخیل‌دیر.

آیریتی‌لی راهنما اوچون [قوناق‌داری منولاری](./theme-builder/hospitality-menus.md) بؤلومونه باخین.

## قورولوش

بوتون تنظیم‌لر WordPress یؤنه‌تیم بؤلمه‌سینده **Gratis AI Agent → Settings** آلتین‌دادیر. Superdav AI Agent بو قورولوش سئچنه‌کلرینی آرتیرار:

| تنظیم | آچیقلاما |
|---|---|
| **Internet Search Provider** | Brave Search و Tavily آراسیندان سئچین (ایختیاری) |
| **Tavily API Key** | Tavily اینتئرنت آختاریشی اوچون API آچاری (اگر Tavily سئچیلیبسه) |
| **Managed Superdav Service** | دَسته‌کلنن site-لاری اوتوماتیک endpoint حاضرلاما ایله میزبان‌لی Superdav خیدمتینه باغلایین |
| **Google Calendar Credentials** | agent تقویم‌لری و اؤیونتی‌لری اوخویابیلسین دئیه تقویم گئریشینی قورون |
| **TextBee SMS Provider** | اونایلانمیش SMS بیلدیریش ایش آقیشلاری اوچون TextBee کیملیک بیلگی‌لرینی قورون |

## ایره‌لی یانداش بسته

Superdav AI Agent v1.18.0 ائتیبارلی توسعه‌چی ایش آقیشلاری اوچون آیری پایلانان Advanced یانداش بسته آرتیرار. بو قابلیّت‌لر WordPress.org اؤزک بسته‌سینه قوشولماییب. یانداش بسته‌نی آنجاق یؤنه‌دیجی‌لر agent-ین توسعه‌چی ابزارلارینا گئریشینه آچیقجا گؤونن site-لاردا قورون.

Advanced ابزارلارا فایل‌سیستم عمللری، بیلگی‌بانک گئریشی، WP-CLI، REST dispatcher ایش آقیشلاری، plugin builder یاردیمچی‌لاری، git snapshots، ایشلدن یؤنه‌تیمی کؤمکچی‌لری و benchmark ابزارلاری داخیل‌دیر. بو ابزارلاری ائتیبارلی یؤنه‌دیجی‌لرله محدود ساخلیین و production site-لاردا اوناخمازدان قاباق approval-gate قورولوشونو گؤزدن کئچیرین.
