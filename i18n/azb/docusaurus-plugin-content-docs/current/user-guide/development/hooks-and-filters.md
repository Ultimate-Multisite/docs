# Ultimate Multisite Hooks and filters

Ultimate Multisite is composed of a lot of functions and database queries. It has hooks and filters that let you modify or add new code to existing functions.

We created a list of these hooks and filters as a reference to help advanced users customize their network even further. You can access the list on Github by clicking the link below.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
