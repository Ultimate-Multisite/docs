# Codeable'dan Özel Geliştirme Yardımı Nasıl Alınır? {#how-to-get-custom-development-help-from-codeable}

Bazen ağımızın tam olarak istediğimiz gibi akması için harika bir özelleştirmeye ihtiyacımız olur. Eğer aradığınız şey bu ise, Ultimate Multisite uzmanlığına sahip deneyimli geliştiriciler bulabileceğiniz bir yer olan [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE)'ı tavsiye ederiz.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), müşterileri her büyüklükteki projede çalışabilecek nitelikli WordPress serbest geliştiricileriyle buluşturan bir platformdur. Aklınızdaki görev için özel olarak nitelikli uzmanlarla eşleştiğinizden emin olmak için titiz bir başvuru sürecine sahiptirler.

### Codeable'da bir proje nasıl oluşturulur? {#how-to-create-a-project-in-codeable}

Öncelikle [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE)'a gidin ve **Start a Project** düğmesine tıklayın.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Şimdi, bir proje göndermek için soldaki **Start a Project** seçeneğine tıklayın.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

"I need help with" (Yardım ihtiyacım) kısmında **Customizing** (Özelleştirme) seçeneğini seçin:

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

"My" (Benim) kısmında **Plugin** (Eklenti) seçeneğini seçin:

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Ardından **Continue** (Devam et) düğmesine tıklayın.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Şimdi, projenizi ve ihtiyaçlarınızı en iyi şekilde tarif etme zamanı. Mümkün olduğunca fazla bilgi vermeye çalışın!

<!-- Screenshot unavailable: Codeable project description form -->

Fiyatlandırmaya yardımcı olmak için aşağıdaki soruları yanıtlayın.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Bundan sonra, henüz bir [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) hesabınız yoksa, bir tane oluşturun ve **Publish My Project** (Projemizi Yayınla) düğmesine tıklayın.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) projenizi yayınladıktan sonra, platform sizi projeniz için gereken beceri setine uygun geliştiricilerle buluşturacaktır.

Bu süreç, WordPress ve Ultimate Multisite konusunda deneyimli, işiniz için doğru kişiyi bulmanıza yardımcı olacaktır!

Codeable'ın minimum ücreti saatte 70–120 USD'dir. Tahminler, işin kapsamına, karmaşıklığına ve projenin aciliyetine göre değişecektir.

Tahmininizi aldıktan sonra, ihtiyaçlarınıza en uygun geliştiriciyi işe alabilir ve projenize birlikte başlayabilirsiniz!
