# Header دامنه آختاریشی

بو قورولوشو او واخت ایشلدین کی، Header‌ده کیچیک بیر فرم دامنه آختاریشینا باشلاسین، سونرا موشتری Ultimate Multisite حسابلاشماسی ایچینده موجود دامنه‌لردن سئچه بیلسین.

## گرکلی‌لر {#requirements}

- Ultimate Multisite شبکه‌ده فعال اولمالیدیر.
- Multisite Ultimate Domain Seller شبکه‌ده فعال اولمالیدیر.
- ان آزی بیر فعال دامنه قئید محصولو اولمالیدیر:
  - `domain_provider` قورولموش بیر ساخلایجییا تنظیم اولسون.
  - دستکلنن TLD‌لر قورولسون، مثلن `com`، `net` و `org`.
- **دامنه سئچیمی** ساحه‌سینی ایچرن گئچرلی بیر حسابلاشما فرمی.

## حسابلاشما فرمی {#checkout-form}

1. قئید صفحه‌سینده ایشلنن حسابلاشما فرمینی یارادین یا دَییشدیرین.
2. عادی گرکلی حسابلاشما/Account ساحه‌لرینی، **قوللانیجی آدی** دا داخیل اولماقلا، آرتیرین. یالنیز دامنه ساحه‌سی اولان حسابلاشما فرمی Ultimate Multisite یوخلاماسی طرفیندن رد اولونور.
3. بیر **دامنه سئچیمی** ساحه‌سی آرتیرین.
4. آقیش آزاد آلت‌دامنه‌لر یا وار اولان دامنه‌لر یئرینه قئید اولونان دامنه‌لره یؤنلمه‌لیدیرسه، Domain Selection حالینی **یالنیز قئید** قویون.
5. دامنه قئید محصولونو او ساحه‌یه باغلایین.

قئید صفحه‌سی حسابلاشما فرمینی گؤسترمَلیدیر، مثلن:

```text
[wu_checkout slug="domain-form"]
```

## Header فرمی {#header-form}

سایت Header‌اینه کیچیک بیر `GET` فرمی آرتیرین کی، یازیلمیش آختاریش سؤزون `domain_name` کیمی حسابلاشما صفحه‌سینه گؤندرسین:

```html
<form class="ums-header-domain-search" action="/register/" method="get">
  <label class="screen-reader-text" for="ums-header-domain-search-input">Search for a domain</label>
  <input id="ums-header-domain-search-input" name="domain_name" type="text" placeholder="Find your domain" autocomplete="off">
  <button type="submit">Search</button>
</form>
```

اؤزل Header JavaScript‌اینده دامنه‌نی قاباقجادان سئچمه‌یین. Header یالنیز آختاریش سؤزون گؤندرمَلیدیر. Domain Seller حسابلاشما اسکریپتی `?domain_name=example` اوخور، حسابلاشما آختاریش قوتوسونو دولدورور و موجودلوق آختاریشینی ایشلدیر کی، موشتری قایتاریلان دامنه‌لردن سئچه بیلسین.

## گؤزلنن داورانیش {#expected-behaviour}

Header‌ده `example` آختاریشی بو صفحه‌نی آچمالیدیر:

```text
/register/?domain_name=example
```

سونرا حسابلاشما سئچیله بیلن نتیجه‌لری گؤسترمَلیدیر، مثلن:

- `example.com`
- `example.net`
- `example.org` موجود دئییل
- ساخلایجی طرفیندن دستکلنن باشقا TLD‌لر

موجود بیر نتیجه سئچیلندن سونرا، سیفاریش اؤزه‌تی دامنه قئید محصولونو و سئچیلن دامنه آدینی ایچرمَلیدیر.

## یوخلاما {#verification}

1. ائو صفحه‌سینی آچین.
2. یالین بیر آد آختارین، مثلن `example`.
3. حسابلاشما URL‌اینده `?domain_name=example` اولدوغونو دوغرولایین.
4. حسابلاشما دامنه ساحه‌سینده `example` اولدوغونو دوغرولایین.
5. دامنه سئچیم‌لری لیستینین گؤروندویونو دوغرولایین.
6. موجود بیر دامنه اوچون **سئچ** دئین.
7. سیفاریش اؤزه‌تینده `Domain Registration - example.com` یا سئچیلن دامنه اولدوغونو دوغرولایین.

## عیب‌یابی {#troubleshooting}

- هیچ لیست گؤرونمورسه، براوزرین شبکه تب‌ینده `admin-ajax.php?action=wu_domain_search` یوخلایین و بوش اولمایان `domains` یا `results` قایتاردیغینی دوغرولایین.
- حسابلاشما فرمی ساخلاما زامانی یوخلامادان کئچمیرسه، **قوللانیجی آدی** کیمی گرکلی Account ساحه‌لرینی آرتیرین.
- دامنه سئچمک سبدی ینیله‌میرسه، `window.wu_checkout_form` وارلیغینی و Domain Seller حسابلاشما فایل‌لرینین حسابلاشما صفحه‌سینده یوکلندیگینی دوغرولایین.
