# PRO Theme ile Admin Panel'inizi Özelleştirme

Müşteri projeleri yaparken, bazen müşterilerimize WordPress admin dashboard'unda özel bir deneyim sunmamız gerekebiliyor. Daha profesyonel bir görünüm ve his arıyorsanız, PRO Theme size yardımcı olabilir.

PRO Theme, müşterilerinizin Dashboard'unu özelleştirmenize yardımcı olmak için geliştirilmiş bir admin theme'idir. Ayrıca Ultimate Multisite'ın premium eklentileri çekirdeğinin bir parçasıdır.

Kurulumunu yaptıktan sonra, _Plugins_ sayfanıza gidin ve Network'inizde etkinleştirmek için tıklayın.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Plugins sayfanızdan network genelinde etkinleştirin_

_**NOT:**_ _Sadece kurarsanız, bu, müşterinizin sadece o kullanıcıya tema ekleyebileceği bir seçenek olarak görünecektir._

Artık admin arayüzünüz tamamen yeniden tasarlandı! Ayrıca soldaki sütunda PRO Theme adında yeni bir menüye sahipsiniz.
