# Ultimate Multisite ile Ağınızda Reklam Yerleştirme: Ad Injector {#placing-ads-on-your-network-with-ultimate-multisite-ad-injector}

Ultimate Multisite: Ad Injector, planlarınızın doğrudan düzenleme sayfasından reklam yerleştirmenizi sağlayan bir **ücretsiz eklentidir**.

Ultimate Multisite: Ad Injector'ı kurduktan sonra, eklentiyi etkinleştirmek için Eklentiler sayfanıza gidin.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Ağda Etkinleştir (Network Activate) düğmesine tıklayın_

Eklenti etkinleştirildikten sonra, plan düzenleme ekranındaki _Gelişmiş Seçenekler (Advanced Options)_ meta kutusunda _Reklam Ayarları (Ad Settings)_ adında yeni bir seçenek göreceksiniz.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_Artık her bir planınız için reklam ayarlarını düzenleyebilirsiniz_

Bu sekmede, ön yüzde (front-end) gönderiler ve sayfalar için, içeriğin öncesine ve sonrasına reklam kodlarını ekleyebilirsiniz.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Reklam kodunuzu ekledikten sonra, bu reklam o belirli planın gönderilerinde ve sayfalarında görünecektir._

Ayrıca, o planın kullanıcıları için Yönetici Sayfalarının (Admin Pages) başlığına da reklam ekleyebilirsiniz.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Kullanıcınızın arka yüzünde (back-end) reklam göstermek istiyorsanız, Yönetici Paneli Reklam Kodu (Admin Panel Ad Code) seçeneğine gidin_

### Reklamcılığın Ötesinde {#beyond-advertising}

Ultimate Multisite: Ad Injector'ı, üst düzey planlara destek sunmak için de kullanabilirsiniz.

Sadece _Yönetici Paneli Reklamları (Admin Panel Ads)_ seçeneğine gidip destek aracınız için HTML kodunu ekleyin.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Belirli planlara müşteri desteği ekleyin_

Bu sayede insanlar doğrudan Yönetici Panellerinden size ulaşabilir!
