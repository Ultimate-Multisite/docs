# İlk defa hesap sayfamıza nasıl ulaşabilirim?

Ultimate Multisite'a hoş geldiniz!

Eklentimizi satın aldıktan sonra e-postanıza gelen bağlantı üzerinden hesap sayfanıza ulaşabilirsiniz:

<!-- Screenshot unavailable: Account page email with access link -->

Satın alım sırasında verdiğiniz e-posta adresini ve e-postanızda aldığınız şifreyi kullanın.

<!-- Screenshot unavailable: Login page with email and password fields -->

Bu hesaba ilk kez giriş yaptığınız için, şifrenizi değiştirmeniz istenecektir. Şifrenizi değiştirin ve _Save_ düğmesine tıklayın.

<!-- Screenshot unavailable: Change password form with Save button -->

Hesap sayfanızda, faturalandırma, yenilemeler, iptaller ve diğer birçok şey dahil olmak üzere tüm hesap bilgilerinizi yönetebilirsiniz.
