# Müşterinizin Web Siteleri (v2)

Network'ünüze kaydolduktan sonra, müşterileriniz dashboard'larına erişebilirler.

Ödemeleri, üyelikleri, domain'leri, plan kısıtlamaları ve daha fazlasıyla ilgili önemli bilgileri görebilecekleri **Account** sayfasının yanı sıra, aynı zamanda tüm web sitelerini görebilecekleri ve yönetebilecekleri **My Sites** sayfasına da erişebilirler. Yapmaları gereken tek şey, dashboard'larındaki My Sites bağlantısına tıklamaktır.

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

Burada tüm web siteleri listelenecektir ve her birinin hemen altında bir **Manage** butonu bulunur. Bu butona tıkladıklarında, doğrudan **bu spesifik web sitesinin dashboard'una** yönlendirilirler.

Ayrıca, birden fazla siteye izin veren planlar için hangi sitelerinin **birincil (primary)** olduğunu da görebilirler.

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

Site düzenleme arayüzünün tam görünümü aşağıdadır:

![Site edit full interface](/img/admin/site-edit-full.png)

Birincil web sitelerini değiştirmek isterlerse, **Account** menüsüne tıklayarak Account sayfasına gitmeleri gerekir.

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

Ayrıca müşteri detaylarını görüntüleyebilir ve düzenleyebilirsiniz:

![Customer edit interface](/img/admin/customer-edit.png)

Ardından, **Change Default Site** butonuna tıklamalıdırlar.

![Change Default Site button on account page](/img/account-page/change-default-site.png)

Bir açılır pencere görünecektir. Açılır listeden, birincil web sitesi yapmak istediğiniz siteyi seçin.

![Change Default Site dropdown selection](/img/account-page/change-default-site-dropdown.png)

Ardından, sadece **Change Default Site** butonuna tıklamanız yeterli ve işlem tamamdır!

![Confirming the default site change](/img/account-page/change-default-site-confirm.png)
