# Agents ທີ່ມີໃຫ້ໃຊ້ໃນຕົວ

Gratis AI Agent v1.9.0 ມາພ້ອມ agents ທີ່ມີໃຫ້ໃຊ້ໃນຕົວຫ້າຕົວ, ແຕ່ລະຕົວຖືກຕັ້ງຄ່າລ່ວງໜ້າດ້ວຍຊຸດເຄື່ອງມືທີ່ເນັ້ນສະເພາະ, system prompt ທີ່ປັບໃຫ້ເໝາະ, ແລະຄຳແນະນຳເລີ່ມຕົ້ນທີ່ກົງກັບວຽກທົ່ວໄປໃນດ້ານນັ້ນ. ການສະຫຼັບລະຫວ່າງ agents ຈະປ່ຽນສິ່ງທີ່ຜູ້ຊ່ວຍສາມາດເຮັດໄດ້ ແລະວິທີທີ່ມັນຕອບກັບ — ໂດຍບໍ່ຕ້ອງມີການຕັ້ງຄ່າໃດໆຈາກຝ່າຍທ່ານ. Superdav AI Agent v1.18.0 ສາມາດເພີ່ມເຄື່ອງມືທີ່ຮູ້ກຳນົດເວລາ, ບັນທຶກການເຕືອນ, ດ່ານອະນຸມັດ, ແລະການແຈ້ງເຕືອນ SMS ເຂົ້າໃນຂະບວນວຽກເຫຼົ່ານີ້ໄດ້ ເມື່ອການເຊື່ອມຕໍ່ທີ່ກ່ຽວຂ້ອງຖືກຕັ້ງຄ່າແລ້ວ.

## Agent ແມ່ນຫຍັງ?

ແຕ່ລະ agent ແມ່ນໂປຣໄຟລ໌ການຕັ້ງຄ່າທີ່ມີຊື່ ເຊິ່ງລວມ:

- **ເຄື່ອງມື** — ຄວາມສາມາດທີ່ agent ໄດ້ຮັບອະນຸຍາດໃຫ້ເອີ້ນໃຊ້ (ເຊັ່ນ Content Writer ມີການເຂົ້າເຖິງຄວາມສາມາດສ້າງໂພສ; Design Studio ມີການເຂົ້າເຖິງ CSS ແລະ theme.json)
- **System prompt** — ຄຳແນະນຳທີ່ກຳນົດນ້ຳສຽງ, ລຳດັບຄວາມສຳຄັນ, ແລະຂໍ້ຈຳກັດຂອງ agent
- **ຄຳແນະນຳ** — prompt ທີ່ຂຽນໄວ້ລ່ວງໜ້າ ທີ່ສະແດງໃນອິນເຕີເຟດແຊັດ ເພື່ອຊ່ວຍໃຫ້ທ່ານເລີ່ມຕົ້ນໄດ້ຢ່າງໄວ

## ການເຂົ້າເຖິງ Agent Picker

1. ເປີດແຜງ **Gratis AI Agent** ໃນແຖບຂ້າງຜູ້ດູແລ WordPress.
2. ຄລິກ **ໄອຄອນ agent** ຢູ່ມຸມຊ້າຍເທິງຂອງຫົວຂໍ້ແຊັດ (ໄອຄອນຈະປ່ຽນເພື່ອສະທ້ອນ agent ທີ່ກຳລັງໃຊ້ຢູ່).
3. **Agent Picker** ຈະເປີດເປັນຊັ້ນທັບແບບຕາຕະລາງຟອມ. ແຕ່ລະ agent ຖືກລາຍຊື່ພ້ອມໄອຄອນ, ຊື່, ແລະຄຳອະທິບາຍແຖວດຽວ.
4. ຄລິກແຖວ agent ເພື່ອເປີດໃຊ້ງານ. ຫົວຂໍ້ແຊັດຈະອັບເດດທັນທີ.

ທ່ານຍັງສາມາດສະຫຼັບ agents ກາງການສົນທະນາໄດ້ — system prompt ຂອງ agent ໃໝ່ຈະມີຜົນຕັ້ງແຕ່ຂໍ້ຄວາມຖັດໄປ.

## Agents ທີ່ມີໃຫ້ໃຊ້ໃນຕົວຫ້າຕົວ

### Content Writer

**ຈຸດເນັ້ນ:** ການສ້າງແລະແກ້ໄຂໂພສ, ໜ້າ, ແລະແບບຟອມຕິດຕໍ່.

**ເຄື່ອງມືທີ່ມີ:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`. ເມື່ອເປີດໃຊ້ການເຊື່ອມຕໍ່ Superdav AI Agent v1.18.0, ບໍລິບົດປະຕິທິນທີ່ຕັ້ງຄ່າແລ້ວ, ດ່ານອະນຸມັດ, ການເຕືອນ, ແລະເຄື່ອງມືແຈ້ງເຕືອນ SMS ອາດພ້ອມໃຫ້ໃຊ້ສຳລັບຂະບວນວຽກທີ່ໄດ້ຮັບອະນຸມັດ.

**ສິ່ງທີ່ເຮັດໄດ້ດີ:**
- ຮ່າງແລະເຜີຍແຜ່ໂພສບລັອກຈາກ brief ຫຼືໂຄງຮ່າງ
- ສ້າງຊຸດໜ້າ landing ສຳລັບເວັບໄຊໃໝ່
- ສ້າງແບບຟອມຕິດຕໍ່ແລະສອບຖາມ
- ຕັ້ງຮູບເດັ່ນໃນໂພສຈາກ URL ຫຼືການຄົ້ນຫາ
- ຮ່າງຂໍ້ຄວາມຕິດຕາມຫຼັງງານຈາກບໍລິບົດ Google Calendar ທີ່ຕັ້ງຄ່າແລ້ວ, ແລ້ວຢຸດພັກເພື່ອລໍຖ້າອະນຸມັດກ່ອນສົ່ງການແຈ້ງເຕືອນ

**ຄຳແນະນຳເລີ່ມຕົ້ນ:**
- *ຂຽນໂພສບລັອກ 500 ຄຳ ກ່ຽວກັບປະໂຫຍດຂອງ WordPress multisite.*
- *ສ້າງໜ້າ ກ່ຽວກັບ, ບໍລິການ, ແລະຕິດຕໍ່ ແລ້ວເຜີຍແຜ່.*
- *ເພີ່ມແບບຟອມສອບຖາມການຈອງໃສ່ໜ້າຕິດຕໍ່.*
- *ຮ່າງການເຕືອນສຳລັບຜູ້ເຂົ້າຮ່ວມໃນງານປະຕິທິນທີ່ຕັ້ງຄ່າໄວ້ຂອງມື້ອື່ນ ແລະລໍຖ້າອະນຸມັດກ່ອນສົ່ງ.*

### Site Builder

**ຈຸດເນັ້ນ:** ການສ້າງເວັບໄຊຕັ້ງແຕ່ຕົ້ນຈົນຈົບຈາກ prompt ດຽວ.

**ເຄື່ອງມືທີ່ມີ:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`. ດ້ວຍ Superdav AI Agent v1.18.0, ເຄື່ອງມືບໍລິການທີ່ມີການຈັດການ, ການອະນຸມັດ, ການເຕືອນ, ປະຕິທິນ, ແລະ SMS ທີ່ຕັ້ງຄ່າແລ້ວ ອາດພ້ອມໃຫ້ໃຊ້ໃນບ່ອນທີ່ຜູ້ດູແລເປີດໃຊ້.

**ສິ່ງທີ່ເຮັດໄດ້ດີ:**
- ສ້າງແຜນການສ້າງເວັບໄຊຫຼາຍໄລຍະສຳລັບປະເພດທຸລະກິດທີ່ອະທິບາຍໄວ້
- ດຳເນີນການແຕ່ລະໄລຍະແບບອັດຕະໂນມັດ — ໂຄງສ້າງ, ເນື້ອຫາ, ການນຳທາງ, ການອອກແບບ
- ຟື້ນຕົວຈາກຂໍ້ຜິດພາດກາງແຜນ ໂດຍບໍ່ຕ້ອງການການແຊກແຊງດ້ວຍມື
- ຕິດຕັ້ງ plugins ທີ່ແນະນຳເປັນສ່ວນໜຶ່ງຂອງການສ້າງ
- ສ້າງແບບຟອມຕິດຕໍ່ໂດຍກົງຈາກອິນເຕີເຟດແຊັດ (Superdav AI Agent v1.10.0+)
- ປະສານງານການເຕືອນການເປີດຕົວ ຫຼືການຕິດຕາມຜູ້ເຂົ້າຮ່ວມ ໂດຍບໍ່ມີການແຈ້ງເຕືອນຊ້ຳ ເມື່ອດ່ານອະນຸມັດແລະບັນທຶກການເຕືອນຖືກເປີດໃຊ້

**ຄຳແນະນຳເລີ່ມຕົ້ນ:**
- *ສ້າງເວັບໄຊພອດໂຟລິໂອການຖ່າຍຮູບ ພ້ອມປະເພດໂພສຄັງຮູບ, ໜ້າຈອງ, ແລະແບບຟອມຕິດຕໍ່.*
- *ສ້າງເວັບໄຊຮ້ານອາຫານ ພ້ອມເມນູອອນລາຍ, ເວລາເປີດ, ແລະແບບຟອມສອບຖາມການຈອງໂຕະ.*
- *ຕັ້ງຄ່າເວັບໄຊທີ່ປຶກສາອິດສະລະ ພ້ອມໜ້າບໍລິການ, ສ່ວນພອດໂຟລິໂອ, ແລະບລັອກ.*
- *ເພີ່ມແບບຟອມຕິດຕໍ່ໃສ່ໜ້າຕິດຕໍ່ ໂດຍໃຊ້ site builder.*
- *ຫຼັງຈາກລາຍການກວດກາການເປີດຕົວເວັບໄຊຖືກອະນຸມັດ, ສົ່ງການເຕືອນ SMS ໄປຫາຜູ້ຕິດຕໍ່ຜູ້ມີສ່ວນໄດ້ສ່ວນເສຍທີ່ຕັ້ງຄ່າໄວ້.*

### Design Studio

**ຈຸດເນັ້ນ:** ການປັບແຕ່ງພາບລັກ — ສີ, ຕົວພິມ, CSS, ແລະແບບແຜນ block.

**ເຄື່ອງມືທີ່ມີ:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**ສິ່ງທີ່ເຮັດໄດ້ດີ:**
- ນຳໃຊ້ preset ຂອງ theme ທີ່ມີຊື່ (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- ປັບແຕ່ງຕົວພິມລວມ ແລະຊຸດສີຢ່າງລະອຽດຜ່ານ theme.json
- ໃສ່ CSS ກຳນົດເອງສຳລັບການປັບທັບທີ່ເຈາະຈົງຕາມແບຣນ
- ຖ່າຍພາບໜ້າຈໍຂອງໜ້າໜຶ່ງ ແລະກວດທົບທວນຫາບັນຫາດ້ານການອອກແບບ

**ຄຳແນະນຳເລີ່ມຕົ້ນ:**
- *ນຳໃຊ້ preset warm-editorial ແລ້ວຕັ້ງສີຫຼັກເປັນ #2d6a4f.*
- *ຖ່າຍພາບໜ້າຈໍຂອງໜ້າຫຼັກ ແລະບອກຂ້ອຍວ່າເຈົ້າຈະປັບປຸງຫຍັງ.*
- *ສ້າງແບບແຜນ hero block ທີ່ນຳກັບໃຊ້ໄດ້ ພ້ອມຮູບພື້ນຫຼັງເຕັມຄວາມກວ້າງ ແລະຫົວຂໍ້ຢູ່ກາງ.*

### Plugin Manager

**ຈຸດເນັ້ນ:** ການຄົ້ນພົບ, ຕິດຕັ້ງ, ແລະຈັດການ WordPress plugins.

**ເຄື່ອງມືທີ່ມີ:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**ສິ່ງທີ່ເຮັດໄດ້ດີ:**
- ແນະນຳ plugin ທີ່ດີທີ່ສຸດສຳລັບກໍລະນີການໃຊ້ງານທີ່ອະທິບາຍໄວ້
- ຕິດຕັ້ງຊຸດຄວາມສາມາດຈາກທະບຽນ
- ເບິ່ງ catalogue ຄວາມສາມາດທີ່ມີຕາມໝວດໝູ່

**ຄຳແນະນຳເລີ່ມຕົ້ນ:**
- *Plugin ໃດດີທີ່ສຸດສຳລັບໄດເຣັກທໍຣີສະມາຊິກ?*
- *ຕິດຕັ້ງຊຸດຄວາມສາມາດ WooCommerce.*
- *ສະແດງຊຸດຄວາມສາມາດ ecommerce ທີ່ມີທັງໝົດໃຫ້ຂ້ອຍເບິ່ງ.*

- ຕອບຄຳຖາມ "ການຕັ້ງຄ່ານີ້ເຮັດຫຍັງ?" ໂດຍການອ່ານຄ່າຈິງທີ່ກຳລັງໃຊ້ງານ

## ການຜະສານງານອັດຕະໂນມັດຂອງ Superdav

ເມື່ອການຜະສານງານ Superdav AI Agent v1.18.0 ຖືກກຳນົດຄ່າແລ້ວ, ຕົວແທນທີ່ມີໃນຕົວສາມາດເຂົ້າຮ່ວມໃນເວີກໂຟລວອັດຕະໂນມັດທີ່ປອດໄພກວ່າ ແລະ ຮັບຮູ້ຕາຕະລາງໄດ້:

- **ເຄື່ອງມືອ່ານ Google Calendar** ອະນຸຍາດໃຫ້ຕົວແທນກວດກາປະຕິທິນ ແລະ ເຫດການທີ່ຖືກກຳນົດຄ່າໄວ້ ກ່ອນຮ່າງວຽກຕິດຕາມ.
- **ການຈັບຄູ່ຜູ້ຕິດຕໍ່ ແລະ ຜູ້ເຂົ້າຮ່ວມ** ຊ່ວຍຈັບຄູ່ຜູ້ຮ່ວມເຫດການກັບຜູ້ໃຊ້ WordPress ຫຼື ຜູ້ຕິດຕໍ່ທີ່ຮູ້ຈັກ.
- **ດ່ານການອະນຸມັດໂດຍມະນຸດ** ຢຸດການກະທຳທີ່ອ່ອນໄຫວໄວ້ຊົ່ວຄາວ ຈົນກວ່າຜູ້ໃຊ້ທີ່ໄດ້ຮັບອຳນາດຈະກວດທານ ແລະ ຢືນຢັນ.
- **ບັນທຶກການເຕືອນ** ປ້ອງກັນການແຈ້ງເຕືອນຊ້ຳ ເມື່ອວຽກທີ່ຕັ້ງຕາຕະລາງໄວ້ລອງໃໝ່ ຫຼື ເຮັດຊ້ຳ.
- **ການແຈ້ງເຕືອນ TextBee SMS** ສົ່ງຂໍ້ຄວາມຕາມທີ່ກຳນົດໄວ້ ກໍ່ຕໍ່ເມື່ອຂໍ້ມູນຮັບຮອງ SMS ແລະ ສິດອະນຸຍາດເວີກໂຟລວຖືກເປີດໃຊ້.

ເວີກໂຟລວທີ່ແນະນຳ: ຂໍໃຫ້ຕົວແທນກະກຽມຂໍ້ຄວາມ ຫຼື ການກະທຳ, ກວດທານຄຳຂໍອະນຸມັດ, ແລ້ວອະນຸຍາດໃຫ້ການກະທຳທີ່ອະນຸມັດແລ້ວດຳເນີນຕໍ່. ສຳລັບການເຕືອນທີ່ເກີດຊ້ຳ, ໃຫ້ເປີດໃຊ້ການຕັດການຊ້ຳຂອງການເຕືອນໄວ້ ເພື່ອບໍ່ໃຫ້ເຫດການ ຫຼື ຜູ້ຕິດຕໍ່ດຽວກັນຖືກແຈ້ງເຕືອນຊ້ຳໆ.

## ການປັບແຕ່ງຕົວແທນ

ຕົວແທນທີ່ມີໃນຕົວແຕ່ລະຕົວສາມາດຂະຫຍາຍ ຫຼື ແທນທີ່ໄດ້ຜ່ານ filter `gratis_ai_agent_agents`.

### ເພີ່ມ system prompt ກຳນົດເອງໃຫ້ຕົວແທນທີ່ມີຢູ່

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### ລົງທະບຽນຕົວແທນໃໝ່

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

ຕົວແທນໃໝ່ຈະປາກົດໃນຕົວເລືອກຕົວແທນທັນທີຫຼັງຈາກ filter ເຮັດວຽກ.

### ລຶບຕົວແທນທີ່ມີໃນຕົວອອກ

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
