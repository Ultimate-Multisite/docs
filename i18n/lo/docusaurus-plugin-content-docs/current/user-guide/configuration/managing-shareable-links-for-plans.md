# ການຈັດການລິ້ງທີ່ແບ່ງປັນໄດ້ສຳລັບແຜນການຕ່າງໆ {#managing-shareable-links-for-plans}

Ultimate Multisite ໃຫ້ທາງເລືອກກັບເຈົ້າໃນການແບ່ງປັນລິ້ງຂອງແຜນການໂດຍກົງກັບລູກຄ້າຂອງເຈົ້າ. ເພື່ອເຮັດແບບນັ້ນ, ໄປທີ່ໜ້າຕ່າງການແກ້ໄຂ (edit pages) ຂອງແຜນການໃດໜຶ່ງຂອງເຈົ້າ ແລ້ວໃຫ້ກົດປຸ່ມ _Click to copy the Shareable Link_ (ກົດເພື່ອຄັດລອກລິ້ງທີ່ແບ່ງປັນໄດ້).

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_ໃນໜ້າຕ່າງການແກ້ໄຂຂອງແຜນການ, ເຈົ້າຈະພົບປຸ່ມສຳລັບການສ້າງລິ້ງທີ່ແບ່ງປັນໄດ້_.

ໃຫ້ສັງເກດວ່າ ກົນໄກນີ້ໃຊ້ຄວາມຖີ່ການເກັບຄ່າທໍານຽມ (billing frequency) ໂດຍຄ່າເລີ່ມຕົ້ນເພື່ອສ້າງລິ້ງ. ນີ້ໝາຍຄວາມວ່າ ຖ້າການເກັບຄ່າທໍານຽມຄ່າເລີ່ມຕົ້ນຂອງເຈົ້າແມ່ນປະຈຳເດືອນ, ສຸດທ້າຍແລ້ວເຈົ້າຈະມີລິ້ງແບບນີ້: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

ຖ້າເຈົ້າຕ້ອງການໃຫ້ລິ້ງທີ່ແບ່ງປັນໄດ້ເປັນຄ່າເລີ່ມຕົ້ນປະຈຳໄຕມາດ (quarterly), ເຈົ້າຈະຕ້ອງເພີ່ມ 3 ນັ້ນໄວ້ທ້າຍ:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

ເພື່ອເລືອກແຜນການ _premium_ ໃນຕາຕະລາງປະຈຳປີ, ເຈົ້າຈະເພີ່ມ 1 ແລ້ວຕາມດ້ວຍຄຳວ່າ:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

ຄວນຈື່ໄວ້ວ່າ ເຈົ້າບໍ່ສາມາດມີລິ້ງດຽວສຳລັບຄວາມຖີ່ການເກັບຄ່າທໍານຽມທີ່ແຕກຕ່າງກັນໄດ້. ເຈົ້າຄວນແບ່ງປັນລິ້ງສະເພາະໜຶ່ງໃຫ້ກັບຄວາມຖີ່ການເກັບຄ່າທໍານຽມຂອງເຈົ້າແຕ່ລະຢ່າງ.

ເມື່ອເຈົ້າແບ່ງປັນລິ້ງກັບລູກຄ້າຂອງເຈົ້າແລ້ວ, ມັນຈະນຳພວກເຂົາໄປສູ່ຂັ້ນຕອນທຳອິດຂອງການลงทะเบียน (signup flow) ຂອງເຈົ້າ.
