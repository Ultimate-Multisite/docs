# ການຄົ້ນຫາໃນອິນເຕີເນັດ

Gratis AI Agent v1.5.0 ເພີ່ມຄວາມສາມາດໃນການ **Internet Search** (ການຄົ້ນຫາໃນອິນເຕີເນັດ) ເຊິ່ງເຮັດໃຫ້ຜູ້ຊ່ວຍ AI ສາມາດດຶງຂໍ້ມູນທີ່ທັນສະໄໝຈາກເວັບໄດ້ໃນລະຫວ່າງການສົນທະນາ. ນີ້ແມ່ນໃຊ້ງານໂດຍ [Brave Search API](https://brave.com/search/api/) ຫຼື [Tavily API](https://tavily.com/).

## ມັນເຮັດວຽກແນວໃດ {#how-it-works}

ເມື່ອເປີດໃຊ້ການຄົ້ນຫາໃນອິນເຕີເນັດ, ຜູ້ຊ່ວຍຈະສາມາດສອບຖາມຜູ້ໃຫ້ບໍລິການຄົ້ນຫາທີ່ທ່ານຕັ້ງຄ່າໄວ້ໂດຍອັດຕະໂນມັດ ເມື່ອມັນຮູ້ວ່າຄຳຖາມນັ້ນຕ້ອງການຂໍ້ມູນທີ່ທັນສະໄໝ ຫຼື ຂໍ້ມູນພາຍນອກ — ຕົວຢ່າງເຊັ່ນ: ຂ່າວຫຼ້າສຸດ, ລາຄາສົດໆ, ຫຼື ບົດບັນຍັດທີ່ອາດຈະປ່ຽນແປງຕັ້ງແຕ່ເວລາທີ່ model ຖືກຝຶກມາ.

ຜົນການຄົ້ນຫາຈະຖືກດຶງມາແບບທັນທີ ແລະ ເຂົ້າໄປໃນ context (ສະພາບແວດລ້ອມ) ຂອງຜູ້ຊ່ວຍ ກ່ອນທີ່ຈະສ້າງຄຳຕອບ. ຜູ້ຊ່ວຍຈະບອກໃຫ້ຮູ້ເມື່ອມັນໄດ້ໃຊ້ຜົນການຄົ້ນຫາເພື່ອຕອບຄຳຖາມນັ້ນແລ້ວ.

## ການເປີດໃຊ້ Internet Search {#enabling-internet-search}

ການຄົ້ນຫາໃນອິນເຕີເນັດ ຮຽກຮ້ອງ API key ຈາກຜູ້ໃຫ້ບໍລິການຄົ້ນຫາທີ່ທ່ານເລືອກ. ເພື່ອຕັ້ງຄ່າ:

1. ໄປທີ່ **Gratis AI Agent → Settings → Advanced**.
2. ຊອກຫາ dropdown **Internet Search Provider** ແລະ ເລືອກລະຫວ່າງ **Brave Search** ຫຼື **Tavily**.
3. ໃສ່ API key ຂອງທ່ານໃນຊ່ອງທີ່ກ່ຽວຂ້ອງ. ລິ້ງການສະໝັກໃຊ້ຈະສະແດງຢູ່ຂ້າງແຕ່ລະຊ່ອງ.
4. ກົດ **Save Settings**.

ເມື່ອບັນທຶກ key ແລ້ວ, ຄວາມສາມາດ Internet Search ຈະມີໃຫ້ຜູ້ຊ່ວຍໂດຍອັດຕະໂນມັດ.

## Brave Search {#brave-search}

### ການໄດ້ຮັບ Brave Search API Key {#obtaining-a-brave-search-api-key}

1. ເຂົ້າໄປທີ່ [Brave Search API page](https://brave.com/search/api/).
2. ລົງທະບຽນແຜນການໃຊ້ງານ. ມີລະດັບຟຣີໃຫ້ໃຊ້ໂດຍມີຈຳກັດການຮ້ອງຂໍຕໍ່ເດືອນ.
3. ຄັດລອກ Brave Search API key ຂອງທ່ານຈາກ developer dashboard.
4. ວາງມັນໃສ່ໃນຊ່ອງ **Brave Search API Key** ໃນການຕັ້ງຄ່າຂອງ Gratis AI Agent.

### ຂີດຈຳກັດການນຳໃຊ້ {#usage-limits}

การใช้งานจะถูกเรียกเก็บเงินโดย Brave Search ตามจำนวนคำค้นหาที่ทำไป แต่ทุกๆ การตอบกลับของ AI ที่กระตุ้นให้มีการค้นหานับเป็นหนึ่งคำค้นหา คุณสามารถตรวจสอบการใช้งานของคุณได้ที่ [Brave Search developer dashboard](https://brave.com/search/api/) เพื่อหลีกเลี่ยงค่าใช้จ่ายที่ไม่คาดคิด

## Tavily {#tavily}

Superdav AI Agent v1.10.0 ได้เพิ่ม **Tavily** เป็นตัวเลือกสำหรับผู้ให้บริการค้นหาอินเทอร์เน็ต ซึ่งให้ผลการค้นหาที่ละเอียดกว่าและความสามารถในการวิจัยขั้นสูง

### การรับ Tavily API Key {#obtaining-a-tavily-api-key}

1. ไปที่ [Tavily API page](https://tavily.com/)
2. สมัครบัญชี คุณสามารถใช้ระดับฟรีที่มีขีดจำกัดจำนวนคำขอต่อเดือนได้
3. คัดลอก API key ของคุณจากแดชบอร์ดของ Tavily
4. วางคีย์นั้นลงในช่อง **Tavily API Key** ในการตั้งค่า Gratis AI Agent

### ขีดจำกัดการใช้งาน {#usage-limits-1}

การใช้งานจะถูกเรียกเก็บเงินโดย Tavily ตามจำนวนครั้งของการเรียกใช้ API แต่ละการตอบกลับของ AI ที่กระตุ้นให้มีการค้นหานับเป็นการเรียกใช้หนึ่งครั้ง คุณสามารถตรวจสอบการใช้งานของคุณได้ที่ [Tavily dashboard](https://tavily.com/) เพื่อหลีกเลี่ยงค่าใช้จ่ายที่ไม่คาดคิด

## การปิดการค้นหาอินเทอร์เน็ต {#disabling-internet-search}

นำ API key ออกจากช่องผู้ให้บริการค้นหาที่ใช้งานอยู่แล้วบันทึก จากนั้นความสามารถในการค้นหาอินเทอร์เน็ตจะไม่ถูกเสนอให้ผู้ช่วยอีกต่อไป

:::note
การค้นหาอินเทอร์เน็ตจะทำให้เกิดความล่าช้าในการตอบกลับ เพราะผู้ช่วยจะต้องรอผลการค้นหาก่อนที่จะสร้างคำตอบ สำหรับกรณีที่ต้องการข้อมูลแบบเรียลไทม์ ให้พิจารณาว่าจำเป็นต้องมีการค้นหาแบบเรียลไทม์หรือไม่ หรือความรู้ที่มีมาพร้อมกับผู้ช่วยก็เพียงพอแล้ว
:::
