# ทักษะที่รู้เรื่องธีม (Theme-Aware Skills)

Superdav AI Agent v1.10.0 มีทักษะใหม่ 4 อย่างที่ฉลาดและปรับตัวตามธีม WordPress ที่คุณใช้งานอยู่โดยอัตโนมัติ ทักษะเหล่านี้จะให้คำแนะนำและความสามารถเฉพาะทางที่ออกแบบมาเพื่อสถาปัตยกรรมและฟีเจอร์ของธีมของคุณโดยเฉพาะ

## ทักษะที่รู้เรื่องธีมคืออะไร? {#what-are-theme-aware-skills}

ทักษะที่รู้เรื่องธีม คือชุดความรู้พื้นฐาน (knowledge bases) และชุดเครื่องมือ (tool sets) ที่ถูกตั้งค่าไว้ล่วงหน้า ซึ่งผู้ช่วย AI จะเลือกมาให้โดยอัตโนมัติตามธีม WordPress ที่กำลังใช้งานอยู่บนเว็บไซต์ของคุณ เมื่อคุณเปลี่ยนธีม ทักษะที่มีให้ของผู้ช่วยก็จะอัปเดตโดยอัตโนมัติ ไม่ต้องตั้งค่าด้วยตัวเองเลย

แต่ละทักษะจะประกอบด้วย:

- **เอกสารเฉพาะสำหรับธีม** — คำแนะนำเกี่ยวกับการใช้งานและการปรับแต่งธีม
- **การอ้างอิงบล็อกและรูปแบบ (Block and pattern references)** — บล็อกที่มีให้, รูปแบบ และตัวเลือกการออกแบบต่างๆ
- **ตัวอย่างการปรับแต่ง (Customization examples)** — ส่วนของโค้ดและรูปแบบการตั้งค่าสำหรับงานทั่วไป
- **แนวทางปฏิบัติที่ดีที่สุด (Best practices)** — คำแนะนำเกี่ยวกับสถาปัตยกรรมและขั้นตอนการทำงานของธีม

## ทักษะที่รู้เรื่องธีมที่มีให้ใช้งาน {#available-theme-aware-skills}

### ธีมแบบบล็อก (Block Themes) {#block-themes}

**ใช้กับ:** ธีมที่ใช้สถาปัตยกรรมแบบบล็อกของ WordPress (Full Site Editing)

ทักษะ Block Themes จะให้คำแนะนำเกี่ยวกับ:

- การสร้างและแก้ไขเทมเพลตโดยใช้ block editor
- การทำงานกับ block patterns และ reusable blocks
- การปรับแต่งสไตล์ทั่วโลก (global styles) และการตั้งค่า theme.json
- การใช้ theme blocks และ variations
- การสร้าง custom block patterns สำหรับเว็บไซต์ของคุณ

**จะเปิดใช้งานโดยอัตโนมัติเมื่อ:** ธีมที่กำลังใช้งานอยู่เป็นธีมแบบบล็อก (รองรับฟีเจอร์ `block-templates`)

### ธีมแบบคลาสสิก (Classic Themes) {#classic-themes}

**ใช้กับ:** ธีม WordPress แบบดั้งเดิมที่ใช้ PHP templates และ classic editor

ทักษะ Classic Themes จะให้คำแนะนำเกี่ยวกับ:

การทำงานกับไฟล์ template ของ PHP และ hooks
การปรับแต่งรูปลักษณ์ของธีมผ่าน Customizer
การใช้ widget areas และ sidebars
การแก้ไข CSS และการพัฒนา child theme
การทำความเข้าใจลำดับชั้นของธีมและ template tags

**จะเปิดใช้งานโดยอัตโนมัติเมื่อ:** ธีมที่ใช้งานอยู่ของคุณเป็นธีมแบบคลาสสิก (ไม่ใช่แบบ block)

### Kadence Blocks {#kadence-blocks}

**ใช้ได้กับ:** เว็บไซต์ที่ใช้ปลั๊กอิน Kadence Blocks สำหรับการออกแบบขั้นสูงด้วยบล็อก

ทักษะ Kadence Blocks จะให้คำแนะนำเกี่ยวกับ:

- การใช้ไลบรารีบล็อกขั้นสูงของ Kadence (เช่น Hero, Testimonials, Pricing ฯลฯ)
- การตั้งค่าการตั้งค่าบล็อกและตัวเลือกที่ตอบสนองต่อหน้าจอ (responsive options) ของ Kadence block
- การสร้างเลย์เอาต์แบบกำหนดเองด้วยบล็อก grid และ container ของ Kadence
- การรวม Kadence blocks เข้ากับธีมของคุณ
- การใช้ประโยชน์จาก design system และ presets ของ Kadence

**จะเปิดใช้งานโดยอัตโนมัติเมื่อ:** ปลั๊กอิน Kadence Blocks เปิดใช้งานอยู่บนเว็บไซต์ของคุณ

### Kadence Theme {#kadence-theme}

**ใช้ได้กับ:** เว็บไซต์ที่ใช้ Kadence theme สำหรับการออกแบบและการปรับแต่งด้วยบล็อก

ทักษะ Kadence Theme จะให้คำแนะนำเกี่ยวกับ:

- การปรับแต่งธีม Kadence ผ่าน global styles และ theme.json
- การใช้ block patterns และ templates ที่มีมาในตัวของ Kadence
- การตั้งค่าและตัวเลือกของ Kadence theme
- การสร้างดีไซน์แบบกำหนดเองด้วย design system ของ Kadence
- การรวม Kadence เข้ากับปลั๊กอินและเครื่องมือยอดนิยม

**จะเปิดใช้งานโดยอัตโนมัติเมื่อ:** ธีม Kadence เป็นธีมที่ใช้งานอยู่ของคุณ

## วิธีการเลือก Skills (ทักษะ) {#how-skills-are-selected}

ผู้ช่วยจะตรวจจับธีมที่ใช้งานอยู่และปลั๊กอินที่ติดตั้งไว้บนแต่ละข้อความโดยอัตโนมัติ หากมี skill ที่เกี่ยวข้องกับธีมตรงกัน ระบบจะโหลดเข้าไปใน context ของผู้ช่วยโดยอัตโนมัติ คุณไม่จำเป็นต้องเปิดหรือสลับ skill ด้วยตนเอง

### Multiple Skills (หลายทักษะ) {#multiple-skills}

ຖ້າทักษะหลายอย่างเกี่ยวข้องกับเว็บไซต์ของคุณ (ຕົວຢ່າງ, ប្រសិនបើអ្នកប្រើទាំង Kadence Blocks និង Kadence Theme) អ្នកជំនួយការនឹងអាចចូលប្រើທຸກທັກສະທີ່ນຳໃຊ້ໄດ້ ແລະ ສາມາດອ້າງອີງຄຳແນະນຳຈາກທຸກຢ່າງໄດ້.

### ການປ່ຽນເນື້ອຫາ (Switching Themes) {#switching-themes}

ເມື່ອທ່ານປ່ຽນເນື້ອຫາ (theme) ທີ່ກຳລັງໃຊ້ຢູ່, ທັກສະທີ່ມີໃຫ້ຂອງຜູ້ຊ່ວຍຈະອັບເດດໂດຍອັດຕະໂນມັດໃນຂໍ້ຄວາມຕໍ່ໄປ. ຕົວຢ່າງ:

1. ທ່ານກຳລັງໃຊ້ theme ແບບ block ແລະ ມີທັກສະ **Block Themes** ເປັນທີ່ເປີດໃຊ້ຢູ່.
2. ທ່ານປ່ຽນໄປໃຊ້ theme ແບບ classic.
3. ໃນຂໍ້ຄວາມຕໍ່ໄປຂອງທ່ານ, ທັກສະ **Classic Themes** ຈະຖືກໂຫຼດໂດຍອັດຕະໂນມັດ ແລະ ທັກສະ **Block Themes** ຈະບໍ່ມີໃຫ້ໃຊ້ອີກ.

## ການໃຊ້ທັກສະທີ່ຮູ້ຈັກ về Theme (Using Theme-Aware Skills) {#using-theme-aware-skills}

ເພື່ອໃຊ້ປະໂຫຍດຈາກທັກສະທີ່ຮູ້ຈັກ về theme, ພຽງແຕ່ອະທິບາຍວ່າທ່ານຕ້ອງການເຮັດຫຍັງໃນ giao diệnແຊັດ. ຜູ້ຊ່ວຍຈະອ້າງອີງຄຳແນະນຳຂອງທັກສະທີ່ເໝາະສົມໂດຍອັດຕະໂນມັດ.

### ຕົວຢ່າງການສັ່ງງານ (Example Prompts) {#example-prompts}

**ສຳລັບ Block Themes:**
> "ສ້າງ hero section ທີ່ມີຮູບພື້ນຫຼັງ ແລະ ເນື້ອໃນຢູ່ກາງໂດຍໃຊ້ block patterns."

**ສຳລັບ Classic Themes:**
> "ເພີ່ມ area widget ທີ່ເປັນແບບ custom ໃສ່ sidebar ໂດຍໃຊ້ child theme."

**ສຳລັບ Kadence Blocks:**
> "ສ້າງ section testimonial ໂດຍໃຊ້ Kadence Testimonials block."

**ສຳລັບ Kadence Theme:**
> "ປັບຮູບແບບ layout header ແລະ ການຈັດວາງ menu navigation."

ຜູ້ຊ່ວຍຈະໃຫ້ຄຳແນະນຳສະເພາະຂອງ theme ແລະ code examples ທີ່ຖືກອອກແບບມາໃຫ້ເຂົ້າກັບ theme ແລະ plugin ທີ່ທ່ານກຳລັງໃຊ້ຢູ່.

ທັກສະທີ່ຮູ້ຈັກ về theme ມີໃຫ້ໂດຍອັດຕະໂນມັດໃນ Superdav AI Agent v1.10.0 ແລະ ລ່າກວ່າ. ບໍ່ຕ້ອງການການຕັ້ງຄ່າ ຫຼື ການកំណត់ໃດໆເພີ່ມເຕີມ.
