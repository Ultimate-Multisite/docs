# ການຕັ້ງຄ່າການຊຳລະເງິນແບບ Manual (v2)

_**ໝາຍເຫດສຳຄັນ: ບົດຄວາມນີ້ອ້າງອີງເຖິງ Ultimate Multisite ເວີຊັນ 2.x.**_

ການຊຳລະເງິນແບບ manual ເປັນວິທີໃຫ້ທ່ານສາມາດສະເໜີວິທີຊຳລະເງິນອື່ນໆ ໃນກໍລະນີທີ່ **Stripe** ຫຼື **PayPal** ບໍ່ພ້ອມໃຊ້ສຳລັບຜູ້ໃຊ້ຂອງທ່ານ. ອາດເປັນການໂອນຜ່ານສາຍທະນາຄານ ຫຼື ໂອນຜ່ານທະນາຄານ ຫຼືວິທີຊຳລະເງິນອື່ນໃດທີ່ມີໃຫ້ຜູ້ໃຊ້ຂອງທ່ານໃນທ້ອງຖິ່ນ.

## ວິທີເປີດໃຊ້ Manual Payments

ການຕັ້ງຄ່າການຊຳລະເງິນແບບ manual ແມ່ນງ່າຍຫຼາຍ. ທ່ານພຽງແຕ່ຕ້ອງເປີດໃຊ້ມັນພາຍໃຕ້ເກດເວຍ໌ການຊຳລະເງິນ ແລະໃສ່ຄຳແນະນຳລາຍລະອຽດວ່າຜູ້ໃຊ້ຄວນສົ່ງເງິນຢ່າງໃດ.

ກ່ອນອື່ນ, ໄປທີ່ **Ultimate Multisite > Settings > Payments**. ພາຍໃຕ້ **Payment Gateways** , ເປີດສະວິດ **Manual**. ທ່ານຈະເຫັນວ່າກ່ອງ **Payment Instructions** ຈະປາກົດຂຶ້ນໃຫ້ທ່ານ.

ເພີ່ມຂໍ້ມູນທີ່ລູກຄ້າຂອງທ່ານຕ້ອງການເພື່ອຊຳລະເງິນໃສ່ໃນກ່ອງນີ້. ອາດເປັນລາຍລະອຽດບັນຊີທະນາຄານຂອງທ່ານ ແລະອີເມວຂອງທ່ານ ເພື່ອໃຫ້ລູກຄ້າສາມາດສົ່ງການຢືນຢັນການຊຳລະເງິນໃຫ້ທ່ານ, ຕົວຢ່າງເຊັ່ນ.

![ສະວິດເກດເວຍ໌ການຊຳລະເງິນແບບ Manual ພ້ອມພື້ນທີ່ຂໍ້ຄວາມ Payment Instructions](/img/config/manual-gateway-expanded.png)

ນີ້ແມ່ນອິນເຕີເຟດການຕັ້ງຄ່າ manual gateway:

![ການຕັ້ງຄ່າ manual gateway](/img/config/manual-gateway-settings.png)

ຫຼັງຈາກຕັ້ງຄ່າແລ້ວ, ພຽງແຕ່ຄລິກທີ່ **Save Settings** ແລະກໍສຳເລັດ. ເມື່ອຜູ້ໃຊ້ລົງທະບຽນໃນເຄືອຂ່າຍຂອງທ່ານ, ພວກເຂົາຈະເຫັນຂໍ້ຄວາມບອກວ່າພວກເຂົາຈະໄດ້ຮັບຄຳແນະນຳຂອງທ່ານເພື່ອດຳເນີນການຊື້ໃຫ້ສຳເລັດ.

![ຂໍ້ຄວາມຢືນຢັນການລົງທະບຽນທີ່ບອກຜູ້ໃຊ້ວ່າພວກເຂົາຈະໄດ້ຮັບຄຳແນະນຳການຊຳລະເງິນ](/img/frontend/registration-manual-notice.png)

ແລະພວກເຂົາຍັງຈະໄດ້ຮັບຂໍ້ຄວາມໃນໜ້າ **Thank You** ຂອງທ່ານ ພ້ອມຄຳແນະນຳການຊຳລະເງິນຂອງທ່ານ.

## ການຢືນຢັນການຊຳລະເງິນແບບ manual

ເພື່ອຢືນຢັນການຊຳລະເງິນແບບ manual, ໄປທີ່ເມນູ **Payments** ໃນແຖບດ້ານຊ້າຍ. ທີ່ນັ້ນທ່ານສາມາດເຫັນການຊຳລະເງິນທັງໝົດໃນເຄືອຂ່າຍຂອງທ່ານ ແລະລາຍລະອຽດຂອງມັນ, ລວມທັງ **ສະຖານະ** ຂອງມັນ. ການຊຳລະເງິນແບບ manual ຈະມີສະຖານະ **Pending** ສະເໝີ ຈົນກວ່າທ່ານຈະປ່ຽນມັນດ້ວຍຕົນເອງ.

![ລາຍການການຊຳລະເງິນທີ່ສະແດງການຊຳລະເງິນແບບ manual ທີ່ລໍຖ້າຢູ່](/img/admin/payments-list.png)

ເຂົ້າໄປໜ້າການຊຳລະເງິນໂດຍຄລິກທີ່ **ລະຫັດອ້າງອີງ**. ໃນໜ້ານີ້ທ່ານມີລາຍລະອຽດທັງໝົດຂອງການຊຳລະເງິນທີ່ລໍຖ້າຢູ່, ເຊັ່ນ ID ອ້າງອີງ, ຜະລິດຕະພັນ, ເວລາປະທັບ ແລະອື່ນໆ.

![ໜ້າລາຍລະອຽດການຊຳລະເງິນທີ່ສະແດງລະຫັດອ້າງອີງ, ຜະລິດຕະພັນ ແລະຍອດລວມ](/img/admin/payment-edit.png)

ໃນຄໍລຳດ້ານຂວາ, ທ່ານສາມາດປ່ຽນສະຖານະຂອງການຊຳລະເງິນໄດ້. ການປ່ຽນມັນເປັນ **Completed** ແລະ **ເປີດສະວິດຕົວເລືອກ Activate Membership** ຈະເປີດໃຊ້ເວັບໄຊຂອງລູກຄ້າຂອງທ່ານ ແລະສະມາຊິກຂອງພວກເຂົາຈະເປັນສະຖານະໃຊ້ງານ.

![ໜ້າແກ້ໄຂການຊຳລະເງິນທີ່ຕັ້ງ Status ເປັນ Completed ແລະສະວິດ Activate Membership](/img/admin/payment-activate-membership.png)
