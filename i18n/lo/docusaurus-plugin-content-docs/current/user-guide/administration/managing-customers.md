# การจัดการลูกค้า

หน้า **Customers** จะแสดงรายการสมาชิกที่ลงทะเบียนในเครือข่ายของคุณ หน้านี้จะแสดงข้อมูลผู้ใช้พื้นฐาน รวมถึงชื่อ อีเมล วันที่เข้าสู่ระบบครั้งล่าสุด และลิงก์ **switch to** ที่ให้คุณเข้าสู่ระบบเว็บไซต์ย่อยของผู้ใช้นั้นในฐานะผู้ดูแลระบบเว็บไซต์

![Customers list page](/img/admin/customers-list.png)

คลิก **Manage** ใต้การ์ดของลูกค้าเพื่อดูข้อมูลโดยละเอียดเกี่ยวกับลูกค้ารายนั้น

บนหน้าแสดงรายละเอียดลูกค้า บล็อก **At a Glance** จะแสดงรหัสลูกค้า เวลาเข้าสู่ระบบครั้งล่าสุด และยอดรวมที่ลูกค้าใช้จ่าย

บล็อก **Customer Options** จะให้คุณตั้งค่าลูกค้าคนนี้เป็น **VIP** (แท็กเพื่อทำให้ค้นหาง่ายขึ้น) แท็บ **Billing Info** จะแสดงข้อมูลการเรียกเก็บเงิน เช่น อีเมล ที่อยู่ และประเทศ

บล็อก **Fields** จะแสดงชื่อผู้ใช้ของลูกค้า อีเมล และให้คุณเปลี่ยนสถานะ **Email Verification**

ด้านล่างบล็อก Customer Options คุณจะพบอีกสี่บล็อก:

- **Memberships** — การสมัครสมาชิกปัจจุบันที่ลูกค้ารายนี้สมัครไว้
- **Payments** — ประวัติการชำระเงินของลูกค้า
- **Sites** — เว็บไซต์ที่ลูกค้ารายนี้เป็นเจ้าของ
- **Events** — เหตุการณ์ที่เกี่ยวข้องกับลูกค้ารายนี้ (เช่น การยืนยันอีเมล, การเปลี่ยนแปลงการเรียกเก็บเงิน ฯลฯ)

คลิกไอคอนลูกศร (→) ถัดจากรายการใดๆ เพื่อไปยังหน้าแสดงรายละเอียด คลิกที่ลูกศรถัดจากเว็บไซต์เป็นตัวอย่าง จะนำคุณไปยังหน้าแก้ไขเว็บไซต์ ซึ่งคุณสามารถแก้ไขการเชื่อมโยงสมาชิก ประเภทของเว็บไซต์ และสถานะการใช้งานได้

สิ่งเดียวกันนี้ใช้กับ Memberships และ Payments — แต่ละรายการจะลิงก์ไปยังหน้าแสดงรายละเอียดที่คุณสามารถแก้ไขสถานะสมาชิก ลูกค้าที่เกี่ยวข้อง ผลิตภัณฑ์ การตั้งค่าเกตเวย์การชำระเงิน และแม้กระทั่งดำเนินการคืนเงินได้

ทางด้านขวาของหน้าลูกค้า คุณจะพบ:

- **การเข้าสู่ระบบครั้งล่าสุดและที่อยู่ IP** — เวลาเข้าสู่ระบบครั้งสุดท้าย, ที่อยู่ IP, และประเทศ
- **เวลาประทับ (Timestamps)** — เวลาที่บัญชีลูกค้าถูกสร้างขึ้นและแก้ไขล่าสุด
- **ลบ (Delete)** — ตัวเลือกในการลบบัญชีลูกค้าอย่างถาวร

:::warning
การลบบัญชีลูกค้าเป็นการดำเนินการที่ถาวรและไม่สามารถย้อนกลับได้
:::
