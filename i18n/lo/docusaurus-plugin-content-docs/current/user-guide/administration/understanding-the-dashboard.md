# ການເຂົ້າໃຈ Ultimate Multisite Dashboard

ກ່ອນที่เราຈະເຈາະເລິກເຂົ້າໄປໃນຕົວເລືອກການຕັ້ງຄ່າທີ່ສັບສົນ ແລະ ດ້ານການເຮັດວຽກຂອງ Ultimate Multisite, ເຊິ່ງມາເບິ່ງ dashboard ກ່ອນ.

ຄືກັບ plugin WordPress ອື່ນໆ, Ultimate Multisite ສ້າງເມນູລະດັບເຄືອຂ່າຍທີ່ມີຊື່ວ່າ **Ultimate Multisite** ພ້ອມກັບເມນູຮອງຢູ່ໃຕ້ຫຼາຍລາຍການ.

## Widgets ຂອງ Network Dashboard

Ultimate Multisite ຈະເພີ່ມ widgets ທີ່ເປັນປະໂຫຍດສຳລັບ network-level dashboard. Dashboard ນີ້ສາມາດຊອກເຫັນໄດ້ໂດຍການໄປທີ່ **My Sites → Network Admin → Dashboard**. Widgets ເຫຼົ່ານີ້ຖືກອອກແບບມາເພື່ອໃຫ້ເຂົ້າເຖິງຂໍ້ມູນ ແລະ ການເຮັດວຽກທົ່ວໄປໄດ້ງ່າຍສຳລັບຜູ້ບໍລິຫານເຄືອຂ່າຍ.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

### ຂັ້ນຕອນທໍາອິດ (First Steps)

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

Widget ນີ້ຈະປາກົດໃນການຕິດຕັ້ງ Ultimate Multisite ໃໝ່ ແລະ ເປັນການເຕືອນຜູ້ບໍລິຫານເຄືອຂ່າຍໃຫ້ເຮັດວຽກເພີ່ມເຕີມ ເຊັ່ນ: ການຕັ້ງຄ່າວິທີການຈ່າຍເງິນ ແລະ ການສ້າງບັນຊີທົດສອບ.

### ສະຫຼຸບ (Summary)

![Summary widget](/img/admin/network-dashboard-summary.png)

Widget Summary ຈະລາຍງານຈຳນວນຜູ້ລົງທະບຽນ ແລະ ລາຍຮັບຂອງພວກເຂົາໃນມື້ນັ້ນ. MRR (Monthly Recurring Revenue) ຈະຊີ້ໃຫ້ເຫັນຍອດລວມທີ່ຄາດໄວ້ທັງໝົດຂອງລູກຄ້າທີ່ມີການເປັນສະມາຊິກທີ່ສ້າງລາຍໄດ້.

### ສະຖິຕິກິດຈະກໍາ (Activity Stream)

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

Widget ນີ້ຈະລາຍງານເຫດການຕ່າງໆໃນ Ultimate Multisite. ເຫດການເຫຼົ່ານັ້ນປະກອບມີ ການລົງທະບຽນ, ການຍົກເລີກ, ການສ້າງໄຊທ໌, ການລຶບ ແລະ ກິດຈະກໍາອື່ນໆ.

ກະລຸນາເບິ່ງສ່ວນ Events ໃນ documentation ເພື່ອຂໍ້ຄວາມທີ່ລະອຽດກວ່າ.

### ຕອນນີ້ (Right Now)

![Right Now widget](/img/admin/network-dashboard-right-now.png)

วิดเจ็ตยูทิลิตี้ (utility widget) นี้จะแสดงสรุปสั้นๆ เกี่ยวกับผู้ใช้และเว็บไซต์ในเครือข่าย มีทางลัดที่ด้านบนช่วยให้คุณสามารถสร้างเว็บไซต์หรือผู้ใช้ใหม่ได้ด้วยการคลิกเพียงครั้งเดียว

### ข่าวสารและการอภิปราย (News and Discussions)

![วิดเจ็ตข่าวสารและการอภิปราย](/img/admin/network-dashboard-news.png)

วิดเจ็ตนี้จะดึงและแสดงข่าวสารล่าสุดของ Ultimate Multisite ให้คุณดู หมั่นตรวจสอบวิดเจ็ตนี้เพื่อรับข้อมูลเกี่ยวกับการอัปเดต บั๊กที่สำคัญ และการแก้ไขความปลอดภัยต่างๆ

## แดชบอร์ด Ultimate Multisite (Ultimate Multisite Dashboard)

ในขณะที่แดชบอร์ดระดับเครือข่ายจะแสดงข้อมูลภาพรวมทั้งหมดเกี่ยวกับเครือข่าย แดชบอร์ด Ultimate Multisite (ซึ่งอยู่ที่ส่วนบนสุดของเมนู Ultimate Multisite) จะแสดงข้อมูลเฉพาะเกี่ยวกับบริการนั้นๆ

![แดชบอร์ด Ultimate Multisite](/img/admin/um-dashboard-top.png)

### การเติบโตของรายได้ประจำรายเดือน (Monthly Recurring Revenue Growth)

กราฟ MRR (Monthly Recurring Revenue หรือ รายได้ที่เกิดขึ้นซ้ำในแต่ละเดือน) จะแสดงรายละเอียดรายเดือนของรายได้ในช่วงเวลาปฏิทิน 12 เดือน

![กราฟการเติบโตของ MRR](/img/admin/um-dashboard-mrr.png)

รายได้ใหม่จะถูกติดตามด้วยสีน้ำเงิน ในขณะที่การยกเลิกจะแสดงด้วยสีแดง

### การรองรับช่วงวันที่ (Date-range Support)

ขึ้นอยู่กับจำนวนลูกค้าและกิจกรรม สถิติอาจดูเยอะเกินไป การรองรับช่วงวันที่ที่มีอยู่ในแดชบอร์ดช่วยให้ผู้ดูแลระบบเครือข่ายสามารถมุ่งเน้นไปที่ข้อมูลและช่วงเวลาที่จำเป็นเท่านั้น

![ตัวเลือกช่วงวันที่และสถิติ](/img/admin/um-dashboard-date-range.png)

ช่วงวันที่นี้จะควบคุมระยะเวลาของข้อมูลที่แสดงโดยวิดเจ็ตด้านล่างโดยตรง ซึ่งรวมถึงรายได้ การสมัครสมาชิกใหม่ การลงทะเบียนตามประเทศ การลงทะเบียนตามฟอร์ม และเว็บไซต์ที่มีการเข้าชมมากที่สุด

![วิดเจ็ตสถิติแดชบอร์ด](/img/admin/um-dashboard-stats.png)

:::tip
เพื่อให้วิดเจ็ต **Most Visited Sites** ทำงานได้ ให้เปิดใช้งานการนับจำนวนการเข้าชมที่ **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**
:::
