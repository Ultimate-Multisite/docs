# ການຈັດການສະມາຊິກ (Managing Memberships) {#managing-memberships}

ໃນ Ultimate Multisite, ເຈົ້າສາມາດຈັດການສະມາຊິກ ແລະ ເຂົ້າເຖິງການສະໝັກໃຊ້ທັງໝົດໃນເຄືອຂ່າຍຂອງເຈົ້າໄດ້. ເຈົ້າສາມາດກວດສອບ ແລະ ແກ້ໄຂລາຍລະອຽດການສະໝັກໃຊ້ຕ່າງໆ ເຊັ່ນ: ແຜນການ (plans) ແລະ ຜະລິດຕະພັນ (products), ຈຳນວນໄຊທ໌, ຈຳນວນເງິນໃນການເກັບຄ່າທຳນຽມ ແລະ ຄວາມຖີ່ຂອງການເກັບຄ່າທຳນຽມ, ປະຫວັດການຈ່າຍເງິນ ແລະ ເວລາທີ່ບັນທຶກໄວ້.

ເພື່ອເຂົ້າເຖິງໜ້າພ这ນີ້, ໃຫ້ໄປທີ່ແຖບ **Memberships** ໃນເມນູ Ultimate Multisite.

![Memberships list page](/img/admin/memberships-list.png)

ໃນໜ້າ Memberships ເຈົ້າຈະເຫັນລາຍຊື່ລູກຄ້າຂອງເຈົ້າ ແລະ ຜະລິດຕະພັນທີ່ກ່ຽວຂ້ອງກັບບັນຊີຂອງພວກເຂົາ. ເຈົ້າສາມາດກັ່ນຕອງສະມາຊິກຕາມສະຖານະໄດ້: active (ກຳລັງໃຊ້ງານ), pending (ລໍຖ້າ), on hold (ຖືກផ្ອຍໄວ້), expired (ໝົດອາຍຸ), ຫຼື cancelled (ຖືກຍົກເລີກ).

ສະມາຊິກຟຣີ ຈະຖືກຈັດການເປັນສະມາຊິກແບບ **lifetime** (ຕະຫຼອດໄປ). ພວກມັນບໍ່ມີວັນໝົດອາຍຸ ແລະ ຈະບໍ່ໝົດອາຍຸໂດຍອັດຕະໂນມັດ. ຖ້າແຜນຟຣີຄວນຈະສິ້ນສຸດລົງຫຼັງຈາກການທົດລອງໃຊ້ງານ ຫຼື ໄລຍະເວລາທີ່ກຳນົດໄວ້, ໃຫ້ໃຊ້ຜະລິດຕະພັນແບບຈ່າຍເງິນທີ່ມີການທົດລອງໃຊ້ງານ, ຍົກເລີກສະມາຊິກດ້ວຍຕົນເອງ, ຫຼື ຍ້າຍລູກຄ້າໄປໃສ່ແຜນອື່ນເມື່ອການເຂົ້າເຖິງຂອງພວກເຂົາຕ້ອງການປ່ຽນແປງ.

ໃຫ້ຄລິກໃສ່ສະມາຊິກໃດໜຶ່ງເພື່ອເຂົ້າເຖິງໜ້າທີ່ເຈົ້າສາມາດແກ້ໄຂລາຍລະອຽດຂອງສະມາຊິກໄດ້ — ເງື່ອນໄຂການເກັບຄ່າທຳນຽມ, ຂີດຈຳກັດໄຊທ໌ ຫຼື ການເຂົ້າເຖິງ ແລະ ອື່ນໆ.

![Membership edit interface](/img/admin/membership-edit.png)

ນີ້ແມ່ນພາບລວມຂອງໜ້າແກ້ໄຂສະມາຊິກ:

![Membership edit full interface](/img/admin/membership-edit-full.png)

ເຈົ້າສາມາດເພີ່ມສະມາຊິກໃໝ່ທີ່ກ່ຽວຂ້ອງກັບຜູ້ໃຊ້ທີ່ເຄີຍລົງທະບຽນຢູ່ໃນເຄືອຂ່າຍຂອງເຈົ້າໄດ້ຄືກັນ. ຄລິກ **Add Membership** ແລະ ຕັ້ງຄ່າລາຍລະອຽດ.

![Add Membership modal](/img/admin/membership-add-modal.png)

ຈາກນັ້ນໃຫ້ຄລິກ **Create Membership**. ສະມາຊິກໃໝ່ຈະປາກົດຢູ່ໃນລາຍຊື່ສະມາຊິກ.
