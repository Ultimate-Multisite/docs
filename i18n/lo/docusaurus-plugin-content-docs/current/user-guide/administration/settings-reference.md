# ອ້າງອີງ Settings

ໜ້ານີ້ຕິດຕາມ Settings ທີ່ມີຜົນຕໍ່ການບໍລິຫານປະຈຳວັນ ແລະການປ່ຽນແປງພຶດຕິກຳຫຼ້າສຸດໃນ Ultimate Multisite.

## ຕົວເລືອກອື່ນໆ

ພື້ນທີ່ **ຕົວເລືອກອື່ນໆ** ປາກົດຢູ່ໃຕ້ **Ultimate Multisite > Settings > ເຂົ້າລະບົບ & ລົງທະບຽນ**.

| Setting | ຄຳອະທິບາຍ |
|---|---|
| **ເປີດໃຊ້ Jumper** | ສະແດງເຄື່ອງມືນຳທາງດ່ວນ Jumper ໃນພື້ນທີ່ຜູ້ບໍລິຫານ. ໃຊ້ມັນເພື່ອຂ້າມໄປຍັງໜ້າຈໍ Ultimate Multisite, ວັດຖຸເຄືອຂ່າຍ, ແລະປາຍທາງຜູ້ບໍລິຫານທີ່ຮອງຮັບໂດຍກົງ. ປິດໃຊ້ງານມັນຖ້າທ່ານບໍ່ຕ້ອງການໃຫ້ທາງລັດນີ້ປາກົດ. |

## ການລາຍງານຂໍ້ຜິດພາດ ແລະ telemetry

Settings ເລືອກເຂົ້າຮ່ວມການລາຍງານຂໍ້ຜິດພາດກ່ອນໜ້ານີ້ໄດ້ຖືກລຶບອອກຈາກໜ້າ Settings ແລ້ວ. telemetry ແບບບໍ່ລະບຸຕົວຕົນຖືກປິດໃຊ້ງານ ແລະບໍ່ມີ UI toggle ໃຫ້ເປີດໃຊ້ມັນ.

ຖ້າທ່ານຮັກສາ runbooks ພາຍໃນ ຫຼືຮູບພາບໜ້າຈໍສຳລັບໜ້າ Settings, ໃຫ້ລຶບການອ້າງອີງເຖິງ field ເລືອກເຂົ້າຮ່ວມການລາຍງານຂໍ້ຜິດພາດເກົ່າ ເພື່ອໃຫ້ຜູ້ບໍລິຫານບໍ່ຊອກຫາ Setting ທີ່ບໍ່ມີອີກແລ້ວ.

## Settings ນຳເຂົ້າ/ສົ່ງອອກ

ແຖບ Settings **ນຳເຂົ້າ/ສົ່ງອອກ** ອະທິບາຍວ່າມັນຄວບຄຸມ Settings ໃດ ແລະເຊື່ອມໂຍງໂດຍກົງໄປຫາ **Ultimate Multisite > ສົ່ງອອກເວັບໄຊ** ສຳລັບຄັງເກັບເວັບໄຊ ແລະເຄືອຂ່າຍ. ໃຊ້ແຖບ Settings ສຳລັບການຕັ້ງຄ່ານຳເຂົ້າ/ສົ່ງອອກ, ໃຊ້ **Tools > ສົ່ງອອກ & ນຳເຂົ້າ** ສຳລັບກະແສວຽກສົ່ງອອກ/ນຳເຂົ້າເວັບໄຊດຽວ, ແລະໃຊ້ເຄື່ອງມື Site Export ເມື່ອທ່ານຕ້ອງການຄັງເກັບ Network Export ເຕັມຮູບແບບ.

## ຄຳເຕືອນຍອດເງິນ Domain Seller HostAfrica

ເມື່ອ addon Domain Seller ຖືກເຊື່ອມຕໍ່ກັບ HostAfrica, ຜູ້ບໍລິຫານເຄືອຂ່າຍຕອນນີ້ເຫັນຄຳເຕືອນຍອດເງິນຕ່ຳທີ່ສາມາດປິດໄດ້ ເມື່ອຍອດເງິນບັນຊີ reseller ຕ່ຳເກີນໄປສຳລັບການປະມວນຜົນການຈົດທະບຽນ ຫຼືຕໍ່ອາຍຸໂດເມນຢ່າງເຊື່ອຖືໄດ້.

ໃຫ້ຖືປະກາດນີ້ເປັນຄຳເຕືອນດ້ານການດຳເນີນງານ: ເຕີມເງິນຍອດບັນຊີ reseller HostAfrica ກ່ອນຮັບການຈົດທະບຽນໂດເມນແບບຈ່າຍເງິນເພີ່ມຕື່ມ, ຈາກນັ້ນກັບໄປທີ່ Settings Domain Seller ຫຼືໜ້າຈໍຕິດຕາມໂດເມນ ເພື່ອຢືນຢັນວ່າການຈົດທະບຽນ ແລະການຕໍ່ອາຍຸສາມາດດຳເນີນຕໍ່ໄປໄດ້ຕາມປົກກະຕິ.

## Settings ຕົວເຊື່ອມຕໍ່ AI provider

Settings ຕົວເຊື່ອມຕໍ່ AI provider ຕອນນີ້ເປີດເຜີຍສະເພາະກຸ່ມ Account OAuth ທີ່ຮອງຮັບ:

| Provider | ກະແສການຕັ້ງຄ່າ |
|---|---|
| **Anthropic Max** | ເຊື່ອມຕໍ່ Account Anthropic Max ໜຶ່ງບັນຊີ ຫຼືຫຼາຍບັນຊີດ້ວຍປຸ່ມ OAuth. ໃຊ້ທາງເລືອກສຳຮອງ OAuth ແບບຄູ່ມື ເມື່ອ browser ທີ່ຖືກ sandbox ບໍ່ສາມາດສຳເລັດການ redirect ໂດຍອັດຕະໂນມັດ. |
| **OpenAI ChatGPT/Codex** | ເຊື່ອມຕໍ່ Account ChatGPT ຜ່ານກະແສວຽກກຸ່ມ OAuth ດຽວກັນ. ການດຳເນີນງານທີ່ຕົວເຊື່ອມຕໍ່ຮອງຮັບສາມາດໃຊ້ການເອີ້ນເຄື່ອງມື ChatGPT Codex ຫຼັງຈາກ Account ຖືກເຊື່ອມຕໍ່ແລ້ວ. |
| **Google AI Pro** | ເຊື່ອມຕໍ່ Account Google AI Pro ຜ່ານ OAuth, ຈາກນັ້ນ refresh ຕົວເຊື່ອມຕໍ່ຖ້າລາຍການ Account ບໍ່ອັບເດດທັນທີ. |

Cursor Pro ບໍ່ແມ່ນ provider ທີ່ຮອງຮັບອີກແລ້ວ. ລຶບຮູບພາບໜ້າຈໍພາຍໃນເກົ່າ, runbooks, ຫຼືຂັ້ນຕອນ onboarding ທີ່ກ່າວເຖິງ fields ການຕັ້ງຄ່າ Cursor Pro ຫຼືເສັ້ນທາງຕົວເຊື່ອມຕໍ່.

ເມື່ອເພີ່ມ ຫຼືລຶບ Account provider, ໃຫ້ປ້ອນທີ່ຢູ່ email ທີ່ຖືກຕ້ອງສຳລັບ Account ທີ່ກຳລັງ refresh ຫຼືຖືກລຶບ ແລະບັນທຶກ Settings provider ກ່ອນທົດສອບການດຳເນີນງານທີ່ມີຕົວເຊື່ອມຕໍ່ຮອງຮັບ.
