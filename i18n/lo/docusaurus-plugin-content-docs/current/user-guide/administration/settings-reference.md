# ຕາຕະລາງການຕັ້ງຄ່າອ້າງອີງ

ໜ້າພ这ຕິດຕາມການຕັ້ງຄ່າທີ່ສົ່ງຜົນກະທົບຕໍ່ການບໍລິຫານປະຈຳວັນ ແລະ ການປ່ຽນແປງຂອງພຶດຕິກຳບໍ່ດົນໃນ Ultimate Multisite.

## ທາງເລືອກອື່ນໆ

ສ່ວນ **Other Options** (ທາງເລືອກອື່ນໆ) ຈະສະແດງຢູ່ໃຕ້ **Ultimate Multisite > Settings > Login & Registration**.

| ການຕັ້ງຄ່າ | ຄຳອະທິບາຍ |
|---|---|
| **Enable Jumper** | ສະແດງເຄື່ອງມື quick navigation ຊື່ວ່າ Jumper ໃນ khu vực admin. ໃຊ້ເພື່ອຂ້າມໄປຫາໜ້າ Ultimate Multisite, object ຂອງ network ແລະ ຈຸດປາຍທາງ admin ທີ່ຮອງຮັບໂດຍກົງ. ປິດມັນຖ້າທ່ານບໍ່ຕ້ອງການໃຫ້ shortcut ນີ້ເຫັນ. |

## ການລາຍງານຂໍ້ຜິດພາດ ແລະ telemetry

ການຕັ້ງຄ່າເພີ່ມເຕີມເພື່ອສະໝັກເຂົ້າໃນການລາຍງານຂໍ້ຜິດພາດທີ່ເຄີຍມີຢູ່ໄດ້ຖືກເອົາອອກຈາກໜ້າຕັ້ງຄ່າແລ້ວ. Telemetry ແບບບໍ່ເປີດເຜີຍຂໍ້ມູນ (Anonymous telemetry) ຖືກປິດໄວ້ ແລະ ບໍ່ມີ UI toggle ເພື່ອເປີດມັນ.

ຖ້າທ່ານຮັກສາ runbooks ຫຼື screenshots ຂອງໜ້າຕັ້ງຄ່າໄວ້, ກະລຸນາເອົາການອ້າງອີງເຖິງຊ່ອງ error-reporting ເກົ່າອອກ ເພື່ອໃຫ້ຜູ້ບໍລິຫານບໍ່ໄດ້ຊອກຫາການຕັ້ງຄ່າທີ່ບໍ່ມີຢູ່ແລ້ວ.

## ການຕັ້ງຄ່າ Import/Export

ແທັບ **Import/Export** (ນຳເຂົ້າ/ສົ່ງອອກ) ຈະອະທິບາຍວ່າການຕັ້ງຄ່ານີ້ຄວບຄຸມຫຍັງ ແລະ ເຊື່ອມຕໍ່ໂດຍກົງກັບ **Ultimate Multisite > Site Export** ສຳລັບການເກັບຂໍ້ມູນ archive ຂອງ site ແລະ network. ໃຊ້ແທັບການຕັ້ງຄ່າເພື່ອການຕັ້ງຄ່າ import/export, ໃຊ້ **Tools > Export & Import** ສຳລັບ workflow ການ export/import ແບບ single-site, ແລະ ໃຊ້ເຄື່ອງມື Site Export ເມື່ອທ່ານຕ້ອງການ archive ການ export Network ຢ່າງເຕັມທີ່.

## ການຕັ້ງຄ່າ AI provider connector settings

ການຕັ້ງຄ່າ AI provider connector ແມ່ນສະແດງໃຫ້ເຫັນພຽງແຕ່ account pools OAuth ທີ່ຮອງຮັບເທົ່ານັ້ນ:

| ผู้ให้บริการ | ขั้นตอนการตั้งค่า |
|---|---|
| **Anthropic Max** | เชื่อมต่อบัญชี Anthropic Max หนึ่งบัญชีหรือมากกว่าด้วยปุ่ม OAuth ใช้ตัวเลือก OAuth แบบแมนนวลหากเบราว์เซอร์แบบ sandboxed ไม่สามารถทำ redirect ได้โดยอัตโนมัติ |
| **OpenAI ChatGPT/Codex** | เชื่อมต่อบัญชี ChatGPT ผ่านเวิร์กโฟลว์ OAuth เดียวกัน การดำเนินการที่รองรับโดย Connector สามารถใช้เครื่องมือเรียกของ ChatGPT Codex หลังจากเชื่อมต่อบัญชีแล้ว |
| **Google AI Pro** | เชื่อมต่อบัญชี Google AI Pro ผ่าน OAuth จากนั้นรีเฟรช Connector หากรายการบัญชีไม่ได้รับการอัปเดตทันที |

Cursor Pro ไม่ใช่ผู้ให้บริการที่รองรับอีกต่อไป โปรดลบภาพหน้าจอภายใน, runbooks หรือขั้นตอนการเริ่มต้นใช้งานเก่าๆ ที่กล่าวถึงฟิลด์การตั้งค่าหรือเส้นทางของ connector สำหรับการตั้งค่า Cursor Pro ออกไป

เมื่อเพิ่มหรือลบบัญชีผู้ให้บริการ ให้กรอกที่อยู่อีเมลที่ถูกต้องสำหรับบัญชีที่จะรีเฟรชหรือลบทิ้ง และบันทึกการตั้งค่าของผู้ให้บริการก่อนทดสอบการทำงานที่มี Connector รองรับ
