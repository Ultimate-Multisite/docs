# สำรวจแผงผู้ดูแลระบบ (Admin Panel)

Ultimate Multisite ตั้งใจให้ใช้งานง่ายและเข้าใจได้ด้วยตัวเองที่สุด แต่ด้วยเครื่องมือที่ทรงพลังขนาดนี้ มันก็สามารถช่วยให้เรามีคู่มือแนะนำการใช้งานได้ครับ มาดูกันทีละหน้าในแอดมินเพื่อช่วยให้คุณคุ้นเคยกันนะครับ

## แดชบอร์ด (Dashboard)

**Dashboard** ของ Ultimate Multisite จะแสดงรายงานพื้นฐานและข้อมูลวิเคราะห์ต่างๆ เช่น รายได้, กิจกรรมของเว็บไซต์, การเติบโตของสมาชิก, จำนวนผู้เข้าชม และข้อมูลทางภูมิศาสตร์ครับ

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

นี่คือมุมมองเต็มๆ ของแดชบอร์ดครับ:

![Ultimate Multisite Dashboard full page](/img/admin/um-dashboard-full.png)

และส่วนด้านล่างของแดชบอร์ด:

![Ultimate Multisite Dashboard bottom section](/img/admin/um-dashboard-bottom.png)

นี่คือมุมมองเต็มๆ ของแดชบอร์ดเครือข่ายด้วยครับ:

![Network Dashboard full page](/img/admin/network-dashboard-full-page.png)

## แบบฟอร์มการชำระเงิน (Checkout Forms)

**Checkout Forms** ให้ความยืดหยุ่นในการสร้างหน้าลงทะเบียนแบบกำหนดเองได้ คุณสามารถปรับแต่งช่องต่างๆ, ใส่คลาสและสคริปต์ที่กำหนดเอง และจำกัดการเข้าถึงตามประเทศหรือตำแหน่งของผู้เข้าชมได้ด้วยครับ

![Checkout Forms list](/img/admin/checkout-forms-list.png)

## ผลิตภัณฑ์ (Products)

ส่วน **Products** คือที่ที่คุณจะสร้างแผน, แพ็กเกจ และบริการต่างๆ สำหรับเครือข่ายของคุณ กำหนดราคา, ความถี่ในการเรียกเก็บเงิน และข้อจำกัด/โควต้าในระดับผลิตภัณฑ์ได้เลยครับ

![Products list](/img/admin/products-list.png)

## สมาชิก (Memberships)

หน้า **Memberships** จะแสดงการสมัครสมาชิกทั้งหมดในเครือข่ายของคุณ คุณสามารถตรวจสอบและแก้ไขรายละเอียดการสมัครสมาชิก รวมถึงแผน, ผลิตภัณฑ์, จำนวนเงินที่เรียกเก็บ และความถี่ในการเรียกเก็บเงิน, ประวัติการชำระเงิน และเวลาที่บันทึกไว้ได้ครับ

![Memberships list](/img/admin/memberships-list.png)

## การชำระเงิน (Payments)

ໜ້າ **Payments** ຈະສະແດງລາຍການປະຫວັດການຈ່າຍເງິນທົ່ວເຄືອຂ່າຍຂອງທ່ານຢ່າງໄວວາ, ພ້ອມຂໍ້ມູນລາຍລະອຽດຂອງທຸກທຸລະກໍາ ລວມທັງຜະລິດຕະພັນ ແລະ ຈຳນວນເງິນທີ່ຊapatillas.

![Payments list](/img/admin/payments-list.png)

## Customers

ໜ້າ **Customers** ຈະສະແດງສະມາຊິກທຸກຄົນທີ່ລົງທະບຽນຢູ່ໃນເຄືອຂ່າຍຂອງທ່ານພ້ອມຂໍ້ມູນຜູ້ໃຊ້ພື້ນຖານ ເຊັ່ນ: ຊື່, ອີເມວ, ການເຂົ້າສູ່ລະບົບຄັ້ງສຸດທ້າຍ ແລະ ລິ້ງ "ປ່ຽນໄປຫາ" ເພື່ອເຂົ້າສູ່ລະບົບຂອງ subsites ຂອງເຂົາເຈົ້າ.

![Customers list](/img/admin/customers-list.png)

## Sites

ໜ້າ **Sites** ຈະສະແດງລາຍການ subsites ທັງໝົດໃນເຄືອຂ່າຍຂອງທ່ານ ລວມທັງ template website ແລະ subsites ເປັນຂອງລູກຄ້າ. ສາມາດຈັດການ domain map, ຂີດຈຳກັດ ແລະ quota, plugin ແລະ theme, ແລະ thumbnail website ໄດ້ຢ່າງງ່າຍດາຍ.

![Sites list](/img/admin/sites-list.png)

## Domains

ໜ້າ **Domains** ແມ່ນສຳລັບ custom domains ທີ່ຖືກ map ກັບ subsites. ໃນຖານະ super admin, ທ່ານສາມາດເພີ່ມ ຫຼື map ຊື່ domain ທີ່ເປັນຂອງເຈົ້າໄດ້ດ້ວຍຕົນເອງ ແລະ ເບິ່ງ DNS records ແລະ logs ໄດ້.

![Domains list](/img/admin/domains-list.png)

## Discount Codes

**Discount Codes** ອະນຸຍາດໃຫ້ທ່ານສ້າງ coupon codes ເພື່ອສະເໜີສ່ວນຫຼຸດໃຫ້ກັບ end-users ຂອງທ່ານ. ສາມາດລະບຸຄ່າສ່ວນຫຼຸດ ແລະ ເປົ້າໝາຍຜູ້ໃຊ້ ຫຼື ກຸ່ມໃດເປັນພິເສດໄດ້.

![Discount Codes list](/img/admin/discount-codes-list.png)

## Broadcasts

**Broadcasts** ແມ່ນເຄື່ອງມືຊ່ວຍໃນການເຜີຍແຜ່ການປະກາດ ຫຼື ສົ່ງຂໍ້ຄວາມສ່ວນຕົວໄປຫາຜູ້ໃຊ້ subsites ຂອງທ່ານ. ສາມາດກຳນົດກຸ່ມຜູ້ໃຊ້ສະເພາະໂດຍອີງຕາມ plan ຫຼື product ທີ່ເຂົາເຈົ້າໄດ້ສະໝັກໄວ້.

![Broadcasts list](/img/admin/broadcasts-list.png)

## Settings

ໜ້າ **Settings** ແມ່ນບ່ອນທີ່ທ່ານສາມາດຕັ້ງຄ່າ Ultimate Multisite — ການຕັ້ງຄ່າການລົງທະບຽນ, ການຈ່າຍເງິນ, API ແລະ webhooks, ການ map domain, ແລະ ການເຊື່ອມຕໍ່ອື່ນໆ.

การตั้งค่า AI connector แสดงรายการกลุ่ม OAuth provider ที่รองรับในปัจจุบัน: Anthropic Max, OpenAI ChatGPT/Codex และ Google AI Pro แต่ละการ์ดของ provider จะให้ super admin สามารถเชื่อมต่อบัญชี, รีเฟรชบัญชีที่บันทึกไว้, ลบบัญชีตามอีเมล และใช้ fallback แบบ manual OAuth เมื่อสภาพแวดล้อมแบบ sandboxed บล็อกการเปลี่ยนเส้นทางของเบราว์เซอร์ ตัวเลือกการตั้งค่า Cursor Pro ได้ถูกนำออกจากหน้า admin แล้ว

บัญชี ChatGPT/Codex รองรับการใช้งานเครื่องมือที่เชื่อมต่อกับ connector ซึ่งหมายความว่าการดำเนินการที่อนุญาตให้ใช้เครื่องมือ สามารถใช้พฤติกรรมเครื่องมือของ Codex ได้ หลังจากที่เชื่อมต่อบัญชี OpenAI เรียบร้อยแล้ว

## Events (เหตุการณ์)

หน้านี้จะเก็บประวัติของทุกเหตุการณ์และบันทึกในเครือข่ายของคุณ มันจะติดตามกิจกรรมต่างๆ เช่น การเปลี่ยนแปลงแผน, การลงทะเบียน และการเคลื่อนไหวอื่นๆ ซึ่งมีประโยชน์สำหรับการตรวจสอบ multisite ของคุณ

![Events list](/img/admin/events-list.png)

## Webhooks (เว็บฮุก)

**Webhooks** ช่วยให้คุณส่งข้อมูลไปยังแอปพลิเคชันอื่น มีประโยชน์สำหรับการส่งข้อมูลจาก Ultimate Multisite ไปยังแพลตฟอร์มอย่าง Zapier

![Webhooks list](/img/admin/webhooks-list.png)
