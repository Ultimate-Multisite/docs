# ການຈັດການໄຊທ໌

ໄຊທ໌ (ຫຼື subsites) ແມ່ນຫົວໃຈຫຼັກຂອງທຸລະກິດ WaaS ຂອງທ່ານ. Ultimate Multisite ມີ 3 ປະເພດຂອງໄຊທ໌:

- **Customer-owned** — ໄຊທ໌ທີ່ເປັນຂອງລູກຄ້າສະເພາະ
- **Site Templates** — ໄຊທ໌ທີ່ຖືກກຳນົດໄວ້ລ່ວງໜ້າ ເຊິ່ງລູກຄ້າສາມາດເລືອກໃຊ້ເປັນຈຸດເລີ່ມຕົ້ນໄດ້
- **Main Site** — ໄຊທ໌ຫຼັກຂອງເຄືອຂ່າຍຂອງທ່ານ

## ການເບິ່ງໄຊທ໌

ໄປທີ່ **Ultimate Multisite → Sites** ເພື່ອເບິ່ງ subsites ທັງໝົດໃນເຄືອຂ່າຍຂອງທ່ານ. ແຕ່ລະໄຊທ໌ຈະມີປ້າຍກຳນົດປະເພດ ແລະ ທ່ານສາມາດກອງ (filter) ຕາມ All Sites, Customer-Owned, Templates, ຫຼື Pending ໄດ້ເລີຍ.

![Sites list page](/img/admin/sites-list.png)

## ການເພີ່ມໄຊທ໌ໃໝ່

ກົດປຸ່ມ **Add Site** ເພື່ອສ້າງໄຊທ໌ໃໝ່. ທ່ານຈະຕ້ອງໃຫ້ຂໍ້ມູນດັ່ງນີ້:

- **Site Title** — ຊື່ຂອງໄຊທ໌ໃໝ່
- **Site URL** — ໂດເມນ/ເສັ້ນທາງ (path) ຂອງໄຊທ໌
- **Site Type** — ເປັນໄຊທ໌ຂອງລູກຄ້າ, template, ຫຼື regular site

ຕົວເລືອກ **Copy Site** ອະນຸຍາດໃຫ້ທ່ານສ້າງໄຊທ໌ໃໝ່ໂດຍອີງໃສ່ template ທີ່ມີຢູ່ແລ້ວ. ເມື່ອເປີດໃຊ້ງານ, ທ່ານສາມາດເລືອກໄດ້ວ່າຈະໃຊ້ template ໃດເປັນຈຸດເລີ່ມຕົ້ນ. ຂໍໃຫ້ກວດສອບວ່າ **Copy Media on Duplication** ຖືກເປີດໃຊ້ເພື່ອລວບລວມໄຟລ໌ media ນຳເຂົ້າດ້ວຍ.

## ການຈັດການໄຊທ໌ທີ່ມີຢູ່ແລ້ວ

ກົດ **Manage** ໃສ່ໄຊທ໌ໃດໜຶ່ງ ເພື່ອເປີດໜ້າ **Edit Site**. ທີ່ນີ້ທ່ານຈະພົບກັບ:

### ຂໍ້ມູນພື້ນຖານ

ຊື່ໄຊທ໌, ປະເພດ, site ID ແລະ ຄຳອະທິບາຍ. ທ່ານຍັງຈະເຫັນໂດເມນທີ່ຖືກแมપ (mapped domain), ການເປັນສະມາຊິກທີ່ກ່ຽວຂ້ອງ, ແລະ ບັນຊີລູກຄ້າທີ່ເປັນເຈົ້າຂອງໄຊທ໌ນັ້ນໆ.

### ຕົວເລືອກໄຊທ໌

ຕັ້ງຄ່າຄວາມສາມາດ ແລະ ຂີດຈຳກັດຂອງໄຊທ໌:

- **Visit limits** — ຈຳນວນການເຂົ້າເບິ່ງໄຊທ໌ສູງສຸດ
- **User account limits** — ຂີດຈຳກັດຕໍ່ໜ້າທີ່ຂອງຜູ້ໃຊ້ແຕ່ລະຄົນ
- **Disk space** — ຂໍ້ຈຳກັດພື້ນທີ່ເກັບຂໍ້ມູນ (storage quota) ສຳລັບໄຊທ໌ນັ້ນໆ
- **Custom domain** — ເປີດການแมપໂດເມນສຳລັບໄຊທ໌ນີ້
- **Plugin and theme visibility** — ຄວບຄຸມວ່າ plugin ແລະ theme ໃດຈະສະແດງ, ຖືກເຊື່ອງໄວ້, ຫຼື ຖືກເປີດໃຊ້ງານລ່ວງໜ້າ

ໂດຍປົກກະຕິແລ້ວ, ເວັບໄຊທ໌ຈະປະຕິບັດຕາມຂີດຈຳກັດທີ່ໄດ້ກຳນົດໄວ້ໃນລະດັບການເປັນສະມາຊິກ (membership level). ການຕັ້ງຄ່າຂີດຈຳກັດໃນລະດັບຂອງເວັບໄຊທ໌ຈະມີອຳນາດສູງກວ່າການຕັ້ງຄ່າຂອງການເປັນສະມາຊິກ.

### ການເຊື່ອມໂຍງ (Associations)

ຢູ່ໃຕ້ຕົວເລືອກຂອງເວັບໄຊທ໌, ທ່ານຈະພົບຂໍ້ມູນກ່ຽວກັບ:

- **Mapped domains** (ໂດເມນທີ່ຖືກເຊື່ອມຕໍ່) ທີ່ກ່ຽວຂ້ອງກັບເວັບໄຊທ໌
- **Membership** (ການເປັນສະມາຊິກ) ທີ່ເວັບໄຊທ໌ນັ້ນຢູ່ພາຍໃຕ້
- **Customer account** (ບັນຊີລູກຄ້າ) ທີ່ເຊື່ອມໂຍງກັບເວັບໄຊທ໌

### ແຖບຂ້າງຂວາ (Right Sidebar)

ທາງດ້ານຂວາທ່ານສາມາດ:

- **Enable/disable the site** (ເປີດ/ປິດເວັບໄຊທ໌) ໂດຍໃຊ້ປຸ່ມ toggle
- **Change the site type** (ປ່ຽນປະເພດຂອງເວັບໄຊທ໌) ຫຼື ຈັດສິດເຈົ້າຂອງໃໝ່
- **Set a site image/thumbnail** (ຕັ້ງຮູບພາບ/Thumbnail ຂອງເວັບໄຊທ໌) (ຈະສະແດງຢູ່ໜ້າເວັບ (frontend) ສຳລັບ template ຂອງເວັບໄຊທ໌)
- **Delete the site** (ລຶບເວັບໄຊທ໌) ຄືນແບບຖາວອນ

:::warning
ການລຶບເວັບໄຊທ໌ແມ່ນບໍ່ສາມາດກູ້ຄືນໄດ້. ເວັບໄຊທ໌ ແລະ ເນື້ອໃນທັງໝົດຈະຖືກລຶບອອກຢ່າງຖາວອນ.
:::
