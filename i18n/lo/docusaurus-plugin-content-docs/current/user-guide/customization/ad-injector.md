# ການដាក់ໂຄສະນາໃນເຄືອຂ່າຍຂອງທ່ານດ້ວຍ Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector ແມ່ນ **add-on ຟຣີ** ອັນໃຫ້ເຄື່ອງມືແກ້ໄຂໂຄສະນາໃສ່ແຜນການຕ່າງໆໄດ້ໂດຍກົງຈາກໜ້າແກ້ໄຂຂອງມັນ.

ຫຼັງຈາກຕິດຕັ້ງ Ultimate Multisite: Ad Injector ແລ້ວ, ໄປທີ່ໜ້າ _Plugins_ ຂອງທ່ານເພື່ອເປີດໃຊ້ງານມັນ.

<!-- ຮູບພາບບໍ່ມີให้: ໜ້າ WP Plugins ສະແດງປຸ່ມ Network Activate ສຳລັບ add-on Ad Injector -->

_ກົດປຸ່ມ Network Activate_

ເມື່ອ add-on ເປີດໃຊ້ງານແລ້ວ, ບ່ອນ _Advanced Options_ ໃນໜ້າແກ້ໄຂແຜນການຈະມີຕົວເລືອກໃໝ່ຊື່ວ່າ _Ad Settings_.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_ດຽວນີ້ທ່ານສາມາດແກ້ໄຂການຕັ້ງຄ່າໂຄສະນາສຳລັບແຕ່ລະແຜນຂອງທ່ານໄດ້ເລີຍ_

ໃນ tab ນັ້ນ, ທ່ານຈະສາມາດເພີ່ມ ad codes ສຳລັບ posts ແລະ pages ໃນ front-end, ທັງຢູ່ກ່ອນ ແລະ ຫຼັງເນື້ອຫາ.

<!-- ຮູບພາບບໍ່ມີໃຫ້: ແຖບ ad code ສຳລັບ posts ແລະ pages ກ່ອນເນື້ອຫາ -->

<!-- ຮູບພາບບໍ່ມີໃຫ້: ຕົວຢ່າງການສະແດງໂຄສະນາໃນໜ້າ post ສຳລັບແຜນໃດໜຶ່ງ -->

_ຫຼັງຈາກໃສ່ ad code ຂອງທ່ານເຂົ້າໄປ, ມັນກໍຈະສະແດງຢູ່ໃນ posts ແລະ pages ຂອງແຜນນັ້ນໆ._

ທ່ານຍັງສາມາດມີໂຄສະນາລວມຢູ່ໃນ header ຂອງ Admin Pages ສຳລັບຜູ້ໃຊ້ຂອງແຜນນັ້ນໄດ້ອີກ.

<!-- ຮູບພາບບໍ່ມີໃຫ້: ຕົວເລືອກ Ad Code ໃນ Admin Panel ຂອງ plan settings -->

<!-- ຮູບພາບບໍ່ມີໃຫ້: ຕົວຢ່າງການສະແດງໂຄສະນາໃນ user admin panel -->

_ໄປທີ່ຕົວເລືອກ Ad Code ໃນ Admin Panel ຖ້າທ່ານຕ້ອງການສະແດງໂຄສະນາໃນ back-end ຂອງຜູ້ໃຊ້_

### ເກີນກວ່າການໂຄສະນາ

ທ່ານຍັງສາມາດໃຊ້ Ultimate Multisite: Ad Injector ເພື່ອສະໜອງການຊ່ວຍເຫຼືອໃຫ້ກັບແຜນທີ່ມີລະດັບສູງ (high-tier plans) ໄດ້ອີກ.

ພຽງແຕ່ໄປທີ່ຕົວເລືອກ _Admin Panel Ads_ ແລະ ເພີ່ມ HTML code ສຳລັບ support tool ຂອງທ່ານໃສ່.

<!-- ຮູບພາບບໍ່ມີໃຫ້: ຕົວເລືອກ Admin Panel Ads ທີ່ມີ HTML code support tool -->

_ເພີ່ມການຊ່ວຍເຫຼືອລູກຄ້າໃສ່ແຜນສະໝandroidx ທີ່ພິເສດ_

ແບບນີ້ຄືໃຫ້ຄົນສາມາດຕິດຕໍ່ຫາທ່ານໄດ້ໂດຍກົງຈາກໜ້າ Admin Panel ເຫຼົ່ານັ້ນເລີຍ!
