# การปรับแต่งหน้า Admin ของคุณด้วย PRO Theme {#customizing-your-admin-panel-with-pro-theme}

บ่อยครั้งเวลาที่เราทำงานให้ลูกค้า เราก็มีความต้องการที่จะมอบประสบการณ์ที่ปรับแต่งเองในแดชบอร์ดของ WordPress ให้กับลูกค้าของเรา ถ้าคุณกำลังมองหาลุคและฟีลลิ่งที่เป็นมืออาชีพมากขึ้น PRO Theme อาจช่วยคุณได้

PRO Theme เป็นธีมสำหรับ Admin ที่พัฒนาขึ้นมาเพื่อช่วยคุณในการปรับแต่ง Dashboard ของลูกค้า และยังเป็นส่วนหนึ่งของ core plugin ระดับพรีเมียมของ Ultimate Multisite ด้วย

หลังจากติดตั้งแล้ว ให้ไปที่หน้า _Plugins_ ของคุณ แล้วคลิกเพื่อเปิดใช้งานบน Network ของคุณ

<!-- รูปภาพไม่พร้อมใช้งาน: หน้า WP Plugins แสดงปุ่ม Activate สำหรับ PRO Theme บน Network -->

_เปิดใช้งานทั่วทั้ง Network บนหน้า Plugins ของคุณ_

_**หมายเหตุ:**_ _ถ้าคุณเพิ่งติดตั้ง มันจะแสดงเป็นตัวเลือกให้ลูกค้าของคุณสามารถทำเครื่องหมายเพื่อเพิ่มธีมนี้ให้กับผู้ใช้นั้นได้เท่านั้น._

ตอนนี้ คุณก็มีอินเทอร์เฟซ Admin ที่ได้รับการออกแบบใหม่ทั้งหมดแล้ว! คุณยังมีเมนูใหม่ทางด้านซ้ายที่ชื่อว่า PRO Theme

<!-- รูปภาพไม่พร้อมใช้งาน: อินเทอร์เฟซ Admin ที่ออกแบบใหม่ของ PRO Theme -->

_ลุคใหม่สำหรับหน้า Admin ของคุณ_

ในเมนูนี้ คุณจะสามารถเข้าถึงแท็บห้าอันได้:

<!-- รูปภาพไม่พร้อมใช้งาน: แท็บ Styling Settings ของ PRO Theme -->

_Styling Settings_ (การตั้งค่าสไตล์)

<!-- รูปภาพไม่พร้อมใช้งาน: แท็บ Functionality ของ PRO Theme -->

_Functionality_ (ฟังก์ชันการทำงาน)

<!-- รูปภาพไม่พร้อมใช้งาน: แท็บ Custom CSS ของ PRO Theme -->

_Custom CSS_ (CSS ที่กำหนดเอง)

<!-- รูปภาพไม่พร้อมใช้งาน: แท็บ Import/Export ของ PRO Theme -->

_Import/Export_ (นำเข้า/ส่งออก)

<!-- รูปภาพไม่พร้อมใช้งาน: แท็บ Activate ของ PRO Theme -->

_Activate PRO Theme_ (เปิดใช้งาน PRO Theme)

ในแต่ละแท็บ คุณสามารถเปิดหรือปิดรายการสไตล์, อนุญาตหรือบล็อกฟังก์ชันการทำงาน และมอบการปรับแต่งที่คุณอาจกำลังมองหาให้กับลูกค้าของคุณบนเครือข่ายเว็บไซต์ของคุณได้

นี่คือตัวอย่างสิ่งที่คุณจะได้รับหลังจากมีการปรับแต่งบางอย่าง:

<!-- ບໍ່ມີຮູບພາບ: ແດັດບອດຂອງລູກຄ້າທີ່ມີການປັບແຕ່ງດ້ວຍ PRO Theme -->

_ແດັດບອດຂອງລູກຄ້າ_

<!-- ບໍ່ມີຮູບພາບ: ໜ້າເຂົ້າສູ່ລະບົບທີ່ປັບແຕ່ງດ້ວຍ PRO Theme -->

_ໜ້າເຂົ້າສູ່ລະບົບທີ່ປັບແຕ່ງ_
