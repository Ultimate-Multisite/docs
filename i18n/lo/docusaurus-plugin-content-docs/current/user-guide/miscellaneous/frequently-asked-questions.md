# ຄຳຖາມທີ່ພົບເລື້ອຍ (Frequently Asked Questions)

ນີ້ແມ່ນຄຳຖາມທີ່ມັກຈະຖືກຖາມມາໃນເວລາຮ້ອງຂໍການຊ່ວຍເຫຼືອຕ່າງໆ.

  * **Ultimate Multisite ແມ່ນຫຍັງ?**

Ultimate Multisite ແມ່ນ plugin ຂອງ WordPress, ເຊິ່ງສ້າງຂຶ້ນມາເພື່ອຕິດຕັ້ງ WordPress multisite, ມັນຈະປ່ຽນການຕິດຕັ້ງ WordPress ຂອງທ່ານໃຫ້ກາຍເປັນເຄືອຂ່າຍເວັບໄຊທ໌ພຣີມຽມ – ຄືກັບ [WordPress.com](https://WordPress.com) – ເຮັດໃຫ້ລູກຄ້າສາມາດສ້າງເວັບໄຊໄດ້ຜ່ານຄ່າບໍລິການປະຈຳເດືອນ, ປະຈຳໄຕມາດ, ຫຼື ປະຈຳປີ (ທ່ານຍັງສາມາດສ້າງແຜນຟຣີໄດ້ອີກ).

  * **Ultimate Multisite ຈະເຮັດວຽກໂດຍບໍ່ຕ້ອງມີ Multisite ໄດ້ບໍ່?**

ບໍ່ໄດ້. Ultimate Multisite ຮຽກຮ້ອງໃຫ້ມີ WordPress multisite, subdirectory, ຫຼື subdomain.

  * **ຂໍ້ກຳນົດສຳລັບ Ultimate Multisite ແມ່ນຫຍັງ?**

Ultimate Multisite ຮຽກຮ້ອງໃຫ້ມີ:

**WordPress Version** : v5.3+ (ແນະນຳ: ໃຊ້ເວີຊັນທີ່ໝັ້ນຄົງສຸດທ້າຍ)
**PHP Version** : 7.4.x (ຈະມີການສະໜັບສະໜູນ > 8.0 ໃນໄວໆນີ້)
**MySQL Version** : v5+ (ແນະນຳ: ເວີຊັນ 5.6, ໃນກໍລະນີທີ່ທ່ານບໍ່ມີ 8.0 ຢູ່ໃນ hosting provider ຂອງທ່ານ)

_Multisite (subdomain ຫຼື subdirectory) ກໍຕ້ອງໄດ້ຮັບການເປີດໃຊ້ງານດ້ວຍ._ [ວິທີຕິດຕັ້ງ WordPress Multisite](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **Ultimate Multisite ສາມາດໃຊ້ເພື່ອຈຳກັດ plugin ແລະ theme ທີ່ຜູ້ໃຊ້ສາມາດໃຊ້ໄດ້ບໍ່?**

ໄດ້. ດ້ວຍ Ultimate Multisite, ທ່ານສາມາດເລືອກໄດ້ວ່າ plugin ແລະ theme ໃດທີ່ມີໃຫ້ແຕ່ລະແຜນໃນເຄືອຂ່າຍຂອງທ່ານ.

  * **Ultimate Multisite ສາມາດໃຊ້ເພື່ອຈຳກັດຈຳນວນ post ທີ່ຜູ້ໃຊ້ສາມາດສ້າງໄດ້ໃນເວັບໄຊຂອງຕົນເອງບໍ່?**

ใช่ครับ ด้วย Ultimate Multisite คุณสามารถตั้งโควต้าโพสต์สำหรับ **ทุกประเภทของโพสต์ที่มีอยู่** ในเครือข่ายของคุณได้ นอกจากนี้ คุณยังสามารถจำกัดจำนวนผู้ใช้ที่แอดมินสามารถเชิญเข้ามา, **พื้นที่ดิสก์**, และอื่นๆ อีกมากมายได้ด้วย

  * **Ultimate Multisite รองรับ Site Templates ไหม?**

รองรับครับ ด้วย Ultimate Multisite คุณสามารถสร้าง Site Template ได้ตามที่คุณต้องการ Site Template ไม่ใช่ธีม แต่เป็นเว็บไซต์ปกติในเครือข่ายของคุณ คุณสามารถเปิดใช้งานปลั๊กอิน ธีม และปรับแต่ง Site Template นั้นได้ทุกแบบ หากผู้ใช้เลือกเทมเพลตนั้น เว็บไซต์ของเขาจะถูกสร้างขึ้นเป็นสำเนาของเทมเพลตนั้น พร้อมกับการตั้งค่าทั้งหมดที่คุณทำไว้

คุณยังสามารถกำหนดเทมเพลตเฉพาะสำหรับแผนใดแผนหนึ่งได้ การทำเช่นนั้น หมายความว่าผู้ใช้ทุกคนที่สมัครในแผนนั้นจะมีเว็บไซต์ของตนถูกสร้างขึ้นเป็นสำเนาของ Site Template นั้น

  * **Ultimate Multisite เชื่อมต่อกับ WooCommerce ได้ไหม?**

เชื่อมต่อได้ครับ!

หากคุณต้องการให้ผู้ใช้ในเครือข่ายของคุณสามารถเข้าถึง WooCommerce เพื่อตั้งร้านค้าได้ นั่นก็ทำได้โดยใช้ Ultimate Multisite คุณยังสามารถจำกัดจำนวนผลิตภัณฑ์ที่ลูกค้าของคุณจะสร้างได้ในแต่ละระดับแผนด้วย

สำหรับเรื่องการชำระค่าบริการของ Ultimate Multisite คุณสามารถใช้ [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) เพื่อให้ลูกค้า Ultimate Multisite ของคุณสามารถชำระเงินโดยใช้ WooCommerce gateways บนเว็บไซต์หลักของคุณได้ ด้วยวิธีนี้ คุณจะสามารถเข้าถึงคลัง payment gateways ขนาดใหญ่ที่พัฒนาสำหรับ WooCommerce และนำมาใช้เรียกเก็บเงินจากลูกค้า Ultimate Multisite ของคุณได้

  * **มีวิธีโยกย้ายการสมัครสมาชิกจาก WPMUDEV Pro Sites ไหม?**

ใช่แล้ว Ultimate Multisite ตอนนี้มี [เครื่องมือที่เป็นประโยชน์ช่วยให้คุณย้ายข้อมูลได้ง่ายขึ้น](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) สำหรับเครือข่าย Pro Sites ของคุณ

  * **ทำไมแท็บ “Allowed Plugins” ถึงว่างในหน้าแก้ไขแผนของฉัน?**

Ultimate Multisite จะอนุญาตหรือปฏิเสธการเข้าถึงปลั๊กอินที่ไม่ได้เปิดใช้งานในเครือข่ายของคุณเท่านั้น เมื่อปลั๊กอินถูกเปิดใช้งานในเครือข่ายแล้ว โดยธรรมชาติของมัน ปลั๊กอินนั้นจะถูกโหลดโดยอัตโนมัติในทุกไซต์ของเครือข่ายคุณ

หากคุณไม่เห็นปลั๊กอินใดๆ ในแท็บ “Allowed Plugins” ให้ลองตรวจสอบว่าปลั๊กอินที่คุณต้องการนั้นเปิดใช้งานในเครือข่ายหรือไม่

  * **ฉันสามารถใช้การออกแบบตารางราคาของฉันเองได้ไหม?**

ได้ครับ ตั้งแต่เวอร์ชัน 1.4.0 เป็นต้นมา Ultimate Multisite รองรับการเขียนทับเทมเพลต (template overrides) แล้ว

  * **Ultimate Multisite ใช้กับ WP Engine ได้ไหม?**

ใช้ได้ครับ Ultimate Multisite เคยรองรับโดย WP Engine มาตลอด แต่การทำให้การแมปโดเมนทำงานต้องมีการตั้งค่าเพิ่มเติมด้วยตนเองจากแอดมิน (การเพิ่มโดเมนในแผงควบคุมของ WP Engine) **ตอนนี้ไม่ใช่แบบนั้นแล้วตั้งแต่เวอร์ชัน 1.6.0** Ultimate Multisite เชื่อมต่อโดยตรงกับ API ของ WP Engine และโดเมนจะถูกเพิ่มเข้าไปในแผงควบคุมโดยอัตโนมัติหลังจากแมปเสร็จ

  * **คุณแนะนำผู้ให้บริการโฮสติ้งรายไหน?**

Ultimate Multisite สามารถใช้ได้กับผู้ให้บริการโฮสติ้งทุกราย ความแตกต่างเพียงอย่างเดียวคือ บริษัทโฮสติ้งบางแห่ง เช่น Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane และ [Runcloud.io](http://Runcloud.io) มีการผสานรวมที่แน่นหนากว่าสำหรับการแมปโดเมน ทำให้มันใช้งานได้ทันที สำหรับผู้ให้บริการโฮสติ้งรายอื่น คุณอาจต้องตั้งค่าการแมปโดเมนด้วยตนเอง
