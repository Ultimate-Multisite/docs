# ການຍົກລະດັບຈາກ V1 {#migrating-from-v1}

## Ultimate Multisite ໄດ້ປ່ຽນມາໃຊ້ແຟມ Gia 2.x ແທນແຟມ 1.x ເດີມຂອງມັນ. {#ultimate-multisite-has-switched-from-its-original-1x-family-of-releases-to-the-2x-family-of-releases}

Ultimate Multisite ໃສ່ເວີຊັນ 2.0 ແລະ ຂຶ້ນໄປ ແມ່ນການຂຽນໂຄດໃໝ່ທັງໝົດ, ຊຶ່ງໝາຍຄວາມວ່າ ມີສ່ວນຮ່ວມໜ້ອຍຫຼາຍລະຫວ່າງເວີຊັນເກົ່າກັບໃໝ່. ເພາະສະນັ້ນ, ເມື່ອອັບເດດຈາກ 1.x ໄປເປັນ 2.x, ຂໍ້ມູນຂອງທ່ານຈະຕ້ອງຖືກຍ້າຍໄປໃນຮູບແບບທີ່ເວີຊັນໃໝ່ສາມາດເຂົ້າໃຈໄດ້.

ຂອບໃຈກັບ Ultimate Multisite 2.0+ **ມີຕົວຊ່ວຍຍ້າຍ (migrator)** ຕິດຕັ້ງມາພ້ອມກັບ core ເຊິ່ງສາມາດກວດຫາຂໍ້ມູນຈາກເວີຊັນເກົ່າ ແລະ ປ່ຽນມັນເປັນຮູບແບບໃໝ່ໄດ້. ການຍ້າຍນີ້ຈະເກີດຂຶ້ນໃນລະຫວ່າງ **Setup Wizard** ຂອງເວີຊັນ 2.0+.

ບົດຮຽນນີ້ຈະກວມເອົາວິທີການເຮັດວຽກຂອງ migrator, ສິ່ງທີ່ຕ້ອງເຮັດໃນກໍລະນີເກີດຂໍ້ຜິດພາດ, ແລະ ວິທີແກ້ໄຂບັນຫາທີ່ອາດຈະເກີດຂຶ້ນໃນຂະບວນການນີ້.

_**ສຳຄັນ: ກ່ອນທີ່ທ່ານຈະເລີ່ມຍົກລະດັບຈາກເວີຊັນ 1.x ໄປເປັນເວີຊັນ 2.0, ກະລຸນາເຮັດການສຳຮອງ (backup) ຖານຂໍ້ມູນຂອງໄຊທ໌ຂອງທ່ານໃຫ້ແນ່ໃຈ.**_

## ຂັ້ນຕອນທຳອິດ {#first-steps}

ຂັ້ນຕອນທຳອິດແມ່ນການດາວໂຫຼດໄຟລ໌ plugin .zip ແລະ ຕິດຕັ້ງເວີຊັນ 2.0 ໃນ dashboard ຂອງ network admin ຂອງທ່ານ.

ຫຼັງຈາກທີ່ທ່ານ [install and activate version 2.0](../getting-started/installing-ultimate-multisite.md), ລະບົບຈະກວດພົບໂດຍອັດຕະໂນມັດວ່າ Multisite ຂອງທ່ານກຳລັງໃຊ້ເວີຊັນເກົ່າ ແລະ ທ່ານຈະເຫັນຂໍ້ຄວາມນີ້ຢູ່ເທິງໜ້າ plugin.

_**ໝາຍເຫດ: ຖ້າທ່ານມີ Ultimate Multisite 1.x ຕິດຕັ້ງຢູ່ໃນ Multisite ຂອງທ່ານ, ທ່ານຈະມີທາງເລືອກໃນການປ່ຽນ plugin ເປັນເວີຊັນທີ່ທ່ານໄດ້ດາວໂຫຼດມາໃໝ່. ກະລຸນາເຮັດຕໍ່ໄປ ແລະ ຄລິກເພື່ອ **Replace current with uploaded** (ປ່ຽນຂອງປັດຈຸບັນດ້ວຍໄຟລ໌ທີ່ອັບໂຫຼດ).**_

<!-- ພາບໜ້າຈໍບໍ່ມີ: ໜ້າ plugin ສະແດງທາງເລືອກໃນການປ່ຽນ v1 ເປັນເວີຊັນ v2 ທີ່ອັບໂຫຼດມາ -->

ໜ້າຕໍ່ໄປຈະແຈ້ງໃຫ້ທ່ານຮູ້ວ່າທ່ານໄດ້ຕິດຕັ້ງ legacy add-ons (ສ່ວນຂະຫຍາຍເກົ່າ) ອะไรໄວ້ພ້ອມກັບເວີຊັນ 1.x. ໜ້າ这ຈະມີຄຳແນະນຳວ່າເວີຊັນທີ່ທ່ານກຳລັງໃຊ້ແມ່ນເຂົ້າກັນໄດ້ກັບເວີຊັນ 2.0 ຫຼື ບໍ່, ຫຼື ທ່ານຕ້ອງການຕິດຕັ້ງ add-on ທີ່ອັບເກຣດແລ້ວຫຼັງຈາກການຍົກລະດັບ (migration) ຫຼື ບໍ່.

<!-- Screenshot unavailable: Message on the top of the plugins page about updating to v2.0, with list of add-ons that need updating -->

ເມື່ອທ່ານພ້ອມທີ່ຈະດຳເນີນຕໍ່, ທ່ານສາມາດກົດປຸ່ມທີ່ຂຽນວ່າ **Visit the Installer to finish the upgrade** (ເຂົ້າໄປທີ່ຕົວຕິດຕັ້ງເພື່ອສຳເລັດການອັບເກຣດ).

<!-- Screenshot unavailable: Button saying Visit the Installer to finish the upgrade -->

ຈາກນັ້ນ, ມັນຈະພາທ່ານໄປໜ້າ setup wizard (ຄູ່ມືການຕັ້ງຄ່າ) ພ້ອມກັບຂໍ້ຄວາມຕ້ອນຮັບບາງຢ່າງ. ທ່ານພຽງແຕ່ຕ້ອງກົດ **Get Started** (ເລີ່ມຕົ້ນ) ເພື່ອໄປໜ້າຕໍ່ໄປ.

<!-- Screenshot unavailable: Setup Wizard welcoming page with Get Started button -->

ຫຼັງຈາກກົດ **Get Started**, ມັນຈະປ່ຽນໜ້າໄປຫາ Pre-install Checks_._ ໜ້າ这ຈະສະແດງຂໍ້ມູນລະບົບ (System Information) ແລະ ການຕິດຕັ້ງ WordPress ຂອງທ່ານໃຫ້ເຫັນວ່າຕອບສະໜອງຕາມຂໍ້ກຳນົດຂອງ Ultimate Multisite ຫຼື ບໍ່.

<!-- Screenshot unavailable: Pre-install Checks page showing system requirements confirmation -->

ຂັ້ນຕອນຕໍ່ໄປແມ່ນການໃສ່ license key ຂອງ Ultimate Multisite ແລະ ເປີດໃຊ້ add-on ນັ້ນ. ອັນນີ້ຈະຮັບປະກັນວ່າທຸກຄຸນສົມບັດ, ລວມທັງ add-ons, ຈະສາມາດໃຊ້ງານໄດ້ໃນເວັບໄຊຂອງທ່ານ.

<!-- Screenshot unavailable: License activation page with license key input and Agree and Activate button -->

ຫຼັງຈາກໃສ່ key ແລ້ວ, ກົດ **Agree & Activate** (ຕົກລົງ ແລະ ເປີດໃຊ້).

ຫຼັງຈາກເປີດໃຊ້ license ແລ້ວ, ທ່ານສາມາດເລີ່ມການຕິດຕັ້ງຕົວຈິງໄດ້ໂດຍການກົດ **Install** (ຕິດຕັ້ງ) ໃນໜ້າຕໍ່ໄປ. ອັນນີ້ຈະສ້າງໄຟລ໌ ແລະ database ທີ່ຈຳເປັນໃຫ້ເວີຊັນ 2.0 ສາມາດເຮັດວຽກໄດ້ໂດຍອັດຕະໂນມັດ.

## ตอนนี้ถึงขั้นตอนการย้ายข้อมูล (migration) {#now-the-migration}

ตัว migrator มีระบบความปลอดภัยในตัวครับ คือมันจะทำการตรวจสอบทั้ง multisite ของคุณทั้งหมด เพื่อให้แน่ใจว่าข้อมูล Ultimate Multisite ทั้งหมดของคุณสามารถย้ายมาได้โดยไม่มีปัญหา ให้คลิกปุ่ม **Run Check** เพื่อเริ่มกระบวนการนี้ครับ

หลังจากรันการตรวจสอบเสร็จแล้ว คุณจะมีสองทางเลือกคือ ผลลัพธ์จะเป็นแบบ **มีข้อผิดพลาด (with error)** หรือ **ไม่มีข้อผิดพลาด (without error)** ครับ

### มีข้อผิดพลาด (With Error) {#with-error}

ถ้าคุณได้รับข้อความแจ้งข้อผิดพลาด คุณจะต้องติดต่อทีมสนับสนุนของเราเพื่อให้พวกเขาช่วยแก้ไขข้อผิดพลาดนั้นครับ อย่าลืม **แนบ log ข้อผิดพลาด (error log)** ไปด้วยตอนที่คุณสร้าง ticket นะครับ คุณสามารถดาวน์โหลด log ได้ หรือจะคลิกลิงก์ที่เขียนว่า contact our support team ก็ได้ มันจะเปิดหน้าต่างช่วยเหลือ (help widget) ที่อยู่ทางด้านขวาของหน้าเว็บคุณ ซึ่งจะมีช่องให้กรอกข้อมูลมาให้พร้อม รวมถึงมี error logs อยู่ในส่วนคำอธิบายด้วย

_**เนื่องจากระบบพบข้อผิดพลาด คุณจะไม่สามารถดำเนินการย้ายไปเป็นเวอร์ชัน 2.0 ได้ครับ จากนั้นคุณสามารถย้อนกลับไปใช้เวอร์ชัน 1.x เพื่อให้เครือข่ายของคุณทำงานต่อไปได้จนกว่าจะแก้ไขข้อผิดพลาดเสร็จสิ้นนะครับ**_

### ไม่มีข้อผิดพลาด (Without Error) {#without-error}

ถ้าระบบไม่พบข้อผิดพลาดใดๆ คุณจะเห็นข้อความแจ้งว่าสำเร็จ พร้อมปุ่ม **Migrate** ที่ด้านล่าง ซึ่งจะอนุญาตให้คุณดำเนินการย้ายข้อมูลต่อได้ ในหน้านี้ ระบบจะเตือนให้คุณสร้าง backup ฐานข้อมูลก่อนที่จะดำเนินการต่อ ซึ่งเราแนะนำอย่างยิ่งครับ ถ้าคุณมี backup อยู่แล้ว ให้กด **Migrate** ได้เลยครับ

และแค่นี้ก็เสร็จเรียบร้อย!

ເຈົ້າສາມາດເລືອກທີ່ຈະສືບຕໍ່ໃຊ້ Wizard setup ເພື່ອອັບເດດໂລໂກ້ ແລະ ສິ່ງຕ່າງໆໃນເຄືອຂ່າຍຂອງເຈົ້າ ຫຼື ເລີ່ມນຳໃຊ້ເມນູ ແລະ ໂຕໜ້າໃໝ່ຂອງ Ultimate Multisite version 2.0 ກໍໄດ້. ລອງຊອກຫາຄວາມມ່ວນກັບມັນເລີຍ!
