# ປິດການສະໝັກໃຊ້ຂອງຂ້ອຍໄດ້ແນວໃດ?

ຖ້າທ່ານມີ license Ultimate Multisite, ທ່ານສາມາດຍົກເລີກການຕໍ່ອາຍຸ (renewal) ໄດ້ທຸກເວລາທີ່ທ່ານຕ້ອງການ. ເພື່ອເຮັດນັ້ນ, ໃຫ້ເຮັດຕາມຂັ້ນຕອນខាងລຸ່ມນີ້:

<!-- Screenshot unavailable: Account page email with access link -->

ໃຊ້ອີເມວທີ່ທ່ານໄດ້ໃສ່ໃນເວລາຊື້ ແລະ ລະຫັດຜ່ານຂອງທ່ານ.

<!-- Screenshot unavailable: Login page with email and password fields -->

ນີ້ແມ່ນໜ້າເບິ່ງບັນຊີ (Account Page) ເປັນຄັ້ງທຳອິດຂອງທ່ານ:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

ເພື່ອຍົກເລີກການສະໝັກໃຊ້, ໃຫ້ໄປທີ່ເມນູທາງຊ້າຍ, ຊື່ວ່າ _Renewing & Billing._

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

ກົດໃສ່ລູກໜ້າ (arrow) ຢູ່ທາງຂວາເພື່ອເປີດປ່ອງຢ້ຽມຂ້າງ. ແລ້ວ, ທ່ານຄວນເລືອກຕົວເລືອກທີ່ຈະຍົກເລີກການຕໍ່ອາຍຸແບບ **auto-renew**.

ລະບົບຈະສະແດງຂໍ້ຄວາມຢືນຢັນໃຫ້ທ່ານເຫັນ.

ຖ້າທ່ານໝັ້ນໃຈວ່າຕ້ອງການຍົກເລີກການສະໝັກໃຊ້, ພຽງແຕ່ກົດປຸ່ມ _**Cancel Renewals**_. ຫຼັງຈາກເຮັດການກະທຳນີ້ແລ້ວ, ທ່ານຈະຖືກຂໍຕອບແບບສຳຫຼວດສັ້ນໆ.

ສຳເລັດ! ການສະໝັກໃຊ້ຂອງທ່ານຈະບໍ່ຖືກຕໍ່ອາຍຸໂດຍອັດຕະໂນມັດອີກ.

**ທ່ານຈະມີ key ທີ່ໃຊ້ໄດ້ຈົນກວ່າການສະໝັກໃຊ້ຂອງທ່ານຈະໝົດອາຍຸ**. ຖ້າທ່ານຕ້ອງການເປີດການສະໝັກໃຊ້ຄືນໃໝ່, ທ່ານຈະຕ້ອງເຮັດດ້ວຍຕົນເອງ.
