# ການຍົກເລີກສະມາຊິກ (v2) {#canceling-a-membership-v2}

_**ចំណាំສຳຄັນ: ບົດຄວາມນີ້ອ້າງອີງເຖິງ Ultimate Multisite version 2.x.**_

ການຍົກເລີກສະມາຊິກໃນຝ່າຍລູກຄ້າແມ່ນເຮັດໄດ້ງ່າຍພໍສົມຄວນ.

ຖ້າຜູ້ໃຊ້ຕ້ອງການຍົກເລີກສະມາຊິກຂອງຕົນ, ພວກເຂົາຄວນເຂົ້າໄປທີ່ໜ້າ **Account** ກ່ອນ.

ຢູ່ທີ່ນັ້ນ, ພວກເຂົາຄວນຄລິກໄປທີ່ສ່ວນ **Danger Zone**, ແລະ ຄລິກເພື່ອ **Delete Account**.

![Account page Danger Zone with Delete Account button](/img/account-page/danger-zone.png)

ຈະມີໂມດູນປ່ອຍຂຶ້ນມາຂໍການຢືນຢັນຂອງພວກເຂົາ. ພວກເຂົາຄວນເປີດໃຊ້ (toggle on) ຕົວເລືອກ **Confirm Account Deletion** ແລະ ຄລິກເພື່ອ **Delete Account**.

![Confirm Account Deletion dialog](/img/account-page/delete-account-confirm.png)

ການເຮັດແບບນີ້ຈະເຮັດໃຫ້ພວກເຂົາ Log out ອອກຈາກໄຊທ໌ຂອງທ່ານ ແລະ ຍົກເລີກສະມາຊິກຂອງພວກເຂົາໃນເຄືອຂ່າຍຂອງທ່ານ, ລວມທັງລຶບໄຊທ໌ຂອງພວກເຂົາອອກຈາກເຄືອຂ່າຍຂອງທ່ານ.

![Logged out after account deletion](/img/account-page/logged-out-after-delete.png)

![Membership canceled confirmation](/img/account-page/membership-canceled.png)

**ສິ່ງທີ່ຄວນຈື່:** ເຫັນວ່າການກະທຳນີ້ຈະບໍ່ເຮັດໃຫ້ລູກຄ້າຖືກໄລ່ອອກຈາກເຄືອຂ່າຍຂອງທ່ານ, ເພາະວ່າລູກຄ້າໜຶ່ງສາມາດມີສະມາຊິກໄດ້ຫຼາຍກວ່າໜຶ່ງ ຫຼື ບໍ່ມີສະມາຊິກໃດໆທີ່ເຊື່ອມໂຍງກັບພວກເຂົາ.
