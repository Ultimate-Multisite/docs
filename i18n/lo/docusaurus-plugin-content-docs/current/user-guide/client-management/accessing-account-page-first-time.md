# ວິທີເຂົ້າເຖິງໜ້າບັນຊີຂອງເຈົ້າເປັນຄັ້ງທຳອິດ?

ຍິນດີຕ້ອນຮັບສູ່ Ultimate Multisite!

ໃຫ້ເຂົ້າເຖິງໜ້າບັນຊີຂອງເຈົ້າຜ່ານລິ້ງທີ່ໄດ້ຮັບໃນອີເມວຫຼັງຈາກຊື້ plugin ຂອງພວກເຮົາ:

<!-- ຮູບພາບບໍ່ມີ: ອີເມວໜ້າບັນຊີທີ່ມີລິ້ງເຂົ້າເຖິງ -->

ໃຊ້ອີເມວທີ່ເຈົ້າໄດ້ໃຫ້ໄວ້ໃນເວລາຊື້ ແລະ ລະຫັດຜ່ານທີ່ເຈົ້າໄດ້ຮັບໃນອີເມວ.

<!-- ຮູບພາບບໍ່ມີ: ໜ້າເຂົ້າສູ່ລະບົບພ້ອມປ້ອນອີເມວແລະລະຫັດຜ່ານ -->

ເນື່ອງຈາກນີ້ແມ່ນຄັ້ງທຳອິດທີ່ເຈົ້າເຂົ້າສູ່ລະບົບບັນຊີນີ້, ເຈົ້າຈະຖືກຮ້ອງຂໍໃຫ້ປ່ຽນລະຫັດຜ່ານ. ໃຫ້ປ່ຽນມັນແລ້ວចុចປຸ່ມ _Save_ (ບັນທຶກ).

<!-- ຮູບພາບບໍ່ມີ: ແບບຟອມປ່ຽນລະຫັດຜ່ານພ້ອມປຸ່ມ Save -->

ໃນໜ້າບັນຊີຂອງເຈົ້າ, ເຈົ້າສາມາດຈັດການກັບລາຍລະອຽດບັນຊີທັງໝົດຂອງເຈົ້າໄດ້ ເຊັ່ນ: ການເກັບຄ່າບໍລິການ (billings), ການຕໍ່ອາຍຸ (renewals), ການຍົກເລີກ (cancellations) ແລະ ສິ່ງອື່ນໆອີກຫຼາຍຢ່າງ.
