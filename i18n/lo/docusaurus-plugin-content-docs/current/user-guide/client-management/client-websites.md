# เว็บไซต์ของลูกค้าคุณ (v2)

หลังจากลงทะเบียนในเครือข่ายของคุณแล้ว ลูกค้าของคุณจะสามารถเข้าถึงแดชบอร์ดของตัวเองได้

นอกจากหน้า **Account** ซึ่งเป็นที่ที่พวกเขาจะเห็นข้อมูลสำคัญเกี่ยวกับค่าใช้จ่าย การเป็นสมาชิก โดเมน ข้อจำกัดของแผน และอื่นๆ แล้ว พวกเขายังสามารถเข้าถึงหน้า **My Sites** ได้ด้วย ซึ่งเป็นที่ที่พวกเขาสามารถดูและจัดการเว็บไซต์ทั้งหมดของตนได้ สิ่งที่ต้องทำก็คือ คลิกที่ My Sites บนแดชบอร์ดของพวกเขา

![ลิงก์ My Sites ในแดชบอร์ดลูกค้า](/img/account-page/my-sites-link.png)

ระบบจะแสดงเว็บไซต์ทั้งหมดของพวกเขา พร้อมปุ่ม **Manage** อยู่ใต้แต่ละเว็บไซต์ หากพวกเขาคลิกที่ปุ่มนั้น ระบบจะนำไปยัง **dashboard ของเว็บไซต์นั้นโดยเฉพาะ**

พวกเขายังสามารถดูได้ว่าเว็บไซต์ใดเป็นเว็บไซต์ **หลัก (primary)** สำหรับแผนที่อนุญาตให้มีเว็บไซต์มากกว่าหนึ่งเว็บ

![หน้า My Sites แสดงเว็บไซต์ของลูกค้าพร้อมปุ่ม Manage](/img/account-page/my-sites-page.png)

นี่คือภาพรวมทั้งหมดของส่วนแก้ไขเว็บไซต์:

![อินเทอร์เฟซการแก้ไขเว็บไซต์แบบเต็ม](/img/admin/site-edit-full.png)

หากพวกเขาต้องการเปลี่ยนเว็บไซต์หลัก พวกเขาต้องไปที่หน้า Account โดยคลิกที่เมนู **Account**

![ลิงก์เมนู Account ในแดชบอร์ดลูกค้า](/img/account-page/account-menu.png)

พวกเขายังสามารถดูและแก้ไขรายละเอียดของลูกค้าได้ด้วย:

![อินเทอร์เฟซการแก้ไขลูกค้า](/img/admin/customer-edit.png)

จากนั้น พวกเขาควรคลิกที่ **Change Default Site** (เปลี่ยนเว็บไซต์เริ่มต้น)

![ปุ่ม Change Default Site บนหน้า Account](/img/account-page/change-default-site.png)

จะมีหน้าต่างป๊อปอัปปรากฏขึ้น ให้คลิกที่รายการแบบดรอปดาวน์แล้วเลือกเว็บไซต์ที่คุณต้องการให้เป็นเว็บไซต์หลัก

![การเลือกในเมนู Change Default Site dropdown](/img/account-page/change-default-site-dropdown.png)

จากนั้น แค่คลิกที่ **Change Default Site** แล้วก็เสร็จเรียบร้อย!

![ຢືນຢັນການປ່ຽນໄຊທ໌ເລີ່ມຕົ້ນ](https://example.com/img/account-page/change-default-site-confirm.png)
