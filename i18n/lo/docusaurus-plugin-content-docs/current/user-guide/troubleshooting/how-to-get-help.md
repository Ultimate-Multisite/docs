# ວິທີຂໍຄວາມຊ່ວຍເຫຼືອ

ເຈົ້າໄດ້ຮັບ License key ທີ່ຖືກຕ້ອງແລ້ວ ແລະ ມີການເຊື່ອມຕໍ່ກັບທີມງານຂອງພວກເຮົາເພື່ອຊ່ວຍဖြေရှင်းບັນຫາຕ່າງໆທີ່ເຈົ້າອາດຈະພົບໃນຂະນະທີ່ກຳລັງຕັ້ງຄ່າ Ultimate Multisite.

ບົດຮຽນນີ້ຈະກວມເອົາຊ່ອງທາງການສະໜັບສະໜູນທີ່ມີໃຫ້ສຳລັບຜູ້ຖື License ແລະ ຂໍ້ມູນປະເພດໃດທີ່ເຈົ້າຄວນມີໄວ້ເພື່ອຕິດຕໍ່ກັບທີມງານສະໜັບສະໜູນຂອງພວກເຮົາ.

## Help Site (ເວັບໄຊຊ່ວຍເຫຼືອ)

ເຈົ້າສາມາດເຂົ້າເຖິງບົດຄວາມ ແລະ ຄູ່ມືຕ່າງໆຈາກ help site ຂອງພວກເຮົາໄດ້ ເຊິ່ງເຈົ້າສາມາດພົບມັນໄດ້ໃນລິ້ງខាងລຸ່ມນີ້.

[)

ພວກເຮົາຂໍແນະນຳໃຫ້ຜູ້ໃຊ້ເຮັດຄວາມເຂົ້າໃຈກັບຄຸນສົມບັດ ແລະ ໜ້າທີ່ພື້ນຖານຂອງ Ultimate Multisite. ມີບົດຄວາມ "Getting Started" (ເລີ່ມຕົ້ນ) ຢູ່ໃນ help site ຂອງພວກເຮົາ ແລະ ຄຳຕອບຕໍ່ຄຳຖາມທົ່ວໄປກໍສາມາດຊອກໄດ້ນັ້ນ ເຊັ່ນ: ວິທີຕິດຕັ້ງ add-ons, ວິທີເຊື່ອມຕໍ່ payment gateways ແລະ ອື່ນໆ.

## Contact Support (ຕິດຕໍ່ສະໜັບສະໜູນ)

ຖ້າເຈົ້າជាប់ບັນຫາ ແລະ ຊອກຄຳຕອບຈາກ help site ຂອງພວກເຮົາບໍ່ພົບ, ເຈົ້າສາມາດຕິດຕໍ່ກັບທີມງານສະໜັບສະໜູນຂອງພວກເຮົາໄດ້ໂດຍການກົດທີ່ໄອຄອນຮູບຟອງສັບ (speech bubble icon) ທີ່ຢູ່ທາງຂ້າງລຸ່ມຂວາຂອງເວັບໄຊຂອງພວກເຮົາ ຫຼື ສົ່ງອີເມວໂດຍກົງໄປທີ່ support@ultimatemultisite.com.

ເມື່ອຕິດຕໍ່ສະໜັບສະໜູນເພື່ອບັນຫາດ້ານເຕັກນິກ, ກະລຸນາແນ່ໃຈວ່າເຈົ້າໄດ້ໃຫ້ຄຳອະທິບາຍລາຍລະອຽດຂອງບັນຫາດັ່ງກ່າວພ້ອມກັບຮູບພາບຈໍ (screenshot), ຂໍ້ຄວາມຜິດພາດ (error messages) ຫຼື ຂໍ້ມູນໃດໆທີ່ຈະເປັນປະໂຫຍດຕໍ່ພະນັກງານສະໜັບສະໜູນຂອງພວກເຮົາ ເພື່ອໃຫ້ເຂົາເຈົ້າເຂົ້າໃຈ ແລະ ປະເມີນບັນຫາຂອງເຈົ້າໄດ້ງ່າຍ. ເຈົ້າກໍຕ້ອງແຈ້ງ License key ຂອງເຈົ້າດ້ວຍ. ມີພຽງຜູ້ຖື license ທີ່ເຄື່ອນໄຫວເທົ່ານັ້ນທີ່ຈະໄດ້ຮັບການສະໜັບສະໜູນ.

## Community Site (ເວັບໄຊຊຸມຊົນ)

[community site](https://community.ultimatemultisite.com/) ຂອງພວກເຮົາ ແມ່ນບ່ອນໃຫ້ຜູ້ໃຊ້ Ultimate Multisite ສາມາດໂຕ້ຕອບກັນໄດ້. ມັນເຮັດໜ້າທີ່ເປັນລະບົບຂໍ້ມູນບ່ອນທີ່ສະມາຊິກສາມາດໂພສ, ຄອມเมนต์ໃນການສົນທະນາ, ໃຫ້ຄຳແນະນຳ, ຮ່ວມມື ຫຼື ເກັບຂໍ້ມູນຮ່ວມກັນໄດ້. ເຈົ້າຈະຖືກຮຽກຮ້ອງໃຫ້ໃສ່ Ultimate Multisite license key ຂອງເຈົ້າເພື່ອປົດລັອກພື້ນທີ່/ໜ້າສ່ວນຕົວ.
