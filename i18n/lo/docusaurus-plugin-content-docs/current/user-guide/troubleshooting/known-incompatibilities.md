# ความไม่เข้ากันที่ทราบกับปลั๊กอินอื่น ๆ

หลังจากติดตั้งและเปิดใช้งาน Ultimate Multisite แล้ว คุณอาจเจอปัญหาเนื่องจากมีความไม่เข้ากันบางอย่างกับปลั๊กอินตัวอื่นที่คุณยังไม่ได้แก้ไข

เรากำลังพยายามลดรายการนี้ลงในทุกการอัปเดตใหม่ แต่ถ้ามีปลั๊กอินใดอยู่ในรายการนี้ คุณอาจต้องปิดใช้งานมันเพื่อให้เครือข่าย Ultimate Multisite ทำงานได้อย่างสมบูรณ์

นี่คือรายการของปลั๊กอินเหล่านั้นและสิ่งที่คุณควรทำในกรณีเหล่านี้:

**ปลั๊กอิน WPMU DEV บางตัว** ปลั๊กอินบางตัวที่ _ไม่ได้ได้รับการดูแลโดย WPMU DEV อีกต่อไป_ เป็นที่ทราบกันว่าทำให้ขั้นตอนการลงทะเบียน Ultimate Multisite หยุดชะงักในขั้นตอนสุดท้าย (ผู้ใช้ไม่สามารถสร้างบัญชีได้) นี่คือรายการปลั๊กอินที่ก่อให้เกิดปัญหาเหล่านั้น:

  * Signup Language: _Ultimate Multisite มีส่วนเสริมที่ให้บริการฟังก์ชันเดียวกัน เรียกว่า Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _สิ่งนี้ถูกสร้างมาใน Ultimate Multisite แล้ว จึงไม่จำเป็นต้องใช้ปลั๊กอินเพิ่มเติม._

  * New Blog Templates: _สิ่งนี้ถูกสร้างมาใน Ultimate Multisite แล้ว จึงไม่จำเป็นต้องใช้ปลั๊กอินเพิ่มเติม._

  * Set Password on Multisite Blog Creation: _สิ่งนี้ถูกสร้างมาใน Ultimate Multisite แล้ว จึงไม่จำเป็นต้องใช้ปลั๊กอินเพิ่มเติม._

  * Signup TOS: _สิ่งนี้ถูกสร้างมาใน Ultimate Multisite แล้ว จึงไม่จำเป็นต้องใช้ปลั๊กอินเพิ่มเติม._

  * Anti-Splog: แม้ว่าจะมีความสามารถน้อยกว่า แต่ปลั๊กอินนี้ก็ _ถูกสร้างมาใน Ultimate Multisite ด้วยเช่นกัน._

**หมายเหตุ**: ปลั๊กอิน WPMU Dev อื่น ๆ ทั้งหมดสามารถใช้งานร่วมกับ Ultimate Multisite ได้ตามปกติ ตัวอย่างเช่น _Smush_ , _Forminator_ , _Defender,_ เป็นต้น
