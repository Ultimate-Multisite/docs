# ประกาศการนำโหมด Site Builder ออก {#site-builder-mode-removal-notice}

**โหมด Site Builder ได้ถูกนำออกใน Superdav AI Agent v1.12.0 แล้ว** หากคุณเคยใช้โหมด Site Builder คุณควรย้ายไปใช้ **Setup Assistant agent** สำหรับการสร้างธีมและการตั้งค่าเว็บไซต์แทนครับ/ค่ะ

## เกิดอะไรขึ้น? {#what-happened}

### โหมด Site Builder (แบบเก่า) {#site-builder-mode-legacy}

Site Builder mode เป็นหน้าจอที่ใช้ขั้นตอนเหมือนไกด์ (wizard-based interface) สำหรับ:

- การสร้างเว็บไซต์จากเทมเพลต
- การตั้งค่าพื้นฐานต่างๆ
- การเลือกธีม
- การตั้งค่าเนื้อหาเริ่มต้น

### สิ่งที่เข้ามาแทน? {#what-replaced-it}

**Setup Assistant agent** ตอนนี้จัดการฟังก์ชันทั้งหมดของ Site Builder ด้วย:

- การตั้งค่าที่ยืดหยุ่นกว่าและนำทางโดย Agent
- ตัวเลือกการปรับแต่งธีมที่ดีขึ้น
- การเชื่อมต่อกับการแนะนำ Theme Builder
- หน่วยความจำ site_brief ที่คงอยู่สำหรับการใช้งานในครั้งต่อไป

## หากคุณเคยใช้โหมด Site Builder {#if-you-were-using-site-builder-mode}

### เว็บไซต์ของคุณยังปลอดภัย {#your-sites-are-safe}

- เว็บไซต์ที่มีอยู่ซึ่งสร้างด้วย Site Builder mode ยังคงใช้งานได้ตามปกติ
- ไม่มีข้อมูลสูญหายหรือการขัดข้องของเว็บไซต์
- คุณสามารถจัดการเว็บไซต์ของคุณได้อย่างเป็นปกติ

### ย้ายไปใช้ Setup Assistant Agent {#migrate-to-setup-assistant-agent}

สำหรับการตั้งค่าเว็บไซต์ใหม่หรือการเปลี่ยนแปลงธีม ให้ใช้ Setup Assistant agent:

```
"Help me set up a new site"
```

หรือ

```
"Start the Theme Builder onboarding"
```

Setup Assistant agent มีฟังก์ชันที่เหมือนเดิมแต่มีความยืดหยุ่นมากขึ้นครับ/ค่ะ

## การเปรียบเทียบ: Site Builder กับ Setup Assistant {#comparison-site-builder-vs-setup-assistant}

| คุณสมบัติ | Site Builder (ถูกนำออก) | Setup Assistant (ใหม่) |
|---------|----------------------|----------------------|
| วิธีการตั้งค่า | แบบฟอร์มไกด์ (Wizard form) | การสนทนากับ Agent |
| การเลือกธีม | เทมเพลตที่กำหนดไว้ล่วงหน้า | สร้างขึ้นเองตามต้องการ |
| การปรับแต่ง | ตัวเลือกจำกัด | ระบบออกแบบเต็มรูปแบบ |
| site brief | ไม่ถูกบันทึก | หน่วยความจำที่คงอยู่ (Persistent memory) |
| เซสชันในอนาคต | ตั้งค่าซ้ำ | ใช้ site_brief ที่บันทึกไว้ |
| ความยืดหยุ่น | ขั้นตอนการทำงานแบบตายตัว | การสนทนาที่ปรับเปลี่ยนได้ตามสถานการณ์ |

## การย้ายไปใช้ Setup Assistant Agent {#migrating-to-setup-assistant-agent}

### สำหรับเว็บไซต์ใหม่ {#for-new-sites}

แทนที่จะใช้โหมด Site Builder:

1. คำขอ: "ช่วยฉันตั้งค่าเว็บไซต์ใหม่หน่อย"
2. ตัวช่วยตั้งค่า (Setup Assistant agent) จะแนะนำคุณผ่านขั้นตอนเหล่านี้:
   - วัตถุประสงค์และเป้าหมายของเว็บไซต์
   - กลุ่มเป้าหมาย
   - เอกลักษณ์ของแบรนด์
   - การสร้างธีม (Theme generation)
   - การตั้งค่าเริ่มต้น

### สำหรับเว็บไซต์ที่มีอยู่แล้ว {#for-existing-sites}

หากคุณมีเว็บไซต์ที่สร้างจากโหมด Site Builder มาก่อน:

1. คุณสามารถใช้งานมันต่อไปได้ตามเดิมเลย
2. หากต้องการอัปเดตธีม ให้ขอว่า: "ออกแบบเว็บไซต์ใหม่" (Redesign my site)
3. ตัวช่วยตั้งค่าจะช่วยคุณสร้างธีมใหม่ให้
4. ข้อมูลของเว็บไซต์ของคุณจะไม่เปลี่ยนแปลง

### สำหรับการเปลี่ยนแปลงธีม {#for-theme-changes}

แทนที่จะเลือกธีมในโหมด Site Builder:

1. คำขอ: "เปลี่ยนธีมของฉัน" (Change my theme)
2. ตัวช่วยตั้งค่าจะดำเนินการดังนี้:
   - สอบถามเกี่ยวกับความชอบในการออกแบบของคุณ
   - สร้างธีมที่กำหนดเองให้
   - เปิดใช้งานบนเว็บไซต์ของคุณ

## ความแตกต่างที่สำคัญ {#key-differences}

### โหมด Site Builder {#site-builder-mode}

```
1. เลือกเทมเพลต (template)
2. เลือกธีม (theme)
3. ตั้งค่าการตั้งค่าพื้นฐาน (basic settings)
4. เสร็จสิ้น
```

### ตัวช่วยตั้งค่า (Setup Assistant Agent) {#setup-assistant-agent}

```
1. อธิบายวัตถุประสงค์ของเว็บไซต์ของคุณ
2. กำหนดกลุ่มเป้าหมายของคุณ
3. เลือกความชอบในการออกแบบ
4. เอเจนต์สร้างธีมที่กำหนดเอง
5. เอเจนต์เปิดใช้งานธีม
6. บันทึกข้อมูลสรุปของเว็บไซต์ไว้สำหรับการใช้งานในครั้งต่อไป
```

## ประโยชน์ของตัวช่วยตั้งค่า (Setup Assistant Agent) {#benefits-of-setup-assistant-agent}

### ยืดหยุ่นกว่า {#more-flexible}

- อธิบายเว็บไซต์ของคุณด้วยภาษาพูดธรรมดา
- รับคำแนะนำที่ปรับให้เข้ากับความต้องการเฉพาะของคุณได้
- ปรับให้เข้ากับความต้องการเฉพาะของคุณได้

### การปรับแต่งที่ดีขึ้น {#better-customization}

- สร้างธีมที่กำหนดเอง
- การตัดสินใจเกี่ยวกับระบบการออกแบบ (Design system decisions)
- โทเค็นการออกแบบที่คงอยู่ (Persistent design tokens)

### หน่วยความจำที่คงอยู่ {#persistent-memory}

- ข้อมูลสรุปของเว็บไซต์ (`site_brief`) จะถูกเก็บไว้
- เอเจนต์ในอนาคตจะเข้าใจเว็บไซต์ของคุณ
- ไม่จำเป็นต้องใส่ข้อมูลการตั้งค่าซ้ำอีก

### ขั้นตอนการทำงานแบบบูรณาการ (Integrated Workflow) {#integrated-workflow}

- การเริ่มต้นสร้างธีม (Theme Builder onboarding)
- ทักษะด้านสุนทรียศาสตร์ของระบบการออกแบบ (Design System Aesthetics skill)
- ความสามารถในการควบคุมการมองเห็น (Ability Visibility controls)
- ทุกอย่างทำงานร่วมกันได้อย่างราบรื่น

## การแก้ไขปัญหา (Troubleshooting) {#troubleshooting}

### ฉันหาโหมด Site Builder ไม่เจอ {#i-cant-find-site-builder-mode}

โหมด Site Builder ได้ถูกนำออกแล้ว โปรดใช้ตัวช่วยตั้งค่า (Setup Assistant agent) แทน:

"ช่วยฉันตั้งค่าเว็บไซต์ใหม่หน่อย"

### ฉันต้องการสร้างเว็บไซต์ใหม่จาก Site Builder {#i-want-to-recreate-a-site-from-site-builder}

คุณสามารถสร้างเว็บไซต์ใหม่ได้ด้วย Setup Assistant agent:

1. ขอความช่วยเหลือ: "Help me set up a new site" (ช่วยฉันตั้งค่าเว็บไซต์ใหม่)
2. อธิบายวัตถุประสงค์และการออกแบบของเว็บไซต์เดิมของคุณ
3. Agent จะสร้างธีมที่คล้ายกันให้คุณ
4. ข้อมูลสรุปเว็บไซต์ (site_brief) ของคุณจะถูกเก็บไว้เพื่อใช้อ้างอิงในอนาคต

### เว็บไซต์ Site Builder ที่มีอยู่ของฉันใช้งานไม่ได้ {#my-existing-site-builder-site-isnt-working}

เว็บไซต์ที่สร้างขึ้นด้วยโหมด Site Builder ยังคงใช้งานได้ หากคุณประสบปัญหา:

1. ตรวจสอบว่าธีมของคุณยังเปิดใช้งานอยู่หรือไม่
2. ตรวจสอบว่าปลั๊กอินของคุณถูกเปิดใช้งานแล้ว
3. ตรวจสอบบันทึกข้อผิดพลาดของ WordPress (WordPress error logs)
4. ติดต่อฝ่ายสนับสนุนหากปัญหายังคงอยู่

### ฉันยังสามารถใช้เทมเพลต Site Builder เก่าได้ไหม? {#can-i-still-use-my-old-site-builder-templates}

เทมเพลต Site Builder ไม่พร้อมใช้งานอีกต่อไป อย่างไรก็ตาม:

- เว็บไซต์ที่มีอยู่ของคุณยังคงใช้งานได้
- คุณสามารถสร้างเว็บไซต์ที่คล้ายกันใหม่ได้ด้วย Setup Assistant agent
- Setup Assistant agent มีตัวเลือกในการปรับแต่งให้มากขึ้น

## ขั้นตอนต่อไป {#next-steps}

1. **สำหรับเว็บไซต์ใหม่**: ใช้ Setup Assistant agent
2. **สำหรับเว็บไซต์ที่มีอยู่แล้ว**: ใช้ต่อไปตามเดิม
3. **สำหรับการเปลี่ยนแปลงธีม**: ขอความช่วยเหลือจาก Setup Assistant agent
4. **สำหรับการปรับปรุงการออกแบบ**: ใช้ทักษะ Design System Aesthetics

## หัวข้อที่เกี่ยวข้อง {#related-topics}

- **Theme Builder Onboarding**: การตั้งค่าแบบมีคำแนะนำสำหรับธีมที่กำหนดเอง
