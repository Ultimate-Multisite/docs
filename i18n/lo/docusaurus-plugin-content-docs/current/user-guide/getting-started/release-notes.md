# บันทึกการออกเวอร์ชัน (Release Notes) {#release-notes}

## เวอร์ชัน 2.13.0 — เปิดตัวเมื่อวันที่ 2026-06-05 {#version-2130--released-on-2026-06-05}

- ใหม่: เพิ่มการรองรับ sovereign-tenant สำหรับบัญชีลูกค้า, การชำระเงิน, การเรียกเก็บเงิน, เว็บไซต์, ใบแจ้งหนี้, การสลับเทมเพลต และการแมปโดเมน เพื่อให้เครือข่ายของเทนเนนต์สามารถนำลูกค้ากลับไปยังเว็บไซต์หลักสำหรับการดำเนินการที่ต้องจัดการได้
- ใหม่: เพิ่มการตรวจสอบข้อมูลการต่ออายุสำหรับสมาชิกแบบสมัครสมาชิกซ้ำ (recurring memberships) เพื่อให้ gateway สามารถปิดการต่ออายุอัตโนมัติได้เมื่อไม่มีข้อตกลงการเรียกเก็บเงิน, การสมัครสมาชิก หรือ vault token ที่บันทึกไว้
- ใหม่: เพิ่ม HMAC-verified loopback publishing สำหรับการสร้างเว็บไซต์ที่รอดำเนินการ เพื่อให้การเตรียมพร้อมของ checkout ไปยังเว็บไซต์มีความน่าเชื่อถือมากขึ้นบนโฮสต์ที่งานเบื้องหลังล่าช้า
- ใหม่: เพิ่มจุดเชื่อมต่อสำหรับนักพัฒนา (developer extension points) สำหรับ SSO URLs, โดเมนพื้นฐานของฟอร์ม checkout และการสร้าง record ของโดเมนอัตโนมัติ
- แก้ไข: SSO มีความน่าเชื่อถือมากขึ้นในโดเมนที่แมปไว้, การเข้าชม anonymous broker, การออกจากระบบ และความขัดแย้งระหว่างปลั๊กอินต่างๆ
- แก้ไข: ตอนนี้การสร้างเว็บไซต์ที่รอดำเนินการจะสามารถกู้คืนจาก flag publish ที่เก่าและหลีกเลี่ยงการทำให้ลูกค้าติดอยู่ที่หน้าสร้างเว็บไซต์ได้อีก
- แก้ไข: ไม่มีการสร้าง record โดเมนสำหรับโดเมนพื้นฐานของฟอร์ม checkout ที่ใช้ร่วมกัน และงานเบื้องหลังของผู้ให้บริการโฮสต์ที่ไม่ได้ใช้งานจะถูกข้ามไปเมื่อไม่มีการเชื่อมต่อใดๆ ทำงานอยู่
- แก้ไข: กรณีพิเศษต่างๆ เช่น การชำระเงิน, ที่อยู่ใบแจ้งหนี้, การรีเซ็ตรหัสผ่าน, การยืนยันอีเมล, การสลับเทมเพลต, ทัวร์ และแดชบอร์ดลูกค้า ไม่ได้ขัดขวางขั้นตอนปกติของลูกค้าอีกต่อไป
- แก้ไข: อีเมลบรอดแคสต์ตอนนี้จะเก็บความเป็นส่วนตัวของผู้รับในขณะที่หลีกเลี่ยงข้อผิดพลาดร้ายแรงของ SMTP/ปลั๊กอินเมื่อรายการผู้รับหรือการส่งอีเมลล้มเหลว
- แก้ไข: กรณีพิเศษเกี่ยวกับการต่ออายุสมาชิก, การแสดงวันหมดอายุ และการเรียกเก็บเงิน ตอนนี้จะหลีกเลี่ยงการหมดอายุทันที, การค้าง หรือการพลาดการชำระเงินที่จำเป็น
- ปรับปรุง: ความเข้ากันได้กับ WordPress ได้รับการทดสอบจนถึงเวอร์ชัน 7.0, asset ของ Vue สำหรับ production ถูกสร้างใหม่จากแหล่งที่มาของ npm และ Cypress end-to-end coverage ตอนนี้จะทดสอบขั้นตอน checkout, การตั้งค่า, SSO และ gateway มากขึ้น

## 2.12.0 — ထုတ်လွှတ်သည့်နေ့ ၂၀၂၆-၀၅-၁၅ {#version-2120--released-on-2026-05-15}

- အသစ်ထည့်သွင်းမှု: Hostinger (hPanel) ကို ဒိုမိန်တွဲဖက်မှုနဲ့ ပေါင်းစပ်ပြီး ထောက်ပံ့တဲ့ host provider တစ်ခုအဖြစ် ထည့်သွင်းပေးထားပါတယ်
- အသစ်ထည့်သွင်းမှု: Site Exporter က အင်တာနက်ကွန်ရက်တစ်ခုလုံးအတွက် စာမျက်နှာပြန်လည်တည်ဆောက်မှုကို ပိုမိုလွယ်ကူစေဖို့ network import bundles တွေကို ကိုင်တွယ်နိုင်လာပါပြီ
- ပြင်ဆင်ချက်: BCC broadcast email တွေမှာ လက်ခံသူရဲ့ လိပ်စာတွေ မပေါက်ကြားအောင် header တစ်ခုကို သုံးစွဲလာပါတယ်
- ပြင်ဆင်ချက်: ရက်စွဲမဟုတ်တဲ့ တန်ဖိုးနဲ့ သိမ်းတဲ့အခါ အဖွဲ့ဝင်သက်တမ်းကုန်ဆုံးရက် မှားယွင်းတာတွေကို ပြုပြင်ပေးထားပါတယ်
- ပြင်ဆင်ချက်: Stripe အဖွဲ့ဝင်အခြေအနေတွေမှာ discount တွေကို သတ်မှတ်မလိုဘဲ ပျောက်သွားအောင် လုပ်ဆောင်နိုင်ဖို့ deprecated deleteDiscount API ကို မခေါ်တော့ပါဘူး
- ပြင်ဆင်ချက်: ဒိုမိန်တွဲဖက်ထားတဲ့ စာမျက်နှာတွေပေါ်က SSO ပြန်လည်ညွှန်းဆိုမှုတွေကို အဆုံးမရှိ လည်ပတ်တာမျိုး မဖြစ်အောင် ကန့်သတ်ပေးထားပါတယ်
- ပြင်ဆင်ချက်: Setup wizard မှာ ပုံရိပ်ရွေးတဲ့အခါ နောက်ကွယ်က data model ကို မှန်ကန်စွာ အပ်ဒိတ်လုပ်နိုင်ဖို့ ပြုပြင်ပေးထားပါတယ်
- ပြင်ဆင်ချက်: Site Exporter CLI က သတ်မှတ်ထားတဲ့ network site ရွေးချယ်မှုကို မှန်ကန်စွာ ထိန်းသိမ်းထားနိုင်အောင် လုပ်ဆောင်ပေးထားပါတယ်
- တိုးတက်မှု: plugin အရွယ်အစား လျှော့ချဖို့ bundled wp-cli ကို plugin package ထဲကနေ ဖယ်ရှားလိုက်ပါတယ်

## เวอร์ชัน 2.9.0 — เปิดตัวเมื่อวันที่ 2026-04-30 {#version-290--released-on-2026-04-30}

- แก้ไข: ไฟล์ ZIP สำหรับการส่งออกจะถูกเสิร์ฟผ่าน endpoint ดาวน์โหลดที่ต้องมีการยืนยันตัวตนแล้ว

- แก้ไข: บันทึก pending_site ที่หลุดหายจะถูกล้างออกเมื่อไม่มีสมาชิกอยู่
- แก้ไข: การจัดระยะห่าง (padding) ของแถบนำทาง Settings และ anchor search ถูกแก้ไขแล้ว
- แก้ไข: เว็บไซต์ที่รอการเผยแพร่จะแสดงเป็นอันดับแรกในมุมมอง All Sites
- แก้ไข: เพิ่ม User-Agent header สำหรับผู้ให้บริการ Screenshot (mShots) เพื่อป้องกันข้อผิดพลาด 403
- แก้ไข: ปัญหา circular dependency ของ cron schedule ในการนำเข้าได้รับการแก้ไขแล้ว
- แก้ไข: Tour IDs ถูกปรับให้เป็น underscore ในคีย์การตั้งค่าผู้ใช้
- ปรับปรุง: ตอนนี้ใช้ ZipArchive แทน Alchemy/Zippy เพื่อความเข้ากันได้ที่ดีขึ้น

- แก้ไข: ฟังก์ชัน `get_site_url()` จะไม่คืนค่าว่างอีกต่อไปเมื่อโดเมนมีพอร์ตอยู่ด้วย
- แก้ไข: การคัดลอกไฟล์สื่อ (media files) ถูกคัดลอกอย่างถูกต้องเมื่อการตั้งค่า `copy_media` เป็นค่าว่าง
- แก้ไข: Object cache จะถูกยกเลิกการใช้งานอย่างถูกต้องหลังจากเขียน sitemeta หลังจากการเปิดใช้งานเครือข่าย (network-activate)
- แก้ไข: โดเมนที่กำหนดเองจะถูกโปรโมตเป็นหลักโดยอัตโนมัติเมื่อมีการตรวจสอบ DNS สำหรับโดเมนที่มี 3 ส่วน
- แก้ไข: การสมัครสมาชิกที่ค้างอยู่จะถูกยกเลิกเมื่อการชำระเงินหมดอายุได้รับการทำความสะอาดแล้ว
- แก้ไข: ตัวตรวจสอบความแข็งแกร่งของรหัสผ่าน (Password strength checker) จะกลับมาทำงานใหม่หลังจากปิดหน้าต่างแจ้งเตือนการเข้าสู่ระบบแบบอินไลน์ (inline login prompt dismissed)
- แก้ไข: การโหลดหน้าเว็บซ้ำไม่สิ้นสุดบนหน้าขอบคุณจะหยุดทำงานเมื่อเว็บไซต์ถูกสร้างขึ้นแล้ว

- แก้ไข: เพิ่มการป้องกันค่าว่าง (Null expiration guard) ใน `calculate_expiration` เพื่อให้เข้ากันได้กับ PHP 8.4
