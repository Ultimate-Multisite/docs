# ເປັນຫຍັງຕ້ອງໃຊ້ Ultimate Multisite

ຖ້າທ່ານໄດ້ມາຮອດບົດຄວາມນີ້, ອາດຈະໄດ້ຍິນວ່າ plugin ຂອງພວກເຮົາອາດເປັນຄຳຕອບທີ່ຈະເຮັດໃຫ້ແນວຄິດທຸລະກິດຂອງທ່ານເປັນຈິງ. ພວກເຮົາຫວັງວ່າທ່ານຈະໄດ້ຮັບຄຳຕອບທີ່ຕ້ອງການໃນທ້າຍບົດຄວາມນີ້.

## Ultimate Multisite ແມ່ນຫຍັງ? {#what-is-ultimate-multisite}

Ultimate Multisite ແມ່ນ plugin ຂອງ WordPress Multisite ເຊິ່ງຊ່ວຍໃຫ້ທ່ານສາມາດສະເໜີບໍລິການ WaaS (Website as a Service) ແກ່ລູກຄ້າໄດ້. ດ້ວຍ plugin ຂອງພວກເຮົາ, ທ່ານສາມາດມີລະດັບການສະໝັກສະມາຊິກທີ່ຫຼາກຫຼາຍ ແລະ ໃຫ້ລູກຄ້າຈ່າຍຄ່າທຳນຽມເປັນປະຈຳເພື່ອໃຫ້ມີເວັບໄຊຕ໌ຖືກໂຮດໃນເຄືອຂ່າຍ Multisite ຂອງທ່ານ.

Ultimate Multisite ຊ່ວຍໃຫ້ທ່ານສາມາດຮັບເງິນຜ່ານ gateway ທີ່ແຕກຕ່າງກັນ (ເຊັ່ນ: WooCommerce ຫຼື Stripe), ສ້າງ template ເພື່ອເຮັດໃຫ້ຊີວິດຂອງທ່ານງ່າຍຂຶ້ນເມື່ອມີຜູ້ໃຊ້ໃໝ່ລົງທະບຽນໃນເຄືອຂ່າຍຂອງທ່ານ, ຈັດການ domain mapping, ແລະ ເຮັດໃຫ້ແບບຟອມລົງທະບຽນຂອງທ່ານເປັນສ່ວນຕົວເພື່ອສ້າງປະສົບການທີ່ປັບແຕ່ງສະເພາະເຈາະຈົງໃຫ້ກັບລູກຄ້າ.

ແລະ ນີ້ພຽງແຕ່ແມ່ນການເລີ່ມຕົ້ນເທົ່ານັ້ນ ເພາະທ່ານສາມາດຂະຫຍາຍຟັງຊັນຂອງມັນດ້ວຍ add-ons ແລະ webhooks ໄດ້ອີກ.

## ຄຸນສົມບັດອື່ນໆ {#other-features}

Ultimate Multisite ຍັງມີ:

  * ການເຊື່ອມຕໍ່ Domain Mapping ກັບຜູ້ໃຫ້ບໍລິການ Managed WordPress hosting: Ultimate Multisite ຈະເພີ່ມ domain mapping ອັດຕະໂນມັດເຂົ້າໃນ admin panels ຂອງ WP Engine, Cloudways, cPanel, RunCloud, Closte ແລະ ອື່ນໆ ເມື່ອມີ domain ໃໝ່ຖືກ map, ເພື່ອບໍ່ໃຫ້ທ່ານຕ້ອງເຮັດດ້ວຍຕົນເອງ. ໃນບາງກໍລະນີ, ພວກເຮົາແມ່ນຈະສັ່ງໃຫ້ດຶງ SSL certificate ມາໃຫ້ທ່ານເລີຍ.

  * Webhooks: Ultimate Multisite ມີລະບົບ Webhook ທີ່ຊົງພະລັງ ເຊິ່ງຊ່ວຍໃຫ້ທ່ານສາມາດເຊື່ອມຕໍ່ເຄືອຂ່າຍຂອງທ່ານກັບບໍລິການຕ່າງໆໄດ້ຫຼາຍຢ່າງ. ພວກເຮົາແມ່ນຈະສະເໜີການເຊື່ອມຕໍ່ໂດຍກົງກັບ Zapier, ເຊິ່ງເຮັດໃຫ້ເຄືອຂ່າຍຂອງທ່ານເຂົ້າເຖິງແອັບ ແລະ ບໍລິການຫຼາຍກວ່າ 1000 ແອັບທີ່ມີຢູ່ໃນ platform ຂອງເຂົາເຈົ້າໄດ້ເລີຍ.

หน้า Admin ที่ทรงพลังเพื่อช่วยลูกค้าของคุณใช้ Page Builder ที่คุณชื่นชอบ: Ultimate Multisite มี add-on ที่มีประโยชน์ชื่อว่า Admin Pages Creator ซึ่งโดยพื้นฐานแล้วช่วยให้คุณเพิ่ม Admin Pages ที่ปรับแต่งได้อย่างเต็มรูปแบบให้กับแดชบอร์ดของลูกค้า โดยใช้เทมเพลตจาก Elementor, Beaver Builder และ Brizy

ธีม Admin สองแบบที่แตกต่างกันเพื่อปรับแต่งประสบการณ์แดชบอร์ดของลูกค้าของคุณ: Check Material WP และ PRO

และฟีเจอร์ขั้นสูงอื่นๆ:

* การรองรับ Template Overrides: ด้วย Ultimate Multisite คุณสามารถแทนที่เทมเพลต HTML เริ่มต้นสำหรับขั้นตอนการสมัครสมาชิกด้วยเวอร์ชันของคุณเองได้ โดยไม่ต้องกังวลว่าจะต้องแก้ไขไฟล์หลักของ plugin
* Search and Replace API: คุณยังสามารถใส่ placeholder ลงใน site templates เช่น {{site_name}} และใช้ Search and Replace API เพื่อเติมข้อมูลเหล่านั้นโดยอัตโนมัติ ด้วยข้อมูลที่ลูกค้าให้มาหลังจากสมัครสมาชิกเสร็จสิ้น

## กรณีการใช้งาน (Use cases) {#use-cases}

ตอนนี้ถึงเวลาที่เราจะมาดูตัวอย่างกรณีการใช้งานที่เป็นไปได้ของ Ultimate Multisite กันครับ

### กรณีที่ 1: เอเจนซี่ {#case-1-an-agency}

โดยปกติแล้วทักษะหลักของเอเจนซี่จะอยู่ที่การออกแบบเว็บไซต์ โดยมีส่วนต่างๆ เช่น การโฮสต์หรือการตลาดถูกระบุเป็นบริการเพิ่มเติม

สำหรับเอเจนซี่ Ultimate Multisite นำเสนอข้อเสนอที่มีคุณค่าอย่างเหลือเชื่อในความสามารถในการโฮสต์และจัดการเว็บไซต์หลายแห่งบนแพลตฟอร์มเดียว ยิ่งไปกว่านั้น เอเจนซี่ที่กำหนดมาตรฐานการออกแบบของตนด้วยธีมเฉพาะ เช่น GeneratePress, Astra, OceanWP หรืออื่นๆ สามารถใช้ประโยชน์จากความสามารถของ Ultimate Multisite เพื่อเปิดใช้งานธีมเหล่านี้โดยอัตโนมัติสำหรับแต่ละไซต์ใหม่

คุณจะสามารถควบคุม plugin และ theme ได้อย่างใกล้ชิดในระดับผลิตภัณฑ์ โดยผ่านการตั้งค่าของ Ultimate Multisite ซึ่งช่วยให้สามารถทำให้ plugin และ theme พร้อมใช้งานหรือซ่อนได้ รวมถึงสถานะการเปิดใช้งานเมื่อมีการสร้าง (instantiated) สำหรับไซต์ใหม่

เอเจนซี่ต่างๆ จะรู้สึกสบายใจมากขึ้นด้วย Ultimate Multisite ที่ช่วยให้พวกเขาสามารถทำในสิ่งที่พวกเขาเก่งที่สุดได้ นั่นคือ การออกแบบเว็บไซต์ที่ยอดเยี่ยม

### กรณีศึกษาที่ 2: ผู้ให้บริการเฉพาะกลุ่ม (Niche Provider) {#case-2-niche-provider}

มีคำกล่าวเก่าๆ ว่า "ทำสิ่งใดสิ่งหนึ่งให้ดี" สำหรับผู้เชี่ยวชาญหลายคน นี่หมายถึงการสร้างผลิตภัณฑ์หรือบริการโดยมีแนวคิดหลักเพียงอย่างเดียว

ถ้าคุณเน้นไปที่การพัฒนาเครือข่ายเว็บไซต์เฉพาะกลุ่ม คุณอาจต้องการให้บริการโดยใช้โครงสร้างและแพลตฟอร์มที่เป็นมาตรฐาน นั่นคือจุดที่เทมเพลตเว็บไซต์จะช่วยได้!

หนึ่งในฟีเจอร์สุดล้ำของ Ultimate Multisite คือ template site ซึ่งก็คือ เว็บไซต์ที่มีธีม (theme) ติดตั้งและเปิดใช้งานแล้ว มีปลั๊กอินที่จำเป็นติดตั้งและเปิดใช้งาน พร้อมด้วยโพสต์หรือหน้าตัวอย่างที่สร้างไว้ เมื่อลูกค้าสร้างเว็บไซต์ใหม่โดยใช้เทมเพลต เนื้อหาและการตั้งค่าของเทมเพลตจะถูกคัดลอกไปยังเว็บไซต์ที่สร้างขึ้นใหม่นั้น

สำหรับผู้ให้บริการเว็บไซต์และบริการเฉพาะกลุ่ม สิ่งนี้ให้ความได้เปรียบที่ไม่มีใครเทียบได้ในด้านความสามารถในการสร้างเว็บไซต์ที่พร้อมใช้งานทันที พร้อมปลั๊กอินและการออกแบบที่กำหนดเอง ลูกค้าเพียงแค่ต้องให้ข้อมูลขั้นต่ำเพื่อทำให้บริการเสร็จสมบูรณ์เท่านั้น

### กรณีศึกษาที่ 3: การโฮสต์ WordPress เว็บไซต์ (WordPress Web Hosting) {#case-3-wordpress-web-hosting}

มีหลายวิธีในการโฮสต์เว็บไซต์ WordPress แต่แทบไม่เคยง่ายเท่ากับการจัดเตรียมพื้นที่บนเว็บให้กับลูกค้าพร้อมกับเวอร์ชันของ WordPress ที่ติดตั้งไว้ล่วงหน้า นี่เป็นเพราะต้องมีการตัดสินใจและข้อควรพิจารณาหลายอย่างมารวมกันเพื่อให้ได้บริการที่มีความหมาย

Ultimate Multisite โดดเด่นในด้านนี้ด้วยการนำเสนอโซลูชันแบบครบวงจร (turnkey solution) สำหรับการโฮสต์เว็บไซต์ WordPress ในส่วนของโซลูชันประกอบด้วยกลไกหลักเพื่อให้บริการแบบสมัครสมาชิก การเก็บเงิน ฟอร์มชำระเงิน แบบฟอร์มเช็คเอาท์ คูปองส่วนลด และการสื่อสารกับลูกค้า

งานส่วนใหญ่ที่ต้องทำในการติดตั้ง ตั้งค่า และดูแลรักษา WordPress Multisite ให้ถูกต้องนั้น จะได้รับการอำนวยความสะดวกโดย Ultimate Multisite ซึ่งช่วยให้ผู้ดูแลระบบเครือข่ายไม่ต้องกังวลกับเรื่องต่างๆ ที่เกี่ยวข้องกับบริการหรือกลุ่มเป้าหมายของตน เช่น ระดับผลิตภัณฑ์ ราคา และข้อเสนอการบริการ

สำหรับนักพัฒนาที่ต้องการเชื่อมต่อกับ Ultimate Multisite ทางโซลูชันนี้ยังมี RESTful API และ Webhooks สำหรับแจ้งเตือนเหตุการณ์อย่างครบถ้วนด้วย

โดยไม่ต้องพึ่งพาปลั๊กอินและใบอนุญาตภายนอกมากมาย Ultimate Multisite ก็เป็นโซลูชันที่มีฟีเจอร์เยอะและเทียบเคียงได้กับ Wix, Squarespace, [WordPress.com] และอื่นๆ ครับ
