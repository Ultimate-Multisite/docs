# ຕົວເລືອກພາສາ {#language-selector}

Multisite Ultimate: ຕົວເລືອກພາສາໃນການລົງທະບຽນ

ເພີ່ມຊ່ອງຕົວເລືອກພາສາໃສ່ໃນຂັ້ນຕອນການລົງທະບຽນ.

## ການຕິດຕັ້ງ {#installation}

1. ອັບໂຫຼດ 'ultimate-multisite-language-selector' ໄປທີ່ໄດເຣັກທໍຣີ '/wp-content/plugins/'
2. ເປີດໃຊ້ປລັກອິນຜ່ານເມນູ 'ປລັກອິນ' ໃນ WordPress Network Admin ຂອງທ່ານ

## ຄູ່ມືການໃຊ້ງານ {#usage-guide}

ດ້ວຍ add-on ຟຣີນີ້ ທ່ານສາມາດໃຫ້ຜູ້ໃຊ້ຂອງທ່ານເລືອກພາສາທີ່ພວກເຂົາຢາກເຫັນໃນຂັ້ນຕອນການລົງທະບຽນ ແລະໃນໜ້າລັອກອິນ.

ຫຼັງຈາກ [ເປີດໃຊ້ໃບອະນຸຍາດຂອງທ່ານ](../../user-guide/getting-started/installing-ultimate-multisite.md) ແລ້ວ ທ່ານຈະສາມາດຕິດຕັ້ງ Ultimate Multisite add-ons ທີ່ໃບອະນຸຍາດຂອງທ່ານໃຫ້ສິດເຂົ້າເຖິງ.

ໄປທີ່ໜ້າການຕັ້ງຄ່າ Ultimate Multisite ຂອງທ່ານ ແລະຈາກນັ້ນ ໃນເມນູທາງຊ້າຍ ໃຫ້ຄລິກປ້າຍ _Add-ons_.

![ໜ້າ Add-ons](/img/admin/addons-page.png)

_ໄປທີ່ໜ້າ Add-ons ເພື່ອເລີ່ມຕິດຕັ້ງຕົວເລືອກພາສາ_

ຢູ່ທີ່ນັ້ນ ທ່ານຈະພົບ add-ons ທັງໝົດທີ່ພວກເຮົານຳສະເໜີ.

Add-ons ຟຣີມີໃຫ້ກັບລູກຄ້າຂອງພວກເຮົາທຸກຄົນ ໂດຍບໍ່ຂຶ້ນກັບແຜນການສະໝັກໃຊ້ທີ່ພວກເຂົາລົງທະບຽນ.

ຄລິກປຸ່ມ _ຕິດຕັ້ງຕອນນີ້!_ ຢູ່ດ້ານລຸ່ມຮູບຕົວຢ່າງຂອງ add-on. ຈາກນັ້ນເປີດໃຊ້ມັນໃນໜ້າປລັກອິນ.

![ໜ້າ Add-ons ພ້ອມປຸ່ມຕິດຕັ້ງ](/img/admin/addons-page.png)

_ຄລິກປຸ່ມເພື່ອຕິດຕັ້ງ add-on_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_ເປີດໃຊ້ມັນໃນໜ້າປລັກອິນຂອງທ່ານ_

ຕອນນີ້ ໄປທີ່ເມນູການຕັ້ງຄ່າຂອງ Ultimate Multisite ແລະເຂົ້າແທັບ _ການຕັ້ງຄ່າເຄືອຂ່າຍ_. ຢູ່ທີ່ນັ້ນ ດ້ານລຸ່ມຂອງໜ້າ ທ່ານຈະພົບສ່ວນໃໝ່ທີ່ເອີ້ນວ່າ _ພາສາ_.

![ໜ້າການຕັ້ງຄ່າ](/img/config/settings-general.png)

_ສ່ວນໃໝ່ສົດໆປາກົດຂຶ້ນ!_

ທ່ານສາມາດປະໄວ້ຫວ່າງເພື່ອສະແດງພາສາທັງໝົດເປັນຕົວເລືອກ ຫຼືເລືອກພາສາທີ່ທ່ານຢາກໃຫ້ພ້ອມໃຊ້. ພາສາເຫຼົ່ານີ້ແມ່ນພາສາທີ່ທ່ານໄດ້ຕິດຕັ້ງໃນການຕັ້ງຄ່າ WordPress ຂອງທ່ານ.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_ລາຍຊື່ພາສາທີ່ພ້ອມໃຊ້ໃນການຕິດຕັ້ງ WP ນີ້_

ຫຼັງຈາກບັນທຶກການປ່ຽນແປງທີ່ທ່ານໄດ້ເຮັດ ເມນູເລືອກພ້ອມພາສາທີ່ທ່ານເລືອກຈະຖືກສະແດງໃນໜ້າລັອກອິນ ລວມທັງໃນຂັ້ນຕອນການລົງທະບຽນ.

![ການຕັ້ງຄ່າລັອກອິນ ແລະການລົງທະບຽນ](/img/config/settings-login-registration.png)

_ຕົວເລືອກພາສາໃນໜ້າລັອກອິນ_

## ການຕັ້ງຄ່າພາສາຜ່ານ URL {#setting-the-language-via-the-url}

ທ່ານຍັງສາມາດປ່ຽນພາສາຂອງໜ້າໂດຍສົ່ງພາລາມິເຕີ locale ໄປຍັງ URL. ຖ້າທ່ານຕ້ອງການປ່ຽນເສັ້ນທາງໄປຫາໜ້າລົງທະບຽນໃນພາສາອິຕາລີ ຕົວຢ່າງເຊັ່ນ ເພີ່ມ ?locale=it_IT ໃສ່ URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

ພາສານັ້ນຕ້ອງຖືກຕິດຕັ້ງໃນເຄືອຂ່າຍ WordPress ຂອງທ່ານເພື່ອໃຫ້ສິ່ງນີ້ເຮັດວຽກໄດ້.
