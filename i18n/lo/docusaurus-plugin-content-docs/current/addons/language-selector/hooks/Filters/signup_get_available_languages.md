# ຕົວກັ່ນຕອງ: signup_get_available_languages {#filter-signupgetavailablelanguages}

ກັ່ນຕອງລາຍຊື່ພາສາທີ່ມີໃຫ້ສຳລັບການລົງທະບຽນເວັບໄຊດ້ານໜ້າ.

ການສົ່ງອາເຣເປົ່າໃຫ້ hook ນີ້ຈະປິດການສະແດງຜົນຂອງການຕັ້ງຄ່າໃນແບບຟອມລົງທະບຽນ, ແລະພາສາເລີ່ມຕົ້ນຈະຖືກໃຊ້ເມື່ອສ້າງເວັບໄຊ. ພາສາທີ່ຍັງບໍ່ໄດ້ຕິດຕັ້ງຈະຖືກຕັດອອກ.

## ພາຣາມິເຕີ {#parameters}

| ຊື່ | ປະເພດ | ຄຳອະທິບາຍ |
|------|------|-------------|
| $available_languages | `array` | ພາສາທີ່ມີໃຫ້. |
