# ຕົວສົ່ງອອກເວັບໄຊ {#site-exporter}

## ເນື້ອຫາ {#content}

ພວກເຮົາເຂົ້າໃຈຄວາມຈຳເປັນໃນການທີ່ສາມາດຍ້າຍເວັບໄຊຍ່ອຍພາຍໃນເຄືອຂ່າຍຂອງທ່ານໄປຫາການຕິດຕັ້ງເວັບໄຊດຽວ ແລະ ກັບກັນ ຖ້າລູກຄ້າຂອງທ່ານຕັດສິນໃຈຍ້າຍເວັບໄຊຂອງພວກເຂົາ. ສ່ວນເສີມຕົວສົ່ງອອກ ແລະ ຕົວນຳເຂົ້າເວັບໄຊເຮັດໃຫ້ວຽກນີ້ງ່າຍຂຶ້ນ. ເຄື່ອງມືນີ້ສົ່ງອອກ ແລະ ນຳເຂົ້າເວັບໄຊໂດຍໃຊ້ໄຟລ໌ zip ເຊິ່ງຍັງສາມາດໃຊ້ເປັນສຳເນົາສຳຮອງໄດ້.

## ຄຸນສົມບັດ {#features}

* ສົ່ງອອກ ແລະ ນຳເຂົ້າເວັບໄຊ

* ອິນເຕີເຟດທີ່ໃຊ້ງານງ່າຍ

* ສ້າງສຳເນົາສຳຮອງເວັບໄຊ

## ເອກະສານປະກອບ {#documentation}
**ວິທີສົ່ງອອກ ແລະ ນຳເຂົ້າເວັບໄຊ**

## ການຕິດຕັ້ງ {#installation}

1. ອັບໂຫຼດ **ultimate-multisite-site-exporter** ໄປທີ່ໄດເຣັກທໍຣີ **/wp-content/plugins/**
2. ເປີດໃຊ້ງານ plugin ຜ່ານເມນູ **Plugins** ໃນ WordPress
3. ປະຕິບັດຕາມ Wizard ແບບຂັ້ນຕອນເພື່ອຕັ້ງຄ່າ plugin

## ຄູ່ມືການໃຊ້ງານ {#usage-guide}

ດ້ວຍ Ultimate Multisite ທ່ານສາມາດສົ່ງອອກເວັບໄຊຈາກເຄືອຂ່າຍຂອງທ່ານ ຫຼື ນຳເຂົ້າເວັບໄຊໄປຫາການຕິດຕັ້ງແບບຫຼາຍເວັບໄຊ ຫຼື ເວັບໄຊດຽວຂອງທ່ານ.

ເພື່ອເລີ່ມຂະບວນການ, ໄປທີ່ໜ້າສ່ວນເສີມຂອງພວກເຮົາ ແລະ ຕິດຕັ້ງສ່ວນເສີມ Ultimate Multisite: ຕົວສົ່ງອອກ ແລະ ຕົວນຳເຂົ້າເວັບໄຊ.![ໜ້າສ່ວນເສີມ](/img/admin/addons-page.png)

![ໜ້າສ່ວນເສີມພ້ອມຕົວເລືອກຕິດຕັ້ງ](/img/admin/addons-page.png)

ໄປທີ່ໜ້າ Plugins ຂອງທ່ານ ແລະ ຄລິກເພື່ອ **ເປີດໃຊ້ງານໃນເຄືອຂ່າຍ** ສ່ວນເສີມທີ່ທ່ານຫາກໍຕິດຕັ້ງ.<!-- ບໍ່ມີພາບໜ້າຈໍ: ໜ້າ WP Plugins ທີ່ສະແດງປຸ່ມເປີດໃຊ້ງານໃນເຄືອຂ່າຍສຳລັບສ່ວນເສີມຕົວສົ່ງອອກເວັບໄຊ -->

ຕົວເລືອກ **ສົ່ງອອກ ແລະ ນຳເຂົ້າ** ສາມາດພົບໄດ້ໃນເມນູ Ultimate Multisite ທາງດ້ານຊ້າຍ.

<!-- ບໍ່ມີພາບໜ້າຈໍ: ເມນູ Ultimate Multisite ທີ່ສະແດງຕົວເລືອກສົ່ງອອກ ແລະ ນຳເຂົ້າ -->

## ວິທີສົ່ງອອກເວັບໄຊ {#how-to-export-a-site}

ເພື່ອສົ່ງອອກເວັບໄຊຈາກເຄືອຂ່າຍຂອງທ່ານ, ໄປທີ່ **ສົ່ງອອກ ແລະ ນຳເຂົ້າ** ໃນເມນູ Ultimate Multisite ແລະ, ໃນແຖບ **ສົ່ງອອກເວັບໄຊ**, ຄລິກເພື່ອ **ສ້າງການສົ່ງອອກເວັບໄຊໃໝ່**.

<!-- ບໍ່ມີພາບໜ້າຈໍ: ແຖບສົ່ງອອກເວັບໄຊພ້ອມປຸ່ມສ້າງການສົ່ງອອກເວັບໄຊໃໝ່ -->

ຄົ້ນຫາເວັບໄຊທີ່ທ່ານຕ້ອງການສົ່ງອອກ ແລະ ເລືອກລາຍການທີ່ທ່ານຕ້ອງການລວມໃນໄຟລ໌ທີ່ສົ່ງອອກ. ທ່ານສາມາດລວມ theme, plugin, ແລະ uploads. ຖ້າເວັບໄຊຂອງທ່ານໃຫຍ່ເກີນໄປ, ທ່ານສາມາດເລືອກໃຫ້ຮັນຂະບວນການສົ່ງອອກໃນພື້ນຫຼັງ.<!-- ບໍ່ມີພາບໜ້າຈໍ: ຕົວເລືອກສົ່ງອອກເວັບໄຊພ້ອມການຄົ້ນຫາເວັບໄຊ, ກ່ອງໝາຍເພື່ອລວມ theme/plugin/uploads -->

ຄລິກປຸ່ມ **ສົ່ງອອກເວັບໄຊ**. ເວັບໄຊທີ່ສົ່ງອອກຂອງທ່ານຈະມີໃຫ້ໃນລາຍຊື່ **ການສົ່ງອອກທີ່ມີຢູ່**. ຄລິກປຸ່ມ **ດາວໂຫຼດ ZIP** ເພື່ອດາວໂຫຼດໄຟລ໌. ZIP ສົ່ງອອກຈະລວມ `index.php` ແບບເລີ່ມຕົ້ນເອງ ແລະ `readme.txt`, ດັ່ງນັ້ນຈຶ່ງສາມາດຕິດຕັ້ງໃນ host ໃໝ່ໄດ້ໂດຍບໍ່ຕ້ອງຕິດຕັ້ງ plugin ຕົວນຳເຂົ້າແຍກຕ່າງຫາກກ່ອນ.<!-- ບໍ່ມີພາບໜ້າຈໍ: ລາຍຊື່ການສົ່ງອອກທີ່ມີຢູ່ພ້ອມປຸ່ມດາວໂຫຼດ ZIP -->

## ວິທີສົ່ງອອກເຄືອຂ່າຍທັງໝົດ {#how-to-export-the-whole-network}

ເພື່ອສົ່ງອອກເວັບໄຊຍ່ອຍທັງໝົດຮ່ວມກັນ, ໄປທີ່ **ສົ່ງອອກ ແລະ ນຳເຂົ້າ** ໃນເມນູ Ultimate Multisite ແລະ ໃຊ້ຕົວເລືອກ **ສົ່ງອອກເຄືອຂ່າຍ** ໃນໜ້າຈໍສົ່ງອອກ. ຄັງເອກະສານທີ່ສ້າງຂຶ້ນມີເວັບໄຊຍ່ອຍທັງໝົດໃນ ZIP ດຽວ ສຳລັບການຍ້າຍ host, ການສ້າງ staging ໃໝ່, ຫຼື ສຳເນົາສຳຮອງລະດັບເຄືອຂ່າຍ.

ການສົ່ງອອກເຄືອຂ່າຍອາດມີຂະໜາດໃຫຍ່. ຮັນເມື່ອການເຂົ້າຊົມຕ່ຳ ແລະ ຢືນຢັນວ່າ server ມີພື້ນທີ່ຈັດເກັບພຽງພໍສຳລັບຄັງເອກະສານ ແລະ uploads, plugin, ແລະ theme ໃດໆທີ່ລວມຢູ່.

## ວິທີນຳເຂົ້າເວັບໄຊ {#how-to-import-a-site}

ເພື່ອນຳເຂົ້າເວັບໄຊ, ໄປທີ່ແຖບ **ນຳເຂົ້າເວັບໄຊ** ແລະ ອັບໂຫຼດໄຟລ໌ .zip ຂອງເວັບໄຊທີ່ທ່ານຕ້ອງການນຳເຂົ້າ. ທ່ານສາມາດປ່ຽນ URL ເປັນອັນໃໝ່ ຫຼື ເກັບ URL ເກົ່າໄວ້ໂດຍປ່ອຍຊ່ອງວ່າງໄວ້. ນອກຈາກນັ້ນ, ທ່ານສາມາດໝາຍກ່ອງເພື່ອລຶບໄຟລ໌ .zip ອອກຈາກການຕິດຕັ້ງຂອງທ່ານຫຼັງຈາກນຳເຂົ້າ. ຈາກນັ້ນຄລິກ **ເລີ່ມການນຳເຂົ້າ**.

<!-- ບໍ່ມີພາບໜ້າຈໍ: ແຖບນຳເຂົ້າເວັບໄຊພ້ອມຊ່ອງອັບໂຫຼດ zip ແລະ ປຸ່ມເລີ່ມການນຳເຂົ້າ -->

ຂະບວນການອາດໃຊ້ເວລາສັກໜ່ອຍ. ທ່ານຈະເຫັນການນຳເຂົ້າທີ່ລໍຖ້າຢູ່ໃນລາຍຊື່ດ້ານລຸ່ມຕົວນຳເຂົ້າ.

<!-- ບໍ່ມີພາບໜ້າຈໍ: ການນຳເຂົ້າທີ່ລໍຖ້າຢູ່ໃນລາຍຊື່ດ້ານລຸ່ມຕົວນຳເຂົ້າ -->

ທ່ານສາມາດຢຸດຂະບວນການໄດ້ໂດຍຄລິກ **ຍົກເລີກການນຳເຂົ້າ**.

<!-- ບໍ່ມີພາບໜ້າຈໍ: ປຸ່ມຍົກເລີກການນຳເຂົ້າເພື່ອຢຸດຂະບວນການນຳເຂົ້າ -->

ຫຼັງຈາກຂະບວນການສຳເລັດ, ເວັບໄຊຂອງທ່ານກໍພ້ອມໃຊ້ງານໃນເຄືອຂ່າຍຂອງທ່ານແລ້ວ.

## ວິທີນຳເຂົ້າເວັບໄຊໃນການຕິດຕັ້ງເວັບໄຊດຽວ {#how-to-import-a-site-on-a-single-install}

ເພື່ອນຳເຂົ້າເວັບໄຊໄປຫາການຕິດຕັ້ງເວັບໄຊດຽວຂອງທ່ານ ກ່ອນອື່ນໃຫ້ຕິດຕັ້ງ ແລະ ເປີດໃຊ້ງານຕົວນຳເຂົ້າໃນເວັບໄຊຂອງທ່ານ. ທ່ານສາມາດພົບລິ້ງເພື່ອດາວໂຫຼດມັນໄດ້ໃນແຖບຕົວສົ່ງອອກ.

<!-- ບໍ່ມີພາບໜ້າຈໍ: ແຖບຕົວສົ່ງອອກພ້ອມລິ້ງເພື່ອດາວໂຫຼດຕົວນຳເຂົ້າເວັບໄຊດຽວ -->

ທ່ານສາມາດພົບຕົວນຳເຂົ້າໄດ້ໃນເມນູ **ເຄື່ອງມື**, ທາງດ້ານຊ້າຍ, ໃນລາຍການຍ່ອຍ **ສົ່ງອອກ ແລະ ນຳເຂົ້າ**.

<!-- ບໍ່ມີພາບໜ້າຈໍ: ເມນູເຄື່ອງມືເວັບໄຊດຽວທີ່ສະແດງເມນູຍ່ອຍສົ່ງອອກ ແລະ ນຳເຂົ້າ -->

ອັບໂຫຼດໄຟລ໌ .zip ຂອງເວັບໄຊທີ່ທ່ານຕ້ອງການນຳເຂົ້າ. ທ່ານສາມາດປ່ຽນ URL ເປັນອັນໃໝ່ ຫຼື ເກັບ URL ເກົ່າໄວ້ໂດຍປ່ອຍຊ່ອງວ່າງໄວ້. ນອກຈາກນັ້ນ, ທ່ານສາມາດໝາຍກ່ອງເພື່ອລຶບໄຟລ໌ .zip ອອກຈາກການຕິດຕັ້ງຂອງທ່ານຫຼັງຈາກນຳເຂົ້າ. ຈາກນັ້ນຄລິກ **ເລີ່ມການນຳເຂົ້າ**.

ຂະບວນການອາດໃຊ້ເວລາສັກໜ່ອຍ. ທ່ານຈະເຫັນການນຳເຂົ້າທີ່ລໍຖ້າຢູ່ໃນລາຍຊື່ດ້ານລຸ່ມຕົວນຳເຂົ້າ.

<!-- ບໍ່ມີພາບໜ້າຈໍ: ການນຳເຂົ້າເວັບໄຊດຽວທີ່ລໍຖ້າຢູ່ໃນລາຍຊື່ -->

ທ່ານສາມາດຢຸດຂະບວນການໄດ້ໂດຍຄລິກ **ຍົກເລີກການນຳເຂົ້າ**.

<!-- ບໍ່ມີພາບໜ້າຈໍ: ປຸ່ມຍົກເລີກການນຳເຂົ້າເພື່ອຢຸດຂະບວນການນຳເຂົ້າ -->

ຫຼັງຈາກຂະບວນການສຳເລັດ, ເວັບໄຊຂອງທ່ານຈະຖືກແທນທີ່ໂດຍເວັບໄຊທີ່ທ່ານຫາກໍນຳເຂົ້າ.
