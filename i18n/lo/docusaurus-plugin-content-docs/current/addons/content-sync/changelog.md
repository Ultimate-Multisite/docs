# ບັນທຶກການປ່ຽນແປງ Content Sync {#content-sync-changelog}

### ເວີຊັນ 1.1.0 - ປ່ອຍໃຊ້ໃນວັນທີ 2025-12-31 {#version-110---released-on-2025-12-31}
* ໃໝ່: ຮອງຮັບການຊິງຂໍ້ມູນສຳລັບປະເພດໂພສທັງໝົດ (ໂພສ, ໜ້າ, ປະເພດໂພສແບບກຳນົດເອງ)
* ໃໝ່: ການກັ່ນຕອງປະເພດໂພສໃນຂັ້ນຕອນການເລືອກເນື້ອຫາ
* ໃໝ່: ຂັ້ນຕອນການເລືອກເວັບໄຊຕົ້ນທາງ - ຊິງຂໍ້ມູນຈາກເວັບໄຊໃດກໍໄດ້, ບໍ່ແມ່ນພຽງແຕ່ເວັບໄຊຫຼັກ
* ໃໝ່: ການເລືອກເວັບໄຊລູກລ່ວງໜ້າສຳລັບເວັບໄຊແມ່ແບບ
* ໃໝ່: ການດຶງສື່ຈາກ post_content ສຳລັບປະເພດໂພສທັງໝົດ
* ໃໝ່: ການຊິງຂໍ້ມູນ Taxonomy (ໝວດໝູ່, ແທັກ, taxonomy ແບບກຳນົດເອງ)
* ປ່ຽນແປງ: ປ່ຽນຊື່ຈາກ "Elementor Template Sync" ເປັນ "Content Sync"
* ປ່ຽນແປງ: ຂັ້ນຕອນ "Select Templates" ປ່ຽນຊື່ເປັນ "Select Data"
* ປັບປຸງ: ໂຄດສະເພາະ Elementor ຕອນນີ້ເຮັດວຽກສະເພາະກັບປະເພດໂພສ elementor_library ເທົ່ານັ້ນ

### ເວີຊັນ 1.0.0 - ປ່ອຍໃຊ້ໃນວັນທີ 2025-12-19 {#version-100---released-on-2025-12-19}
* ການປ່ອຍໃຊ້ຄັ້ງທຳອິດ
* ວິຊາດຫຼາຍຂັ້ນຕອນສຳລັບການເລືອກແມ່ແບບແລະເວັບໄຊ
* ການປະມວນຜົນເປັນຊຸດພ້ອມຄຳຕິຊົມຄວາມຄືບໜ້າແບບເວລາຈິງ
* ການຊິງສື່ຢ່າງຄົບຖ້ວນພ້ອມການແມັບ ID ໃໝ່ອັດຕະໂນມັດ
* ການຜະສານການຕັ້ງຄ່າກັບ Ultimate Multisite
* ຮອງຮັບການຍົກເວັ້ນເວັບໄຊຜ່ານການຕັ້ງຄ່າເຄືອຂ່າຍ
* ການລ້າງແຄຊ Elementor ຫຼັງຈາກຊິງຂໍ້ມູນ
* ການຮັກສາ taxonomy ຂອງແມ່ແບບ
