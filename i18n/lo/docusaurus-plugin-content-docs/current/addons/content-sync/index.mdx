# ການຊິງໂຄຣໄນເນື້ອຫາ {#content-sync}

Multisite Ultimate: ການຊິງໂຄຣໄນເນື້ອຫາ

ສ່ວນເສີມນີ້ສຳລັບ Multisite Ultimate ອະນຸຍາດໃຫ້ຜູ້ດູແລເຄືອຂ່າຍຊິງໂຄຣໄນເນື້ອຫາຈາກເວັບໄຊໃດກໍໄດ້ໄປຍັງເວັບໄຊອື່ນໆ ໃນເຄືອຂ່າຍ. ມັນຮອງຮັບປະເພດໂພສທັງໝົດ ລວມທັງໂພສ, ໜ້າ, ປະເພດໂພສກຳນົດເອງ, ແລະ Elementor templates. ສ່ວນເສີມນີ້ປະກອບມີອິນເຕີເຟດຕົວຊ່ວຍແບບຫຼາຍຂັ້ນຕອນສຳລັບການເລືອກເນື້ອຫາ ແລະເວັບໄຊເປົ້າໝາຍ, ພ້ອມການປະມວນຜົນເປັນຊຸດ ແລະຂໍ້ມູນຄວາມຄືບໜ້າແບບເວລາຈິງ.

## ຄຸນສົມບັດ {#features}

* ການເລືອກເນື້ອຫາ: ຕາຂ່າຍບັດແບບພາບພ້ອມຮູບຫຍໍ້ ແລະການກັ່ນຕອງປະເພດໂພສ
* ປະເພດໂພສທັງໝົດ: ຊິງໂຄຣໄນໂພສ, ໜ້າ, ສິນຄ້າ, Elementor templates, ແລະປະເພດໂພສກຳນົດເອງໃດໆ
* ການເລືອກເວັບໄຊ: ເລືອກເວັບໄຊທີ່ຈະຊິງໂຄຣໄນເນື້ອຫາໄປຫາ, ພ້ອມການຍົກເວັ້ນເວັບໄຊຕົ້ນທາງ ແລະການເລືອກເວັບໄຊລູກໄວ້ລ່ວງໜ້າ
* ການປະມວນຜົນເປັນຊຸດ: ຊິງໂຄຣໄນໄປຍັງ 10 ເວັບໄຊຕໍ່ຄັ້ງ ພ້ອມການອັບເດດຄວາມຄືບໜ້າ
* ການຊິງໂຄຣໄນສື່: ຄັດລອກ ແລະແມບໄຟລ໌ສື່ (ຮູບພາບ, ວິດີໂອ) ຈາກເນື້ອຫາໂພສໄປຍັງເວັບໄຊເປົ້າໝາຍໂດຍອັດຕະໂນມັດ
* ການຊິງໂຄຣໄນ Taxonomy: ຮັກສາໝວດໝູ່, ແທັກ, ແລະ taxonomy ກຳນົດເອງ
* ການຮອງຮັບ Elementor: ຮອງຮັບ Elementor templates ຢ່າງເຕັມຮູບແບບ ພ້ອມການສ້າງ CSS ໃໝ່ ແລະການແມບ ID ເທມເພລດໃໝ່
* ການແກ້ໄຂຂໍ້ຂັດແຍ່ງ: ຕົວເລືອກເພື່ອແທນທີ່ເນື້ອຫາທີ່ມີຢູ່ ຫຼືຂ້າມມັນ
* ການຈັດການແຄຊ: ລ້າງ Elementor cache ຢູ່ເວັບໄຊເປົ້າໝາຍຫຼັງຈາກຊິງໂຄຣໄນໂດຍອັດຕະໂນມັດ

## ຂໍ້ກຳນົດ {#requirements}

* WordPress 5.3 ຫຼືສູງກວ່າ
* PHP 7.4 ຫຼືສູງກວ່າ
* plugin Multisite Ultimate (ເປີດໃຊ້ງານຢູ່)
* plugin Elementor (ບໍ່ບັງຄັບ, ຈຳເປັນສຳລັບການຊິງໂຄຣໄນ Elementor template)

## ການຕິດຕັ້ງ {#installation}

1. ອັບໂຫຼດ 'ultimate-multisite-content-sync' ໄປຍັງໄດເຣັກທໍຣີ '/wp-content/plugins/'
2. ເປີດໃຊ້ງານ plugin ຜ່ານເມນູ 'ສ່ວນເສີມ' ໃນ WordPress Network Admin ຂອງທ່ານ
3. ກຳນົດຄ່າການຕັ້ງຄ່າໃນ Ultimate Multisite → ການຕັ້ງຄ່າ → ການຊິງໂຄຣໄນເນື້ອຫາ
4. ໄປທີ່ Ultimate Multisite → ການຊິງໂຄຣໄນເນື້ອຫາ ເພື່ອເລີ່ມຊິງໂຄຣໄນ
