# ຕິກເກັດສະໜັບສະໜູນ {#support-tickets}

**ຕິກເກັດສະໜັບສະໜູນ** ແມ່ນ addon ທີ່ມີພະລັງສຳລັບ Ultimate Multisite ທີ່ເພີ່ມລະບົບຕິກເກັດສະໜັບສະໜູນແບບຄົບຊຸດໃຫ້ກັບເຄືອຂ່າຍ WordPress ຂອງທ່ານ. ເໝາະສົມຢ່າງຍິ່ງສຳລັບແພລດຟອມ SaaS, ຜູ້ໃຫ້ບໍລິການ WaaS (ເວັບໄຊເປັນບໍລິການ), ແລະເຄືອຂ່າຍ multisite ທີ່ຕ້ອງການໃຫ້ການສະໜັບສະໜູນລູກຄ້າທັງໃນລະດັບເຄືອຂ່າຍ ແລະລະດັບເວັບໄຊແຕ່ລະແຫ່ງ.

## ເປັນຫຍັງຈຶ່ງເລືອກຕິກເກັດສະໜັບສະໜູນ? {#why-choose-support-tickets}

* **ການສະໜັບສະໜູນຫຼາຍລະດັບ**: ຜູ້ດູແລເຄືອຂ່າຍຈັດການຕິກເກັດທົ່ວເຄືອຂ່າຍ ໃນຂະນະທີ່ເຈົ້າຂອງເວັບໄຊຈັດການການສະໜັບສະໜູນລູກຄ້າຂອງຕົນເອງ
* **ຂັ້ນຕອນການເຮັດວຽກແບບມືອາຊີບ**: ການຈັດການວົງຈອນຊີວິດຂອງຕິກເກັດຢ່າງຄົບຖ້ວນ ພ້ອມລະດັບຄວາມສຳຄັນ, ສະຖານະ, ແລະການມອບໝາຍ
* **ປະສົບການລູກຄ້າ**: ອິນເຕີເຟດທີ່ສວຍງາມ ແລະຕອບສະໜອງດີ ທີ່ເຮັດໃຫ້ລູກຄ້າຂໍຄວາມຊ່ວຍເຫຼືອໄດ້ງ່າຍ
* **ປອດໄພ ແລະເຊື່ອຖືໄດ້**: ສ້າງຂຶ້ນດ້ວຍແນວປະຕິບັດຄວາມປອດໄພທີ່ດີທີ່ສຸດຂອງ WordPress ແລະປັບປຸງປະສິດທິພາບຖານຂໍ້ມູນ
* **ການຜະສານງານທີ່ລຽບລື່ນ**: ເຮັດວຽກໄດ້ຢ່າງສົມບູນກັບການຈັດການລູກຄ້າ ແລະເວັບໄຊຂອງ Ultimate Multisite

## ຄຸນສົມບັດຫຼັກ {#key-features}

**ການຈັດການຕິກເກັດຢ່າງຄົບຖ້ວນ**

* ລະບົບໝາຍເລກຕິກເກັດທີ່ບໍ່ຊ້ຳກັນເພື່ອໃຫ້ອ້າງອີງໄດ້ງ່າຍ
* ລະດັບຄວາມສຳຄັນທີ່ຕັ້ງຄ່າໄດ້ (ຕ່ຳ, ປົກກະຕິ, ສູງ, ວິກິດ)
* ຂັ້ນຕອນສະຖານະຢ່າງຄົບຖ້ວນ (ເປີດ, ກຳລັງດຳເນີນການ, ກຳລັງລໍຖ້າ, ແກ້ໄຂແລ້ວ, ປິດ)
* ການຈັດໝວດໝູ່ຕິກເກັດ (ທົ່ວໄປ, ເຕັກນິກ, ການຮຽກເກັບເງິນ, ຄຳຂໍຄຸນສົມບັດ, ລາຍງານຂໍ້ຜິດພາດ)
* ມອບໝາຍຕິກເກັດໃຫ້ຕົວແທນສະໜັບສະໜູນ ພ້ອມຕົວເລືອກການມອບໝາຍອັດຕະໂນມັດ

**ການສົນທະນາແບບເປັນສາຍກະທູ້**

* ການຈັດລຽງການສົນທະນາເຕັມຮູບແບບສຳລັບການຕອບກັບຕິກເກັດ
* ຮູບແບບການສະແດງທີ່ແຕກຕ່າງກັນສຳລັບຄຳຕອບຂອງພະນັກງານ ແລະລູກຄ້າ
* ໝາຍເຫດສ່ວນຕົວຂອງພະນັກງານທີ່ລູກຄ້າບໍ່ສາມາດເຫັນ
* ອິນເຕີເຟດແບບເວລາຈິງທີ່ຂັບເຄື່ອນດ້ວຍ AJAX
* ຕົວແກ້ໄຂຂໍ້ຄວາມທັນສະໄໝພ້ອມຕົວເລືອກການຈັດຮູບແບບ

**ໄຟລ໌ແນບ**

* ການອັບໂຫຼດໄຟລ໌ທີ່ປອດໄພ ພ້ອມການຮອງຮັບລາກແລ້ວວາງ
* ຂໍ້ຈຳກັດປະເພດໄຟລ໌ ແລະຂະໜາດທີ່ຕັ້ງຄ່າໄດ້
* ຄວາມປອດໄພຂັ້ນສູງພ້ອມການກວດຈັບໄຟລ໌ອັນຕະລາຍ
* ການດາວໂຫຼດທີ່ຖືກປົກປ້ອງພ້ອມການຄວບຄຸມການເຂົ້າເຖິງ

**ການແຈ້ງເຕືອນອັດສະລິຍະ**

* ການແຈ້ງເຕືອນອີເມວອັດຕະໂນມັດສຳລັບຕິກເກັດໃໝ່, ການຕອບກັບ, ແລະການປ່ຽນສະຖານະ
* ແມ່ແບບອີເມວທີ່ປັບແຕ່ງໄດ້
* ການແຈ້ງເຕືອນຕາມບົດບາດສຳລັບພະນັກງານ ແລະລູກຄ້າ
* ຕົວແປແມ່ແບບແບບໄດນາມິກສຳລັບຂໍ້ຄວາມສ່ວນຕົວ

**ອິນເຕີເຟດຜູ້ດູແລທີ່ມີພະລັງ**

* ລາຍການຕິກເກັດຂັ້ນສູງພ້ອມການຈັດຮຽງ ແລະການກັ່ນຕອງ
* ການແກ້ໄຂດ່ວນສຳລັບສະຖານະ, ຄວາມສຳຄັນ, ແລະການມອບໝາຍ
* ການດຳເນີນການແບບຈຳນວນຫຼາຍເພື່ອການຈັດການທີ່ມີປະສິດທິພາບ
* Dashboard ສະຖິຕິພ້ອມຕົວຊີ້ວັດປະສິດທິພາບ
* ແຜງການຕັ້ງຄ່າຢ່າງຄົບຖ້ວນ

**ພອດທັນລູກຄ້າທີ່ສວຍງາມ**

* ອິນເຕີເຟດສະອາດ ແລະຕອບສະໜອງດີໃນທຸກອຸປະກອນ
* ການອອກແບບໂດຍເລີ່ມຈາກມືຖືກ່ອນ
* ການຄົ້ນຫາ ແລະການກັ່ນຕອງຂັ້ນສູງ
* ຕົວແກ້ໄຂທັນສະໄໝພ້ອມແຖບເຄື່ອງມືການຈັດຮູບແບບ
* ແບບຟອມສົ່ງຕິກເກັດທີ່ໃຊ້ງ່າຍ

**ການຜະສານງານທີ່ຍືດຫຍຸ່ນ**

* ໃຊ້ shortcodes ຫຼືອົງປະກອບ UI ຂອງ Ultimate Multisite
* ສະແດງແບບຟອມຕິກເກັດ ແລະລາຍການໄດ້ທຸກບ່ອນໃນເວັບໄຊຂອງທ່ານ
* ເຮັດວຽກຢ່າງລຽບລື່ນກັບໂຄງສ້າງເຄືອຂ່າຍ multisite
* ຂະຫຍາຍໄດ້ດ້ວຍ hooks ແລະ filters

## ເໝາະສຳລັບ {#perfect-for}

* ແພລດຟອມ SaaS ທີ່ຕ້ອງການການສະໜັບສະໜູນລູກຄ້າ
* ຜູ້ໃຫ້ບໍລິການ WaaS ທີ່ຈັດການເວັບໄຊລູກຄ້າຫຼາຍແຫ່ງ
* ເວັບໄຊສະມາຊິກທີ່ມີການສະໜັບສະໜູນເປັນລຳດັບຊັ້ນ
* ຜູ້ດູແລເຄືອຂ່າຍທີ່ຈັດການຫຼາຍແບຣນ
* ເຈົ້າຂອງເວັບໄຊທີ່ໃຫ້ບໍລິການລູກຄ້າ

## ການຕິດຕັ້ງ {#installation}

1. ໃຫ້ແນ່ໃຈວ່າ Ultimate Multisite plugin ຖືກຕິດຕັ້ງ ແລະເປີດໃຊ້ງານໃນເຄືອຂ່າຍ WordPress ຂອງທ່ານແລ້ວ
2. ອັບໂຫຼດໄຟລ໌ addon ໄປຍັງໄດເຣັກທໍຣີ `/wp-content/plugins/` ຂອງທ່ານ
3. ເປີດໃຊ້ງານ plugin ໃນລະດັບເຄືອຂ່າຍຜ່ານເມນູ 'ປລັກອິນ' ໃນ WordPress
4. ໄປທີ່ ຜູ້ດູແລເຄືອຂ່າຍ > ຕິກເກັດສະໜັບສະໜູນ ເພື່ອກຳນົດຄ່າການຕັ້ງຄ່າຂອງທ່ານ

## ຄຳຖາມທີ່ພົບເລື້ອຍ {#frequently-asked-questions}

## ສິ່ງນີ້ຕ້ອງໃຊ້ Ultimate Multisite ບໍ? {#does-this-require-ultimate-multisite}

ແມ່ນແລ້ວ, addon ນີ້ຕ້ອງການໃຫ້ຕິດຕັ້ງ ແລະເປີດໃຊ້ງານ Ultimate Multisite. ມັນຜະສານງານຢ່າງເລິກຊຶ້ງກັບຄຸນສົມບັດການຈັດການລູກຄ້າ ແລະເວັບໄຊຂອງ Ultimate Multisite.

## ເຈົ້າຂອງເວັບໄຊສາມາດຈັດການການສະໜັບສະໜູນລູກຄ້າຂອງຕົນເອງໄດ້ບໍ? {#can-site-owners-handle-their-own-customer-support}

ໄດ້ແນ່ນອນ! ເຈົ້າຂອງເວັບໄຊສາມາດໃຊ້ລະບົບຕິກເກັດເພື່ອໃຫ້ການສະໜັບສະໜູນແກ່ລູກຄ້າຂອງຕົນເອງ, ໃນຂະນະທີ່ຜູ້ດູແລເຄືອຂ່າຍສາມາດຈັດການຕິກເກັດສະໜັບສະໜູນທົ່ວເຄືອຂ່າຍໄດ້.

## ລູກຄ້າສົ່ງຕິກເກັດໄດ້ແນວໃດ? {#how-do-customers-submit-tickets}

ທ່ານສາມາດເພີ່ມແບບຟອມສົ່ງຕິກເກັດໄດ້ທຸກບ່ອນໃນເວັບໄຊຂອງທ່ານໂດຍໃຊ້ shortcodes ຫຼືອົງປະກອບ UI ຂອງ Ultimate Multisite. ລູກຄ້າພຽງແຕ່ຕື່ມແບບຟອມ ແລະສົ່ງຄຳຂໍຂອງພວກເຂົາ.

## ຂ້ອຍສາມາດປັບແຕ່ງສະຖານະ ແລະລະດັບຄວາມສຳຄັນຂອງຕິກເກັດໄດ້ບໍ? {#can-i-customize-the-ticket-statuses-and-priorities}

ໄດ້! ທ່ານສາມາດປັບແຕ່ງລະດັບຄວາມສຳຄັນ, ສະຖານະ, ແລະປະເພດຕິກເກັດໄດ້ຢ່າງເຕັມທີ່ໃນການຕັ້ງຄ່າຜູ້ດູແລເຄືອຂ່າຍ ເພື່ອໃຫ້ກົງກັບຂັ້ນຕອນການສະໜັບສະໜູນຂອງທ່ານ.

## ມັນຮອງຮັບໄຟລ໌ແນບບໍ? {#does-it-support-file-attachments}

ແມ່ນແລ້ວ, ທັງລູກຄ້າ ແລະພະນັກງານສາມາດແນບໄຟລ໌ໃສ່ຕິກເກັດ ແລະຄຳຕອບໄດ້. ທ່ານມີການຄວບຄຸມຢ່າງເຕັມທີ່ຕໍ່ປະເພດໄຟລ໌ທີ່ອະນຸຍາດ ແລະຂີດຈຳກັດຂະໜາດເພື່ອຄວາມປອດໄພ.

## ການແຈ້ງເຕືອນຖືກຈັດການແນວໃດ? {#how-are-notifications-handled}

ລະບົບຈະສົ່ງການແຈ້ງເຕືອນອີເມວອັດຕະໂນມັດສຳລັບຕິກເກັດໃໝ່, ການຕອບກັບ, ແລະການປ່ຽນສະຖານະ. ທ່ານສາມາດປັບແຕ່ງແມ່ແບບອີເມວທັງໝົດໄດ້ໃນການຕັ້ງຄ່າ.

## ຂ້ອຍສາມາດມອບໝາຍຕິກເກັດໃຫ້ຕົວແທນສະໜັບສະໜູນສະເພາະຄົນໄດ້ບໍ? {#can-i-assign-tickets-to-specific-support-agents}

ໄດ້, ທ່ານສາມາດມອບໝາຍຕິກເກັດໃຫ້ຕົວແທນດ້ວຍຕົນເອງ ຫຼືເປີດໃຊ້ການມອບໝາຍອັດຕະໂນມັດເພື່ອແຈກຈ່າຍຕິກເກັດອັດຕະໂນມັດ.

## ມີໝາຍເຫດສ່ວນຕົວສຳລັບພະນັກງານບໍ? {#are-there-private-notes-for-staff}

ແມ່ນແລ້ວ, ສະມາຊິກພະນັກງານສາມາດເພີ່ມຄຳຕອບສ່ວນຕົວທີ່ເຫັນໄດ້ສະເພາະສະມາຊິກພະນັກງານຄົນອື່ນເທົ່ານັ້ນ, ບໍ່ແມ່ນລູກຄ້າ.

## ມັນເປັນມິດກັບມືຖືບໍ? {#is-it-mobile-friendly}

ໄດ້ແນ່ນອນ! ອິນເຕີເຟດທັງໝົດຖືກສ້າງດ້ວຍແນວທາງທີ່ເລີ່ມຈາກມືຖືກ່ອນ ແລະເຮັດວຽກໄດ້ຢ່າງສວຍງາມໃນທຸກອຸປະກອນ.

## ຂ້ອຍສາມາດສະແດງລາຍການຕິກເກັດ ແລະແບບຟອມໂດຍໃຊ້ shortcodes ໄດ້ບໍ? {#can-i-display-ticket-lists-and-forms-using-shortcodes}

ໄດ້! shortcodes ທີ່ມີໃຫ້:
* `[wu_support_tickets]` - ສະແດງລາຍການຕິກເກັດຂອງລູກຄ້າ
* `[wu_support_ticket_form]` - ສະແດງແບບຟອມສົ່ງຕິກເກັດໃໝ່
* `[wu_support_ticket_view ticket_id="123"]` - ສະແດງຕິກເກັດສະເພາະ

## ປະເພດໄຟລ໌ໃດສາມາດອັບໂຫຼດໄດ້? {#what-file-types-can-be-uploaded}

ທ່ານສາມາດກຳນົດຄ່າປະເພດໄຟລ໌ທີ່ອະນຸຍາດໃນການຕັ້ງຄ່າ. ລະບົບມີການສະແກນຄວາມປອດໄພຂັ້ນສູງລວມຢູ່ ເພື່ອປ້ອງກັນໄຟລ໌ອັນຕະລາຍ.

## ຂ້ອຍສາມາດສົ່ງອອກຂໍ້ມູນຕິກເກັດໄດ້ບໍ? {#can-i-export-ticket-data}

ໄດ້, addon ນີ້ມີຄວາມສາມາດສົ່ງອອກສຳລັບຂໍ້ມູນຕິກເກັດ ແລະສະຖິຕິ.
