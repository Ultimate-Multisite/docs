# ຕົວຈັດການປລັກອິນ ແລະ ທີມ

Multisite Ultimate: ຕົວຈັດການປລັກອິນ ແລະ ທີມ

ແກ້ໄຂ ຫຼື ເຊື່ອງຂໍ້ມູນເມຕາ (ຊື່, ຜູ້ຂຽນ, ຮູບຫຍໍ້ ແລະ ຄຳອະທິບາຍ) ຂອງປລັກອິນ ແລະ ທີມທີ່ຕິດຕັ້ງແລ້ວ. ສ້າງໝວດໝູ່ສຳລັບປລັກອິນ ແລະ ທີມເພື່ອໃຫ້ຜູ້ໃຊ້ຂອງທ່ານກັ່ນຕອງພວກມັນໄດ້ ແລະ ເພີ່ມໜ້າປລັກອິນແບບກຳນົດເອງທີ່ສວຍງາມໃສ່ແຜງຄວບຄຸມຂອງຜູ້ໃຊ້ຂອງທ່ານ.

ເພື່ອໃຊ້ປລັກອິນ, ທ່ານພຽງແຕ່ໄປທີ່ໜ້າປລັກອິນ ຫຼື ທີມຂອງທ່ານໃນຜູ້ບໍລິຫານເຄືອຂ່າຍ ແລ້ວຄລິກທີ່ "ແກ້ໄຂຂໍ້ມູນປລັກອິນ/ທີມ".

## ການຕິດຕັ້ງ

1. ອັບໂຫຼດ 'ultimate-multisite-plugin-and-theme-manager' ໄປທີ່ໄດເຣັກທໍຣີ '/wp-content/plugins/'
2. ເປີດໃຊ້ປລັກອິນຜ່ານເມນູ 'ປລັກອິນ' ໃນຜູ້ບໍລິຫານເຄືອຂ່າຍ WordPress ຂອງທ່ານ

## ຄູ່ມືການໃຊ້ງານ

Ultimate Multisite: ຕົວຈັດການປລັກອິນ ແລະ ທີມ ແມ່ນສ່ວນເສີມ Ultimate Multisite ແບບ **ພຣີມຽມ** ທີ່ມີພະລັງ ເຊິ່ງໃຫ້ທ່ານຄວບຄຸມໄດ້ເຕັມທີ່ຕໍ່ວິທີທີ່ຜູ້ໃຊ້ເຄືອຂ່າຍຂອງທ່ານເຫັນ ແລະ ໂຕ້ຕອບກັບຕົວເລືອກປລັກອິນ ແລະ ທີມໃນເຄືອຂ່າຍຂອງທ່ານ.

ສ່ວນເສີມນີ້ມີຈຸດປະສົງສາມຢ່າງ:

  1. ມັນໃຫ້ທ່ານປ່ຽນຂໍ້ມູນເມຕາຂອງທັງປລັກອິນ ແລະ ທີມໄດ້ຢ່າງສົມບູນ. ນັ້ນລວມເຖິງຊື່, ຄຳອະທິບາຍ, ຊື່ຜູ້ຂຽນ ແລະ ແມ່ນແຕ່ຮູບຫຍໍ້!

  2. ມັນໃຫ້ທ່ານຈັດໝວດໝູ່ປລັກອິນ ແລະ ທີມ, ເພື່ອໃຫ້ຜູ້ໃຊ້ຂອງທ່ານຈັດຮຽງພວກມັນໄດ້ຢ່າງງ່າຍດາຍ.

  3. ມັນໃຫ້ຕົວເລືອກແກ່ທ່ານໃນການແທນທີ່ໜ້າປລັກອິນເກົ່າໆ ໜ້າເບື່ອ ດ້ວຍໜ້າໃໝ່ທີ່ຄ້າຍກັບໜ້າທີມ ພ້ອມການຈັດຮຽງຕາມໝວດໝູ່.

ຫຼັງຈາກດາວໂຫຼດ ແລະ ຕິດຕັ້ງມັນແລ້ວ, ແທັບໃໝ່ - _ການຕັ້ງຄ່າສ່ວນເສີມ_ \- ຈະສະແດງຢູ່ໃນໜ້າການຕັ້ງຄ່າ Ultimate Multisite.

_ທີ່ນີ້ທ່ານຈະພົບຕົວເລືອກສ່ວນກາງທີ່ເພີ່ມໂດຍ Ultimate Multisite: ຕົວຈັດການປລັກອິນ ແລະ ທີມ_

ໃນໜ້າ _ປລັກອິນ_ ຂອງເຄືອຂ່າຍທ່ານ ຕອນນີ້ທ່ານຈະສາມາດແກ້ໄຂຂໍ້ມູນເມຕາຂອງປລັກອິນທັງໝົດໄດ້, ເຊັ່ນ ຊື່, ຜູ້ຂຽນ ແລະ ແມ່ນແຕ່ຮູບຫຍໍ້ຂອງປລັກອິນ.

_ທ່ານຍັງສາມາດເລືອກໄດ້ວ່າຈະສະແດງ ຫຼື ບໍ່ສະແດງຂໍ້ມູນປລັກອິນສະເພາະ_

ຕົວຈັດການປລັກອິນ ແລະ ທີມ ມີປະເພດການສະແດງສອງແບບ: ແບບປລັກອິນ ແລະ ແບບທີມ. ດັ່ງນັ້ນ, ໃນທີ່ສຸດ ຫຼັງຈາກປັບແຕ່ງຕາມທີ່ທ່ານຕ້ອງການແລ້ວ, ນີ້ຄືວິທີທີ່ລູກຄ້າຂອງທ່ານຈະເຫັນໜ້າ _ປລັກອິນ_ ຂອງພວກເຂົາ:

_ໜ້າປລັກອິນຂອງລູກຄ້າທີ່ສະແດງແບບປລັກອິນ_
