# Loco Translate {#loco-translate}

ສ່ວນເສີມການເຊື່ອມຕໍ່ Ultimate Multisite: Loco Translate ນຳຄວາມສາມາດການແປພາສາສະເພາະແຕ່ລະເວັບໄຊມາໃຫ້ເຄືອຂ່າຍ WordPress multisite ຂອງທ່ານ. ຖ້າທ່ານເຄີຍຕ້ອງການໃຫ້ເວັບໄຊຕ່າງໆໃນເຄືອຂ່າຍຂອງທ່ານມີຄຳແປທີ່ແຕກຕ່າງກັນສຳລັບ plugin ຫຼື theme ດຽວກັນ, ສ່ວນເສີມນີ້ແມ່ນຄຳຕອບທີ່ເໝາະສົມທີ່ສຸດ.

**ສ່ວນເສີມນີ້ເຮັດຫຍັງ**

ສ່ວນເສີມນີ້ເຊື່ອມຕໍ່ Ultimate Multisite ກັບ plugin Loco Translate ຍອດນິຍົມຢ່າງລົງຕົວ, ເຮັດໃຫ້ແຕ່ລະ subsite ໃນເຄືອຂ່າຍຂອງທ່ານສາມາດຮັກສາຄຳແປທີ່ປັບແຕ່ງເອງໄດ້. ບໍ່ວ່າທ່ານກຳລັງດຳເນີນແພລດຟອມຫຼາຍພູມິພາກ, ໃຫ້ບໍລິການຊຸມຊົນຫຼາຍພາສາທີ່ແຕກຕ່າງກັນ, ຫຼືພຽງແຕ່ຕ້ອງການຄວາມຍືດຫຍຸ່ນດ້ານການແປພາສາທົ່ວເຄືອຂ່າຍຂອງທ່ານ, ສ່ວນເສີມນີ້ເຮັດໃຫ້ມັນງ່າຍດາຍ.

**ເໝາະສຳລັບ:**

* ທຸລະກິດຫຼາຍພູມິພາກທີ່ມີຄວາມຕ້ອງການການປັບໃຫ້ເໝາະກັບທ້ອງຖິ່ນທີ່ແຕກຕ່າງກັນ
* ສະຖາບັນການສຶກສາທີ່ໃຫ້ບໍລິການຊຸມຊົນທີ່ຫຼາກຫຼາຍ
* ແພລດຟອມ SaaS ທີ່ສະໜອງການຮອງຮັບຫຼາຍພາສາ
* ເອເຈນຊີເວັບທີ່ຈັດການເວັບໄຊລູກຄ້າຫຼາຍແຫ່ງ
* ເຄືອຂ່າຍ multisite ໃດໆທີ່ຕ້ອງການຄວາມຍືດຫຍຸ່ນດ້ານການແປພາສາ

## ການຕິດຕັ້ງ {#installation}

1. ໃຫ້ແນ່ໃຈວ່າ Ultimate Multisite ຖືກຕິດຕັ້ງແລະເປີດໃຊ້ງານໃນລະດັບເຄືອຂ່າຍແລ້ວ
2. ຕິດຕັ້ງແລະເປີດໃຊ້ plugin Loco Translate
3. ອັບໂຫຼດສ່ວນເສີມນີ້ໄປຍັງໄດເຣັກທໍຣີ plugins ຂອງທ່ານ
4. ເປີດໃຊ້ສ່ວນເສີມ Loco Translate Integration ໃນລະດັບເຄືອຂ່າຍ
5. ເລີ່ມສ້າງຄຳແປສະເພາະແຕ່ລະເວັບໄຊຜ່ານ Loco Translate ໃນ subsite ໃດກໍໄດ້

ເທົ່ານັ້ນ! ການເຊື່ອມຕໍ່ເຮັດວຽກໂດຍອັດຕະໂນມັດໂດຍບໍ່ຕ້ອງມີການກຳນົດຄ່າເພີ່ມເຕີມ.

## ຄຳຖາມທີ່ຖາມເລື້ອຍໆ {#frequently-asked-questions}

## ຂ້ອຍຈຳເປັນຕ້ອງກຳນົດຄ່າຫຍັງຫຼັງຈາກຕິດຕັ້ງບໍ? {#do-i-need-to-configure-anything-after-installation}

ບໍ່! ການເຊື່ອມຕໍ່ເປັນແບບອັດຕະໂນມັດ. ເມື່ອທັງສ່ວນເສີມແລະ Loco Translate ເປີດໃຊ້ງານແລ້ວ, subsites ສາມາດເລີ່ມສ້າງຄຳແປທີ່ປັບແຕ່ງເອງໄດ້ທັນທີ.

## ຈະເກີດຫຍັງຂຶ້ນຖ້າ subsite ບໍ່ສ້າງຄຳແປທີ່ປັບແຕ່ງເອງ? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

ເວັບໄຊຈະໃຊ້ຄຳແປທົ່ວເຄືອຂ່າຍເປັນຕົວສຳຮອງໂດຍອັດຕະໂນມັດ. ບໍ່ມີຫຍັງເສຍຫາຍ, ແລະທຸກຢ່າງເຮັດວຽກຢ່າງລົງຕົວ.

## subsites ສາມາດເຂົ້າເຖິງຄຳແປທົ່ວເຄືອຂ່າຍໄດ້ບໍ? {#can-subsites-access-network-wide-translations}

ຜູ້ດູແລເວັບໄຊທົ່ວໄປຈະເຫັນແລະແກ້ໄຂພຽງຄຳແປຂອງເວັບໄຊຕົນເອງເທົ່ານັ້ນ. Super admins ສາມາດເຂົ້າເຖິງທັງຄຳແປທົ່ວເຄືອຂ່າຍແລະຄຳແປສະເພາະເວັບໄຊ ຂຶ້ນກັບວ່າພວກເຂົາກຳລັງເບິ່ງເວັບໄຊໃດ.

## ສິ່ງນີ້ໃຊ້ງານໄດ້ກັບທັງ plugins ແລະ themes ບໍ? {#does-this-work-with-both-plugins-and-themes}

ໄດ້! ຄຳແປສະເພາະເວັບໄຊໃຊ້ງານໄດ້ກັບທັງ WordPress plugins ແລະ themes.

## ສິ່ງນີ້ຈະເຮັດໃຫ້ເຄືອຂ່າຍ multisite ຂອງຂ້ອຍຊ້າລົງບໍ? {#will-this-slow-down-my-multisite-network}

ບໍ່. ສ່ວນເສີມໃຊ້ການໂຫຼດໄຟລ໌ທີ່ມີປະສິດທິພາບ ເຊິ່ງກວດຫາຄຳແປສະເພາະເວັບໄຊກໍຕໍ່ເມື່ອຈຳເປັນເທົ່ານັ້ນ. ບໍ່ມີຜົນກະທົບຕໍ່ປະສິດທິພາບທີ່ສັງເກດໄດ້.

## ຂ້ອຍສາມາດຄັດລອກຄຳແປຂອງເຄືອຂ່າຍໄປຍັງເວັບໄຊສະເພາະເປັນຈຸດເລີ່ມຕົ້ນໄດ້ບໍ? {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

ໄດ້! ໃນຂະນະທີ່ແກ້ໄຂຄຳແປໃນ Loco Translate, ທ່ານສາມາດເລີ່ມຈາກຄຳແປຂອງເຄືອຂ່າຍໄດ້ຢ່າງງ່າຍດາຍ ແລ້ວປັບແຕ່ງມັນສຳລັບແຕ່ລະເວັບໄຊ.

## ສິ່ງນີ້ໃຊ້ງານໄດ້ກັບ WooCommerce ແລະ plugins ອື່ນໆບໍ? {#does-this-work-with-woocommerce-and-other-plugins}

ແນ່ນອນ! plugin ຫຼື theme ໃດໆທີ່ໃຊ້ຟັງຊັນການແປພາສາມາດຕະຖານຂອງ WordPress ຈະໃຊ້ງານໄດ້ກັບຄຳແປສະເພາະເວັບໄຊ.
