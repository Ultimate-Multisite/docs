# ບັນທຶກການປ່ຽນແປງຂອງ Loco Translate

### 1.0.1 {#101}
* ປັບປຸງຄວາມເຂົ້າກັນໄດ້ກັບ Loco Translate ເວີຊັນຫຼ້າສຸດ
* ເພີ່ມປະສິດທິພາບການຄວບຄຸມການເຂົ້າເຖິງເພື່ອຄວາມປອດໄພທີ່ດີຂຶ້ນ
* ການປັບແຕ່ງປະສິດທິພາບ

### 1.0.0 {#100}
* ການປ່ອຍຄັ້ງທຳອິດ
* ໄດເຣັກທໍຣີການແປສະເພາະເວັບໄຊ
* ລະບົບລຳດັບຄວາມສຳຄັນຂອງການແປ
* ການນຳໃຊ້ການຄວບຄຸມການເຂົ້າເຖິງ
* ການຜະສານ UI ກັບ Loco Translate ຢ່າງລື່ນໄຫຼ
