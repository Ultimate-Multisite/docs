# Fluent Forms

add-on ນີ້ເຮັດໃຫ້ສາມາດເພີ່ມຂີດຈຳກັດໃຫ້ Fluent Forms ໃນ Multisite Ultimate ໄດ້ ໃນຜະລິດຕະພັນ, ສະມາຊິກພາບ ຫຼື ເວັບໄຊ.
Fluent Forms ໃຊ້ຕາຕະລາງແບບກຳນົດເອງແທນ custom post types ດັ່ງນັ້ນຕົວເລືອກການຈຳກັດປົກກະຕິຈຶ່ງບໍ່ເຮັດວຽກກັບ Fluent Forms.
