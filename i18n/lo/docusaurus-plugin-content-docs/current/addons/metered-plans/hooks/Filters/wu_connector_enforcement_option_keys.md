# ຕົວກອງ: wu_connector_enforcement_option_keys {#filter-wuconnectorenforcementoptionkeys}

ກອງລາຍຊື່ຄີຕົວເລືອກຂອງຜູ້ໃຫ້ບໍລິການ AI ທີ່ຖືກບັງຄັບຈາກເວັບໄຊຫຼັກ.

ເພີ່ມຄີຕົວເລືອກສຳລັບ plugin ຜູ້ໃຫ້ບໍລິການ AI ແບບກຳນົດເອງ ຫຼື ຂອງບຸກຄົນທີສາມ ເພື່ອໃຫ້ການຕັ້ງຄ່າຂອງພວກມັນຖືກສືບທອດຈາກເວັບໄຊຫຼັກໃນເວັບໄຊຍ່ອຍດ້ວຍ.
