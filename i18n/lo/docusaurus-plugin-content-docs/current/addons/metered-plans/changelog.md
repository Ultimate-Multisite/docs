# ບັນທຶກການປ່ຽນແປງແຜນຄິດຕາມການໃຊ້ງານ {#metered-plans-changelog}

ເວີຊັນ 1.1.0 - ປ່ອຍໃຫ້ໃຊ້ໃນວັນທີ 2026-05-05
- ໃໝ່: ການຄິດເງິນ AI token ສຳລັບເວັບຍ່ອຍຂອງ multisite — ຕິດຕາມແລະຄິດເງິນການໃຊ້ AI token ໃນທົ່ວເວັບຂອງລູກຄ້າ ດ້ວຍອັດຕາຕໍ່ token ທີ່ກຳນົດຄ່າໄດ້
- ໃໝ່: ການບັງຄັບໃຊ້ຕົວເຊື່ອມຕໍ່ຖືກສ້າງໃໝ່ດ້ວຍການຄົ້ນພົບຂີດຈຳກັດແບບໄດນາມິກ ແລະການຂຽນຜ່ານ, ເພື່ອຮັບປະກັນຄວາມຖືກຕ້ອງແບບເວລາຈິງໃນທົ່ວຕົວເຊື່ອມຕໍ່ທັງໝົດ
- ແກ້ໄຂ: ການອັບເກຣດຕາຕະລາງຖານຂໍ້ມູນຕອນນີ້ຈັດໃຫ້ສອດຄ່ອງຢ່າງຖືກຕ້ອງກັບຄຳນິຍາມ schema ຂອງ BerlinDB, ປ້ອງກັນການອັບເກຣດລົ້ມເຫຼວໃນການຕິດຕັ້ງໃໝ່
- ແກ້ໄຂ: callbacks ການອັບເກຣດຖານຂໍ້ມູນຖືກແປງເປັນຮູບແບບທີ່ຖືກຕ້ອງ, ແກ້ໄຂຄວາມລົ້ມເຫຼວໃນການອັບເກຣດແບບບໍ່ມີຂໍ້ຄວາມແຈ້ງ
- ແກ້ໄຂ: ຕອນນີ້ຍອມຮັບຄ່າເສດສ່ວນໃນຊ່ອງປ້ອນ AI Usage Overage Markup
- ແກ້ໄຂ: ແກ້ໄຂ fatal errors ແລະບັນຫາການເລີ່ມຕົ້ນຊ້ຳສອງຄັ້ງໃນຕອນໂຫຼດປລັກອິນ
- ປັບປຸງ: ເພີ່ມ check-env npm script ເພື່ອໃຫ້ສະພາບແວດລ້ອມຂອງນັກພັດທະນາກຳນົດຄ່າເອງໃນການແລ່ນຄັ້ງທຳອິດ
