# ບັນທຶກການປ່ຽນແປງ VAT

ເວີຊັນ 1.0.7 - ປ່ອຍໃຊ້ໃນ 2026-02-03

* ແກ້ໄຂ: ເລກ VAT ຂອງເຢຍລະມັນ (DE) ກວດສອບບໍ່ຜ່ານ ເນື່ອງຈາກຂໍ້ຜິດພາດ MS_MAX_CONCURRENT_REQ ຂອງບໍລິການ VIES SOAP. ໄດ້ປ່ຽນຈາກ SOAP API ແບບເກົ່າໄປໃຊ້ EU VIES REST API ທາງການ ເພື່ອການກວດສອບທີ່ເຊື່ອຖືໄດ້ຫຼາຍຂຶ້ນ.
* ແກ້ໄຂ: ຕອນນີ້ເລກ VAT ຖືກຍອມຮັບເມື່ອບໍລິການ VIES ບໍ່ພ້ອມໃຊ້ຊົ່ວຄາວ ແທນທີ່ຈະຖືກປະຕິເສດຢ່າງຜິດພາດ. ມີການສ້າງລາຍການບັນທຶກໄວ້ເພື່ອການກວດທານພາຍຫຼັງ.
* ແກ້ໄຂ: ຕອນນີ້ຟິວ VAT/Tax ID ສະແດງສຳລັບທຸກປະເທດ ບໍ່ແມ່ນສະເພາະສະມາຊິກ EU ເທົ່ານັ້ນ. Tax ID ທີ່ບໍ່ແມ່ນ EU (ເຊັ່ນ ເລກ Swiss CHE) ຖືກເກັບໄວ້ສຳລັບການສະແດງໃນໃບແຈ້ງໜີ້ ໂດຍບໍ່ມີການກວດສອບ VIES.
* ແກ້ໄຂ: ການຄົ້ນຫາຄຳນຳໜ້າ VAT ສຳລັບກຣີຊ (GR), ໂມນາໂກ (MC), ແລະ Isle of Man (IM) ໃຊ້ງານບໍ່ໄດ້ ເນື່ອງຈາກຕັກກະການຄົ້ນຫາໃນອາເຣບໍ່ຖືກຕ້ອງ.
* ປັບປຸງ: ລຶບ dependency ສຳລັບການພັດທະນາ wp-cli-bundle ທີ່ບໍ່ຈຳເປັນ ເຊິ່ງອາດເຮັດໃຫ້ເກີດຂໍ້ຜິດພາດຮ້າຍແຮງເມື່ອໃຊ້ຄູ່ກັບ addon Site Exporter.
* ປັບປຸງ: ເພີ່ມຊຸດທົດສອບ unit ແບບຄົບຖ້ວນ (53 ການທົດສອບ).

ເວີຊັນ 1.0.6 - ປ່ອຍໃຊ້ໃນ 2026-01-25
