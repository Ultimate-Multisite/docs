# VAT {#vat}

Multisite Ultimate: VAT ຂອງເອີຣົບ

ເພີ່ມຄຸນສົມບັດແລະເຄື່ອງມືເພີ່ມເຕີມໃຫ້ກັບການຈັດການພາສີເລີ່ມຕົ້ນຂອງ Multisite Ultimate ເພື່ອຮອງຮັບ VAT ຂອງເອີຣົບໄດ້ດີຂຶ້ນ.
ເພີ່ມອັດຕາ EU VAT ໄປຍັງໜ້າອັດຕາພາສີໂດຍອັດຕະໂນມັດຈາກ

## ການຕິດຕັ້ງ {#installation}

1. ອັບໂຫຼດ 'ultimate-multisite-vat' ໄປຍັງໄດເຣັກທໍຣີ '/wp-content/plugins/'
2. ເປີດໃຊ້ງານປລັກອິນຜ່ານເມນູ 'ປລັກອິນ' ໃນ WordPress
3. ໄປທີ່ Ultimate Multisite > ການຕັ້ງຄ່າ > ພາສີ ແລ້ວເປີດໃຊ້ EU VAT.
4. ໄປຄລິກທີ່ອັດຕາພາສີ ແລະອັບເດດອັດຕາ EU VAT ຢູ່ດ້ານລຸ່ມ ແລ້ວບັນທຶກອັດຕາພາສີ.
5. ສຳເລັດ!
