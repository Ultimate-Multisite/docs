# ຕົວແທນຝ່າຍສະໜັບສະໜູນ

ການຈັດການເຄືອຂ່າຍທັງໝົດດ້ວຍຕົນເອງບໍ່ແມ່ນວຽກທີ່ງ່າຍ. ເມື່ອເຄືອຂ່າຍຂອງທ່ານເລີ່ມເຕີບໂຕ, ທ່ານຈະເລີ່ມເຫັນຄວາມຕ້ອງການການສະໜັບສະໜູນທີ່ເພີ່ມຂຶ້ນ. ການນຳຄົນເຂົ້າມາຊ່ວຍຫຼຸດພາລະວຽກຂອງທ່ານຈະເປັນຄວາມຄິດທີ່ດີ. ຢ່າງໃດກໍຕາມ, ການໃຫ້ການເຂົ້າເຖິງ/ສິດອະນຸຍາດເຕັມຮູບແບບແມ່ນມີຄວາມສ່ຽງ ແລະອາດເຮັດໃຫ້ເຄືອຂ່າຍຂອງທ່ານຖືກຄຸກຄາມໄດ້. ນັ້ນຄືຈຸດທີ່ add-on ຕົວແທນຝ່າຍສະໜັບສະໜູນນີ້ເຂົ້າມາຊ່ວຍ.

add-on ນີ້ອະນຸຍາດໃຫ້ທ່ານສ້າງຜູ້ໃຊ້ WordPress ປະເພດໃໝ່ທີ່ເອີ້ນວ່າ ຕົວແທນຝ່າຍສະໜັບສະໜູນ, ແລະໃຫ້ທ່ານຄວບຄຸມສິດອະນຸຍາດທີ່ຜູ້ໃຊ້ນັ້ນມີໄດ້ຢ່າງເຕັມທີ່. ການໃຫ້ສິດອະນຸຍາດແມ່ນງ່າຍພຽງແຕ່ເປີດປຸ່ມສະຫຼັບ ແລະປິດມັນຫາກທ່ານຕັດສິນໃຈລຶບສິດນັ້ນອອກ.

## ຄຸນສົມບັດ {#features}

- ລາຍການແບບແຍກລາຍການຂອງສິດອະນຸຍາດ WordPress ແລະ Ultimate Multisite
- ເປີດໃຊ້ ແລະປິດໃຊ້ການເຂົ້າເຖິງໄດ້ຢ່າງງ່າຍດາຍ
- ສ້າງຈຳນວນບັນຊີຕົວແທນໄດ້ບໍ່ຈຳກັດ

## ການຕິດຕັ້ງ {#installation}

1. ອັບໂຫຼດ 'ultimate-multisite-support-agents' ໄປທີ່ໄດເຣັກທໍຣີ '/wp-content/plugins/'
2. ເປີດໃຊ້ plugin ຜ່ານເມນູ 'ປລັກອິນ' ໃນ WordPress
3. ປະຕິບັດຕາມ Wizard ແບບຂັ້ນຕອນຕໍ່ຂັ້ນຕອນເພື່ອຕັ້ງຄ່າ plugin
