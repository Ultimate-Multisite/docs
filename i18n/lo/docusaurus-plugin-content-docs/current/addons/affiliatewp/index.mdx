# AffiliateWP {#affiliatewp}

Ultimate Multisite: ການເຊື່ອມຕໍ່ AffiliateWP

ໃຊ້ AffiliateWP ທີ່ມີພະລັງເພື່ອຂະຫຍາຍຖານລູກຄ້າຂອງເຄືອຂ່າຍ Ultimo ຂອງທ່ານ!

## ການຕິດຕັ້ງ {#installation}

1. ອັບໂຫຼດ 'ultimate-multisite-git ' ໄປທີ່ໄດເຣັກທໍຣີ '/wp-content/plugins/'
2. ເປີດໃຊ້ງານປັກອິນຜ່ານເມນູ 'ປັກອິນ' ໃນ WordPress
3. ປະຕິບັດຕາມຕົວຊ່ວຍແບບຂັ້ນຕອນຕໍ່ຂັ້ນເພື່ອຕັ້ງຄ່າປັກອິນ

## ຄູ່ມືການໃຊ້ງານ {#usage-guide}

[AffiliateWP](https://affiliatewp.com/) ໄດ້ກາຍເປັນວິທີແກ້ໄຂຫຼັກເມື່ອສ້າງໂປຣແກຣມພັນທະມິດສຳລັບທຸລະກິດອອນລາຍຂອງທ່ານ. Ultimate Multisite: AffiliateWP ແມ່ນສະພານທີ່ຈະເຮັດໃຫ້ທ່ານສາມາດສ້າງສິ່ງດຽວກັນສຳລັບເຄືອຂ່າຍພຣີມຽມຂອງທ່ານ.

## ເລີ່ມຕົ້ນ {#getting-started}

ກ່ອນອື່ນໝົດ, ທ່ານ **ຈຳເປັນຕ້ອງມີ** [**AffiliateWP**](https://affiliatewp.com/) **ເປີດໃຊ້ງານຢູ່**! ທ່ານຈະບໍ່ສາມາດໃຊ້ add-on ຂອງພວກເຮົາໄດ້ຖ້າທ່ານບໍ່ມີ **ປັກອິນທີ່ເຊື່ອມລວມກັບມັນ**. ໃນກໍລະນີທີ່ທ່ານບໍ່ໄດ້ວາງແຜນໃຫ້ລູກຄ້າຂອງທ່ານໃຊ້ມັນໃນເວັບໄຊຂອງພວກເຂົາດ້ວຍ, AffiliateWP ຄວນເປີດໃຊ້ງານສະເພາະໃນເວັບໄຊຫຼັກຂອງທ່ານເທົ່ານັ້ນ.

ນອກຈາກນີ້, ທ່ານຈະຕ້ອງມີທັງ **Ultimate Multisite** ແລະ **Ultimate Multisite: AffiliateWP** **ເປີດໃຊ້ງານ** **ທົ່ວເຄືອຂ່າຍ**.

## ການເພີ່ມ Ultimate Multisite ເປັນການເຊື່ອມລວມ AffiliateWP {#adding-ultimate-multisite-as-an-affiliatewp-integration}

ໃນເວັບໄຊຫຼັກຂອງທ່ານ, ໄປທີ່ການຕັ້ງຄ່າ Affiliates, ແລະໃນແທັບ Integrations, ໃຫ້ໝາຍເລືອກ Ultimate Multisite. ນີ້ຈະບອກໃຫ້ [AffiliateWP](https://affiliatewp.com/) ເລີ່ມຕິດຕາມການເຂົ້າຊົມ ແລະການປ່ຽນເປັນລູກຄ້າຈາກຂັ້ນຕອນການລົງທະບຽນຂອງ Ultimate Multisite.

<!-- ບໍ່ມີພາບໜ້າຈໍ: ແທັບການຕັ້ງຄ່າ AffiliateWP Integrations ພ້ອມກ່ອງໝາຍ Ultimate Multisite -->

_ໝາຍເລືອກກ່ອງ Ultimate Multisite ຢູ່ດ້ານລຸ່ມ ແລະບັນທຶກ_

ນັ້ນແມ່ນເກືອບທັງໝົດທີ່ຕ້ອງເຮັດ. ການເຂົ້າຊົມທີ່ມາຈາກລິ້ງອ້າງອີງຈະຖືກຕິດຕາມ ແລະເມື່ອຜູ້ໃຊ້ນັ້ນຊຳລະເງິນຄັ້ງທຳອິດໃນພາຍຫຼັງ, ການປ່ຽນເປັນລູກຄ້າຈະຖືກສ້າງ ແລະຄ່າຄອມມິຊັນຈະຖືກກັນໄວ້ໃຫ້ພັນທະມິດທີ່ຮັບຜິດຊອບຕໍ່ການປ່ຽນເປັນລູກຄ້ານັ້ນ.

ຢ່າງໃດກໍຕາມ, ຈະຕິດຕາມສະເພາະການຊຳລະເງິນຄັ້ງທຳອິດທີ່ໄດ້ຮັບຈາກຜູ້ໃຊ້ທີ່ສ້າງໃໝ່ເທົ່ານັ້ນ. ເພື່ອເປີດໃຊ້ການຊຳລະຄ່າອ້າງອີງແບບຊ້ຳ, ທ່ານຈະຕ້ອງມີ [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## ການເປີດໃຊ້ Recurring Referrals {#enabling-recurring-referrals}

ຫຼັງຈາກຕິດຕັ້ງ [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/), ໄປທີ່ແທັບ Affiliates Settings Recurring Referrals, ແລະເປີດໃຊ້ການໃຊ້ການອ້າງອີງແບບຊ້ຳ.

<!-- ບໍ່ມີພາບໜ້າຈໍ: ການຕັ້ງຄ່າ AffiliateWP Recurring Referrals ພ້ອມປຸ່ມສະຫຼັບ Enable Recurring Referrals -->

_ງ່າຍເທົ່າທີ່ເປັນໄປໄດ້: ເປີດໃຊ້ Recurring Referrals ແລະທ່ານກໍພ້ອມແລ້ວ!_

ອີກຄັ້ງ, ນັ້ນແມ່ນທັງໝົດທີ່ຕ້ອງເຮັດ!

ຕອນນີ້, ທຸກຄັ້ງທີ່ມີການຊຳລະເງິນໃໝ່ເຂົ້າມາໃນບັນຊີທີ່ຖືກອ້າງອີງ, ພັນທະມິດທີ່ຮັບຜິດຊອບຈະໄດ້ຮັບສ່ວນແບ່ງຂອງຕົນເອງໂດຍອັດຕະໂນມັດ!
