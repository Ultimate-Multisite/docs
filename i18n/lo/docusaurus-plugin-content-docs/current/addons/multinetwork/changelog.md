# ບັນທຶກການປ່ຽນແປງ Multi-Network

### 1.3.0 {#130}
* ໃໝ່: ຕົວສະແດງຕົວຢ່າງແມ່ແບບເຄືອຂ່າຍ — ເລືອກເບິ່ງ ແລະສະແດງຕົວຢ່າງແມ່ແບບເຄືອຂ່າຍດ້ວຍແຜງສົດກ່ອນຊື້.
* ໃໝ່: ຕົວເລືອກແມ່ແບບເຄືອຂ່າຍທີ່ການຊຳລະເງິນ — ລູກຄ້າສາມາດເລືອກແມ່ແບບເຄືອຂ່າຍໃນຂະບວນການລົງທະບຽນຊຳລະເງິນ.
* ໃໝ່: SSO ດ້ວຍລິ້ງວິເສດຂ້າມເຄືອຂ່າຍ — ລູກຄ້າສາມາດເຂົ້າໃຊ້ເຄືອຂ່າຍຍ່ອຍໄດ້ຢ່າງລາບລື່ນຜ່ານການຢືນຢັນຕົວຕົນດ້ວຍລິ້ງວິເສດ.
* ໃໝ່: ການໂຄລນສື່ເຄືອຂ່າຍ — ສື່ຂອງເວັບໄຊຖືກສຳເນົາຢ່າງຖືກຕ້ອງເມື່ອໂຄລນເຄືອຂ່າຍຈາກແມ່ແບບ.
* ແກ້ໄຂ: ການໂຄລນເຄືອຂ່າຍຕອນນີ້ເຮັດວຽກໄດ້ໂດຍບໍ່ຕ້ອງມີ addon ຫຼາຍຜູ້ເຊົ່າ; ສິດ super admin ຖືກມອບໃຫ້ອີກຄັ້ງຫຼັງຈາກໂຄລນ.
* ແກ້ໄຂ: ການສ້າງເຄືອຂ່າຍຕອນນີ້ເຂົ້າກັນໄດ້ກັບ WooCommerce ແລະທຸກຊ່ອງທາງຊຳລະເງິນ.
* ແກ້ໄຂ: ຕອນນີ້ມີການລວມເລກພອດໃນການສ້າງໂດເມນສຳຮອງ.
* ແກ້ໄຂ: CSS ຂອງຕົວເລືອກແມ່ແບບຖືກຍ້າຍໄປຍັງ stylesheet ທີ່ຖືກເຂົ້າຄິວເພື່ອການສະແດງຜົນທີ່ຖືກຕ້ອງໃນທຸກທີມ.
* ແກ້ໄຂ: ປ້ອງກັນ key `network_id` ທີ່ບໍ່ໄດ້ນິຍາມໃນຂອບເຂດຄຳສັ່ງຄົ້ນຫາເວັບໄຊ.
* ແກ້ໄຂ: ການກວດສອບສິດເຄືອຂ່າຍກ່ອນການສະຫຼັບບໍລິບົດຊ່ວຍປ້ອງກັນຂໍ້ຜິດພາດ property ຂອງຕາຕະລາງ `wpdb`.
