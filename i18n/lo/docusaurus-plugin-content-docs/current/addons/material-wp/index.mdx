# Material WP {#material-wp}

ສ່ວນຕິດຕໍ່ຜູ້ບໍລິຫານຂອງ WordPress ຍັງຄົງເປັນແບບເກົ່າມາດົນນານແລ້ວ. ແຕ່ເຈົ້າເຄີຍສົງໄສບໍ່ວ່າ ສ່ວນຕິດຕໍ່ຜູ້ບໍລິຫານໃນແບບ Material Design ຈະເບິ່ງເປັນແນວໃດ? 

Material WP ແມ່ນທີມຜູ້ບໍລິຫານທີ່ພັດທະນາຂຶ້ນເພື່ອໃຫ້ເຈົ້າໄດ້ຮູບລັກແລະຄວາມຮູ້ສຶກແບບນັ້ນ, ຊ່ວຍໃຫ້ເຈົ້າມອບ Dashboard ທີ່ປັບແຕ່ງໄດ້ຫຼາຍຂຶ້ນໃຫ້ກັບລູກຄ້າຂອງເຈົ້າ.

## ຄຸນສົມບັດ {#features}

- ຮອງຮັບ CSS ແບບກຳນົດເອງ
- ການຕັ້ງຄ່າສະໄຕລ໌ທີ່ງ່າຍ
- ການຈັດລຳດັບເມນູໃໝ່
- ນຳເຂົ້າ/ສົ່ງອອກໄດ້ງ່າຍ

## ເອກະສານ {#documentation}

**ການປັບແຕ່ງແຜງຜູ້ບໍລິຫານຂອງເຈົ້າດ້ວຍ Material WP**

## ການຕິດຕັ້ງ {#installation}

1. ອັບໂຫຼດ 'material-wp' ໄປທີ່ໄດເຣັກທໍຣີ '/wp-content/plugins/'
2. ເປີດໃຊ້ plugin ຜ່ານເມນູ 'Plugins' ໃນ WordPress

## ຄູ່ມືການໃຊ້ງານ {#usage-guide}

ສ່ວນຕິດຕໍ່ຜູ້ບໍລິຫານຂອງ WordPress ຍັງຄົງເປັນແບບເກົ່າມາດົນນານແລ້ວ. ແຕ່ເຈົ້າເຄີຍສົງໄສບໍ່ວ່າ ສ່ວນຕິດຕໍ່ຜູ້ບໍລິຫານໃນແບບ Material Design ຈະເບິ່ງເປັນແນວໃດ?

Material WP ແມ່ນທີມຜູ້ບໍລິຫານທີ່ພັດທະນາຂຶ້ນເພື່ອໃຫ້ເຈົ້າໄດ້ຮູບລັກແລະຄວາມຮູ້ສຶກແບບນັ້ນ, ຊ່ວຍໃຫ້ເຈົ້າມອບ Dashboard ທີ່ປັບແຕ່ງໄດ້ຫຼາຍຂຶ້ນໃຫ້ກັບລູກຄ້າຂອງເຈົ້າ. Material WP ຍັງເປັນສ່ວນໜຶ່ງຂອງແກນ plugin ພຣີມຽມຂອງ Ultimate Multisite ອີກດ້ວຍ.

ຫຼັງຈາກ **ຕິດຕັ້ງມັນ**, ໄປທີ່ໜ້າ _Plugins_ ຂອງເຈົ້າ ແລ້ວຄລິກເພື່ອເປີດໃຊ້ມັນໃນເຄືອຂ່າຍຂອງເຈົ້າ.

_ເປີດໃຊ້ມັນທົ່ວເຄືອຂ່າຍໃນໜ້າ Plugins ຂອງເຈົ້າ_

_**ໝາຍເຫດ:**_ _ຖ້າເຈົ້າພຽງແຕ່ເປີດໃຊ້ມັນ, ມັນຈະສະແດງໃຫ້ລູກຄ້າຂອງເຈົ້າເປັນຕົວເລືອກທີ່ສາມາດໝາຍເພື່ອເພີ່ມທີມໃຫ້ສະເພາະຜູ້ໃຊ້ນັ້ນເທົ່ານັ້ນ._

ຕອນນີ້ ສ່ວນຕິດຕໍ່ຜູ້ບໍລິຫານຂອງເຈົ້າຖືກອອກແບບໃໝ່ທັງໝົດແລ້ວ! ເຈົ້າຍັງມີເມນູໃໝ່ໃນຄໍລຳຊ້າຍທີ່ຊື່ວ່າ Material WP
