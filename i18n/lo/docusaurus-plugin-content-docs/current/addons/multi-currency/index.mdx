# Multi-Currency

Ultimate Multisite: Multi-Currency ເຮັດໃຫ້ເຄືອຂ່າຍຂອງທ່ານຮັບການຊຳລະເງິນໄດ້ໃນຫຼາຍສະກຸນເງິນ. ມັນແປງລາຄາໂດຍອັດຕະໂນມັດໂດຍໃຊ້ອັດຕາແລກປ່ຽນແບບສົດ ຫຼື ກຳນົດເອງ, ໃຫ້ທ່ານຕັ້ງລາຄາຕໍ່ສິນຄ້າແບບແນ່ນອນສຳລັບສະກຸນເງິນໃດກໍໄດ້, ແລະກວດຈັບສະກຸນເງິນທີ່ຜູ້ເຂົ້າຊົມແຕ່ລະຄົນມັກໃຊ້ຜ່ານການລະບຸຕຳແໜ່ງທາງພູມສາດ — ທັງໝົດຈັດການໄດ້ຈາກ Dashboard ຜູ້ດູແລດຽວ.

## ຄຸນສົມບັດຫຼັກ {#key-features}

- **ການແປງອັດຕາແລກປ່ຽນອັດຕະໂນມັດ** — ລາຄາຈະຖືກແປງຈາກສະກຸນເງິນພື້ນຖານຂອງທ່ານໂດຍໃຊ້ອັດຕາທີ່ດຶງມາຈາກຜູ້ໃຫ້ບໍລິການທີ່ທ່ານເລືອກ
- **ການກຳນົດລາຄາຕໍ່ສິນຄ້າແບບແທນຄ່າ** — ຕັ້ງລາຄາສະເພາະສະກຸນເງິນແບບແນ່ນອນໃນສິນຄ້າໃດກໍໄດ້ ເພື່ອຂ້າມການແປງອັດຕະໂນມັດ
- **ຊ່ອງເລືອກສະກຸນເງິນໃນການລົງທະບຽນ** — ໃຫ້ລູກຄ້າເລືອກສະກຸນເງິນທີ່ຕົນມັກໃນລະຫວ່າງ checkout/ລົງທະບຽນ
- **ຜູ້ໃຫ້ບໍລິການອັດຕາແລກປ່ຽນຫຼາຍລາຍ** — ECB, Frankfurter, Stripe, ຫຼືອັດຕາກຳນົດເອງທັງໝົດ
- **ການກວດຈັບສະກຸນເງິນຕາມຕຳແໜ່ງທາງພູມສາດ** — ເລືອກສະກຸນເງິນທີ່ກົງກັບປະເທດຂອງຜູ້ເຂົ້າຊົມໄວ້ລ່ວງໜ້າໂດຍອັດຕະໂນມັດ
- **Dashboard ຜູ້ດູແລ** — ຈັດການສະກຸນເງິນທີ່ເປີດໃຊ້, ອັດຕາແລກປ່ຽນ, ແລະການຕັ້ງຄ່າຜູ້ໃຫ້ບໍລິການຈາກບ່ອນດຽວ

## ຂໍ້ກຳນົດ {#requirements}

- Ultimate Multisite 2.0.0 ຫຼືສູງກວ່າ
- ການຕິດຕັ້ງ WordPress Multisite
- PHP 7.4 ຫຼືສູງກວ່າ

## ການຕິດຕັ້ງ {#installation}

1. ອັບໂຫຼດໄຟລ໌ plugin ໄປທີ່ `/wp-content/plugins/ultimate-multisite-multi-currency`
2. ເປີດໃຊ້ plugin ຜ່ານໜ້າຈໍ **Plugins** ໃນ WordPress (Network Activate)
3. ໄປທີ່ **Ultimate Multisite → Multi-Currency** ເພື່ອເປີດ Dashboard ຜູ້ດູແລ
4. ເພີ່ມສະກຸນເງິນທີ່ທ່ານຕ້ອງການຮັບ ແລະກຳນົດຄ່າຜູ້ໃຫ້ບໍລິການອັດຕາແລກປ່ຽນຂອງທ່ານ

## ການກຳນົດຄ່າ {#configuration}

### Dashboard ຜູ້ດູແລ {#admin-dashboard}

Dashboard ຜູ້ດູແລ Multi-Currency (**Ultimate Multisite → Multi-Currency**) ເປັນສູນກາງສຳລັບ:

- ເພີ່ມ, ເປີດໃຊ້, ຫຼືປິດໃຊ້ສະກຸນເງິນ
- ເລືອກຜູ້ໃຫ້ບໍລິການອັດຕາແລກປ່ຽນ ແລະຣີເຟຣຊອັດຕາຕາມຕ້ອງການ
- ກວດເບິ່ງອັດຕາປັດຈຸບັນສຳລັບສະກຸນເງິນທີ່ເປີດໃຊ້ທັງໝົດ
- ຕັ້ງອັດຕາກຳນົດເອງເມື່ອຜູ້ໃຫ້ບໍລິການອັດຕະໂນມັດບໍ່ເໝາະສົມ

### ຜູ້ໃຫ້ບໍລິການອັດຕາແລກປ່ຽນ {#exchange-rate-providers}

| ຜູ້ໃຫ້ບໍລິການ | ຄຳອະທິບາຍ |
|---|---|
| **ECB** | ທະນາຄານກາງເອີຣົບ — ອັດຕາປະຈຳວັນຟຣີສຳລັບເຄືອຂ່າຍທີ່ອີງໃສ່ EUR |
| **Frankfurter** | API ສຳເນົາ ECB ແບບໂອເພນຊອດ — ຟຣີ, ບໍ່ຕ້ອງມີ API key |
| **Stripe** | ໃຊ້ອັດຕາ FX ແບບສົດຂອງ Stripe — ຕ້ອງມີການຜະສານລວມ Stripe ທີ່ເປີດໃຊ້ຢູ່ |
| **Manual** | ທ່ານປ້ອນອັດຕາໂດຍກົງ; ບໍ່ມີການເອີ້ນ API ພາຍນອກ |

ເລືອກຜູ້ໃຫ້ບໍລິການຂອງທ່ານພາຍໃຕ້ **Ultimate Multisite → Multi-Currency → Settings → Exchange Rate Provider**. ອັດຕາສາມາດຣີເຟຣຊດ້ວຍຕົນເອງໄດ້ທຸກເວລາ ຫຼືກຳນົດເວລາໃຫ້ອັບເດດອັດຕະໂນມັດ.

### ການກຳນົດລາຄາຕໍ່ສິນຄ້າແບບແທນຄ່າ {#per-product-pricing-overrides}

ເມື່ອການແປງອັດຕະໂນມັດບໍ່ແມ່ນຍຳພໍ, ທ່ານສາມາດຕັ້ງລາຄາແນ່ນອນສຳລັບສະກຸນເງິນສະເພາະໃນສິນຄ້າໃດກໍໄດ້:

1. ເປີດຕົວແກ້ໄຂສິນຄ້າ (**Ultimate Multisite → Products → Edit**)
2. ເລື່ອນໄປຫາພາກສ່ວນ **Currency Pricing**
3. ປ້ອນລາຄາແນ່ນອນສຳລັບແຕ່ລະສະກຸນເງິນທີ່ທ່ານຕ້ອງການແທນຄ່າ
4. ບັນທຶກສິນຄ້າ

ສິນຄ້າທີ່ບໍ່ມີການແທນຄ່າຈະໃຊ້ລາຄາທີ່ແປງມາຈາກສະກຸນເງິນພື້ນຖານຂອງທ່ານ.

### ຊ່ອງເລືອກສະກຸນເງິນໃນການລົງທະບຽນ {#currency-selector-signup-field}

ຊ່ອງເລືອກສະກຸນເງິນສາມາດເພີ່ມເຂົ້າໃນຂັ້ນຕອນລົງທະບຽນ/checkout ຂອງທ່ານ ເພື່ອໃຫ້ລູກຄ້າເລືອກສະກຸນເງິນທີ່ຕົນມັກໃນຂະນະລົງທະບຽນ:

1. ໄປທີ່ **Ultimate Multisite → Multi-Currency → Settings**
2. ເປີດໃຊ້ **Show currency selector on signup**
3. ທາງເລືອກ ຕັ້ງສະກຸນເງິນຄ່າເລີ່ມຕົ້ນ ຫຼືໃຫ້ການລະບຸຕຳແໜ່ງທາງພູມສາດກຳນົດຄ່າທີ່ເລືອກໄວ້ລ່ວງໜ້າ

ສະກຸນເງິນທີ່ເລືອກຈະຖືກເກັບໄວ້ກັບການສະໝັກໃຊ້ ແລະໃຊ້ສຳລັບການຮຽກເກັບເງິນທັງໝົດຕໍ່ໄປ.

### ການກວດຈັບສະກຸນເງິນຕາມຕຳແໜ່ງທາງພູມສາດ {#geolocation-based-currency-detection}

ເມື່ອເປີດໃຊ້ການກວດຈັບຕຳແໜ່ງທາງພູມສາດ, addon ຈະຈັບຄູ່ປະເທດຂອງຜູ້ເຂົ້າຊົມກັບສະກຸນເງິນ ແລະເລືອກມັນໄວ້ລ່ວງໜ້າໃນຊ່ອງເລືອກສະກຸນເງິນ. ເພື່ອເປີດໃຊ້:

1. ໄປທີ່ **Ultimate Multisite → Multi-Currency → Settings**
2. ເປີດໃຊ້ **Detect currency from visitor location**
3. ຈັບຄູ່ປະເທດກັບສະກຸນເງິນໃນຕາຕະລາງ **Geolocation Mapping**

ຖ້າບໍ່ມີການຈັບຄູ່ສຳລັບປະເທດຂອງຜູ້ເຂົ້າຊົມ, ສະກຸນເງິນພື້ນຖານຂອງເຄືອຂ່າຍຈະຖືກໃຊ້ເປັນຄ່າສຳຮອງ.

## ຄຳຖາມທີ່ພົບເລື້ອຍ {#frequently-asked-questions}

### ຈະເກີດຫຍັງຂຶ້ນຖ້າຜູ້ໃຫ້ບໍລິການອັດຕາແລກປ່ຽນບໍ່ພ້ອມໃຊ້? {#what-happens-if-an-exchange-rate-provider-is-unavailable}

addon ຈະກັບໄປໃຊ້ອັດຕາທີ່ດຶງມາສຳເລັດຄັ້ງຫຼ້າສຸດ. ຖ້າຍັງບໍ່ເຄີຍມີການດຶງອັດຕາມາເລີຍ, ມັນຈະກັບໄປໃຊ້ອັດຕາກຳນົດເອງທີ່ທ່ານໄດ້ກຳນົດຄ່າໄວ້. ແນະນຳໃຫ້ກຳນົດຄ່າອັດຕາກຳນົດເອງເປັນຕາຂ່າຍຄວາມປອດໄພ.

### ຂ້ອຍສາມາດໃຊ້ຜູ້ໃຫ້ບໍລິການຕ່າງກັນສຳລັບສະກຸນເງິນຕ່າງກັນໄດ້ບໍ? {#can-i-use-different-providers-for-different-currencies}

ບໍ່ — ຜູ້ໃຫ້ບໍລິການໜຶ່ງລາຍຈະເປີດໃຊ້ໃນເວລາດຽວສຳລັບສະກຸນເງິນທັງໝົດ. ໃຊ້ **Manual** ຖ້າທ່ານຕ້ອງການຄວບຄຸມອັດຕາແຕ່ລະອັດຕາຢ່າງເຕັມທີ່.

### ການລະບຸຕຳແໜ່ງທາງພູມສາດຕ້ອງມີ plugin ແຍກຕ່າງຫາກບໍ? {#does-geolocation-require-a-separate-plugin}

ບໍ່ຕ້ອງມີ plugin ລະບຸຕຳແໜ່ງທາງພູມສາດພາຍນອກ. addon ໃຊ້ການກວດຈັບ IP-to-country ໃນຕົວຂອງ WordPress. ເພື່ອຄວາມແມ່ນຍຳສູງຂຶ້ນ, ບໍລິການລະບຸຕຳແໜ່ງທາງພູມສາດແບບສະເພາະສາມາດຜະສານລວມໄດ້ຜ່ານ filter hooks ທີ່ຈັດໃຫ້.

### ການແທນຄ່າຕໍ່ສິນຄ້າຖືກນຳໃຊ້ກ່ອນ ຫຼື ຫຼັງພາສີ? {#are-per-product-overrides-applied-before-or-after-tax}

ການແທນຄ່າຕໍ່ສິນຄ້າຕັ້ງລາຄາກ່ອນພາສີໃນສະກຸນເງິນເປົ້າໝາຍ. ພາສີຈະຖືກຄຳນວນເພີ່ມເທິງລາຄາແທນຄ່າໂດຍໃຊ້ການກຳນົດຄ່າພາສີທີ່ມີຢູ່ຂອງທ່ານ.
