# ບັນທຶກການປ່ຽນແປງ Ultimate AI Connector ສຳລັບ Endpoint ທີ່ເຂົ້າກັນໄດ້

## ເວີຊັນ 2.0.0 — ປ່ອຍໃຊ້ເມື່ອ 2026-04-24

* ໃໝ່: ຮອງຮັບຫຼາຍຜູ້ໃຫ້ບໍລິການ — ກຳນົດຄ່າ AI endpoint ຫຼາຍອັນ ແລະ ນຳທາງຄຳຂໍດ້ວຍການສຳຮອງອັດຕະໂນມັດຂ້າມຜູ້ໃຫ້ບໍລິການ.
* ແກ້ໄຂ: ການຜະສານ Multi-provider SDK ພ້ອມ provider ID, URL ລົງທະບຽນ, ແລະ ການຈຳກັດຂອບເຂດ HTTP filter ຕໍ່ຜູ້ໃຫ້ບໍລິການທີ່ຖືກຕ້ອງ.
* ແກ້ໄຂ: ບັດຜູ້ໃຫ້ບໍລິການໃໝ່ຈະຂະຫຍາຍອັດຕະໂນມັດເມື່ອເພີ່ມ; ລ້າງ cache script ເມື່ອອັບເດດ plugin.
* ແກ້ໄຂ: namespace ຂອງຄລາສຜູ້ໃຫ້ບໍລິການແບບໄດນາມິກສຳລັບ eval() ແລະ ພຶດຕິກຳຂະຫຍາຍອັດຕະໂນມັດສຳລັບຜູ້ໃຫ້ບໍລິການທີ່ເພີ່ມໃໝ່.
* ແກ້ໄຂ: ໃຊ້ຄອມໂພເນັນ Card/CardBody/CardHeader/CardDivider ທີ່ສະຖຽນແລ້ວ (ບໍ່ແມ່ນທົດລອງອີກຕໍ່ໄປ) ເພື່ອຄວາມເຂົ້າກັນໄດ້ກັບ WordPress 6.9+.
* ແກ້ໄຂ: ແທນທີ່ DragHandle ທີ່ບໍ່ໄດ້ນິຍາມ ດ້ວຍໄອຄອນຈັບແບບ unicode ສຳລັບລາກເພື່ອຈັດລຳດັບຜູ້ໃຫ້ບໍລິການໃໝ່.
* ແກ້ໄຂ: ກຳຈັດຄຳຂໍ HTTP ທີ່ບລັອກ ເຊິ່ງຖືກເອີ້ນໃນທຸກຄັ້ງທີ່ໂຫຼດໜ້າ.
* ປັບປຸງ: GitHub Actions workflows ອັບເກຣດເປັນ Node.js 24.
