# ບັນທຶກການປ່ຽນແປງ

## 1.18.0 — ປ່ອຍໃຊ້ເມື່ອ 2026-06-29 {#1180--released-on-2026-06-29}

### ໃໝ່ {#new}

- **ເຄື່ອງມື Google Calendar** — ອ່ານປະຕິທິນແລະເຫດການທີ່ກຳນົດຄ່າໄວ້ ສຳລັບລະບົບອັດຕະໂນມັດທີ່ຮັບຮູ້ຕາຕະລາງ
- **ການຈັບຄູ່ຜູ້ຕິດຕໍ່ ແລະຕົວຊ່ວຍຜູ້ເຂົ້າຮ່ວມ** — ຈັບຄູ່ຜູ້ເຂົ້າຮ່ວມໃນປະຕິທິນກັບຜູ້ໃຊ້ແລະຜູ້ຕິດຕໍ່ຂອງເວັບໄຊ
- **ປະຕູອະນຸມັດໂດຍມະນຸດ ແລະບັນທຶກເຕືອນຄວາມຈຳ** — ຢຸດລະບົບອັດຕະໂນມັດໄວ້ເພື່ອການກວດທານ ແລະຫຼີກລ່ຽງການແຈ້ງເຕືອນຊ້ຳ
- **ຜູ້ໃຫ້ບໍລິການ TextBee SMS** — ສົ່ງການແຈ້ງເຕືອນຂໍ້ຄວາມຕາມທີ່ກຳນົດຄ່າໄວ້ຜ່ານ TextBee
- **ແພັກເກດຄູ່ຮ່ວມຂັ້ນສູງ** — ເພີ່ມເຄື່ອງມືລະບົບໄຟລ໌ສຳລັບນັກພັດທະນາທີ່ໄວ້ໃຈໄດ້, ຖານຂໍ້ມູນ, WP-CLI, ຕົວສົ່ງ REST, ຕົວສ້າງ plugin, ສະແນັບຊັອດ git, ການຈັດການຜູ້ໃຊ້, ແລະເຄື່ອງມືທົດສອບປະສິດທິພາບ ທີ່ແຈກຢາຍແຍກຕ່າງຫາກຈາກບິວ WordPress.org

### ປັບປຸງ {#improved}

- **ການຕັ້ງຄ່າບໍລິການ Superdav ທີ່ມີການຈັດການ** — ເພີ່ມ endpoints ຂອງບໍລິການທີ່ໂຮສໄວ້ ແລະການຈັດຫາການເຊື່ອມຕໍ່ອັດຕະໂນມັດສຳລັບເວັບໄຊທີ່ຮອງຮັບ
- **ການຈັດແພັກເກດການປ່ອຍໃຊ້** — ສ້າງ ZIP ຫຼັກແລະ Advanced ແຍກກັນ, ເຜີຍແຜ່ທັງສອງໃນ GitHub, ແລະສົ່ງສະເພາະແພັກເກດຫຼັກໄປຫາ WordPress.org

### ແກ້ໄຂ {#fixed}

- **ຄວາມນ່າເຊື່ອຖືຂອງຄຳຂໍ AI** — ປັບປຸງການເລືອກໂມເດວ, ເວລາໝົດອາຍຸຂອງຄຳຂໍ, ໂມເດວການຕັ້ງຄ່າປະລິຍາຍ, ການຈັດການຂໍ້ຄວາມໃຊ້ເຫດຜົນ, ແລະຄຳແນະນຳການລອງໃໝ່ສຳລັບການເອີ້ນເຄື່ອງມືທີ່ບໍ່ຖືກຕ້ອງ
- **ການເສີມຄວາມແຂງແຮງໃຫ້ປະຕິທິນ ແລະການເຕືອນຄວາມຈຳ** — ເສີມຄວາມແຂງແຮງໃຫ້ໂທເຄັນ Google Calendar ແລະການລົບຄວາມຊ້ຳຂອງການເຕືອນຄວາມຈຳ
- **ການເລີ່ມນຳໃຊ້ ແລະການດຳເນີນຕໍ່ຫຼັງການອະນຸມັດ** — ແກ້ໄຂການເປີດການເລີ່ມນຳໃຊ້ໃນ frontend ແລະການດຳເນີນຕໍ່ຂອງຄວາມສາມາດທີ່ຢືນຢັນແລ້ວ
- **ບັນຫາການທົບທວນການຈັດແພັກເກດ WordPress.org** — ແກ້ໄຂຄຳເຫັນຕອບກັບຈາກການທົບທວນການຈັດແພັກເກດສຳລັບການປ່ອຍໃຊ້ຫຼັກ

## 1.16.0 — ປ່ອຍໃຊ້ເມື່ອ 2026-05-20 {#1160--released-on-2026-05-20}

### ໃໝ່ {#new-1}

- **ຄວາມສາມາດສ້າງ Logo SVG** — Theme Builder ຕອນນີ້ສາມາດສ້າງແລະຝັງ SVG ໂລໂກ້ກຳນົດເອງພ້ອມການຊຳລະລ້າງທີ່ປອດໄພຕໍ່ namespace
- **ການອັບໂຫຼດຮູບໃນການສຳພາດຄົ້ນຫາ** — ການສຳພາດຄົ້ນຫາຂອງ Theme Builder ຕອນນີ້ມີຂັ້ນຕອນອັບໂຫຼດຮູບ ເພື່ອບໍລິບົດດ້ານການອອກແບບທີ່ລະອຽດຂຶ້ນ
- **ຄວາມສາມາດກວດສອບຄວາມຕ່າງຂອງຊຸດສີ** — ກວດຄູ່ສີໃຫ້ສອດຄ່ອງກັບ WCAG ກ່ອນນຳໃຊ້ກັບ theme
- **ເມນູສຳລັບທຸລະກິດຕ້ອນຮັບ** — Theme Builder ຕອນນີ້ສາມາດສ້າງໜ້າເມນູອາຫານແລະເຄື່ອງດື່ມແບບມີໂຄງສ້າງ ສຳລັບທຸລະກິດຕ້ອນຮັບ
- **ການສະແດງຕົວຢ່າງບົນເດັສທັອບ ແລະມືຖື** — ເບິ່ງຕົວຢ່າງດີໄຊນ໌ຂອງທ່ານບົນອຸປະກອນເດັສທັອບແລະມືຖື ໃນລະຫວ່າງການເລືອກທິດທາງການອອກແບບ
- **ພາຣາມິເຕີປ້າຍນຳທາງ** — ຄວາມສາມາດ Create Menu ຕອນນີ້ຮອງຮັບ `navigation_label` ທີ່ແຍກຕ່າງຫາກຈາກຊື່ໜ້າ
- **ຄວາມພ້ອມໃຊ້ຂອງເຄື່ອງມື Tier 1** — sd-ai-agent/site-scrape ຕອນນີ້ເປັນເຄື່ອງມື Tier 1 ທີ່ມີໃຫ້ໃຊ້ໂດຍປະລິຍາຍໃນ Theme Builder

### ແກ້ໄຂ {#fixed-1}

- **ແຄຊ AI Client** — ຕອນນີ້ຮອງຮັບດ້ວຍ transients ສຳລັບການຄົງຢູ່ຂ້າມຄຳຂໍ, ປ້ອງກັນການສູນເສຍຂໍ້ມູນໃນງານ agent ທີ່ໃຊ້ເວລາດົນ
- **ລິ້ງການກະທຳໃນແຖວ plugin** — ແກ້ໄຂແລະປ່ຽນຊື່ເພື່ອຄວາມຊັດເຈນ

## 1.10.0 — ປ່ອຍໃຊ້ເມື່ອ 2026-05-05 {#1100--released-on-2026-05-05}

### ໃໝ່ {#new-2}

- **ການຄົ້ນຫາອິນເຕີເນັດ Tavily** — ເພີ່ມ Tavily ເປັນຜູ້ໃຫ້ບໍລິການຄົ້ນຫາ ເພື່ອຜົນການຄົ້ນຫາອິນເຕີເນັດທີ່ລະອຽດຂຶ້ນ ຄຽງຄູ່ກັບ Brave Search
- **ທັກສະໃນຕົວທີ່ຮັບຮູ້ theme** — ຄູ່ມືທັກສະ Block Themes, Classic Themes, Kadence Blocks, ແລະ Kadence Theme ຕອນນີ້ຖືກສົ່ງມາພ້ອມກັບ plugin
- **ຄວາມສາມາດຟອມຕິດຕໍ່ຂອງຕົວສ້າງເວັບໄຊ** — ເພີ່ມຟອມຕິດຕໍ່ໃສ່ໜ້າໃດກໍໄດ້ໂດຍກົງຈາກອິນເຕີເຟດແຊັດ

### ປັບປຸງ {#improved-1}

- **ປັບໂຄງສ້າງໃໝ່ການເຊື່ອມຮວມ WooCommerce** — ຕອນນີ້ໃຊ້ WooCommerce APIs ແບບ native ເພື່ອຄວາມນ່າເຊື່ອຖືແລະຄວາມເຂົ້າກັນໄດ້ທີ່ດີຂຶ້ນ
- **ລາຍຊື່ຜູ້ໃຫ້ບໍລິການຣີເຟຣຊອັດຕະໂນມັດ** — ເມື່ອ plugin ໃດໜຶ່ງຖືກເປີດໃຊ້ ຫຼືປິດໃຊ້

### ແກ້ໄຂ {#fixed-2}

- **ຄວາມສາມາດ navigate-to** — ແກ້ໄຂວົງຈອນໂຫຼດໃໝ່ບໍ່ສິ້ນສຸດໃນບາງໜ້າ admin
- **ຄວາມສາມາດ list-posts** — ຕອນນີ້ແປງຊື່ໝວດໝູ່ແລະແທັກເປັນ slugs ໄດ້ຖືກຕ້ອງ
- **ຄຳສັ່ງ WP-CLI** — ກູ້ຄືນນາມແຝງ namespace ທີ່ຂາດຫາຍ ຫຼັງຈາກການປັບໂຄງສ້າງໃໝ່ກ່ອນໜ້ານີ້
- **ລະບົບອັດຕະໂນມັດເຫດການ** — ຈັດການໄດ້ຢ່າງນຸ່ມນວນກັບເວັບໄຊທີ່ຕາຕະລາງອັດຕະໂນມັດຍັງບໍ່ຖືກສ້າງ
- **ຄວາມສາມາດ memory-save** — ຕອນນີ້ໃຊ້ຄຳນຳໜ້າ namespace ທີ່ຖືກຕ້ອງໃນຕົວສ້າງຄຳແນະນຳລະບົບ
- **ຜົນລັບເຄື່ອງມືແບບ Scalar** — ຕອນນີ້ຖືກຫໍ່ຢ່າງຖືກຕ້ອງກ່ອນສົ່ງກັບໄປຫາ AI
- **ສະຖິຕິການໃຊ້ງານ** — ຕອນນີ້ຈັດການຮູບແບບຄີຄວາມສາມາດເກົ່າໄດ້ຖືກຕ້ອງ ເມື່ອອັບເກຣດຈາກເວີຊັນເກົ່າກວ່າ
