# ການສຳພາດຄົ້ນຫາ {#discovery-interview}

**ການສຳພາດຄົ້ນຫາ** ແມ່ນຂັ້ນຕອນທຳອິດໃນຂະບວນການອອກແບບຂອງ Theme Builder. ມັນຮວບຮວມຂໍ້ມູນກ່ຽວກັບເວັບໄຊ, ທຸລະກິດ, ແລະຄວາມມັກດ້ານການອອກແບບຂອງທ່ານ ເພື່ອສ້າງທີມທີ່ປັບໃຫ້ເຂົ້າກັບທ່ານໂດຍສະເພາະ.

## ພາບລວມ {#overview}

ການສຳພາດຄົ້ນຫາຈະນຳພາທ່ານຜ່ານຊຸດຄຳຖາມເພື່ອເຂົ້າໃຈແບຣນ, ກຸ່ມເປົ້າໝາຍ, ແລະເປົ້າໝາຍດ້ານການອອກແບບຂອງທ່ານ. ໃນເວີຊັນ 1.16.0, ການສຳພາດຕອນນີ້ລວມມີ **ຂັ້ນຕອນອັບໂຫຼດຮູບພາບ** ທີ່ອະນຸຍາດໃຫ້ທ່ານໃຫ້ແຮງບັນດານໃຈດ້ານພາບສຳລັບການອອກແບບຂອງທ່ານ.

## ຂັ້ນຕອນການສຳພາດ {#interview-steps}

### 1. ຂໍ້ມູນທຸລະກິດ {#1-business-information}

- ຊື່ເວັບໄຊ ແລະຄຳຂວັນ
- ປະເພດທຸລະກິດ ຫຼືອຸດສາຫະກຳ
- ເປົ້າໝາຍຫຼັກຂອງທຸລະກິດ
- ຄຳອະທິບາຍກຸ່ມເປົ້າໝາຍ

### 2. ຄວາມມັກດ້ານການອອກແບບ {#2-design-preferences}

- ໂຄງສີທີ່ມັກ
- ສະໄຕລ໌ການອອກແບບ (ທັນສະໄໝ, ຄລາສສິກ, ມິນິມອລ, ສົດໃສຂີ້ຫຼິ້ນ, ແລະອື່ນໆ)
- ເວັບໄຊ ຫຼືແບຣນທີ່ເປັນແຮງບັນດານໃຈ
- ອົງປະກອບການອອກແບບທີ່ຈຳເປັນຕ້ອງມີ

### 3. ອັບໂຫຼດຮູບພາບ (ໃໝ່ໃນ v1.16.0) {#3-photo-upload-new-in-v1160}

ອັບໂຫຼດຮູບພາບໜຶ່ງຮູບ ຫຼືຫຼາຍຮູບ ເພື່ອສົ່ງອິດທິພົນຕໍ່ທິດທາງການອອກແບບ:

#### ຮູບແບບທີ່ຮອງຮັບ {#accepted-formats}

| ຮູບແບບ | ນາມສະກຸນໄຟລ໌ | ລາຍລະອຽດ |
|--------|-----------|---------|
| **JPEG** | `.jpg`, `.jpeg` | ແນະນຳສຳລັບຮູບຖ່າຍ |
| **PNG** | `.png` | ຮອງຮັບຄວາມໂປ່ງໃສ |
| **WebP** | `.webp` | ຮູບແບບທັນສະໄໝ, ຂະໜາດໄຟລ໌ນ້ອຍກວ່າ |
| **GIF** | `.gif` | ຮູບນິ່ງເທົ່ານັ້ນ (ບໍ່ມີອະນິເມຊັນ) |

#### ຂໍ້ກຳນົດໄຟລ໌ {#file-requirements}

- **ຂະໜາດໄຟລ໌ສູງສຸດ:** 10 MB
- **ຂະໜາດຕ່ຳສຸດ:** 400 × 300 ພິກເຊວ
- **ຂະໜາດທີ່ແນະນຳ:** 1200 × 800 ພິກເຊວ ຫຼືໃຫຍ່ກວ່າ
- **ອັດຕາສ່ວນພາບ:** ອັດຕາສ່ວນໃດກໍໄດ້

#### ຮູບພາບສົ່ງອິດທິພົນຕໍ່ທິດທາງການອອກແບບແນວໃດ {#how-photos-influence-design-direction}

ຮູບພາບທີ່ທ່ານອັບໂຫຼດຊ່ວຍໃຫ້ Theme Builder ເຂົ້າໃຈ:

- **ຊຸດສີ** — ສີເດັ່ນຈະຖືກດຶງອອກມາ ແລະນຳໃຊ້ໃນທີມ
- **ສະໄຕລ໌ພາບ** — ທັນສະໄໝ, ແບບຊົນນະບົດ, ມິນິມອລ, ສີສັນສົດໃສ, ແລະອື່ນໆ
- **ອາລົມ ແລະບັນຍາກາດ** — ມືອາຊີບ, ສ້າງສັນ, ເປັນມິດ, ຫຼູຫຼາ, ແລະອື່ນໆ
- **ອົງປະກອບການອອກແບບ** — ລວດລາຍ, ຜິວສຳຜັດ, ແລະສະໄຕລ໌ການຈັດອົງປະກອບ
- **ແຮງບັນດານໃຈດ້ານຕົວອັກສອນ** — ສະໄຕລ໌ຟອນຕ໌ທີ່ເຂົ້າກັບທີມດ້ານພາບ

### 4. ໂຄງສ້າງເນື້ອຫາ {#4-content-structure}

- ໜ້າຫຼັກທີ່ຕ້ອງການ (ໜ້າທຳອິດ, ກ່ຽວກັບ, ບໍລິການ, ຕິດຕໍ່, ແລະອື່ນໆ)
- ຕ້ອງມີສ່ວນບລັອກ ຫຼືຂ່າວສານ
- ຕ້ອງມີຟັງຊັນອີຄອມເມີຊ
- ຄຸນສົມບັດພິເສດ ຫຼືການເຊື່ອມຕໍ່

### 5. ການທົບທວນ ແລະການຢືນຢັນ {#5-review-and-confirmation}

- ສະຫຼຸບຂໍ້ມູນທັງໝົດທີ່ໃຫ້ໄວ້
- ການຢືນຢັນກ່ອນທີ່ການສ້າງການອອກແບບຈະເລີ່ມຕົ້ນ

## ແນວປະຕິບັດທີ່ດີໃນການອັບໂຫຼດຮູບພາບ {#photo-upload-best-practices}

### ສິ່ງທີ່ຄວນອັບໂຫຼດ {#what-to-upload}

- **ແຮງບັນດານໃຈຂອງແບຣນ** — ໂລໂກ້, ຕົວຢ່າງສີ, ຫຼືຄູ່ມືແນວທາງແບຣນ
- **ເວັບໄຊຄູ່ແຂ່ງ** — ພາບໜ້າຈໍຂອງເວັບໄຊທີ່ທ່ານຊື່ນຊົມ
- **ກະດານອາລົມ** — ຊຸດຮູບພາບທີ່ແທນຄວາມງາມທີ່ທ່ານຕ້ອງການ
- **ຮູບພາບສິນຄ້າ** — ຖ້າທ່ານມີສິນຄ້າທີ່ຈະນຳສະແດງ
- **ຮູບພາບທີມງານ ຫຼືຫ້ອງການ** — ສຳລັບການສ້າງແບຣນສ່ວນຕົວ ຫຼືມືອາຊີບ
- **ຮູບພາບວິຖີຊີວິດ** — ທີ່ແທນກຸ່ມເປົ້າໝາຍຂອງທ່ານ

### ສິ່ງທີ່ຄວນຫຼີກລ້ຽງ {#what-to-avoid}

- **ຮູບພາບຄຸນນະພາບຕ່ຳ** — ຮູບພາບມົວ ຫຼືແຕກເປັນພິກເຊວ
- **ຮູບພາບທີ່ບໍ່ກ່ຽວຂ້ອງ** — ຮູບພາບທີ່ບໍ່ແທນແບຣນຂອງທ່ານ
- **ຮູບພາບທີ່ມີຂໍ້ຄວາມຫຼາຍ** — ພາບໜ້າຈໍທີ່ມີຂໍ້ຄວາມຫຼາຍ
- **ວັດຖຸທີ່ມີລິຂະສິດ** — ໃຫ້ແນ່ໃຈວ່າທ່ານມີສິດໃນການນຳໃຊ້ຮູບພາບ
- **GIF ແບບອະນິເມຊັນ** — ປະມວນຜົນຮູບນິ່ງເທົ່ານັ້ນ

### ເຄັດລັບເພື່ອຜົນລັບທີ່ດີທີ່ສຸດ {#tips-for-best-results}

1. **ອັບໂຫຼດ 2-3 ຮູບພາບ** — ຮູບພາບຫຼາຍຮູບໃຫ້ບໍລິບົດທີ່ດີກວ່າ
2. **ໃຊ້ຮູບພາບຄຸນນະພາບສູງ** — ຮູບພາບຊັດເຈນ ແລະມີແສງດີ ໃຫ້ຜົນດີທີ່ສຸດ
3. **ໃສ່ຄວາມຫຼາກຫຼາຍ** — ປະສົມສີ, ສະໄຕລ໌, ແລະຫົວຂໍ້ເນື້ອຫາ
4. **ໃຫ້ບໍລິບົດ** — ອະທິບາຍສັ້ນໆວ່າເປັນຫຍັງທ່ານເລືອກແຕ່ລະຮູບ
5. **ໃຫ້ລະບຸຊັດເຈນ** — ຖ້າທ່ານມັກອົງປະກອບໃດໜຶ່ງ, ໃຫ້ຊີ້ອອກມາ

## ຜົນລັບທິດທາງການອອກແບບ {#design-direction-output}

ຫຼັງຈາກການສຳພາດຄົ້ນຫາ, Theme Builder ຈະສ້າງ:

- **ຊຸດສີ** — ສີຫຼັກ, ສີຮອງ, ແລະສີເນັ້ນ
- **ຕົວອັກສອນ** — ຄອບຄົວຟອນຕ໌ສຳລັບຫົວຂໍ້ ແລະຂໍ້ຄວາມເນື້ອຫາ
- **ສະໄຕລ໌ເລຢ໌ເອົາດ໌** — ໄລຍະຫ່າງ, ການຈັດວາງແນວ, ແລະການຈັດຮຽງອົງປະກອບ
- **ອົງປະກອບພາບ** — ໄອຄອນ, ລວດລາຍ, ແລະອົງປະກອບຕົກແຕ່ງ
- **ກະດານອາລົມ** — ການສະແດງດ້ວຍພາບຂອງທິດທາງການອອກແບບ

## ຂັ້ນຕອນຕໍ່ໄປ {#next-steps}

ຫຼັງຈາກສຳເລັດການສຳພາດຄົ້ນຫາ:

1. ທົບທວນທິດທາງການອອກແບບທີ່ສ້າງຂຶ້ນ
2. ໃຫ້ຄຳຕິຊົມ ຫຼືຂໍການປັບແກ້
3. ດຳເນີນຕໍ່ໄປຫາຂັ້ນຕອນ [ທິດທາງການອອກແບບ](./design-direction.md)
4. Theme Builder ສ້າງທີມກຳນົດເອງຂອງທ່ານ
