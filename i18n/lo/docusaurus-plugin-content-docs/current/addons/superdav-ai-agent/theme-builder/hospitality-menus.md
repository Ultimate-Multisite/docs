# ເມນູສຳລັບທຸລະກິດບໍລິການອາຫານ ແລະ ທີ່ພັກ

ຟີເຈີ **ເມນູສຳລັບທຸລະກິດບໍລິການອາຫານ ແລະ ທີ່ພັກ** ຊ່ວຍໃຫ້ Theme Builder ສາມາດສ້າງ ແລະ ຝັງໜ້າເມນູອາຫານ ແລະ ເຄື່ອງດື່ມທີ່ມີໂຄງສ້າງໄດ້ໂດຍກົງໃສ່ WordPress ເວັບໄຊຂອງທ່ານ.

## ພາບລວມ

Theme Builder ຕອນນີ້ສາມາດສ້າງໜ້າເມນູທີ່ເປັນມືອາຊີບ ແລະ ມີໂຄງສ້າງ ສຳລັບທຸລະກິດບໍລິການອາຫານ ແລະ ທີ່ພັກ ລວມທັງຮ້ານອາຫານ, ຄາເຟ, ບາ, ແລະ ບໍລິການຈັດອາຫານ. ເມນູເຫຼົ່ານີ້ຖືກຜະສານເຂົ້າກັບການອອກແບບເວັບໄຊຂອງທ່ານຢ່າງຄົບຖ້ວນ ແລະ ສາມາດອັບເດດ ແລະ ຈັດການໄດ້ຢ່າງງ່າຍດາຍ.

## ປະເພດທຸລະກິດບໍລິການອາຫານ ແລະ ທີ່ພັກທີ່ຮອງຮັບ

- **ຮ້ານອາຫານ** — ເມນູອາຫານແບບບໍລິການເຕັມຮູບແບບ
- **ຄາເຟ** — ເມນູກາເຟ ແລະ ອາຫານເບົາໆ
- **ບາ ແລະ ເລາຈ໌** — ເມນູເຄື່ອງດື່ມ ແລະ ອາຫານຮອງທ້ອງ
- **ຮ້ານເບເກີຣີ** — ເມນູຂະໜົມອົບ ແລະ ເຂົ້າຈີ່
- **ບໍລິການຈັດອາຫານ** — ຕົວເລືອກເມນູສຳລັບງານອີເວັນ
- **ລົດຂາຍອາຫານ** — ເມນູບໍລິການອາຫານເຄື່ອນທີ່
- **ໂຮງເບຍ ແລະ ໂຮງເຫຼົ້າແວງ** — ເມນູເຄື່ອງດື່ມພ້ອມຄຳອະທິບາຍ

## ໂຄງສ້າງເມນູ

### ໝວດໝູ່ເມນູ

ເມນູຖືກຈັດລຽງເປັນໝວດໝູ່:

- **ອາຫານເປີດຄາບ** — ອາຫານເລີ່ມຕົ້ນ ແລະ ຈານນ້ອຍ
- **ອາຫານຈານຫຼັກ** — ຄອສຫຼັກ
- **ອາຫານເຄີຍງ** — ເຄື່ອງເຄີຍງ ແລະ ຜັກ
- **ຂອງຫວານ** — ຂອງຫວານແຊບໆ
- **ເຄື່ອງດື່ມ** — ເຄື່ອງດື່ມ (ມີແອລກໍຮໍ ແລະ ບໍ່ມີແອລກໍຮໍ)
- **ລາຍການພິເສດ** — ລາຍການພິເສດປະຈຳວັນ ຫຼື ຕາມລະດູການ

### ຮູບແບບລາຍການເມນູ

ແຕ່ລະລາຍການເມນູປະກອບມີ:

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### ຊ່ອງຂໍ້ມູນລາຍການ

| ຊ່ອງຂໍ້ມູນ | ປະເພດ | ຄຳອະທິບາຍ |
|-------|------|-------------|
| `name` | string | ຊື່ອາຫານ ຫຼື ເຄື່ອງດື່ມ |
| `description` | string | ຄຳອະທິບາຍລະອຽດຂອງລາຍການ |
| `price` | string | ລາຄາ (ຈັດຮູບແບບພ້ອມສະກຸນເງິນ) |
| `dietary_info` | array | ຄຸນລັກສະນະດ້ານອາຫານ (ວີແກນ, ບໍ່ມີກລູເຕນ, ແລະ ອື່ນໆ) |
| `allergens` | array | ສານກໍ່ພູມແພ້ທົ່ວໄປ (ໝາກຖົ່ວ, ສັດນ້ຳມີເປືອກ, ແລະ ອື່ນໆ) |
| `availability` | string | ເວລາທີ່ມີໃຫ້ບໍລິການ (ປະຈຳວັນ, ຕາມລະດູການ, ແລະ ອື່ນໆ) |

## ການສ້າງເມນູສຳລັບທຸລະກິດບໍລິການອາຫານ ແລະ ທີ່ພັກ

### ຂັ້ນຕອນ 1: ໃຫ້ຂໍ້ມູນເມນູ

ບອກ Theme Builder ກ່ຽວກັບເມນູຂອງທ່ານ:

```
ສ້າງເມນູຮ້ານອາຫານສຳລັບຮ້ານອາຫານອິຕາລຽນຂອງຂ້ອຍ ໂດຍມີອາຫານເປີດຄາບ,
ອາຫານເສັ້ນພາສຕາ, ອາຫານຈານຫຼັກ, ແລະ ຂອງຫວານ. ໃຫ້ມີລາຄາ ແລະ ຄຳອະທິບາຍນຳ.
```

### ຂັ້ນຕອນ 2: Theme Builder ສ້າງເມນູ

Theme Builder:

1. ສ້າງໜ້າເມນູທີ່ມີໂຄງສ້າງ
2. ອອກແບບໃຫ້ເຂົ້າກັບ theme ຂອງເວັບໄຊທ່ານ
3. ຈັດລຽງລາຍການເປັນໝວດໝູ່
4. ຈັດຮູບແບບລາຄາ ແລະ ຄຳອະທິບາຍ
5. ເພີ່ມຂໍ້ມູນດ້ານອາຫານ ແລະ ສານກໍ່ພູມແພ້

### ຂັ້ນຕອນ 3: ກວດທານ ແລະ ປັບແຕ່ງ

ທ່ານສາມາດ:

1. ແກ້ໄຂລາຍການເມນູ ແລະ ລາຄາ
2. ເພີ່ມ ຫຼື ລຶບໝວດໝູ່
3. ຈັດລຳດັບລາຍການໃໝ່ພາຍໃນໝວດໝູ່
4. ອັບເດດຄຳອະທິບາຍ ແລະ ຂໍ້ມູນດ້ານອາຫານ
5. ປັບຮູບແບບສະໄຕລ໌ ແລະ ເລເອົາຕ໌

## ຕົວເລືອກການສະແດງເມນູ

### ໜ້າເມນູເຕັມ

ໜ້າສະເພາະທີ່ສະແດງເມນູທັງໝົດຂອງທ່ານ:

- ຈັດລຽງຕາມໝວດໝູ່
- ຄົ້ນຫາ ແລະ ກັ່ນຕອງໄດ້
- ເລເອົາຕ໌ທີ່ເໝາະສຳລັບການພິມ
- ການອອກແບບທີ່ຕອບສະໜອງກັບມືຖື

### ວິດເຈັດເມນູ

ຝັງສ່ວນເມນູໃສ່ໜ້າອື່ນໆ:

- ລາຍການແນະນຳໃນໜ້າຫຼັກ
- ລາຍການພິເສດປະຈຳວັນໃນແຖບຂ້າງ
- ເມນູເຄື່ອງດື່ມໃນໜ້າບາ
- ການຈັດໂຊວ໌ຂອງຫວານໃນສ່ວນທ້າຍໜ້າ

### ເມນູ PDF

ສ້າງເມນູ PDF ທີ່ດາວໂຫຼດໄດ້:

- ການຈັດຮູບແບບແບບມືອາຊີບ
- ຄຸນນະພາບພ້ອມພິມ
- ມີຮູບພາບ ແລະ ຄຳອະທິບາຍ
- ແບ່ງປັນ ແລະ ສົ່ງອີເມວໄດ້ງ່າຍ

## ຂໍ້ມູນດ້ານອາຫານ ແລະ ສານກໍ່ພູມແພ້

### ຄຸນລັກສະນະດ້ານອາຫານ

ໝາຍລາຍການດ້ວຍຂໍ້ມູນດ້ານອາຫານ:

- **ວີແກນ** — ບໍ່ມີຜະລິດຕະພັນຈາກສັດ
- **ມັງສະວິລັດ** — ບໍ່ມີຊີ້ນ
- **ບໍ່ມີກລູເຕນ** — ປອດໄພສຳລັບໂລກແພ້ກລູເຕນ
- **ບໍ່ມີນົມ** — ບໍ່ມີຜະລິດຕະພັນນົມ
- **ບໍ່ມີໝາກຖົ່ວ** — ບໍ່ມີໝາກຖົ່ວຈາກຕົ້ນໄມ້ ຫຼື ຖົ່ວດິນ
- **ຄາໂບໄຮເດຣດຕ່ຳ** — ຫຼຸດຄາໂບໄຮເດຣດ
- **ໂປຣຕີນສູງ** — ອຸດົມດ້ວຍໂປຣຕີນ

### ຄຳເຕືອນສານກໍ່ພູມແພ້

ລວມສານກໍ່ພູມແພ້ທົ່ວໄປ:

- **ໝາກຖົ່ວ** — ໝາກຖົ່ວຈາກຕົ້ນໄມ້ ແລະ ຖົ່ວດິນ
- **ສັດນ້ຳມີເປືອກ** — ສັດນ້ຳພວກກຸ້ງປູ ແລະ ຫອຍ
- **ປາ** — ປາທຸກຊະນິດ
- **ນົມ** — ນົມ ແລະ ຜະລິດຕະພັນນົມ
- **ໄຂ່** — ໄຂ່ໄກ່
- **ຖົ່ວເຫຼືອງ** — ຜະລິດຕະພັນຖົ່ວເຫຼືອງ
- **ກລູເຕນ** — ເຂົ້າສາລີ ແລະ ທັນຍາພືດທີ່ມີກລູເຕນ
- **ງາ** — ເມັດງາ ແລະ ນ້ຳມັນງາ

## ການຈັດການເມນູ

### ການອັບເດດລາຄາ

ອັບເດດລາຄາເມນູໄດ້ຢ່າງງ່າຍດາຍ:

1. ໄປທີ່ໜ້າເມນູ
2. ຄລິກ "ແກ້ໄຂເມນູ"
3. ອັບເດດລາຄາສຳລັບລາຍການ
4. ບັນທຶກການປ່ຽນແປງ
5. ການປ່ຽນແປງຈະປາກົດທັນທີໃນເວັບໄຊຂອງທ່ານ

### ການເພີ່ມລາຍການຕາມລະດູການ

ສ້າງຮູບແບບເມນູຕາມລະດູການ:

1. ສ້າງເວີຊັນເມນູໃໝ່
2. ເພີ່ມລາຍການຕາມລະດູການ
3. ໝາຍລາຍການເປັນ "ຕາມລະດູການ"
4. ກຳນົດຕາຕະລາງວັນທີທີ່ມີໃຫ້ບໍລິການ
5. ສະແດງອັດຕະໂນມັດໃນຊ່ວງລະດູການ

### ການຈັດການລາຍການພິເສດ

ສະແດງລາຍການພິເສດປະຈຳວັນ ຫຼື ປະຈຳອາທິດ:

1. ສ້າງໝວດໝູ່ "ລາຍການພິເສດ"
2. ເພີ່ມລາຍການພ້ອມວັນທີທີ່ມີໃຫ້ບໍລິການ
3. ເນັ້ນລາຍການພິເສດໃນໜ້າຫຼັກ
4. ອັບເດດປະຈຳວັນ ຫຼື ປະຈຳອາທິດ
5. ເກັບລາຍການພິເສດເກົ່າເຂົ້າຄັງ

## ການຜະສານກັບ Theme Builder

ເມື່ອໃຊ້ Theme Builder ສຳລັບເວັບໄຊບໍລິການອາຫານ ແລະ ທີ່ພັກ:

1. **ການກວດພົບເມນູອັດຕະໂນມັດ** — ລະບຸວ່າທ່ານເປັນທຸລະກິດອາຫານ/ເຄື່ອງດື່ມຫຼືບໍ່
2. **ການສ້າງໜ້າເມນູ** — ສ້າງໜ້າເມນູແບບມືອາຊີບ
3. **ການຈັບຄູ່ການອອກແບບ** — ເມນູເຂົ້າກັບ theme ຂອງເວັບໄຊທ່ານ
4. **ການປັບໃຫ້ເໝາະກັບມືຖື** — ເມນູສະແດງຢ່າງສວຍງາມໃນໂທລະສັບ
5. **ການປັບໃຫ້ເໝາະກັບ SEO** — ເມນູເປັນມິດກັບເຄື່ອງມືຄົ້ນຫາ

## ແນວປະຕິບັດທີ່ດີທີ່ສຸດ

### ການອອກແບບເມນູ

- **ການຈັດລຽງທີ່ຊັດເຈນ** — ໂຄງສ້າງໝວດໝູ່ທີ່ເປັນເຫດຜົນ
- **ຄຳອະທິບາຍທີ່ອ່ານງ່າຍ** — ນ່າກິນ ແລະ ໃຫ້ຂໍ້ມູນ
- **ລາຄາທີ່ສະເໝີກັນ** — ສະກຸນເງິນ ແລະ ການຈັດຮູບແບບທີ່ຊັດເຈນ
- **ຮູບພາບມືອາຊີບ** — ຮູບອາຫານຄຸນນະພາບສູງ
- **ພື້ນທີ່ວ່າງ** — ຢ່າໃຫ້ໜ້າແອອັດເກີນໄປ

### ເນື້ອຫາ

- **ຄຳອະທິບາຍທີ່ຖືກຕ້ອງ** — ອະທິບາຍລາຍການໃຫ້ຖືກຕ້ອງ
- **ເນັ້ນລາຍການພິເສດ** — ເຮັດໃຫ້ລາຍການພິເສດໂດດເດັ່ນ
- **ລວມສານກໍ່ພູມແພ້** — ລະບຸສານກໍ່ພູມແພ້ສະເໝີ
- **ອັບເດດເປັນປະຈຳ** — ຮັກສາລາຄາ ແລະ ລາຍການໃຫ້ເປັນປັດຈຸບັນ
- **ໃຊ້ພາສາທີ່ຊວນຢາກກິນ** — ເຮັດໃຫ້ລາຍການຟັງແລ້ວແຊບ

### ການເຂົ້າເຖິງ

- **ຟອນຕ໌ທີ່ອ່ານງ່າຍ** — ໃຊ້ຮູບແບບຕົວອັກສອນທີ່ຊັດ ແລະ ອ່ານງ່າຍ
- **ຄອນທຣາສທີ່ພຽງພໍ** — ໃຫ້ແນ່ໃຈວ່າຂໍ້ຄວາມອ່ານໄດ້
- **ປ້າຍກຳກັບດ້ານອາຫານ** — ໝາຍຕົວເລືອກດ້ານອາຫານໃຫ້ຊັດເຈນ
- **ຄຳເຕືອນສານກໍ່ພູມແພ້** — ສະແດງສານກໍ່ພູມແພ້ໃຫ້ເຫັນເດັ່ນຊັດ
- **ເປັນມິດກັບມືຖື** — ທົດສອບໃນອຸປະກອນທຸກປະເພດ
