# ສ້າງໂລໂກ້ SVG {#generate-logo-svg}

ຄວາມສາມາດ **ສ້າງໂລໂກ້ SVG** ຊ່ວຍໃຫ້ Theme Builder ສ້າງ ແລະຝັງໂລໂກ້ SVG ແບບກຳນົດເອງໄດ້ໂດຍກົງໃນເວັບໄຊ WordPress ຂອງທ່ານ ພ້ອມການກວດລ້າງແບບ namespace-safe ອັດຕະໂນມັດ.

## ພາບລວມ {#overview}

ຄວາມສາມາດນີ້ສ້າງໂລໂກ້ກຣາຟິກເວັກເຕີທີ່ປັບຂະໜາດໄດ້ (SVG) ໂດຍອີງຕາມທິດທາງແບຣນດິ້ງ ແລະຄວາມມັກດ້ານການອອກແບບຂອງເວັບໄຊຂອງທ່ານ. SVG ທີ່ຖືກສ້າງຂຶ້ນຈະຖືກກວດລ້າງອັດຕະໂນມັດເພື່ອຮັບປະກັນວ່າປອດໄພສຳລັບການໃຊ້ໃນ WordPress ໂດຍຍັງຄົງຄວາມສົມບູນທາງພາບໄວ້.

## ພາຣາມິເຕີ {#parameters}

| ພາຣາມິເຕີ | ປະເພດ | ຈຳເປັນ | ຄຳອະທິບາຍ |
|-----------|------|----------|-------------|
| `site_name` | string | ແມ່ນ | ຊື່ຂອງເວັບໄຊທີ່ຈະສ້າງໂລໂກ້ໃຫ້ |
| `style` | string | ແມ່ນ | ສະໄຕລ໌ການອອກແບບ (ເຊັ່ນ "ທັນສະໄໝ", "ຄລາສສິກ", "ມິນິມອລ", "ສົດໃສມ່ວນຊື່ນ") |
| `colors` | array | ບໍ່ | ອາເຣຂອງລະຫັດສີ hex ເພື່ອໃຊ້ໃນໂລໂກ້ (ເຊັ່ນ `["#678233", "#ffffff"]`) |
| `width` | number | ບໍ່ | ຄວາມກວ້າງ SVG ເປັນພິກເຊວ (ຄ່າເລີ່ມຕົ້ນ: 200) |
| `height` | number | ບໍ່ | ຄວາມສູງ SVG ເປັນພິກເຊວ (ຄ່າເລີ່ມຕົ້ນ: 200) |
| `include_text` | boolean | ບໍ່ | ຈະລວມຊື່ເວັບໄຊເປັນຂໍ້ຄວາມໃນໂລໂກ້ຫຼືບໍ່ (ຄ່າເລີ່ມຕົ້ນ: true) |

## ຮູບແບບຜົນລັບ {#output-format}

ຄວາມສາມາດນີ້ສົ່ງຄືນສະຕຣິງ SVG ທີ່ມີໂຄງສ້າງດັ່ງນີ້:

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## ພຶດຕິກຳການກວດລ້າງ SVG {#svg-sanitisation-behaviour}

SVG ທີ່ຖືກສ້າງຂຶ້ນຈະຜ່ານການກວດລ້າງແບບ namespace-safe ອັດຕະໂນມັດເພື່ອ:

- **ລຶບແອັດທຣິບິວທີ່ບໍ່ປອດໄພ** — ຕັດຕົວຈັດການເຫດການ, ສະຄຣິບ ແລະແອັດທຣິບິວທີ່ອາດເປັນອັນຕະລາຍອອກ
- **ຮັກສາ namespace** — ຄົງໄວ້ namespace ຂອງ SVG (xmlns, xlink) ເພື່ອການສະແດງຜົນທີ່ຖືກຕ້ອງ
- **ກວດສອບໂຄງສ້າງ** — ຮັບປະກັນວ່າ SVG ສອດຄ່ອງກັບມາດຕະຖານ W3C
- **ເຂົ້າລະຫັດເອນທິຕີ** — ຫຼີກອັກຂະລະພິເສດໃນເນື້ອຫາຂໍ້ຄວາມຢ່າງຖືກຕ້ອງ
- **ລຶບການອ້າງອີງພາຍນອກ** — ຕັດສະໄຕລ໌ຊີດພາຍນອກ ແລະການອ້າງອີງຮູບພາບອອກ

ສິ່ງນີ້ຮັບປະກັນວ່າ SVG ປອດໄພທີ່ຈະຝັງໂດຍກົງໃນ WordPress ໂດຍບໍ່ຕ້ອງການການກວດລ້າງເພີ່ມເຕີມ.

## ຕົວຢ່າງການໃຊ້ງານ {#usage-example}

**ພຣອມພ໌:**
```
ສ້າງໂລໂກ້ແບບທັນສະໄໝສຳລັບສະຕາດອັບເທັກໂນໂລຊີຂອງຂ້ອຍທີ່ຊື່ "CloudSync" ໂດຍໃຊ້ສີຟ້າ ແລະສີຂາວ.
```

**ຜົນລັບ:**
ຄວາມສາມາດນີ້ສ້າງໂລໂກ້ SVG ທີ່:
- ລວມຊື່ເວັບໄຊ "CloudSync"
- ໃຊ້ໂຄງສີຟ້າ ແລະສີຂາວທີ່ລະບຸໄວ້
- ປະຕິບັດຕາມຫຼັກການອອກແບບແບບທັນສະໄໝ
- ຖືກກວດລ້າງອັດຕະໂນມັດ ແລະພ້ອມໃຊ້ງານ

## ການປະສານກັບ Theme Builder {#integration-with-theme-builder}

ເມື່ອໃຊ້ການເລືອກທິດທາງການອອກແບບຂອງ Theme Builder, ຄວາມສາມາດສ້າງໂລໂກ້ SVG ຈະ:

1. ວິເຄາະທິດທາງການອອກແບບ ແລະຈານສີຂອງທ່ານ
2. ສ້າງໂລໂກ້ SVG ແບບກຳນົດເອງທີ່ກົງກັບຄວາມມັກຂອງທ່ານ
3. ຝັງໂລໂກ້ເຂົ້າໃນສ່ວນຫົວ/ພື້ນທີ່ແບຣນດິ້ງຂອງເວັບໄຊຂອງທ່ານອັດຕະໂນມັດ
4. ຈັດເກັບ SVG ເປັນໂລໂກ້ແບບກຳນົດເອງໃນສື່ WordPress

## ແນວທາງປະຕິບັດທີ່ດີທີ່ສຸດ {#best-practices}

- **ໃຫ້ຄວາມມັກດ້ານສະໄຕລ໌ທີ່ຊັດເຈນ** — ອະທິບາຍສະໄຕລ໌ການອອກແບບທີ່ທ່ານຕ້ອງການ (ທັນສະໄໝ, ຄລາສສິກ, ສົດໃສມ່ວນຊື່ນ, ແລະອື່ນໆ)
- **ລະບຸສີ** — ລວມສີແບຣນດ໌ຂອງທ່ານເພື່ອຄວາມສອດຄ່ອງ
- **ທົດສອບການສະແດງຜົນ** — ກວດຢືນຢັນວ່າໂລໂກ້ສະແດງຢ່າງຖືກຕ້ອງໃນຂະໜາດໜ້າຈໍຕ່າງໆ
- **ປັບແຕ່ງເພີ່ມເຕີມ** — ໃຊ້ເຄື່ອງມືປັບແຕ່ງໂລໂກ້ຂອງ WordPress ເພື່ອປັບຂະໜາດ ແລະຕຳແໜ່ງ

## ຂໍ້ຈຳກັດ {#limitations}

- ໂລໂກ້ SVG ຖືກສ້າງເປັນກຣາຟິກນິ່ງ (ບໍ່ແມ່ນແອນິເມຊັນ)
- ໂລໂກ້ທີ່ຊັບຊ້ອນມີອົງປະກອບຫຼາຍອາດຕ້ອງການການປັບປຸງດ້ວຍມື
- ບໍ່ຮອງຮັບຟອນຕ໌ແບບກຳນົດເອງ; ຂໍ້ຄວາມໃຊ້ຟອນຕ໌ຂອງລະບົບ
- ຂະໜາດທີ່ໃຫຍ່ຫຼາຍ ຫຼືນ້ອຍຫຼາຍອາດສົ່ງຜົນຕໍ່ຄຸນນະພາບ
