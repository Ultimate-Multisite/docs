# Superdav AI Agent {#superdav-ai-agent}

Superdav AI Agent ແມ່ນຜູ້ຊ່ວຍ AI WordPress ຂັ້ນສູງ ທີ່ຂະຫຍາຍ Gratis AI Agent ດ້ວຍທັກສະທີ່ເຂົ້າໃຈທີມ, ຄວາມສາມາດຄົ້ນຫາອິນເຕີເນັດທີ່ດີຂຶ້ນ, ຄຸນສົມບັດຕົວສ້າງເວັບໄຊ, ແລະການຜະສານລະບົບອັດຕະໂນມັດແບບເລືອກໄດ້. ເວີຊັນ 1.18.0 ເພີ່ມເຄື່ອງມືອ່ານ Google Calendar, ການແມັບຜູ້ຕິດຕໍ່ແລະຜູ້ເຂົ້າຮ່ວມ, ປະຕູອະນຸມັດໂດຍມະນຸດ, ການກຳຈັດການເຕືອນຊ້ຳ, ການແຈ້ງເຕືອນ SMS ຜ່ານ TextBee, ການຈັດສັນບໍລິການ Superdav ແບບມີການຈັດການ, ແລະແພັກເກດຄູ່ຮ່ວມຂັ້ນສູງທີ່ແຈກຢາຍແຍກຕ່າງຫາກສຳລັບເວີກໂຟລວຂອງນັກພັດທະນາທີ່ໄວ້ໃຈໄດ້.

## ຄຸນສົມບັດຫຼັກ {#key-features}

- **ທັກສະທີ່ເຂົ້າໃຈທີມ** — ປັບຄຳແນະນຳແລະຄວາມສາມາດອັດຕະໂນມັດ ຕາມທີມ WordPress ທີ່ກຳລັງໃຊ້ ແລະປລັກອິນທີ່ຕິດຕັ້ງໄວ້ຂອງທ່ານ
- **ຜູ້ໃຫ້ບໍລິການຄົ້ນຫາອິນເຕີເນັດຫຼາຍລາຍ** — ເລືອກລະຫວ່າງ Brave Search ແລະ Tavily ເພື່ອຜົນການຄົ້ນຫາທີ່ສົມບູນກວ່າ ແລະຄວາມສາມາດວິໄຈຂັ້ນສູງ
- **ແບບຟອມຕິດຕໍ່ຂອງຕົວສ້າງເວັບໄຊ** — ສ້າງແບບຟອມຕິດຕໍ່ທີ່ໃຊ້ງານໄດ້ຄົບຖ້ວນໂດຍກົງຈາກອິນເຕີເຟດແຊັດ ໂດຍໃຊ້ປລັກອິນແບບຟອມທີ່ທ່ານຕິດຕັ້ງໄວ້
- **ຕົວສ້າງເວັບໄຊທີ່ປັບປຸງດີຂຶ້ນ** — ແຜນສ້າງເວັບໄຊຫຼາຍຂັ້ນຕອນທີ່ປັບປຸງແລ້ວ ພ້ອມການຄົ້ນພົບປລັກອິນແລະການກູ້ຄືນຈາກຂໍ້ຜິດພາດທີ່ດີກວ່າ
- **ທັກສະທີມໃນຕົວ** — ຄຳແນະນຳສະເພາະທາງສຳລັບ Block Themes, Classic Themes, Kadence Blocks, ແລະ Kadence Theme
- **ການຜະສານ WooCommerce ທີ່ປັບປຸງດີຂຶ້ນ** — ຈັດໂຄງສ້າງໃໝ່ໃຫ້ໃຊ້ WooCommerce APIs ແບບເຈົ້າຂອງເອງ ເພື່ອຄວາມເຊື່ອຖືໄດ້ແລະຄວາມເຂົ້າກັນໄດ້ທີ່ດີກວ່າ
- **ການກວດພົບຜູ້ໃຫ້ບໍລິການອັດຕະໂນມັດ** — ກວດພົບແລະຣີເຟຣຊປລັກອິນແລະທີມທີ່ມີຢູ່ອັດຕະໂນມັດ ເມື່ອພວກມັນຖືກເປີດໃຊ້ ຫຼືປິດໃຊ້
- **ການສ້າງໂລໂກ້ກຳນົດເອງ** — ສ້າງແລະຝັງໂລໂກ້ SVG ກຳນົດເອງ ພ້ອມການທຳຄວາມສະອາດແບບປອດໄພຕໍ່ namespace ອັດຕະໂນມັດ
- **ການກວດສອບຄວາມຕ່າງຂອງພາເລດ** — ກວດຄູ່ສີໃຫ້ສອດຄ່ອງກັບມາດຕະຖານການເຂົ້າເຖິງ WCAG ກ່ອນນຳໃຊ້ກັບທີມຂອງທ່ານ
- **ການຄົ້ນພົບທີ່ເສີມດ້ວຍຮູບພາບ** — ອັບໂຫຼດຮູບພາບລະຫວ່າງການສຳພາດຄົ້ນພົບຂອງ Theme Builder ເພື່ອສົ່ງຜົນຕໍ່ທິດທາງການອອກແບບ
- **ເມນູສຳລັບທຸລະກິດຕ້ອນຮັບ** — ສ້າງໜ້າເມນູອາຫານແລະເຄື່ອງດື່ມແບບມີໂຄງສ້າງ ສຳລັບຮ້ານອາຫານ, ຮ້ານກາເຟ, ແລະທຸລະກິດຕ້ອນຮັບ
- **ຕົວຢ່າງເບິ່ງລ່ວງໜ້າເທິງ desktop ແລະ mobile** — ເບິ່ງຕົວຢ່າງການອອກແບບຂອງທ່ານເທິງອຸປະກອນ desktop ແລະ mobile ລະຫວ່າງການເລືອກທິດທາງການອອກແບບ
- **ການດຶງຂໍ້ມູນເວັບໄຊ Tier 1** — sd-ai-agent/site-scrape ຕອນນີ້ມີໃຫ້ໃຊ້ໂດຍຄ່າເລີ່ມຕົ້ນ ສຳລັບວິເຄາະເວັບໄຊຄູ່ແຂ່ງ ແລະເວັບໄຊແຮງບັນດານໃຈ
- **ເຄື່ອງມືອ່ານ Google Calendar** — ອ່ານປະຕິທິນແລະເຫດການທີ່ກຳນົດຄ່າໄວ້ ເພື່ອໃຫ້ອັດຕະໂນມັດສາມາດໃຊ້ເຫດຜົນກ່ຽວກັບບໍລິບົດຕາຕະລາງທີ່ຈະມາເຖິງ
- **ການແມັບຜູ້ຕິດຕໍ່ແລະຜູ້ເຂົ້າຮ່ວມ** — ຈັບຄູ່ຜູ້ຮ່ວມໃນປະຕິທິນກັບຜູ້ໃຊ້ແລະຜູ້ຕິດຕໍ່ໃນເວັບໄຊທີ່ຮູ້ຈັກ ກ່ອນສ້າງວຽກຕິດຕາມ
- **ປະຕູອະນຸມັດໂດຍມະນຸດ** — ຢຸດອັດຕະໂນມັດທີ່ອ່ອນໄຫວໄວ້ຊົ່ວຄາວເພື່ອກວດທານ ກ່ອນສົ່ງຂໍ້ຄວາມ, ດຳເນີນວຽກຕໍ່, ຫຼືເຮັດການປ່ຽນແປງ
- **ບັນທຶກການເຕືອນແລະການກຳຈັດການຊ້ຳ** — ຕິດຕາມການເຕືອນທີ່ຖືກສົ່ງແລ້ວ ເພື່ອໃຫ້ການຮັນອັດຕະໂນມັດຊ້ຳບໍ່ແຈ້ງເຕືອນຜູ້ຮັບຄົນເດີມສອງເທື່ອ
- **ການແຈ້ງເຕືອນ SMS ຜ່ານ TextBee** — ສົ່ງການແຈ້ງເຕືອນຂໍ້ຄວາມທີ່ກຳນົດຄ່າໄວ້ຜ່ານ TextBee ເມື່ອຜູ້ໃຫ້ບໍລິການ SMS ຖືກເປີດໃຊ້
- **ການຈັດສັນບໍລິການ Superdav ແບບມີການຈັດການ** — ເຊື່ອມຕໍ່ເວັບໄຊທີ່ຮອງຮັບໄປຫາ endpoint Superdav ທີ່ໂຮສໄວ້ ພ້ອມການຕັ້ງຄ່າອັດຕະໂນມັດແທນການກຳນົດຄ່າ endpoint ດ້ວຍມື
- **ແພັກເກດຄູ່ຮ່ວມຂັ້ນສູງ** — ເຄື່ອງມືນັກພັດທະນາທີ່ໄວ້ໃຈໄດ້ ທີ່ແຈກຢາຍແຍກຕ່າງຫາກ ສຳລັບເວີກໂຟລວລະບົບໄຟລ໌, ຖານຂໍ້ມູນ, WP-CLI, REST dispatcher, ຕົວສ້າງປລັກອິນ, ພາບຖ່າຍ git, ການຈັດການຜູ້ໃຊ້, ແລະເບັນຊ໌ມາກ

## ຂໍ້ກຳນົດ {#requirements}

- WordPress 6.0 ຫຼືສູງກວ່າ
- PHP 8.1 ຫຼືສູງກວ່າ
- WordPress Multisite (ເປີດໃຊ້ລະດັບເຄືອຂ່າຍ) ຫຼືການຕິດຕັ້ງແບບເວັບໄຊດຽວ
- ປລັກອິນເຊື່ອມຕໍ່ຜູ້ໃຫ້ບໍລິການ AI ທີ່ເປີດໃຊ້ຢູ່ (Anthropic, OpenAI, ຫຼືທີ່ເຂົ້າກັນໄດ້)
- Gratis AI Agent v1.9.0 ຫຼືສູງກວ່າ (Superdav AI Agent ຂະຫຍາຍ Gratis AI Agent)

## ການຕິດຕັ້ງ {#installation}

1. ອັບໂຫຼດໄຟລ໌ປລັກອິນໄປທີ່ `/wp-content/plugins/superdav-ai-agent`
2. ເປີດໃຊ້ປລັກອິນຜ່ານໜ້າ **ປລັກອິນ** ໃນ WordPress
3. ໃຫ້ແນ່ໃຈວ່າ Gratis AI Agent ຖືກເປີດໃຊ້ດ້ວຍ
4. ໄປທີ່ **Gratis AI Agent → ການຕັ້ງຄ່າ** ແລະກຳນົດຄ່າຜູ້ໃຫ້ບໍລິການຄົ້ນຫາອິນເຕີເນັດ, ການເຊື່ອມຕໍ່ບໍລິການ Superdav ແບບມີການຈັດການ, Google Calendar, ຫຼືຂໍ້ມູນຢືນຢັນ TextBee ຕາມທີ່ຈຳເປັນ

ແພັກເກດ WordPress.org ມີຄຸນສົມບັດຫຼັກຂອງ Superdav AI Agent. ແພັກເກດຄູ່ຮ່ວມຂັ້ນສູງຖືກສ້າງແລະແຈກຢາຍແຍກຕ່າງຫາກ ສຳລັບສະພາບແວດລ້ອມທີ່ເຄື່ອງມືນັກພັດທະນາທີ່ໄວ້ໃຈໄດ້ຖືກຕິດຕັ້ງແລະອະນຸຍາດຢ່າງຊັດເຈນ.

## ເລີ່ມໃຊ້ຢ່າງໄວ {#quick-start}

ເມື່ອເປີດໃຊ້ແລ້ວ, Superdav AI Agent ຈະປັບປຸງການຕິດຕັ້ງ Gratis AI Agent ຂອງທ່ານໂດຍອັດຕະໂນມັດ ດ້ວຍທັກສະທີ່ເຂົ້າໃຈທີມ ແລະຄວາມສາມາດເພີ່ມເຕີມ. ເປີດແຜງ **Gratis AI Agent** ໃນແຖບດ້ານຂ້າງຜູ້ດູແລ WordPress ແລະເລີ່ມໃຊ້ຄຸນສົມບັດໃໝ່.

## ທັກສະທີ່ເຂົ້າໃຈທີມ {#theme-aware-skills}

| ຜູ້ໃຫ້ບໍລິການ | ຄຸນສົມບັດ | ການຕັ້ງຄ່າ |

| **Tavily** | ຜົນການຄົ້ນຫາທີ່ລະອຽດກວ່າ, ຄວາມສາມາດດ້ານການຄົ້ນຄວ້າຂັ້ນສູງ | [Tavily API](https://tavily.com/) |

ເບິ່ງ [ການຄົ້ນຫາອິນເຕີເນັດ](../../user-guide/configuration/internet-search) ສຳລັບຄຳແນະນຳການກຳນົດຄ່າ.

## ຂັ້ນຕອນວຽກຂອງປະຕິທິນ, ການອະນຸມັດ ແລະການແຈ້ງເຕືອນ {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 ຂະຫຍາຍຂັ້ນຕອນວຽກອັດຕະໂນມັດໃຫ້ເກີນກວ່າວຽກເນື້ອຫາ ແລະການສ້າງເວັບໄຊ:

- **ເຄື່ອງມື Google Calendar** ໃຫ້ agent ອ່ານປະຕິທິນທີ່ກຳນົດຄ່າໄວ້ ແລະລາຍລະອຽດເຫດການໄດ້ ເມື່ອມີຂໍ້ມູນຮັບຮອງປະຕິທິນ.
- **ຕົວຊ່ວຍລາຍຊື່ຕິດຕໍ່ ແລະຜູ້ເຂົ້າຮ່ວມ** ຈັບຄູ່ຜູ້ເຂົ້າຮ່ວມເຫດການກັບຜູ້ໃຊ້ WordPress ຫຼືລາຍຊື່ຕິດຕໍ່ທີ່ຮູ້ຈັກ ກ່ອນການກະກຽມການເຕືອນ ຫຼືວຽກຕິດຕາມ.
- **ປະຕູການອະນຸມັດໂດຍມະນຸດ** ອະນຸຍາດໃຫ້ການອັດຕະໂນມັດຢຸດຊົ່ວຄາວຈົນກວ່າຜູ້ໃຊ້ທີ່ໄດ້ຮັບອຳນາດຈະກວດທານການກະທຳທີ່ສະເໜີ.
- **ບັນທຶກການເຕືອນ** ປ້ອງກັນການເຕືອນປະຕິທິນ ຫຼືການແຈ້ງເຕືອນຊ້ຳ ເມື່ອການອັດຕະໂນມັດຖືກລອງໃໝ່ ຫຼືຖືກກຳນົດເວລາຊ້ຳໆ.
- **ການຮອງຮັບ TextBee SMS** ເປີດໃຫ້ໃຊ້ການແຈ້ງເຕືອນທາງຂໍ້ຄວາມ ເມື່ອກຳນົດຄ່າຜູ້ໃຫ້ບໍລິການ TextBee ແລ້ວ.

ໃຊ້ເຄື່ອງມືເຫຼົ່ານີ້ສຳລັບການຊ່ວຍເຫຼືອທີ່ຮັບຮູ້ຕາຕະລາງ, ການຕິດຕາມຜູ້ເຂົ້າຮ່ວມ ແລະຂັ້ນຕອນວຽກການເຕືອນທີ່ຍັງຕ້ອງການການກວດທານໂດຍມະນຸດກ່ອນທີ່ຈະສົ່ງຂໍ້ຄວາມທີ່ອ່ອນໄຫວ.

## ແບບຟອມຕິດຕໍ່ຂອງຕົວສ້າງເວັບໄຊ {#site-builder-contact-forms}

agent ຕົວສ້າງເວັບໄຊຕອນນີ້ສາມາດສ້າງແບບຟອມຕິດຕໍ່ໄດ້ໂດຍກົງຈາກອິນເຕີເຟດແຊັດ. ພຽງແຕ່ຂໍໃຫ້ຕົວສ້າງເວັບໄຊສ້າງແບບຟອມຕິດຕໍ່, ແລະມັນຈະກວດພົບ plugin ແບບຟອມທີ່ຕິດຕັ້ງໄວ້ຂອງທ່ານໂດຍອັດຕະໂນມັດ (Contact Form 7, WPForms, Fluent Forms, ຫຼື Gravity Forms) ແລະສ້າງແບບຟອມທີ່ພ້ອມໃຊ້ງານ.

## ຄຸນສົມບັດ Theme Builder {#theme-builder-features}

Theme Builder ໃນ Superdav AI Agent v1.16.0 ມີຄວາມສາມາດໃໝ່ທີ່ມີພະລັງ:

### ການສຳພາດຄົ້ນພົບພ້ອມການອັບໂຫຼດຮູບພາບ {#discovery-interview-with-photo-upload}

ການສຳພາດຄົ້ນພົບຂອງ Theme Builder ຕອນນີ້ມີຂັ້ນຕອນອັບໂຫຼດຮູບພາບ ທີ່ທ່ານສາມາດໃຫ້ແຮງບັນດານໃຈທາງສາຍຕາໄດ້. ອັບໂຫຼດຮູບພາບຂອງແບຣນຂອງທ່ານ, ເວັບໄຊຄູ່ແຂ່ງ ຫຼືແຮງບັນດານໃຈດ້ານການອອກແບບ ເພື່ອຊ່ວຍໃຫ້ Theme Builder ເຂົ້າໃຈທິດທາງການອອກແບບຂອງທ່ານ.

ເບິ່ງ [ການສຳພາດຄົ້ນພົບ](./theme-builder/discovery-interview.md) ສຳລັບຄຳແນະນຳແບບລະອຽດ.

### ທິດທາງການອອກແບບພ້ອມຕົວຢ່າງເບິ່ງກ່ອນໃນເດັສທັອບ ແລະມືຖື {#design-direction-with-desktop-and-mobile-previews}

ກວດທານການອອກແບບຂອງທ່ານທັງໃນອຸປະກອນເດັສທັອບ ແລະມືຖື ກ່ອນທີ່ Theme Builder ຈະສ້າງ theme ທີ່ສົມບູນຂອງທ່ານ. ເບິ່ງຕົວຢ່າງເບິ່ງກ່ອນແບບເວລາຈິງໃນຫຼາຍຈຸດປັບຕາມຂະໜາດໜ້າຈໍ.

ເບິ່ງ [ທິດທາງການອອກແບບ](./theme-builder/design-direction.md) ສຳລັບຄຳແນະນຳແບບລະອຽດ.

### ເມນູສຳລັບທຸລະກິດຕ້ອນຮັບ {#hospitality-menus}

ເບິ່ງ [ເມນູສຳລັບທຸລະກິດຕ້ອນຮັບ](./theme-builder/hospitality-menus.md) ສຳລັບຄຳແນະນຳແບບລະອຽດ.

## ແພັກເກດຄູ່ຮ່ວມຂັ້ນສູງ {#advanced-companion-package}

Superdav AI Agent v1.18.0 ເພີ່ມແພັກເກດຄູ່ຮ່ວມຂັ້ນສູງທີ່ແຈກຈ່າຍແຍກຕ່າງຫາກ ສຳລັບຂັ້ນຕອນວຽກຂອງນັກພັດທະນາທີ່ໄວ້ໃຈໄດ້. ຄວາມສາມາດເຫຼົ່ານີ້ບໍ່ໄດ້ລວມຢູ່ໃນແພັກເກດຫຼັກ WordPress.org. ຕິດຕັ້ງແພັກເກດຄູ່ຮ່ວມສະເພາະໃນເວັບໄຊທີ່ຜູ້ຄຸ້ມຄອງໄວ້ໃຈ agent ຢ່າງຊັດເຈນໃຫ້ເຂົ້າເຖິງເຄື່ອງມືນັກພັດທະນາເທົ່ານັ້ນ.

ເຄື່ອງມືຂັ້ນສູງລວມມີການດຳເນີນການໃນລະບົບໄຟລ໌, ການເຂົ້າເຖິງຖານຂໍ້ມູນ, WP-CLI, ຂັ້ນຕອນວຽກ REST dispatcher, ຢູທິລິຕີຕົວສ້າງ plugin, git snapshots, ຕົວຊ່ວຍຈັດການຜູ້ໃຊ້ ແລະເຄື່ອງມື benchmark. ຈຳກັດເຄື່ອງມືເຫຼົ່ານີ້ໄວ້ໃຫ້ຜູ້ຄຸ້ມຄອງທີ່ໄວ້ໃຈໄດ້ ແລະກວດທານການກຳນົດຄ່າປະຕູການອະນຸມັດກ່ອນເປີດໃຊ້ໃນເວັບໄຊຜະລິດຈິງ.
