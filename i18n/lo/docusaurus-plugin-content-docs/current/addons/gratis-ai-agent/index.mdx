# Gratis AI Agent

Gratis AI Agent ແມ່ນຜູ້ຊ່ວຍ AI ແບບອັດຕະໂນມັດສຳລັບ WordPress ທີ່ສາມາດວາງແຜນ, ສ້າງ, ແລະຈັດການເວັບໄຊ WordPress ທີ່ສົມບູນໄດ້ຜ່ານຄຳສັ່ງພາສາທຳມະຊາດ. ມັນເປີດໃຫ້ໃຊ້ຊຸດ **ຄວາມສາມາດ** ທີ່ມີໂຄງສ້າງ — ການກະທຳແບບແຍກສ່ວນ ແລະທົດສອບໄດ້ທີ່ຕົວແທນສາມາດເອີ້ນໃຊ້ — ຄອບຄຸມທຸກຢ່າງຕັ້ງແຕ່ການຈັດການເນື້ອຫາໄປຈົນເຖິງການປັບແຕ່ງລະບົບອອກແບບ. ເວີຊັນ 1.9.0 ເພີ່ມຄວາມສາມາດການສ້າງເນື້ອຫາ, ການກວດທານໜ້າແບບພາບຜ່ານພາບໜ້າຈໍຝັ່ງລູກຄ້າ, ໂປຣໄຟລ໌ຕົວແທນໃນຕົວ 5 ລາຍການ, ແລະທຸງຟີເຈີສຳລັບການຄວບຄຸມການເຂົ້າເຖິງ ແລະການສ້າງແບຣນ.

## ຟີເຈີຫຼັກ

- **ລະບົບຄວາມສາມາດ** — ລາຍການການກະທຳແບບໂມດູລາ ແລະຂະຫຍາຍໄດ້ທີ່ຕົວແທນສາມາດເອີ້ນໃຊ້ໃນການຕິດຕັ້ງ WordPress ຂອງທ່ານ
- **ການຈັດການເນື້ອຫາ** — ສ້າງບົດຄວາມ ແລະໜ້າຕ່າງໆ (ລວມທັງການກຳນົດແມ່ແບບໜ້າ), ສ້າງບົດຄວາມຫຼາຍລາຍການແບບຊຸດໃນການເອີ້ນຄັ້ງດຽວ, ກຳນົດຮູບພາບເດັ່ນ, ແລະສ້າງແບບຟອມຕິດຕໍ່
- **ການກວດທານແບບພາບ** — ຈັບພາບໜ້າຈໍຂອງໜ້າສົດ, ປຽບທຽບຄວາມຕ່າງລະຫວ່າງສະຖານະກ່ອນ/ຫຼັງ, ແລະຮັບການກວດທານການອອກແບບທີ່ສ້າງໂດຍ AI ຄອບຄຸມໂຄງຮ່າງ, ຕົວພິມ, ສີ, ແລະການເຂົ້າເຖິງ
- **ຕົວແທນໃນຕົວ 5 ລາຍການ** — ຜູ້ຂຽນເນື້ອຫາ, ຜູ້ສ້າງເວັບໄຊ, ສະຕູດິໂອອອກແບບ, ຜູ້ຈັດການປັກອິນ, ແລະຜູ້ຊ່ວຍສະໜັບສະໜູນ; ສາມາດສະຫຼັບໄດ້ຜ່ານຕົວເລືອກຕົວແທນໃນສ່ວນຫົວຂອງແຊັດ
- **ທຸງຟີເຈີ** — ຕົວສະຫຼັບການຄວບຄຸມການເຂົ້າເຖິງ ແລະການສ້າງແບຣນໃນ Settings → Feature Flags ສຳລັບການຈຳກັດຕາມບົດບາດ, ການຕັ້ງຊື່ແບບ white-label, ແລະການປັບແຕ່ງວິດເຈັດ
- **ການຈັດການປະເພດບົດຄວາມກຳນົດເອງ** — ລົງທະບຽນ, ສະແດງລາຍການ, ແລະລຶບປະເພດບົດຄວາມກຳນົດເອງພ້ອມການຄົງຢູ່ຂ້າມການເລີ່ມໃໝ່
- **ການຈັດການຈຳແນກກຳນົດເອງ** — ລົງທະບຽນ, ສະແດງລາຍການ, ແລະລຶບຈຳແນກກຳນົດເອງພ້ອມການຄົງຢູ່
- **ຄວາມສາມາດລະບົບອອກແບບ** — ແຊກ CSS ກຳນົດເອງ, ຈັດການຮູບແບບບລັອກທີ່ຄັດເລືອກແລ້ວ, ກຳນົດໂລໂກ້ເວັບໄຊ, ແລະນຳໃຊ້ຄ່າຕັ້ງລ່ວງໜ້າ theme.json
- **ການຈັດການສະໄຕລ໌ທົ່ວໂລກ** — ອ່ານ ແລະຂຽນຄ່າ theme.json ສຳລັບຊຸດສີ, ຕົວພິມ, ໄລຍະຫ່າງ, ແລະອື່ນໆ
- **ການຈັດການເມນູນຳທາງ** — ສ້າງ ແລະອັບເດດເມນູນຳທາງ ແລະລາຍການຂອງມັນແບບໂປຣແກຣມ
- **ການຈັດການຕົວເລືອກ** — ອ່ານ ແລະຂຽນຕົວເລືອກ WordPress ພ້ອມບັນຊີບລັອກຄວາມປອດໄພໃນຕົວເພື່ອປົກປ້ອງການຕັ້ງຄ່າສຳຄັນ
- **ທະບຽນຄວາມສາມາດທີ່ຕິດຕັ້ງໄດ້** — ຄົ້ນພົບ ແລະເປີດໃຊ້ຊຸດຄວາມສາມາດເພີ່ມເຕີມໂດຍບໍ່ແກ້ໄຂໄຟລ໌ຫຼັກຂອງປັກອິນ
- **ການປະສານການສ້າງເວັບໄຊ v2** — ແຜນການສ້າງເວັບໄຊຫຼາຍຂັ້ນຕອນພ້ອມການຄົ້ນພົບປັກອິນ, ການຕິດຕາມຄວາມຄືບໜ້າ, ແລະການກູ້ຄືນຈາກຂໍ້ຜິດພາດ
- **ຄຳສັ່ງທົດສອບປະສິດທິພາບ WP-CLI** — ແລ່ນຊຸດທົດສອບປະສິດທິພາບຄວາມສາມາດຂອງຕົວແທນຈາກແຖວຄຳສັ່ງສຳລັບ CI ແລະການປະເມີນແບບຈຳລອງ

## ຂໍ້ກຳນົດ

- WordPress 6.0 ຫຼືສູງກວ່າ
- PHP 8.1 ຫຼືສູງກວ່າ
- WordPress Multisite (ເປີດໃຊ້ໃນເຄືອຂ່າຍ) ຫຼືການຕິດຕັ້ງເວັບໄຊດຽວ
- ປັກອິນຕົວເຊື່ອມຕໍ່ຜູ້ໃຫ້ບໍລິການ AI ທີ່ເປີດໃຊ້ຢູ່ (Anthropic, OpenAI, ຫຼືທີ່ເຂົ້າກັນໄດ້)

## ການຕິດຕັ້ງ

1. ອັບໂຫຼດໄຟລ໌ປັກອິນໄປທີ່ `/wp-content/plugins/gratis-ai-agent`
2. ເປີດໃຊ້ປັກອິນຜ່ານໜ້າຈໍ **ປັກອິນ** ໃນ WordPress
3. ໄປທີ່ **Gratis AI Agent → Settings** ແລະເຊື່ອມຕໍ່ຜູ້ໃຫ້ບໍລິການ AI ຂອງທ່ານ
4. (ບໍ່ບັງຄັບ) ເບິ່ງ **ທະບຽນຄວາມສາມາດ** ເພື່ອຕິດຕັ້ງຊຸດຄວາມສາມາດເພີ່ມເຕີມ

## ເລີ່ມຕົ້ນໄວ

ເມື່ອເປີດໃຊ້ ແລະເຊື່ອມຕໍ່ກັບຜູ້ໃຫ້ບໍລິການ AI ແລ້ວ, ໃຫ້ເປີດແຜງ **Gratis AI Agent** ໃນແຖບຂ້າງຜູ້ດູແລ WordPress. ພິມຄຳສັ່ງພາສາທຳມະຊາດ ແລະຕົວແທນຈະວາງແຜນ ແລະດຳເນີນຂັ້ນຕອນທີ່ຈຳເປັນໂດຍໃຊ້ຄວາມສາມາດທີ່ມີຢູ່.

```
ສ້າງປະເພດບົດຄວາມກຳນົດເອງ "Portfolio" ພ້ອມຈຳແນກ "Project Category".
```

```
ແຊກຊຸດສີໂໝດມືດເຂົ້າໃນທີມ ແລະຕັ້ງຄ່າສີຫຼັກເປັນ #1a1a2e.
```

```
ສ້າງເວັບໄຊຮ້ານອາຫານພ້ອມເມນູອອນລາຍ, ແບບຟອມຈອງ, ແລະໜ້າຕິດຕໍ່.
```

## ພາບລວມຄວາມສາມາດ

ຄວາມສາມາດແມ່ນການກະທຳຫົວໜ່ວຍພື້ນຖານທີ່ຕົວແທນສາມາດດຳເນີນການໄດ້. ແຕ່ລະຄວາມສາມາດແມ່ນຄລາສ PHP ທີ່ລົງທະບຽນໄວ້ ເຊິ່ງເປີດໃຫ້ໃຊ້ສະກີມາ (ພາລາມິເຕີ ແລະປະເພດຄ່າສົ່ງຄືນ) ທີ່ຕົວແທນອ່ານໃນເວລາແລ່ນ. ຄວາມສາມາດຖືກຈັດກຸ່ມເປັນຂົງເຂດຟັງຊັນ:

| ຂົງເຂດ | ຄວາມສາມາດ |
|---|---|
| **ການຈັດການເນື້ອຫາ** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **ການກວດທານແບບພາບ** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **ປະເພດບົດຄວາມກຳນົດເອງ** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **ຈຳແນກກຳນົດເອງ** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **ລະບົບອອກແບບ** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **ສະໄຕລ໌ທົ່ວໂລກ** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **ເມນູນຳທາງ** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **ການຈັດການຕົວເລືອກ** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **ຄວາມສາມາດທີ່ຕິດຕັ້ງໄດ້** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **ຜູ້ສ້າງເວັບໄຊ** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

ເບິ່ງ [ອ້າງອີງຄວາມສາມາດ](./abilities) ສຳລັບເອກະສານພາລາມິເຕີຄົບຖ້ວນ ແລະຕົວຢ່າງການໃຊ້ງານ.

## WP-CLI

Gratis AI Agent ມາພ້ອມຊຸດຄຳສັ່ງ `wp gratis-ai-agent` ສຳລັບແລ່ນການທົດສອບປະສິດທິພາບ ແລະຈັດການຄວາມສາມາດຈາກແຖວຄຳສັ່ງ. ເບິ່ງ [ອ້າງອີງ WP-CLI](./wp-cli) ສຳລັບລາຍການຄຳສັ່ງຄົບຖ້ວນ.

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## ການກຳນົດຄ່າ

ການຕັ້ງຄ່າທັງໝົດຢູ່ພາຍໃຕ້ **Gratis AI Agent → Settings** ໃນຜູ້ດູແລ WordPress:

| ການຕັ້ງຄ່າ | ຄຳອະທິບາຍ |
|---|---|
| **ຜູ້ໃຫ້ບໍລິການ AI** | ເລືອກ ແລະກຳນົດຄ່າຕົວເຊື່ອມຕໍ່ຜູ້ໃຫ້ບໍລິການ AI ທີ່ເປີດໃຊ້ |
| **ແບບຈຳລອງ** | ເລືອກແບບຈຳລອງພາສາທີ່ຈະໃຊ້ສຳລັບເຊສຊັນຕົວແທນ |
| **ໂທເຄັນສູງສຸດ** | ງົບປະມານໂທເຄັນຕໍ່ຄຳຂໍ (ຄ່າເລີ່ມຕົ້ນ 4096) |

## ບັນຊີບລັອກຄວາມປອດໄພການຈັດການຕົວເລືອກ

ຄວາມສາມາດການຈັດການ Options ມີລາຍການບລັອກໃນຕົວ ທີ່ປ້ອງກັນ agent ຈາກການອ່ານ ຫຼື ຂຽນ WordPress options ທີ່ລະອຽດອ່ອນ. ລາຍການບລັອກເລີ່ມຕົ້ນຄອບຄຸມ:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (ຕົວເລືອກ URL — ໃຊ້ເຄື່ອງມືຍ້າຍຂໍ້ມູນສະເພາະສຳລັບສິ່ງເຫຼົ່ານີ້)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (ກະແຈລັບ)
- `active_plugins`, `template` (ການເປີດໃຊ້ plugin/theme — ໃຊ້ຄວາມສາມາດສະເພາະ)
- ຂໍ້ມູນລັບຂອງປະຕູຊຳລະເງິນ ແລະ API keys ທີ່ຖືກລົງທະບຽນໂດຍ plugins ທີ່ຕິດຕັ້ງແລ້ວ

ສາມາດເພີ່ມລາຍການເພີ່ມເຕີມໄດ້ຜ່ານ filter `gratis_ai_agent_options_blocklist`.

## Hooks ແລະ Filters

### Actions

- `gratis_ai_agent_ability_registered` — ຖືກເອີ້ນຫຼັງຈາກລົງທະບຽນຄວາມສາມາດແລ້ວ, ຮັບຊື່ class ຂອງຄວາມສາມາດ
- `gratis_ai_agent_plan_started` — ຖືກເອີ້ນເມື່ອແຜນຕົວສ້າງເວັບໄຊເລີ່ມການປະຕິບັດ
- `gratis_ai_agent_plan_completed` — ຖືກເອີ້ນເມື່ອແຜນສຳເລັດຢ່າງສຳເລັດຜົນ
- `gratis_ai_agent_plan_failed` — ຖືກເອີ້ນເມື່ອແຜນພົບຂໍ້ຜິດພາດທີ່ກູ້ຄືນບໍ່ໄດ້

### Filters

- `gratis_ai_agent_abilities` — ປັບປ່ຽນລາຍການຄວາມສາມາດທີ່ລົງທະບຽນໄວ້ ກ່ອນທີ່ agent ຈະໂຫຼດມັນ
- `gratis_ai_agent_options_blocklist` — ຂະຫຍາຍ ຫຼື ຫຼຸດລາຍການບລັອກຄວາມປອດໄພຂອງ options

- `gratis_ai_agent_plan_plugins` — ປັບປ່ຽນລາຍການການຄົ້ນພົບ plugin ໃນລະຫວ່າງການສ້າງແຜນເວັບໄຊ
