# ຕົວກອງ: wu_admin_notices

ອະນຸຍາດໃຫ້ນັກພັດທະນາກັ່ນຕອງການແຈ້ງເຕືອນຜູ້ຄຸ້ມຄອງທີ່ເພີ່ມໂດຍ Ultimate Multisite.

## ພາຣາມິເຕີ

| ຊື່ | ປະເພດ | ຄຳອະທິບາຍ |
|------|------|-------------|
| $notices | `array` | ລາຍການຂອງການແຈ້ງເຕືອນສຳລັບແຜງນັ້ນໂດຍສະເພາະ. |
| $all_notices | `array` | ລາຍການຂອງການແຈ້ງເຕືອນທີ່ເພີ່ມເຂົ້າມາ, ແຍກຕາມແຜງ. |
| $panel | `string` | ແຜງທີ່ຈະດຶງການແຈ້ງເຕືອນ. |
| $filter | `string` | ຖ້າການແຈ້ງເຕືອນທີ່ປິດໄດ້ຖືກກັ່ນອອກໄປແລ້ວ. |
| $dismissed_messages | `array` | ລາຍການຂອງຄີການແຈ້ງເຕືອນທີ່ຖືກປິດໄປແລ້ວ. |
