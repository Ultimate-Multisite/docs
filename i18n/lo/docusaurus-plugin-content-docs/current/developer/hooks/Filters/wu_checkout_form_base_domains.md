# Filter: wu_checkout_form_base_domains

ກັ່ນຕອງໂດເມນພື້ນຖານຂອງແບບຟອມຊຳລະເງິນທີ່ໃຊ້ຮ່ວມກັນ ເຊິ່ງບໍ່ຄວນກາຍເປັນລະບຽນໂດເມນທີ່ຖືກແມັບ.

ໃຊ້ filter ນີ້ເມື່ອການເຊື່ອມຕໍ່ໃຫ້ໂດເມນພື້ນຖານເພີ່ມເຕີມສຳລັບຟິວ **URL ຂອງເວັບໄຊ** ໃນແບບຟອມຊຳລະເງິນ. ໂດເມນທີ່ສົ່ງຄືນໂດຍ filter ນີ້ຈະຖືກປະຕິບັດເປັນໂຮສລົງທະບຽນທີ່ໃຊ້ຮ່ວມກັນ ແທນທີ່ຈະເປັນໂດເມນກຳນົດເອງຕໍ່ແຕ່ລະເວັບໄຊ.
