# ຕົວກັ່ນຕອງ: wu_sso_enabled

ເປີດ/ປິດ ຄວາມສາມາດການເຂົ້າສູ່ລະບົບຄັ້ງດຽວຂ້າມໂດເມນ.

ກັ່ນຕອງຄ່ານີ້ເພື່ອປິດການເຂົ້າສູ່ລະບົບຄັ້ງດຽວທັງໝົດ, ຫຼືເປີດໃຊ້ແບບມີເງື່ອນໄຂແທນ.

|------|------|-------------|
