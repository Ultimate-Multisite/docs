# ຕົວກອງ: wu_stripe_existing_product_id

ກອງ ID ຂອງຜະລິດຕະພັນທີ່ຈະກວດສອບ. ຖ້າມີອັນນີ້ຢູ່, ການສະໝັກສະມາຊິກໃໝ່ຈະໃຊ້ຜະລິດຕະພັນນີ້.

|------|------|-------------|
| $product_id | `string` | ID ຂອງຜະລິດຕະພັນ Stripe ທີ່ຈະກວດສອບ. |
| $name | `string` | ຊື່ຜະລິດຕະພັນ. |
