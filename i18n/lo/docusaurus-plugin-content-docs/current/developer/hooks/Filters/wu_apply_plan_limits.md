# ຕົວກອງ: wu_apply_plan_limits

ອະນຸຍາດໃຫ້ນັກພັດທະນາປລັກອິນຂ້າມຂໍ້ຈຳກັດໄດ້.

ທ່ານສາມາດໃຊ້ຕົວກອງນີ້ເພື່ອເອີ້ນໃຊ້ໂຄດໃດໆ ກ່ອນທີ່ຂໍ້ຈຳກັດໃດໆ ຈະຖືກເລີ່ມຕົ້ນ. ຖ້າຕົວກອງຂອງທ່ານສົ່ງຄ່າໃດໆ ທີ່ເປັນຈິງກັບຄືນມາ, ຂະບວນການຈະດຳເນີນຕໍ່ໄປ; ຖ້າມັນສົ່ງຄ່າໃດໆ ທີ່ເປັນເທັດກັບຄືນມາ, ໂຄດຈະສົ່ງຄືນ ແລະ hooks ດ້ານລຸ່ມນີ້ຈະບໍ່ຖືກເອີ້ນໃຊ້.
