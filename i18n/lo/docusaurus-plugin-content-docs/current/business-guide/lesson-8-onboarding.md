# ບົດຮຽນ 8: ການແນະນຳລູກຄ້າໃໝ່ໃຫ້ເລີ່ມໃຊ້ງານ

ການເຮັດໃຫ້ລູກຄ້າລົງທະບຽນແມ່ນເຮັດວຽກໄດ້ພຽງຄຶ່ງດຽວ. ຖ້າພວກເຂົາເຂົ້າລະບົບແລ້ວຮູ້ສຶກວ່າຫຍຸ້ງຍາກເກີນໄປ ແລະບໍ່ກັບມາອີກ, ທ່ານກໍໄດ້ເສຍລູກຄ້ານັ້ນໄປແລ້ວ. ບົດຮຽນນີ້ອອກແບບປະສົບການທີ່ປ່ຽນຜູ້ລົງທະບຽນໃໝ່ໃຫ້ກາຍເປັນລູກຄ້າທີ່ໃຊ້ງານຈິງ ແລະມີສ່ວນຮ່ວມ.

## ຈຸດທີ່ເຮົາຢຸດໄວ້

FitSite ໄດ້ຖືກປັບແບຣນຢ່າງຄົບຖ້ວນ ພ້ອມກັບຂັ້ນຕອນຊຳລະເງິນທີ່ໃຊ້ງານໄດ້ແລ້ວ. ຕອນນີ້ເຮົາຈະມຸ່ງເນັ້ນສິ່ງທີ່ເກີດຂຶ້ນຫຼັງຈາກເຈົ້າຂອງສະຕູດິໂອຟິດເນສລົງທະບຽນສຳເລັດ ແລະເຂົ້າມາທີ່ເວັບໄຊໃໝ່ຂອງພວກເຂົາເປັນຄັ້ງທຳອິດ.

## ເປັນຫຍັງການແນະນຳໃຫ້ເລີ່ມໃຊ້ງານຈຶ່ງສຳຄັນ

30 ນາທີທຳອິດຫຼັງຈາກລົງທະບຽນຈະກຳນົດວ່າລູກຄ້າຈະຢູ່ຕໍ່ ຫຼືຈະເລີກໃຊ້. ເຈົ້າຂອງສະຕູດິໂອຟິດເນສທີ່:

- ເຂົ້າລະບົບແລ້ວເຫັນເວັບໄຊທີ່ເບິ່ງເປັນເວັບໄຊຟິດເນສຢູ່ແລ້ວ → ຈະຢູ່ຕໍ່
- ຮູ້ຢ່າງຊັດເຈນວ່າຕ້ອງເຮັດຫຍັງຕໍ່ໄປ → ຈະຢູ່ຕໍ່
- ຮູ້ສຶກຫຼົງທາງໃນ WordPress dashboard ແບບທົ່ວໄປ → ຈະອອກໄປ

ແມ່ແບບເຉພາະນິຊຂອງທ່ານ (ບົດຮຽນ 4) ຈັດການຈຸດທຳອິດແລ້ວ. ບົດຮຽນນີ້ຈັດການຈຸດທີສອງ.

## ປະສົບການເຂົ້າລະບົບຄັ້ງທຳອິດ

### ວິດເຈັດຕ້ອນຮັບໃນ Dashboard

ສ້າງ dashboard widget ແບບກຳນົດເອງທີ່ທັກທາຍລູກຄ້າໃໝ່ ແລະນຳທາງພວກເຂົາຜ່ານການຕັ້ງຄ່າ. ສິ່ງນີ້ຄວນປາກົດຢ່າງເດັ່ນຊັດເມື່ອພວກເຂົາເຂົ້າລະບົບຄັ້ງທຳອິດ.

**FitSite ເລີ່ມຕົ້ນດ່ວນ:**

1. **ເພີ່ມຊື່ ແລະໂລໂກສະຕູດິໂອຂອງທ່ານ** -- ລິ້ງໄປຫາ Customizer ຫຼືການຕັ້ງຄ່າຕົວຕົນເວັບໄຊ
2. **ອັບເດດຕາຕະລາງຄລາສຂອງທ່ານ** -- ລິ້ງໂດຍກົງໄປຫາຕົວແກ້ໄຂໜ້າຄລາສ
3. **ເພີ່ມຄູຝຶກຂອງທ່ານ** -- ລິ້ງໄປຫາຕົວແກ້ໄຂໜ້າຄູຝຶກ
4. **ກຳນົດລາຍລະອຽດການຕິດຕໍ່ຂອງທ່ານ** -- ລິ້ງໄປຫາຕົວແກ້ໄຂໜ້າຕິດຕໍ່
5. **ເບິ່ງຕົວຢ່າງເວັບໄຊຂອງທ່ານ** -- ລິ້ງໄປຫາດ້ານໜ້າເວັບໄຊ

ແຕ່ລະຂັ້ນຕອນລິ້ງໂດຍກົງໄປຫາໜ້າ ຫຼືການຕັ້ງຄ່າທີ່ກ່ຽວຂ້ອງ. ບໍ່ຕ້ອງຄົ້ນຫາຜ່ານເມນູ.

### ເຮັດໃຫ້ Dashboard ງ່າຍຂຶ້ນ

ລູກຄ້າໃໝ່ບໍ່ຈຳເປັນຕ້ອງເຫັນທຸກລາຍການເມນູຂອງ WordPress. ພິຈາລະນາ:

- ເຊື່ອງລາຍການເມນູທີ່ບໍ່ກ່ຽວຂ້ອງກັບການຈັດການເວັບໄຊຟິດເນສ (ເຊັ່ນ Comments ຖ້າບໍ່ໄດ້ໃຊ້)
- ຈັດລຳດັບເມນູໃໝ່ໃຫ້ລາຍການທີ່ໃຊ້ບ່ອຍທີ່ສຸດຢູ່ກ່ອນ
- ເພີ່ມປ້າຍຊື່ເມນູແບບກຳນົດເອງທີ່ເຂົ້າກັບນິຊ ("ສະຕູດິໂອຂອງທ່ານ" ແທນ "ຮູບລັກສະນະ")

addon [ຕົວຈັດການ Plugin ແລະ theme](/addons/plugin-and-theme-manager) ສາມາດຊ່ວຍຄວບຄຸມສິ່ງທີ່ລູກຄ້າເຫັນໄດ້.

## ຊຸດອີເມວຕ້ອນຮັບ

ອີເມວຕ້ອນຮັບພຽງສະບັບດຽວບໍ່ພຽງພໍ. ຕັ້ງຄ່າຊຸດອີເມວທີ່ນຳທາງລູກຄ້າຜ່ານອາທິດທຳອິດຂອງພວກເຂົາ:

### ອີເມວ 1: ຕ້ອນຮັບ (ທັນທີຫຼັງຈາກລົງທະບຽນ)

- ຫົວຂໍ້: "ຍິນດີຕ້ອນຮັບສູ່ FitSite -- ເວັບໄຊສະຕູດິໂອຂອງທ່ານໃຊ້ງານໄດ້ແລ້ວ"
- ເນື້ອຫາ: ລິ້ງເຂົ້າລະບົບ, ຂັ້ນຕອນເລີ່ມຕົ້ນດ່ວນ, ລິ້ງໄປຫາຊັບພະຍາກອນຊ່ວຍເຫຼືອ
- ນ້ຳສຽງ: ຕື່ນເຕັ້ນ, ໃຫ້ກຳລັງໃຈ, ເຉພາະດ້ານຟິດເນສ

### ອີເມວ 2: ຜົນສຳເລັດດ່ວນ (ມື້ທີ 1)

- ຫົວຂໍ້: "3 ສິ່ງທີ່ຄວນເຮັດກ່ອນໃນ FitSite ຂອງທ່ານ"
- ເນື້ອຫາ: ເພີ່ມໂລໂກຂອງທ່ານ, ອັບເດດຮູບພາບຮີໂຣໜ້າຫຼັກ, ເພີ່ມຕາຕະລາງຄລາສຂອງທ່ານ
- ໃສ່ພາບໜ້າຈໍທີ່ສະແດງແບບຊັດເຈນວ່າຕ້ອງຄລິກບ່ອນໃດ

### ອີເມວ 3: ເຮັດໃຫ້ເປັນຂອງທ່ານ (ມື້ທີ 3)

- ຫົວຂໍ້: "ເຮັດໃຫ້ເວັບໄຊຟິດເນສຂອງທ່ານໂດດເດັ່ນ"
- ເນື້ອຫາ: ປັບແຕ່ງສີ, ເພີ່ມຮູບຄູຝຶກ, ຂຽນເລື່ອງລາວຂອງສະຕູດິໂອຂອງທ່ານ
- ລິ້ງໄປຫາຕົວຢ່າງເວັບໄຊຟິດເນສທີ່ດີຢູ່ໃນແພລດຟອມ

### ອີເມວ 4: ເປີດໃຊ້ງານຈິງ (ມື້ທີ 7)

- ຫົວຂໍ້: "ພ້ອມແບ່ງປັນ FitSite ຂອງທ່ານໃຫ້ໂລກຮູ້ແລ້ວບໍ?"
- ເນື້ອຫາ: ລາຍການກວດສອບສິ່ງທີ່ຕ້ອງຢືນຢັນກ່ອນແບ່ງປັນ, ວິທີເຊື່ອມຕໍ່ໂດເມນກຳນົດເອງ (ຖ້າຢູ່ໃນ Growth/Pro), ເຄັດລັບການແບ່ງປັນທາງສັງຄົມອອນລາຍ

:::tip ອີເມວອັດຕະໂນມັດ
ໃຊ້ [Webhooks](/user-guide/integrations/webhooks) ຫຼື [Zapier](/user-guide/integrations/zapier) ເພື່ອກະຕຸ້ນອີເມວເຫຼົ່ານີ້ຜ່ານແພລດຟອມການຕະຫຼາດອີເມວຂອງທ່ານ. ສິ່ງນີ້ໃຫ້ທ່ານຄວບຄຸມເວລາສົ່ງໄດ້ຫຼາຍຂຶ້ນ ແລະຊ່ວຍໃຫ້ທ່ານຕິດຕາມການມີສ່ວນຮ່ວມໄດ້.
:::

## ຊັບພະຍາກອນຊ່ວຍເຫຼືອ

ສ້າງເນື້ອຫາຊ່ວຍເຫຼືອເຉພາະນິຊທີ່ຕອບຄຳຖາມທີ່ເຈົ້າຂອງສະຕູດິໂອຟິດເນສຖາມຈິງ:

### ບົດຄວາມຖານຄວາມຮູ້

- "ວິທີອັບເດດຕາຕະລາງຄລາສຂອງທ່ານ"
- "ການເພີ່ມ ແລະແກ້ໄຂໂປຣໄຟລ໌ຄູຝຶກ"
- "ການປ່ຽນໂລໂກ ແລະສີຂອງສະຕູດິໂອຂອງທ່ານ"
- "ການເຊື່ອມຕໍ່ຊື່ໂດເມນຂອງທ່ານເອງ" (ສຳລັບລູກຄ້າ Growth/Pro)
- "ການເພີ່ມວິດເຈັດຈອງເຂົ້າໃນເວັບໄຊຂອງທ່ານ"

ຂຽນສິ່ງເຫຼົ່ານີ້ສຳລັບຜູ້ໃຊ້ທີ່ບໍ່ແມ່ນສາຍເຕັກນິກ. ໃຊ້ພາບໜ້າຈໍ. ຫຼີກລ່ຽງຄຳສັບເຉພາະຂອງ WordPress.

### ວິດີໂອນຳທາງ

ການບັນທຶກໜ້າຈໍສັ້ນໆ (2-3 ນາທີ) ທີ່ສະແດງ:

- ການເຂົ້າລະບົບຄັ້ງທຳອິດ ແລະການປັບຕົວ
- ການແກ້ໄຂໜ້າຫຼັກ
- ການອັບເດດຕາຕະລາງຄລາສ
- ການເພີ່ມຄູຝຶກໃໝ່

ສິ່ງເຫຼົ່ານີ້ບໍ່ຈຳເປັນຕ້ອງເປັນງານຜະລິດທີ່ຂັດເງົາສົມບູນ. ຊັດເຈນ, ເປັນປະໂຫຍດ, ແລະເຉພາະນິຊ ແມ່ນສິ່ງທີ່ສຳຄັນ.

## ໜ້າ Account

Ultimate Multisite ມີ [ໜ້າ Account](/user-guide/client-management/account-page) ສຳລັບລູກຄ້າ ບ່ອນທີ່ລູກຄ້າຈັດການການສະໝັກໃຊ້ຂອງພວກເຂົາ. ປັບແຕ່ງສິ່ງນີ້ເພື່ອ:

- ສະແດງແຜນ FitSite ປັດຈຸບັນຂອງພວກເຂົາ
- ສະແດງຕົວເລືອກອັບເກຣດພ້ອມປະໂຫຍດເຉພາະດ້ານຟິດເນສ
- ໃຫ້ປະຫວັດການຮຽກເກັບເງິນ ແລະການດາວໂຫຼດໃບແຈ້ງໜີ້
- ລິ້ງໄປຫາຊັບພະຍາກອນຊ່ວຍເຫຼືອ

## ການວັດແທກຄວາມສຳເລັດຂອງການແນະນຳໃຫ້ເລີ່ມໃຊ້ງານ

ຕິດຕາມຕົວຊີ້ວັດເຫຼົ່ານີ້ເພື່ອຮູ້ວ່າການແນະນຳໃຫ້ເລີ່ມໃຊ້ງານຂອງທ່ານໃຊ້ໄດ້ຜົນບໍ:

- **ອັດຕາການເປີດໃຊ້ງານ**: ມີຜູ້ລົງທະບຽນກີ່ເປີເຊັນທີ່ປັບແຕ່ງເວັບໄຊຂອງຕົນຈິງພາຍໃນອາທິດທຳອິດ?
- **ການເຂົ້າລະບົບໃນອາທິດທຳອິດ**: ລູກຄ້າໃໝ່ເຂົ້າລະບົບຈັກຄັ້ງໃນອາທິດທຳອິດຂອງພວກເຂົາ?
- **ຕິກເກັດຊ່ວຍເຫຼືອຈາກລູກຄ້າໃໝ່**: ປະລິມານສູງໝາຍຄວາມວ່າການແນະນຳໃຫ້ເລີ່ມໃຊ້ງານຂອງທ່ານມີຊ່ອງຫວ່າງ
- **ການປ່ຽນຈາກທົດລອງເປັນຈ່າຍເງິນ**: ຖ້າທ່ານສະເໜີການທົດລອງໃຊ້, ມີກີ່ເປີເຊັນທີ່ປ່ຽນເປັນລູກຄ້າຈ່າຍເງິນ?

## ສິ່ງທີ່ເຮົາສ້າງໃນບົດຮຽນນີ້

- **ປະສົບການເຂົ້າລະບົບຄັ້ງທຳອິດແບບມີການນຳທາງ** ພ້ອມວິດເຈັດເລີ່ມຕົ້ນດ່ວນ
- **Dashboard ທີ່ງ່າຍຂຶ້ນ** ທີ່ເນັ້ນວຽກຈັດການເວັບໄຊຟິດເນສ
- **ຊຸດອີເມວຕ້ອນຮັບ** ທີ່ນຳທາງລູກຄ້າຜ່ານອາທິດທຳອິດຂອງພວກເຂົາ
- **ຊັບພະຍາກອນຊ່ວຍເຫຼືອເຉພາະນິຊ** ທີ່ຂຽນສຳລັບເຈົ້າຂອງສະຕູດິໂອຟິດເນສທີ່ບໍ່ແມ່ນສາຍເຕັກນິກ
- **ໂຕຊີ້ວັດການແນະນຳໃຫ້ເລີ່ມໃຊ້ງານ** ເພື່ອຕິດຕາມ ແລະປັບປຸງປະສົບການ
