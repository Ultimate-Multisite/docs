# ບົດຮຽນ 5: ການອອກແບບແຜນຂອງທ່ານ {#lesson-5-designing-your-plans}

ລະດັບແຜນຂອງທ່ານບໍ່ແມ່ນພຽງແຕ່ລະດັບລາຄາ -- ມັນແມ່ນພາບສະທ້ອນວ່າລູກຄ້າໃນຕະຫຼາດສະເພາະຂອງທ່ານດຳເນີນງານຈິງແນວໃດ. ໃນບົດຮຽນນີ້ ທ່ານຈະອອກແບບລະດັບຜະລິດຕະພັນທີ່ກົງກັບຄວາມຕ້ອງການຈິງຂອງທຸລະກິດຟິດເນສໃນແຕ່ລະຂັ້ນ.

## ຈຸດທີ່ເຮົາຢຸດໄວ້ {#where-we-left-off}

FitSite ມີແມ່ແບບສາມອັນພ້ອມແລ້ວ (Studio Essential, Gym Pro, Fitness Chain). ຕອນນີ້ເຮົາຈະສ້າງແຜນທີ່ກຳນົດວ່າລູກຄ້າຈະໄດ້ເຂົ້າເຖິງຫຍັງ ແລະຈະຈ່າຍເທົ່າໃດ.

## ການຄິດຕາມກຸ່ມລູກຄ້າ {#thinking-in-customer-segments}

ຂໍ້ຜິດພາດທີ່ຄົນສ່ວນໃຫຍ່ເຮັດແມ່ນການອອກແບບແຜນໂດຍອີງໃສ່ຄຸນສົມບັດທາງເຕັກນິກ (ພື້ນທີ່ເກັບຂໍ້ມູນ, ແບນວິດ, ຈຳນວນໜ້າ). ລູກຄ້າໃນຕະຫຼາດສະເພາະຂອງທ່ານບໍ່ໄດ້ຄິດໃນແນວນັ້ນ. ເຈົ້າຂອງຍິມຈະຄິດເຖິງສິ່ງທີ່ພວກເຂົາຕ້ອງການເພື່ອດຳເນີນທຸລະກິດ.

ສຳລັບສະຕູດິໂອຟິດເນສ, ມີສາມກຸ່ມທຳມະຊາດ:

| ກຸ່ມ | ພວກເຂົາແມ່ນໃຜ | ສິ່ງທີ່ພວກເຂົາຕ້ອງການ |
|---------|-------------|----------------|
| **ຄູຝຶກສ່ວນຕົວ / ສະຕູດິໂອນ້ອຍ** | ພະນັກງານ 1-3 ຄົນ, ສະຖານທີ່ດຽວ, ກຳລັງເລີ່ມຕົ້ນ | ເວັບໄຊມືອາຊີບ, ຂໍ້ມູນຄລາສ, ແບບຟອມຕິດຕໍ່ |
| **ຍິມທີ່ຕັ້ງຕົວແລ້ວ** | ພະນັກງານ 5-20 ຄົນ, ສະຖານທີ່ດຽວ, ກຳລັງເຕີບໂຕ | ທຸກຢ່າງຂ້າງເທິງ ພ້ອມກັບການຈອງ, ບລັອກ, ແກລເລີຣີ, ໂດເມນກຳນົດເອງ |
| **ເຄືອຂ່າຍຟິດເນສ** | ຫຼາຍສະຖານທີ່, ແບຣນທີ່ຕັ້ງຕົວແລ້ວ | ທຸກຢ່າງຂ້າງເທິງ ພ້ອມກັບ multi-site, ໜ້າສະຖານທີ່, ໄດເຣັກທໍຣີພະນັກງານ |

ແຜນຂອງທ່ານຄວນຈັບຄູ່ກັບກຸ່ມເຫຼົ່ານີ້, ບໍ່ແມ່ນກັບຊຸດຄຸນສົມບັດທີ່ກຳນົດຂຶ້ນແບບສຸ່ມ.

## ການສ້າງແຜນ FitSite {#creating-the-fitsite-plans}

ໄປທີ່ **Ultimate Multisite > ຜະລິດຕະພັນ > ເພີ່ມຜະລິດຕະພັນ** ສຳລັບແຕ່ລະແຜນ.

### ແຜນ 1: FitSite Starter -- $49/ເດືອນ {#plan-1-fitsite-starter----49month}

**ເປົ້າໝາຍ**: ຄູຝຶກສ່ວນຕົວ ແລະ ສະຕູດິໂອນ້ອຍ

**ແຖບຄຳອະທິບາຍ**:
- ຊື່: FitSite Starter
- ຄຳອະທິບາຍ: "ທຸກຢ່າງທີ່ຄູຝຶກສ່ວນຕົວ ຫຼື ສະຕູດິໂອນ້ອຍຕ້ອງການເພື່ອໃຫ້ເບິ່ງເປັນມືອາຊີບໃນອອນໄລນ໌."

**ແຖບທົ່ວໄປ**:
- ປະເພດຜະລິດຕະພັນ: ແຜນ
- ບົດບາດລູກຄ້າ: ຜູ້ດູແລ

**ແຖບແມ່ແບບເວັບໄຊ**:
- ອະນຸຍາດແມ່ແບບເວັບໄຊ: ເປີດໃຊ້
- ແມ່ແບບທີ່ມີໃຫ້: Studio Essential ເທົ່ານັ້ນ

**ຂໍ້ຈຳກັດ**:
- ເວັບໄຊ: 1
- ພື້ນທີ່ດິສກ໌: 1 GB
- ໂດເມນກຳນົດເອງ: ປິດໃຊ້ (ໃຊ້ `studioname.fitsite.com`)

**ແຖບ plugin**:
- plugin ແບບຟອມຕິດຕໍ່: ບັງຄັບເປີດໃຊ້
- plugin SEO: ບັງຄັບເປີດໃຊ້
- plugin ການຈອງ: ບໍ່ມີໃຫ້ (ແຮງຈູງໃຈໃຫ້ອັບເກຣດ)

**ແຖບ theme**:
- theme ທີ່ທ່ານເລືອກ: ບັງຄັບເປີດໃຊ້
- theme ອື່ນທັງໝົດ: ເຊື່ອງ

### ແຜນ 2: FitSite Growth -- $99/ເດືອນ {#plan-2-fitsite-growth----99month}

**ເປົ້າໝາຍ**: ຍິມສະຖານທີ່ດຽວທີ່ຕັ້ງຕົວແລ້ວ

**ແຖບຄຳອະທິບາຍ**:
- ຊື່: FitSite Growth
- ຄຳອະທິບາຍ: "ສຳລັບຍິມທີ່ຕັ້ງຕົວແລ້ວ ພ້ອມທີ່ຈະເຕີບໂຕການປະກົດຕົວອອນໄລນ໌ ແລະ ດຶງດູດສະມາຊິກໃໝ່."

**ແຖບແມ່ແບບເວັບໄຊ**:
- ແມ່ແບບທີ່ມີໃຫ້: Studio Essential ແລະ Gym Pro

**ຂໍ້ຈຳກັດ**:
- ເວັບໄຊ: 1
- ພື້ນທີ່ດິສກ໌: 5 GB
- ໂດເມນກຳນົດເອງ: ເປີດໃຊ້

**ແຖບ plugin**:
- ທຸກຢ່າງໃນ Starter, ພ້ອມກັບ:
- plugin ການຈອງ: ບັງຄັບເປີດໃຊ້
- plugin ແກລເລີຣີ: ບັງຄັບເປີດໃຊ້
- ຟັງຊັນບລັອກ: ມີໃຫ້

**ແຖບອັບເກຣດ ແລະ ດາວເກຣດ**:
- ກຸ່ມແຜນ: FitSite Plans
- ລຳດັບຜະລິດຕະພັນ: 2

### ແຜນ 3: FitSite Pro -- $199/ເດືອນ {#plan-3-fitsite-pro----199month}

**ເປົ້າໝາຍ**: ເຄືອຂ່າຍຟິດເນສຫຼາຍສະຖານທີ່

**ແຖບຄຳອະທິບາຍ**:
- ຊື່: FitSite Pro
- ຄຳອະທິບາຍ: "ແພລດຟອມຄົບຖ້ວນສຳລັບແບຣນຟິດເນສທີ່ມີຫຼາຍສະຖານທີ່."

**ແຖບແມ່ແບບເວັບໄຊ**:
- ແມ່ແບບທີ່ມີໃຫ້: ແມ່ແບບທັງສາມ

**ຂໍ້ຈຳກັດ**:
- ເວັບໄຊ: 5 (ໜຶ່ງອັນຕໍ່ສະຖານທີ່)
- ພື້ນທີ່ດິສກ໌: 20 GB
- ໂດເມນກຳນົດເອງ: ເປີດໃຊ້

**ແຖບ plugin**:
- ທຸກຢ່າງໃນ Growth, ພ້ອມກັບ:
- plugin ພຣີມຽມທັງໝົດ: ບັງຄັບເປີດໃຊ້

**ແຖບອັບເກຣດ ແລະ ດາວເກຣດ**:
- ກຸ່ມແຜນ: FitSite Plans
- ລຳດັບຜະລິດຕະພັນ: 3

## ການຕັ້ງຄ່າກຸ່ມແຜນ {#setting-up-the-plan-group}

ກຸ່ມແຜນຊ່ວຍໃຫ້ລູກຄ້າສາມາດອັບເກຣດ ຫຼື ດາວເກຣດໄດ້ພາຍໃນຄອບຄົວແຜນ FitSite ເທົ່ານັ້ນ. ໃນແຖບ **ອັບເກຣດ ແລະ ດາວເກຣດ** ຂອງແຕ່ລະແຜນ:

1. ຕັ້ງ **ກຸ່ມແຜນ** ເປັນ "FitSite Plans" ສຳລັບແຜນທັງສາມ
2. ຕັ້ງ **ລຳດັບຜະລິດຕະພັນ** ເປັນ 1 (Starter), 2 (Growth), 3 (Pro)

ສິ່ງນີ້ສ້າງເສັ້ນທາງອັບເກຣດທີ່ຊັດເຈນ: Starter → Growth → Pro.

## ການເພີ່ມ Order Bumps {#adding-order-bumps}

Order bumps ແມ່ນຜະລິດຕະພັນເສີມທີ່ສະເໜີໃນລະຫວ່າງ checkout. ສຳລັບ FitSite, ພິຈາລະນາ:

- **ຊຸດພື້ນທີ່ເກັບຂໍ້ມູນເພີ່ມ** ($19/ເດືອນ) -- ພື້ນທີ່ດິສກ໌ເພີ່ມ 5 GB
- **ການສະໜັບສະໜູນແບບສຳຄັນກ່ອນ** ($29/ເດືອນ) -- ເວລາຕອບກັບໄວຂຶ້ນ
- **ເວັບໄຊເພີ່ມ** ($39/ເດືອນ) -- ສຳລັບລູກຄ້າທີ່ຕ້ອງການເວັບໄຊຫຼາຍກວ່າທີ່ແຜນຂອງພວກເຂົາອະນຸຍາດ

ສ້າງສິ່ງເຫຼົ່ານີ້ເປັນຜະລິດຕະພັນປະເພດ **ແພັກເກດ** ໃນ Ultimate Multisite ແລະ ເຊື່ອມໂຍງພວກມັນກັບແຜນທີ່ກ່ຽວຂ້ອງ.

## ເປັນຫຍັງໂຄງສ້າງນີ້ຈຶ່ງໃຊ້ໄດ້ດີ {#why-this-structure-works}

- **Starter** ລົບອຸປະສັກໃນການເລີ່ມຕົ້ນ -- ລາຄາຕ່ຳ, ຂໍ້ສະເໜີງ່າຍໆ, ເຮັດໃຫ້ຄູຝຶກຂຶ້ນອອນໄລນ໌ໄດ້ໄວ
- **Growth** ເພີ່ມຄຸນສົມບັດທີ່ຍິມຂໍຈິງໆ -- ການຈອງ, ແກລເລີຣີ, ໂດເມນກຳນົດເອງ
- **Pro** ຮອງຮັບກຸ່ມມູນຄ່າສູງທີ່ຕ້ອງການການສະໜັບສະໜູນຫຼາຍສະຖານທີ່
- **Order bumps** ໃຫ້ລູກຄ້າປັບແຕ່ງໄດ້ໂດຍບໍ່ເຮັດໃຫ້ແຜນຫຼັກຊັບຊ້ອນ
- **ເສັ້ນທາງອັບເກຣດທີ່ຊັດເຈນ** ໝາຍຄວາມວ່າລູກຄ້າເຕີບໂຕໄປກັບທ່ານແທນທີ່ຈະຈາກໄປ

## ເຄືອຂ່າຍ FitSite ຈົນເຖິງຕອນນີ້ {#the-fitsite-network-so-far}

```
FitSite Network
├── WordPress Multisite (subdomain mode)
├── Ultimate Multisite (configured)
├── Hosting with wildcard SSL + domain mapping
├── Site Templates (Studio Essential, Gym Pro, Fitness Chain)
├── Products
│   ├── FitSite Starter ($49/mo) → Studio Essential template
│   ├── FitSite Growth ($99/mo) → Studio Essential + Gym Pro templates
│   ├── FitSite Pro ($199/mo) → All templates
│   └── Order Bumps (Extra Storage, Priority Support, Additional Site)
└── Ready for checkout configuration (next lesson)
```

## ສິ່ງທີ່ເຮົາສ້າງໃນບົດຮຽນນີ້ {#what-we-built-this-lesson}

- **ລະດັບແຜນສາມລະດັບ** ທີ່ຈັບຄູ່ກັບກຸ່ມທຸລະກິດຟິດເນສຈິງ
- **ການກຳນົດສິດຄຸນສົມບັດ** ໂດຍໃຊ້ການຄວບຄຸມ plugin ແລະ ແມ່ແບບຂອງ Ultimate Multisite
