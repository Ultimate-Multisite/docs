# Ultimate Multisite hujjatlari {#ultimate-multisite-documentation}

Ultimate Multisite hujjatlariga xush kelibsiz. Ultimate Multisite WordPress Multisite’ni to‘liq Website as a Service (WaaS) platformasiga aylantiradi.

## Hujjat bo‘limlari {#documentation-sections}

### [Foydalanuvchi qo‘llanmasi](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

WaaS platformangizni o‘rnatish, sozlash va boshqarish uchun kerak bo‘lgan barcha narsa. O‘rnatish, to‘lov shlyuzlari, domen xaritalash, mijozlarni boshqarish va boshqalarni qamrab oladi.

### [Dasturchi hujjatlari](/developer/hooks) {#developer-documentation}

Ultimate Multisite asosida yaratayotgan dasturchilar uchun texnik ma’lumotnoma. Hooklar ma’lumotnomasi, REST API hujjatlari va qo‘shimcha plaginlarni ishlab chiqish qo‘llanmalarini o‘z ichiga oladi.

### [Qo‘shimchalar](/addons) {#addons}

Ultimate Multisite’ni WooCommerce integratsiyasi, tahlillar, domen sotish va boshqa qo‘shimcha imkoniyatlar bilan kengaytiradigan pulli qo‘shimcha plaginlar uchun hujjatlar.
