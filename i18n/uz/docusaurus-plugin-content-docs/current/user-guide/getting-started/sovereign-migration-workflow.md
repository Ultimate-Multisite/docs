# Монархияли Миграция Процесс {#sovereign-migration-workflow}

Ultimate Multisite: Multi-Tenancy 1.2.0 стандартного субсайта суверенный арендатор инфраструктуре ксли миграция верификация вороталарни қўшиб беради.

## Бошқачадан олдин {#before-you-start}

Сизда қуйидагилар борлигини тасдиқланг:

- Манба сайтнинг ҳозирда бўлган бэкап.
- Multi-Tenancy аддон фаол.
- Мақсад арендатор базаси, файлосининг асосий тупиғи ва домен тайёр.
- Мақсад муҳитга мос келадиган база хости боғланиши (binding).
- Тармоқ учун WP-CLI буйруқларини ишлашга кириш имконияти.

## Тавсия этиладиган процесс {#recommended-workflow}

1. Мақсад арендатор базаси ва файлосини тайёрлаш.
2. Арендатор изоляция sozlamларини роштибон қилиш ёки янгилаш.
3. Арендатор миграциясини ишлатиш.
4. Арендатор фойдаланучларини тақдим этиш ёки текшириш.
5. Асинх миграция жуйеларини очиб юбориш (drain).
6. Суверен миграция верификацию ишлатиш.
7. SSO билан арендаторга кириш.
8. Верификация ўтганлигидан сўнг фақат DNS ёки маршрутизацияни ўзгартириш.

## Верификация вороқлари {#verification-gates}

Верификация жараёни миграцияни бир нечта бурчадан текширади:

- Арендатор схемаси мавжуд ва арендатор базаси ёзувчиси томонидан ёзиш мумкин.
- Созланган база хостида ҳақиқатли рухсатлар (grants) борлиги.
- Арендатор фойдаланучлари мавжуд ва кутилган суверен ўрнатилган фойдалануч сони билан мос келади.
- Асинх пуш кутучаси муваффақиятли очиб юборилиши мумкин.
- Суверен арендатор учун эски тармоқ томонидаги маълумот йўналишлари (data paths) кўздан чиқилмаган.

Верификацияда келиб чиққан муаммоларни ишга туширишдан олдин тўсиқлар сифатида қабул қилинг. База, фойдалануч, кутуча ёки маршрутизация муаммосини аниқлаб тузатинг, сўнгра арендаторни мижозларга тақдим этишдан олдин верификацию қайта ишлатинг.

## Биринчи продукцияга кириш {#first-production-visit}

Верификация ўтганлигидан сўнг, биринчи административ кириш учун сайт бошқаруви экрандан **Visit (SSO)** ни ишлатинг. Бу арендатор маршрутизациясини, SSO токенини қайта ишлашни, манбани белгилаш ва арендатор томонидаги фойдаланучларни тайёрлаш барчани бир контрол қилинган қадамда тасдиқлайди.
