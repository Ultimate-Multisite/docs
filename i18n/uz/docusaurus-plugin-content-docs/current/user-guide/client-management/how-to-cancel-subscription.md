# Abonamentimni qanday bekor qilishim mumkin?

Agar sizda Ultimate Multisite litsenziyasi bo'lsa, uni qayta yangilashni istalgan vaqtda bekor qilishingiz mumkin. Faqat quyidagi qadamlarni bajaring:

<!-- Screenshot unavailable: Account page email with access link -->

Xarid paytida berilgan elektron pochta manzilingiz va parolingizdan foydalaning.

<!-- Screenshot unavailable: Login page with email and password fields -->

Bu sizning Hisob sahifangizdagi boshlang'ich ekraningizdir:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

Abonamentingizni bekor qilish uchun, chap menyuda **_Renewing & Billing._** (Qayta yangilash va hisob-kitob) bo'limiga o'ting.

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

O'ng tomondagi yo'naltirilgan tugmani bosib yon oynani oching. Keyin, **_auto-renew._** (avtomatik yangilanish)ni bekor qiladigan variantni tanlashingiz kerak.

Tizim sizga tasdiqlash xabari beradi.

Agar abonamentingizni bekor qilishdan amin bo'lsangiz, shunchaki **_Cancel Renewals._** (Qayta yangilashlarni bekor qilish) tugmasini bosing. Ushbu harakatdan so'ng sizdan qisqa so'rov javobi berishingiz so'raladi.

Buxida! Sizning abonamentingiz avtomatik ravishda qayta yangillanmaydi.

**Abonamentingiz muddati tugaguncha sizda amal qiladigan kalit bo'ladi**. Agar abonamentingizni qayta faollashtirmoqchi bo'lsangiz, buni qo'lda qilish kerak bo'ladi.
