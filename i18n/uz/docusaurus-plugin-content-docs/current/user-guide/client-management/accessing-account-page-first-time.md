# Birinchi marta akkaunt sahifamga qanday kirishim mumkin? {#how-can-i-access-my-account-page-for-the-first-time}

Ultimate Multisite'ga xush kelibsiz!

Bizning pluginimizni sotib olganingizdan so'ng sizga yuborilgan elektron pochta orqali olingan link orqali akkaunt sahifangizga kirishingiz mumkin:

<!-- Screenshot unavailable: Account page email with access link -->

Sotish paytida kiritgan elektron pochta va sizga yuborilgan parol bilan kirishingiz mumkin.

<!-- Screenshot unavailable: Login page with email and password fields -->

Bu akkauntga birinchi marta kirishingiz uchun parolingizni o'zgartirishingiz so'raladi. Uni o'zgartiring va _Save_ tugmasini bosing.

<!-- Screenshot unavailable: Change password form with Save button -->

Akkaunt sahifasida siz hisobingizdagi barcha ma'lumotlarni, masalan, hisoblashlar (billings), yangilashlar (renewals), bekor qilish va boshqa ko'plab narsalarni boshqara olasiz.
