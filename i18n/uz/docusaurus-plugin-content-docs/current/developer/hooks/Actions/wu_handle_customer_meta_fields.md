# Harakat: wu_handle_customer_meta_fields {#action-wuhandlecustomermetafields}

Plagin ishlab chiquvchilariga kerak bo‘lsa, meta ma’lumotlarni turli usullarda saqlashga ruxsat beradi.

## Parametrlar {#parameters}

| Nomi | Turi | Tavsif |
|------|------|-------------|
| $meta_repository | `array` | Meta maydonlar ro‘yxati, key =&gt; value tuzilmasida. |
| $customer | `\Customer` | Ultimate Multisite mijoz obyekti. |
| $checkout | `\Checkout` | To‘lovni yakunlash klassi. |
