# Filter: wu_magic_link_enforce_ip {#filter-wumagiclinkenforceip}

IP manzilni tekshirishni majburlash yoki majburlamaslikni filter qiladi.

Tokenlar turli tarmoqlardan ishlashiga ruxsat berish uchun false qilib sozlang. Bu xavfsizlikni kamaytiradi, lekin foydalanish qulayligini oshiradi (masalan, tarmoqlarni almashtirayotgan mobil foydalanuvchilar uchun).

## Parametrlar {#parameters}

| Name | Type | Tavsif |
|------|------|-------------|
| $enforce | `bool` | IP manzil mosligini majburlash kerakmi. |
