# Filtr: wu_admin_notices

Dasturchilarga Ultimate Multisite tomonidan qo‘shilgan admin bildirishnomalarini filtrlashga ruxsat beradi.

## Parametrlar {#parameters}

| Nomi | Turi | Tavsif |
|------|------|-------------|
| $notices | `array` | Muayyan panel uchun bildirishnomalar ro‘yxati. |
| $all_notices | `array` | Panel bo‘yicha ajratilgan qo‘shilgan bildirishnomalar ro‘yxati. |
| $panel | `string` | Bildirishnomalarni olish uchun panel. |
| $filter | `string` | Agar yopib qo‘yiladigan bildirishnomalar chiqarib tashlangan bo‘lsa. |
| $dismissed_messages | `array` | Yopib qo‘yilgan bildirishnoma kalitlari ro‘yxati. |
