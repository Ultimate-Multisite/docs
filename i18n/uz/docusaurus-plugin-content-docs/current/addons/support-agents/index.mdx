# Support Agents {#support-agents}

Butun tarmoqni o‘zingiz boshqarish oson vazifa emas. Tarmog‘ingiz o‘sishni boshlagani sari, qo‘llab-quvvatlashga bo‘lgan talab ortib borayotganini ko‘ra boshlaysiz. Ish yukingizni yengillashtirishga yordam beradigan odamlarni jalb qilish yaxshi fikr bo‘ladi. Biroq, to‘liq kirish/ruxsat berish xavfli va tarmog‘ingiz xavfsizligiga putur yetkazishi mumkin. Aynan shu yerda ushbu Support Agent qo‘shimchasi yordamga keladi.

Ushbu qo‘shimcha Support Agent deb nomlangan yangi turdagi WordPress foydalanuvchisini yaratishga imkon beradi va bu foydalanuvchi qanday ruxsatlarga ega bo‘lishi ustidan to‘liq nazorat beradi. Ruxsat berish toggle tugmasini yoqishdek oson, uni olib tashlashga qaror qilsangiz, o‘chirib qo‘yish kifoya.

## Xususiyatlar {#features}

- WordPress va Ultimate Multisite ruxsatlarining bandma-band ro‘yxati
- Kirishni oson yoqish va o‘chirish
- Cheklanmagan miqdorda Agent Accountlarini yarating

## O‘rnatish {#installation}

1. 'ultimate-multisite-support-agents' ni '/wp-content/plugins/' katalogiga yuklang
2. WordPressdagi 'Plugins' menyusi orqali plaginni faollashtiring
3. Plaginni sozlash uchun bosqichma-bosqich Wizard ko‘rsatmalariga amal qiling
