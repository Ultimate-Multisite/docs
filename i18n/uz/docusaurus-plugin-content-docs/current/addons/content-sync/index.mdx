# Content Sync

Multisite Ultimate: Content Sync

Multisite Ultimate uchun ushbu addon tarmoq administratorlariga kontentni istalgan saytdan tarmoqdagi boshqa saytlarga sinxronlash imkonini beradi. U postlar, sahifalar, maxsus post turlari va Elementor shablonlarini o‘z ichiga olgan barcha post turlarini qo‘llab-quvvatlaydi. Addon kontent va maqsad saytlarni tanlash uchun ko‘p bosqichli wizard interfeysini, batch qayta ishlash va real vaqtdagi jarayon feedback bilan birga o‘z ichiga oladi.

## Xususiyatlar {#features}

* Kontent tanlash: Thumbnails va post turi filtrlashi bilan vizual karta panjarasi
* Barcha post turlari: Postlar, sahifalar, mahsulotlar, Elementor shablonlari va istalgan maxsus post turini sinxronlaydi
* Sayt tanlash: Kontentni qaysi saytlarga sinxronlashni tanlang, manba saytni chiqarib tashlash va child saytlarni oldindan tanlash bilan
* Batch qayta ishlash: Jarayon yangilanishlari bilan bir vaqtda 10 ta saytga sinxronlaydi
* Media sinxronlash: Media fayllarni (rasmlar, videolar) post kontentidan maqsad saytlarga avtomatik nusxalaydi va qayta xaritalaydi
* Taksonomiya sinxronlash: Kategoriyalar, teglar va maxsus taksonomiyalarni saqlab qoladi
* Elementor qo‘llab-quvvatlovi: CSS qayta yaratish va shablon ID larini qayta xaritalash bilan Elementor shablonlari uchun to‘liq qo‘llab-quvvatlash
* Mojarolarni hal qilish: Mavjud kontentni almashtirish yoki uni o‘tkazib yuborish opsiyasi
* Kesh boshqaruvi: Sinxronlashdan keyin maqsad saytlarda Elementor keshini avtomatik tozalaydi

## Talablar {#requirements}

* WordPress 5.3 yoki undan yuqori
* PHP 7.4 yoki undan yuqori
* Multisite Ultimate plugin (faol)
* Elementor plugin (ixtiyoriy, Elementor shablon sinxronlash uchun talab qilinadi)

## O‘rnatish {#installation}

1. 'ultimate-multisite-content-sync' ni '/wp-content/plugins/' katalogiga yuklang
2. WordPress Network Admin ichidagi 'Plugins' menyusi orqali plugin ni faollashtiring
3. Ultimate Multisite → Settings → Content Sync bo‘limida sozlamalarni sozlang
4. Sinxronlashni boshlash uchun Ultimate Multisite → Content Sync bo‘limiga o‘ting
