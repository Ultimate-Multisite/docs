# Material WP

WordPress admin interfeysi yillar davomida bir xil bo‘lib kelgan. Ammo admin interfeysining Material Design versiyasi qanday ko‘rinishda bo‘lishi haqida hech o‘ylab ko‘rganmisiz? 

Material WP — sizga ana shu ko‘rinish va hissiyotni berish uchun ishlab chiqilgan admin theme bo‘lib, mijozlaringizga yanada moslashtirilgan Dashboard taqdim etishingizga yordam beradi.

## Xususiyatlar

- Maxsus CSS qo‘llab-quvvatlashi
- Oson uslublash sozlamalari
- Menyuni qayta tartiblash
- Oson import/eksport

## Hujjatlar

**Admin panelingizni Material WP bilan moslashtirish**

## O‘rnatish

1. 'material-wp'ni '/wp-content/plugins/' katalogiga yuklang
2. WordPress ichidagi 'Plaginlar' menyusi orqali pluginni faollashtiring

## Foydalanish qo‘llanmasi

WordPress admin interfeysi yillar davomida bir xil bo‘lib kelgan. Ammo admin interfeysining Material Design versiyasi qanday ko‘rinishda bo‘lishi haqida hech o‘ylab ko‘rganmisiz?

Material WP — sizga ana shu ko‘rinish va hissiyotni berish uchun ishlab chiqilgan admin theme bo‘lib, mijozlaringizga yanada moslashtirilgan Dashboard taqdim etishingizga yordam beradi. Material WP shuningdek Ultimate Multisite’ning premium pluginlar yadrosiga ham kiradi.

Uni **o‘rnatganingizdan** so‘ng, _Plaginlar_ sahifangizga o‘ting va uni Network’ingizda faollashtirish uchun bosing.

Endi admin interfeysingiz butunlay qayta dizayn qilindi! Chap ustunda Material WP deb nomlangan yangi menyu ham mavjud
