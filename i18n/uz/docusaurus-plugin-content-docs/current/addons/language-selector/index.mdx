# Til tanlagich

Multisite Ultimate: Ro‘yxatdan o‘tishda Til tanlagich

Ro‘yxatdan o‘tish jarayoniga til tanlash maydonini qo‘shing.

## O‘rnatish {#installation}

1. 'ultimate-multisite-language-selector' ni '/wp-content/plugins/' katalogiga yuklang
2. WordPress Network Admin ichidagi 'Plaginlar' menyusi orqali plaginni faollashtiring

## Foydalanish qo‘llanmasi {#usage-guide}

Ushbu bepul qo‘shimcha yordamida foydalanuvchilaringiz ro‘yxatdan o‘tish jarayonida va kirish sahifasida qaysi tilni ko‘rishni xohlashlarini tanlashiga imkon berishingiz mumkin.

[Litsenziyangizni faollashtirgandan](../../user-guide/getting-started/installing-ultimate-multisite.md) so‘ng, litsenziyangiz sizga kirish huquqini beradigan Ultimate Multisite qo‘shimchalarini o‘rnatishingiz mumkin bo‘ladi.

Ultimate Multisite sozlamalari sahifasiga o‘ting, so‘ng chapdagi menyuda _Qo‘shimchalar_ yorlig‘ini bosing.

![Qo‘shimchalar sahifasi](/img/admin/addons-page.png)

_Language Selector o‘rnatishni boshlash uchun Qo‘shimchalar sahifasiga o‘ting_

U yerda biz taklif qiladigan barcha qo‘shimchalarni topasiz.

Bepul qo‘shimchalar obuna rejasidan qat’i nazar, barcha mijozlarimiz uchun mavjud.

Qo‘shimcha eskizi ostidagi _Hozir o‘rnating!_ tugmasini bosing. Keyin uni Plaginlar sahifasida faollashtiring.

![O‘rnatish tugmasi bilan Qo‘shimchalar sahifasi](/img/admin/addons-page.png)

_Qo‘shimchani o‘rnatish uchun tugmani bosing_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_Uni Plaginlar sahifangizda faollashtiring_

Endi Ultimate Multisite sozlamalari menyusiga o‘ting va _Tarmoq sozlamalari_ varaqasiga kiring. U yerda, sahifaning pastki qismida _Tillar_ deb nomlangan yangi bo‘limni topasiz.

![Sozlamalar sahifasi](/img/config/settings-general.png)

_Butunlay yangi bo‘lim paydo bo‘ladi!_

Barcha tillarni variant sifatida ko‘rsatish uchun uni bo‘sh qoldirishingiz yoki qaysilarini mavjud qilishni xohlashingizni tanlashingiz mumkin. Tillar WordPress sozlamangizda o‘rnatilgan tillardir.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_Ushbu WordPress o‘rnatilishida mavjud tillar ro‘yxati_

Kiritgan o‘zgarishlaringizni saqlaganingizdan so‘ng, siz tanlagan tillar bilan tanlash menyusi Kirish sahifasida ham, Ro‘yxatdan o‘tish jarayonida ham ko‘rsatiladi.

![Kirish va ro‘yxatdan o‘tish sozlamalari](/img/config/settings-login-registration.png)

_Kirish sahifasidagi Til tanlagich_

![Ro‘yxatdan o‘tish sahifasi](/img/config/registration-page.png)

_Ro‘yxatdan o‘tish jarayonidagi Til tanlagich_

## URL orqali tilni sozlash {#setting-the-language-via-the-url}

Shuningdek, URL ga locale parametrini yuborish orqali sahifa tilini o‘zgartirishingiz mumkin. Masalan, italyan tilidagi ro‘yxatdan o‘tish sahifasiga yo‘naltirmoqchi bo‘lsangiz, URL ga ?locale=it_IT qo‘shing.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Buning ishlashi uchun til WordPress tarmog‘ingizda o‘rnatilgan bo‘lishi kerak.
