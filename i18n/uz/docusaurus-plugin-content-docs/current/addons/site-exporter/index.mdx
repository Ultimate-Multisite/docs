# Sayt eksportchisi

## Mazmun

Mijozingiz o‘z veb-saytini ko‘chirishga qaror qilsa, tarmog‘ingiz ichidagi subsaytlarni yakka sayt o‘rnatmasiga va aksincha ko‘chirish imkoniyati zarurligini tushunamiz. Sayt eksportchisi va importchisi qo‘shimchasi bu vazifani osonlashtiradi. Bu vosita saytlarni zip fayllar yordamida eksport va import qiladi, ular zaxira nusxa sifatida ham xizmat qilishi mumkin.

## Xususiyatlar

* Saytlarni eksport va import qilish

* Foydalanish oson interfeys

* Sayt zaxira nusxasini yaratish

## Hujjatlar
**Saytni qanday eksport va import qilish**

## O‘rnatish

1. **ultimate-multisite-site-exporter** ni **/wp-content/plugins/** katalogiga yuklang
2. WordPress dagi **Pluginlar** menyusi orqali pluginni faollashtiring
3. Pluginni sozlash uchun bosqichma-bosqich Wizard ko‘rsatmalariga amal qiling

## Foydalanish qo‘llanmasi

Ultimate Multisite yordamida siz tarmog‘ingizdan saytni eksport qilishingiz yoki saytingizni multisite yoki yakka sayt o‘rnatmasiga import qilishingiz mumkin.

Jarayonni boshlash uchun add-onlar sahifamizga o‘ting va Ultimate Multisite: Sayt eksportchisi va importchisi add-onini o‘rnating.![Add-onlar sahifasi](/img/admin/addons-page.png)

![O‘rnatish opsiyasi bor add-onlar sahifasi](/img/admin/addons-page.png)

Plugins sahifangizga o‘ting va hozirgina o‘rnatgan add-onni **Tarmoq bo‘yicha faollashtirish** uchun bosing.<!-- Skrinshot mavjud emas: WP Plugins sahifasida Site Exporter add-oni uchun Network Activate tugmasi ko‘rsatilgan -->

**Eksport va import** opsiyasini Ultimate Multisite menyusida, chap tomonda topish mumkin.

<!-- Skrinshot mavjud emas: Ultimate Multisite menyusida Export and Import opsiyasi ko‘rsatilgan -->

## Saytni qanday eksport qilish

Tarmog‘ingizdan saytni eksport qilish uchun Ultimate Multisite menyusidagi **Eksport va import** bo‘limiga o‘ting va **Saytlarni eksport qilish** varag‘ida **Yangi sayt eksportini yaratish** tugmasini bosing.

<!-- Skrinshot mavjud emas: Export Sites varag‘ida Generate new Site Export tugmasi ko‘rsatilgan -->

Eksport qilmoqchi bo‘lgan saytingizni qidiring va eksport qilinadigan faylga kiritmoqchi bo‘lgan elementlarni tanlang. Siz theme, pluginlar va yuklamalarni kiritishingiz mumkin. Agar saytingiz juda katta bo‘lsa, eksport jarayonini fonda ishga tushirishni tanlashingiz mumkin.<!-- Skrinshot mavjud emas: Sayt eksport opsiyalarida sayt qidiruvi, theme/pluginlar/yuklamalarni kiritish belgilash kataklari ko‘rsatilgan -->

**Saytni eksport qilish** tugmasini bosing. Eksport qilingan saytingiz **Mavjud eksportlar** ro‘yxatida mavjud bo‘ladi. Faylni yuklab olish uchun **ZIP yuklab olish** tugmasini bosing. Eksport ZIP fayllari o‘zi ishga tushuvchi `index.php` va `readme.txt` ni o‘z ichiga oladi, shuning uchun ularni alohida importer pluginni avval o‘rnatmasdan yangi hostga o‘rnatish mumkin.<!-- Skrinshot mavjud emas: Existing Exports ro‘yxatida Download ZIP tugmasi ko‘rsatilgan -->

## Butun tarmoqni qanday eksport qilish

Barcha subsaytlarni birga eksport qilish uchun Ultimate Multisite menyusidagi **Eksport va import** bo‘limiga o‘ting va eksport ekranidagi **Tarmoq eksporti** opsiyasidan foydalaning. Yaratilgan arxiv host migratsiyalari, stagingni qayta qurishlar yoki tarmoq darajasidagi zaxira nusxalar uchun barcha subsaytlarni bitta ZIP ichida saqlaydi.

Tarmoq eksportlari katta bo‘lishi mumkin. Ularni trafik past bo‘lgan paytda ishga tushiring va serverda arxiv hamda kiritilgan har qanday yuklamalar, pluginlar va themelar uchun yetarli saqlash joyi borligini tasdiqlang.

## Saytni qanday import qilish

Saytni import qilish uchun **Saytni import qilish** varag‘iga o‘ting va import qilmoqchi bo‘lgan saytingizning .zip faylini yuklang. URLni yangisiga o‘zgartirishingiz yoki joyni bo‘sh qoldirib eski URLni saqlashingiz mumkin. Shuningdek, importdan keyin .zip faylini o‘rnatmangizdan o‘chirish uchun katakchani belgilashingiz mumkin. Keyin **Importni boshlash** tugmasini bosing.

<!-- Skrinshot mavjud emas: Import a Site varag‘ida zip yuklash maydoni va Begin Import tugmasi ko‘rsatilgan -->

Jarayon biroz vaqt olishi mumkin. Importer ostidagi ro‘yxatda kutilayotgan importni ko‘rasiz.

<!-- Skrinshot mavjud emas: Importer ostidagi ro‘yxatda kutilayotgan import -->

**Importni bekor qilish** tugmasini bosib jarayonni to‘xtatishingiz mumkin.

<!-- Skrinshot mavjud emas: Import jarayonini to‘xtatish uchun Cancel Import tugmasi -->

Jarayon tugagandan so‘ng, saytingiz tarmog‘ingizda allaqachon mavjud bo‘ladi.

## Yakka o‘rnatmada saytni qanday import qilish

Saytni yakka o‘rnatmangizga import qilish uchun avval saytingizda importerni o‘rnating va faollashtiring. Uni yuklab olish havolasini exporter varag‘ida topishingiz mumkin.

<!-- Skrinshot mavjud emas: Exporter varag‘ida yakka sayt importerni yuklab olish havolasi ko‘rsatilgan -->

Importerni chap tomondagi **Vositalar** menyusida, **Eksport va import** quyi bandida topishingiz mumkin.

<!-- Skrinshot mavjud emas: Yakka sayt Tools menyusida Export and Import quyi menyusi ko‘rsatilgan -->

Import qilmoqchi bo‘lgan saytingizning .zip faylini yuklang. URLni yangisiga o‘zgartirishingiz yoki joyni bo‘sh qoldirib eski URLni saqlashingiz mumkin. Shuningdek, importdan keyin .zip faylini o‘rnatmangizdan o‘chirish uchun katakchani belgilashingiz mumkin. Keyin **Importni boshlash** tugmasini bosing.

<!-- Skrinshot mavjud emas: Yakka sayt import shaklida zip yuklash va Begin Import tugmasi -->

Jarayon biroz vaqt olishi mumkin. Importer ostidagi ro‘yxatda kutilayotgan importni ko‘rasiz.

<!-- Skrinshot mavjud emas: Ro‘yxatda yakka sayt bo‘yicha kutilayotgan import -->

**Importni bekor qilish** tugmasini bosib jarayonni to‘xtatishingiz mumkin.

<!-- Skrinshot mavjud emas: Import jarayonini to‘xtatish uchun Cancel Import tugmasi -->

Jarayon tugagandan so‘ng, saytingiz hozirgina import qilgan saytingiz bilan almashtiriladi.
