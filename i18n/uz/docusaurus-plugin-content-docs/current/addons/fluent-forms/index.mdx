# Fluent Forms

Bu kengaytma Multisite Ultimate’da mahsulotlar, a’zoliklar yoki saytlarda Fluent Forms uchun cheklovlar qo‘shish imkonini beradi.
Fluent Forms custom post types o‘rniga maxsus jadvallardan foydalanadi, shuning uchun odatiy cheklash opsiyalari Fluent Forms bilan ishlamaydi.

## Xususiyatlar {#features}

- Fluent Forms yoki Conversational Fluent Forms uchun cheklovlar qo‘shish
- Mijozlar limitga yetganida, ularga planini yangilashni taklif qiluvchi tushunarli xabar ko‘rsatiladi
