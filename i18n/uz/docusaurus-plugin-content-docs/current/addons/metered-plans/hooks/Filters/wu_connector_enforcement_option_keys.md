# Filter: wu_connector_enforcement_option_keys {#filter-wuconnectorenforcementoptionkeys}

Asosiy saytdan majburiy qo‘llanadigan AI provayder option keylari ro‘yxatini filterlaydi.

Maxsus yoki uchinchi tomon AI provayder pluginlari uchun option keylarini qo‘shing, shunda ularning sozlamalari ham quyi saytlarda asosiy saytdan meros qilib olinadi.

|------|------|-------------|
| $keys | `string[]` | Majburiy qo‘llanadigan option key nomlari (allaqachon dinamik aniqlangan connector keylari va EXTRA_PROVIDER_OPTIONSni o‘z ichiga oladi). |
