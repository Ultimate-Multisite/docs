# Superdav AI Agent

Superdav AI Agent — Gratis AI Agent imkoniyatlarini mavzuga mos ko‘nikmalar, kengaytirilgan internet qidiruvi imkoniyatlari, sayt quruvchi funksiyalari va ixtiyoriy avtomatlashtirish integratsiyalari bilan kengaytiradigan ilg‘or WordPress AI yordamchisi. 1.18.0 versiyasi Google Calendar o‘qish vositalari, kontakt va ishtirokchilarni moslashtirish, inson tasdig‘i darvozalari, eslatmalar deduplikatsiyasi, TextBee SMS bildirishnomalari, boshqariladigan Superdav xizmatini ta’minlash va ishonchli dasturchi ish jarayonlari uchun alohida tarqatiladigan Advanced hamroh paketini qo‘shadi.

## Asosiy imkoniyatlar {#key-features}

- **Mavzuga mos ko‘nikmalar** — faol WordPress mavzusi va o‘rnatilgan plaginlaringiz asosida yo‘l-yo‘riq va imkoniyatlarni avtomatik moslashtiradi
- **Bir nechta internet qidiruv provayderlari** — boyroq qidiruv natijalari va ilg‘or tadqiqot imkoniyatlari uchun Brave Search va Tavily orasidan tanlang
- **Sayt quruvchi kontakt formalari** — o‘rnatilgan forma plaginingizdan foydalanib, to‘liq ishlaydigan kontakt formalarini bevosita chat interfeysidan yarating
- **Kengaytirilgan sayt quruvchi** — plaginlarni yaxshiroq aniqlash va xatolardan tiklanish bilan yaxshilangan ko‘p bosqichli sayt qurish rejalari
- **Ichki mavzu ko‘nikmalari** — Block Themes, Classic Themes, Kadence Blocks va Kadence Theme uchun ixtisoslashgan yo‘l-yo‘riq
- **Yaxshilangan WooCommerce integratsiyasi** — ishonchlilik va moslikni oshirish uchun native WooCommerce APIlaridan foydalanishga qayta ishlangan
- **Avtomatik provayder aniqlash** — mavjud plaginlar va mavzular faollashtirilganda yoki o‘chirilganda ularni avtomatik aniqlaydi va yangilaydi
- **Maxsus logo yaratish** — avtomatik namespace-xavfsiz sanitizatsiya bilan maxsus SVG logolarni yarating va joylashtiring
- **Palitra kontrastini tekshirish** — mavzuga qo‘llashdan oldin rang juftliklarini WCAG kirish imkoniyati talablariga mosligini tekshiring
- **Fotosurat bilan boyitilgan aniqlash** — dizayn yo‘nalishiga ta’sir qilish uchun Theme Builder aniqlash suhbati davomida fotosuratlar yuklang
- **Mehmondo‘stlik menyulari** — restoranlar, kafelar va mehmondo‘stlik bizneslari uchun tuzilmali taom va ichimlik menyusi sahifalarini yarating
- **Desktop va mobil oldindan ko‘rishlar** — dizayn yo‘nalishini tanlash davomida dizayningizni desktop va mobil qurilmalarda oldindan ko‘ring
- **1-darajali sayt scraping** — raqobatchi va ilhom beruvchi veb-saytlarni tahlil qilish uchun sd-ai-agent/site-scrape endi sukut bo‘yicha mavjud
- **Google Calendar o‘qish vositalari** — avtomatlashtirishlar yaqinlashayotgan jadval kontekstini tahlil qila olishi uchun sozlangan kalendarlar va tadbirlarni o‘qiydi
- **Kontakt va ishtirokchilarni moslashtirish** — keyingi ishlarni yaratishdan oldin kalendar ishtirokchilarini ma’lum sayt foydalanuvchilari va kontaktlar bilan moslashtiradi
- **Inson tasdig‘i darvozalari** — xabarlar yuborish, ishni davom ettirish yoki o‘zgarishlar kiritishdan oldin sezgir avtomatlashtirishlarni ko‘rib chiqish uchun pauza qiladi
- **Eslatma yozuvlari va deduplikatsiya** — takroriy avtomatlashtirish ishga tushirishlari bir xil oluvchiga ikki marta bildirishnoma yubormasligi uchun allaqachon yuborilgan eslatmalarni kuzatadi
- **TextBee SMS bildirishnomalari** — SMS provayderi yoqilganida TextBee orqali sozlangan matnli xabar bildirishnomalarini yuboradi
- **Boshqariladigan Superdav xizmatini ta’minlash** — qo‘lda endpoint sozlash o‘rniga, qo‘llab-quvvatlanadigan saytlarni avtomatik sozlash bilan hosted Superdav endpointlariga ulaydi
- **Advanced hamroh paketi** — fayl tizimi, ma’lumotlar bazasi, WP-CLI, REST dispatcher, plagin builder, git snapshot, foydalanuvchi boshqaruvi va benchmark ish jarayonlari uchun alohida tarqatiladigan ishonchli dasturchi vositalari

## Talablar {#requirements}

- WordPress 6.0 yoki undan yuqori
- PHP 8.1 yoki undan yuqori
- WordPress Multisite (tarmoq bo‘yicha faollashtirilgan) yoki bitta saytli o‘rnatish
- Faol AI provayder ulovchi plagini (Anthropic, OpenAI yoki mos keluvchi)
- Gratis AI Agent v1.9.0 yoki undan yuqori (Superdav AI Agent Gratis AI Agent imkoniyatlarini kengaytiradi)

## O‘rnatish {#installation}

1. Plagin fayllarini `/wp-content/plugins/superdav-ai-agent` manziliga yuklang
2. Plaginni WordPress ichidagi **Plaginlar** ekrani orqali faollashtiring
3. Gratis AI Agent ham faollashtirilganiga ishonch hosil qiling
4. **Gratis AI Agent → Sozlamalar** bo‘limiga o‘ting va kerak bo‘lsa internet qidiruv provayderingiz, boshqariladigan Superdav xizmati ulanishi, Google Calendar yoki TextBee credentials ma’lumotlarini sozlang

WordPress.org paketi asosiy Superdav AI Agent imkoniyatlarini o‘z ichiga oladi. Advanced hamroh paketi ishonchli dasturchi vositalari aniq o‘rnatilgan va ruxsat berilgan muhitlar uchun alohida quriladi va tarqatiladi.

## Mavzuga mos ko‘nikmalar {#theme-aware-skills}

Superdav AI Agent faol mavzuingiz va o‘rnatilgan plaginlaringizni avtomatik aniqlaydi hamda mos mavzuga mos ko‘nikmani yordamchi kontekstiga yuklaydi. Mavjud ko‘nikmalar quyidagilarni o‘z ichiga oladi:

| Ko‘nikma | Nimalarga qo‘llanadi |
|---|---|
| **Block Themes** | WordPress blok-asosli (Full Site Editing) arxitekturasidan foydalanadigan mavzular |
| **Classic Themes** | PHP shablonlari va klassik editordan foydalanadigan an’anaviy WordPress mavzulari |
| **Kadence Blocks** | Kadence Blocks plagini faol bo‘lgan saytlar |
| **Kadence Theme** | Faol mavzu sifatida Kadence theme ishlatilayotgan saytlar |

Har bir ko‘nikma bo‘yicha batafsil yo‘l-yo‘riq uchun [Mavzuga mos ko‘nikmalar](../../user-guide/configuration/theme-aware-skills) sahifasiga qarang.

## Internet qidiruv provayderlari {#internet-search-providers}

Superdav AI Agent v1.10.0 internet qidiruv so‘rovlari uchun Brave Search muqobili sifatida **Tavily**ni qo‘shadi. Har ikkala provayder real vaqt qidiruv natijalarini taqdim etadi va **Gratis AI Agent → Sozlamalar → Kengaytirilgan** bo‘limida sozlanishi mumkin.

| Provayder | Xususiyatlar | Sozlash |
|---|---|---|
| **Brave Search** | Tezkor, maxfiylikka yo‘naltirilgan qidiruv natijalari | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Boyroq qidiruv natijalari, ilg‘or tadqiqot imkoniyatlari | [Tavily API](https://tavily.com/) |

Sozlash bo‘yicha ko‘rsatmalar uchun [Internet qidiruvi](../../user-guide/configuration/internet-search) bo‘limiga qarang.

## Taqvim, tasdiqlash va bildirishnoma ish jarayonlari {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 avtomatlashtirish ish jarayonlarini kontent va sayt yaratish vazifalaridan tashqariga kengaytiradi:

- **Google Calendar vositalari** taqvim hisob ma’lumotlari mavjud bo‘lganda agentga sozlangan taqvimlar va tadbir tafsilotlarini o‘qish imkonini beradi.
- **Kontakt va ishtirokchi yordamchilari** eslatmalar yoki keyingi vazifalarni tayyorlashdan oldin tadbir ishtirokchilarini WordPress foydalanuvchilari yoki ma’lum kontaktlar bilan moslaydi.
- **Inson tasdiqlash darvozalari** vakolatli foydalanuvchi taklif qilingan amalni ko‘rib chiqmaguncha avtomatlashtirishni pauza qilish imkonini beradi.
- **Eslatma yozuvlari** avtomatlashtirish qayta urinilganda yoki takroran rejalashtirilganda takroriy taqvim yoki bildirishnoma eslatmalarining oldini oladi.
- **TextBee SMS qo‘llab-quvvatlashi** TextBee provayderi sozlanganda matnli xabar bildirishnomalarini yoqadi.

Bu vositalardan jadvaldan xabardor yordam, ishtirokchilarni kuzatish va nozik xabarlar yuborilishidan oldin baribir inson ko‘rigini talab qiladigan eslatma ish jarayonlari uchun foydalaning.

## Sozlash {#configuration}

Barcha sozlamalar WordPress adminidagi **Gratis AI Agent → Settings** ostida joylashgan. Superdav AI Agent quyidagi sozlash variantlarini qo‘shadi:

| Sozlama | Tavsif |
|---|---|
| **Internet Search Provider** | Brave Search va Tavily orasidan tanlang (ixtiyoriy) |
| **Tavily API Key** | Tavily internet qidiruvi uchun API kaliti (agar Tavily tanlangan bo‘lsa) |
| **Managed Superdav Service** | Qo‘llab-quvvatlanadigan saytlarni avtomatik endpoint ta’minoti bilan hosted Superdav xizmatiga ulang |
| **Google Calendar Credentials** | Agent taqvimlar va tadbirlarni o‘qishi uchun taqvimga kirishni sozlang |
| **TextBee SMS Provider** | Tasdiqlangan SMS bildirishnoma ish jarayonlari uchun TextBee hisob ma’lumotlarini sozlang |

## Ilg‘or hamroh paket {#advanced-companion-package}

Superdav AI Agent v1.18.0 ishonchli dasturchi ish jarayonlari uchun alohida tarqatiladigan Advanced hamroh paketini qo‘shadi. Bu imkoniyatlar WordPress.org asosiy paketiga kiritilmagan. Hamroh paketni faqat administratorlar agentning dasturchi vositalariga kirishiga aniq ishonadigan saytlargagina o‘rnating.

Ilg‘or vositalar fayl tizimi amallari, ma’lumotlar bazasiga kirish, WP-CLI, REST dispatcher ish jarayonlari, plugin builder utilitalari, git snapshotlari, foydalanuvchi boshqaruvi yordamchilari va benchmark vositalarini o‘z ichiga oladi. Bu vositalarni ishonchli administratorlar bilan cheklang va ularni production saytlarda yoqishdan oldin approval-gate sozlamasini ko‘rib chiqing.
