# Multi-Network o‘zgarishlar jurnali

### 1.3.0
* Yangi: Network shablonini oldindan ko‘rish vositasi — xarid qilishdan oldin jonli panel orqali network shablonlarini ko‘ring va oldindan ko‘rib chiqing.
* Yangi: Checkout paytida network shablonini tanlash vositasi — mijozlar ro‘yxatdan o‘tish checkout jarayonida network shablonini tanlashi mumkin.
* Yangi: Networklararo magic link SSO — mijozlar magic-link autentifikatsiyasi orqali sub-networklarga muammosiz kira oladi.
* Yangi: Network media klonlash — shablondan networklarni klonlashda site medialari to‘g‘ri nusxalanadi.
* Tuzatish: Network klonlash endi multi-tenancy addonisiz ishlaydi; klondan keyin super admin imtiyozlari qayta beriladi.
* Tuzatish: Network yaratish endi WooCommerce va barcha to‘lov gatewaylari bilan mos.
* Tuzatish: Port raqami endi fallback domain yaratishga kiritiladi.
* Tuzatish: Template picker CSS barcha themelarda to‘g‘ri render qilish uchun navbatga qo‘yilgan stylesheetga ko‘chirildi.
* Tuzatish: Site query scope ichida aniqlanmagan `network_id` keyidan himoya qiladi.
* Tuzatish: Context switchingdan oldingi network ruxsatini tekshirish `wpdb` jadval property xatolarining oldini oladi.
