# PayFast

Multisite Ultimate: Payfast shlyuzi

Janubiy Afrikadagi tezkor va xavfsiz onlayn to‘lovlar shlyuzi bo‘lgan Payfast’ni Multisite Ultimate to‘lov varianti sifatida qo‘shadi.

## O‘rnatish

1. 'ultimate-multisite-payfast'ni '/wp-content/plugins/' katalogiga yuklang
2. Plugin’ni WordPress’dagi 'Plugins' menyusi orqali faollashtiring
3. Plugin’ni sozlash uchun bosqichma-bosqich Wizard’ga amal qiling

## Foydalanish qo‘llanmasi

PayFast Janubiy Afrikadagi bizneslar uchun yetakchi onlayn to‘lov protsessoridir. U onlayn savdogarlarga tranzaksiyalarni silliq va xavfsiz qilish orqali o‘sishga yordam berishga qaratilgan. Ultimate Multisite foydalanuvchisining so‘roviga ko‘ra, biz PayFast bilan integratsiyani mumkin qiladigan addon yaratishga qaror qildik. Ushbu maqola Ultimate Multisite tarmog‘ingizda to‘lov qabul qilish uchun PayFast’ni to‘lov shlyuzi sifatida qanday sozlashni ko‘rsatadi.

Birinchi qadam — add on’ni o‘rnatish.

# Payfast add-on’ini qanday o‘rnatish kerak:

Add-ons sahifasiga o‘ting va Ultimate Multisite: Payfast Gateway add-on’ini tanlang.

![Add-ons sahifasi](/img/admin/addons-page.png)

Add-on oynasi ochiladi. Install Now tugmasini bosing.

![O‘rnatish varianti bor Add-ons sahifasi](/img/admin/addons-page.png)

O‘rnatish tugagach, siz plugins sahifasiga yo‘naltirilasiz. Bu yerda shunchaki Network Activate tugmasini bosing va _Payfast_ add-on’i tarmog‘ingizda faollashtiriladi.

# PayFast’ni sozlash

Add-on’ni o‘rnatgandan so‘ng, endi Ultimate Multisite sozlamalariga o‘tib, PayFast’ni sozlashni boshlashingiz mumkin.

**Ultimate Multisite settings** sahifasida "Payments" yorlig‘ini topishingiz mumkin. Ushbu yorliq ostida ushbu to‘lov shlyuzini yoqish yoki o‘chirish uchun toggle tugmasi bilan birga "**Payfast** "ni ko‘rasiz.

![To‘lov shlyuzi sozlamalari](/img/config/settings-payment-gateways.png)Davom etib, uni yoqish uchun toggle tugmasini bosing. Shundan so‘ng u qutini kengaytiradi va to‘ldirishingiz kerak bo‘lgan turli maydonlarni ko‘rsatadi, masalan **Mechant ID, Merchant Key va PassPhrase.** Bu ma’lumotni **Payfast** Account’ingizdan olishingiz mumkin

<!-- Skrinshot mavjud emas: PayFast kengaytirilgan sozlamalarida Merchant ID, Merchant Key va PassPhrase maydonlari ko‘rsatilgan -->

Keyingi qadam — kerakli ma’lumotni olish uchun PayFast Account’ingizga login qilish.

Login qilganingizdan so‘ng, kerakli hisob ma’lumotlarini darhol **Payfast** Account dashboard’ingizda ko‘rishingiz mumkin.

<!-- Skrinshot mavjud emas: PayFast Account dashboard’ida Merchant ID va Merchant Key hisob ma’lumotlari ko‘rsatilgan -->Siz hisob ma’lumotlarini shunchaki nusxalab qo‘yishingiz va Ultimate Multisite sozlamalaringizdagi tafsilotlarni to‘ldirishingiz mumkin. Keyin o‘zgarishlarni saqlash uchun **"Save settings** " tugmasini bosing.

<!-- Skrinshot mavjud emas: hisob ma’lumotlari to‘ldirilgan Ultimate Multisite PayFast sozlamalari va Save tugmasi -->

Bo‘ldi, PayFast to‘lov integratsiyangiz yakunlandi!
