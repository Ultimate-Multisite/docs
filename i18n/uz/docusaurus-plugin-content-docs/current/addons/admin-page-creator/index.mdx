# Admin Page Creator

## Kontent {#content}
Admin Page Creator WordPress admin dashboard ichida yangi admin sahifalar va quyi sahifalarni oson qo‘shish imkonini beradi. Undan yangi foydalanuvchilar uchun boshlang‘ich sahifa vazifasini bajaradigan xush kelibsiz sahifasi yoki qo‘llanma sahifasini yaratish uchun foydalanish mumkin. Sahifalarni qiynalmasdan loyihalash uchun Beaver Builder, Elementor, Brizy va Oxygen kabi sahifa yaratuvchilardan foydalanishingiz mumkin.

U shuningdek maxsus HTML, CSS va JavaScript qo‘llab-quvvatlashini, shuningdek plan va foydalanuvchi roli asosidagi kirish nazoratini taklif qiladi!

## Xususiyatlar {#features}
* Maxsus HTML va CSS qo‘llab-quvvatlash
* JavaScript qo‘llab-quvvatlash
* PHP kontent rejimi
* Foydalanuvchi roliga asoslangan ruxsat
* Sahifa yaratuvchilarni qo‘llab-quvvatlash

## O‘rnatish {#installation}

1. **ultimate-multisite-admin-page-creator** ni **/wp-content/plugins/** katalogiga yuklang
2. WordPress Network Admin’ingizdagi **Pluginlar** menyusi orqali pluginni faollashtiring

## Foydalanish qo‘llanmasi {#usage-guide}

WordPress saytlarning old tomoniga kontent qo‘shishni juda oson qilsa-da, admin tomonida sahifalar va quyi sahifalar yaratish ba’zi asosiy kod yozish ko‘nikmalarini talab qiladi. Ultimate Multisite: Admin Page Creator tarmoq saytlaringizning orqa tomoniga old tomondagi kabi foydalanish qulayligini olib kirishni maqsad qiladi.

## Nega Admin Page Creator’dan foydalanish kerak? {#why-use-admin-page-creator}

Ehtimol, mijozlaringizning admin panelida qo‘llanmalar bilan maxsus sahifa taklif qilmoqchidirsiz, yoki FAQ va aloqa vidjeti bilan maxsus sahifa qo‘shmoqchidirsiz. Admin Page Creator bilan imkoniyatlar cheksiz va bugungi qo‘llanmada aynan shuni ko‘rib chiqamiz.

## Birinchi maxsus admin sahifangizni yaratish {#creating-your-first-custom-admin-page}

Ultimate Multisite: Admin Page Creator’ni o‘rnatib faollashtirgandan so‘ng, tarmoq admin menyusiga o‘ting va Admin Pages bo‘limiga kiring.

![Network Admin’dagi maxsus admin sahifalar ro‘yxati](/img/admin/admin-pages-list.png)

_Network Admin’dagi maxsus admin sahifalar ro‘yxati_

Hozirgacha yaratgan barcha admin sahifalaringiz ro‘yxatini ko‘rasiz (bu birinchi tashrifingiz bo‘lgani uchun ro‘yxat bo‘sh bo‘ladi). Davom eting va **Yangi admin sahifa qo‘shish** tugmasini bosing. Siz Admin Page tahrirlagichiga yo‘naltirilasiz.

## Menyu opsiyalari {#menu-options}

Yon tomondagi General Options meta-qutisida menyu turi, yorlig‘i va yana ko‘p narsalarni sozlashingiz mumkin bo‘ladi.

### Menyu turlari {#menu-types}

Yangi maxsus admin sahifangiz ikki xil menyu turiga ega bo‘lishi mumkin: yuqori darajali menyu va quyi menyu.

Yuqori darajali menyu sahifasi, nomidan ko‘rinib turibdiki, yangi yaratilgan sahifangizni standart WordPress sahifalari bilan birga admin menyuga qo‘shadi.

Agar yuqori darajali opsiyani tanlasangiz, menyu tartibini va menyu ikonkasini ham tanlashingiz mumkin bo‘ladi.

Agar bu sahifani quyi menyu sahifasi qilmoqchi bo‘lsangiz, ushbu sahifani biriktirish uchun ota sahifani tanlashingiz mumkin bo‘ladi. Maxsus sahifangizni standart WordPress sahifalariga yoki o‘zingizning yuqori darajali maxsus admin sahifalaringizga qo‘shishingiz mumkin.

<!-- Screenshot unavailable: Admin Page Creator menu type options with label, order, and icon settings -->

_Ushbu maxsus sahifa uchun menyu elementini to‘liq moslashtiring_

<!-- Screenshot unavailable: Admin Page Creator sub-menu parent page selection -->

_Bu sahifani boshqa menyu sahifalarining, jumladan o‘zingizning yuqori darajali maxsus sahifalaringizning quyi menyusi sifatida ham qo‘shishingiz mumkin._

## Kengaytirilgan opsiyalar {#advanced-options}

Maxsus admin sahifalaringizga turli xil kontent qo‘sha olishingiz kerak. Bunga CSS qoidalari va fayllari, JavaScript hamda tashqi kutubxonalar ham kiradi. Buni Advanced Options tabidan foydalanib qilishingiz mumkin.

### Maxsus CSS {#custom-css}

CSS tabida o‘zingizning maxsus CSS kodingizni qo‘shishingiz, shuningdek masofaviy saytlardan (CDN’lar, Google Fonts va hokazo kabi) CSS fayllarini import qilishingiz mumkin.

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_Maxsus CSS ham qo‘llab-quvvatlanadi. Tashqi uslub fayllarini ham kiritishingiz mumkin!_

### Maxsus JavaScript {#custom-javascript}

Xuddi shu narsa JavaScript uchun ham amal qiladi. Maxsus JavaScript kodingizda foydalanish uchun tashqi kutubxonalarni qo‘shishingiz mumkin.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_O‘zingizning JavaScript kodingizni qo‘shing!_

## Ruxsatlar {#permissions}

Advanced Options ichida, shuningdek, qaysi quyi saytlar ushbu sahifaga kirish huquqiga ega bo‘lishini tanlashingiz mumkin bo‘ladi. Bu, masalan, turli plan darajalariga turli eksklyuziv kontent taqdim etishni rejalashtirayotgan bo‘lsangiz foydalidir.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Bu sahifa qayerda ko‘rinishini to‘liq nazorat qilasiz_

**Muhim** : Sahifa ko‘rsatilishi uchun ikkala shart ham bajarilishi kerak. Bu shuni anglatadiki, agar foydalanuvchi berilgan reja mijozi bo‘lsa, lekin tanlangan rollardan biriga ega bo‘lmasa (yoki aksincha), sahifa ularning panelida ko‘rinmaydi.

## Faol {#active}

Agar maxsus admin sahifani o‘chirmasdan yoki ruxsatlar bilan ovora bo‘lmasdan o‘chirib qo‘ymoqchi bo‘lsangiz, shunchaki Active opsiyasidan belgini olib tashlang.

Yuqori darajadagi sahifani nofaollashtirish ushbu ota sahifaga biriktirilgan barcha quyi sahifalarni avtomatik ravishda nofaollashtiradi.
