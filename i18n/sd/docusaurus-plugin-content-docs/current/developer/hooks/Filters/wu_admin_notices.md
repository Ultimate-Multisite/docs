# فلٽر: wu_admin_notices

ڊولپرن کي Ultimate Multisite طرفان شامل ڪيل admin notices فلٽر ڪرڻ جي اجازت ڏيو.

## پيرا ميٽر {#parameters}

| نالو | قسم | وضاحت |
|------|------|-------------|
| $notices | `array` | ان خاص پينل لاءِ notices جي فهرست. |
| $all_notices | `array` | شامل ڪيل notices جي فهرست، پينل موجب الڳ ڪيل. |
| $panel | `string` | notices حاصل ڪرڻ لاءِ پينل. |
| $filter | `string` | جيڪڏهن dismissable notices فلٽر ڪري ڪڍيا ويا آهن. |
| $dismissed_messages | `array` | dismissed notice keys جي فهرست. |

### کان وٺي {#since}

- 2.0.0
