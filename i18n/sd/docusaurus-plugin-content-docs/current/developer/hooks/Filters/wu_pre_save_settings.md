# فلٽر: wu_pre_save_settings

ڊولپرز کي اجازت ڏيو ته Ultimate Multisite پاران محفوظ ڪرڻ کان اڳ سيٽنگون فلٽر ڪن.

## پيرا ميٽر {#parameters}

| نالو | قسم | وضاحت |
|------|------|-------------|
| $settings | `array` | محفوظ ٿيڻ واريون سيٽنگون. |
| $settings_to_save | `array` | شامل ڪرڻ لاءِ نيون سيٽنگون. |
| $saved_settings | `array` | موجوده محفوظ ڪيل سيٽنگون. |

### کان وٺي {#since}

- 2.0.18
