# فلٽر: wu_country_get_administrative_division_name

ملڪ جي انتظامي ذيلي ورهاستن جو سهڻو نالو موٽائي ٿو.

## پيرا ميٽر

| نالو | قسم | وضاحت |
|------|------|-------------|
| $name | `string` | ورهاست جو نالو. عام طور تي state، province، region وغيره جهڙو ڪجهه. |
| $country_code | `string` | ملڪ لاءِ ٻن اکرن وارو ISO ڪوڊ. |
| $state_code | `string` | state لاءِ ٻن اکرن وارو ISO ڪوڊ. |
| $ucwords | `bool` | جيڪڏهن اسان لفظن کي وڏن اکرن ۾ ڪريون. |
| $current_country | `\WP_Ultimo\Country\Country` | موجوده ڪلاس جو انسٽنس. |
