# Filter: wu_get_site_domain_and_path

ڊولپرز کي domain/path جوڙن ۾ ڦيرڦار ڪرڻ جي اجازت ڏئي ٿو.

هي ڪيترين ئي شين لاءِ ڪارائتو ٿي سگهي ٿو، جهڙوڪ ڪنهن قسم جو staging حل لاڳو ڪرڻ، مختلف servers، وغيره.

## پيراميٽر

| نالو | قسم | وضاحت |
|------|------|-------------|
| $d | `object` | موجوده object جنهن ۾ domain ۽ path keys شامل آهن. |
| $path_or_subdomain | `string` | function ڏانهن موڪليل اصل path/subdomain. |
