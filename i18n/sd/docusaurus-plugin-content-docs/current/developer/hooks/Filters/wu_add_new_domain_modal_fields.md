# فلٽر: wu_add_new_domain_modal_fields

نئون ڊومين شامل ڪرڻ واري موڊل لاءِ فيلڊز کي فلٽر ڪري ٿو.

ايڊآنز (مثال طور Domain Seller) کي ڊومين رجسٽريشن فيلڊز ۾ تبديلي ڪرڻ يا انهن کي مٽائڻ جي اجازت ڏئي ٿو.

## پيرا ميٽر {#parameters}

| نالو | قسم | وضاحت |
|------|------|-------------|
| $fields | `array` | فارم جا فيلڊز. |
