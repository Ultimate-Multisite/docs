# فلٽر: wp_ultimo_render_vars

plugin ڊولپرن کي عالمي طور render context ۾ اضافي variable شامل ڪرڻ جي اجازت ڏئي ٿو.

## پيرا ميٽر

| نالو | قسم | وضاحت |
|------|------|-------------|
| $args | `array` | render call ذريعي پاس ڪيل variables تي مشتمل array. |
| $view | `string` | رينڊر ٿيڻ واري view جو نالو. |
| $default_view | `string` | fallback_view جو نالو |

### کان وٺي

- 2.0.0

[`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L31) ۾ line 31 تي بيان ڪيل
