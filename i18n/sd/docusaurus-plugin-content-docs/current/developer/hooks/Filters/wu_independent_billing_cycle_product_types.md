# فلٽر: wu_independent_billing_cycle_product_types

انهن پيداوار جي قسمن کي فلٽر ڪريو جن جا آزاد بلنگ چڪر آهن.

آزاد بلنگ چڪر رکندڙ پيداوارن کي ڪارٽ ۾ پلان جي بلنگ مدت سان ملائڻ لاءِ مجبور نه ڪيو ويندو آهي. اهو انهن پيداوارن لاءِ ڪارائتو آهي جهڙوڪ ڊومين رجسٽريشن، جيڪي هميشه سالياني بل ڪن ٿيون، ڀلي پلان ماهوار هجي يا ساليانو.

## پيرا ميٽر

| نالو | قسم | وضاحت |
|------|------|-------------|
| $types | `array` | آزاد بلنگ چڪر رکندڙ پيداوار جي قسمن جي Array. |
