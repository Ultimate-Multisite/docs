# سبق 8: گراهڪ جي شروعاتي رهنمائي {#lesson-8-customer-onboarding}

گراهڪ کي سائن اپ ڪرائڻ ڪم جو صرف اڌ حصو آهي. جيڪڏهن هو لاگ اِن ڪن، گهٻرائجي وڃن، ۽ ڪڏهن واپس نه اچن، ته توهان انهن کي وڃائي ڇڏيو. هي سبق اهو تجربو ترتيب ڏئي ٿو جيڪو نئين سائن اپ کي هڪ سرگرم، دلچسپي رکندڙ گراهڪ ۾ بدلائي ٿو.

## اسان ڪٿي ڇڏيو هو {#where-we-left-off}

FitSite مڪمل طور برانڊ ٿيل آهي ۽ ان ۾ ڪم ڪندڙ چيڪ آئوٽ وهڪرو موجود آهي. هاڻي اسان ان ڳالهه تي ڌيان ڏيون ٿا ته جڏهن هڪ فٽنيس اسٽوڊيو مالڪ سائن اپ مڪمل ڪري پهريون ڀيرو پنهنجي نئين سائيٽ تي پهچي ٿو، ته پوءِ ڇا ٿئي ٿو.

## شروعاتي رهنمائي ڇو اهم آهي {#why-onboarding-matters}

سائن اپ کان پوءِ پهريان 30 منٽ اهو طئي ڪن ٿا ته گراهڪ رهي ٿو يا ڇڏي وڃي ٿو. هڪ فٽنيس اسٽوڊيو مالڪ جيڪو:

- لاگ اِن ڪري ۽ اهڙي سائيٽ ڏسي جيڪا اڳ ئي فٽنيس ويب سائيٽ جهڙي لڳي → رهي ٿو
- بلڪل ڄاڻي ٿو ته اڳتي ڇا ڪرڻو آهي → رهي ٿو
- هڪ عام WordPress Dashboard ۾ پاڻ کي وڃايل محسوس ڪري → ڇڏي وڃي ٿو

توهان جا مخصوص شعبي وارا ٽيمپليٽس (سبق 4) پهرين نقطي کي سنڀالين ٿا. هي سبق ٻئي کي سنڀالي ٿو.

## پهرئين لاگ اِن جو تجربو {#the-first-login-experience}

### ڀليڪار Dashboard Widget {#welcome-dashboard-widget}

هڪ ڪسٽم dashboard widget ٺاهيو جيڪو نون گراهڪن کي ڀليڪار ڪري ۽ انهن کي سيٽ اپ ۾ رهنمائي ڪري. جڏهن اهي پهريون ڀيرو لاگ اِن ڪن، ته اهو نمايان طور ظاهر ٿيڻ گهرجي.

**FitSite تڪڙي شروعات:**

1. **پنهنجي اسٽوڊيو جو نالو ۽ لوگو شامل ڪريو** -- Customizer يا Site Identity سيٽنگز ڏانهن لنڪ
2. **پنهنجو ڪلاس شيڊول اپڊيٽ ڪريو** -- سڌو Classes صفحي جي ايڊيٽر ڏانهن لنڪ
3. **پنهنجا ٽرينر شامل ڪريو** -- Trainers صفحي جي ايڊيٽر ڏانهن لنڪ
4. **پنهنجا رابطا تفصيل سيٽ ڪريو** -- Contact صفحي جي ايڊيٽر ڏانهن لنڪ
5. **پنهنجي سائيٽ جو پريويو ڏسو** -- فرنٽ اينڊ ڏانهن لنڪ

هر قدم لاڳاپيل صفحي يا سيٽنگ ڏانهن سڌو لنڪ ڪري ٿو. مينيوز ۾ ڳولا ڪرڻ جي ضرورت ناهي.

### Dashboard کي سادو ڪريو {#simplify-the-dashboard}

نون گراهڪن کي هر WordPress مينيو آئٽم ڏسڻ جي ضرورت ناهي. غور ڪريو:

- اهي مينيو آئٽمز لڪائڻ جيڪي فٽنيس سائيٽ مينيجمينٽ سان لاڳاپيل ناهن (مثال طور، Comments جيڪڏهن استعمال نه ٿيندا هجن)
- مينيو کي ٻيهر ترتيب ڏيڻ ته جيئن سڀ کان وڌيڪ استعمال ٿيندڙ آئٽمز پهريان اچن
- اهڙا ڪسٽم مينيو ليبل شامل ڪرڻ جيڪي مخصوص شعبي لاءِ سمجهه ۾ اچن ("Appearance" جي بدران "Your Studio")

[Plugin & Theme Manager](/addons/plugin-and-theme-manager) addon گراهڪن کي ڇا ڏسڻ ۾ اچي، ان تي ڪنٽرول ڪرڻ ۾ مدد ڪري سگهي ٿو.

## ڀليڪار اي ميل سلسلو {#welcome-email-sequence}

هڪ واحد ڀليڪار اي ميل ڪافي ناهي. اهڙو سلسلو سيٽ ڪريو جيڪو گراهڪن کي سندن پهرين هفتي ۾ رهنمائي ڪري:

### اي ميل 1: ڀليڪار (سائن اپ کان فوراً پوءِ) {#email-1-welcome-immediately-after-signup}

- موضوع: "FitSite ۾ ڀليڪار -- توهان جي اسٽوڊيو ويب سائيٽ لائيو آهي"
- مواد: لاگ اِن لنڪ، تڪڙي شروعات جا قدم، مدد وسيلن ڏانهن لنڪ
- لهجو: پرجوش، همٿائيندڙ، فٽنيس-مخصوص

### اي ميل 2: تڪڙيون ڪاميابيون (ڏينهن 1) {#email-2-quick-wins-day-1}

- موضوع: "توهان جي FitSite تي پهريان ڪرڻ لاءِ 3 شيون"
- مواد: پنهنجو لوگو شامل ڪريو، هوم پيج هيرو تصوير اپڊيٽ ڪريو، پنهنجو ڪلاس شيڊول شامل ڪريو
- اسڪرين شاٽس شامل ڪريو جيڪي بلڪل ڏيکارين ته ڪٿي ڪلڪ ڪرڻو آهي

### اي ميل 3: ان کي پنهنجو بڻايو (ڏينهن 3) {#email-3-make-it-yours-day-3}

- موضوع: "پنهنجي فٽنيس سائيٽ کي نمايان بڻايو"
- مواد: رنگ ڪسٽمائيز ڪريو، ٽرينر تصويرون شامل ڪريو، پنهنجي اسٽوڊيو جي ڪهاڻي لکو
- پليٽ فارم تي بهترين فٽنيس سائيٽن جي مثالن ڏانهن لنڪ

### اي ميل 4: لائيو وڃو (ڏينهن 7) {#email-4-go-live-day-7}

- موضوع: "ڇا توهان پنهنجي FitSite دنيا سان شيئر ڪرڻ لاءِ تيار آهيو؟"
- مواد: شيئر ڪرڻ کان اڳ تصديق ڪرڻ وارين شين جي چيڪ لسٽ، ڪسٽم ڊومين ڪيئن ڳنڍجي (جيڪڏهن Growth/Pro تي هجي)، سماجي شيئرنگ جا ٽوٽڪا

:::tip اي ميل خودڪارڪاري
اهي اي ميلون پنهنجي اي ميل مارڪيٽنگ پليٽ فارم ذريعي ٽرگر ڪرڻ لاءِ [Webhooks](/user-guide/integrations/webhooks) يا [Zapier](/user-guide/integrations/zapier) استعمال ڪريو. هي توهان کي وقتبندي تي وڌيڪ ڪنٽرول ڏئي ٿو ۽ توهان کي مصروفيت ٽريڪ ڪرڻ جي اجازت ڏئي ٿو.
:::

## مدد جا وسيلا {#help-resources}

اهڙو مخصوص شعبي وارو مددگار مواد ٺاهيو جيڪو انهن سوالن جا جواب ڏئي جيڪي فٽنيس اسٽوڊيو مالڪ واقعي پڇن ٿا:

### ڄاڻ بنياد جا مضمون {#knowledge-base-articles}

- "پنهنجو ڪلاس شيڊول ڪيئن اپڊيٽ ڪجي"
- "ٽرينر پروفائيل شامل ڪرڻ ۽ ايڊٽ ڪرڻ"
- "پنهنجي اسٽوڊيو جو لوگو ۽ رنگ بدلائڻ"
- "پنهنجو ڊومين نالو ڳنڍڻ" (Growth/Pro گراهڪن لاءِ)
- "پنهنجي سائيٽ ۾ بڪنگ widget شامل ڪرڻ"

اهي غير-ٽيڪنيڪل استعمال ڪندڙن لاءِ لکو. اسڪرين شاٽس استعمال ڪريو. WordPress جرگن کان پاسو ڪريو.

### وڊيو واڪ ٿروز {#video-walkthroughs}

مختصر (2-3 منٽ) اسڪرين رڪارڊنگز ڏيکارينديون:

- پهريون لاگ اِن ۽ واقفيت
- هوم پيج ايڊٽ ڪرڻ
- ڪلاس شيڊول اپڊيٽ ڪرڻ
- نئون ٽرينر شامل ڪرڻ

انهن کي تمام گهڻو پالش ٿيل پيداوارن جي ضرورت ناهي. صاف، مددگار، ۽ مخصوص شعبي وارو هجڻ ئي اهم آهي.

## Account Page {#the-account-page}

Ultimate Multisite ۾ گراهڪن لاءِ سامهون وارو [Account Page](/user-guide/client-management/account-page) شامل آهي جتي گراهڪ پنهنجي سبسڪرپشن سنڀالين ٿا. هن کي ڪسٽمائيز ڪريو ته جيئن:

- انهن جو موجوده FitSite پلان ڏيکاري
- فٽنيس-مخصوص فائدن سان اپ گريڊ آپشنز ڏيکاري
- بلنگ تاريخ ۽ انوائس ڊائون لوڊ مهيا ڪري
- مدد وسيلن ڏانهن لنڪ ڪري

## شروعاتي رهنمائي جي ڪاميابي ماپڻ {#measuring-onboarding-success}

اهي اشارا ٽريڪ ڪريو ته ڄاڻو توهان جي شروعاتي رهنمائي ڪم ڪري رهي آهي يا نه:

- **فعال ٿيڻ جي شرح**: سائن اپس مان ڪيترا سيڪڙو پهرين هفتي اندر واقعي پنهنجي سائيٽ ڪسٽمائيز ڪن ٿا؟
- **پهرين هفتي جا لاگ اِن**: نئون گراهڪ پنهنجي پهرين هفتي دوران ڪيترا ڀيرا لاگ اِن ڪري ٿو؟
- **نون گراهڪن کان سپورٽ ٽڪيٽس**: وڌيڪ مقدار جو مطلب آهي ته توهان جي شروعاتي رهنمائي ۾ خاميون آهن
- **ٽرائل کان ادا ڪيل ۾ تبديلي**: جيڪڏهن توهان ٽرائلز پيش ڪريو ٿا، ته ڪيترا سيڪڙو تبديل ٿين ٿا؟

## هن وقت تائين FitSite نيٽ ورڪ {#the-fitsite-network-so-far}

```
FitSite نيٽ ورڪ
├── WordPress Multisite (سب ڊومين موڊ)
├── Ultimate Multisite (ترتيب ڏنل + برانڊ ٿيل)
├── پليٽ فارم ڊومين (fitsite.com + wildcard SSL)
├── سائيٽ ٽيمپليٽس (Studio Essential, Gym Pro, Fitness Chain)
├── پراڊڪٽس (Starter, Growth, Pro + Order Bumps)

├── برانڊنگ (لاگ اِن، Dashboard، اي ميلون، انوائسز، مارڪيٽنگ سائيٽ)
├── گراهڪ جي شروعاتي رهنمائي
│   ├── تڪڙي شروعات dashboard widget
│   ├── فٽنيس سائيٽ مينيجمينٽ لاءِ سادو ڪيل Dashboard
│   ├── 4-اي ميل ڀليڪار سلسلو
│   ├── مخصوص شعبي وارو ڄاڻ بنياد
│   ├── وڊيو واڪ ٿروز
│   └── ڪسٽمائيز ٿيل Account Page
└── قيمت واري حڪمت عملي لاءِ تيار (اڳيون سبق)
```

## هن سبق ۾ اسان ڇا ٺاهيو {#what-we-built-this-lesson}

- **هڪ رهنمائي ٿيل پهرئين لاگ اِن جو تجربو** Quick Start widget سان
- **هڪ سادو Dashboard** جيڪو فٽنيس سائيٽ مينيجمينٽ ڪمن تي ڌيان ڏئي ٿو
- **هڪ ڀليڪار اي ميل سلسلو** جيڪو گراهڪن کي سندن پهرين هفتي ۾ رهنمائي ڪري ٿو
- **مخصوص شعبي وارا مدد وسيلا** جيڪي غير-ٽيڪنيڪل فٽنيس اسٽوڊيو مالڪن لاءِ لکيل آهن
