# ریلیز نوٹس (Release Notes) {#release-notes}

## ورژن 2.13.0 — 2026-06-05 تائين ريليس ٿيو {#version-2130--released-on-2026-06-05}

- نئين: کسٹمر اکاؤنٹ، چیک آؤٹ، بلنگ، سائٹ، انوائس، ٹیمپلیٹ سوئچنگ، ۽ دومين ميپنگ واري عملن لاءِ سوورین-ٹینٽ سپورٽ شامل ڪئي آهي ته جيئن ٽينينet گهرجي ڪارن کي ماني سائيٽ ڏانهن واپس ريڊيڪٽ ڪري سگهن.
- نئين: باقاعده ممبرشپ لاءِ رينيو ڪرڊينشل جا هيٺيون شامل ڪيا ويا آهن ته جيئن گيٽويز ان آخيران کي ختم ڪري سگهن جڏهن ڪا محفوظ بلنگ entspreچ، سبسڪريپشن، يا واؤٽ ٽوڪن نٿي ملي.
- نئين: پنهنجي سائٹ جي ترتيب (pending site creation) لاءِ HMAC-verified loopback publishing شامل ڪئي آهي ته جيئن گهرجي جسن جتي پس منظر جا جوٻاي (background jobs) سست ٿين ٿا، چیک آؤٹ کان سائيٽ فراهم ڪرڻ واري عمل کي وڌيڪ قابل اعتماد بڻائي سگهجي.
- نئين: SSO URLs، چیک آؤٹ-فورم بيس دومينيوم، ۽ خود بخود دومين رڪڊن جي ترتيب لاءِ ڊولپرز ايڪسٽينشن پوائنٽس شامل ڪيا ويا آهن.
- تصحيح: مانيڊ ميپ ٿيل دومينيمن، انوائنيس بروکريٽر جي رهڻ، لاگ آؤٹ، ۽ پليجن کي پارٽي ڪنفسي (cross-plugin) تنازوه ۾ SSO وڌيڪ قابل اعتماد آهي.
- تصحيح: مانيڊ سائيٽ جي ترتيب جنهن لاءِ انتظار ڪيل هو، هاڻي پرانے پبلش فلگز کان ٻاهر ٿي سگهي ٿي ۽ کيس ڪارن کي سائيٽ جي ترتيب جو سڪرين تي رڪاوٽ نه ڏئي.
- تصحيح: شئرڊ چیک آؤٹ-فورم بيس دومينيمن لاءِ دومين رڪڊ ناهن ٺاهيون وينديون، ۽ جتي ڪا انٽيغريشن فعال نه هجي ته غير استعمال ٿيل گهرجي پراجڪٽ جي پس منظر جا جوٻاي کائڻ وارا عمل کائڻ کان بچيا ويندا آهن.
- تصحيح: چیک آؤٹ، بلنگ adhesion، پاسور ريسٽ، ايميل فيريڪيشن، ٹیمپلیٽ سوئچنگ، ٹورز، ۽ ڪسٹمرแดشبوڊ جا هيٺيون عام کسٹمرن جي عمل کي روڪڻ کان سواءِ نه ٿيندا.
- تصحيح: برڊڪاسٹ ايميل هاڻي وصول کنندڙن کي رازداري ۾ رکون ٿا جڏهن ته SMTP/پليجن جي فتال غلطين کان بچيون ٿا جڏهن وصول کنندڙن جي لسٹ يا ميل ترانسپورٽ ڪاميو نه ٿين.
- تصحيح: ممبرشپ رينيو، ختم ٿيڻ جو ڏيڪس، ۽ دفعي جمع ڪرائڻ جا هيٺيون فاسي (edge cases) هاڻي فوري ختم ٿيڻ، کراش، يا ضروري دفعات جي ادائیگی ڪرڻ کان مڪمل ناهن کي روڪيندا آهن.
- بهتر: WordPress سان مطابقت 7.0 تائين تجربو ڪئي وئي آهي، پروڊڪشن Vue ائسٽس ان امپ (npm) سورسز کان ٻيهر ٻڌل ٿا، ۽ Cypress جو ايڪيند-تو-ايڪيند ڪورج اب چیک آؤٹ، سیٽ اپ، SSO، ۽ گيٽوي عملن کي وڌيڪ آزمائش ڏئي ٿو.

## ورژن 2.12.0 — 2026-05-15 تيدار {#version-2120--released-on-2026-05-15}

- نئين: Hostinger (hPanel) کي سپورٽڊ هاستنگ پراجڪٽ طور شامل ڪيو آهي جنهن ۾ دومين ميپنگ جو نظام شامل آهي
- نئين: Site Exporter ان هتي همسائي ماڻهن جي سڌي مدد لاءِ نئين رينڊر ڪرڻ واري بيلز (network import bundles) کي سنڀالندي آهي، جنهن سان سڄي ورڪائڊ تي سائيٽ جو بحال ٿيڻ آسان ٿيندو
- تصحيح: BCC برڊڪاسٹ اي ميلز ۾ هدا مڪمل نه ڏيکيل وصول کنندڙن جي هڊر استعمال ڪئي ويندي آهي ته جيئن وصول کنندڙن جا پيٽي ظاهر ٿيڻ کان روڪجي سگهجي
- تصحيح: جڏهن غير تاريخي قيمت سان محفوظ ڪيو ويندو آهي ته ممبر شپ جو ختم ٿيڻ جو ڏينهن خراب نه ٿيندو
- تصحيح: Stripe ممبر شپ جي اپڊيٽس هاڻي پراني (deprecated) deleteDiscount API کي هلائڻ کان سواءِ گونج کي صحيح طور تي صاف ڪن ٿا
- تصحيح: دومين ميپنگ واري سائيٽن تي SSO ريڊيڪٽس کي ان حد ۾ رکيو ويو آهي ته جيئن لامحدود ريڊيڪٽ لوپ (infinite redirect loops) کان بچي سگهجي
- تصحيح: سټيپ اپ ويزر جو ايميج پيكر چونڊ هاڻي اندر موجود ڊيٽا ماڊل کي صحيح طور تي اپڊيٽ ڪري ٿو
- تصحيح: Site Exporter CLI ۾ همسائي سائيٽ جي انتخاب جو صحيح डिफلت سلڪشن برقرار رکيل آهي
- بهتر: پليجن پيڪيج کان bundled wp-cli کي ختم ڪيو ويو آهي، جنهن سان پليجن جو حجم گھٽ ٿيو آهي
