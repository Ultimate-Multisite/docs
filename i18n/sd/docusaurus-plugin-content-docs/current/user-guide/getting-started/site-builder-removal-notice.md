# साइट بلڊر موڊ ختم ڪرڻ جو نوٽيس

**Superdav AI Agent v1.12.0 ۾ Site Builder mode ختم ڪري ڇڏيو ويو آهي.** جيڪڏهن توهان Site Builder mode استعمال ڪندا هئيو، ته توهان کي ٿيم (theme) جي ٺاهڻ ۽ سائيٽ جي ترتیب لاءِ **Setup Assistant agent** تي منتقل ٿيڻ گهرجي.

## ڇا ٿيو؟

### Site Builder Mode (پرانا نظام)

Site Builder mode ان لاءِ هڪ ويزرڊ (wizard)-بنيو انٽرفيس هو:

- ٽيمپليٽس کان سائيٽون ٺاهڻ
- بنيادي ترتي بيان (basic settings) کي ترتیب ڏيڻ
- ڪا ٿيم چونڊڻ
- شروعاتي مواد (initial content) جو تربيت ڪرڻ

### ان جي جاءِ تي ڇا آهي؟

**Setup Assistant agent** هاڻي Site Builder جا سڀني ڪم ڪري ٿو، جنهن سان:

- وڌيڪ لچڪدار، اڳيان رهندي (agent-guided) ترتیب
- ٿيم کي ترتي بيان ڪرڻ لاءِ بهتر خيار
- Theme Builder جي شروعاتي (onboarding) سان جوڙجڪڻ
- مستقبل جا سشن لاءِ پائيدار site_brief memory

## جيڪڏهن توهان Site Builder Mode استعمال ڪندا هئيو

### توهان جون سائيٽون محفوظ آهن

- Site Builder mode سان ٺهي وئي موجودہ سائيٽون کام ڪندري ٿيون
- ڪا به معلومات ختم نه ٿيندي يا سائيٽ ۾ خرابيء نه اچي
- توهان عام طرح کان پنهنجي سائيٽن جو انتظام جاری رک سکد ٿا

### Setup Assistant Agent تي منتقل ٿيو

نئين سائيٽ جي ترتیب لاءِ يا ٿيم ۾ تبديلي لاءِ، Setup Assistant agent استعمال ڪريو:

```
"Help me set up a new site"
```

يا

```
"Start the Theme Builder onboarding"
```

Setup Assistant agent وڌيڪ لچڪدار انداز ۾ وٺندڙ ئي هيٺيون ڪم ڪري سگهي ٿو.

## موازنہ: Site Builder vs. Setup Assistant

| خاصيت | Site Builder (ختم) | Setup Assistant (نئين) |
|---|---|---|
| ترتیب جو طريقو | ويزرڊ فارم | Agent مکالمو (conversation) |
| ٿيم چونڊڻ | پنهنجي ترتي بيان (Predefined templates) | خود ٺاهيو (Custom generation) |
| ترتي بيان ڪرڻ | محدود خيار | پوري تصميم نظام (Full design system) |
| سائيٽ جو خلاصو (Site brief) | محفوظ نه ٿيندو | پائيدار memory |
| مستقبل جا سشن | دہرائي ترتيب | محفوظ site_brief استعمال ڪريو |
| لچڪ | ثابت ڪيل ڪمڪاري (Fixed workflow) | بدلون وارو مکالمو (Adaptive conversation) |

## Setup Assistant Agent تي منتقل ٿيڻ

### نئين سائيٽن لاءِ

साइट बिल्डर موڈ استعمال کرنے کے بجائے:

1. درخواست: "مجھے ایک نئی سائٹ سیٹ اپ کرنے میں مدد کریں"
2. سیٹ اپ اسسٹنٹ ایجنٹ آپ کی رہنمائی کرے گا:
   - سائٹ کا مقصد اور اہداف
   - ہدف آڈینس (Target Audience)
   - برانڈ کی شناخت (Brand Identity)
   - تھیم جنریشن
   - ابتدائی ترتیب

### موجودہ سائٹس کے لیے

اگر آپ کے پاس سائٹ بلڈر موڈ سے کوئی موجودہ سائٹ ہے:

1. آپ اسے ویسے ہی استعمال جاری رکھ سکتے ہیں
2. تھیم کو اپ ڈیٹ کرنے کے لیے، یہ درخواست کریں: "میری سائٹ کا دوبارہ ڈیزائن کرو" (Redesign my site)
3. سیٹ اپ اسسٹنٹ ایجنٹ آپ کو ایک نئی تھیم بنانے میں مدد کرے گا
4. آپ کی سائٹ کا ڈیٹا تبدیل نہیں ہوگا۔

### تھیم تبدیلیوں کے لیے

سائٹ بلڈر موڈ کے تھیم انتخاب کے بجائے:

1. درخواست: "میرا تھیم تبدیل کریں" (Change my theme)
2. سیٹ اپ اسسٹنٹ ایجنٹ یہ کرے گا:
   - آپ کی ڈیزائن کی ترجیحات کے بارے میں پوچھے گا
   - ایک کسٹم تھیم تیار کرے گا
   - اسے آپ کی سائٹ پر فعال کر دے گا

## اہم فرق

### سائٹ بلڈر موڈ (Site Builder Mode)

```
1. ٹیمپلیٹ منتخب کریں
2. تھیم منتخب کریں
3. بنیادی سیٹنگز ترتیب دیں
4. ہو گیا
```

### سیٹ اپ اسسٹنٹ ایجنٹ (Setup Assistant Agent)

```
1. اپنی سائٹ کا مقصد بیان کریں
2. اپنے ہدف آڈینس کی تعریف کریں
3. ڈیزائن کی ترجیحات منتخب کریں
4. ایجنٹ کسٹم تھیم تیار کرے گا
5. ایجنٹ تھیم کو فعال کرے گا
6. مستقبل کے سیشنز کے لیے سائٹ کا خلاصہ (site_brief) محفوظ کیا جائے گا
```

## سیٹ اپ اسسٹنٹ ایجنٹ کے فوائد

### زیادہ لچکدار (More Flexible)

- اپنی سائٹ کو عام زبان میں بیان کریں
- کسٹم سفارشات حاصل کریں
- آپ کی مخصوص ضروریات کے مطابق ڈھل جائیں

### بہتر کسٹمائزیشن (Better Customization)

- کسٹم تھیم جنریشن
- ڈیزائن سسٹم کے فیصلے
- مستقل ڈیزائن ٹوکنز (Persistent design tokens)

### مستقل یادداشت (Persistent Memory)

- آپ کا `site_brief` محفوظ رہتا ہے
- مستقبل کے ایجنٹ آپ کی سائٹ کو سمجھ جاتے ہیں
- سیٹ اپ کی معلومات دہرانے کی ضرورت نہیں پڑتی

### مربوط ورک فلو (Integrated Workflow)

- تھیم بلڈر آن بورڈنگ
- ڈیزائن سسٹم کی خوبصورتی کا ہنر (Design System Aesthetics skill)
- ویزیبلٹی کنٹرولز کی صلاحیت
- سب کام بغیر کسی رکاوٹ کے ایک ساتھ چلتے ہیں

## ٹرابل شوٹنگ

### مجھے سائٹ بلڈر موڈ نہیں مل رہا

سائٹ بلڈر موڈ ہٹا دیا گیا ہے۔ اس کے بجائے سیٹ اپ اسسٹنٹ ایجنٹ کا استعمال کریں:

"مجھ کو نئی سائٹ سیٹ اپ کرنے میں مدد کریں"

### میں سائٹ بلڈر سے ایک سائٹ دوبارہ بنانا چاہتا ہوں

آپ اسے سیٹ اپ اسسٹنٹ ایجنٹ کے ذریعے دوبارہ بنا سکتے ہیں:

1. درخواست دیں: "مجھ کو نئی سائٹ سیٹ اپ کرنے میں مدد کریں"
2. اپنی اصل سائٹ کے مقصد اور ڈیزائن کی تفصیل بتائیں
3. ایجنٹ ایک ملتی جلتی تھیم تیار کرے گا
4. آپ کی `site_brief` مستقبل کے حوالے کے لیے محفوظ ہو جائے گی

### میری موجودہ سائٹ بلڈر سائٹ کام نہیں کر رہی

سائٹ بلڈر موڈ سے بنائی گئی موجودہ سائٹس اب بھی کام کرتی رہتی ہیں۔ اگر آپ کو کوئی مسئلہ پیش آ رہا ہے تو:

1. چیک کریں کہ آپ کی تھیم ابھی بھی فعال (active) ہے
2. تصدیق کریں کہ آپ کے پگ انز (plugins) فعال ہیں
3. ورڈپریس ایرر لاگز (error logs) چیک کریں
4. اگر مسائل برقرار رہتے ہیں تو سپورٹ سے رابطہ کریں

### کیا میں اب بھی اپنے پرانے سائٹ بلڈر ٹیمپلیٹس استعمال کر سکتا ہوں؟

سائٹ بلڈر ٹیمپلیٹس اب دستیاب نہیں ہیں۔ تاہم:

- آپ کی موجودہ سائٹس کام کرتی رہیں گی
- آپ سیٹ اپ اسسٹنٹ ایجنٹ کے ذریعے ملتی جلتی سائٹس دوبارہ بنا سکتے ہیں
- سیٹ اپ اسسٹنٹ ایجنٹ میں مزید کسٹمائزیشن کے اختیارات فراہم کیے جاتے ہیں

## اگلے اقدامات

1. **نئی سائٹس کے لیے**: سیٹ اپ اسسٹنٹ ایجنٹ کا استعمال کریں
2. **موجودہ سائٹس کے لیے**: انہیں ویسے ہی استعمال کرتے رہیں
3. **تھیم تبدیلیاں کرنے کے لیے**: سیٹ اپ اسسٹنٹ ایجنٹ سے مدد طلب کریں
4. **ڈیزائن کو بہتر بنانے کے لیے**: ڈیزائن سسٹم ایکسٹسٹیکس (Design System Aesthetics) سکل کا استعمال کریں

## متعلقہ موضوعات

- **تھیم بلڈر آن بورڈنگ**: کسٹم تھیمز کے لیے گائیڈڈ سیٹ اپ
- **سیٹ اپ اسسٹنٹ ایجنٹ**: ایجنٹ کی رہنمائی سے سائٹ سیٹ اپ
- **سائٹ سپیسیفیکیشن سکل**: اپنی سائٹ کے اہداف اور سامعین کو متعین کریں
- **ڈیزائن سسٹم ایکسٹسٹیکس سکل**: اپنی سائٹ کی بصری شناخت (visual identity) کو بہتر بنائیں
