# تھیم بلڈر کی صلاحیتیں: اسکیفڈ اور ایکٹیویٹ بلاک تھیمز {#theme-builder-abilities-scaffold-and-activate-block-themes}

Superdav AI Agent v1.12.0 دو طاقتور صلاحیتیں لایا ہے جو آپ کو چیٹ انٹرفیس سے کسٹم بلاک تھیمز بنانے اور ڈپلائے کرنے کی اجازت دیتی ہیں۔

## جائزہ {#overview}

**scaffold-block-theme** اور **activate-theme** صلاحیتیں ایجنٹس کو یہ کرنے کے قابل بناتی ہیں:
- آپ کی تفصیلات کی بنیاد پر مکمل، پروڈکشن کے لیے تیار بلاک تھیمز جنریٹ کرنا
- دستی مداخلت کے بغیر خود بخود اپنی سائٹ پر تھیمز کو ایکٹیویٹ کرنا
- رہنمائی شدہ ڈیزائن کے فیصلوں کے ذریعے ہم آہنگ بصری شناخت (visual identities) تخلیق کرنا

## اسکیفڈ بلاک تھیم {#scaffold-block-theme}

**scaffold-block-theme** صلاحیت نئی WordPress بلاک تھیم بناتی ہے جس میں مکمل تھیم کی ساخت شامل ہوتی ہے، بشمول:

- ڈیزائن ٹوکنز کے ساتھ `theme.json` کنفیگریشن
- عام لے آؤٹ کے لیے بلاک ٹیمپلیٹس
- کسٹم بلاک اسٹائل اور ویریئیشنز
- تھیم میٹا ڈیٹا اور سپورٹ کی تفصیلات

### کیسے استعمال کریں {#how-to-invoke}

Superdav AI Agent کے ساتھ اپنی چیٹ میں، آپ تھیم جنریشن کا مطالبہ کر سکتے ہیں:

```
"ایک 'Modern Agency' نامی بلاک تھیم بنائیں جس میں نیلا اور سفید رنگوں کا سکیم ہو،
sans-serif ٹائپوگرافی ہو، اور ایک پیشہ ورانہ لے آؤٹ ہو۔"
```

ایجنٹ یہ کرے گا:
1. بات چیت کے ذریعے آپ کی ڈیزائن کی ترجیحات اکٹھا کرے گا
2. مکمل تھیم ڈھانچہ جنریٹ کرے گا
3. تمام ضروری تھیم فائلیں بنائے گا
4. ایکٹیویشن کے لیے تھیم کو تیار کرے گا

### متوقع آؤٹ پٹ {#expected-output}

جب یہ صلاحیت کامیابی سے چلتی ہے، تو آپ دیکھیں گے:

- اس کا تصدیق کہ تھیم اسکیفڈ ہو گئی ہے
- تھیم کا نام اور مقام
- لاگو کیے گئے ڈیزائن ٹوکنز کا خلاصہ (رنگ، ٹائپوگرافی، اسپیسنگ)
- ایکٹیویٹ کرنے کے لیے تیار کی حیثیت

مثال آؤٹ پٹ:
```
✓ تھیم "Modern Agency" کامیابی سے اسکیفڈ ہو گیا
  مقام: /wp-content/themes/modern-agency/
  رنگ: پرائمری #0066CC، سیکنڈری #FFFFFF
  ٹائپوگرافی: Inter (sans-serif)
  حالت: ایکٹیویٹ کرنے کے لیے تیار
```

## تھیم کو ایکٹیویٹ کریں {#activate-theme}

**activate-theme** جي صلاحيت توهان جي سائيٽ کي نوي بنائي واري يا موجود block theme تي منتقل ڪرڻ لاءِ استعمال ٿي ٿي.

### ڪيئن استعمال ڪجي {#how-to-invoke-1}

Theme بنائڻ کان پوءِ، توهان ان کي فوري طور تي فعال ڪري سگهو ٿا:

```
"Modern Agency theme کي فعال ڪريو"
```

يا موجوده ڪنه به theme کي فعال ڪريو:

```
"Twentytwentyfour theme تي منتقل ٿيو"
```

### توقع ڪيل نتيجي {#expected-output-1}

جڏهن فعال ڪرڻ ۾ ڪاميابي ٿئي ته:

- فعال سميائي واري theme جو تصديق
- پُچھلا theme جو نالو (حواله لاءِ)
- سائيٽ جو URL جتي theme موجوده آهي
- ڪنه به theme مخصوص set up جي نوٽس

مثال نتيجي:
```
✓ Theme ڪاميابن سان فعال ٿيو
  فعال سميائي واري theme: Modern Agency
  پُچھلا theme: Twentytwentyfour
  لائیو آهي: https://yoursite.com
  نوٽ: لے آؤپ کي تصديق ڪرڻ لاءِ پنهنجو homepage چيڪ ڪريو
```

## ڪم ڪندڙ عمل (Workflow): Scaffold ۽ فعال ڪرڻ {#workflow-scaffold-and-activate}

عام ڪم ڪندڙ عمل ۾ ٻئي صلاحيتون شامل ٿين ٿيون:

1. **Theme جي ٺهڻ جو درخواست**: "منه جي SaaS لاندين لاءِ هڪ block theme banaؤ"
2. **Agent Theme کي scaffold ڪري ٿو**: فائلون ۽ design tokens بنائندا آهي
3. **جنهن کي ڏسڻ ۽ بهتر ڪرڻ**: جيڪڏهن ضرورت پئي ته design جي تبديلين تي بحث ڪيو ويندو
4. **فعال ڪرڻ**: "Theme کي هله فعال ڪريو"
5. **تصديق ڪرڻ**: نئين design جو لائیو ٿيڻ کان پوءِ سائيٽ بجيڪ ڪري ڏسو

## Design Tokens ۽ Customization {#design-tokens-and-customization}

Scaffold ڪيل themes WordPress design tokens (via `theme.json`) جو استعمال ڪن:

- **Colors**: Primary، secondary، accent، neutral palette
- **Typography**: Font families، sizes، weights، line heights
- **Spacing**: Padding، margin، gap scales
- **Borders**: Radius ۽ width tokens
- **Shadows**: Elevation levels

اهي tokens `theme.json` ۾ مركزيت ٿين ٿا، جنهن سان توهان هڪ فائل کان پنهنجي سڄي design system کي آسان بڻائي سگهو ٿا.

## محدوديتون ۽ نوٽس {#limitations-and-notes}

थीमहरू `/wp-content/themes/` मा ढाँचा (scaffold) गरिन्छ र तिनीले वर्डप्रेसको नामकरण नियमहरू पालना गर्नुपर्छ।
सक्रिय (Activation) गर्नका लागि तपाईंको वर्डप्रेस साइटमा उपयुक्त अनुमति (permissions) चाहिन्छ।
थीमहरूमा कस्टम PHP कोड कम प्रयोग गर्नुहोस्; जटिल कामहरूको लागि प्लगइन प्रयोग गर्नुहोस्।
ब्लक थीम्सहरू वर्डप्रेस ५.९ र त्यसपछि राम्रोसँग चल्छन्।

## समस्या समाधान (Troubleshooting) {#troubleshooting}

**ढाँचा बनाएपछि थीम देखिँदैन**
- जाँच गर्नुहोस् कि थीमको डाइरेक्टरी छ र उसलाई सही अनुमति (permissions) दिइएको छ।
- हेर्नुहोस् कि `theme.json` वैध JSON हो।
- सुनिश्चित गर्नुहोस् कि थीमको नाम पहिले नै भएका कुनै थीमसँग मिल्दैन।

**सक्रिय हुन असफल (Activation fails)**
- तपाईंमा एडमिन अनुमति छन् भने पुष्टि गर्नुहोस्।
- जाँच गर्नुहोस् कि वर्डप्रेसले थीमको डाइरेक्टरी पढ्न सक्छ।
- विवरणका लागि वर्डप्रेसको एरर लगहरू (error logs) हेर्नुहोस्।

**डिजाईन टोकन लागू हुँदैन**
- `theme.json` को सिन्ट्याक्स सही छ भने पुष्टि गर्नुहोस्।
- कुनै पनि क्याचिङ प्लगइन सफा गर्नुहोस्।
- जाँच गर्नुहोस् कि तपाईंको वर्डप्रेस संस्करणले प्रयोग गरिरहनुभएको टोकनलाई समर्थन गर्छ।

## अर्को कदम (Next Steps) {#next-steps}

तपाईंको थीम सक्रिय गरेपछि, तपाईं निम्न कुराहरू गर्न सक्नुहुन्छ:
- टाइपोग्राफी, रंग र स्पेसलिङ सुधार गर्न **Design System Aesthetics skill** प्रयोग गर्नुहोस्।
- वर्डप्रेस ब्लक एडिटर मार्फत व्यक्तिगत ब्लक शैलीहरू अनुकूलित (customize) गर्नुहोस्।
- थीमको `style.css` फाइलमा कस्टम CSS थप्नुहोस्।
- विशिष्ट पेज प्रकारहरूको लागि कस्टम ब्लक टेम्प्लेट बनाउनुहोस्।
