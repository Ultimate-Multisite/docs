# आफ्नो एडमिन प्यानललाई PRO थीमले कसरी बनाउन सकिन्छ

कहिलेकाहीँ क्लायन्टको काम गर्दा हामीलाई वर्डप्रेसको एडमिन ड्यासबोर्डमा आफ्ना क्लायन्टहरूलाई एउटा विशेष अनुभव दिनुपर्ने आवश्यकता पर्छ। यदि तपाईंलाई अझ व्यावसायिक देखिने र अनुभव गर्न लायक लुक चाहिएको छ भने PRO Theme ले मद्दत गर्न सक्छ।

PRO Theme एक एडमिन थीम हो जुन तपाईंलाई आफ्ना क्लायन्टहरूको ड्यासबोर्ड कस्टमाइज़ गर्दा सहयोग गर्न बनाइएको हो। यो Ultimate Multisite को प्रीमियम प्लगइन कोरको पनि हिस्सा हो।

यो इन्स्टल गरेपछि, आफ्नो _Plugins_ पेजमा जानुहोस् र आफ्नो नेटवर्कमा यसलाई सक्रिय (activate) गर्न क्लिक गर्नुहोस्।

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_आफ्नो Plugins पेजमा नेटवर्क-व्यापी रूपमा सक्रिय गर्नुहोस्_

_**नोट:**_ _यदि तपाईंले यो इन्स्टल गर्नुभयो भने, यसले तपाईंको क्लायन्टहरूलाई एउटा विकल्पको रूपमा देखाउनेछ जसलाई केवल त्यो प्रयोगकर्ताका लागि थीम थप्न चिह्नित गर्न सकिन्छ._

अब, तपाईंको एडमिन इन्टरफेस पूर्ण रूपमा नयाँ डिजाइन गरिएको छ! बायाँ स्तम्भमा 'PRO Theme' नामको एउटा नयाँ मेनु पनि तपाईंसँग हुनेछ।
