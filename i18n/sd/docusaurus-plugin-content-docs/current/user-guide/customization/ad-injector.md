# آپৰ নেটৱৰकमा বিজ্ঞাপন থকাৰ বাবে Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector এটা **ফ্ৰী অ্যাড-অন** যিয়ে আপোনাক তেওঁলোকৰ এডিট পেজৰ পৰাই নিজৰ প্লেনত বিজ্ঞাপন থকাৰ বাবে সকলো সা-সুবিধা দিয়ে।

Ultimate Multisite: Ad Injector ইনষ্টল কৰাৰ পিছত, ইয়াক অ্যাক্টিভেট কৰিবলৈ আপোনাৰ _Plugins_ পেজত जानुपर्छ।

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Network Activate বাছনিটো ক্লিক কৰক_

অ্যাড-অনটো অ্যাক্টিভেট হোৱাৰ পিছত, প্লেন এডিট স্ক্ৰীণৰ _Advanced Options_ মেটা বক্সত '_Ad Settings_' নামৰ এটা নতুন অপচন থাকিব।

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_এতিয়া আপুনি আপোনাৰ প্ৰতিটো প্লেনৰ বাবে বিজ্ঞাপনৰ সেটিংবোৰ এডিট কৰিব পাৰিব_

সেই ট্যাবত, আপুনি ফ্ৰণ্ট-এণ্ডৰ পোষ্ট আৰু পেজবোৰৰ বাবে অ্যাড কোডবোৰ যোগ কৰিব পাৰিব, কন্টেন্টৰ আগতে আৰু পিছতো।

### বিজ্ঞাপনৰ বাহিৰত

আপুনি উচ্চ স্তৰৰ প্লেনবোৰলৈ সমৰ্থন দিবলৈও Ultimate Multisite: Ad Injector ব্যৱহাৰ কৰিব পাৰে।

আপুনি কেৱল _Admin Panel Ads_ অপচনটো লৈ আপোনাৰ সমৰ্থন টুলৰ HTML কোড যোগ কৰক।
