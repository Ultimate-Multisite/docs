# ٻين پيگمنٽس سان جاني ٿيل غير مطابقت (Known incompatibilities with other Plugins)

Ultimate Multisite کي انسٹال ۽ فعال ڪرڻ کان پوءِ، توهان کي ٻين پيگمنٽس سان ڪجهه مسئلا اچڻ ۾ ٿي سگهن ٿا جيڪي حل نه ڪيا ويا آهن.

اسان هر نئين ريلس (release) سان اسان هن لست کي گهٽائي لاءِ ڪم ڪري رهيا آهيون، پر جيڪڏهن ڪو پيگمنٽ هيٺ ڏنل آهي، ته توهان کي ان کي غير فعال ڪرڻ جي ضرورت پئي ٿي ته جيئن Ultimate Multisite Network پورو ڪامياب طور تي کام ڪري سگهي.

هي ان جي لست ۽ ان صورت ۾ توهان کي چهري ڪهڙا ڪارنام ڪرڻ گهرجي انهن جي باري آهي:

**ڪجهه WPMU DEV پيگمنٽس** ڪجهه پيگمنٽس جيڪي WPMU DEV طرفان مائڻ بند ٿي ويا آهن، انهن کي جاني آهي ته Ultimate Multisite جي ريجيسٽري فنڪس (registration flow) آخري قدم تي رکجي ويندي آهي (يعني صارف اڪاونٽ نه bana سکدا). هيٺ ڏنل پيگمنٽس ان مسئلن کا سبب بڻجن:

  * Signup Language: _Ultimate Multisite ۾ هڪ add-on آهي جيڪو هيٺ ڏنل ڪارڪردگي جو يقيني ڏئي ٿو، جنهن جو نالو Ultimate Multisite: Language Switcher آهي._

  * Set Blog Description on Blog Creation: _هي Ultimate Multisite ۾ بڻايو ويو آهي، تنهنجي لاءِ اضافي پيگمنٽ جي ضرورت نه آهي._

  * New Blog Templates: _هي Ultimate Multisite ۾ بڻايو ويو آهي، تنهنجي لاءِ اضافي پيگمنٽ جي ضرورت نه آهي._

  * Set Password on Multisite Blog Creation: _هي Ultimate Multisite ۾ بڻايو ويو آهي، تنهنجي لاءِ اضافي پيگمنٽ جي ضرورت نه آهي._

  * Signup TOS: _هي Ultimate Multisite ۾ بڻايو ويو آهي، تنهنجي لاءِ اضافي پيگمنٽ جي ضرورت نه آهي._

  * Anti-Splog: جيتوڻيڪ اهو ٿورو کمزور آهي، پر هي به _Ultimate Multisite ۾ بڻايو ويو آهي._

**نوٽ:** ٻيا سڀ WPMU Dev پيگمنٽس Ultimate Multisite سان عام طور تي استعمال ڪري سگهجن ٿا. مثالن ۾ _Smush_ ، _Forminator_ ، _Defender,_ تنهنجي لاءِ ڪم ڪرڻ گهرجي.
