# مدد کیسے حاصل کریں {#how-to-get-help}

آپ کے پاس لائسنس کی کلید ہے، اور اس کے ساتھ ہماری ٹیم سے مدد لینے کا اختیار بھی ہے تاکہ آپ کو Ultimate Multisite سیٹ اپ کرتے وقت پیش آنے والے مسائل حل کرنے میں مدد مل سکے۔

یہ سبق لائسنس ہولڈرز کے لیے دستیاب سپورٹ چینلز اور ہمارے سپورٹ ٹیم سے رابطہ کرنے کے لیے آپ کو کس طرح کی معلومات تیار رکھنی ہوں، یہ بتاتا ہے۔

## مدد سائٹ (Help Site) {#help-site}

آپ ہماری مدد سائٹ سے مختلف آرٹیکلز اور ٹیوٹوریلز تک رسائی حاصل کر سکتے ہیں جو نیچے دیے گئے لنک میں مل سکتی ہیں۔

[)

ہم اپنے صارفین کو Ultimate Multisite کی بنیادی خصوصیات اور کاموں سے واقف ہونے کی ترغیب دیتے ہیں۔ ہماری مدد سائٹ پر "Getting Started" کے آرٹیکلز دستیاب ہیں اور عام سوالات کے جوابات بھی وہاں مل جاتے ہیں، جیسے ایڈ-آؤن (add-ons) کیسے انسٹال کریں، ادائیگی کے گیٹ ویز (payment gateways) کو کیسے انٹیگریٹ کریں وغیرہ۔

## سپورٹ سے رابطہ (Contact Support) {#contact-support}

اگر آپ کسی مسئلے میں پھنس جائیں اور ہماری مدد سائٹ سے جواب نہیں مل پاتا تو آپ ہمارے سپورٹ ٹیم سے رابطہ کر سکتے ہیں یا ہماری ویب سائٹ کے نچلے دائیں کونے میں موجود سپیچ بلبل آئیکن پر کلک کرکے براہ راست support@ultimatemultisite.com پر ای میل بھی بھیج سکتے ہیں۔

اگر تکنیکی مسائل کے لیے سپورٹ سے رابطہ کرتے وقت، تو براہ کرم اپنے مسئلے کی تفصیل ایک اسکرین شاٹ، ایرر میسجز یا کسی بھی ایسی معلومات کے ساتھ فراہم کریں جو ہمارے سپورٹ لوگوں کو آپ کے مسئلے کو آسانی سے سمجھنے اور جانچنے میں مدد دے سکے۔ آپ کو اپنی لائسنس کی کلید بھی فراہم کرنی ہوگی۔ صرف فعال (active) لائسنس ہولڈرز ہی سپورٹ کے حقدار ہیں۔

## کمیونٹی سائٹ (Community Site) {#community-site}

ہماری [کمیونٹی سائٹ](https://community.ultimatemultisite.com/) Ultimate Multisite صارفین کو ایک دوسرے کے ساتھ بات چیت کرنے کا ایک جگہ فراہم کرتی ہے۔ یہ ایک معلومات کا نظام ہے جہاں ممبرز پوسٹ کر سکتے ہیں، مباحثوں پر تبصرہ کر سکتے ہیں، مشورہ دے سکتے ہیں، تعاون کر سکتے ہیں یا لوگوں سے مدد مانگ سکتے ہیں۔ نجی جگہیں/صفحات کھولنے کے لیے آپ کو اپنی Ultimate Multisite لائسنس کی کلید فراہم کرنے کی ضرورت ہوگی۔
