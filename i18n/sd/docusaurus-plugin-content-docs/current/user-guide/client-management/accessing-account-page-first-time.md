# میں پہلی بار اپنا اکاؤنٹ پیج کیسے دیکھ سکتا ہوں؟ {#how-can-i-access-my-account-page-for-the-first-time}

Ultimate Multisite میں خوش آمدید!

اپنے اکاؤنٹ پیج تک پہنچنے کے لیے اس لنک کا استعمال کریں جو آپ کو ہمارے پگ ان (plugin) خریدنے کے بعد ای میل میں ملا ہوگا:

<!-- Screenshot unavailable: Account page email with access link -->

وہ ای میل استعمال کریں جو آپ نے خریداری کے دوران فراہم کی تھی اور وہ پاس ورڈ جو آپ کو ای میل میں ملا ہوگا۔

<!-- Screenshot unavailable: Login page with email and password fields -->

چونکہ یہ پہلی بار ہے جب آپ اس اکاؤنٹ میں لاگ ان کر رہے ہیں، تو آپ سے اپنا پاس ورڈ تبدیل کرنے کے لیے کہا جائے گا۔ اسے تبدیل کریں اور _Save_ بٹن پر کلک کریں۔

<!-- Screenshot unavailable: Change password form with Save button -->

اپنے اکاؤنٹ پیج پر، آپ اپنے تمام اکاؤنٹ کی تفصیلات جیسے بلنگ (billings)، تجدید (renewals)، منسوخیوں (cancellations)، اور بہت سی دوسری چیزیں دیکھ اور سنبھال سکتے ہیں۔
