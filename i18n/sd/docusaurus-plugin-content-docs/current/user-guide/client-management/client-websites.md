# توهان جي ڪلائين جا ويب سائيٽس (v2) {#your-clients-websites-v2}

پنهنجي نڪرن تي رجسٽر ڪرڻ کان پوءِ، توهان جا گهڻا ڪلائين پنهنجو dashboard استعمال ڪري سگهجن ٿا.

**Account** صفحو، جتي اهي پنهنجي دفعات، ممزرو (memberships)، دومين، پلان جي حدن ۽ ٻين اهم معلومات ڏس سگهن ٿا، ان کان علاوه **My Sites** صفحو تي به وصول ڪندا آهن، جتي اهي پنهنجي سڀني ويب سائيٽس کي ڏسي سگهن ۽ منظم ڪري سگهن ٿا. انهن لاءِ بس پنهنجي dashboard تي My Sites تي ڇڪڻ گهرجي.

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

اهن توهان جي سڀني ويب سائيٽس ڏسي ويندا، جن جو هيٺ هڪ **Manage** بٽون رکيل آهي. جيڪڏهن اهي ان تي ڪلڪائين ٿا، ته انهن کي ان خاص ويب سائيٽ جو **dashboard** ڏانهن ريڊيڪٽ ڪيو ويندو.

اهي اهو به ڏس سگهو ٿا ته توهان جي ڪهڙي ويب سائيٽ پلان لاءِ **primary** آهي جيڪڏهن پلان هڪ کان وڌيڪ ويب سائيٽون رکڻ جي اجازت ڏئي ٿو.

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

هي ويب سائيٽ جو هيٺ ڏنل كامل ويوز آهي:

![Site edit full interface](/img/admin/site-edit-full.png)

جيڪڏهن اهي پنهنجي primary ويب سائيٽ بدلڻ چاهين، ته انهن کي **Account** صفحو تي وڃڻ گهرجي ۽ **Account** مينيوي تي ڪلڪ ڪرڻ گهرجي.

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

توهان گهڻي کليان جي تفصيل ڏسڻ ۽ ان ۾ تبديلي ڪرڻ به سگهو ٿا:

![Customer edit interface](/img/admin/customer-edit.png)

پھر، انهن کي **Change Default Site** تي ڪلڪ ڪرڻ گهرجي.

![Change Default Site button on account page](/img/account-page/change-default-site.png)

هڪ پاپ اپ ويندو. 드롭-ڊاؤن لسٹ تي ڪلڪ ڪريو ۽ انتخاب ڪريو ته توهان ڪهڙي ويب سائيٽ کي primary ويب سائيٽ بڻائڻ چاهيو ٿا.

![Change Default Site dropdown selection](/img/account-page/change-default-site-dropdown.png)

پھر، بس **Change Default Site** تي ڪلڪ ڪريو ۽ توهان جو ڪم ختم!

![Default site بدل ڪيو واري تصوير](https://example.com/img/account-page/change-default-site-confirm.png)
