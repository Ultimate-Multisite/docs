# V1 کان ميو (Migrating from V1)

## Ultimate Multisite پنهنجي اصل 1.x جي ريليوزن مان جو 2.x جي ريليوزن مان جو سوي آهي.

Ultimate Multisite جو ورژن 2.0 ۽ ان کان وڌيڪ، ڪو مکمل نئين ڪوڊ (rewrite) آهي، جس کا مطلب آهي ته پرانا ۽ نئين ورژن ۾ گهڻو شيئر ڪرڻ جو گهڻو محدود آهي. ان ڪري، جڏهن توهان 1.x کان 2.x تي اپ گریڊ ڪري رهيا آهيو، تنهنجي ڊيٽا کي هڪ اهڙي فارمٽ ۾ منتقل ڪرڻ جي ضرورت پوندي ته جنه نئين ورژن سمجهي سگهن.

خوشڪاريءَ سان، Ultimate Multisite 2.0+ **هڪ مائجريٽر (migrator)** همراه آهي جيڪو ڪور (core) ۾ بڻايو ويو آهي ۽ پرانے ورژن کان ڊيٽا کي محسوس ڪري سگهي ٿو ۽ ان کي نئين فارمٽ ۾ تبديل ڪري سگهي ٿو. هي مائجريشن ورژن 2.0+ جي **Setup Wizard** دوران ٿيندي آهي.

هي سبق اسان ٻڌائي ٿو ته مائجريٽر ڪيئن ڪم ڪري ٿو، غلطي پوندي صورت ۾ کیا ڪرڻ گهرجي، ۽ ان عمل دوران اچڻ واري مسئلن کي ڪيئن حل (troubleshoot) ڪجي.

_**اهم: 1.x کان 2.0 تي اپ گریڊ ڪرڻ شروع ڪرڻ کان اڳ توهان کي پنهنجي سائيٽ جي ڊيٽا جو بڪ اپ (backup) ضرور بنايو رکڻ گهرجي**_

## پهرين قدم

پهرين قدم ۾، توهان کي پليجن (.zip) فائل ڈاؤنلوڊ ڪرڻ ۽ ان کي پنهنجي نڪريل اڊمنแดشبوڊ تي ورژن 2.0 نصب ڪرڻ آهي.

جڏهن توهان [install and activate version 2.0](../getting-started/installing-ultimate-multisite.md) ڪندا، ته سسٽم خود هيءَ محسوس ڪندو ته توهان جو Multisite پراني ورژن تي رن ٿو ۽ توهان کي پليجن جي صفحي جي اوپري طرف اهو ميساج ڏسڻ ۾ ايندو.

_**نوٽ:** جيڪڏهن توهان جي Multisite تي Ultimate Multisite 1.x نصب آهي، ته توهان کي پليجن کي ان ورژن سان بدلائڻ جو چوڻ ملندو آهي جيڪو توهان هن وقت ڈاؤنلوڊ ڪيو آهي. مهرباني ڪري، **Replace current with uploaded** تي کليڪ ڪرڻ گهرجي._

<!-- Screenshot unavailable: Plugin page showing option to replace v1 with uploaded v2 version -->

ناليجي پيچ پر توهان کي معلوم ٿيندو ته توهان ڪهڙا ليجسٽ اڊ-آنيس (legacy add-ons) نصب ڪيا آهن جنه ورژن 1.x سان همراه آهن. ان ۾ تعليمات به هونديون ته جيڪڏهن توهان جو استعمال ڪيو ويو ورژن 2.0 سان مطابقت رکي ٿو يا هلڪو منتقل ٿيڻ کان پوءِ اڊ-آنيس جو اپ گریڊ (upgraded version) نصب ڪرڻ جي ضرورت آهي.

<!-- Screenshot unavailable: Message on the top of the plugins page about updating to v2.0, with list of add-ons that need updating -->

جڏهن توهان هيٺ ڏنل ڪيل شيءِ کي ختم ڪرڻ لاءِ تيار ٿي آيو، ته توهان ان بٽون تي کليڪ ڪري سگهو ٿا جيڪو **Visit the Installer to finish the upgrade** لکيو آهي.

<!-- Screenshot unavailable: Button saying Visit the Installer to finish the upgrade -->

پھر اهو توهان کي ان installation wizard جو صفحو ڏانهن رکو ويندو جتي ڪجهه خوش آمديد جو پيغام هوندا آهن. توهان کي بس **Get Started** تي کليڪ ڪرڻ جي ضرورت آهي ته هيٺين صفحي تي وڃڻ لاءِ.

<!-- Screenshot unavailable: Setup Wizard welcoming page with Get Started button -->

**Get Started** تي کليڪ ڪرڻ کان پوءِ، اهو توهان کي Pre-install Checks_._ تي ريڊيڪٽ ڪندو. ان ۾ توهان جي سسٽم جي معلومات ۽ WordPress installation ڏيکاري ٿو ۽ توهان کي ٻڌائي ڇڏيندو ته هي **Ultimate Multisite** جي گهرج (requirements) سان مطابقت رکي ٿو يا نه.

<!-- Screenshot unavailable: Pre-install Checks page showing system requirements confirmation -->

هيٺ جو قدم اهو آهي ته توهان پنهنجو Ultimate Multisite license key داخل ڪريو ۽ plugin کي فعال (activate) ڪريو. هي اسان کي يقيني بڻائي ڇڏيندو ته سسٽي تي تمام خاص طور تي اڊ-آنيس، شامل اهڙا به، موجود هوندا.

<!-- Screenshot unavailable: License activation page with license key input and Agree and Activate button -->

پنهنجو ڪي (key) داخل ڪرڻ کان پوءِ، **Agree & Activate** تي کليڪ ڪريو.

License فعال ٿيڻ کان پوءِ، توهان اصل installation شروع ڪري سگهو ٿا جڏهن توهان هيٺين صفحي تي **Install** تي کليڪ ڪندا آهيو. اهو خود بخود ان ضروري فائلون ۽ database کي ٺاهي ڇڏيندو جيڪي ورژن 2.0 کي کام ڪرڻ لاءِ گهرجن.

## ته، مِيجريشن (Migration)

مِيجريٹر ۾ هڪ اندروني سيڪيورٽي فيچر آهي جنهن سان هو توهان جي سڄي ملٽي سائيٽ کي چيڪ ڪندو آهي ته هي ٺيڪ ٿيندي آهي ته توهان جو سڀئي Ultimate Multisite ڊيٽا ڪنهن به مسئلي کان سواءِ مِيجريٽ ڪري سگهجي ٿو. عمل شروع ڪرڻ لاءِ **Run Check** بٽڻ تي ڪلڪ ڪريو.

بعد مان چيڪ رن ڪرڻ کان پوءِ، توهان جي دو ئي رستو هوندا آهن: نتيجا يا ته غلطي (error) سان آهي يا نه آهي.

### غلطي سان ٿيڻ صورت ۾ (With Error)

جيڪڏهن توهان کي غلطي جو ميغير ملي، ته توهان کي اسان جي سپورٽ ٽيم تائين رابطو ڪرڻو پوندو ته جنهن سٺي مدد ڪري سگهون آهن غلطي کي درست ڪرڻ ۾. ڪارارڊ (ticket) ٺاهندو وقت **error log** ضرور ڏيو. توهان لَگ کي ڊائونلو ڪري سگهو ٿا يا ان لِنڪ تي ڪلڪ ڪري سگهو ٿا جيڪا "contact our support team" لکي ٿي. اهو توهان جي صفح سان هيڊ-سايد ۾ هڪ مدد وِجٽ (help widget) کيلڪ ڪرڻ سان ٺهي ويندو، جتي توهان لاءِ پورا فارم پري پورو هوندا آهن جن ۾ غلطي جا لاگ (error logs) بيان ڪيا ويا هوندا.

_**چڱو آهي ته چونڊ ٿيو آهي ته توهان 2.0 اي تي مِيجريٽ نه ڪري سگهو ٿا. تڏهن توهان 1.x تي واپس وڃي سگهو ٿا ته جيئن توهان جو نڪتون (network) غلطي درست ٿيڻ تائين چلوندي رهيو.**_

### غلطي کان سواءِ (Without Error)

جيڪڏهن نظام کي ڪا غلطي نه ملي، ته توهان کي هڪ سنڀال جو ميغير ڏسيل ويندو ۽ هيٺ **Migrate** بٽو هوندي جيڪا توهان کي مِيجريشن ۾ اڳتي وڌڻ جي اجازت ڏيندي. هن صفحا تي، توهان کي اهو یاد دلائي ويندو ته پنهنجي ڊيٽا جو بڪ اپ (backup) ٺاهيو ته قدم وٺڻ کان اڳ، جنهن سڀني لاءِ اسان سخت سفارش ڪندون آهيون. جيڪڏهن توهان جي پاسي پري بڪ اپ موجود آهي ته **Migrate** تي ڪلڪ ڪريو.

هي بس اتلي آهي!

تجيری کے لیے، آپ ویزارد سیٹ اپ جاری رکھ سکتے ہیں تاکہ اپنے نیٹ ورک پر اپنا لوگو اور دیگر چیزیں اپ ڈیٹ کر سکیں یا پھر اپنے Ultimate Multisite ورژن 2.0 کے مینو اور اس نئے انٹرفیس میں گھومنا شروع کر دیں۔ آگے بڑھیں اور مزے کریں۔
