# Ultimate Multisite Hooks and filters

Ultimate Multisite ۾ ڪيترن ئي فنڪشنز ۽ ڊيٽابيس ڪويريون شامل آهن. ان ۾ هوڪس (hooks) ۽ فلٽر (filters) آهن جيڪي توهان کي موجودہ فنڪشنز ۾ ڪو نون ڪوڊ هلائڻ يا تبديل ڪرڻ جي اجازت ڏين ٿا.

اسان هنن هوڪس ۽ فلٽرن جي هڪ لسٹ هيٺ ڏني آهي ته advanced users کي پنهنجي network کي اڳتي ڪيئن customize ڪرڻ لاءِ مدد ملي سگهي. توهان Github کان ڊسي (link) تي کليڪ ڪري ان لسٹ تک رسائي سگهو ٿا.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
