# Ultimate Multisite ترجمو

Ultimate Multisite ان English ۾ ٺهي آهي، پر هي plugin پوري طرح ترجمي لاءِ کليل آهي ۽ اسين ترجمي جو تعاون گرم جوار استقبال ڪندون آهيون.

## Core Plugin Translation (مرڪزي plugin جي ترجمو)

Core Ultimate Multisite plugin کي آفيسيل **WordPress.org translation platform** ترو ترجمو ڪيو ويندو آهي:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

هي standard WordPress community translation system آهي. جڏهن ترجمون ڪافي منظوري حاصل ڪري ويندا آهن، تڏهن اهي WordPress plugin updates جي ذريعي سڀني Ultimate Multisite users کي خود بخود ڇائع جنون ٿا.

**اسان کي توهان جي مدد گهرجي!** جيڪڏهن توهان English کان ٻيو ٻولي بولندا آهيو، ته ترجمو ڏيڻ جو سوچيو. ڪجهه strings ترجمو ڪرڻ سان به مدد ملندي آهي. شروع ڪرڻ لاءِ:

1. [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/) تي وڃو
2. پنهنجي ٻولي چونڊيو
3. "Contribute Translation" ڇڪڻ تي ڪلڪ ڪريو
4. توهان کي WordPress.org جو مفت account سڀني لاءِ گهرجي

 جيڪڏهن توهان پنهنجي ٻولي لاءِ translation editor بڻجي چاهيو (ترجمون منظور ڪرڻ جي صلاحيت رکڻ)، ته [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) ذريعي رابطو ڪري سگهو ٿا.

## Addon Translations (Addon جو ترجمو)

Addon plugins کي الگ کان:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)** تي ترجمو ڪيو ويندو آهي.

Addon translations موجوده وقت AI جي مدد سان شروع ڪيا ويا آهن. اهي سڀني سمجهيل ٻولين کي کوري ڇڏون ٿا پر انهن ۾ ڪجهه awkward phrasing يا غلطيون ٿي سگهن ٿيون. اسين human translators کي انهن کي وٺڻ ۽ انهن کي وڌيڪ قدرتي ۽ درست بڻائڻ لاءِ مشورو ڏيڻ جو استقبال ڪندون آهيون.

## Local Translations (مقامي ترجمو)

اگر آپ اپنے پروجیکٹ کے لیے مقامی طور پر ترجمہ کرنا پسند کرتے ہیں بجائے اس کے کہ اپسٹریم (upstream) میں حصہ لیں، تو دو طریقے موجود ہیں:

### Poedit کا استعمال کرنا

آپ `.pot` فائل کو ترجمہ کرنے کے لیے ڈیسک ٹاپ ایپلیکیشن **[Poedit](https://poedit.net)** استعمال کر سکتے ہیں:

1. [poedit.net](https://poedit.net) سے Poedit ڈاؤن لوڈ کریں۔
2. `wp-content/plugins/ultimate-multisite/lang/` سے `.pot` فائل کھولیں۔
3. Poedit کے انٹرفیس میں اسٹرنگز (strings) کا ترجمہ کریں۔
4. محفوظ کریں -- Poedit خود `.po` اور `.mo` فائلیں بنا دیتا ہے۔
5. آؤٹ پٹ فائلیں `lang` فولڈر میں رکھ دیں۔

### Loco Translate کا استعمال کرنا

**[Loco Translate](https://wordpress.org/plugins/loco-translate/)** ورڈپریس پلگ ان آپ کو کسی ڈیسک ٹاپ ٹولز کے بغیر براہ راست ایڈمن پینل سے ترجمہ کرنے کی اجازت دیتا ہے۔ یہ آپ کے لیے ترجمہ فائلیں بنانے اور کمپائل کرنے کا کام سنبھالتا ہے۔
