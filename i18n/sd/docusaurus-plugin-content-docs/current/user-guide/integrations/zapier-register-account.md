# ایونٹ: Zapier کے ذریعے اکاؤنٹ رجسٹر کرنا

اس آرٹیکل [Integrating Ultimate Multisite with Zapier](zapier.md) میں، ہم نے اس بارے میں بات کی تھی کہ کیسے آپ ٹرگرز اور ایونٹس کی بنیاد پر Ultimate Multisite کے اندر مختلف کام کرنے کے لیے Zapier کا استعمال کر سکتے ہیں۔ اس آرٹیکل میں، ہم دکھائیں گے کہ آپ تھرڈ پارٹی ایپلی کیشنز کو کیسے جوڑ سکتے ہیں۔ ہم ڈیٹا کے ذرائع کے طور پر Google Sheets کا استعمال کریں گے اور معلومات کو Ultimate Multisite کو بھیجیں تاکہ اکاؤنٹ رجسٹر ہو سکے۔

Zapier سے پہلے کنیکٹ کرنے کے لیے، **Ultimate Multisite > Settings > API & Webhooks** پر جائیں اور تصدیق کریں کہ API فعال (enabled) ہے۔ جب Zapier آپ سے Ultimate Multisite اکاؤنٹ کی تفصیلات مانگے تو اسکرین سے API Key اور API Secret کاپی کر لیں۔

![API and Webhooks settings with API Key, API Secret, and Enable API options](/img/admin/settings-api-webhooks.png)

سب سے پہلے، آپ کو اپنے Google Drive کے اندر ایک **Google Sheet** بنانی ہوگی۔ اس بات کو یقینی بنائیں کہ آپ ہر کالم کو ٹھیک سے ڈیفائن کریں تاکہ بعد میں ڈیٹا کو آسانی سے مپ (map) کر سکیں۔

ایک Google sheet بنانے کے بعد، آپ اپنے Zapier اکاؤنٹ میں لاگ ان کر سکتے ہیں اور ایک zap بنانا شروع کر سکتے ہیں۔

**"App event"** کے سرچ فیلڈ کے نیچے **"Google Sheets"** منتخب کریں۔

پھر "**Event** " فیلڈ کے لیے **"New spreadsheet row"** منتخب کریں اور "**Continue** " پر کلک کریں۔

اگلا قدم آپ سے ایک **Google Account** منتخب کرنے کے لیے پوچھے گا جہاں **Google Sheet** محفوظ ہے۔ لہذا، بس یہ یقینی بنائیں کہ صحیح گوگل اکاؤنٹ مخصوص کیا گیا ہے۔

**"Set up trigger** " کے نیچے، آپ کو اس گوگل اسپریڈ شیٹ اور ورکشیت کو منتخب اور بتانا ہوگا جس کا استعمال آپ ڈیٹا حاصل کرنے کے لیے کریں گے۔ بس آگے بڑھیں اور اسے پُر کریں اور "**Continue** " پر کلک کریں۔

اگلا قدم یہ ہے کہ **"test your trigger** " تاکہ یہ یقینی بنایا جا سکے کہ آپ کی گوگل شیت ٹھیک سے منسلک ہے۔

اگر آپ کا ٹیسٹ کامیاب ہو جاتا ہے، تو آپ کو اپنے اسپریڈ شیٹس سے کچھ ویلیوز دکھاتے ہوئے نتیجہ نظر آنا چاہیے، آگے بڑھنے کے لیے "**Continue** " پر کلک کریں۔

نالي نتيجه هي دوسرا عمل جو سیٹ اپ کرنا ہے جس سے Ultimate Multisite ۾ ڪا اڪاونٽ (account) ٺهي وڃي ٿي يا ريجيسٽري هجي. سرچ فيلڊ ۾ "**Ultimate Multisite(2.0.2)** " چونڊ ڪريو.

"**Event**" جي فیلڊ تحت، "**Register an Account in Ultimate Multisite** " چونڊ ڪريو ۽ پوءِ "**Continue** " بٽڻ تي ڪلڪ ڪريو.

"**Set up an action**" تحت، توهان کي ڪسٽمر جي ڊيٽا، ممبرشپ (memberships)، پروڊڪٹس يا ٻين شين لاءِ مختلف فیلڊ ڏسڻ گهرجي ٿي. توهان گوگل شيت ۾ قيمت (values) کي مئپ ڪري سگهو ٿا ۽ انهن کي ان مناسب فیلڊ تي چونڊ ڪري سگهو ٿا جتي انهن کي پوري ڪيو ويندو آهي جهڙوڪ هيٺ ڏنل اسڪريئن ۾ ڏيکيل آهي.

قيمت مئپ ڪرڻ کان پوءِ، توهان عمل (action) کي ٽيسٽ ڪري سگهو ٿا.
