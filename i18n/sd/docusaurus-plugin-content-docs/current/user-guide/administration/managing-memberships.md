# رکنیت (Memberships) کو سنبھالنا

Ultimate Multisite ۾ توهان پنهنجي نٽويڪ (network) تي سڀني سبسڪريپشنز ۽ رسائي کي سنڀال سگهو ٿا. توهان پلانز ۽ پروڊڪٽس، سائٽن جي تعداد، بلنگ جو مقدار ۽ ڪ freqency، دفعي وارو تاريخ، پياملين جي تاريخ ۽ ٽائم اسٽمبس جهڙيون سبسڪريپشن جي تفصيل کي ٻيهر ڏسي سگهو ٿا ۽ ان ۾ تبديلي ڪري سگهو ٿا.

ان صفحتي تي پهچڻ لاءِ، Ultimate Multisite منيو ۾ **Memberships** ٽاب (tab) تي وڃو.

![Memberships list page](/img/admin/memberships-list.png)

Memberships صفحتي تي توهان پنهنجي گهرجي ۽ انهن حسابن سان جڙيل پروڊڪٽس جي فهرست ڏسي سگهو ٿا. توهان حالت (status) جي لحاظ کان رکنيت کي فلٽر ڪري سگهو ٿا: active، pending، on hold، expired، يا cancelled.

فري سبسڪريپشنز کي **lifetime** سبسڪريپشنز طور سمجهيو ويندو آهي. انهن کي ختم ٿيڻ جو ڪو ختم وقت نه ڏني ويندي ۽ اهي خود بخاري نه ٿين. جيڪڏهن ڪا فري پلان ٹرائل يا مقرر وقت کان پوءِ ختم ٿيڻ گهرجي، ته هڪ ٹرائل وارو پيڊ پروڊڪٽ استعمال ڪريو، رکنيت کي هيٺ ماني طور تي ختم ڪريو، يا جڏهن توهان جي رسائي تبديلي ٿي، تڏهن ڪسائي ٻئي پلان تي منتقل ڪريو.

اڪثر رکنيت تي ڪلڪ ڪريو ته ان صفحتي تي وڃو جتي توهان رکنيت جي تفصيل—بلنگ معلومات، سائٽ يا وزٽ جي حد، ۽ انهيءَ طرح جون ٻيا ڳالهيون—تبديلي ڪري سگهو ٿا.

![Membership edit interface](/img/admin/membership-edit.png)

هتي رکنيت جي تبديلي جو پورو ڏوه آهي:

![Membership edit full interface](/img/admin/membership-edit-full.png)

توهان پنهنجي نٽويڪ تي قبضو ڪيل صارفن سان جڙيل نون رکنيتس کي به شامل ڪري سگهو ٿا جيڪي پراني وقت کان توهان جي نٽويڪ تي رجستري ٿيا هئا. **Add Membership** تي ڪلڪ ڪريو ۽ تفصيل مقرر ڪريو.

![Add Membership modal](/img/admin/membership-add-modal.png)

پھر **Create Membership** تي ڪلڪ ڪريو. نون رکنيت Memberships جي فهرست ۾ ظاهر ٿيندي.
