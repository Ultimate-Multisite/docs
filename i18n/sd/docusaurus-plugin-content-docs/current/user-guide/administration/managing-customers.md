# گاہکاں کا انتظام (Managing Customers) {#managing-customers}

**Customers** صفحہ آپ کے نیٹ ورک میں رجسٹرڈ ممبرز کی فہرست دکھاتا ہے۔ اس صفحے پر نام، ای میل ایڈریس، آخری لاگ ان، اور ایک **switch to** لنک ہوتا ہے جو آپ کو سائٹ ایڈمنسٹریٹر کے طور پر صارف کی سب سائٹ میں لاگ ان کرنے دیتا ہے۔

![Customers list page](/img/admin/customers-list.png)

کسی گاہک کے کارڈ کے نیچے **Manage** پر کلک کریں تاکہ اس گاہک کے بارے میں مزید تفصیلات دیکھ سکیں۔

گاہک کی تفصیل والے صفحے پر، **At a Glance** بلاک میں گاہک کا آئی ڈی، آخری لاگ ان کا وقت، اور اس گاہک نے کل کتنا خرچ کیا ہے یہ سب دکھایا جاتا ہے۔

**Customer Options** بلاک آپ کو اس گاہک کو ایک **VIP** (ایک ٹیگ جو انہیں تلاش کرنا آسان بناتا ہے) کے طور پر سیٹ کرنے کی اجازت دیتا ہے۔ **Billing Info** ٹیب میں بلنگ کی معلومات جیسے ای میل، پتہ، اور ملک دکھائی جاتی ہیں۔

**Fields** بلاک میں گاہک کا یوزر نیم، ای میل دکھایا جاتا ہے، اور آپ **Email Verification** کی حیثیت تبدیل کر سکتے ہیں۔

Customer Options بلاک کے نیچے آپ کو چار اضافی بلاکس ملیں گے:

- **Memberships** — وہ موجودہ سبسکرپشنز جن پر گاہک سبسکرائب ہے۔
- **Payments** — گاہک کی ادائیگیوں کا ریکارڈ۔
- **Sites** — وہ سائٹس جن کا مالک یہ گاہک ہے۔
- **Events** — اس گاہک سے متعلق ایونٹس (ای میل تصدیق، بلنگ میں تبدیلیاں وغیرہ)۔

کسی بھی آئٹم کے ساتھ موجود تیر کے نشان (→) پر کلک کریں تاکہ اس کے تفصیل والے صفحے پر جا سکیں۔ مثال کے طور پر، کسی سائٹ کے آگے تیر پر کلک کرنے سے آپ سائٹ ایڈیٹ پیج پر چلے جائیں گے جہاں آپ ممبرشپ کی وابستگی، سائٹ کا قسم اور فعال حیثیت تبدیل کر سکتے ہیں۔

یہ بات ممبرشپ اور ادائیگیوں کے لیے بھی یہی ہے — ہر ایک تفصیل والے صفحات سے لنک ہوتا ہے جہاں آپ ممبرشپ کی حیثیت، متعلقہ گاہک، مصنوعات، پیمنٹ گیٹ وے سیٹنگز، اور یہاں تک کہ ریفنڈز پر عمل کرنے کا انتظام بھی کر سکتے ہیں۔

گاہک کے صفحے کے دائیں جانب آپ کو یہ چیزیں ملیں گی:

- **آخر لاگ ان ۽ IP ها** — آخر ڪڏهن لڳي، IP آڊريس، ۽ ملڪ
- **تايم سٽمپس** — جڏهن ڪلائڻ جي حساب جو ٺهيو هو ۽ آخر ڪڏهن تعديل ڪيو ويو
- **منتقلي (Delete)** — ڪلائڻ کي باقائي طور تي مڪمل طور تي مڪمل ڪرڻ جو چوڻ

:::warning
ڪلائڻ کي حذف ڪرڻ باقائي آهي ۽ ان کي واپس نه ڪري سگهجي ٿو.
:::
