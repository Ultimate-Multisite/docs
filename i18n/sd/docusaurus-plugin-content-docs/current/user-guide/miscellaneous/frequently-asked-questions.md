# کثرت پوڇل (Frequently Asked Questions)

هي ڪجهه **کثرت پوڇل** آهن جيڪي سپورٽ جي درخواستن دوران عام طور تي اچن ٿا.

  * **Ultimate Multisite ڇا آهي؟**

Ultimate Multisite هڪ WordPress plugin آهي، جيڪو WordPress multisite installation لاءِ banaيو ويو آهي، جيڪو توهان جي WordPress installation کي سائيٽن جو هڪ پریميم نڪر (network) بڻائي ٿو – جهڙوڪ [WordPress.com](https://WordPress.com) – جنهن سان ڪلائين ماهمتي، چوتري يا سالي جي فیس تي سائيٽون bana سگهن ٿا (توهان مفت پلانز به bana سگهو ٿا).

  * **ڇا Ultimate Multisite Multisite کان سواءِ ڪم ڪندو؟**

نه. Ultimate Multisite کي WordPress multisite، subdirectory، يا subdomain جي ضرورت آهي.

  * **Ultimate Multisite جا گهرجن (requirements) ڇا آهن؟**

Ultimate Multisite کي مندرج آهي:

**WordPress Version** : v5.3+ (تجاويز: آخري مستحڪم نسخه)
**PHP Version** : 7.4.x (8.0 کان وڌيڪ جي سپورٽ قريب اچڻ وارو آهي)
**MySQL Version** : v5+ (تجاويز: نسخه 5.6، جيڪڏهن توهان کي پنهنجي هاوسنگ پروائيڊر تي 8.0 موجود نه هجي ته)

_Multisite (subdomain يا subdirectory) کي به فعال ڪرڻ جي ضرورت آهي._ [WordPress Multisite ڪيئن نصب ڪجي](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **ڇا Ultimate Multisite Subdirectories سان گڏ Multisit هتي ڪم ڪندو؟**

ها. Ultimate Multisite subdirectory installation ۽ subdirectory installation ٻئي کي سپورٽ ڪري ٿو.

  * **ڇا Ultimate Multisite Subdomains سان گڏ Multisit هتي ڪم ڪندو؟**

ها. Ultimate Multisite subdirectory installation ۽ subdomain installation ٻئي کي سپورٽ ڪري ٿو.

  * **ڇا مان Ultimate Multisite جو استعمال ڪري سگهان ٿو ته جيئن اسان صارفين کي ڪهڙيون Plugins ۽ Themes استعمال ڪرڻ گهرجي ان کي محدود ڪري سگهان؟**

ها. Ultimate Multisite سان توهان پنهنجي نڪر (network) تي هر پلان لاءِ ڪهڙا plugins ۽ themes موجود آهن انهن کي چونڊي سگهو ٿا.

  * **ڇا مان Ultimate Multisite جو استعمال ڪري سگهان ٿو ته جيئن اسان صارف هڪ سائيٽ تي کذوب ڪرڻ جي سمورن پوسٽن جي تعداد کي محدود ڪري سگهون؟**

ہاں۔ Ultimate Multisite کے ساتھ آپ اپنے نیٹ ورک پر موجود ہر دستیاب پوسٹ ٹائپ کے لیے پوسٹ کوٹہ (post quotas) بنا سکتے ہیں۔ آپ ایڈمن کو بلانے والے صارفین کی تعداد، **ڈسک اسپیس کی مقدار،** اور بہت کچھ بھی محدود کر سکتے ہیں۔

* **کیا Ultimate Multisite سائٹ ٹیمپلیٹس (Site Templates) کو سپورٹ کرتا ہے؟**

جی ہاں۔ Ultimate Multisite کے ساتھ آپ جتنے چاہیں Site Templates بنا سکتے ہیں۔ Site Templates تھیمز نہیں ہوتے، یہ آپ کے نیٹ ورک پر عام سائٹس ہوتی ہیں۔ آپ کوئی بھی پلے ان، تھیم اور اس Site Template کو اپنی مرضی کے مطابق کسٹمائز کر سکتے ہیں۔ اگر کوئی صارف وہ ٹیمپلیٹ منتخب کرتا ہے، تو اس کی سائٹ اس ٹیمپلیٹ کی کاپی کے طور پر بنائی جائے گی، جس میں تمام کسٹمائزیشنز موجود ہوں گی۔

آپ کسی خاص پلان کے لیے ایک مخصوص ٹیمپلیٹ بھی سیٹ کر سکتے ہیں۔ ایسا کرنے سے، جو بھی صارف اس پلان پر سائن اپ کرے گا، اس کی سائٹ اس Site Template کی نقل کے طور پر بنائی جائے گی۔

* **کیا Ultimate Multisite WooCommerce کے ساتھ انٹیگریٹ ہوتا ہے؟**

جی ہاں!

اگر آپ چاہتے ہیں کہ آپ کے نیٹ ورک کے صارفین کے لیے WooCommerce دستیاب ہو تاکہ وہ اسٹور سیٹ اپ کر سکیں، تو یہ Ultimate Multisite کا استعمال کرتے ہوئے ممکن ہے۔ آپ یہاں تک بھی محدود کر سکتے ہیں کہ ہر پلان ٹیر پر آپ کے کلائنٹس کتنی مصنوعات بنا سکیں گے۔

Ultimate Multisite کے سبسکرپشن ادائیگیوں کے حوالے سے، آپ [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) کا استعمال کر سکتے ہیں تاکہ اپنے Ultimate Multisite کلائنٹس کو اپنی مین سائٹ پر WooCommerce گیٹ ویز کا استعمال کرتے ہوئے ادائیگی کرنے کی اجازت دے سکیں۔ اس طرح، آپ WooCommerce کے لیے تیار کردہ بہت بڑے پے گیٹ ویز لائبریری سے فائدہ اٹھا سکتے ہیں اور اپنے Ultimate Multisite کلائنٹس کو بلانے کے لیے ان کا استعمال کر سکتے ہیں۔

* **کیا WPMUDEV Pro سائٹس سے سبسکرپشنز منتقل کرنے کا کوئی طریقہ ہے؟**

ہاں، Ultimate Multisite اب آپ کو اپنے Pro Sites نیٹ ورک کے لیے [ایک مددگار ٹول فراہم کرتا ہے جو آپ کو منتقلی کرنے میں مدد دے](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/)۔

  * **میرے Plan Edit Page پر “Allowed Plugins” ٹیب کیوں خالی ہے؟**

Ultimate Multisite صرف ان پلگنز کی اجازت دیتا ہے جو آپ کے نیٹ ورک پر فعال (network-activated) نہ ہوں۔ جب کوئی پلگ ان نیٹ ورک پر فعال ہوتا ہے، تو اپنی فطرت کے مطابق، وہ آپ کے نیٹ ورک کے ہر سائٹ پر ڈیفالٹ طور پر لوڈ ہو جاتا ہے۔

اگر آپ کو “Allowed Plugins” ٹیب پر کوئی پلگ ان نظر نہیں آ رہا ہے، تو چیک کریں کہ کیا مطلوبہ پلگ ان نیٹ ورک پر فعال ہے یا نہیں۔

  * **کیا میں اپنے اپنی قیمتوں کی جدول کا ڈیزائن استعمال کر سکتا ہوں؟**

جی ہاں۔ ورژن 1.4.0 سے، Ultimate Multisite ٹیمپلیٹ اووررائڈز (template overrides) کو سپورٹ کرتا ہے۔

  * **کیا Ultimate Multisite WP Engine کے ساتھ کام کرتا ہے؟**

جی ہاں۔ Ultimate Multisite ہمیشہ WP Engine کے ساتھ سپورٹ میں رہا ہے، لیکن ڈومین 매핑 (domain mapping) کو کام کرنے کے لیے ایڈمن کی طرف سے دستی اضافی اقدامات درکار ہوتے تھے (WP Engine پینل میں ڈومین شامل کرنا)۔ **یہ ورژن 1.6.0 سے اب ایسا نہیں ہے۔** Ultimate Multisite براہ راست WP Engine API کے ساتھ انٹیگریٹ ہوتا ہے اور 매핑 کے بعد ڈومین خود بخود پینل میں شامل ہو جاتے ہیں۔

  * **آپ کس ہوسٹنگ پرووائڈر کی سفارش کرتے ہیں؟**

Ultimate Multisite کسی بھی ہوسٹنگ پرووائڈر کے ساتھ کام کر سکتا ہے۔ واحد فرق یہ ہے کہ کچھ ہوسٹنگ کمپنیاں جیسے Closte، Serverpilot، WPMUDEV، WP Engine، Cloudways، Gridpane اور [Runcloud.io](http://Runcloud.io) میں ڈومین 매핑 کی صلاحیتوں کے لیے زیادہ مضبوط انٹیگریشن ہوتی ہے اس لیے وہ بغیر کسی اضافی کام کے ہی کام کرتے ہیں۔ دیگر ہوسٹنگز کے لیے، آپ کو ڈومین 매핑 دستی طور پر سیٹ اپ کرنا پڑ سکتا ہے۔
