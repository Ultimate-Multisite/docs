# تھیم کے مطابق مہارتیں {#theme-aware-skills}

Superdav AI Agent v1.10.0 چار نئی تھیم سے واقف ہونے والی ان بلٹ-ان مہارتوں کو لانچ کرتا ہے جو آپ کی فعال WordPress تھیم کے مطابق خود بخود ایڈجسٹ ہوتی ہیں۔ یہ مہارتیں آپ کی تھیم کی ساخت اور خصوصیات کے مطابق مخصوص رہنمائی اور صلاحیتیں فراہم کرتی ہیں۔

## تھیم سے واقف مہارتیں کیا ہیں؟ {#what-are-theme-aware-skills}

تھیم سے واقف مہارتیں پہلے سے ترتیب دی گئی معلومات کی بنیاد پر بنائے گئے علم کے ذخیرے (knowledge bases) اور ٹول سیٹ ہوتے ہیں جنہیں AI اسسٹنٹ آپ کی سائٹ پر فی الحال فعال WordPress تھیم کی بنیاد پر خود منتخب کرتا ہے۔ جب آپ تھیمز بدلتے ہیں، تو اسسٹنٹ کی دستیاب مہارتیں خود بخود اپ ڈیٹ ہو جاتی ہیں — کسی دستی ترتیب کی ضرورت نہیں ہوتی۔

ہر مہارت میں شامل ہوتا ہے:

- **تھیم کے مخصوص دستاویزات** — تھیم کو استعمال کرنے اور اپنی مرضی کے مطابق بنانے کے بارے میں رہنمائی
- **بلاک اور پیٹرن کے حوالے** — دستیاب بلاکس، پیٹرنز، اور ڈیزائن کے اختیارات
- **تخصیص کی مثالیں** — عام کاموں کے لیے کوڈ کے ٹکڑے اور کنفیگریشن کے طریقے
- **بہترین طریقے** — تھیم کی ساخت اور ورک فلو کے لیے سفارشات

## دستیاب تھیم سے واقف مہارتیں {#available-theme-aware-skills}

### بلاک تھیمز (Block Themes) {#block-themes}

**لائیڈ کرتا ہے:** ان تھیمز پر جو WordPress بلاک پر مبنی (Full Site Editing) آرکیٹیکچر استعمال کرتے ہیں۔

بلاک تھیمز کی مہارت یہ رہنمائی فراہم کرتی ہے:

- بلاک ایڈیٹر کا استعمال کرتے ہوئے ٹیمپلیٹس بنانا اور ایڈٹ کرنا
- بلاک پیٹرنز اور دوبارہ استعمال ہونے والے بلاکس کے ساتھ کام کرنا
- عالمی اسٹائلز (global styles) اور theme.json سیٹنگز کو اپنی مرضی کے مطابق بنانا
- تھیم بلاکس اور ان کی مختلف شکلوں کا استعمال کرنا
- اپنی سائٹ کے لیے کسٹم بلاک پیٹرنز بنانا

**خود بخود فعال ہوتے ہیں جب:** آپ کی فعال تھیم ایک بلاک تھیم ہو (یہ `block-templates` فیچر کو سپورٹ کرتی ہے)۔

### کلکیک تھیمز (Classic Themes) {#classic-themes}

**لائیڈ کرتا ہے:** روایتی WordPress تھیمز جو PHP ٹیمپلیٹس اور کلکیک ایڈیٹر استعمال کرتے ہیں۔

کلکیک تھیمز کی مہارت یہ رہنمائی فراہم کرتی ہے:

PHP template files ۽ hooks سان کام ڪرڻ
Customizer جي ذريعي ٿيم جو ڏسڻ ۾ تبديلي ڪرڻ
Widget areas ۽ sidebars جو استعمال
CSS کي مڪمل ڪرڻ ۽ child theme جو ڊولپمينٽ
Theme hierarchy ۽ template tags جو سمجهڻ

**جيڪڏهن خود بخود فعال ٿئي ٿو ته:** توهان جو فعال theme classic (non-block) theme هجي.

### Kadence Blocks {#kadence-blocks}

**ڪهڙي لاءِ لاڳو آهي:** اهڙا سائٽس جيڪي advanced block-based design لاءِ Kadence Blocks plugin استعمال ڪن.

Kadence Blocks جو اسڪل هيٺ ڏينهن گهجهه رهنمائي پيش ڪري ٿو:

- Kadence جي advanced block library (Hero, Testimonials, Pricing, etc.) جو استعمال
- Kadence block settings ۽ responsive options جو configuration
- Kadence جي grid ۽ container blocks سان custom layouts جو ٺهيل ڪرڻ
- Kadence blocks کي توهان جي theme سان integrate ڪرڻ
- Kadence جي design system ۽ presets جو فائدو وٺڻ

**جيڪڏهن خود بخود فعال ٿئي ٿو ته:** Kadence Blocks plugin توهان جي سائيٽ تي active هجي.

### Kadence Theme {#kadence-theme}

**ڪهڙي لاءِ لاڳو آهي:** اهڙا سائٽس جيڪي block-based design ۽ customization لاءِ Kadence theme استعمال ڪن.

Kadence Theme جو اسڪل هيٺ ڏينهن گهجهه رهنمائي پيش ڪري ٿو:

- global styles ۽ theme.json جي ذريعي Kadence theme جو customization
- Kadence جي built-in block patterns ۽ templates جو استعمال
- Kadence theme settings ۽ options جو configuration
- Kadence جي design system سان custom designs جو ٺهيل ڪرڻ
- popular plugins ۽ tools سان Kadence جو integration

**جيڪڏهن خود بخود فعال ٿئي ٿو ته:** Kadence theme توهان جو active theme هجي.

## Skills ڪيئن چونڊيا ويندا آهن {#how-skills-are-selected}

Assistant هر message تي توهان جي active theme ۽ install ڪيل plugins کي خود موڪل ڪري ٿو. جيڪڏهن ڪا match هلي واري theme-aware skill موجود هجي، ته اهو خود بخود assistant جي context ۾ لادو ٿيندي آهي. توهان کي skills کي manual طور تي enable ڪرڻ يا switch ڪرڻ جي ضرورت نه آهي.

### Multiple Skills {#multiple-skills}

جيڪڏهن توهان جي سائيٽ کي ڪيترن ئي صلاحيتون لاڳو ٿيل آهن (مثال طور، جيڪڏهن توهان Kadence Blocks ۽ Kadence Theme ٻئي فعال رکڻ ۾ آهي)، ته اسان جو اسست (assistant) تمام صلاحيتون حاصل ڪري سگهي ٿو ۽ هر هڪ کان رهنمائي حاصل ڪري سگهي ٿو.

### ٿيم کي تبديل ڪرڻ (Switching Themes) {#switching-themes}

جڏهن توهان پنهنجي فعال ٿيڻ واري ٿيم کي تبديل ڪندا آهيو، ته اسان جو اسست جي موجود صلاحيتون هيٺ ڏنل پيغام تي خود بخود اپڊيٽ ٿين وينديون. مثال طور:

1. توهان **Block Themes** صلاحيت فعال رکڻ وارو هڪ بلاڪ ٿيم استعمال ڪري رهيا آهيو.
2. توهان ڪلاسي ٿيم (classic theme) تي تبديل ٿين ٿا.
3. پنهنجي هيٺيون پيغام ۾، **Classic Themes** صلاحيت خود بخود لودي ويندي آهي، ۽ **Block Themes** صلاحيت اب موجود نه هجي.

## ٿيم-سمجهڻ واري صلاحيتون استعمال ڪرڻ (Using Theme-Aware Skills) {#using-theme-aware-skills}

ڪو هميشه ٿيم سمجهڻ واري صلاحيت کي استعمال ڪرڻ لاءِ، بس چٽي ڪيو ته توهان ڇا ڪرڻ چاهيو ان چٽي رابطي جو حصي ۾. اسست خود بخود مناسب صلاحيت جي رهنمائي طرف رجوع ڪندو.

### مثال پيغام (Example Prompts) {#example-prompts}

**Block Themes لاءِ:**
> "بلاڪ پينٽرن استعمال ڪندي هڪ هيرو سيكشن بنائو جنهن پس منظر جو تصوير ۽ سنڌي متن واري هجي."

**Classic Themes لاءِ:**
> "چيلڊ ٿيم (child theme) استعمال ڪري سائیڊبين کي هڪ ڪسمائن وِجٽ ايريا ڏيو."

**Kadence Blocks لاءِ:**
> "Kadence Testimonials بلاڪ جو استعمال ڪندي هڪ تجربن جو حصو بنائو."

**Kadence Theme لاءِ:**
> "هيدر جي لاءُ آؤٽٽ ۽ نيويجي مينيءُ جي سٽلنگ کي ترقي ڏيو."

اسست توهان جي فعال ٿيڻ واري ٿيم ۽ پنهنجي پلگينز لاءِ مخصوص رهنمائي ۽ ڪوڊ جا مثال فراهم ڪندو.

:::note
Theme-aware صلاحيتون Superdav AI Agent v1.10.0 ۽ ان کان بعد خود بخود موجود هونديون آهن. ڪا به اضافي ترتیب يا ڪيفيڪيشن جي ضرورت نه آهي.
:::
