# پلانز لاءِ شیئر ایبل لنکس سنڀالڻ {#managing-shareable-links-for-plans}

Ultimate Multisite توهان کي پنهنجي ڪلائين سان پلان لينڪس direktly شيئر ڪرڻ جو چوڻ ڏئي ٿو. ان لاءِ، پنهنجي ڪنهن به پلان جي edit pages تي وڃو ۽ _Click to copy the Shareable Link_ بٽون تي ڪلڪ ڪريو.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_Plan جي Edit Page تي، توهان کي شيئر ایبل لنکس کڻڻ لاءِ بٽون ملي ويندي آهي_

ان ڳالهه جو ڌيان رکجو ته هي ميڪانيزم لنک کي بنائڻ لاءِ default billing frequency جو استعمال ڪري ٿو. ان جو مطلب اهو آهي ته جيڪڏهن توهان جي default billing ماهي (monthly) آهي، مثال طور تي، آخر ۾ توهان کي هيٺ ڏنل طرح جو لنڪ ملندو: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

جيڪڏهن توهان چاهيو ته شيئر ایبل لنڪ quarterly (توهاني) بڻجي، تنهنجي آخر ۾ 3 شامل ڪرڻو پوندو:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Plan _premium_ کي yearly (سال وار) شيڊول تي چونڊڻ لاءِ، توهان 1 کان پوءِ هي لفظ شامل ڪندا:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

يه رکجو ته توهان مختلف billing frequencies لاءِ صرف هڪ لنڪ نه بنا سکدا. توهان کي پنهنجي هر billing frequency لاءِ هڪ خاص لنڪ شيئر ڪرڻ گهرجي.

جڏهن به توهان رابطين سان لنڪ شيئر ڪندا، اهو انهن کي توهان جي signup flow جو پهريون قدم تي ريڊيڪٽ ڪري ڇڏيندو.
