# ملٽي-نيٽ ورڪ تبديلي نامو

### 1.3.0 {#130}
* نئون: نيٽ ورڪ ٽيمپليٽ پري ويوئر — خريداري کان اڳ لائيو پينل سان نيٽ ورڪ ٽيمپليٽس کي برائوز ۽ پريويو ڪريو.
* نئون: checkout تي نيٽ ورڪ ٽيمپليٽ چونڊيندڙ — گراهڪ signup checkout وهڪري دوران نيٽ ورڪ ٽيمپليٽ چونڊي سگهن ٿا.
* نئون: ڪراس-نيٽ ورڪ ميجڪ لنڪ SSO — گراهڪ magic-link تصديق ذريعي sub-networks تائين بنا رنڊڪ رسائي حاصل ڪري سگهن ٿا.
* نئون: نيٽ ورڪ ميڊيا ڪلوننگ — ٽيمپليٽ مان نيٽ ورڪ ڪلون ڪرڻ وقت سائيٽ ميڊيا صحيح نموني نقل ٿئي ٿي.
* درستگي: نيٽ ورڪ ڪلوننگ هاڻي multi-tenancy addon کان سواءِ ڪم ڪري ٿي؛ super admin اختيار ڪلون کان پوءِ ٻيهر ڏنا وڃن ٿا.
* درستگي: نيٽ ورڪ ٺاهڻ هاڻي WooCommerce ۽ سڀني payment gateways سان مطابقت رکي ٿو.
* درستگي: fallback domain ٺهڻ ۾ هاڻي پورٽ نمبر شامل آهي.
* درستگي: ٽيمپليٽ چونڊيندڙ CSS سڀني themes تي صحيح rendering لاءِ enqueued stylesheet ڏانهن منتقل ڪيو ويو.
* درستگي: سائيٽ query scope ۾ اڻ-تعريف ٿيل `network_id` key کان بچاءُ.
* درستگي: context switching کان اڳ نيٽ ورڪ اجازت چيڪ `wpdb` table property errors کي روڪي ٿو.
