# عمل: move_site

تڏهن هلندو آهي جڏهن ڪا سائيٽ نئين نيٽ ورڪ ڏانهن منتقل ڪئي وئي هجي.

## پيرا ميٽر {#parameters}

| نالو | قسم | وضاحت |
|------|------|-------------|
| $site_id | `int` | ان سائيٽ جو ID جيڪا منتقل ڪئي وئي آهي. |
| $old_network_id | `int` | سائيٽ لاءِ اصل نيٽ ورڪ جو ID. |
| $new_network_id | `int` | ان نيٽ ورڪ جو ID جنهن ڏانهن سائيٽ منتقل ڪئي وئي آهي. |
