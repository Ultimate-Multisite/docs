# گهڻ-ڪرنسي

Ultimate Multisite: گهڻ-ڪرنسي توهان جي نيٽ ورڪ کي ڪيترين ڪرنسين ۾ ادائگيون قبول ڪرڻ جي قابل بڻائي ٿو. اهو زندهه يا دستي مٽاسٽا جي شرح استعمال ڪري قيمتن کي پاڻمرادو تبديل ڪري ٿو، توهان کي ڪنهن به ڪرنسي لاءِ هر پراڊڪٽ جون صحيح قيمتون مقرر ڪرڻ ڏئي ٿو، ۽ جغرافيائي مقام ذريعي هر مهمان جي پسنديده ڪرنسي سڃاڻي ٿو — سڀ ڪجهه هڪ ئي انتظامي Dashboard مان سنڀاليو وڃي ٿو.

## اهم خاصيتون {#key-features}

- **پاڻمرادو مٽاسٽا جي شرح جي تبديلي** — قيمتون توهان جي بنيادي ڪرنسي مان انهن شرحن سان تبديل ٿين ٿيون جيڪي توهان جي چونڊيل فراهم ڪندڙ کان حاصل ڪيون وڃن ٿيون
- **هر پراڊڪٽ لاءِ قيمت جا اوور رائيڊ** — پاڻمرادو تبديلي کي نظرانداز ڪرڻ لاءِ ڪنهن به پراڊڪٽ تي ڪرنسي-مخصوص صحيح قيمتون مقرر ڪريو
- **ڪرنسي چونڊيندڙ رجسٽريشن فيلڊ** — گراهڪن کي چيڪ آئوٽ/رجسٽريشن دوران پنهنجي پسنديده ڪرنسي چونڊڻ ڏئي ٿو
- **ڪيترائي مٽاسٽا جي شرح فراهم ڪندڙ** — ECB، Frankfurter، Stripe، يا مڪمل طور دستي شرحون
- **جغرافيائي مقام تي ٻڌل ڪرنسي جي سڃاڻپ** — مهمان جي ملڪ سان ملندڙ ڪرنسي پاڻمرادو اڳواٽ چونڊي ٿو
- **انتظامي Dashboard** — فعال ڪرنسين، مٽاسٽا جي شرحن، ۽ فراهم ڪندڙ جي سيٽنگن کي هڪ هنڌان سنڀاليو

## گهرجون {#requirements}

- Ultimate Multisite 2.0.0 يا وڌيڪ
- WordPress Multisite انسٽاليشن
- PHP 7.4 يا وڌيڪ

## انسٽاليشن {#installation}

1. پلگ ان جون فائلون `/wp-content/plugins/ultimate-multisite-multi-currency` تي اپ لوڊ ڪريو
2. WordPress ۾ **پلگ انز** اسڪرين ذريعي پلگ ان فعال ڪريو (نيٽ ورڪ فعال ڪريو)
3. انتظامي Dashboard کولڻ لاءِ **Ultimate Multisite → گهڻ-ڪرنسي** ڏانهن وڃو
4. اهي ڪرنسيون شامل ڪريو جيڪي توهان قبول ڪرڻ چاهيو ٿا ۽ پنهنجو مٽاسٽا جي شرح فراهم ڪندڙ ترتيب ڏيو

## ترتيب {#configuration}

### انتظامي Dashboard {#admin-dashboard}

گهڻ-ڪرنسي انتظامي Dashboard (**Ultimate Multisite → گهڻ-ڪرنسي**) مرڪزي هنڌ آهي جتي توهان:

- ڪرنسيون شامل، فعال، يا غير فعال ڪري سگهو ٿا
- مٽاسٽا جي شرح فراهم ڪندڙ چونڊي سگهو ٿا ۽ ضرورت پوڻ تي شرحون تازيون ڪري سگهو ٿا
- سڀني فعال ڪرنسين لاءِ موجوده شرحون ڏسي سگهو ٿا
- جڏهن پاڻمرادو فراهم ڪندڙ مناسب نه هجن ته دستي شرحون مقرر ڪري سگهو ٿا

### مٽاسٽا جي شرح فراهم ڪندڙ {#exchange-rate-providers}

| فراهم ڪندڙ | وضاحت |
|---|---|
| **ECB** | يورپي مرڪزي بئنڪ — EUR تي ٻڌل نيٽ ورڪن لاءِ مفت روزاني شرحون |
| **Frankfurter** | اوپن سورس ECB آئينو API — مفت، API چاٻي گهربل ناهي |
| **Stripe** | Stripe جون زندهه FX شرحون استعمال ڪري ٿو — فعال Stripe انٽيگريشن گهربل آهي |
| **دستي** | توهان شرحون سڌو داخل ڪريو ٿا؛ ٻاهريون API ڪالون ناهن |

پنهنجو فراهم ڪندڙ **Ultimate Multisite → گهڻ-ڪرنسي → سيٽنگون → مٽاسٽا جي شرح فراهم ڪندڙ** هيٺ چونڊيو. شرحون ڪنهن به وقت دستي طور تازيون ڪري سگهجن ٿيون يا پاڻمرادو اپڊيٽ ٿيڻ لاءِ شيڊول ڪري سگهجن ٿيون.

### هر پراڊڪٽ لاءِ قيمت جا اوور رائيڊ {#per-product-pricing-overrides}

جڏهن پاڻمرادو تبديلي ڪافي درست نه هجي، ته توهان ڪنهن به پراڊڪٽ تي مخصوص ڪرنسي لاءِ صحيح قيمت مقرر ڪري سگهو ٿا:

1. پراڊڪٽ ايڊيٽر کوليو (**Ultimate Multisite → پراڊڪٽس → ترميم ڪريو**)
2. **ڪرنسي قيمت بندي** حصي تائين اسڪرول ڪريو
3. هر انهيءَ ڪرنسي لاءِ صحيح قيمت داخل ڪريو جنهن کي توهان اوور رائيڊ ڪرڻ چاهيو ٿا
4. پراڊڪٽ محفوظ ڪريو

جن پراڊڪٽس لاءِ اوور رائيڊ ناهي، اهي توهان جي بنيادي ڪرنسي مان تبديل ٿيل قيمت استعمال ڪن ٿا.

### ڪرنسي چونڊيندڙ رجسٽريشن فيلڊ {#currency-selector-signup-field}

ڪرنسي چونڊيندڙ فيلڊ توهان جي رجسٽريشن/چيڪ آئوٽ وهڪري ۾ شامل ڪري سگهجي ٿو ته جيئن گراهڪ رجسٽريشن وقت پنهنجي پسنديده ڪرنسي چونڊين:

1. **Ultimate Multisite → گهڻ-ڪرنسي → سيٽنگون** ڏانهن وڃو
2. **رجسٽريشن تي ڪرنسي چونڊيندڙ ڏيکاريو** فعال ڪريو
3. اختياري طور هڪ ڊفالٽ ڪرنسي مقرر ڪريو يا جغرافيائي مقام کي اڳواٽ چونڊيل قدر طئي ڪرڻ ڏيو

چونڊيل ڪرنسي سبسڪرپشن سان محفوظ ڪئي وڃي ٿي ۽ سڀني بعد وارين بلنگ لاءِ استعمال ٿئي ٿي.

### جغرافيائي مقام تي ٻڌل ڪرنسي جي سڃاڻپ {#geolocation-based-currency-detection}

جڏهن جغرافيائي مقام جي سڃاڻپ فعال هجي، addon مهمان جي ملڪ کي ڪرنسي سان ميپ ڪري ٿو ۽ ان کي ڪرنسي چونڊيندڙ فيلڊ ۾ اڳواٽ چونڊي ٿو. فعال ڪرڻ لاءِ:

1. **Ultimate Multisite → گهڻ-ڪرنسي → سيٽنگون** ڏانهن وڃو
2. **مهمان جي مقام مان ڪرنسي سڃاڻيو** فعال ڪريو
3. **جغرافيائي مقام ميپنگ** ٽيبل ۾ ملڪن کي ڪرنسين سان ميپ ڪريو

جيڪڏهن ڪنهن مهمان جي ملڪ لاءِ ڪا ميپنگ موجود نه هجي، ته نيٽ ورڪ جي بنيادي ڪرنسي fallback طور استعمال ڪئي ويندي آهي.

## اڪثر پڇيا ويندڙ سوال {#frequently-asked-questions}

### جيڪڏهن مٽاسٽا جي شرح فراهم ڪندڙ دستياب نه هجي ته ڇا ٿيندو؟ {#what-happens-if-an-exchange-rate-provider-is-unavailable}

addon آخري ڀيرو ڪاميابي سان حاصل ڪيل شرحن ڏانهن موٽي ٿو. جيڪڏهن ڪڏهن به شرحون حاصل نه ڪيون ويون هجن، ته اهو توهان جي ترتيب ڏنل دستي شرحن ڏانهن موٽي ٿو. حفاظت جي جال طور دستي شرحون ترتيب ڏيڻ جي صلاح ڏني وڃي ٿي.

### ڇا مان مختلف ڪرنسين لاءِ مختلف فراهم ڪندڙ استعمال ڪري سگهان ٿو؟ {#can-i-use-different-providers-for-different-currencies}

نه — سڀني ڪرنسين لاءِ هڪ وقت ۾ هڪ فراهم ڪندڙ فعال هوندو آهي. جيڪڏهن توهان کي انفرادي شرحن تي مڪمل ڪنٽرول گهرجي ته **دستي** استعمال ڪريو.

### ڇا جغرافيائي مقام لاءِ الڳ پلگ ان گهربل آهي؟ {#does-geolocation-require-a-separate-plugin}

ڪو ٻاهريون جغرافيائي مقام پلگ ان گهربل ناهي. addon WordPress جي اندروني IP-کان-ملڪ سڃاڻپ استعمال ڪري ٿو. وڌيڪ درستگي لاءِ، مهيا ڪيل filter hooks ذريعي هڪ مخصوص جغرافيائي مقام خدمت ضم ڪري سگهجي ٿي.

### ڇا هر پراڊڪٽ جا اوور رائيڊ ٽيڪس کان اڳ لاڳو ٿين ٿا يا پوءِ؟ {#are-per-product-overrides-applied-before-or-after-tax}

هر پراڊڪٽ جا اوور رائيڊ هدف ڪرنسي ۾ ٽيڪس کان اڳ واري قيمت مقرر ڪن ٿا. ٽيڪس توهان جي موجوده ٽيڪس ترتيب استعمال ڪندي اوور رائيڊ قيمت جي مٿان حساب ڪيو ويندو آهي.
