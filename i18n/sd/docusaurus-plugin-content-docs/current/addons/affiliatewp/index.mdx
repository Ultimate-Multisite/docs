# AffiliateWP {#affiliatewp}

Multisite Ultimate: AffiliateWP انٽيگريشن

پنهنجي Ultimo Network جي گراهڪن جو بنياد وڌائڻ لاءِ طاقتور AffiliateWP استعمال ڪريو!

## انسٽاليشن {#installation}

1. 'ultimate-multisite-git ' کي '/wp-content/plugins/' ڊائريڪٽري ۾ اپلوڊ ڪريو
2. WordPress ۾ 'Plugins' مينيو ذريعي پلگ ان کي فعال ڪريو
3. پلگ ان سيٽ اپ ڪرڻ لاءِ قدم بہ قدم Wizard جي پيروي ڪريو

## استعمال جي رهنمائي {#usage-guide}

[AffiliateWP](https://affiliatewp.com/) توهان جي آن لائن ڪاروبار لاءِ افليئيٽ پروگرام ٺاهڻ وقت پسنديده حل بڻجي چڪو آهي. Ultimate Multisite: AffiliateWP اهو پل آهي جيڪو توهان کي پنهنجي پريميئم نيٽ ورڪ لاءِ ساڳي شيءِ ٺاهڻ جي قابل بڻائيندو.

## شروع ڪرڻ {#getting-started}

سڀ کان پهرين، توهان وٽ [**AffiliateWP**](https://affiliatewp.com/) **فعال هجڻ لازمي آهي**! جيڪڏهن توهان وٽ **اهو پلگ ان ناهي جيڪو ان سان انٽيگريٽ ٿئي ٿو** ته توهان اسان جو ايڊ-آن استعمال نه ڪري سگهندا. جيڪڏهن توهان پنهنجي گراهڪن کي به انهن جي سائيٽن تي ان کي استعمال ڪرڻ جي اجازت ڏيڻ جو ارادو نٿا رکو، ته AffiliateWP صرف توهان جي مکيه سائيٽ تي فعال هجڻ گهرجي.

ان سان گڏ، توهان کي **Ultimate Multisite** ۽ **Ultimate Multisite: AffiliateWP** ٻنهي کي **network-wide** **فعال** ڪرڻو پوندو.

## Ultimate Multisite کي AffiliateWP انٽيگريشن طور شامل ڪرڻ {#adding-ultimate-multisite-as-an-affiliatewp-integration}

پنهنجي مکيه سائيٽ تي، افليئيٽس سيٽنگن ڏانهن وڃو، ۽ انٽيگريشنز ٽيب تي، Ultimate Multisite کي چيڪ ڪريو. هي [AffiliateWP](https://affiliatewp.com/) کي ٻڌائيندو ته Ultimate Multisite سائن اپ فلو مان ايندڙ وزٽن ۽ ڪنورشنز کي ٽريڪ ڪرڻ شروع ڪري.

<!-- اسڪرين شاٽ دستياب ناهي: AffiliateWP انٽيگريشنز سيٽنگز ٽيب Ultimate Multisite چيڪ باڪس سان -->

_هيٺان Ultimate Multisite باڪس چيڪ ڪريو ۽ محفوظ ڪريو_

اهو ڪرڻ لاءِ لڳ ڀڳ ايترو ئي ڪافي آهي. ريفرل لنڪس مان ايندڙ وزٽس ٽريڪ ڪيا ويندا ۽ جڏهن اهو يوزر پوءِ پنهنجي پهرين ادائيگي ڪندو، ته ڪنورشن ٺاهي ويندي ۽ ڪنورشن لاءِ ذميوار افليئيٽ لاءِ ڪميشن الڳ رکي ويندي.

بهرحال، نئين ٺهيل يوزر کان ملي صرف پهرين ادائيگي ٽريڪ ڪئي ويندي. بار بار ٿيندڙ ريفرل ادائيگيون فعال ڪرڻ لاءِ، توهان کي [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) جي ضرورت پوندي

## بار بار ٿيندڙ ريفرلز فعال ڪرڻ {#enabling-recurring-referrals}

[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) انسٽال ڪرڻ کان پوءِ، افليئيٽس سيٽنگز Recurring Referrals ٽيب ڏانهن وڃو، ۽ بار بار ٿيندڙ ريفرلز جو استعمال فعال ڪريو.

<!-- اسڪرين شاٽ دستياب ناهي: AffiliateWP Recurring Referrals سيٽنگز Enable Recurring Referrals ٽوگل سان -->

_جيئن سادو ٿي سگهي ٿو: Enable Recurring Referrals فعال ڪريو ۽ توهان مڪمل طور تي تيار آهيو!_

وري به، اهو ئي سڀ ڪجهه آهي!

هاڻي، هر ڀيري جڏهن ريفر ٿيل Account تي نئين ادائيگي ملي ٿي، ذميوار افليئيٽ کي پنهنجو حصو پاڻمرادو ملي ويندو!
