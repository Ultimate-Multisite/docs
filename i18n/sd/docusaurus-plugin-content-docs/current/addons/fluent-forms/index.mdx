# Fluent Forms

هي add-on Multisite Ultimate ۾ مصنوعات، رڪنيتن يا سائيٽن تي Fluent Forms لاءِ حدون شامل ڪرڻ ممڪن بڻائي ٿو.
Fluent Forms custom post types جي بدران custom tables استعمال ڪري ٿو، تنهنڪري عام حد لڳائڻ وارا اختيار Fluent Forms سان ڪم نٿا ڪن.

## خاصيتون {#features}

- Fluent Forms يا Conversational Fluent Forms لاءِ حدون شامل ڪريو
- گراهڪن کي حد تي پهچڻ وقت پنهنجو plan upgrade ڪرڻ لاءِ هڪ دوستانه پيغام ملي ٿو
