# فلٽر: signup_get_available_languages {#filter-signupgetavailablelanguages}

فرنٽ-اينڊ سائيٽ سائن اپس لاءِ دستياب ٻولين جي فهرست کي فلٽر ڪري ٿو.

هن hook ڏانهن خالي array موڪلڻ سان signup form تي setting جي آئوٽ پُٽ غير فعال ٿي ويندي، ۽ سائيٽ ٺاهڻ وقت ڊيفالٽ ٻولي استعمال ڪئي ويندي. جيڪي ٻوليون اڳ ۾ انسٽال ٿيل نه آهن، اهي هٽايون وينديون.

## پيرا ميٽر {#parameters}

| نالو | قسم | وضاحت |
|------|------|-------------|
| $available_languages | `array` | دستياب ٻوليون. |
