# ٻولي چونڊيندڙ

Multisite Ultimate: سائن اپ تي ٻولي چونڊيندڙ

سائن اپ وهڪري ۾ ٻولي چونڊڻ جو خانو شامل ڪريو.

## انسٽاليشن

1. 'ultimate-multisite-language-selector' کي '/wp-content/plugins/' ڊائريڪٽري ۾ اپلوڊ ڪريو
2. پنهنجي WordPress نيٽورڪ ايڊمن ۾ 'پلگ انز' مينيو ذريعي پلگ ان کي فعال ڪريو

## استعمال جي رهنمائي

هن مفت اضافي سان توهان پنهنجي استعمال ڪندڙن کي سائن اپ وهڪري ۽ لاگ ان صفحي تي اها ٻولي چونڊڻ ڏئي سگهو ٿا جيڪا اهي ڏسڻ چاهين ٿا.

[پنهنجو لائسنس فعال ڪرڻ](../../user-guide/getting-started/installing-ultimate-multisite.md) کان پوءِ توهان اهي Ultimate Multisite اضافا انسٽال ڪري سگهندا جن تائين توهان جو لائسنس توهان کي رسائي ڏئي ٿو.

پنهنجي Ultimate Multisite سيٽنگز صفحي تي وڃو ۽ پوءِ، کاٻي پاسي واري مينيو ۾، _اضافا_ ليبل تي ڪلڪ ڪريو.

![اضافن جو صفحو](/img/admin/addons-page.png)

_ٻولي چونڊيندڙ انسٽال ڪرڻ شروع ڪرڻ لاءِ اضافن جي صفحي تي وڃو_

اتي، توهان کي اسان جا سڀ اضافا ملندا.

مفت اضافا اسان جي سڀني گراهڪن لاءِ موجود آهن، ڀلي انهن ڪهڙي به سبسڪرپشن رٿا لاءِ سائن اپ ڪيو هجي.

اضافي جي ٿمب نيل هيٺان _هاڻي انسٽال ڪريو!_ بٽڻ تي ڪلڪ ڪريو. پوءِ Plugins صفحي تي ان کي فعال ڪريو.

![انسٽال بٽڻ سان اضافن جو صفحو](/img/admin/addons-page.png)

_اضافو انسٽال ڪرڻ لاءِ بٽڻ تي ڪلڪ ڪريو_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_ان کي پنهنجي Plugins صفحي تي فعال ڪريو_

هاڻي، Ultimate Multisite جي سيٽنگز مينيو ڏانهن وڃو ۽ _نيٽورڪ سيٽنگز_ ٽيب ۾ داخل ٿيو. اتي، صفحي جي تري ۾، توهان کي _ٻوليون_ نالي هڪ نئون سيڪشن ملندو.

![سيٽنگز صفحو](/img/config/settings-general.png)

_هڪ بلڪل نئون سيڪشن ظاهر ٿئي ٿو!_

توهان ان کي خالي ڇڏي سگهو ٿا ته جيئن سڀ ٻوليون اختيارن طور ڏيکارجن يا چونڊي سگهو ٿا ته ڪهڙيون موجود ڪرڻ چاهيو ٿا. ٻوليون اهي آهن جيڪي توهان پنهنجي WordPress سيٽ اپ ۾ انسٽال ڪيون آهن.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_هن WordPress انسٽال تي موجود ٻولين جي فهرست_

توهان ڪيل تبديليون محفوظ ڪرڻ کان پوءِ، توهان چونڊيل ٻولين سان هڪ چونڊ مينيو Login صفحي تي ۽ Sign-up Flow تي پڻ ڏيکاريو ويندو.

![لاگ ان ۽ رجسٽريشن سيٽنگز](/img/config/settings-login-registration.png)

_لاگ ان صفحي تي ٻولي چونڊيندڙ_

![رجسٽريشن صفحو](/img/config/registration-page.png)

_سائن اپ وهڪري تي ٻولي چونڊيندڙ_

## URL ذريعي ٻولي مقرر ڪرڻ

توهان URL ۾ locale پيرا ميٽر موڪلي به ڪنهن صفحي جي ٻولي تبديل ڪري سگهو ٿا. مثال طور، جيڪڏهن توهان اطالوي ۾ رجسٽر صفحي ڏانهن ريڊائريڪٽ ڪرڻ چاهيو ٿا، ته URL ۾ ?locale=it_IT شامل ڪريو.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

هي ڪم ڪرڻ لاءِ اها ٻولي توهان جي WordPress نيٽورڪ تي انسٽال ٿيل هجڻ گهرجي.
