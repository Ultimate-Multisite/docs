# Ultimate AI Connector for Compatible Endpoints تبديلي نامو

## Version 2.0.0 — 2026-04-24 تي جاري ڪيو ويو {#version-200--released-on-2026-04-24}

* نئون: گھڻن مهياڪارن جي حمايت — ڪيترائي AI endpoint ترتيب ڏيو ۽ مهياڪارن وچ ۾ خودڪار fallback سان درخواستون route ڪريو.
* درستگي: صحيح مهياڪار IDs، registration URLs، ۽ هر مهياڪار لاءِ HTTP filter scoping سان گھڻن مهياڪارن جي SDK integration.
* درستگي: نئون مهياڪار card هاڻي شامل ڪرڻ تي پاڻمرادو کلي ٿو؛ plugin update تي script cache busting.
* درستگي: eval() لاءِ dynamic provider class namespace ۽ نون شامل ڪيل مهياڪارن لاءِ پاڻمرادو کلڻ وارو رويو.
* درستگي: WordPress 6.9+ compatibility لاءِ مستحڪم Card/CardBody/CardHeader/CardDivider components استعمال ڪري ٿو (هاڻي experimental نه آهن).
* درستگي: مهياڪار drag-to-reorder لاءِ undefined DragHandle کي unicode grip icon سان تبديل ڪيو ويو.
* درستگي: هر page load تي هلندڙ blocking HTTP request ختم ڪئي وئي.
* بهتري: GitHub Actions workflows کي Node.js 24 ڏانهن upgrade ڪيو ويو.
