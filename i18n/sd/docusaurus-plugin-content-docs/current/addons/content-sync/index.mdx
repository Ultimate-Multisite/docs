# مواد هم وقت سازي

Multisite Ultimate: مواد هم وقت سازي

Multisite Ultimate لاءِ هي اضافو نيٽورڪ منتظمن کي اجازت ڏئي ٿو ته ڪنهن به سائيٽ مان مواد کي نيٽورڪ جي ٻين سائيٽن ڏانهن هم وقت ڪن. اهو سڀني پوسٽ قسمن کي سپورٽ ڪري ٿو، جن ۾ پوسٽون، صفحا، ڪسٽم پوسٽ قسم، ۽ Elementor ٽيمپليٽ شامل آهن. اضافي ۾ مواد ۽ هدف سائيٽن چونڊڻ لاءِ گهڻ-قدم وارو وزرڊ انٽرفيس شامل آهي، بيچ پروسيسنگ ۽ حقيقي وقت جي پيش رفت موٽ سان.

## خاصيتون {#features}

* مواد جي چونڊ: ٿمب نيلز ۽ پوسٽ قسم فلٽرنگ سان بصري ڪارڊ گرڊ
* سڀ پوسٽ قسم: پوسٽون، صفحا، پراڊڪٽس، Elementor ٽيمپليٽ، ۽ ڪنهن به ڪسٽم پوسٽ قسم کي هم وقت ڪريو
* سائيٽ جي چونڊ: چونڊيو ته مواد ڪهڙين سائيٽن ڏانهن هم وقت ڪيو وڃي، ماخذ سائيٽ جي اخراج ۽ چائلڊ سائيٽ جي اڳ-چونڊ سان
* بيچ پروسيسنگ: پيش رفت اپڊيٽس سان هڪ وقت ۾ 10 سائيٽن ڏانهن هم وقت ڪري ٿو
* ميڊيا هم وقت سازي: پوسٽ مواد مان ميڊيا فائلون (تصويرون، وڊيوز) پاڻمرادو نقل ڪري هدف سائيٽن تي ٻيهر ميپ ڪري ٿو
* ٽيڪسونامي هم وقت سازي: ڪيٽيگريون، ٽيگ، ۽ ڪسٽم ٽيڪسوناميز برقرار رکي ٿو
* Elementor سپورٽ: CSS ٻيهر ٺاهڻ ۽ ٽيمپليٽ ID ٻيهر ميپنگ سان Elementor ٽيمپليٽس لاءِ مڪمل سپورٽ
* تڪرار جو حل: موجوده مواد کي بدلائڻ يا ان کي ڇڏي ڏيڻ جو اختيار
* ڪيش انتظام: هم وقت سازي کان پوءِ هدف سائيٽن تي Elementor ڪيش پاڻمرادو صاف ڪري ٿو

## گهرجون {#requirements}

* WordPress 5.3 يا وڌيڪ
* PHP 7.4 يا وڌيڪ
* Ultimate Multisite پلگ ان (فعال)
* Elementor پلگ ان (اختياري، Elementor ٽيمپليٽ هم وقت سازي لاءِ گهربل)

## انسٽاليشن {#installation}

1. 'ultimate-multisite-content-sync' کي '/wp-content/plugins/' ڊاريڪٽري ۾ اپلوڊ ڪريو
2. پنهنجي WordPress نيٽورڪ ايڊمن ۾ 'پلگ انز' مينيو ذريعي پلگ ان فعال ڪريو
3. Ultimate Multisite → سيٽنگون → مواد هم وقت سازي ۾ سيٽنگون ترتيب ڏيو
4. هم وقت سازي شروع ڪرڻ لاءِ Ultimate Multisite → مواد هم وقت سازي ڏانهن وڃو
