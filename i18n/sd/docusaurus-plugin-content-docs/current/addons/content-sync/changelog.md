# مواد هم وقت سازي تبديلي نامو

### نسخو 1.1.0 - 2025-12-31 تي جاري ٿيو {#version-110---released-on-2025-12-31}
* نئون: سڀني post قسمن جي هم وقت سازي لاءِ سهائتا (posts، pages، custom post types)
* نئون: مواد چونڊڻ واري قدم ۾ post قسم فلٽرنگ
* نئون: ماخذ سائيٽ چونڊڻ وارو قدم - ڪنهن به سائيٽ مان هم وقت سازي، رڳو مکيه سائيٽ مان نه
* نئون: ٽيمپليٽ سائيٽن لاءِ ذيلي سائيٽ جي اڳواٽ چونڊ
* نئون: سڀني post قسمن لاءِ post_content مان ميڊيا ڪڍڻ
* نئون: Taxonomy هم وقت سازي (categories، tags، custom taxonomies)
* تبديل ٿيل: "Elementor Template Sync" مان "Content Sync" نالو رکيو ويو
* تبديل ٿيل: "Select Templates" قدم جو نالو "Select Data" رکيو ويو
* بهتر ڪيل: Elementor-مخصوص ڪوڊ هاڻي رڳو elementor_library post قسم لاءِ هلندو آهي

### نسخو 1.0.0 - 2025-12-19 تي جاري ٿيو {#version-100---released-on-2025-12-19}
* شروعاتي رليز
* ٽيمپليٽ ۽ سائيٽ چونڊڻ لاءِ گهڻ-قدمي وزارڊ
* حقيقي وقت جي پيش رفت موٽ سان Batch پروسيسنگ
* خودڪار ID ٻيهر ميپنگ سان مڪمل ميڊيا هم وقت سازي
* Ultimate Multisite سان سيٽنگن جي انضمام
* نيٽ ورڪ سيٽنگن ذريعي سائيٽ خارج ڪرڻ جي سهائتا
* هم وقت سازي کان پوءِ Elementor cache صاف ڪرڻ
* ٽيمپليٽ taxonomy محفوظ رکڻ
