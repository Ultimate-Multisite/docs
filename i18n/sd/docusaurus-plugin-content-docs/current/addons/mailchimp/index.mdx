# Mailchimp

Multisite Ultimate: Mailchimp انضمام

پنهنجي Multisite Ultimate نيٽ ورڪ کي پنهنجي Mailchimp سان ضم ڪريو. توهان مخصوص پلانن کي Mailchimp فهرستن سان ڳنڍي سگهو ٿا ۽ سائن اپ وقت انهن فهرستن ۾ سبسڪرائبر پاڻمرادو شامل ڪري سگهو ٿا.

## انسٽاليشن {#installation}

1. 'ultimate-multisite-mailchimp' کي '/wp-content/plugins/' ڊائريڪٽري ۾ اپلوڊ ڪريو
2. پنهنجي WordPress Network Admin ۾ 'Plugins' مينيو ذريعي plugin کي فعال ڪريو

## استعمال جي رهنمائي {#usage-guide}

اسان جي Ultimate Multisite: MailChimp Integration Add-on سان پنهنجي سبسڪرائبرن جون اي ميلون پنهنجي MailChimp فهرستن سان هم وقت ڪرڻ بلڪل آسان آهي. هن سبق جو مقصد توهان کي سڀ ڪجهه سيٽ اپ ڪرڻ ۾ مدد ڪرڻ آهي!

## API key حاصل ڪرڻ {#getting-an-api-key}

ان کان اڳ جو اسان موجود مختلف سيٽنگن ۾ وڃون، اسان کي هڪ MailChimp API key جي ضرورت آهي. هي Ultimate Multisite کي MailChimp API سان ڳالهائڻ جي اجازت ڏيندو، ته جيئن فهرستون، گروپ، ۽ جڏهن استعمال ڪندڙ سائن اپ ڪن ته انهن فهرستن ۽ گروپن ۾ اي ميل پتا شامل ڪري سگهجن.

پنهنجي MailChimp API key حاصل ڪرڻ لاءِ هي سبق پيروي ڪريو: [API Keys بابت | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

_پنهنجي MailChimp API key هتي شامل ڪريو_ _**Ultimate Multisite Settings - > Add-on Settings -> MailChimp**_

## انضمام موڊ {#integration-mode}

MailChimp Integration add-on ٻه “انضمام موڊ” پيش ڪري ٿو، جنهن جو مطلب آهي ته add-on استعمال ڪري اي ميل پتا هم وقت ڪرڻ جا ٻه مختلف طريقا آهن.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_وڌيڪ لچڪ لاءِ ٻه انضمام موڊ!_

### موڊ 1: گهڻيون فهرستون {#mode-1-multiple-lists}

ڊفالٽ موڊ **Multiple Lists** استعمال ڪري ٿو. هي موڊ توهان کي هر پلان لاءِ ڪيترين MailChimp فهرستن کي چونڊڻ جي اجازت ڏيندو. جيڪڏهن توهان پنهنجن استعمال ڪندڙن کي مختلف فهرستن ۾ ورهايو ٿا ته هي بهترين اختيار آهي.

مثال طور، اچو ته فرض ڪريون ته توهان جي MailChimp Account تي 3 مختلف فهرستون آهن: **Newsletter** ، **Plan A** ، ۽ **Plan B**. توهان چاهيو ٿا ته سڀ استعمال ڪندڙ، سندن پلان کان قطع نظر، Newsletter فهرست ۾ به شامل هجن، ته جيئن اهي توهان جون تازه ڪاريون نه وڃائين.

اهو حاصل ڪرڻ لاءِ، توهان Multiple Lists موڊ استعمال ڪري **Newsletter** ۽ **Plan A** ٻنهي فهرستن کي Plan A MailChimp سيٽنگن ۾ شامل ڪري سگهو ٿا، ۽ **Newsletter** ۽ **Plan B** کي Plan B MailChimp سيٽنگن ۾.

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_نون اڪائونٽن ۽ منسوخ ٿيل اڪائونٽن لاءِ ڊفالٽ فهرستون چونڊيو_

### موڊ 2: هڪ فهرست + گهڻا گروپ {#mode-2-single-list--multiple-groups}

MailChimp توهان کي پنهنجي فهرست کي استعمال ڪندڙن جي ننڍن گروپن ۾ ورهائڻ جي اجازت ڏئي ٿو، جن کي **Groups** چيو وڃي ٿو. هي توهان کي وڌيڪ ڪنٽرول ڏئي ٿو، ڇاڪاڻ ته توهان پنهنجي سبسڪرائبرن جي ڪنهن مخصوص حصي ڏانهن مهم موڪلي سگهو ٿا، بغير ان جي ته MailChimp تي ساڳي مهم جون گهڻيون ڪاپيون ٺاهڻيون پون (جنهن سان پوءِ ان مهم جي ميٽرڪس جو تجزيو ڪرڻ ڏکيو ٿي وڃي ٿو).

مثال طور، توهان وٽ **My Network Subscribers,** نالي هڪ ئي فهرست ٿي سگهي ٿي، ۽ ان جي اندر **Plan A** ۽ **Plan B** لاءِ گروپ هجن.

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_جيڪڏهن “هڪ فهرست + گهڻا گروپ” انضمام موڊ استعمال ڪري رهيا آهيو، ته ڊفالٽ فهرستون ۽ گروپ چونڊيو_

## ڊفالٽ فهرستون ۽ گروپ {#default-lists-and-groups}

جڏهن توهان چونڊيو ته ڪهڙو انضمام موڊ توهان جي مخصوص استعمال جي صورت لاءِ وڌيڪ مناسب آهي، تڏهن ڊفالٽ فهرستون (۽ موڊ جي لحاظ کان گروپ) چونڊڻ جو وقت آهي. ڊفالٽ فهرست تڏهن استعمال ڪئي ويندي جڏهن ڪنهن ڏنل پلان لاءِ مخصوص فهرست چونڊيل نه هوندي.

توهان منسوخ ٿيل اڪائونٽن لاءِ به ڊفالٽ فهرست چونڊي سگهو ٿا. جڏهن به ڪو استعمال ڪندڙ پنهنجي سبسڪرپشن ختم ڪري ۽ پنهنجو اڪائونٽ توهان جي نيٽ ورڪ مان هٽائي، سندس اي ميل پتو منسوخ ٿيل اڪائونٽن لاءِ مقرر ڪيل فهرست ڏانهن منتقل ڪيو ويندو.

## ٻٽي سبسڪرپشن فعال ڪرڻ {#enabling-double-subscription}

ساڳي سيٽنگن واري صفحي تي، توهان **Double Subscription** فعال ڪري سگهندا. هي اختيار استعمال ڪندڙ کي چونڊيل ڊفالٽ فهرست ۽ انهي استعمال ڪندڙ جي پلان لاءِ چونڊيل فهرست ٻنهي ۾ شامل ڪندو. جيڪڏهن هي اختيار فعال نه هجي، ته استعمال ڪندڙ صرف انهي فهرست ۾ شامل ٿيندو جيڪا استعمال ڪندڙ جي پلان لاءِ چونڊيل آهي. جيڪڏهن انهي مخصوص پلان لاءِ ڪا به مقرر نه هجي، ته پوءِ استعمال ڪندڙ چونڊيل ڊفالٽ فهرست ۾ شامل ڪيو ويندو.

<!-- Screenshot unavailable: MailChimp double subscription toggle setting -->

_توهان سائن اپ وقت پنهنجن استعمال ڪندڙن کي ڊفالٽ فهرست ۽ پلان جي فهرست ٻنهي ۾ سبسڪرائب ڪرائي سگهو ٿا_

## هر پلان لاءِ فهرستون ۽ گروپ چونڊڻ {#selecting-lists-and-groups-for-each-of-the-plans}

پنهنجن **Plans** ڏانهن وڃو ۽ اهو پلان چونڊيو جنهن کي توهان ايڊٽ ڪرڻ چاهيو ٿا. نوٽ ڪريو ته **Advanced Options** meta-box تي، MailChimp نالي هڪ نئون ٽيب موجود هوندو.

ان ٽيب تي، توهان انهي پلان لاءِ گهڻيون فهرستون يا هڪ فهرست ۽ گهڻا گروپ (انضمام موڊ جي لحاظ کان) چونڊي سگهندا.

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_انهن فهرستن ۽ گروپن کي چونڊيو جيڪي توهان پنهنجي هر پلان لاءِ استعمال ڪرڻ چاهيو ٿا_

جڏهن ڪو نئون استعمال ڪندڙ انهي پلان لاءِ سائن اپ ڪندو، سندس اي ميل پتو چونڊيل فهرستن ۽ گروپن ۾ شامل ڪيو ويندو!
