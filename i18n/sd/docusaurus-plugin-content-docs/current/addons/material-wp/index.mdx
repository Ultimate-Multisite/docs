# Material WP {#material-wp}

WordPress جو منتظم انٽرفيس سالن کان ساڳيو رهيو آهي. پر ڇا توهان ڪڏهن سوچيو آهي ته منتظم انٽرفيس جو Material Design نسخو ڪيئن نظر ايندو؟ 

Material WP هڪ منتظم ٿيم آهي، جيڪو توهان کي اهو ڏيک ۽ احساس ڏيڻ لاءِ تيار ڪيو ويو آهي، جنهن سان توهان پنهنجي گراهڪن کي وڌيڪ ترتيب ڏنل Dashboard مهيا ڪري سگهو ٿا.

## خاصيتون {#features}

- ڪسٽم CSS سپورٽ
- آسان اسٽائلنگ سيٽنگون
- مينيو جي ٻيهر ترتيب
- آسان درآمد/برآمد

## دستاويز {#documentation}

**Material WP سان پنهنجي منتظم پينل کي ترتيب ڏيڻ**

## انسٽاليشن {#installation}

1. 'material-wp' کي '/wp-content/plugins/' ڊائريڪٽري ۾ اپلوڊ ڪريو
2. WordPress ۾ 'پلگ اِنز' مينيو ذريعي پلگ اِن کي فعال ڪريو

## استعمال جي رهنمائي {#usage-guide}

WordPress جو منتظم انٽرفيس سالن کان ساڳيو رهيو آهي. پر ڇا توهان ڪڏهن سوچيو آهي ته منتظم انٽرفيس جو Material Design نسخو ڪيئن نظر ايندو؟

Material WP هڪ منتظم ٿيم آهي، جيڪو توهان کي اهو ڏيک ۽ احساس ڏيڻ لاءِ تيار ڪيو ويو آهي، جنهن سان توهان پنهنجي گراهڪن کي وڌيڪ ترتيب ڏنل Dashboard مهيا ڪري سگهو ٿا. Material WP، Ultimate Multisite جي پريميئم پلگ اِنز ڪور جو به حصو آهي.

ان کي **انسٽال ڪرڻ** کان پوءِ، پنهنجي _پلگ اِنز_ صفحي تي وڃو ۽ پنهنجي نيٽ ورڪ تي ان کي فعال ڪرڻ لاءِ ڪلڪ ڪريو.
