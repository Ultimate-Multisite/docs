# VAT

Multisite Ultimate: يورپي VAT

Multisite Ultimate جي ڊفالٽ ٽيڪس سنڀال ۾ اضافي خاصيتون ۽ اوزار شامل ڪري ٿو ته جيئن يورپي VAT کي بهتر سهارو ملي.
خودڪار طور يورپي يونين VAT شرحون ٽيڪس شرحون واري صفحي ۾ شامل ڪري ٿو مان

## انسٽاليشن {#installation}

1. 'ultimate-multisite-vat' کي '/wp-content/plugins/' ڊائريڪٽري ۾ اپلوڊ ڪريو
2. WordPress ۾ 'پلگ انز' مينيو ذريعي پلگ ان کي فعال ڪريو
3. Ultimate Multisite > سيٽنگون > ٽيڪسز ڏانهن وڃو ۽ يورپي يونين VAT فعال ڪريو.
4. ٽيڪس شرحون تي ڪلڪ ڪريو ۽ تري ۾ يورپي يونين VAT شرحون اپڊيٽ ڪريو ۽ ٽيڪس شرحون محفوظ ڪريو.
5. ٿي ويو!
