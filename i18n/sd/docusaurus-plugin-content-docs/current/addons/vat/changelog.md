# VAT تبديلي لاگ {#vat-changelog}

نسخو 1.0.7 - 2026-02-03 تي جاري ڪيو ويو

* درستگي: جرمن (DE) VAT نمبر VIES SOAP سروس جي MS_MAX_CONCURRENT_REQ غلطين سبب تصديق ۾ ناڪام ٿي رهيا هئا. وڌيڪ ڀروسي جوڳي تصديق لاءِ legacy SOAP API مان سرڪاري EU VIES REST API ڏانهن منتقل ڪيو ويو.
* درستگي: VIES سروس عارضي طور دستياب نه هئڻ وقت VAT نمبر هاڻي قبول ڪيا وڃن ٿا، غلط طور رد ڪرڻ بدران. بعد ۾ جائزي لاءِ لاگ داخلا ٺاهي وڃي ٿي.
* درستگي: VAT/Tax ID فيلڊ هاڻي سڀني ملڪن لاءِ نظر اچي ٿي، صرف EU ميمبرن لاءِ نه. غير-EU ٽيڪس IDs (مثال طور Swiss CHE نمبر) VIES تصديق کان سواءِ انوائس ڏيک لاءِ محفوظ ڪيا وڃن ٿا.
* درستگي: Greece (GR)، Monaco (MC)، ۽ Isle of Man (IM) لاءِ VAT اڳياڙي ڳولا غلط array lookup منطق سبب خراب هئي.
* بهتري: غير ضروري wp-cli-bundle dev dependency هٽائي وئي جيڪا Site Exporter ايڊآن سان گڏ استعمال ڪرڻ وقت fatal errors جو سبب بڻجي سگهي ٿي.
* بهتري: جامع unit test suite شامل ڪيو ويو (53 ٽيسٽ).

نسخو 1.0.6 - 2026-01-25 تي جاري ڪيو ويو

* درستگي: گهريلو B2B ٽرانزيڪشنز هاڻي صحيح طور VAT چارج ڪن ٿيون. EU VAT قاعدن موجب، reverse charge صرف سرحد پار B2B ٽرانزيڪشنز تي لاڳو ٿئي ٿو، نه جڏهن ڪسٽمر جو ملڪ ڪمپني جي ملڪ سان ملي ٿو.

نسخو 1.0.5 - 2026-01-22 تي جاري ڪيو ويو

* درستگي: ايڊآن لوڊ نه ٿي رهيو هو يا صحيح ڪم نه ڪري رهيو هو.
* تبديلي: VAT شرح ڊيٽا جو ذريعو euvatrates.com مان ibericode/vat-rates repository ڏانهن تبديل ڪيو ويو، وڌيڪ ڀروسي جوڳي ۽ فعال طور سنڀاليل ڊيٽا لاءِ.
* درستگي: super_reduced_rates آپشن ويليو جي ٽائيپو درست ڪئي وئي.
* بهتري: غلطين کان بچڻ لاءِ ملڪ ڪوڊ سنڀالڻ ۾ null checks شامل ڪيا ويا.
* تبديلي: Traduttore ذريعي خودڪار اپڊيٽس جي حق ۾ گڏ ڪيل ترجمي فائلون هٽايون ويون.
