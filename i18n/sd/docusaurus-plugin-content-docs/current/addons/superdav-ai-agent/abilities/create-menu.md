# مينيو ٺاهيو

**مينيو ٺاهيو** صلاحيت WordPress نيويگيشن مينيو ٺاهي ٿي، جنهن ۾ صفحي جي عنوانن کان الڳ مخصوص نيويگيشن ليبلز جي سهائتا هوندي آهي.

## جائزو

هي صلاحيت معياري مينيو ٺاهڻ واري ڪارڪردگي کي `navigation_label` پيرا ميٽر بيان ڪرڻ جي صلاحيت سان وڌائي ٿي. هي توهان کي اهڙا مينيو ٺاهڻ جي اجازت ڏئي ٿو جتي نيويگيشن ۾ ڏيکاريل ليبل صفحي جي عنوان کان مختلف هجي، جنهن سان سائيٽ جي ڍانچي ۽ صارف تجربي ۾ وڌيڪ لچڪ ملي ٿي.

## پيرا ميٽر

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | string | Yes | مينيو جو نالو، مثال طور `Primary Navigation` |
| `location` | string | No | هن مينيو کي تفويض ڪرڻ لاءِ ٿيم جي جڳهه، مثال طور `primary` |
| `navigation_label` | string | No | نيويگيشن ۾ ڏيکارڻ لاءِ ليبل (صفحي جي عنوان کان الڳ) |

## واپسي قدر

```json
{
  "success": true,
  "menu_id": 7,
  "name": "Primary Navigation",
  "navigation_label": "Main Menu"
}
```

## نيويگيشن ليبل بمقابله صفحي جو عنوان

`navigation_label` پيرا ميٽر توهان کي اندروني مينيو نالي کي صارفن کي ڏيکاريل ليبل کان الڳ ڪرڻ جي اجازت ڏئي ٿو:

- **`name`** — WordPress پاران استعمال ٿيندڙ اندروني مينيو سڃاڻپ ڪندڙ (مثال طور، "Primary Navigation")
- **`navigation_label`** — نيويگيشن ۾ سائيٽ جي مهمانن کي ڏيکاريل ليبل (مثال طور، "Main Menu")

هي تڏهن فائديمند آهي جڏهن:
- توهان جو اندروني نالو رکڻ جو طريقو صارفن کي ڏيکاريل ليبلز کان مختلف هجي
- توهان ايڊمن پينل جي ڀيٽ ۾ نيويگيشن ۾ ننڍا ليبل چاهيو ٿا
- توهان کي مختلف ليبل ڊگهاين سان گهڻين ٻولين جي سهائتا ڪرڻي هجي
- توهان مخصوص علائقن يا صارف گروپن لاءِ مينيو ٺاهي رهيا آهيو

## استعمال جا مثال

### مثال 1: نيويگيشن ليبل سان سادو مينيو

**اشارو:**
```
Create a primary navigation menu called "Main Navigation" with the navigation label "Menu".
```

**نتيجو:**
```json
{
  "success": true,
  "menu_id": 7,
  "name": "Main Navigation",
  "navigation_label": "Menu"
}
```

### مثال 2: مخصوص ٿيم جڳهه لاءِ مينيو

**اشارو:**
```
Create a footer menu called "Footer Links" with navigation label "Quick Links" and assign it to the footer location.
```

**نتيجو:**
```json
{
  "success": true,
  "menu_id": 8,
  "name": "Footer Links",
  "navigation_label": "Quick Links",
  "location": "footer"
}
```

## Theme Builder سان انضمام

Theme Builder استعمال ڪرڻ وقت، مينيو ٺاهيو صلاحيت:

1. موجود ٿيم مينيو جڳهن کي خودڪار طور سڃاڻي ٿي
2. توهان جي ڊيزائن لاءِ مناسب نيويگيشن ليبلز سان مينيو ٺاهي ٿي
3. مينيوز کي صحيح ٿيم جڳهن تي تفويض ڪري ٿي
4. ٺاهڻ کان پوءِ مينيو شيون شامل ڪرڻ جي سهائتا ڪري ٿي

## لاڳاپيل صلاحيتون

- **`add_menu_item`** — موجود مينيو ۾ شيون شامل ڪريو
- **`update_menu`** — مينيو جو نالو تبديل ڪريو يا ان کي ٿيم جي جڳهه تي ٻيهر تفويض ڪريو
- **`delete_menu`** — پنهنجي سائيٽ مان مينيو هٽايو

## بهترين طريقا

- **واضح نيويگيشن ليبلز استعمال ڪريو** — صارفن لاءِ ليبل مختصر ۽ وضاحتي رکو
- **ٿيم جڳهن سان ملائي ڇڏيو** — صحيح نمائش لاءِ مينيوز کي صحيح ٿيم جڳهه تي تفويض ڪريو
- **مينيو ڍانچي جي رٿابندي ڪريو** — شيون ٺاهڻ کان اڳ پنهنجي مينيو درجابندي بابت فيصلو ڪريو
- **جوابدهي جي جانچ ڪريو** — پڪ ڪريو ته مينيوز موبائل ڊوائيسز تي صحيح طرح ڏيکارجن ٿا
- **ليبلز کي مقامي بڻايو** — مختلف ٻولي وارن نسخن لاءِ مختلف نيويگيشن ليبلز استعمال ڪريو

## حدون

- نيويگيشن ليبلز صرف ڏيکارڻ لاءِ آهن؛ اندروني `name` اڃا به WordPress سڃاڻپ لاءِ استعمال ٿيندو آهي
- ٿيم جي سهائتا مختلف هوندي آهي؛ سڀئي ٿيم سڀني مينيو جڳهن جي سهائتا نٿا ڪن
- مينيو ٺاهڻ کان پوءِ مينيو شيون الڳ طور شامل ڪرڻيون پونديون
- نيويگيشن ليبل تبديل ڪرڻ لاءِ مينيو کي اپڊيٽ ڪرڻ جي ضرورت هوندي آهي
