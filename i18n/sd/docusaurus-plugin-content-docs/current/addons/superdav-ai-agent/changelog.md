- **Google Calendar اوزار** — شيڊول کان واقف خودڪاريءَ لاءِ ترتيب ڏنل ڪئلينڊر ۽ واقعا پڙهو

- **TextBee SMS فراهم ڪندڙ** — ترتيب ڏنل ٽيڪسٽ-ميسيج اطلاع TextBee ذريعي موڪليو
