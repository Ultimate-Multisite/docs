# اوزار ۽ ترتيب {#tools-and-configuration}

Theme Builder توهان جي WordPress سائيٽ جو تجزيو ڪرڻ، ڊزائين ڪرڻ، ۽ ٺاهڻ لاءِ اوزارن جو هڪ سيٽ استعمال ڪري ٿو. ورزن 1.16.0 ۾، **sd-ai-agent/site-scrape هاڻي Tier 1 اوزار آهي**، جنهن سان اهو Theme Builder ۾ ڊفالٽ طور دستياب آهي. Superdav AI Agent v1.18.0 ڀروسي لائق ڊولپر ڪم جي وهڪرن لاءِ الڳ ورهايل Advanced ساٿي پيڪيج پڻ متعارف ڪرائي ٿو؛ اهي اوزار WordPress.org ڪور پيڪيج جو حصو نه آهن ۽ انهن کي الڳ انسٽال ۽ مجاز ڪرڻو پوندو.

## دستياب اوزار {#available-tools}

### Tier 1 اوزار (هميشه دستياب) {#tier-1-tools-always-available}

Tier 1 اوزار Theme Builder ۾ ڪنهن به اضافي ترتيب کانسواءِ ڊفالٽ طور دستياب آهن.

#### sd-ai-agent/site-scrape {#sd-ai-agentsite-scrape}

**مقصد:** موجوده ويب سائيٽن جو تجزيو ڪري ڊزائين جي الهام، مواد جي بناوٽ، ۽ ڪارڪردگي ڪڍڻ.

**صلاحيتون:**

- **ويب سائيٽ تجزيو** — مقابلي ڪندڙ يا الهامي ويب سائيٽن کي scrape ۽ تجزيو ڪريو
- **ڊزائين ڪڍڻ** — رنگ، فونٽ، ۽ لي آئوٽ نمونا سڃاڻو
- **مواد جي بناوٽ** — صفحي جي تنظيم ۽ درجابندي سمجهو
- **خصوصيت سڃاڻپ** — plugins ۽ ڪارڪردگي سڃاڻو
- **ڪارڪردگي تجزيو** — صفحي جي رفتار ۽ بهتري چيڪ ڪريو
- **SEO تجزيو** — meta tags ۽ structured data جو جائزو وٺو

**استعمال:**

```
example.com جي ڊزائين جو تجزيو ڪريو ۽ ان کي منهنجي سائيٽ لاءِ الهام طور استعمال ڪريو.
```

**اهو ڇا ڪڍي ٿو:**

- رنگن جو پيليٽ ۽ رنگن جو استعمال
- Typography (فونٽ ۽ سائيزون)
- لي آئوٽ بناوٽ ۽ وٿي
- نيويگيشن نمونا
- مواد جي تنظيم
- تصويرون ۽ ميڊيا جو استعمال
- تعاملي عنصر
- موبائل جوابدهي

**حدون:**

- پاسورڊ سان محفوظ سائيٽن کي scrape نٿو ڪري سگهي
- robots.txt ۽ سائيٽ پاليسين جو احترام ڪري ٿو
- متحرڪ مواد پڪڙي نه سگهي
- JavaScript-واريون ڳريون سائيٽون محدود تجزيو رکي سگهن ٿيون
- وڏين سائيٽن جو تجزيو وڌيڪ وقت وٺي سگهي ٿو

### Tier 2 اوزار (اختياري) {#tier-2-tools-optional}

Tier 2 اوزار تڏهن دستياب هوندا آهن جڏهن Theme Builder سيٽنگن ۾ واضح طور فعال ڪيا وڃن.

#### Advanced Analytics {#advanced-analytics}

تفصيلي ڪارڪردگي ماپون فراهم ڪري ٿو:

- صفحي لوڊ ٿيڻ جا وقت
- Core Web Vitals
- SEO اسڪور
- رسائي پذيري اسڪور
- بهترين عملن جو اسڪور

#### Content Optimizer {#content-optimizer}

هيٺين لاءِ تجزيو ڪري ٿو ۽ بهتريون تجويز ڪري ٿو:

- پڙهڻ جي آساني
- SEO بهتري
- لفظن جو استعمال
- مواد جي بناوٽ
- Call-to-action جي جڳهه

### Tier 3 اوزار (Premium) {#tier-3-tools-premium}

Tier 3 اوزارن لاءِ اضافي ترتيب يا premium خصوصيتون گهرجن.

#### AI Content Generator {#ai-content-generator}

هيٺين لاءِ مواد ٺاهي ٿو:

- پيداوار جون وضاحتون
- خدمتن جا صفحا
- بلاگ پوسٽون
- Meta وضاحتون
- Call-to-action نقل

#### Advanced Design Tools {#advanced-design-tools}

ترقي يافته ڊزائين صلاحيتون فراهم ڪري ٿو:

- ڪسٽم CSS ٺاهڻ
- Animation ٺاهڻ
- تعاملي عنصر ڊزائين
- ترقي يافته رنگن جو نظريو
- Typography بهتري

### Advanced ساٿي اوزار (الڳ پيڪيج) {#advanced-companion-tools-separate-package}

Advanced ساٿي پيڪيج ڪور Superdav AI Agent رليز کان الڳ ورهايو ويندو آهي. اهو ڀروسي لائق ڊولپر ۽ سائيٽ مالڪ ماحولن لاءِ آهي جتي منتظم واضح طور agent کي وڌيڪ اثرائتي اوزارڪاري استعمال ڪرڻ جي اجازت ڏين ٿا.

Advanced ساٿي صلاحيتن ۾ شامل ٿي سگهن ٿا:

- **Filesystem اوزار** — ڀروسي لائق ماحولن ۾ منظور ٿيل فائلون جانچيو ۽ تبديل ڪريو
- **Database اوزار** — واضح اجازت ملڻ تي سائيٽ ڊيٽا query يا update ڪريو
- **WP-CLI اوزار** — WordPress سار سنڀال ۽ جانچ جا حڪم هلائو
- **REST dispatcher اوزار** — ڪنٽرول ٿيل dispatch ڪم جي وهڪرن ذريعي رجسٽر ٿيل REST endpoints کي call ڪريو
- **Plugin builder اوزار** — plugin code جو scaffold ٺاهيو ۽ ان تي ورجائي ڪم ڪريو
- **Git snapshot اوزار** — خطري وارن ڊولپر آپريشنن کان اڳ snapshots ٺاهيو
- **User-management اوزار** — اجازت هئڻ تي user records جو جائزو وٺڻ يا ترتيب ڏيڻ ۾ مدد ڪريو
- **Benchmark اوزار** — ڊولپمينٽ جائزي لاءِ ڪارڪردگي يا صلاحيت benchmarks هلائو

انهن کي هر انسٽاليشن لاءِ عام طور دستياب Theme Builder اوزار طور دستاويز نه ڪريو. اهي صرف تڏهن دستياب آهن جڏهن Advanced ساٿي پيڪيج انسٽال، فعال، ۽ ڀروسي لائق منتظمن تائين محدود هجي.

## اوزارن جي ترتيب {#tool-configuration}

### اوزار فعال ڪرڻ {#enabling-tools}

Theme Builder ۾ اضافي اوزار فعال ڪرڻ لاءِ:

1. **Gratis AI Agent → Settings** ڏانهن وڃو
2. **Theme Builder → Tools** ڏانهن وڃو
3. ضرورت موجب اوزار on/off ڪريو
4. سيٽنگون محفوظ ڪريو

### اوزارن جون اجازتون {#tool-permissions}

ڪجهه اوزارن کي اجازتون گهرجن:

- **سائيٽ scraping** — انٽرنيٽ رسائي گهرجي
- **Analytics** — Google Analytics integration گهرجي
- **مواد ٺاهڻ** — API keys گهرجن
- **ترقي يافته خصوصيتون** — premium subscription گهربل ٿي سگهي ٿي
- **Advanced ساٿي اوزار** — الڳ ورهايل Advanced پيڪيج ۽ واضح منتظم ڀروسو گهرجن

### API Keys ۽ Credentials {#api-keys-and-credentials}

انهن اوزارن لاءِ API keys ترتيب ڏيو جن کي اهي گهرجن:

1. **Gratis AI Agent → Settings → API Keys** ڏانهن وڃو
2. هر اوزار لاءِ credentials داخل ڪريو
3. ڪنيڪشن ٽيسٽ ڪريو
4. محفوظ نموني محفوظ ڪريو

### Advanced ساٿي پيڪيج انسٽال ڪرڻ {#installing-the-advanced-companion-package}

Advanced ساٿي پيڪيج WordPress.org ڪور ZIP کان الڳ جاري ڪيو ويندو آهي. ان کي صرف رليز لاءِ سرڪاري پروجيڪٽ ورهاست چينل مان انسٽال ڪريو، پوءِ رسائي ڀروسي لائق منتظمن تائين محدود ڪريو. production سائيٽ تي filesystem، database، WP-CLI، REST dispatcher، plugin builder، user-management، يا benchmark اوزار فعال ڪرڻ کان اڳ انساني منظوري gates ۽ git snapshot رويي جو جائزو وٺو.

## sd-ai-agent/site-scrape استعمال ڪرڻ {#using-sd-ai-agentsite-scrape}

### بنيادي استعمال {#basic-usage}

Theme Builder کان ويب سائيٽ جو تجزيو ڪرڻ لاءِ پڇو:

```
competitor.com تي منهنجي مقابلي ڪندڙ جي سائيٽ جي ڊزائين جو تجزيو ڪريو
۽ منهنجي سائيٽ لاءِ ڊزائين بهتريون تجويز ڪريو.
```

### مخصوص تجزيو {#specific-analysis}

تجزيي جا مخصوص قسم درخواست ڪريو:

```
example.com مان رنگن جو پيليٽ ڪڍو ۽ ان کي الهام طور استعمال ڪريو.
```

```
example.com جي نيويگيشن بناوٽ جو تجزيو ڪريو ۽ ساڳي
تنظيم منهنجي سائيٽ تي لاڳو ڪريو.
```

```
example.com جي موبائل جوابدهي چيڪ ڪريو ۽ يقيني بڻايو ته منهنجي سائيٽ
ايتري ئي جوابده آهي.
```

### ڀيٽ وارو تجزيو {#comparison-analysis}

گهڻن سائيٽن جي ڀيٽ ڪريو:

```
site1.com ۽ site2.com جي ڊزائينن جي ڀيٽ ڪريو ۽ اهڙي ڊزائين ٺاهيو
جيڪا ٻنهي جا بهترين عنصر گڏ ڪري.
```

## اوزارن جون حدون ۽ ويچار {#tool-limitations-and-considerations}

### شرح محدود ڪرڻ {#rate-limiting}

- سرور تي وڌاءُ بار روڪڻ لاءِ اسڪريپنگ تي شرح جي حد لاڳو آهي
- ساڳي سائيٽ ڏانهن گهڻيون درخواستون سست ڪيون وڃي سگهن ٿيون
- وڏين سائيٽن جو تجزيو ڪرڻ ۾ وڌيڪ وقت لڳي سگهي ٿو

### مواد جي درستگي {#content-accuracy}

- متحرڪ مواد شايد مڪمل طور تي نه پڪڙجي
- JavaScript سان رينڊر ٿيل مواد شايد اڻپورو هجي
- حقيقي وقت واري ڊيٽا شايد ظاهر نه ٿئي
- ڪجهه مواد شايد paywalls پٺيان هجي

### قانوني ۽ اخلاقي خيال {#legal-and-ethical-considerations}

- robots.txt ۽ سائيٽ جي پاليسين جو احترام ڪريو
- ڪاپي رائيٽ ٿيل مواد کي ٻيهر شايع ڪرڻ لاءِ اسڪريپ نه ڪريو
- تجزيو نقل ڪرڻ لاءِ نه، پر الهام لاءِ استعمال ڪريو
- تصديق ڪريو ته ڪڍيل مواد استعمال ڪرڻ جا حق توهان وٽ آهن
- تجزيو ڪيل سائيٽن جي خدمت جي شرطن تي عمل ڪريو

### ڪارڪردگيءَ تي اثر {#performance-impact}

- وڏين سائيٽن جي اسڪريپنگ ۾ وقت لڳي سگهي ٿو
- هڪ ئي وقت گهڻيون اسڪريپنگون وڌيڪ سست ٿي سگهن ٿيون
- نيٽ ورڪ ڪنيڪٽيوٽي رفتار تي اثرانداز ٿئي ٿي
- وڏين ڊيٽاسيٽن کي وڌيڪ پروسيسنگ جي ضرورت پئجي سگهي ٿي

## بهترين طريقا {#best-practices}

### سائيٽ تجزيو استعمال ڪرڻ {#using-site-analysis}

1. **گهڻيون سائيٽون تجزيو ڪريو** — ڪيترن ذريعن مان الهام گڏ ڪريو
2. **ڍانچي تي ڌيان ڏيو** — layout ۽ تنظيم جا نمونا سکو
3. **رنگ ڪڍو** — رنگن جي پيلٽس کي شروعاتي نقطن طور استعمال ڪريو
4. **ٽائپوگرافي جو مطالعو ڪريو** — اهي font ميلاپ سڃاڻو جيڪي توهان کي پسند آهن
5. **نيويگيشن جو جائزو وٺو** — menu جي تنظيم کي سمجھو

### اخلاقي اسڪريپنگ {#ethical-scraping}

1. **robots.txt جو احترام ڪريو** — سائيٽ جي هدايتن تي عمل ڪريو
2. **مواد نقل نه ڪريو** — تجزيو صرف الهام لاءِ استعمال ڪريو
3. **حقن جي تصديق ڪريو** — پڪ ڪريو ته توهان ڪڍيل مواد استعمال ڪري سگهو ٿا
4. **ذريعن جو حوالو ڏيو** — الهام جي ذريعن کي ڪريڊٽ ڏيو
5. **شرطن تي عمل ڪريو** — سائيٽ جي خدمت جي شرطن جي پابندي ڪريو

### اوزار جي اثرائتي وڌائڻ {#maximizing-tool-effectiveness}

1. **مخصوص ٿيو** — تجزيي جا مخصوص قسم درخواست ڪريو
2. **پسمنظر ڏيو** — پنهنجي سائيٽ جو مقصد بيان ڪريو
3. **اميدون مقرر ڪريو** — پنهنجا design مقصد بيان ڪريو
4. **ورجاءُ ڪريو** — نتيجن جي بنياد تي بهتر ڪريو
5. **اوزار گڏ استعمال ڪريو** — جامع تجزيي لاءِ گهڻا اوزار استعمال ڪريو

## مسئلا حل ڪرڻ {#troubleshooting}

### سائيٽ اسڪريپ نٿي ٿئي {#site-wont-scrape}

- چيڪ ڪريو ته سائيٽ عوامي طور تي رسائي لائق آهي يا نه
- تصديق ڪريو ته robots.txt اسڪريپنگ جي اجازت ڏئي ٿو
- ڪا ٻي سائيٽ آزمائي ڏسو
- انٽرنيٽ ڪنيڪشن چيڪ ڪريو
- جيڪڏهن مسئلو برقرار رهي ته سپورٽ سان رابطو ڪريو

### اڻپورو تجزيو {#incomplete-analysis}

- سائيٽ ۾ متحرڪ مواد ٿي سگهي ٿو
- JavaScript شايد مڪمل طور تي رينڊر نه ٿيو هجي
- وڏيون سائيٽون timeout ٿي سگهن ٿيون
- بدران مخصوص صفحن جو تجزيو آزمائي ڏسو
- تجزيي جا مخصوص قسم درخواست ڪريو

### سست ڪارڪردگي {#slow-performance}

- وڏين سائيٽن جو تجزيو ڪرڻ ۾ وڌيڪ وقت لڳندو آهي
- هڪ ئي وقت گهڻيون اسڪريپنگون وڌيڪ سست هونديون آهن
- نيٽ ورڪ ڪنيڪٽيوٽي رفتار تي اثرانداز ٿئي ٿي
- گهٽ رش وارن وقتن دوران آزمائي ڏسو
- پهرين ننڍن حصن جو تجزيو ڪريو
