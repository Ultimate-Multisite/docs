# مهمان نوازي مينيوز

**مهمان نوازي مينيوز** خاصيت Theme Builder کي توهان جي WordPress سائيٽ ۾ سڌو سنئون منظم کاڌي ۽ مشروبات جا مينيو صفحا ٺاهڻ ۽ شامل ڪرڻ جي اجازت ڏئي ٿي.

## جائزو {#overview}

Theme Builder هاڻي مهمان نوازي جي ڪاروبارن لاءِ پيشورانه، منظم مينيو صفحا ٺاهي سگهي ٿو، جن ۾ ريسٽورنٽ، ڪيفي، بار، ۽ ڪيٽرنگ خدمتون شامل آهن. اهي مينيوز توهان جي سائيٽ جي ڊيزائن ۾ مڪمل طور ضم ٿيل آهن ۽ آساني سان اپڊيٽ ۽ منظم ڪري سگهجن ٿا.

## سهائتا ڪيل مهمان نوازي جا قسم {#supported-hospitality-types}

- **ريسٽورنٽ** — مڪمل خدمت وارا کاڌي جا مينيوز
- **ڪيفي** — ڪافي ۽ هلڪي خوراڪ جا مينيوز
- **بار ۽ لائونجز** — مشروبات ۽ هلڪن ناشتن جا مينيوز
- **بيڪريون** — پيسٽري ۽ ماني جا مينيوز
- **ڪيٽرنگ خدمتون** — تقريبن لاءِ مينيو اختيار
- **فوڊ ٽرڪس** — موبائل کاڌي جي خدمت جا مينيوز
- **بروئريون ۽ وائنريون** — وضاحتن سان مشروبات جا مينيوز

## مينيو جي جوڙجڪ {#menu-structure}

### مينيو ڪيٽيگريون {#menu-categories}

مينيوز کي ڪيٽيگرين ۾ منظم ڪيو ويندو آهي:

- **هلڪا ناشتا** — شروعاتي کاڌا ۽ ننڍيون پليٽون
- **مکيه کاڌا** — بنيادي ڪورس
- **پاسي جا کاڌا** — ساٿي کاڌا ۽ ڀاڄيون
- **مٺايون** — مٺا کاڌا
- **مشروبات** — پيئڻ واريون شيون (الڪوحل واريون ۽ غير الڪوحل)
- **خاص شيون** — روزاني يا موسمي خاص شيون

### مينيو آئٽم جو فارميٽ {#menu-item-format}

هر مينيو آئٽم ۾ شامل آهي:

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### آئٽم فيلڊز {#item-fields}

| فيلڊ | قسم | وضاحت |
|-------|------|-------------|
| `name` | string | ڊش يا مشروب جو نالو |
| `description` | string | آئٽم جي تفصيلي وضاحت |
| `price` | string | قيمت (ڪرنسي سان فارميٽ ٿيل) |
| `dietary_info` | array | غذائي خاصيتون (Vegan, Gluten-Free, وغيره) |
| `allergens` | array | عام الرجنز (Nuts, Shellfish, وغيره) |
| `availability` | string | ڪڏهن موجود آهي (Daily, Seasonal, وغيره) |

## مهمان نوازي مينيوز ٺاهڻ {#creating-hospitality-menus}

### قدم 1: مينيو جي معلومات فراهم ڪريو {#step-1-provide-menu-information}

Theme Builder کي پنهنجي مينيو بابت ٻڌايو:

```
Create a restaurant menu for my Italian restaurant with appetizers,
pasta dishes, main courses, and desserts. Include prices and descriptions.
```

### قدم 2: Theme Builder مينيو ٺاهي ٿو {#step-2-theme-builder-generates-menu}

Theme Builder:

1. هڪ منظم مينيو صفحو ٺاهي ٿو
2. ان کي توهان جي سائيٽ جي ٿيم سان ملائڻ لاءِ ڊيزائن ڪري ٿو
3. آئٽمز کي ڪيٽيگرين ۾ منظم ڪري ٿو
4. قيمتن ۽ وضاحتن کي فارميٽ ڪري ٿو
5. غذائي ۽ الرجن معلومات شامل ڪري ٿو

### قدم 3: جائزو وٺو ۽ ترتيب ڏيو {#step-3-review-and-customize}

توهان ڪري سگهو ٿا:

1. مينيو آئٽمز ۽ قيمتون ايڊٽ ڪريو
2. ڪيٽيگريون شامل يا ختم ڪريو
3. ڪيٽيگرين اندر آئٽمز جي ترتيب تبديل ڪريو
4. وضاحتون ۽ غذائي معلومات اپڊيٽ ڪريو
5. اسٽائلنگ ۽ لي آئوٽ ترتيب ڏيو

## مينيو ڏيکارڻ جا اختيار {#menu-display-options}

### مڪمل مينيو صفحو {#full-menu-page}

توهان جو مڪمل مينيو ڏيکاريندڙ هڪ مخصوص صفحو:

- ڪيٽيگري موجب منظم
- ڳولا ۽ فلٽر ڪرڻ لائق
- پرنٽ لاءِ موافق لي آئوٽ
- موبائل لاءِ جوابدار ڊيزائن

### مينيو وجيٽ {#menu-widget}

ٻين صفحن تي مينيو سيڪشن شامل ڪريو:

- هوم پيج تي نمايان آئٽمز
- سائڊبار تي روزاني خاص شيون
- بار صفحي تي مشروبات جو مينيو
- فوٽر تي مٺاين جي نمائش

### مينيو PDF {#menu-pdf}

ڊائون لوڊ لائق PDF مينيو ٺاهيو:

- پيشورانه فارميٽنگ
- پرنٽ لاءِ تيار معيار
- تصويرون ۽ وضاحتون شامل
- شيئر ۽ اي ميل ڪرڻ ۾ آسان

## غذائي ۽ الرجن معلومات {#dietary-and-allergen-information}

### غذائي خاصيتون {#dietary-attributes}

آئٽمز کي غذائي معلومات سان نشان لڳايو:

- **Vegan** — ڪوبه جانورن مان ٺهيل پيداوار نه
- **Vegetarian** — ڪوبه گوشت نه
- **Gluten-Free** — celiac disease لاءِ محفوظ
- **Dairy-Free** — ڪابه کير واري پيداوار نه
- **Nut-Free** — ڪوبه وڻن جا نٽ يا مونگڦليون نه
- **Low-Carb** — گهٽ ڪاربوهائيڊريٽس
- **High-Protein** — پروٽين سان ڀرپور

### الرجن چتاءَ {#allergen-warnings}

عام الرجنز شامل ڪريو:

- **Nuts** — وڻن جا نٽ ۽ مونگڦليون
- **Shellfish** — ڪرسٽيشينز ۽ مولسڪس
- **Fish** — مڇي جا سڀ قسم
- **Dairy** — کير ۽ کير جون پيداوارون
- **Eggs** — ڪڪڙ جا انڊا
- **Soy** — سويا پيداوارون
- **Gluten** — ڪڻڪ ۽ گلوٽن تي مشتمل اناج
- **Sesame** — تل جا ٻج ۽ تيل

## مينيو جو انتظام {#menu-management}

### قيمتون اپڊيٽ ڪرڻ {#updating-prices}

مينيو جون قيمتون آساني سان اپڊيٽ ڪريو:

1. مينيو صفحي ڏانهن وڃو
2. "مينيو ايڊٽ ڪريو" تي ڪلڪ ڪريو
3. آئٽمز لاءِ قيمتون اپڊيٽ ڪريو
4. تبديليون محفوظ ڪريو
5. تبديليون توهان جي سائيٽ تي فوري طور ظاهر ٿينديون

### موسمي آئٽمز شامل ڪرڻ {#adding-seasonal-items}

موسمي مينيو جا فرق ٺاهيو:

1. مينيو جو نئون نسخو ٺاهيو
2. موسمي آئٽمز شامل ڪريو
3. آئٽمز کي "موسمي" طور نشان لڳايو
4. دستيابي جون تاريخون شيڊول ڪريو
5. موسم دوران پاڻمرادو ڏيکاريو

### خاص شين جو انتظام {#managing-specials}

روزاني يا هفتيوار خاص شيون ڏيکاريو:

1. "خاص شيون" ڪيٽيگري ٺاهيو
2. دستيابي جي تاريخن سان آئٽمز شامل ڪريو
3. هوم پيج تي خاص شيون نمايان ڪريو
4. روزاني يا هفتيوار اپڊيٽ ڪريو
5. پراڻيون خاص شيون آرڪائيو ڪريو

## Theme Builder سان انٽيگريشن {#integration-with-theme-builder}

جڏهن مهمان نوازي سائيٽن لاءِ Theme Builder استعمال ڪيو وڃي:

1. **پاڻمرادو مينيو سڃاڻپ** — سڃاڻي ٿو ته ڇا توهان کاڌي/مشروبات جو ڪاروبار آهيو
2. **مينيو صفحي جي تخليق** — پيشورانه مينيو صفحا ٺاهي ٿو
3. **ڊيزائن ملائڻ** — مينيوز توهان جي سائيٽ جي ٿيم سان ملن ٿا
4. **موبائل بهترڪاري** — مينيوز فونن تي خوبصورت ڏيکارين ٿا
5. **SEO بهترڪاري** — مينيوز سرچ انجڻ لاءِ موافق آهن

## بهترين طريقا {#best-practices}

### مينيو ڊيزائن {#menu-design}

- **واضح تنظيم** — منطقي ڪيٽيگري جوڙجڪ
- **پڙهڻ لائق وضاحتون** — دلڪش ۽ معلوماتي
- **هڪجهڙي قيمت بندي** — واضح ڪرنسي ۽ فارميٽنگ
- **پيشورانه تصويرون** — اعليٰ معيار جون کاڌي جون تصويرون
- **خالي جاءِ** — صفحي کي گهڻو ڀريل نه ڪريو

### مواد {#content}

- **درست وضاحتون** — آئٽمز کي درست نموني بيان ڪريو
- **خاص شيون نمايان ڪريو** — خاص شين کي الڳ ڏيکاريو
- **الرجنز شامل ڪريو** — هميشه الرجنز لسٽ ڪريو
- **باقاعدگي سان اپڊيٽ ڪريو** — قيمتون ۽ آئٽمز موجوده رکو
- **دلڪش ٻولي استعمال ڪريو** — آئٽمز کي لذيذ محسوس ڪرايو

### رسائي {#accessibility}

- **پڙهڻ لائق فونٽس** — واضح، پڙهڻ ۾ آسان ٽائپ فيسز استعمال ڪريو
- **ڪافي تضاد** — يقيني بڻايو ته متن پڙهڻ لائق آهي
- **غذائي ليبلز** — غذائي اختيارن کي واضح طور نشان لڳايو
- **الرجن چتاءَ** — الرجنز کي نمايان طور ڏيکاريو
- **موبائل لاءِ موافق** — سڀني ڊوائيسز تي ٽيسٽ ڪريو
