# Superdav AI Agent

Superdav AI Agent هڪ ترقي يافته WordPress AI مددگار آهي، جيڪو Gratis AI Agent کي ٿيم-واقف صلاحيتن، بهتر ڪيل انٽرنيٽ ڳولا جي قابليتن، سائيٽ ٺاهيندڙ خاصيتن، ۽ اختياري خودڪاري انٽيگريشنز سان وڌائي ٿو. ورزن 1.18.0 Google Calendar پڙهڻ جا اوزار، رابطن ۽ شرڪت ڪندڙن جي ميپنگ، انساني منظوري جا دروازا، ياد ڏياريندڙن جي ٻيهر ورجاءَ کان بچاءُ، TextBee SMS نوٽيفڪيشنز، منظم Superdav خدمت جي فراهمي، ۽ ڀروسي لائق ڊولپر ورڪ فلو لاءِ الڳ ورهايل Advanced ساٿي پيڪيج شامل ڪري ٿو.

## اهم خاصيتون

- **ٿيم-واقف صلاحيتون** — توهان جي فعال WordPress ٿيم ۽ انسٽال ٿيل پلگ انز جي بنياد تي هدايت ۽ قابليتن کي پاڻمرادو ترتيب ڏئي ٿو
- **گهڻا انٽرنيٽ ڳولا فراهم ڪندڙ** — وڌيڪ ڀرپور ڳولا نتيجن ۽ ترقي يافته تحقيقاتي قابليتن لاءِ Brave Search ۽ Tavily مان چونڊيو
- **سائيٽ ٺاهيندڙ رابطو فارم** — پنهنجي انسٽال ٿيل فارم پلگ ان استعمال ڪندي چيٽ انٽرفيس مان سڌو مڪمل طور ڪم ڪندڙ رابطو فارم ٺاهيو
- **بهتر ڪيل سائيٽ ٺاهيندڙ** — بهتر پلگ ان دريافت ۽ غلطي بحالي سان بهتر ڪيل گهڻ-مرحلي سائيٽ تعمير منصوبا
- **تعمير ٿيل ٿيم صلاحيتون** — بلاڪ ٿيمز، ڪلاسڪ ٿيمز، Kadence Blocks، ۽ Kadence Theme لاءِ خاص هدايت
- **بهتر WooCommerce انٽيگريشن** — وڌيڪ ڀروسي ۽ هم آهنگي لاءِ native WooCommerce APIs استعمال ڪرڻ لاءِ ٻيهر ترتيب ڏنل
- **خودڪار فراهم ڪندڙ سڃاڻپ** — پلگ انز ۽ ٿيمز کي فعال يا غير فعال ڪرڻ وقت موجود پلگ انز ۽ ٿيمز کي پاڻمرادو سڃاڻي ۽ تازو ڪري ٿو
- **ڪسٽم لوگو ٺاهڻ** — خودڪار namespace-safe صافڪاري سان ڪسٽم SVG لوگو ٺاهيو ۽ شامل ڪريو
- **پيليٽ ڪنٽراسٽ جي تصديق** — پنهنجي ٿيم تي لاڳو ڪرڻ کان اڳ WCAG رسائي تعميل لاءِ رنگن جا جوڙا چيڪ ڪريو
- **تصويرن سان بهتر دريافت** — ڊزائن جي رخ تي اثر وجهڻ لاءِ Theme Builder دريافت انٽرويو دوران تصويرون اپلوڊ ڪريو
- **مهمانداري مينيوز** — ريسٽورنٽن، ڪيفيز، ۽ مهمانداري ڪاروبارن لاءِ منظم کاڌي ۽ مشروبات جي مينيو صفحا ٺاهيو
- **ڊيسڪ ٽاپ ۽ موبائل ڏيک** — ڊزائن-رخ چونڊ دوران پنهنجي ڊزائن کي ڊيسڪ ٽاپ ۽ موبائل ڊوائيسز تي اڳواٽ ڏسو
- **Tier 1 سائيٽ اسڪريپنگ** — sd-ai-agent/site-scrape هاڻي مقابلي ڪندڙ ۽ الهامي ويب سائيٽن جي تجزيي لاءِ ڊفالٽ طور موجود آهي
- **Google Calendar پڙهڻ جا اوزار** — ترتيب ڏنل ڪئلينڊر ۽ ايونٽس پڙهو ته جيئن خودڪاريون ايندڙ شيڊيول جي پسمنظر بابت دليل ڏئي سگهن
- **رابطن ۽ شرڪت ڪندڙن جي ميپنگ** — فالو-اپ ڪم ٺاهڻ کان اڳ ڪئلينڊر شرڪت ڪندڙن کي ڄاتل سائيٽ استعمال ڪندڙن ۽ رابطن سان ملائي ٿو
- **انساني منظوري جا دروازا** — پيغام موڪلڻ، ڪم ٻيهر شروع ڪرڻ، يا تبديليون ڪرڻ کان اڳ حساس خودڪارين کي جائزي لاءِ روڪي ٿو
- **ياد ڏياريندڙ رڪارڊ ۽ ٻيهر ورجاءَ کان بچاءُ** — اڳ ۾ موڪليل ياد ڏياريندڙن کي ٽريڪ ڪريو ته جيئن ورجندڙ خودڪاري هلڻ ساڳئي وصول ڪندڙ کي ٻه ڀيرا اطلاع نه ڏين
- **TextBee SMS نوٽيفڪيشنز** — جڏهن SMS فراهم ڪندڙ فعال هجي ته TextBee ذريعي ترتيب ڏنل ٽيڪسٽ-پيغام نوٽيفڪيشنز موڪليو
- **منظم Superdav خدمت فراهمي** — دستي endpoint ترتيب جي بدران پاڻمرادو سيٽ اپ سان سهائتا ٿيل سائيٽن کي hosted Superdav endpoints سان ڳنڍيو
- **Advanced ساٿي پيڪيج** — filesystem، database، WP-CLI، REST dispatcher، plugin builder، git snapshot، user-management، ۽ benchmark ورڪ فلو لاءِ الڳ ورهايل ڀروسي لائق ڊولپر اوزار

## گهرجون

- WordPress 6.0 يا وڌيڪ
- PHP 8.1 يا وڌيڪ
- WordPress Multisite (network-activated) يا single-site انسٽاليشن
- هڪ فعال AI فراهم ڪندڙ connector پلگ ان (Anthropic، OpenAI، يا compatible)
- Gratis AI Agent v1.9.0 يا وڌيڪ (Superdav AI Agent، Gratis AI Agent کي وڌائي ٿو)

## انسٽاليشن

1. پلگ ان فائلن کي `/wp-content/plugins/superdav-ai-agent` تي اپلوڊ ڪريو
2. WordPress ۾ **Plugins** اسڪرين ذريعي پلگ ان کي فعال ڪريو
3. پڪ ڪريو ته Gratis AI Agent پڻ فعال آهي
4. **Gratis AI Agent → Settings** ڏانهن وڃو ۽ ضرورت موجب پنهنجو انٽرنيٽ ڳولا فراهم ڪندڙ، منظم Superdav خدمت ڪنيڪشن، Google Calendar، يا TextBee سندون ترتيب ڏيو

WordPress.org پيڪيج ۾ Superdav AI Agent جون بنيادي خاصيتون شامل آهن. Advanced ساٿي پيڪيج انهن ماحولن لاءِ الڳ ٺهندو ۽ ورهايو ويندو آهي، جتي ڀروسي لائق ڊولپر اوزار واضح طور انسٽال ۽ مجاز ڪيا ويا هجن.

## تڪڙي شروعات

فعال ٿيڻ کان پوءِ، Superdav AI Agent خودڪار طور تي توهان جي Gratis AI Agent انسٽاليشن کي ٿيم-واقف صلاحيتن ۽ اضافي قابليتن سان بهتر ڪري ٿو. WordPress ايڊمن سائڊبار ۾ **Gratis AI Agent** پينل کوليو ۽ نيون خاصيتون استعمال ڪرڻ شروع ڪريو.

**مثالي هدايتون:**

```
رابطو صفحي ۾ رابطو فارم شامل ڪريو.
```

```
تازين WordPress سيڪيورٽي بهترين عملن لاءِ ويب ڳوليو.
```

```
فعال ٿيم جي بلاڪ نمونن کي استعمال ڪندي hero سيڪشن ٺاهيو.
```

```
سڀاڻي جي ترتيب ڏنل Google Calendar ايونٽس جو جائزو وٺو ۽ شرڪت ڪندڙن لاءِ ياد ڏياريندڙن جا مسودا ٺاهيو.
```

```
منهنجي پيغام منظور ڪرڻ کان پوءِ SMS ياد ڏياريندڙ موڪليو.
```

## ٿيم-واقف صلاحيتون

Superdav AI Agent توهان جي فعال ٿيم ۽ انسٽال ٿيل پلگ انز کي پاڻمرادو سڃاڻي ٿو، ۽ مناسب ٿيم-واقف صلاحيت کي مددگار جي پسمنظر ۾ لوڊ ڪري ٿو. موجود صلاحيتن ۾ شامل آهن:

| صلاحيت | لاڳو ٿئي ٿو |
|---|---|
| **بلاڪ ٿيمز** | WordPress بلاڪ-بنياد (مڪمل سائيٽ ايڊيٽنگ) آرڪيٽيڪچر استعمال ڪندڙ ٿيمز |
| **ڪلاسڪ ٿيمز** | PHP ٽيمپليٽس ۽ ڪلاسڪ ايڊيٽر استعمال ڪندڙ روايتي WordPress ٿيمز |
| **Kadence Blocks** | Kadence Blocks پلگ ان فعال رکندڙ سائيٽون |
| **Kadence Theme** | Kadence theme کي فعال ٿيم طور رکندڙ سائيٽون |

هر صلاحيت بابت تفصيلي هدايت لاءِ [ٿيم-واقف صلاحيتون](../../user-guide/configuration/theme-aware-skills) ڏسو.

## انٽرنيٽ ڳولا فراهم ڪندڙ

Superdav AI Agent v1.10.0 انٽرنيٽ ڳولا سوالن لاءِ Brave Search جي متبادل طور **Tavily** شامل ڪري ٿو. ٻئي فراهم ڪندڙ حقيقي-وقت ڳولا نتيجا پيش ڪن ٿا ۽ **Gratis AI Agent → Settings → Advanced** ۾ ترتيب ڏئي سگهجن ٿا.

| مهيا ڪندڙ | خصوصيتون | سيٽ اپ |
|---|---|---|
| **Brave Search** | تيز، رازداري تي مرڪوز ڳولا جا نتيجا | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | وڌيڪ شاهوڪار ڳولا جا نتيجا، ترقي يافته تحقيق جون صلاحيتون | [Tavily API](https://tavily.com/) |

ترتيب ڏيڻ جي هدايتن لاءِ [انٽرنيٽ ڳولا](../../user-guide/configuration/internet-search) ڏسو.

## ڪئلينڊر، منظوري، ۽ اطلاع جا ڪم-وهڪرا

Superdav AI Agent v1.18.0 خودڪاري ڪم-وهڪرن کي مواد ۽ سائيٽ ٺاهڻ وارن ڪمن کان اڳتي وڌائي ٿو:

- **Google Calendar اوزار** ايجنٽ کي ترتيب ڏنل ڪئلينڊر ۽ واقعي جا تفصيل پڙهڻ ڏين ٿا جڏهن ڪئلينڊر سندون موجود هجن.
- **رابطي ۽ شرڪت ڪندڙ مددگار** ياد ڏيارڻ يا فالو-اپ ڪم تيار ڪرڻ کان اڳ واقعي جي شرڪت ڪندڙن کي WordPress استعمال ڪندڙن يا ڄاتل رابطن سان ملائين ٿا.
- **انساني منظوري جا گيٽ** هڪ خودڪاري کي روڪڻ جي اجازت ڏين ٿا جيستائين ڪو بااختيار استعمال ڪندڙ تجويز ڪيل عمل جو جائزو نه وٺي.
- **ياد ڏيارڻ وارا رڪارڊ** جڏهن خودڪاري ٻيهر ڪوشش ڪئي وڃي يا بار بار شيڊول ڪئي وڃي ته نقل ڪئلينڊر يا اطلاع ياد ڏيارڻ کان بچائين ٿا.
- **TextBee SMS سپورٽ** جڏهن TextBee مهيا ڪندڙ ترتيب ڏنل هجي ته ٽيڪسٽ-پيغام اطلاعن کي فعال ڪري ٿي.

اهي اوزار شيڊول کان باخبر مدد، شرڪت ڪندڙن جي فالو-اپ، ۽ ياد ڏيارڻ وارن ڪم-وهڪرن لاءِ استعمال ڪريو، جن ۾ حساس پيغام موڪلڻ کان اڳ اڃا به انساني جائزي جي ضرورت هجي.

## سائيٽ ٺاهيندڙ جا رابطا فارم

سائيٽ ٺاهيندڙ ايجنٽ هاڻي چيٽ انٽرفيس مان سڌو رابطا فارم ٺاهي سگهي ٿو. بس سائيٽ ٺاهيندڙ کي رابطا فارم ٺاهڻ لاءِ پڇو، ۽ اهو پاڻمرادو توهان جو نصب ٿيل فارم پلگ ان (Contact Form 7، WPForms، Fluent Forms، يا Gravity Forms) سڃاڻندو ۽ استعمال لاءِ تيار فارم ٺاهيندو.

تفصيلي رهنمائي لاءِ [سائيٽ ٺاهيندڙ رابطا فارم](../../user-guide/administration/site-builder-contact-forms) ڏسو.

## ترتيب

سڀ سيٽنگون WordPress ايڊمن ۾ **Gratis AI Agent → سيٽنگون** هيٺ آهن. Superdav AI Agent هيٺيان ترتيب جا اختيار شامل ڪري ٿو:

| سيٽنگ | وضاحت |

## ترقي يافته ساٿي پيڪيج

Superdav AI Agent v1.18.0 ڀروسي جوڳن ڊولپر ڪم-وهڪرن لاءِ الڳ ورهايل ترقي يافته ساٿي پيڪيج شامل ڪري ٿو. اهي صلاحيتون WordPress.org core پيڪيج ۾ گڏ ٿيل ناهن. ساٿي پيڪيج صرف انهن سائيٽن تي نصب ڪريو جتي منتظمين ايجنٽ کي ڊولپر اوزارن تائين رسائي لاءِ واضح طور تي ڀروسو ڪن.

ترقي يافته اوزارن ۾ فائل سسٽم آپريشنز، ڊيٽابيس رسائي، WP-CLI، REST dispatcher ڪم-وهڪرا، پلگ ان ٺاهيندڙ يوٽلٽيز، git snapshots، استعمال ڪندڙ-انتظام مددگار، ۽ benchmark اوزار شامل آهن. اهي اوزار ڀروسي جوڳن منتظمين تائين محدود رکو ۽ پيداوار سائيٽن تي انهن کي فعال ڪرڻ کان اڳ approval-gate ترتيب جو جائزو وٺو.
