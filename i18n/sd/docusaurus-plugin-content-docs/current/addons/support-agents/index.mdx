# سپورٽ ايجنٽس

سڄي نيٽ ورڪ کي پاڻ سنڀالڻ آسان ڪم ناهي. جيئن توهان جو نيٽ ورڪ وڌڻ شروع ٿيندو، توهان سپورٽ جي وڌندڙ گهرج ڏسڻ شروع ڪندا. اهڙن ماڻهن کي شامل ڪرڻ سٺو خيال هوندو جيڪي توهان جو ڪم جو بار گهٽ ڪرڻ ۾ مدد ڪندا. تنهن هوندي، مڪمل رسائي/اجازت ڏيڻ خطري وارو آهي ۽ توهان جي نيٽ ورڪ کي متاثر ڪري سگهي ٿو. هتي هي سپورٽ ايجنٽ add-on ڪم اچي ٿو.

هي add-on توهان کي WordPress صارف جو هڪ نئون قسم ٺاهڻ جي اجازت ڏئي ٿو، جنهن کي سپورٽ ايجنٽ سڏيو وڃي ٿو، ۽ توهان کي ان صارف وٽ موجود اجازتن تي مڪمل ڪنٽرول ڏئي ٿو. اجازت ڏيڻ ايترو ئي آسان آهي جيترو هڪ ٽوگل بٽڻ آن ڪرڻ، ۽ جيڪڏهن توهان ان کي هٽائڻ جو فيصلو ڪريو ته ان کي آف ڪرڻ.

## خاصيتون

- WordPress ۽ Ultimate Multisite اجازتن جي شين وار فهرست
- آساني سان رسائي فعال ۽ غير فعال ڪريو
- Agent اڪائونٽس جو لامحدود تعداد ٺاهيو

## انسٽاليشن

1. 'ultimate-multisite-support-agents' کي '/wp-content/plugins/' ڊائريڪٽري ۾ اپلوڊ ڪريو
2. WordPress ۾ 'پلگ انز' مينيو ذريعي plugin کي فعال ڪريو
3. plugin کي سيٽ اپ ڪرڻ لاءِ قدم بہ قدم Wizard جي پيروي ڪريو
