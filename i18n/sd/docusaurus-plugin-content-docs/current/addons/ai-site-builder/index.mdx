# AI سائيٽ ٺاهيندڙ

ultimate-multisite-ai-site-builder

## گهرجون {#requirements}

- WordPress 5.3 يا وڌيڪ
- PHP 7.4 يا وڌيڪ
- Multisite Ultimate پلگ اِن (فعال)

## تنصيب {#installation}

1. اضافي جون فائلون پنهنجي `/wp-content/plugins/` ڊائريڪٽري ۾ اپلوڊ ڪريو
2. WordPress ۾ 'Plugins' مينيو ذريعي پلگ اِن فعال ڪريو
3. نيٽ ورڪ منتظم ۾ اضافي جون سيٽنگون ترتيب ڏيو

## ترقياتي ڪم {#development}

### ترتيب {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### منصوبي جي بناوت {#project-structure}

```
ultimate-multisite-ai-site-builder/
├── inc/                          # PHP ڪلاس ۽ شاملات
│   ├── class-multisite-ultimate-updater.php  # اپڊيٽ سنڀاليندڙ
│   └── class-ai_site_builder-main.php             # مکيه ڪارڪردگي
├── tests/                        # يونٽ ٽيسٽون
│   ├── bootstrap.php            # ٽيسٽ bootstrap
│   ├── class-*-base.php         # بنيادي ٽيسٽ ڪلاس
│   └── class-*-test.php         # ٽيسٽ ڪيس
├── views/                       # ٽيمپليٽ فائلون
├── assets/                      # جامد اثاثا
│   ├── css/                     # اسٽائل شيٽون
│   ├── js/                      # JavaScript فائلون
│   └── images/                  # تصويرون
├── lang/                        # ٻولي جون فائلون
├── composer.json                # PHP انحصار
├── package.json                 # Node.js انحصار
├── phpunit.xml.dist             # PHPUnit ترتيب
├── .phpcs.xml.dist              # ڪوڊ معيارن جي ترتيب
└── rector.php                   # ڪوڊ جديد ڪرڻ جا قاعدا
```

### ٽيسٽنگ {#testing}

اضافي ۾ هڪ جامع ٽيسٽنگ فريم ورڪ شامل آهي:

- **يونٽ ٽيسٽون**: WordPress ٽيسٽ سوئٽ سان PHPUnit-بنياد ٽيسٽون
- **بنيادي ٽيسٽ ڪلاس**: عام ٽيسٽ منظرنامن لاءِ مددگار طريقا مهيا ڪري ٿو
- **ٽيسٽ ڪوريج**: ڪوڊ ڪوريج کي ٽريڪ ڪرڻ لاءِ ترتيب ڏنل
- **Multisite ٽيسٽنگ**: ٽيسٽون multisite ماحول ۾ هلن ٿيون

### ڪوڊ معيار {#code-standards}

- **WordPress ڪوڊنگ معيار**: PHPCS ذريعي لاڳو ڪيل
- **PHP 7.4+ مطابقت**: جديد PHP خاصيتن جي حمايت
- **جامد تجزيو**: قسم چيڪنگ لاءِ PHPStan انٽيگريشن
- **ڪوڊ جديد ڪرڻ**: PHP اپگريڊن لاءِ Rector قاعدا

### بلڊ سسٽم {#build-system}

- **اثاثن جي پروسيسنگ**: CSS/JS فائلن لاءِ minification
- **ترجمو**: POT فائل جي تياري
- **پيڪيجنگ**: ورڇ لاءِ آرڪائيو ٺاهڻ
- **ترقياتي موڊ**: debugging لاءِ اڻ-minified اثاثا

## ترتيب {#configuration}

اضافو نيٽ ورڪ منتظم انٽرفيس ذريعي رسائي لائق مختلف ترتيب اختيارن کي سپورٽ ڪري ٿو.

## Hooks ۽ Filters {#hooks-and-filters}

### Actions {#actions}

- `ultimate-multisite-ai-site-builder_init` - اضافي جي شروعات کان پوءِ هلندو آهي
- `ultimate-multisite-ai-site-builder_loaded` - سڀ اضافي فائلون لوڊ ٿيڻ کان پوءِ هلندو آهي

### Filters {#filters}

- `ultimate-multisite-ai-site-builder_settings` - اضافي جون سيٽنگون تبديل ڪريو
- `ultimate-multisite-ai-site-builder_enabled` - اضافي جي فعال/غير فعال حالت کي override ڪريو
