# Mailster {#mailster}

پنهنجي multisite نيٽ ورڪ کي Ultimate Multisite Mailster Integration اضافي سان هڪ طاقتور اي ميل فهرست ٺاهڻ واري مشين ۾ تبديل ڪريو. جڏهن نوان گراهڪ سائن اپ ڪن يا خريداريون ڪن، ته انهن کي پاڻمرادو پنهنجي Mailster اي ميل فهرستن ۾ شامل ڪريو، جنهن سان توهان جي رڪنيت پليٽ فارم ۽ اي ميل مارڪيٽنگ حڪمت عملي جي وچ ۾ بي رڪاوٽ ڳانڍاپو ٺهي ٿو.

## اهم خاصيتون {#key-features}

**پاڻمرادو سبسڪرائبر هم وقت سازي**
* جڏهن گراهڪ سائن اپ ڪن، ته انهن کي پاڻمرادو Mailster فهرستن ۾ شامل ڪريو
* سائن اپ تي فوري هم وقت سازي ڪرڻ يا ادائيگي جي تصديق جو انتظار ڪرڻ چونڊيو
* دستي مداخلت کانسواءِ پنهنجون اي ميل فهرستون تازيون ۽ اپ ٽو ڊيٽ رکو

**لچڪدار فهرست انتظام**
* سڀني نون گراهڪن لاءِ عالمي ڊفالٽ فهرستون مقرر ڪريو
* هدف ٿيل مهمن لاءِ پيداوار-مخصوص فهرستن سان ڊفالٽس کي اوور رائيڊ ڪريو
* Mailster جي واقف انٽرفيس سان فهرست تفويضن کي آساني سان منظم ڪريو

**GDPR-مطابق آپٽ-ان اختيار**
* پاڻمرادو سبسڪرپشن يا چيڪ باڪس تي ٻڌل رضامندي مان چونڊيو
* تصديق ٿيل سبسڪرپشنز لاءِ ڊبل آپٽ-ان فعال ڪريو
* پنهنجي تعميل گهرجن جي بنياد تي آپٽ-ان چيڪ باڪس ڏيکاريو/لڪايو

**سمارٽ ڊيٽا ميپنگ**
* گراهڪن جا نالا ۽ رابطا ڄاڻ پاڻمرادو هم وقت ساز ڪريو
* بلنگ پتا سبسڪرائبر فيلڊن سان ميپ ڪريو
* هدف ٿيل مهمن لاءِ سائن اپ تاريخون ٽريڪ ڪريو

**پيداوار-سطح ڪنٽرول**
* مختلف پيداوارن کي مختلف فهرستون تفويض ڪريو
* گراهڪن جي خريد ڪيل شين جي بنياد تي هدف ٿيل مهمون ٺاهيو
* ضرورت هجي ته مخصوص پيداوارن لاءِ Mailster غير فعال ڪريو

**پيشوراڻو تجربو**
* فهرست چونڊ لاءِ مقامي Mailster UI انٽيگريشن
* مسئلا حل ڪرڻ ۽ تصديق لاءِ تفصيلي لاگنگ
* موجوده سبسڪرائبرن جون بي رڪاوٽ اپڊيٽون

## هنن لاءِ ڀرپور {#perfect-for}

* اي ميل فهرستون ٺاهيندڙ رڪنيت سائيٽ مالڪ
* نون گراهڪن کي نکارڻ چاهيندڙ SaaS پليٽ فارمز
* داخلا جي بنياد تي شاگردن کي ورهائيندڙ ڪورس ٺاهيندڙ
* Mailster سان Ultimate Multisite هلائيندڙ هر ڪو

## اهو ڪيئن ڪم ڪري ٿو {#how-it-works}

1. پنهنجي Ultimate Multisite نيٽ ورڪ تي اضافو انسٽال ۽ فعال ڪريو
2. Ultimate Multisite سيٽنگن ۾ عالمي ڊفالٽ فهرستون ترتيب ڏيو
3. اختياري طور پيداوار-مخصوص فهرستن سان اوور رائيڊ ڪريو
4. پنهنجو آپٽ-ان موڊ ۽ سبسڪرپشن جو وقت چونڊيو
5. پنهنجي Mailster فهرستن کي پاڻمرادو وڌندي ڏسو!

## انسٽاليشن {#installation}

1. پلگ ان فائلون پنهنجي Ultimate Multisite اضافن واري ڊائريڪٽري ۾ اپلوڊ ڪريو
2. پڪ ڪريو ته Mailster توهان جي مکيه سائيٽ تي انسٽال ۽ فعال آهي
3. Ultimate Multisite اضافن واري صفحي ذريعي اضافو فعال ڪريو
4. ترتيب ڏيڻ لاءِ سيٽنگون > Mailster Integration ڏانهن وڃو
5. پنهنجون ڊفالٽ فهرستون چونڊيو ۽ آپٽ-ان ترجيحات ترتيب ڏيو

## اڪثر پڇيا ويندڙ سوال {#frequently-asked-questions}

## ڇا هي اضافو Ultimate Multisite کانسواءِ ڪم ڪري ٿو؟ {#does-this-addon-work-without-ultimate-multisite}

نه، هن اضافي کي ڪم ڪرڻ لاءِ Ultimate Multisite گهربل آهي. اهو خاص طور تي Mailster کي Ultimate Multisite جي گراهڪ ۽ پيداوار نظام سان ڳنڍڻ لاءِ ٺاهيو ويو آهي.

## ڇا مون کي سڀني سائيٽن تي Mailster انسٽال ڪرڻ جي ضرورت آهي؟ {#do-i-need-mailster-installed-on-all-sites}

نه، Mailster صرف توهان جي مکيه نيٽ ورڪ سائيٽ تي انسٽال ۽ فعال هجڻ گهرجي. اضافو سڀني سب سائيٽن مان پاڻمرادو ان سان ڳنڍجي ويندو.

## ڇا مان مختلف پيداوارن کي مختلف فهرستون تفويض ڪري سگهان ٿو؟ {#can-i-assign-different-lists-to-different-products}

ها! توهان عالمي ڊفالٽ فهرستون مقرر ڪري سگهو ٿا، پوءِ انهن کي هر پيداوار جي بنياد تي اوور رائيڊ ڪري سگهو ٿا. اهو توهان کي گراهڪن جي خريد ڪيل شين جي بنياد تي انتهائي هدف ٿيل اي ميل مهمون ٺاهڻ جي اجازت ڏئي ٿو.

## جيڪڏهن ڪو گراهڪ اڳ ۾ ئي Mailster ۾ موجود هجي ته ڇا ٿيندو؟ {#what-happens-if-a-customer-already-exists-in-mailster}

ڊفالٽ طور، اضافو موجوده سبسڪرائبرن کي اپڊيٽ ڪندو ۽ انهن کي نين فهرستن ۾ شامل ڪندو. جيڪڏهن توهان موجوده سبسڪرائبرن کي ڇڏڻ پسند ڪريو ٿا ته توهان سيٽنگن ۾ اهو غير فعال ڪري سگهو ٿا.

## ڇا هي GDPR مطابق آهي؟ {#is-this-gdpr-compliant}

ها! اضافي ۾ چيڪ باڪس تي ٻڌل آپٽ-ان موڊ شامل آهي، جتي گراهڪن کي اي ميلون وصول ڪرڻ لاءِ واضح طور رضامندي ڏيڻ لازمي آهي. مناسب رازداري پاليسين سان گڏ، اهو GDPR گهرجن کي پورو ڪري ٿو.

## ڇا مان ڊبل آپٽ-ان استعمال ڪري سگهان ٿو؟ {#can-i-use-double-opt-in}

بلڪل! ڊبل آپٽ-ان سيٽنگ فعال ڪريو ۽ Mailster سبسڪرپشنز فعال ڪرڻ کان اڳ تصديقي اي ميلون موڪليندو.
