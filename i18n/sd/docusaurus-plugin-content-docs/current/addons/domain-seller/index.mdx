# ڊومين وڪڻندڙ

**Multisite Ultimate: ڊومين وڪڻندڙ** توهان جي WordPress Multisite نيٽ ورڪ کي هڪ طاقتور ڊومين وڪڻڻ واري پليٽ فارم ۾ تبديل ڪري ٿو. گراهڪن کي سائن اپ دوران يا پنهنجي Dashboard مان سڌو سنئون ڪسٽم ڊومين رجسٽر ڪرڻ ڏيو، حقيقي وقت دستيابي جاچ ۽ شفاف قيمت سان.

### ڊومين وڪڻندڙ ڇو؟

ڊومين رجسٽريشن ڪنهن به WaaS (Website as a Service) پليٽ فارم لاءِ سڀ کان وڌيڪ گهرج وارن اپ سيلز مان هڪ آهي. پنهنجي گراهڪن کي ڪنهن ٽئين ڌر رجسٽرار ڏانهن موڪلڻ بدران، توهان سڌو سنئون ڊومين پيش ڪري سگهو ٿا — پنهنجي نيٽ ورڪ ۾ وڌيڪ آمدني رکندي، گراهڪن کي بي رڪاوٽ تجربو ڏيندي.

### اهم خاصيتون

#### ڪيترن ئي ڊومين رجسٽرارن جي سپورٽ
* **OpenSRS** — مڪمل رجسٽريشن، تجديد، ۽ انتظام
* **Namecheap** — WhoisGuard ۽ سينڊباڪس جاچ سان مڪمل انٽيگريشن
* **HostAfrica** — Domains Reseller API ذريعي ڊومين ڳولا، قيمت، رجسٽريشن، منتقلي، DNS، ۽ ID تحفظ
* **Openprovider** — ريسيلر قيمت، رجسٽريشن، تجديد، منتقليون، DNS، WHOIS رازداري، ۽ TLD هم وقت سازي
* **Hostinger** — گڏيل Hostinger انٽيگريشن ٽوڪن استعمال ڪندي ڊومين ڳولا ۽ رجسٽريشن
* **وسعت پذير آرڪيٽيڪچر** — Integration Registry نموني تي ٺهيل، جنهن سان وڌيڪ فراهم ڪندڙ شامل ڪرڻ آسان ٿئي ٿو
* **بلٽ اِن ڪنيڪشن جاچ** — لائيو وڃڻ کان اڳ فراهم ڪندڙ جي اسناد ۽ ڪنيڪٽيوٽي جي تصديق ڪريو

#### رهنمائي ڪيل سيٽ اپ وزرڊ
* **هڪ-ڪلڪ آن بورڊنگ** — توهان کي فراهم ڪندڙ جي ترتيب، TLD امپورٽ، ۽ ڊومين پراڊڪٽ ٺاهڻ مان گذاري ٿو
* **خودڪار TLD هم وقت سازي** — توهان جي ڳنڍيل فراهم ڪندڙن مان سڀ موجود TLDs ۽ هول سيل قيمت امپورٽ ڪري ٿو
* **ڊفالٽ پراڊڪٽ ٺاهڻ** — مناسب مارڪ اپ ڊفالٽس سان استعمال لاءِ تيار ڊومين پراڊڪٽ ٺاهي ٿو

#### لچڪدار ڊومين قيمت
* **ٽي مارڪ اپ موڊ** — فيصد مارڪ اپ، مقرر مارڪ اپ، يا في TLD مقرر قيمت
* **TLD-مخصوص پراڊڪٽس** — مختلف TLD گروپن لاءِ الڳ ڊومين پراڊڪٽس ٺاهيو (.com/.net، .io، ملڪي ڪوڊز، وغيره)
* **تعارفي قيمت** — نون رجسٽريشنز کي راغب ڪرڻ لاءِ پهرين سال جي رعايتي قيمت پيش ڪريو
* **حقيقي وقت قيمت** — ذهين ڪيشنگ سان فراهم ڪندڙن مان لائيو هول سيل قيمت
* **ڪوپن سپورٽ** — Multisite Ultimate رعايتي ڪوڊز سان مڪمل مطابقت

#### WHOIS رازداري تحفظ
* **في-پراڊڪٽ ترتيب** — WHOIS رازداري کي Disabled، Always Included، يا Customer Choice تي سيٽ ڪريو
* **گراهڪ آپٽ-اِن** — جڏهن Customer Choice تي سيٽ هجي، چيڪ آئوٽ چيڪ باڪس گراهڪن کي رازداري تحفظ شامل ڪرڻ ڏئي ٿو
* **ترتيب ڏئي سگهجندڙ قيمت** — رازداري ايڊ-آن لاءِ في سال قيمت سيٽ ڪريو، يا اها مفت پيش ڪريو
* **رجسٽرار انٽيگريشن** — رجسٽريشن وقت پاڻمرادو WhoisGuard (Namecheap) يا WHOIS رازداري (OpenSRS) فعال ڪري ٿو

#### گڏيل چيڪ آئوٽ تجربو
* **ٽي ڊومين موڊ** — گراهڪ سب ڊومين (مفت)، نئون ڊومين رجسٽر ڪرڻ، يا اڳ ۾ پنهنجي ملڪيت وارو موجوده ڊومين ميپ ڪرڻ مان چونڊين ٿا
* **حقيقي وقت ڊومين ڳولا** — AJAX سان هلندڙ دستيابي جاچ بغير صفحو ٻيهر لوڊ ڪرڻ جي
* **سمارٽ تجويزون** — پسنديده ڊومين ورتل هجي ته متبادل TLD تجويزون
* **شفاف قيمت** — رجسٽريشن قيمت، تجديد قيمت، ۽ اختياري WHOIS رازداري فيس جي صاف ڏيک
* **رجسٽرنٽ رابطي جا فيلڊ** — چيڪ آئوٽ دوران گهربل رجسٽرنٽ معلومات (نالو، پتو، فون) اِن لائن گڏ ڪري ٿو
* **پاڻمرادو ٺهيل سائيٽ URL** — سائيٽ پتو چونڊيل ڊومين مان پاڻمرادو نڪتل هوندو آهي

#### خودڪار تجديد نظام
* **ميمبرشپ سان ڳنڍيل تجديدون** — جڏهن گراهڪ جي ميمبرشپ فعال هجي، ڊومين پاڻمرادو تجديد ٿين ٿا
* **سمارٽ شيڊولنگ** — تجديد جا ڪم ڊومين جي ختم ٿيڻ وارين تاريخن جي بنياد تي قطار ۾ رکيا وڃن ٿا
* **ٻيهر ڪوشش لاجڪ** — ناڪام تجديد ڪوششن لاءِ ذهين ٻيهر ڪوششون
* **اي ميل اطلاع** — ايندڙ تجديدن، ڪامياب تجديدن، ناڪامين، ۽ ختم ٿيڻ لاءِ ترتيب ڏئي سگهجندڙ الرٽس

#### گراهڪ DNS انتظام
* **سيلف سروس DNS** — گراهڪ پنهنجي Dashboard مان DNS رڪارڊز (A، AAAA، CNAME، MX، TXT) شامل، ايڊٽ، ۽ ڊليٽ ڪري سگهن ٿا
* **ڊومين اسٽيٽس** — حقيقي وقت رجسٽريشن اسٽيٽس، ختم ٿيڻ جون تاريخون، ۽ نيم سرور معلومات
* **خودڪار ڊومين ميپنگ** — رجسٽر ٿيل يا موجوده ڊومين پاڻمرادو گراهڪ جي سائيٽ سان ميپ ٿين ٿا

#### ايڊمن اوزار
* **دستي ڊومين رجسٽريشن** — ايڊمن پينل مان گراهڪن جي طرفان ڊومين رجسٽر ڪريو
* **ڊومين مانيٽرنگ** — هڪ صفحي تان سڀ گراهڪ ڊومين، اسٽيٽس، ۽ ختم ٿيڻ جون تاريخون ڏسو
* **فراهم ڪندڙ-مخصوص لاگنگ** — هر رجسٽرار لاءِ الڳ لاگ چينلز (مثال طور، domain-seller-namecheap.log)
* **رجسٽريشن تفصيل** — بنيادي ڊومين ايڊٽ صفحي تي رجسٽرنٽ معلومات ۽ رجسٽريشن ڊيٽا ڏسو

### سپورٽ ٿيل فراهم ڪندڙ

* **OpenSRS** — رجسٽريشن، تجديد، DNS انتظام، ۽ WHOIS رازداري سان مڪمل انٽيگريشن
* **Namecheap** — رجسٽريشن، تجديد، بيچ دستيابي جاچ، ۽ مفت WhoisGuard سان مڪمل انٽيگريشن
* **HostAfrica** — رجسٽريشن، تجديد، منتقليون، DNS انتظام، EPP ڪوڊ حاصل ڪرڻ، رجسٽرار لاڪ، ۽ ID تحفظ
* **Openprovider** — رجسٽريشن، تجديد، منتقليون، DNS انتظام، EPP ڪوڊ حاصل ڪرڻ، رجسٽرار لاڪ، WHOIS رازداري، ۽ ريسيلر TLD هم وقت سازي
* **Hostinger** — رجسٽريشن، ڊومين lookup، نيم سرور اپڊيٽس، رجسٽرار لاڪ، ۽ WHOIS رازداري؛ تجديدون Hostinger آٽو-رينيو سنڀاليندو آهي
* **GoDaddy** — رجسٽريشن ۽ تجديد
* **ResellerClub** — رجسٽريشن، تجديد، ۽ DNS انتظام
* **NameSilo** — رجسٽريشن ۽ تجديد
* **Enom** — رجسٽريشن، تجديد، ۽ DNS انتظام

### ٽيڪنيڪي تفصيل

* **گهربل:** PHP 7.4+، Multisite Ultimate 2.4.12+
* **آرڪيٽيڪچر:** Domain_Selling_Capability انٽرفيس سان Integration Registry نمونو
* **Autoloading:** PSR-4 ڪلاس ميپنگ سان Jetpack autoloader
* **معيار:** WordPress ڪوڊنگ معيار، strict typing، prepared statements

## انسٽاليشن

1. پڪ ڪريو ته **Multisite Ultimate v2.4.12+** انسٽال ٿيل ۽ نيٽ ورڪ-فعال ٿيل آهي
2. plugin کي `/wp-content/plugins/ultimate-multisite-domain-seller/` تي اپلوڊ ڪريو
3. WordPress نيٽ ورڪ ايڊمن ۾ **Plugins** مان plugin کي نيٽ ورڪ-فعال ڪريو
4. **سيٽ اپ وزرڊ** پهرين فعال ٿيڻ تي پاڻمرادو شروع ٿئي ٿو — هنن قدمن تي عمل ڪريو:
   * پنهنجي ڊومين رجسٽرار کي ڳنڍيو
   * ڪنيڪشن کي ٽيسٽ ڪريو
   * موجود TLDs ۽ هول سيل قيمتون درآمد ڪريو
   * قيمتي واڌ سان پنهنجو پهريون ڊومين پراڊڪٽ ٺاهيو
5. **Ultimate Multisite > چيڪ آئوٽ فارمز** ۾ پنهنجي چيڪ آئوٽ فارم ۾ **ڊومين چونڊ** فيلڊ شامل ڪريو

## اڪثر پڇيا ويندڙ سوال

### Multisite Ultimate جو ڪهڙو ورزن گهربل آهي؟

ورزن 2.4.12 يا ان کان مٿي. هي addon WP Ultimo v1.x يا 2.4.12 کان هيٺ وارن Ultimate Multisite ورزنز سان مطابقت نٿو رکي.

### ڪهڙا ڊومين فراهم ڪندڙ سپورٽ ٿيل آهن؟

نو رجسٽرار سپورٽ ٿيل آهن: **OpenSRS**، **Namecheap**، **HostAfrica**، **Openprovider**، **Hostinger**، **GoDaddy**، **ResellerClub**، **NameSilo**، ۽ **Enom**. صلاحيتون فراهم ڪندڙ موجب مختلف ٿين ٿيون، ڇاڪاڻ ته رجسٽرار APIs مختلف آپريشن ظاهر ڪن ٿيون؛ فراهم ڪندڙ-وار جائزي لاءِ سيٽ اپ گائيڊ ڏسو. addon جي Integration Registry اڏاوت ڪسٽم فراهم ڪندڙ شامل ڪرڻ کي سڌو بڻائي ٿي — [ڊولپر گائيڊ](/developer/integration-guide/custom-registrar) ڏسو.

### ڇا مان WHOIS رازداري تحفظ پيش ڪري سگهان ٿو؟

ها. هر ڊومين پراڊڪٽ ۾ WHOIS Privacy سيٽنگ آهي جنهن جا ٽي موڊ آهن: غيرفعال، هميشه شامل، يا گراهڪ جي چونڊ. جڏهن گراهڪ جي چونڊ تي سيٽ ڪيو وڃي، گراهڪ چيڪ آئوٽ دوران قيمت سان گڏ هڪ چيڪ باڪس ڏسن ٿا (يا Namecheap جي WhoisGuard لاءِ "مفت"). رازداري فليگ رجسٽريشن وقت پاڻمرادو رجسٽرار ڏانهن موڪليو وڃي ٿو.

### ڇا مان مختلف TLDs لاءِ مختلف قيمتون مقرر ڪري سگهان ٿو؟

ها. مختلف TLD گروپن لاءِ الڳ ڊومين پراڊڪٽس ٺاهيو. هر پراڊڪٽ جو پنهنجو مارڪ اپ قسم (سيڪڙو، مقرر مارڪ اپ، يا مقرر قيمت)، سپورٽ ٿيل TLDs، ۽ تعارفي قيمت ٿي سگهي ٿي.

### گراهڪ ڪهڙا چيڪ آئوٽ اختيار ڏسن ٿا؟

ڊومين چونڊ فيلڊ گراهڪن کي ٽي ٽيب ڏئي ٿو: **سب ڊومين** (توهان جي نيٽ ورڪ تي مفت سب ڊومين استعمال ڪريو)، **نئون ڊومين رجسٽر ڪريو** (نئون ڊومين ڳوليو ۽ رجسٽر ڪريو)، ۽ **موجوده ڊومين** (اهو ڊومين ميپ ڪريو جيڪو انهن وٽ اڳ ۾ ئي آهي). صرف اهي موڊ ظاهر ٿيندا جيڪي توهان فعال ڪندا.

### ڇا اهو پاڻمرادو تجديدن کي سپورٽ ڪري ٿو؟

ها. ڊومين پاڻمرادو تجديد لاءِ سيٽ ڪري سگهجن ٿا ۽ گراهڪ جي رڪنيت واري حيثيت سان ڳنڍيل آهن. تجديد ادائگيون گراهڪ جي موجوده ادائيگي گيٽ وي ذريعي پروسيس ڪيون وڃن ٿيون.

### ڇا گراهڪ پنهنجا DNS ريڪارڊس سنڀالي سگهن ٿا؟

ها. گراهڪن کي پنهنجي ڊيش بورڊ ۾ DNS مينيجمينٽ انٽرفيس ملي ٿو ته جيئن A، AAAA، CNAME، MX، ۽ TXT ريڪارڊس شامل، ايڊٽ، ۽ حذف ڪري سگهن. DNS مينيجمينٽ OpenSRS، ResellerClub، Enom، HostAfrica، ۽ Openprovider سان موجود آهي. hosted ڊومينز لاءِ Hostinger DNS بنيادي Hostinger ڊومين-ميپنگ انٽيگريشن سنڀالي ٿي.

### ڇا سيٽ اپ وزرڊ موجود آهي؟

ها. سيٽ اپ وزرڊ پهرين فعال ٿيڻ تي پاڻمرادو هلندو آهي ۽ توهان کي رجسٽرار ڳنڍڻ، TLDs درآمد ڪرڻ، ۽ پنهنجو پهريون ڊومين پراڊڪٽ ٺاهڻ ذريعي وٺي ويندو آهي. توهان addon سيٽنگز مان وزرڊ ٻيهر به هلائي سگهو ٿا.
