# Filter: wu_connector_enforcement_option_keys

مکيه سائيٽ مان لاڳو ڪيل AI provider option keys جي فهرست کي فلٽر ڪريو.

ڪسٽم يا ٽئين ڌر جي AI provider plugins لاءِ option keys شامل ڪريو ته جيئن انهن جون سيٽنگون به ذيلي سائيٽن تي مکيه سائيٽ مان ورثي ۾ ملن.

## پيرا ميٽر {#parameters}

| نالو | قسم | وضاحت |
|------|------|-------------|
| $keys | `string[]` | لاڳو ڪرڻ لاءِ option key نالا (اڳ ۾ ئي متحرڪ طور دريافت ڪيل connector keys ۽ EXTRA_PROVIDER_OPTIONS شامل آهن). |

### کان وٺي {#since}

- 1.2.0
