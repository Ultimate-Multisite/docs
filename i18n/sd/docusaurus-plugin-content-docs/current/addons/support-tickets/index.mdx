# سپورٽ ٽڪيٽون

**سپورٽ ٽڪيٽون** Ultimate Multisite لاءِ هڪ طاقتور addon آهي، جيڪو توهان جي WordPress نيٽ ورڪ ۾ هڪ جامع سپورٽ ٽڪيٽ نظام شامل ڪري ٿو. SaaS پليٽفارمن، WaaS (Website as a Service) فراهم ڪندڙن، ۽ multisite نيٽ ورڪن لاءِ بهترين، جن کي نيٽ ورڪ ۽ انفرادي سائيٽ ٻنهي سطحن تي گراهڪ سپورٽ مهيا ڪرڻي هجي.

## سپورٽ ٽڪيٽون ڇو چونڊيو؟

* **گهڻ-سطحي سپورٽ**: نيٽ ورڪ منتظم نيٽ ورڪ-وسيع ٽڪيٽون سنڀالين ٿا، جڏهن ته سائيٽ مالڪ پنهنجي گراهڪ سپورٽ پاڻ منظم ڪن ٿا
* **پيشوراڻو ڪم وهڪرو**: ترجيحن، حالتن، ۽ تفويضن سان مڪمل ٽڪيٽ حياتي چڪر جو انتظام
* **گراهڪ تجربو**: خوبصورت، responsive انٽرفيس جيڪو گراهڪن لاءِ مدد حاصل ڪرڻ آسان بڻائي ٿو
* **محفوظ ۽ ڀروسي جوڳو**: WordPress سيڪيورٽي جي بهترين عملن سان ٺهيل ۽ بهتر ڪيل ڊيٽابيس ڪارڪردگي سان
* **بي رڪاوٽ انضمام**: Ultimate Multisite جي گراهڪ ۽ سائيٽ انتظام سان مڪمل طور ڪم ڪري ٿو

## اهم خاصيتون

**مڪمل ٽڪيٽ انتظام**

* آسان حوالي لاءِ منفرد ٽڪيٽ نمبرنگ نظام
* ترتيب ڏئي سگهجندڙ ترجيحي سطحون (گهٽ، نارمل، اعليٰ، نازڪ)
* جامع حالت ڪم وهڪرو (کليل، جاري، انتظار ۾، حل ٿيل، بند)
* ٽڪيٽ درجابندي (عام، فني، بلنگ، خاصيت جي درخواست، بگ رپورٽ)
* اختياري پاڻمرادو تفويض سان سپورٽ ايجنٽن کي ٽڪيٽون تفويض ڪريو

**سلسليوار گفتگو**

* ٽڪيٽ جوابن لاءِ مڪمل گفتگو سلسلا
* عملي ۽ گراهڪ جي جوابن لاءِ الڳ انداز
* خانگي عملي نوٽس جيڪي گراهڪ نٿا ڏسي سگهن
* حقيقي وقت AJAX-طاقتور انٽرفيس
* فارميٽنگ اختيارن سان جديد متن ايڊيٽر

**فائل اٽيچمينٽس**

* drag-and-drop سپورٽ سان محفوظ فائل اپلوڊ
* ترتيب ڏئي سگهجندڙ فائل قسم ۽ سائيز جون پابنديون
* نقصانڪار فائل سڃاڻپ سان ترقي يافته سيڪيورٽي
* پهچ ڪنٽرولن سان محفوظ ڊائونلوڊ

**هوشيار اطلاع**

* نون ٽڪيٽن، جوابن، ۽ حالت تبديلين لاءِ پاڻمرادو اي ميل الرٽ
* ترتيب ڏئي سگهجندڙ اي ميل templates
* عملي ۽ گراهڪن لاءِ ڪردار-بنياد اطلاع
* ذاتي پيغامن لاءِ متحرڪ template متغير

**طاقتور منتظم انٽرفيس**

* sorting ۽ filtering سان ترقي يافته ٽڪيٽ فهرست
* حالت، ترجيح، ۽ تفويضن لاءِ تڪڙي ترميم
* ڪارائتي انتظام لاءِ گڏيل عمل
* ڪارڪردگي ماپن سان انگ اکر Dashboard
* جامع سيٽنگون panel

**خوبصورت گراهڪ پورٽل**

* سڀني ڊوائيسز تي صاف، responsive انٽرفيس
* mobile-first ڊيزائن
* ترقي يافته ڳولا ۽ filtering
* فارميٽنگ toolbar سان جديد ايڊيٽر
* استعمال ۾ آسان ٽڪيٽ جمع ڪرائڻ جا فارم

**لچڪدار انضمام**

* shortcodes يا Ultimate Multisite UI عنصر استعمال ڪريو
* پنهنجي سائيٽ تي ڪٿي به ٽڪيٽ فارم ۽ فهرستون ڏيکاريو
* multisite نيٽ ورڪ جوڙجڪ سان بي رڪاوٽ ڪم ڪري ٿو
* hooks ۽ filters سان وڌائي سگهجندڙ

## انهن لاءِ بهترين

* گراهڪ سپورٽ جي ضرورت رکندڙ SaaS پليٽفارم
* گهڻن client سائيٽن جو انتظام ڪندڙ WaaS فراهم ڪندڙ
* مرحليوار سپورٽ سان membership سائيٽون
* گهڻن brands جو انتظام ڪندڙ نيٽ ورڪ منتظم
* گراهڪ سروس مهيا ڪندڙ سائيٽ مالڪ

## انسٽاليشن

1. پڪ ڪريو ته Ultimate Multisite plugin توهان جي WordPress نيٽ ورڪ تي انسٽال ۽ فعال ٿيل آهي
2. addon فائلون پنهنجي `/wp-content/plugins/` ڊائريڪٽري ۾ اپلوڊ ڪريو
3. WordPress ۾ 'Plugins' مينيو ذريعي plugin کي نيٽ ورڪ تي فعال ڪريو
4. پنهنجي سيٽنگون ترتيب ڏيڻ لاءِ Network Admin > Support Tickets ڏانهن وڃو

## اڪثر پڇيا ويندڙ سوال

## ڇا هن لاءِ Ultimate Multisite گهربل آهي؟

ها، هن addon لاءِ Ultimate Multisite انسٽال ۽ فعال هجڻ ضروري آهي. اهو Ultimate Multisite جي گراهڪ ۽ سائيٽ انتظام خاصيتن سان گهريءَ طرح ضم ٿئي ٿو.

## ڇا سائيٽ مالڪ پنهنجي گراهڪ سپورٽ پاڻ سنڀالي سگهن ٿا؟

بلڪل! سائيٽ مالڪ پنهنجي گراهڪن کي سپورٽ مهيا ڪرڻ لاءِ ٽڪيٽ نظام استعمال ڪري سگهن ٿا، جڏهن ته نيٽ ورڪ منتظم نيٽ ورڪ-وسيع سپورٽ ٽڪيٽون سنڀالي سگهن ٿا.

## گراهڪ ٽڪيٽون ڪيئن جمع ڪرائين ٿا؟

توهان shortcodes يا Ultimate Multisite UI عنصر استعمال ڪندي پنهنجي سائيٽ تي ڪٿي به ٽڪيٽ جمع ڪرائڻ جا فارم شامل ڪري سگهو ٿا. گراهڪ صرف فارم ڀرين ٿا ۽ پنهنجي درخواست جمع ڪرائين ٿا.

## ڇا مان ٽڪيٽ حالتون ۽ ترجيحون ترتيب ڏئي سگهان ٿو؟

ها! توهان پنهنجي سپورٽ ڪم وهڪري سان ملائڻ لاءِ نيٽ ورڪ منتظم سيٽنگن ۾ ترجيحي سطحون، حالتون، ۽ ٽڪيٽ قسم مڪمل طور ترتيب ڏئي سگهو ٿا.

## ڇا اهو فائل اٽيچمينٽس کي سپورٽ ڪري ٿو؟

ها، گراهڪ ۽ عملو ٻنهي ٽڪيٽن ۽ جوابن سان فائلون ڳنڍي سگهن ٿا. سيڪيورٽي لاءِ اجازت ڏنل فائل قسمن ۽ سائيز حدن تي توهان کي مڪمل ڪنٽرول آهي.

## اطلاع ڪيئن سنڀاليا وڃن ٿا؟

نظام نون ٽڪيٽن، جوابن، ۽ حالت تبديلين لاءِ پاڻمرادو اي ميل اطلاع موڪلي ٿو. توهان سيٽنگن ۾ سڀ اي ميل templates ترتيب ڏئي سگهو ٿا.

## ڇا مان مخصوص سپورٽ ايجنٽن کي ٽڪيٽون تفويض ڪري سگهان ٿو؟

ها، توهان هٿ سان ٽڪيٽون ايجنٽن کي تفويض ڪري سگهو ٿا يا ٽڪيٽن کي پاڻمرادو ورهائڻ لاءِ auto-assignment فعال ڪري سگهو ٿا.

## ڇا عملي لاءِ خانگي نوٽس آهن؟

ها، عملي جا ميمبر خانگي جواب شامل ڪري سگهن ٿا، جيڪي صرف ٻين عملي ميمبرن کي نظر اچن ٿا، گراهڪن کي نه.

## ڇا اهو موبائل-دوست آهي؟

بلڪل! سڄو انٽرفيس mobile-first انداز سان ٺهيل آهي ۽ سڀني ڊوائيسز تي خوبصورتي سان ڪم ڪري ٿو.

## ڇا مان shortcodes استعمال ڪندي ٽڪيٽ فهرستون ۽ فارم ڏيکاري سگهان ٿو؟

ها! موجود shortcodes:
* `[wu_support_tickets]` - گراهڪ ٽڪيٽ فهرست ڏيکاريو
* `[wu_support_ticket_form]` - نئين ٽڪيٽ جمع ڪرائڻ جو فارم ڏيکاريو
* `[wu_support_ticket_view ticket_id="123"]` - مخصوص ٽڪيٽ ڏيکاريو

## ڪهڙا فائل قسم اپلوڊ ڪري سگهجن ٿا؟

توهان سيٽنگن ۾ اجازت ڏنل فائل قسم ترتيب ڏئي سگهو ٿا. نظام ۾ نقصانڪار فائلن کان بچاءَ لاءِ ترقي يافته سيڪيورٽي scanning شامل آهي.

## ڇا مان ٽڪيٽ ڊيٽا برآمد ڪري سگهان ٿو؟

ها، addon ۾ ٽڪيٽ ڊيٽا ۽ انگ اکرن لاءِ برآمد ڪرڻ جون صلاحيتون شامل آهن.
