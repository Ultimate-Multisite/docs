# PayFast

Multisite Ultimate: Payfast گيٽ وي

Payfast - ڏکڻ آفريڪا ۾ هڪ تيز ۽ محفوظ آن لائن ادائيگين جو گيٽ وي - کي Multisite Ultimate ادائيگي اختيار طور شامل ڪري ٿو.

## انسٽاليشن

1. 'ultimate-multisite-payfast' کي '/wp-content/plugins/' ڊائريڪٽري ۾ اپلوڊ ڪريو
2. WordPress ۾ 'پلگ اِنز' مينيو ذريعي پلگ اِن کي چالو ڪريو
3. پلگ اِن سيٽ اپ ڪرڻ لاءِ قدم بہ قدم وزرڊ جي پيروي ڪريو

## استعمال جي رهنمائي

PayFast ڏکڻ آفريڪا ۾ ڪاروبارن لاءِ هڪ اڳواڻ آن لائن ادائيگي پروسيسر آهي. اهو آن لائن واپارين کي ٽرانزيڪشنز کي آسان ۽ محفوظ بڻائي وڌڻ ۾ مدد ڏيڻ لاءِ وقف آهي. Ultimate Multisite استعمال ڪندڙ جي درخواست موجب، اسان هڪ add-on ٺاهڻ جو فيصلو ڪيو جيڪو PayFast سان انٽيگريشن کي ممڪن بڻائيندو. هي مضمون ڏيکاريندو ته توهان جي Ultimate Multisite نيٽ ورڪ ۾ ادائيگي وصول ڪرڻ لاءِ PayFast کي توهان جي ادائيگي گيٽ وي طور ڪيئن سيٽ اپ ڪجي.

پهريون قدم add-on انسٽال ڪرڻ آهي.

# PayFast سيٽ اپ ڪرڻ

add-on انسٽال ڪرڻ کان پوءِ، هاڻي توهان Ultimate Multisite سيٽنگز ڏانهن وڃي PayFast ترتيب ڏيڻ شروع ڪري سگهو ٿا.

**Ultimate Multisite سيٽنگز** صفحي هيٺ، توهان "Payments" ٽيب ڳولي سگهو ٿا. هن ٽيب هيٺ، توهان کي هن ادائيگي گيٽ وي کي فعال يا غير فعال ڪرڻ لاءِ ٽوگل بٽڻ سان گڏ "**Payfast** " ڏسڻ گهرجي.

![ادائيگي گيٽ وي سيٽنگز](/img/config/settings-payment-gateways.png)اڳتي وڌو ۽ ان کي فعال ڪرڻ لاءِ ٽوگل بٽڻ تي ڪلڪ ڪريو. پوءِ اهو باڪس کي وڌائيندو ته مختلف فيلڊز ڏيکاري جن کي توهان کي ڀرڻو آهي جهڙوڪ **Mechant ID, Merchant Key and PassPhrase.** توهان هي معلومات پنهنجي **Payfast** Account مان حاصل ڪري سگهو ٿا

<!-- اسڪرين شاٽ دستياب ناهي: PayFast وڌايل سيٽنگز Merchant ID، Merchant Key، ۽ PassPhrase فيلڊز ڏيکاري رهيون آهن -->

اڳيون قدم اها معلومات حاصل ڪرڻ لاءِ توهان جي PayFast Account ۾ لاگ اِن ڪرڻ آهي جنهن جي توهان کي ضرورت آهي.

لاگ اِن ٿيڻ تي، توهان پنهنجي **Payfast** Account Dashboard مان فوري طور اهي سندون ڏسي سگهو ٿا جن جي توهان کي ضرورت آهي.

<!-- اسڪرين شاٽ دستياب ناهي: PayFast Account Dashboard Merchant ID ۽ Merchant Key سندون ڏيکاري رهيو آهي -->توهان آساني سان سندون ڪاپي ۽ پيسٽ ڪري پنهنجي Ultimate Multisite سيٽنگز ۾ تفصيل ڀري سگهو ٿا. پوءِ تبديليون محفوظ ڪرڻ لاءِ **"Save settings** " بٽڻ تي ڪلڪ ڪريو.

<!-- اسڪرين شاٽ دستياب ناهي: Ultimate Multisite PayFast سيٽنگز سندن سان ڀريل ۽ Save بٽڻ سان -->
