# 内容同步变更日志

### 版本 1.1.0 - 发行于 2025-12-31
* 新增：支持同步所有文章类型（文章、页面、自定义文章类型）
* 新增：在内容选择步骤中进行文章类型过滤
* 新增：源站点选择步骤——可从任何站点同步，而不仅限于主站点
* 新增：模板站点的子站点预选
* 新增：从 post_content 中提取媒体，适用于所有文章类型
* 新增：分类法同步（分类、标签、自定义分类法）
* 更改：将“Elementor Template Sync”改名为“Content Sync”
* 更改：“Select Templates”步骤改名为“Select Data”
* 改进：仅在 elementor_library 文章类型中执行 Elementor 特定代码

### 版本 1.0.0 - 发行于 2025-12-19
* 初始发布
* 多步骤向导，用于模板和站点选择
* 批量处理，实时进度反馈
* 完整媒体同步，自动 ID 重映射
* 与 Ultimate Multisite 集成设置
* 通过网络设置支持站点排除
* 同步后清除 Elementor 缓存
* 模板分类法保留
