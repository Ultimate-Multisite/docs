# PayFast

Multisite Ultimate: Payfast 网关

Adds Payfast——南非快速且安全的在线支付网关——作为 Multisite Ultimate 的支付选项。

## Installation

1. 将 'ultimate-multisite-payfast' 上传到 '/wp-content/plugins/' 目录  
2. 通过 WordPress 的 `Plugins` 菜单激活插件  
3. 按照逐步向导设置插件  

## Usage Guide

PayFast 是南非领先的在线支付处理商。它致力于通过使交易无缝且安全来帮助在线商家成长。根据 Ultimate Multisite 用户的请求，我们决定创建一个插件，使与 PayFast 的集成成为可能。本文展示如何将 PayFast 设置为您的支付网关，以在您的 Ultimate Multisite 网络中接收付款。

第一步是安装插件。

# How to install the Payfast add-on:

转到插件页面并选择 Ultimate Multisite: Payfast Gateway 插件。

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-BmLWqj4yjt.png)

插件窗口将弹出。点击 `Install Now`。

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-1jpCakOHNy.png)

安装完成后，您将被重定向到插件页面。在此，只需点击 `Network Activate`，_Payfast_ 插件将在您的网络上激活。

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-49OQHBwPxk.png)

# Setting Up PayFast

安装插件后，您现在可以进入 Ultimate Multisite 设置并开始配置 PayFast。

在 **Ultimate Multisite settings** 页面，您可以找到“Payments”选项卡。在此选项卡下，您应该看到“**Payfast**”以及一个切换按钮，用于启用或禁用此支付网关。

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-07bPkaGJka.png)继续点击切换按钮以启用它。随后会展开一个框，显示您需要填写的不同字段，例如 **商户 ID、商户密钥和密码短语**。您可以从您的 **Payfast** 账户获取这些信息。

下一步是登录您的 PayFast 账户以获取所需信息。

登录后，您可以立即在您的 **Payfast** 账户仪表盘中看到所需的凭据。

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-KW0WnsxHIW.png)您可以直接复制并粘贴凭据，并在 Ultimate Multisite 设置中填写详细信息。然后点击 **"保存设置"** 按钮保存更改。

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-5MIrG21r3e.png)

就这样，您的 PayFast 支付集成完成！
