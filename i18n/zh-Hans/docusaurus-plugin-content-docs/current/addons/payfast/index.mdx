# PayFast

Multisite Ultimate: Payfast 网关

Adds Payfast - a fast and secure online payments gateway in South Africa - as a Multisite Ultimate payment option.

## 安装

1. 将 'ultimate-multisite-payfast' 上传到 '/wp-content/plugins/' 目录  
2. 在 WordPress 的 'Plugins' 菜单中激活插件  
3. 按照逐步向导设置插件

## 使用指南

PayFast 是南非领先的在线支付处理商，致力于通过实现无缝安全的交易来帮助在线商家成长。根据 Ultimate Multisite 用户的需求，我们决定创建一个插件，使与 PayFast 的集成成为可能。本文将展示如何将 PayFast 设置为您的支付网关，以在 Ultimate Multisite 网络中接收付款。

第一步是安装插件。

# 如何安装 Payfast 插件：

前往插件页面并选择 Ultimate Multisite: Payfast Gateway 插件。

![插件页面](/img/admin/addons-page.png)

插件窗口将弹出。点击“Install Now”。

![插件页面（带安装选项）](/img/admin/addons-page.png)

安装完成后，您将被重定向到插件页面。在此，只需点击“Network Activate”，Payfast 插件将在您的网络中激活。

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# 设置 PayFast

安装插件后，您现在可以进入 Ultimate Multisite 设置并开始配置 PayFast。

在 **Ultimate Multisite settings** 页面中，您可以找到“Payments”选项卡。在此选项卡下，您应该看到“Payfast”，并配有一个切换按钮，用于启用或禁用此支付网关。

![支付网关设置](/img/config/settings-payment-gateways.png)

点击切换按钮启用它。随后会展开一个框，显示您需要填写的不同字段，例如 **Merchant ID、Merchant Key 和 PassPhrase**。您可以从您的 **Payfast** 账户获取这些信息。

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

下一步是登录您的 PayFast 账户以获取所需信息。

登录后，您可以立即在 **Payfast** 账户仪表盘中看到所需凭证。

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->

您可以直接复制并粘贴凭证，然后在 Ultimate Multisite 设置中填写详细信息。随后点击 **“Save settings”** 按钮保存更改。

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

完成！您的 PayFast 支付集成已完成!
