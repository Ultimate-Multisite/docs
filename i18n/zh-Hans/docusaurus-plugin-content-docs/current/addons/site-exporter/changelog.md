# 站点导出器更新日志 {#site-exporter-changelog}

版本：1.0.6 - 发布于 2026-05-11
* 新增：站点导出现在会打包一个可自启动的 index.php，因此 ZIP 可以安装到全新主机上，无需单独安装插件
* 新增：网络导出允许管理员从站点导出管理页面将所有子站点导出到单个归档文件中

版本：1.0.5 - 发布于 2026-05-05
* 修复：向 WP_CLI polyfill stub 添加了 __callStatic，以处理未模拟的方法（如 get_config），防止测试引导失败

版本：1.0.4 - 发布于 2025-11-25

* 修复与 Ultimate Multisite 2.4 的兼容性。

版本：1.0.2 - 发布于 2025-09-28

* 将前缀重命名为 ultimate-multisite；更新文本域；提升版本号。

### 版本 1.0.1 - 发布于 2023-08-09 {#version-101---released-on-2023-08-09}

* 改进：现在确保获取导入器文件的正确文件路径。
* 修复：禁用对象缓存以防止导入期间出错。
* 修复：修复导入器插件下载链接无法工作的问题。

### 版本 1.0.0 - 发布于 2022-12-23 {#version-100---released-on-2022-12-23}

* 修复：导出表单弹窗未加载；
* 修复：导出页面上的 Javascript 文件未加载；
* 内部：新的构建流程；

### 版本 1.0.0-beta.4 - 发布于 2022-01-13 {#version-100-beta4---released-on-2022-01-13}

* 内部：添加了钩子和过滤器生成器；
* 内部：添加了 WP Ultimo stub，以提升开发者体验；
