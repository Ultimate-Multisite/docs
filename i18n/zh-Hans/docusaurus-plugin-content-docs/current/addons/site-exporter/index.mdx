# 站点导出器

## 内容

我们了解在网络中移动子站点到单站点安装以及相反的需求，以便当您的客户决定迁移他们的网站时。Site Exporter 和 Importer 插件使这项任务更容易。该工具使用 zip 文件导出和导入站点，也可用作备份。

## 功能

* 导出和导入站点

* 易于使用的界面

* 创建站点备份

## 文档
**如何导出和导入站点**

## 安装

1. 将 **ultimate-multisite-site-exporter** 上传到 **/wp-content/plugins/** 目录  
2. 在 WordPress 的 **插件** 菜单中激活插件  
3. 按步骤向导设置插件

## 如何导出站点

要从网络导出站点，请在 Ultimate Multisite 菜单中转到 **导出与导入**，在 **导出站点** 选项卡上，点击 **生成新的站点导出**。![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-_bBztobh1xIFoj0Zo3zo7bx842u3XFMgpc2b8kiHmGmBU6RjuxlSOSY4ZNzaWTPOJrb0TIW_Q0Dv9WvT0yD-WwOkZ7bR36Ij5aJa_l19cDTfuj9DADQqI5dGy3eIeUmoQA=s0)

搜索您想要导出的站点，并选择要包含在导出文件中的项目。您可以包含主题、插件和上传文件。如果您的站点太大，您可以选择在后台运行导出过程。![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-fk-i4RP0uZFj8kgfcQeRfAqWtCBelOjpOQARFfeBisVkeRSc1Qywq6Dpy66MnB6oBbnvdfJM359LdRSgqCLU1tNozo1u1OFH5svxtld8TZAXHY-CPGh92gWz9HQ8nlUcjg=s0)

点击 **导出站点** 按钮。您的导出站点将显示在 **现有导出** 列表中。点击 **下载 ZIP** 按钮下载文件。![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-mouH16ZCUuNaUg0PwF2JmMhhsKPJOztau21QIeB7DXqAcX4VI5FL_itxvnKmFo439xuGUWsntM3doG6yBkjS2gi6bNTjg-kLHC19m19br5xsgBl0XH8hy5LvuPN1uNtx9g=s0)

## 如何导入站点

要导入站点，请转到 **导入站点** 选项卡并上传您想要导入的站点的 .zip 文件。您可以将 URL 更改为新地址，或通过留空保持旧 URL。您还可以勾选删除导入后 .zip 文件的复选框。然后点击 **开始导入**。![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-Uq6wNJ4TPTuAtSJqKunxSJEn5cli7vbwYGgL1G3fcHwdvU7gSwW_J4xlBAtZ1fyVbDU7Pupi8uNFhw1Lzm4S2fjVTkbQoz7fiZPIc1qiy8qj3iGt-MNVlyLWmIZXFxPLYA=s0)

该过程可能需要一点时间。您将在导入器下方的列表中看到待处理的导入。![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-kMkwSY3bYrE5vjfJtLviYImhNBn_PwmuamaRDlQKS9rGwIy3a_jXV0glMNTU8fBmV6IKkNN7obXNQFqNaAJtAAbWDftzX76-qdCDKCrqdru_9taciFCayE9FNszZJTODvA=s0)

## 如何在单站点安装中导入站点

要将站点导入到单站点安装，首先在您的站点上安装并激活导入器。您可以在导出器选项卡中找到下载链接。![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-BV8-2YzBoNf7snFztjmCSqYxyJLGArMW3Gdk0gyxSBQjDQt9iazVDgy01eQWtijvF0qywh5mA8QRYMEVpr3IaPlJ2A8djjxwkIoDCT-tVLQKVfA2exXgW-EYWLrjNXMFLw=s0)

您可以在 **工具** 菜单左侧的子项 **导出与导入** 中找到导入器。![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-qxzJG3PwurqcqgiiB7ZGQ-dtjTVnLQLsqw7CrSjPTBtFCIpuaPN_BBNvMt6rSg__ezhrza_-tUzJpxifEB-r9ns0u_jflR1VONWSvUlYlIalhqmvan-dzXuAHx3BmvYoww=s0)

上传您想要导入的站点的 .zip 文件。您可以将 URL 更改为新地址，或通过留空保持旧 URL。您还可以勾选导入后删除 .zip 文件的复选框。然后点击 **开始导入**。![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-TR4lY9W8hsYWGJ2dcY7HC9cF2-uPtEJpWxCfHjdqdKy0Sg0OuZmfcxB3ubTO0dx6Odq_PBcx4q2NNBOrec_7nNOzsOK4zSS3VdF3IeXXioP12CrnA20dy2f0HOuEfsB8Gw=s0)

该过程可能需要一点时间。您将在导入器下方的列表中看到待处理的导入。![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-GTjHNmq2vUc930m_MzbKlA6uyO9iNBLAcGmINdlOE87DKsjAT-T5mipPiqVfmELkpD4sHCopAs3Bzn56QUk6XHybE9A0HKL4qo6RbGIwSQ4Gc5zapKhGpBcyhVAZi4qqYA=s0)

您可以点击 **取消导入** 停止该过程。![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-oGmtCq7br6Y9GAfw_Ch53TP8nkJ0hw77sQJpSCQ7fEj5qk2lJo3WuMl7NYqB9c8SmORis53bWHILwGLxUTbpv4trWhxibu-yu5DNu6yNWXfkRo0bcnAZJ5F2K1hYIMH6PQ=s0)
