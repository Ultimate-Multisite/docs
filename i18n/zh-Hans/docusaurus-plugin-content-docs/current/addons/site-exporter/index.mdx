# 站点导出器

## 内容

我们理解在网络中移动子站点到单站点安装以及相反的需求，以便客户决定迁移他们的网站。站点导出器和导入器插件使这项任务更容易。该工具使用 zip 文件导出和导入站点，也可用作备份。

## 功能

* 导出和导入站点
* 易于使用的界面
* 创建站点备份

## 文档

**如何导出和导入站点**

## 安装

1. 将 **ultimate-multisite-site-exporter** 上传到 **/wp-content/plugins/** 目录
2. 在 WordPress 的 **插件** 菜单中激活插件
3. 按照逐步向导设置插件

## 使用指南

使用 Ultimate Multisite，您可以从网络中导出站点，或将站点导入到您的多站点或单站点安装中。

要开始此过程，请转到我们的插件页面并安装 Ultimate Multisite：站点导出器 & 导入器插件。![插件页面](/img/admin/addons-page.png)

![插件页面（安装选项）](/img/admin/addons-page.png)

转到您的插件页面，点击 **网络激活** 您刚安装的插件。<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Site Exporter addon -->

**导出与导入** 选项位于 Ultimate Multisite 菜单的左侧。<!-- Screenshot unavailable: Ultimate Multisite menu showing Export and Import option -->

## 如何导出站点

要从网络导出站点，请转到 Ultimate Multisite 菜单中的 **导出与导入**，在 **导出站点** 选项卡上，点击 **生成新站点导出**。<!-- Screenshot unavailable: Export Sites tab with Generate new Site Export button -->

搜索您想要导出的站点，并选择要包含在导出文件中的项目。您可以包含主题、插件和上传文件。如果站点太大，您可以选择在后台运行导出过程。<!-- Screenshot unavailable: Site export options with site search, include themes/plugins/uploads checkboxes -->

点击 **导出站点** 按钮。导出的站点将显示在 **现有导出** 列表中。点击 **下载 ZIP** 按钮下载文件。<!-- Screenshot unavailable: Existing Exports list with Download ZIP button -->

## 如何导入站点

要导入站点，请转到 **导入站点** 选项卡并上传您想要导入的站点的 .zip 文件。您可以将 URL 更改为新地址，或通过留空保持旧 URL。您还可以勾选删除导入后 .zip 文件的复选框。然后点击 **开始导入**。<!-- Screenshot unavailable: Import a Site tab with zip upload field and Begin Import button -->

此过程可能需要一点时间。您将在导入器下方的列表中看到待处理的导入。<!-- Screenshot unavailable: Pending import in the list below the importer -->

您可以点击 **取消导入** 停止该过程。<!-- Screenshot unavailable: Cancel Import button to stop the import process -->

完成后，您的站点将已在网络中可用。

## 如何在单站点安装中导入站点

要将站点导入到单站点安装，首先在您的站点上安装并激活导入器。您可以在导出器选项卡中找到下载链接。<!-- Screenshot unavailable: Exporter tab with link to download single-site importer -->

您可以在左侧的 **工具** 菜单中的子项 **导出与导入** 找到导入器。<!-- Screenshot unavailable: Single-site Tools menu showing Export and Import submenu -->

上传您想要导入的站点的 .zip 文件。您可以将 URL 更改为新地址，或通过留空保持旧 URL。您还可以勾选导入后删除 .zip 文件的复选框。然后点击 **开始导入**。<!-- Screenshot unavailable: Single-site import form with zip upload and Begin Import button -->

此过程可能需要一点时间。您将在导入器下方的列表中看到待处理的导入。<!-- Screenshot unavailable: Single-site pending import in the list -->

您可以点击 **取消导入** 停止该过程。<!-- Screenshot unavailable: Cancel Import button to stop the import process -->

完成后，您的站点将被刚刚导入的站点替换。
