# 过滤器: wu_site_exporter_plugin_exclusion_list

允许开发者管理一个插件列表，该列表可能会被排除在生成的 zip 之外。

## 参数

| 名称 | 类型 | 描述 |
|------|------|------|
| $plugin_list | `array` | 将被排除的插件。 |

### 自

- 2.0.0

### 来源

插件列表。
