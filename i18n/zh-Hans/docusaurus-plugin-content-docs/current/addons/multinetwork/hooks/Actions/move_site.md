# 动作: move_site

在站点被移动到新网络后触发。

## 参数

| 名称 | 类型 | 描述 |
|------|------|------|
| $site_id | `int` | 已移动站点的 ID。 |
| $old_network_id | `int` | 原始网络的 ID。 |
| $new_network_id | `int` | 站点已移动到的网络的 ID。 |
