# 多网络

Ultimate Multisite: Multinetwork 插件扩展了您的 Multisite Ultimate 平台，提供强大的多网络功能。此插件使您的客户能够购买和管理完整的 WordPress 多站点网络，而不仅仅是单个站点。

## 关键功能

* **网络部署**: 自动为您的客户创建并配置完整的 WordPress 多站点网络
* **网络管理**: 为多网络环境提供完整的管理功能
* **隔离网络**: 每个客户网络独立运行，拥有自己的数据库表和配置
* **可扩展架构**: 通过优化资源分配，高效处理多个网络
* **集成**: 与 Multisite Ultimate 的订阅和计费系统无缝集成

## 用例

* **白标托管**: 为客户提供自有品牌的多站点网络
* **代理方案**: 使代理机构能够在单一平台上管理多个客户网络
* **企业部署**: 支持大型组织的复杂多网络需求
* **教育平台**: 为不同学校或部门创建独立网络

## 需求

* Multisite Ultimate 2.0.0 或更高版本
* WordPress Multisite 安装
* PHP 7.4 或更高版本
* MySQL 5.6 或更高版本

## 安装

1. 将插件文件上传到 `/wp-content/plugins/ultimate-multisite-multinetwork` 目录
2. 在 WordPress 的 '插件' 页面激活插件
3. 通过 Multisite Ultimate → 设置配置多网络设置

## 常见问题

## 什么是多网络？

多网络是 WordPress 的一项功能，允许您在单个 WordPress 安装中运行多个独立的多站点网络。

## 这与普通多站点有何不同？

普通多站点允许在一个网络内拥有多个站点。多网络则允许多个网络，每个网络都有自己的站点，所有网络都在同一个 WordPress 安装中。

## 这需要特殊的服务器配置吗？

是的，多网络需要特定的服务器设置和配置。请参阅文档以获取详细的设置说明。
