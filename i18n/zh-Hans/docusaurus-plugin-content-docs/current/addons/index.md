# Ultimate Multisite Addons

Addons 扩展 Ultimate Multisite，提供额外功能和集成。大多数插件是付费的，可在 [ultimatemultisite.com](https://ultimatemultisite.com) 购买。

## Installation

Addons 可以通过两种方式安装：

1. **Via the Addon Page** -- 在 WordPress 管理后台导航到您站点的插件管理页面
2. **Manual Download** -- 从 [your account](https://ultimatemultisite.com/my-account/downloads/) 下载并通过 Plugins > Add New 上传
