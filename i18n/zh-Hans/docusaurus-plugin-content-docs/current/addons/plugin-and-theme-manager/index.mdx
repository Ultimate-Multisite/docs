# 插件与主题管理器

Multisite Ultimate：插件与主题管理器

编辑或隐藏已安装插件和主题的元信息（标题、作者、缩略图和描述）。为插件和主题创建分类，以便让您的用户筛选它们，并为您的用户面板添加一个漂亮的自定义插件页面。

要使用此插件，您只需在网络管理员的插件或主题页面中导航，然后点击“编辑插件/主题信息”。

## 安装

1. 将 'ultimate-multisite-plugin-and-theme-manager' 上传到 '/wp-content/plugins/' 目录
2. 通过 WordPress 网络管理员中的 'Plugins' 菜单激活插件

## 使用指南

Ultimate Multisite：插件与主题管理器是一个强大的 **高级** Ultimate Multisite 插件，能够让您完全控制网络用户在网络中查看和交互插件与主题选项的方式。

此插件有三个用途：

  1. 它让您完全更改插件和主题的元数据，包括名称、描述、作者名称，甚至缩略图！
  2. 它让您对插件和主题进行分类，以便让您的用户轻松筛选。
  3. 它让您可以将旧的、无聊的插件页面替换为类似主题的新页面，并按类别排序。
