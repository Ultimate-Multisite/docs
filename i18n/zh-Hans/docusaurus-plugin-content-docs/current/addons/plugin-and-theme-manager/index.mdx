# 插件与主题管理器

Multisite Ultimate: 插件与主题管理器

编辑或隐藏已安装插件和主题的元信息（标题、作者、缩略图和描述）。为插件和主题创建分类，以便让您的用户对其进行筛选，并为用户面板添加一个漂亮的自定义插件页面。

要使用此插件，您只需在网络管理后台的插件或主题页面中导航，然后点击“编辑插件/主题信息”。

## 安装

1. 将 'ultimate-multisite-plugin-and-theme-manager' 上传到 '/wp-content/plugins/' 目录  
2. 在您的 WordPress 网络管理后台的“插件”菜单中激活该插件

## 使用指南

Ultimate Multisite: 插件与主题管理器是一个强大的 **高级** Ultimate Multisite 插件，能够让您完全控制网络用户在网络中查看和交互插件与主题选项的方式。

该插件有三个用途：

1. 它让您可以完全更改插件和主题的元数据，包括名称、描述、作者名称，甚至缩略图！  
2. 它让您对插件和主题进行分类，以便让您的用户轻松筛选。  
3. 它为您提供了将旧的、无聊的插件页面替换为类似主题页面的选项，并按类别进行排序。

下载并安装后，Ultimate Multisite 设置页面将出现一个新标签——_插件设置_。
