# Fluent Forms

此插件使得在 Multisite Ultimate 的产品、会员或站点中为 Fluent Forms 添加限制成为可能。  
Fluent Forms 使用自定义表而非自定义文章类型，因此常规限制选项无法与 Fluent Forms 一起使用。

## Features

- 为 Fluent Forms 或 Conversational Fluent Forms 添加限制  
- 当客户达到限制时，系统会显示友好的提示信息，建议他们升级套餐
