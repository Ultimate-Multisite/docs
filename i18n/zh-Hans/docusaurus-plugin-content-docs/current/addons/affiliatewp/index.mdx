# Multisite Ultimate: AffiliateWP 集成

使用强大的 AffiliateWP 来扩大您 Ultimo Network 的客户基础！

## 安装

1. 将 'ultimate-multisite-git ' 上传到 '/wp-content/plugins/' 目录  
2. 通过 WordPress 的 'Plugins' 菜单激活插件  
3. 按照逐步向导设置插件  

## 使用指南

[AffiliateWP](https://affiliatewp.com/) 已成为为您的在线业务构建联盟计划的首选解决方案。Ultimate Multisite: AffiliateWP 是一座桥梁，使您能够为您的高级网络构建同样的功能。

## 入门

首先，您 **需要激活** [**AffiliateWP**](https://affiliatewp.com/)！如果没有与之集成的插件，您将无法使用我们的插件。在您不打算让客户在他们的网站上使用它的情况下，AffiliateWP 应仅在您的主站点激活。

此外，您还需要在整个网络范围内激活 **Ultimate Multisite** 和 **Ultimate Multisite: AffiliateWP**。

## 将 Ultimate Multisite 添加为 AffiliateWP 集成

在您的主站点，前往 Affiliates Settings（联盟设置），在 Integrations（集成）选项卡中勾选 Ultimate Multisite。这将告诉 [AffiliateWP](https://affiliatewp.com/) 开始跟踪来自 Ultimate Multisite 注册流程的访问和转化。

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_在底部勾选 Ultimate Multisite 框并保存_

这基本上就是全部。来自推荐链接的访问将被跟踪，当该用户随后进行首次付款时，将创建转化，并为负责该转化的联盟分配佣金。

不过，只会跟踪新创建用户的首次付款。若要启用递归推荐付款，您需要 [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)。

## 启用递归推荐

安装完 [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) 后，前往 Affiliates Settings Recurring Referrals（联盟设置递归推荐）选项卡，启用递归推荐功能。

<!-- Screenshot unavailable: AffiliateWP Recurring Referrals settings with Enable Recurring Referrals toggle -->

_就这么简单：启用递归推荐，您就完成了！_

同样，这就是全部！

现在，每当在被推荐的账户收到新付款时，负责的联盟将自动获得其份额！
