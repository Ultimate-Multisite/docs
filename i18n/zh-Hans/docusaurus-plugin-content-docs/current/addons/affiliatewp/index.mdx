# AffiliateWP

Multisite Ultimate: AffiliateWP 集成

Use the powerful AffiliateWP to grow the client base of your Ultimo Network!

## Installation

1. 将 'ultimate-multisite-git ' 上传到 '/wp-content/plugins/' 目录
2. 通过 WordPress 的 'Plugins' 菜单激活插件
3. 按照逐步向导设置插件

## Usage Guide

[AffiliateWP](https://affiliatewp.com/) 已成为为您的在线业务构建联盟计划的首选解决方案。Ultimate Multisite: AffiliateWP 是让您为您的高级网络构建相同功能的桥梁。

## Getting Started

首先，您 **需要** 激活 [**AffiliateWP**](https://affiliatewp.com/)！如果没有 **与其集成的插件**，您将无法使用我们的插件。若您不打算让客户在他们的网站上使用它，AffiliateWP 只需在您的主站激活。

此外，您还需要在整个网络范围内激活 **Ultimate Multisite** 和 **Ultimate Multisite: AffiliateWP**。

## Adding Ultimate Multisite as an AffiliateWP Integration

在您的主站，前往 Affiliates Settings，然后在 Integrations 选项卡中勾选 Ultimate Multisite。这将告诉 [AffiliateWP](https://affiliatewp.com/) 开始跟踪来自 Ultimate Multisite 注册流程的访问和转化。

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-602126536867724dfc6f0a54-zkWh3TFms-1F9087E6-928C-4043-94B9-F85966AF05DA.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-602126536867724dfc6f0a54-zkWh3TFms-1F9087E6-928C-4043-94B9-F85966AF05DA.png)

_在底部勾选 Ultimate Multisite 框并保存_

这基本上就是全部。来自推荐链接的访问将被跟踪，当该用户随后完成首次付款时，转化将被创建，佣金将分配给负责该转化的联盟。

不过，只会跟踪新创建用户的首次付款。若要启用循环推荐付款，您需要 [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## Enabling Recurring Referrals

安装完 [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) 后，前往 Affiliates Settings Recurring Referrals 选项卡，启用循环推荐功能。

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-602126536867724dfc6f0a54-bmSN5h7Lk-4697DC4E-ABB9-4C34-BFD8-97E4B62C261C.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-602126536867724dfc6f0a54-bmSN5h7Lk-4697DC4E-ABB9-4C34-BFD8-97E4B62C261C.png)

_简单到极致：启用循环推荐，您就完成了！_

同样，这就是全部！

现在，每当在被推荐账户收到新付款时，负责的联盟将自动获得其份额！
