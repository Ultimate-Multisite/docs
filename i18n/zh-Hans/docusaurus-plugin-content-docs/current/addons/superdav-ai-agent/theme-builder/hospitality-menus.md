# 餐饮菜单

**餐饮菜单**功能允许 Theme Builder 直接在您的 WordPress 网站中生成和嵌入结构化的食品和饮料菜单页面。

## 概述

现在，Theme Builder 可以为酒店业（包括餐厅、咖啡馆、酒吧和餐饮服务）创建专业、结构化的菜单页面。这些菜单与您的网站设计完全集成，并且可以轻松更新和管理。

## 支持的餐饮类型

- **餐厅** — 全方位服务的餐饮菜单
- **咖啡馆** — 咖啡和轻食菜单
- **酒吧和休息室** — 饮品和开胃小吃菜单
- **面包店** —糕点和面包菜单
- **餐饮服务** — 活动菜单选项
- **餐车** — 移动餐饮服务菜单
- **酒厂和酒庄** — 带描述的饮品菜单

## 菜单结构

### 菜单分类

菜单会按照以下分类进行组织：

- **开胃小吃** — 前菜和小点心
- **主菜** — 主餐
- **配菜** — 搭配和蔬菜
- **甜点** — 甜食
- **饮品** — 饮料（含酒精和非酒精）
- **特色菜** — 当日或季节特色

### 菜单项格式

每个菜单项都包含以下信息：

```json
{
  "name": "烤三文鱼",
  "description": "新鲜大西洋三文鱼配柠檬黄油酱",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### 字段说明

| Field | Type | Description |
|-------|------|-------------|
| `name` | string | 菜品或饮品名称 |
| `description` | string | 详细的描述 |
| `price` | string | 价格（已格式化货币） |
| `dietary_info` | array | 饮食属性（纯素、无麸质等） |
| `allergens` | array | 常见过敏原（坚果、贝类等） |
| `availability` | string | 可用时间（每日、季节性等） |

## 创建餐饮菜单

### 步骤 1：提供菜单信息

告诉 Theme Builder 关于您的菜单：

```
为我的意大利餐厅创建一个菜单，包括开胃小吃、意大利面主食、主菜和甜点。请包含价格和描述。
```

### 步骤 2：Theme Builder 生成菜单

Theme Builder 将：

1. 创建一个结构化的菜单页面
2. 根据您的网站主题进行设计
3. 将项目组织到不同的分类中
4. 格式化价格和描述
5. 添加饮食和过敏原信息

### 步骤 3：审核和自定义

您可以：

1. 编辑菜单项和价格
2. 添加或删除分类
3. 重新排列分类内的项目顺序
4. 更新描述和饮食信息
5. 调整样式和布局

## 菜单展示选项

### 完整菜单页面

一个专门展示您完整菜单的页面：

- 按分类组织
- 可搜索和筛选
- 适合打印的布局
- 响应式移动端设计

### 菜单小工具 (Menu Widget)

在其他页面嵌入菜单部分：

- 首页特色项目
- 侧边栏当日特价
- 酒吧页面饮品菜单
- 页脚甜点展示

### 菜单 PDF

生成可下载的 PDF 菜单：

- 专业格式
- 印刷级质量
- 包含图片和描述
- 易于分享和通过电子邮件发送

## 饮食和过敏原信息

### 饮食属性

标记具有饮食信息的项目：

- **Vegan** — 不含任何动物产品
- **Vegetarian** — 不含肉类
- **Gluten-Free** — 适合乳糜泻患者
- **Dairy-Free** — 不含奶制品
- **Nut-Free** — 不含坚果或花生
- **Low-Carb** — 低碳水化合物
- **High-Protein** — 富含蛋白质

### 过敏原警告

包含常见过敏原：

- **Nuts** — 坚果和花生
- **Shellfish** — 甲壳类和软体动物
- **Fish** — 所有鱼类物种
- **Dairy** — 牛奶和奶制品
- **Eggs** — 鸡鸡蛋
- **Soy** — 大豆产品
- **Gluten** — 小麦和含麸质谷物
- **Sesame** — 芝麻籽和油

## 菜单管理

### 更新价格

轻松更新菜单价格：

1. 导航到菜单页面
2. 点击“编辑菜单”
3. 更新项目价格
4. 保存更改
5. 更改会立即显示在您的网站上

### 添加季节性项目

创建季节性菜单变体：

1. 创建一个新的菜单版本
2. 添加季节性项目
3. 将项目标记为“季节性”
4. 设置可用日期时间表
5. 在季节期间自动显示

### 管理特色菜

展示每日或每周特色菜：

1. 创建一个“特色菜”分类
2. 添加带有可用日期时间的项目
3. 在首页突出显示特色菜
4. 每日或每周更新
5. 归档旧的特色菜

## 与 Theme Builder 的集成

当您使用 Theme Builder 为酒店业网站时：

1. **自动菜单检测** — 识别您是否是食品/饮料企业
2. **菜单页面创建** — 生成专业的菜单页面
3. **设计匹配** — 菜单与您的网站主题保持一致
4. **移动端优化** — 菜单在手机上显示美观
5. **SEO 优化** — 菜单对搜索引擎友好

## 最佳实践

### 菜单设计

- **清晰的组织结构** — 合理的分类结构
- **易读的描述** — 诱人且信息丰富
- **一致的定价** — 清晰的货币和格式
- **专业的照片** — 高质量的食物图片
- **留白** — 不要让页面过于拥挤

### 内容

- **准确的描述** — 准确描述项目
- **突出特色菜** — 让特色菜脱颖而出
- **包含过敏原** — 始终列出过敏原
- **定期更新** — 保持价格和项目的新鲜度
- **使用诱人的语言** — 让项目听起来美味可口

### 可访问性

- **易读的字体** — 使用清晰、易读的字体
- **足够的对比度** — 确保文本易于阅读
- **饮食标签** — 清楚标记饮食选项
- **过敏原警告** — 醒目地显示过敏原
- **移动友好** — 在所有设备上进行测试
