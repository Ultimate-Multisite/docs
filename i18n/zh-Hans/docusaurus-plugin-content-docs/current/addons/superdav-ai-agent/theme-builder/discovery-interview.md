# 调研访谈

**调研访谈 (Discovery Interview)** 是 Theme Builder 设计流程的第一步。它会收集关于您的网站、业务和设计偏好等信息，从而为您打造一个个性化的主题。

## 概述

调研访谈会引导您完成一系列问题，帮助我们了解您的品牌、目标受众和设计目标。在 1.16.0 版本中，访谈流程新增了**照片上传步骤**，让您能够提供视觉灵感，指导我们的设计方向。

## 访谈步骤

### 1. 业务信息

- 网站名称和标语
- 业务类型或行业
- 主要业务目标
- 目标受众描述

### 2. 设计偏好

- 首选配色方案
- 设计风格（现代、经典、极简、活泼等）
- 灵感网站或品牌
- 必须具备的设计元素

### 3. 照片上传（v1.6.0 新增）

上传一张或多张照片，以影响设计方向：

#### 支持格式

| 格式 | 扩展名 | 详情 |
|--------|-----------|---------|
| **JPEG** | `.jpg`, `.jpeg` | 推荐用于照片 |
| **PNG** | `.png` | 支持透明度 |
| **WebP** | `.webp` | 现代格式，文件更小 |
| **GIF** | `.gif` | 仅支持静态图片（不支持动画） |

#### 文件要求

- **最大文件大小：** 10 MB
- **最小尺寸：** 400 × 300 像素
- **推荐尺寸：** 1200 × 800 像素或更大
- **宽高比：** 任何比例均可接受

#### 照片如何影响设计方向

您上传的照片有助于 Theme Builder 理解：

- **配色方案** — 会提取主导颜色并在主题中使用
- **视觉风格** — 现代、质朴、极简、充满活力等
- **情绪和氛围** — 专业、创意、友好、奢华等
- **设计元素** — 图案、纹理和构图风格
- **字体灵感** — 与视觉主题相匹配的字体风格

### 4. 内容结构

- 需要的主要页面（首页、关于我们、服务、联系方式等）
- 是否需要博客或新闻板块
- 是否需要电商功能
- 特殊功能或集成需求

### 5. 审核与确认

- 所有提供信息的总结
- 在设计生成开始前进行确认

## 照片上传最佳实践

### 建议上传的内容

- **品牌灵感** — Logo、色块或品牌指南
- **竞争对手网站** — 您欣赏的网站截图
- **情绪板 (Mood boards)** — 代表您所需美学的图片合集
- **产品照片** — 如果您有需要展示的产品
- **团队或办公室照片** — 用于个人或专业品牌展示
- **生活方式图片** — 代表您的目标受众

### 避免上传的内容

- **低质量图片** — 模糊或像素化的照片
- **不相关的图片** — 与您的品牌不相关的照片
- **文字过多的图片** — 包含大量文字的截图
- **受版权保护的材料** — 请确保您有权使用这些图片
- **动画 GIF** — 仅处理静态图片

### 最佳结果提示

1. **上传 2-3 张图片** — 多张图片能提供更全面的背景信息
2. **使用高质量照片** — 清晰、光线充足的图片效果最佳
3. **包含多样性** — 混合不同颜色、风格和主题的图片
4. **提供背景信息** — 简要描述您选择每张图片的原因
5. **具体化描述** — 如果您喜欢某个特定元素，请指出它

## 设计方向输出

完成调研访谈后，Theme Builder 将生成：

- **配色方案** — 主色、辅色和强调色
- **字体** — 标题和正文的字体家族
- **布局风格** — 间距、对齐和组件排列方式
- **视觉元素** — 图标、图案和装饰元素
- **情绪板** — 设计方向的视觉呈现

## 后续步骤

完成调研访谈后：

1. 审核生成的设计方向
2. 提供反馈或要求调整
3. 进入 [设计方向](./design-direction.md) 步骤
4. Theme Builder 将创建您的定制主题
