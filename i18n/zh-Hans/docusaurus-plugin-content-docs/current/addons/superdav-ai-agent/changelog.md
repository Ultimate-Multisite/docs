# 更新日志 {#changelog}

## 1.18.0 — 发布于 2026-06-29 {#1180--released-on-2026-06-29}

### 新增 {#new}

- **Google Calendar 工具** — 读取已配置的日历和事件，用于感知日程的自动化
- **联系人映射和参会者辅助工具** — 将日历参与者匹配到站点用户和联系人
- **人工审批门和提醒记录** — 暂停自动化以供审核，并避免重复通知
- **TextBee SMS 提供商** — 通过 TextBee 发送已配置的短信通知
- **高级伴随包** — 添加受信任开发者的文件系统、数据库、WP-CLI、REST 调度器、plugin 构建器、git 快照、用户管理和基准测试工具，并与 WordPress.org 构建分开发行

### 改进 {#improved}

- **托管 Superdav 服务设置** — 为受支持的站点添加托管服务 endpoint 和自动连接预配
- **发布打包** — 构建单独的核心和高级 ZIP，在 GitHub 上同时发布，并仅将核心包发送到 WordPress.org

### 修复 {#fixed}

- **AI 请求可靠性** — 改进模型选择、请求超时、默认设置模型、推理文本处理，以及对无效工具调用的重试指导
- **日历和提醒加固** — 强化 Google Calendar token 和提醒去重
- **入门引导和审批恢复** — 修复前端入门引导启动和已确认能力恢复
- **WordPress.org 打包审核问题** — 处理核心发布的打包审核反馈

## 1.16.0 — 发布于 2026-05-20 {#1160--released-on-2026-05-20}

### 新增 {#new-1}

- **生成 Logo SVG 能力** — Theme Builder 现在可以生成并嵌入自定义 Logo SVG，并进行命名空间安全的清理
- **发现访谈中的照片上传** — Theme Builder 发现访谈现在包含照片上传步骤，以提供更丰富的设计上下文
- **验证调色板对比度能力** — 在应用到 theme 之前检查颜色对是否符合 WCAG
- **酒店餐饮菜单** — Theme Builder 现在可以为酒店餐饮企业生成结构化的食品和饮品菜单页面
- **桌面和移动端预览渲染** — 在设计方向选择期间，在桌面和移动设备上预览你的设计
- **导航标签参数** — 创建菜单能力现在支持独立于页面标题的 `navigation_label`
- **第 1 层工具可用性** — sd-ai-agent/site-scrape 现在是 Theme Builder 中默认可用的第 1 层工具

### 修复 {#fixed-1}

- **AI Client 缓存** — 现在由 transients 支持以实现跨请求持久化，防止长时间运行的 agent 任务中的数据丢失
- **Plugin 行操作链接** — 已更正并重命名以提高清晰度

## 1.10.0 — 发布于 2026-05-05 {#1100--released-on-2026-05-05}

### 新增 {#new-2}

- **Tavily 互联网搜索** — 添加 Tavily 作为搜索提供商，与 Brave Search 一起提供更丰富的互联网搜索结果
- **感知 theme 的内置技能** — Block Themes、Classic Themes、Kadence Blocks 和 Kadence Theme 技能指南现在随 plugin 一起提供
- **站点构建器联系表单能力** — 直接从聊天界面向任何页面添加联系表单

### 改进 {#improved-1}

- **WooCommerce 集成已重构** — 现在使用原生 WooCommerce API，以获得更好的可靠性和兼容性
- **提供商列表自动刷新** — 当任何 plugin 被激活或停用时

### 修复 {#fixed-2}

- **navigate-to 能力** — 修复某些管理页面上的无限重新加载循环
- **list-posts 能力** — 现在可正确将分类和标签名称解析为 slugs
- **WP-CLI 命令** — 在之前的重构后恢复缺失的命名空间别名
- **事件自动化** — 优雅处理尚未创建自动化表的站点
- **memory-save 能力** — 现在在系统指令构建器中使用正确的命名空间前缀
- **标量工具结果** — 现在会在返回给 AI 之前正确包装
- **使用统计** — 现在在从旧版本升级时正确处理旧版能力键格式
