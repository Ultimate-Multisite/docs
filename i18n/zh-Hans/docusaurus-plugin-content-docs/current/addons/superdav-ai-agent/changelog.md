# 更新日志

## 1.16.0 — 发布于 2026-05-20

### 新增功能

- **生成 Logo SVG 能力** — 主题构建器现在可以生成和嵌入自定义的、经过命名空间安全清理的 SVG 标志。
- **发现访谈中的照片上传** — 主题构建器的发现访谈现在包含照片上传步骤，以获取更丰富的设计背景信息。
- **验证调色板对比度能力** — 在将颜色应用到主题之前，检查颜色组合是否符合 WCAG 标准。
- **酒店餐饮菜单** — 主题构建器现在可以为酒店业生成结构化的食品和饮料菜单页面。
- **桌面和移动端预览渲染** — 在选择设计方向时，可以在桌面和移动设备上预览您的设计。
- **导航标签参数** — 创建菜单功能现在支持一个独立的 `navigation_label`，该标签与页面标题分开。
- **Tier 1 工具可用性** — sd-ai-agent/site-scrape 现在是主题构建器默认可用的 Tier 1 工具。

### 修复问题

- **AI 客户端缓存** — 现在使用瞬态（transients）支持跨请求持久化，防止长时间运行的代理任务丢失数据。
- **插件行操作链接** — 已更正并重命名，以提高清晰度。

## 1.10.0 — 发布于 2026-05-05

### 新增功能

- **Tavily 互联网搜索** — 添加 Tavily 作为搜索提供商，与 Brave Search 一起提供更丰富的互联网搜索结果。
- **具备主题意识的内置技能** — Block Themes、Classic Themes、Kadence Blocks 和 Kadence Theme 的技能指南现在随插件提供。
- **网站构建器联系表单功能** — 可直接从聊天界面向任何页面添加联系表单。

### 改进优化

- **WooCommerce 集成重构** — 现在使用原生的 WooCommerce API，以提高可靠性和兼容性。
- **提供商列表自动刷新** — 当任何插件被激活或禁用时。

### 修复问题

- **`navigate-to` 功能** — 修复了某些管理页面上的无限重新加载循环。
- **`list-posts` 功能** — 现在能正确地将分类和标签名称解析为 slug。
- **WP-CLI 命令** — 在之前的重构后，恢复了丢失的命名空间别名。
- **事件自动化** — 优雅地处理了尚未创建自动化表的站点。
- **`memory-save` 功能** — 现在在系统指令构建器中使用正确的命名空间前缀。
- **Scalar 工具结果** — 现在在返回给 AI 之前进行了正确的封装。
- **使用统计数据** — 现在能正确处理从旧版本升级时遗留的 ability key 格式。
