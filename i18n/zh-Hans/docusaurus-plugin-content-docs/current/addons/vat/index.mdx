# 增值税

Multisite Ultimate: 欧洲增值税

为 Multisite Ultimate 的默认税务处理添加额外功能和工具，以更好地支持欧洲增值税。  
自动将欧盟增值税税率添加到税率页面。

## 安装

1. 将 'ultimate-multisite-vat' 上传到 '/wp-content/plugins/' 目录  
2. 通过 WordPress 的 'Plugins' 菜单激活插件  
3. 前往 Ultimate Multisite > Settings > Taxes 并启用 EU Vat。  
4. 点击 Tax Rates 并在底部更新 EU VAT Rates，然后保存税率。  
5. 完成！
