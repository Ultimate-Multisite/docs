# 支持代理变更日志

版本：1.0.13 - 发布于 2025-09-28

* 将前缀重命名为 ultimate-multisite；更新文本域；版本升级。

版本 1.0.12 - 发布于 2025-09-17

* 修复：Ultimate Multisite 2.4.4 兼容性

版本 1.0.11 - 发布于 2023-11-21

* 修复：Ultimate Multisite 2.3.0 兼容性

版本 1.0.10 - 发布于 2023-08-30

* 修复：批量删除支持代理

版本 1.0.9 - 发布于 2023-08-09

* 添加：添加读取结账表单的功能。
* 修复：支持代理权限表单布局；
* 修复：支持代理删除表单上的警告信息；
* 修复：隐藏网络仪表盘小部件给未授权用户。
* 修复：纠正支持代理编辑和删除权限。
* 修复：修正创建支持代理的流程。
* 修复：阻止将客户添加为支持代理。
* 修复：将未授权用户重定向到编辑支持代理的管理页面。
* 修复：改进支持代理权限表单的布局。
* 内部：实现了新的构建流程。

版本 1.0.8 - 发布于 2022-01-21

* 内部：添加了钩子和过滤器生成器；
* 内部：添加了 Ultimate Multisite 存根以提升开发者体验；
* 添加：结账表单功能；
* 修复：Ultimate Multisite 表单与此插件选项之间的权限名称不匹配；

版本 1.0.7 - 发布于 2021-12-02

* 修复：删除支持代理无效；
* 修复：批量删除支持代理无效；

版本 1.0.6 - 发布于 2021-11-22

* 修复：在核心性能优化后，新增模态窗口无效；
* 修复：核心中存在 Grid-item 视图，而插件中不存在；

版本 1.0.5 - 发布于 2021-10-31

* 修复：与 PHP 8 的小兼容性问题；
* 改进：wp_ultimo_skip_network_active_check 现在使用 wu_is_must_use();

版本 1.0.4 - 2021-09-24

* 添加：针对 mu-plugins 设置的 wp_ultimo_skip_network_active_check 过滤器；

版本 1.0.3 - 发布于 2021-08-29

* 添加：对 REST API 的支持——支持代理可以通过 REST 创建；

版本 1.0.2 - 发布于 2021-07-26

* 修复：更新代理权限无效；

版本 1.0.1 - 发布于 2021-05-28

* 添加：当用户已是超级管理员时的提示；
* 添加：允许用户切换的权限；
* 改进：更好的表单字段；

版本 1.0.0 - 初始发布
