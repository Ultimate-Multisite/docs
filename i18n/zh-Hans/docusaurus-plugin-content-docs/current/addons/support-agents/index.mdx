# 支持代理

单独管理整个网络并不容易。随着网络的增长，你会发现对支持的需求不断增加。请考虑引入人员来减轻你的工作负担。然而，授予完全访问/权限存在风险，可能会危及你的网络安全。正是因为这个原因，才有了支持代理插件。

## Features

- WordPress 与 Ultimate Multisite 权限的分项列表
- 轻松启用和禁用访问权限
- 创建无限数量的代理账户

## Installation

1. 将 'ultimate-multisite-support-agents' 上传到 '/wp-content/plugins/' 目录
2. 在 WordPress 的 'Plugins' 菜单中激活插件
3. 按照逐步向导设置插件
