# 多租户

将子站点分布在服务器和数据库中，以提升性能和安全性

## Requirements

- WordPress 5.3 或更高版本
- PHP 7.4 或更高版本
- Multisite Ultimate 插件（已激活）

## Installation

1. 将插件文件上传到您的 `/wp-content/plugins/` 目录
2. 通过 WordPress 的 '插件' 菜单激活插件
3. 在网络管理中配置插件设置

## Development

### Setup

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Testing

插件包含全面的测试框架：

- **单元测试**：基于 PHPUnit 的测试，使用 WordPress 测试套件
- **基类测试**：提供常见测试场景的辅助方法
- **测试覆盖率**：配置以跟踪代码覆盖率
- **多站点测试**：在多站点环境中运行测试

### Code Standards

- **WordPress 编码规范**：通过 PHPCS 强制执行
- **PHP 7.4+ 兼容性**：支持现代 PHP 特性
- **静态分析**：集成 PHPStan 进行类型检查
- **代码现代化**：Rector 规则用于 PHP 升级

### Build System

- **资产处理**：CSS/JS 文件的压缩
- **翻译**：POT 文件生成
- **打包**：分发归档创建
- **开发模式**：调试用的未压缩资产

## Configuration

插件支持多种可通过网络管理界面访问的配置选项。

## Hooks and Filters

### Actions

- `ultimate-multisite-addon-template_init` - 插件初始化后触发
- `ultimate-multisite-addon-template_loaded` - 所有插件文件加载后触发

### Filters

- `ultimate-multisite-addon-template_settings` - 修改插件设置
- `ultimate-multisite-addon-template_enabled` - 覆盖插件启用/禁用状态
