# 邮件功能

Ultimate Multisite: Emails 附加功能允许网络所有者在管理网站和会员资格的同时，配置和管理客户的电子邮件账户。版本 1.1.0 新增了面向客户的邮件账户管理、Webmail 单点登录（SSO）、CyberPanel 转发器支持，并改进了结账时的账户配置流程。

## 主要功能

* **客户邮件账户小组件**: 客户无需提交支持工单，即可在账户页面创建和管理邮件账户。
* **Webmail SSO 和魔术链接**: 当提供商支持时，客户可以直接从账户页面通过一键认证访问 Webmail。
* **密码更改**: 客户和管理员都可以通过邮件管理小组件更新邮箱密码。
* **结账时配置**: 客户邮件账户的创建可以在结账过程中运行，确保购买流程完成后邮箱即可使用。
* **CyberPanel 转发器**: 基于 CyberPanel 的网络可以作为邮件服务流程的一部分创建和管理邮件转发器。
* **提供商感知设置**: 设置页面会筛选出当前激活和已配置的集成提供商，确保管理员只看到可用的选项。

## 要求

* Ultimate Multisite 2.0.0 或更高版本
* WordPress Multisite 安装
* 至少配置了一个邮件提供商集成
* 提供商 API 凭证，并具有创建邮箱、更改密码和提供 Webmail 访问链接（如果支持）的权限

## 安装

1. 将插件文件上传到 `/wp-content/plugins/ultimate-multisite-emails`。
2. 在 WordPress 网络管理器的 **Plugins** 中，网络激活此附加功能。
3. 导航到 **Ultimate Multisite** > **Settings**，打开附加功能的邮件设置。
4. 启用您希望客户使用的邮件提供商集成。
5. 保存设置，然后使用非生产客户或暂存站点测试账户创建流程。

## 客户账户邮件管理

版本 1.1.0 在客户账户页面添加了邮件管理小组件。当客户需要自行维护和创建自己网站的邮箱时，可以使用此功能。

从账户页面，客户可以：

1. 查看连接到其当前会员资格或网站的邮件账户。
2. 在其套餐允许额外账户时，创建新邮箱。
3. 更改现有邮箱的密码。
4. 通过 SSO 或魔术链接登录访问支持的 Webmail 提供商。

该小组件使用了 Ultimate Multisite 账户页面的当前系统，因此所有操作都限定在客户正在查看的账户上下文内。这可以防止客户管理属于其他会员资格或网站的邮件账户。

## Webmail SSO 和魔术链接登录

Webmail SSO 为客户提供了从账户页面直接 **Open webmail** 的操作。当活动提供商支持此功能时，附加功能会请求一个短期访问 URL，并将客户重定向到 Webmail，而无需要求他们复制邮箱凭证。

提供商的行为可能有所不同：

* 具有原生 SSO API 的提供商可以创建直接认证的 Webmail 会话。
* 支持魔术链接的提供商可以生成有时限的登录链接。
* 没有 SSO 或魔术链接支持的提供商仍然会显示邮箱详情和密码管理操作，但不会显示一键 Webmail 按钮。

请仅为已配置所需 API 凭证的提供商启用 Webmail 访问。如果某个提供商缺少此操作，请首先确认该提供商是否已激活、已配置，并且支持 Webmail 登录链接。

## 结账时配置

附加功能可以在结账过程中创建客户邮件账户。当套餐默认包含邮箱，或者结账表单收集了邮件账户详情时，这非常有用。

推荐的部署流程：

1. 在启用结账时配置功能之前，先配置好提供商和套餐限制。
2. 使用暂存产品进行购买测试，并确认邮箱在结账后已创建。
3. 检查客户账户页面，确认新邮箱已出现在邮件小组件中。
4. 在为生产产品启用流程之前，确认密码更改和 Webmail 操作功能。

如果配置失败，客户账户和会员资格仍然可以存在，但邮箱需要管理员后续跟进。在重试之前，请检查 Ultimate Multisite 的活动日志和提供商日志。

## CyberPanel 邮件转发器

对于基于 CyberPanel 的网络，版本 1.1.0 添加了邮件转发器支持。当客户需要地址（例如 `info@customer-domain.test`）将邮件重定向到另一个收件箱，而无需创建完整的邮箱时，可以使用转发器。

在提供转发器之前，请确认 CyberPanel 集成已配置且可访问。有关所需的主机、用户名、密码和包常量，请参阅 [CyberPanel Integration](../../user-guide/administration/cyberpanel-integration) 指南。

## 常见问题解答

## 客户可以创建自己的邮件账户吗？

可以。当客户账户小组件启用且套餐允许邮件账户时，客户可以在账户页面创建和管理自己的邮箱。

## 每个提供商都支持 Webmail SSO 吗？

不一定。SSO 和魔术链接支持取决于活动的邮件提供商。没有访问链接 API 的提供商仍然可以支持邮箱创建和密码更改。

## 管理员可以为客户创建邮件账户吗？

可以。管理员可以在管理界面创建邮件账户，包括设置初始密码。

## 转发器和邮箱一样吗？

不一样。邮箱用于存储邮件并具有登录凭证。转发器用于将邮件重定向到另一个地址，不提供独立的收件箱。
