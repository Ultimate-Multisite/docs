# 语言选择器

Multisite Ultimate：注册时的语言选择器

在注册流程中添加语言选择字段。

## 安装

1. 将 'ultimate-multisite-language-selector' 上传到 '/wp-content/plugins/' 目录
2. 通过 WordPress 网络管理员中的 'Plugins' 菜单激活插件

## 使用指南

使用此免费插件，您可以让用户选择他们希望在注册流程和登录页面中看到的语言。

在[激活您的许可证](../../user-guide/getting-started/installing-ultimate-multisite.md)后，您将能够安装许可证允许访问的 Ultimate Multisite 插件。

转到您的 Ultimate Multisite 设置页面，然后在左侧菜单中点击 _Add-ons_ 标签。

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-rzVqJCVpY1.png)

_转到插件页面开始安装语言选择器_

在那里，您将找到我们提供的所有插件。

免费插件对所有客户开放，无论他们订阅了哪种计划。

点击插件缩略图下方的 _Install it Now!_ 按钮。然后在 Plugins 页面激活它。

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-bJgpO3QvLh.png)

_点击按钮安装插件_

_在您的 Plugins 页面激活它_

现在，转到 Ultimate Multisite 的 Settings 菜单并进入 _Network Settings_ 选项卡。在页面底部，您将找到一个名为 _Languages_ 的新部分。

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-CaH7BHDh50.png)

_出现一个全新的部分！_

您可以将其留空以显示所有语言作为选项，或选择您想要提供的语言。语言是您在 WordPress 设置中已安装的语言。

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-PGljAd4C24.png)

_此 WP 安装中可用的语言列表_

保存更改后，所选语言的下拉菜单将显示在登录页面和注册流程中。

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-qUnJL7JHxY.png)

_登录页面上的语言选择器_

_注册流程中的语言选择器_

## 通过 URL 设置语言

您还可以通过在 URL 中添加 locale 参数来更改页面语言。例如，如果您想重定向到意大利语注册页面，请在 URL 中添加 ?locale=it_IT。

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

此功能需要在您的 WordPress 网络中安装该语言。
