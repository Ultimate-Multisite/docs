# Material WP

WordPress 管理界面一直保持不变。但你有没有想过，Material Design 版本的管理界面会是什么样子？

Material WP 是一个管理员主题，旨在为你提供这种外观和感觉，帮助你为客户提供更个性化的仪表盘。

## 功能

- 自定义 CSS 支持
- 简单的样式设置
- 菜单重新排序
- 简单的导入/导出

## 文档

**使用 Material WP 自定义你的管理面板**

## 安装

1. 将 'material-wp' 上传到 '/wp-content/plugins/' 目录
2. 在 WordPress 的 '插件' 菜单中激活插件

## 使用指南

WordPress 管理界面一直保持不变。但你有没有想过，Material Design 版本的管理界面会是什么样子？

在**安装**后，前往你的 _插件_ 页面，在网络中点击激活。
