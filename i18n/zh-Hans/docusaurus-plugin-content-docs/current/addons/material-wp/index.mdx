# Material WP

WordPress 管理界面一直保持不变。但您是否曾想过，Material Design 版的管理界面会是什么样子？

Material WP 是一个管理主题，旨在为您提供这种外观和感觉，帮助您为客户提供更个性化的仪表盘。

## 功能

- 自定义 CSS 支持
- 轻松的样式设置
- 菜单重新排序
- 轻松导入/导出

## 文档

**使用 Material WP 自定义您的管理面板**

## 安装

1. 将 'material-wp' 上传到 '/wp-content/plugins/' 目录
2. 通过 WordPress 的 'Plugins' 菜单激活插件

## 使用指南

WordPress 管理界面一直保持不变。但您是否曾想过，Material Design 版的管理界面会是什么样子？
