# 计量计费方案

Ultimate Multisite: Metered Plans 插件通过引入灵活的计量计费功能，改变了您向客户计费的方式。与在超出硬性限制时阻止用户的传统做法不同，它允许产生超额费用，并自动向客户收取额外使用量。

## 关键功能

* **计量计费**：按实际使用量向客户收费，而非固定层级
* **超额管理**：自动计费超出计划限制的使用量
* **使用跟踪**：实时监控客户资源消耗
* **灵活定价**：为不同资源（存储、带宽、站点等）设置自定义费率
* **自动开票**：当使用阈值被超过时自动生成发票
* **使用报告**：为客户和管理员提供详细的使用分析和报告

## 适用场景

* **托管服务商**：提供可随客户需求增长的可扩展方案
* **SaaS 平台**：按实际资源消耗计费
* **代理网络**：为客户项目提供灵活的定价模式
* **转售托管**：让合作伙伴提供基于使用量的定价

## 工作原理

1. 配置计量资源（磁盘空间、带宽、站点数量等）
2. 设置基础计划限制和超额费率
3. 系统会自动跟踪整个网络的使用情况
4. 当客户超出限制时，系统会计算超额费用
5. 根据计费周期自动生成发票
6. 客户可以实时查看他们的使用量和即将产生的费用

## 支持的资源

* 磁盘存储
* 带宽/数据传输
* 站点数量
* 用户数量
* 自定义资源类型

## 系统要求

* Multisite Ultimate 2.0.0 或更高版本
* WordPress Multisite 安装
* PHP 7.4 或更高版本

## 安装

1. 将插件文件上传到 `/wp-content/plugins/ultimate-multisite-metered-plans`
2. 在 WordPress 的 '插件' 页面激活插件
3. 导航到 Multisite Ultimate → Settings → Metered Plans
4. 配置您的计量资源和定价
5. 在产品编辑器中为产品启用计量计费

## 常见问题

### 当客户超出限制时会发生什么？

系统不会阻止访问，而是跟踪超额使用，并根据您配置的费率将其计入下一张发票。

### 我可以为不同产品设置不同的超额费率吗？

可以，每个产品都可以拥有独特的计量资源和超额计费方案。

### 如何通知客户使用情况？

客户可以在仪表盘中查看当前使用量，并在接近或超出限制时接收可选通知。

### 我仍然可以使用硬性限制吗？

可以，计量计费是按产品可选的。您可以将传统硬性限制产品与计量产品混合使用。
