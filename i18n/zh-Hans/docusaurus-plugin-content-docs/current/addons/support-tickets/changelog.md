# Support Tickets 更新日志

### 1.0.4 - 2026-05-05 {#104---2026-05-05}
* 改进：从 Git 跟踪中移除了 vendor/ 目录（已由 .gitignore 覆盖），减小仓库大小
* 改进：测试兼容至 WordPress 7.0

### 1.0.3 - 2026-05-01 {#103---2026-05-01}
* 修复：在可为空的工单模型 setter 中接受 null 值
* 修复：将所有工单存储在全局网络级表中，以正确支持多站点
* 修复：在客户新建工单表单中隐藏员工字段，并修复空白页重定向
* 修复：将未定义的 add_meta() 调用替换为正确的 update_meta()，用于元数据存储
* 修复：将未注册的权限检查替换为正确的 wu_view_all_support_tickets
* 修复：为工单状态、分配和快速编辑操作添加缺失的 AJAX 处理程序
* 修复：更正工单视图中员工回复检测的方法名称
* 修复：合并重复的回复处理程序，并对齐 nonce 操作名称
* 修复：为 [wu_submit_ticket] shortcode 添加缺失的前端视图
* 修复：添加缺失的 user_id 列，并修复 Support_Ticket::get_user_id() 方法
* 修复：移除优先级筛选选项标签中的多余双引号
* 修复：为超级管理员添加网络管理工单管理面板
* 改进：将管理员 CSS 合并到单个外部样式表中
* 改进：从子站点管理员菜单中移除 Settings 子菜单
* 改进：仅在支持工单页面按条件加载前端资源
* 改进：当 Bedrock 根 autoloader 已加载依赖项时跳过 plugin autoloader

### 1.0.2 - 2025-12-11 {#102---2025-12-11}
* 新增：管理员和前端的完整工单查看功能
* 新增：用于工单回复提交的 AJAX 处理程序
* 新增：支持具备正确表单处理的工单回复功能
* 新增：在管理员中为工单提交和回复正确显示通知
* 新增：客户自行提交时自动将工单关联到当前用户
* 新增：安全增强，防止客户覆盖工单所有权
* 新增：缺失的辅助函数（wu_format_date、wu_user_can_view_ticket 等）
* 新增：正确的文件附件下载和处理
* 新增：用于工单回复和状态变更的电子邮件通知系统
* 修复：工单回复表单现在包含必要的工单 ID
* 修复：管理员工单视图中的通知可见性问题
* 修复：支持工单函数中的语法错误
* 改进：为回复和附件提供正确的查询与架构类，改进数据库结构
* 改进：迁移为工单模型使用真实属性而不是 attributes
