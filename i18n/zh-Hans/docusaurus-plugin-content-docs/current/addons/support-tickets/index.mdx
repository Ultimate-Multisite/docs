# 支持工单

**支持工单** 是 Ultimate Multisite 的强大插件，为您的 WordPress 网络添加了全面的支持工单系统。非常适合 SaaS 平台、WaaS（Website as a Service）提供商以及需要在网络层和单个站点层面提供客户支持的多站点网络。

## 为什么选择支持工单？

* **多级支持**：网络管理员处理全网工单，而站点所有者管理自己的客户支持
* **专业工作流**：完整的工单生命周期管理，包括优先级、状态和分配
* **客户体验**：美观、响应式界面，让客户轻松获得帮助
* **安全可靠**：采用 WordPress 安全最佳实践构建，数据库性能优化
* **无缝集成**：与 Ultimate Multisite 的客户和站点管理完美配合

## 关键功能

**完整工单管理**

* 唯一工单编号系统，便于引用
* 可配置的优先级（低、正常、高、关键）
* 全面的状态工作流（开放、进行中、等待、已解决、已关闭）
* 工单分类（通用、技术、计费、功能请求、错误报告）
* 将工单分配给支持代理，支持可选自动分配

**线程化对话**

* 完整的对话线程，用于工单回复
* 员工和客户回复的独特样式
* 私人员工备注，客户不可见
* 实时 AJAX 驱动的界面
* 现代文本编辑器，支持格式化选项

**文件附件**

* 安全文件上传，支持拖放
* 可配置的文件类型和大小限制
* 高级安全，恶意文件检测
* 受保护的下载，访问控制

**智能通知**

* 新工单、回复和状态更改的自动电子邮件提醒
* 可自定义的电子邮件模板
* 基于角色的通知，适用于员工和客户
* 动态模板变量，用于个性化消息

**强大的管理员界面**

* 高级工单列表，支持排序和过滤
* 快速编辑状态、优先级和分配
* 批量操作，提升管理效率
* 统计仪表盘，显示性能指标
* 综合设置面板

**美观的客户门户**

* 清晰、响应式界面，适用于所有设备
* 移动优先设计
* 高级搜索和过滤
* 现代编辑器，带格式化工具栏
* 易于使用的工单提交表单

**灵活集成**

* 使用短代码或 Ultimate Multisite UI 元素
* 在站点任何位置显示工单表单和列表
* 与多站点网络结构无缝配合
* 可通过挂钩和过滤器扩展

## 适用场景

* 需要客户支持的 SaaS 平台
* 管理多个客户站点的 WaaS 提供商
* 具有分层支持的会员网站
* 管理多个品牌的网络管理员
* 提供客户服务的站点所有者

## 安装

1. 确保 Ultimate Multisite 插件已安装并在您的 WordPress 网络中激活
2. 将插件文件上传到您的 `/wp-content/plugins/` 目录
3. 通过 WordPress 的 'Plugins' 菜单在网络层激活插件
4. 导航至 Network Admin > Support Tickets 配置设置

## 常见问题

## 这需要 Ultimate Multisite 吗？

是的，这个插件需要安装并激活 Ultimate Multisite。它与 Ultimate Multisite 的客户和站点管理功能深度集成。

## 站点所有者可以自行处理客户支持吗？

当然可以！站点所有者可以使用工单系统为自己的客户提供支持，而网络管理员可以处理全网工单。

## 客户如何提交工单？

您可以使用短代码或 Ultimate Multisite UI 元素在站点任何位置添加工单提交表单。客户只需填写表单并提交请求。

## 我可以自定义工单状态和优先级吗？

可以！您可以在网络管理员设置中完全自定义优先级、状态和工单类型，以匹配您的支持工作流程。

## 它支持文件附件吗？

是的，客户和员工都可以将文件附加到工单和回复中。您可以完全控制允许的文件类型和大小限制，以确保安全。

## 通知如何处理？

系统会自动发送新工单、回复和状态更改的电子邮件通知。您可以在设置中自定义所有电子邮件模板。

## 我可以将工单分配给特定支持代理吗？

可以，您可以手动将工单分配给代理，或启用自动分配以自动分发工单。

## 是否有供员工使用的私人备注？

是的，员工可以添加仅对其他员工可见的私人回复，而客户不可见。

## 它支持移动端吗？

当然！整个界面采用移动优先设计，在所有设备上都能良好运行。

## 我可以使用短代码显示工单列表和表单吗？

可以！可用的短代码：

* `[wu_support_tickets]` - 显示客户工单列表
* `[wu_support_ticket_form]` - 显示新工单提交表单
* `[wu_support_ticket_view ticket_id="123"]` - 显示指定工单

## 可以上传哪些文件类型？

您可以在设置中配置允许的文件类型。系统包含高级安全扫描，以防止恶意文件。

## 我可以导出工单数据吗？

可以，插件提供工单数据和统计信息的导出功能。
