# Mailster

将您的多站点网络转变为强大的电子邮件列表构建机器，使用 Ultimate Multisite Mailster Integration 插件。新客户在注册或购买时会自动添加到 Mailster 邮件列表中，创建您的会员平台与电子邮件营销策略之间的无缝连接。

## 关键功能

**自动订阅者同步**
* 在用户注册时自动将客户添加到 Mailster 列表
* 选择在注册时立即同步，或等待付款确认后再同步
* 在无需手动干预的情况下保持您的邮件列表新鲜且最新

**灵活的列表管理**
* 为所有新客户设置全局默认列表
* 用产品特定列表覆盖默认值，以进行针对性活动
* 使用 Mailster 熟悉的界面轻松管理列表分配

**符合 GDPR 的选择加入选项**
* 在自动订阅或基于复选框的同意之间进行选择
* 为已确认的订阅启用双重选择加入
* 根据您的合规要求显示/隐藏选择加入复选框

**智能数据映射**
* 自动同步客户姓名和联系信息
* 将账单地址映射到订阅者字段
* 跟踪注册日期以进行针对性活动

**产品级控制**
* 为不同产品分配不同的列表
* 根据客户购买的产品创建针对性活动
* 在需要时为特定产品禁用 Mailster

**专业体验**
* 原生 Mailster UI 集成用于列表选择
* 详细日志记录用于故障排除和验证
* 无缝更新现有订阅者

## 适用人群

* 正在构建邮件列表的会员网站所有者
* 希望培养新客户的 SaaS 平台
* 按注册情况对学生进行细分的课程创作者
* 任何使用 Ultimate Multisite 与 Mailster 的人

## 工作原理

1. 在您的 Ultimate Multisite 网络上安装并激活插件
2. 在 Ultimate Multisite 设置中配置全局默认列表
3. 可选地使用产品特定列表覆盖
4. 选择您的选择加入模式和订阅时间
5. 观看您的 Mailster 列表自动增长！

## 安装

1. 将插件文件上传到您的 Ultimate Multisite addons 目录
2. 确保 Mailster 已在您的主站点上安装并激活
3. 通过 Ultimate Multisite addons 页面激活插件
4. 导航至 Settings > Mailster Integration 进行配置
5. 选择您的默认列表并配置选择加入偏好

## 常见问题

## 这个插件在没有 Ultimate Multisite 的情况下能工作吗？

不，这个插件需要 Ultimate Multisite 才能运行。它专门设计用于将 Mailster 与 Ultimate Multisite 的客户和产品系统集成。

## 我需要在所有站点上安装 Mailster 吗？

不，Mailster 只需在您的主网络站点上安装并激活。插件会自动从所有子站点连接到它。

## 我可以为不同产品分配不同的列表吗？

可以！您可以设置全局默认列表，然后按产品覆盖它们。这使您能够根据客户购买的产品创建高度针对性的电子邮件活动。

## 如果客户已存在于 Mailster 中会发生什么？

默认情况下，插件会更新现有订阅者并将其添加到新列表中。如果您想跳过现有订阅者，可以在设置中禁用此功能。

## 这符合 GDPR 吗？

是的！插件包含基于复选框的选择加入模式，客户必须明确同意接收电子邮件。这与适当的隐私政策相结合，满足 GDPR 要求。

## 我可以使用双重选择加入吗？

当然可以！启用双重选择加入设置后，Mailster 将在激活订阅前发送确认电子邮件。
