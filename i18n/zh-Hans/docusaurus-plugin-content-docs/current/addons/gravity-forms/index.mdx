## 🚦 工作原理 {#-how-it-works}

### 1. **设计您的表单** {#1-design-your-form}
使用 Gravity Forms 的拖放式构建器创建完美的注册体验。

### 2. **映射您的字段** {#2-map-your-fields}
使用我们的可视化映射界面，将表单字段连接到 Multisite Ultimate 数据点。

### 3. **配置支付** {#3-configure-payment}
在结账重定向或 WooCommerce 集成之间选择，以进行支付处理。

### 4. **上线** {#4-go-live}
发布您的表单，开始将访客转化为付费客户！
