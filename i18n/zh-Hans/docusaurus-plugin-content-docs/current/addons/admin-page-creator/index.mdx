# Admin Page Creator

## 内容
Admin Page Creator 将让你轻松在 WordPress 管理仪表盘中添加新的管理页面和子页面。它可用于创建欢迎页面或教程页面，作为新用户的登陆页面。你可以使用 Beaver Builder、Elementor、Brizy 和 Oxygen 等页面构建器，无需费力即可设计页面。  
它还提供自定义 HTML、CSS 和 JavaScript 支持，以及基于套餐和用户角色的访问控制！

## 功能
* 支持自定义 HTML 和 CSS  
* 支持 JavaScript  
* PHP 内容模式  
* 基于用户角色的权限  
* 支持页面构建器  

## 文档
[使用 Multisite Ultimate 创建自定义管理页面和子页面：Admin Page Creator](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## 安装

1. 将 **ultimate-multisite-admin-page-creator** 上传到 **/wp-content/plugins/** 目录  
2. 通过 WordPress 网络管理中的 **插件** 菜单激活插件  

## 使用指南

虽然 WordPress 让在站点前端添加内容变得非常容易，但在后台创建页面和子页面需要一些基本的编码技能。Ultimate Multisite: Admin Page Creator 旨在将前端的同样易用性带到你网络站点的后台。

## 为什么使用 Admin Page Creator？

也许你想在客户的管理面板上提供一个带有教程的自定义页面，也许你想添加一个带 FAQ 和联系小部件的自定义页面。使用 Admin Page Creator 的可能性是无穷无尽的，这正是我们今天教程将要探讨的内容。

## 创建你的第一个自定义管理页面

安装并激活 Ultimate Multisite: Admin Page Creator 后，前往你的网络管理菜单，进入 Admin Pages。

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212627fb34b55df443e4a5-Aml1wdfXt-69C790DA-7E3F-4DDD-A381-7F6633D91B2A.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212627fb34b55df443e4a5-Aml1wdfXt-69C790DA-7E3F-4DDD-A381-7F6633D91B2A.png)

_网络管理中的自定义管理页面列表_

你将看到你目前创建的所有管理页面的列表（由于这是你第一次访问，列表将为空）。继续点击 **Add new Admin Page**。你将被重定向到 Admin Page 编辑器。

在 Admin Page 编辑器中，你将拥有多种不同的选项来定制你的页面。

### 页面标题

_输入页面标题_

### 内容类型选择器

Ultimate Multisite: Admin Page Creator 在页面内容方面为你提供两种选择：你可以使用默认的 WordPress WYSIWYG 编辑器，或者切换到 HTML 编辑器。  
如果你不熟悉 HTML，保持默认的 WordPress 编辑器。你可以像编辑普通文章或页面一样编辑页面内容。

_使用默认的 WordPress WYSIWYG 编辑器_  
_使用 HTML 编辑器选项，带有语法高亮和错误提示_

### 菜单选项

在侧边的 General Options 元框中，你可以配置菜单类型、标签等更多设置。

### 菜单类型

你新创建的自定义管理页面可以有两种不同的菜单类型：顶级菜单和子菜单。  
顶级菜单页面，顾名思义，会将你新创建的页面添加到管理菜单中，与默认的 WordPress 页面并列。  
如果你选择顶级选项，你还可以选择菜单顺序和菜单图标。  
如果你想将此页面设为子菜单页面，你可以选择一个父页面来附加此页面。你可以将自定义页面添加到默认的 WordPress 页面或你的顶级自定义管理页面中。

_完全自定义此自定义页面的菜单项_  
_你也可以将此页面添加为其他菜单页面的子菜单，包括你自己的自定义顶级菜单。_

## 高级选项

你应该能够为自定义管理页面添加各种内容，包括 CSS 规则和文件、JavaScript，以及外部库。你可以使用高级选项标签来完成此操作。

### 自定义 CSS

_也支持自定义 CSS。你还可以包含外部样式文件！_

### 自定义 JavaScript

_添加你自己的 JavaScript 代码！_

## 权限

在高级选项中，你还可以选择哪些子站点将访问此页面。如果你计划为不同的套餐层级提供不同的专属内容，这将非常有用。

_你可以完全控制此页面的显示位置_

**重要**：必须满足两个条件才能显示页面。这意味着如果用户是某个套餐的客户，但没有所选角色之一（或相反），页面将不会出现在他们的面板上。

## 激活

如果你想禁用自定义管理页面而不删除它或更改权限，只需取消选中 Active 选项。  
停用顶级页面将自动停用所有附加到该父页面的子页面。
