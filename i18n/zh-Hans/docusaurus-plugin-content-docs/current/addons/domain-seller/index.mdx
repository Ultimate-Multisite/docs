# Domain Seller

**Multisite Ultimate: Domain Seller** 将您的 WordPress Multisite 网络转变为强大的域名销售平台。此插件与 Multisite Ultimate v2 无缝集成，提供：

### 关键功能

#### 🌐 多域名提供商支持
* **OpenSRS Integration** - 完整支持 OpenSRS 域名注册和管理
* **Extensible Architecture** - 框架用于添加 Namecheap、GoDaddy 以及其他提供商
* **Provider Testing** - 内置连接测试和配置验证
* **Failover Support** - 在提供商之间自动故障转移以提高可靠性

#### 💰 高级定价与产品
* **Domain Products** - 原生 Multisite Ultimate v2 产品类型用于域名
* **Flexible Markup** - 固定金额或百分比加价，基于批发价
* **TLD-Specific Pricing** - 针对每个顶级域名（.com、.net 等）采用不同的定价策略
* **Real-Time Pricing** - 来自提供商的实时价格，配合智能缓存
* **Coupon Integration** - 完整支持 Multisite Ultimate 优惠券和折扣

#### 🛒 无缝结账体验
* **Integrated Domain Search** - 在结账过程中实时查询域名可用性
* **AJAX-Powered Interface** - 快速、响应式域名搜索，无需页面刷新
* **Smart Suggestions** - 当首选域名不可用时提供替代域名建议
* **Transparent Pricing** - 清晰展示价格及加价明细
* **One-Click Selection** - 简单的域名选择和结账流程

#### 🔄 智能自动续费系统
* **Membership Integration** - 域名与客户会员状态绑定
* **Automatic Scheduling** - 基于到期日期的智能续费调度
* **Payment Processing** - 通过现有客户网关进行自动续费支付
* **Retry Logic** - 针对续费失败的智能重试系统
* **Notification System** - 续费、失败和到期的电子邮件提醒

#### 👨‍💼 全面管理工具
* **Domain Management Dashboard** - 从单一界面监控所有客户域名
* **Provider Configuration** - 轻松设置和测试域名提供商
* **Manual Operations** - 手动域名注册、续费和管理
* **Detailed Logging** - 完整的域名操作审计跟踪
* **Bulk Operations** - 一次处理多个域名

#### 👤 增强客户体验
* **Customer Dashboard** - 自助域名管理界面
* **Renewal Management** - 客户可查看和管理自动续费
* **Domain Status** - 实时状态更新和到期信息
* **Transfer Tools** - 域名转移和 DNS 管理功能

### 支持的提供商

* **OpenSRS** - 完整集成注册、续费和管理
* **Extensible Architecture** - 轻松添加其他提供商

## 安装

1. 确保已安装并激活 **Multisite Ultimate v2.0.0+**  
2. 将插件文件上传至 `/wp-content/plugins/ultimate-multisite-domain-seller/`  
3. 通过 WordPress 网络管理后台的 '插件' 菜单激活插件  
4. 导航至 **Ultimate Multisite > Settings > Domain Seller** 配置提供商  
5. 在 **Ultimate Multisite > Products** 创建域名产品

## 常见问题

## 需要哪个版本的 Multisite Ultimate？

此插件需要 Multisite Ultimate 版本 2.0.0 或更高。它不兼容 Ultimate Multisite v1.x。

## 支持哪些域名提供商？

目前仅完整支持 OpenSRS。插件已设计为易于支持其他提供商——未来版本将添加更多。

## 我可以为不同的顶级域名设置不同的定价吗？

可以！您可以为不同的顶级域名组创建单独的域名产品，每个都有自己的定价加价和配置。

## 它支持自动续费吗？

可以，域名可以设置为自动续费，并与客户会员集成，实现无缝计费。

## 客户可以管理他们的域名吗？

可以，客户在其面板中获得域名管理界面，可查看注册、续费日期和基本管理功能。
