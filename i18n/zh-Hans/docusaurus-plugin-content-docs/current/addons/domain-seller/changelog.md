# 域名卖家更新日志

版本: 2.0.1 - 发布于 2025-09-28

* 将前缀重命名为 ultimate-multisite；更新文本域；版本提升。

### 版本 2.0.0 - 2024-XX-XX

**Multisite Ultimate v2 的重大重写**

* **新增：** 完全重写，采用现代 PHP 7.4+ 架构
* **新增：** 与 Multisite Ultimate v2 结账系统无缝集成
* **新增：** 域名产品管理，支持灵活定价选项
* **新增：** 多域名提供商支持架构
* **新增：** 自动续费和订阅集成
* **新增：** 客户域名管理界面
* **新增：** 管理员域名监控和日志
* **新增：** 域名产品的优惠券支持
* **新增：** 综合设置管理
* **新增：** 开发者友好的可扩展代码库
* **改进：** 更新 OpenSRS 提供商，完整功能支持
* **改进：** 现代 UI 与 Multisite Ultimate v2 一致
* **修复：** 所有已弃用的 v1 代码已更新为 v2 标准
* **移除：** 旧版 v1 兼容性（破坏性更改）
