# Ultimate Multisite 文档

欢迎来到 Ultimate Multisite 文档。Ultimate Multisite 将 WordPress Multisite 转变为完整的网站即服务（WaaS）平台。

## 文档章节

### [用户指南](/user-guide/getting-started/ultimate-multisite-101)

这里包含了您设置、配置和管理 WaaS 平台所需的一切。涵盖安装、支付网关、域名映射、客户管理等。

### [开发者文档](/developer/hooks)

为在 Ultimate Multisite 上构建的开发者提供技术参考。包括钩子参考、REST API 文档和插件开发指南。

### [插件](/addons)

付费插件文档，扩展 Ultimate Multisite，提供 WooCommerce 集成、分析、域名销售等附加功能。

## 获取帮助

- [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) -- 提问并分享想法

- [GitHub Issues](https://github.com/Ultimate-Multisite/ultimate-multisite/issues) -- 报告错误

- [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) -- 下载核心插件
