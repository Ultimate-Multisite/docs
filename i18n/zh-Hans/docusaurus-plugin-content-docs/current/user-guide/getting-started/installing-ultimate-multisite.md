# 安装 Ultimate Multisite

:::note
本教程假设您已安装并配置了 WordPress Multisite。若想了解如何操作，请查看 WP Beginner 的 [此教程](https://www.wpbeginner.com/glossary/multisite/)。
:::

## 安装插件

Ultimate Multisite 可在 [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) 免费获取。

在您的 **Network Admin Dashboard** 中，转到 **Plugins → Add New Plugin**。

![网络管理员添加新插件页面](/img/installation/add-new-plugin.png)

搜索 **"Ultimate Multisite"**（使用引号进行精确匹配），它将作为第一个结果出现。点击 **Install Now**。

![搜索结果显示 Ultimate Multisite](/img/installation/search-ultimate-multisite.png)

安装完成后，点击 **Network Activate**，在整个网络中激活插件。

![插件已安装，显示 Network Activate 按钮](/img/installation/plugin-installed.png)

激活后，您将自动重定向到设置向导。

![插件已激活并重定向到向导](/img/installation/plugin-activated.png)

## 设置向导

设置向导将在大约 10 分钟内引导您完成 Ultimate Multisite 的配置。

### 欢迎

点击 **Get Started** 开始。

![设置向导欢迎屏幕](/img/installation/wizard-welcome.png)

### 预安装检查

此步骤检查您的系统信息和 WordPress 安装，以确保满足 Ultimate Multisite 的要求。如果一切正常，请点击 **Go to the Next Step**。

![预安装检查显示系统要求](/img/installation/wizard-pre-install-checks.png)

### 安装

安装程序将创建必要的数据库表并安装 Ultimate Multisite 所需的 `sunrise.php` 文件。点击 **Install** 继续。

![安装步骤显示数据库表和 sunrise.php](/img/installation/wizard-installation.png)

### 您的公司

填写您的公司信息并设置默认货币。此信息将在整个 WaaS 平台中使用。完成后点击 **Continue**。

![您的公司配置步骤](/img/installation/wizard-your-company.png)

### 默认内容

此步骤允许您安装预定义的模板、产品和其他入门内容。这是熟悉 Ultimate Multisite 功能的好方法。点击 **Install** 添加默认内容，或跳过此步骤以从零开始。

![默认内容安装步骤](/img/installation/wizard-default-content.png)

### 推荐插件

可选择安装推荐的配套插件。点击 **Install** 添加，或跳过继续。

![推荐插件步骤](/img/installation/wizard-recommended-plugins.png)

### 准备就绪！

完成！您的 Ultimate Multisite 安装已完成。您现在可以从 **Network Admin Dashboard** 开始构建您的 Website as a Service 平台。

![设置完成 - 准备就绪屏幕](/img/installation/wizard-ready.png)

![网络管理员仪表盘，Ultimate Multisite 已激活](/img/installation/network-dashboard.png)

继续，玩得开心吧！
