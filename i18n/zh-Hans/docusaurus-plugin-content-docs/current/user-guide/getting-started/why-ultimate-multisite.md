# 为什么选择 Ultimate Multisite

如果你来到这篇文章，可能已经听说我们的插件可能是实现你商业想法的解决方案。我们希望在本文结束时，你能得到所需的答案。

## 什么是 Ultimate Multisite？

Ultimate Multisite 是一款 WordPress Multisite 插件，允许你为客户提供 WaaS（网站即服务）。使用我们的插件，你可以设置不同的订阅层级，让客户支付定期费用，在你的 Multisite 网络中托管网站。

Ultimate Multisite 让你可以通过不同的支付网关（如 WooCommerce 或 Stripe）接收付款，创建站点模板，使当新用户在你的网络上注册时更轻松，映射域名，并个性化你的注册表单，为客户打造定制化体验。

这仅仅是开始，因为你可以通过插件和 Webhook 扩展其功能。

## 其他功能

Ultimate Multisite 还提供：

* 与托管 WordPress 主机提供商的域名映射集成：当映射新域名时，Ultimate Multisite 会自动将映射的域名添加到 WP Engine、Cloudways、cPanel、RunCloud、Closte 等的管理面板中，省去你手动操作的麻烦。在某些情况下，我们甚至会为你触发 SSL 证书获取。

* Webhook：Ultimate Multisite 拥有强大的 Webhook 系统，允许你将网络与多种不同服务集成。我们甚至提供与 Zapier 的直接集成，使你的网络可以接入其平台上 1000+ 的应用和服务。

* 强大的管理页面，帮助你的客户使用你喜爱的页面构建器：Ultimate Multisite 有一个实用插件叫 Admin Pages Creator，基本上允许你使用 Elementor、Beaver Builder 和 Brizy 模板向客户的仪表盘添加完全自定义的管理页面。

* 两种不同的管理主题，定制客户仪表盘体验：Material WP 和 PRO。

以及更高级的功能：

* 支持模板覆盖：使用 Ultimate Multisite，你可以用自己的版本替换注册流程的默认 HTML 模板，而无需担心更改核心插件文件。

* 搜索和替换 API：你还可以在站点模板中使用 {{site_name}} 等占位符，并使用搜索和替换 API 在客户注册后自动填充这些占位符。

## 使用场景

现在让我们看看 Ultimate Multisite 的一些可能使用场景。

### 案例 1：代理机构

通常，代理机构的核心技能在于网站设计，而托管或营销等方面则列为附加服务。

对于代理机构，Ultimate Multisite 在单一平台上托管和管理多个网站的能力上提供了极大的价值。更进一步，标准化使用 GeneratePress、Astra、OceanWP 或其他主题的代理机构，可以利用 Ultimate Multisite 的功能为每个新站点自动激活这些主题。

通过 Ultimate Multisite 的设置，按产品维度对插件和主题进行严格控制，允许在为新站点实例化时将插件和主题设为可用或隐藏，并控制其激活状态。

代理机构将因 Ultimate Multisite 让他们专注于擅长的工作——设计卓越的网站而获得安心。

### 案例 2：细分领域提供商

有句老话说：“专注做一件事并做好”。对于许多专业人士来说，这意味着围绕单一核心理念创建产品或服务。

如果你专注于开发细分领域的网站网络，你可能想基于通用框架和平台提供服务。这时站点模板就能派上用场！

Ultimate Multisite 的一项创新功能是模板站点，即已安装并激活主题、必要插件以及创建了示例文章或页面。当客户基于模板创建新站点时，模板的内容和设置会被复制到新站点。

对于细分领域站点和服务的提供商而言，这提供了无与伦比的优势，能够即时创建已准备就绪的站点，配备自定义插件和设计。客户只需提供最少的输入即可完成服务。

### 案例 3：WordPress 网站托管

托管 WordPress 网站有无数种方式，但很少像为客户提供预装 WordPress 的网页空间那样简单。这是因为需要将许多决策和考虑因素结合起来，才能提供有意义的服务。

Ultimate Multisite 在此领域表现出色，提供了全面的一站式 WordPress 托管解决方案。该方案包含提供订阅服务、收款、结账表单、折扣券和客户沟通的核心机制。

Ultimate Multisite 在很大程度上简化了正确安装、配置和维护 WordPress Multisite 所需的核心工作，使网络管理员只需关注与其服务或细分领域相关的方面，例如产品层级、定价和服务优惠。

对于希望与 Ultimate Multisite 集成的开发者，该方案还提供了全面的 RESTful API 和 Webhook，用于事件通知。

不依赖大量外部插件和许可证，Ultimate Multisite 提供了功能丰富且可与 Wix、Squarespace、[WordPress.com](https://WordPress.com) 等相媲美的解决方案。
