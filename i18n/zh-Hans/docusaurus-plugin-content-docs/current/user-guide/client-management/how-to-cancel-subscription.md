# 如何取消我的订阅？

如果您拥有 Ultimate Multisite 许可证，您可以随时取消其续订。只需按照以下步骤操作：

<!-- Screenshot unavailable: Account page email with access link -->

使用您在购买时提供的电子邮件和密码。

<!-- Screenshot unavailable: Login page with email and password fields -->

这是您的账户页面初始屏幕：

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

要取消订阅，请在左侧菜单中转到 _Renewing & Billing_。

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

点击右侧的箭头打开侧边窗口。然后，您应该选择将取消 **auto-renew** 的选项。

系统会向您显示确认消息。

如果您确定要取消订阅，只需点击 _**Cancel Renewals**_ 按钮。执行此操作后，系统会要求您完成一项简短调查。

完成！您的订阅将不再自动续订。

**您将在订阅到期前拥有有效的密钥**。如果您想重新激活订阅，需要手动操作。
