# 您客户的网站（v2）

注册到您的网络后，您的客户可以访问他们的仪表盘。

除了 **Account** 页面，客户可以在该页面查看有关其付款、会员资格、域名、计划限制等重要信息，他们还可以访问 **My Sites** 页面，在那里查看并管理所有网站。只需在仪表盘上点击 My Sites 即可。

![客户仪表盘中的 My Sites 链接](/img/admin/customers-list.png)

他们将看到所有网站，并在每个网站下方显示一个 **Manage** 按钮。点击后，他们将被重定向到该网站的 **dashboard of this specific website**。

他们还可以查看哪些网站是 **primary**（主要）网站，适用于允许拥有多个网站的计划。

![My Sites 页面显示客户网站及 Manage 按钮](/img/admin/customers-list.png)

如果他们想更改主要网站，需要通过点击 **Account** 菜单进入 Account 页面。

![客户仪表盘中的 Account 菜单链接](/img/admin/customers-list.png)

然后，点击 **Change Default Site**。

![账户页面上的 Change Default Site 按钮](/img/admin/customers-list.png)

弹出窗口出现后，点击下拉列表并选择您想设为主要网站的网站。

![Change Default Site 下拉列表选择](/img/admin/customers-list.png)

最后，只需点击 **Change Default Site**，完成即可！

![确认默认站点更改](/img/admin/customers-list.png)
