# 你客户的网站 (v2)

在你的网络上注册后，你的客户可以访问他们的 Dashboard。

除了 **Account** 页面（他们可以在其中查看有关付款、会员、域名、计划限制等的重要信息）之外，他们还可以访问 **My Sites** 页面，在那里他们可以查看并管理自己的所有网站。他们只需在自己的 Dashboard 上点击 My Sites。

![客户 Dashboard 中的 My Sites 链接](/img/account-page/my-sites-link.png)

系统会显示他们的所有网站，每个网站正下方都有一个 **Manage** 按钮。如果他们点击它，就会被重定向到**该特定网站的 Dashboard**。

对于允许拥有多个网站的计划，他们还可以看到自己哪个网站是**主**网站。

![My Sites 页面显示带有 Manage 按钮的客户网站](/img/account-page/my-sites-page.png)

下面是网站编辑界面的完整视图：

![网站编辑完整界面](/img/admin/site-edit-full.png)

如果他们想更改自己的主网站，需要点击 **Account** 菜单进入 Account 页面。

![客户 Dashboard 中的 Account 菜单链接](/img/account-page/account-menu.png)

你也可以查看并编辑客户详情：

![客户编辑界面](/img/admin/customer-edit.png)

然后，他们应点击 **Change Default Site**。

![Account 页面上的 Change Default Site 按钮](/img/account-page/change-default-site.png)

会出现一个弹出窗口。点击下拉列表，然后选择你想设为主网站的网站。

![Change Default Site 下拉选择](/img/account-page/change-default-site-dropdown.png)

然后，只需点击 **Change Default Site**，就完成了！

![确认默认网站更改](/img/account-page/change-default-site-confirm.png)
