# 浏览管理面板

Ultimate Multisite 旨在尽可能简单易懂，但凭借如此强大的工具，提供引导式演练会有所帮助。让我们一起浏览管理页面，帮助你快速上手。

## 仪表盘

Ultimate Multisite **仪表盘** 显示基本报告和分析，包括收入、站点活动、会员增长、访客数量和地理数据。

![Ultimate Multisite 仪表盘](/img/admin/um-dashboard-top.png)

## 结账表单

**结账表单** 让你可以灵活创建自定义注册页面。你可以自定义字段、插入自定义类和脚本，并根据访客所在国家或位置限制访问。

![结账表单列表](/img/admin/checkout-forms-list.png)

## 产品

**产品** 部分是你为网络创建不同计划、套餐和服务的地方。可在产品层面定义价格、计费频率以及限制和配额。

![产品列表](/img/admin/products-list.png)

## 会员

**会员** 页面显示网络中所有订阅。可查看并编辑订阅详情，包括计划、产品、计费金额和频率、支付历史以及时间戳。

![会员列表](/img/admin/memberships-list.png)

## 付款

**付款** 页面快速查看整个网络的支付历史，包含详细交易信息，包括具体产品和金额。

![付款列表](/img/admin/payments-list.png)

## 客户

**客户** 页面显示网络中所有已注册成员的基本用户信息，包括姓名、电子邮件地址、上次登录时间，以及一个 "切换到" 链接，用于登录他们的子站点。

![客户列表](/img/admin/customers-list.png)

## 站点

**站点** 页面列出网络中的所有子站点，包括站点模板和客户拥有的站点。可轻松管理映射域名、限制与配额、插件与主题以及站点缩略图。

![站点列表](/img/admin/sites-list.png)

## 域名

**域名** 页面专门用于映射到子站点的自定义域名。作为超级管理员，你可以手动添加或映射自定义域名，并查看 DNS 记录和日志。

![域名列表](/img/admin/domains-list.png)

## 折扣码

**折扣码** 让你创建优惠券代码，为最终用户提供折扣。指定折扣金额并针对特定用户或群组。

![折扣码列表](/img/admin/discount-codes-list.png)

## 广播

**广播** 是用于发布公告或向子站点用户发送私信的工具。可根据用户订阅的计划或产品定位特定用户群体。

![广播列表](/img/admin/broadcasts-list.png)

## 设置

**设置** 页面用于配置 Ultimate Multisite — 注册设置、支付、API 与 Webhook、域名映射以及其他集成。

![设置页面](/img/admin/settings-general.png)

## 事件

**事件** 页面记录网络中所有事件和日志。它跟踪计划变更、注册以及其他操作 — 对监控多站点非常有用。

![事件列表](/img/admin/events-list.png)

## Webhook

**Webhook** 允许你将数据发送到其他应用程序。可用于将 Ultimate Multisite 的数据发送到 Zapier 等平台。

![Webhook 列表](/img/admin/webhooks-list.png)
