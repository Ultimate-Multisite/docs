# 管理系统邮件

Ultimate Multisite 会为注册确认、付款收据、会员变更等事件发送各种系统邮件。您可以自定义这些邮件模板以匹配您的品牌。

## 查看系统邮件

导航至 **Ultimate Multisite → Settings** 并点击 **Emails** 选项卡查看与邮件相关的设置。

![Settings - Emails tab](/img/admin/settings-emails.png)

在那里，您可以访问 **System Emails** 页面查看所有邮件模板。

![System Emails list](/img/admin/system-emails-list.png)

## 导入新邮件模板

新版本的 Ultimate Multisite 和插件可能会注册新的邮件模板。为防止冲突，新模板不会自动添加——您需要手动导入。

导入新邮件模板的步骤：

1. 前往 **System Emails** 页面
2. 点击页面顶部的 **Reset & Import** 按钮
3. 切换 **Import Emails** 选项以查看可用模板
4. 选择要导入的模板并确认

## 重置邮件模板

如果您已自定义邮件模板并想恢复默认内容，有两种选择：

1. **Delete and re-import** — 删除系统邮件并重新导入（这会清除发送指标）
2. **Reset via the tool** — 使用 **Reset & Import** 工具，切换 **Reset** 选项，然后选择要重置的邮件

优先使用 Reset 选项，因为它在恢复默认内容的同时保留发送指标和其他元数据。
