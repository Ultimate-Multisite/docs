# 设置参考

本页面记录了影响 Ultimate Multisite 日常管理和近期行为变更的设置。

## 其他选项

**其他选项**区域位于 **Ultimate Multisite > 设置 > 登录与注册** 下。

| 设置 | 描述 |
|---|---|
| **启用 Jumper** | 在管理区域显示 Jumper 快速导航工具。您可以使用它直接跳转到 Ultimate Multisite 屏幕、网络对象和支持的管理目的地。如果您不希望看到此快捷方式，请禁用此选项。 |

## 错误报告和遥测

之前用于选择加入的错误报告设置已从设置页面移除。匿名遥测功能已禁用，并且不再有用户界面开关来启用它。

如果您维护内部运行手册或设置页面截图，请移除对旧错误报告选择加入字段的引用，以免管理员寻找一个已不存在的设置。

## 导入/导出设置

**导入/导出**设置选项卡描述了它控制哪些设置，并直接链接到 **Ultimate Multisite > 站点导出**，用于站点和网络存档。请使用此设置选项卡进行导入/导出配置；使用 **工具 > 导出与导入** 进行单站点导出/导入工作流程；当您需要完整的网络导出存档时，请使用站点导出工具。
