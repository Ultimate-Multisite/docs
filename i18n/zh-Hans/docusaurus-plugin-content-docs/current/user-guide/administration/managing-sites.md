# 管理站点

站点（或子站点）是您的 WaaS 业务核心。Ultimate Multisite 有三种类型的站点：

- **客户拥有** — 属于特定客户的站点
- **站点模板** — 预定义站点，客户可将其作为起点
- **主站点** — 您的主网络站点

## 查看站点

导航至 **Ultimate Multisite → 站点** 查看网络中的所有子站点。每个站点按类型标记，您可以按 **所有站点**、**客户拥有**、**模板** 或 **待处理** 进行筛选。

![站点列表页面](/img/admin/sites-list.png)

## 添加新站点

点击 **添加站点** 按钮创建新站点。您需要提供：

- **站点标题** — 新站点的名称
- **站点 URL** — 站点的域名/路径
- **站点类型** — 是否为客户站点、模板或普通站点

**复制站点** 选项允许您基于现有站点模板创建新站点。启用后，您可以选择用作起点的模板。确保 **复制媒体在复制时** 已启用，以包含媒体文件。

## 管理现有站点

点击任何站点上的 **管理** 打开 **编辑站点** 页面。在此您将找到：

### 基本信息

站点名称、类型、站点 ID 和描述。您还将看到映射域名、关联会员以及拥有该站点的客户账户。

### 站点选项

配置站点的功能和限制：

- **访问限制** — 站点访问的最大次数
- **用户账户限制** — 每个用户角色的限制
- **磁盘空间** — 站点存储配额
- **自定义域名** — 为此站点启用域名映射
- **插件和主题可见性** — 控制哪些插件和主题可见、隐藏或预激活

默认情况下，站点遵循会员级别设置的限制。在站点级别设置限制将覆盖会员设置。

### 关联

在站点选项下方，您将找到有关以下信息：

- 与站点关联的映射域名
- 站点所属会员
- 与站点关联的客户账户

### 右侧边栏

在右侧，您可以：

- 使用切换按钮启用/禁用站点
- 更改站点类型或重新分配所有权
- 设置站点图片/缩略图（在前端显示给站点模板）
- 永久删除站点

:::warning
删除站点是不可逆的。站点及其所有内容将被永久删除。
:::
