# 管理计划的可共享链接

Ultimate Multisite 为您提供直接与客户共享计划链接的选项。为此，请转到任何计划的编辑页面并点击 _点击复制可共享链接_ 按钮。

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-2DJoMvapXh.png)

_在计划编辑页面，您可以找到创建可共享链接的按钮_

请注意，此机制使用默认计费频率来构建链接。这意味着，如果您的默认计费是每月，例如，最终您将得到类似这样的链接： [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

如果您希望可共享链接默认按季度计费，您需要在末尾添加 3

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

要在年度计划上选择 _premium_ 计划，您需要添加 1，然后是单词：

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

请记住，您不能为不同的计费频率使用相同的链接。您应该为每个计费频率分享一个特定的链接。

一旦您将链接分享给客户，它将把他们重定向到注册流程的第一步。
