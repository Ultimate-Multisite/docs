# 站点模板中的自动搜索和替换 (v2)

_**本教程需要 WP UItimo 2.x 版本。**_

Ultimate Multisite 最强大的功能之一是能够在注册表单中添加任意文本、颜色和选择字段。捕获到这些数据后，我们可以用它来预填充所选站点模板的某些部分。然后，当新站点发布时，Ultimate Multisite 将把占位符替换为注册时输入的实际信息。

例如，你可以在模板站点中使用占位符。占位符应被双大括号包围 - {{placeholder_name}}。

然后，你只需添加一个对应的注册字段来捕获该数据。

客户随后可以在注册时填写该字段。

Ultimate Multisite 将自动用客户提供的数据替换占位符。

## **解决“占位符满满的模板”问题**

这一切都很好，但我们确实遇到了一个丑陋的问题：现在我们的站点模板——客户可以访问的模板——充满了无聊的占位符，无法提供任何信息。

为了解决这个问题，我们提供了为占位符设置假值的选项，并在客户访问时使用这些值在模板站点上搜索并替换其内容。

你可以通过前往 **Ultimate Multisite > Settings > Sites**，然后在侧边栏点击 Edit Placeholders 链接来访问模板占位符编辑器。

这将带你进入占位符内容编辑器，在那里你可以添加占位符及其对应的内容。
