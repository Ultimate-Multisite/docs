# 如何获得帮助

您已获得有效的 License key，并且随之获得了我们团队的支持，以帮助您在设置 Ultimate Multisite 期间解决可能遇到的问题。

本节内容涵盖了许可持有者可用的支持渠道，以及在联系支持团队时需要准备的相关信息。

## Help Site

您可以通过以下链接访问我们的帮助站点，获取不同的文章和教程。

[)

我们鼓励用户熟悉 Ultimate Multisite 的基本功能和特性。我们的帮助站点提供了“快速入门”文章，并且可以在其中找到常见问题的答案，例如如何安装插件、如何集成支付网关等。

## Contact Support

如果您遇到困难，无法在帮助站点找到答案，您可以通过点击网站右下角的聊天气泡图标，或直接发送电子邮件至 support@ultimatemultisite.com 与我们的支持团队取得联系。

在联系技术支持时，请确保提供详细的故障描述，并附上截图、错误信息或任何有助于支持人员快速理解和评估问题的资料。同时，请提供您的 License key。仅活跃的 License key 持有者有权获得支持。

## Community Site

我们的 [community site](https://community.ultimatemultisite.com/) 是 Ultimate Multisite 用户相互交流的平台。它是一个信息系统，成员可以发布、评论讨论、提供建议、协作或众包。您需要提供您的 Ultimate Multisite license key 才能解锁私人空间/页面。
