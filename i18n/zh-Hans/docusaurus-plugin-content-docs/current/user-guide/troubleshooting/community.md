# 如何加入 Ultimate Multisite 社区

Ultimate Multisite Community 是一个可以与其他 Ultimate Multisite 用户交流并分享技巧、经验等的空间。要访问 Ultimate Multisite Community，请访问 https://community.ultimatemultisite.com/home。

在欢迎页面，点击“**在此创建您的账户**”链接。

<!-- Screenshot unavailable: Community welcome page with creating your account link -->

您将被重定向到我们的注册页面。现在需要输入用户名、电子邮件和密码。完成验证码后，点击 **注册** 按钮。

<!-- Screenshot unavailable: Community register page with username, email, password fields and Captcha -->

注册完成后，您可以创建个人资料。在此页面，您可以上传照片、输入全名、写一段简短的自我介绍等。同时，添加您的社交媒体链接并设置账户权限。

<!-- Screenshot unavailable: Community profile creation page with photo upload and bio fields -->

现在您已进入“Introductions”频道，您可以在此介绍自己并了解其他成员。点击 **新帖** 按钮即可创建帖子。

<!-- Screenshot unavailable: Community Introductions channel with New Post button -->

Lounge 区域和 “Dev Logs” 频道对所有人开放。然而，您需要特殊权限才能访问社区中的其他频道。

<!-- Screenshot unavailable: Community channels showing Lounge section and Dev Logs channel -->

当您提供有效的 Ultimate Multisite 许可证密钥后，即可获得此权限。请点击任意私有频道链接，然后点击 **解锁** 按钮。

<!-- Screenshot unavailable: Community private channel with Unlock button -->

您将被重定向到我们的解锁访问页面。输入您的 Ultimate Multisite 许可证密钥并点击 **解锁访问** 按钮：

<!-- Screenshot unavailable: Community unlock access page with license key input -->

您的许可证将被验证，随后您即可访问私有频道。此过程可能需要最多 24 小时，请点击链接返回社区，在我们处理您的请求期间享受其他空间。
