# 如何加入 Ultimate Multisite Community {#how-to-join-ultimate-multisite-community}

Ultimate Multisite Community 是一个空间，你可以在这里与其他 Ultimate Multisite 用户联系，并分享技巧、经验等内容。要访问 Ultimate Multisite Community，请前往 https://community.ultimatemultisite.com/home。

在欢迎页面，前往“**在这里创建你的 Account** ”链接。

<!-- Screenshot unavailable: Community welcome page with creating your account link -->

你将被重定向到我们的注册页面。现在你需要输入用户名、你的电子邮件和密码。完成 Captcha，然后点击 **Register** 按钮。

<!-- Screenshot unavailable: Community register page with username, email, password fields and Captcha -->

注册后，就该创建你的个人资料了。在此页面，你可以上传你的照片、输入你的全名、写一段简短的个人简介等等。另外，添加你的社交媒体链接，并为你的 Account 设置权限。

<!-- Screenshot unavailable: Community profile creation page with photo upload and bio fields -->

现在你位于“Introductions”频道，在这里你可以介绍自己，并稍微了解其他成员。你可以点击 **New Post** 按钮来创建你的帖子。

<!-- Screenshot unavailable: Community Introductions channel with New Post button -->

Lounge 部分的频道和“Dev Logs”频道向所有人开放。不过，你需要特殊权限才能访问社区中的其他频道。

<!-- Screenshot unavailable: Community channels showing Lounge section and Dev Logs channel -->

一旦你向我们提供有效的 Ultimate Multisite license key，此访问权限就会被授予。要执行此操作，请点击任意私密频道链接，然后点击 **Unlock** 按钮。

<!-- Screenshot unavailable: Community private channel with Unlock button -->

你将被重定向到我们的解锁访问页面。输入你的 Ultimate Multisite license key，然后点击 Unlock Access 按钮：

<!-- Screenshot unavailable: Community unlock access page with license key input -->

你的许可证将被验证，随后你将能够访问私密频道。此过程最多可能需要 24 小时，因此请点击链接返回社区，并在我们处理你的请求时享受其他空间。

<!-- Screenshot unavailable: Community license validation confirmation page -->
