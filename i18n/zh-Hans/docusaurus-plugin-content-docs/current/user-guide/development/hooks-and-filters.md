# Ultimate Multisite 钩子与过滤器

Ultimate Multisite 由大量函数和数据库查询组成。它拥有钩子和过滤器，允许你修改或向现有函数添加新代码。

我们创建了一个这些钩子和过滤器的列表，作为参考，帮助高级用户进一步自定义他们的网络。你可以点击下面的链接从 GitHub 访问该列表。

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
