# 使用 Ultimate Multisite: Ad Injector 在你的网络中投放广告

Ultimate Multisite: Ad Injector 是一个**免费 add-on**，可让你直接从套餐的编辑页面获得在套餐中投放广告的工具。

安装 Ultimate Multisite: Ad Injector 后，前往你的 _Plugins_ 页面将其激活。

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_点击 Network Activate 按钮_

add-on 激活后，套餐编辑界面上的 _Advanced Options_ 元数据框会出现一个名为 _Ad Settings_ 的新选项。

![套餐编辑界面上的广告设置](/img/config/product-edit-full.png)

_现在你可以为每个套餐编辑广告设置_

在该选项卡中，你可以为前端中的文章和页面添加广告代码，显示在内容之前和之后。

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_插入你的广告代码后，它会显示在该特定套餐的文章和页面上。_

你还可以在该套餐用户的 Admin Pages 页眉中包含广告。

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_如果你想在用户的后端显示广告，请前往 Admin Panel Ad Code 选项_

### 不止广告 {#beyond-advertising}

你还可以使用 Ultimate Multisite: Ad Injector 为高阶套餐提供支持。

只需前往 _Admin Panel Ads_ 选项，并添加你的支持工具的 HTML 代码。

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_为特定套餐添加客户支持_

这样用户就可以直接从他们的 Admin Panel 联系你！
