# 自定义您的管理面板使用 PRO Theme

在为客户工作时，我们经常需要为他们的 WordPress 管理仪表盘提供定制化体验。如果您正在寻找更专业的外观和感觉，PRO Theme 可能会帮到您。

PRO Theme 是一款为您在定制客户仪表盘时提供帮助的管理主题。它也是 Ultimate Multisite 高级插件核心的一部分。

安装后，前往您的 _插件_ 页面，在网络中激活它。

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_在插件页面全网激活它_

**注意:** 如果您只是安装它，它会在您的客户那里显示为一个选项，您可以勾选以仅将主题添加到该用户。

现在，您的管理界面已完全重新设计！左侧栏还新增了一个名为 PRO Theme 的菜单。
