# 使用 PRO Theme 自定义你的管理面板

在为客户工作时，我们经常需要在 WordPress 的管理 Dashboard 中为客户提供自定义体验。如果你正在寻找更专业的外观与体验，PRO Theme 可能会帮到你。

PRO Theme 是一个管理 theme，旨在帮助你自定义客户的 Dashboard。它也是 Ultimate Multisite 高级 plugins 核心的一部分。

安装后，前往你的 _Plugins_ 页面，并点击在你的 Network 上启用它。

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_在你的 Plugins 页面上进行全 Network 启用_

_**注意：**_ _如果你只是安装它，它会在你的客户侧显示为一个可选项，可被勾选以仅为该用户添加该 theme。_

现在，你的管理界面已经完全重新设计！左侧栏中还会出现一个名为 PRO Theme 的新菜单。
