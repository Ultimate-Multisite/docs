# 第二课：选择你的细分市场 {#lesson-2-picking-your-niche}

在第一课中，我们了解了为什么细分市场优于泛泛而谈。现在，你需要选择你的细分市场。本课将为你提供一个框架，帮助你在投入时间构建之前，找到并验证一个细分市场。

## 回顾进度 {#where-we-left-off}

我们定义了 FitSite 概念：一个为健身工作室打造的网站平台。但我们是如何确定专门针对健身工作室的呢？本课将带你走过整个流程，让你能够将这个方法应用到寻找自己的细分市场。

## 三个标准 {#the-three-criteria}

一个好的网站平台细分市场必须满足三个条件：

### 1. 他们需要网站，但难以获得高质量的网站 {#1-they-need-websites-but-struggle-to-get-good-ones}

寻找那些符合以下条件的行业：

- 企业普遍需要拥有网站存在感
- 该行业目前大多数网站质量较差
- 通用建站工具无法解决他们特定的需求
- 为普通规模的企业定制开发成本过高

**健身工作室符合**：会员期望在线找到课程表、价格和预订信息。大多数工作室的网站过时，或者使用了通用模板，无法真正满足需求。

### 2. 这些企业能够负担月度订阅费用 {#2-the-businesses-can-afford-a-monthly-subscription}

你的目标客户需要具备以下特征：

- 足够成熟，拥有市场营销预算
- 习惯为软件订阅付费（例如预订系统、支付处理等）
- 运营规模达到每月 $49-$199 是一个合理的业务开支

**健身工作室符合**：他们已经为预订软件、支付处理和营销工具付费。网站订阅是自然而然的补充。

### 3. 他们之间相互交流 {#3-they-talk-to-each-other}

最好的细分市场具备以下特点：

- 行业协会、会议或贸易展览会
- 活跃的线上社区（Facebook 群组、论坛、Reddit 子版块）
- 业务所有者分享推荐的本地人脉网络

**健身工作室符合**：工作室老板会参加健身行业的活动，参与线上社区，并经常与同行分享工具和服务。

## 如何寻找细分市场候选对象 {#how-to-find-niche-candidates}

### 从你了解的领域开始 {#start-with-what-you-know}

你现有的经验是一项优势。可以考虑：

- 你曾工作或为之构建过网站的行业
- 你了解其社区的爱好或兴趣
- 你已经所属的专业人脉网络

### 寻找痛点信号 {#look-for-pain-signals}

搜索证据，证明某个细分市场的企业在网站方面存在困难：

- 在 Google 搜索 “[行业] 网站构建器”，看看目前存在哪些选择
- 浏览行业论坛，寻找关于网站存在感的抱怨
- 查看该细分市场实际的网站——大多数是否都很差？
- 检查是否存在竞争对手，以及他们服务市场的程度如何

### 评估经济可行性 {#evaluate-the-economics}

对于每一个候选细分市场，估算以下内容：

- **市场规模**：这个细分市场有多少企业？（先从你的国家/地区开始）
- **支付意愿**：他们目前为类似工具花费多少钱？
- **潜在终身价值**：这些企业是否稳定，还是会快速流失？

## 验证你的细分市场 {#validating-your-niche}

在构建任何东西之前，必须验证需求：

### 与潜在客户交谈 {#talk-to-potential-customers}

联系目标细分市场中的 10-15 位企业主。询问他们：

- 你们现在的网站是怎么弄到的？
- 你们希望它能做到，但目前做不到的是什么？
- 目前你们为网站支付了多少费用？
- 如果有一个专门为 [你的细分市场] 构建的平台，你们是否有兴趣？

你现在不是在销售。你是在倾听。

### 检查竞争格局 {#check-competitor-landscape}

- **没有竞争对手**：可能意味着没有需求，也可能意味着这是一个尚未开发的机遇。客户的对话会告诉你答案。
- **少数竞争对手**：这是一个好兆头。它验证了需求。研究他们做得好的地方和不足的地方。
- **大量竞争对手**：进入难度较大，但如果你能清晰地做出差异化，仍然是可行的。

### 建立着陆页 (Landing Page) {#build-a-landing-page}

在构建平台之前，先创建一个简单的着陆页，描述你的细分市场解决方案，并收集电子邮件注册。运行少量定向广告来测试兴趣。如果人们注册了，你就获得了验证。

## 将此方法应用于 FitSite {#applying-this-to-fitsite}

以下是 FitSite 相较于这些标准的评分：

| 标准 | 评估 |
| :--- | :--- |
| 需要网站但难以获得 | 是——大多数工作室网站是通用的或过时的 |
| 能负担订阅费用 | 是——他们已经为预订软件支付了 $50-$200/月 |
| 相互交流 | 是——活跃的社区、行业活动、本地网络 |
| 市场规模 | 仅在美国就有 100,000+ 个设施 |
| 竞争格局 | 很少有针对特定细分市场的网站平台 |

验证确认了这是一个机会。现在我们开始构建。

## 可考虑的细分市场点子 {#niche-ideas-to-consider}

如果健身不是你的市场，这里还有其他符合标准的细分市场：

- **餐厅和咖啡馆** — 菜单、预订、营业时间、外卖集成
- **房地产经纪人** — 房源列表、经纪人资料、社区指南
- **律师事务所** — 专业领域、律师简介、咨询预订
- **教会和非营利组织** — 活动、布道、捐款、社区建设
- **贸易承包商**（水管工、电工、暖通空调）— 服务区域、预订、评论
- **沙龙和水疗中心** — 服务项目、预订、造型师资料、图库
- **兽医诊所** — 服务项目、宠物门户、预约预订
- **音乐学校** — 教师资料、课程安排、毕业表演信息

这些市场都具有相同的特征：明确的网站需求、支付能力和社区口碑传播。

## 本课学到的内容 {#what-we-built-this-lesson}

- **细分市场选择框架**：用于评估任何细分市场的三个标准
- **验证方法**：客户对话、竞争对手分析、着陆页测试
- **FitSite 验证**：确认健身工作室符合所有标准
- **替代细分市场**：一份你可以应用此框架的其他垂直领域列表
