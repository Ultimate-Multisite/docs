# 第 7 课：打造属于你的品牌

您的客户不应该觉得他们在使用“某个 WordPress 插件”。他们应该感觉在使用 FitSite——一个专为他们的行业打造的平台。本课将涵盖品牌化、白标（white-labeling）以及如何让整个平台看起来像一个成熟的产品。

## 我们上次讲到哪里了

FitSite 已经拥有了一个可用的结账流程，能引导健身房老板从选择套餐到拥有一个实时的网站。现在，我们要让整个用户体验感觉像一个统一、品牌化的产品。

## 您的平台域名

您品牌的基石是您的域名。对于 FitSite 来说：

- **主域名 (Main domain)**：`fitsite.com`（您的营销网站和网络根目录）
- **客户网站 (Customer sites)**：`studioname.fitsite.com`（子域名）
- **自定义域名 (Custom domains)**：Growth 和 Pro 套餐的客户可以映射他们自己的域名

### 设置您的域名

1. 注册您的平台域名
2. 将其指向您的主机提供商
3. 为客户子域名配置通配符 DNS (`*.fitsite.com`)
4. 确保通配符 SSL 已激活

有关详细说明，请参阅 [如何配置域名映射](/user-guide/domain-mapping/how-to-configure-domain-mapping)。

## 白标化后台体验

当健身房老板登录其网站后台时，他们应该看到的是您的品牌，而不是 WordPress 或 Ultimate Multisite 的品牌标识。

### 自定义登录页面

自定义 WordPress 登录页面，展示以下内容：

- 您的 FitSite logo
- 适合健身行业的背景图片
- 您的品牌颜色

### 后台品牌化

使用 [Admin Page Creator](/addons/admin-page-creator) 插件或自定义 CSS 来：

- 用您的 FitSite logo 替换 WordPress logo
- 自定义后台配色方案，使其与您的品牌匹配
- 添加一个自定义的后台小工具，包含健身相关的快捷链接和帮助资源

### 自定义后台页面

考虑创建自定义后台页面，将健身房老板最相关的操作放在最显眼的位置：

- “编辑您的课程表”
- “更新教练资料”
- “查看您的网站”

这样做可以减少学习曲线，将与行业相关的核心操作放在最前面，而不是让用户在标准的 WordPress 菜单中深挖才能找到。

## 品牌化您的沟通内容

每一封邮件、每一张发票和每一个通知都应该强化您的品牌形象。

### 系统邮件

导航到 **Ultimate Multisite > Settings > Emails**，并自定义所有系统邮件：

- **发件人名称 (From name)**：FitSite
- **发件人邮箱 (From email)**：hello@fitsite.com
- **邮件模板 (Email templates)**：使用您的品牌颜色和 logo
- **语言 (Language)**：全程使用健身行业术语

需要自定义的关键邮件包括：

| 邮件 | 通用版本 | FitSite 版本 |
|-------|----------------|-----------------|
| 欢迎信 | “您的新网站已就绪” | “您的健身房网站已上线” |
| 付款收据 | “已收到付款” | “FitSite 订阅付款已确认” |
| 试用结束 | “您的试用即将结束” | “您的 FitSite 试用将在 3 天后结束——请保持您的健身房网站在线” |

### 发票

使用以下内容自定义发票模板：

- 您的 FitSite logo 和品牌颜色
- 您的企业详细信息
- 健身相关的产品名称（而不是通用的套餐 ID）

## 客户可见的网站

您的主域名 (`fitsite.com`) 需要一个营销网站来销售整个平台。这与 Ultimate Multisite 网络后台是分开的——它是您业务的公共门面。

关键页面包括：

- **首页 (Homepage)**：为健身企业提供清晰的价值主张
- **功能 (Features)**：FitSite 能做什么，用健身行业的语言描述
- **定价 (Pricing)**：您的三个套餐，并进行行业特定的功能对比
- **案例 (Examples)**：展示使用该平台构建的网站
- **注册 (Sign Up)**：链接到您的结账表单

:::tip 您的营销网站可以是一个网络站点
您可以将营销网站创建为您自己网络中的一个站点。这样您就可以从同一个后台管理它，同时也展示了您自己平台的强大功能。
:::

## 客户的自定义域名

对于包含自定义域名的套餐客户，请清晰地记录流程：

1. 客户在域名注册商处注册或转移其域名
2. 客户更新 DNS，将其指向您的平台（提供确切的记录）
3. Ultimate Multisite 处理域名映射和 SSL

请为这个流程创建一个帮助文章或知识库条目，并且要用非技术背景的健身房老板的语言来撰写。

```
│   └── fitsite.com 上的营销网站

```
