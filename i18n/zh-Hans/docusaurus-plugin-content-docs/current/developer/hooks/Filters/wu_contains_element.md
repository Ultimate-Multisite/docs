# Filter: wu_contains_element

允许开发者更改初始搜索的结果。

这对第三方构建器等很有用。

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $contains_elements | `bool` | 如果元素包含在内容中。 |
| $content | `string` | 正在检查的内容。 |
| $element | `self` | 当前元素。 |
| $post | `null\|\WP_Post` | 要检查的帖子。 |

### 自

- 2.0.0
