# 过滤器：wu_plan_product_types

过滤应被视为计划的产品类型。

此过滤器允许插件注册额外的产品类型，以便在验证和分离时将其识别为计划。

## 参数

| 名称 | 类型 | 描述 |
|------|------|------|
| $plan_types | `array` | 用于将其视为计划的产品类型数组。 |

### 版本

- 2.3.0

### 源码
