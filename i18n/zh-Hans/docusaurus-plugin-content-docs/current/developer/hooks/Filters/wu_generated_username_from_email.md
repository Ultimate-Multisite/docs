# Filter: wu_generated_username_from_email

过滤生成的客户用户名。

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $username | `string` | 生成的用户名。 |
| $email | `string` | 新的客户电子邮件地址。 |
| $new_user_args | `array` | 新的用户参数数组，可能包含名和姓。 |
| $suffix | `string` | 追加字符串到用户名以使其唯一。 |
