# Filter: wu_password_require_uppercase

过滤器，用于决定密码是否需要包含大写字母。

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | 是否需要大写字母。默认在强制规则时为 true。 |
| $defender_active | `bool` | Defender Pro Strong Password 是否启用。 |

### Since

- 2.4.0
