# 过滤器: wu_password_require_number

过滤器，用于决定密码是否需要包含数字。

## 参数

| 名称 | 类型 | 描述 |
|------|------|------|
| $require | `bool` | 是否需要数字。默认在强制规则时为 true。 |
| $defender_active | `bool` | Defender Pro Strong Password 是否激活。 |

### 版本

- 2.4.0
