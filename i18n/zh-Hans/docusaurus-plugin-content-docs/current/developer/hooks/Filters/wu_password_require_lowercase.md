# 过滤器：wu_password_require_lowercase

过滤是否在密码中要求使用小写字母。

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | 是否要求使用小写字母。默认在强制规则时为 true。 |
| $defender_active | `bool` | Defender Pro Strong Password 是否处于激活状态。 |

### Since

- 2.4.0
