# 过滤器：wu_documentation_get_link

允许插件开发者过滤链接。

不确定这会有什么用，但拥有它也无妨。

## 参数

| 名称 | 类型 | 描述 |
|------|------|-------------|
| $link | `string` | 已注册的链接 |
| $slug | `string` | 用于检索链接的 slug |
| $default_link | `string` | 已注册的默认链接 |

### 版本

- 1.7.0

### 源码

定义在 [`inc/class-documentation.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-documentation.php#L125) 的第 125 行
