# 过滤器：wu_add_new_domain_modal_fields

过滤添加新域名模态框的字段。

允许插件（例如 Domain Seller）修改或替换域名注册字段。

## 参数

| 名称 | 类型 | 描述 |
|------|------|------|
| $fields | `array` | 表单字段。 |

### 自 2.1.0
