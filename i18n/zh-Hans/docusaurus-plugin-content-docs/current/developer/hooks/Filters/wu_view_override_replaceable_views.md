# Filter: wu_view_override_replaceable_views

允许开发者向可替换列表添加额外文件夹。

请注意，允许额外文件夹可能导致加载过时的副本，而不是 Ultimate Multisite 版本。

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $replaceable_views | `array` | 允许的文件夹列表。 |

### Since

- 2.0.0

### Source

在 [`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L50) 定义，位于第 50 行
