# 动作: wu_page_before_render

允许插件开发者在我们打印页面之前添加额外内容。

## 参数

| 名称 | 类型 | 描述 |
|------|------|------|
| `$page_id` | `string` | 此页面的 ID。 |
| `$page` | `object` | 页面对象。 |

- 1.8.2
