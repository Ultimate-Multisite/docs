# 动作: wu_page_after_render

允许插件开发者在我们打印页面后添加额外内容

## 参数

| 名称 | 类型 | 描述 |
|------|------|------|
| $page_id | `string` | 此页面的 id |
| $page | `object` | 页面对象。 |
