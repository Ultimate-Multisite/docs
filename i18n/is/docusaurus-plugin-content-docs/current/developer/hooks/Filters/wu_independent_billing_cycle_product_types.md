# Filter: wu_independent_billing_cycle_product_types

Sía þær vörutegundir sem hafa sjálfstæð rukkunartímabil.

Vörur með sjálfstæð rukkunartímabil eru ekki neyddar til að passa við rukkunartímabil áætlunarinnar í körfunni. Þetta er gagnlegt fyrir vörur eins og lénaskráningar sem eru alltaf rukkaðar árlega, óháð því hvort áætlunin er mánaðarleg eða árleg.

## Færibreytur

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | Fylki af vörutegundum með sjálfstæð rukkunartímabil. |

### Síðan

- 2.5.0
