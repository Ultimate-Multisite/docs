# Sía: wu_get_site_domain_and_path

Leyfir þróunaraðilum að breyta léns-/slóðarpörum.

Þetta getur verið gagnlegt fyrir ýmislegt, svo sem að innleiða einhvers konar sviðsetningarlausn, mismunandi þjóna o.s.frv.

## Færibreytur {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $d | `object` | Núverandi hlutur sem inniheldur lykla fyrir lén og slóð. |
| $path_or_subdomain | `string` | Upprunalega slóðin/undirlénið sem sent var í fallið. |

### Frá útgáfu {#since}

- 2.0.0
### Uppruni {#source}

Skilgreint í [`inc/functions/site.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/site.php#L235) í línu 235


## Skilar {#returns}
Hlut sem inniheldur lykla fyrir lén og slóð.
