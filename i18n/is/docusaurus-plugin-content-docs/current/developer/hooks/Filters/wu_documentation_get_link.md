# Sía: wu_documentation_get_link

Leyfir þróunaraðilum viðbóta að sía tenglana.

Ekki viss hvernig það gæti verið gagnlegt, en það sakar ekki að hafa það

## Færibreytur

| Nafn | Tegund | Lýsing |
|------|------|-------------|
| $link | `string` | Skráði tengillinn |
| $slug | `string` | Slug sem er notað til að sækja tengilinn |
| $default_link | `string` | Sjálfgefni skráði tengillinn |
