# Sía: wp_ultimo_skip_network_active_check {#filter-wpultimoskipnetworkactivecheck}

Þetta er gagnlegt þegar notaðar eru composer-grunnuð og önnur sérsniðin uppsetning, eins og til dæmis Bedrock, þar sem notkun viðbóta sem mu-plugins er venjan.
