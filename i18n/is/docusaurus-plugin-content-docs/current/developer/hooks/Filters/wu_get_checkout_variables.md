# Sía: wu_get_checkout_variables {#filter-wugetcheckoutvariables}

Leyfir plugin-hönnuðum að sía fyrirfram skilgreindar stillingar afgreiðslusíðu.

Farðu varlega, lyklar sem vantar geta alveg bilað afgreiðsluna á framendanum.

## Færibreytur {#parameters}

| Heiti | Tegund | Lýsing |
|------|------|-------------|
| $variables | `array` | Staðfærðar breytur. |
| $checkout | `\Checkout` | Afgreiðslu-klasinn. |

### Frá útgáfu {#since}

- 2.0.0
### Uppruni {#source}

Skilgreint í [`inc/checkout/class-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout.php#L1970) í línu 1970


## Skilar {#returns}
Nýja breytufylkið.
