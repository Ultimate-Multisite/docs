# Sía: wu_minimum_password_strength

Sía lágmarksstyrk lykilorðs sem krafist er (zxcvbn-stig).

Styrkleikastig: - 0, 1: Mjög veikt - 2: Veikt - 3: Miðlungs - 4: Sterkt (sjálfgefið)

## Færibreytur {#parameters}

| Nafn | Tegund | Lýsing |
|------|------|-------------|
| $min_strength | `int` | Lágmarksstyrkleikastig sem krafist er. |
| $strength_setting | `string` | Gildi admin-stillingar (medium, strong, super_strong). |
