# Sía: wu_minimum_password_length

Sía lágmarkslengd lykilorðs.

Aðeins framfylgt þegar wu_enforce_password_rules er true.

## Færibreytur {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $min_length | `int` | Lágmarkslengd lykilorðs. Sjálfgefið 12 (samsvarar Defender Pro). |
| $defender_active | `bool` | Hvort Defender Pro Strong Password sé virkt. |

### Síðan {#since}

- 2.4.0
