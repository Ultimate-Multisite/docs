# Sía: wu_maybe_redirect_to_admin_panel_exclusion_list

Búðu til útilokunarlista yfir færibreytur sem koma í veg fyrir sjálfvirka áframsendingu.

Þetta er nauðsynlegt vegna þess að annars getur síðusmiðurinn ekki breytt innskráningarsíðunni þegar hún hefur verið skilgreind.
