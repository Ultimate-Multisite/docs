# Ultimate Multisite Dokumentation

Velkomið til dokumentationinn fyrir Ultimate Multisite. Ultimate Multisite breytir WordPress Multisite yfir í fullan Website as a Service (WaaS) plattforma.

## Dokumentasjóð {#documentation-sections}

### [Notkunarleiðbeining](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

Allt það sem þú þarft til að sett upp, innreiða og stjórna WaaS plattforma þinni. Innebergurinn dektar innreiðslu, betalingsgerðir, domænnareiningar, viðskiptavarkjarnir og meira.

### [Notkunarleiðbeining fyrir þróunarmiðlinga](/developer/hooks) {#developer-documentation}

Téknilemiðling fyrir þróunarmiðlinga sem byggja á Ultimate Multisite. Innebergurinn hefur referens fyrir hooks, REST API dokumentation og leiðbeiningar fyrir viðbótartilboð (addon development guides).

### [Viðbótartilboð](/addons) {#addons}

Dokumentation fyrir betaldu viðbótartilboð (paid addon plugins) sem breyta Ultimate Multisite í það með að bæta við aðferðar eins og WooCommerce integration, vísindaleg skýrslur (analytics), domænnareiningar og meira.
