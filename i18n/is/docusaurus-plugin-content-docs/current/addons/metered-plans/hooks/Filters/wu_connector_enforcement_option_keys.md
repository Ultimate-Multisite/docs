# Sía: wu_connector_enforcement_option_keys {#filter-wuconnectorenforcementoptionkeys}

Síar listann yfir valkostalykla gervigreindarveitna sem framfylgt er frá aðalvefnum.

Bættu við valkostalyklum fyrir sérsniðin eða þriðja aðila plugin fyrir gervigreindarveitur svo stillingar þeirra erfist einnig frá aðalvefnum á undirvefjum.

## Færibreytur {#parameters}

| Heiti | Gerð | Lýsing |
|------|------|-------------|
| $keys | `string[]` | Heiti valkostalykla sem á að framfylgja (inniheldur þegar tengilykla sem fundnir eru sjálfkrafa og EXTRA_PROVIDER_OPTIONS). |

### Frá útgáfu {#since}

- 1.2.0
