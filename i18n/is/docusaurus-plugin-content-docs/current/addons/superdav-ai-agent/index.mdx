# Superdav AI Agent

Superdav AI Agent er háþróaður WordPress AI-aðstoðarmaður sem útvíkkar Gratis AI Agent með þemameðvituðum færniþáttum, aukinni getu til netleitar, vefsmíðareiginleikum og valfrjálsum sjálfvirknisamþættingum. Útgáfa 1.18.0 bætir við lesverkfærum fyrir Google Calendar, vörpun tengiliða og þátttakenda, samþykktargáttum manna, afritunarvörn fyrir áminningar, TextBee SMS-tilkynningum, stýrðri Superdav-þjónustuuppsetningu og sérdreifðum Advanced-fylgipakka fyrir traust þróunarverkflæði.

## Helstu eiginleikar

- **Þemameðvitaðir færniþættir** — aðlagar leiðbeiningar og getu sjálfkrafa út frá virku WordPress-þema þínu og uppsettum plugins
- **Margir netleitarveitendur** — veldu á milli Brave Search og Tavily fyrir ríkari leitarniðurstöður og háþróaða rannsóknargetu
- **Samskiptaform fyrir vefsmíði** — búðu til fullvirk samskiptaform beint úr spjallviðmótinu með uppsettu form-plugin þínu
- **Bætt vefsmíði** — endurbættar margra fasa vefsmíðaáætlanir með betri plugin-greiningu og villubata
- **Innbyggðir þemafærniþættir** — sérhæfð leiðsögn fyrir Block Themes, Classic Themes, Kadence Blocks og Kadence Theme
- **Bætt WooCommerce-samþætting** — endurunnin til að nota innfædd WooCommerce API fyrir meiri áreiðanleika og samhæfni
- **Sjálfvirk greining veitenda** — greinir og endurnýjar tiltæk plugins og þemu sjálfkrafa þegar þau eru virkjuð eða óvirkjuð
- **Sérsniðin merkjagerð** — búðu til og felldu inn sérsniðin SVG-merki með sjálfvirkri hreinsun sem er örugg fyrir nafnsvið
- **Staðfesting birtuskila í litaspjaldi** — athugaðu litapör með tilliti til WCAG-aðgengis áður en þau eru notuð á þemað þitt
- **Ljósmyndabætt uppgötvun** — hladdu upp ljósmyndum í uppgötvunarviðtali Theme Builder til að hafa áhrif á hönnunarstefnu
- **Matseðlar fyrir veitinga- og gistigeirann** — búðu til skipulagðar síður fyrir matar- og drykkjarseðla fyrir veitingastaði, kaffihús og fyrirtæki í veitinga- og gistigeiranum
- **Forskoðanir fyrir borðtölvu og farsíma** — forskoðaðu hönnun þína á borðtölvu- og farsímatækjum þegar hönnunarstefna er valin
- **Tier 1 vefskrap** — sd-ai-agent/site-scrape er nú sjálfgefið tiltækt til að greina vefsíður samkeppnisaðila og innblásturs
- **Google Calendar lesverkfæri** — lesa stillt dagatöl og viðburði svo sjálfvirknir geti rökhugsað út frá komandi dagskrársamhengi
- **Vörpun tengiliða og þátttakenda** — para dagatalsþátttakendur við þekkta notendur og tengiliði vefs áður en eftirfylgniverk eru búin til
- **Samþykktargáttir manna** — gera hlé á viðkvæmum sjálfvirknum til yfirferðar áður en skilaboð eru send, vinna er hafin aftur eða breytingar gerðar
- **Áminningarfærslur og afritunarvörn** — fylgjast með áminningum sem þegar hafa verið sendar svo endurteknar sjálfvirknikeyrslur tilkynni ekki sama viðtakanda tvisvar
- **TextBee SMS-tilkynningar** — sendu stilltar textaskilaboðatilkynningar í gegnum TextBee þegar SMS-veitandi er virkjaður
- **Stýrð Superdav-þjónustuuppsetning** — tengdu studda vefi við hýsta Superdav endpoints með sjálfvirkri uppsetningu í stað handvirkrar endpoint-stillingar
- **Advanced-fylgipakki** — sérdreifð verkfæri fyrir trausta þróunaraðila fyrir skráakerfi, gagnagrunn, WP-CLI, REST-sendi, plugin-smíði, git-skyndimynd, notendastjórnun og viðmiðunarprófunarverkflæði

## Kröfur

- WordPress 6.0 eða nýrra
- PHP 8.1 eða nýrra
- WordPress Multisite (virkjað á neti) eða uppsetning á einum vef
- Virkt AI-veitendatengingar-plugin (Anthropic, OpenAI eða samhæft)
- Gratis AI Agent v1.9.0 eða nýrra (Superdav AI Agent útvíkkar Gratis AI Agent)

## Uppsetning

1. Hladdu plugin-skránum upp í `/wp-content/plugins/superdav-ai-agent`
2. Virkjaðu plugin í gegnum **Plugins**-skjáinn í WordPress
3. Gakktu úr skugga um að Gratis AI Agent sé einnig virkjað
4. Farðu í **Gratis AI Agent → Settings** og stilltu netleitarveitanda, stýrða Superdav-þjónustutengingu, Google Calendar eða TextBee-auðkenni eftir þörfum

WordPress.org-pakkinn inniheldur kjarnaeiginleika Superdav AI Agent. Advanced-fylgipakkinn er byggður og dreifður sérstaklega fyrir umhverfi þar sem verkfæri fyrir trausta þróunaraðila eru sérstaklega uppsett og heimiluð.

## Þemameðvitaðir færniþættir

Superdav AI Agent greinir sjálfkrafa virka þemað þitt og uppsett plugins og hleður viðeigandi þemameðvituðum færniþætti inn í samhengi aðstoðarmannsins. Tiltækir færniþættir eru meðal annars:

| Færniþáttur | Á við um |
|---|---|
| **Block Themes** | Þemu sem nota blokkamiðaðan WordPress-arkitektúr (Full Site Editing) |
| **Classic Themes** | Hefðbundin WordPress-þemu sem nota PHP-sniðmát og klassíska ritilinn |
| **Kadence Blocks** | Vefi með Kadence Blocks-plugin virkt |
| **Kadence Theme** | Vefi með Kadence-þemað sem virkt þema |

Sjá [Þemameðvitaðir færniþættir](../../user-guide/configuration/theme-aware-skills) fyrir ítarlega leiðsögn um hvern færniþátt.

## Netleitarveitendur

Superdav AI Agent v1.10.0 bætir við **Tavily** sem valkosti við Brave Search fyrir netleitarfyrirspurnir. Báðir veitendur bjóða upp á leitarniðurstöður í rauntíma og hægt er að stilla þá í **Gratis AI Agent → Settings → Advanced**.

| Þjónustuveita | Eiginleikar | Uppsetning |
|---|---|---|
| **Brave Search** | Hraðar leitarniðurstöður með áherslu á persónuvernd | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Ítarlegri leitarniðurstöður, háþróaðir rannsóknarmöguleikar | [Tavily API](https://tavily.com/) |

Sjá [Internet Search](../../user-guide/configuration/internet-search) fyrir leiðbeiningar um stillingar.

## Verkflæði fyrir dagatal, samþykki og tilkynningar

Superdav AI Agent v1.18.0 víkkar út sjálfvirkniverkflæði út fyrir efnis- og vefsmíðaverkefni:

- **Google Calendar verkfæri** leyfa umboðsmanninum að lesa stillt dagatöl og upplýsingar um viðburði þegar dagatalsskilríki eru tiltæk.
- **Hjálparar fyrir tengiliði og þátttakendur** tengja þátttakendur viðburða við WordPress-notendur eða þekkta tengiliði áður en áminningar eða eftirfylgniverkefni eru undirbúin.
- **Samþykkishlið manna** leyfa sjálfvirkni að stöðvast þar til heimilaður notandi yfirfer fyrirhugaða aðgerð.
- **Áminningarfærslur** koma í veg fyrir tvíteknar dagatals- eða tilkynningaáminningar þegar sjálfvirkni er reynd aftur eða tímasett endurtekið.
- **TextBee SMS stuðningur** gerir tilkynningar með textaskilaboðum mögulegar þegar TextBee-þjónustuveitan er stillt.

Notaðu þessi verkfæri fyrir aðstoð sem tekur mið af tímaáætlun, eftirfylgni með þátttakendum og áminningarverkflæði sem krefjast enn mannlegrar yfirferðar áður en viðkvæm skilaboð eru send.

Sjá [Tengiliðaform Site Builder](../../user-guide/administration/site-builder-contact-forms) fyrir ítarlegar leiðbeiningar.

Sjá [Uppgötvunarviðtal](./theme-builder/discovery-interview.md) fyrir ítarlegar leiðbeiningar.

Sjá [Hönnunarstefna](./theme-builder/design-direction.md) fyrir ítarlegar leiðbeiningar.

Sjá [Matseðlar fyrir gestrisnifyrirtæki](./theme-builder/hospitality-menus.md) fyrir ítarlegar leiðbeiningar.

- **[Búa til valmynd](./abilities/create-menu.md)** — búa til leiðsöguvalmyndir með aðgreindum leiðsögumerkingum

Sjá [Verkfæri og stillingar](./theme-builder/tools.md) fyrir upplýsingar um tiltæk verkfæri og Tier 1 vefskrapunartólið.

## Stillingar

Allar stillingar eru undir **Gratis AI Agent → Stillingar** í WordPress-stjórnborðinu. Superdav AI Agent bætir við eftirfarandi stillingarmöguleikum:

| Stilling | Lýsing |
|---|---|
| **Internet Search Provider** | Veldu á milli Brave Search og Tavily (valfrjálst) |
| **Tavily API Key** | API-lykill fyrir Tavily internetleit (ef Tavily er valið) |
| **Managed Superdav Service** | Tengdu studdar síður við hýstu Superdav-þjónustuna með sjálfvirkri útvegun endapunkta |
| **Google Calendar Credentials** | Stilltu dagatalsaðgang svo umboðsmaðurinn geti lesið dagatöl og viðburði |
| **TextBee SMS Provider** | Stilltu TextBee-skilríki fyrir samþykkt SMS-tilkynningaverkflæði |

## Háþróaður fylgipakki

Superdav AI Agent v1.18.0 bætir við sérdreifðum háþróuðum fylgipakka fyrir traust verkflæði þróara. Þessir möguleikar eru ekki innifaldir í grunnpakkanum á WordPress.org. Settu fylgipakkann aðeins upp á síðum þar sem stjórnendur treysta umboðsmanninum sérstaklega til að fá aðgang að þróunarverkfærum.

Háþróuð verkfæri innihalda aðgerðir á skráakerfi, gagnagrunnsaðgang, WP-CLI, REST-sendiforritsverkflæði, hjálpartól fyrir viðbótarsmíði, git-skyndimyndir, hjálpara fyrir notendastjórnun og viðmiðunarprófunarverkfæri. Takmarkaðu þessi verkfæri við trausta stjórnendur og yfirfarðu stillingar samþykkishliða áður en þau eru virkjuð á framleiðslusíðum.
