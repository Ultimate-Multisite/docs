# Fluent Forms {#fluent-forms}

Þessi viðbót gerir mögulegt að bæta takmörkunum við Fluent Forms á Multisite Ultimate í vörum, aðildum eða vefjum.
Fluent Forms notar sérsniðnar töflur í stað sérsniðinna færslutegunda, þannig að venjulegu takmörkunarvalkostirnir virka ekki með Fluent Forms.

## Eiginleikar {#features}

- Bættu við takmörkunum fyrir Fluent Forms eða Conversational Fluent Forms
- Viðskiptavinir fá vingjarnleg skilaboð sem hvetja þá til að uppfæra áætlun sína þegar þeir ná takmörkuninni
