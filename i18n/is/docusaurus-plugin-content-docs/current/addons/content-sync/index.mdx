# Samstilling efnis {#content-sync}

Multisite Ultimate: Samstilling efnis

Þessi viðbót fyrir Multisite Ultimate gerir netkerfisstjórnendum kleift að samstilla efni frá hvaða vef sem er yfir á aðra vefi í netkerfinu. Hún styður allar færslutegundir, þar á meðal færslur, síður, sérsniðnar færslutegundir og Elementor sniðmát. Viðbótin inniheldur fjölþrepa leiðarviðmót til að velja efni og markvefi, með lotuvinnslu og rauntíma framvindusvörun.

## Eiginleikar {#features}

* Efnisval: Sjónrænt kortayfirlit með smámyndum og síun eftir færslutegund
* Allar færslutegundir: Samstillir færslur, síður, vörur, Elementor sniðmát og hvaða sérsniðna færslutegund sem er
* Vefval: Veldu hvaða vefi á að samstilla efni við, með útilokun upprunavefs og forvali undirvefja
* Lotuvinnsla: Samstillir við 10 vefi í einu með framvinduuppfærslum
* Miðlasamstilling: Afritar og endurvarpar miðlaskrám (myndum, myndböndum) sjálfkrafa úr færsluefni yfir á markvefi
* Flokkunarfræði-samstilling: Varðveitir flokka, merki og sérsniðnar flokkunarfræði
* Elementor-stuðningur: Fullur stuðningur við Elementor sniðmát með endurgerð CSS og endurvörpun sniðmátsauðkenna
* Árekstralausn: Valkostur til að skipta út núverandi efni eða sleppa því
* Skyndiminnisstjórnun: Hreinsar Elementor skyndiminni sjálfkrafa á markvefjum eftir samstillingu

## Kröfur {#requirements}

* WordPress 5.3 eða nýrra
* PHP 7.4 eða nýrra
* Multisite Ultimate viðbót (virk)
* Elementor viðbót (valfrjáls, nauðsynleg fyrir samstillingu Elementor sniðmáta)

## Uppsetning {#installation}

1. Hladdu 'ultimate-multisite-content-sync' upp í '/wp-content/plugins/' möppuna
2. Virkjaðu viðbótina í gegnum 'Viðbætur' valmyndina í WordPress netstjórninni þinni
3. Stilltu valkosti í Ultimate Multisite → Stillingar → Samstilling efnis
4. Farðu í Ultimate Multisite → Samstilling efnis til að hefja samstillingu
