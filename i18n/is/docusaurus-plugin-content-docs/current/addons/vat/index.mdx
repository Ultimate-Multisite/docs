# VAT {#vat}

Ultimate Multisite: evrópskt VAT

Bætir við viðbótareiginleikum og verkfærum við sjálfgefna skattameðhöndlun Ultimate Multisite til að styðja betur við evrópskt VAT.
Bætir sjálfkrafa EU VAT-hlutföllum við síðuna Skatthlutföll frá

## Uppsetning {#installation}

1. Hladdu upp 'ultimate-multisite-vat' í '/wp-content/plugins/' möppuna
2. Virkjaðu viðbótina í gegnum valmyndina 'Viðbætur' í WordPress
3. Farðu í Ultimate Multisite > Settings > Taxes og virkjaðu EU VAT.
4. Smelltu á Skatthlutföll og Uppfærðu EU VAT-hlutföll neðst og Vistaðu skatthlutföll.
5. Lokið!
