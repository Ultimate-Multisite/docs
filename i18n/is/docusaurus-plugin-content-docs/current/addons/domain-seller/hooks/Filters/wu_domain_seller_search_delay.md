# Sía: wu_domain_seller_search_delay {#filter-wudomainsellersearchdelay}

Síar biðtíma debounce fyrir lénaleit í millisekúndum.

Hækkaðu þetta gildi til að draga úr API-köllum á hægum tengingum.

## Færibreytur {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $delay | `int` | Biðtími debounce í millisekúndum. Sjálfgefið 500. |
