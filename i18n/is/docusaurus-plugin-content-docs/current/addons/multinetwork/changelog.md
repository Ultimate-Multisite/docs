# Breytingaskrá fyrir Multi-Network

### 1.3.0 {#130}
* Nýtt: Forskoðari netkerfissniðmáta — skoðaðu og forskoðaðu netkerfissniðmát með lifandi spjaldi áður en keypt er.
* Nýtt: Val á netkerfissniðmáti í checkout — viðskiptavinir geta valið netkerfissniðmát í signup checkout ferlinu.
* Nýtt: Cross-network magic link SSO — viðskiptavinir geta fengið aðgang að undirnetkerfum hnökralaust með magic-link auðkenningu.
* Nýtt: Klónun netkerfismiðla — miðlar site eru rétt afritaðir þegar netkerfi eru klónuð úr sniðmáti.
* Lagfæring: Klónun netkerfa virkar nú án multi-tenancy viðbótarinnar; super admin réttindi eru veitt aftur eftir klónun.
* Lagfæring: Stofnun netkerfis er nú samhæf við WooCommerce og allar greiðslugáttir.
* Lagfæring: Gáttarnúmer er nú innifalið í gerð varaléns.
* Lagfæring: CSS fyrir sniðmátaval var flutt í skrásett stílblað til réttrar birtingar í öllum þemum.
* Lagfæring: Vörn gegn óskilgreindum `network_id` lykli í site query scope.
* Lagfæring: Athugun á netkerfisheimildum áður en skipt er um samhengi kemur í veg fyrir villur í `wpdb` töflueiginleikum.
