# Plugin- og þemastjóri

Multisite Ultimate: Plugin- og þemastjóri

Breyttu eða feldu lýsiupplýsingar (titil, höfund, smámynd og lýsingu) uppsettra viðbóta og þema. Búðu til flokka fyrir viðbætur og þemu svo notendur þínir geti síað þau og bættu fallegri sérsniðinni viðbótasíðu við stjórnborð notenda þinna.

Til að nota viðbótina geturðu einfaldlega farið á síðurnar Viðbætur eða Þemu hjá netstjórnanda og smellt á „Breyta upplýsingum viðbótar/þema“.

## Uppsetning {#installation}

1. Hladdu upp 'ultimate-multisite-plugin-and-theme-manager' í '/wp-content/plugins/' möppuna
2. Virkjaðu viðbótina í gegnum 'Viðbætur' valmyndina í WordPress Network Admin

## Notkunarleiðbeiningar {#usage-guide}

Ultimate Multisite: Plugin- og þemastjóri er öflug **premium** Ultimate Multisite viðbót sem veitir þér fulla stjórn á því hvernig netnotendur þínir sjá og hafa samskipti við valkosti viðbóta og þema á netinu þínu.

Þessi viðbót þjónar þremur tilgangi:

  1. Hún gerir þér kleift að breyta lýsigögnum bæði viðbóta og þema algjörlega. Það felur í sér nafn, lýsingu, höfundarnafn og jafnvel smámyndir!

  2. Hún gerir þér kleift að flokka viðbætur og þemu, svo notendur þínir geti auðveldlega raðað þeim.

  3. Hún gefur þér möguleika á að skipta út gömlu, leiðinlegu viðbótasíðunni fyrir nýja þemalíka síðu með röðun eftir flokkum.

Eftir að hafa hlaðið henni niður og sett hana upp birtist nýr flipi - _Stillingar viðbótar_ \- á stillingasíðu Ultimate Multisite.
