# Support Agents {#support-agents}

Að stjórna heilu neti einn er ekki auðvelt verkefni. Þegar netið þitt fer að vaxa sérðu aukna eftirspurn eftir aðstoð. Það er góð hugmynd að fá fólk til að hjálpa þér að létta á vinnuálaginu. Hins vegar er áhættusamt að veita fullan aðgang/heimild og það getur stofnað netinu þínu í hættu. Þar kemur þessi Support Agent viðbót til sögunnar.

Þessi viðbót gerir þér kleift að búa til nýja tegund af WordPress notanda sem kallast Support Agent og veitir þér fulla stjórn á því hvaða heimildir sá notandi hefur. Að veita heimild er jafn auðvelt og að kveikja á rofahnappi og slökkva á honum ef þú ákveður að fjarlægja hana.

## Eiginleikar {#features}

- Sundurliðaður listi yfir WordPress & Ultimate Multisite heimildir
- Virkja og óvirkja aðgang með auðveldum hætti
- Búa til ótakmarkaðan fjölda Agent accounts

## Uppsetning {#installation}

1. Hladdu 'ultimate-multisite-support-agents' upp í '/wp-content/plugins/' möppuna
2. Virkjaðu plugin í gegnum 'Plugins' valmyndina í WordPress
3. Fylgdu Wizard skref fyrir skref til að setja plugin upp
