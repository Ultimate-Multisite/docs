# Hvernig get ég aðgang til reikningarsíðu í fyrsta sem?

Velkomin í Ultimate Multisite!

Hetta er hvernig þú getur aðgang til reikningarsíðu með linknum sem fór í tölvupóstinn eftir að þú hefur kaupa þennan plugin:

<!-- Screenshot unavailable: Account page email with access link -->

Notu tölvupóstinn sem þú sýndir á við kaupin og löseni sem fór í tölvupóstinum.

<!-- Screenshot unavailable: Login page with email and password fields -->

Eftir að þetta sé fyrsta skiljanlegur innlogingin í reikninginn þitt, munu þeir spyrja þig um að breyta lösenið. Breyt það og klikk á _Save_ (Spara) knappa.

<!-- Screenshot unavailable: Change password form with Save button -->

Á reikningarsíðuna þinni geturð þú stýrt yfir allar þína reikningarskilpur, eins og billings, uppfærslu, afhendingar og mikið annað.
