# Stjórnun delingarskila fyrir planum

Ultimate Multisite ger þér möguleika að deila skila-skila (shareable) linkum beint með viðskiptavinum þínum. Til þess, farðu í einhver af útrekkjastum (edit) sínum plana og klikka á knappa _Click to copy the Shareable Link_.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_Á útrekkjastum sínum planins geturðu finna knappa til að skapa skila-skila linkum._

Litið það að þessi virkni notar almenn fjárfestingartíma (billing frequency) til að byggja linkinn. Þetta þýðir að ef almenn fjárfesting þín er mánuðar, t.d., í endanlegt viljað umskiptur þú ert með link sem hefur formið: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Ef þú vilt að skila-skila linkinn verði almennlegur (quarterly), þurfaðu að bæta 3 við endanlegt:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Til að velja planinn _premium_ á ársstjórnu (yearly schedule), þurfaðu að bæta 1 og orðinu:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Hugsaðu þér að þú getur ekki haft bara eina link fyrir ólík persónuafjörðunartíma (billing frequencies). Þú þarft að deila einn sérstakan link fyrir hverja fjárfestingartíma sem þín er.

Eftir að þú hefur deilt linkinn með viðskiptavinum þínum, mun hann leða þeim í fyrsta skrefi í skráningarprosessið þitt.
