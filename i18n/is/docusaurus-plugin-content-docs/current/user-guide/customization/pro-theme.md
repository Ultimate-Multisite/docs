# Skráning þínar Administra-panel með PRO Theme

Ofanlega þegar við gerum vinnu fyrir viðskiptavini þurfa við að veita þeim sérstaka upplifun í administrasjóninu á WordPress. Ef þú leitarð eftir meira faglegum sjónar og kjönnum, getur PRO Theme hjálpa þér.

PRO Theme er administrasjón sem hefur verið byggt til að hjálpa þér við að skrá og breyta dashboard-i fyrir viðskiptavina þín. Það er einnig hluti af kerfinum fyrir premium plugin-a í Ultimate Multisite.

Eftir að þú hefur hlað það í, ferðu yfir síðuna _Plugins_ og klikka á að aktiva það á netverkinn þínu.

<!-- Skjupa ekki til: WP Plugins síða með knapp til aktiva PRO Theme -->

_Aktiva það á netverkinu á síðunni þinni um Plugins_

_**ATBURSTI:**_ _Ef þú bara hlaðir það í, mun það sýna sér sem val fyrir viðskiptavininn þinn sem getur markmið að bæta themeins við þennan notanda._

En nú hefur þú fullyrðu administrasjón settar upp á ný! Þú hefur einnig nýtt menulinum á vinstri hleðingu kallinn PRO Theme.

<!-- Skjupa ekki til: PRO Theme breytilegur administrasjón -->

_Ný sjón fyrir administrasjón þinni_

Á þessum menulinum hefur þú aðgang að fem tabur:

<!-- Skjupa ekki til: PRO Theme Styling Settings tab -->

_Styling Settings_ (Stylingarstillingar)

<!-- Skjupa ekki til: PRO Theme Functionality tab -->

_Functionality_ (Funktionalitet)

<!-- Skjupa ekki til: PRO Theme Custom CSS tab -->

_Custom CSS_ (Sérstaklegr CSS)

<!-- Skjupa ekki til: PRO Theme Import/Export tab -->

_Import/Export_ (Innfærð/Útferð)

<!-- Skjupa ekki til: PRO Theme Activate tab -->

_Activate PRO Theme_ (Aktiva PRO Theme)

Í hverjum þeirrum geturð þú aktivað og inaktivað stýlingar, leyfja eða blokkera funskjalit, og veita viðskiptavinum þínum sér sérstakla breytingar sem þeir gæti leitt eftir á netverk þinn sína.

Hér er dæmi um það sem þú getur fá eftir nokkrar breytingar:
