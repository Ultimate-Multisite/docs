# Hvernig fáir þú sérstaka aðstoð í þróun frá Codeable? {#how-to-get-custom-development-help-from-codeable}

Í nokkrum tilfellum þurfa við okkur svo ótrúlega sérstakla aðlögunar til að gera netverkssambönd okkar nákvæmlega eins og við viljum. Ef það er það sem þú leitar eftir, mælum við með [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), staðnum þar sem þú getur verið í tengslum við erfitt upplýst þróunarframkvæmdavinnu með sérfræði í Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) er plattaform sem tengir viðskiptavini við fullkomlega kvalfðar WordPress freelance þróunarframkvæmdavinnu, sem geta að vinnja á verkefnum almennri háðni. Þeir hafa nákvæma skráningaraferð til að tryggja að þú sért tengd sérfræðingum sem eru sérstaklega kvalfðar fyrir uppgåva sem þú hefur í huga.

### Hvernig skapa verkefni í Codeable? {#how-to-create-a-project-in-codeable}

Fyrst, ferðu yfir [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) og klikka á knappa **Start a Project** (Byrja verkefni).

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

En nú, til að sýna verkefnið þitt, klikka á vinstri hálf yfir í **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

Í "I need help with" (Ég þarf aðstoð með), veldu **Customizing** (Sérstaklega aðlögun):

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

Í "My" (Mín), veldu **Plugin** (Addon):

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Þá klikka á knappa **Continue** (Viðfelu).

<!-- Screenshot unavailable: Codeable project form Continue button -->

En nú er tími til að lýsa verkefni þínu og neitir bestum hátt sem þú getur. Reðið í mögulega hvaða upplýsingar þú getur gefið!

<!-- Screenshot unavailable: Codeable project description form -->

Svaraðu eftirfarandi spurningum til að hjálpa við setningu verðs.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Eftir það, ef þú hefur ekki enn [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) reikning, skapa eitt og klikka á knappa **Publish My Project** (Blaðmi mitt).

<!-- Skjupa ekki til að sýna: Skapning Codeable reikninga og Publish My Project knappa -->

Nánast eftir því sem þú hefur blaðmiðuð [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) projekt þitt, mun plattforma tengja þig við þróunarmenn sem passar til skilyrði og ferli þarf fyrir projekt þitt.

Þessi ferli hjálpar þér að finna rétta menn til vinnu sem hafa erfitt með WordPress og Ultimate Multisite!

Minimumsverð Codeable er $70–$120 USD á tíma. Skattar eru enn framtíðar eftir því sem í gangi er fjölda verkefna, flóknleiki og nauðsynlegheit projektins.

Eftir að þú hefur tekið skattarinn, geturðu hérað þennan þróunarmann sem passar best til þess sem þitt þarf og byrjað með projektinu saman!
