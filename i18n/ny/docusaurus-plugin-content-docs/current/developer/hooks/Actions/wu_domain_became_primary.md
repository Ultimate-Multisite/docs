# Chochita: wu_domain_became_primary

Imayambitsidwa domain ikakhala domain yayikulu ya site.

Chochitachi chimayambitsidwa pamene chizindikiro cha primary_domain cha domain chakhazikitsidwa kukhala true, kaya popanga domain yatsopano yayikulu kapena posintha domain yomwe ilipo kuti ikhale yayikulu.

## Magawo

| Dzina | Mtundu | Kufotokozera |
|------|------|-------------|
| $domain | `\WP_Ultimo\Models\Domain` | Domain yomwe yakhala yayikulu. |
| $blog_id | `int` | ID ya blog ya site yomwe yakhudzidwa. |
| $was_new | `bool` | Kaya iyi ndi domain yomwe yangopangidwa kumene. |
