# Action: wu_page_added {#action-wupageadded}

Lolani opanga ma plugin kuyendetsa zinthu zina pamene masamba alembetsedwa.

Mosiyana ndi wu_page_load, yomwe imayenda kokha pamene tsamba linalake likuwonedwa, hook iyi imayenda pa kulembetsa kwa tsamba lililonse la admin lomwe likuwonjezedwa pogwiritsa ntchito code ya Ultimate Multisite.

## Ma Parameter {#parameters}

| Dzina | Mtundu | Kufotokozera |
|------|------|-------------|
| $page_id | `string` | ID ya tsamba ili. |
| $page_hook | `string` | Dzina la hook la tsamba ili. |

- 2.0.0
