# Fyuluta: wu_should_create_domain_record_for_site

Imasefa ngati Ultimate Multisite iyenera kupanga rekodi ya domain ya tsamba latsopano lopangidwa.

Gwiritsani ntchito fyulutayi kuletsa kapena kuchedwetsa kupanga rekodi ya domain kokha pa masamba omwe amagwiritsa ntchito domain yoyambira yogawana ya fomu yolipirira, host wamkati, kapena domain yomwe kuphatikiza kwina kudzayiyang'anira padera.

## Magawo {#parameters}

| Dzina | Mtundu | Kufotokozera |
|------|------|-------------|
| $create | `bool` | Ngati rekodi ya domain iyenera kupangidwa. |
| $site | `WP_Site` | Chinthu cha tsamba latsopano lopangidwa. |

### Kuyambira {#since}

- 2.13.0

### Gwero {#source}

Yafotokozedwa mu `inc/functions/domain.php`.


## Zobweza {#returns}

Boolean yosonyeza ngati kupanga rekodi ya domain.
