# Filter: wu_magic_link_enforce_ip

Sefa ngati kutsimikizira adilesi ya IP kuyenera kukakamizidwa.

Ikani ku false kuti ma token agwire ntchito kuchokera ku ma netiweki osiyanasiyana. Izi zimachepetsa chitetezo koma zimawonjezera kugwiritsidwa ntchito mosavuta (mwachitsanzo, kwa ogwiritsa ntchito mafoni omwe akusintha ma netiweki).

## Ma Parameter {#parameters}

| Dzina | Mtundu | Kufotokozera |
|------|------|-------------|
| $enforce | `bool` | Ngati kufananitsa adilesi ya IP kuyenera kukakamizidwa. |
