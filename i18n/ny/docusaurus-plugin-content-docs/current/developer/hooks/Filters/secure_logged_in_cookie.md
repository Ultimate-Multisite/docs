# Sefa: secure_logged_in_cookie {#filter-secureloggedincookie}

Imasefa ngati cookie yolowera iyenera kutumizidwa kudzera pa HTTPS kokha.

## Magawo {#parameters}

| Dzina | Mtundu | Kufotokozera |
|------|------|-------------|
| $secure_logged_in_cookie | `bool` | Ngati cookie yolowera iyenera kutumizidwa kudzera pa HTTPS kokha. |
| $user_id | `int` | ID ya wogwiritsa ntchito. |
| $secure | `bool` | Ngati cookie yotsimikizira iyenera kutumizidwa kudzera pa HTTPS kokha. |
