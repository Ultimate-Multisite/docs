# Fyuluta: wu_enforce_password_rules

Fyuluta ngati kukakamiza malamulo owonjezera a mawu achinsinsi.

Ikakhala true, imakakamiza kutalika kochepa ndi zofunikira za zilembo. Imayatsidwa yokha pa kasinthidwe ka "Super Strong" kapena pamene mbali ya Strong Password ya Defender Pro ikugwira ntchito.

## Magawo

| Dzina | Mtundu | Kufotokozera |
|------|------|-------------|
| $enforce_rules | `bool` | Ngati kukakamiza malamulo owonjezera. |
| $strength_setting | `string` | Mtengo wa kasinthidwe ka admin. |
| $defender_active | `bool` | Ngati Defender Pro Strong Password ikugwira ntchito. |

- 2.4.0
