# Fyuluta: wu_magic_link_enforce_user_agent

Fyuluta ngati kutsimikizira user agent kuyenera kukakamizidwa.

Ikani kukhala false kuti ma token agwire ntchito pa ma browser/zida zosiyanasiyana. Izi zimachepetsa chitetezo koma zimawonjezera kugwiritsa ntchito mosavuta.

## Ma Parameter {#parameters}

| Dzina | Mtundu | Kufotokozera |
|------|------|-------------|
| $enforce | `bool` | Ngati kufanana kwa user agent kuyenera kukakamizidwa. |
