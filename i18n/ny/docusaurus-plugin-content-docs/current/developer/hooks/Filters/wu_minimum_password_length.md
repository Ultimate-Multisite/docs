# Fyuluta: wu_minimum_password_length

Fyulutani kutalika kochepa kwa mawu achinsinsi.

Imangokakamizidwa pamene wu_enforce_password_rules ili true.

## Ma Parameters

| Dzina | Mtundu | Kufotokozera |
|------|------|-------------|
| $min_length | `int` | Kutalika kochepa kwa mawu achinsinsi. Default 12 (ikufanana ndi Defender Pro). |
| $defender_active | `bool` | Kaya Defender Pro Strong Password ndi yogwira ntchito. |
