# Sefa: wu_get_site_domain_and_path

Lolani opanga mapulogalamu kusintha magulu a domain/njira.

Izi zingakhale zothandiza pa zinthu zingapo, monga kukhazikitsa yankho lina la staging, maseva osiyanasiyana, ndi zina zotero.

## Magawo {#parameters}

| Dzina | Mtundu | Kufotokozera |
|------|------|-------------|
| $d | `object` | Chinthu chapano chomwe chili ndi makiyi a domain ndi njira. |
| $path_or_subdomain | `string` | Njira/subdomain yoyambirira yomwe inaperekedwa ku function. |

### Kuyambira {#since}

- 2.0.0
### Gwero {#source}

Yafotokozedwa mu [`inc/functions/site.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/site.php#L235) pa mzere 235


## Zobwezera {#returns}
Chinthu chomwe chili ndi makiyi a domain ndi njira.
