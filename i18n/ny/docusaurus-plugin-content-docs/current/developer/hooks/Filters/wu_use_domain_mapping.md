# Sefa: wu_use_domain_mapping

Dziwani ngati kulumikiza kuyenera kugwiritsidwa ntchito

Nthawi zambiri, mudzafuna kulola kulumikiza komwe kuli active kokha kuti kugwiritsidwe ntchito. Komabe, ngati mukufuna kugwiritsa ntchito logic yapamwamba kwambiri, kapena kulola ma domain omwe si active kuti nawonso alumikizidwe, ingosefani apa.

| Dzina | Mtundu | Kufotokozera |
|------|------|-------------|
| $is_active | `bool` | Kodi kulumikizaku kuyenera kutengedwa ngati active? |

| $domain | `string` |  |
