# Sefa: wu_admin_notices

Lolani opanga mapulogalamu kusefa zidziwitso za admin zomwe zawonjezedwa ndi Ultimate Multisite.

## Ma Parameter {#parameters}

| Dzina | Mtundu | Kufotokozera |
|------|------|-------------|
| $notices | `array` | Mndandanda wa zidziwitso za panel imeneyo. |
| $all_notices | `array` | Mndandanda wa zidziwitso zomwe zawonjezedwa, zogawidwa malinga ndi panel. |
| $panel | `string` | Panel yoti mutenge zidziwitso. |
| $filter | `string` | Ngati zidziwitso zomwe zingachotsedwe zasefedwa kuchotsedwa. |
| $dismissed_messages | `array` | Mndandanda wa makiyi a zidziwitso zomwe zachotsedwa. |

- 2.0.0
