# Fyuluta: wu_bypass_unset_current_user

Lolani opanga mapulogalamu kuti adumphe code yochotsa wogwiritsa ntchito wapano.

Kubweza chilichonse chosiyana ndi null kudzadumpha kuchotsa wogwiritsa ntchito wapano amene walowa. Izi zingakhale zothandiza m'mikhalidwe ina, mwachitsanzo, mukamagwira ntchito ndi masamba ang'onoang'ono omwe akugwiritsidwa ntchito ngati mapanelo a oyang'anira.

## Ma parameter {#parameters}

| Dzina | Mtundu | Kufotokozera |
|------|------|-------------|
| $null_or_bypass | `mixed` | Null kuti mupitirire, china chilichonse kuti mudumphe izi. |
| $current_user | `false\|\WP_User` | Chinthu cha wogwiritsa ntchito wapano. |
