# Fyuluta: wu_checkout_form_base_domains

Imasefa madomeni oyambira a fomu yolipirira omwe amagawidwa kuti asakhale zolemba za madomeni olumikizidwa.

Gwiritsani ntchito fyulutayi pamene cholumikizira chimapereka madomeni ena oyambira a minda ya **URL ya Tsamba** ya fomu yolipirira. Madomeni obwezedwa ndi fyulutayi amatengedwa ngati mahosti ogawidwa olembetsera m'malo mwa madomeni apadera a tsamba lililonse.

## Magawo

| Dzina | Mtundu | Kufotokozera |
|------|------|-------------|
| $domains | `array` | Madomeni oyambira ogawidwa omwe asonkhanitsidwa kuchokera ku kasinthidwe ka fomu yolipirira. |

### Kuyambira

- 2.13.0

### Gwero

Zafotokozedwa mu `inc/functions/domain.php`.


## Zobwezedwa

Array ya madomeni oyambira a fomu yolipirira omwe asanjidwa bwino.
