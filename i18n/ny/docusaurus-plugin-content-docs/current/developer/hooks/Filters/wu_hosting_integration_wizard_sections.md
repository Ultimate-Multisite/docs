# Fyuluta: wu_hosting_integration_wizard_sections

Imasefa magawo a wizard a kukhazikitsa kulumikizana kwa hosting.

Imalola addons kuwonjezera, kuchotsa, kapena kusintha magawo a wizard.

## Ma parameter {#parameters}

| Dzina | Mtundu | Kufotokozera |
|------|------|-------------|
| $sections | `array` | Magawo a wizard. |
| $integration | `\WP_Ultimo\Integrations\Host_Providers\Base_Host_Provider\|\WP_Ultimo\Integrations\Integration` | Kulumikizana komwe kukukonzedwa. |
| $page | `\Hosting_Integration_Wizard_Admin_Page` | Instance ya tsamba la wizard. |
