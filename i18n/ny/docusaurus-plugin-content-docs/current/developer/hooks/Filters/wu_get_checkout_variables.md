# Sefa: wu_get_checkout_variables {#filter-wugetcheckoutvariables}

Lolani opanga plugin kusefa zokonzedwa kale za tsamba la checkout.

Samalani, ma key osowa akhoza kuwononga checkout yonse pa front-end.

## Ma Parameter {#parameters}

| Dzina | Mtundu | Kufotokozera |
|------|------|-------------|
| $variables | `array` | Ma variable a localized. |
| $checkout | `\Checkout` | Kalasi ya checkout. |

### Kuyambira {#since}

- 2.0.0
### Gwero {#source}

Zafotokozedwa mu [`inc/checkout/class-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout.php#L1970) pa mzere 1970


## Zobwerera {#returns}
Array yatsopano ya ma variable.
