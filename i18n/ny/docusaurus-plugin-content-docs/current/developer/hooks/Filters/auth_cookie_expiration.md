# Fyuluta: auth_cookie_expiration

Imasefa kutalika kwa nthawi yotha ntchito ya cookie yotsimikizira.

## Magawo

| Dzina | Mtundu | Kufotokozera |
|------|------|-------------|
| $length | `int` | Kutalika kwa nthawi yotha ntchito mu masekondi. |
| $user_id | `int` | ID ya wogwiritsa ntchito. |
| $remember | `bool` | Kaya kukumbukira kulowa kwa wogwiritsa ntchito. Chokhazikika ndi false. |

### Kuyambira

- 2.8.0
