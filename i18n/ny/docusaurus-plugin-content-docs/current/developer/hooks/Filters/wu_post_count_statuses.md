# Fyuluta: wu_post_count_statuses

Lolani opanga plugin kusintha post status yomwe iyenera kuwerengedwa. Mwachisawawa, posts osindikizidwa ndi achinsinsi amawerengedwa

## Ma Parameters {#parameters}

| Dzina | Mtundu | Kufotokozera |
|------|------|-------------|
| $post_status | `array` | Mndandanda wa ma post status |
| $post_type | `string` | Slug ya mtundu wa post |
