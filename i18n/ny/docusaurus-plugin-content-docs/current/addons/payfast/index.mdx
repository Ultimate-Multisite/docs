# PayFast

Multisite Ultimate: Payfast Gateway

Imawonjezera Payfast - njira yolipirira pa intaneti yachangu komanso yotetezeka ku South Africa - ngati njira yolipirira ya Multisite Ultimate.

## Kuyika

1. Kwezani 'ultimate-multisite-payfast' ku chikwatu cha '/wp-content/plugins/'
2. Yatsani plugin kudzera pa menyu ya 'Plugins' mu WordPress
3. Tsatirani Wizard pang'onopang'ono kuti mukonze plugin

## Kalozera Wogwiritsa Ntchito

PayFast ndi kampani yotsogola pokonza zolipira pa intaneti kwa mabizinesi ku South Africa. Imadzipereka kuthandiza amalonda a pa intaneti kukula popanga malonda kukhala osavuta komanso otetezeka. Malinga ndi pempho la wogwiritsa ntchito Ultimate Multisite, tinaganiza zopanga addon yomwe idzapangitse kulumikizana ndi PayFast kukhala kotheka. Nkhaniyi ikusonyeza momwe mungakhazikitsire PayFast ngati njira yanu yolipirira kuti mulandire malipiro mu netiweki yanu ya Ultimate Multisite.

Gawo loyamba ndi kuyika add on.

# Momwe mungayikire Payfast add-on:

Pitani ku tsamba la add-ons ndipo sankhani Ultimate Multisite: Payfast Gateway add-on.

![Tsamba la add-ons](/img/admin/addons-page.png)

Zenera la add-on lidzatuluka. Dinani Install Now.

![Tsamba la add-ons lokhala ndi njira yoyika](/img/admin/addons-page.png)

Kuyika kukamalizidwa, mudzatumizidwa ku tsamba la ma plugin. Apa, ingodinani Network Activate ndipo _Payfast_ add-on idzayatsidwa pa netiweki yanu.

# Kukhazikitsa PayFast

Mukayika add-on, tsopano mutha kupita ku zokonda za Ultimate Multisite ndi kuyamba kukonza PayFast.

Pansi pa tsamba la **zokonda za Ultimate Multisite**, mutha kupeza tabu ya "Payments". Pansi pa tabu iyi, muyenera kuona "**Payfast** " limodzi ndi batani losinthira kuti muyatse kapena kuzimitsa njira yolipirira iyi.

![Zokonda za njira yolipirira](/img/config/settings-payment-gateways.png)Pitilizani ndipo dinani batani losinthira kuti muyiyatsitse. Kenako bokosilo lidzatambasuka kuti lionetse minda yosiyanasiyana yomwe muyenera kudzaza monga **Mechant ID, Merchant Key ndi PassPhrase.** Mutha kupeza zambiri izi kuchokera ku **Payfast** account yanu

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Gawo lotsatira ndi kulowa mu PayFast account yanu kuti mupeze zambiri zomwe mukufuna.

Mukangolowa, mutha kuona nthawi yomweyo zidziwitso zomwe mukufuna kuchokera pa dashboard ya **Payfast** account yanu.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->Mutha kungokopera ndi kumata zidziwitsozo ndi kudzaza tsatanetsatane pa zokonda zanu za Ultimate Multisite. Kenako dinani batani la **"Save settings** " kuti musunge zosinthazo.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Ndi zimenezo, kulumikizana kwanu kwa malipiro a PayFast kwatha!
