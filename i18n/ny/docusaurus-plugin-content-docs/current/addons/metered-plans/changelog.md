# Mbiri ya Kusintha kwa Mapulani a Metered {#metered-plans-changelog}

Version 1.1.0 - Yatulutsidwa pa 2026-05-05
- Chatsopano: Kulipiritsa kwa ma token a AI kwa ma subsite a multisite — tsatirani ndi kulipiritsa kugwiritsa ntchito ma token a AI pamasaiti a makasitomala ndi mitengo yosinthika pa token iliyonse
- Chatsopano: Kukakamiza kwa Connector kumangidwanso ndi kupeza malire mosintha komanso write-through, kuonetsetsa kulondola kwa nthawi yeniyeni pa ma connector onse
- Kukonza: Kukweza matebulo a database tsopano kwagwirizana bwino ndi matanthauzo a schema a BerlinDB, kupewa kulephera kwa kukweza pa ma install atsopano
- Kukonza: Ma callback a kukweza database asinthidwa kukhala mtundu wolondola, kuthetsa kulephera kwa kukweza kosawoneka
- Kukonza: Ma value a fractional tsopano avomerezedwa mu field yolowetsa ya AI Usage Overage Markup
- Kukonza: Zolakwa za fatal ndi mavuto a double-initialization pa kutsegula plugin zathetsedwa
- Zasintha: Yawonjezera script ya check-env npm kuti malo a developer azidzikhazikitsa okha pa kuyendetsa koyamba
