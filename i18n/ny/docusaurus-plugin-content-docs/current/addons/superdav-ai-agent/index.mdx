# Superdav AI Agent

Superdav AI Agent ndi wothandizira wa AI wa WordPress wapamwamba amene amakulitsa Gratis AI Agent ndi maluso ozindikira mutu, luso lowonjezereka lofufuza pa intaneti, zinthu zomangira tsamba lawebusayiti, ndi zolumikizira zosankha zokha. Mtundu 1.18.0 umawonjezera zida zowerengera Google Calendar, kulumikiza anthu olumikizana nawo ndi opezeka pamsonkhano, zipata zovomereza ndi munthu, kuchotsa zikumbutso zobwerezabwereza, zidziwitso za TextBee SMS, kupereka ntchito ya Superdav yoyendetsedwa, ndi phukusi la Advanced companion lomwe limagawidwa payokha pa kayendedwe ka ntchito ka opanga odalirika.

## Zinthu Zofunika {#key-features}

- **Maluso ozindikira mutu** — amasintha okha malangizo ndi kuthekera kutengera mutu wanu wa WordPress womwe ukugwiritsidwa ntchito ndi mapulagini omwe aikidwa
- **Othandizira angapo ofufuza pa intaneti** — sankhani pakati pa Brave Search ndi Tavily kuti mupeze zotsatira zofufuza zolemera komanso luso lapamwamba lofufuza mozama
- **Mafomu olumikizana a womanga tsamba lawebusayiti** — pangani mafomu olumikizana ogwira ntchito mokwanira kuchokera mwachindunji pa mawonekedwe a macheza pogwiritsa ntchito pulagini ya mafomu yomwe mwaika
- **Womanga tsamba lawebusayiti wowonjezedwa** — mapulani omanga tsamba lawebusayiti a magawo angapo okonzedwa bwino okhala ndi kupeza mapulagini kwabwino ndi kubwezeretsa zolakwika
- **Maluso a mitu omangidwa mkati** — malangizo apadera a Block Themes, Classic Themes, Kadence Blocks, ndi Kadence Theme
- **Kulumikizana kwa WooCommerce kokonzedwa bwino** — kwasinthidwanso kuti kugwiritse ntchito ma API achibadwa a WooCommerce kuti pakhale kudalirika ndi kugwirizana kwabwino
- **Kuzindikira othandizira kokha** — amazindikira ndi kutsitsimutsa okha mapulagini ndi mitu yomwe ilipo ikayatsidwa kapena kuzimitsidwa
- **Kupanga logo ya makonda** — pangani ndi kuyika ma logo a SVG a makonda ndi kuyeretsa kokha kotetezeka ku namespace
- **Kutsimikizira kusiyana kwa palette** — onani awiri a mitundu kuti agwirizane ndi zofunikira za kupezeka kwa WCAG musanagwiritse ntchito pa mutu wanu
- **Kuzindikira kowonjezedwa ndi zithunzi** — kwezani zithunzi pa nthawi ya kufunsana kozindikira kwa Theme Builder kuti zikhudze njira ya kapangidwe
- **Menyu za kuchereza alendo** — pangani masamba a menyu a chakudya ndi zakumwa okonzedwa bwino a maresitilanti, makafe, ndi mabizinesi ochereza alendo
- **Zowoneratu pa kompyuta ndi pa foni** — wonerani kapangidwe kanu pa zida za kompyuta ndi za m'manja pa nthawi yosankha njira ya kapangidwe
- **Kukwapula tsamba lawebusayiti kwa Gawo 1** — sd-ai-agent/site-scrape tsopano ilipo mwachisawawa posanthula mawebusayiti a opikisana nawo ndi a kudzoza
- **Zida zowerengera Google Calendar** — werengani makalendala ndi zochitika zomwe zakonzedwa kuti automations zithe kuganiza za nkhani ya ndandanda yomwe ikubwera
- **Kulumikiza anthu olumikizana nawo ndi opezeka pamsonkhano** — fananitsani otenga nawo mbali pa kalendala ndi ogwiritsa ntchito tsamba lawebusayiti ndi anthu olumikizana nawo omwe amadziwika musanapange ntchito yotsatira
- **Zipata zovomereza ndi munthu** — imitsani automations zosakhwima kuti ziwunikidwe musanatumize mauthenga, kuyambiranso ntchito, kapena kusintha zinthu
- **Zolemba za zikumbutso ndi kuchotsa zobwerezabwereza** — tsatirani zikumbutso zomwe zatumizidwa kale kuti kuthamanga kwa automation kobwereza kusadziwitse wolandira yemweyo kawiri
- **Zidziwitso za TextBee SMS** — tumizani zidziwitso za meseji zomwe zakonzedwa kudzera pa TextBee pamene wothandizira SMS wayatsidwa
- **Kupereka ntchito ya Superdav yoyendetsedwa** — lumikizani mawebusayiti othandizidwa ku malo olumikizira a Superdav omwe amasamaliridwa ndi kukonza kokha m'malo mokonza malo olumikizira pamanja
- **Phukusi la Advanced companion** — zida za opanga odalirika zogawidwa payokha za filesystem, database, WP-CLI, REST dispatcher, womanga pulagini, git snapshot, kasamalidwe ka ogwiritsa ntchito, ndi kayendedwe ka ntchito ka benchmark

## Zofunikira {#requirements}

- WordPress 6.0 kapena yapamwamba
- PHP 8.1 kapena yapamwamba
- WordPress Multisite (yoyatsidwa pa netiweki) kapena kuyika kwa tsamba limodzi
- Pulagini yolumikizira wothandizira AI yogwira ntchito (Anthropic, OpenAI, kapena yogwirizana)
- Gratis AI Agent v1.9.0 kapena yapamwamba (Superdav AI Agent imakulitsa Gratis AI Agent)

## Kuyika {#installation}

1. Kwezani mafayilo a pulagini ku `/wp-content/plugins/superdav-ai-agent`
2. Yatsani pulagini kudzera pa chophimba cha **Mapulagini** mu WordPress
3. Onetsetsani kuti Gratis AI Agent yayatsidwanso
4. Pitani ku **Gratis AI Agent → Settings** ndipo konzani wothandizira wanu wofufuza pa intaneti, kulumikizana kwa ntchito ya Superdav yoyendetsedwa, Google Calendar, kapena zizindikiro za TextBee monga zikufunika

Phukusi la WordPress.org lili ndi zinthu zazikulu za Superdav AI Agent. Phukusi la Advanced companion limamangidwa ndi kugawidwa payokha kwa malo amene zida za opanga odalirika zaikidwa ndi kuvomerezedwa momveka bwino.

## Maluso Ozindikira Mutu {#theme-aware-skills}

Superdav AI Agent imazindikira yokha mutu wanu womwe ukugwiritsidwa ntchito ndi mapulagini omwe aikidwa, ndikulowetsa luso loyenera lozindikira mutu mu nkhani ya wothandizirayo. Maluso omwe alipo akuphatikizapo:

| Luso | Limagwira Ntchito Kwa |
|---|---|
| **Block Themes** | Mitu yogwiritsa ntchito zomangamanga za WordPress zozikidwa pa ma block (Full Site Editing) |
| **Classic Themes** | Mitu yachikhalidwe ya WordPress yogwiritsa ntchito ma template a PHP ndi editor wachikhalidwe |
| **Kadence Blocks** | Mawebusayiti okhala ndi pulagini ya Kadence Blocks yogwira ntchito |
| **Kadence Theme** | Mawebusayiti okhala ndi mutu wa Kadence ngati mutu wogwiritsidwa ntchito |

Onani [Maluso Ozindikira Mutu](../../user-guide/configuration/theme-aware-skills) kuti mupeze malangizo atsatanetsatane pa luso lililonse.

## Othandizira Ofufuza pa Intaneti {#internet-search-providers}

Superdav AI Agent v1.10.0 imawonjezera **Tavily** ngati njira ina m'malo mwa Brave Search pa mafunso ofufuza pa intaneti. Othandizira onse awiri amapereka zotsatira zofufuza za nthawi yeniyeni ndipo akhoza kukonzedwa mu **Gratis AI Agent → Settings → Advanced**.

| Wopereka | Makhalidwe | Kukhazikitsa |
|---|---|---|
| **Brave Search** | Zotsatira zosaka zachangu, zolunjika pa zachinsinsi | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Zotsatira zosaka zambiri, luso lapamwamba lofufuza | [Tavily API](https://tavily.com/) |

Onani [Kusaka pa Intaneti](../../user-guide/configuration/internet-search) kuti mupeze malangizo okonza.

## Mayendedwe a Kalendala, Kuvomereza, ndi Zidziwitso {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 imakulitsa mayendedwe a automation kupitirira ntchito za zomwe zili ndi zomangamanga za site:

- **Zida za Google Calendar** zimalola agent kuwerenga makalendala okonzedwa ndi tsatanetsatane wa zochitika pamene zidziwitso zolowera ku kalendala zilipo.
- **Othandizira olumikizana ndi opezeka pamwambo** amafananitsa otenga nawo mbali pamwambo ndi ogwiritsa ntchito a WordPress kapena anthu olumikizana nawo odziwika asanakonze zikumbutso kapena ntchito zotsatira.
- **Zipata zovomerezedwa ndi munthu** zimalola automation kuyima mpaka wogwiritsa ntchito wovomerezeka awunikenso zochita zomwe zaperekedwa.
- **Mbiri za zikumbutso** zimalepheretsa zikumbutso za kalendala kapena zidziwitso kubwerezedwa pamene automation yayesedwanso kapena yakonzedwa mobwerezabwereza.
- **Thandizo la TextBee SMS** limathandiza zidziwitso za mauthenga a pa foni pamene wopereka TextBee wakonzedwa.

Gwiritsani ntchito zida izi pa thandizo lodziwa ndandanda, kutsatira opezeka pamwambo, ndi mayendedwe a zikumbutso omwe akufunikirabe kuwunikiridwa ndi munthu musanatumize mauthenga okhudzidwa.

## Kukonza {#configuration}

Zokonda zonse zili pansi pa **Gratis AI Agent → Settings** mu admin wa WordPress. Superdav AI Agent imawonjezera zosankha zotsatirazi zokonza:

| Zokonda | Kufotokozera |
|---|---|
| **Internet Search Provider** | Sankhani pakati pa Brave Search ndi Tavily (posankha) |
| **Tavily API Key** | API key ya kusaka pa intaneti kwa Tavily (ngati Tavily yasankhidwa) |
| **Managed Superdav Service** | Lumikizani ma site othandizidwa ku service ya Superdav yosungidwa ndi kupereka endpoint kokha |
| **Google Calendar Credentials** | Konzani mwayi wa kalendala kuti agent athe kuwerenga makalendala ndi zochitika |
| **TextBee SMS Provider** | Konzani zidziwitso zolowera za TextBee pa mayendedwe a zidziwitso za SMS ovomerezedwa |

## Advanced Companion Package {#advanced-companion-package}

Superdav AI Agent v1.18.0 imawonjezera Advanced companion package yogawidwa padera ya mayendedwe a opanga mapulogalamu odalirika. Maluso amenewa saphatikizidwa mu phukusi la core la WordPress.org. Ikani companion package kokha pa ma site kumene oyang'anira amakhulupirira momveka agent kuti ipeze zida za opanga mapulogalamu.

Zida za Advanced zikuphatikizapo ntchito za filesystem, mwayi wa database, WP-CLI, mayendedwe a REST dispatcher, zida za plugin builder, git snapshots, othandizira oyang'anira ogwiritsa ntchito, ndi zida za benchmark. Sungani zida izi kwa oyang'anira odalirika okha ndipo unikani kukonza kwa approval-gate musanaziyatse pa ma site a production.
