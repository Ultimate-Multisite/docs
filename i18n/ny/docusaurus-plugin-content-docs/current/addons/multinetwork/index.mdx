# Multi-Network

Addon ya Ultimate Multisite: Multinetwork imakulitsa nsanja yanu ya Ultimate Multisite ndi magwiridwe amphamvu a multinetwork. Addon iyi imathandiza makasitomala anu kugula ndi kuyang'anira ma network athunthu a WordPress multisite, osati ma site payekha okha.

## Zinthu Zofunika

* **Kupanga Network**: Pangani ndi kukonza zokha ma network athunthu a WordPress multisite kwa makasitomala anu
* **Ma Template a Network**: Lolani makasitomala kusakatula, kuwoneratu, ndi kusankha template ya network yokonzedwa kale asanalembetse
* **Kuyang'anira Network**: Kuthekera konse koyang'anira malo a multinetwork
* **Ma Network Odzipatula**: Network iliyonse ya kasitomala imagwira ntchito payokha ndi matebulo ake a database ndi makonzedwe ake
* **Cross-Network SSO**: Perekani makasitomala mwayi wolowa mosavuta ku ma sub-network awo pogwiritsa ntchito kutsimikizira kwa magic-link
* **Kapangidwe Kosavuta Kukulitsa**: Gwirani ma network angapo moyenera pogwiritsa ntchito kugawa chuma kokonzedwa bwino
* **Kuphatikiza**: Imaphatikizana mosavuta ndi makina a subscription ndi billing a Ultimate Multisite

## Ntchito Zake

* **White Label Hosting**: Perekani makasitomala ma network awo a multisite okhala ndi brand yawo
* **Mayankho a Agency**: Thandizani ma agency kuyang'anira ma network angapo a makasitomala kuchokera pa nsanja imodzi
* **Kuyika kwa Enterprise**: Thandizani mabungwe akuluakulu omwe ali ndi zofunikira zovuta za multi-network
* **Nsanja za Maphunziro**: Pangani ma network osiyana a masukulu kapena madipatimenti osiyanasiyana

## Zofunikira

* Ultimate Multisite 2.0.0 kapena yapamwamba
* Kuyika kwa WordPress Multisite
* PHP 7.4 kapena yapamwamba
* MySQL 5.6 kapena yapamwamba

## Kuyika

1. Kwezani mafayilo a plugin ku chikwatu cha `/wp-content/plugins/ultimate-multisite-multinetwork`
2. Yatsani plugin kudzera pa skrini ya 'Plugins' mu WordPress
3. Konzani makonzedwe a multinetwork kudzera pa Ultimate Multisite → Settings

## Ma Template a Network

Mtundu wa 1.3.0 umawonjezera njira yopezera ndi kusankha template kwa makasitomala omwe akugula ma network atsopano. Gwiritsani ntchito ma template a network mukafuna kuti network iliyonse yatsopano ya kasitomala iyambe kuchokera ku kapangidwe kokonzedwa kale, design, laibulale ya zomwe zili, kapena kukonzekera kokhudzana ndi gawo linalake.

### Chowonera template

Chowonera template ya network chimalola makasitomala kusakatula ma template omwe alipo asanafike pa checkout. Template iliyonse ikhoza kutsegulidwa mu gawo lowonera la live, kuti makasitomala athe kuyerekezera ma layout ndi zomwe zili asanagule network.

Gwiritsani ntchito chowonera kuthandiza makasitomala:

* Kuwunika maonekedwe ndi kapangidwe ka template iliyonse ya network yomwe ilipo
* Kuyerekezera ma template angapo osasiya njira yogulira
* Kusankha template yomwe ikugwirizana bwino ndi ntchito yawo asanalembetse

Sungani mayina ndi mafotokozedwe a template kuti akhale osavuta kwa makasitomala. Chowonera chimakhala chothandiza kwambiri makasitomala akamamvetsa ntchito yomwe cholinga chake ndi, monga ma network a agency, ma network a sukulu, ma network a franchise, kapena ma bundle a white-label hosting.

### Chosankhira template pa checkout

Chosankhira template chikhoza kuwonjezedwa ku njira ya signup checkout ngati field. Ikakonzedwa, makasitomala amasankha template ya network yomwe akufuna panthawi ya checkout, ndipo template yosankhidwayo imagwiritsidwa ntchito pamene network yawo yatsopano ikupangidwa.

Kuti mugwiritse ntchito chosankhira mu njira ya checkout:

1. Konzani ma network omwe mukufuna kupereka ngati ma template.
2. Yatsani field yosankha template ya Multinetwork mu fomu yoyenera ya signup kapena checkout.
3. Sankhani ma template omwe ayenera kupezeka pa njira imeneyo ya checkout.
4. Sindikizani njira ya checkout ndipo yesani signup kuti mutsimikizire kuti template yosankhidwayo yagwiritsidwa ntchito pa network yatsopano.

Ngati njira ya checkout imangopereka product imodzi ya network, chosankhira chingakhalebe chothandiza ngati sitepe yotsogolera onboarding. Kwa ma funnel a ma plan angapo, sonyezani kokha ma template omwe amagwirizana ndi product kapena plan yosankhidwa.

## Cross-Network Magic Link SSO

Cross-network magic link SSO imalola makasitomala kusuntha pakati pa dera la Account yawo yayikulu ndi ma sub-network popanda kulowanso pamanja. Kasitomala akatsatira ulalo wolowera wa cross-network, Ultimate Multisite imapanga magic link yokhala ndi nthawi yochepa ndipo imatsimikizira kasitomalayo kulowa mu network yomwe akupita.

Izi zimakulitsa zokumana nazo za makasitomala omwe amayang'anira network yoposa imodzi kapena omwe akufuna kupeza ma network a template omwe akopidwa kumene nthawi yomweyo atagula.

Mndandanda wa admin:

* Tsimikizani kuti addon ya Multinetwork ndi yogwira pa kuyika konse kwa WordPress multisite.
* Onetsetsani kuti makasitomala ali ndi mwayi wolowa mu network yoyenera asanapatsidwe ma link a cross-network.
* Sungani ma domain a network ndi ma domain omwe amamapidwa atakonzedwa bwino kuti ma magic link afike ku network yomwe ikufunidwa.
* Yesani dashboard ya kasitomala ndi mabatani aliwonse olowera pambuyo pa checkout mutasintha makonzedwe a network, domain, kapena checkout.

Ma magic link ayenera kuchitidwa ngati ma link olowera. Amapangidwa kuti makasitomala alowe mosavuta ndipo sayenera kutumizidwa kapena kugwiritsidwanso ntchito kunja kwa session yomwe inali cholinga chake.

## Mafunso Ofunsidwa Kawirikawiri

### Kodi multinetwork ndi chiyani?

Multinetwork ndi mbali ya WordPress yomwe imakulolani kuyendetsa ma network angapo odziyimira pawokha a multisite pa kuyika kumodzi kwa WordPress.

### Izi zimasiyana bwanji ndi multisite wamba?

Multisite wamba amalola ma site angapo mkati mwa network imodzi. Multinetwork imalola ma network angapo, iliyonse ndi ma site ake, onse mkati mwa kuyika kumodzi kwa WordPress.

### Kodi izi zimafuna makonzedwe apadera a server?

Inde, multinetwork imafuna kukonzekera ndi makonzedwe apadera a server. Chonde onani documentation kuti mupeze malangizo atsatanetsatane okhazikitsa.
