# Zosintha za Multi-Network {#multi-network-changelog}

### 1.3.0 {#130}
* Chatsopano: Wowonera template ya network — sakatulani ndi kuwona template za network ndi panel yamoyo musanagule.
* Chatsopano: Wosankha template ya network pa checkout — makasitomala angasankhe template ya network panthawi ya njira ya kulembetsa pa checkout.
* Chatsopano: Cross-network magic link SSO — makasitomala angapeze ma sub-network mosavuta kudzera pa kutsimikizira kwa magic-link.
* Chatsopano: Kukopera media ya network — media ya site imakopedwa molondola pokopera ma network kuchokera ku template.
* Kukonza: Kukopera network tsopano kumagwira ntchito popanda addon ya multi-tenancy; ufulu wa super admin umaperekedwanso pambuyo pokopera.
* Kukonza: Kupanga network tsopano kukugwirizana ndi WooCommerce ndi ma payment gateways onse.
* Kukonza: Nambala ya port tsopano ikuphatikizidwa popanga domain ya fallback.
* Kukonza: CSS ya wosankha template yasamutsidwa kupita ku stylesheet yoyikidwa mu enqueue kuti iwonetsedwe molondola pa themes onse.
* Kukonza: Kuteteza kuti pasakhale kiyi ya `network_id` yosatanthauzidwa mu scope ya kufufuza site.
* Kukonza: Kufufuza chilolezo cha network musanasinthe context kumalepheretsa zolakwika za katundu wa tebulo la `wpdb`.
