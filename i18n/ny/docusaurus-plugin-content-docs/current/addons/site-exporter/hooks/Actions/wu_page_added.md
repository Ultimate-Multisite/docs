# Zochita: wu_page_added

Lolani opanga mapulagini kuyendetsa zinthu zina pamene masamba alembetsedwa.

Mosiyana ndi wu_page_load, yomwe imangoyenda pamene tsamba linalake likuwonedwa, hook iyi imayenda pa kulembetsa kwa tsamba lililonse la admin lomwe likuwonjezedwa pogwiritsa ntchito code ya WP Ultimo.

| $page_hook | `string\|\WP_Ultimo_Site_Exporter\Admin_Pages\Base_Admin_Page` | Hook yopangidwa ku tsambali. |
