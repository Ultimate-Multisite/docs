# Mailchimp {#mailchimp}

Multisite Ultimate: Kulumikizana kwa Mailchimp

Lumikizani netiweki yanu ya Multisite Ultimate ku Mailchimp yanu. Mutha kulumikiza mapulani ena ku Mailchimp Lists ndipo kuwonjezera olembetsa okha ku mindandanda imeneyo akalembetsa.

## Kuyika {#installation}

1. Kwezani 'ultimate-multisite-mailchimp' ku chikwatu cha '/wp-content/plugins/'
2. Yambitsani plugin kudzera pa menyu ya 'Plugins' mu WordPress Network Admin yanu

## Kalozera Wogwiritsa Ntchito {#usage-guide}

Kusakaniza maimelo a olembetsa ndi mindandanda yanu ya MailChimp n'kosavuta kwambiri ndi Ultimate Multisite: MailChimp Integration Add-on yathu. Phunziroli likufuna kukuthandizani kukonza zonse!

## Kupeza API key {#getting-an-api-key}

Tisanayambe kulowa mu makonzedwe osiyanasiyana omwe alipo, tikufuna MailChimp API key. Izi zidzalola Ultimate Multisite kulankhula ndi MailChimp API kuti itenge mindandanda, magulu, ndi kuwonjezera ma adiresi a imelo ku mindandanda ndi magulu amenewo pamene ogwiritsa ntchito alembetsa.

Tsatirani phunziro ili kuti mupeze MailChimp API key yanu: [Za API Keys | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

<!-- Screenshot unavailable: MailChimp API key settings in Ultimate Multisite Add-on Settings -->

_Onjezani MailChimp API key yanu pa_ _**Ultimate Multisite Settings - > Add-on Settings -> MailChimp**_

## Njira Yolumikizira {#integration-mode}

MailChimp Integration add-on imapereka “njira zolumikizira” ziwiri, zomwe zikutanthauza kuti pali njira ziwiri zosiyanasiyana zomwe mungagwiritsire ntchito add-on kuti musakanize ma adiresi a imelo.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_Njira ziwiri zolumikizira kuti pakhale kusinthasintha kwakukulu!_

### Njira 1: Mindandanda Yambiri {#mode-1-multiple-lists}

Njira yokhazikika imagwiritsa ntchito **Mindandanda Yambiri**. Njirayi idzakulolani kusankha mindandanda yambiri ya MailChimp pa pulani iliyonse. Iyi ndi njira yabwino kwambiri ngati mumagawa ogwiritsa ntchito anu m'mindandanda yosiyanasiyana.

Mwachitsanzo, tiyerekeze kuti muli ndi mindandanda 3 yosiyanasiyana pa Account yanu ya MailChimp: **Kalata ya Nkhani** , **Pulani A** , ndi **Pulani B**. Mukufuna ogwiritsa ntchito onse, mosasamala kanthu za pulani yawo, kuti nawonso aphatikizidwe pa mndandanda wa Kalata ya Nkhani, kuti asaphonye zosintha zanu.

Kuti mukwaniritse zimenezo, mutha kugwiritsa ntchito njira ya Mindandanda Yambiri kuti muwonjezere mindandanda yonse iwiri **Kalata ya Nkhani** ndi **Pulani A** ku MailChimp Settings ya Pulani A, ndi **Kalata ya Nkhani** ndi **Pulani B** ku MailChimp Settings ya Pulani B.

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_Sankhani mindandanda yokhazikika ya ma Account atsopano ndi ma Account oletsedwa_

### Njira 2: Mndandanda Umodzi + Magulu Ambiri {#mode-2-single-list--multiple-groups}

MailChimp imakulolani kugawa mndandanda wanu kukhala magulu ang'onoang'ono a ogwiritsa ntchito otchedwa **Magulu**. Izi zimakupatsani ulamuliro wambiri chifukwa mutha kutumiza kampeni ku gawo linalake la olembetsa anu popanda kupanga makope ambiri a kampeni yomweyo pa MailChimp (zomwe zimapangitsa kukhala kovuta kusanthula miyeso ya kampeniyo pambuyo pake).

Mwachitsanzo, mutha kukhala ndi mndandanda umodzi wotchedwa **Olembetsa a Netiweki Yanga,** ndipo mkati mwake muli magulu a **Pulani A** ndi **Pulani B**.

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_Sankhani mindandanda ndi magulu okhazikika, ngati mukugwiritsa ntchito njira yolumikizira ya “mndandanda umodzi + magulu ambiri”_

## Mindandanda ndi Magulu Okhazikika {#default-lists-and-groups}

Mukatha kusankha njira yolumikizira yomwe ikugwirizana kwambiri ndi momwe mukufuna kugwiritsira ntchito, tsopano ndi nthawi yosankha mindandanda yokhazikika (ndi magulu, kutengera njira). Mndandanda wokhazikika udzagwiritsidwa ntchito pamene pulani inayake ilibe Mndandanda wapadera wosankhidwa.

Muthanso kusankha mndandanda wokhazikika wa ma Account oletsedwa. Nthawi iliyonse wogwiritsa ntchito akamaliza subscription yake ndi kuchotsa Account yake ku netiweki yanu, adiresi yake ya imelo idzasamutsidwa ku mndandanda womwe wakonzedwa wa ma Account oletsedwa.

## Kuyambitsa Kulembetsa Kawiri {#enabling-double-subscription}

Pa tsamba lomwelo la makonzedwe, mudzatha kuyambitsa **Kulembetsa Kawiri**. Njirayi idzawonjezera wogwiritsa ntchito ku mndandanda wokhazikika wosankhidwa NDI mndandanda wosankhidwa pa pulani ya wogwiritsa ntchitoyo. Ngati njirayi siyayatsidwa, wogwiritsa ntchito adzawonjezedwa kokha ku mndandanda wosankhidwa wa pulani ya wogwiritsa ntchitoyo. Ngati palibe chomwe chakhazikitsidwa pa pulani imeneyo, ndiye kuti wogwiritsa ntchito adzawonjezedwa ku mndandanda wokhazikika wosankhidwa.

<!-- Screenshot unavailable: MailChimp double subscription toggle setting -->

_Mutha kukhala ndi ogwiritsa ntchito anu olembetsedwa ku mndandanda wokhazikika komanso mndandanda wa pulani akalembetsa_

## Kusankha Mindandanda ndi Magulu pa Pulani iliyonse {#selecting-lists-and-groups-for-each-of-the-plans}

Pitani ku **Mapulani** anu ndipo sankhani pulani yomwe mukufuna kusintha. Dziwani kuti pa meta-box ya **Advanced Options**, tabu yatsopano yotchedwa MailChimp idzapezeka.

Pa tabu imeneyo, mudzatha kusankha mindandanda yambiri kapena mndandanda umodzi ndi magulu ambiri (kutengera njira yolumikizira) pa pulani imeneyo.

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_Sankhani Mindandanda ndi Magulu omwe mukufuna kugwiritsa ntchito pa pulani iliyonse yanu_

Pamene wogwiritsa ntchito watsopano alembetsa pa pulani imeneyo, adiresi yake ya imelo idzawonjezedwa ku mindandanda ndi magulu osankhidwa!
