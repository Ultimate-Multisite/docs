# Loco Translate {#loco-translate}

Ultimate Multisite: Loco Translate Integration addon imabweretsa kuthekera kwa kumasulira kosiyanasiyana pa tsamba lililonse ku WordPress multisite network yanu. Ngati munafunikapo kuti masamba osiyanasiyana mu network yanu akhale ndi matanthauzidwe osiyanasiyana a plugin kapena theme yomweyo, addon iyi ndi yankho labwino kwambiri.

**Zimene Addon Iyi Imachita**

Addon iyi imalumikiza Ultimate Multisite mosavuta ndi plugin yotchuka ya Loco Translate, kulola subsite iliyonse mu network yanu kusunga matanthauzidwe ake apadera. Kaya mukuyendetsa nsanja ya zigawo zambiri, kutumikira magulu a zilankhulo zosiyanasiyana, kapena kungofuna kusinthasintha kwa kumasulira pa network yanu yonse, addon iyi imapangitsa zimenezi kukhala zosavuta.

**Yabwino Kwa:**

* Mabizinesi a zigawo zambiri omwe ali ndi zosowa zosiyanasiyana za localization
* Mabungwe a maphunziro omwe amatumikira magulu osiyanasiyana
* Nsanja za SaaS zomwe zimapereka chithandizo cha zilankhulo zambiri
* Mabungwe a web omwe amayang'anira masamba ambiri a makasitomala
* Multisite network iliyonse yomwe imafuna kusinthasintha kwa kumasulira

## Kuyika {#installation}

1. Onetsetsani kuti Ultimate Multisite yaikidwa ndipo yatsegulidwa pa network
2. Ikani ndi kutsegula plugin ya Loco Translate
3. Kwezani addon iyi ku chikwatu cha maplugin anu
4. Tsegulani Loco Translate Integration addon pa network
5. Yambani kupanga matanthauzidwe apadera a tsamba kudzera mu Loco Translate pa subsite iliyonse

Ndizo basi! Kulumikizanaku kumagwira ntchito zokha popanda kukonza kwina kulikonse.

## Mafunso Ofunsidwa Kawirikawiri {#frequently-asked-questions}

## Kodi ndiyenera kukonza chilichonse nditamaliza kuyika? {#do-i-need-to-configure-anything-after-installation}

Ayi! Kulumikizanaku ndi kodzichitira kokha. Addon ndi Loco Translate zikangotsegulidwa, ma subsite angayambe nthawi yomweyo kupanga matanthauzidwe awo apadera.

## Chimachitika n’chiyani ngati subsite siyipanga matanthauzidwe apadera? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

Tsambalo lidzagwiritsa ntchito zokha matanthauzidwe a network yonse ngati njira yobwerera. Palibe chomwe chimawonongeka, ndipo zonse zimagwira ntchito mosalala.

## Kodi ma subsite angapeze matanthauzidwe a network yonse? {#can-subsites-access-network-wide-translations}

Ma admin wamba a tsamba amangoona ndi kusintha matanthauzidwe a tsamba lawo lokha. Ma super admin angathe kupeza matanthauzidwe a network yonse komanso a tsamba lapadera kutengera tsamba limene akuona.

## Kodi izi zimagwira ntchito ndi ma plugin ndi ma theme onse? {#does-this-work-with-both-plugins-and-themes}

Inde! Matanthauzidwe apadera a tsamba amagwira ntchito pa ma WordPress plugin ndi ma theme.

## Kodi izi zidzachedwetsa multisite network yanga? {#will-this-slow-down-my-multisite-network}

Ayi. Addon imagwiritsa ntchito kutsegula mafayilo moyenera komwe kumangoyang'ana matanthauzidwe apadera a tsamba pakafunika. Palibe kukhudzidwa kooneka pa magwiridwe antchito.

## Kodi ndingakopere matanthauzidwe a network ku tsamba lapadera ngati poyambira? {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

Inde! Pamene mukusintha matanthauzidwe mu Loco Translate, mungayambe mosavuta ndi matanthauzidwe a network ndi kuwasintha kuti agwirizane ndi tsamba lililonse.

## Kodi izi zimagwira ntchito ndi WooCommerce ndi ma plugin ena? {#does-this-work-with-woocommerce-and-other-plugins}

Ndithudi! Plugin kapena theme iliyonse yomwe imagwiritsa ntchito ma function wamba a kumasulira a WordPress idzagwira ntchito ndi matanthauzidwe apadera a tsamba.
