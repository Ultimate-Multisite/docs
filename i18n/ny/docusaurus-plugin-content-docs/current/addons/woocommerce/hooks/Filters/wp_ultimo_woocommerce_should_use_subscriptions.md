# Fyuluta: wp_ultimo_woocommerce_should_use_subscriptions

Lolani opanga kuti akakamize kugwiritsa ntchito WooCommerce Subscriptions.

Mwachisawawa add-on imayang'ana ngati (1) WooCommerce Subscriptions ilipo (mwina plugin yonse kapena core yophatikizidwa), kenako (2) imayang'ana setting yolola kugwiritsa ntchito WooCommerce Subscriptions.

## Ma Parameter

| Dzina | Mtundu | Kufotokozera |
|------|------|-------------|
| $enable_woocommerce_subscription_integration | `bool` | Ngati panopa tikufunika kugwiritsa ntchito Woo Subs. |
| $woo_gateway | `\WP_Ultimo_WooCommerce\Gateways\WooCommerce_Gateway` | Object yapano. |

### Gwero

Yafotokozedwa mu [`inc/gateways/class-woocommerce-gateway.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-woocommerce/blob/main/inc/gateways/class-woocommerce-gateway.php#L361) pa mzere 361


## Zobweza
true kuti mugwiritse ntchito WooCommerce Subs, false kuti musagwiritse ntchito.
