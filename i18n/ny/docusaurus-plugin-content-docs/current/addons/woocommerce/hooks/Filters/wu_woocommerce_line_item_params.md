# Sefa: wu_woocommerce_line_item_params

Sefa ma parameter omwe amagwiritsidwa ntchito kupanga ma line items pa WooCommerce Cart, musanatumize kasitomala ku WooCommerce checkout.

## Ma parameter {#parameters}

| Dzina | Mtundu | Kufotokozera |
|------|------|-------------|
| $line_item_params | `array` | Ma parameter a cart line item. |
| $wu_line_item | `\WP_Ultimo\Checkout\Line_Item` | Instance ya Ultimate Multisite line item. |
| $wc_product | `\WC_Product` | WooCommerce product. |
