# Sefa: wp_ultimo_woocommerce_checkout_fields_to_clean

Imalola opanga mapulogalamu kusefa minda yokhazikika ya adilesi yolipira ya WooCommerce.

Popeza Ultimate Multisite ili ndi minda ya adilesi yolipira, zingakhale zomveka kuchotsa minda yokhazikika yomwe WooCommerce imasonyeza pa fomu ya checkout. Mwachisawawa, palibe mtengo umene umaperekedwa ku mndandandawu, choncho minda yonse yokonzedwa pa WooCommerce kuti iwonetsedwe imasungidwa.

### Kuyambira {#since}

- 2.0.0
### Gwero {#source}

Yafotokozedwa mu [`inc/gateways/class-woocommerce-gateway.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-woocommerce/blob/main/inc/gateways/class-woocommerce-gateway.php#L277) pa mzere 277


## Zobwerera {#returns}
Mndandanda wa minda yochotsa. Mitengo yomwe ilipo ikuphatikiza billing_first_name, billing_last_name, billing_company, billing_country, billing_address_1, billing_address_2, billing_city, billing_state, billing_postcode, billing_email, ndi billing_phone.
