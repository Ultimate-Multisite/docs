# Fyuluta: signup_get_available_languages

Imasefa mndandanda wa zilankhulo zomwe zilipo pa kulembetsa kwa site ya kutsogolo.

Kupereka array yopanda kanthu ku hook iyi kudzayimitsa kutulutsa kwa setting pa fomu yolembetsa, ndipo chilankhulo chokhazikika chidzagwiritsidwa ntchito popanga site. Zilankhulo zomwe sizinayikidwe kale zidzachotsedwa.

## Ma Parameter {#parameters}

| Dzina | Mtundu | Kufotokozera |
|------|------|-------------|
| $available_languages | `array` | Zilankhulo zomwe zilipo. |
