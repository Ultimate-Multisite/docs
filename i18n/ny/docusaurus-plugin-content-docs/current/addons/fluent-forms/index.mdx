# Fluent Forms {#fluent-forms}

Add-on iyi imapangitsa kuti zitheke kuwonjezera malire ku Fluent Forms pa Multisite Ultimate mu zinthu, umembala kapena masaiti.
Fluent Forms imagwiritsa ntchito matebulo apadera m'malo mwa mitundu ya post yapadera, kotero zosankha zanthawi zonse zoyika malire sizigwira ntchito ndi Fluent Forms.

## Zinthu Zake {#features}

- Onjezani malire a Fluent Forms kapena Conversational Fluent Forms
- Makasitomala amalandira uthenga waubwenzi wowalimbikitsa kukweza dongosolo lawo akafika pa malire
