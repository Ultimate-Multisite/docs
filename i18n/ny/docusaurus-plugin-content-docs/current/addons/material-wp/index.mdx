# Material WP

Mawonekedwe oyang'anira a WordPress akhala ofanana kwa nthawi yaitali. Koma kodi munayamba mwadzifunsapo momwe mtundu wa Material Design wa mawonekedwe oyang'anira ungaonekere? 

Material WP ndi mutu wa admin wopangidwa kuti ukupatseni maonekedwe ndi kumverera kumeneko, kukuthandizani kupatsa makasitomala anu Dashboard yosinthidwa kwambiri mwamakonda.

## Zinthu zake

- Thandizo la CSS yosinthidwa mwamakonda
- Zokonda zosavuta za masitayilo
- Kukonzanso dongosolo la menyu
- Kulowetsa/kutumiza kosavuta

## Zolemba

**Kusintha Admin Panel yanu mwamakonda ndi Material WP**

## Kuyika

1. Kwezani 'material-wp' ku chikwatu cha '/wp-content/plugins/'
2. Yambitsani chowonjezera kudzera mu menyu ya 'Zowonjezera' mu WordPress
