# Content Sync

Multisite Ultimate: Content Sync

Addon iyi ya Multisite Ultimate imalola oyang'anira netiweki kulunzanitsa zomwe zili kuchokera pa tsamba lililonse kupita kumasamba ena mu netiweki. Imathandiza mitundu yonse ya zolemba kuphatikiza zolemba, masamba, mitundu ya zolemba zosinthidwa, ndi ma template a Elementor. Addon iyi ili ndi mawonekedwe a wizard a masitepe angapo osankhira zomwe zili ndi masamba omwe akulandira, yokhala ndi kukonza m'magulu ndi mayankho a kupita patsogolo munthawi yeniyeni.

## Zinthu {#features}

* Kusankha Zomwe Zili: Gridi ya makadi owoneka ndi zithunzi zazing'ono ndi kusefa kwa mtundu wa zolemba
* Mitundu Yonse ya Zolemba: Lunzanitsani zolemba, masamba, zinthu, ma template a Elementor, ndi mtundu uliwonse wa zolemba zosinthidwa
* Kusankha Tsamba: Sankhani masamba omwe mungalunzanitse zomwe zili kwa iwo, ndi kuchotsa tsamba lochokera komanso kusankha kale masamba a ana
* Kukonza M'magulu: Imalunzanitsa ku masamba 10 nthawi imodzi ndi zosintha za kupita patsogolo
* Kulunzanitsa Media: Imakopera ndi kusinthanso maulalo a mafayilo a media (zithunzi, makanema) kuchokera ku zomwe zili mu zolemba kupita kumasamba omwe akulandira
* Kulunzanitsa Taxonomy: Imasunga magulu, ma tag, ndi ma taxonomy osinthidwa
* Thandizo la Elementor: Thandizo lathunthu la ma template a Elementor ndi kupanga CSS kachiwiri komanso kusinthanso ma ID a template
* Kuthetsa Mikangano: Njira yosinthira zomwe zilipo kale kapena kuzilumpha
* Kasamalidwe ka Cache: Imachotsa yokha cache ya Elementor pamasamba omwe akulandira pambuyo pa kulunzanitsa

## Zofunikira {#requirements}

* WordPress 5.3 kapena kupitilira apo
* PHP 7.4 kapena kupitilira apo
* Plugin ya Multisite Ultimate (yoyatsidwa)
* Plugin ya Elementor (yosankha, yofunikira pa kulunzanitsa ma template a Elementor)

## Kuyika {#installation}

1. Kwezani 'ultimate-multisite-content-sync' ku chikwatu cha '/wp-content/plugins/'
2. Yatsani plugin kudzera mu menyu ya 'Plugins' mu WordPress Network Admin yanu
3. Konzani zoikamo mu Ultimate Multisite → Settings → Content Sync
4. Pitani ku Ultimate Multisite → Content Sync kuti muyambe kulunzanitsa
