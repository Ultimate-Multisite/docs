# Womanga Tsamba wa AI

ultimate-multisite-ai-site-builder

## Zofunikira

- WordPress 5.3 kapena yapamwamba
- PHP 7.4 kapena yapamwamba
- pulogalamu yowonjezera ya Multisite Ultimate (yogwira ntchito)

## Kuyika

1. Kwezani mafayilo a chowonjezeracho ku chikwatu chanu cha `/wp-content/plugins/`
2. Yambitsani pulogalamu yowonjezerayo kudzera mu menyu ya 'Plugins' mu WordPress
3. Konzani makonda a chowonjezeracho mu woyang'anira network

## Kupanga

### Kukhazikitsa

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Kapangidwe ka Pulojekiti

```
ultimate-multisite-ai-site-builder/
├── inc/                          # Makalasi a PHP ndi zophatikizidwa
│   ├── class-multisite-ultimate-updater.php  # Wosamalira zosintha
│   └── class-ai_site_builder-main.php             # Ntchito yayikulu
├── tests/                        # Mayeso a unit
│   ├── bootstrap.php            # Kuyambitsa mayeso
│   ├── class-*-base.php         # Kalasi yoyambira ya mayeso
│   └── class-*-test.php         # Milandu ya mayeso
├── views/                       # Mafayilo a ma template
├── assets/                      # Katundu wosasunthika
│   ├── css/                     # Ma stylesheet
│   ├── js/                      # Mafayilo a JavaScript
│   └── images/                  # Zithunzi
├── lang/                        # Mafayilo a chinenero
├── composer.json                # Zodalira za PHP
├── package.json                 # Zodalira za Node.js
├── phpunit.xml.dist             # Kasinthidwe ka PHPUnit
├── .phpcs.xml.dist              # Kasinthidwe ka miyezo ya code
└── rector.php                   # Malamulo okonzanso code kukhala yamakono
```

### Kuyesa

Chowonjezerachi chili ndi dongosolo loyesera lokwanira:

- **Mayeso a Unit**: Mayeso ozikidwa pa PHPUnit okhala ndi WordPress test suite
- **Kalasi Yoyambira ya Mayeso**: Imapereka njira zothandizira pa zochitika zofala za mayeso
- **Kuphimbidwa kwa Mayeso**: Kwakonzedwa kuti kutsate kuchuluka kwa code yomwe yayesedwa
- **Kuyesa kwa Multisite**: Mayeso amayenda mu chilengedwe cha multisite

### Miyezo ya Code

- **Miyezo ya Kulemba Code ya WordPress**: Imakakamizidwa kudzera pa PHPCS
- **Kugwirizana ndi PHP 7.4+**: Zinthu zamakono za PHP zimathandizidwa
- **Kusanthula Kosasunthika**: Kuphatikizidwa kwa PHPStan kuti muwone mitundu
- **Kukonzanso Code Kukhala Yamakono**: Malamulo a Rector a kukweza PHP

### Dongosolo Lomangira

- **Kukonza Katundu**: Kuchepetsa mafayilo a CSS/JS
- **Kumasulira**: Kupanga fayilo ya POT
- **Kupakira**: Kupanga archive kuti igawidwe
- **Njira ya Kupanga**: Katundu wosachepetsedwa kuti athandize kuthetsa zolakwika

## Kasinthidwe

Chowonjezerachi chimathandiza zosankha zosiyanasiyana za kasinthidwe zomwe zimapezeka kudzera mu mawonekedwe a woyang'anira network.

## Ma Hooks ndi Ma Filters

### Zochita

- `ultimate-multisite-ai-site-builder_init` - Imayambitsidwa pambuyo poyambitsa chowonjezera
- `ultimate-multisite-ai-site-builder_loaded` - Imayambitsidwa mafayilo onse a chowonjezera atakwezedwa

### Ma Filters

- `ultimate-multisite-ai-site-builder_settings` - Sinthani makonda a chowonjezera
- `ultimate-multisite-ai-site-builder_enabled` - Pitirirani pa momwe chowonjezera chiyenera kuyatsidwa/kuzimitsidwa
