# Ndalama Zosiyanasiyana

Ultimate Multisite: Ndalama Zosiyanasiyana imathandiza netiweki yanu kuvomereza malipiro mu ndalama zosiyanasiyana. Imasintha mitengo yokha pogwiritsa ntchito mitengo yosinthira ndalama ya pompopompo kapena ya pamanja, imakulolani kukhazikitsa mitengo yeniyeni pa malonda aliwonse pa ndalama iliyonse, ndipo imazindikira ndalama zomwe mlendo aliyense amakonda kudzera pa geolocation — zonsezi zikuyendetsedwa kuchokera pa Dashboard imodzi ya woyang'anira.

## Zinthu Zofunika

- **Kusintha kwa mitengo yosinthira ndalama kokha** — mitengo imasinthidwa kuchokera ku ndalama yanu yoyambira pogwiritsa ntchito mitengo yomwe yatengedwa kwa wopereka amene mwasankha
- **Kusintha mitengo pa malonda aliyense** — khazikitsani mitengo yeniyeni ya ndalama inayake pa malonda aliwonse kuti mudutse kusintha kokha
- **Gawo losankha ndalama polembetsa** — limalola makasitomala kusankha ndalama zomwe amakonda pa nthawi yomaliza kugula/kulembetsa
- **Opereka mitengo yosinthira ndalama ambiri** — ECB, Frankfurter, Stripe, kapena mitengo ya pamanja kwathunthu
- **Kuzindikira ndalama pogwiritsa ntchito geolocation** — imasankha kale yokha ndalama yomwe ikugwirizana ndi dziko la mlendo
- **Dashboard ya woyang'anira** — yendetsani ndalama zogwira ntchito, mitengo yosinthira ndalama, ndi makonda a wopereka kuchokera malo amodzi

## Zofunikira

- Ultimate Multisite 2.0.0 kapena yapamwamba
- Kuyika kwa WordPress Multisite
- PHP 7.4 kapena yapamwamba

## Kuyika

1. Kwezani mafayilo a pulagini ku `/wp-content/plugins/ultimate-multisite-multi-currency`
2. Yatsani pulagini kudzera pa skrini ya **Mapulagini** mu WordPress (Yatsani pa Netiweki)
3. Pitani ku **Ultimate Multisite → Ndalama Zosiyanasiyana** kuti mutsegule Dashboard ya woyang'anira
4. Onjezani ndalama zomwe mukufuna kuvomereza ndi kukonza wopereka mitengo yosinthira ndalama wanu

## Kukonza

### Dashboard ya Woyang'anira

Dashboard ya woyang'anira ya Ndalama Zosiyanasiyana (**Ultimate Multisite → Ndalama Zosiyanasiyana**) ndi malo apakati oti:

- Kuwonjezera, kuyatsa, kapena kuzimitsa ndalama
- Kusankha wopereka mitengo yosinthira ndalama ndi kutsitsimutsa mitengo pakafunika
- Kuwunika mitengo yapano ya ndalama zonse zogwira ntchito
- Kukhazikitsa mitengo ya pamanja pamene opereka okha sali oyenera

### Opereka Mitengo Yosinthira Ndalama

| Wopereka | Kufotokozera |
|---|---|
| **ECB** | European Central Bank — mitengo yaulere ya tsiku ndi tsiku ya netiweki zochokera pa EUR |
| **Frankfurter** | API yaulere ya ECB mirror ya open-source — yaulere, safuna API key |
| **Stripe** | Imagwiritsa ntchito mitengo ya FX ya pompopompo ya Stripe — imafuna kuphatikizidwa kwa Stripe kogwira ntchito |
| **Pamanja** | Mumalowetsa mitengo mwachindunji; palibe kuyitana kwa API yakunja |

Sankhani wopereka wanu pansi pa **Ultimate Multisite → Ndalama Zosiyanasiyana → Makonda → Wopereka Mitengo Yosinthira Ndalama**. Mitengo ingatsitsimutshidwe pamanja nthawi iliyonse kapena kukonzedwa kuti isinthe yokha.

### Kusintha Mitengo pa Malonda Aliwonse

Pamene kusintha kokha sikuli kolondola mokwanira, mungakhazikitse mtengo weniweni wa ndalama inayake pa malonda aliwonse:

1. Tsegulani mkonzi wa malonda (**Ultimate Multisite → Malonda → Sinthani**)
2. Pitani pansi mpaka gawo la **Mitengo ya Ndalama**
3. Lowetsani mtengo weniweni wa ndalama iliyonse yomwe mukufuna kusintha
4. Sungani malonda

Malonda opanda kusintha amagwiritsa ntchito mtengo wosinthidwa kuchokera ku ndalama yanu yoyambira.

### Gawo Losankha Ndalama Polembetsa

Gawo losankha ndalama lingawonjezedwe ku njira yanu yolembetsa/yomaliza kugula kuti makasitomala asankhe ndalama zomwe amakonda pa kulembetsa:

1. Pitani ku **Ultimate Multisite → Ndalama Zosiyanasiyana → Makonda**
2. Yatsani **Onetsani chosankha ndalama polembetsa**
3. Mwachisankho, khazikitsani ndalama yokhazikika kapena lolani geolocation kusankha mtengo wosankhidwa kale

Ndalama yosankhidwayo imasungidwa ndi subscription ndipo imagwiritsidwa ntchito pa kulipiritsa konse kotsatira.

### Kuzindikira Ndalama Pogwiritsa Ntchito Geolocation

Pamene kuzindikira kwa geolocation kwayatsidwa, addon imalumikiza dziko la mlendo ndi ndalama ndipo imaisankha kale mu gawo losankha ndalama. Kuti muyatse:

1. Pitani ku **Ultimate Multisite → Ndalama Zosiyanasiyana → Makonda**
2. Yatsani **Zindikira ndalama kuchokera ku malo a mlendo**
3. Lumikizani maiko ndi ndalama mu tebulo la **Kulumikiza kwa Geolocation**

Ngati palibe kulumikiza kwa dziko la mlendo, ndalama yoyambira ya netiweki imagwiritsidwa ntchito ngati yobwerera.

## Mafunso Ofunsidwa Kawirikawiri

### Chimachitika n'chiyani ngati wopereka mitengo yosinthira ndalama sakupezeka?

Addon imabwerera ku mitengo yomwe inatengedwa bwino komaliza. Ngati palibe mitengo yomwe inatengedwa kale, imabwerera ku mitengo ya pamanja yomwe mwakonza. Ndikulimbikitsidwa kukonza mitengo ya pamanja ngati njira yotetezera.

### Kodi ndingagwiritse ntchito opereka osiyanasiyana pa ndalama zosiyanasiyana?

Ayi — wopereka m'modzi amakhala wogwira ntchito nthawi imodzi pa ndalama zonse. Gwiritsani ntchito **Pamanja** ngati mukufuna kuwongolera kwathunthu mitengo iliyonse payokha.

### Kodi geolocation imafuna pulagini yosiyana?

Palibe pulagini yakunja ya geolocation yomwe ikufunika. Addon imagwiritsa ntchito kuzindikira kwa IP-kupita-ku-dziko komwe kuli mkati mwa WordPress. Kuti mupeze kulondola kwakukulu, ntchito yapadera ya geolocation ingaphatikizidwe kudzera pa filter hooks omwe aperekedwa.

### Kodi kusintha pa malonda aliwonse kumagwiritsidwa ntchito musanawonjezere kapena mutawonjezera msonkho?

Kusintha pa malonda aliwonse kumakhazikitsa mtengo usanaperekedwe msonkho mu ndalama yomwe mukufuna. Msonkho umawerengedwa pamwamba pa mtengo wosinthidwa pogwiritsa ntchito makonzedwe anu amsonkho omwe alipo.
