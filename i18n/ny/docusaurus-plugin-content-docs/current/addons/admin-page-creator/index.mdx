# Admin Page Creator {#admin-page-creator}

## Zomwe zili {#content}
Admin page creator ikulolani kuwonjezera mosavuta masamba atsopano a admin ndi masamba ang’onoang’ono mkati mwa WordPress admin dashboard. Ingagwiritsidwe ntchito kupanga tsamba lolandirira kapena tsamba la malangizo lomwe lingakhale tsamba loyambira kwa ogwiritsa ntchito atsopano. Mutha kugwiritsa ntchito page builder monga Beaver Builder, Elementor, Brizy, ndi Oxygen kuti mupange masambawo popanda vuto.

Imaperekanso chithandizo cha HTML, CSS ndi JavaScript zosinthidwa mwamakonda, komanso kuwongolera mwayi wolowa kutengera plan ndi udindo wa wogwiritsa ntchito!

## Zinthu zake {#features}
* Imathandiza Custom HTML ndi CSS
* Chithandizo cha JavaScript
* PHP Content Mode
* Chilolezo kutengera udindo wa wogwiritsa ntchito
* Chithandizo kwa page builders

## Zolemba {#documentation}
[Kupanga Masamba a Admin ndi Masamba Ang’onoang’ono Osinthidwa Mwamakonda pogwiritsa ntchito Multisite Ultimate: Admin Page Creator](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## Kuyika {#installation}

1. Kwezani **ultimate-multisite-admin-page-creator** ku chikwatu cha **/wp-content/plugins/**
2. Yambitsani plugin kudzera mu menyu ya **Plugins** mu WordPress Network Admin yanu

## Kalozera Wogwiritsa Ntchito {#usage-guide}

Ngakhale WordPress imapangitsa kukhala kosavuta kwambiri kuwonjezera zomwe zili ku mbali yakutsogolo ya masaiti, kupanga masamba ndi masamba ang’onoang’ono mbali ya admin kumafuna luso lina loyambirira la kulemba code. Ultimate Multisite: Admin Page Creator cholinga chake ndi kubweretsa kumasuka komweko kogwiritsa ntchito kwa mbali yakutsogolo ku mbali yakumbuyo ya masaiti a netiweki yanu.

## Chifukwa chiyani mugwiritse ntchito Admin Page Creator? {#why-use-admin-page-creator}

Mwina mukufuna kupereka tsamba losinthidwa mwamakonda pa panel ya admin ya makasitomala anu yokhala ndi malangizo, mwina mukufuna kuwonjezera tsamba losinthidwa mwamakonda lokhala ndi ma FAQ ndi contact widget. Zotheka ndi zambiri ndi Admin Page Creator, ndipo ndi zimene tifufuza mu phunziro la lero.

## Kupanga Tsamba lanu loyamba la Custom Admin Page {#creating-your-first-custom-admin-page}

Mukayika ndi kuyambitsa Ultimate Multisite: Admin Page Creator, pitani ku menyu ya network admin yanu ndipo pitani ku Admin Pages.

![Mndandanda wa Custom Admin Pages pa Network Admin](/img/admin/admin-pages-list.png)

_Mndandanda wa Custom Admin Pages pa Network Admin_

Mudzawona mndandanda wa masamba onse a admin omwe mwapanga mpaka pano (popeza uwu ndi ulendo wanu woyamba, mndandandawo udzakhala wopanda kanthu). Pitirizani ndipo dinani **Add new Admin Page**. Mudzatumizidwa ku editor ya Admin Page.

## Zinthu za Custom Admin Page {#the-elements-of-a-custom-admin-page}

Pa editor ya Admin Page, mudzakhala ndi zosankha zingapo zosiyanasiyana zosinthira tsamba lanu mwamakonda.

### Mutu wa Tsamba {#page-title}

Uwu ndi mutu womwe udzawonetsedwa pamwamba pa tsamba, osati label ya menyu ya admin.

<!-- Chithunzi sichikupezeka: Editor ya Admin Page ikusonyeza field ya Page Title -->

_Lowetsani Mutu wa Tsamba_

### Chosankha Mtundu wa Zomwe zili {#content-type-selector}

Ultimate Multisite: Admin Page Creator imakupatsani zosankha ziwiri pankhani ya zomwe zili patsambalo: mutha kugwiritsa ntchito editor yokhazikika ya WordPress WYSIWYG kapena kusintha kupita ku editor ya HTML.

Ngati simudziwa bwino HTML, khalani ndi editor yokhazikika ya WordPress. Mudzatha kusintha zomwe zili patsambalo monga momwe mungachitire ndi Post kapena Page wamba.

<!-- Chithunzi sichikupezeka: Admin Page Creator ikugwiritsa ntchito editor yokhazikika ya WordPress WYSIWYG -->

## Zosankha za Menyu {#menu-options}

Kumbali, pa meta-box ya General Options, mudzatha kukonza mtundu wa menyu, label, ndi zambiri zina.

### Mitundu ya Menyu {#menu-types}

Tsamba lanu latsopano la admin losinthidwa mwamakonda lingakhale ndi mitundu iwiri yosiyanasiyana ya menyu: menyu ya mulingo wapamwamba ndi submenu.

Tsamba la menyu ya mulingo wapamwamba, monga dzinalo likusonyezera, lidzawonjezera tsamba lanu lomwe mwangopanga ku menyu ya admin limodzi ndi masamba okhazikika a WordPress.

Ngati musankha njira ya mulingo wapamwamba, mudzatha kusankha dongosolo la menyu komanso icon ya menyu.

Ngati mukufuna kupanga tsambali kukhala tsamba la sub-menu, mudzatha kusankha tsamba la kholo kuti mulumikize tsambali nalo. Mutha kuwonjezera tsamba lanu losinthidwa mwamakonda ku masamba okhazikika a WordPress kapena ku masamba anu a admin osinthidwa mwamakonda a mulingo wapamwamba.

<!-- Chithunzi sichikupezeka: Zosankha za mtundu wa menyu wa Admin Page Creator zokhala ndi label, dongosolo, ndi makonzedwe a icon -->

_Sinthani kwathunthu chinthu cha menyu cha Custom Page iyi_

<!-- Chithunzi sichikupezeka: Kusankha tsamba la kholo la sub-menu mu Admin Page Creator -->

_Mutha kuwonjezeranso tsambali ngati sub-menu ya masamba ena a menyu, kuphatikizapo anu osinthidwa mwamakonda a mulingo wapamwamba._

## Zosankha Zapamwamba {#advanced-options}

Muyenera kutha kuwonjezera mitundu yonse ya zomwe zili ku masamba anu a admin osinthidwa mwamakonda. Izi zikuphatikizapo malamulo ndi mafayilo a CSS, JavaScript, komanso malaibulale akunja. Mutha kuchita zimenezi pogwiritsa ntchito Advanced Options Tab.

### Custom CSS {#custom-css}

Pa tab ya CSS, mutha kuwonjezera code yanu ya CSS yosinthidwa mwamakonda, komanso kulowetsa mafayilo a CSS kuchokera ku masaiti akutali (monga CDNs, Google Fonts, ndi zina).

<!-- Chithunzi sichikupezeka: Editor ya custom CSS ya Admin Page Creator yokhala ndi kulowetsa stylesheet yakunja -->

_Custom CSS imathandizidwanso. Mutha kuphatikizanso mafayilo akunja a style!_

### Custom JavaScript {#custom-javascript}

Zomwezi zimagwiranso ntchito pa JavaScript. Mutha kuwonjezera malaibulale akunja kuti mugwiritse ntchito pa code yanu ya custom JavaScript.

<!-- Chithunzi sichikupezeka: Editor ya custom JavaScript ya Admin Page Creator yokhala ndi kulowetsa library yakunja -->

_Onjezani code yanu ya JavaScript!_

## Zilolezo {#permissions}

Komanso mu Advanced Options, mudzatha kusankha ma sub-sites omwe adzakhale ndi mwayi wolowa patsambali. Izi n’zothandiza ngati mukukonzekera kupereka zomwe zili zapadera zosiyanasiyana ku magawo osiyanasiyana a plan, mwachitsanzo.

<!-- Chithunzi sichikupezeka: Makonzedwe a zilolezo za Admin Page Creator pa kuwongolera mwayi wolowa kutengera plan ndi udindo -->

_Muli ndi ulamuliro wonse pa kumene tsambali lidzawonekere_

**Chofunika** : Zinthu ziwirizi ziyenera kukwaniritsidwa kuti tsambalo liwonetsedwe. Izi zikutanthauza kuti ngati wogwiritsa ntchito ndi kasitomala wa plan inayake, koma alibe imodzi mwa roles zosankhidwa (kapena mosiyana), tsambalo silidzawoneka pa panel yake.

## Active {#active}

Ngati mukufuna kuyimitsa tsamba la custom admin popanda kulichotsa kapena kusokoneza zilolezo, ingochotsani chizindikiro pa njira ya Active.

Kuyimitsa tsamba la top-level kudzayimitsa zokha ma sub-pages onse omwe alumikizidwa ndi tsamba la parent limenelo.
