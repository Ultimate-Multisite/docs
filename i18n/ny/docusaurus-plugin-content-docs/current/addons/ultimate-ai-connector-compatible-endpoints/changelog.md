# Ultimate AI Connector for Compatible Endpoints Mbiri ya Kusintha

## Version 2.0.0 — Yatulutsidwa pa 2026-04-24

* Chatsopano: Thandizo la opereka ambiri — konzani AI endpoints angapo ndikutumiza zopempha ndi kubwerera yokha kwa opereka ena.
* Kukonza: Kulumikiza kwa SDK ya opereka ambiri ndi ma ID olondola a opereka, ma URL olembetsera, ndi kugawa kwa HTTP filter pa wopereka aliyense.
* Kukonza: Khadi latsopano la wopereka tsopano limadzitambasula lokha likawonjezedwa; kuchotsa cache ya script pa kusintha kwa plugin.
* Kukonza: Namespace ya class ya wopereka yosinthika ya eval() ndi khalidwe lodzitambasula lokha kwa opereka omwe awonjezedwa kumene.
* Kukonza: Imagwiritsa ntchito zigawo za Card/CardBody/CardHeader/CardDivider zokhazikika (sizilinso zoyesera) kuti zigwirizane ndi WordPress 6.9+.
* Kukonza: M'malo mwa DragHandle yosafotokozedwa yaikidwa unicode grip icon ya kukoka-kukonza dongosolo la opereka.
* Kukonza: Yachotsa pempho la HTTP loletsa lomwe limayambitsidwa pa kutsegula kwa tsamba lililonse.
* Zasinthidwa: GitHub Actions workflows zasinthidwa kupita ku Node.js 24.
