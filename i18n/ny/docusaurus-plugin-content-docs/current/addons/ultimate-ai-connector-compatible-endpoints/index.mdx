# Ultimate AI Connector ya Malo Omaliza Ogwirizana {#ultimate-ai-connector-for-compatible-endpoints}

Ultimate AI Connector ya Malo Omaliza Ogwirizana imaphatikiza ma API a opereka AI ogwirizana ndi OpenAI ndi netiweki yanu ya WordPress multisite. Konzani opereka AI angapo — aliyense ali ndi endpoint URL yake, API key yake, ndi model yake yokhazikika — ndipo tumizani zopempha ndi kubwerera kwadzidzidzi pamene wopereka sakugwira ntchito.

## Zinthu Zofunika {#key-features}

- **Thandizo la opereka ambiri**: Onjezani makadi angapo a opereka AI, lililonse lili ndi endpoint URL yake, API key yake, ndi model yake yokhazikika
- **Kukoka kuti musinthe dongosolo**: Konzani opereka malinga ndi kufunikira pogwiritsa ntchito chogwirira pa khadi lililonse; zopempha zimagwiritsa ntchito opereka motsatira dongosolo
- **Njira yobwerera yokha**: Ngati wopereka wamkulu sakugwira ntchito, zopempha zimabwerera zokha kwa wopereka wotsatira amene wakonzedwa
- **Kukonzekera kwa wopereka aliyense**: Wopereka aliyense amakonzedwa payekha ndi zidziwitso zake ndi kusankha kwa model
- **UI ya zokonda yozikidwa pa makadi**: Yomangidwa pa zigawo za Card zokhazikika za WordPress 6.9+ (CardBody, CardHeader, CardDivider)
- **Kutseguka kokha mukawonjezera**: Makadi atsopano a opereka amatseguka okha kuti mutha kudzaza tsatanetsatane nthawi yomweyo
- **Kuchotsa cache ya script**: Ma script a opereka amachotsedwa mu cache pakusintha plugin kuti zitsimikizire kuti mtundu waposachedwa ukutsitsidwa

## Zofunikira {#requirements}

- WordPress 5.3 kapena kupitirira
- PHP 7.4 kapena kupitirira
- Ultimate Multisite plugin (yogwira ntchito)
- Malo omaliza a API ogwirizana ndi OpenAI (odzichititsa nokha kapena a gulu lachitatu)

## Kuyika {#installation}

1. Kwezani mafayilo a addon ku chikwatu chanu cha `/wp-content/plugins/`
2. Yambitsani plugin kudzera mu menyu ya 'Plugins' mu WordPress
3. Pitani ku **Ultimate Multisite → AI Connector** kuti mukonze addon

## Kukonzekera {#configuration}

### Kukhazikitsa Opereka Ambiri {#multi-provider-setup}

Chophimba chokonzera chimagwiritsa ntchito kapangidwe kozika pa makadi komwe khadi lililonse limayimira wopereka AI m'modzi. Mutha kuwonjezera opereka ambiri momwe mukufunira.

#### Kuwonjezera Wopereka {#adding-a-provider}

1. Pitani ku **Ultimate Multisite → AI Connector** mu oyang'anira netiweki
2. Dinani **Onjezani Wopereka**
3. Khadi latsopano la wopereka limawoneka ndipo limatseguka lokha
4. Dzazani tsatanetsatane wa wopereka:
   - **Dzina la Wopereka** — chizindikiro chodziwira wopereka uyu (mwachitsanzo "OpenAI", "Mistral", "Local Ollama")
   - **Endpoint URL** — URL yoyambira ya API yogwirizana ndi OpenAI (mwachitsanzo `https://api.openai.com/v1`)
   - **API Key** — API key yanu ya wopereka uyu
   - **Model Yokhazikika** — chizindikiritso cha model chogwiritsidwa ntchito mwachisawawa kwa wopereka uyu (mwachitsanzo `gpt-4o`, `mistral-small`)
5. Dinani **Sungani Zokonda**

#### Kuchotsa Wopereka {#removing-a-provider}

Dinani batani la **Chotsani** pa khadi lililonse la wopereka kuti muchotse. Sungani zokonda kuti kusinthaku kuyambe kugwira ntchito.

#### Kusintha Dongosolo la Opereka {#reordering-providers}

Kokani chizindikiro chogwirira (⣿) kumanzere kwa khadi lililonse la wopereka kuti musinthe dongosolo lake. Dongosololi limatsimikizira kufunikira kwa kubwerera: zopempha zimatumizidwa kwa wopereka woyamba ndipo zimabwerera kwa opereka otsatira ngati pempho lalephera.

### Njira Yobwerera {#fallback-routing}

Pamene pempho kwa wopereka wamkulu lalephera — chifukwa cha kutha kwa nthawi, malire a kuchuluka kwa zopempha, kapena kulephera kwa ntchito — cholumikizira chimayesanso chokha ndi wopereka wotsatira pa mndandanda. Izi zimapitiriza kutsika pa mndandanda mpaka wopereka wapambana kapena opereka onse ayesedwa.

| Chochitika | Zomwe zimachitika |
|----------|-----------|
| Wopereka wamkulu ayankha | Pempho limathandizidwa mwachibadwa |
| Wopereka wamkulu alephera | Pempho limayesedwanso ndi wopereka wotsatira |
| Opereka onse alephera | Cholakwika chimabwezedwa kwa woyitana |

Kubwerera sikuwoneka kwa plugin kapena theme yomwe ikuyitana — amalandira mwina yankho lopambana kapena cholakwika chomaliza, mosasamala kanthu za kuchuluka kwa opereka omwe ayesedwa.

### Buku la Kukonzekera Wopereka {#provider-configuration-reference}

| Gawo | Chofunikira | Kufotokozera |
|-------|----------|-------------|
| Dzina la Wopereka | Inde | Chizindikiro chowonetsera cha wopereka uyu |
| Endpoint URL | Inde | URL yoyambira ya API yogwirizana ndi OpenAI |
| API Key | Inde | Kiyi yotsimikizira ya wopereka uyu |
| Model Yokhazikika | Inde | Chizindikiritso cha model chogwiritsidwa ntchito pamene palibe model yomwe yatchulidwa mu pempho |

## Kuthetsa Mavuto {#troubleshooting}

### Palibe opereka omwe akuwonetsedwa {#no-providers-are-listed}

Ngati mndandanda wa opereka uli wopanda kanthu mutasunga, tsimikizirani kuti:

- Mudadina **Sungani Zokonda** mutawonjezera opereka
- Msakatuli wanu sanatseke tsamba la zokonda kuti lisasunge (yang'anani console ya msakatuli kuti muwone zolakwika za JavaScript)
- Plugin ikugwira ntchito ndipo muli pa chophimba cha oyang'anira netiweki

### Zopempha zikulephera kwa opereka onse {#requests-are-failing-for-all-providers}

- Tsimikizirani kuti endpoint URL ya wopereka aliyense ikufikika kuchokera pa seva yanu (osati kuchokera pa msakatuli wanu wokha)
- Onani kuti API key iliyonse ndi yolondola ndipo sinathe ntchito
- Unikaninso malire a kuchuluka kwa zopempha ndi momwe quota ya woperekayo ilili mu dashboard yawo
- Yambitsani kulemba kwa debug kwa WordPress (`WP_DEBUG_LOG`) kuti mugwire mauthenga atsatanetsatane a zolakwika kuchokera ku cholumikizira

### Wopereka sakugwiritsidwa ntchito pa kubwerera {#a-provider-is-not-being-used-for-fallback}

Kukoka kuti musinthe dongosolo kumasintha dongosolo la kubwerera. Tsimikizirani kuti wopereka yemwe mukuyembekezera kuti alandire zopempha zobwerera ali pamalo oyenera pa mndandanda ndipo endpoint URL yake ndi API key yake ndi zolondola.

### Khadi latsopano la wopereka silitseguka lokha {#new-provider-card-does-not-expand-automatically}

Kutseguka kokha kumafuna kuti JavaScript ikhale yoyatsidwa mu msakatuli wanu. Ngati khadi silitseguka mukadina, yang'anani console ya msakatuli kuti muwone zolakwika. Zomwe zili pa khadi zimapezeka nthawi zonse podina mutu wa khadi ngakhale kutseguka kokha sikuyambika.
