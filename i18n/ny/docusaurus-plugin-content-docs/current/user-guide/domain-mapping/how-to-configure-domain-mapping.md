## Kuti ya Domain Mapping (Domain Mapping)

Kuti ya domain mapping ndi chiyani?

Mwachidule, domain mapping ndi njira yoperekera kuti m'chonde wanu woyenera (website) uliwe ndi dziko lalikulu la zinthu zina. Mwachitsanzo, muli ndi website imodzi, koma mukuona kuti ndipo muli ndi website imodzi kale. Domain mapping imaperekera kuyesawo (mapathula) domain yanu ya chisoni (domain name) kuti ite ku dziko lalikulu la website lanu.

Mwachitsanzo:
*   Muli ndi domain yanu yomwe ndi `www.example.com`.
*   Muli ndi website imodzi lomwe limapangidwa m'server imodzi, koma mukuona kuti ndipo muli ndi website imodzi lina lomwe limapangidwa m'server lina (mwachitsanzo, `blog.example.com`).

Domain mapping imaperekera kuyesawo domain yanu ya chisoni kuti ite ku dziko lalikulu la website lanu, ndipo inalimbikitsa kuti zinthu zonse zimene zikuyenera zikupitirira m'dziko lalikulu.

**Ndi kutha kodi? (Why do we need it?)**

1.  **Kuperekera Mwayi (Traffic Management):** Mumapeza kuti ndipo muli ndi website imodzi, koma mukuona kuti ndipo muli ndi website imodzi lina lomwe limapangidwa m'server lina. Domain mapping imaperekera kuyesawo domain yanu ya chisoni kuti ite ku dziko lalikulu la website lanu, ndipo inalimbikitsa kuti zinthu zonse zimene zikupitirira m'dziko lalikulu.
2.  **Kukonza Zomwe Zili M'Chonde (Organizing Content):** Mumapeza kuti ndipo muli ndi website imodzi, koma mukuona kuti ndipo muli ndi website imodzi lina lomwe limapangidwa m'server lina. Domain mapping imaperekera kuyesawo domain yanu ya chisoni kuti ite ku dziko lalikulu la website lanu, ndipo inalimbikitsa kuti zinthu zonse zimene zikupitirira m'dziko lalikulu.
3.  **Kukonza Zomwe Zili M'Chonde (SEO):** Kuti Google ndi zingathe kuona ndi kuyenera website yanu, domain mapping imaperekera kuyesawo domain yanu ya chisoni kuti ite ku dziko lalikulu la website lanu.

Mwachidule, domain mapping ndi njira yoperekera kuti muli ndi ule woyenera (website) wanu wosangalatsa ndipo mukuona kuti ndipo muli ndi ule woyenera wosangalatsa imodzi lomwe limapangidwa m'server lina.

Kodiwa koti, domain mapping ndi ule wopereka ndi Ultimate Multisite kuti upatsa mawuwo omwe muli ndi domain yomwe yomwe, ndipo kumapatsa lonjezo lilo ndi site lilo m'mwayi ndi domain yomwe yomwe.

### Kufunir mapano domain mapping pa mtindo wanu wa Ultimate Multisite

Kufunira domain mapping kuphuna kulengeza kuchokera m'ndindo wanu. Kuti kuliye, Ultimate Multisite kumapulula mwayi wanu wosangalala kuti muwoneze kufufuza mwayi wanu bwino.

Pindi muli m'pela Ultimate Multisite, m'pela wizard imakopya ndipo imalimbikira **sunrise.php** ku folder iliyonseka. Imakopya imalimbikira mpaka muperekeza mpaka mwayi ndi luso liliwoneka.

### Kuti kuti kuti m'ndau ya domain DNS imayamba bwino

Kodi muli ndi zinthu zomwe mukuwoneka, ndiyenera kukhala m'ndau (DNS) ya domain yanu imapangidwa bwino. Izi ndi zofunika kwambiri kuti anthu wosangalala amwezi angathe kupeza website yanu.

Mwachitsanzo, muli monga mukuwona kuti:
*   **A Record:** Ili ndi kukhazikitsa kuti domain yanu ikupitira ku server imodzi.
*   **MX Record:** Ili ndi kukhazikitsa kuti zopangidwa za email (email) zikupitira welo wosangalala.
*   **CNAME Record:** Ili ndi kukhazikitsa kuti domain yanu ikupitira ku server imodzi omwe ali ndi dzina la chinthu china.

Muli monga mukuwona kuti:
1.  **Check DNS Settings Yanu:** Muli monga mukuwona kuti m'ndau ya domain yanu imayamba bwino? Muli monga mukuwona kuti muli ndi zofunika kuti muwone ndipo muli ndi mafunso omwe muli ndi ulendo wosangalala.
2.  **Update Records:** Ngati muli ndi records osiyanasiyana, muli ndi chifukwa kuti muli ndi chidziwitso (update) lina lomwe muli ndi kukhazikitsa.

### Momwe Kuti Domain Iye Yowonjezo (About Domain Syncing)

Kukonza Domain Syncing ndi njira imene Ultimate Multisite ikatsera dzina la domain lalikulu (custom domain) ku m'changu wanu wa hosting monga domain yomwe yomwe kuti mapeto a domain (domain mapping) akuleke.

Kutete koti koti kuli m'moyo (domain syncing) kutete kutete m'moyo ngati wopemba wanu umene uli na integration ndi yoteteza Ultimate Multisite domain mapping feature. Kuti nthawi ya izi, wopemba wanu ndi _Runcloud_, _Closte_, _WP Engine_, _Gridpane_, _WPMU Dev_, Cloudways,_ ndipo _Cpanel_.

Pema muli pansi ya host-provider integration ikulimbikiza Ultimate Multisite kuti yendetsa ndalama za DNS kapena subdomain zilizachitidwa kwa sites zilizachitidwa nthawi zina. Ngati pano pano sipali wopereka (listening) lonselo lilo, job la m'ndalama (background job) liliyendetsedwa kuti muchezi wa queue osavomereze kuona entries zomwe siyo ndi chinthu chokhazikika (no-op). Zomwe zikufotokozera DNS ndi SSL kwa domains zomwe zikuperekedwa kumfunika kulemavuta m'ndalama ya domain yomwe yomwe.

Kodiya mukufuna kukwera (activate) integration iyi pa Ultimate Multisite settings m'maka **Integration** tab.
