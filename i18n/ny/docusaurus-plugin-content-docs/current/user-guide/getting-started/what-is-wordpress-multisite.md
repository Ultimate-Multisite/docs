# Kodi WordPress Multisite ndi chiyani?

M'moyo wamene, WordPress imapereka chinthu chamene muliwo chiyenera chamene chimene chitsogolera kuti "Multisite". Izi zikuyenera ku mpaka 2010 pamene WordPress 3.0 iliperekedwa. Komwe m'maka, imapereka zambiri kuti zimene nkhani zake zikugwira zinthu zatsanitsa zatsanitsa ndi kukhazikitsa kukhala kwakukulu.

M'moyo wamene, WordPress multisite yomwe mungupatsedwe kuyesa ndi chonde: Njira imodzi ya Onchero (University) imapereka installation imodzi la WordPress koma m'maka, m'maka imodzi limene limapereka site lina la WordPress.

## The Super Admin

Single-site WordPress installations zimapangira kuti mupereke nthawi yomwe yosonyeza wosonyeza (users) ndi malo omwe ali ndi roles lina ndi ulemu lina.

M'moyo wopanga Multisite, muli ndi aina yosonyeza yotchedwa: **the super admin** – ndipo muli ndi panel ya admin yotchedwa: **the network admin panel**.

Kodi monga m'chiyembekezero, the super admin ali ndi mphamvu zambiri pa network yanu, kuti akwoneza subsites onse, plugins, themes, ndi zinthu zonse!

Poyamba momwe mupanga single-site WordPress wanu kuti ndi multisite, the original admin ya single site idzakhala ndi super admin kukhala m'moyo.

Plugins ndi themes zimenezi zikupangira kapena kuchepa zimenezi zikupangira m'network admin panel ndi a super admins. Subsite admins angayesera kuti zikuyendetsa plugins kapena themes ngati ali, ngati osonyeza wosonyeza (super admin) ya network imapangira plugin, zomwe zimapanga kuti zimakhale ali kwa subsites onse m'moyo.

_Note: muliwiri, muliwiri wosomwe, ndikukumbutsa kuti kuita wone m'muliwiri wanu ndi kuyesa munthu wanu ndi kupereka iwo super admin status kumphatikizira wosoweka wanu m'muliwiri wanu. Mwachitsanzo, ena omwe ndi super admins angathe kuondoza super admin status yanu, ndipo izi zimapangitsa kuti muli wosoweka wanu woyamba wanu wosowekera wanu. Kuti uthangano Ultimate Multisite wamene anayesera kuti akulekeze kufunsa omwe ena a super admin angathe kuchita chiyani, tili ndi add-on lomwe limakwanira kukhala Support Agents. Add-on ili limakukwanira kupeza m'muliwiri wosoweka wina – agent – ndi kupereka iwo permissions zomwe mungikufunje kuti muperekeza ntchito zawo m'muliwiri wanu._

## Kodi zimene zikuyenera m'muliwiri wosomwe ndizo zomwe sikuyenera

Kodi tikukumbutsa m'muliwiri wosomwe, chimodzi cha zokhudza kwambiri ndi kuti m'muliwiri onse imakugwiritsa ntchito mfundo zosonyeza (configurations), files za moyera (core files), themes, plugins, WordPress core files, ndi zina.

Komabe, kuli monga zinthu zomwe zimapereka bwino kwa m'muliwiri wosomwe wina.

- Mwachitsanzo, muli wosomwe wiri woyamba wanu muliwiri wosomwe wiri woyamba wanu wosowekera (uploads folder) lina. Komanso, zinthu zomwe anaperekedwa ndi anthu omwe ali m'muliwiri wosomwe wiri woyamba wanu sikuperekedwa pa m'muliwiri wosomwe wina.

- Muli wosomwe wiri woyamba wanu muliwiri wosomwe wiri woyamba wanu ndi chitsanzo cha admin lina, ndipo mungakuyendetsa (activate) kapena kukuyendetsa (deactivate) plugins kapena themes ngati siye kuti zikuyenera m'muliwiri wosoweka wanu.

- Zinthu zingapo za database zimaperekedwa kwa muliwiri wosomwe wina, zomwezi ndi posts, comments, pages, settings, ndi zina.

## Ulamulira wa anthu m'muliwiri wosomwe (User management on WordPress Multisite)

Kukonzekera kwa anthu m'muliwiri wosomwe ndi chinthu chomwe chimakhala chofunikira kwambiri. Table ya users ya WordPress ndi imodzi monga momwe imaperekedwa m'muliwiri onse.

Mwachidule, izi zingathe kupangitsa mavuto monga momwe mungupange nkhani yomwe mungupange m'muliwiri wanu. Mwachitsanzo monga monga monga monga monga:

Muli muli, uthandiza WordPress multisite network uliyoona ndi subsites zomwe munthu amagwiritsa ntchito kwa mlopelo wosiyanasiyana m'mabwiri.

Muli ndi munthu woyamba wamene ali ndi malingaliro (customer) — John. Mumaliza site imodzi kwa John pa network yanu, mumapanga plugins zonse zomwe zikalazidwa, ndipo mumapanga user imodzi kwa John kuti athangane store yake.

Kuti nthawi imodzi, muli ndi munthu wosiyanasiyana — Alice. Mumaliza zinthu zina zokhudza iye komanso, ndipo pano ali ndi store imodzi pa network yanu.

John ndi Alice ndi anthu amene muli malingaliro, koma sembedwa siwanji. Komabe, ngati imodzi mwonse wamene ali mu store ya imodzi yomwe imayamba kuwoneka kwa imodzi, pali njira yomwe mungowone kuti iye imodzi ndi network imodzi la subsites.

Muli monga m'malo, John anayenera kupanga chinthu chiri cha zitsomeko (shoes) ndipo amapita kuwona zomwe zikuyenera mu store ya Alice. Pamene akuti atanthane kukhala ndi chidziwitso (purchase), ali ndi mufotokozi wochitika kuti "email imodzi yomwe imayendetsedwa" (email already in use) — zimene zimakhudza kwambiri, koma John amakhulupirira kuti ndi nthawi yoyamba yake yomwe akuti ali ndi Alice.

Chiyembekezo chiyenera kukhala kuti user ya John imodzi kwa m'malo wosiyanasiyana wose pa network. Pankati pomwe akuti atanthane kupeza account kuti athandane kupanga (checkout) pa site ya Alice, WordPress idzindikire kuti pali user imodzi ndi email imodzi yomwe imayendetsedwa ndipo idzathandiza mufotokozi wochitika.

_Kumbukira: Timakhudza kuti zimene zikuyenera kukhala zikukhudza kwambiri kuposa, chifukwa cha njira yomwe Ultimate Multisite imapereka kuti muli ndi chagera lomwe limapota zochitika za user wosiyanasiyana, kumaliza kuti muli ndi accounts zingapo zomwe zimayendetsedwa ndi email imodzi. Kukhala kwa account limodzi limayendetsedwa ndi subsite imodzi, chifukwa chidziwitso cha kuposa kimene chimakhala kotsatira kwambiri. M'malo wosiyanasiyana m'malo, John sichapereka mufotokozi wochitika ndipo adzathandiza kupanga zitsomeko hizo m'malo ndi kusintha._ Chogera limene limakuthandizira kukhala ndi chidziwitso cha kuposa, limapereka kuti muli ndi accounts zingapo zomwe zimayendetsedwa ndi email imodzi. Kukhala kwa account limodzi limayendetsedwa ndi subsite imodzi, chifukwa chidziwitso cha kuposa kimene chimakhala kotsatira kwambiri. M'malo wosiyanasiyana m'malo, John sichapereka mufotokozi wochitika ndipo adzathandiza kupanga zitsomeko hizo m'malo ndi kusintha._

Koma ndipo kuti chonde, pamene table ya users ikulimbikira, users angayendwa ndi subsites ndi admin a subsite kapena super admin, ndipo angayendera roles zosiyanasiyana pa subsites.

## Zomwe Ziyenera Kuti Muli Ndi Kufunsa (Performance considerations)

WordPress multisite imakuti ndi umodzi wogwira ntchito kwambiri pamene kukhala ndi anthu odziwo kuti imathandiza nkhani zambiri za sites. Izi yomwe mungutse ndi kuona kuti [WordPress.com](https://WordPress.com), Edublogs, ndi Campuspress ndi services zomwe zimakhala multisite, ndipo m'host imodzi imathandiza anthu zingapo zingapo.

Koma m'moyo wopereka (in theory) pano pali nkhauksa yomwe sikuti muli ndi anthu odziwo kuti muli ndi sites zingapo zomwe mungutse, m'moyo wopereka (in practice) ndalama ya sites zomwe mungutse kuyendera bwino imakutsa kwambiri kuposa m'zinthu zingapo: kutanthauza kuti sites zimakhala zotsegula (dynamic) ngati, plugins zikuwogolera subsites ndi zina.
