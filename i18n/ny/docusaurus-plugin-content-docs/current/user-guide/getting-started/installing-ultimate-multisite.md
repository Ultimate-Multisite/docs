# Kufunga Ultimate Multisite

:::note
Ili kulemba i tutorial ulemu, muli ndi WordPress Multisite wokhala wokhala ndi wokhala. Ili kuti mufunse kulowa bwanji, mufunse kuona [ile tutorial](https://www.wpbeginner.com/glossary/multisite/) ya WP Beginner.
:::

## Kufunga Plugin

Ultimate Multisite ndi yenera kwa kutayira pa [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/).

Kuchokera m'**Network Admin Dashboard**, ndi lankhula **Plugins → Add New Plugin**.

![Network Admin Add New Plugin page](/img/installation/add-new-plugin.png)

Tsimikira **"Ultimate Multisite"** (nkhani yomwe imapereka kuti ikukhale m'mwayi wolembedwa) ndipo idzakhala kuyamba. Kuti, chonde thandiza ndi **Install Now**.

![Search results showing Ultimate Multisite](/img/installation/search-ultimate-multisite.png)

Poyamba imaphatikira, thandiza ndi **Network Activate** kuti mupatike plugin wopereka kwa mtundu wanu wosangalatsa wosangalatsa.

![Plugin installed with Network Activate button](/img/installation/plugin-installed.png)

Poyamba imaphatikira, mudzakhala m'mwayi woperekera (Setup Wizard).

![Plugin activated and redirected to wizard](/img/installation/plugin-activated.png)

## Mwayi Woperekera (Setup Wizard)

Mwayi Woperekera uthandizira kuti muphatike Ultimate Multisite m'miniti 10 kale.

### Kukhulupirika

Thandiza ndi **Get Started** kuti mupereke.

![Setup Wizard welcome screen](/img/installation/wizard-welcome.png)

### Zomwe Zikuyamba Kufunga (Pre-install Checks)

Ili lino limaperekera mawu awo pa mfundo za system yanu ndi WordPress kuti mupereke kuti imapereka zolembedwa ndi Ultimate Multisite. Ngati zonse zimapereka bwino, thandiza ndi **Go to the Next Step**.

![Pre-install checks showing system requirements](/img/installation/wizard-pre-install-checks.png)

:::note Button ya "Network Activate" (v2.6.1+)
Muli Ultimate Multisite yali wamutsa, koma **sikuya kuli** — monga muli mwapereka **Activate** (single-site) muli monga **Network Activate** ku m'moyo wa plugins — muli ya Pre-install Checks imakumbutsa izi ndipo imalipira button ya **Network Activate**.

Muli muliyeza **Network Activate**, imalipira plugin yali mwapereka wanuwo wamutsa wanuwo wanuwo wanuwo. Muli muliyeza, wizard imakwera monga muli wa installation. Muli seya state ya activation muliyeza.
:::

### Installation (Kusangira)

Installer imalipira database tables zokufuna kuli ndipo imalipira file `sunrise.php` yali Ultimate Multisite yali mwapereka kuti yali mwapereka. Muli ya **Install** muliyeza kupita monga muli wa installation.

![Installation step showing database tables and sunrise.php](/img/installation/wizard-installation.png)

### Your Company (Wanuwo Wanuwo Wanuwo)

Muli yali mwapereka info ya wanuwo wanuwo wanuwo ndipo muli set your default currency. Info iyi imalipira monga muli wa platform yali WaaS yali wanuwo wanuwo wanuwo. Muli ya **Continue** muliyeza kupita muliyeza.

![Your Company configuration step](/img/installation/wizard-your-company.png)

### Default Content (Mawu Oya Oya)

Iyi step imalipira kuti muli mwapereka templates, products, ndipo monga muli content ya mbiri. Iyi ndi njira yomwe imalipira kuti muli woyenera features za Ultimate Multisite. Muli ya **Install** muliyeza kupanga default content, kapena muliyeza kupita step iyi muliyeza muliyeza.

![Default content installation step](/img/installation/wizard-default-content.png)

### Recommended Plugins (Plugins Zokunja)

Muli mwapereka plugins zomwe zikufuna kuli. Muli ya **Install** muliyeza kupanga kuti muliyeza kapena muliyeza kupita.

![Recommended plugins step](/img/installation/wizard-recommended-plugins.png)

### Ready! (Muli Woyenera!)

Kodi! Kuli ya Ultimate Multisite yanu yaliyendera. Muliye kuyendera ku Website as a Service platform monga **Network Admin Dashboard**.

![Setup complete - Ready screen](/img/installation/wizard-ready.png)

![Network Admin Dashboard with Ultimate Multisite active](/img/installation/network-dashboard.png)

Muliye kulembedwa!
