# Usimu wa Site Specification Skill {#site-specification-skill}

Usimu wa **Site Specification skill** ndi njira yomwe imapulogera (structured approach) yomwe imapulogera monga zolemba za sita lanu, wopereka lanu, ndi chidziwitso cha brand lanu. Kufotokoza limodzi limalimbikira m'memory ya **site_brief** yanu, yomwe imapulogera amagwira ntchito m'moyo wosiyanasiyana kuti apereke chithandizo chomwe chimakhulupirira ndi zomwe zili pansi.

## Site Specification ndi chiyani? {#what-is-site-specification}

Site Specification ndi njira yoperekera:

- **Chikhulupiriro cha sita**: Sitayopereka sita lanu limakwera ndi chifukwa chake
- **Wopereka wopereka (Target audience)**: Ndiwo amene amapita pa sita lanu ndipo ali ndi chiyani chomwe ali m'malo
- **Chidziwitso cha brand**: Mwayi wanu, mwayi wanu, ndi mtindo wanu wa zithunzi
- **Zomwe zimeneza bizinesi (Business goals)**: Zomwe zimeneza kuti sita lanu likupereka chiyani
- **Mwayi wa zinthu za content**: Sitayoperekera sita lanu limapereka bwanji

Site specification imapereka **site_brief** yanu, memory yomwe imapulogera amagwira ntchito kuti mwayi wanu wosiyanasiyana amveza zomwe sita lanu ndi chiyani.

## Kodi Muthoza Kuthandizira Site Specification? {#why-use-site-specification}

### Kukhulupirira M'moyo Wosiyanasiyana (Consistency Across Sessions) {#consistency-across-sessions}

Ndikudziwa site_brief, muli ndi kuyenera kuphunzitsa chikhulupiriro cha sita lanu nthawi zina m'moyo wosiyanasiyana. Ndiwo, imapulogera amveza m'malo:

- Zomwe zimeneza kuti sita lanu likupereka ndi wopereka wanu
- Mwayi wanu wa brand ndi mwayi wanu
- Mwayi wa zinthu za content
- Zomwe zimeneza bizinesi yanu

### Kukhala ndi Chithandizo Chomwe Chimeneza (Better Recommendations) {#better-recommendations}

Imapulogera amagwira ntchito site_brief yanu kuti:

- Yendetsa chinthu chomwe chimakwera ndi chikhulupiriro cha sita lanu
- Yendetsa mwayi wa zinthu za content zomwe zimeneza zomwe zimeneza
- Yendetsa mtindo wosiyanasiyana omwe umeneza brand lanu
- Kukhala ndi chidziwitso chomwe sikupereka

### Kukhala Ndi M'moyo Wosiyanasiyana (Faster Onboarding) {#faster-onboarding}

Imapulogera atsopano (or agents m'sessions zanu zifukira) imapereka mwayi wanu wosiyanasiyana kwa kuthandizira mwachangu m'site_brief yanu m'malo m'kufunsa mavuto.

## Kukhudza Site Specification {#initiating-site-specification}

## Kufatira ndi Site_brief Yanu

### Onesha Site_brief Yanu

Mati wamukondani:

```
"Onesha site brief yanu"
```

kapena

```
"Kodi mukuona za sitiyanu?"
```

Wamukondani mudzindikira zokhudza zanu zomwe mumakonda.

### Kufotokozera Site_brief Yanu

Ngati sitiyanu lalikupitilira, muli ndi ulemu wanu:

```
"Fotokozera site brief yanu: tili tikufuna makatundu kwa B2B"
```

kapena

```
"Redha site specification yanu"
```

Izi zimakukonza upatanirano wa Site Specification ndi zomwe muli ndi zinthu zina monga chifukwa cha kuchokera.

## Kodi Agents Imakonda Site_brief Bwera?

### Zomwe Zikufotokozera kwa Malingaliro (Design Recommendations)

Pamene mukuona kuti muli ndi kusintha kwa design, agents imakumbutsa site_brief yanu:

```
Wono: "Redesign homepage ya sitiyanu"
Agenti: "Monga momwe muli ndi site brief yanu, muli tikufuna akazi omwe ali ndi zaka 25-45 ndi moyo wopereka. Ndipo ndikukonza design zomwe zimakumbutsa makatundu anu kwa mkati ndi mfundo za kusintha."
```

### Zomwe Zikufotokozera kwa Malingaliro (Feature Suggestions)

Agents zimakukonza zinthu zomwe zimakubwera ndi zomwe muli ndi zolemba:

```
Wono: "Zomwe zimakonda kuti ndipereke?"
Agenti: "Monga sitiyanu la e-commerce lomwe limakufuna kukhala ndi kulumbitsa kwa makatundu, ndikukonza: loyalty program, reviews za makatundu, email newsletter, ndi zolemba za makatundu."
```

### Mtsogoleri wa Zomwe Zikufotokozera (Content Structure)

Agents zimakuthandiza kupeza momwe muli ndi kusintha kwa zomwe mumakonda:

Kondi: "Ndi ndipale nji m'bale blog wangu?"
Mwanditsi: "Chifukwa mukati ndi product catalog ndi blog, ndikufotokoza kuti muli ndi chondeledwe kukhazikitsa posts pa category ya product ndikugwira 'styling guides' zomwe zimapereka zinthu zingapo m'maka."

## Zomwe Zili Ndi Best Practices (Zomwe Zili Ndi Zomwe Zili Zina Kukhala Zomwe Zili Zina)

### Kodi Muli ndi Zomwe Zili Zina? (Be Specific)

Mmodzi mukufotokoza "moyo wosavuta," muli ndi kusonyeza kuti ndiyo ndi ndiyo:

- ✓ "Wamukwanakwamba omwamba 25-45, omwe ali ndi chidziwitso cha fashion yomwe ndi yomwe yomwe"
- ✗ "Onse"

### Kukhazikitsa M'maka (Update Regularly)

Poyenera m'maka wanu, kudzipereka site_brief wanu:

- Pamene muli ndi chidziwitso cha womwe wosavuta
- Pamene muli ndi zinthu zatsopano za product
- Pamene chidziwitso cha brand yanu kukhala lina
- Pamene zolemba za bizinesi yanu zimakhala lina

### Kutsogolera Mankhwala Osame (Use Consistent Terminology)

Gwiritsa ntchira mankhwala osamukwaniritsa m'maka:

- ✓ Ndikuti "sustainable jewelry" (osati "eco-friendly jewelry" ndi "green products")
- ✓ Ndi kutsegula chidziwitso cha womwe wanu m'njira yomwe yomwe

### Kukhala ndi Zomwe Zili M'maka (Include Context)

Pereka zinthu zomwe zimapereka kuti agents kuti muli ndi chondeledwe za zolemba lanu:

- "Tili tikufunsa anthu omwamba omwe amakuti malo ndi yomwe yomwe kuposa"
- "Moyo wathu ndi wosavuta m'maka ndipo tikufuna njira zomwe zimakhala ndi chisonkhira cha m'maka"
- "Tili ndi startup yomwe imapereka zinthu, chifukwa pano tikufuna njira zomwe zimapereka kukhala ndi chidziwitso cha m'maka"

## Mwayi ndi Onboarding ya Theme Builder

Skill ya Site Specification ikuti m'maka wopereka (Theme Builder onboarding flow). Pamene muli ndi onboarding, site_brief wanu kumaliza kwawo ndi zimene mumapereka.

Muli ndi ulemu wosavuta Site Specification ngati mukufuna:

- Kukhazikitsa zolemba lanu pambuyo pa kukula
- Kukhazikitsa m'maka wanu pamene site yanu ikula
- Kukhazikitsa specification yomwe ndi mfundo kwambiri poyenera Theme Builder
