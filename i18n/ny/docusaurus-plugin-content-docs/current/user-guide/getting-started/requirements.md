## WordPress Multisite

Nkhawa, tili tikukonza malo kale za WordPress Multisite.

WordPress Multisite imapereka kupeza kuti muli ndi mitundu ya sites pa chinthu chimodzi cha installation. Iyi ndi feature yomwe imapereka m'malo, koma si imapereka kwa mpaka m'malo. Mukuwona kuti muli ndi feature iyi m'malo m'WordPress ndi control panel ya web hosting yanu.

Kukonza WordPress ndi Multisite kukhala **chifukwa cha ndalama za Ultimate Multisite**. Chifukwa chake, ntchito yomwe yomwe ya Ultimate Multisite ndi kuchilengedwa kwa anthu kuti muli ndi sites omwe mukuwona ndi kupereka monga service, zimene kukhala kulimbikizidwa pa WordPress ndi Multisite.

Tili ndi **article yomwe imapereka kufufuza m'malo m'zambiri zomwe mukuwona kuti muli ndi feature ya Multisite pa installation ya WordPress.**

Poyenera kuti muli ndi Multisite pa WordPress, ndiyenera kukhala ndi install Ultimate Multisite. Kuti muli ndi install yomwe imapereka bwino Ultimate Multisite, mukuwona kuti muli ndi:

  * WordPress: v5.3+ (Kukondetsedwa: Last stable version)

  * PHP: 7.4.x (Chifukwa cha kupeza > 8.0 kuli kuchokera)

  * MySQL: v5+ (Kukondetsedwa: 5.6, monga momwe m'malo wanu wa hosting si imapereka 8.0)

Zilolo ndi zowonjezera zomwe zikufunika kuti Ultimate Multisite ikwoneke.

Kuti mufunse, Ultimate Multisite mungakwanirere kutumikira PHP 8.0 koma tikufunse kuti muli ndi PHP 7.4.x.

Komanso, tikufunse kuti muli ndi kuyesa ku **main domains** (zomwe zikuyenera), osati subdomains kapena subdirectories. Zikhoza kutumikira pa subdomains/subdirectories koma mungone kukhala ndi mavuto kuchokera.

Poyenera chitsanzo chofunikira momwe mungukwanire Ultimate Multisite, **kudziwitsa pano**.
