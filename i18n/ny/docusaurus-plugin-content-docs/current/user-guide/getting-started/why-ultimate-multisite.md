# Ndiye Ultimate Multisite

Muli muli wamukira ilo ndalama, muli ndi chonde kuti muli ndi mavuto omwe plugin yathu yotsogolera kuti muli ndi chidziwitso lomaliza. Tili ndi chonde kuti muli ndi zokhudza zomwe mukuwona m'moyo wopereka.

## Kodi Ultimate Multisite ndi chiyani? {#what-is-ultimate-multisite}

Ultimate Multisite ndi plugin ya WordPress Multisite yomwe imapereka kuwona kwa wamukondedwa WaaS (Website as a Service). ndi plugin yathu, muli ndi njira yomwe mungupereke mbiri zoshipulira (subscription tiers) ndipo wamukondedwa angathe kulipira mlankhuso woperekera kuti muli ndi site yomwe imaperekedwa m'moyo woperekera wanu.

Ultimate Multisite imapereka njira yomwe mungupereke malipiro m'njira zoshipulira (gateways) osiyanasiyana (mwachitsanzo, WooCommerce kapena Stripe), kupanga site templates kuti zimapangire moyo wanu woyenera kwambiri pamene wamukondedwa watsatira m'moyo wanu, kupanga domains, ndipo kusonetsedwa zolemba za registration (registration forms) kuti mupereke mfundo yomwe imapangidwa kwa wamukondedwa wanu.

Ndipo izi ndi chifukwa cha m'moyo woperekera, komwe mungupereke funksheni zina monga add-ons ndi webhooks.

## Zinthu zina {#other-features}

Ultimate Multisite imapereka:

  * Kukhazikitsa domains (Domain Mapping) ndi wpemera woperekera WordPress: Ultimate Multisite imapangidwa kuti imapeza domains zomwe zimapangidwa m'moyo woperekera wa WP Engine, Cloudways, cPanel, RunCloud, Closte, ndi zina pamene domain yatsopano imapezedwa, chifukwa imapereka zinthu zimene muli ndi kuyenera kulipira. M'moyo woperekera, timapereka kuti muli ndi kusintha SSL certificate kwa inu.

  * Webhooks: Ultimate Multisite ndi ndi system ya Webhook yomwe imapereka njira yomwe mungupereke kuwunika (integrate) network yanu ndi zinthu zingapo. Timapereka kuyenera kulipira m'moyo woperekera ndi Zapier, komwe imapereka network yanu kuti ipeze apps ndi services zambiri zomwe zili pa platform yake.

* Admin Pages Zomwezi Zomakula Kuti Mndandawire Wanu: Ultimate Multisite uma na add-on woyamba wamba, Admin Pages Creator, womwe umukono woyambatsa kuti mupereke Admin Pages zosonyeza kwambiri kwa mndandawire wanu monga Elementor, Beaver Builder, ndi Brizy templates.

* M'malo mwina wa Themes Ziwiri Kuti Mumakula Mwayi Wopangidwa (Dashboard Experience) kwa mndandawire wanu: Check Material WP ndi PRO.

Ndi zambiri zokunena kwambiri:

* Kuthandiza Template Overrides: Na Ultimate Multisite, muli ndi ulemu wosonyeza kuti mupereke template HTML zomwe zimapangidwa kwa mpaka wopanga (signup flow) ndi zinthu zina monga wanu osati mukukumbutsa kuti mukucheza files za core plugin.

* Search and Replace API: Muli ndi ulemu womwe mumakula site templates ndi placeholders monga {{site_name}} ndipo mukuwunika Search and Replace API kuti mupereke zinthu zomwe wanu amapereka pansi pa mpaka wopanga (right after sign-up), monga m'malo.

## Zomwe Mungathe Kuti Muli ndi Zomwe Zikuyenera (Use cases) {#use-cases}

Ndi nthawi yomwe tikuyenera kuona zomwe mungathandizire Ultimate Multisite.

### Case 1: Agency {#case-1-an-agency}

Kukonzekera kwa agency kumayamba ndi kupanga websites ndi zinthu monga hosting kapena marketing kuti zimeneledwe monga zinthu zina.

Poyamba, Ultimate Multisite imapereka mwayi wopangidwa kwambiri m'malo wopereka ndi kuwongolera websites ambiri pa platform imodzi. Kuti mndandawire wanu amakula designs monga GeneratePress, Astra, OceanWP, kapena zina, mungayendetsedwe ndi ulemu wa Ultimate Multisite kuti mumapereke themes zimeneledwe kwa mpaka wopanga (new site) monga m'malo.

Kuthandiza kukhulupirira plugins ndi themes kumakhala m'malo monga m'malo woyamba, monga Ultimate Multisite imapereka malo kuti plugins ndi themes zikuyenera kapena zimeneledwe monga m'malo wopanga (activation state) pamene zimapangidwa kwa mpaka wopanga.

Agencies zikomo chifundo chokuti Ultimate Multisite chimapera kwa agensi kuti mupereke chisonkhwe chokuti zimenezi - kukhazikitsa website zosonyeza.

### Case 2: Mupereka Wopereka (Niche Provider) {#case-2-niche-provider}

Kuna chifundo chokuti, "Mefunsa chilichonse chimodzi ndi kulipirira bwino." Kwa anthu ambiri omwe ali ndi chitsanzo cha sikwambiri, izi zimenezi kukhala kupanga zochitika kapena zosiyanasiyana zomwe zili m'moyo woyenera.

Ngati muli ndi kufunsa kulipirira m'nkhoto ya sites zosonyeza, muli kuyenera kupereka zosiyanasiyana kwa mfundo yomwe zimenezi zina. Kodi site templates (zithunzi za sitete) zingathamulirira?

Chimenezo cha Ultimate Multisite chokuti template site ndi chimenezo chomwe theme imaperekedwa ndikuyenera, plugins zosonyeza zimaperekedwa ndikuyenera, ndipo m'moyo woyenera (sample posts) kapena pages zikuyenera zikuyenera. Pamene m'moyo woyenera imapanga site lili ndi chinthu cha template, m'moyo ndi zinthu za template zimaperekedwa ku site yomwe imapanga kwa nthawi yomwe imapanga.

Poyenera wopereka sites ndi zosiyanasiyana ndi zosonyeza, izi zimapereka chifundo chokuti muli ndi ulemu wosonyeza kuti mupanga site lili likuyenera kuyamba ndi plugins ndi design yomwe imenezi. M'moyo woyenera kumenezi kulipirira chinthu chofunika kwa kukhazikitsa ntchito.

### Case 3: WordPress Web Hosting {#case-3-wordpress-web-hosting}

Kuna mwayi ambiri wokupereka WordPress sites koma osati ndi luso kwambiri kupereka web space kwa wopereka ndi WordPress yomwe imaperekedwa m'moyo woyenera. Izi ndikuphunzira chifukwa kukhala kulemba zinthu zingali zikuyenera kuyenera kuti mupereke ntchito yomwe ndi yenera.

Ultimate Multisite imapera m'mwayi wopereka kwa kupereka WordPress sites monga ndimenezi yomwe imaperekedwa. M'moyo wopereka umapereka zinthu zomwe zimaperekedwa kuti mupereke hudugulira (subscription services), kulipira, checkout forms, discount vouchers, ndi kuphunzitsa kwa wopereka.

Mphamvu pa ntchito zosangalatsa koti kuinstalira, kukonza, ndi kuteteza WordPress Multisite m'mwayi wopambana, Ultimate Multisite imapangitsa kuti administrators a network amakonda kufikirira zinthu zomwe zimakhudzidwa ndi ntchito kapena niche yawo, monga mwachitsanzo, kupanga tiers za zinthu, kuphatikiza, ndi mafunso a ntchito.

Pamene developers ali ndi chidziwitso cha kukhala ndi Ultimate Multisite, ndipo mfundo imeneyi imapereka RESTful API yomwe yomwe yomwe yomwe (API) ndi Webhooks kuti mufotokozo wa zinthu zimene.
