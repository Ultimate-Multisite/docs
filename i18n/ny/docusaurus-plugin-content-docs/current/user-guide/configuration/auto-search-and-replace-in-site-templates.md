# Auto Search and Replace mu Site Templates (v2) {#auto-search-and-replace-in-site-templates-v2}

_**Iziya izi ndi WP UItimo version 2.x.**_

Modziwo monga momwe Ultimate Multisite ndi chinthu chokuti mumagwira mawu, miyambo, ndipo m'malo omwe muli kukhulupirira (select fields) ku registration form. Pamene tikapereka data imodzi, tingathe kutsegula kuti tikatseza malo omwe muli kukhulupirira m'malo monga momwe muli kukhulupirira. Kodi site yomwe yachitidwa (published) kumene Ultimate Multisite idzathandiza kuchotsa placeholders ndi mawu osati m'malo omwe muli kukhulupirira ndi zomwe mumagwera m'registration.

Mwachitsanzo, mungatseze site yomwe muli kukhulupirira ndi placeholders. Placeholders mungayendetsedwe m'malo omwe muli kukhulupirira - {{placeholder_name}}.

Kuti nthawi imodzi, mungapeza field ya registration yomwe ikuti imapereke data imodzi.

Kuti Ultimate Multisite idzathandiza kuchotsa placeholders ndi data yomwe wamene amagwera.
