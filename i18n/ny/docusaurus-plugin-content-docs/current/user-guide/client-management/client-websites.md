# Zomwezi za Website za Wopambiri wanu (v2) {#your-clients-websites-v2}

Panda m'pingo pa lanu, wopambiri wanu angamukira dashboard yanu.

Poma ya kurasa ya **Account**, pamene angamukira maluso omwe angamukira ku mpaka wa lanu, membership, domains, mapambana a plan, ndi zina zina. Koma, muli ndi ulimi wosonyeza **My Sites** page, pamene mungone ndi kudzipereka ndikuyendetsa zonse za websites zanu. Zilipo ndi kukhala luso ku My Sites pa dashboard yanu.

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

Mungone zonse za websites zanu, ndi button ya **Manage** m'moyo wopambiri wa m'moyo wanu m'moyo wanu. Ngati mungamukire, mudzakhala m'moyo wopambiri wa website lili lopambana.

Mungone kuti muli ndiyo website yomwe ndi **primary** (yomwe yomwe) kwa plans zomwe zimulemfa kuti muli ndi websites zingapo.

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

Pansi ndi chidziwitso chosonyeza mpaka pa site edit:

![Site edit full interface](/img/admin/site-edit-full.png)

Ngati kuti tikufuna kusintha website yomwe yomwe yomwe, tikufuna kuenda pa Account page m'moyo wopambiri wa menu ya **Account**.

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

Mungone kuti mungone ndi kudzipereka za wopambiri:

![Customer edit interface](/img/admin/customer-edit.png)

Poma, mungamukire **Change Default Site**.

![Change Default Site button on account page](/img/account-page/change-default-site.png)
