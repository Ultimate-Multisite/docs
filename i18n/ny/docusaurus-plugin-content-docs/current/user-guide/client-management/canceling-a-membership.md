# Kufunga Membershipu (v2)

_**KUFUNDI MUYAMBALA: Izi zikufiwe Ultimate Multisite version 2.x.**_

Kufunga membership kuwowo wa wena ndi luso kwambiri.

Ngati mwana wanu ali na mpaka kuti akufune kufunsa membership yake, adali kukhala akuyamba kupereka **Account** page yake.

Mmodzi momwe ali pano, adali kukhala akupereka ku **Danger Zone**, ndipo akupereka ku **Delete Account**.

![Account page Danger Zone with Delete Account button](/img/account-page/danger-zone.png)

Kukuti mmodzi woyamba kufunsa kuti akufune, adali kukhala akupereka **Confirm Account Deletion** option ndipo akupereka ku **Delete Account**.

![Confirm Account Deletion dialog](/img/account-page/delete-account-confirm.png)

Izi zimapereka kuti mmodzi woyamba akuyamba kupita (log out) pa site yanu ndipo kufunsa membership yake kuwoneka ku network yanu, koma kulimbikitsa site yake kuwoneka ku network yanu.

![Logged out after account deletion](/img/account-page/logged-out-after-delete.png)

![Membership canceled confirmation](/img/account-page/membership-canceled.png)

**Zofunzi:** Kodi mukuona kuti luso lili ndi kufunsa lilo, koma kukhala mwana wanu ali na membership yomwe yomwe kapena osati ali na membership yomwe yomwe kuwoneka ku network yanu.

![Customer still exists in the network after cancellation](/img/admin/customer-after-cancel.png)

![Customer with no active memberships](/img/admin/memberships-customer-empty.png)
