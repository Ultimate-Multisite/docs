# Mwayi ndikuphunzira kuti ndikayesera?

Ngati muli ndi Ultimate Multisite license, muli ndi ulimi wopereka kuwamba kwawo monga mukuphunzira nthawi yomwe mukuphunzira. Kodi mukuphunzira monga mukuphunzira monga mukuphunzira monga mukuphunzira?

<!-- Screenshot unavailable: Account page email with access link -->

Gawira imelanso yomwe muliye mukuphunzira ndi chitsanzo chanu. Kodi ndi chitsanzo chanu cha kukhulupirira (password).

<!-- Screenshot unavailable: Login page with email and password fields -->

Iyi ndi mwayi wopereka kwawo woperekera:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

Kuti mukuphunzire kuwamba kwawo, pamene muli m'menu yomwe m'malo m'm'lino, ndi penda _Renewing & Billing._

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

Gawira arrow yomwe m'm'lino kuti mukuphunzire window lomalanso. Kodi mukuphunzira chitsanzo chomwe chimayesera kuwamba kwawo (auto-renew).

Mwayi wopereka utanthauza mwayi woperekera.

Ngati mukuphunzira kuti mukuphunzire kuwamba kwawo, mwapereka button yomwe imayesera _**Cancel Renewals**_. Mmodzi m'malo mwa mopereka, mudzakhulupiridwa survey yomwe yomwe.

Kukonza! Kuwamba kwawo kumayesera monga mukuphunzira.

**Muli ndi key yomwe imayesera mpaka kuwamba kwawo kumayesera**. Ngati mukuphunzira kuti mukuphunzire kuwamba kwawo, mudzakhala ndi kuyesera m'malo.
