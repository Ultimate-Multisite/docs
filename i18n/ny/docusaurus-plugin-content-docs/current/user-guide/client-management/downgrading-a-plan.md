# Kufunsa plan (v2)

_**KUFUNSA: Izi nkhani iyi imaliza Ultimate Multisite v2.x.**_

Kufunsa plan kapena subscription ndi chilichonse chomwe anthu ena angakugwira ngati ali ndi ndalama yomwe yomwe (limited budget) kapena ali ndi chonde kuti sindikukonza zinthu zingapo kuti muli ndi subsite.

## Kodi mungufunse plan bwanji?

Anthu ena angayendetsa plan yawo m'malo woyamba kwa logging in ku admin dashboard ya subsite yawo ndipo yendera **Change** (Mwandirira) pansi pa account page yawo.

![Customer account page with Your Membership card and Change button](/img/account-page/membership-change-button.png)

Poyendera butunzi la **Change**, user/client adzakhala kuyendera ku checkout page pamene angayendera plan yomwe ali ndi chonde kuti mwandirire subscription yawo.

![Downgrade plan options page on the customer side](/img/account-page/downgrade-picker.png)

M'malo omwe tikufunsa, tikufunsa plan kuchokera **Premium** ku **Free**.

Kuti mupitire, user aliyense adzakhala kuyendera butunzi la **Complete Checkout**. Kuti nthawi yomwe imapereka ndikugwira kukhazikitsa (pending change) kwa membership. Zomwe zikufunsa zimapereka kutonja pa **billing cycle** yawo yomwe yomwe.

![Account page showing pending membership change banner](/img/account-page/pending-change.png)

### Kodi kuli ndi chiyani pamene user amakufunsa plan?

Kuti ndi zofunsa kuti kufunsa plan sikugwira kusintha monga momwe kuli pansi kwa subsite yawo.

Sikugwira kusintha template ya site monga momwe ndimene kusintha template kumaliza ndi subsite yomwe. Izi zikuyenera kuti musintha data yosiyanasiyana. Choncho, disk space, themes, plugins ndi zina zimapereka kutonja mpaka posts.

Tiliwoni kuti chonde, mwayi wanu woyenera kuthandizira kuti ndikufuna kudziwa kuti mwayi wanu woyenera kwambiri ndi kufufuza limi ndi quota zomwe mulipeza kwa mpaka m'malo. Komabe, tikufuna kumvetsa kuti kodi kudzipereka kapena kuchotsa zinthu zina za subsite yanu kulipeza kutanthauza kuti kodi kudangokera ndikupatsa nkhawa lofunika kwa woyenera wanu wosangalala.

Poyenera posts zomwe zikukwera ku quota yomwe mulipeza, muli ndi milengedwe imodzi imodzi: **Kupereka posts kodi** *,* **Kutanthauza posts mu trash** *,* kapena **Kutanthauza posts mu draft** *.* Muli ndi ulimi wopereka izi m'malo ya Ultimate Multisite settings.

![Network Admin Settings Sites page showing post-limit-exceeded behaviour options](/img/account-page/settings-sites-post-limit.png)

### Kodi kulipeza ku payment?

M'malo mwamene 2.0, sikuleni kutambira zinthu za proration (kudzipereka m'malo).

Iyi ndipo chifukwa cha kuti system ikupereka kwa membership yomwe yomonse kulipeza **kuti imapereke tsiku lamtengo lalikulu la kupewa (billing cycle) mopambana ndi mpaka** plan/membership yatsopano itambire. Tsiku lamtengo lalikulu la membership yatsopano litambira ndikuperekedwa ndi kutambira kwa billing cycle lili lopambana.
