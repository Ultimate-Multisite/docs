# Kodiwera Kufamba Ntchito Yopambana ndi Mwachidule kwa Codeable?

Mwachidule, tikufunika kukhala ndi zochitika zomwe zimapambana kuti mwayi wathu wosiyanasiyana umwe (network flows) ubonje monga tili tikufuna. Ngati ndi izi zomwe mukufuna, tikukonza [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), seya yomwe mungupatseni wopambana wosavuta ndi zozina za Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) ndi platform yomwe imapanga anthu omwe ali ndi chidziwitso (qualified WordPress freelance developers) ndi anthu omwe angakonzekera m'njira ya zochitika, zomwe mungakwaniritseni projects ndi mfundo yopambana kuti mumagwirizana ndi wopambana omwe ali ndi chidziwitso kwa zomwe mukufuna.

### Kodi mungapeza project m'Codeable bwanji?

Mmodzi, ndi lero ku [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) ndikupanga button ya **Start a Project**.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Ngati pano, kuti mupatseni project, chonde pambani kwa lero ku **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

M'malo m' "I need help with" (Ndikufuna madiningidwe ndi), chonde chita **Customizing** :

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

M'malo m' "My" (Yomwe ndi wanu), chonde chita **Plugin** :

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Kuti nthawi yomwe, chita button ya **Continue**.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Ngati pano, ndiyenera kupanga za project yanu ndi mwayi wanu wopambana kwambiri. Kuti mungupatseni zolemba zambiri!

<!-- Screenshot unavailable: Codeable project description form -->

Mawonjeza mavuto omwe muli kuti mufunse kukhala ndi chisoni (pricing).

<!-- Screenshot unavailable: Codeable project pricing questions -->

Puma, ngati simukulu ya [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) yomwe, mupereka imwe ndipo mupereke **Publish My Project** button.

<!-- Screenshot is not available: Codeable account creation and Publish My Project button -->

Pamene utapereka [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) yomwe, mpangamutsa ndi developers omwe ali ndi chiyembekezo lomwe lili lozi wanu.

Iyi njira idzathandiza kukhala munthu woyenera kwawo, wamene ali ndi kufunika kwa WordPress ndi Ultimate Multisite!
