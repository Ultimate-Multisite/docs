# Ultimate Multisite Hooks and filters

Ultimate Multisite ndi mfundo yomwe imapulogalamu onse ndi functions ndi database queries. Imene inali hooks ndi filters zomwe zimapangitsa kuti mupereke kapena kupereka code zatsopano pa functions zomwe zili pansi.

Tsimene list ya hooks ndi filters iyi kuti mupereke chidziwika kwa anthu omwe ali ndi mfundo zosiyanasiyana kuti mumakhalipira (customize) network yanu kwambiri. Mumakwaniritsa list iyi ku Github mwa kupanga link limodzi pano.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
