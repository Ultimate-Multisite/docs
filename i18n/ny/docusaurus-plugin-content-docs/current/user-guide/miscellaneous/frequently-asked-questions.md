# Matanka Zikomo (Frequently Asked Questions)

Ndi zikomo zokhawo zomwe zimeneza m'moyo pa zolemba za chidziwika.

  * **Kodi Ultimate Multisite ndi chiyani?**

Ultimate Multisite ndi plugin la WordPress, lomwe limapangidwa kuti m'WordPress multisite, limapangidwa lino m'malo wosangalatsa wa sites – monga [WordPress.com](https://WordPress.com) – lomwe kumene anthu angathe kupanga sites m'malo m'maka (mungathe kupanga njira za m'mabwera, m'mabwera m'maka, kapena m'mabwera m'maka). (Mungathe kudziwa kuti muli ndi njira zosangalatsa.)

  * **Kuti Ultimate Multisite kulipeza m'malo osati ndi Multisite?**

Osati. Ultimate Multisite imayenera WordPress multisite, subdirectory, kapena subdomain.

  * **Zimenezo za Ultimate Multisite ndi zikuwera kuti?**

Ultimate Multisite imayenera:

**WordPress Version** : v5.3+ (Kukonzekera: Last Stable Version)
**PHP Version** : 7.4.x (Kuthandiza ku > 8.0 kulipeza m'malo)
**MySQL Version** : v5+ (Kukonzekera: Version 5.6, monga muli ndi 8.0 osati muli wosangalatsa wanu)

_Multisite (subdomain kapena subdirectory) kumayenera kuthandizidwa._ [Mwayi wopanga WordPress Multisite](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **Kuti Ultimate Multisite kulipeza ndi Multisite ndi Subdirectories?**

E. Ultimate Multisite imayenera njira zosangalatsa (subdirectory) ndi njira zosangalatsa (subdirectory).

  * **Kuti Ultimate Multisite kulipeza ndi Multisite ndi Subdomains?**

E. Ultimate Multisite imayenera njira zosangalatsa (subdirectory) ndi njira zosangalatsa (subdomain).

  * **Kodi ndiyenera kutsegula Ultimate Multisite kuti mupereke chidziwitso zomwe anthu angathe kudzera ku Plugins ndi Themes?**

E. Ndi. Kwa Ultimate Multisite, mungathandize kuyesa kuti plugins ndi themes zikuwera kwa njira zonse pa network yanu.

  * **Kodi ndiyenera kutsegula Ultimate Multisite kuti mupereke nambala ya posts zomwe munthu amungathe kupanga pa site yake?**

Ndipo ndi Ultimate Multisite muli ndi ulemu woperekera post quotas kwa **post type onse zomwe zili pansi pa network yanu**. Muli ndi ulemu wosonyeza komwe kudzera kwa users kuti admin angathe kuita, **kufotokozera kwa disk space**, ndipo zimene zingali.

  * **Ultimate Multisite imaperekera Site Templates?**

E. Yes. Na Ultimate Multisite muli ndi ulemu woperekera Site Templates zomwe mukuwoneza. Site Templates si themes, koma ndi sites zosonyeza pansi pa network yanu. Muli ndi ulemu woperekera plugins, themes, ndipo muli ndi ulemu wosonyeza kuti muperekeze Site Template iwo kwa njira yomwe mukuwoneza. Ngati user imapereka template iwo, site yawo idzakhala kopitiliza (copy) ya template iwo, ndi zomwe zonse zikuyenera.

Muli ndi ulemu wosonyeza kuti muperekeze template yomwe yomwe. Mwachidule, ndipo user aliyense imaperekera plan iwo idzakhalira site yawo kupitiliza (duplicate) ya Site Template iwo.

  * **Ultimate Multisite imaperekera WooCommerce?**

E!

Ngati mukuwoneza kuti users wa network yanu adzakhale ndi WooCommerce, kumene kuumpatsa kuti muperekeze stores, zimene zili wosonyeza kwa Ultimate Multisite. Muli ndi ulemu woperekera kuti muperekeze anthu awo kuti awoneze products ngati m'plan tier imodzi.

Moyenera za subscription payments kwa Ultimate Multisite, muli ndi ulemu woperekera [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) kuti muperekeze kuti clients wa Ultimate Multisite iwo amaperekeze monga kwa WooCommerce gateways pa site yanu yomwe yomwe. Mwachidule, muli ndi ulemu wosonyeza kuti mukuwoneza library yomwe yomwe ya payment gateways yomwe yomwe imaperekedwa kwa WooCommerce ndipo muperekeze kuti muperekeze clients wa Ultimate Multisite iwo.

  * **Kodi pali njira yoperekera subscriptions kuchokera ku WPMUDEV Pro Sites?**

Ndili, Ultimate Multisite wena muli mpaka [zote zotani kuti mupereke chombo chofunika kukhala mupita (migrate)](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) kwa mtandawuni wanu wa Pro Sites.

  * **Kodi tab ya “Allowed Plugins” ikale yomwezi pa Page Edit ya Plan ya muli?**

Ultimate Multisite imupereke chifukwa kuti mupereke kapena kukatswa ulemu (access) kwa plugins zosakhala zikafunika ku mtandawuni wanu. Pamene plugin ikafunika ku mtandawuni, pamene imapereka, imapereka kwa kutetezaulo kwayo pa site zonse za mtandawuni wanu.

Kuti mupereke plugins zosangalala pa tab ya “Allowed Plugins”, chabwiri kuti mupereke kapena kukatswa ulemu (access) kwa plugin yomwe muli wokufuna, muli kukhala mupereke kapena sikukafunika ku mtandawuni.

  * **Kodi ndikubatsa n’maka ya kutengera beiya wangu?**

Eee. Since version 1.4.0, Ultimate Multisite imapereka kuti mupereke zotenga template (template overrides).

  * **Ultimate Multisite ikufunika ndi WP Engine?**

Ee. Ultimate Multisite imapereka kwa WP Engine wosakhala muli, koma kuti mupereke kutengera domain mapping kukhala kofunika kuti admin yomwe imapereka (adding the domain to the WP Engine panel). **Isi ndi chinthu chokhala muli since version 1.6.0.** Ultimate Multisite ikufunika m’mwayi pa API ya WP Engine ndipo domains zimaperekedwa kudzera m’mwayi pa panel muli pambuyo pomwe imapereka (after mapping).

  * **Muli ndi wopereka wotengera kuwona?**

Ultimate Multisite ikufunika kutumikira ndi anthu onse omwe amapereke hosting. Chinthu chokhala ndi kuti ndalama zambiri monga Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane ndi [Runcloud.io](http://Runcloud.io) imapereka kutengera domain mapping kukhala kofunika kwambiri, chifukwa imapereke m’mwayi pa panel pambuyo pomwe imapereka (work out of the box). Kwa hosting ena, muli wokonzekera kuti mupereke kutengera domain mapping m’mwayi.
