# Event: Register an account via Zapier {#event-register-an-account-via-zapier}

M'mulo m'mulo ku mpaka ku [Integrating Ultimate Multisite with Zapier](zapier.md), tsimbuka boso boso kuti muiteka nazo Zapier kuti mupereke zinthu zosonyeza m'Ultimate Multisite monga momwe mukuiteka ndi triggers ndi events. M'mulo, mudzindikire momwe mungakumbukire (integrate) applications za zina. Mudzindikire kuti mutanthauze Google Sheets monga chombo chokhala ndi data ndipo muperekeza ulendo wina kwa Ultimate Multisite kuti mupereke zinthu zomwe zimene.

M'mulo kukhala m'Zapier, ndi ulutsa ku **Ultimate Multisite > Settings > API & Webhooks** ndikumbukire kuti API imene. Kopya API Key ndi API Secret kuchokera kwa skrini iyi pamene Zapier ikupereka zinthu za account ya Ultimate Multisite.

![API and Webhooks settings with API Key, API Secret, and Enable API options](/img/admin/settings-api-webhooks.png)

M'mulo, mukupeza **Google Sheet** m'Drive yanu. Mungathe kuti muperekeza bwino m'column zina kuti muperekeze data m'malo momwe mungayendera m'malo.

Pambuyo pomwe mukupeza Google sheet, mukuwoneka ku account ya Zapier yanu ndikuyamba kukhala ndi zap.

M'mulo kwa search field ya **"App event"** mukuphuka **"Google Sheets"**.


M'mulo kwa field ya "**Event** ", mukuphuka **"New spreadsheet row** " ndipo muperekeza **"Continue** ".

M'mulo, nthawi yomwe yomwe ikupereka kukhala ndi chondecha kuti muperekeze **Google Account** pamene **Google Sheet** imene ikuperekedwa. Mwayi mwambiri wamene mwayi wosiyesera (second action) wokukwera kapena kukhazikira akaunti mu Ultimate Multisite. Mu search field, chita "**Ultimate Multisite(2.0.2)** ".

Mu field "Event", chita "**Register an Account in Ultimate Multisite**" muli, ndipo chita button "Continue".

Mu "Set up an action", muli mwayi wosiyesera kwa maaka awo omwe ali ndi data ya mwanawo (customer data), memberships, products, ndi zina zinthu. Mungathe kukhazikira mavuto (values) mu google sheet yanu ndipo kumayesa ku field yomwe imayenera kupakanyika monga momwe muli wamwona mu screenshot wosiyanasiya.

Pambuyo pa kukwera mavuto, mungathe kuyesa action.
