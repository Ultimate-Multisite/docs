# Zomwezi ya Dashboard {#dashboard-overview}

Ultimate Multisite inalipanga **Dashboard** iliyonkhulupanga m'malo wa Ultimate Multisite, ndipo inalimbikisa widgets ku Network Dashboard. Iyi article inalipanga kuti muli ndi chiyambira m'malowowo.

## Network Dashboard {#the-network-dashboard}

Ultimate Multisite inalimbikisa widgets zofunika ku dashboard ya network yomwe imayendera m'My Sites → Network Admin → Dashboard.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

Iyi ndi chiyambira chomwe kukhala m'malowo:

**First Steps:** Kukhala m'malo wosavuta ndipo kumalimbikisa kuti muwoneke m'malozo ya kukonza ntchito monga kuyendetsa payment method ndi kupeza account yomwe imayendera.

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Summary:** Imalimbikisa anthu omwe alipira ndipo mapulogolo ya tsiku. MRR (Monthly Recurring Revenue) imalimbikisa ntchito yomwe imayendera m'malozo kwa anthu omwe ali ndi membership omwe imayendera mapulogolo.

![Summary widget](/img/admin/network-dashboard-summary.png)

**Activity Stream:** Imalimbikisa zinthu zomwe zimayendera m'Ultimate Multisite — kupira, kuchepetsa, kukhala kwa sites, kukhalitsa, ndi zinthu zina.

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** Imalimbikisa chotsatira cha anthu ndi sites m'malo wosavuta wa network ndi shortcuts kuti mupeza sites kapena anthu atsopano.

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** Imalimbikisa ndimaka za Ultimate Multisite zomwe zikuwoneka kwambiri, kuphatikiza updates, bugs zosonyeza, ndi mwayi wopanga chisoni.

![News widget](/img/admin/network-dashboard-news.png)

## Ultimate Multisite Dashboard {#the-ultimate-multisite-dashboard}

Pamene network dashboard imalimbikisa malingaliro omwe ali m'malozo wa network, Ultimate Multisite dashboard (m'malo wosavuta wa Ultimate Multisite) imalimbikisa malingaliro a service yomwe imayendera.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Kutera Kukhulupuka kwa Reruvu Lwamba (Monthly Recurring Revenue Growth):** Iyi ndiyo inalipisa utumiki wa reruvu lwamba mwezi mwezi pa mwaka woshanzi wa miaka 12. Reruvu lwamba lili mpya lilioneka m'ndulo ya chiviri (blue) pamene, m'ndulo ya kuondoka lilioneka m'ndulo ya chiviri (red).

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)
