# Kupuka na Kupanga

Ultimate Multisite 2.9.0 imapanga ya **Export & Import** yomwe mwezi mmoja mwa **Tools > Export & Import**. Tumia ikuti mukati mwake muliipanga WordPress site mwezi mmoja kama ZIP file, kurekebisha hiyo ZIP, au kupanga site kati ya Ultimate Multisite na single-site WordPress installations zilizolingana.

## Ruhusa Zinazohitajika

Lazidi kuingia kama admin aliye na uwezo wa kufikia menyu ya **Tools** ya WordPress kwenye site inayopangwa au inayopangwa. Kwenye mtandao mkuu (multisite network), tumia akaunti ya admin ya mtandao wakati wa kupanga au kupanga subsites kutoka kwa zana za Ultimate Multisite za kiwanda (network-level).

Kupanga ZIP downloads zikutolewa kupitia endpoint la download lililithibitishwa, basi chagua session ya admin iendeze mpaka download itakamilike na usishiriki URLs za download zilizotengenezwa kwa umma.

## Kupanga site kwenda ZIP

1. Katika admin ya WordPress ya site unayotaka kuikopi, nenda kwenye **Tools > Export & Import**.
2. Fungua eneo la kupanga na chagua site unayotaka kuipanga.
3. Chagua maudhui yasiyohitajika (optional content) ya kuingiza, kama vile uploads, plugins, na themes, wakati hizo chaguo ziko wapo.
4. Anza kupanga na subiri mchakato ukakamilike. Sites kubwa zinaweza kuhitaji kukamilika nyuma kabla ZIP haikupangwa tayari.
5. Download ZIP iliyokamilika kutoka kwenye orodha ya exports.

Weka ZIP katika mahali salama. Inaweza kuwa na maudhui ya site, mipangilio (settings), faili za media, na vipengele vya code vilivyochaguliwa.

## Kile kinachopangwa

ZIP ya kupanga inaweza kujumuisha:

- Maudhui ya database na mipangilio ya site iliyochaguliwa.
- Faili za media zilizopangwa wakati uploads zikijumuishwa.
- Plugins na themes wakati chaguo hilo likichaguliwa.
- Metadata ya kuingiza inayotumika na zana ya Export & Import ili kujenga upya site kwenye installation inayolengwa.

M'ndu wamwamba wa ZIP ulipambana ndi kutengera monga koti kuwoneka kwa media, plugins, ndithu (themes), komanso ukulu kwa database tables za site.

Mawu wopolo wa kuupaka mwayi wamene mwayi wosokoneka (uploads directories) kapena media libraries zikati zikulu zingozera ZIP files zikulu. Ndikumbutsa kuti muwone limit za PHP upload, execution limits, disk space, memory, na server timeout settings monga momwe muli kupanga kapena kuingiza sites zikulu.
Media libraries zikulu zingakhale zifunika kutulizidwa m'maka pafupi pa utumiki (low-traffic maintenance window).
Kutanthauza, WordPress yosokoneka (target installation) yofunika kukhala ndi WordPress, PHP, Ultimate Multisite, plugin, na theme versions zomwe zikhoza kutumana.
Import ya site imodzi imayendetsa site yosokoneka. Isi ndi tool yopanga mwayi wosokoneka (merge tool).
Zomwe zikuti Multisite-to-single-site na single-site-to-multisite moves zikhoza kutumana monga momwe m'maka pafupi pa utumiki (low-traffic maintenance window) ndipo site yosokoneka ikhoza kuyendetsa plugins, themes, URLs, ndi components zomwe zifunika za Ultimate Multisite.
Kupanga ZIP iye kukhale lina. Imayesa kuti imatha kukhala ndi database content, files zomwe zimapaka, na configuration details kuchokera kwa site yosokoneka.

Poyamba mwayi wopanga mwayi wosokoneka wa network (network-level export workflows) omwe ali wosona, mwayi wosonyeza [Site Export](/user-guide/administration/site-export).
