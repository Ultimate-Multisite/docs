# Zomweka ndi zinthu zomwe si yenera kuti m'malo ndi Plugins zina

Pambuyo pa kukhazikitsa ndikhatuliza Ultimate Multisite, mumveza kukumbutsa mavuto omwe osalira m'malo ndi Plugins zina osalira.

Tili tikuchitira zochitika kuti tikuchepetsa lilo laziro lilo lilo m'malo ndi kukhala kwa mpulogolo yomwe yenera, koma ngati plugin imayenera pano, mumveza kuti mukuchepa kuti muchepe kuti Ultimate Multisite Network yenera ikhale yenera.

Ili ndi lilo laziro lilo ndi zomwe mukufuna kuti mudzire:

**Zilipo za WPMU DEV plugins Zomwe ziri kuchokera kwa WPMU DEV** Kuchokera kwa plugin zina zomwe si yenera kwambiri ndi WPMU DEV zimene kudzidwa kuti zimapangitse kusintha m'malo wa Ultimate Multisite kuoneka m'malo omwe wokhala (wosavomerezeka kukhala account). Ili ndi lilo laziro lilo lomwe limapangitsa mavuto:

  * Signup Language: _Ultimate Multisite ndi chinthu chomwe kumapangidwa kuti muli ndi mfundo yomwe yenera, koma ndi chinthu chamene kuli Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Ili limapangidwa m'malo ndi Ultimate Multisite, chiyenera kulimbikitsa plugin inayo.*

  * New Blog Templates: _Ili limapangidwa m'malo ndi Ultimate Multisite, chiyenera kulimbikitsa plugin inayo.*

  * Set Password on Multisite Blog Creation: _Ili limapangidwa m'malo ndi Ultimate Multisite, chiyenera kulimbikitsa plugin inayo.*

  * Signup TOS: _Ili limapangidwa m'malo ndi Ultimate Multisite, chiyenera kulimbikitsa plugin inayo.*

  * Anti-Splog: Ngakhale si yenera kwambiri, ili _limapangidwa m'malo ndi Ultimate Multisite._

**Kumbukira** : Plugins zina za WPMU Dev zingakhale zikugwiritsidwa ntchito bwino m'malo ndi Ultimate Multisite. Malingaliro ndi: _Smush_ , _Forminator_ , _Defender,_ ndi zina.
