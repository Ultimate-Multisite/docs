# Kodiwera kuingilira Komuniti ya Ultimate Multisite {#how-to-join-ultimate-multisite-community}

Ultimate Multisite Community ndi malo omwe mungapeze kukhala ndi mwayi ndi anthu ena a Ultimate Multisite ndikupatsa zolemba, zokhudza kufunika, ndi zina. Mukupeza Ultimate Multisite Community m'malo: https://community.ultimatemultisite.com/home.

Pa upeti woyamba, ndiupatsani ndi link ya “**kugonera akaunti yanu pano** ”.

<!-- Screenshot is not available: Kodiwera kwa upeti wa Komuniti ndi link la kugonera akaunti -->

Mungaperekedwa ku upeti woyenera wathu. Ngati, mukupeza username, email yanu, ndi password. Yendetsa Captcha ndikupatsani button ya **Register**.

<!-- Screenshot is not available: Kodiwera kwa upeti wa Komuniti ndi maphu ya username, email, password ndi Captcha -->

Poyenera akaunti, ndi nthawi yomwe mungagone profile yanu. Pa upeti wosiyanasiyana, mungapeze kuwerenga pikhulupiriro lanu, kuyesa limi lanu, ndipo zina. Kodi mukuwunika kukhala ndi links za social media lanu ndipo kumapereka udindo (permissions) kwa akaunti yanu.

<!-- Screenshot is not available: Upeti wa kugonera profile woyenera ndi maphu a kuwerenga pikhulupiriro ndi limi -->

Ngati, mukupeza channel ya “Introductions”, pamene mungapeze kukhala ndi chidziwitso lanu ndipo kumakuthandiza kudziwa malo ena a anthu. Mungagone post yanu monga momwe mungapeze kuwerenga button ya **New Post**.

<!-- Screenshot is not available: Channel ya Introductions kwa Lounge section ndi channel ya Dev Logs ndi button ya New Post -->

Channels pa lounge section ndi channel ya “Dev Logs” ndi wopereka kwa anthu onse. Komabe, mukuwunika udindo wosonyeza kuti muapeze channels zina m'mkomuniti.

<!-- Screenshot is not available: Channels za Komuniti zokwera Lounge section ndi Dev Logs channel -->

Udindo wopereka wina wamapereka pamene mungapereke nazo license key yanu ya Ultimate Multisite yomwe ikulimbikira. Mukupeza ndipo mukupeza button ya **Unlock**.

<!-- Screenshot is not available: Channel ya mwayi wosayewu ndi button ya Unlock -->

Muli kudziwitsa pa ndauzo yomwe imapululira (unlock access page). Ndikumbukira kuti muli ndi key ya Ultimate Multisite license wanzerani ndipo mupulule button ya Unlock Access:

<!-- Screenshot is not available: Channel ya unlock access pafupi ndi kuwerenga key -->

License yanu idzayesa ndipo mudzakhala ndi ulemu wosayewu (private channels). Izi zikukonza nthawi yomwe yomweyi kudzera m'maka 24, chidziwitsa link kuti muli kuyenda ku community ndipo mupathe ulemu wosayewu wina pamene tikupanga mawu anu.
