# Moleni kuthandiza ntchito

Uli na License key wanu yopambana, ndipo kuona kule nazo ule ufufuza kuti mndzi wathu wamane kuti athandize kufula zvinhu zvinogona kuona pamwe nemaka pakutanga Ultimate Multisite.

I lesson iyi inozivisa mitambo yakuti mndzi wanu wamambwa (license holders) anogona kutamba nenzani kuti athandize kufula zvinhu zvakakosha pamwe nemaka pakutanga Ultimate Multisite.

## Help Site

Mungafanira mitambo yakanaka uye tutorials kubva munzira iyi inotapa link inotsika pano.

[)

Tiri kudzidza kuti mndzi wathu wamambwa anogona kuona zvinhu zvinobva Ultimate Multisite pakutanga. Kune "Getting Started" articles (mitambo yakutanga) munzira iyi inotapa, uye mavho akakosha anogona kuona pamwe nemaka pakutanga, zvakadzama kuti mupfungwa sei kutanga add-ons, sei kudzivirira payment gateways, ndokuti zvinhu.

## Contact Support

Kana uri kupfuura uye usina kuona mazano munzira iyi inotapa, unogona kuita contact na mndzi wathu wamane (support team) nekuti unogona kutamba icon ya speech bubble inotsika pamwe nemaka pakutanga pawebsite yedu OR nekuita email dzokuti support@ultimatemultisite.com.

Kana uri kutaura nenduro dzinotanga, ndichida kuti uone zvinhu zvakadzama nezvinhu zvinotanga pamwe nemaka pakutanga, zvichiri screenshot, error messages, kana kunyangwe kunyangwe kwakawanda kwakakosha kuita kuti vanonzi vachida kuona uye kuona mupfungwa wako. Unofanira kutaura License key yako. Zvinhu zvinogona kutamba support zvakaita mndzi wanu wamambwa anogona kuona.

## Community Site

[community site] yedu (https://community.ultimatemultisite.com/) ndiyo chigumbi chinogona kuti mndzi wemweya wakaenda Ultimate Multisite anogone kutaura nemunhu wose. Inoita zvinhu zvinobva system yekuti vanhu vashandisa (members) vanogone kuita posts, kucomment kana mafungiro, kuipa matsviki, kutanga kunge mweya (collaborate), kana kuratidza zvinhu pamwe chete. Unofanira kutaura Ultimate Multisite license key yako kuti ufula spaces/pages dzakakwana.
