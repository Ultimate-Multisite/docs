# Kufunsa Panel Yanu ndi PRO Theme {#customizing-your-admin-panel-with-pro-theme}

Mmodzi pamene mukuyenda ndi zinthu za wamene amagwira ntchito, tikufunsa kuti tiye mwayi wosonyeza (custom experience) m'dashboard ya WordPress kwa wamene atili. Ngati mukuphunzira mpaka mwayi wosonyeza wopambana kwambiri, PRO Theme mungathandize.

PRO Theme ndi theme yomwe imapambana kuti mukuwunje mwayi wosonyeza wa clients wanene. Ndipo ndi chinthu cha core la plugins zokhudza Ultimate Multisite.

Poyamba pamene muliyo, ndi lankhula pa page ya _Plugins_ ndikupambana kuti mupambane (activate) pa Network yanu.

<!-- Screenshot is not available: WP Plugins page showing Network Activate button for PRO Theme -->

_Pambanani kwa mtundu wosonyeza wose pa page ya Plugins_

_**CHONCHO:**_ _Ngati mukuwunje poyamba, kumapambana kuwonetsa kwa clients wanene kuti ndi chagera lomwe mungupambane kuti muwunje theme imene kwa wamene. _

Nkhawa, muli ndi mwayi wosonyeza wa admin yanu womwe umupambanira kuyamba! Muli ndi menu yanu yanu yanu pa chinthu cha m'left column lomwe limapambana PRO Theme.
