# Kule Ads pa m'ndzi wanu ndi Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector ndi **add-on yosangalatsa** yomene imapereka zikukonza adu pa njira ya kopeka yanu.

Panda mukuwera Ultimate Multisite: Ad Injector, ndiuka pa _Plugins_ page yanu kuti muwoneze kuyendetsa.

<!-- Screenshot is not available: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Kukonza button ya Network Activate_

Panda add-on imakwera, _Advanced Options_ meta box pa skrini ya kopeka yanu ndi chitsanzo chitsancho cha _Ad Settings_.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_Nkhawa mukuwera ad settings kwa njira zonse zanu_

Pa tab imodzi, mungakwanise kuwamba kodi za ads pa posts ndi pages pamene mukuwera, monga m'moyo wopereka ndi pansi pa mwayi.

<!-- Screenshot is not available: Ad code fields for posts and pages before content -->

<!-- Screenshot is not available: Example of ad displayed on a post page for a specific plan -->

_Panda mungawamba kodi za ads, zimakwera pa posts ndi pages pamene mukuwera imodzi._

Mungakwaniritsa kuti muli na ads pa header ya Admin Pages kwa anthu omwe ali pa njira imodzi.

<!-- Screenshot is not available: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot is not available: Example of ad displayed on the user admin panel -->

_Mukuwera Admin Panel Ad Code option kungayenera kuwamba ads pa back-end ya anthu wanu_

### Pansi pa Kulemba Ads {#beyond-advertising}

Mungakwaniritsa kuti muli na Ultimate Multisite: Ad Injector kuti muwoneze kukonza chisoni kwa njira zokhudzana ndi njira zikulu.

Kukonza saja kuwamba pa option ya _Admin Panel Ads_ ndikuwamba HTML code yopereka wopereka wanu.
