# നിങ്ങളുടെ ക്ലയന്റിന്റെ വെബ്സൈറ്റുകൾ (v2)

നിങ്ങളുടെ നെറ്റ്‌വർക്കിൽ രജിസ്റ്റർ ചെയ്തതിനുശേഷം, നിങ്ങളുടെ കസ്റ്റമർമാർക്ക് അവരുടെ dashboard ആക്‌സസ് ചെയ്യാൻ കഴിയും.

**Account** പേജിൽ അവരുടെ പേയ്‌മെന്റുകൾ, മെമ്പർഷിപ്പുകൾ, ഡൊമെയ്‌നുകൾ, പ്ലാൻ പരിമിതികൾ തുടങ്ങിയ പ്രധാനപ്പെട്ട വിവരങ്ങൾ കാണാൻ കഴിയും. കൂടാതെ, **My Sites** പേജിലേക്കും അവർക്ക് ആക്‌സസ് ഉണ്ട്, അവിടെ അവരുടെ എല്ലാ വെബ്സൈറ്റുകളും കാണാനും മാനേജ് ചെയ്യാനും കഴിയും. അവരുടെ dashboard-ൽ My Sites എന്നതിൽ ക്ലിക്ക് ചെയ്താൽ മതി.

![കസ്റ്റമർ dashboard-ലെ My Sites ലിങ്ക്](/img/admin/customers-list.png)

അവരുടെ എല്ലാ വെബ്സൈറ്റുകളും കാണിക്കും, ഓരോന്നിന്റെയും താഴെ ഒരു **Manage** ബട്ടൺ ഉണ്ടായിരിക്കും. അതിൽ ക്ലിക്ക് ചെയ്താൽ, **ആ പ്രത്യേക വെബ്സൈറ്റിന്റെ dashboard-ലേക്ക്** അവർ റീഡയറക്ട് ചെയ്യപ്പെടും.

ഒന്നിൽ കൂടുതൽ വെബ്സൈറ്റുകൾ അനുവദിക്കുന്ന പ്ലാനുകളിൽ, ഏത് വെബ്സൈറ്റാണ് **പ്രൈമറി** എന്നും അവർക്ക് കാണാൻ കഴിയും.

![Manage ബട്ടണുകളോടുകൂടിയ കസ്റ്റമർ വെബ്സൈറ്റുകൾ കാണിക്കുന്ന My Sites പേജ്](/img/admin/customers-list.png)

അവരുടെ പ്രൈമറി വെബ്സൈറ്റ് മാറ്റണമെങ്കിൽ, **Account** മെനുവിൽ ക്ലിക്ക് ചെയ്ത് Account പേജിലേക്ക് പോകണം.

![കസ്റ്റമർ dashboard-ലെ Account മെനു ലിങ്ക്](/img/admin/customers-list.png)

തുടർന്ന്, **Change Default Site** എന്നതിൽ ക്ലിക്ക് ചെയ്യണം.

![Account പേജിലെ Change Default Site ബട്ടൺ](/img/admin/customers-list.png)

ഒരു പോപ്പ്-അപ്പ് വിൻഡോ ദൃശ്യമാകും. ഡ്രോപ്പ്-ഡൗൺ ലിസ്റ്റിൽ ക്ലിക്ക് ചെയ്ത് ഏത് വെബ്സൈറ്റ് പ്രൈമറി വെബ്സൈറ്റാക്കണം എന്ന് തിരഞ്ഞെടുക്കുക.

![Change Default Site ഡ്രോപ്പ്ഡൗൺ സെലക്ഷൻ](/img/admin/customers-list.png)

അതിനുശേഷം, **Change Default Site** എന്നതിൽ ക്ലിക്ക് ചെയ്താൽ മതി, കാര്യം കഴിഞ്ഞു!

![ഡിഫോൾട്ട് സൈറ്റ് മാറ്റം സ്ഥിരീകരിക്കുന്നു](/img/admin/customers-list.png)
