# നിങ്ങളുടെ ക്ലയന്റിന്റെ വെബ്സൈറ്റുകൾ (v2) {#your-clients-websites-v2}

നിങ്ങളുടെ നെറ്റ്വർക്കിൽ രജിസ്റ്റർ ചെയ്ത ശേഷം, നിങ്ങളുടെ ഉപഭോക്താക്കൾക്ക് അവരുടെ Dashboard ആക്സസ് ചെയ്യാം.

അവരുടെ പേയ്മെന്റുകൾ, അംഗത്വങ്ങൾ, ഡൊമെയ്‌നുകൾ, പ്ലാൻ പരിധികൾ എന്നിവയും മറ്റു പ്രധാന വിവരങ്ങളും കാണാനാകുന്ന **Account** പേജിനുപുറമേ, എല്ലാ വെബ്സൈറ്റുകളും കാണാനും നിയന്ത്രിക്കാനും കഴിയുന്ന **എന്റെ സൈറ്റുകൾ** പേജിലേക്കും അവർക്ക് ആക്സസ് ഉണ്ട്. അവർ ചെയ്യേണ്ടത് അവരുടെ Dashboard-ൽ My Sites ക്ലിക്ക് ചെയ്യുക മാത്രമാണ്.

![ഉപഭോക്തൃ Dashboard-യിലെ My Sites ലിങ്ക്](/img/account-page/my-sites-link.png)

അവരുടെ എല്ലാ വെബ്സൈറ്റുകളും കാണിക്കും, ഓരോന്നിന്റെയും താഴെ **നിയന്ത്രിക്കുക** ബട്ടൺ ഉണ്ടായിരിക്കും. അതിൽ ക്ലിക്ക് ചെയ്താൽ, അവർ **ഈ പ്രത്യേക വെബ്സൈറ്റിന്റെ Dashboard**-ലേക്ക് തിരിച്ചുവിടപ്പെടും.

ഒന്നിലധികം വെബ്സൈറ്റ് അനുവദിക്കുന്ന പ്ലാനുകളിൽ, അവരുടെ ഏത് വെബ്സൈറ്റാണ് **പ്രാഥമിക** വെബ്സൈറ്റ് എന്നും അവർക്ക് കാണാം.

![Manage ബട്ടണുകളുള്ള ഉപഭോക്തൃ വെബ്സൈറ്റുകൾ കാണിക്കുന്ന My Sites പേജ്](/img/account-page/my-sites-page.png)

സൈറ്റ് തിരുത്തൽ ഇന്റർഫേസിന്റെ പൂർണ്ണ കാഴ്ച ഇതാ:

![സൈറ്റ് തിരുത്തൽ പൂർണ്ണ ഇന്റർഫേസ്](/img/admin/site-edit-full.png)

പ്രാഥമിക വെബ്സൈറ്റ് മാറ്റണമെങ്കിൽ, **Account** മെനുവിൽ ക്ലിക്ക് ചെയ്ത് Account പേജിലേക്ക് പോകണം.

![ഉപഭോക്തൃ Dashboard-യിലെ Account മെനു ലിങ്ക്](/img/account-page/account-menu.png)

നിങ്ങൾക്ക് ഉപഭോക്തൃ വിശദാംശങ്ങൾ കാണാനും തിരുത്താനും കഴിയും:

![ഉപഭോക്തൃ തിരുത്തൽ ഇന്റർഫേസ്](/img/admin/customer-edit.png)

തുടർന്ന്, അവർ **ഡീഫോൾട്ട് സൈറ്റ് മാറ്റുക** ക്ലിക്ക് ചെയ്യണം.

![Account പേജിലെ Change Default Site ബട്ടൺ](/img/account-page/change-default-site.png)

ഒരു പോപ്പ്-അപ്പ് വിൻഡോ പ്രത്യക്ഷപ്പെടും. ഡ്രോപ്പ്-ഡൗൺ ലിസ്റ്റിൽ ക്ലിക്ക് ചെയ്ത് ഏത് വെബ്സൈറ്റിനെയാണ് പ്രാഥമിക വെബ്സൈറ്റാക്കണമെന്ന് തിരഞ്ഞെടുക്കുക.

![Change Default Site ഡ്രോപ്പ്-ഡൗൺ തിരഞ്ഞെടുപ്പ്](/img/account-page/change-default-site-dropdown.png)

തുടർന്ന്, **ഡീഫോൾട്ട് സൈറ്റ് മാറ്റുക** ക്ലിക്ക് ചെയ്താൽ മതി, നിങ്ങൾ പൂർത്തിയാക്കി!

![ഡീഫോൾട്ട് സൈറ്റ് മാറ്റം സ്ഥിരീകരിക്കുന്നു](/img/account-page/change-default-site-confirm.png)
