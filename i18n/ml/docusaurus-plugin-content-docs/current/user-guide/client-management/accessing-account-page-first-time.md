# എന്റെ അക്കൗണ്ട് പേജ് ആദ്യമായി എങ്ങനെ ആക്സസ് ചെയ്യാം?

Ultimate Multisite-ലേക്ക് സ്വാഗതം!

ഞങ്ങളുടെ plugin വാങ്ങിയ ശേഷം നിങ്ങളുടെ ഇ-മെയിലിൽ ലഭിച്ച ലിങ്ക് വഴി നിങ്ങളുടെ അക്കൗണ്ട് പേജ് ആക്സസ് ചെയ്യുക:

<!-- Screenshot unavailable: Account page email with access link -->

വാങ്ങുമ്പോൾ നൽകിയ ഇമെയിലും നിങ്ങളുടെ ഇമെയിലിൽ ലഭിച്ച പാസ്‌വേഡും ഉപയോഗിക്കുക.

<!-- Screenshot unavailable: Login page with email and password fields -->

ഈ അക്കൗണ്ടിലേക്ക് നിങ്ങൾ ആദ്യമായി ലോഗിൻ ചെയ്യുന്നതിനാൽ, പാസ്‌വേഡ് മാറ്റാൻ ആവശ്യപ്പെടും. അത് മാറ്റി _Save_ ബട്ടൺ ക്ലിക്ക് ചെയ്യുക.

<!-- Screenshot unavailable: Change password form with Save button -->

നിങ്ങളുടെ അക്കൗണ്ട് പേജിൽ, ബില്ലിംഗുകൾ, പുതുക്കലുകൾ, റദ്ദാക്കലുകൾ, മറ്റ് പല കാര്യങ്ങളും പോലുള്ള എല്ലാ അക്കൗണ്ട് വിശദാംശങ്ങളും നിങ്ങൾക്ക് മാനേജ് ചെയ്യാൻ കഴിയും.
