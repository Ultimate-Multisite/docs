# എന്റെ സബ്‌സ്‌ക്രിപ്ഷൻ എങ്ങനെ റദ്ദാക്കാം?

നിങ്ങൾക്ക് Ultimate Multisite ലൈസൻസ് ഉണ്ടെങ്കിൽ, എപ്പോൾ വേണമെങ്കിലും അതിന്റെ പുതുക്കൽ റദ്ദാക്കാം. താഴെ കൊടുത്തിരിക്കുന്ന ഘട്ടങ്ങൾ പിന്തുടരുക:


<!-- Screenshot unavailable: Account page email with access link -->

വാങ്ങുമ്പോൾ നൽകിയ ഇമെയിലും പാസ്‌വേഡും ഉപയോഗിക്കുക.

<!-- Screenshot unavailable: Login page with email and password fields -->

ഇതാണ് നിങ്ങളുടെ Account Page-ന്റെ ആദ്യ സ്‌ക്രീൻ:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

നിങ്ങളുടെ സബ്‌സ്‌ക്രിപ്ഷൻ റദ്ദാക്കാൻ, ഇടതുവശത്തെ മെനുവിൽ നിന്ന് _Renewing & Billing_-ലേക്ക് പോകുക.

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

ഒരു സൈഡ് വിൻഡോ തുറക്കാൻ വലതുവശത്തെ അമ്പടയാളത്തിൽ ക്ലിക്ക് ചെയ്യുക. തുടർന്ന്, **auto-renew** റദ്ദാക്കുന്ന ഓപ്ഷൻ തിരഞ്ഞെടുക്കുക.

സിസ്റ്റം ഒരു സ്ഥിരീകരണ സന്ദേശം കാണിക്കും.

നിങ്ങളുടെ സബ്‌സ്‌ക്രിപ്ഷൻ റദ്ദാക്കണമെന്ന് ഉറപ്പാണെങ്കിൽ, _**Cancel Renewals**_ ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക. ഈ പ്രവർത്തനത്തിന് ശേഷം, ഒരു ചെറിയ സർവേയിൽ പങ്കെടുക്കാൻ ആവശ്യപ്പെടും.

കഴിഞ്ഞു! നിങ്ങളുടെ സബ്‌സ്‌ക്രിപ്ഷൻ സ്വയമേവ പുതുക്കപ്പെടില്ല.

**നിങ്ങളുടെ സബ്‌സ്‌ക്രിപ്ഷൻ കാലാവധി തീരുന്നതുവരെ നിങ്ങൾക്ക് സാധുവായ ഒരു കീ ഉണ്ടായിരിക്കും**. സബ്‌സ്‌ക്രിപ്ഷൻ വീണ്ടും സജീവമാക്കണമെങ്കിൽ, അത് മാനുവലായി ചെയ്യേണ്ടതുണ്ട്.
