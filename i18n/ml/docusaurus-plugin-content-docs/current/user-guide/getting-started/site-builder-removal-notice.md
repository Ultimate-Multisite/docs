# സൈറ്റ് ബിൽഡർ മോഡ് നീക്കം ചെയ്യുന്നതിനെക്കുറിച്ചുള്ള അറിയിപ്പ്

**Superdav AI Agent v1.12.0-ൽ സൈറ്റ് ബിൽഡർ മോഡ് (Site Builder mode) നീക്കം ചെയ്തിരിക്കുന്നു.** നിങ്ങൾ സൈറ്റ് ബിൽഡർ മോഡ് ഉപയോഗിക്കുകയാണെങ്കിൽ, തീം ഉണ്ടാക്കുന്നതിനും സൈറ്റ് സെറ്റപ്പ് ചെയ്യുന്നതിനും നിങ്ങൾ **Setup Assistant ഏജന്റിനൊപ്പം** (Setup Assistant agent) മാറണം.

## എന്ത് സംഭവിച്ചു? {#what-happened}

### സൈറ്റ് ബിൽഡർ മോഡ് (പഴയത്) {#site-builder-mode-legacy}

സൈറ്റ് ബിൽഡർ മോഡ് എന്നത് താഴെ പറയുന്ന കാര്യങ്ങൾക്കായി വിസാർഡ് അടിസ്ഥാനമാക്കിയുള്ള ഒരു ഇന്റർഫേസ് ആയിരുന്നു:

- ടെംപ്ലേറ്റുകളിൽ നിന്ന് സൈറ്റുകൾ ഉണ്ടാക്കുക
- അടിസ്ഥാന ക്രമീകരണങ്ങൾ കോൺഫിഗർ ചെയ്യുക
- ഒരു തീം തിരഞ്ഞെടുക്കുക
- ആദ്യത്തെ ഉള്ളടക്കം സജ്ജമാക്കുക

### അതിന് പകരമായി വന്നത് എന്ത്? {#what-replaced-it}

ഇപ്പോൾ **Setup Assistant ഏജന്റ്** എല്ലാ സൈറ്റ് ബിൽഡർ പ്രവർത്തനങ്ങളും കൈകാര്യം ചെയ്യുന്നു. ഇതിൽ ഉൾപ്പെടുന്നത്:

- കൂടുതൽ വഴക്കമുള്ള, ഏജന്റ് വഴിയുള്ള സെറ്റപ്പ്
- മികച്ച തീം കസ്റ്റമൈസേഷൻ ഓപ്ഷനുകൾ
- തീം ബിൽഡർ ഓൺബോർഡിംഗുമായി സംയോജനം (Integration)
- ഭാവി സെഷനുകൾക്കായി സ്ഥിരമായ site_brief മെമ്മറി

## നിങ്ങൾ സൈറ്റ് ബിൽഡർ മോഡ് ഉപയോഗിച്ചിരുന്നെങ്കിൽ {#if-you-were-using-site-builder-mode}

### നിങ്ങളുടെ സൈറ്റുകൾ സുരക്ഷിതമാണ് {#your-sites-are-safe}

- സൈറ്റ് ബിൽഡർ മോഡിൽ ഉണ്ടാക്കിയ നിലവിലുള്ള സൈറ്റുകൾ പ്രവർത്തിക്കുന്നത് തുടരും
- ഡാറ്റ നഷ്ടമോ സൈറ്റ് തടസ്സമോ ഉണ്ടാകില്ല
- നിങ്ങളുടെ സൈറ്റുകൾ സാധാരണ പോലെ നിങ്ങൾക്ക് കൈകാര്യം ചെയ്യാൻ കഴിയും

### Setup Assistant ഏജന്റിലേക്ക് മാറുക {#migrate-to-setup-assistant-agent}

പുതിയ സൈറ്റ് സെറ്റപ്പ് ചെയ്യാനോ തീം മാറ്റാനോ, Setup Assistant ഏജന്റ് ഉപയോഗിക്കുക:

```
"Help me set up a new site"
```

അല്ലെങ്കിൽ

```
"Start the Theme Builder onboarding"
```

Setup Assistant ഏജന്റ് കൂടുതൽ വഴക്കത്തോടെ അതേ പ്രവർത്തനക്ഷമത നൽകുന്നു.

## താരതമ്യം: സൈറ്റ് ബിൽഡർ vs. Setup Assistant {#comparison-site-builder-vs-setup-assistant}

| സവിശേഷത | സൈറ്റ് ബിൽഡർ (നീക്കം ചെയ്തു) | Setup Assistant (പുതിയത്) |
|---------|----------------------|----------------------|
| സെറ്റപ്പ് രീതി | വിസാർഡ് ഫോം | ഏജന്റ് സംഭാഷണം |
| തീം തിരഞ്ഞെടുക്കൽ | മുൻകൂട്ടി നിർവചിച്ച ടെംപ്ലേറ്റുകൾ | കസ്റ്റം ജനറേഷൻ |
| കസ്റ്റമൈസേഷൻ | പരിമിതമായ ഓപ്ഷനുകൾ | പൂർണ്ണ ഡിസൈൻ സിസ്റ്റം |
| സൈറ്റ് ബ്രീഫ് | സംഭരിക്കില്ല | സ്ഥിരമായ മെമ്മറി |
| ഭാവി സെഷനുകൾ | സെറ്റപ്പ് ആവർത്തിക്കണം | സംഭരിച്ച site_brief ഉപയോഗിക്കാം |
| വഴക്കം | നിശ്ചിത വർക്ക്ഫ്ലോ | മാറിക്കൊണ്ടിരിക്കുന്ന സംഭാഷണം |

## Setup Assistant ഏജന്റിലേക്ക് മൈഗ്രേറ്റ് ചെയ്യുന്നത് എങ്ങനെ {#migrating-to-setup-assistant-agent}

### പുതിയ സൈറ്റുകൾക്ക് {#for-new-sites}

സൈറ്റ് ബിൽഡർ മോഡ് ഉപയോഗിക്കുന്നതിന് പകരം:

1. ആവശ്യപ്പെടുക: "Help me set up a new site"
2. Setup Assistant ഏജന്റ് നിങ്ങളെ താഴെ പറയുന്ന കാര്യങ്ങളിലൂടെ നയിക്കും:
   - സൈറ്റിന്റെ ഉദ്ദേശ്യവും ലക്ഷ്യങ്ങളും
   - ടാർഗെറ്റ് ഓഡിയൻസ്
   - ബ്രാൻഡ് ഐഡന്റിറ്റി
   - തീം ജനറേഷൻ
   - ആദ്യത്തെ കോൺഫിഗറേഷൻ

### നിലവിലുള്ള സൈറ്റുകൾക്ക് {#for-existing-sites}

സൈറ്റ് ബിൽഡർ മോഡിൽ നിന്ന് നിങ്ങൾക്ക് ഒരു നിലവിലുള്ള സൈറ്റ് ഉണ്ടെങ്കിൽ:

1. നിങ്ങൾക്ക് അത് അതുപോലെ ഉപയോഗിക്കാൻ കഴിയും
2. തീം അപ്ഡേറ്റ് ചെയ്യാൻ, ആവശ്യപ്പെടുക: "Redesign my site"
3. Setup Assistant ഏജന്റ് പുതിയ തീം ഉണ്ടാക്കാൻ നിങ്ങളെ സഹായിക്കും
4. നിങ്ങളുടെ സൈറ്റ് ഡാറ്റ മാറ്റമില്ലാതെ തുടരും

### തീം മാറ്റങ്ങൾക്കായി {#for-theme-changes}

സൈറ്റ് ബിൽഡർ മോഡിന്റെ തീം തിരഞ്ഞെടുക്കുന്നതിന് പകരം:

1. ആവശ്യപ്പെടുക: "Change my theme"
2. Setup Assistant ഏജന്റ് ചെയ്യും:
   - നിങ്ങളുടെ ഡിസൈൻ ഇഷ്ടാനിഷ്ടങ്ങളെക്കുറിച്ച് ചോദിക്കും
   - ഒരു കസ്റ്റം തീം ജനറേറ്റ് ചെയ്യും
   - അത് നിങ്ങളുടെ സൈറ്റിൽ ആക്ടിവേറ്റ് ചെയ്യും

## പ്രധാന വ്യത്യാസങ്ങൾ {#key-differences}

### സൈറ്റ് ബിൽഡർ മോഡ് {#site-builder-mode}

```
1. ഒരു ടെംപ്ലേറ്റ് തിരഞ്ഞെടുക്കുക
2. ഒരു തീം തിരഞ്ഞെടുക്കുക
3. അടിസ്ഥാന ക്രമീകരണങ്ങൾ കോൺഫിഗർ ചെയ്യുക
4. കഴിഞ്ഞു
```

### Setup Assistant ഏജന്റ് {#setup-assistant-agent}

```
1. നിങ്ങളുടെ സൈറ്റിന്റെ ഉദ്ദേശ്യം വിവരിക്കുക
2. നിങ്ങളുടെ ടാർഗെറ്റ് ഓഡിയൻസ് നിർവചിക്കുക
3. ഡിസൈൻ ഇഷ്ടാനിഷ്ടങ്ങൾ തിരഞ്ഞെടുക്കുക
4. ഏജന്റ് കസ്റ്റം തീം ജനറേറ്റ് ചെയ്യുന്നു
5. ഏജന്റ് തീം ആക്ടിവേറ്റ് ചെയ്യുന്നു
6. ഭാവി സെഷനുകൾക്കായി സൈറ്റ് ബ്രീഫ് സംഭരിക്കുന്നു
```

## Setup Assistant ഏജന്റിന്റെ ഗുണങ്ങൾ {#benefits-of-setup-assistant-agent}

### കൂടുതൽ വഴക്കമുള്ളത് (More Flexible) {#more-flexible}

- നിങ്ങളുടെ സൈറ്റ് സ്വാഭാവിക ഭാഷയിൽ വിവരിക്കാം
- കസ്റ്റം ശുപാർശകൾ ലഭിക്കും
- നിങ്ങളുടെ പ്രത്യേക ആവശ്യങ്ങൾക്കനുസരിച്ച് മാറാൻ കഴിയും

### മികച്ച കസ്റ്റമൈസേഷൻ (Better Customization) {#better-customization}

- കസ്റ്റം തീം ജനറേഷൻ
- ഡിസൈൻ സിസ്റ്റം തീരുമാനങ്ങൾ
- സ്ഥിരമായ ഡിസൈൻ ടോക്കണുകൾ

### സ്ഥിരമായ മെമ്മറി (Persistent Memory) {#persistent-memory}

- നിങ്ങളുടെ site_brief സംഭരിക്കപ്പെടുന്നു
- ഭാവി ഏജന്റുമാർക്ക് നിങ്ങളുടെ സൈറ്റിനെ മനസ്സിലാക്കാൻ കഴിയും
- സെറ്റപ്പ് വിവരങ്ങൾ ആവർത്തിക്കേണ്ട ആവശ്യമില്ല

### സംയോജിത വർക്ക്ഫ്ലോ (Integrated Workflow) {#integrated-workflow}

- തീം ബിൽഡർ ഓൺബോർഡിംഗ്
- ഡിസൈൻ സിസ്റ്റം എസ്തറ്റിക്സ് സ്കിൽ
- എബിലിറ്റി വിസിബിലിറ്റി കൺട്രോൾസ്
- എല്ലാം തടസ്സമില്ലാതെ ഒരുമിച്ച് പ്രവർത്തിക്കുന്നു

## ട്രബിൾഷൂട്ടിംഗ് {#troubleshooting}

### എനിക്ക് സൈറ്റ് ബിൽഡർ മോഡ് കണ്ടെത്താൻ കഴിയുന്നില്ല {#i-cant-find-site-builder-mode}

സൈറ്റ് ബിൽഡർ മോഡ് നീക്കം ചെയ്തിരിക്കുന്നു. പകരം Setup Assistant ഏജന്റ് ഉപയോഗിക്കുക:

```
"Help me set up a new site"
```

### സൈറ്റ് ബിൽഡറിൽ നിന്ന് ഒരു സൈറ്റ് പുനഃസൃഷ്ട করতে ഞാൻ ആഗ്രഹിക്കുന്നു {#i-want-to-recreate-a-site-from-site-builder}

നിങ്ങൾക്ക് Setup Assistant ഏജന്റ് ഉപയോഗിച്ച് ഇത് പുനഃസൃഷ്ടിക്കാൻ കഴിയും:

1. ആവശ്യപ്പെടുക: "Help me set up a new site"
2. നിങ്ങളുടെ യഥാർത്ഥ സൈറ്റിന്റെ ഉദ്ദേശ്യവും ഡിസൈനും വിവരിക്കുക
3. ഏജന്റ് സമാനമായ ഒരു തീം ജനറേറ്റ് ചെയ്യും
4. നിങ്ങളുടെ site_brief ഭാവി റഫറൻസിനായി സംഭരിക്കപ്പെടും

### എന്റെ നിലവിലുള്ള സൈറ്റ് ബിൽഡർ സൈറ്റ് പ്രവർത്തിക്കുന്നില്ല {#my-existing-site-builder-site-isnt-working}

സൈറ്റ് ബിൽഡർ മോഡിൽ ഉണ്ടാക്കിയ നിലവിലുള്ള സൈറ്റുകൾ പ്രവർത്തിക്കുന്നത് തുടരും. നിങ്ങൾക്ക് പ്രശ്നങ്ങൾ നേരിടുന്നുണ്ടെങ്കിൽ:

1. നിങ്ങളുടെ തീം ഇപ്പോഴും ആക്റ്റീവ് ആണോ എന്ന് പരിശോധിക്കുക
2. നിങ്ങളുടെ പ്ലഗിനുകൾ എനേബിൾ ചെയ്തിട്ടുണ്ടോ എന്ന് പരിശോധിക്കുക
3. വേർഡ്പ്രസ്സ് എറർ ലോഗുകൾ പരിശോധിക്കുക
4. പ്രശ്നങ്ങൾ തുടരുകയാണെങ്കിൽ സപ്പോർട്ടുമായി ബന്ധപ്പെടുക

### എനിക്ക് എൻ്റെ പഴയ സൈറ്റ് ബിൽഡർ ടെംപ്ലേറ്റുകൾ ഉപയോഗിക്കാമോ? {#can-i-still-use-my-old-site-builder-templates}

സൈറ്റ് ബിൽഡർ ടെംപ്ലേറ്റുകൾ ഇനി ലഭ്യമല്ല. എന്നിരുന്നാലും:

- നിങ്ങളുടെ നിലവിലുള്ള സൈറ്റുകൾ പ്രവർത്തിക്കുന്നത് തുടരും
- Setup Assistant ഏജന്റ് ഉപയോഗിച്ച് സമാനമായ സൈറ്റുകൾ നിങ്ങൾക്ക് പുനഃസൃഷ്ടിക്കാൻ കഴിയും
- Setup Assistant ഏജന്റ് കൂടുതൽ കസ്റ്റമൈസേഷൻ ഓപ്ഷനുകൾ നൽകുന്നു
