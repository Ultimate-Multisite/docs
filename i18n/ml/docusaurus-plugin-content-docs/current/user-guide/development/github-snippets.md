# GitHub റിപ്പോസിറ്ററിയിലെ Ultimate Multisite സ്നിപ്പറ്റുകൾ എങ്ങനെ ഉപയോഗിക്കാം

സൈൻ-അപ്പ് പേജുകളിൽ Google Analytics സ്ക്രിപ്റ്റ് ചേർക്കുക അല്ലെങ്കിൽ അഡ്മിൻ dashboard-ൽ നിന്ന് ഒരു meta box മറയ്ക്കുക തുടങ്ങിയ ചെറിയ ഫീച്ചറുകൾ ചേർക്കാൻ ആഗ്രഹിക്കുന്ന Ultimate Multisite ഉപയോക്താക്കൾ പതിവായി ആവശ്യപ്പെടാറുള്ള കോഡ് സ്നിപ്പറ്റുകൾ GitHub റിപ്പോസിറ്ററിയിൽ ലഭ്യമാണ്.

ഈ കോഡുകൾ എങ്ങനെ ഉപയോഗിക്കാം, കൂടുതൽ കൃത്യമായി പറഞ്ഞാൽ എവിടെ ചേർക്കണം എന്ന് ഈ ലേഖനം കാണിച്ചുതരും.

താഴെയുള്ള ലിങ്കിൽ നിങ്ങൾക്ക് സ്നിപ്പറ്റുകൾ കണ്ടെത്താം.

https://github.com/next-press/wp-ultimo-snippets/

കോഡ് ചേർക്കാൻ 2 വഴികളുണ്ട്

  1. നിങ്ങളുടെ theme-ന്റെ functions.php ഫയലിൽ.

  2. Must-Use Plugins (mu-plugins)

# നിങ്ങളുടെ theme-ന്റെ functions.php ഫയലിൽ സ്നിപ്പറ്റ് എങ്ങനെ ചേർക്കാം.

  1. നിങ്ങളുടെ WordPress Network അഡ്മിൻ dashboard-ൽ ലോഗിൻ ചെയ്ത് Themes > Theme Editor-ലേക്ക് പോകുക (താഴെയുള്ള സ്ക്രീൻഷോട്ട് കാണുക).

  2. "Edit Themes" പേജിൽ, നിങ്ങളുടെ സ്ക്രീനിന്റെ മുകളിൽ വലതുഭാഗത്തുള്ള dropdown ഫീൽഡിൽ നിങ്ങളുടെ സജീവ theme തിരഞ്ഞെടുത്തിട്ടുണ്ടെന്ന് ഉറപ്പാക്കുക (താഴെയുള്ള സ്ക്രീൻഷോട്ടിൽ #3).

  3. ഫയൽ ലോഡ് ചെയ്യാൻ "Theme Files" വിഭാഗത്തിന് കീഴിലുള്ള functions.php ഫയലിൽ ക്ലിക്ക് ചെയ്യുക. താഴേക്ക് സ്ക്രോൾ ചെയ്ത് GitHub റിപ്പോസിറ്ററിയിൽ നിന്ന് ലഭിച്ച Ultimate Multisite സ്നിപ്പറ്റ് പേസ്റ്റ് ചെയ്യുക.

# Must-Use Plugins (mu-plugins) എങ്ങനെ സൃഷ്ടിക്കാം

"Must-Use Plugins" അല്ലെങ്കിൽ ചുരുക്കത്തിൽ "mu-plugins" എന്ന് വിളിക്കുന്ന ഇഷ്ടാനുസൃത ഫങ്ഷണാലിറ്റി ലോഡ് ചെയ്യാൻ അനുവദിക്കുന്ന ഒരു ഫീച്ചർ WordPress-ൽ ഉണ്ട്.

ഈ പ്രത്യേക mu-plugins മറ്റെല്ലാ സാധാരണ plugins-നും മുമ്പായി ലോഡ് ചെയ്യപ്പെടുന്നു, അവ നിഷ്ക്രിയമാക്കാൻ കഴിയില്ല. ഒരു multisite network-ൽ, ഈ mu-plugins-ലെ കോഡ് നിങ്ങളുടെ ഇൻസ്റ്റാളേഷനിലെ എല്ലാ സൈറ്റുകളിലും ലോഡ് ചെയ്യപ്പെടും.

1\. നിങ്ങളുടെ WordPress ഇൻസ്റ്റാളിന്റെ ഫയൽസിസ്റ്റം ആക്സസ് ചെയ്യാൻ FTP അല്ലെങ്കിൽ SSH ഉപയോഗിക്കുക.

2\. നിങ്ങളുടെ WordPress ഇൻസ്റ്റാളിന്റെ wp-content ഡയറക്ടറിക്കുള്ളിൽ, mu-plugins എന്ന പേരിൽ ഒരു പുതിയ ഡയറക്ടറി സൃഷ്ടിക്കുക.

<!-- Screenshot unavailable: File manager showing wp-content directory with mu-plugins folder -->

3\. Notepad അല്ലെങ്കിൽ ഏതെങ്കിലും കോഡ് എഡിറ്റർ ഉപയോഗിച്ച് നിങ്ങളുടെ കമ്പ്യൂട്ടറിൽ wu-snippet.php എന്ന പേരിൽ ഒരു പുതിയ PHP ഫയൽ സൃഷ്ടിക്കുക.

4\. GitHub റിപ്പോസിറ്ററിയിൽ നിന്ന് ലഭിച്ച Ultimate Multisite കോഡ് സ്നിപ്പറ്റ് ഫയലിൽ ചേർത്ത് സേവ് ചെയ്യുക. നിങ്ങളുടെ mu plugin-ന് ലേബൽ ചെയ്യാൻ കോഡ് സ്നിപ്പറ്റിന്റെ മുകളിൽ ഈ കോഡും ചേർക്കാം.
