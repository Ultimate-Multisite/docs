# Ultimate Multisite Hooks-ഉം Filters-ഉം

Ultimate Multisite നിരവധി ഫങ്ഷനുകളും ഡാറ്റാബേസ് ക്വറികളും ഉപയോഗിച്ചാണ് നിർമ്മിച്ചിരിക്കുന്നത്. നിലവിലുള്ള ഫങ്ഷനുകൾ മാറ്റം വരുത്താനോ പുതിയ കോഡ് ചേർക്കാനോ നിങ്ങളെ സഹായിക്കുന്ന hooks-ഉം filters-ഉം ഇതിലുണ്ട്.

പരിചയസമ്പന്നരായ ഉപയോക്താക്കൾക്ക് അവരുടെ നെറ്റ്‌വർക്ക് കൂടുതൽ ഇഷ്ടാനുസൃതമാക്കാൻ സഹായിക്കുന്നതിന്, ഈ hooks-ഉം filters-ഉം ഒരു റഫറൻസ് ലിസ്റ്റായി ഞങ്ങൾ തയ്യാറാക്കിയിട്ടുണ്ട്. താഴെയുള്ള ലിങ്കിൽ ക്ലിക്ക് ചെയ്ത് Github-ൽ നിന്ന് ഈ ലിസ്റ്റ് നിങ്ങൾക്ക് ആക്സസ് ചെയ്യാം.
