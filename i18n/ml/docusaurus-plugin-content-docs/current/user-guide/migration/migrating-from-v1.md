# V1-ൽ നിന്ന് മൈഗ്രേറ്റ് ചെയ്യുന്നതിനുള്ള മാർഗ്ഗനിർദ്ദേശം

## Ultimate Multisite അതിന്റെ യഥാർത്ഥ 1.x റിലീസുകളിൽ നിന്ന് 2.x റിലീസുകളിലേക്ക് മാറിയിരിക്കുന്നു.

Ultimate Multisite വേർഷൻ 2.0-ഉം അതിനു മുകളിലുള്ളവയും കോഡ്‌ബേസിന്റെ പൂർണ്ണമായ പുനർനിർമ്മാണമാണ്, അതിനാൽ പഴയ വേർഷനും പുതിയതും തമ്മിൽ വളരെ കുറച്ച് സാമ്യമേയുള്ളൂ. ഇക്കാരണത്താൽ, 1.x-ൽ നിന്ന് 2.x-ലേക്ക് അപ്‌ഗ്രേഡ് ചെയ്യുമ്പോൾ, പുതിയ വേർഷനുകൾക്ക് മനസ്സിലാക്കാൻ കഴിയുന്ന ഫോർമാറ്റിലേക്ക് നിങ്ങളുടെ ഡാറ്റ മൈഗ്രേറ്റ് ചെയ്യേണ്ടതുണ്ട്.

ഭാഗ്യവശാൽ, Ultimate Multisite 2.0+-ൽ പഴയ വേർഷനിൽ നിന്നുള്ള ഡാറ്റ കണ്ടെത്തി പുതിയ ഫോർമാറ്റിലേക്ക് മാറ്റാൻ കഴിവുള്ള ഒരു **മൈഗ്രേറ്റർ ഉൾപ്പെടുത്തിയിട്ടുണ്ട്**. ഈ മൈഗ്രേഷൻ വേർഷൻ 2.0+-ന്റെ **Setup Wizard**-ൽ വച്ചാണ് നടക്കുന്നത്.

മൈഗ്രേറ്റർ എങ്ങനെ പ്രവർത്തിക്കുന്നു, പരാജയപ്പെട്ടാൽ എന്ത് ചെയ്യണം, ഈ പ്രക്രിയയിൽ ഉണ്ടാകാവുന്ന പ്രശ്നങ്ങൾ എങ്ങനെ പരിഹരിക്കാം എന്നിവയെല്ലാം ഈ പാഠത്തിൽ ഉൾപ്പെടുത്തിയിട്ടുണ്ട്.

_**പ്രധാനം: വേർഷൻ 1.x-ൽ നിന്ന് വേർഷൻ 2.0-ലേക്ക് അപ്‌ഗ്രേഡ് ചെയ്യാൻ തുടങ്ങുന്നതിന് മുമ്പ് നിങ്ങളുടെ സൈറ്റ് ഡാറ്റാബേസിന്റെ ഒരു ബാക്കപ്പ് എടുക്കാൻ മറക്കരുത്**_

## ആദ്യ ഘട്ടങ്ങൾ

ആദ്യ ഘട്ടം plugin .zip ഫയൽ ഡൗൺലോഡ് ചെയ്ത് നിങ്ങളുടെ network admin dashboard-ൽ വേർഷൻ 2.0 ഇൻസ്റ്റാൾ ചെയ്യുക എന്നതാണ്.

നിങ്ങൾ [വേർഷൻ 2.0 ഇൻസ്റ്റാൾ ചെയ്ത് ആക്ടിവേറ്റ് ചെയ്തതിന്](../getting-started/installing-ultimate-multisite.md) ശേഷം, നിങ്ങളുടെ Multisite പഴയ വേർഷനിൽ പ്രവർത്തിക്കുന്നുണ്ടെന്ന് സിസ്റ്റം സ്വയം കണ്ടെത്തുകയും plugin പേജിന്റെ മുകളിൽ ഈ സന്ദേശം കാണിക്കുകയും ചെയ്യും.

_**കുറിപ്പ്:** നിങ്ങളുടെ Multisite-ൽ Ultimate Multisite 1.x ഇൻസ്റ്റാൾ ചെയ്തിട്ടുണ്ടെങ്കിൽ, നിങ്ങൾ ഇപ്പോൾ ഡൗൺലോഡ് ചെയ്ത വേർഷൻ ഉപയോഗിച്ച് നിലവിലുള്ള plugin മാറ്റിസ്ഥാപിക്കാനുള്ള ഓപ്ഷൻ കാണും. ദയവായി **Replace current with uploaded** ക്ലിക്ക് ചെയ്യുക._

<!-- Screenshot unavailable: Plugin page showing option to replace v1 with uploaded v2 version -->

അടുത്ത പേജിൽ വേർഷൻ 1.x-നൊപ്പം നിങ്ങൾ ഇൻസ്റ്റാൾ ചെയ്ത പഴയ add-on-കൾ എന്തൊക്കെയാണെന്ന് കാണിക്കും. നിങ്ങൾ ഉപയോഗിക്കുന്ന വേർഷൻ 2.0-യുമായി യോജിക്കുന്നുണ്ടോ അല്ലെങ്കിൽ മൈഗ്രേഷന് ശേഷം add-on-ന്റെ അപ്‌ഗ്രേഡ് ചെയ്ത വേർഷൻ ഇൻസ്റ്റാൾ ചെയ്യേണ്ടതുണ്ടോ എന്നതിനെക്കുറിച്ചുള്ള നിർദ്ദേശങ്ങളും അതിൽ ഉണ്ടാകും.

<!-- Screenshot unavailable: Message on the top of the plugins page about updating to v2.0, with list of add-ons that need updating -->

മുന്നോട്ട് പോകാൻ തയ്യാറായിക്കഴിഞ്ഞാൽ, **Visit the Installer to finish the upgrade** എന്ന ബട്ടൺ ക്ലിക്ക് ചെയ്യാം.

<!-- Screenshot unavailable: Button saying Visit the Installer to finish the upgrade -->

ഇത് നിങ്ങളെ ചില സ്വാഗത സന്ദേശങ്ങളുള്ള installation wizard പേജിലേക്ക് കൊണ്ടുപോകും. അടുത്ത പേജിലേക്ക് പോകാൻ **Get Started** ക്ലിക്ക് ചെയ്താൽ മതി.

<!-- Screenshot unavailable: Setup Wizard welcoming page with Get Started button -->

**Get Started** ക്ലിക്ക് ചെയ്തതിന് ശേഷം, Pre-install Checks പേജിലേക്ക് റീഡയറക്ട് ചെയ്യും. ഇത് നിങ്ങളുടെ System Information-ഉം WordPress installation-ഉം കാണിക്കുകയും **Ultimate Multisite-ന്റെ ആവശ്യകതകൾ** പാലിക്കുന്നുണ്ടോ എന്ന് അറിയിക്കുകയും ചെയ്യും.

<!-- Screenshot unavailable: Pre-install Checks page showing system requirements confirmation -->

അടുത്ത ഘട്ടം നിങ്ങളുടെ Ultimate Multisite license key നൽകി plugin ആക്ടിവേറ്റ് ചെയ്യുക എന്നതാണ്. ഇത് add-on-കൾ ഉൾപ്പെടെ എല്ലാ ഫീച്ചറുകളും നിങ്ങളുടെ സൈറ്റിൽ ലഭ്യമാണെന്ന് ഉറപ്പാക്കും.

<!-- Screenshot unavailable: License activation page with license key input and Agree and Activate button -->

നിങ്ങളുടെ key നൽകിയ ശേഷം, **Agree & Activate** ക്ലിക്ക് ചെയ്യുക.

License activation-ന് ശേഷം, അടുത്ത പേജിൽ **Install** ക്ലിക്ക് ചെയ്ത് യഥാർത്ഥ ഇൻസ്റ്റലേഷൻ ആരംഭിക്കാം. ഇത് വേർഷൻ 2.0 പ്രവർത്തിക്കാൻ ആവശ്യമായ ഫയലുകളും ഡാറ്റാബേസും സ്വയം സൃഷ്ടിക്കും.

<!-- Screenshot unavailable: Installation page showing what will be updated with Install button -->

## ഇനി, മൈഗ്രേഷൻ

മൈഗ്രേറ്ററിൽ ഒരു സുരക്ഷാ ഫീച്ചർ ഉൾപ്പെടുത്തിയിട്ടുണ്ട്, അത് നിങ്ങളുടെ Ultimate Multisite ഡാറ്റ മുഴുവൻ യാതൊരു പ്രശ്നവുമില്ലാതെ മൈഗ്രേറ്റ് ചെയ്യാൻ കഴിയുമെന്ന് ഉറപ്പാക്കാൻ നിങ്ങളുടെ മുഴുവൻ multisite-ഉം പരിശോധിക്കും. പ്രക്രിയ ആരംഭിക്കാൻ **Run Check** ബട്ടൺ ക്ലിക്ക് ചെയ്യുക.

<!-- Screenshot unavailable: Migration page with Run Check button to verify data can be converted -->

പരിശോധന നടത്തിയ ശേഷം, രണ്ട് സാധ്യതകളുണ്ട്: ഫലം **എറർ ഉള്ളതോ** **എറർ ഇല്ലാത്തതോ** ആകാം.

### എറർ ഉണ്ടെങ്കിൽ

നിങ്ങൾക്ക് ഒരു എറർ സന്ദേശം ലഭിച്ചാൽ, എറർ പരിഹരിക്കാൻ സഹായിക്കുന്നതിന് ഞങ്ങളുടെ support team-നെ ബന്ധപ്പെടേണ്ടതുണ്ട്. ടിക്കറ്റ് സൃഷ്ടിക്കുമ്പോൾ **error log നൽകാൻ മറക്കരുത്**. നിങ്ങൾക്ക് log ഡൗൺലോഡ് ചെയ്യാം അല്ലെങ്കിൽ contact our support team എന്ന ലിങ്ക് ക്ലിക്ക് ചെയ്യാം. ഇത് നിങ്ങളുടെ പേജിന്റെ വലതുവശത്ത് help widget തുറക്കും, description-ന് കീഴിൽ error logs ഉൾപ്പെടെ ഫീൽഡുകൾ മുൻകൂട്ടി പൂരിപ്പിച്ചിട്ടുണ്ടാകും.

_**സിസ്റ്റം ഒരു എറർ കണ്ടെത്തിയതിനാൽ, വേർഷൻ 2.0-ലേക്ക് മൈഗ്രേറ്റ് ചെയ്യാൻ നിങ്ങൾക്ക് കഴിയില്ല. എറർ പരിഹരിക്കുന്നതുവരെ നിങ്ങളുടെ network പ്രവർത്തിപ്പിക്കുന്നത് തുടരാൻ നിങ്ങൾക്ക് വേർഷൻ 1.x-ലേക്ക് റോൾ ബാക്ക് ചെയ്യാം.**_

### എറർ ഇല്ലെങ്കിൽ

സിസ്റ്റം യാതൊരു എററും കണ്ടെത്തിയില്ലെങ്കിൽ, നിങ്ങൾക്ക് ഒരു success സന്ദേശവും താഴെ മൈഗ്രേഷനുമായി മുന്നോട്ട് പോകാൻ അനുവദിക്കുന്ന **Migrate** ബട്ടണും കാണാം. ഈ പേജിൽ, മുന്നോട്ട് പോകുന്നതിന് മുമ്പ് നിങ്ങളുടെ ഡാറ്റാബേസിന്റെ ഒരു ബാക്കപ്പ് സൃഷ്ടിക്കാൻ ഓർമ്മിപ്പിക്കും, ഇത് ഞങ്ങൾ ശക്തമായി ശുപാർശ ചെയ്യുന്നു. നിങ്ങൾക്ക് ഇതിനകം ഒരു ബാക്കപ്പ് ഉണ്ടെങ്കിൽ **Migrate** ക്ലിക്ക് ചെയ്യുക.

ഇത്രമാത്രമേ വേണ്ടൂ!

നിങ്ങളുടെ network-ൽ ലോഗോയും മറ്റ് കാര്യങ്ങളും അപ്‌ഡേറ്റ് ചെയ്യാൻ Wizard setup തുടരാം അല്ലെങ്കിൽ നിങ്ങളുടെ Ultimate Multisite വേർഷൻ 2.0 മെനുവും അതിന്റെ പുതിയ interface-ഉം നാവിഗേറ്റ് ചെയ്യാൻ തുടങ്ങാം. മുന്നോട്ട് പോയി ആസ്വദിക്കൂ.
