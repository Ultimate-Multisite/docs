# PRO Theme ഉപയോഗിച്ച് നിങ്ങളുടെ Admin Panel ഇഷ്ടാനുസൃതമാക്കുക

ക്ലയന്റ് വർക്കുകൾ ചെയ്യുമ്പോൾ, WordPress-ന്റെ admin dashboard-ൽ ഞങ്ങളുടെ ക്ലയന്റുകൾക്ക് ഒരു ഇഷ്ടാനുസൃത അനുഭവം നൽകേണ്ട ആവശ്യം പലപ്പോഴും വരാറുണ്ട്. കൂടുതൽ പ്രൊഫഷണൽ ലുക്കും ഫീലും തേടുകയാണെങ്കിൽ, PRO Theme നിങ്ങളെ സഹായിച്ചേക്കാം.

നിങ്ങളുടെ ക്ലയന്റുകളുടെ Dashboard ഇഷ്ടാനുസൃതമാക്കുമ്പോൾ സഹായിക്കാൻ വികസിപ്പിച്ചെടുത്ത ഒരു admin theme ആണ് PRO Theme. ഇത് Ultimate Multisite-ന്റെ premium plugins core-ന്റെ ഭാഗവുമാണ്.

ഇത് ഇൻസ്റ്റാൾ ചെയ്ത ശേഷം, നിങ്ങളുടെ _Plugins_ പേജിലേക്ക് പോയി Network-ൽ activate ചെയ്യാൻ ക്ലിക്ക് ചെയ്യുക.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Plugins പേജിൽ നെറ്റ്‌വർക്ക്-വൈഡായി activate ചെയ്യുക_

_**ശ്രദ്ധിക്കുക:**_ _ഇത് ഇൻസ്റ്റാൾ ചെയ്യുക മാത്രം ചെയ്താൽ, ആ ഉപയോക്താവിന് മാത്രം theme ചേർക്കാൻ തിരഞ്ഞെടുക്കാവുന്ന ഒരു ഓപ്ഷനായി നിങ്ങളുടെ ക്ലയന്റിന്റെ ഭാഗത്ത് ഇത് കാണിക്കും._

ഇപ്പോൾ, നിങ്ങളുടെ admin interface പൂർണ്ണമായും പുതുക്കിയിരിക്കുന്നു! ഇടത് കോളത്തിൽ PRO Theme എന്ന പേരിൽ ഒരു പുതിയ മെനുവും ഉണ്ട്.
