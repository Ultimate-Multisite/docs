# Ultimate Multisite: Ad Injector ഉപയോഗിച്ച് നിങ്ങളുടെ നെറ്റ്‌വർക്കിൽ പരസ്യങ്ങൾ പ്രദർശിപ്പിക്കുന്നു

Ultimate Multisite: Ad Injector ഒരു **സൗജന്യ add-on** ആണ്, ഇത് നിങ്ങളുടെ പ്ലാനുകളുടെ edit page-ൽ നിന്ന് തന്നെ പരസ്യങ്ങൾ പ്രദർശിപ്പിക്കാനുള്ള ടൂളുകൾ നൽകുന്നു.

Ultimate Multisite: Ad Injector ഇൻസ്റ്റാൾ ചെയ്ത ശേഷം, അത് ആക്ടിവേറ്റ് ചെയ്യാൻ നിങ്ങളുടെ _Plugins_ പേജിലേക്ക് പോകുക.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Network Activate ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക_

add-on ആക്ടീവ് ആയിക്കഴിഞ്ഞാൽ, plan edit സ്ക്രീനിലെ _Advanced Options_ meta box-ൽ _Ad Settings_ എന്ന പുതിയ ഓപ്ഷൻ കാണാം.

![plan edit സ്ക്രീനിലെ Ad Settings](/img/config/product-edit-full.png)

_ഇപ്പോൾ നിങ്ങളുടെ ഓരോ പ്ലാനിന്റെയും ad settings എഡിറ്റ് ചെയ്യാം_

ആ ടാബിൽ, front-end-ലെ posts-ലും pages-ലും, content-ന് മുമ്പും ശേഷവും ad codes ചേർക്കാൻ കഴിയും.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_നിങ്ങളുടെ പരസ്യത്തിന്റെ code ചേർത്തുകഴിഞ്ഞാൽ, അത് ആ നിർദ്ദിഷ്ട പ്ലാനിന്റെ posts-ലും pages-ലും പ്രദർശിപ്പിക്കും._

ആ പ്ലാനിലെ ഉപയോക്താക്കളുടെ Admin Pages-ന്റെ header-ൽ പരസ്യങ്ങൾ ഉൾപ്പെടുത്താനും കഴിയും.

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_നിങ്ങളുടെ ഉപയോക്താവിന്റെ back-end-ൽ പരസ്യങ്ങൾ പ്രദർശിപ്പിക്കണമെങ്കിൽ Admin Panel Ad Code ഓപ്ഷനിലേക്ക് പോകുക_

### പരസ്യത്തിനപ്പുറം

ഉയർന്ന തലത്തിലുള്ള പ്ലാനുകൾക്ക് സപ്പോർട്ട് നൽകാനും Ultimate Multisite: Ad Injector ഉപയോഗിക്കാം.

_Admin Panel Ads_ ഓപ്ഷനിലേക്ക് പോയി നിങ്ങളുടെ support tool-ന്റെ HTML code ചേർക്കുക.
