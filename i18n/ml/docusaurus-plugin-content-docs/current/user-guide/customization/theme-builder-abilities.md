# Theme Builder Abilities: ബ്ലോക്ക് തീമുകൾ രൂപകൽപ്പന ചെയ്യുകയും സജീവമാക്കുകയും ചെയ്യുക {#theme-builder-abilities-scaffold-and-activate-block-themes}

Superdav AI Agent v1.12.0 രണ്ട് ശക്തമായ കഴിവുകൾ അവതരിപ്പിക്കുന്നു. ഇവ ചാറ്റ് ഇന്റർഫേസിൽ നിന്ന് തന്നെ കസ്റ്റം ബ്ലോക്ക് തീമുകൾ സൃഷ്ടിക്കാനും വിന്യസിക്കാനും നിങ്ങളെ സഹായിക്കുന്നു.

## Overview {#overview}

**scaffold-block-theme** ഉം **activate-theme** ഉം എന്നീ കഴിവുകൾ ഏജന്റുമാരെ താഴെ പറയുന്ന കാര്യങ്ങൾ ചെയ്യാൻ പ്രാപ്തരാക്കുന്നു:
- നിങ്ങളുടെ ആവശ്യകതകൾ അനുസരിച്ച് പൂർണ്ണവും, ഉപയോഗിക്കാൻ തയ്യാറായതുമായ ബ്ലോക്ക് തീമുകൾ സൃഷ്ടിക്കുക.
- മാനുവൽ ഇടപെടലുകൾ ഇല്ലാതെ തന്നെ നിങ്ങളുടെ സൈറ്റിൽ തീമുകൾ ഓട്ടോമാറ്റിക്കായി സജീവമാക്കുക.
- മാർഗ്ഗനിർദ്ദേശിക്കപ്പെട്ട ഡിസൈൻ തീരുമാനങ്ങളിലൂടെ ഒരുമിച്ചൊരു വിഷ്വൽ ഐഡന്റിറ്റി സൃഷ്ടിക്കുക.

## Scaffold Block Theme {#scaffold-block-theme}

**scaffold-block-theme** കഴിവ് ഒരു പൂർണ്ണമായ തീം ഘടനയോടുകൂടിയ പുതിയ WordPress ബ്ലോക്ക് തീം സൃഷ്ടിക്കുന്നു. ഇതിൽ ഉൾപ്പെടുന്നത്:

- ഡിസൈൻ ടോക്കണുകളോടുകൂടിയ `theme.json` കോൺഫിഗറേഷൻ
- പൊതുവായ ലേഔട്ടുകൾക്കായുള്ള ബ്ലോക്ക് ടെംപ്ലേറ്റ് ഫയലുകൾ
- കസ്റ്റം ബ്ലോക്ക് സ്റ്റൈലുകളും വേരിയേഷനുകളും
- തീം മെറ്റാഡാറ്റയും സപ്പോർട്ട് ഡിക്ലറേഷനുകളും

### എങ്ങനെ ഉപയോഗിക്കണം (How to Invoke) {#how-to-invoke}

Superdav AI Agent-മായി നിങ്ങൾ നടത്തുന്ന ചാറ്റിൽ, നിങ്ങൾക്ക് തീം സൃഷ്ടിക്കാൻ ആവശ്യപ്പെടാം:

```
"Create a block theme called 'Modern Agency' with a blue and white color scheme,
sans-serif typography, and a professional layout"
```

ഏജന്റ് ചെയ്യുക:
1. സംഭാഷണത്തിലൂടെ നിങ്ങളുടെ ഡിസൈൻ ഇഷ്ടങ്ങൾ ശേഖരിക്കും.
2. പൂർണ്ണമായ തീം ഘടന സൃഷ്ടിക്കും.
3. ആവശ്യമായ എല്ലാ തീം ഫയലുകളും ഉണ്ടാക്കും.
4. സജീവമാക്കുന്നതിനായി തീം തയ്യാറാക്കും.

### പ്രതീക്ഷിക്കുന്ന ഔട്ട്പുട്ട് (Expected Output) {#expected-output}

ഈ കഴിവ് വിജയകരമായി പ്രവർത്തിക്കുമ്പോൾ, നിങ്ങൾക്ക് കാണാൻ കഴിയുന്നത്:

- തീം രൂപകൽപ്പന ചെയ്തുവെന്ന സ്ഥിരീകരണം.
- തീമിന്റെ പേരും ലൊക്കേഷനും.
- പ്രയോഗിച്ച ഡിസൈൻ ടോക്കണുകളുടെ സംഗ്രഹം (നിറങ്ങൾ, ടൈപ്പോഗ്രാഫി, സ്പേസിംഗ്).
- സജീവമാക്കാൻ തയ്യാറായ സ്റ്റാറ്റസ്.

ഉദാഹരണ ഔട്ട്പുട്ട്:
```
✓ Theme "Modern Agency" scaffolded successfully
  Location: /wp-content/themes/modern-agency/
  Colors: Primary #0066CC, Secondary #FFFFFF
  Typography: Inter (sans-serif)
  Status: Ready to activate
```

## Activate Theme {#activate-theme}

**activate-theme** കഴിവ് നിങ്ങളുടെ സൈറ്റിനെ പുതിയതായി രൂപകൽപ്പന ചെയ്തതോ അല്ലെങ്കിൽ നിലവിലുള്ളതോ ആയ ഒരു ബ്ലോക്ക് തീമിലേക്ക് മാറ്റുന്നു.

### എങ്ങനെ ഉപയോഗിക്കണം (How to Invoke) {#how-to-invoke-1}

ഒരു തീം രൂപകൽപ്പന ചെയ്ത ശേഷം, നിങ്ങൾക്ക് അത് ഉടൻ തന്നെ സജീവമാക്കാം:

```
"Activate the Modern Agency theme"
```

അല്ലെങ്കിൽ നിലവിലുള്ള ഏത് തീമും സജീവമാക്കാം:

```
"Switch to the Twentytwentyfour theme"
```

### പ്രതീക്ഷിക്കുന്ന ഔട്ട്പുട്ട് (Expected Output) {#expected-output-1}

സജീവമാക്കൽ വിജയിക്കുമ്പോൾ:

- സജീവമായ തീമിന്റെ സ്ഥിരീകരണം.
- മുമ്പത്തെ തീമിന്റെ പേര് (വിവരാനത്തിനായി).
- തീം ഇപ്പോൾ സജീവമായ സൈറ്റ് URL.
- തീം-നിർദ്ദിഷ്ട സജ്ജീകരണ കുറിപ്പുകൾ.

ഉദാഹരണ ഔട്ട്പുട്ട്:
```
✓ Theme activated successfully
  Active theme: Modern Agency
  Previous theme: Twentytwentyfour
  Live at: https://yoursite.com
  Note: Check your homepage to verify the layout
```

## Workflow: Scaffold and Activate {#workflow-scaffold-and-activate}

ഒരു സാധാരണ വർക്ക്‌ഫ്ലോ ഈ രണ്ട് കഴിവുകളും സംയോജിപ്പിക്കുന്നു:

1. **തീം സൃഷ്ടിക്കാൻ ആവശ്യപ്പെടുക**: "എന്റെ SaaS ല Landing പേജിനായുള്ള ഒരു ബ്ലോക്ക് തീം ഉണ്ടാക്കുക"
2. **ഏജന്റ് തീം രൂപകൽപ്പന ചെയ്യുന്നു**: ഫയലുകളും ഡിസൈൻ ടോക്കണുകളും സൃഷ്ടിക്കുന്നു.
3. **പരിശോധിക്കുകയും മെച്ചപ്പെടുത്തുകയും ചെയ്യുക**: ആവശ്യമെങ്കിൽ ഡിസൈൻ മാറ്റങ്ങളെക്കുറിച്ച് ചർച്ച ചെയ്യുക.
4. **സജീവമാക്കുക**: "Activate the theme now"
5. **പരിശോധിക്കുക**: പുതിയ ഡിസൈൻ സജീവമാണോ എന്ന് ഉറപ്പാക്കാൻ നിങ്ങളുടെ സൈറ്റ് സന്ദർശിക്കുക.

## Design Tokens and Customization {#design-tokens-and-customization}

രൂപകൽപ്പന ചെയ്ത തീമുകൾ (scaffolded themes) ഡിസൈൻ ടോക്കണുകൾ (theme.json വഴി) ഉപയോഗിക്കുന്നു. ഇത് താഴെ പറയുന്ന കാര്യങ്ങൾക്കായി ഉപയോഗിക്കുന്നു:

- **Colors**: Primary, secondary, accent, neutral palette
- **Typography**: Font families, sizes, weights, line heights
- **Spacing**: Padding, margin, gap scales
- **Borders**: Radius and width tokens
- **Shadows**: Elevation levels

ഈ ടോക്കണുകൾ `theme.json`-ൽ കേന്ദ്രീകൃതമായി (centralized) ആയതിനാൽ, ഒരു ഫയലിൽ നിന്ന് തന്നെ നിങ്ങളുടെ മുഴുവൻ ഡിസൈൻ സിസ്റ്റവും എളുപ്പത്തിൽ ക്രമീകരിക്കാൻ കഴിയും.

## Limitations and Notes {#limitations-and-notes}

- തീമുകൾ `/wp-content/themes/`-ൽ രൂപകൽപ്പന ചെയ്യണം, കൂടാതെ WordPress നെയിമിംഗ് കൺവെൻഷനുകൾ പാലിക്കണം.
- സജീവമാക്കുന്നതിന് നിങ്ങളുടെ WordPress സൈറ്റിൽ അനുയോജ്യമായ പെർമിഷനുകൾ ആവശ്യമാണ്.
- തീമുകളിലെ കസ്റ്റം PHP കോഡ് കുറവാണ്; സങ്കീർണ്ണമായ പ്രവർത്തനങ്ങൾക്കായി പ്ലഗിനുകൾ ഉപയോഗിക്കുക.
- ബ്ലോക്ക് തീമുകൾ WordPress 5.9 അല്ലെങ്കിൽ അതിനുശേഷമുള്ള പതിപ്പുകളുമായി ഏറ്റവും നന്നായി പ്രവർത്തിക്കുന്നു.

## Troubleshooting {#troubleshooting}

**Scaffolding ചെയ്ത ശേഷം തീം കാണുന്നില്ല**
- തീം ഡയറക്ടറി നിലവിലുണ്ടോ എന്നും ശരിയായ പെർമിഷനുകൾ ഉണ്ടോ എന്നും പരിശോധിക്കുക.
- `theme.json` സാധുവായ JSON ആണെന്ന് ഉറപ്പാക്കുക.
- തീമിന്റെ പേര് നിലവിലുള്ള തീമുകളുമായി衝突 (conflict) ചെയ്യുന്നില്ലെന്ന് ഉറപ്പാക്കുക.

**Activation പരാജയപ്പെടുന്നു**
- നിങ്ങൾക്ക് അഡ്മിനിസ്ട്രേറ്റർ പെർമിഷനുകൾ ഉണ്ടോയെന്ന് സ്ഥിരീകരിക്കുക.
- തീം ഡയറക്ടറി WordPress-ന് വായിക്കാൻ കഴിയുന്നുണ്ടോയെന്ന് പരിശോധിക്കുക.
- വിശദാംശങ്ങൾക്കായി WordPress error logs പരിശോധിക്കുക.

**Design tokens പ്രയോഗിക്കുന്നില്ല**
- `theme.json` സിന്റാക്സ് ശരിയാണോയെന്ന് പരിശോധിക്കുക.
- ഏതെങ്കിലും കാഷിംഗ് പ്ലഗിനുകൾ ക്ലിയർ ചെയ്യുക.
- നിങ്ങൾ ഉപയോഗിക്കുന്ന ടോക്കണുകൾ നിങ്ങളുടെ WordPress പതിപ്പ് പിന്തുണയ്ക്കുന്നുണ്ടോയെന്ന് പരിശോധിക്കുക.

## Next Steps {#next-steps}

നിങ്ങളുടെ തീം സജീവമാക്കിയ ശേഷം, നിങ്ങൾക്ക് ചെയ്യാൻ കഴിയുന്നത്:
- ടൈപ്പോഗ്രാഫി, നിറങ്ങൾ, സ്പേസിംഗ് എന്നിവ മെച്ചപ്പെടുത്താൻ **Design System Aesthetics skill** ഉപയോഗിക്കുക.
- WordPress ബ്ലോക്ക് എഡിറ്ററിലൂടെ വ്യക്തിഗത ബ്ലോക്ക് സ്റ്റൈലുകൾ കസ്റ്റമൈസ് ചെയ്യുക.
- തീമിന്റെ `style.css` ഫയലിൽ കസ്റ്റം CSS ചേർക്കുക.
- പ്രത്യേക പേജ് തരങ്ങൾക്കായി കസ്റ്റം ബ്ലോക്ക് ടെംപ്ലേറ്റുകൾ സൃഷ്ടിക്കുക.
