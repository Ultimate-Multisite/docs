# ഇന്റർനെറ്റ് സെർച്ച് {#internet-search}

Gratis AI Agent v1.5.0, **ഇന്റർനെറ്റ് സെർച്ച്** എന്ന ഒരു കഴിവ് കൂട്ടിച്ചേർക്കുന്നു. ഇത് AI അസിസ്റ്റന്റിനെ സംഭാഷണത്തിനിടയിൽ വെബ്സൈറ്റിൽ നിന്ന് ഏറ്റവും പുതിയ വിവരങ്ങൾ ശേഖരിക്കാൻ സഹായിക്കുന്നു. ഇത് [Brave Search API](https://brave.com/search/api/) വഴിയോ [Tavily API](https://tavily.com/) വഴിയോ പ്രവർത്തിപ്പിക്കുന്നു.

## ഇത് എങ്ങനെ പ്രവർത്തിക്കുന്നു {#how-it-works}

ഇന്റർനെറ്റ് സെർച്ച് പ്രവർത്തനക്ഷമമാക്കുമ്പോൾ, ചോദ്യത്തിന് നിലവിലെതോ പുറത്തുള്ളതോ ആയ വിവരങ്ങൾ ആവശ്യമാണെന്ന് അസിസ്റ്റന്റ് സ്വയം മനസ്സിലാക്കുകയാണെങ്കിൽ, കോൺഫിഗർ ചെയ്ത സെർച്ച് പ്രൊവൈഡറിൽ നിന്ന് ഇത് ഓട്ടോമാറ്റിക്കായി വിവരങ്ങൾ ശേഖരിക്കും. ഉദാഹരണത്തിന്, സമീപകാല വാർത്തകൾ, ലൈവ് വിലകൾ, അല്ലെങ്കിൽ മോഡലിന്റെ പരിശീലന സമയത്തിനുശേഷം മാറിയേക്കാവുന്ന ഡോക്യുമെന്റേഷൻ എന്നിവയാണിത്.

ഫലങ്ങൾ തത്സമയം ശേഖരിക്കുകയും, പ്രതികരണം നൽകുന്നതിന് മുമ്പ് അസിസ്റ്റന്റിന്റെ സന്ദർഭത്തിലേക്ക് (context) ഉൾപ്പെടുത്തുകയും ചെയ്യുന്നു. ചോദ്യത്തിന് ഉത്തരം നൽകാൻ സെർച്ച് ഫലങ്ങൾ ഉപയോഗിച്ചുവെങ്കിൽ, അസിസ്റ്റന്റ് അത് സൂചിപ്പിക്കും.

## ഇന്റർനെറ്റ് സെർച്ച് പ്രവർത്തനക്ഷമമാക്കൽ {#enabling-internet-search}

ഇന്റർനെറ്റ് സെർച്ച് പ്രവർത്തിപ്പിക്കണമെങ്കിൽ, നിങ്ങൾ തിരഞ്ഞെടുക്കുന്ന സെർച്ച് പ്രൊവൈഡറിൽ നിന്ന് ഒരു API key ആവശ്യമാണ്. ഇത് കോൺഫിഗർ ചെയ്യാൻ:

1. **Gratis AI Agent → Settings → Advanced** എന്നതിലേക്ക് പോകുക.
2. **Internet Search Provider** എന്ന ഡ്രോപ്പ്ഡൗൺ ലിഖിംഗിൽ നിന്ന് **Brave Search** അല്ലെങ്കിൽ **Tavily** എന്നിവയിൽ ഒന്ന് തിരഞ്ഞെടുക്കുക.
3. ബന്ധപ്പെട്ട ഫീൽഡിൽ നിങ്ങളുടെ API key നൽകുക. ഓരോ ഫീൽഡിനരികിലും സൈൻ-അപ്പ് ലിങ്കുകൾ കാണാം.
4. **Save Settings** ക്ലിക്ക് ചെയ്യുക.

key സേവ് ചെയ്തുകഴിഞ്ഞാൽ, ഇന്റർനെറ്റ് സെർച്ച് കഴിവ് അസിസ്റ്റന്റിന് ഓട്ടോമാറ്റിക്കായി ലഭ്യമാകും.

## Brave Search {#brave-search}

### Brave Search API Key നേടുന്നത് {#obtaining-a-brave-search-api-key}

1. [Brave Search API പേജ്](https://brave.com/search/api/) സന്ദർശിക്കുക.
2. ഒരു പ്ലാനിനായി സൈൻ അപ്പ് ചെയ്യുക. പ്രതിമാസ റിക്വസ്റ്റ് പരിധിയോടെ ഒരു ഫ്രീ ടയർ ലഭ്യമാണ്.
3. Brave Search ഡെവലപ്പർ ഡാഷ്‌ബോർഡിൽ നിന്ന് നിങ്ങളുടെ API key കോപ്പി ചെയ്യുക.
4. Gratis AI Agent സജ്ജീകരണങ്ങളിലെ **Brave Search API Key** ഫീൽഡിൽ പേസ്റ്റ് ചെയ്യുക.

### ഉപയോഗ പരിധികൾ (Usage Limits) {#usage-limits}

നിങ്ങൾ ചെയ്യുന്ന ക്വറികളുടെ എണ്ണം അനുസരിച്ചാണ് Brave Search വഴി ബില്ലിംഗ് നടക്കുന്നത്. സെർച്ച് ട്രിഗർ ചെയ്യുന്ന ഓരോ AI പ്രതികരണവും ഒരു ക്വറിയായി കണക്കാക്കും. അപ്രതീക്ഷിത ചാർജുകൾ ഒഴിവാക്കാൻ [Brave Search developer dashboard](https://brave.com/search/api/) വഴി നിങ്ങളുടെ ഉപയോഗം നിരീക്ഷിക്കുക.

## Tavily {#tavily}

Superdav AI Agent v1.10.0, കൂടുതൽ സമ്പന്നമായ സെർച്ച് ഫലങ്ങളും അഡ്വാൻസ്ഡ് റിസേർച്ച് കഴിവുകളും വാഗ്ദാനം ചെയ്യുന്ന ഒരു ബദൽ ഇന്റർനെറ്റ് സെർച്ച് പ്രൊവൈഡറായി **Tavily** കൂട്ടിച്ചേർക്കുന്നു.

### Tavily API Key നേടുന്നത് {#obtaining-a-tavily-api-key}

1. [Tavily API പേജ്](https://tavily.com/) സന്ദർശിക്കുക.
2. ഒരു അക്കൗണ്ടിനായി സൈൻ അപ്പ് ചെയ്യുക. പ്രതിമാസ റിക്വസ്റ്റ് പരിധിയോടെ ഒരു ഫ്രീ ടയർ ലഭ്യമാണ്.
3. Tavily ഡാഷ്‌ബോർഡിൽ നിന്ന് നിങ്ങളുടെ API key കോപ്പി ചെയ്യുക.
4. Gratis AI Agent സജ്ജീകരണങ്ങളിലെ **Tavily API Key** ഫീൽഡിൽ പേസ്റ്റ് ചെയ്യുക.

### ഉപയോഗ പരിധികൾ (Usage Limits) {#usage-limits-1}

നിങ്ങൾ നടത്തുന്ന API കോളുകളുടെ എണ്ണം അനുസരിച്ചാണ് Tavily വഴി ബില്ലിംഗ് നടക്കുന്നത്. സെർച്ച് ട്രിഗർ ചെയ്യുന്ന ഓരോ AI പ്രതികരണവും ഒരു കോളായി കണക്കാക്കും. അപ്രതീക്ഷിത ചാർജുകൾ ഒഴിവാക്കാൻ [Tavily dashboard](https://tavily.com/) വഴി നിങ്ങളുടെ ഉപയോഗം നിരീക്ഷിക്കുക.

## ഇന്റർനെറ്റ് സെർച്ച് പ്രവർത്തനരഹിതമാക്കൽ {#disabling-internet-search}

ആക്ടീവ് സെർച്ച് പ്രൊവൈഡർ ഫീൽഡിൽ നിന്ന് API key നീക്കം ചെയ്യുകയും സേവ് ചെയ്യുകയും ചെയ്യുക. ഇന്റർനെറ്റ് സെർച്ച് കഴിവ് അസിസ്റ്റന്റിന് আর ലഭിക്കില്ല.

:::note
സഹായകരമായ മറുപടികൾ നൽകുന്നതിന് അസിസ്റ്റന്റ് സെർച്ച് ഫലങ്ങൾക്കായി കാത്തിരിക്കേണ്ടതിനാൽ, ഇന്റർനെറ്റ് സെർച്ച് പ്രതികരണങ്ങളിൽ കാലതാമസം (latency) ഉണ്ടാക്കുന്നു. സമയബന്ധിതമായ ഉപയോഗങ്ങൾക്കായി, തത്സമയ സെർച്ച് ആവശ്യമുണ്ടോ അതോ അസിസ്റ്റന്റിന്റെ ബിൽറ്റ്-ഇൻ അറിവ് മതിയാകുമോ എന്ന് പരിഗണിക്കുക.
:::
