# പ്ലാനുകൾക്കുള്ള പങ്കിടാവുന്ന ലിങ്കുകൾ കൈകാര്യം ചെയ്യുന്നത്

നിങ്ങളുടെ ക്ലയന്റുകളുമായി പ്ലാൻ ലിങ്കുകൾ നേരിട്ട് പങ്കിടാനുള്ള സൗകര്യം Ultimate Multisite നിങ്ങൾക്ക് നൽകുന്നു. ഇത് ചെയ്യാൻ, നിങ്ങളുടെ ഏതെങ്കിലും പ്ലാനിന്റെ എഡിറ്റ് പേജിലേക്ക് പോയി _Click to copy the Shareable Link_ ബട്ടൺ ക്ലിക്ക് ചെയ്യുക.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_പ്ലാനിന്റെ എഡിറ്റ് പേജിൽ, പങ്കിടാവുന്ന ലിങ്കുകൾ സൃഷ്ടിക്കാനുള്ള ബട്ടൺ നിങ്ങൾക്ക് കാണാം_

ഈ സംവിധാനം ലിങ്ക് നിർമ്മിക്കാൻ ഡിഫോൾട്ട് ബില്ലിംഗ് ഫ്രീക്വൻസി ഉപയോഗിക്കുന്നു എന്ന കാര്യം ശ്രദ്ധിക്കുക. അതായത്, നിങ്ങളുടെ ഡിഫോൾട്ട് ബില്ലിംഗ് മാസംതോറും ആണെങ്കിൽ, ഉദാഹരണത്തിന്, അവസാനം നിങ്ങൾക്ക് ഇതുപോലൊരു ലിങ്ക് ലഭിക്കും: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

പങ്കിടാവുന്ന ലിങ്ക് ക്വാർട്ടർലി ആയി ഡിഫോൾട്ട് ചെയ്യണമെങ്കിൽ, അവസാനം 3 ചേർക്കേണ്ടതുണ്ട്

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

വാർഷിക ഷെഡ്യൂളിൽ _premium_ പ്ലാൻ തിരഞ്ഞെടുക്കാൻ, 1 എന്ന് ചേർത്ത് തുടർന്ന് ഈ വാക്ക് ചേർക്കുക:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

വ്യത്യസ്ത ബില്ലിംഗ് ഫ്രീക്വൻസികൾക്ക് ഒരൊറ്റ ലിങ്ക് മാത്രം ഉണ്ടാക്കാൻ കഴിയില്ല എന്ന കാര്യം ഓർക്കുക. നിങ്ങളുടെ ഓരോ ബില്ലിംഗ് ഫ്രീക്വൻസിക്കും പ്രത്യേകം ഒരു ലിങ്ക് പങ്കിടണം.

നിങ്ങൾ കസ്റ്റമേഴ്സുമായി ലിങ്ക് പങ്കിട്ടുകഴിഞ്ഞാൽ, അത് അവരെ നിങ്ങളുടെ സൈൻഅപ്പ് ഫ്ലോയുടെ ആദ്യ ഘട്ടത്തിലേക്ക് റീഡയറക്ട് ചെയ്യും.
