# സൈറ്റ് ടെംപ്ലേറ്റുകളിൽ സ്വയമേവ തിരയലും മാറ്റിസ്ഥാപിക്കലും (v2) {#auto-search-and-replace-in-site-templates-v2}

_**ഈ ട്യൂട്ടോറിയലിന് WP UItimo പതിപ്പ് 2.x ആവശ്യമാണ്.**_

Ultimate Multisite-ന്റെ ഏറ്റവും ശക്തമായ സവിശേഷതകളിലൊന്നാണ് രജിസ്ട്രേഷൻ ഫോമിൽ ഇഷ്ടാനുസൃത ടെക്സ്റ്റ്, നിറം, തിരഞ്ഞെടുപ്പ് ഫീൽഡുകൾ ചേർക്കാനുള്ള കഴിവ്. ആ ഡാറ്റ പിടിച്ചെടുത്താൽ, തിരഞ്ഞെടുത്ത സൈറ്റ് ടെംപ്ലേറ്റിലെ ചില ഭാഗങ്ങളിലെ ഉള്ളടക്കം മുൻകൂട്ടി പൂരിപ്പിക്കാൻ അത് ഉപയോഗിക്കാം. തുടർന്ന് പുതിയ സൈറ്റ് പ്രസിദ്ധീകരിക്കുമ്പോൾ, രജിസ്ട്രേഷൻ സമയത്ത് നൽകിയ യഥാർത്ഥ വിവരങ്ങൾ ഉപയോഗിച്ച് Ultimate Multisite പ്ലേസ്‌ഹോൾഡറുകൾ മാറ്റിസ്ഥാപിക്കും.

ഉദാഹരണത്തിന്, പ്ലേസ്‌ഹോൾഡറുകൾ ഉപയോഗിച്ച് നിങ്ങളുടെ ടെംപ്ലേറ്റ് സൈറ്റുകൾ തയ്യാറാക്കാം. പ്ലേസ്‌ഹോൾഡറുകൾ ഇരട്ട കർളി ബ്രേസുകൾക്കുള്ളിൽ ചേർക്കണം - {{placeholder_name}}.

തുടർന്ന്, ആ ഡാറ്റ പിടിച്ചെടുക്കാൻ പൊരുത്തപ്പെടുന്ന ഒരു രജിസ്ട്രേഷൻ ഫീൽഡ് എളുപ്പത്തിൽ ചേർക്കാം

നിങ്ങളുടെ ഉപഭോക്താവിന് രജിസ്ട്രേഷൻ സമയത്ത് ആ ഫീൽഡ് പൂരിപ്പിക്കാൻ കഴിയും.

തുടർന്ന് Ultimate Multisite ഉപഭോക്താവ് നൽകിയ ഡാറ്റ ഉപയോഗിച്ച് പ്ലേസ്‌ഹോൾഡറുകൾ സ്വയമേവ മാറ്റിസ്ഥാപിക്കും.

## **"പ്ലേസ്‌ഹോൾഡറുകൾ നിറഞ്ഞ ടെംപ്ലേറ്റ്" പ്രശ്നം പരിഹരിക്കൽ** {#solving-the-template-full-of-placeholders-problem}

ഇതെല്ലാം മികച്ചതാണ്, പക്ഷേ ഒരു മോശം പ്രശ്നം ഉണ്ടാകുന്നു: നമ്മുടെ ഉപഭോക്താക്കൾക്ക് സന്ദർശിക്കാനാകുന്ന സൈറ്റ് ടെംപ്ലേറ്റുകൾ ഇപ്പോൾ ഒന്നും കാര്യമായി പറയാത്ത മോശം പ്ലേസ്‌ഹോൾഡറുകൾ കൊണ്ട് നിറഞ്ഞിരിക്കുന്നു.

അത് പരിഹരിക്കാൻ, പ്ലേസ്‌ഹോൾഡറുകൾക്ക് വ്യാജ മൂല്യങ്ങൾ സജ്ജമാക്കാനുള്ള ഓപ്ഷൻ ഞങ്ങൾ നൽകുന്നു. നിങ്ങളുടെ ഉപഭോക്താക്കൾ ടെംപ്ലേറ്റ് സൈറ്റുകൾ സന്ദർശിക്കുമ്പോൾ, അവയിലെ ഉള്ളടക്കം തിരഞ്ഞ് മാറ്റിസ്ഥാപിക്കാൻ ഞങ്ങൾ ആ മൂല്യങ്ങൾ ഉപയോഗിക്കുന്നു.

**Ultimate Multisite > Settings > Sites** എന്നതിലേക്ക് പോയി, Site Template Options ഭാഗത്തേക്ക് സ്ക്രോൾ ചെയ്ത്, തുടർന്ന് **Edit Placeholders** ലിങ്ക് ക്ലിക്ക് ചെയ്‌താൽ ടെംപ്ലേറ്റ് പ്ലേസ്‌ഹോൾഡർ എഡിറ്റർ ആക്സസ് ചെയ്യാം.

![Sites ക്രമീകരണ പേജിലെ സൈറ്റ് ടെംപ്ലേറ്റ് ഓപ്ഷൻസിന്റെ ഭാഗം](/img/config/settings-sites-templates-section.png)

അത് നിങ്ങളെ പ്ലേസ്‌ഹോൾഡറുകളുടെ ഉള്ളടക്ക എഡിറ്ററിലേക്ക് കൊണ്ടുപോകും. അവിടെ നിങ്ങൾക്ക് പ്ലേസ്‌ഹോൾഡറുകളും അവയ്ക്കുള്ള അനുബന്ധ ഉള്ളടക്കവും ചേർക്കാം.

![ടെംപ്ലേറ്റ് പ്ലേസ്‌ഹോൾഡർ എഡിറ്ററിലേക്കുള്ള പ്രവേശന സ്ഥലം](/img/config/settings-sites-templates-section.png)
