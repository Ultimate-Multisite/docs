# സൈറ്റ് ടെംപ്ലേറ്റുകളിൽ ഓട്ടോ സെർച്ച് ആൻഡ് റീപ്ലേസ് (v2)

_**ഈ ട്യൂട്ടോറിയലിന് WP UItimo version 2.x ആവശ്യമാണ്.**_

Ultimate Multisite-ന്റെ ഏറ്റവും ശക്തമായ ഫീച്ചറുകളിലൊന്ന്, രജിസ്ട്രേഷൻ ഫോമിലേക്ക് ടെക്സ്റ്റ്, കളർ, സെലക്ട് ഫീൽഡുകൾ എന്നിവ ആവശ്യാനുസരണം ചേർക്കാനുള്ള കഴിവാണ്. ആ ഡാറ്റ ശേഖരിച്ചുകഴിഞ്ഞാൽ, തിരഞ്ഞെടുത്ത സൈറ്റ് ടെംപ്ലേറ്റിന്റെ ചില ഭാഗങ്ങളിൽ ഉള്ളടക്കം മുൻകൂട്ടി നിറയ്ക്കാൻ നമുക്ക് അത് ഉപയോഗിക്കാം. പുതിയ സൈറ്റ് പബ്ലിഷ് ചെയ്യുമ്പോൾ, രജിസ്ട്രേഷൻ സമയത്ത് നൽകിയ യഥാർത്ഥ വിവരങ്ങൾ കൊണ്ട് Ultimate Multisite പ്ലേസ്‌ഹോൾഡറുകളെ മാറ്റിസ്ഥാപിക്കും.

ഉദാഹരണത്തിന്, നിങ്ങളുടെ ടെംപ്ലേറ്റ് സൈറ്റുകളിൽ പ്ലേസ്‌ഹോൾഡറുകൾ ഉപയോഗിക്കാം. പ്ലേസ്‌ഹോൾഡറുകൾ ഇരട്ട കർളി ബ്രേസുകൾക്കുള്ളിൽ ചേർക്കണം - {{placeholder_name}}.

അതിനുശേഷം, ആ ഡാറ്റ ശേഖരിക്കാൻ അനുയോജ്യമായ ഒരു രജിസ്ട്രേഷൻ ഫീൽഡ് ലളിതമായി ചേർക്കാം

രജിസ്ട്രേഷൻ സമയത്ത് നിങ്ങളുടെ ഉപഭോക്താവിന് ആ ഫീൽഡ് പൂരിപ്പിക്കാൻ കഴിയും.

ഉപഭോക്താവ് നൽകിയ ഡാറ്റ ഉപയോഗിച്ച് Ultimate Multisite പ്ലേസ്‌ഹോൾഡറുകളെ സ്വയമേവ മാറ്റിസ്ഥാപിക്കും.

## **"പ്ലേസ്‌ഹോൾഡറുകൾ നിറഞ്ഞ ടെംപ്ലേറ്റ്" എന്ന പ്രശ്നം പരിഹരിക്കൽ**

ഇതെല്ലാം നല്ലതാണ്, പക്ഷേ നമുക്ക് ഒരു അഭംഗിയായ പ്രശ്നം നേരിടേണ്ടിവരുന്നു: ഇപ്പോൾ നമ്മുടെ സൈറ്റ് ടെംപ്ലേറ്റുകൾ - ഉപഭോക്താക്കൾക്ക് സന്ദർശിക്കാൻ കഴിയുന്നവ - കാര്യമായി ഒന്നും പറയാത്ത വിരൂപമായ പ്ലേസ്‌ഹോൾഡറുകൾ നിറഞ്ഞതാണ്.

ഇത് പരിഹരിക്കാൻ, പ്ലേസ്‌ഹോൾഡറുകൾക്ക് ഫേക്ക് വാല്യൂകൾ സെറ്റ് ചെയ്യാനുള്ള ഓപ്ഷൻ ഞങ്ങൾ നൽകുന്നു, നിങ്ങളുടെ ഉപഭോക്താക്കൾ സന്ദർശിക്കുമ്പോൾ ടെംപ്ലേറ്റ് സൈറ്റുകളിൽ അവയുടെ ഉള്ളടക്കങ്ങൾ സെർച്ച് ആൻഡ് റീപ്ലേസ് ചെയ്യാൻ ഞങ്ങൾ ആ വാല്യൂകൾ ഉപയോഗിക്കുന്നു.

**Ultimate Multisite > Settings > Sites** എന്നതിലേക്ക് പോയി, സൈഡ്ബാറിൽ Edit Placeholders ലിങ്കിൽ ക്ലിക്ക് ചെയ്താൽ ടെംപ്ലേറ്റ് പ്ലേസ്‌ഹോൾഡർ എഡിറ്ററിലേക്ക് നിങ്ങൾക്ക് ആക്സസ് ലഭിക്കും.

![Sites സെറ്റിംഗ്സ് സൈഡ്ബാറിലെ Edit Placeholders ലിങ്ക്](/img/config/settings-sites.png)

അത് നിങ്ങളെ പ്ലേസ്‌ഹോൾഡർ കണ്ടന്റ് എഡിറ്ററിലേക്ക് കൊണ്ടുപോകും, അവിടെ നിങ്ങൾക്ക് പ്ലേസ്‌ഹോൾഡറുകളും അവയുടെ അനുബന്ധ ഉള്ളടക്കവും ചേർക്കാം.

![പ്ലേസ്‌ഹോൾഡർ പേരുകളും വാല്യൂകളും ഉള്ള പ്ലേസ്‌ഹോൾഡർ കണ്ടന്റ് എഡിറ്റർ](/img/config/settings-sites.png)
