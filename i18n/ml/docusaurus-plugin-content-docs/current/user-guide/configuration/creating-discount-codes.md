# ഡിസ്കൗണ്ട് കോഡുകൾ സൃഷ്ടിക്കൽ (v2)

_**പ്രധാന കുറിപ്പ്: ഈ ലേഖനം Ultimate Multisite പതിപ്പ് 2.x-നെ സംബന്ധിച്ചുള്ളതാണ്.**_

Ultimate Multisite ഉപയോഗിച്ച് നിങ്ങൾക്ക് ഡിസ്കൗണ്ട് കോഡുകൾ സൃഷ്ടിച്ച് നിങ്ങളുടെ ക്ലയന്റുകൾക്ക് അവരുടെ സബ്സ്ക്രിപ്ഷനുകളിൽ കിഴിവുകൾ നൽകാം. ഇവ സൃഷ്ടിക്കുന്നത് വളരെ എളുപ്പവുമാണ്!

## ഡിസ്കൗണ്ട് കോഡുകൾ സൃഷ്ടിക്കലും എഡിറ്റ് ചെയ്യലും

ഒരു ഡിസ്കൗണ്ട് കോഡ് സൃഷ്ടിക്കാനോ എഡിറ്റ് ചെയ്യാനോ, **Ultimate Multisite > Discount Codes** എന്നതിലേക്ക് പോകുക.

![](/img/config/discount-codes-list.png)

അവിടെ നിങ്ങൾ മുമ്പ് സൃഷ്ടിച്ച ഡിസ്കൗണ്ട് കോഡുകളുടെ ഒരു ലിസ്റ്റ് കാണാം.

ഒരു പുതിയ കൂപ്പൺ സൃഷ്ടിക്കാൻ **Add Discount** **Code** ക്ലിക്ക് ചെയ്യാം, അല്ലെങ്കിൽ നിലവിലുള്ളവയുടെ മുകളിൽ മൗസ് കൊണ്ടുവന്ന് **Edit** ക്ലിക്ക് ചെയ്ത് എഡിറ്റ് ചെയ്യാം.

![Hover actions ഉള്ള ഡിസ്കൗണ്ട് കോഡുകളുടെ ലിസ്റ്റ്](/img/config/discount-codes-list.png)

![Add Discount Code ബട്ടൺ](/img/config/discount-codes-list.png)

നിങ്ങളുടെ കൂപ്പൺ കോഡ് സൃഷ്ടിക്കുകയോ എഡിറ്റ് ചെയ്യുകയോ ചെയ്യുന്ന പേജിലേക്ക് നിങ്ങളെ റീഡയറക്ട് ചെയ്യും. ഈ ഉദാഹരണത്തിൽ നമ്മൾ ഒരു പുതിയത് സൃഷ്ടിക്കും.

![ഡിസ്കൗണ്ട് കോഡ് എഡിറ്റ് പേജ്](/img/config/discount-codes-list.png)

ഇവിടെ ലഭ്യമായ ക്രമീകരണങ്ങൾ നോക്കാം:

**Enter Discount Code:** ഇത് നിങ്ങളുടെ ഡിസ്കൗണ്ട് കോഡിന്റെ പേര് മാത്രമാണ്. checkout ഫോമിൽ നിങ്ങളുടെ ഉപഭോക്താക്കൾ ഉപയോഗിക്കേണ്ട കോഡ് ഇതല്ല.

**Description:** ഈ കൂപ്പൺ എന്തിനുവേണ്ടിയാണെന്ന് ഇവിടെ ചുരുക്കമായി വിവരിക്കാം.

![ഡിസ്കൗണ്ട് കോഡ് പേരും വിവരണ ഫീൽഡുകളും](/img/config/discount-codes-list.png)

**Coupon code:** checkout സമയത്ത് നിങ്ങളുടെ ഉപഭോക്താക്കൾ നൽകേണ്ട കോഡ് ഇവിടെയാണ് നിർവചിക്കുന്നത്.

![Coupon code ഫീൽഡ്](/img/config/discount-codes-list.png)

**Discount:** ഇവിടെ, നിങ്ങളുടെ ഡിസ്കൗണ്ട് കോഡിന് **ശതമാനം** അല്ലെങ്കിൽ **നിശ്ചിത തുക** സജ്ജീകരിക്കാം.

![ഡിസ്കൗണ്ട് ശതമാനം അല്ലെങ്കിൽ നിശ്ചിത തുക ക്രമീകരണം](/img/config/discount-codes-list.png)

**Apply to renewals:** ഈ ഓപ്ഷൻ ഓഫ് ആണെങ്കിൽ, ഈ ഡിസ്കൗണ്ട് കോഡ് **ആദ്യ പേയ്മെന്റിന് മാത്രം** ബാധകമാകും. മറ്റെല്ലാ പേയ്മെന്റുകൾക്കും കിഴിവ് ഉണ്ടാകില്ല. ഈ ഓപ്ഷൻ ഓൺ ആണെങ്കിൽ, ഭാവിയിലെ എല്ലാ പേയ്മെന്റുകൾക്കും ഡിസ്കൗണ്ട് കോഡ് സാധുവായിരിക്കും.

**Setup fee discount:** ഈ ഓപ്ഷൻ ഓഫ് ആണെങ്കിൽ, കൂപ്പൺ കോഡ് ഓർഡറിന്റെ **setup fee-യിൽ ഒരു കിഴിവും നൽകില്ല**. ഈ ഓപ്ഷൻ ഓൺ ആണെങ്കിൽ, നിങ്ങളുടെ പ്ലാനുകളുടെ setup fee-യിൽ ഈ കൂപ്പൺ കോഡ് നൽകുന്ന കിഴിവ് (ശതമാനം അല്ലെങ്കിൽ നിശ്ചിത തുക) സജ്ജീകരിക്കാം.

![Apply to renewals, setup fee discount ഓപ്ഷനുകൾ](/img/config/discount-codes-list.png)

**Active:** ഈ കൂപ്പൺ കോഡ് മാനുവലായി ആക്ടിവേറ്റ് അല്ലെങ്കിൽ ഡീആക്ടിവേറ്റ് ചെയ്യുക.

![ഡിസ്കൗണ്ട് കോഡിനുള്ള Active toggle](/img/config/discount-codes-list.png)

**Advanced Optio** **ns** -ന് കീഴിൽ, ഇനിപ്പറയുന്ന ക്രമീകരണങ്ങൾ ഉണ്ട്:

**Limit uses:**

  * **Uses:** ഡിസ്കൗണ്ട് കോഡ് എത്ര തവണ ഉപയോഗിച്ചുവെന്ന് ഇവിടെ കാണാം.

  * **Max uses:** ഉപയോക്താക്കൾക്ക് ഈ ഡിസ്കൗണ്ട് കോഡ് എത്ര തവണ ഉപയോഗിക്കാമെന്ന് ഇത് പരിമിതപ്പെടുത്തും. ഉദാഹരണത്തിന്, ഇവിടെ 10 എന്ന് നൽകിയാൽ, കൂപ്പൺ 10 തവണ മാത്രമേ ഉപയോഗിക്കാൻ കഴിയൂ. ഈ പരിധി കഴിഞ്ഞാൽ, കൂപ്പൺ കോഡ് ഇനി ഉപയോഗിക്കാൻ കഴിയില്ല.

![Uses, max uses ഫീൽഡുകളുള്ള Limit uses ക്രമീകരണം](/img/config/discount-codes-list.png)**Start & expiration dates:** നിങ്ങളുടെ കൂപ്പണിന് ഒരു ആരംഭ തീയതിയും/അല്ലെങ്കിൽ കാലാവധി അവസാന തീയതിയും ചേർക്കാനുള്ള ഓപ്ഷൻ ഇവിടെയുണ്ട്.

![ആരംഭ, കാലാവധി അവസാന തീയതി ഫീൽഡുകൾ](/img/config/discount-codes-list.png)

**Limit products:** **Select products** ഓൺ ചെയ്താൽ, നിങ്ങളുടെ എല്ലാ ഉൽപ്പന്നങ്ങളും കാണിക്കും. ഏത് ഉൽപ്പന്നത്തിന് ഈ കൂപ്പൺ കോഡ് സ്വീകരിക്കാമെന്ന് മാനുവലായി തിരഞ്ഞെടുക്കാം (ഓൺ അല്ലെങ്കിൽ ഓഫ് ചെയ്ത്). ഇവിടെ ഓഫ് ചെയ്ത ഉൽപ്പന്നങ്ങൾക്ക് നിങ്ങളുടെ ഉപഭോക്താക്കൾ ഈ കൂപ്പൺ കോഡ് ഉപയോഗിക്കാൻ ശ്രമിച്ചാലും ഒരു മാറ്റവും കാണില്ല.

![Limit products തിരഞ്ഞെടുപ്പ് toggles](/img/config/discount-codes-list.png)

ഈ ഓപ്ഷനുകളെല്ലാം സജ്ജീകരിച്ച ശേഷം, നിങ്ങളുടെ കൂപ്പൺ സേവ് ചെയ്യാൻ **Save Discount Code** ക്ലിക്ക് ചെയ്യുക, അത്രമാത്രം!

![Save Discount Code ബട്ടൺ](/img/config/discount-codes-list.png)

കൂപ്പൺ ഇപ്പോൾ നിങ്ങളുടെ ലിസ്റ്റിലുണ്ട്, അവിടെ നിന്ന് അത് **എഡിറ്റ് അല്ലെങ്കിൽ ഡിലീറ്റ്** ചെയ്യാൻ ക്ലിക്ക് ചെയ്യാം.

![എഡിറ്റ്, ഡിലീറ്റ് ഓപ്ഷനുകളുള്ള ലിസ്റ്റിലെ ഡിസ്കൗണ്ട് കോഡ്](/img/config/discount-codes-list.png)

### URL Parameters ഉപയോഗിക്കൽ:

നിങ്ങളുടെ pricing tables ഇഷ്ടാനുസൃതമാക്കാനോ നിങ്ങളുടെ വെബ്സൈറ്റിനായി ഒരു നല്ല കൂപ്പൺ കോഡ് പേജ് നിർമ്മിക്കാനോ നിങ്ങളുടെ checkout ഫോമിലേക്ക് സ്വയമേവ ഒരു ഡിസ്കൗണ്ട് കോഡ് ചേർക്കാനോ ആഗ്രഹിക്കുന്നുവെങ്കിൽ, URL parameters വഴി ഇത് ചെയ്യാം.

ആദ്യം, നിങ്ങളുടെ പ്ലാനിന്റെ shareable link ലഭിക്കണം. ഇതിനായി, **Ultimate Multisite > Products** എന്നതിലേക്ക് പോയി ഒരു പ്ലാൻ തിരഞ്ഞെടുക്കുക.

**Click to Copy Shareable Link** ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക. ഇത് ഈ നിർദ്ദിഷ്ട പ്ലാനിലേക്കുള്ള shareable link നൽകും. ഞങ്ങളുടെ കാര്യത്തിൽ, ലഭിച്ച shareable link [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/)_ ആയിരുന്നു._

![Shareable link ബട്ടണുള്ള Product പേജ്](/img/config/products-list.png)

ഈ നിർദ്ദിഷ്ട പ്ലാനിലേക്ക് നിങ്ങളുടെ ഡിസ്കൗണ്ട് കോഡ് ചേർക്കാൻ, URL-ലേക്ക് **?discount_code=XXX** എന്ന parameter ചേർക്കുക. ഇവിടെ **XXX** എന്നത് കൂപ്പൺ കോഡാണ്.

ഈ ഉദാഹരണത്തിൽ, ഈ നിർദ്ദിഷ്ട ഉൽപ്പന്നത്തിലേക്ക് **50OFF** എന്ന കൂപ്പൺ കോഡ് ചേർക്കും.

50OFF ഡിസ്കൗണ്ട് കോഡ് ചേർത്ത ഈ നിർദ്ദിഷ്ട പ്ലാനിന്റെ URL ഇങ്ങനെയായിരിക്കും: [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/) _**?discount_code=50OFF**_.
