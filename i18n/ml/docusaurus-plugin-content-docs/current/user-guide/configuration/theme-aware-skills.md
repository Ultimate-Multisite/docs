# തീം അനുസരിച്ചുള്ള കഴിവുകൾ (Theme-Aware Skills) {#theme-aware-skills}

Superdav AI Agent v1.10.0-ൽ നാല് പുതിയ തീം-അനുസരിച്ചുള്ള (theme-aware) ബിൽറ്റ്-ഇൻ കഴിവുകൾ ഉൾപ്പെടുത്തിയിരിക്കുന്നു. നിങ്ങളുടെ സജീവമായ WordPress തീമിന് അനുസരിച്ച് ഇവ സ്വയം ക്രമീകരിക്കപ്പെടും. നിങ്ങളുടെ തീമിന്റെ ആർക്കിടെക്ചറിനും ഫീച്ചറുകൾക്കും അനുയോജ്യമായ പ്രത്യേക മാർഗ്ഗനിർദ്ദേശങ്ങളും കഴിവുകളും ഈ കഴിവുകൾ നൽകുന്നു.

## തീം-അനുസരിച്ചുള്ള കഴിവുകൾ എന്നാൽ എന്ത്? {#what-are-theme-aware-skills}

തീം-അനുസരിച്ചുള്ള കഴിവുകൾ (Theme-aware skills) എന്നത്, നിങ്ങളുടെ സൈറ്റിൽ നിലവിൽ സജീവമായ WordPress തീമിനെ അടിസ്ഥാനമാക്കി AI അസിസ്റ്റന്റ് സ്വയമേവ തിരഞ്ഞെടുക്കുന്ന, മുൻകൂട്ടി സജ്ജീകരിച്ച അറിവിന്റെ ശേഖരങ്ങളും ടൂൾ സെറ്റുകളുമാണ്. നിങ്ങൾ തീമുകൾ മാറ്റുമ്പോൾ, അസിസ്റ്റന്റിന്റെ ലഭ്യമായ കഴിവുകൾ സ്വയമേവ അപ്ഡേറ്റ് ചെയ്യപ്പെടും — ഇതിനായി പ്രത്യേക കോൺഫിഗറേഷൻ ആവശ്യമില്ല.

ഓരോ കഴിവിലും ഇവ ഉൾപ്പെടുന്നു:

- **തീം-നിർദ്ദിഷ്ട ഡോക്യുമെന്റേഷൻ** — തീം ഉപയോഗിക്കുന്നതിനും കസ്റ്റമൈസ് ചെയ്യുന്നതിനും വേണ്ട മാർഗ്ഗനിർദ്ദേശങ്ങൾ
- **ബ്ലോക്ക്, പാറ്റേൺ റഫറൻസുകൾ** — ലഭ്യമായ ബ്ലോക്കുകൾ, പാറ്റേണുകൾ, ഡിസൈൻ ഓപ്ഷനുകൾ
- **കസ്റ്റമൈസേഷൻ ഉദാഹരണങ്ങൾ** — സാധാരണ ജോലികൾക്കുള്ള കോഡ് സ്നിപ്പെറ്റുകളും കോൺഫിഗറേഷൻ പാറ്റേണുകളും
- **മികച്ച രീതികൾ (Best practices)** — തീമിന്റെ ആർക്കിടെക്ചറിസിനും വർക്ക്ഫ്ലോയ്ക്കും വേണ്ട ശുപാർശകൾ

## ലഭ്യമായ തീം-അനുസരിച്ചുള്ള കഴിവുകൾ {#available-theme-aware-skills}

### ബ്ലോക്ക് തീമുകൾ (Block Themes) {#block-themes}

**ഇതിന് ബാധകം:** WordPress ബ്ലോക്ക് അടിസ്ഥാനമാക്കിയുള്ള (Full Site Editing) ആർക്കിടെക്ചർ ഉപയോഗിക്കുന്ന തീമുകൾ.

ബ്ലോക്ക് തീമുകൾ കഴിവ് (Block Themes skill) നൽകുന്നത് താഴെ പറയുന്ന കാര്യങ്ങളിലുള്ള മാർഗ്ഗനിർദ്ദേശങ്ങളാണ്:

- ബ്ലോക്ക് എഡിറ്റർ ഉപയോഗിച്ച് ടെംപ്ലേറ്റുകൾ സൃഷ്ടിക്കുന്നതിനും എഡിറ്റ് ചെയ്യുന്നതിനും
- ബ്ലോക്ക് പാറ്റേണുകളുമായും പുനരുപയോഗിക്കാവുന്ന ബ്ലോക്കുകളുമായും പ്രവർത്തിക്കുന്നതിനും
- ഗ്ലോബൽ സ്റ്റൈലുകളും theme.json ക്രമീകരണങ്ങളും കസ്റ്റമൈസ് ചെയ്യുന്നതിനും
- തീം ബ്ലോക്കുകളും വേരിയേഷനുകളും ഉപയോഗിക്കുന്നതിനും
- നിങ്ങളുടെ സൈറ്റിനായി കസ്റ്റം ബ്ലോക്ക് പാറ്റേണുകൾ നിർമ്മിക്കുന്നതിനും

**സ്വയമേവ സജീവമാവുന്നത്:** നിങ്ങളുടെ സജീവമായ തീം ഒരു ബ്ലോക്ക് തീം ആണെങ്കിൽ (ഇത് `block-templates` ഫീച്ചർ പിന്തുണയ്ക്കുന്നു).

### ക്ലാസിക് തീമുകൾ (Classic Themes) {#classic-themes}

**ഇതിന് ബാധകം:** PHP ടെംപ്ലേറ്റുകളും ക്ലാസിക് എഡിറ്ററും ഉപയോഗിക്കുന്ന പരമ്പരാഗത WordPress തീമുകൾ.

ക്ലാസിക് തീമുകൾ കഴിവ് (Classic Themes skill) നൽകുന്നത് താഴെ പറയുന്ന കാര്യങ്ങളിലുള്ള മാർഗ്ഗനിർദ്ദേശങ്ങളാണ്:

- PHP ടെംപ്ലേറ്റ് ഫയലുകളുമായും ഹുക്കുകളുമായും (hooks) പ്രവർത്തിക്കുന്നതിനും
- കസ്റ്റമൈസറിലൂടെ (Customizer) തീമിന്റെ രൂപം കസ്റ്റമൈസ് ചെയ്യുന്നതിനും
- 위ഡ്ജെറ്റ് ഏരിയകളും സൈഡ്ബാർസും ഉപയോഗിക്കുന്നതിനും
- CSS മാറ്റുന്നതിനും ചൈൽഡ് തീം ഡെവലപ്‌മെന്റിനും
- തീം ഹൈറാർക്കി, ടെംപ്ലേറ്റ് ടാഗുകൾ എന്നിവ മനസ്സിലാക്കുന്നതിനും

**സ്വയമേവ സജീവമാവുന്നത്:** നിങ്ങളുടെ സജീവമായ തീം ഒരു ക്ലാസിക് (non-block) തീം ആണെങ്കിൽ.

### Kadence Blocks {#kadence-blocks}

**ഇതിന് ബാധകം:** അഡ്വാൻസ്ഡ് ബ്ലോക്ക് അടിസ്ഥാനമാക്കിയുള്ള ഡിസൈനിനായി Kadence Blocks plugin ഉപയോഗിക്കുന്ന സൈറ്റുകൾ.

Kadence Blocks കഴിവ് (Kadence Blocks skill) നൽകുന്നത് താഴെ പറയുന്ന കാര്യങ്ങളിലുള്ള മാർഗ്ഗനിർദ്ദേശങ്ങളാണ്:

- Kadence-ന്റെ അഡ്വാൻസ്ഡ് ബ്ലോക്ക് ലൈബ്രറി ഉപയോഗിക്കുന്നത് (Hero, Testimonials, Pricing, മുതലായവ)
- Kadence ബ്ലോക്ക് ക്രമീകരണങ്ങളും റെസ്‌പോൺസീവ് ഓപ്ഷനുകളും കോൺഫിഗർ ചെയ്യുന്നത്
- Kadence-ന്റെ ഗ്രിഡ്, കണ്ടെയ്‌നർ ബ്ലോക്കുകൾ ഉപയോഗിച്ച് കസ്റ്റം ലേഔട്ടുകൾ നിർമ്മിക്കുന്നത്
- നിങ്ങളുടെ തീമുമായി Kadence ബ്ലോക്കുകൾ സംയോജിപ്പിക്കുന്നത്
- Kadence-ന്റെ ഡിസൈൻ സിസ്റ്റവും പ്രീസെറ്റുകളും ഉപയോഗപ്പെടുത്തുന്നത്

**സ്വയമേവ സജീവമാവുന്നത്:** നിങ്ങളുടെ സൈറ്റിൽ Kadence Blocks plugin സജീവമാകുമ്പോൾ.

### Kadence Theme {#kadence-theme}

**ഇതിന് ബാധകം:** ബ്ലോക്ക് അടിസ്ഥാനമാക്കിയുള്ള ഡിസൈനിനും കസ്റ്റമൈസേഷനുമായി Kadence തീം ഉപയോഗിക്കുന്ന സൈറ്റുകൾ.

Kadence Theme കഴിവ് (Kadence Theme skill) നൽകുന്നത് താഴെ പറയുന്ന കാര്യങ്ങളിലുള്ള മാർഗ്ഗനിർദ്ദേശങ്ങളാണ്:

- ഗ്ലോബൽ സ്റ്റൈലുകൾ വഴിയും theme.json വഴിയും Kadence തീം കസ്റ്റമൈസ് ചെയ്യുന്നത്
- Kadence-ന്റെ ബിൽറ്റ്-ഇൻ ബ്ലോക്ക് പാറ്റേണുകളും ടെംപ്ലേറ്റുകളും ഉപയോഗിക്കുന്നത്
- Kadence തീം ക്രമീകരണങ്ങളും ഓപ്ഷനുകളും കോൺഫിഗർ ചെയ്യുന്നത്
- Kadence-ന്റെ ഡിസൈൻ സിസ്റ്റം ഉപയോഗിച്ച് കസ്റ്റം ഡിസൈനുകൾ നിർമ്മിക്കുന്നത്
- ജനപ്രിയ പ്ലഗിനുകളുമായും ടൂളുകളുമായും Kadence സംയോജിപ്പിക്കുന്നത്

**സ്വയമേവ സജീവമാവുന്നത്:** Kadence തീം നിങ്ങളുടെ സജീവമായ തീം ആണെങ്കിൽ.

## കഴിവുകൾ എങ്ങനെ തിരഞ്ഞെടുക്കപ്പെടുന്നു {#how-skills-are-selected}

ഓരോ സന്ദേശത്തിലും അസിസ്റ്റന്റ് നിങ്ങളുടെ സജീവമായ തീമും ഇൻസ്റ്റാൾ ചെയ്ത പ്ലഗിനുകളും സ്വയമേവ കണ്ടെത്തുന്നു. ഒരു അനുയോജ്യമായ തീം-അനുസരിച്ചുള്ള കഴിവ് ലഭ്യമാണെങ്കിൽ, അത് അസിസ്റ്റന്റിന്റെ സന്ദർഭത്തിലേക്ക് (context) സ്വയമേവ ലോഡ് ചെയ്യപ്പെടും. കഴിവുകൾを手വഴിയായി പ്രവർത്തനക്ഷമമാക്കാനോ മാറ്റാനോ നിങ്ങൾക്ക് ആവശ്യമില്ല.

### ഒന്നിലധികം കഴിവുകൾ (Multiple Skills) {#multiple-skills}

നിങ്ങളുടെ സൈറ്റിന് ഒന്നിലധികം കഴിവുകൾ ബാധകമാണെങ്കിൽ (ഉദാഹരണത്തിന്, നിങ്ങൾക്ക് Kadence Blocks-ഉം Kadence Theme-ഉം സജീവമാണെങ്കിൽ), അസിസ്റ്റന്റിന് എല്ലാ ബാധകമായ കഴിവുകളിലേക്കും പ്രവേശനം ഉണ്ടായിരിക്കും, കൂടാതെ ഓരോന്നിന്റെയും മാർഗ്ഗനിർദ്ദേശങ്ങൾ റഫർ ചെയ്യാൻ കഴിയും.

### തീമുകൾ മാറ്റുമ്പോൾ (Switching Themes) {#switching-themes}

നിങ്ങൾ നിങ്ങളുടെ സജീവമായ തീം മാറ്റുമ്പോൾ, അടുത്ത സന്ദേശത്തിൽ അസിസ്റ്റന്റിന്റെ ലഭ്യമായ കഴിവുകൾ സ്വയമേവ അപ്ഡേറ്റ് ചെയ്യപ്പെടും. ഉദാഹരണത്തിന്:

1. നിങ്ങൾ **Block Themes** കഴിവ് സജീവമാക്കി ഒരു ബ്ലോക്ക് തീം ഉപയോഗിക്കുകയാണ്.
2. നിങ്ങൾ ഒരു ക്ലാസിക് തീമിലേക്ക് മാറുന്നു.
3. നിങ്ങളുടെ അടുത്ത സന്ദേശത്തിൽ, **Classic Themes** കഴിവ് സ്വയമേവ ലോഡ് ചെയ്യപ്പെടുകയും, **Block Themes** കഴിവ് ഇനി ലഭ്യമാകാതിരിക്കുകയും ചെയ്യും.

## തീം-അനുസരിച്ചുള്ള കഴിവുകൾ ഉപയോഗിക്കൽ {#using-theme-aware-skills}

ഒരു തീം-അനുസരിച്ചുള്ള കഴിവ് ഉപയോഗിക്കണമെങ്കിൽ, ചാറ്റ് ഇന്റർഫേസിൽ നിങ്ങൾ എന്ത് ചെയ്യാൻ ആഗ്രഹിക്കുന്നു എന്ന് ലളിതമായി വിവരിക്കുക. അസിസ്റ്റന്റ് സ്വയമേവ അനുയോജ്യമായ കഴിവின் മാർഗ്ഗനിർദ്ദേശങ്ങൾ റഫർ ചെയ്യും.

### ഉദാഹരണ പ്രോംപ്റ്റുകൾ (Example Prompts) {#example-prompts}

**Block Themes-ന് വേണ്ടി:**
> "ബ്ലോക്ക് പാറ്റേണുകൾ ഉപയോഗിച്ച് ഒരു ബാക്ക്ഗ്രൗണ്ട് ഇമേജോടും സെൻട്രൽ ടെക്സ്റ്റോടും കൂടിയ ഒരു ഹീറോ സെക്ഷൻ ഉണ്ടാക്കുക."

**Classic Themes-ന് വേണ്ടി:**
> "ഒരു ചൈൽഡ് തീം ഉപയോഗിച്ച് സൈഡ്ബാറിൽ ഒരു കസ്റ്റം 위ഡ്ജെറ്റ് ഏരിയ ചേർക്കുക."

**Kadence Blocks-ന് വേണ്ടി:**
> "Kadence Testimonials ബ്ലോക്ക് ഉപയോഗിച്ച് ഒരു സാക്ഷ്യപത്രങ്ങൾ (testimonials) സെക്ഷൻ നിർമ്മിക്കുക."

**Kadence Theme-ന് വേണ്ടി:**
> "ഹെഡർ ലേഔട്ടും नेവിഗേഷൻ മെനു സ്റ്റൈലിംഗും കസ്റ്റമൈസ് ചെയ്യുക."

നിങ്ങളുടെ സജീവമായ തീമിനും പ്ലഗിനുകൾക്കും അനുയോജ്യമായ തീം-നിർദ്ദിഷ്ട മാർഗ്ഗനിർദ്ദേശങ്ങളും കോഡ് ഉദാഹരണങ്ങളും അസിസ്റ്റന്റ് നൽകും.
