# പതിവായി ചോദിക്കുന്ന ചോദ്യങ്ങൾ

സപ്പോർട്ട് അഭ്യർത്ഥനകളിൽ സാധാരണയായി ഉയർന്നുവരുന്ന ചില **പതിവ് ചോദ്യങ്ങൾ** ഇതാ.

  * **Ultimate Multisite എന്താണ്?**

Ultimate Multisite ഒരു WordPress plugin ആണ്, WordPress multisite ഇൻസ്റ്റാളുകൾക്കായി നിർമ്മിച്ചത്, നിങ്ങളുടെ WordPress ഇൻസ്റ്റാളിനെ [WordPress.com](https://WordPress.com) പോലെ ഒരു പ്രീമിയം സൈറ്റുകളുടെ നെറ്റ്‌വർക്കാക്കി മാറ്റുന്നു – ക്ലയന്റുകൾക്ക് പ്രതിമാസ, ത്രൈമാസ, അല്ലെങ്കിൽ വാർഷിക ഫീസ് വഴി സൈറ്റുകൾ സൃഷ്ടിക്കാൻ അനുവദിക്കുന്നു (സൗജന്യ പ്ലാനുകളും സൃഷ്ടിക്കാം).

  * **Multisite ഇല്ലാതെ Ultimate Multisite പ്രവർത്തിക്കുമോ?**

ഇല്ല. Ultimate Multisite-ന് WordPress multisite, subdirectory, അല്ലെങ്കിൽ subdomain ആവശ്യമാണ്.

  * **Ultimate Multisite-ന്റെ ആവശ്യകതകൾ എന്തൊക്കെയാണ്?**

Ultimate Multisite-ന് ആവശ്യമാണ്:

**WordPress പതിപ്പ്** : v5.3+ (ശുപാർശ ചെയ്യുന്നത്: അവസാന സ്ഥിര പതിപ്പ്)  
**PHP പതിപ്പ്** : 7.4.x (> 8.0-ലേക്കുള്ള പിന്തുണ ഉടൻ വരുന്നു)  
**MySQL പതിപ്പ്** : v5+ (ശുപാർശ ചെയ്യുന്നത്: പതിപ്പ് 5.6, നിങ്ങളുടെ ഹോസ്റ്റിംഗ് ദാതാവിൽ 8.0 ലഭ്യമല്ലെങ്കിൽ)

_Multisite (subdomain അല്ലെങ്കിൽ subdirectory) കൂടി സജീവമാക്കേണ്ടതുണ്ട്._ [WordPress Multisite ഇൻസ്റ്റാൾ ചെയ്യുന്നതെങ്ങനെ](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **Subdirectories ഉള്ള Multisite-ൽ Ultimate Multisite പ്രവർത്തിക്കുമോ?**

അതെ. Ultimate Multisite subdirectory ഇൻസ്റ്റാളുകളും subdirectory ഇൻസ്റ്റാളുകളും പിന്തുണയ്ക്കുന്നു.

  * **Subdomains ഉള്ള Multisite-ൽ Ultimate Multisite പ്രവർത്തിക്കുമോ?**

അതെ. Ultimate Multisite subdirectory ഇൻസ്റ്റാളുകളും subdomain ഇൻസ്റ്റാളുകളും പിന്തുണയ്ക്കുന്നു.

  * **ഉപയോക്താക്കൾക്ക് ഏതൊക്കെ Plugins-ഉം Themes-ഉം ഉപയോഗിക്കാമെന്ന് പരിമിതപ്പെടുത്താൻ Ultimate Multisite ഉപയോഗിക്കാമോ?**

അതെ. Ultimate Multisite ഉപയോഗിച്ച് നിങ്ങളുടെ നെറ്റ്‌വർക്കിലെ ഓരോ പ്ലാനിനും ഏതൊക്കെ plugins-ഉം themes-ഉം ലഭ്യമാണെന്ന് തിരഞ്ഞെടുക്കാം.

  * **ഒരു ഉപയോക്താവിന് തന്റെ സൈറ്റിൽ സൃഷ്ടിക്കാവുന്ന പോസ്റ്റുകളുടെ എണ്ണം പരിമിതപ്പെടുത്താൻ Ultimate Multisite ഉപയോഗിക്കാമോ?**

അതെ. Ultimate Multisite ഉപയോഗിച്ച് നിങ്ങളുടെ നെറ്റ്‌വർക്കിലെ **ലഭ്യമായ എല്ലാ post type-നും** പോസ്റ്റ് ക്വോട്ടകൾ സ്ഥാപിക്കാം. അഡ്മിന് ക്ഷണിക്കാവുന്ന ഉപയോക്താക്കളുടെ എണ്ണം, **ഡിസ്ക് സ്പേസിന്റെ അളവ്,** എന്നിവയും മറ്റും പരിമിതപ്പെടുത്താനും കഴിയും.

  * **Ultimate Multisite Site Templates പിന്തുണയ്ക്കുന്നുണ്ടോ?**

അതെ. Ultimate Multisite ഉപയോഗിച്ച് നിങ്ങൾക്ക് ആവശ്യമുള്ളത്ര Site Templates സൃഷ്ടിക്കാം. Site Templates themes അല്ല, അവ നിങ്ങളുടെ നെറ്റ്‌വർക്കിലെ സാധാരണ സൈറ്റുകളാണ്. നിങ്ങൾക്ക് plugins, themes എന്നിവ സജീവമാക്കാനും ആ Site Template നിങ്ങൾക്ക് ആവശ്യമുള്ള രീതിയിൽ ഇഷ്ടാനുസൃതമാക്കാനും കഴിയും. ഒരു ഉപയോക്താവ് ആ template തിരഞ്ഞെടുത്താൽ, അയാളുടെ സൈറ്റ് എല്ലാ ഇഷ്ടാനുസൃതമാക്കലുകളോടും കൂടി ആ template-ന്റെ ഒരു പകർപ്പായി സൃഷ്ടിക്കപ്പെടും.

ഒരു പ്ലാനിന് ഒരു പ്രത്യേക template സജ്ജമാക്കാനും കഴിയും. അങ്ങനെ ചെയ്യുമ്പോൾ, ആ പ്ലാനിൽ സൈൻ അപ്പ് ചെയ്യുന്ന ഓരോ ഉപയോക്താവിന്റെയും സൈറ്റ് ആ Site Template-ന്റെ ഒരു ഡ്യൂപ്ലിക്കേറ്റായി സൃഷ്ടിക്കപ്പെടും.

  * **Ultimate Multisite WooCommerce-മായി സംയോജിക്കുന്നുണ്ടോ?**

അതെ!

നിങ്ങളുടെ നെറ്റ്‌വർക്ക് ഉപയോക്താക്കൾക്ക് WooCommerce ലഭ്യമാക്കാനും സ്റ്റോറുകൾ സജ്ജമാക്കാൻ അനുവദിക്കാനും നിങ്ങൾ ആഗ്രഹിക്കുന്നുവെങ്കിൽ, Ultimate Multisite ഉപയോഗിച്ച് അത് സാധ്യമാണ്. ഓരോ പ്ലാൻ ടയറിലും നിങ്ങളുടെ ക്ലയന്റുകൾക്ക് സൃഷ്ടിക്കാവുന്ന ഉൽപ്പന്നങ്ങളുടെ എണ്ണം പോലും പരിമിതപ്പെടുത്താം.

Ultimate Multisite-നുള്ള സബ്‌സ്ക്രിപ്ഷൻ പേയ്‌മെന്റുകളുടെ കാര്യത്തിൽ, നിങ്ങളുടെ Ultimate Multisite ക്ലയന്റുകൾക്ക് നിങ്ങളുടെ മെയിൻ സൈറ്റിലെ WooCommerce gateways ഉപയോഗിച്ച് പണം നൽകാൻ അനുവദിക്കുന്നതിന് [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) ഉപയോഗിക്കാം. ഈ രീതിയിൽ, WooCommerce-നായി വികസിപ്പിച്ച payment gateways-ന്റെ വലിയ ലൈബ്രറി ഉപയോഗിച്ച് നിങ്ങളുടെ Ultimate Multisite ക്ലയന്റുകൾക്ക് ബിൽ ചെയ്യാം.

  * **WPMUDEV Pro Sites-ൽ നിന്ന് subscriptions മൈഗ്രേറ്റ് ചെയ്യാൻ വഴിയുണ്ടോ?**

അതെ, Ultimate Multisite ഇപ്പോൾ നിങ്ങളുടെ Pro Sites നെറ്റ്‌വർക്ക് [മൈഗ്രേറ്റ് ചെയ്യാൻ സഹായകരമായ ഒരു ടൂൾ](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) വാഗ്ദാനം ചെയ്യുന്നു.

  * **എന്റെ Plan Edit Page-ൽ "Allowed Plugins" ടാബ് ശൂന്യമായിരിക്കുന്നത് എന്തുകൊണ്ട്?**

നിങ്ങളുടെ നെറ്റ്‌വർക്കിൽ network-activated അല്ലാത്ത plugins-ലേക്ക് മാത്രമേ Ultimate Multisite ആക്‌സസ് അനുവദിക്കാനോ നിരസിക്കാനോ അനുവദിക്കൂ. ഒരു plugin network-activated ആകുമ്പോൾ, അതിന്റെ സ്വഭാവത്താൽ തന്നെ, നിങ്ങളുടെ നെറ്റ്‌വർക്കിലെ എല്ലാ സൈറ്റിലും അത് ഡിഫോൾട്ടായി ലോഡ് ചെയ്യപ്പെടും.

"Allowed Plugins" ടാബിൽ ഒരു plugins-ഉം ലിസ്റ്റ് ചെയ്തിരിക്കുന്നില്ലെങ്കിൽ, ആവശ്യമുള്ള plugin network-active ആണോ അല്ലയോ എന്ന് പരിശോധിക്കുക.

  * **എനിക്ക് സ്വന്തം Pricing Table ഡിസൈൻ ഉപയോഗിക്കാമോ?**

അതെ. 1.4.0 പതിപ്പ് മുതൽ, Ultimate Multisite template overrides പിന്തുണയ്ക്കുന്നു.

  * **WP Engine-ൽ Ultimate Multisite പ്രവർത്തിക്കുമോ?**

അതെ. Ultimate Multisite എല്ലായ്പ്പോഴും WP Engine പിന്തുണച്ചിരുന്നു, പക്ഷേ domain mapping പ്രവർത്തിപ്പിക്കുന്നതിന് അഡ്മിനിൽ നിന്ന് മാനുവൽ അധിക ഘട്ടങ്ങൾ ആവശ്യമായിരുന്നു (WP Engine പാനലിലേക്ക് domain ചേർക്കൽ). **1.6.0 പതിപ്പ് മുതൽ ഇത് ഇനി ആവശ്യമില്ല.** Ultimate Multisite WP Engine API-യുമായി നേരിട്ട് സംയോജിക്കുന്നു, mapping-ന് ശേഷം domains പാനലിലേക്ക് സ്വയമേവ ചേർക്കപ്പെടുന്നു.

  * **നിങ്ങൾ ശുപാർശ ചെയ്യുന്ന hosting provider ഏതാണ്?**

Ultimate Multisite ഏത് hosting provider-ലും പ്രവർത്തിക്കണം. ഒരേയൊരു വ്യത്യാസം Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane, [Runcloud.io](http://Runcloud.io) തുടങ്ങിയ ചില hosting കമ്പനികൾക്ക് domain mapping കഴിവുകൾക്കായി കൂടുതൽ ഇഴുകിച്ചേർന്ന സംയോജനമുണ്ട്, അതിനാൽ അവ ബോക്സിൽ നിന്ന് തന്നെ പ്രവർത്തിക്കുന്നു. മറ്റ് hostings-ന്, നിങ്ങൾക്ക് domain mapping മാനുവലായി സജ്ജമാക്കേണ്ടി വന്നേക്കാം.
