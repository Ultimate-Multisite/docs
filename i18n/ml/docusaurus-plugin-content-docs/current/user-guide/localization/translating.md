# Ultimate Multisite വിവർത്തനം ചെയ്യുന്നു

Ultimate Multisite ഇംഗ്ലീഷിൽ വികസിപ്പിച്ചതാണ്, പക്ഷേ ഈ പ്ലഗിൻ പൂർണ്ണമായും വിവർത്തനം ചെയ്യാൻ കഴിയുന്നതാണ്, വിവർത്തന സംഭാവനകളെ ഞങ്ങൾ സജീവമായി സ്വാഗതം ചെയ്യുന്നു.

## കോർ പ്ലഗിൻ വിവർത്തനങ്ങൾ

കോർ Ultimate Multisite പ്ലഗിൻ ഔദ്യോഗിക **WordPress.org വിവർത്തന പ്ലാറ്റ്ഫോം** വഴിയാണ് വിവർത്തനം ചെയ്യുന്നത്:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

ഇത് സ്റ്റാൻഡേർഡ് WordPress കമ്മ്യൂണിറ്റി വിവർത്തന സംവിധാനമാണ്. വിവർത്തനങ്ങൾ ആവശ്യമായ അംഗീകാര പരിധിയിലെത്തുമ്പോൾ, WordPress പ്ലഗിൻ അപ്ഡേറ്റുകൾ വഴി എല്ലാ Ultimate Multisite ഉപയോക്താക്കൾക്കും അവ സ്വയമേവ വിതരണം ചെയ്യപ്പെടും.

**നിങ്ങളുടെ സഹായം ആവശ്യമാണ്!** ഇംഗ്ലീഷല്ലാത്ത മറ്റൊരു ഭാഷ നിങ്ങൾക്ക് അറിയാമെങ്കിൽ, ദയവായി വിവർത്തനങ്ങൾ സംഭാവന ചെയ്യാൻ പരിഗണിക്കുക. കുറച്ച് strings വിവർത്തനം ചെയ്താൽ പോലും അത് സഹായകരമാണ്. ആരംഭിക്കാൻ:

1. [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/) സന്ദർശിക്കുക
2. നിങ്ങളുടെ ഭാഷ തിരഞ്ഞെടുക്കുക
3. "Contribute Translation" ക്ലിക്ക് ചെയ്യുക
4. നിങ്ങൾക്ക് ഒരു സൗജന്യ WordPress.org അക്കൗണ്ട് ആവശ്യമാണ്

നിങ്ങളുടെ ഭാഷയ്ക്കുള്ള ഒരു വിവർത്തന എഡിറ്ററായി (വിവർത്തനങ്ങൾ അംഗീകരിക്കാൻ കഴിയുന്ന) മാറാൻ നിങ്ങൾ ആഗ്രഹിക്കുന്നെങ്കിൽ, [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) വഴി ഞങ്ങളെ ബന്ധപ്പെടുക.

## Addon വിവർത്തനങ്ങൾ

Addon പ്ലഗിനുകൾ പ്രത്യേകമായി ഇവിടെ വിവർത്തനം ചെയ്യുന്നു:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Addon വിവർത്തനങ്ങൾ നിലവിൽ ഒരു ആരംഭ പോയിന്റായി AI-ജനറേറ്റ് ചെയ്തവയാണ്. അവ എല്ലാ പിന്തുണയ്ക്കുന്ന ഭാഷകളും ഉൾക്കൊള്ളുന്നു, പക്ഷേ അസ്വാഭാവിക വാക്യഘടനയോ തെറ്റുകളോ ഉണ്ടായേക്കാം. അവ കൂടുതൽ സ്വാഭാവികവും കൃത്യവുമാക്കാൻ മനുഷ്യ വിവർത്തകർ അവ അവലോകനം ചെയ്ത് മെച്ചപ്പെടുത്തലുകൾ നിർദ്ദേശിക്കുന്നതിനെ ഞങ്ങൾ സ്വാഗതം ചെയ്യുന്നു.

## ലോക്കൽ വിവർത്തനങ്ങൾ

അപ്സ്ട്രീമിലേക്ക് സംഭാവന ചെയ്യുന്നതിനു പകരം നിങ്ങളുടെ സ്വന്തം പ്രോജക്ടിനായി ലോക്കലായി വിവർത്തനങ്ങൾ സൃഷ്ടിക്കാൻ നിങ്ങൾ താൽപ്പര്യപ്പെടുന്നെങ്കിൽ, രണ്ട് ഓപ്ഷനുകളുണ്ട്:

### Poedit ഉപയോഗിക്കുന്നു

`.pot` ഫയൽ വിവർത്തനം ചെയ്യാൻ **[Poedit](https://poedit.net)** ഡെസ്ക്ടോപ്പ് ആപ്ലിക്കേഷൻ ഉപയോഗിക്കാം:

1. [poedit.net](https://poedit.net)-ൽ നിന്ന് Poedit ഡൗൺലോഡ് ചെയ്യുക
2. `wp-content/plugins/ultimate-multisite/lang/`-ൽ നിന്ന് `.pot` ഫയൽ തുറക്കുക
3. Poedit ഇന്റർഫേസിൽ strings വിവർത്തനം ചെയ്യുക
4. സേവ് ചെയ്യുക -- Poedit `.po`, `.mo` ഫയലുകൾ സൃഷ്ടിക്കും
5. ഔട്ട്പുട്ട് ഫയലുകൾ `lang` ഫോൾഡറിൽ വയ്ക്കുക

### Loco Translate ഉപയോഗിക്കുന്നു

**[Loco Translate](https://wordpress.org/plugins/loco-translate/)** WordPress പ്ലഗിൻ ഉപയോഗിച്ച് ഡെസ്ക്ടോപ്പ് ടൂളുകൾ ഇല്ലാതെ നേരിട്ട് അഡ്മിൻ പാനലിൽ നിന്ന് വിവർത്തനം ചെയ്യാം. വിവർത്തന ഫയലുകൾ സൃഷ്ടിക്കുന്നതും കംപൈൽ ചെയ്യുന്നതും ഇത് നിങ്ങൾക്കായി കൈകാര്യം ചെയ്യുന്നു.
