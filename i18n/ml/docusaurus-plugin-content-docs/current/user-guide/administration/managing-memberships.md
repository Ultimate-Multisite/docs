# അംഗത്വങ്ങൾ നിയന്ത്രിക്കൽ {#managing-memberships}

Ultimate Multisite-ൽ നിങ്ങളുടെ നെറ്റ്‌വർക്കിലെ അംഗത്വങ്ങൾ നിയന്ത്രിക്കുകയും എല്ലാ സബ്‌സ്‌ക്രിപ്ഷനുകളും ആക്‌സസ് ചെയ്യുകയും ചെയ്യാം. പ്ലാനുകൾ, ഉൽപ്പന്നങ്ങൾ, സൈറ്റുകളുടെ എണ്ണം, ബില്ലിംഗ് തുകയും ആവർത്തനവും, പേയ്‌മെന്റ് ചരിത്രം, സമയമുദ്രകൾ തുടങ്ങിയ സബ്‌സ്‌ക്രിപ്ഷൻ വിശദാംശങ്ങൾ പരിശോധിക്കുകയും തിരുത്തുകയും ചെയ്യാം.

ഈ പേജ് ആക്‌സസ് ചെയ്യാൻ, Ultimate Multisite മെനുവിലെ **അംഗത്വങ്ങൾ** ടാബിലേക്ക് പോകുക.

![അംഗത്വങ്ങളുടെ പട്ടിക പേജ്](/img/admin/memberships-list.png)

അംഗത്വങ്ങൾ പേജിൽ നിങ്ങളുടെ ഉപഭോക്താക്കളുടെയും അവരുടെ അക്കൗണ്ടുകളുമായി ബന്ധപ്പെട്ട ഉൽപ്പന്നങ്ങളുടെയും പട്ടിക കാണാം. നില അനുസരിച്ച് അംഗത്വങ്ങൾ ഫിൽട്ടർ ചെയ്യാം: സജീവം, തീർപ്പാകാത്തത്, താൽക്കാലികമായി നിർത്തിയത്, കാലഹരണപ്പെട്ടത്, അല്ലെങ്കിൽ റദ്ദാക്കിയത്.

സൗജന്യ അംഗത്വങ്ങളെ **ജീവിതകാല** അംഗത്വങ്ങളായി കണക്കാക്കുന്നു. അവയ്ക്ക് കാലഹരണ തീയതി ലഭിക്കില്ല, സ്വയമേവ കാലഹരണപ്പെടുകയുമില്ല. ട്രയലിനോ നിശ്ചിത കാലയളവിനോ ശേഷം ഒരു സൗജന്യ പ്ലാൻ അവസാനിക്കണമെങ്കിൽ, ട്രയൽ ഉള്ള പണമടച്ചുള്ള ഉൽപ്പന്നം ഉപയോഗിക്കുക, അംഗത്വം കൈമാറി റദ്ദാക്കുക, അല്ലെങ്കിൽ ആക്‌സസ് മാറേണ്ട സമയത്ത് ഉപഭോക്താവിനെ മറ്റൊരു പ്ലാനിലേക്ക് മാറ്റുക.

അംഗത്വ വിശദാംശങ്ങൾ തിരുത്താൻ കഴിയുന്ന പേജിലേക്ക് പോകാൻ ഒരു അംഗത്വത്തിൽ ക്ലിക്ക് ചെയ്യുക — ബില്ലിംഗ് വിവരം, സൈറ്റ് അല്ലെങ്കിൽ സന്ദർശന പരിധികൾ, കൂടാതെ മറ്റു വിവരങ്ങളും.

![അംഗത്വം തിരുത്തുന്ന ഇന്റർഫേസ്](/img/admin/membership-edit.png)

അംഗത്വം തിരുത്തുന്ന പേജിന്റെ പൂർണ്ണ ദൃശ്യം ഇതാ:

![അംഗത്വം തിരുത്തുന്ന പൂർണ്ണ ഇന്റർഫേസ്](/img/admin/membership-edit-full.png)

നിങ്ങളുടെ നെറ്റ്‌വർക്കിൽ മുമ്പ് രജിസ്റ്റർ ചെയ്ത ഉപയോക്താക്കളുമായി ബന്ധപ്പെട്ട പുതിയ അംഗത്വങ്ങളും ചേർക്കാം. **അംഗത്വം ചേർക്കുക** ക്ലിക്ക് ചെയ്ത് വിശദാംശങ്ങൾ ക്രമീകരിക്കുക.

![അംഗത്വം ചേർക്കുന്ന മോഡൽ](/img/admin/membership-add-modal.png)

ശേഷം **അംഗത്വം സൃഷ്ടിക്കുക** ക്ലിക്ക് ചെയ്യുക. പുതിയ അംഗത്വം അംഗത്വങ്ങളുടെ പട്ടികയിൽ പ്രത്യക്ഷപ്പെടും.
