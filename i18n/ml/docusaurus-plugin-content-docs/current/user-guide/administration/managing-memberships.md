# അംഗത്വങ്ങൾ നിയന്ത്രിക്കൽ

Ultimate Multisite-ൽ നിങ്ങൾക്ക് അംഗത്വങ്ങൾ നിയന്ത്രിക്കാനും നിങ്ങളുടെ നെറ്റ്‌വർക്കിലെ എല്ലാ സബ്‌സ്ക്രിപ്ഷനുകളും ആക്‌സസ് ചെയ്യാനും കഴിയും. പ്ലാനുകളും ഉൽപ്പന്നങ്ങളും, സൈറ്റുകളുടെ എണ്ണം, ബില്ലിംഗ് തുകയും ആവൃത്തിയും, പേയ്‌മെന്റ് ചരിത്രം, ടൈംസ്റ്റാമ്പുകൾ എന്നിവ പോലുള്ള സബ്‌സ്ക്രിപ്ഷൻ വിശദാംശങ്ങൾ അവലോകനം ചെയ്യുകയും എഡിറ്റ് ചെയ്യുകയും ചെയ്യാം.

ഈ പേജിലേക്ക് പോകാൻ, Ultimate Multisite മെനുവിലെ **Memberships** ടാബിൽ ക്ലിക്ക് ചെയ്യുക.

![അംഗത്വ ലിസ്റ്റ് പേജ്](/img/admin/memberships-list.png)

Memberships പേജിൽ നിങ്ങളുടെ ഉപഭോക്താക്കളുടെയും അവരുടെ അക്കൗണ്ടുകളുമായി ബന്ധപ്പെട്ട ഉൽപ്പന്നങ്ങളുടെയും ഒരു ലിസ്റ്റ് കാണാം. active, pending, on hold, expired, അല്ലെങ്കിൽ cancelled എന്നീ സ്റ്റാറ്റസ് അനുസരിച്ച് അംഗത്വങ്ങൾ ഫിൽട്ടർ ചെയ്യാം.

അംഗത്വ വിശദാംശങ്ങൾ — ബില്ലിംഗ് വിവരങ്ങൾ, സൈറ്റ് അല്ലെങ്കിൽ സന്ദർശന പരിധികൾ, മറ്റും — എഡിറ്റ് ചെയ്യാനുള്ള പേജ് ആക്‌സസ് ചെയ്യാൻ ഒരു അംഗത്വത്തിൽ ക്ലിക്ക് ചെയ്യുക.

നിങ്ങളുടെ നെറ്റ്‌വർക്കിൽ മുമ്പ് രജിസ്റ്റർ ചെയ്ത ഉപയോക്താക്കളുമായി ബന്ധപ്പെടുത്തി പുതിയ അംഗത്വങ്ങൾ ചേർക്കാനും കഴിയും. **Add Membership** ക്ലിക്ക് ചെയ്ത് വിശദാംശങ്ങൾ ക്രമീകരിക്കുക.

![അംഗത്വം ചേർക്കൽ മോഡൽ](/img/admin/membership-add-modal.png)

തുടർന്ന് **Create Membership** ക്ലിക്ക് ചെയ്യുക. പുതിയ അംഗത്വം അംഗത്വ ലിസ്റ്റിൽ കാണാം.
