# സിസ്റ്റം ഇമെയിലുകൾ കൈകാര്യം ചെയ്യുന്നു

സൈൻഅപ്പ് സ്ഥിരീകരണങ്ങൾ, പേയ്‌മെന്റ് രസീതുകൾ, മെമ്പർഷിപ്പ് മാറ്റങ്ങൾ തുടങ്ങിയ ഇവന്റുകൾക്കായി Ultimate Multisite വിവിധ സിസ്റ്റം ഇമെയിലുകൾ അയയ്ക്കുന്നു. നിങ്ങളുടെ ബ്രാൻഡിന് അനുയോജ്യമായ രീതിയിൽ ഈ ഇമെയിൽ ടെംപ്ലേറ്റുകൾ ഇഷ്ടാനുസരണം മാറ്റാവുന്നതാണ്.

## സിസ്റ്റം ഇമെയിലുകൾ കാണുന്നു

**Ultimate Multisite → Settings** എന്നതിലേക്ക് പോയി ഇമെയിൽ സംബന്ധമായ ക്രമീകരണങ്ങൾ കാണാൻ **Emails** ടാബിൽ ക്ലിക്ക് ചെയ്യുക.

![Settings - Emails tab](/img/admin/settings-emails.png)

അവിടെ നിന്ന്, എല്ലാ ഇമെയിൽ ടെംപ്ലേറ്റുകളും കാണാൻ **System Emails** പേജിലേക്ക് പ്രവേശിക്കാം.

![System Emails list](/img/admin/system-emails-list.png)

## പുതിയ ഇമെയിൽ ടെംപ്ലേറ്റുകൾ ഇംപോർട്ട് ചെയ്യുന്നു

പുതിയ Ultimate Multisite വേർഷനുകളും ആഡ്-ഓണുകളും പുതിയ ഇമെയിൽ ടെംപ്ലേറ്റുകൾ രജിസ്റ്റർ ചെയ്തേക്കാം. പ്രശ്നങ്ങൾ ഒഴിവാക്കാൻ, പുതിയ ടെംപ്ലേറ്റുകൾ സ്വയമേവ ചേർക്കപ്പെടില്ല — നിങ്ങൾ അവ സ്വമേധയാ ഇംപോർട്ട് ചെയ്യേണ്ടതുണ്ട്.

പുതിയ ഇമെയിൽ ടെംപ്ലേറ്റുകൾ ഇംപോർട്ട് ചെയ്യാൻ:

1. **System Emails** പേജിലേക്ക് പോകുക
2. പേജിന്റെ മുകളിലുള്ള **Reset & Import** ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക
3. ലഭ്യമായ ടെംപ്ലേറ്റുകൾ കാണാൻ **Import Emails** ഓപ്ഷൻ ടോഗിൾ ചെയ്യുക
4. നിങ്ങൾക്ക് ഇംപോർട്ട് ചെയ്യേണ്ട ടെംപ്ലേറ്റുകൾ തിരഞ്ഞെടുത്ത് സ്ഥിരീകരിക്കുക

## ഇമെയിൽ ടെംപ്ലേറ്റുകൾ റീസെറ്റ് ചെയ്യുന്നു

നിങ്ങൾ ഒരു ഇമെയിൽ ടെംപ്ലേറ്റ് ഇഷ്ടാനുസരണം മാറ്റിയിട്ടുണ്ടെങ്കിൽ, ഡിഫോൾട്ട് ഉള്ളടക്കത്തിലേക്ക് മടങ്ങാൻ നിങ്ങൾക്ക് രണ്ട് വഴികളുണ്ട്:

1. **ഡിലീറ്റ് ചെയ്ത് വീണ്ടും ഇംപോർട്ട് ചെയ്യുക** — സിസ്റ്റം ഇമെയിൽ ഡിലീറ്റ് ചെയ്ത് വീണ്ടും ഇംപോർട്ട് ചെയ്യുക (ഇത് അയച്ച കണക്കുകൾ മായ്ക്കും)
2. **ടൂൾ ഉപയോഗിച്ച് റീസെറ്റ് ചെയ്യുക** — **Reset & Import** ടൂൾ ഉപയോഗിക്കുക, **Reset** ഓപ്ഷൻ ടോഗിൾ ചെയ്യുക, റീസെറ്റ് ചെയ്യേണ്ട ഇമെയിലുകൾ തിരഞ്ഞെടുക്കുക

ഡിഫോൾട്ട് ഉള്ളടക്കം പുനഃസ്ഥാപിക്കുമ്പോൾ അയച്ച കണക്കുകളും മറ്റ് മെറ്റാഡേറ്റയും സംരക്ഷിക്കുന്നതിനാൽ Reset ഓപ്ഷനാണ് കൂടുതൽ അഭികാമ്യം.
