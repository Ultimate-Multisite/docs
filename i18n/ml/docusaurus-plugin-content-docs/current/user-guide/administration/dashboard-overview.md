# ഡാഷ്ബോർഡ് അവലോകനം

Ultimate Multisite മെനുവിൽ ഒരു പ്രത്യേക **Dashboard** ഉണ്ട്, കൂടാതെ Network Dashboard-ലേക്ക് വിജറ്റുകളും ചേർക്കുന്നു. രണ്ട് സ്ഥലങ്ങളിലും നിങ്ങൾ കാണുന്നവ എന്തൊക്കെയെന്ന് ഈ ലേഖനം വിശദീകരിക്കുന്നു.

## Network Dashboard

**My Sites → Network Admin → Dashboard**-ൽ കാണുന്ന network-level dashboard-ലേക്ക് Ultimate Multisite ഉപയോഗപ്രദമായ വിജറ്റുകൾ ചേർക്കുന്നു.

![Ultimate Multisite വിജറ്റുകളോടുകൂടിയ Network Dashboard](/img/admin/network-dashboard-full.png)

ഓരോ വിജറ്റും എന്താണ് നൽകുന്നതെന്ന് നോക്കാം:

**First Steps:** പുതിയ ഇൻസ്റ്റാളേഷനിൽ ഇത് പ്രത്യക്ഷപ്പെടുകയും പേയ്മെന്റ് രീതി ക്രമീകരിക്കുക, ടെസ്റ്റ് അക്കൗണ്ട് സൃഷ്ടിക്കുക തുടങ്ങിയ സെറ്റപ്പ് ടാസ്കുകൾ പൂർത്തിയാക്കാൻ നിങ്ങളെ ഓർമ്മിപ്പിക്കുകയും ചെയ്യുന്നു.

![First Steps വിജറ്റ്](/img/admin/network-dashboard-first-steps.png)

**Summary:** ദിവസത്തെ സൈൻഅപ്പുകളുടെയും വരുമാനത്തിന്റെയും എണ്ണം റിപ്പോർട്ട് ചെയ്യുന്നു. MRR (Monthly Recurring Revenue) വരുമാനമുള്ള മെമ്പർഷിപ്പുകളുള്ള ഉപഭോക്താക്കളുടെ പ്രതീക്ഷിത മൊത്തം തുക സൂചിപ്പിക്കുന്നു.

![Summary വിജറ്റ്](/img/admin/network-dashboard-summary.png)

**Activity Stream:** Ultimate Multisite-ലുടനീളമുള്ള ഇവന്റുകൾ — സൈൻഅപ്പുകൾ, റദ്ദാക്കലുകൾ, സൈറ്റ് സൃഷ്ടികൾ, ഇല്ലാതാക്കലുകൾ, മറ്റ് പ്രവർത്തനങ്ങൾ എന്നിവ റിപ്പോർട്ട് ചെയ്യുന്നു.

![Activity Stream വിജറ്റ്](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** നെറ്റ്‌വർക്കിലെ ഉപയോക്താക്കളുടെയും സൈറ്റുകളുടെയും ഒരു ചെറിയ സംഗ്രഹം പുതിയ സൈറ്റുകളോ ഉപയോക്താക്കളെയോ സൃഷ്ടിക്കാനുള്ള കുറുക്കുവഴികളോടൊപ്പം പ്രദർശിപ്പിക്കുന്നു.

![Right Now വിജറ്റ്](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** അപ്ഡേറ്റുകൾ, ഗുരുതരമായ ബഗ്ഗുകൾ, സുരക്ഷാ പരിഹാരങ്ങൾ എന്നിവയുൾപ്പെടെ ഏറ്റവും പുതിയ Ultimate Multisite വാർത്തകൾ ശേഖരിച്ച് പ്രദർശിപ്പിക്കുന്നു.

![News വിജറ്റ്](/img/admin/network-dashboard-news.png)

## Ultimate Multisite Dashboard

Network dashboard നെറ്റ്‌വർക്കിനെക്കുറിച്ചുള്ള സമഗ്രമായ വിവരങ്ങൾ അവതരിപ്പിക്കുമ്പോൾ, Ultimate Multisite dashboard (Ultimate Multisite മെനുവിന്റെ ഏറ്റവും മുകളിൽ) സേവനത്തെക്കുറിച്ചുള്ള വിവരങ്ങൾ അവതരിപ്പിക്കുന്നു.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Monthly Recurring Revenue Growth:** 12 മാസത്തെ കാലയളവിൽ വരുമാനത്തിന്റെ മാസാടിസ്ഥാനത്തിലുള്ള വിശദാംശങ്ങൾ പ്രദർശിപ്പിക്കുന്നു. പുതിയ വരുമാനം നീലയിലും റദ്ദാക്കലുകൾ ചുവപ്പിലും കാണിക്കുന്നു.

![MRR Growth ചാർട്ട്](/img/admin/um-dashboard-mrr.png)

**Date-range Support:** നിങ്ങൾക്ക് ആവശ്യമായ വിവരങ്ങളിൽ ശ്രദ്ധ കേന്ദ്രീകരിക്കാൻ കാലയളവ് അനുസരിച്ച് dashboard ഡാറ്റ ഫിൽട്ടർ ചെയ്യുക. തിരഞ്ഞെടുത്ത തീയതി ശ്രേണി താഴെയുള്ള വിജറ്റുകളിൽ എന്താണ് പ്രദർശിപ്പിക്കുന്നതെന്ന് നിയന്ത്രിക്കുന്നു.

![തീയതി ശ്രേണിയും dashboard സ്ഥിതിവിവരക്കണക്കുകളും](/img/admin/um-dashboard-date-range.png)

Dashboard വിജറ്റുകളിൽ ഇവ ഉൾപ്പെടുന്നു:

- **Revenue** — MRR-ഉം ഉൽപ്പന്നാടിസ്ഥാനത്തിലുള്ള വിശദാംശങ്ങളും ഉൾപ്പെടെ ആ കാലയളവിലെ മൊത്തം വരുമാനം
- **New Memberships** — തിരഞ്ഞെടുത്ത കാലയളവിൽ എത്ര പുതിയ മെമ്പർഷിപ്പുകൾ സൃഷ്ടിച്ചു
- **Signups by Countries** — സൈൻഅപ്പുകളുടെ ഭൂമിശാസ്ത്രപരമായ വിശദാംശങ്ങൾ
- **Signups by Forms** — ഓരോ checkout ഫോം വഴിയും അഡ്മിൻമാർ ചേർത്തതും എത്ര ഉപഭോക്താക്കൾ സൈൻ അപ്പ് ചെയ്തു
- **Most Visited Sites** — നിങ്ങളുടെ നെറ്റ്‌വർക്കിലെ ഏറ്റവും കൂടുതൽ സന്ദർശിച്ച സൈറ്റുകൾ (**Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**-ൽ visit counting പ്രവർത്തനക്ഷമമാക്കേണ്ടതുണ്ട്)

![Dashboard സ്ഥിതിവിവരക്കണക്കുകൾ](/img/admin/um-dashboard-stats.png)
