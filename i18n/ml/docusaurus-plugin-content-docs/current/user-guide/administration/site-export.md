# സൈറ്റ് എക്സ്പോർട്ട് (Site Export)

**സൈറ്റ് എക്സ്പോർട്ട്** അഡ്മിൻ പേജ് ഉപയോഗിച്ച് നെറ്റ്‌വർക്ക് അഡ്മിനിസ്ട്രേറ്റർമാർക്ക് ഒരു സൈറ്റ്, അല്ലെങ്കിൽ മുഴുവൻ നെറ്റ്‌വർക്കിലുള്ള സൈറ്റുകളും, മൈഗ്രേഷൻ, ബാക്കപ്പ്, അല്ലെങ്കിൽ കൈമാറ്റം ചെയ്യാനുള്ള ഡൗൺലോഡബിൾ ആർക്കൈവായി (archive) പാക്കേജ് ചെയ്യാൻ കഴിയും.

## ഒരു സൈറ്റ് എക്സ്പോർട്ട് ചെയ്യുമ്പോൾ

**Ultimate Multisite > Site Export** എന്നതിലേക്ക് പോവുക, തുടർന്ന് **Generate new Site Export** തിരഞ്ഞെടുക്കുക. എക്സ്പോർട്ട് ചെയ്യേണ്ട സബ്സൈറ്റ് തിരഞ്ഞെടുക്കുക, അതിനുശേഷം ആർക്കൈവിൽ അപ്‌ലോഡുകൾ (uploads), പ്ലഗിനുകൾ (plugins), തീമുകൾ (themes) എന്നിവ ഉൾപ്പെടുത്തണമോ എന്ന് തിരഞ്ഞെടുക്കുക.

എക്സ്പോർട്ട് പൂർത്തിയാകുമ്പോൾ, **Existing Exports** ലിസ്റ്റിൽ നിന്ന് ZIP ഫയൽ ഡൗൺലോഡ് ചെയ്യുക. എക്സ്പോർട്ട് ZIP-കളിൽ ഇപ്പോൾ ഒരു സ്വയം-ബൂട്ടിംഗ് `index.php` ഉം ഒരു `readme.txt` ഉം ഉൾപ്പെടുന്നു. അതിനാൽ, ഈ ആർക്കൈവ് ഒരു പുതിയ ഹോസ്റ്റിലേക്ക് അപ്‌ലോഡ് ചെയ്യുകയും, പ്രത്യേക ഇംപോർട്ടർ പ്ലഗിൻ ഇൻസ്റ്റാൾ ചെയ്യാതെ തന്നെ പ്രവർത്തിപ്പിക്കുകയും ചെയ്യാം.

## മുഴുവൻ നെറ്റ്‌വർക്കും എക്സ്പോർട്ട് ചെയ്യുമ്പോൾ

നെറ്റ്‌വർക്കിലെ എല്ലാ സബ്സൈറ്റുകളും ഉൾക്കൊള്ളുന്ന ഒരു ഒറ്റ ആർക്കൈവ് ആവശ്യമായി വരുമ്പോൾ, സൈറ്റ് എക്സ്പോർട്ട് പേജിലെ **Network Export** ഉപയോഗിക്കുക. ഹോസ്റ്റ് മൈഗ്രേഷനുകൾക്ക് മുൻപോ, ദുരന്ത നിവാരണ ഡ്രില്ലുകൾക്കോ, അല്ലെങ്കിൽ ഓരോ സബ്സൈറ്റും ഒരുമിച്ച് യാത്ര ചെയ്യേണ്ട സ്റ്റേജിംഗ് റീബിൽഡുകൾക്കോ ഇത് വളരെ ഉപകാരപ്രദമാണ്.

ഒരു നെറ്റ്‌വർക്ക് എക്സ്പോർട്ട് ഒറ്റ സൈറ്റ് എക്സ്പോർട്ടേക്കാൾ വളരെ വലുതാകാൻ സാധ്യതയുള്ളതിനാൽ, കുറഞ്ഞ ട്രാഫിക്കുള്ള സമയത്താണ് ഇത് പ്രവർത്തിപ്പിക്കേണ്ടത്. കൂടാതെ, അപ്‌ലോഡുകൾ, പ്ലഗിനുകൾ, തീമുകൾ, സൃഷ്ടിക്കപ്പെട്ട ZIP ഫയലുകൾ എന്നിവയ്ക്കായി ലക്ഷ്യസ്ഥാന സ്റ്റോറേജിൽ (target storage) ആവശ്യത്തിന് ഫ്രീ സ്പേസ് ഉണ്ടോ എന്നും ഉറപ്പുവരുത്തണം.

### നെറ്റ്‌വർക്ക് ഇംപോർട്ട് ബണ്ടിലുകൾ (Network Import Bundles)

Ultimate Multisite 2.12.0 മുതൽ, സൈറ്റ് എക്സ്പോർട്ടർ **നെറ്റ്‌വർക്ക് ഇംപോർട്ട് ബണ്ടിലുകൾ** സൃഷ്ടിക്കാൻ കഴിയും. നെറ്റ്‌വർക്ക് തലത്തിലുള്ള സൈറ്റ് പുനഃസ്ഥാപനത്തിനായി രൂപകൽപ്പന ചെയ്ത പ്രത്യേക ആർക്കൈവുകളാണിവ. ഒന്നിലധികം സൈറ്റുകൾ ഒരു പുതിയ നെറ്റ്‌വർക്ക് ഇൻസ്റ്റാളേഷനിലേക്ക് പുനഃസ്ഥാപിക്കാൻ ആവശ്യമായ എല്ലാ ഫയലുകളും മെറ്റാഡാറ്റയും ഒരു നെറ്റ്‌വർക്ക് ഇംപോർട്ട് ബണ്ടിലിൽ അടങ്ങിയിരിക്കുന്നു.

#### നെറ്റ്‌വർക്ക് ഇംപോർട്ട് ബണ്ടിൽ സൃഷ്ടിക്കൽ

1. **Ultimate Multisite > Site Export** ലേക്ക് പോകുക
2. **Generate new Network Export** ക്ലിക്ക് ചെയ്യുക
3. എക്സ്പോർട്ട് തരം (export type) ആയി **Network Import Bundle** തിരഞ്ഞെടുക്കുക
4. അപ്‌ലോഡുകൾ, പ്ലഗിനുകൾ, തീമുകൾ എന്നിവ ഉൾപ്പെടുത്തണമോ എന്ന് തിരഞ്ഞെടുക്കുക
5. **Generate** ക്ലിക്ക് ചെയ്യുക
6. **Existing Exports** ലിസ്റ്റിൽ നിന്ന് ബണ്ടിൽ ZIP ഡൗൺലോഡ് ചെയ്യുക

#### നെറ്റ്‌വർക്ക് ഇംപോർട്ട് ബണ്ടിൽ നിന്ന് പുനഃസ്ഥാപിക്കൽ

ഒരു നെറ്റ്‌വർക്ക് ഇംപോർട്ട് ബണ്ടിൽ നിന്ന് സൈറ്റുകൾ പുനഃസ്ഥാപിക്കണമെങ്കിൽ:

1. നിങ്ങളുടെ ലക്ഷ്യസ്ഥാന ഹോസ്റ്റിൽ (target host) Ultimate Multisite ഇൻസ്റ്റാൾ ചെയ്യുക
2. മൾട്ടിസൈറ്റ് സജ്ജീകരണ മാന്ത്രികത (multisite setup wizard) പൂർത്തിയാക്കുക
3. പുതിയ നെറ്റ്‌വർക്കിൽ **Ultimate Multisite > Site Export** ലേക്ക് പോകുക
4. **Import Network Bundle** ക്ലിക്ക് ചെയ്യുക
5. നെറ്റ്‌വർക്ക് ഇംപോർട്ട് ബണ്ടിൽ ZIP ഫയൽ അപ്‌ലോഡ് ചെയ്യുക
6. സ്ക്രീനിൽ കാണിക്കുന്ന ഇംപോർട്ട് നിർദ്ദേശങ്ങൾ പിന്തുടരുക
7. ഇംപോർട്ട് പ്രക്രിയ എല്ലാ സൈറ്റുകളും അവയുടെ ഉള്ളടക്കവും (content) കോൺഫിഗറേഷനുകളും പുനഃസ്ഥാപിക്കും.

നെറ്റ്‌വർക്ക് ഇംപോർട്ട് ബണ്ടിലുകൾ സംരക്ഷിക്കുന്നത്:
- സൈറ്റ് ഉള്ളടക്കം (പോസ്റ്റുകൾ, പേജുകൾ, കസ്റ്റം പോസ്റ്റ് ടൈപ്പുകൾ)
- സൈറ്റ് ക്രമീകരണങ്ങളും ഓപ്ഷനുകളും
- യൂസർ റോളുകളും പെർമിഷനുകളും
- പ്ലഗിനുകളും തീമുകളും (ബണ്ടിലിൽ ഉൾപ്പെടുത്തിയിട്ടുണ്ടെങ്കിൽ)
- മീഡിയ അപ്‌ലോഡുകൾ (ബണ്ടിലിൽ ഉൾപ്പെടുത്തിയിട്ടുണ്ടെങ്കിൽ)
- കസ്റ്റം ഡാറ്റാബേസ് ടേബിളുകളും ഡാറ്റയും

## സ്വയം-ബൂട്ടിംഗ് എക്സ്പോർട്ട് ZIP ഇൻസ്റ്റാൾ ചെയ്യൽ

ഒരു പുതിയ ഹോസ്റ്റിൽ സ്വയം-ബൂട്ടിംഗ് ZIP പുനഃസ്ഥാപിക്കാൻ:

1. എക്സ്പോർട്ട് ചെയ്ത ZIP ഉള്ളടക്കങ്ങൾ ലക്ഷ്യസ്ഥാന വെബ് റൂട്ടിൽ (target web root) അപ്‌ലോഡ് ചെയ്യുക.
2. ബ്രൗസറിൽ അപ്‌ലോഡ് ചെയ്ത `index.php` തുറക്കുക.
3. ബണ്ടിൽ ചെയ്ത എക്സ്പോർട്ട് പാക്കേജിൽ നിന്നുള്ള ഓൺ-സ്‌ക്രീൻ ഇൻസ്റ്റാളർ നിർദ്ദേശങ്ങൾ പിന്തുടരുക.
4. താൽക്കാലിക ഫയലുകൾ നീക്കം ചെയ്യുന്നതിന് മുമ്പ്, ബണ്ടിലായ `readme.txt` പരിശോധിച്ച് എക്സ്പോർട്ട് സംബന്ധമായ കുറിപ്പുകൾ ശ്രദ്ധിക്കുക.

കൂടുതൽ വിവരങ്ങൾക്കായി, [Site Exporter addon documentation](/addons/site-exporter) കാണുക.

Ultimate Multisite 2.9.0-ൽ ചേർത്ത ഒറ്റ-സൈറ്റ് ടൂളിനായി, [Export & Import](/user-guide/administration/export-import) കാണുക.
