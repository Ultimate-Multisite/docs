# പേയ്‌മെന്റുകളും ഇൻവോയ്‌സുകളും കൈകാര്യം ചെയ്യൽ

## പേയ്‌മെന്റ് ക്രമീകരണങ്ങൾ

പേയ്‌മെന്റുകൾ സ്വീകരിക്കാൻ തുടങ്ങുന്നതിന് മുമ്പ്, പേയ്‌മെന്റുമായി ബന്ധപ്പെട്ട ക്രമീകരണങ്ങൾ കോൺഫിഗർ ചെയ്യേണ്ടതുണ്ട്. **Ultimate Multisite → Settings**-ലേക്ക് പോയി **Payment** ടാബിൽ ക്ലിക്ക് ചെയ്യുക.

![Payment settings tab](/img/admin/settings-payments-top.png)

### പൊതു പേയ്‌മെന്റ് ഓപ്ഷനുകൾ

പൊതു ക്രമീകരണങ്ങളിൽ നിങ്ങൾക്ക് ഇവ കോൺഫിഗർ ചെയ്യാം:

- **Currency** — ഇടപാടുകൾക്ക് ഉപയോഗിക്കുന്ന ഡിഫോൾട്ട് കറൻസി
- **Currency Position** — കറൻസി ചിഹ്നം എവിടെ പ്രദർശിപ്പിക്കണം (തുകയുടെ മുമ്പിൽ/പിന്നിൽ)

![Payment gateway settings](/img/admin/settings-payments-options.png)

### പേയ്‌മെന്റ് ഗേറ്റ്‌വേകൾ

Ultimate Multisite ഒന്നിലധികം പേയ്‌മെന്റ് ഗേറ്റ്‌വേകളെ പിന്തുണയ്ക്കുന്നു. Payment settings ടാബിൽ നിന്ന് ഓരോന്നും എനേബിൾ ചെയ്യാനും കോൺഫിഗർ ചെയ്യാനും കഴിയും.

![Payment gateway configuration](/img/admin/settings-payments-gateways.png)

ലഭ്യമായ ഗേറ്റ്‌വേകളിൽ ഉൾപ്പെടുന്നവ:

- **Stripe** — Stripe വഴിയുള്ള ക്രെഡിറ്റ് കാർഡ് പേയ്‌മെന്റുകൾ
- **PayPal** — PayPal പേയ്‌മെന്റുകൾ
- **Manual** — ഓഫ്‌ലൈൻ അല്ലെങ്കിൽ കസ്റ്റം പേയ്‌മെന്റ് പ്രോസസ്സിംഗിന്

ഓരോ ഗേറ്റ്‌വേയ്ക്കും API കീകളും മറ്റ് ക്രമീകരണങ്ങളും നൽകാനുള്ള സ്വന്തം കോൺഫിഗറേഷൻ വിഭാഗമുണ്ട്.

![Additional gateway settings](/img/admin/settings-payments-gateways-2.png)

### Sandbox Mode

പേയ്‌മെന്റ് ഇന്റഗ്രേഷൻ ലൈവാകുന്നതിന് മുമ്പ് പരീക്ഷിക്കാൻ **Sandbox Mode** എനേബിൾ ചെയ്യാം. Sandbox mode ആക്ടീവ് ആയിരിക്കുമ്പോൾ, യഥാർത്ഥ ചാർജുകൾ ഒന്നും ഈടാക്കില്ല.

## പേയ്‌മെന്റുകൾ കാണൽ

നിങ്ങളുടെ നെറ്റ്‌വർക്കിലെ എല്ലാ ഇടപാടുകളും കാണാൻ Ultimate Multisite-ന് കീഴിലുള്ള **Payments** പേജിലേക്ക് പോകുക.

![Payments list](/img/admin/payments-list.png)

സ്റ്റാറ്റസ് അനുസരിച്ച് (completed, pending, failed, refunded) പേയ്‌മെന്റുകൾ ഫിൽട്ടർ ചെയ്യാനും നിർദ്ദിഷ്ട ഇടപാടുകൾക്കായി തിരയാനും കഴിയും.

ലൈൻ ഐറ്റങ്ങൾ, ബന്ധപ്പെട്ട മെമ്പർഷിപ്പ്, കസ്റ്റമർ വിവരങ്ങൾ, പേയ്‌മെന്റ് ഗേറ്റ്‌വേ ഡാറ്റ എന്നിവയുൾപ്പെടെ മുഴുവൻ വിശദാംശങ്ങളും കാണാൻ ഒരു പേയ്‌മെന്റിൽ ക്ലിക്ക് ചെയ്യുക.

## ഇൻവോയ്‌സുകൾ

പേയ്‌മെന്റുകൾക്കായി Ultimate Multisite-ന് സ്വയമേവ ഇൻവോയ്‌സുകൾ സൃഷ്ടിക്കാൻ കഴിയും. Payment settings-ൽ നിന്ന് ഇൻവോയ്‌സ് ടെംപ്ലേറ്റും നമ്പറിംഗ് ഫോർമാറ്റും ഇഷ്ടാനുസൃതമാക്കാം.

ഇൻവോയ്‌സ് കസ്റ്റമൈസേഷൻ ഓപ്ഷനുകളിൽ ഉൾപ്പെടുന്നവ:

- ഇൻവോയ്‌സുകളിൽ പ്രദർശിപ്പിക്കുന്ന **കമ്പനിയുടെ പേരും വിലാസവും**
- **ഇൻവോയ്‌സ് നമ്പറിംഗ്** ഫോർമാറ്റും സീക്വൻസും
- ഇൻവോയ്‌സ് ഹെഡറിൽ പ്രദർശിപ്പിക്കുന്ന **ലോഗോ**
- നിബന്ധനകൾ, കുറിപ്പുകൾ, അല്ലെങ്കിൽ നിയമപരമായ വിവരങ്ങൾക്കുള്ള **കസ്റ്റം ഫൂട്ടർ ടെക്സ്റ്റ്**

ഇൻവോയ്‌സ് ടെംപ്ലേറ്റ് ഇഷ്ടാനുസൃതമാക്കാൻ, **Ultimate Multisite → Settings → Payment**-ലേക്ക് പോയി ഇൻവോയ്‌സുമായി ബന്ധപ്പെട്ട ക്രമീകരണങ്ങൾ കണ്ടെത്തുക.
