# പേയ്‌മെന്റുകളും ഇൻവോയിസുകളും കൈകാര്യം ചെയ്യൽ

## പേയ്‌മെന്റ് ക്രമീകരണങ്ങൾ {#payment-settings}

പേയ്‌മെന്റുകൾ സ്വീകരിക്കാൻ തുടങ്ങുന്നതിന് മുമ്പ്, പേയ്‌മെന്റുമായി ബന്ധപ്പെട്ട ക്രമീകരണങ്ങൾ സജ്ജീകരിക്കണം. **Ultimate Multisite → ക്രമീകരണങ്ങൾ** എന്നതിലേക്ക് പോയി **പേയ്‌മെന്റ്** ടാബിൽ ക്ലിക്ക് ചെയ്യുക.

![പേയ്‌മെന്റ് ക്രമീകരണങ്ങൾ ടാബ്](/img/admin/settings-payments-top.png)

പേയ്‌മെന്റ് ക്രമീകരണ പേജിന്റെ പൂർണ്ണ ദൃശ്യം ഇതാ:

![പേയ്‌മെന്റ് ക്രമീകരണങ്ങളുടെ പൂർണ്ണ പേജ്](/img/admin/settings-payments-full.png)

### പൊതുവായ പേയ്‌മെന്റ് ഓപ്ഷനുകൾ {#general-payment-options}

പൊതുവായ ക്രമീകരണങ്ങളിൽ നിങ്ങൾക്ക് സജ്ജീകരിക്കാനാകുന്നത്:

- **കറൻസി** — ഇടപാടുകൾക്കായി ഉപയോഗിക്കുന്ന സ്ഥിരസ്ഥിതി കറൻസി
- **കറൻസി സ്ഥാനം** — കറൻസി ചിഹ്നം എവിടെ പ്രത്യക്ഷപ്പെടും (തുകയ്ക്ക് മുമ്പ്/ശേഷം)

![പേയ്‌മെന്റ് ഗേറ്റ്‌വേ ക്രമീകരണങ്ങൾ](/img/admin/settings-payments-options.png)

### പേയ്‌മെന്റ് ഗേറ്റ്‌വേകൾ {#payment-gateways}

Ultimate Multisite ഒന്നിലധികം പേയ്‌മെന്റ് ഗേറ്റ്‌വേകളെ പിന്തുണയ്ക്കുന്നു. പേയ്‌മെന്റ് ക്രമീകരണങ്ങൾ ടാബിൽ നിന്ന് ഓരോന്നും സജീവമാക്കി സജ്ജീകരിക്കാം.

![പേയ്‌മെന്റ് ഗേറ്റ്‌വേ കോൺഫിഗറേഷൻ](/img/admin/settings-payments-gateways.png)

ലഭ്യമായ ഗേറ്റ്‌വേകളിൽ ഉൾപ്പെടുന്നത്:

- **Stripe** — Stripe വഴി ക്രെഡിറ്റ് കാർഡ് പേയ്‌മെന്റുകൾ
- **PayPal** — PayPal പേയ്‌മെന്റുകൾ
- **മാനുവൽ** — ഓഫ്‌ലൈൻ അല്ലെങ്കിൽ ഇഷ്ടാനുസൃത പേയ്‌മെന്റ് പ്രോസസ്സിംഗിനായി

ഓരോ ഗേറ്റ്‌വേയ്ക്കും സ്വന്തം കോൺഫിഗറേഷൻ വിഭാഗമുണ്ട്. അവിടെ നിങ്ങൾ API കീകളും മറ്റു ക്രമീകരണങ്ങളും നൽകുന്നു.

![അധിക ഗേറ്റ്‌വേ ക്രമീകരണങ്ങൾ](/img/admin/settings-payments-gateways-2.png)

### സാൻഡ്‌ബോക്സ് മോഡ് {#sandbox-mode}

ലൈവിലേക്ക് പോകുന്നതിന് മുമ്പ് നിങ്ങളുടെ പേയ്‌മെന്റ് ഇന്റഗ്രേഷൻ പരീക്ഷിക്കാൻ **സാൻഡ്‌ബോക്സ് മോഡ്** സജീവമാക്കാം. സാൻഡ്‌ബോക്സ് മോഡ് സജീവമായിരിക്കുമ്പോൾ യഥാർത്ഥ ചാർജുകൾ ഉണ്ടാകില്ല.

## പേയ്‌മെന്റുകൾ കാണൽ {#viewing-payments}

നിങ്ങളുടെ നെറ്റ്‌വർക്കിലുടനീളമുള്ള എല്ലാ ഇടപാടുകളും കാണാൻ Ultimate Multisite-ന്റെ കീഴിലുള്ള **പേയ്‌മെന്റുകൾ** പേജിലേക്ക് പോകുക.

![പേയ്‌മെന്റുകളുടെ പട്ടിക](/img/admin/payments-list.png)

സ്ഥിതി അനുസരിച്ച് (പൂർത്തിയായത്, ബാക്കിയുള്ളത്, പരാജയപ്പെട്ടത്, റീഫണ്ട് ചെയ്തത്) പേയ്‌മെന്റുകൾ ഫിൽട്ടർ ചെയ്യാനും പ്രത്യേക ഇടപാടുകൾ തിരയാനും കഴിയും.

ലൈൻ ഇനങ്ങൾ, ബന്ധപ്പെട്ട അംഗത്വം, ഉപഭോക്തൃ വിവരങ്ങൾ, പേയ്‌മെന്റ് ഗേറ്റ്‌വേ ഡാറ്റ എന്നിവ ഉൾപ്പെടെയുള്ള പൂർണ്ണ വിശദാംശങ്ങൾ കാണാൻ ഒരു പേയ്‌മെന്റിൽ ക്ലിക്ക് ചെയ്യുക.

## ഇൻവോയിസുകൾ {#invoices}

Ultimate Multisite പേയ്‌മെന്റുകൾക്കായി ഇൻവോയിസുകൾ സ്വയമേവ സൃഷ്ടിക്കാം. പേയ്‌മെന്റ് ക്രമീകരണങ്ങളിൽ നിന്ന് ഇൻവോയിസ് ടെംപ്ലേറ്റും നമ്പറിംഗ് ഫോർമാറ്റും ഇഷ്ടാനുസൃതമാക്കാം.

ഇൻവോയിസ് ഇഷ്ടാനുസൃതമാക്കൽ ഓപ്ഷനുകളിൽ ഉൾപ്പെടുന്നത്:

- **കമ്പനി പേരും വിലാസവും** ഇൻവോയിസുകളിൽ പ്രദർശിപ്പിക്കുന്നു
- **ഇൻവോയിസ് നമ്പറിംഗ്** ഫോർമാറ്റും ക്രമവും
- **ലോഗോ** ഇൻവോയിസ് ഹെഡറിൽ പ്രദർശിപ്പിക്കുന്നു
- **ഇഷ്ടാനുസൃത ഫൂട്ടർ ടെക്സ്റ്റ്** നിബന്ധനകൾ, കുറിപ്പുകൾ, അല്ലെങ്കിൽ നിയമപരമായ വിവരങ്ങൾക്കായി

ഇൻവോയിസ് ടെംപ്ലേറ്റ് ഇഷ്ടാനുസൃതമാക്കാൻ, **Ultimate Multisite → ക്രമീകരണങ്ങൾ → പേയ്‌മെന്റ്** എന്നതിലേക്ക് പോയി ഇൻവോയിസുമായി ബന്ധപ്പെട്ട ക്രമീകരണങ്ങൾ കണ്ടെത്തുക.
