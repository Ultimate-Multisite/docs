# ഉപഭോക്താക്കളെ കൈകാര്യം ചെയ്യുന്നത്

**Customers** പേജിൽ നിങ്ങളുടെ നെറ്റ്‌വർക്കിൽ രജിസ്റ്റർ ചെയ്ത അംഗങ്ങളുടെ ലിസ്റ്റ് കാണാം. ഈ പേജിൽ പേര്, ഇമെയിൽ വിലാസം, അവസാനമായി ലോഗിൻ ചെയ്ത സമയം എന്നിവ ഉൾപ്പെടെ അടിസ്ഥാന യൂസർ വിവരങ്ങൾ പ്രദർശിപ്പിക്കുന്നു. കൂടാതെ, ഒരു **switch to** ലിങ്കും ഉണ്ട്, ഇത് ഉപയോഗിച്ച് നിങ്ങൾക്ക് സൈറ്റ് അഡ്മിനിസ്ട്രേറ്ററായി യൂസറിന്റെ subsite-ലേക്ക് ലോഗിൻ ചെയ്യാം.

![Customers list page](/img/admin/customers-list.png)

ഒരു ഉപഭോക്താവിനെക്കുറിച്ചുള്ള വിശദമായ വിവരങ്ങൾ കാണാൻ അവരുടെ കാർഡിന് താഴെയുള്ള **Manage** ക്ലിക്ക് ചെയ്യുക.

ഉപഭോക്താവിന്റെ വിശദാംശ പേജിൽ, **At a Glance** ബ്ലോക്കിൽ ഉപഭോക്തൃ ID, അവസാനമായി ലോഗിൻ ചെയ്ത സമയം, ഈ ഉപഭോക്താവ് ചെലവഴിച്ച മൊത്തം തുക എന്നിവ കാണാം.

**Customer Options** ബ്ലോക്ക് ഉപയോഗിച്ച് ഈ ഉപഭോക്താവിനെ **VIP** ആയി സജ്ജീകരിക്കാം (അവരെ എളുപ്പത്തിൽ കണ്ടെത്താനുള്ള ഒരു ടാഗ്). **Billing Info** ടാബിൽ ഇമെയിൽ, വിലാസം, രാജ്യം തുടങ്ങിയ ബില്ലിംഗ് വിവരങ്ങൾ കാണിക്കുന്നു.

**Fields** ബ്ലോക്കിൽ ഉപഭോക്താവിന്റെ യൂസർനെയിം, ഇമെയിൽ എന്നിവ കാണാം, കൂടാതെ **Email Verification** സ്റ്റാറ്റസ് മാറ്റാനും കഴിയും.

Customer Options ബ്ലോക്കിന് താഴെ നാല് അധിക ബ്ലോക്കുകൾ കൂടി കാണാം:

- **Memberships** — ഉപഭോക്താവ് സബ്‌സ്‌ക്രൈബ് ചെയ്തിട്ടുള്ള നിലവിലെ സബ്‌സ്‌ക്രിപ്ഷനുകൾ
- **Payments** — ഉപഭോക്താവിന്റെ പേയ്‌മെന്റ് ചരിത്രം
- **Sites** — ഈ ഉപഭോക്താവിന്റെ ഉടമസ്ഥതയിലുള്ള സൈറ്റുകൾ
- **Events** — ഈ ഉപഭോക്താവുമായി ബന്ധപ്പെട്ട ഇവന്റുകൾ (ഇമെയിൽ വെരിഫിക്കേഷൻ, ബില്ലിംഗ് മാറ്റങ്ങൾ മുതലായവ)

ഏതെങ്കിലും ഇനത്തിന്റെ വിശദാംശ പേജിലേക്ക് പോകാൻ അതിന് അടുത്തുള്ള അമ്പടയാള ഐക്കൺ (→) ക്ലിക്ക് ചെയ്യുക. ഉദാഹരണത്തിന്, ഒരു സൈറ്റിന് അടുത്തുള്ള അമ്പടയാളം ക്ലിക്ക് ചെയ്താൽ സൈറ്റ് എഡിറ്റ് പേജിലേക്ക് പോകും, അവിടെ membership association, സൈറ്റ് ടൈപ്പ്, ആക്ടീവ് സ്റ്റാറ്റസ് എന്നിവ മാറ്റാൻ കഴിയും.

Memberships-നും payments-നും ഇതേ രീതി ബാധകമാണ് — ഓരോന്നും വിശദാംശ പേജുകളിലേക്ക് ലിങ്ക് ചെയ്യുന്നു, അവിടെ membership status, ബന്ധപ്പെട്ട ഉപഭോക്താവ്, ഉൽപ്പന്നങ്ങൾ, payment gateway settings എന്നിവ എഡിറ്റ് ചെയ്യാനും റീഫണ്ടുകൾ പ്രോസസ്സ് ചെയ്യാനും കഴിയും.

ഉപഭോക്താവിന്റെ പേജിന്റെ വലതുവശത്ത് ഇവ കാണാം:

- **Last Login & IPs** — അവസാനമായി ലോഗിൻ ചെയ്ത സമയം, IP വിലാസം, രാജ്യം
- **Timestamps** — ഉപഭോക്തൃ അക്കൗണ്ട് സൃഷ്ടിച്ചതും അവസാനമായി മാറ്റം വരുത്തിയതും എപ്പോൾ
- **Delete** — ഉപഭോക്താവിനെ ശാശ്വതമായി ഇല്ലാതാക്കാനുള്ള ഓപ്ഷൻ

:::warning
ഒരു ഉപഭോക്താവിനെ ഇല്ലാതാക്കുന്നത് ശാശ്വതമാണ്, ഇത് പഴയപടിയാക്കാൻ കഴിയില്ല.
:::
