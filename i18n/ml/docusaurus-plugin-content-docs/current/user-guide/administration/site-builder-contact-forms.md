# സൈറ്റ് ബിൽഡർ കോൺടാക്റ്റ് ഫോമുകൾ

Superdav AI Agent v1.10.0 ഉപയോഗിച്ച്, Site Builder agent വഴി ചാറ്റ് ഇന്റർഫേസിൽ നിന്ന് നേരിട്ട് കോൺടാക്റ്റ് ഫോമുകൾ ഉണ്ടാക്കാൻ സാധിക്കും. ചാറ്റ് വിട്ട് പോകാതെ തന്നെ ഏത് പേജിലും പൂർണ്ണമായി പ്രവർത്തിക്കുന്ന കോൺടാക്റ്റ് ഫോമുകൾ ചേർക്കാൻ ഇത് നിങ്ങളെ സഹായിക്കുന്നു.

## Overview {#overview}

Site Builder-ലെ കോൺടാക്റ്റ് ഫോം സൗകര്യം (`create_contact_form`) നിങ്ങൾ ഇൻസ്റ്റാൾ ചെയ്തിട്ടുള്ള ഫോം plugin ഓട്ടോമാറ്റിക്കായി തിരിച്ചറിയുകയും, ആ plugin-ന്റെ native capabilities ഉപയോഗിച്ച് ഒരു കോൺടാക്റ്റ് ഫോം സൃഷ്ടിക്കുകയും ചെയ്യുന്നു. പിന്തുണയ്ക്കുന്ന ഫോം plugins ഇവയാണ്:

- Contact Form 7
- WPForms
- Fluent Forms
- Gravity Forms

നിങ്ങളുടെ സൈറ്റിൽ ലഭ്യമായ ഏറ്റവും മികച്ച ഫോം plugin ഓട്ടോമാറ്റിക്കായി അസിസ്റ്റന്റ് തിരഞ്ഞെടുക്കുകയും, ആ plugin-ന് അനുയോജ്യമായ ഒരു കോൺടാക്റ്റ് ഫോം സൃഷ്ടിക്കുകയും ചെയ്യുന്നു.

## ഒരു കോൺടാക്റ്റ് ഫോം സൃഷ്ടിക്കൽ {#creating-a-contact-form}

Site Builder ഉപയോഗിച്ച് ഒരു കോൺടാക്റ്റ് ഫോം സൃഷ്ടിക്കാൻ:

1. WordPress admin-ൽ **Gratis AI Agent** ചാറ്റ് പാനൽ തുറക്കുക.
2. ചാറ്റ് ഹെഡറിലെ agent ഐക്കണിൽ ക്ലിക്ക് ചെയ്ത് **Site Builder** agent-ലേക്ക് മാറുക.
3. നിങ്ങൾക്ക് സൃഷ്ടിക്കാൻ കഴിയുന്ന കോൺടാക്റ്റ് ഫോമിനെക്കുറിച്ച് വിവരിക്കുക. ഉദാഹരണത്തിന്:

   > "പേര്, ഇമെയിൽ, സന്ദേശം, ഫോൺ നമ്പർ എന്നിവയ്ക്കായി ഫീൽഡുകളുള്ള ഒരു കോൺടാക്റ്റ് ഫോം Contact പേജിൽ ചേർക്കുക."

   അല്ലെങ്കിൽ ലളിതമായി:

   > "Contact പേജിനായി ഒരു കോൺടാക്റ്റ് ഫോം സൃഷ്ടിക്കുക."

4. Site Builder കോൺടാക്റ്റ് ഫോം സൃഷ്ടിക്കുകയും, അത് എമബഡ് ചെയ്യാൻ തയ്യാറായ ഒരു shortcode തിരികെ നൽകുകയും ചെയ്യും.

## സൃഷ്ടിച്ച shortcode ഉപയോഗിക്കൽ {#using-the-generated-shortcode}

Site Builder കോൺടാക്റ്റ് ഫോം സൃഷ്ടിച്ച ശേഷം, അത് ഒരു shortcode നൽകും (ഉദാഹരണത്തിന്, `[contact-form-7 id="123"]`). നിങ്ങൾക്ക് ഇത്:

1. **ഒരു പേജിലോ പോസ്റ്റിലോ എമബഡ് ചെയ്യാം** — shortcode കോപ്പി ചെയ്ത്, block editor അല്ലെങ്കിൽ classic editor ഉപയോഗിച്ച് ഏത് പേജിലോ പോസ്റ്റിലോ പേസ്റ്റ് ചെയ്യുക.
2. **Site Builder വഴി ചേർക്കാം** — ഒരു പ്രത്യേക പേജിലേക്ക് ഫോം ഓട്ടോമാറ്റിക്കായി ചേർക്കാൻ Site Builder-യോട് ആവശ്യപ്പെടുക:

   > "Contact പേജിലേക്ക് കോൺടാക്റ്റ് ഫോം ചേർക്കുക."

3. **ഒരു template-ൽ ഉപയോഗിക്കാം** — നിങ്ങൾക്ക് PHP-യിൽ പരിജ്ഞാനം ഉണ്ടെങ്കിൽ, ഈ shortcode ഒരു theme template ഫയലിൽ ചേർക്കാൻ കഴിയും.

## കോൺടാക്റ്റ് ഫോം കസ്റ്റമൈസ് ചെയ്യൽ {#customizing-the-contact-form}

Site Builder കോൺടാക്റ്റ് ഫോം സൃഷ്ടിച്ച ശേഷം, നിങ്ങൾക്ക് അതിനെ കൂടുതൽ കസ്റ്റമൈസ് ചെയ്യാൻ കഴിയും:

### ചാറ്റ് ഇന്റർഫേസ് വഴി {#via-the-chat-interface}

ഫോം മാറ്റങ്ങൾ വരുത്താൻ Site Builder-യോട് ആവശ്യപ്പെടുക:

> "ഒരു subject field കൂടി ചേർക്കാനും, message field-നെ mandatory ആക്കാനും കോൺടാക്റ്റ് ഫോം അപ്ഡേറ്റ് ചെയ്യുക."

Site Builder ഫോം അപ്ഡേറ്റ് ചെയ്യുകയും അപ്ഡേറ്റ് ചെയ്ത shortcode തിരികെ നൽകുകയും ചെയ്യും.

### ഫോം plugin-ന്റെ admin ഇന്റർഫേസ് വഴി {#via-the-form-plugins-admin-interface}

നിങ്ങളുടെ ഫോം plugin-ന്റെ സെറ്റിംഗ്സിൽ നേരിട്ടും ഫോം കസ്റ്റമൈസ് ചെയ്യാൻ കഴിയും:

1. WordPress admin-ൽ **Contact Form 7** (അല്ലെങ്കിൽ നിങ്ങൾ ഇൻസ്റ്റാൾ ചെയ്ത ഫോം plugin) എന്നതിലേക്ക് പോകുക.
2. Site Builder സൃഷ്ടിച്ച ഫോം കണ്ടെത്തുക.
3. ആവശ്യാനുസരണം ഫോം ഫീൽഡുകൾ, validation rules, ഇമെയിൽ അറിയിപ്പുകൾ എന്നിവ എഡിറ്റ് ചെയ്യുക.

## ഫോം plugins ഉം അനുയോജ്യതയും {#form-plugins-and-compatibility}

നിങ്ങളുടെ സൈറ്റിൽ ഏത് ഫോം plugin ആണ് ഇൻസ്റ്റാൾ ചെയ്തിരിക്കുന്നതെന്ന് Site Builder ഓട്ടോമാറ്റിക്കായി തിരിച്ചറിയുകയും അത് ഉപയോഗിച്ചാണ് കോൺടാക്റ്റ് ഫോം സൃഷ്ടിക്കുന്നത്. ഒന്നിലധികം ഫോം plugins ഇൻസ്റ്റാൾ ചെയ്തിട്ടുണ്ടെങ്കിൽ, Site Builder അവയ്ക്ക് ഈ ക്രമത്തിൽ മുൻഗണന നൽകും:

1. Contact Form 7
2. WPForms
3. Fluent Forms
4. Gravity Forms

ഈ plugins-ൽ ഒന്നും ഇൻസ്റ്റാൾ ചെയ്തിട്ടില്ലെങ്കിൽ, കോൺടാക്റ്റ് ഫോം സൃഷ്ടിക്കുന്നതിന് മുമ്പ് ഒന്ന് ഇൻസ്റ്റാൾ ചെയ്യാൻ Site Builder ശുപാർശ ചെയ്യും.

## ഇമെയിൽ അറിയിപ്പുകൾ {#email-notifications}

Site Builder സൃഷ്ടിക്കുന്ന കോൺടാക്റ്റ് ഫോമുകൾ ഡിഫോൾട്ടായി സൈറ്റ് അഡ്മിനിസ്ട്രേറ്റർക്ക് ഇമെയിൽ അറിയിപ്പുകൾ അയക്കാനായി കോൺഫിഗർ ചെയ്തിരിക്കുന്നു. നിങ്ങൾക്ക് സ്വീകരിക്കുന്ന ഇമെയിൽ വിലാസവും അറിയിപ്പ് സന്ദേശവും കസ്റ്റമൈസ് ചെയ്യാൻ കഴിയും:

1. നിങ്ങളുടെ ഫോം plugin-ന്റെ admin ഇന്റർഫേസിലേക്ക് പോകുക.
2. Site Builder സൃഷ്ടിച്ച ഫോം കണ്ടെത്തുക.
3. ഇമെയിൽ അറിയിപ്പ് സെറ്റിംഗ്സ് എഡിറ്റ് ചെയ്യുക.

കൂടുതൽ വിശദമായ നിർദ്ദേശങ്ങൾക്കായി, നിങ്ങളുടെ ഫോം plugin-ന്റെ documentation കാണുക:

- [Contact Form 7 Email Notifications](https://contactform7.com/docs/)
- [WPForms Email Notifications](https://wpforms.com/docs/)
- [Fluent Forms Email Notifications](https://fluentforms.com/docs/)
- [Gravity Forms Email Notifications](https://docs.gravityforms.com/)

## ഉദാഹരണ ഉപയോഗ സാഹചര്യങ്ങൾ {#example-use-cases}

### ലളിതമായ കോൺടാക്റ്റ് ഫോം {#simple-contact-form}

> "പേര്, ഇമെയിൽ, സന്ദേശം എന്നിവയ്ക്കായി ലളിതമായ ഒരു കോൺടാക്റ്റ് ഫോം സൃഷ്ടിക്കുക."

### Multi-Step ഫോം {#multi-step-form}

> "കോൺടാക്റ്റ് വിവരങ്ങൾക്കായുള്ള ആദ്യ ഘട്ടവും, സന്ദേശത്തിനും ഇഷ്ടപ്പെട്ട കോൺടാക്റ്റ് രീതിക്കുമുള്ള രണ്ടാമത്തെ ഘട്ടവും ഉള്ള ഒരു കോൺടാക്റ്റ് ഫോം സൃഷ്ടിക്കുക."

### Conditional Logic ഉള്ള ഫോം {#form-with-conditional-logic}

> "ഒരു ഡ്രോഡൗണിൽ ഉപയോക്താവ് നൽകുന്നതിനെ അടിസ്ഥാനമാക്കി വ്യത്യസ്ത ഫീൽഡുകൾ കാണിക്കുന്ന ഒരു കോൺടാക്റ്റ് ഫോം സൃഷ്ടിക്കുക."

### File Upload ഉള്ള ഫോം {#form-with-file-upload}

> "ഉപയോക്താക്കൾക്ക് ഒരു ഫയലോ അറ്റാച്ച്മെന്റോ അപ്‌ലോഡ് ചെയ്യാൻ കഴിയുന്ന ഒരു കോൺടാക്റ്റ് ഫോം സൃഷ്ടിക്കുക."

:::note

:::
