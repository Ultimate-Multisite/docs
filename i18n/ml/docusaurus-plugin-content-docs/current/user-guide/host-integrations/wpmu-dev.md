# WPMU DEV ഇന്റഗ്രേഷൻ

## അവലോകനം
WPMU DEV എന്നത് WordPress സൈറ്റുകൾക്ക് ഹോസ്റ്റിംഗ്, plugins, സേവനങ്ങൾ എന്നിവ നൽകുന്ന ഒരു സമ്പൂർണ്ണ WordPress പ്ലാറ്റ്ഫോമാണ്. ഈ ഇന്റഗ്രേഷൻ Ultimate Multisite-ഉം WPMU DEV ഹോസ്റ്റിംഗും തമ്മിൽ ഓട്ടോമാറ്റിക് ഡൊമെയ്ൻ സിൻക്കിംഗും SSL സർട്ടിഫിക്കറ്റ് മാനേജ്മെന്റും സാധ്യമാക്കുന്നു.

## ഫീച്ചറുകൾ
- ഓട്ടോമാറ്റിക് ഡൊമെയ്ൻ സിൻക്കിംഗ്
- SSL സർട്ടിഫിക്കറ്റ് മാനേജ്മെന്റ്
- കൂടുതൽ SSL സർട്ടിഫിക്കറ്റ് വെരിഫിക്കേഷൻ ശ്രമങ്ങൾ

## ആവശ്യകതകൾ
നിങ്ങൾ WPMU DEV-ൽ ഹോസ്റ്റ് ചെയ്യുന്നുണ്ടെങ്കിൽ ഇന്റഗ്രേഷൻ അത് സ്വയം കണ്ടെത്തി built-in API ഉപയോഗിക്കും. WPMU DEV-ൽ ഹോസ്റ്റ് ചെയ്യുകയാണെങ്കിൽ അധിക കോൺഫിഗറേഷൻ ആവശ്യമില്ല.

WPMU DEV-ൽ ഹോസ്റ്റ് ചെയ്യുമ്പോൾ സ്വയം നിർവചിക്കപ്പെടുന്ന `WPMUDEV_HOSTING_SITE_ID` constant ഉണ്ടോ എന്ന് ഇന്റഗ്രേഷൻ പരിശോധിക്കുന്നു.

## സെറ്റപ്പ് നിർദ്ദേശങ്ങൾ

### 1. WPMU DEV ഹോസ്റ്റിംഗ് സ്ഥിരീകരിക്കുക

നിങ്ങൾ WPMU DEV-ൽ ഹോസ്റ്റ് ചെയ്യുകയാണെങ്കിൽ, ആവശ്യമായ constants ഇതിനകം നിർവചിച്ചിട്ടുണ്ടാകും. ഇനിപ്പറയുന്നവ സ്ഥിരീകരിക്കുക:

1. നിങ്ങളുടെ environment-ൽ `WPMUDEV_HOSTING_SITE_ID` constant നിർവചിച്ചിട്ടുണ്ട്
2. API ആക്സസ് ഉള്ള ഒരു ആക്ടീവ് WPMU DEV മെമ്പർഷിപ്പ് നിങ്ങൾക്കുണ്ട്

### 2. ഇന്റഗ്രേഷൻ പ്രവർത്തനക്ഷമമാക്കുക

1. നിങ്ങളുടെ WordPress admin-ൽ, Ultimate Multisite > Settings-ലേക്ക് പോകുക
2. "Domain Mapping" ടാബിലേക്ക് നാവിഗേറ്റ് ചെയ്യുക
3. "Host Integrations" എന്നതിലേക്ക് സ്ക്രോൾ ചെയ്യുക
4. WPMU DEV ഇന്റഗ്രേഷൻ പ്രവർത്തനക്ഷമമാക്കുക
5. "Save Changes" ക്ലിക്ക് ചെയ്യുക

## ഇത് എങ്ങനെ പ്രവർത്തിക്കുന്നു

### ഡൊമെയ്ൻ സിൻക്കിംഗ്

Ultimate Multisite-ൽ ഒരു ഡൊമെയ്ൻ മാപ്പ് ചെയ്യുമ്പോൾ:

1. നിങ്ങളുടെ ഹോസ്റ്റിംഗ് അക്കൗണ്ടിലേക്ക് ഡൊമെയ്ൻ ചേർക്കാൻ ഇന്റഗ്രേഷൻ WPMU DEV API ഉപയോഗിക്കുന്നു
2. ഡൊമെയ്നിന്റെ www വേർഷനും സ്വയം ചേർക്കുന്നു
3. ഡൊമെയ്ൻ കോൺഫിഗറേഷനും SSL സർട്ടിഫിക്കറ്റ് ഇഷ്യൂ ചെയ്യലും WPMU DEV കൈകാര്യം ചെയ്യുന്നു

### SSL സർട്ടിഫിക്കറ്റ് മാനേജ്മെന്റ്

SSL സർട്ടിഫിക്കറ്റുകൾ ഇഷ്യൂ ചെയ്ത് ഇൻസ്റ്റാൾ ചെയ്യാൻ കുറച്ച് സമയമെടുത്തേക്കാമെന്നതിനാൽ, WPMU DEV ഹോസ്റ്റിംഗിനായി SSL സർട്ടിഫിക്കറ്റ് വെരിഫിക്കേഷൻ ശ്രമങ്ങളുടെ എണ്ണം കൂട്ടാൻ ഇന്റഗ്രേഷൻ കോൺഫിഗർ ചെയ്തിട്ടുണ്ട്. സ്റ്റാൻഡേർഡ് 5 ശ്രമങ്ങൾക്ക് പകരം, SSL സർട്ടിഫിക്കറ്റ് വെരിഫിക്കേഷനായി ഡിഫോൾട്ടായി 10 തവണ വരെ ശ്രമിക്കും.

## പ്രധാന കുറിപ്പുകൾ

### ഡൊമെയ്ൻ നീക്കംചെയ്യൽ

നിലവിൽ, ഡൊമെയ്നുകൾ നീക്കംചെയ്യാനുള്ള മാർഗം WPMU DEV API നൽകുന്നില്ല. Ultimate Multisite-ൽ ഒരു ഡൊമെയ്ൻ മാപ്പിംഗ് നീക്കംചെയ്യുമ്പോൾ, ഡൊമെയ്ൻ നിങ്ങളുടെ WPMU DEV ഹോസ്റ്റിംഗ് അക്കൗണ്ടിൽ തുടരും. ആവശ്യമെങ്കിൽ WPMU DEV ഹോസ്റ്റിംഗ് dashboard-ൽ നിന്ന് അത് manually നീക്കംചെയ്യേണ്ടതുണ്ട്.

### API Authentication

നിങ്ങളുടെ WordPress database-ൽ `wpmudev_apikey` option ആയി സ്റ്റോർ ചെയ്തിരിക്കുന്ന WPMU DEV API key ഇന്റഗ്രേഷൻ ഉപയോഗിക്കുന്നു. നിങ്ങളുടെ സൈറ്റ് WPMU DEV-മായി കണക്റ്റ് ചെയ്യുമ്പോൾ ഇത് സ്വയം സെറ്റപ്പ് ചെയ്യപ്പെടുന്നു.

## ട്രബിൾഷൂട്ടിംഗ്

### API കണക്ഷൻ പ്രശ്നങ്ങൾ
- നിങ്ങളുടെ സൈറ്റ് WPMU DEV-മായി ശരിയായി കണക്റ്റ് ചെയ്തിട്ടുണ്ടോ എന്ന് പരിശോധിക്കുക
- നിങ്ങളുടെ WordPress database-ൽ `wpmudev_apikey` option സെറ്റ് ചെയ്തിട്ടുണ്ടോ എന്ന് പരിശോധിക്കുക
- നിങ്ങളുടെ WPMU DEV മെമ്പർഷിപ്പ് ആക്ടീവ് ആണെന്ന് ഉറപ്പാക്കുക

### SSL സർട്ടിഫിക്കറ്റ് പ്രശ്നങ്ങൾ
- SSL സർട്ടിഫിക്കറ്റുകൾ ഇഷ്യൂ ചെയ്യാൻ WPMU DEV-ന് കുറച്ച് സമയമെടുത്തേക്കാം (സാധാരണയായി 5-15 മിനിറ്റ്)
- SSL സർട്ടിഫിക്കറ്റുകൾക്കായി 10 തവണ വരെ പരിശോധിക്കാൻ ഇന്റഗ്രേഷൻ കോൺഫിഗർ ചെയ്തിട്ടുണ്ട്
- ഒന്നിലധികം ശ്രമങ്ങൾക്ക് ശേഷവും SSL സർട്ടിഫിക്കറ്റുകൾ ഇഷ്യൂ ചെയ്യുന്നില്ലെങ്കിൽ, WPMU DEV support-മായി ബന്ധപ്പെടുക

### ഡൊമെയ്ൻ ചേർക്കുന്നില്ല
- ഏതെങ്കിലും error messages ഉണ്ടോ എന്ന് Ultimate Multisite logs പരിശോധിക്കുക
- ഡൊമെയ്ൻ ഇതിനകം WPMU DEV-ലേക്ക് ചേർത്തിട്ടില്ലെന്ന് ഉറപ്പാക്കുക
- നിങ്ങൾ ചേർക്കുന്ന ഡൊമെയ്നുകളുടെ എണ്ണത്തെ നിങ്ങളുടെ WPMU DEV ഹോസ്റ്റിംഗ് പ്ലാൻ പിന്തുണയ്ക്കുന്നുണ്ടെന്ന് ഉറപ്പാക്കുക
