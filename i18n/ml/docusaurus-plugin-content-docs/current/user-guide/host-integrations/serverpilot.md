# ServerPilot ഇന്റഗ്രേഷൻ

## അവലോകനം
DigitalOcean, Amazon, Google, അല്ലെങ്കിൽ മറ്റേതെങ്കിലും സെർവർ പ്രൊവൈഡറിലെ സെർവറുകളിൽ WordPress-ഉം മറ്റ് PHP വെബ്‌സൈറ്റുകളും ഹോസ്റ്റ് ചെയ്യുന്നതിനുള്ള ഒരു ക്ലൗഡ് സേവനമാണ് ServerPilot. Ultimate Multisite-ഉം ServerPilot-ഉം തമ്മിൽ ഓട്ടോമാറ്റിക് ഡൊമെയ്ൻ സിങ്കിംഗും SSL സർട്ടിഫിക്കറ്റ് മാനേജ്‌മെന്റും സാധ്യമാക്കുന്നതാണ് ഈ ഇന്റഗ്രേഷൻ.

## ഫീച്ചറുകൾ
- ഓട്ടോമാറ്റിക് ഡൊമെയ്ൻ സിങ്കിംഗ്
- Let's Encrypt ഉപയോഗിച്ചുള്ള SSL സർട്ടിഫിക്കറ്റ് മാനേജ്‌മെന്റ്
- ഓട്ടോമാറ്റിക് SSL പുതുക്കൽ

## ആവശ്യകതകൾ
നിങ്ങളുടെ `wp-config.php` ഫയലിൽ താഴെപ്പറയുന്ന constants നിർവചിച്ചിരിക്കണം:

```php
define('WU_SERVER_PILOT_CLIENT_ID', 'your_client_id');
define('WU_SERVER_PILOT_API_KEY', 'your_api_key');
define('WU_SERVER_PILOT_APP_ID', 'your_app_id');
```

## സെറ്റപ്പ് നിർദ്ദേശങ്ങൾ

### 1. നിങ്ങളുടെ ServerPilot API ക്രെഡൻഷ്യലുകൾ നേടുക

1. നിങ്ങളുടെ ServerPilot dashboard-ൽ ലോഗിൻ ചെയ്യുക
2. "Account" > "API" എന്നതിലേക്ക് പോകുക
3. നിങ്ങൾക്ക് ഇതുവരെ ഒരു API key ഇല്ലെങ്കിൽ പുതിയതൊന്ന് സൃഷ്ടിക്കുക
4. നിങ്ങളുടെ Client ID-യും API Key-യും കോപ്പി ചെയ്യുക

### 2. നിങ്ങളുടെ App ID നേടുക

1. നിങ്ങളുടെ ServerPilot dashboard-ൽ "Apps" എന്നതിലേക്ക് പോകുക
2. നിങ്ങളുടെ WordPress multisite ഹോസ്റ്റ് ചെയ്തിരിക്കുന്ന app തിരഞ്ഞെടുക്കുക
3. URL-ൽ App ID കാണാം: `https://manage.serverpilot.io/apps/{APP_ID}`

### 3. wp-config.php-ൽ Constants ചേർക്കുക

നിങ്ങളുടെ `wp-config.php` ഫയലിൽ താഴെപ്പറയുന്ന constants ചേർക്കുക:

```php
define('WU_SERVER_PILOT_CLIENT_ID', 'your_client_id');
define('WU_SERVER_PILOT_API_KEY', 'your_api_key');
define('WU_SERVER_PILOT_APP_ID', 'your_app_id');
```

### 4. ഇന്റഗ്രേഷൻ പ്രവർത്തനക്ഷമമാക്കുക

1. നിങ്ങളുടെ WordPress admin-ൽ Ultimate Multisite > Settings എന്നതിലേക്ക് പോകുക
2. "Domain Mapping" ടാബിലേക്ക് നാവിഗേറ്റ് ചെയ്യുക
3. "Host Integrations" എന്നതിലേക്ക് സ്ക്രോൾ ചെയ്യുക
4. ServerPilot ഇന്റഗ്രേഷൻ പ്രവർത്തനക്ഷമമാക്കുക
5. "Save Changes" ക്ലിക്ക് ചെയ്യുക

## ഇത് എങ്ങനെ പ്രവർത്തിക്കുന്നു

### ഡൊമെയ്ൻ സിങ്കിംഗ്

Ultimate Multisite-ൽ ഒരു ഡൊമെയ്ൻ മാപ്പ് ചെയ്യുമ്പോൾ:

1. ഇന്റഗ്രേഷൻ ServerPilot-ൽ നിന്ന് നിലവിലുള്ള ഡൊമെയ്ൻ ലിസ്റ്റ് ശേഖരിക്കുന്നു
2. പുതിയ ഡൊമെയ്ൻ ലിസ്റ്റിലേക്ക് ചേർക്കുന്നു (ബാധകമെങ്കിൽ www പതിപ്പും ഉൾപ്പെടെ)
3. അപ്‌ഡേറ്റ് ചെയ്ത ലിസ്റ്റ് API വഴി ServerPilot-ലേക്ക് അയയ്ക്കുന്നു
4. ServerPilot നിങ്ങളുടെ ആപ്ലിക്കേഷന്റെ ഡൊമെയ്ൻ ലിസ്റ്റ് അപ്‌ഡേറ്റ് ചെയ്യുന്നു

### SSL സർട്ടിഫിക്കറ്റ് മാനേജ്‌മെന്റ്

ഡൊമെയ്നുകൾ സിങ്ക് ചെയ്തതിന് ശേഷം:

1. ഇന്റഗ്രേഷൻ നിങ്ങളുടെ ആപ്ലിക്കേഷനായി AutoSSL സ്വയമേവ പ്രവർത്തനക്ഷമമാക്കുന്നു
2. Let's Encrypt ഉപയോഗിച്ച് SSL സർട്ടിഫിക്കറ്റ് ഇഷ്യു ചെയ്യലും ഇൻസ്റ്റാൾ ചെയ്യലും ServerPilot കൈകാര്യം ചെയ്യുന്നു
3. SSL സർട്ടിഫിക്കറ്റുകളുടെ ഓട്ടോമാറ്റിക് പുതുക്കലും ServerPilot കൈകാര്യം ചെയ്യുന്നു

## SSL സർട്ടിഫിക്കറ്റ് വെരിഫിക്കേഷൻ

ServerPilot-ന് SSL സർട്ടിഫിക്കറ്റുകൾ ഇഷ്യു ചെയ്യാനും ഇൻസ്റ്റാൾ ചെയ്യാനും കുറച്ച് സമയം എടുത്തേക്കാം എന്നതിനാൽ, SSL സർട്ടിഫിക്കറ്റ് വെരിഫിക്കേഷൻ ശ്രമങ്ങളുടെ എണ്ണം വർദ്ധിപ്പിക്കാൻ ഇന്റഗ്രേഷൻ ക്രമീകരിച്ചിരിക്കുന്നു. സ്ഥിരസ്ഥിതിയായി, ഇത് 5 തവണ വരെ ശ്രമിക്കും, എന്നാൽ ഇത് filters ഉപയോഗിച്ച് ക്രമീകരിക്കാവുന്നതാണ്.

## ട്രബിൾഷൂട്ടിംഗ്

### API കണക്ഷൻ പ്രശ്നങ്ങൾ
- നിങ്ങളുടെ Client ID-യും API Key-യും ശരിയാണെന്ന് ഉറപ്പാക്കുക
- നിങ്ങളുടെ App ID ശരിയാണെന്ന് പരിശോധിക്കുക
- നിങ്ങളുടെ ServerPilot അക്കൗണ്ടിന് ആവശ്യമായ അനുമതികൾ ഉണ്ടെന്ന് ഉറപ്പാക്കുക

### SSL സർട്ടിഫിക്കറ്റ് പ്രശ്നങ്ങൾ
- SSL സർട്ടിഫിക്കറ്റുകൾ ഇഷ്യു ചെയ്യുന്നതിന് മുമ്പ് ഡൊമെയ്നുകൾക്ക് നിങ്ങളുടെ സെർവറിലേക്ക് പോയിന്റ് ചെയ്യുന്ന സാധുവായ DNS റെക്കോർഡുകൾ ഉണ്ടായിരിക്കണമെന്ന് ServerPilot ആവശ്യപ്പെടുന്നു
- SSL സർട്ടിഫിക്കറ്റുകൾ ഇഷ്യു ചെയ്യുന്നില്ലെങ്കിൽ, നിങ്ങളുടെ ഡൊമെയ്നുകൾ നിങ്ങളുടെ സെർവറിന്റെ IP വിലാസത്തിലേക്ക് ശരിയായി പോയിന്റ് ചെയ്യുന്നുണ്ടോ എന്ന് പരിശോധിക്കുക
- ServerPilot-ന് SSL സർട്ടിഫിക്കറ്റുകൾ ഇഷ്യു ചെയ്യാനും ഇൻസ്റ്റാൾ ചെയ്യാനും കുറച്ച് സമയം എടുത്തേക്കാം (സാധാരണയായി 5-15 മിനിറ്റ്)

### ഡൊമെയ്ൻ ചേർക്കപ്പെട്ടില്ല
- എന്തെങ്കിലും പിശക് സന്ദേശങ്ങൾ ഉണ്ടോ എന്ന് Ultimate Multisite ലോഗുകൾ പരിശോധിക്കുക
- ഡൊമെയ്ൻ ഇതിനകം ServerPilot-ൽ ചേർത്തിട്ടില്ലെന്ന് ഉറപ്പാക്കുക
- നിങ്ങൾ ചേർക്കുന്ന ഡൊമെയ്നുകളുടെ എണ്ണം നിങ്ങളുടെ ServerPilot പ്ലാൻ പിന്തുണയ്ക്കുന്നുണ്ടെന്ന് ഉറപ്പാക്കുക

### ഡൊമെയ്ൻ നീക്കംചെയ്യൽ
- നിലവിൽ, ServerPilot API വ്യക്തിഗത ഡൊമെയ്നുകൾ നീക്കംചെയ്യാനുള്ള മാർഗ്ഗം നൽകുന്നില്ല
- Ultimate Multisite-ൽ ഒരു ഡൊമെയ്ൻ മാപ്പിംഗ് നീക്കംചെയ്യുമ്പോൾ, നീക്കംചെയ്ത ഡൊമെയ്ൻ ഒഴിവാക്കി ServerPilot-ലെ ഡൊമെയ്ൻ ലിസ്റ്റ് ഇന്റഗ്രേഷൻ അപ്‌ഡേറ്റ് ചെയ്യും
