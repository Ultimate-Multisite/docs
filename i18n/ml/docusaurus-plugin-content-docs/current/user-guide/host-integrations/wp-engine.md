# WP Engine ഇന്റഗ്രേഷൻ

## അവലോകനം
WP Engine ഒരു പ്രീമിയം മാനേജ്ഡ് WordPress ഹോസ്റ്റിംഗ് പ്ലാറ്റ്‌ഫോമാണ്, WordPress സൈറ്റുകൾക്ക് ഒപ്റ്റിമൈസ് ചെയ്ത പെർഫോമൻസ്, സെക്യൂരിറ്റി, സ്കേലബിലിറ്റി എന്നിവ നൽകുന്നു. ഈ ഇന്റഗ്രേഷൻ Ultimate Multisite-ഉം WP Engine-ഉം തമ്മിലുള്ള ഓട്ടോമാറ്റിക് ഡൊമെയ്ൻ സിങ്കിംഗ് സാധ്യമാക്കുന്നു.

## ഫീച്ചറുകൾ
- ഓട്ടോമാറ്റിക് ഡൊമെയ്ൻ സിങ്കിംഗ്
- മൾട്ടിസൈറ്റ് ഇൻസ്റ്റാളേഷനുകൾക്ക് സബ്‌ഡൊമെയ്ൻ പിന്തുണ
- WP Engine-ന്റെ നിലവിലുള്ള സിസ്റ്റങ്ങളുമായി തടസ്സമില്ലാത്ത ഇന്റഗ്രേഷൻ

## ആവശ്യകതകൾ
നിങ്ങൾ WP Engine-ൽ ഹോസ്റ്റ് ചെയ്യുന്നുണ്ടോ എന്ന് ഈ ഇന്റഗ്രേഷൻ ഓട്ടോമാറ്റിക്കായി കണ്ടെത്തുകയും ബിൽറ്റ്-ഇൻ WP Engine API ഉപയോഗിക്കുകയും ചെയ്യുന്നു. WP Engine plugin ആക്ടീവും ശരിയായി കോൺഫിഗർ ചെയ്തതുമാണെങ്കിൽ അധിക കോൺഫിഗറേഷൻ ആവശ്യമില്ല.

എന്നിരുന്നാലും, നിങ്ങൾക്ക് ഇന്റഗ്രേഷൻ മാനുവലായി കോൺഫിഗർ ചെയ്യണമെങ്കിൽ, നിങ്ങളുടെ `wp-config.php` ഫയലിൽ ഈ കോൺസ്റ്റന്റുകളിൽ ഒന്ന് നിർവചിക്കാം:

```php
define('WPE_APIKEY', 'your_api_key'); // ശുപാർശ ചെയ്യുന്ന രീതി
// അല്ലെങ്കിൽ
define('WPE_API', 'your_api_key'); // ഇതര രീതി
```

## സെറ്റപ്പ് നിർദ്ദേശങ്ങൾ

### 1. WP Engine Plugin പരിശോധിക്കുക

നിങ്ങൾ WP Engine-ൽ ഹോസ്റ്റ് ചെയ്യുകയാണെങ്കിൽ, WP Engine plugin ഇതിനകം ഇൻസ്റ്റാൾ ചെയ്ത് ആക്ടിവേറ്റ് ചെയ്തിരിക്കണം. ഇനിപ്പറയുന്നവ പരിശോധിക്കുക:

1. WP Engine plugin ആക്ടീവാണ്
2. `wp-content/mu-plugins/wpengine-common/class-wpeapi.php` ഫയൽ നിലവിലുണ്ട്

### 2. ഇന്റഗ്രേഷൻ എനേബിൾ ചെയ്യുക

1. നിങ്ങളുടെ WordPress admin-ൽ, Ultimate Multisite > Settings-ലേക്ക് പോകുക
2. "Domain Mapping" ടാബിലേക്ക് നാവിഗേറ്റ് ചെയ്യുക
3. "Host Integrations" എന്നതിലേക്ക് സ്ക്രോൾ ചെയ്യുക
4. WP Engine ഇന്റഗ്രേഷൻ എനേബിൾ ചെയ്യുക
5. "Save Changes" ക്ലിക്ക് ചെയ്യുക

## ഇത് എങ്ങനെ പ്രവർത്തിക്കുന്നു

### ഡൊമെയ്ൻ സിങ്കിംഗ്

Ultimate Multisite-ൽ ഒരു ഡൊമെയ്ൻ മാപ്പ് ചെയ്യുമ്പോൾ:

1. നിങ്ങളുടെ WP Engine ഇൻസ്റ്റാളേഷനിലേക്ക് ഡൊമെയ്ൻ ചേർക്കാൻ ഇന്റഗ്രേഷൻ WP Engine API ഉപയോഗിക്കുന്നു
2. ഡൊമെയ്ൻ കോൺഫിഗറേഷനും SSL സർട്ടിഫിക്കറ്റ് ഇഷ്യൂ ചെയ്യലും WP Engine കൈകാര്യം ചെയ്യുന്നു
3. ഡൊമെയ്ൻ മാപ്പിംഗ് നീക്കം ചെയ്യുമ്പോൾ, WP Engine-ൽ നിന്ന് ഡൊമെയ്ൻ ഇന്റഗ്രേഷൻ നീക്കം ചെയ്യും

### സബ്‌ഡൊമെയ്ൻ പിന്തുണ

സബ്‌ഡൊമെയ്ൻ മൾട്ടിസൈറ്റ് ഇൻസ്റ്റാളേഷനുകൾക്ക്:

1. പുതിയ സൈറ്റ് സൃഷ്ടിക്കുമ്പോൾ ഓരോ സബ്‌ഡൊമെയ്നും WP Engine-ലേക്ക് ഇന്റഗ്രേഷൻ ചേർക്കുന്നു
2. സബ്‌ഡൊമെയ്ൻ കോൺഫിഗറേഷൻ WP Engine കൈകാര്യം ചെയ്യുന്നു
3. സൈറ്റ് ഡിലീറ്റ് ചെയ്യുമ്പോൾ, WP Engine-ൽ നിന്ന് സബ്‌ഡൊമെയ്ൻ ഇന്റഗ്രേഷൻ നീക്കം ചെയ്യും

## പ്രധാനപ്പെട്ട കുറിപ്പുകൾ

### വൈൽഡ്കാർഡ് ഡൊമെയ്നുകൾ

സബ്‌ഡൊമെയ്ൻ മൾട്ടിസൈറ്റ് ഇൻസ്റ്റാളേഷനുകൾക്ക്, വൈൽഡ്കാർഡ് ഡൊമെയ്ൻ കോൺഫിഗറേഷൻ അഭ്യർത്ഥിക്കാൻ WP Engine സപ്പോർട്ടുമായി ബന്ധപ്പെടാൻ ശുപാർശ ചെയ്യുന്നു. ഇത് ഓരോ സബ്‌ഡൊമെയ്നും വ്യക്തിഗതമായി ചേർക്കേണ്ടതില്ലാതെ എല്ലാ സബ്‌ഡൊമെയ്നുകളും ഓട്ടോമാറ്റിക്കായി പ്രവർത്തിക്കാൻ അനുവദിക്കുന്നു.

### SSL സർട്ടിഫിക്കറ്റുകൾ

ഈ ഇന്റഗ്രേഷൻ വഴി ചേർക്കുന്ന എല്ലാ ഡൊമെയ്നുകൾക്കും SSL സർട്ടിഫിക്കറ്റ് ഇഷ്യൂ ചെയ്യലും പുതുക്കലും WP Engine ഓട്ടോമാറ്റിക്കായി കൈകാര്യം ചെയ്യുന്നു. അധിക കോൺഫിഗറേഷൻ ആവശ്യമില്ല.

## പ്രശ്നപരിഹാരം

### API കണക്ഷൻ പ്രശ്നങ്ങൾ
- WP Engine plugin ആക്ടീവും ശരിയായി കോൺഫിഗർ ചെയ്തതുമാണെന്ന് പരിശോധിക്കുക
- നിങ്ങൾ API key മാനുവലായി നിർവചിച്ചിട്ടുണ്ടെങ്കിൽ, അത് ശരിയാണോ എന്ന് പരിശോധിക്കുക
- API-യുമായി പ്രശ്നങ്ങൾ ഉണ്ടെങ്കിൽ WP Engine സപ്പോർട്ടുമായി ബന്ധപ്പെടുക

### ഡൊമെയ്ൻ ചേർക്കപ്പെട്ടില്ല
- എന്തെങ്കിലും എറർ മെസേജുകൾക്കായി Ultimate Multisite logs പരിശോധിക്കുക
- ഡൊമെയ്ൻ ഇതിനകം WP Engine-ലേക്ക് ചേർത്തിട്ടില്ലെന്ന് ഉറപ്പാക്കുക
- നിങ്ങൾ ചേർക്കുന്ന ഡൊമെയ്നുകളുടെ എണ്ണത്തെ നിങ്ങളുടെ WP Engine പ്ലാൻ പിന്തുണയ്ക്കുന്നുണ്ടെന്ന് ഉറപ്പാക്കുക

### സബ്‌ഡൊമെയ്ൻ പ്രശ്നങ്ങൾ
- സബ്‌ഡൊമെയ്നുകൾ പ്രവർത്തിക്കുന്നില്ലെങ്കിൽ, വൈൽഡ്കാർഡ് ഡൊമെയ്ൻ കോൺഫിഗറേഷൻ അഭ്യർത്ഥിക്കാൻ WP Engine സപ്പോർട്ടുമായി ബന്ധപ്പെടുക
- പ്രധാന ഡൊമെയ്നിനും സബ്‌ഡൊമെയ്നുകൾക്കും നിങ്ങളുടെ DNS സെറ്റിംഗ്സ് ശരിയായി കോൺഫിഗർ ചെയ്തിട്ടുണ്ടെന്ന് ഉറപ്പാക്കുക
