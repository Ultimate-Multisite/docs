# സഹായം എങ്ങനെ ലഭിക്കും

നിങ്ങൾക്ക് സാധുവായ License key ലഭിച്ചു, അതോടൊപ്പം Ultimate Multisite സജ്ജീകരിക്കുമ്പോൾ നേരിടുന്ന പ്രശ്നങ്ങൾ പരിഹരിക്കാൻ ഞങ്ങളുടെ ടീമിന്റെ സഹായവും ലഭ്യമാണ്.

ലൈസൻസ് ഉടമകൾക്ക് ലഭ്യമായ സപ്പോർട്ട് ചാനലുകളും ഞങ്ങളുടെ സപ്പോർട്ട് ടീമിനെ ബന്ധപ്പെടാൻ ആവശ്യമായ വിവരങ്ങളും ഈ പാഠത്തിൽ വിശദീകരിക്കുന്നു.

## ഹെൽപ്പ് സൈറ്റ്

ചുവടെയുള്ള ലിങ്കിൽ ലഭ്യമായ ഞങ്ങളുടെ ഹെൽപ്പ് സൈറ്റിൽ നിന്ന് വിവിധ ലേഖനങ്ങളും ട്യൂട്ടോറിയലുകളും നിങ്ങൾക്ക് കണ്ടെത്താം.

[)

Ultimate Multisite-ന്റെ അടിസ്ഥാന ഫീച്ചറുകളും പ്രവർത്തനങ്ങളും പരിചയപ്പെടാൻ ഞങ്ങൾ ഉപയോക്താക്കളെ പ്രോത്സാഹിപ്പിക്കുന്നു. ഹെൽപ്പ് സൈറ്റിൽ "Getting Started" ലേഖനങ്ങൾ ലഭ്യമാണ്, കൂടാതെ add-ons എങ്ങനെ ഇൻസ്റ്റാൾ ചെയ്യാം, payment gateways എങ്ങനെ സംയോജിപ്പിക്കാം തുടങ്ങിയ സാധാരണ ചോദ്യങ്ങൾക്കുള്ള ഉത്തരങ്ങളും അവിടെ കാണാം.

## സപ്പോർട്ടിനെ ബന്ധപ്പെടുക

നിങ്ങൾ എവിടെയെങ്കിലും കുടുങ്ങിപ്പോയാൽ, ഹെൽപ്പ് സൈറ്റിൽ നിന്ന് ഉത്തരം കണ്ടെത്താൻ കഴിയുന്നില്ലെങ്കിൽ, ഞങ്ങളുടെ വെബ്സൈറ്റിന്റെ വലതുവശത്ത് താഴെയായുള്ള സ്പീച്ച് ബബിൾ ഐക്കണിൽ ക്ലിക്ക് ചെയ്തോ support@ultimatemultisite.com എന്ന വിലാസത്തിലേക്ക് നേരിട്ട് ഇമെയിൽ അയച്ചോ ഞങ്ങളുടെ സപ്പോർട്ട് ടീമിനെ ബന്ധപ്പെടാം.

സാങ്കേതിക പ്രശ്നങ്ങൾക്കായി സപ്പോർട്ടിനെ ബന്ധപ്പെടുമ്പോൾ, നിങ്ങളുടെ പ്രശ്നത്തിന്റെ വിശദമായ വിവരണം, സ്ക്രീൻഷോട്ട്, എറർ മെസേജുകൾ, അല്ലെങ്കിൽ ഞങ്ങളുടെ സപ്പോർട്ട് ടീമിന് നിങ്ങളുടെ പ്രശ്നം എളുപ്പത്തിൽ മനസ്സിലാക്കാനും വിലയിരുത്താനും സഹായകമാകുന്ന മറ്റേതെങ്കിലും വിവരങ്ങളും നൽകുന്നുവെന്ന് ഉറപ്പാക്കുക. നിങ്ങളുടെ license key കൂടി നൽകേണ്ടതുണ്ട്. സജീവ ലൈസൻസ് ഉടമകൾക്ക് മാത്രമേ സപ്പോർട്ടിന് അർഹതയുള്ളൂ.

## കമ്മ്യൂണിറ്റി സൈറ്റ്

ഞങ്ങളുടെ [കമ്മ്യൂണിറ്റി സൈറ്റ്](https://community.ultimatemultisite.com/) Ultimate Multisite ഉപയോക്താക്കൾക്ക് പരസ്പരം സംവദിക്കാനുള്ള ഒരു വേദിയാണ്. അംഗങ്ങൾക്ക് പോസ്റ്റ് ചെയ്യാനും, ചർച്ചകളിൽ കമന്റ് ചെയ്യാനും, ഉപദേശം നൽകാനും, സഹകരിക്കാനും, ക്രൗഡ്സോഴ്സ് ചെയ്യാനും കഴിയുന്ന ഒരു വിവര വ്യവസ്ഥയായി ഇത് പ്രവർത്തിക്കുന്നു. സ്വകാര്യ സ്പേസുകൾ/പേജുകൾ അൺലോക്ക് ചെയ്യാൻ നിങ്ങളുടെ Ultimate Multisite license key നൽകേണ്ടതുണ്ട്.
