# പാഠം 7: ഇത് നിങ്ങളുടെ സ്വന്തമാക്കുക

നിങ്ങളുടെ ഉപഭോക്താക്കൾ ഒരിക്കലും "ഒരു വേറെ WordPress plugin ആണ് ഉപയോഗിക്കുന്നത്" എന്ന തോന്നൽ ഉണ്ടാകരുത്. അവർക്ക് തോന്നേണ്ടത്, ഇത് FitSite ആണ്—അവരുടെ വ്യവസായത്തിനായി നിർമ്മിച്ച ഒരു പ്ലാറ്റ്‌ഫോം. ബ്രാൻഡിംഗിനെക്കുറിച്ചും, വൈറ്റ്-ലേബലിംഗിനെക്കുറിച്ചും, ഈ പ്ലാറ്റ്‌ഫോമിനെ ഒരു ഉൽപ്പന്നം പോലെ തോന്നിക്കാൻ വേണ്ട കാര്യങ്ങളെക്കുറിച്ചാണ് ഈ പാഠം പറയുന്നത്.

## നമ്മൾ എവിടെ വെച്ച് നിർത്തി

FitSite-ന് ഒരു പ്രവർത്തനക്ഷമമായ checkout flow ഉണ്ട്. ഇത് ഫിറ്റ്നസ് സ്റ്റുഡിയോ ഉടമകളെ പ്ലാൻ തിരഞ്ഞെടുക്കുന്നതിൽ നിന്ന് ഒരു ലൈവ് സൈറ്റിലേക്ക് എത്തിക്കുന്നു. ഇപ്പോൾ, ഈ മുഴുവൻ അനുഭവത്തെയും ഒത്തൊരുമയുള്ള, ബ്രാൻഡഡ് ഉൽപ്പന്നം പോലെ തോന്നအောင် നമുക്ക് മാറ്റാം.

## നിങ്ങളുടെ പ്ലാറ്റ്‌ഫോം ഡൊമെയ്ൻ

നിങ്ങളുടെ ബ്രാൻഡിംഗിന്റെ അടിത്തറ നിങ്ങളുടെ ഡൊമെയ്നാണ്. FitSite-ന് വേണ്ടി:

- **പ്രധാന ഡൊമെയ്ൻ (Main domain)**: `fitsite.com` (നിങ്ങളുടെ മാർക്കറ്റിംഗ് സൈറ്റും നെറ്റ്‌വർക്ക് റൂട്ട് ആണ് ഇത്)
- **ഉപഭോക്തൃ സൈറ്റുകൾ (Customer sites)**: `studioname.fitsite.com` (subdomains)
- **കസ്റ്റം ഡൊമെയ്‌നുകൾ (Custom domains)**: Growth, Pro പ്ലാനുകളിലുള്ള ഉപഭോക്താക്കൾക്ക് സ്വന്തമായി ഡൊമെയ്ൻ മാപ്പ് ചെയ്യാൻ കഴിയും

### നിങ്ങളുടെ ഡൊമെയ്ൻ സജ്ജമാക്കൽ

1. നിങ്ങളുടെ പ്ലാറ്റ്‌ഫോം ഡൊമെയ്ൻ രജിസ്റ്റർ ചെയ്യുക
2. അത് നിങ്ങളുടെ ഹോസ്റ്റിംഗ് പ്രൊവൈഡറിലേക്ക് പോയിന്റ് ചെയ്യുക
3. ഉപഭോക്തൃ സബ്ഡൊമെയ്‌നുകൾക്കായി വൈൽഡ്കാർഡ് DNS (`*.fitsite.com`) സജ്ജമാക്കുക
4. വൈൽഡ്കാർഡ് SSL സജീവമാണെന്ന് ഉറപ്പാക്കുക

വിശദമായ നിർദ്ദേശങ്ങൾക്കായി [How to Configure Domain Mapping](/user-guide/domain-mapping/how-to-configure-domain-mapping) കാണുക.

## അഡ്മിൻ അനുഭവം വൈറ്റ്-ലേബൽ ചെയ്യുന്നത്

ഒരു ഫിറ്റ്നസ് സ്റ്റുഡിയോ ഉടമ അവരുടെ സൈറ്റ് ഡാഷ്‌ബോർഡിൽ ലോഗിൻ ചെയ്യുമ്പോൾ, അവർക്ക് WordPress അല്ലെങ്കിൽ Ultimate Multisite-ന്റെ ബ്രാൻഡിംഗ് കാണരുത്. അവർക്ക് നിങ്ങളുടെ ബ്രാൻഡ് കാണണം.

### കസ്റ്റം ലോഗിൻ പേജ്

WordPress ലോഗിൻ പേജ് കസ്റ്റമൈസ് ചെയ്ത് കാണിക്കുക:

- നിങ്ങളുടെ FitSite ലോഗോ
- ഫിറ്റ്നസിന് അനുയോജ്യമായ പശ്ചാത്തല ചിത്രങ്ങൾ
- നിങ്ങളുടെ ബ്രാൻഡ് നിറങ്ങൾ

### ഡാഷ്‌ബോർഡ് ബ്രാൻഡിംഗ്

[Admin Page Creator](/addons/admin-page-creator) ആഡ്ഓൺ അല്ലെങ്കിൽ കസ്റ്റം CSS ഉപയോഗിച്ച്:

- WordPress ലോഗോ നിങ്ങളുടെ FitSite ലോഗോ ഉപയോഗിച്ച് മാറ്റിസ്ഥാപിക്കുക
- നിങ്ങളുടെ ബ്രാൻഡിന് അനുയോജ്യമായ രീതിയിൽ അഡ്മിൻ കളർ സ്കീം കസ്റ്റമൈസ് ചെയ്യുക
- ഫിറ്റ്നസ്-നിർദ്ദിഷ്ട ക്വിക്ക് ലിങ്കുകളും സഹായ വിഭവങ്ങളും ഉൾപ്പെടുത്തി ഒരു കസ്റ്റം ഡാഷ്‌ബോർഡ് വിഡ്ജറ്റ് ചേർക്കുക

### കസ്റ്റം അഡ്മിൻ പേജുകൾ

ഫിറ്റ്നസ് സ്റ്റുഡിയോ ഉടമകൾക്ക് ഏറ്റവും പ്രസക്തമായ പ്രവർത്തനങ്ങൾ എളുപ്പത്തിൽ കാണിക്കുന്ന കസ്റ്റം അഡ്മിൻ പേജുകൾ ഉണ്ടാക്കാൻ പരിഗണിക്കുക:

- "Edit Your Class Schedule" (നിങ്ങളുടെ ക്ലാസ് ഷെഡ്യൂൾ എഡിറ്റ് ചെയ്യുക)
- "Update Trainer Profiles" (ട്രെയിനർ പ്രൊഫൈലുകൾ അപ്ഡേറ്റ് ചെയ്യുക)
- "View Your Site" (നിങ്ങളുടെ സൈറ്റ് കാണുക)

ഇത് സാധാരണ WordPress മെനുവിൽ മറച്ചുപിടിക്കുന്നതിന് പകരം, ആവശ്യമായ പ്രവർത്തനങ്ങൾ മുൻപന്തിയിൽ കൊണ്ടുവരുന്നതിലൂടെ പഠിക്കാനുള്ള ബുദ്ധിമുട്ട് കുറയ്ക്കുന്നു.

## നിങ്ങളുടെ ആശയവിനിമയങ്ങൾ ബ്രാൻഡിംഗ് ചെയ്യുക

ഓരോ ഇമെയിലും, ഇൻവോയ്സും, അറിയിപ്പുകളും നിങ്ങളുടെ ബ്രാൻഡിനെ ശക്തിപ്പെടുത്തണം.

### സിസ്റ്റം ഇമെയിലുകൾ

**Ultimate Multisite > Settings > Emails** എന്നതിലേക്ക് പോവുക, എല്ലാ സിസ്റ്റം ഇമെയിലുകളും കസ്റ്റമൈസ് ചെയ്യുക:

- **From name**: FitSite
- **From email**: hello@fitsite.com
- **Email templates**: നിങ്ങളുടെ ബ്രാൻഡ് നിറങ്ങളും ലോഗോയും ഉപയോഗിക്കുക
- **Language**: ഫിറ്റ്നസ്-നിർദ്ദിഷ്ടമായി എല്ലാ കാര്യങ്ങളിലും

കസ്റ്റമൈസ് ചെയ്യേണ്ട പ്രധാന ഇമെയിലുകൾ:

| ഇമെയിൽ | പൊതുവായ പതിപ്പ് (Generic Version) | FitSite പതിപ്പ് |
|-------|----------------|-----------------|
| സ്വാഗതം (Welcome) | "നിങ്ങളുടെ പുതിയ സൈറ്റ് തയ്യാറാണ്" | "നിങ്ങളുടെ ഫിറ്റ്നസ് സ്റ്റുഡിയോ വെബ്സൈറ്റ് ലൈവാണ്" |
| പേയ്‌മെന്റ് രസീത് (Payment receipt) | "പേയ്‌മെന്റ് ലഭിച്ചു" | "FitSite സബ്‌സ്‌ക്രിപ്‌ഷൻ പേയ്‌മെന്റ് സ്ഥിരീകരിച്ചു" |
| ട്രയൽ അവസാനിക്കുമ്പോൾ (Trial ending) | "നിങ്ങളുടെ ട്രയൽ ഉടൻ അവസാനിക്കാൻ പോകുന്നു" | "നിങ്ങളുടെ FitSite ട്രയൽ 3 ദിവസത്തിനുള്ളിൽ അവസാനിക്കുന്നു -- നിങ്ങളുടെ സ്റ്റുഡിയോ വെബ്സൈറ്റ് ലൈവായി നിലനിർത്തുക" |

### ഇൻവോയ്സുകൾ

ഇൻവോയ്സ് ടെംപ്ലേറ്റുകൾ കസ്റ്റമൈസ് ചെയ്യുമ്പോൾ ഇവ ഉൾപ്പെടുത്തുക:

- നിങ്ങളുടെ FitSite ലോഗോയും ബ്രാൻഡ് നിറങ്ങളും
- നിങ്ങളുടെ ബിസിനസ് വിവരങ്ങൾ
- ഫിറ്റ്നസ്-നിർദ്ദിഷ്ട ഉൽപ്പന്ന പേരുകൾ (പൊതുവായ പ്ലാൻ ഐഡികൾ അല്ല)

## ഉപഭോക്താക്കൾ കാണുന്ന സൈറ്റ്

നിങ്ങളുടെ പ്രധാന ഡൊമെയ്ൻ (`fitsite.com`) പ്ലാറ്റ്‌ഫോം വിൽക്കുന്ന ഒരു മാർക്കറ്റിംഗ് സൈറ്റ് ആവശ്യമാണ്. ഇത് Ultimate Multisite നെറ്റ്‌വർക്ക് അഡ്മിന്റിൽ നിന്ന് വേറിട്ടതാണ്—ഇതാണ് നിങ്ങളുടെ ബിസിനസ്സിന്റെ പൊതുവായ മുഖം.

പ്രധാന പേജുകൾ:

- **ഹോംപേജ് (Homepage)**: ഫിറ്റ്നസ് ബിസിനസ്സുകൾക്കുള്ള വ്യക്തമായ മൂല്യ പ്രസ്താവന (value proposition)
- **ఫీച്ചറുകൾ (Features)**: FitSite എന്ത് ചെയ്യുന്നു, ഫിറ്റ്നസ് പദങ്ങളിൽ
- **Pricing**: നിങ്ങളുടെ മൂന്ന് പ്ലാനുകളും, വ്യവസായ-നിർദ്ദിഷ്ട ഫീച്ചർ താരതമ്യങ്ങൾ
- **Examples**: പ്ലാറ്റ്‌ഫോമിൽ നിർമ്മിച്ച സൈറ്റുകൾ പ്രദർശിപ്പിക്കുക
- **Sign Up**: നിങ്ങളുടെ checkout ഫോമിലേക്കുള്ള ലിങ്കുകൾ

:::tip നിങ്ങളുടെ മാർക്കറ്റിംഗ് സൈറ്റ് ഒരു നെറ്റ്‌വർക്ക് സൈറ്റായിരിക്കാം
നിങ്ങളുടെ സ്വന്തം നെറ്റ്‌വർക്കിലെ ഒരു സൈറ്റായി നിങ്ങളുടെ മാർക്കറ്റിംഗ് സൈറ്റ് സൃഷ്ടിക്കുക. ഇത് നിങ്ങൾക്ക് ഒരേ ഡാഷ്‌ബോർഡിൽ നിന്ന് അത് കൈകാര്യം ചെയ്യാൻ അനുവദിക്കുകയും നിങ്ങളുടെ സ്വന്തം പ്ലാറ്റ്‌ഫോമിന്റെ കഴിവുകൾ പ്രദർശിപ്പിക്കുകയും ചെയ്യും.
:::

## ഉപഭോക്താക്കൾക്കുള്ള കസ്റ്റം ഡൊമെയ്ൻ

കസ്റ്റം ഡൊമെയ്‌നുകൾ ഉൾപ്പെടുന്ന പ്ലാനുകളിലുള്ള ഉപഭോക്താക്കൾക്കായി, ഈ പ്രക്രിയ വ്യക്തമായി രേഖപ്പെടുത്തുക:

1. ഉപഭോക്താവ് അവരുടെ ഡൊമെയ്ൻ ഒരു രജിസ്ട്രാറിലേക്ക് രജിസ്റ്റർ ചെയ്യുകയോ ട്രാൻസ്ഫർ ചെയ്യുകയോ ചെയ്യുന്നു
2. ഉപഭോക്താവ് DNS അപ്ഡേറ്റ് ചെയ്ത് നിങ്ങളുടെ പ്ലാറ്റ്‌ഫോമിലേക്ക് പോയിന്റ് ചെയ്യുന്നു (കൃത്യമായ റെക്കോർഡുകൾ നൽകുക)
3. Ultimate Multisite ഡൊമെയ്ൻ മാപ്പിംഗും SSL-ഉം കൈകാര്യം ചെയ്യുന്നു

ഈ പ്രക്രിയയ്ക്കായി, സാങ്കേതിക പരിജ്ഞാനം ഇല്ലാത്ത ഫിറ്റ്നസ് സ്റ്റുഡിയോ ഉടമകൾക്കായി എഴുതിയ ഒരു സഹായ ലേഖനം അല്ലെങ്കിൽ knowledge base എൻട്രി സൃഷ്ടിക്കുക.

- ബ്രാൻഡഡ് അനുഭവം നൽകാൻ **പ്ലാറ്റ്‌ഫോം ഡൊമെയ്‌നിയും DNS-ഉം** സജ്ജമാക്കി
- FitSite ബ്രാൻഡിംഗോടെ **വൈറ്റ്-ലേബൽ ചെയ്ത അഡ്മിൻ**
- **കസ്റ്റമൈസ് ചെയ്ത ആശയവിനിമയങ്ങൾ** -- ഇമെയിലുകൾ, ഇൻവോയ്സുകൾ, അറിയിപ്പുകൾ എല്ലാം ബ്രാൻഡഡ്
- ഫിറ്റ്നസ് സ്റ്റുഡിയോ ഉടമകൾക്ക് FitSite വിൽക്കുന്ന ഒരു **മാർക്കറ്റിംഗ് സൈറ്റ്**
- സ്വന്തമായി ഡൊമെയ്ൻ ആഗ്രഹിക്കുന്ന ഉപഭോക്താക്കൾക്കായി **കസ്റ്റം ഡൊമെയ്ൻ ഡോക്യുമെന്റേഷൻ**
