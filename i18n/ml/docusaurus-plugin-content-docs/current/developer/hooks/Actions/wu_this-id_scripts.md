# Action: wu_{$this->id}_scripts {#action-wuthis-idscripts}

സ്ക്രിപ്റ്റുകൾ എൻക്യൂ ചെയ്യാനുള്ള (enqueue) ഹുക്ക് ഇത് പ്രവർത്തിപ്പിക്കുന്നു.

ഈ എലമെന്റിന് അതിന്റെ `register_scripts` മെത്തേഡ് ഹുക്ക് ചെയ്യാൻ ഇത് ഉപയോഗിക്കുന്നു.

### Since {#since}

- 2.0.0

[`inc/ui/class-base-element.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/ui/class-base-element.php#L613) എന്ന ഫയലിലെ 613-ാം വരിയിൽ നിർവചിച്ചിരിക്കുന്നു.
