# Action: wu_mcp_abilities_registered

MCP അബിലിറ്റികൾ ഒരു എൻ്റിറ്റിക്ക് വേണ്ടി രജിസ്റ്റർ ചെയ്ത ശേഷം ഇത് പ്രവർത്തിപ്പിക്കുന്നു.

## പാരാമീറ്ററുകൾ {#parameters}

| Name | Type | വിവരണം |
|------|------|-------------|
| $ability_prefix | `string` | അബിലിറ്റി പ്രിഫിക്സ്. |
| $model_name | `string` | മോഡൽ നാമം. |
| $this | `object` | മാനേജർ ഇൻസ്റ്റൻസ്. |

### ആരംഭിച്ചത് {#since}

- 2.5.0
