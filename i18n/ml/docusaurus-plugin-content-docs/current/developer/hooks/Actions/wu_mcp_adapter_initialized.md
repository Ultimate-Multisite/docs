# Action: wu_mcp_adapter_initialized

MCP adapter initialize ചെയ്തതിന് ശേഷം ഇത് പ്രവർത്തിപ്പിക്കുന്നു (Fires).

മറ്റ് പ്ലഗിനുകൾക്കും തീമുകൾക്കും അവയുടെ സ്വന്തം കഴിവുകൾ (abilities) രജിസ്റ്റർ ചെയ്യാൻ ഇത് അനുവദിക്കുന്നു.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | MCP adapter-ന്റെ ഇൻസ്റ്റൻസ്. |

### Since {#since}

- 2.5.0
### Source {#source}

[`inc/class-mcp-adapter.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-mcp-adapter.php#L98) എന്ന ഫയലിലെ 98-ാം വരിയിൽ നിർവചിച്ചിരിക്കുന്നു.
