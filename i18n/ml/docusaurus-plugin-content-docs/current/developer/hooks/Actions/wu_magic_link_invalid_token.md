# Action: wu_magic_link_invalid_token

അസാധുവായ മാജിക് ലിങ്ക് ടോക്കൺ (invalid magic link token) ലഭിക്കുമ്പോൾ ഇത് പ്രവർത്തിപ്പിക്കുന്നു.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $reason | `string` | ടോക്കൺ അസാധുവായതിനുള്ള കാരണം. |

### Since {#since}

- 2.0.0
