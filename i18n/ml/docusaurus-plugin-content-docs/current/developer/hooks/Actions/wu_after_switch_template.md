# Action: wu_after_switch_template

ഒരു യൂസർ അല്ലെങ്കിൽ സൂപ്പർ അഡ്മിൻ സൈറ്റ് ടെംപ്ലേറ്റ് മാറ്റിയതിന് ശേഷം പ്ലഗിൻ ഡെവലപ്പർമാർക്ക് ഫംഗ്ഷനുകൾ ഹുക്ക് ചെയ്യാൻ ഇത് അനുവദിക്കുന്നു.

## Parameters
