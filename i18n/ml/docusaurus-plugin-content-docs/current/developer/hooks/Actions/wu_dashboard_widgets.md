# Action: wu_dashboard_widgets

പ്ലഗിൻ ഡെവലപ്പർമാർക്ക് നെറ്റ്‌വർക്ക് ഡാഷ്‌ബോർഡ് പാനലിൽ വിഡ്ജറ്റുകൾ ചേർക്കാൻ ഇത് അനുവദിക്കുന്നു.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $tab | `string` | നിലവിൽ സജീവമായ ടാബ് ഏതാണെന്ന് ഇത് സൂചിപ്പിക്കുന്നു. |
| $screen | `\WP_Screen` | സ്ക്രീൻ ഒബ്ജക്റ്റ്. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Ultimate Multisite അഡ്മിൻ പേജ് ഇൻസ്റ്റൻസ്. |
