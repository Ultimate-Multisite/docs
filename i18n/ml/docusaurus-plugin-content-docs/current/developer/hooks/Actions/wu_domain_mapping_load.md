# Action: wu_domain_mapping_load

നമ്മുടെ പ്രധാന Domain Mapping ലോഡ് ചെയ്തതിന് ശേഷം ഇത് പ്രവർത്തിക്കുന്നു (Fired).

ഇതിലേക്ക് ഹുക്ക് ചെയ്ത് (Hook into this) നിങ്ങൾക്ക് ഏത് അധിക സവിശേഷതകളും (add-on functionality) ചേർക്കാൻ സാധിക്കും.
