# Action: wu_page_{$this->id}_before_render {#action-wupagethis-idbeforerender}

പേജ് പ്രദർശിപ്പിക്കുന്നതിന് (print) മുമ്പ് അധിക ഉള്ളടക്കം ചേർക്കാൻ പ്ലഗിൻ ഡെവലപ്പർമാർക്ക് ഇത് അനുവദിക്കുന്നു.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $page_id | `string` | ഈ പേജിന്റെ ഐഡി. |
| $page | `object` | പേജ് ഒബ്ജക്റ്റ്. |
