# Action: wu_site_created {#action-wusitecreated}

ഒരു സൈറ്റ് ആദ്യമായി സൃഷ്ടിക്കപ്പെടുമ്പോൾ ഇത് പ്രവർത്തിപ്പിക്കുന്നു.

ഒരു ടെംപ്ലേറ്റിൽ നിന്ന് ഡ്യൂപ്ലിക്കേറ്റ് ചെയ്താൽ ഇത് പ്രവർത്തിക്കില്ല.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $data | `array` | സ്റ്റോർ ചെയ്യേണ്ട ഒബ്ജക്റ്റ് ഡാറ്റ. |
| $site | `\Site` | ഒബ്ജക്റ്റ് ഇൻസ്റ്റൻസ്. |
