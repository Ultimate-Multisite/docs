# Action: wu_before_light_ajax

ചില സന്ദർഭങ്ങളിൽ, ആക്ഷനുകൾ കൈകാര്യം ചെയ്യാൻ കൂടുതൽ കാര്യങ്ങൾ (dependencies) ലോഡ് ചെയ്യേണ്ടി വരും.

ആവശ്യത്തിന് കൂടുതൽ ഡിപൻഡൻസികൾ ലോഡ് ചെയ്യാൻ ഈ ആക്ഷൻ ഉപയോഗിക്കാവുന്നതാണ്.

- 2.0.0
