# Filter: wu_wp_cli_enabled_sub_commands

ഈ എൻറ്റിറ്റിക്കായി ഏത് സബ്-കമാൻഡുകളാണ് പ്രവർത്തനക്ഷമമാക്കിയിരിക്കുന്നതെന്ന് ഇത് ഫിൽട്ടർ ചെയ്യുന്നു.

## പാരാമീറ്ററുകൾ {#parameters}

| Name | Type | വിവരണം |
|------|------|-------------|
| $sub_commands | `array` | ഡിഫോൾട്ട് സബ്-കമാൻഡുകൾ. |
| $command_base | `string` | റൂട്ടിന് തൊട്ടുപിന്നാലെ കമാൻഡിൽ ഉപയോഗിക്കുന്ന ബേസ്. |
| $this | `\Base_Manager` | ഒബ്ജക്റ്റ് ഇൻസ്റ്റൻസ്. |
