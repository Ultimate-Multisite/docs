# Filter: wu_get_post_types {#filter-wugetposttypes}

വിവിധ പോസ്റ്റ് ടൈപ്പുകൾ (post types) ഏതെല്ലാം കാണിക്കണം എന്ന് ഡെവലപ്പർമാർക്ക് തിരഞ്ഞെടുക്കാൻ ഇത് അനുവദിക്കുന്നു.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $post_types | `array` | പോസ്റ്റ് ടൈപ്പുകളുടെ ലിസ്റ്റ്. |

### Since {#since}

- 2.0.0
### Source {#source}

[`inc/ui/class-limits-element.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/ui/class-limits-element.php#L267) എന്ന ഫയലിലെ 267-ാം വരിയിൽ നിർവചിച്ചിരിക്കുന്നു.

## Returns {#returns}
പുതിയ ലിസ്റ്റ്.
