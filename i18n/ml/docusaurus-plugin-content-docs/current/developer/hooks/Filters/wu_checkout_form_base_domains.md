# ഫിൽട്ടർ: wu_checkout_form_base_domains

പങ്കിട്ട ചെക്കൗട്ട്-ഫോം അടിസ്ഥാന ഡൊമെയിനുകൾ മാപ്പ് ചെയ്ത ഡൊമെയിൻ രേഖകളാകാതിരിക്കാൻ ഫിൽട്ടർ ചെയ്യുന്നു.

ഒരു ഇന്റഗ്രേഷൻ ചെക്കൗട്ട് ഫോം **Site URL** ഫീൽഡുകൾക്കായി അധിക അടിസ്ഥാന ഡൊമെയിനുകൾ നൽകുമ്പോൾ ഈ ഫിൽട്ടർ ഉപയോഗിക്കുക. ഈ ഫിൽട്ടർ തിരികെ നൽകുന്ന ഡൊമെയിനുകൾ ഓരോ സൈറ്റിനുമുള്ള കസ്റ്റം ഡൊമെയിനുകളായല്ല, പങ്കിട്ട രജിസ്ട്രേഷൻ ഹോസ്റ്റുകളായി കണക്കാക്കപ്പെടും.

## പരാമീറ്ററുകൾ {#parameters}

| പേര് | തരം | വിവരണം |
|------|------|-------------|
| $domains | `array` | ചെക്കൗട്ട് ഫോം ക്രമീകരണത്തിൽ നിന്ന് ശേഖരിച്ച പങ്കിട്ട അടിസ്ഥാന ഡൊമെയിനുകൾ. |

### മുതൽ {#since}

- 2.13.0

### ഉറവിടം {#source}

`inc/functions/domain.php`-ൽ നിർവചിച്ചിരിക്കുന്നു.


## റിട്ടേണുകൾ {#returns}

സാധാരണവൽക്കരിച്ച ചെക്കൗട്ട്-ഫോം അടിസ്ഥാന ഡൊമെയിനുകളുടെ അറേ.
