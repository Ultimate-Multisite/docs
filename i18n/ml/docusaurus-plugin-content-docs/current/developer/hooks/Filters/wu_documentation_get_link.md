# Filter: wu_documentation_get_link

പ്ലഗിൻ ഡെവലപ്പർമാർക്ക് ലിങ്കുകൾ ഫിൽട്ടർ ചെയ്യാൻ ഇത് അനുവദിക്കുന്നു.

ഇത് എങ്ങനെ ഉപകാരപ്രദമാകുമെന്ന് എനിക്ക് ഉറപ്പില്ല, പക്ഷേ ഇത് ഉണ്ടാകുന്നത് കുഴപ്പമില്ല.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $link | `string` | രജിസ്റ്റർ ചെയ്ത ലിങ്ക് |
| $slug | `string` | ലിങ്ക് എടുക്കാൻ ഉപയോഗിക്കുന്ന സ്ലഗ് |
| $default_link | `string` | രജിസ്റ്റർ ചെയ്ത ഡിഫോൾട്ട് ലിങ്ക് |
