# ഫിൽട്ടർ: wu_sso_url {#filter-wussourl}

ക്രോസ്-ഡൊമെയ്ൻ ഉപഭോക്തൃ പ്രവർത്തനങ്ങൾക്ക് മടക്കി നൽകുന്നതിന് മുമ്പ് സൃഷ്ടിച്ച SSO URL-കൾ ഫിൽട്ടർ ചെയ്യുന്നു.

ഒരു ഇന്റഗ്രേഷനു സ്വതന്ത്ര-ടെനന്റ് SSO ലിങ്കിലേക്ക് വിശ്വസനീയമായ സന്ദർഭം ചേർക്കേണ്ടതുണ്ടെങ്കിൽ, അല്ലെങ്കിൽ Ultimate Multisite-ന്റെ token സാധൂകരണം നിലനിർത്തിക്കൊണ്ട് ബ്രോക്കർ URL മാറ്റിസ്ഥാപിക്കേണ്ടതുണ്ടെങ്കിൽ, ഈ ഫിൽട്ടർ ഉപയോഗിക്കുക.

## പാരാമീറ്ററുകൾ {#parameters}

| പേര് | തരം | വിവരണം |
|------|------|-------------|
| $sso_url | `string` | സൃഷ്ടിച്ച SSO URL. |
| $user | `WP_User` | SSO സന്ദർശനത്തിലൂടെ പ്രാമാണീകരിക്കപ്പെടുന്ന ഉപയോക്താവ്. |
| $site_id | `int` | സന്ദർശനത്തിനുള്ള ലക്ഷ്യ സൈറ്റ് ID. |
| $redirect_to | `string` | വിജയകരമായ SSO സാധൂകരണത്തിന് ശേഷമുള്ള ലക്ഷ്യ URL. |

### മുതൽ {#since}

- 2.13.0

### ഉറവിടം {#source}

`inc/sso/class-sso.php`-ൽ നിർവചിച്ചിരിക്കുന്നു.


## മടക്കുന്നത് {#returns}

ഫിൽട്ടർ ചെയ്ത SSO URL.
