# ഫിൽട്ടർ: wu_should_redirect_to_primary_domain {#filter-wushouldredirecttoprimarydomain}

| $should_redirect | `bool` | റീഡയറക്റ്റ് ചെയ്യണോ വേണ്ടയോ എന്ന് ഇത് സൂചിപ്പിക്കുന്നു. |
