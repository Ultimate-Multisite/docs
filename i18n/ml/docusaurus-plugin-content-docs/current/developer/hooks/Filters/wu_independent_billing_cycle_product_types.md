# Filter: wu_independent_billing_cycle_product_types

സ്വതന്ത്ര ബില്ലിംഗ് സൈക്കിളുകൾ ഉള്ള ഉൽപ്പന്ന തരങ്ങളെ (product types) ഫിൽട്ടർ ചെയ്യുന്നു.

സ്വതന്ത്ര ബില്ലിംഗ് സൈക്കിളുകളുള്ള ഉൽപ്പന്നങ്ങൾ കാർട്ടിലെ പ്ലാനിന്റെ ബില്ലിംഗ് കാലയളവുമായി (billing period) പൊരുത്തപ്പെടാൻ നിർബന്ധിക്കപ്പെടുന്നില്ല. പ്ലാൻ മാസത്തേക്കാണോ വർഷത്തേക്കാണോ എന്നത് പരിഗണിക്കാതെ, എല്ലാ വർഷവും ബിൽ ചെയ്യേണ്ട ഡൊമെയ്ൻ രജിസ്ട്രേഷനുകൾ പോലുള്ള ഉൽപ്പന്നങ്ങൾക്ക് ഇത് വളരെ ഉപകാരപ്രദമാണ്.

## Parameters {#parameters}

| Name | Type | Description |

| $types | `array` | സ്വതന്ത്ര ബില്ലിംഗ് സൈക്കിളുകളുള്ള ഉൽപ്പന്ന തരങ്ങളുടെ (product types) ഒരു അറേ. |
