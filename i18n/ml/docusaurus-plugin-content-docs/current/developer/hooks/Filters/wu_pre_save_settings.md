# Filter: wu_pre_save_settings

Ultimate Multisite വഴി സേവ് ചെയ്യുന്നതിന് മുൻപ് സെറ്റിംഗുകൾ ഫിൽട്ടർ ചെയ്യാൻ ഡെവലപ്പർമാർക്ക് ഇത് അനുവദിക്കുന്നു.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $settings | `array` | സേവ് ചെയ്യേണ്ട സെറ്റിംഗുകൾ. |
| $settings_to_save | `array` | ചേർക്കേണ്ട പുതിയ സെറ്റിംഗുകൾ. |
| $saved_settings | `array` | നിലവിൽ സേവ് ചെയ്തിട്ടുള്ള സെറ്റിംഗുകൾ. |
