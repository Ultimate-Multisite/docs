# Filter: wu_plan_product_types

പ്ലാനുകളായി കണക്കാക്കേണ്ട ഉൽപ്പന്ന തരങ്ങൾ (product types) ഫിൽട്ടർ ചെയ്യുന്നു.

ഈ ഫിൽട്ടർ അഡോണുകൾക്ക് (addons) സാധൂകരണത്തിലും വേർതിരിക്കലിലും (validation and segregation) പ്ലാനുകളായി തിരിച്ചറിയേണ്ട അധിക ഉൽപ്പന്ന തരങ്ങൾ രജിസ്റ്റർ ചെയ്യാൻ അനുവദിക്കുന്നു.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | പ്ലാനുകളായി കണക്കാക്കേണ്ട ഉൽപ്പന്ന തരങ്ങളുടെ അറേ. |
