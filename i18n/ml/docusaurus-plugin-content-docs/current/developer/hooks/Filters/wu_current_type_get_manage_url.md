# Filter: wu_current_\{$type\}_get_manage_url {#filter-wucurrenttypegetmanageurl}

ഈ ഫിൽട്ടർ ഉപയോഗിച്ച് ഡെവലപ്പർമാർക്ക് മാനേജ് സൈറ്റ് URL-ൻ്റെ പാരാമീറ്ററുകളിൽ മാറ്റങ്ങൾ വരുത്താൻ കഴിയും.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $manage_site_url | `string` | മാനേജ് സൈറ്റ് URL. |
| $id | `int` | സൈറ്റ് ID. |
| $site_hash | `string` | സൈറ്റ് ഹാഷ്. |

### Since {#since}

- 2.0.9
### Source {#source}

[`inc/class-current.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-current.php#L214) എന്ന ഫയലിലെ 214-ാം വരിയിൽ നിർവചിച്ചിരിക്കുന്നു.

## Returns {#returns}
മാറ്റം വരുത്തിയ മാനേജ് URL.
