# Filter: wu_password_require_special

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | പ്രത്യേക അക്ഷരങ്ങൾ ആവശ്യമുണ്ടോ. നിയമങ്ങൾ നടപ്പിലാക്കുമ്പോൾ ഡിഫോൾട്ട് ആയി true ആയിരിക്കും. |
| $defender_active | `bool` | Defender Pro Strong Password സജീവമാണോ എന്ന്. |
