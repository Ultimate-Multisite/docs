# Filter: wu_password_require_lowercase

പാസ്‌വേഡുകളിൽ ചെറിയ അക്ഷരങ്ങൾ (lowercase letters) നിർബന്ധമാക്കണോ എന്ന് ഇത് ഫിൽട്ടർ ചെയ്യുന്നു.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | ചെറിയ അക്ഷരങ്ങൾ നിർബന്ധമാക്കണോ. നിയമങ്ങൾ നടപ്പിലാക്കുമ്പോൾ ഡിഫോൾട്ട് ആയി ഇത് `true` ആയിരിക്കും. |
| $defender_active | `bool` | Defender Pro Strong Password സജീവമാണോ (active). |
