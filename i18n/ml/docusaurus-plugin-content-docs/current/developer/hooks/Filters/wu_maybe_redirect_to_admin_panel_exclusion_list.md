# Filter: wu_maybe_redirect_to_admin_panel_exclusion_list

ഓട്ടോ-റിഡയക്റ്റ് തടയുന്ന പാരാമീറ്ററുകൾ അടങ്ങിയ ഒരു ഒഴിവാക്കൽ ലിസ്റ്റ് (exclusion list) സൃഷ്ടിക്കുന്നു.

ഇത് ആവശ്യമായി വരുന്നത് എന്തുകൊണ്ടെന്നാൽ, ഇത് ഇല്ലെങ്കിൽ പേജ് ബിൽഡർക്ക് ലോഗിൻ പേജ് നിർവചിച്ച ശേഷം എഡിറ്റ് ചെയ്യാൻ കഴിയില്ല.
