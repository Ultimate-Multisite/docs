# Filter: wu_stripe_existing_product_id

പരിശോധിക്കാനുള്ള ഉൽപ്പന്നത്തിന്റെ ID ഫിൽട്ടർ ചെയ്യുന്നു. ഇത് നിലവിലുണ്ടെങ്കിൽ, പുതിയ സബ്‌സ്‌ക്രിപ്ഷൻ ഈ ഉൽപ്പന്നം ഉപയോഗിക്കും.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $product_id | `string` | പരിശോധിക്കാനുള്ള Stripe ഉൽപ്പന്നത്തിന്റെ ID. |
| $name | `string` | ഉൽപ്പന്നത്തിന്റെ പേര്. |
