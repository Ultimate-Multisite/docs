# Filter: wp_ultimo_redirect_url_after_signup

റീഡയറക്റ്റ് URL സജ്ജമാക്കുക.

ഇതൊരു പഴയ ഫിൽട്ടറാണ് (legacy filter). $site_id പോലുള്ള ചില പാരാമീറ്ററുകൾ ഇതിലൂടെ ലഭ്യമല്ല.

### മുതൽ {#since}

- 1.1.3: ഡെവലപ്പർമാർക്ക് റീഡയറക്റ്റ് URL ഫിൽട്ടർ ചെയ്യാൻ ഇത് അനുവദിക്കുന്നു.
### സ്രോതസ്സ് {#source}

[`inc/checkout/class-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout.php#L2450) എന്ന ഫയലിലെ 2450-ാം വരിയിൽ നിർവചിച്ചിരിക്കുന്നു.
