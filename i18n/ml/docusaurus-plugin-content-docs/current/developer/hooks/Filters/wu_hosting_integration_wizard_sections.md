# Filter: wu_hosting_integration_wizard_sections {#filter-wuhostingintegrationwizardsections}

ഹോസ്റ്റിംഗ് ഇന്റഗ്രേഷൻ സജ്ജീകരിക്കുന്നതിനുള്ള വിസാർഡ് സെക്ഷനുകൾ ഫിൽട്ടർ ചെയ്യുന്നു.

ആഡ്-ഓണുകൾക്ക് വിസാർഡ് സെക്ഷനുകൾ ചേർക്കാനും, നീക്കം ചെയ്യാനും, അല്ലെങ്കിൽ മാറ്റങ്ങൾ വരുത്താനും ഇത് അനുവദിക്കുന്നു.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $sections | `array` | വിസാർഡ് സെക്ഷനുകൾ. |
| $integration | `\WP_Ultimo\Integrations\Host_Providers\Base_Host_Provider\|\WP_Ultimo\Integrations\Integration` | കോൺഫിഗർ ചെയ്യുന്ന ഇന്റഗ്രേഷൻ. |
| $page | `\Hosting_Integration_Wizard_Admin_Page` | വിസാർഡ് പേജ് ഇൻസ്റ്റൻസ്. |
