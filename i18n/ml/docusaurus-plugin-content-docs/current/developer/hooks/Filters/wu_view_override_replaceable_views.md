# Filter: wu_view_override_replaceable_views

ഡെവലപ്പർമാർക്ക് മാറ്റിസ്ഥാപിക്കാവുന്ന ലിസ്റ്റിലേക്ക് അധിക ഫോൾഡറുകൾ ചേർക്കാൻ ഇത് അനുവദിക്കുന്നു.

ശ്രദ്ധിക്കുക: അധിക ഫോൾഡറുകൾ അനുവദിക്കുന്നത്, Ultimate Multisite പതിപ്പുകൾക്ക് പകരം കാലഹരണപ്പെട്ട കോപ്പികൾ ലോഡ് ചെയ്യാൻ കാരണമായേക്കാം.
