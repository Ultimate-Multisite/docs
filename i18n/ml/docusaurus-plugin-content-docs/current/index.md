# Ultimate Multisite ഡോക്യുമെന്റേഷൻ

Ultimate Multisite ഡോക്യുമെന്റേഷനിലേക്ക് സ്വാഗതം. Ultimate Multisite, WordPress Multisite-നെ ഒരു സമ്പൂർണ്ണ Website as a Service (WaaS) പ്ലാറ്റ്‌ഫോമിലേക്ക് മാറ്റുന്നു.

## ഡോക്യുമെന്റേഷൻ വിഭാഗങ്ങൾ {#documentation-sections}

### [ഉപയോക്തൃ ഗൈഡ്](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

നിങ്ങളുടെ WaaS പ്ലാറ്റ്‌ഫോം സജ്ജീകരിക്കാനും, കോൺഫിഗർ ചെയ്യാനും, കൈകാര്യം ചെയ്യാനും വേണ്ട എല്ലാ കാര്യങ്ങളും ഇതിൽ ഉൾപ്പെടുന്നു. ഇൻസ്റ്റാളേഷൻ, പേയ്‌മെന്റ് ഗേറ്റ്‌വേകൾ, ഡൊമെയ്ൻ മാപ്പിംഗ്, ക്ലയിന്റ് മാനേജ്‌മെന്റ് എന്നിവ ഇതിൽ ഉൾക്കൊള്ളുന്നു.

### [ഡെവലപ്പർ ഡോക്സ്](/developer/hooks) {#developer-documentation}

Ultimate Multisite-ൽ പ്രവർത്തിക്കുന്ന ഡെവലപ്പർമാർക്കുള്ള സാങ്കേതിക റഫറൻസ്. ഇതിൽ hooks reference, REST API ഡോക്യുമെന്റേഷൻ, ആഡ്‌ഓൺ ഡെവലപ്‌മെന്റ് ഗൈഡുകൾ എന്നിവ ഉൾപ്പെടുന്നു.

### [ആഡോണുകൾ](/addons) {#addons}

Ultimate Multisite-നെ WooCommerce ഇന്റഗ്രേഷൻ, അനലിറ്റിക്സ്, ഡൊമെയ്ൻ വിൽപന തുടങ്ങിയ അധിക ഫീച്ചറുകളോടെ വികസിപ്പിക്കുന്ന പെയ്ഡ് ആഡ്‌ഓൺ പ്ലഗിനുകൾക്കുള്ള ഡോക്യുമെന്റേഷൻ.
