# Action: wp_ultimo_woocommerce_load

എല്ലാ ആഡ്-ഓൺ ഡിപൻഡൻസികളും ലോഡ് ചെയ്തതിന് ശേഷം ഇത് പ്രവർത്തിപ്പിക്കുന്നു (Triggers).

പ്ലഗിൻ ഡെവലപ്പർമാർക്ക് പുതിയ ഫംഗ്ഷണാലിറ്റി ചേർക്കാൻ ഇത് അനുവദിക്കുന്നു.

- 2.0.0
