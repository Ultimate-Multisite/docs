# Plugin & Theme Manager

Multisite Ultimate: Plugin and Theme Manager

ഇൻസ്റ്റാൾ ചെയ്ത plugins, themes എന്നിവയുടെ മെറ്റാ വിവരങ്ങൾ (title, author, thumbnail, description) എഡിറ്റ് ചെയ്യാനോ മറച്ചുവെക്കാനോ ചെയ്യാം. നിങ്ങളുടെ ഉപയോക്താക്കൾക്ക് അവയെ ഫിൽട്ടർ ചെയ്യാൻ കഴിയുന്നതിനായി plugins-നും themes-നും വേണ്ടി വിഭാഗങ്ങൾ (categories) സൃഷ്ടിക്കാനും, ഉപയോക്താക്കളുടെ പാനലിൽ മനോഹരമായ കസ്റ്റം Plugins പേജ് ചേർക്കാനും ഇത് സഹായിക്കുന്നു.

ഈ plugin ഉപയോഗിക്കാൻ, നിങ്ങളുടെ network admin-ലെ Plugins അല്ലെങ്കിൽ Themes പേജിലേക്ക് പോവുക, എന്നിട്ട് "Edit Plugin/Theme info" എന്നതിൽ ക്ലിക്ക് ചെയ്യുക.

## ഇൻസ്റ്റാളേഷൻ

1. 'ultimate-multisite-plugin-and-theme-manager' എന്നത് '/wp-content/plugins/' ഡയറക്‌ടറിയിലേക്ക് അപ്‌ലോഡ് ചെയ്യുക
2. നിങ്ങളുടെ WordPress Network Admin-ലെ 'Plugins' മെനു വഴി plugin ആക്ടിവേറ്റ് ചെയ്യുക

## ഉപയോഗ രീതി (Usage Guide)

Ultimate Multisite: Plugin and Theme Manager എന്നത് ഒരു ശക്തമായ **premium** Ultimate Multisite add-on ആണ്. നിങ്ങളുടെ network-ലെ plugin, theme ഓപ്ഷനുകൾ നിങ്ങളുടെ network ഉപയോക്താക്കൾ കാണുകയും ഇടപെടുകയും ചെയ്യുന്ന രീതിയിൽ നിങ്ങൾക്ക് പൂർണ്ണ നിയന്ത്രണം നൽകുന്നു.

ഈ add-on മൂന്ന് ആവശ്യങ്ങൾ നിറവേറ്റുന്നു:

  1. plugins-ന്റെയും themes-ന്റെയും മെറ്റാ ഡാറ്റ പൂർണ്ണമായും മാറ്റാൻ ഇത് നിങ്ങളെ അനുവദിക്കുന്നു. ഇതിൽ പേര്, വിവരണം (description), എഴുത്തുകാരൻ്റെ പേര് (author name), തംബ്നെയിലുകൾ (thumbnails) എന്നിവ ഉൾപ്പെടുന്നു!

  2. നിങ്ങളുടെ ഉപയോക്താക്കൾക്ക് എളുപ്പത്തിൽ ഇവ വേർതിരിക്കാൻ കഴിയുന്നതിനായി plugins-നും themes-നും വേണ്ടി വിഭാഗങ്ങൾ (categories) നൽകാൻ ഇത് നിങ്ങളെ അനുവദിക്കുന്നു.

  3. പഴയതും വിരസവുമായ plugins പേജ്, വിഭാഗങ്ങൾ (categories) അനുസരിച്ച് സോർട്ട് ചെയ്യാവുന്ന ഒരു പുതിയ themes-പോലുള്ള പേജുമായി മാറ്റിസ്ഥാപിക്കാനുള്ള ഓപ്ഷൻ ഇത് നൽകുന്നു.

ഇത് ഡൗൺലോഡ് ചെയ്ത് ഇൻസ്റ്റാൾ ചെയ്ത ശേഷം, Ultimate Multisite Settings പേജിൽ ഒരു പുതിയ ടാബ് - _Add-on Settings_ - കാണാം.

_Ultimate Multisite: Plugin and Theme Manager ചേർക്കുന്ന global ഓപ്ഷനുകൾ ഇവിടെ നിങ്ങൾക്ക് കണ്ടെത്താനാകും._

നിങ്ങളുടെ network-ലെ _Plugins_ പേജിൽ, പേര്, എഴുത്തുകാരൻ, plugin-ൻ്റെ തംബ്നെയിൽ എന്നിവ പോലുള്ള എല്ലാ plugins-ൻ്റെയും മെറ്റാ വിവരങ്ങൾ നിങ്ങൾക്ക് ഇപ്പോൾ എഡിറ്റ് ചെയ്യാൻ കഴിയും.

_ഒരു പ്രത്യേക plugin വിവരങ്ങൾ കാണിക്കണോ വേണ്ടയോ എന്നും നിങ്ങൾക്ക് തിരഞ്ഞെടുക്കാൻ കഴിയും._

Plugin and Theme Manager രണ്ട് ഡിസ്‌പ്ലേ തരങ്ങൾ നൽകുന്നു: Plugin Style, Theme Style. അതിനാൽ, നിങ്ങൾക്ക് ആവശ്യമായ മാറ്റങ്ങൾ വരുത്തിയ ശേഷം, നിങ്ങളുടെ ഉപഭോക്താക്കൾ അവരുടെ _Plugins_ പേജ് കാണുന്നത് ഇങ്ങനെയായിരിക്കും:
