# ഫിൽട്ടർ: wp_ultimo_skip_network_active_check

ഡെവലപ്പർമാർക്ക് ഈ പരിശോധന ഒഴിവാക്കാൻ (short-circuit) ഇത് അനുവദിക്കുന്നു.

composer-based ആയതും Bedrock പോലുള്ള മറ്റ് കസ്റ്റം സെറ്റപ്പുകൾ ഉപയോഗിക്കുമ്പോൾ ഇത് വളരെ ഉപകാരപ്രദമാണ്. കാരണം, അവിടെ mu-plugins ആയി പ്ലഗിനുകൾ ഉപയോഗിക്കുന്നത് സാധാരണമാണ്.

### ലഭ്യമായത്

- 2.0.0
### സ്രോതസ്സ്

[`inc/class-requirements.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-plugin-and-theme-manager/blob/main/inc/class-requirements.php#L227) എന്ന ഫയലിലെ 227-ാം വരിയിൽ നിർവചിച്ചിരിക്കുന്നു.
