# Loco Translate

Ultimate Multisite: Loco Translate Integration അഡോൺ നിങ്ങളുടെ WordPress മൾട്ടിസൈറ്റ് നെറ്റ്‌വർക്കിലേക്ക് സൈറ്റ്-നിർദ്ദിഷ്ട വിവർത്തന ശേഷി കൊണ്ടുവരുന്നു. നിങ്ങളുടെ നെറ്റ്‌വർക്കിലെ വ്യത്യസ്ത സൈറ്റുകൾക്ക് ഒരേ പ്ലഗിൻ അല്ലെങ്കിൽ തീമിന്റെ വ്യത്യസ്ത വിവർത്തനങ്ങൾ ആവശ്യമായി വന്നേക്കാം എങ്കിൽ, ഈ അഡോൺ ഏറ്റവും മികച്ച പരിഹാരമാണ്.

**ഈ അഡോൺ എന്തുചെയ്യുന്നു**

ഈ അഡോൺ Ultimate Multisite-നെ ജനപ്രിയമായ Loco Translate പ്ലഗിനുമായി തടസ്സമില്ലാതെ സംയോജിപ്പിക്കുന്നു. ഇത് നിങ്ങളുടെ നെറ്റ്‌വർക്കിലെ ഓരോ സബ്സൈറ്റിനും അതിൻ്റേതായ കസ്റ്റം വിവർത്തനങ്ങൾ നിലനിർത്താൻ സഹായിക്കുന്നു. നിങ്ങൾ ഒരു മൾട്ടി-റിജിയണൽ പ്ലാറ്റ്‌ഫോം നടത്തുകയാണെങ്കിലും, വ്യത്യസ്ത ഭാഷാ സമൂഹങ്ങൾക്ക് സേവനം നൽകുകയാണെങ്കിലും, അല്ലെങ്കിൽ നിങ്ങളുടെ നെറ്റ്‌വർക്കിലുടനീളം വിവർത്തന വഴക്കം ആവശ്യമാണെങ്കിലും, ഈ അഡോൺ അത് വളരെ എളുപ്പമാക്കുന്നു.

**ആർക്കൊക്കെ മികച്ചത്:**

* വ്യത്യസ്ത ലൊക്കലൈസേഷൻ ആവശ്യങ്ങളുള്ള മൾട്ടി-റിജിയണൽ ബിസിനസ്സുകൾ
* വൈവിധ്യമാർന്ന സമൂഹങ്ങൾക്ക് സേവനം നൽകുന്ന വിദ്യാഭ്യാസ സ്ഥാപനങ്ങൾ
* മൾട്ടി-ലിംഗ്വൽ സപ്പോർട്ട് നൽകുന്ന SaaS പ്ലാറ്റ്‌ഫോമുകൾ
* ഒന്നിലധികം ക്ലയിന്റ് സൈറ്റുകൾ കൈകാര്യം ചെയ്യുന്ന വെബ് ഏജൻസികൾ
* വിവർത്തന വഴക്കം ആവശ്യമുള്ള ഏത് മൾട്ടിസൈറ്റ് നെറ്റ്‌വർക്കിനും

## ഇൻസ്റ്റാളേഷൻ

1. Ultimate Multisite ഇൻസ്റ്റാൾ ചെയ്തിട്ടുണ്ടെന്നും നെറ്റ്‌വർക്ക് ആക്ടിവേറ്റ് ചെയ്തിട്ടുണ്ടെന്നും ഉറപ്പാക്കുക
2. Loco Translate പ്ലഗിൻ ഇൻസ്റ്റാൾ ചെയ്ത് ആക്ടിവേറ്റ് ചെയ്യുക
3. ഈ അഡോൺ നിങ്ങളുടെ പ്ലഗിൻ ഡയറക്‌ടറിയിലേക്ക് അപ്‌ലോഡ് ചെയ്യുക
4. Loco Translate Integration അഡോൺ നെറ്റ്‌വർക്ക് ആക്ടിവേറ്റ് ചെയ്യുക
5. ഏത് സബ്സൈറ്റിലാണെങ്കിലും Loco Translate വഴി സൈറ്റ്-നിർദ്ദിഷ്ട വിവർത്തനങ്ങൾ ഉണ്ടാക്കാൻ തുടങ്ങുക

അത്രയേ ഉള്ളൂ! കൂടുതൽ കോൺഫിഗറേഷൻ ഒന്നും ചെയ്യാതെ തന്നെ ഇന്റഗ്രേഷൻ ഓട്ടോമാറ്റിക്കായി പ്രവർത്തിക്കും.

ഇല്ല! ഇന്റഗ്രേഷൻ ഓട്ടോമാറ്റിക്കാണ്. അഡോണും Loco Translate-ഉം രണ്ടും ആക്റ്റീവ് ആയ ഉടൻ തന്നെ സബ്സൈറ്റുകൾക്ക് അവയുടെ കസ്റ്റം വിവർത്തനങ്ങൾ ഉണ്ടാക്കാൻ തുടങ്ങാം.

## ഒരു സബ്സൈറ്റ് കസ്റ്റം വിവർത്തനങ്ങൾ ഉണ്ടാക്കുന്നില്ലെങ്കിൽ എന്ത് സംഭവിക്കും?

## സബ്സൈറ്റുകൾക്ക് നെറ്റ്‌വർക്ക്-വ്യാപക വിവർത്തനങ്ങൾ ആക്‌സസ് ചെയ്യാൻ കഴിയുമോ?
