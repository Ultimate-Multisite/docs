# Loco Translate മാറ്റങ്ങൾ രേഖപ്പെടുത്തൽ (Changelog) {#loco-translate-changelog}

### 1.0.1 {#101}
* ഏറ്റവും പുതിയ Loco Translate പതിപ്പുമായി മെച്ചപ്പെടുത്തിയ അനുയോജ്യത (Improved compatibility)
* മികച്ച സുരക്ഷയ്ക്കായി ആക്‌സസ് നിയന്ത്രണം മെച്ചപ്പെടുത്തി
* പ്രകടന മെച്ചപ്പെടുത്തലുകൾ (Performance optimizations)

### 1.0.0 {#100}
* ആദ്യ റിലീസ് (Initial release)
* സൈറ്റ്-സ്പെസിഫിക് ട്രാൻസ്‌ലേഷൻ ഡയറക്‌ടറികൾ
* ട്രാൻസ്‌ലേഷൻ പ്രയോറിറ്റി സിസ്റ്റം
* ആക്‌സസ് നിയന്ത്രണം നടപ്പിലാക്കി
* Loco Translate-മായി തടസ്സമില്ലാത്ത UI സംയോജനം (Seamless UI integration)
