# Content Sync-ൻ്റെ മാറ്റങ്ങൾ (Changelog) {#content-sync-changelog}

### Version 1.1.0 - Released on 2025-12-31 {#version-110---released-on-2025-12-31}
* പുതിയത്: എല്ലാ പോസ്റ്റ് ടൈപ്പുകളും (posts, pages, custom post types) sync ചെയ്യാൻ പിന്തുണ നൽകുന്നു.
* പുതിയത്: കണ്ടന്റ് തിരഞ്ഞെടുക്കുന്ന ഘട്ടത്തിൽ പോസ്റ്റ് ടൈപ്പ് ഫിൽട്ടറിംഗ്.
* പുതിയത്: സോഴ്സ് സൈറ്റ് തിരഞ്ഞെടുക്കുന്ന ഘട്ടം - പ്രധാന സൈറ്റിൽ നിന്ന് മാത്രമല്ല, ഏത് സൈറ്റിൽ നിന്നും sync ചെയ്യാൻ കഴിയും.
* പുതിയത്: ടെംപ്ലേറ്റ് സൈറ്റുകൾക്കായി ചൈൽഡ് സൈറ്റ് പ്രീ-സെലക്ഷൻ.
* പുതിയത്: എല്ലാ പോസ്റ്റ് ടൈപ്പുകൾക്കുമുള്ള post_content-ൽ നിന്ന് മീഡിയ എക്സ്ട്രാക്ഷൻ.
* പുതിയത്: ടാക്സോണമി sync ചെയ്യാനുള്ള പിന്തുണ (categories, tags, custom taxonomies).
* മാറ്റം: "Elementor Template Sync" എന്നതിൽ നിന്ന് "Content Sync" എന്ന് പേര് മാറ്റി.
* മാറ്റം: "Select Templates" എന്ന ഘട്ടം "Select Data" എന്ന് പേര് മാറ്റി.
* മെച്ചപ്പെടുത്തിയത്: Elementor-ൻ്റെ പ്രത്യേക കോഡ് ഇപ്പോൾ elementor_library പോസ്റ്റ് ടൈപ്പിന് വേണ്ടി മാത്രമേ പ്രവർത്തിക്കൂ.

### Version 1.0.0 - Released on 2025-12-19 {#version-100---released-on-2025-12-19}
* ആദ്യ റിലീസ്.
* ടെംപ്ലേറ്റ്, സൈറ്റ് തിരഞ്ഞെടുക്കുന്നതിനുള്ള മൾട്ടി-സ്റ്റെപ്പ് വിസാർഡ്.
* റിയൽ-ടൈം പ്രോഗ്രസ് ഫീഡ്ബാക്കോടെ ബാച്ച് പ്രോസസ്സിംഗ്.
* ഓട്ടോമാറ്റിക് ID റീമാപ്പിംഗോടുകൂടിയ പൂർണ്ണ മീഡിയ sync.
* Ultimate Multisite-മായി സെറ്റിംഗ്സ് സംയോജിപ്പിക്കൽ (integration).
* നെറ്റ്‌വർക്ക് സെറ്റിംഗ്‌സ് വഴി സൈറ്റ് ഒഴിവാക്കാനുള്ള പിന്തുണ.
* sync ചെയ്ത ശേഷം Elementor കാഷെ ക്ലിയറിംഗ്.
* ടെംപ്ലേറ്റ് ടാക്സോണമി സംരക്ഷണം.
