# Filter: ultimate_multisite_content_sync_url

പ്ലഗിൻ റൂട്ടിലേക്കുള്ള റിലേറ്റീവ് പാത്ത് കൈകാര്യം ചെയ്യാൻ ഡെവലപ്പർമാർക്ക് ഇത് അനുവദിക്കുന്നു.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $dir | `string` | പ്ലഗിൻ റൂട്ടുമായി ബന്ധപ്പെട്ട, നിങ്ങൾക്ക് ആക്‌സസ് ചെയ്യേണ്ട പാത്ത്. |

### Since {#since}

- 1.0.0
### Source {#source}

[`inc/class-helper.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-content-sync/blob/main/inc/class-helper.php#L52) എന്ന ഫയലിലെ 52-ാം വരിയിൽ നിർവചിച്ചിരിക്കുന്നു.

## Returns {#returns}
പ്ലഗിൻ റൂട്ടുമായി ബന്ധപ്പെട്ട പാത്ത്.
