# Ultimate AI Connector for Compatible Endpoints മാറ്റങ്ങൾ രേഖപ്പെടുത്തൽ {#ultimate-ai-connector-for-compatible-endpoints-changelog}

## Version 2.0.0 — 2026-04-24 ന് പുറത്തിറക്കി {#version-200--released-on-2026-04-24}

* **പുതിയത്:** ഒന്നിലധികം പ്രൊവൈഡർ പിന്തുണ — ഒന്നിലധികം AI എൻഡ്‌പോയിന്റുകൾ സജ്ജീകരിക്കാനും, പ്രൊവൈഡർമാർക്കിടയിൽ ഓട്ടോമാറ്റിക് ഫെല്ലബാക്ക് സഹിതം അഭ്യർത്ഥനകൾ റൂട്ട് ചെയ്യാനും കഴിയും.
* **പരിഹാരം:** ഓരോ പ്രൊവൈഡറിനും ശരിയായ പ്രൊവൈഡർ ID-കൾ, രജിസ്‌ട്രേഷൻ URL-കൾ, കൂടാതെ HTTP ഫിൽട്ടർ സ്കോപ്പിംഗ് എന്നിവയോടെ Multi-provider SDK സംയോജനം.
* **പരിഹാരം:** പുതിയ പ്രൊവൈഡർ കാർഡ് ഇപ്പോൾ ചേർക്കുമ്പോൾ തന്നെ ഓട്ടോമാറ്റിക്കായി വികസിക്കുന്നു; പ്ലഗിൻ അപ്ഡേറ്റ് ചെയ്യുമ്പോൾ സ്ക്രിപ്റ്റ് കാഷ് ബസ്റ്റിംഗ് (script cache busting) നടക്കുന്നു.
* **പരിഹാരം:** eval() എന്നിവയ്ക്കായി ഡൈനാമിക് പ്രൊവൈഡർ ക്ലാസ് നെയിംസ്‌പേസ് (namespace) കൂടാതെ പുതിയതായി ചേർത്ത പ്രൊവൈഡർമാർക്ക് ഓട്ടോ-വികസന സ്വഭാവം (auto-expand behaviour).
* **പരിഹാരം:** WordPress 6.9+ അനുയോജ്യതയ്ക്കായി സ്ഥിരതയുള്ള Card/CardBody/CardHeader/CardDivider കോമ്പോണന്റുകൾ ഉപയോഗിച്ചു (ഇനി പരീക്ഷണാടിസ്ഥാനത്തിലുള്ളവയല്ല).
* **പരിഹാരം:** പ്രൊവൈഡർ വലിച്ചിഴച്ച് ക്രമീകരിക്കുമ്പോൾ (drag-to-reorder) ഉപയോഗിച്ചിരുന്ന അൺഡിഫൈൻഡ് DragHandle മാറ്റി യൂണിക്കോഡ് ഗ്രിപ്പ് ഐക്കൺ ഉപയോഗിച്ചു.
* **പരിഹാരം:** ഓരോ പേജ് ലോഡിലും പ്രവർത്തിച്ചിരുന്ന ബ്ലോക്കിംഗ് HTTP അഭ്യർത്ഥന ഒഴിവാക്കി.
* **മെച്ചപ്പെടുത്തിയത്:** GitHub Actions വർക്ക്‌ഫ്ലോകൾ Node.js 24 ലേക്ക് അപ്ഗ്രേഡ് ചെയ്തു.
