# Filter: wp_ultimo_skip_network_active_check {#filter-wpultimoskipnetworkactivecheck}

ഈ ചെക്ക് ഒഴിവാക്കാൻ (short-circuit) ഡെവലപ്പർമാർക്ക് ഇത് അനുവദിക്കുന്നു.

composer-based ആയതോ Bedrock പോലുള്ള മറ്റ് കസ്റ്റം സെറ്റപ്പുകൾ ഉപയോഗിക്കുമ്പോഴോ ഇത് വളരെ ഉപകാരപ്രദമാണ്. കാരണം, അത്തരം സാഹചര്യങ്ങളിൽ mu-plugins ആയി പ്ലഗിനുകൾ ഉപയോഗിക്കുന്നത് സാധാരണമാണ്.
