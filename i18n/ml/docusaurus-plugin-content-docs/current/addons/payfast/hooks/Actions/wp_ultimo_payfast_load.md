# Action: wp_ultimo_payfast_load

എല്ലാ ഡിപൻഡൻസികളും (dependencies) ലോഡ് ചെയ്യുമ്പോൾ ഇത് ട്രിigger ചെയ്യും.

പ്ലഗിൻ ഡെവലപ്പർമാർക്ക് പുതിയ ഫംഗ്ഷണാലിറ്റി ചേർക്കാൻ ഇത് അനുവദിക്കുന്നു. ഉദാഹരണത്തിന്, പുതിയ ഹോസ്റ്റിംഗ് പ്രൊവൈഡർമാർക്കുള്ള സപ്പോർട്ട് എന്നിവ ചേർക്കാൻ ഇത് ഉപയോഗിക്കാം.
