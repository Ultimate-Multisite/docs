# Action: wp_ultimo_site_exporter_load {#action-wpultimositeexporterload}

എല്ലാ ആശ്രിതത്വങ്ങളും (dependencies) ലോഡ് ചെയ്യുമ്പോൾ ഇത് ട്രിigger ചെയ്യപ്പെടും.

പ്ലഗിൻ ഡെവലപ്പർമാർക്ക് പുതിയ പ്രവർത്തനങ്ങൾ ചേർക്കാൻ ഇത് അനുവദിക്കുന്നു. ഉദാഹരണത്തിന്, പുതിയ ഹോസ്റ്റിംഗ് പ്രൊവൈഡർമാർക്കുള്ള പിന്തുണ എന്നിവ ചേർക്കാൻ ഇത് ഉപയോഗിക്കാം.
