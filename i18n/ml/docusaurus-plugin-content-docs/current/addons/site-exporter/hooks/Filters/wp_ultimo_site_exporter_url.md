# Filter: wp_ultimo_site_exporter_url

പ്ലഗിൻ റൂട്ടിലേക്കുള്ള റിലേറ്റീവ് പാത്ത് (relative path) കൈകാര്യം ചെയ്യാൻ ഡെവലപ്പർമാർക്ക് ഇത് അനുവദിക്കുന്നു.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $dir | `string` | പ്ലഗിൻ റൂട്ടുമായി ബന്ധപ്പെട്ട, നിങ്ങൾക്ക് ആക്‌സസ് ചെയ്യേണ്ട പാത്ത്. |

### Since {#since}

- 0.0.1
### Source {#source}

[`inc/class-helper.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-site-exporter/blob/main/inc/class-helper.php#L52) എന്ന ഫയലിലെ 52-ാം വരിയിൽ നിർവചിച്ചിരിക്കുന്നു.

## Returns {#returns}
പ്ലഗിൻ റൂട്ടുമായി ബന്ധപ്പെട്ട പാത്ത്.
