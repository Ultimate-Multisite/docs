# സൈറ്റ് എക്സ്പോർട്ടർ മാറ്റങ്ങളുടെ രേഖ

Version: 1.0.6 - പുറത്തിറങ്ങിയ തീയതി 2026-05-11
* പുതിയത്: സൈറ്റ് എക്സ്പോർട്ടുകൾ ഇപ്പോൾ ഒരു സ്വയം പ്രവർത്തിക്കുന്ന index.php കൂട്ടിച്ചേർക്കുന്നു. അതിനാൽ, ഒരു പുതിയ ഹോസ്റ്റിൽ വേറെ plugin ഇൻസ്റ്റാൾ ചെയ്യാതെ തന്നെ ഈ ZIP ഫയൽ ഇൻസ്റ്റാൾ ചെയ്യാൻ കഴിയും.
* പുതിയത്: നെറ്റ്‌വർക്ക് എക്സ്പോർട്ട് വഴി, അഡ്മിനിസ്ട്രേറ്റർമാർക്ക് സൈറ്റ് എക്സ്പോർട്ട് അഡ്മിൻ പേജിൽ നിന്ന് എല്ലാ സബ്സൈറ്റുകളും ഒറ്റ ആർക്കൈവായി എക്സ്പോർട്ട് ചെയ്യാൻ കഴിയും.

Version: 1.0.5 - പുറത്തിറങ്ങിയ തീയതി 2026-05-05
* പരിഹരിച്ചു: get_config പോലുള്ള മോക്ക് ചെയ്യാത്ത methods കൈകാര്യം ചെയ്യാൻ WP_CLI polyfill stub-ൽ __callStatic ചേർത്തു. ഇത് ടെസ്റ്റ് ബൂട്ട്സ്ട്രാപ്പ് പരാജയങ്ങൾ തടയും.

Version: 1.0.4 - പുറത്തിറങ്ങിയ തീയതി 2025-11-25

* Ultimate Multisite 2.4-മായി പൊരുത്തക്കേട് പരിഹരിച്ചു.

Version: 1.0.2 - പുറത്തിറങ്ങിയ തീയതി 2025-09-28

* പ്രിഫിക്സ് ultimate-multisite എന്ന് മാറ്റുകയും; ടെക്സ്റ്റ് ഡൊമെയ്ൻ അപ്ഡേറ്റ് ചെയ്യുകയും; പതിപ്പ് നമ്പർ വർദ്ധിപ്പിക്കുകയും ചെയ്തു.

### Version 1.0.1 - പുറത്തിറങ്ങിയ തീയതി 2023-08-09 {#version-101---released-on-2023-08-09}

* മെച്ചപ്പെടുത്തി: ഇംപോർട്ടർ ഫയലിന്റെ ശരിയായ ഫയൽ പാത്ത് ലഭിക്കുന്നു എന്ന് ഇപ്പോൾ ഉറപ്പാക്കുന്നു.
* പരിഹരിച്ചു: ഇംപോർട്ടിംഗിനിടയിൽ വരുന്ന പിശകുകൾ തടയാൻ ഒബ്ജക്റ്റ് കാഷ് (object cache) പ്രവർത്തനരഹിതമാക്കി.
* പരിഹരിച്ചു: ഇംപോർട്ടർ plugin ഡൗൺലോഡ് ലിങ്ക് പ്രവർത്തിക്കാത്ത പ്രശ്നം പരിഹരിച്ചു.
