# മാറ്റരേഖ {#changelog}

## 1.18.0 — 2026-06-29-ന് പുറത്തിറങ്ങി {#1180--released-on-2026-06-29}

### പുതിയത് {#new}

- **Google Calendar ഉപകരണങ്ങൾ** — ഷെഡ്യൂൾ-അറിയുന്ന ഓട്ടോമേഷനുകൾക്കായി ക്രമീകരിച്ച കലണ്ടറുകളും ഇവന്റുകളും വായിക്കുക
- **കോൺടാക്റ്റ് മാപ്പിംഗും പങ്കെടുക്കുന്നവർക്കുള്ള സഹായികളും** — കലണ്ടർ പങ്കാളികളെ സൈറ്റ് ഉപയോക്താക്കളുമായും കോൺടാക്റ്റുകളുമായും പൊരുത്തപ്പെടുത്തുക
- **മനുഷ്യ അംഗീകാര ഗേറ്റുകളും ഓർമ്മപ്പെടുത്തൽ രേഖകളും** — അവലോകനത്തിനായി ഓട്ടോമേഷനുകൾ നിർത്തിവെക്കുക, ആവർത്തിച്ചുള്ള അറിയിപ്പുകൾ ഒഴിവാക്കുക
- **TextBee SMS പ്രൊവൈഡർ** — TextBee വഴി ക്രമീകരിച്ച ടെക്സ്റ്റ്-സന്ദേശ അറിയിപ്പുകൾ അയയ്ക്കുക
- **മുന്നേറ്റ കമ്പാനിയൻ പാക്കേജ്** — WordPress.org ബിൽഡിൽ നിന്ന് വേർതിരിച്ച് വിതരണം ചെയ്യുന്ന വിശ്വസനീയ-ഡെവലപ്പർ ഫയൽസിസ്റ്റം, ഡാറ്റാബേസ്, WP-CLI, REST ഡിസ്പാച്ചർ, plugin ബിൽഡർ, git സ്നാപ്ഷോട്ട്, ഉപയോക്തൃ-മാനേജ്മെന്റ്, ബെഞ്ച്മാർക്ക് ഉപകരണങ്ങൾ ചേർക്കുക

### മെച്ചപ്പെടുത്തി {#improved}

- **മാനേജ്ഡ് Superdav സേവന സജ്ജീകരണം** — പിന്തുണയ്ക്കുന്ന സൈറ്റുകൾക്കായി ഹോസ്റ്റഡ് സേവന endpoint-കളും സ്വയമേവയുള്ള കണക്ഷൻ പ്രൊവിഷനിംഗും ചേർക്കുക
- **റിലീസ് പാക്കേജിംഗ്** — വേർതിരിച്ച core, Advanced ZIP-കൾ ബിൽഡ് ചെയ്യുക, രണ്ടും GitHub-ൽ പ്രസിദ്ധീകരിക്കുക, core പാക്കേജ് മാത്രം WordPress.org-ലേക്ക് അയയ്ക്കുക

### പരിഹരിച്ചു {#fixed}

- **AI അഭ്യർത്ഥന വിശ്വാസ്യത** — മോഡൽ തിരഞ്ഞെടുപ്പ്, അഭ്യർത്ഥന timeout-കൾ, സ്ഥിരസ്ഥിതി സജ്ജീകരണ മോഡൽ, reasoning-text കൈകാര്യം ചെയ്യൽ, അസാധുവായ tool call-ുകൾക്കായുള്ള വീണ്ടും ശ്രമിക്കാനുള്ള മാർഗ്ഗനിർദ്ദേശം എന്നിവ മെച്ചപ്പെടുത്തുക
- **കലണ്ടർ, ഓർമ്മപ്പെടുത്തൽ കാഠിന്യം** — Google Calendar ടോക്കണുകളും ഓർമ്മപ്പെടുത്തൽ ഡീഡ്യൂപ്ലിക്കേഷനും ശക്തിപ്പെടുത്തുക
- **ഓൺബോർഡിംഗും അംഗീകാര പുനരാരംഭങ്ങളും** — frontend ഓൺബോർഡിംഗ് ലോഞ്ചുകളും സ്ഥിരീകരിച്ച കഴിവ് പുനരാരംഭങ്ങളും പരിഹരിക്കുക
- **WordPress.org പാക്കേജിംഗ് അവലോകന പ്രശ്നങ്ങൾ** — core റിലീസിനായുള്ള പാക്കേജിംഗ് അവലോകന പ്രതികരണം പരിഹരിക്കുക

## 1.16.0 — 2026-05-20-ന് പുറത്തിറങ്ങി {#1160--released-on-2026-05-20}

### പുതിയത് {#new-1}

- **ലോഗോ SVG സൃഷ്ടിക്കുന്ന കഴിവ്** — Theme Builder-ന് ഇനി namespace-സുരക്ഷിത sanitisation സഹിതം കസ്റ്റം ലോഗോ SVG-കൾ സൃഷ്ടിച്ച് എംബെഡ് ചെയ്യാം
- **ഡിസ്കവറി അഭിമുഖത്തിലെ ഫോട്ടോ അപ്‌ലോഡ്** — സമൃദ്ധമായ ഡിസൈൻ സാഹചര്യത്തിനായി Theme Builder ഡിസ്കവറി അഭിമുഖത്തിൽ ഇപ്പോൾ ഫോട്ടോ അപ്‌ലോഡ് ഘട്ടം ഉൾപ്പെടുന്നു
- **പാലറ്റ് കോൺട്രാസ്റ്റ് സാധൂകരിക്കുന്ന കഴിവ്** — ഒരു theme-ലേക്ക് പ്രയോഗിക്കുന്നതിന് മുമ്പ് നിറ ജോഡികൾ WCAG പാലിക്കുന്നുണ്ടോ എന്ന് പരിശോധിക്കുക
- **ഹോസ്പിറ്റാലിറ്റി മെനുകൾ** — ഹോസ്പിറ്റാലിറ്റി ബിസിനസുകൾക്കായി ഘടനാപരമായ ഭക്ഷണ-പാനീയ മെനു പേജുകൾ Theme Builder-ന് ഇനി സൃഷ്ടിക്കാം
- **ഡെസ്ക്ടോപ്പ്, മൊബൈൽ പ്രിവ്യൂ റെൻഡറിംഗ്** — ഡിസൈൻ-ദിശ തിരഞ്ഞെടുക്കുമ്പോൾ നിങ്ങളുടെ ഡിസൈൻ ഡെസ്ക്ടോപ്പിലും മൊബൈൽ ഉപകരണങ്ങളിലും പ്രിവ്യൂ ചെയ്യുക
- **നാവിഗേഷൻ ലേബൽ പാരാമീറ്റർ** — Create Menu കഴിവ് ഇപ്പോൾ പേജ് തലക്കെട്ടിൽ നിന്ന് വേറിട്ട പ്രത്യേക `navigation_label` പിന്തുണയ്ക്കുന്നു
- **Tier 1 tool ലഭ്യത** — sd-ai-agent/site-scrape ഇപ്പോൾ Theme Builder-ൽ സ്ഥിരസ്ഥിതിയായി ലഭ്യമായ ഒരു Tier 1 tool ആണ്

### പരിഹരിച്ചു {#fixed-1}

- **AI Client cache** — ഇപ്പോൾ അഭ്യർത്ഥനകൾക്കിടയിലെ നിലനിൽപ്പിനായി transients പിന്തുണയ്ക്കുന്നു, ദീർഘകാലം പ്രവർത്തിക്കുന്ന agent ജോലികളിൽ ഡാറ്റ നഷ്ടം തടയുന്നു
- **Plugin വരി action link-കൾ** — വ്യക്തതയ്ക്കായി തിരുത്തി പുനർനാമകരണം ചെയ്തു

## 1.10.0 — 2026-05-05-ന് പുറത്തിറങ്ങി {#1100--released-on-2026-05-05}

### പുതിയത് {#new-2}

- **Tavily ഇന്റർനെറ്റ് തിരയൽ** — Brave Search-ിനൊപ്പം കൂടുതൽ സമൃദ്ധമായ ഇന്റർനെറ്റ് തിരയൽ ഫലങ്ങൾക്കായി Tavily-യെ തിരയൽ പ്രൊവൈഡറായി ചേർക്കുക
- **Theme-അറിയുന്ന built-in skill-കൾ** — Block Themes, Classic Themes, Kadence Blocks, Kadence Theme skill ഗൈഡുകൾ ഇപ്പോൾ plugin-നൊപ്പം ലഭ്യമാണ്
- **Site builder കോൺടാക്റ്റ് ഫോം കഴിവ്** — chat interface-ൽ നിന്ന് നേരിട്ട് ഏത് പേജിലേക്കും ഒരു കോൺടാക്റ്റ് ഫോം ചേർക്കുക

### മെച്ചപ്പെടുത്തി {#improved-1}

- **WooCommerce സംയോജനം പുനർഘടിപ്പിച്ചു** — കൂടുതൽ വിശ്വാസ്യതക്കും അനുയോജ്യതയ്ക്കുമായി ഇപ്പോൾ native WooCommerce API-കൾ ഉപയോഗിക്കുന്നു
- **പ്രൊവൈഡർ പട്ടിക സ്വയമേവ പുതുക്കുന്നു** — ഏതെങ്കിലും plugin സജീവമാക്കുകയോ നിർജ്ജീവമാക്കുകയോ ചെയ്യുമ്പോൾ

### പരിഹരിച്ചു {#fixed-2}

- **navigate-to കഴിവ്** — ചില admin പേജുകളിലെ അനന്ത reload loop പരിഹരിച്ചു
- **list-posts കഴിവ്** — ഇപ്പോൾ category, tag പേരുകൾ ശരിയായി slug-കളാക്കി പരിഹരിക്കുന്നു
- **WP-CLI commands** — മുൻ refactor-ിന് ശേഷം നഷ്ടമായ namespace alias-കൾ പുനഃസ്ഥാപിച്ചു
- **ഇവന്റ് ഓട്ടോമേഷൻ** — ഓട്ടോമേഷൻ പട്ടികകൾ ഇതുവരെ സൃഷ്ടിച്ചിട്ടില്ലാത്ത സൈറ്റുകൾ സുന്ദരമായി കൈകാര്യം ചെയ്യുന്നു
- **memory-save കഴിവ്** — ഇപ്പോൾ system instruction builder-ൽ ശരിയായ namespace prefix ഉപയോഗിക്കുന്നു
- **Scalar tool ഫലങ്ങൾ** — ഇപ്പോൾ AI-ലേക്ക് തിരികെ നൽകുന്നതിന് മുമ്പ് ശരിയായി പൊതിയുന്നു
- **ഉപയോഗ സ്ഥിതിവിവരക്കണക്കുകൾ** — പഴയ പതിപ്പുകളിൽ നിന്ന് upgrade ചെയ്യുമ്പോൾ legacy ability key format ഇപ്പോൾ ശരിയായി കൈകാര്യം ചെയ്യുന്നു
