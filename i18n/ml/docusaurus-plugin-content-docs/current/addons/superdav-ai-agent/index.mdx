# Superdav AI Agent

Superdav AI Agent എന്നത് Theme പരിചയമുള്ള കഴിവുകൾ, മെച്ചപ്പെടുത്തിയ ഇന്റർനെറ്റ് തിരച്ചിൽ ശേഷികൾ, സൈറ്റ് ബിൽഡർ സവിശേഷതകൾ, ഐച്ഛിക ഓട്ടോമേഷൻ ഇന്റഗ്രേഷനുകൾ എന്നിവ ഉപയോഗിച്ച് Gratis AI Agent വിപുലീകരിക്കുന്ന ഒരു പുരോഗമിച്ച WordPress AI അസിസ്റ്റന്റാണ്. പതിപ്പ് 1.18.0 Google Calendar വായനാ ടൂളുകൾ, കോൺടാക്റ്റ്, പങ്കെടുക്കുന്നവരുടെ മാപ്പിംഗ്, മനുഷ്യ അംഗീകാര കവാടങ്ങൾ, റിമൈൻഡർ ആവർത്തനനീക്കം, TextBee SMS അറിയിപ്പുകൾ, നിയന്ത്രിത Superdav സേവന പ്രൊവിഷനിംഗ്, വിശ്വസനീയ ഡെവലപ്പർ പ്രവർത്തനപ്രവാഹങ്ങൾക്കായി വേർതിരിച്ച് വിതരണം ചെയ്യുന്ന Advanced കമ്പാനിയൻ പാക്കേജ് എന്നിവ ചേർക്കുന്നു.

## പ്രധാന സവിശേഷതകൾ {#key-features}

- **Theme പരിചയമുള്ള കഴിവുകൾ** — നിങ്ങളുടെ സജീവ WordPress Theme, ഇൻസ്റ്റാൾ ചെയ്തിരിക്കുന്ന plugins എന്നിവയെ അടിസ്ഥാനമാക്കി മാർഗനിർദേശവും കഴിവുകളും സ്വയം ക്രമീകരിക്കുന്നു
- **ഒന്നിലധികം ഇന്റർനെറ്റ് തിരച്ചിൽ ദാതാക്കൾ** — സമ്പന്നമായ തിരച്ചിൽ ഫലങ്ങൾക്കും പുരോഗമിച്ച ഗവേഷണ ശേഷികൾക്കുമായി Brave Search, Tavily എന്നിവയിൽ നിന്ന് തിരഞ്ഞെടുക്കുക
- **സൈറ്റ് ബിൽഡർ കോൺടാക്റ്റ് ഫോമുകൾ** — നിങ്ങൾ ഇൻസ്റ്റാൾ ചെയ്ത ഫോം plugin ഉപയോഗിച്ച് ചാറ്റ് ഇന്റർഫേസിൽ നിന്ന് നേരിട്ട് പൂർണ്ണമായി പ്രവർത്തിക്കുന്ന കോൺടാക്റ്റ് ഫോമുകൾ സൃഷ്ടിക്കുക
- **മെച്ചപ്പെടുത്തിയ സൈറ്റ് ബിൽഡർ** — മികച്ച plugin കണ്ടെത്തലും പിശക് വീണ്ടെടുക്കലും ഉള്ള മെച്ചപ്പെടുത്തിയ ബഹുഘട്ട സൈറ്റ് നിർമ്മാണ പദ്ധതികൾ
- **അന്തർനിർമ്മിത Theme കഴിവുകൾ** — Block Themes, Classic Themes, Kadence Blocks, Kadence Theme എന്നിവയ്ക്കുള്ള പ്രത്യേക മാർഗനിർദേശം
- **മെച്ചപ്പെടുത്തിയ WooCommerce ഇന്റഗ്രേഷൻ** — കൂടുതൽ വിശ്വാസ്യതയ്ക്കും അനുയോജ്യതയ്ക്കുമായി സ്വദേശ WooCommerce APIകൾ ഉപയോഗിക്കാൻ പുനർഘടിപ്പിച്ചു
- **സ്വയമേവയുള്ള ദാതാവ് കണ്ടെത്തൽ** — plugins, themes എന്നിവ സജീവമാക്കുകയോ നിർജ്ജീവമാക്കുകയോ ചെയ്യുമ്പോൾ ലഭ്യമായവ സ്വയം കണ്ടെത്തി പുതുക്കുന്നു
- **കസ്റ്റം ലോഗോ സൃഷ്ടിക്കൽ** — സ്വയമേവയുള്ള namespace-സുരക്ഷിത sanitisation സഹിതം കസ്റ്റം SVG ലോഗോകൾ സൃഷ്ടിച്ച് ഉൾച്ചേർക്കുക
- **പാലറ്റ് കോൺട്രാസ്റ്റ് സാധൂകരണം** — നിങ്ങളുടെ Theme-ൽ പ്രയോഗിക്കുന്നതിന് മുമ്പ് WCAG ആക്സസിബിലിറ്റി പാലനത്തിനായി നിറജോടികൾ പരിശോധിക്കുക
- **ഫോട്ടോ ഉപയോഗിച്ച് മെച്ചപ്പെടുത്തിയ കണ്ടെത്തൽ** — ഡിസൈൻ ദിശയെ സ്വാധീനിക്കാൻ Theme Builder കണ്ടെത്തൽ അഭിമുഖത്തിനിടെ ഫോട്ടോകൾ അപ്ലോഡ് ചെയ്യുക
- **ഹോസ്പിറ്റാലിറ്റി മെനുകൾ** — റസ്റ്റോറന്റുകൾ, കഫേകൾ, ഹോസ്പിറ്റാലിറ്റി ബിസിനസുകൾ എന്നിവയ്ക്കായി ഘടനാപരമായ ഭക്ഷണ-പാനീയ മെനു പേജുകൾ സൃഷ്ടിക്കുക
- **ഡെസ്ക്ടോപ്പ്, മൊബൈൽ പ്രിവ്യൂകൾ** — ഡിസൈൻ ദിശ തിരഞ്ഞെടുക്കുമ്പോൾ ഡെസ്ക്ടോപ്പ്, മൊബൈൽ ഉപകരണങ്ങളിൽ നിങ്ങളുടെ ഡിസൈൻ പ്രിവ്യൂ ചെയ്യുക
- **ടിയർ 1 സൈറ്റ് സ്ക്രേപ്പിംഗ്** — മത്സരക്കാരുടെയും പ്രചോദനത്തിനായുള്ള വെബ്‌സൈറ്റുകളുടെയും വിശകലനത്തിനായി sd-ai-agent/site-scrape ഇപ്പോൾ ഡിഫോൾട്ടായി ലഭ്യമാണ്
- **Google Calendar വായനാ ടൂളുകൾ** — വരാനിരിക്കുന്ന ഷെഡ്യൂൾ സന്ദർഭം ഓട്ടോമേഷനുകൾക്ക് മനസ്സിലാക്കാൻ ക്രമീകരിച്ച കലണ്ടറുകളും ഇവന്റുകളും വായിക്കുക
- **കോൺടാക്റ്റ്, പങ്കെടുക്കുന്നവരുടെ മാപ്പിംഗ്** — തുടർപ്രവർത്തനങ്ങൾ സൃഷ്ടിക്കുന്നതിന് മുമ്പ് കലണ്ടർ പങ്കാളികളെ അറിയപ്പെടുന്ന സൈറ്റ് ഉപയോക്താക്കളോടും കോൺടാക്റ്റുകളോടും പൊരുത്തപ്പെടുത്തുക
- **മനുഷ്യ അംഗീകാര കവാടങ്ങൾ** — സന്ദേശങ്ങൾ അയയ്ക്കുന്നതിനോ ജോലി പുനരാരംഭിക്കുന്നതിനോ മാറ്റങ്ങൾ വരുത്തുന്നതിനോ മുമ്പ് അവലോകനത്തിനായി സെൻസിറ്റീവ് ഓട്ടോമേഷനുകൾ താൽക്കാലികമായി നിർത്തുക
- **റിമൈൻഡർ രേഖകളും ആവർത്തനനീക്കവും** — ഒരേ സ്വീകർത്താവിനെ ആവർത്തിച്ചുള്ള ഓട്ടോമേഷൻ റൺസ് രണ്ടുതവണ അറിയിക്കാതിരിക്കാൻ ഇതിനകം അയച്ച റിമൈൻഡറുകൾ ട്രാക്ക് ചെയ്യുക
- **TextBee SMS അറിയിപ്പുകൾ** — ഒരു SMS ദാതാവ് പ്രവർത്തനക്ഷമമാക്കിയിരിക്കുമ്പോൾ TextBee വഴി ക്രമീകരിച്ച ടെക്സ്റ്റ്-സന്ദേശ അറിയിപ്പുകൾ അയയ്ക്കുക
- **നിയന്ത്രിത Superdav സേവന പ്രൊവിഷനിംഗ്** — മാനുവൽ endpoint ക്രമീകരണത്തിന് പകരം സ്വയമേവയുള്ള സജ്ജീകരണത്തോടെ പിന്തുണയുള്ള സൈറ്റുകളെ ഹോസ്റ്റ് ചെയ്ത Superdav endpoints-ലേക്ക് ബന്ധിപ്പിക്കുക
- **Advanced കമ്പാനിയൻ പാക്കേജ്** — filesystem, database, WP-CLI, REST dispatcher, plugin builder, git snapshot, user-management, benchmark പ്രവർത്തനപ്രവാഹങ്ങൾക്കായി വേർതിരിച്ച് വിതരണം ചെയ്യുന്ന വിശ്വസനീയ ഡെവലപ്പർ ടൂളുകൾ

## ആവശ്യകതകൾ {#requirements}

- WordPress 6.0 അല്ലെങ്കിൽ അതിലധികം
- PHP 8.1 അല്ലെങ്കിൽ അതിലധികം
- WordPress Multisite (നെറ്റ്‌വർക്കിൽ സജീവമാക്കിയ) അല്ലെങ്കിൽ single-site ഇൻസ്റ്റലേഷൻ
- സജീവമായ AI ദാതാവ് കണക്റ്റർ plugin (Anthropic, OpenAI, അല്ലെങ്കിൽ അനുയോജ്യം)
- Gratis AI Agent v1.9.0 അല്ലെങ്കിൽ അതിലധികം (Superdav AI Agent, Gratis AI Agent വിപുലീകരിക്കുന്നു)

## ഇൻസ്റ്റലേഷൻ {#installation}

1. plugin ഫയലുകൾ `/wp-content/plugins/superdav-ai-agent` എന്നതിലേക്ക് അപ്ലോഡ് ചെയ്യുക
2. WordPress-ലെ **Plugins** സ്ക്രീൻ വഴി plugin സജീവമാക്കുക
3. Gratis AI Agent സജീവമാക്കിയിട്ടുണ്ടെന്നും ഉറപ്പാക്കുക
4. **Gratis AI Agent → Settings** എന്നതിലേക്ക് പോയി ആവശ്യാനുസരണം നിങ്ങളുടെ ഇന്റർനെറ്റ് തിരച്ചിൽ ദാതാവ്, നിയന്ത്രിത Superdav സേവന കണക്ഷൻ, Google Calendar, അല്ലെങ്കിൽ TextBee credentials ക്രമീകരിക്കുക

WordPress.org പാക്കേജിൽ പ്രധാന Superdav AI Agent സവിശേഷതകൾ അടങ്ങിയിരിക്കുന്നു. വിശ്വസനീയ ഡെവലപ്പർ ടൂളുകൾ വ്യക്തമായി ഇൻസ്റ്റാൾ ചെയ്ത് അധികാരപ്പെടുത്തിയിരിക്കുന്ന പരിസരങ്ങൾക്കായി Advanced കമ്പാനിയൻ പാക്കേജ് വേർതിരിച്ച് നിർമ്മിക്കുകയും വിതരണം ചെയ്യുകയും ചെയ്യുന്നു.

## വേഗത്തിലുള്ള തുടക്കം {#quick-start}

സജീവമാക്കിയതിന് ശേഷം, Theme പരിചയമുള്ള കഴിവുകളും അധിക ശേഷികളും ഉപയോഗിച്ച് Superdav AI Agent നിങ്ങളുടെ Gratis AI Agent ഇൻസ്റ്റലേഷൻ സ്വയം മെച്ചപ്പെടുത്തുന്നു. WordPress അഡ്മിൻ സൈഡ്ബാറിലെ **Gratis AI Agent** പാനൽ തുറന്ന് പുതിയ സവിശേഷതകൾ ഉപയോഗിച്ച് തുടങ്ങുക.

**ഉദാഹരണ നിർദേശങ്ങൾ:**

```
സമ്പർക്ക പേജിൽ ഒരു കോൺടാക്റ്റ് ഫോം ചേർക്കുക.
```

```
ഏറ്റവും പുതിയ WordPress സുരക്ഷാ മികച്ച രീതികൾക്കായി വെബിൽ തിരയുക.
```

```
സജീവ Theme-ന്റെ block patterns ഉപയോഗിച്ച് ഒരു hero section സൃഷ്ടിക്കുക.
```

```
നാളെയുടെ ക്രമീകരിച്ച Google Calendar ഇവന്റുകൾ അവലോകനം ചെയ്ത് പങ്കെടുക്കുന്നവർക്ക് റിമൈൻഡറുകൾ കരട് തയ്യാറാക്കുക.
```

```
ഞാൻ സന്ദേശം അംഗീകരിച്ച ശേഷം ഒരു SMS റിമൈൻഡർ അയയ്ക്കുക.
```

## Theme പരിചയമുള്ള കഴിവുകൾ {#theme-aware-skills}

Superdav AI Agent നിങ്ങളുടെ സജീവ Theme ഉം ഇൻസ്റ്റാൾ ചെയ്ത plugins ഉം സ്വയം കണ്ടെത്തി, അനുയോജ്യമായ Theme പരിചയമുള്ള കഴിവ് അസിസ്റ്റന്റിന്റെ സന്ദർഭത്തിലേക്ക് ലോഡ് ചെയ്യുന്നു. ലഭ്യമായ കഴിവുകളിൽ ഉൾപ്പെടുന്നത്:

| കഴിവ് | ബാധകമാകുന്നത് |
|---|---|
| **Block Themes** | WordPress block-based (Full Site Editing) ആർക്കിടെക്ചർ ഉപയോഗിക്കുന്ന Themes |
| **Classic Themes** | PHP templates ഉം classic editor ഉം ഉപയോഗിക്കുന്ന പരമ്പരാഗത WordPress themes |
| **Kadence Blocks** | Kadence Blocks plugin സജീവമായ സൈറ്റുകൾ |
| **Kadence Theme** | സജീവ Theme ആയി Kadence theme ഉള്ള സൈറ്റുകൾ |

ഓരോ കഴിവിനുമുള്ള വിശദമായ മാർഗനിർദേശത്തിനായി [Theme പരിചയമുള്ള കഴിവുകൾ](../../user-guide/configuration/theme-aware-skills) കാണുക.

## ഇന്റർനെറ്റ് തിരച്ചിൽ ദാതാക്കൾ {#internet-search-providers}

Superdav AI Agent v1.10.0 ഇന്റർനെറ്റ് തിരച്ചിൽ അന്വേഷണങ്ങൾക്കായി Brave Search-ന്റെ പകരമായി **Tavily** ചേർക്കുന്നു. രണ്ട് ദാതാക്കളും തത്സമയ തിരച്ചിൽ ഫലങ്ങൾ നൽകുന്നു, കൂടാതെ **Gratis AI Agent → Settings → Advanced** എന്നതിൽ ക്രമീകരിക്കാം.

| ദാതാവ് | സവിശേഷതകൾ | സജ്ജീകരണം |
|---|---|---|
| **Brave Search** | വേഗമേറിയ, സ്വകാര്യത കേന്ദ്രീകൃത തിരയൽ ഫലങ്ങൾ | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | കൂടുതൽ സമ്പന്നമായ തിരയൽ ഫലങ്ങൾ, പുരോഗമിച്ച ഗവേഷണ ശേഷികൾ | [Tavily API](https://tavily.com/) |

## കലണ്ടർ, അംഗീകാരം, അറിയിപ്പ് പ്രവർത്തനക്രമങ്ങൾ {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 ഉള്ളടക്കത്തിനും സൈറ്റ് നിർമ്മാണ ജോലികൾക്കും അതീതമായി ഓട്ടോമേഷൻ പ്രവർത്തനക്രമങ്ങൾ വിപുലീകരിക്കുന്നു:

- **Google Calendar ഉപകരണങ്ങൾ** കലണ്ടർ ക്രെഡൻഷ്യലുകൾ ലഭ്യമായിരിക്കുമ്പോൾ സജ്ജീകരിച്ച കലണ്ടറുകളും ഇവന്റ് വിശദാംശങ്ങളും വായിക്കാൻ agent-നെ അനുവദിക്കുന്നു.
- **കോൺടാക്റ്റ്, പങ്കെടുക്കുന്നവരുടെ സഹായികൾ** ഓർമ്മപ്പെടുത്തലുകളോ തുടർ ജോലികളോ തയ്യാറാക്കുന്നതിന് മുമ്പ് ഇവന്റ് പങ്കാളികളെ WordPress ഉപയോക്താക്കളുമായോ അറിയാവുന്ന കോൺടാക്റ്റുകളുമായോ മാപ്പ് ചെയ്യുന്നു.
- **മാനുഷിക അംഗീകാര ഗേറ്റുകൾ** നിർദ്ദേശിച്ച നടപടി ഒരു അംഗീകൃത ഉപയോക്താവ് പരിശോധിക്കുന്നതുവരെ ഓട്ടോമേഷൻ താൽക്കാലികമായി നിർത്താൻ അനുവദിക്കുന്നു.
- **ഓർമ്മപ്പെടുത്തൽ റെക്കോർഡുകൾ** ഒരു ഓട്ടോമേഷൻ വീണ്ടും ശ്രമിക്കുമ്പോഴോ ആവർത്തിച്ച് ഷെഡ്യൂൾ ചെയ്യുമ്പോഴോ ഇരട്ട കലണ്ടർ അല്ലെങ്കിൽ അറിയിപ്പ് ഓർമ്മപ്പെടുത്തലുകൾ തടയുന്നു.
- **TextBee SMS പിന്തുണ** TextBee ദാതാവ് സജ്ജീകരിച്ചിരിക്കുമ്പോൾ ടെക്സ്റ്റ് സന്ദേശ അറിയിപ്പുകൾ സാധ്യമാക്കുന്നു.

സൂക്ഷ്മമായ സന്ദേശങ്ങൾ അയയ്ക്കുന്നതിന് മുമ്പ് മാനുഷിക പരിശോധന ഇപ്പോഴും ആവശ്യമുള്ള ഷെഡ്യൂൾ-അവബോധമുള്ള സഹായം, പങ്കെടുക്കുന്നവരുടെ തുടർനടപടി, ഓർമ്മപ്പെടുത്തൽ പ്രവർത്തനക്രമങ്ങൾ എന്നിവയ്ക്ക് ഈ ഉപകരണങ്ങൾ ഉപയോഗിക്കുക.

## സൈറ്റ് ബിൽഡർ കോൺടാക്റ്റ് ഫോമുകൾ {#site-builder-contact-forms}

സൈറ്റ് ബിൽഡർ agent-ന് ഇപ്പോൾ ചാറ്റ് ഇന്റർഫേസിൽ നിന്ന് നേരിട്ട് കോൺടാക്റ്റ് ഫോമുകൾ സൃഷ്ടിക്കാം. ഒരു കോൺടാക്റ്റ് ഫോം സൃഷ്ടിക്കാൻ സൈറ്റ് ബിൽഡറോട് ആവശ്യപ്പെടുക. അത് നിങ്ങളുടെ ഇൻസ്റ്റാൾ ചെയ്ത ഫോം plugin (Contact Form 7, WPForms, Fluent Forms, അല്ലെങ്കിൽ Gravity Forms) സ്വയം കണ്ടെത്തി ഉപയോഗിക്കാൻ തയ്യാറായ ഒരു ഫോം സൃഷ്ടിക്കും.

വിശദമായ മാർഗ്ഗനിർദ്ദേശങ്ങൾക്ക് [സൈറ്റ് ബിൽഡർ കോൺടാക്റ്റ് ഫോമുകൾ](../../user-guide/administration/site-builder-contact-forms) കാണുക.

## Theme Builder സവിശേഷതകൾ {#theme-builder-features}

Superdav AI Agent v1.16.0-ലെ Theme Builder ശക്തമായ പുതിയ ശേഷികൾ ഉൾക്കൊള്ളുന്നു:

### ഫോട്ടോ അപ്‌ലോഡോടുകൂടിയ കണ്ടെത്തൽ അഭിമുഖം {#discovery-interview-with-photo-upload}

Theme Builder കണ്ടെത്തൽ അഭിമുഖത്തിൽ ഇപ്പോൾ ദൃശ്യ പ്രചോദനം നൽകാൻ കഴിയുന്ന ഒരു ഫോട്ടോ അപ്‌ലോഡ് ഘട്ടം ഉൾപ്പെടുന്നു. നിങ്ങളുടെ ഡിസൈൻ ദിശ Theme Builder-ന് മനസ്സിലാക്കാൻ സഹായിക്കാൻ നിങ്ങളുടെ ബ്രാൻഡ്, മത്സരിക്കുന്ന സൈറ്റുകൾ, അല്ലെങ്കിൽ ഡിസൈൻ പ്രചോദനം എന്നിവയുടെ ഫോട്ടോകൾ അപ്‌ലോഡ് ചെയ്യുക.

വിശദമായ മാർഗ്ഗനിർദ്ദേശങ്ങൾക്ക് [കണ്ടെത്തൽ അഭിമുഖം](./theme-builder/discovery-interview.md) കാണുക.

### ഡെസ്ക്ടോപ്പ്, മൊബൈൽ പ്രിവ്യൂകളോടുകൂടിയ ഡിസൈൻ ദിശ {#design-direction-with-desktop-and-mobile-previews}

Theme Builder നിങ്ങളുടെ പൂർണ്ണമായ തീം സൃഷ്ടിക്കുന്നതിന് മുമ്പ് ഡെസ്ക്ടോപ്പിലും മൊബൈൽ ഉപകരണങ്ങളിലും നിങ്ങളുടെ ഡിസൈൻ പരിശോധിക്കുക. വിവിധ പ്രതികരണാത്മക ബ്രേക്ക്പോയിന്റുകളിൽ തത്സമയ പ്രിവ്യൂകൾ കാണുക.

വിശദമായ മാർഗ്ഗനിർദ്ദേശങ്ങൾക്ക് [ഡിസൈൻ ദിശ](./theme-builder/design-direction.md) കാണുക.

### ഹോസ്പിറ്റാലിറ്റി മെനുകൾ {#hospitality-menus}

Theme Builder-ന് ഇപ്പോൾ റെസ്റ്റോറന്റുകൾ, കഫേകൾ, ബാറുകൾ, മറ്റ് ഹോസ്പിറ്റാലിറ്റി ബിസിനസുകൾ എന്നിവയ്ക്കായി ഘടനാപരമായ ഭക്ഷണ, പാനീയ മെനു പേജുകൾ സൃഷ്ടിക്കാം. മെനുകളിൽ വിലനിർണ്ണയം, വിവരണങ്ങൾ, ഭക്ഷണ സംബന്ധമായ വിവരങ്ങൾ, അലർജൻ മുന്നറിയിപ്പുകൾ എന്നിവ ഉൾപ്പെടുന്നു.

വിശദമായ മാർഗ്ഗനിർദ്ദേശങ്ങൾക്ക് [ഹോസ്പിറ്റാലിറ്റി മെനുകൾ](./theme-builder/hospitality-menus.md) കാണുക.

## പുരോഗമിച്ച കമ്പാനിയൻ പാക്കേജ് {#advanced-companion-package}

Superdav AI Agent v1.18.0 വിശ്വസനീയമായ ഡെവലപ്പർ പ്രവർത്തനക്രമങ്ങൾക്കായി വേർതിരിച്ച് വിതരണം ചെയ്യുന്ന ഒരു പുരോഗമിച്ച കമ്പാനിയൻ പാക്കേജ് ചേർക്കുന്നു. ഈ ശേഷികൾ WordPress.org കോർ പാക്കേജിൽ ഉൾപ്പെടുത്തിയിട്ടില്ല. ഡെവലപ്പർ ടൂളിംഗിലേക്ക് agent-ന് പ്രവേശനം നൽകുന്നതിനെ അഡ്മിനിസ്ട്രേറ്റർമാർ വ്യക്തമായി വിശ്വസിക്കുന്ന സൈറ്റുകളിൽ മാത്രം കമ്പാനിയൻ പാക്കേജ് ഇൻസ്റ്റാൾ ചെയ്യുക.

പുരോഗമിച്ച ഉപകരണങ്ങളിൽ ഫയൽസിസ്റ്റം പ്രവർത്തനങ്ങൾ, ഡാറ്റാബേസ് ആക്സസ്, WP-CLI, REST ഡിസ്പാച്ചർ പ്രവർത്തനക്രമങ്ങൾ, plugin ബിൽഡർ യൂട്ടിലിറ്റികൾ, git സ്നാപ്ഷോട്ടുകൾ, ഉപയോക്തൃ-മാനേജ്മെന്റ് സഹായികൾ, ബെഞ്ച്മാർക്ക് ടൂളിംഗ് എന്നിവ ഉൾപ്പെടുന്നു. ഈ ഉപകരണങ്ങൾ വിശ്വസനീയമായ അഡ്മിനിസ്ട്രേറ്റർമാർക്ക് മാത്രമായി പരിമിതപ്പെടുത്തുക. പ്രൊഡക്ഷൻ സൈറ്റുകളിൽ ഇവ പ്രവർത്തനക്ഷമമാക്കുന്നതിന് മുമ്പ് അംഗീകാര-ഗേറ്റ് സജ്ജീകരണം പരിശോധിക്കുക.
