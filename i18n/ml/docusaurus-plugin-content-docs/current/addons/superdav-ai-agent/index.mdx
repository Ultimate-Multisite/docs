# Superdav AI Agent

Superdav AI Agent എന്നത് ഒരു നൂതനമായ WordPress AI അസിസ്റ്റന്റാണ്. ഇത് Gratis AI Agent-നെ theme-aware കഴിവുകൾ, മെച്ചപ്പെടുത്തിയ ഇന്റർനെറ്റ് സെർച്ച് ശേഷികൾ, മികച്ച സൈറ്റ് ബിൽഡർ ഫീച്ചറുകൾ എന്നിവ ഉപയോഗിച്ച് വികസിപ്പിക്കുന്നു. പതിപ്പ് 1.16.0-ൽ കസ്റ്റം ലോഗോ ജനറേഷൻ, പാലറ്റ് കോൺട്രാസ്റ്റ് വാലിഡേഷൻ, ഡിസ്കവറി ഇന്റർവ്യൂവിൽ ഫോട്ടോ അപ്‌ലോഡ് ചെയ്യാനുള്ള സൗകര്യം, ഘടനാപരമായ ഹോസ്പിറ്റാലിറ്റി മെനു ജനറേഷൻ, Theme Builder-ൽ ഡെസ്ക്ടോപ്പ്/മൊബൈൽ പ്രിവ്യൂ റെൻഡറിംഗ് എന്നിവ കൂട്ടിച്ചേർത്തു.

## പ്രധാന സവിശേഷതകൾ (Key Features)

- **Theme-aware skills** — നിങ്ങളുടെ സജീവമായ WordPress തീമും ഇൻസ്റ്റാൾ ചെയ്ത പ്ലഗിനുകളും അനുസരിച്ച് ഗൈഡൻസും കഴിവുകളും സ്വയമേവ ക്രമീകരിക്കുന്നു.
- **ഒന്നിലധികം ഇന്റർനെറ്റ് സെർച്ച് പ്രൊവൈഡർമാർ** — സമ്പന്നമായ സെർച്ച് റിസൾട്ടുകൾക്കും അഡ്വാൻസ്ഡ് ഗവേഷണ ശേഷികൾക്കുമായി Brave Search, Tavily എന്നിവയിൽ നിന്ന് തിരഞ്ഞെടുക്കുക.
- **സൈറ്റ് ബിൽഡർ കോൺടാക്റ്റ് ഫോമുകൾ** — നിങ്ങളുടെ ഇൻസ്റ്റാൾ ചെയ്ത ഫോം പ്ലഗിൻ ഉപയോഗിച്ച് ചാറ്റ് ഇന്റർഫേസിൽ നിന്ന് തന്നെ പൂർണ്ണമായും പ്രവർത്തനക്ഷമമായ കോൺടാക്റ്റ് ഫോമുകൾ സൃഷ്ടിക്കുക.
- **മെച്ചപ്പെടുത്തിയ സൈറ്റ് ബിൽഡർ** — മികച്ച പ്ലഗിൻ ഡിസ്കവറിയോടും എറർ റിക്കവറിയോടും കൂടിയ മെച്ചപ്പെടുത്തിയ മൾട്ടി-ഫേസ് സൈറ്റ് ബിൽഡ് പ്ലാനുകൾ.
- **ബിൽറ്റ്-ഇൻ തീം സ്കില്ലുകൾ** — Block Themes, Classic Themes, Kadence Blocks, Kadence Theme എന്നിവയ്ക്കായി പ്രത്യേക ഗൈഡൻസ്.
- **മെച്ചപ്പെടുത്തിയ WooCommerce സംയോജനം** — മികച്ച വിശ്വാസ്യതയ്ക്കും അനുയോജ്യതയ്ക്കുമായി native WooCommerce APIs ഉപയോഗിക്കാൻ പുനഃക്രമീകരിച്ചു.
- **ഓട്ടോമാറ്റിക് പ്രൊവൈഡർ ഡിറ്റക്ഷൻ** — പ്ലഗിനുകളും തീമുകളും ആക്ടിവേറ്റ് ചെയ്യുമ്പോഴോ ഡിആക്ടിവേറ്റ് ചെയ്യുമ്പോഴോ ലഭ്യമായവയെ സ്വയമേവ കണ്ടെത്തി പുതുക്കുന്നു.
- **കസ്റ്റം ലോഗോ ജനറേഷൻ** — ഓട്ടോമാറ്റിക് namespace-safe sanitisation സഹിതം കസ്റ്റം SVG ലോഗോകൾ ജനറേറ്റ് ചെയ്യാനും ഉൾപ്പെടുത്താനും കഴിയും.
- **പാലറ്റ് കോൺട്രാസ്റ്റ് വാലിഡേഷൻ** — നിങ്ങളുടെ തീമിൽ പ്രയോഗിക്കുന്നതിന് മുമ്പ് WCAG ആക്സസിബിലിറ്റി പാലിക്കാനുള്ള നിറങ്ങളുടെ ജോഡികൾ പരിശോധിക്കുന്നു.
- **ഫോട്ടോ-എൻഹാൻസ്ഡ് ഡിസ്കവറി** — Theme Builder discovery interview സമയത്ത് ഡിസൈൻ ദിശയെ സ്വാധീനിക്കാൻ ഫോട്ടോകൾ അപ്‌ലോഡ് ചെയ്യുക.
- **ഹോസ്പിറ്റാലിറ്റി മെനുകൾ** — റെസ്റ്റോറന്റുകൾ, കഫേകൾ, ഹോസ്പിറ്റാലിറ്റി ബിസിനസ്സുകൾ എന്നിവയ്ക്കായി ഘടനാപരമായ ഭക്ഷണ, പാനീയ മെനു പേജുകൾ ജനറേറ്റ് ചെയ്യുക.
- **ഡെസ്ക്ടോപ്പ്, മൊബൈൽ പ്രിവ്യൂകൾ** — ഡിസൈൻ ദിശ തിരഞ്ഞെടുക്കുന്ന സമയത്ത് ഡെസ്ക്ടോപ്പ്, മൊബൈൽ ഉപകരണങ്ങളിൽ നിങ്ങളുടെ ഡിസൈൻ പ്രിവ്യൂ ചെയ്യുക.
- **Tier 1 സൈറ്റ് സ്ക്രാപ്പിംഗ്** — എതിരാളികളുടെയും പ്രചോദന സൈറ്റുകളുടെയും വിശകലനത്തിനായി sd-ai-agent/site-scrape ഇപ്പോൾ ഡിഫോൾട്ടായി ലഭ്യമാണ്.

## ആവശ്യകതകൾ (Requirements)

- WordPress 6.0 അല്ലെങ്കിൽ അതിലും ഉയർന്നത്
- PHP 8.1 അല്ലെങ്കിൽ അതിലും ഉയർന്നത്
- WordPress Multisite (network-activated) അല്ലെങ്കിൽ single-site installation
- ഒരു സജീവ AI പ്രൊവൈഡർ കണക്റ്റർ പ്ലഗിൻ (Anthropic, OpenAI, അല്ലെങ്കിൽ അനുയോജ്യം)
- Gratis AI Agent v1.9.0 അല്ലെങ്കിൽ അതിലും ഉയർന്നത് (Superdav AI Agent, Gratis AI Agent-നെ വികസിപ്പിക്കുന്നു)

## ഇൻസ്റ്റാളേഷൻ

1. പ്ലഗിൻ ഫയലുകൾ `/wp-content/plugins/superdav-ai-agent` എന്നതിലേക്ക് അപ്‌ലോഡ് ചെയ്യുക.
2. WordPress-ലെ **Plugins** സ്ക്രീനിലൂടെ പ്ലഗിൻ ആക്ടിവേറ്റ് ചെയ്യുക.
3. Gratis AI Agent-ഉം ആക്ടിവേറ്റ് ചെയ്തിട്ടുണ്ടെന്ന് ഉറപ്പാക്കുക.
4. **Gratis AI Agent → Settings** എന്നതിലേക്ക് പോയി നിങ്ങളുടെ ഇന്റർനെറ്റ് സെർച്ച് പ്രൊവൈഡർ (ഓപ്ഷണൽ) കോൺഫിഗർ ചെയ്യുക.

## പെട്ടെന്നുള്ള തുടക്കം (Quick Start)

ആക്ടിവേറ്റ് ചെയ്തുകഴിഞ്ഞാൽ, Superdav AI Agent നിങ്ങളുടെ Gratis AI Agent ഇൻസ്റ്റാളേഷനെ സ്വയമേവ theme-aware കഴിവുകളോടും അധിക ശേഷികളോടും കൂടി മെച്ചപ്പെടുത്തും. WordPress അഡ്മിൻ സൈഡ്ബാറിലെ **Gratis AI Agent** പാനൽ തുറന്ന് പുതിയ ഫീച്ചറുകൾ ഉപയോഗിക്കാൻ തുടങ്ങുക.

**ഉദാഹരണ നിർദ്ദേശങ്ങൾ:**

```
Add a contact form to the Contact page.
```

```
Search the web for the latest WordPress security best practices.
```

```
Create a hero section using the active theme's block patterns.
```

## Theme-Aware Skills

Superdav AI Agent നിങ്ങളുടെ സജീവമായ തീമും ഇൻസ്റ്റാൾ ചെയ്ത പ്ലഗിനുകളും സ്വയമേവ കണ്ടെത്തി, അനുയോജ്യമായ theme-aware skill അസിസ്റ്റന്റിന്റെ സന്ദർഭത്തിലേക്ക് ലോഡ് ചെയ്യുന്നു. ലഭ്യമായ കഴിവുകൾ ഇവയാണ്:

| Skill | Applies To |
|---|---|
| **Block Themes** | WordPress block-based (Full Site Editing) architecture ഉപയോഗിക്കുന്ന തീമുകൾ |
| **Classic Themes** | PHP ടെംപ്ലേറ്റുകളും ക്ലാസിക് എഡിറ്ററും ഉപയോഗിക്കുന്ന പരമ്പരാഗത WordPress തീമുകൾ |
| **Kadence Blocks** | Kadence Blocks പ്ലഗിൻ സജീവമായ സൈറ്റുകൾ |
| **Kadence Theme** | സജീവമായ തീമായി Kadence theme ഉള്ള സൈറ്റുകൾ |

ഓരോ സ്കില്ലിനെക്കുറിച്ചുമുള്ള വിശദമായ ഗൈഡൻസിനായി [Theme-Aware Skills](../../user-guide/configuration/theme-aware-skills) കാണുക.

## Internet Search Providers

Superdav AI Agent v1.10.0 ഇന്റർനെറ്റ് സെർച്ച് ക്വറികൾക്കായി Brave Search-ന് ഒരു ബദലായി **Tavily** കൂട്ടിച്ചേർക്കുന്നു. രണ്ട് പ്രൊവൈഡർമാരും തത്സമയ സെർച്ച് റിസൾട്ടുകൾ നൽകുന്നു, കൂടാതെ ഇവയെല്ലാം **Gratis AI Agent → Settings → Advanced** എന്നതിൽ കോൺഫിഗർ ചെയ്യാവുന്നതാണ്.

| Provider | Features | Setup |
|---|---|---|
| **Brave Search** | വേഗതയേറിയ, പ്രൈവസി-ഫോക്കസ്ഡ് സെർച്ച് റിസൾട്ടുകൾ | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | സമ്പന്നമായ സെർച്ച് റിസൾട്ടുകൾ, അഡ്വാൻസ്ഡ് ഗവേഷണ ശേഷികൾ | [Tavily API](https://tavily.com/) |

കോൺഫിഗറേഷൻ നിർദ്ദേശങ്ങൾക്കായി [Internet Search](../../user-guide/configuration/internet-search) കാണുക.

## Site Builder Contact Forms

Site Builder agent ഇപ്പോൾ ചാറ്റ് ഇന്റർഫേസിൽ നിന്ന് നേരിട്ട് കോൺടാക്റ്റ് ഫോമുകൾ സൃഷ്ടിക്കാൻ കഴിയും. Site Builder-യോട് ഒരു കോൺടാക്റ്റ് ഫോം ഉണ്ടാക്കാൻ ആവശ്യപ്പെടുക മാത്രം ചെയ്യുക, അത് നിങ്ങളുടെ ഇൻസ്റ്റാൾ ചെയ്ത ഫോം പ്ലഗിൻ (Contact Form 7, WPForms, Fluent Forms, അല്ലെങ്കിൽ Gravity Forms) സ്വയമേവ കണ്ടെത്തി ഉപയോഗിക്കാൻ തയ്യാറായ ഒരു ഫോം ജനറേറ്റ് ചെയ്യും.

വിശദമായ ഗൈഡൻസിനായി [Site Builder Contact Forms](../../user-guide/administration/site-builder-contact-forms) കാണുക.

## Theme Builder Features

Superdav AI Agent v1.16.0-ലെ Theme Builder-ൽ ശക്തമായ പുതിയ കഴിവുകൾ ഉൾപ്പെടുന്നു:

### Photo Upload സഹിതമുള്ള Discovery Interview

Theme Builder discovery interview-ൽ ഇപ്പോൾ ഒരു ഫോട്ടോ അപ്‌ലോഡ് സ്റ്റെപ്പ് ഉൾപ്പെടുന്നു, അവിടെ നിങ്ങൾക്ക് വിഷ്വൽ ഇൻസ്പിറേഷൻ നൽകാൻ കഴിയും. നിങ്ങളുടെ ബ്രാൻഡിന്റെയോ, എതിരാളികളുടെ സൈറ്റുകളുടെയോ, അല്ലെങ്കിൽ ഡിസൈൻ പ്രചോദനത്തിന്റെയോ ഫോട്ടോകൾ അപ്‌ലോഡ് ചെയ്യുന്നത് Theme Builder-ന് നിങ്ങളുടെ ഡിസൈൻ ദിശ മനസ്സിലാക്കാൻ സഹായിക്കും.

വിശദമായ ഗൈഡൻസിനായി [Discovery Interview](./theme-builder/discovery-interview.md) കാണുക.

### ഡെസ്ക്ടോപ്പ്, മൊബൈൽ പ്രിവ്യൂ സഹിതം ഡിസൈൻ ദിശ

Theme Builder നിങ്ങളുടെ പൂർണ്ണമായ തീം ജനറേറ്റ് ചെയ്യുന്നതിന് മുമ്പ് ഡെസ്ക്ടോപ്പ്, മൊബൈൽ ഉപകരണങ്ങളിൽ നിങ്ങളുടെ ഡിസൈൻ അവലോകനം ചെയ്യുക. ഒന്നിലധികം റെസ്‌പോൺസീവ് ബ്രേക്ക്പോയിന്റുകളിൽ തത്സമയ പ്രിവ്യൂകൾ കാണുക.

വിശദമായ ഗൈഡൻസിനായി [Design Direction](./theme-builder/design-direction.md) കാണുക.

### Hospitality Menus

Theme Builder-ന് ഇപ്പോൾ റെസ്റ്റോറന്റുകൾ, കഫേകൾ, ബാറുകൾ, മറ്റ് ഹോസ്പിറ്റാലിറ്റി ബിസിനസ്സുകൾ എന്നിവയ്ക്കായി ഘടനാപരമായ ഭക്ഷണ, പാനീയ മെനു പേജുകൾ ജനറേറ്റ് ചെയ്യാൻ കഴിയും. മെനുകളിൽ വിലകൾ, വിവരണങ്ങൾ, ഭക്ഷണക്രമ വിവരങ്ങൾ, അലർജൻ മുന്നറിയിപ്പുകൾ എന്നിവ ഉൾപ്പെടുന്നു.

വിശദമായ ഗൈഡൻസിനായി [Hospitality Menus](./theme-builder/hospitality-menus.md) കാണുക.

### പുതിയ കഴിവുകൾ (New Abilities)

- **[Generate Logo SVG](./abilities/generate-logo-svg.md)** — ഓട്ടോമാറ്റിക് sanitisation സഹിതം കസ്റ്റം SVG ലോഗോകൾ സൃഷ്ടിക്കുക.
- **[Validate Palette Contrast](./abilities/validate-palette-contrast.md)** — WCAG പാലിക്കാനുള്ള നിറങ്ങളുടെ ജോഡികൾ പരിശോധിക്കുക.
- **[Create Menu](./abilities/create-menu.md)** — വ്യക്തമായ നാവിഗേഷൻ ലേബലുകളോടുകൂടിയ നാവിഗേഷൻ മെനുകൾ സൃഷ്ടിക്കുക.

ലഭ്യമായ ടൂളുകളെക്കുറിച്ചും Tier 1 site-scrape ടൂളിനെക്കുറിച്ചുമുള്ള വിവരങ്ങൾക്കായി [Tools and Configuration](./theme-builder/tools.md) കാണുക.

## കോൺഫിഗറേഷൻ (Configuration)

എല്ലാ സെറ്റിംഗുകളും WordPress അഡ്മിനിൽ **Gratis AI Agent → Settings** എന്നതിൽ ആണ് ഉള്ളത്. Superdav AI Agent താഴെ പറയുന്ന കോൺഫിഗറേഷൻ ഓപ്ഷനുകൾ കൂട്ടിച്ചേർക്കുന്നു:

| Setting | Description |
|---|---|
| **Internet Search Provider** | Brave Search, Tavily എന്നിവയിൽ നിന്ന് തിരഞ്ഞെടുക്കുക (ഓപ്ഷണൽ) |
| **Tavily API Key** | Tavily ഇന്റർനെറ്റ് സെർച്ചിനായുള്ള API key (Tavily തിരഞ്ഞെടുക്കുകയാണെങ്കിൽ) |

## v1.16.0-ലെ മെച്ചപ്പെടുത്തലുകൾ (Improvements in v1.16.0)

- **AI Client cache** — ഇപ്പോൾ ട്രാൻസിയന്റുകൾ (transients) ഉപയോഗിച്ച് ബാക്കപ്പ് ചെയ്യുന്നു, ഇത് ദീർഘദൈർഘ്യമുള്ള ഏജന്റ് ടാസ്ക്കുകളിൽ ഡാറ്റ നഷ്ടപ്പെടുന്നത് തടയുന്നു.
- **Tier 1 site-scrape tool** — എതിരാളികളുടെയും പ്രചോദന സൈറ്റുകളുടെയും വിശകലനത്തിനായി Theme Builder-ൽ sd-ai-agent/site-scrape ഇപ്പോൾ ഡിഫോൾട്ടായി ലഭ്യമാണ്.
- **Plugin row action links** — വ്യക്തതയ്ക്കായി തിരുത്തുകയും പേര് മാറ്റുകയും ചെയ്തു.

## v1.10.0-ലെ മെച്ചപ്പെടുത്തലുകൾ (Improvements in v1.10.0)

- **WooCommerce integration refactored** — മികച്ച വിശ്വാസ്യതയ്ക്കും അനുയോജ്യതയ്ക്കുമായി ഇപ്പോൾ native WooCommerce APIs ഉപയോഗിക്കുന്നു.
- **Provider list auto-refresh** — പ്ലഗിനുകൾ ആക്ടിവേറ്റ് ചെയ്യുമ്പോഴോ ഡിആക്ടിവേറ്റ് ചെയ്യുമ്പോഴോ ലഭ്യമായ പ്രൊവൈഡർ ലിസ്റ്റ് സ്വയമേവ കണ്ടെത്തി അപ്‌ഡേറ്റ് ചെയ്യുന്നു.
- **Enhanced site builder** — മികച്ച എറർ റിക്കവറിയോടുകൂടിയ മെച്ചപ്പെടുത്തിയ മൾട്ടി-ഫേസ് സൈറ്റ് ബിൽഡ് പ്ലാനുകൾ.
- **Theme-aware skills** — നിങ്ങളുടെ സജീവമായ തീമുമായി സ്വയമേവ ക്രമീകരിക്കുന്ന നാല് പുതിയ ബിൽറ്റ്-ഇൻ കഴിവുകൾ.

## v1.10.0-ലെ ബഗ് ഫിക്സുകൾ (Bug Fixes in v1.10.0)

- ചില അഡ്മിൻ പേജുകളിൽ അനന്തമായ റീലോഡ് ലൂപ്പുകൾ ഉണ്ടാക്കിയിരുന്ന `navigate-to` കഴിവ് പരിഹരിച്ചു.
- കാറ്റഗറി, ടാഗ് പേരുകൾ സ്ലഗ്‌കളായി ശരിയായി പരിഹരിക്കാത്ത `list-posts` കഴിവ് പരിഹരിച്ചു.
- മുൻ refactor-ന് ശേഷം WP-CLI കമാൻഡുകൾക്ക് namespace aliases നഷ്ടപ്പെട്ടിരുന്നത് പരിഹരിച്ചു.
- ഓട്ടോമേഷൻ ടേബിളുകൾ ഇതുവരെ സൃഷ്ടിക്കാത്ത സൈറ്റുകൾ കൈകാര്യം ചെയ്യുന്ന രീതിയിൽ ഇവന്റ് ഓട്ടോമേഷൻ പരിഹരിച്ചു.
- സിസ്റ്റം ഇൻസ്ട്രക്ഷൻ ബിൽഡറിൽ ശരിയായ namespace prefix ഉപയോഗിക്കുന്ന `memory-save` കഴിവ് പരിഹരിച്ചു.
- AI-ക്ക് തിരികെ നൽകുന്നതിന് മുമ്പ് സ്കെയിലർ ടൂൾ റിസൾട്ടുകൾ ശരിയായി റാപ്പ് ചെയ്യാനുള്ള പ്രശ്നം പരിഹരിച്ചു.
- പഴയ പതിപ്പുകളിൽ നിന്ന് അപ്‌ഗ്രേഡ് ചെയ്യുമ്പോൾ ലെഗസി കഴിവ് കീ ഫോർമാറ്റ് ശരിയായി കൈകാര്യം ചെയ്യുന്ന രീതിയിൽ യൂസേജ് സ്റ്റാറ്റിസ്റ്റിക്സ് പരിഹരിച്ചു.
