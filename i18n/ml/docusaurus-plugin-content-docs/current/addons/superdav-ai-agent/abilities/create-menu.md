# മെനു സൃഷ്ടിക്കൽ (Create Menu) {#create-menu}

**Create Menu** എന്ന ഈ സംവിധാനം, പേജ് ടൈറ്റിലുകളിൽ നിന്ന് വേറിട്ട നാവിഗേഷൻ ലേബലുകൾ സപ്പോർട്ട് ചെയ്യുന്ന WordPress നാവിഗേഷൻ മെനുകൾ സൃഷ്ടിക്കാൻ സഹായിക്കുന്നു.

## Overview (വിവരം) {#overview}

ഈ സംവിധാനം സാധാരണ മെനു നിർമ്മാണ പ്രവർത്തനത്തെ വികസിപ്പിക്കുകയാണ് ചെയ്യുന്നത്. ഇതിലൂടെ `navigation_label` എന്ന ഒരു പാരമീറ്റർ നൽകാൻ സാധിക്കുന്നു. ഇത് വഴി, നാവിഗേഷനിൽ കാണിക്കുന്ന ലേബൽ പേജ് ടൈറ്റിലിൽ നിന്ന് വ്യത്യസ്തമായിരിക്കാൻ കഴിയുന്ന മെനുകൾ നിങ്ങൾക്ക് സൃഷ്ടിക്കാൻ കഴിയും. ഇത് സൈറ്റ് ഘടനയിലും യൂസർ എക്സ്പീരിയൻസിലും കൂടുതൽ വഴക്കം നൽകുന്നു.

## Parameters (പാരമീറ്ററുകൾ) {#parameters}

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | string | Yes | മെനുവിന്റെ പേര്, ഉദാഹരണത്തിന്: `Primary Navigation` |
| `location` | string | No | ഈ മെനു ഏത് തീം ലൊക്കേഷനിൽ (Theme location) നൽകണം എന്ന് നിർവചിക്കുന്നു, ഉദാഹരണത്തിന്: `primary` |
| `navigation_label` | string | No | നാവിഗേഷനിൽ കാണിക്കാനുള്ള ലേബൽ (പേജ് ടൈറ്റിലിൽ നിന്ന് വ്യത്യസ്തമായി) |

## Return Value (തിരിച്ചറിയുന്ന മൂല്യം) {#return-value}

```json
{
  "success": true,
  "menu_id": 7,
  "name": "Primary Navigation",
  "navigation_label": "Main Menu"
}
```

## Navigation Label vs Page Title (നാവിഗേഷൻ ലേബലും പേജ് ടൈറ്റിലും തമ്മിലുള്ള വ്യത്യാസം) {#navigation-label-vs-page-title}

`navigation_label` പാരമീറ്റർ ഉപയോഗിച്ച്, ആന്തരിക മെനുവിന്റെ പേരും (internal menu name) ഉപയോക്താക്കൾക്ക് കാണിക്കുന്ന ലേബലും വേർതിരിക്കാൻ സാധിക്കുന്നു:

- **`name`** — WordPress ഉപയോഗിക്കുന്ന ആന്തരിക മെനു ഐഡന്റിഫയർ (ഉദാഹരണത്തിന്: "Primary Navigation")
- **`navigation_label`** — സൈറ്റ് സന്ദർശകർക്ക് നാവിഗേഷനിൽ കാണിക്കുന്ന ലേബൽ (ഉദാഹരണത്തിന്: "Main Menu")

ഇത് എപ്പോഴാണ് ഉപകാരപ്രദമാകുന്നത്:
- നിങ്ങളുടെ ആന്തരിക നാമകരണ രീതി (internal naming convention) ഉപയോക്താക്കൾ കാണുന്ന ലേബലുകളിൽ നിന്ന് വ്യത്യസ്തമാണെങ്കിൽ
- അഡ്മിൻ പാനലിൽ കാണുന്നതിനേക്കാൾ ചെറു ലേബലുകൾ നാവിഗേഷനിൽ വേണമെങ്കിൽ
- വ്യത്യസ്ത ലേബൽ നീളങ്ങൾ ഉപയോഗിച്ച് ഒന്നിലധികം ഭാഷകൾ സപ്പോർട്ട് ചെയ്യേണ്ടതുണ്ടെങ്കിൽ
- പ്രത്യേക പ്രദേശങ്ങൾക്കോ ഉപയോക്തൃ ഗ്രൂപ്പുകൾക്കോ വേണ്ടി മെനുകൾ നിർമ്മിക്കുമ്പോൾ

## Usage Examples (ഉപയോഗ ഉദാഹരണങ്ങൾ) {#usage-examples}

### Example 1: Simple Menu with Navigation Label (നാവിഗേഷൻ ലേബലടങ്ങിയ ലളിതമായ മെനു) {#example-1-simple-menu-with-navigation-label}

**Prompt (ആവശ്യം):**
```
Create a primary navigation menu called "Main Navigation" with the navigation label "Menu".
```

**Result (ഫലം):**
```json
{
  "success": true,
  "menu_id": 7,
  "name": "Main Navigation",
  "navigation_label": "Menu"
}
```

### Example 2: Menu for Specific Theme Location (ഒരു പ്രത്യേക തീം ലൊക്കേഷനുള്ള മെനു) {#example-2-menu-for-specific-theme-location}

**Prompt (ആവശ്യം):**
```
Create a footer menu called "Footer Links" with navigation label "Quick Links" and assign it to the footer location.
```

**Result (ഫലം):**
```json
{
  "success": true,
  "menu_id": 8,
  "name": "Footer Links",
  "navigation_label": "Quick Links",
  "location": "footer"
}
```

## Integration with Theme Builder (തീം ബിൽഡറുമായുള്ള സംയോജനം) {#integration-with-theme-builder}

Theme Builder ഉപയോഗിക്കുമ്പോൾ, Create Menu സംവിധാനം:

1. ലഭ്യമായ തീം മെനു ലൊക്കേഷനുകൾ സ്വയമേവ കണ്ടെത്തുന്നു
2. നിങ്ങളുടെ ഡിസൈനിന് അനുയോജ്യമായ നാവിഗേഷൻ ലേബലുകളോടെ മെനുകൾ സൃഷ്ടിക്കുന്നു
3. മെനുകളെ ശരിയായ തീം ലൊക്കേഷനുകളിലേക്ക് നൽകുന്നു
4. സൃഷ്ടിച്ചതിന് ശേഷം മെനു ഇനങ്ങൾ ചേർക്കാൻ പിന്തുണ നൽകുന്നു

## Related Abilities (ബന്ധപ്പെട്ട സംവിധാനങ്ങൾ) {#related-abilities}

- **`add_menu_item`** — നിലവിലുള്ള മെനുവിൽ ഇനങ്ങൾ ചേർക്കാൻ
- **`update_menu`** — ഒരു മെനുവിന്റെ പേര് മാറ്റാനോ അല്ലെങ്കിൽ ഒരു തീം ലൊക്കേഷനിലേക്ക് വീണ്ടും നൽകാനോ
- **`delete_menu`** — നിങ്ങളുടെ സൈറ്റിൽ നിന്ന് ഒരു മെനു നീക്കം ചെയ്യാൻ

## Best Practices (മികച്ച രീതികൾ) {#best-practices}

- **വ്യക്തമായ നാവിഗേഷൻ ലേബലുകൾ ഉപയോഗിക്കുക** — ലേബലുകൾ സംക്ഷിപ്തവും വിവരണാത്മകവുമാക്കുക.
- **തീം ലൊക്കേഷനുകൾ പൊരുത്തപ്പെടുത്തുക** — ശരിയായ ഡിസ്പ്ലേയ്ക്കായി മെനുകൾ ശരിയായ തീം ലൊക്കേഷനിലേക്ക് നൽകുക.
- **മെനു ഘടന ആസൂത്രണം ചെയ്യുക** — ഇനങ്ങൾ സൃഷ്ടിക്കുന്നതിന് മുമ്പ് നിങ്ങളുടെ മെനു ഹൈറാർക്കി (hierarchy) തീരുമാനിക്കുക.
- **റെസ്‌പോൺസീവ്‌നെസ് ടെസ്റ്റ് ചെയ്യുക** — മൊബൈൽ ഉപകരണങ്ങളിൽ മെനു ശരിയായി കാണപ്പെടുന്നുണ്ടോ എന്ന് പരിശോധിക്കുക.
- **ലേബലുകൾ പ്രാദേശികവൽക്കരിക്കുക** — വ്യത്യസ്ത ഭാഷാ പതിപ്പുകൾക്കായി വ്യത്യസ്ത നാവിഗേഷൻ ലേബലുകൾ ഉപയോഗിക്കുക.

## Limitations (പരിമിതികൾ) {#limitations}

- നാവിഗേഷൻ ലേബലുകൾ കാണിക്കാനുള്ളവ മാത്രമാണ്; ആന്തരിക `name` ഇപ്പോഴും WordPress തിരിച്ചറിയലിനായി ഉപയോഗിക്കുന്നു.
- തീം സപ്പോർട്ട് വ്യത്യാസപ്പെട്ടിരിക്കും; എല്ലാ തീമുകളും എല്ലാ മെനു ലൊക്കേഷനുകളും സപ്പോർട്ട് ചെയ്യുന്നില്ല.
- മെനു സൃഷ്ടിച്ചതിന് ശേഷം മെനു ഇനങ്ങൾ പ്രത്യേകമായി ചേർക്കേണ്ടതുണ്ട്.
- ഒരു നാവിഗേഷൻ ലേബൽ മാറ്റണമെങ്കിൽ മെനു അപ്ഡേറ്റ് ചെയ്യേണ്ടതുണ്ട്.
