# ഡിസ്കവറി ഇന്റർവ്യൂ

**ഡിസ്കവറി ഇന്റർവ്യൂ** എന്നത് Theme Builder-ന്റെ ഡിസൈൻ പ്രക്രിയയിലെ ആദ്യ പടിയാണ്. നിങ്ങളുടെ സൈറ്റിനെക്കുറിച്ചും ബിസിനസിനെക്കുറിച്ചും ഡിസൈൻ ഇഷ്ടങ്ങളെക്കുറിച്ചുമുള്ള വിവരങ്ങൾ ഇത് ശേഖരിക്കുന്നു. ഇത് ഉപയോഗിച്ച് ഒരു വ്യക്തിഗതമാക്കിയ തീം (personalized theme) സൃഷ്ടിക്കാൻ സാധിക്കും.

## Overview

നിങ്ങളുടെ ബ്രാൻഡ്, ടാർഗെറ്റ് ഓഡിയൻസ്, ഡിസൈൻ ലക്ഷ്യങ്ങൾ എന്നിവ മനസ്സിലാക്കാൻ സഹായിക്കുന്ന ചോദ്യങ്ങളുടെ ഒരു പരമ്പരയിലൂടെയാണ് ഡിസ്കവറി ഇന്റർവ്യൂ നിങ്ങളെ നയിക്കുന്നത്. v1.16.0 പതിപ്പിൽ, ഈ ഇന്റർവ്യൂവിൽ **ഫോട്ടോ അപ്‌ലോഡ് സ്റ്റെപ്പ്** ഉൾപ്പെടുത്തിയിട്ടുണ്ട്. ഇത് നിങ്ങളുടെ ഡിസൈനിനായുള്ള വിഷ്വൽ പ്രചോദനം (visual inspiration) നൽകാൻ നിങ്ങളെ അനുവദിക്കുന്നു.

## Interview Steps

### 1. Business Information

- സൈറ്റ് പേരും ടാഗ് ലൈനും
- ബിസിനസ് തരം അല്ലെങ്കിൽ വ്യവസായം
- പ്രധാന ബിസിനസ് ലക്ഷ്യങ്ങൾ
- ടാർഗെറ്റ് ഓഡിയൻസ് വിവരണം

### 2. Design Preferences

- ഇഷ്ടപ്പെട്ട കളർ സ്കീം
- ഡിസൈൻ ശൈലി (മോഡേൺ, ക്ലാസിക്, മിനിമലിസ്റ്റ്, കളിയായ, തുടങ്ങിയവ)
- പ്രചോദന വെബ്സൈറ്റുകളോ ബ്രാൻഡുകളോ
- നിർബന്ധമായും വേണ്ട ഡിസൈൻ ഘടകങ്ങൾ

### 3. Photo Upload (v1.16.0-ൽ പുതിയത്)

ഡിസൈൻ ദിശയെ സ്വാധീനിക്കാൻ ഒന്നോ അതിലധികമോ ഫോട്ടോകൾ അപ്‌ലോഡ് ചെയ്യുക:

#### Accepted Formats

| Format | Extension | Details |
|--------|-----------|---------|
| **JPEG** | `.jpg`, `.jpeg` | ഫോട്ടോഗ്രാഫുകൾക്ക് ശുപാർശ ചെയ്യുന്നത് |
| **PNG** | `.png` | ട്രാൻസ്പരൻസി (transparency) പിന്തുണയ്ക്കുന്നു |
| **WebP** | `.webp` | ആധുനിക ഫോർമാറ്റ്, ചെറിയ ഫയൽ സൈസ് |
| **GIF** | `.gif` | സ്റ്റാറ്റിക് ചിത്രങ്ങൾ മാത്രം (ആനിമേഷൻ ഇല്ല) |

#### File Requirements

- **Maximum file size:** 10 MB
- **Minimum dimensions:** 400 × 300 pixels
- **Recommended dimensions:** 1200 × 800 pixels അല്ലെങ്കിൽ അതിൽ കൂടുതൽ
- **Aspect ratio:** ഏത് അനുപാതവും സ്വീകാര്യമാണ്

#### How Photos Influence Design Direction

നിങ്ങൾ അപ്‌ലോഡ് ചെയ്യുന്ന ഫോട്ടോകൾ Theme Builder-ന് മനസ്സിലാക്കാൻ സഹായിക്കുന്നു:

- **Color palette** — പ്രധാന നിറങ്ങൾ വേർതിരിച്ചെടുക്കുകയും തീമിൽ ഉപയോഗിക്കുകയും ചെയ്യുന്നു
- **Visual style** — മോഡേൺ, റസ്റ്റിക്, മിനിമലിസ്റ്റ്, വൈബ്രന്റ്, തുടങ്ങിയവ
- **Mood and atmosphere** — പ്രൊഫഷണൽ, ക്രിയേറ്റീവ്, ഫ്രണ്ട്ലി, ലക്ഷ്വറി തുടങ്ങിയവ
- **Design elements** — പാറ്റേണുകൾ, ടെക്സ്ചറുകൾ, ഘടനാപരമായ ശൈലികൾ
- **Typography inspiration** — വിഷ്വൽ തീമിന് അനുയോജ്യമായ ഫോണ്ട് ശൈലികൾ

### 4. Content Structure

- ആവശ്യമായ പ്രധാന പേജുകൾ (Home, About, Services, Contact, തുടങ്ങിയവ)
- ബ്ലോഗോ വാർത്താ വിഭാഗം ആവശ്യമുണ്ടോ
- ഇ-കൊമേഴ്‌സ് പ്രവർത്തനങ്ങൾ ആവശ്യമുണ്ടോ
- പ്രത്യേക ഫീച്ചറുകളോ ഇന്റഗ്രേഷനുകളോ

### 5. Review and Confirmation

- നൽകിയ എല്ലാ വിവരങ്ങളുടെയും സംഗ്രഹം
- ഡിസൈൻ ജനറേഷൻ ആരംഭിക്കുന്നതിന് മുമ്പുള്ള സ്ഥിരീകരണം

## Photo Upload Best Practices

### What to Upload

- **Brand inspiration** — ലോഗോകൾ, കളർ സ്വാച്ചുകൾ, അല്ലെങ്കിൽ ബ്രാൻഡ് ഗൈഡ്‌ലൈനുകൾ
- **Competitor sites** — നിങ്ങൾക്ക് ഇഷ്ടപ്പെട്ട സൈറ്റുകളുടെ സ്ക്രീൻഷോട്ടുകൾ
- **Mood boards** — നിങ്ങൾ ആഗ്രഹിക്കുന്ന സൗന്ദര്യം പ്രതിനിധീകരിക്കുന്ന ചിത്രങ്ങളുടെ ശേഖരം
- **Product photos** — പ്രദർശിപ്പിക്കാനുള്ള ഉൽപ്പന്നങ്ങളുണ്ടെങ്കിൽ
- **Team or office photos** — വ്യക്തിഗത അല്ലെങ്കിൽ പ്രൊഫഷണൽ ബ്രാൻഡിംഗിനായി
- **Lifestyle images** — നിങ്ങളുടെ ടാർഗെറ്റ് ഓഡിയൻസിനെ പ്രതിനിധീകരിക്കുന്ന ചിത്രങ്ങൾ

### What to Avoid

- **Low-quality images** — മങ്ങിക്കിടക്കുന്നതോ പിക്സലേറ്റഡ് ആയതോ ആയ ഫോട്ടോകൾ
- **Unrelated images** — നിങ്ങളുടെ ബ്രാൻഡിനെ പ്രതിനിധീകരിക്കുന്നില്ലാത്ത ചിത്രങ്ങൾ
- **Text-heavy images** — ധാരാളം ടെക്സ്റ്റ് ഉള്ള സ്ക്രീൻഷോട്ടുകൾ
- **Copyrighted material** — ചിത്രങ്ങൾ ഉപയോഗിക്കാൻ നിങ്ങൾക്ക് അവകാശമുണ്ടെന്ന് ഉറപ്പാക്കുക
- **Animated GIFs** — സ്റ്റാറ്റിക് ചിത്രങ്ങൾ മാത്രമേ പ്രോസസ്സ് ചെയ്യുകയുള്ളൂ

### Tips for Best Results

1. **Upload 2-3 images** — ഒന്നിലധികം ചിത്രങ്ങൾ മികച്ച സന്ദർഭം നൽകുന്നു
2. **Use high-quality photos** — വ്യക്തവും നന്നായി പ്രകാശമുള്ളതുമായ ചിത്രങ്ങൾ മികച്ചതാണ്
3. **Include variety** — നിറം, ശൈലി, വിഷയം എന്നിവയുടെ മിശ്രിതം നൽകുക
4. **Provide context** — ഓരോ ചിത്രവും എന്തിനാണ് തിരഞ്ഞെടുത്തതെന്ന് ചുരുക്കി വിവരിക്കുക
5. **Be specific** — ഒരു പ്രത്യേക ഘടകം നിങ്ങൾക്ക് ഇഷ്ടമാണെങ്കിൽ, അത് ചൂണ്ടിക്കാണിക്കുക

## Design Direction Output

ഡിസ്കവറി ഇന്റർവ്യൂവിനു ശേഷം, Theme Builder താഴെ പറയുന്നവ സൃഷ്ടിക്കുന്നു:

- **Color palette** — പ്രാഥമിക, രണ്ടാമത്തെ, ആക്സന്റ് നിറങ്ങൾ
- **Typography** — ഹെഡിംഗുകൾക്കും ബോഡി ടെക്സ്റ്റിനുമുള്ള ഫോണ്ട് കുടുംബങ്ങൾ
- **Layout style** — സ്പേസിംഗ്, അലൈൻമെന്റ്, ഘടകങ്ങളുടെ ക്രമീകരണം
- **Visual elements** — ഐക്കണുകൾ, പാറ്റേണുകൾ, അലങ്കാര ഘടകങ്ങൾ
- **Mood board** — ഡിസൈൻ ദിശയുടെ വിഷ്വൽ പ്രാതിനിധ്യം

## Next Steps

ഡിസ്കവറി ഇന്റർവ്യൂ പൂർത്തിയാക്കിയ ശേഷം:

1. സൃഷ്ടിക്കപ്പെട്ട ഡിസൈൻ ദിശ അവലോകനം ചെയ്യുക
2. ഫീഡ്‌ബാക്ക് നൽകുക അല്ലെങ്കിൽ മാറ്റങ്ങൾ ആവശ്യപ്പെടുക
3. [Design Direction](./design-direction.md) സ്റ്റെപ്പിലേക്ക് പോകുക
4. Theme Builder നിങ്ങളുടെ കസ്റ്റം തീം സൃഷ്ടിക്കുന്നു
