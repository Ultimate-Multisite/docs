# ചക്ക് നോറിസ് ഫാക്ട്സ് മാറ്റങ്ങൾ

### 1.0.1 - 2025-11-03
* കൂട്ടിച്ചേർത്തു: Ultimate Multisite update server വഴി ഓട്ടോമാറ്റിക് അപ്ഡേറ്റ് പ്രവർത്തനം.
* അപ്ഡേറ്റ് ചെയ്തു: WordPress 6.6 വരെ ടെസ്റ്റ് ചെയ്തു.
* അപ്ഡേറ്റ് ചെയ്തു: composer autoloader ഉപയോഗിച്ച് plugin ഘടന മെച്ചപ്പെടുത്തി.
* അപ്ഡേറ്റ് ചെയ്തു: ആധുനിക WordPress ഇൻസ്റ്റാളേഷനുകളുമായി അനുയോജ്യത വർദ്ധിപ്പിച്ചു.

### 1.0.0 - 2024-10-21
* ആദ്യ റിലീസ്.
* 110-ൽ അധികം ചക്ക് നോറിസ് ഫാക്ട്സ്.
* പൂർണ്ണമായ വിവർത്തന പിന്തുണ.
* എല്ലാ സ്ക്രീൻ സൈസുകൾക്കുമുള്ള പ്രതികരണശേഷിയുള്ള ഡിസൈൻ.
* RTL ഭാഷാ പിന്തുണ.
* ബ്ലോക്ക് എഡിറ്റർ അനുയോജ്യത.
* സ്ക്രീൻ റീഡർ പിന്തുണയോടെ എളുപ്പത്തിൽ ഉപയോഗിക്കാനുള്ള സവിശേഷതകൾ.
