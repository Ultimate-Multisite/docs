# Domain Seller മാറ്റപ്പട്ടിക

Version 1.3.0 - 2026-06-02-ന് റിലീസ് ചെയ്തത്
- പുതിയത്: HostAfrica reseller ബാലൻസ് വളരെ കുറഞ്ഞാൽ network-admin മുന്നറിയിപ്പ് ചേർത്തു
- പുതിയത്: പുതുതായി രജിസ്റ്റർ ചെയ്ത ഡൊമെയ്‌നുകൾ network site-ലേക്ക് സ്വയം മാപ്പ് ചെയ്യൽ ചേർത്തു
- പരിഹാരം: പുതിയ ഡൊമെയ്ൻ രജിസ്റ്റർ ചെയ്യുമ്പോൾ മാത്രം registrant ഫീൽഡ് ആവശ്യകതകൾ പ്രയോഗിച്ചു
- പരിഹാരം: ബാലൻസ് നിരീക്ഷണ അറിയിപ്പുകൾ അടയ്ക്കാവുന്നതാക്കി
- പരിഹാരം: WooCommerce registrant ബില്ലിംഗ് വിശദാംശങ്ങൾ നിലനിർത്തുന്നുവെന്ന് ഉറപ്പാക്കി
- പരിഹാരം: രജിസ്ട്രേഷൻ സമയത്ത് registrant കോൺടാക്റ്റ് ആവശ്യകതകൾ നിർബന്ധമാക്കി
- പരിഹാരം: ഡൊമെയ്ൻ രജിസ്ട്രേഷൻ ഉൽപ്പന്നങ്ങൾ 0% markup-ോടെ സൃഷ്ടിക്കപ്പെടുന്നത് തടഞ്ഞു
- പരിഹാരം: checkout സെഷൻ പ്രവാഹത്തിലുടനീളം ഡൊമെയ്ൻ തിരഞ്ഞെടുപ്പുകളും വിലനിർണ്ണയവും നിലനിർത്തി
- പരിഹാരം: HostAfrica ഡൊമെയ്ൻ വിലനിർണ്ണയ കറൻസി പ്രദർശനം മെച്ചപ്പെടുത്തി
- പരിഹാരം: WP-core query-var പൊരുത്തക്കേടുകൾ തടയാൻ checkout form-action പെരുമാറ്റം മെച്ചപ്പെടുത്തി
- മെച്ചപ്പെടുത്തി: setup മാർഗനിർദേശത്തിൽ HostAfrica reseller കോൺഫിഗറേഷൻ ഡോക്യുമെന്റേഷൻ ലിങ്ക് ചെയ്തു
