# ഡൊമെയിൻ വിൽപ്പനക്കാരൻ

**Multisite Ultimate: ഡൊമെയിൻ വിൽപ്പനക്കാരൻ** നിങ്ങളുടെ WordPress Multisite നെറ്റ്വർക്കിനെ ശക്തമായ ഡൊമെയിൻ വിൽപ്പന പ്ലാറ്റ്ഫോമാക്കി മാറ്റുന്നു. സൈൻ-അപ്പിനിടെ അല്ലെങ്കിൽ അവരുടെ Dashboard-ൽ നിന്ന് തന്നെ ഉപഭോക്താക്കൾക്ക് ഇഷ്ടാനുസൃത ഡൊമെയിനുകൾ രജിസ്റ്റർ ചെയ്യാൻ അനുവദിക്കുക. തത്സമയ ലഭ്യതാ പരിശോധനയും സുതാര്യമായ വിലനിർണ്ണയവും ലഭിക്കും.

### ഡൊമെയിൻ വിൽപ്പനക്കാരൻ എന്തിന്? {#why-domain-seller}

ഏത് WaaS (Website as a Service) പ്ലാറ്റ്ഫോമിനും ഏറ്റവും കൂടുതൽ ആവശ്യപ്പെടുന്ന അധിക വിൽപ്പനകളിൽ ഒന്നാണ് ഡൊമെയിൻ രജിസ്ട്രേഷൻ. ഉപഭോക്താക്കളെ മൂന്നാം കക്ഷി രജിസ്ട്രാറിലേക്ക് അയയ്ക്കുന്നതിനു പകരം, നിങ്ങൾക്ക് നേരിട്ട് ഡൊമെയിനുകൾ നൽകാം — ഇതിലൂടെ നിങ്ങളുടെ നെറ്റ്വർക്കിൽ കൂടുതൽ വരുമാനം നിലനിർത്തുകയും ഉപഭോക്താക്കൾക്ക് തടസ്സമില്ലാത്ത അനുഭവം നൽകുകയും ചെയ്യാം.

### പ്രധാന സവിശേഷതകൾ {#key-features}

#### നിരവധി ഡൊമെയിൻ രജിസ്ട്രാർ പിന്തുണ {#multiple-domain-registrar-support}
* **OpenSRS** — പൂർണ്ണ രജിസ്ട്രേഷൻ, പുതുക്കൽ, മാനേജ്മെന്റ്
* **Namecheap** — WhoisGuard, സാൻഡ്ബോക്സ് ടെസ്റ്റിംഗ് എന്നിവയോടെയുള്ള പൂർണ്ണ സംയോജനം
* **HostAfrica** — Domains Reseller API വഴി ഡൊമെയിൻ തിരച്ചിൽ, വിലനിർണ്ണയം, രജിസ്ട്രേഷൻ, കൈമാറ്റം, DNS, ID സംരക്ഷണം
* **Openprovider** — റീസെല്ലർ വിലനിർണ്ണയം, രജിസ്ട്രേഷൻ, പുതുക്കൽ, കൈമാറ്റങ്ങൾ, DNS, WHOIS സ്വകാര്യത, TLD സിങ്ക്
* **Hostinger** — പങ്കിട്ട Hostinger സംയോജന ടോക്കൺ ഉപയോഗിച്ച് ഡൊമെയിൻ തിരച്ചിലും രജിസ്ട്രേഷനും
* **വിപുലീകരിക്കാവുന്ന ഘടന** — Integration Registry പാറ്റേണിൽ നിർമ്മിച്ചതിനാൽ കൂടുതൽ പ്രൊവൈഡർമാരെ ചേർക്കുന്നത് എളുപ്പമാണ്
* **അന്തർനിർമ്മിത കണക്ഷൻ ടെസ്റ്റിംഗ്** — ലൈവിലേക്ക് പോകുന്നതിന് മുമ്പ് പ്രൊവൈഡർ ക്രെഡൻഷ്യലുകളും കണക്റ്റിവിറ്റിയും സാധൂകരിക്കുക

#### മാർഗനിർദ്ദേശമുള്ള സജ്ജീകരണ വിസാർഡ് {#guided-setup-wizard}
* **ഒറ്റ ക്ലിക്കിൽ ആരംഭിക്കൽ** — പ്രൊവൈഡർ കോൺഫിഗറേഷൻ, TLD ഇറക്കുമതി, ഡൊമെയിൻ ഉൽപ്പന്ന സൃഷ്ടി എന്നിവയിലൂടെ നിങ്ങളെ നയിക്കുന്നു
* **സ്വയമേവയുള്ള TLD സിങ്ക്** — ബന്ധിപ്പിച്ച പ്രൊവൈഡർമാരിൽ നിന്ന് ലഭ്യമായ എല്ലാ TLD-കളും ഹോൾസെയിൽ വിലകളും ഇറക്കുമതി ചെയ്യുന്നു
* **ഡീഫോൾട്ട് ഉൽപ്പന്ന സൃഷ്ടി** — യുക്തമായ ഡീഫോൾട്ട് മാർക്കപ്പുകളോടെ ഉപയോഗിക്കാൻ തയ്യാറായ ഡൊമെയിൻ ഉൽപ്പന്നം സൃഷ്ടിക്കുന്നു

#### ഇഷ്ടാനുസൃത ഡൊമെയിൻ വിലനിർണ്ണയം {#flexible-domain-pricing}
* **മൂന്ന് മാർക്കപ്പ് മോഡുകൾ** — ശതമാന മാർക്കപ്പ്, നിശ്ചിത മാർക്കപ്പ്, അല്ലെങ്കിൽ ഓരോ TLD-ക്കും നിശ്ചിത വില
* **TLD-വിശിഷ്ട ഉൽപ്പന്നങ്ങൾ** — വ്യത്യസ്ത TLD ഗ്രൂപ്പുകൾക്കായി (.com/.net, .io, രാജ്യ കോഡുകൾ മുതലായവ) വേർതിരിച്ച ഡൊമെയിൻ ഉൽപ്പന്നങ്ങൾ സൃഷ്ടിക്കുക
* **ആരംഭകാല വിലനിർണ്ണയം** — പുതിയ രജിസ്ട്രേഷനുകൾ ആകർഷിക്കാൻ കുറഞ്ഞ ആദ്യ വർഷ വില നൽകുക
* **തത്സമയ വിലനിർണ്ണയം** — ബുദ്ധിപരമായ കാഷിംഗോടെ പ്രൊവൈഡർമാരിൽ നിന്ന് ലൈവ് ഹോൾസെയിൽ വില
* **കൂപ്പൺ പിന്തുണ** — Multisite Ultimate ഡിസ്കൗണ്ട് കോഡുകളുമായി പൂർണ്ണ അനുയോജ്യത

#### WHOIS സ്വകാര്യത സംരക്ഷണം {#whois-privacy-protection}
* **ഓരോ ഉൽപ്പന്നത്തിനും കോൺഫിഗറേഷൻ** — WHOIS സ്വകാര്യത Disabled, Always Included, അല്ലെങ്കിൽ Customer Choice ആയി സജ്ജമാക്കുക
* **ഉപഭോക്തൃ തിരഞ്ഞെടുപ്പ്** — Customer Choice ആയി സജ്ജമാക്കിയാൽ, checkout ചെക്ക്ബോക്സ് വഴി ഉപഭോക്താക്കൾക്ക് സ്വകാര്യത സംരക്ഷണം ചേർക്കാം
* **കോൺഫിഗർ ചെയ്യാവുന്ന വിലനിർണ്ണയം** — സ്വകാര്യത add-on-ന് ഓരോ വർഷത്തേക്കുള്ള വില നിശ്ചയിക്കാം, അല്ലെങ്കിൽ സൗജന്യമായി നൽകാം
* **രജിസ്ട്രാർ സംയോജനം** — രജിസ്ട്രേഷൻ സമയത്ത് WhoisGuard (Namecheap) അല്ലെങ്കിൽ WHOIS സ്വകാര്യത (OpenSRS) സ്വയമേവ പ്രവർത്തനക്ഷമമാക്കുന്നു

#### ഏകീകൃത checkout അനുഭവം {#unified-checkout-experience}
* **മൂന്ന് ഡൊമെയിൻ മോഡുകൾ** — ഉപഭോക്താക്കൾക്ക് subdomain (സൗജന്യം), പുതിയ ഡൊമെയിൻ രജിസ്റ്റർ ചെയ്യൽ, അല്ലെങ്കിൽ ഇതിനകം സ്വന്തമായുള്ള നിലവിലെ ഡൊമെയിൻ mapping എന്നിവയിൽ തിരഞ്ഞെടുക്കാം
* **തത്സമയ ഡൊമെയിൻ തിരച്ചിൽ** — പേജ് റീലോഡ് ഇല്ലാതെ AJAX-അധിഷ്ഠിത ലഭ്യതാ പരിശോധന
* **സ്മാർട്ട് നിർദ്ദേശങ്ങൾ** — ഇഷ്ടപ്പെട്ട ഡൊമെയിൻ ലഭ്യമല്ലെങ്കിൽ പകരം TLD നിർദ്ദേശങ്ങൾ
* **സുതാര്യമായ വിലനിർണ്ണയം** — രജിസ്ട്രേഷൻ വില, പുതുക്കൽ വില, ഐച്ഛിക WHOIS സ്വകാര്യത ഫീസ് എന്നിവയുടെ വ്യക്തമായ പ്രദർശനം
* **രജിസ്ട്രന്റ് കോൺടാക്റ്റ് ഫീൽഡുകൾ** — checkout സമയത്ത് ആവശ്യമായ രജിസ്ട്രന്റ് വിവരങ്ങൾ (പേര്, വിലാസം, ഫോൺ) ഇൻലൈൻ ആയി ശേഖരിക്കുന്നു
* **സ്വയം സൃഷ്ടിക്കുന്ന site URL** — തിരഞ്ഞെടുത്ത ഡൊമെയിനിൽ നിന്ന് site വിലാസം സ്വയമേവ നിർണ്ണയിക്കുന്നു

#### സ്വയമേവ പുതുക്കൽ സിസ്റ്റം {#auto-renewal-system}
* **membership-ബന്ധിത പുതുക്കലുകൾ** — ഉപഭോക്താവിന്റെ membership സജീവമായിരിക്കുമ്പോൾ ഡൊമെയിനുകൾ സ്വയമേവ പുതുക്കും
* **സ്മാർട്ട് ഷെഡ്യൂളിംഗ്** — ഡൊമെയിൻ കാലഹരണ തീയതികളെ അടിസ്ഥാനമാക്കി പുതുക്കൽ ജോലികൾ ക്യൂവിൽ ചേർക്കുന്നു
* **വീണ്ടും ശ്രമിക്കുന്ന ലോജിക്** — പരാജയപ്പെട്ട പുതുക്കൽ ശ്രമങ്ങൾക്കായി ബുദ്ധിപരമായ വീണ്ടും ശ്രമങ്ങൾ
* **ഇമെയിൽ അറിയിപ്പുകൾ** — വരാനിരിക്കുന്ന പുതുക്കലുകൾ, വിജയകരമായ പുതുക്കലുകൾ, പരാജയങ്ങൾ, കാലഹരണങ്ങൾ എന്നിവയ്ക്കായി കോൺഫിഗർ ചെയ്യാവുന്ന അലർട്ടുകൾ

#### ഉപഭോക്തൃ DNS മാനേജ്മെന്റ് {#customer-dns-management}
* **സ്വയം സേവന DNS** — ഉപഭോക്താക്കൾക്ക് അവരുടെ Dashboard-ൽ നിന്ന് DNS records (A, AAAA, CNAME, MX, TXT) ചേർക്കാനും തിരുത്താനും ഇല്ലാതാക്കാനും കഴിയും
* **ഡൊമെയിൻ നില** — തത്സമയ രജിസ്ട്രേഷൻ നില, കാലഹരണ തീയതികൾ, nameserver വിവരങ്ങൾ
* **സ്വയമേവയുള്ള ഡൊമെയിൻ mapping** — രജിസ്റ്റർ ചെയ്തതോ നിലവിലുള്ളതോ ആയ ഡൊമെയിനുകൾ ഉപഭോക്താവിന്റെ site-ലേക്ക് സ്വയമേവ map ചെയ്യും

#### Admin ഉപകരണങ്ങൾ {#admin-tools}
* **മാനുവൽ ഡൊമെയിൻ രജിസ്ട്രേഷൻ** — admin പാനലിൽ നിന്ന് ഉപഭോക്താക്കളുടെ പേരിൽ ഡൊമെയിനുകൾ രജിസ്റ്റർ ചെയ്യുക
* **ഡൊമെയിൻ നിരീക്ഷണം** — ഒരു പേജിൽ നിന്ന് എല്ലാ ഉപഭോക്തൃ ഡൊമെയിനുകളും നിലകളും കാലഹരണ തീയതികളും കാണുക
* **പ്രൊവൈഡർ-വിശിഷ്ട ലോഗിംഗ്** — ഓരോ രജിസ്ട്രാറിനും വേർതിരിച്ച log ചാനലുകൾ (ഉദാ., domain-seller-namecheap.log)
* **രജിസ്ട്രേഷൻ വിശദാംശങ്ങൾ** — പ്രധാന ഡൊമെയിൻ edit പേജിൽ രജിസ്ട്രന്റ് വിവരങ്ങളും രജിസ്ട്രേഷൻ ഡാറ്റയും കാണുക

### പിന്തുണയുള്ള പ്രൊവൈഡർമാർ {#supported-providers}

* **OpenSRS** — രജിസ്ട്രേഷൻ, പുതുക്കൽ, DNS മാനേജ്മെന്റ്, WHOIS സ്വകാര്യത എന്നിവയോടെയുള്ള പൂർണ്ണ സംയോജനം
* **Namecheap** — രജിസ്ട്രേഷൻ, പുതുക്കൽ, batch ലഭ്യതാ പരിശോധന, സൗജന്യ WhoisGuard എന്നിവയോടെയുള്ള പൂർണ്ണ സംയോജനം
* **HostAfrica** — രജിസ്ട്രേഷൻ, പുതുക്കൽ, കൈമാറ്റങ്ങൾ, DNS മാനേജ്മെന്റ്, EPP കോഡ് വീണ്ടെടുക്കൽ, രജിസ്ട്രാർ ലോക്ക്, ID സംരക്ഷണം
* **Openprovider** — രജിസ്ട്രേഷൻ, പുതുക്കൽ, കൈമാറ്റങ്ങൾ, DNS മാനേജ്മെന്റ്, EPP കോഡ് വീണ്ടെടുക്കൽ, രജിസ്ട്രാർ ലോക്ക്, WHOIS സ്വകാര്യത, റീസെല്ലർ TLD സിങ്ക്
* **Hostinger** — രജിസ്ട്രേഷൻ, ഡൊമെയിൻ lookup, nameserver അപ്ഡേറ്റുകൾ, രജിസ്ട്രാർ ലോക്ക്, WHOIS സ്വകാര്യത; പുതുക്കലുകൾ Hostinger auto-renew കൈകാര്യം ചെയ്യുന്നു
* **GoDaddy** — രജിസ്ട്രേഷനും പുതുക്കലും
* **ResellerClub** — രജിസ്ട്രേഷൻ, പുതുക്കൽ, DNS മാനേജ്മെന്റ്
* **NameSilo** — രജിസ്ട്രേഷനും പുതുക്കലും
* **Enom** — രജിസ്ട്രേഷൻ, പുതുക്കൽ, DNS മാനേജ്മെന്റ്

### സാങ്കേതിക വിശദാംശങ്ങൾ {#technical-details}

* **ആവശ്യമാണ്:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **ഘടന:** Domain_Selling_Capability interface ഉള്ള Integration Registry പാറ്റേൺ
* **Autoloading:** PSR-4 class mapping ഉള്ള Jetpack autoloader
* **മാനദണ്ഡങ്ങൾ:** WordPress coding standards, strict typing, prepared statements

## ഇൻസ്റ്റലേഷൻ {#installation}

1. **Multisite Ultimate v2.4.12+** ഇൻസ്റ്റാൾ ചെയ്തതും നെറ്റ്‌വർക്ക്-സജീവമാക്കിയതുമാണെന്ന് ഉറപ്പാക്കുക
2. plugin `/wp-content/plugins/ultimate-multisite-domain-seller/`-ലേക്ക് അപ്‌ലോഡ് ചെയ്യുക
3. WordPress Network Admin-യിലെ **Plugins**-ൽ നിന്ന് plugin നെറ്റ്‌വർക്ക്-സജീവമാക്കുക
4. ആദ്യ സജീവമാക്കലിൽ **Setup Wizard** സ്വയം ആരംഭിക്കും — ഈ ഘട്ടങ്ങൾ പിന്തുടരുക:
   * നിങ്ങളുടെ ഡൊമെയ്ൻ രജിസ്ട്രാറുമായി ബന്ധിപ്പിക്കുക
   * കണക്ഷൻ പരിശോധിക്കുക
   * ലഭ്യമായ TLD-കളും ഹോൾസെയിൽ വിലനിർണ്ണയവും ഇറക്കുമതി ചെയ്യുക
   * വില മാർക്കപ്പോടെ നിങ്ങളുടെ ആദ്യ ഡൊമെയ്ൻ ഉൽപ്പന്നം സൃഷ്ടിക്കുക
5. **Ultimate Multisite > Checkout Forms**-ൽ നിങ്ങളുടെ checkout ഫോമിലേക്ക് **Domain Selection** ഫീൽഡ് ചേർക്കുക

## പതിവായി ചോദിക്കുന്ന ചോദ്യങ്ങൾ {#frequently-asked-questions}

### Multisite Ultimate-ന്റെ ഏത് പതിപ്പാണ് ആവശ്യമായത്? {#what-version-of-multisite-ultimate-is-required}

പതിപ്പ് 2.4.12 അല്ലെങ്കിൽ അതിനുമുകളിലുള്ളത്. addon WP Ultimo v1.x-ുമായോ 2.4.12-നു താഴെയുള്ള Ultimate Multisite പതിപ്പുകളുമായോ പൊരുത്തപ്പെടുന്നില്ല.

### ഏത് ഡൊമെയ്ൻ പ്രൊവൈഡറുകളാണ് പിന്തുണയ്ക്കുന്നത്? {#which-domain-providers-are-supported}

ഒൻപത് രജിസ്ട്രാറുകൾ പിന്തുണയ്ക്കുന്നു: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo**, കൂടാതെ **Enom**. രജിസ്ട്രാർ API-കൾ വ്യത്യസ്ത പ്രവർത്തനങ്ങൾ നൽകുന്നതിനാൽ കഴിവുകൾ പ്രൊവൈഡർ അനുസരിച്ച് മാറും; ഓരോ പ്രൊവൈഡറിനെയും കുറിച്ചുള്ള അവലോകനത്തിന് setup ഗൈഡ് കാണുക. addon's Integration Registry ആർക്കിടെക്ചർ ഇഷ്ടാനുസൃത പ്രൊവൈഡറുകൾ ചേർക്കുന്നത് എളുപ്പമാക്കുന്നു — [ഡെവലപ്പർ ഗൈഡ്](/developer/integration-guide/custom-registrar) കാണുക.

### WHOIS സ്വകാര്യതാ പരിരക്ഷ നൽകാനാകുമോ? {#can-i-offer-whois-privacy-protection}

അതെ. ഓരോ ഡൊമെയ്ൻ ഉൽപ്പന്നത്തിനും മൂന്ന് മോഡുകളുള്ള WHOIS Privacy ക്രമീകരണമുണ്ട്: Disabled, Always Included, അല്ലെങ്കിൽ Customer Choice. Customer Choice ആയി സജ്ജമാക്കിയാൽ, ഉപഭോക്താക്കൾ checkout സമയത്ത് വിലയോടുകൂടിയ ഒരു ചെക്ക്ബോക്സ് കാണും (അല്ലെങ്കിൽ Namecheap-ന്റെ WhoisGuard-ന് "Free"). രജിസ്ട്രേഷൻ സമയത്ത് സ്വകാര്യതാ ഫ്ലാഗ് സ്വയം രജിസ്ട്രാറിലേക്ക് കൈമാറും.

### വ്യത്യസ്ത TLD-കൾക്ക് വ്യത്യസ്ത വിലനിർണ്ണയം സജ്ജമാക്കാനാകുമോ? {#can-i-set-different-pricing-for-different-tlds}

അതെ. വ്യത്യസ്ത TLD ഗ്രൂപ്പുകൾക്കായി വേർതിരിച്ച ഡൊമെയ്ൻ ഉൽപ്പന്നങ്ങൾ സൃഷ്ടിക്കുക. ഓരോ ഉൽപ്പന്നത്തിനും സ്വന്തം മാർക്കപ്പ് തരം (ശതമാനം, സ്ഥിരമായ മാർക്കപ്പ്, അല്ലെങ്കിൽ സ്ഥിരവില), പിന്തുണയ്ക്കുന്ന TLD-കൾ, പരിചയപ്പെടുത്തൽ വിലനിർണ്ണയം എന്നിവ ഉണ്ടായിരിക്കാം.

### checkout-ൽ ഉപഭോക്താക്കൾക്ക് എന്തെല്ലാം ഓപ്ഷനുകൾ കാണാം? {#what-checkout-options-do-customers-see}

Domain Selection ഫീൽഡ് ഉപഭോക്താക്കൾക്ക് മൂന്ന് ടാബുകൾ നൽകുന്നു: **Subdomain** (നിങ്ങളുടെ നെറ്റ്‌വർക്കിൽ സൗജന്യ സബ്‌ഡൊമെയ്ൻ ഉപയോഗിക്കുക), **Register New Domain** (പുതിയ ഡൊമെയ്ൻ തിരഞ്ഞ് രജിസ്റ്റർ ചെയ്യുക), കൂടാതെ **Existing Domain** (അവർക്ക് ഇതിനകം ഉള്ള ഡൊമെയ്ൻ മാപ്പ് ചെയ്യുക). നിങ്ങൾ സജീവമാക്കുന്ന മോഡുകൾ മാത്രം പ്രത്യക്ഷപ്പെടും.

### സ്വയമേവയുള്ള പുതുക്കലുകൾ പിന്തുണയ്ക്കുന്നുണ്ടോ? {#does-it-support-automatic-renewals}

അതെ. ഡൊമെയ്‌നുകൾ auto-renew ആയി സജ്ജമാക്കാം, അവ ഉപഭോക്താവിന്റെ അംഗത്വ നിലയുമായി ബന്ധിപ്പിച്ചിരിക്കും. പുതുക്കൽ പേയ്മെന്റുകൾ ഉപഭോക്താവിന്റെ നിലവിലുള്ള പേയ്മെന്റ് ഗേറ്റ്‌വേയിലൂടെ പ്രോസസ്സ് ചെയ്യപ്പെടും.

### ഉപഭോക്താക്കൾക്ക് അവരുടെ DNS റെക്കോർഡുകൾ കൈകാര്യം ചെയ്യാനാകുമോ? {#can-customers-manage-their-dns-records}

അതെ. ഉപഭോക്താക്കൾക്ക് അവരുടെ dashboard-ൽ A, AAAA, CNAME, MX, TXT റെക്കോർഡുകൾ ചേർക്കാനും തിരുത്താനും ഇല്ലാതാക്കാനുമുള്ള DNS മാനേജ്മെന്റ് ഇന്റർഫേസ് ലഭിക്കും. DNS മാനേജ്മെന്റ് OpenSRS, ResellerClub, Enom, HostAfrica, Openprovider എന്നിവയോടൊപ്പം ലഭ്യമാണ്. hosted ഡൊമെയ്‌നുകൾക്കുള്ള Hostinger DNS core Hostinger ഡൊമെയ്ൻ-മാപ്പിംഗ് ഇന്റഗ്രേഷൻ കൈകാര്യം ചെയ്യുന്നു.

### setup wizard ഉണ്ടോ? {#is-there-a-setup-wizard}

അതെ. setup wizard ആദ്യ സജീവമാക്കലിൽ സ്വയം പ്രവർത്തിക്കുകയും ഒരു രജിസ്ട്രാറുമായി ബന്ധിപ്പിക്കൽ, TLD-കൾ ഇറക്കുമതി ചെയ്യൽ, നിങ്ങളുടെ ആദ്യ ഡൊമെയ്ൻ ഉൽപ്പന്നം സൃഷ്ടിക്കൽ എന്നിവയിലൂടെ നിങ്ങളെ നയിക്കുകയും ചെയ്യും. addon ക്രമീകരണങ്ങളിൽ നിന്ന് wizard വീണ്ടും പ്രവർത്തിപ്പിക്കാനും കഴിയും.
