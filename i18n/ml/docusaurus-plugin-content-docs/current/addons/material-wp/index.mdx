# Material WP

WordPress-ന്റെ അഡ്മിൻ ഇന്റർഫേസ് വർഷങ്ങളായി ഒരേപോലെയാണ്. എന്നാൽ, ഒരു Material Design പതിപ്പിലുള്ള അഡ്മിൻ ഇന്റർഫേസ് എങ്ങനെയിരിക്കും എന്ന് നിങ്ങൾ എപ്പോഴെങ്കിലും ചിന്തിച്ചിട്ടുണ്ടോ?

Material WP എന്നത് ഒരു അഡ്മിൻ തീമാണ്. നിങ്ങൾക്ക് ആ രൂപവും ഭാവവും നൽകാൻ ഇത് വികസിപ്പിച്ചതാണ്. നിങ്ങളുടെ ക്ലയിന്റുകൾക്ക് കൂടുതൽ കസ്റ്റമൈസ് ചെയ്ത ഒരു Dashboard നൽകാൻ ഇത് നിങ്ങളെ സഹായിക്കും.

## സവിശേഷതകൾ {#features}

- കസ്റ്റം CSS സപ്പോർട്ട്
- എളുപ്പമുള്ള സ്റ്റൈലിംഗ് സെറ്റിംഗുകൾ
- മെനു പുനഃക്രമീകരണം
- എളുപ്പമുള്ള ഇംപോർട്ട്/എക്സ്പോർട്ട്

## ഡോക്യുമെന്റേഷൻ {#documentation}

**Material WP ഉപയോഗിച്ച് നിങ്ങളുടെ അഡ്മിൻ പാനൽ കസ്റ്റമൈസ് ചെയ്യുക**

## ഇൻസ്റ്റാളേഷൻ {#installation}

1. 'material-wp' എന്നത് '/wp-content/plugins/' ഡയറക്‌ടറിയിലേക്ക് അപ്‌ലോഡ് ചെയ്യുക
2. WordPress-ലെ 'Plugins' മെനു വഴി പ്ലഗിൻ ആക്ടിവേറ്റ് ചെയ്യുക

## ഉപയോഗ രീതി {#usage-guide}

WordPress-ന്റെ അഡ്മിൻ ഇന്റർഫേസ് വർഷങ്ങളായി ഒരേപോലെയാണ്. എന്നാൽ, ഒരു Material Design പതിപ്പിലുള്ള അഡ്മിൻ ഇന്റർഫേസ് എങ്ങനെയിരിക്കും എന്ന് നിങ്ങൾ എപ്പോഴെങ്കിലും ചിന്തിച്ചിട്ടുണ്ടോ?

Material WP എന്നത് ഒരു അഡ്മിൻ തീമാണ്. നിങ്ങൾക്ക് ആ രൂപവും ഭാവവും നൽകാൻ ഇത് വികസിപ്പിച്ചതാണ്. നിങ്ങളുടെ ക്ലയിന്റുകൾക്ക് കൂടുതൽ കസ്റ്റമൈസ് ചെയ്ത ഒരു Dashboard നൽകാൻ ഇത് നിങ്ങളെ സഹായിക്കും. Material WP എന്നത് Ultimate Multisite-ന്റെ പ്രീമിയം പ്ലഗിൻസ് കോറിന്റെ ഭാഗവുമാണ്.

ഇത് **ഇൻസ്റ്റാൾ ചെയ്ത ശേഷം**, നിങ്ങളുടെ _Plugins_ പേജിൽ പോയി നെറ്റ്വർക്കിൽ ഇത് ആക്ടിവേറ്റ് ചെയ്യുക.
