# Action: wu_captcha_integrations_init

എല്ലാ ഇന്റഗ്രേഷനുകളും (integrations) സജ്ജമാക്കിയ ശേഷം ഇത് പ്രവർത്തിപ്പിക്കുന്നു (fires).

മറ്റേതെങ്കിലും сторон (third-party) ഇന്റഗ്രേഷനുകൾക്ക് സ്വന്തമായി ചേർക്കാൻ ഇവിടെ hook ചെയ്യാവുന്നതാണ്.

- 1.2.0
