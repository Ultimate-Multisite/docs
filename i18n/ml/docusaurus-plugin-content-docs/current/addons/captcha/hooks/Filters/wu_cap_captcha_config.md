# Filter: wu_cap_captcha_config {#filter-wucapcaptchaconfig}

Cap Captcha കോൺഫിഗറേഷൻ ഫിൽട്ടർ ചെയ്യുക.

Cap Captcha സജ്ജീകരണങ്ങൾ കസ്റ്റമൈസ് ചെയ്യാൻ ഇത് ഡെവലപ്പർമാർക്ക് അനുവദിക്കുന്നു.

## പാരാമീറ്ററുകൾ {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $config | `array` | Cap കോൺഫിഗറേഷൻ അറേ. |
| $level | `string` | സുരക്ഷാ നിലവാരം (fast, medium, max). |
