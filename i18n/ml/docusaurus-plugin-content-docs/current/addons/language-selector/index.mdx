# ഭാഷാ തിരഞ്ഞെടുക്കൽ (Language Selector)

Multisite Ultimate: സൈൻ-അപ്പിൽ ഭാഷാ തിരഞ്ഞെടുക്കൽ

സൈൻ-അപ്പ് പ്രക്രിയയിലേക്ക് (sign-up flow) ഒരു ഭാഷാ തിരഞ്ഞെടുക്കൽ ഫീൽഡ് ചേർക്കുക.

## ഇൻസ്റ്റാളേഷൻ (Installation)

1. 'ultimate-multisite-language-selector' എന്നത് '/wp-content/plugins/' ഡയറക്‌ടറിയിലേക്ക് അപ്‌ലോഡ് ചെയ്യുക
2. നിങ്ങളുടെ WordPress Network Admin-ലെ 'Plugins' മെനുവിലൂടെ പ്ലഗിൻ ആക്ടിവേറ്റ് ചെയ്യുക

## ഉപയോഗ രീതി (Usage Guide)

ഈ സൗജന്യ ആഡ്-ഓൺ ഉപയോഗിച്ച്, സൈൻ-അപ്പ് പ്രക്രിയയിലും ലോഗിൻ പേജിലും ഉപയോക്താക്കൾക്ക് ഏത് ഭാഷയാണ് വേണ്ടതെന്ന് തിരഞ്ഞെടുക്കാൻ നിങ്ങൾക്ക് അനുവദിക്കാൻ കഴിയും.

[നിങ്ങളുടെ ലൈസൻസ് ആക്ടിവേറ്റ് ചെയ്ത ശേഷം](../../user-guide/getting-started/installing-ultimate-multisite.md), നിങ്ങളുടെ ലൈസൻസ് നൽകുന്ന Ultimate Multisite ആഡ്-ഓണുകൾ ഇൻസ്റ്റാൾ ചെയ്യാൻ നിങ്ങൾക്ക് കഴിയും.

നിങ്ങളുടെ Ultimate Multisite സെറ്റിംഗ്സ് പേജിലേക്ക് പോവുക, തുടർന്ന് ഇടതുവശത്തുള്ള മെനുവിൽ നിന്ന് _Add-ons_ എന്ന ലേബലിൽ ക്ലിക്ക് ചെയ്യുക.

![Add-ons page](/img/admin/addons-page.png)

_Language Selector ഇൻസ്റ്റാൾ ചെയ്യാൻ Add-ons പേജിലേക്ക് പോകുക_

അവിടെ, ഞങ്ങൾ നൽകുന്ന എല്ലാ ആഡ്-ഓണുകളും നിങ്ങൾക്ക് കാണാൻ കഴിയും.

സബ്‌സ്‌ക്രിപ്‌ഷൻ പ്ലാൻ ഏതാണെങ്കിലും, ഞങ്ങളുടെ എല്ലാ ഉപഭോക്താക്കൾക്കും സൗജന്യ ആഡ്-ഓണുകൾ ലഭ്യമാണ്.

ആഡ്-ഓണിന്റെ തംബ്നെയിലിന് താഴെയുള്ള _Install it Now!_ ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക. തുടർന്ന് Plugins പേജിൽ അത് ആക്ടിവേറ്റ് ചെയ്യുക.

![Add-ons page with install button](/img/admin/addons-page.png)

_ആഡ്-ഓൺ ഇൻസ്റ്റാൾ ചെയ്യാൻ ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_നിങ്ങളുടെ Plugins പേജിൽ അത് ആക്ടിവേറ്റ് ചെയ്യുക_

ഇപ്പോൾ, Ultimate Multisite-യുടെ Settings മെനുവിലേക്ക് പോവുക, തുടർന്ന് _Network Settings_ ടാബിൽ പ്രവേശിക്കുക. അവിടെ, പേജിന്റെ താഴെയായി, _Languages_ എന്ന് പേരുള്ള ഒരു പുതിയ സെക്ഷൻ കാണാം.

![Settings page](/img/config/settings-general.png)

_ഒരു പുതിയ സെക്ഷൻ പ്രത്യക്ഷപ്പെട്ടു!_

എല്ലാ ഭാഷകളെയും ഓപ്ഷനുകളായി കാണിക്കണമെങ്കിൽ ഇത് ഒഴിച്ചിടാവുന്നതാണ്, അല്ലെങ്കിൽ നിങ്ങൾക്ക് ലഭ്യമാക്കാൻ ആഗ്രഹിക്കുന്ന ഭാഷകൾ തിരഞ്ഞെടുക്കാവുന്നതാണ്. നിങ്ങളുടെ WordPress സെറ്റപ്പിൽ ഇൻസ്റ്റാൾ ചെയ്തിട്ടുള്ള ഭാഷകളാണ് ഇവ.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_ഈ WP ഇൻസ്റ്റാളേഷനിൽ ലഭ്യമായ ഭാഷകളുടെ ലിസ്റ്റ്_

നിങ്ങൾ വരുത്തിയ മാറ്റങ്ങൾ സേവ് ചെയ്ത ശേഷം, നിങ്ങൾ തിരഞ്ഞെടുത്ത ഭാഷകളോടുകൂടിയ ഒരു സെലക്ഷൻ മെനു ലോഗിൻ പേജിലും സൈൻ-അപ്പ് പ്രക്രിയയിലും കാണിക്കും.

_ലോഗിൻ പേജിലെ ഭാഷാ തിരഞ്ഞെടുക്കൽ_

![Registration page](/img/config/registration-page.png)

_സൈൻ-അപ്പ് പ്രക്രിയയിലെ ഭാഷാ തിരഞ്ഞെടുക്കൽ_

## URL വഴി ഭാഷ സജ്ജമാക്കൽ

നിങ്ങൾക്ക് ലൊക്കൽ (locale) പാരാമീറ്റർ ഒഴിവാക്കി URL വഴി ഒരു പേജിന്റെ ഭാഷ മാറ്റാനും കഴിയും. ഉദാഹരണത്തിന്, ഇറ്റാലിയൻ ഭാഷയിലുള്ള ഒരു രജിസ്റ്റർ പേജിലേക്ക് റീഡയറക്ട് ചെയ്യണമെങ്കിൽ, URL-ൽ ?locale=it_IT എന്ന് ചേർക്കുക.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

ഇത് പ്രവർത്തിക്കണമെങ്കിൽ, ആ ഭാഷ നിങ്ങളുടെ WordPress നെറ്റ്‌വർക്കിൽ ഇൻസ്റ്റാൾ ചെയ്തിരിക്കണം.
