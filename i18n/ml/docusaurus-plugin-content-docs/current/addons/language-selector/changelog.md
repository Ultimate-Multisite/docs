Version: 1.1.3 - പുറത്തിറങ്ങിയ തീയതി: 2026-05-05
* പരിഹരിച്ചു: ഇൻപുട്ട് വേരിയബിളുകൾ ഇപ്പോൾ ശുദ്ധീകരിച്ചിരിക്കുന്നു (sanitized). കൂടാതെ, മൂന്നാം കക്ഷി പ്ലഗിനുകളുമായി ഉണ്ടാകാൻ സാധ്യതയുള്ള ගැටലുകൾ ഒഴിവാക്കാൻ കസ്റ്റം hook പേരുകൾക്ക് ഒരു പ്രിഫിക്സ് നൽകിയിട്ടുണ്ട്.
