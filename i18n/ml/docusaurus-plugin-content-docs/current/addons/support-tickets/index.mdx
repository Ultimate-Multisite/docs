# സപ്പോർട്ട് ടിക്കറ്റുകൾ (Support Tickets)

**സപ്പോർട്ട് ടിക്കറ്റുകൾ** എന്നത് Ultimate Multisite-നായുള്ള ഒരു ശക്തമായ അഡോൺ ആണ്. ഇത് നിങ്ങളുടെ WordPress നെറ്റ്‌വർക്കിലേക്ക് ഒരു സമഗ്രമായ സപ്പോർട്ട് ടിക്കറ്റ് സിസ്റ്റം കൂട്ടിച്ചേർക്കുന്നു. SaaS പ്ലാറ്റ്‌ഫോമുകൾ, WaaS (Website as a Service) പ്രൊവൈഡർമാർ, നെറ്റ്‌വർക്കിലും വ്യക്തിഗത സൈറ്റുകളിലും ഉപഭോക്തൃ പിന്തുണ നൽകേണ്ട എല്ലാ മൾട്ടിസൈറ്റ് നെറ്റ്‌വർക്കുകൾക്കും ഇത് അനുയോജ്യമാണ്.

## സപ്പോർട്ട് ടിക്കറ്റുകൾ എന്തിന് തിരഞ്ഞെടുക്കണം? {#why-choose-support-tickets}

* **മൾട്ടി-ലെവൽ സപ്പോർട്ട്**: നെറ്റ്‌വർക്ക് അഡ്മിനിസ്ട്രേറ്റർമാർ നെറ്റ്‌വർക്ക് മുഴുവൻ ബാധകമായ ടിക്കറ്റുകൾ കൈകാര്യം ചെയ്യുമ്പോൾ, സൈറ്റ് ഉടമകൾ സ്വന്തം ഉപഭോക്തൃ പിന്തുണ കൈകാര്യം ചെയ്യുന്നു.
* **പ്രൊഫഷണൽ വർക്ക്‌ഫ്ലോ**: മുൻഗണനകൾ (priorities), സ്റ്റാറ്റസുകൾ (statuses), അസൈൻമെന്റുകൾ എന്നിവ ഉൾപ്പെടെ ടിക്കറ്റ് ലൈഫ്‌സൈക്കിൾ മാനേജ്മെന്റ് പൂർണ്ണമായി നൽകുന്നു.
* **ഉപയോക്തൃ അനുഭവം (Customer Experience)**: ഉപഭോക്താക്കൾക്ക് സഹായം എളുപ്പത്തിൽ ലഭിക്കാൻ സഹായിക്കുന്ന മനോഹരവും പ്രതികരിക്കുന്നതുമായ (responsive) ഇന്റർഫേസ്.
* **സുരക്ഷിതവും വിശ്വസനീയവും**: WordPress സുരക്ഷാ മികച്ച രീതികൾ ഉപയോഗിച്ചാണ് ഇത് നിർമ്മിച്ചിരിക്കുന്നത്, കൂടാതെ ഡാറ്റാബേസ് പ്രകടനം ഒപ്റ്റിമൈസ് ചെയ്തിരിക്കുന്നു.
* **സുഗമമായ സംയോജനം (Seamless Integration)**: Ultimate Multisite-ന്റെ ഉപഭോക്തൃ, സൈറ്റ് മാനേജ്മെന്റ് എന്നിവയുമായി ഇത് മികച്ച രീതിയിൽ പ്രവർത്തിക്കുന്നു.

## പ്രധാന സവിശേഷതകൾ (Key Features) {#key-features}

**പൂർണ്ണമായ ടിക്കറ്റ് മാനേജ്മെന്റ്**

* എളുപ്പമുള്ള റഫറൻസിനായി തനതായ ടിക്കറ്റ് നമ്പറിംഗ് സിസ്റ്റം
* കോൺഫിഗറബിൾ പ്രയോറിറ്റി ലെവലുകൾ (Low, Normal, High, Critical)
* സമഗ്രമായ സ്റ്റാറ്റസ് വർക്ക്‌ഫ്ലോ (Open, In Progress, Waiting, Resolved, Closed)
* ടിക്കറ്റ് തരംതിരിക്കൽ (General, Technical, Billing, Feature Request, Bug Report)
* ഓപ്ഷണൽ ഓട്ടോ-അസൈൻമെന്റോടുകൂടി ടിക്കറ്റുകൾ സപ്പോർട്ട് ഏജന്റുമാർക്ക് അസൈൻ ചെയ്യാം

**ത്രെഡഡ് സംഭാഷണങ്ങൾ (Threaded Conversations)**

* ടിക്കറ്റ് മറുപടികൾക്കായി പൂർണ്ണമായ സംഭാഷണ ത്രെഡിംഗ്.
* സ്റ്റാഫുകൾക്കും ഉപഭോക്താക്കൾക്കുമുള്ള മറുപടികൾക്ക് പ്രത്യേക സ്റ്റൈലിംഗ്.
* ഉപഭോക്താക്കൾക്ക് കാണാൻ കഴിയാത്ത സ്വകാര്യ സ്റ്റാഫ് നോട്ടുകൾ.
* റിയൽ-ടൈം AJAX-പവർഡ് ഇന്റർഫേസ്.
* ഫോർമാറ്റിംഗ് ഓപ്ഷനുകളോടുകൂടിയ ആധുനിക ടെക്സ്റ്റ് എഡിറ്റർ.

**ഫയൽ അറ്റാച്ച്മെന്റുകൾ**

* ഡ്രാഗ്-ആൻഡ്-ഡ്രോപ്പ് സപ്പോർട്ടോടുകൂടിയ സുരക്ഷിത ഫയൽ അപ്‌ലോഡ്.
* കോൺഫിഗറബിൾ ഫയൽ തരം, വലുപ്പ പരിമിതികൾ.
*悪 (malicious) ഫയലുകൾ കണ്ടെത്താനുള്ള അഡ്വാൻസ്ഡ് സുരക്ഷ.
* ആക്‌സസ് നിയന്ത്രണങ്ങളോടുകൂടിയ സംരക്ഷിത ഡൗൺലോഡുകൾ.

**സ്മാർട്ട് നോട്ടിഫിക്കേഷനുകൾ**

* പുതിയ ടിക്കറ്റുകൾ, മറുപടികൾ, സ്റ്റാറ്റസ് മാറ്റങ്ങൾ എന്നിവയ്ക്കായി ഓട്ടോമാറ്റിക് ഇമെയിൽ അലേർട്ടുകൾ.
* കസ്റ്റമൈസബിൾ ഇമെയിൽ ടെംപ്ലേറ്റുകൾ.
* സ്റ്റാഫുകൾക്കും ഉപഭോക്താക്കൾക്കുമുള്ള റോൾ അടിസ്ഥാനമാക്കിയുള്ള നോട്ടിഫിക്കേഷനുകൾ.
* വ്യക്തിഗതമാക്കിയ സന്ദേശങ്ങൾക്കായുള്ള ഡൈനാമിക് ടെംപ്ലേറ്റ് വേരിയബിളുകൾ.

**ശക്തമായ അഡ്മിൻ ഇന്റർഫേസ്**

* സോർട്ടിംഗും ഫിൽട്ടറിംഗും ഉള്ള അഡ്വാൻസ്ഡ് ടിക്കറ്റ് ലിസ്റ്റ്.
* സ്റ്റാറ്റസ്, പ്രയോറിറ്റി, അസൈൻമെന്റുകൾ എന്നിവ വേഗത്തിൽ എഡിറ്റ് ചെയ്യാനുള്ള സൗകര്യം.
* കാര്യക്ഷമമായ മാനേജ്മെന്റിനായുള്ള ബൾക്ക് ആക്ഷനുകൾ.
* പ്രകടന അളവുകളോടുകൂടിയ സ്റ്റാറ്റിസ്റ്റിക്സ് ഡാഷ്‌ബോർഡ്.
* സമഗ്രമായ സെറ്റിംഗ്സ് പാനൽ.

**മനോഹരമായ കസ്റ്റമർ പോർട്ടൽ**

* എല്ലാ ഉപകരണങ്ങളിലും വൃത്തിയുള്ള, പ്രതികരിക്കുന്ന ഇന്റർഫേസ്.
* മൊബൈൽ-ഫസ്റ്റ് ഡിസൈൻ.
* അഡ്വാൻസ്ഡ് സെർച്ച്, ഫിൽട്ടറിംഗ്.
* ഫോർമാറ്റിംഗ് ടൂൾബാർ ഉള്ള ആധുനിക എഡിറ്റർ.
* ഉപയോഗിക്കാൻ എളുപ്പമുള്ള ടിക്കറ്റ് സമർമിഷൻ ഫോമുകൾ.

**ഫ്ലെക്സിബിൾ ഇന്റഗ്രേഷൻ**

* ഷോർട്ട്കോഡുകളോ Ultimate Multisite UI ഘടകങ്ങളോ ഉപയോഗിക്കുക.
* നിങ്ങളുടെ സൈറ്റിലെ എവിടെ വേണമെങ്കിലും ടിക്കറ്റ് ഫോമുകളും ലിസ്റ്റുകളും പ്രദർശിപ്പിക്കുക.
* മൾട്ടിസൈറ്റ് നെറ്റ്‌വർക്ക് ഘടനയുമായി ഇത് തടസ്സമില്ലാതെ പ്രവർത്തിക്കുന്നു.
* Hooks, filters എന്നിവ ഉപയോഗിച്ച് വികസിപ്പിക്കാൻ കഴിയും.

## എന്തിനുവേണ്ടി മികച്ചത്? {#perfect-for}

* ഉപഭോക്തൃ പിന്തുണ ആവശ്യമുള്ള SaaS പ്ലാറ്റ്‌ഫോമുകൾ.
* ഒന്നിലധികം ക്ലയിന്റ് സൈറ്റുകൾ കൈകാര്യം ചെയ്യുന്ന WaaS പ്രൊവൈഡർമാർ.
* ടയേർഡ് സപ്പോർട്ട് ഉള്ള മെമ്പർഷിപ്പ് സൈറ്റുകൾ.
* ഒന്നിലധികം ബ്രാൻഡുകൾ കൈകാര്യം ചെയ്യുന്ന നെറ്റ്‌വർക്ക് അഡ്മിനിസ്ട്രേറ്റർമാർ.
* ഉപഭോക്തൃ സേവനം നൽകുന്ന സൈറ്റ് ഉടമകൾ.

## ഇൻസ്റ്റാളേഷൻ {#installation}

1. നിങ്ങളുടെ WordPress നെറ്റ്‌വർക്കിൽ Ultimate Multisite പ്ലഗിൻ ഇൻസ്റ്റാൾ ചെയ്ത് ആക്ടിവേറ്റ് ചെയ്തിട്ടുണ്ടെന്ന് ഉറപ്പാക്കുക.
2. അഡോൺ ഫയലുകൾ നിങ്ങളുടെ `/wp-content/plugins/` ഡയറക്‌ടറിയിലേക്ക് അപ്‌ലോഡ് ചെയ്യുക.
3. WordPress-ലെ 'Plugins' മെനുവിലൂടെ നെറ്റ്‌വർക്ക് ആക്ടിവേറ്റ് ചെയ്യുക.
4. നിങ്ങളുടെ സെറ്റിംഗുകൾ കോൺഫിഗർ ചെയ്യാൻ Network Admin > Support Tickets എന്നതിലേക്ക് പോകുക.

## പതിവായി ചോദിക്കുന്ന ചോദ്യങ്ങൾ (Frequently Asked Questions) {#frequently-asked-questions}

## ഇതിന് Ultimate Multisite ആവശ്യമുണ്ടോ? {#does-this-require-ultimate-multisite}

അതെ, ഈ അഡോൺ പ്രവർത്തിക്കണമെങ്കിൽ Ultimate Multisite ഇൻസ്റ്റാൾ ചെയ്യുകയും ആക്റ്റീവ് ആക്കുകയും വേണം. Ultimate Multisite-ന്റെ ഉപഭോക്തൃ, സൈറ്റ് മാനേജ്മെന്റ് സവിശേഷതകളുമായി ഇത് ആഴത്തിൽ സംയോജിക്കുന്നു.

## സൈറ്റ് ഉടമകൾക്ക് സ്വന്തമായി ഉപഭോക്തൃ പിന്തുണ കൈകാര്യം ചെയ്യാൻ കഴിയുമോ? {#can-site-owners-handle-their-own-customer-support}

തീർച്ചയായും! സൈറ്റ് ഉടമകൾക്ക് അവരുടെ സ്വന്തം ഉപഭോക്താക്കൾക്ക് പിന്തുണ നൽകാൻ ടിക്കറ്റ് സിസ്റ്റം ഉപയോഗിക്കാനാകും. അതേസമയം, നെറ്റ്‌വർക്ക് അഡ്മിനിസ്ട്രേറ്റർമാർക്ക് നെറ്റ്‌വർക്ക് മുഴുവൻ ബാധകമായ സപ്പോർട്ട് ടിക്കറ്റുകൾ കൈകാര്യം ചെയ്യാം.

## ഉപഭോക്താക്കൾ എങ്ങനെ ടിക്കറ്റുകൾ സമർപ്പിക്കും? {#how-do-customers-submit-tickets}

ഷോർട്ട്കോഡുകളോ Ultimate Multisite UI ഘടകങ്ങളോ ഉപയോഗിച്ച് നിങ്ങളുടെ സൈറ്റിലെ എവിടെ വേണമെങ്കിലും ടിക്കറ്റ് സമർപ്പിക്കാനുള്ള ഫോമുകൾ ചേർക്കാൻ നിങ്ങൾക്ക് കഴിയും. ഉപഭോക്താക്കൾ ഫോം പൂരിപ്പിച്ച് അവരുടെ ആവശ്യം സമർപ്പിക്കുക മാത്രം ചെയ്താൽ മതി.

## എനിക്ക് ടിക്കറ്റ് സ്റ്റാറ്റസ്സുകളും പ്രയോറിറ്റികളും കസ്റ്റമൈസ് ചെയ്യാൻ കഴിയുമോ? {#can-i-customize-the-ticket-statuses-and-priorities}

അതെ! നിങ്ങളുടെ സപ്പോർട്ട് വർക്ക്‌ഫ്ലോയ്ക്ക് അനുയോജ്യമായ രീതിയിൽ പ്രയോറിറ്റി ലെവലുകൾ, സ്റ്റാറ്റസ്സുകൾ, ടിക്കറ്റ് തരങ്ങൾ എന്നിവ നെറ്റ്‌വർക്ക് അഡ്മിൻ സെറ്റിംഗ്സിൽ പൂർണ്ണമായി കസ്റ്റമൈസ് ചെയ്യാൻ നിങ്ങൾക്ക് കഴിയും.

## ഇത് ഫയൽ അറ്റാച്ച്മെന്റുകൾ സപ്പോർട്ട് ചെയ്യുന്നുണ്ടോ? {#does-it-support-file-attachments}

അതെ, ഉപഭോക്താക്കൾക്കും സ്റ്റാഫുകൾക്കും ടിക്കറ്റുകളിലേക്കും മറുപടികളിലേക്കും ഫയലുകൾ അറ്റാച്ച് ചെയ്യാൻ കഴിയും. സുരക്ഷയ്ക്കായി അനുവദനീയമായ ഫയൽ തരങ്ങൾ, വലുപ്പ പരിധികൾ എന്നിവയിൽ നിങ്ങൾക്ക് പൂർണ്ണ നിയന്ത്രണമുണ്ട്.

## നോട്ടിഫിക്കേഷനുകൾ എങ്ങനെ കൈകാര്യം ചെയ്യുന്നു? {#how-are-notifications-handled}

പുതിയ ടിക്കറ്റുകൾ, മറുപടികൾ, സ്റ്റാറ്റസ് മാറ്റങ്ങൾ എന്നിവയ്ക്കായി സിസ്റ്റം ഓട്ടോമാറ്റിക്കായി ഇമെയിൽ നോട്ടിഫിക്കേഷനുകൾ അയയ്ക്കുന്നു. എല്ലാ ഇമെയിൽ ടെംപ്ലേറ്റുകളും സെറ്റിംഗ്സിൽ കസ്റ്റമൈസ് ചെയ്യാൻ നിങ്ങൾക്ക് കഴിയും.

## ടിക്കറ്റുകൾ പ്രത്യേക സപ്പോർട്ട് ഏജന്റുമാർക്ക് അസൈൻ ചെയ്യാൻ കഴിയുമോ? {#can-i-assign-tickets-to-specific-support-agents}

അതെ, ടിക്കറ്റുകൾ ഏജന്റുമാർക്ക് മാനുവലായി അസൈൻ ചെയ്യാനോ, ടിക്കറ്റുകൾ ഓട്ടോമാറ്റിക്കായി വിതരണം ചെയ്യുന്നതിനായി ഓട്ടോ-അസൈൻമെന്റ് എനേബിൾ ചെയ്യാനോ നിങ്ങൾക്ക് കഴിയും.

## സ്റ്റാഫുകൾക്കായി സ്വകാര്യ നോട്ടുകൾ ഉണ്ടോ? {#are-there-private-notes-for-staff}

അതെ, സ്റ്റാഫ് അംഗങ്ങൾക്ക് സ്വകാര്യ മറുപടികൾ ചേർക്കാൻ കഴിയും. ഇത് മറ്റ് സ്റ്റാഫ് അംഗങ്ങൾക്ക് മാത്രമേ കാണാൻ കഴിയൂ, ഉപഭോക്താക്കൾക്ക് കഴിയില്ല.

## ഇത് മൊബൈൽ-ഫ്രണ്ട്ലിയാണോ? {#is-it-mobile-friendly}

തീർച്ചയായും! മുഴുവൻ ഇന്റർഫേസും മൊബൈൽ-ഫസ്റ്റ് സമീപനത്തോടെയാണ് നിർമ്മിച്ചിരിക്കുന്നത്, കൂടാതെ എല്ലാ ഉപകരണങ്ങളിലും ഇത് മനോഹരമായി പ്രവർത്തിക്കുന്നു.

## ഷോർട്ട്കോഡുകൾ ഉപയോഗിച്ച് ടിക്കറ്റ് ലിസ്റ്റുകളും ഫോമുകളും പ്രദർശിപ്പിക്കാൻ കഴിയുമോ? {#can-i-display-ticket-lists-and-forms-using-shortcodes}

അതെ! ലഭ്യമായ ഷോർട്ട്കോഡുകൾ:
* `[wu_support_tickets]` - ഉപഭോക്തൃ ടിക്കറ്റ് ലിസ്റ്റ് പ്രദർശിപ്പിക്കുന്നു
* `[wu_support_ticket_form]` - പുതിയ ടിക്കറ്റ് സമർപ്പിക്കാനുള്ള ഫോം പ്രദർശിപ്പിക്കുന്നു
* `[wu_support_ticket_view ticket_id="123"]` - പ്രത്യേക ടിക്കറ്റ് പ്രദർശിപ്പിക്കുന്നു

## എങ്ങനെയുള്ള ഫയൽ തരങ്ങളാണ് അപ്‌ലോഡ് ചെയ്യാൻ കഴിയുക? {#what-file-types-can-be-uploaded}

സെറ്റിംഗ്സിൽ അനുവദനീയമായ ഫയൽ തരങ്ങൾ കോൺഫിഗർ ചെയ്യാൻ നിങ്ങൾക്ക് കഴിയും.悪 (malicious) ഫയലുകൾക്കെതിരെ സംരക്ഷണം നൽകാൻ സിസ്റ്റം അഡ്വാൻസ്ഡ് സുരക്ഷാ സ്കാനിംഗ് ഉൾക്കൊള്ളുന്നു.

## ടിക്കറ്റ് ഡാറ്റ എക്‌സ്‌പോർട്ട് ചെയ്യാൻ കഴിയുമോ? {#can-i-export-ticket-data}

അതെ, ടിക്കറ്റ് ഡാറ്റയും സ്റ്റാറ്റിസ്റ്റിക്സുകളും എക്‌സ്‌പോർട്ട് ചെയ്യാനുള്ള കഴിവുകൾ അഡോണിൽ ഉൾപ്പെടുത്തിയിട്ടുണ്ട്.
