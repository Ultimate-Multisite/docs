### 1.0.3 - 2026-05-01 {#103---2026-05-01}
* പരിഹരിച്ചു: nullable ticket model setters-ൽ null values സ്വീകരിക്കുന്നു.
* പരിഹരിച്ചു: ശരിയായ multisite പിന്തുണയ്ക്കായി എല്ലാ ടിക്കറ്റുകളും ഗ്ലോബൽ നെറ്റ്‌വർക്ക്-വ്യാപകമായ ടേബിളുകളിൽ സ്റ്റോർ ചെയ്യുന്നു.
* പരിഹരിച്ചു: സ്റ്റാഫ് ഫീൽഡുകൾ മറച്ചു, കൂടാതെ customer new-ticket ഫോമിൽ blank-page redirect പ്രശ്നം പരിഹരിച്ചു.
* പരിഹരിച്ചു: മെറ്റാഡാറ്റ സ്റ്റോറിംഗിനായി undefined add_meta() കോളുകൾ ശരിയായ update_meta() ഉപയോഗിച്ച് മാറ്റിസ്ഥാപിച്ചു.
* പരിഹരിച്ചു: രജിസ്റ്റർ ചെയ്യാത്ത capability check ശരിയായ wu_view_all_support_tickets ഉപയോഗിച്ച് മാറ്റിസ്ഥാപിച്ചു.
* പരിഹരിച്ചു: ടിക്കറ്റ് സ്റ്റാറ്റസ്, അസൈൻമെന്റ്, ക്വിക്ക്-എഡിറ്റ് ആക്ഷനുകൾക്കായി നഷ്ടപ്പെട്ട AJAX ഹാൻഡ്ലറുകൾ കൂട്ടിച്ചേർത്തു.
* പരിഹരിച്ചു: ടിക്കറ്റ് വ്യൂവുകളിലെ സ്റ്റാഫ് റെസ്‌പോൺസ് ഡിറ്റക്ഷনের രീതിയുടെ പേര് ശരിയാക്കി.
* പരിഹരിച്ചു: ഡ്യൂപ്ലിക്കേറ്റ് ആയ റിപ്ലൈ ഹാൻഡ്ലറുകൾ സംയോജിപ്പിക്കുകയും nonce action പേരുകൾ ക്രമീകരിക്കുകയും ചെയ്തു.
* പരിഹരിച്ചു: [wu_submit_ticket] ഷോർട്ട്കോഡിനായുള്ള നഷ്ടപ്പെട്ട ഫ്രണ്ട്എൻഡ് വ്യൂ കൂട്ടിച്ചേർത്തു.
* പരിഹരിച്ചു: നഷ്ടപ്പെട്ട user_id കോളം കൂട്ടിച്ചേർക്കുകയും Support_Ticket::get_user_id() രീതി പരിഹരിക്കുകയും ചെയ്തു.
* പരിഹരിച്ചു: പ്രയോറിറ്റി ഫിൽട്ടർ ഓപ്ഷൻ ടാഗിലെ അധിക ഡബിൾ-കോട്ട് നീക്കം ചെയ്തു.
* പരിഹരിച്ചു: സൂപ്പർ അഡ്മിൻസിനായി നെറ്റ്‌വർക്ക് അഡ്മിൻ ടിക്കറ്റ് മാനേജ്മെന്റ് പാനൽ കൂട്ടിച്ചേർത്തു.
* മെച്ചപ്പെടുത്തി: അഡ്മിൻ CSS ഒറ്റ എക്സ്റ്റേണൽ സ്റ്റൈൽഷീറ്റിൽ സംയോജിപ്പിച്ചു.
* മെച്ചപ്പെടുത്തി: സബ്സൈറ്റ് അഡ്മിൻ മെനുവിൽ നിന്ന് Settings സബ്മെനു നീക്കം ചെയ്തു.
* മെച്ചപ്പെടുത്തി: സപ്പോർട്ട് ടിക്കറ്റ് പേജുകളിൽ മാത്രം ഫ്രണ്ട്എൻഡ് ആസറ്റുകൾ കണ്ടീഷണലായി ലോഡ് ചെയ്യുന്നു.
* മെച്ചപ്പെടുത്തി: Bedrock root autoloader ഇതിനകം ഡിപൻഡൻസികൾ ലോഡ് ചെയ്തിട്ടുണ്ടെങ്കിൽ, പ്ലഗിൻ autoloader ഒഴിവാക്കുന്നു.

### 1.0.2 - 2025-12-11 {#102---2025-12-11}
* കൂട്ടിച്ചേർത്തു: അഡ്മിനും ഫ്രണ്ട്എൻഡിനും വേണ്ടി മുഴുവൻ ടിക്കറ്റ് വ്യൂവിംഗ് ഫംഗ്ഷണാലിറ്റിയും.
* കൂട്ടിച്ചേർത്തു: ടിക്കറ്റ് റിപ്ലൈ സമർമിഷനുള്ള AJAX ഹാൻഡ്ലർ.
* കൂട്ടിച്ചേർത്തു: ശരിയായ ഫോം ഹാൻഡ്ലിംഗോടുകൂടിയ ടിക്കറ്റ് റിപ്ലൈ ഫംഗ്ഷണാലിറ്റി പിന്തുണ.
* കൂട്ടിച്ചേർത്തു: അഡ്മിനിൽ ടിക്കറ്റ് സമർമിഷുകൾക്കും റിപ്ലൈകൾക്കുമായി ശരിയായ നോട്ടിസ് ഡിസ്‌പ്ലേ.
* കൂട്ടിച്ചേർത്തു: കസ്റ്റമർമാർ സ്വയം സമർപ്പിക്കുമ്പോൾ നിലവിലെ യൂസറുമായി ടിക്കറ്റ് ഓട്ടോമാറ്റിക്കായി ബന്ധിപ്പിക്കുന്നു.
* കൂട്ടിച്ചേർത്തു: ടിക്കറ്റ് ഉടമസ്ഥാവകാശം കസ്റ്റമർമാർക്ക് മാറ്റാൻ കഴിയാതിരിക്കാൻ സുരക്ഷാ മെച്ചപ്പെടുത്തൽ.
* കൂട്ടിച്ചേർത്തു: നഷ്ടപ്പെട്ട Helper ഫംഗ്ഷനുകൾ (wu_format_date, wu_user_can_view_ticket, തുടങ്ങിയവ).
* കൂട്ടിച്ചേർത്തു: ശരിയായ ഫയൽ അറ്റാച്ച്മെന്റ് ഡൗൺലോഡ് ചെയ്യാനുള്ളതും കൈകാര്യം ചെയ്യാനുള്ളതുമായ സംവിധാനം.
* കൂട്ടിച്ചേർത്തു: ടിക്കറ്റ് റിപ്ലൈകൾക്കും സ്റ്റാറ്റസ് മാറ്റങ്ങൾക്കും ഇമെയിൽ നോട്ടിഫിക്കേഷൻ സിസ്റ്റം.
* പരിഹരിച്ചു: ടിക്കറ്റ് റിപ്ലൈ ഫോമിൽ ഇപ്പോൾ ആവശ്യമായ ടിക്കറ്റ് ID ഉൾപ്പെടുന്നു.
* പരിഹരിച്ചു: അഡ്മിൻ ടിക്കറ്റ് വ്യൂവുകളിലെ നോട്ടിസ് കാണാനുള്ള പ്രശ്നങ്ങൾ.
* പരിഹരിച്ചു: സപ്പോർട്ട് ടിക്കറ്റ് ഫംഗ്ഷനുകളിലെ സിന്റാക്സ് പിശകുകൾ.
* മെച്ചപ്പെടുത്തി: റിപ്ലൈകൾക്കും അറ്റാച്ച്മെന്റുകൾക്കുമായി ശരിയായ ക്വറി, സ്കീമ ക്ലാസുകളോടുകൂടിയ ഡാറ്റാബേസ് ഘടന.
* മെച്ചപ്പെടുത്തി: ടിക്കറ്റ് മോഡലുകൾക്കായി അറ്റാട്രിബ്യൂട്ടുകൾക്ക് പകരം യഥാർത്ഥ പ്രോപ്പർട്ടികളിലേക്ക് മൈഗ്രേഷൻ.
