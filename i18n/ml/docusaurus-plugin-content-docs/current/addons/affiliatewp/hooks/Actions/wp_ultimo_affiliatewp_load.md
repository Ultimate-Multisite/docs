# Action: wp_ultimo_affiliatewp_load {#action-wpultimoaffiliatewpload}

എല്ലാ ഡിപൻഡൻസികളും ലോഡ് ചെയ്യുമ്പോൾ ഇത് പ്രവർത്തിപ്പിക്കുന്നു (Triggers).

പ്ലഗിൻ ഡെവലപ്പർമാർക്ക് പുതിയ ഫംഗ്ഷണാലിറ്റി കൂട്ടിച്ചേർക്കാൻ ഇത് അവസരം നൽകുന്നു. ഉദാഹരണത്തിന്, പുതിയ ഹോസ്റ്റിംഗ് പ്രൊവൈഡർമാർക്കുള്ള സപ്പോർട്ട് എന്നിവ ചേർക്കാൻ ഇത് ഉപയോഗിക്കാം.
