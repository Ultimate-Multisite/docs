# AffiliateWP {#affiliatewp}

Multisite Ultimate: AffiliateWP Integration

നിങ്ങളുടെ Ultimo Network-ന്റെ ക്ലയിന്റ് ബേസ് വളർത്താൻ ശക്തമായ AffiliateWP ഉപയോഗിക്കുക!

## Installation {#installation}

1. 'ultimate-multisite-git ' എന്ന ഫയൽ '/wp-content/plugins/' ഡയറക്‌ടറിയിലേക്ക് അപ്‌ലോഡ് ചെയ്യുക.
2. WordPress-ലെ 'Plugins' മെനു വഴി പ്ലഗിൻ ആക്റ്റീവേറ്റ് ചെയ്യുക.
3. പ്ലഗിൻ സജ്ജമാക്കാൻ ഘട്ടം ഘട്ടമായുള്ള Wizard പിന്തുടരുക.

## Usage Guide {#usage-guide}

നിങ്ങളുടെ ഓൺലൈൻ ബിസിനസ്സിനായി ഒരു അഫിലിയേറ്റ് പ്രോഗ്രാം ഉണ്ടാക്കുമ്പോൾ [AffiliateWP](https://affiliatewp.com/) ആണ് ഏറ്റവും മികച്ച പരിഹാരം. നിങ്ങളുടെ പ്രീമിയം നെറ്റ്‌വർക്കിനായി ഇതേ കാര്യം ചെയ്യാൻ നിങ്ങളെ സഹായിക്കുന്ന പാലമാണ് Ultimate Multisite: AffiliateWP.

## Getting Started {#getting-started}

എല്ലാറ്റിനുമുപരിയായി, നിങ്ങൾക്ക് [**AffiliateWP**](https://affiliatewp.com/) **ആക്റ്റീവ്** ആയിരിക്കണം! ഇതിനോട് സംയോജിക്കുന്ന **പ്ലഗിൻ** നിങ്ങളുടെ പക്കലില്ലെങ്കിൽ നിങ്ങൾക്ക് ഞങ്ങളുടെ ആഡ്-ഓൺ ഉപയോഗിക്കാൻ കഴിയില്ല. നിങ്ങളുടെ ക്ലയിന്റുകൾ അവരുടെ സൈറ്റുകളിലും ഇത് ഉപയോഗിക്കാൻ ഉദ്ദേശിക്കുന്നില്ലെങ്കിൽ, AffiliateWP നിങ്ങളുടെ പ്രധാന സൈറ്റിൽ മാത്രമേ ആക്റ്റീവ് ആയിരിക്കാവൂ.

കൂടാതെ, **Ultimate Multisite** ഉം **Ultimate Multisite: AffiliateWP** ഉം **നെറ്റ്‌വർക്ക് മുഴുവൻ** ആക്റ്റീവ് ആയിരിക്കണം.

## Adding Ultimate Multisite as an AffiliateWP Integration {#adding-ultimate-multisite-as-an-affiliatewp-integration}

നിങ്ങളുടെ പ്രധാന സൈറ്റിൽ, Affiliates Settings-ലേക്ക് പോവുക, അവിടെ Integrations ടാബിൽ Ultimate Multisite ടിക്ക് ചെയ്യുക. ഇത് [AffiliateWP](https://affiliatewp.com/) നെ Ultimate Multisite സൈൻ-അപ്പ് ഫ്ലോയിൽ നിന്നുള്ള സന്ദർശനങ്ങളും കൺവേർഷനുകളും ട്രാക്ക് ചെയ്യാൻ തുടങ്ങാൻ പറയും.

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_താഴെയുള്ള Ultimate Multisite ബോക്സ് ടിക്ക് ചെയ്ത് സേവ് ചെയ്യുക_

അത്രയേ വേണ്ടൂ. റഫറൽ ലിങ്കുകളിൽ നിന്നുള്ള സന്ദർശനങ്ങൾ ട്രാക്ക് ചെയ്യപ്പെടും, ആ ഉപയോക്താവ് പിന്നീട് ആദ്യ പേയ്‌മെന്റ് നടത്തുമ്പോൾ, കൺവേർഷൻ സൃഷ്ടിക്കപ്പെടുകയും ആ കൺവേർഷന് ഉത്തരവാദിത്തമുള്ള അഫിലിയേറ്റിന് ഒരു കമ്മീഷൻ മാറ്റിവെക്കപ്പെടുകയും ചെയ്യും.

എങ്കിലും, പുതിയതായി സൃഷ്ടിക്കപ്പെട്ട ഉപയോക്താവിൽ നിന്ന് ലഭിക്കുന്ന ആദ്യ പേയ്‌മെന്റ് മാത്രമേ ട്രാക്ക് ചെയ്യվൂ. ആവർത്തിച്ചുള്ള റഫറൽ പേയ്‌മെന്റുകൾ പ്രവർത്തനക്ഷമമാക്കാൻ, നിങ്ങൾക്ക് [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) ആവശ്യമാണ്.

## Enabling Recurring Referrals {#enabling-recurring-referrals}

[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) ഇൻസ്റ്റാൾ ചെയ്ത ശേഷം, Affiliates Settings-ലെ Recurring Referrals ടാബിലേക്ക് പോവുക, തുടർന്ന് ആവർത്തിച്ചുള്ള റഫറലുകൾ ഉപയോഗിക്കുന്നത് പ്രവർത്തനക്ഷമമാക്കുക.
