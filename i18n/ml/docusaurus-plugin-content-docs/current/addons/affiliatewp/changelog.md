# AffiliateWP മാറ്റങ്ങളുടെ രേഖ

Version: 2.0.1 - Released on 2025-09-28

* പ്രിഫിക്സ് ultimate-multisite എന്ന് മാറ്റുന്നു; ടെക്സ്റ്റ് ഡൊമെയ്ൻ അപ്ഡേറ്റ് ചെയ്യുന്നു; പതിപ്പ് വർദ്ധിപ്പിച്ചു.

Version: 2.0.0 - Released on 2022-08-05

* ultimo പേയ്‌മെന്റുകളെ അടിസ്ഥാനമാക്കി പ്രവർത്തിക്കുന്ന രീതിയിൽ പ്ലഗിൻ പുനർനിർമ്മിച്ചു.

Version: 2.0.0-beta.2 - Released on 2022-01-21

* ആന്തരികം: hooks ഉം filters ഉം ഉണ്ടാക്കുന്ന ജനറേറ്റർ ചേർത്തു;
* ആന്തരികം: ഡെവലപ്പർമാർക്ക് എളുപ്പത്തിനായി Multisite Ultimate stubs ചേർത്തു;
* പരിഹരിച്ചു: ഒരു കാർട്ടിൽ ഡിസ്‌കൗണ്ട് കോഡ് ഇല്ലാത്തപ്പോൾ വരുന്ന ഗുരുതരമായ പിശക്;
* പരിഹരിച്ചു: സന്ദർശകരെ ശരിയായി കൺവേർട്ട് ചെയ്യുന്നില്ല എന്ന പ്രശ്നം;

Version 2.0.0-beta.1 - Released on 2021-12-29

* 2.0 അടിസ്ഥാനമാക്കിയുള്ള പതിപ്പിന്റെ ആദ്യ റിലീസ്;
