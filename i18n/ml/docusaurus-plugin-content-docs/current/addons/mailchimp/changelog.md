# Mailchimp മാറ്റങ്ങളുടെ രേഖ (Changelog)

Version: 1.1.1 - 2025-09-28 ന് പുറത്തിറക്കി

* പ്രിഫിക്സ് (prefix) 'ultimate-multisite' എന്ന് മാറ്റുകയും; ടെക്സ്റ്റ് ഡൊമെയ്ൻ അപ്ഡേറ്റ് ചെയ്യുകയും; പതിപ്പ് വർദ്ധിപ്പിക്കുകയും ചെയ്തു.

Version 1.1.0 - 03/05/2018
- പരിഹരിച്ചു: നിരവധി ചെറിയ പ്രശ്നങ്ങളും ബഗ്ഗുകളും;
- കൂട്ടിച്ചേർത്തു: MailChimp ഗ്രൂപ്പുകൾ വഴി ഒറ്റ ലിസ്റ്റും സെഗ്മെന്റും ഉപയോഗിക്കാനുള്ള സൗകര്യം;
