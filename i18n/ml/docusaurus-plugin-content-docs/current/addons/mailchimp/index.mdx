# MailChimp

Multisite Ultimate: MailChimp സംയോജനം (Integration)

നിങ്ങളുടെ Multisite Ultimate നെറ്റ്വർക്കിനെ MailChimp-മായി ബന്ധിപ്പിക്കുക. ചില പ്ലാനുകൾ MailChimp ലിസ്റ്റുകളുമായി ബന്ധിപ്പിക്കാനും, യൂസർമാർ സൈൻ അപ്പ് ചെയ്യുമ്പോൾ ഓട്ടോമാറ്റിക്കായി അവരെ ആ ലിസ്റ്റുകളിൽ ചേർക്കാനും നിങ്ങൾക്ക് സാധിക്കും.

## ഇൻസ്റ്റാളേഷൻ (Installation) {#installation}

1. 'ultimate-multisite-mailchimp' എന്ന ഫയൽ '/wp-content/plugins/' ഡയറക്‌ടറിയിലേക്ക് അപ്‌ലോഡ് ചെയ്യുക.
2. നിങ്ങളുടെ WordPress Network Admin-ലെ 'Plugins' മെനുവിലൂടെ പ്ലഗിൻ ആക്ടിവേറ്റ് ചെയ്യുക.

## ഉപയോഗ രീതി (Usage Guide) {#usage-guide}

ഞങ്ങളുടെ Ultimate Multisite: MailChimp Integration Add-on ഉപയോഗിച്ച് നിങ്ങളുടെ സബ്‌സ്‌ക്രൈബർമാരുടെ ഇമെയിലുകൾ നിങ്ങളുടെ MailChimp ലിസ്റ്റുകളുമായി സിങ്ക് ചെയ്യുന്നത് വളരെ എളുപ്പമാണ്. എല്ലാം സെറ്റ് ചെയ്യാൻ സഹായിക്കുന്നതാണ് ഈ ട്യൂട്ടോറിയൽ!

## ഒരു API key നേടുന്നത് (Getting an API key) {#getting-an-api-key}

വിവിധ സെറ്റിംഗുകൾ എന്തൊക്കെയാണെന്ന് നോക്കുന്നതിന് മുൻപ്, നമുക്കൊരു MailChimp API key ആവശ്യമാണ്. യൂസർമാർ സൈൻ അപ്പ് ചെയ്യുമ്പോൾ ലിസ്റ്റുകൾ, ഗ്രൂപ്പുകൾ എന്നിവ എടുക്കാനും, ആ ലിസ്റ്റുകളിലും ഗ്രൂപ്പുകളിലും ഇമെയിൽ വിലാസങ്ങൾ ചേർക്കാനും Ultimate Multisite-ക്ക് MailChimp API യുമായി സംസാരിക്കാൻ ഇത് അവസരം നൽകുന്നു.

നിങ്ങളുടെ സ്വന്തം MailChimp API key നേടാൻ ഈ ട്യൂട്ടോറിയൽ പിന്തുടരുക: [About API Keys | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

<!-- Screenshot unavailable: MailChimp API key settings in Ultimate Multisite Add-on Settings -->

_നിങ്ങളുടെ MailChimp API key ഇവിടെ ചേർക്കുക_ _**Ultimate Multisite Settings - > Add-on Settings -> MailChimp**_

## സംയോജന രീതി (Integration Mode) {#integration-mode}

MailChimp Integration add-on രണ്ട് "സംയോജന രീതികൾ" (integration modes) നൽകുന്നു. അതായത്, ഇമെയിൽ വിലാസങ്ങൾ സിങ്ക് ചെയ്യാൻ നിങ്ങൾക്ക് രണ്ട് വ്യത്യസ്ത വഴികൾ ഉപയോഗിക്കാൻ കഴിയും.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_കൂടുതൽ വഴക്കത്തിനായി രണ്ട് സംയോജന രീതികൾ!_

### മോഡ് 1: ഒന്നിലധികം ലിസ്റ്റുകൾ (Multiple Lists) {#mode-1-multiple-lists}

ഡിഫോൾട്ട് മോഡ് **ഒന്നിലധികം ലിസ്റ്റുകൾ** ഉപയോഗിക്കുന്നു. ഈ മോഡ് ഉപയോഗിച്ച്, ഓരോ പ്ലാനിനും വേണ്ടി നിങ്ങൾക്ക് ഒന്നിലധികം MailChimp ലിസ്റ്റുകൾ തിരഞ്ഞെടുക്കാൻ കഴിയും. നിങ്ങളുടെ യൂസർമാരെ വ്യത്യസ്ത ലിസ്റ്റുകളായി വേർതിരിക്കണമെങ്കിൽ ഇത് ഏറ്റവും മികച്ച ഓപ്ഷനാണ്.

ഉദാഹരണത്തിന്, നിങ്ങളുടെ MailChimp അക്കൗണ്ടിൽ മൂന്ന് വ്യത്യസ്ത ലിസ്റ്റുകൾ ഉണ്ടെന്ന് കരുതുക: **Newsletter**, **Plan A**, കൂടാതെ **Plan B** എന്നിവയും. നിങ്ങളുടെ പ്ലാൻ എന്തുതന്നെയായാലും, എല്ലാ യൂസർമാരെയും Newsletter ലിസ്റ്റിൽ ഉൾപ്പെടുത്തണം, എങ്കിൽ മാത്രമേ അവർക്ക് നിങ്ങളുടെ അപ്‌ഡേറ്റുകൾ നഷ്ടമാകാതിരിക്കാൻ കഴിയൂ.

അത് നേടാൻ, നിങ്ങൾ Multiple Lists മോഡ് ഉപയോഗിച്ച് Plan A MailChimp Settings-ൽ **Newsletter** ലിസ്റ്റും **Plan A** ലിസ്റ്റും ചേർക്കുകയും, Plan B MailChimp Settings-ൽ **Newsletter** ലിസ്റ്റും **Plan B** ലിസ്റ്റും ചേർക്കുകയും ചെയ്യാം.

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_പുതിയ അക്കൗണ്ടുകൾക്കും റദ്ദാക്കിയ അക്കൗണ്ടുകൾക്കുമായി ഡിഫോൾട്ട് ലിസ്റ്റുകൾ തിരഞ്ഞെടുക്കുക_

### മോഡ് 2: ഒറ്റ ലിസ്റ്റ് + ഒന്നിലധികം ഗ്രൂപ്പുകൾ (Single List + Multiple Groups) {#mode-2-single-list--multiple-groups}

നിങ്ങളുടെ ലിസ്റ്റിനെ **ഗ്രൂപ്പുകൾ** എന്ന് വിളിക്കപ്പെടുന്ന ചെറിയ യൂസർ ഗ്രൂപ്പുകളായി വേർതിരിക്കാൻ MailChimp അനുവദിക്കുന്നു. ഇത് നിങ്ങൾക്ക് കൂടുതൽ നിയന്ത്രണം നൽകുന്നു, കാരണം MailChimp-ൽ ഒരേ കാമ്പെയ്‌നിന്റെ ഒന്നിലധികം കോപ്പികൾ ഉണ്ടാക്കേണ്ട ആവശ്യം വരാതെ, നിങ്ങളുടെ സബ്‌സ്‌ക്രൈബർമാരുടെ ഒരു പ്രത്യേക സബ്‌സെറ്റിന് ഒരു കാമ്പെയ്‌ൻ അയക്കാൻ കഴിയും (ഇത് പിന്നീട് ആ കാമ്പെയ്‌നിന്റെ മെട്രിക്‌സ് വിശകലനം ചെയ്യുന്നത് എളുപ്പമാക്കുന്നു).

ഉദാഹരണത്തിന്, **My Network Subscribers** എന്ന് പേരുള്ള ഒരു ഒറ്റ ലിസ്റ്റ് നിങ്ങൾക്ക് ഉണ്ടാക്കാം, അതിനുള്ളിൽ **Plan A** എന്നും **Plan B** എന്നും ഗ്രൂപ്പുകൾ ഉണ്ടാക്കാം.

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_“ഒറ്റ ലിസ്റ്റ് + ഒന്നിലധികം ഗ്രൂപ്പുകൾ” സംയോജന മോഡ് ഉപയോഗിക്കുകയാണെങ്കിൽ, ഡിഫോൾട്ട് ലിസ്റ്റുകളും ഗ്രൂപ്പുകളും തിരഞ്ഞെടുക്കുക_

## ഡിഫോൾട്ട് ലിസ്റ്റുകളും ഗ്രൂപ്പുകളും (Default Lists and Groups) {#default-lists-and-groups}

ഏത് സംയോജന രീതിയാണ് നിങ്ങളുടെ പ്രത്യേക ഉപയോഗത്തിന് കൂടുതൽ അനുയോജമെന്ന് നിങ്ങൾ തിരഞ്ഞെടുした後, ഡിഫോൾട്ട് ലിസ്റ്റുകൾ (കൂടാതെ മോഡിനനുസരിച്ച് ഗ്രൂപ്പുകളും) തിരഞ്ഞെടുക്കേണ്ട സമയമാണിത്. ഒരു പ്രത്യേക ലിസ്റ്റ് തിരഞ്ഞെടുക്കാത്ത പ്ലാനിന് വേണ്ടി ഡിഫോൾട്ട് ലിസ്റ്റ് ഉപയോഗിക്കപ്പെടും.

കൂടാതെ, റദ്ദാക്കിയ അക്കൗണ്ടുകൾക്കായുള്ള ഡിഫോൾട്ട് ലിസ്റ്റും നിങ്ങൾക്ക് തിരഞ്ഞെടുക്കാൻ കഴിയും. ഒരു യൂസർ തന്റെ സബ്‌സ്‌ക്രിപ്ഷൻ അവസാനിപ്പിച്ച് നിങ്ങളുടെ നെറ്റ്വർക്കിൽ നിന്ന് അക്കൗണ്ട് നീക്കം ചെയ്യുമ്പോൾ, അദ്ദേഹത്തിന്റെ ഇമെയിൽ വിലാസം റദ്ദാക്കിയ അക്കൗണ്ടുകൾക്കായി സെറ്റ് ചെയ്ത ലിസ്റ്റിലേക്ക് മാറും.

## ഡബിൾ സബ്‌സ്‌ക്രിപ്ഷൻ എനേബിൾ ചെയ്യൽ (Enabling Double Subscription) {#enabling-double-subscription}

അതേ സെറ്റിംഗ്സ് പേജിൽ, നിങ്ങൾക്ക് **Double Subscription** എനേബിൾ ചെയ്യാൻ കഴിയും. ഈ ഓപ്ഷൻ യൂസറെ ഡിഫോൾട്ട് ലിസ്റ്റിലും, ആ യൂസറിന്റെ പ്ലാനിനായി തിരഞ്ഞെടുക്കപ്പെട്ട ലിസ്റ്റിലും ചേർക്കും. ഈ ഓപ്ഷൻ എനേബിൾ ചെയ്തിട്ടില്ലെങ്കിൽ, യൂസറെ ആ യൂസറിന്റെ പ്ലാനിനായി തിരഞ്ഞെടുക്കപ്പെട്ട ലിസ്റ്റിൽ മാത്രമേ ചേർക്കുകയുള്ളൂ. ആ പ്രത്യേക പ്ലാനിനായി ഒന്നും സെറ്റ് ചെയ്തിട്ടില്ലെങ്കിൽ, യൂസറെ ഡിഫോൾട്ട് ലിസ്റ്റിൽ ചേർക്കും.

_സൈൻ അപ്പ് ചെയ്യുമ്പോൾ നിങ്ങളുടെ യൂസർമാരെ ഡിഫോൾട്ട് ലിസ്റ്റിലും പ്ലാനിന്റെ ലിസ്റ്റിലും സബ്‌സ്‌ക്രൈബ് ചെയ്യാൻ കഴിയും_

## ഓരോ പ്ലാനിനുവേണ്ടി ലിസ്റ്റുകളും ഗ്രൂപ്പുകളും തിരഞ്ഞെടുക്കൽ (Selecting Lists and Groups for each of the Plans) {#selecting-lists-and-groups-for-each-of-the-plans}

നിങ്ങളുടെ **Plans** എന്ന ഭാഗത്തേക്ക് പോവുക, എഡിറ്റ് ചെയ്യാൻ ആഗ്രഹിക്കുന്ന പ്ലാൻ തിരഞ്ഞെടുക്കുക. **Advanced Options** മെറ്റാ-ബോക്സിൽ, MailChimp എന്ന് പേരുള്ള ഒരു പുതിയ ടാബ് ലഭ്യമാണെന്ന് ശ്രദ്ധിക്കുക.

ആ ടാബിൽ, ആ പ്ലാനിനായി ഒന്നിലധികം ലിസ്റ്റുകളോ അല്ലെങ്കിൽ ഒറ്റ ലിസ്റ്റും ഒന്നിലധികം ഗ്രൂപ്പുകളോ (സംയോജന മോഡിനനുസരിച്ച്) നിങ്ങൾക്ക് തിരഞ്ഞെടുക്കാൻ കഴിയും.

_നിങ്ങളുടെ ഓരോ പ്ലാനിനുവേണ്ടി ഉപയോഗിക്കാൻ ആഗ്രഹിക്കുന്ന ലിസ്റ്റുകളും ഗ്രൂപ്പുകളും തിരഞ്ഞെടുക്കുക_

ഒരു പുതിയ യൂസർ ആ പ്ലാനിനായി സൈൻ അപ്പ് ചെയ്യുമ്പോൾ, അദ്ദേഹത്തിന്റെ ഇമെയിൽ വിലാസം തിരഞ്ഞെടുക്കപ്പെട്ട ലിസ്റ്റുകളിലും ഗ്രൂപ്പുകളിലും ചേർക്കപ്പെടും!
