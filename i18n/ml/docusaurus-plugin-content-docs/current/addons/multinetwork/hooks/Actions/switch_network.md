# Action: switch_network

നിലവിലെ നെറ്റ്വർക്ക് സന്ദർഭം (network context) മാറുമ്പോൾ ഇത് പ്രവർത്തിപ്പിക്കുന്നു.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $new_network_id | `int` | മാറിയേക്കാവുന്ന നെറ്റ്വർക്കിന്റെ ഐഡി (ID). |
| $old_network_id | `int` | മുമ്പ് നിലവിലുണ്ടായിരുന്ന നെറ്റ്വർക്കിന്റെ ഐഡി (ID). |
