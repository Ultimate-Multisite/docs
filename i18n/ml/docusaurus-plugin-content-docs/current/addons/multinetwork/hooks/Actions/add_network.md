# Action: add_network

ഒരു പുതിയ നെറ്റ്‌വർക്ക് ചേർした後 ഇത് പ്രവർത്തിപ്പിക്കുന്നു.

## പാരാമീറ്ററുകൾ {#parameters}

| Name | Type | വിവരണം |
|------|------|-------------|
| $new_network_id | `int` | ചേർത്ത നെറ്റ്‌വർക്കിന്റെ ID. |
| $r | `array` | നെറ്റ്‌വർക്ക് ആർഗ്യുമെന്റുകളുടെ മുഴുവൻ അസോസിയേറ്റീവ് അറേ. |
