# Action: update_network {#action-updatenetwork}

നിലവിലുള്ള ഒരു നെറ്റ്‌വർക്ക് അപ്ഡേറ്റ് ചെയ്തതിന് ശേഷം ഈ ആക്ഷൻ (action) പ്രവർത്തിപ്പിക്കുന്നു.

## പാരാമീറ്ററുകൾ {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $network_id | `int` | ചേർത്ത നെറ്റ്‌വർക്കിന്റെ ID. |
| $args | `array` | നെറ്റ്‌വർക്ക് ആർഗ്യുമെന്റുകളുടെ ഒരു അസോസിയേറ്റീവ് ആറേ (associative array). |
