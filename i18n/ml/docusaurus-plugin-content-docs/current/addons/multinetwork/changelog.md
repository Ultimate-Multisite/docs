# മൾട്ടി-നെറ്റ്‌വർക്ക് ചേഞ്ച്‌ലോഗ്

### 1.3.0
* പുതിയത്: നെറ്റ്‌വർക്ക് ടെംപ്ലേറ്റ് പ്രിവ്യൂവറ — വാങ്ങുന്നതിന് മുമ്പ് ലൈവ് പാനലിൽ നെറ്റ്‌വർക്ക് ടെംപ്ലേറ്റുകൾ ബ്രൗസ് ചെയ്യാനും പ്രിവ്യൂ ചെയ്യാനും കഴിയും.
* പുതിയത്: ചെക്ക്ഔട്ടിലെ നെറ്റ്‌വർക്ക് ടെംപ്ലേറ്റ് പിക്കർ — സൈൻഅപ്പ് ചെക്ക്ഔട്ട് പ്രക്രിയയ്ക്കിടെ ഉപഭോക്താക്കൾക്ക് ഒരു നെറ്റ്‌വർക്ക് ടെംപ്ലേറ്റ് തിരഞ്ഞെടുക്കാൻ കഴിയും.
* പുതിയത്: ക്രോസ്-നെറ്റ്‌വർക്ക് മാജിക് ലിങ്ക് SSO — മാജിക്-ലിങ്ക് ഓതന്റിക്കേഷൻ വഴി ഉപഭോക്താക്കൾക്ക് സബ്-നെറ്റ്‌വർക്കുകളിൽ തടസ്സമില്ലാതെ പ്രവേശിക്കാൻ കഴിയും.
* പുതിയത്: നെറ്റ്‌വർക്ക് മീഡിയ ക്ലോണിംഗ് — ഒരു ടെംപ്ലേറ്റിൽ നിന്ന് നെറ്റ്‌വർക്കുകൾ ക്ലോൺ ചെയ്യുമ്പോൾ സൈറ്റ് മീഡിയ ശരിയായി ഡ്യൂപ്ലിക്കേറ്റ് ചെയ്യപ്പെടുന്നു.
* പരിഹാരം: മൾട്ടി-ടെനൻസി അഡോൺ ഇല്ലാതെ നെറ്റ്‌വർക്ക് ക്ലോണിംഗ് ഇപ്പോൾ പ്രവർത്തിക്കുന്നു; ക്ലോൺ ചെയ്ത ശേഷം സൂപ്പർ അഡ്മിൻ പ്രിവിലേജുകൾ വീണ്ടും നൽകുന്നു.
* പരിഹാരം: നെറ്റ്‌വർക്ക് സൃഷ്ടിക്കൽ ഇപ്പോൾ WooCommerce-ഉം എല്ലാ പേയ്‌മെന്റ് ഗേറ്റ്‌വേകളും സപ്പോർട്ട് ചെയ്യുന്നു.
* പരിഹാരം: ഫോൾബാക്ക് ഡൊമെയ്ൻ ജനറേഷനിൽ പോർട്ട് നമ്പർ ഇപ്പോൾ ഉൾപ്പെടുത്തിയിരിക്കുന്നു.
* പരിഹാരം: എല്ലാ തീമുകളിലും ശരിയായ റെൻഡറിംഗിനായി ടെംപ്ലേറ്റ് പിക്കർ CSS ഒരു എൻക്യൂ ചെയ്ത സ്റ്റൈൽഷീറ്റിലേക്ക് മാറ്റി.
* പരിഹാരം: സൈറ്റ് ക്വറി സ്കോപ്പിലെ നിർവചിക്കാത്ത `network_id` കീക്കെതിരെ സംരക്ഷണം നൽകി.
* പരിഹാരം: കോൺടെക്സ്റ്റ് സ്വിച്ചിംഗിന് മുമ്പുള്ള നെറ്റ്‌വർക്ക് പെർമിഷൻ ചെക്ക് `wpdb` ടേബിൾ പ്രോപ്പർട്ടി പിശകുകൾ തടയുന്നു.
