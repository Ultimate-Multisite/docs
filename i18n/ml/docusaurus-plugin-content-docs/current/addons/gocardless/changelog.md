# GoCardless മാറ്റങ്ങൾ (Changelog)

Version: 1.0.3 - Released on 2026-05-05
* പരിഹരിച്ചു: ഒരു GoCardless membership 'pending' അല്ലെങ്കിൽ 'on-hold' സ്റ്റേറ്റിൽ നിന്ന് റദ്ദാക്കുമ്പോൾ, അനാഥമായ (orphaned) pending സൈറ്റുകൾ ഇപ്പോൾ വൃത്തിയാക്കപ്പെടുന്നു.
* മെച്ചപ്പെടുത്തി: Bedrock root autoloader ഇതിനകം dependencies ലോഡ് ചെയ്തിട്ടുണ്ടെങ്കിൽ, Plugin autoloader പ്രവർത്തിപ്പിക്കുന്നത് ഒഴിവാക്കിയിരിക്കുന്നു.

Version: 1.0.1 - Released on 2025-09-28

* പ്രിഫിക്സ് ultimate-multisite എന്ന് മാറ്റിച്ചു; ടെക്സ്റ്റ് ഡൊമെയ്ൻ അപ്ഡേറ്റ് ചെയ്തു; വേർഷൻ നമ്പർ വർദ്ധിപ്പിച്ചു.

2021-09-24 - Version 1.0.0-beta.4

* കൂട്ടിച്ചേർത്തു: mu-plugins അടിസ്ഥാനമാക്കിയുള്ള സെറ്റപ്പുകൾക്കായി wp_ultimo_skip_network_active_check എന്ന filter ചേർത്തു;

2021-06-16 - Version 1.0.0-beta.3

* പരിഹരിച്ചു: സിഗ്നേച്ചറുകൾ സ്ഥിരീകരിക്കാൻ കഴിയാത്ത Webhook handlers;

2021-06-14 - Version 1.0.0-beta.2
