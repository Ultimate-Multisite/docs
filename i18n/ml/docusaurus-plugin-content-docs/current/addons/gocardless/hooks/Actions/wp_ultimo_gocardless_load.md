# Action: wp_ultimo_gocardless_load {#action-wpultimogocardlessload}

എല്ലാ ഡിപൻഡൻസികളും (dependencies) ലോഡ് ചെയ്യുമ്പോൾ ഇത് പ്രവർത്തിപ്പിക്കുന്നു.

പ്ലഗിൻ ഡെവലപ്പർമാർക്ക് പുതിയ ഫംഗ്ഷണാലിറ്റി കൂട്ടിച്ചേർക്കാൻ ഇത് അനുവദിക്കുന്നു. ഉദാഹരണത്തിന്, പുതിയ ഹോസ്റ്റിംഗ് പ്രൊവൈഡർമാർക്കുള്ള സപ്പോർട്ട് നൽകാൻ.

- 1.0.0
