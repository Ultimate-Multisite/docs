# ഫിൽട്ടർ: wp_ultimo_skip_network_active_check

ഡെവലപ്പർമാർക്ക് ഈ പരിശോധന ഒഴിവാക്കാൻ (short-circuit) ഇത് അനുവദിക്കുന്നു.

composer-based ആയതും Bedrock പോലുള്ള മറ്റ് കസ്റ്റം സജ്ജീകരണങ്ങൾ ഉപയോഗിക്കുമ്പോഴും ഇത് വളരെ ഉപകാരപ്രദമാണ്. കാരണം, അത്തരം സാഹചര്യങ്ങളിൽ mu-plugins ആയി പ്ലഗിനുകൾ ഉപയോഗിക്കുന്നത് സാധാരണമാണ്.

### ലഭ്യമായത്

- 2.0.0
### ഉറവിടം

[`inc/class-requirements.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-gocardless/blob/main/inc/class-requirements.php#L229) എന്ന ഫയലിലെ 229-ാം വരിയിലാണ് ഇത് നിർവചിച്ചിരിക്കുന്നത്.
