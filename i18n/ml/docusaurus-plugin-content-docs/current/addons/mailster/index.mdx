# Mailster {#mailster}

Ultimate Multisite Mailster Integration addon ഉപയോഗിച്ച് നിങ്ങളുടെ multisite network-നെ ഒരു ശക്തമായ ഇമെയിൽ ലിസ്റ്റ് നിർമ്മാണ യന്ത്രമാക്കി മാറ്റുക. ഉപയോക്താക്കൾ സൈൻ അപ്പ് ചെയ്യുമ്പോഴോ പർച്ചേസ് ചെയ്യുമ്പോഴോ പുതിയ ഉപഭോക്താക്കളെ ഓട്ടോമാറ്റിക്കായി നിങ്ങളുടെ Mailster ഇമെയിൽ ലിസ്റ്റുകളിലേക്ക് ചേർക്കുന്നു. ഇത് നിങ്ങളുടെ membership പ്ലാറ്റ്‌ഫോമും ഇമെയിൽ മാർക്കറ്റിംഗ് തന്ത്രവും തമ്മിൽ ഒരു തടസ്സമില്ലാത്ത ബന്ധം സൃഷ്ടിക്കുന്നു.

## പ്രധാന സവിശേഷതകൾ {#key-features}

**ഓട്ടോമാറ്റിക് സബ്‌സ്‌ക്രൈബർ Sync**
* സൈൻ അപ്പ് ചെയ്യുമ്പോൾ ഉപഭോക്താക്കളെ ഓട്ടോമാറ്റിക്കായി Mailster ലിസ്റ്റുകളിലേക്ക് ചേർക്കുന്നു
* സൈൻ അപ്പ് ചെയ്യുമ്പോൾ തന്നെ sync ചെയ്യണമോ അതോ പേയ്‌മെന്റ് സ്ഥിരീകരണം വരെ കാത്തിരിക്കണമോ തിരഞ്ഞെടുക്കാം
* മാനുവൽ ഇടപെടൽ കൂടാതെ നിങ്ങളുടെ ഇമെയിൽ ലിസ്റ്റുകൾ പുതിയതും അപ്‌ഡേറ്റായതുമായി നിലനിർത്തുക

**ഫ്ലെക്സിബിൾ ലിസ്റ്റ് മാനേജ്‌മെന്റ്**
* എല്ലാ പുതിയ ഉപഭോക്താക്കൾക്കുമായി ഗ്ലോബൽ ഡിഫോൾട്ട് ലിസ്റ്റുകൾ സജ്ജമാക്കുക
* ടാർഗെറ്റഡ് കാമ്പെയ്‌നുകൾക്കായി ഉൽപ്പന്ന-നിർദ്ദിഷ്ട ലിസ്റ്റുകൾ ഉപയോഗിച്ച് ഡിഫോൾട്ടുകൾ മാറ്റാൻ കഴിയും
* Mailster-ൻ്റെ പരിചിതമായ ഇന്റർഫേസിലൂടെ എളുപ്പത്തിൽ ലിസ്റ്റ് അസൈൻമെന്റുകൾ കൈകാര്യം ചെയ്യുക

**GDPR-കംപ്ലയിന്റ് ഓപ്റ്റ്-ഇൻ ഓപ്ഷനുകൾ**
* ഓട്ടോമാറ്റിക് സബ്‌സ്‌ക്രിപ്ഷൻ അല്ലെങ്കിൽ ചെക്ക്ബോക്സ് അടിസ്ഥാനമാക്കിയുള്ള സമ്മതം എന്നിവയിൽ നിന്ന് തിരഞ്ഞെടുക്കുക
* സ്ഥിരീകരിച്ച സബ്‌സ്‌ക്രിപ്‌ഷനുകൾക്കായി ഡബിൾ ഓപ്റ്റ്-ഇൻ പ്രവർത്തനക്ഷമമാക്കുക
* നിങ്ങളുടെ കംപ്ലയൻസ് ആവശ്യകതകൾ അനുസരിച്ച് ഓപ്റ്റ്-ഇൻ ചെക്ക്ബോക്സ് കാണിക്കുക/മറിക്കുക

**സ്മാർട്ട് ഡാറ്റ മാപ്പിംഗ്**
* ഉപഭോക്താക്കളുടെ പേരുകളും കോൺടാക്റ്റ് വിവരങ്ങളും ഓട്ടോമാറ്റിക്കായി sync ചെയ്യുക
* ബില്ലിംഗ് വിലാസങ്ങൾ സബ്‌സ്‌ക്രൈബർ ഫീൽഡുകളുമായി മാപ്പ് ചെയ്യുക
* ടാർഗെറ്റഡ് കാമ്പെയ്‌നുകൾക്കായി സൈൻ അപ്പ് തീയതികൾ ട്രാക്ക് ചെയ്യുക

**ഉൽപ്പന്ന-തല നിയന്ത്രണം**
* വ്യത്യസ്ത ഉൽപ്പന്നങ്ങൾക്ക് വ്യത്യസ്ത ലിസ്റ്റുകൾ നൽകുക
* ഉപഭോക്താക്കൾ എന്ത് വാങ്ങുന്നു എന്നതിനെ അടിസ്ഥാനമാക്കി ടാർഗെറ്റഡ് കാമ്പെയ്‌നുകൾ സൃഷ്ടിക്കുക
* ആവശ്യമുള്ളപ്പോൾ പ്രത്യേക ഉൽപ്പന്നങ്ങൾക്ക് Mailster പ്രവർത്തനരഹിതമാക്കുക

**പ്രൊഫഷണൽ അനുഭവം**
* ലിസ്റ്റ് തിരഞ്ഞെടുക്കുന്നതിനായി Native Mailster UI സംയോജനം
* ട്രബിൾഷൂട്ടിംഗിനും വെരിഫിക്കേഷനുമുള്ള വിശദമായ ലോഗിംഗ്
* നിലവിലുള്ള സബ്‌സ്‌ക്രൈബർമാരെ തടസ്സമില്ലാതെ അപ്‌ഡേറ്റ് ചെയ്യുക

## ആർക്കൊക്കെ ഉപയോഗപ്രദമാകും {#perfect-for}

* ഇമെയിൽ ലിസ്റ്റുകൾ നിർമ്മിക്കുന്ന മെമ്പർഷിപ്പ് സൈറ്റ് ഉടമകൾ
* പുതിയ ഉപഭോക്താക്കളെ പരിപോഷിപ്പിക്കാൻ ആഗ്രഹിക്കുന്ന SaaS പ്ലാറ്റ്‌ഫോമുകൾ
* എൻറോൾമെൻ്റ് അനുസരിച്ച് വിദ്യാർത്ഥികളെ വേർതിരിക്കുന്ന കോഴ്സ് ക്രിയേറ്റർമാർ
* Mailster ഉപയോഗിച്ച് Ultimate Multisite പ്രവർത്തിപ്പിക്കുന്ന ആർക്കും

## ഇത് എങ്ങനെ പ്രവർത്തിക്കുന്നു {#how-it-works}

1. നിങ്ങളുടെ Ultimate Multisite network-ൽ ആഡ്‌ഓൺ ഇൻസ്റ്റാൾ ചെയ്യുകയും ആക്ടിവേറ്റ് ചെയ്യുകയും ചെയ്യുക
2. Ultimate Multisite ക്രമീകരണങ്ങളിൽ ഗ്ലോബൽ ഡിഫോൾട്ട് ലിസ്റ്റുകൾ കോൺഫിഗർ ചെയ്യുക
3. ആവശ്യമെങ്കിൽ ഉൽപ്പന്ന-നിർദ്ദിഷ്ട ലിസ്റ്റുകൾ ഉപയോഗിച്ച് മാറ്റങ്ങൾ വരുത്തുക
4. നിങ്ങളുടെ ഓപ്റ്റ്-ഇൻ മോഡും സബ്‌സ്‌ക്രിപ്ഷൻ സമയവും തിരഞ്ഞെടുക്കുക
5. നിങ്ങളുടെ Mailster ലിസ്റ്റുകൾ ഓട്ടോമാറ്റിക്കായി വളരുന്നത് കാണുക!

## ഇൻസ്റ്റാളേഷൻ {#installation}

1. പ്ലഗിൻ ഫയലുകൾ നിങ്ങളുടെ Ultimate Multisite addons ഡയറക്‌ടറിയിലേക്ക് അപ്‌ലോഡ് ചെയ്യുക
2. നിങ്ങളുടെ പ്രധാന സൈറ്റിൽ Mailster ഇൻസ്റ്റാൾ ചെയ്യുകയും ആക്റ്റീവ് ആണെന്ന് ഉറപ്പാക്കുക
3. Ultimate Multisite addons പേജ് വഴി ആഡ്‌ഓൺ ആക്ടിവേറ്റ് ചെയ്യുക
4. കോൺഫിഗർ ചെയ്യാൻ Settings > Mailster Integration എന്നതിലേക്ക് പോകുക
5. നിങ്ങളുടെ ഡിഫോൾട്ട് ലിസ്റ്റുകൾ തിരഞ്ഞെടുക്കുകയും ഓപ്റ്റ്-ഇൻ മുൻഗണനകൾ കോൺഫിഗർ ചെയ്യുകയും ചെയ്യുക

## പതിവായി ചോദിക്കുന്ന ചോദ്യങ്ങൾ {#frequently-asked-questions}

## Ultimate Multisite ഇല്ലാതെ ഈ ആഡ്‌ഓൺ പ്രവർത്തിക്കുമോ? {#does-this-addon-work-without-ultimate-multisite}

ഇല്ല, ഈ ആഡ്‌ഓൺ പ്രവർത്തിക്കാൻ Ultimate Multisite ആവശ്യമാണ്. Ultimate Multisite-ൻ്റെ ഉപഭോക്തൃ, ഉൽപ്പന്ന സംവിധാനവുമായി Mailster സംയോജിപ്പിക്കാനാണ് ഇത് പ്രത്യേകമായി രൂപകൽപ്പന ചെയ്തിരിക്കുന്നത്.

## എല്ലാ സൈറ്റുകളിലും എനിക്ക് Mailster ഇൻസ്റ്റാൾ ചെയ്യേണ്ട কি? {#do-i-need-mailster-installed-on-all-sites}

ഇല്ല, Mailster നിങ്ങളുടെ പ്രധാന നെറ്റ്‌വർക്ക് സൈറ്റിൽ ഇൻസ്റ്റാൾ ചെയ്യുകയും ആക്റ്റീവ് ആക്കുകയും ചെയ്താൽ മതി. എല്ലാ സബ്‌സൈറ്റുകളിൽ നിന്നും ആഡ്‌ഓൺ അതിലേക്ക് ഓട്ടോമാറ്റിക്കായി കണക്ട് ചെയ്യും.

## വിവിധ ഉൽപ്പന്നങ്ങൾക്ക് വ്യത്യസ്ത ലിസ്റ്റുകൾ നൽകാൻ കഴിയുമോ? {#can-i-assign-different-lists-to-different-products}

അതെ! നിങ്ങൾക്ക് ഗ്ലോബൽ ഡിഫോൾട്ട് ലിസ്റ്റുകൾ സജ്ജമാക്കാനും, അതിനുശേഷം ഓരോ ഉൽപ്പന്നത്തിനും പ്രത്യേകമായി മാറ്റങ്ങൾ വരുത്താനും കഴിയും. ഉപഭോക്താക്കൾ എന്ത് വാങ്ങുന്നു എന്നതിനെ അടിസ്ഥാനമാക്കി ഇത് വളരെ ടാർഗെറ്റഡ് ഇമെയിൽ കാമ്പെയ്‌നുകൾ സൃഷ്ടിക്കാൻ നിങ്ങളെ അനുവദിക്കുന്നു.

## ഒരു ഉപഭോക്താവ് ഇതിനകം Mailster-ൽ ഉണ്ടെങ്കിൽ എന്ത് സംഭവിക്കും? {#what-happens-if-a-customer-already-exists-in-mailster}

ഡിഫോൾട്ടായി, നിലവിലുള്ള സബ്‌സ്‌ക്രൈബർമാരെ ആഡ്‌ഓൺ അപ്‌ഡേറ്റ് ചെയ്യുകയും പുതിയ ലിസ്റ്റുകളിലേക്ക് അവരെ ചേർക്കുകയും ചെയ്യും. നിലവിലുള്ള സബ്‌സ്‌ക്രൈബർമാരെ ഒഴിവാക്കാൻ നിങ്ങൾ ആഗ്രഹിക്കുന്നുവെങ്കിൽ, നിങ്ങൾക്ക് ഇത് ക്രമീകരണങ്ങളിൽ പ്രവർത്തനരഹിതമാക്കാം.

## ഇത് GDPR-കംപ്ലയിന്റ് ആണോ? {#is-this-gdpr-compliant}

അതെ! ഉപഭോക്താക്കൾ ഇമെയിലുകൾ സ്വീകരിക്കാൻ വ്യക്തമായി സമ്മതം നൽകേണ്ട ഒരു ചെക്ക്ബോക്സ് അടിസ്ഥാനമാക്കിയുള്ള ഓപ്റ്റ്-ഇൻ മോഡ് ഈ ആഡ്‌ഓണിൽ ഉൾപ്പെടുന്നു. ശരിയായ സ്വകാര്യ നയങ്ങളുമായി സംയോജിപ്പിക്കുമ്പോൾ ഇത് GDPR ആവശ്യകതകൾ പാലിക്കുന്നു.

## എനിക്ക് ഡബിൾ ഓപ്റ്റ്-ഇൻ ഉപയോഗിക്കാൻ കഴിയുമോ? {#can-i-use-double-opt-in}

തീർച്ചയായും! ഡബിൾ ഓപ്റ്റ്-ഇൻ സെറ്റിംഗ് പ്രവർത്തനക്ഷമമാക്കുക, സബ്‌സ്‌ക്രിപ്ഷനുകൾ സജീവമാക്കുന്നതിന് മുമ്പ് Mailster സ്ഥിരീകരണ ഇമെയിലുകൾ അയയ്ക്കും.
