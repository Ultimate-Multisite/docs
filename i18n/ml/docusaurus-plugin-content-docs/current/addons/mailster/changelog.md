# Mailster മാറ്റങ്ങളുടെ പട്ടിക (Changelog)

### 1.0.0
* ആദ്യ റിലീസ് (Initial release)
* സൈൻഅപ്പ് ചെയ്യുമ്പോഴോ പണം അടയ്ക്കുമ്പോഴോ ഓട്ടോമാറ്റിക് സബ്‌സ്‌ക്രൈബർ sync
* ഉൽപ്പന്നം അടിസ്ഥാനമാക്കിയുള്ള ലിസ്റ്റ് അസൈൻമെന്റുകൾ (Product-specific list assignments)
* GDPR പാലിക്കലിനായുള്ള ചെക്ക്ബോക്സ് ഓപ്റ്റ്-ഇൻ (Checkbox-based opt-in for GDPR compliance)
* ഡബിൾ ഓപ്റ്റ്-ഇൻ സപ്പോർട്ട് (Double opt-in support)
* സമഗ്രമായ ഫീൽഡ് മാപ്പിംഗ് (Comprehensive field mapping)
* വിശദമായ ലോഗിംഗ് (Detailed logging)
