# അളക്കപ്പെട്ട പ്ലാനുകളുടെ മാറ്റങ്ങളുടെ രേഖ (Metered Plans Changelog) {#metered-plans-changelog}

Version 1.1.0 - പുറത്തിറങ്ങിയ തീയതി: 2026-05-05
- പുതിയത്: മൾട്ടിസൈറ്റ് സബ്സൈറ്റുകൾക്കുള്ള AI ടോക്കൺ ബില്ലിംഗ് — കോൺഫിഗറബിൾ ടോക്കൺ നിരക്കുകളോടെ ഉപഭോക്തൃ സൈറ്റുകളിലുടനീളം AI ടോക്കൺ ഉപയോഗം ട്രാക്ക് ചെയ്യാനും ബിൽ ചെയ്യാനും സാധിക്കും.
- പുതിയത്: കണക്റ്റർ എൻഫോഴ്സ്മെന്റ് ഡൈനാമിക് ലിമിറ്റ് ഡിസ്കവറിയോടും റൈറ്റ്-ത്രൂ സംവിധാനത്തോടും കൂടി പുനർനിർമ്മിച്ചു. എല്ലാ കണക്റ്ററുകളിലും തത്സമയ കൃത്യത ഉറപ്പാക്കുന്നു.
- പരിഹരിച്ചു: ഡാറ്റാബേസ് ടേബിൾ അപ്ഗ്രേഡുകൾ ഇപ്പോൾ ബെർലിൻഡിബി (BerlinDB) സ്കീമ നിർവചനങ്ങളുമായി ശരിയായി യോജിച്ചു, പുതിയ ഇൻസ്റ്റാളേഷനുകളിൽ അപ്ഗ്രേഡ് പരാജയങ്ങൾ ഒഴിവാക്കുന്നു.
- പരിഹരിച്ചു: ഡാറ്റാബേസ് അപ്ഗ്രേഡ് കോൾബാക്കുകൾ ശരിയായ ഫോർമാറ്റിലേക്ക് മാറ്റിയത്, ഇത് നിശബ്ദമായ അപ്ഗ്രേഡ് പരാജയങ്ങൾ പരിഹരിച്ചു.
- പരിഹരിച്ചു: AI Usage Overage Markup ഇൻപുട്ട് ഫീൽഡിൽ ഭിന്ന മൂല്യങ്ങൾ (Fractional values) ഇപ്പോൾ സ്വീകരിക്കുന്നു.
- പരിഹരിച്ചു: പ്ലഗ്ഇൻ ലോഡ് ചെയ്യുമ്പോൾ ഉണ്ടായ ഫറ്റൽ എററുകളും ഡബിൾ-ഇനിഷ്യലൈസേഷൻ പ്രശ്നങ്ങളും പരിഹരിച്ചു.
- മെച്ചപ്പെടുത്തി: check-env npm സ്ക്രിപ്റ്റ് ചേർത്തു. ഇത് ഡെവലപ്പർ എൻവയോൺമെന്റുകൾ ആദ്യമായി പ്രവർത്തിപ്പിക്കുമ്പോൾ സ്വയം കോൺഫിഗർ ചെയ്യാൻ സഹായിക്കുന്നു.
