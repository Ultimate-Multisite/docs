# Fluent Forms {#fluent-forms}

ഈ അഡ്-ഓൺ ഉപയോഗിച്ച് Ultimate Multisite-ൽ ഉൽപ്പന്നങ്ങൾ, അംഗത്വങ്ങൾ അല്ലെങ്കിൽ സൈറ്റുകൾ എന്നിവയിൽ Fluent Forms-ന് പരിമിതികൾ (limits) ഏർപ്പെടുത്താൻ സാധിക്കും.
Fluent Forms കസ്റ്റം പോസ്റ്റ് ടൈപ്പുകൾക്ക് പകരം കസ്റ്റം ടേബിളുകൾ ഉപയോഗിക്കുന്നതിനാൽ, സാധാരണ പരിമിതപ്പെടുത്താനുള്ള ഓപ്ഷനുകൾ Fluent Forms-മായി പ്രവർത്തിക്കില്ല.

## സവിശേഷതകൾ (Features) {#features}

- Fluent Forms അല്ലെങ്കിൽ Conversational Fluent Forms എന്നിവയ്ക്ക് പരിമിതികൾ ഏർപ്പെടുത്താം.
- പരിധിയിലെത്തുന്ന ഉപഭോക്താക്കൾക്ക്, അവരുടെ പ്ലാൻ അപ്ഗ്രേഡ് ചെയ്യാൻ ആവശ്യപ്പെടുന്ന ഒരു സൗഹൃദ സന്ദേശം ലഭിക്കും.
