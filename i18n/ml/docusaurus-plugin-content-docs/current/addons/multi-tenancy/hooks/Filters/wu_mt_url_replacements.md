# Filter: wu_mt_url_replacements {#filter-wumturlreplacements}

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $replacements | `array` | നിലവിലുള്ള മാറ്റിസ്ഥാപിക്കാനുള്ള ജോഡികൾ. |
| $old_url | `string` | പഴയ URL. |
| $new_url | `string` | പുതിയ URL. |
| $blog_id | `int` | ബ്ലോഗ് ഐഡി. |
