# മൾട്ടി-ടെനൻസി

വേഗതയ്ക്കും സുരക്ഷയ്ക്കുമായി സബ്സൈറ്റുകൾ സെർവറുകളിലും ഡാറ്റാബേസുകളിലും വിതരണം ചെയ്യുക.

## ആവശ്യകതകൾ

- WordPress 5.3 അല്ലെങ്കിൽ അതിലും ഉയർന്നത്
- PHP 7.4 അല്ലെങ്കിൽ അതിലും ഉയർന്നത്
- Multisite Ultimate പ്ലഗിൻ (active)

## ഇൻസ്റ്റാളേഷൻ

1. നിങ്ങളുടെ `/wp-content/plugins/` ഡയറക്‌ടറിയിലേക്ക് ആഡ്‌ഓൺ ഫയലുകൾ അപ്‌ലോഡ് ചെയ്യുക.
2. വേർഡ്രസ്സിലെ 'Plugins' മെനുവിലൂടെ പ്ലഗിൻ ആക്ടിവേറ്റ് ചെയ്യുക.
3. നെറ്റ്‌വർക്ക് അഡ്മിനിൽ ആഡ്‌ഓൺ സെറ്റിംഗുകൾ കോൺഫിഗർ ചെയ്യുക.

## ഡെവലപ്‌മെന്റ്

### സജ്ജീകരണം

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

```

```

### ടെസ്റ്റിംഗ്

ഈ ആഡ്‌ഓണിൽ ഒരു സമഗ്രമായ ടെസ്റ്റിംഗ് ഫ്രെയിംവർക്ക് ഉൾപ്പെടുന്നു:

- **Unit Tests**: വേർഡ്രസ് ടെസ്റ്റ് സ്യൂട്ടുമായി ബന്ധപ്പെട്ട PHPUnit അടിസ്ഥാനമാക്കിയുള്ള ടെസ്റ്റുകൾ.
- **Base Test Class**: സാധാരണ ടെസ്റ്റ് സാഹചര്യങ്ങൾക്കായുള്ള സഹായ രീതികൾ (helper methods) നൽകുന്നു.
- **Test Coverage**: കോഡ് കവറേജ് ട്രാക്ക് ചെയ്യാൻ സജ്ജീകരിച്ചിരിക്കുന്നു.
- **Multisite Testing**: മൾട്ടിസൈറ്റ് എൻവയോൺമെന്റിൽ ടെസ്റ്റുകൾ പ്രവർത്തിപ്പിക്കുന്നു.

### കോഡ് സ്റ്റാൻഡേർഡുകൾ

- **WordPress Coding Standards**: PHPCS വഴി നടപ്പിലാക്കുന്നു.
- **PHP 7.4+ അനുയോജ്യത**: ആധുനിക PHP ഫീച്ചറുകൾ പിന്തുണയ്ക്കുന്നു.
- **Static Analysis**: ടൈപ്പ് ചെക്കിംഗിനായി PHPStan സംയോജിപ്പിക്കുന്നു.
- **Code Modernization**: PHP അപ്‌ഗ്രേഡുകൾക്കായുള്ള Rector നിയമങ്ങൾ.

### ബിൽഡ് സിസ്റ്റം

- **Asset Processing**: CSS/JS ഫയലുകൾക്ക് മിനിഫിക്കേഷൻ.
- **Translation**: POT ഫയൽ ജനറേഷൻ.
- **Packaging**: വിതരണത്തിനായി ആർക്കൈവ് സൃഷ്ടിക്കൽ.
- **Development Mode**: ഡീബഗ്ഗിംഗിനായി മിനിഫൈ ചെയ്യാത്ത ആസ്തികൾ.

## കോൺഫിഗറേഷൻ

നെറ്റ്‌വർക്ക് അഡ്മിൻ ഇന്റർഫേസ് വഴി ലഭ്യമാകുന്ന വിവിധ കോൺഫിഗറേഷൻ ഓപ്ഷനുകൾ ഈ ആഡ്‌ഓൺ പിന്തുണയ്ക്കുന്നു.

## Hooks ഉം Filters ഉം

### Actions

- `ultimate-multisite-addon-template_init` - ആഡ്‌ഓൺ ഇനിഷ്യലൈസേഷൻ കഴിഞ്ഞ ശേഷം പ്രവർത്തിപ്പിക്കുന്നു.
- `ultimate-multisite-addon-template_loaded` - എല്ലാ ആഡ്‌ഓൺ ഫയലുകളും ലോഡ് ചെയ്ത ശേഷം പ്രവർത്തിപ്പിക്കുന്നു.

### Filters

- `ultimate-multisite-addon-template_settings` - ആഡ്‌ഓൺ സെറ്റിംഗുകൾ മാറ്റാൻ.
- `ultimate-multisite-addon-template_enabled` - ആഡ്‌ഓൺ പ്രവർത്തനക്ഷമമാക്കാനുള്ള/നിർജ്ജീവമാക്കാനുള്ള സ്റ്റാറ്റസ് മാറ്റാൻ.
