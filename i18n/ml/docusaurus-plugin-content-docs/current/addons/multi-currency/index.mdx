# Multi-Currency {#multi-currency}

Ultimate Multisite: Multi-Currency ഉപയോഗിച്ച് നിങ്ങളുടെ നെറ്റ്വർക്ക് ഒന്നിലധികം കറൻസികളിൽ പേയ്‌മെന്റുകൾ സ്വീകരിക്കാൻ സാധിക്കും. ഇത് ലൈവ് അല്ലെങ്കിൽ മാനുവൽ എക്സ്ചേഞ്ച് നിരക്കുകൾ ഉപയോഗിച്ച് സ്വയമേവ വിലകൾ മാറ്റുന്നു, ഏത് കറൻസിയിലുമുള്ള ഉൽപ്പന്നങ്ങൾക്ക് കൃത്യമായ വിലകൾ സജ്ജമാക്കാൻ അനുവദിക്കുന്നു, കൂടാതെ ജിയോലൊക്കേഷൻ വഴി ഓരോ സന്ദർശകന്റെയും ഇഷ്ടപ്പെട്ട കറൻസി കണ്ടെത്തുന്നു — ഇതെല്ലാം ഒരു single admin dashboard-ൽ നിന്ന് തന്നെ കൈകാര്യം ചെയ്യാം.

## പ്രധാന സവിശേഷതകൾ (Key Features) {#key-features}

- **ഓട്ടോമാറ്റിക് എക്സ്ചേഞ്ച് റേറ്റ് കൺവേർഷൻ** — നിങ്ങൾ തിരഞ്ഞെടുത്ത പ്രൊവൈഡറിൽ നിന്ന് ലഭിക്കുന്ന നിരക്കുകൾ ഉപയോഗിച്ച് നിങ്ങളുടെ ബേസ് കറൻസിയിൽ നിന്ന് വിലകൾ മാറ്റുന്നു.
- **ഉൽപ്പന്നമനുസരിച്ചുള്ള വില മാറ്റങ്ങൾ (Per-product pricing overrides)** — ഓട്ടോമാറ്റിക് കൺവേർഷൻ ഒഴിവാക്കി ഏത് ഉൽപ്പന്നത്തിലും കറൻസി-നിർദ്ദിഷ്ടമായ കൃത്യമായ വിലകൾ സജ്ജമാക്കാം.
- **കറൻസി സെലക്ടർ സൈൻഅപ്പ് ഫീൽഡ്** — ചെക്ക്ഔട്ട്/സൈൻഅപ്പ് സമയത്ത് ഉപഭോക്താക്കൾക്ക് അവരുടെ ഇഷ്ടപ്പെട്ട കറൻസി തിരഞ്ഞെടുക്കാൻ ഇത് സഹായിക്കുന്നു.
- **ഒന്നിലധികം എക്സ്ചേഞ്ച് റേറ്റ് പ്രൊവൈഡർമാർ** — ECB, Frankfurter, Stripe, അല്ലെങ്കിൽ പൂർണ്ണമായും മാനുവൽ നിരക്കുകൾ.
- **ജിയോലൊക്കേഷൻ അടിസ്ഥാനമാക്കിയുള്ള കറൻസി കണ്ടെത്തൽ** — സന്ദർശകന്റെ രാജ്യത്തോട് ചേരുന്ന കറൻസി സ്വയമേവ തിരഞ്ഞെടുക്കുന്നു.
- **admin dashboard** — സജീവമായ കറൻസികൾ, എക്സ്ചേഞ്ച് നിരക്കുകൾ, പ്രൊവൈഡർ ക്രമീകരണങ്ങൾ എന്നിവ ഒരിടത്ത് നിന്ന് കൈകാര്യം ചെയ്യാം.

## ആവശ്യകതകൾ (Requirements) {#requirements}

- Ultimate Multisite 2.0.0 അല്ലെങ്കിൽ അതിലും ഉയർന്നത്
- WordPress Multisite ഇൻസ്റ്റാളേഷൻ
- PHP 7.4 അല്ലെങ്കിൽ അതിലും ഉയർന്നത്

## ഇൻസ്റ്റാളേഷൻ (Installation) {#installation}

1. പ്ലഗിൻ ഫയലുകൾ `/wp-content/plugins/ultimate-multisite-multi-currency` എന്നതിലേക്ക് അപ്‌ലോഡ് ചെയ്യുക.
2. WordPress-ലെ **Plugins** സ്ക്രീനിലൂടെ പ്ലഗിൻ ആക്ടിവേറ്റ് ചെയ്യുക (Network Activate).
3. admin dashboard തുറക്കാൻ **Ultimate Multisite → Multi-Currency** എന്നതിലേക്ക് പോകുക.
4. സ്വീകരിക്കാൻ ആഗ്രഹിക്കുന്ന കറൻസികൾ ചേർക്കുകയും നിങ്ങളുടെ എക്സ്ചേഞ്ച് റേറ്റ് പ്രൊവൈഡർ ക്രമീകരിക്കുകയും ചെയ്യുക.

## കോൺഫിഗറേഷൻ (Configuration) {#configuration}

### Admin Dashboard {#admin-dashboard}

Multi-Currency admin dashboard (**Ultimate Multisite → Multi-Currency**) എന്നത് താഴെ പറയുന്ന കാര്യങ്ങൾ ചെയ്യുന്നതിനുള്ള കേന്ദ്ര സ്ഥലമാണ്:

- കറൻസികൾ ചേർക്കുക, പ്രവർത്തനക്ഷമമാക്കുക അല്ലെങ്കിൽ പ്രവർത്തനരഹിതമാക്കുക.
- ഒരു എക്സ്ചേഞ്ച് റേറ്റ് പ്രൊവൈഡർ തിരഞ്ഞെടുക്കുകയും ആവശ്യാനുസരണം നിരക്കുകൾ റിഫ്രഷ് ചെയ്യുകയും ചെയ്യുക.
- സജീവമായ എല്ലാ കറൻസികളുടെയും നിലവിലെ നിരക്കുകൾ അവലോകനം ചെയ്യുക.
- ഓട്ടോമാറ്റിക് പ്രൊവൈഡർമാർ അനുയോജ്യമല്ലാത്തപ്പോൾ മാനുവൽ നിരക്കുകൾ സജ്ജമാക്കുക.

### Exchange Rate Providers {#exchange-rate-providers}

| Provider | Description |
|---|---|
| **ECB** | യൂറോപ്യൻ സെൻട്രൽ ബാങ്ക് — യൂറോ (EUR) അടിസ്ഥാനമാക്കിയുള്ള നെറ്റ്വർക്കുകൾക്ക് സൗജന്യ ദൈനംദിന നിരക്കുകൾ. |
| **Frankfurter** | ഓപ്പൺ-സോഴ്‌സ് ECB മിറർ API — സൗജന്യം, API key ആവശ്യമില്ല. |
| **Stripe** | Stripe-ന്റെ ലൈവ് FX നിരക്കുകൾ ഉപയോഗിക്കുന്നു — സജീവമായ Stripe ഇന്റഗ്രേഷൻ ആവശ്യമാണ്. |
| **Manual** | നിങ്ങൾ നേരിട്ട് നിരക്കുകൾ നൽകുന്നു; ബാഹ്യ API കോളുകൾ ആവശ്യമില്ല. |

**Ultimate Multisite → Multi-Currency → Settings → Exchange Rate Provider** എന്നതിൻ കീഴിൽ നിങ്ങളുടെ പ്രൊവൈഡർ തിരഞ്ഞെടുക്കുക. നിരക്കുകൾ എപ്പോൾ വേണമെങ്കിലും മാനുവലായി റിഫ്രഷ് ചെയ്യാനോ അല്ലെങ്കിൽ ഓട്ടോമാറ്റിക്കായി അപ്‌ഡേറ്റ് ചെയ്യാൻ ഷെഡ്യൂൾ ചെയ്യാനോ സാധിക്കും.

### Per-Product Pricing Overrides {#per-product-pricing-overrides}

ഓട്ടോമാറ്റിക് കൺവേർഷൻ മതിയായ കൃത്യത നൽകുന്നില്ലെങ്കിൽ, ഏത് ഉൽപ്പന്നത്തിലും ഒരു പ്രത്യേക കറൻസിക്ക് കൃത്യമായ വില സജ്ജമാക്കാൻ നിങ്ങൾക്ക് കഴിയും:

1. ഉൽപ്പന്ന എഡിറ്റർ തുറക്കുക (**Ultimate Multisite → Products → Edit**).
2. **Currency Pricing** എന്ന വിഭാഗത്തിലേക്ക് സ്ക്രോൾ ചെയ്യുക.
3. മാറ്റം വരുത്താൻ ആഗ്രഹിക്കുന്ന ഓരോ കറൻസിക്ക് കൃത്യമായ വില നൽകുക.
4. ഉൽപ്പന്നം സേവ് ചെയ്യുക.

override ചെയ്യാത്ത ഉൽപ്പന്നങ്ങൾ നിങ്ങളുടെ ബേസ് കറൻസിയിൽ നിന്ന് മാറ്റിയ വില ഉപയോഗിക്കുന്നു.

### Currency Selector Signup Field {#currency-selector-signup-field}

ഉപയോക്താക്കൾക്ക് രജിസ്ട്രേഷൻ സമയത്ത് അവരുടെ ഇഷ്ടപ്പെട്ട കറൻസി തിരഞ്ഞെടുക്കാൻ കഴിയുന്നതിനായി കറൻസി സെലക്ടർ ഫീൽഡ് നിങ്ങളുടെ സൈൻഅപ്പ്/ചെക്ക്ഔട്ട് ഫ്ലോയിൽ ചേർക്കാൻ കഴിയും:

1. **Ultimate Multisite → Multi-Currency → Settings** എന്നതിലേക്ക് പോകുക.
2. **Show currency selector on signup** ഓൺ ചെയ്യുക.
3. ഓപ്ഷണലായി ഒരു ഡിഫോൾട്ട് കറൻസി സജ്ജമാക്കുക അല്ലെങ്കിൽ ജിയോലൊക്കേഷൻ വഴി മുൻകൂട്ടി തിരഞ്ഞെടുക്കാനുള്ള മൂല്യം തീരുമാനിക്കാൻ അനുവദിക്കുക.

തിരഞ്ഞെടുത്ത കറൻസി സബ്‌സ്‌ക്രിപ്ഷനൊപ്പം സംഭരിക്കുകയും തുടർ എല്ലാ ബില്ലിംഗിനും ഉപയോഗിക്കുകയും ചെയ്യുന്നു.

### Geolocation-Based Currency Detection {#geolocation-based-currency-detection}

ജിയോലൊക്കേഷൻ കണ്ടെത്തൽ പ്രവർത്തനക്ഷമമാക്കുമ്പോൾ, ഈ ആഡ്‌ഓൺ സന്ദർശകന്റെ രാജ്യത്തെ ഒരു കറൻസിയിലേക്ക് മാപ്പ് ചെയ്യുകയും കറൻസി സെലക്ടർ ഫീൽഡിൽ അത് മുൻകൂട്ടി തിരഞ്ഞെടുക്കുകയും ചെയ്യുന്നു. ഇത് പ്രവർത്തനക്ഷമമാക്കാൻ:

1. **Ultimate Multisite → Multi-Currency → Settings** എന്നതിലേക്ക് പോകുക.
2. **Detect currency from visitor location** ഓൺ ചെയ്യുക.
3. **Geolocation Mapping** ടേബിളിൽ രാജ്യങ്ങളെ കറൻസികളുമായി മാപ്പ് ചെയ്യുക.

ഒരു സന്ദർശകന്റെ രാജ്യത്തിന് മാപ്പിംഗ് ഇല്ലെങ്കിൽ, നെറ്റ്വർക്കിന്റെ ബേസ് കറൻസി ഒരു fallback ആയി ഉപയോഗിക്കുന്നു.

## പതിവായി ചോദിക്കുന്ന ചോദ്യങ്ങൾ (Frequently Asked Questions) {#frequently-asked-questions}

### ഒരു എക്സ്ചേഞ്ച് റേറ്റ് പ്രൊവൈഡർ ലഭ്യമല്ലെങ്കിൽ എന്ത് സംഭവിക്കും? {#what-happens-if-an-exchange-rate-provider-is-unavailable}

ആഡ്‌ഓൺ അവസാനമായി വിജയകരമായി ലഭിച്ച നിരക്കുകളിലേക്ക് fallback ചെയ്യും. ഇതുവരെ നിരക്കുകൾ ലഭിച്ചിട്ടില്ലെങ്കിൽ, നിങ്ങൾ ക്രമീകരിച്ച മാനുവൽ നിരക്കുകളിലേക്ക് fallback ചെയ്യും. ഒരു സുരക്ഷാ വലയമായി മാനുവൽ നിരക്കുകൾ ക്രമീകരിക്കാൻ ശുപാർശ ചെയ്യുന്നു.

### വ്യത്യസ്ത കറൻസികൾക്കായി വ്യത്യസ്ത പ്രൊവൈഡർമാർ ഉപയോഗിക്കാമോ? {#can-i-use-different-providers-for-different-currencies}

ഇല്ല — എല്ലാ കറൻസികൾക്കും ഒരേ സമയം ഒരു പ്രൊവൈഡർ മാത്രമേ സജീവമാകൂ. വ്യക്തിഗത നിരക്കുകളിൽ പൂർണ്ണ നിയന്ത്രണം വേണമെങ്കിൽ **Manual** ഉപയോഗിക്കുക.

### ജിയോലൊക്കേഷന് പ്രത്യേക പ്ലഗിൻ ആവശ്യമുണ്ടോ? {#does-geolocation-require-a-separate-plugin}

ബാഹ്യ ജിയോലൊക്കേഷൻ പ്ലഗിൻ ആവശ്യമില്ല. ആഡ്‌ഓൺ വേർഡ്പ്രസ്സിന്റെ ബിൽറ്റ്-ഇൻ IP-ടു-കൺട്രി കണ്ടെത്തൽ ഉപയോഗിക്കുന്നു. കൂടുതൽ കൃത്യതയ്ക്ക്, നൽകിയിട്ടുള്ള ഫിൽട്ടർ ഹുക്കുകൾ വഴി ഒരു പ്രത്യേക ജിയോലൊക്കേഷൻ സർവീസ് സംയോജിപ്പിക്കാൻ കഴിയും.

### ഉൽപ്പന്നമനുസരിച്ചുള്ള മാറ്റങ്ങൾ നികുതിക്ക് മുൻപോ ശേഷോ ആണ് ബാധകമാകുന്നത്? {#are-per-product-overrides-applied-before-or-after-tax}

ഉൽപ്പന്നമനുസരിച്ചുള്ള മാറ്റങ്ങൾ ടാർഗെറ്റ് കറൻസിയിൽ നികുതിക്ക് മുൻപുള്ള വില സജ്ജമാക്കുന്നു. നിലവിലുള്ള നികുതി കോൺഫിഗറേഷൻ ഉപയോഗിച്ച് ഈ മാറ്റത്തിന് മുകളിലായി നികുതി കണക്കാക്കുന്നു.
