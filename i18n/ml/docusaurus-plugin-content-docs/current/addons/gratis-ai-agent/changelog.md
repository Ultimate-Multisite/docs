# Changelog {#changelog}

## 1.9.0 — Released on 2026-04-28 {#190--released-on-2026-04-28}

### New {#new}

- **`create_contact_form` ability** — സജീവമായ ഫോം പ്ലഗിൻ (Contact Form 7, WPForms, Fluent Forms, അല്ലെങ്കിൽ Gravity Forms) ഉപയോഗിച്ച് ഒരു കോൺടാക്റ്റ് ഫോം സൃഷ്ടിക്കുകയും, അത് ഏത് പോസ്റ്റിലോ പേജിലോ ഉൾപ്പെടുത്താൻ തയ്യാറായ ഒരു ഷോർട്ട്കോഡ് തിരികെ നൽകുകയും ചെയ്യുന്നു.
- **`set_featured_image` ability** — നിലവിലുള്ള മീഡിയ ലൈബ്രറി അറ്റാച്ച്മെന്റ് ID അല്ലെങ്കിൽ ഒരു റിമോട്ട് URL ഉപയോഗിച്ച് ഒരു പോസ്റ്റിനോ പേജിനോ ഫീച്ചർ ഇമേജ് നൽകുന്നു; ഒരു URL നൽകുകയാണെങ്കിൽ, ചിത്രം സ്വയമേവ ഇംപോർട്ട് ചെയ്യും.
- **`batch_create_posts` ability** — ഒറ്റ അബിലിറ്റി കോളിൽ ഒന്നിലധികം പോസ്റ്റുകൾ സൃഷ്ടിക്കാൻ ഇത് സഹായിക്കുന്നു. `create_post`-ന് സമാനമായ പാരാമീറ്ററുകൾ പിന്തുണയ്ക്കുന്നു, ഓരോ പോസ്റ്റിന്റെയും വിജയം/വിFailure റിപ്പോർട്ട് ചെയ്യുന്നു, കൂടാതെ ഓപ്ഷണൽ `stop_on_error` മോഡും ഇതിനുണ്ട്.
- **`page_template` parameter** — `create_post` ലും `update_post`-ലും ചേർത്തു. സൃഷ്ടിക്കുമ്പോഴോ അപ്ഡേറ്റ് ചെയ്യുമ്പോഴോ ഒരു PHP പേജ് ടെംപ്ലേറ്റ് ഫയൽ (ഉദാഹരണത്തിന്, `page-full-width.php`) നൽകാൻ ഇത് സഹായിക്കുന്നു. `update_post`-ൽ ഒരു ખાલી സ്ട്രിംഗ് നൽകുന്നത് തീമിന്റെ ഡിഫോൾട്ട് സജ്ജീകരണത്തിലേക്ക് മടങ്ങാൻ സഹായിക്കും.
- **Client-side screenshot abilities** — `capture_screenshot`, `compare_screenshots`, കൂടാതെ `review_page_design` എന്നിവ. ഒരു സെർവർ-സൈഡ് ഹെഡ്‌ലെസ് ബ്രൗസർ വഴി ലൈവ് പേജുകളുടെ മുഴുവൻതോതിലുള്ളതോ അല്ലെങ്കിൽ Viewport-ന്റെതോ ആയ സ്ക്രീൻഷോട്ടുകൾ എടുക്കുന്നു, രണ്ട് സ്ക്രീൻഷോട്ടുകൾ തമ്മിലുള്ള വ്യത്യാസം കണ്ടെത്തുന്നു, കൂടാതെ ലേഔട്ട്, ടൈപ്പോഗ്രാഫി, നിറം, ആക്സസ്സിബിലിറ്റി എന്നിവ ഉൾക്കൊള്ളുന്ന AI-ജനറേറ്റഡ് ഡിസൈൻ റിവ്യൂ നൽകുന്നു.
- **Five built-in agents** — Content Writer, Site Builder, Design Studio, Plugin Manager, കൂടാതെ Support Assistant എന്നിവ. ഓരോ ഏജന്റിനും പ്രത്യേക ടൂളുകളുടെ ഒരു കൂട്ടം, ഒരു പ്രത്യേക സിസ്റ്റം പ്രോംപ്റ്റ്, കൂടാതെ സ്റ്റാർട്ടർ നിർദ്ദേശങ്ങൾ എന്നിവയുണ്ട്. ചാറ്റ് ഹെഡറിലെ പുതിയ **Agent Picker** വഴി ഇത് മാറ്റാൻ കഴിയും. [Built-in Agents](../../user-guide/configuration/built-in-agents) കാണുക.
- **Feature flags** — ആക്‌സസ്-കൺട്രോൾ ടോഗിളുകളോടുകൂടിയ പുതിയ **Settings → Feature Flags** ടാബ്. (അഡ്മിനിസ്ട്രേറ്റർമാർക്ക് മാത്രം പരിമിതപ്പെടുത്തുക, നെറ്റ്‌വർക്ക് അഡ്മിനിസ്ട്രേറ്റർമാർക്ക് മാത്രം പരിമിതപ്പെടുത്തുക, സബ്‌സ്‌ക്രൈബർ ആക്‌സസ്, അംഗമല്ലാത്തവർക്ക് പ്രവർത്തനരഹിതമാക്കുക) കൂടാതെ ബ്രാൻഡിംഗ് ഓപ്ഷനുകളും (footer attribution മറയ്ക്കുക, കസ്റ്റം ഏജന്റ് പേര്, ഏജന്റ് പിക്കർ മറയ്ക്കുക, ചാറ്റ് അവതാർ ആയി സൈറ്റ് ഐക്കൺ ഉപയോഗിക്കുക) ഇതിലുണ്ട്. [Gratis AI Agent Settings](../../user-guide/administration/gratis-ai-agent-settings) കാണുക.
- **Restore last session** — പേജ് ലോഡ് ചെയ്യുമ്പോഴും വിഡ്ജറ്റ് തുറക്കുമ്പോഴും ചാറ്റ് പാനൽ ഏറ്റവും പുതിയ സംഭാഷണം സ്വയമേവ റീലോഡ് ചെയ്യുന്നു, അതിനാൽ പേജ് നാവിഗേഷനുകളിലുടനീളം സന്ദർഭം നഷ്ടപ്പെടില്ല.
- **Plugin action links** — പ്ലഗിൻ വിവരണത്തിന് താഴെ, വേർഡ്പ്രസ്സിന്റെ **Plugins → Installed Plugins** സ്‌ക്രീനിൽ ഇപ്പോൾ സെറ്റിംഗുകളിലേക്കും Abilities Registry-യിലേക്കുമുള്ള വേഗത്തിലുള്ള ലിങ്കുകൾ കാണാം.

### Improved {#improved}

- **Image source retry** — ഡൗൺലോഡ് ചെയ്യുന്നതിൽ പരാജയപ്പെട്ടാൽ, ഏജന്റ് ഇപ്പോൾ കോൺഫിഗർ ചെയ്ത എല്ലാ സൗജന്യ ഇമേജ് സ്രോതസ്സുകളും വീണ്ടും ശ്രമിക്കുകയും, അതിനുശേഷം AI-ജനറേറ്റഡ് ഇമേജിലേക്ക് മാറുകയും ചെയ്യുന്നു.
- **Model info panel** — ചാറ്റ് ഹെഡറിൽ എപ്പോഴും കാണാം; ആദ്യ സന്ദേശത്തിന് ശേഷം മറയുന്നില്ല.
- **Auto-scroll behaviour** — ഉപയോക്താവ് വായിക്കാൻ വേണ്ടി സ്ക്രോൾ ചെയ്യുമ്പോൾ ഓട്ടോ-സ്‌ക്രോൾ താൽക്കാലികമായി നിർത്തും; ഒരു ഫ്ലോട്ടിംഗ് **Scroll to bottom** ബട്ടൺ പ്രത്യക്ഷപ്പെടുകയും ഉപയോക്താവ് ഏറ്റവും പുതിയ സന്ദേശത്തോളം എത്തുമ്പോൾ സ്വയമേവ മറയുകയും ചെയ്യും.
- **Agent Picker UI** — ഏജന്റ് ഐക്കണുകളോടുകൂടിയ ഒരു ഫോം-ടേബിൾ ഓവർലേ ആയി പുനർരൂപകൽപ്പന ചെയ്തിരിക്കുന്നു, ഇത് ഏജന്റുകളെ ഒറ്റനോട്ടത്തിൽ തിരിച്ചറിയാനും മാറ്റാനും എളുപ്പമാക്കുന്നു.
- **Lazy-loaded JS chunks** — ചാറ്റ് വിഡ്ജറ്റിന്റെ ആദ്യത്തെ JavaScript ബണ്ടിൽ ഇപ്പോൾ ലേസി-ലോഡഡ് ചങ്ക്സായി വിഭജിച്ചിരിക്കുന്നു, ഇത് ആദ്യത്തെ ബണ്ടിൽ വലിപ്പം 75–90% കുറയ്ക്കുന്നു.
- **Chat widget redesign** — യൂണൈഫൈഡ് AI ഐക്കൺ മുൻ കസ്റ്റം അവതാറിന് പകരമായി വന്നിരിക്കുന്നു; ഇത് ബിൽറ്റ്-ഇൻ ഏജന്റ് സിസ്റ്റവുമായി പൊരുത്തപ്പെടുന്നു.
- **URL linkification** — സിസ്റ്റം സന്ദേശങ്ങളിലും എറർ സന്ദേശ ബബിളുകളിലും കാണുന്ന URL-കൾ ഇപ്പോൾ ക്ലിക്കബിൾ ലിങ്കുകളായി റെൻഡർ ചെയ്യുന്നു.

### Fixed {#fixed}

- **Ability discoverability** — എല്ലാ അബിലിറ്റികളും ഏജന്റിന്റെ ടൂൾ ലിസ്റ്റിൽ വിശ്വസനീയമായി പ്രത്യക്ഷപ്പെടുന്നതിന് വിവരണങ്ങൾ, സിസ്റ്റം പ്രോംപ്റ്റ് റഫറൻസുകൾ, നെയിംസ്‌പേസ് അലൈൻമെന്റ് എന്നിവ തിരുത്തി.
- **Providers cache** — മൾട്ടിസൈറ്റ് നെറ്റ്‌വർക്കുകളിൽ സ്റ്റേൾ-പ്രൊവൈഡർ പ്രശ്നങ്ങൾ തടയാൻ പ്രൊവൈഡർമാർ ഇപ്പോൾ ഒരു വേർഷൻ കൗണ്ടർ വഴി സൈറ്റ്-വൈഡ് കാഷ് ചെയ്യപ്പെടുന്നു.
- **`ability_invalid_output`** — 12 അബിലിറ്റി ഹാൻഡിലറുകളിലുടനീളം പരിഹരിച്ചു; എല്ലാം ശരിയായി ഘടനാപരമായ JSON പ്രതികരണങ്ങൾ നൽകുന്നു.
- **`pending_client_tool_calls` pipeline** — ക്ലയിന്റ്-സൈഡ് ടൂൾ കോളുകൾ ശരിയായി പൂർത്തിയാക്കാനും ഫലങ്ങൾ മോഡലിലേക്ക് തിരികെ നൽകാനും എൻഡ്-ടു-എൻഡ് വയർ ചെയ്തു.
- **History drawer** — റീവർട്ട് ചെയ്യാൻ കഴിയാത്ത മാറ്റങ്ങൾ റീവർട്ട് ലിസ്റ്റിൽ നിന്ന് ഒഴിവാക്കിയിരിക്കുന്നു; **View full history** ലിങ്ക് ഇപ്പോൾ ശരിയായി പ്രവർത്തിക്കുന്നു.
- **Changes/revert system** — അഞ്ച് പ്രത്യേക ബഗ്ഗുകൾ പരിഹരിച്ച് പുതിയ അഡ്മിൻ ഇന്റർഫേസിൽ ഏകീകരിച്ചു.
- **Save Settings toast** — **Save Settings** ക്ലിക്കുചെയ്തതിന് ശേഷം സ്നാക്ക്ബാർ നോട്ടിഫിക്കേഷൻ ഇപ്പോൾ വിശ്വസനീയമായി പ്രത്യക്ഷപ്പെടുന്നു.
- **Trash context menu** — **Delete Permanently** ഓപ്ഷൻ ചേർത്തു, അതിനാൽ ട്രാഷ് വ്യൂവിനെ വിട്ട് ഇനങ്ങൾ ഹാർഡ്-ഡിലീറ്റ് ചെയ്യാൻ കഴിയും.
- **Edit & resend** — എഡിറ്റ് ഐക്കൺ ക്ലിക്കുചെയ്യുന്നത് ഇപ്പോൾ തടസ്സത്തിലെ എല്ലാ സന്ദേശങ്ങൾക്കുമുള്ള എഡിറ്റ് മോഡിലേക്ക് പോകാതെ, ക്ലിക്കുചെയ്ത സന്ദേശത്തിന് മാത്രം എഡിറ്റ് മോഡിലേക്ക് പ്രവേശിക്കുന്നു.
- **Chat layout height** — പ്ലഗിൻ ഇൻജക്റ്റ് ചെയ്ത ഉള്ളടക്കം (അഡ്മിൻ നോട്ടീസുകൾ, ബാനറുകൾ) പേജിന് മുകളിൽ പ്രത്യക്ഷപ്പെടുമ്പോൾ ചാറ്റ് പാനൽ അതിന്റെ ഉയരം ക്രമീകരിക്കുന്നു, ഇത് ഇൻപുട്ട് ഏരിയ സ്ക്രീനിൽ നിന്ന് മാഞ്ഞുപോകുന്നത് തടയുന്നു.

## 1.4.0 — Released on 2026-04-09 {#140--released-on-2026-04-09}

### New {#new-1}

- **WP-CLI benchmark command** (`wp gratis-ai-agent benchmark`) — CI പൈപ്പ്ലൈനുകൾക്കും മോഡൽ വിലയിരുത്തൽ വർക്ക്‌ഫ്ലോകൾക്കുമായി കമാൻഡ് ലൈനിൽ ഏജന്റ് കപ്പാസിറ്റീസ് v1 ബെഞ്ച്മാർക്ക് സ്യൂട്ട് പ്രവർത്തിപ്പിക്കുന്നു. ചോദ്യങ്ങൾ അനുസരിച്ച് പ്രവർത്തിപ്പിക്കാനും, പ്രൊവൈഡർ/മോഡൽ ഓവർറൈഡുകൾ നൽകാനും, JSON/CSV ഔട്ട്പുട്ട് പിന്തുണയ്ക്കുന്നു.
- **Agent Capabilities v1 benchmark suite** — പൂർണ്ണമായ അബിലിറ്റി സർഫേസ് വ്യായാമം ചെയ്യുന്ന, സ്കോറിംഗും ടോക്കൺ/സമയ റിപ്പോർട്ടിംഗും ഉള്ള സങ്കീർണ്ണമായ, ഒന്നിലധികം ഘട്ടങ്ങളുള്ള പ്രോംപ്റ്റുകളുടെ ഒരു ഘടനാപരമായ കൂട്ടം.
- **Custom Post Type abilities** — `register_post_type`, `list_post_types`, `delete_post_type`. രജിസ്ട്രേഷനുകൾ ഓപ്ഷൻസ് ടേബിൾ വഴി റീസ്റ്റാർട്ടുകൾക്കിടയിലും നിലനിൽക്കുന്നു.
- **Custom Taxonomy abilities** — `register_taxonomy`, `list_taxonomies`, `delete_taxonomy`. ഓപ്ഷണൽ റൈറ്റ്റൈറ്റ് സ്ലഗുകൾ സഹിതം ഹൈറാർക്കിക് (hierarchical) കൂടാതെ ഫ്ലാറ്റ് ടാക്സോണമികൾ പിന്തുണയ്ക്കുന്നു.
- **Design System abilities** — `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset`. `minimal-dark`, `warm-editorial`, `corporate-blue`, `vibrant-startup`, `classic-blog` എന്നിങ്ങനെ അഞ്ച് കൂറേറ്റഡ് പ്രീസെറ്റുകൾ ഉൾപ്പെടുന്നു.
- **Global Styles abilities** — `get_global_styles`, `set_global_styles`, `reset_global_styles`. വേർഡ്പ്രസ്സിന്റെ Global Styles API വഴി theme.json മൂല്യങ്ങൾ വായിക്കാനും എഴുതാനും ഇത് സഹായിക്കുന്നു.
- **Navigation Menu management ability** — `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus`. Nested menus, കൂടാതെ തീം ലൊക്കേഷൻ അസൈൻമെന്റ് എന്നിവ പിന്തുണയ്ക്കുന്നു.
- **Options Management ability** — `get_option`, `set_option`, `delete_option`, `list_options`. നിർണ്ണായകമായ വേർഡ്പ്രസ്സ് ഓപ്ഷനുകൾ മാറ്റുന്നത് സംരക്ഷിക്കുന്ന ഒരു ബിൽറ്റ്-ഇൻ സുരക്ഷാ ബ്ലോക്ക്‌ലിസ്റ്റ് ഇതിൽ ഉൾപ്പെടുന്നു.
- **Installable Abilities Registry** — `list_available_abilities`, `install_ability`, `recommend_plugin`. വേർഡ്പ്രസ്സ് പ്ലഗിനുകളായി വിതരണം ചെയ്യപ്പെടുന്ന അബിലിറ്റി പാക്കുകൾ കണ്ടെത്താനും സജീവമാക്കാനും ഇത് സഹായിക്കുന്നു.
- **Site Builder Orchestration v2** — `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`. പ്ലഗിൻ ഡിസ്കവറി, സ്റ്റെപ്പ് ഔട്ട്പുട്ട് റഫറൻസുകൾ, പ്രോഗ്രസ് ട്രാക്കിംഗ്, സ്വയംഭരണ പിശക് പരിഹാരം എന്നിവയോടുകൂടിയ മൾട്ടി-ഫേസ് സൈറ്റ് ബിൽഡ് പ്ലാനുകൾ.
- **Restaurant website benchmark question** (`q-restaurant-website`) — CPT രജിസ്ട്രേഷൻ, ഡിസൈൻ സിസ്റ്റം, नेവിഗേഷൻ, പ്ലഗിൻ ഡിസ്കവറി എന്നിവ ഉൾക്കൊള്ളുന്ന എൻഡ്-ടു-എൻഡ് ടെസ്റ്റ്.
- **AI provider connector plugins** വേഗത്തിലുള്ള ലോക്കൽ ഡെവലപ്‌മെന്റ് സെറ്റപ്പിനായി WordPress Playground blueprints-ൽ ചേർത്തു.
