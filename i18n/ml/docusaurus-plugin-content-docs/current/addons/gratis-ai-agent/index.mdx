# Gratis AI Agent

Gratis AI Agent എന്നത് ഒരു സ്വയംഭരണ WordPress AI അസിസ്റ്റന്റാണ്. സ്വാഭാവിക ഭാഷയിലുള്ള നിർദ്ദേശങ്ങൾ ഉപയോഗിച്ച് ഇത് പൂർണ്ണമായ WordPress സൈറ്റുകൾ പ്ലാൻ ചെയ്യാനും നിർമ്മിക്കാനും കൈകാര്യം ചെയ്യാനും കഴിയും. ഉള്ളടക്ക മാനേജ്‌മെന്റ് മുതൽ ഡിസൈൻ സിസ്റ്റം കസ്റ്റമൈസേഷൻ വരെയുള്ള കാര്യങ്ങൾ ഉൾക്കൊള്ളുന്ന, **അബിലിറ്റികൾ** (abilities) എന്ന ഘടനാപരമായ കൂട്ടം പ്രവർത്തനങ്ങൾ ഇത് ലഭ്യമാക്കുന്നു. പതിപ്പ് 1.9.0-ൽ ഉള്ളടക്ക നിർമ്മാണ അബിലിറ്റികൾ, ക്ലയിന്റ്-സൈഡ് സ്ക്രീൻഷോട്ടുകൾ വഴി വിഷ്വൽ പേജ് റിവ്യൂ, അഞ്ച് ബിൽറ്റ്-ഇൻ ഏജന്റ് പ്രൊഫൈലുകൾ, ആക്‌സസ് കൺട്രോൾ, ബ്രാൻഡിംഗ് എന്നിവയ്ക്കായുള്ള ഫീച്ചർ ഫ്ലാഗുകൾ എന്നിവ കൂട്ടിച്ചേർത്തു.

## പ്രധാന സവിശേഷതകൾ (Key Features)

- **അബിലിറ്റീസ് സിസ്റ്റം (Abilities system)** — നിങ്ങളുടെ WordPress ഇൻസ്റ്റാളേഷനിൽ ഏജന്റിന് ഉപയോഗിക്കാൻ കഴിയുന്ന പ്രവർത്തനങ്ങളുടെ ഒരു മൊഡുലാർ, വിപുലീകരിക്കാവുന്ന каталог.
- **ഉള്ളടക്ക മാനേജ്‌മെന്റ് (Content Management)** — പോസ്റ്റുകളും പേജുകളും സൃഷ്ടിക്കുക (പേജ് ടെംപ്ലേറ്റുകൾ നൽകുന്നത് ഉൾപ്പെടെ), ഒറ്റ കോളിൽ ഒന്നിലധികം പോസ്റ്റുകൾ ബാച്ച്-സൃഷ്ടിക്കുക, ഫീച്ചർ ചെയ്ത ചിത്രങ്ങൾ സജ്ജമാക്കുക, കോൺടാക്റ്റ് ഫോമുകൾ നിർമ്മിക്കുക.
- **വിഷ്വൽ റിവ്യൂ (Visual Review)** — ലൈവ് പേജുകളുടെ സ്ക്രീൻഷോട്ടുകൾ എടുക്കുക, മുമ്പും ശേഷവുമുള്ള അവസ്ഥകൾ താരതമ്യം ചെയ്യുക, ലേഔട്ട്, ടൈപ്പോഗ്രാഫി, നിറം, ആക്‌സസിബിലിറ്റി എന്നിവ ഉൾക്കൊള്ളുന്ന AI-ജനറേറ്റഡ് ഡിസൈൻ റിവ്യൂ ലഭിക്കുക.
- **അഞ്ച് ബിൽറ്റ്-ഇൻ ഏജന്റുകൾ (Five built-in agents)** — Content Writer, Site Builder, Design Studio, Plugin Manager, Support Assistant എന്നിവ. ചാറ്റ് ഹെഡറിലെ ഏജന്റ് പിക്കർ വഴി ഇവ മാറ്റാവുന്നതാണ്.
- **ഫീച്ചർ ഫ്ലാഗുകൾ (Feature flags)** — റോളുകൾ നിയന്ത്രിക്കാനും, വൈറ്റ്-ലേബൽ നാമകരണത്തിനും, വിഡ്ജറ്റ് കസ്റ്റമൈസേഷനും വേണ്ടി Settings → Feature Flags-ൽ ലഭ്യമായ ആക്‌സസ് കൺട്രോൾ ടോഗിളുകൾ.
- **കസ്റ്റം പോസ്റ്റ് ടൈപ്പ് മാനേജ്‌മെന്റ് (Custom Post Type management)** — റീജിസ്റ്റർ ചെയ്യുക, ലിസ്റ്റ് ചെയ്യുക, ഡിലീറ്റ് ചെയ്യുക എന്നിവ കസ്റ്റം പോസ്റ്റ് ടൈപ്പുകൾക്ക്. റീസ്റ്റാർട്ടുകൾക്കിടയിലും ഡാറ്റ നിലനിർത്തുന്നു.
- **കസ്റ്റം ടാക്സോണമി മാനേജ്‌മെന്റ് (Custom Taxonomy management)** — റീജിസ്റ്റർ ചെയ്യുക, ലിസ്റ്റ് ചെയ്യുക, ഡിലീറ്റ് ചെയ്യുക എന്നിവ കസ്റ്റം ടാക്സോണമികൾക്ക്. ഡാറ്റ നിലനിർത്തുന്നു.
- **ഡിസൈൻ സിസ്റ്റം അബിലിറ്റികൾ (Design System abilities)** — കസ്റ്റം CSS ഇൻജക്റ്റ് ചെയ്യുക, കൂറേറ്റ് ചെയ്ത ബ്ലോക്ക് പാറ്റേണുകൾ കൈകാര്യം ചെയ്യുക, സൈറ്റ് ലോഗോ സജ്ജമാക്കുക, theme.json പ്രീസെറ്റുകൾ പ്രയോഗിക്കുക.
- **ഗ്ലോബൽ സ്റ്റൈൽസ് മാനേജ്‌മെന്റ് (Global Styles management)** — കളർ പാലറ്റുകൾ, ടൈപ്പോഗ്രാഫി, സ്പേസിംഗ് എന്നിവയ്ക്കായി theme.json മൂല്യങ്ങൾ വായിക്കാനും എഴുതാനും കഴിയും.
- **നാവിഗേഷൻ മെനു മാനേജ്‌മെന്റ് (Navigation Menu management)** — നാവിഗേഷൻ മെനുവുകളും അതിലെ ഇനങ്ങളും പ്രോഗ്രാമാറ്റിക്കായി സൃഷ്ടിക്കാനും അപ്ഡേറ്റ് ചെയ്യാനും കഴിയും.
- **ഓപ്ഷൻസ് മാനേജ്‌മെന്റ് (Options Management)** — പ്രധാനപ്പെട്ട സെറ്റിംഗുകൾ സംരക്ഷിക്കാൻ ബിൽറ്റ്-ഇൻ സേഫ്റ്റി ബ്ലോക്ക്‌ലിസ്റ്റ് ഉള്ള ഒരു സംവിധാനത്തിലൂടെ WordPress ഓപ്ഷനുകൾ വായിക്കാനും എഴുതാനും കഴിയും.
- **ഇൻസ്റ്റാളബിൾ അബിലിറ്റീസ് രജിസ്ട്രി (Installable Abilities Registry)** — കോർ പ്ലഗിൻ ഫയലുകൾ മാറ്റാതെ അധിക അബിലിറ്റി പാക്കുകൾ കണ്ടെത്താനും ആക്ടിവേറ്റ് ചെയ്യാനും കഴിയും.
- **സൈറ്റ് ബിൽഡർ ഓർക്കസ്ട്രേഷൻ v2 (Site Builder Orchestration v2)** — പ്ലഗിൻ കണ്ടെത്തൽ, പ്രോഗ്രസ് ട്രാക്കിംഗ്, എറർ റിക്കവറി എന്നിവയോടുകൂടിയ ഒന്നിലധികം ഘട്ടങ്ങളുള്ള സൈറ്റ് ബിൽഡ് പ്ലാനുകൾ.
- **WP-CLI ബെഞ്ച്മാർക്ക് കമാൻഡ് (WP-CLI benchmark command)** — CI-ക്കും മോഡൽ വിലയിരുത്തലിനുമായി കമാൻഡ് ലൈനിൽ നിന്ന് ഏജന്റ് കഴിവിന്റെ ബെഞ്ച്മാർക്ക് സ്യൂട്ട് പ്രവർത്തിപ്പിക്കുക.

## ആവശ്യകതകൾ (Requirements)

- WordPress 6.0 അല്ലെങ്കിൽ അതിലും ഉയർന്നത്
- PHP 8.1 അല്ലെങ്കിൽ അതിലും ഉയർന്നത്
- WordPress Multisite (network-activated) അല്ലെങ്കിൽ single-site ഇൻസ്റ്റാളേഷൻ
- ഒരു സജീവ AI പ്രൊവൈഡർ കണക്റ്റർ പ്ലഗിൻ (Anthropic, OpenAI, അല്ലെങ്കിൽ അനുയോജ്യമായത്)

## ഇൻസ്റ്റാളേഷൻ (Installation)

1. പ്ലഗിൻ ഫയലുകൾ `/wp-content/plugins/gratis-ai-agent` എന്നതിലേക്ക് അപ്‌ലോഡ് ചെയ്യുക.
2. WordPress-ലെ **Plugins** സ്ക്രീനിലൂടെ പ്ലഗിൻ ആക്ടിവേറ്റ് ചെയ്യുക.
3. **Gratis AI Agent → Settings** എന്നതിലേക്ക് പോയി നിങ്ങളുടെ AI പ്രൊവൈഡറെ കണക്ട് ചെയ്യുക.
4. (ഓപ്ഷണൽ) അധിക അബിലിറ്റി പാക്കുകൾ ഇൻസ്റ്റാൾ ചെയ്യാൻ **Abilities Registry** പരിശോധിക്കുക.

## വേഗത്തിലുള്ള തുടക്കം (Quick Start)

ആക്ടിവേറ്റ് ചെയ്യുകയും ഒരു AI പ്രൊവൈഡറുമായി കണക്ട് ചെയ്യുകയും ചെയ്ത ശേഷം, WordPress അഡ്മിൻ സൈഡ്ബാറിൽ **Gratis AI Agent** പാനൽ തുറക്കുക. ഒരു സ്വാഭാവിക ഭാഷയിലുള്ള നിർദ്ദേശം ടൈപ്പ് ചെയ്യുക, തുടർന്ന് ഏജന്റ് ലഭ്യമായ അബിലിറ്റികൾ ഉപയോഗിച്ച് ആവശ്യമായ ഘട്ടങ്ങൾ പ്ലാൻ ചെയ്യുകയും എക്സിക്യൂട്ട് ചെയ്യുകയും ചെയ്യും.

**ഉദാഹരണ നിർദ്ദേശങ്ങൾ:**

```
Create a "Portfolio" custom post type with a "Project Category" taxonomy.
```

```
Inject a dark-mode colour palette into the theme and set the primary colour to #1a1a2e.
```

```
Build a restaurant website with an online menu, booking form, and contact page.
```

## അബിലിറ്റീസ് അവലോകനം (Abilities Overview)

അബിലിറ്റികൾ എന്നത് ഏജന്റിന് ചെയ്യാൻ കഴിയുന്ന ആറ്റോമിക് പ്രവർത്തനങ്ങളാണ്. ഓരോ അബിലിറ്റിയും ഒരു രജിസ്റ്റർ ചെയ്ത PHP ക്ലാസ്സാണ്. ഇത് ഒരു സ്കീമ (parameters, return types) പുറത്തുവിടുന്നു, ഇത് ഏജന്റ് റൺടൈമിൽ വായിക്കുന്നു. അബിലിറ്റികൾ പ്രവർത്തന മേഖലകളായി ഗ്രൂപ്പ് ചെയ്തിരിക്കുന്നു:

| Area | Abilities |
|---|---|
| **Content Management** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **Visual Review** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **Custom Post Types** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **Custom Taxonomies** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **Design System** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **Global Styles** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **Navigation Menus** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **Options Management** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **Installable Abilities** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **Site Builder** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

പൂർണ്ണമായ parameter ഡോക്യുമെന്റേഷനും ഉപയോഗ ഉദാഹരണങ്ങൾക്കുമായി [Abilities Reference](./abilities) കാണുക.

## WP-CLI

ബെഞ്ച്മാർക്കുകൾ പ്രവർത്തിപ്പിക്കാനും കമാൻഡ് ലൈനിൽ നിന്ന് അബിലിറ്റികൾ കൈകാര്യം ചെയ്യാനും Gratis AI Agent ഒരു `wp gratis-ai-agent` കമാൻഡ് കുടുംബം നൽകുന്നു. മുഴുവൻ കമാൻഡ് ലിസ്റ്റിനും [WP-CLI Reference](./wp-cli) കാണുക.

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## കോൺഫിഗറേഷൻ (Configuration)

എല്ലാ സെറ്റിംഗുകളും WordPress അഡ്മിനിൽ **Gratis AI Agent → Settings** എന്നതിലുണ്ട്:

| Setting | Description |
|---|---|
| **AI Provider** | സജീവമായ AI പ്രൊവൈഡർ കണക്റ്റർ തിരഞ്ഞെടുക്കുകയും കോൺഫിഗർ ചെയ്യുകയും ചെയ്യുക |
| **Model** | ഏജന്റ് സെഷനുകൾക്കായി ഉപയോഗിക്കേണ്ട ഭാഷാ മോഡൽ തിരഞ്ഞെടുക്കുക |
| **Max Tokens** | ഓരോ റിക്വസ്റ്റിനുള്ള ടോക്കൺ ബഡ്ജറ്റ് (ഡിഫോൾട്ട് 4096) |
| **Ability Timeout** | ഒരു വ്യക്തിഗത അബിലിറ്റി പ്രവർത്തിക്കാൻ കഴിയുന്ന പരമാവധി സെക്കൻഡുകൾ (ഡിഫോൾട്ട് 30) |
| **Debug Logging** | അബിലിറ്റി കോളുകളും ഏജന്റിന്റെ യുക്തിയും WordPress ഡീബഗ് ലോഗിലേക്ക് രേഖപ്പെടുത്തുക |

## ഓപ്ഷൻസ് മാനേജ്‌മെന്റ് സേഫ്റ്റി ബ്ലോക്ക്‌ലിസ്റ്റ് (Options Management Safety Blocklist)

ഓപ്ഷൻസ് മാനേജ്‌മെന്റ് അബിലിറ്റിയിൽ ഒരു ബിൽറ്റ്-ഇൻ ബ്ലോക്ക്‌ലിസ്റ്റ് ഉൾപ്പെടുന്നു. ഇത് ഏജന്റിന് സെൻസിറ്റീവായ WordPress ഓപ്ഷനുകൾ വായിക്കാനോ എഴുതാനോ തടയുന്നു. ഡിഫോൾട്ട് ബ്ലോക്ക്‌ലിസ്റ്റ് ഇവയെ ഉൾക്കൊള്ളുന്നു:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (URL ഓപ്ഷനുകൾ — ഇവയ്ക്കായി ഒരു പ്രത്യേക മൈഗ്രേഷൻ ടൂൾ ഉപയോഗിക്കുക)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (രഹസ്യ കീകൾ)
- `active_plugins`, `template` (പ്ലഗിൻ/തീം ആക്ടിവേഷൻ — പ്രത്യേക അബിലിറ്റികൾ ഉപയോഗിക്കുക)
- ഇൻസ്റ്റാൾ ചെയ്ത പ്ലഗിനുകൾ വഴി രജിസ്റ്റർ ചെയ്ത പേയ്‌മെന്റ് ഗേറ്റ്‌വേ ക്രെഡൻഷ്യലുകളും API കീകളും

`gratis_ai_agent_options_blocklist` ഫിൽട്ടർ വഴി അധിക എൻട്രികൾ ചേർക്കാൻ കഴിയും.

## Hooks and Filters

### Actions

- `gratis_ai_agent_ability_registered` — ഒരു അബിലിറ്റി രജിസ്റ്റർ ചെയ്തതിന് ശേഷം ട്രിഗർ ചെയ്യപ്പെടുന്നു, അബിലിറ്റി ക്ലാസ് നാമം സ്വീകരിക്കുന്നു.
- `gratis_ai_agent_plan_started` — ഒരു സൈറ്റ് ബിൽഡർ പ്ലാൻ എക്സിക്യൂഷൻ ആരംഭിക്കുമ്പോൾ ട്രിഗർ ചെയ്യപ്പെടുന്നു.
- `gratis_ai_agent_plan_completed` — ഒരു പ്ലാൻ വിജയകരമായി പൂർത്തിയാകുമ്പോൾ ട്രിഗർ ചെയ്യപ്പെടുന്നു.
- `gratis_ai_agent_plan_failed` — ഒരു പ്ലാൻ റിക്കവറി ചെയ്യാൻ കഴിയാത്ത പിശക് നേരിടുമ്പോൾ ട്രിഗർ ചെയ്യപ്പെടുന്നു.

### Filters

- `gratis_ai_agent_abilities` — ഏജന്റ് ഇത് ലോഡ് ചെയ്യുന്നതിന് മുമ്പ് രജിസ്റ്റർ ചെയ്ത അബിലിറ്റി ലിസ്റ്റ് മാറ്റാൻ.
- `gratis_ai_agent_options_blocklist` — ഓപ്ഷൻസ് സേഫ്റ്റി ബ്ലോക്ക്‌ലിസ്റ്റ് വികസിപ്പിക്കാനോ കുറയ്ക്കാനോ.
- `gratis_ai_agent_benchmark_questions` — വിലയിരുത്തൽ സ്യൂട്ടിലേക്ക് കസ്റ്റം ബെഞ്ച്മാർക്ക് ചോദ്യങ്ങൾ ചേർക്കാൻ.
- `gratis_ai_agent_plan_plugins` — സൈറ്റ് പ്ലാൻ ജനറേഷൻ സമയത്ത് പ്ലഗിൻ കണ്ടെത്തൽ ലിസ്റ്റ് മാറ്റാൻ.
