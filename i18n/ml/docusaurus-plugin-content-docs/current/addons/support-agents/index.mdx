# Support Agents {#support-agents}

ഒരു മുഴുവൻ നെറ്റ്വർക്കും ഒറ്റയ്ക്ക് കൈകാര്യം ചെയ്യുന്നത് എളുപ്പമുള്ള കാര്യമല്ല. നിങ്ങളുടെ നെറ്റ്വർക്ക് വളരാൻ തുടങ്ങുമ്പോൾ, സപ്പോർട്ടിനായുള്ള ആവശ്യം വർദ്ധിക്കുന്നത് നിങ്ങൾ കാണും. നിങ്ങളുടെ ജോലിഭാരം കുറയ്ക്കാൻ സഹായിക്കുന്ന ആളുകളെ കൊണ്ടുവരുന്നത് നല്ല ആശയമാണ്. എന്നിരുന്നാലും, മുഴുവൻ ആക്‌സസ്സോ പെർമിഷനോ നൽകുന്നത് അപകടകരമാണ്, അത് നിങ്ങളുടെ നെറ്റ്വർക്കിനെ ബാധിച്ചേക്കാം. ഇവിടെയാണ് ഈ Support Agent add-on സഹായകമാകുന്നത്.

Support Agent എന്ന് പേരുള്ള ഒരു പുതിയ തരം WordPress user-നെ സൃഷ്ടിക്കാൻ ഈ add-on നിങ്ങളെ അനുവദിക്കുന്നു. ആ user-ന് എന്ത് പെർമിഷനുകൾ വേണമെന്ന് നിങ്ങൾക്ക് പൂർണ്ണ നിയന്ത്രണം ലഭിക്കും. പെർമിഷൻ നൽകുന്നത് ഒരു ടോഗിൾ ബട്ടൺ ഓൺ ചെയ്യുന്നത്ര എളുപ്പമാണ്, അത് നീക്കം ചെയ്യണമെന്ന് തീരുമാനിച്ചാൽ ഓഫ് ചെയ്യുകയേ ചെയ്താൽ മതി.

## Features {#features}

* WordPress & Ultimate Multisite പെർമിഷനുകളുടെ വിശദമായ ലിസ്റ്റ്
* എളുപ്പത്തിൽ ആക്‌സസ് പ്രവർത്തനക്ഷമമാക്കാനും പ്രവർത്തനരഹിതമാക്കാനും കഴിയും
* പരിധിയില്ലാത്ത എണ്ണം ഏജന്റ് അക്കൗണ്ടുകൾ സൃഷ്ടിക്കാൻ കഴിയും

## Installation {#installation}

1. 'ultimate-multisite-support-agents' എന്നത് '/wp-content/plugins/' ഡയറക്‌ടറിയിലേക്ക് അപ്‌ലോഡ് ചെയ്യുക
2. WordPress-ലെ 'Plugins' മെനു വഴി പ്ലഗിൻ ആക്ടിവേറ്റ് ചെയ്യുക
3. പ്ലഗിൻ സജ്ജമാക്കാൻ ഘട്ടം ഘട്ടമായുള്ള Wizard പിന്തുടരുക
