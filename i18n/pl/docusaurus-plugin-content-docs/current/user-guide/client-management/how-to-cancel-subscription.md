# Jak mogę anulować subskrypcję?

Jeśli posiadasz licencję Ultimate Multisite, możesz w dowolnym momencie anulować jej odnowienie. Wystarczy wykonać poniższe kroki:


<!-- Screenshot unavailable: Account page email with access link -->

Użyj adresu e-mail podanego podczas zakupu oraz swojego hasła.

<!-- Screenshot unavailable: Login page with email and password fields -->

Oto ekran początkowy Twojego konta:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

Aby anulować subskrypcję, przejdź w menu po lewej stronie do sekcji _Renewing & Billing._

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

Kliknij strzałkę po prawej stronie, aby otworzyć boczny panel. Następnie wybierz opcję, która anuluje **automatyczne odnawianie**.

System wyświetli komunikat z prośbą o potwierdzenie.

Jeśli na pewno chcesz anulować subskrypcję, po prostu kliknij przycisk _**Cancel Renewals**_. Po tej czynności zostaniesz poproszony o wypełnienie krótkiej ankiety.

Gotowe! Twoja subskrypcja nie będzie automatycznie odnawiana.

**Twój klucz będzie ważny do momentu wygaśnięcia subskrypcji**. Jeśli zechcesz ponownie aktywować subskrypcję, będziesz musiał to zrobić ręcznie.
