# Jak uzyskać dostęp do strony konta po raz pierwszy?

Witaj w Ultimate Multisite!

Aby uzyskać dostęp do strony konta, skorzystaj z linku, który otrzymałeś w wiadomości e-mail po zakupie naszego pluginu:

<!-- Screenshot unavailable: Account page email with access link -->

Użyj adresu e-mail podanego podczas zakupu oraz hasła, które otrzymałeś w wiadomości.

<!-- Screenshot unavailable: Login page with email and password fields -->

Ponieważ logujesz się na to konto po raz pierwszy, zostaniesz poproszony o zmianę hasła. Zmień je i kliknij przycisk _Zapisz_.

<!-- Screenshot unavailable: Change password form with Save button -->

Na stronie konta możesz zarządzać wszystkimi szczegółami swojego konta, takimi jak płatności, odnowienia, anulowania i wiele innych.
