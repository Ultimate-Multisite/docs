# Jak uzyskać pomoc w tworzeniu niestandardowych rozwiązań od Codeable?

Czasami naprawdę potrzebujemy świetnych dostosowań, aby nasze przepływy sieciowe działały dokładnie tak, jak chcemy. Jeśli właśnie tego szukasz, polecamy [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) — miejsce, gdzie znajdziesz doświadczonych programistów ze znajomością Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) to platforma łącząca klientów z wykwalifikowanymi freelancerami WordPress, którzy mogą pracować nad projektami dowolnej wielkości. Mają rygorystyczny proces weryfikacji, który zapewnia dopasowanie do ekspertów posiadających odpowiednie kwalifikacje do zadania, które masz na myśli.

### Jak utworzyć projekt w Codeable?

Najpierw wejdź na stronę [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) i kliknij przycisk **Start a Project**.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Teraz, aby przesłać projekt, kliknij po lewej stronie **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

W polu „I need help with" wybierz **Customizing**:

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

W polu „My" wybierz **Plugin**:

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Następnie kliknij przycisk **Continue**.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Teraz nadszedł czas, aby jak najlepiej opisać swój projekt i potrzeby. Postaraj się podać jak najwięcej informacji!

<!-- Screenshot unavailable: Codeable project description form -->

Odpowiedz na poniższe pytania, które pomogą w ustaleniu wyceny.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Następnie, jeśli nie masz jeszcze konta w [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), utwórz je i kliknij przycisk **Publish My Project**.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

Po opublikowaniu projektu w [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) platforma połączy Cię z programistami, którzy posiadają umiejętności wymagane do Twojego projektu.

Ten proces pomoże znaleźć odpowiednią osobę do Twojego zadania, która ma doświadczenie z WordPress i Ultimate Multisite!

Minimalna stawka w Codeable wynosi 70–120 USD za godzinę. Wyceny będą się różnić w zależności od zakresu prac, złożoności i pilności projektu.

Po otrzymaniu wyceny będziesz mógł zatrudnić programistę, który najlepiej odpowiada Twoim potrzebom, i wspólnie rozpocząć pracę nad projektem!
