# Hooki i filtry Ultimate Multisite

Ultimate Multisite składa się z wielu funkcji i zapytań do bazy danych. Zawiera hooki i filtry, które pozwalają modyfikować istniejące funkcje lub dodawać do nich nowy kod.

Przygotowaliśmy listę tych hooków i filtrów jako materiał referencyjny dla zaawansowanych użytkowników, którzy chcą jeszcze bardziej dostosować swoją sieć do własnych potrzeb. Listę znajdziesz na Githubie, klikając poniższy link.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
