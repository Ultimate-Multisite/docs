# Znane niekompatybilności z innymi wtyczkami

Po zainstalowaniu i aktywacji Ultimate Multisite możesz napotkać problemy wynikające z nierozwiązanych niekompatybilności z innymi wtyczkami.

Aktywnie pracujemy nad zmniejszeniem tej listy z każdą nową wersją, ale jeśli wtyczka znajduje się na tej liście, może być konieczne jej dezaktywowanie, aby Ultimate Multisite Network działał w pełni poprawnie.

Oto lista takich wtyczek oraz informacje, co należy zrobić w poszczególnych przypadkach:

**Niektóre wtyczki WPMU DEV** Niektóre wtyczki _już nierozwijane_ przez WPMU DEV powodują zatrzymanie procesu rejestracji Ultimate Multisite na ostatnim kroku (użytkownik nie może utworzyć konta). Oto lista wtyczek powodujących te problemy:

  * Signup Language: _Ultimate Multisite oferuje dodatek o tej samej funkcjonalności o nazwie Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Ta funkcja jest wbudowana w Ultimate Multisite, więc dodatkowa wtyczka nie jest potrzebna._

  * New Blog Templates: _Ta funkcja jest wbudowana w Ultimate Multisite, więc dodatkowa wtyczka nie jest potrzebna._

  * Set Password on Multisite Blog Creation: _Ta funkcja jest wbudowana w Ultimate Multisite, więc dodatkowa wtyczka nie jest potrzebna._

  * Signup TOS: _Ta funkcja jest wbudowana w Ultimate Multisite, więc dodatkowa wtyczka nie jest potrzebna._

  * Anti-Splog: Choć w mniej zaawansowanej wersji, ta funkcja również jest _wbudowana w Ultimate Multisite._

**Uwaga**: Wszystkie pozostałe wtyczki WPMU Dev można używać normalnie wraz z Ultimate Multisite. Przykłady to _Smush_, _Forminator_, _Defender_ itp.
