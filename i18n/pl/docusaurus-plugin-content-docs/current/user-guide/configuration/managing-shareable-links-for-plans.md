# Zarządzanie linkami do udostępniania planów

Ultimate Multisite umożliwia udostępnianie linków do planów bezpośrednio Twoim klientom. Aby to zrobić, przejdź do strony edycji dowolnego planu i kliknij przycisk _Kliknij, aby skopiować link do udostępnienia_.

![Strona edycji produktu z przyciskiem linku do udostępnienia](/img/config/product-edit-full.png)

_Na stronie edycji planu znajdziesz przycisk do tworzenia linków do udostępnienia_

Pamiętaj, że ten mechanizm używa domyślnej częstotliwości rozliczeń do budowania linku. Oznacza to, że jeśli Twoim domyślnym rozliczeniem jest na przykład miesięczne, otrzymasz link w takiej postaci: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Jeśli chcesz, aby link domyślnie kierował do rozliczenia kwartalnego, musisz dodać 3 na końcu:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Aby wybrać plan _premium_ z rozliczeniem rocznym, dodaj 1, a następnie słowo:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Pamiętaj, że nie możesz mieć jednego linku dla różnych częstotliwości rozliczeń. Powinieneś udostępniać osobny link dla każdej częstotliwości rozliczeń.

Gdy udostępnisz link swoim klientom, przekieruje ich on do pierwszego kroku procesu rejestracji.
