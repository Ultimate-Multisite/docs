# Umieszczanie reklam w Twojej sieci z Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector to **darmowy dodatek**, który pozwala umieszczać reklamy w planach bezpośrednio z poziomu ich edycji.

Po zainstalowaniu Ultimate Multisite: Ad Injector przejdź do strony _Wtyczki_, aby go aktywować.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Kliknij przycisk Włącz w całej sieci_

Po aktywacji dodatku w meta boxie _Opcje zaawansowane_ na ekranie edycji planu pojawi się nowa opcja _Ustawienia reklam_.

![Ustawienia reklam na ekranie edycji planu](/img/config/product-edit-full.png)

_Teraz możesz edytować ustawienia reklam dla każdego ze swoich planów_

W tej zakładce możesz dodać kody reklamowe dla wpisów i stron wyświetlanych na front-endzie — przed treścią i po niej.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Po wstawieniu kodu reklamy będzie ona wyświetlana na wpisach i stronach danego planu._

Możesz również umieszczać reklamy w nagłówku panelu administracyjnego dla użytkowników danego planu.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Przejdź do opcji Kod reklamy w panelu administracyjnym, jeśli chcesz wyświetlać reklamy w zapleczu użytkowników_

### Nie tylko reklamy

Ultimate Multisite: Ad Injector możesz wykorzystać również do oferowania wsparcia w planach wyższego poziomu.

Wystarczy przejść do opcji _Reklamy w panelu administracyjnym_ i dodać kod HTML swojego narzędzia wsparcia.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Dodaj obsługę klienta do wybranych planów_

W ten sposób użytkownicy mogą skontaktować się z Tobą bezpośrednio ze swojego panelu administracyjnego!
