# Dostosowywanie panelu administracyjnego za pomocą PRO Theme

Pracując nad projektami dla klientów, często musimy zapewnić im spersonalizowane doświadczenie w panelu administracyjnym WordPressa. Jeśli szukasz bardziej profesjonalnego wyglądu i stylu, PRO Theme może Ci w tym pomóc.

PRO Theme to motyw administracyjny stworzony, aby ułatwić Ci dostosowywanie kokpitu dla Twoich klientów. Jest również częścią zestawu wtyczek premium Ultimate Multisite.

Po zainstalowaniu przejdź do strony _Wtyczki_ i kliknij, aby aktywować motyw w całej sieci.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Aktywuj go dla całej sieci na stronie Wtyczki_

_**UWAGA:**_ _Jeśli tylko go zainstalujesz, będzie wyświetlany u Twoich klientów jako opcja, którą można zaznaczyć, aby dodać motyw tylko dla danego użytkownika._

Teraz Twój interfejs administracyjny został całkowicie przeprojektowany! W lewej kolumnie pojawi się również nowe menu o nazwie PRO Theme.
