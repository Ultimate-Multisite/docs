# Filtr: wu_current_set_site

Pozwala deweloperom zmodyfikować domyślne zachowanie i ustawić bieżącą witrynę w inny sposób.

## Parametry

| Nazwa | Typ | Opis |
|------|------|-------------|
| $site | `\WP_Ultimo\Models\Site` | Bieżąca witryna, którą należy ustawić. |
| $ | `self` | Bieżąca instancja klasy. |
