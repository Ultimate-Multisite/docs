# Filtr: wu_apply_plan_limits {#filter-wuapplyplanlimits}

Pozwala deweloperom pluginów na obejście ograniczeń.

Możesz użyć tego filtra, aby uruchomić dowolny kod przed rozpoczęciem sprawdzania jakichkolwiek ograniczeń. Jeśli filtr zwróci jakąkolwiek wartość uznaną za prawdziwą, proces będzie kontynuowany; jeśli zwróci jakąkolwiek wartość uznaną za fałszywą, kod się zatrzyma i żaden z poniższych hooków nie zostanie uruchomiony.

- 1.7.0
