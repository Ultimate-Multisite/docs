# Filtr: wu_password_require_special

Filtr, który decyduje, czy hasła muszą zawierać znaki specjalne.

## Parametry {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | Czy wymagać znaków specjalnych. Domyślnie `true`, gdy zasady są egzekwowane. |
| $defender_active | `bool` | Czy Defender Pro Strong Password jest aktywny. |
