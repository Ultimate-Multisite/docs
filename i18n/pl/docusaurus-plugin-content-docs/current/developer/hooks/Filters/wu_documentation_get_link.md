# Filtrowanie: wu_documentation_get_link {#filter-wudocumentationgetlink}

Pozwala deweloperom pluginów filtrować linki.

Nie jestem pewien, jak to może być przydatne, ale nie zaszkodzi mieć takiej opcji.

## Parametry {#parameters}

| Nazwa | Typ | Opis |
|------|------|-------------|
| $link | `string` | Zarejestrowany link |
| $slug | `string` | Slug używany do pobrania linku |
| $default_link | `string` | Zarejestrowany link domyślny |

### Od {#since}

- 1.7.0
