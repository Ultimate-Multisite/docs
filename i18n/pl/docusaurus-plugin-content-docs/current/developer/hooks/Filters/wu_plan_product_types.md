# Filtr: wu_plan_product_types {#filter-wuplanproducttypes}

Filtr, który określa, które typy produktów powinny być traktowane jako plany.

Ten filtr pozwala dodatkom (addons) rejestrować dodatkowe typy produktów, które mają być rozpoznawane jako plany podczas walidacji i segregacji.

## Parametry {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | Tablica typów produktów, które mają być traktowane jako plany. |

### Since {#since}

- 2.3.0
