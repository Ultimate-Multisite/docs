# Filter: wu_generated_username_from_email

Filtruje wygenerowane imię użytkownika klienta.

## Parametry

| Nazwa | Typ | Opis |
|------|------|-------------|
| $username | `string` | Wygenerowane imię użytkownika. |
| $email | `string` | Nowy adres e-mail klienta. |
| $new_user_args | `array` | Tablica argumentów dla nowego użytkownika, która może zawierać imię i nazwisko. |
| $suffix | `string` | Dodatkowy ciąg znaków do imienia użytkownika, aby uczynić go unikalnym. |
