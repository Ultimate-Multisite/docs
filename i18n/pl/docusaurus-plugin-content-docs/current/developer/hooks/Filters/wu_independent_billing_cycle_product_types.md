# Filtr: wu_independent_billing_cycle_product_types {#filter-wuindependentbillingcycleproducttypes}

Filtruje typy produktów, które mają niezależne cykle rozliczeniowe.

Produkty z niezależnymi cyklami rozliczeniowymi nie są zmuszone do dopasowania się do okresu rozliczeniowego planu w koszyku. Jest to przydatne dla produktów, takich jak rejestracje domen, które zawsze są rozliczane rocznie, niezależnie od tego, czy plan jest miesięczny, czy roczny.

## Parametry {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | Tablica typów produktów z niezależnymi cyklami rozliczeniowymi. |

- 2.5.0
