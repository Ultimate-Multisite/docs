# Filtr: wu_maybe_redirect_to_admin_panel_exclusion_list {#filter-wumayberedirecttoadminpanelexclusionlist}

Tworzy listę wykluczeń parametrów, które zapobiegają automatycznemu przekierowaniu.

Jest to konieczne, ponieważ w przeciwnym razie edytor stron (page builder) nie będzie mógł edytować strony logowania po jej zdefiniowaniu.
