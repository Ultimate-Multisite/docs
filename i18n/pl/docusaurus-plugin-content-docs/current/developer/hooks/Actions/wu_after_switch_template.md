# Akcja: wu_after_switch_template {#action-wuafterswitchtemplate}

Pozwala deweloperom pluginów podpiąć funkcje, które zostaną wywołane po tym, jak użytkownik lub superadmin zmieni szablon strony.

## Parametry {#parameters}

| Nazwa | Typ | Opis |
|------|------|-------------|
| $id | `int` | ID strony |

### Od wersji {#since}

- 1.9.8
