# Action: wu_\{$this->id\}_scripts {#action-wuthis-idscripts}

Wyzwala (triggeruje) hook do dodawania skryptów (enqueue scripts).

Element używa tego hooka, aby przypisać (hook) swoją metodę `register_scripts`.
