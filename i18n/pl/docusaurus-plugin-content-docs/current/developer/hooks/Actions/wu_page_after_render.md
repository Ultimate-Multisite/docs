# Akcja: wu_page_after_render

Pozwala deweloperom pluginów dodać dodatkową zawartość po wyświetleniu strony.

## Parametry

| Name | Type | Description |
|------|------|-------------|
| $page_id | `string` | Identyfikator tej strony |
| $page | `object` | Obiekt strony. |

### Od wersji

- 1.8.2
