# Akcja: wu_page_before_render {#action-wupagebeforerender}

Pozwala deweloperom pluginów na dodanie dodatkowej zawartości przed wyświetleniem strony.

## Parametry {#parameters}

| Nazwa | Typ | Opis |
|------|------|-------------|
| $page_id | `string` | Id tej strony. |
| $page | `object` | Obiekt strony. |
