# Dokumentacja Ultimate Multisite

Witamy w dokumentacji Ultimate Multisite. Ultimate Multisite przekształca WordPress Multisite w pełną platformę Website as a Service (WaaS).

## Sekcje Dokumentacji

### [Przewodnik dla Użytkownika](/user-guide/getting-started/ultimate-multisite-101)

Wszystko, czego potrzebujesz, aby skonfigurować, ustawić i zarządzać swoją platformą WaaS. Obejmuje instalację, bramki płatności, mapowanie domen, zarządzanie klientami i więcej.

### [Dokumentacja dla Deweloperów](/developer/hooks)

Techniczny przewodnik dla deweloperów tworzących na bazie Ultimate Multisite. Zawiera odniesienie do hooków, dokumentację REST API oraz przewodniki po tworzeniu addonów.

### [Addony](/addons)

Dokumentacja dla płatnych addon pluginów, które rozszerzają Ultimate Multisite o dodatkowe funkcje, takie jak integracja z WooCommerce, analityka, sprzedaż domen i inne.
