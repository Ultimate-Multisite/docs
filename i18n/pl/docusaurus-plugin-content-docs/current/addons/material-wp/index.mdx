# Material WP

Interfejs administracyjny WordPressa nie zmieniał się od lat. Ale zastanawiałeś się kiedyś, jak wyglądałby interfejs administracyjny w wersji Material Design?

Material WP to motyw administracyjny stworzony, aby nadać mu ten wygląd i wrażenia, pomagając Ci dostarczyć klientom bardziej spersonalizowany Dashboard.

## Features {#features}

- Custom CSS support (Obsługa niestandardowego CSS)
- Easy styling settings (Łatwe ustawienia stylizacji)
- Menu reordering (Ponowne układanie menu)
- Easy import/export (Łatwy import/eksport)

## Documentation {#documentation}

**Customizing your Admin Panel with Material WP**

## Installation {#installation}

1. Upload 'material-wp' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

## Usage Guide {#usage-guide}
