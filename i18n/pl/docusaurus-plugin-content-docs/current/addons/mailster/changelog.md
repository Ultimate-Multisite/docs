# Dziennik zmian Mailster {#mailster-changelog}

### 1.0.0 {#100}
* Pierwsze wydanie
* Automatyczna synchronizacja subskrybentów po rejestracji lub dokonaniu płatności
* Możliwość przypisywania subskrybentów do list dedykowanych konkretnemu produktowi
* Zgoda oparta na checkboxach, zapewniająca zgodność z RODO
* Obsługa podwójnej zgody (double opt-in)
* Kompleksowe mapowanie pól
* Szczegółowe logowanie działań
