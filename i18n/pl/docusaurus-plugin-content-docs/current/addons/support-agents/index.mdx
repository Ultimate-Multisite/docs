# Agenci Wsparcia

Zarządzanie całą siecią samodzielnie nie jest łatwym zadaniem. W miarę jak Twoja sieć zaczyna rosnąć, zauważysz rosnące zapotrzebowanie na wsparcie. Dobrym pomysłem będzie zatrudnienie osób, które pomogą Ci zmniejszyć obciążenie pracą. Jednak nadawanie pełnego dostępu/uprawnień jest ryzykowne i może narazić Twoją sieć na kompromitację. Właśnie tutaj wchodzi dodatek Support Agent.

Ten dodatek pozwala stworzyć nowy typ użytkownika WordPress o nazwie Support Agent i daje Ci pełną kontrolę nad uprawnieniami tego użytkownika. Nadanie uprawnień jest tak proste, jak włączenie przełącznika, a wyłączenie jest równie łatwe, jeśli zdecydujesz się zrezygnować z tego dostępu.

## Funkcje {#features}

- Szczegółowa lista uprawnień WordPress i Ultimate Multisite
- Łatwe włączanie i wyłączanie dostępu
- Możliwość utworzenia nieograniczonej liczby kont agentów

## Instalacja {#installation}

1. Prześlij 'ultimate-multisite-support-agents' do katalogu '/wp-content/plugins/'
2. Aktywuj plugin w menu 'Plugins' w WordPress
3. Postępuj zgodnie z kreatorem krok po kroku, aby skonfigurować plugin
