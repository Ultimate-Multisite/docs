# Akcja: wu_page_enqueue_scripts

Dodajemy tu niestandardową akcję, aby mieliśmy miejsce na ładowanie skryptów i stylów w razie potrzeby oraz aby upewnić się, że zostaną one załadowane tylko na odpowiednich stronach.

## Parametry

| Nazwa | Typ | Opis |
|------|------|-------------|
| $ | `\WU_Page` | Aktualna strona administracyjna |

### Od

- 1.3.0
### Źródło

Zdefiniowane w [`inc/class-wu-admin-pages.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-admin-page-creator/blob/main/inc/class-wu-admin-pages.php#L1123) w wierszu 1123
