# Admin Page Creator

## Content {#content}
Admin Page Creator pozwoli Ci łatwo dodawać nowe strony administracyjne i podstrony w panelu administracyjnym WordPress. Można go wykorzystać do stworzenia strony powitalnej lub strony instruktażowej, która może pełnić funkcję strony docelowej dla nowych użytkowników. Możesz użyć wizualnych edytorów stron, takich jak Beaver Builder, Elementor, Brizy czy Oxygen, aby zaprojektować strony bez problemów.

Oferuje również obsługę niestandardowego HTML, CSS i JavaScript, a także kontrolę dostępu opartą na planie i roli użytkownika!

## Features {#features}
* Obsługa niestandardowego HTML i CSS
* Obsługa JavaScript
* Tryb treści PHP
* Uprawnienia oparte na roli użytkownika
* Obsługa wizualnych edytorów stron (page builders)

## Documentation {#documentation}
[Tworzenie niestandardowych stron i podstron administracyjnych za pomocą Ultimate Multisite: Admin Page Creator](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## Installation {#installation}

1. Prześlij **ultimate-multisite-admin-page-creator** do katalogu **/wp-content/plugins/**
2. Aktywuj wtyczkę w menu **Plugins** w panelu administracyjnym Twojej sieci WordPress.

## Usage Guide {#usage-guide}

Podczas gdy WordPress sprawia, że dodawanie treści na stronę główną jest bardzo proste, tworzenie stron i podstron po stronie administracyjnej wymaga pewnych podstawowych umiejętności kodowania. Ultimate Multisite: Admin Page Creator ma na celu przeniesienie tej samej łatwości użytkowania, jaką znamy ze strony front-end, do back-end Twojej sieci stron.

## Why use Admin Page Creator? {#why-use-admin-page-creator}

Może chcesz zaoferować niestandardową stronę na panelu administracyjnym swoich klientów z poradnikami, może chcesz dodać niestandardową stronę z FAQ i widgetem kontaktowym. Możliwości z Admin Page Creator są nieograniczone, a właśnie to dziś sprawdzimy w poradniku.

## Creating your first Custom Admin Page {#creating-your-first-custom-admin-page}

Po zainstalowaniu i aktywacji Ultimate Multisite: Admin Page Creator przejdź do menu administracyjnego sieci i wybierz Admin Pages.

![Custom Admin Pages list on Network Admin](/img/admin/admin-pages-list.png)

_Lista niestandardowych stron administracyjnych w panelu sieci_

Zobaczysz listę wszystkich stron administracyjnych, które stworzyłeś do tej pory (ponieważ jest to Twoja pierwsza wizyta, lista będzie pusta). Kliknij **Add new Admin Page**. Przekieruje Cię to do edytora strony administracyjnej.

## The elements of a Custom Admin Page {#the-elements-of-a-custom-admin-page}

W edytorze strony administracyjnej masz wiele różnych opcji, aby dostosować swoją stronę.

### Page Title {#page-title}

Jest to tytuł, który pojawi się na górze strony, a nie etykieta w menu administracyjnym.

Jeśli nie znasz HTML, trzymaj się domyślnego edytora WordPressa. Będziesz mógł edytować zawartość strony tak samo, jakbyś robił to w normalnym Poście lub Stronie.

## Menu Options {#menu-options}

Po prawej stronie, w meta-boksie General Options, będziesz mógł skonfigurować typ menu, etykietę i wiele więcej.

### Menu Types {#menu-types}

Twoja nowa niestandardowa strona administracyjna może mieć dwa różne typy menu: menu główne (top-level menu) i podmenu (submenu).

Strona menu głównego, jak sugeruje nazwa, doda Twoją nowo utworzoną stronę do menu administracyjnego obok domyślnych stron WordPressa.

Jeśli wybierzesz opcję top-level, będziesz mógł również wybrać kolejność menu i ikonę menu.

Jeśli chcesz, aby ta strona była podstroną, możesz wybrać stronę nadrzędną, do której chcesz przypisać tę stronę. Możesz dodać swoją niestandardową stronę do domyślnych stron WordPressa lub do swoich niestandardowych stron głównych.

## Advanced Options {#advanced-options}

Powinieneś być w stanie dodać wszelkiego rodzaju treści do swoich niestandardowych stron administracyjnych. Obejmuje to reguły i pliki CSS, JavaScript oraz zewnętrzne biblioteki. Możesz to zrobić za pomocą zakładki Advanced Options.

### Custom CSS {#custom-css}

W zakładce CSS możesz dodać własny niestandardowy kod CSS, a także importować pliki CSS z zewnętrznych stron (takie jak CDN, Google Fonts itp.).

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_Wspierany jest również niestandardowy CSS. Możesz również dołączać zewnętrzne pliki stylów!_

### Custom JavaScript {#custom-javascript}

To samo dotyczy JavaScript. Możesz dodawać zewnętrzne biblioteki do użycia w swoim niestandardowym kodzie JavaScript.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Dodaj własny kod JavaScript!_

## Permissions {#permissions}

Także w Advanced Options będziesz mógł wybrać, które podstrony będą miały dostęp do tej strony. Jest to przydatne, jeśli planujesz dostarczać różne ekskluzywne treści dla różnych poziomów planów, na przykład.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Masz pełną kontrolę nad tym, gdzie ta strona się wyświetli_

**Important** : Aby strona była wyświetlana, muszą być spełnione oba warunki. Oznacza to, że jeśli użytkownik jest klientem danego planu, ale nie ma jednej z wybranych ról (lub odwrotnie), strona nie pojawi się na jego panelu.

## Active {#active}

Jeśli chcesz wyłączyć niestandardową stronę administracyjną, nie usuwając jej ani nie bawiąc się uprawnieniami, po prostu odznacz opcję Active.

Wyłączenie strony głównej automatycznie wyłączy wszystkie podstrony przypisane do tej strony nadrzędnej.
