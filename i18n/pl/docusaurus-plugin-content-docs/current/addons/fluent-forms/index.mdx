# Fluent Forms

Ten dodatek umożliwia dodanie limitów do Fluent Forms w Ultimate Multisite, stosując je w produktach, członkostwach lub na stronach.
Fluent Forms używa niestandardowych tabel zamiast niestandardowych typów wpisów, dlatego standardowe opcje ograniczające nie działają z Fluent Forms.

## Funkcje {#features}

- Dodawanie limitów dla Fluent Forms lub Conversational Fluent Forms
- Klienci otrzymują przyjazną wiadomość zachęcającą ich do ulepszenia planu, gdy osiągną limit
