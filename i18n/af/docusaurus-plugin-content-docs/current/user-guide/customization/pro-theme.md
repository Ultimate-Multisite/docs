# Pas jou Admin Paneel aan met PRO Theme {#customizing-your-admin-panel-with-pro-theme}

Dikwels, wanneer ons kliëntwerk doen, kom die behoefte om aan ons kliënte 'n aangepaste ervaring in die WordPress admin dashboard te bied. As jy op soek is na 'n meer professionele voorkoms en gevoel, kan PRO Theme jou help.

PRO Theme is 'n admin-tema wat ontwikkel is om jou te help wanneer jy jou kliënte se Dashboard aanpas. Dit is ook deel van Ultimate Multisite se premium plugins core.

Nadat jy dit geïnstalleer het, gaan na jou _Plugins_ bladsy en klik om dit op jou Netwerk te aktiveer.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Activate it network-wide on your Plugins page_

_**NOTA:**_ _As jy dit net installeer, sal dit op jou kliënt se skerm as 'n opsie verskyn wat gemerk kan word om die tema slegs aan daardie gebruiker by te voeg._

Nou het jy jou admin-koppelvlak heeltemal herontwerp! Jy het ook 'n nuwe meny op die linkerkolom wat PRO Theme genoem word.
