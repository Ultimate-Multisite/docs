# Plaas Advertensies op jou netwerk met Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector is 'n **gratis bykomende funksie** wat jou die gereedskap gee om advertensies op jou planne te plaas direk vanaf hul redigeerblad.

Nadat jy Ultimate Multisite: Ad Injector geïnstalleer het, gaan na jou _Plugins_-bladsy om dit te aktiveer.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Klik op die Network Activate-knoppie_

Sodra die bykomende funksie aktief is, sal die _Advanced Options_-meta-boks op die planredigeerblad 'n nuwe opsie hê genaamd _Ad Settings_.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_Nou kan jy advertensie-instellings vir elkeen van jou planne redigeer_

Op daardie tab kan jy die advertensie-kodes vir plase en bladsye in die front-end, voor en na die inhoud, byvoeg.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Nadat jy die kode van jou advertensie ingesit het, sal dit op die plase en bladsye van daardie spesifieke plan verskyn._

Jy kan ook advertensies laat insluit op die kop van Admin Pages vir gebruikers van daardie plan.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Gaan na die Admin Panel Ad Code opsie as jy advertensies op jou gebruiker se back-end wil wys_

### Verder as Advertensies {#beyond-advertising}

Jy kan ook Ultimate Multisite: Ad Injector gebruik om ondersteuning te bied aan hoër-tier planne.

Gaan net na die _Admin Panel Ads_-opsie en voeg die HTML-kode vir jou ondersteuningsgereedskap by.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Voeg kliëntondersteuning by aan spesifieke planne_

Op hierdie manier kan mense jou direk vanaf hul Admin Panel bereik!
