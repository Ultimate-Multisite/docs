# Hoe om spesiale ontwikkelingshulp van Codeable te kry?

Soms het ons regtig fantastiese aanpassings nodig om ons netwerk presies so te laat loop hoe ons wil. As dit is wat jy soek, beveel ons [Codeable] aan, 'n plek waar jy ervare ontwikkelaars met Ultimate Multisite-kennis kan vind.

[Codeable] is 'n platform wat kliënte verbind met gekwalifiseerde WordPress freelance ontwikkelaars, wat op projekte van enige grootte kan werk. Hulle het 'n noukeurige aansoekproses om te verseker dat jy gekoppel word aan kundiges wat spesifiek gekwalifiseer is vir die taak wat jy in gedagte het.

### Hoe om 'n projek in Codeable te skep? {#how-to-create-a-project-in-codeable}

Gaan eers na [Codeable] en klik op die **Start a Project** knoppie.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Om nou 'n projek in te dien, klik aan die linkerkant op **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

Kies **Customizing** in "I need help with" (Ek het hulp met):

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

Kies **Plugin** in "My" (My):

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Klik dan op die **Continue** knoppie.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Nou is dit tyd om jou projek en behoeftes op die beste manier moontlik te beskryf. Probeer om soveel moontlike inligting te gee!

<!-- Screenshot unavailable: Codeable project description form -->

Beantwoord die volgende vrae om te help met die prysbepaling.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Daarna, indien jy nog nie 'n [Codeable] rekening het nie, skep dan een en klik op die **Publish My Project** knoppie.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

Sodra jy jou [Codeable] projek gepubliseer het, sal die platform jou verbind met ontwikkelaars wat by die vaardigheidsset wat vir jou projek benodig word, pas.

Hierdie proses sal help om die regte persoon vir jou werk te vind, iemand wat ervaring het met WordPress en Ultimate Multisite!

Codeable se minimum tarief is $70–$120 USD per uur. Kwantifiseringe sal verander volgens die omvang van die werk, kompleksiteit en dringendheid van 'n projek.

Sodra jy jou kwantifisering ontvang het, sal jy in staat wees om die ontwikkelaar te huur wat die beste by jou behoeftes pas en jou projek saam te begin!
