# Ultimate Multisite Hooks en filters

Ultimate Multisite bestaan uit 'n hele lading funksies en database-navrae. Dit het *hooks* en *filters* wat jou sal toelaat om bestaande funksies te wysig of nuwe kode by te voeg.

Ons het 'n lys van hierdie *hooks* en *filters* geskep as 'n verwysing om gevorderde gebruikers te help om hul netwerk verder aan te pas. Jy kan die lys vanaf Github raadpleeg deur op die skakel hieronder te klik.
