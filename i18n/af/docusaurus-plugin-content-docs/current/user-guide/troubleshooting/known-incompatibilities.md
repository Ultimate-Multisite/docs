# Bekende onverenigbaarhede met ander Plugins

Nadat jy Ultimate Multisite geïnstalleer en geaktiveer het, kan jy probleme ondervind as gevolg van onbehandelde onverenigbaarhede met ander plugins.

Ons werk aktief daaraan om hierdie lys met elke nuwe vrystelling te verminder, maar as 'n plugin hier gelys word, mag jy dit deaktiveer moet om 'n volkome funksionele Ultimate Multisite Network te hê.

Hier is 'n lys daarvan en wat jy in hierdie gevalle moet doen:

**Sommige WPMU DEV plugins** Sommige van die plugins wat nie meer deur WPMU DEV onderhou word nie, is bekend daarvoor dat hulle die Ultimate Multisite-registrasieproses op die laaste stap laat stop (die gebruiker kan nie 'n rekening skep nie). Hier is 'n lys van die plugins wat daardie probleme veroorsaak:

  * Signup Language: _Ultimate Multisite het 'n add-on wat dieselfde funksionaliteit bied, genaamd Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Dit is ingebou in Ultimate Multisite, so daar is nie 'n ekstra plugin nodig nie._

  * New Blog Templates: _Dit is ingebou in Ultimate Multisite, so daar is nie 'n ekstra plugin nodig nie._

  * Set Password on Multisite Blog Creation: _Dit is ingebou in Ultimate Multisite, so daar is nie 'n ekstra plugin nodig nie._

  * Signup TOS: _Dit is ingebou in Ultimate Multisite, so daar is nie 'n ekstra plugin nodig nie._

  * Anti-Splog: Alhoewel minder kragtig, is dit ook _ingebou in Ultimate Multisite._

**Nota** : Alle ander WPMU Dev plugins kan normaalweg saam met Ultimate Multisite gebruik word. Voorbeelde sluit in _Smush_ , _Forminator_ , _Defender,_ ens.
