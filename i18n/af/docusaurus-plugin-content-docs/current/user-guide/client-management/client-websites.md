# Jou Klant se Webwerwe (v2)

Nadat hulle op jou netwerk geregistreer het, kan jou kliënte hul dashboard bereik.

Behalwe die **Account**-bladsy, waar hulle belangrike inligting oor hul betalings, lidmaatskappe, domeine, planbeperkings en meer kan sien, het hulle ook toegang tot die **My Sites**-bladsy. Hier kan hulle al hul webwerwe sien en bestuur. Alles wat hulle hoef te doen, is om op My Sites op hul dashboard te klik.

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

Hulle sal al hul webwerwe gesien, met 'n **Manage**-knoppie direk onder elkeen. As hulle daarop klik, sal hulle na die **dashboard van hierdie spesifieke webwerf** herlei word.

Hulle kan ook sien watter een van hul webwerwe die **primêre** een is vir planne wat hulle toelaat om meer as een webwerf te hê.

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

Hier is 'n volledige uitsig van die webwerf-redigeer-koppelvlak:

![Site edit full interface](/img/admin/site-edit-full.png)

As hulle hul primêre webwerf wil verander, moet hulle na die Account-bladsy gaan deur op die **Account**-menu te klik.

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

Jy kan ook kliëntbesonderhede sien en redigeer:

![Customer edit interface](/img/admin/customer-edit.png)

Daarna moet hulle op **Change Default Site** klik.

![Change Default Site button on account page](/img/account-page/change-default-site.png)

'n Pop-up venster sal verskyn. Klik op die afhanglys en kies watter webwerf jy as die primêre webwerf wil stel.

![Change Default Site dropdown selection](/img/account-page/change-default-site-dropdown.png)

Klik dan net op **Change Default Site** en jy is klaar!

![Confirming the default site change](/img/account-page/change-default-site-confirm.png)
