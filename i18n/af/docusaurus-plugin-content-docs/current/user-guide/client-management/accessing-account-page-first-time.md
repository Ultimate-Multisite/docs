# Hoe kan ek vir die eerste keer toegang tot my rekeningblad kry? {#how-can-i-access-my-account-page-for-the-first-time}

Welkom by Ultimate Multisite!

Toegang tot jou rekeningblad kry jy deur die skakel wat jy in jou e-pos ontvang het nadat jy ons plugin gekoop het:

<!-- Screenshot unavailable: Account page email with access link -->

Gebruik die e-posadres wat jy tydens die aankoop verskaf het en die wagwoord wat jy in jou e-pos ontvang het.

<!-- Screenshot unavailable: Login page with email and password fields -->

Aangesien dit jou eerste keer is wat jy in hierdie rekening aanmeld, sal jy gevra word om jou wagwoord te verander. Verander dit en klik op die _Save_ knoppie.

<!-- Screenshot unavailable: Change password form with Save button -->

Op jou rekeningblad kan jy al jou rekeningbesonderhede bestuur, soos fakturering, hernuwings, kansellasies, en baie ander dinge.
