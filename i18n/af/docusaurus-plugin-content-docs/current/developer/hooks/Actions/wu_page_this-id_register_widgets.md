# Action: wu_page_{$this->id}_register_widgets

Word geaktiveer nadat widgets vir hierdie bladsy geregistreer is.

Die dinamiese deel van die hook se naam, `$this->id`, verwys na die bladsy-ID.

## Parameters {#parameters}

| Naam | Tipe | Beskrywing |
|------|------|-------------|
| $id | `string` | Die bladsy-ID. |
| $page_hook | `string` | Die bladsy-hook. |
| $page | `object` | Die bladsy-objek. |
