# Aksie: wu_site_created {#action-wusitecreated}

Word geaktiveer nadat 'n webwerf vir die eerste keer geskep is.

Word nie geaktiveer as dit van 'n sjabloon gekopieer (geduplikeer) word nie.

## Parameters {#parameters}

| Naam | Tipe | Beskrywing |
|------|------|-------------|
| $data | `array` | Die objekdata wat gestoor sal word. |
| $site | `\Site` | Die objek-instansie. |
