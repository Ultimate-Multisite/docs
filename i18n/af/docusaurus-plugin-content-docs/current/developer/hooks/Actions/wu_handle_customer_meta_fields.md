# Aksie: wu_handle_customer_meta_fields {#action-wuhandlecustomermetafields}

Dit laat plugin-ontwikkelaars toe om meta-data op verskillende maniere te stoor indien hulle dit nodig het.

## Parameters {#parameters}

| Naam | Tipe | Beskrywing |
|------|------|-------------|
| $meta_repository | `array` | Die lys van meta-velde, gestruktureer as sleutel => waarde. |
| $customer | `\Customer` | Die Ultimate Multisite kliënt-objek. |
| $checkout | `\Checkout` | Die checkout-klas. |

- 2.0.0
