# Aksie: wu_page_after_render

Dit laat plugin-ontwikkelaars toe om bykomende inhoud by te voeg nadat ons die bladsy gerender het.

## Parameters {#parameters}

| Naam | Tipe | Beskrywing |
|------|------|-------------|
| $page_id | `string` | Die ID van hierdie bladsy. |
| $page | `object` | Die bladsy-objek. |
