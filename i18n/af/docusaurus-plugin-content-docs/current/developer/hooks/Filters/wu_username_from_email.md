# Filter: wu_username_from_email

Filter die gebruikersnaam vir 'n nuwe kliënt.

## Parameters

| Naam | Tipe | Beskrywing |
|------|------|-------------|
| $username | `string` | Die kliënt se gebruikersnaam. |
| $email | `string` | Die nuwe kliënt se e-posadres. |
| $new_user_args | `array` | 'n Array van nuwe gebruikersargs, wat dalk voor- en vanname insluit. |
| $suffix | `string` | Voeg 'n string by die gebruikersnaam om dit uniek te maak. |
