# Filter: wu_independent_billing_cycle_product_types

Filter die produktipe wat onafhanklike faktuurwenkele het.

Produkte met onafhanklike faktuurwenkele word nie gedwing om by die plan se faktuurperiode in die winkelmand te pas nie. Dit is nuttig vir produkte soos domeinregistrasies wat altyd jaarliks factureer, ongeag of die plan maandeliks of jaarliks is.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | Array van produktipe met onafhanklike faktuurwenkele. |
