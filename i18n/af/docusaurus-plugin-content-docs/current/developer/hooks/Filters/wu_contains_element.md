# Filter: wu_contains_element

Dit laat ontwikkelaars toe om die resultate van die aanvanklike soektog aan te pas.

Dit is nuttig vir derdepartye bouers en soortgelyke dinge.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $contains_elements | `bool` | Indien die element op die inhoud ingesluit is. |
| $content | `string` | Die inhoud wat ondersoek word. |
| $element | `self` | Die huidige element. |
| $post | `null\|\WP_Post` | die pos om te kontroleer. |
