# Filter: wu_hosting_integration_wizard_sections {#filter-wuhostingintegrationwizardsections}

Filter die wizard-seksies vir hosting-integrasie-opstelling.

Dit laat bykomende modules (addons) toe om wizard-seksies by te voeg, te verwyder, of te wysig.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $sections | `array` | Wizard-seksies. |
| $integration | `\WP_Ultimo\Integrations\Host_Providers\Base_Host_Provider\|\WP_Ultimo\Integrations\Integration` | Die integrasie wat gekonfigureer word. |
| $page | `\Hosting_Integration_Wizard_Admin_Page` | Die wizard-bladsy-instansie. |
