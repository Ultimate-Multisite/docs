# Filter: wp_ultimo_render_vars

Laat plugin-ontwikkelaars ekstra veranderlikes aan die render-konteks wêreldwyd byvoeg.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $args | `array` | 'n Array wat veranderlikes bevat wat deur die render-oproep oorgedra word. |
| $view | `string` | Naam van die weergawes wat gerender word. |
| $default_view | `string` | Naam van die terugval-weergawes. |

### Sedert

- 2.0.0
### Bron

Gedefinieer in [`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L31) op lyn 31
