# Filter: wu_apply_plan_limits {#filter-wuapplyplanlimits}

Stel plugin-ontwikkelaars in staat om die beperkings te omseil.

Jy kan hierdie filter gebruik om willekeurige kode te laat loop voordat enige van die beperkings geïnisieer word. As die filter enige waar-waarde (truthy value) teruggee, sal die proses voortgaan; as dit enige onwaar-waarde (falsy value) teruggee, sal die kode terugkeer en sal nie enige van die hooks hieronder loop nie.
