# Filter: wu_stripe_existing_product_id

Filter die ID van die produk wat nagegaan moet word. As dit bestaan, sal die nuwe intekening hierdie produk gebruik.

## Parameters {#parameters}

| Naam | Tipe | Beskrywing |
|------|------|-------------|
| $product_id | `string` | Die ID van die Stripe-produk wat nagegaan moet word. |
| $name | `string` | Die produknaam. |
