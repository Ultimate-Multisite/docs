# Filter: wu_documentation_get_link

Stel plugin-ontwikkelaars in staat om skakels te filter.

Ek is nie seker hoe dit nuttig kan wees nie, maar dit skad nie om dit te hê.

## Parameters

| Naam | Tipe | Beskrywing |
|------|------|-------------|
| $link | `string` | Die geregistreerde skakel |
| $slug | `string` | Die slug wat gebruik word om die skakel te haal |
| $default_link | `string` | Die geregistreerde standaard skakel |

### Sedert

- 1.7.0
