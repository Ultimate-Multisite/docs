# Filter: wu_connector_enforcement_option_keys

Filter die lys van AI-verskaffer-optie sleutels wat vanaf die hoofwebwerf geafdwing word.

Voeg optie sleutels by vir maatgemaakte of derdeparty AI-verskaffer-plugins sodat hul instellings ook op subwerwe van die hoofwebwerf geërf word.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $keys | `string[]` | Naam van optie sleutels wat geafdwing moet word (sluit reeds dinamies ontdekte koppelvlak-sleutels en EXTRA_PROVIDER_OPTIONS in). |
