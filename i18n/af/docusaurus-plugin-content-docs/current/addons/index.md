# Ultimate Multisite Addons

Add-ons brei Ultimate Multisite uit met ekstra funksies en integrasies. Die meeste add-ons is betaal en kan gekoop word vanaf [ultimatemultisite.com](https://ultimatemultisite.com).

## Installasie

Add-ons kan op twee maniere geïnstalleer word:

1. **Via die Add-on Bladsy** -- Navigeer na die add-on bestuur bladsy van jou webwerf in die WordPress admin
2. **Handmatige Aflaai** -- Laai af vanaf [your account](https://ultimatemultisite.com/my-account/downloads/) en laai op via Plugins > Add New
