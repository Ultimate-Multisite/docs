# Material WP

Die WordPress admin-koppelvlak het al jare dieselfde geword. Maar het jy al ooit gewonder hoe 'n Material Design-weergawe van die admin-koppelvlak sou lyk?

Material WP is 'n admin-tema wat ontwikkel is om jou daardie voorkoms en gevoel te gee, en jou te help om jou kliënte 'n meer aangepaste Dashboard te bied.

## Features

- Custom CSS ondersteuning
- Eenvoudige styling-instellings
- Menureordering
- Eenvoudige invoer/uitvoer

## Documentation

**Jou Admin Paneel met Material WP aanpas**

## Installation

1. Laai 'material-wp' op die '/wp-content/plugins/' gids
2. Aktiveer die plugin via die 'Plugins' menu in WordPress
