# Multi-Currency

Ultimate Multisite: Multi-Currency laat jou netwerk toe om betalings in verskeie valuut te aanvaar. Dit konverteer pryse outomaties met behulp van live of handmatige wisselkoerse, laat jou om presiese prys per produk vir enige valuut te stel, en detecteer elke besoeker se voorkeurvaluut via geolokasie — alles bestuur vanaf een enkele admin dashboard.

## Sleutelkenmerke

- **Outomatiese wisselkoerskonversie** — pryse word van jou basisvaluut gekonverteer met behulp van koerse wat van jou gekose verskaffer verkry word
- **Oortredings van prys per produk** — stel presiese valuut-spesifieke pryse op enige produk om outomatiese konversie te omseil
- **Valuutkiezer-registrasieveld** — laat kliënte om hul voorkeurvaluut te kies tydens afrekening/registrasie
- **Verskeie wisselkoersverskaffers** — ECB, Frankfurter, Stripe, of volledig handmatige koerse
- **Valuutdeteksie gebaseer op geolokasie** — stel outomaties die valuut in wat by die besoeker se land pas
- **Admin dashboard** — bestuur aktiewe valuut, wisselkoerse en verskafferinstellings vanaf een plek

## Vereistes

- Ultimate Multisite 2.0.0 of hoër
- WordPress Multisite installasie
- PHP 7.4 of hoër

## Instalering

1. Laai die plugin-lêre op na `/wp-content/plugins/ultimate-multisite-multi-currency`
2. Aktiveer die plugin deur die **Plugins** skerm in WordPress (Netwerk Aktiveer)
3. Navigeer na **Ultimate Multisite → Multi-Currency** om die admin dashboard oop te maak
4. Voeg die valuut by wat jy wil aanvaar en konfigureer jou wisselkoersverskaffer

## Konfigurasie

### Admin Dashboard

Die Multi-Currency admin dashboard (**Ultimate Multisite → Multi-Currency**) is die sentrale plek om:

- Valuut by te voeg, te aktiveer of te deaktiveer
- 'n wisselkoersverskaffer te kies en koerse op aanvraag te verfris
- Huidige koerse vir alle aktiewe valuut te hersien
- Handmatige koerse te stel wanneer outomatiese verskaffers nie geskik is nie

### Wisselkoersverskaffers

| Verskaffer | Beskrywing |
|---|---|
| **ECB** | European Central Bank — gratis daaglikse koerse vir EUR-gebaseerde netwerke |
| **Frankfurter** | Open-source ECB spieël API — gratis, geen API sleutel vereis nie |
| **Stripe** | Gebruik Stripe se live FX koerse — vereis 'n aktiewe Stripe integrasie |
| **Handmatig** | Jy voer koerse direk in; geen eksterne API oproepe nie |

Kies jou verskaffer onder **Ultimate Multisite → Multi-Currency → Settings → Exchange Rate Provider**. Koerse kan op enige tyd handmatig verfris word of geskeduleer word om outomaties op te dateer.

### Oortredings van Prys per Produk

Wanneer outomatiese konversie nie presies genoeg is nie, kan jy 'n presiese prys vir 'n spesifieke valuut op enige produk stel:

1. Maak die produkredigeerder oop (**Ultimate Multisite → Products → Edit**)
2. Skrol na die **Currency Pricing** afdeling
3. Voer die presiese prys vir elke valuut in wat jy wil oortskry
4. Stoor die produk

Produkte sonder 'n oortreding gebruik die gekonverteerde prys van jou basisvaluut.

### Valuutkiezer-Registrasieveld

Die valuutkiezerveld kan by jou registrasie/afrekening-proses bygevoeg word sodat kliënte hul voorkeurvaluut by registrasie kies:

1. Gaan na **Ultimate Multisite → Multi-Currency → Settings**
2. Aktiveer **Show currency selector on signup**
3. Stel opsioneel 'n standaardvaluut in of laat geolokasie die voorafgekoese waarde bepaal

Die gekose valuut word gestoor met die inskrywing en gebruik vir alle daaropvolgende fakturering.

### Valuutdeteksie Gebaseer op Geolokasie

Wanneer geolokasie-deteksie geaktiveer is, kaart die addon die besoeker se land aan 'n valuut en stel dit vooraf in die valuutkiezerveld. Om dit te aktiveer:

1. Gaan na **Ultimate Multisite → Multi-Currency → Settings**
2. Aktiveer **Detect currency from visitor location**
3. Kaart lande na valuut in die **Geolocation Mapping** tabel

As geen kaarting bestaan vir 'n besoeker se land nie, word die netwerk se basisvaluut as die terugval gebruik.

## Algemene Vrae

### Wat gebeur as 'n wisselkoersverskaffer nie beskikbaar is nie?

Die addon val terug op die laaste suksesvol verkryde koerse. As nog koerse verkry is, val dit terug op die handmatige koerse wat jy gekonfigureer het. Dit word aanbeveel om handmatige koerse as 'n veiligheidsnet te konfigureer.

### Kan ek verskillende verskaffers vir verskillende valuut gebruik?

Nee — een verskaffer is op enige oomblik aktief vir alle valuut. Gebruik **Handmatig** as jy volledige beheer oor individuele koerse benodig.

### Vereis geolokasie 'n aparte plugin?

Geen eksterne geolokasie plugin is nodig nie. Die addon gebruik WordPress se ingeboude IP-na-land deteksie. Vir hoër akkuraatheid kan 'n toegewyde geolokasie diens geïntegreer word via die voorsiene filter-hooks.

### Word prys-oortredings toegepas voor of na belasting?

Prys-oortredings stel die prys voor belasting in die teikenvaluut. Belasting word bereken op die oortredingsprys met behulp van jou bestaande belastingkonfigurasie.
