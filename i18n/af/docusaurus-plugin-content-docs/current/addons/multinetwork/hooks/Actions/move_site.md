# Aksie: move_site

Word geaktiveer nadat 'n webwerf na 'n nuwe netwerk beweeg is.

## Parameters

| Naam | Tipe | Beskrywing |
|------|------|-------------|
| $site_id | `int` | Die ID van die webwerf wat beweeg is. |
| $old_network_id | `int` | Die ID van die oorspronklike netwerk vir die webwerf. |
| $new_network_id | `int` | Die ID van die netwerk waarna die webwerf beweeg is. |

- 1.3.0
