# Aksie: wp_ultimo_support_agents_load {#action-wpultimosupportagentsload}

Dit laat ontwikkelaars van plugins toe om nuwe funksionaliteit by te voeg. Byvoorbeeld, ondersteuning vir nuwe Hosting-aanbieders, ens.
