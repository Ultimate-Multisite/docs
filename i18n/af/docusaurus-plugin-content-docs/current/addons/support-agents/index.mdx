# Ondersteuningsagents

Om 'n heel hele netwerk alleen te bestuur, is nie 'n maklike taak nie. Soos jou netwerk begin groei, sal jy 'n toenemende vraag na ondersteuning sien. Dit sal 'n goeie idee wees om mense in te roep wat jou werkdruk kan verlig. Die gee van volle toegang/vergunning is egter risikvol en kan jou netwerk kompromitteer. Dit is waar hierdie Support Agent-inkoppelting in die spel kom.

Hierdie add-on laat jou toe om 'n nuwe tipe WordPress-gebruiker genaamd Ondersteuningsagent te skep, en gee jou volledige beheer oor watter toestemmings daardie gebruiker het. Die toekenning van toestemming is so maklik soos om 'n wisselaarknoppie aan te skakel en dit af te skakel as jy besluit dit te verwyder.

## Kenmerke

- Itemiseerde lys van WordPress & Ultimate Multisite-toestemming
- Aktiveer & deaktiveer toegang met gemak
- Skep onbeperkte aantal Agent-rekeninge

## Installasie

1. Laai 'ultimate-multisite-support-agents' op na die '/wp-content/plugins/' gids
2. Aktiveer die plugin via die 'Plugins' kieslys in WordPress
3. Volg die stap-vir-stap Wizard om die plugin op te stel
