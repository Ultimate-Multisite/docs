# Admin Page Creator

## Inhoud
Admin page creator sal jou toelaat om maklik nuwe admin-sidene en sub-sidene binne die WordPress admin dashboard by te voeg. Dit kan gebruik word om 'n welkom-skyf of leer-skyf te skep wat as 'n landingskyf vir nuwe gebruikers kan dien. Jy kan 'n paginabouer soos Beaver Builder, Elementor, Brizy en Oxygen gebruik om die skyfie sonder moeite te ontwerp.

Dit bied ook ondersteuning vir aangepaste HTML, CSS en JavaScript, sowel as toegangsbeheer gebaseer op plan en gebruikersrol!

## Funksies
* Ondersteun aangepaste HTML en CSS
* Ondersteun JavaScript
* PHP Inhoud Modus
* Toestemming gebaseer op gebruikersrol
* Ondersteun paginabouers

## Dokumentasie
[Skep Aangepaste Admin-Sidene en Sub-sidene met Multisite Ultimate: Admin Page Creator](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## Installasie

1. Laai **ultimate-multisite-admin-page-creator** op na die **/wp-content/plugins/** gids
2. Aktiveer die plugin via die **Plugins** menu in jou WordPress Network Admin

## Gebruiksgids

Terwyl WordPress dit regtig maklik maak om inhoud by die voorkant van webwerwe te voeg, vereis die skep van skyfie en sub-skyfie aan die admin-kant 'n paar basiese kodeskillinge. Ultimate Multisite: Admin Page Creator streef daarna om daardie gemak van die voorkant na die agterkant van jou netwerkwebwerwe te bring.

## Hoekom gebruik Admin Page Creator?

Miskien wil jy 'n aangepaste skyfie op jou kliënte se admin-paneel aanbied met tutorials, miskien wil jy 'n aangepaste skyfie met FAQ's en 'n kontak-widget byvoeg. Die moontlikhede is eindeloos met Admin Page Creator, en dit is wat ons vandag se tutorial sal verken.

## Menu Opsies

Op die kant, in die Algemene Opsies meta-box, sal jy die menu-type, etiket en baie meer kan konfigureer.

### Menu-types

Jou nuwe aangepaste admin-skyfie kan twee verskillende menu-types hê: top-level menu en submenu.

'n Top-level menu-skyfie, soos die naam voorstel, sal jou onlangs geskepte skyfie by die admin-menu voeg langs die standaard WordPress-skyfie.

As jy die top-level opsie kies, sal jy die menu-volgorde en 'n menu-ikoon ook kan kies.

As jy hierdie skyfie 'n sub-menu-skyfie wil maak, sal jy 'n ouerskyfie kan kies om hierdie skyfie aan te heg. Jy kan jou aangepaste skyfie by die standaard WordPress-skyfie of jou top-level aangepaste admin-skyfie voeg.

## Gevorderde Opsies

Jy moet in staat wees om alle tipes inhoud by jou aangepaste admin-skyfie te voeg. Dit sluit CSS-regels en lêers, JavaScript en eksterne biblioteke in. Jy kan dit doen met die Gevorderde Opsies Tab.

### Aangepaste CSS

Op die CSS-tab kan jy jou eie aangepaste CSS-kode byvoeg, sowel as CSS-lêers van afgeleë sites (soos CDNs, Google Fonts, ens.).

![Aangepaste CSS word ook ondersteun. Jy kan eksterne styl-lêers insluit!](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212627fb34b55df443e4a5-NwRQA4Lgk-AFD43C6C-1200-48C9-BDD9-64420A03EEB7.png)

### Aangepaste JavaScript

Dit geld dieselfde vir JavaScript. Jy kan eksterne biblioteke byvoeg om op jou aangepaste JavaScript-kode te gebruik.

![Voeg jou eie JavaScript-kode by!](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212627fb34b55df443e4a5-Uspm22WJI-E1D47212-D5C5-4B19-8E41-2BA85EADC4FE.png)

## Toestemmings

Ook in die Gevorderde Opsies kan jy kies watter sub-webwerwe toegang tot hierdie skyfie sal hê. Dit is nuttig as jy van plan is om verskillende eksklusiewe inhoud aan verskillende plan-tiers te bedien, byvoorbeeld.

![Jy het volledige beheer oor waar hierdie skyfie sal verskyn](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212627fb34b55df443e4a5-AJ3iDzsqH-E1389350-44A7-4D61-BE8A-6D5330B02333.png)

**Belangrik** : Albei voorwaardes moet voldoen vir die skyfie om vertoon te word. Dit beteken dat as die gebruiker 'n kliënt van 'n gegewe plan is, maar nie een van die gekose rolle het nie (of omgekeerd), sal die skyfie nie op hul paneel verskyn nie.

## Aktief

As jy 'n aangepaste admin-skyfie wil deaktiveer sonder om dit te verwyder of die toestemmings te verwar, ontkies net die Aktief opsie.

Deaktiveer 'n top-level skyfie sal outomaties al die sub-skyfie wat aan daardie ouerskyfie geheg is, deaktiveer.
