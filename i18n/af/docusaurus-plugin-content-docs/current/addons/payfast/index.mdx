# PayFast

Multisite Ultimate: Payfast Gateway

Voeg Payfast - 'n vinnige en veilige aanlyn betalingspoort in Suid-Afrika - as 'n Multisite Ultimate betalingsopsie.

## Installation

1. Laai 'ultimate-multisite-payfast' op die '/wp-content/plugins/' gids.
2. Aktiveer die plugin via die 'Plugins' menu in WordPress.
3. Volg die stap-vir-stap Wizard om die plugin op te stel.

## Usage Guide

PayFast is 'n vooraanstaande aanlyn betalingsverwerker vir besighede in Suid-Afrika. Dit is toegewy aan die hulp van aanlyn handelaars om te groei deur transaksies naatloos en veilig te maak. Volgens die versoek van 'n Ultimate Multisite gebruiker het ons besluit om 'n add-on te skep wat integrasie met PayFast moontlik maak. Hierdie artikel wys hoe om PayFast as jou betalingspoort op te stel om betaling in jou Ultimate Multisite netwerk te ontvang.

Die eerste stap is om die add-on te installeer.

# How to install the Payfast add-on:

Gaan na die add-ons bladsy en kies die Ultimate Multisite: Payfast Gateway add-on.

![Add-ons page](/img/admin/addons-page.png)

Die add-on venster sal opduik. Klik op Install Now.

![Add-ons page with install option](/img/admin/addons-page.png)

Na die installasie is voltooid, sal jy na die plugins bladsy herlei word. Hier klik net op Network Activate en die _Payfast_ add-on sal op jou netwerk geaktiveer word.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# Setting Up PayFast

Na die installasie van die add-on, kan jy nou na Ultimate Multisite instellings gaan en begin om PayFast te konfigureer.

Onder **Ultimate Multisite settings** bladsy, kan jy die "Payments" tab vind. Onder hierdie tab, moet jy '**Payfast**' saam met 'n toggle knoppie hÃª om hierdie betalingspoort te aktiveer of te deaktiveer.

![Payment gateway settings](/img/config/settings-payment-gateways.png)Go ahead and click the toggle button to enable it. It will then expand the box to show different fields that you need to populate such as **Mechant ID, Merchant Key and PassPhrase.** You can get this information from your **Payfast** account

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Die volgende stap is om in te log by jou PayFast rekening om die nodige inligting te kry.

Na inlog, kan jy dadelik die kredensies sien wat jy nodig het vanaf jou **Payfast** rekening dashboard.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->You can simply copy and paste the credential and populate the details on your Ultimate Multisite settings. Then click the **"Save settings** " button to save the changes.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Dit is dit, jou PayFast betalingsintegrasie is voltooi!
