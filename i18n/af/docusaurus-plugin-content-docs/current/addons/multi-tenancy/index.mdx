# Multi-Tenancy

Verdeel subsites oor bedieners en databasisse vir prestasie en sekuriteit

## Requirements

- WordPress 5.3 or higher
- PHP 7.4 or higher
- Multisite Ultimate plugin (active)

## Installation

1. Laai die add-on lêers na jou `/wp-content/plugins/` gids
2. Aktiveer die plugin via die 'Plugins' menu in WordPress
3. Konfigureer die add-on instellings in die netwerkadmin

## Development

### Setup

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Testing

Die add-on bevat 'n omvattende toetsraamwerk:

- **Unit Tests**: PHPUnit-gebaseerde toetse met WordPress toetsuite
- **Base Test Class**: Verskaf hulpmiddels vir algemene toets scenario's
- **Test Coverage**: Geconfigureer om kode dekking te volg
- **Multisite Testing**: Toetse hardloop in multisite omgewing

### Code Standards

- **WordPress Coding Standards**: Geëndeer via PHPCS
- **PHP 7.4+ Compatibility**: Moderne PHP funksies ondersteun
- **Static Analysis**: PHPStan integrasie vir tipe kontrole
- **Code Modernization**: Rector reëls vir PHP opgradering

### Build System

- **Asset Processing**: Minifikasie vir CSS/JS lêers
- **Translation**: POT lêer generasie
- **Packaging**: Argief skepping vir verspreiding
- **Development Mode**: Onminifiseerde hulpbronne vir debugging

## Configuration

Die add-on ondersteun verskeie konfigurasie opsies wat toeganklik is via die netwerkadmin koppelvlak.

## Hooks and Filters

### Actions

- `ultimate-multisite-addon-template_init` - Gevuur na add-on initialisering
- `ultimate-multisite-addon-template_loaded` - Gevuur na alle add-on lêers gelaai

### Filters

- `ultimate-multisite-addon-template_settings` - Verander add-on instellings
- `ultimate-multisite-addon-template_enabled` - Oorskryf add-on aktiveer/deaktiveer status
