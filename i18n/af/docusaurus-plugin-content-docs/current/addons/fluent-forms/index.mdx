# Fluent Forms

Hierdie add-on maak dit moontlik om beperkings by te voeg aan Fluent Forms op Multisite Ultimate in produkte, lidmaatskap of werf. Fluent Forms gebruik aangepaste tabelle in plaas van aangepaste possoorte, so die normale beperkingsopsies werk nie met Fluent Forms.

## Features

- Voeg beperkings by vir Fluent Forms of Gespreklike Fluent Forms
- Klante kry 'n vriendelike boodskap wat hulle aanmoedig om hul plan op te gradeer wanneer hulle die limiet bereik
