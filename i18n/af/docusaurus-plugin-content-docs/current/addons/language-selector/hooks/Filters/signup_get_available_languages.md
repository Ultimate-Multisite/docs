# Filter: signup_get_available_languages

Filter die lys van beskikbare tale vir front-end-webwerfregistrasies.

As jy 'n leë array aan hierdie hook oordra, sal die uitsetting van die instelling op die registrasieformulier gedeaktiveer word, en die standaardtaal sal gebruik word wanneer die webwerf geskep word. Tale wat nog nie geïnstalleer is nie, sal verwyder word.

## Parameters {#parameters}

| Naam | Tipe | Beskrywing |
|------|------|-------------|
| $available_languages | `array` | Beskikbare tale. |

- 4.4.0
