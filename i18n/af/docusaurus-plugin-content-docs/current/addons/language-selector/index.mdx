# Multisite Ultimate: Language Selector op aanmelding

# Language Selector

# Installasie

1. Laai 'ultimate-multisite-language-selector' op na die '/wp-content/plugins/' gids
2. Aktiveer die plugin via die 'Plugins' menu in jou WordPress Network Admin

# Gebruiksrig

Met hierdie gratis add-on kan jy jou gebruikers laat kies watter taal hulle graag wil sien op die aanmeldvloei en op die aanmeldbladsy.

Na [aktivering van jou lisensie] sal jy in staat wees om Ultimate Multisite add-ons te installeer waarop jou lisensie jou toegang gee.

Gaan na jou Ultimate Multisite-instellingsbladsy en klik dan op die menu aan die links, op die _Add-ons_ etiket.

![Add-ons page](/img/admin/addons-page.png)

_Gaan na Add-ons-bladsy om Language Selector te begin installeer_

Daar sal jy al die add-ons vind wat ons aanbied.

Gratis add-ons is beskikbaar vir al ons kliënte, ongeag die inskrywingplan waarop hulle aangemeld is.

Klik die _Install it Now!_ knoppie onder die add-on se thumbnail. Dan aktiveer dit op die Plugins-bladsy.

![Add-ons page with install button](/img/admin/addons-page.png)

_Klik die knoppie om die add-on te installeer_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_Aktiveer dit op jou Plugins-bladsy_

Nou, gaan na Ultimate Multisite se Instellingsmenu en voer die _Network Settings_ tab in. Daar, onderaan die bladsy, sal jy 'n nuwe afdeling vind genaamd _Languages_.

![Settings page](/img/config/settings-general.png)

_'n Merk nuwe afdeling verskyn!_

Jy kan dit leeg laat om al die tale as opsies te vertoon of kies watter tale jy wil beskikbaar maak. Die tale is die een wat jy in jou WordPress-opstelling geïnstalleer het.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_Lys van tale beskikbaar op hierdie WP-installasie_

Na die stoor van die veranderings wat jy gemaak het, sal 'n keuse-menu met die tale wat jy gekies het, op die Login-bladsy en op die Sign-up Flow vertoon word.

![Login and Registration settings](/img/config/settings-login-registration.png)

_Language Selector op Login-bladsy_

![Registration page](/img/config/registration-page.png)

_Language Selector op die Sign-up Flow_

# Stel die taal via die URL

Jy kan ook die taal van 'n bladsy verander deur die locale-parameter aan die URL te omseil. As jy byvoorbeeld na 'n registreerbladsy in Italiaans wil omskep, voeg ?locale=it_IT by die URL.

The language must be installed on your WordPress network for this to work.
