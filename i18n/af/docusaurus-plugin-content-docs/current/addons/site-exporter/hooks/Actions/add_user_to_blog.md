# Aksie: add_user_to_blog

Word geaktiveer onmiddellik nadat 'n gebruiker by 'n webwerf (site) bygevoeg word.

## Parameters {#parameters}

| Naam | Tipe | Beskrywing |
|------|------|-------------|
| $user_id | `int` | Die ID van die gebruiker. |
| $role | `string` | Die rol van die gebruiker. |
| $blog_id | `int` | Die ID van die blog. |
