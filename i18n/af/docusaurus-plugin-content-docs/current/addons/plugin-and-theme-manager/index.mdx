# Plugin & Theme Manager

Multisite Ultimate: Plugin en Tema Beheer

Wysig of verberg die meta-inligting (titel, skrywer, miniatyur en beskrywing) van geïnstalleerde plugins en temas. Skep kategorieë vir plugins en temas om jou gebruikers toe te laat om hulle te filter en voeg 'n pragtige aangepaste Plugins-bladsy by jou gebruikers se paneel.

Om die plugin te gebruik, kan jy net na jou Plugins of Themes-bladsye op jou netwerkadmin gaan en klik op `Edit Plugin/Theme info`.

## Installation

1. Laai 'ultimate-multisite-plugin-and-theme-manager' op na die `/wp-content/plugins/` gids
2. Aktiveer die plugin via die `Plugins` kieslys in jou WordPress Network Admin

## Usage Guide

Ultimate Multisite: Plugin en Tema Beheer is 'n kragtige **premium** Ultimate Multisite add-on wat jou volledige beheer gee oor hoe jou netwerkgebruikers die plugin en tema-opsies op jou netwerk sien en interaksie.

Hierdie add-on dien drie doeleindes uit:

  1. Dit laat jou toe om die metadata van beide plugins en temas volledig te verander. Dit sluit naam, beskrywing, skrywernaam en selfs miniatyure in!

  2. Dit laat jou toe om plugins en temas te kategoriseer, sodat jou gebruikers dit maklik kan sorteer.

  3. Dit gee jou die opsie om die ou, eentonige plugins-bladsy te vervang met 'n nuwe, tema-agtige bladsy met sortering volgens kategorieë.

Na aflaai en installasie, sal 'n nuwe oortjie - _Add-on Settings_ \- verskyn op die Ultimate Multisite Settings-bladsy.
