# Admin Page Creator {#admin-page-creator}

## Ọdịnaya {#content}
Admin Page Creator ga-enye gị ohere ịgbakwunye peeji admin ọhụrụ na peeji nta n’ime WordPress admin dashboard n’ụzọ dị mfe. Enwere ike iji ya mepụta peeji nnabata ma ọ bụ peeji nkuzi nke nwere ike ịbụ peeji mbata maka ndị ọrụ ọhụrụ. Ị nwere ike iji page builder dị ka Beaver Builder, Elementor, Brizy, na Oxygen hazie peeji ndị ahụ n’enweghị nsogbu.

Ọ na-enyekwa nkwado HTML, CSS na JavaScript ahaziri iche, yana njikwa ohere dabere na atụmatụ na ọrụ onye ọrụ!

## Njirimara {#features}
* Nkwado HTML na CSS ahaziri iche
* Nkwado JavaScript
* Ọnọdụ ọdịnaya PHP
* Ikike dabere na ọrụ onye ọrụ
* Nkwado maka page builders

## Akwụkwọ nduzi {#documentation}
[Ịmepụta Peeji Admin ahaziri iche na Peeji nta site na iji Ultimate Multisite: Admin Page Creator](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## Nwụnye {#installation}

1. Bulite **ultimate-multisite-admin-page-creator** gaa na ndekọ **/wp-content/plugins/**
2. Mee ka plugin ahụ rụọ ọrụ site na menu **Plugins** n’ime WordPress Network Admin gị

## Nduzi Ojiji {#usage-guide}

Ọ bụ ezie na WordPress na-eme ka ịgbakwunye ọdịnaya na akụkụ ihu nke saịtị dị mfe nke ukwuu, ịmepụta peeji na peeji nta n’akụkụ admin chọrọ ụfọdụ nka ide koodu bụ isi. Ultimate Multisite: Admin Page Creator bu n’obi iweta otu mfe ojiji ahụ nke akụkụ ihu gaa n’azụ nke saịtị netwọkụ gị.

## Gịnị mere eji eji Admin Page Creator? {#why-use-admin-page-creator}

Ikekwe ịchọrọ inye peeji ahaziri iche na panel admin nke ndị ahịa gị nwere nkuzi, ma ọ bụ ịchọrọ ịgbakwunye peeji ahaziri iche nwere ajụjụ a na-ajụkarị na widget kọntaktị. Ohere ndị dị na Admin Page Creator enweghị ngwụcha, nke ahụ bụkwa ihe anyị ga-enyocha na nkuzi taa.

## Ịmepụta Peeji Admin ahaziri iche mbụ gị {#creating-your-first-custom-admin-page}

Mgbe ị wụnyere ma mee ka Ultimate Multisite: Admin Page Creator rụọ ọrụ, gaa na menu admin netwọkụ gị ma gaa na Peeji Admin.

![Ndepụta Peeji Admin ahaziri iche na Network Admin](/img/admin/admin-pages-list.png)

_Ndepụta Peeji Admin ahaziri iche na Network Admin_

## Nhọrọ Menu {#menu-options}

N’akụkụ, na meta-box Nhọrọ Izugbe, ị ga-enwe ike ịhazi ụdị menu, akara, na ọtụtụ ihe ndị ọzọ.

### Ụdị Menu {#menu-types}

Peeji admin ahaziri iche ọhụrụ gị nwere ike inwe ụdị menu abụọ dị iche: menu ọkwa elu na submenu.

Peeji menu ọkwa elu, dịka aha ya na-egosi, ga-agbakwunye peeji ọhụrụ ị mepụtara na menu admin n’akụkụ peeji ndabara nke WordPress.

Ọ bụrụ na ịhọrọ nhọrọ ọkwa elu, ị ga-enwe ike ịhọrọ usoro menu na akara ngosi menu kwa.

Ọ bụrụ na ịchọrọ ime peeji a ka ọ bụrụ peeji sub-menu, ị ga-enwe ike ịhọrọ peeji nne na nna iji jikọta peeji a na ya. Ị nwere ike ịgbakwunye peeji ahaziri iche gị na peeji ndabara nke WordPress ma ọ bụ na peeji admin ahaziri iche ọkwa elu nke gị.

## Nhọrọ Dị Elu {#advanced-options}

Ị kwesịrị inwe ike ịgbakwunye ụdị ọdịnaya niile na peeji admin ahaziri iche gị. Nke ahụ gụnyere iwu na faịlụ CSS, JavaScript, yana ọba akwụkwọ mpụga kwa. Ị nwere ike ime nke ahụ site na iji Advanced Options Tab.

### CSS ahaziri iche {#custom-css}

Na tab CSS, ị nwere ike ịgbakwunye koodu CSS ahaziri iche nke gị, yana ibubata faịlụ CSS site na saịtị dị anya (dị ka CDNs, Google Fonts, wdg).

<!-- Nseta ihuenyo adịghị: Onye ndezi CSS ahaziri iche nke Admin Page Creator nwere mbubata stylesheet mpụga -->

_A na-akwadokwa CSS ahaziri iche. Ị nwekwara ike itinye faịlụ style mpụga!_

### JavaScript ahaziri iche {#custom-javascript}

Otu ihe ahụ dịkwa irè maka JavaScript. Ị nwere ike ịgbakwunye ọba akwụkwọ mpụga iji na koodu JavaScript ahaziri iche gị.

<!-- Nseta ihuenyo adịghị: Onye ndezi JavaScript ahaziri iche nke Admin Page Creator nwere mbubata ọba akwụkwọ mpụga -->

_Tinye koodu JavaScript nke gị!_

## Ikikere {#permissions}

Ọzọkwa n’ime Advanced Options, ị ga-enwe ike ịhọrọ saịtị nta ndị ga-enweta ohere na peeji a. Nke a bara uru ma ọ bụrụ na ị na-eme atụmatụ inye ọdịnaya pụrụ iche dị iche iche nye ọkwa atụmatụ dị iche iche, dịka ọmụmaatụ.

<!-- Nseta ihuenyo adịghị: Ntọala ikikere Admin Page Creator maka njikwa ohere dabere na atụmatụ na ọrụ -->

_Ị nwere njikwa zuru oke n’ebe peeji a ga-apụta_

**Dị mkpa** : A ga-emezurịrị ọnọdụ abụọ ahụ ka e wee gosipụta peeji ahụ. Nke a pụtara na ọ bụrụ na onye ọrụ bụ onye ahịa nke plan e nyere, mana enweghị otu n’ime roles ahọpụtara (ma ọ bụ nke ọzọ), peeji ahụ agaghị apụta na panel ha.

## Na-arụ ọrụ {#active}

Ọ bụrụ na ịchọrọ ime ka peeji nchịkwa ahaziri ghara ịrụ ọrụ na-enweghị ihichapụ ya ma ọ bụ imegharị ikike, naanị wepụ akara na nhọrọ Na-arụ ọrụ.

Ime ka peeji ọkwa elu ghara ịrụ ọrụ ga-eme ka peeji nta niile ejikọtara na peeji nne na nna ahụ ghara ịrụ ọrụ na-akpaghị aka.
