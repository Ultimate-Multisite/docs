# Ndekọ mgbanwe Multi-Network {#multi-network-changelog}

### 1.3.0 {#130}
* Ọhụrụ: Onye nlele ndebiri netwọkụ — chọgharịa ma lelee ndebiri netwọkụ site na panel dị ndụ tupu ịzụta.
* Ọhụrụ: Onye nhọpụta ndebiri netwọkụ na checkout — ndị ahịa nwere ike ịhọrọ ndebiri netwọkụ n'oge usoro signup checkout.
* Ọhụrụ: Cross-network magic link SSO — ndị ahịa nwere ike ịnweta sub-networks n'enweghị nsogbu site na nkwenye magic-link.
* Ọhụrụ: Mmegharị media netwọkụ — a na-emegharị media saịtị nke ọma mgbe a na-eme clone netwọkụ site na ndebiri.
* Mmezi: Clone netwọkụ na-arụ ọrụ ugbu a na-enweghị multi-tenancy addon; a na-enyeghachi ikike super admin mgbe clone gasịrị.
* Mmezi: Mepụta netwọkụ dakọtara ugbu a na WooCommerce na payment gateways niile.
* Mmezi: A na-etinye nọmba port ugbu a na mmepụta fallback domain.
* Mmezi: Ebugharịla CSS nke onye nhọpụta ndebiri gaa na stylesheet e tinyere n'ahịrị maka ngosi ziri ezi na themes niile.
* Mmezi: Chebe megide igodo `network_id` a na-akọwaghị na scope ajụjụ saịtị.
* Mmezi: Nlele ikike netwọkụ tupu ịgbanwe context na-egbochi njehie property tebụl `wpdb`.
