# Nzacha: signup_get_available_languages

Na-enyocha ndepụta asụsụ dị maka ndebanye aha saịtị n’akụkụ ihu.

Ịnyefe array efu na hook a ga-agbanyụọ mmepụta nke ntọala ahụ na fọm ndebanye aha, a ga-ejikwa asụsụ ndabara mgbe a na-emepụta saịtị ahụ. A ga-ewepụ asụsụ ndị etinyebeghị.

## Paramita {#parameters}

| Aha | Ụdị | Nkọwa |
|------|------|-------------|
| $available_languages | `array` | Asụsụ dị. |

### Kemgbe {#since}

- 4.4.0
