# Filter: wu_connector_enforcement_option_keys {#filter-wuconnectorenforcementoptionkeys}

Nyochaa ndepụta nke AI provider option keys a na-amanye site na saịtị isi.

Tinye option keys maka plugin AI provider ahaziri iche ma ọ bụ nke ndị ọzọ ka ntọala ha sikwa na saịtị isi bụrụ nke a na-eketa na subsites.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $keys | `string[]` | Aha option key a ga-amanye (gụnyelarị connector keys achọpụtara n’ụzọ dinamiki na EXTRA_PROVIDER_OPTIONS). |
