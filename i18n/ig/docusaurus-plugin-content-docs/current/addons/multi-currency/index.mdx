# Multi-Currency

Ultimate Multisite: Multi-Currency na-eme ka netwọkụ gị nwee ike ịnakwere ịkwụ ụgwọ n'ọtụtụ ụdị ego. Ọ na-agbanwe ọnụahịa na-akpaghị aka site n'iji ọnụego mgbanwe dị ndụ ma ọ bụ nke aka, na-enye gị ohere ịtọ ọnụahịa kpọmkwem kwa-ngwaahịa maka ụdị ego ọ bụla, ma chọpụta ụdị ego onye ọbịa ọ bụla kacha amasị site na nchọpụta ọnọdụ ala — ihe niile a na-ejikwa site n'otu Dashboard onye nchịkwa.

## Njirimara Ndị Dị Mkpa {#key-features}

- **Mgbanwe ọnụego mgbanwe ego na-akpaghị aka** — a na-agbanwe ọnụahịa site na ụdị ego ntọala gị site n'iji ọnụego ewetara n'aka onye na-enye ọrụ ị họọrọ
- **Ndochi ọnụahịa kwa-ngwaahịa** — tọọ ọnụahịa kpọmkwem maka ụdị ego pụrụ iche na ngwaahịa ọ bụla iji gafee mgbanwe akpaka
- **Ogige nhọrọ ụdị ego maka ndebanye aha** — na-enye ndị ahịa ohere ịhọrọ ụdị ego ha kacha amasị n'oge ịkwụ ụgwọ/ndebanye aha
- **Ọtụtụ ndị na-enye ọnụego mgbanwe ego** — ECB, Frankfurter, Stripe, ma ọ bụ ọnụego aka kpamkpam
- **Nchọpụta ụdị ego dabere na ọnọdụ ala** — na-ahọrọ ụdị ego dabara na obodo onye ọbịa na-akpaghị aka
- **Dashboard onye nchịkwa** — jikwaa ụdị ego ndị na-arụ ọrụ, ọnụego mgbanwe ego, na ntọala ndị na-enye ọrụ site n'otu ebe

## Nrụnye {#installation}

1. Bulite faịlụ ngwa mgbakwunye ahụ na `/wp-content/plugins/ultimate-multisite-multi-currency`
2. Mee ka ngwa mgbakwunye ahụ rụọ ọrụ site na ihuenyo **Ngwa mgbakwunye** na WordPress (Mee ka ọ rụọ ọrụ na Netwọkụ)
3. Gaa na **Ultimate Multisite → Multi-Currency** iji mepee Dashboard onye nchịkwa
4. Tinye ụdị ego ịchọrọ ịnakwere ma hazie onye na-enye ọnụego mgbanwe ego gị

## Nhazi {#configuration}

### Dashboard Onye Nchịkwa {#admin-dashboard}

Dashboard onye nchịkwa Multi-Currency (**Ultimate Multisite → Multi-Currency**) bụ ebe etiti maka:

- Ịgbakwunye, ime ka ọ rụọ ọrụ, ma ọ bụ ime ka ụdị ego kwụsị ọrụ
- Ịhọrọ onye na-enye ọnụego mgbanwe ego ma mee ka ọnụego dị ọhụrụ mgbe achọrọ
- Ịlele ọnụego ugbu a maka ụdị ego niile na-arụ ọrụ
- Ịtọ ọnụego aka mgbe ndị na-enye ọrụ akpaka adịghị mma

### Ndị Na-enye Ọnụego Mgbanwe Ego {#exchange-rate-providers}

| Onye na-enye ọrụ | Nkọwa |
|---|---|
| **ECB** | European Central Bank — ọnụego kwa ụbọchị n'efu maka netwọkụ dabere na EUR |
| **Frankfurter** | API mirror ECB nke isi mmalite mepere emepe — n'efu, achọghị API key |
| **Stripe** | Na-eji ọnụego FX dị ndụ nke Stripe — chọrọ njikọta Stripe na-arụ ọrụ |
| **Manual** | Ị na-etinye ọnụego ozugbo; enweghị oku API mpụga |

Họrọ onye na-enye ọrụ gị n'okpuru **Ultimate Multisite → Multi-Currency → Settings → Exchange Rate Provider**. Enwere ike ime ka ọnụego dị ọhụrụ n'aka n'oge ọ bụla ma ọ bụ hazie ya ka ọ melite na-akpaghị aka.

### Ndochi Ọnụahịa Kwa-Ngwaahịa {#per-product-pricing-overrides}

Mgbe mgbanwe akpaka ezughị oke nke ọma, ị nwere ike ịtọ ọnụahịa kpọmkwem maka ụdị ego pụrụ iche na ngwaahịa ọ bụla:

1. Mepee onye ndezi ngwaahịa (**Ultimate Multisite → Products → Edit**)
2. Pịgharịa ruo ngalaba **Currency Pricing**
3. Tinye ọnụahịa kpọmkwem maka ụdị ego ọ bụla ịchọrọ dochie
4. Chekwaa ngwaahịa ahụ

Ngwaahịa na-enweghị ndochi na-eji ọnụahịa agbanwere site na ụdị ego ntọala gị.

### Ogige Nhọrọ Ụdị Ego Maka Ndebanye Aha {#currency-selector-signup-field}

Enwere ike itinye ogige nhọrọ ụdị ego na usoro ndebanye aha/ịkwụ ụgwọ gị ka ndị ahịa họrọ ụdị ego ha kacha amasị n'oge ndebanye aha:

1. Gaa na **Ultimate Multisite → Multi-Currency → Settings**
2. Mee ka **Show currency selector on signup** rụọ ọrụ
3. Ọ bụrụ na ịchọrọ, tọọ ụdị ego ndabara ma ọ bụ hapụ nchọpụta ọnọdụ ala ka ọ chọpụta uru ahọrọ tupu oge eruo

A na-echekwa ụdị ego ahọrọ na ndenye aha ma jiri ya maka ịgba ụgwọ niile na-esote.

### Nchọpụta Ụdị Ego Dabere na Ọnọdụ Ala {#geolocation-based-currency-detection}

Mgbe enyere nchọpụta ọnọdụ ala aka, addon ahụ na-ejikọta obodo onye ọbịa na ụdị ego ma họrọ ya tupu oge eruo n'ogige nhọrọ ụdị ego. Iji mee ka ọ rụọ ọrụ:

1. Gaa na **Ultimate Multisite → Multi-Currency → Settings**
2. Mee ka **Detect currency from visitor location** rụọ ọrụ
3. Jikọọ obodo na ụdị ego na tebụl **Geolocation Mapping**

Ọ bụrụ na enweghị njikọ maka obodo onye ọbịa, a na-eji ụdị ego ntọala netwọkụ ahụ dị ka ndabere.

## Ajụjụ A Na-ajụkarị {#frequently-asked-questions}

### Gịnị na-eme ma ọ bụrụ na onye na-enye ọnụego mgbanwe ego adịghị? {#what-happens-if-an-exchange-rate-provider-is-unavailable}

### Nchọpụta ọnọdụ ala ọ chọrọ ngwa mgbakwunye dị iche? {#does-geolocation-require-a-separate-plugin}

Achọghị ngwa mgbakwunye nchọpụta ọnọdụ ala mpụga. Addon ahụ na-eji nchọpụta IP-ruo-obodo nke WordPress nwere n'ime ya. Maka izi ezi ka elu, enwere ike ijikọ ọrụ nchọpụta ọnọdụ ala pụrụ iche site na filter hooks enyere.

### A na-etinye ndochi kwa-ngwaahịa tupu ma ọ bụ mgbe ụtụ isi gasịrị? {#are-per-product-overrides-applied-before-or-after-tax}
