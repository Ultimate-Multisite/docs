# Tutor Multisite Compatibility

Ngwa mgbakwunye WordPress nke na-agbakwunye ndozi ndakọrịta maka Tutor LMS n'ime gburugburu multisite.

## Nkọwa

Ngwa mgbakwunye a na-enye ndozi ndakọrịta dị mkpa maka ịrụ Tutor LMS na netwọkụ multisite WordPress. Ọ na-elekọta idebanye ndị ọrụ, njikwa ndị nkuzi, mmachi nkuzi, na arụmọrụ ndị ọzọ pụrụ iche maka multisite.

## Njirimara

- **Njikwa Tebụl Ebe Nchekwa Data**: Na-achọpụta ma mepụta tebụl Tutor LMS ndị na-efu efu n'akpaghị aka, ndị dara n'ihi mmachi foreign key nke multisite
- **Ndebanye Onye Ọrụ Multisite**: Ndị njikwa AJAX ahaziri ahazi maka ijikwa ndebanye ndị ọrụ n'ofe saịtị netwọkụ
- **Njikwa Ọrụ Onye Nkuzi**: Ịkenye na ijikwa ọrụ onye nkuzi n'akpaghị aka n'ime gburugburu multisite
- **Mmachi Nkuzi**: Njikọta na Ultimate Multisite maka ijikwa oke imepụta nkuzi na funnel dabere na atụmatụ saịtị
- **Nlele Admin Ahaziri Ahazi**: Ndepụta ndị nkuzi na interface njikwa nke maara multisite
- **Ngwaọrụ Admin**: Interface njikwa ebe nchekwa data maka ndị nchịkwa netwọkụ na saịtị

## Ihe A Chọrọ

- WordPress 5.0+
- PHP 7.4+
- Ngwa mgbakwunye Tutor LMS (a na-amanye ya n'akpaghị aka site na header `Requires Plugins`)
- Ntinye WordPress Multisite
- Ngwa mgbakwunye Ultimate Multisite (maka arụmọrụ mmachi nkuzi)

## Ntinye

1. Bulite faịlụ ngwa mgbakwunye ahụ na `/wp-content/plugins/tutor-multisite-compatibility/`
2. Gbaa mbọ hụ na etinyere ma mee ka Tutor LMS rụọ ọrụ
3. Mee ka ngwa mgbakwunye ahụ rụọ ọrụ site na menu 'Plugins' na WordPress
4. Ngwa mgbakwunye ahụ ga-ejikọta na Tutor LMS n'akpaghị aka

## Arụmọrụ Ebugara Site na MU Plugin

Ngwa mgbakwunye a nwere arụmọrụ nke dịbu na `wp-content/mu-plugins/daves.php`:

- Mmachi mgbanwe ọnọdụ nkuzi
- Mmachi imepụta nkuzi
- Njikọta na oke CartFlows
- Njikwa ọrụ onye nkuzi Tutor
- Ijikwa ndebanye onye ọrụ multisite
- Ngosipụta ID ngwaahịa nkuzi

## Njikwa Ebe Nchekwa Data

### Nsogbu Ahụ

N'ime gburugburu multisite WordPress, imepụta tebụl Tutor LMS na-ada maka tebụl nwere mmachi foreign key na tebụl ndị ọrụ. Nke a na-eme n'ihi na:

- Tebụl pụrụ iche maka saịtị na-eji prefixes dịka `wp_2_tutor_carts`
- Foreign keys na-ezo aka na `wp_2_users` (nke na-adịghị adị)
- Tebụl ndị ọrụ n'ezie bụ `wp_users` (a na-ekekọrịta ya n'ofe netwọkụ)

### Ngwọta Ahụ

Ngwa mgbakwunye a na-eme ihe ndị a n'akpaghị aka:

1. **Na-achọpụta Tebụl Ndị Na-efu Efuu**: Na-enyocha tebụl ndị na-emeghị ka e mepụta ha n'ihi mmachi foreign key
2. **Na-emepụta Tebụl nwere Ntụaka Ziri Ezi**: Na-eji aha tebụl onye ọrụ kwesịrị ekwesị (`wp_users` na multisite)
3. **Na-elekọta Ọtụtụ Ọnọdụ**:
   - Ime ka ngwa mgbakwunye rụọ ọrụ/rụọ ọrụ ọzọ
   - Imepụta saịtị ọhụrụ na netwọkụ
   - Nlele tebụl aka site na interface admin

### Tebụl Ndị Metụtara

- `tutor_coupon_usages` - Nwere foreign key gaa na tebụl ndị ọrụ
- `tutor_carts` - Nwere foreign key gaa na tebụl ndị ọrụ
- `tutor_notification_preferences` - Tebụl TutorPro nwere foreign key gaa na tebụl ndị ọrụ

### Interface Admin

Nweta ngwaọrụ njikwa ebe nchekwa data na:

- **Network Admin**: Settings → Tutor Multisite DB
- **Site Admin**: Tools → Tutor Multisite DB

Interface ahụ na-enye gị ohere:
- Lelee tebụl ndị na-efu efu n'ofe saịtị niile
- Kpalite imepụta tebụl aka
- Lelee nhazi ebe nchekwa data
- Lelee ozi ọnọdụ zuru ezu
