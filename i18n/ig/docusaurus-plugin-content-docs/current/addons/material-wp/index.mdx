# Material WP {#material-wp}

Ọdịdị nchịkwa WordPress adịla otu ihe ruo ogologo oge. Ma ị chetụla otú ụdị Material Design nke ọdịdị nchịkwa ga-adị?

Material WP bụ theme nchịkwa e mepụtara iji nye gị ọdịdị na mmetụta ahụ, na-enyere gị aka inye ndị ahịa gị Dashboard ahaziri nke ọma karị.

## Njirimara {#features}

- Nkwado Custom CSS
- Ntọala ịkpụzi dị mfe
- Nhazigharị menu
- Mbubata/mbupụ dị mfe

## Akwụkwọ nduzi {#documentation}

**Iji Material WP hazie Panel Nchịkwa gị**

## Ntinye {#installation}

1. Bulite 'material-wp' na ndekọ '/wp-content/plugins/'
2. Mee ka plugin ahụ rụọ ọrụ site na menu 'Plugins' na WordPress
