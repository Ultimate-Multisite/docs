# Omume: wu_page_added

Kwe ka ndị mmepe mgbakwunye mee ihe ndị ọzọ mgbe a na-edebanye peeji aha.

N'adịghị ka wu_page_load, nke na-arụ naanị mgbe a na-ele peeji kpọmkwem, hook a na-arụ n'oge ndebanye aha maka peeji nchịkwa ọ bụla a na-agbakwunye site na iji koodu WP Ultimo.

## Paramita

| Aha | Ụdị | Nkọwa |
|------|------|-------------|
| $page_id | `string\|\WP_Ultimo_Site_Exporter\Admin_Pages\Base_Admin_Page` | ID nke peeji a. |
| $page_hook | `string\|\WP_Ultimo_Site_Exporter\Admin_Pages\Base_Admin_Page` | Hook emepụtara maka peeji a. |
