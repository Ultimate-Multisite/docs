# Ngwaọrụ na Nhazi {#tools-and-configuration}

Theme Builder na-eji otu ìgwè ngwaọrụ nyochaa, mepụta atụmatụ, ma wuo WordPress saịtị gị. Na ụdị 1.16.0, **sd-ai-agent/site-scrape bụzi ngwaọrụ Tier 1**, nke na-eme ka ọ dị na ndabara na Theme Builder. Superdav AI Agent v1.18.0 na-ewebatakwa ngwugwu Advanced ibe a na-ekesa iche maka usoro ọrụ ndị mmepe a tụkwasịrị obi; ngwaọrụ ndị ahụ abụghị akụkụ nke ngwugwu isi WordPress.org ma a ga-etinyerịrị ma nye ha ikike iche.

## Ngwaọrụ Dị {#available-tools}

### Ngwaọrụ Tier 1 (Na-adị Mgbe Niile) {#tier-1-tools-always-available}

Ngwaọrụ Tier 1 dị na ndabara na Theme Builder na-enweghị nhazi ọzọ ọ bụla.

#### sd-ai-agent/site-scrape {#sd-ai-agentsite-scrape}

**Ebumnuche:** Nyochaa weebụsaịtị ndị dị ugbu a iji wepụ mkpali imewe, nhazi ọdịnaya, na arụmọrụ.

**Ikike:**

- **Nyocha weebụsaịtị** — hichapụ ma nyochaa weebụsaịtị ndị asọmpi ma ọ bụ ndị na-enye mkpali
- **Mwepụta imewe** — chọpụta agba, mkpụrụedemede, na usoro nhazi
- **Nhazi ọdịnaya** — ghọta nhazi peeji na usoro ọkwa
- **Nchọpụta njirimara** — chọpụta plugins na arụmọrụ
- **Nyocha arụmọrụ** — lelee ọsọ peeji na mmezi
- **Nyocha SEO** — nyochaa meta tags na data ahaziri ahazi

**Ojiji:**

```
Nyochaa imewe nke example.com ma jiri ya dị ka mkpali maka saịtị m.
```

**Ihe Ọ Na-ewepụta:**

- Paleti agba na ojiji agba
- Typography (mkpụrụedemede na nha)
- Nhazi layout na oghere
- Usoro navigation
- Nhazi ọdịnaya
- Ojiji onyonyo na mgbasa ozi
- Ihe mmekọrịta
- Mmeghachi omume na mobile

**Mmachi:**

- Enweghị ike ihichapụ saịtị ndị e ji okwuntughe chebe
- Na-asọpụrụ robots.txt na atumatu saịtị
- O nwere ike ọ gaghị ejide ọdịnaya na-agbanwe agbanwe
- Saịtị nwere JavaScript dị ukwuu nwere ike inwe nyocha pere mpe
- Saịtị buru ibu nwere ike iwe ogologo oge inyocha

### Ngwaọrụ Tier 2 (Nhọrọ) {#tier-2-tools-optional}

Ngwaọrụ Tier 2 dị mgbe enyere ha ikike n'ụzọ doro anya na ntọala Theme Builder.

#### Advanced Analytics {#advanced-analytics}

Na-enye metrik arụmọrụ zuru ezu:

- Oge ibu peeji
- Core Web Vitals
- Akara SEO
- Akara nnweta
- Akara omume kacha mma

#### Content Optimizer {#content-optimizer}

Na-enyocha ma na-atụ aro mmezi maka:

- Mfe ọgụgụ
- Mmezi SEO
- Ojiji keyword
- Nhazi ọdịnaya
- Idobe oku-ime-ihe

### Ngwaọrụ Tier 3 (Premium) {#tier-3-tools-premium}

Ngwaọrụ Tier 3 chọrọ nhazi ọzọ ma ọ bụ njirimara premium.

#### AI Content Generator {#ai-content-generator}

Na-emepụta ọdịnaya maka:

- Nkọwa ngwaahịa
- Peeji ọrụ
- Edemede blog
- Nkọwa meta
- Ederede oku-ime-ihe

#### Advanced Design Tools {#advanced-design-tools}

Na-enye ikike imewe dị elu:

- Mmepụta Custom CSS
- Mepụta animation
- Imewe ihe mmekọrịta
- Ozizi agba dị elu
- Mmezi typography

### Ngwaọrụ Advanced Ibe (Ngwugwu Iche) {#advanced-companion-tools-separate-package}

A na-ekesa ngwugwu Advanced ibe iche na mbipụta isi Superdav AI Agent. Ezubere ya maka gburugburu ndị mmepe na ndị nwe saịtị a tụkwasịrị obi ebe ndị nchịkwa na-enye agent ikike n'ụzọ doro anya iji ngwaọrụ nwere mmetụta dị elu.

Ikike Advanced ibe nwere ike ịgụnye:

- **Ngwaọrụ filesystem** — nyochaa ma gbanwee faịlụ akwadoro na gburugburu a tụkwasịrị obi
- **Ngwaọrụ database** — jụọ ma ọ bụ melite data saịtị mgbe enyere ikike n'ụzọ doro anya
- **Ngwaọrụ WP-CLI** — gbaa iwu mmezi na nyocha WordPress
- **Ngwaọrụ REST dispatcher** — kpọọ REST endpoints edebanyere aha site na usoro ọrụ dispatch a na-achịkwa
- **Ngwaọrụ plugin builder** — wuo scaffold ma megharịa plugin code
- **Ngwaọrụ Git snapshot** — mepụta snapshots tupu ọrụ mmepe nwere ihe ize ndụ
- **Ngwaọrụ njikwa ndị ọrụ** — nyere aka nyochaa ma ọ bụ gbanwee ndekọ ndị ọrụ ebe enyere ikike
- **Ngwaọrụ benchmark** — gbaa benchmark arụmọrụ ma ọ bụ ikike maka nyocha mmepe

Edela ha dịka ngwaọrụ Theme Builder dị n’ozuzu maka ntinye ọ bụla. Ha dị naanị mgbe etinyere ngwugwu Advanced ibe, enyere ya ikike, ma kpachie ya nye ndị nchịkwa a tụkwasịrị obi.

## Nhazi Ngwaọrụ {#tool-configuration}

### Ime Ka Ngwaọrụ Rụọ Ọrụ {#enabling-tools}

Iji mee ka ngwaọrụ ndị ọzọ rụọ ọrụ na Theme Builder:

1. Gaa na **Gratis AI Agent → Settings**
2. Gaa na **Theme Builder → Tools**
3. Gbanyụọ ma ọ bụ gbanye ngwaọrụ dịka achọrọ
4. Chekwaa ntọala

### Ikike Ngwaọrụ {#tool-permissions}

Ụfọdụ ngwaọrụ chọrọ ikike:

- **Ihichapụ saịtị** — chọrọ ohere ịntanetị
- **Analytics** — chọrọ njikọ Google Analytics
- **Mmepụta ọdịnaya** — chọrọ API keys
- **Njirimara dị elu** — nwere ike ịchọ premium subscription
- **Ngwaọrụ Advanced ibe** — chọrọ ngwugwu Advanced a na-ekesa iche na ntụkwasị obi nchịkwa doro anya

### API Keys na Credentials {#api-keys-and-credentials}

Hazie API keys maka ngwaọrụ ndị chọrọ ha:

1. Gaa na **Gratis AI Agent → Settings → API Keys**
2. Tinye credentials maka ngwaọrụ ọ bụla
3. Nwale njikọ
4. Chekwaa n’ụzọ nchebe

### Ịwụnye Ngwugwu Advanced Ibe {#installing-the-advanced-companion-package}

A na-ewepụta ngwugwu Advanced ibe iche na WordPress.org isi ZIP. Wụnye ya naanị site na ụzọ nkesa ọrụ gọọmentị maka mbipụta ahụ, mgbe ahụ kpachie ohere nye ndị nchịkwa a tụkwasịrị obi. Nyochaa ọnụ ụzọ nkwado mmadụ na omume git snapshot tupu ime ka filesystem, database, WP-CLI, REST dispatcher, plugin builder, njikwa ndị ọrụ, ma ọ bụ ngwaọrụ benchmark rụọ ọrụ na saịtị mmepụta.

## Iji sd-ai-agent/site-scrape {#using-sd-ai-agentsite-scrape}

### Ojiji Dị Mfe {#basic-usage}

Rịọ Theme Builder ka o nyochaa weebụsaịtị:

```
Nyochaa imewe nke saịtị onye asọmpi m na competitor.com
ma tụọ aro mmezi imewe maka saịtị m.
```

### Nyocha Kpọmkwem {#specific-analysis}

Rịọ ụdị nyocha kpọmkwem:

```
Wepụ paleti agba site na example.com ma jiri ya dị ka mkpali.
```

```
Nyochaa nhazi navigation nke example.com ma tinye ụdị nhazi yiri ya
na saịtị m.
```

```
Lelee mmeghachi omume mobile nke example.com ma hụ na saịtị m
na-emeghachi omume otu ahụ.
```

### Nyocha Ntụnyere {#comparison-analysis}

Tụnyere ọtụtụ saịtị:

```
Tụnyere imewe nke site1.com na site2.com ma mepụta imewe
nke jikọtara ihe kacha mma n’ime ha abụọ.
```

## Mmachi Ngwaọrụ na Ihe A Ga-atụle {#tool-limitations-and-considerations}

### Mgbochi Ọnụọgụ Iji {#rate-limiting}

- A na-amachi ọsọ ịnakọta data iji gbochie ibu karịrị akarị n'ọrụ sava
- Arịrịọ ọtụtụ ugboro n'otu saịtị nwere ike bụrụ ihe a na-amachi ọsọ
- Saịtị buru ibu nwere ike iwekwu oge iji nyochaa

### Izizi nke Ọdịnaya {#content-accuracy}

- Enweghị ike ijide ọdịnaya na-agbanwe agbanwe kpamkpam
- Ọdịnaya JavaScript na-egosipụta nwere ike ghara izu oke
- Enweghị ike igosi data oge-ugbu a
- Ụfọdụ ọdịnaya nwere ike ịdị n'azụ paywalls

### Ntụle Iwu na Omume Ziri Ezi {#legal-and-ethical-considerations}

- Sọpụrụ robots.txt na atumatu saịtị
- Ejila ịnakọta data depụtaghachi ọdịnaya nwere ikike nwebisiinka
- Jiri nyocha maka mkpali, ọ bụghị maka idetuo
- Nyochaa na ị nwere ikike iji ọdịnaya ewepụtara
- Soro usoro ọrụ nke saịtị ndị enyochara

### Mmetụta na Ọrụ {#performance-impact}

- Ịnakọta data na saịtị buru ibu nwere ike iwe oge
- Ịnakọta data ọtụtụ n'otu oge nwere ike ịdị nwayọ
- Njikọ netwọkụ na-emetụta ọsọ
- Datasets buru ibu nwere ike ịchọ nhazi karịa

## Omume Kachasị Mma {#best-practices}

### Iji Nnyocha Saịtị {#using-site-analysis}

1. **Nyochaa ọtụtụ saịtị** — nakọta mkpali site n'ọtụtụ isi mmalite
2. **Lekwasị anya na nhazi** — mụta ụkpụrụ nhazi na ịhazi ihe
3. **Wepụta agba** — jiri palettes agba dịka ebe mbido
4. **Mụọ typography** — chọpụta ngwakọta font ndị masịrị gị
5. **Nyochaa ngagharị** — ghọta nhazi menu

### Ịnakọta Data n'Omume Ziri Ezi {#ethical-scraping}

1. **Sọpụrụ robots.txt** — soro ntuziaka saịtị
2. **Edetula ọdịnaya** — jiri nyocha maka mkpali naanị
3. **Nyochaa ikike** — hụ na ị nwere ike iji ọdịnaya ewepụtara
4. **Kpọọ isi mmalite aha** — nye isi mmalite mkpali otuto
5. **Soro usoro** — mee ihe kwekọrọ na usoro ọrụ saịtị

### Ịbawanye Irè Ngwá Ọrụ {#maximizing-tool-effectiveness}

1. **Kọwaa nke ọma** — rịọ ụdị nyocha kpọmkwem
2. **Nye ọnọdụ** — kọwaa ebumnuche saịtị gị
3. **Tọọ atụmanya** — kọwaa ebumnuche imewe gị
4. **Meegharị** — meziwanye dabere na nsonaazụ
5. **Jikọta ngwá ọrụ** — jiri ọtụtụ ngwá ọrụ maka nyocha zuru oke

## Idozi Nsogbu {#troubleshooting}

### Saịtị Agaghị Ekwe Ka Anakọta Data {#site-wont-scrape}

- Lelee ma saịtị ahụ dị maka ọha na eze
- Nyochaa na robots.txt na-enye ohere ịnakọta data
- Gbalịa saịtị ọzọ
- Lelee njikọ ịntanetị
- Kpọtụrụ support ma ọ bụrụ na nsogbu ahụ aga n'ihu

### Nnyocha Ezughị Ezu {#incomplete-analysis}

- Saịtị nwere ike ịnwe ọdịnaya na-agbanwe agbanwe
- JavaScript nwere ike ghara igosipụta kpamkpam
- Saịtị buru ibu nwere ike timeout
- Gbalịa nyochaa peeji kpọmkwem kama
- Rịọ ụdị nyocha kpọmkwem

### Ọrụ Dị Nwayọ {#slow-performance}

- Saịtị buru ibu na-ewekwu oge iji nyochaa
- Ịnakọta data ọtụtụ n'otu oge na-adị nwayọ karịa
- Njikọ netwọkụ na-emetụta ọsọ
- Gbalịa n'oge awa na-abụghị nke ọrụ kacha elu
- Nyochaa ngalaba pere mpe mbụ
