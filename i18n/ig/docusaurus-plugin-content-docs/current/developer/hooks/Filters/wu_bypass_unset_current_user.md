# Filter: wu_bypass_unset_current_user

Kwe ka ndị mmepe gafee koodu unset current user.

Ịlaghachi ihe ọ bụla na-abụghị null ga-eme ka a gafee unset nke onye ọrụ dị ugbu a batara. Nke a nwere ike ịba uru n’ọnọdụ ụfọdụ, dịka ọmụmaatụ, mgbe a na-emeso sub-sites a na-eji dịka admin panels.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $null_or_bypass | `mixed` | Null iji gaa n’ihu, ihe ọ bụla ọzọ iji gafee ya. |
| $current_user | `false\|\WP_User` | Ihe onye ọrụ dị ugbu a. |
