# Nzacha: wu_should_create_domain_record_for_site {#filter-wushouldcreatedomainrecordforsite}

Na-enyocha ma Ultimate Multisite kwesịrị ịmepụta ndekọ domain maka saịtị emepụtara ọhụrụ.

Jiri nzacha a gbochie ma ọ bụ yigharịa imepụta ndekọ domain na-akpaghị aka maka saịtị ndị na-eji domain ntọala nke fọm checkout a na-ekekọrịta, host ime, ma ọ bụ domain nke integration ọzọ ga-elekọta iche.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $create | `bool` | Ma ekwesịrị ịmepụta ndekọ domain. |
| $site | `WP_Site` | Objekt saịtị emepụtara ọhụrụ. |

### Kemgbe {#since}

- 2.13.0

### Isi mmalite {#source}

Akọwapụtara na `inc/functions/domain.php`.


## Nloghachi {#returns}

Boolean na-egosi ma a ga-emepụta ndekọ domain.
