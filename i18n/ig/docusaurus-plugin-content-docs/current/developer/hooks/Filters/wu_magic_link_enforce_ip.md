# Filter: wu_magic_link_enforce_ip

Filter ma a ga-amanye nkwenye adreesị IP.

Tọọ ya ka ọ bụrụ false iji kwe ka tokens rụọ ọrụ site na netwọkụ dị iche iche. Nke a na-ebelata nchekwa mana na-abawanye ịdị mfe iji ya (dịka ọmụmaatụ, maka ndị ọrụ ekwentị na-agbanwe netwọkụ).

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $enforce | `bool` | Ma a ga-amanye ka adreesị IP kwekọọ. |
