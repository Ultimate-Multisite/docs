# Nzacha: wu_minimum_password_length

Nzacha ogologo paswọọdụ kacha nta.

A na-amanye ya naanị mgbe wu_enforce_password_rules bụ true.

## Paramita

| Aha | Ụdị | Nkọwa |
|------|------|-------------|
| $min_length | `int` | Ogologo paswọọdụ kacha nta. Ndabara 12 (dakọtara na Defender Pro). |
| $defender_active | `bool` | Ma Defender Pro Strong Password dị n'ọrụ. |

### Kemgbe

- 2.4.0
### Isi mmalite
