# Nzacha: wu_admin_notices

## Paramita {#parameters}

| Aha | Ụdị | Nkọwa |
|------|------|-------------|
| $notices | `array` | Ndepụta ọkwa maka ogwe ahụ kpọmkwem. |
| $all_notices | `array` | Ndepụta ọkwa agbakwunyere, ekewapụtara dịka ogwe si dị. |
| $panel | `string` | Ogwe a ga-esi nweta ọkwa ndị ahụ. |
| $filter | `string` | Ma ọ bụrụ na ewepụla ọkwa ndị a pụrụ iwepụ site na nzacha. |
| $dismissed_messages | `array` | Ndepụta igodo ọkwa e wepụrụ. |
