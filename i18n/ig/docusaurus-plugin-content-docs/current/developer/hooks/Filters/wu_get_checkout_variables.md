# Nzacha: wu_get_checkout_variables {#filter-wugetcheckoutvariables}

Na-enye ndị mmepe plugin ohere ịza ntọala edobere tupu nke ibe ịkwụ ụgwọ.

Kpachara anya, igodo efu nwere ike imebi ịkwụ ụgwọ kpamkpam n’akụkụ ihu.

## Paramita {#parameters}

| Aha | Ụdị | Nkọwa |
|------|------|-------------|
| $variables | `array` | Mgbanwe emere ka ọ dabara mpaghara. |
| $checkout | `\Checkout` | Klas ịkwụ ụgwọ. |

### Kemgbe {#since}

- 2.0.0
### Isi mmalite {#source}

A kọwara ya na [`inc/checkout/class-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout.php#L1970) n’ahịrị 1970


## Ihe ọ na-eweghachi {#returns}
Usoro ọhụrụ nke mgbanwe.
