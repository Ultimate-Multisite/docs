# Omume: wu_domain_became_primary

Ọ na-eme mgbe ngalaba ghọrọ ngalaba isi maka saịtị.

A na-akpalite omume a mgbe etinyere akara primary_domain nke ngalaba ka ọ bụrụ true, ma ọ bụ mgbe a na-emepụta ngalaba isi ọhụrụ ma ọ bụ mgbe a na-emelite ngalaba dị adị ka ọ bụrụ isi.

## Parameters {#parameters}

| Aha | Ụdị | Nkọwa |
|------|------|-------------|
| $domain | `\WP_Ultimo\Models\Domain` | Ngalaba ahụ ghọrọ isi. |
| $blog_id | `int` | Blog ID nke saịtị emetụtara. |
| $was_new | `bool` | Ma nke a bụ ngalaba emepụtara ọhụrụ. |

### Kemgbe {#since}

- 2.0.0
