# Omume: wu_page_added

Kwe ka ndị mmepe mgbakwunye mee ihe ndị ọzọ mgbe edebanyere ibe.

N'adịghị ka wu_page_load, nke na-agba naanị mgbe a na-ele otu ibe kpọmkwem, hook a na-agba n'oge ndebanye maka ibe nchịkwa ọ bụla a na-agbakwunye site na iji koodu Ultimate Multisite.

## Paramita

| Aha | Ụdị | Nkọwa |
|------|------|-------------|
| $page_id | `string` | ID nke ibe a. |
| $page_hook | `string` | Aha hook nke ibe a. |
