Maka ebe akụkọ ọrụ gị, ị ga-abịa ịgwara ihe niile nke akụkọ ọrụ gị dị ka billings (ịgụọ nkwado), renewals (ịgụọ dịka ụta), cancelations (ịgụọ dịka anọchịrị), na ihe ndị ọzọ.
