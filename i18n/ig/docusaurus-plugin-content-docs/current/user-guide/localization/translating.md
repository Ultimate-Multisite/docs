# Chú Gốc Ultimate Multisite {#translating-ultimate-multisite}

Ultimate Multisite được phát triển bằng tiếng Anh, nhưng plugin này hoàn toàn có thể dịch và chúng tôi nhiệt liệt chào đón những đóng góp dịch thuật của bạn.

## Dịch Thuật Plugin Cốt Lõi {#core-plugin-translations}

Plugin cốt lõi Ultimate Multisite được dịch thông qua nền tảng dịch thuật chính thức của **WordPress.org**:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

Đây là hệ thống dịch cộng đồng tiêu chuẩn của WordPress. Khi bản dịch đạt đến ngưỡng phê duyệt đủ, chúng sẽ tự động được phân phối cho tất cả người dùng Ultimate Multisite thông qua các cập nhật plugin WordPress.

**Chúng tôi cần sự giúp đỡ của bạn!** Nếu bạn nói một ngôn ngữ khác tiếng Anh, xin hãy cân nhắc đóng góp bản dịch. Ngay cả việc dịch một vài chuỗi cũng giúp ích rất nhiều. Để bắt đầu:

1. Truy cập [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)
2. Chọn ngôn ngữ của bạn
3. Nhấn "Contribute Translation" (Đóng góp Bản dịch)
4. Bạn sẽ cần một tài khoản WordPress.org miễn phí

Nếu bạn muốn trở thành biên tập viên dịch thuật cho ngôn ngữ của mình (có khả năng phê duyệt bản dịch), hãy liên hệ qua [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions).

## Dịch Thuật Addon {#addon-translations}

Các plugin addon được dịch riêng tại:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Hiện tại, các bản dịch addon đang được tạo bằng AI làm điểm khởi đầu. Chúng bao gồm tất cả các ngôn ngữ được hỗ trợ nhưng có thể chứa những cách diễn đạt gượng gạo hoặc không chính xác. Chúng tôi hoan nghênh người dịch thuật là con người xem xét và đề xuất cải tiến để chúng trở nên tự nhiên và chính xác hơn.

### Ụdị Poedit (Using Poedit) {#using-poedit}

Ị nwere ike iji application desktop **[Poedit](https://poedit.net)** ị chịkọta file `.pot` ahụ:

1. Download Poedit na [poedit.net](https://poedit.net)
2. Open file `.pot` ahụ tunu n' `wp-content/plugins/ultimate-multisite/lang/`
3. Translate strings ndị a n'interface Poedit
4. Save -- Poedit ga jere file `.po` na `.mo`
5. Jiji output files ndị a n'folder `lang`

### Ụdị Loco Translate (Using Loco Translate) {#using-loco-translate}

Plugin WordPress **[Loco Translate](https://wordpress.org/plugins/loco-translate/)** na-eme ka ị chịkọta direkt n'admin panel anụ ahụ ma ọ bụ anyị na-agba desktop tools ahụ. Ọ ga-eme ka ị jere na-compile translation files maka gị.
