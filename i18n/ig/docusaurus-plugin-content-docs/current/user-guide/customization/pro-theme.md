# Customizing Admin Panel-nwaa-m-PRO Theme {#customizing-your-admin-panel-with-pro-theme}

O n'ụrụ ịrụ aka na atụike akụkọ ọrụ ndị ahụ, anyị ga-anọrọ ịnweta ihe dị mma na dashboard WordPress nke client. Ka ọ bụrụ na ị na-iji ihe dị mkpa na ịhụ ihe dị profọfessional na nkwupụta, PRO Theme ga-anọrọ ịkpọta gị.

PRO Theme bụ akụkọ admin (admin theme) nke a na-adị mma karị ọrụ gị mgbe ị chọrọ ịchọta dashboard ndị client ahụ. Ọ bụkwa ọ dị n'ime core plugins premium Ultimate Multisite.

Ọ bụrụ na ị na-agbanwe ya, jiji ị na _Plugins_ page-nwaa, ji kpọta ya ka ị chọta ya na Network-nwaa gị.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Jiji ya n'ime Network-nwaa gị na Plugin page_

_**NOTE:**_ _Ọ bụrụ na ị na-agbanwe ya, ọ ga-anọrọ na client-nwaa gị ka ihe dị mma nke a na-abụghị ike ịkpọta ya ka ị chọta ya na user ahụ n'gụrụ._

Nke a bụ dashboard admin gị nke a na-agbanwe ụdị! Ọkụ ọzọ, ị na-agba menu ọhụrụ na column akụkọ kiri na-adị PRO Theme.
