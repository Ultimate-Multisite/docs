# Plác Adah na ọkụmụta gị na Ultimate Multisite: Ad Injector {#placing-ads-on-your-network-with-ultimate-multisite-ad-injector}

Ultimate Multisite: Ad Injector bụ **plugin mai** nke na-eme ka ị nwere ihe ndị ahụ iji plác adah na ọkụmụta gị direkt na sahọdụ edit page ha.

Ọ bụrụ na ị chọrọ Ultimate Multisite: Ad Injector, jikọpụta (go) _Plugins_ page-onye maka ị activate ya.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Jikọpụta nút Network Activate_

Ọ bụrụ na add-on ahụ nwere, _Advanced Options_ meta box na screen edit plan ahụ ga-anọ nwere ihe ọhụrụ a na-adị na-ezi _Ad Settings_.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_Ọ bụ n'ebe ị ga ị chọpụta ad settings maka ọkụmụta gị ahụ._

Na tab ahụ, ị ga ị nwere ike ị chọpụta ad codes maka posts na pages na front-end, ọtụtụ ihe ndị a naịa ọrụ content ahụ.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Ọ bụrụ na ị chọpụta ad code-onye, ọ ga-anọ n'posts na pages ahụ ahụ._

Ị chụrụ adah na header Admin Pages maka ndị na-eme ọrụ ọkụmụta ahụ.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Jikọpụta (go) Admin Panel Ad Code option ma ị chọrọ ịplác adah na back-end nke ndị na-eme ọrụ ahụ._

### Ọ dị n'ụ allá Adịghị Nwere {#beyond-advertising}

Ị chụrụ Ultimate Multisite: Ad Injector maka ịnwale ihe ọrụ maka plans ndị bi nwere kedu.

Jikọpụta (go) _Admin Panel Ads_ option ma jikọpụta HTML code maka support tool ahụ.
