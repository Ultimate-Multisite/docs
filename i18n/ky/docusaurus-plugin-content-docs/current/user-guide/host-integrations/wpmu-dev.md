# WPMU DEV Интеграция {#wpmu-dev-integration}

## Обзор {#overview}
WPMU DEV — это комплексная платформа WordPress, которая предлагает хостинг, плагины и услуги для сайтов на WordPress. Эта интеграция позволяет автоматически синхронизировать домены и управлять SSL-сертификатами между Ultimate Multisite и хостингом WPMU DEV.

## Возможности {#features}
- Автоматическая синхронизация доменов
- Управление SSL-сертификатами
- Расширенные попытки проверки SSL-сертификатов

## Требования {#requirements}
Интеграция автоматически определяет, размещаете ли вы сайт на WPMU DEV, и использует встроенный API. Если вы размещаете сайт на WPMU DEV, дополнительная настройка не требуется.

Интеграция проверяет наличие константы `WPMUDEV_HOSTING_SITE_ID`, которая автоматически определяется при размещении на WPMU DEV.

## Инструкции по настройке {#setup-instructions}

### 1. Проверка хостинга на WPMU DEV {#1-verify-wpmu-dev-hosting}

Если вы размещаете сайт на WPMU DEV, необходимые константы уже должны быть определены. Убедитесь, что:

1. Константа `WPMUDEV_HOSTING_SITE_ID` определена в вашей среде
2. У вас есть активная подписка на WPMU DEV с доступом к API

### 2. Включение интеграции {#2-enable-the-integration}

1. В админке WordPress перейдите в Ultimate Multisite > Settings (Настройки)
2. Перейдите на вкладку "Domain Mapping" (Сопоставление доменов)
3. Прокрутите вниз до раздела "Host Integrations" (Интеграции хостинга)
4. Включите интеграцию WPMU DEV
5. Нажмите "Save Changes" (Сохранить изменения)

## Как это работает {#how-it-works}

### Синхронизация доменов {#domain-syncing}

Когда домен сопоставляется в Ultimate Multisite:

1. Интеграция использует API WPMU DEV для добавления домена в ваш аккаунт хостинга
2. Она также автоматически добавляет версию с `www` доменом
3. WPMU DEV занимается настройкой домена и выдачей SSL-сертификатов

### Управление SSL-сертификатами {#ssl-certificate-management}

Интеграция WPMU DEV хостинг үчүн SSL сертификатын верификацияга колдонуу санын көбөйтүүгө карата конфигурацияланган, анткени SSL сертификаттары бекитилip орнолгондо убакыт кетиши мүмкүн. Дефолtdа ал стандарттык 5 буйрукка салыштырмалуу SSL сертификатын верификациясы үчүн 10 жолго чейин аракет кылат.

## Маанилүү эскертүүлөр {#important-notes}

### Доменди алып салуу {#domain-removal}

Азыркы учурда WPMU DEV API домендерди алып салуу жолун камсыз этпейт. Ultimate Multisiteде домен маппинг алында болсо, домен сиздин WPMU DEV хостинг эсебиңизде калат. Зарыл болсо, аны WPMU DEV хостинг дашбордунан колдонуп алып салууңуз керек.

### API аутентификациясы {#api-authentication}

Интеграция WordPress базаңызга `wpmudev_apikey` опциясы реквизит катары сакталган WPMU DEV API ачкычын колдонот. Сайтты WPMU DEVге туузуп кармап жатканда бул автоматтык түрдө орнотулат.

## Ар кандай көйгөйлөрдү чечүү {#troubleshooting}

### API байланыш маселелери {#api-connection-issues}
- Сайттын WPMU DEVге туура кабыл алынганын текшериңиз.
- `wpmudev_apikey` опциясынын WordPress базаңызга коюлганын текшериңиз.
- Сиздин WPMU DEV мүчөлүгү активдүү экенин камсыз кылыңыз.

### SSL сертификат маселелери {#ssl-certificate-issues}
- WPMU DEV SSL сертификаттарын бекитүүгө убакыт кетиши мүмкүн (көбүнчө 5-15 мүнөт).
- Интеграция SSL сертификаттары үчүн 10 жолго чейин текшерип колдонууга конфигурацияланган.
- Көп буйруктан кийин да SSL сертификат бекитилбесе, WPMU DEV колдоо кызматы менен байланышыңыз.

### Домен кошулбаган {#domain-not-added}
- Ultimate Multisite логдорунда кандайдыр бир ката билдирүүлөр бар экенин текшериңиз.
- Домен алдын ала WPMU DEVге кошулган эмес экенин текшериңиз.
- Сиздин WPMU DEV хостинг планы сиз кошо турган домендер санын колдонуп турганын камсыз кылыңыз.
