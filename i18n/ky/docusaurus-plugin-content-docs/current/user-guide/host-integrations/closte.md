# Closte Интеграция {#closte-integration}

## Обзор {#overview}
Closte — бул Google Cloud инфраструктурасы дээр курулган башкарылган WordPress хостинг платформасы. Бу интеграция Ultimate Multisite жана Closte ортосунда домендерди автоматтык түрдө синхролоо жана SSL сертификаттарын башкарууну мүмкүн кылат.

## Функциялар {#features}
- Домендерди автоматтык түрдө синхролоо
- SSL сертификатын башкаруу
- Wildcard домендерди колдонуу
- Closte дээр иштесе, конфигурацияга болбойт

## Талаптар {#requirements}
Эгер сиз Closte дээр хостинг кылсаңыз, `wp-config.php` файлыңызда төмөнкү константаны аныктошуңуз керек:

```php
define('CLOSTE_CLIENT_API_KEY', 'your_api_key');
```

Бул константа Closte дээр хостинг кылсаңыз, адатта да аныкталган болот.

## Орнотуу Нускамалары {#setup-instructions}

### 1. Closte API Ключünüzü Текшериңиз {#1-verify-your-closte-api-key}

Эгер сиз Closte дээр хостинг кылсаңыз, `CLOSTE_CLIENT_API_KEY` константасы `wp-config.php` файлыңызда да аныкталган болушу керек. Бул сунушту `wp-config.php` файлын текшерип аныктай аласыз.

### 2. Интеграцияны Иштетиңиз {#2-enable-the-integration}

1. WordPress администратордук панелинде Ultimate Multisite > Settings (Сотустар) бөлүмүнө кириңиз
2. "Domain Mapping" (Домен картасы) табасына барыңыз
3. Төмөндө "Host Integrations" (Хостинг интеграциялары) деген жерге жетүү үчүн төмөндө жылчуңуз
4. Closte интеграциясын иштетиңиз
5. "Save Changes" (Өзгөртүүлөрдү сактоо) батырууну унутуңуз

## Иш Жүргөнүгүн {#how-it-works}

Ultimate Multisiteде домен картасы жасалганда:

1. Интеграция Closte API-сына доменди ваша программаңызга кошуу үчүн сураныч (request) жөнөтөт
2. Closte SSL сертификатын автоматтык түрдө даярдап берет
3. Домен картасы алынганда, интеграция доменди Closteден алып салат

Интеграция Ultimate Multisiteдеги DNS текшерүү аралыгын (DNS check interval) менен да иштешет, бул система домендин таралышын жана SSL сертификатын берилишин канча жиикат текшерерин конфигурациялоо мүмкүнчүлүгүн берет.

## Домен Калдыгын Жасактоо {#domain-record-creation}

Бул интеграция сайты түзүлгөн же көчүрүлгөн кезде домен катмарын автоматтык түрдө түзүлүп кала турганын камсыз кылат. Бул Closte интеграциясы үчүн өтө маанилүү, анткени домен катмарын түзүү Closte API-сын домен жана SSL сертификатын түзүүгө уруксат берет.
