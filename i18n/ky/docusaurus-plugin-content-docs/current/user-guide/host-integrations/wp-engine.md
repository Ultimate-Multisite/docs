# WP Engine Интеграция {#wp-engine-integration}

## Жалпы маалымат (Overview) {#overview}
WP Engine — бул WordPress сайттары үчүн оптималдаштырылган аткарууну, коопсуздугун жана көлөмдөөнү камсыз кылган 프리миум башкарылган WordPress хостинг платформасы. Бул интеграция Ultimate Multisite менен WP Engine ортосунда домендерди автоматтык түрдө синхролоону мүмкүн кылат.

## Функциялар (Features) {#features}
- Домендерди автоматтык түрдө синхролоо
- мультисайт орнотуулары үчүн субдомендерди колдоо
- WP Engine'дин бардык системалары менен үзгүлтүксүз интеграция

## Талаптар (Requirements) {#requirements}
Интеграция сиз WP Engine дээр хостинг кылганыңды автоматтык түрдө аныктайт жана киргизилген WP Engine API-сын колдонот. Эгерде WP Engine плагини активдүү болсо жана туура конфигурацияланган болсо, кошумча конфигурация талап кылынбайт.

Бирок, интеграцияны өзүңүз колдонуп конфигурациялоо керек болсо, сиз `wp-config.php` файлыңызда бул констанналардын бирин аныктай аласыз:

```php
define('WPE_APIKEY', 'your_api_key'); // Эң сунушталган ыкма
// ЖЕРГЕ
define('WPE_API', 'your_api_key'); // Башка ыкма
```

## Орнотуу Нускамалары (Setup Instructions) {#setup-instructions}

### 1. WP Engine Плагинын текшериңиз {#1-verify-wp-engine-plugin}

Эгер сиз WP Engine дээр хостинг кылсаңыз, WP Engine плагины дагы эле орнотулуп жана активдүү болушу керек. Төмөнкүлөрдү текшериңиз:

1. WP Engine плагины активдүү
2. `wp-content/mu-plugins/wpengine-common/class-wpeapi.php` файлы барбы

### 2. Интеграцияны иштетиңиз (Enable the Integration) {#2-enable-the-integration}

1. WordPress администратордук панелинде Ultimate Multisite > Settings бөлүмүнө кириңиз
2. "Domain Mapping" (Домен картасы) таасир этүүчү вкладкага өтүңүз
3. Төмөнкү жерге, "Host Integrations" (Хостинг интеграциялары) деген бөлүмгө жылчуңуз
4. WP Engine интеграциясын иштетиңиз (Enable the WP Engine integration)
5. "Save Changes" (Өзгөртүүлөрдү сактоо) баскыңыз

## Ал кандай иштейт (How It Works) {#how-it-works}

### Домендерди синхролоо (Domain Syncing) {#domain-syncing}

Ultimate Multisiteде домен картасы жасалганда:

1. Интеграция WP Engine API-сын колдонуп, доменди сиздин WP Engine орнотумуңузга кошот
2. WP Engine домен конфигурациясын жана SSL сертификатын берип берүүсүн башкарат
3. Домен картасы алында болсо, интеграция доменди WP Engine'ден алып салат

### Субдомендерди колдоо (Subdomain Support) {#subdomain-support}

Субдомен мультисайт орнотуулары үчүн:

1. Интеграция жаңы сайт түзүлгэне ар бир субдоменди WP Engine'ге кошот
2. WP Engine субдомен конфигурациясын башкарат
3. Сайт өчүрүлгөндө интеграция субдоменди WP Engine'ден алып салат

## Маанилүү эскертүүлөр {#important-notes}

### Жайырмалуу домендер (Wildcard Domains) {#wildcard-domains}

Субдомен мультисайт орнотуулары үчүн, бардык субдомендерди ар бирине кошоп киргизүүгө болбойт. Ошондуктан, сиз WP Engine колдоо кызматы менен байланышып, жайырмалуу домен (wildcard domain) конфигурациясын сурап алууну сунуштайбыз. Бул бардык субдомендердин автоматтык түрдө иштөөсүн камсыз кылат.

### SSL сертификаттары {#ssl-certificates}

Бул интеграция аркылуу кошулган бардык домендер үчүн WP Engine автоматтык түрдө SSL сертификатын берип берүү жана жаңыртууну башкарат. Кошумча конфигурация талап кылынбайт.

## Ар кандай көйгөйлөрдү чечүү (Troubleshooting) {#troubleshooting}

### API байланышына байланыштуу көйгөйлөр {#api-connection-issues}
- WP Engine плагини активдүү экенин жана туура конфигурациялангандыгын текшериңиз
- Эгер сиз API ачкычын өзүңүз колдонуп аныктаган болсоңуз, ал туура экенин текшериңиз
- API менен көйгөй жаралса, WP Engine колдоо кызматы менен байланышыңыз

### Домен кошолбайт {#domain-not-added}
- Эч кандай ката билдирүүлөрү бар экенин Ultimate Multisite логдорунан текшериңиз
- Домен дагы эле WP Engine'ге кошулбагандыгын текшериңиз
- Сиздин WP Engine планы сиз кошууну пландаган домендер санына туура келишине кам көрүңүз

### Субдомен көйгөйлөрү {#subdomain-issues}
- Эгер субдомендер иштебесе, жайырмалуу домен конфигурациясын сурап алуу үчүн WP Engine колдоо кызматы менен байланышыңыз
- Негизги домен жана субдомендер үчүн DNS жөндөмдү туура конфигурациялап алганыңызды текшериңиз
