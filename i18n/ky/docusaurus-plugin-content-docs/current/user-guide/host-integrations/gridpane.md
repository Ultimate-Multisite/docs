# GridPane Интеграция

## Обзор {#overview}
GridPane — это специальная панель управления хостингом для WordPress, созданная для серьезных профессионалов в этой сфере. Эта интеграция позволяет автоматически синхронизировать домены и управлять SSL-сертификатами между Ultimate Multisite и GridPane.

## Возможности {#features}
- Автоматическая синхронизация доменов
- Управление SSL-сертификатами
- Автоматическая настройка константы SUNRISE

## Требования {#requirements}
В вашем файле `wp-config.php` должны быть определены следующие константы:

```php
define('WU_GRIDPANE', true);
define('WU_GRIDPANE_API_KEY', 'ваш_api_ключ');
define('WU_GRIDPANE_SERVER_ID', 'ваш_server_id');
define('WU_GRIDPANE_APP_ID', 'ваш_site_id');
```

## Инструкции по настройке {#setup-instructions}

### 1. Получите учетные данные API GridPane {#1-get-your-gridpane-api-credentials}

1. Войдите в свою панель управления GridPane
2. Перейдите в "Настройки" > "API"
3. Сгенерируйте ключ API, если у вас его еще нет
4. Скопируйте свой ключ API

### 2. Получите ID вашего сервера и сайта {#2-get-your-server-and-site-ids}

1. В панели управления GridPane перейдите в раздел "Серверы" (Servers)
2. Выберите сервер, где размещен ваш WordPress мультисайт
3. Запомните Server ID (он виден в URL или на странице деталей сервера)
4. Перейдите в "Сайты" (Sites) и выберите свой сайт WordPress
5. Запомните Site ID (он виден в URL или на странице деталей сайта)

### 3. Добавьте константы в wp-config.php {#3-add-constants-to-wp-configphp}

Добавьте следующие константы в ваш файл `wp-config.php`:

```php
define('WU_GRIDPANE', true);
define('WU_GRIDPANE_API_KEY', 'ваш_api_ключ');
define('WU_GRIDPANE_SERVER_ID', 'ваш_server_id');
define('WU_GRIDPANE_APP_ID', 'ваш_site_id');
```

### 4. Включите интеграцию {#4-enable-the-integration}

1. В админке WordPress перейдите в Ultimate Multisite > Настройки (Settings)
2. Перейдите на вкладку "Domain Mapping" (Сопоставление доменов)
3. Прокрутите вниз до раздела "Host Integrations" (Интеграции хостов)
4. Включите интеграцию GridPane
5. Нажмите "Save Changes" (Сохранить изменения)

## Как это работает {#how-it-works}

Когда домен сопоставляется в Ultimate Multisite:

1. Интеграция домендиңизди сайтқа кошуу үчүн GridPane APIга сураныч жөнөтөт.
2. GridPane SSL сертификатын автоматтык түрдө даярдайт (provisioning).
3. Домен маппинг алынса, интеграция домени GridPaneдан алып салат.

Интеграция ошондой эле `wp-config.php` файлындагы SUNRISE константасын автоматтык түрдө башкарат, бул доменди маппинг туура иштөө үчүн зарыл.

## SUNRISE Константасын Баскыч (Management) {#sunrise-constant-management}

GridPane интеграциясынын өзгөчөлүгү – ал `wp-config.php` файлындагы SUNRISE константасын автоматтык түрдө кайра коюп берет, бул GridPaneнын өзүнүн доменди маппинг системасы менен кезиши мүмкүн болгон чырчалууларды алдын алат. Бул эки системанын бирге иштөөсүн катасыз жүргүзүүгө мүмкүндүк берет.

## Ооруларды Артыруу (Troubleshooting) {#troubleshooting}

### API Байланышы Керектөөлөрүндө {#api-connection-issues}
- API ключуңу туура экенин текшериңиз.
- Сервердин жана сайттын ID'лери туура экенин текшериңиз.
- GridPane эсебиңизде зарыл укуктары бар экенин камсыздан.

### SSL Сертификаты Керектөөлөрүндө {#ssl-certificate-issues}
- GridPane SSL сертификатту чыгарууга бир аз убакыт кетиши мүмкүн.
- Домендердин сиздин сервердин IP дарегине туура багытталганын текшериңиз.
- Сайт үчүн GridPaneнын SSL жөндөмдөрүн текшериңиз.

### Домен Кошулбаган {#domain-not-added}
- Ultimate Multisite логдорунда кандайдыр бир ката билдирүүлөрү бар экенин текшериңиз.
- Домендин дагы эле GridPaneга кошулган эмес экенин текшериңиз.
- Домендүн DNS катмарлары туура конфигурацияланган экенин камсыздан.
