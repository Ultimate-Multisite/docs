# V1-денди миграциялоо {#migrating-from-v1}

## Ultimate Multisite эскизиси 1.x версиясынан 2.x версиясына өтүп кетти. {#ultimate-multisite-has-switched-from-its-original-1x-family-of-releases-to-the-2x-family-of-releases}

Ultimate Multisite 2.0 жана андан жогору версиясы кодду толук кайра жазылган, бул эски версия менен жаңысы арасында өтө аз дал келгени дегенди билдирет. Ошондуктан, 1.x-ден 2.x-ке 업그레이д учурунда сиздин маалыматыңар жаңы версиялар түшүнө турган форматка миграциялануу керек болот.

Баарын айтканда, Ultimate Multisite 2.0+ **корутундучу (migrator)** менен келет, ал эски версиядан маалыматтарды аныктап, аларды жаңы форматка которууга кубаттуу. Бул миграция 2.0+ версиясынын **Setup Wizard** учурунда болот.

Бул сабакта мигратор кантип иштегенин, ката келип чыкса эмне кылуу керектигин жана бул процесс учурунда пайда болушу мүмкүн болгон көйгөйлөрдү кантип чечүү керектигин камтыйбыз.

_**МААЛЫМДАР: 1.x версиясынан 2.0 версиясына 업그레이д кылууга баштагандын алдында сайт маалыматтарынын резерлөөсүн жасап алганыңызга экенин текшериңиз**_

## Биринчи кадамдар {#first-steps}

Биринчи кадам — плагин .zip файлына жүктөп алуу жана версия 2.0-ды ваша сеть администратордук панелинде орнотуу.

Сиз [install and activate version 2.0](../getting-started/installing-ultimate-multisite.md) менен версия 2.0'ды орнотуп, активкактагандан кийин, система сиздин Multisite эски версияда иштейт дегенди автоматтык түрдө аныктайт жана плагин баасынын үстүндө бул билдирүүнү көрөсүз.

_**Эскертүү:** Эгерде сизде Multisite-те Ultimate Multisite 1.x орнотулган болсо, сиз плагинти азыр жүктөп алган версия менен алмаштыруу мүмкүнчүлүгүн аласыз. Сураныч, **Replace current with uploaded** баскычын басып көрүңүз._

<!-- Скриншот жок: v1-ди жүктөлгөн v2 версиясы менен алмаштыруу опциясын көрсөткөн плагин баасы -->

Кийинки баракча сизге версия 1.x менен орнотулган эски плагиндерди көрсөтөт. Ал сиз колдонуп жаткан версия 2.0 менен дал келбеши же миграциядан кийин плагиндин жаңыланган версиясын орнотуу керектиги жөнүндө маалымдама берет.

<!-- Screenshot unavailable: Message on the top of the plugins page about updating to v2.0, with list of add-ons that need updating -->

Жаңы баскычты, **Upgrade'ды аткаруу үчүн Installerди кезектеги** деп жазылган баскычты басып койсоңуз, уланта аласыз.

<!-- Screenshot unavailable: Button saying Visit the Installer to finish the upgrade -->

Андан кийин сизди акыл-эс маалымдамалары бар орнотуу wizards (wizard) баракчасына алып баратпайт. Экинчи баракчага өтүү үчүн жөн гана **Get Started** дегенди басыңыз.

<!-- Screenshot unavailable: Setup Wizard welcoming page with Get Started button -->

**Get Started**-ды басып койгондон кийин, сиз Pre-install Checks_._ баракчасына кайра багытталасыз. Бул сизге Сиздин Система маалыматтарын жана WordPress орнотулушуңузду көрсөтөт жана алар **Ultimate Multisite'тын талаптарына** туура келерин айтып берет.

<!-- Screenshot unavailable: Pre-install Checks page showing system requirements confirmation -->

Кийинки кадам — Ultimate Multisite лицензиялык ключ (license key) сизге киргизүү жана плагинтикти активдештирүү. Бул сиздин сайтта, анын ичинде плагиндерди да камтыган бардык функциялар колдонууга мүмкүн болгонун камсыз кылат.

<!-- Screenshot unavailable: License activation page with license key input and Agree and Activate button -->

Ключту киргизгенден кийин, **Agree & Activate**-ды басыңыз.

Лицензия активдештирилгенден кийин, сиз чыныгы орнотууну улантуу үчүн кийинки баракчадан **Install**-ды баса аласыз. Бул версия 2.0 иштөө үчүн зарыл болгон керектүү файлдарды жана базалык маалыматтарды автоматтык түрдө түзөт.

## Тепер миграция {#now-the-migration}

Мигратордогуңунда кире берилген коопсуздук функция бар. Ал сиздин Ultimate Multisite маалыматын эч кандай көйгөйлөргүзбүүсүз миграциялашы мүмкүн экенин текшерип чыгат. Процес баштага үчүн **Run Check** баскычын басыңыз.

Миграцияны текшергенден кийин, сизде эки мүмкүнчүлүк болот: натыйжа **карыт** же **карытсыз**.

### Карыт менен {#with-error}

Эгер сиз ката билдирүүсүн алсаңыз, анда биздин колдоо кызматтарына кайрылышыңыз керек. Алар катаны оңдоодо сизге жардам бере алышат. Тикет жаратууда **каталогду (error log) берге** кам көрүңүз. Логду жүктөп алсаңыз болот же "contact our support team" деп жазылган шилтемеге басып жибере аласыз. Бул сиздин бетиң оң жагындагы жардам элементтерин ачат, анда сизге каталогдор киргизилген талаалар менен даяр болот.

_**Система катаны таап алса, сиз 2.0 версиясына миграциялоого өтө албасыз. Андан кийин, ката оңдолгонго чейин тармагыңызды иштөөгө ылайык 1.x версиясына кайра келе аласыз.**_

### Карытсуз {#without-error}

Эгер система эч кандай катаны таап албаса, сиз ийгилик билдирүүсүн жана миграцияга өтүүнү мүмкүн болгон төмөндөгү **Migrate** баскычын көрөсүз. Бул бетте сизге андан ары жүрүүдөн мурун базалык маалыматтын резерптин (backup) жасашыңызды эскеришет, биз бул өтүнүчтү өтө сунуштайбыз. Эгерде сиздин резербиңиз бар болсоңуз, **Migrate** басыңыз.

Иш ушундай сонун!

Сиз либончаWizard kurulumunu улантып, кызматтардын логотибин жана башка нерселерди тазаласаңыз болот, же Ultimate Multisite версия 2.0 менюсуна жана анын жаңы интерфейшине баштасаңыз болот. Баары сиздин чечимиңизде. Көңүл буруңуз!
