# Ultimate Multisite Хуктар жана фильтрлер

Ultimate Multisite көптөгөн функциялар жана базалык суроолордон турат. Анда сиз бардык функцияларды өзгөртүү же жаңы кодду кошуу мүмкүн болгон хуктар (hooks) жана фильтрлер (filters) бар.

Бул хуктар жана фильтрлердин тизмесин колдонуучулардын өз тармагын дарооча ылайыкташтыруу үчүн референс катары түзүп салганыбызды билдиребиз. Сиз бул тизмеге Github-дан астынкы шилтеме аркылуу кире аласыз:

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
