# Ultimate Multisite фрагменттерингини GitHub репозиторийде кантип колдонуу

Ultimate Multisite колдонуучулары көп сураган код фрагменттери (snippets) GitHub репозиторийде бар. Булар сиз жаңы каттоо эшілген сапта Google Analytics скриптинди кошуу же администратордун панелиндеги мета боксту жашыруу сыяктуу кичинекей функцияларды кошуу үчүн колдоно аласыз.

Бул макала сизге бул коддорду кантип колдонууну же тактап айтканда, кайсы жерге коюу керектигин көрсөтөт.

Фрагменттерин төмөнкү шилтемеден таба аласыз:

https://github.com/next-press/wp-ultimo-snippets/

Кодду кошуу үчүн сизге 2 жол бар:

  1. Темаңыздын functions.php файлына.

  2. Must-Use Plugins (mu-plugins).

# Фрагменттерин темаңыздын functions.php файлына кантип кошуу керек.

  1. WordPress Network администратордук панелинде кириңиз жана Themes > Theme Editor (тистик четин төмөнкү скриншоту караңыз) багытынан өтүңүз.

  2. "Edit Themes" деген бетте, экрандын оң жээги жагындагы түшүрүүчү талаада активдүү темаңызды тандаганыңызга кам көрүңүз (#3 скриншотунун төмөнкү кадрасында).

  3. "Theme Files" бөлүмүнөн functions.php файлына басып, файлды жүктөө үчүн ачыңыз. Төмөндөгү эң төмөндө барып, GitHub репозиторийден алган Ultimate Multisite фрагменттерин чаптаңыз.

# Must-Use Plugins (mu-plugins) кантип түзүү керек

WordPressде "Must-Use Plugins" же кыскача "mu-plugins" деп аталган, кастомдык функцияларды жүктөөгө мүмкүндүк берген функция бар.

Бул атайын mu-плагиндер башка бардык адаттагы плагиндерден мурун жүктөлөт жана аларды өчүрө албасыз. Multisite тармагында бул mu-плагиндердеги код сиздин орнотулган бардык сайттарда жүктөөчү болот.

1. WordPress орнотулган файлынын файлоо системасына FTP же SSH аркылуу кирүү үчүн колдонуңуз.

2. WordPress орнотуңуздун `wp-content` директориясынын ичинде `mu-plugins` деп аталган жаңы директорияны түзүңүз.

<!-- Скриншот жок: Файл менеджерде wp-content директориясы жана mu-plugins папкасы көрсөтүлгөн -->

3. Компьютериңизде Notepad же башка код редактору аркылуу `wu-snippet.php` деп аталган жаңы PHP файлы түзүңүз.

4. GitHub репозиторийден алган Ultimate Multisite код бөлүктүн файлга коюп сактаңыз. Ошондой эле, mu плагиниңизди белгилөө үчүн бул коддун үстүнө аны кошуп койсоңуз болот.
