# Ultimate Multisite-ты-бирдиги

Ultimate Multisite ангilis тилде иштелип чыккан, бирок плагин толук которууга ылайыктуу жана биз которууга активсиздүү кубанычтабыз.

## Негизги Плагин Котормолору

Негизги Ultimate Multisite плагини расмий **WordPress.org которуу платформасы** аркылуу которулат:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

Бул стандарттык WordPress коомчулугунун которуу системасы. Котормолор жетиштүү бекилоо чегине жеткенде, алар WordPress плагин жаңыртуулары аркылуу Ultimate Multisite колдонуучуларына автоматтык түрдө таратылат.

**Бизге жардам керек!** Эгер сиз англис тилинен башка бир тил билсеңиз, котормолорду сунуштооңузду карап көрүңүз. Бир нече строкту которуу да жардам берет. Башталышы үчүн:

1. [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/) сайтына кириңиз
2. Өз тилиңизди тандаңыз
3. "Contribute Translation" (Котормо сунуштоо) баскычын басыңыз
4. Сизге WordPress.org үчүн акысыз эсеп керек болот

Эгер сиз өз тилиңиздин которуу редактору болуп калууну кааласаңыз (котормолорду бекитүүгө ыкчам), [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) аркылуу ма相談 кылыңыз.

## Дорогу Плагиндердин Котормолору

Дорогу плагиндерди атайын которуу:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Дорогу плагин котормолору учурда башталыш nuqta катары AI аркылуу жасалган. Алар колдоого алынган бардык тилдерди камтыйт, бирок аларда кылдат сөздөр же тактыктысы жок болушу мүмкүн. Аларды көбүрөөк табигый жана так кылуу үчүн адам которуучуларынан карап чыгууну жана жакшыртууларды сунуштооңузду кубанычтабыз.

## Жергиликтүү Котормолор

Эгер сиз өз долбооруңуз үчүн котормолорду жергиликтүү түзүүнү кааласаңыз, жогорку деңгээлде колдоо көрсөтүүгө кирбесеңиз да, эки вариант бар:

### Poedit'тен пайдалануу

Сиз `.pot` файлына которуу үчүн **[Poedit](https://poedit.net)** десктоп тиркемесин колдонсоңуз болот:

1. [poedit.net](https://poedit.net) сайтынан Poedit'ти жүктөп алыңыз
2. `wp-content/plugins/ultimate-multisite/lang/` дарек астындагы `.pot` файлын ачыңыз
3. Poedit интерфейсиндеги тексттерди которуңуз
4. Сайга сактаңыз -- Poedit `.po` жана `.mo` файлдарын түзөт
5. Жыйынтыгын `lang` папкасына коюңуз

### Loco Translate'тен пайдалануу

**[Loco Translate](https://wordpress.org/plugins/loco-translate/)** WordPress плагини сизге десктоп куралдарысыз администратордук панелден түздөн-түз которууну мүмкүнчүлүк берет. Ал сиз үчүн котормо файлдарын түзүп, бириктирет.
