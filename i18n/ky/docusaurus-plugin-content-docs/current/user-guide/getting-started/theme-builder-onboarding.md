# Тема Билдер Интеграция Процесс (Theme Builder Onboarding Flow)

Superdav AI Agent v1.12.0 сизга баштапкы орнотоодо ылайыктуу **Тема Билдер интеграция процессин** сунуштайт, бул сиздин алгачкы курулушуңузда өзүңүздүн уникалдуу блоктон теманы түзүүгө жардам берет. Бул эски Site Builder режимине орду алган, көбүрөөк ийкемдүү жана агентдин жардамдашы менен иштөө ыкмасы болуп саналат.

## Тема Билдер Интеграция Процесси деген эмне? {#what-is-the-theme-builder-onboarding-flow}

Тема Билдер интеграция процесси — бул интерактивдүү орнотоо wizards (сунушчу) болот, ал:

- Дизайн чечимдери боюнча сизди багыттайт (рангдар, типография, макет).
- Сайттын көрүнүшү жөнүндөгү каалоолоруңузду колдойт.
- Сиздин муктаждыктарыңызга ылайыкталган уникалдуу блоктон теманы түзөт.
- Аяктаганда теманы автоматтык түрдө активдештирет.

Бул процесс **Setup Assistant агент** аркылуу ишке ашырылат, ал тактоочу суроолорду берип, сиздин теманы кадам-кадам курат.

## Тема Билдер Интеграция Процессин Башталышы {#starting-the-theme-builder-onboarding}

### Биринчи Кезекте Орнотоо (First-Run Setup) {#first-run-setup}

Эгерде сиз Superdav AI Agentти жаңы WordPress орнотушунда биринчи жолу иштетсеңиз, төмөнкүлөрдү көрөсүз:

```
Superdav AI Agent'го кош келиңиз!

Сиз эмне кылууну каалайсыз?
1. Уникалдуу тема куруу (Theme Builder)
2. Маанилүү теманы колдонуу
3. Азыр үчүн орнотоону өткөрүү
```

Тема Билдер интеграция процессине кирүү үчүн **"Уникалдуу тема куруу"** дегенди тандаңыз.

### Кол менен Активдештирүү (Manual Activation) {#manual-activation}

Сиз Тема Билдер интеграция процессин кез келген убакта сурап баштай аласыз:

```
"Тема Билдер интеграция процессин башта"
```

же

```
"Уникалдуу тема түзүүгө жардам бергиңиз"
```

## Интеграция Процессинин Кадамдары {#the-onboarding-steps}

### 1-Кадам: Режимди Тандоо (Mode Selection) {#step-1-mode-selection}

Setup Assistant агент сиздин каалооңуз жөнүндө сурашат:

```
Темаңызды кантип курууну каалайсыз?
- Багытталган (Мен суроолор берип, сизге курам)
- Кол менен иштөө (Мен сизге тандоо мүмкүнчүлүктөрүн көрсөтөм жана сиз чечимиңизди бериңиз)
```

**Багытталган режим** көпчүлүк колдонуучулар үчүн сунушталган; агент сиздин өнөңүз жана максаттарыңызга негизделген дизайн сунуштарын берет.

### 2-Кадам: Сайттын Аталышы (Site Specification) {#step-2-site-specification}

Сизден сайт жөнүндө сурашат:

- **Сайт максаты**: Электрондук товарлар сатуу (E-commerce), блог, портфолио, SaaS ж.б.
- **Максаттуу аудитория**: Сиздин киришелер кимдер?
- **Бренд түстөрүн**: Негизги жана экинчи түстөр (же "Мага так эмес")
- **Тон (Аткарылыш стили)**: Профессионал, чыгармачылык, ойноок, минималдуу ж.б.

Бул маалымат сиздин **site_brief** эсэтиңизде сакталат жана келечек сессияларда агенттер ошону карашат.

### 3-чара: Дизайн системасы чечимдери {#step-3-design-system-decisions}

Агент сизге дизайн токендерди тандоо боюнча багыт берет:

- **Типография**: Шрифт үй-бүлөсү (serif, sans-serif, monospace) жана өлчөм масштабдары
- **Түстөр палитрасы**: Негизги, экинчи, акцент жана нейтралдуу түстөр
- **Убакыттар (Spacing)**: Кичинекей, нормалдуу же кеңири убакыттуу макеттер
- **Кискоо (Motion)**: Анимациялар жана өтүү эффекттери (каалаган болсо)

### 4-чара: Теманы түзүү {#step-4-theme-generation}

Setup Assistant агент сиздин кастом блок темаңызды төмөнкүлөр менен каркастайт:

- Бардык дизайн токендери бар `theme.json`
- Көрсөткүч макеттер (башкы бет, блог, контакт) үчүн Блок шаблондору
- Сиздин дизайн системаңызга дал келген кастом блок стилдери
- Тема метамаалыматтары жана WordPress колдоо билдирүүлөрү

### 5-чара: Жүргүзүү жана текшерүү {#step-5-activation-and-verification}

Тема автоматтык түрдө активдештирилет, жана сиз төмөнкүлөрдү көрөсүз:

```
✓ Сиздин кастом темаңыз азыр иштейт!
  Тема аталышы: [Сайтыңыздын Аты] Темасы
  Түстөр: [Негизги] / [Экинчи]
  Типография: [Шрифт Үй-бүлөсү]

  Жаңы дизайны көрүү үчүн сайтыңызды ачыңыз.
```

Андан кийин, тема туура көрсөтүлүп жатканын текшерүү үчүн сайтыңызды баса аласыз.

## Сайттын талаптары жана site_brief эсети {#site-specification-and-sitebrief-memory}

Киришүү учурунда агент сиздин сайтыңыздын маалыматтарын **site_brief** эсэти категориясына түшүрөт. Бул төмөнкүлөрдү камтыйт:

- Сайттын максаты жана максаттары
- Максаттуу аудитория
- Бренд түстөрүн жана тонду
- Дизайн предпочэлери
- Мазмун структурасы

### site_brief эмне үчүн маанилүү? {#why-sitebrief-matters}

Келечек сессияларда агенттер сиздин site_brief эсетин карап, төмөнкүлөрдү аткарат:

### Дизайнды өзгөртүүлөрдү сактау {#viewing-your-sitebrief}

*   Өзгөртүүлөрдүн кезінде дизайндын бирдей болушун сактаңыз.
*   Сайтыңыздын максатына ылайыкталган функцияларды сунуштаңыз.
*   Контекстке байланыштуу сунуштарды бериңиз.
*   Установка боюнча суроолорду кайталоодон качыңыз.

### Сайттинди көрсөтүү (site_brief) {#customizing-after-onboarding}

Сиз агентке төмөнкүлөрдү сурасаңыз болот:

```
"Show me my site brief"
```

же

```
"What do you know about my site?"
```

Агент сакталып калган сайт спецификасын көрсөтөт.

## Интеграциядан кийин өзгөртүү (Customizing After Onboarding) {#use-the-design-system-aesthetics-skill}

Тема куруу (Theme Builder) иштеп чыгуу процесси аяктагандан кийин, сиз төмөнкүлөрдү кыла аласыз:

### Дизайн система эстетикасы көндүмдөрүн колдонуу {#edit-themejson-directly}

Дизайнды тактоо сураңыз:

```
"Refine the typography to be more modern"
```

же

```
"Adjust the color palette to be warmer"
```

**Design System Aesthetics skill** сизге натыйжалуу дизайн өзгөртүүлөрүн көрсөтөт.

### theme.json-ду түз редакциялоо {#create-custom-block-templates}

Көрсөтмөлүү колдонуучулар үчүн, төмөнкүлөрдү жөнгө салуу үчүн `/wp-content/themes/[theme-name]/theme.json` файлына түз редакциялаңыз:

*   Цвет токендерин (Color tokens)
*   Типографика шкалаларын (Typography scales)
*   Убакыт аралык маанилерди (Spacing values)
*   Чекит жана көлөкөлөрдүн аныктамаларын (Border and shadow definitions)

### Өзгөртүр блоктордун үлгүлөрүн түзүү {#comparison-old-vs-new-onboarding}

WordPress блок редакторун колдонуп төмөнкүлөр үчүн өзүңүздүн үлгүлөрүңүздү түзүңүз:

*   Үндүрүштүн башкы бети (Homepage layouts)
*   Блог макалалары сапка (Blog post pages)
*   Өнүмдүн беттери (Product pages)
*   Байланыш формалары (Contact forms)

## Салыштыруу: Эски vs. Жаңы Интеграциядан кийинки процесс {#troubleshooting}

| Функция | Сайт куруучу (Эски) | Тема куруучу (Жаңы) |
| :--- | :--- | :--- |
| Установка ыкмасы | Визуалдык көрсөтмөлүү форма | Агент тарабынан багытталган сүйлөшүү |
| Теманы түзүү | Четелген үлгүлөр | Өзгөртүр каркас (Custom scaffolding) |
| Дизайн токендери | Колу менен киргизүү | Багытталган чечимдер |
| Ийгиндүүлүк | Бекитилген варианттар | Жөнгө салууга мүмкүн болгон |
| Келечектеги жаңыртуулар | Элтирилбейт | site_brief-те сакталат |

## Ооруларды чечүү (Troubleshooting) {#next-steps}

**Интеграциядан кийинки процесс аяктабады**
*   Процесди кайра баштоо: "Start the Theme Builder onboarding"
*   WordPress орнотуңуздун актуалды экенин текшериңиз.
*   Setup Assistant агенти күйүп жатканын текшериңиз.

**Менин сайт_brief колдонуппайрылбайт**
- Агентке эс тутумга кирүү мүмкүнчүлүгү бар экенин текшериңиз.
- Агентке "менин сайт_brief-имди эске алып бер" деп сураңыз.
- Эс тутумunuz жөндөмдөрүнүн орондунда күчтүрүп жатканыңызды текшериңиз.

**Женерацияланган тема менин каалоолууларыма дал келбейт**
- Аны тактоо үчүн Design System Aesthetics колдонуу көндүмдүүлүгүн колдонуңуз.
- Агентке "[айырмачылыктар менен] теманы кайра жасоо" деп сураңыз.
- Так көзөмөл жүргүзүү үчүн theme.json файлын түз редактор аркылуу өзгөртүңүз.

## Кийинки кадамдар

Тема куруу (Theme Builder) окутууну аяктагандан кийин:

1. **Сайты текшериңиз**: Жаңы теманы көрүү үчүн сайттыңызды басыңыз.
2. **Дизайнда тактоо**: Айрыхтмалар үчүн Design System Aesthetics көндүмдүүлүгүн колдонуңуз.
3. **Мазмун кошуу**: Сайттын мазмунун курууга баштайсыз.
4. **Көндүмдөрдү изилдеңиз**: scaffold-block-theme жана activate-theme сыяктуу башка агент көндүмдөр жөнүндө үйрөнүңүз.
