# Жаңылыктар {#release-notes}

## 2.13.0 версиясы — 2026-06-05 күнү чыкты {#version-2130--released-on-2026-06-05}

- Жаңы функция: Клиенттик аккаунт, чекаут (checkout), төлөм, сайт, инвойс (invoice), шаблон алмаштыруу жана домен картасы агымдары үчүн суверейн-тенент (sovereign-tenant) поддержку кошуuldu. Бул тенентдик тармактардын колдонуучуларды башкарылган иштер үчүн негизги сайтта кайтарып коюусуна мүмкүндүк берет.
- Жаңы функция: Кайра жаңартуулар (renewal) үчүн которуу (credential) текшерүүлөрү кошуuldu, ошондук-соо gateway'лор сакталып калган төлөм келишими, жазылуу же vault токени болбогон учурда автоматтык жаңартууну өчүрө алат.
- Жаңы функция: Сайттын даярдалышы үчүн күтүлгөн (pending) сайттарды HMAC-текшерүү аркылуу жарыялоо кошуuldu, бул артыда жүрүүчү жумуштар кечигип калган хосттордо чекауттан сайтта колдонууга өтүүнү дагы ишенимдүү кылат.
- Жаңы функция: SSO URL'лери, чекаут-форма базалык домендер жана автоматтык домен-жазылуулар (domain-record) түзүү үчүн өнөктөштөр (extension points) кошуuldu.
- Оңдолгон функция: Карталанган домендер аркылуу SSO, аноним блоктордун кире бериши, логиттоо жана плагиндин кесип чыгышындагы конфликттер аркылуу SSO дагы ишенимдүү болуп калды.
- Оңдолгон функция: Сайттын даярдалышы үчүн күтүлгөн сайттар эми эски жарыялоо белгилеринен (stale publish flags) оордой алат жана колдонуучуларды сайтты түзүү экранында убада калуудан сактайт.
- Оңдолгон функция: Баалуу чекаут-форма базалык домендер үчүн домен жазылуулары эми түзүлбөйт, ал эми интеграция активдүү эмес болгондо колдонулган хост-провайдердин артыда жүрүүчү жумуштары өткөрүлбөйт.
- Оңдолгон функция: Чекаут, төлөм дареги, парольди кайра коюу, электрондук почтады текшерүү, шаблон алмаштыруу, турлар (tours) жана колдонуучу панелиндеги чекит учурлары нормалдуу колдонуучу агымдарын токтотпойт.
- Оңдолгон функция: Блоктуар электрондук каттар эми алуучуларды жашыруун сактап, ал эми алуучу тизмелери же кат жөнөтүү системалары иштебей калган учурда SMTP/плагиндин каталарын болбосо да аткарылат.
- Оңдолгон функция: Мемлекеттик жаңартуулар, мөөнөтү өткөн көрсөтүү жана төлөмдөрдү топтоо чекит учурлары эми дароо мөөнөтүн өткөрүүгө, программалоону токтотуп калууга же зарыл төлөмдөрдү упурууга жол бербейт.
- Жакшыртылган функция: WordPress 7.0 чегин текшерилди, өндүрүштөгү Vue активтери npm булактарынан кайра курулду жана Cypress эң алдыңкы (end-to-end) баалоосы эми дагы көп чекаут, орнотуу, SSO жана gateway агымдарын текшерет.
