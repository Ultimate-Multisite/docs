# Сайт Билдир Модулу Жок Болоткуу Эскертүүсү

**Site Builder modu Superdav AI Agent v1.12.0-да алынды.** Эгер сиз Site Builder modunu колдонуп жатсаңыз, темаларды жана сайттарды куруу үчүн **Setup Assistant agent**ге өтүшүңүз керек.

## Эмне болду?

### Site Builder Modу (Эски)

Site Builder modu төмөнкүлөр үчүн wizards (өткөм) 기반 интерфейси болгон:

- Шаблондордон сайттарды түзүү
- Базалык параметрлерди конфигурациялоо
- Теманы тандоо
- Баштапкы мазмунду орнотуу

### Анын ордуна эмне келди?

**Setup Assistant agent** эми төмөнкүдөр менен Site Builder функциялары бардык иштерди башкарат:

- Көбүрөөк ийкемдүү, агент тарабынан багытталган орнотуу
- Теманы ылайыкташтыруу үчүн жакшыраак тандоо опциялары
- Theme Builderге кирүү (onboarding) менен интеграция
- Келечек сессиялар үчүн `site_brief` убадалуу эс тутум

## Эгер сиз Site Builder Modunu Колдонуп Жатсаңыз

### Сайттарыңыз Коопсуз

- Site Builder modu менен түзүлгөн бардык сайттар иштеп жатат
- Маалымат жоголбойт же сайтта бузуку болбойт
- Сиз сайттарыңызды колдонууну кадимкидей уланта аласыз

### Setup Assistant Agentге Өтүңүз

Жаңы сайтты орнотуу же теманы өзгөртүү үчүн Setup Assistant agent'ти колдонуңуз:

```
"Help me set up a new site"
```

же

```
"Start the Theme Builder onboarding"
```

Setup Assistant agent дагы көп ийкемдүүлүктү камтыйт.

## Салыштыруу: Site Builder vs. Setup Assistant

| Функция | Site Builder (Жок болду) | Setup Assistant (Жаңы) |
|---|---|---|
| Орнотуу ыкмасы | Wizard формасы | Agent сүйлөшү |
| Теманы тандоо | Алдын ала аныкталган шаблондор | Өздей түзүү |
| Ылайыкташтыруу | Четелген опциялар | Толук дизайн системасы |
| Сайттин кыскача маалыматы (site brief) | Саفوظу мүмкүн эмес | Убадалуу эс тутум |
| Келечек сессиялар | кайталоочу орнотуу | сакталган site_brief-ти колдонуу |
| Ийкемдүүлүк | Беделген иш кагымдуулугу | Адаптивдүү сүйлөшүү |

## Setup Assistant Agentге Өтүү

### Жаңы Сайттар Үчүн

Сайт құру режимін қолданбаңыз:

1. Сұраныс: "Мен жаңа сайт орнатуға көмектесеңіз"
2. Орнату ассистент агенті сізге келесілер арқылы бағыт береді:
   - Сайттың мақсаты мен міндеттері
   - Мақсатты аудитория
   - Брендтің сипаты
   - Теманы генерациялау
   - Бастапқы конфигурация

### Қазіргі сайттар үшін

Егер сізде Site Builder режимінен жасалған бар сайт болса:

1. Оны өз күйінде қолдануға жалғастыруға болады
2. Теманы жаңарту үшін сұраңыз: "Сайтты қайта дизайнын жасаңыз"
3. Орнату ассистент агенті сізге жаңа тема құруға көмектеседі
4. Сайт деректері өзгермейді

### Тема өзгерістері үшін

Site Builder режимінің теманы таңдауын алмастырған кезде:

1. Сұраныс: "Теманымды өзгерт"
2. Орнату ассистент агенті мыналарды істейді:
   - Сіздің дизайн қалауыңыз туралы сұрақ қояды
   - Қолданылған (custom) тему генерациялайды
   - Оны сіздің сайтыңызда белсендіреді

## Негізгі айырмашылықтар

### Site Builder Режимі

```
1. Шаблон таңдау
2. Теманы таңдау
3. Бастапқы параметрлерді реттеу
4. Бітті
```

### Орнату Ассистент Агенті

```
1. Сайттың мақсатын сипаттаңыз
2. Мақсатты аудиторияны анықтаңыз
3. Дизайн қалауларыңызды таңдаңыз
4. Агент қолданылған тему генерациялайды
5. Агент темуны белсендіреді
6. Болашақ сессиялар үшін сайттың қысқаша сипаттамасы сақталады
```

## Орнату Ассистент Агентінің артықшылықтары

### Көбірек икемділік

- Сайтты табиғи тілде сипаттаңыз
- Қолданылған ұсыныстарды алыңыз
- Сіздің нақты қажеттіліктеріңізге бейімделу

### Жақсы реттеу мүмкіндігі

- Қолданылған (custom) тема генерациялау
- Дизайн жүйесінің шешімдерін жасау
- Тұрақты дизайн токендері

### Тұрақты жад

- Сіздің site_brief-і сақталады
- Болашақ агенттер сіздің сайтыңызды түсінеді
- Орнату ақпараттарын қайта айту қажеттілігі жоқ

### Бірыңғай жұмыс істеу

- Тема құру кезіндегі басталу (onboarding)
- Дизайн жүйесінің эстетикасы дағдысы
- Көру бақылауларын қолдану мүмкіндігі
- Барлығы бір-бірімен үздіксіз жұмыс істейді

## Қателіктерді шешу

### Site Builder режимін таба алмаймын

Site Builder режимі жойылды. Орынды Орнату ассистент агентін қолданыңыз:

"Мен мага жаңы сайт орнотууга жардам бериңиз"

### Site Builderden сайтты кайра түзгүм келет

Сайтты Setup Assistant агенти аркылуу кайра түзө аласыз:

1. Сураныч: "Мен жаңы сайт орнотууга жардам бериңиз"
2. Оригиналды сайтыңыздын максатын жана дизайнын сүрөттөп бериңиз
3. Агент окшош теманы жасайт
4. Сиздин `site_brief` (сайт жөнүндө кыскача маалымат) келечекте караш үчүн сакталат

### Менин бар Site Builder сайтым иштебей жатат

Site Builder режими менен түзүлгөн бардык сайттар азыр да иштеп турат. Эгер сиз көйгөйлөрдү кабырсаңыз:

1. Темаңыз активдүү экенин текшериңиз
2. Плагиндердин (plugins) күчтөлдүрүлүп жатканын текшериңиз
3. WordPress ката журналдарын (error logs) текшериңиз
4. Көйгөйлөр улап турса, колдоо кызматы менен байланышыңыз

### Эски Site Builder шаблондорумду колдонуп жата абы?

Site Builder шаблондору артыкчылыксыз болуп калды. Бирок:

- Сиздин бардык сайттары да иштеп турат
- Setup Assistant агенти аркылуу окшош сайттарды кайра түзө аласыз
- Setup Assistant агенти көбүрөөк ыңгайлаштыруу мүмкүнчүлүктөрүн берет

## Кийинки кадамдар

1. **Жаңы сайттар үчүн**: Setup Assistant агентин колдонуңуз
2. **Бардык сайттар үчүн**: Аларды азыркы абалында колдонууга уланыңыз
3. **Теманы өзгөртүү үчүн**: Setup Assistant агентиден жардам сураңыз
4. **Дизайнды тактоо үчүн**: Design System Aestheticsスキル (skill) аркылуу колдонуңуз

## Байланыш темалары

- **Theme Builder Onboarding**: Сиздин кастом темаларыңыз үчүн багытталган орнотуу
- **Setup Assistant Agent**: Агент менен багытталган сайтты орнотуу
- **Site Specification Skill**: Сайттын максаттарын жана аудиториясын аныктоо
- **Design System Aesthetics Skill**: Сайттын көрүнүшүн тактоо
