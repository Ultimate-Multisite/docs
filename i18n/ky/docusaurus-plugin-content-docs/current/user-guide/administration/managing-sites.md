# Сайттарды башкаруу

Сайттар (же субсайттар) сиздин WaaS бизнесиңиздин негизги бөлүгү. Ultimate Multisiteде 3 түрү бар сайттар:

- **Кардар ээликтеги** — Атайын кардарларга тиешелүү сайттар
- **Сайт шаблондору** — Кардарлар башталыш нүкте катары тандаса болот мурунку аныкталган сайттар
- **Негизги сайт** — Сиздин негизги тармак сайтыңыз

## Сайттарды көрүү

Тармактарыңыздагы бардык субсайттарды көрүү үчүн **Ultimate Multisite → Sites** баракчасына кириңиз. Ар бир сайт тиби менен белгиленет жана сиз "Бардык сайттар", "Кардар ээликтеги", "Шаблондор" же "Күтүлбөй турган" деп фильтрдеп көрө аласыз.

![Sites list page](/img/admin/sites-list.png)

## Жаңы сайт кошуу

Жаңы сайт түзүү үчүн **Add Site** баскычын басыңыз. Сиз төмөнкүлөрдү киргизишиңиз керек:

- **Сайт аталышы** — Жаңы сайттын аты
- **Сайт URL-и** — Сайттын домендү/жолу
- **Сайт тиби** — Ал кардар сайтыбы, шаблонбу же адаттагы сайтпы?

**Copy Site** опциясы сизге мурунку сайт шаблонунун негизинде жаңы сайт түзүүгө мүмкүндүк берет. Бул активдүү болгондо, кайсы шаблонду башталыш нүкте катары колдонуу керектигин тандасаңыз болот. Медиа файлдарды камтуу үчүн **Copy Media on Duplication** активдүү экенин унутпаңыз.

## Учурдагы сайтты башкаруу

Сайттын **Edit Site** баракчасын ачуу үчүн кайсы сайтка болсо, ошол сайдын жанындагы **Manage**-ды басыңыз. Бул жерде сиз төмөнкүлөрдү табасыз:

### Негизги маалыматтар

Сайттын аталышы, тиби, сайт ID'си жана сүрөттөмөсү. Ошондой эле карталанган домен, байланышкан мүшенныектер жана сайтты ээлей турган кардардын аккаунту да көрсөтүлөт.

### Сайт опциялары

Сайттын мүмкүнчүлүктөрүн жана чектөөлөрүн конфигурациялаңыз:

- **Көрсөтүү чектөөсү** — Сайттагы максималдуу сайттарды издөө саны
- **Колдонмо аккаунтунун чектөөлөрү** — Ар бир колдонуучу ролу үчүн чектөөлөр
- **Диск аймагы** — Сайт үчүн сактагыч чеги
- **Өзгөрткөн домен** — Бул сайт үчүн домен картасын активдештирүү
- **Плагин жана тема көрүнүшү** — Кайсы плагин жана темалар көрүнүп турушун, жашырылуусун же алдын ала активдештирилишин контролдоо

Дегенмен, сайттар дефолтта мүшендик деңгээлинен белгиленген чектөөлөргө ылайы турушат. Сайт деңгээлинде чектөөлөрдү коюу, мүшендик жөндөмдөрүн үстөмдөгөн болот.

### Байланыштар (Associations)

Сайт опцияларынан кийин төмөнкү маалыматтарды табасыз:

- Сайт менен байланышкан **Мапталган домендер**
- Сайттын кайсы **Мүшендик** астында экени
- Сайт менен байланышкан **Кардардын аккаунту**

### Оң жактагы баярлык (Right Sidebar)

Оң жакта сиз төмөнкүлөрдү кыла аласыз:

- Баяндау баскычы аркылуу сайтты **ички/ички эмес режимде иштетүү**
- Сайттын түрүн **өзүңүз өзгөртүү** же ээликке кайра таштап берүү
- Сайт үчүн **сайт сүрөтүн/миниатюруну коюу** (сайт шаблондорунда алдын ала көрсөтүлөт)
- Сайтты **катуу жоюу**

:::warning
Сайтты жабуу мүмкүн эмес. Сайт жана анын бардык мазмуну катуу алынат.
:::
