# Экспорт жана импорт

Ultimate Multisite 2.9.0 дагы **Tools > Export & Import** (Чыгаруу жана импорт) деген бөлүмдө бир сайтты ZIP файлы катары пакеттештирүү, андан ошону калыбына келтирүү же бирCompatible Ultimate Multisite жана single-site WordPress орнотуулары арасында сайтты которуу керек болгон учурда колдонууга болот.

## Ректрөөлөр талап кылынат

Сайттын экспорту же импортторуна кирүүгө мүмкүнчүлүгү бар администратор катары киришиңиз керек. Multisite тармагында, сеть-уровду Ultimate Multisite куралдарынан подсайттарды экспорту же импорттоо учурунда сеть администратору эсебинен колдонуңуз.

Экспорт ZIP жүктөп алуучу конекси аркылуу көрсөтүлөт, ошондуктан администратор сессиясын жүктөө аяктаганча активде калтырыңыз жана түзүлгөн жүктөө URL'лерин публик жерде бөлүшпөңүзбөй.

## Сайтты ZIP форматына экспорту

1. Копилоону каласаңыз, сайттын WordPress администраторунда **Tools > Export & Import** (Чыгаруу жана импорт) дегенге барыңыз.
2. Экспорт аймагын ачып, пакеттештиргиңиз келген сайтты тандаңыз.
3. Эгер ошол опциялар бар болсо, жүктөлгөн файлдарды (uploads), плагиндерди жана темаларды тандоо аркылуу каалаган мазмунду көрсөтүңүз.
4. Экспортту баштаңыз жана процесстин аякташына күтүңүз. Чоң сайттар ZIP даяр болгонго чейин артында аякташы мүмкүн.
5. Аяктаган ZIP файлын экспорлар тизмесинен жүктөп алыңыз.

ZIP файлына коопсуз жерде сактаңыз. Ал сайттын мазмунун, жөнгө салууларды, медиа файлдарын жана танланган код активтерин камтышы мүмкүн.

## Экспорт эмнени камтыйт

Экспорт ZIP форматы төмөнкүлөрдү камтый алат:

- Тандалган сайттын базалык мазмуну жана конфигурациясы.
- Жүкөлгөн файлдарды жүктөлгөн файлдар киргизилсе.
- Ошол опциялар тандалса, плагиндер жана темалар.
- Сайтты максаттуу орнотууда кайра куруу үчүн Экспорт & Import куралы тарабынан колдонулган импорт метадата.

Жазуунун так ZIP өлчөмү медиа материалдарынын көлөмүнө, тандалган плагиндерге жана темаларга, ошол сайттын базалык таблицаларынын өлчөмүнө жараша болот.

## Сайтты ZIP файлыдан импорттоо

1. Багыткы WordPress сайтында **Tools > Export & Import** (Инструменттер > Экспорттоо жана импорттоо) бөлүмүнө кириңиз.
2. Импорт аймагын ачып, Экспорттоо жана импорттоо куралы аркылуу түзүлгөн ZIP файлын жүктөп коюңуз.
3. Сайт жаңы дарекке колдонулушу керек болсо, алмаштыруу URL-ын киргизиңиз, же эски URL-ды сактагыңыз келсе аяктоочу талаага бош калтырыңыз.
4. Импортто заверте болгондон кийин жүктөлгөн ZIP файлын өчүрүүгө мажбур болосузбу? деген суроого жооп бериңиз.
5. **Begin Import** (Импортту баштоо) баскычын басыңыз.
6. Импортто күткүп турганды аяктаганча көзөмөлдөңүз. Процес бүтүүдөн мурун токтотуу керек болсо гана **Cancel Import** (Импортту жокко чыгаруу) баскычын колдонуңуз.
7. Кадимки трафикти же кардарлардын кирүүсүн уруксат берүүгө алдында импортталган сайтты текшериңиз.

Бир сайттуу WordPress орносо, ZIP файлыдан импорттоо учурдагы сайтты импортталган сайт менен алмаштырат. Баштапкы иш-аракеттерден мурун максаттуу сайттын толук резерв копиюсун жасаңыз жана бир эле убакта бирдей сайттар үчүн көптөгөн импортторду баштоодон сактаныңыз.

## Чектөөлөр жана ыңгайлаштыруу жөнүндө маалымдама

Көп маалыматтуу (Very large) жүктөр же медиа китепканалары чоң ZIP файлдары болушу мүмкүн. Чоң сайттарды экспорттоодон же импорттоодон мурун PHP жүктөө чектөөлөрүн, аткаруу чектөөлөрүн, диск убактысын, эс тутумду жана сервердин убакыт чектерин текшерип алыңыз.
Көп маалыматтуу медиа китепканалары төмөн трафикийдүү тейлөө чуркасында жылдырылышы керек болушу мүмкүн.
Максаттагы WordPress орнотулган версиясы, PHP, Ultimate Multisite, плагин жана тема версиялары менен дал келиши керек.
Бир сайтты импорттоо максаттуу сайты алмаштырат. Бул биригиш куралы эмес.
Multisite-ден single-siteге жана single-site-дан multisiteге өтүү операциялары гана максаттуу чөйрөнү экспортталган сайттын плагиндерин, темаларын, URL'дерин жана Ultimate Multisite үчүн зарыл болгон компоненттерин иштете алса колдоого алынат.
ZIP файлын жашыруун (private) каласың. Ал экспортирүлгөн сайттан базалык маалыматтарды, жүктөлгөн файлдарды жана конфигурация деталдарын камтышы мүмкүн.

Эски тармак деңгээли боюнча экспорт иш-аракеттары үчүн [Site Export](/user-guide/administration/site-export) окуу колдонмосуна караңыз.
