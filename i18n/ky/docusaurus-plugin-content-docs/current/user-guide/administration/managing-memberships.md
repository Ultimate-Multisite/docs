# Чөйрөлдерди башкаруу {#managing-memberships}

Ultimate Multisite системасында сиз тармагыңыздагы бардык жазылууларды жана жеткирүү (subscription) маселелерин башкара аласыз. Пландар жана продукттардын, сайттардын санынын, төлөм мөөнөтү жана жиылылыгынын, төлөм тарыхынын жана убакыт белгилеринин сизге керектүү маалыматтарын карап чыгып, редакцияласаңыз болот.

Бул баракка кирүү үчүн Ultimate Multisite менюсундагы **Memberships** (Чөйрөлөр) табына өтүңүз.

![Memberships list page](/img/admin/memberships-list.png)

Memberships баарында сиз мурунку кардыларыңыздын жана алардын аккаунттары менен байланышкан продукттардын тизмесин көрө аласыз. Чөйрөлөрдү статус боюнча фильтрлей аласыз: активдүү (active), күтүүдө (pending), убада берилгенде (on hold), мөөнөтү өткөн (expired) же жокко чыгарылган (cancelled).

Байттардагы чөйрөлөр **убакытсыз** (lifetime) чөйрөлөр катары каралат. Аларда мөөнөтү белгилөөсү болбойт жана автоматтык түрдө убактысы өтпөйт. Эгерде акысыз план сынакты же белгилүү мезгилден кийин аякташы керек болсо, сынакты камтыган төлөмдүү продуктду колдонуңуз, чөйрөлүүнү колдонуп жокко чыгарыңыз же жеткилоосу өзгөрүшү керектиги учурда кардыр башка планга өткөрсе болот.

Чөйрөлгөнүн тандасаңыз, сиз чөйрөлүнүн маалыматтарын — төлөм маалыматын, сайт же визит чектөөлөрүн жогорудан карап редакцияласаңыз болот.

![Membership edit interface](/img/admin/membership-edit.png)

Чөйрөлүүнү редакциялоонун толук көзү:

![Membership edit full interface](/img/admin/membership-edit-full.png)

Ошондой эле, мурунку тармагыңызда катталган колдонуучуларга тиешелүү жаңы чөйрөлөрдү кошуп жиберсеңиз болот. **Add Membership** (Чөйрөлгөнү кошуу) басын басып, маалыматтарды конфигурациялаңыз.

![Add Membership modal](/img/admin/membership-add-modal.png)

Андан кийин **Create Membership** (Чөйрөлгөнү түзүү) басын басыңыз. Жаңы чөйрөлө Memberships тизмесинде пайда болот.
