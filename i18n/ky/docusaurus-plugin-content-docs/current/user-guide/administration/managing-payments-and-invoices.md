# Толуптарды жана сурамжылоолорду башкаруу {#managing-payments-and-invoices}

## Төлөм 설정дору {#payment-settings}

Төлөм алууга баштоодон мурун, төлөм менен байланышкан параметрлерди даярдашыңыз керек. **Ultimate Multisite → Settings**-ке барыңыз жана **Payment** (Төлөм) таамырына басыңыз.

![Payment settings tab](/img/admin/settings-payments-top.png)

Бул төлөм 설정дорунун толук көзү:

![Payment settings full page](/img/admin/settings-payments-full.png)

### Жалпы Төлөм Параметрлери {#general-payment-options}

Жалпы параметрлерде сиз төмөнкүлөрдү конфигурацияласаңыз болот:

- **Currency** (Валюта) — Транзакциялар үчүн колдонулган дефолт валюта
- **Currency Position** (Валюта орду) — Валюта символу кайсы жерде чыгат (максаттан мурун/андан кийин)

![Payment gateway settings](/img/admin/settings-payments-options.png)

### Төлөм Эскертүүчүлөр (Payment Gateways) {#payment-gateways}

Ultimate Multisite бир нече төлөм эскертүүчүлөрдү колдойт. Ар бирин аларды төлөм 설정дорунан күйгүзүп, конфигурациялоо мүмкүн.

![Payment gateway configuration](/img/admin/settings-payments-gateways.png)

Кабыл алынган эскертүүчүлөр:

- **Stripe** — Stripe аркылуу кредиттик карталар менен төлөө
- **PayPal** — PayPal төлөмдөрү
- **Manual** (Кол менен) — Офлайн же ысымыңиздеги төлөм иштетүү үчүн

Ар бир эскертүүчү өзүнө тиешелүү конфигурация бөлүмү бар, мында сиз API ключтерин жана башка параметрлерди киргизесиз.

![Additional gateway settings](/img/admin/settings-payments-gateways-2.png)

### Sandbox Режими (Sandbox Mode) {#sandbox-mode}

Төлөм интеграцияңызды иш жүзүндө иштетүүдөн мурун сынап көрүү үчүн **Sandbox Mode**-ду күйгүзүшүңүз мүмкүн. Sandbox режими активдүү болгондо, чыныгы төлөмдөр жасалбайт.

## Төлөмдөрдү Көрсөтүү {#viewing-payments}

Ultimate Multisite астындагы **Payments** (Төлөмдөр) баракчага барып, тармагы боюнча бардык транзакцияларды көрө аласыз.

![Payments list](/img/admin/payments-list.png)

Сиз төлөмдөрдү статусуна (аяктаган, күтүүдө, ишке ашырылган жок, кайтарылган) ылайык фильтрдеп жана атайын транзакцияларды издөө аркылуу карасаңыз болот.

Төлөмдү басып, анын толук маалыматтарын, анын чегинен турган товарларды, убактылуу мүчөлүктөргө тиешелүү маалыматтарды жана төлөм ээсинин кызматтык маалыматтарын көрсөтсө болот.

## Инвойстар (Invoices) {#invoices}

Ultimate Multisite төлөмдөр үчүн инвойстарды автоматтык түрдө түзebilir. Сиз инвойсттердин үлгүсүн жана номерирүү форматын Төлөм параметрлеринен өзүңүз кала аласыз.

Инвойсттерди ылайыкташтыруу мүмкүнчүлүктөрү төмөнкүлөрдү камтыйт:

- Инвойсттордо көрсөтүлө турган **Компаниянын аталышы жана дареги**
- **Инвойст номерин** форматы жана тартиби
- Инвойст башкы бөлүгүнө коюла турган **Логотип**
- Шарттар, эскертүүлөр же мыйзамдуу маалыматтар үчүн **Өзгөрткөн футер тексти**

Инвойст үлгүсүн өзүңүз калагыңыз келсе, **Ultimate Multisite → Settings → Payment** баракчасына кирип, инвойст менен байланышкан параметрлерди издеңиз.
