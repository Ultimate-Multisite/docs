# Сайтты Экспорттоо {#site-export}

**Site Export** администратордук баракча аркылуу сезимдерди башкаруучулар бир сайтты же бүтүндөй сетьти миграция, бэкап алуу же өткөрүү иштери үчүн жүктөп алууга мүмкүн болгон архивге салып жиберүүгө болот.

## Бир сайтты экспорттоо {#exporting-one-site}

**Ultimate Multisite > Site Export**-ка кириңиз жана **Generate new Site Export**-у тандаңыз. Экспорттоомду экспорттоочу сайтын тандаңыз, андан кийин архивде жүктөлгөн файлдарды (uploads), плагиндерди (plugins) жана темаларды (themes) камтышы керекпи же жокпуросун тандаңыз.

Экспорттоо бүткөндөн кийин, **Existing Exports** тизмесинен ZIP файлын жүктөп алыңыз. Учурда экспорттагы ZIP файлдары өзүн-өзү иштете турган `index.php` жана `readme.txt`-ти камтыйт, ошондуктан архивди ажыратылган импорт плагинин алдын ала орнобоосуз жаңы хостко жүктөп алып, андан кийин баштоо үчүн колдонууга болот.

## Бүтүндөй сетьти экспорттоо {#exporting-the-whole-network}

Сезимдерде бир гана архивте сетьтеги бардык сайтууларды камтыган бирдиктүү архив керек болгон учурда Site Export баракчасында **Network Export**-у колдонуңуз. Бул хост миграцияларынан мурун, катастрофадан качкандыгын текшерүү жаттоодон же ар бир сайттын биригип жүрүшү зарыл болгон этапта (staging rebuilds) пайдалуу болот.

Сегменттик экспорттон көбүрөөк архив болушу мүмкүн болгондуктан, аны трафики төмөндө турган мезгилде аткарыңыз жана багытталган сактоочу орду жүктөлгөн файлдарды, плагиндерди, темаларды жана түзүлгөн ZIP файлдарын камтуу үчүн жетиштүү бош орун бар экенин текшериңиз.

### Сегменттик импорттогу пакеттер (Network Import Bundles) {#network-import-bundles}

Ultimate Multisite 2.12.0-ден баштап, Site Exporter **network import bundles** — сетьтеги сайттарды жөнөкөйлөтүлгөн түрдө кайра куруу үчүн арналган атайын архивдерди түзө алат. Сегменттик импорт пакети бир нече сайтты жаңы сетьке орнотуу үчүн зарыл болгон бардык файлдарды жана метадеаттарды камтыйт.

#### Сегменттик импорт пакетін түзүү {#generating-a-network-import-bundle}

1. **Ultimate Multisite > Site Export**-катына киришти барыңыз
2. **Generate new Network Export**-ына басыңыз
3. Экспорт түрү катары **Network Import Bundle**-ды тандаңыз
4. Жүкөлгөнүмдөрдү, плагинддерди жана темаларды камтыш керекпи экенин тандоо үчүн
5. **Generate**-ка басыңыз
6. **Existing Exports** тизмесинен пакет ZIP файлын жүктөп алыңыз

#### Network Import Bundleден калыбытуу (Restoring from a Network Import Bundle) {#restoring-from-a-network-import-bundle}

Сайттарды сеть импорт пакетине карата калыбытуу үчүн:

1. Ultimate Multisite-ты максаттуу хостго орнотуңуз
2. мультисайт орнотуучу wizards'уну аяктаңыз
3. Жаңы сетьте **Ultimate Multisite > Site Export**-ка барыңыз
4. **Import Network Bundle**-ына басыңыз
5. Сеть импорт пакеттерин ZIP файлы менен жүктөңүз
6. Экрандагы импорт көрсөтмөлөрүн аткарыңыз
7. Импорт процесси бардык сайттарды, алардын мазмунун жана конфигурацияларын калыбытпайт

Network import bundles төмөнкүлөрдү сактап турат:
- Сайт мазмуну (посттор, беттер, кошумча пост түрлөрү)
- Сайт жөндөмдөрүн жана параметрлерин
- Колдонуучу ролдору жана укуктары
- Плагиндер жана темалар (эгерде пакетте камтылган болсо)
- Медиа жүктөлгөнүшү (эгерде пакетте камтылган болсо)
- Кошумча базалык таблицалар жана маалыматтар

## Өзүн-өзү иштеп чыккан ZIP файлын орнотуу (Installing a self-booting export ZIP) {#installing-a-self-booting-export-zip}

Жаңы хостго өзүн-өзү иштеп чыккан ZIP файлына калыбытуу үчүн:

1. Экспортталган ZIP мазмунун максаттуу веб түшүлүмдөгү köк катапка жүктөңүз.
2. Жүктелген `index.php` файлын браузерде ачыңыз.
3. Пакеттен келген экрандагы орнотуу көрсөтмөлөрүн аткарыңыз.
4. Убактылуу файлдарды алып салуудан мурун, экспортко тиешелүү эскертүүлөрдү камтыган `readme.txt` файлын карап чыгыңыз.

Кошумчалар боюнча орнотуу жана импорттун кылдаттык маалыматтары үчүн [Site Exporter addon documentation](/addons/site-exporter) баарын караңыз.

Ultimate Multisite 2.9.0-да кошулган бир сайттык алаткыч (single-site tool) жөнүндө көбүрөөк маалымат алуу үчүн [Export & Import](/user-guide/administration/export-import)-тушка караңыз.
