# Сайт Билдирме Контакт Формалары {#site-builder-contact-forms}

Superdav AI Agent v1.10.0 сиз Site Builder агенти аркылуу чат интерфейсинен контакт формаларын түзүү мүмкүнчүлүгүн кошот. Бул сизге чаттан чыкпай, кез келген баракка толук иштеп жаткан контакт формаларын кошууну мүмкүндүк берет.

## Жалпы маалымат {#overview}

Site Builder контакт формасын түзүү мүмкүнчүлүгү (`create_contact_form`) орнотулган форма плагиниңизди автоматтык түрдө аныктап, анын киретей мүмкүнчүлүктөрүн колдонуп контакт формасын түзөт. Ден соода болгон форма плагинилердин мисалдары:

- Contact Form 7
- WPForms
- Fluent Forms
- Gravity Forms

Ассистент сиздин сайтыңызда бар эң мыкты форма плагинын автоматтык түрдө тандап, ошол плагинге ылайыкталган контакт формасын түзөт.

## Контакт Формасын Создау {#creating-a-contact-form}

Site Builder аркылуу контакт формасын түзүү үчүн:

1. WordPress администратордунда **Gratis AI Agent** чат панелин ачыңыз.
2. Чат башталышындагы агент иконканы басып, **Site Builder** агентине өтүңүз.
3. Создаamak istediğiniz контакт формасын сүрөттөп бериңиз. Мисалы:

   > "Contact страницыга аты, электрондук почта, билдирүү жана телефон номери үчүн талааларды кошуңуз."

   Же жөн эле:

   > "Contact страницы үчүн контакт формасын түзүңүз."

4. Site Builder контакт формасын түзүп берет жана киргизүүгө даяр болгон шорткодду (shortcode) кайтарып берет.

## Генерацияланган Шорткоду Колдонуу {#using-the-generated-shortcode}

Site Builder контакт формасын түздүгөнүнөн кийин, ал шорткодду (мисалы, `[contact-form-7 id="123"]`) кайтарып берет. Сиз:

1. **Баракка же посту киргизе аласыз** — Шорткодду көчүрүп алып, блок редактору же классикалык редактор аркылуу кез келген баракка же посту чаптаңыз.
2. **Site Builder аркылуу аны кошуңуз** — Site Builder-ге атайын баракка форманы автоматтык түрдө кошууну сураңыз:

   > "Контакт формасын Contact страницуга кошуңуз."

3. **Ол шаблондо колдону** — Эгер сиз PHP менен ыңгайлашып болсоңуз, кыска кодду тема шаблону файлына кошуп койсоңуз болот.

## Контакт формасын өзгөртүү {#customizing-the-contact-form}

Site Builder контакт формасын жараткандан кийин, аны дагы олуттуу өзгөртө аласыз:

### Чат интерфейси аркылуу {#via-the-chat-interface}

Site Builder-ге форманы өзгөртүүнү сураңыз:

> "Контакт формасын тема сапка (subject) талабын кошуу жана билдирүү талабы болушу керек кылуу үчүн жаңылаңыз."

Site Builder форманы жаңылап, жаңыланган shortcode'ду кайтарып берет.

### Форма плагининин административдик интерфейси аркылуу {#via-the-form-plugins-admin-interface}

Сиз да форманы өзүнүн форма плагининин жөндөөлөрүндө түз өзгөртө аласыз:

1. WordPress администратордук панелинде **Contact Form 7** (же орнотулган форма плагиниңизди) ачыңыз.
2. Site Builder тарабынан жаралган форманы табыңыз.
3. Зарыл болгон учурда форма талааларын, валидация эрежелерин жана электрондук билдирүүлөрдү редакциялаңыз.

## Форма плагинилери жана ыңгайлуулук {#form-plugins-and-compatibility}

Site Builder сиздин сайтыңызда кайсы форма плагини орнотулганын автоматтык түрдө аныктап алат жана контакт формасын түзүү үчүн аны колдонуп алат. Эгер бир нече форма плагини орнотулган болсо, Site Builder аларды төмөнкү тартипте сунуштайт:

1. Contact Form 7
2. WPForms
3. Fluent Forms
4. Gravity Forms

Эгер бул плагинилердин эч каймаса орнотулган болсо, Site Builder контакт формасын жасоого мурун аларды орнотууну сунуштайт.

## Электрондук билдирүүлөр {#email-notifications}

Site Builder тарабынан жаралган контакт формалары дефолtdээр сайт администраторуна электрондук билдирүүлөрдү жөнөтүү үчүн конфигурацияланган болот. Сиз алуучу электрондук дарекке жана билдирүү 메시жине өзгөртө аласыз:

1. Форма плагининин административдик интерфейсине барыңыз.
2. Site Builder тарабынан жаралган форманы табыңыз.
3. Электрондук билдирүү жөндөөлөрүн редакциялаңыз.

Толук маалымат үчүн, форма плагиниңиздин документациясына кайрылыңыз:

- [Contact Form 7 Электрондук билдирүүлөр](https://contactform7.com/docs/)
- [WPForms Электрондук билдирүүлөр](https://wpforms.com/docs/)
- [Fluent Forms Электрондук билдирүүлөр](https://fluentforms.com/docs/)
- [Gravity Forms Электрондук билдирүүлөр](https://docs.gravityforms.com/)

## Мисалдардагы колдонуу учурлары (Example Use Cases) {#example-use-cases}

### Жөнөкөй контакт формасы (Simple Contact Form) {#simple-contact-form}

> "Аты, электрондук почта жана каттын тексттери үчүн жөнөкөй контакт формасын түзүңүз."

### Көп кадамдуу форма (Multi-Step Form) {#multi-step-form}

> "Контакт маалыматтары үчүн биринчи кадамды жана билдирүү менен ылайыктуу байланыш ыкмасы үчүн экинчи кадамды бар контакт формасын түзүңүз."

### Шарттуу логикага ээ болгон форма (Form with Conditional Logic) {#form-with-conditional-logic}

> "Төмөнкү менюдан (dropdown) колдонуучунун тандаган бөлүмүнө жараша ар кандай талааларды көрсөткөн контакт формасын түзүңүз."

### Файл жүктөөгө мүмкүнчүлүгү бар форма (Form with File Upload) {#form-with-file-upload}

> "Колдонуучулардын файлды же тиркемени жүктөөгө мүмкүндүк берген контакт формасын түзүңүз."

:::note
Контакт формасын түзүү Superdav AI Agent v1.10.0 жана андан жаңы версияларында жеткиликтүү. Бул мүмкүнчүлүктү колдонуу үчүн Сайт куруучу (Site Builder) агент активдүү болушу керек.
:::
