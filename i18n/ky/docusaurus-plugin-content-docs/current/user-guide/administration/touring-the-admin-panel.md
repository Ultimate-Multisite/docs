# Администратор панелинды арылуу

Ultimate Multisite жөнөкөй жана өзүн түшүндүрүп кала турган система болуп саналат, бирок ушундай күчтүү курал менен аны багытталган турду өткөрүүгө жардам береbilir. Сизге орнонуңузду табууңуздар көмөктөшүү үчүн администратордук саптарга карап көрөлү.

## Dashboard (Башкы экран)

Ultimate Multisite **Dashboard** кирешелер, сайт активдikleri, мүшендик өсүшү, кызматкерлер саны жана географиялык маалымат сыяктуу негизги отчетторду жана аналитиканы көрсөтөт.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

Бул дашбордун толук көзөлгөнү:

![Ultimate Multisite Dashboard full page](/img/admin/um-dashboard-full.png)

Ал эми дашбордун төмөнкү бөлүгү:

![Ultimate Multisite Dashboard bottom section](/img/admin/um-dashboard-bottom.png)

Бул жерде тармак дашбордун толук көзөлгөнү да бар:

![Network Dashboard full page](/img/admin/network-dashboard-full-page.png)

## Checkout Forms (Сырткы төлөм формалары)

**Checkout Forms** сизге өзүнчө каттоо саптарын түзүү үчүн ыңгайлуулук берет. Сиз талааларды кастомалап, өзүңүздүн класстарыңызды жана скрипттериңизди киргизе аласыз, ошондой эле кирип жаткан өлкө же аймакка негизделген чектөөлөрдү коюшуңузга болот.

![Checkout Forms list](/img/admin/checkout-forms-list.png)

## Products (Өнүмдөр)

**Products** бөлүмү — тармагыңыз үчүн ар кандай пландарды, пакеттерди жана кызматтарды түзүүчү жер. Бааларды, төлөөдөгү жиырыктарды жана чектөөлөрдү/квоталарды өнүм деңгээлинде аныктасаңыз болот.

![Products list](/img/admin/products-list.png)

## Memberships (Мүшендик мамилелери)

**Memberships** баракча тармагыңыздагы бардык жазылууларды көрсөтөт. Пландарды, өнүмдөрдү, төлөө мөөнөттөрүн жана жиырыктарды, төлөм тарыхын жана убакыт белгилерин камтыган жазылуу деталдарын карап чыгып, редакцияласаңыз болот.

![Memberships list](/img/admin/memberships-list.png)

## Payments (Төлөмдөр)

**Tözoqlar** барак ар бир тармагыңыздагы төлөм тарыхын тез карап көрүүгө мүмкүндүк берет; анын ичинде конкреттүү продукт жана суммалар камтылган кызматтар жөнүндө толук транзакция маалыматтары болот.

![Payments list](/img/admin/payments-list.png)

## Кардарлар

**Кардарлар** барак ар бир тармакка катталган бардык мүчөлөрдү көрсөтөт, алардын аталышы, электрондук почта дареги, акыркы кириш తేదీ жана алардын поддиш сайтына кирүү үчүн "кетирүү" (switch to) шилтемеси сыяктуу негизги колдонуучу маалыматтары камтылган.

![Customers list](/img/admin/customers-list.png)

## Сайттар

**Сайттар** барак ар бир тармакка кирген бардык поддиш сайттарды, алардын сайт шаблондорун жана кардарлар тарабынан ээ колдонулган сайттарды тизмелейт. Мапталган домендерди, чектөөлөрдү жана квоталарды, плагиндерди жана темаларды, ошондой эле сайттын миниатюраларын оңой башкара аласыз.

![Sites list](/img/admin/sites-list.png)

## Домендер

**Домендер** барак поддиш сайттарға мапталган өзүнчө домендерге арналган. Супер администратор катары сиз өзүңүз колдонулган домендик аталыштарды кошо аласыз же мапталсаңыз, DNS каттарын жана журналдарды көрө аласыз.

![Domains list](/img/admin/domains-list.png)

## Арзандатуу коддору (Discount Codes)

**Арзандатуу коддору** сиздин акыркы колдонуучуларыңызга арзандатууларды сунуштоо үчүн купон коддорун түзүүгө мүмкүндүк берет. Арзандатуунун баасын көрсөтүп, атайын колдонуучуларды же топторду аныктай аласыз.

![Discount Codes list](/img/admin/discount-codes-list.png)

## Жарнамалар (Broadcasts)

**Жарнамалар** — поддиш сайттын колдонуучуларына билдирүүлөрдү жайылтуу же жеке каттар göndermek үчүн курал. Алардын жазылган планга же продуктуна негизделген атайын колдонуучу топторду багыттай аласыз.

![Broadcasts list](/img/admin/broadcasts-list.png)

## Жооптор (Settings)

**Жооптор** барак — Ultimate Multisite'ти конфигурациялоо үчүн жер. каттоо 설정дору, төлөмдөр, API жана webhooks, домен маппасы жана башка интеграциялар камтылган.

AI коннектор жогорку колдонулган OAuth провайдер пулдары тизмесин көрсөтөт: Anthropic Max, OpenAI ChatGPT/Codex жана Google AI Pro. Ар бир провайдер картасы супер администраторлорго аккаунттарды туузуу, сакталып калган аккаунттарды жаңыртуу, электрондук почта аркылуу аккаунттарды алып салуу жана браузердин багыты бөгөдөндө сыноочу чөйрөнү токтотсогу үчүн колдонулуп турган колдонуучу OAuth резервди пайдалануу мүмкүнчүлүгүн берет. Cursor Pro жөндөөлөр администратордук панелден алынып салынды.

ChatGPT/Codex аккаунттары коннекторго колдоо көрсөткөн куралдарды колдонууга жардам берет, анда операцияларга куралдар ушу мүмкүн болсо, OpenAI аккаунту туузулгандан кийин Codex куралдын жүрүм-турумун колдонуу үчүн администратордук иш процесстери иштеп чыгууга болот.

## Окунул алар (Events)

**Окунул алар (Events)** баракчаңыздагы бардык окуяларды жана журналдарды сактап турат. Ал пландын өзгөрүшү, каттоодон өтүү жана башка кыймылдар сыяктуу иш-аракеттерди излейт — бул сиздин мультисайтты көзөмөлдөө үчүн пайдалуу.

![Events list](/img/admin/events-list.png)

## Вебхуктар (Webhooks)

**Вебхуктар (Webhooks)** сизге маалыматты башка программага жөнөтүүгө мүмкүндүк берет. Ultimate Multisiteдан Zapier сыяктуу платформаларга маалымат жөнөтүү үчүн пайдалуу.

![Webhooks list](/img/admin/webhooks-list.png)
