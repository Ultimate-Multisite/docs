# Ойрон: Zapier аркылуу аккаунт каттоо {#event-register-an-account-via-zapier}

[Ultimate Multisite-ти Zapier менен интеграциялоо] макаарясында биз Zapierди Ultimate Multisite ичиндеги ар кандай иш-аракеттерди триггерлер жана окуялар негизинде кантип колдонуп көрүү керектигин талкуулак. Бул макаланда биз үчүнчү тараптын тиркемелерди (3rd party applications) кантип интеграциялоону көрсөтөбүз. Биз Google Sheets-ти маалымат булагы катары колдонуп, Ultimate Multisiteге аккаунт каттоо үчүн маалымат жөнөтөбүз.

Zapier менен туташуудан мурун, **Ultimate Multisite > Settings > API & Webhooks** бөлүмүнө барыңыз жана API-nin активдүү экенин текшериңиз. Zapier Ultimate Multisite аккаунтунун кирүү маалыматтарын сураганда API Key (API Ачкычы) жана API Secret (API Сыктыгыч) сыяктуу нерселерди бул экрандан көчүрүп алыңыз.

![API иштетүү жана Webhooks жөндөөлөрү, анда API Ачкычы, API Сыктыгычы жана API-ни активдүү кылуу опциялары көрсөтүлгөн](/img/admin/settings-api-webhooks.png)

Биринчиден, Google Drive'де **Google Sheet** түзүшүңүз керек. Маалыматтарды кийинчерээк оңой карталоо үчүн ар бир колонканы туура аныктап коюуну унутпаңыз.

Google sheet-ти түздүргөндөн sonra, Zapier аккаунтуңузга кирип, zap түзүүнү баштасаңыз болот.

**"App event"** (Тиркеме окуясы) издөө талаасында **"Google Sheets"** (Google Таблицалар) тандаңыз.

Андан соң, "**Event**" (Окуя) талабына **"New spreadsheet row"** (Жаңы таблица сапы) тандап, "**Continue**" (Уланууга өтүү) басыңыз.

Кийинки кадамда сиз маалыматтар келе турган **Google Sheet**-ти тандашыңыз керек. Демек, туура Google аккаунту көрсөтүлүп жатканын текшериңиз.

**"Set up trigger"** (Триггерди орнотуу) талабы астында, маалыматтардын келе турган Google spreadsheet-ти жана worksheet'тин тандап белгилөшү керек. Ошол жерде маалыматтар кайдан келет экенин көрсөтүңүз. Аларды толтуруп, "**Continue**" басыңыз.

Кийинки кадам — **"test your trigger"** (триггерди сынап көргөнүңүз) — Google sheet'тин туура байланышканын текшерүү үчүн керек.

Эгер сиздин тест ийгиликтүү болсо, сиз Google spreadsheet'териңизден бир нече маанилер көрсөтүлгөн жыйынтыкты көрөсүз. Улантуу үчүн "**Continue**" басыңыз.

Кийинки кадам — Ultimate Multisite-та аккаунтду түзүү же каттоодон өткөрүү үчүн экинчи аракетти орнотуу. Издеу талаасында "**Ultimate Multisite(2.0.2)**" дегенди тандаңыз.

"**Event**" талаасынын астында, "**Register an Account in Ultimate Multisite**" (Ultimate Multisite-та аккаунт каттоодон өтүү) дегенди тандап, андан кийин "**Continue**" (Улантуу) баскычын басыңыз.

"**Set up an action**" (Аракетти орнотуу) бөлүмүндө сиз кардар маалыматтары, мүчөлүктөр, товарлар ж.б. үчүн ар кандай талааларды көрөсүз. Сиз бул маанилерди өз Google Sheet-иңиздеги талааларга карталап, аларды скриншотто көрсөтүлгөн туура талаага таштасаңыз болот.

Маанилерди карталгандан кийин, сиз аракетти текшере аласыз.
