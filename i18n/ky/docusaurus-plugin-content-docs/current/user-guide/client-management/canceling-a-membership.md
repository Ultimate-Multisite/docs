# Чыгармалыкты (Membership) Абаюуну Жоюу (v2) {#canceling-a-membership-v2}

_**МААЛЫЙ ЭСАС КӨРҮЗҮ: Бул макала Ultimate Multisite 2.x версиясына тиешелүү.**_

Кардарларыңыздын тарабынан чыгармалыкты жоюу өтө оңой.

Эгерде колдонуучу өзүнүн чыгармалыгын жоюп келсе, ал биринчи кезекте алардын ****Аккаунт**** баатында кирүүсү керек.

Ошол жерден алар **Кыйын аймакка (Danger Zone)** басуу жана **Аккаунтту Жоюу** дегенге басуу керек.

![Account page Danger Zone with Delete Account button](/img/account-page/danger-zone.png)

Модуль ачылып, алардын тастыктоосун сурап турат. Алар **Аккаунтты Жоюуну Тастыктоо** опциясын күйгүзүп (toggle) жана **Аккаунтту Жоюу** дегенге басуу керек.

![Confirm Account Deletion dialog](/img/account-page/delete-account-confirm.png)

Бул аларды сайттан кирип чыгарат жана сиздин тармагыңыздагы чыгармалыгын жоयो, ошондой эле алардын сайтынын сиздин тармагыңыздан өчүрүлөт.

![Logged out after account deletion](/img/account-page/logged-out-after-delete.png)

![Membership canceled confirmation](/img/account-page/membership-canceled.png)

**Маанилүү:** Бул аракет кардарды сиздин тармагыңыздан алып салуучу эмес экенин байлаңыз, анткени кардардын бирден ар кандай чыгармалыгы же ага тиешелүү чыгармалык жок болушу мүмкүн.

![Customer still exists in the network after cancellation](/img/admin/customer-after-cancel.png)

![Customer with no active memberships](/img/admin/memberships-customer-empty.png)
