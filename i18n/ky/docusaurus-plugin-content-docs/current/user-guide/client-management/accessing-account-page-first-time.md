# Биринчи кезекте аккаунтума кантип кире алам? {#how-can-i-access-my-account-page-for-the-first-time}

Ultimate Multisite-тагыраңыз!

Биздин плагинине сатып алгандан кийин сиз кабыл алган электрондук катта болгон шилтеме аркылуу аккаунтуңузга кирсе болот:

<!-- Скриншот жок: Аккаунт үчүн ээлик шилтеме бар электрондук кат -->

Сатып алууда колдонулган электрондук дарегиңизди жана электрондук катта алган паролу колдонуңуз.

<!-- Скриншот жок: Электрондук дарек жана пароль талап кылынган кириш бет -->

Бул аккаунтуна биринчи жолу кирсеңиз, сиз паролуңузду өзгөртүүнү суралат. Өзгөртүп, _Save_ баскычын басыңыз.

<!-- Скриншот жок: Паролду өзгөртүү формасы жана Save баскычы -->

Аккаунт үчүн кирип жатканда сиз төлөмдөр, жаңыртуулар, калыбына келтирүү (renewals), жокко чыгаруу жана башка көп нерселер сыяктуу бардык аккаунт маалыматтарын башкара аласыз.
