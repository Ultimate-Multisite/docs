# Клиенттердин веб-сайттарыңыз (v2)

Сетьке каттангынан кирип алгандан кийин, купуялар өзүнүн дашбордуна кирүүгө мүмкүнчүлүк алышат.

Маанилүү төлөмдөр, мүчөлүктөр, домендер, план чектөөлөрү жана башка нерселер жөнүндө маалыматтарды көрө турган **Account** баракчадан тышкары, алар бардык веб-сайттарын көрө жана башкарууга мүмкүнчүлүк берген **My Sites** баракчасына да кире алышат. Алар жөн гана дашбордундагы My Sites дегенге басышы керек.

![Купуянын дашбордундагы My Sites шилтемеси](/img/account-page/my-sites-link.png)

Алардын бардык веб-сайттары көрсөтүлөт жана ар биринин астында **Manage** баскычы болот. Эгер алар ошону баса берсе, алар ошол конкреттүү веб-сайтын **dashboard**ундагига жөнөтүлөт.

Ошондой эле, планлары бирденден көп веб-сайттарды колдонууга мүмкүндүк берген учурда, кайсы веб-сайты **primary** (башкы) экенин көрө алышат.

![My Sites баракчасында купуянын веб-сайттары жана Manage баскычтары көрсөтүлгөн](/img/account-page/my-sites-page.png)

Бул сайттын редакциялоо интерфейсинин толук көзү:

![Сайтты редакциялоонун толук интерфейси](/img/admin/site-edit-full.png)

Эгер алар башкы веб-сайтын өзгөрткүп келсе, алар **Account** менюсуна басып, Account баракчасына барышы керек.

![Купуянын дашбордундагы Account менюсу шилтемеси](/img/account-page/account-menu.png)

Алар купуялардын маалыматтарын да көрө жана редакцияласа болот:

![Купуяны редакциялоо интерфейси](/img/admin/customer-edit.png)

Андан кийин, алар **Change Default Site** (Баштапкы сайтты өзгөртүү) дегенге басушу керек.

![Account баракчасында Change Default Site баскычы](/img/account-page/change-default-site.png)

Чыгармалы терезе ачылат. Төмөнкү менюдан тандаңыз жана кайсы веб-сайты башкы сайт кылууну кааласаңыз, аны тандаңыз.

![Change Default Site 드롭다운 тандоо](/img/account-page/change-default-site-dropdown.png)

Андан кийин, жөн гана **Change Default Site** баскычын басыңыз жана сиз аяктадыңыз!

![Стандарттык сайтты өзгөртүүнү тастыктоо](/img/account-page/change-default-site-confirm.png)
