# Жардык кантип жардам алуу

Сиздин күчтүү Лицензиялык ключ бар жана ал биздин командага маселелерди чечүүгө жардам берүү үчүн жеткиликтүүлүктү берип турат, анда Ultimate Multisite'ты орнотуу учурунда келетген мүмкүн болгон көйгөйлөрдөн.

Бул сабак лицензия ээлери үчүн жеткиликтүү колдоо каналдарын жана биздин колдоо командасы менен байланышуу үчүн даяр болушу керек болгон маалыматтардын кандай түрлөр керектиги жөнүндө маалымат берет.

## Жардам сайты {#help-site}

Сиз астындагы шилтемеде таба турган жардам сайтыбыз ар кандай макалаларды жана сабактарды колдоно аласыз.

[)

Биз колдонуучуларга Ultimate Multisite'тын негизги функциялары жана иш-ализациялары менен таанышып алууну сунуштайбыз. Жардам сайтыбызда "Башталыш" макалалары бар жана анда колун жибергенде кандайдыр бир плагинди кантип орнотуу керек, төлөм багыттарын интеграциялоо сыяктуу жалпы суроолорго жооптор да табылат.

## Колдоого кайрылуу {#contact-support}

Эгер сиз маселелер менен токтоп калсаңыз жана жардам сайтыбызда жообу таба албасаңыз, сиз биздин колдоо командасына сайттын оң жагындагы төмөнкү шилтеме иконкасын басуу аркылуу же түз эле support@ultimatemultisite.com дарегине электрондук кат жөнөтүү аркылуу кайрыла аласыз.

Техникалык маселелер боюнча колдоо менен байланыша алганда, сурооңузду так сүрөттөп бергенде, ошол убакта сиздин маселени оңой түшүнүп баалоо жана чечүү үчүн биздин колдоо адамдарына жардам бере турган скриншотторду, ката билдирүүлөрүн же башка бардык маалыматтарды камсыз кылыңыз. Ошондой эле лицензиялык ключуңуз да бериши керек. Гана активдүү лицензия ээлери колдоого укуктуу.

## Коомдук сайт {#community-site}

Биздин [community site](https://community.ultimatemultisite.com/) Ultimate Multisite колдонуучулары өз ара аракеттениши үчүн жер бети. Ал мүчөлөрдүн талкууларга жазып, комментарий калтырууга, кеңеш берүүгө, кызматташууну же коллективдүү булактан алууга мүмкүнчүлүк берет. Жашыруун аймактарды/баракчаларды ачуу үчүн Ultimate Multisite лицензиялык ключуңузду беришиңиз талап кылынат.
