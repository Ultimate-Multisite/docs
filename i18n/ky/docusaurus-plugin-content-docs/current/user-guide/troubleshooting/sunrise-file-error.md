# Sunrise Файлын Орнотууданда Катаалган

sunrise.php файлы — бул WordPress өзүн иштеп жатканда издөөчү атайын файл. WordPress `sunrise.php` файлын таануу үчүн ал **wp-content папкасынын** ичинде болушу керек.

Ultimate Multisite-ты активдештиргенде жана скриншотто көрсөтүлгөн сыяктуу орнотуучу программаларды өткөрө алганда, Ultimate Multisite биздин `sunrise.php` файлын wp-content папкасына көчүрүүгө аракет кылат.

<!-- Скриншот жок: sunrise.php орнотушун көрсөткөн орнотуучу программанын саадаты -->

Көбүнчө учурларда биз файлды ийгиликтүү көчүрө алабыз жана ар бир нерсе туура иштеп чыгат. Бирок, эгерде кандайдыр бир нерсе туура коюлбаса (мисалы, папканын укуктары), Ultimate Multisite файлды көчүрө албаган учурларга туш болушуңуз мүмкүн.

Эгерде Ultimo сизге берген ката билдирүүсүн окусаңыз, бул жерде так эмне болгонун көрөсүз: **Sunrise көчүрмө ийгиликтүү болгон эмес**.

<!-- Скриншот жок: Sunrise көчүрмө ийгиликтүү болбоону көрсөткөн ката билдирүү -->

Бул маселени чечүү үчүн, жөн гана `wp-ultimo` плагининин папкасынын ичиндеги `sunrise.php` файлын көчүрүп алсаңыз жана аны өзүңүздүн wp-content папкасына чаптасаңыз болот. Бул ишке ашырылгандан кийин, орнотуучу программанын саадаты менен кайра жүктөө керек жана текшерүү ийгиликтүү болушу керек.

<!-- Скриншот жок: wp-ultimo плагининин папкасындагы sunrise.php файлын көрсөткөн файл менеджеры --> Кандайдыр бир учурда, келечекте маселелердин туулушун болбогон үчүн (Ultimate Multisite менен гана эмес, башка плагиндер жана темалар менен да) папка укуктарын жалпы текшерүү зарыл болушу мүмкүн.

WordPressтин өзүнүн бөлүгү болгон **Health Check tool** (бул сиз негизги сайтыңыздын **admin panel > Tools > Health Check** аркылуу жеткире аласыз), папка укуктары WordPress менен маселелерди алып келиши мүмкүн болгон бааларга коюлуп электигин билүүгө мүмкүндүк берет.
