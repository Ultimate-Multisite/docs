# Башкаруучу плагинддер менен белгилүү кескиндуктар

Ultimate Multisite-ты орнотуп, активдештиргенден кийин, башка плагиндер менен чечилбеген кескиндуктардан таасирленсеңиз болот.

Биз ар бир жаңы релисте бул тизмеди азайтуу боюнча активдүү иштеп жатабыз, бирок эгерде плагин бул жерде тизмеде болсо, толук иштей турган Ultimate Multisite Network-уздан болушу үчүн аны өчүрүшүңүз керек болушу мүмкүн.

Бул тизме жана андай учурларда эмне кылышы керектиги жөнүндө маалымат:

**Кээ бир WPMU DEV плагиндерди** WPMU DEV тарабынан **азыр колдонулуп келбеген** кээ бир плагиндер Ultimate Multisite каттоо процессинин акыркы кадамында (колдонуучу аккаунт түзө албайт) токтоп калууга алып келиши белгилүү. Бул маселелерди жаратуучу плагиндердин тизмеси төмөнкүлөр:

  * Signup Language: _Ultimate Multisite-та ошол функцияны камтыган Ultimate Multisite: Language Switcher деген кошумча (add-on) бар._

  * Set Blog Description on Blog Creation: _Бул Ultimate Multisite-ка киргизилген, ошондуктан кошумча плагинду колдонуу зарыл эмес._

  * New Blog Templates: _Бул Ultimate Multisite-ка киргизилген, ошондуктан кошумча плагинду колдонуу зарыл эмес._

  * Set Password on Multisite Blog Creation: _Бул Ultimate Multisite-ка киргизилген, ошондуктан кошумча плагинду колдонуу зарыл эмес._

  * Signup TOS: _Бул Ultimate Multisite-ка киргизилген, ошондуктан кошумча плагинду колдонуу зарыл эмес._

  * Anti-Splog: Ал артыкчылыктуулугу азыр, бул да _Ultimate Multisite-ка киргизилген._

**Эскертүү**: Башка бардык WPMU Dev плагиндерди Ultimate Multisite менен нормалдуу колдонууга болот. Мисалдар катары _Smush_, _Forminator_, _Defender_ ж.б. болушу мүмкүн.
