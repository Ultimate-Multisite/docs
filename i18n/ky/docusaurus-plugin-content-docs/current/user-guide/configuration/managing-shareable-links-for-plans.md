# Пландар үчүн бөлүшүү мүмкүнчүлүктөрүн башкаруу {#managing-shareable-links-for-plans}

Ultimate Multisite сизге планリンクтерин түздөн-түз кардарларыңыз менен бөлүшүү мүмкүнчүлүгүн сунуштайт. Бул үчүн, пландарыңыздын редакциялоо сапка кирип, _Shareable Linkти көчүрүү_ баскычын басыңыз.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_Пландын редакциялоо сапка бөлүшүү мүмкүнчүлүктөрүн түзүү үчүн баскычту таба аласыз_

Бул механизмдинリンクти түзүүдө дефолт төлөм жиктерди колдонот экенин байкаңыз. Бул дегенде, мисалы, сиздин дефолт төлөм ай сайын болсо, аягында сизде мындай шилтеме болот: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Эгер сиз бөлүшүүчү шилтемени кварталдык болуп дефолт кылууну кааласаңыз, аягына 3 санын кошуп беришиңиз керек:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Эгер сиз _premium_ планын жылдык график боюнча тандоо үчүн, 1 санынан соң сөзү кошуп беришиңиз керек:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Кечиресиз, ар кандай төлөм жиктер үчүн бир гана шилтеме болушу мүмкүн эмес. Ар бир төлөм жиктер үчүн бир гана атайын шилтемени бөлүшүңүз керек.

Шилтемени кардарларыңыз менен бөлүштүргенден кийин, ал аларды сиздин каттоо процессиңиздин биринчи кадамына багыттайт.
