# Темага Жакотка Адаптивдүү Көндүмдөр {#theme-aware-skills}

Superdav AI Agent v1.10.0 төрт жаңы тема-адаптивдүү киргизүүчү көндүмдөрдү (skills) менен келет, алар сиздин активдүү WordPress темаңызга автоматтык түрдө ылайыкташат. Бул көндүмдөр сиздин темаңыздын архитектурасына жана функцияларына ылайыкталган атайын багыт берүүчү жана мүмкүнчүлүктөрдү сунуштайт.

## Темага Адаптивдүү Көндүмдөр деген эмне? {#what-are-theme-aware-skills}

Темага адаптивдүү көндүмдөр — бул AI жардамчысы сиздин сайтыңызда учурда активдүү болгон WordPress темасына негизделген, ал автоматтык түрдө тандайт. Темаларды алмаштырсаңыз, жардамчынын жеткиликтүү көндүмдөрү автоматтык түрдө жаңыланып турат — колдонуучу тарабынан эч кандай кошумча конфигурация талап кылынбайт.

Ар бир көндүмдө төмөнкүлөр камтылган:

- **Темага тиешелүү документтер** — теманы колдонуу жана аны ылайыкташтыруу боюнча багыттар
- **Блок жана шаблон (pattern) маалыматтары** — жеткиликтүү блоктар, шаблондор жана дизайн опциялары
- **Ылайыкташтыруу мисалдары** — кеңири таралган тапшырмалар үчүн код бөлүктөрү жана конфигурация үлгүлөрү
- **Ички иш-ализация (Best practices)** — теманын архитектурасы жана жумуш процесси боюнча сунушталган нерселер

## Жеткиликтүү Темага Адаптивдүү Көндүмдөр {#available-theme-aware-skills}

### Блок Темалар (Block Themes) {#block-themes}

**Кимге тиешелүү:** WordPress блок-негизиндеги (Full Site Editing) архитектурадан колдонуучу темаларга.

Блок Темалар көндүмү төмөнкүлөрдү көрсөтөт:

- Блок редакторунун аркылуу шаблондорду түзүү жана редакциялоо
- Блок шаблондору жана кайра колдонууга мүмкүн болгон блоктар менен иштөө
- Женерал стилдерди жана theme.json жөндөмдөрүн ылайыкташтыруу
- Тема блоктарын жана алардын варианттарын колдонуу
- Сайт үчүн өзүңүздүн блок шаблондорун түзүү

**Автоматтык түрдө активдештирилет:** Активдүү темаңыз блок тема болсо (блок-шаблон функциясын колдойт).

### Классикалык Темалар (Classic Themes) {#classic-themes}

**Кимге тиешелүү:** PHP шаблондорун жана классикалык редакторду колдонуучу адаттагы WordPress темаларына.

Классикалык Темалар көндүмү төмөнкүлөрдү көрсөтөт:

PHP шаблон файлдары жана хуктар менен иштөө
Теманын көрүнүшүн Customizer аркылуу өзгөртүү
Виджет аймактарын жана жанындагы панелдарды колдонуу
CSS-ти жана чачылган теманы (child theme) иштеп чыгуу
Тема иерархиясын жана шаблон тегдерин түшүнүү

**Автоматтык түрдө активдүү болот:** Активдүү темаңыз классикалык (блок эмес) тема болгон учурда.

### Kadence Blocks {#kadence-blocks}

**Колдонулат:** Кесиптик блок-негизделген дизайн үчүн Kadence Blocks плагининен фойдалануучу сайттарда.

Kadence Blocks көндүмдөр төмөнкүлөрдү көрсөтөт:

- Kadencenдин артыкчылыктуу блок китебинен (Hero, Testimonials, Pricing ж.б.) колдонуу
- Kadence блокторунун параметрлерин жана адаптивдүү опцияларын конфигурациялоо
- Kadence'дин сеткасы (grid) жана контейнер блоктору менен кастомдык макеттерди куруу
- Kadence блокторун темаңыз менен интеграциялоо
- Kadencenдин дизайн системасын жана пресеттерин пайдалануу

**Автоматтык түрдө активдүү болот:** Сайтта Kadence Blocks плагини активдүү болгон учурда.

### Kadence Theme {#kadence-theme}

**Колдонулат:** Блок-негизделген дизайн жана кастомизация үчүн Kadence темасын колдонуп жаткан сайттарда.

Kadence Theme көндүмдөр төмөнкүлөрдү көрсөтөт:

- Жолугучу стилдер (global styles) жана theme.json аркылуу Kadence темасын кастомизациялоо
- Kadencenдин киргизилген блок моделдери жана шаблондорун колдонуу
- Kadence темасынын параметрлерин жана опцияларын конфигурациялоо
- Kadencenдин дизайн системасы менен кастомдык дизайн куруу
- Kadencенди популярдуу плагиндер жана куралдар менен интеграциялоо

**Автоматтык түрдө активдүү болот:** Kadence темаңыз активдүү тема болгон учурда.

## Көндүмдөр кантип тандалат {#how-skills-are-selected}

Ассистент ар бир билдирүүдө сиздин активдүү темаңызды жана орнотулган плагиниңизди автоматтык түрдө аныктайт. Эгерде дал келген теманы тааныган көндүм бар болсо, ал ассистенттин контекстине автоматтык түрдө жүктөлөт. Көндүмдөрдү колдонуп же алмаштыруу үчүн колуңизди колдобоңуз.

### Көптүрк көндүмдөр {#multiple-skills}

Эгер сиздин сайтыңызга бир нече көндүмдөр (мисалы, эгерде сизде Kadence Blocks жана Kadence Theme активдүү болсо) колдонулуп жатса, ассистент бардык тиешелүү көндүмдөргө жеткиет жана ар биринен кеңештерди карап чыгышы мүмкүн.

### Темаларды өзгөртүү {#switching-themes}

Активдүү темаңызды өзгөртсөңүз, ассистентин жеткиликтүү көндүмдөрү кийинки билдирүүдө автоматтык түрдө жаңыланып турат. Мисалы:

1. Сиз **Block Themes** (Блок темалары) көндүмү активдүү болгон блок темасын колдонуп жатасыз.
2. Сиз классикалык теманы колдоносуз.
3. Кийинки билдирүүңүздө, **Classic Themes** (Классикалык темалар) көндүмү автоматтык түрдө жүктөөчү болот жана **Block Themes** көндүмү дагы жеткиликтүү болбойт.

## Теманы билип колдонуу {#using-theme-aware-skills}

Теманы билип колдонуучу көндүмдөрдөн пайдалануу үчүн, жөн гана чат интерфейсинде эмне кылышыңыз керектигин сүрөттөп бериңиз. Ассистент автоматтык түрдө тиешелүү көндүмдүн кеңешине кайрылат.

### Мисалдардагы суроолор {#example-prompts}

**Block Themes үчүн:**
> "Блок моделдерин колдонуп, фондук сүрөтү жана борборго жазылган текст менен герой секциясын түзүңүз."

**Classic Themes үчүн:**
> "Бала теманы колдонуп, сайдбарга кошумча виджет аймагын кошуңуз."

**Kadence Blocks үчүн:**
> "Kadence Testimonials блогун колдонуп, баалуулуктар секциясын куруңуз."

**Kadence Theme үчүн:**
> "Башкы бөлүктүн (header) сыяктуу дизейнды жана навигация менюсунун стилин өзгөртүңүз."

Ассистент активдүү темаңызга жана плагиндерге ылайыкталган тема-спецификалык кеңештерди жана код мисалдарына жооп берет.

:::note
Теманы билип колдонуучу көндүмдөр Superdav AI Agent v1.10.0 жана андан жаңы версияларында автоматтык түрдө жеткиликтүү болот. Эч кандай кошумча даярдоо же конфигурация талап кылынбайт.
:::
