# Интернет издөө

Gratis AI Agent v1.5.0-га сөздөрүнүн жүрүшүндө ИИ ассистендин интернеттен актуалдуу маалыматтарды алуусун мүмкүнчүлүк берген **Интернет издөө** (Internet Search) функциясын кошот. Бул [Brave Search API](https://brave.com/search/api/) же [Tavily API](https://tavily.com/) аркылуу ишке ашырылган.

## Иш кандай иштейт? {#how-it-works}

Интернет издөө күчөтүлгөн болсо, ассистент суроо учурдагы же сырткы маалыматтарды талап кылганын аныктаган учурда сиз жөнгө салгандай колдонулган издөө көрсөтүүчүлөрүн автоматтык түрдө сурашы мүмкүн — мисалы, акыркы жаңылыктар, учурдагы баалар же моделдин окутуудан кийин өзгөрүп калышы мүмкүн болгон документтер.

Натыйжалар реалдуу убакта алынып, ассистент жооп берүүдөн мурун алдын ала контекстке кошулат. Ассистент суроого жооп берүү үчүн издөө натыйжаларын колдонуп жатканын билдирет.

## Интернет издөөнү күчөтүү {#enabling-internet-search}

Интернет издөө сиз тандаган издөө көрсөтүүчүлөрдөн API ключу талап кылат. аны конфигурациялоо үчүн:

1. **Gratis AI Agent → Settings → Advanced**-га кириңиз.
2. **Internet Search Provider** (Интернет издөө көрсөтүүчүсү) түшмөсүн таап, андан **Brave Search** же **Tavily**-ди тандаңыз.
3. API ключуңузду тиешелүү талаага киргизиңиз. Ар бир талаанын жанында каттоодон өтүү (Sign-up) шилтемелери көрсөтүлөт.
4. **Save Settings** (Сырткы параметрлерди сактоо) баскыңыз.

Ключ сакталгандан кийин, ассистентке Интернет издөө мүмкүнчүлүгү автоматтык түрдө жеткиликтүү болот.

## Brave Search {#brave-search}

### Brave Search API ключу алуу {#obtaining-a-brave-search-api-key}

1. [Brave Search API бетине](https://brave.com/search/api/) барыңыз.
2. Планга катталыңыз. Ай сайын суроолор саны чектөө менен акысыз деңгээл бар.
3. Brave Search өнөктөсүнүн (developer dashboard) аркылуу API ключуңузду көчүрүңүз.
4. Аны Gratis AI Agent параметрлерине кирген **Brave Search API Key** талаасына чаптаңыз.

### Колдонуу чектөөлөрү {#usage-limits}

Использование оплачивается Brave Search-ом на основе количества сделанных запросов. Каждое ответное сообщение ИИ, которое вызывает поиск, считается одним запросом. Следите за своим использованием в [панели разработчика Brave Search](https://brave.com/search/api/), чтобы избежать неожиданных списаний.

## Tavily {#tavily}

Superdav AI Agent v1.10.0 добавляет **Tavily** как альтернативный провайдер интернет-поиска, предлагая более богатые результаты поиска и расширенные возможности для исследований.

### Получение API-ключа Tavily {#obtaining-a-tavily-api-key}

1. Перейдите на [страницу API Tavily](https://tavily.com/).
2. Зарегистрируйтесь. Доступен бесплатный уровень с лимитом запросов в месяц.
3. Скопируйте свой API-ключ из панели управления Tavily.
4. Вставьте его в поле **Tavily API Key** в настройках Gratis AI Agent.

### Лимиты использования {#usage-limits-1}

Использование оплачивается Tavily на основе количества сделанных вызовов API. Каждое ответное сообщение ИИ, которое вызывает поиск, считается одним вызовом. Следите за своим использованием в [панели управления Tavily](https://tavily.com/), чтобы избежать неожиданных списаний.

## Отключение интернет-поиска {#disabling-internet-search}

Удалите API-ключ из поля активного провайдера поиска и сохраните изменения. Возможность поиска в интернете больше не будет предлагаться ассистенту.

:::note
Поиск в интернете добавляет задержку ответам, потому что ассистент должен дождаться результатов поиска перед генерацией ответа. Для задач, требующих высокой скорости, подумайте, нужен ли поиск в реальном времени или достаточно встроенных знаний ассистента.
:::
