# Камтылган Agents

Gratis AI Agent v1.9.0 беш камтылган agent менен келет; ар бири белгилүү бир tools топтому, ылайыкташтырылган system prompt жана ошол тармактагы жалпы тапшырмаларга туура келген баштапкы сунуштар менен алдын ала конфигурацияланган. Agentтердин ортосунда которулуу жардамчы эмне кыла аларын жана кантип жооп берерин өзгөртөт — сиз тараптан эч кандай конфигурациясыз. Superdav AI Agent v1.18.0 тиешелүү интеграциялар конфигурацияланганда бул иш агымдарына графикти эске алган tools, эскертме жазуулары, бекитүү дарбазалары жана SMS билдирмелерин кошо алат.

## Agent деген эмне? {#what-is-an-agent}

Ар бир agent төмөнкүлөрдү бириктирген аталыштуу конфигурация профили:

- **Tools** — agent чакырууга уруксат берилген жөндөмдөр (мисалы, Content Writer пост түзүү жөндөмдөрүнө кире алат; Design Studio CSS жана theme.json жөндөмдөрүнө кире алат)
- **System prompt** — agentтин тону, артыкчылыктары жана чектөөлөрүн белгилеген көрсөтмөлөр
- **Сунуштар** — тез баштоого жардам берүү үчүн чат интерфейсинде көрсөтүлгөн алдын ала жазылган promptтор

## Agent Pickerге кирүү {#accessing-the-agent-picker}

1. WordPress админ каптал тилкесинен **Gratis AI Agent** панелин ачыңыз.
2. Чат башкы бөлүгүнүн жогорку сол жагындагы **agent сөлөкөтүн** басыңыз (сөлөкөт активдүү agentти чагылдыруу үчүн өзгөрөт).
3. **Agent Picker** form-table катмар катары ачылат. Ар бир agent өзүнүн сөлөкөтү, аталышы жана бир саптык сүрөттөмөсү менен тизмеленет.
4. Agent сабын басып, аны активдештириңиз. Чат башкы бөлүгү дароо жаңыланат.

Сүйлөшүүнүн ортосунда да agentтерди которсоңуз болот — жаңы agentтин system promptу кийинки билдирүүдөн тартып күчүнө кирет.

## Беш камтылган Agents {#the-five-built-in-agents}

### Content Writer {#content-writer}

**Багыты:** Постторду, барактарды жана байланыш формаларын түзүү жана түзөтүү.

**Жеткиликтүү tools:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`. Superdav AI Agent v1.18.0 интеграциялары иштетилгенде, конфигурацияланган календарь контексти, бекитүү дарбазалары, эскертмелер жана SMS билдирме tools бекитилген иш агымдары үчүн да жеткиликтүү болушу мүмкүн.

**Эмнени жакшы аткарат:**
- Кыскача маалыматтан же пландан блог постторун жазып, жарыялоо
- Жаңы сайт үчүн landing pageлердин топтомдорун түзүү
- Байланыш жана суроо-талап формаларын куруу
- URL же издөө аркылуу постторго өзгөчөлөнгөн сүрөттөрдү коюу
- Конфигурацияланган Google Calendar контекстинен иш-чарадан кийинки билдирмелерди даярдап, билдирмелерди жөнөтүүдөн мурун бекитүү үчүн токтотуу

**Баштапкы сунуштар:**
- *WordPress multisite артыкчылыктары жөнүндө 500 сөздүк блог пост жазыңыз.*
- *About, Services жана Contact барактарын түзүп, аларды жарыялаңыз.*
- *Contact барагына брондоо суроо-талап формасын кошуңуз.*
- *Эртеңки конфигурацияланган календарь иш-чарасынын катышуучулары үчүн эскертме даярдап, аны жөнөтүүдөн мурун бекитүүнү күтүңүз.*

### Site Builder {#site-builder}

**Багыты:** Бир promptтон баштап толук вебсайт түзүү.

**Жеткиликтүү tools:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`. Superdav AI Agent v1.18.0 менен, администраторлор иштеткен жерлерде конфигурацияланган башкарылуучу кызмат, бекитүү, эскертме, календарь жана SMS tools жеткиликтүү болушу мүмкүн.

**Эмнени жакшы аткарат:**
- Сүрөттөлгөн бизнес түрү үчүн көп баскычтуу сайт куруу планын түзүү
- Ар бир баскычты өз алдынча аткаруу — түзүм, мазмун, навигация, дизайн
- План ортосундагы каталардан кол менен кийлигишүүнү талап кылбастан калыбына келүү
- Куруунун бир бөлүгү катары сунушталган pluginдерди орнотуу
- Байланыш формаларын түздөн-түз чат интерфейсинен түзүү (Superdav AI Agent v1.10.0+)
- Бекитүү дарбазалары жана эскертме жазуулары иштетилгенде, кайталанма билдирмелерсиз ишке киргизүү эскертмелерин же катышуучуларга кийинки байланыштарды координациялоо

**Баштапкы сунуштар:**
- *Галерея post type, брондоо барагы жана байланыш формасы бар фотография портфолио сайтын куруңуз.*
- *Онлайн менюсу, иш убактысы жана үстөл брондоо суроо-талап формасы бар ресторан вебсайтын түзүңүз.*
- *Кызмат барактары, портфолио бөлүмү жана блог менен фриланс консалтинг сайтын орнотуңуз.*
- *Site builderди колдонуп Contact барагына байланыш формасын кошуңуз.*
- *Сайтты ишке киргизүү текшерүү тизмеси бекитилгенден кийин, конфигурацияланган кызыкдар тараптын байланышына SMS эскертме жөнөтүңүз.*

### Design Studio {#design-studio}

**Багыты:** Визуалдык ыңгайлаштыруу — түстөр, типография, CSS жана блок үлгүлөрү.

**Жеткиликтүү tools:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Эмнени жакшы аткарат:**
- Аталыштуу theme presetтерди колдонуу (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- theme.json аркылуу глобалдык типографияны жана түс палитраларын так жөндөө
- Брендге тиешелүү үстөм өзгөртүүлөр үчүн ыңгайлаштырылган CSS киргизүү
- Барактын screenshotун алып, аны дизайн маселелери боюнча карап чыгуу

**Баштапкы сунуштар:**
- *warm-editorial presetин колдонуп, андан кийин негизги түстү #2d6a4f кылып коюңуз.*
- *Башкы беттин screenshotун алып, эмнени жакшыртарыңызды айтыңыз.*
- *Толук тууралуу фон сүрөтү жана борборлоштурулган баш аталышы бар кайра колдонулуучу hero block pattern түзүңүз.*

### Plugin Manager {#plugin-manager}

**Багыты:** WordPress pluginдерин табуу, орнотуу жана башкаруу.

**Жеткиликтүү tools:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Эмнени жакшы аткарат:**
- Сүрөттөлгөн колдонуу учурга эң ылайыктуу pluginди сунуштоо
- Реестрден ability packтерди орнотуу
- Жеткиликтүү ability каталогун категория боюнча карап чыгуу

**Баштапкы сунуштар:**
- *Мүчөлүк каталогу үчүн эң мыкты plugin кайсы?*
- *WooCommerce abilities packти орнотуңуз.*
- *Бардык жеткиликтүү ecommerce ability packтерди көрсөтүңүз.*

### Support Assistant {#support-assistant}

**Багыты:** Сайт мазмуну, жөндөөлөрү жана WordPress конфигурациясы жөнүндө суроолорго жооп берүү.

**Жеткиликтүү tools:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Ал эмнени жакшы аткарат:**
- Учурдагы сайт жөндөөлөрүн жана параметрлерин издөө
- Сайтта кайсы жазуу түрлөрү, таксономиялар жана менюлар конфигурацияланганын түшүндүрүү
- Жандуу маанилерди окуу аркылуу "бул жөндөө эмне кылат?" деген суроолорго жооп берүү
- Өзгөртүүлөрдү киргизүүдөн мурун окуу үчүн гана диагностикалык катмар катары кызмат кылуу

**Баштапкы сунуштар:**
- *Бул сайтта учурда кайсы плагиндер жана жөндөөлөр активдүү?*
- *Бул сайтта катталган бардык ыңгайлаштырылган жазуу түрлөрүн тизмектеңиз.*
- *Кайсы навигация менюлары бар жана алар кайсы жерлерге дайындалган?*

## Superdav Automation интеграциялары {#superdav-automation-integrations}

Superdav AI Agent v1.18.0 интеграциялары конфигурацияланганда, камтылган агенттер коопсузураак, графикти эске алган автоматташтыруу иш агымдарына катыша алат:

- **Google Calendar окуу куралдары** агенттерге кийинки иштерди даярдоодон мурун конфигурацияланган календарларды жана окуяларды текшерүүгө мүмкүндүк берет.
- **Байланыш жана катышуучуларды дал келтирүү** окуя катышуучуларын WordPress колдонуучулары же белгилүү байланыштар менен дал келтирүүгө жардам берет.
- **Адамдын жактыруу дарбазалары** сезимтал аракеттерди ыйгарым укуктуу колдонуучу карап чыгып, ырастаганга чейин токтотуп турат.
- **Эскертме жазуулары** пландаштырылган тапшырмалар кайра аракет кылганда же кайталанганда кайталанма билдирүүлөрдү алдын алат.
- **TextBee SMS билдирүүлөрү** SMS эсептик дайындары жана иш агымынын уруксаттары иштетилгенде гана конфигурацияланган текст билдирүүлөрдү жөнөтөт.

Сунушталган иш агымы: агенттен билдирүүнү же аракетти даярдоону сураныңыз, жактыруу чакыруусун карап чыгыңыз, андан кийин жактырылган аракеттин улантылышына уруксат бериңиз. Кайталануучу эскертмелер үчүн бир эле окуя же байланыш кайра-кайра кабарланбашы үчүн эскертмелердин кайталануусун жок кылууну иштетилген бойдон калтырыңыз.

## Агенттерди ыңгайлаштыруу {#customising-agents}

Ар бир камтылган агент `gratis_ai_agent_agents` фильтри аркылуу кеңейтиле же алмаштырыла алат.

### Учурдагы агентке ыңгайлаштырылган системалык промпт кошуу {#adding-a-custom-system-prompt-to-an-existing-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### Жаңы агентти каттоо {#registering-a-new-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

Жаңы агент фильтр иштегенден кийин дароо Agent Picker ичинде пайда болот.

### Камтылган агентти алып салуу {#removing-a-built-in-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
