# Жабылган Агенттер

Gratis AI Agent v1.9.0 беш даяр агент менен келет, ар бири атайын куралдар жинагы, ылайыкташтырылган система промпты жана ошол аймактын кеңири тапшырмаларына дал келген башталыш сунуштары менен алдын ала конфигурацияланган. Агенттердин ортосунда өтүү ассистенттин эмне кыла ала турганын жана кандай жооп берет, сиздин катышууңузсуз өзгөртүлөт.

## Агент деген эмне?

Ар бир агент — аталышы бар конфигурация профили, ал төмөнкүлөрдү бириктирет:

- **Куралдар (Tools)** — агент колдонууга уруксат берилген жөндөмдөр (мисалы, Контент жазуучу постту жасоо мүмкүнчүлүгүнө ээ; Дизайн Студио CSS жана theme.json жөндөмдөрүнө жетүүгө мүмкүндүк берет).
- **Система промпты (System prompt)** — агенттин тонусун, приоритеттөрүн жана чектөөлөрүн аныктаган көрсөтмөлөр.
- **Сунуштар (Suggestions)** — чат интерфейсинде тез башташыңызга жардам берүү үчүн жазылган алдын ала промпттар.

## Агент тандоочусуна жетүү

1. WordPress администратордун жанындагы **Gratis AI Agent** панелин ачыңыз.
2. Чат башчынын сол жогорку бурчундагы **агент иконкасына** басыңыз (иконка активдүү агентке ылайык өзгөрөт).
3. **Agent Picker** чат интерфейсинин үстүнкү форма-таблица катары ачылат. Ар бир агент икондосу, аталышы жана бир саптагы сүрөттөлүшү менен тизмеде көрсөтүлөт.
4. Агентдин сарына басып аны активдештириңиз. Чат башчы дароо жаңыланып турат.

Сиз да сүйлөшүү аягында агенттерди өзгөртө аласыз — жаңы агенттин система промпты кийинки билдирүүдөн баштап күч алсын.

## Беш даяр агент

### Контент жазуучу (Content Writer)

**Көңүл буруу:** постторду, саптарга жана контакт ишкердик формаларына жасоо жана редакциялоо.

**Кайра колдонулуучу куралдар:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`

**Канча эмнени жакшы алат:**
- Кыскача маалымат же пландан жазып, блог постуларды даярдоо жана жарыялоо
- Жаңы сайт үчүн лендинг беттеринин топторду түзүү
- Байланыш жана суроо формаларын куруу
- URL же издөө аркылуу посттардагы акыркы сүрөттөрдү коюу

**Баштапкы сунуштар:**
- *WordPress multisite пайдалары жөнүндө 500 сөздөгү блог посту жазыңыз.*
- *Байланыш, кызматтар жана контакттык беттерди түзүп, жарыялаңыз.*
- *Контакттык бетке бөгөөлөнүү суроо формасын кошуңуз.*

---

### Сайт куруучу (Site Builder)

**Көңүл буруу:** Бир сунуштан бүтүндөй сайтты түзүү.

**Кайра турган куралдар:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`

**Канча эмнени жакшы алат:**
- Сүрөттөлгөн бизнес түрү үчүн көп этаптуу сайт куруу планы түзүү
- Ар бир этапты өз алдынча аткаруу — структура, мазмун, навигация, дизайн
- Пландын ортосундагы каталардан колдонуучу аракети талап кылбай айырмалануу (Recovering from errors mid-plan without requiring manual intervention)
- Куруу процессинин бир бөлүгү катары сунушталган плагиндерди орнотуу
- чат интерфейси аркылуу контакттык формаларды түзүү (Superdav AI Agent v1.10.0+)

**Баштапкы сунуштар:**
- *Галерея пост түрү, бөгөөлөнүү беттери жана контакттык формасы менен фотограф портфолио сайты куруңуз.*
- *Онлайн меню, ачуу сааттары жана столго жабуу суроо формасы бар ресторан сайты түзүңүз.*
- *Кызматтар беттери, портфолио бөлүмү жана блог менен фрилансер консультация сайты орносоңуз.*
- *Сайт куруучу аркылуу Контакттык бетке контакттык формасын кошуңуз.*

---

### Дизайн Студия (Design Studio)

**Көңүл буруу:** Визуалдык кастомизация — түстөр, типография, CSS жана блоктордун үлгүлөрү.

**Кайры бар куралдык аспаптар:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Канчалык жакшы иштешет:**
- Аталыштырылган тема пресеттерин (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog) колдонуу.
- theme.json аркылуу глобал типографияны жана түстүү палитраларды тактыруу.
- Брендге тиешелүү өзгөртүүлөрдү киргизүү үчүн кастом CSSды кошуу.
- Барактын скриншотун алуу жана дизайн маселелерин текшерүү.

**Баштапкы сунушчулар:**
- *warm-editorial пресеттерин колдонуп, андан кийин негизги түстү #2d6a4f деп коюңуз.*
- *Үндүрүштүн башкы беттин скриншотун алып, эмнени жакшыртуу керектигин айтып бериңиз.*
- *Толук көлөмдөгү фондук сүрөт жана борбордо жазылган аталыш менен кайра колдонууга мүмкүн болгон hero block pattern түзүңүз.*

---

### Plugin Manager (Плагин Менежеги)

**Көңүл буруу:** WordPress плагиндерін табуу, орнотуу жана башкаруу.

**Кайры бар аспаптар:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Канчалык жакшы иштешет:**
- Сүрөттөлгөн колдонуу учуру үчүн эң мыкты плагинди сунуштоо.
- реестрден ability packs орнотуу.
- Мамлекеттик категориялар аркылуу жеткиликтүү ability каталогуну карап чыгуу.

**Баштапкы сунушчулар:**
- *Мемлекеттик каталог үчүн эң мыкты плагин кайсы?*
- *WooCommerce ability packs орнотуңуз.*
- *Бардык жеткиликтүү электрондук коммерция (ecommerce) ability packs көрсөтүңүз.*

---

### Support Assistant (Дعمди колдоочу ассистент)

**Көңүл буруу:** Сайттын мазмуну, жөнгө салуулар жана WordPress конфигурациясы жөнүндө суроолорго жооп берүү.

**Кайры бар аспаптар:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Канча-качан ийманды жактары:**
- Сайттын учурдагы жөндөөлөрүн жана опцияларын текшерүү.
- Сайтта кандай пост түрлөрү, таксификациялар жана менюлар колдонулуп жатканын түшүндүрүү.
- Чыныгы маанилерди окуу аркылуу "бул жөндөм эмне кылат?" деген суроолорго жооп берүү.
- Өзгөртүүлөрдү киргизүүдөн мурун окуучулук диагностика катмары катары кызмат кылуу.

**Баштапкы сунуштор:**
- *Бул сайтта учурда кандай плагин жана жөндөөлөр активде?*
- *Бул сайтта катталган бардык кошумча пост түрлөрүн тизмелей.*
- *Кай дагы навигация менюлары бар жана алар кайсы жерге таандык?*

## Агенттерди кастомизациялоо

Ар бир киргизилген агентти `gratis_ai_agent_agents` фильтри аркылуу кеңейте аласыз же алмаштыра аласыз.

### Учурдагы агентке кошумча система промптын (system prompt) кошуу

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### Жаңы агентти каттоо

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Постторду жана беттерди издөө кызматы үчүн оптималдаштырат.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'Сиз SEO адисисисиз. Кирүү түшүнүктөрүн, мета-сүрөттөрдү жана структураланган маалыматты оптималдаштырууга басым жасаңыз.',
        'suggestions'   => [
            'Башкы беттин аталышын жана мета-сүрөтүн текшериңиз.',
            'Эң акыркы беш пост үчүн аталыш теги (title tag) жакшыртууну сунуштаңыз.',
        ],
    ];
    return $agents;
} );
```

Жаңы агент фильтр иштей алдында, Агент таңгаруучу (Agent Picker) дароবদа пайда болот.

### Ички агентти алып салуу

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
