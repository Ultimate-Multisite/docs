# Тема Билдердигиси: Блок Темаларды Скелетттоо жана Ишке Кирүү {#theme-builder-abilities-scaffold-and-activate-block-themes}

Superdav AI Agent v1.12.0 эки күчтүү жөндөмдү киргизет, алар сизге чат интерфейси аркылуу өз алдынча блок темаларды түзүүгө жана кызмат көрсөтүүгө мүмкүндүк берет.

## Жалпы маалымат {#overview}

**scaffold-block-theme** жана **activate-theme** жөндөмдөр агенттерге төмөнкүлөрдү жүргүзүүгө мүмкүндүк берет:
- Сиздин талаптарыңызга негизделген толук, өндүрүш даяр блок темаларды түзүү
- Мануал аракетисиз сайтта темаларды автоматтык түрдө ишке киргизүү
- Багытталган дизайн чечимдер аркылуу бирдей визуалдык идентификацияларды түзүү

## Блок Теманы Скелеттөө (Scaffold Block Theme) {#scaffold-block-theme}

**scaffold-block-theme** жөндөмү толук тема структурасы менен жаңы WordPress блок теманды түзөт, анын ичине төмөнкүлөр камтылат:

- дизайн токендери бар `theme.json` конфигурациясы
- жалпы макеттер үчүн Блок шаблон файлдары
- Кастом блок стилдери жана варианттары
- Тема метадатасы жана колдоо билдирүүлөрү

### Иш-аракетти Көрсөтүү (How to Invoke) {#how-to-invoke}

Superdav AI Agent менен чат аркылуу теманы түзүүнү сурасаңыз болот:

```
"Modern Agency" деп аталган блок теманды, көк жана ак түстүү схемасы,
sans-serif типографиясы жана профессионал макет менен түзүңүз.
```

Агент төмөнкүлөрдү ишке ашырат:
1. Сиздин дизайн каалоолоруңузду сүйлөшүү аркылуу чогултуу
2. Толук тема структурасын түзүү
3. Бардык зарыл тема файлдарын түзүү
4. Теманы ишке киргизүүгө даярдоо

### Күтүлгөн Натыйжа (Expected Output) {#expected-output}

Жөндөмдү ийгиликтүү аткарганда, сиз төмөнкүлөрдү көрөсүз:

- Тема скелеттелгени жөнүндө тастыктоо
- Темандын аталышы жана орду
- Колдонулган дизайн токендеринин кыскача маалыматы (цветдер, типография, аралык)
- Ишке киргизүүгө даяр абалы

Мисал:
```
✓ "Modern Agency" темасы ийгиликтүү скелеттелди
  Орду: /wp-content/themes/modern-agency/
  Цветтер: Негизги #0066CC, Экинчи #FFFFFF
  Типография: Inter (sans-serif)
  Абалы: Ишке киргизүүгө даяр
```

## Теманы Ишке Киргизүү (Activate Theme) {#activate-theme}

**activate-theme** мүмiyatı сиздин сайтыңызды жаңы құрылған немесе бар блок темасына ауыстырады.

### Қолдану тәсілі {#how-to-invoke-1}

Теманы құрдыktan кейін, оны дереу белсендіре аласыз:

```
"Modern Agency темун белсендіру"
```

Немесе кез келген бар теманы белсендіріңіз:

```
"Twentytwentyfour темасына ауысу"
```

### Күтілетін нәтиже {#expected-output-1}

Белсендіру сәтті болғанда:

- Белсендір된 тема туралы растау
- Алдыңғы тема атауы (қарақты тексеру үшін)
- Тема енді қай жерде жасайтыны туралы сайт URL-і
- Темаға арналған кез келген орнату ескертулері

Мысал:
```
✓ Тема сәтті белсендірілді
  Белсендір된 тема: Modern Agency
  Алдыңғы тема: Twentytwentyfour
  Жайда: https://yoursite.com
  Ескерту: Дизайнды тексеру үшін басты бетіңізді қараңыз
```

## Жұмыс ағымы: Құру және белсендіру (Scaffold and Activate) {#workflow-scaffold-and-activate}

Тиісті жұмыс ағымы екі мүмкіндікті біріктіреді:

1. **Теманы жасауды сұрау**: "Менің SaaS бетім үшін блок темасын жасаңыз"
2. **Агент теманы құрайды (scaffolds)**: Файлдар мен дизайн токендерін жасайды
3. **Тексеру және жетілдіру**: қажет болса, дизайн өзгерістерін талқылау
4. **Белсендіру**: "Теманы дереу белсендіру"
5. **Тексеру**: Жаңа дизайн жасайтынын растау үшін сайтыңызды басыңыз

## Дизайн токендері және жекелеу (Customization) {#design-tokens-and-customization}

Құрылған темалар WordPress дизайн токендерін (`theme.json` арқылы) қолданады:

- **Түстер**: Негізгі, екінші, акцент, нейтралды палитра
- **Типография**: Шрифттар, өлшемдер, салмақтар, жол биіктіктері
- **Асық (Spacing)**: Padding, margin, gap шкалалары
- **Шығындар (Borders)**: Радиус және ендік токендері
- **Түтін (Shadows)**: Жоғары деңгейлер

Бұл токендер `theme.json` файлында орталықтандырылған, бұл бүкіл дизайн жүйеңізді бір файлдан 쉽게 реттеуге мүмкіндік береді.

## Шешімдер және ескертулер {#limitations-and-notes}

- Темалар `/wp-content/themes/` даректе каркас (scaffold) кылынат жана алар WordPress аталыш каада-карашка ылайык болушу керек
- Активация үчүн WordPress сайтыңызда тиешелүү укуктар болушу шарт
- Темаларда кошумча PHP коду минималдуу болушу керек; татаал функциялар үчүн плагинддерди колдонуңуз
- Блок темалар WordPress 5.9 жана андан жаңы версиялары менен эң жакшы иштешет

## Кырдаалды чечүү (Troubleshooting) {#troubleshooting}

**Каркастан кийин тема пайда болбойт**
- Тема директорунун бар экенин жана туура укуктардын болушун текшериңиз
- `theme.json` файлдын туура JSON форматтагы экенин текшериңиз
- Теманын аталышы бардык темалар менен татаалбагарын камсыз кылыңыз

**Активация ишке ашпайт**
- Администратор укуктарыңыз бар экенин ырастаңыз
- Тема директорунун WordPress тарабынан окулуп жаткандыгын текшериңиз
- Деталдар үчүн WordPress ката журналдарын карап чыгыңыз

**Дизайн токендерди колдонуп берилбейт**
- `theme.json` синтаксиси туура экенин текшериңиз
- Качтаруу плагинддерин (caching plugins) тазалаңыз
- WordPress версияңыз колдонуп жаткан токендерди колдонуп берип жатканын камсыз кылыңыз

## Кийинки кадамдар {#next-steps}

Теманы активка кылып алгандан кийин сиз төмөнкүлөрдү кыла аласыз:
- Типографияны, түстөрдү жана аралыкты (spacing) таклоо үчүн **Design System Aesthetics** көндүмдөрүн колдонуңуз
- WordPress блок редактору аркылуу ар бир блок стилдерин ылайыкташтырыңыз
- Теманын `style.css` файлына кошумча CSS кошуңуз
- Атайын бет түрлөрү үчүн ар кандай блок шаблондорун түзүңү
