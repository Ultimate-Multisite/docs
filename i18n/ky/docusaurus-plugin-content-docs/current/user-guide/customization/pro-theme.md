# Администратор панелиндиңизди PRO темасы менен ка떻даруу

Клиенттердин ишинде көп учурда, WordPress администратордук панелине өзүнө ылайыктуу тажрыйбаны берүү керек болот. Эгер сиз көбүрөөк профессионалдуу көрүнүш жана сезимди издеп жатсаңыз, PRO Theme сизге жардам бере алат.

PRO Theme — бул клиенттердин дашбордун дизайнын өзү менен иштөөдө сизге жардам берүү үчүн иштелип чыккан администратор темасы. Ал ошондой эле Ultimate Multisite'тин 프리миум плагиндердин негизги бөлүгү болуп саналат.

Аны орносодон кийин, _Plugins_ баракчасына өтүп, аны Сетиңизде (Network) активдештирүү үчүн клика бериңиз.

<!-- Скриншот жок: WP Plugins баракчасы PRO Theme үчүн Сети Активдештирүү баскычын көрсөтүп турат -->

_Plugins баракчасында Сети бүтүндөй тармагына активдештириңиз_

_**ЭСКЕРТҮҮ:**_ _Эгер сиз аны жөн гана орносоңуз, ал клиенттердин үчүн опция катары көрсөтүлөт жана аларды ошол колдонуучуга гана теманы кошуу үчүн белгилөөгө болот._

Азыр сизде администратор интерфейси толук кайра дизайны! Ошондой эле сол колонкада PRO Theme деген жаңы меню бар.
