# Сетьдеревеңизге рекламаларды Ultimate Multisite аркылуу орнотуу: Ad Injector {#placing-ads-on-your-network-with-ultimate-multisite-ad-injector}

Ultimate Multisite: Ad Injector — бул сиздин планларыңыздагы жарнамаларды атайында редакциялоо баракчадан эле орнотууга мүмкүндүк берген **белөөлөнө турган (free) плагин**.

Ultimate Multisite: Ad Injector-ди орнотуп алгандан кийин, аны активдештирүү үчүн _Plugins_ баракчасына барыңыз.

<!-- Скриншот жок: WP Plugins баракчасында Ad Injector плагинине арналган Сеть активдештиру кнопкасы -->

_Сеть активдештиру кночуна басыңыз_

Плагин активдүү болгондон кийин, планды редакциялоо экранындагы _Advanced Options_ мета боксогунда _Ad Settings_ деп аталган жаңы опция пайда болот.

![Планды редакциялоо экранындагы Ad Settings](/img/config/product-edit-full.png)

_Азыр сиз ар бир план үчүн жарнама параметрлерин редакцияласаңыз болот_

Ошол вкладкада сиз контенттин алдын, андан кийин жана андан кийин фронтенддеги посттор жана беттер үчүн жарнама коддорун кошуп жиберсеңиз болот.

<!-- Скриншот жок: Контенттин алдындагы жана артындагы посттор жана беттер үчүн Ad code талаалары -->

<!-- Скриншот жок: Атайын планга карата пост баракчасында көрсөтүлгөн жарнаманын мисалы -->

_Жарнама кодуңузду киргизе алгандан кийин, ал ошол атайын пландын постторунда жана беттеринде көрсөтүлөт._

Ошондой эле, сиз ошол пландын колдонуучулары үчүн Администратордук баракчанын баш мүчөсүндө жарнамаларды камтууну да мүмкүн кыласыз.

<!-- Скриншот жок: План параметрлериндеги Администратордук панельдеги Жарнама коду опциясы -->

<!-- Скриншот жок: Колдонуучунун администратордук панелинда көрсөтүлгөн жарнаманын мисалы -->

_Эгер сиз колдонуучуңуздун артындагы бөлүмдө жарнамаларды көрсөткүңүз келсе, Администратордук панелдеги Жарнама коду опциясына барыңыз_

### Жарнамадан тышкары {#beyond-advertising}

Сиз да Ultimate Multisite: Ad Injector-ди жогорку деңгээлдеги планларга колдоо көрсөтүү үчүн колдонуп турушуңуз мүмкүн.

Кечине _Admin Panel Ads_ опциясына барып, колдоо куралыңыздын HTML кодун кошуңуз.
