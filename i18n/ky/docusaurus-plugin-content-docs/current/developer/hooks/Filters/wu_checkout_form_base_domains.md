# Фильтр: wu_checkout_form_base_domains {#filter-wucheckoutformbasedomains}

Төлөмдү аяктоо формасынын жалпы базалык домендерин чыпкалайт, алар mapped-domain жазууларына айланбашы керек.

Интеграция төлөмдү аяктоо формасынын **сайттын URL дареги** талаалары үчүн кошумча базалык домендерди бергенде бул фильтрди колдонуңуз. Бул фильтр кайтарган домендер ар бир сайт үчүн өзүнчө бажы домендердин ордуна жалпы каттоо хосттору катары каралат.

## Параметрлер {#parameters}

| Аты | Түрү | Сүрөттөмө |
|------|------|-------------|
| $domains | `array` | Төлөмдү аяктоо формасынын конфигурациясынан чогултулган жалпы базалык домендер. |

### Чыгарылгандан бери {#since}

- 2.13.0

### Булак {#source}

`inc/functions/domain.php` ичинде аныкталган.


## Кайтаруулар {#returns}

Нормалдаштырылган төлөмдү аяктоо формасынын базалык домендеринин массиви.
