# Фильтр: wu_password_require_special

Сырсөздөрдө атайын символдор талап кылынаарын чыпкалоо.

## Параметрлер

| Аты | Түрү | Сүрөттөмө |
|------|------|-------------|
| $require | `bool` | Атайын символдор талап кылынаарын аныктайт. Эрежелер күчүндө болгондо демейкиси true. |
| $defender_active | `bool` | Defender Pro Strong Password активдүү экенин аныктайт. |

- 2.4.0
