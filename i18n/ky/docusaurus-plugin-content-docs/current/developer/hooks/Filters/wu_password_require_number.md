# Чыпка: wu_password_require_number

Сырсөздөрдө сандарды талап кылуу же кылбоону чыпкалайт.

## Параметрлер {#parameters}

| Аты | Түрү | Сүрөттөмө |
|------|------|-------------|
| $require | `bool` | Сандарды талап кылуу же кылбоо. Эрежелер колдонулганда демейкиси true. |
| $defender_active | `bool` | Defender Pro Strong Password активдүүбү же жокпу. |
