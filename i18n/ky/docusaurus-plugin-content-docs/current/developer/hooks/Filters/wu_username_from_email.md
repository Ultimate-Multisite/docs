# Filter: wu_username_from_email

Жаңы кардардын колдонуучу атын чыпкалоо.

## Параметрлер {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $username | `string` | Кардардын колдонуучу аты. |
| $email | `string` | Жаңы кардардын email дареги. |
| $new_user_args | `array` | Жаңы колдонуучунун аргументтеринин массиви, балким аты жана фамилиясы камтылат. |
| $suffix | `string` | Аны уникалдуу кылуу үчүн колдонуучу атына сап кошуу. |
