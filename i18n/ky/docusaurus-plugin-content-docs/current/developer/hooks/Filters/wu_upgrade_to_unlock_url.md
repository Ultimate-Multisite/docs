# Filter: wu_upgrade_to_unlock_url

Иштеп чыгуучуларга кулпуну ачуу үчүн жаңыртуу URLин өзгөртүүгө уруксат берет

## Параметрлер

| Аты | Түрү | Сүрөттөмө |
|------|------|-------------|
| $url | `string` | Жаңыртуу URLи. |
| $args | `array` | Жаңыртуу талап кылынган модуль жана чектөөнүн түрү. |
