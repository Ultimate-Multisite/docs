# Чыпка: wu_hosting_integration_wizard_sections

Хостинг интеграциясын жөндөө үчүн устанын бөлүмдөрүн чыпкалайт.

Кошумчаларга устанын бөлүмдөрүн кошууга, алып салууга же өзгөртүүгө мүмкүндүк берет.

## Параметрлер {#parameters}

| Аты | Түрү | Сүрөттөмө |
|------|------|-------------|
| $sections | `array` | Устанын бөлүмдөрү. |
| $integration | `\WP_Ultimo\Integrations\Host_Providers\Base_Host_Provider\|\WP_Ultimo\Integrations\Integration` | Жөндөлүп жаткан интеграция. |
| $page | `\Hosting_Integration_Wizard_Admin_Page` | Уста барагынын экземпляры. |

### Качандан бери {#since}

- 2.5.0
