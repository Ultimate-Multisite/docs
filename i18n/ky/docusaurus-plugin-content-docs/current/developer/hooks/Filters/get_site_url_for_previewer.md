# Фильтр: get_site_url_for_previewer {#filter-getsiteurlforpreviewer}

Plugin иштеп чыгуучуларына previewer ичинде колдонулган URL'ди чыпкалоого мүмкүндүк берет

## Параметрлер {#parameters}

| Аты | Түрү | Сүрөттөмө |
|------|------|-------------|
| $domain | `string` | Азыр колдонулуп жаткан демейки домен, өзгөртүүлөр үчүн пайдалуу |
| $domain_options | `array` | Ultimate Multisite Settings -&gt; Network Settings -&gt; Domain Options ичинде киргизилген бардык домен параметрлеринин тизмеси |

### Бери {#since}

- 1.7.2
### Булак {#source}

[`inc/checkout/class-legacy-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-legacy-checkout.php#L812) ичинде 812-сапта аныкталган


## Кайтарат {#returns}
Колдонулуучу жаңы домен
