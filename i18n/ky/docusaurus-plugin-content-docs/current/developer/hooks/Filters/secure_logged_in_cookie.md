# Чыпка: secure_logged_in_cookie

Кирген cookie HTTPS аркылуу гана жөнөтүлүшү керекпи же жокпу чыпкалайт.

## Параметрлер

| Name | Type | Description |
|------|------|-------------|
| $secure_logged_in_cookie | `bool` | Кирген cookie HTTPS аркылуу гана жөнөтүлүшү керекпи же жокпу. |
| $user_id | `int` | Колдонуучу ID. |
| $secure | `bool` | Auth cookie HTTPS аркылуу гана жөнөтүлүшү керекпи же жокпу. |
