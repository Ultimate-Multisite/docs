# Чыпка: wu_billing_address_fields

Plugin иштеп чыгуучуларына эсептешүү дареги талааларын чыпкалоого уруксат берет.

## Параметрлер

| Аты | Түрү | Сүрөттөмө |
|------|------|-------------|
| $fields | `array` | Эсептешүү дарегинин массиви. |
| $zip_only | `bool` | Эгер бизге почта индекси жана өлкө гана керек болсо. |
