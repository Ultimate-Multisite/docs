# Filter: wu_current_\{$type\}_get_manage_url {#filter-wucurrenttypegetmanageurl}

| Аты | Түрү | Сүрөттөмө |

| $id | `int` | Сайт ID. |
| $site_hash | `string` | Сайт хэши. |
