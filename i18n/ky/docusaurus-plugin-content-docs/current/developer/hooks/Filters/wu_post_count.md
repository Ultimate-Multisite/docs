# Фильтр: wu_post_count

Плагин иштеп чыгуучуларына жалпы санды өзгөртүүгө уруксат берет

## Параметрлер

| Аталышы | Түрү | Сүрөттөмө |
|------|------|-------------|
| $count | `int` | Жалпы пост саны |
| $post_counts | `object` | wp_count_posts функциясы кайтарган WordPress объекти |
| $post_type | `string` | Пост түрүнүн кыска аталышы |

### Версиядан тартып

- 1.9.1
