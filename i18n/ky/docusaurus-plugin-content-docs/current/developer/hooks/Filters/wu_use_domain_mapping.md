# Filter: wu_use_domain_mapping

Карталоону колдонуу керекпи же жокпу аныктайт

Адатта, сиз активдүү карталоолорду гана колдонууга уруксат берүүнү каалайсыз. Бирок, эгер өркүндөтүлгөн логиканы колдонгуңуз келсе же активдүү эмес домендерди да карталоого уруксат бергиңиз келсе, бул жерден фильтрлеңиз.

## Параметрлер {#parameters}

| Аты | Түрү | Сүрөттөмө |
|------|------|-------------|
| $is_active | `bool` | Карталоо активдүү деп каралышы керекпи? |
| $mapping | `\Domain` | Биз текшерип жаткан карталоо |
| $domain | `string` |  |
