# Filter: wu_view_override_replaceable_views {#filter-wuviewoverridereplaceableviews}

Иштеп чыгуучуларга алмаштырылуучу тизмеге кошумча папкаларды кошууга мүмкүндүк берет.

Сак болуңуз, анткени кошумча папкаларга уруксат берүү Ultimate Multisite версияларынын ордуна эскирген көчүрмөлөрдүн жүктөлүшүнө алып келиши мүмкүн.

## Параметрлер {#parameters}

| Аты | Түрү | Сүрөттөмө |
|------|------|-------------|
| $replaceable_views | `array` | Уруксат берилген папкалардын тизмеси. |

### Чыгарылган версиясы {#since}

- 2.0.0
