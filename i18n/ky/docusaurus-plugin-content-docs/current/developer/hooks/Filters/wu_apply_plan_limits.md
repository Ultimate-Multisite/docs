# Фильтр: wu_apply_plan_limits

Плагин иштеп чыгуучуларга чектөөлөрдү айланып өтүүгө уруксат берет.

Бул фильтрди чектөөлөрдүн кайсынысы болбосун ишке киргизилгенге чейин каалагандай кодду аткаруу үчүн колдоно аласыз. Эгер фильтриңиз чындык мааниге бааланган каалаган маанини кайтарса, процесс уланат; эгер жалган мааниге бааланган каалаган маанини кайтарса, код кайтып, төмөндөгү хуктардын эч бири иштебейт.

### Версиядан тартып

- 1.7.0
