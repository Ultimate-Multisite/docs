# Фильтр: wu_magic_link_enforce_user_agent

Колдонуучу агентин текшерүүнү мажбурлап колдонуу-келбесин чыпкалайт.

Токендердин ар башка браузерлерде/түзмөктөрдө иштешине уруксат берүү үчүн false кылып коюңуз. Бул коопсуздукту азайтат, бирок колдонууга ыңгайлуулукту жогорулатат.

## Параметрлер {#parameters}

| Аты | Түрү | Сүрөттөмө |
|------|------|-------------|
| $enforce | `bool` | Колдонуучу агентинин дал келишин мажбурлап колдонуу керекпи. |
