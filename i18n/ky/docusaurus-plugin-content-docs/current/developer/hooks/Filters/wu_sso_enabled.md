# Чыпка: wu_sso_enabled

Домендер аралык single-sign-on мүмкүнчүлүгүн иштетүү/өчүрүү.

Бул маанини чыпкалап, single-sign-on'ду толугу менен өчүрүңүз же анын ордуна шарттуу түрдө иштетиңиз.

## Параметрлер

| Аты | Түрү | Сүрөттөмө |
|------|------|-------------|
| $enabled | `bool` | SSO иштетилиши керекпи? Күйгүзүү үчүн true, өчүрүү үчүн false сыяктуу маани. |

### Кайсы версиядан бери

- 2.0.11
### Булагы

[`inc/sso/class-sso.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/sso/class-sso.php#L110) ичинде 110-сапта аныкталган
