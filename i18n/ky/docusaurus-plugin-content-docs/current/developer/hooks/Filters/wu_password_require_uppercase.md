# Чыпка: wu_password_require_uppercase

Сырсөздөрдө чоң тамгаларды талап кылуу керекпи-жокпу чыпкалайт.

## Параметрлер {#parameters}

| Аты | Түрү | Сүрөттөмө |
|------|------|-------------|
| $require | `bool` | Чоң тамгаларды талап кылуу керекпи. Эрежелер күчүндө болгондо демейкиси true. |
| $defender_active | `bool` | Defender Pro Strong Password активдүүбү. |
