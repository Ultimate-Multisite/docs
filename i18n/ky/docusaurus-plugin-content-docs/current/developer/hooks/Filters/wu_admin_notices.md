# Фильтр: wu_admin_notices {#filter-wuadminnotices}

Иштеп чыгуучуларга Ultimate Multisite тарабынан кошулган админ билдирүүлөрүн чыпкалоого мүмкүндүк берет.

## Параметрлер {#parameters}

| Аты | Түрү | Сүрөттөмө |
|------|------|-------------|
| $notices | `array` | Ошол белгилүү панел үчүн билдирүүлөрдүн тизмеси. |
| $all_notices | `array` | Панел боюнча бөлүнүп кошулган билдирүүлөрдүн тизмеси. |
| $panel | `string` | Билдирүүлөрдү алуу үчүн панел. |
| $filter | `string` | Эгер жабууга боло турган билдирүүлөр чыпкаланып чыгарылган болсо. |
| $dismissed_messages | `array` | Жабылган билдирүү ачкычтарынын тизмеси. |

### Версиядан тартып {#since}

- 2.0.0
