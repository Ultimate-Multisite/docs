# Фильтр: wu_documentation_get_link

Плагин иштеп чыгуучуларына шилтемелерди чыпкалоого мүмкүндүк берет.

Бул кандай пайдалуу болору так эмес, бирок анын болгону зыян келтирбейт

## Параметрлер

| Аты | Түрү | Сүрөттөмө |
|------|------|-------------|
| $link | `string` | Катталган шилтеме |
| $slug | `string` | Шилтемени алуу үчүн колдонулган slug |
| $default_link | `string` | Катталган демейки шилтеме |
