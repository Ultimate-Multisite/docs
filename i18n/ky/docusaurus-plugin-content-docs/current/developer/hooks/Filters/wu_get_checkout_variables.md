# Чыпка: wu_get_checkout_variables {#filter-wugetcheckoutvariables}

Плагин иштеп чыгуучуларына checkout барагынын алдын ала орнотууларын чыпкалоого мүмкүндүк берет.

Этият болуңуз, жетишпеген ачкычтар front-end'деги checkout'ту толугу менен бузуп коюшу мүмкүн.

## Параметрлер {#parameters}

| Аты | Түрү | Сүрөттөмө |
|------|------|-------------|
| $variables | `array` | Локалдаштырылган өзгөрмөлөр. |
| $checkout | `\Checkout` | Checkout классы. |

### Баштап {#since}

- 2.0.0
