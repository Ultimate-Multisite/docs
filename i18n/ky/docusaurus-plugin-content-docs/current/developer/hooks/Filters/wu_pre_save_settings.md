# Filter: wu_pre_save_settings {#filter-wupresavesettings}

Иштеп чыгуучуларга жөндөөлөрдү Ultimate Multisite тарабынан сактоодон мурун чыпкалоого уруксат берет.

## Параметрлер {#parameters}

| Аталышы | Түрү | Сүрөттөмө |
|------|------|-------------|
| $settings | `array` | Сактала турган жөндөөлөр. |
| $settings_to_save | `array` | Кошула турган жаңы жөндөөлөр. |
| $saved_settings | `array` | Учурда сакталган жөндөөлөр. |

### Кайсы версиядан бери {#since}

- 2.0.18
