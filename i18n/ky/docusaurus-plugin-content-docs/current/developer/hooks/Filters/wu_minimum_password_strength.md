# Сүзгүч: wu_minimum_password_strength

Талап кылынган эң төмөнкү сырсөз күчүн сүзгүчтөн өткөрүңүз (zxcvbn упайы).

Күч деңгээлдери: - 0, 1: Өтө алсыз - 2: Алсыз - 3: Орточо - 4: Күчтүү (демейки)

## Параметрлер

| Аты | Түрү | Сүрөттөмө |
|------|------|-------------|
| $min_strength | `int` | Талап кылынган эң төмөнкү күч деңгээли. |
| $strength_setting | `string` | Админ жөндөөсүнүн мааниси (medium, strong, super_strong). |

### Баштап

- 2.4.0
