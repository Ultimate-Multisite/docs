# Фильтр: wu_domain_has_correct_dns

Плагин иштеп чыгуучуларына жыйынтыктарды аныктоо үчүн жаңы текшерүүлөрдү кошууга уруксат берет.

## Параметрлер {#parameters}

| Аты | Түрү | Сүрөттөмө |
|------|------|-------------|
| $result | `bool` | учурдагы жыйынтык. |
| $domain | `self` | Учурдагы домен нускасы. |
| $domains_and_ips | `array` | DNS издөөсүндө табылган домендердин жана IP даректеринин тизмеси. |
