# Фильтр: wp_ultimo_render_vars {#filter-wpultimorendervars}

Plugin иштеп чыгуучуларына render контекстине глобалдык түрдө кошумча өзгөрмө кошууга мүмкүндүк берет.

## Параметрлер {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $args | `array` | render чакыруусу аркылуу өткөрүлгөн өзгөрмөлөрдү камтыган массив. |
| $view | `string` | Render кылына турган көрүнүштүн аталышы. |
| $default_view | `string` | fallback_view аталышы |

### Чыгарылган версиясы {#since}

- 2.0.0
### Булак {#source}

[`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L31) ичинде 31-сапта аныкталган
