# Filter: wu_magic_link_enforce_ip

IP даректи текшерүүнү мажбурлоону чыпкалоо.

Токендер ар башка тармактардан иштешине уруксат берүү үчүн false деп коюңуз. Бул коопсуздукту төмөндөтөт, бирок колдонууга ыңгайлуулукту жогорулатат (мисалы, тармак алмаштырган мобилдик колдонуучулар үчүн).

## Параметрлер

| Аты | Түрү | Сүрөттөмө |
|------|------|-------------|
| $enforce | `bool` | IP даректин дал келишин мажбурлоо керекпи. |

- 2.0.0
