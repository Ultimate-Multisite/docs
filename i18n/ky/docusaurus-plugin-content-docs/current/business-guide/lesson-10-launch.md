# Сабак 10: Ишке киргизүү күнү

Баары курулду. Эшиктерди ачардан мурун, бул сабак ишке киргизүүгө чейинки текшерүү тизмесинен өтүп, эч нерсе бузук, жетишсиз же уят абалда эмес экенине ынанууга жардам берет.

## Кайсы жерден токтогонбуз {#where-we-left-off}

FitSite'те шаблондор, пландар, төлөм тариздөө, брендинг, баштапкы тааныштыруу жана баалар толугу менен жөндөлгөн. Эми баары иштеп жатканын текшерип, түз эфирге чыгарабыз.

## Ишке киргизүүгө чейинки текшерүү тизмеси {#pre-launch-checklist}

Ар бир пункттан өтүңүз. Эч бирин өткөрүп жибербеңиз.

### Платформа инфраструктурасы {#platform-infrastructure}

- [ ] Хостинг туруктуу жана жүктөм астында жакшы иштеп жатат
- [ ] Wildcard SSL активдүү жана бардык субдомендер HTTPS аркылуу кызмат көрсөтөт
- [ ] Домен байланыштыруу иштейт -- сайт түзүп, жеке доменди байланыштырып көрүңүз
- [ ] Камдык көчүрмөлөр жөндөлгөн жана текшерилген (текшерүү үчүн жок дегенде бирин калыбына келтириңиз)
- [ ] Мониторинг орнотулган -- платформа иштен чыкса, сиз билесиз

### Шаблондор {#templates}

- [ ] Үч шаблондун баары жаңы сайттарда туура жүктөлөт
- [ ] Убактылуу мазмун пайдалуу жана катасыз
- [ ] Бардык сүрөттөр тийиштүү лицензияланган (сток сүрөттөрдүн суу белгилери жок)
- [ ] Ар бир шаблон бетинде мобилдик ыңгайлашуу иштейт
- [ ] Бет жүктөлүү ылдамдыгы алгылыктуу (GTmetrix же PageSpeed Insights сыяктуу курал менен текшериңиз)
- [ ] Эч бир шаблондо бузук шилтемелер же жетишпеген ресурстар жок

### Пландар жана өнүмдөр {#plans-and-products}

- [ ] Үч пландын баары активдүү жана көрүнүп турат
- [ ] План сүрөттөмөлөрү так жана тармакка ылайык
- [ ] Баалар туура (айлык жана жылдык)
- [ ] Орнотуу төлөмдөрү туура пландарда жөндөлгөн
- [ ] Сыноо мөөнөтү Starter планында иштейт
- [ ] Кошумча сунуштар төлөм тариздөө учурунда туура көрүнөт
- [ ] Плагин жана тема чектөөлөрү ар бир план боюнча туура колдонулат

### Төлөм тариздөө агымы {#checkout-flow}

- [ ] Ар бир планда толук тесттик катталууну бүтүрүңүз (тесттик төлөм режимин колдонуңуз)
- [ ] Шаблон тандоодо ар бир планга туура келген шаблондор көрсөтүлөт
- [ ] Төлөм ийгиликтүү өтөт
- [ ] Кардар катталгандан кийин кош келиңиз катын алат
- [ ] Жаңы сайт туура шаблон менен түзүлөт
- [ ] Кардар жаңы сайтына дароо кире алат
- [ ] Арзандатуу коддору туура иштейт

### Брендинг {#branding}

- [ ] Кирүү бетинде FitSite брендинги көрсөтүлөт
- [ ] Admin Dashboard FitSite брендингин көрсөтөт
- [ ] Бардык системалык каттар FitSite брендингин жана фитнеске тиешелүү тилди колдонот
- [ ] Эсеп-дүмүрчөктөр бизнес маалыматыңыз менен туура көрсөтүлөт
- [ ] Маркетинг сайты түз эфирде жана төлөм тариздөө формасына шилтеме берет

### Баштапкы тааныштыруу {#onboarding}

- [ ] Quick Start виджети жаңы кардар Dashboard-дарында көрүнөт
- [ ] Бардык Quick Start шилтемелери туура беттерге багыттайт
- [ ] Кош келиңиз каттар тизмеги жөндөлгөн жана текшерилген
- [ ] Билим базасынын макалалары жарыяланган жана жеткиликтүү
- [ ] Account бети туура план маалыматын жана жаңыртуу параметрлерин көрсөтөт

### Юридикалык жана бизнес {#legal-and-business}

- [ ] Кызмат көрсөтүү шарттары жарыяланган жана төлөм тариздөөдөн шилтемеленген
- [ ] Купуялык саясаты жарыяланган жана жеткиликтүү
- [ ] Кайтаруу саясаты аныкталган жана документтештирилген
- [ ] Төлөмдөрдү кабыл алуу үчүн бизнес субъекти түзүлгөн
- [ ] Төлөм шлюзу түз режимде (тест режиминде эмес)
- [ ] Салык жөндөөлөрү сиздин юрисдикцияңызга туура келет

## Жумшак ишке киргизүү жана катуу ишке киргизүү {#soft-launch-vs-hard-launch}

Эки баскычтуу ишке киргизүүнү карап көрүңүз:

### 1-баскыч: Жумшак ишке киргизүү {#phase-1-soft-launch}

Коомдук ишке киргизүүдөн мурун 5-10 фитнес студия ээлерин катталууга чакырыңыз. Булар сиздин бета кардарларыңыз. Төмөнкүлөрдүн ордуна аларга олуттуу арзандатуу сунуштаңыз (өмүр бою 50% арзандатуу же 3 ай акысыз):

- Катталуу жана баштапкы тааныштыруу тажрыйбасы боюнча чынчыл пикир
- Алардын сайтын көргөзмө мисалы катары колдонууга уруксат
- Алар туш болгон мүчүлүштүктөрдү же көйгөйлөрдү билдирүү

Бул коомчулукка ачылганга чейин чыныгы кардар пикирлерин жана көрсөтө турган түз сайттарды берет.

### 2-баскыч: Коомдук ишке киргизүү {#phase-2-public-launch}

Жумшак ишке киргизүүдөн алынган пикирлер киргизилгенден кийин:

- Төлөм шлюзун түз режимге которуңуз
- Маркетинг сайтыңызды жарыялаңыз
- Кардар тартууну баштаңыз (Сабак 11)
- Тиешелүү фитнес тармагынын каналдарында жарыялаңыз

## Ишке киргизүү күнүндөгү аракеттер {#launch-day-actions}

Коомчулукка чыккан күнү:

1. **Түз төлөмдөргө которуңуз** -- төлөм шлюзуңуздагы тест режимин өчүрүңүз
2. **Дагы бир жолу текшериңиз** -- чыныгы төлөм менен толук тесттик катталуу жасаңыз (андан кийин өзүңүзгө кайтарып бериңиз)
3. **Жакындан көзөмөлдөңүз** -- каталарды, ийгиликсиз катталууларды же төлөм көйгөйлөрүн байкаңыз
4. **Жеткиликтүү болуңуз** -- биринчи чыныгы кардарларыңызга жардам керек болушу мүмкүн, ал эми тез жооп ишеним жаратат
5. **Кыскача майрамдаңыз** -- анан кайра ишке киришиңиз

## Эмне туура эмес болушу мүмкүн {#what-can-go-wrong}

Төмөнкүлөргө даяр болуңуз:

- **Төлөм шлюзу көйгөйлөрү**: Шлюз провайдериңиздин колдоо байланышын даяр кармаңыз
- **SSL сертификаты көйгөйлөрү**: Сертификаттарды кантип текшерүү жана жаңыртуу керек экенин билиңиз
- **Кат жеткирүү мүчүлүштүктөрү**: Каттар чынында жетерин текшериңиз (спам папкаларын караңыз)
- **Жүктөм астындагы өндүрүмдүүлүк**: Трафик күтүүсүз көбөйсө, хостингиңизди кантип масштабдоо керек экенин билиңиз
- **Кардардын түшүнбөстүгү**: Билим базаңызды жана колдоо каналдарыңызды даярдаңыз

## Бул сабакта эмнени курдук {#what-we-built-this-lesson}

- **Комплекстүү ишке киргизүүгө чейинки текшерүү тизмеси** инфраструктураны, мазмунду, төлөмдөрдү жана юридикалык маселелерди камтыйт
- **Жумшак ишке киргизүү стратегиясы** коомчулукка чыгаардан мурун чыныгы пикир алуу үчүн
- **Ишке киргизүү күнүнүн жол-жоболору** ишеним менен түз эфирге чыгуу үчүн
- **Күтүлбөгөн кырдаалдар планы** ишке киргизүү күнүндөгү жалпы көйгөйлөр үчүн
