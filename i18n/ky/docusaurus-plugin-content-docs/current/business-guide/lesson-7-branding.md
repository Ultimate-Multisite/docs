# Сабак 7: Аны өзүңүздүкү кылуу {#lesson-7-making-it-yours}

Кардарларыңыз эч качан "кандайдыр бир WordPress plugin" колдонуп жаткандай сезбеши керек. Алар FitSite -- өз тармагына ылайык курулган платформа -- колдонуп жаткандай сезиши керек. Бул сабак брендингди, white-label кылууну жана платформаны чыныгы өнүмдөй сездирүүнү камтыйт.

## Кайсы жерден токтогонбуз {#where-we-left-off}

FitSite фитнес-студия ээлерин план тандоодон жандуу сайтка чейин алып барган иштеп турган төлөмгө өтүү агымына ээ. Эми биз бүт тажрыйбаны бирдиктүү, брендделген өнүмдөй сездиребиз.

## Платформаңыздын домени {#your-platform-domain}

Брендиңиздин негизи — домениңиз. FitSite үчүн:

- **Негизги домен**: `fitsite.com` (маркетинг сайтыңыз жана тармактын түпкү дареги)
- **Кардар сайттары**: `studioname.fitsite.com` (субдомендер)
- **Өз домендери**: Growth жана Pro пландарындагы кардарлар өз доменин байлай алышат

### Домениңизди жөндөө {#setting-up-your-domain}

1. Платформа домениңизди каттаңыз
2. Аны хостинг провайдериңизге багыттаңыз
3. Кардар субдомендери үчүн wildcard DNS (`*.fitsite.com`) конфигурациялаңыз
4. Wildcard SSL активдүү экенин текшериңиз

Толук нускамалар үчүн [Domain Mappingди кантип конфигурациялоо керек](/user-guide/domain-mapping/how-to-configure-domain-mapping) бөлүмүн караңыз.

## Админ тажрыйбасын white-label кылуу {#white-labeling-the-admin-experience}

Фитнес-студия ээси өз сайтынын Dashboardуна киргенде, WordPress же Ultimate Multisite брендин эмес, сиздин брендиңизди көрүшү керек.

### Ыңгайлаштырылган кирүү барагы {#custom-login-page}

WordPress кирүү барагын төмөнкүлөрдү көрсөтө тургандай ыңгайлаштырыңыз:

- FitSite логотибиңиз
- Фитнеске ылайыктуу фон сүрөттөрү
- Бренд түстөрүңүз

### Dashboard брендинги {#dashboard-branding}

Төмөнкүлөр үчүн [Admin Page Creator](/addons/admin-page-creator) кошумчасын же ыңгайлаштырылган CSS колдонуңуз:

- WordPress логотибин FitSite логотибиңизге алмаштыруу
- Админ түстөр схемасын брендиңизге ылайыктоо
- Фитнеске тиешелүү тез шилтемелер жана жардам ресурстары бар ыңгайлаштырылган Dashboard виджетин кошуу

### Ыңгайлаштырылган админ барактар {#custom-admin-pages}

Фитнес-студия ээлери үчүн эң маанилүү аракеттерди көрсөтө турган ыңгайлаштырылган админ барактарды түзүүнү карап көрүңүз:

- "Сабактар графигиңизди түзөтүңүз"
- "Машыктыруучу профилдерин жаңыртыңыз"
- "Сайтыңызды көрүңүз"

Бул нишага тиешелүү аракеттерди стандарттуу WordPress менюсунун ичине жашырбай, алдыга жана борборго коюу аркылуу үйрөнүү ийри сызыгын азайтат.

## Байланыш билдирүүлөрүңүздү бренддөө {#branding-your-communications}

Ар бир email, эсеп-фактура жана билдирме брендиңизди бекемдеши керек.

### Тутумдук emailдер {#system-emails}

**Ultimate Multisite > Settings > Emails** бөлүмүнө өтүп, бардык тутумдук emailдерди ыңгайлаштырыңыз:

- **Жөнөтүүчү аты**: FitSite
- **Жөнөтүүчү emailи**: hello@fitsite.com
- **Email шаблондору**: Бренд түстөрүңүздү жана логотибиңизди колдонуңуз
- **Тил**: Бүтүндөй фитнеске багытталган болсун

Ыңгайлаштырууга маанилүү emailдер:

| Email | Жалпы нуска | FitSite нускасы |
|-------|----------------|-----------------|
| Кош келиңиз | "Жаңы сайтыңыз даяр" | "Фитнес-студияңыздын веб-сайты жанданды" |
| Төлөм дүмүрчөгү | "Төлөм кабыл алынды" | "FitSite жазылуу төлөмү ырасталды" |
| Сыноо мөөнөтү аяктоодо | "Сыноо мөөнөтүңүз жакында аяктайт" | "FitSite сыноо мөөнөтүңүз 3 күндөн кийин аяктайт -- студияңыздын веб-сайтын жандуу бойдон калтырыңыз" |

### Эсеп-фактуралар {#invoices}

Эсеп-фактура шаблондорун төмөнкүлөр менен ыңгайлаштырыңыз:

- FitSite логотибиңиз жана бренд түстөрүңүз
- Бизнес маалыматтарыңыз
- Фитнеске тиешелүү өнүм аталыштары (жалпы план IDлери эмес)

## Кардарга көрүнгөн сайт {#the-customer-facing-site}

Негизги домениңизге (`fitsite.com`) платформаны саткан маркетинг сайты керек. Бул Ultimate Multisite тармак админинен өзүнчө -- бизнесиңиздин коомчулукка көрүнгөн жүзү.

Негизги барактар:

- **Башкы бет**: Фитнес бизнеси үчүн так баалуулук сунушу
- **Өзгөчөлүктөр**: FitSite фитнес тили менен эмне кылат
- **Баалар**: Нишага тиешелүү өзгөчөлүк салыштыруулары менен үч планыңыз
- **Мисалдар**: Платформада курулган сайттарды көрсөтүү
- **Катталуу**: Төлөмгө өтүү формаңызга шилтемелер

:::tip Маркетинг сайтыңыз тармак сайты боло алат
Маркетинг сайтыңызды өз тармагыңыздын ичиндеги сайт катары түзүңүз. Бул аны ошол эле Dashboard аркылуу башкарууга мүмкүндүк берет жана өз платформаңыздын мүмкүнчүлүктөрүн көрсөтөт.
:::

## Кардарлар үчүн өз домени {#custom-domain-for-customers}

Өз домендерин камтыган пландардагы кардарлар үчүн процессти так документтештириңиз:

1. Кардар доменин каттайт же регистраторго өткөрөт
2. Кардар DNSти платформаңызга багыттай тургандай жаңыртат (так жазууларды бериңиз)
3. Ultimate Multisite домен байлоосун жана SSLди иштетет

Бул процесс үчүн техникалык эмес фитнес-студия ээлерине арналып жазылган жардам макаласын же билим базасынын жазуусун түзүңүз.

## Азыркыга чейин FitSite тармагы {#the-fitsite-network-so-far}

```
FitSite тармагы
├── WordPress Multisite (субдомен режими)
├── Ultimate Multisite (конфигурацияланган + брендделген)
├── Платформа домени (fitsite.com + wildcard SSL)
├── Сайт шаблондору (Studio Essential, Gym Pro, Fitness Chain)
├── Өнүмдөр (Starter, Growth, Pro + Order Bumps)
├── Төлөмгө өтүү агымы (нишага тиешелүү, текшерилген)
├── Брендинг
│   ├── Ыңгайлаштырылган кирүү барагы
│   ├── Брендделген админ Dashboard
│   ├── Нишага тиешелүү тутумдук emailдер
│   ├── Брендделген эсеп-фактуралар
│   └── fitsite.com дарегиндеги маркетинг сайты
└── Onboarding агымына даяр (кийинки сабак)
```

## Бул сабакта эмнени курдук {#what-we-built-this-lesson}

- **Платформа домени жана DNS** брендделген тажрыйба үчүн конфигурацияланды
- **White-labeled админ** бүтүндөй FitSite брендинги менен
- **Ыңгайлаштырылган байланыш билдирүүлөрү** -- emailдер, эсеп-фактуралар жана билдирмелердин баары брендге ылайык
- **Маркетинг сайты** FitSiteти фитнес-студия ээлерине сатат
- **Өз домени боюнча документация** өз доменин каалаган кардарлар үчүн
