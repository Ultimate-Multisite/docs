# PayFast {#payfast}

Multisite Ultimate: Payfast Gateway

Payfast'ты — Түштүк Африкадагы тез жана коопсуз онлайн төлөмдөр шлюзун — Multisite Ultimate төлөм тандоосу катары кошот.

## Орнотуу {#installation}

1. 'ultimate-multisite-payfast' файлын '/wp-content/plugins/' каталогуна жүктөңүз
2. Плагинди WordPress ичиндеги 'Плагиндер' менюсу аркылуу активдештириңиз
3. Плагинди жөндөө үчүн кадам-кадам Wizard'ды аткарыңыз

## Колдонуу колдонмосу {#usage-guide}

PayFast — Түштүк Африкадагы бизнестер үчүн алдыңкы онлайн төлөм иштетүүчү. Ал онлайн сатуучуларга транзакцияларды үзгүлтүксүз жана коопсуз кылуу аркылуу өсүүгө жардам берүүгө арналган. Ultimate Multisite колдонуучусунун өтүнүчүнө ылайык, биз PayFast менен интеграцияны мүмкүн кылган кошумча түзүүнү чечтик. Бул макала Ultimate Multisite тармагыңызда төлөм алуу үчүн PayFast'ты төлөм шлюзу катары кантип жөндөөнү көрсөтөт.

Биринчи кадам — кошумчаны орнотуу.

# Payfast кошумчасын кантип орнотуу керек: {#how-to-install-the-payfast-add-on}

Кошумчалар барагына өтүп, Ultimate Multisite: Payfast Gateway кошумчасын тандаңыз.

![Кошумчалар барагы](/img/admin/addons-page.png)

Кошумча терезеси ачылат. Install Now баскычын басыңыз.

Орнотуу аяктагандан кийин, сиз плагиндер барагына багытталасыз. Бул жерден Network Activate баскычын гана басыңыз, ошондо _Payfast_ кошумчасы тармагыңызда активдештирилет.

# PayFast'ты жөндөө {#setting-up-payfast}

Кошумчаны орноткондон кийин, эми Ultimate Multisite жөндөөлөрүнө өтүп, PayFast'ты конфигурациялай баштасаңыз болот.

**Ultimate Multisite жөндөөлөрү** барагынын астынан "Төлөмдөр" өтмөгүн таба аласыз. Бул өтмөктүн астында бул төлөм шлюзун иштетүү же өчүрүү үчүн которгуч баскыч менен бирге "**Payfast** " болушу керек.

![Төлөм шлюзунун жөндөөлөрү](/img/config/settings-payment-gateways.png)Алга жылып, аны иштетүү үчүн которгуч баскычты басыңыз. Андан кийин куту кеңейип, толтурушуңуз керек болгон ар кандай талааларды көрсөтөт, мисалы **Merchant ID, Merchant Key жана PassPhrase.** Бул маалыматты **Payfast** Account'уңуздан ала аласыз

<!-- Скриншот жеткиликтүү эмес: PayFast кеңейтилген жөндөөлөрүндө Merchant ID, Merchant Key жана PassPhrase талаалары көрсөтүлгөн -->

Кийинки кадам — керектүү маалыматты алуу үчүн PayFast Account'уңузга кирүү.

Киргенден кийин, керектүү дайындарды дароо **Payfast** Account Dashboard'уңуздан көрө аласыз.

<!-- Скриншот жеткиликтүү эмес: PayFast Account Dashboard'унда Merchant ID жана Merchant Key дайындары көрсөтүлгөн -->Дайындарды жөн эле көчүрүп, чаптап, Ultimate Multisite жөндөөлөрүңүздөгү маалыматтарды толтурсаңыз болот. Андан кийин өзгөртүүлөрдү сактоо үчүн **"Save settings** " баскычын басыңыз.

<!-- Скриншот жеткиликтүү эмес: Ultimate Multisite PayFast жөндөөлөрүндө дайындар толтурулган жана Save баскычы көрсөтүлгөн -->

Болду, PayFast төлөм интеграцияңыз аяктады!
