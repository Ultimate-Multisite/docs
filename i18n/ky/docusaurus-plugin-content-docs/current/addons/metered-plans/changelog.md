# Metered Plans өзгөртүүлөр журналы {#metered-plans-changelog}

Версия 1.1.0 - 2026-05-05 күнү чыгарылды
- Жаңы: multisite кошумча сайттары үчүн AI token эсептөө — кардар сайттары боюнча AI token колдонулушун көзөмөлдөп жана эсептеп, ар бир token үчүн конфигурациялануучу тарифтерди колдонуңуз
- Жаңы: Connector мажбурлоосу динамикалык чекти аныктоо жана түз жазуу менен кайра курулду, бардык connector боюнча реалдуу убакыттагы тактыкты камсыздайт
- Оңдоо: Database таблица жаңыртуулары эми BerlinDB схема аныктамалары менен туура шайкеш келтирилди, жаңы орнотуулардагы жаңыртуу каталарын алдын алат
- Оңдоо: Database жаңыртуу callback`тери туура форматка өткөрүлдү, байкалбай калган жаңыртуу каталарын чечет
- Оңдоо: Бөлчөк маанилер эми AI Usage Overage Markup киргизүү талаасында кабыл алынат
- Оңдоо: plugin жүктөлгөндө fatal error`лор жана кош инициализация маселелери чечилди
- Жакшыртылды: developer чөйрөлөрү биринчи иштетүүдө өзүн-өзү конфигурациялашы үчүн check-env npm script кошулду
