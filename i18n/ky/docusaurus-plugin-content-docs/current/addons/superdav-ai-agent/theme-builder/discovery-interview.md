# Ачылыш маеги

**Ачылыш маеги** — Theme Builder'дин дизайн процессиндеги биринчи кадам. Ал жекелештирилген тема түзүү үчүн сиздин сайт, бизнес жана дизайн артыкчылыктарыңыз тууралуу маалымат чогултат.

## Жалпы маалымат

Ачылыш маеги брендиңизди, максаттуу аудиторияңызды жана дизайн максаттарыңызды түшүнүү үчүн сизди бир катар суроолор аркылуу жетектейт. 1.16.0 версиясында маек эми дизайныңыз үчүн визуалдык илхам берүүгө мүмкүндүк берген **сүрөт жүктөө кадамын** камтыйт.

## Маек кадамдары

### 1. Бизнес маалыматы

- Сайттын аталышы жана урааны
- Бизнес түрү же тармагы
- Негизги бизнес максаттары
- Максаттуу аудиториянын сүрөттөмөсү

### 2. Дизайн артыкчылыктары

- Артык көрүлгөн түстөр схемасы
- Дизайн стили (заманбап, классикалык, минималисттик, оюнкараак ж.б.)
- Илхам берген веб-сайттар же бренддер
- Милдеттүү дизайн элементтери

### 3. Сүрөт жүктөө (v1.16.0 версиясында жаңы)

Дизайн багытына таасир берүү үчүн бир же бир нече сүрөт жүктөңүз:

#### Кабыл алынган форматтар

| Формат | Кеңейтүү | Чоо-жайы |
|--------|-----------|---------|
| **JPEG** | `.jpg`, `.jpeg` | Фотосүрөттөр үчүн сунушталат |
| **PNG** | `.png` | Тунук фонду колдойт |
| **WebP** | `.webp` | Заманбап формат, файл өлчөмү кичирээк |
| **GIF** | `.gif` | Статикалык сүрөттөр гана (анимация жок) |

#### Файл талаптары

- **Файлдын эң чоң өлчөмү:** 10 MB
- **Эң кичине өлчөмдөр:** 400 × 300 пиксел
- **Сунушталган өлчөмдөр:** 1200 × 800 пиксел же андан чоң
- **Тараптар катышы:** Каалаган катыш кабыл алынат

#### Сүрөттөр дизайн багытына кантип таасир этет

Сиз жүктөгөн сүрөттөр Theme Builder'ге төмөнкүлөрдү түшүнүүгө жардам берет:

- **Түс палитрасы** — басымдуу түстөр алынат жана темада колдонулат
- **Визуалдык стиль** — заманбап, рустик, минималисттик, жандуу ж.б.
- **Маанай жана атмосфера** — кесипкөй, чыгармачыл, достук, люкс ж.б.
- **Дизайн элементтери** — оймо-чиймелер, текстуралар жана композициялык стилдер
- **Типографиялык илхам** — визуалдык темага шайкеш келген шрифт стилдери

### 4. Контент түзүмү

- Керектүү негизги барактар (Башкы бет, Биз жөнүндө, Кызматтар, Байланыш ж.б.)
- Блог же жаңылыктар бөлүмү керекпи
- Электрондук соода функционалдуулугу керекпи
- Өзгөчө функциялар же интеграциялар

### 5. Карап чыгуу жана ырастоо

- Берилген бардык маалыматтын кыскача жыйындысы
- Дизайн түзүү башталардан мурун ырастоо

## Сүрөт жүктөө боюнча мыкты тажрыйбалар

### Эмне жүктөө керек

- **Бренд илхамы** — логотиптер, түс үлгүлөрү же бренд көрсөтмөлөрү
- **Атаандаштардын сайттары** — сизге жаккан сайттардын скриншоттору
- **Маанай такталары** — каалаган эстетикаңызды чагылдырган сүрөттөр жыйнагы
- **Продукт сүрөттөрү** — көрсөтө турган продукттарыңыз болсо
- **Команда же кеңсе сүрөттөрү** — жеке же кесипкөй брендинг үчүн
- **Жашоо образы сүрөттөрү** — максаттуу аудиторияңызды чагылдырган

### Эмнеден качуу керек

- **Сапаты төмөн сүрөттөр** — бүдөмүк же пикселденген сүрөттөр
- **Байланышсыз сүрөттөр** — брендиңизди чагылдырбаган сүрөттөр
- **Тексти көп сүрөттөр** — тексти көп скриншоттор
- **Автордук укук менен корголгон материал** — сүрөттөрдү колдонууга укугуңуз бар экенин текшериңиз
- **Анимацияланган GIF'тер** — статикалык сүрөттөр гана иштетилет

### Эң жакшы натыйжалар үчүн кеңештер

1. **2-3 сүрөт жүктөңүз** — бир нече сүрөт жакшыраак контекст берет
2. **Жогорку сапаттагы сүрөттөрдү колдонуңуз** — так, жакшы жарыктанган сүрөттөр эң жакшы иштейт
3. **Ар түрдүүлүктү кошуңуз** — түс, стиль жана тема боюнча аралашма
4. **Контекст бериңиз** — ар бир сүрөттү эмне үчүн тандаганыңызды кыскача түшүндүрүңүз
5. **Так болуңуз** — эгер белгилүү бир элемент жакса, аны көрсөтүңүз

## Дизайн багытынын жыйынтыгы

Ачылыш маегинен кийин Theme Builder төмөнкүлөрдү түзөт:

- **Түс палитрасы** — негизги, экинчи даражадагы жана акцент түстөр
- **Типография** — баш аттар жана негизги текст үчүн шрифт үй-бүлөлөрү
- **Макет стили** — аралык, тегиздөө жана компоненттердин жайгашуусу
- **Визуалдык элементтер** — иконкалар, оймо-чиймелер жана кооздоочу элементтер
- **Маанай тактасы** — дизайн багытынын визуалдык чагылдырылышы

## Кийинки кадамдар

Ачылыш маегин аяктагандан кийин:

1. Түзүлгөн дизайн багытын карап чыгыңыз
2. Пикир калтырыңыз же түзөтүүлөрдү сураңыз
3. [Дизайн багыты](./design-direction.md) кадамына өтүңүз
4. Theme Builder сиздин ыңгайлаштырылган темаңызды түзөт
