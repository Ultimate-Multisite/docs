# Куралдар жана конфигурация

Theme Builder сиздин WordPress сайтыңызды талдоо, долбоорлоо жана куруу үчүн куралдар топтомун колдонот. 1.16.0 версиясында **sd-ai-agent/site-scrape эми 1-деңгээлдеги курал**, ошондуктан ал Theme Builder ичинде демейки боюнча жеткиликтүү. Superdav AI Agent v1.18.0 ишенимдүү иштеп чыгуучу иш агымдары үчүн өзүнчө таратылган Advanced коштоочу топтомун да киргизет; ал куралдар WordPress.org негизги топтомунун бөлүгү эмес жана өзүнчө орнотулуп, уруксат берилиши керек.

## Жеткиликтүү куралдар

### 1-деңгээлдеги куралдар (дайыма жеткиликтүү)

1-деңгээлдеги куралдар эч кандай кошумча конфигурациясыз Theme Builder ичинде демейки боюнча жеткиликтүү.

#### sd-ai-agent/site-scrape

**Максаты:** Дизайн илхамын, мазмун түзүмүн жана функционалдуулукту алуу үчүн бар веб-сайттарды талдоо.

**Мүмкүнчүлүктөрү:**

- **Веб-сайтты талдоо** — атаандаш же илхам берүүчү веб-сайттарды чогултуп талдоо
- **Дизайнды алуу** — түстөрдү, шрифттерди жана жайгашуу үлгүлөрүн аныктоо
- **Мазмун түзүмү** — барак уюштуруусун жана иерархиясын түшүнүү
- **Функцияларды аныктоо** — плагиндерди жана функционалдуулукту аныктоо
- **Өндүрүмдүүлүктү талдоо** — барак ылдамдыгын жана оптималдаштырууну текшерүү
- **SEO талдоосу** — мета тегдерди жана түзүмдөлгөн маалыматтарды карап чыгуу

**Колдонуу:**

```
example.com дизайнын талдап, аны менин сайтым үчүн илхам катары колдон.
```

**Ал эмнелерди алат:**

- Түстөр палитрасы жана түстөрдү колдонуу
- Типография (шрифттер жана өлчөмдөр)
- Жайгашуу түзүмү жана аралык коюу
- Навигация үлгүлөрү
- Мазмунду уюштуруу
- Сүрөттөрдү жана медианы колдонуу
- Интерактивдүү элементтер
- Мобилдик ыңгайлашуучулук

**Чектөөлөрү:**

- Сырсөз менен корголгон сайттарды чогулта албайт
- robots.txt жана сайт саясаттарын сактайт
- Динамикалык мазмунду толук кармабай калышы мүмкүн
- JavaScript көп колдонулган сайттарда талдоо чектелүү болушу мүмкүн
- Чоң сайттарды талдоо көбүрөөк убакыт алышы мүмкүн

### 2-деңгээлдеги куралдар (милдеттүү эмес)

2-деңгээлдеги куралдар Theme Builder жөндөөлөрүндө ачык иштетилгенде жеткиликтүү болот.

#### Өркүндөтүлгөн аналитика

Деталдуу өндүрүмдүүлүк көрсөткүчтөрүн берет:

- Барактын жүктөлүү убакыттары
- Core Web Vitals
- SEO упайы
- Жеткиликтүүлүк упайы
- Мыкты тажрыйбалар упайы

#### Мазмун оптималдаштыргычы

Төмөнкүлөр үчүн жакшыртууларды талдайт жана сунуштайт:

- Окумдуулук
- SEO оптималдаштыруу
- Ачкыч сөздөрдү колдонуу
- Мазмун түзүмү
- Аракетке чакырууну жайгаштыруу

### 3-деңгээлдеги куралдар (Premium)

3-деңгээлдеги куралдар кошумча конфигурацияны же premium функцияларды талап кылат.

#### AI мазмун генератору

Төмөнкүлөр үчүн мазмун түзөт:

- Товар сүрөттөмөлөрү
- Кызмат барактары
- Блог жазуулары
- Мета сүрөттөмөлөр
- Аракетке чакыруу тексти

#### Өркүндөтүлгөн дизайн куралдары

Өркүндөтүлгөн дизайн мүмкүнчүлүктөрүн берет:

- Ыңгайлаштырылган CSS түзүү
- Анимация түзүү
- Интерактивдүү элемент дизайны
- Өркүндөтүлгөн түс теориясы
- Типографияны оптималдаштыруу

### Advanced коштоочу куралдары (өзүнчө топтом)

Advanced коштоочу топтому негизги Superdav AI Agent чыгарылышынан өзүнчө таратылат. Ал администраторлор агентке жогорку таасирдүү куралдарды колдонууга ачык уруксат берген ишенимдүү иштеп чыгуучу жана сайт ээси чөйрөлөрү үчүн арналган.

Advanced коштоочу мүмкүнчүлүктөрү төмөнкүлөрдү камтышы мүмкүн:

- **Файл тутуму куралдары** — ишенимдүү чөйрөлөрдө бекитилген файлдарды текшерүү жана өзгөртүү
- **Маалымат базасы куралдары** — ачык уруксат берилгенде сайт маалыматтарын сурап алуу же жаңыртуу
- **WP-CLI куралдары** — WordPress тейлөө жана текшерүү буйруктарын иштетүү
- **REST диспетчер куралдары** — көзөмөлдөнгөн диспетчер иш агымдары аркылуу катталган REST endpointтерин чакыруу
- **Плагин куруучу куралдар** — плагин кодун каркас кылып түзүү жана кайталап өркүндөтүү
- **Git snapshot куралдары** — кооптуу иштеп чыгуучу операцияларынан мурун snapshot түзүү
- **Колдонуучу башкаруу куралдары** — уруксат берилген жерлерде колдонуучу жазууларын карап чыгууга же тууралоого жардам берүү
- **Benchmark куралдары** — иштеп чыгуудагы кароо үчүн өндүрүмдүүлүк же мүмкүнчүлүк benchmarkтарын иштетүү

Булардын ар бир орнотуу үчүн жалпысынан жеткиликтүү Theme Builder куралдары катары документтештирбеңиз. Алар Advanced коштоочу топтому орнотулуп, иштетилип жана ишенимдүү администраторлор менен чектелгенде гана жеткиликтүү.

## Курал конфигурациясы

### Куралдарды иштетүү

Theme Builder ичинде кошумча куралдарды иштетүү үчүн:

1. **Gratis AI Agent → Settings** бөлүмүнө өтүңүз
2. **Theme Builder → Tools** бөлүмүнө өтүңүз
3. Керек болгондо куралдарды күйгүзүп/өчүрүңүз
4. Жөндөөлөрдү сактаңыз

### Курал уруксаттары

Айрым куралдар уруксаттарды талап кылат:

- **Сайтты чогултуу** — интернетке жетүүнү талап кылат
- **Аналитика** — Google Analytics интеграциясын талап кылат
- **Мазмун түзүү** — API ачкычтарын талап кылат
- **Өркүндөтүлгөн функциялар** — premium жазылууну талап кылышы мүмкүн
- **Advanced коштоочу куралдары** — өзүнчө таратылган Advanced топтомун жана администратордун ачык ишенимин талап кылат

### API ачкычтары жана эсептик маалыматтар

Аларды талап кылган куралдар үчүн API ачкычтарын конфигурациялаңыз:

1. **Gratis AI Agent → Settings → API Keys** бөлүмүнө өтүңүз
2. Ар бир курал үчүн эсептик маалыматтарды киргизиңиз
3. Байланышты сынаңыз
4. Коопсуз сактаңыз

### Advanced коштоочу топтомун орнотуу

Advanced коштоочу топтому WordPress.org негизги ZIP файлынан өзүнчө чыгарылат. Аны чыгарылыш үчүн расмий долбоор таратуу каналынан гана орнотуңуз, андан кийин жетүүнү ишенимдүү администраторлор менен чектеңиз. Өндүрүштүк сайтта файл тутуму, маалымат базасы, WP-CLI, REST диспетчер, плагин куруучу, колдонуучу башкаруу же benchmark куралдарын иштетүүдөн мурун адам бекитүү дарбазаларын жана git snapshot жүрүм-турумун карап чыгыңыз.

## sd-ai-agent/site-scrape колдонуу

### Негизги колдонуу

Theme Builderден веб-сайтты талдоону сураңыз:

```
competitor.com дарегиндеги атаандашымдын сайтынын дизайнын талдап,
менин сайтым үчүн дизайн жакшыртууларын сунушта.
```

### Белгилүү бир талдоо

Белгилүү талдоо түрлөрүн сураңыз:

```
example.com сайтынан түстөр палитрасын алып, аны илхам катары колдон.
```

```
example.com сайтынын навигация түзүмүн талдап, менин сайтыма окшош
уюштурууну колдон.
```

```
example.com сайтынын мобилдик ыңгайлашуучулугун текшерип, менин сайтым
ошондой эле ыңгайлашуучу экенин камсыз кыл.
```

### Салыштыруу талдоосу

Бир нече сайтты салыштырыңыз:

```
site1.com жана site2.com дизайндарын салыштырып, экөөнүн тең эң мыкты
элементтерин бириктирген дизайн түз.
```

## Курал чектөөлөрү жана эске алуулар

### Ылдамдыкты чектөө

- Scraping серверге ашыкча жүк түшүрбөө үчүн ылдамдык боюнча чектелет
- Бир эле сайтка бир нече суроо-талап жөнөтүлсө, чектелиши мүмкүн
- Чоң сайттарды талдоо көбүрөөк убакыт алышы мүмкүн

### Мазмундун тактыгы

- Динамикалык мазмун толук камтылбай калышы мүмкүн
- JavaScript аркылуу көрсөтүлгөн мазмун толук болбошу мүмкүн
- Чыныгы убакыттагы маалыматтар чагылдырылбай калышы мүмкүн
- Айрым мазмун төлөм дубалынын артында болушу мүмкүн

### Укуктук жана этикалык жагдайлар

- robots.txt жана сайт саясаттарын урматтаңыз
- Автордук укук менен корголгон мазмунду кайра чыгаруу үчүн Scraping кылбаңыз
- Талдоону көчүрүү үчүн эмес, шыктануу үчүн колдонуңуз
- Алынган мазмунду колдонууга укугуңуз бар экенин текшериңиз
- Талданган сайттардын кызмат көрсөтүү шарттарын сактаңыз

### Өндүрүмдүүлүккө таасири

- Чоң сайттарды Scraping кылуу убакыт алышы мүмкүн
- Бир убакта бир нече Scraping жайыраак болушу мүмкүн
- Тармак туташуусу ылдамдыкка таасир этет
- Чоң маалымат топтомдору көбүрөөк иштетүүнү талап кылышы мүмкүн

## Мыкты практикалар

### Сайт талдоосун колдонуу

1. **Бир нече сайтты талдаңыз** — бир нече булактан шыктануу алыңыз
2. **Түзүмгө көңүл буруңуз** — жайгашуу жана уюштуруу үлгүлөрүн үйрөнүңүз
3. **Түстөрдү чыгарып алыңыз** — түс палитраларын баштапкы чекит катары колдонуңуз
4. **Типографияны изилдеңиз** — өзүңүзгө жаккан шрифт айкалыштарын аныктаңыз
5. **Навигацияны карап чыгыңыз** — меню уюштурулушун түшүнүңүз

### Этикалык Scraping

1. **robots.txtти урматтаңыз** — сайт көрсөтмөлөрүн аткарыңыз
2. **Мазмунду көчүрбөңүз** — талдоону шыктануу үчүн гана колдонуңуз
3. **Укуктарды текшериңиз** — алынган мазмунду колдоно аларыңызды камсыздаңыз
4. **Булактарды көрсөтүңүз** — шыктануу булактарына шилтеме бериңиз
5. **Шарттарды сактаңыз** — сайттын кызмат көрсөтүү шарттарына ылайык иш кылыңыз

### Куралдын натыйжалуулугун максималдаштыруу

1. **Так болуңуз** — талдоонун белгилүү түрлөрүн сураңыз
2. **Контекст бериңиз** — сайтыңыздын максатын түшүндүрүңүз
3. **Күтүүлөрдү коюңуз** — дизайн максаттарыңызды сүрөттөңүз
4. **Кайталап жакшыртыңыз** — жыйынтыктарга негизделип тактаңыз
5. **Куралдарды айкалыштырыңыз** — ар тараптуу талдоо үчүн бир нече куралды колдонуңуз

## Көйгөйлөрдү чечүү

### Сайт Scraping кылынбай жатат

- Сайт жалпыга жеткиликтүү экенин текшериңиз
- robots.txt Scraping кылууга уруксат берерин текшериңиз
- Башка сайтты колдонуп көрүңүз
- Интернет туташуусун текшериңиз
- Маселе улана берсе, колдоо кызматына кайрылыңыз

### Толук эмес талдоо

- Сайтта динамикалык мазмун болушу мүмкүн
- JavaScript толук көрсөтүлбөшү мүмкүн
- Чоң сайттарда убакыт бүтүп калышы мүмкүн
- Анын ордуна белгилүү барактарды талдап көрүңүз
- Талдоонун белгилүү түрлөрүн сураңыз

### Жай өндүрүмдүүлүк

- Чоң сайттарды талдоо көбүрөөк убакыт алат
- Бир убакта бир нече Scraping жайыраак болот
- Тармак туташуусу ылдамдыкка таасир этет
- Жүктөм аз болгон убакта колдонуп көрүңүз
- Алгач кичирээк бөлүмдөрдү талдаңыз
