# Меймандостук менюлары

**Меймандостук менюлары** функциясы Theme Builder’ге түзүмдөлгөн тамак-аш жана суусундук меню барактарын түзүп, аларды түздөн-түз WordPress сайтыңызга жайгаштырууга мүмкүндүк берет.

## Жалпы маалымат {#overview}

Theme Builder эми ресторандар, кафелер, барлар жана кейтеринг кызматтары сыяктуу меймандостук бизнеси үчүн кесипкөй, түзүмдөлгөн меню барактарын түзө алат. Бул менюлар сайтыңыздын дизайнына толук интеграцияланат жана аларды оңой жаңыртып, башкарса болот.

## Колдоого алынган меймандостук түрлөрү {#supported-hospitality-types}

- **Ресторандар** — толук тейлөөлүү тамактануу менюлары
- **Кафелер** — кофе жана жеңил тамак менюлары
- **Барлар жана лаунждар** — суусундук жана жеңил закуска менюлары
- **Нан бышыруучу жайлар** — кондитердик азыктар жана нан менюлары
- **Кейтеринг кызматтары** — иш-чара меню варианттары
- **Фудтрактар** — көчмө тамак-аш тейлөө менюлары
- **Сыра жана шарап өндүрүү жайлары** — сүрөттөмөлөрү бар суусундук менюлары

## Меню түзүмү {#menu-structure}

### Меню категориялары {#menu-categories}

Менюлар категорияларга уюштурулат:

- **Закускалар** — баштапкы жана чакан порциядагы тамактар
- **Негизги тамактар** — негизги курстар
- **Кошумчалар** — гарнирлер жана жашылчалар
- **Десерттер** — таттуу даамдар
- **Суусундуктар** — ичимдиктер (алкоголдук жана алкоголсуз)
- **Атайын сунуштар** — күнүмдүк же сезондук атайын сунуштар

### Меню пунктунун форматы {#menu-item-format}

Ар бир меню пункту төмөнкүлөрдү камтыйт:

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### Пункт талаалары {#item-fields}

| Талаа | Түрү | Сүрөттөмө |
|-------|------|-------------|
| `name` | string | Тамактын же суусундуктун аталышы |
| `description` | string | Пункттун кеңири сүрөттөмөсү |
| `price` | string | Баа (валюта менен форматталган) |
| `dietary_info` | array | Тамактануу атрибуттары (Веган, Глютенсиз ж.б.) |
| `allergens` | array | Кеңири таралган аллергендер (Жаңгактар, Моллюскалар ж.б.) |
| `availability` | string | Качан жеткиликтүү (Күн сайын, Сезондук ж.б.) |

## Меймандостук менюларын түзүү {#creating-hospitality-menus}

### 1-кадам: Меню маалыматын бериңиз {#step-1-provide-menu-information}

Theme Builder’ге менюңуз тууралуу айтыңыз:

```
Италия рестораным үчүн закускалар, паста тамактары, негизги тамактар
жана десерттер камтылган ресторан менюсун түзүңүз. Бааларды жана сүрөттөмөлөрдү кошуңуз.
```

### 2-кадам: Theme Builder меню түзөт {#step-2-theme-builder-generates-menu}

Theme Builder:

1. Түзүмдөлгөн меню барагын түзөт
2. Аны сайтыңыздын темасына ылайык дизайндайт
3. Пункттарды категорияларга уюштурат
4. Бааларды жана сүрөттөмөлөрдү форматтайт
5. Тамактануу жана аллерген тууралуу маалыматты кошот

### 3-кадам: Карап чыгып, ыңгайлаштырыңыз {#step-3-review-and-customize}

Сиз төмөнкүлөрдү кыла аласыз:

1. Меню пункттарын жана бааларды түзөтүү
2. Категорияларды кошуу же алып салуу
3. Категориялардын ичиндеги пункттардын тартибин өзгөртүү
4. Сүрөттөмөлөрдү жана тамактануу маалыматын жаңыртуу
5. Стилдөө жана жайгашууну тууралоо

## Меню көрсөтүү варианттары {#menu-display-options}

### Толук меню барагы {#full-menu-page}

Толук менюңузду көрсөткөн өзүнчө барак:

- Категория боюнча уюштурулган
- Издөөгө жана чыпкалоого болот
- Басып чыгарууга ыңгайлуу жайгашуу
- Мобилдик түзмөктөргө ылайыкташкан дизайн

### Меню виджети {#menu-widget}

Меню бөлүмдөрүн башка барактарга жайгаштырыңыз:

- Башкы беттеги тандалган пункттар
- Каптал тилкесиндеги күнүмдүк атайын сунуштар
- Бар барагындагы суусундук менюсу
- Футердеги десерт көргөзмөсү

### Меню PDF {#menu-pdf}

Жүктөп алууга боло турган PDF меню түзүңүз:

- Кесипкөй форматтоо
- Басып чыгарууга даяр сапат
- Сүрөттөрдү жана сүрөттөмөлөрдү камтыйт
- Бөлүшүүгө жана email аркылуу жөнөтүүгө оңой

## Тамактануу жана аллерген тууралуу маалымат {#dietary-and-allergen-information}

### Тамактануу атрибуттары {#dietary-attributes}

Пункттарды тамактануу маалыматы менен белгилеңиз:

- **Веган** — жаныбарлардан алынган азыктар жок
- **Вегетариандык** — эт жок
- **Глютенсиз** — целиакия оорусу барлар үчүн коопсуз
- **Сүт азыктары жок** — сүт азыктары жок
- **Жаңгаксыз** — дарак жаңгактары же арахис жок
- **Карбонгидраты аз** — карбонгидраттар азайтылган
- **Белогу көп** — белокко бай

### Аллерген эскертүүлөрү {#allergen-warnings}

Кеңири таралган аллергендерди кошуңуз:

- **Жаңгактар** — дарак жаңгактары жана арахис
- **Моллюскалар** — рак сымалдар жана моллюскалар
- **Балык** — бардык балык түрлөрү
- **Сүт азыктары** — сүт жана сүт азыктары
- **Жумуртка** — тоок жумурткалары
- **Соя** — соя азыктары
- **Глютен** — буудай жана глютен камтыган дан өсүмдүктөрү
- **Күнжүт** — күнжүт уруктары жана майы

## Менюну башкаруу {#menu-management}

### Бааларды жаңыртуу {#updating-prices}

Меню бааларын оңой жаңыртыңыз:

1. Меню барагына өтүңүз
2. "Менюну түзөтүү" дегенди басыңыз
3. Пункттардын бааларын жаңыртыңыз
4. Өзгөртүүлөрдү сактаңыз
5. Өзгөртүүлөр сайтыңызда дароо көрүнөт

### Сезондук пункттарды кошуу {#adding-seasonal-items}

Сезондук меню варианттарын түзүңүз:

1. Жаңы меню версиясын түзүңүз
2. Сезондук пункттарды кошуңуз
3. Пункттарды "Сезондук" деп белгилеңиз
4. Жеткиликтүүлүк даталарын пландаштырыңыз
5. Сезон учурунда автоматтык түрдө көрсөтүңүз

### Атайын сунуштарды башкаруу {#managing-specials}

Күнүмдүк же жумалык атайын сунуштарды көрсөтүңүз:

1. "Атайын сунуштар" категориясын түзүңүз
2. Жеткиликтүүлүк даталары бар пункттарды кошуңуз
3. Атайын сунуштарды башкы бетте белгилеп көрсөтүңүз
4. Күн сайын же жума сайын жаңыртыңыз
5. Эски атайын сунуштарды архивдеңиз

## Theme Builder менен интеграция {#integration-with-theme-builder}

Меймандостук сайттары үчүн Theme Builder колдонгондо:

1. **Менюну автоматтык аныктоо** — тамак-аш/суусундук бизнеси экениңизди аныктайт
2. **Меню барагын түзүү** — кесипкөй меню барактарын түзөт
3. **Дизайнды шайкеш келтирүү** — менюлар сайтыңыздын темасына шайкеш келет
4. **Мобилдик оптималдаштыруу** — менюлар телефондордо сонун көрүнөт
5. **SEO оптималдаштыруу** — менюлар издөө системаларына ылайыктуу

## Мыкты практикалар {#best-practices}

### Меню дизайны {#menu-design}

- **Так уюштуруу** — логикалуу категория түзүмү
- **Окула турган сүрөттөмөлөр** — табит ачарлык жана маалыматтуу
- **Ырааттуу баалоо** — так валюта жана форматтоо
- **Кесипкөй сүрөттөр** — жогорку сапаттагы тамак сүрөттөрү
- **Бош мейкиндик** — баракты ашыкча толтурбаңыз

### Мазмун {#content}

- **Так сүрөттөмөлөр** — пункттарды так сүрөттөп жазыңыз
- **Атайын сунуштарды белгилөө** — атайын сунуштарды айырмаланып тургандай кылыңыз
- **Аллергендерди кошуу** — аллергендерди ар дайым тизмектеңиз
- **Үзгүлтүксүз жаңыртуу** — бааларды жана пункттарды актуалдуу кармаңыз
- **Табит ачар тил колдонуу** — пункттар даамдуу угулсун

### Жеткиликтүүлүк {#accessibility}

- **Окула турган ариптер** — так, окууга жеңил ариптерди колдонуңуз
- **Жетиштүү контраст** — тексттин окулушун камсыз кылыңыз
- **Тамактануу белгилери** — тамактануу варианттарын так белгилеңиз
- **Аллерген эскертүүлөрү** — аллергендерди көрүнүктүү көрсөтүңүз
- **Мобилдикке ыңгайлуу** — бардык түзмөктөрдө сынап көрүңүз
