# Меню түзүү

**Меню түзүү** мүмкүнчүлүгү барак аталыштарынан өзүнчө навигация белгилерин колдоо менен WordPress навигация менюларын түзөт.

## Жалпы маалымат {#overview}

Бул мүмкүнчүлүк стандарттуу меню түзүү функциясын `navigation_label` параметрин көрсөтүү мүмкүнчүлүгү менен кеңейтет. Бул навигацияда көрсөтүлгөн белги барак аталышынан айырмаланган менюларды түзүүгө мүмкүндүк берет, сайт түзүмүндө жана колдонуучу тажрыйбасында көбүрөөк ийкемдүүлүк берет.

## Параметрлер {#parameters}

| Параметр | Түрү | Милдеттүү | Сүрөттөмө |
|-----------|------|----------|-------------|
| `name` | string | Ооба | Меню аты, мисалы `Primary Navigation` |
| `location` | string | Жок | Бул меню дайындала турган тема жайгашуусу, мисалы `primary` |
| `navigation_label` | string | Жок | Навигацияда көрсөтүлүүчү белги (барак аталышынан өзүнчө) |

## Кайтарылган маани {#return-value}

```json
{
  "success": true,
  "menu_id": 7,
  "name": "Primary Navigation",
  "navigation_label": "Main Menu"
}
```

## Навигация белгиси жана барак аталышы {#navigation-label-vs-page-title}

`navigation_label` параметри ички меню атын колдонуучуларга навигацияда көрсөтүлгөн белгиден бөлүүгө мүмкүндүк берет:

- **`name`** — WordPress тарабынан колдонулган ички меню идентификатору (мисалы, "Негизги навигация")
- **`navigation_label`** — Навигацияда сайт конокторуна көрсөтүлгөн белги (мисалы, "Башкы меню")

Бул төмөнкү учурларда пайдалуу:
- Ички ат коюу эрежелериңиз колдонуучуга көрүнгөн белгилерден айырмаланганда
- Навигациядагы белгилерди админ панелине караганда кыскараак кылгыңыз келгенде
- Ар башка белги узундуктары бар бир нече тилди колдоо керек болгондо
- Белгилүү аймактар же колдонуучу топтору үчүн менюларды түзүп жатканыңызда

## Колдонуу мисалдары {#usage-examples}

### 1-мисал: Навигация белгиси бар жөнөкөй меню {#example-1-simple-menu-with-navigation-label}

**Сурам:**
```
"Башкы навигация" деп аталган негизги навигация менюсун "Меню" навигация белгиси менен түзүңүз.
```

**Натыйжа:**
```json
{
  "success": true,
  "menu_id": 7,
  "name": "Main Navigation",
  "navigation_label": "Menu"
}
```

### 2-мисал: Белгилүү тема жайгашуусу үчүн меню {#example-2-menu-for-specific-theme-location}

**Сурам:**
```
"Футер шилтемелери" деп аталган футер менюсун "Ыкчам шилтемелер" навигация белгиси менен түзүп, аны футер жайгашуусуна дайындаңыз.
```

**Натыйжа:**
```json
{
  "success": true,
  "menu_id": 8,
  "name": "Footer Links",
  "navigation_label": "Quick Links",
  "location": "footer"
}
```

## Theme Builder менен интеграция {#integration-with-theme-builder}

Theme Builder колдонулганда, Меню түзүү мүмкүнчүлүгү:

1. Жеткиликтүү тема меню жайгашууларын автоматтык түрдө аныктайт
2. Дизайныңызга ылайыктуу навигация белгилери бар менюларды түзөт
3. Менюларды туура тема жайгашууларына дайындайт
4. Түзүлгөндөн кийин меню элементтерин кошууну колдойт

## Байланыштуу мүмкүнчүлүктөр {#related-abilities}

- **`add_menu_item`** — Учурдагы менюга элементтерди кошуу
- **`update_menu`** — Менюну кайра атоо же аны тема жайгашуусуна кайра дайындоо
- **`delete_menu`** — Сайтыңыздан менюну алып салуу

## Мыкты тажрыйбалар {#best-practices}

- **Так навигация белгилерин колдонуңуз** — белгилерди колдонуучулар үчүн кыска жана сүрөттөмөлүү кылыңыз
- **Тема жайгашууларына дал келтириңиз** — туура көрсөтүлүшү үчүн менюларды туура тема жайгашуусуна дайындаңыз
- **Меню түзүмүн пландаңыз** — элементтерди түзүүдөн мурун меню иерархияңызды чечип алыңыз
- **Ыңгайлашуучулукту сынаңыз** — менюлар мобилдик түзмөктөрдө туура көрсөтүлгөнүн текшериңиз
- **Белгилерди локалдаштырыңыз** — ар башка тил версиялары үчүн ар башка навигация белгилерин колдонуңуз

## Чектөөлөр {#limitations}

- Навигация белгилери көрсөтүү үчүн гана; ички `name` дагы эле WordPress идентификациясы үчүн колдонулат
- Тема колдоосу ар кандай болот; бардык темалар бардык меню жайгашууларын колдобойт
- Меню түзүлгөндөн кийин меню элементтери өзүнчө кошулушу керек
- Навигация белгисин өзгөртүү менюну жаңыртууну талап кылат
