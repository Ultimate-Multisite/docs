# Logo SVG түзүү

**Generate Logo SVG** жөндөмү Theme Builder’ге ыңгайлаштырылган логотип SVG’лерин түзүп, аларды автоматтык namespace-коопсуз тазалоо менен түздөн-түз WordPress сайтыңызга жайгаштырууга мүмкүндүк берет.

## Жалпы маалымат {#overview}

Бул жөндөм сайтыңыздын брендинг багытына жана дизайн артыкчылыктарына негизделген масштабдалуучу вектордук графика (SVG) логотиптерин түзөт. Түзүлгөн SVG’лер WordPress’те колдонууга коопсуз болушу үчүн автоматтык түрдө тазаланат, ошол эле учурда визуалдык бүтүндүк сакталат.

## Параметрлер {#parameters}

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `site_name` | string | Yes | Логотип түзүлө турган сайттын аталышы |
| `style` | string | Yes | Дизайн стили (мис., "modern", "classic", "minimalist", "playful") |
| `colors` | array | No | Логотипте колдонулуучу hex түс коддорунун массиви (мис., `["#678233", "#ffffff"]`) |
| `width` | number | No | SVG туурасы пиксел менен (демейки: 200) |
| `height` | number | No | SVG бийиктиги пиксел менен (демейки: 200) |
| `include_text` | boolean | No | Логотипке сайт аталышын текст катары кошуу керекпи (демейки: true) |

## Чыгуу форматы {#output-format}

Жөндөм төмөнкү түзүмдөгү SVG сапты кайтарат:

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## SVG тазалоо жүрүм-туруму {#svg-sanitisation-behaviour}

Түзүлгөн SVG’лер автоматтык namespace-коопсуз тазалоодон өтөт, булар үчүн:

- **Кооптуу атрибуттарды алып салуу** — окуя иштетүүчүлөрүн, скрипттерди жана мүмкүн болгон кооптуу атрибуттарды алып салат
- **Namespace’терди сактоо** — туура көрсөтүү үчүн SVG namespace’терин (xmlns, xlink) сактайт
- **Түзүмдү текшерүү** — SVG W3C стандарттарына шайкеш экенин камсыздайт
- **Entity’лерди коддоо** — текст мазмунундагы атайын белгилерди туура экрандейт
- **Тышкы шилтемелерди алып салуу** — тышкы стилдик барактарды жана сүрөт шилтемелерин алып салат

Бул SVG кошумча тазалоону талап кылбастан түздөн-түз WordPress’ке жайгаштырууга коопсуз экенин камсыздайт.

## Колдонуу мисалы {#usage-example}

**Сурам:**
```
"CloudSync" деп аталган менин технологиялык стартапым үчүн көк жана ак түстөрдү колдонуп заманбап логотип түзүңүз.
```

**Натыйжа:**
Жөндөм төмөнкүлөрдү камтыган SVG логотип түзөт:
- "CloudSync" сайт аталышын камтыйт
- Көрсөтүлгөн көк жана ак түс схемасын колдонот
- Заманбап дизайн принциптерин сактайт
- Автоматтык түрдө тазаланган жана колдонууга даяр

## Theme Builder менен интеграция {#integration-with-theme-builder}

Theme Builder’дин дизайн багытын тандоосун колдонгондо, Generate Logo SVG жөндөмү:

1. Дизайн багытыңызды жана түс палитраңызды талдайт
2. Артыкчылыктарыңызга туура келген ыңгайлаштырылган SVG логотип түзөт
3. Логотипти автоматтык түрдө сайтыңыздын башкы бөлүгүнө/брендинг аймагына жайгаштырат
4. SVG’ни WordPress медиасында ыңгайлаштырылган логотип катары сактайт

## Мыкты практикалар {#best-practices}

- **Так стиль артыкчылыктарын бериңиз** — каалаган дизайн стилин сүрөттөп бериңиз (заманбап, классикалык, ойноок ж.б.)
- **Түстөрдү көрсөтүңүз** — ырааттуулук үчүн брендиңиздин түстөрүн кошуңуз
- **Көрсөтүлүшүн сынаңыз** — логотип ар кандай экран өлчөмдөрүндө туура көрсөтүлөрүн текшериңиз
- **Андан ары ыңгайлаштырыңыз** — өлчөмүн жана жайгашуусун тууралоо үчүн WordPress’тин логотипти ыңгайлаштыруу куралдарын колдонуңуз

## Чектөөлөр {#limitations}

- SVG логотиптери статикалык графика катары түзүлөт (анимацияланган эмес)
- Көп элементтери бар татаал логотиптер кол менен тактоону талап кылышы мүмкүн
- Ыңгайлаштырылган шрифттер колдоого алынбайт; текст системалык шрифттерди колдонот
- Өтө чоң же өтө кичине өлчөмдөр сапатка таасир этиши мүмкүн
