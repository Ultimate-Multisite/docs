# Өзгөртүүлөр журналы

## 1.18.0 — 2026-06-29 күнү чыгарылды

### Жаңы

- **Google Calendar куралдары** — графикти эске алган автоматташтыруулар үчүн жөндөлгөн календарларды жана окуяларды окуу
- **Байланыштарды дал келтирүү жана катышуучуларга жардамчылар** — календарь катышуучуларын сайт колдонуучуларына жана байланыштарга дал келтирүү
- **Адам жактыруусунун дарбазалары жана эскертме жазуулары** — текшерүү үчүн автоматташтырууларды токтотуп туруу жана кайталанган билдирмелерден сактануу
- **TextBee SMS провайдери** — жөндөлгөн тексттик билдирме эскертүүлөрүн TextBee аркылуу жөнөтүү
- **Өркүндөтүлгөн кошумча топтом** — WordPress.org түзүмүнөн өзүнчө таратылган ишенимдүү иштеп чыгуучу үчүн файл системасы, маалымат базасы, WP-CLI, REST диспетчери, плагин түзгүч, git сүрөтү, колдонуучу башкаруу жана өндүрүмдүүлүктү өлчөө куралдарын кошуу

### Жакшыртылды

- **Башкарылган Superdav кызматын орнотуу** — колдоого алынган сайттар үчүн жайгаштырылган кызмат endpointтерин жана автоматтык туташуу камсыздоосун кошуу
- **Чыгарылышты таңгактоо** — өзүнчө негизги жана Advanced ZIP файлдарын түзүү, экөөнү тең GitHubда жарыялоо жана WordPress.org сайтына негизги топтомду гана жөнөтүү

### Оңдолду

- **AI сурамдарынын ишенимдүүлүгү** — моделди тандоону, сурамдын күтүү убакыттарын, демейки орнотуу моделин, ой жүгүртүү текстин иштетүүнү жана жараксыз курал чакыруулары үчүн кайра аракет кылуу боюнча көрсөтмөлөрдү жакшыртуу
- **Календарды жана эскертмени бекемдөө** — Google Calendar токендерин жана эскертмелердин кайталануусун жок кылууну күчөтүү
- **Баштапкы тааныштыруу жана жактыруудан кийин улантуу** — frontend баштапкы тааныштыруу ишке киргизүүлөрүн жана ырасталган мүмкүнчүлүктөрдү улантууну оңдоо
- **WordPress.org таңгактоо текшерүүсүнүн маселелери** — негизги чыгарылыш үчүн таңгактоо текшерүүсүнүн пикирлерин чечүү

## 1.16.0 — 2026-05-20 күнү чыгарылды

### Жаңы

- **Logo SVG түзүү мүмкүнчүлүгү** — Theme Builder эми namespace коопсуз тазалоо менен ыңгайлаштырылган logo SVGлерди түзүп жана кыстара алат
- **Ачылыш маегинде сүрөт жүктөө** — Theme Builder ачылыш маеги эми байыраак дизайн контексти үчүн сүрөт жүктөө кадамын камтыйт
- **Palette Contrast текшерүү мүмкүнчүлүгү** — темага колдонордон мурун түстөрдүн жуптарын WCAG талаптарына шайкештигин текшерүү
- **Меймандостук менюлары** — Theme Builder эми меймандостук бизнеси үчүн түзүмдөлгөн тамак-аш жана суусундук меню барактарын түзө алат
- **Иш такта жана мобилдик алдын ала көрүүнү көрсөтүү** — дизайн багытын тандоо учурунда дизайныңызды иш такта жана мобилдик түзмөктөрдө алдын ала көрүү
- **Навигация энбелгисинин параметри** — Create Menu мүмкүнчүлүгү эми барактын аталышынан өзүнчө болгон `navigation_label` параметрин колдойт
- **Tier 1 куралынын жеткиликтүүлүгү** — sd-ai-agent/site-scrape эми Theme Builder ичинде демейки боюнча жеткиликтүү Tier 1 куралы

### Оңдолду

- **AI Client кэши** — эми сурамдар аралык туруктуулук үчүн transientтер менен колдоого алынат, узакка созулган агент тапшырмаларында маалымат жоготуунун алдын алат
- **Плагин саптарындагы аракет шилтемелери** — тактык үчүн оңдолуп жана кайра аталды

## 1.10.0 — 2026-05-05 күнү чыгарылды

### Жаңы

- **Tavily интернет издөөсү** — Brave Search менен катар байыраак интернет издөө натыйжалары үчүн Tavily издөө провайдерин кошуу
- **Темага ылайыкташкан камтылган көндүмдөр** — Block Themes, Classic Themes, Kadence Blocks жана Kadence Theme көндүм колдонмолору эми плагин менен кошо келет
- **Сайт түзгүчтүн байланыш формасы мүмкүнчүлүгү** — чат интерфейсинен түздөн-түз каалаган баракка байланыш формасын кошуу

### Жакшыртылды

- **WooCommerce интеграциясы кайра түзүлдү** — эми жакшыраак ишенимдүүлүк жана шайкештик үчүн нативдүү WooCommerce APIлерин колдонот
- **Провайдерлер тизмеси автоматтык түрдө жаңыланат** — каалаган плагин активдештирилгенде же өчүрүлгөндө

### Оңдолду

- **navigate-to мүмкүнчүлүгү** — айрым админ барактарындагы чексиз кайра жүктөө цикли оңдолду
- **list-posts мүмкүнчүлүгү** — эми категория жана тег аттарын slugларга туура чечет
- **WP-CLI буйруктары** — мурунку кайра түзүүдөн кийин жок болгон namespace лакап аттары калыбына келтирилди
- **Окуя автоматташтыруусу** — автоматташтыруу таблицалары али түзүлө элек сайттарды жумшак иштетет
- **memory-save мүмкүнчүлүгү** — эми системалык нускама түзгүчүндө туура namespace префиксин колдонот
- **Скалярдык курал натыйжалары** — эми AIга кайтарылардан мурун туура оролот
- **Колдонуу статистикасы** — эми эски версиялардан жаңыртканда мурунку мүмкүнчүлүк ачкычынын форматын туура иштетет
