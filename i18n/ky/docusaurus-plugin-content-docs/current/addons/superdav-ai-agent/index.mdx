# Superdav AI Agent

Superdav AI Agent — Gratis AI Agentти темадан кабардар көндүмдөр, кеңейтилген интернет издөө мүмкүнчүлүктөрү, сайт куруучу функциялар жана кошумча автоматташтыруу интеграциялары менен кеңейтүүчү өркүндөтүлгөн WordPress AI жардамчысы. 1.18.0 версиясы Google Calendar окуу куралдарын, байланыш жана катышуучу дал келтирүүнү, адамдын бекитүү дарбазаларын, эскертмелерди кайталабоону, TextBee SMS билдирүүлөрүн, башкарылуучу Superdav кызматын даярдоону жана ишенимдүү иштеп чыгуучу иш агымдары үчүн өзүнчө таратылган Advanced коштомо пакетин кошот.

## Негизги мүмкүнчүлүктөр

- **Темага ылайыкташкан көндүмдөр** — активдүү WordPress темаңызга жана орнотулган плагиндерге жараша көрсөтмөлөрдү жана мүмкүнчүлүктөрдү автоматтык түрдө ылайыкташтырат
- **Бир нече интернет издөө провайдерлери** — байыраак издөө натыйжалары жана өркүндөтүлгөн изилдөө мүмкүнчүлүктөрү үчүн Brave Search менен Tavilyнин бирин тандаңыз
- **Сайт куруучунун байланыш формалары** — орнотулган форма плагиниңизди колдонуп, түздөн-түз чат интерфейсинен толук иштеген байланыш формаларын түзүңүз
- **Жакшыртылган сайт куруучу** — плагиндерди жакшыраак табуу жана катадан калыбына келтирүү менен өркүндөтүлгөн көп баскычтуу сайт куруу пландары
- **Камтылган тема көндүмдөрү** — Block Themes, Classic Themes, Kadence Blocks жана Kadence Theme үчүн адистештирилген көрсөтмөлөр
- **Жакшыртылган WooCommerce интеграциясы** — жакшыраак ишенимдүүлүк жана шайкештик үчүн нативдүү WooCommerce APIлерин колдонууга кайра түзүлгөн
- **Провайдерди автоматтык аныктоо** — плагиндер жана темалар активдештирилгенде же өчүрүлгөндө жеткиликтүү плагиндерди жана темаларды автоматтык түрдө аныктайт жана жаңылайт
- **Ыңгайлаштырылган логотип түзүү** — аталыш мейкиндигине коопсуз автоматтык тазалоо менен ыңгайлаштырылган SVG логотиптерин түзүп, кыстарыңыз
- **Палитра контрастын текшерүү** — темаңызга колдонордон мурун түс жуптарын WCAG жеткиликтүүлүк талаптарына шайкештигин текшериңиз
- **Сүрөт менен кеңейтилген табуу** — дизайн багытына таасир этүү үчүн Theme Builder табуу маегинде сүрөттөрдү жүктөңүз
- **Меймандостук менюлары** — ресторандар, кафелер жана меймандостук ишканалары үчүн түзүмдөлгөн тамак-аш жана суусундук меню барактарын түзүңүз
- **Иш такта жана мобилдик алдын ала көрүүлөр** — дизайн багытын тандоо учурунда дизайныңызды иш такта жана мобилдик түзмөктөрдө алдын ала көрүңүз
- **1-деңгээл сайт кыргычы** — атаандаш жана шыктандыруучу веб-сайттарды талдоо үчүн sd-ai-agent/site-scrape эми демейки боюнча жеткиликтүү
- **Google Calendar окуу куралдары** — автоматташтыруулар жакынкы график контекстин талдай алышы үчүн жөндөлгөн календарларды жана окуяларды окуйт
- **Байланыш жана катышуучу дал келтирүү** — кийинки иштерди түзүүдөн мурун календарь катышуучуларын белгилүү сайт колдонуучуларына жана байланыштарга дал келтириңиз
- **Адамдын бекитүү дарбазалары** — билдирүүлөрдү жөнөтүүдөн, ишти улантуудан же өзгөртүүлөрдү киргизүүдөн мурун кароо үчүн сезимтал автоматташтырууларды токтотуп туруңуз
- **Эскертме жазуулары жана кайталабоо** — автоматташтыруу кайра иштетилгенде бир эле алуучуга эки жолу билдирүү барбашы үчүн буга чейин жөнөтүлгөн эскертмелерди көзөмөлдөңүз
- **TextBee SMS билдирүүлөрү** — SMS провайдери иштетилгенде TextBee аркылуу жөндөлгөн текст билдирүү билдирмелерин жөнөтүңүз
- **Башкарылуучу Superdav кызматын даярдоо** — колдоого алынган сайттарды кол менен endpoint жөндөөнүн ордуна автоматтык орнотуу менен жайгаштырылган Superdav endpointтерине туташтырыңыз
- **Advanced коштомо пакети** — файл системасы, маалымат базасы, WP-CLI, REST диспетчери, плагин куруучу, git snapshot, колдонуучуларды башкаруу жана benchmark иш агымдары үчүн өзүнчө таратылган ишенимдүү иштеп чыгуучу куралдары

## Талаптар

- WordPress 6.0 же андан жогору
- PHP 8.1 же андан жогору
- WordPress Multisite (тармакта активдештирилген) же бир сайттуу орнотуу
- Активдүү AI провайдер туташтыргыч плагини (Anthropic, OpenAI же шайкеш)
- Gratis AI Agent v1.9.0 же андан жогору (Superdav AI Agent Gratis AI Agentти кеңейтет)

## Орнотуу

1. Плагин файлдарын `/wp-content/plugins/superdav-ai-agent` дарегине жүктөңүз
2. Плагинди WordPress ичиндеги **Плагиндер** экраны аркылуу активдештириңиз
3. Gratis AI Agent да активдештирилгенин текшериңиз
4. **Gratis AI Agent → Settings** бөлүмүнө өтүп, керектүү учурда интернет издөө провайдериңизди, башкарылуучу Superdav кызматына туташууну, Google Calendar же TextBee эсептик маалыматтарын жөндөңүз

WordPress.org пакети негизги Superdav AI Agent мүмкүнчүлүктөрүн камтыйт. Advanced коштомо пакети ишенимдүү иштеп чыгуучу куралдары ачык орнотулуп, уруксат берилген чөйрөлөр үчүн өзүнчө курулуп жана таратылат.

## Тез баштоо

Активдештирилгенден кийин, Superdav AI Agent Gratis AI Agent орнотууңузду темадан кабардар көндүмдөр жана кошумча мүмкүнчүлүктөр менен автоматтык түрдө жакшыртат. WordPress администратор каптал тилкесинен **Gratis AI Agent** панелин ачып, жаңы мүмкүнчүлүктөрдү колдоно баштаңыз.

Add a contact form to the Contact page.

## Темага ылайыкташкан көндүмдөр

Superdav AI Agent активдүү темаңызды жана орнотулган плагиндерди автоматтык түрдө аныктап, жардамчынын контекстине тиешелүү темадан кабардар көндүмдү жүктөйт. Жеткиликтүү көндүмдөр төмөнкүлөрдү камтыйт:

| Көндүм | Эмнеге колдонулат |
|---|---|
| **Block Themes** | WordPress блокко негизделген (Full Site Editing) архитектурасын колдонгон темалар |
| **Classic Themes** | PHP шаблондорун жана классикалык редакторду колдонгон салттуу WordPress темалары |
| **Kadence Blocks** | Kadence Blocks плагини активдүү болгон сайттар |
| **Kadence Theme** | Активдүү тема катары Kadence темасы колдонулган сайттар |

Ар бир көндүм боюнча кеңири көрсөтмө үчүн [Темага ылайыкташкан көндүмдөр](../../user-guide/configuration/theme-aware-skills) бөлүмүн караңыз.

## Интернет издөө провайдерлери

Superdav AI Agent v1.10.0 интернет издөө сурамдары үчүн Brave Searchка альтернатива катары **Tavily** кошот. Эки провайдер тең реалдуу убакыттагы издөө натыйжаларын сунуштайт жана **Gratis AI Agent → Settings → Advanced** бөлүмүндө жөндөлө алат.

| Провайдер | Мүмкүнчүлүктөр | Орнотуу |
|---|---|---|
| **Brave Search** | Ылдам, купуялуулукка багытталган издөө натыйжалары | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Байыраак издөө натыйжалары, өнүккөн изилдөө мүмкүнчүлүктөрү | [Tavily API](https://tavily.com/) |

Конфигурация нускамалары үчүн [Интернеттен издөө](../../user-guide/configuration/internet-search) бөлүмүн караңыз.

## Календарь, бекитүү жана билдирүү иш агымдары

Superdav AI Agent v1.18.0 автоматташтыруу иш агымдарын мазмун жана сайт куруу тапшырмаларынан да кеңейтет:

- **Google Calendar куралдары** календарь эсептик дайындары жеткиликтүү болгондо агентке конфигурацияланган календарларды жана окуянын чоо-жайын окууга мүмкүндүк берет.
- **Байланыш жана катышуучу жардамчылары** эскертмелерди же кийинки тапшырмаларды даярдоодон мурун окуя катышуучуларын WordPress колдонуучуларына же белгилүү байланыштарга дал келтирет.
- **Адамдын бекитүү дарбазалары** автоматташтырууну ыйгарым укуктуу колдонуучу сунушталган аракетти карап чыкканга чейин токтотуп турууга мүмкүндүк берет.
- **Эскертме жазуулары** автоматташтыруу кайра аракет кылынганда же кайра-кайра пландалганда кайталанган календарь же билдирүү эскертмелерин алдын алат.
- **TextBee SMS колдоосу** TextBee провайдери конфигурацияланганда тексттик билдирүү билдирмелерин иштетет.

Бул куралдарды графикти эске алган жардам, катышуучуларга кийинки байланыш жана сезимтал билдирүүлөр жөнөтүлгөнгө чейин дагы эле адамдын карап чыгуусун талап кылган эскертме иш агымдары үчүн колдонуңуз.

## Site Builder байланыш формалары

Site Builder агенти эми чат интерфейсинен түздөн-түз байланыш формаларын түзө алат. Жөн гана Site Builderден байланыш формасын түзүүнү сураңыз, ал орнотулган форма pluginиңизди (Contact Form 7, WPForms, Fluent Forms же Gravity Forms) автоматтык түрдө аныктап, колдонууга даяр форма түзөт.

Толук көрсөтмө үчүн [Site Builder байланыш формалары](../../user-guide/administration/site-builder-contact-forms) бөлүмүн караңыз.

## Theme Builder мүмкүнчүлүктөрү

Superdav AI Agent v1.16.0 ичиндеги Theme Builder күчтүү жаңы мүмкүнчүлүктөрдү камтыйт:

### Сүрөт жүктөө менен ачылыш маеги

Theme Builder ачылыш маеги эми визуалдык шыктануу бере ала турган сүрөт жүктөө кадамын камтыйт. Theme Builder сиздин дизайн багытыңызды түшүнүшү үчүн брендиңиздин, атаандаш сайттардын же дизайн шыктануусунун сүрөттөрүн жүктөңүз.

Толук көрсөтмө үчүн [Ачылыш маеги](./theme-builder/discovery-interview.md) бөлүмүн караңыз.

### Иш такта жана мобилдик алдын ала көрүүлөр менен дизайн багыты

Theme Builder толук themeиңизди түзгөнгө чейин дизайныңызды иш тактада да, мобилдик түзмөктөрдө да карап чыгыңыз. Бир нече жооп берүүчү чекиттерде реалдуу убакыттагы алдын ала көрүүлөрдү көрүңүз.

Толук көрсөтмө үчүн [Дизайн багыты](./theme-builder/design-direction.md) бөлүмүн караңыз.

### Меймандостук менюлары

Theme Builder эми ресторандар, кафелер, барлар жана башка меймандостук ишканалары үчүн түзүмдөлгөн тамак-аш жана суусундук меню барактарын түзө алат. Менюлар бааларды, сүрөттөмөлөрдү, диеталык маалыматты жана аллерген эскертүүлөрүн камтыйт.

Толук көрсөтмө үчүн [Меймандостук менюлары](./theme-builder/hospitality-menus.md) бөлүмүн караңыз.

### Жаңы жөндөмдөр

- **[Logo SVG түзүү](./abilities/generate-logo-svg.md)** — автоматтык тазалоо менен ыңгайлаштырылган SVG логотиптерди түзүү
- **[Палитра контрастын текшерүү](./abilities/validate-palette-contrast.md)** — түстөр жуптарын WCAG талаптарына шайкештик боюнча текшерүү
- **[Меню түзүү](./abilities/create-menu.md)** — өзүнчө навигация энбелгилери менен навигация менюларын түзүү

Жеткиликтүү куралдар жана Tier 1 сайтты кыргыч куралы жөнүндө маалымат үчүн [Куралдар жана конфигурация](./theme-builder/tools.md) бөлүмүн караңыз.

## Конфигурация

Бардык жөндөөлөр WordPress админ бөлүгүндө **Gratis AI Agent → Жөндөөлөр** астында. Superdav AI Agent төмөнкү конфигурация параметрлерин кошот:

| Жөндөө | Сүрөттөмө |
|---|---|
| **Интернеттен издөө провайдери** | Brave Search менен Tavily арасынан тандаңыз (милдеттүү эмес) |
| **Tavily API Key** | Tavily интернеттен издөөсү үчүн API ачкычы (эгер Tavily тандалса) |
| **Башкарылган Superdav кызматы** | Колдоого алынган сайттарды автоматтык endpoint камсыздоо менен жайгаштырылган Superdav кызматына туташтырыңыз |
| **Google Calendar эсептик дайындары** | Агент календарларды жана окуяларды окуй алышы үчүн календарга кирүүнү конфигурациялаңыз |
| **TextBee SMS провайдери** | Бекитилген SMS билдирүү иш агымдары үчүн TextBee эсептик дайындарын конфигурациялаңыз |

## Advanced коштоочу пакети

Superdav AI Agent v1.18.0 ишенимдүү иштеп чыгуучу иш агымдары үчүн өзүнчө таратылган Advanced коштоочу пакетин кошот. Бул мүмкүнчүлүктөр WordPress.org өзөк пакетине киргизилген эмес. Коштоочу пакетти администраторлор агентке иштеп чыгуучу куралдарына кирүүгө ачык ишенген сайттарга гана орнотуңуз.

Advanced куралдары файл тутуму операцияларын, базага кирүүнү, WP-CLI, REST dispatcher иш агымдарын, plugin куруучу утилиталарды, git snapshots, колдонуучуларды башкаруу жардамчыларын жана benchmark куралдарын камтыйт. Бул куралдарды ишенимдүү администраторлор менен гана чектеп, production сайттарда иштетүүдөн мурун бекитүү дарбазасынын конфигурациясын карап чыгыңыз.

- **TextBee SMS провайдери** — талап кылынган бекитүүлөрдөн кийин конфигурацияланган тексттик билдирүү билдирмелерин жөнөтүү
- **Башкарылган Superdav кызматын орнотуу** — колдоого алынган сайттар үчүн жайгаштырылган кызмат endpointтерин автоматтык түрдө камсыздоо
- **Advanced коштоочу таратуу** — ишенимдүү иштеп чыгуучу куралдарын WordPress.org өзөк түзүмүнөн өзүнчө жарыялоо
- **Ишенимдүүлүк оңдоолору** — моделди тандоону, сурам убакыт чектерин, демейки орнотуу моделин иштетүүнү, reasoning-text иштетүүнү, кайра аракет кылуу көрсөтмөлөрүн, календарь tokenин бекемдөөнү, эскертмени бекемдөөнү, onboarding ишке киргизүүлөрүн, ырасталган жөндөмдөрдү улантууну жана WordPress.org пакеттөө текшерүүсү маселелерин жакшыртуу
