# Loco Translate

Ultimate Multisite: Loco Translate Integration кошумчасы WordPress multisite тармагыңызга сайтка мүнөздүү котормо мүмкүнчүлүктөрүн алып келет. Эгер тармагыңыздагы ар башка сайттарга бир эле plugin же theme үчүн ар башка котормолор керек болгон учур болсо, бул кошумча эң ылайыктуу чечим.

**Бул кошумча эмне кылат**

Бул кошумча Ultimate Multisiteти популярдуу Loco Translate plugin менен үзгүлтүксүз интеграциялап, тармагыңыздагы ар бир субсайтка өзүнүн ыңгайлаштырылган котормолорун сактоого мүмкүнчүлүк берет. Сиз көп аймактуу платформа иштетип жатасызбы, ар башка тил коомчулуктарын тейлейсизби же тармагыңыз боюнча жөн гана котормо ийкемдүүлүгү керекпи — бул кошумча муну оңой кылат.

**Төмөнкүлөр үчүн идеалдуу:**

* Ар башка локалдаштыруу муктаждыктары бар көп аймактуу ишканалар
* Ар түрдүү коомчулуктарды тейлеген билим берүү мекемелери
* Көп тилдүү колдоо сунуштаган SaaS платформалары
* Бир нече кардар сайттарын башкарган веб-агенттиктер
* Котормо ийкемдүүлүгүн талап кылган каалаган multisite тармагы

## Орнотуу {#installation}

1. Ultimate Multisite орнотулганын жана тармакта активдештирилгенин текшериңиз
2. Loco Translate plugin орнотуп, активдештириңиз
3. Бул кошумчаны plugins директорияңызга жүктөңүз
4. Loco Translate Integration кошумчасын тармак боюнча активдештириңиз
5. Каалаган субсайтта Loco Translate аркылуу сайтка мүнөздүү котормолорду түзө баштаңыз

Бүттү! Интеграция эч кандай кошумча конфигурациясыз автоматтык түрдө иштейт.

## Көп берилүүчү суроолор {#frequently-asked-questions}

## Орнотуудан кийин бир нерсени конфигурациялашым керекпи? {#do-i-need-to-configure-anything-after-installation}

Жок! Интеграция автоматтык. Кошумча да, Loco Translate да активдүү болгондон кийин, субсайттар дароо өздөрүнүн ыңгайлаштырылган котормолорун түзө баштай алышат.

## Эгер субсайт ыңгайлаштырылган котормолорду түзбөсө эмне болот? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

Сайт автоматтык түрдө тармак боюнча котормолорду резервдик вариант катары колдонот. Эч нерсе бузулбайт жана баары үзгүлтүксүз иштейт.

## Субсайттар тармак боюнча котормолорго кире алабы? {#can-subsites-access-network-wide-translations}

Кадимки сайт админдери өз сайтынын котормолорун гана көрүп жана түзөтө алышат. Super админдер көрүп жаткан сайтына жараша тармак боюнча жана сайтка мүнөздүү котормолордун экөөнө тең кире алышат.

## Бул plugins жана themes экөөндө тең иштейби? {#does-this-work-with-both-plugins-and-themes}

Ооба! Сайтка мүнөздүү котормолор WordPress plugins жана themes үчүн тең иштейт.

## Бул менин multisite тармагымды жайлатабы? {#will-this-slow-down-my-multisite-network}

Жок. Кошумча зарыл болгондо гана сайтка мүнөздүү котормолорду текшерген натыйжалуу файл жүктөөнү колдонот. Өндүрүмдүүлүккө байкаларлык таасир жок.

## Тармак котормолорун баштапкы чекит катары белгилүү бир сайтка көчүрө аламбы? {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

Ооба! Loco Translate ичинде котормолорду түзөтүп жатканда, тармак котормолорунан оңой баштап, аларды ар бир сайт үчүн ыңгайлаштыра аласыз.

## Бул WooCommerce жана башка plugins менен иштейби? {#does-this-work-with-woocommerce-and-other-plugins}

Албетте! Стандарттуу WordPress котормо функцияларын колдонгон каалаган plugin же theme сайтка мүнөздүү котормолор менен иштейт.
