# Fluent Forms

Бул кошумча Multisite Ultimate ичинде өнүмдөргө, мүчөлүктөргө же сайттарга Fluent Forms үчүн чектөөлөрдү кошууга мүмкүнчүлүк берет.
Fluent Forms ыңгайлаштырылган жазма түрлөрүнүн ордуна ыңгайлаштырылган таблицаларды колдонот, ошондуктан кадимки чектөө параметрлери Fluent Forms менен иштебейт.

## Өзгөчөлүктөр {#features}

- Fluent Forms же Conversational Fluent Forms үчүн чектөөлөрдү кошуу
- Кардарлар чектөөгө жеткенде, планын жаңыртууга чакырган жагымдуу билдирүү алышат
