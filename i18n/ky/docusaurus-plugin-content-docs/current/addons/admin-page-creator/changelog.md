# Админ баракча түзгүчтүн өзгөртүүлөр журналы

### Версия 1.9.0 - 2026-01-18 күнү чыгарылды

* Оңдолду: Активдештирүүдө өлүмгө алып келген каталар.
* Кайра түзүлдү: Мазмун булагынын класстары туура ата-эне конструкторун баштапкы абалга келтирүү менен Singleton trait колдонууга өткөрүлдү
* Оңдолду: Мазмун түрүн тандоочунун көрсөтүлүшүнө тоскоол болгон Vue.js шаблон компиляция каталары
* Оңдолду: Жетишпеген Vue маалымат касиеттери (template_id, external_link_url_embedable, external_link_url_checking)
* Оңдолду: TinyMCE редактору баракча жүктөлгөндө туура башталбай жаткан
* Оңдолду: Divi колдоо файлы Composer аркылуу автоматтык жүктөлбөй жаткан
* Оңдолду: Divi колдоо классындагы бузулган PHP синтаксиси
* Жакшыртылды: JavaScript активдерин уюштуруу жана кичирейтүү
* Жакшыртылды: Акыркы WordPress жана баракча түзгүч версиялары менен шайкештик
