# Admin Page Creator {#admin-page-creator}

## Мазмун {#content}
Admin page creator сизге WordPress админ Dashboard ичинде жаңы админ барактарды жана кошумча барактарды оңой кошууга мүмкүндүк берет. Аны жаңы колдонуучулар үчүн конуу барагы катары кызмат кыла турган саламдашуу барагын же окуу куралы барагын түзүү үчүн колдонсо болот. Барактарды түйшүксүз долбоорлоо үчүн Beaver Builder, Elementor, Brizy жана Oxygen сыяктуу барак куруучуну колдоно аласыз.

Ал ошондой эле ыңгайлаштырылган HTML, CSS жана JavaScript колдоосун, ошондой эле планга жана колдонуучунун ролуна негизделген кирүү көзөмөлүн сунуштайт!

## Мүмкүнчүлүктөр {#features}
* Ыңгайлаштырылган HTML жана CSS колдоосу
* JavaScript колдоосу
* PHP мазмун режими
* Колдонуучу ролуна негизделген уруксат
* Барак куруучуларды колдоо

## Орнотуу {#installation}

1. **ultimate-multisite-admin-page-creator** файлын **/wp-content/plugins/** каталогуна жүктөңүз
2. WordPress тармак админиңиздеги **Плагиндер** менюсу аркылуу плагинди активдештириңиз

## Колдонуу колдонмосу {#usage-guide}

WordPress сайттардын алдыңкы бөлүгүнө мазмун кошууну чындап эле жеңил кылганы менен, админ тарабында барактарды жана кошумча барактарды түзүү үчүн айрым негизги коддоо көндүмдөрү талап кылынат. Ultimate Multisite: Admin Page Creator тармак сайттарыңыздын арткы бөлүгүнө алдыңкы бөлүктөгүдөй эле колдонуу жеңилдигин алып келүүнү максат кылат.

## Эмне үчүн Admin Page Creator колдонуу керек? {#why-use-admin-page-creator}

Балким, сиз кардарларыңыздын админ панелинде окуу куралдары менен ыңгайлаштырылган барак сунуштагыңыз келет, балким, FAQ жана байланыш виджети менен ыңгайлаштырылган барак кошкуңуз келет. Admin Page Creator менен мүмкүнчүлүктөр чексиз, жана бүгүнкү окуу куралында биз дал ошону карап чыгабыз.

## Меню параметрлери {#menu-options}

Капталда, Жалпы параметрлер meta-box ичинде, меню түрүн, энбелгисин жана дагы көп нерселерди конфигурациялай аласыз.

### Меню түрлөрү {#menu-types}

Жаңы ыңгайлаштырылган админ барагыңызда эки башка меню түрү болушу мүмкүн: жогорку деңгээлдеги меню жана кошумча меню.

Жогорку деңгээлдеги меню барагы, аталышы айтып тургандай, жаңы түзүлгөн барагыңызды демейки WordPress барактары менен бирге админ менюга кошот.

Эгер жогорку деңгээл вариантын тандасаңыз, меню тартибин жана меню сөлөкөтүн да тандай аласыз.

Эгер бул баракты кошумча меню барагы кылгыңыз келсе, бул баракты тиркөө үчүн ата-эне баракты тандай аласыз. Ыңгайлаштырылган барагыңызды демейки WordPress барактарына же жогорку деңгээлдеги ыңгайлаштырылган админ барактарыңызга кошо аласыз.

## Өркүндөтүлгөн параметрлер {#advanced-options}

Ыңгайлаштырылган админ барактарыңызга ар кандай мазмунду кошо алышыңыз керек. Буга CSS эрежелери жана файлдары, JavaScript, ошондой эле тышкы китепканалар кирет. Муну Өркүндөтүлгөн параметрлер өтмөгү аркылуу жасай аласыз.

### Ыңгайлаштырылган CSS {#custom-css}

CSS өтмөгүндө өзүңүздүн ыңгайлаштырылган CSS кодуңузду кошо аласыз, ошондой эле алыскы сайттардан CSS файлдарын импорттой аласыз (мисалы, CDNs, Google Fonts ж.б.).

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_Ыңгайлаштырылган CSS да колдоого алынат. Тышкы стиль файлдарын да кошо аласыз!_

### Ыңгайлаштырылган JavaScript {#custom-javascript}

Ушундай эле нерсе JavaScript үчүн да жарактуу. Ыңгайлаштырылган JavaScript кодуңузда колдонуу үчүн тышкы китепканаларды кошо аласыз.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Өзүңүздүн JavaScript кодуңузду кошуңуз!_

## Уруксаттар {#permissions}

Өркүндөтүлгөн параметрлерде бул баракка кайсы кошумча сайттар кире аларын да тандай аласыз. Мисалы, ар кандай план деңгээлдерине ар башка эксклюзивдүү мазмун берүүнү пландап жатсаңыз, бул пайдалуу.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Бул барак кайда көрүнөрүн толук көзөмөлдөй аласыз_

**Маанилүү** : Барак көрсөтүлүшү үчүн эки шарт тең аткарылышы керек. Бул деген, эгер колдонуучу берилген пландын кардары болсо, бирок тандалган ролдордун бири жок болсо (же тескерисинче), барак анын панелинде көрүнбөйт.

## Активдүү {#active}

Эгер ыңгайлаштырылган admin баракты өчүрбөстөн же уруксаттарды бузбастан убактылуу иштен чыгаргыңыз келсе, жөн гана Active опциясынан белгини алып салыңыз.

Жогорку деңгээлдеги баракты иштен чыгаруу ошол ата-эне баракка тиркелген бардык ички барактарды автоматтык түрдө иштен чыгарат.
