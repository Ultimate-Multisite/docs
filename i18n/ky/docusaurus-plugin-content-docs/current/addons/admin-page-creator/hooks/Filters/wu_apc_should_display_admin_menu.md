# Filter: wu_apc_should_display_admin_menu

Администраторлор ыңгайлаштыруу процессин аяктаган болсо, admin menu барактарын жашырууга уруксат берет.

## Параметрлер

| Аталышы | Түрү | Сүрөттөмө |
|------|------|-------------|
| $should_display_admin_menu | `bool` | Барактарды көрсөтүү керек болсо. |
| $network_active | `bool` | Плагин network admin активдүү экенин текшерет. |
