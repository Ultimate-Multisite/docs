# GoCardless

Multisite Ultimate: GoCardless төлөмү

Кардарларыңызга Multisite Ultimate мүчөлүктөрү жана өнүмдөрү үчүн GoCardless аркылуу түз дебет менен төлөөгө мүмкүнчүлүк бериңиз!

## Орнотуу

1. 'ultimate-multisite-gocardless' файлын '/wp-content/plugins/' каталогуна жүктөңүз
2. Плагинди WordPress ичиндеги 'Плагиндер' менюсу аркылуу активдештириңиз
3. Плагинди жөндөө үчүн кадам-кадам Wizard көрсөтмөсүн аткарыңыз

## Колдонуу колдонмосу

_GoCardless_ — **Европада жана Улуу Британияда** жеткиликтүү болгон, **түз дебет** төлөмдөрүнө мүмкүндүк берген төлөм шлюзу.

**Ultimate Multisite: GoCardless Gateway** Ultimate Multisite үчүн _GoCardless_ колдоосун кошуп, ошол аймактагы кардарларыңыздан түз дебет аркылуу төлөм алууга мүмкүндүк берет.

Бул окуу куралы Ultimate Multisite тармагыңызда _GoCardless_ аркылуу төлөмдөрдү кабыл ала баштоо үчүн аны кантип жөндөөнү камтыйт.

_**Эскертүү: GoCardless төмөнкү валюталар менен гана иштейт:**_ **EUR, GBP, AUD, SEK жана DKK.**

## Кошумча модулду орнотуу

Эң оболу, Ultimate Multisite үчүн GoCardless кошумча модулун орнотушуңуз керек. Муну **Ultimate Multisite > Жөндөөлөр > Кошумча модулдар (каптал тилкесинде)** бөлүмүнө өтүп, ошол тизмеден GoCardless кошумча модулун тандап жасай аласыз.

Кошумча модуль орнотулуп жана активдештирилгенден кийин, **Ultimate Multisite > Жөндөөлөр > Төлөмдөр** бөлүмүнө өтүп, GoCardless опциясын күйгүзүңүз жана сактаңыз.

![Төлөм шлюзунун жөндөөлөрү](/img/config/settings-payment-gateways.png)

## Жетүү токенин алуу

_GoCardless_ ичинде Account түзүңүз (же алгач **сынап көргүңүз келсе**, сунушталгандай _GoCardless_ sandbox ичинде). Андан кийин, кирген соң, **Иштеп чыгуучулар** меню пунктуна өтүңүз.

<!-- Скриншот жеткиликсиз: GoCardless dashboard ичинде Developers менюсунун жайгашуусу көрсөтүлгөн -->

**Иштеп чыгуучулар** барагында экрандын жогорку оң бурчундагы **Түзүү** баскычын басыңыз, андан кийин **Жетүү токени** опциясын басыңыз.

<!-- Скриншот жеткиликсиз: GoCardless Developers барагы Create баскычы жана Access Token опциясы менен -->

Бул жаңы модалдык терезени ачып, жаңы токен түзүүгө мүмкүндүк берет. Ага таанымал ат бериңиз жана **Көлөм** менюсунда **Окуу-жазуу жеткиликтүүлүгү** опциясын тандаганыңызды текшериңиз.

<!-- Скриншот жеткиликсиз: Read-write көлөмү менен жаңы жетүү токенин түзүү үчүн GoCardless модалы -->

Жетүү токени экранда пайда болот. Аны көчүрүп алып, **Ultimate Multisite > Жөндөөлөр > Төлөмдөр > GoCardless** менюсуна чаптаңыз.

Аны туура талаага койгонуңузга ишениңиз. Эгер бул **жандуу токе** **н** болсо, токенди чаптоодон мурун **sandbox режимин өчүрүңүз**.

![GoCardless менен төлөм шлюзунун жөндөөлөрү](/img/config/settings-payment-gateways.png)

<!-- Скриншот жеткиликсиз: Ultimate Multisite төлөм жөндөөлөрүндөгү GoCardless токен талаалары -->

Биринчи кадам үчүн ушуну менен бүттү. Эми webhook угуучусун жөндөшүңүз керек.

## Webhook угуучусун жөндөө

Жазылуулар менен иштегенде, Ultimate Multisite жазылуу абалындагы өзгөрүүлөр, жаңы төлөмдөрдүн жүргүзүлүшү жана ушул сыяктуу башка нерселер жөнүндө билип турушу керек.

Төлөм шлюздары Ultimate Multisite'ка бул өзгөрүүлөр жөнүндө **webhook чакырууларын** колдонуп кабарлайт. Ошондуктан интеграция 100% иштеши үчүн, _GoCardless_ бардык ошол маалымат менен кайсы URL'ге кайрылышы керек экенин көрсөтүшүбүз керек.

**GoCardless > Иштеп чыгуучулар** панелиңизге кайра барып, Create баскычын дагы бир жолу басыңыз, бирок бул жолу **Webhook endpoint** опциясын тандаңыз.

<!-- Скриншот жеткиликсиз: GoCardless Developers барагы Webhook endpoint опциясы менен -->Webhook endpoint опциялары менен жаңы терезе ачылат. Ал webhook URL'ин сурайт, ошондуктан **Ultimate Multisite жөндөөлөрү > Төлөмдөр > GoCardless** бөлүмүнө кайра өтүп, ал жерде көрсөтүлгөн webhook URL'ин көчүрүңүз:

<!-- Скриншот жеткиликсиз: webhook URL көрсөтүлгөн Ultimate Multisite GoCardless жөндөөлөрү -->

Кошумча катары, webhook чакырууларыңызга кол коюу үчүн колдонула турган **webhook сырын** киргизсеңиз болот. Ал талаага эмне киргизсеңиз да, аны **Ultimate Multisite Webhook Secret талаасына** да киргизишиңиз керек.

<!-- Скриншот жеткиликсиз: URL жана сыр талаалары бар GoCardless webhook endpoint конфигурация формасы -->

<!-- Скриншот жеткиликсиз: Ultimate Multisite GoCardless webhook сыр талаасы -->

Аяктоо үчүн **Create webhook endpoint** баскычын басып endpoint'ти сактаңыз, андан кийин **Ultimate Multisite жөндөөлөрүңүздү да сактаңыз**.

Эми _GoCardless_ кабыл алууга даяр болушуңуз керек!
