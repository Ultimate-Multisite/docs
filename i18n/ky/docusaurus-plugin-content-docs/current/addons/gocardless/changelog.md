# GoCardless өзгөрүүлөр журналы

Версия: 1.0.3 - 2026-05-05 күнү чыгарылган
* Оңдоо: GoCardless мүчөлүгү күтүлүүдө же кармалууда абалында жокко чыгарылганда, ээси жок күтүлүүдөгү сайттар эми тазаланат
* Жакшыртылды: Bedrock тамыр autoloader көз карандылыктарды буга чейин жүктөгөн болсо, плагин autoloader өткөрүп жиберилет

Версия: 1.0.1 - 2025-09-28 күнү чыгарылган

* Префикс ultimate-multisite болуп өзгөртүлдү; текст домени жаңыртылды; версия көтөрүлдү.



2021-09-24 - Версия 1.0.0-beta.4

* Кошулду: mu-plugins негизиндеги орнотуулар үчүн wp_ultimo_skip_network_active_check чыпкасы;

2021-06-16 - Версия 1.0.0-beta.3

* Оңдоо: Webhook иштетүүчүлөрү кол тамгаларды ырастай албай жаткан;

2021-06-14 - Версия 1.0.0-beta.2

* Жакшыртылды: GoCardless mandate түзүлгөндөн кийин сайтты жарыялоо;

Версия 1.0.0-beta.1 - Баштапкы версия
