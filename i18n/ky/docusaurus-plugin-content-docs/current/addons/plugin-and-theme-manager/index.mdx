# Плагин жана тема башкаргычы

Multisite Ultimate: Плагин жана тема башкаргычы

Орнотулган плагиндердин жана темалардын мета маалыматын (аталышы, автору, миниатюрасы жана сүрөттөмөсү) түзөтүңүз же жашырыңыз. Колдонуучуларыңыз аларды чыпкалай алышы үчүн плагиндер жана темалар үчүн категорияларды түзүңүз жана колдонуучуларыңыздын панелине кооз ыңгайлаштырылган Плагиндер барагын кошуңуз.

Плагинди колдонуу үчүн тармак администраторуңуздагы Плагиндер же Темалар барактарына өтүп, "Плагин/тема маалыматын түзөтүү" баскычын бассаңыз болот.

## Орнотуу {#installation}

1. 'ultimate-multisite-plugin-and-theme-manager' файлын '/wp-content/plugins/' каталогуна жүктөңүз
2. WordPress Network Admin ичиндеги 'Плагиндер' менюсу аркылуу плагинди активдештириңиз

## Колдонуу колдонмосу {#usage-guide}

Ultimate Multisite: Плагин жана тема башкаргычы — тармагыңыздагы колдонуучулар тармагыңыздагы плагин жана тема параметрлерин кандай көрүп, алар менен кандай иштешерин толук көзөмөлдөөгө мүмкүндүк берген күчтүү **премиум** Ultimate Multisite кошумча модулу.

Бул кошумча модуль үч максатты аткарат:

  1. Ал плагиндердин да, темалардын да метамаалыматтарын толугу менен өзгөртүүгө мүмкүндүк берет. Буга аталышы, сүрөттөмөсү, автордун аты жана ал тургай миниатюралар да кирет!

  2. Ал колдонуучуларыңыз аларды оңой иреттей алышы үчүн плагиндерди жана темаларды категорияларга бөлүүгө мүмкүндүк берет.

  3. Ал эски, кызыксыз плагиндер барагын категориялар боюнча иреттөө мүмкүнчүлүгү бар жаңы, темаларга окшош баракка алмаштыруу опциясын берет.

Аны жүктөп алып орноткондон кийин, Ultimate Multisite Жөндөөлөр барагында жаңы өтмөк - _Кошумча модулдун жөндөөлөрү_ \- пайда болот.

Тармагыңыздын _Плагиндер_ барагында эми бардык плагиндердин мета маалыматын, мисалы атын, авторун жана ал тургай плагиндин миниатюрасын түзөтө аласыз.
