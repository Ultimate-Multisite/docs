# Site Exporter өзгөртүүлөр журналы {#site-exporter-changelog}

Версия: 1.0.6 - 2026-05-11 күнү чыгарылды
* Жаңы: Site экспорттору эми өзүн-өзү жүктөгөн index.php файлын камтыйт, ошондуктан ZIP өзүнчө plugin орнотуусуз жаңы хостко орнотула алат
* Жаңы: Network экспорту администраторлорго Site Export админ барагынан бардык subsite'тарды бир архивге экспорттоого мүмкүндүк берет

Версия: 1.0.5 - 2026-05-05 күнү чыгарылды
* Оңдоо: get_config сыяктуу mock кылынбаган методдорду иштетүү үчүн WP_CLI polyfill stub'уна __callStatic кошулду, бул тесттин bootstrap каталарын алдын алат

Версия: 1.0.4 - 2025-11-25 күнү чыгарылды

* Ultimate Multisite 2.4 менен шайкештик оңдолду.

Версия: 1.0.2 - 2025-09-28 күнү чыгарылды

* Префиксти ultimate-multisite деп өзгөртүү; текст доменин жаңыртуу; версияны көтөрүү.

### Версия 1.0.1 - 2023-08-09 күнү чыгарылды {#version-101---released-on-2023-08-09}

* Жакшыртылды: Эми importer файлынын туура файл жолу алынганын камсыздайт.
* Оңдолду: Импорт учурунда каталардын алдын алуу үчүн object cache өчүрүлдү.
* Оңдолду: importer plugin жүктөп алуу шилтемеси иштебей жаткан маселе оңдолду.

### Версия 1.0.0 - 2022-12-23 күнү чыгарылды {#version-100---released-on-2022-12-23}

* Ички: Жаңы түзүү процесси;

### Версия 1.0.0-beta.4 - 2022-01-13 күнү чыгарылды {#version-100-beta4---released-on-2022-01-13}
