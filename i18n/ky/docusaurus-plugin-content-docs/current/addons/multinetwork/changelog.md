# Multi-Network өзгөрүүлөр журналы {#multi-network-changelog}

### 1.3.0 {#130}
* Жаңы: Тармак шаблонун алдын ала көрүүчү — сатып алуудан мурун жандуу панель менен тармак шаблондорун карап чыгып, алдын ала көрүңүз.
* Жаңы: Төлөмдү аяктоодо тармак шаблонун тандагыч — кардарлар катталуудагы төлөмдү аяктоо агымында тармак шаблонун тандай алышат.
* Жаңы: Тармактар аралык magic link SSO — кардарлар magic-link аутентификациясы аркылуу суб-тармактарга тоскоолдуксуз кире алышат.
* Жаңы: Тармак медиасын клондоо — тармактарды шаблондон клондоодо сайт медиасы туура көчүрүлөт.
* Оңдоо: Тармакты клондоо эми multi-tenancy addon жок эле иштейт; клондон кийин super admin укуктары кайра берилет.
* Оңдоо: Тармак түзүү эми WooCommerce жана бардык төлөм шлюздары менен шайкеш.
* Оңдоо: Порт номери эми fallback домен түзүүдө камтылат.
* Оңдоо: Шаблон тандагычтын CSS'и бардык темаларда туура көрсөтүлүшү үчүн enqueued stylesheet'ке көчүрүлдү.
* Оңдоо: Сайт суроо scope'унда аныкталбаган `network_id` ачкычынан коргоо.
* Оңдоо: Контекстти алмаштыруудан мурун тармак уруксатын текшерүү `wpdb` таблица касиетинин каталарын алдын алат.
