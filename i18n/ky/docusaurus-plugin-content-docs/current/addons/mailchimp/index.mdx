# Mailchimp

Multisite Ultimate: Mailchimp интеграциясы

Multisite Ultimate тармагыңызды Mailchimp'иңизге интеграциялаңыз. Белгилүү пландарды Mailchimp тизмелерине байланыштырып, катталуу учурунда жазылуучуларды ошол тизмелерге автоматтык түрдө кошо аласыз.

## Орнотуу {#installation}

1. 'ultimate-multisite-mailchimp' файлын '/wp-content/plugins/' каталогуна жүктөңүз
2. Плагинди WordPress Network Admin'иңиздеги 'Plugins' менюсу аркылуу активдештириңиз

## Колдонуу боюнча колдонмо {#usage-guide}

Жазылуучулардын электрондук почталарын MailChimp тизмелериңиз менен шайкештештирүү биздин Ultimate Multisite: MailChimp Integration Add-on аркылуу абдан оңой. Бул нускама сизге баарын жөндөөгө жардам берүүгө багытталган!

## API ачкычын алуу {#getting-an-api-key}

Жеткиликтүү ар кандай жөндөөлөргө өтүүдөн мурун, бизге MailChimp API ачкычы керек. Бул Ultimate Multisite'ка тизмелерди жана топторду алуу, ошондой эле колдонуучулар катталганда алардын электрондук почта даректерин ошол тизмелерге жана топторго кошуу үчүн MailChimp API менен байланышууга мүмкүндүк берет.

Өзүңүздүн MailChimp API ачкычыңызды алуу үчүн бул нускаманы аткарыңыз: [API ачкычтары жөнүндө | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

## Интеграция режими {#integration-mode}

MailChimp Integration add-on эки “интеграция режимин” сунуштайт, бул электрондук почта даректерин шайкештештирүү үчүн add-on'ду колдонуунун эки башка жолу бар дегенди билдирет.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_Көбүрөөк ийкемдүүлүк үчүн эки интеграция режими!_

### Режим 1: Бир нече тизме {#mode-1-multiple-lists}

Демейки режим **Multiple Lists** колдонот. Бул режим ар бир план үчүн бир нече MailChimp тизмесин тандоого мүмкүндүк берет. Колдонуучуларыңызды ар башка тизмелерге бөлсөңүз, бул эң жакшы вариант.

Мисалы, MailChimp Account'уңузда 3 башка тизме бар деп элестетели: **Newsletter** , **Plan A** , жана **Plan B**. Планына карабастан бардык колдонуучулар Newsletter тизмесине да киргизилишин каалайсыз, ошондо алар жаңыртууларыңызды өткөрүп жиберишпейт.

Муну ишке ашыруу үчүн, Multiple Lists режимин колдонуп, **Newsletter** жана **Plan A** тизмелерин Plan A MailChimp Settings'ке, ал эми **Newsletter** жана **Plan B** тизмелерин Plan B MailChimp Settings'ке кошо аласыз.

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_Жаңы аккаунттар жана жокко чыгарылган аккаунттар үчүн демейки тизмелерди тандаңыз_

### Режим 2: Жалгыз тизме + бир нече топ {#mode-2-single-list--multiple-groups}

MailChimp тизмеңизди **Groups** деп аталган колдонуучулардын майда топторуна бөлүүгө мүмкүндүк берет. Бул сизге көбүрөөк көзөмөл берет, анткени MailChimp'те бир эле кампаниянын бир нече көчүрмөсүн түзбөстөн, жазылуучуларыңыздын белгилүү бир кичи бөлүгүнө кампания жөнөтө аласыз (бул кийин ошол кампаниянын метрикаларын талдоону кыйындатат).

Мисалы, сизде **My Network Subscribers,** деп аталган бир гана тизме болуп, анын ичинде **Plan A** жана **Plan B** үчүн топтор болушу мүмкүн.

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_Эгер “жалгыз тизме + бир нече топ” интеграция режимин колдонсоңуз, демейки тизмелерди жана топторду тандаңыз_

## Демейки тизмелер жана топтор {#default-lists-and-groups}

Белгилүү колдонуу жагдайыңызга кайсы интеграция режими көбүрөөк туура келерин тандагандан кийин, демейки тизмелерди (жана режимге жараша топторду) тандоо убактысы келет. Демейки тизме берилген планда атайын тизме тандалбаганда колдонулат.

Ошондой эле жокко чыгарылган аккаунттар үчүн демейки тизмени тандай аласыз. Колдонуучу өзүнүн жазылуусун токтотуп, аккаунтун тармагыңыздан алып салган сайын, анын электрондук почта дареги жокко чыгарылган аккаунттар үчүн коюлган тизмеге жылдырылат.

## Кош жазылууну иштетүү {#enabling-double-subscription}

Ошол эле жөндөөлөр барагында **Double Subscription** иштете аласыз. Бул параметр колдонуучуну тандалган демейки тизмеге ДА, ошол колдонуучунун планы үчүн тандалган тизмеге ДА кошот. Эгер бул параметр иштетилбесе, колдонуучу колдонуучунун планы үчүн тандалган тизмеге гана кошулат. Эгер ошол конкреттүү план үчүн эч нерсе коюлбаса, анда колдонуучу тандалган демейки тизмеге кошулат.

_Катталуу учурунда колдонуучуларыңызды демейки тизмеге да, пландын тизмесине да жаздыра аласыз_

## Ар бир план үчүн тизмелерди жана топторду тандоо {#selecting-lists-and-groups-for-each-of-the-plans}

**Plans** бөлүмүнө өтүп, түзөткүңүз келген планды тандаңыз. **Advanced Options** meta-box'унда MailChimp деп аталган жаңы өтмөк жеткиликтүү болорун эске алыңыз.

Ошол өтмөктө ошол план үчүн бир нече тизмени же бир тизмени жана бир нече топту (интеграция режимине жараша) тандай аласыз.

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_Ар бир планыңыз үчүн колдонгуңуз келген тизмелерди жана топторду тандаңыз_

Жаңы колдонуучу ошол планга катталганда, анын электрондук почта дареги тандалган тизмелерге жана топторго кошулат!
