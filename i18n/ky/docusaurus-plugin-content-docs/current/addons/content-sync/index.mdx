# Контентти синхрондоштуруу

Multisite Ultimate: Контентти синхрондоштуруу

Multisite Ultimate үчүн бул кошумча тармак администраторлоруна каалаган сайттан тармактагы башка сайттарга контентти синхрондоштурууга мүмкүндүк берет. Ал жазмалар, барактар, ыңгайлаштырылган жазма түрлөрү жана Elementor шаблондору сыяктуу бардык жазма түрлөрүн колдойт. Кошумча контентти жана максаттуу сайттарды тандоо үчүн көп кадамдуу устат интерфейсин, топтук иштетүүнү жана реалдуу убакыттагы жүрүш тууралуу пикирди камтыйт.

## Мүмкүнчүлүктөр

* Контентти тандоо: Миниатюралар жана жазма түрү боюнча чыпкалоо менен визуалдык карта торчосу
* Бардык жазма түрлөрү: Жазмаларды, барактарды, өнүмдөрдү, Elementor шаблондорун жана каалаган ыңгайлаштырылган жазма түрүн синхрондоштуруу
* Сайтты тандоо: Контент кайсы сайттарга синхрондоштуруларын тандаңыз, булак сайтты алып салуу жана туунду сайттарды алдын ала тандоо менен
* Топтук иштетүү: Жүрүш жаңыртуулары менен бир убакта 10 сайтка чейин синхрондоштурат
* Медиа синхрондоштуруу: Жазма контентинен медиа файлдарды (сүрөттөр, видеолор) автоматтык түрдө көчүрүп, максаттуу сайттарга кайра байланыштырат
* Таксономия синхрондоштуруу: Категорияларды, тегдерди жана ыңгайлаштырылган таксономияларды сактайт
* Elementor колдоосу: CSS кайра түзүү жана шаблон ID кайра байланыштыруу менен Elementor шаблондору үчүн толук колдоо
* Чыр-чатактарды чечүү: Учурдагы контентти алмаштыруу же өткөрүп жиберүү мүмкүнчүлүгү
* Кэшти башкаруу: Синхрондоштуруудан кийин максаттуу сайттардагы Elementor кэшин автоматтык түрдө тазалайт

## Талаптар

* WordPress 5.3 же андан жогору
* PHP 7.4 же андан жогору
* Multisite Ultimate плагини (активдүү)
* Elementor плагини (милдеттүү эмес, Elementor шаблонун синхрондоштуруу үчүн талап кылынат)

## Орнотуу

1. 'ultimate-multisite-content-sync' файлын '/wp-content/plugins/' каталогуна жүктөңүз
2. WordPress тармак администраторуңуздагы 'Плагиндер' менюсу аркылуу плагинди активдештириңиз
3. Ultimate Multisite → Жөндөөлөр → Контентти синхрондоштуруу бөлүмүнөн жөндөөлөрдү конфигурациялаңыз
4. Синхрондоштурууну баштоо үчүн Ultimate Multisite → Контентти синхрондоштуруу бөлүмүнө өтүңүз
