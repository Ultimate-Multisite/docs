# Content Sync өзгөрүүлөр журналы {#content-sync-changelog}

### Version 1.1.0 - 2025-12-31 күнү чыгарылды {#version-110---released-on-2025-12-31}
* Жаңы: Бардык жазма түрлөрүн синхрондоону колдоо (жазмалар, барактар, ыңгайлаштырылган жазма түрлөрү)
* Жаңы: Мазмун тандоо кадамында жазма түрү боюнча чыпкалоо
* Жаңы: Булак сайтты тандоо кадамы - негизги сайттан гана эмес, каалаган сайттан синхрондоо
* Жаңы: Үлгү сайттары үчүн бала сайтты алдын ала тандоо
* Жаңы: Бардык жазма түрлөрү үчүн post_content ичинен медианы чыгарып алуу
* Жаңы: Таксономияларды синхрондоо (категориялар, тегдер, ыңгайлаштырылган таксономиялар)
* Өзгөртүлдү: "Elementor Template Sync" аталышы "Content Sync" болуп өзгөртүлдү
* Өзгөртүлдү: "Select Templates" кадамынын аталышы "Select Data" болуп өзгөртүлдү
* Жакшыртылды: Elementor-спецификалык код эми elementor_library жазма түрү үчүн гана иштейт

### Version 1.0.0 - 2025-12-19 күнү чыгарылды {#version-100---released-on-2025-12-19}
* Баштапкы чыгарылыш
* Үлгү жана сайт тандоо үчүн көп кадамдуу устат
* Чыныгы убакыттагы жүрүш боюнча кайтарым байланыш менен топтомдук иштетүү
* Автоматтык ID кайра шайкештештирүүсү менен толук медиа синхрондоо
* Ultimate Multisite менен жөндөөлөр интеграциясы
* Тармак жөндөөлөрү аркылуу сайтты чыгарып салууну колдоо
* Синхрондоодон кийин Elementor кэшин тазалоо
* Үлгү таксономиясын сактоо
