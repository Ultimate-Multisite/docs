# Чыпка: signup_get_available_languages {#filter-signupgetavailablelanguages}

Алдыңкы бөлүктөгү сайтка катталуулар үчүн жеткиликтүү тилдердин тизмесин чыпкалайт.

Бул hook'ка бош массив өткөрүлсө, катталуу формасында бул жөндөөнүн чыгарылышы өчүрүлөт жана сайт түзүлгөндө демейки тил колдонулат. Мурда орнотулбаган тилдер алып салынат.

## Параметрлер {#parameters}

| Аты | Түрү | Сүрөттөмө |
|------|------|-------------|
| $available_languages | `array` | Жеткиликтүү тилдер. |
