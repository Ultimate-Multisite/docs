# Тил тандоочу

Multisite Ultimate: Катталууда тил тандоочу

Катталуу агымына тил тандоо талаасын кошуңуз.

## Орнотуу {#installation}

1. 'ultimate-multisite-language-selector' файлын '/wp-content/plugins/' каталогуна жүктөңүз
2. Плагинди WordPress Network Admin ичиндеги 'Плагиндер' менюсу аркылуу активдештириңиз

## Колдонуу боюнча колдонмо {#usage-guide}

Бул акысыз кошумча менен колдонуучуларыңызга катталуу агымында жана кирүү барагында кайсы тилди көргүсү келерин тандоого мүмкүнчүлүк бере аласыз.

[Лицензияңызды активдештиргенден](../../user-guide/getting-started/installing-ultimate-multisite.md) кийин, лицензияңыз кирүү мүмкүнчүлүгүн берген Ultimate Multisite кошумчаларын орното аласыз.

Ultimate Multisite жөндөөлөр барагыңызга өтүп, андан соң сол жактагы менюдан _Кошумчалар_ энбелгисин басыңыз.

![Кошумчалар барагы](/img/admin/addons-page.png)

_Language Selector орнотууну баштоо үчүн Кошумчалар барагына өтүңүз_

Ал жерде биз сунуштаган бардык кошумчаларды табасыз.

Акысыз кошумчалар биздин бардык кардарларга, алар катталган жазылуу планына карабастан, жеткиликтүү.

Кошумчанын кичи сүрөтүнүн астындагы _Азыр орнотуу!_ баскычын басыңыз. Андан кийин аны Плагиндер барагында активдештириңиз.

![Орнотуу баскычы бар Кошумчалар барагы](/img/admin/addons-page.png)

_Кошумчаны орнотуу үчүн баскычты басыңыз_

<!-- Скриншот жеткиликсиз: WP Плагиндер барагында Language Selector кошумчасы үчүн Network Activate баскычы көрсөтүлгөн -->

_Аны Плагиндер барагыңызда активдештириңиз_

Эми Ultimate Multisite'тын Жөндөөлөр менюсуна өтүп, _Тармак жөндөөлөрү_ өтмөгүнө кириңиз. Ал жерде, барактын ылдый жагында, _Тилдер_ деп аталган жаңы бөлүмдү табасыз.

![Жөндөөлөр барагы](/img/config/settings-general.png)

_Жапжаңы бөлүм пайда болду!_

Бардык тилдерди вариант катары көрсөтүү үчүн аны бош калтырсаңыз болот же кайсыларын жеткиликтүү кылгыңыз келерин тандасаңыз болот. Тилдер — бул WordPress орнотууңузда орнотулган тилдер.

<!-- Скриншот жеткиликсиз: WP орнотуусунда жеткиликтүү тилдерди көрсөткөн тил тандоо ачылма менюсу -->

_Бул WP орнотуусунда жеткиликтүү тилдердин тизмеси_

Киргизген өзгөртүүлөрүңүздү сактагандан кийин, сиз тандаган тилдери бар тандоо менюсу Кирүү барагында, ошондой эле Катталуу агымында көрсөтүлөт.

![Кирүү жана катталуу жөндөөлөрү](/img/config/settings-login-registration.png)

_Кирүү барагындагы Language Selector_

![Катталуу барагы](/img/config/registration-page.png)

_Катталуу агымындагы Language Selector_

## Тилди URL аркылуу коюу {#setting-the-language-via-the-url}

Ошондой эле URL'ге locale параметрин өткөрүп берип, барактын тилин өзгөртө аласыз. Мисалы, италия тилиндеги катталуу барагына багыттагыңыз келсе, URL'ге ?locale=it_IT кошуңуз.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Бул иштеши үчүн тил WordPress тармагыңызда орнотулган болушу керек.
