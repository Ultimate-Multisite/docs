# Tutor Multisite Compatibility

Multisite чөйрөлөрүндө Tutor LMS үчүн шайкештик оңдоолорун кошкон WordPress плагини.

## Сүрөттөмө {#description}

Бул плагин WordPress multisite тармактарында Tutor LMS иштетүү үчүн маанилүү шайкештик оңдоолорун камсыздайт. Ал колдонуучуларды каттоо, инструкторлорду башкаруу, курс чектөөлөрү жана башка multisiteке мүнөздүү функцияларды иштетет.

## Өзгөчөлүктөр {#features}

- **Маалымат базасынын таблицаларын башкаруу**: Multisite тышкы ачкыч чектөөлөрүнөн улам түзүлбөй калган Tutor LMS таблицаларын автоматтык түрдө аныктайт жана түзөт
- **Multisite колдонуучуларын каттоо**: Тармак сайттары боюнча колдонуучуларды каттоону башкаруу үчүн ыңгайлаштырылган AJAX иштетүүчүлөрү
- **Инструктор ролдорун башкаруу**: Multisite чөйрөлөрүндө инструктор ролун автоматтык дайындоо жана башкаруу
- **Курс чектөөлөрү**: Сайт пландарына жараша курс жана funnel түзүү чектөөлөрүн башкаруу үчүн Ultimate Multisite менен интеграция
- **Ыңгайлаштырылган админ көрүнүштөрү**: Multisiteти эске алган инструкторлор тизмеси жана башкаруу интерфейси
- **Админ куралдары**: Тармак жана сайт администраторлору үчүн маалымат базасын башкаруу интерфейси

## Талаптар {#requirements}

- WordPress 5.0+
- PHP 7.4+
- Tutor LMS плагини (`Requires Plugins` header аркылуу автоматтык түрдө талап кылынат)
- WordPress Multisite орнотуусу
- Ultimate Multisite плагини (курс чектөөлөрү функциясы үчүн)

## Орнотуу {#installation}

1. Плагин файлдарын `/wp-content/plugins/tutor-multisite-compatibility/` дарегине жүктөңүз
2. Tutor LMS орнотулганын жана активдештирилгенин текшериңиз
3. Плагинди WordPress ичиндеги 'Плагиндер' менюсу аркылуу активдештириңиз
4. Плагин Tutor LMS менен автоматтык түрдө интеграцияланат

## MU pluginден көчүрүлгөн функциялар {#functionality-moved-from-mu-plugin}

Бул плагин мурда `wp-content/mu-plugins/daves.php` ичинде болгон функцияларды камтыйт:

- Курстун статусун өзгөртүү чектөөлөрү
- Курс түзүү чектөөлөрү
- CartFlows интеграциясы жана чектөөлөрү
- Tutor инструктор ролдорун башкаруу
- Multisite колдонуучуларын каттоону иштетүү
- Курстун продукт IDсин көрсөтүү

## Маалымат базасын башкаруу {#database-management}

### Маселе {#the-problem}

WordPress multisite чөйрөлөрүндө Tutor LMS таблицаларын түзүү users таблицасына тышкы ачкыч чектөөлөрү бар таблицалар үчүн ишке ашпай калат. Бул төмөнкү себептерден улам болот:

- Сайтка тиешелүү таблицалар `wp_2_tutor_carts` сыяктуу префикстерди колдонот
- Тышкы ачкычтар `wp_2_users` таблицасына шилтеме берет (ал жок)
- Чыныгы users таблицасы — `wp_users` (тармак боюнча жалпы)

### Чечим {#the-solution}

Бул плагин автоматтык түрдө:

1. **Жок таблицаларды аныктайт**: Тышкы ачкыч чектөөлөрүнөн улам түзүлбөй калган таблицаларды текшерет
2. **Туура шилтемелер менен таблицаларды түзөт**: Туура user таблица атын колдонот (multisite ичинде `wp_users`)
3. **Бир нече сценарийди иштетет**:
   - Плагинди активдештирүү/кайра активдештирүү
   - Тармакта жаңы сайт түзүү
   - Админ интерфейси аркылуу таблицаларды кол менен текшерүү

### Таасир тийген таблицалар {#affected-tables}

- `tutor_coupon_usages` - Users таблицасына тышкы ачкычты камтыйт
- `tutor_carts` - Users таблицасына тышкы ачкычты камтыйт
- `tutor_notification_preferences` - Users таблицасына тышкы ачкычы бар TutorPro таблицасы

### Админ интерфейси {#admin-interface}

Маалымат базасын башкаруу куралдарына бул жерден кириңиз:

- **Тармак админи**: Жөндөөлөр → Tutor Multisite DB
- **Сайт админи**: Куралдар → Tutor Multisite DB

Интерфейс сизге төмөнкүлөргө мүмкүндүк берет:
- Бардык сайттар боюнча жок таблицаларды көрүү
- Таблицаларды түзүүнү кол менен иштетүү
- Маалымат базасынын конфигурациясын текшерүү
- Толук статус маалыматын көрүү
