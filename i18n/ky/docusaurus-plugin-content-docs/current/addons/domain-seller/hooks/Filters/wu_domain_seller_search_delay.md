# Filter: wu_domain_seller_search_delay {#filter-wudomainsellersearchdelay}

Домен издөө debounce кечигүүсүн миллисекунд менен чыпкалайт.

Жай туташууларда API чакырууларын азайтуу үчүн бул маанини көбөйтүңүз.

## Параметрлер {#parameters}

| Аты | Түрү | Сүрөттөмө |
|------|------|-------------|
| $delay | `int` | Debounce кечигүүсү миллисекунд менен. Демейки 500. |
