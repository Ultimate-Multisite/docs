# Домен сатуучунун өзгөрүүлөр журналы

Version 1.3.0 - 2026-06-02 күнү чыгарылды
- Жаңы: HostAfrica кайра сатуучусунун балансы өтө төмөндөп кеткенде тармак администраторуна эскертүү кошулду
- Жаңы: Жаңы катталган домендерди тармак сайтына автоматтык байланыштыруу кошулду
- Оңдоо: Каттоочунун талаа талаптары жаңы домен катталганда гана колдонулду
- Оңдоо: Балансты көзөмөлдөө билдирүүлөрүн жабууга мүмкүн болду
- Оңдоо: WooCommerce каттоочусунун эсеп-фактура маалыматтары сакталышы камсыздалды
- Оңдоо: Каттоо учурунда каттоочунун байланыш талаптары милдеттүү кылынды
- Оңдоо: Домен каттоо өнүмдөрүнүн 0% үстөк менен түзүлүшүнө жол берилбеди
- Оңдоо: Төлөм жол-жобосу сессиясынын агымы аркылуу домен тандоолору жана баалар сакталды
- Оңдоо: HostAfrica домен бааларынын валюта көрсөтүлүшү жакшыртылды
- Оңдоо: WP-core сурам өзгөрмөлөрүнүн дал келбестигин алдын алуу үчүн төлөм формасынын аракет жүрүм-туруму жакшыртылды
- Жакшыртылды: Орнотуу көрсөтмөсүнө HostAfrica кайра сатуучу конфигурациясынын документациясына шилтеме кошулду
