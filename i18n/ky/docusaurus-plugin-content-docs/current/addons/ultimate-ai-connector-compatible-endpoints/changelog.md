# Compatible Endpoints үчүн Ultimate AI Connector өзгөртүүлөр журналы

## 2.0.0 версиясы — 2026-04-24 чыгарылды {#version-200--released-on-2026-04-24}

* Жаңы: Көп-провайдер колдоосу — бир нече AI endpointти жөндөп, сурамдарды провайдерлер арасында автоматтык fallback менен багыттаңыз.
* Оңдоо: Туура провайдер IDлери, каттоо URLлери жана ар бир провайдер үчүн HTTP фильтр чөйрөсү менен көп-провайдер SDK интеграциясы.
* Оңдоо: Жаңы провайдер картасы эми кошулганда автоматтык түрдө ачылат; plugin жаңыртылганда скрипт кэшин жаңылоо.
* Оңдоо: eval() үчүн динамикалык провайдер класс namespace жана жаңы кошулган провайдерлер үчүн автоматтык ачылуу жүрүм-туруму.
* Оңдоо: WordPress 6.9+ шайкештиги үчүн туруктуу Card/CardBody/CardHeader/CardDivider компоненттерин колдонот (мындан ары эксперименталдык эмес).
* Оңдоо: Провайдерди сүйрөп кайра иреттөө үчүн аныкталбаган DragHandle unicode кармоо белгисине алмаштырылды.
* Оңдоо: Ар бир барак жүктөлгөндө иштей турган бөгөттөөчү HTTP сурамы жоюлду.
* Жакшыртылды: GitHub Actions иш агымдары Node.js 24кө жаңыртылды.
