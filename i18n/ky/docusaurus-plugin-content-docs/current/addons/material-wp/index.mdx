# Material WP

WordPress админ интерфейси көптөн бери өзгөрбөй келген. Бирок админ интерфейсинин Material Design версиясы кандай көрүнөрүн ойлонуп көрдүңүз беле? 

Material WP — сизге ошол көрүнүштү жана сезимди берүү үчүн иштелип чыккан админ темасы; ал кардарларыңызга көбүрөөк ыңгайлаштырылган Dashboard сунуштоого жардам берет.

## Мүмкүнчүлүктөр

- Ыңгайлаштырылган CSS колдоосу
- Оңой стиль жөндөөлөрү
- Менюну кайра иреттөө
- Оңой импорт/экспорт

## Документация

**Material WP менен админ панелиңизди ыңгайлаштыруу**

## Орнотуу

1. 'material-wp' файлын '/wp-content/plugins/' каталогуна жүктөңүз
2. Плагинди WordPress ичиндеги 'Плагиндер' менюсу аркылуу активдештириңиз

## Колдонуу колдонмосу

WordPress админ интерфейси көптөн бери өзгөрбөй келген. Бирок админ интерфейсинин Material Design версиясы кандай көрүнөрүн ойлонуп көрдүңүз беле?

Material WP — сизге ошол көрүнүштү жана сезимди берүү үчүн иштелип чыккан админ темасы; ал кардарларыңызга көбүрөөк ыңгайлаштырылган Dashboard сунуштоого жардам берет. Material WP ошондой эле Ultimate Multisite’тын премиум плагиндер өзөгүнүн бир бөлүгү.

Аны **орноткондон кийин**, _Плагиндер_ барагыңызга өтүп, аны Тармагыңызда активдештирүү үчүн чыкылдатыңыз.
