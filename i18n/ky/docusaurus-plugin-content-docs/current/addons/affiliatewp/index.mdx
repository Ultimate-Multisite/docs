# AffiliateWP

Multisite Ultimate: AffiliateWP интеграциясы

Ultimo тармагыңыздын кардарлар базасын өстүрүү үчүн кубаттуу AffiliateWP колдонунуз!

## Орнотуу

1. 'ultimate-multisite-git ' файлын '/wp-content/plugins/' каталогуна жүктөңүз
2. Плагинди WordPress ичиндеги 'Плагиндер' менюсу аркылуу иштетиңиз
3. Плагинди жөндөө үчүн кадам-кадам устатты ээрчиңиз

## Колдонуу колдонмосу

[AffiliateWP](https://affiliatewp.com/) онлайн бизнесиңиз үчүн өнөктөштүк программа түзүүдө негизги чечим болуп калды. Ultimate Multisite: AffiliateWP — премиум тармагыңыз үчүн ушундай эле нерсени түзүүгө мүмкүнчүлүк бере турган көпүрө.

## Баштоо

Эң оболу, сизде [**AffiliateWP**](https://affiliatewp.com/) **активдүү болушу керек**! Эгер сизде **аны менен интеграциялануучу плагин** жок болсо, биздин кошумчаны колдоно албайсыз. Кардарларыңыз аны өз сайттарында да колдонсун деген планыңыз жок болсо, AffiliateWP негизги сайтыңызда гана активдүү болушу керек.

Ошондой эле **Ultimate Multisite** жана **Ultimate Multisite: AffiliateWP** экөө тең **тармак боюнча** **активдештирилген** болушу керек.

## Ultimate Multisite'ты AffiliateWP интеграциясы катары кошуу

Негизги сайтыңызда Өнөктөштөр жөндөөлөрүнө өтүп, Интеграциялар өтмөгүндө Ultimate Multisite'ты белгилеңиз. Бул [AffiliateWP](https://affiliatewp.com/) үчүн Ultimate Multisite катталуу агымынан келген кирүүлөрдү жана конверсияларды көзөмөлдөй баштоону билдирет.

<!-- Скриншот жеткиликсиз: AffiliateWP Интеграциялар жөндөөлөрү өтмөгү, Ultimate Multisite белгилөө кутучасы менен -->

_Төмөн жактагы Ultimate Multisite кутучасын белгилеп, сактаңыз_

Мунун баары ушунча эле. Рефералдык шилтемелерден келген кирүүлөр көзөмөлдөнөт жана ал колдонуучу кийин биринчи төлөмүн жасаганда, конверсия түзүлүп, конверсия үчүн жооптуу өнөктөшкө комиссия бөлүнүп коюлат.

Бирок жаңы түзүлгөн колдонуучудан алынган биринчи төлөм гана көзөмөлдөнөт. Кайталануучу рефералдык төлөмдөрдү иштетүү үчүн сизге [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) керек болот

## Кайталануучу рефералдарды иштетүү

[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) орнотулгандан кийин, Өнөктөштөр жөндөөлөрүнүн Кайталануучу рефералдар өтмөгүнө өтүп, кайталануучу рефералдарды колдонууну иштетиңиз.

_Өтө жөнөкөй: Кайталануучу рефералдарды иштетиңиз — баары даяр!_
