# Колдоо агенттери {#support-agents}

Бүтүндөй тармакты өз алдынча башкаруу оңой иш эмес. Тармагыңыз өсө баштаганда, колдоого болгон суроо-талаптын көбөйгөнүн байкай баштайсыз. Жумуш жүгүңүздү жеңилдетүүгө жардам бере турган адамдарды тартуу жакшы ой болот. Бирок толук жеткиликтүүлүк/уруксат берүү кооптуу жана тармагыңыздын коопсуздугуна доо кетириши мүмкүн. Дал ушул жерде бул Колдоо агенттери кошумча модулу жардамга келет.

Бул кошумча модуль Support Agent деп аталган WordPress колдонуучусунун жаңы түрүн түзүүгө мүмкүндүк берет жана ал колдонуучунун кандай уруксаттары бар экенин толугу менен көзөмөлдөөгө мүмкүнчүлүк берет. Уруксат берүү которгуч баскычты күйгүзгөндөй эле оңой, ал эми алып салууну чечсеңиз, аны өчүрүп койсоңуз болот.

## Мүмкүнчүлүктөр {#features}

- WordPress жана Ultimate Multisite уруксаттарынын пункттарга бөлүнгөн тизмеси
- Жеткиликтүүлүктү оңой күйгүзүү жана өчүрүү
- Agent аккаунттарынын чексиз санын түзүү

## Орнотуу {#installation}

1. 'ultimate-multisite-support-agents' файлын '/wp-content/plugins/' каталогуна жүктөңүз
2. Плагинди WordPress ичиндеги 'Плагиндер' менюсу аркылуу активдештириңиз
3. Плагинди жөндөө үчүн кадам-кадам устаттагы нускамаларды аткарыңыз
