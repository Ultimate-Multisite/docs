# Чыпка: wu_cap_captcha_config

Cap Captcha конфигурациясын чыпкалаңыз.

Иштеп чыгуучуларга Cap Captcha жөндөөлөрүн ыңгайлаштырууга мүмкүндүк берет.

## Параметрлер {#parameters}

| Аталышы | Түрү | Сүрөттөмө |
|------|------|-------------|
| $config | `array` | Cap конфигурациясынын массиви. |
| $level | `string` | Коопсуздук деңгээли (fast, medium, max). |

### Версиядан баштап {#since}

- 1.2.0
