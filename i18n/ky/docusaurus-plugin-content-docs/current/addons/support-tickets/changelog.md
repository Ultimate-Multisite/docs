### 1.0.3 - 2026-05-01 {#103---2026-05-01}
* Оңдоо: Nullable билет моделинин setterлеринде null маанилерин кабыл алуу
* Оңдоо: Туура multisite колдоосу үчүн бардык билеттерди глобалдык тармак боюнча таблицаларда сактоо
* Оңдоо: Кардардын жаңы билет формасында кызматкерлер талааларын жашыруу жана бош баракка багыттоо көйгөйүн оңдоо
* Оңдоо: Метамаалыматты сактоо үчүн аныкталбаган add_meta() чакырууларын туура update_meta() менен алмаштыруу
* Оңдоо: Катталбаган мүмкүнчүлүк текшерүүсүн туура wu_view_all_support_tickets менен алмаштыруу
* Оңдоо: Билет статусу, дайындоо жана тез түзөтүү аракеттери үчүн жетишпеген AJAX иштетүүчүлөрүн кошуу
* Оңдоо: Билет көрүнүштөрүндө кызматкер жообун аныктоо үчүн методдун атын тууралоо
* Оңдоо: Кайталанган жооп иштетүүчүлөрүн бириктирүү жана nonce аракет аттарын шайкеш келтирүү
* Оңдоо: [wu_submit_ticket] shortcode үчүн жетишпеген frontend көрүнүшүн кошуу
* Оңдоо: Жетишпеген user_id тилкесин кошуу жана Support_Ticket::get_user_id() методун оңдоо
* Оңдоо: Артыкча кош тырмакчаны priority чыпкасынын option тегинен алып салуу
* Оңдоо: Супер администраторлор үчүн тармак администраторунун билет башкаруу панелин кошуу
* Жакшыртылды: Администратор CSS бир тышкы стиль барагына бириктирилди
* Жакшыртылды: Подсайт администратор менюсунан Settings ички менюсу алып салынды
* Жакшыртылды: Frontend ресурстарын колдоо билет барактарында гана шарттуу жүктөө
* Жакшыртылды: Bedrock түпкү autoloader көз карандылыктарды буга чейин жүктөгөндө plugin autoloader өткөрүп жиберилет

### 1.0.2 - 2025-12-11 {#102---2025-12-11}
* Кошулду: Администратор жана frontend үчүн толук билет көрүү функционалдуулугу
* Кошулду: Билетке жооп жөнөтүүлөрү үчүн AJAX иштетүүчү
* Кошулду: Туура форма иштетүү менен билетке жооп берүү функционалдуулугун колдоо
* Кошулду: Администратордо билет жөнөтүүлөрү жана жооптору үчүн туура билдирүү көрсөтүү
* Кошулду: Кардардын өз алдынча жөнөтүүлөрү үчүн билеттерди учурдагы колдонуучу менен автоматтык байланыштыруу
* Кошулду: Кардарлардын билет ээлигин өзгөртүп жиберишин алдын алуу үчүн коопсуздук жакшыртуусу
* Кошулду: Жетишпеген жардамчы функциялар (wu_format_date, wu_user_can_view_ticket, ж.б.)
* Кошулду: Файл тиркемелерин туура жүктөп алуу жана иштетүү
* Кошулду: Билет жооптору жана статус өзгөрүүлөрү үчүн email билдирүү системасы
* Оңдолду: Билетке жооп берүү формасы эми керектүү билет ID камтыйт
* Оңдолду: Администратордун билет көрүнүштөрүндөгү билдирүү көрүнүмдүүлүгү маселелери
* Оңдолду: Колдоо билети функцияларындагы синтаксис каталары
* Жакшыртылды: Жооптор жана тиркемелер үчүн туура суроо жана схема класстары бар маалымат базасынын түзүмү
* Жакшыртылды: Билет моделдери үчүн атрибуттардын ордуна чыныгы касиеттерге миграция
