# Өзгөртүүлөр журналы {#changelog}

## 1.9.0 — 2026-04-28 күнү чыгарылды {#190--released-on-2026-04-28}

### Жаңы {#new}

- **`create_contact_form` жөндөмү** — активдүү форма pluginин (Contact Form 7, WPForms, Fluent Forms же Gravity Forms) колдонуп байланыш формасын түзөт жана каалаган жазмага же баракка жайгаштырууга даяр shortcode кайтарат.
- **`set_featured_image` жөндөмү** — бар болгон Media Library тиркемесинин IDсинен же алыстагы URLден жазмага же баракка негизги сүрөт дайындайт; URL берилгенде сүрөттү автоматтык түрдө импорттойт.
- **`batch_create_posts` жөндөмү** — бир жөндөм чакыруусунда бир нече жазма түзөт. `create_post` менен бирдей параметрлерди колдойт, ар бир жазма боюнча ийгилик/ийгиликсиздик тууралуу кабарлайт жана кошумча `stop_on_error` режимине ээ.
- **`page_template` параметри** — `create_post` жана `update_post` үчүн кошулду. Түзүү же жаңыртуу учурунда PHP барак шаблон файлын (мисалы, `page-full-width.php`) дайындайт. Тема демейкисине кайтуу үчүн `update_post`ко бош сап өткөрүңүз.
- **Кардар тараптагы скриншот жөндөмдөрү** — `capture_screenshot`, `compare_screenshots` жана `review_page_design`. Сервер тараптагы headless браузер аркылуу жандуу барактардан толук же viewport скриншотторун тартат, эки скриншотту салыштырат жана жайгашуу, типография, түс жана жеткиликтүүлүк боюнча AI түзгөн дизайн кароосун берет.
- **Беш камтылган агент** — Content Writer, Site Builder, Design Studio, Plugin Manager жана Support Assistant. Ар бир агентте атайын куралдар топтому, ылайыкташтырылган system prompt жана баштапкы сунуштар бар. Чат башындагы жаңы **Agent Picker** аркылуу алмаштырылат. [Камтылган агенттер](../../user-guide/configuration/built-in-agents) бөлүмүн караңыз.
- **Функция желектери** — жаңы **Settings → Feature Flags** өтмөгү, кирүү мүмкүнчүлүгүн көзөмөлдөө которгучтары (администраторлор менен чектөө, тармак админдери менен чектөө, жазылуучунун кирүүсү, мүчө эместер үчүн өчүрүү) жана брендинг параметрлери (footer атрибуциясын жашыруу, ыңгайлаштырылган агент аты, agent pickerди жашыруу, сайт сүрөтчөсүн чат аватары катары колдонуу) менен. [Gratis AI Agent Settings](../../user-guide/administration/gratis-ai-agent-settings) бөлүмүн караңыз.
- **Акыркы сессияны калыбына келтирүү** — чат панели эми барак жүктөлгөндө жана виджет ачылганда эң акыркы сүйлөшүүнү автоматтык түрдө кайра жүктөйт, ошондуктан барактар арасында өтүүдө контекст эч качан жоголбойт.
- **Plugin аракет шилтемелери** — Settings жана Abilities Registry үчүн тез шилтемелер эми WordPress **Plugins → Installed Plugins** экранында plugin сүрөттөмөсүнүн астында көрүнөт.

### Жакшыртылды {#improved}

- **Сүрөт булагын кайра аракет кылуу** — агент эми жүктөө ишке ашпай калганда AI түзгөн сүрөткө өтүүдөн мурун бардык конфигурацияланган акысыз сүрөт булактарын кайра сынайт.
- **Модель маалымат панели** — чат башында ар дайым көрүнүп турат; биринчи билдирүүдөн кийин мындан ары жашырылбайт.
- **Авто-жылдырып жүрүм-туруму** — колдонуучу окуу үчүн өйдө жылдырганда авто-жылдырып токтойт; калкыма **Төмөнгө жылдыруу** баскычы пайда болуп, колдонуучу акыркы билдирүүгө жеткенде автоматтык түрдө жок болот.
- **Agent Picker UI** — ар бир агенттин сүрөтчөлөрү бар form-table overlay катары кайра иштелип чыкты, бул агенттерди бир караганда аныктап, алмаштырууну жеңилдетет.
- **Жалкоо жүктөлүүчү JS бөлүктөрү** — чат виджетинин баштапкы JavaScript топтому эми жалкоо жүктөлүүчү бөлүктөргө бөлүнүп, баштапкы топтом өлчөмдөрүн 75–90%га азайтат.
- **Чат виджетин кайра дизайн кылуу** — мурунку ыңгайлаштырылган аватарды бирдиктүү AI сүрөтчөсү алмаштырат; камтылган агент системасына шайкеш.
- **URL шилтемелештирүү** — system билдирүүлөрүндө жана ката билдирүү көбүкчөлөрүндө көрүнгөн URLдер эми басылуучу шилтемелер катары көрсөтүлөт.

### Оңдолду {#fixed}

- **Жөндөмдөрдүн табылышы** — бардык жөндөмдөр агенттин куралдар тизмесинде ишенимдүү көрүнүшү үчүн сүрөттөмөлөр, system prompt шилтемелери жана namespace шайкештиги оңдолду.
- **Провайдерлер кэши** — провайдерлер эми версия эсептегичи аркылуу сайт боюнча кэштелет, бул multisite тармактарында эскирип калган провайдер маселелерин алдын алат.
- **`ability_invalid_output`** — 12 жөндөм иштетүүчүсүндө чечилди; баары туура түзүмдөлгөн JSON жоопторун кайтарат.
- **`pending_client_tool_calls` pipeline** — кардар тараптагы курал чакыруулары туура аяктап, натыйжаларды модельге кайтарышы үчүн башынан аягына чейин туташтырылды.
- **Тарых тартмасы** — артка кайтарылбай турган өзгөрүүлөр revert тизмесинен чыгарылды; **Толук тарыхты көрүү** шилтемеси эми туура иштейт.
- **Өзгөртүүлөр/артка кайтаруу системасы** — беш өзүнчө ката оңдолуп, жаңы admin интерфейсинин астында бириктирилди.
- **Save Settings toast** — **Save Settings** басылгандан кийин snackbar эскертмеси эми ишенимдүү көрүнөт.
- **Таштанды контекст менюсу** — **Биротоло жок кылуу** опциясы кошулду, ошондуктан элементтер таштанды көрүнүшүнөн чыкпай эле толук жок кылына алат.
- **Түзөтүү жана кайра жөнөтүү** — түзөтүү сүрөтчөсүн басуу эми тизмектеги бардык билдирүүлөр үчүн эмес, басылган билдирүү үчүн гана түзөтүү режимине киргизет.
- **Чат жайгашуусунун бийиктиги** — барактын үстүндө plugin киргизген контент (admin эскертмелери, баннерлер) пайда болгондо чат панели бийиктигин ылайыктап, киргизүү аймагы экрандан чыгып кетишин алдын алат.

## 1.4.0 — 2026-04-09 күнү чыгарылды {#140--released-on-2026-04-09}

### Жаңы {#new-1}

- **WP-CLI benchmark command** (`wp gratis-ai-agent benchmark`) — CI pipelines жана модель баалоо иш агымдары үчүн буйрук сабынан Agent Capabilities v1 benchmark suite иштетет. Ар бир суроо боюнча иштетүүнү, provider/model кайра аныктоолорун жана JSON/CSV чыгаруусун колдойт.
- **Agent Capabilities v1 benchmark suite** — толук жөндөмдөр аймагын текшерген татаал, көп кадамдуу promptтордун түзүмдөлгөн топтому; баалоо жана token/узактык отчеттору менен.
- **Custom Post Type жөндөмдөрү** — `register_post_type`, `list_post_types`, `delete_post_type`. Каттоолор options таблицасы аркылуу кайра иштетилгенден кийин да сакталат.
- **Custom Taxonomy жөндөмдөрү** — `register_taxonomy`, `list_taxonomies`, `delete_taxonomy`. Кошумча rewrite slugтары менен иерархиялык жана жалпак таксономияларды колдойт.
- **Design System жөндөмдөрү** — `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset`. Беш тандалган preset камтыйт: `minimal-dark`, `warm-editorial`, `corporate-blue`, `vibrant-startup`, `classic-blog`.
- **Global Styles жөндөмдөрү** — `get_global_styles`, `set_global_styles`, `reset_global_styles`. theme.json маанилерин WordPress Global Styles API аркылуу окуйт жана жазат.
- **Navigation Menu башкаруу жөндөмү** — `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus`. Ичме менюларды жана theme жайгашуу дайындоосун колдойт.
- **Options Management жөндөмү** — `get_option`, `set_option`, `delete_option`, `list_options`. Маанилүү WordPress параметрлерин өзгөртүүдөн коргогон камтылган коопсуздук blocklist камтыйт.
- **Installable Abilities Registry** — `list_available_abilities`, `install_ability`, `recommend_plugin`. WordPress pluginдери катары таратылган жөндөм топтомдорун табыңыз жана активдештириңиз.
- **Site Builder Orchestration v2** — `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`. Plugin табуу, кадам чыгарылышына шилтемелер, прогрессти көзөмөлдөө жана автономдуу ката калыбына келтирүү менен көп фазалуу сайт куруу пландары.
- **Ресторан вебсайтынын benchmark суроосу** (`q-restaurant-website`) — CPT каттоосун, design systemди, навигацияны жана plugin табууну камтыган баштан-аяк тест.
- Жергиликтүү иштеп чыгуу орнотуусун тездетүү үчүн WordPress Playground blueprintтерине **AI provider connector pluginдери** кошулду.

### Жакшыртылды {#improved-1}

- README AI provider connector документтештирүүсү жана орнотуу нускамалары менен жаңыртылды.

### Оңдолду {#fixed-1}

- `main` бутагындагы 25 PHPUnit тест мүчүлүштүгү чечилди.
- `blueprint.json` ичиндеги GitHub releases URL форматы оңдолду.
- Эски IDлер менен кагылышууларды болтурбоо үчүн тапшырма IDлерин кайра номерлөө.
