# Gratis AI Agent {#gratis-ai-agent}

Gratis AI Agent — табигый тилдеги нускамалар аркылуу толук WordPress сайттарын пландап, куруп жана башкара алган автономдуу WordPress AI жардамчысы. Ал агент чакыра ала турган **жөндөмдөрдүн** түзүмдөлгөн топтомун — өз-өзүнчө, текшериле турган аракеттерди — сунуштайт; булар контентти башкаруудан тартып дизайн системасын ыңгайлаштырууга чейинки бардык нерсени камтыйт. 1.9.0 версиясы контент түзүү жөндөмдөрүн, кардар тараптагы скриншоттор аркылуу баракты визуалдык кароону, беш камтылган агент профилин жана кирүүнү көзөмөлдөө менен брендинг үчүн функция желектерин кошот.

## Негизги мүмкүнчүлүктөр {#key-features}

- **Жөндөмдөр системасы** — агент сиздин WordPress орнотууңузда чакыра ала турган аракеттердин модулдук, кеңейтилүүчү каталогу
- **Контентти башкаруу** — жазмаларды жана барактарды түзүү (анын ичинде барак шаблондорун дайындоо), бир чакырууда бир нече жазманы топтом менен түзүү, өзгөчөлөнгөн сүрөттөрдү коюу жана байланыш формаларын куруу
- **Визуалдык кароо** — жандуу барактардын скриншотторун тартуу, мурунку/кийинки абалдарды салыштыруу жана жайгашуу, типография, түс жана жеткиликтүүлүк боюнча AI түзгөн дизайн кароосун алуу
- **Беш камтылган агент** — Контент жазуучу, Сайт куруучу, Дизайн студиясы, Plugin башкаргычы жана Колдоо жардамчысы; чат башындагы Agent Picker аркылуу которулат
- **Функция желектери** — ролдук чектөөлөр, white-label аталышы жана виджетти ыңгайлаштыруу үчүн Settings → Feature Flags ичиндеги кирүүнү көзөмөлдөө жана брендинг которгучтары
- **Custom Post Type башкаруу** — custom post type түрлөрүн кайра иштетүүлөр арасында сакталуусу менен каттоо, тизмелөө жана өчүрүү
- **Custom Taxonomy башкаруу** — custom taxonomy түрлөрүн сакталуусу менен каттоо, тизмелөө жана өчүрүү
- **Дизайн системасынын жөндөмдөрү** — custom CSS киргизүү, тандалган блок үлгүлөрүн башкаруу, сайт логотибин коюу жана theme.json пресеттерин колдонуу
- **Global Styles башкаруу** — түс палитралары, типография, аралыктар жана башка нерселер үчүн theme.json маанилерин окуу жана жазуу
- **Навигация менюсун башкаруу** — навигация менюларын жана алардын элементтерин программалык түрдө түзүү жана жаңыртуу
- **Параметрлерди башкаруу** — маанилүү жөндөөлөрдү коргоо үчүн камтылган коопсуздук бөгөт тизмеси менен WordPress параметрлерин окуу жана жазуу
- **Орнотула турган жөндөмдөр реестри** — негизги plugin файлдарын өзгөртпөстөн кошумча жөндөм топтомдорун табуу жана активдештирүү
- **Сайт куруучу оркестрациясы v2** — plugin табуу, прогрессти көзөмөлдөө жана катадан калыбына келтирүү менен көп кадамдуу сайт куруу пландары
- **WP-CLI benchmark буйругу** — CI жана моделди баалоо үчүн агент мүмкүнчүлүктөрүнүн benchmark топтомун буйрук сабынан иштетүү

## Талаптар {#requirements}

- WordPress 6.0 же андан жогору
- PHP 8.1 же андан жогору
- WordPress Multisite (тармак боюнча активдештирилген) же бир сайттуу орнотуу
- Активдүү AI provider connector plugin (Anthropic, OpenAI же шайкеш)

## Орнотуу {#installation}

1. Plugin файлдарын `/wp-content/plugins/gratis-ai-agent` ичине жүктөңүз
2. Pluginди WordPress ичиндеги **Plugins** экраны аркылуу активдештириңиз
3. **Gratis AI Agent → Settings** бөлүмүнө өтүп, AI провайдериңизди туташтырыңыз
4. (Милдеттүү эмес) Кошумча жөндөм топтомдорун орнотуу үчүн **Abilities Registry** бөлүмүн карап чыгыңыз

## Ыкчам баштоо {#quick-start}

Активдештирилип, AI провайдерине туташкандан кийин, WordPress админ каптал тилкесинен **Gratis AI Agent** панелин ачыңыз. Табигый тилде нускама жазыңыз, агент жеткиликтүү жөндөмдөрүн колдонуп керектүү кадамдарды пландап жана аткарат.

**Мисал нускамалар:**

```
"Portfolio" custom post type түрүн "Project Category" taxonomy менен түзүңүз.
```

```
Theme ичине караңгы режимдин түс палитрасын киргизип, негизги түстү #1a1a2e кылып коюңуз.
```

```
Онлайн менюсу, брондоо формасы жана байланыш барагы бар ресторан вебсайтын куруңуз.
```

## Жөндөмдөргө сереп {#abilities-overview}

Жөндөмдөр — агент аткара ала турган атомдук аракеттер. Ар бир жөндөм — агент иштөө учурунда окуй турган схеманы (параметрлер жана кайтаруу түрлөрү) сунуштаган катталган PHP классы. Жөндөмдөр функционалдык аймактарга топтолгон:

| Аймак | Жөндөмдөр |
|---|---|
| **Контентти башкаруу** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **Визуалдык кароо** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **Custom Post Types** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **Custom Taxonomies** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **Дизайн системасы** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **Global Styles** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **Навигация менюлары** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **Параметрлерди башкаруу** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **Орнотула турган жөндөмдөр** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **Сайт куруучу** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

Толук параметр документтери жана колдонуу мисалдары үчүн [Жөндөмдөр маалымдамасын](./abilities) караңыз.

## WP-CLI {#wp-cli}

Gratis AI Agent буйрук сабынан benchmark иштетүү жана жөндөмдөрдү башкаруу үчүн `wp gratis-ai-agent` буйрук үй-бүлөсү менен келет. Толук буйрук тизмеси үчүн [WP-CLI маалымдамасын](./wp-cli) караңыз.

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## Конфигурация {#configuration}

Бардык жөндөөлөр WordPress админ ичиндеги **Gratis AI Agent → Settings** астында жайгашкан:

| Жөндөө | Сүрөттөмө |
|---|---|
| **AI Provider** | Активдүү AI провайдер коннекторун тандап жана конфигурациялаңыз |
| **Model** | Агент сессиялары үчүн колдонулуучу тил моделин тандаңыз |
| **Max Tokens** | Ар бир сурам үчүн токен бюджети (демейки 4096) |
| **Ability Timeout** | Жеке жөндөм иштей ала турган эң көп секунд (демейки 30) |
| **Debug Logging** | Жөндөм чакырууларын жана агенттин ой жүгүртүүсүн WordPress debug журналына жазыңыз |

## Параметрлерди башкаруунун коопсуздук бөгөт тизмеси {#options-management-safety-blocklist}

Параметрлерди башкаруу мүмкүнчүлүгүндө агенттин сезимтал WordPress параметрлерин окушун же жазышын алдын алган камтылган бөгөт тизмеси бар. Демейки бөгөт тизмеси төмөнкүлөрдү камтыйт:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (URL параметрлери — булар үчүн атайын көчүрүү куралын колдонуңуз)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (жашыруун ачкычтар)
- `active_plugins`, `template` (плагин/тема активдештирүү — атайын мүмкүнчүлүктөрдү колдонуңуз)
- Орнотулган плагиндер тарабынан катталган төлөм шлюзунун эсептик дайындары жана API ачкычтары

Кошумча жазууларды `gratis_ai_agent_options_blocklist` чыпкасы аркылуу кошсо болот.

## Илгичтер жана чыпкалар {#hooks-and-filters}

### Аракеттер {#actions}

- `gratis_ai_agent_ability_registered` — мүмкүнчүлүк катталгандан кийин иштетилет, мүмкүнчүлүк классынын атын алат
- `gratis_ai_agent_plan_started` — сайт куруучу план аткарууну баштаганда иштетилет
- `gratis_ai_agent_plan_completed` — план ийгиликтүү аяктаганда иштетилет
- `gratis_ai_agent_plan_failed` — план калыбына келтирилгис катага туш болгондо иштетилет

### Чыпкалар {#filters}

- `gratis_ai_agent_abilities` — агент аны жүктөгөнгө чейин катталган мүмкүнчүлүктөр тизмесин өзгөртүңүз
- `gratis_ai_agent_options_blocklist` — параметрлердин коопсуздук бөгөт тизмесин кеңейтиңиз же кыскартыңыз
- `gratis_ai_agent_benchmark_questions` — баалоо топтомуна ыңгайлаштырылган эталон суроолорун кошуңуз
- `gratis_ai_agent_plan_plugins` — сайт планы түзүлүп жатканда плагиндерди табуу тизмесин өзгөртүңүз
