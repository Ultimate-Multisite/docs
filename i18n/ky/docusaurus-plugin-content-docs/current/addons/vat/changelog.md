# VAT өзгөрүүлөр журналы

Version 1.0.7 - 2026-02-03 күнү чыгарылды

* Оңдоо: German (DE) VAT номерлери VIES SOAP service MS_MAX_CONCURRENT_REQ каталарынан улам текшерүүдөн өтпөй жаткан. Ишенимдүүрөөк текшерүү үчүн эски SOAP APIден расмий EU VIES REST APIге которулду.
* Оңдоо: VIES service убактылуу жеткиликсиз болгондо VAT номерлери эми туура эмес четке кагылбастан кабыл алынат. Кийин карап чыгуу үчүн журнал жазуусу түзүлөт.
* Оңдоо: VAT/Tax ID талаасы эми EU мүчөлөрү үчүн гана эмес, бардык өлкөлөр үчүн көрүнөт. EU эмес салык IDлери (мис., Swiss CHE номерлери) эсеп-фактурада көрсөтүү үчүн VIES текшерүүсүз сакталат.
* Оңдоо: Greece (GR), Monaco (MC) жана Isle of Man (IM) үчүн VAT префиксин издөө массивден издөө логикасы туура эмес болгондуктан бузулган.
* Жакшыртуу: Site Exporter кошумчасы менен бирге колдонулганда олуттуу каталарды жаратышы мүмкүн болгон керексиз wp-cli-bundle dev көз карандылыгы алып салынды.
* Жакшыртуу: Толук unit test топтому кошулду (53 тест).

Version 1.0.6 - 2026-01-25 күнү чыгарылды

* Оңдоо: Ички B2B транзакциялары эми VATты туура эсептейт. EU VAT эрежелерине ылайык, reverse charge кардардын өлкөсү компаниянын өлкөсү менен дал келгенде эмес, чек ара аралык B2B транзакцияларына гана колдонулат.

Version 1.0.5 - 2026-01-22 күнү чыгарылды

* Оңдоо: Кошумча жүктөлбөй же туура иштебей жаткан.
* Өзгөртүү: Ишенимдүүрөөк жана активдүү колдоого алынган дайындар үчүн VAT ставкасынын дайындар булагы euvatrates.com сайтынан ibericode/vat-rates репозиторийине которулду.
* Оңдоо: super_reduced_rates параметринин маанисиндеги тамга катасы оңдолду.
* Жакшыртуу: Каталарды алдын алуу үчүн өлкө кодун иштетүүгө null текшерүүлөрү кошулду.
* Өзгөртүү: Traduttore аркылуу автоматтык жаңыртуулардын пайдасына топтомдолгон котормо файлдары алып салынды.
