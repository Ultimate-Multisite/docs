# VAT

Multisite Ultimate: Европалык VAT

Европалык VAT'ты жакшыраак колдоо үчүн Multisite Ultimate'тин демейки салык иштетүүсүнө кошумча мүмкүнчүлүктөрдү жана куралдарды кошот.
EU VAT чендерин Салык чендери барагына автоматтык түрдө кошот

## Орнотуу {#installation}

1. 'ultimate-multisite-vat' папкасын '/wp-content/plugins/' каталогуна жүктөңүз
2. WordPress ичиндеги 'Плагиндер' менюсу аркылуу плагинди активдештириңиз
3. Ultimate Multisite > Орнотуулар > Салыктар бөлүмүнө барып, EU VAT'ты иштетиңиз.
4. Салык чендерин басып, ылдый жагындагы EU VAT чендерин жаңыртыңыз жана Салык чендерин сактаңыз.
5. Даяр!
