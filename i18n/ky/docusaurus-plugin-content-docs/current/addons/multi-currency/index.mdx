# Көп валюталуу

Ultimate Multisite: Көп валюталуу сиздин тармакка бир нече валютада төлөмдөрдү кабыл алууга мүмкүндүк берет. Ал бааларды жандуу же кол менен коюлган алмашуу курстары аркылуу автоматтык түрдө которот, каалаган валюта үчүн ар бир өнүмгө так бааларды коюуга мүмкүндүк берет жана геолокация аркылуу ар бир келүүчүнүн артык көргөн валютасын аныктайт — мунун баары бир админ Dashboard аркылуу башкарылат.

## Негизги мүмкүнчүлүктөр {#key-features}

- **Алмашуу курсун автоматтык которуу** — баалар сиз тандаган камсыздоочудан алынган курстарды колдонуп, негизги валютаңыздан которулат
- **Ар бир өнүм үчүн бааны өзүнчө белгилөө** — автоматтык которууну айланып өтүү үчүн каалаган өнүмдө валютага тиешелүү так бааларды коюңуз
- **Катталуудагы валюта тандоо талаасы** — кардарларга төлөө/катталуу учурунда артык көргөн валютасын тандоого мүмкүндүк берет
- **Бир нече алмашуу курсу камсыздоочулары** — ECB, Frankfurter, Stripe же толугу менен кол менен коюлган курстар
- **Геолокацияга негизделген валюта аныктоо** — келүүчүнүн өлкөсүнө ылайык келген валютаны автоматтык түрдө алдын ала тандайт
- **Админ Dashboard** — активдүү валюталарды, алмашуу курстарын жана камсыздоочу жөндөөлөрүн бир жерден башкарыңыз

## Талаптар {#requirements}

- Ultimate Multisite 2.0.0 же жогору
- WordPress көп сайттуу орнотуусу
- PHP 7.4 же жогору

## Орнотуу {#installation}

1. Плагин файлдарын `/wp-content/plugins/ultimate-multisite-multi-currency` дарегине жүктөңүз
2. Плагинди WordPress ичиндеги **Плагиндер** экраны аркылуу активдештириңиз (Тармак боюнча активдештирүү)
3. Админ Dashboard ачуу үчүн **Ultimate Multisite → Көп валюталуу** бөлүмүнө өтүңүз
4. Кабыл алгыңыз келген валюталарды кошуп, алмашуу курсу камсыздоочуңузду жөндөңүз

## Конфигурация {#configuration}

### Админ Dashboard {#admin-dashboard}

Көп валюталуу админ Dashboard (**Ultimate Multisite → Көп валюталуу**) төмөнкүлөр үчүн борбордук жер болуп саналат:

- Валюталарды кошуу, иштетүү же өчүрүү
- Алмашуу курсу камсыздоочусун тандоо жана курстарды талап боюнча жаңыртуу
- Бардык активдүү валюталар үчүн учурдагы курстарды карап чыгуу
- Автоматтык камсыздоочулар ылайыктуу болбогондо кол менен курстарды коюу

### Алмашуу курсу камсыздоочулары {#exchange-rate-providers}

| Камсыздоочу | Сүрөттөмө |
|---|---|
| **ECB** | European Central Bank — EUR негизиндеги тармактар үчүн акысыз күнүмдүк курстар |
| **Frankfurter** | Ачык булактуу ECB күзгү API — акысыз, API ачкычы талап кылынбайт |
| **Stripe** | Stripe'тын жандуу FX курстарын колдонот — активдүү Stripe интеграциясын талап кылат |
| **Кол менен** | Курстарды түз киргизесиз; тышкы API чакыруулары жок |

Камсыздоочуңузду **Ultimate Multisite → Көп валюталуу → Жөндөөлөр → Алмашуу курсу камсыздоочусу** бөлүмүнөн тандаңыз. Курстарды каалаган убакта кол менен жаңыртууга же автоматтык жаңыртууга пландаштырууга болот.

### Ар бир өнүм үчүн бааны өзүнчө белгилөө {#per-product-pricing-overrides}

Автоматтык которуу жетиштүү так болбогондо, каалаган өнүмдө белгилүү бир валюта үчүн так баа коё аласыз:

1. Өнүм редакторун ачыңыз (**Ultimate Multisite → Өнүмдөр → Түзөтүү**)
2. **Валюта баалары** бөлүмүнө чейин ылдый жылдырыңыз
3. Өзүнчө белгилегиңиз келген ар бир валюта үчүн так бааны киргизиңиз
4. Өнүмдү сактаңыз

Өзүнчө белгилөөсү жок өнүмдөр негизги валютаңыздан которулган бааны колдонот.

### Катталуудагы валюта тандоо талаасы {#currency-selector-signup-field}

Валюта тандоо талаасын катталуу/төлөө агымыңызга кошсо болот, ошондо кардарлар катталууда артык көргөн валютасын тандай алышат:

1. **Ultimate Multisite → Көп валюталуу → Жөндөөлөр** бөлүмүнө өтүңүз
2. **Катталууда валюта тандоочуну көрсөтүү** параметрин иштетиңиз
3. Кааласаңыз, демейки валютаны коюңуз же геолокация алдын ала тандалган маанини аныктасын

Тандалган валюта жазылуу менен сакталат жана кийинки бардык эсептешүүлөр үчүн колдонулат.

### Геолокацияга негизделген валюта аныктоо {#geolocation-based-currency-detection}

Геолокация аркылуу аныктоо иштетилгенде, кошумча келүүчүнүн өлкөсүн валютага байланыштырып, аны валюта тандоо талаасында алдын ала тандайт. Иштетүү үчүн:

1. **Ultimate Multisite → Көп валюталуу → Жөндөөлөр** бөлүмүнө өтүңүз
2. **Келүүчүнүн жайгашкан жеринен валютаны аныктоо** параметрин иштетиңиз
3. **Геолокация байланыштыруу** таблицасында өлкөлөрдү валюталарга байланыштырыңыз

Эгер келүүчүнүн өлкөсү үчүн байланыш жок болсо, тармактын негизги валютасы резервдик вариант катары колдонулат.

## Көп берилген суроолор {#frequently-asked-questions}

### Алмашуу курсу камсыздоочусу жеткиликсиз болсо эмне болот? {#what-happens-if-an-exchange-rate-provider-is-unavailable}

Кошумча акыркы жолу ийгиликтүү алынган курстарга кайтып келет. Эгер курстар эч качан алынбаган болсо, сиз конфигурациялаган кол менен коюлган курстарга кайтып келет. Коопсуздук чарасы катары кол менен коюлган курстарды конфигурациялоо сунушталат.

### Ар башка валюталар үчүн ар башка камсыздоочуларды колдоно аламбы? {#can-i-use-different-providers-for-different-currencies}

Жок — бардык валюталар үчүн бир убакта бир гана камсыздоочу активдүү болот. Жеке курстарды толук көзөмөлдөө керек болсо, **Кол менен** параметрин колдонуңуз.

### Геолокация өзүнчө плагинди талап кылабы? {#does-geolocation-require-a-separate-plugin}

Тышкы геолокация плагини талап кылынбайт. Кошумча WordPress'тин камтылган IP-ден өлкөнү аныктоо мүмкүнчүлүгүн колдонот. Жогорку тактык үчүн, атайын геолокация кызматын берилген чыпка хуктары аркылуу интеграциялоого болот.

### Ар бир өнүм үчүн өзүнчө белгилөөлөр салыкка чейинби же кийинби колдонулат? {#are-per-product-overrides-applied-before-or-after-tax}

Ар бир өнүм үчүн өзүнчө белгилөөлөр максаттуу валютадагы салыкка чейинки бааны коёт. Салык учурдагы салык конфигурацияңызды колдонуп, өзүнчө белгиленген баанын үстүнө эсептелет.
