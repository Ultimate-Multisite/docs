# የUltimate Multisite ሰነድ

ወደ Ultimate Multisite ሰነድ እንኳን በደህና መጡ። Ultimate Multisite የWordPress Multisiteን ወደ ሙሉ Website as a Service (WaaS) መድረክ ይለውጣል።

## የሰነድ ክፍሎች {#documentation-sections}

### [የተጠቃሚ መመሪያ](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

WaaS መድረክዎን ለማዘጋጀት፣ ለማዋቀር እና ለማስተዳደር የሚያስፈልግዎ ሁሉ። መጫን (installation)፣ የክፍያ መግቢያዎች (payment gateways)፣ የዶሜን ካርታ አወጣጥ (domain mapping)፣ የደንበኛ አስተዳደር (client management) እና ሌሎችንም ይሸፍናል።

### [የገንቢ መመሪያዎች](/developer/hooks) {#developer-documentation}

በUltimate Multisite ላይ የሚገነቡ ገንቢዎች የቴክኒክ ማጣቀሻ። የhooks ማጣቀሻ፣ የREST API ሰነድ እና የአድ-ኦን ልማት መመሪያዎችን ያካትታል።

### [ተጨማሪዎች](/addons) {#addons}

Ultimate Multisiteን የሚያስፋፉ ክፍያ የሚጠይቁ የadd-on plugins ሰነድ። እንደ WooCommerce integration፣ analytics፣ የዶሜን ሽያጭ እና ሌሎችንም ተጨማሪ ባህሪያት ይጨምራሉ።

## እርዳታን ማግኘት {#getting-help}

- [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) -- ጥያቄዎችን ይጠይቁ እና ሀሳቦችን ያካፍሉ።
- [GitHub Issues](https://github.com/Ultimate-Multisite/ultimate-multisite/issues) -- ስህተቶችን (bugs) ሪፖርት ያድርጉ።
- [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) -- ዋናውን plugin ያውርዱ።
