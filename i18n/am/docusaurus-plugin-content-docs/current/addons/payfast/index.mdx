# PayFast

Multisite Ultimate: Payfast Gateway

PayFastን - በደቡብ አፍሪካ ውስጥ ፈጣን እና ደህንነቱ የተጠበቀ የመስመር ላይ ክፍያ መግቢያ (gateway) - በMultisite Ultimate የክፍያ አማራጭ አድርጎ ይጨምራል።

## መጫን (Installation)

1. 'ultimate-multisite-payfast' የሚለውን ፋይል ወደ '/wp-content/plugins/' ዳይሬክቶሪ ይጫኑ።
2. በWordPress ውስጥ ባለው 'Plugins' ሜኑ አማካኝነት ፕልጊኑን ያንቁ (Activate)።
3. ፕልጊኑን ለማዘጋጀት ደረጃ በደረጃ የሚመራውን አስማተኛ (Wizard) ይከተሉ።

## የአጠቃቀም መመሪያ (Usage Guide)

PayFast በደቡብ አፍሪካ ውስጥ ለንግዶች መሪ የመስመር ላይ ክፍያ አቅራቢ ነው። የኦንላይን ነጋዴዎች ግብይቶችን እንከን የለሽ እና ደህንነቱ የተጠበቀ እንዲሆኑ በማድረግ ዕድገታቸውን እንዲያሳድጉ ትኩረት ሰጥቷል። እንደ Ultimate Multisite ተጠቃሚዎች ጥያቄ፣ ከPayFast ጋር መዋሃድ (integration) እንዲቻል የሚያስችል ተጨማሪ (addon) ለመፍጠር ወስነናል። ይህ ጽሑፍ በUltimate Multisite ኔትወርክዎ ውስጥ ክፍያ ለመቀበል PayFastን እንደ የክፍያ መግቢያ (payment gateway) እንዴት ማዘጋጀት እንደሚቻል ያሳያል።

የመጀመሪያው እርምጃ ተጨማሪውን (add-on) መጫን ነው።

# የPayfast ተጨማሪውን (add-on) እንዴት መጫን እንደሚቻል፡

ወደ ተጨማሪዎች (add-ons) ገጽ ይሂዱ እና Ultimate Multisite: Payfast Gateway ተጨማሪውን ይምረጡ።

![Add-ons page](/img/admin/addons-page.png)

የተጨማሪው መስኮት ይከፈታል። "Install Now" የሚለውን ይጫኑ።

![Add-ons page with install option](/img/admin/addons-page.png)

መጫኑ ከተጠናቀቀ በኋላ ወደ ፕልጊኖች ገጽ ይመራሉ። እዚህ ላይ፣ "Network Activate" የሚለውን በመጫን የ_Payfast_ ተጨማሪው በኔትወርክዎ ላይ ይንቀሳቀሳል።

# PayFastን ማዘጋጀት (Setting Up PayFast)

ተጨማሪውን ከጫኑ በኋላ፣ ወደ Ultimate Multisite ቅንብሮች (settings) በመሄድ PayFastን ማዋቀር (configure) መጀመር ይችላሉ።

በ**Ultimate Multisite settings** ገጽ ስር፣ "Payments" የሚል ታብ (tab) ላይ ያገኛሉ። በዚህ ታብ ስር፣ ይህን የክፍያ መግቢያ ለማንቃት ወይም ለማጥፋት የሚያስችል "Payfast" እና መቀያየር (toggle) ቁልፍ ይኖራል።

![Payment gateway settings](/img/config/settings-payment-gateways.png)እየተመራዎት ነው፣ ስለዚህ እሱን ለማንቃት (enable) መቀያየር (toggle) ቁልፉን ይጫኑ። ከዚያም **Merchant ID, Merchant Key, እና PassPhrase** ባሉባቸው የተለያዩ መስኮቶች እንዲሞሉ ሳጥኑን ያሰፋዋል። ይህን መረጃ ከ**Payfast** አካውንትዎ ማግኘት ይችላሉ።

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

ቀጣዩ እርምጃ የሚያስፈልጉዎትን መረጃ ለማግኘት ወደ PayFast አካውንትዎ መግባት (login) ነው።

ግባktan በኋላ፣ ከ**Payfast** አካውንትዎ ዳሽቦርድ (dashboard) የሚያስፈልጉዎትን መለያዎች (credentials) ወዲያውኑ ማየት ይችላሉ።

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->የመለያውን መረጃ በቀላሉ መቅዳት (copy) እና መለጠፍ (paste) እና በUltimate Multisite ቅንብሮችዎ ውስጥ ማስገባት ይችላሉ። ከዚያም ለውጥ ለማስቀመጥ **"Save settings"** የሚለውን ቁልፍ ይጫኑ።
