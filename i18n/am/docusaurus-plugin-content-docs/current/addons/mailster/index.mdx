# Mailster

በUltimate Multisite Mailster Integration addon አማካይነት የብዙ ድረ-ገጽ አውታረ መረብዎን ወደ ኃይለኛ የኢሜል ዝርዝር መገንባት ማሽን ይለውጡ። አዳዲስ ደንበኞች ሲመዘገቡ ወይም ሲገዙ በራስ-ሰር ወደ Mailster የኢሜል ዝርዝርዎ እንዲጨመሩ ያደርጋል፣ ይህም የየተቋቋሙ የትምህርት መድረክዎን እና የኢሜል ግብይት ስትራቴጂዎን በቅልጥፍና ያገናኛል።

## ቁልፍ ባህሪያት (Key Features) {#key-features}

**በራስ-ሰር ተመዝጋቢ መመሳሰል (Automatic Subscriber Sync)**
* ደንበኞች ሲመዘገቡ በራስ-ሰር ወደ Mailster ዝርዝሮች ይጨመራሉ።
* ወዲያውኑ ሲመዘገቡ እንዲመሳሰል ወይስ ክፍያ እስኪረጋገጥ እንዲጠብቅ መምረጥ ይችላሉ።
* በእጅ ጣልቃ ሳይገቡ የኢሜል ዝርዝርዎን አዲስ እና ወቅታዊ ያድርጉ።

**ተጣጣፊ የዝርዝር አስተዳደር (Flexible List Management)**
* ለሁሉም አዳዲስ ደንበኞች ዓለም አቀፋዊ መነሻ ዝርዝሮችን (global default lists) ያዘጋጁ።
* ለተወሰኑ ዘመቻዎች (targeted campaigns) መነሻዎችን በምርት ደረጃ መሻር ይችላሉ።
* የዝርዝር ምደባዎችን በMailster በሚታወቀው በይነገጽ በቀላሉ ያስተዳድሩ።

**በGDPR ተስማሚ የOpt-In አማራጮች (GDPR-Compliant Opt-In Options)**
* በራስ-ሰር ምዝገባ ወይም በቼክቦክስ ላይ የተመሰረተ ስምምነት (consent) መምረጥ ይችላሉ።
* የተረጋገጡ ምዝገባዎች እንዲሆኑ Double Opt-in ማብራት ይችላሉ።
* እንደ የጥበቃ መስፈርቶችዎ መሠረት የOpt-in ቼክቦክስን ማሳየት/ማስወገድ ይችላሉ።

**ብልህ የውሂብ መመደብ (Smart Data Mapping)**
* የደንበኞችን ስም እና የእውቂያ መረጃ በራስ-ሰር ያመሳስላሉ።
* የሂሳብ አድራሻዎችን ወደ ተመዝጋቢ መስኮች (subscriber fields) መመደብ ይችላሉ።
* ለተወሰኑ ዘመቻዎች የመመዝገቢያ ቀኖችን ይከታተሉ።

**በምርት ደረጃ ቁጥጥር (Product-Level Control)**
* ለተለያዩ ምርቶች የተለያዩ ዝርዝሮችን መመደብ ይችላሉ።
* ደንበኞች ምን እንደሚገዙ በመሠረት የተወሰኑ ዘመቻዎችን ይፍጠሩ።
* አስፈላጊ ሲሆን Mailsterን ለተወሰኑ ምርቶች ማጥፋት ይችላሉ።

**ሙያዊ ተሞክሮ (Professional Experience)**
* ለዝርዝር ምርጫ ተፈጥሯዊ Mailster UI ውህደት።
* ለችግር መፈተሽ እና ማረጋገጫ ዝርዝር መመዝገብ (Detailed logging)።
* ያሉ ተመዝጋቢዎችን ያለችግር ማዘመን።

## ለማን ተስማሚ ነው (Perfect For) {#perfect-for}

* የኢሜል ዝርዝሮችን እየገነቡ የትምህርት ድረ-ገጽ ባለቤቶች
* አዳዲስ ደንበኞችን ለማሳደግ የሚፈልጉ SaaS መድረኮች
* ተማሪዎችን በቅጃጀት (segmenting) የሚከፋፍሉ የኮርስ ፈጣሪዎች
* Ultimate Multisiteን ከMailster ጋር የሚያስኬዱ ሁሉ

## እንዴት ይሰራል (How It Works) {#how-it-works}

1. addonን በUltimate Multisite አውታረ መረብዎ ላይ ይጫኑ እና ያንቀሳቅሱ።
2. በአጠቃላይ የUltimate Multisite ቅንብሮች ውስጥ ዓለም አቀፋዊ መነሻ ዝርዝሮችን ያዘጋጁ።
3. በምርት-ተኮር ዝርዝሮች አማራጭ መሻር ይችላሉ።
4. የOpt-in ሁነታዎን እና የደንበኝነት ጊዜን ይምረጡ።
5. የMailster ዝርዝሮችዎ በራስ-ሰር እያደጉ መታየታቸውን ይመልከቱ!

## መጫን (Installation) {#installation}

1. የፕልጊን ፋይሎችን ወደ Ultimate Multisite addons ዳይሬክቶሪዎ ይስቀሉ።
2. Mailster በዋናው ድረ-ገጽዎ ላይ መጫን እና መሥራቱን ማረጋገጥ።
3. addonን በUltimate Multisite addons ገጽမှ አንቀሳቅሱ።
4. ለማዋቀር ወደ Settings > Mailster Integration ይሂዱ።
5. መነሻ ዝርዝሮችዎን ይምረጡ እና የOpt-in ምርጫዎችን ያዘጋጁ።

## በተደጋጋሚ የሚጠየቁ ጥያቄዎች (Frequently Asked Questions) {#frequently-asked-questions}

## ይህ addon Ultimate Multisite ባይኖርበት ይሰራል? {#does-this-addon-work-without-ultimate-multisite}

አይ፣ ይህ addon እንዲሰራ Ultimate Multisite ያስፈልገዋል። Mailsterን ከUltimate Multisite የደንበኛ እና የምርት ስርዓት ጋር ለማዋሃድ በተለይ የተነደፈ ነው።

## በሁሉም ድረ-ገጾች ላይ Mailster መጫን አለብኝ? {#do-i-need-mailster-installed-on-all-sites}

አይ፣ Mailster በዋናው የኔትወርክ ድረ-ገጽዎ ላይ መጫን እና መሥራቱን ብቻ ይፈልጋል። addonው ከሁሉም ንዑስ ድረ-ገጾች (subsites) በራስ-ሰር ይገናኛል።

## ለተለያዩ ምርቶች የተለያዩ ዝርዝሮችን መመደብ እችላለሁ? {#can-i-assign-different-lists-to-different-products}

አዎ! ዓለም አቀፋዊ መነሻ ዝርዝሮችን ማዘጋጀት ይችላሉ፣ ከዚያም በየምርቱ ደረጃ መሻር ይችላሉ። ይህ ደንበኞች ምን እንደሚገዙ በመሠረት እጅግ በጣም የተወሰኑ የኢሜል ዘመቻዎችን እንዲፈጥሩ ያስችልዎታል።

## ደንበኛው ቀድሞውንም በMailster ካለ ምን ይሆናል? {#what-happens-if-a-customer-already-exists-in-mailster}

በመነሻ ሁኔታ፣ addonው ያሉ ተመዝጋቢዎችን ያዘምናል እና ወደ አዳዲስ ዝርዝሮች ይጨምራቸዋል። ካሉ ተመዝጋቢዎች መዝለል እንደሚመርጡ ቅንብሮች ውስጥ ይህንን ማጥፋት ይችላሉ።

## ይህ GDPR ተስማሚ ነው? {#is-this-gdpr-compliant}

አዎ! addonው ደንበኞች ኢሜይሎችን ለመቀበል በግልጽ እንዲስማሙ የሚያስገድድ የቼክቦክስ-ተኮር Opt-in ሁነታን ያካትታል። ይህ በትክክለኛ የግል ጥበቃ ፖሊሲዎች ስብስብ ሲሆን የGDPR መስፈርቶችን ያሟላል።

## Double opt-in ልጠቀም እችላለሁ? {#can-i-use-double-opt-in}

በእርግጠኝነት! የdouble opt-in ቅንብሩን ያንቀሳቅሱ እና Mailster ምዝገባዎችን ከማንቃት በፊት የማረጋገጫ ኢሜይሎችን ይልካል።
