# ለተጣጣሙ ኤንድፖይንቶች (Endpoints) የሚያገለግል ምርጥ AI Connector የለውጥ መዝገብ {#ultimate-ai-connector-for-compatible-endpoints-changelog}

## ስሪት 2.0.0 — የተለቀቀበት ቀን 2026-04-24 {#version-200--released-on-2026-04-24}

* **አዲስ:** ብዙ አቅራቢዎችን መደገፍ — በርካታ AI endpointsዎችን ማዋቀር እና በሁሉም አቅራቢዎች መካከል በራስ-ሰር ወደ ሌላ አማራጭ እንዲሄድ በማድረግ ጥያቄዎችን መላክ።
* **ማስተካከያ:** ለተለያዩ አቅራቢዎች ትክክለኛ የቅራቢ መለያዎች (provider IDs)፣ የምዝገባ URLዎች (registration URLs) እና የ HTTP filter ስኮፒንግ (scoping) ያለው የብዙ አቅራቢ SDK ውህደት።
* **ማስተካከያ:** አዲሱ የቅራቢ ካርድ (provider card) ሲጨመር በራስ-ሰር ይከፈታል፤ በፕልጋይኑ ዝመና ጊዜ የስክሪፕት ካሽ ብስቲንግ (script cache busting) ተተክሏል።
* **ማስተካከያ:** ለeval() እና ለአዲስ የተጨመሩ አቅራቢዎች በራስ-ሰር እንዲሰፋ የሚያስችል ተለዋዋጭ የቅራቢ ክፍል ስም ቦታ (namespace)።
* **ማስተካከያ:** ለWordPress 6.9+ ተኳሃኝነት ሲባል ያልሆኑ (no longer experimental) የተረጋጋ የCard/CardBody/CardHeader/CardDivider ክፍሎችን ተጠቅሟል።
* **ማስተካከያ:** የቅራቢዎችን ለመጎተት እና እንደገና ለማስቀመጥ (drag-to-reorder) ጥቅም ላይ የነበረውን ያልተገለጸ የDragHandle በዩኒኮድ የያዘ መያዣ አዶ (unicode grip icon) ተተክሏል።
* **ማስተካከያ:** በየገጹ መጫን ጊዜ ይፈጠረ የነበረውን የሚያደናቅፍ የ HTTP ጥያቄ ማስወገድ።
* **የተሻሻለ:** የGitHub Actions የሥራ ፍሰቶች (workflows) ወደ Node.js 24 ተዘምዋል።
