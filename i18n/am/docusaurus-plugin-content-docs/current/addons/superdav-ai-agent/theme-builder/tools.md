# መሳሪያዎች እና ቅንብሮች

Theme Builder የWordPress ድረ-ገጽዎን ለመተንተን፣ ለመቅረጽ እና ለመገንባት የሚያስችል የመሳሪያ ስብስብ ይጠቀማል። በ1.16.0 ስሪት ላይ፣ **sd-ai-agent/site-scrape አሁን የደረጃ 1 መሳሪያ (Tier 1 tool) ሆኗል**፣ ይህም በTheme Builder ውስጥ በነባሪነት እንዲገኝ አድርጓል።

## የሚገኙ መሳሪያዎች

### የደረጃ 1 መሳሪያዎች (ሁልጊዜ የሚገኙ)

የደረጃ 1 መሳሪያዎች ምንም ተጨማሪ ቅንብር ሳያስፈልጋቸው በTheme Builder ውስጥ በነባሪነት ይገኛሉ።

#### sd-ai-agent/site-scrape

**ዓላማ:** የነባር ድረ-ገጾችን በመተንተን የንድፍ መነሳሳት፣ የይዘት አወቃቀር እና ተግባር (functionality) ለማውጣት።

**አቅሞች:**

- **የድረ-ገጽ ትንተና** — ተወዳዳሪ ወይም መነሳሳት የሆኑ ድረ-ገጾችን መቧጨር እና መተንተን
- **የንድፍ መረጃ አወጣጥ** — ቀለሞችን፣ ፊደላትን (fonts) እና የቅንብር ዘይቤዎችን መለየት
- **የይዘት አወቃቀር** — የገጽ አደረጃጀትን እና ተዋረደድን መረዳት
- **የባህሪ መለየት** — የplugins እና ተግባራትን መለየት
- **የአፈጻጸም ትንተና** — የገጽ ፍጥነት እና ማመቻቸት መፈተሽ
- **የSEO ትንተና** — ሜታ ታጎችን (meta tags) እና የተዋቀረ መረጃን (structured data) መገምገም

**አጠቃቀም:**

```
example.comን ንድፉን በመተንተን ለገጼ መነሳሳት አድርጉ።
```

**የሚያወጣው:**

- የቀለም ቤተ-ስዕል እና የቀለም አጠቃቀም
- የፊደል ስልት (fonts እና መጠኖች)
- የቅንብር አወቃቀር እና ክፍተት (spacing)
- የአሰሳ ዘይቤዎች (Navigation patterns)
- የይዘት አደረጃጀት
- ምስሎች እና ሚዲያ አጠቃቀም
- መስተጋብራዊ ክፍሎች (Interactive elements)
- ለሞባይል ተስማሚነት (Mobile responsiveness)

**ገደቦች:**

- በይለፍ ቃል የተጠበቁ ድረ-ገጾችን መቧጨር አይችልም
- `robots.txt` እና የጣቢያ ፖሊሲዎችን ያከብራል
- ተለዋዋጭ ይዘትን (dynamic content) ላይይ ይችላል
- በጃቫስክሪፕት (JavaScript) የበዛባቸው ድረ-ገጾች ውስን ትንተና ሊኖራቸው ይችላል
- ትልልቅ ድረ-ገጾች ለመተንተን ተጨማሪ ጊዜ ሊወስዱ ይችላሉ

### የደረጃ 2 መሳሪያዎች (በፈቃድ)

የደረጃ 2 መሳሪያዎች በTheme Builder ቅንብሮች ውስጥ በግልጽ ሲበራከው ብቻ ይገኛሉ።

#### Advanced Analytics

ዝርዝር የአፈጻጸም መለኪያዎች ያቀርባል:

- የገጽ መጫን ጊዜዎች
- Core Web Vitals
- የSEO ውጤት (score)
- የአካል ብቃት ደረጃ (Accessibility score)
- የላቀ አሠራር ውጤት (Best practices score)

#### Content Optimizer

የሚከተሉትን ለማተን እና መሻሻል የሚጠቁም:

- የአንባብ ቀላልነት (Readability)
- የSEO ማመቻቸት
- የቁልፍ ቃል አጠቃቀም (Keyword usage)
- የይዘት አወቃቀር
- የድርጊት ጥሪ (Call-to-action) አቀማመጥ

### የደረጃ 3 መሳሪያዎች (ፕሪሚየም)

የደረጃ 3 መሳሪያዎች ተጨማሪ ቅንብር ወይም ፕሪሚየም ባህሪያት ይጠይቃሉ።

#### AI Content Generator

ይዘትን ይፈጥራል:

- የምርት መግለጫዎች (Product descriptions)
- የአገልግሎት ገጾች
- የብሎግ ልጥፎች
- ሜታ መግለጫዎች (Meta descriptions)
- የድርጊት ጥሪ ጽሑፎች (Call-to-action copy)

#### Advanced Design Tools

የላቀ የንድፍ አቅሞችን ያቀርባል:

- የቅንብር CSS መፍጠር (Custom CSS generation)
- አኒሜሽን መፍጠር (Animation creation)
- መስተጋብራዊ ክፍሎች ንድፍ (Interactive element design)
- የላቀ የቀለም ንድፈ ሐሳብ (Advanced color theory)
- የፊደል ስልት ማመቻቸት (Typography optimization)

## የመሳሪያ ቅንብር

### መሳሪያዎችን ማብራት

በTheme Builder ውስጥ ተጨማሪ መሳሪያዎችን ለማብራት:

1. ወደ **Gratis AI Agent → Settings** ይሂዱ
2. ወደ **Theme Builder → Tools** ይሂዱ
3. መሳሪያዎችን እንደ አስፈላጊነቱ ያብሩ/ያጥፉ (Toggle)
4. ቅንብሮችን ያስቀምጡ (Save settings)

### የመሳሪያ ፈቃዶች (Tool Permissions)

አንዳንድ መሳሪያዎች ፈቃድ ይጠይቃሉ:

- **የጣቢያ መቧጨር (Site scraping)** — የኢንተርኔት መዳረሻ ይጠይቃል
- **ትንተና (Analytics)** — የGoogle Analytics ውህደት ይጠይቃል
- **የይዘት መፍጠር (Content generation)** — የAPI ቁልፎችን ይጠይቃል
- **የላቀ ባህሪያት (Advanced features)** — ፕሪሚየም የደንበኝነት ምዝገባ ሊጠይቅ ይችላል

### የAPI ቁልፎች እና መለያዎች (API Keys and Credentials)

የሚጠይቁባቸው መሳሪያዎች የAPI ቁልፎችን ለማስገባት ቅንብሮችን ያድርጉ:

1. ወደ **Gratis AI Agent → Settings → API Keys** ይሂዱ
2. ለእያንዳንዱ መሳሪያ መለያዎችን ያስገቡ
3. ግንኙነቱን ይፈትሹ (Test connection)
4. በአስተማማኝ ሁኔታ ያስቀምጡ (Save securely)

## sd-ai-agent/site-scrapeን መጠቀም

### መሰረታዊ አጠቃቀም

Theme Builder ድረ-ገጽ እንዲተነትን ይጠይቁ:

```
my competitor's site at competitor.com ንድፉን በመተንተን ለገጼ የንድፍ መሻሻል ሀሳብ ይስጡ።
```

### የተወሰነ ትንተና

የተወሰኑ የትንተና አይነቶችን ይጠይቁ:

```
example.comን የቀለም ቤተ-ስዕል አውጣጥ እና እንደ መነሳሳት ይጠቀሙ።
```

```
example.com የአሰሳ አወቃቀሩን በመተንተን ተመሳሳይ አደረጃጀትን በገጼ ላይ ይተግብሩ።
```

```
example.com ለሞባይል ተስማሚነት መፈተንን እና ገጼም በተመሳሳይ መልኩ ተስማሚ መሆኑን ያረጋግጡ።
```

### ንጽጽር ትንተና

በርካታ ድረ-ገጾችን ያነፃፅሩ:

```
site1.com እና site2.com ንድፎችን በማነፃፅር የሁለቱንም ምርጥ ክፍሎች የሚያጣምር ንድፍ ይፍጠሩ።
```

## የመሳሪያ ገደቦች እና ግምት ውስጥ ማስገባት ያለባቸው ነገሮች

### የፍጥነት ገደብ (Rate Limiting)

- የሰርቨርን ከመጫን ለመከላከል መቧጨር በየጊዜው ይገደባል።
- በተመሳሳይ ጣቢያ ላይ ብዙ ጥያቄዎችን መላክ ሊገደብ ይችላል።
- ትልልቅ ድረ-ገጾች ለመተንተን ተጨማሪ ጊዜ ሊወስዱ ይችላሉ።

### የይዘት ትክክለኛነት (Content Accuracy)

- ተለዋዋጭ ይዘት ሙሉ በሙሉ ላይይ ይችላል።
- በጃቫስክሪፕት (JavaScript) የተፈጠረ ይዘት不 ሙሉ ሊሆን ይችላል።
- የእውነተኛ ጊዜ መረጃ ላይን ይችላል።
- አንዳንድ ይዘቶች በክፍያ ግድግዳ (paywalls) ሊቀበሩ ይችላሉ።

### ህጋዊ እና ስነምግባር ግምት ውስጥ ማስገባት

- `robots.txt` እና የጣቢያ ፖሊሲዎችን ያክብሩ።
- የተበcopyright ይዘትን እንደገና ለመቅዳት አይቧጨሩ።
- ትንተናን ለመነሳሳት ይጠቀሙ እንጂ ለመቅዳት አይደለም።
- የወጣውን ይዘት ለመጠቀም መብት እንዳለዎት ያረጋግጡ።
- የተተነተኑ ድረ-ገጾችን የአገልግሎት ውል (terms of service) ይከተሉ።

### የአፈጻጸም ተጽዕኖ (Performance Impact)

- ትልልቅ ድረ-ገጾችን መቧጨር ጊዜ ሊወስድ ይችላል።
- ብዙ በተመሳሳይ ጊዜ የሚደረጉ መቧጨሮች ቀስ ብለው ሊሆኑ ይችላሉ።
- የኔትወርክ ግንኙነት ፍጥነትን ይነካል።
- ትልልቅ የውሂብ ስብስቦች ተጨማሪ ሂደት ሊጠይቁ ይችላሉ።

## ምርጥ አሠራሮች (Best Practices)

### የጣቢያ ትንተናን መጠቀም

1. **በርካታ ጣቢያዎችን ይተንትኑ** — ከበርካታ ምንጮች መነሳሳት ይስበሉ።
2. **በአወቃቀር ላይ ያተኩሩ** — የቅንብር እና የአደረጃጀት ዘይቤዎችን ይማሩ።
3. **ቀለሞችን ያውጡ** — የቀለም ቤተ-ስዕሎችን እንደ መነሻ ይጠቀሙ።
4. **ፊደል ስልትን ያጥኑ** — የሚወዱትን የፊደል ጥምረት ይለዩ።
5. **አሰሳን ይገምግሙ** — የመენიበሩ አደረጃጀት ምንነት ይረዱ።

### ስነምግባር ያለው መቧጨር (Ethical Scraping)

1. **`robots.txt` ያክብሩ** — የጣቢያ መመሪያዎችን ይከተሉ።
2. **ይዘትን አይቅዱ** — ትንተናን ለመነሳሳት ብቻ ይጠቀሙ።
3. **መብቶችን ያረጋግጡ** — የወጣውን ይዘት መጠቀም እንደሚችሉ ያረጋግጡ።
4. **ምንጮችን ይጥቀሱ** — የመነሳሳት ምንጮችን ክሬዲት ይስጡ።
5. **ቅንብሮችን ይከተሉ** — የጣቢያ የአገልግሎት ውልን ያክብሩ።

### የመሳሪያውን ውጤታማነት ከፍ ማድረግ

1. **በግልጽ ይሁኑ** — የተወሰኑ የትንተና አይነቶችን ይጠይቁ።
2. **ዐውደ-ጽሑፍ ይስጡ** — የገጽዎ ዓላማን ያስረዱ።
3. **ስሜት ይስጡ** — የንድፍ ግቦችን ይግለጹ።
4. **ይሻሻሉ** — በውጤቶች ላይ በመመርኮዝ ያጥብቁ።
5. **መሳሪያዎችን ያጣምሩ** — ሁሉን አቀፍ ትንተና ለማግኘት ብዙ መሳሪያዎችን ይጠቀሙ።

## ችግር መፍታት (Troubleshooting)

### ጣቢያው አይቧጨርም

- ጣቢያው በይፋ ተደራሽ መሆኑን ያረጋግጡ።
- `robots.txt` መቧጨርን እንደሚፈቅድ ያረጋግጡ።
- ሌላ ጣቢያ በመሞከር ይሞክሩ።
- የኢንተርኔት ግንኙነትን ያረጋግጡ።
- ችግሩ ከቀጠለ ድጋፍን ያነጋግሩ።

### ያልተሟላ ትንተና

- ጣቢያው ተለዋዋጭ ይዘት ሊኖረው ይችላል።
- ጃቫስክሪፕት ሙሉ በሙሉ ላይፈስ ይችላል።
- ትልልቅ ጣቢያዎች ጊዜ ሊወስዱ ይችላሉ።
- በተቃራኒው የተወሰኑ ገጾችን መተንተን ይሞክሩ።
- የተወሰኑ የትንተና አይነቶችን ይጠይቁ።

### ቀስተኛ አፈጻጸም

- ትልልቅ ጣቢያዎች ለመተንተን ተጨማሪ ጊዜ ይወስዳሉ።
- ብዙ በተመሳሳይ ጊዜ የሚደረጉ መቧጨሮች ቀስ ብለው ይሆናል።
- የኔትወርክ ግንኙነት ፍጥነትን ይነካል።
- በስራ ሰዓት ውጪ ለመሞከር ይሞክሩ።
- በመጀመሪያ ትናንሽ ክፍሎችን መተንተን ይጀምሩ።
