# የለውጦች መዝገብ

## 1.18.0 — በ2026-06-29 ተለቋል {#1180--released-on-2026-06-29}

### አዲስ {#new}

- **Google Calendar መሳሪያዎች** — ለመርሐግብር-አዋቂ አውቶሜሽኖች የተዋቀሩ ካሌንደሮችን እና ክስተቶችን ያንብቡ
- **የእውቂያ ካርታ ማድረግ እና የተሳታፊ ረዳቶች** — የካሌንደር ተሳታፊዎችን ከጣቢያ ተጠቃሚዎች እና እውቂያዎች ጋር ያዛምዱ
- **የሰው ማጽደቂያ በሮች እና የማስታወሻ መዝገቦች** — ለግምገማ አውቶሜሽኖችን ለጊዜው ያቁሙ እና የተደጋገሙ ማሳወቂያዎችን ያስወግዱ
- **TextBee SMS አቅራቢ** — የተዋቀሩ የጽሑፍ-መልዕክት ማሳወቂያዎችን በTextBee በኩል ይላኩ
- **የላቀ ተጓዳኝ ጥቅል** — ከWordPress.org ግንባታ በተናጠል የሚሰራጩ የታመነ-ገንቢ ፋይል ስርዓት፣ ዳታቤዝ፣ WP-CLI፣ REST አሰራጭ፣ plugin ገንቢ፣ git ቅጽበታዊ ቅጂ፣ የተጠቃሚ-አስተዳደር እና የመለኪያ መሳሪያዎችን ያክሉ

### ተሻሽሏል {#improved}

- **Managed Superdav አገልግሎት ማዋቀር** — ለሚደገፉ ጣቢያዎች የተስተናገዱ የአገልግሎት endpoints እና ራስ-ሰር የግንኙነት ዝግጅት ያክሉ
- **የልቀት ጥቅል ማዘጋጀት** — የተለዩ core እና Advanced ZIPs ይገንቡ፣ ሁለቱንም በGitHub ላይ ያትሙ፣ እና ወደ WordPress.org የcore ጥቅሉን ብቻ ይላኩ

### ተስተካክሏል {#fixed}

- **የAI ጥያቄ አስተማማኝነት** — የmodel ምርጫ፣ የጥያቄ ጊዜ ገደቦች፣ ነባሪ የማዋቀሪያ model፣ የምክንያታዊ-ጽሑፍ አያያዝ፣ እና ለትክክል ያልሆኑ የመሳሪያ ጥሪዎች የድጋሚ ሙከራ መመሪያን ያሻሽሉ
- **የካሌንደር እና ማስታወሻ ማጠናከሪያ** — Google Calendar tokens እና የማስታወሻ ድግግሞሽ ማስወገድን ያጠናክሩ
- **መጀመሪያ ማስጀመር እና የማጽደቂያ ቀጥሎ መቀጠል** — የfrontend መጀመሪያ ማስጀመሪያዎችን እና የተረጋገጡ ችሎታዎች ቀጥሎ መቀጠልን ያስተካክሉ
- **የWordPress.org ጥቅል ማዘጋጀት ግምገማ ጉዳዮች** — ለcore ልቀት የጥቅል ማዘጋጀት ግምገማ ግብረመልስን ይፍቱ
