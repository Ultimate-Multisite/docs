# Superdav AI Agent

Superdav AI Agent ገጽታን የሚያውቁ ክህሎቶች፣ የተሻሻሉ የኢንተርኔት ፍለጋ ችሎታዎች፣ የድረ ገጽ ገንቢ ባህሪያት፣ እና አማራጭ የራስ-ሰር ማዋሃዶችን በመጨመር Gratis AI Agentን የሚያስፋ የላቀ WordPress AI ረዳት ነው። ስሪት 1.18.0 የGoogle Calendar ንባብ መሣሪያዎችን፣ የእውቂያ እና የተሳታፊ ካርታ ማድረግን፣ የሰው ማጽደቂያ በሮችን፣ የማስታወሻ ድግግሞሽ ማስወገድን፣ TextBee SMS ማሳወቂያዎችን፣ የሚተዳደር Superdav አገልግሎት ማቅረብን፣ እና ለታመኑ የገንቢ የሥራ ፍሰቶች በተናጠል የሚሰራጭ Advanced ተጓዳኝ ጥቅልን ይጨምራል።

## ቁልፍ ባህሪያት {#key-features}

- **ገጽታን የሚያውቁ ክህሎቶች** — በንቁ WordPress ገጽታዎ እና በተጫኑ ተሰኪዎች መሠረት መመሪያን እና ችሎታዎችን በራስ-ሰር ያስተካክላል
- **በርካታ የኢንተርኔት ፍለጋ አቅራቢዎች** — የበለጸጉ የፍለጋ ውጤቶችን እና የላቁ የምርምር ችሎታዎችን ለማግኘት በBrave Search እና Tavily መካከል ይምረጡ
- **የድረ ገጽ ገንቢ የእውቂያ ቅጾች** — የተጫነውን የቅጽ ተሰኪዎን በመጠቀም ሙሉ ተግባራዊ የእውቂያ ቅጾችን በቀጥታ ከውይይት በይነገጽ ይፍጠሩ
- **የተሻሻለ የድረ ገጽ ገንቢ** — በተሻለ የተሰኪ ግኝት እና የስህተት መልሶ ማግኛ የተሻሻሉ ባለብዙ-ደረጃ የድረ ገጽ ግንባታ ዕቅዶች
- **አብሮገነብ የገጽታ ክህሎቶች** — ለBlock Themes፣ Classic Themes፣ Kadence Blocks፣ እና Kadence Theme ልዩ መመሪያ
- **የተሻሻለ WooCommerce ማዋሃድ** — ለተሻለ አስተማማኝነት እና ተኳኋኝነት ተወላጅ WooCommerce APIዎችን እንዲጠቀም እንደገና ተዋቅሯል
- **በራስ-ሰር የአቅራቢ መለየት** — ተሰኪዎች እና ገጽታዎች ሲነቁ ወይም ሲሰናከሉ ያሉትን ተሰኪዎች እና ገጽታዎች በራስ-ሰር ይለያል እና ያድሳል
- **ብጁ የአርማ ማመንጨት** — በራስ-ሰር namespace-ደህንነቱ የተጠበቀ ማጽዳት ጋር ብጁ SVG አርማዎችን ያመንጩ እና ያካትቱ
- **የቀለም ስብስብ ኮንትራስት ማረጋገጫ** — ወደ ገጽታዎ ከመተግበርዎ በፊት የቀለም ጥንዶችን ለWCAG ተደራሽነት ተገዢነት ያረጋግጡ
- **በፎቶ የተሻሻለ ግኝት** — የንድፍ አቅጣጫን ለመቅረጽ በTheme Builder የግኝት ቃለመጠይቅ ወቅት ፎቶዎችን ይስቀሉ
- **የእንግዳ መስተንግዶ ምናሌዎች** — ለሬስቶራንቶች፣ ለካፌዎች፣ እና ለእንግዳ መስተንግዶ ንግዶች የተዋቀሩ የምግብ እና የመጠጥ ምናሌ ገጾችን ያመንጩ
- **የዴስክቶፕ እና የሞባይል ቅድመ-እይታዎች** — በንድፍ-አቅጣጫ ምርጫ ወቅት ንድፍዎን በዴስክቶፕ እና በሞባይል መሣሪያዎች ላይ ቅድመ-ይመልከቱ
- **Tier 1 የድረ ገጽ መቧጠጥ** — sd-ai-agent/site-scrape አሁን የተፎካካሪ እና የአነሳሽነት ድረ ገጾችን ለመተንተን በነባሪ ይገኛል
- **የGoogle Calendar ንባብ መሣሪያዎች** — ራስ-ሰር ሂደቶች ስለ መጪ የመርሃግብር አውድ ማመዛዘን እንዲችሉ የተዋቀሩ ቀን መቁጠሪያዎችን እና ክስተቶችን ያንብቡ
- **የእውቂያ እና የተሳታፊ ካርታ ማድረግ** — የክትትል ሥራ ከመፍጠር በፊት የቀን መቁጠሪያ ተሳታፊዎችን ከሚታወቁ የድረ ገጽ ተጠቃሚዎች እና እውቂያዎች ጋር ያዛምዱ
- **የሰው ማጽደቂያ በሮች** — መልዕክቶችን ከመላክ፣ ሥራን ከመቀጠል፣ ወይም ለውጦችን ከማድረግ በፊት ስሱ ራስ-ሰር ሂደቶችን ለግምገማ ያቁሙ
- **የማስታወሻ መዝገቦች እና ድግግሞሽ ማስወገድ** — ተደጋጋሚ የራስ-ሰር ሂደት ሩጫዎች አንድን ተቀባይ ሁለት ጊዜ እንዳያሳውቁ አስቀድመው የተላኩ ማስታወሻዎችን ይከታተሉ
- **TextBee SMS ማሳወቂያዎች** — የSMS አቅራቢ ሲነቃ በTextBee በኩል የተዋቀሩ የጽሑፍ-መልዕክት ማሳወቂያዎችን ይላኩ
- **የሚተዳደር Superdav አገልግሎት ማቅረብ** — ከእጅ የendpoint ውቅር ይልቅ በራስ-ሰር ማዋቀር የሚደገፉ ድረ ገጾችን ከተስተናገዱ Superdav endpointዎች ጋር ያገናኙ
- **Advanced ተጓዳኝ ጥቅል** — ለፋይል ስርዓት፣ ዳታቤዝ፣ WP-CLI፣ REST dispatcher፣ ተሰኪ ገንቢ፣ git snapshot፣ የተጠቃሚ-አስተዳደር፣ እና benchmark የሥራ ፍሰቶች በተናጠል የሚሰራጩ የታመነ-ገንቢ መሣሪያዎች

## መስፈርቶች {#requirements}

- WordPress 6.0 ወይም ከዚያ በላይ
- PHP 8.1 ወይም ከዚያ በላይ
- WordPress Multisite (በአውታረ መረብ የነቃ) ወይም ነጠላ-ድረ ገጽ መጫኛ
- ንቁ የAI አቅራቢ አገናኝ ተሰኪ (Anthropic፣ OpenAI፣ ወይም ተኳኋኝ)
- Gratis AI Agent v1.9.0 ወይም ከዚያ በላይ (Superdav AI Agent Gratis AI Agentን ያስፋል)

## መጫን {#installation}

1. የተሰኪውን ፋይሎች ወደ `/wp-content/plugins/superdav-ai-agent` ይስቀሉ
2. ተሰኪውን በWordPress ውስጥ ባለው **ተሰኪዎች** ማያ ገጽ በኩል ያንቁ
3. Gratis AI Agent እንዲሁም መንቃቱን ያረጋግጡ
4. ወደ **Gratis AI Agent → Settings** ይሂዱ እና እንደ አስፈላጊነቱ የኢንተርኔት ፍለጋ አቅራቢዎን፣ የሚተዳደር Superdav አገልግሎት ግንኙነትን፣ Google Calendarን፣ ወይም የTextBee ምስክርነቶችን ያዋቅሩ

የWordPress.org ጥቅል ዋናዎቹን የSuperdav AI Agent ባህሪያት ይዟል። Advanced ተጓዳኝ ጥቅል የታመኑ የገንቢ መሣሪያዎች በግልጽ በሚጫኑባቸው እና በሚፈቀዱባቸው አካባቢዎች ለመጠቀም በተናጠል ይገነባል እና ይሰራጫል።

## ፈጣን መጀመሪያ {#quick-start}

ከነቃ በኋላ፣ Superdav AI Agent የGratis AI Agent መጫኛዎን በገጽታን የሚያውቁ ክህሎቶች እና ተጨማሪ ችሎታዎች በራስ-ሰር ያሻሽላል። በWordPress አስተዳዳሪ የጎን አሞሌ ውስጥ ያለውን **Gratis AI Agent** ፓነል ይክፈቱ እና አዲሶቹን ባህሪያት መጠቀም ይጀምሩ።

**የምሳሌ መመሪያዎች፦**

```
ወደ እውቂያ ገጹ የእውቂያ ቅጽ ያክሉ።
```

```
ለቅርብ ጊዜዎቹ የWordPress ደህንነት ምርጥ ልምዶች ድሩን ይፈልጉ።
```

```
የንቁ ገጽታውን የብሎክ ንድፎች በመጠቀም የhero ክፍል ይፍጠሩ።
```

```
የነገን የተዋቀሩ Google Calendar ክስተቶች ይገምግሙ እና ለተሳታፊዎች የማስታወሻ ረቂቆችን ያዘጋጁ።
```

```
መልዕክቱን ካጸደቅኩ በኋላ የSMS ማስታወሻ ይላኩ።
```

## ገጽታን የሚያውቁ ክህሎቶች {#theme-aware-skills}

| ክህሎት | የሚተገበርበት |

| **Classic Themes** | PHP ቴምፕሌቶችን እና ክላሲክ አርታዒን የሚጠቀሙ ባህላዊ WordPress ገጽታዎች |
| **Kadence Blocks** | Kadence Blocks ተሰኪ ንቁ የሆነባቸው ድረ ገጾች |
| **Kadence Theme** | Kadence ገጽታ እንደ ንቁ ገጽታ ያላቸው ድረ ገጾች |

## የቀን መቁጠሪያ፣ ማጽደቅ እና ማሳወቂያ የሥራ ፍሰቶች {#calendar-approval-and-notification-workflows}

- **የGoogle Calendar መሣሪያዎች** የቀን መቁጠሪያ ምስክርነቶች ሲገኙ ወኪሉ የተዋቀሩ ቀን መቁጠሪያዎችን እና የክስተት ዝርዝሮችን እንዲያነብ ያስችላሉ።
- **የእውቂያ እና የተሳታፊ ረዳቶች** ማስታወሻዎችን ወይም የክትትል ተግባራትን ከማዘጋጀት በፊት የክስተት ተሳታፊዎችን ከWordPress ተጠቃሚዎች ወይም ከሚታወቁ እውቂያዎች ጋር ያዛምዳሉ።
- **የሰው ማጽደቅ መግቢያዎች** አውቶሜሽን የታቀደውን እርምጃ ፈቃድ ያለው ተጠቃሚ እስኪገመግም ድረስ እንዲቆም ያስችላሉ።
- **የማስታወሻ መዝገቦች** አውቶሜሽን እንደገና ሲሞከር ወይም በተደጋጋሚ ሲታቀድ የተባዙ የቀን መቁጠሪያ ወይም የማሳወቂያ ማስታወሻዎችን ይከላከላሉ።
- **የTextBee SMS ድጋፍ** የTextBee አቅራቢ ሲዋቀር የጽሑፍ-መልእክት ማሳወቂያዎችን ያስችላል።

እነዚህን መሣሪያዎች ለመርሐግብር-አዋቂ እገዛ፣ ለተሳታፊ ክትትል፣ እና ለማስታወሻ የሥራ ፍሰቶች ይጠቀሙ፤ እነዚህ ግን ሚስጥራዊ መልእክቶች ከመላካቸው በፊት አሁንም የሰው ግምገማ ይፈልጋሉ።
