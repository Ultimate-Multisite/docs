# የይዘት መመሳሰል (Content Sync)

Multisite Ultimate: የይዘት መመሳሰል (Content Sync)

ይህ ለMultisite Ultimate የተዘጋጀው ተጨማሪ (addon) የኔትወርክ አስተዳዳሪዎች በኔትወርኩ ውስጥ ካሉ በማንኛውም ድረ-ገጽ ይዘትን ወደ ሌሎች ድረ-ገጾች ለማመሳሰል ያስችላል። ይህ ተጨማሪ የፖስቶች (posts)፣ የገጾች (pages)፣ የተበጁ የፖስት አይነቶች (custom post types) እና Elementor templatesን ጨምሮ ሁሉንም የፖስት አይነቶች ይደግፋል። ይዘትን ለመምረጥ እና መድረሻ ድረ-ገጾችን ለመምረጥ ብዙ ደረጃ ያለው መመሪያ (multi-step wizard) በይነገጽን፣ በቡድን መሥራት (batch processing) እና በቅጽበታዊ የሂደት ግብረመልስ (real-time progress feedback) አቅም አለው።

## ባህሪያት (Features)

* **የይዘት ምርጫ (Content Selection):** በትንንሽ ምስሎች (thumbnails) እና በፖስት አይነቶች መጣራት (post type filtering) አማካኝነት የሚታይ ካርድ መረብ።
* **ሁሉም የፖስት አይነቶች (All Post Types):** ፖስቶችን፣ ገጾችን፣ ምርቶችን፣ Elementor templatesን እና ማንኛውንም የተበጀ የፖስት አይነቶችን መመሳሰል ይችላሉ።
* **የድረ-ገጽ ምርጫ (Site Selection):** ይዘትን ወደየትኞቹ ድረ-ገጾች ማመሳሰል እንዳለብዎ ይምረጡ። የምንጩ ድረ-ገጽን ማስወገድ እና የልጅ ድረ-ገጽን አስቀድሞ መምረጥ ይቻላል።
* **በቡድን መሥራት (Batch Processing):** በአንድ ጊዜ እስከ 10 ድረ-ገጾች ድረስ በመመሳሰል የሂደት ዝመናዎችን ያሳያል።
* **የሚዲያ መመሳሰል (Media Sync):** ከፖስት ይዘት ወደ መድረሻ ድረ-ገጾች የሚላኩ የሚዲያ ፋይሎችን (ምስሎች፣ ቪዲዮዎች) በራስ-ሰር ይገለብጣል እና ቦታውን ያድሳል።
* **የታክሶኖሚ መመሳሰል (Taxonomy Sync):** ምድቦችን (categories)፣ መለያዎችን (tags) እና የተበጁ ታክሶኖሚዎችን ይጠብቃል።
* **Elementor ድጋፍ (Elementor Support):** CSSን እንደገና በማመንጨት እና የtemplate IDን ቦታ በመቀየር Elementor templatesን ሙሉ በሙሉ ይደግፋል።
* **የግጭት መፍታት (Conflict Resolution):** ያለውን ይዘት ለመተካት ወይም ለማለፍ (skip) አማራጭ ይሰጣል።
* **የካሽ አያያዝ (Cache Management):** ከተመሳሰለ በኋላ በመድረሻ ድረ-ገጾች ላይ ያለውን Elementor cache በራስ-ሰር ያጽዳል።

## መስፈርቶች (Requirements)

* WordPress 5.3 ወይም ከዚያ በላይ
* PHP 7.4 ወይም ከዚያ በላይ
* Multisite Ultimate plugin (በሥራ ላይ መሆን አለበት)
* Elementor plugin (አማራጭ ነው፣ ነገር ግን Elementor template sync ሲፈልጉ ያስፈልጋል)

## መጫን (Installation)

1. 'ultimate-multisite-content-sync' የሚለውን ፋይል ወደ '/wp-content/plugins/' ዳይሬክቶሪ ይጫኑ።
2. በWordPress Network Admin ውስጥ ባለው 'Plugins' ሜኑ አማካኝነት 플uginটিকে ያንቀሳቅሱ (Activate)።
3. ቅንብሮችን በUltimate Multisite → Settings → Content Sync ውስጥ ያዋቅሩ።
4. መመሳሰልን (syncing) ለመጀመር ወደ Ultimate Multisite → Content Sync ይሂዱ።
