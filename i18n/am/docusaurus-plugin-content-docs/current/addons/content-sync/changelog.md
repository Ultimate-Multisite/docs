# የይዘት መመሳሰል የለውጥ መዝገብ {#content-sync-changelog}

### Version 1.1.0 - የተለቀቀበት ቀን 2025-12-31 {#version-110---released-on-2025-12-31}
* አዲስ፡ ሁሉንም የፖስት አይነቶች (posts, pages, custom post types) የመመሳሰል ድጋፍ።
* አዲስ፡ በይዘት የመምረጥ ደረጃ የፖስት አይነቶችን መጣራት (filtering)።
* አዲስ፡ የምንጭ ድረ-ገጽ የመምረጥ ደረጃ - ዋናውን ድረ-ገጽ ብቻ ሳይሆን ከማንኛውም ድረ-ገጽ መመሳሰል።
* አዲስ፡ ለቴምፕሌት ድረ-ገጾች የልጅ ድረ-ገጽ ቅድመ-ምርጫ።
* አዲስ፡ ሁሉንም የፖስት አይነቶች ከpost_content ሚዲያ መውጣት (Media extraction)።
* አዲስ፡ ታክሶኖሚ መመሳሰል (categories, tags, custom taxonomies)።
* ተቀየረ፡ ከ "Elementor Template Sync" ወደ "Content Sync" ተለውጧል።
* ተቀየረ፡ "Select Templates" ደረጃ ወደ "Select Data" ተለውጧል።
* የተሻሻለ፡ በ Elementor ላይ የተመሰረተ ኮድ አሁን በ elementor_library የፖስት አይነቶች ላይ ብቻ ይሰራል።

### Version 1.0.0 - የተለቀቀበት ቀን 2025-12-19 {#version-100---released-on-2025-12-19}
* የመጀመሪያ ስሪት።
* ለቴምፕሌት እና ለሳይት ምርጫ ባለ ብዙ ደረጃ አስማሚ (Multi-step wizard)።
* በቅጽበት የሂደት ግብረመልስ (real-time progress feedback) ያለው በቡድን ሂደት (Batch processing)።
* በራስ-ሰር የ ID መልሶ ማቀየሪያ (automatic ID remapping) ያለው ሙሉ ሚዲያ መመሳሰል።
* ከ Ultimate Multisite ጋር የቅንብሮች ውህደት።
* በኔትወርክ ቅንብሮች አማካኝነት የሳይት መገለል ድጋፍ።
* ከመሳሰሉ በኋላ የ Elementor cache ማጽዳት።
* የቴምፕሌት ታክሶኖሚ ጥበቃ።
