# Material WP

የ WordPress የአስተዳደር ገጽ (admin interface) ለረጅም ጊዜ አንድ አይነት ነው። ግን የ Material Design ስሪት የአስተዳደር ገጽ ምን እንደሚመስል አስበህ ታውቃለህ?

Material WP እርስዎ ያንን መልክና ስሜት እንዲያገኙ የሚያስችል የተሰራ የአስተዳደር ቴም (admin theme) ነው። ደንበኞችን ይበልጥ የተበጀ Dashboard እንዲሰጡት ይረዳዎታል።

## ባህሪያት {#features}

- ለCustom CSS ድጋፍ
- ቀላል የቅይጥ (styling) ቅንብሮች
- ሜኑን እንደገና ማስቀመጥ (Menu reordering)
- ቀላል ማስመጥ/ማስላክ (import/export)

## ሰነድ አቀራረብ (Documentation) {#documentation}

**Material WP በመጠቀም የአስተዳደር ፓነልዎን ማበጀት**

## መጫን (Installation) {#installation}

1. 'material-wp' ወደ '/wp-content/plugins/' ዳይሬክቶሪ ይስቀሉ።
2. በ WordPress ውስጥ ባለው 'Plugins' ሜኑ አማካኝነት ፕልጊኑን ያንቁ።

## የአጠቃቀም መመሪያ (Usage Guide) {#usage-guide}

የ WordPress የአስተዳደር ገጽ (admin interface) ለረጅም ጊዜ አንድ አይነት ነው። ግን የ Material Design ስሪት የአስተዳደር ገጽ ምን እንደሚመስል አስበህ ታውቃለህ?

Material WP እርስዎ ያንን መልክና ስሜት እንዲያገኙ የሚያስችል የተሰራ የአስተዳደር ቴም (admin theme) ነው። ደንበኞችን ይበልጥ የተበጀ Dashboard እንዲሰጡት ይረዳዎታል። Material WP በተጨማሪም የ Ultimate Multisite የፕሪሚየም ፕልጊን ኮር አካል ነው።

**ከመጫኑ በኋላ**፣ ወደ _Plugins_ ገጽዎ ይሂዱ እና በኔትዎርክዎ ላይ ለማንቃት ይጫኑ።
