# Plugin & Theme Manager

Multisite Ultimate: Plugin and Theme Manager

የተጫኑ የplugins እና የthemes ሜታ መረጃዎችን (ርዕስ፣ ደራሲ፣ ምስል እና መግለጫ) ማርትዕ ወይም መደበቅ። ተጠቃሚዎችዎ በቀላሉ እንዲfiltter የሚያስችሉ የplugins እና themes ምድቦችን ይፍጠሩ እና ለተጠቃሚዎችዎ ፓነል የሚያምር የPlugins ገጽ ይጨምሩ።

ፕልጊኑን ለመጠቀም፣ በኔትወርክ አድሚንዎ ላይ ወደ Plugins ወይም Themes ገጾች በመሄድ "Edit Plugin/Theme info" የሚለውን በመጫን መጀመር ይችላሉ።

## Installation {#installation}

1. 'ultimate-multisite-plugin-and-theme-manager' የሚለውን ወደ '/wp-content/plugins/' ዳይሬክቶሪ ይጫኑ።
2. በWordPress Network Adminዎ ውስጥ ባለው 'Plugins' ሜኑ አማካይነት ፕልጊኑን ያግኙ እና ያክምሩት (Activate)።

## Usage Guide {#usage-guide}

Ultimate Multisite: Plugin and Theme Manager እርስዎ በኔትወርክዎ ላይ የplugins እና የthemes አማራጮችን የሚያዩበት እና የሚገናኙበትን መንገድ ሙሉ ቁጥጥር የሚሰጥ **ፕሪሚየም** Ultimate Multisite add-on ነው።

ይህ add-on ሶስት ዓላማዎችን ያገለግላል፦

  1. የplugins እና የthemes ሜታ መረጃዎችን ሙሉ በሙሉ እንዲቀይሩ ያስችልዎታል። ይህ ስም፣ መግለጫ፣ የደራሲ ስም እና አልፎ ተርፎም ምስል (thumbnails)ን ያካትታል።

  2. ተጠቃሚዎችዎ በቀላሉ እንዲከፋፍሉ plugins እና themes እንዲመድቡ ያስችልዎታል።

  3. የድሮውን፣ አሰልቺ የplugins ገጽ በአማካይነት በምድቦች የተደራጀ አዲስ themes-እንደ ገጽ እንዲተካ ያስችልዎታል።

ከማውረድ እና ከማስቀመጥ በኋላ፣ አዲስ ታብ - _Add-on Settings_ - በUltimate Multisite Settings ገጽ ላይ ይታያል።
