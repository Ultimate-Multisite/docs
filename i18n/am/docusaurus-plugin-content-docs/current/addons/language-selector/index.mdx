# Language Selector

Multisite Ultimate: Sign-up ላይ የቋንቋ መምረጥ (Language Selector)

በመመዝገቢያ ሂደት (sign-up flow) ውስጥ የቋንቋ መምረጥ መስክ ይጨምሩ።

## Installation

1. 'ultimate-multisite-language-selector' የሚለውን ወደ '/wp-content/plugins/' ዳይሬክቶሪ ይጫኑ።
2. በWordPress Network Admin ውስጥ ባለው 'Plugins' ሜኑ አማካይነት ፕልጊኑን ያንቁ (Activate)።

## Usage Guide

በዚህ ነጻ add-on አማካኝነት ተጠቃሚዎችዎ በመመዝገቢያ ሂደት እና በlogin ገጽ ላይ የትኛውን ቋንቋ ማየት እንደሚፈልጉ እንዲመርጡ መፍቀድ ይችላሉ።

[የライሰንስዎን አጠቃቀም ካነሱት](../../user-guide/getting-started/installing-ultimate-multisite.md)፣ በライሰንስዎ የሚሰጡዎትን Ultimate Multisite add-ons መጫን ይችላሉ።

ወደ Ultimate Multisite settings page ይሂዱና ከዚያም በግራ በኩል ባለው ሜኑ ላይ _Add-ons_ የሚለውን ምልክት ይጫኑ።

![Add-ons page](/img/admin/addons-page.png)

_Language Selector ለመጫን ወደ Add-ons page ይሂዱ_

እዚያም የምናቀርባቸውን ሁሉም add-ons ያገኛሉ።

ነጻ add-ons በደንበኞቻችን ሁሉ፣ የሰበክራፕሽን ዕቅድ ምንም ይሁን ምን፣ ይገኛሉ።

ከadd-on ምስል በታች ባለው _Install it Now!_ ቁልፍ ላይ ይጫኑ። ከዚያም በPlugins page ላይ ያንቁት።

![Add-ons page with install button](/img/admin/addons-page.png)

_add-on ለመጫን ቁልፉን ይጫኑ_

## Setting the Language via the URL

የገጹን ቋንቋ ወደ URL የlocale parameter ሳይተላለፉ መቀየር ይችላሉ። ለምሳሌ፣ ወደ ጣሊያንኛ የመመዝገቢያ ገጽ መላክ ከፈለጉ፣ ?locale=it_IT ወደ URL ላይ ይጨምሩ።

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

ይህ እንዲሰራ ቋንቋው በWordPress networkዎ ላይ መጫን አለበት።
