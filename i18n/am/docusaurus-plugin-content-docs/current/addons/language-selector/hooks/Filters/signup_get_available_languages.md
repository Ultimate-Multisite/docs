# ማጣሪያ፡ signup_get_available_languages {#filter-signupgetavailablelanguages}

የፊት ለፊት (front-end) የድር ጣቢያዎች መመዝገቢያ የሚደረጉበትን የሚገኙ ቋንቋዎች ዝርዝር ማጣራት።

ለዚህ hook ባዶ አረፋ (empty array) መስጠት፣ በመመዝገቢያ ቅጽ ላይ ያንን ቅንብር ውጤት እንዳይሰጥ ያደርጋል፤ ጣቢያ ሲፈጠር መደበኛው ቋንቋ ጥቅም ላይ ይውላል። አስቀድሞ ያልተጫኑ ቋንቋዎች ይነሳሉ።

## መለኪያዎች (Parameters) {#parameters}

| ስም | አይነት | መግለጫ |
|------|------|-------------|
| `$available_languages` | `array` | የሚገኙት ቋንቋዎች። |
