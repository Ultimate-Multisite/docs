# የቶከን ክፍያ ዕቅዶች የለውጥ መዝገብ

Version 1.1.0 - የተለቀቀበት ቀን 2026-05-05
- አዲስ፡ ለብዙ ድረ-ገጽ ተابعة ድረ-ገጾች (multisite subsites) የAI ቶከን ክፍያ — በደንበኛ ድረ-ገጾች ላይ የAI ቶከን አጠቃቀምን መከታተል እና በየቶከኑ ሊቀመጥ በሚችል መጠን ክፍያ መፈጸም።
- አዲስ፡ የኮኔክተር (Connector) አተገባበር በዳይናሚክ ገደብ ፍለጋ (dynamic limit discovery) እና በጽሑፍ ማለፍ (write-through) ተስተካክሏል፣ ይህም በሁሉም ኮኔክተሮች ላይ ትክክለኛነትን በቅጽበት ያረጋግጣል።
- ማስተካከያ፡ የዳታቤዝ ሰንጠረዥ ማሻሻያዎች አሁን ከBerlinDB የስхема ትርጓሜዎች ጋር በትክክል ተስተካክለዋል፣ ይህም በአዲስ መጫን ጊዜ የሚከሰቱ የማሻሻያ ውድቀቶችን ይከላከላል።
- ማስተካከያ፡ የዳታቤዝ ማሻሻያ የጥሪ መልስ (callbacks) ወደ ትክክለኛው ቅርጸት ተቀይረዋል፣ ይህም በዝምተኛ የማሻሻያ ውድቀቶች ችግርን ፈቷል።
- ማስተካከያ፡ በAI Usage Overage Markup የግብዓት መስክ ውስጥ ክፍልፋይ እሴቶች (Fractional values) ተቀባይነት አግኝተዋል።
- ማስተካከያ፡ በplugin ሲጫን የሚከሰቱ የሞት ስህተቶች (fatal errors) እና የሁለት ጊዜ መጀመር (double-initialization) ችግሮች ተፈቷል።
- የተሻሻለ፡ የdeveloper environments ላይ የመጀመሪያ ጊዜ ሲሰራ ራሱን እንዲያዘጋጅ የሚያስችል `check-env` npm script ተጨምሯል።
