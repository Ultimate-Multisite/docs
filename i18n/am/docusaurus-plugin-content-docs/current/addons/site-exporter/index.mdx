# የድረ-ገጽ አቅራቢ (Site Exporter)

## ይዘት (Content) {#content}

የደንበኛዎ ድረ-ገጽ ለመቀየር (migrate) ሲወስን፣ በኔትወርክዎ ውስጥ ያሉ የሰባ-ሳይት (subsites) ወደ አንድ የሳይት ተከላ (single site installation) መዛወር እና በተቃራኒው መሆኑን እናውቃለን። የSite Exporter እና Importer add-on ይህንን ስራ ቀላል ያደርገዋል። ይህ መሳሪያ የሳይቶችን በzip ፋይሎች እንዲያወጽም (export) እና እንዲያስገባ (import) የሚያስችል ሲሆን፣ እንደ ምትኬ (backup)ም ሊያገለግል ይችላል።

## ባህሪያት (Features) {#features}

* የሳይቶችን ማስወ출 (Export) እና ማስገባት (Import)
* ለመጠቀም ቀላል የሆነ የይዘት ገጽ (interface)
* የሳይት ምትኬ መፍጠር (Create site backup)

## ሰነድ (Documentation) {#documentation}
**የሳይት ማስወ출 እና ማስገባት ዘዴ**

## መጫን (Installation) {#installation}

1. **ultimate-multisite-site-exporter** ን ወደ **/wp-content/plugins/** ዳይሬክቶሪ ይጫኑ።
2. በWordPress ውስጥ ባለው **Plugins** ሜኑ አማካይነት ፕልጊኑን ያንቁ (Activate)።
3. ፕልጊኑን ለማዘጋጀት ደረጃ በደረጃ የሚመራውን አስማተኛ (Wizard) ይከተሉ።

## የአጠቃቀም መመሪያ (Usage Guide) {#usage-guide}

በUltimate Multisite አማካኝነት የሳይትዎን ኔትወርክ ላይ ያለ ሳይት ማስወ출 (export) ወይም ወደ multi-site ወይም single-site ተከላ ማስገባት (import) ይችላሉ።

ሂደቱን ለመጀመር፣ ወደ add-ons ገጻችን በመሄድ Ultimate Multisite: Site Exporter & Importer add-on ን ይጫኑ።![Add-ons page](/img/admin/addons-page.png)

![Add-ons page with install option](/img/admin/addons-page.png)

ወደ Plugins ገጽ በመሄድ፣ የጫኑትን add-on **Network Activate** ማድረግ ላይ ጠቅ ያድርጉ።<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Site Exporter addon -->

የ**Export & Import** አማራጭ በUltimate Multisite ሜኑ፣ በግራ በኩል ይገኛል።

<!-- Screenshot unavailable: Ultimate Multisite menu showing Export and Import option -->

## የሳይት ማስወ출 ዘዴ (How to export a site) {#how-to-export-a-site}

ከኔትወርክዎ የሳይት ለማስወ출፣ በUltimate Multisite ሜኑ ላይ ወደ **Export & Import** ይሂዱ እና በ**Export Sites** ታብ ላይ **Generate new Site Export** ላይ ጠቅ ያድርጉ።

<!-- Screenshot unavailable: Export Sites tab with Generate new Site Export button -->

ማስወ출 (export) ሊያደርጉት የሚፈልጉትን የሳይት ስም ይፈልጉ እና በፋይሉ ውስጥ ለማካተት የሚፈልጓቸውን ክፍሎች ይምረጡ። ቴሞች (themes)፣ ፕልጊኖች (plugins) እና የፋይሎች ዝውውር (uploads) ማካተት ይችላሉ። የሳይትዎ መጠን በጣም ትልቅ ከሆነ፣ ማስወ출 ሂደቱን በበስተጀርባ (background) እንዲሰራ መምረጥ ይችላሉ።<!-- Screenshot unavailable: Site export options with site search, include themes/plugins/uploads checkboxes -->

በ**Export Site** ቁልፍ ላይ ይጫኑ። የወጣው የሳይትዎ ስብስብ (exported site) በ**Existing Exports** ዝርዝር ውስጥ ይገኛል። ፋይሉን ለማውረድ በ**Download ZIP** ቁልፍ ላይ ጠቅ ያድርጉ። የExport ZIP ፋይሎች ራሳቸውን የሚያነሳሳ `index.php` እና `readme.txt` ስለሚይዙ፣ ያለ መለየተ importer plugin መጫን ሳይኖር በአዲስ ሆስት ላይ ሊጫኑ ይችላሉ።<!-- Screenshot unavailable: Existing Exports list with Download ZIP button -->

## መላውን ኔትወርክ ማስወ출 ዘዴ (How to export the whole network) {#how-to-export-the-whole-network}

ሁሉንም የሰባ-ሳይት (subsites) በአንድ ላይ ለማስወ출፣ ወደ Ultimate Multisite ሜኑ ላይ **Export & Import** ይሂዱ እና በexport screen ላይ ባለው **Network Export** አማራጭ ይጠቀሙ። የተፈጠረው አርካይቭ (archive) ለሆስት መዛወር (host migrations)፣ ለstaging rebuilds፣ ወይም ለኔትወርክ ደረጃ ምትኬዎች ሁሉንም የሰባ-ሳይት በአንድ ZIP ፋይል ውስጥ ይይዛል።

የኔትወርክ ማስወ출 (Network exports) ትልቅ ሊሆን ይችላል። ስለዚህ ትራፊክ ዝቅተኛ በሚሆንበት ጊዜ ያካሂዱ እና አርካይቭውን እና የተካተቱ የፋይሎች ዝውውር (uploads)፣ ፕልጊኖች (plugins) እና ቴሞች (themes) ለማስቀመጥ ሰርቨሩ በቂ ማከማቻ እንዳለው ያረጋግጡ።

## የሳይት ማስገባት ዘዴ (How to import a site) {#how-to-import-a-site}

የሳይት ለማስገባት (import)፣ ወደ **Import a Site** ታብ ይሂዱ እና ማስገባት የሚፈልጉትን የሳይት .zip ፋይል ይስቀሉ። የዩአርኤል (URL) ወደ አዲስ ሊቀይሩት ይችላሉ ወይም ቦታውን ባዶ በመተው የድሮውን ዩአርኤል ሊጠብቁ ይችላሉ። በተጨማሪም፣ ከማስገባቱ በኋላ የ.zip ፋይሉን ከትዕዛዝዎ ለማጥፋት ሳጥኑን ምልክት ማድረግ ይችላሉ። ከዚያም በ**Begin Import** ላይ ጠቅ ያድርጉ።

<!-- Screenshot unavailable: Import a Site tab with zip upload field and Begin Import button -->

ሂደቱ ትንሽ ጊዜ ሊወስድ ይችላል። በimporter ስር ባለው ዝርዝር ውስጥ የሚታየው በመጠባበቅ ላይ ያለ ማስገባት (pending import) ይሆናል።

<!-- Screenshot unavailable: Pending import in the list below the importer -->

ሂደቱን በ**Cancel Import** ላይ ጠቅ በማድረግ ማቆም ይችላሉ።

<!-- Screenshot unavailable: Cancel Import button to stop the import process -->

ሂደቱ ከተጠናቀቀ በኋላ፣ የሳይትዎ ስብስብ (site) በኔትወርክዎ ላይ ዝግጁ ይሆናል።

## በአንድ ተከላ ላይ የሳይት ማስገባት ዘዴ (How to import a site on a single install) {#how-to-import-a-site-on-a-single-install}

ወደ single installዎ የሳይት ለማስገባት፣ በመጀመሪያ በሳይትዎ ላይ importer ን ይጫኑ እና ያንቁ። ለማውረድ የሚረዳውን ሊንክ በexporter ታብ ላይ ማግኘት ይችላሉ።

<!-- Screenshot unavailable: Exporter tab with link to download single-site importer -->

importer ን በ**Tools** ሜኑ፣ በግራ በኩል፣ በ**Export & Import** ንዑስ ክፍል ውስጥ ማግኘት ይችላሉ።

<!-- Screenshot unavailable: Single-site Tools menu showing Export and Import submenu -->

ማስገባት የሚፈልጉትን የሳይት .zip ፋይል ይስቀሉ። የዩአርኤል (URL) ወደ አዲስ ሊቀይሩት ይችላሉ ወይም ቦታውን ባዶ በመተው የድሮውን ዩአርኤል ሊጠብቁ ይችላሉ። በተጨማሪም፣ ከማስገባቱ በኋላ የ.zip ፋይሉን ከትዕዛዝዎ ለማጥፋት ሳጥኑን ምልክት ማድረግ ይችላሉ። ከዚያም በ**Begin Import** ላይ ጠቅ ያድርጉ።
