# VAT {#vat}

Multisite Ultimate: የአውሮፓ ቫት (European VAT)

ይህ ተጨማሪ ባህሪያት እና መሳሪያዎችን በመጨመር፣ Multisite Ultimate መደበኛ የግብር አያያዝን (tax handling) ለማሻሻል እና የአውሮፓን ቫት (VAT) በተሻለ ሁኔታ ለመደገፍ ይረዳል።
የEU የቫት መጠኖችን በራስ-ሰር ከ

## መጫን (Installation) {#installation}

1. 'ultimate-multisite-vat'ን ወደ '/wp-content/plugins/' ዳይሬክቶሪ ይጫኑ።
2. በWordPress ውስጥ ባለው 'Plugins' ሜኑ አማካይነት 플uginን ያክምቱ።
3. ወደ Ultimate Multisite > Settings > Taxes ይሂዱ እና EU Vatን ያንቁ።
4. በTax Rates ላይ ይጫኑ፣ ከታች ባለው የEU የቫት መጠኖች ላይ ዝመና ያድርጉ እና Tax Ratesን ያስቀምጡ።
5. ተከናውኗ!
