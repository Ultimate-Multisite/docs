# Loco Translate

የUltimate Multisite: Loco Translate Integration addon ወደ WordPress multisite networkዎ የጣቢያ-ልዩ (site-specific) የትርጉም አቅም ያመጣል። በኔትዎ ውስጥ ያሉ የተለያዩ ጣቢያዎች አንድ አይነት plugin ወይም theme ቢኖራቸውም የተለያዩ ትርጉሞች እንዲኖራቸው ከፈለጉ፣ ይህ addon ፍጹም መፍትሄ ነው።

**ይህ Addon ምን ያደርጋል**

ይህ addon Ultimate Multisiteን ከታዋቂው Loco Translate plugin ጋር ያለችግር ያዋህዳል፣ ይህም በኔትዎ ውስጥ እያንዳንዱ ንዑስ ጣቢያ (subsite) የራሱን የቅንብረት ትርጉም (custom translations) እንዲይዝ ያስችላል። ብዙ ክልል ያለው መድረክ እየሰሩ ቢሆንም፣ የተለያዩ የቋንቋ ማህበረሰቦችን እየሰሩ ቢሆንም፣ ወይም በኔትዎ ውስጥ የትርጉም ተጣጣፊነት ብቻ የሚያስፈልግዎ ከሆነ፣ ይህ addon በጣም ቀላል ያደርገዋል።

**ለማን ተስማሚ ነው:**

*   የተለያዩ የቦታ አቀማመጥ ያላቸው (multi-regional) እና የተለያዩ የቅንብረት ፍላጎት ያላቸው ንግዶች
*   የተለያዩ ማህበረሰቦችን የሚያገለግሉ የትምህርት ተቋማት
*   ብዙ ቋንቋ ድጋፍ የሚሰጡ SaaS መድረኮች
*   በርካታ የደንበኛ ጣቢያዎችን የሚያስተዳድሩ የዌብ ኤጀንሲዎች
*   የትርጉም ተጣጣፊነት የሚያስፈልገው ማንኛውም multisite network

## መጫን (Installation)

1.  Ultimate Multisite ተጭኖ እና network ተንቀሳቅሶ መጀመር እንዳለበት ያረጋግጡ።
2.  Loco Translate pluginን ይጫኑ እና ያንቀሳቅሱ።
3.  ይህን addon ወደ plugins ዳይሬክቶሪዎ ይስቀሉ።
4.  Loco Translate Integration addonን በኔትዎ ደረጃ ያንቀሳቅሱ።
5.  በማንኛውም ንዑስ ጣቢያ (subsite) በLoco Translate አማካኝነት የጣቢያ-ልዩ ትርጉም መፍጠር ይጀምሩ።

ያ ብቻ ነው! ትስስሩ ምንም ተጨማሪ ቅንብር ሳያስፈልገው በራስ-ሰር ይሰራል።

## በተደጋጋሚ የሚጠየቁ ጥያቄዎች

## ከተጫኑ በኋላ ሌላ ነገር ማቅረብ አለብኝ?

አይ! ትስስሩ በራስ-ሰር ነው። addon እና Loco Translate ሁለቱም ሲንቀሳቀሱ፣ ንዑስ ጣቢያዎች ወዲያውኑ የራሳቸውን የቅንብረት ትርጉም መፍጠር ይችላሉ።

## አንድ ንዑስ ጣቢያ የቅንብረት ትርጉም ካላመጣ ምን ይሆናል?

ጣቢያው እንደ አማራጭ (fallback) የኔት-አቀፍ ትርጉሞችን በራስ-ሰር ይጠቀማል። ምንም ነገር አይበላሽም፣ ሁሉም ነገር ያለችግር ይሰራል።

## ንዑስ ጣቢያዎች የኔት-አቀፍ ትርጉሞችን መድረስ ይችላሉ?

መደበኛ የጣቢያ አስተዳዳሪዎች (site admins) የራሳቸውን የጣቢያ ትርጉሞች ብቻ ያያሉ እና ያርትዑታል። ሱፐር አስተዳዳሪዎች (Super admins) የትኛውን ጣቢያ እያዩ እንደሆነ በመወሰን የኔት-አቀፍ እና የጣቢያ-ልዩ ትርጉሞችን መድረስ ይችላሉ።

## ከplugins እና ከthemes ሁለቱም ጋር ይሰራል?

አዎ! የጣቢያ-ልዩ ትርጉሞች ለWordPress plugins እና themes ሁለቱም ይሰራሉ።

## የmultisite networkዬን ያቀዘቅዘዋል?

አይ። הaddon የሚያስፈልገው ጊዜ ብቻ የጣቢያ-ልዩ ትርጉሞችን የሚፈትሽ ቀልጣፋ የፋይል መጫኛን ይጠቀማል። የሚታወቅ የአፈጻጸም መዳከም የለም።

## የኔት ትርጉሞችን እንደ መነሻ ወደ አንድ የተወሰነ ጣቢያ መቅዳት እችላለሁ?

አዎ! በLoco Translate ውስጥ ትርጉሞችን በሚያርትዑበት ጊዜ፣ በቀላሉ በኔት ትርጉሞች መጀመር እና ለእያንዳንዱ ጣቢያ ማበጀት ይችላሉ።

## ከWooCommerce እና ከሌሎች plugins ጋር ይሰራል?

በእርግጠኝነት! መደበኛ የWordPress የትርጉም ተግባራትን የሚጠቀሙ ሁሉም plugins ወይም themes ከጣቢያ-ልዩ ትርጉሞች ጋር ይሰራሉ።
