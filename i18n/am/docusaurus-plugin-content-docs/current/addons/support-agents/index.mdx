# የድጋፍ ወኪሎች (Support Agents)

ሙሉ ኔትወርክን ራሳችን ብቻ ማስተዳደር ቀላል ስራ አይደለም። ኔትወርክዎ ሲያድግ፣ የድጋፍ ፍላጎት እየጨመረ መምጣቱን ያያሉ። የሥራ ጫናዎን ለማቃለል የሚረዱ ሰዎችን ማምጣት ጥሩ ሀሳብ ነው። ሆኖም፣ ሙሉ መዳረሻ/ፈቃድ መስጠት አደገኛ ነው እና ኔትወርክዎን ሊጎዳ ይችላል። እዚህ ላይ ነው ይህ Support Agent add-on የሚመጣው።

ይህ add-on Support Agent ተብሎ የሚጠራ አዲስ የWordPress user አይነት እንዲፈጥሩ ያስችልዎታል፣ እናም ያንን user ምን አይነት ፈቃዶች እንዳሉት ሙሉ ቁጥጥር ይሰጥዎታል። ፈቃድ መስጠት እንደ toggle button ማብራት ቀላል ነው፣ እና ማስወገድ ከወሰኑት ደግሞ እንደገና ማጥፋት ብቻ ነው።

## ባህሪያት (Features)

*   የWordPress እና Ultimate Multisite ፈቃዶች ዝርዝር
*   መዳረሻን በቀላሉ ማብራት እና ማጥፋት
*   ያልተገደበ የAgent አካውንቶችን መፍጠር

## መጫን (Installation)

1.  'ultimate-multisite-support-agents' ን ወደ '/wp-content/plugins/' ዳይሬክቶሪ ይጫኑ።
2.  በWordPress ውስጥ ባለው 'Plugins' ሜኑ አማካይነት plugin ን ያክቲቬት করুন።
3.  plugin ን ለማስገቢያ ደረጃ በደረጃ የሚመራውን Wizard ይከተሉ።
