# የደንበኛዎ ድረ-ገጾች (v2) {#your-clients-websites-v2}

በኔትዎ ላይ ከተመዘገሩ በኋላ፣ ደንበኞችዎ ወደ ዳሽቦርድ (dashboard) ሊገቡ ይችላሉ።

በክፍሉ ላይ አስፈላጊ መረጃዎችን በተመለከተ፣ እንደ ክፍያዎቻቸው፣ የአባልነት መብትዎቻቸው፣ የዶሜንዎቻቸው፣ የፕላን ገደቦች እና ሌሎችንም ማየት የሚችሉበት **Account** ገጽ ቢኖርም፣ ሁሉንም ድረ-ገጾቻቸውን ማየት እና ማስተዳደር የሚችሉበት **My Sites** ገጽም አላቸው። የሚያደርጉት ነገር ቢኖር በዳሽቦርድ ላይ ባለው My Sites ላይ መጫን ብቻ ነው።

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

ሁሉንም ድረ-ገጾቻቸውን ያያሉ፣ ከእያንዳንዳቸው ስርም **Manage** የሚል አዝራር አለ። እሱን ከጫኑ፣ ወደ **የዚህ የተወሰነ ድረ-ገጽ ዳሽቦርድ** ይመራቸዋል።

በተጨማሪም፣ ብዙ ድረ-ገጾችን እንዲይዙ የሚያስችላቸው ፕላን ካላቸው የትኛው ድረ-ገጽ **ዋናው (primary)** እንደሆነ ማየት ይችላሉ።

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

የsite edit ሙሉ በሙሉ መስተጋብር (interface) እዚህ አለ፦

![Site edit full interface](/img/admin/site-edit-full.png)

ዋና ድረ-ገጽ ሊቀይሩት ከፈለጉ፣ በ**Account** ሜኑ ላይ በመጫን ወደ Account ገጹ መሄድ አለባቸው።

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

የደንበኛ ዝርዝሮችን መመልከት እና ማርትዕም ይችላሉ፦

![Customer edit interface](/img/admin/customer-edit.png)

ከዚያም፣ **Change Default Site** የሚለውን መጫን አለባቸው።

![Change Default Site button on account page](/img/account-page/change-default-site.png)

ፖፕ-አፕ (pop-up) መስኮት ይከፈታል። በድብደባ ዝርዝር (drop-down list) ላይ በመጫን የትኛውን ድረ-ገጽ ዋና ድረ-ገጽ ማድረግ እንደሚፈልጉ ይምረጡ።

![Change Default Site dropdown selection](/img/account-page/change-default-site-dropdown.png)

ከዚያም፣ **Change Default Site** ብለው በመጫን ተከናውኗል!

![Confirming the default site change](/img/account-page/change-default-site-confirm.png)
