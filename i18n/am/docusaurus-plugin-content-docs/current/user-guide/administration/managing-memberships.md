# የአባላትነት (Memberships) አስተዳደር {#managing-memberships}

በUltimate Multisite ውስጥ የአባላትነት ሁኔታዎችን (memberships) መቆጣጠር እና በኔትዎርክዎ ላይ ያሉትን ሁሉንም የደንበኝነት ምዝገባዎች (subscriptions) መዳረሻ ሊሰጥ ይችላሉ። የደንበኝነት ምዝገባ ዝርዝሮችን፣ እንደ ዕቅዶች (plans) እና ምርቶች (products)፣ የየድረ-ገጽ ብዛት፣ የክፍያ መጠን እና ድግግሞሽ፣ የክፍያ ታሪክ እና የጊዜ ማህተሞችን (timestamps) መገምገል እና ማርትዕ ይችላሉ።

ይህንን ገጽ ለመድረስ፣ በUltimate Multisite ሜኑ ውስጥ ወደ **Memberships** ትብሌ (tab) ይሂዱ።

![Memberships list page](/img/admin/memberships-list.png)

በMemberships ገጽ ላይ የደንበኞችን ዝርዝር እና ከखाታዎቻቸው ጋር የተያያዙ ምርቶችን ማየት ይችላሉ። የአባላትነት ሁኔታን መሰረት በማድረግ ማጣራት (filter) ይችላሉ፡ ንቁ (active)፣ በመጠባበቅ ላይ ያለ (pending)፣ ተቆልቋሪ (on hold)፣ ጊዜ ያለፈበት (expired)፣ ወይም የተሰረዘ (cancelled)።

ነጻ የአባላትነት ሁኔታዎች (Free memberships) እንደ **ዕድሜ ልክ** (lifetime) አባላትነት ይቆጠራሉ። የጊዜ መቋረጥ ቀን አይኖራቸውም እና በራስ-ሰር አይሰረዙም። አንድ ነጻ ዕቅድ ከሙከራ ጊዜ (trial) ወይም ከተወሰነ ጊዜ በኋላ ካለቀቀ፣ በሙከራ ጊዜ (trial) ባለው ክፍያ ምርት ይጠቀሙ፣ የአባላትነት ሁኔታውን በእጅ ይሰርዙ፣ ወይም የደንበኛው መዳረሻ መለወጥ ሲገባው ወደ ሌላ ዕቅድ ያዛውሩ።

የአባላትነት ዝርዝሮችን — የክፍያ መረጃ፣ የድረ-ገጽ ወይም የጎብኝ ገደቦች፣ እና ሌሎችንም — ማርትዕ የሚችሉበት ገጽ ለመድረስ በአንድ የአባላትነት ሁኔታ ላይ ይጫኑ።

![Membership edit interface](/img/admin/membership-edit.png)

የባለሙያ የአባላትነት ማርትዕ ገጽ ሙሉ እይታ እነሆ፦

![Membership edit full interface](/img/admin/membership-edit-full.png)

በተጨማሪም፣ ቀደም ሲል በኔትዎርክዎ ላይ የተመዘገቡ ተጠቃሚዎችን የሚያገናኙ አዳዲስ የአባላትነት ሁኔታዎችን መጨመር ይችላሉ። **Add Membership** ላይ ይጫኑ እና ዝርዝሮችን ያዘጋጁ።

![Add Membership modal](/img/admin/membership-add-modal.png)

ከዚያም **Create Membership** ላይ ይጫኑ። አዲሱ የአባላትነት ሁኔታ በmemberships ዝርዝር ውስጥ ይታያል።
