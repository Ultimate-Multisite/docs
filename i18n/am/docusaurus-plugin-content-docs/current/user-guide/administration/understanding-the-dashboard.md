# የUltimate Multisite Dashboardን መረዳት

በከፍተኛ የቅንብር አማራጮች እና የUltimate Multisite የአሠራር ገጽታዎች ውስጥ ከመግባታችን በፊት፣ በመጀመሪያ ወደ Dashboard እንየው።

እንደ ብዙ የWordPress plugins ሁሉ፣ Ultimate Multisite በ**Ultimate Multisite** በሚል ርዕስ የኔትወርክ ደረጃ ሜኑ አማራጭ ይፈጥራል፣ እና ከዚህ ስርም በርካታ ንዑስ-ሜኑ አማራጮችን ይይዛል።

## የNetwork Dashboard Widgets

Ultimate Multisite በኔትወርክ ደረጃ Dashboard ውስጥ ጥቂት ጠቃሚ widgets ይጨምራል። ይህ Dashboard የሚገኘው ወደ **My Sites → Network Admin → Dashboard** በመሄድ ነው። እነዚህ widgets ለኔትወርክ አስተዳዳሪው መረጃን እና የተለመዱ ሥራዎችን በቀላሉ እንዲያገኙ ተደርገው ተዘጋጅተዋል።

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

### የመጀመሪያ እርምጃዎች (First Steps)

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

ይህ widget በUltimate Multisite አዲስ ሲጫን ይታያል፣ እና የኔትወርክ አስተዳዳሪው የክፍያ ዘዴን ማዋቀር እና የሙከራ አካውንት መፍጠር የመሳሰሉ ተጨማሪ ስራዎችን እንዲጨርስ ያስታውሰዋል።

### ማጠቃለያ (Summary)

![Summary widget](/img/admin/network-dashboard-summary.png)

የSummary widget የየቀኑ የመመዝገቢያ ብዛት እና ገቢውን (revenue) ሪፖርት ያደርጋል። MRR (Monthly Recurring Revenue) ደግሞ ገቢ የሚያመጣ የሆኑ አባላትን በመጠቀም የሚጠበቀውን አጠቃላይ ገቢ ያሳያል።

### የሥራ እንቅስቃሴ ፍሰት (Activity Stream)

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

ይህ widget በUltimate Multisite ውስጥ የተፈጠሩ ክስተቶችን (events) ሪፖርት ያደርጋል። ክስተቶች መመዝገብ (signups)፣ መሰረዝ (cancellations)፣ የsite መፍጠር፣ ማስወገድ እና ሌሎች እንቅስቃሴዎችን ያካትታሉ።

በበለጠ ዝርዝር መግለጫ ለማግኘት እባክዎ በdocumentation ውስጥ ባለው የEvents ክፍል ይመልከቱ።

### አሁን (Right Now)

![Right Now widget](/img/admin/network-dashboard-right-now.png)

ይህ የጥቅም widget በኔትወርኩ ውስጥ ያሉ ተጠቃሚዎችን እና sites አጭር ማጠቃለያ ያሳያል። በላይ ባሉት ቦታዎች ላይ ያሉ ምቹ አቋራጮች (shortcuts) አዳዲስ sites ወይም ተጠቃሚዎችን በአንድ ጠቅታ እንዲፈጥሩ ያስችላሉ።

### ዜና እና ውይይቶች (News and Discussions)

![News and Discussions widget](/img/admin/network-dashboard-news.png)

ይህ widget የቅርብ ጊዜውን የUltimate Multisite ዜና ይስባል እና ያሳያል። ስለ ዝመናዎች፣ ወሳኝ ስህተቶች (bugs) እና የደህንነት ማስተካከያዎች መረጃ ለማግኘት በዚህ widget ላይ ትኩረት ያድርጉ።

## Ultimate Multisite Dashboard

የኔትወርክ ደረጃ Dashboard አጠቃላይ መረጃን ስለኔትወርኩ የሚያሳይ ከሆነ፣ የUltimate Multisite dashboard (በUltimate Multisite ሜኑ ውስጥ በከፍተኛ ደረጃ የሚገኘው) ስለ አገልግሎቱ መረጃን ያቀርባል።

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

### የወር ተደጋጋሚ ገቢ ዕድገት (Monthly Recurring Revenue Growth)

የMRR (Monthly Recurring Revenue) ገበቻ በ12 ወር የካላንደር ጊዜ ውስጥ ያለውን የገቢ ወርኃዊ ክፍፍል ያሳያል።

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

አዲስ ገቢ በሰማያዊ ቀለም ተከታትሏል፣ መሰረዝ ደግሞ በቀይ ቀለም ተንጸባርቋል።

### የቀን ክልል ድጋፍ (Date-range Support)

በደንበኞች ብዛት እና እንቅስቃሴ ላይ በመመስረት፣ ስታትስቲክስ ከመጠን በላይ ሊሆኑ ይችላሉ። በDashboard ውስጥ የተገነባው የቀን ክልል ድጋፍ የኔትወርክ አስተዳዳሪው የሚያስፈልገውን መረጃ እና የጊዜ ክልል ላይ ብቻ እንዲያተኩር ያስችለዋል።

![Date range selector and stats](/img/admin/um-dashboard-date-range.png)

የቀን ክልሉ ከታች ባሉት widgets በሚታየው መረጃ ጊዜን ይቆጣጠራል፤ ይህም ገቢን፣ አዲስ አባላትን፣ በሀገር መመዝገብ፣ በform መመዝገብ እና በብዛት የተጎበኙ sitesን ያካትታል።

![Dashboard statistics widgets](/img/admin/um-dashboard-stats.png)

:::tip
**Most Visited Sites** widget እንዲሰራ፣ እባክዎ በ**Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** ላይ የጎብኝ ብዛት መቁጠርን (visit counting) ያንቁ።
:::
