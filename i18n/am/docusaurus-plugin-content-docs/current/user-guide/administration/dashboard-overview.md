# የዳሽቦርድ አጠቃላይ እይታ

Ultimate Multisite በUltimate Multisite ሜኑ ስር የተለየ **Dashboard** አለው፣ እንዲሁም በNetwork Dashboard ላይ widgets ይጨምራል። ይህ ጽሑፍ በሁለቱም ቦታዎች ምን እንደሚመለከቱ ያብራራል።

## የኔትወርክ ዳሽቦርድ {#the-network-dashboard}

Ultimate Multisite ጠቃሚ widgets በ**My Sites → Network Admin → Dashboard** በሚገኘው የኔትወርክ ደረጃ ዳሽቦርድ ውስጥ ያስገባል።

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

እያንዳንዱ widget ምን እንደሚሰጥ እነሆ፡

**የመጀመሪያ እርምጃዎች (First Steps):** አዲስ ሲጫን ይታያል እና የክፍያ ዘዴን ማዋቀር (configuring a payment method) እና የሙከራ አካውንት መፍጠር (creating a test account) የመሳሰሉ የማዋቀር ስራዎችን እንድትጨርሱ ያስታውሳል።

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**ማጠቃለያ (Summary):** የዕለቱን የመመዝገቢያ (signups) ብዛት እና ገቢ (revenue) ሪፖርት ያደርጋል። MRR (Monthly Recurring Revenue) ደግሞ ገቢ የሚያመጣ የደንበኝነት ምዝገባ (memberships) ባላቸው ደንበኞች ላይ የተገመተ አጠቃላይ ድምርን ያሳያል።

![Summary widget](/img/admin/network-dashboard-summary.png)

**የእንቅስቃሴ ስርአት (Activity Stream):** በUltimate Multisite ላይ የተፈጠሩ ክስተቶችን — መመዝገቢያዎችን፣ መሰረዝን፣ የጣቢያ ፍጥረቶችን፣ መደምሰስን እና ሌሎች እንቅስቃሴዎችን — ሪፖርት ያደርጋል።

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**አሁን (Right Now):** በኔትወርኩ ውስጥ ያሉ ተጠቃሚዎችን እና ጣቢያዎችን አጭር ማጠቃለያ ያሳያል፣ እንዲሁም አዳዲስ ጣቢያዎችን ወይም ተጠቃሚዎችን ለመፍጠር አቋራጮችን (shortcuts) ይሰጣል።

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**ዜና እና ውይይቶች (News & Discussions):** ዝመናዎችን (updates)፣ ወሳኝ ስህተቶችን (critical bugs) እና የደህንነት ማስተካከያዎችን (security fixes) ጨምሮ የቅርብ ጊዜውን የUltimate Multisite ዜና አምጥቶ ያሳያል።

![News widget](/img/admin/network-dashboard-news.png)

## የUltimate Multisite ዳሽቦርድ {#the-ultimate-multisite-dashboard}

የኔትወርክ ዳሽቦርድ ስለ ኔትወርኩ አጠቃላይ መረጃን ሲያቀርብ፣ የUltimate Multisite ዳሽቦርድ (በUltimate Multisite ሜኑ ከፍተኛ ደረጃ ላይ) ስለ አገልግሎቱ መረጃን ያቀርባል።

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**የወርሃዊ ገቢ ዕድገት (Monthly Recurring Revenue Growth):** በ12 ወር የካላንደር ጊዜ ውስጥ የገቢ ወርሃዊ ክፍፍልን ያሳያል። አዲስ ገቢ በሰማያዊ ሲታይ፣ መሰረዝ (cancellations) ደግሞ በቀይ ይታያል።

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**የቀን ክልል ድጋፍ (Date-range Support):** የሚያስፈልግዎ መረጃ ላይ ለማተኮር የዳሽቦርድ መረጃን በጊዜ ክፍተት (time period) መጣራት (filter) ይችላሉ። የቀን ክልል (date range) የሚታዩት widgets ስር ምን እንደሚታይ ይቆጣጠራል።

![Date range and dashboard stats](/img/admin/um-dashboard-date-range.png)

የዳሽቦርድ widgets የሚከተሉትን ያካትታሉ፡

- **ገቢ (Revenue)** — ለወቅቱ አጠቃላይ ገቢ፣ MRR እና ለእያንዳንዱ ምርት የተከፋፈሉ መረጃዎች።
- **አዲስ የደንበኝነት ምዝገባዎች (New Memberships)** — በተመረጠው ጊዜ ውስጥ ስንት አዲስ የደንበኝነት ምዝገባዎች ተፈጥረዋል።
- **በሀገር መመዝገቢያዎች (Signups by Countries)** — የመመዝገቢያዎች የጂኦግራፊያዊ ክፍፍል።
- **በፎርሞች መመዝገቢያዎች (Signups by Forms)** — ደንበኞች በየቼክአውት ፎርም (checkout form) ስንት ተመዝግበዋል ወይስ በአድሚን ስንት ተጨምሯል።
- **በብዛት የተጎበኙ ጣቢያዎች (Most Visited Sites)** — በኔትወርኩዎ ላይ በብዛት የተጎበኙ ጣቢያዎች (በ**Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** ላይ የጎበኘት ብዛት ማስላትን ማብራት ይጠበቅቅብዎታል)።
