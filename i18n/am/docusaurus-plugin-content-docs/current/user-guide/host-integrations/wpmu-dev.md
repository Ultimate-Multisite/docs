# WPMU DEV ማገናኛ (Integration)

## አጠቃላይ እይታ (Overview)
WPMU DEV ለWordPress ድረ-ገጾች አስተናጋጅ (hosting)፣ ፕልጊን (plugins) እና አገልግሎቶችን የሚያቀርብ ሰፊ የWordPress መድረክ ነው። ይህ ማገናኛ (integration) በUltimate Multisite እና በWPMU DEV hosting መካከል የዶሜይን መመሳሰልን (domain syncing) እና የSSL ሰርተፍኬት አስተዳደርን በራስ-ሰር ያደርጋል።

## ባህሪያት (Features)
- በራስ-ሰር የዶሜይን መመሳሰል (Automatic domain syncing)
- የSSL ሰርተፍኬት አስተዳደር (SSL certificate management)
- የተራዘመ የSSL ሰርተፍኬት የማረጋገጫ ሙከራዎች (Extended SSL certificate verification attempts)

## መስፈርቶች (Requirements)
ይህ ማገናኛ (integration) በWPMU DEV ላይ እየተስተናገዱ ከሆነ በራስ-ሰር ይለያል እና የተገጠመውን API ይጠቀማል። በWPMU DEV ላይ እየተስተናገዱ ከሆነ ተጨማሪ ቅንብር (configuration) አያስፈልግም።

ይህ ማገናኛ (integration) በWPMU DEV ላይ ሲስተናገዱ በራስ-ሰር የሚገለጽ የ`WPMUDEV_HOSTING_SITE_ID` constant መኖሩን ይፈትሻል።

## የማስጫ ስልጠናዎች (Setup Instructions)

### 1. WPMU DEV Hostingን ማረጋገጥ (Verify WPMU DEV Hosting)

በWPMU DEV ላይ እየተስተናገዱ ከሆነ፣ አስፈላጊ የሆኑ ቋሚ ተለዋዋጮች (constants) ቀድሞውኑ መገለጻቸውን ማረጋገጥ አለብዎት። የሚከተሉትን ያረጋግጡ፦

1. የ`WPMUDEV_HOSTING_SITE_ID` constant በስራ አካባቢዎ (environment) ተገልጿል
2. በAPI መዳረሻ ያለው ንቁ WPMU DEV አባልነት (membership) አለዎት

### 2. ማገናኛውን ማብራት (Enable the Integration)

1. በWordPress adminዎ ውስጥ፣ ወደ Ultimate Multisite > Settings ይሂዱ።
2. ወደ "Domain Mapping" ታብ (tab) ይጓዙ።
3. እስከ "Host Integrations" ድረስ ዝቅ ይሏቸው።
4. የWPMU DEV ማገናኛውን ያብሩ (Enable)።
5. "Save Changes" የሚለውን ይጫኑ።

## እንዴት ይሰራል (How It Works)

### የዶሜይን መመሳሰል (Domain Syncing)

አንድ ዶሜይን በUltimate Multisite ሲመዘገብ (mapped)፦

1. ማገናኛው (integration) ዶሜይኑን ወደ hosting አካውንትዎ ለመጨመር የWPMU DEV APIን ይጠቀማል።
2. በተጨማሪም የዶሜይኑን www ስሪት በራስ-ሰር ይጨምራል።
3. WPMU DEV የዶሜይን ቅንብርን እና የSSL ሰርተፍኬት አሰጣጥን (issuance) ይቆጣጠራል።

### የSSL ሰርተፍኬት አስተዳደር (SSL Certificate Management)

የSSL ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት ሰርተፍኬት
