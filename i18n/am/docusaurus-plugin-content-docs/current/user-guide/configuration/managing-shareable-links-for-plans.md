# የፕላን ሊንኮችን መጋራት (መቆጣጠር)

Ultimate Multisite በደንበኞችዎ ዘንድ የፕላን ሊንኮችን በቀጥታ የመጋራት አማራጭ ይሰጣል። ይህንን ለማድረግ፣ ወደ ማንኛውም የፕላን ማረሚያ ገጽ (edit page) በመሄድ የ _Click to copy the Shareable Link_ የሚለውን ቁልፍ መጫን ያስፈልግዎታል።

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_በየፕላኑ ማረሚያ ገጽ ላይ፣ ሊንኮችን ለመፍጠር የሚያስችል ቁልፍ ያገኛሉ።_

ይህ ዘዴ ሊንኩን ለመገንባት የሚያገለግለው መደበኛ የክፍያ ድግግሞሽ (default billing frequency) መሆኑን ልብ ይበሉ። ለምሳሌ፣ መደበኛ ክፍያዎ ወርሃዊ ከሆነ፣ በመጨረሻም እንዲህ ያለ ሊንክ ይኖራዎታል፦ [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

የሚጋሩት ሊንክ በሩብ ዓመት (quarterly) እንዲሆን ከፈለጉ፣ መጨረሻ ላይ '3' መጨመር አለብዎት።

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

የፕላኑን 'premium' በየዓመት (yearly) መርሃግብር ለመምረጥ፣ '1' ጨምረው ከዚያም 'year' የሚለውን ቃል መጨመር አለብዎት።

እዚህ ላይ ልብ ሊሉት ይገባል፤ ለየየ የክፍያ ድግግሞሽ (billing frequency) አንድ ሊንክ ብቻ ሊኖርዎት አይችልም። ለእያንዳንዱ የክፍያ ድግግሞሽ የተለየ ሊንክ መጋራት አለብዎት።

ሊንኩን ለደንበኞችዎ ከሰጡ በኋላ፣ ወደ ምዝገባ ሂደትዎ (signup flow) የመጀመሪያ ደረጃ ይመራቸዋል።
