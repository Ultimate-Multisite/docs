# አብሮ የተገነቡ Agents

Gratis AI Agent v1.9.0 አምስት አብሮ የተገነቡ agents ይዞ ይመጣል፤ እያንዳንዱም በተኮረ የመሣሪያዎች ስብስብ፣ በተስማሚ የስርዓት መመሪያ፣ እና በዚያ አካባቢ ከተለመዱ ተግባራት ጋር በተዛመዱ የመጀመሪያ ጥቆማዎች ቀድሞ ተዋቅሯል። በagents መካከል መቀየር ረዳቱ ምን ማድረግ እንደሚችል እና እንዴት እንደሚመልስ ይቀይራል — ከእርስዎ በኩል ምንም ውቅር ሳይፈለግ። Superdav AI Agent v1.18.0 ተዛማጅ ውህዶች ሲዋቀሩ በእነዚህ የሥራ ፍሰቶች ላይ መርሐግብርን የሚያውቁ መሣሪያዎች፣ የማስታወሻ መዝገቦች፣ የማጽደቅ መግቢያዎች፣ እና SMS ማሳወቂያዎችን ሊጨምር ይችላል።

## Agent ምንድነው?

እያንዳንዱ agent የሚከተሉትን የሚያጣምር ስም ያለው የውቅር መገለጫ ነው፦

- **መሣሪያዎች** — agent እንዲጠቀምባቸው የተፈቀዱለት ችሎታዎች (ለምሳሌ፣ Content Writer የልጥፍ ፈጠራ ችሎታዎችን ያገኛል፤ Design Studio የCSS እና theme.json ችሎታዎችን ያገኛል)
- **የስርዓት መመሪያ** — የagentን ቃና፣ ቅድሚያዎች፣ እና ገደቦች የሚያስቀምጡ መመሪያዎች
- **ጥቆማዎች** — በፍጥነት እንዲጀምሩ ለማገዝ በውይይት በይነገጽ ውስጥ የሚታዩ ቀድሞ የተጻፉ ጥያቄዎች

## Agent Picker መድረስ

1. በWordPress አስተዳዳሪ ጎን አሞሌ ውስጥ የ**Gratis AI Agent** ፓነልን ይክፈቱ።
2. በውይይት ራስጌ በላይ-ግራ ያለውን **agent አዶ** ጠቅ ያድርጉ (አዶው ንቁውን agent ለማሳየት ይቀየራል)።
3. **Agent Picker** እንደ ቅጽ-ሰንጠረዥ መሸፈኛ ይከፈታል። እያንዳንዱ agent በአዶው፣ በስሙ፣ እና በአንድ-መስመር መግለጫው ይዘረዘራል።
4. ለማንቃት agent ረድፍን ጠቅ ያድርጉ። የውይይት ራስጌው ወዲያውኑ ይዘምናል።

በውይይት መካከልም agents መቀየር ይችላሉ — የአዲሱ agent የስርዓት መመሪያ ከሚቀጥለው መልእክት ጀምሮ ተፈጻሚ ይሆናል።

## ወኪሎችን ማበጀት

እያንዳንዱ አብሮገነብ ወኪል በ`gratis_ai_agent_agents` filter በኩል ሊራዘም ወይም ሊተካ ይችላል።

### ብጁ የsystem prompt ወደ ነባር ወኪል መጨመር

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### አዲስ ወኪል መመዝገብ

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

አዲሱ ወኪል filter ከተሰራ በኋላ ወዲያውኑ በAgent Picker ውስጥ ይታያል።

### አብሮገነብ ወኪል ማስወገድ

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
