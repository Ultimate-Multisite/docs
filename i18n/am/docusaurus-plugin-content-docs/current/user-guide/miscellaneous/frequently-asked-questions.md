# በተደጋጋሚ የሚጠየቁ ጥያቄዎች

በድጋፍ ጥያቄዎች ጊዜ ብዙ ጊዜ የሚነሱ አንዳንድ **በተደጋጋሚ የሚጠየቁ ጥያቄዎች** እነሆ።

  * **Ultimate Multisite ምንድነው?**

Ultimate Multisite ለWordPress multisite installations የተሰራ የWordPress plugin ሲሆን፣ የWordPress installationዎን ወደ ባለሙያ የድረ-ገጾች መረብ (network) ይለውጠዋል—እንደ [WordPress.com](https://WordPress.com)። ይህም ደንበኞች በየወሩ፣ በየሩብ ዓመቱ ወይም በየዓመቱ ክፍያ የራሳቸውን ድረ-ገጽ እንዲፈጥሩ ያስችላል (ነጻ ዕቅዶችንም መፍጠር ይችላሉ)።

  * **Ultimate Multisite Multisite ባይኖርበት ይሰራል?**

አይ። Ultimate Multisite የWordPress multisite፣ subdirectory፣ ወይም subdomain ያስፈልገዋል።

  * **የUltimate Multisite መስፈርቶች ምንድናቸው?**

Ultimate Multisite የሚከተሉትን ይጠይቃል፦

**የWordPress Version** : v5.3+ (የሚመከር፡ የመጨረሻው የተረጋጋ Version)
**የPHP Version** : 7.4.x (እስከ > 8.0 ድጋፍ በቅርቡ ይመጣል)
**የMySQL Version** : v5+ (የሚመከር፡ Version 5.6፣ በሆስቲንግ ሰጪዎ ላይ 8.0 ካልተገኘ)

_Multisite (subdomain ወይም subdirectory) መብላት አለበት።_ [የWordPress Multisite እንዴት መጫን እንደሚቻል](/user-guide/getting-started/how-to-install-wordpress-multisite)።

  * **Ultimate Multisite በSubdirectories ባለው Multisite ይሰራል?**

አዎ። Ultimate Multisite የsubdirectory installations እና የsubdirectory installations ሁለቱንም ይደግፋል።

  * **Ultimate Multisite በSubdomains ባለው Multisite ይሰራል?**

አዎ። Ultimate Multisite የsubdirectory installations እና የsubdomain installations ሁለቱንም ይደግፋል።

  * **በተጠቃሚዎች ምን አይነት Plugins እና Themes መጠቀም እንደሚችሉ ለመገደብ Ultimate Multisite ልጠቀም እችላለሁ?**

አዎ። በUltimate Multisite አማካኝነት በኔትዎርክዎ ውስጥ ለእያንዳንዱ ዕቅድ (plan) ምን አይነት plugins እና themes መገኘት እንደሚችሉ መምረጥ ይችላሉ።

  * **በተጠቃሚው ድረ-ገጽ ላይ ሊፈጥረው የሚችለውን የፖስት ብዛት ለመገደብ Ultimate Multisite ልጠቀም እችላለሁ?**

አዎ። በUltimate Multisite አማካኝነት በኔትዎርክዎ ውስጥ ባሉ **ሁሉም ሊገኙ የሚችሉ የፖስት አይነቶች** (post types) ላይ የፖስት መጠን ገደብ (quota) ማስቀመጥ ይችላሉ። በተጨማሪም የአስተዳሚው ሊጋብቃቸው የሚችላቸው ተጠቃሚዎች ብዛት፣ **የዲስክ ቦታ መጠን**፣ እና ብዙ ተጨማሪ ነገሮችን መገደብ ይችላሉ።

  * **Ultimate Multisite የSite Templates ይደግፋል?**

አዎ። በUltimate Multisite አማካኝነት በሚፈልጉት መጠን የSite Templates መፍጠር ይችላሉ። Site Templates themes አይደሉም፤ በኔትዎርክዎ ውስጥ ያሉ መደበኛ ድረ-ገጾች ናቸው። plugins፣ themes መפעል እና የSite Templateን በፈለጉት መንገድ ማበጀት ይችላሉ። ተጠቃሚው ይህን template ቢመርጥ፣ የራሱ ድረ-ገጽ ሁሉንም ማበበጅቶች ተይዘው እንደዚህ template ቅጂ ይፈጠራል።

በተጨማሪም ለተወሰነ ዕቅድ (plan) የተለየ template ማስቀመጥ ይችላሉ። ይህን በማድረግ፣ በዚያ ዕቅድ የሚመዘገብ እያንዳንዱ ተጠቃሚ የራሱን ድረ-ገጽ እንደዚህ Site Template ቅጂ እንዲፈጠር ያደርጋል።

  * **Ultimate Multisite ከWooCommerce ጋር ይዋሃዳል?**

አዎ!

የኔትዎርክ ተጠቃሚዎች WooCommerce እንዲጠቀሙ እና መደብሮች እንዲመሰረቱ ከፈለጉ፣ ይህን ማድረግ በUltimate Multisite አማካኝነት ይቻላል። ደንበኞች በየዕቅድ ደረጃ (plan tier) ምን ያህል ምርቶችን መፍጠር እንደሚችሉ እንኳን መገደብ ይችላሉ።

በUltimate Multisite የደንበኝነት ክፍያ ክፍያዎች (subscription payments) አንፃር፣ ዋናው ድረ-ገጽዎ ላይ Ultimate Multisite ደንበኞች በWooCommerce gateways እንዲከፍሉ የሚያስችለው [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) መጠቀም ይችላሉ። በዚህ መንገድ፣ ለWooCommerce የተዘጋጁትን ትልቅ የክፍያ গেትዌይ ቤተ-መጽሐፍት መጠቀም እና የUltimate Multisite ደንበኞችን ክፍያ መፈጸም ይችላሉ።

  * **ከWPMUDEV Pro Sites የደንበኝነት ክፍያዎችን ወደ ሌላ ቦታ ለማዛወር መንገድ አለ?**

አዎ፣ Ultimate Multisite አሁን የPro Sites ኔትዎርክዎን ለማዛወር የሚረዳ [መሳሪያ አቅርቧል](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/)።

  * **በPlan Edit Page ላይ ያለው “Allowed Plugins” ክፍል ባዶ ለምን ነው?**

Ultimate Multisite በኔትዎርክዎ ላይ ባልተንቀሳቀሱ (not network-activated) plugins ላይ ብቻ መዳረሻን እንዲፈቅዱ ወይም እንዲከለክሉ ያስችላል። አንድ plugin በኔትዎርክ ደረጃ ከተንቀሳቀሰ (network-activated)፣ በተፈጥሮው በኔትዎርክዎ ውስጥ ባሉ ሁሉም ድረ-ገጾች በቅድፋት ይጫናል (loaded by default)።

በ“Allowed Plugins” ክፍል ውስጥ ምንም plugins ካልተመለከቱ፣ የሚፈልጉት plugin በኔትዎርክ ደረጃ ተንቀሳቅሷል ወይስ አልተንቀሳቀሰም ብለው ይፈትሹ።

  * **የራሴ የዋጋ ሰንጠረዥ (Pricing Table) ዲዛይን ልጠቀም እችላለሁ?**

አዎ። ከversion 1.4.0 ጀምሮ፣ Ultimate Multisite template overridesን ይደግፋል።

  * **Ultimate Multisite ከWP Engine ጋር ይሰራል?**

አዎ። Ultimate Multisite ሁልጊዜም በWP Engine የሚደገፍ ነበር፣ ነገር ግን የdomain mapping እንዲሰራ የሚያስፈልገው የአስተዳሚው (admin) ተጨማሪ በእጅ እርምጃዎች ነበር (ዶሜኑን ወደ WP Engine panel መጨመር)። **ይህ በversion 1.6.0 ጀምሮ አልሆነም።** Ultimate Multisite በቀጥታ ከWP Engine API ጋር ይዋሃዳል እና ዶሜኖቹ ከማዛመድ በኋላ ወደ panel በራስ-ሰር ይጨመራሉ።

  * **ምን አይነት የሆስቲንግ ሰጪን ይመክራሉ?**

Ultimate Multisite ከማንኛውም የሆስቲንግ ሰጪ ጋር መስራት አለበት። ብቸኛው ልዩነት አንዳንድ የሆስቲንግ ኩባንያዎች እንደ Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane እና [Runcloud.io](http://Runcloud.io) ያሉ የdomain mapping አቅም ጥብቅ ውህደት ስላላቸው ከሳጥኑ ውጭ (out of the box) እንዲሰሩ ያደርጋቸዋል። ለሌሎች ሆስቲንግስ ደግሞ የdomain mappingን በእጅ ማዘጋጀት ሊኖርብዎ ይችላል።
