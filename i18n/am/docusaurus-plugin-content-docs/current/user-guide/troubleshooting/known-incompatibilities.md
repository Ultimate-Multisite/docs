# ከሌሎች ፕልጊኖች ጋር የሚታወቁ አለመጣጣሞች {#known-incompatibilities-with-other-plugins}

Ultimate Multisiteን ከጫኑ እና ከaktyivate (activating) በኋላ፣ ያልተስተካከሉ አለመጣጣሞች ምክንያት ችግር ሊያጋጥምዎት ይችላል።

በእያንዳንዱ አዲስ ስሪት ይህን ዝርዝር ለመቀነስ በንቃት እየሰራን ቢሆንም፣ አንድ ፕልጊን እዚህ ከተዘረዘረ፣ ሙሉ በሙሉ የሚሰራ የUltimate Multisite Network እንዲኖር እሱን ማጥፋት (deactivate) ሊያስፈልግዎት ይችላል።

እነዚህን ፕልጊኖች ዝርዝር እና በእንደዚህ ዓይነት ሁኔታዎች ምን ማድረግ እንዳለብዎ እነሆ፡

**የWPMU DEV ፕልጊኖች ጥቂቶቹ** WPMU DEV የማይጠብቁት (no longer maintained) አንዳንድ ፕልጊኖች የUltimate Multisite የመመዝገቢያ ፍሰት በመጨረሻው ደረጃ እንዲቆም ያደርጋሉ (ተጠቃሚው አካውንት መፍጠር አይችልም)። እነዚህን ችግሮች የሚያስከትሉ ፕልጊኖች ዝርዝር እነሆ፡
