# Ultimate Multisite: Ad Injector በመጠቀም በኔትዎ ላይ ማስታወቂያዎችን ማስቀመጥ

Ultimate Multisite: Ad Injector እርስዎ በፕላኖችዎ ላይ ማስታወቂያዎችን እንዲያስቀምጡ የሚያስችል **ነጻ ተጨማሪ (add-on)** ነው። ይህንን በቀጥታ ከየፕላን ማረሚያ ገጽ (edit page) ላይ መጠቀም ይችላሉ።

Ultimate Multisite: Ad Injector ከጫኑ በኋላ፣ ለማንቃት ወደ _Plugins_ ገጽዎ ይሂዱ።

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_የNetwork Activate የሚለውን ቁልፍ ይጫኑ_

ተጨማሪው (add-on) ከተንቀሳቀሰ በኋላ፣ በፕላን ማረሚያ ስክሪን ላይ የሚገኘው _Advanced Options_ ሜታ ቦክስ (meta box) ላይ _Ad Settings_ የሚል አዲስ አማራጭ ይኖራል።

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_አሁን ለእያንዳንዱ ፕላን የማስታወቂያ ቅንብሮችን (ad settings) ማርትዕ ይችላሉ_

በዚያ ትብሌ ላይ፣ በፈረንጓዳው (front-end) ለፖስቶች እና ለገጾች፣ ከይዘቱ በፊት እና ከይዘቱ በኋላ ያሉትን የማስታወቂያ ኮዶች ማስገባት ይችላሉ።

_የማስታወቂያ ኮድዎን ከሰቀሉ በኋላ፣ በዚያ የተወሰነ ፕላን ላይ ባሉ ፖስቶች እና ገጾች ላይ ይታያል።_

ለዚያ ፕላን ተጠቃሚዎች የአስተዳዳሪ ገጾች (Admin Pages) ርዕስ (header) ላይ ማስታወቂያዎችን ማስገባትም ይችላሉ።

_የተጠቃሚው የኋላ ክፍል (back-end) ላይ ማስታወቂያዎችን ማሳየት ከፈለጉ ወደ Admin Panel Ad Code አማራጭ ይሂዱ_

### ከማስታወቂያ ባሻገር

Ultimate Multisite: Ad Injectorን ከፍተኛ ደረጃ ባላቸው ፕላኖች ላይ ድጋፍ (support) ለመስጠትም መጠቀም ይችላሉ።

ወደ _Admin Panel Ads_ አማራጭ ብቻ ይሂዱ እና ለድጋፍዎ መሣሪያ (support tool) የHTML ኮድ ይጨምሩ።

በዚህ መንገድ ሰዎች በቀጥታ ከየየራሳቸው የአስተዳዳሪ ፓነል (Admin Panel) ሊያገኙዎት ይችላሉ!
