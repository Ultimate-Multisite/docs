# የአስተዳዳሪ ፓነልዎን በPRO Theme ማበጀት

በደንበኛ ስራ ስንሰራ፣ ለደንበኞቻችን በWordPress የአስተዳዳሪ ዳሽቦርድ ውስጥ የተበጀ (custom) ተሞክሮ እንዲሰጣቸው የሚያስፈልግ ጊዜ አለ። ይበልጥ ሙያዊ መልክ እና ስሜት እየፈለጉ ከሆነ፣ PRO Theme ሊረዳዎት ይችላል።

PRO Theme የደንበኞቻችሁን ዳሽቦርድ ስታበጁበት እንዲረዳዎት የተዘጋጀ የአስተዳዳሪ ንድፍ (admin theme) ነው። እንዲሁም የUltimate Multisite የፕሪሚየም ፕልጊኖች ዋና ክፍል አካል ነው።

ከመጫኑ በኋላ፣ ወደ `_Plugins_` ገጽ በመሄድ በኔትዎርክዎ ላይ ለማግበር (activate) ይጫኑ።

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_በፕልጊኖች ገጽዎ ላይ በኔትዎርክ ደረጃ ያግበሱ (Activate it network-wide)_

_**ማስታወሻ:**_ _በቀላሉ ከጫኑት፣ ይህ ንድፍ (theme) ለደንበኛዎ ብቻ እንዲጨመርበት ምልክት ሊደረግበት የሚችል አማራጭ ሆኖ ይታያል።_

አሁን የአስተዳዳሪ በይነገጽዎ ሙሉ በሙሉ እንደገና ተዘጋጅቷል! በተጨማሪም በግራ በኩል ባለው ዓምድ ላይ PRO Theme የሚል አዲስ ምናሌ (menu) አግኝተዋል።
