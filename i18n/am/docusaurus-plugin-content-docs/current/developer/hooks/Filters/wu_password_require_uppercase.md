# Filter: wu_password_require_uppercase {#filter-wupasswordrequireuppercase}

በይለፍ ቃላት (passwords) ውስጥ ትልልቅ ፊደሎችን (uppercase letters) መጠየቅ መሆኑን የሚቆጣጠር ማጣሪያ ነው።

## መለኪያዎች {#parameters}

| ስም | አይነት | መግለጫ |
|------|------|-------------|
| $require | `bool` | ትልልቅ ፊደሎችን መጠየቅ መሆኑን። ህጎቹ ሲተገበሩ በቅድሚያ እውነት (true) ነው። |
| $defender_active | `bool` | Defender Pro Strong Password ን ብቃት ባለው መልኩ እየሰራ መሆኑን። |
