# Filter: wu_plan_product_types {#filter-wuplanproducttypes}

እንደ ዕቅድ (plans) መታየት ያለባቸውን የውሂብ አይነቶች (product types) የሚያጣጥር (filter) ነው።

ይህ filter add-ons ተጨማሪ የውሂብ አይነቶችን (product types) በመመዝገብ፣ እነዚህ አይነቶች በስርዓት ማረጋገጫ (validation) እና መለያየት (segregation) ውስጥ እንደ ዕቅድ እንዲታወቁ ያስችላል።
