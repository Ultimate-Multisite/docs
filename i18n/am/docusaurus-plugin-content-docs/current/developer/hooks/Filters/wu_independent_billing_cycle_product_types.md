# Filter: wu_independent_billing_cycle_product_types

በራ የክፍያ ዑደት ያላቸውን የምርት አይነቶች (product types) የሚያጣጥር (filter) ነው።

በራ የክፍያ ዑደት ያላቸው ምርቶች በካርట్‌లో የዕቅዱን የክፍያ ጊዜ መከተል አይጠበቅባቸውም። ይህ ለምሳሌ የዶሜን ምዝገባዎች (domain registrations) ባሉ ምርቶች ጠቃሚ ነው፤ እነዚህ ምርቶች ዕቅዱ ወርኃዊ (monthly) ቢሆንም ወይም ዓመታዊ (annual) ቢሆንም ሁልጊዜ በዓመት ክፍያ የሚጠይቁ በመሆናቸው ነው።

## Parameters {#parameters}

| Name | Type | Description |
