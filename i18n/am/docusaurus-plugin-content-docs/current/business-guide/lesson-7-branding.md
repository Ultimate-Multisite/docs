# ትምህርት 7፡ የራሳችሁን ስም ማስቀመጥ

ደንበኞችዎ "የሆነ የ WordPress plugin" እየተጠቀሙ ነው ብለው በፍጹም ሊሰማቸው አይገባም። በምትኩ፣ ለኢንዱስትሪው ስራ የተሰራ መድረክ እንደሆነ—የ FitSite—እየተጠቀሙ ነው ብለው እንዲሰማቸው ማድረግ አለባችሁ። ይህ ትምህርት ብራንዲንግን (branding)፣ የብራንድ ስም ማስቀመጥን (white-labeling) እና መድረኩ እንደ ምርት እንዲሰማው ማድረግን ይዳስሳል።

## የት ቆምን ነበር {#where-we-left-off}

FitSite የጤና ተቋማት ባለቤቶችን ከየዕቅድ ምርጫ እስከ ቀጥተኛ ስራ (live site) የሚያደርስ የሚሰራ የቼክአው ተከታታይ ሂደት አለው። አሁን ደግሞ፣ ሙሉ ልምዱ አንድ ወጥ፣ በብራንድዎ የተዋቀረ ምርት እንዲመስል እናደርገዋለን።

## የፕላትፎርምዎ ዶሜን (Platform Domain) {#your-platform-domain}

የብራንድዎ መሠረት የዶሜንዎ ነው። ለ FitSite፡

- **ዋና ዶሜን (Main domain)**፡ `fitsite.com` (የግብይት ድረ-ገጽዎ እና የኔትወርክ ሥር)
- **የደንበኛ ድረ-ገጾች (Customer sites)**፡ `studioname.fitsite.com` (subdomains)
- **የተበጁ ዶሜኖች (Custom domains)**፡ በGrowth እና Pro ዕቅዶች ላይ ያሉ ደንበኞች የራሳቸውን ዶሜን ማስቀመጥ ይችላሉ።

### የዶሜንዎን አቀማመጥ (Setting Up Your Domain) {#setting-up-your-domain}

1. የፕላትፎርምዎን ዶሜን ይመዝግቡ
2. ወደ ሆስቲንግ ሰጪዎ ያገናኙት (Point it to your hosting provider)
3. ለደንበኛ subdomains የwildcard DNS (`*.fitsite.com`) ያዘጋጁ
4. የwildcard SSL ን ማረጋገጥዎን ያረጋግጡ

ዝርዝር መመሪያዎችን ለማግኘት [How to Configure Domain Mapping](/user-guide/domain-mapping/how-to-configure-domain-mapping) ይመልከቱ።

## የአስተዳደር ልምድን በብራንድዎ ማስቀመጥ (White-Labeling the Admin Experience) {#white-labeling-the-admin-experience}

የጤና ተቋም ባለቤት ወደ የድረ-ገጹ ዳሽቦርድ ሲገባ፣ የ WordPress ወይም የ Ultimate Multisite ብራንዲንግ ሳይሆን የእናንተ ብራንድ ማየት አለበት።

### የተበጀ የግባ መግቢያ ገጽ (Custom Login Page) {#custom-login-page}

የ WordPress የመግቢያ ገጽን በማበጀት የሚከተሉትን ያሳዩት፡

- የ FitSite አርማዎ
- ለጤና ተቋማት ተስማሚ የሆነ የጀርባ ምስል (background imagery)
- የብራንድዎ ቀለሞች

### የዳሽቦርድ ብራንዲንግ (Dashboard Branding) {#dashboard-branding}

[Admin Page Creator](/addons/admin-page-creator) addonን ወይም የብራንድዎ ቀለም ጋር እንዲመሳሰል የቅንብረቶች CSS (custom CSS) በመጠቀም የሚከተሉትን ያድርጉ፡

- የ WordPress አርማን በ FitSite አርማዎ ይተኩ
- የአስተዳደር ቀለም ስብስብን (admin color scheme) የብራንድዎ ቀለም ጋር ያመሳስሉ።
- የጤና ተቋም ተስማሚ የሆኑ ፈጣን ሊንኮችን እና የጥቅም መረጃዎችን (help resources) የያዘ የተበጀ የዳሽቦርድ widget ይጨምሩ።

### የተበጁ የአስተዳደር ገጾች (Custom Admin Pages) {#custom-admin-pages}

ለጤና ተቋም ባለቤቶች በጣም ተዛማጅ የሆኑ ድርጊቶችን የሚያሳዩ የተበጁ የአስተዳደር ገጾችን መፍጠር ያስቡ፡

- "የክፍል መርሃ ግብረ-ጥለትዎን ያርትዑ"
- "የአሰልጣኝ መገለጫዎችን ያዘምኑ"
- "የእርስዎን ድረ-ገጽ ይመልከቱ"

ይህ ደንበኛው አዲስ ነገር ለመማር የሚያስፈልገውን ጊዜ ይቀንሳል፣ ምክንያቱም ተዛማጅ የሆኑ ድርጊቶችን በዋናው የ WordPress ምናሌ ውስጥ ከመቀበል ይልቅ በግልጽ በማቅረብ ነው።

## ግንኙነቶችዎን በብራንድዎ ማስቀመጥ (Branding Your Communications) {#branding-your-communications}

እያንዳንዱ ኢሜይል፣ የክፍያ ደረሰኝ (invoice) እና ማሳሰቢያ የእናንተን ብራንድ ማጠናከር አለበት።

### የስርዓት ኢሜይሎች (System Emails) {#system-emails}

ወደ **Ultimate Multisite > Settings > Emails** ይሂዱ እና ሁሉንም የስርዓት ኢሜይሎች ያበጁ፡

- **ከስም (From name)**፡ FitSite
- **ከኢሜይል (From email)**፡ hello@fitsite.com
- **የኢሜይል አብነቶች (Email templates)**፡ የብራንድዎ ቀለሞች እና አርማ ይጠቀሙ
- **ቋንቋ (Language)**፡ በጤና ተቋም ተስማሚ በሆነ መልኩ

የማበጀት ቁልፍ ኢሜይሎች፡

| ኢሜይል | አጠቃላይ ስሪት (Generic Version) | የ FitSite ስሪት (FitSite Version) |
|-------|----------------|-----------------|
| እንኳን ደህና መጣችሁ | "አዲሱ ድረ-ገጽዎ ዝግጁ ነው" | "የጤና ተቋም ድረ-ገጽዎ በስራ ላይ ነው" |
| የክፍያ ደረሰኝ | "ክፍያ ተቀብሏል" | "የ FitSite የደንበኝነት ክፍያ ተረጋግጧል" |
| የሙከራ ጊዜ መቋረጥ | "የእርስዎ ሙከራ ጊዜ በቅርቡ ሊያልቅ ነው" | "የ FitSite ሙከራዎ ባሉት 3 ቀናት ውስጥ ያበቃል — የስቱዲዮ ድረ-ገጽዎን በስራ ላይ ያቆዩት" |

### የክፍያ ደረሰኞች (Invoices) {#invoices}

የክፍያ ደረሰኝ አብነቶችን በሚከተሉት ነገሮች ያበጁ፡

- የ FitSite አርማዎ እና የብራንድ ቀለሞች
- የንግድዎ ዝርዝሮች
- የጤና ተቋም ተስማሚ የሆኑ የምርት ስሞች (አልባ የዕቅድ መለያዎች ሳይሆኑ)

## ለደንበኛው የሚታየው ድረ-ገጽ (The Customer-Facing Site) {#the-customer-facing-site}

ዋና ዶሜንዎ (`fitsite.com`) መድረኩን የሚሸጥ የግብይት ድረ-ገጽ ሊኖረው ይገባል። ይህ ከ Ultimate Multisite የኔትወርክ አስተዳደር የተለየ ነው — የንግድዎ የህዝብ ገጽ ነው።

ቁልፍ ገጾች፡

- **የመነሻ ገጽ (Homepage)**፡ ለጤና ተቋማት ንግዶች ግልጽ የዋጋ ጥቅም መግለጫ
- **ባህሪያት (Features)**፡ FitSite ምን እንደሚያደርግ፣ በጤና ተቋም አነጋገር
- **ዋጋ (Pricing)**፡ በኢንዱስትሪው ተስማሚ የሆኑ የባህሪ ንጽጽሮች ባሉበት ሶስት ዕቅዶችዎ
- **ምሳሌዎች (Examples)**፡ በመድረኩ ላይ የተገነቡ ድረ-ገጾችን ማሳየት
- **መመዝገብ (Sign Up)**፡ ወደ የቼክአው ቅጽዎ የሚያገናኙ ሊንኮች

:::tip የግብይት ድረ-ገጽዎ የኔትወርክ ድረ-ገጽ ሊሆን ይችላል
የግብይት ድረ-ገጽዎን በራስዎ ኔትወርክ ውስጥ እንደ ድረ-ገጽ ይፍጠሩ። ይህ ከዚሁ ዳሽቦርድ እንዲያስተዳድሩት ያስችላል እና የራስዎን መድረክ አቅም ያሳያል።
:::

## ለደንበኞች የተበጀ ዶሜን (Custom Domain for Customers) {#custom-domain-for-customers}

የተበጁ ዶሜኖች የያዙ ዕቅዶች ላይ ለሚገኙ ደንበኞች፣ ሂደቱን በግልጽ ሰነድ ያቅዱ፡

1. ደንበኛው የዶሜኑን መዝጋቢ (registrar) ይመዝግባል ወይም ያስተላልፋል
2. ደንበኛው DNS ን ወደ መድረክዎ እንዲያመለክት ያዘምናል (ትክክለኛ መዝገቦችን ያቅርቡ)
3. Ultimate Multisite የዶሜን አቀማመጥን እና SSL ን ይይዛል

ለዚህ ሂደት ብቻ የተዘጋጀ፣ ለቴክኒክ ያልሆኑ የጤና ተቋም ባለቤቶች የተጻፈ የጥቅም መጣጥፍ ወይም የዕውቀት መሠረት (knowledge base entry) ይፍጠሩ።

## የ FitSite ኔትወርክ እስካሁን የሰራነው {#the-fitsite-network-so-far}

```
FitSite Network

│   └── Marketing site on fitsite.com
└── Ready for onboarding flow (next lesson)
```

## በዚህ ትምህርት የሰራነው {#what-we-built-this-lesson}

- ለጤና ተቋም ባለቤቶች FitSiteን የሚሸጥ **የግብይት ድረ-ገጽ**
