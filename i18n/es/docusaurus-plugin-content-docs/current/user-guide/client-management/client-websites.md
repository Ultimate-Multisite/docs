# Los sitios web de tu cliente (v2)

Después de registrarse en tu red, tus clientes pueden acceder a su panel de control.

Además de la página **Account**, donde pueden ver información importante sobre sus pagos, membresías, dominios, limitaciones de planes y más, también tienen acceso a la página **My Sites**, donde pueden ver y gestionar todos sus sitios web. Todo lo que tienen que hacer es hacer clic en My Sites en su panel de control.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-giAvstK90F.png)

Se les mostrará todos sus sitios web, con un botón **Manage** justo debajo de cada uno de ellos. Si hacen clic en él, serán redirigidos al **dashboard de este sitio web específico**.

También pueden ver cuál de sus sitios web es el **primary** para los planes que les permiten tener más de un sitio web.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-UKHdV8cyYl.png)

Si desean cambiar su sitio web principal, deben ir a la página **Account** haciendo clic en el menú **Account**.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-lW9oJNcdVe.png)

Luego, deben hacer clic en **Change Default Site**.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-YCnbsxW8IB.png)

Aparecerá una ventana emergente. Haz clic en la lista desplegable y selecciona qué sitio web quieres hacer el sitio principal.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-SLSxU7NnEF.png)

Luego, simplemente haz clic en **Change Default Site** y ¡listo!

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-FQiGxBjht8.png)
