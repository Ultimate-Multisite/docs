# Los sitios web de tu cliente (v2) {#your-clients-websites-v2}

Después de registrarse en tu red, tus clientes pueden acceder a su Dashboard.

Además de la página **Account**, donde pueden ver información importante sobre sus pagos, membresías, dominios, limitaciones del plan y más, también tienen acceso a la página **Mis sitios**, donde pueden ver y gestionar todos sus sitios web. Lo único que tienen que hacer es hacer clic en Mis sitios en su Dashboard.

![Enlace Mis sitios en el Dashboard del cliente](/img/account-page/my-sites-link.png)

Se les mostrarán todos sus sitios web, con un botón **Gestionar** justo debajo de cada uno de ellos. Si hacen clic en él, serán redirigidos al **Dashboard de este sitio web específico**.

También pueden ver cuál de sus sitios web es el **principal** para los planes que les permiten tener más de un sitio web.

![Página Mis sitios que muestra los sitios web del cliente con botones Gestionar](/img/account-page/my-sites-page.png)

Aquí tienes una vista completa de la interfaz de edición del sitio:

![Interfaz completa de edición del sitio](/img/admin/site-edit-full.png)

Si quieren cambiar su sitio web principal, tienen que ir a la página Account haciendo clic en el menú **Account**.

![Enlace del menú Account en el Dashboard del cliente](/img/account-page/account-menu.png)

También puedes ver y editar los detalles del cliente:

![Interfaz de edición del cliente](/img/admin/customer-edit.png)

Luego, deben hacer clic en **Change Default Site**.

![Botón Change Default Site en la página Account](/img/account-page/change-default-site.png)

Aparecerá una ventana emergente. Haz clic en la lista desplegable y selecciona qué sitio web quieres establecer como sitio web principal.

![Selección desplegable de Change Default Site](/img/account-page/change-default-site-dropdown.png)

Luego, simplemente haz clic en **Change Default Site** y ¡listo!

![Confirmación del cambio de sitio predeterminado](/img/account-page/change-default-site-confirm.png)
