# Gestión de Clientes

La página **Customers** muestra la lista de miembros registrados en tu red. Esta página muestra información básica del usuario, incluyendo nombre, dirección de correo electrónico, último inicio de sesión y un enlace **switch to** que te permite iniciar sesión en el subsitio del usuario como administrador del sitio.

![Página de lista de clientes](/img/admin/customers-list.png)

Haz clic en **Manage** debajo de la tarjeta de un cliente para ver información detallada sobre ese cliente.

En la página de detalle del cliente, el bloque **At a Glance** muestra el ID del cliente, la hora del último inicio de sesión y el monto total gastado por este cliente.

El bloque **Customer Options** te permite marcar a este cliente como **VIP** (una etiqueta para facilitar su búsqueda). La pestaña **Billing Info** muestra información de facturación como correo electrónico, dirección y país.

El bloque **Fields** muestra el nombre de usuario del cliente, el correo electrónico y te permite cambiar el estado de **Email Verification**.

Debajo del bloque Customer Options encontrarás cuatro bloques adicionales:

- **Memberships** — Suscripciones actuales a las que el cliente está suscrito
- **Payments** — Historial de pagos del cliente
- **Sites** — Sitios que posee este cliente
- **Events** — Eventos relacionados con este cliente (verificación de correo electrónico, cambios de facturación, etc.)

Haz clic en el icono de flecha (→) junto a cualquier elemento para navegar a su página de detalle. Por ejemplo, hacer clic en la flecha junto a un sitio te lleva a la página de edición del sitio donde puedes modificar la asociación de membresía, el tipo de sitio y el estado activo.

Lo mismo aplica para membresías y pagos: cada uno enlaza a páginas de detalle donde puedes editar el estado de la membresía, el cliente asociado, los productos, la configuración del gateway de pago e incluso procesar reembolsos.

En el lado derecho de la página del cliente encontrarás:

- **Last Login & IPs** — Hora del último inicio de sesión, dirección IP y país
- **Timestamps** — Cuando se creó la cuenta del cliente y la última vez que se modificó
- **Delete** — Opción para eliminar permanentemente al cliente

:::warning
Eliminar un cliente es permanente y no se puede deshacer.
:::
