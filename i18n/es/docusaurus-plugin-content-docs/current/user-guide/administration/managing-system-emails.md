# Gestionando correos del sistema

Ultimate Multisite envía varios correos del sistema para eventos como confirmaciones de registro, recibos de pago y cambios de membresía. Puedes personalizar estas plantillas de correo para que coincidan con tu marca.

## Visualizando correos del sistema

Navega a **Ultimate Multisite → Settings** y haz clic en la pestaña **Emails** para ver la configuración relacionada con el correo.

![Settings - Emails tab](/img/admin/settings-emails.png)

Desde allí, puedes acceder a la página **System Emails** para ver todas las plantillas de correo.

![System Emails list](/img/admin/system-emails-list.png)

## Importando nuevas plantillas de correo

Las nuevas versiones de Ultimate Multisite y los complementos pueden registrar nuevas plantillas de correo. Para evitar conflictos, las nuevas plantillas no se añaden automáticamente; debes importarlas manualmente.

Para importar nuevas plantillas de correo:

1. Ve a la página **System Emails**
2. Haz clic en el botón **Reset & Import** en la parte superior de la página
3. Activa la opción **Import Emails** para ver las plantillas disponibles
4. Selecciona las plantillas que deseas importar y confirma

## Restableciendo plantillas de correo

Si has personalizado una plantilla de correo y deseas volver al contenido predeterminado, tienes dos opciones:

1. **Delete and re-import** — Elimina el correo del sistema e impórtalo de nuevo (esto borra las métricas de envío)
2. **Reset via the tool** — Usa la herramienta **Reset & Import**, activa la opción **Reset**, y selecciona los correos que deseas restablecer

La opción Reset es preferida ya que conserva las métricas de envío y otros metadatos mientras restaura el contenido predeterminado.
