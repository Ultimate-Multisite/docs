# Colocar anuncios en tu red con Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector es un **add-on gratuito** que te proporciona las herramientas para colocar anuncios en tus planes directamente desde su página de edición.

Después de instalar Ultimate Multisite: Ad Injector, ve a tu página de _Plugins_ para activarlo.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Haz clic en el botón Network Activate_

Una vez que el add-on esté activo, el cuadro meta _Advanced Options_ en la pantalla de edición del plan tendrá una nueva opción llamada _Ad Settings_.

![Ad Settings en la pantalla de edición del plan](/img/config/product-edit-full.png)

_Ahora puedes editar la configuración de anuncios para cada uno de tus planes_

En esa pestaña, podrás añadir los códigos de anuncio para publicaciones y páginas en el front-end, antes y después del contenido.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Después de insertar el código de tu anuncio, se mostrará en las publicaciones y páginas de ese plan específico._

También puedes incluir anuncios en el encabezado de las páginas de administración para los usuarios de ese plan.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Ve a la opción Admin Panel Ad Code si quieres mostrar anuncios en el back-end de tu usuario_

### Más allá de la publicidad {#beyond-advertising}

También puedes usar Ultimate Multisite: Ad Injector para ofrecer soporte a planes de nivel alto.

Solo ve a la opción _Admin Panel Ads_ y añade el código HTML de tu herramienta de soporte.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Añade soporte al cliente a planes específicos_

¡De esta manera, las personas pueden contactarte directamente desde su Admin Panel!
