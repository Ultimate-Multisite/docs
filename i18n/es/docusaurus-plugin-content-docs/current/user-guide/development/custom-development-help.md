# ¿Cómo obtener ayuda de desarrollo personalizado de Codeable?

A veces, realmente necesitamos alguna personalización increíble para hacer que los flujos de nuestra red funcionen exactamente como queremos. Si eso es lo que estás buscando, recomendamos [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), un lugar donde puedes encontrar desarrolladores experimentados con experiencia en Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) es una plataforma que conecta a clientes con desarrolladores freelance de WordPress cualificados, capaces de trabajar en proyectos de cualquier tamaño. Tienen un proceso de solicitud meticuloso para garantizar que se te asigne a expertos específicamente cualificados para la tarea que tienes en mente.

### ¿Cómo crear un proyecto en Codeable?

Primero, ve a [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) y haz clic en el botón **Start a Project**.

<!-- Captura de pantalla no disponible: página de inicio de Codeable con el botón Start a Project -->

Ahora, para enviar un proyecto, haz clic a la izquierda en **Start a Project**.

<!-- Captura de pantalla no disponible: página de envío de proyectos de Codeable con la opción Start a Project -->

En "I need help with", elige **Customizing** :

<!-- Captura de pantalla no disponible: formulario de proyecto de Codeable con Customizing seleccionado -->

En "My", elige **Plugin** :

<!-- Captura de pantalla no disponible: formulario de proyecto de Codeable con Plugin seleccionado -->

Luego haz clic en el botón **Continue**.

<!-- Captura de pantalla no disponible: botón Continue del formulario de proyecto de Codeable -->

Ahora es el momento de describir tu proyecto y tus necesidades de la mejor manera posible. ¡Intenta proporcionar tanta información como puedas!

<!-- Captura de pantalla no disponible: formulario de descripción de proyecto de Codeable -->

Responde las siguientes preguntas para ayudar con la fijación de precios.

<!-- Captura de pantalla no disponible: preguntas de precios del proyecto de Codeable -->

Después de eso, si aún no tienes una cuenta de [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), crea una y haz clic en el botón **Publish My Project**.

<!-- Captura de pantalla no disponible: creación de cuenta de Codeable y botón Publish My Project -->

Una vez que hayas publicado tu proyecto de [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), la plataforma te conectará con desarrolladores que coincidan con el conjunto de habilidades requerido para tu proyecto.

¡Este proceso ayudará a encontrar a la persona adecuada para tu trabajo, que tenga experiencia con WordPress y Ultimate Multisite!

La tarifa mínima de Codeable es de $70–$120 USD por hora. Las estimaciones cambiarán según el alcance del trabajo, la complejidad y la urgencia de un proyecto.

Una vez que recibas tu estimación, podrás contratar al desarrollador que mejor se adapte a tus necesidades y comenzar tu proyecto juntos!
