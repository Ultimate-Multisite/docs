# Cómo obtener ayuda de desarrollo personalizado de Codeable?

A veces, realmente necesitamos una personalización increíble para que nuestros flujos de red funcionen exactamente como queremos. Si eso es lo que buscas, recomendamos [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), un lugar donde puedes encontrar desarrolladores experimentados con experiencia en Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) es una plataforma que conecta a clientes con desarrolladores freelance calificados de WordPress, capaces de trabajar en proyectos de cualquier tamaño. Tienen un proceso de solicitud meticuloso para asegurar que te emparejen con expertos específicamente calificados para la tarea que tienes en mente.

### Cómo crear un proyecto en Codeable?

Primero, ve a [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) y haz clic en el botón **Start a Project**.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Ahora, para enviar un proyecto, haz clic a la izquierda en **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

En 'I need help with', elige **Customizing** :

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

En 'My', elige **Plugin** :

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Luego haz clic en el botón **Continue**.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Ahora, es hora de describir tu proyecto y tus necesidades de la mejor manera posible. ¡Intenta proporcionar tanta información como sea posible!

<!-- Screenshot unavailable: Codeable project description form -->

Responde las siguientes preguntas para ayudar con la fijación de precios.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Después de eso, si aún no tienes una cuenta de [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), crea una y haz clic en el botón **Publish My Project**.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

Una vez que hayas publicado tu proyecto en [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), la plataforma te conectará con desarrolladores que coincidan con el conjunto de habilidades requerido para tu proyecto.

¡Este proceso te ayudará a encontrar a la persona adecuada para tu trabajo, que tenga experiencia con WordPress y Ultimate Multisite!

La tarifa mínima de Codeable es de $70–$120 USD por hora. Las estimaciones cambiarán según el alcance del trabajo, la complejidad y la urgencia del proyecto.

Una vez que obtengas tu estimación, podrás contratar al desarrollador que mejor se adapte a tus necesidades y comenzar tu proyecto juntos!
