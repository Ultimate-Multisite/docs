# Ganchos y filtros de Ultimate Multisite

Ultimate Multisite está compuesto por una gran cantidad de funciones y consultas a la base de datos. Tiene ganchos y filtros que te permitirán modificar o añadir nuevo código a las funciones existentes.

Creamos una lista de estos ganchos y filtros como referencia para ayudar a usuarios avanzados a personalizar aún más su red. Puedes acceder a la lista desde Github haciendo clic en el enlace de abajo.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
