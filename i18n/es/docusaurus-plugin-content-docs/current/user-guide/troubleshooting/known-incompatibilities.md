# Conocidas incompatibilidades con otros plugins

Después de instalar y activar Ultimate Multisite, podrías encontrar problemas debido a incompatibilidades no resueltas con otros plugins.

Estamos trabajando activamente para reducir esta lista con cada nueva versión, pero si un plugin aparece aquí, es posible que necesites desactivarlo para tener una red Ultimate Multisite totalmente funcional.

Aquí tienes una lista de ellos y lo que debes hacer en estos casos:

**Algunos plugins de WPMU DEV** Algunos de los plugins _ya no_ mantenidos por WPMU DEV son conocidos por hacer que el flujo de registro de Ultimate Multisite se detenga en el último paso (el usuario no puede crear una cuenta). Aquí tienes una lista de los plugins que causan esos problemas:

  * Signup Language: _Ultimate Multisite tiene un complemento que ofrece la misma funcionalidad, llamado Ultimate Multisite: Language Switcher._
  * Set Blog Description on Blog Creation: _Esto está integrado en Ultimate Multisite, por lo que no es necesario un plugin adicional._
  * New Blog Templates: _Esto está integrado en Ultimate Multisite, por lo que no es necesario un plugin adicional._
  * Set Password on Multisite Blog Creation: _Esto está integrado en Ultimate Multisite, por lo que no es necesario un plugin adicional._
  * Signup TOS: _Esto está integrado en Ultimate Multisite, por lo que no es necesario un plugin adicional._
  * Anti-Splog: Aunque es menos potente, también está _integrado en Ultimate Multisite._

**Nota** : Todos los demás plugins de WPMU Dev pueden usarse normalmente junto con Ultimate Multisite. Ejemplos incluyen _Smush_, _Forminator_, _Defender_, etc.
