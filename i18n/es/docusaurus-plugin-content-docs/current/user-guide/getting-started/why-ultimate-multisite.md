# Por qué Ultimate Multisite

Si llegaste a este artículo, probablemente hayas escuchado que nuestro plugin podría ser la solución para hacer realidad tu idea de negocio. Esperamos que tengas las respuestas que necesitas al final de este artículo.

## ¿Qué es Ultimate Multisite?

Ultimate Multisite es un plugin de WordPress Multisite que te permite ofrecer a los clientes un WaaS (un sitio web como servicio). Con nuestro plugin, puedes tener varios niveles de suscripción y hacer que los clientes te paguen una tarifa recurrente para que un sitio esté alojado en tu red Multisite.

Ultimate Multisite te permite recibir tus pagos con diferentes pasarelas (como WooCommerce o Stripe), crear plantillas de sitio para facilitarte la vida cuando un nuevo usuario se registre en tu red, mapear dominios y personalizar tus formularios de registro para crear una experiencia personalizada para tus clientes.  

Y esto es solo el comienzo, ya que puedes ampliar sus funciones con complementos y webhooks.

## Otras características

Ultimate Multisite también ofrece:

* Integración de mapeo de dominios con proveedores de hosting gestionado de WordPress: Ultimate Multisite agrega automáticamente los dominios mapeados a los paneles de administración de WP Engine, Cloudways, cPanel, RunCloud, Closte y más cuando se mapea un nuevo dominio, así que no tienes que hacerlo manualmente. En algunos casos, incluso activaremos la obtención de certificados SSL por ti.
* Webhooks: Ultimate Multisite tiene un potente sistema de Webhooks que te permite integrar tu red con una serie de servicios diferentes. Incluso ofrecemos integración directa con Zapier, lo que abre tu red a más de 1000 aplicaciones y servicios disponibles en su plataforma.
* Páginas de administración potentes para ayudar a tus clientes a usar tu constructor de páginas favorito: Ultimate Multisite tiene un complemento útil llamado Admin Pages Creator, que básicamente te permite agregar Páginas de Administración totalmente personalizadas al panel de tus clientes usando plantillas de Elementor, Beaver Builder y Brizy.
* Dos temas de administración diferentes para personalizar la experiencia del panel de tus clientes: Check Material WP y PRO.

Y cosas más avanzadas:

* Soporte a sobrescrituras de plantillas: Con Ultimate Multisite, puedes reemplazar las plantillas HTML predeterminadas del flujo de registro con tus propias versiones sin preocuparte por cambiar archivos del plugin principal.
* API de búsqueda y reemplazo: También puedes anotar plantillas de sitio con marcadores de posición como {{site_name}} y usar la API de búsqueda y reemplazo para rellenar automáticamente esos marcadores con la información proporcionada por tu cliente justo después del registro, automáticamente.

## Casos de uso

Ahora es hora de ver algunos de los posibles casos de uso para Ultimate Multisite.

### Caso 1: Una Agencia

Normalmente, las habilidades centrales de una agencia se centran en el diseño de sitios web, con aspectos como su alojamiento o marketing listados como servicios adicionales.

Para las agencias, Ultimate Multisite presenta una propuesta de valor increíble en su capacidad para alojar y gestionar múltiples sitios web en una sola plataforma. Más aún, las agencias que estandarizan sus diseños en temas particulares como GeneratePress, Astra, OceanWP u otros pueden aprovechar las capacidades de Ultimate Multisite para activar automáticamente estos temas en cada nuevo sitio.

Se mantiene un control estricto sobre los plugins y temas a nivel de producto mediante la configuración de Ultimate Multisite, que permite que los plugins y temas estén disponibles o ocultos, así como su estado de activación cuando se instancian para un nuevo sitio.

Las agencias encontrarán tranquilidad con Ultimate Multisite, que les permite hacer lo que mejor saben hacer: diseñar sitios web excepcionales.

### Caso 2: Proveedor de nicho

Existe un viejo dicho que dice: “Haz una cosa y hazla bien”. Para muchos especialistas, esto significa crear un producto o servicio alrededor de una idea central única.

Si te concentras en desarrollar una red de sitios de nicho, probablemente quieras ofrecer servicios basados en un marco y plataforma comunes. ¡Ahí es donde las plantillas de sitio pueden ayudar!

Una de las características innovadoras de Ultimate Multisite, un sitio plantilla, es aquel donde el tema ha sido instalado y activado, los plugins necesarios instalados y activados, y se han creado publicaciones o páginas de muestra. Cuando un cliente crea un nuevo sitio basado en la plantilla, el contenido y la configuración de la plantilla se copian al sitio recién creado.

Para un proveedor de sitios y servicios de nicho, esto brinda una ventaja inigualable en la capacidad de crear instantáneamente un sitio listo para usar con plugins y diseño personalizados. El cliente solo necesita proporcionar una entrada mínima para completar el servicio.

### Caso 3: Hosting web de WordPress

Hay una miríada de formas de alojar sitios WordPress, pero rara vez es tan simple como proporcionar espacio web a un cliente con una versión preinstalada de WordPress. Esto se debe a que una serie de decisiones y consideraciones deben unirse para ofrecer un servicio significativo.

Ultimate Multisite sobresale en esta área al proporcionar una solución integral llave en mano para el alojamiento de sitios WordPress. La solución incluye los mecanismos centrales para ofrecer servicios de suscripción, recolección de pagos, formularios de pago, cupones de descuento y comunicaciones con el cliente.

Gran parte del trabajo integral necesario para instalar, configurar y mantener correctamente un WordPress Multisite es facilitado por Ultimate Multisite en la medida en que los administradores de la red solo necesitan considerar aspectos relacionados con su servicio o nicho, como niveles de producto, precios y ofertas de servicio.

Para los desarrolladores que deseen integrarse con Ultimate Multisite, la solución también ofrece una API RESTful completa y Webhooks para notificaciones de eventos.

Sin depender de una miríada de plugins y licencias externas, Ultimate Multisite ofrece una solución rica en funciones y comparable a la de Wix, Squarespace, [WordPress.com](https://WordPress.com) y otros.
