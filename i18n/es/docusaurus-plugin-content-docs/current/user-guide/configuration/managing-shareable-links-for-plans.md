# Gestionar enlaces compartibles para planes

Ultimate Multisite te ofrece la opción de compartir enlaces de planes directamente con tus clientes. Para hacerlo, ve a cualquiera de las páginas de edición de tus planes y haz clic en el botón _Haga clic para copiar el enlace compartible_.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-2DJoMvapXh.png)

_En la página de edición del plan, puedes encontrar el botón para crear enlaces compartibles_

Tenga en cuenta que este mecanismo utiliza la frecuencia de facturación predeterminada para construir el enlace. Esto significa que si su facturación predeterminada es mensual, por ejemplo, al final tendrá un enlace como este: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Si desea que el enlace compartible tenga por defecto la facturación trimestral, deberá agregar 3 al final

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Para seleccionar el plan _premium_ en un calendario anual, agregaría 1 y luego la palabra:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Tenga en cuenta que no puede tener un solo enlace para diferentes frecuencias de facturación. Debe compartir un enlace específico para cada una de sus frecuencias de facturación.

Una vez que comparta el enlace con sus clientes, lo redirigirá al primer paso de su flujo de registro.
