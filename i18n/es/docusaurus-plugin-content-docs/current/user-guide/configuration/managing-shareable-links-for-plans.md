# Gestionar enlaces compartibles para planes {#managing-shareable-links-for-plans}

Ultimate Multisite te ofrece la opción de compartir enlaces de planes directamente con tus clientes. Para hacer esto, ve a cualquiera de las páginas de edición de tus planes y haz clic en el botón _Haz clic para copiar el enlace compartible_.

![Página de edición de producto con botón de enlace compartible](/img/config/product-edit-full.png)

_En la página de edición del plan, puedes encontrar el botón para crear enlaces compartibles_

Ten en cuenta que este mecanismo utiliza la frecuencia de facturación predeterminada para crear el enlace. Esto significa que, si tu facturación predeterminada es mensual, por ejemplo, al final tendrás un enlace como este: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Si deseas que el enlace compartible tenga como valor predeterminado la frecuencia trimestral, tendrás que añadir 3 al final

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Para seleccionar el plan _premium_ con una programación anual, añadirías 1 y luego la palabra:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Ten en cuenta que no puedes tener un solo enlace para diferentes frecuencias de facturación. Debes compartir un enlace específico para cada una de tus frecuencias de facturación.

Una vez que compartas el enlace con tus clientes, los redirigirá al primer paso de tu flujo de registro.
