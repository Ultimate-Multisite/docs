# Action: wu_site_created

Se dispara después de que se crea un sitio por primera vez.

No se dispara si se duplica desde una plantilla.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $data | `array` | Los datos del objeto que se almacenarán. |
| $site | `\Site` | La instancia del objeto. |
