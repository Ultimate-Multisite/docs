# Acción: wu_register_capabilities

Se dispara para permitir que los complementos adjunten módulos de capacidad a las integraciones.

## Parámetros {#parameters}

| Nombre | Tipo | Descripción |
|------|------|-------------|
| $registry | `\Integration_Registry` | La instancia del registro. |
