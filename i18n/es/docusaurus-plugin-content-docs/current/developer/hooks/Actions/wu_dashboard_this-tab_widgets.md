# Action: wu_dashboard_\{$this->tab\}_widgets

Permitir a los desarrolladores de plugins agregar widgets al Panel de la Red.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $tab | `string` | La pestaña actual. |
| $screen | `\WP_Screen` | El objeto de pantalla. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Instancia de la página de administración de Ultimate Multisite. |
