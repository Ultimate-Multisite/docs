# Action: wu_mcp_abilities_registered {#action-wumcpabilitiesregistered}

Se dispara después de que las habilidades MCP se registran para una entidad.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $ability_prefix | `string` | El prefijo de la habilidad. |
| $model_name | `string` | El nombre del modelo. |
| $this | `object` | La instancia del gestor. |
