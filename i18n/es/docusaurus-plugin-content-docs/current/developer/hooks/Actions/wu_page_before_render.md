# Action: wu_page_before_render

Permite a los desarrolladores de plugins agregar contenido adicional antes de imprimir la página.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $page_id | `string` | El ID de esta página. |
| $page | `object` | El objeto de la página. |

### Since

- 1.8.2
