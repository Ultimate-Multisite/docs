# Acción: wu_magic_link_login

Se dispara después de que un usuario inicie sesión mediante un enlace mágico.

## Parámetros

| Nombre | Tipo | Descripción |
|--------|------|-------------|
| $user_id | `int` | El ID del usuario. |
| $site_id | `int` | El ID del sitio. |
