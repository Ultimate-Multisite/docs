# Filtro: wu_generated_username_from_email

Filtro generado de nombre de usuario del cliente.

## Parámetros

| Nombre | Tipo | Descripción |
|------|------|-------------|
| $username | `string` | Nombre de usuario generado. |
| $email | `string` | Dirección de correo electrónico del nuevo cliente. |
| $new_user_args | `array` | Array de argumentos de nuevo usuario, que puede incluir nombres y apellidos. |
| $suffix | `string` | Añadir cadena al nombre de usuario para hacerlo único. |
