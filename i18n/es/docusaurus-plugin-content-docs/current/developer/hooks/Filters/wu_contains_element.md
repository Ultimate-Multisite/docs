# Filtro: wu_contains_element

Permitir a los desarrolladores cambiar los resultados de la búsqueda inicial.

Esto es útil para constructores de terceros y demás.

## Parámetros

| Nombre | Tipo | Descripción |
|------|------|-------------|
| $contains_elements | `bool` | Si el element está contenido en el content. |
| $content | `string` | El content que se está examinando. |
| $element | `self` | El element actual. |
| $post | `null\|\WP_Post` | post para comprobar. |

### Desde

- 2.0.0
