# Filtro: wu_independent_billing_cycle_product_types

Filtrar los tipos de producto que tienen ciclos de facturación independientes.

Los productos con ciclos de facturación independientes no se ven obligados a coincidir con el período de facturación del plan en el carrito. Esto es útil para productos como los registros de dominios que siempre se facturan anualmente, independientemente de si el plan es mensual o anual.

## Parámetros

| Nombre | Tipo | Descripción |
|------|------|-------------|
| $types | `array` | Array de tipos de producto con ciclos de facturación independientes. |
