# Filter: wu_\{$id\}_form_fields

Filtra los campos de un formulario. El formulario se identifica por el ID en el nombre del filtro.

## Parámetros

| Nombre | Tipo | Descripción |
|--------|------|-------------|
| $fields | `array` | Lista de campos del formulario. |

- 2.0.0
