# Filtro: wu_username_from_email

Filtrar el nombre de usuario del nuevo cliente.

## Parámetros

| Nombre | Tipo | Descripción |
|--------|------|-------------|
| $username | `string` | Nombre de usuario del cliente. |
| $email | `string` | Dirección de correo electrónico del nuevo cliente. |
| $new_user_args | `array` | Matriz de argumentos de nuevo usuario, que puede incluir nombres y apellidos. |
| $suffix | `string` | Añadir cadena al nombre de usuario para hacerlo único. |
