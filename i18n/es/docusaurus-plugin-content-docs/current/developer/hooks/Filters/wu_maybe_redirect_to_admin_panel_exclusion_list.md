# Filtro: wu_maybe_redirect_to_admin_panel_exclusion_list

Crea una lista de exclusión de parámetros que impidan la redirección automática.

Esto es necesario porque, de lo contrario, el constructor de páginas no podrá editar la página de inicio de sesión una vez que esté definida.
