## Parámetros

| Nombre | Tipo | Descripción |
|------|------|-------------|
| $require | `bool` | Si se requiere minúsculas. Por defecto verdadero cuando se aplican las reglas. |
| $defender_active | `bool` | Si Defender Pro Strong Password está activo. |
