# Filter: wu_current_\{$type\}_get_manage_url

Permite a los desarrolladores modificar los parámetros de la URL de gestión del sitio.

## Parámetros

| Nombre | Tipo | Descripción |
|------|------|-------------|
| $manage_site_url | `string` | La URL de gestión del sitio. |
| $id | `int` | El ID del sitio. |
| $site_hash | `string` | El hash del sitio. |

### Desde

- 2.0.9

### Fuente

Definido en [`inc/class-current.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-current.php#L214) en la línea 214

## Devuelve

La URL de gestión modificada.
