# Filtro: secure_auth_cookie

Filtra si la cookie de autenticación debe enviarse solo a través de HTTPS.

## Parámetros

| Nombre | Tipo | Descripción |
|--------|------|-------------|
| $secure | `bool` | Si la cookie debe enviarse solo a través de HTTPS. |
| $user_id | `int` | ID de usuario. |

### Desde

- 3.1.0
