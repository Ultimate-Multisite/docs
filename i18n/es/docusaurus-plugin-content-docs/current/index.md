# Ultimate Multisite Documentation

Bienvenido a la documentación de Ultimate Multisite. Ultimate Multisite transforma WordPress Multisite en una plataforma completa de Website as a Service (WaaS).

## Documentation Sections

### [User Guide](/user-guide/getting-started/ultimate-multisite-101)

Todo lo que necesitas para configurar, administrar y gestionar tu plataforma WaaS. Cubre la instalación, pasarelas de pago, mapeo de dominios, gestión de clientes y más.

### [Developer Documentation](/developer/hooks)

Referencia técnica para desarrolladores que construyen sobre Ultimate Multisite. Incluye referencia de hooks, documentación de la API REST y guías de desarrollo de complementos.

### [Addons](/addons)

Documentación de complementos de pago que extienden Ultimate Multisite con características adicionales como integración de WooCommerce, análisis, venta de dominios y más.
