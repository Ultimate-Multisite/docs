# AI Site Builder

ultimate-multisite-ai-site-builder

## Requisitos

- WordPress 5.3 o superior
- PHP 7.4 o superior
- Multisite Ultimate plugin (activo)

## Instalación

1. Sube los archivos del addon a tu directorio `/wp-content/plugins/`
2. Activa el plugin a través del menú 'Plugins' en WordPress
3. Configura la configuración del addon en el administrador de la red

## Desarrollo

### Configuración

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Pruebas

El addon incluye un marco de pruebas integral:

- **Pruebas Unitarias**: pruebas basadas en PHPUnit con el conjunto de pruebas de WordPress
- **Clase Base de Pruebas**: proporciona métodos auxiliares para escenarios de prueba comunes
- **Cobertura de Pruebas**: configurado para rastrear la cobertura de código
- **Pruebas Multisitio**: las pruebas se ejecutan en un entorno multisite

### Estándares de Código

- **Estándares de Codificación de WordPress**: aplicados mediante PHPCS
- **Compatibilidad con PHP 7.4+**: se admiten características modernas de PHP
- **Análisis Estático**: integración de PHPStan para la comprobación de tipos
- **Modernización de Código**: reglas de Rector para actualizaciones de PHP

### Sistema de Construcción

- **Procesamiento de Activos**: minificación de archivos CSS/JS
- **Traducción**: generación de archivos POT
- **Empaquetado**: creación de archivos de distribución
- **Modo de Desarrollo**: activos sin minificar para depuración

## Configuración

El addon admite varias opciones de configuración accesibles a través de la interfaz del administrador de la red.

## Ganchos y Filtros

### Acciones

- `ultimate-multisite-ai-site-builder_init` - Se dispara después de la inicialización del addon
- `ultimate-multisite-ai-site-builder_loaded` - Se dispara después de que se cargan todos los archivos del addon

### Filtros

- `ultimate-multisite-ai-site-builder_settings` - Modificar la configuración del addon
- `ultimate-multisite-ai-site-builder_enabled` - Sobrescribir el estado de habilitación/deshabilitación del addon
