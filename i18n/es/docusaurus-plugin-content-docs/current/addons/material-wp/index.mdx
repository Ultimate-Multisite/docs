# Material WP

La interfaz de administración de WordPress ha sido la misma durante mucho tiempo. Pero, ¿alguna vez te has preguntado cómo se vería una versión de Material Design de la interfaz de administración?

Material WP es un tema de administración desarrollado para brindarte ese aspecto y sensación, ayudándote a ofrecer a tus clientes un panel de control más personalizado.

## Features

- Soporte de CSS personalizado
- Configuraciones de estilo fáciles
- Reordenar menús
- Importación/exportación sencilla

## Documentation

**Personalizando tu panel de administración con Material WP**

## Installation

1. Sube 'material-wp' al directorio '/wp-content/plugins/'
2. Activa el plugin a través del menú 'Plugins' en WordPress

## Usage Guide

La interfaz de administración de WordPress ha sido la misma durante mucho tiempo. Pero, ¿alguna vez te has preguntado cómo se vería una versión de Material Design de la interfaz de administración?

Después de **instalarlo**, ve a tu página de _Plugins_ y haz clic para activarlo en tu Red.
