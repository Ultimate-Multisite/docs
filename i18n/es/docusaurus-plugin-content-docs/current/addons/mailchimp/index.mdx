# Mailchimp

Multisite Ultimate: Integración con Mailchimp

Integra tu red Multisite Ultimate con Mailchimp. Puedes vincular ciertos planes a Listas de Mailchimp y agregar automáticamente suscriptores a esas listas al registrarse.

## Instalación

1. Sube 'ultimate-multisite-mailchimp' al directorio '/wp-content/plugins/'
2. Activa el plugin a través del menú 'Plugins' en el Administrador de la Red de WordPress

## Guía de uso

Sincronizar los correos electrónicos de los suscriptores con tus listas de Mailchimp es muy sencillo con nuestro complemento Ultimate Multisite: Integración con Mailchimp. ¡Este tutorial tiene como objetivo ayudarte a configurar todo!

## Obtener una clave API

Antes de comenzar a explorar las diferentes configuraciones disponibles, necesitamos una clave API de Mailchimp. Esto permitirá que Ultimate Multisite se comunique con la API de Mailchimp para recuperar listas, grupos y agregar direcciones de correo electrónico a esas listas y grupos cuando los usuarios se registren.

Sigue este tutorial para obtener tu propia clave API de Mailchimp: [Acerca de las claves API | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

_Añade tu clave API de MailChimp en_ _**Ultimate Multisite Settings - > Add-on Settings -> MailChimp**_

## Modo de integración

El complemento de Integración con Mailchimp ofrece dos “modos de integración”, lo que significa que hay dos formas diferentes de usar el complemento para sincronizar direcciones de correo electrónico.

<!-- Pantalla de selección del modo de integración de MailChimp con opciones Múltiples Listas y Lista Única -->

_Two integration modes for greater flexibility!_

### Modo 1: Múltiples Listas

El modo predeterminado utiliza **Múltiples Listas**. Este modo te permitirá seleccionar múltiples listas de Mailchimp para cada uno de los planes. Es la mejor opción si segmentas a tus usuarios en diferentes listas.

Por ejemplo, supongamos que tienes 3 listas diferentes en tu cuenta de Mailchimp: **Newsletter**, **Plan A** y **Plan B**. Quieres que todos los usuarios, independientemente de su plan, también estén incluidos en la lista Newsletter, para que no se pierdan tus actualizaciones.

Para lograr eso, puedes usar el modo Múltiples Listas para agregar ambas listas **Newsletter** y **Plan A** a la configuración de Mailchimp del Plan A, y **Newsletter** y **Plan B** a la configuración de Mailchimp del Plan B.

<!-- Pantalla de selección de listas predeterminadas de MailChimp para cuentas nuevas y cuentas canceladas -->

_Selecciona las listas predeterminadas para cuentas nuevas y para cuentas canceladas_

### Modo 2: Lista Única + Grupos Múltiples

Mailchimp te permite segmentar tu lista en grupos más pequeños de usuarios llamados **Grupos**. Esto te brinda mayor control, ya que puedes enviar una campaña a un subconjunto específico de tus suscriptores sin tener que crear múltiples copias de la misma campaña en Mailchimp (lo que dificulta analizar las métricas de esa campaña después).

Por ejemplo, puedes tener una única lista llamada **My Network Subscribers**, y dentro de ella tener grupos para **Plan A** y **Plan B**.

<!-- Pantalla de selección de lista única y grupos de Mailchimp para listas predeterminadas -->

_Selecciona las listas y grupos predeterminados, si usas el modo de integración “lista única + grupos múltiples”_

## Listas y Grupos Predeterminados

Después de seleccionar qué modo de integración tiene más sentido para tu caso de uso particular, es hora de seleccionar las listas predeterminadas (y grupos, según el modo). La lista predeterminada se usará cuando un plan determinado no tenga una lista específica seleccionada.

También puedes seleccionar la lista predeterminada para cuentas canceladas. Cada vez que un usuario termina su suscripción y elimina su cuenta de tu red, su dirección de correo electrónico se moverá a la lista establecida para cuentas canceladas.

## Habilitar Suscripción Doble

En la misma página de configuración, podrás habilitar **Suscripción Doble**. Esta opción añadirá al usuario tanto a la lista predeterminada seleccionada como a la lista seleccionada para el plan del usuario. Si esta opción no está habilitada, el usuario solo se añadirá a la lista seleccionada para su plan. Si no se establece ninguna para ese plan en particular, entonces el usuario se añadirá a la lista predeterminada seleccionada.

<!-- Pantalla de activación de suscripción doble de Mailchimp -->

_Puedes suscribir a tus usuarios tanto a la lista predeterminada como a la lista del plan al registrarse_

## Selección de Listas y Grupos para cada Plan

Ve a tus **Planes** y selecciona el plan que deseas editar. Ten en cuenta que en el meta-caja **Opciones Avanzadas**, estará disponible una nueva pestaña llamada MailChimp.

En esa pestaña, podrás seleccionar múltiples listas o una lista única y múltiples grupos (dependiendo del modo de integración) para ese plan.

<!-- Pantalla de la pestaña MailChimp en la página de edición de plan mostrando la selección de listas y grupos -->

_Selecciona las Listas y Grupos que deseas usar para cada uno de tus planes_

¡Cuando un nuevo usuario se registre para ese plan, su dirección de correo electrónico se añadirá a las listas y grupos seleccionados!
