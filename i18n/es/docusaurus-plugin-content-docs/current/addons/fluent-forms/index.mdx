# Fluent Forms

Este complemento permite añadir límites a Fluent Forms en Multisite Ultimate en productos, membresías o sitios.  
Fluent Forms utiliza tablas personalizadas en lugar de tipos de publicaciones personalizadas, por lo que las opciones de limitación normales no funcionan con Fluent Forms.

## Features

- Añadir límites para Fluent Forms o Conversational Fluent Forms  
- Los clientes reciben un mensaje amigable que les sugiere actualizar su plan cuando alcanzan el límite
