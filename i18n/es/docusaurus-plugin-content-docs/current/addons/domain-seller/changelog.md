# Registro de cambios del Vendedor de Dominios

Versión: 2.0.1 - Lanzado el 2025-09-28

* Renombrar prefijo a ultimate-multisite; actualizar dominio de texto; incremento de versión.

### Versión 2.0.0 - 2024-XX-XX

**Reescritura importante para Multisite Ultimate v2**

* **Nuevo:** Reescritura completa con arquitectura moderna PHP 7.4+
* **Nuevo:** Integración fluida con el sistema de pago Multisite Ultimate v2
* **Nuevo:** Gestión de productos de dominio con opciones de precios flexibles
* **Nuevo:** Arquitectura de soporte para múltiples proveedores de dominio
* **Nuevo:** Renovación automática e integración de suscripciones
* **Nuevo:** Interfaz de gestión de dominios para clientes
* **Nuevo:** Monitoreo de dominios y registros de administrador
* **Nuevo:** Soporte de cupones para productos de dominio
* **Nuevo:** Gestión completa de configuraciones
* **Nuevo:** Base de código extensible y amigable para desarrolladores
* **Mejorado:** Proveedor OpenSRS actualizado con soporte completo de funciones
* **Mejorado:** Interfaz moderna consistente con Multisite Ultimate v2
* **Corregido:** Todo el código obsoleto v1 actualizado a estándares v2
* **Eliminado:** Compatibilidad heredada v1 (cambio disruptivo)

* Corregido: Incompatibilidad con Groundhogg CRM

* Corregido: Se eliminó el campo de clave de licencia

* Búsqueda y registro de dominios simples
* Permisos de dominio basados en planes
