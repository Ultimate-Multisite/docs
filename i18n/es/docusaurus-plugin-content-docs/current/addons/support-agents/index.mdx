# Agentes de Soporte

Gestionar una red completa por tu cuenta no es una tarea fácil. A medida que tu red comience a crecer, comenzarás a ver una demanda creciente de soporte. Será una buena idea incorporar personas que te ayuden a aliviar tu carga de trabajo. Sin embargo, otorgar acceso/permiso completo es arriesgado y puede comprometer tu red. Ahí es donde entra este complemento Agente de Soporte.

Este complemento te permite crear un nuevo tipo de usuario de WordPress llamado Agente de Soporte, y te brinda control total sobre los permisos que tiene ese usuario. Otorgar permiso es tan sencillo como activar un botón de alternancia y desactivarlo si decides retirarlo.

## Features

- Lista de permisos de WordPress y Ultimate Multisite
- Habilitar y deshabilitar el acceso con facilidad
- Crear un número ilimitado de cuentas de Agente

## Installation

1. Sube 'ultimate-multisite-support-agents' al directorio '/wp-content/plugins/'
2. Activa el complemento a través del menú 'Plugins' en WordPress
3. Sigue el asistente paso a paso para configurar el complemento
