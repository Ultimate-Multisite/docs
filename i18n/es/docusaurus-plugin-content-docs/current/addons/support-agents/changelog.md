# Registro de cambios de Agentes de Soporte

Versión: 1.0.13 - Publicado el 2025-09-28

* Renombrar prefijo a ultimate-multisite; actualizar dominio de texto; aumento de versión.

Versión 1.0.12 - Publicado el 2025-09-17

* Corregido: compatibilidad con Ultimate Multisite 2.4.4

Versión 1.0.11 - Publicado el 2023-11-21

* Corregido: compatibilidad con Ultimate Multisite 2.3.0

Versión 1.0.10 - Publicado el 2023-08-30

* Corregido: eliminación masiva de Agentes de Soporte

Versión 1.0.9 - Publicado el 2023-08-09

* Añadido: Se añadió la capacidad de leer formularios de pago.
* Corregido: diseño del formulario de permisos de Agentes de Soporte;
* Corregido: mensaje de advertencia en el formulario de eliminación de Agentes de Soporte;
* Corregido: ocultar widgets del panel de red para usuarios no autorizados.
* Corregido: corregidas las capacidades de edición y eliminación de Agentes de Soporte.
* Corregido: corregido el proceso de creación de Agentes de Soporte.
* Corregido: impedido agregar un cliente como Agente de Soporte.
* Corregido: redirigidos usuarios no autorizados a la página de administración de edición de Agentes de Soporte.
* Corregido: mejorado el diseño del formulario de permisos de Agentes de Soporte.
* Interno: implementado un nuevo proceso de compilación.

Versión 1.0.8 - Publicado el 2022-01-21

* Interno: añadido generador de ganchos y filtros;
* Interno: añadidos stubs de Ultimate Multisite para la calidad de vida del desarrollador;
* Añadido: capacidades de formulario de pago;
* Corregido: desajuste de nombre de capacidad entre formularios de Ultimate Multisite y opciones de este complemento;

Versión 1.0.7 - Publicado el 2021-12-02

* Corregido: eliminación de Agentes de Soporte no funciona;
* Corregido: eliminación masiva de Agentes de Soporte no funciona;

Versión 1.0.6 - Publicado el 2021-11-22

* Corregido: el modal de añadir nuevo no funciona después de optimizaciones de rendimiento en el núcleo;
* Corregido: vista de elemento de cuadrícula presente en el núcleo y no en el complemento;

Versión 1.0.5 - Publicado el 2021-10-31

* Corregido: pequeñas incompatibilidades con PHP 8;
* Mejorado: wp_ultimo_skip_network_active_check ahora usa wu_is_must_use();

Versión 1.0.4 - 2021-09-24

* Añadido: filtro wp_ultimo_skip_network_active_check para configuraciones basadas en mu-plugins;

Versión 1.0.3 - Publicado el 2021-08-29

* Añadido: soporte a la API REST - los Agentes de Soporte pueden ser creados vía REST;

Versión 1.0.2 - Publicado el 2021-07-26

* Corregido: la actualización de permisos de agente no funciona;

Versión 1.0.1 - Publicado el 2021-05-28

* Añadido: Aviso cuando el usuario ya es super administrador;
* Añadido: Permiso para permitir el cambio de usuario;
* Mejorado: mejores campos de formulario;

Versión 1.0.0 - Lanzamiento inicial
