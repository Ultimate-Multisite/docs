# Exportador de Sitios

## Contenido

Entendemos la necesidad de poder mover subsitios dentro de su red hacia una instalación de sitio único y viceversa si su cliente decide migrar su sitio web. El complemento Exportador e Importador de Sitios facilita esta tarea. Esta herramienta exporta e importa sitios usando archivos zip que también pueden servir como respaldo.

## Funcionalidades

* Exportar e Importar Sitios
* Interfaz fácil de usar
* Crear copia de seguridad del sitio

## Documentación

**Cómo Exportar e Importar un Sitio**

## Instalación

1. Suba **ultimate-multisite-site-exporter** al directorio **/wp-content/plugins/**
2. Active el plugin a través del menú **Plugins** en WordPress
3. Siga el asistente paso a paso para configurar el plugin

## Guía de Uso

Con Ultimate Multisite puede exportar un sitio desde su red o importar un sitio a su instalación multisite o de sitio único.

Para iniciar el proceso, vaya a nuestra página de complementos y instale el complemento Ultimate Multisite: Site Exporter & Importer.![Página de complementos](/img/admin/addons-page.png)

![Página de complementos con opción de instalación](/img/admin/addons-page.png)

Vaya a su página de Plugins y haga clic en **Network Activate** el complemento que acaba de instalar.<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Site Exporter addon -->

La opción **Export & Import** se encuentra en el menú Ultimate Multisite, a la izquierda.

<!-- Screenshot unavailable: Ultimate Multisite menu showing Export and Import option -->

## Cómo exportar un sitio

Para exportar un sitio desde su red, vaya a **Export & Import** en el menú Ultimate Multisite y, en la pestaña **Export Sites**, haga clic en **Generate new Site Export**.

<!-- Screenshot unavailable: Export Sites tab with Generate new Site Export button -->

Busque el sitio que desea exportar y seleccione los elementos que desea incluir en el archivo exportado. Puede incluir temas, plugins y archivos subidos. Si su sitio es demasiado grande, puede elegir ejecutar el proceso de exportación en segundo plano.<!-- Screenshot unavailable: Site export options with site search, include themes/plugins/uploads checkboxes -->

Haga clic en el botón **Export Site**. Su sitio exportado estará disponible en la lista **Existing Exports**. Haga clic en el botón **Download ZIP** para descargar el archivo.<!-- Screenshot unavailable: Existing Exports list with Download ZIP button -->

## Cómo importar un sitio

Para importar un sitio, vaya a la pestaña **Import a Site** y cargue el archivo .zip del sitio que desea importar. Puede cambiar la URL a una nueva o mantener la URL antigua dejando el espacio en blanco. Además, puede marcar la casilla para eliminar el archivo .zip de su instalación después de la importación. Luego haga clic en **Begin Import**.

<!-- Screenshot unavailable: Import a Site tab with zip upload field and Begin Import button -->

El proceso puede tardar un poco. Verá una importación pendiente en la lista debajo del importador.

<!-- Screenshot unavailable: Pending import in the list below the importer -->

Puede detener el proceso haciendo clic en **Cancel Import**.

<!-- Screenshot unavailable: Cancel Import button to stop the import process -->

Después de que el proceso termine, su sitio ya estará disponible en su red.

## Cómo importar un sitio en una instalación única

Para importar un sitio a su instalación única, primero instale y active el importador en su sitio. Puede encontrar el enlace para descargarlo en la pestaña exportador.

<!-- Screenshot unavailable: Exporter tab with link to download single-site importer -->

Puede encontrar el importador en el menú **Tools**, a la izquierda, en el subelemento **Export & Import**.

<!-- Screenshot unavailable: Single-site Tools menu showing Export and Import submenu -->

Cargue el archivo .zip del sitio que desea importar. Puede cambiar la URL a una nueva o mantener la URL antigua dejando el espacio en blanco. Además, puede marcar la casilla para eliminar el archivo .zip de su instalación después de la importación. Luego haga clic en **Begin Import**.

<!-- Screenshot unavailable: Single-site import form with zip upload and Begin Import button -->

El proceso puede tardar un poco. Verá una importación pendiente en la lista debajo del importador.

<!-- Screenshot unavailable: Single-site pending import in the list -->

Puede detener el proceso haciendo clic en **Cancel Import**.

<!-- Screenshot unavailable: Cancel Import button to stop the import process -->

Después de que el proceso termine, su sitio será reemplazado por el sitio que acaba de importar.
