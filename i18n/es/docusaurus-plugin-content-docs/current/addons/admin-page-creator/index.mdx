# Admin Page Creator

## Contenido
Admin page creator te permitirá añadir fácilmente nuevas páginas de administración y subpáginas dentro del panel de administración de WordPress. Se puede usar para crear una página de bienvenida o una página de tutorial que sirva como página de destino para nuevos usuarios. Puedes usar constructores de páginas como Beaver Builder, Elementor, Brizy y Oxygen para diseñar las páginas sin complicaciones.

También ofrece soporte para HTML, CSS y JavaScript personalizados, así como control de acceso basado en el plan y el rol del usuario.

## Características
* Soporte de HTML y CSS personalizados
* Soporte de JavaScript
* Modo de contenido PHP
* Permiso basado en el rol del usuario
* Soporte para constructores de páginas

## Documentación
[Creando Páginas de Administración Personalizadas y Subpáginas usando Multisite Ultimate: Admin Page Creator](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## Instalación

1. Sube **ultimate-multisite-admin-page-creator** al directorio **/wp-content/plugins/**
2. Activa el plugin a través del menú **Plugins** en tu Administración de Red de WordPress

## Guía de Uso

Mientras WordPress facilita mucho añadir contenido al front-end de los sitios, crear páginas y subpáginas en el lado de administración requiere algunas habilidades básicas de codificación. Ultimate Multisite: Admin Page Creator tiene como objetivo llevar esa misma facilidad de uso del front-end al back-end de tus sitios en la red.

## ¿Por qué usar Admin Page Creator?

Quizás quieras ofrecer una página personalizada en el panel de administración de tus clientes con tutoriales, o tal vez quieras añadir una página personalizada con preguntas frecuentes y un widget de contacto. Las posibilidades son infinitas con Admin Page Creator, y eso es lo que exploraremos en el tutorial de hoy.

_¡Crear una página de tutoriales con una lista de reproducción de YouTube incrustada ahora es posible!_

_Aquí tienes el resultado final en el panel de control del usuario_

## Creando tu primera Página de Administración Personalizada

Después de instalar y activar Ultimate Multisite: Admin Page Creator, ve al menú de administración de la red y dirígete a Páginas de Administración.

_Custom Admin Pages list on the Network Admin_

Verás una lista de todas las páginas de administración que has creado hasta ahora (ya que es tu primera visita, la lista estará vacía). Haz clic en **Agregar nueva Página de Administración**. Serás redirigido al editor de Páginas de Administración.

## Los elementos de una Página de Administración Personalizada

En el editor de Páginas de Administración, tendrás varias opciones diferentes para personalizar tu página.

### Título de la Página

Este es el título que se mostrará en la parte superior de la página, no la etiqueta del menú de administración.

_Introduce un título de página_

### Selector de Tipo de Contenido

Ultimate Multisite: Admin Page Creator te ofrece dos opciones en cuanto al contenido de la página: puedes usar el editor WYSIWYG predeterminado de WordPress o cambiar a un editor HTML.

Si no estás familiarizado con HTML, mantente con el editor predeterminado de WordPress. Podrás editar el contenido de la página como lo harías con una entrada o página normal.

_Usando el editor WYSIWYG predeterminado de WordPress_

_Usando la opción de editor HTML, con resaltado de sintaxis y alertas de error_

## Opciones de Menú

En el lateral, en el meta-caja de Opciones Generales, podrás configurar el tipo de menú, la etiqueta y mucho más.

### Tipos de Menú

Tu nueva página de administración personalizada puede tener dos tipos de menú diferentes: menú de nivel superior y submenú.

Una página de menú de nivel superior, como su nombre indica, añadirá tu página recién creada al menú de administración junto con las páginas predeterminadas de WordPress.

Si seleccionas la opción de nivel superior, también podrás elegir el orden del menú y un icono de menú.

Si deseas que esta página sea un submenú, podrás seleccionar una página principal a la que adjuntar esta página. Puedes añadir tu página personalizada a las páginas predeterminadas de WordPress o a tus páginas de administración personalizadas de nivel superior.

_Personaliza completamente el elemento de menú para esta Página Personalizada_

_También puedes añadir esta página como submenú de otras páginas de menú, incluidas tus propias páginas de nivel superior personalizadas._

## Opciones Avanzadas

Deberías poder agregar todo tipo de contenido a tus páginas de administración personalizadas. Eso incluye reglas y archivos CSS, JavaScript y bibliotecas externas. Puedes hacer eso usando la pestaña Opciones Avanzadas.

### CSS Personalizado

En la pestaña CSS, puedes agregar tu propio código CSS personalizado, así como importar archivos CSS desde sitios remotos (como CDNs, Google Fonts, etc.).

_El CSS personalizado también es compatible. ¡También puedes incluir archivos de estilo externos!_

### JavaScript Personalizado

Lo mismo aplica para JavaScript. Puedes agregar bibliotecas externas para usar en tu código JavaScript personalizado.

_¡Añade tu propio código JavaScript!_

## Permisos

También en las Opciones Avanzadas, podrás seleccionar qué sub-sitios tendrán acceso a esta página. Esto es útil si planeas ofrecer contenido exclusivo diferente a distintos niveles de plan, por ejemplo.

_Tienes control total sobre dónde aparecerá esta página_

**Importante**: Ambas condiciones deben cumplirse para que la página se muestre. Eso significa que si el usuario es cliente de un plan determinado, pero no tiene uno de los roles seleccionados (o viceversa), la página no aparecerá en su panel.

## Activo

Si deseas desactivar una página de administración personalizada sin eliminarla o alterar los permisos, simplemente desmarca la opción Activo.

Desactivar una página de nivel superior desactivará automáticamente todas las subpáginas adjuntas a esa página principal.

_¡El resultado final en el panel del usuario!_
