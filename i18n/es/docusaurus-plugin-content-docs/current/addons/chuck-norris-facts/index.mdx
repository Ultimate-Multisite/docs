# Chuck Norris Facts

Añade un poco de humor a tus tareas diarias de administración de WordPress con el addon **Ultimate Multisite - Chuck Norris Facts**. Cuando esté activado, verás un hecho aleatorio de Chuck Norris mostrado en la esquina superior derecha de cada página de administración.

Perfecto para sacar sonrisas a tu equipo, aligerar el ambiente durante largas sesiones de administración o simplemente celebrar la legendaria genialidad que es Chuck Norris.

## Por qué Chuck Norris Facts?

* **Impulsar la moral del equipo**: Añade un elemento divertido a tu administración de red
* **Romper el hielo**: Excelente tema de conversación para nuevos miembros del equipo
* **Impulso de productividad**: Una risa rápida puede mejorar el enfoque y la productividad
* **Humor atemporal**: Los hechos de Chuck Norris nunca pasan de moda

## Características clave

**110+ Hechos legendarios**  
Una colección completa de chistes y hechos clásicos de Chuck Norris que celebran a la propia leyenda.

**Visualización no intrusiva**  
Los hechos aparecen en la barra de administración de manera no intrusiva que no interfiere con tu flujo de trabajo.

**Listo para traducción**  
Todos los hechos están envueltos en funciones de traducción, lo que facilita la localización a tu idioma o crear hechos personalizados para tu equipo.

**Totalmente responsive**  
Los hechos se muestran correctamente en pantallas de escritorio, tablet y móvil: Chuck Norris es genial en todos los dispositivos.

**Soporte RTL**  
Soporte completo para idiomas de derecha a izquierda porque Chuck Norris trasciende las barreras lingüísticas.

**Compatible con el editor de bloques**  
Los hechos se ocultan inteligentemente en el editor de bloques para evitar conflictos de interfaz.

**Enfoque en accesibilidad**  
Incluye texto adecuado para lectores de pantalla para cumplir con la accesibilidad total.

* Administradores de redes multisite que buscan añadir personalidad

## Instalación

1. Asegúrate de que el plugin Ultimate Multisite esté instalado y activado en tu red de WordPress
2. Sube los archivos del addon a tu directorio `/wp-content/plugins/`
3. Activa el plugin en la red a través del menú 'Plugins' en WordPress
4. ¡Disfruta de hechos aleatorios de Chuck Norris en cada página de administración!

¡No se necesita configuración - simplemente activa y sonríe!

## Preguntas frecuentes

## ¿Requiere Ultimate Multisite?

Aunque está diseñado como un addon de Ultimate Multisite, el plugin funciona de forma independiente y puede ejecutarse en cualquier instalación de WordPress.

## ¿Puedo añadir mis propios hechos?

¡Sí! Los hechos están listos para traducción. Puedes crear tu propio archivo `.po` con hechos personalizados y colocarlo en el directorio `/lang`. O puedes filtrar el array de hechos programáticamente.

## ¿Esto ralentiza mi administración?

¡Para nada! El plugin es extremadamente ligero y no afecta el rendimiento. Sólo muestra texto, sin llamadas a API externas ni procesamiento pesado.

## ¿Puedo desactivarlo en ciertas páginas?

Los hechos se ocultan automáticamente en las páginas donde podrían interferir (como el editor de bloques). También puedes usar ganchos de WordPress para personalizar dónde aparecen los hechos.

## ¿Son los hechos apropiados para todas las audiencias?

¡Sí! Todos los hechos son humor limpio y apto para la familia, celebrando el legendario estatus de Chuck Norris.

## ¿Puedo traducir los hechos a otro idioma?

¡Absolutamente! Todos los hechos utilizan funciones de traducción de WordPress. Crea un archivo de traducción para tu idioma y colócalo en el directorio `/lang`. Dominio de texto: `ultimate-multisite-chuck-norris-facts`

## ¿Funciona con multisite?

¡Sí! Está diseñado específicamente para redes multisite y puede activarse en la red.
