# Acción: switch_network

Se dispara cuando se cambia el contexto de la red actual.

## Parámetros

| Nombre | Tipo | Descripción |
|------|------|-------------|
| $new_network_id | `int` | ID de la red a la que se está cambiando. |
| $old_network_id | `int` | ID de la red que era la actual previamente. |

### Desde

- 1.3.0
