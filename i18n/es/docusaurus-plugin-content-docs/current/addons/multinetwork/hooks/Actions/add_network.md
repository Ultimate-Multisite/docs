# Acción: add_network

Se dispara después de que se haya añadido una nueva red.

## Parameters

| Name | Type | Descripción |
|------|------|-------------|
| $new_network_id | `int` | ID de la red añadida. |
| $r | `array` | Array asociativo completo de argumentos de la red. |
