# Filtro: networks_user_is_network_admin

Filtra las redes en las que un usuario es administrador.

## Parámetros

| Nombre | Tipo | Descripción |
|------|------|-------------|
| $ | `array\|bool` | Lista de IDs de red o falso si no hay redes para el usuario. |
| $ | `int` | ID de usuario para el cual se deben devolver las redes. |
