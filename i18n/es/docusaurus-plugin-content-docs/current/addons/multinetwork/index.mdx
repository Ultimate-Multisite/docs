# Multi-Network

The Ultimate Multisite: Multinetwork addon extends your Multisite Ultimate platform with powerful multinetwork functionality. Este addon permite a sus clientes comprar y gestionar redes completas de WordPress multisite, no solo sitios individuales.

## Key Features

* **Network Provisioning**: Cree y configure automáticamente redes completas de WordPress multisite para sus clientes
* **Network Management**: Capacidades completas de gestión para entornos multinetwork
* **Isolated Networks**: Cada red de cliente opera de forma independiente con sus propias tablas de base de datos y configuración
* **Scalable Architecture**: Maneje múltiples redes de manera eficiente con asignación de recursos optimizada
* **Integration**: Se integra sin problemas con los sistemas de suscripción y facturación de Multisite Ultimate

## Use Cases

* **White Label Hosting**: Proporcione a los clientes sus propias redes multisite con marca propia
* **Agency Solutions**: Permita a las agencias gestionar múltiples redes de clientes desde una sola plataforma
* **Enterprise Deployments**: Apoye a grandes organizaciones con requisitos complejos de multi-red
* **Education Platforms**: Cree redes separadas para diferentes escuelas o departamentos

## Requirements

* Multisite Ultimate 2.0.0 o superior
* Instalación de WordPress Multisite
* PHP 7.4 o superior
* MySQL 5.6 o superior

## Installation

1. Suba los archivos del plugin al directorio `/wp-content/plugins/ultimate-multisite-multinetwork`
2. Active el plugin a través de la pantalla 'Plugins' en WordPress
3. Configure las opciones multinetwork a través de Multisite Ultimate → Settings

## Frequently Asked Questions

## What is multinetwork?

Multinetwork es una característica de WordPress que le permite ejecutar múltiples redes multisite independientes en una sola instalación de WordPress.

## How is this different from regular multisite?

Un multisite regular permite múltiples sitios dentro de una sola red. Multinetwork permite múltiples redes, cada una con sus propios sitios, todas dentro de una sola instalación de WordPress.
