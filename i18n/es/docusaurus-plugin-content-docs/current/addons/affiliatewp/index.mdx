# AffiliateWP

Multisite Ultimate: AffiliateWP Integración

¡Utiliza el potente AffiliateWP para hacer crecer la base de clientes de tu Ultimo Network!

## Instalación

1. Sube 'ultimate-multisite-g ' al directorio '/wp-content/plugins/'
2. Activa el plugin a través del menú 'Plugins' en WordPress
3. Sigue el asistente paso a paso para configurar el plugin

## Guía de Uso

[AffiliateWP](https://affiliatewp.com/) se ha convertido en la solución de referencia al crear un programa de afiliados para tu negocio en línea. Ultimate Multisite: AffiliateWP es el puente que te permitirá crear lo mismo para tu red premium.

## Empezando

En primer lugar, ¡necesitas tener **AffiliateWP** activo! No podrás usar nuestro complemento si no tienes **el plugin que se integra con él**. En caso de que no planees permitir que tus clientes lo utilicen en sus sitios también, AffiliateWP debe estar activo solo en tu sitio principal.

Además, necesitarás que tanto **Ultimate Multisite** como **Ultimate Multisite: AffiliateWP** estén **activados** **en toda la red**.

## Añadiendo Ultimate Multisite como una Integración de AffiliateWP

En tu sitio principal, ve a Configuración de Afiliados y, en la pestaña Integraciones, marca Ultimate Multisite. Esto indicará a [AffiliateWP](https://affiliatewp.com/) que comience a rastrear visitas y conversiones del flujo de registro de Ultimate Multisite.

<!-- Pantalla no disponible: pestaña de configuración de Integraciones de AffiliateWP con casilla de verificación Ultimate Multisite -->

_Marca la casilla Ultimate Multisite en la parte inferior y guarda_

Eso es prácticamente todo lo que se necesita. Las visitas provenientes de enlaces de referencia se rastrearán y cuando ese usuario realice su primer pago, se creará la conversión y se reservará una comisión para el afiliado responsable de la conversión.

Sin embargo, solo se rastreará el primer pago recibido del usuario recién creado. Para habilitar pagos recurrentes de referencia, necesitarás el [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## Habilitando Referencias Recurrentes

Después de instalar el [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/), ve a la pestaña Configuración de Afiliados Referencias Recurrentes y habilita el uso de referencias recurrentes.

<!-- Pantalla no disponible: configuración de Referencias Recurrentes de AffiliateWP con interruptor Habilitar Referencias Recurrentes -->

_Tan simple como se puede: habilita Referencias Recurrentes y ya está listo!_

¡De nuevo, eso es todo lo que se necesita!

Ahora, cada vez que se reciba un nuevo pago en una cuenta referida, el afiliado responsable obtendrá su propia comisión automáticamente!
