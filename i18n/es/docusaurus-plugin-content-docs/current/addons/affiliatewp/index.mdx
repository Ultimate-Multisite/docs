# AffiliateWP

Multisite Ultimate: Integración de AffiliateWP

¡Utiliza el potente AffiliateWP para hacer crecer la base de clientes de tu Ultimo Network!

## Instalación

1. Sube 'ultimate-multisite-git' al directorio '/wp-content/plugins/'
2. Activa el plugin a través del menú 'Plugins' en WordPress
3. Sigue el asistente paso a paso para configurar el plugin

## Guía de Uso

[AffiliateWP](https://affiliatewp.com/) se ha convertido en la solución de referencia al crear un programa de afiliados para tu negocio en línea. Ultimate Multisite: AffiliateWP es el puente que te permitirá crear lo mismo para tu red premium.

## Empezando

En primer lugar, ¡**necesitas tener** [**AffiliateWP**](https://affiliatewp.com/) **activo**! No podrás usar nuestro complemento si no tienes **el plugin que se integra con él**. En caso de que no planees que tus clientes lo utilicen en sus sitios también, AffiliateWP debe estar activo solo en tu sitio principal.

Además, necesitarás tanto **Ultimate Multisite** como **Ultimate Multisite: AffiliateWP** **activados** **en toda la red**.

## Añadiendo Ultimate Multisite como una Integración de AffiliateWP

En tu sitio principal, ve a Ajustes de Afiliados, y en la pestaña Integraciones, marca Ultimate Multisite. Esto indicará a [AffiliateWP](https://affiliatewp.com/) que comience a rastrear visitas y conversiones del flujo de registro de Ultimate Multisite.

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-602126536867724dfc6f0a54-zkWh3TFms-1F9087E6-928C-4043-94B9-F85966AF05DA.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-602126536867724dfc6f0a54-zkWh3TFms-1F9087E6-928C-4043-94B9-F85966AF05DA.png)

_Marca la casilla Ultimate Multisite en la parte inferior y guarda_

Eso es prácticamente todo lo que se necesita. Las visitas provenientes de enlaces de referencia se rastrearán y cuando ese usuario realice su primer pago, se creará la conversión y se reservará una comisión al afiliado responsable de la conversión.

Sin embargo, solo se rastreará el primer pago recibido del usuario recién creado. Para habilitar pagos recurrentes de referencia, necesitarás el [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/).

## Habilitando Referencias Recurrentes

Después de instalar el [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/), ve a la pestaña Ajustes de Afiliados Referencias Recurrentes y habilita el uso de referencias recurrentes.

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-602126536867724dfc6f0a54-bmSN5h7Lk-4697DC4E-ABB9-4C34-BFD8-97E4B62C261C.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-602126536867724dfc6f0a54-bmSN5h7Lk-4697DC4E-ABB9-4C34-BFD8-97E4B62C261C.png)

_Tan simple como parece: habilita Referencias Recurrentes y ya está listo!_

¡De nuevo, eso es todo lo que se necesita!

Ahora, cada vez que se reciba un nuevo pago en una cuenta referida, el afiliado responsable obtendrá su propia comisión automáticamente!
