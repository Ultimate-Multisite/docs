# Tickets de Soporte

**Tickets de Soporte** es un complemento potente para Ultimate Multisite que añade un sistema completo de tickets de soporte a tu red de WordPress. Ideal para plataformas SaaS, proveedores WaaS (Website as a Service) y redes multisite que necesiten ofrecer soporte al cliente tanto a nivel de red como de sitio individual.

## ¿Por qué elegir Tickets de Soporte?

* **Soporte multinivel**: Los administradores de la red gestionan tickets a nivel de red mientras que los propietarios de los sitios manejan su propio soporte al cliente
* **Flujo de trabajo profesional**: Gestión completa del ciclo de vida de los tickets con prioridades, estados y asignaciones
* **Experiencia del cliente**: Interfaz atractiva y responsive que facilita a los clientes obtener ayuda
* **Seguro y confiable**: Construido con las mejores prácticas de seguridad de WordPress y rendimiento optimizado de la base de datos
* **Integración perfecta**: Funciona perfectamente con la gestión de clientes y sitios de Ultimate Multisite

## Características Clave

**Gestión completa de tickets**

* Sistema único de numeración de tickets para referencia sencilla
* Niveles de prioridad configurables (Baja, Normal, Alta, Crítica)
* Flujo de trabajo de estados completo (Abierto, En progreso, En espera, Resuelto, Cerrado)
* Categorización de tickets (General, Técnico, Facturación, Solicitud de función, Informe de error)
* Asignar tickets a agentes de soporte con asignación automática opcional

**Conversaciones en hilo**

* Hilo completo de conversación para respuestas de tickets
* Estilo distinto para respuestas de personal y clientes
* Notas privadas del personal que los clientes no pueden ver
* Interfaz en tiempo real impulsada por AJAX
* Editor de texto moderno con opciones de formato

**Adjuntos de archivos**

* Subida segura de archivos con soporte de arrastrar y soltar
* Restricciones configurables de tipo y tamaño de archivo
* Seguridad avanzada con detección de archivos maliciosos
* Descargas protegidas con controles de acceso

**Notificaciones inteligentes**

* Alertas por correo electrónico automáticas para nuevos tickets, respuestas y cambios de estado
* Plantillas de correo electrónico personalizables
* Notificaciones basadas en roles para personal y clientes
* Variables dinámicas de plantilla para mensajes personalizados

**Interfaz administrativa potente**

* Lista avanzada de tickets con ordenación y filtrado
* Edición rápida de estado, prioridad y asignaciones
* Acciones masivas para gestión eficiente
* Panel de estadísticas con métricas de rendimiento
* Panel de configuración completo

**Portal de cliente atractivo**

* Interfaz limpia y responsive en todos los dispositivos
* Diseño mobile-first
* Búsqueda y filtrado avanzados
* Editor moderno con barra de herramientas de formato
* Formularios de envío de tickets fáciles de usar

**Integración flexible**

* Utilizar shortcodes o elementos UI de Ultimate Multisite
* Mostrar formularios y listas de tickets en cualquier lugar de tu sitio
* Funciona sin problemas con la estructura de red multisite
* Extensible con hooks y filtros

## Instalación

1. Asegúrate de que el plugin Ultimate Multisite esté instalado y activado en tu red de WordPress
2. Sube los archivos del complemento a tu directorio `/wp-content/plugins/`
3. Activa el plugin a nivel de red a través del menú 'Plugins' en WordPress
4. Navega a Network Admin > Support Tickets para configurar tus ajustes

## Preguntas frecuentes

## ¿Esto requiere Ultimate Multisite?

Sí, este complemento requiere que Ultimate Multisite esté instalado y activo. Se integra profundamente con las funciones de gestión de clientes y sitios de Ultimate Multisite.

## ¿Pueden los propietarios de sitios manejar su propio soporte al cliente?

¡Absolutamente! Los propietarios de sitios pueden usar el sistema de tickets para brindar soporte a sus propios clientes, mientras que los administradores de red pueden manejar tickets de soporte a nivel de red.

## ¿Cómo envían los clientes tickets?

Puedes agregar formularios de envío de tickets en cualquier lugar de tu sitio usando shortcodes o elementos UI de Ultimate Multisite. Los clientes simplemente completan el formulario y envían su solicitud.

## ¿Puedo personalizar los estados y prioridades de los tickets?

¡Sí! Puedes personalizar completamente los niveles de prioridad, estados y tipos de tickets en la configuración del administrador de la red para que coincidan con tu flujo de trabajo de soporte.

## ¿Soporta adjuntos de archivos?

Sí, tanto los clientes como el personal pueden adjuntar archivos a tickets y respuestas. Tienes control total sobre los tipos de archivo permitidos y los límites de tamaño por razones de seguridad.

## ¿Cómo se manejan las notificaciones?

El sistema envía automáticamente notificaciones por correo electrónico para nuevos tickets, respuestas y cambios de estado. Puedes personalizar todas las plantillas de correo electrónico en la configuración.

## ¿Puedo asignar tickets a agentes de soporte específicos?

Sí, puedes asignar manualmente tickets a agentes o habilitar la asignación automática para distribuir tickets automáticamente.

## ¿Hay notas privadas para el personal?

Sí, los miembros del personal pueden agregar respuestas privadas que solo son visibles para otros miembros del personal, no para los clientes.

## ¿Es móvil-amigable?

¡Absolutamente! Toda la interfaz está construida con un enfoque mobile-first y funciona maravillosamente en todos los dispositivos.

## ¿Puedo mostrar listas y formularios de tickets usando shortcodes?

¡Sí! Shortcodes disponibles:

* `[wu_support_tickets]` - Mostrar lista de tickets de clientes
* `[wu_support_ticket_form]` - Mostrar formulario de envío de nuevo ticket
* `[wu_support_ticket_view ticket_id="123"]` - Mostrar ticket específico

## ¿Qué tipos de archivos se pueden subir?

Puedes configurar los tipos de archivo permitidos en la configuración. El sistema incluye escaneo de seguridad avanzado para proteger contra archivos maliciosos.

## ¿Puedo exportar datos de tickets?

Sí, el complemento incluye capacidades de exportación para datos y estadísticas de tickets.
