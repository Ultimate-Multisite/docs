# Integración con WooCommerce

La Integración con WooCommerce funciona de manera muy similar a nuestro Manual Gateway. Ofrece mucha flexibilidad a tus usuarios, permitiéndoles elegir cualquiera de los formularios de pago disponibles en tu instalación de WooCommerce. Y es simple: una vez que termina el período de facturación del usuario, la suscripción del usuario se pone en espera. Al mismo tiempo, se creará un pedido en la instalación de WooCommerce del sitio principal de la red. El usuario luego recibe una factura en su correo electrónico, con un enlace para realizar el pago. Una vez recibido el pago, la suscripción se renueva.

## Features

- Integración sencilla entre Ultimate Multisite y WooCommerce
- Varias opciones de pago disponibles
- Proceso completamente automatizado con WooCommerce Subscriptions

## Usage Guide

_**NOTA IMPORTANTE: Este artículo se refiere a la versión 2.x de Ultimate Multisite.**_

_**ATENCIÓN:** Ultimate Multisite: WooCommerce Integration requiere que WooCommerce esté activado al menos en tu sitio principal._

Entendemos que _Stripe_ y _PayPal_ no están disponibles en algunos países, lo que limita o dificulta a los usuarios de Ultimate Multisite usar eficazmente nuestro plugin. Por eso creamos un **add-on para integrar _WooCommerce_**.

_WooCommerce_ es un plugin de eCommerce muy popular. Desarrolladores de todo el mundo crearon add-ons para integrar diferentes pasarelas de pago. Aprovechamos esto para ampliar las pasarelas de pago que puedes usar en el sistema de facturación de Ultimate Multisite.

## Setting It Up

Primero, necesitas instalar el add-on WooCommerce Integration, que es **gratuito para usuarios con licencia**. Para instalarlo, haz clic en Ultimate Multisite en tu panel de super administrador y selecciona **Add-ons**.

![Página de Add-ons](/img/admin/addons-page.png)

Aquí puedes encontrar todos los add-ons de Ultimate Multisite. Haz clic en el add-on **Ultimate Multisite: WooCommerce Integration**.

![Página de Add-ons mostrando la Integración con WooCommerce](/img/admin/addons-page.png)

Se abrirá una ventana con los detalles del add-on. Simplemente haz clic en **Install Now** *.*

<!-- Screenshot unavailable: WooCommerce Integration addon install popup with Install Now button -->

Una vez finalizada la instalación, serás redirigido a la página de plugins. Aquí, simplemente haz clic en **Network Activate** y el add-on _WooCommerce_ se activará en tu red.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for WooCommerce Integration -->

Después de activarlo, si aún no tienes instalado y activado el plugin WooCommerce en tu sitio web, **recibirás un recordatorio**.

<!-- Screenshot unavailable: Admin notice reminding to install and activate WooCommerce plugin -->

Una vez instalado, verás una opción adicional bajo la configuración de pagos de Ultimate Multisite. Luego puedes **habilitar** la opción _WooCommerce_ desde allí.

![Configuración de la pasarela de pago](/img/config/settings-payment-gateways.png)

Puedes configurar algunos ajustes básicos como el nombre que se mostrará en la interfaz y las instrucciones para guiar a tus usuarios en su flujo de pago. Luego puedes ir a la configuración de **_WooCommerce_ del sitio principal** para habilitar y configurar la pasarela de pago que prefieras usar.

## How to manage the payment

La Integración con WooCommerce funciona de manera muy similar a nuestro **Manual Gateway**. La desventaja es que cada pago debe realizarse manualmente al final del período de facturación. Así es como funciona el flujo:

* Cuando termina el ciclo de facturación del usuario, la membresía se pone **en espera**. Al mismo tiempo, se creará un pedido en la instalación de _WooCommerce_ del sitio principal de tu red.
* Los usuarios reciben una factura en su correo electrónico, con un enlace para realizar el pago (el enlace también aparece en su historial de facturación desde el panel de su subsitio). Ese enlace redirige al usuario al **checkout** de _**WooCommerce**_, donde podrán seleccionar una de las **pasarelas _WooCommerce_ disponibles** para procesar su pago.
* Una vez recibido el pago, la membresía se **renueva**, el pago se registra en Ultimate Multisite y el pedido se marca como completado.

### Algunas notas importantes:

* La funcionalidad predeterminada de Ultimate Multisite sigue funcionando con este formulario de integración. Por ejemplo, **emitir reembolsos** desde la pantalla **Subscription Management** se comunicará con WooCommerce para emitir el reembolso usando el formulario de pago seleccionado.
* Ultimate Multisite: WooCommerce Integration depende de WP-Cron para funcionar. Esto funciona muy bien la mayoría de las veces, pero si el sitio tiene poco tráfico, el cron no se activa para comprobar eventos programados y el evento se pierde, lo que puede causar retrasos en la ejecución.

## Woocommerce Subscriptions Plugin

Iniciar manualmente un pago cada mes no es ideal para los usuarios finales, ya que se espera que una suscripción se renueve automáticamente cada ciclo de facturación. Por eso diseñamos el add-on WooCommerce Integration para **soportar el plugin WooCommerce Subscriptions**. Esto automatizará el pago para facilitar la transacción a tus usuarios finales.

Todo lo que necesitas hacer es **instalar** el [plugin Woocommerce Subscriptions](https://woocommerce.com/products/woocommerce-subscriptions/) y tenerlo **activado en tu sitio principal**. Luego vuelve a la configuración de pagos de Ultimate Multisite y **habilita la integración**.

Y eso es todo. Tus usuarios finales ya no necesitarán realizar un pago manual al final de su ciclo de facturación.

Además de que se cree un pedido en tu sitio principal bajo _Woocommerce_, también se creará una suscripción para cada cuenta.
