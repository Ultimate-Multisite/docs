# Superdav AI Agent

Superdav AI Agent es un asistente de IA avanzado para WordPress que amplía Gratis AI Agent con habilidades conscientes del tema, capacidades mejoradas de búsqueda en internet, funciones de constructor de sitios e integraciones de automatización opcionales. La versión 1.18.0 añade herramientas de lectura de Google Calendar, mapeo de contactos y asistentes, puertas de aprobación humana, deduplicación de recordatorios, notificaciones SMS de TextBee, aprovisionamiento del servicio gestionado Superdav y un paquete complementario Advanced distribuido por separado para flujos de trabajo de desarrolladores de confianza.

## Funciones clave {#key-features}

- **Habilidades conscientes del tema** — adapta automáticamente la orientación y las capacidades según tu tema activo de WordPress y los plugins instalados
- **Múltiples proveedores de búsqueda en internet** — elige entre Brave Search y Tavily para obtener resultados de búsqueda más completos y capacidades avanzadas de investigación
- **Formularios de contacto del constructor de sitios** — crea formularios de contacto totalmente funcionales directamente desde la interfaz de chat usando tu plugin de formularios instalado
- **Constructor de sitios mejorado** — planes de creación de sitios multifase mejorados con mejor detección de plugins y recuperación de errores
- **Habilidades de tema integradas** — orientación especializada para Block Themes, Classic Themes, Kadence Blocks y Kadence Theme
- **Integración mejorada con WooCommerce** — refactorizada para usar APIs nativas de WooCommerce para una mejor fiabilidad y compatibilidad
- **Detección automática de proveedores** — detecta y actualiza automáticamente los plugins y temas disponibles cuando se activan o desactivan
- **Generación de logotipos personalizados** — genera e inserta logotipos SVG personalizados con sanitización automática segura para espacios de nombres
- **Validación de contraste de paleta** — comprueba pares de colores para cumplir con la accesibilidad WCAG antes de aplicarlos a tu tema
- **Descubrimiento mejorado con fotos** — sube fotos durante la entrevista de descubrimiento de Theme Builder para influir en la dirección del diseño
- **Menús de hostelería** — genera páginas estructuradas de menús de comida y bebida para restaurantes, cafeterías y negocios de hostelería
- **Vistas previas de escritorio y móvil** — previsualiza tu diseño en dispositivos de escritorio y móviles durante la selección de la dirección de diseño
- **Extracción de sitios Tier 1** — sd-ai-agent/site-scrape ahora está disponible de forma predeterminada para analizar sitios web de competidores e inspiración
- **Herramientas de lectura de Google Calendar** — lee calendarios y eventos configurados para que las automatizaciones puedan razonar sobre el contexto de la programación próxima
- **Mapeo de contactos y asistentes** — relaciona participantes del calendario con usuarios y contactos conocidos del sitio antes de generar trabajo de seguimiento
- **Puertas de aprobación humana** — pausa automatizaciones sensibles para revisión antes de enviar mensajes, reanudar trabajo o realizar cambios
- **Registros de recordatorios y deduplicación** — rastrea los recordatorios que ya se han enviado para que las ejecuciones repetidas de automatización no notifiquen dos veces al mismo destinatario
- **Notificaciones SMS de TextBee** — envía notificaciones configuradas por mensaje de texto a través de TextBee cuando un proveedor de SMS está habilitado
- **Aprovisionamiento del servicio gestionado Superdav** — conecta sitios compatibles a endpoints de Superdav alojados con configuración automática en lugar de configuración manual de endpoints
- **Paquete complementario Advanced** — herramientas para desarrolladores de confianza distribuidas por separado para flujos de trabajo de sistema de archivos, base de datos, WP-CLI, despachador REST, constructor de plugins, instantánea git, gestión de usuarios y benchmarks

## Requisitos {#requirements}

- WordPress 6.0 o superior
- PHP 8.1 o superior
- WordPress Multisite (activado en red) o instalación de sitio único
- Un plugin conector de proveedor de IA activo (Anthropic, OpenAI o compatible)
- Gratis AI Agent v1.9.0 o superior (Superdav AI Agent amplía Gratis AI Agent)

## Instalación {#installation}

1. Sube los archivos del plugin a `/wp-content/plugins/superdav-ai-agent`
2. Activa el plugin mediante la pantalla **Plugins** en WordPress
3. Asegúrate de que Gratis AI Agent también esté activado
4. Ve a **Gratis AI Agent → Settings** y configura tu proveedor de búsqueda en internet, conexión al servicio gestionado Superdav, Google Calendar o credenciales de TextBee según sea necesario

El paquete de WordPress.org contiene las funciones principales de Superdav AI Agent. El paquete complementario Advanced se compila y distribuye por separado para entornos donde las herramientas de desarrollador de confianza se instalan y autorizan explícitamente.

## Habilidades conscientes del tema {#theme-aware-skills}

Superdav AI Agent detecta automáticamente tu tema activo y los plugins instalados, cargando la habilidad consciente del tema adecuada en el contexto del asistente. Las habilidades disponibles incluyen:

| Habilidad | Se aplica a |
|---|---|
| **Block Themes** | Temas que usan la arquitectura basada en bloques de WordPress (edición completa del sitio) |
| **Classic Themes** | Temas tradicionales de WordPress que usan plantillas PHP y el editor clásico |
| **Kadence Blocks** | Sitios con el plugin Kadence Blocks activo |
| **Kadence Theme** | Sitios con el tema Kadence como tema activo |

Consulta [Habilidades conscientes del tema](../../user-guide/configuration/theme-aware-skills) para obtener orientación detallada sobre cada habilidad.

## Proveedores de búsqueda en internet {#internet-search-providers}

Superdav AI Agent v1.10.0 añade **Tavily** como alternativa a Brave Search para consultas de búsqueda en internet. Ambos proveedores ofrecen resultados de búsqueda en tiempo real y se pueden configurar en **Gratis AI Agent → Settings → Advanced**.

| Proveedor | Funciones | Configuración |
|---|---|---|
| **Brave Search** | Resultados de búsqueda rápidos y centrados en la privacidad | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Resultados de búsqueda más completos, capacidades de investigación avanzadas | [Tavily API](https://tavily.com/) |

Consulta [Búsqueda en Internet](../../user-guide/configuration/internet-search) para obtener instrucciones de configuración.

## Flujos de trabajo de calendario, aprobación y notificaciones {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 amplía los flujos de trabajo de automatización más allá de las tareas de contenido y creación de sitios:

- Las **herramientas de Google Calendar** permiten que el agente lea calendarios configurados y detalles de eventos cuando las credenciales de calendario están disponibles.
- Los **asistentes de contactos y asistentes a eventos** asignan participantes de eventos a usuarios de WordPress o contactos conocidos antes de preparar recordatorios o tareas de seguimiento.
- Las **puertas de aprobación humana** permiten que una automatización se pause hasta que un usuario autorizado revise la acción propuesta.
- Los **registros de recordatorios** evitan recordatorios duplicados de calendario o notificación cuando una automatización se reintenta o se programa repetidamente.
- La **compatibilidad con SMS de TextBee** habilita notificaciones por mensaje de texto cuando el proveedor TextBee está configurado.

Usa estas herramientas para asistencia con conocimiento de agenda, seguimiento de asistentes y flujos de trabajo de recordatorios que aún requieren revisión humana antes de enviar mensajes sensibles.

## Formularios de contacto de Site Builder {#site-builder-contact-forms}

Simplemente pide a Site Builder que cree un formulario de contacto, y detectará automáticamente tu plugin de formularios instalado (Contact Form 7, WPForms, Fluent Forms o Gravity Forms) y generará un formulario listo para usar.

Consulta [Formularios de contacto de Site Builder](../../user-guide/administration/site-builder-contact-forms) para obtener orientación detallada.

## Configuración {#configuration}

Todos los ajustes están en **Gratis AI Agent → Ajustes** en la administración de WordPress. Superdav AI Agent añade las siguientes opciones de configuración:

| Ajuste | Descripción |
|---|---|
| **Proveedor de búsqueda en Internet** | Elige entre Brave Search y Tavily (opcional) |
| **Tavily API Key** | API key para la búsqueda en Internet de Tavily (si Tavily está seleccionado) |
| **Managed Superdav Service** | Conecta sitios compatibles al servicio alojado de Superdav con aprovisionamiento automático de endpoints |
| **Credenciales de Google Calendar** | Configura el acceso al calendario para que el agente pueda leer calendarios y eventos |
| **Proveedor de SMS de TextBee** | Configura las credenciales de TextBee para flujos de trabajo aprobados de notificaciones SMS |

## Paquete complementario avanzado {#advanced-companion-package}

Superdav AI Agent v1.18.0 añade un paquete complementario Advanced distribuido por separado para flujos de trabajo de desarrolladores de confianza. Estas capacidades no están incluidas en el paquete core de WordPress.org. Instala el paquete complementario solo en sitios donde los administradores confíen explícitamente en que el agente acceda a herramientas de desarrollador.

Las herramientas Advanced incluyen operaciones del sistema de archivos, acceso a bases de datos, WP-CLI, flujos de trabajo de despachador REST, utilidades de creación de plugins, instantáneas git, asistentes de gestión de usuarios y herramientas de benchmark. Mantén estas herramientas limitadas a administradores de confianza y revisa la configuración de puertas de aprobación antes de habilitarlas en sitios de producción.

- **Compatibilidad con Google Calendar** — lee calendarios y eventos configurados para automatizaciones con conocimiento de agenda
- **Asignación de contactos y asistentes** — relaciona participantes del calendario con usuarios y contactos del sitio

- **Distribución de complemento avanzado** — publica herramientas para desarrolladores de confianza por separado de la compilación core de WordPress.org
- **Correcciones de fiabilidad** — mejora la selección de modelos, los tiempos de espera de solicitudes, el manejo del modelo de configuración predeterminado, el manejo de texto de razonamiento, la orientación de reintentos, el endurecimiento del token de calendario, el endurecimiento de recordatorios, los lanzamientos de onboarding, las reanudaciones de capacidades confirmadas y los problemas de revisión de empaquetado de WordPress.org

## Mejoras en v1.16.0 {#improvements-in-v1160}

- **Herramienta de raspado de sitio de Nivel 1** — sd-ai-agent/site-scrape ahora está disponible de forma predeterminada en Theme Builder para analizar sitios web de competidores e inspiración

- **Integración con WooCommerce refactorizada** — ahora usa APIs nativas de WooCommerce para mayor fiabilidad y compatibilidad
- **Actualización automática de la lista de proveedores** — detecta automáticamente cuándo se activan o desactivan extensiones y actualiza la lista de proveedores disponibles
- **Constructor de sitios mejorado** — planes de construcción de sitios multifase mejorados con mejor recuperación de errores
- **Habilidades adaptadas al tema** — cuatro nuevas habilidades integradas que se adaptan automáticamente a tu tema activo

## Correcciones de errores en v1.10.0 {#bug-fixes-in-v1100}

- Corregida la habilidad `navigate-to` que causaba bucles de recarga infinitos en algunas páginas de administración
- Corregida la habilidad `list-posts` que no resolvía correctamente los nombres de categorías y etiquetas a slugs
- Corregidos comandos WP-CLI a los que les faltaban alias de namespace después de una refactorización anterior
- Corregida la automatización de eventos para manejar correctamente sitios donde las tablas de automatización aún no se han creado
- Corregida la habilidad `memory-save` para usar el prefijo de namespace correcto en el constructor de instrucciones del sistema
- Corregidos resultados escalares de herramientas para que se envolvieran correctamente antes de devolverse a la AI
- Corregidas estadísticas de uso para manejar correctamente el formato de clave de habilidad heredado al actualizar desde versiones anteriores
