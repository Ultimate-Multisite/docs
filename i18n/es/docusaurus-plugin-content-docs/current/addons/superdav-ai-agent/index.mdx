# Superdav AI Agent

Superdav AI Agent es un asistente de IA avanzado para WordPress que mejora Gratis AI Agent añadiéndole habilidades que entienden el tema, capacidades mejoradas de búsqueda en internet y funciones optimizadas para el constructor de sitios. La versión 1.16.0 añade generación de logos personalizados, validación de contraste de paletas, carga de fotos en la entrevista de descubrimiento, generación de menús de hostelería estructurados y renderizado de previsualización para escritorio/móvil en Theme Builder.

## Características Clave

- **Habilidades conscientes del tema** — adapta automáticamente la guía y las capacidades basándose en el tema activo de WordPress y los plugins instalados
- **Múltiples proveedores de búsqueda en internet** — elige entre Brave Search y Tavily para obtener resultados de búsqueda más ricos y capacidades de investigación avanzadas
- **Formularios de contacto del constructor de sitios** — crea formularios de contacto totalmente funcionales directamente desde la interfaz de chat usando tu plugin de formularios instalado
- **Constructor de sitios mejorado** — planes de construcción de sitios multifase mejorados con mejor descubrimiento de plugins y recuperación de errores
- **Habilidades integradas del tema** — guía especializada para Block Themes, Classic Themes, Kadence Blocks y Kadence Theme
- **Integración mejorada con WooCommerce** — refactorizada para usar las APIs nativas de WooCommerce, lo que mejora la fiabilidad y la compatibilidad
- **Detección automática de proveedores** — detecta y actualiza automáticamente los plugins y temas disponibles cuando se activan o desactivan
- **Generación de logos personalizados** — genera e incrusta logos SVG personalizados con sanitización automática segura del espacio de nombres
- **Validación de contraste de paletas** — comprueba los pares de colores para el cumplimiento de la accesibilidad WCAG antes de aplicarlos a tu tema
- **Descubrimiento mejorado con fotos** — carga fotos durante la entrevista de descubrimiento de Theme Builder para influir en la dirección del diseño
- **Menús de hostelería** — genera páginas de menús de comida y bebida estructurados para restaurantes, cafeterías y negocios de hostelería
- **Previsualizaciones de escritorio y móvil** — previsualiza tu diseño en dispositivos de escritorio y móvil durante la selección de la dirección del diseño
- **Raspado de sitios de Nivel 1** — sd-ai-agent/site-scrape ahora está disponible por defecto para analizar sitios web competidores e inspiración

## Requisitos

- WordPress 6.0 o superior
- PHP 8.1 o superior
- WordPress Multisite (activado a nivel de red) o instalación de sitio único
- Un plugin con conector de proveedor de IA activo (Anthropic, OpenAI o compatible)
- Gratis AI Agent v1.9.0 o superior (Superdav AI Agent extiende Gratis AI Agent)

## Instalación

1. Sube los archivos del plugin a `/wp-content/plugins/superdav-ai-agent`
2. Activa el plugin desde la pantalla **Plugins** de WordPress
3. Asegúrate de que Gratis AI Agent también esté activado
4. Navega a **Gratis AI Agent → Settings** y configura tu proveedor de búsqueda en internet (opcional)

## Inicio Rápido

Una vez activado, Superdav AI Agent mejora automáticamente tu instalación de Gratis AI Agent con habilidades conscientes del tema y capacidades adicionales. Abre el panel de **Gratis AI Agent** en la barra lateral del administrador de WordPress y comienza a usar las nuevas funciones.

**Ejemplos de instrucciones:**

```
Add a contact form to the Contact page.
```

```
Search the web for the latest WordPress security best practices.
```

```
Create a hero section using the active theme's block patterns.
```

## Habilidades Conscientes del Tema

Superdav AI Agent detecta automáticamente tu tema activo y los plugins instalados, cargando la habilidad consciente del tema apropiada en el contexto del asistente. Las habilidades disponibles incluyen:

| Skill | Applies To |
|---|---|
| **Block Themes** | Temas que utilizan la arquitectura basada en bloques de WordPress (Full Site Editing) |
| **Classic Themes** | Temas tradicionales de WordPress que usan plantillas PHP y el editor clásico |
| **Kadence Blocks** | Sitios con el plugin Kadence Blocks activo |
| **Kadence Theme** | Sitios con el tema Kadence como tema activo |

Consulta [Theme-Aware Skills](../../user-guide/configuration/theme-aware-skills) para obtener una guía detallada sobre cada habilidad.

## Proveedores de Búsqueda en Internet

Superdav AI Agent v1.10.0 añade **Tavily** como alternativa a Brave Search para las consultas de búsqueda en internet. Ambos proveedores ofrecen resultados de búsqueda en tiempo real y se pueden configurar en **Gratis AI Agent → Settings → Advanced**.

| Provider | Features | Setup |
|---|---|---|
| **Brave Search** | Resultados de búsqueda rápidos y centrados en la privacidad | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Resultados de búsqueda más ricos, capacidades de investigación avanzadas | [Tavily API](https://tavily.com/) |

Consulta [Internet Search](../../user-guide/configuration/internet-search) para obtener instrucciones de configuración.

## Formularios de Contacto del Constructor de Sitios

El agente del Constructor de Sitios ahora puede crear formularios de contacto directamente desde la interfaz de chat. Simplemente pídele al Constructor de Sitios que cree un formulario de contacto, y detectará automáticamente tu plugin de formularios instalado (Contact Form 7, WPForms, Fluent Forms o Gravity Forms) y generará un formulario listo para usar.

Consulta [Site Builder Contact Forms](../../user-guide/administration/site-builder-contact-forms) para obtener una guía detallada.

## Funciones del Constructor de Temas

Theme Builder en Superdav AI Agent v1.16.0 incluye potentes nuevas capacidades:

### Entrevista de Descubrimiento con Carga de Fotos

La entrevista de descubrimiento de Theme Builder ahora incluye un paso de carga de fotos donde puedes proporcionar inspiración visual. Sube fotos de tu marca, sitios competidores o inspiración de diseño para ayudar a Theme Builder a entender tu dirección de diseño.

Consulta [Discovery Interview](./theme-builder/discovery-interview.md) para obtener una guía detallada.

### Dirección de Diseño con Previsualizaciones de Escritorio y Móvil

Revisa tu diseño tanto en dispositivos de escritorio como móviles antes de que Theme Builder genere tu tema completo. Consulta previsualizaciones en tiempo real en múltiples puntos de ruptura responsivos.

Consulta [Design Direction](./theme-builder/design-direction.md) para obtener una guía detallada.

### Menús de Hostelería

Theme Builder ahora puede generar páginas de menús de comida y bebida estructurados para restaurantes, cafeterías, bares y otros negocios de hostelería. Los menús incluyen precios, descripciones, información dietética y advertencias de alérgenos.

Consulta [Hospitality Menus](./theme-builder/hospitality-menus.md) para obtener una guía detallada.

### Nuevas Habilidades

- **[Generate Logo SVG](./abilities/generate-logo-svg.md)** — crea logos SVG personalizados con sanitización automática
- **[Validate Palette Contrast](./abilities/validate-palette-contrast.md)** — comprueba los pares de colores para el cumplimiento WCAG
- **[Create Menu](./abilities/create-menu.md)** — crea menús de navegación con etiquetas de navegación distintas

Consulta [Tools and Configuration](./theme-builder/tools.md) para obtener información sobre las herramientas disponibles y la herramienta de raspado de sitios de Nivel 1.

## Configuración

Todas las configuraciones se encuentran en **Gratis AI Agent → Settings** en el administrador de WordPress. Superdav AI Agent añade las siguientes opciones de configuración:

| Setting | Description |
|---|---|
| **Internet Search Provider** | Elige entre Brave Search y Tavily (opcional) |
| **Tavily API Key** | Clave API para la búsqueda en internet de Tavily (si se selecciona Tavily) |

## Mejoras en v1.16.0

- **Cache del cliente de IA** — ahora respaldado por transients para persistencia entre solicitudes, evitando la pérdida de datos en tareas de agente de larga duración
- **Herramienta de raspado de sitios de Nivel 1** — sd-ai-agent/site-scrape ahora está disponible por defecto en Theme Builder para analizar sitios web competidores e inspiración
- **Enlaces de acción de filas de plugins** — corregidos y renombrados para mayor claridad

## Mejoras en v1.10.0

- **Integración de WooCommerce refactorizada** — ahora utiliza las APIs nativas de WooCommerce para una mejor fiabilidad y compatibilidad
- **Actualización automática de la lista de proveedores** — detecta automáticamente cuándo se activan o desactivan los plugins y actualiza la lista de proveedores disponibles
- **Constructor de sitios mejorado** — planes de construcción de sitios multifase mejorados con mejor recuperación de errores
- **Habilidades conscientes del tema** — cuatro nuevas habilidades integradas que se adaptan automáticamente a tu tema activo

## Corrección de Errores en v1.10.0

- Corregida la habilidad `navigate-to` que causaba bucles de recarga infinitos en algunas páginas de administración
- Corregida la habilidad `list-posts` que no resolvía correctamente los nombres de categorías y etiquetas a slugs
- Corregidos los comandos WP-CLI que carecían de alias de espacio de nombres después de una refactorización anterior
- Corregida la gestión de eventos para que maneje con normalidad los sitios donde las tablas de automatización aún no han sido creadas
- Corregida la habilidad `memory-save` para usar el prefijo de espacio de nombres correcto en el constructor de instrucciones del sistema
- Corregidos los resultados de herramientas escalares para que se envuelvan correctamente antes de ser devueltos a la IA
- Corregida la estadística de uso para manejar correctamente el formato antiguo de clave de habilidad al actualizar desde versiones anteriores
