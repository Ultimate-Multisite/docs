# Acción: wp_ultimo_ptm_edit_page_after_title_input

Permite a los desarrolladores de plugins agregar información adicional debajo de la entrada de texto

## Parámetros

| Nombre | Tipo | Descripción |
|--------|------|-------------|
| $ | `object` | Objeto que contiene la información |
| $ | `\WU_Page` | Instancia de Multisite Ultimate Page |
