# PayFast

Multisite Ultimate: Payfast Gateway

Añade Payfast - una pasarela de pagos en línea rápida y segura en Sudáfrica - como una opción de pago de Multisite Ultimate.

## Installation

1. Upload 'ultimate-multisite-payfast' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Sigue el asistente paso a paso para configurar el plugin

## Usage Guide

PayFast es un procesador de pagos en línea líder para negocios en Sudáfrica. Está dedicado a ayudar a los comerciantes en línea a crecer haciendo que las transacciones sean fluidas y seguras. Según la solicitud de un usuario de Ultimate Multisite, decidimos crear un addon que permita la integración con PayFast. Este artículo muestra cómo configurar PayFast como tu pasarela de pagos para recibir pagos en tu red de Ultimate Multisite.

El primer paso es instalar el addon.

# How to install the Payfast add-on:

Ve a la página de addons y selecciona el addon Ultimate Multisite: Payfast Gateway.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-BmLWqj4yjt.png)

La ventana del addon aparecerá. Haz clic en Install Now.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-1jpCakOHNy.png)

Una vez finalizada la instalación, serás redirigido a la página de plugins. Aquí, simplemente haz clic en Network Activate y el addon _Payfast_ se activará en tu red.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-49OQHBwPxk.png)

# Setting Up PayFast

Después de instalar el addon, ahora puedes ir a la configuración de Ultimate Multisite y comenzar a configurar PayFast.

En la página **Ultimate Multisite settings**, puedes encontrar la pestaña "Payments". En esta pestaña, deberías ver "**Payfast**" junto con un botón de alternancia para habilitar o deshabilitar esta pasarela de pagos.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-07bPkaGJka.png)Haz clic en el botón de alternancia para habilitarlo. Se expandirá la caja para mostrar diferentes campos que debes completar, como **Mechant ID, Merchant Key y PassPhrase**. Puedes obtener esta información de tu cuenta **Payfast**.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-pocxO2VPfM.png)

El siguiente paso es iniciar sesión en tu cuenta PayFast para obtener la información que necesitas.

Al iniciar sesión, puedes ver inmediatamente las credenciales que necesitas en el panel de tu cuenta **Payfast**.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-KW0WnsxHIW.png)Puedes copiar y pegar la credencial y completar los detalles en la configuración de Ultimate Multisite. Luego haz clic en el botón **"Save settings"** para guardar los cambios.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-5MIrG21r3e.png)

¡Eso es todo, tu integración de pagos PayFast está completa!
