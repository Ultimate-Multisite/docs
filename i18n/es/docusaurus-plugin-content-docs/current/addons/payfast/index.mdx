# PayFast

Multisite Ultimate: Payfast Gateway

Añade Payfast - una pasarela de pagos en línea rápida y segura en Sudáfrica - como opción de pago en Multisite Ultimate.

## Instalación

1. Sube 'ultimate-multisite-payfast' al directorio '/wp-content/plugins/'
2. Activa el plugin a través del menú 'Plugins' en WordPress
3. Sigue el asistente paso a paso para configurar el plugin

## Guía de uso

PayFast es un procesador de pagos en línea líder para negocios en Sudáfrica. Está dedicado a ayudar a los comerciantes en línea a crecer haciendo que las transacciones sean fluidas y seguras. Según la solicitud de los usuarios de Ultimate Multisite, decidimos crear un complemento que permita la integración con PayFast. Este artículo muestra cómo configurar PayFast como tu pasarela de pagos para recibir pagos en tu red de Ultimate Multisite.

El primer paso es instalar el complemento.

# Cómo instalar el complemento Payfast:

Ve a la página de complementos y selecciona el complemento Ultimate Multisite: Payfast Gateway.

![Página de complementos](/img/admin/addons-page.png)

La ventana del complemento aparecerá. Haz clic en Instalar ahora.

![Página de complementos con opción de instalación](/img/admin/addons-page.png)

Una vez finalizada la instalación, serás redirigido a la página de plugins. Aquí, simplemente haz clic en Activar en red y el complemento _Payfast_ se activará en tu red.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# Configuración de PayFast

Después de instalar el complemento, ahora puedes ir a la configuración de Ultimate Multisite y comenzar a configurar PayFast.

En la página **Ultimate Multisite settings**, puedes encontrar la pestaña 'Payments'. En esta pestaña, deberías ver '**Payfast**' junto con un botón de alternancia para habilitar o deshabilitar esta pasarela de pagos.

![Configuración de la pasarela de pagos](/img/config/settings-payment-gateways.png)

Haz clic en el botón de alternancia para habilitarlo. Se expandirá la caja para mostrar diferentes campos que debes completar, como **Merchant ID, Merchant Key y PassPhrase**. Puedes obtener esta información de tu cuenta **Payfast**.

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

El siguiente paso es iniciar sesión en tu cuenta PayFast para obtener la información que necesitas.

Al iniciar sesión, puedes ver inmediatamente las credenciales que necesitas en el panel de tu cuenta **Payfast**.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->

Puedes copiar y pegar la credencial y completar los detalles en la configuración de Ultimate Multisite. Luego haz clic en el botón **"Save settings"** para guardar los cambios.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

¡Eso es todo, tu integración de pagos PayFast está completa!
