# Mailster

Преобразуйте вашу мультисайт‑сеть в мощную машину для создания списков рассылки с помощью дополнения Ultimate Multisite Mailster Integration. Автоматически добавляйте новых клиентов в списки Mailster, когда они регистрируются или совершают покупки, создавая бесшовную связь между вашей платформой членства и стратегией email‑маркетинга.

## Key Features

**Автоматическая синхронизация подписчиков**
* Автоматически добавлять клиентов в списки Mailster при регистрации
* Выбрать синхронизацию сразу при регистрации или дождаться подтверждения оплаты
* Поддерживать списки рассылки свежими и актуальными без ручного вмешательства

**Гибкое управление списками**
* Установить глобальные списки по умолчанию для всех новых клиентов
* Перезаписать значения по умолчанию списками, специфичными для продукта, для целевых кампаний
* Легко управлять назначением списков через знакомый интерфейс Mailster

**Опции согласия, соответствующие GDPR**
* Выбрать между автоматической подпиской или согласием через флажок
* Включить двойное подтверждение для подтверждённых подписок
* Показать/скрыть флажок согласия в зависимости от ваших требований к соответствию

**Умное сопоставление данных**
* Автоматически синхронизировать имена и контактную информацию клиентов
* Сопоставлять адреса выставления счетов с полями подписчика
* Отслеживать даты регистрации для целевых кампаний

**Контроль на уровне продукта**
* Назначать разные списки разным продуктам
* Создавать целевые кампании на основе того, что покупают клиенты
* Отключать Mailster для конкретных продуктов при необходимости

**Профессиональный опыт**
* Интеграция с нативным UI Mailster для выбора списка
* Подробные журналы для устранения неполадок и проверки
* Бесшовные обновления существующих подписчиков

## How It Works

1. Установите и активируйте дополнение в вашей сети Ultimate Multisite
2. Настройте глобальные списки по умолчанию в настройках Ultimate Multisite
3. При желании переопределите списки, специфичными для продукта
4. Выберите режим согласия и время подписки
5. Наблюдайте, как ваши списки Mailster растут автоматически!

## Installation

1. Загрузите файлы плагина в каталог дополнений Ultimate Multisite
2. Убедитесь, что Mailster установлен и активен на вашем основном сайте
3. Активируйте дополнение через страницу дополнений Ultimate Multisite
4. Перейдите в Настройки > Mailster Integration для настройки
5. Выберите ваши списки по умолчанию и настройте предпочтения согласия

## Frequently Asked Questions

## Работает ли это дополнение без Ultimate Multisite?

Нет, это дополнение требует Ultimate Multisite для работы. Оно специально разработано для интеграции Mailster с системой клиентов и продуктов Ultimate Multisite.

## Нужно ли устанавливать Mailster на всех сайтах?

Нет, Mailster нужно установить и активировать только на основном сайте сети. Дополнение автоматически подключится к нему со всех подсайтов.

## Могу ли я назначать разные списки разным продуктам?

Да! Вы можете установить глобальные списки по умолчанию, а затем переопределить их для каждого продукта. Это позволяет создавать высоко целевые email‑кампании на основе того, что покупают клиенты.

## Что произойдет, если клиент уже существует в Mailster?

По умолчанию дополнение обновит существующих подписчиков и добавит их в новые списки. Вы можете отключить это в настройках, если предпочитаете пропускать существующих подписчиков.

## Соответствует ли это GDPR?

Да! Дополнение включает режим согласия через флажок, при котором клиенты явно соглашаются получать письма. Это соответствует требованиям GDPR при наличии соответствующих политик конфиденциальности.

## Могу ли я использовать двойное подтверждение?

Конечно! Включите настройку двойного подтверждения, и Mailster отправит письма с подтверждением перед активацией подписок.
