# Фильтр: ultimate_multisite_content_sync_url

Позволяет разработчикам управлять относительным путем к корню плагина.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $dir | `string` | Относительный путь к корню плагина, который вы хотите получить. |
