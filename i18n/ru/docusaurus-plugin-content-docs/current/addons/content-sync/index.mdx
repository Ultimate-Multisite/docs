# Синхронизация контента

Multisite Ultimate: Синхронизация контента

Этот аддон для Multisite Ultimate позволяет сетевым администраторам синхронизировать контент с любого сайта на другие сайты в сети. Он поддерживает все типы записей, включая записи, страницы, пользовательские типы записей и шаблоны Elementor. Аддон включает многошаговый мастер для выбора контента и целевых сайтов, с пакетной обработкой и обратной связью о ходе выполнения в реальном времени.

## Features

* Выбор контента: визуальная сетка карточек с миниатюрами и фильтрацией по типу записей
* Все типы записей: синхронизация записей, страниц, продуктов, шаблонов Elementor и любых пользовательских типов записей
* Выбор сайтов: выберите, на какие сайты синхронизировать контент, с исключением исходного сайта и предварительным выбором дочерних сайтов
* Пакетная обработка: синхронизация до 10 сайтов одновременно с обновлениями прогресса
* Синхронизация медиа: автоматически копирует и переназначает медиафайлы (изображения, видео) из контента записей на целевые сайты
* Синхронизация таксономий: сохраняет категории, теги и пользовательские таксономии
* Поддержка Elementor: полная поддержка шаблонов Elementor с регенерацией CSS и переназначением ID шаблонов
* Разрешение конфликтов: опция замены существующего контента или пропуска
* Управление кэшем: автоматически очищает кэш Elementor на целевых сайтах после синхронизации

## Requirements

* WordPress 5.3 или выше
* PHP 7.4 или выше
* плагин Multisite Ultimate (активен)
* плагин Elementor (необязательно, требуется для синхронизации шаблонов Elementor)

## Installation

1. Загрузите 'ultimate-multisite-content-sync' в каталог '/wp-content/plugins/'
2. Активируйте плагин через меню 'Plugins' в админке сети WordPress
3. Настройте параметры в Ultimate Multisite → Settings → Синхронизация контента
4. Перейдите в Ultimate Multisite → Синхронизация контента, чтобы начать синхронизацию
