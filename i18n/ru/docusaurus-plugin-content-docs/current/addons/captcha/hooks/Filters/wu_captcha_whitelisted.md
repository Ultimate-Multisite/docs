# Фильтр: wu_captcha_whitelisted

Фильтр, определяющий, следует ли включить текущего пользователя/запрос в белый список.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $whitelisted | `bool` | Определяет, следует ли пропустить проверку капчи. |

### С 1.2.0

### Источник
