# Действие: wu_captcha_after_validation

Вызывается после проверки капчи.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $result | `array` | Результат проверки. |
| $token | `string` | Токен, который был проверен. |

- 1.2.0
