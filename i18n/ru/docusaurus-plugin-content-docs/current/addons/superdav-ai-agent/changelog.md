# Изменения в версии (Changelog)

## 1.16.0 — Выпущено 2026-05-20

### Новое

- **Возможность генерации SVG-логотипа** — Theme Builder теперь может генерировать и встраивать пользовательские SVG-логотипы с безопасной очисткой (sanitisation) с учетом пространства имен.
- **Загрузка фото в интервью по поиску контекста** — Интервью по поиску контекста в Theme Builder теперь включает этап загрузки фотографии для более глубокого понимания дизайна.
- **Проверка контрастности палитры** — Проверяйте цветовые пары на соответствие WCAG, прежде чем применять их к теме.
- **Меню для сферы гостеприимства** — Theme Builder теперь может генерировать структурированные страницы меню блюд и напитков для предприятий сферы гостеприимства.
- **Предпросмотр на десктопе и мобильном устройстве** — Просматривайте свой дизайн на десктопных и мобильных устройствах во время выбора направления дизайна.
- **Параметр метки навигации** — Функция создания меню (Create Menu) теперь поддерживает отдельную `navigation_label`, отличную от заголовка страницы.
- **Доступность инструмента Tier 1** — sd-ai-agent/site-scrape теперь является инструментом Tier 1 и доступен по умолчанию в Theme Builder.

### Исправлено

- **Кэш AI-клиента** — теперь использует транзиенты (transients) для сохранения данных между запросами, предотвращая потерю данных при длительных задачах агента.
- **Ссылки действий в строке плагина** — исправлено и переименовано для большей ясности.

## 1.10.0 — Выпущено 2026-05-05

### Новое

- **Поиск в интернете через Tavily** — Добавлен Tavily в качестве поискового провайдера для получения более полных результатов поиска в интернете, наряду с Brave Search.
- **Встроенные навыки, учитывающие тему** — Блочные темы (Block Themes), Классические темы (Classic Themes), Kadence Blocks и руководства по навыкам Kadence Theme теперь поставляются вместе с плагином.
- **Возможность создания контактной формы в конструкторе сайтов** — Добавьте контактную форму на любую страницу прямо из интерфейса чата.

### Улучшено

- **Рефакторинг интеграции WooCommerce** — Теперь использует нативные API WooCommerce для повышения надежности и совместимости.
- **Автоматическое обновление списка провайдеров** — При активации или деактивации любого плагина.

### Исправлено

- **Функция navigate-to** — Исправлена бесконечная перезагрузка на некоторых страницах админ-панели.
- **Функция list-posts** — Теперь корректно преобразует названия категорий и тегов в слаги.
- **Команды WP-CLI** — Восстановлены отсутствующие псевдонимы пространств имен после предыдущего рефакторинга.
- **Автоматизация событий** — Корректно обрабатывает сайты, где таблицы автоматизации еще не созданы.
- **Функция memory-save** — Теперь использует правильный префикс пространства имен в конструкторе системных инструкций.
- **Результаты инструмента Scalar** — Теперь корректно оборачиваются перед возвратом в AI.
- **Статистика использования** — Теперь корректно обрабатывает устаревший формат ключа функции при обновлении со старых версий.
