# Журнал изменений {#changelog}

## 1.18.0 — Выпущено 2026-06-29 {#1180--released-on-2026-06-29}

### Новое {#new}

- **Инструменты Google Calendar** — чтение настроенных календарей и событий для автоматизаций, учитывающих расписание
- **Сопоставление контактов и помощники для участников** — сопоставление участников календаря с пользователями и контактами сайта
- **Шлюзы утверждения человеком и записи напоминаний** — приостанавливайте автоматизации для проверки и избегайте дублирующих уведомлений
- **Провайдер SMS TextBee** — отправка настроенных уведомлений в текстовых сообщениях через TextBee
- **Расширенный сопутствующий пакет** — добавлены инструменты для файловой системы доверенного разработчика, базы данных, WP-CLI, диспетчера REST, сборщика плагинов, git-снимков, управления пользователями и бенчмарков, распространяемые отдельно от сборки WordPress.org

### Улучшено {#improved}

- **Настройка управляемого сервиса Superdav** — добавлены endpoint размещённого сервиса и автоматическое предоставление подключений для поддерживаемых сайтов
- **Упаковка релиза** — сборка отдельных ZIP-архивов ядра и Advanced, публикация обоих на GitHub и отправка только пакета ядра в WordPress.org

### Исправлено {#fixed}

- **Надёжность AI-запросов** — улучшены выбор модели, тайм-ауты запросов, модель настройки по умолчанию, обработка текста рассуждений и рекомендации по повторным попыткам для недействительных вызовов инструментов
- **Усиление календаря и напоминаний** — усилены токены Google Calendar и дедупликация напоминаний
- **Возобновление онбординга и утверждений** — исправлены запуски онбординга на frontend и возобновление подтверждённых возможностей
- **Проблемы проверки упаковки WordPress.org** — учтены замечания проверки упаковки для релиза ядра

## 1.16.0 — Выпущено 2026-05-20 {#1160--released-on-2026-05-20}

### Новое {#new-1}

- **Возможность Generate Logo SVG** — Theme Builder теперь может генерировать и встраивать пользовательские SVG-логотипы с безопасной для пространств имён санитизацией
- **Загрузка фото в интервью для изучения требований** — интервью Theme Builder для изучения требований теперь включает шаг загрузки фото для более богатого дизайнерского контекста
- **Возможность Validate Palette Contrast** — проверка цветовых пар на соответствие WCAG перед применением к теме
- **Меню для hospitality-сферы** — Theme Builder теперь может генерировать структурированные страницы меню еды и напитков для hospitality-бизнесов
- **Рендеринг предпросмотра на компьютере и мобильных устройствах** — просматривайте свой дизайн на компьютере и мобильных устройствах во время выбора направления дизайна
- **Параметр метки навигации** — возможность Create Menu теперь поддерживает отдельный `navigation_label`, отличный от заголовка страницы
- **Доступность инструментов Tier 1** — sd-ai-agent/site-scrape теперь является инструментом Tier 1, доступным по умолчанию в Theme Builder

### Исправлено {#fixed-1}

- **Кэш AI Client** — теперь поддерживается транзиентами для сохранения между запросами, предотвращая потерю данных при длительных задачах агента
- **Ссылки действий в строке плагина** — исправлены и переименованы для ясности

## 1.10.0 — Выпущено 2026-05-05 {#1100--released-on-2026-05-05}

### Новое {#new-2}

- **Интернет-поиск Tavily** — добавлен Tavily как поисковый провайдер для более насыщенных результатов интернет-поиска наряду с Brave Search
- **Встроенные навыки с учётом темы** — руководства по навыкам Block Themes, Classic Themes, Kadence Blocks и Kadence Theme теперь поставляются с плагином
- **Возможность формы контактов для конструктора сайта** — добавьте форму контактов на любую страницу прямо из интерфейса чата

### Улучшено {#improved-1}

- **Интеграция WooCommerce переработана** — теперь использует нативные API WooCommerce для лучшей надёжности и совместимости
- **Список провайдеров обновляется автоматически** — при активации или деактивации любого плагина

### Исправлено {#fixed-2}

- **Возможность navigate-to** — исправлен бесконечный цикл перезагрузки на некоторых страницах администрирования
- **Возможность list-posts** — теперь корректно преобразует названия рубрик и тегов в слаги
- **Команды WP-CLI** — восстановлены отсутствующие псевдонимы пространств имён после предыдущего рефакторинга
- **Автоматизация событий** — корректно обрабатывает сайты, где таблицы автоматизации ещё не созданы
- **Возможность memory-save** — теперь использует правильный префикс пространства имён в построителе системных инструкций
- **Скалярные результаты инструментов** — теперь корректно оборачиваются перед возвратом в AI
- **Статистика использования** — теперь корректно обрабатывает устаревший формат ключа возможности при обновлении со старых версий
