# Superdav AI Agent

Superdav AI Agent — это продвинутый помощник на базе искусственного интеллекта для WordPress, который расширяет возможности Gratis AI Agent, добавляя навыки, учитывающие тему оформления, улучшенный поиск в интернете и улучшенные функции конструктора сайтов. В версии 1.16.0 добавлены генерация пользовательских логотипов, проверка контрастности цветовой палитры, загрузка фотографий во время интервью по поиску информации, генерация структурированных меню для сферы гостеприимства и предварительный просмотр на десктопе/мобильных устройствах в Theme Builder.

## Основные функции

- **Навыки, учитывающие тему** — автоматически адаптируют рекомендации и возможности в зависимости от активной темы WordPress и установленных плагинов
- **Множественные поисковые провайдеры в интернете** — выбирайте между Brave Search и Tavily для получения более полных результатов поиска и расширенных исследовательских возможностей
- **Формы обратной связи в конструкторе сайтов** — создавайте полностью функциональные формы обратной связи прямо из чата, используя ваш установленный плагин для форм
- **Улучшенный конструктор сайтов** — улучшенные многоэтапные планы создания сайтов с лучшим обнаружением плагинов и восстановлением после ошибок
- **Встроенные навыки для тем** — специализированные рекомендации для Block Themes, Classic Themes, Kadence Blocks и Kadence Theme
- **Улучшенная интеграция с WooCommerce** — переработана с использованием нативных API WooCommerce для повышения надежности и совместимости
- **Автоматическое обнаружение провайдера** — автоматически обнаруживает и обновляет доступные плагины и темы при их активации или деактивации
- **Генерация пользовательских логотипов** — генерируйте и встраивайте пользовательские SVG-логотипы с автоматической очисткой (sanitisation), безопасной для пространства имен
- **Проверка контрастности палитры** — проверяйте цветовые пары на соответствие требованиям WCAG, прежде чем применять их к вашей теме
- **Поиск информации с помощью фотографий** — загружайте фотографии во время интервью по поиску информации в Theme Builder, чтобы повлиять на направление дизайна
- **Меню для сферы гостеприимства** — генерируйте структурированные страницы меню блюд и напитков для ресторанов, кафе и предприятий сферы гостеприимства
- **Предварительный просмотр на десктопе и мобильных устройствах** — просматривайте свой дизайн на десктопе и мобильных устройствах во время выбора направления дизайна
- **Скрапинг сайтов Уровня 1** — sd-ai-agent/site-scrape теперь доступен по умолчанию для анализа сайтов конкурентов и источников вдохновения

## Требования

- WordPress 6.0 или выше
- PHP 8.1 или выше
- WordPress Multisite (активировано на уровне сети) или установка для одного сайта
- Активный плагин-коннектор для AI-провайдера (Anthropic, OpenAI или совместимый)
- Gratis AI Agent v1.9.0 или выше (Superdav AI Agent расширяет Gratis AI Agent)

## Установка

1. Загрузите файлы плагина в `/wp-content/plugins/superdav-ai-agent`
2. Активируйте плагин на экране **Плагины** в WordPress
3. Убедитесь, что Gratis AI Agent также активирован
4. Перейдите в **Gratis AI Agent → Настройки** и настройте вашего поискового провайдера (необязательно)

## Быстрый старт

После активации Superdav AI Agent автоматически улучшает вашу установку Gratis AI Agent, добавляя навыки, учитывающие тему, и дополнительные возможности. Откройте панель **Gratis AI Agent** в боковой панели администратора WordPress и начните использовать новые функции.

## Навыки, учитывающие тему

Superdav AI Agent автоматически обнаруживает вашу активную тему и установленные плагины, загружая соответствующий навык, учитывающий тему, в контекст помощника. Доступные навыки включают:

| Навык | Применяется к |
|---|---|
| **Block Themes** | Темы, использующие архитектуру на основе блоков WordPress (Full Site Editing) |
| **Classic Themes** | Традиционные темы WordPress, использующие PHP-шаблоны и классический редактор |
| **Kadence Blocks** | Сайты с активным плагином Kadence Blocks |
| **Kadence Theme** | Сайты с активной темой Kadence |

Подробные инструкции по каждому навыку см. в [Навыки, учитывающие тему](../../user-guide/configuration/theme-aware-skills).

## Поисковые провайдеры в интернете

Superdav AI Agent v1.10.0 добавляет **Tavily** в качестве альтернативы Brave Search для поисковых запросов в интернете. Оба провайдера предлагают результаты поиска в реальном времени и могут быть настроены в **Gratis AI Agent → Настройки → Расширенные**.

| Провайдер | Возможности | Настройка |
|---|---|---|
| **Brave Search** | Быстрые результаты поиска с акцентом на конфиденциальность | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Более полные результаты поиска, расширенные исследовательские возможности | [Tavily API](https://tavily.com/) |

Инструкции по настройке см. в [Поиск в интернете](../../user-guide/configuration/internet-search).

## Формы обратной связи в конструкторе сайтов

Теперь агент Site Builder может создавать формы обратной связи прямо из чата. Просто попросите Site Builder создать форму обратной связи, и он автоматически обнаружит ваш установленный плагин для форм (Contact Form 7, WPForms, Fluent Forms или Gravity Forms) и сгенерирует готовую к использованию форму.

Подробные инструкции см. в [Формы обратной связи в конструкторе сайтов](../../user-guide/administration/site-builder-contact-forms).

## Функции Theme Builder

В Superdav AI Agent v1.16.0 Theme Builder включает мощные новые возможности:

### Интервью по поиску информации с загрузкой фотографий

Интервью по поиску информации в Theme Builder теперь включает этап загрузки фотографий, где вы можете предоставить визуальное вдохновение. Загрузите фотографии вашего бренда, сайтов конкурентов или источники вдохновения, чтобы помочь Theme Builder понять ваше направление дизайна.

Подробные инструкции см. в [Интервью по поиску информации](./theme-builder/discovery-interview.md).

### Направление дизайна с предварительным просмотром на десктопе и мобильных устройствах

Просмотрите свой дизайн как на десктопных, так и на мобильных устройствах, прежде чем Theme Builder сгенерирует вашу полную тему. Смотрите предварительный просмотр в реальном времени на нескольких адаптивных точках останова.

Подробные инструкции см. в [Направление дизайна](./theme-builder/design-direction.md).

### Меню для сферы гостеприимства

Theme Builder теперь может генерировать структурированные страницы меню блюд и напитков для ресторанов, кафе, баров и других предприятий сферы гостеприимства. Меню включают цены, описания, информацию о диетических ограничениях и предупреждения об аллергенах.

Подробные инструкции см. в [Меню для сферы гостеприимства](./theme-builder/hospitality-menus.md).

### Новые возможности

- **[Генерация SVG-логотипа](./abilities/generate-logo-svg.md)** — создание пользовательских SVG-логотипов с автоматической очисткой
- **[Проверка контрастности палитры](./abilities/validate-palette-contrast.md)** — проверка цветовых пар на соответствие WCAG
- **[Создание меню](./abilities/create-menu.md)** — создание навигационных меню с отдельными метками навигации

Информация о доступных инструментах и инструменте скрапинга сайтов Уровня 1 см. в [Инструменты и конфигурация](./theme-builder/tools.md).

## Конфигурация

Все настройки находятся в **Gratis AI Agent → Настройки** в админ-панели WordPress. Superdav AI Agent добавляет следующие опции конфигурации:

| Настройка | Описание |
|---|---|
| **Поисковый провайдер в интернете** | Выберите между Brave Search и Tavily (необязательно) |
| **API-ключ Tavily** | API-ключ для поиска в интернете Tavily (если выбран Tavily) |

## Улучшения в v1.16.0

- **Кэш клиента AI** — теперь поддерживается транзиентами для сохранения данных между запросами, предотвращая потерю данных при длительных задачах агента
- **Инструмент скрапинга сайтов Уровня 1** — sd-ai-agent/site-scrape теперь доступен по умолчанию в Theme Builder для анализа сайтов конкурентов и источников вдохновения
- **Ссылки действий в строке плагина** — исправлены и переименованы для большей ясности

## Улучшения в v1.10.0

- **Переработана интеграция с WooCommerce** — теперь использует нативные API WooCommerce для повышения надежности и совместимости
- **Автоматическое обновление списка провайдеров** — автоматически обнаруживает, когда плагины активируются или деактивируются, и обновляет список доступных провайдеров
- **Улучшенный конструктор сайтов** — улучшенные многоэтапные планы создания сайтов с лучшим восстановлением после ошибок
- **Навыки, учитывающие тему** — четыре новых встроенных навыка, которые автоматически адаптируются к вашей активной теме

## Исправления ошибок в v1.10.0

- Исправлена возможность `navigate-to`, вызывавшая бесконечные циклы перезагрузки на некоторых страницах админ-панели
- Исправлена возможность `list-posts`, которая некорректно разрешала имена категорий и тегов в слаги
- Исправлены команды WP-CLI, где отсутствовали псевдонимы пространства имен после предыдущей переработки
- Исправлена обработка событий, которая корректно обрабатывает сайты, где таблицы автоматизации еще не созданы
- Исправлена возможность `memory-save`, использующая правильный префикс пространства имен в генераторе системных инструкций
- Исправлено корректное оборачивание результатов скалярного инструмента перед возвратом их в AI
- Исправлена статистика использования, которая корректно обрабатывает устаревший формат ключа возможности при обновлении со старых версий
