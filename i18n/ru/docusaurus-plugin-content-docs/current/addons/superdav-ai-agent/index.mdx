# Superdav AI Agent

Superdav AI Agent — это продвинутый AI-ассистент для WordPress, который расширяет Gratis AI Agent навыками с учетом темы, улучшенными возможностями поиска в интернете, функциями конструктора сайта и дополнительными интеграциями автоматизации. Версия 1.18.0 добавляет инструменты чтения Google Calendar, сопоставление контактов и участников, точки обязательного подтверждения человеком, дедупликацию напоминаний, SMS-уведомления TextBee, управляемое предоставление сервиса Superdav и отдельно распространяемый сопутствующий пакет Advanced для рабочих процессов доверенных разработчиков.

## Ключевые возможности

- **Навыки с учетом темы** — автоматически адаптирует рекомендации и возможности на основе вашей активной темы WordPress и установленных плагинов
- **Несколько провайдеров интернет-поиска** — выбирайте между Brave Search и Tavily для более богатых результатов поиска и расширенных исследовательских возможностей
- **Контактные формы конструктора сайта** — создавайте полностью функциональные контактные формы прямо из интерфейса чата с помощью установленного плагина форм
- **Улучшенный конструктор сайта** — улучшенные многоэтапные планы сборки сайта с более эффективным обнаружением плагинов и восстановлением после ошибок
- **Встроенные навыки для тем** — специализированные рекомендации для Block Themes, Classic Themes, Kadence Blocks и Kadence Theme
- **Улучшенная интеграция WooCommerce** — переработана для использования нативных WooCommerce API для большей надежности и совместимости
- **Автоматическое определение провайдеров** — автоматически обнаруживает и обновляет доступные плагины и темы при их активации или деактивации
- **Генерация пользовательского логотипа** — генерируйте и встраивайте пользовательские SVG-логотипы с автоматической очисткой, безопасной для пространств имен
- **Проверка контрастности палитры** — проверяйте цветовые пары на соответствие требованиям доступности WCAG перед применением к вашей теме
- **Обнаружение с использованием фотографий** — загружайте фотографии во время ознакомительного интервью Theme Builder, чтобы повлиять на направление дизайна
- **Меню для сферы гостеприимства** — генерируйте структурированные страницы меню еды и напитков для ресторанов, кафе и предприятий сферы гостеприимства
- **Предпросмотр на компьютере и мобильных устройствах** — просматривайте свой дизайн на компьютере и мобильных устройствах во время выбора направления дизайна
- **Скрапинг сайтов уровня 1** — sd-ai-agent/site-scrape теперь доступен по умолчанию для анализа сайтов конкурентов и сайтов для вдохновения
- **Инструменты чтения Google Calendar** — читайте настроенные календари и события, чтобы автоматизации могли учитывать контекст предстоящего расписания
- **Сопоставление контактов и участников** — сопоставляйте участников календаря с известными пользователями и контактами сайта перед созданием последующих задач
- **Точки обязательного подтверждения человеком** — приостанавливайте чувствительные автоматизации для проверки перед отправкой сообщений, возобновлением работы или внесением изменений
- **Записи напоминаний и дедупликация** — отслеживайте уже отправленные напоминания, чтобы повторные запуски автоматизации не уведомляли одного и того же получателя дважды
- **SMS-уведомления TextBee** — отправляйте настроенные текстовые уведомления через TextBee, когда включен SMS-провайдер
- **Управляемое предоставление сервиса Superdav** — подключайте поддерживаемые сайты к размещенным конечным точкам Superdav с автоматической настройкой вместо ручной конфигурации конечных точек
- **Сопутствующий пакет Advanced** — отдельно распространяемые инструменты для доверенных разработчиков для рабочих процессов с файловой системой, базой данных, WP-CLI, REST-диспетчером, конструктором плагинов, снимком git, управлением пользователями и бенчмарками

## Требования

- WordPress 6.0 или выше
- PHP 8.1 или выше
- WordPress Multisite (с активацией по сети) или установка на один сайт
- Активный плагин-коннектор AI-провайдера (Anthropic, OpenAI или совместимый)
- Gratis AI Agent v1.9.0 или выше (Superdav AI Agent расширяет Gratis AI Agent)

## Установка

1. Загрузите файлы плагина в `/wp-content/plugins/superdav-ai-agent`
2. Активируйте плагин через экран **Плагины** в WordPress
3. Убедитесь, что Gratis AI Agent также активирован
4. Перейдите в **Gratis AI Agent → Настройки** и при необходимости настройте своего провайдера интернет-поиска, подключение к управляемому сервису Superdav, Google Calendar или учетные данные TextBee

Пакет WordPress.org содержит основные возможности Superdav AI Agent. Сопутствующий пакет Advanced собирается и распространяется отдельно для сред, где инструменты доверенных разработчиков явно установлены и авторизованы.

## Навыки с учетом темы

Superdav AI Agent автоматически определяет вашу активную тему и установленные плагины, загружая соответствующий навык с учетом темы в контекст ассистента. Доступные навыки включают:

| Навык | Применяется к |
|---|---|
| **Block Themes** | Темы, использующие блочную архитектуру WordPress (Full Site Editing) |
| **Classic Themes** | Традиционные темы WordPress, использующие шаблоны PHP и классический редактор |
| **Kadence Blocks** | Сайты с активным плагином Kadence Blocks |
| **Kadence Theme** | Сайты с темой Kadence в качестве активной темы |

См. [Навыки с учетом темы](../../user-guide/configuration/theme-aware-skills) для подробных рекомендаций по каждому навыку.

## Провайдеры интернет-поиска

Superdav AI Agent v1.10.0 добавляет **Tavily** как альтернативу Brave Search для поисковых запросов в интернете. Оба провайдера предлагают результаты поиска в реальном времени и могут быть настроены в **Gratis AI Agent → Настройки → Расширенные**.

| Поставщик | Возможности | Настройка |
|---|---|---|
| **Brave Search** | Быстрые результаты поиска с акцентом на конфиденциальность | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Более насыщенные результаты поиска, расширенные возможности исследования | [Tavily API](https://tavily.com/) |

См. [Интернет-поиск](../../user-guide/configuration/internet-search) для инструкций по настройке.

## Рабочие процессы календаря, утверждения и уведомлений

Superdav AI Agent v1.18.0 расширяет рабочие процессы автоматизации за пределы задач, связанных с контентом и созданием сайтов:

- **Инструменты Google Calendar** позволяют агенту читать настроенные календари и сведения о событиях, когда доступны учетные данные календаря.
- **Помощники для контактов и участников** сопоставляют участников событий с пользователями WordPress или известными контактами перед подготовкой напоминаний или последующих задач.
- **Шлюзы утверждения человеком** позволяют автоматизации приостанавливаться, пока авторизованный пользователь не проверит предложенное действие.
- **Записи напоминаний** предотвращают дублирование календарных или уведомительных напоминаний, когда автоматизация повторяется после сбоя или запускается по расписанию многократно.
- **Поддержка TextBee SMS** включает уведомления текстовыми сообщениями, когда настроен поставщик TextBee.

Используйте эти инструменты для помощи с учетом расписания, последующих действий с участниками и рабочих процессов напоминаний, которые всё еще требуют проверки человеком перед отправкой чувствительных сообщений.

## Контактные формы Site Builder

Агент Site Builder теперь может создавать контактные формы прямо из интерфейса чата. Просто попросите Site Builder создать контактную форму, и он автоматически обнаружит установленный у вас плагин форм (Contact Form 7, WPForms, Fluent Forms или Gravity Forms) и сгенерирует готовую к использованию форму.

См. [Контактные формы Site Builder](../../user-guide/administration/site-builder-contact-forms) для подробного руководства.

## Конфигурация

Все настройки находятся в **Gratis AI Agent → Settings** в админке WordPress. Superdav AI Agent добавляет следующие параметры конфигурации:

| Настройка | Описание |
|---|---|
| **Internet Search Provider** | Выберите между Brave Search и Tavily (необязательно) |
| **Tavily API Key** | API-ключ для интернет-поиска Tavily (если выбран Tavily) |
| **Managed Superdav Service** | Подключите поддерживаемые сайты к размещенному сервису Superdav с автоматическим предоставлением endpoint |
| **Google Calendar Credentials** | Настройте доступ к календарю, чтобы агент мог читать календари и события |
| **TextBee SMS Provider** | Настройте учетные данные TextBee для утвержденных рабочих процессов SMS-уведомлений |

## Расширенный сопутствующий пакет

Superdav AI Agent v1.18.0 добавляет отдельно распространяемый расширенный сопутствующий пакет для доверенных рабочих процессов разработчиков. Эти возможности не входят в основной пакет WordPress.org. Устанавливайте сопутствующий пакет только на сайтах, где администраторы явно доверяют агенту доступ к инструментам разработчика.

Расширенные инструменты включают операции с файловой системой, доступ к базе данных, WP-CLI, рабочие процессы REST-диспетчера, утилиты для создания плагинов, git-снимки, помощники по управлению пользователями и инструменты бенчмаркинга. Ограничьте эти инструменты только доверенными администраторами и проверьте конфигурацию шлюзов утверждения перед включением их на производственных сайтах.
