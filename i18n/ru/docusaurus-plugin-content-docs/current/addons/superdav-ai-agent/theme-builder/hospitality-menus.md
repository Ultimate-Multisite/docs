# Меню для заведений

Функция **Меню для заведений** позволяет Theme Builder создавать и встраивать структурированные страницы меню блюд и напитков прямо на ваш сайт WordPress.

## Обзор {#overview}

Теперь Theme Builder может создавать профессиональные, структурированные страницы меню для предприятий сферы гостеприимства, включая рестораны, кафе, бары и кейтеринговые службы. Эти меню полностью интегрируются в дизайн вашего сайта и легко обновляются и управляются.

## Поддерживаемые типы заведений {#supported-hospitality-types}

- **Рестораны** — меню полного цикла обслуживания
- **Кафе** — меню кофе и легких закусок
- **Бары и лаунж-зоны** — меню напитков и закусок
- **Пекарни** — меню выпечки и хлеба
- **Кейтеринговые службы** — варианты меню для мероприятий
- **Фудтраки** — меню мобильного питания
- **Пивоварни и винодельни** — меню напитков с описаниями

## Структура меню {#menu-structure}

### Категории меню {#menu-categories}

Меню организованы по категориям:

- **Закуски** — стартеры и небольшие блюда
- **Основные блюда** — основные блюда
- **Гарниры** — дополнения и овощи
- **Десерты** — сладкие угощения
- **Напитки** — напитки (алкогольные и безалкогольные)
- **Специальные предложения** — ежедневные или сезонные блюда

### Формат элемента меню {#menu-item-format}

Каждый элемент меню включает:

```json
{
  "name": "Жареный лосось",
  "description": "Свежий атлантический лосось с соусом из лимонного масла",
  "price": "$24.95",
  "dietary_info": ["Без глютена", "Высокое содержание белка"],
  "allergens": ["Рыба"],
  "availability": "Ежедневно"
}
```

#### Поля элемента {#item-fields}

| Поле | Тип | Описание |
|-------|------|-------------|
| `name` | string | Название блюда или напитка |
| `description` | string | Подробное описание элемента |
| `price` | string | Цена (с форматированием валюты) |
| `dietary_info` | array | Диетические атрибуты (Веганское, Без глютена и т. д.) |
| `allergens` | array | Распространенные аллергены (Орехи, Ракообразные и т. д.) |
| `availability` | string | Когда доступно (Ежедневно, Сезонно и т. д.) |

## Создание меню для заведений {#creating-hospitality-menus}

### Шаг 1: Предоставьте информацию о меню {#step-1-provide-menu-information}

Расскажите Theme Builder о вашем меню:

```
Создай меню для итальянского ресторана: закуски, паста, основные блюда и десерты. Включи цены и описания.
```

### Шаг 2: Theme Builder генерирует меню {#step-2-theme-builder-generates-menu}

Theme Builder:

1. Создает структурированную страницу меню
2. Оформляет ее в соответствии с темой вашего сайта
3. Организует элементы по категориям
4. Форматирует цены и описания
5. Добавляет информацию о диетических ограничениях и аллергенах

### Шаг 3: Просмотр и настройка {#step-3-review-and-customize}

Вы можете:

1. Редактировать элементы меню и цены
2. Добавлять или удалять категории
3. Изменять порядок элементов внутри категорий
4. Обновлять описания и информацию о диетических ограничениях
5. Настраивать стиль и макет

## Варианты отображения меню {#menu-display-options}

### Полная страница меню {#full-menu-page}

Выделенная страница, отображающая все ваше меню:

- Организовано по категориям
- Возможно поиск и фильтрация
- Макет, удобный для печати
- Адаптивный дизайн для мобильных устройств

### Виджет меню {#menu-widget}

Встраивайте разделы меню на другие страницы:

- Избранные блюда на главной странице
- Ежедневные спецпредложения в сайдбаре
- Меню напитков на странице бара
- Витрина десертов в футере

### PDF-меню {#menu-pdf}

Генерируйте скачиваемое PDF-меню:

- Профессиональное оформление
- Качество, готовое к печати
- Включает изображения и описания
- Легко делиться и отправлять по электронной почте

## Диетические ограничения и аллергены {#dietary-and-allergen-information}

### Диетические атрибуты {#dietary-attributes}

Помечайте блюда информацией о диетических ограничениях:

- **Веганское** — без продуктов животного происхождения
- **Вегетарианское** — без мяса
- **Без глютена** — безопасно для людей с целиакией
- **Без молочных продуктов** — без продуктов из молока
- **Без орехов** — без древесных орехов или арахиса
- **Низкоуглеводное** — с пониженным содержанием углеводов
- **Высокое содержание белка** — богато белком

### Предупреждения об аллергенах {#allergen-warnings}

Включайте информацию о распространенных аллергенах:

- **Орехи** — древесные орехи и арахис
- **Ракообразные** — крабы и моллюски
- **Рыба** — все виды рыбы
- **Молочные продукты** — молоко и молочные продукты
- **Яйца** — куриные яйца
- **Соя** — соевые продукты
- **Глютен** — пшеница и злаки, содержащие глютен
- **Кунжут** — кунжутные семена и масло

## Управление меню {#menu-management}

### Обновление цен {#updating-prices}

Легко обновляйте цены в меню:

1. Перейдите на страницу меню
2. Нажмите «Редактировать меню»
3. Обновите цены для блюд
4. Сохраните изменения
5. Изменения появятся на вашем сайте немедленно

### Добавление сезонных блюд {#adding-seasonal-items}

Создавайте сезонные вариации меню:

1. Создайте новую версию меню
2. Добавьте сезонные блюда
3. Отметьте блюда как «Сезонные»
4. Установите даты доступности
5. Автоматически отображать в течение сезона

### Управление спецпредложениями {#managing-specials}

Отображайте ежедневные или еженедельные спецпредложения:

1. Создайте категорию «Спецпредложения»
2. Добавьте блюда с датами доступности
3. Выделите спецпредложения на главной странице
4. Обновляйте ежедневно или еженедельно
5. Архивируйте старые спецпредложения

## Интеграция с Theme Builder {#integration-with-theme-builder}

При использовании Theme Builder для сайтов заведений:

1. **Автоматическое обнаружение меню** — определяет, является ли ваш бизнес в сфере питания/напитков
2. **Создание страницы меню** — генерирует профессиональные страницы меню
3. **Соответствие дизайну** — меню соответствуют теме вашего сайта
4. **Оптимизация для мобильных устройств** — меню красиво отображаются на телефонах
5. **SEO-оптимизация** — меню дружественны для поисковых систем

## Лучшие практики {#best-practices}

### Дизайн меню {#menu-design}

- **Четкая организация** — логичная структура категорий
- **Читаемые описания** — аппетитные и информативные
- **Единообразное ценообразование** — понятная валюта и форматирование
- **Профессиональные фотографии** — изображения блюд высокого качества
- **Белое пространство** — не перегружайте страницу

### Контент {#content}

- **Точные описания** — описывайте блюда корректно
- **Выделение спецпредложений** — сделайте спецпредложения заметными
- **Указание аллергенов** — всегда перечисляйте аллергены
- **Регулярное обновление** — следите за актуальностью цен и блюд
- **Использование аппетитного языка** — заставляйте блюда выглядеть вкусными

### Доступность {#accessibility}

- **Читаемые шрифты** — используйте четкие, легко читаемые шрифты
- **Достаточный контраст** — убедитесь, что текст хорошо читается
- **Диетические метки** — четко отмечайте диетические опции
- **Предупреждения об аллергенах** — заметно отображайте аллергены
- **Удобство для мобильных устройств** — тестируйте на всех устройствах
