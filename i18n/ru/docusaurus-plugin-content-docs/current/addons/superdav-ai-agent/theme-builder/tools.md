# Инструменты и настройка

Theme Builder использует набор инструментов для анализа, проектирования и создания вашего сайта на WordPress. В версии 1.16.0 **sd-ai-agent/site-scrape стал инструментом Tier 1**, что означает, что он доступен по умолчанию в Theme Builder.

## Доступные инструменты

### Инструменты Tier 1 (Всегда доступны)

Инструменты Tier 1 доступны по умолчанию в Theme Builder без дополнительной настройки.

#### sd-ai-agent/site-scrape

**Назначение:** Анализ существующих веб-сайтов для извлечения идей дизайна, структуры контента и функциональности.

**Возможности:**

- **Анализ сайта** — сканирование и анализ сайтов-конкурентов или сайтов-источников вдохновения
- **Извлечение дизайна** — определение цветовой палитры, шрифтов и паттернов макета
- **Структура контента** — понимание организации и иерархии страниц
- **Обнаружение функций** — выявление плагинов и функционала
- **Анализ производительности** — проверка скорости загрузки и оптимизации страниц
- **SEO-анализ** — проверка метатегов и структурированных данных

**Пример использования:**

```
Проанализируй дизайн example.com и используй его как вдохновение для моего сайта.
```

**Что извлекается:**

- Цветовая палитра и использование цветов
- Типографика (шрифты и размеры)
- Структура макета и отступы
- Паттерны навигации
- Организация контента
- Использование изображений и медиа
- Интерактивные элементы
- Адаптивность для мобильных устройств

**Ограничения:**

- Не может сканировать сайты, защищенные паролем
- Уважает robots.txt и политики сайта
- Может не уловить динамический контент
- Сайты с большим количеством JavaScript могут иметь ограниченный анализ
- Крупные сайты могут требовать больше времени для анализа

### Инструменты Tier 2 (Опционально)

Инструменты Tier 2 доступны при явном включении в настройках Theme Builder.

#### Advanced Analytics

Предоставляет подробные метрики производительности:

- Время загрузки страниц
- Core Web Vitals
- SEO-показатель
- Показатель доступности (Accessibility score)
- Показатель лучших практик (Best practices score)

#### Content Optimizer

Анализирует и предлагает улучшения для:

- Читабельности
- SEO-оптимизации
- Использования ключевых слов
- Структуры контента
- Размещения призывов к действию (Call-to-action)

### Инструменты Tier 3 (Премиум)

Инструменты Tier 3 требуют дополнительной настройки или премиум-функций.

#### AI Content Generator

Генерирует контент для:

- Описаний товаров
- Страниц услуг
- Постов в блоге
- Мета-описаний
- Текстов призывов к действию

#### Advanced Design Tools

Предоставляет расширенные возможности дизайна:

- Генерация пользовательского CSS
- Создание анимации
- Дизайн интерактивных элементов
- Продвинутая теория цвета
- Оптимизация типографики

## Настройка инструментов

### Включение инструментов

Чтобы включить дополнительные инструменты в Theme Builder:

1. Перейдите в **Gratis AI Agent → Settings**
2. Выберите **Theme Builder → Tools**
3. Включите/выключите инструменты по мере необходимости
4. Сохраните настройки

### Разрешения инструментов

Некоторые инструменты требуют разрешений:

- **Site scraping** — требует доступа в Интернет
- **Analytics** — требует интеграции с Google Analytics
- **Content generation** — требует ключей API
- **Advanced features** — может требовать премиум-подписки

### Ключи API и учетные данные

Настройте ключи API для инструментов, которые их требуют:

1. Перейдите в **Gratis AI Agent → Settings → API Keys**
2. Введите учетные данные для каждого инструмента
3. Проверьте соединение
4. Сохраните данные в безопасном месте

## Ограничения и особенности работы инструментов

### Ограничение скорости (Rate Limiting)

- Сканирование ограничено для предотвращения перегрузки сервера
- Многократные запросы к одному и тому же сайту могут быть замедлены
- Крупные сайты могут требовать больше времени для анализа

### Точность контента

- Динамический контент может быть не полностью захвачен
- Контент, сгенерированный JavaScript, может быть неполным
- Данные в реальном времени могут не отражаться
- Некоторые материалы могут быть скрыты за платными стенами (paywalls)

### Юридические и этические соображения

- Уважайте robots.txt и политики сайта
- Не сканируйте контент, защищенный авторским правом, для воспроизведения
- Используйте анализ для вдохновения, а не для копирования
- Убедитесь, что у вас есть права на использование извлеченного контента
- Соблюдайте условия обслуживания анализируемых сайтов

### Влияние на производительность

- Сканирование крупных сайтов может занять время
- Несколько одновременных сканирований могут быть медленнее
- Скорость зависит от сетевого соединения
- Большие наборы данных могут потребовать больше ресурсов для обработки

## Лучшие практики

### Использование анализа сайта

1. **Анализируйте несколько сайтов** — соберите вдохновение из нескольких источников
2. **Сосредоточьтесь на структуре** — изучите паттерны макета и организации
3. **Извлекайте цвета** — используйте цветовые палитры в качестве отправной точки
4. **Изучайте типографику** — определите комбинации шрифтов, которые вам нравятся
5. **Просматривайте навигацию** — поймите организацию меню

### Этичное сканирование

1. **Уважайте robots.txt** — следуйте правилам сайта
2. **Не копируйте контент** — используйте анализ только для вдохновения
3. **Проверяйте права** — убедитесь, что вы можете использовать извлеченный контент
4. **Указывайте источники** — цитируйте источники вдохновения
5. **Соблюдайте условия** — следуйте условиям обслуживания сайта

### Максимизация эффективности инструментов

1. **Будьте конкретны** — запрашивайте конкретные типы анализа
2. **Предоставляйте контекст** — объясните цель вашего сайта
3. **Устанавливайте ожидания** — опишите свои дизайнерские цели
4. **Итерируйте** — уточняйте результаты на основе полученных данных
5. **Комбинируйте инструменты** — используйте несколько инструментов для комплексного анализа

## Устранение неполадок

### Сайт не сканируется

- Проверьте, доступен ли сайт публично
- Убедитесь, что robots.txt разрешает сканирование
- Попробуйте другой сайт
- Проверьте подключение к Интернету
- Обратитесь в службу поддержки, если проблема сохраняется

### Неполный анализ

- Сайт может содержать динамический контент
- JavaScript может быть не полностью отрендерен
- Крупные сайты могут вызвать тайм-аут
- Попробуйте проанализировать конкретные страницы вместо всего сайта
- Запросите конкретные типы анализа

### Низкая производительность

- Анализ крупных сайтов занимает больше времени
- Несколько одновременных сканирований медленнее
- Скорость зависит от сетевого соединения
- Попробуйте в часы наименьшей нагрузки
- Сначала проанализируйте более мелкие разделы
