# Fluent Forms

Этот аддон позволяет добавить ограничения к Fluent Forms на Multisite Ultimate в продуктах, членствах или сайтах.  
Fluent Forms использует пользовательские таблицы вместо пользовательских типов записей, поэтому обычные варианты ограничения не работают с Fluent Forms.

## Features

- Добавить ограничения для Fluent Forms или Conversational Fluent Forms  
- Клиенты получают дружелюбное сообщение, побуждающее их обновить план, когда они достигают лимита
