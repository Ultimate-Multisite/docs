# План с измерением

Дополнение Ultimate Multisite: План с измерением преобразует способ выставления счетов клиентам, вводя гибкие возможности измеряемого биллинга. Вместо жестких ограничений, которые блокируют пользователей при превышении, разрешите начисление дополнительных платежей за сверхпотребление, автоматически выставляя счета клиентам за дополнительное использование.

## Ключевые особенности

* **Измеряемый биллинг**: начислять клиентам в соответствии с фактическим использованием, а не фиксированными уровнями
* **Управление сверхпотреблением**: автоматически начислять за использование, превышающее лимиты плана
* **Отслеживание использования**: мониторинг потребления ресурсов клиентом в реальном времени
* **Гибкая ценовая политика**: устанавливать индивидуальные тарифы для разных ресурсов (хранение, пропускная способность, сайты и т.д.)
* **Автоматическое выставление счетов**: автоматически генерировать счета, когда пороги использования превышены
* **Отчёты об использовании**: подробная аналитика и отчёты об использовании для клиентов и администраторов

## Подходит для

* **Провайдеры хостинга**: предлагайте масштабируемые планы, которые растут вместе с потребностями клиентов
* **Платформы SaaS**: выставляйте счета на основе фактического потребления ресурсов
* **Сети агентств**: гибкие модели ценообразования для клиентских проектов
* **Реселлеры хостинга**: позволяйте партнёрам предлагать ценообразование на основе использования

## Как это работает

1. Настройте измеряемые ресурсы (объём диска, пропускная способность, количество сайтов и т.д.)
2. Установите базовые лимиты плана и тарифы за сверхпотребление
3. Система автоматически отслеживает использование по всей вашей сети
4. Когда клиенты превышают лимиты, начисляются платежи за сверхпотребление
5. Счета автоматически генерируются в соответствии с циклом выставления счетов
6. Клиенты могут просматривать своё использование и предстоящие платежи в реальном времени

## Поддерживаемые ресурсы

* Дисковое хранилище
* Пропускная способность/передача данных
* Количество сайтов
* Количество пользователей
* Пользовательские типы ресурсов

## Требования

* Multisite Ultimate 2.0.0 или выше
* Установка WordPress Multisite
* PHP 7.4 или выше

## Установка

1. Загрузите файлы плагина в `/wp-content/plugins/ultimate-multisite-metered-plans`
2. Активируйте плагин через экран «Плагины» в WordPress
3. Перейдите в Multisite Ultimate → Настройки → План с измерением
4. Настройте ваши измеряемые ресурсы и цены
5. Включите измеряемый биллинг для продуктов в редакторе продукта

## Часто задаваемые вопросы

## Что происходит, когда клиент превышает свой лимит?

Вместо блокировки доступа система отслеживает сверхпотребление и добавляет его к следующему счету на основе настроенных вами тарифов.

## Могу ли я установить разные тарифы за сверхпотребление для разных продуктов?

Да, каждый продукт может иметь уникальные измеряемые ресурсы и тарифы за сверхпотребление.

## Как клиенты уведомляются о использовании?

Клиенты могут просматривать текущее использование в своей панели управления и получать необязательные уведомления при приближении к лимиту или его превышении.

## Могу ли я всё ещё использовать жесткие лимиты?

Да, измеряемый биллинг является опциональным для каждого продукта. Вы можете сочетать традиционные продукты с жесткими лимитами с измеряемыми продуктами.
