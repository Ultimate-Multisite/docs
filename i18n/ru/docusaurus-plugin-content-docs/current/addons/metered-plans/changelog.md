# Журнал изменений Metered Plans

Версия 1.1.0 — выпущена 2026-05-05
- Новое: биллинг AI токенов для подсайтов multisite — отслеживайте и выставляйте счета за использование AI токенов на сайтах клиентов с настраиваемыми ставками за токен
- Новое: принудительное применение коннекторов переработано с динамическим обнаружением лимитов и сквозной записью, что обеспечивает точность в реальном времени для всех коннекторов
- Исправлено: обновления таблиц базы данных теперь корректно согласованы с определениями схемы BerlinDB, что предотвращает сбои обновления при новых установках
- Исправлено: обратные вызовы обновления базы данных преобразованы в правильный формат, что устраняет незаметные сбои обновления
- Исправлено: дробные значения теперь принимаются в поле ввода AI Usage Overage Markup
- Исправлено: устранены фатальные ошибки и проблемы двойной инициализации при загрузке плагина
- Улучшено: добавлен npm-скрипт check-env, чтобы среды разработки самонастраивались при первом запуске
