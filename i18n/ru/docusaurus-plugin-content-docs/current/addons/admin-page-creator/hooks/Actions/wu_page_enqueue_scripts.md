# Действие: wu_page_enqueue_scripts {#action-wupageenqueuescripts}

Добавляет здесь пользовательское действие, чтобы у нас было место для подключения скриптов и стилей по мере необходимости\ и чтобы убедиться, что они загружаются только на нужных страницах

## Параметры {#parameters}

| Имя | Тип | Описание |
|------|------|-------------|
| $ | `\WU_Page` | Текущая страница администратора |

### С версии {#since}

- 1.3.0
