# Filter: wu_apc_should_display_admin_menu

Разрешить администраторам скрывать страницы меню администратора, если они завершили процесс настройки.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $should_display_admin_menu | `bool` | Если необходимо отображать страницы. |
| $network_active | `bool` | Проверьте, активен ли плагин в режиме сети администратора. |
