# Admin Page Creator

## Содержание {#content}
Admin Page Creator позволит вам легко добавлять новые страницы администрирования и подстраницы в административной панели WordPress. Его можно использовать для создания приветственной страницы или страницы с руководством, которая может служить целевой страницей для новых пользователей. Вы можете использовать конструкторы страниц, такие как Beaver Builder, Elementor, Brizy и Oxygen, чтобы без лишних сложностей оформлять страницы.

Он также предлагает поддержку пользовательских HTML, CSS и JavaScript, а также контроль доступа на основе плана и роли пользователя!

## Возможности {#features}
* Поддержка пользовательских HTML и CSS
* Поддержка JavaScript
* Режим PHP-контента
* Разрешения на основе роли пользователя
* Поддержка конструкторов страниц

## Установка {#installation}

1. Загрузите **ultimate-multisite-admin-page-creator** в директорию **/wp-content/plugins/**
2. Активируйте плагин через меню **Плагины** в вашем WordPress Network Admin

## Руководство по использованию {#usage-guide}

Хотя WordPress делает добавление контента во фронтенд сайтов действительно простым, создание страниц и подстраниц в административной части требует базовых навыков кодирования. Ultimate Multisite: Admin Page Creator призван привнести такое же удобство использования фронтенда в бэкенд сайтов вашей сети.

## Зачем использовать Admin Page Creator? {#why-use-admin-page-creator}

Возможно, вы хотите предложить пользовательскую страницу в административной панели ваших клиентов с руководствами, а возможно, хотите добавить пользовательскую страницу с часто задаваемыми вопросами и виджетом контактов. Возможности с Admin Page Creator безграничны, и именно это мы рассмотрим в сегодняшнем руководстве.

## Создание вашей первой пользовательской страницы администрирования {#creating-your-first-custom-admin-page}

После установки и активации Ultimate Multisite: Admin Page Creator перейдите в меню администрирования сети и откройте Admin Pages.

![Список пользовательских страниц администрирования в Network Admin](/img/admin/admin-pages-list.png)

_Список пользовательских страниц администрирования в Network Admin_

Вы увидите список всех страниц администрирования, которые вы уже создали (поскольку это ваш первый визит, список будет пустым). Нажмите **Добавить новую страницу администрирования**. Вы будете перенаправлены в редактор страницы администрирования.

## Опции меню {#menu-options}

Сбоку, в метабоксе общих опций, вы сможете настроить тип меню, метку и многое другое.

### Типы меню {#menu-types}

Ваша новая пользовательская страница администрирования может иметь два разных типа меню: меню верхнего уровня и подменю.

Страница меню верхнего уровня, как следует из названия, добавит вашу только что созданную страницу в меню администрирования рядом со стандартными страницами WordPress.

Если вы выберете опцию верхнего уровня, вы также сможете выбрать порядок меню и значок меню.

Если вы хотите сделать эту страницу страницей подменю, вы сможете выбрать родительскую страницу, к которой нужно прикрепить эту страницу. Вы можете добавить свою пользовательскую страницу к стандартным страницам WordPress или к своим пользовательским страницам администрирования верхнего уровня.

## Расширенные опции {#advanced-options}

Вы сможете добавлять любой контент на свои пользовательские страницы администрирования. Это также включает правила и файлы CSS, JavaScript и внешние библиотеки. Вы можете сделать это с помощью вкладки расширенных опций.

### Пользовательский CSS {#custom-css}

На вкладке CSS вы можете добавить собственный пользовательский CSS-код, а также импортировать CSS-файлы с удаленных сайтов (например, CDN, Google Fonts и т. д.).

## Разрешения {#permissions}

Также в расширенных опциях вы сможете выбрать, какие подсайты будут иметь доступ к этой странице. Это полезно, если вы планируете предоставлять разный эксклюзивный контент для разных уровней планов, например.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_У вас есть полный контроль над тем, где будет отображаться эта страница_

**Важно** : Для отображения страницы должны быть выполнены оба условия. Это означает, что если пользователь является клиентом данного плана, но не имеет одной из выбранных ролей (или наоборот), страница не появится на его панели.

## Активно {#active}

Если вы хотите отключить пользовательскую страницу администратора, не удаляя её и не изменяя разрешения, просто снимите флажок с опции Active.

Деактивация страницы верхнего уровня автоматически деактивирует все подстраницы, прикреплённые к этой родительской странице.
