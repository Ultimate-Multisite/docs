# Журнал изменений Создателя страниц администратора

### Версия 1.9.0 - Выпущено 2026-01-18

* Исправлено: Фатальные ошибки при активации.
* Переработано: Классы источников контента используют трейт Singleton с правильной инициализацией конструктора родителя.
* Исправлено: Ошибки компиляции шаблона Vue.js, препятствующие отображению селектора типа контента.
* Исправлено: Отсутствующие свойства данных Vue (template_id, external_link_url_embedable, external_link_url_checking).
* Исправлено: TinyMCE редактор не инициализируется корректно при загрузке страницы.
* Исправлено: Файл поддержки Divi не загружается автоматически через Composer.
* Исправлено: Повреждённый синтаксис PHP в классе поддержки Divi.
* Улучшено: Организация и минификация JavaScript-активов.
* Улучшено: Совместимость с последними версиями WordPress и конструкторов страниц.
