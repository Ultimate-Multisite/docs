# Журнал изменений Ultimate AI Connector для совместимых эндпоинтов

## Версия 2.0.0 — Выпущено 2026-04-24 {#version-200--released-on-2026-04-24}

* Новое: Поддержка нескольких провайдеров — настройте несколько AI-эндпоинтов и маршрутизируйте запросы с автоматическим переключением между провайдерами.
* Исправлено: Интеграция SDK для нескольких провайдеров с корректными ID провайдеров, URL регистрации и областью действия HTTP-фильтров для каждого провайдера.
* Исправлено: Новая карточка провайдера теперь автоматически разворачивается при добавлении; реализован обход кэша при обновлении плагина.
* Исправлено: Динамическое пространство имен класса провайдера для eval() и автоматическое разворачивание для недавно добавленных провайдеров.
* Исправлено: Используются стабильные компоненты Card/CardBody/CardHeader/CardDivider (больше не экспериментальные) для совместимости с WordPress 6.9+.
* Исправлено: Заменили неопределенный DragHandle на иконку захвата Unicode для переупорядочивания провайдеров перетаскиванием.
* Исправлено: Устранен блокирующий HTTP-запрос, который срабатывал при каждой загрузке страницы.
* Улучшено: Рабочие процессы GitHub Actions обновлены до Node.js 24.
