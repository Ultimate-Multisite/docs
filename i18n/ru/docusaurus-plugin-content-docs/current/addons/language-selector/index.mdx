# Language Selector

Multisite Ultimate: Language Selector on Sign-up

Добавьте поле выбора языка в поток регистрации.

## Installation

1. Загрузите 'ultimate-multisite-language-selector' в каталог '/wp-content/plugins/'
2. Активируйте плагин через меню 'Plugins' в админке сети WordPress

## Usage Guide

С этим бесплатным дополнением вы можете позволить пользователям выбрать язык, который они хотели бы видеть в процессе регистрации и на странице входа.

После [активации вашей лицензии](../../user-guide/getting-started/installing-ultimate-multisite.md) вы сможете установить дополнения Ultimate Multisite, к которым ваша лицензия предоставляет доступ.

Перейдите на страницу настроек Ultimate Multisite, а затем в меню слева нажмите на ярлык _Add-ons_.

![Add-ons page](/img/admin/addons-page.png)

_Перейдите на страницу дополнений, чтобы начать установку Language Selector_

Там вы найдете все дополнения, которые мы предлагаем.

Бесплатные дополнения доступны всем нашим клиентам, независимо от подписки, на которую они подписались.

Нажмите кнопку _Install it Now!_ под миниатюрой дополнения. Затем активируйте его на странице Plugins.

![Add-ons page with install button](/img/admin/addons-page.png)

_Нажмите кнопку, чтобы установить дополнение_

_Активируйте его на странице Plugins_

Теперь перейдите в меню Настройки Ultimate Multisite и откройте вкладку _Network Settings_. Внизу страницы вы найдете новый раздел под названием _Languages_.

Вы можете оставить его пустым, чтобы отобразить все языки в качестве вариантов, или выбрать те, которые хотите сделать доступными. Языки — это те, которые вы установили в своей конфигурации WordPress.

![Settings page](/img/config/settings-general.png)

_Список языков, доступных в этой установке WP_

После сохранения внесенных изменений меню выбора с выбранными вами языками будет отображено на странице входа, а также в процессе регистрации.

## Setting the Language via the URL

Вы также можете изменить язык страницы, обойдя параметр locale в URL. Если вы хотите перенаправить на страницу регистрации на итальянском, например, добавьте ?locale=it_IT к URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Язык должен быть установлен в вашей сети WordPress, чтобы это работало.
