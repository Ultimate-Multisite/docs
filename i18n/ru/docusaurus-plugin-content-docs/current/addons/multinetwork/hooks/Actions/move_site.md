# Действие: move_site

Вызывается после того, как сайт был перемещён в новую сеть.

## Параметры

| Name | Type | Description |
|------|------|-------------|
| $site_id | `int` | ID сайта, который был перемещён. |
| $old_network_id | `int` | ID исходной сети для сайта. |
| $new_network_id | `int` | ID сети, в которую перемещён сайт. |

### С версии

- 1.3.0
