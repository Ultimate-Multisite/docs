# Action: update_network

Вызывается после обновления существующей сети.

## Parameters

| Имя | Тип | Описание |
|------|------|-------------|
| $network_id | `int` | ID добавленной сети. |
| $args | `array` | Ассоциативный массив аргументов сети. |

### Начиная с

- 1.3.0
