# Действие: delete_network

Вызывается после удаления сети.

## Параметры

| Name | Type | Описание |
|------|------|----------|
| $network | `\WP_Network` | Удаленный объект сети. |

### С версии

- 1.3.0

### Источник

Определено в [`inc/functions/network.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-multinetwork/blob/main/inc/functions/network.php#L1494) на строке 1494
