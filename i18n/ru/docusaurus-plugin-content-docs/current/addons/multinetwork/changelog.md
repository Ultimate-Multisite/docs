# Журнал изменений Multi-Network

### 1.3.0 {#130}
* Новое: средство предпросмотра шаблонов сети — просматривайте и предварительно оценивайте шаблоны сети в живой панели перед покупкой.
* Новое: выбор шаблона сети при checkout — клиенты могут выбрать шаблон сети во время процесса регистрации и checkout.
* Новое: SSO по magic link между сетями — клиенты могут беспрепятственно получать доступ к подсетям через аутентификацию по magic-link.
* Новое: клонирование медиа сети — медиа сайта корректно дублируются при клонировании сетей из шаблона.
* Исправление: клонирование сети теперь работает без addon для мультитенантности; права super admin повторно назначаются после клонирования.
* Исправление: создание сети теперь совместимо с WooCommerce и всеми платежными шлюзами.
* Исправление: номер порта теперь включается в генерацию резервного домена.
* Исправление: CSS выбора шаблона перенесен в подключаемую таблицу стилей для корректного отображения во всех темах.
* Исправление: защита от неопределенного ключа `network_id` в области запроса сайта.
* Исправление: проверка разрешений сети перед переключением контекста предотвращает ошибки свойства таблицы `wpdb`.
