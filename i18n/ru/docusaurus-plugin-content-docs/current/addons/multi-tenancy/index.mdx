# Мультиарендность

Распределяйте подсайты по серверам и базам данных для повышения производительности и безопасности

## Требования

- WordPress 5.3 или выше
- PHP 7.4 или выше
- Multisite Ultimate plugin (активный)

## Установка

1. Загрузите файлы аддона в ваш каталог `/wp-content/plugins/`
2. Активируйте плагин через меню 'Plugins' в WordPress
3. Настройте параметры аддона в сетевом админке

## Разработка

### Настройка

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Тестирование

Аддон включает всестороннюю тестовую инфраструктуру:

- **Модульные тесты**: Тесты на основе PHPUnit с тестовым набором WordPress
- **Base Test Class**: Предоставляет вспомогательные методы для общих сценариев тестирования
- **Test Coverage**: Настроено для отслеживания покрытия кода
- **Multisite Testing**: Тесты выполняются в многосайтовой среде

### Стандарты кода

- WordPress Coding Standards: Применяются через PHPCS
- PHP 7.4+ Compatibility: Поддерживаются современные функции PHP
- Static Analysis: Интеграция PHPStan для проверки типов
- Code Modernization: Правила Rector для обновления PHP

### Система сборки

- Asset Processing: Минификация файлов CSS/JS
- Translation: Генерация файлов POT
- Packaging: Создание архива для распространения
- Development Mode: Не минифицированные ресурсы для отладки

## Конфигурация

Аддон поддерживает различные параметры конфигурации, доступные через интерфейс сетевого администрирования.

## Хуки и фильтры

### Действия

- `ultimate-multisite-addon-template_init` - Вызывается после инициализации аддона
- `ultimate-multisite-addon-template_loaded` - Вызывается после загрузки всех файлов аддона

### Фильтры

- `ultimate-multisite-addon-template_settings` - Изменяет настройки аддона
- `ultimate-multisite-addon-template_enabled` - Перезаписывает статус включения/выключения аддона
