# Фильтр: wu_mt_syncable_user_meta_keys

Фильтрует список мета-ключей пользователя, которые должны быть синхронизированы с удалёнными узлами.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $syncable_keys | `array` | Ключи мета по умолчанию для синхронизации. |

### С версии

- 1.0.0
