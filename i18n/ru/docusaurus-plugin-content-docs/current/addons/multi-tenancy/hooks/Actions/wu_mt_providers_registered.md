# Action: wu_mt_providers_registered

Вызывается после регистрации стандартных провайдеров.

Используйте этот хук для регистрации пользовательских провайдеров.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $manager | `\Provider_Manager` | Экземпляр менеджера провайдеров. |

### С версии

- 1.0.0
