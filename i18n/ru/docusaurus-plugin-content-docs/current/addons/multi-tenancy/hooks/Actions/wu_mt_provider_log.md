# Action: wu_mt_provider_log

Вызывается, когда провайдер записывает сообщение в журнал.

## Parameters

| Имя | Тип | Описание |
|------|------|-------------|
| $message | `string` | Сообщение журнала. |
| $level | `string` | Уровень журнала. |
| $context | `array` | Дополнительный контекст. |

### С 1.0.0
