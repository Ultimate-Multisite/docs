# Журнал изменений НДС {#vat-changelog}

Версия 1.0.7 - Выпущена 2026-02-03

* Исправление: немецкие (DE) номера НДС не проходили проверку из-за ошибок MS_MAX_CONCURRENT_REQ сервиса VIES SOAP. Выполнен переход с устаревшего SOAP API на официальный EU VIES REST API для более надежной проверки.
* Исправление: номера НДС теперь принимаются, когда сервис VIES временно недоступен, вместо некорректного отклонения. Для последующей проверки создается запись в журнале.
* Исправление: поле НДС/налогового идентификатора теперь видно для всех стран, а не только для членов ЕС. Налоговые идентификаторы стран вне ЕС (например, швейцарские номера CHE) сохраняются для отображения в счете без проверки VIES.
* Исправление: поиск префикса НДС для Греции (GR), Монако (MC) и острова Мэн (IM) не работал из-за некорректной логики поиска в массиве.
* Улучшение: удалена ненужная dev-зависимость wp-cli-bundle, которая могла вызывать фатальные ошибки при использовании вместе с дополнением Site Exporter.
* Улучшение: добавлен комплексный набор модульных тестов (53 теста).

Версия 1.0.6 - Выпущена 2026-01-25

* Исправление: для внутренних B2B-транзакций теперь корректно начисляется НДС. Согласно правилам НДС ЕС, обратное начисление применяется только к трансграничным B2B-транзакциям, а не когда страна клиента совпадает со страной компании.

Версия 1.0.5 - Выпущена 2026-01-22

* Исправление: дополнение не загружалось или работало некорректно.
* Изменение: источник данных ставок НДС переключен с euvatrates.com на репозиторий ibericode/vat-rates для более надежных и активно поддерживаемых данных.
* Исправление: исправлена опечатка в значении опции super_reduced_rates.
* Улучшение: добавлены проверки на null при обработке кода страны для предотвращения ошибок.
* Изменение: удалены встроенные файлы переводов в пользу автоматических обновлений через Traduttore.

Версия: 1.0.3 - Выпущена 2025-09-28

* Переименован префикс в ultimate-multisite; обновлен text domain; повышена версия.

Версия 1.0.0-beta.4 - 2021-09-24

* Добавлено: фильтр wp_ultimo_skip_network_active_check для настроек на основе mu-plugins;

Версия 1.0.0 - 05/08/2021 - Первоначальный выпуск
