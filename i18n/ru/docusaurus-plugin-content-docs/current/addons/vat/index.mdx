# VAT

Multisite Ultimate: Европейский НДС

Добавляет дополнительные функции и инструменты к стандартной системе налогообложения Multisite Ultimate для лучшей поддержки Европейского НДС.  
Автоматически добавляет ставки НДС ЕС на страницу ставок налогов из

## Installation

1. Загрузите `ultimate-multisite-vat` в каталог `/wp-content/plugins/`  
2. Активируйте плагин через меню `Plugins` в WordPress  
3. Перейдите в Ultimate Multisite > Settings > Taxes и включите EU Vat.  
4. Нажмите на `Tax Rates` и обновите ставки EU VAT внизу, затем сохраните ставки налогов.  
5. Готово!
