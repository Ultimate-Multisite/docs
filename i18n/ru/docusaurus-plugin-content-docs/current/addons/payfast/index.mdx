# PayFast

Multisite Ultimate: Payfast Gateway

Adds Payfast - a fast and secure online payments gateway in South Africa - as a Multisite Ultimate payment option.

## Установка

1. Загрузите 'ultimate-multisite-payfast' в каталог '/wp-content/plugins/'
2. Активируйте плагин через меню 'Plugins' в WordPress
3. Следуйте пошаговому мастеру, чтобы настроить плагин

## Руководство по использованию

PayFast — ведущий онлайн‑платёжный процессор для бизнеса в Южной Африке. Он посвящён тому, чтобы помогать онлайн‑торговцам расти, делая транзакции бесшовными и безопасными. По запросу пользователя Ultimate Multisite мы решили создать аддон, который позволит интегрировать PayFast. В этой статье показано, как настроить PayFast в качестве вашего платежного шлюза для получения платежей в вашей сети Ultimate Multisite.

Первый шаг — установить аддон.

# Как установить аддон Payfast:

Перейдите на страницу аддонов и выберите аддон Ultimate Multisite: Payfast Gateway.

![Add-ons page](/img/admin/addons-page.png)

Откроется окно аддона. Нажмите «Установить сейчас».

![Add-ons page with install option](/img/admin/addons-page.png)

После завершения установки вы будете перенаправлены на страницу плагинов. Здесь просто нажмите «Network Activate», и аддон _Payfast_ будет активирован в вашей сети.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

## Настройка PayFast

После установки аддона вы можете перейти в настройки Ultimate Multisite и начать настраивать PayFast.

На странице **Ultimate Multisite settings** вы найдёте вкладку «Payments». В этой вкладке вы увидите **Payfast** вместе с переключателем, чтобы включить или отключить этот платежный шлюз.

![Payment gateway settings](/img/config/settings-payment-gateways.png)Go ahead and click the toggle button to enable it. It will then expand the box to show different fields that you need to populate such as **Mechant ID, Merchant Key and PassPhrase.** You can get this information from your **Payfast** account

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Следующий шаг — войти в вашу учетную запись PayFast, чтобы получить необходимую информацию.

После входа вы сразу увидите необходимые учетные данные в панели управления вашей учетной записью **Payfast**.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->You can simply copy and paste the credential and populate the details on your Ultimate Multisite settings. Then click the **"Save settings** " button to save the changes.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Вот и всё, ваша интеграция PayFast завершена!
