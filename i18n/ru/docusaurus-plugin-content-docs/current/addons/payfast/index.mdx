# PayFast {#payfast}

Ultimate Multisite: Payfast Gateway

Добавляет Payfast — быстрый и безопасный шлюз онлайн-платежей в Южной Африке — как платежную опцию Ultimate Multisite.

## Установка {#installation}

1. Загрузите 'ultimate-multisite-payfast' в директорию '/wp-content/plugins/'
2. Активируйте плагин через меню 'Plugins' в WordPress
3. Следуйте пошаговому мастеру, чтобы настроить плагин

## Руководство по использованию {#usage-guide}

PayFast — ведущий обработчик онлайн-платежей для компаний в Южной Африке. Он помогает онлайн-продавцам развиваться, делая транзакции простыми и безопасными. По запросу пользователя Ultimate Multisite мы решили создать дополнение, которое сделает возможной интеграцию с PayFast. В этой статье будет показано, как настроить PayFast в качестве платежного шлюза для получения платежей в вашей сети Ultimate Multisite.

Первый шаг — установить дополнение.

# Как установить дополнение Payfast: {#how-to-install-the-payfast-add-on}

Перейдите на страницу дополнений и выберите дополнение Ultimate Multisite: Payfast Gateway.

![Страница дополнений](/img/admin/addons-page.png)

Появится окно дополнения. Нажмите Install Now.

![Страница дополнений с опцией установки](/img/admin/addons-page.png)

После завершения установки вы будете перенаправлены на страницу плагинов. Здесь просто нажмите Network Activate, и дополнение _Payfast_ будет активировано в вашей сети.

<!-- Скриншот недоступен: страница плагинов WP с кнопкой Network Activate для дополнения PayFast -->

# Настройка PayFast {#setting-up-payfast}

После установки дополнения вы можете перейти к настройкам Ultimate Multisite и начать настраивать PayFast.

На странице **настроек Ultimate Multisite** вы можете найти вкладку "Платежи". На этой вкладке вы увидите "**Payfast** " вместе с переключателем для включения или отключения этого платежного шлюза.

![Настройки платежного шлюза](/img/config/settings-payment-gateways.png)Нажмите переключатель, чтобы включить его. После этого блок развернется и покажет различные поля, которые нужно заполнить, такие как **ID продавца, ключ продавца и парольная фраза.** Вы можете получить эту информацию в своем **Payfast** Account

<!-- Скриншот недоступен: развернутые настройки PayFast с полями ID продавца, ключ продавца и парольная фраза -->

Следующий шаг — войти в свой Account PayFast, чтобы получить нужную информацию.

После входа вы сразу сможете увидеть необходимые учетные данные в Dashboard вашего **Payfast** Account.

<!-- Скриншот недоступен: Dashboard Account PayFast с учетными данными ID продавца и ключ продавца -->Вы можете просто скопировать и вставить учетные данные, а затем заполнить сведения в настройках Ultimate Multisite. Затем нажмите кнопку **"Сохранить настройки** ", чтобы сохранить изменения.

<!-- Скриншот недоступен: настройки PayFast в Ultimate Multisite с заполненными учетными данными и кнопкой сохранения -->

Вот и все, ваша интеграция платежей PayFast завершена!
