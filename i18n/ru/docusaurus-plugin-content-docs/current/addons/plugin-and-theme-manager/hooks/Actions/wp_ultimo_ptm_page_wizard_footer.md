# Действие: wp_ultimo_ptm_page_wizard_footer

Позволяет разработчикам plugin добавлять скрипты в нижнюю часть страницы

## Параметры

| Название | Тип | Описание |
|------|------|-------------|
| $ | `\WU_Page` | Экземпляр страницы Multisite Ultimate |
