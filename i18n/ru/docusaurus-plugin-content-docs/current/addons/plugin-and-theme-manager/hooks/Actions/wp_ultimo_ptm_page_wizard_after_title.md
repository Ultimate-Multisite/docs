# Action: wp_ultimo_ptm_page_wizard_after_title

Позволяет разработчикам plugin добавлять дополнительные кнопки на страницы списка

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $ | `\WU_Page` | Экземпляр страницы Ultimate Multisite |

### С версии

- 1.8.2
### Исходный код

- Определено в [`views/base/settings.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-plugin-and-theme-manager/blob/main/views/base/settings.php#L22) в строке 22
- Определено в [`views/base/wizard.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-plugin-and-theme-manager/blob/main/views/base/wizard.php#L22) в строке 22
