# Действие: wp_ultimo_ptm_page_edit_footer

Позволяет разработчикам плагинов добавлять скрипты в нижнюю часть страницы

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $ | `object` | Объект, содержащий информацию |
| $ | `\WU_Page` | Экземпляр Multisite Ultimate Page |

### С версии

- 1.8.2
