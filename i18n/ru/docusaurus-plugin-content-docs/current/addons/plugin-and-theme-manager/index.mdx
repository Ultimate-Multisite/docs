# Plugin & Theme Manager

Multisite Ultimate: Plugin and Theme Manager

Редактируйте или скрывайте метаинформацию (заголовок, автора, миниатюру и описание) установленных плагинов и тем. Создавайте категории для плагинов и тем, чтобы ваши пользователи могли фильтровать их, и добавьте красивую пользовательскую страницу плагинов в панель ваших пользователей.

Чтобы использовать плагин, вы можете просто перейти на страницы плагинов или тем в админ-панели сети и нажать «Редактировать информацию о плагине/теме».

## Установка {#installation}

1. Загрузите 'ultimate-multisite-plugin-and-theme-manager' в директорию '/wp-content/plugins/'
2. Активируйте плагин через меню 'Плагины' в админ-панели сети WordPress

## Руководство по использованию {#usage-guide}

Ultimate Multisite: Plugin and Theme Manager — это мощное **премиум**-дополнение Ultimate Multisite, которое дает вам полный контроль над тем, как пользователи вашей сети видят параметры плагинов и тем в вашей сети и взаимодействуют с ними.

Это дополнение служит трем целям:

  1. Оно позволяет полностью изменять метаданные как плагинов, так и тем. Это включает название, описание, имя автора и даже миниатюры!

  2. Оно позволяет распределять плагины и темы по категориям, чтобы ваши пользователи могли легко их сортировать.

  3. Оно дает вам возможность заменить старую скучную страницу плагинов новой страницей в стиле тем с сортировкой по категориям.
