# Действие: wu_page_load

Позволяет разработчикам плагинов добавлять дополнительные хуки к нашим страницам.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $page_id | `string|WP_Ultimo_Site_Exporter\Admin_Pages\Base_Admin_Page` | Идентификатор этой страницы. |
| $page_hook | `string|WP_Ultimo_Site_Exporter\Admin_Pages\Base_Admin_Page` | Генерируемый хук для этой страницы. |

### С версии

- 1.8.2
