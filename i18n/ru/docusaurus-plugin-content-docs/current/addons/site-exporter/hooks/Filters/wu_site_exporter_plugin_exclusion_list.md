# Filter: wu_site_exporter_plugin_exclusion_list

Позволяет разработчикам управлять списком плагинов, которые могут быть исключены из сгенерированного zip.

## Parameters

| Имя | Тип | Описание |
|------|------|-------------|
| $plugin_list | `array` | Плагины, которые будут исключены. |

### С версии

- 2.0.0

### Источник

Определено в [`inc/class-wp-ultimo-site-exporter.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-site-exporter/blob/main/inc/class-wp-ultimo-site-exporter.php#L210) на строке 210


## Возвращает
Список плагинов.
