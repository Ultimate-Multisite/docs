# Фильтр: wp_ultimo_site_exporter_url

Позволяет разработчикам управлять относительным путем к корню плагина.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $dir | `string` | Путь относительно корня плагина, к которому вы хотите получить доступ. |

### С версии

- 0.0.1

### Источник

Определено в [`inc/class-helper.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-site-exporter/blob/main/inc/class-helper.php#L52) на строке 52

## Возвращает
Путь относительно корня плагина.
