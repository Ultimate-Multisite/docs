# Site Exporter Журнал изменений

Версия: 1.0.4 - Выпущено 2025-11-25

* Исправлена совместимость с Ultimate Multisite 2.4.

Версия: 1.0.2 - Выпущено 2025-09-28

* Переименован префикс в ultimate-multisite; обновлен текстовый домен; обновлена версия.

### Версия 1.0.1 - Выпущено 2023-08-09

* Улучшено: теперь гарантируется, что корректный путь к файлу импортера получен.
* Исправлено: отключен кэш объектов, чтобы предотвратить ошибки при импорте.
* Исправлено: устранена проблема с неработающей ссылкой на скачивание плагина импортера.

### Версия 1.0.0 - Выпущено 2022-12-23

* Исправлено: всплывающее окно формы экспорта не загружается;
* Исправлено: файлы Javascript не загружаются на странице экспорта;
* Внутреннее: новый процесс сборки;

### Версия 1.0.0-beta.4 - Выпущено 2022-01-13

* Внутреннее: добавлен генератор хуков и фильтров;
* Внутреннее: добавлены stubs Ultimate Multisite для удобства разработчиков;
