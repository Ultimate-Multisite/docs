# Журнал изменений Site Exporter

Версия: 1.0.6 — выпущена 2026-05-11
* Новое: экспорты сайта теперь включают самозагружающийся index.php, поэтому ZIP можно установить на новый хост без отдельной установки плагина
* Новое: экспорт сети позволяет администраторам экспортировать все подсайты в один архив со страницы администрирования Site Export

Версия: 1.0.5 — выпущена 2026-05-05
* Исправление: добавлен __callStatic в заглушку полифилла WP_CLI для обработки незамоканных методов, таких как get_config, что предотвращает сбои начальной загрузки тестов

Версия: 1.0.4 — выпущена 2025-11-25

* Исправлена совместимость с Ultimate Multisite 2.4.

Версия: 1.0.2 — выпущена 2025-09-28

* Переименован префикс в ultimate-multisite; обновлён текстовый домен; повышена версия.

### Версия 1.0.1 — выпущена 2023-08-09

* Улучшено: теперь гарантируется получение правильного пути к файлу импортера.
* Исправлено: отключён кэш объектов для предотвращения ошибок во время импорта.
* Исправлено: устранена проблема, из-за которой ссылка на скачивание плагина импортера не работала.

### Версия 1.0.0 — выпущена 2022-12-23

* Исправлено: всплывающее окно формы экспорта не загружалось;
* Исправлено: файлы Javascript не загружались на странице экспорта;
* Внутреннее: новый процесс сборки;

### Версия 1.0.0-beta.4 — выпущена 2022-01-13

* Внутреннее: добавлен генератор хуков и фильтров;
* Внутреннее: добавлены заглушки WP Ultimo для удобства разработчиков;
