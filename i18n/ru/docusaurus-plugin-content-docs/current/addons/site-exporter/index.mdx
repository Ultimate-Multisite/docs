# Экспортёр сайтов

## Содержание {#content}

Мы понимаем, насколько важно иметь возможность переносить подсайты внутри вашей сети в установку отдельного сайта и наоборот, если ваш клиент решит мигрировать свой веб-сайт. Дополнение для экспорта и импорта сайтов упрощает эту задачу. Этот инструмент экспортирует и импортирует сайты с помощью zip-файлов, которые также могут служить резервной копией.

## Возможности {#features}

* Экспорт и импорт сайтов

* Простой в использовании интерфейс

* Создание резервной копии сайта

## Документация {#documentation}
**Как экспортировать и импортировать сайт**

## Установка {#installation}

1. Загрузите **ultimate-multisite-site-exporter** в каталог **/wp-content/plugins/**
2. Активируйте plugin через меню **Plugins** в WordPress
3. Следуйте пошаговому мастеру, чтобы настроить plugin

## Руководство по использованию {#usage-guide}

С Ultimate Multisite вы можете экспортировать сайт из своей сети или импортировать сайт в свою multisite- или single-site-установку.

Чтобы начать процесс, перейдите на нашу страницу дополнений и установите дополнение Ultimate Multisite: Site Exporter & Importer.![Страница дополнений](/img/admin/addons-page.png)

![Страница дополнений с вариантом установки](/img/admin/addons-page.png)

Перейдите на страницу Plugins и нажмите **Network Activate** для дополнения, которое вы только что установили.<!-- Скриншот недоступен: страница WP Plugins с кнопкой Network Activate для дополнения Site Exporter -->

Опцию **Export & Import** можно найти в меню Ultimate Multisite слева.

<!-- Скриншот недоступен: меню Ultimate Multisite с опцией Export and Import -->

## Как экспортировать сайт {#how-to-export-a-site}

Чтобы экспортировать сайт из вашей сети, перейдите в **Export & Import** в меню Ultimate Multisite и на вкладке **Export Sites** нажмите **Generate new Site Export**.

<!-- Скриншот недоступен: вкладка Export Sites с кнопкой Generate new Site Export -->

Найдите сайт, который хотите экспортировать, и выберите элементы, которые хотите включить в экспортируемый файл. Вы можете включить темы, plugins и загрузки. Если ваш сайт слишком большой, вы можете выбрать запуск процесса экспорта в фоновом режиме.<!-- Скриншот недоступен: параметры экспорта сайта с поиском сайта и флажками включения тем/plugins/загрузок -->

Нажмите кнопку **Export Site**. Экспортированный сайт будет доступен в списке **Existing Exports**. Нажмите кнопку **Download ZIP**, чтобы скачать файл. Экспортные ZIP-файлы включают самозагружающийся `index.php` и `readme.txt`, поэтому их можно установить на новый хост без предварительной установки отдельного importer plugin.<!-- Скриншот недоступен: список Existing Exports с кнопкой Download ZIP -->

## Как экспортировать всю сеть {#how-to-export-the-whole-network}

Чтобы экспортировать все подсайты вместе, перейдите в **Export & Import** в меню Ultimate Multisite и используйте опцию **Network Export** на экране экспорта. Созданный архив содержит все подсайты в одном ZIP для миграций хоста, пересборки staging-сред или резервных копий на уровне сети.

Экспорт сети может быть большим. Запускайте его, когда трафик низкий, и убедитесь, что на сервере достаточно места для архива и любых включённых загрузок, plugins и тем.

## Как импортировать сайт {#how-to-import-a-site}

Чтобы импортировать сайт, перейдите на вкладку **Import a Site** и загрузите .zip-файл сайта, который хотите импортировать. Вы можете изменить URL на новый или сохранить старый URL, оставив поле пустым. Также вы можете отметить флажок, чтобы удалить .zip-файл из вашей установки после импорта. Затем нажмите **Begin Import**.

<!-- Скриншот недоступен: вкладка Import a Site с полем загрузки zip и кнопкой Begin Import -->

Процесс может занять некоторое время. Вы увидите ожидающий импорт в списке ниже импортера.

<!-- Скриншот недоступен: ожидающий импорт в списке ниже импортера -->

Вы можете остановить процесс, нажав **Cancel Import**.

<!-- Скриншот недоступен: кнопка Cancel Import для остановки процесса импорта -->

После завершения процесса ваш сайт уже будет доступен в вашей сети.

## Как импортировать сайт в отдельную установку {#how-to-import-a-site-on-a-single-install}

Чтобы импортировать сайт в вашу отдельную установку, сначала установите и активируйте импортер на вашем сайте. Ссылку для его скачивания можно найти на вкладке экспортера.

<!-- Скриншот недоступен: вкладка экспортера со ссылкой для скачивания импортера для отдельного сайта -->

Импортер можно найти в меню **Tools** слева, в подпункте **Export & Import**.

<!-- Скриншот недоступен: меню Tools для отдельного сайта с подменю Export and Import -->

Загрузите .zip-файл сайта, который хотите импортировать. Вы можете изменить URL на новый или сохранить старый URL, оставив поле пустым. Также вы можете отметить флажок, чтобы удалить .zip-файл из вашей установки после импорта. Затем нажмите **Begin Import**.

<!-- Скриншот недоступен: форма импорта для отдельного сайта с загрузкой zip и кнопкой Begin Import -->

Процесс может занять некоторое время. Вы увидите ожидающий импорт в списке ниже импортера.

<!-- Скриншот недоступен: ожидающий импорт для отдельного сайта в списке -->

Вы можете остановить процесс, нажав **Cancel Import**.

<!-- Скриншот недоступен: кнопка Cancel Import для остановки процесса импорта -->

После завершения процесса ваш сайт будет заменён сайтом, который вы только что импортировали.
