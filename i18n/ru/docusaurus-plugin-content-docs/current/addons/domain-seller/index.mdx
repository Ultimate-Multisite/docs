# Domain Seller

**Multisite Ultimate: Domain Seller** превращает вашу сеть WordPress Multisite в мощную платформу для продажи доменов. Позвольте клиентам регистрировать собственные домены прямо во время регистрации или из их Dashboard, с проверкой доступности в реальном времени и прозрачными ценами.

### Почему Domain Seller?

Регистрация доменов — один из самых востребованных дополнительных предложений для любой платформы WaaS (Website as a Service). Вместо того чтобы отправлять клиентов к стороннему регистратору, вы можете предлагать домены напрямую — сохраняя больше дохода в своей сети и обеспечивая клиентам удобный опыт без лишних препятствий.

### Ключевые возможности

#### Поддержка нескольких регистраторов доменов
* **OpenSRS** — Полная регистрация, продление и управление
* **Namecheap** — Полная интеграция с WhoisGuard и тестированием в песочнице
* **HostAfrica** — Поиск доменов, цены, регистрация, перенос, DNS и защита ID через Domains Reseller API
* **Openprovider** — Цены для реселлеров, регистрация, продление, переносы, DNS, приватность WHOIS и синхронизация TLD
* **Hostinger** — Поиск и регистрация доменов с использованием общего интеграционного токена Hostinger
* **Расширяемая архитектура** — Построена на шаблоне Integration Registry, что упрощает добавление новых провайдеров
* **Встроенное тестирование подключения** — Проверяйте учетные данные провайдера и соединение перед запуском

#### Пошаговый мастер настройки
* **Подключение в один клик** — Проводит вас через настройку провайдера, импорт TLD и создание доменного продукта
* **Автоматическая синхронизация TLD** — Импортирует все доступные TLD и оптовые цены от подключенных провайдеров
* **Создание продукта по умолчанию** — Создает готовый к использованию доменный продукт с разумными настройками наценки по умолчанию

#### Гибкие цены на домены
* **Три режима наценки** — Процентная наценка, фиксированная наценка или фиксированная цена для каждого TLD
* **Продукты для конкретных TLD** — Создавайте отдельные доменные продукты для разных групп TLD (.com/.net, .io, коды стран и т. д.)
* **Вводные цены** — Предлагайте скидочную цену на первый год, чтобы привлечь новые регистрации
* **Цены в реальном времени** — Актуальные оптовые цены от провайдеров с интеллектуальным кэшированием
* **Поддержка купонов** — Полная совместимость с кодами скидок Multisite Ultimate

#### Защита приватности WHOIS
* **Настройка для каждого продукта** — Установите приватность WHOIS как отключенную, всегда включенную или на выбор клиента
* **Согласие клиента** — Если выбран вариант на выбор клиента, чекбокс на checkout позволяет клиентам добавить защиту приватности
* **Настраиваемые цены** — Установите цену за год для дополнения приватности или предлагайте его бесплатно
* **Интеграция с регистратором** — Автоматически включает WhoisGuard (Namecheap) или приватность WHOIS (OpenSRS) во время регистрации

#### Единый процесс checkout
* **Три режима домена** — Клиенты выбирают между поддоменом (бесплатно), регистрацией нового домена или привязкой существующего домена, которым они уже владеют
* **Поиск домена в реальном времени** — Проверка доступности на базе AJAX без перезагрузки страницы
* **Умные предложения** — Предложения альтернативных TLD, когда предпочтительный домен занят
* **Прозрачные цены** — Понятное отображение цены регистрации, цены продления и необязательной платы за приватность WHOIS
* **Поля контакта регистранта** — Собирает необходимую информацию о регистранте (имя, адрес, телефон) прямо во время checkout
* **Автоматически сгенерированный URL сайта** — Адрес сайта автоматически формируется на основе выбранного домена

#### Система автопродления
* **Продления, связанные с membership** — Домены автоматически продлеваются, пока membership клиента активен
* **Умное планирование** — Задания продления ставятся в очередь на основе дат истечения доменов
* **Логика повторных попыток** — Интеллектуальные повторные попытки для неудачных попыток продления
* **Email-уведомления** — Настраиваемые оповещения о предстоящих продлениях, успешных продлениях, сбоях и истечениях

#### Управление DNS для клиентов
* **Самостоятельное управление DNS** — Клиенты могут добавлять, редактировать и удалять DNS-записи (A, AAAA, CNAME, MX, TXT) из своего Dashboard
* **Статус домена** — Статус регистрации, даты истечения и информация о nameserver в реальном времени
* **Автоматическая привязка домена** — Зарегистрированные или существующие домены автоматически привязываются к сайту клиента

#### Инструменты администратора
* **Ручная регистрация домена** — Регистрируйте домены от имени клиентов из панели администратора
* **Мониторинг доменов** — Просматривайте все домены клиентов, статусы и даты истечения на одной странице
* **Логирование по провайдерам** — Отдельные каналы логов для каждого регистратора (например, domain-seller-namecheap.log)
* **Детали регистрации** — Просматривайте информацию о регистранте и регистрационные данные на основной странице редактирования домена

### Поддерживаемые провайдеры

* **OpenSRS** — Полная интеграция с регистрацией, продлением, управлением DNS и приватностью WHOIS
* **Namecheap** — Полная интеграция с регистрацией, продлением, пакетной проверкой доступности и бесплатным WhoisGuard
* **HostAfrica** — Регистрация, продление, переносы, управление DNS, получение EPP-кода, блокировка регистратора и защита ID
* **Openprovider** — Регистрация, продление, переносы, управление DNS, получение EPP-кода, блокировка регистратора, приватность WHOIS и синхронизация TLD для реселлеров
* **Hostinger** — Регистрация, поиск домена, обновления nameserver, блокировка регистратора и приватность WHOIS; продления обрабатываются автопродлением Hostinger
* **GoDaddy** — Регистрация и продление
* **ResellerClub** — Регистрация, продление и управление DNS
* **NameSilo** — Регистрация и продление
* **Enom** — Регистрация, продление и управление DNS

### Технические детали

* **Требуется:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **Архитектура:** шаблон Integration Registry с интерфейсом Domain_Selling_Capability
* **Автозагрузка:** автозагрузчик Jetpack с сопоставлением классов PSR-4
* **Стандарты:** стандарты кодирования WordPress, строгая типизация, подготовленные выражения

## Установка

1. Убедитесь, что **Multisite Ultimate v2.4.12+** установлен и активирован для сети
2. Загрузите plugin в `/wp-content/plugins/ultimate-multisite-domain-seller/`
3. Активируйте plugin для сети из **Plugins** в WordPress Network Admin
4. **Setup Wizard** запускается автоматически при первой активации — следуйте шагам, чтобы:
   * Подключить регистратора доменов
   * Проверить соединение
   * Импортировать доступные TLD и оптовые цены
   * Создать первый доменный продукт с ценовой наценкой
5. Добавьте поле **Выбор домена** в форму оформления заказа в **Ultimate Multisite > Checkout Forms**

## Часто задаваемые вопросы

### Какая версия Multisite Ultimate требуется?

Версия 2.4.12 или выше. Addon несовместим с WP Ultimo v1.x или версиями Ultimate Multisite ниже 2.4.12.

### Какие доменные провайдеры поддерживаются?

Поддерживаются девять регистраторов: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo** и **Enom**. Возможности различаются в зависимости от провайдера, поскольку API регистраторов предоставляют разные операции; см. руководство по настройке для обзора по каждому провайдеру. Архитектура Integration Registry в addon позволяет легко добавлять пользовательских провайдеров — см. [руководство разработчика](/developer/integration-guide/custom-registrar).

### Могу ли я предлагать защиту конфиденциальности WHOIS?

Да. У каждого доменного продукта есть настройка конфиденциальности WHOIS с тремя режимами: отключено, всегда включено или выбор клиента. Если установлен режим «Выбор клиента», клиенты видят флажок при оформлении заказа с ценой (или «Бесплатно» для WhoisGuard от Namecheap). Флаг конфиденциальности автоматически передается регистратору во время регистрации.

### Могу ли я установить разные цены для разных TLD?

Да. Создайте отдельные доменные продукты для разных групп TLD. У каждого продукта может быть собственный тип наценки (процент, фиксированная наценка или фиксированная цена), поддерживаемые TLD и вводная цена.

### Какие варианты оформления заказа видят клиенты?

Поле «Выбор домена» показывает клиентам три вкладки: **Поддомен** (использовать бесплатный поддомен в вашей сети), **Зарегистрировать новый домен** (найти и зарегистрировать новый домен) и **Существующий домен** (привязать домен, которым они уже владеют). Появятся только те режимы, которые вы включите.

### Поддерживает ли это автоматическое продление?

Да. Домены могут быть настроены на автоматическое продление и привязаны к статусу членства клиента. Платежи за продление обрабатываются через существующий платежный шлюз клиента.

### Могут ли клиенты управлять своими DNS-записями?

Да. Клиенты получают интерфейс управления DNS в своем dashboard, чтобы добавлять, редактировать и удалять записи A, AAAA, CNAME, MX и TXT. Управление DNS доступно с OpenSRS, ResellerClub, Enom, HostAfrica и Openprovider. DNS Hostinger для размещенных доменов обрабатывается основной интеграцией привязки доменов Hostinger.

### Есть ли мастер настройки?

Да. Мастер настройки запускается автоматически при первой активации и проводит вас через подключение регистратора, импорт TLD и создание первого доменного продукта. Вы также можете повторно запустить мастер из настроек addon.
