# Журнал изменений Domain Seller {#domain-seller-changelog}

Версия 1.3.0 — выпущена 2026-06-02
- Новое: Добавлено предупреждение для администратора сети, когда баланс реселлера HostAfrica становится слишком низким
- Новое: Добавлено автоматическое сопоставление недавно зарегистрированных доменов с сетевым сайтом
- Исправление: Требования к полям регистранта применяются только при регистрации нового домена
- Исправление: Уведомления о контроле баланса теперь можно скрывать
- Исправление: Обеспечено сохранение платежных данных регистранта WooCommerce
- Исправление: Обязательные требования к контактным данным регистранта применяются во время регистрации
- Исправление: Предотвращено создание продуктов регистрации доменов с наценкой 0%
- Исправление: Сохранены выбранные домены и цены в ходе сессии checkout
- Исправление: Улучшено отображение валюты цен на домены HostAfrica
- Исправление: Улучшено поведение атрибута form-action в форме checkout для предотвращения несоответствий query-var в WP-core
- Улучшено: В инструкции по настройке добавлена ссылка на документацию по конфигурации реселлера HostAfrica

Version 1.2.0 - Выпущено 2026-05-25
- Новое: Добавлена HostAfrica как интеграция для продажи доменов с поддержкой checkout, мастера настройки, поиска, TLD/цен, регистрации, продления, трансфера, nameserver, DNS, EPP-кода, registrar lock и ID protection
- Новое: Добавлена Openprovider как интеграция для продажи доменов с поддержкой reseller-цен, регистрации, продления, трансфера, nameserver, DNS, EPP-кода, registrar lock, WHOIS privacy и синхронизации TLD
- Новое: Добавлена Hostinger как интеграция для продажи доменов, использующая общий Hostinger API token из основной интеграции для проверок доступности, регистрации, обновления nameserver, registrar lock и WHOIS privacy
- Улучшено: Добавлены docblock-и для действий и фильтров жизненного цикла домена, чтобы помочь разработчикам с интеграцией
- Улучшено: Метаданные совместимости plugin обновлены до WordPress 7.0 в readme дополнения
- Улучшено: Обновлены шаблоны планирования, используемые для координации предстоящего релиза

Version 1.1.0 - Выпущено 2026-05-08
- Новое: Создание DNS-записей (add_dns_record) реализовано для регистраторов ResellerClub, Enom и OpenSRS
- Исправление: Парсер Default DNS Records теперь допускает токены {DOMAIN} и {SITE_URL}
- Исправление: Слаги поля выбора домена в checkout получили пространство имен, чтобы предотвратить конфликт с site_url

Version 1.0.8 - Выпущено 2026-05-07
- Исправление: Цены доменов ResellerClub теперь получают актуальные себестоимости из правильного API endpoint
