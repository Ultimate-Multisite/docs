### Версия 2.0.0 - 2024-XX-XX

**Основная переделка для Multisite Ultimate v2**

* **Новый:** Полная переделка с современной архитектурой PHP 7.4+
* **Новый:** Бесшовная интеграция с системой оформления заказа Multisite Ultimate v2
* **Новый:** Управление продуктами доменов с гибкими вариантами ценообразования
* **Новый:** Архитектура поддержки нескольких поставщиков доменов
* **Новый:** Автообновление и интеграция подписки
* **Новый:** Интерфейс управления доменами для клиентов
* **Новый:** Мониторинг доменов и журналы для администраторов
* **Новый:** Поддержка купонов для продуктов доменов
* **Новый:** Полное управление настройками
* **Новый:** Расширяемый код, дружественный для разработчиков
* **Улучшено:** Обновлён поставщик OpenSRS с полной поддержкой функций
* **Улучшено:** Современный UI, согласованный с Multisite Ultimate v2
* **Исправлено:** Весь устаревший код v1 обновлён до стандартов v2
* **Удалено:** Совместимость с Legacy v1 (нарушающая совместимость)
