# Действие: wu_domain_seller_register_integrations {#action-wudomainsellerregisterintegrations}

Позволяет другим plugins регистрировать дополнительные интеграции продавцов доменов.

## Параметры {#parameters}

| Имя | Тип | Описание |
|------|------|-------------|
| $registry | `\WP_Ultimo\Integrations\Integration_Registry` | Реестр интеграций. |

### Начиная с версии {#since}

- 2.1.0
