# Действие: wu_centered_content

Позволяет разработчикам плагинов добавлять дополнительный контент внутрь центрального столбца.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $page | `object` | Экземпляр страницы. |

### Начиная с

- 2.1.0
