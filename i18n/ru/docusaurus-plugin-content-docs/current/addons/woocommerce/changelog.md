# WooCommerce Integration Changelog

Version 2.0.6 - Released on 2026-01-16
* Улучшение: Включить основные подписки в дополнение. Теперь не требуется расширение Woocommerce Subscriptinos.

Version 2.0.5 - Released on 2026-01-09
* Улучшение: Загрузить переводы из API glotpress.
* Исправление: Фатальная ошибка в некоторых конструкторах страниц.
* Исправление: Бесконечный редирект, когда клиент является членом главного сайта.

Version 2.0.4 - Released on 2025-11-14
* Добавлено: Переводы на многие дополнительные языки.
* Изменено: Переименовано в Ultimate Multisite: Woocommerce Integration.
* Добавлена совместимость с Woocommerce 10.2.1.
* Добавлена совместимость с Woocommerce Subscriptions 7.7.0.
* Исправление: Совместимость с PHP 8.4
* Исправление: Перенаправление, когда страница аккаунта WC не существует.

Version 2.0.3 - Released on 2025-08-13
* Изменено: Включены автоматические обновления с новым маркетплейсом.

Version 2.0.2 - Released on 2025-07-05
* Изменено: Переименовано в Multisite Ultimate: Woocommerce Integration.
* Добавлена совместимость с Woocommerce 9.8.1.
* Добавлена совместимость с Woocommerce Subscriptions 7.3.0.
* Исправление: Отмена подписки клиентом.
* Исправление: Фатальная ошибка при использовании блока оформления заказа.
* Улучшение: Теперь совместим с высокопроизводительными пользовательскими таблицами заказов Woocommerce.
* Исправление: Отмена при оформлении заказа WooCommerce может по-прежнему обновлять членство.

Version 2.0.1 - Released on 2023-08-09

* Добавлена совместимость с Woocommerce 7.9.0.
* Добавлена совместимость с Woocommerce Subscriptions 5.3.0.
* Добавлена поддержка обновлений членства.
* Добавлены уведомления о пробных периодах и сборах за настройку в Woocommerce.
* Добавлена идентификация продуктов Ultimate Multisite Woocommerce с помощью мета-значения.
* Добавлено: Вставлен одноразовый исправление для пометки всех продуктов Ultimate Multisite, связанных с Woocommerce.
* Добавлено: Удалены продукты, созданные Ultimate Multisite, из списка Woocommerce.
* Улучшение: Создан непериодический скидочный купон Woocommerce для применения к корзине.
* Улучшение: Восстановлена периодическая скидка для продукта Woocommerce.
* Улучшение: Добавлен ярлык периодической скидки к продукту Woocommerce.
* Улучшение: Обеспечен тип продукта при оформлении заказа.
* Исправление: Сохранен статус членства во время процесса понижения уровня.
* Исправление: Проверено наличие подписки, чтобы избежать ошибок во время процесса отмены.
* Исправление: Добавлена дата начала подписки для использования в подписках Woocommerce.
* Внутреннее: Реализован новый процесс сборки PHP 8.1.

Version 2.0.0 - Complete rewrite.

* Добавлен метод обработки отмены, чтобы удалить подписку woo при смене шлюза или отмене членства;
* Добавлен обработчик для понижения и повышения уровня членства;
* Улучшение: Загрузка зависимостей woocommerce в форме обновления клиента на подсайтах для разрешения обновления аккаунта;
* Улучшение: Корректная загрузка корзины Woocommerce, если она не существует;
* Улучшение: Убедиться, что мы работаем с таблицами главного сайта при обработке оформления заказа;
* Улучшение: Сделать заказ обновления Ultimo на основе стоимости заказа подписки Woocommerce, а не последнего платежа;
* Исправление: Перейти по ссылке кнопки WU Membership;
* Исправление: Установить заказ Ultimo как оплаченный, когда обновление подписки Woocommerce оплачено;
* Сборка: Добавить MPB в качестве конструктора;

Version 2.0.0-beta-5 - Released on 2022-01-21

* Внутреннее: Добавлен генератор хуков и фильтров;
* Внутреннее: Добавлены заглушки Ultimate Multisite для удобства разработчика;
* Исправлено: Предотвращено создание нескольких продуктов, когда это не требуется;

Version 2.0.0-beta.4 - 2021-09-23

* Исправление: Требование, чтобы WooCommerce был активен в сети, а не только на главном сайте;
* Улучшение: Добавлен фильтр, позволяющий использовать дополнение как mu-plugin;

Version 2.0.0-beta.3 - 2021-05-28

* Исправление: Контроль доступа к панели управления был слишком строгим;
* Улучшение: Добавлены ссылки справки WooCommerce в верхнее меню Ultimate Multisite;

Version 2.0.0-beta.2 - 2021-05-04

* Улучшение: Создает ожидающие платежи в Ultimo при создании заказа обновления WCS;
* Улучшение: Предзаполняет поля оплаты данными клиента Ultimate Multisite;
* Улучшение: Восстанавливает поля оплаты для шлюзов;

Version 2.0.0-beta.1 - 2021-05-04

* Первый бета-релиз
