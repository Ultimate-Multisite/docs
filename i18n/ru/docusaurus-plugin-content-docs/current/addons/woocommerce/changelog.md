# Журнал изменений интеграции WooCommerce

Версия 2.2.0 - Выпущена 2026-07-01
* Новое: суммы налогов Ultimate Multisite теперь отображаются как отдельные строки сборов WooCommerce при оформлении заказа, делая итоговые суммы налогов понятнее до оплаты.
* Новое: добавлена необязательная настройка "Немедленно приостанавливать членства при неудачных продлениях" и фильтр `wu_woo_suspend_on_payment_failure` для сайтов, которым нужна приостановка из-за неудачного продления во время окна повторных попыток WooCommerce Subscriptions.
* Исправление: согласованы статусы WooCommerce Subscription, которые могли оставаться несинхронизированными с членствами Ultimate Multisite после неудачных или восстановленных продлений.
* Исправление: валюта магазина WooCommerce добавлена в список валют Ultimate Multisite, когда она отсутствовала.
* Исправление: сохранены платежные данные клиента при перенаправлении подписчиков к оформлению заказа WooCommerce.
* Улучшено: добавлена совместимость с Jetpack Autoloader 5.
* Улучшено: очищена генерация релизного пакета, чтобы zip-архивы GitHub и маркетплейса не содержали вложенных промежуточных каталогов и файлов разработки.

Версия 2.0.6 - Выпущена 2026-01-16
* Улучшение: включены основные подписки в дополнение. Больше не требуется расширение Woocommerce Subscriptinos.

Версия 2.0.5 - Выпущена 2026-01-09
* Улучшение: загрузка переводов из glotpress API.
* Исправление: фатальная ошибка в некоторых конструкторах страниц.
* исправление: бесконечное перенаправление, когда клиент является участником основного сайта.

Версия 2.0.4 - Выпущена 2025-11-14
* Добавлено: переводы для гораздо большего числа языков.
* Изменено: переименовано в Ultimate Multisite: Woocommerce Integration.
* Добавлено: совместимость с Woocommerce 10.2.1.
* Добавлено: совместимость с Woocommerce Subscriptions 7.7.0.
* Исправление: совместимость с PHP 8.4
* Исправление: перенаправление, когда страница Account WC отсутствует.

Версия 2.0.3 - Выпущена 2025-08-13
* Изменено: включены автоматические обновления с новым маркетплейсом.

Версия 2.0.2 - Выпущена 2025-07-05
* Изменено: переименовано в Multisite Ultimate: Woocommerce Integration.
* Добавлено: совместимость с Woocommerce 9.8.1.
* Добавлено: совместимость с Woocommerce Subscriptions 7.3.0.
* Исправление: отмена подписки клиентом.
* Исправление: фатальная ошибка при использовании блока оформления заказа.
* Улучшение: теперь совместимо с пользовательскими таблицами заказов Woocommerce высокой производительности.
* Исправление: отмена при оформлении заказа WooCommerce всё ещё может повысить уровень членства.

Версия 2.0.1 - Выпущена 2023-08-09

* Добавлено: совместимость с Woocommerce 7.9.0.
* Добавлено: совместимость с Woocommerce Subscriptions 5.3.0.
* Добавлено: поддержка обновлений членства.
* Добавлено: уведомления о пробных периодах и установочных сборах в Woocommerce.
* Добавлено: идентификация продуктов Ultimate Multisite Woocommerce с помощью метазначения.
* Добавлено: вставлено одноразовое исправление для пометки всех связанных с Ultimate Multisite продуктов Woocommerce.
* Добавлено: созданные Ultimate Multisite продукты удалены из списка Woocommerce.
* Улучшение: создан нерекуррентный купон Woocommerce для применения к корзине.
* Улучшение: восстановлена рекуррентная скидка для продукта Woocommerce.
* Улучшение: к продукту Woocommerce добавлена метка рекуррентной скидки.
* Улучшение: обеспечен тип продукта при оформлении заказа.
* Исправление: сохранён статус членства во время процесса понижения.
* Исправление: добавлена проверка существования подписки, чтобы избежать ошибок во время процесса отмены.
* Исправление: добавлена дата начала подписки для использования в подписках Woocommerce.
* Внутреннее: реализован новый процесс сборки PHP 8.1.

Версия 2.0.0 - Полная переработка.

* Добавлено: метод обработки отмены для удаления woo-подписки при смене шлюза или отмене членства;
* Добавлено: обработчик для понижения и повышения уровней членств;
* Улучшение: загрузка зависимостей woocommerce в форме обновления клиента на подсайтах, чтобы разрешить обновление Account;
* Улучшение: корректная загрузка корзины Woocommerce, если она не существует;
* Улучшение: обеспечение работы с таблицами основного сайта при обработке оформления заказа;
* Улучшение: создание заказа на продление Ultimo на основе значения заказа подписки Woocommerce, а не последнего платежа;
* Исправление: ссылка кнопки перехода к членству WU;
* Исправление: установка заказа Ultimo как оплаченного, когда продление подписок Woocommerce оплачено;
* Сборка: добавлен MPB как сборщик;
