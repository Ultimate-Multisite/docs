# Ultimate Multisite Addons

Дополнения расширяют Ultimate Multisite дополнительными функциями и интеграциями. Большинство дополнений платные и их можно приобрести на [ultimatemultisite.com](https://ultimatemultisite.com).

## Installation

Дополнения можно установить двумя способами:

1. **Через страницу дополнения** -- Перейдите на страницу управления дополнениями вашего сайта в админке WordPress
2. **Ручная загрузка** -- Скачайте из [вашего аккаунта](https://ultimatemultisite.com/my-account/downloads/) и загрузите через Плагины > Добавить новый
