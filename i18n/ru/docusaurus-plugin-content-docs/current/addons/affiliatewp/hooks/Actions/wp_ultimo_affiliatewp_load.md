# Действие: wp_ultimo_affiliatewp_load

Вызывается, когда все зависимости загружены

Позволяет разработчикам плагинов добавлять новую функциональность. Например, поддержку новых провайдеров хостинга и т.д.

### С версии

- 1.0.0
