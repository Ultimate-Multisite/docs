# AffiliateWP

Multisite Ultimate: AffiliateWP Интеграция

Используйте мощный AffiliateWP, чтобы расширить клиентскую базу вашей сети Ultimo Network!

## Installation

1. Загрузите 'ultimate-multisite-git ' в каталог '/wp-content/plugins/'.
2. Активируйте плагин через меню 'Plugins' в WordPress.
3. Следуйте пошаговому мастеру, чтобы настроить плагин.

## Usage Guide

[AffiliateWP](https://affiliatewp.com/) стал основным решением при создании программы аффилированных продаж для вашего онлайн-бизнеса. Ultimate Multisite: AffiliateWP является мостом, который позволит вам создать то же самое для вашей премиум-сети.

## Getting Started

Прежде всего, вам **нужно иметь** [**AffiliateWP**](https://affiliatewp.com/) **активным**! Вы не сможете использовать наш аддон, если у вас нет **плагина, который с ним интегрируется**. Если вы не планируете позволять вашим клиентам использовать его на своих сайтах, AffiliateWP должен быть активен только на вашем основном сайте.

Также вам понадобится оба **Ultimate Multisite** и **Ultimate Multisite: AffiliateWP**, **активированные** **по всей сети**.

## Adding Ultimate Multisite as an AffiliateWP Integration

На вашем основном сайте перейдите в Настройки аффилиатов, и на вкладке Интеграции отметьте Ultimate Multisite. Это сообщит [AffiliateWP](https://affiliatewp.com/) начать отслеживать посещения и конверсии из потока регистрации Ultimate Multisite.

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_Отметьте поле Ultimate Multisite внизу и сохраните_

Это почти всё, что нужно. Посещения, приходящие с реферальных ссылок, будут отслеживаться, и когда пользователь позже сделает свой первый платеж, будет создана конверсия, и комиссия будет отложена аффилиату, ответственному за конверсию.

Однако будет отслеживаться только первый платеж, полученный от вновь созданного пользователя. Чтобы включить повторяющиеся реферальные платежи, вам понадобится [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/).

## Enabling Recurring Referrals

После установки [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/), перейдите во вкладку Настройки аффилиатов → Повторяющиеся рефералы и включите использование повторяющихся рефералов.

<!-- Screenshot unavailable: AffiliateWP Recurring Referrals settings with Enable Recurring Referrals toggle -->

_Просто как можно проще: включите повторяющиеся рефералы, и вы готовы!_

Опять же, это всё, что нужно!

Теперь, каждый раз, когда на реферируемый аккаунт поступает новый платеж, ответственный аффилиат автоматически получит свою долю!
