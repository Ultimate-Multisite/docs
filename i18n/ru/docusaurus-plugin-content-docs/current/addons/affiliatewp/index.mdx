# AffiliateWP

Интеграция Multisite Ultimate: AffiliateWP

Используйте мощный AffiliateWP, чтобы расширить клиентскую базу вашей сети Ultimo!

## Установка

1. Загрузите 'ultimate-multisite-git ' в каталог '/wp-content/plugins/'
2. Активируйте плагин через меню 'Плагины' в WordPress
3. Следуйте пошаговому мастеру, чтобы настроить плагин

## Руководство по использованию

[AffiliateWP](https://affiliatewp.com/) стал основным решением для создания партнёрской программы для вашего онлайн-бизнеса. Ultimate Multisite: AffiliateWP — это мост, который позволит вам создать то же самое для вашей премиальной сети.

## Начало работы

Прежде всего, у вас **должен быть** [**AffiliateWP**](https://affiliatewp.com/) **активен**! Вы не сможете использовать наше дополнение, если у вас нет **плагина, с которым оно интегрируется**. Если вы не планируете позволять вашим клиентам также использовать его на своих сайтах, AffiliateWP должен быть активен только на вашем основном сайте.

Также вам потребуется, чтобы **Ultimate Multisite** и **Ultimate Multisite: AffiliateWP** были **активированы** **по всей сети**.

## Добавление Ultimate Multisite как интеграции AffiliateWP

На вашем основном сайте перейдите в настройки партнёров, затем на вкладке «Интеграции» отметьте Ultimate Multisite. Это укажет [AffiliateWP](https://affiliatewp.com/) начать отслеживать визиты и конверсии из процесса регистрации Ultimate Multisite.

<!-- Скриншот недоступен: вкладка настроек интеграций AffiliateWP с флажком Ultimate Multisite -->

_Отметьте поле Ultimate Multisite внизу и сохраните_

Это практически всё, что требуется. Визиты, приходящие по реферальным ссылкам, будут отслеживаться, и когда этот пользователь позже совершит свой первый платёж, будет создана конверсия, а комиссия будет отложена для партнёра, ответственного за эту конверсию.

Однако отслеживаться будет только первый платёж, полученный от только что созданного пользователя. Чтобы включить повторяющиеся реферальные платежи, вам потребуется [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## Включение повторяющихся реферальных вознаграждений

После установки [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) перейдите в настройки партнёров на вкладку «Повторяющиеся реферальные вознаграждения» и включите использование повторяющихся реферальных вознаграждений.

_Проще некуда: включите повторяющиеся реферальные вознаграждения — и всё готово!_

Повторимся, это всё, что требуется!

Теперь каждый раз, когда на реферальном аккаунте будет получен новый платёж, ответственный партнёр автоматически получит свою долю!
