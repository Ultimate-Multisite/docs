# Журнал изменений Support Tickets

### 1.0.4 - 2026-05-05
* Улучшено: Удалён каталог vendor/ из отслеживания Git (уже покрыт .gitignore), что уменьшило размер репозитория
* Улучшено: Протестировано до WordPress 7.0

### 1.0.3 - 2026-05-01
* Исправлено: Допускаются значения null в nullable-сеттерах модели тикета
* Исправлено: Все тикеты сохраняются в глобальных таблицах всей сети для корректной поддержки multisite
* Исправлено: Скрыты поля сотрудников и исправлено перенаправление на пустую страницу в форме создания тикета клиентом
* Исправлено: Вызовы неопределённой add_meta() заменены на корректную update_meta() для хранения метаданных
* Исправлено: Проверка незарегистрированного права заменена на корректную wu_view_all_support_tickets
* Исправлено: Добавлены отсутствующие AJAX-обработчики для статуса тикета, назначения и действий быстрого редактирования
* Исправлено: Скорректировано имя метода для определения ответа сотрудника в представлениях тикетов
* Исправлено: Объединены дублирующиеся обработчики ответов и согласованы имена действий nonce
* Исправлено: Добавлено отсутствующее представление frontend для shortcode [wu_submit_ticket]
* Исправлено: Добавлен отсутствующий столбец user_id и исправлен метод Support_Ticket::get_user_id()
* Исправлено: Удалена лишняя двойная кавычка в теге option фильтра приоритета
* Исправлено: Добавлена панель управления тикетами в network admin для super admins
* Улучшено: CSS администратора объединён в одну внешнюю таблицу стилей
* Улучшено: Подменю Settings удалено из меню администратора subsite
* Улучшено: Ресурсы frontend условно загружаются только на страницах тикетов поддержки
* Улучшено: Автозагрузчик plugin пропускается, когда корневой автозагрузчик Bedrock уже загрузил зависимости

### 1.0.2 - 2025-12-11
* Добавлено: Полная функциональность просмотра тикетов для администратора и frontend
* Добавлено: AJAX-обработчик для отправки ответов на тикеты
* Добавлено: Поддержка функциональности ответов на тикеты с корректной обработкой формы
* Добавлено: Корректное отображение уведомлений для отправки тикетов и ответов в admin
* Добавлено: Автоматическая связь тикета с текущим пользователем при самостоятельной отправке клиентом
* Добавлено: Улучшение безопасности для предотвращения переопределения владельца тикета клиентами
* Добавлено: Отсутствующие вспомогательные функции (wu_format_date, wu_user_can_view_ticket и т. д.)
* Добавлено: Корректное скачивание и обработка файловых вложений
* Добавлено: Система email-уведомлений для ответов на тикеты и изменений статуса
* Исправлено: Форма ответа на тикет теперь содержит необходимый ID тикета
* Исправлено: Проблемы видимости уведомлений в представлениях тикетов admin
* Исправлено: Синтаксические ошибки в функциях тикетов поддержки
* Улучшено: Структура базы данных с корректными классами запросов и схемы для ответов и вложений
* Улучшено: Миграция на реальные свойства вместо атрибутов для моделей тикетов

### 1.0.1 - 2025-09-28
* Обновлено: Префикс переименован в ultimate-multisite для единообразия
* Обновлено: Стандартизация text domain
* Исправлено: Небольшие исправления ошибок и улучшения

* Первый выпуск
* Полная система управления тикетами
* Многоуровневый контроль доступа
* Система обсуждений в виде цепочек
