# Support Tickets Changelog

### 1.0.2 - 2025-12-11
* Added: Полная функциональность просмотра заявок для админов и фронтенда
* Added: Обработчик AJAX для отправки ответов на заявки
* Added: Поддержка функциональности ответов на заявки с правильной обработкой форм
* Added: Корректное отображение уведомлений о подаче заявок и ответах в админке
* Added: Автоматическая привязка заявки к текущему пользователю при самостоятельной подаче клиентом
* Added: Улучшение безопасности для предотвращения переопределения владельца заявки клиентами
* Added: Отсутствующие вспомогательные функции (wu_format_date, wu_user_can_view_ticket и т.д.)
* Added: Корректная загрузка и обработка вложенных файлов
* Added: Система email-уведомлений о ответах на заявки и изменениях статуса
* Fixed: Форма ответа на заявку теперь включает необходимый ID заявки
* Fixed: Проблемы видимости уведомлений в админских просмотрах заявок
* Fixed: Синтаксические ошибки в функциях поддержки заявок
* Improved: Структура базы данных с правильными классами запросов и схем для ответов и вложений
* Improved: Миграция к реальным свойствам вместо атрибутов для моделей заявок
