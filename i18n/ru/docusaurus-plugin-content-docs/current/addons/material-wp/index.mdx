# Material WP {#material-wp}

Интерфейс администратора WordPress оставался неизменным целую вечность. Но задумывались ли вы когда-нибудь, как выглядела бы версия интерфейса администратора в стиле Material Design? 

Material WP — это тема администратора, разработанная, чтобы дать вам такой внешний вид и ощущения, помогая предоставить вашим клиентам более персонализированный Dashboard.

## Возможности {#features}

- Поддержка пользовательского CSS
- Простые настройки стилизации
- Изменение порядка меню
- Простой импорт/экспорт

## Документация {#documentation}

**Настройка панели администратора с помощью Material WP**

## Установка {#installation}

1. Загрузите 'material-wp' в каталог '/wp-content/plugins/'
2. Активируйте плагин через меню 'Плагины' в WordPress
