# Mailchimp

Multisite Ultimate: Mailchimp Интеграция

Интегрируйте вашу сеть Multisite Ultimate с Mailchimp. Вы можете связать определённые планы с Mailchimp списками и автоматически добавлять подписчиков в эти списки при регистрации.

## Installation

1. Загрузите 'ultimate-multisite-mailchimp' в каталог '/wp-content/plugins/'
2. Активируйте плагин через меню 'Plugins' в админке сети WordPress

## Usage Guide

Синхронизация электронных адресов подписчиков с вашими списками MailChimp действительно проста с нашим дополнением Ultimate Multisite: MailChimp Integration. Это руководство поможет вам всё настроить!

## Getting an API key

Прежде чем перейти к доступным настройкам, нам нужен API‑ключ MailChimp. Это позволит Ultimate Multisite общаться с API MailChimp для получения списков, групп и добавления адресов электронной почты в эти списки и группы при регистрации пользователей.

Следуйте этому руководству, чтобы получить собственный API‑ключ MailChimp: [About API Keys | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

![image](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-zimbKYFZo-B4D1E28C-1203-4E9A-AC7B-D8E9B5659B93.png)

_Добавьте ваш API‑ключ MailChimp в_ _**Ultimate Multisite Settings - > Add-on Settings -> MailChimp**_

## Integration Mode

Дополнение MailChimp Integration предлагает два «режима интеграции», что означает, что существует два разных способа использования дополнения для синхронизации адресов электронной почты.

![image](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-mc8dVzW5S-B7A9495E-05B8-4FBB-8954-80EBF4A9CB3B.png)

_Dва режима интеграции для большей гибкости!_

### Mode 1: Множественные списки

Стандартный режим использует **Множественные списки**. Этот режим позволит вам выбрать несколько списков MailChimp для каждого плана. Это лучший вариант, если вы сегментируете пользователей по разным спискам.

Например, предположим, у вас есть 3 разных списка в вашем аккаунте MailChimp: **Newsletter**, **Plan A** и **Plan B**. Вы хотите, чтобы все пользователи, независимо от их плана, также были включены в список Newsletter, чтобы они не пропускали ваши обновления.

Для этого вы можете использовать режим Множественные списки, чтобы добавить оба списка **Newsletter** и **Plan A** в настройки MailChimp для плана A, а также **Newsletter** и **Plan B** в настройки MailChimp для плана B.

![image](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-kZS8-S_eh-4145C1B8-B903-4DFF-9D92-AF7E0A61445E.png)

_Выберите списки по умолчанию для новых аккаунтов и для отменённых аккаунтов_

### Mode 2: Один список + несколько групп

MailChimp позволяет сегментировать ваш список на более мелкие группы пользователей, называемые **Groups**. Это позволяет вам иметь больший контроль, поскольку вы можете отправлять кампанию конкретному подмножеству ваших подписчиков, не создавая несколько копий той же кампании в MailChimp (что затрудняет анализ метрик этой кампании позже).

Например, вы можете иметь один список под названием **My Network Subscribers**, и внутри него иметь группы для **Plan A** и **Plan B**.

![image](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-tOsErbJy6-A468DB31-B539-493E-AF5F-F44A4F90A974.png)

_Выберите списки и группы по умолчанию, если используете режим «один список + несколько групп» интеграции_

## Default Lists and Groups

После того как вы выберете, какой режим интеграции имеет больше смысла для вашего конкретного случая, пришло время выбрать списки по умолчанию (и группы, в зависимости от режима). Список по умолчанию будет использоваться, когда для конкретного плана не выбран конкретный список.

Вы также можете выбрать список по умолчанию для отменённых аккаунтов. Каждый раз, когда пользователь прекращает подписку и удаляет свой аккаунт из вашей сети, его адрес электронной почты будет перемещён в список, назначенный для отменённых аккаунтов.

## Enabling Double Subscription

На той же странице настроек вы сможете включить **Double Subscription**. Эта опция добавит пользователя как в выбранный список по умолчанию, так и в список, выбранный для плана пользователя. Если эта опция не включена, пользователь будет добавлен только в список, выбранный для его плана. Если для конкретного плана не задан список, пользователь будет добавлен в выбранный список по умолчанию.

![image](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-OPUDRnO94-5D4C267A-14EB-43B4-B86F-0328D330B521.png)

_Вы можете подписать ваших пользователей как на список по умолчанию, так и на список плана при регистрации_

## Selecting Lists and Groups for each of the Plans

Перейдите к вашим **Plans** и выберите план, который хотите отредактировать. Обратите внимание, что в мета‑боксе **Advanced Options** появится новая вкладка под названием MailChimp.

На этой вкладке вы сможете выбрать несколько списков или один список и несколько групп (в зависимости от режима интеграции) для этого плана.

![image](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-BMHjubpN5-C3183A30-687C-4837-8350-842CF959BB06.png)

_Выберите списки и группы, которые вы хотите использовать для каждого из ваших планов_

Когда новый пользователь зарегистрируется на этот план, его адрес электронной почты будет добавлен в выбранные списки и группы!
