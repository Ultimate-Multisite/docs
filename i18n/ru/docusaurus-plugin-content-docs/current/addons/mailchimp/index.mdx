# Mailchimp

Интеграция Multisite Ultimate с Mailchimp

Интегрируйте вашу сеть Multisite Ultimate с вашим Mailchimp. Вы можете связать определенные тарифы со списками Mailchimp и автоматически добавлять подписчиков в эти списки при регистрации.

## Установка {#installation}

1. Загрузите 'ultimate-multisite-mailchimp' в директорию '/wp-content/plugins/'
2. Активируйте plugin через меню 'Plugins' в админ-панели сети WordPress

## Руководство по использованию {#usage-guide}

Синхронизировать адреса электронной почты подписчиков с вашими списками MailChimp очень просто с нашим дополнением Ultimate Multisite: MailChimp Integration. Это руководство поможет вам все настроить!

## Получение API-ключа {#getting-an-api-key}

Прежде чем мы перейдем к различным доступным настройкам, нам нужен API-ключ MailChimp. Это позволит Ultimate Multisite взаимодействовать с MailChimp API, чтобы получать списки, группы и добавлять адреса электронной почты в эти списки и группы, когда пользователи регистрируются.

Следуйте этому руководству, чтобы получить собственный API-ключ MailChimp: [Об API-ключах | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

_Добавьте ваш API-ключ MailChimp в_ _**Ultimate Multisite Settings - > Add-on Settings -> MailChimp**_

## Режим интеграции {#integration-mode}

Дополнение MailChimp Integration предлагает два «режима интеграции», то есть есть два разных способа использовать дополнение для синхронизации адресов электронной почты.

<!-- Скриншот недоступен: выбор режима интеграции MailChimp с вариантами Multiple Lists и Single List -->

_Два режима интеграции для большей гибкости!_

### Режим 1: несколько списков {#mode-1-multiple-lists}

Режим по умолчанию использует **несколько списков**. Этот режим позволит вам выбирать несколько списков MailChimp для каждого из тарифов. Это лучший вариант, если вы сегментируете пользователей по разным спискам.

Например, предположим, что у вас есть 3 разных списка в вашем аккаунте MailChimp: **Newsletter** , **Plan A** и **Plan B**. Вы хотите, чтобы все пользователи, независимо от их тарифа, также были включены в список Newsletter, чтобы они не пропускали ваши обновления.

Чтобы добиться этого, вы можете использовать режим нескольких списков, чтобы добавить оба списка **Newsletter** и **Plan A** в настройки MailChimp для тарифа Plan A, а **Newsletter** и **Plan B** — в настройки MailChimp для тарифа Plan B.

<!-- Скриншот недоступен: выбор списков MailChimp по умолчанию для новых аккаунтов и отмененных аккаунтов -->

_Выберите списки по умолчанию для новых аккаунтов и для отмененных аккаунтов_

### Режим 2: один список + несколько групп {#mode-2-single-list--multiple-groups}

MailChimp позволяет сегментировать ваш список на меньшие группы пользователей, называемые **группами**. Это дает вам больше контроля, поскольку вы можете отправить кампанию определенному подмножеству ваших подписчиков без необходимости создавать несколько копий одной и той же кампании в MailChimp (из-за чего впоследствии сложнее анализировать метрики этой кампании).

Например, у вас может быть один список под названием **My Network Subscribers,** а внутри него — группы для **Plan A** и **Plan B**.

_Выберите списки и группы по умолчанию, если используете режим интеграции «один список + несколько групп»_

## Списки и группы по умолчанию {#default-lists-and-groups}

После того как вы выберете, какой режим интеграции лучше подходит для вашего конкретного случая использования, пора выбрать списки по умолчанию (и группы, в зависимости от режима). Список по умолчанию будет использоваться, когда для данного тарифа не выбран конкретный список.

Вы также можете выбрать список по умолчанию для отмененных аккаунтов. Каждый раз, когда пользователь прекращает свою подписку и удаляет свой аккаунт из вашей сети, его адрес электронной почты будет перемещен в список, установленный для отмененных аккаунтов.

## Включение двойной подписки {#enabling-double-subscription}

На той же странице настроек вы сможете включить **двойную подписку**. Эта опция добавит пользователя и в выбранный список по умолчанию, И в список, выбранный для тарифа этого пользователя. Если эта опция не включена, пользователь будет добавлен только в список, выбранный для тарифа пользователя. Если для этого конкретного тарифа ничего не задано, пользователь будет добавлен в выбранный список по умолчанию.

_Вы можете подписывать ваших пользователей и на список по умолчанию, и на список тарифа при регистрации_

## Выбор списков и групп для каждого из тарифов {#selecting-lists-and-groups-for-each-of-the-plans}

Перейдите в ваши **тарифы** и выберите тариф, который хотите отредактировать. Обратите внимание, что в метабоксе **Advanced Options** будет доступна новая вкладка под названием MailChimp.

На этой вкладке вы сможете выбрать несколько списков или один список и несколько групп (в зависимости от режима интеграции) для этого тарифа.

<!-- Скриншот недоступен: вкладка MailChimp на странице редактирования тарифа с выбором списков и групп -->

_Выберите списки и группы, которые хотите использовать для каждого из ваших тарифов_

Когда новый пользователь зарегистрируется на этот тариф, его адрес электронной почты будет добавлен в выбранные списки и группы!
