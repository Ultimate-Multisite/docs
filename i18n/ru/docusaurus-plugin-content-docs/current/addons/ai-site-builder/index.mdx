# AI Site Builder

ultimate-multisite-ai-site-builder

## Требования

- WordPress 5.3 или выше
- PHP 7.4 или выше
- Multisite Ultimate plugin (active)

## Установка

1. Загрузите файлы аддона в ваш каталог `/wp-content/plugins/`
2. Активируйте плагин через меню 'Plugins' в WordPress
3. Настройте параметры аддона в админке сети

## Разработка

### Настройка

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Тестирование

Аддон включает всестороннюю тестовую инфраструктуру:

- **Unit Tests**: Тесты на основе PHPUnit с тестовым набором WordPress
- **Base test class**: Предоставляет вспомогательные методы для общих сценариев тестирования
- **Test Coverage**: Настроено отслеживание покрытия кода
- **Multisite Testing**: Тесты запускаются в многосайтовой среде

### Стандарты кода

- **WordPress Coding Standards**: Применяются через PHPCS
- **PHP 7.4+ Compatibility**: Поддерживаются современные функции PHP
- **Static Analysis**: Интеграция PHPStan для проверки типов
- **Code Modernization**: Правила Rector для обновления PHP

### Система сборки

- **Asset Processing**: Минификация файлов CSS/JS
- **Translation**: Генерация POT-файла
- **Packaging**: Создание архива для распространения
- **Development Mode**: Ненарезанные ресурсы для отладки

## Конфигурация

Аддон поддерживает различные параметры конфигурации, доступные через интерфейс админки сети.

## Хуки и фильтры

### Actions

- `ultimate-multisite-ai-site-builder_init` - Вызывается после инициализации аддона
- `ultimate-multisite-ai-site-builder_loaded` - Вызывается после загрузки всех файлов аддона

### Filters

- `ultimate-multisite-ai-site-builder_settings` - Модифицирует настройки аддона
- `ultimate-multisite-ai-site-builder_enabled` - Переопределяет статус включения/выключения аддона
