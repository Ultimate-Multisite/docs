# Журнал изменений Support Agents

Версия: 1.0.13 - Выпущено 2025-09-28

* Переименовать префикс в ultimate-multisite; обновить текстовый домен; обновление версии.

Версия 1.0.12 - Выпущено 2025-09-17

* Исправлено: совместимость с Ultimate Multisite 2.4.4.

Версия 1.0.11 - Выпущено 2023-11-21

* Исправлено: совместимость с Ultimate Multisite 2.3.0.

Версия 1.0.10 - Выпущено 2023-08-30

* Исправлено: удаление массовых Support Agents.

Версия 1.0.9 - Выпущено 2023-08-09

* Добавлено: добавлена возможность читать формы оформления заказа.
* Исправлено: макет формы разрешений Support Agents;
* Исправлено: сообщение предупреждения на форме удаления Support Agent;
* Исправлено: скрыты виджеты панели управления сети для неавторизованных пользователей.
* Исправлено: исправлены возможности редактирования и удаления Support Agents.
* Исправлено: исправлен процесс создания Support Agents.
* Исправлено: предотвращено добавление клиента в качестве Support Agent.
* Исправлено: перенаправление неавторизованных пользователей на страницу редактирования Support Agent в админке.
* Исправлено: улучшен макет формы разрешений Support Agents.
* Внутреннее: реализован новый процесс сборки.

Версия 1.0.8 - Выпущено 2022-01-21

* Внутреннее: добавлен генератор хуков и фильтров;
* Внутреннее: добавлены stubs Ultimate Multisite для удобства разработчика;
* Добавлено: возможности Checkout Form;
* Исправлено: несоответствие названий возможностей между формами Ultimate Multisite и настройками этого дополнения;

Версия 1.0.7 - Выпущено 2021-12-02

* Исправлено: удаление Support Agents не работает;
* Исправлено: массовое удаление Support Agents не работает;

Версия 1.0.6 - Выпущено 2021-11-22

* Исправлено: добавление нового модального окна не работает после оптимизаций производительности ядра;
* Исправлено: представление grid-item присутствует в ядре, но отсутствует в дополнении;

Версия 1.0.5 - Выпущено 2021-10-31

* Исправлено: небольшие несовместимости с PHP 8;
* Улучшено: wp_ultimo_skip_network_active_check теперь использует wu_is_must_use();

Версия 1.0.4 - 2021-09-24

* Добавлено: фильтр wp_ultimo_skip_network_active_check для установок на основе mu-plugins;

Версия 1.0.3 - Выпущено 2021-08-29

* Добавлена поддержка REST API - Support Agents можно создавать через REST;

Версия 1.0.2 - Выпущено 2021-07-26

* Исправлено: обновление прав агента не работает;

Версия 1.0.1 - Выпущено 2021-05-28

* Добавлено: уведомление, когда пользователь уже является супер-админом;
* Добавлено: разрешение на переключение пользователей;
* Улучшено: лучшие поля формы;
