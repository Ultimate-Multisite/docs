# Support Agents

Управлять всей сетью самостоятельно — нелегкая задача. По мере роста сети вы начнете замечать растущий спрос на поддержку. Хорошей идеей будет привлечение людей, которые помогут вам облегчить нагрузку. Однако предоставление полного доступа/разрешений рискованно и может поставить под угрозу вашу сеть. Именно здесь вступает в игру это дополнение Support Agent.

## Features

- Перечисленный список разрешений WordPress и Ultimate Multisite
- Легко включать и отключать доступ
- Создавайте неограниченное количество аккаунтов Agent

## Installation

1. Загрузите 'ultimate-multisite-support-agents' в каталог '/wp-content/plugins/'
2. Активируйте плагин через меню 'Plugins' в WordPress
3. Следуйте пошаговому мастеру, чтобы настроить плагин
