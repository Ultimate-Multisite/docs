# Установка Ultimate Multisite

:::note
В этом руководстве предполагается, что у вас уже установлена и настроена WordPress Multisite. Чтобы узнать, как это сделать, ознакомьтесь с [этим руководством](https://www.wpbeginner.com/glossary/multisite/) от WP Beginner.
:::

## Установка плагина

Ultimate Multisite доступен бесплатно на [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/).

На вашей **Панели администратора сети** перейдите к **Плагины → Добавить новый плагин**.

![Страница добавления нового плагина в панели администратора сети](/img/installation/add-new-plugin.png)

Найдите **"Ultimate Multisite"** (с кавычками для точного совпадения), и он появится первым в списке. Нажмите **Установить сейчас**.

![Результаты поиска, показывающие Ultimate Multisite](/img/installation/search-ultimate-multisite.png)

После установки нажмите **Активировать в сети**, чтобы активировать плагин во всей сети.

![Плагин установлен с кнопкой Network Activate](/img/installation/plugin-installed.png)

После активации вы автоматически будете перенаправлены к мастеру установки.

![Плагин активирован и перенаправлен к мастеру установки](/img/installation/plugin-activated.png)

## Мастер установки

Мастер установки проведет вас через настройку Ultimate Multisite примерно за 10 минут.

### Приветствие

Нажмите **Начать** для начала.

![Экран приветствия мастера установки](/img/installation/wizard-welcome.png)

### Проверка перед установкой

Этот шаг проверяет информацию о системе и установку WordPress, чтобы убедиться, что они соответствуют требованиям Ultimate Multisite. Если всё в порядке, нажмите **Перейти к следующему шагу**.

![Проверка перед установкой, показывающая системные требования](/img/installation/wizard-pre-install-checks.png)

### Установка

Установщик создаст необходимые таблицы базы данных и установит файл `sunrise.php`, который необходим для работы Ultimate Multisite. Нажмите **Установить**, чтобы продолжить.

![Шаг установки, показывающий таблицы базы данных и sunrise.php](/img/installation/wizard-installation.png)

### Ваша компания

Введите информацию о вашей компании и установите валюту по умолчанию. Эти данные будут использоваться во всей вашей платформе WaaS. Нажмите **Продолжить**, когда закончите.

![Шаг конфигурации вашей компании](/img/installation/wizard-your-company.png)

### Контент по умолчанию

Этот шаг позволяет установить предопределённые шаблоны, продукты и другой стартовый контент. Это отличный способ ознакомиться с функциями Ultimate Multisite. Нажмите **Установить**, чтобы добавить контент по умолчанию, или пропустите этот шаг, если предпочитаете начать с нуля.

![Шаг установки контента по умолчанию](/img/installation/wizard-default-content.png)

### Рекомендуемые плагины

При желании установите рекомендуемые сопутствующие плагины. Нажмите **Установить**, чтобы добавить их, или пропустите, чтобы продолжить.

![Шаг рекомендуемых плагинов](/img/installation/wizard-recommended-plugins.png)

### Готово!

Всё! Установка Ultimate Multisite завершена. Теперь вы можете начать создавать вашу платформу Веб-сайт как услуга с **Панелью администратора сети**.

![Экран завершения установки - Готово](/img/installation/wizard-ready.png)

![Панель администратора сети с активированным Ultimate Multisite](/img/installation/network-dashboard.png)

Давайте начнем и повеселимся!
