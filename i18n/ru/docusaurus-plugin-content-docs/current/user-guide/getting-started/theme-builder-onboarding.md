# Онбординг в Theme Builder

Superdav AI Agent v1.12.0 представляет управляемый **процесс онбординга Theme Builder**, который поможет вам создать кастомную блочную тему во время первоначальной настройки. Это заменяет устаревший режим Site Builder на более гибкий подход с помощью агента.

## Что такое процесс онбординга Theme Builder?

Процесс онбординга Theme Builder — это интерактивный мастер настройки, который:

- Проводит вас через решения по дизайну (цвета, типографика, макет)
- Собирает информацию о визуальной идентичности вашего сайта
- Генерирует кастомную блочную тему, адаптированную под ваши нужды
- Автоматически активирует тему по завершении

Этот процесс работает на базе **Setup Assistant agent**, который задает уточняющие вопросы и постепенно создает вашу тему.

## Начало работы с Theme Builder

### Настройка при первом запуске

Когда вы впервые запускаете Superdav AI Agent на новой установке WordPress, вы увидите:

```
Добро пожаловать в Superdav AI Agent!

Что бы вы хотели сделать?
1. Создать кастомную тему (Theme Builder)
2. Использовать существующую тему
3. Пропустить настройку
```

Выберите **"Создать кастомную тему"**, чтобы войти в процесс онбординга Theme Builder.

### Руктивация

Вы также можете начать онбординг Theme Builder в любое время, запросив:

```
"Начать онбординг Theme Builder"
```

или

```
"Помогите мне создать кастомную тему"
```

## Шаги онбординга

### Шаг 1: Выбор режима

Setup Assistant agent спросит о ваших предпочтениях:

```
Как вы хотите создать свою тему?
- В режиме помощи (Я задам вопросы и создам ее за вас)
- В режиме самостоятельной работы (Я покажу вам варианты, а вы решите)
```

**Режим помощи (Guided mode)** рекомендуется большинству пользователей; агент делает рекомендации по дизайну, основываясь на вашей отрасли и целях.

### Шаг 2: Описание сайта

Вам зададут вопросы о вашем сайте:

- **Цель сайта**: Интернет-магазин, блог, портфолио, SaaS и т. д.
- **Целевая аудитория**: Кто ваши посетители?
- **Цвета бренда**: Основные и второстепенные цвета (или "Я не уверен")
- **Тон**: Профессиональный, креативный, игривый, минималистичный и т. д.

Эта информация сохраняется в вашей памяти **site_brief**, на которую агенты ссылаются в будущих сессиях.

### Шаг 3: Решения по дизайн-системе

Агент проведет вас через выбор дизайн-токенов:

- **Типографика**: Семейство шрифтов (с засечками, без засечек, моноширинный) и масштаб размеров
- **Цветовая палитра**: Основные, второстепенные, акцентные и нейтральные цвета
- **Интервалы (Spacing)**: Компактные, стандартные или просторные макеты
- **Анимация (Motion)**: Анимации и переходы (если требуется)

### Шаг 4: Генерация темы

Setup Assistant agent создает каркас вашей кастомной блочной темы, включая:

- `theme.json`, содержащий все ваши дизайн-токены
- Шаблоны блоков для распространенных макетов (главная страница, блог, контакты)
- Кастомные стили блоков, соответствующие вашей дизайн-системе
- Метаданные темы и декларации поддержки WordPress

### Шаг 5: Активация и проверка

Тема активируется автоматически, и вы увидите:

```
✓ Ваша кастомная тема запущена!
  Название темы: [Ваше название сайта] Theme
  Цвета: [Основной] / [Второстепенный]
  Типографика: [Семейство шрифтов]

  Посетите свой сайт, чтобы увидеть новый дизайн.
```

После этого вы можете перейти на свой сайт, чтобы убедиться, что тема отображается корректно.

## Описание сайта и память site_brief

Во время онбординга агент фиксирует спецификацию вашего сайта в категории памяти **site_brief**. Она включает:

- Цель и задачи сайта
- Целевая аудитория
- Цвета и тон бренда
- Дизайнерские предпочтения
- Структура контента

### Почему важен site_brief

В будущих сессиях агенты обращаются к вашему site_brief, чтобы:

- Поддерживать единообразие дизайна при изменениях
- Предлагать функции, соответствующие цели вашего сайта
- Предоставлять контекстно-зависимые рекомендации
- Избегать повторения вопросов настройки

### Просмотр site_brief

Вы можете попросить агента:

```
"Покажи мне мой site brief"
```

или

```
"Что ты знаешь о моем сайте?"
```

Агент отобразит вашу сохраненную спецификацию сайта.

## Кастомизация после онбординга

После завершения онбординга Theme Builder вы можете:

### Использовать навык Design System Aesthetics

Запросить уточнения дизайна:

```
"Усовершенствовать типографику, чтобы она выглядела более современной"
```

или

```
"Отрегулировать цветовую палитру, чтобы она стала теплее"
```

**Design System Aesthetics skill** поможет вам внести целенаправленные обновления дизайна.

### Редактировать theme.json напрямую

Для продвинутых пользователей отредактируйте `/wp-content/themes/[theme-name]/theme.json`, чтобы настроить:

- Токены цвета
- Масштабы типографики
- Значения интервалов
- Определения границ и теней

### Создавать кастомные шаблоны блоков

Используйте блочный редактор WordPress для создания кастомных шаблонов для:

- Макет главной страницы
- Страницы записи блога
- Страницы продукта
- Форм контактов

## Сравнение: Старый vs. Новый онбординг

| Функция | Site Builder (Устаревший) | Theme Builder (Новый) |
|---------|-------------------------|---------------------|
| Метод настройки | Форма на основе мастера | Диалог с агентом |
| Генерация темы | Ограниченные шаблоны | Кастомный каркас |
| Дизайн-токены | Ручной ввод | Управляемые решения |
| Гибкость | Фиксированные опции | Настраиваемый |
| Обновления в будущем | Не учитываются | Сохраняются в site_brief |

## Устранение неполадок

**Процесс онбординга не завершился**
- Перезапустите процесс: "Начать онбординг Theme Builder"
- Убедитесь, что ваша установка WordPress обновлена
- Проверьте, что Setup Assistant agent включен

**Мой site_brief не используется**
- Подтвердите, что у агента есть доступ к памяти
- Попросите агента "вспомнить мой site brief"
- Проверьте, что память включена в настройках

**Сгенерированная тема не соответствует моим предпочтениям**
- Используйте Design System Aesthetics skill для доработки
- Попросите агента "перегенерировать тему с [конкретными изменениями]"
- Отредактируйте theme.json напрямую для точного контроля

## Следующие шаги

После завершения онбординга Theme Builder:

1. **Проверьте сайт**: Посетите свой сайт, чтобы увидеть новую тему
2. **Доработайте дизайн**: Используйте Design System Aesthetics skill для внесения корректировок
3. **Добавьте контент**: Начните создавать контент для вашего сайта
4. **Изучите возможности**: Узнайте о других возможностях агента, таких как scaffold-block-theme и activate-theme
