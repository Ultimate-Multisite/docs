# Возможности конструктора тем: Создание и активация блочных тем

Superdav AI Agent v1.12.0 добавляет две мощные возможности, которые позволяют генерировать и развертывать кастомные блочные темы прямо из чата.

## Обзор

Возможности **scaffold-block-theme** и **activate-theme** позволяют агенту:
- Создавать полноценные блочные темы, готовые к работе, на основе ваших требований
- Автоматически активировать темы на вашем сайте без ручного вмешательства
- Формировать единую визуальную концепцию благодаря управляемым решениям по дизайну

## Создание блочной темы (Scaffold Block Theme)

Возможность **scaffold-block-theme** генерирует новую блочную тему WordPress с полной структурой, включая:

- Конфигурацию `theme.json` с токенами дизайна
- Файлы шаблонов блоков для распространенных макетов
- Пользовательские стили и вариации блоков
- Метаданные темы и декларации поддержки

### Как вызвать

В чате с Superdav AI Agent вы можете запросить генерацию темы:

```
"Create a block theme called 'Modern Agency' with a blue and white color scheme,
sans-serif typography, and a professional layout"
```

Агент выполнит следующие шаги:
1. Собрать ваши предпочтения в дизайне в ходе диалога
2. Сгенерировать полную структуру темы
3. Создать все необходимые файлы темы
4. Подготовить тему к активации

### Ожидаемый результат

После успешного выполнения возможности вы увидите:

- Подтверждение того, что тема была сформирована (scaffolded)
- Название и расположение темы
- Сводку примененных токенов дизайна (цвета, типографика, отступы)
- Статус «Готово к активации»

Пример вывода:
```
✓ Theme "Modern Agency" scaffolded successfully
  Location: /wp-content/themes/modern-agency/
  Colors: Primary #0066CC, Secondary #FFFFFF
  Typography: Inter (sans-serif)
  Status: Ready to activate
```

## Активация темы (Activate Theme)

Возможность **activate-theme** переключает ваш сайт на только что сформированную или существующую блочную тему.

### Как вызвать

После создания базовой структуры темы вы можете активировать ее немедленно:

```
"Activate the Modern Agency theme"
```

Или активировать любую существующую тему:

```
"Switch to the Twentytwentyfour theme"
```

### Ожидаемый результат

При успешной активации:

- Подтверждение активной темы
- Название предыдущей темы (для справки)
- URL сайта, где тема теперь активна
- Любые примечания по настройке, специфичные для темы

Пример вывода:
```
✓ Theme activated successfully
  Active theme: Modern Agency
  Previous theme: Twentytwentyfour
  Live at: https://yoursite.com
  Note: Check your homepage to verify the layout
```

## Рабочий процесс: Создание и активация

Типичный рабочий процесс объединяет обе возможности:

1. **Запрос генерации темы**: "Create a block theme for my SaaS landing page"
2. **Агент формирует тему**: Генерирует файлы и токены дизайна
3. **Просмотр и доработка**: Обсуждение изменений в дизайне, если это необходимо
4. **Активация**: "Activate the theme now"
5. **Проверка**: Посещение вашего сайта для подтверждения, что новый дизайн активен

## Токены дизайна и кастомизация

Темы, созданные с помощью функции scaffolding, используют токены дизайна WordPress (через `theme.json`) для:

- **Цветов**: Основной (Primary), дополнительный (Secondary), акцентный (Accent), нейтральная палитра (Neutral)
- **Типографики**: Семейства шрифтов, размеры, начертания, межстрочный интервал
- **Отступов**: Отступы (Padding), поля (Margin), шкалы промежутков (Gap scales)
- **Границ**: Токены радиуса и ширины
- **Тени**: Уровни возвышения (Elevation levels)

Эти токены централизованы в `theme.json`, что позволяет легко настроить всю вашу систему дизайна из одного файла.

## Ограничения и примечания

- Темы формируются в папке `/wp-content/themes/` и должны соответствовать соглашениям об именовании WordPress.
- Активация требует соответствующих прав доступа на вашем сайте WordPress.
- Пользовательский PHP-код в темах минимален; для сложной функциональности используйте плагины.
- Блочные темы лучше всего работают с WordPress 5.9 и более поздними версиями.

## Устранение неполадок

**Тема не появляется после scaffolding**
- Убедитесь, что каталог темы существует и имеет правильные права доступа.
- Проверьте, что `theme.json` является корректным JSON.
- Убедитесь, что название темы не конфликтует с существующими темами.

**Ошибка активации**
- Подтвердите, что у вас есть права администратора.
- Проверьте, что каталог темы доступен для чтения WordPress.
- Просмотрите журналы ошибок WordPress для получения подробной информации.

**Токены дизайна не применяются**
- Убедитесь, что синтаксис `theme.json` корректен.
- Очистите кэш любых плагинов кэширования.
- Проверьте, что ваша версия WordPress поддерживает используемые вами токены.

## Следующие шаги

После активации темы вы можете:
- Использовать навык **Design System Aesthetics** для доработки типографики, цветов и отступов
- Настраивать стили отдельных блоков через редактор блоков WordPress
- Добавлять пользовательский CSS в файл `style.css` темы
- Создавать пользовательские шаблоны блоков для определенных типов страниц
