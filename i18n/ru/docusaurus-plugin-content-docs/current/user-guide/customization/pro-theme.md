# Настройка вашей Admin Panel с помощью PRO Theme

Часто при работе с клиентами возникает необходимость предоставить нашим клиентам индивидуальный опыт в админ-панели WordPress. Если вы ищете более профессиональный внешний вид и ощущения, PRO Theme может вам помочь.

PRO Theme — это админ-тема, разработанная, чтобы помогать вам при настройке Dashboard ваших клиентов. И она также является частью ядра премиум-плагинов Ultimate Multisite.

После установки перейдите на вашу страницу _Plugins_ и нажмите, чтобы активировать его в вашей сети.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Активируйте его для всей сети на вашей странице Plugins_

_**ПРИМЕЧАНИЕ:**_ _Если вы просто установите его, он будет отображаться у вашего клиента как опция, которую можно отметить, чтобы добавить тему только для этого пользователя._

Теперь ваш интерфейс админ-панели полностью переработан! У вас также появилось новое меню в левой колонке под названием PRO Theme.
