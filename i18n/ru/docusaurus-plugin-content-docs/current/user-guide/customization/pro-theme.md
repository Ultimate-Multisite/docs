# Настройка вашей админ-панели с PRO Theme

Часто при работе с клиентами возникает необходимость предоставить им индивидуальный опыт в админ‑панели WordPress. Если вы ищете более профессиональный внешний вид, PRO Theme может вам помочь.

PRO Theme – это админ‑тема, разработанная, чтобы помочь вам при настройке Dashboard ваших клиентов. И она также является частью ядра премиум‑плагинов Ultimate Multisite.

После установки перейдите на страницу _Plugins_ и нажмите, чтобы активировать его в вашей сети.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-lWHb8N9jkD.png)

_Активируйте его в сети на странице Plugins_

_**ПРИМЕЧАНИЕ:**_ _Если вы просто установите его, он появится у вашего клиента как опция, которую можно отметить, чтобы добавить тему только для этого пользователя._

Теперь ваш админ‑интерфейс полностью переосмыслен! У вас также появилось новое меню в левой колонке под названием PRO Theme.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-EYpOnLd8Lm.png)

_Новый внешний вид вашей админ‑панели_
