# Размещение рекламы в вашей сети с Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector — это **бесплатное дополнение**, которое дает вам инструменты для размещения рекламы в ваших тарифных планах прямо со страницы их редактирования.

После установки Ultimate Multisite: Ad Injector перейдите на страницу _Plugins_, чтобы активировать его.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Нажмите кнопку Network Activate_

Когда дополнение станет активным, метабокс _Advanced Options_ на экране редактирования тарифного плана получит новую опцию под названием _Ad Settings_.

![Ad Settings на экране редактирования тарифного плана](/img/config/product-edit-full.png)

_Теперь вы можете редактировать настройки рекламы для каждого из ваших тарифных планов_

На этой вкладке вы сможете добавить рекламные коды для записей и страниц во фронтенде, до и после содержимого.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_После вставки кода вашей рекламы она будет отображаться в записях и на страницах этого конкретного тарифного плана._

Вы также можете включать рекламу в заголовок страниц администратора для пользователей этого тарифного плана.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Перейдите к опции Admin Panel Ad Code, если хотите показывать рекламу в панели администрирования пользователя_

### Помимо рекламы

Вы также можете использовать Ultimate Multisite: Ad Injector, чтобы предлагать поддержку для тарифных планов высокого уровня.

Просто перейдите к опции _Admin Panel Ads_ и добавьте HTML-код вашего инструмента поддержки.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Добавьте клиентскую поддержку в конкретные тарифные планы_

Так люди смогут связаться с вами прямо из своей панели администратора!
