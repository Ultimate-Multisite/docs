# Перевод Ultimate Multisite

Ultimate Multisite разрабатывается на английском, но плагин полностью переводим и мы активно приветствуем вклад в перевод.

## Переводы ядра плагина

Ядро плагина Ultimate Multisite переводится через официальную **платформу перевода WordPress.org**:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

Это стандартная система перевода сообщества WordPress. Когда переводы достигают достаточного порога одобрения, они автоматически распространяются ко всем пользователям Ultimate Multisite через обновления плагина WordPress.

**Нам нужна ваша помощь!** Если вы говорите на языке, отличном от английского, пожалуйста, рассмотрите возможность внесения переводов. Даже перевод нескольких строк помогает. Чтобы начать:

1. Посетите [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)
2. Выберите ваш язык
3. Нажмите "Внести перевод"
4. Вам понадобится бесплатный аккаунт WordPress.org

Если вы хотите стать редактором перевода для вашего языка (можете одобрять переводы), свяжитесь через [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions).

## Переводы дополнений

Дополнительные плагины переводятся отдельно на:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Переводы дополнений в настоящее время сгенерированы ИИ как отправная точка. Они охватывают все поддерживаемые языки, но могут содержать неестественные формулировки или неточности. Мы приветствуем человеческих переводчиков, чтобы просмотреть и предложить улучшения, чтобы они были более естественными и точными.

## Локальные переводы

Если вы предпочитаете создавать переводы локально для вашего собственного проекта, а не вносить вклад в upstream, есть два варианта:

### Использование Poedit

Вы можете использовать настольное приложение **[Poedit](https://poedit.net)** для перевода файла `.pot`:

1. Скачайте Poedit с [poedit.net](https://poedit.net)
2. Откройте файл `.pot` из `wp-content/plugins/ultimate-multisite/lang/`
3. Переведите строки в интерфейсе Poedit
4. Сохраните — Poedit генерирует файлы `.po` и `.mo`
5. Поместите выходные файлы в папку `lang`

### Использование Loco Translate

Плагин **[Loco Translate](https://wordpress.org/plugins/loco-translate/)** для WordPress позволяет переводить напрямую из панели администратора без использования настольных инструментов. Он автоматически создает и компилирует файлы перевода для вас.
