# Перестановка списка стран Ultimate Multisite

Выпадающий список выбора страны упорядочен по алфавиту на основе английских вариантов названий стран. Это может означать, что иногда домашняя страна вашей компании и/или большинство ваших клиентов находятся в середине огромного списка стран.

Чтобы помочь вам исправить это и добавить вашу страну в начало списка выбора, вы можете использовать следующий фрагмент кода (его можно добавить в mu‑плагин):

```php
add_filter('wu_get_countries', function($countries) {

    // Замените DE на двухбуквенный код вашей страны

    // Пример ниже для Германии.

    unset($countries['DE']);

    // Замените DE на двухбуквенный код вашей страны

    // и название вашей страны.

    // Пример ниже, для Германии.

    $countries = array_merge(array( 'DE' => __('Germany', 'ultimate-multisite'), ), $countries);

    return $countries;

});
```
