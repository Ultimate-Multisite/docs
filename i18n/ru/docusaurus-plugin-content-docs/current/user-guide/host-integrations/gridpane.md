# Интеграция GridPane

## Обзор
GridPane — специализированная панель управления хостингом WordPress, созданная для серьезных профессионалов WordPress. Эта интеграция позволяет автоматически синхронизировать домены и управлять сертификатами SSL между Ultimate Multisite и GridPane.

## Возможности
- Автоматическая синхронизация доменов
- Управление сертификатами SSL
- Автоматическая настройка константы SUNRISE

## Требования
Следующие константы должны быть определены в вашем файле `wp-config.php`:

```php
define('WU_GRIDPANE', true);
define('WU_GRIDPANE_API_KEY', 'your_api_key');
define('WU_GRIDPANE_SERVER_ID', 'your_server_id');
define('WU_GRIDPANE_APP_ID', 'your_app_id');
```

## Инструкции по настройке

### 1. Получите данные API GridPane
1. Войдите в вашу панель управления GridPane
2. Перейдите в «Настройки» > «API»
3. Сгенерируйте ключ API, если у вас его еще нет
4. Скопируйте ваш ключ API

### 2. Получите ID вашего сервера и сайта
1. В панели управления GridPane перейдите в «Сервера»
2. Выберите сервер, на котором размещен ваш мультисайт WordPress
3. Запишите ID сервера (виден в URL или на странице деталей сервера)
4. Перейдите в «Сайты» и выберите ваш сайт WordPress
5. Запишите ID сайта (виден в URL или на странице деталей сайта)

### 3. Добавьте константы в wp-config.php
Добавьте следующие константы в ваш файл `wp-config.php`:

```php
define('WU_GRIDPANE', true);
define('WU_GRIDPANE_API_KEY', 'your_api_key');
define('WU_GRIDPANE_SERVER_ID', 'your_server_id');
define('WU_GRIDPANE_APP_ID', 'your_site_id');
```

### 4. Включите интеграцию
1. В админке WordPress перейдите в Ultimate Multisite > Настройки
2. Перейдите на вкладку «Карта доменов»
3. Прокрутите вниз до «Интеграции хостинга»
4. Включите интеграцию GridPane
5. Нажмите «Сохранить изменения»

## Как это работает
Когда домен привязан в Ultimate Multisite:
1. Интеграция отправляет запрос к API GridPane, чтобы добавить домен на ваш сайт
2. GridPane автоматически обеспечивает выдачу сертификата SSL
3. При удалении привязки домена интеграция удалит домен из GridPane

Интеграция также автоматически обрабатывает константу SUNRISE в вашем файле `wp-config.php`, которая необходима для корректной работы привязки доменов.

## Управление константой SUNRISE
Одной из уникальных особенностей интеграции GridPane является то, что она автоматически откатывает константу SUNRISE в `wp-config.php`, чтобы предотвратить конфликты с собственной системой привязки доменов GridPane. Это обеспечивает совместную работу обеих систем без проблем.

## Устранение неполадок

### Проблемы с подключением к API
- Убедитесь, что ваш ключ API правильный
- Проверьте, что ID вашего сервера и сайта правильные
- Убедитесь, что у вашего аккаунта GridPane есть необходимые разрешения

### Проблемы с сертификатом SSL
- GridPane может занять некоторое время для выдачи сертификатов SSL
- Убедитесь, что ваши домены правильно указывают на IP-адрес вашего сервера
- Проверьте настройки SSL GridPane для вашего сайта

### Домейн не добавлен
- Проверьте журналы Ultimate Multisite на наличие сообщений об ошибках
- Убедитесь, что домен еще не добавлен в GridPane
- Убедитесь, что записи DNS вашего домена правильно настроены
