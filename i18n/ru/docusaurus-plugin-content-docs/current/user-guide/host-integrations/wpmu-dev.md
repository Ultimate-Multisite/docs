# WPMU DEV Интеграция

## Обзор
WPMU DEV — это комплексная платформа WordPress, которая предлагает хостинг, плагины и сервисы для сайтов WordPress. Эта интеграция позволяет автоматически синхронизировать домены и управлять сертификатами SSL между Ultimate Multisite и хостингом WPMU DEV.

## Возможности
- Автоматическая синхронизация доменов
- Управление сертификатами SSL
- Расширенные попытки проверки сертификата SSL

## Требования
Интеграция автоматически определяет, размещаете ли вы сайт на WPMU DEV, и использует встроенный API. Дополнительная настройка не требуется, если вы размещаете сайт на WPMU DEV.

Интеграция проверяет наличие константы `WPMUDEV_HOSTING_SITE_ID`, которая автоматически определяется при размещении на WPMU DEV.

## Инструкции по настройке

### 1. Подтвердите хостинг WPMU DEV
Если вы размещаете сайт на WPMU DEV, необходимые константы уже должны быть определены. Проверьте, что:
1. Константа `WPMUDEV_HOSTING_SITE_ID` определена в вашей среде
2. У вас активная подписка WPMU DEV с доступом к API

### 2. Включите интеграцию
1. В админке WordPress перейдите в Ultimate Multisite > Settings
2. Перейдите на вкладку «Сопоставление доменов»
3. Прокрутите вниз до «Интеграции хостинга»
4. Включите интеграцию WPMU DEV
5. Нажмите «Сохранить изменения»

## Как это работает

### Синхронизация доменов
Когда домен привязан в Ultimate Multisite:
1. Интеграция использует API WPMU DEV для добавления домена в ваш аккаунт хостинга
2. Она также автоматически добавляет www‑версию домена
3. WPMU DEV обрабатывает конфигурацию домена и выдачу сертификата SSL

### Управление сертификатом SSL
Интеграция настроена на увеличение количества попыток проверки сертификата SSL для хостинга WPMU DEV, поскольку выдача и установка сертификатов может занять некоторое время. По умолчанию будет предпринято до 10 попыток проверки сертификата SSL, в отличие от стандартных 5 попыток.

## Важные примечания

### Удаление домена
В настоящее время API WPMU DEV не предоставляет способа удалить домены. Когда привязка домена удаляется в Ultimate Multisite, домен останется в вашем аккаунте хостинга WPMU DEV. При необходимости вам потребуется вручную удалить его из панели управления хостингом WPMU DEV.

### Аутентификация API
Интеграция использует ключ API WPMU DEV, который хранится в базе данных WordPress как опция `wpmudev_apikey`. Он автоматически настраивается при подключении вашего сайта к WPMU DEV.

## Устранение неполадок

### Проблемы с подключением к API
- Проверьте, что ваш сайт корректно подключен к WPMU DEV
- Убедитесь, что опция `wpmudev_apikey` установлена в базе данных WordPress
- Убедитесь, что ваша подписка WPMU DEV активна

### Проблемы с сертификатом SSL
- WPMU DEV может занять некоторое время для выдачи сертификатов SSL (обычно 5–15 минут)
- Интеграция настроена на проверку до 10 раз для сертификатов SSL
- Если сертификаты SSL всё ещё не выдаются после нескольких попыток, свяжитесь со службой поддержки WPMU DEV

### Домейн не добавлен
- Проверьте журналы Ultimate Multisite на наличие сообщений об ошибках
- Убедитесь, что домен ещё не добавлен в WPMU DEV
- Убедитесь, что ваш тарифный план хостинга WPMU DEV поддерживает количество добавляемых доменов
