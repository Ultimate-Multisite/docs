# Closte Integration

## Обзор
Closte — это управляемая платформа хостинга WordPress, построенная на инфраструктуре Google Cloud. Эта интеграция позволяет автоматически синхронизировать домены и управлять сертификатами SSL между Ultimate Multisite и Closte.

## Функции
- Автоматическая синхронизация доменов
- Управление сертификатами SSL
- Поддержка доменов с подстановочными знаками
- Настройка не требуется, если работаете на Closte

## Требования
Следующая константа должна быть определена в вашем файле `wp-config.php`, если вы используете Closte:

```php
define('CLOSTE_CLIENT_API_KEY', 'your_api_key');
```

Эта константа обычно уже определена, если вы размещаете сайт на Closte.

## Инструкции по настройке

### 1. Проверьте ваш ключ API Closte
Если вы размещаете сайт на Closte, константа `CLOSTE_CLIENT_API_KEY` уже должна быть определена в вашем файле `wp-config.php`. Вы можете проверить это, просмотрев ваш файл `wp-config.php`.

### 2. Включите интеграцию
1. В админке WordPress перейдите в Ultimate Multisite > Settings
2. Перейдите на вкладку «Domain Mapping»
3. Прокрутите вниз до «Host Integrations»
4. Включите интеграцию Closte
5. Нажмите «Save Changes»

## Как это работает
Когда домен привязан в Ultimate Multisite:
1. Интеграция отправляет запрос к API Closte, чтобы добавить домен в ваше приложение
2. Closte автоматически обеспечивает выдачу сертификата SSL
3. Когда привязка домена удаляется, интеграция удалит домен из Closte

Интеграция также работает с настройкой интервала проверки DNS в Ultimate Multisite, позволяя настроить, как часто система проверяет распространение DNS и выдачу сертификата SSL.

## Создание записи домена
Эта интеграция гарантирует, что при создании или дублировании сайта запись домена создаётся автоматически. Это особенно важно для интеграции Closte, поскольку создание записи домена запускает API Closte для создания домена и сертификата SSL.

## Устранение неполадок

### Проблемы с подключением к API
- Проверьте, что ваш ключ API Closte корректен
- Убедитесь, что у вашего аккаунта Closte есть необходимые разрешения

### Проблемы с сертификатом SSL
- Closte может занять некоторое время для выдачи сертификатов SSL (обычно 5-10 минут)
- Проверьте, что ваши домены правильно указывают на IP-адрес вашего сервера Closte
- Проверьте записи DNS вашего домена, чтобы убедиться, что они правильно настроены

### Домейн не добавлен
- Проверьте журналы Ultimate Multisite на наличие сообщений об ошибках
- Убедитесь, что домен ещё не добавлен в Closte
- Убедитесь, что записи DNS вашего домена правильно настроены

### Интервал проверки DNS
- Если сертификаты SSL выдаются слишком медленно, вы можете изменить интервал проверки DNS в настройках Domain Mapping
- Значение по умолчанию — 300 секунд (5 минут), но вы можете установить его до 10 секунд для более быстрой проверки во время тестирования
