# Поиск в Интернете

Gratis AI Agent v1.5.0 добавляет возможность **Поиска в Интернете**, которая позволяет помощнику AI получать актуальную информацию из сети во время разговора. Эта функция работает на базе [Brave Search API](https://brave.com/search/api/) или [Tavily API](https://tavily.com/).

## Как это работает {#how-it-works}

Когда поиск в Интернете включен, помощник может автоматически обращаться к вашему настроенному поисковому провайдеру, если он определяет, что вопрос требует текущей или внешней информации — например, последние новости, актуальные цены или документацию, которая могла измениться с момента обучения модели.

Результаты извлекаются в реальном времени и передаются в контекст помощника до того, как он сгенерирует ответ. Помощник указывает, что использовал результаты поиска для ответа на вопрос.

## Включение поиска в Интернете {#enabling-internet-search}

Для использования поиска в Интернете требуется API-ключ от выбранного поискового провайдера. Чтобы настроить его:

1. Перейдите в **Gratis AI Agent → Settings → Advanced**.
2. Найдите выпадающее меню **Internet Search Provider** и выберите **Brave Search** или **Tavily**.
3. Введите ваш API-ключ в соответствующее поле. Ссылки для регистрации отображаются рядом с каждым полем.
4. Нажмите **Save Settings**.

После сохранения ключа функция поиска в Интернете автоматически становится доступной для помощника.

## Brave Search {#brave-search}

### Получение API-ключа Brave Search {#obtaining-a-brave-search-api-key}

1. Посетите [Brave Search API page](https://brave.com/search/api/).
2. Зарегистрируйте план. Доступен бесплатный уровень с ограничением запросов в месяц.
3. Скопируйте ваш API-ключ из панели разработчика Brave Search.
4. Вставьте его в поле **Brave Search API Key** в настройках Gratis AI Agent.

### Лимиты использования {#usage-limits}

Brave Search выставляет счет за использование в зависимости от количества сделанных запросов. Каждый ответ AI, который инициирует поиск, считается одним запросом. Следите за своим использованием в [Brave Search developer dashboard](https://brave.com/search/api/), чтобы избежать непредвиденных списаний.

## Tavily {#tavily}

Superdav AI Agent v1.10.0 добавляет **Tavily** в качестве альтернативного поискового провайдера в Интернете, предлагая более богатые результаты поиска и расширенные возможности для исследований.

### Получение API-ключа Tavily {#obtaining-a-tavily-api-key}

1. Посетите [Tavily API page](https://tavily.com/).
2. Зарегистрируйте учетную запись. Доступен бесплатный уровень с ограничением запросов в месяц.
3. Скопируйте ваш API-ключ из панели управления Tavily.
4. Вставьте его в поле **Tavily API Key** в настройках Gratis AI Agent.

### Лимиты использования {#usage-limits-1}

Tavily выставляет счет за использование в зависимости от количества сделанных вызовов API. Каждый ответ AI, который инициирует поиск, считается одним вызовом. Следите за своим использованием в [Tavily dashboard](https://tavily.com/), чтобы избежать непредвиденных списаний.

## Отключение поиска в Интернете {#disabling-internet-search}

Удалите API-ключ из активного поля поискового провайдера и сохраните изменения. Функция поиска в Интернете больше не будет доступна для помощника.

:::note
Поиск в Интернете увеличивает задержку ответа, потому что помощнику приходится ждать результатов поиска, прежде чем генерировать ответ. Для задач, требующих высокой оперативности, рассмотрите, необходим ли вам поиск в реальном времени или достаточно встроенных знаний помощника.
:::
