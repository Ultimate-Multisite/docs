# Встроенные агенты

Gratis AI Agent v1.9.0 поставляется с пятью встроенными агентами. Каждый из них предварительно настроен сфокусированным набором инструментов, индивидуальным системным промптом и стартовыми подсказками, которые подходят для типичных задач в соответствующей области. Переключение между агентами меняет возможности и стиль ответов помощника — и вам не нужно ничего настраивать.

## Что такое агент?

Каждый агент — это профиль конфигурации с именем, который объединяет:

- **Инструменты (Tools)** — возможности, которые агент может использовать (например, у Контент-писателя есть доступ к функциям создания постов; у Дизайн-студии — доступ к функциям CSS и theme.json).
- **Системный промпт (System prompt)** — инструкции, которые задают тон, приоритеты и ограничения работы агента.
- **Подсказки (Suggestions)** — заранее написанные промпты, которые отображаются в чате, чтобы помочь вам быстро начать работу.

## Доступ к выбору агента

1. Откройте панель **Gratis AI Agent** в боковой панели администратора WordPress.
2. Нажмите на **иконку агента** в левом верхнем углу заголовка чата (иконка меняется в зависимости от активного агента).
3. Откроется оверлей-таблица **Agent Picker**. Каждый агент перечислен с иконкой, названием и однострочным описанием.
4. Нажмите на строку агента, чтобы активировать его. Заголовок чата обновляется немедленно.

Вы также можете переключать агентов во время разговора — системный промпт нового агента начнет действовать со следующего сообщения.

## Пять встроенных агентов

### Content Writer (Контент-писатель)

**Фокус:** Создание и редактирование постов, страниц и контактных форм.

**Доступные инструменты:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`

**Что делает хорошо:**
- Написание и публикация статей в блоге по краткому описанию или плану.
- Создание пакетов страниц для нового сайта.
- Построение контактных и запросных форм.
- Установка главной изображения для постов по URL или поиску.

**Стартовые подсказки:**
- *Напиши статью в блоге объемом 500 слов о преимуществах WordPress multisite.*
- *Создай страницы «О нас», «Услуги» и «Контакты» и опубликуй их.*
- *Добавь форму запроса бронирования на страницу «Контакты».*

---

### Site Builder (Создатель сайтов)

**Фокус:** Создание сайта целиком по одному промпту.

**Доступные инструменты:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`

**Что делает хорошо:**
- Генерация многоэтапного плана создания сайта для описанного типа бизнеса.
- Автономное выполнение каждого этапа — структура, контент, навигация, дизайн.
- Восстановление после ошибок в процессе без необходимости ручного вмешательства.
- Установка рекомендованных плагинов в рамках процесса создания.
- Создание контактных форм прямо из чата (Superdav AI Agent v1.10.0+).

**Стартовые подсказки:**
- *Создай сайт-портфолио фотографа с типом записи «Галерея», страницей бронирования и контактной формой.*
- *Создай сайт ресторана с онлайн-меню, часами работы и формой запроса бронирования столика.*
- *Настрой сайт для фриланс-консультанта со страницами услуг, разделом портфолио и блогом.*
- *Добавь контактную форму на страницу «Контакты» с помощью создателя сайтов.*

---

### Design Studio (Дизайн-студия)

**Фокус:** Визуальная кастомизация — цвета, типографика, CSS и шаблоны блоков.

**Доступные инструменты:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Что делает хорошо:**
- Применение готовых пресетов темы (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog).
- Тонкая настройка глобальной типографики и цветовых палитр через theme.json.
- Внедрение пользовательского CSS для специфических для бренда правок.
- Создание скриншота страницы и проверка ее на наличие проблем с дизайном.

**Стартовые подсказки:**
- *Примени пресет warm-editorial, а затем установи основной цвет на #2d6a4f.*
- *Сделай скриншот главной страницы и скажи, что бы ты улучшил.*
- *Создай многократно используемый шаблон блока «Герой» с фоновым изображением на всю ширину и центрированным заголовком.*

---

### Plugin Manager (Менеджер плагинов)

**Фокус:** Обнаружение, установка и управление плагинами WordPress.

**Доступные инструменты:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Что делает хорошо:**
- Рекомендация лучшего плагина для описанного сценария использования.
- Установка пакетов возможностей из реестра.
- Просмотр доступного каталога возможностей по категориям.

**Стартовые подсказки:**
- *Какой лучший плагин для каталога членства?*
- *Установи пакет возможностей WooCommerce.*
- *Покажи все доступные пакеты возможностей для электронной коммерции.*

---

### Support Assistant (Помощник поддержки)

**Фокус:** Ответы на вопросы о контенте сайта, настройках и конфигурации WordPress.

**Доступные инструменты:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Что делает хорошо:**
- Поиск текущих настроек и опций сайта.
- Объяснение, какие типы записей, таксономии и меню настроены на сайте.
- Ответы на вопросы типа «что делает эта настройка?», с чтением активных значений.
- Выступает в роли режима диагностики только для чтения перед внесением изменений.

**Стартовые подсказки:**
- *Какие плагины и настройки сейчас активны на этом сайте?*
- *Перечисли все пользовательские типы записей, зарегистрированные на этом сайте.*
- *Какие существуют навигационные меню и куда они назначены?*

## Настройка агентов

Каждый встроенный агент можно расширить или заменить с помощью фильтра `gratis_ai_agent_agents`.

### Добавление пользовательского системного промпта к существующему агенту

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### Регистрация нового агента

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

Новый агент появится в Agent Picker сразу после срабатывания фильтра.

### Удаление встроенного агента

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
