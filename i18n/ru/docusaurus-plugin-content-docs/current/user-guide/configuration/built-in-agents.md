# Встроенные агенты {#built-in-agents}

Gratis AI Agent v1.9.0 поставляется с пятью встроенными агентами, каждый из которых предварительно настроен с целевым набором инструментов, адаптированным системным prompt и стартовыми предложениями, соответствующими типовым задачам в этой области. Переключение между агентами меняет то, что ассистент может делать и как он отвечает, — без какой-либо настройки с вашей стороны. Superdav AI Agent v1.18.0 может добавлять в эти рабочие процессы инструменты с учетом расписания, записи напоминаний, шлюзы утверждения и SMS-уведомления, когда настроены связанные интеграции.

## Что такое агент? {#what-is-an-agent}

Каждый агент — это именованный профиль конфигурации, который объединяет:

- **Инструменты** — возможности, которые агенту разрешено вызывать (например, Content Writer имеет доступ к возможностям создания записей; Design Studio имеет доступ к возможностям CSS и theme.json)
- **Системный prompt** — инструкции, задающие тон агента, приоритеты и ограничения
- **Предложения** — заранее написанные prompt, показываемые в интерфейсе чата, чтобы помочь вам быстро начать работу

## Доступ к выбору агента {#accessing-the-agent-picker}

1. Откройте панель **Gratis AI Agent** в боковой панели администратора WordPress.
2. Нажмите **значок агента** в левом верхнем углу заголовка чата (значок меняется, отражая активного агента).
3. **Agent Picker** открывается как наложение в виде таблицы формы. Каждый агент указан со своим значком, названием и описанием в одну строку.
4. Нажмите строку агента, чтобы активировать его. Заголовок чата обновится сразу.

Вы также можете переключать агентов в середине беседы — системный prompt нового агента вступит в силу со следующего сообщения.

## Пять встроенных агентов {#the-five-built-in-agents}

### Content Writer {#content-writer}

**Фокус:** Создание и редактирование записей, страниц и контактных форм.

**Доступные инструменты:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`. При включенных интеграциях Superdav AI Agent v1.18.0 для утвержденных рабочих процессов также могут быть доступны настроенный календарный контекст, шлюзы утверждения, напоминания и инструменты SMS-уведомлений.

**Что он хорошо делает:**
- Подготовка черновиков и публикация записей блога по брифу или плану
- Создание наборов лендингов для нового сайта
- Создание контактных форм и форм запросов
- Установка избранных изображений для записей по URL или поиску
- Подготовка последующих сообщений после события из настроенного контекста Google Calendar, затем пауза для утверждения перед отправкой уведомлений

**Стартовые предложения:**
- *Напишите запись блога на 500 слов о преимуществах WordPress multisite.*
- *Создайте страницы «О нас», «Услуги» и «Контакты» и опубликуйте их.*
- *Добавьте форму запроса на бронирование на страницу «Контакты».*
- *Подготовьте напоминание для участников завтрашнего события из настроенного календаря и дождитесь утверждения перед отправкой.*

---

### Site Builder {#site-builder}

**Фокус:** Полное создание сайта по одному prompt.

**Доступные инструменты:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`. С Superdav AI Agent v1.18.0 могут быть доступны настроенные инструменты управляемого сервиса, утверждения, напоминаний, календаря и SMS там, где администраторы включают их.

**Что он хорошо делает:**
- Генерация многоэтапного плана сборки сайта для описанного типа бизнеса
- Автономное выполнение каждого этапа — структура, контент, навигация, дизайн
- Восстановление после ошибок в середине плана без необходимости ручного вмешательства
- Установка рекомендованных плагинов как часть сборки
- Создание контактных форм непосредственно из интерфейса чата (Superdav AI Agent v1.10.0+)
- Координация напоминаний о запуске или последующих сообщений участникам без дублирующих уведомлений, когда включены шлюзы утверждения и записи напоминаний

**Стартовые предложения:**
- *Создайте сайт-портфолио фотографа с типом записей для галереи, страницей бронирования и контактной формой.*
- *Создайте сайт ресторана с онлайн-меню, часами работы и формой запроса на бронирование столика.*
- *Настройте сайт для внештатного консультанта со страницами услуг, разделом портфолио и блогом.*
- *Добавьте контактную форму на страницу «Контакты» с помощью конструктора сайта.*
- *После утверждения чек-листа запуска сайта отправьте SMS-напоминание настроенному контакту заинтересованной стороны.*

### Design Studio {#design-studio}

**Фокус:** Визуальная настройка — цвета, типографика, CSS и паттерны блоков.

**Доступные инструменты:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Что он хорошо делает:**
- Применение именованных пресетов темы (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- Точная настройка глобальной типографики и цветовых палитр через theme.json
- Внедрение пользовательского CSS для переопределений под бренд
- Создание скриншота страницы и проверка его на проблемы дизайна

**Стартовые предложения:**
- *Примените пресет warm-editorial, а затем установите основной цвет на #2d6a4f.*
- *Сделайте скриншот главной страницы и скажите, что вы бы улучшили.*
- *Создайте многоразовый паттерн hero-блока с фоновым изображением на всю ширину и заголовком по центру.*

### Plugin Manager {#plugin-manager}

**Фокус:** Поиск, установка и управление плагинами WordPress.

**Доступные инструменты:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Что он хорошо делает:**
- Рекомендация лучшего плагина для описанного сценария использования
- Установка пакетов возможностей из реестра
- Просмотр доступного каталога возможностей по категориям

**Стартовые предложения:**
- *Какой лучший плагин для каталога участников?*
- *Установите пакет возможностей WooCommerce.*
- *Покажите все доступные пакеты возможностей ecommerce.*

### Support Assistant {#support-assistant}

**Фокус:** Ответы на вопросы о содержимом сайта, настройках и конфигурации WordPress.

**Доступные инструменты:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Что он делает хорошо:**
- Просмотр текущих настроек и опций сайта
- Объяснение того, какие типы записей, таксономии и меню настроены на сайте
- Ответы на вопросы «что делает эта настройка?» путём чтения актуальных значений
- Работа как диагностический слой только для чтения перед внесением изменений

**Стартовые предложения:**
- *Какие плагины и настройки сейчас активны на этом сайте?*
- *Перечисли все пользовательские типы записей, зарегистрированные на этом сайте.*
- *Какие навигационные меню существуют и куда они назначены?*

## Интеграции автоматизации Superdav {#superdav-automation-integrations}

Когда интеграции Superdav AI Agent v1.18.0 настроены, встроенные агенты могут участвовать в более безопасных рабочих процессах автоматизации с учётом расписания:

- **Инструменты чтения Google Calendar** позволяют агентам проверять настроенные календари и события перед подготовкой последующих действий.
- **Сопоставление контактов и участников** помогает сопоставлять участников события с пользователями WordPress или известными контактами.
- **Шлюзы одобрения человеком** приостанавливают чувствительные действия до тех пор, пока авторизованный пользователь не проверит и не подтвердит их.
- **Записи напоминаний** предотвращают дублирующиеся уведомления, когда запланированные задания повторяются или запускаются повторно.
- **SMS-уведомления TextBee** отправляют настроенные текстовые сообщения только тогда, когда учётные данные SMS и разрешения рабочего процесса включены.

Рекомендуемый рабочий процесс: попросите агента подготовить сообщение или действие, проверьте запрос на одобрение, затем разрешите возобновить одобренное действие. Для повторяющихся напоминаний держите дедупликацию напоминаний включённой, чтобы одно и то же событие или контакт не получали уведомления повторно.

## Настройка агентов {#customising-agents}

Каждый встроенный агент можно расширить или заменить через фильтр `gratis_ai_agent_agents`.

### Добавление пользовательского системного промпта к существующему агенту {#adding-a-custom-system-prompt-to-an-existing-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### Регистрация нового агента {#registering-a-new-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

Новый агент появляется в Agent Picker сразу после выполнения фильтра.

### Удаление встроенного агента {#removing-a-built-in-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
