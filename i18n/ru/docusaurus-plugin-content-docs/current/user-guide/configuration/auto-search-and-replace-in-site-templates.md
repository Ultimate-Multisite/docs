# Автоматический поиск и замена в шаблонах сайтов (v2)

_**Для этого руководства требуется WP UItimo версии 2.x.**_

Одна из самых мощных возможностей Ultimate Multisite — возможность добавлять произвольный текст, цвет и поля выбора в форму регистрации. После того как эти данные получены, мы можем использовать их для предварительного заполнения контента в определённых частях выбранного шаблона сайта. Затем, когда новый сайт будет опубликован, Ultimate Multisite заменит заполнители фактической информацией, введённой во время регистрации.

Например, вы можете создавать свои шаблонные сайты с заполнителями. Заполнители следует добавлять в двойных фигурных скобках — {{placeholder_name}}.

Затем вы можете просто добавить соответствующее поле регистрации, чтобы получить эти данные

После этого ваш клиент сможет заполнить это поле во время регистрации.

Затем Ultimate Multisite автоматически заменит заполнители данными, предоставленными клиентом.

## **Решение проблемы «шаблон, полный заполнителей»** {#solving-the-template-full-of-placeholders-problem}

Всё это отлично, но мы сталкиваемся с неприятной проблемой: теперь наши шаблоны сайтов — которые могут посещать наши клиенты — заполнены некрасивыми заполнителями, которые мало о чём говорят.

Чтобы решить это, мы предлагаем возможность задавать фиктивные значения для заполнителей, и используем эти значения для поиска и замены их содержимого на шаблонных сайтах, пока ваши клиенты их посещают.

Вы можете получить доступ к редактору заполнителей шаблона, перейдя в **Ultimate Multisite > Settings > Sites**, прокрутив до области параметров шаблона сайта, а затем нажав ссылку **Редактировать заполнители**.

![Область параметров шаблона сайта на странице настроек сайтов](/img/config/settings-sites-templates-section.png)

Это приведёт вас в редактор содержимого заполнителей, где вы можете добавить заполнители и соответствующее им содержимое.
