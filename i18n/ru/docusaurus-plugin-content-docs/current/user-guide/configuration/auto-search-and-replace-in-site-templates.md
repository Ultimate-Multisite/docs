# Авто поиск и замена в шаблонах сайтов (v2)

_**Этот учебник требует WP UItimo версии 2.x.**_

Одна из самых мощных функций Ultimate Multisite — возможность добавлять произвольный текст, цвет и поля выбора в форму регистрации. Как только эти данные собраны, мы можем использовать их для предварительного заполнения содержимого определенных частей выбранного шаблона сайта. Затем, когда новый сайт публикуется, Ultimate Multisite заменит заполнители реальной информацией, введённой при регистрации.

Например, вы можете создать шаблоны сайтов с заполнителями. Заполнители должны быть обрамлены двойными фигурными скобками — {{placeholder_name}}.

Затем вы можете просто добавить соответствующее поле регистрации, чтобы захватить эти данные.

Ваш клиент сможет заполнить это поле во время регистрации.

Ultimate Multisite затем автоматически заменит заполнители данными, предоставленными клиентом.

## **Решение проблемы «шаблон, полный заполнителей»**

Всё это замечательно, но у нас возникает неприятная проблема: теперь наши шаблоны сайтов — которые могут посещать наши клиенты — полны некрасивых заполнителей, которые мало что сообщают.

Чтобы решить это, мы предлагаем возможность задать фиктивные значения для заполнителей, и используем эти значения для поиска и замены их содержимого на шаблонах сайтов, пока ваши клиенты их посещают.

Вы можете получить доступ к редактору заполнителей шаблона, перейдя в **Ultimate Multisite > Settings > Sites**, а затем в боковой панели щёлкнув ссылку Edit Placeholders.

![Edit Placeholders link in the Sites settings sidebar](/img/config/settings-sites.png)

Это перенесёт вас в редактор содержимого заполнителей, где вы можете добавить заполнители и их соответствующее содержимое.

![Placeholders content editor with placeholder names and values](/img/config/settings-sites.png)
