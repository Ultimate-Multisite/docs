# Создание кодов скидок (v2)

_**ВАЖНОЕ ПРИМЕЧАНИЕ: эта статья относится к Ultimate Multisite версии 2.x.**_

С Ultimate Multisite вы можете создавать коды скидок, чтобы предоставлять вашим клиентам скидки на их подписки. И создавать их легко!

## Создание и редактирование кодов скидок

Чтобы создать или отредактировать код скидки, перейдите в **Ultimate Multisite > Коды скидок**.

![Список кодов скидок — пустое состояние до создания каких-либо кодов](/img/config/discount-codes-empty.png)

Там вы увидите список кодов скидок, которые уже создали.

Вы можете нажать **Добавить код скидки**, чтобы создать новый купон, или отредактировать имеющиеся, наведя на них курсор и нажав **Редактировать**.

![Список кодов скидок с действиями при наведении, показывающими ссылки Редактировать и Удалить](/img/config/discount-codes-list-hover.png)

![Кнопка Добавить код скидки в заголовке страницы](/img/config/discount-codes-add-button.png)

Вы будете перенаправлены на страницу, где сможете создать или отредактировать код купона. В этом примере мы создадим новый.

![Страница редактирования кода скидки со всеми видимыми разделами](/img/config/discount-code-edit.png)

Давайте рассмотрим доступные здесь настройки:

**Введите код скидки:** Это просто название вашего кода скидки. Это не тот код, который вашим клиентам нужно будет использовать в форме оформления заказа.

**Описание:** Здесь вы можете кратко описать, для чего предназначен этот купон.

![Поля названия и описания кода скидки в верхней части страницы редактирования](/img/config/discount-code-description.png)

Вы также можете просмотреть код скидки как активный или неактивный:

![Активный статус кода скидки](/img/config/discount-code-active.png)

**Код купона:** Здесь вы задаете код, который вашим клиентам нужно будет ввести во время оформления заказа.

![Поле кода купона, куда клиенты вводят код при оформлении заказа](/img/config/discount-code-coupon-field.png)

**Скидка:** Здесь вы можете установить для вашего кода скидки либо **процент**, либо **фиксированную сумму** денег.

![Настройка суммы скидки с выпадающим списком процента или фиксированной суммы](/img/config/discount-code-amount.png)

**Применять к продлениям:** Если эта опция отключена, этот код скидки будет применен только к **первому платежу**. На все остальные платежи скидка не будет распространяться. Если эта опция включена, код скидки будет действителен для всех будущих платежей.

**Скидка на плату за настройку:** Если эта опция отключена, код купона **не даст никакой скидки на плату за настройку** заказа. Если эта опция включена, вы можете задать скидку (процент или фиксированную сумму), которую этот код купона будет применять к плате за настройку ваших планов.

![Опции переключателей применения к продлениям и скидки на плату за настройку](/img/config/discount-code-renewals.png)

**Активен:** Вручную активируйте или деактивируйте этот код купона.

![Переключатель Активен для ручного включения или отключения кода скидки](/img/config/discount-code-active.png)

В разделе **Расширенные параметры** доступны следующие настройки:

![Расширенные параметры кода скидки](/img/config/discount-code-advanced.png)

**Ограничить использование:**

  * **Использований:** Здесь вы можете увидеть, сколько раз был использован код скидки.

  * **Максимум использований:** Это ограничит количество раз, когда пользователи могут использовать этот код скидки. Например, если вы введете здесь 10, купон можно будет использовать только 10 раз. После достижения этого лимита код купона больше нельзя будет использовать.

![Настройка ограничения использования с текущим счетчиком использований и полем максимума использований](/img/config/discount-code-limit-uses.png)

**Даты начала и окончания:** Здесь у вас будет возможность добавить дату начала и/или дату окончания действия купона.

![Поля даты начала и окончания для планирования кода скидки](/img/config/discount-code-dates.png)

**Ограничить продукты:** Если вы включите **Выбрать продукты**, вам будут показаны все ваши продукты. У вас будет возможность вручную выбрать (включая или отключая), какой продукт может принимать этот код купона. Продукты, которые здесь отключены, не покажут никаких изменений, если ваши клиенты попробуют применить к ним этот код купона.

![Раздел ограничения продуктов с переключателями для каждого продукта](/img/config/discount-code-limit-products.png)

После настройки всех этих опций нажмите **Сохранить код скидки**, чтобы сохранить купон, и готово!

![Кнопка Сохранить код скидки в нижней части страницы редактирования](/img/config/discount-code-save.png)

Теперь купон находится в вашем списке, и оттуда вы можете нажать, чтобы **отредактировать или удалить** его.

![Строка кода скидки в списке с действиями Редактировать и Удалить при наведении](/img/config/discount-codes-list-hover.png)

### Использование параметров URL:

Если вы хотите настроить свои таблицы цен или создать красивую страницу кода купона для своего веб-сайта и хотите автоматически применить код скидки к форме оформления заказа, вы можете сделать это с помощью параметров URL.

Сначала вам нужно получить ссылку для общего доступа к вашему плану. Для этого перейдите в **Ultimate Multisite > Продукты** и выберите план.

Нажмите кнопку **Нажмите, чтобы скопировать ссылку для общего доступа**. Это даст вам ссылку для общего доступа к этому конкретному плану. В нашем случае предоставленная ссылка для общего доступа была [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/)_._

![Страница продукта с кнопкой ссылки для общего доступа](/img/config/products-list.png)

Чтобы применить ваш код скидки к этому конкретному плану, просто добавьте параметр **?discount_code=XXX** к URL. Где **XXX** — это код купона.

В нашем примере мы применим код купона **50OFF** к этому конкретному продукту.

URL для этого конкретного плана с примененным кодом скидки 50OFF будет выглядеть так: [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/) _**?discount_code=50OFF**_.
