# Форма оформления заказа

Формы оформления заказа — это простой и гибкий способ экспериментировать с разными подходами при попытке привлечь новых клиентов.

Ultimate Multisite 2.0 предлагает редактор форм оформления заказа, который позволяет создавать столько форм, сколько захотите, с разными полями, предлагаемыми продуктами и т.д.

Чтобы получить доступ к этой функции, перейдите в меню Форма оформления заказа в левой боковой панели.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-TusdCVS7N7.png)

На этой странице вы можете увидеть все имеющиеся у вас формы оформления заказа.

Если хотите создать новую, просто нажмите Добавить форму оформления заказа вверху страницы.

Вы можете выбрать один из этих трёх вариантов в качестве отправной точки: одноступенчатый, многоступенчатый или пустой. Затем нажмите Перейти к редактору.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-OVx7AlxoX5.png)

В качестве альтернативы вы можете редактировать или дублировать уже имеющиеся формы, щёлкнув по пунктам под их названием. Там вы также найдете варианты копирования шорткода формы или удаления формы.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-LdsdAu67WF.png)

### Редактирование формы оформления заказа

Вы можете создавать формы оформления заказа для разных целей. В этом примере мы будем работать с формой регистрации.

После перехода в редактор формы оформления заказа дайте вашей форме имя (будет использоваться только для внутренней ссылки) и slug (используется, например, для создания шорткода).

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-RvbtUn4r3w.png)

Формы состоят из шагов и полей. Вы можете добавить новый шаг, щёлкнув Добавить новый шаг оформления заказа.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-O5GsWoPE35.png)

На первой вкладке модального окна заполните содержимое шага вашей формы. Укажите ему ID, имя и описание. Эти элементы в основном используются внутренне.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-juXHmB0fGH.png)

Далее установите видимость шага. Вы можете выбрать «Всегда показывать», «Показывать только для вошедших пользователей» или «Показывать только для гостей».

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-2gZRyHCxed.png)

Наконец, настройте стиль шага. Это опциональные поля.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-lPHJcayky5.png)

Теперь пришло время добавить поля в наш первый шаг. Просто нажмите Добавить новое поле и выберите тип секции, который хотите.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-z5FpDeUVmD.png)![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-JxtVsd5aGu.png)

У каждого поля есть разные параметры, которые нужно заполнить. Для этого первого ввода мы выберем поле «Имя пользователя».

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-go026QZtjh.png)

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-RroHNTL0TJ.png)

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-OG4O4XkEGu.png)

Вы можете добавить столько шагов и полей, сколько понадобится. Чтобы показать ваши продукты, чтобы клиенты могли выбрать один, используйте поле Таблица цен. Если хотите, чтобы ваши клиенты выбрали шаблон, добавьте поле Выбор шаблона. И так далее.

_**Примечание:** поля имя пользователя, электронная почта, пароль, заголовок сайта, URL сайта, сводка заказа, оплата и кнопка отправки обязательны для создания формы оформления заказа._

Пока вы работаете над своей формой оформления заказа, вы всегда можете использовать кнопку Предпросмотр, чтобы увидеть, как ваши клиенты увидят форму. Вы также можете переключаться между просмотром как существующего пользователя или посетителя

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-kjDtBjBdSH.png)

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-0SKLl73PG5.png)

Наконец, в Дополнительных настройках вы можете настроить сообщение для страницы «Спасибо», добавить фрагменты кода для отслеживания конверсий, добавить пользовательский CSS к вашей форме оформления заказа или ограничить её определёнными странами.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-QC2Q5kIZ9b.png)

Вы также можете вручную включить или отключить вашу форму оформления заказа, переключив эту опцию в правом столбце, или навсегда удалить форму.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-1iRUaR7Gh1.png)

Не забудьте сохранить вашу форму оформления заказа!

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-OKgUbhf41x.png)

Чтобы получить шорткод вашей формы, нажмите Создать шорткод и скопируйте результат, показанный в модальном окне.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-7d68oNc7yI.png)
