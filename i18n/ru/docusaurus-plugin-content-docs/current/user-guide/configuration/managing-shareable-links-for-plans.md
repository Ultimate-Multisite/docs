# Управление совместно используемыми ссылками для планов

Ultimate Multisite предоставляет вам возможность напрямую делиться ссылками на планы с вашими клиентами. Для этого перейдите на страницу редактирования любого из ваших планов и нажмите кнопку _Нажмите, чтобы скопировать совместно используемую ссылку_.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-2DJoMvapXh.png)

_На странице редактирования плана вы можете найти кнопку для создания совместно используемых ссылок_

Обратите внимание, что этот механизм использует частоту выставления счетов по умолчанию для построения ссылки. Это означает, что если ваша частота выставления счетов по умолчанию — ежемесячно, например, в итоге вы получите ссылку вида: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Если вы хотите, чтобы совместно используемая ссылка по умолчанию была ежеквартальной, вам нужно добавить 3 в конце

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Чтобы выбрать план _premium_ с годовым графиком, вы добавите 1, а затем слово:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Имейте в виду, что у вас не может быть только одной ссылки для разных частот выставления счетов. Вы должны делиться отдельной ссылкой для каждой из ваших частот выставления счетов.

После того как вы поделитесь ссылкой со своими клиентами, она перенаправит их на первый шаг вашего процесса регистрации.
