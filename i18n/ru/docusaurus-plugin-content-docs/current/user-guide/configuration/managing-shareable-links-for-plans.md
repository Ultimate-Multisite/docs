# Управление общими ссылками для тарифов

Ultimate Multisite предлагает вам возможность делиться ссылками на тарифы напрямую с вашими клиентами. Чтобы сделать это, перейдите на страницу редактирования любого из ваших тарифов и нажмите кнопку _Нажмите, чтобы скопировать общую ссылку_.

![Страница редактирования продукта с кнопкой общей ссылки](/img/config/product-edit-full.png)

_На странице редактирования тарифа вы можете найти кнопку для создания общих ссылок_

Обратите внимание, что этот механизм использует частоту выставления счетов по умолчанию для построения ссылки. Это означает, что если выставление счетов по умолчанию у вас ежемесячное, например, в итоге вы получите такую ссылку: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Если вы хотите, чтобы общая ссылка по умолчанию вела на ежеквартальную оплату, вам нужно добавить 3 в конце

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Чтобы выбрать тариф _premium_ с ежегодным расписанием, нужно добавить 1, а затем слово:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Имейте в виду, что у вас не может быть только одной ссылки для разных частот выставления счетов. Вам следует делиться отдельной конкретной ссылкой для каждой из ваших частот выставления счетов.

После того как вы поделитесь ссылкой со своими клиентами, она перенаправит их на первый шаг вашего процесса регистрации.
