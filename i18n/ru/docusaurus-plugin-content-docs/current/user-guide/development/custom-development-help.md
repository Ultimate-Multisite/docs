# Как получить помощь по индивидуальной разработке от Codeable?

Иногда нам действительно нужна потрясающая настройка, чтобы наши сетевые потоки работали именно так, как мы хотим. Если это то, что вы ищете, мы рекомендуем [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), место, где вы можете найти опытных разработчиков с экспертизой в Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) — это платформа, соединяющая клиентов с квалифицированными фрилансерами WordPress, способными работать над проектами любого размера. У них тщательный процесс подачи заявок, чтобы вы были сопоставлены с экспертами, специально квалифицированными для задачи, которую вы задумали.

### Как создать проект в Codeable?

Сначала перейдите на [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) и нажмите кнопку **Start a Project**.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Теперь, чтобы отправить проект, нажмите слева **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

В разделе «I need help with» выберите **Customizing** :

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

В разделе «My» выберите **Plugin** :

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Затем нажмите кнопку **Continue**.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Теперь пришло время описать ваш проект и потребности как можно лучше. Постарайтесь предоставить как можно больше информации!

<!-- Screenshot unavailable: Codeable project description form -->

Ответьте на следующие вопросы, чтобы помочь с ценообразованием.

<!-- Screenshot unavailable: Codeable project pricing questions -->

После этого, если у вас еще нет аккаунта на [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), создайте его и нажмите кнопку **Publish My Project**.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

После публикации вашего проекта на [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), платформа свяжет вас с разработчиками, которые соответствуют требуемому набору навыков.

Этот процесс поможет найти подходящего человека для вашей работы, который имеет опыт работы с WordPress и Ultimate Multisite!

Минимальная ставка Codeable составляет $70–$120 USD в час. Оценки будут меняться в зависимости от объема работы, сложности и срочности проекта.

Как только вы получите оценку, вы сможете нанять разработчика, который лучше всего соответствует вашим потребностям, и начать ваш проект вместе!
