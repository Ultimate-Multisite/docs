# Как использовать сниппеты Ultimate Multisite из нашего репозитория GitHub {#how-to-use-ultimate-multisite-snippets-on-our-github-repository}

В репозитории GitHub доступны сниппеты кода, которые часто запрашивают пользователи Ultimate Multisite, желающие добавить небольшие функции, например добавить скрипт Google Analytics на страницы регистрации или скрыть метабокс из панели администратора.

В этой статье показано, как использовать эти коды, а точнее — куда их размещать.

Вы можете найти сниппеты по ссылке ниже.

https://github.com/next-press/wp-ultimo-snippets/

Есть 2 способа добавить код

  1. В файле functions.php вашей темы.

  2. Must-Use Plugins (mu-plugins)

# Как добавить сниппет в файл functions.php вашей темы. {#how-to-add-the-snippet-on-your-themes-functionsphp-file}

  1. Войдите в панель администратора вашей сети WordPress и перейдите в Themes >Theme Editor (см. снимок экрана ниже).

  2. На странице "Edit Themes" убедитесь, что в раскрывающемся поле, расположенном в правом верхнем углу экрана, выбрана ваша активная тема (#3 на снимке экрана ниже).

  3. Нажмите файл functions.php в разделе "Theme Files", чтобы загрузить файл. Прокрутите вниз до конца и вставьте сниппет Ultimate Multisite, который вы получили из репозитория GitHub.

# Как создать Must-Use Plugins (mu-plugins) {#how-to-create-must-use-plugins-mu-plugins}

В WordPress есть функция, которая позволяет загружать пользовательскую функциональность под названием "Must-Use Plugins", или сокращенно "mu-plugins".

Эти специальные mu-plugins загружаются раньше всех остальных обычных плагинов, и их нельзя деактивировать. В multisite-сети код в этих mu-plugins будет загружаться на всех сайтах вашей установки.

1\. Используйте FTP или SSH для доступа к файловой системе вашей установки WordPress.

2\. Внутри каталога wp-content вашей установки WordPress создайте новый каталог с именем: mu-plugins.

<!-- Снимок экрана недоступен: файловый менеджер, показывающий каталог wp-content с папкой mu-plugins -->

3\. Создайте новый PHP-файл на своем компьютере с именем wu-snippet.php, используя Notepad или любой редактор кода.

4\. Поместите сниппет кода Ultimate Multisite, который вы получили из репозитория GitHub, в файл и сохраните его. Вы также можете добавить этот код в начало сниппета кода, чтобы пометить ваш MU plugin.
