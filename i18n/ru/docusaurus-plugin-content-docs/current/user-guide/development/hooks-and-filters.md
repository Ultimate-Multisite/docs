# Ultimate Multisite Горки и фильтры

Ultimate Multisite состоит из множества функций и запросов к базе данных. У него есть горки и фильтры, которые позволяют вам изменять или добавлять новый код к существующим функциям.

Мы создали список этих горок и фильтров как справочник, чтобы помочь продвинутым пользователям дальше настроить свою сеть. Вы можете получить доступ к списку на Github, щёлкнув по ссылке ниже.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
