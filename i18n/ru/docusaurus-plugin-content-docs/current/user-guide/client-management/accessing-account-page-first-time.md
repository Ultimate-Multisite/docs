# Как получить доступ к странице моего аккаунта в первый раз?

Добро пожаловать в Ultimate Multisite!

Получите доступ к странице вашего аккаунта через ссылку, которую вы получили в электронной почте после покупки нашего плагина:

<!-- Screenshot unavailable: Account page email with access link -->

Используйте электронную почту, которую вы указали при покупке, и пароль, который вы получили в письме.

<!-- Screenshot unavailable: Login page with email and password fields -->

Поскольку это ваш первый вход в этот аккаунт, вам будет предложено изменить пароль. Измените его и нажмите кнопку _Save_.

<!-- Screenshot unavailable: Change password form with Save button -->

На странице вашего аккаунта вы можете управлять всеми деталями вашего аккаунта, такими как биллинг, продление, отмены и многими другими вещами.
