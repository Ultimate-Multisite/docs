# Веб-сайты вашего клиента (v2) {#your-clients-websites-v2}

После регистрации в вашей сети ваши клиенты могут получить доступ к своему Dashboard.

Помимо страницы **Account**, где они могут видеть важную информацию о своих платежах, членствах, доменах, ограничениях плана и многом другом, у них также есть доступ к странице **Мои сайты**, где они могут видеть все свои веб-сайты и управлять ими. Всё, что им нужно сделать, — нажать Мои сайты в своём Dashboard.

![Ссылка Мои сайты в Dashboard клиента](/img/account-page/my-sites-link.png)

Им будут показаны все их веб-сайты с кнопкой **Управлять** прямо под каждым из них. Если они нажмут на неё, они будут перенаправлены в **Dashboard этого конкретного веб-сайта**.

Они также могут видеть, какой из их веб-сайтов является **основным** для планов, которые позволяют им иметь более одного веб-сайта.

![Страница Мои сайты, показывающая веб-сайты клиента с кнопками Управлять](/img/account-page/my-sites-page.png)

Вот полный вид интерфейса редактирования сайта:

![Полный интерфейс редактирования сайта](/img/admin/site-edit-full.png)

Если они хотят изменить свой основной веб-сайт, им нужно перейти на страницу Account, нажав на меню **Account**.

![Ссылка меню Account в Dashboard клиента](/img/account-page/account-menu.png)

Вы также можете просматривать и редактировать данные клиента:

![Интерфейс редактирования клиента](/img/admin/customer-edit.png)

Затем им следует нажать **Изменить сайт по умолчанию**.

![Кнопка Изменить сайт по умолчанию на странице Account](/img/account-page/change-default-site.png)

Появится всплывающее окно. Нажмите на раскрывающийся список и выберите, какой веб-сайт вы хотите сделать основным.

![Выбор в раскрывающемся списке Изменить сайт по умолчанию](/img/account-page/change-default-site-dropdown.png)

Затем просто нажмите **Изменить сайт по умолчанию**, и всё готово!

![Подтверждение изменения сайта по умолчанию](/img/account-page/change-default-site-confirm.png)
