# Управление системными письмами

Ultimate Multisite отправляет различные системные письма для таких событий, как подтверждения регистрации, квитанции об оплате и изменения membership. Вы можете настроить эти шаблоны писем в соответствии с вашим брендом.

## Просмотр системных писем {#viewing-system-emails}

Перейдите в **Ultimate Multisite → Settings** и нажмите вкладку **Emails**, чтобы увидеть настройки, связанные с письмами.

![Вкладка Settings - Emails](/img/admin/settings-emails.png)

Оттуда вы можете перейти на страницу **System Emails**, чтобы увидеть все шаблоны писем.

![Список System Emails](/img/admin/system-emails-list.png)

Вы также можете редактировать отдельные шаблоны писем с помощью редактора системных писем:

![Редактор системного письма](/img/admin/system-email-editor.png)

## Импорт новых шаблонов писем {#importing-new-email-templates}

Новые версии Ultimate Multisite и add-ons могут регистрировать новые шаблоны писем. Чтобы предотвратить конфликты, новые шаблоны не добавляются автоматически — вам нужно импортировать их вручную.

Чтобы импортировать новые шаблоны писем:

1. Перейдите на страницу **System Emails**
2. Нажмите кнопку **Reset & Import** в верхней части страницы
3. Переключите опцию **Import Emails**, чтобы увидеть доступные шаблоны
4. Выберите шаблоны, которые хотите импортировать, и подтвердите

## Сброс шаблонов писем {#resetting-email-templates}

Если вы настроили шаблон письма и хотите вернуться к содержимому по умолчанию, у вас есть два варианта:

1. **Удалить и повторно импортировать** — удалите системное письмо и импортируйте его снова (это удалит метрики отправки)
2. **Сбросить через инструмент** — используйте инструмент **Reset & Import**, переключите опцию **Reset** и выберите письма, которые хотите сбросить

Опция Reset предпочтительна, так как она сохраняет метрики отправки и другие метаданные, восстанавливая содержимое по умолчанию.
