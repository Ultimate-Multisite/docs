# Управление системными письмами

Ultimate Multisite отправляет различные системные письма для событий, таких как подтверждения регистрации, квитанции об оплате и изменения членства. Вы можете настроить эти шаблоны писем, чтобы они соответствовали вашему бренду.

## Просмотр системных писем

Перейдите в **Ultimate Multisite → Settings** и нажмите на вкладку **Emails**, чтобы увидеть настройки, связанные с письмами.

![Settings - Emails tab](/img/admin/settings-emails.png)

Оттуда вы можете перейти на страницу **System Emails**, чтобы увидеть все шаблоны писем.

![System Emails list](/img/admin/system-emails-list.png)

## Импорт новых шаблонов писем

Новые версии Ultimate Multisite и дополнения могут регистрировать новые шаблоны писем. Чтобы избежать конфликтов, новые шаблоны не добавляются автоматически — их необходимо импортировать вручную.

Для импорта новых шаблонов писем:

1. Перейдите на страницу **System Emails**
2. Нажмите кнопку **Reset & Import** вверху страницы
3. Включите опцию **Import Emails**, чтобы увидеть доступные шаблоны
4. Выберите шаблоны, которые хотите импортировать, и подтвердите

## Сброс шаблонов писем

Если вы настроили шаблон письма и хотите вернуть его к содержимому по умолчанию, у вас есть два варианта:

1. **Delete and re-import** — Удалите системное письмо и импортируйте его снова (это удалит метрики отправки)
2. **Reset via the tool** — Используйте инструмент **Reset & Import**, включите опцию **Reset** и выберите письма, которые хотите сбросить

Опция сброса предпочтительнее, поскольку она сохраняет метрики отправки и другие метаданные, восстанавливая при этом содержимое по умолчанию.
