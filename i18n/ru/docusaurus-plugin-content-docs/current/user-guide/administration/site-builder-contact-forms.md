# Контактные формы в Site Builder

Superdav AI Agent v1.10.0 добавляет возможность создавать контактные формы прямо из чата с помощью агента Site Builder. Это позволяет добавить полностью функциональные контактные формы на любую страницу, не покидая чат.

## Обзор

Функция создания контактных форм в Site Builder (`create_contact_form`) автоматически определяет установленный вами плагин для форм и создает контактную форму, используя нативные возможности этого плагина. Поддерживаемые плагины для форм включают:

- Contact Form 7
- WPForms
- Fluent Forms
- Gravity Forms

Помощник автоматически выбирает лучший доступный плагин для форм на вашем сайте и генерирует контактную форму, адаптированную под этот плагин.

## Создание контактной формы

Чтобы создать контактную форму с помощью Site Builder, выполните следующие действия:

1. Откройте чат-панель **Gratis AI Agent** в админ-панели WordPress.
2. Переключитесь на агента **Site Builder**, нажав на иконку агента в заголовке чата.
3. Опишите контактную форму, которую вы хотите создать. Например:

   > "Добавь контактную форму на страницу «Контакты» с полями для имени, электронной почты, сообщения и номера телефона."

   Или просто:

   > "Создай контактную форму для страницы «Контакты»."

4. Site Builder сгенерирует контактную форму и вернет шорткод, готовый к вставке.

## Использование сгенерированного шорткода

После того как Site Builder создаст контактную форму, он вернет шорткод (например, `[contact-form-7 id="123"]`). Вы можете:

1. **Вставить его на страницу или запись** — Скопируйте шорткод и вставьте его на любую страницу или запись с помощью блочного или классического редактора.
2. **Добавить через Site Builder** — Попросите Site Builder автоматически добавить форму на определенную страницу:

   > "Добавь контактную форму на страницу «Контакты»."

3. **Использовать в шаблоне** — Если вы знакомы с PHP, вы можете добавить шорткод в файл шаблона темы.

## Настройка контактной формы

После того как Site Builder создаст контактную форму, вы можете настроить ее дополнительно:

### Через интерфейс чата

Попросите Site Builder изменить форму:

> "Обнови контактную форму, добавив поле «Тема» и сделав поле «Сообщение» обязательным."

Site Builder обновит форму и вернет обновленный шорткод.

### Через админ-интерфейс плагина форм

Вы также можете настроить форму непосредственно в настройках вашего плагина форм:

1. Перейдите в **Contact Form 7** (или ваш установленный плагин форм) в админ-панели WordPress.
2. Найдите форму, созданную Site Builder.
3. Отредактируйте поля формы, правила проверки и настройки уведомлений по электронной почте по мере необходимости.

## Плагины форм и совместимость

Site Builder автоматически определяет, какой плагин форм установлен на вашем сайте, и использует его для создания контактной формы. Если установлено несколько плагинов форм, Site Builder расставит приоритеты в следующем порядке:

1. Contact Form 7
2. WPForms
3. Fluent Forms
4. Gravity Forms

Если ни один из этих плагинов не установлен, Site Builder порекомендует установить один из них перед созданием контактной формы.

## Уведомления по электронной почте

Контактные формы, созданные Site Builder, по умолчанию настроены на отправку уведомлений администратору сайта. Вы можете настроить адрес электронной почты получателя и текст уведомления:

1. Перейдите в админ-интерфейс вашего плагина форм.
2. Найдите форму, созданную Site Builder.
3. Отредактируйте настройки уведомлений по электронной почте.

Для получения подробных инструкций обратитесь к документации вашего плагина форм:

- [Contact Form 7 Email Notifications](https://contactform7.com/docs/)
- [WPForms Email Notifications](https://wpforms.com/docs/)
- [Fluent Forms Email Notifications](https://fluentforms.com/docs/)
- [Gravity Forms Email Notifications](https://docs.gravityforms.com/)

## Примеры использования

### Простая контактная форма

> "Создай простую контактную форму с полями для имени, электронной почты и сообщения."

### Многошаговая форма

> "Создай контактную форму с первым шагом для контактной информации и вторым шагом для сообщения и предпочтительного способа связи."

### Форма с условной логикой

> "Создай контактную форму, которая будет показывать разные поля в зависимости от выбора пользователя в выпадающем списке."

### Форма с загрузкой файлов

> "Создай контактную форму, которая позволит пользователям загрузить файл или вложение."

:::note
Создание контактных форм доступно в Superdav AI Agent v1.10.0 и более поздних версиях. Для использования этой функции должен быть активен агент Site Builder.
:::
