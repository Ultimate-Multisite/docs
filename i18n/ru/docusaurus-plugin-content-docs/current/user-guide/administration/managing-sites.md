# Управление сайтами

Сайты (или подсайты) являются ядром вашего бизнеса WaaS. Ultimate Multisite имеет 3 типа сайтов:

- **Customer-owned** — Сайты, принадлежащие конкретным клиентам
- **Site Templates** — Предопределённые сайты, которые клиенты могут выбрать в качестве отправной точки
- **Main Site** — Ваш основной сетевой сайт

## Просмотр сайтов

Перейдите в **Ultimate Multisite → Sites**, чтобы увидеть все подсайты в вашей сети. Каждый сайт помечен типом, и вы можете фильтровать по All Sites, Customer-Owned, Templates или Pending.

![Sites list page](/img/admin/sites-list.png)

## Добавление нового сайта

Нажмите кнопку **Add Site**, чтобы создать новый сайт. Вам нужно предоставить:

- **Site Title** — Название нового сайта
- **Site URL** — Домен/путь для сайта
- **Site Type** — Является ли это клиентским сайтом, шаблоном или обычным сайтом

Опция **Copy Site** позволяет создать новый сайт на основе существующего шаблона сайта. При включении вы можете выбрать, какой шаблон использовать в качестве отправной точки. Убедитесь, что включена опция **Copy Media on Duplication**, чтобы включить медиафайлы.

## Управление существующим сайтом

Нажмите **Manage** на любом сайте, чтобы открыть страницу **Edit Site**. Здесь вы найдете:

### Базовая информация

Имя сайта, тип, ID сайта и описание. Вы также увидите сопоставленный домен, связанное членство и аккаунт клиента, владеющий сайтом.

### Параметры сайта

Настройте возможности и ограничения сайта:

- **Visit limits** — Максимальное количество посещений сайта
- **User account limits** — Ограничения по ролям пользователей
- **Disk space** — Квота хранения для сайта
- **Custom domain** — Включить сопоставление домена для этого сайта
- **Plugin and theme visibility** — Управлять видимостью, скрытием или предактивацией плагинов и тем

По умолчанию сайты следуют ограничениям, установленным на уровне членства. Установка ограничений на уровне сайта переопределит настройки членства.

### Связи

Ниже параметров сайта вы найдете информацию о:

- **Mapped domains** связанные с сайтом
- **Membership** под которым находится сайт
- **Customer account** связанный с сайтом

### Правая боковая панель

С правой стороны вы можете:

- **Enable/disable the site** с помощью переключателя
- **Change the site type** или переназначить владельца
- **Set a site image/thumbnail** (отображается на фронтенде для шаблонов сайтов)
- **Delete the site** навсегда

:::warning
Удаление сайта необратимо. Сайт и весь его контент будут навсегда удалены.
:::
