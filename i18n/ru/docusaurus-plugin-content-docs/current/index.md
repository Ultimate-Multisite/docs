# Документация Ultimate Multisite

Добро пожаловать в документацию Ultimate Multisite. Ultimate Multisite преобразует WordPress Multisite в полноценную платформу Website as a Service (WaaS).

## Разделы документации

### [Руководство пользователя](/user-guide/getting-started/ultimate-multisite-101)

Всё, что вам нужно для настройки, конфигурации и управления вашей платформой WaaS. Охватывает установку, платежные шлюзы, привязку доменов, управление клиентами и многое другое.

### [Документация для разработчиков](/developer/hooks)

Техническая справка для разработчиков, создающих решения на базе Ultimate Multisite. Включает справочник хуков, документацию REST API и руководства по разработке дополнений.

### [Дополнения](/addons)

Документация для платных плагинов-надстроек, которые расширяют Ultimate Multisite дополнительными функциями, такими как интеграция WooCommerce, аналитика, продажа доменов и многое другое.
