# Действие: wu_site_created

Вызывается после того, как сайт создан впервые.

Не вызывается, если сайт дублируется из шаблона.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $data | `array` | Данные объекта, которые будут сохранены. |
| $site | `\Site` | Экземпляр объекта. |

### С версии

- 2.0.0
