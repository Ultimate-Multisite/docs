# Action: wu_after_switch_template

Позволяет разработчикам плагинов подключать функции после того, как пользователь или супер‑админ переключает шаблон сайта

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $id | `int` | Идентификатор сайта |
