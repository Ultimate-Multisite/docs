# Действие: wu_after_switch_template

Позволяет разработчикам плагинов подключать функции после того, как пользователь или супер-администратор переключает шаблон сайта

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $id | `int` | ID сайта |

### С версии

- 1.9.8
