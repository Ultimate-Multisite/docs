# Action: wu_mcp_abilities_registered

Вызывается после регистрации способностей MCP для сущности.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $ability_prefix | `string` | Префикс способности. |
| $model_name | `string` | Имя модели. |
| $this | `object` | Экземпляр менеджера. |

### С версии

- 2.5.0

### Источник

Определено в [`inc/apis/trait-mcp-abilities.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/apis/trait-mcp-abilities.php#L171) на строке 171
