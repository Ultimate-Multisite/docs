# Действие: wu_dashboard_widgets

Позволяет разработчикам плагинов добавлять виджеты в панель сети.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $tab | `string` | Текущая вкладка. |
| $screen | `\WP_Screen` | Объект экрана. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Экземпляр страницы администратора Ultimate Multisite. |

### С версии

- 2.0.0

Определено в [`inc/admin-pages/class-dashboard-admin-page.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/admin-pages/class-dashboard-admin-page.php#L225) на строке 225
