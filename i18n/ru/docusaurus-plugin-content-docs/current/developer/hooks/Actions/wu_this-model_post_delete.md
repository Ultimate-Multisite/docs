# Действие: wu_${this->model}_post_delete

Срабатывает после того, как объект сохранен в базе данных.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $result | `bool` | True, если объект был успешно удален. |
| $this | `\Base_Model` | Экземпляр объекта. |
