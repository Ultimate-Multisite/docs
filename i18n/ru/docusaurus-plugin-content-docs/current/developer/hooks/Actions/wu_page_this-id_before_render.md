# Действие: wu_page_{$this->id}_before_render

Позволяет разработчикам плагинов добавлять дополнительный контент перед выводом страницы.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $page_id | `string` | Идентификатор этой страницы. |
| $page | `object` | Объект страницы. |

### С версии

- 1.8.2

Определено в [`inc/admin-pages/class-base-admin-page.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/admin-pages/class-base-admin-page.php#L398) на строке 398
