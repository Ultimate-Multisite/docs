# Действие: wu_checkout_after_process_order

Позволяет разработчикам вызывать дополнительные хуки.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $checkout | `\Checkout` | Экземпляр объекта оформления заказа. |
| $cart | `\Cart` | Экземпляр корзины оформления заказа. |

### С версии

- 2.0.9

Определено в [`inc/checkout/class-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout.php#L1013) на строке 1013
