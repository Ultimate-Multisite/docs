# Фильтр: wu_get_post_types

Позволяет разработчикам выбирать, какие типы записей должны отображаться.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $post_types | `array` | Список типов записей. |

### С версии

- 2.0.0

### Источник

Определено в [`inc/ui/class-limits-element.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/ui/class-limits-element.php#L267) на строке 267

## Возвращает

Новый список.
