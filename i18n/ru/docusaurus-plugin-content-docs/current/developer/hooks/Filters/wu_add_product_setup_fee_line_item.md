# Фильтр: wu_add_product_setup_fee_line_item

Позволяет разработчикам вносить изменения в позицию платы за настройку.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $setup_fee_line_item | `array` | Параметры позиции платы за настройку. |
| $product | `\WP_Ultimo\Models\Product` | Товар, связанный с платой за настройку. |
| $cart | `\WP_Ultimo\Checkout\Cart` | Объект корзины. |

### С версии

- 2.1

### Источник

Определено в [`inc/checkout/class-cart.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-cart.php#L1790) на строке 1790
