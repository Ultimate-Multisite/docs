# Фильтр: wu_maybe_redirect_to_admin_panel_exclusion_list

Создайте список исключений параметров, которые предотвращают авто-редирект.

Это необходимо, потому что иначе конструктор страниц не сможет редактировать страницу входа после её определения.

### С версии

- 2.0.4

### Источник

Определено в [`inc/checkout/class-checkout-pages.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout-pages.php#L357) на строке 357
