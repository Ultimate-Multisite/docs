# Фильтр: wu_add_new_domain_modal_fields

Фильтрует поля для модального окна добавления нового домена.

Позволяет дополнениям (например, Domain Seller) изменять или заменять поля регистрации домена.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $fields | `array` | Поля формы. |

### С версии

- 2.1.0
