# Filter: wu_independent_billing_cycle_product_types

Отфильтруйте типы продуктов, которые имеют независимые циклы выставления счетов.

Продукты с независимыми циклами выставления счетов не обязаны соответствовать периоду выставления счетов плана в корзине. Это полезно для продуктов, таких как регистрация доменов, которые всегда выставляются ежегодно, независимо от того, является ли план ежемесячным или ежегодным.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | Массив типов продуктов с независимыми циклами выставления счетов. |
