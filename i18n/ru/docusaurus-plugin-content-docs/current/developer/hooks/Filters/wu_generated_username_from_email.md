# Фильтр: wu_generated_username_from_email

Фильтр генерируемого имени пользователя клиента.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $username | `string` | Генерируемое имя пользователя |
| $email | `string` | Новый адрес электронной почты клиента |
| $new_user_args | `array` | Массив аргументов нового пользователя, возможно включающий имя и фамилию |
| $suffix | `string` | Добавить строку к имени пользователя, чтобы сделать его уникальным |

### С версии

- 3.7.0

Определено в [`inc/functions/customer.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/customer.php#L488) на строке 488
