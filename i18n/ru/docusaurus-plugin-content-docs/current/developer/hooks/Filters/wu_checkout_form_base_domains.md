# Фильтр: wu_checkout_form_base_domains

Фильтрует общие базовые домены формы checkout, которые не должны становиться записями сопоставленных доменов.

Используйте этот фильтр, когда интеграция предоставляет дополнительные базовые домены для полей **Site URL** формы checkout. Домены, возвращаемые этим фильтром, рассматриваются как общие хосты регистрации вместо пользовательских доменов для отдельных сайтов.

## Параметры {#parameters}

| Название | Тип | Описание |
|------|------|-------------|
| $domains | `array` | Общие базовые домены, собранные из конфигурации формы checkout. |

### Начиная с версии {#since}

- 2.13.0

### Источник {#source}

Определено в `inc/functions/domain.php`.


## Возвращает {#returns}

Массив нормализованных базовых доменов формы checkout.
