# Фильтр: wu_setup_wizard

Позволяет разработчикам добавлять дополнительные шаги мастера настройки.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $sections | `array` | Текущие разделы. |
| $is_migration | `bool` | Указывает, является ли это миграцией. |
| $wizard | `object` | Текущий экземпляр. |

### С версии

- 2.0.0
