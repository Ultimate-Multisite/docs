# Фильтр: wu_is_mcp_enabled

Позволяет разработчикам плагинов принудительно установить заданное состояние для адаптера MCP.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $enabled | `bool` | Включен ли адаптер MCP. |

### С версии

- 2.5.0

### Источник

Определено в [`inc/class-mcp-adapter.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-mcp-adapter.php#L250) на строке 250
