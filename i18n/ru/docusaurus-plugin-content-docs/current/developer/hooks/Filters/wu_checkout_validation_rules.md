# Фильтр: wu_checkout_validation_rules

Позволяет разработчикам плагинов фильтровать правила проверки.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $validation_rules | `array` | Правила проверки, которые будут использоваться. |
| $checkout | `\Checkout` | Класс оформления заказа. |

### С версии

- 2.0.20
