# Фильтр: wu_is_development_mode

Позволяет разработчикам плагинов добавлять дополнительные тесты для режима разработки.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $is_development_mode | `bool` | Текущее состояние разработки. |
| $site_url | `string` | URL сайта. |

### С версии

- 2.0.0

### Источник

Определено в [`inc/domain-mapping/class-helper.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/domain-mapping/class-helper.php#L63) на строке 63
