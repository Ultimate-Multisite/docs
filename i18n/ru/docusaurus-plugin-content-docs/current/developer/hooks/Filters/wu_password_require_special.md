# Фильтр: wu_password_require_special

Фильтр, определяющий, требуется ли специальные символы в паролях.

## Parameters

| Имя | Тип | Описание |
|------|------|-------------|
| $require | `bool` | Требуется ли специальные символы. По умолчанию true, когда правила применяются. |
| $defender_active | `bool` | Активен ли Defender Pro Strong Password. |

### С версии

- 2.4.0
