# Фильтр: wu_current_set_site {#filter-wucurrentsetsite}

Позволяет разработчикам изменить поведение по умолчанию и установить текущий сайт иначе.

## Параметры {#parameters}

| Имя | Тип | Описание |
|------|------|-------------|
| $site | `\WP_Ultimo\Models\Site` | Текущий сайт, который нужно установить. |
| $ | `self` | Экземпляр класса Current. |

### С версии {#since}
