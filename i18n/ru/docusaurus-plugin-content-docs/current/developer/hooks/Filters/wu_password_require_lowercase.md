# Filter: wu_password_require_lowercase

Фильтр, определяющий, требуется ли использовать строчные буквы в паролях.

## Parameters {#parameters}

| Имя | Тип | Описание |
|------|------|-------------|
| $require | `bool` | Определяет, требуется ли использовать строчные буквы. По умолчанию true, когда правила применяются. |
| $defender_active | `bool` | Определяет, активен ли Defender Pro Strong Password. |
