# Фильтр: wu_password_require_number

Фильтр, определяющий, требуется ли наличие цифр в паролях.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $require | `bool` | Требуется ли наличие цифр. По умолчанию true, когда правила применяются. |
| $defender_active | `bool` | Активен ли Defender Pro Strong Password. |

### С 2.4.0

- 2.4.0
