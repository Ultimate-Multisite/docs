# Filter: wu_view_override_replaceable_views

Позволяет разработчикам добавлять дополнительные папки в список заменяемых.

Будьте осторожны, поскольку добавление дополнительных папок может привести к загрузке устаревших копий вместо версий Ultimate Multisite.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $replaceable_views | `array` | Список разрешенных папок. |

### Since

- 2.0.0
### Source

Определено в `inc/functions/template.php` на строке 50
