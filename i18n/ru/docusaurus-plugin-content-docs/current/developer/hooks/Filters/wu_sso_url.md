# Фильтр: wu_sso_url

Фильтрует сгенерированные SSO URL перед их возвратом для междоменных действий клиента.

Используйте этот фильтр, когда интеграции нужно добавить доверенный контекст к ссылке SSO для суверенного арендатора или заменить URL брокера, сохранив проверку токена Ultimate Multisite.

## Параметры {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $sso_url | `string` | Сгенерированный SSO URL. |
| $user | `WP_User` | Пользователь, который будет аутентифицирован при посещении через SSO. |
| $site_id | `int` | ID целевого сайта для посещения. |
| $redirect_to | `string` | URL назначения после успешной проверки SSO. |

### Начиная с {#since}

- 2.13.0

### Источник {#source}

Определено в `inc/sso/class-sso.php`.


## Возвращает {#returns}

Отфильтрованный SSO URL.
