# Filter: wu_documentation_get_link

Позволяет разработчикам плагинов фильтровать ссылки.

Не уверен, насколько это может быть полезно, но это не помешает иметь его.

## Parameters

| Имя | Тип | Описание |
|------|------|-------------|
| $link | `string` | Зарегистрированная ссылка |
| $slug | `string` | Слаг, используемый для получения ссылки |
| $default_link | `string` | Зарегистрированная ссылка по умолчанию |

- 1.7.0
