## Фильтр: wu_current_set_membership {#filter-wucurrentsetmembership}

Позволяет разработчикам изменить поведение по умолчанию и установить текущее членство по-другому.

## Параметры {#parameters}

| Имя | Тип | Описание |
|------|------|-------------|
| $membership | `\WP_Ultimo\Models\Membership` | Текущее членство, которое нужно установить. |
| $ | `self` | Экземпляр класса Current. |
