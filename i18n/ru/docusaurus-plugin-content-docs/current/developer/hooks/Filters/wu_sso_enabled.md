# Фильтр: wu_sso_enabled

Включить/выключить кросс-доменную функцию единого входа.

Отфильтруйте это значение, чтобы полностью отключить единственный вход, либо условно включить его.

## Параметры

| Имя | Тип | Описание |
|------|------|-------------|
| $enabled | `bool` | Должен ли SSO быть включен? True для включения, false-ish для отключения. |

### С версии

- 2.0.11

### Источник

Определено в [`inc/sso/class-sso.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/sso/class-sso.php#L110) на строке 110

## Возвращаемое значение

Если SSO включен или нет.
