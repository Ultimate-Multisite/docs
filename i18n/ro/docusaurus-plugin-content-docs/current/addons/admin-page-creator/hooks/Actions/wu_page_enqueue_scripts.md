# Action: wu_page_enqueue_scripts

Adăugăm o acțiune personalizată aici, astfel încât să avem un loc unde să înregistreze (enqueue) scripturile și stilurile după cum este necesar, și să ne asigurăm că acestea sunt încărcate doar pe paginile corecte.

|------|------|-------------|
| $ | `\WU_Page` | Pagina de administrare curentă |
