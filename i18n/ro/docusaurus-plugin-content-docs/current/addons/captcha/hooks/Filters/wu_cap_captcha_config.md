# Filtru: wu_cap_captcha_config

Filtrează configurația Cap Captcha.

Acest filtru permite dezvoltatorilor să personalizeze setările Cap Captcha.

## Parametri {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $config | `array` | Matricea de configurare Cap. |
| $level | `string` | Nivelul de securitate (fast, medium, max). |

- 1.2.0
