# Loco Translate {#loco-translate}

Addon-ul Ultimate Multisite: Loco Translate Integration aduce capacități de traducere specifice site-ului tău în rețeaua ta WordPress multisite. Dacă ai avut vreodată nevoie ca site-uri diferite din rețeaua ta să aibă traduceri diferite pentru același plugin sau temă, acest addon este soluția perfectă.

**Ce face acest Addon**

Acest addon integrează perfect Ultimate Multisite cu plugin-ul popular Loco Translate, permițând fiecărui subsite din rețeaua ta să își mențin propriile traduceri personalizate. Indiferent dacă rulezi o platformă multi-regională, servești comunități lingvistice diferite sau ai pur și simplu nevoie de flexibilitate în traduceri în întreaga rețeauă, acest addon face procesul effortless (fără efort).

**Ideal pentru:**

* Afaceri multi-regionale cu nevoi diferite de localizare
* Instituții de învățământ care servește comunități diverse
* Platforme SaaS care oferă suport multilingvistic
* Agenții web care gestionează multiple site-uri de clienți
* Orice rețeaua multisite care necesită flexibilitate în traduceri

## Instalare {#installation}

1. Asigură-te că Ultimate Multisite este instalat și că rețeaua este activată
2. Instalează și activează plugin-ul Loco Translate
3. Încarcă acest addon în directorul tău de plugins
4. Activează la nivel de rețeauă addon-ul Loco Translate Integration
5. Începe să creezi traduceri specifice site-ului prin Loco Translate pe orice subsite

Gata! Integrarea funcționează automat, fără nicio configurare suplimentară.

## Întrebări Frecvente {#frequently-asked-questions}

## Trebuie să configurez ceva după instalare? {#do-i-need-to-configure-anything-after-installation}

Nu! Integrarea este automată. Odată ce atât addon-ul, cât și Loco Translate sunt active, subsite-urile pot începe imediat să își creeze propriile traduceri personalizate.

## Ce se întâmplă dacă un subsite nu creează traduceri personalizate? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

Site-ul va folosi automat traducerile la nivel de rețeauă ca fallback (alternativă). Nimic nu se strică și totul funcționează perfect.

## Subsite-urile pot accesa traducerile la nivel de rețeauă? {#can-subsites-access-network-wide-translations}

Administratorii obișnuiți ai site-urilor văd și editează doar propriile traduceri ale site-ului lor. Super administratorii pot accesa atât traducerile la nivel de rețeauă, cât și cele specifice site-ului, în funcție de site-ul pe care îl vizualizează.

## Funcționează acest lucru atât cu plugin-uri, cât și cu teme? {#does-this-work-with-both-plugins-and-themes}

Da! Traducerile specifice site-ului funcționează atât pentru plugin-urile WordPress, cât și pentru teme.

## Va încetini rețeaua mea multisite? {#will-this-slow-down-my-multisite-network}

Nu. Addon-ul folosește o încărcare eficientă a fișierelor care verifică doar traducerile specifice site-ului atunci când este necesar. Nu există nicio afectare vizibilă a performanței.

## Pot copia traducerile la nivel de rețeauă într-un site specific ca punct de plecare? {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

Da! În timp ce editezi traducerile în Loco Translate, poți începe ușor cu traducerile la nivel de rețeauă și apoi să le customizezi pentru fiecare site.

## Funcționează acest lucru cu WooCommerce și alte plugin-uri? {#does-this-work-with-woocommerce-and-other-plugins}

Absolut! Orice plugin sau temă care folosește funcțiile standard de traducere WordPress va funcționa cu traducerile specifice site-ului.
