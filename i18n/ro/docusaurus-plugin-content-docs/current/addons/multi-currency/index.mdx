# Multi-Currency {#multi-currency}

Ultimate Multisite: Multi-Currency permite rețelei dvs. să accepte plăți în mai multe monede. Acesta convertesc automat prețurile folosind rate de schimb în timp real sau manuale, vă permite să setați prețuri exacte per produs pentru orice monedă și detectează moneda preferată a fiecărui vizitator prin geolocație — toate gestionate dintr-un singur dashboard de administrare.

## Funcționalități Cheie {#key-features}

- **Conversie automată a ratei de schimb** — prețurile sunt convertite din moneda de bază folosind rate prelevate de la furnizorul ales
- **Suprascri scrierea prețurilor per produs** — setați prețuri exacte, specifice unei monede, pe orice produs pentru a ocoli conversia automată
- **Câmp de înregistrare cu selector de monede** — permite clienților să își aleagă moneda preferată în timpul checkout-ului/înregistrării
- **Multiple furnizori de rate de schimb** — ECB, Frankfurter, Stripe sau rate complet manuale
- **Detectarea monedei bazată pe geolocație** — selectează automat moneda care corespunde țării vizitatorului
- **Dashboard de administrare** — gestionați monedele active, ratele de schimb și setările furnizorului dintr-un singur loc

## Cerințe {#requirements}

- Ultimate Multisite 2.0.0 sau mai mare
- Instalare WordPress Multisite
- PHP 7.4 sau mai mare

## Instalare {#installation}

1. Uploadați fișierele plugin-ului în `/wp-content/plugins/ultimate-multisite-multi-currency`
2. Activați plugin-ul prin ecranul **Plugins** din WordPress (Network Activate)
3. Navigați la **Ultimate Multisite → Multi-Currency** pentru a deschide dashboard-ul de administrare
4. Adăugați monedele pe care doriți să le acceptați și configurați furnizorul de rate de schimb

## Configurare {#configuration}

### Dashboard-ul de Administrare {#admin-dashboard}

Dashboard-ul Multi-Currency (**Ultimate Multisite → Multi-Currency**) este locul central pentru:

- Adăugarea, activarea sau dezactivarea monedelor
- Alegerea unui furnizor de rate de schimb și actualizarea ratelor la cerere
- Revizuirea ratelor curente pentru toate monedele active
- Setarea ratelor manuale atunci când furnizorii automatici nu sunt potriviți

### Furnizori de Rate de Schimb {#exchange-rate-providers}

| Provider | Descriere |
|---|---|
| **ECB** | Banca Centrală Europeană — rate zilnice gratuite pentru rețele bazate pe EUR |
| **Frankfurter** | API oglindă open-source al ECB — gratuit, nu necesită cheie API |
| **Stripe** | Utilizează ratele FX live ale Stripe — necesită o integrare Stripe activă |
| **Manual** | Introduceți ratele direct; fără apeluri API externe |

Selectați furnizorul sub **Ultimate Multisite → Multi-Currency → Settings → Exchange Rate Provider**. Ratele pot fi actualizate manual în orice moment sau programate pentru a se actualiza automat.

### Suprascri scrierea prețurilor per produs {#per-product-pricing-overrides}

Când conversia automată nu este suficient de precisă, puteți seta un preț exact pentru o monedă specifică pe orice produs:

1. Deschideți editorul produsului (**Ultimate Multisite → Products → Edit**)
2. Derulați până la secțiunea **Currency Pricing**
3. Introduceți prețul exact pentru fiecare monedă pe care doriți să o suprascrieți
4. Salvați produsul

Produsele fără suprascriere folosesc prețul convertit din moneda de bază.

### Câmp de înregistrare cu selector de monede {#currency-selector-signup-field}

Câmpul selector de monede poate fi adăugat fluxului dvs. de înregistrare/checkout, astfel încât clienții să își aleagă moneda preferată la înregistrare:

1. Mergeți la **Ultimate Multisite → Multi-Currency → Settings**
2. Activați **Show currency selector on signup**
3. Opțional, setați o monedă implicită sau lăsați geolocația să determine valoarea preselectată

Moneda selectată este stocată împreună cu abonamentul și folosită pentru toate facturările ulterioare.

### Detectarea monedei bazată pe geolocație {#geolocation-based-currency-detection}

Când detectarea geolocației este activată, addon-ul mapează țara vizitatorului la o monedă și o selectează preselectat în câmpul selector de monede. Pentru a activa:

1. Mergeți la **Ultimate Multisite → Multi-Currency → Settings**
2. Activați **Detect currency from visitor location**
3. Mappați țările la monede în tabela **Geolocation Mapping**

Dacă nu există o mapare pentru țara vizitatorului, se folosește moneda de bază a rețelei ca fallback.

## Întrebări Frecvente {#frequently-asked-questions}

### Ce se întâmplă dacă un furnizor de rate de schimb nu este disponibil? {#what-happens-if-an-exchange-rate-provider-is-unavailable}

Addon-ul revine la ultimele rate prelevate cu succes. Dacă nu au fost prelevate rate niciodată, revine la ratele manuale pe care le-ați configurat. Este recomandat să configurați ratele manuale ca măsură de siguranță.

### Pot folosi furnizori diferiți pentru monede diferite? {#can-i-use-different-providers-for-different-currencies}

Nu — un singur furnizor este activ în orice moment pentru toate monedele. Folosiți **Manual** dacă aveți nevoie de control complet asupra ratelor individuale.

### Geolocația necesită un plugin separat? {#does-geolocation-require-a-separate-plugin}

Nu este necesar un plugin extern de geolocație. Addon-ul folosește detectarea IP-la-țară încorporată în WordPress. Pentru o precizie mai mare, o serviciu dedicat de geolocație poate fi integrat prin hook-urile de filtrare furnizate.

### Suprascrierile per produs se aplică înainte sau după taxă? {#are-per-product-overrides-applied-before-or-after-tax}

Suprascrierile per produs setează prețul înainte de taxă în moneda țintă. Taxa este calculată peste prețul de suprascriere, folosind configurarea dvs. existentă a taxelor.
