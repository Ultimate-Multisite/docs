# Jurnal de modificări GoCardless

Versiune: 1.0.3 - Lansat pe 2026-05-05
* Corecție: Siturile în așteptare orfane sunt acum curățate atunci când un abonament GoCardless este anulat în timp ce se află în stare de așteptare sau suspendată
* Îmbunătățit: Autoload-er-ul plugin-ului este sărit atunci când autoload-er-ul rădăcinii Bedrock a încărcat deja dependențele

Versiune: 1.0.1 - Lansat pe 2025-09-28

* Renumare prefixului în ultimate-multisite; actualizare domeniului de text; creștere a numărului de versiune.


2021-09-24 - Versiunea 1.0.0-beta.4

* Adăugat: filtrul wp_ultimo_skip_network_active_check pentru configurațiile bazate pe mu-plugins;

2021-06-16 - Versiunea 1.0.0-beta.3

* Corecție: Handler-ele de webhook care eșuau să confirme semneaturile;

2021-06-14 - Versiunea 1.0.0-beta.2

* Îmbunătățit: Publicarea site-ului după ce este creat un mandat GoCardless;
