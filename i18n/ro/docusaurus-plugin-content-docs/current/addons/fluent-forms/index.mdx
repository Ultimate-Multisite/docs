# Fluent Forms {#fluent-forms}

Acest addon permite adăugarea limitelor pentru Fluent Forms pe Multisite Ultimate, fie că vorbim de produse, membrici sau site-uri.
Fluent Forms folosește tabele personalizate (custom tables) în loc de tipuri de postări personalizate (custom post types), așa că opțiunile normale de limitare nu funcționează cu Fluent Forms.

## Funcționalități {#features}

- Adăugarea limitelor pentru Fluent Forms sau Conversational Fluent Forms
- Clienții primesc un mesaj prietenos care îi încurajează să își actualice planul atunci când ating limita
