# AI Site Builder

ultimate-multisite-ai-site-builder

## Cerințe {#requirements}

- WordPress 5.3 sau versiune mai nouă
- PHP 7.4 sau versiune mai nouă
- Plugin-ul Ultimate Multisite (activ)

## Instalare {#installation}

1. Încarcă fișierele addon-ului în directorul tău `/wp-content/plugins/`
2. Activează plugin-ul din meniul „Plugins” din WordPress
3. Configurează setările addon-ului în administratorul rețelei

## Dezvoltare {#development}

### Configurare {#setup}

```bash
# Instalează dependențele
composer install
npm install

# Rulează testele
npm run test

# Rulează verificările standardelor de cod
vendor/bin/phpcs
vendor/bin/phpstan

# Corectează problemele de stil al codului
vendor/bin/phpcbf

# Construiește pentru producție
npm run build
```

### Structura proiectului {#project-structure}

```
ultimate-multisite-ai-site-builder/
├── inc/                          # Clase și include-uri PHP
│   ├── class-multisite-ultimate-updater.php  # Handler de actualizare

│   ├── bootstrap.php            # Bootstrap pentru testare
│   ├── class-*-base.php         # Clasa de testă de bază
│   └── class-*-test.php         # Cazuri de testare
├── views/                       # Fișiere de șablon
├── assets/                      # Asset-uri statice
│   ├── css/                     # Fișiere CSS
│   ├── js/                      # Fișiere JavaScript
│   └── images/                  # Imagini
├── lang/                        # Fișiere de limbă
├── composer.json                # Dependențe PHP

├── phpunit.xml.dist             # Configurația PHPUnit

```

### Testare {#testing}

Addon-ul include un cadru de testare cuprinzător:

- **Teste unitare**: Teste bazate pe PHPUnit, cu suite-ul de test-uri WordPress
- **Clasa de testă de bază**: Oferă metode auxiliare pentru scenarii comune de testare
- **Acoperirea testelor**: Configurat pentru a urmări acoperirea codului
- **Testare Multisite**: Testele rulează într-un mediu multisite

### Standarde de codare {#code-standards}

- **Standarde de codare WordPress**: Aplicat prin PHPCS
- **Compatibilitate PHP 7.4+**: Suport pentru funcționalități moderne PHP
- **Analiză statică**: Integrarea PHPStan pentru verificare de tip
- **Modernizare cod**: Regulile Rector pentru actualizările PHP

### Sistemul de build {#build-system}

- **Procesarea asset-urilor**: Minificare pentru fișierele CSS/JS
- **Traducere**: Generarea fișierelor POT
- **Ambalarea**: Crearea unui arhiv pentru distribuție
- **Mod de dezvoltare**: Asset-uri ne-minificate pentru depanare

## Configurare {#configuration}

Addon-ul suportă diverse opțiuni de configurare, accesibile prin interfața administratorului rețelei.

## Hooks și Filtre {#hooks-and-filters}

### Acțiuni (Actions) {#actions}

- `ultimate-multisite-ai-site-builder_init` - Se declanșează după inițializarea addon-ului
- `ultimate-multisite-ai-site-builder_loaded` - Se declanșează după încărcarea tuturor fișierelor addon-ului

### Filtre (Filters) {#filters}

- `ultimate-multisite-ai-site-builder_settings` - Modifică setările addon-ului
- `ultimate-multisite-ai-site-builder_enabled` - Anulează statutul de activare/dezactivare al addon-ului
