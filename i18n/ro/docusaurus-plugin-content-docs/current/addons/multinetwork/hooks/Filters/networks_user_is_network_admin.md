# Filtru: networks_user_is_network_admin

Filtrează rețelele pentru care un utilizator este administratorul.

## Parametri

| Nume | Tip | Descriere |
|------|------|-------------|
| $ | `array\|bool` | O listă de ID-uri de rețea sau `false` dacă utilizatorul nu are rețele. |
| $ | `int` | ID-ul utilizatorului pentru care trebuie returnate rețelele. |
