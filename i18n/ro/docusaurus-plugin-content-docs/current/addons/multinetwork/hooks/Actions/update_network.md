# Action: update_network

Se declanșează după ce o rețea existentă a fost actualizată.

## Parametri

| Nume | Tip | Descriere |
|------|------|-------------|
| $network_id | `int` | ID-ul rețelei adăugate. |
| $args | `array` | Un array asociativ de argumente de rețea. |
