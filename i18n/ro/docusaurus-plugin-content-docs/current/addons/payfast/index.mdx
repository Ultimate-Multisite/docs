# PayFast

Multisite Ultimate: Gateway-ul Payfast

Adaugă Payfast — un gateway de plăți online rapid și sigur din Africa de Sud — ca opțiune de plată în Ultimate Multisite.

## Instalare

1. Urcați 'ultimate-multisite-payfast' în directorul '/wp-content/plugins/'
2. Activați plugin-ul prin meniul 'Plugins' din WordPress
3. Urmați Asistentul pas cu pas pentru a configura plugin-ul

## Ghid de utilizare

PayFast este un procesator de plăți online de top pentru companiile din Africa de Sud. Se dedică ajutării comercianților online să crească, făcând tranzacțiile fluide și securizate. Conform solicitării utilizatorilor Ultimate Multisite, am decis să creăm un addon care să permită integrarea cu PayFast. Acest articol vă arată cum să setați PayFast ca gateway de plată pentru a primi plăți în rețeaua dvs. Ultimate Multisite.

Primul pas este instalarea addon-ului.

# Cum să instalați addon-ul Payfast:

Mergeți la pagina cu addon-urile și selectați addon-ul Ultimate Multisite: Payfast Gateway.

![Add-ons page](/img/admin/addons-page.png)

Se va deschide o fereastră pentru addon. Faceți clic pe Install Now (Instalează acum).

![Add-ons page with install option](/img/admin/addons-page.png)

După ce instalarea este finalizată, veți fi redirecționați către pagina cu plugin-urile. Aici, doar faceți clic pe Network Activate și addon-ul _Payfast_ va fi activat pe rețeaua dvs.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# Configurarea PayFast

După instalarea addon-ului, puteți merge acum la setările Ultimate Multisite și puteți începe să configurați PayFast.

În pagina **Ultimate Multisite settings**, puteți găsi tab-ul "Payments" (Plăți). Sub acest tab, ar trebui să găsiți "**Payfast** " împreună cu un buton de comutare (toggle button) pentru a activa sau dezactiva acest gateway de plăți.

![Payment gateway settings](/img/config/settings-payment-gateways.png)Mergeți și faceți clic pe butonul de comutare pentru a-l activa. Acesta va extinde apoi câmpul pentru a arăta diferite câmpuri pe care trebuie să le completați, cum ar fi **Merchant ID, Merchant Key și PassPhrase.** Puteți obține aceste informații din contul dvs. **Payfast**

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Următorul pas este să vă autentificați în contul PayFast pentru a obține informațiile necesare.

După autentificare, puteți vedea imediat datele de autentificare de care aveți nevoie din dashboard-ul contului dvs. **Payfast**.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->Puteți copia și lipi simplu datele de autentificare și puteți completa detaliile în setările Ultimate Multisite. Apoi faceți clic pe butonul **"Save settings"** pentru a salva modificările.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Gata, integrarea de plată PayFast este completă!
