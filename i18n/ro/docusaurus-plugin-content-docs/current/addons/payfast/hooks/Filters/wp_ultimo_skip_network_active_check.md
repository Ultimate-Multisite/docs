# Filter: wp_ultimo_skip_network_active_check

Permite dezvoltatorilor să ignore (să "short-circuit") această verificare.

Acest lucru este util atunci când folosiți setări personalizate bazate pe Composer și alte configurații, cum ar fi Bedrock, de exemplu, unde utilizarea plugin-urilor ca mu-plugins este norma.
