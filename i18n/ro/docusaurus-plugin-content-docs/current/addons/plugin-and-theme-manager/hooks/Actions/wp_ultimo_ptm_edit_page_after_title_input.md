# Action: wp_ultimo_ptm_edit_page_after_title_input

Permite dezvoltatorilor de plugin să adauge informații suplimentare sub câmpul de text.

## Parametri

| Nume | Tip | Descriere |
|------|------|-------------|
| $ | `object` | Obiect care conține informațiile |
| $ | `\WU_Page` | Instanță de pagină Ultimate Multisite |

### De la

- 1.8.2
