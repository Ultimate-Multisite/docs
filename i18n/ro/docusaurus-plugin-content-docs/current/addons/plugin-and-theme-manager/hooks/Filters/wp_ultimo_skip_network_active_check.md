# Filter: wp_ultimo_skip_network_active_check

Permite dezvoltatorilor să ignore această verificare.

Acest lucru este util atunci când folosiți configurări personalizate bazate pe Composer și alte setări, cum ar fi Bedrock, de exemplu, unde utilizarea plugin-urilor ca mu-plugins este standard.
