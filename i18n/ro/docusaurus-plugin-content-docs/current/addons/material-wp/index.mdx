# Material WP

Interfața de administrare WordPress nu s-a schimbat de ani de zile. Dar v-ați întrebat vreodată cum ar arăta o versiune Material Design a acestei interfețe?

Material WP este un *theme* de administrare dezvoltat pentru a vă oferi acel aspect și senzație, ajutându-vă să oferiți clienților dumneavoastră un Dashboard mai personalizat.

## Caracteristici

- Suport pentru CSS personalizat
- Setări de stil ușoare
- Reordonarea meniului
- Import/Export ușor

## Documentație

**Personalizarea Panoului de Administrare cu Material WP**

## Instalare

1. Încărcați 'material-wp' în directorul '/wp-content/plugins/'
2. Activați plugin-ul prin meniul 'Plugins' din WordPress
