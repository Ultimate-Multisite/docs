# Action: wp_ultimo_support_agents_load

Se declanșează când toate dependențele au fost încărcate.

Acest hook permite dezvoltatorilor de plugin să adauge funcționalități noi. De exemplu, suport pentru noi furnizori de găzduire (hosting), etc.

### De la

- 1.0.0
### Sursă
