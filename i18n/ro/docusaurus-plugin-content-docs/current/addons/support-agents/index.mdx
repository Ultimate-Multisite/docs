# Agenți de Suport

Gestionarea unui întregă rețele de la singur nu este o sarcină ușoară. Pe măsură ce rețeaua ta începe să crească, vei observa o cerere crescândă de suport. Ar fi o idee bună să aduci oameni care să te ajute să-ți ușuri sarcina. Totuși, acordarea unui acces/permisiune completă este riscantă și poate compromite rețeaua ta. Aici intervine acest addon de Agenți de Suport.

Acest addon îți permite să creezi un nou tip de utilizator WordPress numit Agent de Suport și îți oferă control total asupra permisiunilor pe care le are acel utilizator. Acordarea permisiunii este la fel de simplă ca și apăsarea unui buton de comutare (toggle) și o dezactivezi dacă decizi să îl elimini.

## Funcționalități {#features}

* Listă detaliată de permisiuni WordPress și Ultimate Multisite
* Activare și dezactivare ușoară a accesului
* Crearea unui număr nelimitat de conturi de Agent

## Instalare {#installation}

1. Încarcă 'ultimate-multisite-support-agents' în directorul '/wp-content/plugins/'
2. Activează plugin-ul prin meniul 'Plugins' din WordPress
3. Urmează Wizard-ul pas cu pas pentru a configura plugin-ul
