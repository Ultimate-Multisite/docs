# Documentația Ultimate Multisite

Bine ați venit la documentația Ultimate Multisite. Ultimate Multisite transformă WordPress Multisite într-o platformă completă Website as a Service (WaaS).

## Secțiuni de Documentație {#documentation-sections}

### [Ghid pentru Utilizatori](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

Tot ce aveți nevoie pentru a vă seta, configura și gestiona platforma WaaS. Acoperă instalarea, ghiciurile de plată, maparea domeniilor, managementul clienților și mult mai mult.

### [Documentație pentru Dezvoltatori](/developer/hooks) {#developer-documentation}

Referință tehnică pentru dezvoltatorii care construiesc pe baza Ultimate Multisite. Include referința de hook-uri, documentația REST API și ghiduri de dezvoltare pentru addon-uri.

### [Addon-uri](/addons) {#addons}

Documentația pentru plugin-urile addon plătite care extind Ultimate Multisite cu funcționalități suplimentare, cum ar fi integrarea cu WooCommerce, analitica, vânzarea de domenii și multe altele.
