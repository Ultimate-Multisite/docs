# Action: wu_page_before_render

Permite dezvoltatorilor de plugin să adauge conținut suplimentar înainte de a afișa pagina.

## Parametri {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $page_id | `string` | ID-ul acestei pagini. |
| $page | `object` | Obiectul paginii. |
