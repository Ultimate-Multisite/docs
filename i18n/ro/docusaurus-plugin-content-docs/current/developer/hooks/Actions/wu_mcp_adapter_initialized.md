# Acțiune: wu_mcp_adapter_initialized

Se declanșează după ce adaptorul MCP este inițializat.

Permite altor plugin-uri și teme să își înregistreze propriile capacități.

## Parametri

| Nume | Tip | Descriere |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | Instanța adaptorului MCP. |

### De la

- 2.5.0
### Sursă

Definit în [`inc/class-mcp-adapter.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-mcp-adapter.php#L98) la linia 98
