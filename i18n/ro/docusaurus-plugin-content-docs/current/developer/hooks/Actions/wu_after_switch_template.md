# Action: wu_after_switch_template

Permite dezvoltatorilor de plugin să execute funcții (să "hook") după ce un utilizator sau un super administrator își schimbă template-ul site-ului.

## Parametri {#parameters}

| Nume | Tip | Descriere |
|------|------|-------------|
| $id | `int` | ID-ul site-ului |
