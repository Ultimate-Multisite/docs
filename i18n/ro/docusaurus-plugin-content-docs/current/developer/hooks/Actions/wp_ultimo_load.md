# Acțiune: wp_ultimo_load {#action-wpultimoload}

Se declanșează când toate dependențele au fost încărcate

Permite dezvoltatorilor de plugin-uri să adauge funcționalități noi. De exemplu, suport pentru noi furnizori de găzduire (hosting), etc.
