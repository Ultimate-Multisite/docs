# Filtru: wu_add_new_domain_modal_fields {#filter-wuaddnewdomainmodalfields}

Filtrează câmpurile pentru modulul de adăugare a unui domeniu nou.

Acest filtru permite addon-urilor (de exemplu, Domain Seller) să modifice sau să înlocuiască câmpurile de înregistrare a domeniului.

## Parametri {#parameters}

| Nume | Tip | Descriere |
|------|------|-------------|
| $fields | `array` | Câmpurile formularului. |

- 2.1.0
