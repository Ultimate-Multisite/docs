# Filtru: wu_documentation_get_link

Permite dezvoltatorilor de plugin să filtreze link-urile.

Nu sunt sigur cât de util ar putea fi, dar nu face rău să îl avem.

## Parametri

| Nume | Tip | Descriere |
|------|------|-------------|
| $link | `string` | Link-ul înregistrat |
| $slug | `string` | Slug-ul folosit pentru a recupera link-ul |
| $default_link | `string` | Link-ul implicit înregistrat |

### De la

- 1.7.0
### Sursă

Definit în [`inc/class-documentation.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-documentation.php#L125) la linia 125
