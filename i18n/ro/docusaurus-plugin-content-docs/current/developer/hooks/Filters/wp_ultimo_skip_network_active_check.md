# Filtru: wp_ultimo_skip_network_active_check {#filter-wpultimoskipnetworkactivecheck}

Permite dezvoltatorilor să sară peste această verificare.

Acest lucru este util atunci când folosiți setări personalizate bazate pe Composer și alte setări, cum ar fi Bedrock, de exemplu, unde folosirea plugin-urilor ca mu-plugins este norma.

### De la {#since}

- 2.0.0
### Sursă {#source}
