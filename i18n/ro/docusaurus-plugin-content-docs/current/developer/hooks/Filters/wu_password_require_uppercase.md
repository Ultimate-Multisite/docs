# Filtru: wu_password_require_uppercase

Filtrează dacă este necesară utilizarea literelor majuscule în parolele de protecție.

## Parametri

| Nume | Tip | Descriere |
|------|------|-------------|
| $require | `bool` | Dacă se cere litere majuscule. Implicit este setat pe `true` când regulile sunt aplicate. |
| $defender_active | `bool` | Dacă este activ pluginul Defender Pro Strong Password. |

### De la

- 2.4.0
### Sursă

Definit în [`inc/class-scripts.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-scripts.php#L553) la linia 553
