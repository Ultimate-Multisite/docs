# Filtru: wu_maybe_redirect_to_admin_panel_exclusion_list

Creează o listă de excludere de parametri care previne redirecționarea automată.

Acest lucru este necesar deoarece, altfel, constructorul de pagini nu va putea edita pagina de login odată ce aceasta este definită.

### De la versiunea

- 2.0.4
