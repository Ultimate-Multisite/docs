# Filtru: wu_pre_save_settings {#filter-wupresavesettings}

Permite dezvoltatorilor să filtreze setările înainte de salvare, folosind Ultimate Multisite.

## Parametri {#parameters}

| Nume | Tip | Descriere |
|------|------|-------------|
| $settings | `array` | Setările care urmează a fi salvate. |
| $settings_to_save | `array` | Setările noi de adăugat. |
| $saved_settings | `array` | Setările actuale salvate. |

### Începând cu {#since}

- 2.0.18
