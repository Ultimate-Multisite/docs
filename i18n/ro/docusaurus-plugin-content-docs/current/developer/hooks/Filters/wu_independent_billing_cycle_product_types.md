# Filter: wu_independent_billing_cycle_product_types

Filtrează tipurile de produse care au cicluri de facturare independente.

Produsele cu cicluri de facturare independente nu sunt forțate să se potrivească perioadei de facturare a planului în coș. Acest lucru este util pentru produse precum înregistrările de domenii, care sunt facturate anual indiferent dacă planul este lunar sau anual.

## Parametri {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | Un array de tipuri de produse cu cicluri de facturare independente. |
