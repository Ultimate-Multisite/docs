# Filter: wu_upgrade_to_unlock_url

Permite dezvoltatorilor să modifice URL-ul de upgrade necesar pentru deblocarea conținutului.

## Parametri {#parameters}

| Name | Type | Descriere |
|------|------|-------------|
| $url | `string` | URL-ul de upgrade. |
| $args | `array` | Modul și tipul de limită care necesită upgrade. |
