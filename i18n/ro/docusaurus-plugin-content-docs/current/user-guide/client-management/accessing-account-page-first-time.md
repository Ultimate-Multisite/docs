# Cum pot accesa pagina contului meu pentru prima dată?

Bine ai venit la Ultimate Multisite!

Accesează pagina contului tău prin link-ul primit în e-mail după achiziționarea plugin-ului nostru:

<!-- Screenshot unavailable: Account page email with access link -->

Folosește adresa de e-mail pe care ai furnizat-o la achiziție și parola primită prin e-mail.

<!-- Screenshot unavailable: Login page with email and password fields -->

Deoarece aceasta este prima ta autentificare în acest cont, ți se va cere să schimbi parola. Schimb-o și apasă butonul _Salvează_.

<!-- Screenshot unavailable: Change password form with Save button -->

În pagina contului tău poți gestiona toate detaliile contului, cum ar fi facturarea, reînnoirile, anulările și multe altele.
