# Cum să obții ajutor pentru dezvoltare personalizată de la Codeable?

Uneori, avem nevoie de personalizări avansate pentru ca fluxurile rețelei noastre să funcționeze exact așa cum ne dorim. Dacă asta cauți, îți recomandăm [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), o platformă unde poți găsi dezvoltatori experimentați, cu expertiză în Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) este o platformă care conectează clienții cu dezvoltatori WordPress freelance calificați, capabili să lucreze la proiecte de orice dimensiune. Au un proces riguros de selecție pentru a te asigura că ești pus în legătură cu experți calificați specific pentru sarcina pe care o ai în vedere.

### Cum să creezi un proiect în Codeable?

Mai întâi, accesează [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) și dă clic pe butonul **Start a Project**.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Acum, pentru a trimite un proiect, dă clic în stânga pe **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

La „I need help with", alege **Customizing**:

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

La „My", alege **Plugin**:

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Apoi dă clic pe butonul **Continue**.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Acum e momentul să descrii proiectul și nevoile tale cât mai bine posibil. Încearcă să oferi cât mai multe informații!

<!-- Screenshot unavailable: Codeable project description form -->

Răspunde la următoarele întrebări pentru a ajuta la stabilirea prețului.

<!-- Screenshot unavailable: Codeable project pricing questions -->

După aceea, dacă nu ai încă un cont [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), creează unul și dă clic pe butonul **Publish My Project**.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

După ce ai publicat proiectul tău pe [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), platforma te va conecta cu dezvoltatori care corespund setului de competențe necesar pentru proiectul tău.

Acest proces te va ajuta să găsești persoana potrivită pentru jobul tău, cu experiență în WordPress și Ultimate Multisite!

Tariful minim la Codeable este de 70–120 USD pe oră. Estimările vor varia în funcție de complexitatea lucrării, dificultatea și urgența proiectului.

După ce primești estimarea, vei putea angaja dezvoltatorul care se potrivește cel mai bine nevoilor tale și să începeți proiectul împreună!
