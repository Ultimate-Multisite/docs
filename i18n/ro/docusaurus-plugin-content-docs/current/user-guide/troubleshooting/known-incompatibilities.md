# Incompatibilități cunoscute cu alte pluginuri

După instalarea și activarea Ultimate Multisite, s-ar putea să întâmpini probleme din cauza unor incompatibilități nerezolvate cu alte pluginuri.

Lucrăm activ pentru a reduce această listă cu fiecare nouă versiune lansată, dar dacă un plugin apare aici, este posibil să fie nevoie să îl dezactivezi pentru ca rețeaua Ultimate Multisite să funcționeze corect.

Iată o listă cu acestea și ce ar trebui să faci în aceste cazuri:

**Unele pluginuri WPMU DEV** Unele dintre pluginurile care _nu mai sunt_ întreținute de WPMU DEV pot cauza blocarea fluxului de înregistrare Ultimate Multisite în ultimul pas (utilizatorul nu reușește să își creeze un cont). Iată o listă cu pluginurile care cauzează aceste probleme:

  * Signup Language: _Ultimate Multisite are un add-on care oferă aceeași funcționalitate, numit Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Această funcționalitate este inclusă în Ultimate Multisite, deci nu ai nevoie de un plugin suplimentar._

  * New Blog Templates: _Această funcționalitate este inclusă în Ultimate Multisite, deci nu ai nevoie de un plugin suplimentar._

  * Set Password on Multisite Blog Creation: _Această funcționalitate este inclusă în Ultimate Multisite, deci nu ai nevoie de un plugin suplimentar._

  * Signup TOS: _Această funcționalitate este inclusă în Ultimate Multisite, deci nu ai nevoie de un plugin suplimentar._

  * Anti-Splog: Deși mai puțin puternică, această funcționalitate este de asemenea _inclusă în Ultimate Multisite._

**Notă**: Toate celelalte pluginuri WPMU Dev pot fi folosite fără probleme împreună cu Ultimate Multisite. Printre exemple se numără _Smush_, _Forminator_, _Defender_ etc.
