# Plasarea reclamelor în rețeaua ta cu Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector este un **add-on gratuit** care îți oferă instrumentele necesare pentru a plasa reclame în planurile tale direct din pagina de editare a acestora.

După ce instalezi Ultimate Multisite: Ad Injector, mergi la pagina _Plugins_ pentru a-l activa.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Apasă pe butonul Network Activate_

Odată ce add-on-ul este activ, caseta meta _Advanced Options_ din ecranul de editare a planului va avea o nouă opțiune numită _Ad Settings_.

![Ad Settings în ecranul de editare a planului](/img/config/product-edit-full.png)

_Acum poți edita setările pentru reclame pentru fiecare dintre planurile tale_

În acel tab, vei putea adăuga codurile de reclame pentru articole și pagini în front-end, înainte și după conținut.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_După ce inserezi codul reclamei, aceasta va apărea pe articolele și paginile planului respectiv._

Poți avea, de asemenea, reclame incluse în header-ul paginilor de administrare pentru utilizatorii acelui plan.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Mergi la opțiunea Admin Panel Ad Code dacă vrei să afișezi reclame în back-end-ul utilizatorilor tăi_

### Dincolo de publicitate

Poți folosi Ultimate Multisite: Ad Injector și pentru a oferi suport planurilor de nivel superior.

Mergi pur și simplu la opțiunea _Admin Panel Ads_ și adaugă codul HTML pentru instrumentul tău de suport.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Adaugă suport pentru clienți la planuri specifice_

În acest fel, oamenii te pot contacta direct din panoul lor de administrare!
