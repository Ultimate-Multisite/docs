# Personalizarea panoului de administrare cu PRO Theme

Adesea, când lucrăm pentru clienți, avem nevoie să le oferim o experiență personalizată în panoul de administrare WordPress. Dacă vrei un aspect mai profesional, PRO Theme te poate ajuta.

PRO Theme este o temă de administrare creată special pentru a te ajuta să personalizezi Dashboard-ul clienților tăi. De asemenea, face parte din nucleul de plugin-uri premium al Ultimate Multisite.

După instalare, mergi la pagina _Plugins_ și activează-l în rețeaua ta.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Activează-l la nivel de rețea din pagina Plugins_

_**NOTĂ:**_ _Dacă doar îl instalezi fără a-l activa la nivel de rețea, va apărea pentru clienții tăi ca o opțiune pe care o pot bifa pentru a aplica tema doar contului lor._

Acum ai interfața de administrare complet reproiectată! Vei vedea și un nou meniu în coloana din stânga, numit PRO Theme.
