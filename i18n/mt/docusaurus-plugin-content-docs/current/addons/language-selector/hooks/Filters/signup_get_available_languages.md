# Filtru: signup_get_available_languages {#filter-signupgetavailablelanguages}

Jiffiltra l-lista ta’ lingwi disponibbli għal reġistrazzjonijiet ta’ siti fil-front-end.

Li tgħaddi array vojta lil dan il-hook se jiddiżattiva l-output tas-setting fuq il-formola tar-reġistrazzjoni, u l-lingwa predefinita tintuża meta jinħoloq is-sit. Lingwi li ma jkunux diġà installati jitneħħew.

## Parametri {#parameters}

| Isem | Tip | Deskrizzjoni |
|------|------|-------------|
| $available_languages | `array` | Lingwi disponibbli. |
