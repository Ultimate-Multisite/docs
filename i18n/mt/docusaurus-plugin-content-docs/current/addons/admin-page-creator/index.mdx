# Ħallieq ta' Paġni tal-Admin {#admin-page-creator}

## Kontenut {#content}
Il-ħallieq ta' paġni tal-admin jippermettilek iżżid faċilment paġni tal-admin u sotto-paġni ġodda fi ħdan id-Dashboard tal-admin ta' WordPress. Jista' jintuża biex tinħoloq paġna ta' merħba jew paġna ta' tagħlim li tista' sservi bħala landing page għal utenti ġodda. Tista' tuża page builder bħal Beaver Builder, Elementor, Brizy, u Oxygen biex tiddisinja l-paġni mingħajr taħbit.

Joffri wkoll appoġġ għal HTML, CSS u JavaScript personalizzati, kif ukoll kontroll tal-aċċess ibbażat fuq il-pjan u r-rwol tal-utent!

## Karatteristiċi {#features}
* Appoġġ għal HTML u CSS Personalizzati
* Appoġġ għal JavaScript
* Modalità ta' Kontenut PHP
* Permess ibbażat fuq ir-rwol tal-Utent
* Appoġġ għal page builders

## Dokumentazzjoni {#documentation}

## Installazzjoni {#installation}

1. Tella' **ultimate-multisite-admin-page-creator** fid-direttorju **/wp-content/plugins/**
2. Attiva l-plugin permezz tal-menu **Plugins** fin-Network Admin ta' WordPress tiegħek

## Gwida għall-Użu {#usage-guide}

Filwaqt li WordPress jagħmilha tassew faċli biex iżżid kontenut mal-front end tas-siti, il-ħolqien ta' paġni u sotto-paġni fuq in-naħa tal-admin jeħtieġ xi ħiliet bażiċi ta' kkowdjar. Ultimate Multisite: Ħallieq ta' Paġni tal-Admin għandu l-għan li jġib l-istess faċilità ta' użu tal-front end għall-back end tas-siti tan-network tiegħek.

## Għaliex tuża l-Ħallieq ta' Paġni tal-Admin? {#why-use-admin-page-creator}

Forsi trid toffri paġna personalizzata fuq il-panel tal-admin tal-klijenti tiegħek b'tutorials, forsi trid iżżid paġna personalizzata b'FAQs u widget ta' kuntatt. Il-possibbiltajiet huma bla tmiem bil-Ħallieq ta' Paġni tal-Admin, u dan huwa dak li se nesploraw fit-tutorial tal-lum.

## Ħolqien tal-ewwel Paġna tal-Admin Personalizzata tiegħek {#creating-your-first-custom-admin-page}

Wara li tinstalla u tattiva Ultimate Multisite: Ħallieq ta' Paġni tal-Admin, mur fil-menu tal-admin tan-network tiegħek u mur fuq Paġni tal-Admin.

![Lista ta' Paġni tal-Admin Personalizzati fuq Network Admin](/img/admin/admin-pages-list.png)

_Lista ta' Paġni tal-Admin Personalizzati fuq in-Network Admin_

Se tara lista tal-paġni kollha tal-admin li ħloqt s'issa (peress li din hija l-ewwel żjara tiegħek, il-lista tkun vojta). Kompli u kklikkja **Żid Paġna tal-Admin ġdida**. Tiġi ridirett lejn l-editur tal-Paġna tal-Admin.

## Għażliet tal-Menu {#menu-options}

Fuq in-naħa, fil-meta-box tal-Għażliet Ġenerali, tkun tista' tikkonfigura t-tip tal-menu, it-tikketta, u ħafna aktar.

### Tipi ta' Menu {#menu-types}

Il-paġna tal-admin personalizzata ġdida tiegħek tista' jkollha żewġ tipi ta' menu differenti: menu ta' livell għoli u submenu.

Paġna ta' menu ta' livell għoli, kif jissuġġerixxi l-isem, iżżid il-paġna li għadek kif ħloqt mal-menu tal-admin flimkien mal-paġni predefiniti ta' WordPress.

Jekk tagħżel l-għażla ta' livell għoli, tkun tista' tagħżel l-ordni tal-menu u ikona tal-menu wkoll.

Jekk trid tagħmel din il-paġna paġna ta' sub-menu, tkun tista' tagħżel paġna ġenitur biex torbot din il-paġna magħha. Tista' żżid il-paġna personalizzata tiegħek mal-paġni predefiniti ta' WordPress jew mal-paġni tal-admin personalizzati tiegħek ta' livell għoli.

## Għażliet Avvanzati {#advanced-options}

Għandek tkun tista' żżid kull xorta ta' kontenut mal-paġni tal-admin personalizzati tiegħek. Dan jinkludi regoli u fajls CSS, JavaScript, u libreriji esterni wkoll. Tista' tagħmel dan billi tuża t-Tab tal-Għażliet Avvanzati.

### CSS Personalizzat {#custom-css}

Fuq it-tab CSS, tista' żżid il-kodiċi CSS personalizzat tiegħek stess, kif ukoll timporta fajls CSS minn siti remoti (bħal CDNs, Google Fonts, eċċ).

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_CSS Personalizzat huwa appoġġjat ukoll. Tista' wkoll tinkludi fajls ta' stil esterni!_

### JavaScript Personalizzat {#custom-javascript}

L-istess japplika għal JavaScript. Tista' żżid libreriji esterni biex tużahom fil-kodiċi JavaScript personalizzat tiegħek.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Żid il-kodiċi JavaScript tiegħek stess!_

## Permessi {#permissions}

Ukoll fl-Għażliet Avvanzati, tkun tista' tagħżel liema sotto-siti se jkollhom aċċess għal din il-paġna. Dan huwa utli jekk qed tippjana li sservi kontenut esklussiv differenti għal livelli differenti ta' pjanijiet, pereżempju.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Għandek kontroll totali fuq fejn se tidher din il-paġna_

**Importanti** : Iż-żewġ kundizzjonijiet iridu jiġu sodisfatti biex il-paġna tintwera. Dan ifisser li jekk l-utent ikun klijent ta’ plan partikolari, iżda ma jkollux wieħed mir-rwoli magħżula (jew bil-maqlub) il-paġna ma tidhirx fuq il-pannell tiegħu.

## Attiva {#active}

Id-diżattivazzjoni ta’ paġna ta’ livell għoli se tiddiżattiva awtomatikament is-sub-paġni kollha marbuta ma’ dik il-paġna ġenitur.
