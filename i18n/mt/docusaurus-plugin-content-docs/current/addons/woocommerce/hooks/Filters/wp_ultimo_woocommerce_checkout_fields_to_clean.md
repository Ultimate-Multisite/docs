# Filtru: wp_ultimo_woocommerce_checkout_fields_to_clean

Jippermetti lill-iżviluppaturi jiffiltraw l-oqsma awtomatiċi tal-indirizz tal-kontijiet ta' WooCommerce.

Peress li Ultimate Multisite għandu oqsma tal-indirizz tal-kontijiet, jista' jagħmel sens li jitneħħew l-oqsma awtomatiċi murija minn WooCommerce fuq il-formola tal-checkout. B'mod awtomatiku, l-ebda valur ma jgħaddi lil din il-lista, għalhekk jinżammu l-oqsma kollha kkonfigurati fuq WooCommerce biex jintwerew.

- 2.0.0
### Sors

## Jirritorna
Il-lista ta' oqsma li għandhom jitneħħew. Il-valuri disponibbli jinkludu billing_first_name, billing_last_name, billing_company, billing_country, billing_address_1, billing_address_2, billing_city, billing_state, billing_postcode, billing_email, u billing_phone.
