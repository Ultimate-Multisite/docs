# Bennej ta' Siti bl-AI

ultimate-multisite-ai-site-builder

## Rekwiżiti

- WordPress 5.3 jew ogħla
- PHP 7.4 jew ogħla
- plugin Multisite Ultimate (attiv)

## Installazzjoni

1. Tella' l-fajls tal-addon fid-direttorju tiegħek `/wp-content/plugins/`
2. Attiva l-plugin permezz tal-menu 'Plugins' f'WordPress
3. Ikkonfigura s-settings tal-addon fl-amministratur tan-network

## Żvilupp

### Setup

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Ittestjar

L-addon jinkludi qafas komprensiv għall-ittestjar:

- **Testijiet tal-Unità**: Testijiet ibbażati fuq PHPUnit bis-suite tat-testijiet ta' WordPress
- **Klassi Bażi tat-Test**: Tipprovdi metodi ta' għajnuna għal xenarji komuni tat-testijiet
- **Kopertura tat-Testijiet**: Ikkonfigurata biex issegwi l-kopertura tal-code
- **Ittestjar Multisite**: It-testijiet jitħaddmu f'ambjent multisite

### Standards tal-Code

- **Standards tal-Coding ta' WordPress**: Infurzati permezz ta' PHPCS
- **Kompatibbiltà ma' PHP 7.4+**: Karatteristiċi moderni ta' PHP appoġġjati
- **Analiżi Statika**: Integrazzjoni ta' PHPStan għall-iċċekkjar tat-tipi
- **Immodernizzar tal-Code**: Regoli ta' Rector għal aġġornamenti ta' PHP

### Sistema tal-Build

- **Ipproċessar tal-Assets**: Minifikazzjoni għal fajls CSS/JS
- **Traduzzjoni**: Ġenerazzjoni tal-fajl POT
- **Ippakkjar**: Ħolqien ta' arkivju għad-distribuzzjoni
- **Modalità ta' Żvilupp**: Assets mhux minifikati għad-debugging

## Konfigurazzjoni

L-addon jappoġġja diversi għażliet ta' konfigurazzjoni aċċessibbli permezz tal-interface tal-amministratur tan-network.

## Hooks u Filters

### Azzjonijiet

- `ultimate-multisite-ai-site-builder_init` - Jitħaddem wara l-inizjalizzazzjoni tal-addon
- `ultimate-multisite-ai-site-builder_loaded` - Jitħaddem wara li l-fajls kollha tal-addon jitgħabbew

### Filters

- `ultimate-multisite-ai-site-builder_settings` - Immodifika s-settings tal-addon
- `ultimate-multisite-ai-site-builder_enabled` - Aqbeż l-istatus ta' attivazzjoni/diżattivazzjoni tal-addon
