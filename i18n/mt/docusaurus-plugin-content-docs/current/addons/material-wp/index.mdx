# Material WP {#material-wp}

L-interfaċċa tal-amministrazzjoni ta' WordPress ilha l-istess għal żmien twil. Imma qatt ħsibt kif tkun tidher verżjoni Material Design tal-interfaċċa tal-amministrazzjoni? 

Material WP hija theme tal-amministrazzjoni żviluppata biex tagħtik dik id-dehra u s-sensazzjoni, u tgħinek tipprovdi lill-klijenti tiegħek b'Dashboard aktar personalizzat.

## Karatteristiċi {#features}

- Appoġġ għal CSS personalizzat
- Settings faċli għall-istil
- Tibdil fl-ordni tal-menu
- Importazzjoni/esportazzjoni faċli

## Dokumentazzjoni {#documentation}

**Personalizza l-Panel tal-Amministrazzjoni tiegħek b'Material WP**

## Installazzjoni {#installation}

1. Tella' 'material-wp' fid-direttorju '/wp-content/plugins/'
2. Attiva l-plugin permezz tal-menu 'Plugins' f'WordPress

## Gwida għall-Użu {#usage-guide}

L-interfaċċa tal-amministrazzjoni ta' WordPress ilha l-istess għal żmien twil. Imma qatt ħsibt kif tkun tidher verżjoni Material Design tal-interfaċċa tal-amministrazzjoni?

Material WP hija theme tal-amministrazzjoni żviluppata biex tagħtik dik id-dehra u s-sensazzjoni, u tgħinek tipprovdi lill-klijenti tiegħek b'Dashboard aktar personalizzat. Material WP hija wkoll parti mill-qalba tal-plugins premium ta' Ultimate Multisite.

Wara li **tinstallaha**, mur fil-paġna _Plugins_ tiegħek u kklikkja biex tattivaha fuq in-Netwerk tiegħek.
