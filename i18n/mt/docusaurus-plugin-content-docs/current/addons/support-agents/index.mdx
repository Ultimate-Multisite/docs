# Aġenti ta' Appoġġ {#support-agents}

Il-ġestjoni ta' netwerk sħiħ waħdek mhijiex biċċa xogħol faċli. Hekk kif in-netwerk tiegħek jibda jikber, tibda tara domanda dejjem tikber għall-appoġġ. Ikun tajjeb li ddaħħal nies li jgħinuk tnaqqas il-piż tax-xogħol tiegħek. Madankollu, li tagħti aċċess/permess sħiħ huwa riskjuż u jista' jikkomprometti n-netwerk tiegħek. Hemmhekk jidħol dan l-add-on tal-Aġent ta' Appoġġ.

Dan l-add-on jippermettilek toħloq tip ġdid ta' utent ta' WordPress imsejjaħ Aġent ta' Appoġġ, u jagħtik kontroll totali fuq liema permessi għandu dak l-utent. L-għoti ta' permess huwa faċli daqs li tixgħel buttuna toggle u titfiha jekk tiddeċiedi li tneħħih.

## Karatteristiċi {#features}

- Lista mqassma ta' permessi ta' WordPress u Ultimate Multisite
- Ippermetti u ddiżattiva l-aċċess b'faċilità
- Oħloq numru bla limitu ta' kontijiet ta' Aġenti

## Installazzjoni {#installation}

1. Tella' 'ultimate-multisite-support-agents' fid-direttorju '/wp-content/plugins/'
2. Attiva l-plugin permezz tal-menu 'Plugins' f'WordPress
3. Segwi l-Wizard pass pass biex twaqqaf il-plugin
