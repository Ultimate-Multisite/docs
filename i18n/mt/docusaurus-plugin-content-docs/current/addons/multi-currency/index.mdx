# Multi-Currency {#multi-currency}

Ultimate Multisite: Multi-Currency jippermetti lin-network tiegħek jaċċetta pagamenti f'diversi muniti. Jikkonverti l-prezzijiet awtomatikament billi juża rati tal-kambju diretti jew manwali, iħallik tissettja prezzijiet eżatti għal kull prodott għal kwalunkwe munita, u jidentifika l-munita preferuta ta' kull viżitatur permezz tal-ġeolokalizzazzjoni — kollox immaniġġjat minn Dashboard wieħed tal-amministrazzjoni.

## Karatteristiċi Ewlenin {#key-features}

- **Konverżjoni awtomatika tar-rati tal-kambju** — il-prezzijiet jiġu kkonvertiti mill-munita bażi tiegħek billi jintużaw rati miġjuba mill-fornitur magħżul tiegħek
- **Sovrascrizzjonijiet tal-ipprezzar għal kull prodott** — issettja prezzijiet eżatti speċifiċi għall-munita fuq kwalunkwe prodott biex taqbeż il-konverżjoni awtomatika
- **Qasam tas-selettur tal-munita fir-reġistrazzjoni** — iħalli lill-klijenti jagħżlu l-munita preferuta tagħhom waqt il-ħlas/reġistrazzjoni
- **Fornituri multipli tar-rati tal-kambju** — ECB, Frankfurter, Stripe, jew rati kompletament manwali
- **Identifikazzjoni tal-munita bbażata fuq il-ġeolokalizzazzjoni** — tagħżel minn qabel awtomatikament il-munita li taqbel mal-pajjiż tal-viżitatur
- **Dashboard tal-amministrazzjoni** — immaniġġja muniti attivi, rati tal-kambju, u settings tal-fornitur minn post wieħed

## Rekwiżiti {#requirements}

- Ultimate Multisite 2.0.0 jew ogħla
- Installazzjoni WordPress Multisite
- PHP 7.4 jew ogħla

## Installazzjoni {#installation}

1. Tella' l-fajls tal-estensjoni fuq `/wp-content/plugins/ultimate-multisite-multi-currency`
2. Attiva l-estensjoni permezz tal-iskrin **Estensjonijiet** f'WordPress (Attiva fin-Network)
3. Mur lejn **Ultimate Multisite → Multi-Currency** biex tiftaħ id-Dashboard tal-amministrazzjoni
4. Żid il-muniti li trid taċċetta u kkonfigura l-fornitur tar-rati tal-kambju tiegħek

## Konfigurazzjoni {#configuration}

### Dashboard tal-Amministrazzjoni {#admin-dashboard}

Id-Dashboard tal-amministrazzjoni ta' Multi-Currency (**Ultimate Multisite → Multi-Currency**) huwa l-post ċentrali biex:

- Iżżid, tattiva, jew tiddiżattiva muniti
- Tagħżel fornitur tar-rati tal-kambju u taġġorna r-rati fuq talba
- Tirrevedi r-rati kurrenti għall-muniti attivi kollha
- Tissettja rati manwali meta l-fornituri awtomatiċi ma jkunux xierqa

### Fornituri tar-Rati tal-Kambju {#exchange-rate-providers}

| Fornitur | Deskrizzjoni |
|---|---|
| **ECB** | Bank Ċentrali Ewropew — rati ta' kuljum b'xejn għal networks ibbażati fuq EUR |
| **Frankfurter** | API mera open-source tal-ECB — b'xejn, ma teħtieġ l-ebda API key |
| **Stripe** | Juża r-rati FX diretti ta' Stripe — jeħtieġ integrazzjoni Stripe attiva |
| **Manwali** | Int iddaħħal ir-rati direttament; l-ebda sejħa lil API esterna |

Agħżel il-fornitur tiegħek taħt **Ultimate Multisite → Multi-Currency → Issettjar → Fornitur tar-Rati tal-Kambju**. Ir-rati jistgħu jiġu aġġornati manwalment fi kwalunkwe ħin jew skedati biex jaġġornaw awtomatikament.

### Sovrascrizzjonijiet tal-Ipprezzar għal Kull Prodott {#per-product-pricing-overrides}

Meta l-konverżjoni awtomatika ma tkunx preċiża biżżejjed, tista' tissettja prezz eżatt għal munita speċifika fuq kwalunkwe prodott:

1. Iftaħ l-editur tal-prodott (**Ultimate Multisite → Prodotti → Editja**)
2. Skrollja sas-sezzjoni **Ipprezzar tal-Munita**
3. Daħħal il-prezz eżatt għal kull munita li trid tissovrascrivi
4. Issejvja l-prodott

Prodotti mingħajr sovrascrizzjoni jużaw il-prezz ikkonvertit mill-munita bażi tiegħek.

### Qasam tas-Selettur tal-Munita fir-Reġistrazzjoni {#currency-selector-signup-field}

Il-qasam tas-selettur tal-munita jista' jiżdied mal-fluss tar-reġistrazzjoni/ħlas tiegħek sabiex il-klijenti jagħżlu l-munita preferuta tagħhom waqt ir-reġistrazzjoni:

1. Mur lejn **Ultimate Multisite → Multi-Currency → Issettjar**
2. Attiva **Uri s-selettur tal-munita fir-reġistrazzjoni**
3. B'mod fakultattiv issettja munita predefinita jew ħalli l-ġeolokalizzazzjoni tiddetermina l-valur magħżul minn qabel

Il-munita magħżula tinħażen mal-abbonament u tintuża għall-kontijiet sussegwenti kollha.

### Identifikazzjoni tal-Munita Bbażata fuq il-Ġeolokalizzazzjoni {#geolocation-based-currency-detection}

Meta l-identifikazzjoni tal-ġeolokalizzazzjoni tkun attivata, l-addon jimmappja l-pajjiż tal-viżitatur ma' munita u jagħżilha minn qabel fil-qasam tas-selettur tal-munita. Biex tattiva:

1. Mur lejn **Ultimate Multisite → Multi-Currency → Issettjar**
2. Attiva **Identifika l-munita mill-post tal-viżitatur**
3. Immappja pajjiżi għal muniti fit-tabella **Immappjar tal-Ġeolokalizzazzjoni**

Jekk ma jeżisti l-ebda mmappjar għall-pajjiż ta' viżitatur, tintuża l-munita bażi tan-network bħala l-għażla ta' riżerva.

## Mistoqsijiet Frekwenti {#frequently-asked-questions}

### X'jiġri jekk fornitur tar-rati tal-kambju ma jkunx disponibbli? {#what-happens-if-an-exchange-rate-provider-is-unavailable}

L-addon jaqa' lura għall-aħħar rati miġjuba b'suċċess. Jekk qatt ma nġabu rati, jaqa' lura għar-rati manwali li kkonfigurajt. Huwa rakkomandat li tikkonfigura rati manwali bħala xibka ta' sigurtà.

### Nista' nuża fornituri differenti għal muniti differenti? {#can-i-use-different-providers-for-different-currencies}

Le — fornitur wieħed ikun attiv kull darba għall-muniti kollha. Uża **Manwali** jekk għandek bżonn kontroll sħiħ fuq rati individwali.

### Il-ġeolokalizzazzjoni teħtieġ estensjoni separata? {#does-geolocation-require-a-separate-plugin}

Ma hija meħtieġa l-ebda estensjoni esterna tal-ġeolokalizzazzjoni. L-addon juża l-identifikazzjoni IP-għal-pajjiż integrata f'WordPress. Għal preċiżjoni ogħla, servizz dedikat tal-ġeolokalizzazzjoni jista' jiġi integrat permezz tal-filter hooks ipprovduti.

### Is-sovrascrizzjonijiet għal kull prodott jiġu applikati qabel jew wara t-taxxa? {#are-per-product-overrides-applied-before-or-after-tax}

Is-sovrascrizzjonijiet għal kull prodott jistabbilixxu l-prezz qabel it-taxxa fil-munita fil-mira. It-taxxa tiġi kkalkulata fuq il-prezz tas-sovrascrizzjoni billi tintuża l-konfigurazzjoni tat-taxxa eżistenti tiegħek.
