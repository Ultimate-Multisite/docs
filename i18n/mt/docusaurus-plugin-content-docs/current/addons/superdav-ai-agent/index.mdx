# Superdav AI Agent

Superdav AI Agent huwa assistent AI avvanzat għal WordPress li jestendi Gratis AI Agent b’ħiliet konxji tat-theme, kapaċitajiet imtejba ta’ tiftix fuq l-internet, karatteristiċi ta’ bennej tas-sit, u integrazzjonijiet ta’ awtomazzjoni fakultattivi. Il-verżjoni 1.18.0 iżżid għodod ta’ qari għal Google Calendar, immappjar ta’ kuntatti u parteċipanti, gradi ta’ approvazzjoni umana, deduplikazzjoni tat-tfakkiriet, notifiki SMS ta’ TextBee, proviżjonament ta’ servizz Superdav ġestit, u pakkett Advanced anċillari mqassam separatament għal flussi tax-xogħol ta’ żviluppaturi fdati.

## Karatteristiċi Ewlenin {#key-features}

- **Ħiliet konxji tat-theme** — jadatta awtomatikament il-gwida u l-kapaċitajiet skont it-theme WordPress attiv tiegħek u l-plugins installati
- **Fornituri multipli ta’ tiftix fuq l-internet** — agħżel bejn Brave Search u Tavily għal riżultati tat-tiftix aktar sinjuri u kapaċitajiet avvanzati ta’ riċerka
- **Formoli ta’ kuntatt għall-bennej tas-sit** — oħloq formoli ta’ kuntatt kompletament funzjonali direttament mill-interface taċ-chat billi tuża l-plugin tal-formoli installat tiegħek
- **Bennej tas-sit imtejjeb** — pjanijiet imtejba ta’ bini tas-sit f’diversi fażijiet b’sejba aħjar tal-plugins u rkupru mill-iżbalji
- **Ħiliet tat-theme inkorporati** — gwida speċjalizzata għal Block Themes, Classic Themes, Kadence Blocks, u Kadence Theme
- **Integrazzjoni WooCommerce imtejba** — ristrutturata biex tuża APIs nattivi ta’ WooCommerce għal affidabbiltà u kompatibbiltà aħjar
- **Sejbien awtomatiku tal-fornitur** — isib u jaġġorna awtomatikament il-plugins u t-themes disponibbli meta jiġu attivati jew diżattivati
- **Ġenerazzjoni ta’ logo personalizzat** — iġġenera u inkorpora logos SVG personalizzati b’sanitàzzazzjoni awtomatika sikura għan-namespace
- **Validazzjoni tal-kuntrast tal-paletta** — iċċekkja pari ta’ kuluri għall-konformità mal-aċċessibbiltà WCAG qabel tapplikahom għat-theme tiegħek
- **Skoperta msaħħa bir-ritratti** — tella’ ritratti waqt l-intervista ta’ skoperta ta’ Theme Builder biex tinfluwenza d-direzzjoni tad-disinn
- **Menus għall-ospitalità** — iġġenera paġni strutturati ta’ menus tal-ikel u x-xorb għal ristoranti, kafetteriji, u negozji tal-ospitalità
- **Previews fuq desktop u mobile** — ara preview tad-disinn tiegħek fuq apparati desktop u mobile waqt l-għażla tad-direzzjoni tad-disinn
- **Scraping tas-sit ta’ Livell 1** — sd-ai-agent/site-scrape issa huwa disponibbli b’mod awtomatiku għall-analiżi ta’ websites ta’ kompetituri u ta’ ispirazzjoni
- **Għodod ta’ qari għal Google Calendar** — aqra kalendarji u avvenimenti kkonfigurati sabiex l-awtomazzjonijiet ikunu jistgħu jirraġunaw dwar il-kuntest tal-iskeda li ġejja
- **Immappjar ta’ kuntatti u parteċipanti** — qabbel il-parteċipanti tal-kalendarju ma’ utenti tas-sit u kuntatti magħrufa qabel tiġġenera xogħol ta’ segwitu
- **Gradi ta’ approvazzjoni umana** — waqqaf temporanjament awtomazzjonijiet sensittivi għal reviżjoni qabel ma tibgħat messaġġi, tkompli x-xogħol, jew tagħmel bidliet
- **Rekords u deduplikazzjoni tat-tfakkiriet** — issegwi t-tfakkiriet li diġà ntbagħtu sabiex ġirjiet ripetuti ta’ awtomazzjoni ma jinnotifikawx lill-istess riċevitur darbtejn
- **Notifiki SMS ta’ TextBee** — ibgħat notifiki ta’ messaġġi testwali kkonfigurati permezz ta’ TextBee meta fornitur SMS ikun attivat
- **Proviżjonament ta’ servizz Superdav ġestit** — qabbad siti appoġġati ma’ endpoints Superdav ospitati b’setup awtomatiku minflok konfigurazzjoni manwali tal-endpoint
- **Pakkett Advanced anċillari** — għodod ta’ żviluppaturi fdati mqassma separatament għal flussi tax-xogħol ta’ filesystem, database, WP-CLI, dispatcher REST, bennej tal-plugin, snapshot ta’ git, ġestjoni tal-utenti, u benchmark

## Rekwiżiti {#requirements}

- WordPress 6.0 jew ogħla
- PHP 8.1 jew ogħla
- WordPress Multisite (attivat fuq in-network) jew installazzjoni ta’ sit wieħed
- Plugin attiv ta’ konnettur ta’ fornitur AI (Anthropic, OpenAI, jew kompatibbli)
- Gratis AI Agent v1.9.0 jew ogħla (Superdav AI Agent jestendi Gratis AI Agent)

## Installazzjoni {#installation}

1. Tella’ l-fajls tal-plugin għal `/wp-content/plugins/superdav-ai-agent`
2. Attiva l-plugin permezz tal-iskrin **Plugins** f’WordPress
3. Kun żgur li Gratis AI Agent huwa attivat ukoll
4. Mur lejn **Gratis AI Agent → Settings** u kkonfigura l-fornitur tiegħek tat-tiftix fuq l-internet, il-konnessjoni mas-servizz Superdav ġestit, Google Calendar, jew il-kredenzjali ta’ TextBee kif meħtieġ

Il-pakkett ta’ WordPress.org fih il-karatteristiċi ewlenin ta’ Superdav AI Agent. Il-pakkett Advanced anċillari jinbena u jitqassam separatament għal ambjenti fejn għodod ta’ żviluppaturi fdati jiġu installati u awtorizzati b’mod espliċitu.

## Ħiliet Konxji tat-Theme {#theme-aware-skills}

Superdav AI Agent isib awtomatikament it-theme attiv tiegħek u l-plugins installati, u jgħabbi l-ħila xierqa konxja tat-theme fil-kuntest tal-assistent. Il-ħiliet disponibbli jinkludu:

| Ħila | Japplika Għal |
|---|---|
| **Block Themes** | Themes li jużaw l-arkitettura ta’ WordPress ibbażata fuq blocks (Full Site Editing) |
| **Classic Themes** | Themes tradizzjonali ta’ WordPress li jużaw templates PHP u l-editur klassiku |
| **Kadence Blocks** | Siti bil-plugin Kadence Blocks attiv |
| **Kadence Theme** | Siti bit-theme Kadence bħala t-theme attiv |

Ara [Ħiliet Konxji tat-Theme](../../user-guide/configuration/theme-aware-skills) għal gwida dettaljata dwar kull ħila.

## Fornituri ta’ Tiftix fuq l-Internet {#internet-search-providers}

Superdav AI Agent v1.10.0 iżid **Tavily** bħala alternattiva għal Brave Search għal mistoqsijiet ta’ tiftix fuq l-internet. Iż-żewġ fornituri joffru riżultati tat-tiftix f’ħin reali u jistgħu jiġu kkonfigurati f’**Gratis AI Agent → Settings → Advanced**.

| Provider | Karatteristiċi | Setup |
|---|---|---|
| **Brave Search** | Riżultati tat-tfittxija veloċi u ffukati fuq il-privatezza | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Riżultati tat-tfittxija aktar sinjuri, kapaċitajiet avvanzati ta’ riċerka | [Tavily API](https://tavily.com/) |

Ara [Tfittxija fuq l-Internet](../../user-guide/configuration/internet-search) għal struzzjonijiet ta’ konfigurazzjoni.

## Flussi tax-Xogħol tal-Kalendarju, Approvazzjoni, u Notifiki {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 iwessa’ l-flussi tax-xogħol tal-awtomazzjoni lil hinn mill-kontenut u l-kompiti tal-bini tas-sit:

- **Għodod ta’ Google Calendar** iħallu lill-agent jaqra kalendarji kkonfigurati u dettalji tal-avvenimenti meta jkunu disponibbli l-kredenzjali tal-kalendarju.
- **Għajnuniet għall-kuntatti u l-parteċipanti** jimmappjaw il-parteċipanti tal-avveniment ma’ utenti ta’ WordPress jew kuntatti magħrufa qabel jippreparaw tfakkiriet jew kompiti ta’ segwitu.
- **Bibien ta’ approvazzjoni umana** jippermettu li awtomazzjoni tieqaf sakemm utent awtorizzat jirrevedi l-azzjoni proposta.
- **Rekords ta’ tfakkiriet** jipprevjenu tfakkiriet duplikati tal-kalendarju jew tan-notifiki meta awtomazzjoni terġa’ tiġi ppruvata jew skedata ripetutament.
- **Appoġġ għal TextBee SMS** jippermetti notifiki b’messaġġi ta’ test meta l-provider TextBee jkun ikkonfigurat.

Uża dawn l-għodod għal assistenza konxja tal-iskeda, segwitu mal-parteċipanti, u flussi tax-xogħol ta’ tfakkiriet li xorta jeħtieġu reviżjoni umana qabel jintbagħtu messaġġi sensittivi.

## Konfigurazzjoni {#configuration}

Is-settings kollha jinsabu taħt **Gratis AI Agent → Settings** fl-admin ta’ WordPress. Superdav AI Agent iżid l-għażliet ta’ konfigurazzjoni li ġejjin:

| Setting | Deskrizzjoni |
|---|---|
| **Internet Search Provider** | Agħżel bejn Brave Search u Tavily (mhux obbligatorju) |
| **Tavily API Key** | API key għal tfittxija fuq l-internet ta’ Tavily (jekk Tavily jintgħażel) |
| **Managed Superdav Service** | Qabbad siti appoġġjati mas-servizz ospitat ta’ Superdav b’proviżjoni awtomatika ta’ endpoints |
| **Google Calendar Credentials** | Ikkonfigura aċċess għall-kalendarju sabiex l-agent ikun jista’ jaqra kalendarji u avvenimenti |
| **TextBee SMS Provider** | Ikkonfigura kredenzjali ta’ TextBee għal flussi tax-xogħol ta’ notifiki SMS approvati |

## Pakkett Companion Avvanzat {#advanced-companion-package}

Superdav AI Agent v1.18.0 iżid pakkett companion Avvanzat imqassam separatament għal flussi tax-xogħol ta’ żviluppaturi fdati. Dawn il-kapaċitajiet mhumiex inklużi fil-pakkett ewlieni ta’ WordPress.org. Installa l-pakkett companion biss fuq siti fejn l-amministraturi jafdaw b’mod espliċitu lill-agent biex jaċċessa għodod għall-iżviluppaturi.

L-għodod avvanzati jinkludu operazzjonijiet tas-sistema tal-fajls, aċċess għad-database, WP-CLI, flussi tax-xogħol ta’ dispatcher REST, utilitajiet tal-bini ta’ plugins, snapshots ta’ git, għajnuniet għall-immaniġġjar tal-utenti, u għodod ta’ benchmark. Żomm dawn l-għodod limitati għal amministraturi fdati u irrevedi l-konfigurazzjoni tal-bieb tal-approvazzjoni qabel tattivahom fuq siti ta’ produzzjoni.
