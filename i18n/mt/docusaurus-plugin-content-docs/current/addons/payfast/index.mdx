# PayFast {#payfast}

Ultimate Multisite: Payfast Gateway

Iżid Payfast - gateway tal-ħlasijiet online veloċi u sigur fl-Afrika t’Isfel - bħala għażla ta’ ħlas għal Ultimate Multisite.

## Installazzjoni {#installation}

1. Tella’ 'ultimate-multisite-payfast' fid-direttorju '/wp-content/plugins/'
2. Attiva l-plugin permezz tal-menu 'Plugins' f’WordPress
3. Segwi l-Wizard pass pass biex twaqqaf il-plugin

## Gwida għall-Użu {#usage-guide}

PayFast huwa proċessur ewlieni tal-ħlasijiet online għan-negozji fl-Afrika t’Isfel. Huwa ddedikat biex jgħin lin-negozjanti online jikbru billi jagħmel it-tranżazzjonijiet bla xkiel u siguri. Skont it-talba tal-utent ta’ Ultimate Multisite, iddeċidejna li noħolqu add-on li jagħmel l-integrazzjoni ma’ PayFast possibbli. Dan l-artiklu se juri kif twaqqaf PayFast bħala l-gateway tal-ħlas tiegħek biex tirċievi ħlas fin-network Ultimate Multisite tiegħek.

L-ewwel pass huwa li tinstalla l-add-on.

# Kif tinstalla l-add-on ta’ Payfast: {#how-to-install-the-payfast-add-on}

Mur fil-paġna tal-add-ons u agħżel l-add-on Ultimate Multisite: Payfast Gateway.

![Paġna tal-add-ons](/img/admin/addons-page.png)

It-tieqa tal-add-on se titfaċċa. Ikklikkja fuq Install Now.

![Paġna tal-add-ons b’għażla ta’ installazzjoni](/img/admin/addons-page.png)

Wara li l-installazzjoni tkun lesta, tiġi ridirett lejn il-paġna tal-plugins. Hawnhekk, ikklikkja biss fuq Network Activate u l-add-on _Payfast_ jiġi attivat fuq in-network tiegħek.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# Twaqqif ta’ PayFast {#setting-up-payfast}

Wara li tinstalla l-add-on, issa tista’ tmur fis-settings ta’ Ultimate Multisite u tibda tikkonfigura PayFast.

Taħt il-paġna **settings ta’ Ultimate Multisite**, tista’ ssib it-tab "Payments". Taħt din it-tab, għandek tara "**Payfast** " flimkien ma’ buttuna toggle biex tattiva jew tiddiżattiva dan il-gateway tal-ħlas.

![Settings tal-gateway tal-ħlas](/img/config/settings-payment-gateways.png)Kompli u kklikkja l-buttuna toggle biex tattivah. Imbagħad tespandi l-kaxxa biex turi oqsma differenti li trid timla, bħal **Mechant ID, Merchant Key u PassPhrase.** Tista’ tikseb din l-informazzjoni mill-Account **Payfast** tiegħek

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Il-pass li jmiss huwa li tidħol fil-Account PayFast tiegħek biex tikseb l-informazzjoni li għandek bżonn.

Malli tidħol, tista’ tara minnufih il-kredenzjali li għandek bżonn mid-Dashboard tal-Account **Payfast** tiegħek.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->Tista’ sempliċement tikkopja u twaħħal il-kredenzjali u timla d-dettalji fis-settings ta’ Ultimate Multisite tiegħek. Imbagħad ikklikkja l-buttuna **"Save settings** " biex issalva l-bidliet.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Dak kollox, l-integrazzjoni tal-ħlasijiet PayFast tiegħek hija kompluta!
