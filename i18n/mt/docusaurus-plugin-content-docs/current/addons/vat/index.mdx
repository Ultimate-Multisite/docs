# VAT

Multisite Ultimate: VAT Ewropea

Iżid karatteristiċi u għodod addizzjonali mal-immaniġġjar tat-taxxa predefinit ta' Multisite Ultimate biex jappoġġja aħjar il-VAT Ewropea.
Iżid awtomatikament ir-rati tal-VAT tal-UE mal-paġna tar-Rati tat-Taxxa minn

## Installazzjoni {#installation}

1. Tella' 'ultimate-multisite-vat' fid-direttorju '/wp-content/plugins/'
2. Attiva l-plugin permezz tal-menu 'Estensjonijiet' f'WordPress
3. Mur f'Ultimate Multisite > Settings > Taxes u ppermetti EU VAT.
4. Mur ikklikkja fuq Rati tat-Taxxa u Aġġorna r-Rati tal-VAT tal-UE fil-qiegħ u Ssejvja r-Rati tat-Taxxa.
5. Lest!
