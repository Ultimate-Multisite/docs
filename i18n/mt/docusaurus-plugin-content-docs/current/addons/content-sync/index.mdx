# Sinkronizzazzjoni tal-Kontenut {#content-sync}

Multisite Ultimate: Sinkronizzazzjoni tal-Kontenut

Din l-estensjoni għal Multisite Ultimate tippermetti lill-amministraturi tan-netwerk jissinkronizzaw il-kontenut minn kwalunkwe sit għal siti oħra fin-netwerk. Tappoġġja t-tipi kollha ta' postijiet inklużi postijiet, paġni, tipi ta' postijiet personalizzati, u mudelli ta' Elementor. L-estensjoni tinkludi interface ta' wizard b'diversi passi biex tagħżel il-kontenut u s-siti fil-mira, bi proċessar f'lottijiet u rispons dwar il-progress f'ħin reali.

## Karatteristiċi {#features}

* Għażla tal-Kontenut: Grilja viżwali ta' kards b'minjaturi u filtrazzjoni skont it-tip ta' post
* It-Tipi Kollha ta' Postijiet: Issinkronizza postijiet, paġni, prodotti, mudelli ta' Elementor, u kwalunkwe tip ta' post personalizzat
* Għażla tas-Sit: Agħżel liema siti għandhom jirċievu l-kontenut issinkronizzat, b'esklużjoni tas-sit sors u għażla minn qabel tas-siti sekondarji
* Proċessar f'Lottijiet: Jissinkronizza ma' 10 siti kull darba b'aġġornamenti tal-progress
* Sinkronizzazzjoni tal-Midja: Tikkopja u terġa' timmappja awtomatikament fajls tal-midja (immaġnijiet, videos) mill-kontenut tal-post għas-siti fil-mira
* Sinkronizzazzjoni tat-Tassonomija: Tħares il-kategoriji, it-tags, u tassonomiji personalizzati
* Appoġġ għal Elementor: Appoġġ sħiħ għal mudelli ta' Elementor b'riġenerazzjoni tas-CSS u rimappjar tal-ID tal-mudell
* Riżoluzzjoni tal-Kunflitti: Għażla biex tissostitwixxi kontenut eżistenti jew taqbeż minnu
* Ġestjoni tal-Cache: Tnaddaf awtomatikament il-cache ta' Elementor fuq is-siti fil-mira wara s-sinkronizzazzjoni

## Rekwiżiti {#requirements}

* WordPress 5.3 jew ogħla
* PHP 7.4 jew ogħla
* plugin Multisite Ultimate (attiv)
* plugin Elementor (mhux obbligatorju, meħtieġ għas-sinkronizzazzjoni tal-mudelli ta' Elementor)

## Installazzjoni {#installation}

1. Tella' 'ultimate-multisite-content-sync' fid-direttorju '/wp-content/plugins/'
2. Attiva l-plugin permezz tal-menu 'Plugins' fil-WordPress Network Admin tiegħek
3. Ikkonfigura s-settings f'Ultimate Multisite → Settings → Content Sync
4. Mur f'Ultimate Multisite → Content Sync biex tibda tissinkronizza
