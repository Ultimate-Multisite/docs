# Aħjar Link Shareabili għall-Planijiet

Ultimate Multisite jofflek l-opzione biex tqassma linkijiet tal-planijiet direttament mal-clientijiet tiegħek. Biex tgħmel dan, għandi tmur għal edit pages ta qafas planijiet tiegħek u tikkliċċja fuq il-button _Click to copy the Shareable Link_.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_F'laqt tal-Edit Page tal-Plan, tmur tqajjem il-button biex tgħmli linkijiet shareabili_

Attentjoni: dan l-meccanizmu jipproċessa b'freqwenza ta fattur default. Dan jkun b'mod li, se l-fattur default tiegħek huwa mensili, għall-aħħar tkun għandek link kif dan: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Se tista' li il-link shareabbli jkun b'default quarterly (tqal), għandek tikkedi 3 fl-aħħar

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Biex tsegwji l-plan _premium_ b'skedjola annwaliya, għandek tikkedi 1 u l-parola:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Kejjer xi ħaġa li ma tista' tkun għandek biss link taħt qafas freqwenti tal-fattur differenti. Dovrja tqassma link speċifiku għal kada freqwenza tal-fattur tiegħek.

Mdaqqa: Wakkar li tqassma l-link mal-clientijiet tiegħek, hu jdirli reindirizziment għall-istep il-primar tal-flow ta registrazzjoni tiegħek.
