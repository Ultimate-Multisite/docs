# Il-Sitji tal-Www siti tat-Klient (v2)

Wara li ti tiskru fuq nnetzwerk tieħek, il-klientijiet jistgħu jidħol għall-dashboard tagħhom.

Boltre ta' **Account** page, li hija dakinien li jgħidu informazzjoni importanti dwar il-pagamenti, l-membership, domeni, limitazzjonijiet tal-plan u ktar, huma għandhom aċċess lejn il-page **My Sites**, fejn jistgħu joraw u jidabbru kull siti tagħhom. Il-għadd li jgħandu huwa li jiklik fuq My Sites fuq il-dashboard tagħhom.

![Link ta My Sites fil-dashboard tal-klient](/img/account-page/my-sites-link.png)

Għandhom se joraw kull siti tagħhom, ma' **Manage** button qrib dwar kulla minnhom. Jekk jiklik fih, jinwadda reindirjement għad il-dashboard ta'dha speċifika.

Hum jistgħu joraw ukoll li qualiji minn is-siti tagħhom huwa s-**primary** għall-planijiet li jippermettu li jkollhom aktar min waħda siti.

![My Sites page tura is-siti tal-klient jew button Manage](/img/account-page/my-sites-page.png)

Hawn hija vista kumppleta tal-interfaccia ta'modifika s-siti:

![Interfaccia kumppleta għall-editt tal-siti](/img/admin/site-edit-full.png)

Jekk biddi jgħandu siti primari tagħhom, huma għandhom li jmur għall-Account page biklik fuq il-menu **Account**.

![Link menu Account fil-dashboard tal-klient](/img/account-page/account-menu.png)

Hum jistgħu ukoll joraw u jidabbru dettalji tal-klient:

![Interfaccia ta'editt tal-klient](/img/admin/customer-edit.png)

Imkien, huma għandhom jiklik fuq **Change Default Site**.

![Button Change Default Site fuq il-account page](/img/account-page/change-default-site.png)

Tiddirja finestra pop-up. Jiklik fuq l-lista drop-down u tagħmel selezjoni qualiji siti tista' tgħandu biex tgħmlih s-primary website.

![Selezzjoni dropdown ta Change Default Site](/img/account-page/change-default-site-dropdown.png)

Imkien, jiklik biss fuq **Change Default Site** u għad tmur!

![Konfirmà il cambju del sito predefinitivo](/img/account-page/change-default-site-confirm.png)
