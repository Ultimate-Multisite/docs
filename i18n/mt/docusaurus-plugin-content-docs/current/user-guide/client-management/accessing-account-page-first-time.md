# Come tf issa l-account page għall-ikuża l-ewwel ħaġa?

Ben għal Ultimate Multisite!

Iċċekkja l-page tal-account tiegħek minn link li trabbiet fil-e-mail wara li kintu l-għidna il-plugin tagħna:

<!-- Screenshot unavailable: Account page email with access link -->

Uża l-e-mail li għamti fl-qiegħed mill-iċċekkja u il-password li trabbiet fil-e-mail tiegħek.

<!-- Screenshot unavailable: Login page with email and password fields -->

Jekk dan huwa l-ikuża tiegħek l-ewwel ħaġa li tmur biex tgħallis, titien tiscerti il-password tiegħek. Iċċekkja fl-password tiegħek u għallis bixogħtu l-button _Save_.

<!-- Screenshot unavailable: Change password form with Save button -->

F'laqa' l-page tal-account tiegħek, tista' tmangi kull dettalju tal-account tiegħek bħal il-fatturaturi (billings), il-riġenerazzjonijiet (renewals), l-cancellations, u ħafna oħra ħaġa.
