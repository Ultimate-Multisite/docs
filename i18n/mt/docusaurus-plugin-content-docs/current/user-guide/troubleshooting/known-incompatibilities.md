# Inkompatibilitaj notwarati malajr u plugin oħra {#known-incompatibilities-with-other-plugins}

Wara li tinstalli u attiva Ultimate Multisite, tmur possibbli li ti tqaf f problemi min inkompatibilitaj li ma jkunxġu biex jiġu trattati mal-plugin oħra.

Il-għandna nistawla b'attivitaj għalliex nhabbnu nqassem l-lista dani ma jkunxġu biex niftaħ, iżda jekk plugin huwa mħallat hemm, tmur possibbli li għandu taqfielu biex tkun Network Ultimate Multisite funzjonali.

Hawn lista tagħhom u x'la għandek taxxir fil-każi:

**X'ija Plugin WPMU DEV** Il-plugin tal-WPMU DEV li ma jkunxġu biex jiġu attwalizzati jkun innejna jkun magħruf li jagħmlu li l-flow ta registrazzjoni tal-Ultimate Multisite tqaf fuq l-istep il-aħħar (l-utenti ma jistgħux jcrea kont). Hawn lista tagħhom li jagħmlu dan il-problemi:

  * Signup Language: _L-Ultimate Multisite għandha add-on li joffri l-funzjonalità isxi, u ttel-la Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Dan huwa bbuilt f'Ultimate Multisite, għalfejn ma hemm ħاجة plugin ekstra._

  * New Blog Templates: _Dan huwa bbuilt f'Ultimate Multisite, għalfejn ma hemm ħاجة plugin ekstra._

  * Set Password on Multisite Blog Creation: _Dan huwa bbuilt f'Ultimate Multisite, għalfejn ma hemm ħاجة plugin ekstra._

  * Signup TOS: _Dan huwa bbuilt f'Ultimate Multisite, għalfejn ma hemm ħاجة plugin ekstra._

  * Anti-Splog: Kif ma jkunx powerful meta, dan mhux bbuilt f'Ultimate Multisite.

**Nota**: Kull plugin oħra tal-WPMU Dev tista' tkun użata normalment mal-Ultimate Multisite. Eżempji jinħallu Smush, Forminator, Defender, ecc.
