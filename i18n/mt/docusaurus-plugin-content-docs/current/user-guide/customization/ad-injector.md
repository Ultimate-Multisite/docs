# Aċċettazzjoni ta Ads fuq la-network tieħedha b Ultimate Multisite: Ad Injector {#placing-ads-on-your-network-with-ultimate-multisite-ad-injector}

Ultimate Multisite: Ad Injector huwa **add-on free** li jgħنك b xi strumenti biex tikkunsidi ads fuq il-plans tieħandhom mill-edit page tagħhom.

Wara li tgħallim Ultimate Multisite: Ad Injector, għandi tmur għall- _Plugins_ page tiegħek biex tiattiva.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Iċċekkja l-Network Activate button_

Mda jkun l-add-on attivat, il-meta box ta _Advanced Options_ fuq scherm għall-edit tal-plan se tista' tkun b'opzjoni nuova chiamata _Ad Settings_.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_Dawk l-meta box jista' tiġi emendat biex għal kulle minn il-plans tiegħek._

Fuq dak tab, tista' tikkunsidi kodijiet tad-ad fuq postijiet u pagine fl-front-end, qabel u dwar il-kontenut.

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Wara li tikkunsidi l-kod tad-ad tiegħek, hu se juri fuq postijiet u pagine ta dawk il-plan speċifiku._

Tista' tikkunsidi ads flim header tal-Admin Pages għall-utenti ta dawk il-plan.
