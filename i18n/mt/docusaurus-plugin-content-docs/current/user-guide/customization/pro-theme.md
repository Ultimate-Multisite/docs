# Tweak il-Dashboard Admin Panel tiegħek ma PRO Theme

Sogħti l-aħna, meta nferma fejn nistawli ma għandna biex nipprovdu l-kliemti tagħna esperjenza kustomizzata f'dashboard ta WordPress. Jekk qed tixegħu għal aspetto aktar professionali u ħsieb, il-PRO Theme tista' tgħinek.

Il-PRO Theme huwa tema admin li jkun żviluppat biex jgħinak meta qed tistawli dashboard tal-kliemti tagħek. U hu flimkien mal-core tal-plugin premium ta Ultimate Multisite.

Wara ma tinstallih, għandi tmur għall-pagina _Plugins_ u tikkliċċja biex attivalih fuq la-Network tiegħek.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Attiva fl-network tiegħek fuq la-pagina Plugins_

_**NOTA:**_ _Jekk qed tinstallu l-PRO Theme, hu juri fuq kliemti tagħek bħala opzjoni li tista' tikkliċċja biex tikkawża l-tema għal id-user tiegħek.*

Dawk, għandek l-interface admin tiegħek redesignat kompletament! Għandek ukoll menu ġdid fuq il-kolonna sejjela (left column) inattarja PRO Theme.
