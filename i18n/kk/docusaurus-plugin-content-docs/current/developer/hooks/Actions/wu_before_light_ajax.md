# Әрекет: wu_before_light_ajax {#action-wubeforelightajax}

Кейбір жағдайларда әрекеттерді өңдеу үшін қосымша қуат жүктеуіміз қажет болады.

Бұл әрекетті қажет болғанда қосымша тәуелділіктерді жүктеу үшін пайдалануға болады.

- 2.0.0
