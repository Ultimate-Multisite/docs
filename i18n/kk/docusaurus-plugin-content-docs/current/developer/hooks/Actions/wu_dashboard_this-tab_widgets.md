# Action: wu_dashboard_\{$this->tab\}_widgets

## Параметрлер {#parameters}

| Атауы | Түрі | Сипаттамасы |
|------|------|-------------|
| $tab | `string` | Ағымдағы қойынды. |
| $screen | `\WP_Screen` | Экран нысаны. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Ultimate Multisite әкімші бетінің данасы. |
