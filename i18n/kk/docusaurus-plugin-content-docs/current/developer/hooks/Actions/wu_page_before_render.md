# Әрекет: wu_page_before_render {#action-wupagebeforerender}

Plugin әзірлеушілеріне бетті басып шығармас бұрын қосымша мазмұн қосуға мүмкіндік береді.

## Параметрлер {#parameters}

| Атауы | Түрі | Сипаттамасы |
|------|------|-------------|
| $page_id | `string` | Осы беттің id мәні. |
| $page | `object` | Бет нысаны. |
