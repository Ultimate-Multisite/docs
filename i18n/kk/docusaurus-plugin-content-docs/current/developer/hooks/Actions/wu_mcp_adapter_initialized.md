# Әрекет: wu_mcp_adapter_initialized

MCP адаптері баптандырылғаннан кейін іске қосылады.

Басқа plugin-дер мен theme-дерге өз мүмкіндіктерін тіркеуге мүмкіндік береді.

## Параметрлер {#parameters}

| Атауы | Түрі | Сипаттамасы |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | MCP адаптері данасы. |
