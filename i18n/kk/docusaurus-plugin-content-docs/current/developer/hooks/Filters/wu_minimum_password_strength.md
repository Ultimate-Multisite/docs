# Сүзгі: wu_minimum_password_strength

Қажетті ең төменгі құпиясөз беріктігін (zxcvbn ұпайы) сүзеді.

Беріктік деңгейлері: - 0, 1: Өте әлсіз - 2: Әлсіз - 3: Орташа - 4: Күшті (әдепкі)

## Параметрлер {#parameters}

| Атауы | Түрі | Сипаттамасы |
|------|------|-------------|
| $min_strength | `int` | Қажетті ең төменгі беріктік деңгейі. |
| $strength_setting | `string` | Әкімші баптауының мәні (medium, strong, super_strong). |

- 2.4.0
