# Filter: wu_get_checkout_variables

Plugin әзірлеушілеріне checkout бетінің алдын ала орнатуларын сүзуге мүмкіндік береді.

Абай болыңыз, жетіспейтін кілттер front-end жағындағы checkout-ты толығымен бұзуы мүмкін.

## Параметрлер {#parameters}

| Атауы | Түрі | Сипаттамасы |
|------|------|-------------|
| $variables | `array` | Локализацияланған айнымалылар. |
| $checkout | `\Checkout` | Checkout класы. |

### Бастап {#since}

- 2.0.0
### Дереккөз {#source}

[`inc/checkout/class-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout.php#L1970) ішінде 1970-жолда анықталған


## Қайтарады {#returns}
Жаңа айнымалылар массиві.
