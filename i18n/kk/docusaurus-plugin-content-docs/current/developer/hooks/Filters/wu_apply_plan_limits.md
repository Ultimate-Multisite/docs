# Filter: wu_apply_plan_limits {#filter-wuapplyplanlimits}

Плагин әзірлеушілеріне шектеулерді айналып өтуге мүмкіндік береді.

Бұл filter-ді кез келген шектеулер іске қосылмай тұрып еркін кодты орындау үшін пайдалана аласыз. Егер filter кез келген truthy мән қайтарса, процесс жалғасады; егер кез келген falsy мән қайтарса, код қайтады және төмендегі hooks-тардың ешқайсысы орындалмайды.
