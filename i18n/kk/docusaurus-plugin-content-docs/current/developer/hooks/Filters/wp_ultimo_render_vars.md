# Сүзгі: wp_ultimo_render_vars {#filter-wpultimorendervars}

Plugin әзірлеушілеріне render контекстіне жаһандық түрде қосымша айнымалы қосуға мүмкіндік береді.

## Параметрлер {#parameters}

| Атауы | Түрі | Сипаттамасы |
|------|------|-------------|
| $args | `array` | Render шақыруы арқылы берілген айнымалыларды қамтитын массив. |
| $view | `string` | Render етілетін көріністің атауы. |
| $default_view | `string` | fallback_view атауы |

### Нұсқадан бастап {#since}

- 2.0.0
