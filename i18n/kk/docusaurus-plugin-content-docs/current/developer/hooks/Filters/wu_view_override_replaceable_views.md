# Filter: wu_view_override_replaceable_views {#filter-wuviewoverridereplaceableviews}

Әзірлеушілерге ауыстырылатын тізімге қосымша қалталар қосуға мүмкіндік береді.

Сақ болыңыз, себебі қосымша қалталарға рұқсат беру Ultimate Multisite нұсқаларының орнына ескірген көшірмелердің жүктелуіне әкелуі мүмкін.

## Параметрлер {#parameters}

| Атауы | Түрі | Сипаттамасы |
|------|------|-------------|
| $replaceable_views | `array` | Рұқсат етілген қалталар тізімі. |

### Бастап {#since}

- 2.0.0
