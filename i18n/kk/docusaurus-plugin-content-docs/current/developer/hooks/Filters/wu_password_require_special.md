# Сүзгі: wu_password_require_special

Құпиясөздерде арнайы таңбаларды талап ету-етпеуді сүзеді.

## Параметрлер {#parameters}

| Атауы | Түрі | Сипаттамасы |
|------|------|-------------|
| $require | `bool` | Арнайы таңбаларды талап ету-етпеу. Ережелер мәжбүрлі түрде қолданылғанда әдепкі мәні true. |
| $defender_active | `bool` | Defender Pro Strong Password белсенді ме. |
