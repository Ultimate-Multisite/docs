# Сүзгі: wu_is_mcp_enabled

## Параметрлер

| Атауы | Түрі | Сипаттамасы |
|------|------|-------------|
| $enabled | `bool` | MCP адаптері қосылған-қосылмағаны. |

- 2.5.0
