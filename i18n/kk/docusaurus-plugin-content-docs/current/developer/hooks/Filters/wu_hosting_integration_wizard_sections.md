# Filter: wu_hosting_integration_wizard_sections {#filter-wuhostingintegrationwizardsections}

Хостинг интеграциясын баптауға арналған шебер бөлімдерін сүзгіден өткізеді.

Қосымшаларға шебер бөлімдерін қосуға, жоюға немесе өзгертуге мүмкіндік береді.

## Параметрлер {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $sections | `array` | Шебер бөлімдері. |
| $integration | `\WP_Ultimo\Integrations\Host_Providers\Base_Host_Provider\|\WP_Ultimo\Integrations\Integration` | Бапталып жатқан интеграция. |
| $page | `\Hosting_Integration_Wizard_Admin_Page` | Шебер бетінің данасы. |
