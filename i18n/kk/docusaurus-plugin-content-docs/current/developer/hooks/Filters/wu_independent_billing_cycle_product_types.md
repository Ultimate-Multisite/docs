# Filter: wu_independent_billing_cycle_product_types

Тәуелсіз төлем циклдері бар өнім түрлерін сүзеді.

Тәуелсіз төлем циклдері бар өнімдер себетте жоспардың төлем кезеңіне сәйкес болуға мәжбүрленбейді. Бұл жоспар ай сайынғы немесе жылдық болғанына қарамастан, әрқашан жыл сайын төленетін домен тіркеулері сияқты өнімдер үшін пайдалы.

## Параметрлер {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | Тәуелсіз төлем циклдері бар өнім түрлерінің массиві. |

- 2.5.0
