# Сүзгі: wu_stripe_existing_plan_id

Тексерілетін plan ID-сін сүзеді. Егер бұл бар болса, жаңа жазылым осы plan-ды пайдаланады.

## Параметрлер {#parameters}

| Атауы | Түрі | Сипаттама |
|------|------|-------------|
| $plan_id | `string` | Тексерілетін Stripe plan ID-сі. |
| $membership_level | `object` | Мүшелік деңгейі нысаны. |
