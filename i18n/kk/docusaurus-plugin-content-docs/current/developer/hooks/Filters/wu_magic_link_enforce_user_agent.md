# Сүзгі: wu_magic_link_enforce_user_agent

Пайдаланушы агентін тексеруді мәжбүрлеп қолдану-қолданбауды сүзеді.

Токендердің әртүрлі браузерлерде/құрылғыларда жұмыс істеуіне рұқсат беру үшін false мәніне орнатыңыз. Бұл қауіпсіздікті төмендетеді, бірақ пайдалануға ыңғайлылықты арттырады.

## Параметрлер {#parameters}

| Атауы | Түрі | Сипаттамасы |
|------|------|-------------|
| $enforce | `bool` | Пайдаланушы агентінің сәйкестігін мәжбүрлеп қолдану-қолданбау. |

### Нұсқадан бастап {#since}

- 2.0.0
