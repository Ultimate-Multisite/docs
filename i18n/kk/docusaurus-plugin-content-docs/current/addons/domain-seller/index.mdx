# Domain Seller {#domain-seller}

**Multisite Ultimate: Domain Seller** сіздің WordPress Multisite желіңізді қуатты домен сату платформасына айналдырады. Клиенттерге тіркелу кезінде немесе өздерінің Dashboard-ынан нақты уақыттағы қолжетімділікті тексеру және ашық бағамен жеке домендерді тікелей тіркеуге мүмкіндік беріңіз.

### Неліктен Domain Seller? {#why-domain-seller}

Домен тіркеу — кез келген WaaS (Website as a Service) платформасы үшін ең көп сұралатын қосымша сатылымдардың бірі. Клиенттеріңізді үшінші тарап тіркеушісіне жіберудің орнына, домендерді тікелей ұсына аласыз — клиенттерге кедергісіз тәжірибе бере отырып, желіңізде көбірек табыс сақтайсыз.

### Негізгі мүмкіндіктер {#key-features}

#### Бірнеше домен тіркеушісін қолдау {#multiple-domain-registrar-support}
* **OpenSRS** — Толық тіркеу, жаңарту және басқару
* **Namecheap** — WhoisGuard және sandbox тестілеуімен толық интеграция
* **HostAfrica** — Domains Reseller API арқылы домен іздеу, баға белгілеу, тіркеу, тасымалдау, DNS және ID қорғауы
* **Openprovider** — Реселлерлік баға, тіркеу, жаңарту, тасымалдаулар, DNS, WHOIS құпиялылығы және TLD синхрондау
* **Hostinger** — Ортақ Hostinger интеграция token-ін пайдаланып домен іздеу және тіркеу
* **Кеңейтілетін архитектура** — Integration Registry үлгісінде құрылған, көбірек провайдер қосуды жеңілдетеді
* **Кіріктірілген қосылымды тестілеу** — Іске қоспас бұрын провайдер тіркелгі деректерін және қосылымды тексеріңіз

#### Бағытталған орнату шебері {#guided-setup-wizard}
* **Бір басумен бастапқы баптау** — Провайдер конфигурациясы, TLD импорты және домен өнімін жасау арқылы қадамдап өткізеді
* **Автоматты TLD синхрондау** — Қосылған провайдерлерден барлық қолжетімді TLD-лерді және көтерме бағаларды импорттайды
* **Әдепкі өнім жасау** — Ақылға қонымды үстеме баға әдепкі мәндері бар, бірден қолдануға дайын домен өнімін жасайды

#### Икемді домен бағалары {#flexible-domain-pricing}
* **Үш үстеме баға режимі** — Пайыздық үстеме, бекітілген үстеме немесе әр TLD үшін бекітілген баға
* **TLD-ге арналған өнімдер** — Әртүрлі TLD топтары үшін бөлек домен өнімдерін жасаңыз (.com/.net, .io, ел кодтары және т.б.)
* **Кіріспе баға** — Жаңа тіркеулерді тарту үшін бірінші жылға жеңілдетілген баға ұсыныңыз
* **Нақты уақыттағы баға** — Зияткерлік кэштеумен провайдерлерден алынатын нақты уақыттағы көтерме бағалар
* **Купондарды қолдау** — Multisite Ultimate жеңілдік кодтарымен толық үйлесімділік

#### WHOIS құпиялылық қорғанысы {#whois-privacy-protection}
* **Әр өнім бойынша конфигурация** — WHOIS құпиялылығын Өшірулі, Әрқашан қосылған немесе Клиент таңдауы етіп орнатыңыз
* **Клиенттің келісімі** — Клиент таңдауы орнатылған кезде, checkout-тағы құсбелгі клиенттерге құпиялылық қорғанысын қосуға мүмкіндік береді
* **Бапталатын баға** — Құпиялылық add-on-ы үшін жылдық баға орнатыңыз немесе оны тегін ұсыныңыз
* **Тіркеуші интеграциясы** — Тіркеу кезінде WhoisGuard (Namecheap) немесе WHOIS құпиялылығын (OpenSRS) автоматты түрде қосады

#### Бірыңғай checkout тәжірибесі {#unified-checkout-experience}
* **Үш домен режимі** — Клиенттер субдомен (тегін), жаңа домен тіркеу немесе өздерінде бар қолданыстағы доменді байланыстыру арасында таңдайды
* **Нақты уақыттағы домен іздеу** — Бет қайта жүктелмей, AJAX арқылы қолжетімділікті тексеру
* **Ақылды ұсыныстар** — Қалаған домен бос болмаса, балама TLD ұсыныстары
* **Ашық баға** — Тіркеу бағасын, жаңарту бағасын және қосымша WHOIS құпиялылық төлемін анық көрсету
* **Тіркеуші байланыс өрістері** — Checkout кезінде қажетті тіркеуші ақпаратын (аты, мекенжайы, телефоны) бірден жинайды
* **Автоматты жасалатын site URL** — Сайт мекенжайы таңдалған доменнен автоматты түрде шығарылады

#### Автоматты жаңарту жүйесі {#auto-renewal-system}
* **Membership-ке байланысты жаңартулар** — Клиенттің membership-і белсенді болғанда домендер автоматты түрде жаңартылады
* **Ақылды жоспарлау** — Жаңарту тапсырмалары доменнің аяқталу күндеріне қарай кезекке қойылады
* **Қайталау логикасы** — Сәтсіз жаңарту әрекеттері үшін зияткерлік қайталаулар
* **Email хабарландырулары** — Алдағы жаңартулар, сәтті жаңартулар, сәтсіздіктер және мерзімінің аяқталуы туралы бапталатын ескертулер

#### Клиенттің DNS басқаруы {#customer-dns-management}
* **Өзін-өзі қызмет ету DNS** — Клиенттер өз Dashboard-ынан DNS жазбаларын (A, AAAA, CNAME, MX, TXT) қоса, өңдей және жоя алады
* **Домен күйі** — Нақты уақыттағы тіркеу күйі, аяқталу күндері және nameserver ақпараты
* **Автоматты домен байланыстыру** — Тіркелген немесе қолданыстағы домендер клиенттің сайтына автоматты түрде байланыстырылады

#### Admin құралдары {#admin-tools}
* **Қолмен домен тіркеу** — Admin панелінен клиенттердің атынан домендерді тіркеңіз
* **Домен мониторингі** — Барлық клиент домендерін, күйлерін және аяқталу күндерін бір беттен қараңыз
* **Провайдерге арналған журналдау** — Әр тіркеуші үшін бөлек журнал арналары (мысалы, domain-seller-namecheap.log)
* **Тіркеу мәліметтері** — Негізгі доменді өңдеу бетінде тіркеуші ақпаратын және тіркеу деректерін қараңыз

### Қолдау көрсетілетін провайдерлер {#supported-providers}

* **OpenSRS** — Тіркеу, жаңарту, DNS басқару және WHOIS құпиялылығымен толық интеграция
* **Namecheap** — Тіркеу, жаңарту, топтық қолжетімділікті тексеру және тегін WhoisGuard-пен толық интеграция
* **HostAfrica** — Тіркеу, жаңарту, тасымалдаулар, DNS басқару, EPP кодын алу, тіркеуші құлпы және ID қорғауы
* **Openprovider** — Тіркеу, жаңарту, тасымалдаулар, DNS басқару, EPP кодын алу, тіркеуші құлпы, WHOIS құпиялылығы және реселлерлік TLD синхрондау
* **Hostinger** — Тіркеу, домен іздеу, nameserver жаңартулары, тіркеуші құлпы және WHOIS құпиялылығы; жаңартуларды Hostinger автоматты жаңартуы өңдейді
* **GoDaddy** — Тіркеу және жаңарту
* **ResellerClub** — Тіркеу, жаңарту және DNS басқару
* **NameSilo** — Тіркеу және жаңарту
* **Enom** — Тіркеу, жаңарту және DNS басқару

### Техникалық мәліметтер {#technical-details}

* **Талап етіледі:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **Архитектура:** Domain_Selling_Capability interface-і бар Integration Registry үлгісі
* **Автожүктеу:** PSR-4 class mapping бар Jetpack autoloader
* **Стандарттар:** WordPress кодтау стандарттары, қатаң типтеу, prepared statements

## Орнату {#installation}

1. **Multisite Ultimate v2.4.12+** орнатылғанына және желі деңгейінде белсендірілгеніне көз жеткізіңіз
2. Плагинді `/wp-content/plugins/ultimate-multisite-domain-seller/` ішіне жүктеңіз
3. Плагинді WordPress Network Admin ішіндегі **Плагиндер** бөлімінен желі деңгейінде белсендіріңіз
4. **Орнату шебері** алғашқы белсендіру кезінде автоматты түрде іске қосылады — келесі қадамдарды орындаңыз:
   * Домен тіркеушіңізді қосыңыз
   * Қосылымды тексеріңіз
   * Қолжетімді TLD-лерді және көтерме бағаларды импорттаңыз
   * Баға үстемесі бар алғашқы домен өніміңізді жасаңыз
5. **Ultimate Multisite > Төлем рәсімдеу пішіндері** ішінде төлем рәсімдеу пішініңізге **Домен таңдау** өрісін қосыңыз

## Жиі қойылатын сұрақтар {#frequently-asked-questions}

### Multisite Ultimate нұсқасының қайсысы қажет? {#what-version-of-multisite-ultimate-is-required}

2.4.12 немесе одан жоғары нұсқа. Қосымша WP Ultimo v1.x немесе 2.4.12-ден төмен Ultimate Multisite нұсқаларымен үйлесімді емес.

### Қай домен провайдерлеріне қолдау көрсетіледі? {#which-domain-providers-are-supported}

Тоғыз тіркеушіге қолдау көрсетіледі: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo** және **Enom**. Мүмкіндіктер провайдерге қарай өзгереді, өйткені тіркеуші API-лері әртүрлі операцияларды ұсынады; әр провайдер бойынша шолу үшін орнату нұсқаулығын қараңыз. Қосымшаның Integration Registry архитектурасы жеке провайдерлерді қосуды жеңілдетеді — [әзірлеуші нұсқаулығын](/developer/integration-guide/custom-registrar) қараңыз.

### WHOIS құпиялылық қорғанысын ұсына аламын ба? {#can-i-offer-whois-privacy-protection}

Иә. Әр домен өнімінде үш режимі бар WHOIS құпиялылығы параметрі бар: өшірулі, әрқашан қамтылған немесе клиент таңдауы. Клиент таңдауы орнатылғанда, клиенттер төлем рәсімдеу кезінде бағасы көрсетілген құсбелгіні көреді (немесе Namecheap WhoisGuard үшін "Тегін"). Құпиялылық жалаушасы тіркеу кезінде тіркеушіге автоматты түрде жіберіледі.

### Әртүрлі TLD-лер үшін әртүрлі баға қоя аламын ба? {#can-i-set-different-pricing-for-different-tlds}

Иә. Әртүрлі TLD топтары үшін бөлек домен өнімдерін жасаңыз. Әр өнімнің өз үстеме түрі (пайыздық, бекітілген үстеме немесе бекітілген баға), қолдау көрсетілетін TLD-лері және кіріспе бағалары болуы мүмкін.

### Клиенттер қандай төлем рәсімдеу опцияларын көреді? {#what-checkout-options-do-customers-see}

Домен таңдау өрісі клиенттерге үш қойынды береді: **Субдомен** (желіңіздегі тегін субдоменді пайдалану), **Жаңа доменді тіркеу** (жаңа доменді іздеу және тіркеу) және **Бар домен** (өздерінде бар доменді байланыстыру). Тек сіз қосқан режимдер ғана көрсетіледі.

### Автоматты жаңартуларды қолдай ма? {#does-it-support-automatic-renewals}

Иә. Домендер автоматты жаңартуға орнатыла алады және клиенттің мүшелік күйіне байланысты болады. Жаңарту төлемдері клиенттің қолданыстағы төлем шлюзі арқылы өңделеді.

### Клиенттер өз DNS жазбаларын басқара ала ма? {#can-customers-manage-their-dns-records}

Иә. Клиенттер A, AAAA, CNAME, MX және TXT жазбаларын қосу, өңдеу және жою үшін өз Dashboard-ында DNS басқару интерфейсін алады. DNS басқару OpenSRS, ResellerClub, Enom, HostAfrica және Openprovider арқылы қолжетімді. Орналастырылған домендерге арналған Hostinger DNS негізгі Hostinger доменді байланыстыру интеграциясы арқылы өңделеді.

### Орнату шебері бар ма? {#is-there-a-setup-wizard}

Иә. Орнату шебері алғашқы белсендіру кезінде автоматты түрде іске қосылып, тіркеушіні қосу, TLD-лерді импорттау және алғашқы домен өніміңізді жасау қадамдарынан өткізеді. Сондай-ақ шеберді қосымша баптауларынан қайта іске қоса аласыз.
