# Домен сатушысының өзгерістер журналы {#domain-seller-changelog}

1.3.0 нұсқасы - 2026-06-02 күні шығарылды
- Жаңа: HostAfrica реселлерінің балансы тым төмендегенде network-admin ескертуі қосылды
- Жаңа: Жаңадан тіркелген домендерді желі сайтына автоматты түрде сәйкестендіру қосылды
- Түзету: Тіркеуші өрісінің талаптары тек жаңа доменді тіркеген кезде ғана қолданылатын болды
- Түзету: Баланс туралы бақылау хабарламаларын жабуға болатын етті
- Түзету: WooCommerce тіркеушісінің төлем деректері сақталатыны қамтамасыз етілді
- Түзету: Тіркеу кезінде тіркеуші байланыс деректерінің талаптары міндеттелді
- Түзету: Домен тіркеу өнімдерінің 0% үстемемен жасалуына жол берілмеді
- Түзету: Checkout сессиясы ағымы арқылы домен таңдаулары мен бағалар сақталды
- Түзету: HostAfrica домен бағаларының валюта көрсетілімі жақсартылды
- Түзету: WP-core query-var сәйкессіздіктерін болдырмау үшін checkout form-action әрекеті жақсартылды
- Жақсартылды: Баптау нұсқаулығында HostAfrica реселлер конфигурациясы құжаттамасына сілтеме қосылды

1.2.0 нұсқасы - 2026-05-25 күні шығарылды
- Жаңа: HostAfrica домен сату интеграциясы ретінде checkout, баптау шебері, іздеу, TLD/баға, тіркеу, жаңарту, тасымалдау, nameserver, DNS, EPP code, registrar lock және ID protection қолдауымен қосылды
- Жаңа: Openprovider домен сату интеграциясы ретінде реселлер бағалары, тіркеу, жаңарту, тасымалдау, nameserver, DNS, EPP code, registrar lock, WHOIS privacy және TLD синхрондау қолдауымен қосылды
- Жаңа: Hostinger қолжетімділікті тексеру, тіркеу, nameserver жаңартулары, registrar lock және WHOIS privacy үшін негізгі интеграциядағы ортақ Hostinger API token қолданатын домен сату интеграциясы ретінде қосылды
- Жақсартылды: Әзірлеуші интеграциясына арналған нұсқаулық үшін домен өмірлік циклі әрекеті мен сүзгі docblock-тары қосылды
- Жақсартылды: Addon readme файлындағы plugin үйлесімділік метадеректері WordPress 7.0 нұсқасына жаңартылды
- Жақсартылды: Алдағы шығарылымды үйлестіру үшін қолданылатын жоспарлау үлгілері жаңартылды

* **Жақсартылды:** Баптаулар бетінің барлық жерінде сипаттамалар жақсартылды (TLD іздеу, жаңартулар, DNS, хабарландырулар)

1.0.1 нұсқасы - 2026-02-27

* **Жаңа:** Жаппай бағаны басқаруға арналған TLD импорттау құралы
* **Жаңа:** Домен өнімдері үшін кіріспе баға қолдауы
* **Жаңа:** Cypress бар E2E тест жинағы
* **Жаңа:** Доменнің өмірлік циклі туралы хабарландыруларға арналған email үлгілері
* **Жаңа:** Әкімші доменін тіркеу модалындағы тіркелуші мекенжайы өрістері, баптаулардан алдын ала толтырылады
* **Жаңа:** Жазбаларды қосу, өңдеу және жою қолдауы бар клиенттің DNS басқару интерфейсі
* **Жаңа:** Автоматты домен сәйкестендіруі бар checkout үшін «өзіңіздің доменіңізді әкеліңіз» опциясы
* **Жаңа:** Checkout кезінде домен атауынан сайт URL-ін автоматты түрде жасау
* **Жаңа:** Баптаулардағы әдепкі nameserver және DNS жазбасы конфигурациясы
* **Жаңа:** Негізгі доменді өңдеу бетіндегі домен тіркеу мәліметтері және DNS басқаруы
* **Жаңа:** Орнату шебері орынды әдепкі мәндері бар әдепкі домен өнімін автоматты түрде жасайды
* **Жаңа:** Барлық конфигурацияланған провайдерлер бойынша cron арқылы күнделікті автоматты TLD синхрондау
* **Жаңа:** Әр өнім бойынша конфигурациясы бар WHOIS құпиялылық қорғанысы (әрқашан қосулы, клиент таңдауы немесе өшірулі)
* **Жаңа:** Баға көрсетілімі және қараңғы режим қолдауы бар WHOIS құпиялылығы checkout checkbox-ы
* **Жаңа:** Доменді қолмен тіркеуге арналған Register Domain әкімші беті
* **Жаңа:** Ultimate Multisite жаңарту сервері арқылы автоматты plugin жаңартулары
* **Жаңа:** Өнім тізімі кестесіндегі күлгін badge стилі бар домен өнім түрі сүзгі қойындысы
* **Жаңа:** Домен checkout формасындағы тіркелуші байланыс өрістері (аты, мекенжайы, қаласы, штаты/аймағы, пошта коды, елі, телефоны)
* **Жаңа:** Тіркеуші API-ін шақырмас бұрын анық қате хабарламалары бар тіркелуші өрістерін тексеру
* **Жаңа:** Домен тіркеу оқиғалары үшін провайдерге тән журнал арналары (мысалы, domain-seller-namecheap.log)
* **Жаңа:** Негізгі тіркелу/signup checkout формасындағы тіркелуші байланыс өрістері (домен тіркелгенде көрсетіледі)
* **Жақсартылды:** Domain Search checkout өрісі қосалқы домен, тіркеу және бар домен қойындыларын қолдайтын бірыңғай Domain Selection өрісімен ауыстырылды
* **Жақсартылды:** Домен өнімінің баптаулары негізгі widget жүйесі арқылы өнімді өңдеу бетінде inline көрсетіледі
* **Жақсартылды:** Клиент домені туралы ақпарат дербес metabox орнына негізгі домен сәйкестендіру widget-ына қосылады
* **Жақсартылды:** TLD импорттау шебері барлық провайдерлерден бір рет басу арқылы синхрондауға дейін жеңілдетілді
* **Жақсартылды:** Namecheap домен қолжетімділігі жылдамырақ іздеу үшін batch API шақыруын пайдаланады
* **Жақсартылды:** Namecheap баға API-і дұрыс параметрлерді және жауапты талдауды пайдаланады
* **Жақсартылды:** TLD сақтау бір желілік опцияға орталықтандырылды
* **Жақсартылды:** DNS өзгерістері, трансферлер және конфигурацияны қолдану үшін домен белсенділігін журналдау
* **Жақсартылды:** Batch тексеруі бар IANA негізгі тізімін пайдаланып OpenSRS үшін толық TLD синхрондау
* **Жақсартылды:** Беттелген API сұраулары бар Namecheap үшін толық TLD синхрондау
* **Жақсартылды:** Ескі провайдер кластарын Integration Registry үлгісімен ауыстыру
* **Жақсартылды:** DNS және трансфер конфигурациясы бар баптаулар панелі
* **Жақсартылды:** Телефон нөмірлері автоматты түрде +CC.NNN тіркеуші форматына пішімделеді
* **Жақсартылды:** Телефон өрісін тексеру жіберу алдында пішімдеу таңбаларын алып тастайды
* **Жақсартылды:** Нұсқа талабы анығырақ ескертумен Ultimate Multisite 2.4.12-ге көтерілді
* **Жақсартылды:** CI жұмыс процесі addon және негізгі plugin үшін дұрыс checkout пайдаланады
* **Жақсартылды:** prepare_registrant_info() billing мекенжай fallback-ымен checkout-та сақталған пайдаланушы метадеректерін оқиды
* **Түзетілді:** Checkout кезінде жүйеге кірмеген пайдаланушылар үшін домен іздеу AJAX сәтсіз аяқталуы
* **Түзетілді:** Checkout кезінде жүйеге кірмеген пайдаланушылар үшін домен бағасы AJAX сәтсіз аяқталуы
* **Түзетілді:** WP-CLI командаларын орындағанда Spyc класының қайта жариялануынан болатын fatal error
* **Түзетілді:** Namecheap sandbox API timeout-ы тым қысқа
* **Түзетілді:** Домен іздеудегі Select батырмасы мәтіні жасыл фонда көрінбейді
* **Түзетілді:** Байланыс ақпаратының жоқтығына байланысты "RegistrantFirstName is Missing" қатесімен домен тіркеудің сәтсіз аяқталуы
* **Түзетілді:** Checkout уақытында сайт әлі болмаған кезде blog_id=0 мәнімен домен жазбасының жасалуы
* **Түзетілді:** Әдепкі TLD баптауы талданған массив орнына string ретінде қайтарылды
* **Жойылды:** Дербес Domain Management әкімші беті — енді негізгі домен беттері арқылы өңделеді
