# Filter: wu_domain_seller_search_delay {#filter-wudomainsellersearchdelay}

Доменді іздеу debounce кідірісін миллисекундпен сүзеді.

Баяу қосылымдарда API шақыруларын азайту үшін бұл мәнді арттырыңыз.

## Параметрлер {#parameters}

| Атауы | Түрі | Сипаттамасы |
|------|------|-------------|
| $delay | `int` | Debounce кідірісі миллисекундпен. Әдепкі мәні 500. |
