# VAT өзгерістер журналы

Нұсқа 1.0.7 - 2026-02-03 шығарылды

* Түзету: Германия (DE) VAT нөмірлері VIES SOAP қызметіндегі MS_MAX_CONCURRENT_REQ қателеріне байланысты тексеруден өтпей қалатын. Сенімдірек тексеру үшін бұрынғы SOAP API орнына ресми EU VIES REST API қолдануға ауыстырылды.
* Түзету: VIES қызметі уақытша қолжетімсіз болғанда, VAT нөмірлері енді қате түрде қабылданбай тасталудың орнына қабылданады. Кейін қарап шығу үшін журнал жазбасы жасалады.
* Түзету: VAT/Tax ID өрісі енді тек EU мүшелері үшін ғана емес, барлық елдер үшін көрінеді. EU-ға кірмейтін салық идентификаторлары (мысалы, Swiss CHE нөмірлері) VIES тексеруінсіз шот-фактурада көрсету үшін сақталады.
* Түзету: Грекия (GR), Монако (MC) және Мэн аралы (IM) үшін VAT префиксін іздеу қате массив іздеу логикасына байланысты бұзылған еді.
* Жақсарту: Site Exporter қосымшасымен бірге қолданылғанда өрескел қателер тудыруы мүмкін қажетсіз wp-cli-bundle әзірлеу тәуелділігі алынып тасталды.
* Жақсарту: Толық unit test жинағы қосылды (53 тест).

Нұсқа 1.0.6 - 2026-01-25 шығарылды

* Түзету: Ішкі B2B транзакцияларында енді VAT дұрыс алынады. EU VAT ережелеріне сәйкес, reverse charge тек шекарааралық B2B транзакцияларына қолданылады, клиенттің елі компанияның елімен сәйкес келгенде қолданылмайды.

Нұсқа 1.0.5 - 2026-01-22 шығарылды

* Түзету: Addon жүктелмей немесе дұрыс жұмыс істемей тұрған.
* Өзгеріс: Сенімдірек әрі белсенді жүргізілетін деректер үшін VAT мөлшерлемелері дереккөзі euvatrates.com сайтынан ibericode/vat-rates репозиторийіне ауыстырылды.
* Түзету: super_reduced_rates опция мәніндегі әріп қатесі түзетілді.
* Жақсарту: Қателердің алдын алу үшін ел кодын өңдеуге null тексерулері қосылды.
* Өзгеріс: Traduttore арқылы автоматты жаңартулар пайдасына бірге жеткізілетін аударма файлдары алынып тасталды.

Нұсқа: 1.0.3 - 2025-09-28 шығарылды
