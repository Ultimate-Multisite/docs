# VAT {#vat}

Multisite Ultimate: Еуропалық VAT

Еуропалық VAT-ты жақсырақ қолдау үшін Multisite Ultimate әдепкі салық өңдеуіне қосымша мүмкіндіктер мен құралдар қосады.
EU VAT мөлшерлемелерін Салық мөлшерлемелері бетіне автоматты түрде қосады

## Орнату {#installation}

1. 'ultimate-multisite-vat' файлын '/wp-content/plugins/' каталогіне жүктеңіз
2. Плагинді WordPress ішіндегі 'Плагиндер' мәзірі арқылы белсендіріңіз
3. Ultimate Multisite > Параметрлер > Салықтар бөліміне өтіп, EU Vat мүмкіндігін қосыңыз.
4. Салық мөлшерлемелерін басып, төменгі жақтағы EU VAT мөлшерлемелерін жаңартыңыз да, Салық мөлшерлемелерін сақтаңыз.
5. Дайын!
