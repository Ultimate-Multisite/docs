# Mailster өзгерістер журналы

### 1.0.0 {#100}
* Бастапқы шығарылым
* Тіркелу немесе төлем кезінде жазылушыларды автоматты синхрондау
* Өнімге тән тізім тағайындаулары
* GDPR талаптарына сәйкестік үшін құсбелгіге негізделген келісім беру
* Қос растау қолдауы
* Өрістерді жан-жақты сәйкестендіру
* Егжей-тегжейлі журнал жүргізу
