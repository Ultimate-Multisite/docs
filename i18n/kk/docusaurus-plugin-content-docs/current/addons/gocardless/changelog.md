# GoCardless өзгерістер журналы

Нұсқа: 1.0.3 - 2026-05-05 күні шығарылды
* Түзетілді: GoCardless мүшелігі күтуде немесе тоқтатылған күйде жойылғанда, жетім қалған күтудегі сайттар енді тазаланады
* Жақсартылды: Bedrock түбірлік автожүктегіші тәуелділіктерді жүктеп қойған болса, плагин автожүктегіші өткізіп жіберіледі

Нұсқа: 1.0.1 - 2025-09-28 күні шығарылды

* Префиксті ultimate-multisite деп өзгерту; мәтін доменін жаңарту; нұсқаны көтеру.



2021-09-24 - Нұсқа 1.0.0-beta.4

* Қосылды: mu-plugins негізіндегі баптаулар үшін wp_ultimo_skip_network_active_check сүзгісі;

2021-06-16 - Нұсқа 1.0.0-beta.3

* Түзетілді: вебхук өңдегіштері қолтаңбаларды растай алмады;

2021-06-14 - Нұсқа 1.0.0-beta.2

* Жақсартылды: GoCardless мандаты жасалғаннан кейін сайтты жариялау;

Нұсқа 1.0.0-beta.1 - Бастапқы нұсқа
