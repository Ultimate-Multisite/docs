# Әкімші беттерін жасаушының өзгерістер журналы

### 1.9.0 нұсқасы - 2026-01-18 күні шығарылды

* Түзетілді: белсендіру кезінде болатын Fatal errors.
* Қайта құрылымдалды: Content source кластары дұрыс parent constructor инициализациясымен Singleton trait қолданатындай өзгертілді
* Түзетілді: контент түрі селекторының көрсетілуіне кедергі келтірген Vue.js template компиляция қателері
* Түзетілді: жетіспейтін Vue дерек қасиеттері (template_id, external_link_url_embedable, external_link_url_checking)
* Түзетілді: бет жүктелгенде TinyMCE редакторы дұрыс инициализацияланбауы
* Түзетілді: Divi қолдау файлы Composer арқылы автоматты жүктелмеуі
* Түзетілді: Divi қолдау класындағы бүлінген PHP синтаксисі
* Жақсартылды: JavaScript asset ұйымдастыруы және минификациясы
* Жақсартылды: соңғы WordPress және бет құрастырғыш нұсқаларымен үйлесімділік
