# Admin Page Creator {#admin-page-creator}

## Мазмұн {#content}
Admin page creator WordPress admin dashboard ішінде жаңа әкімші беттері мен ішкі беттерді оңай қосуға мүмкіндік береді. Оны жаңа пайдаланушылар үшін бастапқы бет бола алатын сәлемдесу бетін немесе оқулық бетін жасау үшін пайдалануға болады. Беттерді еш қиындықсыз жобалау үшін Beaver Builder, Elementor, Brizy және Oxygen сияқты бет құрастырушыларды пайдалана аласыз.

Сондай-ақ ол жеке HTML, CSS және JavaScript қолдауын, әрі plan және пайдаланушы рөліне негізделген қатынас бақылауын ұсынады!

## Мүмкіндіктер {#features}
* Жеке HTML және CSS қолдауы
* JavaScript қолдауы
* PHP мазмұн режимі
* Пайдаланушы рөліне негізделген рұқсат
* Бет құрастырушыларға қолдау

## Орнату {#installation}

1. **ultimate-multisite-admin-page-creator** файлын **/wp-content/plugins/** каталогына жүктеңіз
2. WordPress Network Admin ішіндегі **Плагиндер** мәзірі арқылы plugin-ді белсендіріңіз

## Пайдалану нұсқаулығы {#usage-guide}

WordPress сайттардың алдыңғы бөлігіне мазмұн қосуды өте оңай етсе де, әкімші жағында беттер мен ішкі беттер жасау кейбір негізгі кодтау дағдыларын талап етеді. Ultimate Multisite: Admin Page Creator алдыңғы бөліктегі сол пайдалану жеңілдігін желілік сайттарыңыздың артқы бөлігіне әкелуді мақсат етеді.

## Неліктен Admin Page Creator пайдалану керек? {#why-use-admin-page-creator}

## Мәзір опциялары {#menu-options}

Бүйір жақтағы General Options мета-блогында мәзір түрін, белгісін және тағы көп нәрсені конфигурациялай аласыз.

### Мәзір түрлері {#menu-types}

Жаңа жеке әкімші бетіңізде екі түрлі мәзір түрі болуы мүмкін: жоғарғы деңгей мәзірі және ішкі мәзір.

Жоғарғы деңгей мәзір беті, аты айтып тұрғандай, жаңадан жасалған бетіңізді әдепкі WordPress беттерімен қатар әкімші мәзіріне қосады.

Егер жоғарғы деңгей опциясын таңдасаңыз, мәзір ретін және мәзір белгішесін де таңдай аласыз.

Бұл бетті ішкі мәзір беті еткіңіз келсе, осы бетті тіркеу үшін ата-аналық бетті таңдай аласыз. Жеке бетіңізді әдепкі WordPress беттеріне немесе өзіңіздің жоғарғы деңгейдегі жеке әкімші беттеріңізге қоса аласыз.

_Осы жеке бет үшін мәзір элементін толық баптаңыз_

<!-- Скриншот қолжетімсіз: Admin Page Creator ішкі мәзірінің ата-аналық бетін таңдау -->

## Кеңейтілген опциялар {#advanced-options}

Жеке әкімші беттеріңізге әртүрлі мазмұн қоса алуыңыз керек. Бұған CSS ережелері мен файлдары, JavaScript және сыртқы кітапханалар да кіреді. Мұны Advanced Options қойындысы арқылы жасауға болады.

### Жеке CSS {#custom-css}

CSS қойындысында өзіңіздің жеке CSS кодыңызды қоса аласыз, сондай-ақ қашықтағы сайттардан CSS файлдарын импорттай аласыз (мысалы, CDN, Google Fonts және т.б.).

<!-- Скриншот қолжетімсіз: сыртқы стиль кестесін импорттауы бар Admin Page Creator жеке CSS редакторы -->

_Жеке CSS те қолдау табады. Сыртқы стиль файлдарын да қоса аласыз!_

### Жеке JavaScript {#custom-javascript}

Дәл осы JavaScript үшін де жарамды. Жеке JavaScript кодыңызда пайдалану үшін сыртқы кітапханаларды қоса аласыз.

<!-- Скриншот қолжетімсіз: сыртқы кітапхананы импорттауы бар Admin Page Creator жеке JavaScript редакторы -->

_Өзіңіздің JavaScript кодыңызды қосыңыз!_

## Рұқсаттар {#permissions}

Сондай-ақ Advanced Options ішінде осы бетке қай ішкі сайттардың қатынаса алатынын таңдай аласыз. Мысалы, әртүрлі plan деңгейлеріне әртүрлі эксклюзивті мазмұн ұсынуды жоспарласаңыз, бұл пайдалы.

<!-- Скриншот қолжетімсіз: plan және рөлге негізделген қатынас бақылауына арналған Admin Page Creator рұқсат баптаулары -->

_Бұл беттің қай жерде көрсетілетінін толық басқарасыз_

**Маңызды** : Бет көрсетілуі үшін екі шарттың да орындалуы қажет. Бұл дегеніміз, егер пайдаланушы берілген жоспардың клиенті болса, бірақ таңдалған рөлдердің біріне ие болмаса (немесе керісінше), бет олардың панелінде пайда болмайды.

## Белсенді {#active}

Егер пайдаланушыға арналған admin бетін жоймай немесе рұқсаттармен әуре болмай өшіргіңіз келсе, жай ғана Active опциясынан белгіні алып тастаңыз.

Жоғарғы деңгейлі бетті өшіру сол ата-аналық бетке бекітілген барлық ішкі беттерді автоматты түрде өшіреді.
