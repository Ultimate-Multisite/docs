# Support Agents

Бүкіл желіні өзіңіз басқару оңай міндет емес. Желіңіз өсе бастаған сайын, қолдауға деген сұраныстың артқанын байқайсыз. Жұмыс жүктемеңізді жеңілдетуге көмектесетін адамдарды тарту жақсы идея болады. Алайда, толық қолжетімділік/рұқсат беру қауіпті және желіңізге нұқсан келтіруі мүмкін. Дәл осы жерде Support Agent add-on-ы көмекке келеді.

Бұл add-on Support Agent деп аталатын WordPress пайдаланушысының жаңа түрін жасауға мүмкіндік береді және сол пайдаланушыда қандай рұқсаттар болатынын толық бақылауға береді. Рұқсат беру toggle батырмасын қосу сияқты оңай, ал оны алып тастауды шешсеңіз, өшіріп қою жеткілікті.

## Мүмкіндіктер

- WordPress және Ultimate Multisite рұқсаттарының тармақталған тізімі
- Қолжетімділікті оңай қосу және өшіру
- Agent Account-тарының шексіз санын жасау

## Орнату

1. 'ultimate-multisite-support-agents' файлын '/wp-content/plugins/' каталогына жүктеп салыңыз
2. Плагинді WordPress ішіндегі 'Plugins' мәзірі арқылы белсендіріңіз
3. Плагинді баптау үшін қадамдық Wizard нұсқауларын орындаңыз
