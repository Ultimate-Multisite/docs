# AffiliateWP өзгерістер журналы {#affiliatewp-changelog}

Нұсқа: 2.0.1 - 2025-09-28 күні шығарылды

* Префиксті ultimate-multisite деп өзгерту; мәтіндік доменді жаңарту; нұсқаны көтеру.



Нұсқа: 2.0.0 - 2022-08-05 күні шығарылды

* Plugin ultimo төлемдері негізінде жұмыс істеуі үшін қайта құрастырылды

Нұсқа: 2.0.0-beta.2 - 2022-01-21 күні шығарылды

* Ішкі: хуктар мен сүзгілер генераторы қосылды;
* Ішкі: әзірлеушіге ыңғайлы болуы үшін Multisite Ultimate стабтары қосылды;
* Түзетілді: берілген себетте жеңілдік коды болмаған кездегі өлім қатесі;
* Түзетілді: кірулердің дұрыс түрлендірілмеуі;

Нұсқа 2.0.0-beta.1 - 2021-12-29 күні шығарылды

* 2.0 негізіндегі нұсқаның бастапқы шығарылымы;
