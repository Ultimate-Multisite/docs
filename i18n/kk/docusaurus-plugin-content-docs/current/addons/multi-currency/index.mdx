# Көп валюта

Ultimate Multisite: Көп валюта желіңізге бірнеше валютада төлем қабылдауға мүмкіндік береді. Ол нақты уақыттағы немесе қолмен енгізілген айырбас бағамдарын пайдаланып бағаларды автоматты түрде түрлендіреді, кез келген валюта үшін әр өнімге нақты баға қоюға мүмкіндік береді және геолокация арқылы әр келушінің қалаулы валютасын анықтайды — бәрі бір әкімші Dashboard арқылы басқарылады.

## Негізгі мүмкіндіктер {#key-features}

- **Айырбас бағамын автоматты түрде түрлендіру** — бағалар таңдаған провайдеріңізден алынған бағамдар арқылы негізгі валютаңыздан түрлендіріледі
- **Әр өнім бойынша баға қайта анықтаулары** — автоматты түрлендіруді айналып өту үшін кез келген өнімге валютаға тән нақты бағаларды қойыңыз
- **Тіркелу кезіндегі валюта таңдағыш өрісі** — клиенттерге checkout/signup кезінде қалаулы валютасын таңдауға мүмкіндік береді
- **Бірнеше айырбас бағамы провайдері** — ECB, Frankfurter, Stripe немесе толықтай қолмен енгізілетін бағамдар
- **Геолокацияға негізделген валютаны анықтау** — келушінің еліне сәйкес келетін валютаны автоматты түрде алдын ала таңдайды
- **Әкімші Dashboard** — белсенді валюталарды, айырбас бағамдарын және провайдер баптауларын бір жерден басқарыңыз

## Талаптар {#requirements}

- Ultimate Multisite 2.0.0 немесе одан жоғары
- WordPress Multisite орнатымы
- PHP 7.4 немесе одан жоғары

## Орнату {#installation}

1. Plugin файлдарын `/wp-content/plugins/ultimate-multisite-multi-currency` ішіне жүктеңіз
2. Plugin-ді WordPress ішіндегі **Plugin-дер** экраны арқылы белсендіріңіз (Желі бойынша белсендіру)
3. Әкімші Dashboard-ты ашу үшін **Ultimate Multisite → Көп валюта** бөліміне өтіңіз
4. Қабылдағыңыз келетін валюталарды қосып, айырбас бағамы провайдеріңізді баптаңыз

## Баптау {#configuration}

### Әкімші Dashboard {#admin-dashboard}

Көп валюта әкімші Dashboard-ы (**Ultimate Multisite → Көп валюта**) — мыналарды орындауға арналған орталық орын:

- Валюталарды қосу, қосулы ету немесе өшіру
- Айырбас бағамы провайдерін таңдау және бағамдарды сұраныс бойынша жаңарту
- Барлық белсенді валюталардың ағымдағы бағамдарын қарау
- Автоматты провайдерлер жарамсыз болғанда қолмен енгізілетін бағамдарды қою

### Айырбас бағамы провайдерлері {#exchange-rate-providers}

| Провайдер | Сипаттама |
|---|---|
| **ECB** | Еуропалық орталық банк — EUR негізіндегі желілерге арналған тегін күнделікті бағамдар |
| **Frankfurter** | Ашық бастапқы кодты ECB mirror API — тегін, API key қажет емес |
| **Stripe** | Stripe-тың нақты уақыттағы FX бағамдарын пайдаланады — белсенді Stripe интеграциясын талап етеді |
| **Қолмен** | Бағамдарды тікелей өзіңіз енгізесіз; сыртқы API шақырулары жоқ |

Провайдеріңізді **Ultimate Multisite → Көп валюта → Баптаулар → Айырбас бағамы провайдері** бөлімінен таңдаңыз. Бағамдарды кез келген уақытта қолмен жаңартуға немесе автоматты түрде жаңартылатындай жоспарлауға болады.

### Әр өнім бойынша баға қайта анықтаулары {#per-product-pricing-overrides}

Автоматты түрлендіру жеткілікті дәл болмаған кезде, кез келген өнімде белгілі бір валюта үшін нақты баға қоя аласыз:

1. Өнім редакторын ашыңыз (**Ultimate Multisite → Өнімдер → Өңдеу**)
2. **Валюта бағалары** бөліміне дейін айналдырыңыз
3. Қайта анықтағыңыз келетін әр валюта үшін нақты бағаны енгізіңіз
4. Өнімді сақтаңыз

Қайта анықтауы жоқ өнімдер негізгі валютаңыздан түрлендірілген бағаны пайдаланады.

### Тіркелу кезіндегі валюта таңдағыш өрісі {#currency-selector-signup-field}

Клиенттер тіркелу кезінде қалаулы валютасын таңдауы үшін валюта таңдағыш өрісін signup/checkout ағыныңызға қосуға болады:

1. **Ultimate Multisite → Көп валюта → Баптаулар** бөліміне өтіңіз
2. **Тіркелуде валюта таңдағышын көрсету** параметрін қосыңыз
3. Қалауыңызша әдепкі валютаны қойыңыз немесе алдын ала таңдалған мәнді геолокация анықтасын

Таңдалған валюта subscription-мен бірге сақталады және барлық кейінгі шот ұсынуда қолданылады.

### Геолокацияға негізделген валютаны анықтау {#geolocation-based-currency-detection}

Геолокация арқылы анықтау қосулы болғанда, addon келушінің елін валютаға сәйкестендіріп, оны валюта таңдағыш өрісінде алдын ала таңдайды. Қосу үшін:

1. **Ultimate Multisite → Көп валюта → Баптаулар** бөліміне өтіңіз
2. **Келуші орналасқан жерінен валютаны анықтау** параметрін қосыңыз
3. Елдерді валюталарға **Геолокация сәйкестендіруі** кестесінде сәйкестендіріңіз

Егер келушінің елі үшін сәйкестендіру болмаса, желінің негізгі валютасы fallback ретінде пайдаланылады.

## Жиі қойылатын сұрақтар {#frequently-asked-questions}

### Айырбас бағамы провайдері қолжетімсіз болса не болады? {#what-happens-if-an-exchange-rate-provider-is-unavailable}

Addon соңғы сәтті алынған бағамдарға fallback жасайды. Егер ешқашан бағамдар алынбаған болса, ол сіз баптаған қолмен енгізілген бағамдарға fallback жасайды. Қауіпсіздік қоры ретінде қолмен енгізілетін бағамдарды баптау ұсынылады.

### Әртүрлі валюталар үшін әртүрлі провайдерлерді пайдалана аламын ба? {#can-i-use-different-providers-for-different-currencies}

Жоқ — барлық валюталар үшін бір уақытта бір провайдер белсенді болады. Жеке бағамдарды толық бақылау қажет болса, **Қолмен** режимін пайдаланыңыз.

### Геолокация бөлек plugin талап ете ме? {#does-geolocation-require-a-separate-plugin}

Сыртқы геолокация plugin-і қажет емес. Addon WordPress-тің кірістірілген IP-ден елді анықтау мүмкіндігін пайдаланады. Жоғарырақ дәлдік үшін арнайы геолокация қызметін берілген filter hook-тар арқылы интеграциялауға болады.

### Әр өнім бойынша қайта анықтаулар салыққа дейін ме, әлде салықтан кейін ме қолданылады? {#are-per-product-overrides-applied-before-or-after-tax}

Әр өнім бойынша қайта анықтаулар мақсатты валютадағы салыққа дейінгі бағаны қояды. Салық қолданыстағы салық баптауыңыз арқылы қайта анықталған бағаның үстінен есептеледі.
