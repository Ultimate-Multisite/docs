# Құралдар және конфигурация

Theme Builder сіздің WordPress сайтыңызды талдау, жобалау және құру үшін құралдар жиынтығын пайдаланады. 1.16.0 нұсқасында **sd-ai-agent/site-scrape енді Tier 1 құралы**, сондықтан ол Theme Builder ішінде әдепкі бойынша қолжетімді. Superdav AI Agent v1.18.0 сенімді әзірлеуші жұмыс процестері үшін бөлек таратылатын Advanced серіктес пакетін де ұсынады; бұл құралдар WordPress.org негізгі пакетінің бөлігі емес және бөлек орнатылып, бөлек авторизациялануы керек.

## Қолжетімді құралдар

### Tier 1 құралдары (әрдайым қолжетімді)

Tier 1 құралдары Theme Builder ішінде ешқандай қосымша конфигурациясыз әдепкі бойынша қолжетімді.

#### sd-ai-agent/site-scrape

**Мақсаты:** Дизайн шабытын, контент құрылымын және функционалдылықты алу үшін бар веб-сайттарды талдау.

**Мүмкіндіктері:**

- **Веб-сайтты талдау** — бәсекелес немесе шабыт беретін веб-сайттарды жинау және талдау
- **Дизайнды шығарып алу** — түстерді, қаріптерді және орналасу үлгілерін анықтау
- **Контент құрылымы** — бет ұйымдастырылуын және иерархиясын түсіну
- **Мүмкіндіктерді анықтау** — plugin-дерді және функционалдылықты анықтау
- **Өнімділікті талдау** — бет жылдамдығын және оңтайландыруды тексеру
- **SEO талдауы** — meta tag-терді және құрылымдалған деректерді қарап шығу

**Пайдалану:**

```
example.com дизайнын талдап, оны менің сайтым үшін шабыт ретінде пайдаланыңыз.
```

**Ол нені шығарып алады:**

- Түстер палитрасы және түстерді пайдалану
- Типография (қаріптер және өлшемдер)
- Орналасу құрылымы және арақашықтықтар
- Навигация үлгілері
- Контентті ұйымдастыру
- Суреттер және медианы пайдалану
- Интерактивті элементтер
- Мобильді бейімделгіштік

**Шектеулері:**

- Құпиясөзбен қорғалған сайттарды жинай алмайды
- robots.txt және сайт саясаттарын сақтайды
- Динамикалық контентті толық қамтымауы мүмкін
- JavaScript көп қолданылатын сайттарда талдау шектеулі болуы мүмкін
- Үлкен сайттарды талдауға көбірек уақыт кетуі мүмкін

### Tier 2 құралдары (міндетті емес)

Tier 2 құралдары Theme Builder баптауларында нақты қосылған кезде қолжетімді болады.

#### Кеңейтілген аналитика

Толық өнімділік метрикаларын ұсынады:

- Беттің жүктелу уақыттары
- Core Web Vitals
- SEO ұпайы
- Қолжетімділік ұпайы
- Үздік тәжірибелер ұпайы

#### Контентті оңтайландырғыш

Мыналар үшін жақсартуларды талдайды және ұсынады:

- Оқылымдылық
- SEO оңтайландыруы
- Кілтсөздерді пайдалану
- Контент құрылымы
- Әрекетке шақырудың орналасуы

### Tier 3 құралдары (Premium)

Tier 3 құралдары қосымша конфигурацияны немесе premium мүмкіндіктерді қажет етеді.

#### AI контент генераторы

Мыналар үшін контент жасайды:

- Өнім сипаттамалары
- Қызмет беттері
- Блог жазбалары
- Meta сипаттамалар
- Әрекетке шақыру мәтіні

#### Кеңейтілген дизайн құралдары

Кеңейтілген дизайн мүмкіндіктерін ұсынады:

- Жеке CSS жасау
- Анимация жасау
- Интерактивті элемент дизайны
- Кеңейтілген түс теориясы
- Типографияны оңтайландыру

### Advanced серіктес құралдары (бөлек пакет)

Advanced серіктес пакеті негізгі Superdav AI Agent шығарылымынан бөлек таратылады. Ол әкімшілер агентке жоғары әсері бар құралдарды пайдалануға нақты рұқсат беретін сенімді әзірлеуші және сайт иесі орталарына арналған.

Advanced серіктес мүмкіндіктері мыналарды қамтуы мүмкін:

- **Файлдық жүйе құралдары** — сенімді орталарда бекітілген файлдарды тексеру және өзгерту
- **Дерекқор құралдары** — нақты авторизацияланған кезде сайт деректеріне сұрау жасау немесе жаңарту
- **WP-CLI құралдары** — WordPress қызмет көрсету және тексеру командаларын іске қосу
- **REST диспетчер құралдары** — бақыланатын dispatch жұмыс процестері арқылы тіркелген REST endpoint-терін шақыру
- **Plugin құрастыру құралдары** — plugin кодын бастапқы қалыпқа келтіру және итерациялау
- **Git snapshot құралдары** — қауіпті әзірлеуші операцияларына дейін snapshot-тар жасау
- **Пайдаланушыны басқару құралдары** — рұқсат етілген жерде пайдаланушы жазбаларын қарап шығуға немесе түзетуге көмектесу
- **Бенчмарк құралдары** — әзірлеу шолуы үшін өнімділік немесе мүмкіндік бенчмарктарын іске қосу

Бұларды әрбір орнатылым үшін жалпы қолжетімді Theme Builder құралдары ретінде құжаттамаңыз. Олар Advanced серіктес пакеті орнатылған, қосылған және сенімді әкімшілермен шектелген кезде ғана қолжетімді.

## Құрал конфигурациясы

### Құралдарды қосу

Theme Builder ішінде қосымша құралдарды қосу үшін:

1. **Gratis AI Agent → Settings** бөліміне өтіңіз
2. **Theme Builder → Tools** бөліміне өтіңіз
3. Қажетіне қарай құралдарды қосыңыз/өшіріңіз
4. Баптауларды сақтаңыз

### Құрал рұқсаттары

Кейбір құралдар рұқсаттарды қажет етеді:

- **Сайтты жинау** — интернетке қолжетімділікті қажет етеді
- **Аналитика** — Google Analytics интеграциясын қажет етеді
- **Контент жасау** — API кілттерін қажет етеді
- **Кеңейтілген мүмкіндіктер** — premium жазылымды қажет етуі мүмкін
- **Advanced серіктес құралдары** — бөлек таратылатын Advanced пакетін және әкімшінің нақты сенімін қажет етеді

### API кілттері және тіркелгі деректері

Оларды қажет ететін құралдар үшін API кілттерін конфигурациялаңыз:

1. **Gratis AI Agent → Settings → API Keys** бөліміне өтіңіз
2. Әр құрал үшін тіркелгі деректерін енгізіңіз
3. Қосылымды тексеріңіз
4. Қауіпсіз сақтаңыз

### Advanced серіктес пакетін орнату

Advanced серіктес пакеті WordPress.org негізгі ZIP файлынан бөлек шығарылады. Оны тек шығарылымға арналған ресми жоба тарату арнасынан орнатыңыз, содан кейін қолжетімділікті сенімді әкімшілермен шектеңіз. Production сайтында файлдық жүйе, дерекқор, WP-CLI, REST диспетчер, plugin құрастыру, пайдаланушыны басқару немесе бенчмарк құралдарын қоспас бұрын адам мақұлдауы қақпаларын және git snapshot әрекетін қарап шығыңыз.

## sd-ai-agent/site-scrape пайдалану

### Негізгі пайдалану

Theme Builder-ден веб-сайтты талдауды сұраңыз:

```
competitor.com мекенжайындағы бәсекелесімнің сайтының дизайнын талдап,
менің сайтым үшін дизайн жақсартуларын ұсыныңыз.
```

### Нақты талдау

Нақты талдау түрлерін сұраңыз:

```
example.com сайтынан түстер палитрасын шығарып алып, оны шабыт ретінде пайдаланыңыз.
```

```
example.com навигация құрылымын талдап, ұқсас
ұйымдастыруды менің сайтыма қолданыңыз.
```

```
example.com мобильді бейімделгіштігін тексеріп, менің сайтымның
дәл сондай бейімделгіш болуын қамтамасыз етіңіз.
```

### Салыстырмалы талдау

Бірнеше сайтты салыстырыңыз:

```
site1.com және site2.com дизайндарын салыстырып, екеуінің де ең жақсы
элементтерін біріктіретін дизайн жасаңыз.
```

## Құрал шектеулері және ескертпелер

### Жылдамдықты шектеу

- Сервердің шамадан тыс жүктелуін болдырмау үшін scraping жылдамдығы шектеледі
- Бір сайтқа бірнеше сұрау throttled болуы мүмкін
- Үлкен сайттарды талдау ұзағырақ уақыт алуы мүмкін

### Мазмұн дәлдігі

- Динамикалық мазмұн толық қамтылмауы мүмкін
- JavaScript арқылы көрсетілетін мазмұн толық болмауы мүмкін
- Нақты уақыттағы деректер көрсетілмеуі мүмкін
- Кейбір мазмұн paywall артында болуы мүмкін

### Құқықтық және этикалық ескерімдер

- robots.txt және сайт саясаттарын құрметтеңіз
- Авторлық құқықпен қорғалған мазмұнды қайта жариялау үшін scraping жасамаңыз
- Талдауды көшіру үшін емес, шабыт алу үшін пайдаланыңыз
- Алынған мазмұнды пайдалануға құқығыңыз бар екенін тексеріңіз
- Талданған сайттардың қызмет көрсету шарттарын сақтаңыз

### Өнімділікке әсері

- Үлкен сайттарды scraping жасау уақыт алуы мүмкін
- Бір уақытта бірнеше scraping баяуырақ болуы мүмкін
- Желі қосылымы жылдамдыққа әсер етеді
- Үлкен деректер жиындары көбірек өңдеуді қажет етуі мүмкін

## Үздік тәжірибелер

### Сайт талдауын пайдалану

1. **Бірнеше сайтты талдаңыз** — бірнеше дереккөзден шабыт жинаңыз
2. **Құрылымға назар аударыңыз** — орналасу және ұйымдастыру үлгілерін үйреніңіз
3. **Түстерді алыңыз** — түс палитраларын бастапқы нүкте ретінде пайдаланыңыз
4. **Типографияны зерттеңіз** — өзіңізге ұнайтын қаріп үйлесімдерін анықтаңыз
5. **Навигацияны қарап шығыңыз** — мәзір ұйымдастырылуын түсініңіз

### Этикалық scraping

1. **robots.txt құрметтеңіз** — сайт нұсқауларын орындаңыз
2. **Мазмұнды көшірмеңіз** — талдауды тек шабыт алу үшін пайдаланыңыз
3. **Құқықтарды тексеріңіз** — алынған мазмұнды пайдалана алатыныңызға көз жеткізіңіз
4. **Дереккөздерді көрсетіңіз** — шабыт көздеріне сілтеме беріңіз
5. **Шарттарды орындаңыз** — сайттың қызмет көрсету шарттарын сақтаңыз

### Құрал тиімділігін арттыру

1. **Нақты болыңыз** — нақты талдау түрлерін сұраңыз
2. **Мәнмәтін беріңіз** — сайтыңыздың мақсатын түсіндіріңіз
3. **Күтулерді белгілеңіз** — дизайн мақсаттарыңызды сипаттаңыз
4. **Қайталап жетілдіріңіз** — нәтижелер негізінде нақтылаңыз
5. **Құралдарды біріктіріңіз** — жан-жақты талдау үшін бірнеше құралды пайдаланыңыз

## Ақауларды жою

### Сайт scraping жасалмайды

- Сайттың көпшілікке қолжетімді екенін тексеріңіз
- robots.txt scraping жасауға рұқсат беретінін тексеріңіз
- Басқа сайтты қолданып көріңіз
- Интернет қосылымын тексеріңіз
- Мәселе жалғасса, қолдау қызметіне хабарласыңыз

### Толық емес талдау

- Сайтта динамикалық мазмұн болуы мүмкін
- JavaScript толық көрсетілмеуі мүмкін
- Үлкен сайттар timeout болуы мүмкін
- Оның орнына нақты беттерді талдап көріңіз
- Нақты талдау түрлерін сұраңыз

### Баяу өнімділік

- Үлкен сайттарды талдау ұзағырақ уақыт алады
- Бір уақытта бірнеше scraping баяуырақ болады
- Желі қосылымы жылдамдыққа әсер етеді
- Жүктеме аз уақыттарда қолданып көріңіз
- Алдымен шағын бөлімдерді талдаңыз
