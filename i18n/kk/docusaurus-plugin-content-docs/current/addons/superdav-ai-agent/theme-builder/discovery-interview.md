# Танысу сұхбаты {#discovery-interview}

**Танысу сұхбаты** — Theme Builder дизайн процесіндегі алғашқы қадам. Ол дербестендірілген theme жасау үшін сайтыңыз, бизнесіңіз және дизайн қалауларыңыз туралы ақпарат жинайды.

## Шолу {#overview}

Танысу сұхбаты брендіңізді, мақсатты аудиторияңызды және дизайн мақсаттарыңызды түсіну үшін сізді бірқатар сұрақ арқылы бағыттайды. 1.16.0 нұсқасында сұхбат енді дизайныңызға визуалды шабыт беруге мүмкіндік беретін **фото жүктеу қадамын** қамтиды.

## Сұхбат қадамдары {#interview-steps}

### 1. Бизнес туралы ақпарат {#1-business-information}

- Сайт атауы және ұраны
- Бизнес түрі немесе сала
- Негізгі бизнес мақсаттары
- Мақсатты аудитория сипаттамасы

### 2. Дизайн қалаулары {#2-design-preferences}

- Қалаулы түс сұлбасы
- Дизайн стилі (заманауи, классикалық, минималистік, ойнақы және т.б.)
- Шабыт беретін веб-сайттар немесе брендтер
- Міндетті дизайн элементтері

### 3. Фото жүктеу (v1.16.0 нұсқасындағы жаңалық) {#3-photo-upload-new-in-v1160}

Дизайн бағытына ықпал ету үшін бір немесе бірнеше фото жүктеңіз:

#### Қабылданатын форматтар {#accepted-formats}

| Формат | Кеңейтім | Мәліметтер |
|--------|-----------|---------|
| **JPEG** | `.jpg`, `.jpeg` | Фотосуреттер үшін ұсынылады |
| **PNG** | `.png` | Мөлдірлікті қолдайды |
| **WebP** | `.webp` | Заманауи формат, файл өлшемі кішірек |
| **GIF** | `.gif` | Тек статикалық суреттер (анимациясыз) |

#### Файл талаптары {#file-requirements}

- **Ең үлкен файл өлшемі:** 10 MB
- **Ең кіші өлшемдер:** 400 × 300 пиксель
- **Ұсынылатын өлшемдер:** 1200 × 800 пиксель немесе үлкенірек
- **Арақатынасы:** Кез келген арақатынас жарамды

#### Фотолар дизайн бағытына қалай ықпал етеді {#how-photos-influence-design-direction}

Жүктеген фотоларыңыз Theme Builder-ге мыналарды түсінуге көмектеседі:

- **Түс палитрасы** — басым түстер анықталып, theme ішінде қолданылады
- **Визуалды стиль** — заманауи, рустикалық, минималистік, жарқын және т.б.
- **Көңіл күй мен атмосфера** — кәсіби, шығармашылық, достық, сәнді және т.б.
- **Дизайн элементтері** — өрнектер, текстуралар және композициялық стильдер
- **Типография шабыттары** — визуалды theme-мен үйлесетін қаріп стильдері

### 4. Контент құрылымы {#4-content-structure}

- Қажет негізгі беттер (Home, About, Services, Contact және т.б.)
- Блог немесе жаңалықтар бөлімі қажет пе
- E-commerce функционалдығы қажет пе
- Арнайы мүмкіндіктер немесе интеграциялар

### 5. Қарау және растау {#5-review-and-confirmation}

- Берілген барлық ақпараттың қысқаша мазмұны
- Дизайнды генерациялау басталмас бұрын растау

## Фото жүктеудің үздік тәжірибелері {#photo-upload-best-practices}

### Нені жүктеу керек {#what-to-upload}

- **Бренд шабыттары** — логотиптер, түс үлгілері немесе бренд нұсқаулықтары
- **Бәсекелес сайттары** — өзіңізге ұнайтын сайттардың скриншоттары
- **Көңіл күй тақталары** — қалаған эстетикаңызды білдіретін суреттер жинақтары
- **Өнім фотолары** — көрсетуге арналған өнімдеріңіз болса
- **Команда немесе кеңсе фотолары** — жеке немесе кәсіби брендинг үшін
- **Өмір салты суреттері** — мақсатты аудиторияңызды білдіретін

### Неден аулақ болу керек {#what-to-avoid}

- **Сапасы төмен суреттер** — бұлыңғыр немесе пиксельденген фотолар
- **Қатысы жоқ суреттер** — брендіңізді білдірмейтін фотолар
- **Мәтіні көп суреттер** — мәтіні көп скриншоттар
- **Авторлық құқықпен қорғалған материал** — суреттерді пайдалануға құқығыңыз бар екеніне көз жеткізіңіз
- **Анимацияланған GIF-тер** — тек статикалық суреттер өңделеді

### Ең жақсы нәтижелерге арналған кеңестер {#tips-for-best-results}

1. **2-3 сурет жүктеңіз** — бірнеше сурет жақсырақ контекст береді
2. **Жоғары сапалы фотоларды пайдаланыңыз** — анық, жақсы жарықтандырылған суреттер ең жақсы жұмыс істейді
3. **Әртүрлілік қосыңыз** — түс, стиль және тақырып қоспасы
4. **Контекст беріңіз** — әр суретті неге таңдағаныңызды қысқаша сипаттаңыз
5. **Нақты болыңыз** — белгілі бір элемент ұнаса, соны көрсетіңіз

## Дизайн бағытының нәтижесі {#design-direction-output}

Танысу сұхбатынан кейін Theme Builder мыналарды генерациялайды:

- **Түс палитрасы** — негізгі, қосымша және акцент түстер
- **Типография** — тақырыптар мен негізгі мәтінге арналған қаріп отбасылары
- **Орналасу стилі** — аралықтар, туралау және компоненттерді орналастыру
- **Визуалды элементтер** — белгішелер, өрнектер және сәндік элементтер
- **Көңіл күй тақтасы** — дизайн бағытының визуалды көрінісі

## Келесі қадамдар {#next-steps}

Танысу сұхбатын аяқтағаннан кейін:

1. Генерацияланған дизайн бағытын қарап шығыңыз
2. Кері байланыс беріңіз немесе түзетулер сұраңыз
3. [Дизайн бағыты](./design-direction.md) қадамына өтіңіз
4. Theme Builder сіздің custom theme-ді жасайды
