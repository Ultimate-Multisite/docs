# Қонақжайлылық мәзірлері

**Қонақжайлылық мәзірлері** мүмкіндігі Theme Builder-ге құрылымдалған тағам және сусын мәзірі беттерін тікелей WordPress сайтыңызға жасауға және ендіруге мүмкіндік береді.

## Шолу {#overview}

Theme Builder енді мейрамханалар, кафелер, барлар және кейтеринг қызметтері сияқты қонақжайлылық бизнесіне арналған кәсіби, құрылымдалған мәзір беттерін жасай алады. Бұл мәзірлер сайтыңыздың дизайнымен толық біріктіріледі және оларды оңай жаңартып, басқаруға болады.

## Қолдау көрсетілетін қонақжайлылық түрлері {#supported-hospitality-types}

- **Мейрамханалар** — толық қызмет көрсетілетін тамақтану мәзірлері
- **Кафелер** — кофе және жеңіл тағам мәзірлері
- **Барлар мен лаунждар** — сусын және тіскебасар мәзірлері
- **Наубайханалар** — тәтті тоқаш және нан мәзірлері
- **Кейтеринг қызметтері** — іс-шара мәзірі нұсқалары
- **Фудтрактар** — жылжымалы тағам қызметі мәзірлері
- **Сыра және шарап өндірістері** — сипаттамалары бар сусын мәзірлері

## Мәзір құрылымы {#menu-structure}

### Мәзір санаттары {#menu-categories}

Мәзірлер санаттарға ұйымдастырылады:

- **Тіскебасарлар** — бастапқы тағамдар және шағын табақтар
- **Негізгі тағамдар** — басты ас түрлері
- **Қосымша тағамдар** — гарнирлер және көкөністер
- **Десерттер** — тәтті тағамдар
- **Сусындар** — ішімдіктер (алкогольді және алкогольсіз)
- **Арнайы ұсыныстар** — күнделікті немесе маусымдық арнайы ұсыныстар

### Мәзір элементінің пішімі {#menu-item-format}

Әр мәзір элементіне мыналар кіреді:

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### Элемент өрістері {#item-fields}

| Өріс | Түрі | Сипаттама |
|-------|------|-------------|
| `name` | string | Тағам немесе сусын атауы |
| `description` | string | Элементтің егжей-тегжейлі сипаттамасы |
| `price` | string | Баға (валютамен пішімделген) |
| `dietary_info` | array | Диеталық атрибуттар (веган, глютенсіз және т.б.) |
| `allergens` | array | Кең таралған аллергендер (жаңғақтар, теңіз шаяндары және т.б.) |
| `availability` | string | Қашан қолжетімді (күнделікті, маусымдық және т.б.) |

## Қонақжайлылық мәзірлерін жасау {#creating-hospitality-menus}

### 1-қадам: Мәзір ақпаратын беріңіз {#step-1-provide-menu-information}

Theme Builder-ге мәзіріңіз туралы айтыңыз:

```
Create a restaurant menu for my Italian restaurant with appetizers,
pasta dishes, main courses, and desserts. Include prices and descriptions.
```

### 2-қадам: Theme Builder мәзір жасайды {#step-2-theme-builder-generates-menu}

Theme Builder:

1. Құрылымдалған мәзір бетін жасайды
2. Оны сайтыңыздың theme-іне сәйкестендіріп дизайндайды
3. Элементтерді санаттарға ұйымдастырады
4. Бағалар мен сипаттамаларды пішімдейді
5. Диеталық және аллерген ақпаратын қосады

### 3-қадам: Қарап шығып, баптаңыз {#step-3-review-and-customize}

Сіз мыналарды істей аласыз:

1. Мәзір элементтері мен бағаларын өңдеу
2. Санаттарды қосу немесе алып тастау
3. Санаттар ішіндегі элементтердің ретін өзгерту
4. Сипаттамалар мен диеталық ақпаратты жаңарту
5. Стильдеу мен орналасуды реттеу

## Мәзірді көрсету нұсқалары {#menu-display-options}

### Толық мәзір беті {#full-menu-page}

Толық мәзіріңізді көрсететін арнайы бет:

- Санат бойынша ұйымдастырылған
- Іздеуге және сүзуге болады
- Басып шығаруға ыңғайлы орналасу
- Мобильді құрылғыларға бейімделген дизайн

### Мәзір виджеті {#menu-widget}

Мәзір бөлімдерін басқа беттерге ендіріңіз:

- Басты беттегі таңдаулы элементтер
- Бүйірлік тақтадағы күнделікті арнайы ұсыныстар
- Бар бетіндегі сусын мәзірі
- Төменгі колонтитулдағы десерт көрмесі

### Мәзір PDF {#menu-pdf}

Жүктеп алуға болатын PDF мәзір жасаңыз:

- Кәсіби пішімдеу
- Басып шығаруға дайын сапа
- Суреттер мен сипаттамаларды қамтиды
- Бөлісуге және email арқылы жіберуге оңай

## Диеталық және аллерген ақпараты {#dietary-and-allergen-information}

### Диеталық атрибуттар {#dietary-attributes}

Элементтерді диеталық ақпаратпен белгілеңіз:

- **Веган** — жануар текті өнімдер жоқ
- **Вегетариандық** — ет жоқ
- **Глютенсіз** — целиакия ауруы бар адамдар үшін қауіпсіз
- **Сүт өнімдерінсіз** — сүт өнімдері жоқ
- **Жаңғақсыз** — ағаш жаңғақтары немесе жержаңғақ жоқ
- **Көмірсуы аз** — көмірсулары азайтылған
- **Ақуызға бай** — ақуызы мол

### Аллерген ескертулері {#allergen-warnings}

Кең таралған аллергендерді қосыңыз:

- **Жаңғақтар** — ағаш жаңғақтары және жержаңғақ
- **Теңіз шаяндары** — шаянтәрізділер және моллюскалар
- **Балық** — барлық балық түрлері
- **Сүт өнімдері** — сүт және сүт өнімдері
- **Жұмыртқа** — тауық жұмыртқалары
- **Соя** — соя өнімдері
- **Глютен** — бидай және құрамында глютені бар дәнді дақылдар
- **Күнжіт** — күнжіт тұқымдары және майы

## Мәзірді басқару {#menu-management}

### Бағаларды жаңарту {#updating-prices}

Мәзір бағаларын оңай жаңартыңыз:

1. Мәзір бетіне өтіңіз
2. "Мәзірді өңдеу" түймесін басыңыз
3. Элементтердің бағаларын жаңартыңыз
4. Өзгерістерді сақтаңыз
5. Өзгерістер сайтыңызда бірден пайда болады

### Маусымдық элементтерді қосу {#adding-seasonal-items}

Маусымдық мәзір нұсқаларын жасаңыз:

1. Жаңа мәзір нұсқасын жасаңыз
2. Маусымдық элементтерді қосыңыз
3. Элементтерді "Маусымдық" деп белгілеңіз
4. Қолжетімділік күндерін жоспарлаңыз
5. Маусым кезінде автоматты түрде көрсетіңіз

### Арнайы ұсыныстарды басқару {#managing-specials}

Күнделікті немесе апталық арнайы ұсыныстарды көрсетіңіз:

1. "Арнайы ұсыныстар" санатын жасаңыз
2. Қолжетімділік күндері бар элементтерді қосыңыз
3. Басты бетте арнайы ұсыныстарды ерекшелеңіз
4. Күнделікті немесе апта сайын жаңартыңыз
5. Ескі арнайы ұсыныстарды архивтеңіз

## Theme Builder-мен интеграция {#integration-with-theme-builder}

Қонақжайлылық сайттары үшін Theme Builder пайдаланғанда:

1. **Мәзірді автоматты анықтау** — тағам/сусын бизнесі екеніңізді анықтайды
2. **Мәзір бетін жасау** — кәсіби мәзір беттерін жасайды
3. **Дизайн сәйкестігі** — мәзірлер сайтыңыздың theme-іне сәйкес келеді
4. **Мобильді оңтайландыру** — мәзірлер телефондарда әдемі көрсетіледі
5. **SEO оңтайландыру** — мәзірлер іздеу жүйелеріне ыңғайлы

## Үздік тәжірибелер {#best-practices}

### Мәзір дизайны {#menu-design}

- **Айқын ұйымдастыру** — логикалық санат құрылымы
- **Оқуға ыңғайлы сипаттамалар** — тәбет ашатын және ақпаратты
- **Бірізді баға қою** — айқын валюта және пішімдеу
- **Кәсіби фотосуреттер** — жоғары сапалы тағам суреттері
- **Бос кеңістік** — бетті тым толтырмаңыз

### Мазмұн {#content}

- **Дәл сипаттамалар** — элементтерді дәл сипаттаңыз
- **Арнайы ұсыныстарды ерекшелеу** — арнайы ұсыныстарды көзге түсіріңіз
- **Аллергендерді қосу** — аллергендерді әрқашан тізімдеңіз
- **Тұрақты жаңарту** — бағалар мен элементтерді өзекті ұстаңыз
- **Тәбет ашатын тіл қолданыңыз** — элементтерді дәмді етіп сипаттаңыз

### Қолжетімділік {#accessibility}

- **Оқуға ыңғайлы қаріптер** — анық, оңай оқылатын қаріп түрлерін қолданыңыз
- **Жеткілікті контраст** — мәтіннің оқылуын қамтамасыз етіңіз
- **Диеталық белгілер** — диеталық нұсқаларды анық белгілеңіз
- **Аллерген ескертулері** — аллергендерді көрнекті түрде көрсетіңіз
- **Мобильді құрылғыларға ыңғайлы** — барлық құрылғыларда тексеріңіз
