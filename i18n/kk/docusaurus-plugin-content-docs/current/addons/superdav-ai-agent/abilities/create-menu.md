# Мәзір жасау

**Create Menu** мүмкіндігі бет тақырыптарынан бөлек әртүрлі навигация белгілерін қолдай отырып, WordPress навигация мәзірлерін жасайды.

## Шолу

Бұл мүмкіндік стандартты мәзір жасау функционалдығын `navigation_label` параметрін көрсету мүмкіндігімен кеңейтеді. Бұл навигацияда көрсетілетін белгі бет тақырыбынан өзгеше болатын мәзірлер жасауға мүмкіндік береді, сайт құрылымы мен пайдаланушы тәжірибесінде көбірек икемділік береді.

## Параметрлер

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | string | Yes | Мәзір атауы, мысалы, `Primary Navigation` |
| `location` | string | No | Осы мәзір тағайындалатын Theme орналасуы, мысалы, `primary` |
| `navigation_label` | string | No | Навигацияда көрсетілетін белгі (бет тақырыбынан бөлек) |

## Қайтарылатын мән

```json
{
  "success": true,
  "menu_id": 7,
  "name": "Primary Navigation",
  "navigation_label": "Main Menu"
}
```

## Навигация белгісі және бет тақырыбы

`navigation_label` параметрі ішкі мәзір атауын пайдаланушыларға навигацияда көрсетілетін белгіден бөлуге мүмкіндік береді:

- **`name`** — WordPress пайдаланатын ішкі мәзір идентификаторы (мысалы, "Primary Navigation")
- **`navigation_label`** — Сайт келушілеріне навигацияда көрсетілетін белгі (мысалы, "Main Menu")

Бұл мына жағдайларда пайдалы:
- Ішкі атау беру ережеңіз пайдаланушыларға көрінетін белгілерден өзгеше болса
- Навигацияда admin панеліне қарағанда қысқарақ белгілер қажет болса
- Белгі ұзындығы әртүрлі бірнеше тілді қолдау қажет болса
- Белгілі бір өңірлерге немесе пайдаланушы топтарына арналған мәзірлер құрып жатсаңыз

## Қолдану мысалдары

### 1-мысал: Навигация белгісі бар қарапайым мәзір

**Сұрау:**
```
"Menu" навигация белгісі бар "Main Navigation" деп аталатын негізгі навигация мәзірін жасаңыз.
```

**Нәтиже:**
```json
{
  "success": true,
  "menu_id": 7,
  "name": "Main Navigation",
  "navigation_label": "Menu"
}
```

### 2-мысал: Белгілі бір Theme орналасуына арналған мәзір

**Сұрау:**
```
"Quick Links" навигация белгісі бар "Footer Links" деп аталатын төменгі деректеме мәзірін жасаңыз және оны төменгі деректеме орналасуына тағайындаңыз.
```

**Нәтиже:**
```json
{
  "success": true,
  "menu_id": 8,
  "name": "Footer Links",
  "navigation_label": "Quick Links",
  "location": "footer"
}
```

## Theme Builder-пен интеграция

Theme Builder пайдаланылғанда, Create Menu мүмкіндігі:

1. Қолжетімді Theme мәзір орналасуларын автоматты түрде анықтайды
2. Дизайныңызға сәйкес навигация белгілері бар мәзірлер жасайды
3. Мәзірлерді дұрыс Theme орналасуларына тағайындайды
4. Жасалғаннан кейін мәзір элементтерін қосуды қолдайды

## Байланысты мүмкіндіктер

- **`add_menu_item`** — Бар мәзірге элементтер қосу
- **`update_menu`** — Мәзірдің атауын өзгерту немесе оны Theme орналасуына қайта тағайындау
- **`delete_menu`** — Сайтыңыздан мәзірді жою

## Үздік тәжірибелер

- **Анық навигация белгілерін қолданыңыз** — белгілерді пайдаланушылар үшін қысқа әрі сипаттамалы ұстаңыз
- **Theme орналасуларын сәйкестендіріңіз** — дұрыс көрсету үшін мәзірлерді тиісті Theme орналасуына тағайындаңыз
- **Мәзір құрылымын жоспарлаңыз** — элементтер жасамас бұрын мәзір иерархиясын шешіңіз
- **Бейімделгіштікті тексеріңіз** — мәзірлердің мобильді құрылғыларда дұрыс көрсетілетінін тексеріңіз
- **Белгілерді жергіліктендіріңіз** — әртүрлі тіл нұсқалары үшін әртүрлі навигация белгілерін қолданыңыз

## Шектеулер

- Навигация белгілері тек көрсетуге арналған; ішкі `name` әлі де WordPress идентификациясы үшін пайдаланылады
- Theme қолдауы әртүрлі болуы мүмкін; барлық Theme барлық мәзір орналасуларын қолдамайды
- Мәзір элементтері мәзір жасалғаннан кейін бөлек қосылуы керек
- Навигация белгісін өзгерту мәзірді жаңартуды талап етеді
