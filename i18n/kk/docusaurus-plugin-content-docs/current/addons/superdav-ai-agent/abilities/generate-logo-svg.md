# Logo SVG жасау

**Logo SVG жасау** мүмкіндігі Theme Builder-ге автоматты түрде namespace-қауіпсіз тазартумен тікелей WordPress сайтыңызға арнаулы логотип SVG-лерін жасауға және енгізуге мүмкіндік береді.

## Шолу

Бұл мүмкіндік сайтыңыздың брендтік бағыты мен дизайн қалаулары негізінде масштабталатын векторлық графика (SVG) логотиптерін жасайды. Жасалған SVG-лер WordPress-та қолдануға қауіпсіз болуы үшін автоматты түрде тазартылады, сонымен бірге визуалды тұтастығы сақталады.

## Параметрлер

| Параметр | Түрі | Міндетті | Сипаттама |
|-----------|------|----------|-------------|
| `site_name` | string | Иә | Логотип жасалатын сайттың атауы |
| `style` | string | Иә | Дизайн стилі (мысалы, "заманауи", "классикалық", "минималистік", "ойнақы") |
| `colors` | array | Жоқ | Логотипте қолданылатын hex түс кодтарының массиві (мысалы, `["#678233", "#ffffff"]`) |
| `width` | number | Жоқ | SVG ені пиксельмен (әдепкі: 200) |
| `height` | number | Жоқ | SVG биіктігі пиксельмен (әдепкі: 200) |
| `include_text` | boolean | Жоқ | Логотипте сайт атауын мәтін ретінде қосу-қоспау (әдепкі: true) |

## Шығыс пішімі

Бұл мүмкіндік келесі құрылымы бар SVG жолын қайтарады:

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## SVG тазарту әрекеті

Жасалған SVG-лер автоматты namespace-қауіпсіз тазартудан өтеді, ол үшін:

- **Қауіпсіз емес атрибуттарды жою** — оқиға өңдегіштерін, скрипттерді және ықтимал қауіпті атрибуттарды алып тастайды
- **Namespace-терді сақтау** — дұрыс көрсету үшін SVG namespace-терін (xmlns, xlink) сақтайды
- **Құрылымды тексеру** — SVG-нің W3C стандарттарына сәйкес болуын қамтамасыз етеді
- **Нысандарды кодтау** — мәтін мазмұнындағы арнайы таңбаларды дұрыс экрандайды
- **Сыртқы сілтемелерді жою** — сыртқы стиль кестелері мен сурет сілтемелерін алып тастайды

Бұл SVG-ні қосымша тазартуды қажет етпей, тікелей WordPress ішіне енгізуге қауіпсіз етеді.

## Қолдану мысалы

**Сұрау:**
```
"CloudSync" деп аталатын tech стартапым үшін көк және ақ түстерді пайдаланып, заманауи логотип жасаңыз.
```

**Нәтиже:**
Бұл мүмкіндік мыналарды орындайтын SVG логотипін жасайды:
- "CloudSync" сайт атауын қамтиды
- Көрсетілген көк және ақ түс схемасын қолданады
- Заманауи дизайн қағидаларын ұстанады
- Автоматты түрде тазартылады және қолдануға дайын болады

## Theme Builder-пен интеграция

Theme Builder-дің дизайн бағытын таңдауын пайдаланғанда, Logo SVG жасау мүмкіндігі:

1. Дизайн бағытыңыз бен түс палитраңызды талдайды
2. Қалауларыңызға сәйкес арнаулы SVG логотипін жасайды
3. Логотипті сайтыңыздың header/брендинг аймағына автоматты түрде енгізеді
4. SVG-ні WordPress медиасында арнаулы логотип ретінде сақтайды

## Үздік тәжірибелер

- **Айқын стиль қалауларын беріңіз** — қалаған дизайн стилін сипаттаңыз (заманауи, классикалық, ойнақы және т.б.)
- **Түстерді көрсетіңіз** — бірізділік үшін бренд түстеріңізді қосыңыз
- **Көрсетілуін тексеріңіз** — логотиптің әртүрлі экран өлшемдерінде дұрыс көрсетілетінін тексеріңіз
- **Әрі қарай баптаңыз** — өлшемі мен орналасуын реттеу үшін WordPress-тің логотипті баптау құралдарын пайдаланыңыз

## Шектеулер

- SVG логотиптері статикалық графика ретінде жасалады (анимацияланған емес)
- Көп элементі бар күрделі логотиптер қолмен нақтылауды қажет етуі мүмкін
- Арнаулы қаріптерге қолдау көрсетілмейді; мәтін жүйелік қаріптерді пайдаланады
- Өте үлкен немесе өте кіші өлшемдер сапаға әсер етуі мүмкін
