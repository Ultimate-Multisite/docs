# Superdav AI Agent {#superdav-ai-agent}

Superdav AI Agent — тақырыпты ескеретін дағдылармен, жақсартылған интернеттен іздеу мүмкіндіктерімен, сайт құрастырушы функцияларымен және қосымша автоматтандыру интеграцияларымен Gratis AI Agent мүмкіндіктерін кеңейтетін озық WordPress AI көмекшісі. 1.18.0 нұсқасы Google Calendar оқу құралдарын, байланыс және қатысушы сәйкестендіруін, адам мақұлдау қақпаларын, еске салғыштарды қайталанбауын, TextBee SMS хабарландыруларын, басқарылатын Superdav қызметін дайындауды және сенімді әзірлеуші жұмыс процестері үшін бөлек таратылатын Advanced серіктес пакетін қосады.

## Негізгі мүмкіндіктер {#key-features}

- **Тақырыпты ескеретін дағдылар** — белсенді WordPress тақырыбыңызға және орнатылған плагиндерге қарай нұсқаулар мен мүмкіндіктерді автоматты түрде бейімдейді
- **Бірнеше интернет іздеу провайдері** — кеңірек іздеу нәтижелері мен озық зерттеу мүмкіндіктері үшін Brave Search пен Tavily арасынан таңдаңыз
- **Сайт құрастырушының байланыс формалары** — орнатылған форма плагинін пайдаланып, толық жұмыс істейтін байланыс формаларын тікелей чат интерфейсінен жасаңыз
- **Жақсартылған сайт құрастырушы** — плагиндерді жақсырақ анықтау және қателерден қалпына келтіруі жетілдірілген көп кезеңді сайт құрастыру жоспарлары
- **Кіріктірілген тақырып дағдылары** — Block Themes, Classic Themes, Kadence Blocks және Kadence Theme үшін мамандандырылған нұсқаулар
- **Жақсартылған WooCommerce интеграциясы** — сенімділік пен үйлесімділікті арттыру үшін жергілікті WooCommerce API интерфейстерін қолдануға қайта құрылды
- **Провайдерді автоматты анықтау** — плагиндер мен тақырыптар белсендірілгенде немесе өшірілгенде, қолжетімді плагиндер мен тақырыптарды автоматты түрде анықтап, жаңартады
- **Пайдаланушы логотипін жасау** — атаулар кеңістігіне қауіпсіз автоматты тазартумен пайдаланушы SVG логотиптерін жасап, ендіріңіз
- **Палитра контрастын тексеру** — тақырыбыңызға қолданбас бұрын түстер жұптарын WCAG қолжетімділік талаптарына сәйкестігі бойынша тексеріңіз
- **Фотомен толықтырылған анықтау** — дизайн бағытына әсер ету үшін Theme Builder анықтау сұхбаты кезінде фотолар жүктеңіз
- **Қонақжайлық мәзірлері** — мейрамханалар, кафелер және қонақжайлық бизнесі үшін құрылымдалған тағам мен сусын мәзірі беттерін жасаңыз
- **Үстелүсті және мобильді алдын ала қарау** — дизайн бағытын таңдау кезінде дизайныңызды үстелүсті және мобильді құрылғыларда алдын ала қараңыз
- **1-деңгей сайт скрейпингі** — sd-ai-agent/site-scrape енді бәсекелес және шабыт беретін веб-сайттарды талдау үшін әдепкі бойынша қолжетімді
- **Google Calendar оқу құралдары** — автоматтандырулар алдағы кесте контекстін түсінуі үшін бапталған күнтізбелер мен оқиғаларды оқиды
- **Байланыс және қатысушы сәйкестендіруі** — кейінгі жұмыстарды жасамас бұрын күнтізбе қатысушыларын белгілі сайт пайдаланушыларымен және байланыстармен сәйкестендіреді
- **Адам мақұлдау қақпалары** — хабар жіберу, жұмысты жалғастыру немесе өзгеріс енгізу алдында тексеру үшін сезімтал автоматтандыруларды кідіртеді
- **Еске салғыш жазбалары және қайталанбауы** — қайталанатын автоматтандыру іске қосуларында бір алушыға екі рет хабарлама жіберілмеуі үшін бұрын жіберілген еске салғыштарды қадағалайды
- **TextBee SMS хабарландырулары** — SMS провайдері қосылғанда, TextBee арқылы бапталған мәтіндік хабарлама хабарландыруларын жібереді
- **Басқарылатын Superdav қызметін дайындау** — қолдау көрсетілетін сайттарды қолмен endpoint баптаудың орнына автоматты орнатумен хостингтегі Superdav endpoint-теріне қосады
- **Advanced серіктес пакеті** — файл жүйесі, дерекқор, WP-CLI, REST диспетчері, плагин құрастырушы, git snapshot, пайдаланушыны басқару және benchmark жұмыс процестері үшін бөлек таратылатын сенімді әзірлеуші құралдары

## Талаптар {#requirements}

- WordPress 6.0 немесе одан жоғары
- PHP 8.1 немесе одан жоғары
- WordPress Multisite (желі бойынша белсендірілген) немесе бір сайттық орнатылым
- Белсенді AI провайдері қосқыш плагині (Anthropic, OpenAI немесе үйлесімді)
- Gratis AI Agent v1.9.0 немесе одан жоғары (Superdav AI Agent Gratis AI Agent мүмкіндіктерін кеңейтеді)

## Орнату {#installation}

1. Плагин файлдарын `/wp-content/plugins/superdav-ai-agent` ішіне жүктеңіз
2. Плагинді WordPress ішіндегі **Плагиндер** экраны арқылы белсендіріңіз
3. Gratis AI Agent те белсендірілгеніне көз жеткізіңіз
4. **Gratis AI Agent → Settings** бөліміне өтіп, қажетіне қарай интернет іздеу провайдерін, басқарылатын Superdav қызметі қосылымын, Google Calendar немесе TextBee тіркелгі деректерін баптаңыз

WordPress.org пакеті Superdav AI Agent негізгі мүмкіндіктерін қамтиды. Advanced серіктес пакеті сенімді әзірлеуші құралдары анық орнатылып, рұқсат етілген орталар үшін бөлек құрастырылып таратылады.

## Тақырыпты ескеретін дағдылар {#theme-aware-skills}

Superdav AI Agent белсенді тақырыбыңыз бен орнатылған плагиндерді автоматты түрде анықтап, көмекші контекстіне тиісті тақырыпты ескеретін дағдыны жүктейді. Қолжетімді дағдыларға мыналар кіреді:

| Дағды | Қолданылатын жері |
|---|---|
| **Block Themes** | WordPress блокқа негізделген (Full Site Editing) архитектурасын пайдаланатын тақырыптар |
| **Classic Themes** | PHP үлгілерін және классикалық редакторды пайдаланатын дәстүрлі WordPress тақырыптары |
| **Kadence Blocks** | Kadence Blocks плагині белсенді сайттар |
| **Kadence Theme** | Белсенді тақырыбы Kadence тақырыбы болып табылатын сайттар |

Әр дағды бойынша толық нұсқаулық үшін [Тақырыпты ескеретін дағдылар](../../user-guide/configuration/theme-aware-skills) бөлімін қараңыз.

## Интернет іздеу провайдерлері {#internet-search-providers}

Superdav AI Agent v1.10.0 интернет іздеу сұраулары үшін Brave Search баламасы ретінде **Tavily** қосады. Екі провайдер де нақты уақыттағы іздеу нәтижелерін ұсынады және **Gratis AI Agent → Settings → Advanced** бөлімінде баптала алады.

| Провайдер | Мүмкіндіктер | Баптау |
|---|---|---|
| **Brave Search** | Жылдам, құпиялыққа бағытталған іздеу нәтижелері | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Байырақ іздеу нәтижелері, кеңейтілген зерттеу мүмкіндіктері | [Tavily API](https://tavily.com/) |

Баптау нұсқаулары үшін [Интернеттен іздеу](../../user-guide/configuration/internet-search) бөлімін қараңыз.

## Күнтізбе, мақұлдау және хабарландыру жұмыс процестері {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 автоматтандыру жұмыс процестерін контент пен сайт құрастыру тапсырмаларынан тыс кеңейтеді:

- **Google Calendar құралдары** күнтізбе тіркелгі деректері қолжетімді болғанда, агентке бапталған күнтізбелер мен оқиға мәліметтерін оқуға мүмкіндік береді.
- **Байланыс және қатысушы көмекшілері** еске салғыштар немесе кейінгі әрекет тапсырмаларын дайындамас бұрын оқиға қатысушыларын WordPress пайдаланушыларына немесе белгілі контактілерге сәйкестендіреді.
- **Адам мақұлдауының кезеңдері** автоматтандыруға уәкілетті пайдаланушы ұсынылған әрекетті қарап шыққанға дейін кідіртуге мүмкіндік береді.
- **Еске салғыш жазбалары** автоматтандыру қайта орындалғанда немесе бірнеше рет жоспарланғанда қайталанатын күнтізбе немесе хабарландыру еске салғыштарының алдын алады.
- **TextBee SMS қолдауы** TextBee провайдері бапталған кезде мәтіндік хабарлама хабарландыруларын іске қосады.

Бұл құралдарды кестені ескеретін көмек, қатысушылармен кейінгі байланыс және сезімтал хабарлар жіберілмес бұрын әлі де адам тексеруін талап ететін еске салғыш жұмыс процестері үшін пайдаланыңыз.

Толық нұсқаулық үшін [Сайт құрастырушының байланыс формалары](../../user-guide/administration/site-builder-contact-forms) бөлімін қараңыз.

Толық нұсқаулық үшін [Қонақжайлық мәзірлері](./theme-builder/hospitality-menus.md) бөлімін қараңыз.

## Баптау {#configuration}

Барлық параметрлер WordPress әкімші бөлігіндегі **Gratis AI Agent → Settings** астында орналасқан. Superdav AI Agent келесі баптау опцияларын қосады:

| Параметр | Сипаттама |
|---|---|
| **Интернеттен іздеу провайдері** | Brave Search пен Tavily арасында таңдаңыз (міндетті емес) |
| **Tavily API Key** | Tavily интернет іздеуі үшін API кілті (Tavily таңдалса) |
| **Басқарылатын Superdav қызметі** | Қолдау көрсетілетін сайттарды соңғы нүктелерді автоматты дайындау арқылы орналастырылған Superdav қызметіне қосу |
| **Google Calendar тіркелгі деректері** | Агент күнтізбелер мен оқиғаларды оқи алуы үшін күнтізбеге қолжетімділікті баптау |
| **TextBee SMS провайдері** | Мақұлданған SMS хабарландыру жұмыс процестері үшін TextBee тіркелгі деректерін баптау |

## Кеңейтілген серіктес пакет {#advanced-companion-package}

Superdav AI Agent v1.18.0 сенімді әзірлеуші жұмыс процестері үшін бөлек таратылатын кеңейтілген серіктес пакетті қосады. Бұл мүмкіндіктер WordPress.org негізгі пакетіне кіріктірілмеген. Серіктес пакетті әкімшілер агентке әзірлеуші құралдарына қол жеткізуге айқын сенетін сайттарда ғана орнатыңыз.

Кеңейтілген құралдарға файлдық жүйе операциялары, дерекқорға қолжетімділік, WP-CLI, REST диспетчер жұмыс процестері, plugin құрастыру утилиталары, git суреттемелері, пайдаланушыларды басқару көмекшілері және өнімділікті өлшеу құралдары кіреді. Бұл құралдарды сенімді әкімшілермен шектеп, оларды өндірістік сайттарда қоспас бұрын мақұлдау кезеңдерінің баптауын қарап шығыңыз.

## v1.18.0 нұсқасындағы жақсартулар {#improvements-in-v1180}

- **Google Calendar қолдауы** — кестені ескеретін автоматтандырулар үшін бапталған күнтізбелер мен оқиғаларды оқу
- **Байланыс және қатысушы сәйкестендіруі** — күнтізбе қатысушыларын сайт пайдаланушылары мен контактілеріне сәйкестендіру
- **Адам мақұлдауының кезеңдері** — жалғастырмас бұрын сезімтал автоматтандыруларды тексеру үшін кідірту
- **Еске салғыштарды қайталамау** — қайта талпыныстарда немесе қайталанатын іске қосуларда қайталанатын хабарландыруларды болдырмау үшін еске салғыштарды жазу
- **TextBee SMS провайдері** — қажетті мақұлдаулардан кейін бапталған мәтіндік хабарлама хабарландыруларын жіберу
- **Басқарылатын Superdav қызметін баптау** — қолдау көрсетілетін сайттар үшін орналастырылған қызмет соңғы нүктелерін автоматты түрде дайындау
- **Кеңейтілген серіктес таратуы** — сенімді әзірлеуші құралдарын WordPress.org негізгі құрастырылымынан бөлек жариялау
- **Сенімділік түзетулері** — модель таңдауды, сұрау күту уақыттарын, әдепкі баптау моделін өңдеуді, reasoning-text өңдеуін, қайта талпыныс нұсқаулығын, күнтізбе token-ын күшейтуді, еске салғыштарды күшейтуді, onboarding іске қосуларын, расталған қабілет жалғастыруларын және WordPress.org қаптауын тексеру мәселелерін жақсарту

## v1.16.0 нұсқасындағы жақсартулар {#improvements-in-v1160}

- **AI клиент кэші** — енді сұраулар арасында тұрақты сақтау үшін transients арқылы қолдау табады, бұл ұзақ орындалатын агент тапсырмаларында деректердің жоғалуын болдырмайды
- **Tier 1 сайтты сканерлеу құралы** — sd-ai-agent/site-scrape енді бәсекелес және шабыт беретін веб-сайттарды талдау үшін Theme Builder ішінде әдепкі бойынша қолжетімді
- **Плагин жолындағы әрекет сілтемелері** — анығырақ болу үшін түзетілді және қайта аталды

## v1.10.0 нұсқасындағы жақсартулар {#improvements-in-v1100}

- **WooCommerce интеграциясы қайта құрылымдалды** — енді сенімділік пен үйлесімділікті жақсарту үшін native WooCommerce API-лерін пайдаланады
- **Провайдер тізімін автоматты жаңарту** — плагиндер белсендірілгенде немесе өшірілгенде автоматты түрде анықтап, қолжетімді провайдер тізімін жаңартады
- **Жетілдірілген сайт құрастырушы** — қателерден жақсырақ қалпына келуі бар көп кезеңді сайт құрастыру жоспарлары жақсартылды
- **Тақырыпқа бейімделетін дағдылар** — белсенді тақырыбыңызға автоматты түрде бейімделетін төрт жаңа кірістірілген дағды

## v1.10.0 нұсқасындағы қате түзетулері {#bug-fixes-in-v1100}

- Кейбір әкімші беттерінде шексіз қайта жүктеу циклдерін тудыратын `navigate-to` қабілеті түзетілді
- Санат және тег атауларын slug-тарға дұрыс сәйкестендірмейтін `list-posts` қабілеті түзетілді
- Алдыңғы қайта құрылымдаудан кейін namespace бүркеншік аттары жоқ WP-CLI командалары түзетілді
- Автоматтандыру кестелері әлі жасалмаған сайттарда оқиғаларды автоматтандырудың дұрыс өңделуі түзетілді
- Жүйелік нұсқаулық құрастырушыда дұрыс namespace префиксін пайдаланатын `memory-save` қабілеті түзетілді
- Скаляр құрал нәтижелерінің AI-ға қайтарылмас бұрын дұрыс оралуы түзетілді
- Ескі нұсқалардан жаңарту кезінде бұрынғы қабілет кілті пішімін дұрыс өңдейтін пайдалану статистикасы түзетілді
