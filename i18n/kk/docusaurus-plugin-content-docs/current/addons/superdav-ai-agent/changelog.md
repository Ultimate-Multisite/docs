# Өзгерістер журналы {#changelog}

## 1.18.0 — 2026-06-29 күні шығарылды {#1180--released-on-2026-06-29}

### Жаңа {#new}

- **Google Calendar құралдары** — кестеден хабардар автоматтандырулар үшін бапталған күнтізбелер мен оқиғаларды оқу
- **Контактіні сәйкестендіру және қатысушы көмекшілері** — күнтізбе қатысушыларын сайт пайдаланушыларымен және контактілермен сәйкестендіру
- **Адам мақұлдауының қақпалары және еске салу жазбалары** — тексеру үшін автоматтандыруларды кідірту және қайталанатын хабарландыруларды болдырмау
- **TextBee SMS провайдері** — TextBee арқылы бапталған мәтіндік хабарлама хабарландыруларын жіберу
- **Advanced қосымша пакеті** — WordPress.org жинағынан бөлек таратылатын сенімді әзірлеуші файлдық жүйесі, дерекқор, WP-CLI, REST диспетчері, plugin құрастырушысы, git суреті, пайдаланушыны басқару және benchmark құралдарын қосу

### Жақсартылды {#improved}

- **Басқарылатын Superdav қызметін баптау** — қолдау көрсетілетін сайттар үшін орналастырылған қызмет endpoint-терін және автоматты қосылым ұсынуды қосу
- **Шығарылымды қаптау** — бөлек негізгі және Advanced ZIP-терін құрастыру, екеуін де GitHub-та жариялау және WordPress.org-қа тек негізгі пакетті жіберу

### Түзетілді {#fixed}

- **AI сұрауының сенімділігі** — модель таңдауды, сұрау күту уақыттарын, әдепкі баптау моделін, пайымдау мәтінін өңдеуді және жарамсыз құрал шақырулары үшін қайталау нұсқаулығын жақсарту
- **Күнтізбе және еске салуды нығайту** — Google Calendar token-дерін және еске салуды қайталанбауын күшейту
- **Бастапқы баптау және мақұлдаудан кейін жалғастыру** — frontend бастапқы баптауын іске қосуды және расталған қабілеттерді жалғастыруды түзету
- **WordPress.org қаптамасын тексеру мәселелері** — негізгі шығарылымға арналған қаптаманы тексеру пікірлерін шешу

## 1.16.0 — 2026-05-20 күні шығарылды {#1160--released-on-2026-05-20}

### Жаңа {#new-1}

- **Logo SVG жасау қабілеті** — Theme Builder енді namespace-қа қауіпсіз тазартумен жеке logo SVG-лерін жасап, ендіре алады
- **Танысу сұхбатында фото жүктеп салу** — Theme Builder танысу сұхбаты енді байырақ дизайн контексті үшін фото жүктеп салу қадамын қамтиды
- **Палитра контрастын тексеру қабілеті** — theme-ге қолданбас бұрын түс жұптарының WCAG сәйкестігін тексеру
- **Қонақжайлық мәзірлері** — Theme Builder енді қонақжайлық бизнесі үшін құрылымдалған тағам және сусын мәзірі беттерін жасай алады
- **Desktop және mobile алдын ала қарауды көрсету** — дизайн бағытын таңдау кезінде дизайныңызды desktop және mobile құрылғыларда алдын ала қарау
- **Навигация белгісі параметрі** — Create Menu қабілеті енді бет атауынан бөлек жеке `navigation_label` қолдайды
- **Tier 1 құрал қолжетімділігі** — sd-ai-agent/site-scrape енді Theme Builder-де әдепкі бойынша қолжетімді Tier 1 құралы

### Түзетілді {#fixed-1}

- **AI Client кэші** — енді ұзақ жұмыс істейтін агент тапсырмаларында дерек жоғалуын болдырмай, сұраулар арасында тұрақты сақтау үшін transients арқылы қамтамасыз етіледі
- **Plugin жолындағы әрекет сілтемелері** — анықтығы үшін түзетілді және қайта аталды

## 1.10.0 — 2026-05-05 күні шығарылды {#1100--released-on-2026-05-05}

### Жаңа {#new-2}

- **Tavily интернет іздеуі** — Brave Search-пен қатар байырақ интернет іздеу нәтижелері үшін Tavily-ді іздеу провайдері ретінде қосу
- **Theme-нен хабардар кіріктірілген дағдылар** — Block Themes, Classic Themes, Kadence Blocks және Kadence Theme дағды нұсқаулықтары енді plugin-мен бірге жеткізіледі
- **Сайт құрастырушысының контакт формасы қабілеті** — чат интерфейсінен тікелей кез келген бетке контакт формасын қосу

### Жақсартылды {#improved-1}

- **WooCommerce интеграциясы қайта құрылды** — енді жақсырақ сенімділік және үйлесімділік үшін жергілікті WooCommerce API-лерін пайдаланады
- **Провайдер тізімі автоматты түрде жаңартылады** — кез келген plugin белсендірілгенде немесе өшірілгенде

### Түзетілді {#fixed-2}

- **navigate-to қабілеті** — кейбір admin беттеріндегі шексіз қайта жүктеу циклі түзетілді
- **list-posts қабілеті** — енді санат және тег атауларын slug-тарға дұрыс түрлендіреді
- **WP-CLI командалары** — алдыңғы қайта құрылымдаудан кейін жоқ namespace бүркеншік аттары қалпына келтірілді
- **Оқиға автоматтандыруы** — автоматтандыру кестелері әлі жасалмаған сайттарды сыпайы өңдейді
- **memory-save қабілеті** — енді жүйелік нұсқау құрастырушысында дұрыс namespace префиксін пайдаланады
- **Scalar құрал нәтижелері** — енді AI-ға қайтарылмас бұрын дұрыс оралады
- **Пайдалану статистикасы** — енді ескі нұсқалардан жаңарту кезінде бұрынғы қабілет кілті пішімін дұрыс өңдейді
