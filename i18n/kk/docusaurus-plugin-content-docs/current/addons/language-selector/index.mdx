# Тіл таңдағышы

Multisite Ultimate: тіркелудегі тіл таңдағышы

Тіркелу ағынына тіл таңдау өрісін қосыңыз.

## Орнату

1. 'ultimate-multisite-language-selector' файлын '/wp-content/plugins/' директориясына жүктеңіз
2. Плагинді WordPress Network Admin ішіндегі 'Плагиндер' мәзірі арқылы белсендіріңіз

## Пайдалану нұсқаулығы

Осы тегін қосымша арқылы пайдаланушыларыңызға тіркелу ағынында және кіру бетінде қай тілді көргісі келетінін таңдауға мүмкіндік бере аласыз.

[Лицензияңызды белсендіргеннен](../../user-guide/getting-started/installing-ultimate-multisite.md) кейін лицензияңыз қолжеткізуге мүмкіндік беретін Ultimate Multisite қосымшаларын орната аласыз.

Ultimate Multisite баптаулар бетіне өтіп, сол жақтағы мәзірден _Қосымшалар_ белгісін басыңыз.

![Қосымшалар беті](/img/admin/addons-page.png)

_Language Selector орнатуды бастау үшін Қосымшалар бетіне өтіңіз_

Сол жерден біз ұсынатын барлық қосымшаларды табасыз.

Тегін қосымшалар тіркелген жазылым жоспарына қарамастан барлық клиенттерімізге қолжетімді.

Қосымшаның нобайының астындағы _Қазір орнату!_ батырмасын басыңыз. Содан кейін оны Плагиндер бетінде белсендіріңіз.

![Орнату батырмасы бар Қосымшалар беті](/img/admin/addons-page.png)

_Қосымшаны орнату үшін батырманы басыңыз_

<!-- Скриншот қолжетімсіз: WP Плагиндер беті Language Selector қосымшасына арналған Желіде белсендіру батырмасын көрсетеді -->

_Оны Плагиндер бетінде белсендіріңіз_

Енді Ultimate Multisite-тің Баптаулар мәзіріне өтіп, _Желі баптаулары_ қойындысына кіріңіз. Сол жерде, беттің төменгі жағында, _Тілдер_ деп аталатын жаңа бөлімді табасыз.

![Баптаулар беті](/img/config/settings-general.png)

_Мүлде жаңа бөлім пайда болды!_

Барлық тілдерді опциялар ретінде көрсету үшін оны бос қалдыра аласыз немесе қолжетімді еткіңіз келетіндерін таңдай аласыз. Тілдер — WordPress орнатылымыңызда орнатылған тілдер.

<!-- Скриншот қолжетімсіз: WP орнатылымындағы қолжетімді тілдерді көрсететін тіл таңдау ашылмалы тізімі -->

_Осы WP орнатылымында қолжетімді тілдер тізімі_

Енгізген өзгерістеріңізді сақтағаннан кейін таңдаған тілдеріңіз бар таңдау мәзірі Кіру бетінде де, Тіркелу ағынында да көрсетіледі.

![Кіру және тіркелу баптаулары](/img/config/settings-login-registration.png)

_Кіру бетіндегі тіл таңдағышы_

![Тіркелу беті](/img/config/registration-page.png)

_Тіркелу ағынындағы тіл таңдағышы_

## URL арқылы тілді орнату

Сондай-ақ URL-ге locale параметрін беру арқылы беттің тілін өзгерте аласыз. Мысалы, итальян тіліндегі тіркелу бетіне қайта бағыттағыңыз келсе, URL-ге ?locale=it_IT қосыңыз.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Бұл жұмыс істеуі үшін тіл WordPress желіңізде орнатылған болуы керек.
