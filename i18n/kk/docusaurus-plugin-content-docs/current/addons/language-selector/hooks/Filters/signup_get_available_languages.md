# Сүзгі: signup_get_available_languages

Front-end сайт тіркелімдері үшін қолжетімді тілдер тізімін сүзеді.

Бұл hook-қа бос массив беру signup пішініндегі баптаудың шығарылуын өшіреді және сайтты жасау кезінде әдепкі тіл пайдаланылады. Бұрын орнатылмаған тілдер алынып тасталады.

## Параметрлер {#parameters}

| Атауы | Түрі | Сипаттамасы |
|------|------|-------------|
| $available_languages | `array` | Қолжетімді тілдер. |

- 4.4.0
