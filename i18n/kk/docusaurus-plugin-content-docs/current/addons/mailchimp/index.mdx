# Mailchimp {#mailchimp}

Multisite Ultimate: Mailchimp интеграциясы

Multisite Ultimate желіңізді Mailchimp-пен интеграциялаңыз. Белгілі бір жоспарларды Mailchimp тізімдерімен байланыстырып, тіркелу кезінде жазылушыларды сол тізімдерге автоматты түрде қоса аласыз.

## Орнату {#installation}

1. 'ultimate-multisite-mailchimp' файлын '/wp-content/plugins/' каталогына жүктеңіз
2. Плагинді WordPress Network Admin ішіндегі 'Plugins' мәзірі арқылы белсендіріңіз

## Пайдалану нұсқаулығы {#usage-guide}

Жазылушылардың электрондық пошталарын MailChimp тізімдеріңізбен синхрондау біздің Ultimate Multisite: MailChimp Integration қондырмамыз арқылы өте оңай. Бұл нұсқаулық барлығын баптауға көмектесуге арналған!

## API кілтін алу {#getting-an-api-key}

Қолжетімді әртүрлі баптауларға кіріспес бұрын, бізге MailChimp API кілті қажет. Бұл Ultimate Multisite-ке MailChimp API-мен байланысып, тізімдерді, топтарды алуға және пайдаланушылар тіркелген кезде электрондық пошта мекенжайларын сол тізімдер мен топтарға қосуға мүмкіндік береді.

Өзіңіздің MailChimp API кілтіңізді алу үшін мына нұсқаулықты орындаңыз: [API кілттері туралы | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

## Интеграция режимі {#integration-mode}

MailChimp Integration қондырмасы екі “интеграция режимін” ұсынады, яғни электрондық пошта мекенжайларын синхрондау үшін қондырманы пайдаланудың екі түрлі жолы бар.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_Көбірек икемділік үшін екі интеграция режимі!_

### Режим 1: Бірнеше тізім {#mode-1-multiple-lists}

Әдепкі режим **Бірнеше тізімді** пайдаланады. Бұл режим әр жоспар үшін бірнеше MailChimp тізімін таңдауға мүмкіндік береді. Егер пайдаланушыларыңызды әртүрлі тізімдерге бөлсеңіз, бұл ең жақсы нұсқа.

Мысалы, MailChimp Account-ыңызда 3 түрлі тізім бар делік: **Жаңалықтар тарату** , **Жоспар A** , және **Жоспар B**. Жоспарына қарамастан барлық пайдаланушылардың Жаңалықтар тарату тізіміне де қосылғанын қалайсыз, сонда олар жаңартуларыңызды жіберіп алмайды.

Оған қол жеткізу үшін Multiple Lists режимін пайдаланып, Plan A MailChimp Settings ішіне **Жаңалықтар тарату** және **Жоспар A** тізімдерінің екеуін де, ал Plan B MailChimp Settings ішіне **Жаңалықтар тарату** және **Жоспар B** тізімдерін қоса аласыз.

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_Жаңа Account-тар және тоқтатылған Account-тар үшін әдепкі тізімдерді таңдаңыз_

### Режим 2: Бір тізім + бірнеше топ {#mode-2-single-list--multiple-groups}

MailChimp тізіміңізді пайдаланушылардың **Топтар** деп аталатын шағын топтарына бөлуге мүмкіндік береді. Бұл көбірек бақылау береді, себебі MailChimp-та бір науқанның бірнеше көшірмесін жасамай-ақ, жазылушыларыңыздың нақты бір ішкі жиынына науқан жібере аласыз (бұл кейін сол науқанның метрикаларын талдауды қиындатады).

Мысалы, сізде **Менің желі жазылушыларым,** деп аталатын бір тізім болып, оның ішінде **Жоспар A** және **Жоспар B** үшін топтар болуы мүмкін.

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_“Бір тізім + бірнеше топ” интеграция режимін пайдалансаңыз, әдепкі тізімдер мен топтарды таңдаңыз_

## Әдепкі тізімдер мен топтар {#default-lists-and-groups}

Нақты қолдану жағдайыңызға қай интеграция режимі көбірек сәйкес келетінін таңдағаннан кейін, әдепкі тізімдерді (және режимге байланысты топтарды) таңдау уақыты келеді. Әдепкі тізім белгілі бір жоспар үшін нақты тізім таңдалмаған кезде пайдаланылады.

Сондай-ақ тоқтатылған Account-тар үшін әдепкі тізімді таңдай аласыз. Пайдаланушы өз жазылымын тоқтатып, Account-ын желіңізден алып тастаған сайын, оның электрондық пошта мекенжайы тоқтатылған Account-тар үшін орнатылған тізімге жылжытылады.

## Қос жазылымды қосу {#enabling-double-subscription}

Сол баптаулар бетінде **Қос жазылымды** қоса аласыз. Бұл опция пайдаланушыны таңдалған әдепкі тізімге ДЕ, сол пайдаланушының жоспары үшін таңдалған тізімге ДЕ қосады. Егер бұл опция қосылмаған болса, пайдаланушы тек пайдаланушының жоспары үшін таңдалған тізімге қосылады. Егер сол нақты жоспар үшін ештеңе орнатылмаса, онда пайдаланушы таңдалған әдепкі тізімге қосылады.

<!-- Screenshot unavailable: MailChimp double subscription toggle setting -->

_Тіркелу кезінде пайдаланушыларыңызды әдепкі тізімге де, жоспардың тізіміне де жаздыра аласыз_

## Әр жоспар үшін тізімдер мен топтарды таңдау {#selecting-lists-and-groups-for-each-of-the-plans}

**Жоспарлар** бөліміне өтіп, өңдегіңіз келетін жоспарды таңдаңыз. **Advanced Options** мета-боксында MailChimp деп аталатын жаңа қойынды қолжетімді болатынын ескеріңіз.

Сол қойындыда сіз сол жоспар үшін бірнеше тізімді немесе бір тізім мен бірнеше топты (интеграция режиміне байланысты) таңдай аласыз.

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_Әр жоспарыңыз үшін пайдаланғыңыз келетін тізімдер мен топтарды таңдаңыз_

Жаңа пайдаланушы сол жоспарға тіркелгенде, оның электрондық пошта мекенжайы таңдалған тізімдер мен топтарға қосылады!
