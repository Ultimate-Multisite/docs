# Gratis AI Agent {#gratis-ai-agent}

Gratis AI Agent — табиғи тілдегі нұсқаулар арқылы толық WordPress сайттарын жоспарлай, құра және басқара алатын автономды WordPress AI көмекшісі. Ол агент шақыра алатын **мүмкіндіктердің** құрылымдалған жиынын — дискретті, тексерілетін әрекеттерді — ұсынады; олар мазмұнды басқарудан бастап дизайн жүйесін баптауға дейінгі барлық нәрсені қамтиды. 1.9.0 нұсқасы мазмұн жасау мүмкіндіктерін, клиент жағындағы скриншоттар арқылы бетті визуалды шолуды, бес кіріктірілген агент профилін және қолжетімділікті бақылау мен брендингке арналған функция жалаушаларын қосады.

## Негізгі мүмкіндіктер {#key-features}

- **Мүмкіндіктер жүйесі** — агент сіздің WordPress орнатылымыңызда шақыра алатын әрекеттердің модульдік, кеңейтілетін каталогы
- **Мазмұнды басқару** — жазбалар мен беттер жасау (бет үлгілерін тағайындауды қоса), бір шақыруда бірнеше жазбаны топтап жасау, таңдаулы суреттерді орнату және байланыс формаларын құру
- **Визуалды шолу** — тірі беттердің скриншоттарын түсіру, алдыңғы/кейінгі күйлерді салыстыру және орналасу, типография, түс пен қолжетімділік бойынша AI жасаған дизайн шолуын алу
- **Бес кіріктірілген агент** — Мазмұн жазушысы, Сайт құрастырушысы, Дизайн студиясы, Plugin менеджері және Қолдау көмекшісі; чат тақырыбындағы агент таңдау құралы арқылы ауыстыруға болады
- **Функция жалаушалары** — рөл шектеулері, white-label атауы және виджетті баптау үшін Параметрлер → Функция жалаушалары бөліміндегі қолжетімділікті бақылау және брендинг ауыстырғыштары
- **Пайдаланушы жазба түрлерін басқару** — қайта іске қосулар арасында сақталуын қамтамасыз ете отырып, пайдаланушы жазба түрлерін тіркеу, тізімдеу және жою
- **Пайдаланушы таксономияларын басқару** — сақталуын қамтамасыз ете отырып, пайдаланушы таксономияларын тіркеу, тізімдеу және жою
- **Дизайн жүйесі мүмкіндіктері** — пайдаланушы CSS енгізу, іріктелген блок үлгілерін басқару, сайт логотипін орнату және theme.json пресеттерін қолдану
- **Глобал стильдерді басқару** — түс палитралары, типография, аралықтар және тағы басқалар үшін theme.json мәндерін оқу және жазу
- **Навигация мәзірлерін басқару** — навигация мәзірлерін және олардың элементтерін бағдарламалық түрде жасау және жаңарту
- **Параметрлерді басқару** — маңызды параметрлерді қорғауға арналған кіріктірілген қауіпсіз блоктізіммен WordPress параметрлерін оқу және жазу
- **Орнатылатын мүмкіндіктер тізілімі** — негізгі plugin файлдарын өзгертпей, қосымша мүмкіндіктер пакеттерін табу және белсендіру
- **Сайт құрастырушы оркестрациясы v2** — plugin іздеу, прогресті қадағалау және қателерден қалпына келу бар көпқадамды сайт құру жоспарлары
- **WP-CLI benchmark командасы** — CI және модельді бағалау үшін агент қабілеттерінің benchmark жиынын пәрмен жолынан іске қосу

## Талаптар {#requirements}

- WordPress 6.0 немесе одан жоғары
- PHP 8.1 немесе одан жоғары
- WordPress Multisite (желі бойынша белсендірілген) немесе бір сайттық орнатылым
- Белсенді AI провайдер қосқыш plugin-і (Anthropic, OpenAI немесе үйлесімді)

## Орнату {#installation}

1. Plugin файлдарын `/wp-content/plugins/gratis-ai-agent` ішіне жүктеңіз
2. Plugin-ді WordPress ішіндегі **Plugins** экраны арқылы белсендіріңіз
3. **Gratis AI Agent → Параметрлер** бөліміне өтіп, AI провайдеріңізді қосыңыз
4. (Қосымша) Қосымша мүмкіндіктер пакеттерін орнату үшін **Мүмкіндіктер тізілімін** шолыңыз

## Жылдам бастау {#quick-start}

Белсендіріп, AI провайдеріне қосылғаннан кейін WordPress әкімші бүйірлік тақтасындағы **Gratis AI Agent** панелін ашыңыз. Табиғи тілде нұсқау теріңіз, сонда агент қолжетімді мүмкіндіктерін пайдаланып қажетті қадамдарды жоспарлап, орындайды.

**Мысал нұсқаулар:**

```
Create a "Portfolio" custom post type with a "Project Category" taxonomy.
```

```
Inject a dark-mode colour palette into the theme and set the primary colour to #1a1a2e.
```

```
Build a restaurant website with an online menu, booking form, and contact page.
```

## Мүмкіндіктерге шолу {#abilities-overview}

Мүмкіндіктер — агент орындай алатын атомдық әрекеттер. Әр мүмкіндік — агент орындау уақытында оқитын схема (параметрлер және қайтарылатын түрлер) ұсынатын тіркелген PHP класы. Мүмкіндіктер функционалдық салаларға топтастырылған:

| Сала | Мүмкіндіктер |
|---|---|
| **Мазмұнды басқару** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **Визуалды шолу** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **Пайдаланушы жазба түрлері** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **Пайдаланушы таксономиялары** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **Дизайн жүйесі** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **Глобал стильдер** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **Навигация мәзірлері** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **Параметрлерді басқару** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **Орнатылатын мүмкіндіктер** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **Сайт құрастырушы** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

Толық параметр құжаттамасы және қолдану мысалдары үшін [Мүмкіндіктер анықтамасын](./abilities) қараңыз.

## WP-CLI {#wp-cli}

Gratis AI Agent пәрмен жолынан benchmark іске қосу және мүмкіндіктерді басқару үшін `wp gratis-ai-agent` командалар тобымен бірге жеткізіледі. Толық командалар тізімі үшін [WP-CLI анықтамасын](./wp-cli) қараңыз.

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## Конфигурация {#configuration}

Барлық параметрлер WordPress әкімшісіндегі **Gratis AI Agent → Параметрлер** астында орналасқан:

| Параметр | Сипаттама |
|---|---|
| **AI провайдері** | Белсенді AI провайдер қосқышын таңдаңыз және конфигурациялаңыз |
| **Модель** | Агент сессиялары үшін қолданылатын тіл моделін таңдаңыз |
| **Максималды токендер** | Әр сұрауға арналған токен бюджеті (әдепкі 4096) |
| **Мүмкіндік тайм-ауты** | Жеке мүмкіндіктің орындала алатын ең көп секунд саны (әдепкі 30) |
| **Жөндеу журналдауы** | Мүмкіндік шақыруларын және агент пайымдауын WordPress жөндеу журналына жазу |

## Параметрлерді басқарудың қауіпсіз блоктізімі {#options-management-safety-blocklist}

Опцияларды басқару қабілеті агенттің құпия WordPress опцияларын оқуына немесе жазуына жол бермейтін кірістірілген бұғаттау тізімін қамтиды. Әдепкі бұғаттау тізімі мыналарды қамтиды:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (URL опциялары — бұлар үшін арнайы көшіру құралын пайдаланыңыз)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (құпия кілттер)
- `active_plugins`, `template` (плагин/тақырып белсендіру — арнайы қабілеттерді пайдаланыңыз)
- Орнатылған плагиндер тіркеген төлем шлюзі деректері және API кілттері

Қосымша жазбаларды `gratis_ai_agent_options_blocklist` фильтрі арқылы қосуға болады.

## Ілгектер және фильтрлер {#hooks-and-filters}

### Әрекеттер {#actions}

- `gratis_ai_agent_ability_registered` — қабілет тіркелгеннен кейін іске қосылады, қабілет класының атауын алады
- `gratis_ai_agent_plan_started` — сайт құрушы жоспары орындала бастағанда іске қосылады
- `gratis_ai_agent_plan_completed` — жоспар сәтті аяқталғанда іске қосылады
- `gratis_ai_agent_plan_failed` — жоспар түзетілмейтін қатеге тап болғанда іске қосылады

### Фильтрлер {#filters}

- `gratis_ai_agent_abilities` — агент оны жүктемес бұрын тіркелген қабілеттер тізімін өзгерту
- `gratis_ai_agent_options_blocklist` — опциялардың қауіпсіздік бұғаттау тізімін кеңейту немесе қысқарту
- `gratis_ai_agent_benchmark_questions` — бағалау жинағына реттелетін эталон сұрақтарын қосу
- `gratis_ai_agent_plan_plugins` — сайт жоспарын жасау кезінде плагиндерді анықтау тізімін өзгерту
