# Site Exporter өзгерістер журналы

Нұсқа: 1.0.6 - 2026-05-11 күні шығарылды
* Жаңа: Site экспорттары енді өзін-өзі жүктейтін index.php файлын қамтиды, сондықтан ZIP файлын жаңа хостқа бөлек plugin орнатусыз орнатуға болады
* Жаңа: Желі экспорты әкімшілерге Site Export әкімші бетінен барлық қосалқы сайттарды бір архивке экспорттауға мүмкіндік береді

Нұсқа: 1.0.5 - 2026-05-05 күні шығарылды
* Түзету: get_config сияқты мокталмаған әдістерді өңдеу үшін WP_CLI polyfill stub ішіне __callStatic қосылды, бұл тест bootstrap сәтсіздіктерінің алдын алады

Нұсқа: 1.0.4 - 2025-11-25 күні шығарылды

* Түзету: Ultimate Multisite 2.4 нұсқасымен үйлесімділік.

Нұсқа: 1.0.2 - 2025-09-28 күні шығарылды

* Префиксті ultimate-multisite деп өзгерту; text domain жаңарту; нұсқаны көтеру.

### Нұсқа 1.0.1 - 2023-08-09 күні шығарылды

* Жақсартылды: Енді импорттаушы файлдың дұрыс файл жолы алынуын қамтамасыз етеді.
* Түзетілді: Импорт кезінде қателердің алдын алу үшін object cache өшірілді.
* Түзетілді: Импорттаушы plugin жүктеп алу сілтемесінің жұмыс істемеу мәселесі түзетілді.

### Нұсқа 1.0.0 - 2022-12-23 күні шығарылды

* Түзетілді: Экспорт формасының қалқымалы терезесі жүктелмеді;
* Түзетілді: Javascript файлдары экспорт бетінде жүктелмеді;
* Ішкі: Жаңа құрастыру процесі;

### Нұсқа 1.0.0-beta.4 - 2022-01-13 күні шығарылды

* Ішкі: Hooks және filters генераторы қосылды;
* Ішкі: Әзірлеушінің жұмыс ыңғайлылығы үшін WP Ultimo stubs қосылды;
