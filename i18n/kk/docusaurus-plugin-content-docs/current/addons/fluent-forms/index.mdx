# Fluent Forms

Бұл қосымша Multisite Ultimate ішіндегі өнімдерде, мүшеліктерде немесе сайттарда Fluent Forms үшін шектеулер қосуға мүмкіндік береді.
Fluent Forms пайдаланушы жазба түрлерінің орнына пайдаланушы кестелерін қолданады, сондықтан әдеттегі шектеу опциялары Fluent Forms-пен жұмыс істемейді.

## Мүмкіндіктер

- Fluent Forms немесе Conversational Fluent Forms үшін шектеулер қосу
- Клиенттер шекке жеткенде, жоспарын жаңартуды ұсынатын сыпайы хабарлама алады
