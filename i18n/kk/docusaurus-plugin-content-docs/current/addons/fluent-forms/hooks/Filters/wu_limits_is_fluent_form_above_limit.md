# Filter: wu_limits_is_fluent_form_above_limit

Берілген форма түріне осы plan-да рұқсат етілгенін тексереді. plugin әзірлеушілеріне қайтарылатын мәнді сүзуге мүмкіндік береді

## Параметрлер {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $value | `bool` | Форма түрі шектен жоғары ма, жоқ па |
| $form_type | `string` | Тексеріліп жатқан форма түрі |
| $form_count | `int` | Ағымдағы форма саны |
| $quota | `int` | Рұқсат етілген квота |
