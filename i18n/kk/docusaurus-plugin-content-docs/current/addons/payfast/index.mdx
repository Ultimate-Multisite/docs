# PayFast

Ultimate Multisite: Payfast төлем шлюзі

Оңтүстік Африкадағы жылдам әрі қауіпсіз онлайн төлемдер шлюзі Payfast-ты Ultimate Multisite төлем опциясы ретінде қосады.

## Орнату {#installation}

1. 'ultimate-multisite-payfast' файлын '/wp-content/plugins/' каталогына жүктеп салыңыз
2. Плагинді WordPress ішіндегі 'Плагиндер' мәзірі арқылы белсендіріңіз
3. Плагинді баптау үшін қадамдық шеберді орындаңыз

## Пайдалану нұсқаулығы {#usage-guide}

PayFast — Оңтүстік Африкадағы бизнеске арналған жетекші онлайн төлем өңдеушісі. Ол транзакцияларды үздіксіз әрі қауіпсіз ету арқылы онлайн сатушылардың өсуіне көмектесуге арналған. Ultimate Multisite пайдаланушысының сұрауы бойынша біз PayFast-пен интеграция жасауға мүмкіндік беретін қондырма жасауды шештік. Бұл мақалада Ultimate Multisite желіңізде төлем қабылдау үшін PayFast-ты төлем шлюзі ретінде қалай баптау керектігі көрсетіледі.

Бірінші қадам — қондырманы орнату.

# Payfast қондырмасын қалай орнату керек:

Қондырмалар бетіне өтіп, Ultimate Multisite: Payfast Gateway қондырмасын таңдаңыз.

![Қондырмалар беті](/img/admin/addons-page.png)

Қондырма терезесі қалқып шығады. Install Now түймесін басыңыз.

![Орнату опциясы бар қондырмалар беті](/img/admin/addons-page.png)

Орнату аяқталғаннан кейін сіз плагиндер бетіне қайта бағытталасыз. Мұнда Network Activate түймесін бассаңыз болды, _Payfast_ қондырмасы желіңізде белсендіріледі.

# PayFast баптау

Қондырманы орнатқаннан кейін енді Ultimate Multisite параметрлеріне өтіп, PayFast баптауды бастай аласыз.

**Ultimate Multisite параметрлері** бетінде "Төлемдер" қойындысын таба аласыз. Осы қойындының астында бұл төлем шлюзін қосуға немесе өшіруге арналған ауыстырып-қосқыш түймесімен бірге "**Payfast** " көрсетіледі.

![Төлем шлюзі параметрлері](/img/config/settings-payment-gateways.png)Жалғастырып, оны қосу үшін ауыстырып-қосқыш түймесін басыңыз. Содан кейін ол өрісті кеңейтіп, толтыру қажет әртүрлі өрістерді көрсетеді, мысалы **Саудагер ID, саудагер кілті және құпия фраза.** Бұл ақпаратты **Payfast** Account-ыңыздан ала аласыз

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Келесі қадам — қажет ақпаратты алу үшін PayFast Account-ыңызға кіру.

Кіргеннен кейін **Payfast** Account Dashboard-ынан қажетті тіркелгі деректерін бірден көре аласыз.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->Тіркелгі деректерін жай ғана көшіріп, Ultimate Multisite параметрлеріндегі мәліметтерді толтыра аласыз. Содан кейін өзгерістерді сақтау үшін **"Параметрлерді сақтау** " түймесін басыңыз.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Міне, PayFast төлем интеграцияңыз аяқталды!
