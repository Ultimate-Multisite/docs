# Metered Plans өзгерістер журналы {#metered-plans-changelog}

Нұсқа 1.1.0 - 2026-05-05 күні шығарылды
- Жаңа: мультисайт қосалқы сайттары үшін AI token төлемдері — әр token үшін бапталатын мөлшерлемелермен клиент сайттары бойынша AI token қолданылуын қадағалау және есептеу
- Жаңа: Connector қолдануын мәжбүрлеу динамикалық лимитті анықтау және тікелей жазу арқылы қайта құрылды, барлық connector бойынша нақты уақыттағы дәлдікті қамтамасыз етеді
- Түзету: Database кесте жаңартулары енді BerlinDB схема анықтамаларымен дұрыс сәйкестендірілді, жаңа орнатулардағы жаңарту сәтсіздіктерінің алдын алады
- Түзету: Database жаңарту callbacks дұрыс пішімге түрлендірілді, үнсіз жаңарту сәтсіздіктерін шешеді
- Түзету: Бөлшек мәндер енді AI Usage Overage Markup енгізу өрісінде қабылданады
- Түзету: plugin жүктелгенде fatal errors және қосарлы баптандыру мәселелері шешілді
- Жақсартылды: әзірлеуші орталары алғашқы іске қосқанда өздігінен бапталуы үшін check-env npm скрипті қосылды

### 1.0.3 {#103}

* Қолданылуды қадағалау өнімділігі арттырылды
