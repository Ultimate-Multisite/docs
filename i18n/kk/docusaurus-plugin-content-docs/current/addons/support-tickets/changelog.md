# Қолдау тикеттерінің өзгерістер журналы

### 1.0.4 - 2026-05-05 {#104---2026-05-05}
* Жақсартылды: vendor/ каталогы Git бақылауынан алынып тасталды (.gitignore арқылы бұрыннан қамтылған), репозиторий өлшемі азайтылды
* Жақсартылды: WordPress 7.0 нұсқасына дейін тексерілді

### 1.0.3 - 2026-05-01 {#103---2026-05-01}
* Түзетілді: null мәндерін null бола алатын тикет моделі сеттерлерінде қабылдау
* Түзетілді: Дұрыс multisite қолдауы үшін барлық тикеттерді бүкіл желіге ортақ жаһандық кестелерде сақтау
* Түзетілді: Қызметкер өрістерін жасыру және клиенттің жаңа тикет пішініндегі бос бетке қайта бағыттауды түзету
* Түзетілді: Анықталмаған add_meta() шақырулары метадеректерді сақтау үшін дұрыс update_meta() шақыруларымен ауыстырылды
* Түзетілді: Тіркелмеген мүмкіндік тексеруі дұрыс wu_view_all_support_tickets мәнімен ауыстырылды
* Түзетілді: Тикет күйі, тағайындау және жылдам өңдеу әрекеттері үшін жетіспейтін AJAX өңдегіштері қосылды
* Түзетілді: Тикет көріністерінде қызметкер жауабын анықтауға арналған әдіс атауы түзетілді
* Түзетілді: Қайталанатын жауап өңдегіштері біріктіріліп, nonce әрекет атаулары сәйкестендірілді
* Түзетілді: [wu_submit_ticket] shortcode үшін жетіспейтін фронтенд көрінісі қосылды
* Түзетілді: Жетіспейтін user_id бағаны қосылып, Support_Ticket::get_user_id() әдісі түзетілді
* Түзетілді: Басымдық сүзгісінің опция тегіндегі артық қос тырнақша алынып тасталды
* Түзетілді: Супер әкімшілер үшін желі әкімшісінің тикеттерді басқару панелі қосылды
* Жақсартылды: Әкімші CSS бір сыртқы стильдер кестесіне біріктірілді
* Жақсартылды: Қосалқы сайт әкімші мәзірінен Settings ішкі мәзірі алынып тасталды
* Жақсартылды: Фронтенд ресурстары тек қолдау тикеттері беттерінде шартты түрде жүктеледі
* Жақсартылды: Bedrock түбірлік autoloader тәуелділіктерді бұрыннан жүктеген болса, plugin autoloader өткізіп жіберіледі

### 1.0.2 - 2025-12-11 {#102---2025-12-11}
* Қосылды: Әкімші және фронтенд үшін толық тикетті қарау функционалдығы
* Қосылды: Тикет жауабын жіберуге арналған AJAX өңдегіші
* Қосылды: Пішінді дұрыс өңдеумен тикетке жауап беру функционалдығын қолдау
* Қосылды: Әкімші бөлігінде тикет жіберілімдері мен жауаптар үшін дұрыс хабарлама көрсету
* Қосылды: Клиенттің өзі жіберген тикеттерді ағымдағы пайдаланушымен автоматты түрде байланыстыру
* Қосылды: Клиенттердің тикет иелігін қайта анықтауына жол бермейтін қауіпсіздік жақсартуы
* Қосылды: Жетіспейтін көмекші функциялар (wu_format_date, wu_user_can_view_ticket, т.б.)
* Қосылды: Файл тіркемелерін дұрыс жүктеп алу және өңдеу
* Қосылды: Тикет жауаптары мен күй өзгерістері үшін электрондық пошта хабарландыру жүйесі
* Түзетілді: Тикетке жауап беру пішіні енді қажетті тикет ID қамтиды
* Түзетілді: Әкімші тикет көріністеріндегі хабарламалардың көріну мәселелері
* Түзетілді: Қолдау тикеті функцияларындағы синтаксис қателері
* Жақсартылды: Жауаптар мен тіркемелер үшін дұрыс сұрау және схема кластары бар дерекқор құрылымы
* Жақсартылды: Тикет модельдері үшін атрибуттардың орнына нақты қасиеттерге көшу
