# Көпжелілік өзгерістер журналы {#multi-network-changelog}

### 1.3.0 {#130}
* Жаңа: желі үлгісін алдын ала қарау құралы — сатып алмас бұрын желі үлгілерін тікелей панель арқылы шолып, алдын ала қараңыз.
* Жаңа: төлем рәсімдеу кезінде желі үлгісін таңдау — клиенттер тіркелу кезіндегі төлем рәсімдеу ағымында желі үлгісін таңдай алады.
* Жаңа: желілер арасындағы magic link SSO — клиенттер magic-link аутентификациясы арқылы қосалқы желілерге кедергісіз қол жеткізе алады.
* Жаңа: желі медиасын клондау — желілерді үлгіден клондағанда сайт медиасы дұрыс көшіріледі.
* Түзету: желіні клондау енді multi-tenancy қосымша модулінсіз жұмыс істейді; клондаудан кейін super admin артықшылықтары қайта беріледі.
* Түзету: желі жасау енді WooCommerce және барлық төлем шлюздерімен үйлесімді.
* Түзету: порт нөмірі енді резервтік домен жасауда қамтылады.
* Түзету: барлық тақырыптарда дұрыс көрсетілуі үшін үлгі таңдағышының CSS-і enqueued стильдер кестесіне ауыстырылды.
* Түзету: сайт сұрауы аясында анықталмаған `network_id` кілтінен қорғау қосылды.
* Түзету: контексті ауыстырмас бұрын желі рұқсатын тексеру `wpdb` кесте қасиеті қателерінің алдын алады.
