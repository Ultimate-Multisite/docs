# Әрекет: switch_network {#action-switchnetwork}

Ағымдағы желі контексті ауыстырылған кезде іске қосылады.

## Параметрлер {#parameters}

| Атауы | Түрі | Сипаттамасы |
|------|------|-------------|
| $new_network_id | `int` | Ауыстырылып жатқан желінің ID-і. |
| $old_network_id | `int` | Бұрынғы ағымдағы желінің ID-і. |
