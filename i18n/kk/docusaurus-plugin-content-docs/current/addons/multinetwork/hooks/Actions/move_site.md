# Әрекет: move_site

Сайт жаңа желіге ауыстырылғаннан кейін іске қосылады.

## Параметрлер

| Name | Type | Description |
|------|------|-------------|
| $site_id | `int` | Ауыстырылған сайттың ID мәні. |
| $old_network_id | `int` | Сайттың бастапқы желісінің ID мәні. |
| $new_network_id | `int` | Сайт ауыстырылған желінің ID мәні. |
