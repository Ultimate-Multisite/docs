# Сүзгі: networks_pre_user_is_network_admin

Пайдаланушы әкімшісі болып табылатын желілерді сүзеді, үдерісті қысқа жолмен тоқтату үшін.

## Параметрлер

| Атауы | Түрі | Сипаттамасы |
|------|------|-------------|
| $ | `array\|bool\|null` | Желі ID-лерінің тізімі немесе false. null мәнінен басқа кез келген мән үдерісті қысқа жолмен тоқтатады. |
| $ | `int` | Желілер қайтарылуы тиіс пайдаланушы ID-і. |
