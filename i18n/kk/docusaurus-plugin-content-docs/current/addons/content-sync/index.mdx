# Мазмұнды синхрондау {#content-sync}

Multisite Ultimate: Мазмұнды синхрондау

Multisite Ultimate үшін бұл қосымша желі әкімшілеріне кез келген сайттан желідегі басқа сайттарға мазмұнды синхрондауға мүмкіндік береді. Ол жазбалар, беттер, теңшелетін жазба түрлері және Elementor үлгілерін қоса, барлық жазба түрлерін қолдайды. Қосымшада мазмұн мен мақсатты сайттарды таңдауға арналған көпқадамды шебер интерфейсі бар, топтамалық өңдеу және нақты уақыттағы орындалу барысы туралы кері байланыс қамтылған.

## Мүмкіндіктер {#features}

* Мазмұнды таңдау: Нобайлары және жазба түрі бойынша сүзгісі бар көрнекі карточкалар торы
* Барлық жазба түрлері: Жазбаларды, беттерді, өнімдерді, Elementor үлгілерін және кез келген теңшелетін жазба түрін синхрондау
* Сайтты таңдау: Мазмұн синхрондалатын сайттарды таңдаңыз, бастапқы сайтты алып тастау және еншілес сайттарды алдын ала таңдау мүмкіндігі бар
* Топтамалық өңдеу: Бір уақытта 10 сайтқа дейін синхрондайды және орындалу барысының жаңартуларын көрсетеді
* Медианы синхрондау: Жазба мазмұнынан мақсатты сайттарға медиа файлдарды (суреттер, бейнелер) автоматты түрде көшіреді және қайта сәйкестендіреді
* Таксономияны синхрондау: Санаттарды, тегтерді және теңшелетін таксономияларды сақтайды
* Elementor қолдауы: CSS қайта генерациялау және үлгі ID қайта сәйкестендіру арқылы Elementor үлгілерін толық қолдау
* Қайшылықтарды шешу: Бар мазмұнды ауыстыру немесе өткізіп жіберу опциясы
* Кэшті басқару: Синхрондаудан кейін мақсатты сайттардағы Elementor кэшін автоматты түрде тазартады

## Талаптар {#requirements}

* WordPress 5.3 немесе жоғары
* PHP 7.4 немесе жоғары
* Multisite Ultimate плагині (белсенді)
* Elementor плагині (міндетті емес, Elementor үлгілерін синхрондау үшін қажет)

## Орнату {#installation}

1. 'ultimate-multisite-content-sync' файлын '/wp-content/plugins/' каталогіне жүктеңіз
2. Плагинді WordPress желі әкімшісіндегі 'Плагиндер' мәзірі арқылы белсендіріңіз
3. Параметрлерді Ultimate Multisite → Параметрлер → Мазмұнды синхрондау бөлімінде теңшеңіз
4. Синхрондауды бастау үшін Ultimate Multisite → Мазмұнды синхрондау бөліміне өтіңіз
