# WooCommerce интеграциясының өзгерістер журналы {#woocommerce-integration-changelog}

Нұсқа 2.2.0 - 2026-07-01 күні шығарылды
* Жаңа: Ultimate Multisite салық сомалары енді checkout кезінде жеке WooCommerce алым жолдары ретінде көрсетіледі, бұл төлемге дейін салық қорытындыларын айқынырақ етеді.
* Жаңа: WooCommerce Subscriptions қайта әрекет ету терезесі кезінде сәтсіз жаңартудан кейін тоқтатуды қалайтын сайттар үшін қосылуы ерікті "Suspend Memberships Immediately on Failed Renewals" баптауы және `wu_woo_suspend_on_payment_failure` сүзгісі қосылды.
* Түзету: Сәтсіз немесе қалпына келтірілген жаңартулардан кейін Ultimate Multisite мүшеліктерімен синхроннан шығып қалуы мүмкін WooCommerce Subscription күйлері үйлестірілді.
* Түзету: Жоқ болған кезде WooCommerce дүкен валютасы Ultimate Multisite валюта тізіміне қосылды.
* Түзету: Жазылушыларды WooCommerce checkout бетіне қайта бағыттағанда клиенттің төлем деректері сақталды.
* Жақсартылды: Jetpack Autoloader 5-пен үйлесімділік қосылды.
* Жақсартылды: GitHub және marketplace zip файлдары кірістірілген staging каталогтары мен әзірлеу файлдарын қамтымауы үшін шығарылым пакетін жасау тазартылды.

Нұсқа 2.0.6 - 2026-01-16 күні шығарылды
* Жақсарту: Негізгі жазылымдарды addon ішіне қосу. Енді Woocommerce Subscriptinos кеңейтімін қажет етпейді.

Нұсқа 2.0.5 - 2026-01-09 күні шығарылды
* Жақсарту: Аудармаларды glotpress API-ден жүктеу.
* Түзету: Кейбір page builder-лердегі fatal error.
* түзету: Клиент негізгі сайттың мүшесі болған кезде шексіз қайта бағыттау.

Нұсқа 2.0.4 - 2025-11-14 күні шығарылды
* Қосылды: Көптеген қосымша тілдерге аудармалар.
* Өзгертілді: Атауы Ultimate Multisite: Woocommerce Integration болып өзгертілді.
* Қосылды: Woocommerce 10.2.1-пен үйлесімділік.
* Қосылды: Woocommerce Subscriptions 7.7.0-пен үйлесімділік.
* Түзету: PHP 8.4-пен үйлесімділік
* Түзету: WC Account беті болмаған кезде қайта бағыттау.

Нұсқа 2.0.3 - 2025-08-13 күні шығарылды
* Өзгертілді: Жаңа marketplace арқылы автоматты жаңартулар қосылды.

Нұсқа 2.0.2 - 2025-07-05 күні шығарылды
* Өзгертілді: Атауы Multisite Ultimate: Woocommerce Integration болып өзгертілді.
* Қосылды: Woocommerce 9.8.1-пен үйлесімділік.
* Қосылды: Woocommerce Subscriptions 7.3.0-пен үйлесімділік.
* Түзету: Клиенттің жазылымнан бас тартуы.
* Түзету: Checkout блогын пайдаланғанда fatal error.
* Жақсарту: Енді Woocommerce жоғары өнімділікті арнайы тапсырыс кестелерімен үйлесімді.
* Түзету: WooCommerce checkout кезінде бас тарту мүшелікті бәрібір жаңарта алатын.

Нұсқа 2.0.1 - 2023-08-09 күні шығарылды

* Қосылды: Woocommerce 7.9.0-пен үйлесімділік.
* Қосылды: Woocommerce Subscriptions 5.3.0-пен үйлесімділік.
* Қосылды: Мүшелік жаңартуларын қолдау.
* Қосылды: Woocommerce ішіндегі сынақ мерзімдері мен орнату алымдары туралы хабарламалар.
* Қосылды: Ultimate Multisite Woocommerce өнімдерін meta мәні арқылы сәйкестендіру.
* Қосылды: Барлық Ultimate Multisite қатысты Woocommerce өнімдерін белгілеу үшін бір реттік түзету енгізілді.
* Қосылды: Ultimate Multisite жасаған өнімдер Woocommerce тізімінен алынып тасталды.
* Жақсарту: Себетке қолдану үшін қайталанбайтын Woocommerce жеңілдігі жасалды.
* Жақсарту: Қайталанатын жеңілдік Woocommerce өніміне қалпына келтірілді.
* Жақсарту: Woocommerce өніміне қайталанатын жеңілдік белгісі қосылды.
* Жақсарту: Checkout кезінде өнім түрі қамтамасыз етілді.
* Түзету: Төмендету процесі кезінде мүшелік күйі сақталды.
* Түзету: Бас тарту процесі кезінде қателерді болдырмау үшін жазылымның бар-жоғы тексерілді.
* Түзету: Woocommerce subscriptions ішінде пайдалану үшін жазылымның басталу күні қосылды.
* Ішкі: Жаңа PHP 8.1 build процесі іске асырылды.

Нұсқа 2.0.0 - Толық қайта жазу.

* Қосылды: Gateway өзгергенде немесе мүшеліктен бас тартқанда woo жазылымын жоюға арналған бас тартуды өңдеу әдісі;
* Қосылды: Мүшеліктерді төмендету және жаңарту өңдегіші;
* Жақсарту: Account жаңартуға рұқсат беру үшін қосалқы сайттардағы клиент жаңарту формасында woocommerce тәуелділіктерін жүктеу;
* Жақсарту: Бар болмаса, Woocommerce cart-ты дұрыс жүктеу;
* Жақсарту: Checkout өңдегенде негізгі сайт кестелерінде екенімізді қамтамасыз ету;
* Жақсарту: Ultimo жаңарту тапсырысын соңғы төлемнен емес, Woocommerce subscription тапсырысының мәніне негіздеу;
* Түзету: WU Membership түймесінің сілтемесі;
* Түзету: Woocommerce subscriptions жаңартуы төленген кезде Ultimo тапсырысын төленген деп белгілеу;
* Build: MPB-ны builder ретінде қосу;
