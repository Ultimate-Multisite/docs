# Ultimate AI Connector for WebLLM өзгерістер журналы {#ultimate-ai-connector-for-webllm-changelog}

## 1.1.0 нұсқасы — 2026-04-09 күні шығарылды {#version-110--released-on-2026-04-09}

* Жаңа: admin-bar күй көрсеткіші бар қалқымалы чат виджеті — жүйеге кірген кез келген пайдаланушы браузер жағындағы LLM-ге тікелей алдыңғы бөліктен сұрау жібере алады.
* Жаңа: SharedWorker орындалу ортасы — MLC қозғалтқышы енді SharedWorker ішінде жұмыс істейді, сондықтан бірнеше қойынды бір GPU сеансын оған таласудың орнына ортақ пайдаланады.
* Жаңа: apiFetch middleware ұстап алғышы — AI Client SDK үлгісіне сәйкес келетін WordPress REST сұраулары жергілікті WebLLM брокеріне мөлдір түрде бағытталады, loopback HTTP айналма сұрауы қажет емес.
* Жаңа: чат виджетін қосып-өшіруге және auto-prompt әрекетін баптауға арналған Connector панеліндегі виджет параметрлерінің UI-ы.
* Түзету: модель салмақтарын жүктеулер әдепкі Cache API жолын бұзатын HuggingFace xet CDN қайта бағыттауларынан кейін де сақталуы үшін IndexedDB кеш backend-ін мәжбүрлеп қолдану.
* Түзету: embedding модельдері үшін context_window KV-cache қайта анықтауын өткізіп жіберу (оларда decoder жоқ және бұл қайта анықтау орындалу кезінде қате тудырды).
* Түзету: worker қойындысы жүктелмей тұрып /webllm/v1/models ішінде cold-start candidate моделін жариялау, сондықтан SDK тұтынушылары модельді бірден көреді.
* Түзету: модель бапталған кезде AI Experiments мүмкіндігі браузер қозғалтқышына бағытталуы үшін WebLLM-ді wpai_preferred_text_models сүзгісіне қосу.
* Жақсартылды: түпкілікті тестілеу кезінде анықталған cache-busting, мазмұнды қалыпқа келтіру және аппараттық анықтамаларға қатысты түзетулер.
