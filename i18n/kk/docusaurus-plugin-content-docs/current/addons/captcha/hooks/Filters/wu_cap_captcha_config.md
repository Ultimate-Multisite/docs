# Сүзгі: wu_cap_captcha_config

Cap Captcha конфигурациясын сүзеді.

Әзірлеушілерге Cap Captcha параметрлерін баптауға мүмкіндік береді.

## Параметрлер

| Атауы | Түрі | Сипаттамасы |
|------|------|-------------|
| $config | `array` | Cap конфигурация массиві. |
| $level | `string` | Қауіпсіздік деңгейі (fast, medium, max). |
