# Үйлесімді Endpoint-терге арналған Ultimate AI Connector өзгерістер журналы

## 2.0.0 нұсқасы — 2026-04-24 күні шығарылды

* Жаңа: Көп провайдерлік қолдау — бірнеше AI endpoint-ті баптап, сұрауларды провайдерлер арасында автоматты fallback арқылы бағыттаңыз.
* Түзету: Дұрыс провайдер ID-лерімен, тіркеу URL-дарымен және әр провайдерге арналған HTTP filter scoping-пен көп провайдерлік SDK интеграциясы.
* Түзету: Жаңа провайдер картасы енді қосылғанда автоматты түрде ашылады; plugin жаңартылғанда script кэшін айналып өту.
* Түзету: `eval()` үшін динамикалық провайдер класының namespace-і және жаңадан қосылған провайдерлер үшін автоматты ашылу әрекеті.
* Түзету: WordPress 6.9+ үйлесімділігі үшін тұрақты Card/CardBody/CardHeader/CardDivider компоненттерін пайдаланады (енді эксперименттік емес).
* Түзету: Провайдерді сүйреп қайта реттеу үшін анықталмаған DragHandle unicode grip белгішесіне ауыстырылды.
* Түзету: Әр бет жүктелген сайын іске қосылатын бұғаттаушы HTTP сұрауы жойылды.
* Жақсартылды: GitHub Actions жұмыс үрдістері Node.js 24 нұсқасына жаңартылды.
