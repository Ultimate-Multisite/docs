# Material WP {#material-wp}

WordPress әкімшілік интерфейсі ұзақ уақыт бойы өзгеріссіз қалып келеді. Бірақ әкімшілік интерфейстің Material Design нұсқасы қалай көрінетінін ойлап көрдіңіз бе? 

Material WP — сізге дәл сондай көрініс пен сезім беру үшін әзірленген әкімшілік тақырып, ол клиенттеріңізге көбірек бейімделген Dashboard ұсынуға көмектеседі.

## Мүмкіндіктері {#features}

- Таңдамалы CSS қолдауы
- Оңай стильдеу параметрлері
- Мәзір ретін өзгерту
- Оңай импорт/экспорт

## Құжаттама {#documentation}

**Material WP арқылы әкімшілік панеліңізді баптау**

## Орнату {#installation}

1. 'material-wp' файлын '/wp-content/plugins/' каталогына жүктеңіз
2. Плагинді WordPress ішіндегі 'Плагиндер' мәзірі арқылы белсендіріңіз

## Пайдалану нұсқаулығы {#usage-guide}

WordPress әкімшілік интерфейсі ұзақ уақыт бойы өзгеріссіз қалып келеді. Бірақ әкімшілік интерфейстің Material Design нұсқасы қалай көрінетінін ойлап көрдіңіз бе?

Material WP — сізге дәл сондай көрініс пен сезім беру үшін әзірленген әкімшілік тақырып, ол клиенттеріңізге көбірек бейімделген Dashboard ұсынуға көмектеседі. Material WP сондай-ақ Ultimate Multisite премиум плагиндер өзегінің бөлігі болып табылады.

Оны **орнатқаннан кейін**, _Плагиндер_ бетіне өтіп, оны Network желіңізде белсендіру үшін басыңыз.
