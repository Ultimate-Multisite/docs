# Ultimate Multisite фрагменттерін GitHub репозиторийден қолдану туралы {#how-to-use-ultimate-multisite-snippets-on-our-github-repository}

Ultimate Multisite пайдаланушылары жиі сұраған, Google Analytics скриптін тіркелу беттеріне қосу немесе администратор панелінен мета боксты жабу сияқты шағын функционалдық құралдарды қосу арқылы GitHub репозиторийінде қолжетімді код фрагменттері бар.

Бұл мақалада сізге осы кодтарды қалай қолдану керектігін, немесе нақты қай жерге орналастыру керектігін көрсетемін.

Фрагменттерді төмендегі сілтемеден таба аласыз:

https://github.com/next-press/wp-ultimo-snippets/

Кодты қосудің 2 жолы бар:

  1. Темаңыздың functions.php файлына.

  2. Must-Use Plugins (mu-plugins)

# Фрагментті темаңыздың functions.php файлына қалай қосу керек. {#how-to-add-the-snippet-on-your-themes-functionsphp-file}

  1. WordPress желі администратор панеліне кіріп, Themes > Theme Editor (Тұйғыр суретте қараңыз).

  2. "Themes" бетінде экранның оң жақ жоғарғы жағында орналасқан тіктермен айналмалы түймеде белсенді темаңызды таңдағаныңызды растаңыз (#3-ші суретте).

  3. "Theme Files" бөлімінен functions.php файлына басып, файлды жүктеңіз. Төменгі жаққа жылжып, GitHub репозиторийінен алған Ultimate Multisite фрагментін қосыңыз.

# Must-Use Plugins (mu-plugins) қалай жасау керек {#how-to-create-must-use-plugins-mu-plugins}

WordPress-те "Must-Use Plugins" немесе қысқпағында "mu-plugins" деп аталатын арнайы функционалдықты жүктеуге мүмкізу ерекшелік бар.

Бұл арнайы mu-плагиндер басқа әдеттегі плагиндерден бұрын жүктеледі және оларды өшіру мүмкін емес. Multisite желіде, бұл mu-плагиндердегі код орнатылған барлық сайттарда жүктеледі.

1\. FTP немесе SSH арқылы WordPress орнатылыңыздың файл жүйесіне қол жеткізіңіз.

2. WordPress орнатуыңыздың wp-content директориясында `mu-plugins` деп аталатын жаңа директорияны жасаңыз.

<!-- Скриншот қолжетімсіз: Файл менеджеріде wp-content директориясы және mu-plugins папкасы көрсетілген -->

3. Компьютеріңізде Notepad немесе кез келген код редаторы арқылы `wu-snippet.php` деп аталатын жаңа PHP файлы жасаңыз.

4. Ultimate Multisite кодының үлгісін GitHub репозиторийден алып, оны файлға қойып сақтаңыз. Бұл кодтың жоғарғысына mu плагиніңізді белгілеу үшін бұл кодты қосуға да болады.
