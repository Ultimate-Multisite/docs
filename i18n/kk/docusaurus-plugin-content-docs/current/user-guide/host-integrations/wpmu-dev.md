# WPMU DEV Интеграция

## Обзор
WPMU DEV — это комплексная платформа WordPress, которая предлагает хостинг, плагины и услуги для сайтов на WordPress. Эта интеграция позволяет автоматически синхронизировать домены и управлять SSL-сертификатами между Ultimate Multisite и хостингом WPMU DEV.

## Возможности
- Автоматическая синхронизация доменов
- Управление SSL-сертификатами
- Расширенные попытки проверки SSL-сертификатов

## Требования
Интеграция автоматически определяет, размещаете ли вы сайт на WPMU DEV, и использует встроенный API. Если вы размещаете сайт на WPMU DEV, дополнительная настройка не требуется.

Интеграция проверяет наличие константы `WPMUDEV_HOSTING_SITE_ID`, которая автоматически определяется при размещении на WPMU DEV.

## Инструкции по настройке

### 1. Проверка хостинга на WPMU DEV

Если вы размещаете сайт на WPMU DEV, необходимые константы уже должны быть определены. Убедитесь, что:

1. Константа `WPMUDEV_HOSTING_SITE_ID` определена в вашей среде
2. У вас есть активная подписка на WPMU DEV с доступом к API

### 2. Включение интеграции

1. В админке WordPress перейдите в Ultimate Multisite > Settings (Настройки)
2. Перейдите на вкладку "Domain Mapping" (Сопоставление доменов)
3. Прокрутите вниз до раздела "Host Integrations" (Интеграции хостинга)
4. Включите интеграцию WPMU DEV
5. Нажмите "Save Changes" (Сохранить изменения)

## Как это работает

### Синхронизация доменов

Когда домен сопоставляется в Ultimate Multisite:

1. Интеграция использует API WPMU DEV для добавления домена в ваш аккаунт хостинга
2. Она также автоматически добавляет версию с `www` доменом
3. WPMU DEV занимается настройкой домена и выдачей SSL-сертификатов

### Управление SSL-сертификатами

Интеграция WPMU DEV хостинг үшін SSL сертификатты إصدارу және орнату кезінде уақыт алуы мүмкін болғандықтан, SSL сертификатты التحقق үшін артық санын арттыруға бапталуы тиіс. Стандартты 5 реттен бұрын, ол SSL сертификатын التحقق үшін 10 مرة дейін арайды.

## Маңызды Ескертулер

### Доменді алып тарту
Қазіргі уақытта WPMU DEV API-де домендерді алып тартуға арналған жол жоқ. Ultimate Multisite-та домен картасы алынған кезде, домен сіздің WPMU DEV хостинг есебіңізде қалады. Қажет болса, оны WPMU DEV хостинг дашбордынан қолмен алып тартуыңыз керек.

### API Аутентификациясы
Интеграция WordPress дерекқорыңыздағы `wpmudev_apikey` опциясында сақталған WPMU DEV API кілтін қолданады. Бұл сіз сайтты WPMU DEV-пен қоссаңыз, автоматты түрде орнатылады.

## Түзету (Troubleshooting)

### API Байланыс Мәселелері
- Сіздің сайтыңыз WPMU DEV-мен дұрыс қосылғанын тексеріңіз.
- `wpmudev_apikey` опциясы WordPress дерекқорыңызда орнатылғанын тексеріңіз.
- WPMU DEV мүшелігіңіздің белсенді екенін қамтамасыз етіңіз.

### SSL Сертификат Мәселелері
- WPMU DEV SSL сертификаттарын إصدارлауға уақыт алады (әдетте 5-15 минут).
- Интеграция SSL сертификаттарды التحقق үшін 10 مرة дейін тексеруге бапталуы тиіс.
- Көп рет артықталғанға қарамастан, SSL сертификаттары إصدارлмаса, WPMU DEV қолдаумен байланысыңыз.

### Домен Қосылмады
- Ultimate Multisite жолдарында кез келген қате хабарламаларды тексеріңіз.
- Доменді WPMU DEV-ге алдын ала қосылмағанын тексеріңіз.
- Сіздің WPMU DEV хостинг жоспары қосатын домен санын қолдайтынын қамтамасыз етіңіз.
