# Closte Интеграция

## Обзор
Closte — бұл Google Cloud инфрақұрылымы негізінде құрылған басқарылатын WordPress хостинг платформасы. Бұл интеграция Ultimate Multisite мен Closte арасында доменді автоматты синхролдау және SSL сертификаттарын басқаруға мүмкіндік береді.

## Ерекшеліктері
- Доменді автоматты синхролдау
- SSL сертификатын басқару
- Wildcard домендерді қолдау
- Closte-де іске қосу үшін еш конфигурация қажет емес

## Қажеттілігілер
Егер сіз Closte-де хостинг жасасаңыз, `wp-config.php` файлыңызда келесі константаны анықтауыңыз керек:

```php
define('CLOSTE_CLIENT_API_KEY', 'your_api_key');
```

Бұл константа Closte-де хостинг жасасаңыз, әдетте алдын ала анықталған болады.

## Орнату нұсқаулары

### 1. Closte API кілттеріңізді тексеріңіз

Егер сіз Closte-де хостинг жасасаңыз, `CLOSTE_CLIENT_API_KEY` константасы `wp-config.php` файлыңызда алдын ала анықталған болуы керек. Бұл мәліметті тексеру үшін `wp-config.php` файлын қарап көре аласыз.

### 2. Интеграцияны іске қосу

1. WordPress администраторында Ultimate Multisite > Settings-ке өтіңіз
2. "Domain Mapping" (Доменді карталау) табына өтіңіз
3. Төменгі бөлімде "Host Integrations" (Хост интеграциялары) дегенге жетіп қараңыз
4. Closte интеграциясын іске қосыңыз
5. "Save Changes" (Өзгерулерді сақтау) батырмасын басыңыз

## Қалай жұмыс істейді

Ultimate Multisite-те домен карталанған кезде:

1. Интеграция Closte API-ға доменді қолданбарыңызға қосу үшін сұраныс жібереді
2. Closte SSL сертификатын автоматты түрде қамтамасыз етеді
3. Доменді карталау алынып кеткен кезде, интеграция сол доменді Closte-ден алып тастайды

Интеграция Ultimate Multisite-тегі DNS тексеру аралығын (DNS check interval) басқару арқылы жұмыс істейді, бұл жүйенің DNS таралуын және SSL сертификатын шығару жиілігін қалай тексеру керектігін реттеуге мүмкізу етеді.

## Домен тіркемесін жасау

Бұл интеграция сайт жасалған немесе көшірілген кезде домен тіркеуін автоматты түрде жасайды. Бұл Closte интеграциясы үшін ерекше маңызды, өйткені домен тіркеуін жасау Closte API-ді домен мен SSL сертификатын жасауға іске қосады.

## Мәселелерді шешу (Troubleshooting)

### API الاتصال мәселелері
- Closte API кілттерің дұрыс екенін тексеріңіз
- Closte аккаунтыңызда қажетті рұқсаттар бар екенін қамтамасыз етіңіз

### SSL сертификат мәселелері
- Closte SSL сертификаттарын шығаруға уақыт алады (әдетте 5-10 минут)
- Домендеріңіздің Closte серверінің IP-адресіне дұрыс бағытталғанын тексеріңіз
- Доменіңіздің DNS kayıtтарын тексеріп, олардың дұрыс орнатылғанын растаңыз

### Домен қосылмаған
- Ultimate Multisite журналында (logs) кез келген қате хабарлама бар-жоғын тексеріңіз
- Домен Closte-ке әлі қосылмаған екенін растаңыз
- Доменіңіздің DNS kayıtтары дұрыс орнатылғанын қамтамасыз етіңіз

### DNS тексеру аралығы (DNS Check Interval)
- Егер SSL сертификаттары шығару ұзақ уақыт алатын болса, Доменді Схема (Domain Mapping) параметрлерінде DNS тексеру аралығын реттеуге болады
- Дефолт аралық 300 секунд (5 минут), бірақ тест кезінде жылдам тексеру үшін оны 10 секундке дейін қоюға болады
