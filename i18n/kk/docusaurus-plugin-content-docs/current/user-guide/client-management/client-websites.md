# Клиенттің веб-сайттары (v2)

Сіз біздің желіде тіркелгеннен кейін, клиенттері олардың дашбордына қол жеткізе алады.

Олар төлемдері, мүшеліктер, домендері, жоспар шектеулері және т.б туралы маңызды ақпаратты көре алатын **Account** парағынан бөлек, барлық веб-сайттарын көретін және басқаратын **My Sites** парағына да қол жеткізеді. Олар тек дашбордағы My Sites-қа басумен бәрін істей алады.

![Клиент дашборсындағы My Sites сілтемесі](/img/account-page/my-sites-link.png)

Олар барлық веб-сайттарын көретіні, олардың әрқайсысының астындағы **Manage** (Басқару) баты арқылы басқара алады. Егер олар оған басса, олар сол нақты веб-сайтын **dashboard-ына** ауысады.

Олар сондай-ақ, бірнеше веб-сайтқа рұқсат беретін жоспарлар үшін қайсысының **primary** (басты) сайт екенін көре алады.

![Клиенттердің веб-сайттары мен Manage баты бар My Sites парағы](/img/account-page/my-sites-page.png)

Міне, сайтты редакциялау интерфейсінің толық көрінісі:

![Сайтты редакциялаудың толық интерфейсі](/img/admin/site-edit-full.png)

Егер олар басты веб-сайтын өзгергісі келсе, **Account** мәзірінен басқа параққа өтіп, **Account**-қа басуы керек.

![Клиент дашборсындағы Account мәзірінің сілтемесі](/img/account-page/account-menu.png)

Олар клиенттердің мәліметтерін көруге және редакциялауға да болады:

![Клиентті редакциялау интерфейсі](/img/admin/customer-edit.png)

Содан кейін, олар **Change Default Site** (Басты сайтты өзгерту) батына басуы керек.

![Account парағында Change Default Site баты](/img/account-page/change-default-site.png)

Шығу терезе пайда болады. Төменгі тізімге басып, қай веб-сайтты басты сайт ретінде қосу керектігін таңдаңыз.

![Change Default Site төменгі тізімді таңдау](/img/account-page/change-default-site-dropdown.png)

Содан кейін, жай ғана **Change Default Site**-қа басыңыз және біткен!

![Стандартты сайтты өзгертуді растау](/img/account-page/change-default-site-confirm.png)
