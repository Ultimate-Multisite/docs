# Sunrise Файлын Орнатуда Қателік {#error-installing-the-sunrise-file}

`sunrise.php` файлы — бұл WordPress өзін іске қоса отырып іздейтін арнайы файл. WordPress-тің `sunrise.php` файлын анықтай алуы үшін ол **wp-content папкасының** ішінде болуы керек.

Ultimate Multisite-ты активтелеп, скриншоттағыдай орнату wizard-ын өткізген кезде Ultimate Multisite біздің `sunrise.php` файлын wp-content папкасына көшіруді асырады.

<!-- Скриншот қолжетімсіз: sunrise.php орнатылу кезеңін көрсететін орнату wizard-ы беті -->

Көбінесе біз файлды сәтті көшіре аламыз және бәрі жұмыс істейді. Алайда, егер бір нәрсе дұрыс орнатылмаса (мысалы, папка рұқсаттары), Ultimate Multisite файлды көшіре алмауы мүмкін жағдайға тап болуыңыз мүмкін.

Ultimo-ның сізге берген қателік хабарламасын оқысаңыз, мұнда дәл не істегені көресіз: **Sunrise көшіру сәтсіз**.

<!-- Скриншот қолжетімсіз: Sunrise көшіру сәтсіздігін көрсететін қателік хабарлама -->

Бұл мәселені шешу үшін, қарапайым түрде `wp-ultimo` плагинінің ішіндегі `sunrise.php` файлын көшіріп алып, оны өзіңіздің wp-content папкасына қосыңыз. Бұл істегеннен кейін wizard бетін қайта жүктеңіз және тексерулер сәтті өтуі керек.

<!-- Скриншот қолжетімсіз: wp-ultimo плагині папкасындағы sunrise.php файлын көрсететін файл менеджер --> Қайта да, болашақта мәселе тусуыndan аулақ болу үшін папка рұқсаттарыңызды жалпы тексеруді ұсынуға болады (бұл тек Ultimate Multisite-ке ғана емес, басқа плагиндер мен темлерге де).

WordPress-тің бөлігі ретіндегі **Health Check tool** (оны негізгі сайттың **admin panel > Tools > Health Check** арқылы қол жеткізе аласыз) папка рұқсаттары WordPress-ке мәселе тудыруы мүмкін мәндерге орнатылғанын білуге мүмкіндік береді.
