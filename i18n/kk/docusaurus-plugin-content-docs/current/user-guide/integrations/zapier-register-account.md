# Оқиға: Zapier арқылы аккаунт тіркеу

[Ultimate Multisite-ті Zapier-мен интеграциялау туралы] мақаласында біз Ultimate Multisite ішіндегі әртүрлі әрекеттерді оқиғалар мен іс-әрекеттер негізінде қалай қолдану керектігін талқылағанбыз. Бұл мақалада біз үшінші тараптық қосымшаларды (3rd party applications) интеграциялауды көрсетеміз. Біз деректердің көзі ретінде Google Sheets-ті қолданып, Ultimate Multisite-ке аккаунт тіркеу үшін ақпарат жібереміз.

Zapier-ді қосу алдында **Ultimate Multisite > Settings > API & Webhooks** бөліміне өтіңіз және API-дің қосылғанын растаңыз. Zapier Ultimate Multisite-тің аккаунт деректерін сұраған кезде осы экраннан API Key мен API Secret-ті көшіріп алыңыз.

![API және Webhooks параметрлері, API Key, API Secret және API-ді қосу опцияларымен](/img/admin/settings-api-webhooks.png)

Ең алдымен, Google Drive-іңіз ішінде **Google Sheet** жасауыңыз керек. Әр бағанды (column) дұрыс анықтап отырыңыз, солайша кейін деректерді оңай картаға салуға болады.

Google Sheet-ті жасағаннан кейін, Zapier аккаунтыңызға кіріп, zap құруды бастай аласыз.

**"App event"** іздеу өніміне **"Google Sheets"**-ті таңдаңыз.

Содан кейін "**Event**" (Оқиға) өніміне "**New spreadsheet row**" (Жаңа жаzdғы жол) таңдап, "**Continue**" (Бұрынғыдай жалғасу) басыңыз.

Келесі қадамде сізге **Google Sheet-ті сақтайтын Google Account**-ты таңдау керек. Сондықтан дұрыс Google аккаунты көрсетілгеніне көз жеткізіңіз.

**"Set up trigger"** (Іс-әрекетті орнату) бөлімінде, деректердің келетін Google spreadsheet және worksheet-ті таңдап белгілеу қажет болады. Осы ақпаратты толтырып, "**Continue**" басыңыз.

Енді Google Sheet дұрыс қосылғанын тексеру үшін **"test your trigger"** (іс-әрекетті сынау) қадамдары бар.

Тестіңіз сәтті болса, сіздің spreadsheet-теріңізден кейбір мәндерді көрсететін нәтижені көресіз. Жалғасу үшін "**Continue**" басыңыз.

Келесі қадам — Ultimate Multisite-та аккаунт жасау немесе тіркеуді қамтамасыз ететін екінші әрекетті орнату. іздеу өрісінен "**Ultimate Multisite(2.0.2)** " таңдаңыз.

"**Event**" (Оқиға) өрісінің астында "**Register an Account in Ultimate Multisite**" (Ultimate Multisite-та аккаунт тіркеу) таңдап, содан кейін "**Continue**" (Бұрынғыдай жалғасу) батырмасын басыңыз.

"**Set up an action**" (Әрекетті орнату) бөлімінде сізге клиенттер деректері, мүшеліктер, өнімдер және т.б. үшін әртүрлі өрістер көрінеді. Сіз осы мәндерді Google Sheet-тегі құжаттарыңызға сәйкес келетін қатысты өрістерге тарту (mapping) арқылы бөле аласыз, бұл астындағы скриншотта көрсетілгендей толтырылуы керек.

Мәндерді тартудан кейін, сіз әрекетті тексеруіңізді мүмкіндікке ие боласыз.
