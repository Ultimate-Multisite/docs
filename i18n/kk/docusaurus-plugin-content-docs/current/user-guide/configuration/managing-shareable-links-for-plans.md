# Пландар үшін бөлісу мүмкіндіктерін басқару

Ultimate Multisite сізге план сілтемелерін тікелей клиенттеріңізмен бөлісу мүмкіндігін ұсынады. Бұл істі жүзеге асыру үшін кез келген планың редакциялау бетіне өтіп, _Shareable Link-ті көшіру_ (Click to copy the Shareable Link) батырмасын басыңыз.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_Планның редакциялау бетінде бөлісу мүмкіндіктерін жасау батырмасын таба аласыз._

Бұл механизм сіздің стандартты төлем жиілігін пайдаланып сілтемені құруды қолданатынын ескертетін. Бұл ретінде, мысалы, сіздің стандартты төлеміңіз ай сайын болса, соңында сізде мұндай сілтеме болады: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Егер бөлісу сілтемесін тоқсан сайын (quarterly) реттеуді қаласаңыз, соңына 3 қосу қажет болады:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Планды _premium_ де жыл сайын кестеге таңдағыңыз келсе, одан кейін 1 және сөзді қосу қажет:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Әрбір төлем жиілігі үшін бір ғана сілтеме болуы мүмкін екенін ұмытпаңыз. Әрбір төлем жиілігіне арналған нақты бір сілтемені бөлісуіңіз керек.

Сіз сілтемені клиенттеріңізге бөлісе отырып, олар сіздің тіркелу процесінің алғашқы қадамына ауысады.
