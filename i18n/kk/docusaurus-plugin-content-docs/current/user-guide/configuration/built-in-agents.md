# Кіріктірілген агенттер

Gratis AI Agent v1.9.0 бес кіріктірілген агентпен жеткізіледі; әрқайсысы белгілі бір құралдар жиынымен, бейімделген жүйелік нұсқаумен және сол саладағы жиі тапсырмаларға сәйкес бастапқы ұсыныстармен алдын ала бапталған. Агенттер арасында ауысу көмекшінің не істей алатынын және қалай жауап беретінін өзгертеді — сіз тарапынан ешқандай баптау қажет емес. Superdav AI Agent v1.18.0 қатысты интеграциялар бапталған кезде осы жұмыс ағындарына кестені ескеретін құралдарды, еске салу жазбаларын, мақұлдау қақпаларын және SMS хабарландыруларын қоса алады.

## Агент деген не?

Әр агент мыналарды біріктіретін атаулы баптау профилі:

- **Құралдар** — агентке шақыруға рұқсат етілген мүмкіндіктер (мысалы, Content Writer жазба жасау мүмкіндіктеріне қол жеткізе алады; Design Studio CSS және theme.json мүмкіндіктеріне қол жеткізе алады)
- **Жүйелік нұсқау** — агенттің мәнерін, басымдықтарын және шектеулерін белгілейтін нұсқаулар
- **Ұсыныстар** — жылдам бастауға көмектесу үшін чат интерфейсінде көрсетілетін алдын ала жазылған сұраулар

## Агент таңдағышына қол жеткізу

1. WordPress әкімші бүйірлік тақтасындағы **Gratis AI Agent** панелін ашыңыз.
2. Чат тақырыбының жоғарғы сол жағындағы **агент белгішесін** басыңыз (белгіше белсенді агентті көрсету үшін өзгереді).
3. **Agent Picker** пішін-кесте қабаттасуы ретінде ашылады. Әр агент өзінің белгішесімен, атауымен және бір жолдық сипаттамасымен тізімделеді.
4. Оны белсендіру үшін агент жолын басыңыз. Чат тақырыбы бірден жаңартылады.

Сондай-ақ агенттерді әңгіме ортасында ауыстыра аласыз — жаңа агенттің жүйелік нұсқауы келесі хабарламадан бастап күшіне енеді.

## Бес кіріктірілген агент

### Content Writer

**Фокусы:** Жазбалар, беттер және байланыс пішіндерін жасау әрі өңдеу.

**Қолжетімді құралдар:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`. Superdav AI Agent v1.18.0 интеграциялары қосылған кезде, бапталған күнтізбе контексті, мақұлдау қақпалары, еске салулар және SMS хабарландыру құралдары мақұлданған жұмыс ағындары үшін де қолжетімді болуы мүмкін.

**Жақсы орындайтындары:**
- Қысқаша сипаттама немесе жоспар бойынша блог жазбаларын дайындау және жариялау
- Жаңа сайт үшін қону беттерінің топтамаларын жасау
- Байланыс және сұрау пішіндерін құрастыру
- URL немесе іздеу арқылы жазбаларға таңдаулы суреттер қою
- Бапталған Google Calendar контексті бойынша іс-шарадан кейінгі хабарламаларды дайындау, содан кейін хабарландыруларды жібермес бұрын мақұлдауды күту

**Бастапқы ұсыныстар:**
- *WordPress multisite артықшылықтары туралы 500 сөздік блог жазбасын жазыңыз.*
- *About, Services және Contact бетін жасап, оларды жариялаңыз.*
- *Contact бетіне брондау сұрауы пішінін қосыңыз.*
- *Ертеңгі бапталған күнтізбе іс-шарасындағы қатысушыларға еске салу дайындап, оны жібермес бұрын мақұлдауды күтіңіз.*

---

### Site Builder

**Фокусы:** Бір сұраудан бастап толық веб-сайт жасау.

**Қолжетімді құралдар:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`. Superdav AI Agent v1.18.0 арқылы әкімшілер қосқан жерлерде бапталған басқарылатын қызмет, мақұлдау, еске салу, күнтізбе және SMS құралдары қолжетімді болуы мүмкін.

**Жақсы орындайтындары:**
- Сипатталған бизнес түрі үшін көп кезеңді сайт құрастыру жоспарын жасау
- Әр кезеңді дербес орындау — құрылым, мазмұн, навигация, дизайн
- Қолмен араласуды қажет етпей, жоспар ортасындағы қателерден қалпына келу
- Құрастырудың бір бөлігі ретінде ұсынылған плагиндерді орнату
- Байланыс пішіндерін тікелей чат интерфейсінен жасау (Superdav AI Agent v1.10.0+)
- Мақұлдау қақпалары мен еске салу жазбалары қосылған кезде қайталанатын хабарландыруларсыз іске қосу еске салуларын немесе қатысушыларға кейінгі хабарламаларды үйлестіру

**Бастапқы ұсыныстар:**
- *Галерея жазба түрі, брондау беті және байланыс пішіні бар фотография портфолио сайтын құрыңыз.*
- *Онлайн мәзірі, жұмыс уақыты және үстел брондау сұрауы пішіні бар мейрамхана веб-сайтын жасаңыз.*
- *Қызмет беттері, портфолио бөлімі және блогы бар фриланс кеңес беру сайтын баптаңыз.*
- *Site builder арқылы Contact бетіне байланыс пішінін қосыңыз.*
- *Сайтты іске қосу тексеру тізімі мақұлданғаннан кейін, бапталған мүдделі тарап байланысына SMS еске салуын жіберіңіз.*

### Design Studio

**Фокусы:** Көрнекі теңшеу — түстер, типография, CSS және блок үлгілері.

**Қолжетімді құралдар:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Жақсы орындайтындары:**
- Атаулы theme алдын ала баптауларын қолдану (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- theme.json арқылы жаһандық типография мен түс палитраларын дәл баптау
- Брендке тән үстемелер үшін теңшелетін CSS енгізу
- Беттің скриншотын түсіріп, дизайн мәселелері бойынша қарап шығу

**Бастапқы ұсыныстар:**
- *warm-editorial алдын ала баптауын қолданып, содан кейін негізгі түсті #2d6a4f етіп орнатыңыз.*
- *Басты беттің скриншотын түсіріп, нені жақсартар едіңіз, айтыңыз.*
- *Толық енді фон суреті және ортаға тураланған тақырыбы бар қайта қолданылатын hero блок үлгісін жасаңыз.*

### Plugin Manager

**Фокусы:** WordPress плагиндерін табу, орнату және басқару.

**Қолжетімді құралдар:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Жақсы орындайтындары:**
- Сипатталған қолдану жағдайы үшін ең жақсы плагинді ұсыну
- Тізілімнен мүмкіндік бумаларын орнату
- Қолжетімді мүмкіндіктер каталогын санат бойынша шолу

**Бастапқы ұсыныстар:**
- *Мүшелік анықтамалығы үшін ең жақсы плагин қандай?*
- *WooCommerce мүмкіндіктер бумасын орнатыңыз.*
- *Барлық қолжетімді ecommerce мүмкіндіктер бумаларын көрсетіңіз.*

---

### Support Assistant

**Фокусы:** Сайт мазмұны, баптаулар және WordPress конфигурациясы туралы сұрақтарға жауап беру.

**Қолжетімді құралдар:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Нені жақсы орындайды:**
- Ағымдағы сайт баптаулары мен опцияларын іздеп табу
- Сайтта қандай жазба түрлері, таксономиялар және мәзірлер конфигурацияланғанын түсіндіру
- Тікелей мәндерді оқу арқылы "бұл баптау не істейді?" сұрақтарына жауап беру
- Өзгерістер енгізбес бұрын тек оқуға арналған диагностикалық қабат ретінде қызмет ету

**Бастапқы ұсыныстар:**
- *Бұл сайтта қазір қандай plugin-дер мен баптаулар белсенді?*
- *Осы сайтта тіркелген барлық пайдаланушылық жазба түрлерін тізіп бер.*
- *Қандай навигация мәзірлері бар және олар қайда тағайындалған?*

## Superdav автоматтандыру интеграциялары

Superdav AI Agent v1.18.0 интеграциялары конфигурацияланған кезде, кірістірілген агенттер кестені ескеретін қауіпсіздеу автоматтандыру жұмыс процестеріне қатыса алады:

- **Google Calendar оқу құралдары** агенттерге кейінгі жұмысты дайындамас бұрын конфигурацияланған күнтізбелер мен оқиғаларды тексеруге мүмкіндік береді.
- **Байланыс және қатысушыны сәйкестендіру** оқиға қатысушыларын WordPress пайдаланушыларымен немесе белгілі контактілермен сәйкестендіруге көмектеседі.
- **Адам мақұлдауының қақпалары** сезімтал әрекеттерді уәкілетті пайдаланушы қарап, растағанға дейін кідіртеді.
- **Еске салу жазбалары** жоспарланған тапсырмалар қайта орындалғанда немесе қайталанғанда, қайталанатын хабарландырулардың алдын алады.
- **TextBee SMS хабарландырулары** SMS тіркелгі деректері мен жұмыс процесі рұқсаттары қосылған кезде ғана конфигурацияланған мәтіндік хабарларды жібереді.

Ұсынылатын жұмыс процесі: агенттен хабарламаны немесе әрекетті дайындауды сұраңыз, мақұлдау сұрауын қарап шығыңыз, содан кейін мақұлданған әрекетті жалғастыруға рұқсат беріңіз. Қайталанатын еске салулар үшін еске салуларды қайталамау мүмкіндігін қосулы ұстаңыз, сонда бір оқиғаға немесе контактіге қайта-қайта хабарландыру жіберілмейді.

## Агенттерді теңшеу

Әрбір кірістірілген агентті `gratis_ai_agent_agents` filter арқылы кеңейтуге немесе ауыстыруға болады.

### Бар агентке пайдаланушылық жүйелік prompt қосу

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### Жаңа агентті тіркеу

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

Жаңа агент filter іске қосылғаннан кейін бірден агент таңдағышта пайда болады.

### Кірістірілген агентті жою

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
