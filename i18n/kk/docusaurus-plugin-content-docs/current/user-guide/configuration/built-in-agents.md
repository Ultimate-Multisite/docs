# Кіріктірілген Агенттер

Gratis AI Agent v1.9.0 бес кірікті агентпен келеді, әрқайсысы белгілі бір құралдар жиынтығымен, арнайы жүйе промптымен және сол саланың жиі кездесетін тапсырмаларына сәйкес бастама ұсыныстарымен дайын. Агенттер арасында ауысу ассистенттің не істей алатынын және қалай жауап беретінін өз тарапызда ешқандай конфигурациясыз өзгертеді.

## Агент деген не?

Әр агент — атауы бар конфигурация профилі, ол келесілерді біріктіреді:

- **Құралдар (Tools)** — агенттің қолдануға рұқсат етілген мүмкіндіктері (мысалы, Контент жазушы пост жасау мүмкіндігіне; Дизайн Студия CSS және theme.json мүмкіндіктеріне қол жеткізеді).
- **Жүйе промпты (System prompt)** — агенттің үнін, басымдықтары мен шектеулерін белгілейтін нұсқаулықтар.
- **Ұсыныстар (Suggestions)** — сізге тез бастауға көмектесу үшін чат интерфейсінде көрсетілетін алдын ала жазылған промпттар.

## Агент таңдағышқа қол жеткізу

1. WordPress администратор панелінің жақтан **Gratis AI Agent** панелін ашыңыз.
2. Чат банерінің сол жақ шетіндегі **агент иконкасына** (активті агентке сәйкес өзгеріп тұратын иконка) басыңыз.
3. **Agent Picker** чат интерфейсіндегі форма-кесте ретінде ашылады. Әр агент оның иконкасымен, атауымен және бір жолды сипаттамасымен тізіледі.
4. Агент қатарына басып, оны белсендіріңіз. Чат банері дереу жаңаланып өзгереді.

Сіз сондай-ақ әңгіме ішінде агенттерді ауыстай аласыз — жаңа агенттің жүйе промпты келесі хабарламадан бастап күшіне енеді.

## Бес кірікті агент

### Контент жазушы (Content Writer)

**Бағыт:** Бұйрықтарды, беттерді және байланыс формаларын жасау және редакциялау.

**Қолжетімді құралдар:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`

**Негізделді жақсы істейтін мүмкіндіктер:**
- Қысқаша мәтін немесе жоспардан блог посттарды жазу және оларды жариялау
- Жаңа сайт үшін лендинг беттерінің топтарын жасау
- Байланыс және сұраныстар формаларын құру
- URL немесе іздеу бойынша посттарға ерекше суреттерді орнату

**Бастапқы ұсыныстар:**
- *WordPress multisite артықшылықтары туралы 500 сөзді блог пост жазыңыз.*
- *«Тәжірибе», «Қызметтер» және «Байланыс» беттерін жасап, оларды жариялаңыз.*
- *Байланыс бетіне брондау сұранысы формасын қосыңыз.*

---

### Сайт құрушы (Site Builder)

**Бағыты:** Бір тапсырмадан бастап кез келген сайтты толықтай жасау.

**Қолжетімді құралдар:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`

**Негізделді жақсы істейтін мүмкіндіктер:**
- Сипатталған бизнес түріне арналған көп сатылы сайт құру жоспарын генерациялау
- Әр сатыны автоматты түрде орындау — құрылым, контент, навигация, дизайн
- Жоспардың ортасында қолмен араласуды қажет етпей тұрып, қателерден қалпына келтіру
- Құру процесінің бір бөлігі ретінде ұсынылған плагиндерді орнату
- Чат интерфейсінен тікелей байланыс формаларын жасау (Superdav AI Agent v1.10.0+)

**Бастапқы ұсыныстар:**
- *Галерея пост түрі, брондау беті және байланыс формасымен фотограф портфолио сайты құрыңыз.*
- *Онлайн меню, ашылу уақыты және үстелді резервтеу сұранысы формасы бар ресторан сайты жасаңыз.*
- *Қызметтер беттері, портфолио бөлімі және блогпен фрилансер консалтинг сайты орнатыңыз.*
- *Сайт құрушы арқылы Байланыс бетіне байланыс формасын қосыңыз.*

---

### Дизайн Студия (Design Studio)

**Бағыты:** Көріністі өзгерту — түстер, типография, CSS және блок үлгілері.

**Қолжетімді құралдар:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Не істей біліді:**
- Атаулы тема пресеттерін қолдану (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- theme.json арқылы жалпы типография мен түс палитрасын нақты реттеу
- Брендке арналған өзгерістерді енгізу үшін custom CSS қолдану
- Парақшаның скриншотын алу және дизайн мәселелерін қарап көру

**Бастапқы ұсыныстар:**
- *warm-editorial пресетін қолданыңыз, содан кейін негізгі түсті #2d6a4f деп белгілеңіз.*
- *басты бетті скриншот жасаңыз және не жақсарту керектігін айтыңыз.*
- *толық ендірілген фондық сурет пен ортадағы тақырыпқа ие қайта пайдалануға болатын hero block pattern құрыңыз.*

---

### Plugin Manager (Плагин Менеджер)

**Басты мәселе:** WordPress плагиндерін табу, орнату және басқару.

**Қолжетімді құралдар:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Не істей біліді:**
- Сипатталған пайдалану сценарийіне ең қолайлы плагинді ұсыну
- реестрден ability packs орнату
- Қолжетімді ability каталогын категория бойынша қарау

**Бастапқы ұсыныстар:**
- *Мемлекеттік тізім (membership directory) үшін ең жақсы плагин не?*
- *WooCommerce ability pack-ті орнатыңыз.*
- *Барлық қолжетімді электронды коммерция (ecommerce) ability packs-тарды көрсетіңіз.*

---

### Support Assistant (Қолдаушы Ассистент)

**Басты мәселе:** Сайт контенті, параметрлер және WordPress конфигурациясы туралы сұрақтарға жауап беру.

**Қолжетімді құралдар:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Негізделген мүмкіндіктері:**
- Ағымдағы сайт параметрлерін және опцияларын іздеу.
- Сайтта қандай пост түрлері (post types), таксономиялар (taxonomies) және менюлер (menus) орнатылғанын түсіндіру.
- Тікелей мәндерді оқып, "бұл параметр не істейді?" деген сұрақтарға жауап беру.
- Өз өзгертулер енгізуден бұрын оқу ғана диагностикатық қабат ретінде қызмет көрсету.

**Бастапқы ұсыныстар:**
- *Бұл сайтта қазір қай плагиндер және параметрлер белсенді?*
- *Бұл сайтқа тіркелген барлық арнайы пост түрлерін (custom post types) тізімдеңіз.*
- *Қандай навигациялық менюлер бар және олар қай жерге тағайындалған?*

## Агенттерді реттеу (Customising Agents)

Әрбір енгізілген агентті `gratis_ai_agent_agents` фильтрі арқылы кеңейту немесе ауыстыруға болады.

### Қазіргі агентке арнайы систем промпты қосу

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### Жаңа агентті тіркеу

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Posts and pages for search engines үшін оңтайландырады.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'Сіз SEO маманысыз. Кілт сөздерді оңтайландыруға, мета сипаттамаларға және құрылымдалған деректерге басымдық беріңіз.',
        'suggestions'   => [
            'Басты беттің тақырыбы мен мета сипаттамасын қараңыз.',
            'Ең соңғы бес посттың тақырыптарға (title tag) жақсару бойынша ұсыныстар беріңіз.',
        ],
    ];
    return $agents;
} );
```

Жаңа агент сүзгілерді іске қосып жібергеннен кейін Agent Picker-де дереу пайда болады.

### Құрылғыдан жасалған агентті алып тастау

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
