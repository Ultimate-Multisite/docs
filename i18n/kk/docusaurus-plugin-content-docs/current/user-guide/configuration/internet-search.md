# Интернет із пошуком

Gratis AI Agent v1.5.0-де додає функцію **Internet Search**, яка дозволяє AI ассистенту під час розмови отримувати актуальну інформацію з інтернету. Це працює на базі [Brave Search API](https://brave.com/search/api/) або [Tavily API](https://tavily.com/).

## Як це працює {#how-it-works}

Коли інтернет-пошук увімкнено, асистент може автоматично запитувати ваш налаштований пошуковий провайдер, коли він розуміє, що питання вимагають актуальної або зовнішньої інформації — наприклад, останні новини, поточні ціни чи документацію, яка могла змінитися з моменту навчання моделі.

Результати отримуються в режимі реального часу та додаються до контексту асистента перед тим, як він сформує відповідь. Асистент повідомляє, коли використовував результати пошуку для надання відповіді на запитання.

## Увімкнення Інтернету пошуку {#enabling-internet-search}

Для використання інтернету пошуку потрібен API-ключ від обраного вами пошукового провайдера. Щоб його налаштувати:

1. Перейдіть у **Gratis AI Agent → Settings → Advanced**.
2. Знайдіть випадаючий список **Internet Search Provider** і виберіть або **Brave Search**, або **Tavily**.
3. Введіть свій API-ключ у відповідне поле. Посилання для реєстрації відображаються поруч із кожним полем.
4. Натисніть **Save Settings** (Зберегти налаштування).

Після збереження ключа функція Internet Search автоматично стає доступною для асистента.

## Brave Search {#brave-search}

### Отримання API-ключа Brave Search {#obtaining-a-brave-search-api-key}

1. Відвідайте [Brave Search API page](https://brave.com/search/api/).
2. Зареєструйтесь на план. Є безкоштовний рівень з лімітом запитів щомісяця.
3. Скопіюйте свій API-ключ із панелі розробника Brave Search.
4. Вставте його у поле **Brave Search API Key** в налаштуваннях Gratis AI Agent.

### Обмеження використання {#usage-limits}

Қолдану пайдалану санына байланысты Brave Search арқылы есептеледі. Әрбір AI жауабы пошулауды тудырса, ол бір сұрау ретінде есептеледі. Күтпеген алымдардан аулақ болу үшін қолданыс талдауын [Brave Search developer dashboard](https://brave.com/search/api/) бетінде бақылаңыз.

## Tavily {#tavily}

Superdav AI Agent v1.10.0 **Tavily**-ді альтернативті интернет іздеу қызметі ретінде қосады, ол одан да бай іздеу нәтижелері мен жетілдірілген зерттеу мүмкіндіктерін ұсынады.

### Tavily API кілті алу {#obtaining-a-tavily-api-key}

1. [Tavily API бетіне](https://tavily.com/) өтіңіз.
2. Есептік жазбаңыз. Апталық сұрау шегімен бірге тегін деңгей қолжетімді.
3. Tavily дашбордынан API кілтіңізді көшіріп алыңыз.
4. Оны Gratis AI Agent параметрлеріндегі **Tavily API Key** өніміне қосыңыз.

### Қолданыс шектеулері {#usage-limits-1}

Қолданыс Tavily-дің API шақыру санына негізделгенімен есептеледі. Әрбір AI жауабы іздеуді тудырса, ол бір шақыру ретінде есептеледі. Күтпеген алымдардан аулақ болу үшін қолданыс талдауын [Tavily dashboard](https://tavily.com/) бетінде бақылаңыз.

## Интернет іздеуді өшіру {#disabling-internet-search}

API кілтін белсенді іздеу қызметі өнімінен алып тастап, сақтаңыз. Бұл мүмкіндіктер ассистентке ұсырылмауына әкеледі.

:::note
Интернет іздеу жауаптарды генерациялау үшін ассистент іздеу нәтижелерін күтуіне байланысты жауап беруге уақытты арттырады. Уақытқа сезімді қолданылатын жағдайлар үшін нақты уақытта іздеу қажет екенін немесе ассистенттің енгізілген білімі жеткілікті екенін қарастырыңыз.
:::
