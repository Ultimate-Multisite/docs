# Администратор панелін PRO темамен өзгертулер {#customizing-your-admin-panel-with-pro-theme}

Клиент жұмыстары кезінде біздің клиенттерге WordPress администратор дашбордында арнайы тәжірибен ұсыну қажеті туындайды. Егер сіз одан да кәсіби көрініс пен сезімді іздесеңіз, PRO Theme сізге көмектесе алады.

PRO Theme — клиенттердің дашбордысын өзгерту кезінде сізге көмектесуге арналған администратор темасы. Сонымен қатар ол Ultimate Multisite-тың премиум плагиндерінің негізгі бөлігі болып табылады.

Ол орнатылғаннан кейін, _Plugins_ парақшасына өтіп, оны желіңізде белсендіру үшін басыңыз.

<!-- Скриншот қолжетімсіз: WP Плагиндер парағында PRO Theme-ге Желіде белсендіру батыр -->

_Желі бойынша оны белсендіріңіз._

_**Ескерту:**_ _Егер сіз оны тек орнатсаңыз, ол клиенттердің мүмкіндіктерінің бірі ретінде көрсетіледі және бұл тему тек сол пайдаланушыға қосу үшін белгіленуге болады._

Енді сізде администратор интерфейсі толығымен қайта жобаланды! Сонымен қатар, сол жақ бағанда PRO Theme деп аталатын жаңа меню бар.

<!-- Скриншот қолжетімсіз: PRO Theme-тің қайта жобаланған администратор панелі интерфейсі -->

_Администратор панеліне жаңа көрініс._

Бұл менюда сізде бес қойынды болады:

<!-- Скриншот қолжетімсіз: PRO Theme Стильдін Реттеу Параметрлері қойды -->

_Стильді реттеу (Styling Settings)_

<!-- Скриншот қолжетімсіз: PRO Theme Функциялықтық қойынды -->

_Функционалдық (Functionality)_

<!-- Скриншот қолжетімсіз: PRO Theme Қосымша CSS қойынды -->

_Қосымша CSS (Custom CSS)_

<!-- Скриншот қолжетімсіз: PRO Theme Импорт/Экспорт қойынды -->

_Импорт/Экспорт (Import/Export)_

<!-- Скриншот қолжетімсіз: PRO Theme Белсендіру қойынды -->

_PRO Theme-ті белсендіру (Activate PRO Theme)_

Әрқайсысында сіз стиль элементтерін қосу немесе өшіру, функционалдық мүмкіндіктерін беру немесе бөгдеуге болады және өз клиенттеріңізге сайттар желіңізде қажет болуы мүмкін ретінде қажет ететін реттеуді ұсына аласыз.

Бірнеше реттеуден кейін сіз не алатының мысал:

_Сіздің клиентіңіздің Дашборды_
