# Тема Құрастыру Қабілеттері: Блок Темаларды Скелеттеу және Активациялау {#theme-builder-abilities-scaffold-and-activate-block-themes}

Superdav AI Agent v1.12.0 екі қуатты мүмкіндікті енгізеді, олар сізге чат интерфейсінен тікелей арнайы блок темаларды генерациялауға және пайдалануға мүмкізу береді.

## Жалпы Шолу {#overview}

**scaffold-block-theme** және **activate-theme** мүмкіндіктері агенттерге мыналарды жасауға және орындауға мүмкізу береді:
- Сіздің талаптарыңызға негізделген толық, өндіріске дайын блок темаларды генерациялау
- Мануал араласусыз сайтта темага автоматты түрде қол жеткізу
- Бағытталған дизайн шешімдері арқылы үйлесімді визуалды бірегейлік құру

## Блок Темасын Скелеттеу (Scaffold Block Theme) {#scaffold-block-theme}

**scaffold-block-theme** мүмкіндігі толық тема құрылымымен, оның ішінде мыналарды қамтитын жаңа WordPress блок теmasını генерациялайды:

- дизайн токендері бар `theme.json` конфигурациясы
- жиі қолданылатын макеттер үшін Блок үлгі файлдары
- Арнайы блок стилдері және вариативтері
- Тема метадандары және қолдау декларациялары

### Қолдану Жолы {#how-to-invoke}

Superdav AI Agent-пен чат ішінде сіз темага генерация сұрауыңызды жібере аласыз:

```
"Байланыс агентті 'Modern Agency' деп аталатын, көк және ақ түсті схемамен, sans-serif типографиямен және кәсіби макетпен жасау."
```

Агент мыналарды істейді:
1. Сөйлесу арқылы сіздің дизайн қалауларыңызды жинау
2. Толық тема құрылымын генерациялау
3. Барлық қажетті тема файлдарын жасау
4. Теманы активациялауға дайындау

### Күтілетін Нәтиже {#expected-output}

Мүмкіндік сәтті орындалған кезде, сіз мыналарды көресіз:

- Теманың скелеттелгені туралы растау
- Тема атауы және орналасуы
- Қолданылған дизайн токендерінің (түстер, типография, аралық) қысқаша қорытындысы
- Активациялауға дайын мәртебе

Мысал:
```
✓ "Modern Agency" темасы сәтті скелеттелді
  Орналасуы: /wp-content/themes/modern-agency/
  Түстер: Негізгі #0066CC, Екінші #FFFFFF
  Типография: Inter (sans-serif)
  Мәртебе: Активациялауға дайын
```

## Теманы Активациялау (Activate Theme) {#activate-theme}

**activate-theme** мүмкіндігі сіздің сайтыңызды жаңа құрылған немесе бар блок темасына ауыстырады.

### Қолдану тәсілі {#how-to-invoke-1}

Теманы құрдыktan кейін, оны дереу іске қосуға болады:

```
"Modern Agency темун іске қосу"
```

Немесе кез келген бар теманы іске қосу үшін:

```
"Twentytwentyfour темасына ауысу"
```

### Күтілетін нәтиже {#expected-output-1}

Іске қосу сәтті болғанда:

- Ағылшын (активті) тема туралы растау
- Алдыңғы тема атауы (қарақты тексеру үшін)
- Тема енді қай жерде қолжетімді екені туралы сайт URL-і
- Темаға арналған кез келген қосымша құрылымды белгілеу ескертулер

Мысал нәтижесі:
```
✓ Тема сәтті іске қосылды
  Активті тема: Modern Agency
  Алдыңғы тема: Twentytwentyfour
  Қолжетімді: https://yoursite.com
  Ескерту: Дизайнді тексеру үшін басты бетіңізді қараңыз
```

## Жұмыс істеу тәртібі: Құру және Іске қосу (Scaffold and Activate) {#workflow-scaffold-and-activate}

Тиісті жұмыс істеу тәртібі екі мүмкіндікті біріктіреді:

1. **Теманы құруды сұрау**: "Менің SaaS бетім үшін блок темасын жаса"
2. **Агент теmanı құрайды (scaffold)**: Файлдар мен дизайн токендерін генерациялайды
3. **Тексеру және жетілдіру**: Қажет болса, дизайн өзгерістерін талқылау
4. **Іске қосу**: "Теманы дереу іске қосу"
5. **Тексеру**: Жаңа дизайн қолданылғанын растау үшін сайтыңызды қараңыз

## Дизайн токендері және жекелеу (Customization) {#design-tokens-and-customization}

Құрылған темалар WordPress дизайн токендерін (`theme.json` арқылы) келесілер үшін қолданады:

- **Түстер**: Негізгі, екінші, акцент, нейтралды палитра
- **Типография**: Шрифт отырықтары (font families), өлшемдері, салмағы, жол арасының биіктіктері
- **Араласу (Spacing)**: Padding, margin, gap scales
- **Шеттер (Borders)**: Радиус және ендік токендері
- **Түбіретулер (Shadows)**: Деңгейлері

Бұл токендер `theme.json` файлында орталықтандырылған, бұл бүкіл дизайн жүйеңізді бір файлдан 쉽게 реттеуге мүмкіндік береді.

## Шешімдер және Ескертулер {#limitations-and-notes}

## Қолдау және шешімдер {#troubleshooting}

**Тема құрылғыдан кейін пайда болмайды**
- Тема директориясы бар екенін және оның қажетті рұқсаттарға ие екенін тексеріңіз.
- `theme.json` дұрыс JSON форматында екенін тексеріңіз.
- Тема атауы барлық қолданыстағы темалармен қақтығыспайтынын тексеріңіз.

**Қосу (Activation) сәтсіз болады**
- Сізде администратор рұқсаты бар екенін растаңыз.
- Тема директориясы WordPress-ке оқылуы мүмкін екенін тексеріңіз.
- Қазақша мәліметтерді қарау үшін WordPress қате журналдарын (error logs) қарастырыңыз.

**Дизайн токендер қолданылмайды**
- `theme.json` синтаксисінің дұрыс екенін тексеріңіз.
- Кэшті тазалау плагиндерді өшіріңіз.
- WordPress нұсқасы қолданып отырған токендерді қолдайтынын тексеріңіз.

## Келесі қадамдар {#next-steps}

Теманы белсендіргеннен кейін сіз мыналарды істей аласыз:
- **Design System Aesthetics** дағдысын пайдаланып типографияны, түстер мен аралықтарды (spacing) жетілдіруге боласыз.
- WordPress блок редаторы арқылы жеке блок стилдерін өзгертуге боласыз.
- Теманың `style.css` файлына қосымша CSS жазуға боласыз.
- Нақты бет түрлері үшін арнайы блок үлгілерін (templates) жасауға боласыз.
