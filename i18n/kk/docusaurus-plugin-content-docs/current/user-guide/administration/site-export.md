# Сайтты Экспорттау

**Site Export** администратор парақшылықтарды бір сайтты, немесе барлық желіні миграциялау, бэкап алу немесе беру жұмыстары үшін жүктеуге болатын архивке жинауға мүмкізу береді.

## Бір сайтты экспорттау {#exporting-one-site}

**Ultimate Multisite > Site Export**-қа өтіңіз және **Generate new Site Export** (Жаңа Сайт Экспортын генерациялау) таңдаңыз. Экспорттағысы келген субсайтты таңдап, архивке жүктемелерді (uploads), плагиндерді (plugins) және темаларды (themes) қосқаныңызды таңдаңыз.

Экспорт аяқталған кезде ZIP-ті **Existing Exports** (Қазіргі Экспорттар) тізілінен жүктеп алыңыз. Қазіргі экспортта `index.php` және `readme.txt` бар, бұл архивті бөлек импорт плагинін орнатпай, жаңа хостингке көтеріп бастауға болады.

## Барлық желіні экспорттау {#exporting-the-whole-network}

Желідегі барлық субсайттарды қамтитын бір архив қажет болса, Site Export парағында **Network Export** (Желі Экспорты) функциясын қолданыңыз. Бұл хостингтерді ауыстыру алдында, апаттан қалған жағдайларды жою жаттығулары немесе әр субсайт бірге сабиралатын кезеңдегі staging қайта құрылымдамалар үшін өте пайдалы.

Желі экспортыбір сайт экспортынан әлдеқайда үлкен болуы мүмкін, сондықтан оны трафикі төмен кезеңінде жүргізіп, мақсатты сақтаудың жүктемелерге, плагиндерге, темаларға және генерацияланған ZIP файлдарына жеткілікті бос кеңістік бар екенін растаңыз.

### Желі Импортттерін жинау (Network Import Bundles) {#network-import-bundles}

Ultimate Multisite 2.12.0-ден бастап Site Exporter **network import bundles** (желі импортттерін жинауларды) генерациялауға болады — желі бойынша сайтты қайта құруды жеңілдетуге арналған мамандандырылған архивтер. Желі импортттерін жинау бірнеше сайтты жаңа желі орнатуына қайта құру үшін қажетті барлық файлдар мен метадеректерді қамтиды.

#### Желі Импортттерін жинауды генералау {#generating-a-network-import-bundle}

1. **Ultimate Multisite > Site Export**-қа өтіңіз
2. **Generate new Network Export**-ты басыңыз
3. Экспорт түрі ретінде **Network Import Bundle**-ды таңдаңыз
4. Жүктемелерді, плагиндерді және темаларды қосу қажет екенін таңдаңыз
5. **Generate**-ты басыңыз
6. **Existing Exports** тізімінен пакетті ZIP форматында жүктеп алыңыз

#### Network Import Bundle-дан қалпына келтіру {#restoring-from-a-network-import-bundle}

Сайттарды network import bundle-дан қалпына келтіру үшін:

1. Ultimate Multisite-ты мақсатты хостингке орнатыңыз
2. multisite орнату wizard-ын аялаңыз
3. Жаңа network-қа **Ultimate Multisite > Site Export**-қа өтіңіз
4. **Import Network Bundle**-ды басыңыз
5. network import bundle ZIP файлына жүктеңіз
6. Экрандағы импорт нұсқауларын орындаңыз
7. Импорт процесі барлық сайттарды, олардың контентін және конфигурацияларын қалпына келтіреді

Network import bundles мыналарды сақтайды:
- Сайт контенті (посттар, беттер, арнайы пост түрлері)
- Сайт параметрлері мен опциялары
- Қолдалу ролдері және рұқсаттары
- Плагиндер мен темалар (егер пакетке енгізілсе)
- Медиа жүктеулері (егер пакетке енгізілсе)
- Арнайы базалық кестелер мен деректер

## Өзін-өзі іске қосатын export ZIP-ті орнату {#installing-a-self-booting-export-zip}

Жаңа хостингте өзін-өзі іске қосатын ZIP-ті қалпына келтіру үшін:

1. Экспортталған ZIP мазмұнын мақсатты web root-қа жүктеңіз
2. Жүктеген `index.php` файлына браузер арқылы ашыңыз
3. Пакеттегі экспорт пакетінен экрандағы орнату нұсқауларын орындаңыз
4. Кедейсоқ файлдарды өшіруден бұрын, экспортқа арналған ескертулерді қамтитын bundled `readme.txt` файлына қараңыз.

Қосымшаларға арналған орнату және импорт сипаттамалары үшін [Site Exporter addon documentation](/addons/site-exporter) -ды қараңыз.

Ultimate Multisite 2.9.0-да қосылған single-site құралы үшін [Export & Import](/user-guide/administration/export-import)-ты қараңыз.
