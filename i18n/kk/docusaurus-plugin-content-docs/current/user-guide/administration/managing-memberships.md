# Клубтарларды басқару

Ultimate Multisite-та сіз желіңіздегі барлық жазылымдарды және қол жеткізуді басқара аласыз. Жобалар мен өнімдердің жоспарлары, сайттар саны, төлем сомасы мен жиілігі, төлем тарихы және уақыт белгілерін сипаттау сияқты жазылым мәліметтерін қарап, өзгертуге болады.

Бұл бетке қол жеткізу үшін Ultimate Multisite менюіндегі **Memberships** (Клубтар) қойындысына өтіңіз.

![Memberships list page](/img/admin/memberships-list.png)

Memberships (Клубтар) бетінде сіздің клиенттеріңіздің тізімін және олардың есептік жазуларымен байланысты өнімдерді көре аласыз. Жобаларды статус бойынша сүзуге боласыз: активті, күтуде, тоқтап тұрған (on hold), мерзімі өткен (expired) немесе бұзылған (cancelled).

Тегін жазылымдар **өмірлік** жазылымдар ретінде қарастырылады. Олар мерзімсіздік күні алмайды және автоматты түрде өтетін жоқ. Тегін жоспар сынақ кезеңінен немесе белгілі бір мерзімнен кейін аяқталуы керек болса, сынақпен төлемді өнімді пайдаланыңыз, жазылымды қолмен бұйыңыз немесе қол жеткізу өзгерген кезде клиентті басқа жоспарға ауыстырыңыз.

Жазылымға басу арқылы жазылым мәліметтерін — төлем ақпараты, сайттар немесе визит шектеулері және т.б. — өзгертуге болатын бетке кіре аласыз.

![Membership edit interface](/img/admin/membership-edit.png)

Міне, жазылымды өңдеудің толық көрінісі:

![Membership edit full interface](/img/admin/membership-edit-full.png)

Сонымен қатар, желіңізде бұрын тіркелген пайдаланушылармен байланысты жаңа жазылымдарды қосуға да болады. **Add Membership** (Жазылым қосу) басын басып, мәліметтерді реттеңіз.

![Add Membership modal](/img/admin/membership-add-modal.png)

Содан кейін **Create Membership** (Жазылым жасау) басын басыңыз. Жаңа жазылым Memberships тізімінде пайда болады.
