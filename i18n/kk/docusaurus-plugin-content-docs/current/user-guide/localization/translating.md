# Ultimate Multisite аудармасы

Ultimate Multisite ағылшын тілінде дайындалған, бірақ плагин толық аударылуға мүмкіндік береді және біз аудармаларға белсенді қабылдаймыз.

## Негізгі Плагин Аудармалары

Негізгі Ultimate Multisite плагині ресми **WordPress.org аударма платформасы** арқылы аударылады:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

Бұл стандартты WordPress қоғамының аударма жүйесі. Аудармалар жеткілікті келісімді аңғарған кезде, олар WordPress плагині жаңартулары арқылы Ultimate Multisite пайдаланушыларының барлығына автоматты түрде таратылады.

**Бізге көмек керек!** Егер сіз ағылшын тілінен басқа тіл білетін адам болсаңыз, аудармаларға қатысуды қарастырыңыз. Бірнеше сөзді аудару да көмектеседі. Бастау үшін:

1. [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/) сайтына өтіңіз
2. Тілді таңдаңыз
3. "Contribute Translation" (Аударманы жіберу) батырмасын басыңыз
4. Сізге тегін WordPress.org есебі қажет болады

Егер сіз өз тілінің аударма редакторуына айналып, аудармаларды мақұлдай (келісуге мүмкіндік беретін) қызмет көрсеткізгізге орналасуды қаласаңыз, [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) арқылы хабарласыңыз.

## Қосымша Аудармалар (Addon Translations)

Қосымша плагиндер бөлек жерде аударылады:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Қазіргі уақытта қосымша аудармалар бастапқы нүкте ретінде AI арқылы жасалуда. Олар қолдауланатын барлық тілдерді қамтиды, бірақ оларда ұсақ сөздер немесе дәлдіктер бұзылуы мүмкін. Біз адам аудармашыларын қабылдаймыз, оларды табиғи және нақты ету үшін жақсартуға ұсыныстар беруге болады.

## Оңтүстік Аудармалар (Local Translations)

Егер сіз өз жобаңыз үшін жергілікті аударма жасауды қаласаңыз, жоғары деңгейде (upstream) үлестіруден гөрі, екі нұсқа бар:

### Poedit-ті қолдану

Сіз `.pot` файлына аударма жасау үшін **[Poedit](https://poedit.net)** десктоп қосымшасын пайдалана аласыз:

1. [poedit.net](https://poedit.net) сайтынан Poedit-ті жүктеңіз
2. `wp-content/plugins/ultimate-multisite/lang/` жолындағы `.pot` файлын ашыңыз
3. Poedit интерфейсіндегі мәтіндерді аударыңыз
4. Сақтаңыз -- Poedit `.po` және `.mo` файлдарын автоматты түрде жасайды
5. Нәтижелі файлдарды `lang` папкасына орналастырыңыз

### Loco Translate-ті қолдану

**[Loco Translate](https://wordpress.org/plugins/loco-translate/)** WordPress плагині сізге десктоп құралдарысыз тікелей администратор панелінен аударма жасауға мүмкіндік береді. Ол үшін аударма файлдарын жасау және жинақтауды автоматты түрде жүзеге асырады.
