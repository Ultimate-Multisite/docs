# PayPal Баوابة Құрылғысын Орнату (v2)

_**МАҢЫЗДЫ ЕТІНДІ:** Бұл мақала Ultimate Multisite 2.x нұсқасына қатысты._

Сіз төлем параметрлері бетінде Stripe, Stripe Checkout, PayPal және Руманды (Manual) төлем әдістерінің жетінебесті барлық төрт әдісін белсендіре аласыз. Бұл мақалада біз **PayPal**-мен интеграциялауды қалай жүргізу керектігін көреміз.

Stripe сияқты, PayPal онлайн төлемдер үшін, әсіресе WordPress веб-сайттарында кеңінен қолданылады. Бұл мақала сізге PayPal-ді сіздің желіңізде қолжетімді төлем әдісі ретінде қалай пайдалану керектігін бағыттайды.

Бұл интеграция үшін қажетті API құпиясын алу үшін **PayPal Business есебіне** ие болу керектігін ескертемін.

## PayPal-ді желіңізде белсендіру {#enabling-paypal-on-your-network}

PayPal-ді желіңізде қолжетімді төлем әдісі ретінде белсендіру үшін **Ultimate Multisite > Settings > Payments** таңбасына өтіп, PayPal жанындағы шарты белгілеңіз.

![Enabling PayPal in active payment gateways](/img/config/settings-payment-gateways.png)

## Бағытталған орнату құралынан пайдалану {#using-the-guided-setup-wizard}

Ultimate Multisite 2.10.0 төлем әдісі параметрлеріне бағытталған PayPal орнату құралын қосты. PayPal-ді белсендіргеннен кейін, بوابةсын қалай қосу керектігін таңдау және сақтау алдында қандай құпиялар қажет екенін растау үшін **Ultimate Multisite > Settings > Payments** арқылы осы құралды қолданыңыз.

Құрал екі орнату жолын қолдайды:

* **Қолмен құжатты енгізу (Manual credential entry)**: Егер сізде PayPal API куәлігі болса, OAuth орнату мүмкін болмаса немесе куәлікті өзіңіз көшіргіңіз келсе бұл жолды қолданыңыз. PayPal өрістеріне API İstifadəçi атауын (API Username), API Параметрін (API Password) және API Қосындығын (API Signature) енгізіп, төлем параметрлерін сақтаңыз.
* **OAuth қосылыс қақпасы (OAuth connection gate)**: OAuth опция сіздің орнатуыңызда қолжетімді және іске қосылған жағдайда ғана бұл жолды қолданыңыз. Сәлемдесу бағдарламасында OAuth ағыны белгілі бір мүмкіндік (feature flag) артында көрсетілген, сондықтан флагсыз желілер қолмен құжатты енгізу өрістерін пайдалануды жалғастырады.

Егер сіз сәлемдесу бағдарламасында OAuth опциясын көрмесеңіз, төмендегі қолмен құжатты енгізу процесін аялаңыз. Бұл шлюз бұрынғы Ultimate Multisite 2.x إص出ларындағы PayPal Business API куәлігімен жұмыс істейді.

## PayPal API куәлігін алу {#getting-the-paypal-api-credentials}

PayPal төлем шлюзы ретінде қосылған кезде, сізге PayPal API **Истемеші** (Username), PayPal API **Сыртқы пароль** (Password) және PayPal API **Қосындығы** (Signature) өрістерін толтыру қажет болады.

Бұл ақпаратты PayPal [Тіреуді](https://www.paypal.com/home) немесе [Sandbox](https://www.sandbox.paypal.com/home) есебіңізге кіріп енгізу арқылы алуға болады.

(Еске алыңыз, сіз төлемдерді сынау және шлюз дұрыс орнатылғанын көру үшін **sandbox режимін** қолдануға боласыз. Мәселелі бөлімді қосу-алу арқылы оны іске қосыңыз.)

![PayPal API куәлігі өрістері және sandbox режимі қосу/өшіру](https://img/config/settings-payment-gateways.png)

PayPal есебіңіз үшін API Қосындығын немесе Сертификат куәлігін сұрау үшін:

  1. [Есептік жағдайларға](https://www.paypal.com/businessmanage/account/accountAccess) өтіңіз.

  2. **API қолжетімділігі** (API access) бөлімінде **Жаңарту** (Update) батырмасын басыңыз.

3. **NVP/SOAP API интеграциясы (Classic)** бөліміне кіріп, **API құралдарының деректерін басқару**-ға басыңыз.
![PayPal NVP/SOAP API integration Manage API credentials](/img/config/settings-payment-gateways.png)

     * Егер сізде API Signature немесе Sertifikat жасалған болса, сіз деректерді таба алатын бетке ауыстырыласыз.

     * _**Назар аударыңыз:** Егер PayPal hesabınızı тексеруді сұраса, экрандағы нұсқауларды орындаңыз._

  4. Төмендегі опциялардың **бірін** таңдап, содан кейін **Келісу және жіберу**-ға басыңыз.

     * **API Signature сұрау** – API Signature аутентификациясы үшін таңдаңыз.

     * **API Sertifikat сұрау** – API Sertifikat аутентификациясы үшін таңдаңыз.

  5. PayPal сіздің API деректеріңізді келесідей генерациялайды:
![PayPal generated API credentials](/img/config/settings-payment-gateways.png)

     * **API Signature деректері** API İstifadəçi аты (API Username), API Параметрі (API Password) және Signature-дан тұрады, ол мерзімі жоқ. Қошимді қауіпсіздік үшін бұл мәндер бодры болады. Оларды қосу немесе өшіру үшін **Көрсету/Жасыру**-ға басыңыз. Біткенде **Қосу**-ға басыңыз.

     * **API Sertifikat деректері** API İstifadəçi аты (API Username), API Параметрі (API Password) және Сертификаттан тұрады, ол үш жылдан кейін автоматты түрде мерзімі аяқталады. API Сертификатын компьютерге сақтау үшін **Сертификатты жүктеу**-ға басыңыз.

Бұл бар! Сіздің PayPal төлем интеграцияңыз аяқталды!

PayPal параметрлеріне қатысты сұрақтарыңыз болса, сіз PayPal-дың [Қолдау орталығына](https://www.paypal.com/br/smarthelp/home) қарап көре аласыз.
