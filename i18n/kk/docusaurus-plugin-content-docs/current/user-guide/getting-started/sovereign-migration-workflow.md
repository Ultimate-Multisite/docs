# Мемлекеттік миграция құбылысы

Ultimate Multisite: Multi-Tenancy 1.2.0 стандартты субсайтты мемлекеттік арена инфрақұрылымына көшіру үшін миграцияды тексеру кезеңдерін қосты.

## Бастау алдында {#before-you-start}

Төмендегілердің бар екенін растаңыз:

- Деректердің бастапқы сайтының қазіргі бэкапы (backup).
- Multi-Tenancy addon-ы белсенді.
- Бағытталған арена дерекқор, файл жүйесінің түбі және домен дайын болуы керек.
- Бағытталған ортаға сәйкес келетін базалық хост байланысы (database host binding).
- Желі үшін WP-CLI командаларын іске қосуге қол жеткізу.

## Ұсынылатын құбылыс {#recommended-workflow}

1. Бағытталған арена дерекқор мен файл жүйесін дайындаңыз.
2. Ареналды изоляция параметрлерін тіркеңіз немесе жаңартыңыз.
3. Арена миграциясын іске қосыңыз.
4. Ареналды пайдаланушыларды құрыңыз немесе тексеріңіз.
5. Асинхрон миграция жұмыстарын толығымен шығарыңыз (drain).
6. Мемлекеттік миграцияны тексеруді іске қосыңыз.
7. SSO арқылы аренаға кіріңіз.
8. Тексерудан өткеннен кейін ғыснұрлықты немесе маршрутизацияны өзгертіңіз.

## Тексеру кезеңдері {#verification-gates}

Тексеру құбылысы миграцияны бірнеше бұрышынан тексеріп отырады:

- Арена схемасы бар және арена дерекқор жазушы арқылы жазылуға болады.
- Конфигурацияланған дерекқор хостыд дұрыс рұқсаттар (grants) болуы керек.
- Арена пайдаланушылары бар және күтілетін мемлекеттік орнату пайдаланушы санымен сәйкес келеді.
- Асинхрон жіберу тізімін (async push queue) сәтті толығымен шығаруға болады.
- Мемлекеттік арена үшін ескі желі жағындағы дерек жолдары артық болмауы керек.

Тексерудегі қателерді іске қосуден бұрын кедергі ретінде қарастырыңыз. Бағаланған дерекқор, пайдаланушы, тізім немесе маршрутизация мәселесін түзетіп, аренаны клиенттерге көрсету алдында тексеруді қайта іске қосыңыз.

## Бірінші өндірістікі визит {#first-production-visit}

Тексерудан өткеннен кейін, бірінші администраторлық визит үшін сайт басқару парағынан **Visit (SSO)**-ны пайдаланыңыз. Бұл арена маршрутизациясын, SSO токенді басқаруды, шығу нүктесін бекітуді және арена жағындағы пайдаланушыларды құруды бір бақыланатын қадаммен растайды.
