# Сайт Құрастыру Режимінен Ажырату Хабарламасы {#site-builder-mode-removal-notice}

**Site Builder режимі Superdav AI Agent v1.12.0-де жойылды.** Егер сіз Site Builder режимін қолдансаңыз, стиль жасау және сайтты орнату үшін **Setup Assistant agent**-ке көшіруіңіз керек.

## Не Ол Болды? {#what-happened}

### Site Builder Режимі (Ескі) {#site-builder-mode-legacy}

Site Builder режимі келесілер үшін wizard-based интерфейс болды:

- Шаблондардан сайттар жасау
- Негізгі параметрлерді реттеу
- Стиль таңдау
- Баста контентті орнату

### Оны Не Орнатып Жатты? {#what-replaced-it}

**Setup Assistant agent** енді келесі арқылы Site Builder функцияларының барлығын басқарады:

- Көбірек икемді, агент-бағытталған орнату
- Стильді реттеу үшін жақсырақ мүмкіндіктер
- Theme Builder-ге интеграциялау
- Болашақ сессиялар үшін `site_brief` деректерін сақтау

## Егер Сіз Site Builder Режимін Қолданып Отсаңыз {#if-you-were-using-site-builder-mode}

### Сайттарыңыз Қауіпсіз {#your-sites-are-safe}

- Site Builder режимімен жасалған барлық сайттар жұмыс істеуде болады
- Деректердің жоғалуы немесе сайттың бұзылуы жоқ
- Сіз сайттарыңызды қалыпты түрде басқаруды жалғастыра аласыз

### Setup Assistant Agent-ке Көшіріңіз {#migrate-to-setup-assistant-agent}

Жаңа сайт орнату немесе стильді өзгерту үшін Setup Assistant agent-ті қолданыңыз:

```
"Help me set up a new site"
```

немесе

```
"Start the Theme Builder onboarding"
```

Setup Assistant agent сол функцияларды қаттырақ икемді түрде ұсынады.

## Салыстырмалы Талдау: Site Builder vs. Setup Assistant {#comparison-site-builder-vs-setup-assistant}

| Ерекшелік | Site Builder (Жойылған) | Setup Assistant (Жаңа) |
|---------|----------------------|----------------------|
| Орнату әдісі | Wizard формасы | Agent сөйлесуі |
| Стиль таңдау | Алдын ала анықталған шаблондар | Қолданылған генерация |
| Реттеу | Шекаралық мүмкіндіктер | Толық дизайн жүйесі |
| Сайт сипаттамасы | Сақталмады | Ұзақ сақтау памятьі |
| Болашақ сессиялар | Қайта орнату | Сақталған `site_brief`-ті қолдану |
| Икемділік | Белгілі бір құрылым | Түсінікті сөйлесу |

## Setup Assistant Agent-ке Көшіру {#migrating-to-setup-assistant-agent}

### Жаңа Сайттар Үшін {#for-new-sites}

Майкелей Сайт Құрастыру режимін қолданбаңыз:

1. Сұраныс: "Жаңа сайт құруға көмектесіңіз"
2. Құрылымдау ассистент агенті сізге келесілер арқылы бағыт береді:
   - Сайттың мақсаты мен міндеттері
   - Нақты аудиторияңыз
   - Брендтің тұжырымдары
   - Теманы генерациялау
   - Бастапқы конфигурация

### Қазіргі сайттар үшін {#for-existing-sites}

Егер сізде Site Builder режимінен жасалған бар сайт болса:

1. Оны қазіргідей қолдануға жалғастыруға болады
2. Теманы жаңарту үшін сұраныңыз: "Сайтты қайта дизайнын жасаңыз"
3. Құрылымдау ассистент агенті сізге жаңа теманы жасауға көмектеседі
4. Сіздің сайт деректері өзгермейді

### Тема өзгерістері үшін {#for-theme-changes}

Site Builder режимінің тема таңдауының орнына:

1. Сұраныс: "Теманы өзгерт"
2. Құрылымдау ассистент агенті:
   - Сіздің дизайн қалауларыңыз туралы сұрақ қояды
   - Қолданылған (custom) тему генерациялайды
   - Оны сіздің сайтыңызда белсендіреді

## Негізгі айырмашылықтар {#key-differences}

### Site Builder Режимі {#site-builder-mode}

```
1. Шаблон таңдау
2. Теманы таңдау
3. Басты параметрлерді реттеу
4. Бітті
```

### Құрылымдау Ассистент Агенті {#setup-assistant-agent}

```
1. Сайттың мақсатын сипаттаңыз
2. Нақты аудиторияңызды анықтаңыз
3. Дизайн қалауларыңызды таңдаңыз
4. Агент қолданылған (custom) тему генерациялайды
5. Агент темуны белсендіреді
6. Болашақ сессиялар үшін сайт қысқаша сипаттама сақталады
```

## Құрылымдау Ассистент Агентінің артықшылықтары {#benefits-of-setup-assistant-agent}

### Көбірек икемділік {#more-flexible}

- Сайтты табиғи тілде сипаттаңыз
- Қолданылған ұсыныстарды алыңыз
- Сіздің нақты қажеттіліктеріңізге бейімделу

### Жақсы реттеу мүмкіндігі {#better-customization}

- Қолданылған (custom) тема генерациялау
- Дизайн жүйесінің шешімдерін жасау
- Тұрақты дизайн токендері

### Тұрақты жад {#persistent-memory}

- Сіздің site_brief сақталады
- Болашақ агенттер сіздің сайтыңызды түсінеді
- Құрылымдау ақпараттарын қайта айтудың қажеті жоқ

### Бірлескен жұмыс істеу {#integrated-workflow}

- Тема құру кезіндегі onboarding (үйрену)
- Дизайн жүйесінің эстетикасы дағдысы
- Көру бақылауларын қолдану мүмкіндігі
- Барлығы үздіксіз бірлесе жұмыс істейді

## Қателіктерді шешу {#troubleshooting}

### Site Builder режимін таба алмаймын {#i-cant-find-site-builder-mode}

Site Builder режимі жойылды. Оның орнына Құрылымдау ассистент агентін қолданыңыз:

"Мені жаңа сайт орнатуға көмектесіңіз"

### I want to recreate a site from Site Builder (Сайт құрастырудан сайтты қайта жасағым келеді) {#i-want-to-recreate-a-site-from-site-builder}

Оны Setup Assistant агенті арқылы қайта жасауға болады:

1. Сұраныс: "Мен жаңа сайт орнатуға көмектесіңіз"
2. Сіздің бастапқы сайтыңыздың мақсаты мен дизайнын сипаттаңыз
3. Агент ұқсас теманы генерациялайды
4. Сіздің site_brief-і болашақта сілтеу үшін сақталады

### My existing Site Builder site isn't working (Қазіргі Site Builder сайтым жұмыс істемейді) {#my-existing-site-builder-site-isnt-working}

Site Builder режимімен жасалған барлық сайттар әлі жұмыс істейді. Егер сіз мәселелер туындасаңыз:

1. Темаңыздың әлі белсенді екенін тексеріңіз
2. Плагиндердің қосылғанын растаңыз
3. WordPress қате журналдарын (error logs) тексеріңіз
4. Мәселелер баса көрсетілсе, қолдаумен байланысыңыз

### Can I still use my old Site Builder templates? (Ескі Site Builder үлгілерімді әлі қолдануға болады ма?) {#can-i-still-use-my-old-site-builder-templates}

Site Builder үлгілері енді қолжетімді емес. Алайда:

- Сіздің барлық сайттарыңыз жұмыс істеуде
- Setup Assistant агенті арқылы ұқсас сайттарды қайта жасауға болады
- Setup Assistant агенті көбірек мүмкіндіктерді ұсынады

## Next Steps (Келесі қадамдар) {#next-steps}

1. **Жаңа сайттар үшін**: Setup Assistant агентін қолданыңыз
2. **Қазіргі сайттар үшін**: Оларды өз күйінде қоллауды жалғастырыңыз
3. **Теманы өзгерту үшін**: Setup Assistant агентінен көмек сұраңыз
4. **Дизайнды жетілдіру үшін**: Design System Aesthetics skill-ді пайдаланыңыз
