# Ultimate Multisite орнату

:::note
Бұл нұсқаулық WordPress Multisite орнатылған және конфигурацияланған болуын алады деп есептейді. Бұл туралы білу үшін WP Beginner-дің [бұл нұсқаулығын](https://www.wpbeginner.com/glossary/multisite/) қараңыз.
:::

## Плагин орнату

Ultimate Multisite WordPress.org-та тегін қолжетімді.

**Network Admin Dashboard**-ынан **Plugins → Add New Plugin**-ге өтіңіз.

![Network Admin Add New Plugin page](/img/installation/add-new-plugin.png)

**"Ultimate Multisite"** деп іздеңіз (нақты сәйкестендіру үшін кавырша қолданыңыз), ол бірінші нәтиже ретінде шығады. **Install Now**-ға басыңыз.

![Search results showing Ultimate Multisite](/img/installation/search-ultimate-multisite.png)

Орнатылғаннан кейін, плагинді барлық желіде іске қосу үшін **Network Activate**-қа басыңыз.

![Plugin installed with Network Activate button](/img/installation/plugin-installed.png)

Іске қосылғаннан кейін сіз автоматты түрде Нақтылау Wizard-ына (Setup Wizard) ауысасыз.

![Plugin activated and redirected to wizard](/img/installation/plugin-activated.png)

## Нақтылау Wizard

Нақтылау Wizard сізге Ultimate Multisite-ты шамамен 10 минут ішінде конфигурациялауға көмектеседі.

### Қош келдіңіз

Бастау үшін **Get Started**-ға басыңыз.

![Setup Wizard welcome screen](/img/installation/wizard-welcome.png)

### Орнату алдындағы тексерулер

Бұл қадам жүйе ақпаратты және WordPress орнатылуын Ultimate Multisite талаптарына сай екенін тексеріп, сізге көмектеседі. Еңбек барлық жақсы болса, **Go to the Next Step**-ке басыңыз.

![Pre-install checks showing system requirements](/img/installation/wizard-pre-install-checks.png)

:::note Network Activate баты (v2.6.1+)
Егер Ultimate Multisite орнатыл болса, бірақ **ең алдында желіге белсендірілмесе** — мысалы, егер сіз желі плагині экраннан **Network Activate**-ті емес, **Activate** (бір сайт) батымен басқан болса — Pre-install Checks қадамы бұл жағдайды анықтайды және **Network Activate** батын көрсетеді.

**Network Activate**-қа басу плагинді сіздің барлық мультисайт желіңізде автоматты түрде белсендіретін болады. Белсендірілгеннен кейін, wizard (сұрақтар тізбегі) орнату кезеңіне қарай қалыпты жалғасады. Белсендіру күйін түзету үшін wizard-дан шығудың қажеті жоқ.
:::

### Орнату

Инсталлятор қажетті базалық деректер кестелерін жасайды және Ultimate Multisite жұмыс істеуі үшін `sunrise.php` файлын орнатады. Жобалауды жалғастыру үшін **Install**-ға басыңыз.

![Орнату кезеңін көрсететін сурет базалық деректер кестелері мен sunrise.php](https://img/installation/wizard-installation.png)

### Сіздің Компанияңыз

Компанияңыз туралы ақпаратты толтырыңыз және белгілі бір валютаны (default currency) белгілеңіз. Бұл ақпарат WaaS платформаңыздағы барлық уақытта қолданылады. Біткенде **Continue**-қа басыңыз.

![Сіздің компанияңызды конфигурациялау кезеңі](https://img/installation/wizard-your-company.png)

### Деңгейлі Мазмұн (Default Content)

Бұл қадамда сізге алдын ала анықталған шаблондарды, өнімдерді және басқа да бастапқы контентті орната аласыз. Бұл Ultimate Multisite мүмкіндіктерімен танысудың керемет жолы. Деңгейлі контентті қосу үшін **Install**-ға басыңыз немесе егер сіз толық жаңадан бастағыңыз келсе бұл қадамы өткізіп жіберіңіз.

![Деңгейлі контент орнату кезеңі](https://img/installation/wizard-default-content.png)

### Ұсынылатын Плагиндер (Recommended Plugins)

Қосымша ұсынылған плагиндерді опционалды түрде орната аласыз. Оларды қосу үшін **Install**-ға басыңыз немесе жалғасу үшін өткізіп жіберіңіз.

![Ұсынылатын плагиндер кезеңі](https://img/installation/wizard-recommended-plugins.png)

### Дайын!

Бұл барлығы дайын! Сіздің Ultimate Multisite орнатуыңыз аяқталды. Енді **Network Admin Dashboard** арқылы Web as a Service платформасын құруға бастай аласыз.

![Setup complete - Ready screen](/img/installation/wizard-ready.png)

![Network Admin Dashboard with Ultimate Multisite active](/img/installation/network-dashboard.png)

Енді қызығып кетуге болады!
