# Αλλαγή σειράς στη λίστα χωρών του Ultimate Multisite

Το αναπτυσσόμενο μενού επιλογής χώρας είναι ταξινομημένο αλφαβητικά με βάση τα αγγλικά ονόματα των χωρών. Αυτό σημαίνει ότι, μερικές φορές, η χώρα έδρας της εταιρείας σας ή/και των περισσότερων πελατών σας μπορεί να είναι θαμμένη στη μέση μιας τεράστιας λίστας χωρών.

Για να το διορθώσετε και να προσθέσετε τη χώρα της επιλογής σας στην κορυφή της λίστας, μπορείτε να χρησιμοποιήσετε το παρακάτω απόσπασμα κώδικα (μπορείτε να το προσθέσετε σε ένα mu-plugin):

add_filter('wu_get_countries', function($countries) {

// Αλλάξτε το DE στον διψήφιο κωδικό της χώρας σας

// Παράδειγμα παρακάτω για τη Γερμανία.

unset($countries['DE']);

// Αλλάξτε το DE στον διψήφιο κωδικό της χώρας σας

// και το όνομα της χώρας σας.

// Παράδειγμα παρακάτω, για τη Γερμανία.

$countries = array_merge(array( 'DE' => __('Germany', 'ultimate-multisite'), ), $countries);

return $countries;

});
