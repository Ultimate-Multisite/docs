# Μετάφραση του Ultimate Multisite

Το Ultimate Multisite αναπτύσσεται στα Αγγλικά, αλλά το plugin είναι πλήρως μεταφράσιμο και καλωσορίζουμε ενεργά τις συνεισφορές μετάφρασης.

## Μεταφράσεις Βασικού Plugin

Το βασικό plugin Ultimate Multisite μεταφράζεται μέσω της επίσημης **πλατφόρμας μετάφρασης του WordPress.org**:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

Αυτό είναι το τυπικό σύστημα μετάφρασης της κοινότητας του WordPress. Όταν οι μεταφράσεις φτάσουν ένα επαρκές όριο έγκρισης, διανέμονται αυτόματα σε όλους τους χρήστες του Ultimate Multisite μέσω των ενημερώσεων plugin του WordPress.

**Χρειαζόμαστε τη βοήθειά σας!** Αν μιλάτε μια γλώσσα εκτός των Αγγλικών, σκεφτείτε να συνεισφέρετε μεταφράσεις. Ακόμα και η μετάφραση λίγων συμβολοσειρών βοηθάει. Για να ξεκινήσετε:

1. Επισκεφτείτε το [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)
2. Επιλέξτε τη γλώσσα σας
3. Κάντε κλικ στο "Contribute Translation"
4. Θα χρειαστείτε έναν δωρεάν λογαριασμό WordPress.org

Αν θέλετε να γίνετε επιμελητής μετάφρασης για τη γλώσσα σας (με δυνατότητα έγκρισης μεταφράσεων), επικοινωνήστε μέσω του [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions).

## Μεταφράσεις Addons

Τα addon plugins μεταφράζονται ξεχωριστά στο:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Οι μεταφράσεις των addons είναι προς το παρόν δημιουργημένες με τεχνητή νοημοσύνη ως σημείο εκκίνησης. Καλύπτουν όλες τις υποστηριζόμενες γλώσσες, αλλά μπορεί να περιέχουν άβολη διατύπωση ή ανακρίβειες. Καλωσορίζουμε ανθρώπινους μεταφραστές να τις ελέγξουν και να προτείνουν βελτιώσεις ώστε να γίνουν πιο φυσικές και ακριβείς.

## Τοπικές Μεταφράσεις

Αν προτιμάτε να δημιουργήσετε μεταφράσεις τοπικά για το δικό σας έργο αντί να συνεισφέρετε στην κεντρική έκδοση, υπάρχουν δύο επιλογές:

### Χρήση του Poedit

Μπορείτε να χρησιμοποιήσετε την εφαρμογή επιφάνειας εργασίας **[Poedit](https://poedit.net)** για να μεταφράσετε το αρχείο `.pot`:

1. Κατεβάστε το Poedit από το [poedit.net](https://poedit.net)
2. Ανοίξτε το αρχείο `.pot` από το `wp-content/plugins/ultimate-multisite/lang/`
3. Μεταφράστε τις συμβολοσειρές στη διεπαφή του Poedit
4. Αποθηκεύστε -- το Poedit δημιουργεί αρχεία `.po` και `.mo`
5. Τοποθετήστε τα αρχεία εξόδου στον φάκελο `lang`

### Χρήση του Loco Translate

Το plugin **[Loco Translate](https://wordpress.org/plugins/loco-translate/)** για WordPress σας επιτρέπει να μεταφράζετε απευθείας από τον πίνακα διαχείρισης χωρίς εργαλεία επιφάνειας εργασίας. Αναλαμβάνει τη δημιουργία και τη μεταγλώττιση των αρχείων μετάφρασης για εσάς.
