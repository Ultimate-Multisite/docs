# Συχνές Ερωτήσεις

Ακολουθούν μερικές **Συχνές Ερωτήσεις** που εμφανίζονται συχνά κατά τη διάρκεια αιτημάτων υποστήριξης.

  * **Τι είναι το Ultimate Multisite;**

Το Ultimate Multisite είναι ένα plugin για WordPress, σχεδιασμένο για εγκαταστάσεις WordPress multisite, που μετατρέπει την εγκατάσταση WordPress σας σε ένα premium δίκτυο ιστοσελίδων – όπως το [WordPress.com](https://WordPress.com) – επιτρέποντας στους πελάτες να δημιουργούν ιστοσελίδες με μηνιαίες, τριμηνιαίες ή ετήσιες χρεώσεις (μπορείτε επίσης να δημιουργήσετε Δωρεάν πακέτα).

  * **Θα λειτουργήσει το Ultimate Multisite χωρίς Multisite;**

Όχι. Το Ultimate Multisite απαιτεί WordPress multisite, με υποφακέλους ή υποτομείς.

  * **Ποιες είναι οι απαιτήσεις για το Ultimate Multisite;**

Το Ultimate Multisite απαιτεί:

**Έκδοση WordPress** : v5.3+ (Συνιστάται: Τελευταία Σταθερή Έκδοση)  
**Έκδοση PHP** : 7.4.x (Υποστήριξη για > 8.0 έρχεται σύντομα)  
**Έκδοση MySQL** : v5+ (Συνιστάται: Έκδοση 5.6, σε περίπτωση που δεν έχετε διαθέσιμη την 8.0 στον πάροχο φιλοξενίας σας)

_Το Multisite (υποτομέας ή υποφάκελος) πρέπει επίσης να είναι ενεργοποιημένο._ [Πώς να εγκαταστήσετε το WordPress Multisite](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **Θα λειτουργήσει το Ultimate Multisite με Multisite με Υποφακέλους;**

Ναι. Το Ultimate Multisite υποστηρίζει τόσο εγκαταστάσεις με υποφακέλους όσο και εγκαταστάσεις με υποτομείς.

  * **Θα λειτουργήσει το Ultimate Multisite με Multisite με Υποτομείς;**

Ναι. Το Ultimate Multisite υποστηρίζει τόσο εγκαταστάσεις με υποφακέλους όσο και εγκαταστάσεις με υποτομείς.

  * **Μπορώ να χρησιμοποιήσω το Ultimate Multisite για να περιορίσω ποια Plugins και Θέματα μπορούν να χρησιμοποιήσουν οι χρήστες;**

Ναι. Με το Ultimate Multisite μπορείτε να επιλέξετε ποια plugins και θέματα είναι διαθέσιμα σε κάθε πακέτο στο δίκτυό σας.

  * **Μπορώ να χρησιμοποιήσω το Ultimate Multisite για να περιορίσω τον αριθμό των δημοσιεύσεων που μπορεί να δημιουργήσει ένας χρήστης στην ιστοσελίδα του;**

Ναι. Με το Ultimate Multisite μπορείτε να ορίσετε όρια δημοσιεύσεων για **κάθε διαθέσιμο τύπο περιεχομένου** στο δίκτυό σας. Μπορείτε επίσης να περιορίσετε τον αριθμό των χρηστών που μπορεί να προσκαλέσει ο διαχειριστής, τον **διαθέσιμο χώρο αποθήκευσης** και πολλά άλλα.

  * **Υποστηρίζει το Ultimate Multisite Πρότυπα Ιστοσελίδων;**

Ναι. Με το Ultimate Multisite μπορείτε να δημιουργήσετε όσα Πρότυπα Ιστοσελίδων θέλετε. Τα Πρότυπα Ιστοσελίδων δεν είναι θέματα, είναι κανονικές ιστοσελίδες στο δίκτυό σας. Μπορείτε να ενεργοποιήσετε plugins, θέματα και να προσαρμόσετε αυτό το Πρότυπο Ιστοσελίδας με όποιον τρόπο θέλετε. Αν ένας χρήστης επιλέξει αυτό το πρότυπο, η ιστοσελίδα του θα δημιουργηθεί ως αντίγραφο αυτού του προτύπου, με όλες τις προσαρμογές στη θέση τους.

Μπορείτε επίσης να ορίσετε ένα συγκεκριμένο πρότυπο για ένα πακέτο. Με αυτόν τον τρόπο, κάθε χρήστης που εγγράφεται σε αυτό το πακέτο θα έχει την ιστοσελίδα του δημιουργημένη ως αντίγραφο αυτού του Προτύπου Ιστοσελίδας.

  * **Ενσωματώνεται το Ultimate Multisite με το WooCommerce;**

Ναι!

Αν θέλετε οι χρήστες του δικτύου σας να έχουν διαθέσιμο το WooCommerce, επιτρέποντάς τους να δημιουργήσουν καταστήματα, αυτό είναι δυνατό με το Ultimate Multisite. Μπορείτε ακόμη και να περιορίσετε τον αριθμό των προϊόντων που θα μπορούν να δημιουργήσουν οι πελάτες σας σε κάθε επίπεδο πακέτου.

Όσον αφορά τις πληρωμές συνδρομών για το Ultimate Multisite, μπορείτε να χρησιμοποιήσετε το [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) για να επιτρέψετε στους πελάτες του Ultimate Multisite να πληρώνουν χρησιμοποιώντας τις πύλες πληρωμών του WooCommerce στην κύρια ιστοσελίδα σας. Με αυτόν τον τρόπο, μπορείτε να αξιοποιήσετε την τεράστια βιβλιοθήκη πυλών πληρωμών που έχουν αναπτυχθεί για το WooCommerce και να τις χρησιμοποιήσετε για να χρεώνετε τους πελάτες του Ultimate Multisite.

  * **Υπάρχει τρόπος να μεταφέρω συνδρομές από το WPMUDEV Pro Sites;**

Ναι, το Ultimate Multisite σας προσφέρει πλέον [ένα χρήσιμο εργαλείο για να σας βοηθήσει να μεταφέρετε](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) το δίκτυο Pro Sites σας.

  * **Γιατί η καρτέλα "Επιτρεπόμενα Plugins" είναι άδεια στη σελίδα Επεξεργασίας Πακέτου;**

Το Ultimate Multisite σας επιτρέπει να επιτρέψετε ή να απορρίψετε πρόσβαση μόνο σε plugins που δεν είναι ενεργοποιημένα σε επίπεδο δικτύου. Όταν ένα plugin είναι ενεργοποιημένο σε επίπεδο δικτύου, από τη φύση του, θα φορτώνεται από προεπιλογή σε κάθε ιστοσελίδα του δικτύου σας.

Αν δεν βλέπετε κανένα plugin στην καρτέλα "Επιτρεπόμενα Plugins", ελέγξτε αν το επιθυμητό plugin είναι ενεργοποιημένο σε επίπεδο δικτύου ή όχι.

  * **Μπορώ να χρησιμοποιήσω τη δική μου σχεδίαση Πίνακα Τιμολόγησης;**

Ναι. Από την έκδοση 1.4.0, το Ultimate Multisite υποστηρίζει παράκαμψη προτύπων.

  * **Λειτουργεί το Ultimate Multisite με το WP Engine;**

Ναι. Το Ultimate Multisite υποστηριζόταν πάντα από το WP Engine, αλλά η λειτουργία αντιστοίχισης τομέα απαιτούσε επιπλέον χειροκίνητα βήματα από τον διαχειριστή (προσθήκη του τομέα στον πίνακα του WP Engine). **Αυτό δεν ισχύει πλέον από την έκδοση 1.6.0.** Το Ultimate Multisite ενσωματώνεται απευθείας με το API του WP Engine και οι τομείς προστίθενται αυτόματα στον πίνακα μετά την αντιστοίχιση.

  * **Ποιον πάροχο φιλοξενίας προτείνετε;**

Το Ultimate Multisite θα πρέπει να λειτουργεί με οποιονδήποτε πάροχο φιλοξενίας. Η μόνη διαφορά είναι ότι κάποιες εταιρείες φιλοξενίας όπως οι Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane και [Runcloud.io](http://Runcloud.io) έχουν πιο στενή ενσωμάτωση για τις δυνατότητες αντιστοίχισης τομέα, οπότε λειτουργούν χωρίς επιπλέον ρυθμίσεις. Για άλλες φιλοξενίες, μπορεί να χρειαστεί να ρυθμίσετε την αντιστοίχιση τομέα χειροκίνητα.
