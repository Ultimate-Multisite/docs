# Αποστολή Email και Ανακοινώσεων (v2)

_**ΣΗΜΑΝΤΙΚΗ ΣΗΜΕΙΩΣΗ: Αυτό το άρθρο αναφέρεται στην έκδοση 2.x του Ultimate Multisite.**_

Το Ultimate Multisite διαθέτει μια λειτουργία που σας επιτρέπει να επικοινωνείτε με τους πελάτες σας στέλνοντας email σε συγκεκριμένους χρήστες ή σε ομάδες χρηστών, καθώς και να εμφανίζετε ειδοποιήσεις στο admin dashboard τους για τη μετάδοση ανακοινώσεων.

## Προσθέστε ειδοποιήσεις διαχειριστή στο dashboard των πελατών σας με τις Ανακοινώσεις

Χρησιμοποιώντας τη λειτουργία ανακοινώσεων του Ultimate Multisite, μπορείτε να προσθέσετε **ειδοποιήσεις διαχειριστή** στο admin dashboard του subsite των χρηστών σας.

Αυτό είναι εξαιρετικά χρήσιμο όταν χρειάζεται να κάνετε μια ανακοίνωση, όπως για συντήρηση συστήματος ή για να προσφέρετε νέα προϊόντα ή υπηρεσίες στους υπάρχοντες χρήστες σας. Έτσι θα εμφανίζεται η ειδοποίηση διαχειριστή στο dashboard του χρήστη σας.

![Ειδοποίηση ανακοίνωσης που εμφανίζεται στο dashboard του πελάτη](/img/admin/broadcasts-list.png)

Για να ξεκινήσετε μια ειδοποίηση διαχειριστή, μεταβείτε στο network admin dashboard και κάτω από το μενού **Ultimate Multisite**, θα βρείτε την επιλογή **Broadcasts**.

![Μενού Broadcasts στο admin του Ultimate Multisite](/img/admin/broadcasts-list.png)

Από αυτή τη σελίδα, κάντε κλικ στο κουμπί **Add Broadcast** στην κορυφή.

Αυτό θα εμφανίσει το παράθυρο Add broadcast όπου μπορείτε να επιλέξετε τον τύπο ανακοίνωσης που θέλετε να στείλετε.

Επιλέξτε **Message** και κάντε κλικ στο κουμπί **Next Step**.

![Παράθυρο Add broadcast με επιλεγμένο τον τύπο Message](/img/admin/broadcasts-list.png)

Το επόμενο παράθυρο θα σας ζητήσει είτε τον **Target customer** είτε το **Target product**. Σημειώστε ότι μπορείτε να επιλέξετε περισσότερους από έναν χρήστες ή περισσότερα από ένα προϊόντα.

Για να αναζητήσετε είτε λογαριασμό χρήστη είτε προϊόν, πρέπει να αρχίσετε να πληκτρολογείτε τη λέξη-κλειδί μέσα στο πεδίο.

Στο πεδίο **Message type**, μπορείτε να επιλέξετε το χρώμα της ειδοποίησης. Αυτό θα τονίσει τον επείγοντα χαρακτήρα του μηνύματός σας.

Στη συνέχεια, μπορείτε να κάνετε κλικ στο **Next Step**.

![Επιλογή στοχευμένου πελάτη και προϊόντος για την ανακοίνωση](/img/admin/broadcasts-list.png)

Το επόμενο παράθυρο είναι εκεί όπου μπορείτε να συντάξετε το μήνυμά σας εισάγοντας το θέμα και το περιεχόμενο/μήνυμα που θέλετε να μεταδώσετε στους χρήστες.

![Επεξεργαστής θέματος και περιεχομένου ανακοίνωσης](/img/admin/broadcasts-list.png)

Αφού δημιουργήσετε το μήνυμά σας, μπορείτε να πατήσετε το κουμπί **Send**.

Και αυτό ήταν. Η ειδοποίηση διαχειριστή θα εμφανιστεί αμέσως στο dashboard του χρήστη σας.

## Αποστολή email στους πελάτες σας

Χρησιμοποιώντας τη λειτουργία ανακοινώσεων του Ultimate Multisite, μπορείτε να στείλετε email στους χρήστες σας. Έχετε την επιλογή να στείλετε το email μόνο σε συγκεκριμένους χρήστες ή να στοχεύσετε μια συγκεκριμένη ομάδα χρηστών βάσει του προϊόντος ή του πλάνου στο οποίο είναι εγγεγραμμένοι.

Για να ξεκινήσετε μια αποστολή email, μεταβείτε στο network admin dashboard και κάτω από το μενού Ultimate Multisite, θα βρείτε την επιλογή Broadcast.

![Σελίδα Broadcasts στο admin του Ultimate Multisite](/img/admin/broadcasts-list.png)

Από αυτή τη σελίδα, κάντε κλικ στο κουμπί **Add broadcast** στην κορυφή.

Αυτό θα εμφανίσει το παράθυρο Add broadcast όπου μπορείτε να επιλέξετε τον τύπο ανακοίνωσης που θέλετε να στείλετε. Επιλέξτε **Email** και κάντε κλικ στο κουμπί **Next Step**.

![Παράθυρο Add broadcast με επιλεγμένο τον τύπο Email](/img/admin/broadcasts-list.png)

Το επόμενο παράθυρο θα σας ζητήσει είτε τον **Target customer** είτε το **Target product**. Σημειώστε ότι μπορείτε να επιλέξετε περισσότερους από έναν χρήστες ή περισσότερα από ένα προϊόντα.

Για να αναζητήσετε είτε λογαριασμό χρήστη είτε προϊόν, πρέπει να αρχίσετε να πληκτρολογείτε τη λέξη-κλειδί μέσα στο πεδίο.

Αφού επιλέξετε το κοινό-στόχο σας, μπορείτε να κάνετε κλικ στο **Next Step**.

![Επιλογή στοχευμένου πελάτη και προϊόντος για αποστολή email](/img/admin/broadcasts-list.png)

Το επόμενο παράθυρο είναι εκεί όπου μπορείτε να συντάξετε το email σας εισάγοντας το θέμα και το περιεχόμενο/μήνυμα που θέλετε να στείλετε στους χρήστες.

![Επεξεργαστής θέματος και περιεχομένου email](/img/admin/broadcasts-list.png)

Αφού δημιουργήσετε το μήνυμά σας, μπορείτε να πατήσετε το κουμπί **Send**.

Και έτσι απλά μπορείτε να στείλετε email στους τελικούς χρήστες σας χρησιμοποιώντας τη λειτουργία ανακοινώσεων.

## Email συστήματος

Τα email συστήματος στο Ultimate Multisite είναι εκείνες οι **αυτόματες ειδοποιήσεις** που στέλνονται από το σύστημα μετά από συγκεκριμένες ενέργειες όπως εγγραφή, πληρωμή, αντιστοίχιση domain κ.λπ. Αυτά τα email μπορούν να επεξεργαστούν ή να τροποποιηθούν από τις ρυθμίσεις του Ultimate Multisite. Διαθέτει επίσης μια λειτουργία που σας επιτρέπει να επαναφέρετε και να εισάγετε υπάρχουσες ρυθμίσεις από άλλη εγκατάσταση Ultimate Multisite.

### Επαναφορά & Εισαγωγή

Οι νέες εκδόσεις του Ultimate Multisite, καθώς και τα add-ons, μπορούν και θα καταχωρούν νέα email κατά καιρούς.

Για να αποφευχθούν συγκρούσεις και άλλα προβλήματα, **δεν θα προσθέσουμε αυτόματα τα νέα πρότυπα email ως Email Συστήματος στην εγκατάστασή σας**, εκτός αν είναι κρίσιμα για τη σωστή λειτουργία μιας συγκεκριμένης δυνατότητας.

Ωστόσο, οι super admins και οι agents μπορούν να εισάγουν αυτά τα νεοκαταχωρημένα email μέσω του εργαλείου εισαγωγής. Αυτή η διαδικασία θα δημιουργήσει ένα νέο email συστήματος με το περιεχόμενο και τη διαμόρφωση του νέου προτύπου email, επιτρέποντας στον super admin να κάνει οποιεσδήποτε τροποποιήσεις θέλει ή να τα διατηρήσει ως έχουν.

#### Πώς να εισάγετε email συστήματος

Μεταβείτε στη σελίδα Ρυθμίσεων του Ultimate Multisite και πηγαίνετε στην καρτέλα **Emails**.

![Καρτέλα Emails στις ρυθμίσεις του Ultimate Multisite](/img/config/settings-emails.png)

Στη συνέχεια, στην πλαϊνή μπάρα, κάντε κλικ στο κουμπί **Customize System Emails**.

![Κουμπί Customize System Emails στην πλαϊνή μπάρα](/img/config/settings-emails.png)

Στη σελίδα System Emails, θα δείτε το κουμπί ενέργειας **Reset & Import** στην κορυφή. Κάνοντας κλικ σε αυτό το κουμπί θα ανοίξει το παράθυρο εισαγωγής και επαναφοράς.

![Κουμπί ενέργειας Reset and Import στη σελίδα System Emails](/img/config/settings-emails.png)

Στη συνέχεια, μπορείτε να ενεργοποιήσετε τις επιλογές Import Emails για να δείτε ποια email συστήματος είναι διαθέσιμα για εισαγωγή.

![Επιλογές Import Emails που δείχνουν τα διαθέσιμα email συστήματος](/img/config/settings-emails.png)

#### Επαναφορά Email Συστήματος

Άλλες φορές, θα διαπιστώσετε ότι οι αλλαγές που κάνατε σε ένα συγκεκριμένο πρότυπο email δεν σας εξυπηρετούν πλέον και θα θέλατε να το επαναφέρετε στην **προεπιλεγμένη κατάσταση**.

Σε τέτοιες περιπτώσεις, έχετε δύο επιλογές: μπορείτε απλά να διαγράψετε το email συστήματος και να το εισάγετε ξανά (χρησιμοποιώντας τις παραπάνω οδηγίες) - κάτι που θα διαγράψει τα στατιστικά αποστολής και άλλα στοιχεία, γι' αυτό αυτή η μέθοδος δεν προτιμάται.

Ή μπορείτε να χρησιμοποιήσετε το **εργαλείο Reset & Import** για να επαναφέρετε αυτό το πρότυπο email.

Για να επαναφέρετε ένα πρότυπο email, μπορείτε να ακολουθήσετε τα παραπάνω βήματα μέχρι να φτάσετε στο εργαλείο Reset & Import, και στη συνέχεια, ενεργοποιήστε την επιλογή **Reset** και επιλέξτε τα email που θέλετε να επαναφέρετε στο προεπιλεγμένο περιεχόμενό τους.

![Επιλογή Reset για επαναφορά προτύπων email στις προεπιλογές](/img/config/settings-emails.png)
