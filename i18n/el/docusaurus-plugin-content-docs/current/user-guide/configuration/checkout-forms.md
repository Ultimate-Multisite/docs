# Φόρμες Ολοκλήρωσης Αγοράς

Οι Φόρμες Ολοκλήρωσης Αγοράς είναι ένας εύκολος και ευέλικτος τρόπος για να πειραματιστείτε με διαφορετικές προσεγγίσεις όταν προσπαθείτε να μετατρέψετε νέους πελάτες.

Το Ultimate Multisite 2.0 προσφέρει έναν επεξεργαστή Φορμών Ολοκλήρωσης Αγοράς που σας επιτρέπει να δημιουργήσετε όσες φόρμες θέλετε, με διαφορετικά πεδία, προϊόντα προς πώληση κ.λπ.

Για να αποκτήσετε πρόσβαση σε αυτή τη λειτουργία, μεταβείτε στο μενού Checkout Forms, στην αριστερή πλαϊνή μπάρα.

![Λίστα Φορμών Ολοκλήρωσης Αγοράς](/img/config/checkout-forms-list.png)

Σε αυτή τη σελίδα, μπορείτε να δείτε όλες τις φόρμες ολοκλήρωσης αγοράς που έχετε.

Αν θέλετε να δημιουργήσετε μια νέα, απλά κάντε κλικ στο Add Checkout Form στο πάνω μέρος της σελίδας.

Μπορείτε να επιλέξετε μία από αυτές τις τρεις επιλογές ως αφετηρία: single step, multi-step ή blank. Στη συνέχεια, κάντε κλικ στο Go to the Editor.

![Επεξεργαστής Φόρμας Ολοκλήρωσης Αγοράς](/img/config/checkout-form-editor.png)

Εναλλακτικά, μπορείτε να επεξεργαστείτε ή να αντιγράψετε τις φόρμες που έχετε ήδη κάνοντας κλικ στις επιλογές κάτω από το όνομά τους. Εκεί, θα βρείτε επίσης τις επιλογές για να αντιγράψετε το shortcode της φόρμας ή να διαγράψετε τη φόρμα.

![Ενέργειες κατά την αιώρηση στη φόρμα ολοκλήρωσης αγοράς](/img/config/checkout-form-hover-actions.png)

### Επεξεργασία μιας Φόρμας Ολοκλήρωσης Αγοράς

Μπορείτε να δημιουργήσετε φόρμες ολοκλήρωσης αγοράς για διαφορετικούς σκοπούς. Σε αυτό το παράδειγμα θα δουλέψουμε σε μια φόρμα εγγραφής.

Αφού μεταβείτε στον επεξεργαστή φόρμας ολοκλήρωσης αγοράς, δώστε στη φόρμα σας ένα όνομα (που θα χρησιμοποιηθεί μόνο για εσωτερική αναφορά) και ένα slug (που χρησιμοποιείται για τη δημιουργία shortcodes, για παράδειγμα).

![Επεξεργαστής Φόρμας Ολοκλήρωσης Αγοράς](/img/config/checkout-form-editor.png)

Οι φόρμες αποτελούνται από βήματα και πεδία. Μπορείτε να προσθέσετε ένα νέο βήμα κάνοντας κλικ στο Add New Checkout Step.

![Προσθήκη Νέου Βήματος Ολοκλήρωσης Αγοράς](/img/config/checkout-form-add-step.png)

Στην πρώτη καρτέλα του παραθύρου διαλόγου, συμπληρώστε το περιεχόμενο του βήματος της φόρμας σας. Δώστε του ένα ID, ένα όνομα και μια περιγραφή. Αυτά τα στοιχεία χρησιμοποιούνται κυρίως εσωτερικά.

![Βήμα φόρμας ολοκλήρωσης αγοράς](/img/config/checkout-form-step.png)

Στη συνέχεια, ορίστε την ορατότητα του βήματος. Μπορείτε να επιλέξετε μεταξύ "Always show", "Only show for logged in users" ή "Only show for guests".

![Βήμα φόρμας ολοκλήρωσης αγοράς](/img/config/checkout-form-step.png)

Τέλος, διαμορφώστε το στυλ του βήματος. Αυτά είναι προαιρετικά πεδία.

![Βήμα φόρμας ολοκλήρωσης αγοράς](/img/config/checkout-form-step.png)

Τώρα, ήρθε η ώρα να προσθέσουμε πεδία στο πρώτο μας βήμα. Απλά κάντε κλικ στο Add New Field και επιλέξτε τον τύπο της ενότητας που θέλετε.

![Βήμα φόρμας ολοκλήρωσης αγοράς με πεδία](/img/config/checkout-form-step.png)

Κάθε πεδίο έχει διαφορετικές παραμέτρους προς συμπλήρωση. Για αυτή την πρώτη καταχώρηση, θα επιλέξουμε το πεδίο "Username".

![Βήμα φόρμας ολοκλήρωσης αγοράς](/img/config/checkout-form-step.png)

![Βήμα φόρμας ολοκλήρωσης αγοράς](/img/config/checkout-form-step.png)

![Βήμα φόρμας ολοκλήρωσης αγοράς](/img/config/checkout-form-step.png)

Μπορείτε να προσθέσετε όσα βήματα και πεδία χρειάζεστε. Για να εμφανίσετε τα προϊόντα σας ώστε να επιλέξουν οι πελάτες σας, χρησιμοποιήστε το πεδίο Pricing Table. Αν θέλετε να αφήσετε τους πελάτες σας να επιλέξουν ένα πρότυπο, προσθέστε το πεδίο Template Selection. Και ούτω καθεξής.

_**Σημείωση:** τα πεδία username, email, password, site title, site URL, order summary, payment και submit button είναι υποχρεωτικά για τη δημιουργία μιας φόρμας ολοκλήρωσης αγοράς._

Ενώ εργάζεστε στη φόρμα ολοκλήρωσης αγοράς, μπορείτε πάντα να χρησιμοποιήσετε το κουμπί Preview για να δείτε πώς θα βλέπουν οι πελάτες σας τη φόρμα. Μπορείτε επίσης να εναλλάσσετε μεταξύ προβολής ως υπάρχων χρήστης ή ως επισκέπτης.

![Αποθήκευση Φόρμας Ολοκλήρωσης Αγοράς](/img/config/checkout-form-save.png)

![Επεξεργαστής Φόρμας Ολοκλήρωσης Αγοράς](/img/config/checkout-form-editor.png)

Τέλος, στις Advanced Options μπορείτε να διαμορφώσετε το μήνυμα για τη σελίδα "Thank You", να προσθέσετε snippets για παρακολούθηση μετατροπών, να προσθέσετε προσαρμοσμένο CSS στη φόρμα ολοκλήρωσης αγοράς ή να την περιορίσετε σε συγκεκριμένες χώρες.

![Σύνθετες Επιλογές](/img/config/checkout-form-advanced.png)

Μπορείτε επίσης να ενεργοποιήσετε ή να απενεργοποιήσετε χειροκίνητα τη φόρμα ολοκλήρωσης αγοράς εναλλάσσοντας αυτή την επιλογή στη δεξιά στήλη, ή να διαγράψετε μόνιμα τη φόρμα.

![Διακόπτης ενεργοποίησης](/img/config/checkout-form-active.png)

Μην ξεχάσετε να αποθηκεύσετε τη φόρμα ολοκλήρωσης αγοράς!

![Κουμπί αποθήκευσης](/img/config/checkout-form-save.png)

Για να λάβετε το shortcode της φόρμας σας, κάντε κλικ στο Generate Shortcode και αντιγράψτε το αποτέλεσμα που εμφανίζεται στο παράθυρο διαλόγου.

![Κουμπί αποθήκευσης με shortcode](/img/config/checkout-form-save.png)
