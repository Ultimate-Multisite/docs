# Προσαρμογή της φόρμας εγγραφής σας

Για να κάνετε το δίκτυό σας να ξεχωρίζει από τα υπόλοιπα SaaS που βασίζονται στην πλατφόρμα WordPress, το Ultimate Multisite σας επιτρέπει να προσαρμόσετε τις σελίδες εγγραφής και σύνδεσης με τη λειτουργία **Checkout Forms**.

Παρόλο που είναι ένας εύκολος και ευέλικτος τρόπος για να πειραματιστείτε με διαφορετικές προσεγγίσεις όταν προσπαθείτε να μετατρέψετε νέους πελάτες, χρησιμοποιούνται κυρίως για τη δημιουργία εξατομικευμένων φορμών εγγραφής. Αυτό το άρθρο έχει σκοπό να σας δείξει πώς μπορείτε να το κάνετε.

## Σελίδες σύνδεσης και εγγραφής:

Κατά την εγκατάσταση του Ultimate Multisite, δημιουργούνται αυτόματα προσαρμοσμένες σελίδες σύνδεσης και εγγραφής στον κύριο ιστότοπό σας. Μπορείτε να αλλάξετε αυτές τις προεπιλεγμένες σελίδες ανά πάσα στιγμή πηγαίνοντας στη σελίδα **Ultimate Multisite > Settings > Login & Registration**.

![Σελίδα ρυθμίσεων σύνδεσης και εγγραφής](/img/config/settings-general.png)

Ας δούμε κάθε μία από τις επιλογές που μπορείτε να προσαρμόσετε στη σελίδα **Login & Registration**:

  * **Enable registration:** Αυτή η επιλογή ενεργοποιεί ή απενεργοποιεί την εγγραφή στο δίκτυό σας. Αν είναι απενεργοποιημένη, οι πελάτες σας δεν θα μπορούν να εγγραφούν και να εγγραφούν συνδρομητικά στα προϊόντα σας.

  * **Enable email verification:** Αν αυτή η επιλογή είναι ενεργοποιημένη, οι πελάτες που εγγράφονται σε δωρεάν πλάνο ή σε πλάνο επί πληρωμή με δοκιμαστική περίοδο θα λάβουν ένα email επαλήθευσης και θα πρέπει να κάνουν κλικ στον σύνδεσμο επαλήθευσης για να δημιουργηθούν οι ιστότοποί τους.

  * **Default registration page:** Αυτή είναι η προεπιλεγμένη σελίδα εγγραφής. Η σελίδα αυτή πρέπει να είναι δημοσιευμένη στον ιστότοπό σας και να περιέχει μια φόρμα εγγραφής (γνωστή και ως checkout form) - όπου οι πελάτες σας θα εγγράφονται συνδρομητικά στα προϊόντα σας. Μπορείτε να δημιουργήσετε όσες σελίδες εγγραφής και checkout forms θέλετε, απλά θυμηθείτε να τοποθετήσετε το shortcode της checkout form στη σελίδα εγγραφής, αλλιώς δεν θα εμφανιστεί.

  * **Use custom login page:** Αυτή η επιλογή σας επιτρέπει να χρησιμοποιήσετε μια προσαρμοσμένη σελίδα σύνδεσης, διαφορετική από την προεπιλεγμένη σελίδα wp-login.php. Αν αυτή η επιλογή είναι ενεργοποιημένη, μπορείτε να επιλέξετε ποια σελίδα θα χρησιμοποιείται για σύνδεση στην επιλογή **Default login page** (ακριβώς από κάτω).

  * **Obfuscate the original login url (wp-login.php)** : Αν θέλετε να αποκρύψετε το αρχικό URL σύνδεσης, μπορείτε να ενεργοποιήσετε αυτή την επιλογή. Αυτό είναι χρήσιμο για την αποτροπή επιθέσεων brute-force. Αν αυτή η επιλογή είναι ενεργοποιημένη, το Ultimate Multisite θα εμφανίζει σφάλμα 404 όταν κάποιος χρήστης προσπαθεί να αποκτήσει πρόσβαση στον αρχικό σύνδεσμο wp-login.php

  * **Force synchronous site publication:** Μετά την εγγραφή συνδρομής ενός πελάτη σε ένα προϊόν του δικτύου, ο νέος εκκρεμής ιστότοπος πρέπει να μετατραπεί σε πραγματικό ιστότοπο δικτύου. Η διαδικασία δημοσίευσης γίνεται μέσω Job Queue, ασύγχρονα. Ενεργοποιήστε αυτή την επιλογή για να αναγκάσετε τη δημοσίευση να γίνει στο ίδιο αίτημα με την εγγραφή.

Τώρα, ας δούμε άλλες επιλογές που σχετίζονται ακόμα με τη διαδικασία σύνδεσης και εγγραφής. Βρίσκονται ακριβώς κάτω από το **Other options** στην ίδια σελίδα Login & registration:

  * **Default role:** Αυτός είναι ο ρόλος που θα έχουν οι πελάτες σας στον ιστότοπό τους μετά τη διαδικασία εγγραφής.

  * **Add users to the main site as well:** Ενεργοποιώντας αυτή την επιλογή, ο χρήστης θα προστεθεί επίσης στον κύριο ιστότοπο του δικτύου σας μετά τη διαδικασία εγγραφής. Αν ενεργοποιήσετε αυτή την επιλογή, θα εμφανιστεί από κάτω μια επιλογή για να ορίσετε τον **προεπιλεγμένο ρόλο** αυτών των χρηστών στον ιστότοπό σας.

  * **Enable multiple accounts:** Επιτρέπει στους χρήστες να έχουν λογαριασμούς σε διαφορετικούς ιστότοπους του δικτύου σας με την ίδια διεύθυνση email. Αν αυτή η επιλογή είναι απενεργοποιημένη, οι πελάτες σας δεν θα μπορούν να δημιουργήσουν λογαριασμό σε άλλους ιστότοπους που τρέχουν στο δίκτυό σας με την ίδια διεύθυνση email.

Και αυτές είναι όλες οι επιλογές που σχετίζονται με τη σύνδεση και την εγγραφή που μπορείτε να προσαρμόσετε! Μην ξεχάσετε να αποθηκεύσετε τις ρυθμίσεις σας αφού ολοκληρώσετε την επεξεργασία τους.

## Χρήση πολλαπλών φορμών εγγραφής:

Το Ultimate Multisite 2.0 προσφέρει έναν επεξεργαστή checkout form που σας επιτρέπει να δημιουργείτε όσες φόρμες θέλετε, με διαφορετικά πεδία, προϊόντα στην προσφορά, κ.λπ.

Τόσο οι σελίδες σύνδεσης όσο και οι σελίδες εγγραφής είναι ενσωματωμένες με shortcodes: **[wu_login_form]** στη σελίδα σύνδεσης και **[wu_checkout]** για τη σελίδα εγγραφής. Μπορείτε να προσαρμόσετε περαιτέρω τη σελίδα εγγραφής δημιουργώντας checkout forms.

Για να αποκτήσετε πρόσβαση σε αυτή τη λειτουργία, μεταβείτε στο μενού **Checkout Forms**, στην αριστερή πλαϊνή γραμμή.

![Μενού Checkout Forms στην πλαϊνή γραμμή](/img/config/checkout-forms-list.png)

Σε αυτή τη σελίδα, μπορείτε να δείτε όλες τις checkout forms που έχετε.

Αν θέλετε να δημιουργήσετε μια νέα, απλά κάντε κλικ στο **Add Checkout Form** στην κορυφή της σελίδας.

Μπορείτε να επιλέξετε μία από αυτές τις τρεις επιλογές ως αφετηρία: single step, multi-step ή blank. Στη συνέχεια, κάντε κλικ στο **Go to the Editor**.

![Προσθήκη Checkout Form με επιλογές single step, multi-step ή blank](/img/config/checkout-forms-list.png)

Εναλλακτικά, μπορείτε να επεξεργαστείτε ή να αντιγράψετε τις φόρμες που έχετε ήδη κάνοντας κλικ στις επιλογές κάτω από το όνομά τους. Εκεί, θα βρείτε επίσης τις επιλογές για να αντιγράψετε το shortcode της φόρμας ή να διαγράψετε τη φόρμα.

![Ενέργειες hover στο checkout form με επεξεργασία, αντιγραφή και διαγραφή](/img/config/checkout-form-hover-actions.png)

Αν επιλέξετε single step ή multi-step, η checkout form θα είναι ήδη προ-συμπληρωμένη με τα βασικά βήματα για να λειτουργήσει. Στη συνέχεια, αν θέλετε, μπορείτε να προσθέσετε επιπλέον βήματα.

### Επεξεργασία μιας Checkout Form:

Όπως αναφέραμε προηγουμένως, μπορείτε να δημιουργήσετε checkout forms για διαφορετικούς σκοπούς. Σε αυτό το παράδειγμα θα δουλέψουμε σε μια φόρμα εγγραφής.

Αφού πλοηγηθείτε στον επεξεργαστή checkout form, δώστε στη φόρμα σας ένα όνομα (που θα χρησιμοποιείται μόνο για εσωτερική αναφορά) και ένα slug (που χρησιμοποιείται για τη δημιουργία shortcodes, για παράδειγμα).

![Επεξεργαστής checkout form με πεδία ονόματος και slug](/img/config/checkout-form-editor.png)

Οι φόρμες αποτελούνται από βήματα και πεδία. Μπορείτε να προσθέσετε ένα νέο βήμα κάνοντας κλικ στο **Add New Checkout Step**.

![Κουμπί Add New Checkout Step](/img/config/checkout-form-add-step.png)

Στην πρώτη καρτέλα του παραθύρου, συμπληρώστε το περιεχόμενο του βήματος της φόρμας σας. Δώστε του ένα ID, ένα όνομα και μια περιγραφή. Αυτά τα στοιχεία χρησιμοποιούνται κυρίως εσωτερικά.

![Καρτέλα περιεχομένου βήματος checkout με ID, όνομα και περιγραφή](/img/config/checkout-form-step.png)

Στη συνέχεια, ορίστε την ορατότητα του βήματος. Μπορείτε να επιλέξετε μεταξύ **Always show**, **Only show for logged in users** ή **Only show for guests**.

![Επιλογές ορατότητας βήματος checkout](/img/config/checkout-form-step.png)

Τέλος, διαμορφώστε το στυλ του βήματος. Αυτά είναι προαιρετικά πεδία.

![Διαμόρφωση στυλ βήματος checkout](/img/config/checkout-form-step.png)

Τώρα, ήρθε η ώρα να προσθέσουμε πεδία στο πρώτο μας βήμα. Απλά κάντε κλικ στο **Add New Field** και επιλέξτε τον τύπο της ενότητας που θέλετε.

![Κουμπί Add New Field](/img/config/checkout-form-editor.png)![Αναπτυσσόμενο μενού επιλογής τύπου πεδίου](/img/config/checkout-form-step.png)

Κάθε πεδίο έχει διαφορετικές παραμέτρους που πρέπει να συμπληρωθούν. Για αυτή την πρώτη καταχώρηση, θα επιλέξουμε το πεδίο **Username**.

![Διαμόρφωση πεδίου Username](/img/config/checkout-form-step.png)![Παράμετροι πεδίου Username](/img/config/checkout-form-step.png)![Πρόσθετες ρυθμίσεις πεδίου Username](/img/config/checkout-form-step.png)

Μπορείτε να προσθέσετε όσα βήματα και πεδία χρειάζεστε. Για να εμφανίσετε τα προϊόντα σας στους πελάτες σας ώστε να επιλέξουν ένα, χρησιμοποιήστε το πεδίο Pricing Table. Αν θέλετε να αφήσετε τους πελάτες σας να επιλέξουν ένα template, προσθέστε το πεδίο Template Selection. Και ούτω καθεξής.

_**Σημείωση:** Αν δημιουργήσετε ένα προϊόν μετά τη δημιουργία της checkout form σας, θα πρέπει να προσθέσετε το προϊόν στην ενότητα Pricing table. Αν δεν το προσθέσετε, το προϊόν δεν θα εμφανιστεί στους πελάτες σας στη σελίδα εγγραφής._

_**Σημείωση 2:** Τα πεδία username, email, password, site title, site URL, order summary, payment και submit button είναι υποχρεωτικά για τη δημιουργία μιας checkout form._

Ενώ εργάζεστε στη checkout form σας, μπορείτε πάντα να χρησιμοποιείτε το κουμπί Preview για να δείτε πώς θα βλέπουν οι πελάτες σας τη φόρμα. Μπορείτε επίσης να εναλλάσσεστε μεταξύ προβολής ως υπάρχων χρήστης ή ως επισκέπτης.

![Κουμπί Preview στον επεξεργαστή checkout form](/img/config/checkout-form-editor.png)![Προεπισκόπηση checkout form ως επισκέπτης ή υπάρχων χρήστης](/img/config/checkout-form-editor.png)

Τέλος, στις **Advanced Options** μπορείτε να διαμορφώσετε το μήνυμα για τη σελίδα **Thank You**, να προσθέσετε snippets για παρακολούθηση μετατροπών, να προσθέσετε προσαρμοσμένο CSS στη checkout form σας ή να την περιορίσετε σε συγκεκριμένες χώρες.

![Advanced Options με σελίδα Thank You, παρακολούθηση μετατροπών και προσαρμοσμένο CSS](/img/config/checkout-form-advanced.png)

Μπορείτε επίσης να ενεργοποιήσετε ή να απενεργοποιήσετε χειροκίνητα τη checkout form σας εναλλάσσοντας αυτή την επιλογή στη δεξιά στήλη, ή να διαγράψετε μόνιμα τη φόρμα.

![Διακόπτης Active και επιλογή διαγραφής για checkout form](/img/config/checkout-form-active.png)

Μην ξεχάσετε να αποθηκεύσετε τη checkout form σας!

![Κουμπί Save Checkout Form](/img/config/checkout-form-save.png)

Για να λάβετε το shortcode της φόρμας σας, κάντε κλικ στο **Generate Shortcode** και αντιγράψτε το αποτέλεσμα που εμφανίζεται στο παράθυρο.

![Παράθυρο Generate Shortcode με shortcode προς αντιγραφή](/img/config/checkout-form-editor.png)

_**Σημείωση:** Θα πρέπει να προσθέσετε αυτό το shortcode στη σελίδα εγγραφής σας για να προστεθεί αυτή η checkout form σε αυτήν._

## Προεπιλογή προϊόντων και templates μέσω παραμέτρων URL:

Αν θέλετε να δημιουργήσετε προσαρμοσμένους πίνακες τιμολόγησης για τα προϊόντα σας και να προεπιλέξετε στη checkout form το προϊόν ή το template που επιλέγει ο πελάτης σας από τον πίνακα τιμολόγησης ή τη σελίδα templates, μπορείτε να χρησιμοποιήσετε παραμέτρους URL για αυτό.

### **Για πλάνα:**

Μεταβείτε στο **Ultimate Multisite > Products > Επιλέξτε ένα πλάνο**. Θα πρέπει να δείτε το κουμπί **Click to copy Shareable Link** στην κορυφή της σελίδας. Αυτός είναι ο σύνδεσμος που μπορείτε να χρησιμοποιήσετε για να προεπιλέξετε αυτό το συγκεκριμένο πλάνο στη checkout form σας.

![Σελίδα προϊόντος με κουμπί shareable link](/img/config/products-list.png)

Σημειώστε ότι αυτός ο κοινοποιήσιμος σύνδεσμος ισχύει μόνο για **Πλάνα**. Δεν μπορείτε να χρησιμοποιήσετε κοινοποιήσιμους συνδέσμους για πακέτα ή υπηρεσίες.

### Για templates:

Αν θέλετε να προεπιλέξετε templates ιστότοπου στη checkout form σας, μπορείτε να χρησιμοποιήσετε την παράμετρο: **?template_id=X** στο URL της σελίδας εγγραφής σας. Το "X" πρέπει να αντικατασταθεί με τον **αριθμό ID του template ιστότοπου**. Για να λάβετε αυτόν τον αριθμό, μεταβείτε στο **Ultimate Multisite > Sites**.

Κάντε κλικ στο **Manage** ακριβώς κάτω από το template ιστότοπου που θέλετε να χρησιμοποιήσετε. Θα δείτε τον αριθμό SITE ID. Απλά χρησιμοποιήστε αυτόν τον αριθμό για να προεπιλεγεί αυτό το συγκεκριμένο template ιστότοπου στη checkout form σας. Στην περίπτωσή μας εδώ, η παράμετρος URL θα ήταν **?template_id=2**.

![Λίστα ιστότοπων που δείχνει το ID του template ιστότοπου](/img/config/site-templates-list.png)

Ας πούμε ότι ο ιστότοπος του δικτύου μας είναι [**www.mynetwork.com**](http://www.mynetwork.com) και η σελίδα εγγραφής μας με τη checkout form μας βρίσκεται στη σελίδα **/register**. Το πλήρες URL με αυτό το template ιστότοπου προεπιλεγμένο θα μοιάζει με [**www.mynetwork.com/register/?template**](http://www.mynetwork.com/register/?template)**_id=2**.

Και αν θέλετε, μπορείτε να προεπιλέξετε τόσο προϊόντα όσο και templates στη checkout form σας. Το μόνο που χρειάζεται να κάνετε είναι να αντιγράψετε τον κοινοποιήσιμο σύνδεσμο του πλάνου και να επικολλήσετε την παράμετρο template στο τέλος. Θα μοιάζει με [**www.mynetwork.com/register/premium-plan/?template**](http://www.mynetwork.com/register/premium-plan/?template)**_id=2**.
