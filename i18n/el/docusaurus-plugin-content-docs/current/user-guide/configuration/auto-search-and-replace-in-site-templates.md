# Αυτόματη Αναζήτηση και Αντικατάσταση σε Πρότυπα Ιστοσελίδων (v2)

_**Αυτό το tutorial απαιτεί την έκδοση 2.x του WP UItimo.**_

Μία από τις πιο ισχυρές δυνατότητες του Ultimate Multisite είναι η δυνατότητα προσθήκης προσαρμοσμένων πεδίων κειμένου, χρώματος και επιλογής στη φόρμα εγγραφής. Αφού καταγράψουμε αυτά τα δεδομένα, μπορούμε να τα χρησιμοποιήσουμε για να προ-συμπληρώσουμε το περιεχόμενο σε συγκεκριμένα μέρη του επιλεγμένου προτύπου ιστοσελίδας. Στη συνέχεια, όταν δημοσιευτεί η νέα ιστοσελίδα, το Ultimate Multisite θα αντικαταστήσει τα placeholders με τις πραγματικές πληροφορίες που εισήχθησαν κατά την εγγραφή.

Για παράδειγμα, μπορείτε να δημιουργήσετε τα πρότυπα ιστοσελίδων σας με placeholders. Τα placeholders πρέπει να προστεθούν περικυκλωμένα από διπλά άγκιστρα - {{placeholder_name}}.

Στη συνέχεια, μπορείτε απλά να προσθέσετε ένα αντίστοιχο πεδίο εγγραφής για να καταγράψετε αυτά τα δεδομένα

Ο πελάτης σας θα μπορεί να συμπληρώσει αυτό το πεδίο κατά την εγγραφή.

Το Ultimate Multisite θα αντικαταστήσει αυτόματα τα placeholders με τα δεδομένα που παρείχε ο πελάτης.

## **Λύνοντας το πρόβλημα του "προτύπου γεμάτου με placeholders"**

Όλα αυτά είναι υπέροχα, αλλά συναντάμε ένα άσχημο πρόβλημα: τώρα τα πρότυπα ιστοσελίδων μας - που μπορούν να επισκεφτούν οι πελάτες μας - είναι γεμάτα με άσχημα placeholders που δεν λένε πολλά.

Για να το λύσουμε αυτό, προσφέρουμε τη δυνατότητα ορισμού ψευδών τιμών για τα placeholders, και χρησιμοποιούμε αυτές τις τιμές για αναζήτηση και αντικατάσταση του περιεχομένου τους στα πρότυπα ιστοσελίδων ενώ οι πελάτες σας τα επισκέπτονται.

Μπορείτε να αποκτήσετε πρόσβαση στον επεξεργαστή placeholders του προτύπου πηγαίνοντας στο **Ultimate Multisite > Settings > Sites**, και στη συνέχεια, στην πλαϊνή μπάρα, κάνοντας κλικ στον σύνδεσμο Edit Placeholders.

![Σύνδεσμος Edit Placeholders στην πλαϊνή μπάρα ρυθμίσεων Sites](/img/config/settings-sites.png)

Αυτό θα σας μεταφέρει στον επεξεργαστή περιεχομένου placeholders, όπου μπορείτε να προσθέσετε placeholders και τις αντίστοιχες τιμές τους.

![Επεξεργαστής περιεχομένου placeholders με ονόματα και τιμές placeholders](/img/config/settings-sites.png)
