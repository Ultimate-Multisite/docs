# Αυτόματη αναζήτηση και αντικατάσταση στα πρότυπα ιστότοπου (v2) {#auto-search-and-replace-in-site-templates-v2}

_**Αυτό το εκπαιδευτικό άρθρο απαιτεί WP UItimo έκδοση 2.x.**_

Ένα από τα πιο ισχυρά χαρακτηριστικά του Ultimate Multisite είναι η δυνατότητα προσθήκης αυθαίρετων πεδίων κειμένου, χρώματος και επιλογής στη φόρμα εγγραφής. Μόλις καταγράψουμε αυτά τα δεδομένα, μπορούμε να τα χρησιμοποιήσουμε για να προ-συμπληρώσουμε το περιεχόμενο σε ορισμένα μέρη του επιλεγμένου προτύπου ιστότοπου. Έπειτα, όταν ο νέος ιστότοπος δημοσιευτεί, το Ultimate Multisite θα αντικαταστήσει τα placeholders με τις πραγματικές πληροφορίες που εισήχθησαν κατά την εγγραφή.

Για παράδειγμα, μπορείτε να δημιουργήσετε τους ιστότοπους-πρότυπά σας με placeholders. Τα placeholders πρέπει να προστίθενται περικλεισμένα σε διπλά άγκιστρα - {{placeholder_name}}.

Έπειτα, μπορείτε απλώς να προσθέσετε ένα αντίστοιχο πεδίο εγγραφής για να καταγράψετε αυτά τα δεδομένα

Ο πελάτης σας θα μπορεί έπειτα να συμπληρώσει αυτό το πεδίο κατά την εγγραφή.

Το Ultimate Multisite θα αντικαταστήσει έπειτα αυτόματα τα placeholders με τα δεδομένα που παρέχονται από τον πελάτη.

## **Επίλυση του προβλήματος «πρότυπο γεμάτο placeholders»** {#solving-the-template-full-of-placeholders-problem}

Όλα αυτά είναι εξαιρετικά, αλλά πράγματι αντιμετωπίζουμε ένα άσχημο πρόβλημα: τώρα τα πρότυπα ιστότοπού μας - τα οποία μπορούν να επισκέπτονται οι πελάτες μας - είναι γεμάτα με άσχημα placeholders που δεν λένε πολλά.

Για να το λύσουμε αυτό, προσφέρουμε την επιλογή ορισμού ψεύτικων τιμών για τα placeholders και χρησιμοποιούμε αυτές τις τιμές για αναζήτηση και αντικατάσταση του περιεχομένου τους στους ιστότοπους-πρότυπα ενώ οι πελάτες σας τους επισκέπτονται.

Μπορείτε να αποκτήσετε πρόσβαση στον επεξεργαστή placeholders προτύπου πηγαίνοντας στο **Ultimate Multisite > Settings > Sites**, κάνοντας κύλιση στην περιοχή Επιλογές προτύπου ιστότοπου και έπειτα κάνοντας κλικ στον σύνδεσμο **Επεξεργασία Placeholders**.

![Περιοχή επιλογών προτύπου ιστότοπου στη σελίδα ρυθμίσεων Sites](/img/config/settings-sites-templates-section.png)

Αυτό θα σας μεταφέρει στον επεξεργαστή περιεχομένου των placeholders, όπου μπορείτε να προσθέσετε placeholders και το αντίστοιχο περιεχόμενό τους.

![Σημείο εισόδου επεξεργαστή placeholders προτύπου](/img/config/settings-sites-templates-section.png)
