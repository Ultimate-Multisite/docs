# Διαχείριση Συνδέσμων Κοινοποίησης για Πλάνα

Το Ultimate Multisite σας δίνει τη δυνατότητα να μοιράζεστε συνδέσμους πλάνων απευθείας με τους πελάτες σας. Για να το κάνετε αυτό, μεταβείτε στη σελίδα επεξεργασίας οποιουδήποτε πλάνου και κάντε κλικ στο κουμπί _Κάντε κλικ για αντιγραφή του Συνδέσμου Κοινοποίησης_.

![Σελίδα επεξεργασίας προϊόντος με κουμπί συνδέσμου κοινοποίησης](/img/config/product-edit-full.png)

_Στη Σελίδα Επεξεργασίας του Πλάνου, θα βρείτε το κουμπί για τη δημιουργία συνδέσμων κοινοποίησης_

Σημειώστε ότι αυτός ο μηχανισμός χρησιμοποιεί την προεπιλεγμένη συχνότητα χρέωσης για τη δημιουργία του συνδέσμου. Αυτό σημαίνει ότι αν η προεπιλεγμένη χρέωσή σας είναι μηνιαία, για παράδειγμα, στο τέλος θα έχετε έναν σύνδεσμο σαν αυτόν: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Αν θέλετε ο σύνδεσμος κοινοποίησης να έχει ως προεπιλογή τριμηνιαία χρέωση, θα πρέπει να προσθέσετε το 3 στο τέλος

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Για να επιλέξετε το πλάνο _premium_ με ετήσια χρέωση, θα προσθέσετε το 1 και μετά τη λέξη:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Έχετε υπόψη ότι δεν μπορείτε να έχετε μόνο έναν σύνδεσμο για διαφορετικές συχνότητες χρέωσης. Θα πρέπει να μοιράζεστε έναν ξεχωριστό σύνδεσμο για κάθε συχνότητα χρέωσης.

Μόλις μοιραστείτε τον σύνδεσμο με τους πελάτες σας, θα τους ανακατευθύνει στο πρώτο βήμα της διαδικασίας εγγραφής σας.
