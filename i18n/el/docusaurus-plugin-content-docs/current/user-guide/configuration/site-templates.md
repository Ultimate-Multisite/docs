# Πρότυπα Ιστοτόπων (v2)

_**ΣΗΜΕΙΩΣΗ: Αυτό το άρθρο αναφέρεται στην έκδοση 2.x του Ultimate Multisite. Αν χρησιμοποιείτε την έκδοση 1.x,**_ **δείτε αυτό το άρθρο** _**.**_

Ο στόχος μας όταν δημιουργούμε ένα premium δίκτυο με το Ultimate Multisite είναι να αυτοματοποιήσουμε όσο το δυνατόν περισσότερες διαδικασίες, δίνοντας παράλληλα στους πελάτες μας ευελιξία και διαφορετικές επιλογές κατά τη δημιουργία των ιστοτόπων τους. Ένας εύκολος τρόπος για να πετύχουμε αυτή την ισορροπία είναι να αξιοποιήσουμε τη λειτουργία Πρότυπα Ιστοτόπων του Ultimate Multisite.

## Τι είναι ένα Πρότυπο Ιστοτόπου;

Όπως υποδηλώνει το όνομα, ένα Πρότυπο Ιστοτόπου είναι ένας έτοιμος ιστότοπος που μπορεί να χρησιμοποιηθεί ως βάση για τη δημιουργία νέων ιστοτόπων στο δίκτυό σας.

Αυτό σημαίνει ότι μπορείτε να δημιουργήσετε έναν βασικό ιστότοπο, να ενεργοποιήσετε διάφορα plugins, να ορίσετε ένα ενεργό theme και να τον προσαρμόσετε όπως θέλετε. Στη συνέχεια, όταν ο πελάτης σας δημιουργεί έναν νέο λογαριασμό, αντί να λάβει έναν προεπιλεγμένο ιστότοπο WordPress χωρίς ουσιαστικό περιεχόμενο, θα λάβει ένα αντίγραφο του βασικού σας ιστοτόπου με όλες τις προσαρμογές και τα περιεχόμενα ήδη στη θέση τους.

Ακούγεται τέλεια, αλλά πώς δημιουργώ ένα νέο πρότυπο ιστοτόπου; Είναι τόσο απλό όσο γίνεται.

## Δημιουργία και Επεξεργασία ενός νέου Προτύπου Ιστοτόπου

Τα Πρότυπα Ιστοτόπων είναι απλώς κανονικοί ιστότοποι στο δίκτυό σας. Για να δημιουργήσετε ένα νέο πρότυπο, απλά πηγαίνετε στο **Network Admin > Ultimate Multisite > Sites > Add Site.**

**![Κουμπί προσθήκης ιστοτόπου στη σελίδα λίστας ιστοτόπων](/img/config/site-templates-list.png)**

Αυτό θα ανοίξει ένα αναδυόμενο παράθυρο που θα ζητήσει τον **Τίτλο ιστοτόπου, το Domain/διαδρομή του ιστοτόπου** και τον **Τύπο ιστοτόπου**. Στο αναπτυσσόμενο πεδίο **Site Type** βεβαιωθείτε ότι έχετε επιλέξει **Site Template** *.*

_![Αναδυόμενο παράθυρο προσθήκης προτύπου ιστοτόπου με αναπτυσσόμενο μενού τύπου ιστοτόπου](/img/config/site-templates-list.png)_

Στο κάτω μέρος της φόρμας, θα παρατηρήσετε έναν διακόπτη **Copy Site**. Αυτός σας επιτρέπει να δημιουργήσετε ένα νέο πρότυπο ιστοτόπου βασισμένο σε ένα υπάρχον πρότυπο ως αφετηρία, για να εξοικονομήσετε χρόνο αντί να δημιουργήσετε ένα πρότυπο από την αρχή.

![Αναδυόμενο παράθυρο προσθήκης προτύπου ιστοτόπου με διακόπτη αντιγραφής ιστοτόπου](/img/config/site-templates-list.png)

### Προσαρμογή του περιεχομένου ενός Προτύπου Ιστοτόπου

Για να προσαρμόσετε το πρότυπο ιστοτόπου σας, απλά μεταβείτε στον πίνακα ελέγχου του και κάντε τις αλλαγές που χρειάζεστε. Μπορείτε να δημιουργήσετε νέα άρθρα, σελίδες, να ενεργοποιήσετε plugins και να αλλάξετε το ενεργό theme. Μπορείτε ακόμα να μεταβείτε στο Customizer και να αλλάξετε κάθε είδους επιλογές προσαρμογής.

Όλα αυτά τα δεδομένα θα αντιγραφούν όταν ένας πελάτης δημιουργήσει έναν νέο ιστότοπο βασισμένο σε αυτό το Πρότυπο Ιστοτόπου.

### Προηγμένες Επιλογές

Αν γνωρίζετε τα βασικά του προγραμματισμού, μπορείτε να αξιοποιήσετε το Search and Replace API μας για να αντικαταστήσετε αυτόματα πληροφορίες στον νέο ιστότοπο μετά τη δημιουργία του. Αυτό είναι χρήσιμο για πράγματα όπως η αντικατάσταση ονομάτων εταιρειών σε μια σελίδα Σχετικά με εμάς, η αντικατάσταση του email επικοινωνίας στη σελίδα Επικοινωνίας, κ.λπ.

### Χρήση Προτύπων Ιστοτόπων

Εντάξει, δημιουργήσατε λοιπόν διάφορα Πρότυπα Ιστοτόπων με διαφορετικά σχέδια, themes και ρυθμίσεις. Πώς τα κάνετε χρήσιμα στο δίκτυό σας τώρα;

Βασικά, υπάρχουν δύο προσεγγίσεις που μπορείτε να χρησιμοποιήσετε (όχι ταυτόχρονα):

  * Αντιστοίχιση ενός Προτύπου Ιστοτόπου σε κάθε ένα από τα Πακέτα σας

**Ή**

  * Να επιτρέψετε στους πελάτες σας να επιλέγουν οι ίδιοι τα πρότυπα ιστοτόπων κατά την εγγραφή.

#### Λειτουργία 1: Ανάθεση Προτύπου Ιστοτόπου

Σε αυτή τη λειτουργία, οι πελάτες σας δεν θα μπορούν να επιλέξουν πρότυπο όταν δημιουργούν λογαριασμό, αλλά εσείς θα ορίζετε ποιο πρότυπο θα χρησιμοποιείται σε κάθε ένα από τα Πακέτα σας.

Για να το κάνετε αυτό, θα πρέπει να μεταβείτε στο **Ultimate Multisite > Products > Edit**.

![Επεξεργασία προϊόντος για ανάθεση προτύπου ιστοτόπου](/img/config/product-site-templates.png)

Αυτό θα σας μεταφέρει στη σελίδα **Edit Product**. Στην ενότητα **Product Options**, βρείτε την καρτέλα **Site template** και επιλέξτε **Assign Site Template** από το αναπτυσσόμενο πεδίο. Αυτό θα εμφανίσει τη λίστα με τα διαθέσιμα πρότυπα ιστοτόπων και θα σας επιτρέψει να επιλέξετε μόνο ένα πρότυπο ιστοτόπου αφιερωμένο στο προϊόν.

![Καρτέλα προτύπου ιστοτόπου προϊόντος με επιλογή ανάθεσης προτύπου ιστοτόπου](/img/config/product-site-templates.png)

#### Λειτουργία 2: Επιλογή Διαθέσιμου Προτύπου Ιστοτόπου

Σε αυτή τη λειτουργία, θα δώσετε στους πελάτες σας τη δυνατότητα επιλογής κατά τη διαδικασία εγγραφής. Θα μπορούν να επιλέξουν από διαφορετικά πρότυπα ιστοτόπων που ορίζετε στις ρυθμίσεις του προϊόντος. Έχετε την επιλογή να περιορίσετε τα πρότυπα ιστοτόπων από τα οποία μπορούν να επιλέξουν για το επιλεγμένο προϊόν. Αυτό σας επιτρέπει να έχετε διαφορετικά σύνολα προτύπων ιστοτόπων για κάθε προϊόν, κάτι που είναι ιδανικό για να αναδείξετε διαφορετικές λειτουργίες και χαρακτηριστικά για ένα προϊόν υψηλότερης τιμής.

Στη σελίδα **Edit Product**, στην ενότητα **Product Options**, βρείτε την καρτέλα **Site template** και επιλέξτε **Choose Available Site Template** από το αναπτυσσόμενο πεδίο. Αυτό θα εμφανίσει τη λίστα με τα διαθέσιμα πρότυπα ιστοτόπων και θα σας επιτρέψει να επιλέξετε τα πρότυπα ιστοτόπων που θέλετε να είναι διαθέσιμα. Μπορείτε να το κάνετε επιλέγοντας τη Συμπεριφορά: **Available** αν θέλετε το πρότυπο ιστοτόπου να συμπεριλαμβάνεται στη λίστα. _**Not Available**_ αν θέλετε το πρότυπο ιστοτόπου να μην εμφανίζεται ως επιλογή. Και **Pre-selected** αν θέλετε ένα από τα πρότυπα ιστοτόπων στη λίστα να είναι το προεπιλεγμένο.

![Επιλογή διαθέσιμων προτύπων ιστοτόπων με επιλογές συμπεριφοράς](/img/config/product-site-templates.png)

### Προεπιλεγμένη Λειτουργία: Επιλογή προτύπου ιστοτόπου στη φόρμα Checkout

Αν θέλετε όλα τα πρότυπα ιστοτόπων σας να είναι διαθέσιμα κατά την εγγραφή, ή ίσως δεν προτιμάτε να κάνετε επιπλέον δουλειά για την ανάθεση ή τον καθορισμό προτύπων ιστοτόπων για κάθε προϊόν που δημιουργείτε. Τότε μπορείτε απλά να ορίσετε την επιλογή προτύπου ιστοτόπου στη **Checkout Form** σας. Για να το κάνετε αυτό, απλά πηγαίνετε στο **Ultimate Multisite > Checkout Forms**. Στη συνέχεια κάντε κλικ στο **Edit** κάτω από τη φόρμα που θέλετε να διαμορφώσετε.

![Σελίδα λίστας φορμών checkout](/img/config/checkout-forms-list.png)

Αυτό θα εμφανίσει τη σελίδα **Edit Checkout Form**. Βρείτε το πεδίο **Template Selection** και κάντε κλικ στο **Edit** κάτω από αυτό.

![Επεξεργαστής φόρμας checkout με πεδίο επιλογής προτύπου](/img/config/checkout-form-editor.png)

Θα εμφανιστεί ένα αναδυόμενο παράθυρο. Στο πεδίο **Template Sites** μπορείτε να επιλέξετε και να καταχωρήσετε όλα τα πρότυπα ιστοτόπων που θέλετε να είναι διαθέσιμα κατά την εγγραφή. Τα πρότυπα ιστοτόπων που καθορίζετε εδώ θα είναι διαθέσιμα ανεξάρτητα από το προϊόν που έχει επιλέξει ο χρήστης.

![Πεδίο προτύπων ιστοτόπων στον επεξεργαστή φόρμας checkout](/img/config/checkout-form-step.png)

### Επιλογές Προτύπων Ιστοτόπων

Υπάρχουν και άλλες λειτουργίες προτύπων ιστοτόπων που μπορείτε να ενεργοποιήσετε ή να απενεργοποιήσετε στις ρυθμίσεις του Ultimate Multisite.

![Επιλογές προτύπων ιστοτόπων στις ρυθμίσεις του Ultimate Multisite](/img/config/settings-sites.png)

#### Επιτρέπεται η Εναλλαγή Προτύπου

Η ενεργοποίηση αυτής της επιλογής θα επιτρέψει στους πελάτες σας να αλλάξουν το πρότυπο που επέλεξαν κατά τη διαδικασία εγγραφής μετά τη δημιουργία του λογαριασμού και του ιστοτόπου. Αυτό είναι χρήσιμο από την πλευρά του πελάτη, καθώς τους επιτρέπει να επανεπιλέξουν ένα πρότυπο αν αργότερα διαπιστώσουν ότι η αρχική τους επιλογή δεν ήταν η καλύτερη για τις συγκεκριμένες τους ανάγκες.

#### Επιτρέπεται στους Χρήστες να χρησιμοποιούν τον Ιστότοπό τους ως πρότυπα

Δεδομένου ότι οι χρήστες υποιστοτόπων αφιέρωσαν χρόνο για να δημιουργήσουν και να σχεδιάσουν τον δικό τους ιστότοπο, μπορεί να θέλουν να τον κλωνοποιήσουν και να τον χρησιμοποιήσουν ως ένα από τα διαθέσιμα πρότυπα ιστοτόπων κατά τη δημιουργία ενός άλλου υποιστοτόπου στο δίκτυό σας. Αυτή η επιλογή θα τους επιτρέψει να το επιτύχουν.

#### Αντιγραφή Πολυμέσων κατά την Αντιγραφή Προτύπου

Η ενεργοποίηση αυτής της επιλογής θα αντιγράψει τα πολυμέσα που έχουν ανεβεί στο πρότυπο ιστοτόπου στον νεοδημιουργημένο ιστότοπο. Αυτό μπορεί να παρακαμφθεί σε κάθε ένα από τα πακέτα.

#### **Αποτροπή Ευρετηρίασης Προτύπων Ιστοτόπων από τις Μηχανές Αναζήτησης**

Τα πρότυπα ιστοτόπων όπως συζητήθηκαν σε αυτό το άρθρο είναι boilerplate αλλά εξακολουθούν να είναι μέρος του δικτύου σας, πράγμα που σημαίνει ότι είναι ακόμα διαθέσιμα για εύρεση από τις μηχανές αναζήτησης. Αυτή η επιλογή θα σας επιτρέψει να αποκρύψετε τα πρότυπα ιστοτόπων ώστε οι μηχανές αναζήτησης να μην μπορούν να τα ευρετηριάσουν.

## Προ-συμπλήρωση Προτύπων Ιστοτόπων με αυτόματη αναζήτηση-και-αντικατάσταση

Ένα από τα πιο ισχυρά χαρακτηριστικά του Ultimate Multisite είναι η δυνατότητα προσθήκης αυθαίρετων πεδίων κειμένου, χρώματος και επιλογής στη φόρμα εγγραφής. Μόλις καταγράψουμε αυτά τα δεδομένα, μπορούμε να τα χρησιμοποιήσουμε για να προ-συμπληρώσουμε το περιεχόμενο σε συγκεκριμένα μέρη του επιλεγμένου προτύπου ιστοτόπου. Στη συνέχεια, όταν ο νέος ιστότοπος δημοσιεύεται, το Ultimate Multisite θα αντικαταστήσει τα placeholders με τις πραγματικές πληροφορίες που καταχωρήθηκαν κατά την εγγραφή.

Για παράδειγμα, αν θέλετε να λάβετε το όνομα της εταιρείας του τελικού χρήστη κατά την εγγραφή και να τοποθετήσετε αυτόματα το όνομα της εταιρείας στην αρχική σελίδα. Στην αρχική σελίδα του προτύπου ιστοτόπου σας πρέπει να προσθέσετε τα placeholders, όπως στην παρακάτω εικόνα (τα placeholders πρέπει να προστεθούν περιβαλλόμενα από διπλά άγκιστρα - {{placeholder_name}}).

![Αρχική σελίδα με κείμενο placeholder σε άγκιστρα](/img/config/site-templates-list.png)

Στη συνέχεια, μπορείτε απλά να προσθέσετε ένα αντίστοιχο πεδίο εγγραφής στη φόρμα checkout σας για να καταγράψετε αυτά τα δεδομένα:

![Φόρμα checkout με αντίστοιχο πεδίο εγγραφής](/img/config/checkout-form-editor.png)

Ο πελάτης σας θα μπορεί στη συνέχεια να συμπληρώσει αυτό το πεδίο κατά την εγγραφή.

![Πεδίο εγγραφής συμπληρωμένο από τον πελάτη](/img/config/checkout-form-step.png)

![Προεπισκόπηση της φόρμας εγγραφής](/img/config/checkout-form-editor.png)

Το Ultimate Multisite στη συνέχεια θα αντικαταστήσει τα placeholders με τα δεδομένα που παρείχε ο πελάτης αυτόματα.

![Τα placeholders αντικαταστάθηκαν με τα δεδομένα του πελάτη στον ιστότοπο](/img/config/site-templates-list.png)

### Επίλυση του προβλήματος "πρότυπο γεμάτο placeholders"

Όλα αυτά είναι υπέροχα, αλλά αντιμετωπίζουμε ένα άσχημο πρόβλημα: τώρα τα πρότυπα ιστοτόπων μας - που μπορούν να επισκεφθούν οι πελάτες μας - είναι γεμάτα με άσχημα placeholders που δεν λένε πολλά.

Για να το λύσουμε αυτό, προσφέρουμε την επιλογή ορισμού ψεύτικων τιμών για τα placeholders, και χρησιμοποιούμε αυτές τις τιμές για να αναζητήσουμε και να αντικαταστήσουμε τα περιεχόμενά τους στα πρότυπα ιστοτόπων ενώ οι πελάτες σας τα επισκέπτονται.

Μπορείτε να αποκτήσετε πρόσβαση στον επεξεργαστή placeholders προτύπων μεταβαίνοντας στο **Ultimate Multisite > Settings > Sites**, και στη συνέχεια, στην πλαϊνή στήλη, κάνοντας κλικ στον σύνδεσμο **Edit Placeholders**.

![Ρυθμίσεις placeholder στη σελίδα ρυθμίσεων Sites](/img/config/settings-sites.png)

Αυτό θα σας μεταφέρει στον επεξεργαστή περιεχομένου placeholders, όπου μπορείτε να προσθέσετε placeholders και το αντίστοιχο περιεχόμενό τους.

![Επεξεργαστής περιεχομένου placeholders προτύπων](/img/config/settings-sites.png)
