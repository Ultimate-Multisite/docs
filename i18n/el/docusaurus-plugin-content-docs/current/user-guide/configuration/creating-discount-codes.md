# Δημιουργία Κωδικών Έκπτωσης (v2)

_**ΣΗΜΑΝΤΙΚΗ ΣΗΜΕΙΩΣΗ: Αυτό το άρθρο αναφέρεται στην έκδοση 2.x του Ultimate Multisite.**_

Με το Ultimate Multisite μπορείτε να δημιουργήσετε κωδικούς έκπτωσης για να προσφέρετε στους πελάτες σας εκπτώσεις στις συνδρομές τους. Και η δημιουργία τους είναι εύκολη!

## Δημιουργία και Επεξεργασία Κωδικών Έκπτωσης

Για να δημιουργήσετε ή να επεξεργαστείτε έναν κωδικό έκπτωσης, μεταβείτε στο **Ultimate Multisite > Discount Codes**.

![](/img/config/discount-codes-list.png)

Εκεί θα δείτε μια λίστα με τους κωδικούς έκπτωσης που έχετε ήδη δημιουργήσει.

Μπορείτε να κάνετε κλικ στο **Add Discount** **Code** για να δημιουργήσετε ένα νέο κουπόνι ή να επεξεργαστείτε τα υπάρχοντα περνώντας το ποντίκι πάνω τους και κάνοντας κλικ στο **Edit**.

![Λίστα κωδικών έκπτωσης με ενέργειες κατά την αιώρηση](/img/config/discount-codes-list.png)

![Κουμπί Add Discount Code](/img/config/discount-codes-list.png)

Θα μεταφερθείτε στη σελίδα όπου θα δημιουργήσετε ή θα επεξεργαστείτε τον κωδικό κουπονιού σας. Σε αυτό το παράδειγμα θα δημιουργήσουμε έναν νέο.

![Σελίδα επεξεργασίας κωδικού έκπτωσης](/img/config/discount-codes-list.png)

Ας δούμε τις διαθέσιμες ρυθμίσεις:

**Enter Discount Code:** Αυτό είναι απλώς το όνομα του κωδικού έκπτωσής σας. Δεν είναι ο κωδικός που θα πρέπει να χρησιμοποιήσουν οι πελάτες σας στη φόρμα ολοκλήρωσης αγοράς.

**Description:** Εδώ μπορείτε να περιγράψετε σύντομα σε τι χρησιμεύει αυτό το κουπόνι.

![Πεδία ονόματος και περιγραφής κωδικού έκπτωσης](/img/config/discount-codes-list.png)

**Coupon code:** Εδώ ορίζετε τον κωδικό που θα πρέπει να εισάγουν οι πελάτες σας κατά την ολοκλήρωση της αγοράς.

![Πεδίο κωδικού κουπονιού](/img/config/discount-codes-list.png)

**Discount:** Εδώ μπορείτε να ορίσετε είτε ένα **ποσοστό** είτε ένα **σταθερό ποσό** για τον κωδικό έκπτωσής σας.

![Ρύθμιση ποσοστού ή σταθερού ποσού έκπτωσης](/img/config/discount-codes-list.png)

**Apply to renewals:** Αν αυτή η επιλογή είναι απενεργοποιημένη, ο κωδικός έκπτωσης θα εφαρμοστεί μόνο στην **πρώτη πληρωμή**. Όλες οι υπόλοιπες πληρωμές δεν θα έχουν έκπτωση. Αν αυτή η επιλογή είναι ενεργοποιημένη, ο κωδικός έκπτωσης θα ισχύει για όλες τις μελλοντικές πληρωμές.

**Setup fee discount:** Αν αυτή η επιλογή είναι απενεργοποιημένη, ο κωδικός κουπονιού **δεν θα παρέχει καμία έκπτωση στο τέλος εγκατάστασης** της παραγγελίας. Αν αυτή η επιλογή είναι ενεργοποιημένη, μπορείτε να ορίσετε την έκπτωση (ποσοστό ή σταθερό ποσό) που θα εφαρμόζει αυτός ο κωδικός κουπονιού στο τέλος εγκατάστασης των πακέτων σας.

![Επιλογές εφαρμογής σε ανανεώσεις και έκπτωσης τέλους εγκατάστασης](/img/config/discount-codes-list.png)

**Active:** Ενεργοποιήστε ή απενεργοποιήστε χειροκίνητα αυτόν τον κωδικό κουπονιού.

![Διακόπτης ενεργοποίησης κωδικού έκπτωσης](/img/config/discount-codes-list.png)

Στις **Advanced Optio** **ns** έχουμε τις ακόλουθες ρυθμίσεις:

**Limit uses:**

  * **Uses:** Εδώ μπορείτε να δείτε πόσες φορές χρησιμοποιήθηκε ο κωδικός έκπτωσης.

  * **Max uses:** Αυτό περιορίζει τον αριθμό των φορών που μπορούν οι χρήστες να χρησιμοποιήσουν αυτόν τον κωδικό έκπτωσης. Για παράδειγμα, αν βάλετε 10 εδώ, το κουπόνι θα μπορεί να χρησιμοποιηθεί μόνο 10 φορές. Μετά από αυτό το όριο, ο κωδικός κουπονιού δεν μπορεί να χρησιμοποιηθεί πλέον.

![Ρύθμιση περιορισμού χρήσεων με πεδία χρήσεων και μέγιστων χρήσεων](/img/config/discount-codes-list.png)**Start & expiration dates:** Εδώ θα έχετε την επιλογή να προσθέσετε ημερομηνία έναρξης ή/και ημερομηνία λήξης στο κουπόνι σας.

![Πεδία ημερομηνίας έναρξης και λήξης](/img/config/discount-codes-list.png)

**Limit products:** Αν ενεργοποιήσετε την επιλογή **Select products**, θα εμφανιστούν όλα τα προϊόντα σας. Θα έχετε την επιλογή να επιλέξετε χειροκίνητα (ενεργοποιώντας ή απενεργοποιώντας) ποια προϊόντα μπορούν να δέχονται αυτόν τον κωδικό κουπονιού. Τα προϊόντα που είναι απενεργοποιημένα εδώ δεν θα εμφανίσουν καμία αλλαγή αν οι πελάτες σας προσπαθήσουν να χρησιμοποιήσουν αυτόν τον κωδικό κουπονιού σε αυτά.

![Διακόπτες επιλογής περιορισμού προϊόντων](/img/config/discount-codes-list.png)

Αφού ρυθμίσετε όλες αυτές τις επιλογές, κάντε κλικ στο **Save Discount Code** για να αποθηκεύσετε το κουπόνι σας και είστε έτοιμοι!

![Κουμπί Save Discount Code](/img/config/discount-codes-list.png)

Το κουπόνι βρίσκεται τώρα στη λίστα σας και από εκεί μπορείτε να κάνετε κλικ για να το **επεξεργαστείτε ή να το διαγράψετε**.

![Κωδικός έκπτωσης στη λίστα με επιλογές επεξεργασίας και διαγραφής](/img/config/discount-codes-list.png)

### Χρήση Παραμέτρων URL:

Αν θέλετε να προσαρμόσετε τους πίνακες τιμών σας ή να δημιουργήσετε μια όμορφη σελίδα κωδικών κουπονιών για την ιστοσελίδα σας και θέλετε να εφαρμόσετε αυτόματα έναν κωδικό έκπτωσης στη φόρμα ολοκλήρωσης αγοράς, μπορείτε να το κάνετε μέσω παραμέτρων URL.

Πρώτα, πρέπει να λάβετε τον κοινοποιήσιμο σύνδεσμο για το πακέτο σας. Για να το κάνετε αυτό, μεταβείτε στο **Ultimate Multisite > Products** και επιλέξτε ένα πακέτο.

Κάντε κλικ στο κουμπί **Click to Copy Shareable Link**. Αυτό θα σας δώσει τον κοινοποιήσιμο σύνδεσμο για αυτό το συγκεκριμένο πακέτο. Στην περίπτωσή μας, ο κοινοποιήσιμος σύνδεσμος που δόθηκε ήταν [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/)_._

![Σελίδα προϊόντος με κουμπί κοινοποιήσιμου συνδέσμου](/img/config/products-list.png)

Για να εφαρμόσετε τον κωδικό έκπτωσής σας σε αυτό το συγκεκριμένο πακέτο, απλώς προσθέστε την παράμετρο **?discount_code=XXX** στο URL. Όπου **XXX** είναι ο κωδικός κουπονιού.

Στο παράδειγμά μας εδώ, θα εφαρμόσουμε τον κωδικό κουπονιού **50OFF** σε αυτό το συγκεκριμένο προϊόν.

Το URL για αυτό το συγκεκριμένο πακέτο με εφαρμοσμένο τον κωδικό έκπτωσης 50OFF θα είναι: [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/) _**?discount_code=50OFF**_.
