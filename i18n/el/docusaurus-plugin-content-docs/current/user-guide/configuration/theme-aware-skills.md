# Δεξιότητες Συναισθηzące το Theme

Το Superdav AI Agent v1.10.0 περιλαμβάνει τέσσερις νέες ενσωματωμένες δεξιότητες που είναι συναισθηzące το theme και προσαρμόζονται αυτόματα στο ενεργό theme του WordPress σας. Αυτές οι δεξιότητες παρέχουν εξειδικευμένες οδηγίες και δυνατότητες προσαρμοσμένες στην αρχιτεκτονική και τα χαρακτηριστικά του theme σας.

## Τι είναι οι Δεξιότητες Συναισθηzące το Theme; {#what-are-theme-aware-skills}

Οι δεξιότητες συναισθηzące το theme είναι προκαθορισμένες βάσεις γνώσης και σετ εργαλείων που ο βοηθός AI επιλέγει αυτόματα με βάση το theme του WordPress που είναι ενεργό στον ιστότοπό σας. Όταν αλλάζετε themes, οι διαθέσιμες δεξιότητες του βοηθού ενημερώνονται αυτόματα — δεν απαιτείται χειροκίνητη ρύθμιση.

Κάθε δεξιότητα περιλαμβάνει:

- **Εξειδικευμένη τεκμηρίωση του theme** — οδηγίες για τη χρήση και την προσαρμογή του theme
- **Αναφορές σε blocks και patterns** — διαθέσιμα blocks, patterns και επιλογές σχεδιασμού
- **Παραδείγματα προσαρμογών** — κομμάτια κώδικα και μοτίβα διαμόρφωσης για συνηθισμένες εργασίες
- **Καλές πρακτικές** — συστάσεις για την αρχιτεκτονική και τη ροή εργασίας του theme

## Διαθέσιμες Δεξιότητες Συναισθηzące το Theme {#available-theme-aware-skills}

### Block Themes {#block-themes}

**Εφαρμόζεται σε:** Themes που χρησιμοποιούν την αρχιτεκτονική με blocks του WordPress (Full Site Editing).

Η δεξιότητα Block Themes παρέχει οδηγίες για:

- Δημιουργία και επεξεργασία templates χρησιμοποιώντας τον block editor
- Εργασία με block patterns και επαναχρησιμοποιήσιμα blocks
- Προσαρμογή παγκόμιας εμφάνισης (global styles) και ρυθμίσεων theme.json
- Χρήση theme blocks και παραλλαγών
- Δημιουργία προσαρμοσμένων block patterns για τον ιστότοπό σας

**Ενεργοποιείται αυτόματα όταν:** Το ενεργό theme σας είναι ένα block theme (υποστηρίζει το χαρακτηριστικό `block-templates`).

### Classic Themes {#classic-themes}

**Εφαρμόζεται σε:** Πα παραδοσιακά themes του WordPress που χρησιμοποιούν PHP templates και τον classic editor.

Η δεξιότητα Classic Themes παρέχει οδηγίες για:

- Εργασία με αρχεία templates PHP και hooks
- Προσαρμογή της εμφάνισης του theme μέσω του Customizer
- Χρήση περιοχών widgets και sidebars
- Τροποποίηση CSS και ανάπτυξη child theme
- Κατανόηση της ιεραρχίας του theme και των template tags

**Ενεργοποιείται αυτόματα όταν:** Το ενεργό theme σας είναι ένα classic (μη block) theme.

### Kadence Blocks {#kadence-blocks}

**Εφαρμόζεται σε:** Ιστοσελίδες που χρησιμοποιούν το plugin Kadence Blocks για προηγμένο σχεδιασμό με blocks.

Η δεξιότητα Kadence Blocks παρέχει οδηγίες για:

- Χρήση της προηγμένης βιβλιοθήκης blocks της Kadence (Hero, Testimonials, Pricing, κ.λπ.)
- Διαμόρφωση ρυθμίσεων blocks και επιλογών responsive της Kadence
- Δημιουργία προσαρμοσμένων layouts με τα blocks grid και container της Kadence
- Ενσωμάτωση blocks της Kadence με το theme σας
- Εκμετάλλευση του design system και των presets της Kadence

**Ενεργοποιείται αυτόματα όταν:** Το plugin Kadence Blocks είναι ενεργό στον ιστότοπό σας.

### Kadence Theme {#kadence-theme}

**Εφαρμόζεται σε:** Ιστοσελίδες που χρησιμοποιούν το theme Kadence για σχεδιασμό και προσαρμογή με blocks.

Η δεξιότητα Kadence Theme παρέχει οδηγίες για:

- Προσαρμογή του theme Kadence μέσω global styles και theme.json
- Χρήση των ενσωματωμένων block patterns και templates της Kadence
- Διαμόρφωση ρυθμίσεων και επιλογών του theme Kadence
- Δημιουργία προσαρμοσμένων σχεδιασμών με το design system της Kadence
- Ενσωμάτωση της Kadence με δημοφιλή plugins και εργαλεία

**Ενεργοποιείται αυτόματα όταν:** Το theme Kadence είναι το ενεργό theme σας.

## Πώς Επιλέγονται οι Δεξιότητες {#how-skills-are-selected}

Ο βοηθός αναγνωρίζει αυτόματα το ενεργό theme και τα εγκατεστημένα plugins σε κάθε μήνυμα. Εάν υπάρχει διαθέσιμη δεξιότητα συναισθηzące το theme που ταιριάζει, φορτώνεται αυτόματα στο πλαίσιο (context) του βοηθού. Δεν χρειάζεται να ενεργοποιήσετε ή να αλλάξετε δεξιότητες χειροκίνητα.

### Πολλαπλές Δεξιότητες {#multiple-skills}

Εάν πολλές δεξιότητες ισχύουν για τον ιστότοπό σας (π.χ., αν έχετε ενεργοποιημένα τόσο το Kadence Blocks όσο και το Kadence Theme), ο βοηθός έχει πρόσβαση σε όλες τις εφαρμόσιμες δεξιότητες και μπορεί να αναφέρεται σε οδηγίες από κάθε μία.

### Αλλαγή Themes {#switching-themes}

Όταν αλλάζετε το ενεργό theme σας, οι διαθέσιμες δεξιότητες του βοηθού ενημερώνονται αυτόματα στο επόμενο μήνυμα. Για παράδειγμα:

1. Χρησιμοποιείτε ένα block theme με ενεργή τη δεξιότητα **Block Themes**.
2. Αλλάζετε σε ένα classic theme.
3. Στο επόμενο μήνυμα σας, η δεξιότητα **Classic Themes** φορτώνεται αυτόματα και η δεξιότητα **Block Themes** δεν είναι πλέον διαθέσιμη.

## Χρήση των Δεξιοτήτων Συναισθηzące το Theme {#using-theme-aware-skills}

Για να αξιοποιήσετε μια δεξιότητα συναισθηzące το theme, απλώς περιγράψτε τι θέλετε να κάνετε στην διεπαφή chat. Ο βοηθός θα ανατρέξει αυτόματα στις σχετικές οδηγίες της δεξιότητας.

### Παραδείγματα Προτροπών (Prompts) {#example-prompts}

**Για Block Themes:**
> "Δημιουργήστε μια hero section με φόντο εικόνα και κείμενο στο κέντρο χρησιμοποιώντας block patterns."

**Για Classic Themes:**
> "Προσθέστε μια προσαρμοσμένη περιοχή widget στη sidebar χρησιμοποιώντας ένα child theme."

**Για Kadence Blocks:**
> "Δημιουργήστε μια ενότητα μαρτυριών (testimonials) χρησιμοποιώντας το block Kadence Testimonials."

**Για Kadence Theme:**
> "Προσαρμόστε το layout του header και το styling του μενού πλοήγησης."

Ο βοηθός θα παρέχει εξειδικευμένες οδηγίες και παραδείγματα κώδικα προσαρμοσμένα στο ενεργό theme και τα plugins σας.

:::note
Οι δεξιότητες συναισθηzące το theme είναι αυτόματα διαθέσιμες στο Superdav AI Agent v1.10.0 και νεότερες εκδόσεις. Δεν απαιτείται επιπλέον ρύθμιση ή διαμόρφωση.
:::
