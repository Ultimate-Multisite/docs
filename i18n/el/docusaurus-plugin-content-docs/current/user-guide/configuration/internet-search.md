# Αναζήτηση στο Διαδίκτυο {#internet-search}

Το Gratis AI Agent v1.5.0 προσθέτει μια ικανότητα **Αναζήτησης στο Διαδίκτυο** που επιτρέπει στον βοηθό AI να ανακτά ενημερωμένες πληροφορίες από το web κατά τη διάρκεια μιας συνομιλίας. Αυτό λειτουργεί μέσω της [Brave Search API](https://brave.com/search/api/) ή της [Tavily API](https://tavily.com/).

## Πώς Λειτουργεί {#how-it-works}

Όταν η αναζήτηση στο διαδίκτυο είναι ενεργοποιημένη, ο βοηθός μπορεί αυτόματα να κάνει μια αναζήτηση στον προκαθορισμένο πάροχο αναζήτησης όταν διαπιστώσει ότι μια ερώτηση απαιτεί τρέχουσες ή εξωτερικές πληροφορίες — για παράδειγμα, πρόσφατα νέα, ζωντανές τιμές ή τεκμηρίωση που ενδέχεται να έχει αλλάξει από το όριο εκπαίδευσης του μοντέλου.

Τα αποτελέσματα ανακτώνται σε πραγματικό χρόνο και ενσωματώνονται στο πλαίσιο (context) του βοηθού πριν αυτός δημιουργήσει μια απάντηση. Ο βοηθός υποδεικνύει ότι έχει χρησιμοποιήσει αποτελέσματα αναζήτησης για να απαντήσει σε μια ερώτηση.

## Ενεργοποίηση της Αναζήτησης στο Διαδίκτυο {#enabling-internet-search}

Η αναζήτηση στο διαδίκτυο απαιτεί έναν κλειδιά API από τον επιλεγμένο πάροχο αναζήτησης. Για να το ρυθμίσετε:

1. Μεταβείτε στο **Gratis AI Agent → Settings → Advanced**.
2. Βρείτε το αναπτυσσόμενο μενού **Internet Search Provider** και επιλέξτε είτε **Brave Search** είτε **Tavily**.
3. Εισάγετε το API key σας στο αντίστοιχο πεδίο. Τα συνδέσμια εγγραφής εμφανίζονται δίπλα σε κάθε πεδίο.
4. Κάντε κλικ στο **Save Settings**.

Μόλις αποθηκευρωθεί ο κλειδιά, η ικανότητα Αναζήτησης στο Διαδίκτυο είναι αυτόματα διαθέσιμη στον βοηθό.

## Brave Search {#brave-search}

### Λήψη ενός API Key του Brave Search {#obtaining-a-brave-search-api-key}

1. Επισκεφθείτε τη [Brave Search API page](https://brave.com/search/api/).
2. Εγγραφείτε για ένα plan. Διατίθεται ένα δωρεάν επίπεδο με περιορισμό αιτημάτων ανά μήνα.
3. Αντιγράψτε το API key σας από το developer dashboard του Brave Search.
4. Επικολλήστε το στο πεδίο **Brave Search API Key** στις ρυθμίσεις του Gratis AI Agent.

### Όρια Χρήσης {#usage-limits}

Η χρήση χρεώνεται από το Brave Search με βάση τον αριθμό των αναζητήσεων που πραγματοποιούνται. Κάθε απάντηση AI που ενεργοποιεί αναζήτηση μετράται ως μία αναζήτηση. Παρακολουθήστε τη χρήση σας στο [Brave Search developer dashboard](https://brave.com/search/api/) για να αποφύγετε απρόοπτα χρεώσεις.

## Tavily {#tavily}

Το Superdav AI Agent v1.10.0 προσθέτει το **Tavily** ως εναλλακτικό πάροχο αναζήτησης στο διαδίκτυο, προσφέροντας πλουσιότερους αποτελέσματα αναζήτησης και προηγμένες δυνατότητες έρευνας.

### Λήψη ενός API Key του Tavily {#obtaining-a-tavily-api-key}

1. Επισκεφθείτε τη [Tavily API page](https://tavily.com/).
2. Εγγραφείτε για έναν λογαριασμό. Διατίθεται ένα δωρεάν επίπεδο με περιορισμό αιτημάτων ανά μήνα.
3. Αντιγράψτε το API key σας από το Tavily dashboard.
4. Επικολλήστε το στο πεδίο **Tavily API Key** στις ρυθμίσεις του Gratis AI Agent.

### Όρια Χρήσης {#usage-limits-1}

Η χρήση χρεώνεται από το Tavily με βάση τον αριθμό των κλήσεων API που πραγματοποιούνται. Κάθε απάντηση AI που ενεργοποιεί αναζήτηση μετράται ως μία κλήση. Παρακολουθήστε τη χρήση σας στο [Tavily dashboard](https://tavily.com/) για να αποφύγετε απρόοπτα χρεώσεις.

## Απενεργοποίηση της Αναζήτησης στο Διαδίκτυο {#disabling-internet-search}

Αφαιρέστε το API key από το πεδίο του ενεργού παρόχου αναζήτησης και αποθηκεύστε. Η ικανότητα Αναζήτησης στο Διαδίκτυο δεν θα προσφέρεται πλέον στον βοηθό.

:::note
Η αναζήτηση στο διαδίκτυο προσθέτει καθυστέρηση στις απαντήσεις επειδή ο βοηθός πρέπει να περιμένει τα αποτελέσματα αναζήτησης πριν δημιουργήσει μια απάντηση. Για περιπτώσεις που απαιτούν άμεση ανταπόκριση, εξετάστε αν είναι απαραίτητη η αναζήτηση σε πραγματικό χρόνο ή αν είναι αρκετή η ενσωματωμένη γνώση του βοηθού.
:::
