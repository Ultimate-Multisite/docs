# Ενσωματωμένοι πράκτορες

Το Gratis AI Agent v1.9.0 περιλαμβάνει πέντε ενσωματωμένους πράκτορες, καθένας προδιαμορφωμένος με ένα εστιασμένο σύνολο εργαλείων, ένα προσαρμοσμένο system prompt και αρχικές προτάσεις που αντιστοιχούν σε συνηθισμένες εργασίες σε εκείνον τον τομέα. Η εναλλαγή μεταξύ πρακτόρων αλλάζει τι μπορεί να κάνει ο βοηθός και πώς αποκρίνεται — χωρίς καμία διαμόρφωση από την πλευρά σας. Το Superdav AI Agent v1.18.0 μπορεί να προσθέσει εργαλεία με επίγνωση χρονοδιαγράμματος, εγγραφές υπενθυμίσεων, πύλες έγκρισης και ειδοποιήσεις SMS σε αυτές τις ροές εργασίας όταν οι σχετικές ενσωματώσεις έχουν διαμορφωθεί.

## Τι είναι ένας πράκτορας;

Κάθε πράκτορας είναι ένα επώνυμο προφίλ διαμόρφωσης που συνδυάζει:

- **Εργαλεία** — τις δυνατότητες που επιτρέπεται να επικαλεστεί ο πράκτορας (π.χ. ένας Συντάκτης Περιεχομένου έχει πρόσβαση σε δυνατότητες δημιουργίας αναρτήσεων· ένα Στούντιο Σχεδίασης έχει πρόσβαση σε δυνατότητες CSS και theme.json)
- **System prompt** — οδηγίες που ορίζουν τον τόνο, τις προτεραιότητες και τους περιορισμούς του πράκτορα
- **Προτάσεις** — προγραμμένα prompt που εμφανίζονται στη διεπαφή συνομιλίας για να σας βοηθήσουν να ξεκινήσετε γρήγορα

## Πρόσβαση στον επιλογέα πρακτόρων

1. Ανοίξτε το πάνελ **Gratis AI Agent** στην πλευρική γραμμή διαχείρισης του WordPress.
2. Κάντε κλικ στο **εικονίδιο πράκτορα** επάνω αριστερά στην κεφαλίδα της συνομιλίας (το εικονίδιο αλλάζει για να αντικατοπτρίζει τον ενεργό πράκτορα).
3. Ο **Επιλογέας πρακτόρων** ανοίγει ως επικάλυψη πίνακα φόρμας. Κάθε πράκτορας παρατίθεται με το εικονίδιο, το όνομα και μια περιγραφή μίας γραμμής.
4. Κάντε κλικ σε μια γραμμή πράκτορα για να τον ενεργοποιήσετε. Η κεφαλίδα της συνομιλίας ενημερώνεται αμέσως.

Μπορείτε επίσης να αλλάξετε πράκτορες στη μέση της συζήτησης — το system prompt του νέου πράκτορα τίθεται σε ισχύ από το επόμενο μήνυμα.

## Οι πέντε ενσωματωμένοι πράκτορες

### Συντάκτης Περιεχομένου

**Εστίαση:** Δημιουργία και επεξεργασία αναρτήσεων, σελίδων και φορμών επικοινωνίας.

**Διαθέσιμα εργαλεία:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`. Με ενεργοποιημένες τις ενσωματώσεις του Superdav AI Agent v1.18.0, το διαμορφωμένο ημερολογιακό πλαίσιο, οι πύλες έγκρισης, οι υπενθυμίσεις και τα εργαλεία ειδοποιήσεων SMS μπορεί επίσης να είναι διαθέσιμα σε εγκεκριμένες ροές εργασίας.

**Τι κάνει καλά:**
- Σύνταξη και δημοσίευση αναρτήσεων ιστολογίου από μια σύνοψη ή ένα περίγραμμα
- Δημιουργία παρτίδων σελίδων προορισμού για έναν νέο ιστότοπο
- Δημιουργία φορμών επικοινωνίας και ερωτημάτων
- Ορισμός χαρακτηριστικών εικόνων σε αναρτήσεις από URL ή αναζήτηση
- Σύνταξη μηνυμάτων παρακολούθησης συμβάντων από διαμορφωμένο πλαίσιο Google Calendar και στη συνέχεια παύση για έγκριση πριν από την αποστολή ειδοποιήσεων

**Αρχικές προτάσεις:**
- *Γράψε μια ανάρτηση ιστολογίου 500 λέξεων σχετικά με τα οφέλη του WordPress multisite.*
- *Δημιούργησε μια σελίδα Σχετικά, Υπηρεσίες και Επικοινωνία και δημοσίευσέ τες.*
- *Πρόσθεσε μια φόρμα ερωτήματος κράτησης στη σελίδα Επικοινωνία.*
- *Σύνταξε μια υπενθύμιση για τους συμμετέχοντες στο αυριανό διαμορφωμένο ημερολογιακό συμβάν και περίμενε έγκριση πριν την στείλεις.*

### Δημιουργός Ιστότοπου

**Εστίαση:** Πλήρης δημιουργία ιστότοπου από ένα μόνο prompt.

**Διαθέσιμα εργαλεία:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`. Με το Superdav AI Agent v1.18.0, διαμορφωμένα εργαλεία διαχειριζόμενης υπηρεσίας, έγκρισης, υπενθύμισης, ημερολογίου και SMS μπορεί να είναι διαθέσιμα όπου οι διαχειριστές τα ενεργοποιούν.

**Τι κάνει καλά:**
- Δημιουργία πολυφασικού σχεδίου κατασκευής ιστότοπου για έναν περιγραφόμενο τύπο επιχείρησης
- Αυτόνομη εκτέλεση κάθε φάσης — δομή, περιεχόμενο, πλοήγηση, σχεδίαση
- Ανάκαμψη από σφάλματα στη μέση του σχεδίου χωρίς να απαιτείται χειροκίνητη παρέμβαση
- Εγκατάσταση προτεινόμενων plugin ως μέρος της κατασκευής
- Δημιουργία φορμών επικοινωνίας απευθείας από τη διεπαφή συνομιλίας (Superdav AI Agent v1.10.0+)
- Συντονισμός υπενθυμίσεων εκκίνησης ή παρακολούθησης συμμετεχόντων χωρίς διπλές ειδοποιήσεις όταν είναι ενεργοποιημένες οι πύλες έγκρισης και οι εγγραφές υπενθυμίσεων

**Αρχικές προτάσεις:**
- *Δημιούργησε έναν ιστότοπο portfolio φωτογραφίας με τύπο αναρτήσεων συλλογής, σελίδα κράτησης και φόρμα επικοινωνίας.*
- *Δημιούργησε έναν ιστότοπο εστιατορίου με online μενού, ώρες λειτουργίας και φόρμα ερωτήματος κράτησης τραπεζιού.*
- *Ρύθμισε έναν ιστότοπο ανεξάρτητου συμβούλου με σελίδες υπηρεσιών, ενότητα portfolio και ιστολόγιο.*
- *Πρόσθεσε μια φόρμα επικοινωνίας στη σελίδα Επικοινωνία χρησιμοποιώντας τον δημιουργό ιστότοπου.*
- *Αφού εγκριθεί η λίστα ελέγχου εκκίνησης του ιστότοπου, στείλε μια υπενθύμιση SMS στη διαμορφωμένη επαφή ενδιαφερόμενου.*

### Στούντιο Σχεδίασης

**Εστίαση:** Οπτική προσαρμογή — χρώματα, τυπογραφία, CSS και μοτίβα μπλοκ.

**Διαθέσιμα εργαλεία:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Τι κάνει καλά:**
- Εφαρμογή επώνυμων προεπιλογών θέματος (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- Λεπτομερής ρύθμιση καθολικής τυπογραφίας και παλετών χρωμάτων μέσω theme.json
- Έγχυση προσαρμοσμένου CSS για παρακάμψεις ειδικές για την επωνυμία
- Λήψη στιγμιότυπου οθόνης μιας σελίδας και αξιολόγησή του για ζητήματα σχεδίασης

**Αρχικές προτάσεις:**
- *Εφάρμοσε την προεπιλογή warm-editorial και μετά όρισε το κύριο χρώμα σε #2d6a4f.*
- *Πάρε ένα στιγμιότυπο οθόνης της αρχικής σελίδας και πες μου τι θα βελτίωνες.*
- *Δημιούργησε ένα επαναχρησιμοποιήσιμο μοτίβο hero block με εικόνα φόντου πλήρους πλάτους και κεντραρισμένη επικεφαλίδα.*

### Διαχειριστής Plugin

**Εστίαση:** Ανακάλυψη, εγκατάσταση και διαχείριση WordPress plugin.

**Διαθέσιμα εργαλεία:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Τι κάνει καλά:**
- Πρόταση του καλύτερου plugin για μια περιγραφόμενη περίπτωση χρήσης
- Εγκατάσταση πακέτων δυνατοτήτων από το μητρώο
- Περιήγηση στον διαθέσιμο κατάλογο δυνατοτήτων ανά κατηγορία

**Αρχικές προτάσεις:**
- *Ποιο είναι το καλύτερο plugin για έναν κατάλογο membership;*
- *Εγκατάστησε το πακέτο δυνατοτήτων WooCommerce.*
- *Δείξε μου όλα τα διαθέσιμα πακέτα δυνατοτήτων ηλεκτρονικού εμπορίου.*

---

### Βοηθός Υποστήριξης

**Εστίαση:** Απάντηση σε ερωτήσεις σχετικά με το περιεχόμενο ιστότοπου, τις ρυθμίσεις και τη διαμόρφωση WordPress.

**Διαθέσιμα εργαλεία:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Τι κάνει καλά:**
- Αναζήτηση των τρεχουσών ρυθμίσεων και επιλογών του ιστότοπου
- Εξήγηση των τύπων αναρτήσεων, ταξινομιών και μενού που έχουν διαμορφωθεί στον ιστότοπο
- Απάντηση σε ερωτήσεις τύπου «τι κάνει αυτή η ρύθμιση;» διαβάζοντας ζωντανές τιμές
- Λειτουργία ως διαγνωστικό επίπεδο μόνο για ανάγνωση πριν γίνουν αλλαγές

**Προτάσεις για αρχή:**
- *Ποια plugins και ρυθμίσεις είναι ενεργά αυτήν τη στιγμή σε αυτόν τον ιστότοπο;*
- *Κατάγραψε όλους τους προσαρμοσμένους τύπους αναρτήσεων που είναι καταχωρισμένοι σε αυτόν τον ιστότοπο.*
- *Ποια μενού πλοήγησης υπάρχουν και πού έχουν ανατεθεί;*

## Ενσωματώσεις αυτοματισμού Superdav

Όταν έχουν διαμορφωθεί οι ενσωματώσεις Superdav AI Agent v1.18.0, οι ενσωματωμένοι πράκτορες μπορούν να συμμετέχουν σε ασφαλέστερες ροές εργασίας αυτοματισμού που λαμβάνουν υπόψη το πρόγραμμα:

- **Εργαλεία ανάγνωσης Google Calendar** επιτρέπουν στους πράκτορες να επιθεωρούν διαμορφωμένα ημερολόγια και συμβάντα πριν συντάξουν επακόλουθη εργασία.
- **Αντιστοίχιση επαφών και συμμετεχόντων** βοηθά στην αντιστοίχιση συμμετεχόντων συμβάντων με χρήστες WordPress ή γνωστές επαφές.
- **Πύλες ανθρώπινης έγκρισης** διακόπτουν προσωρινά ευαίσθητες ενέργειες μέχρι ένας εξουσιοδοτημένος χρήστης να τις ελέγξει και να τις επιβεβαιώσει.
- **Εγγραφές υπενθυμίσεων** αποτρέπουν διπλές ειδοποιήσεις όταν προγραμματισμένες εργασίες επαναδοκιμάζουν ή επαναλαμβάνονται.
- **Ειδοποιήσεις SMS TextBee** στέλνουν διαμορφωμένα μηνύματα κειμένου μόνο όταν είναι ενεργοποιημένα τα διαπιστευτήρια SMS και τα δικαιώματα ροής εργασίας.

Προτεινόμενη ροή εργασίας: ζητήστε από τον πράκτορα να προετοιμάσει το μήνυμα ή την ενέργεια, ελέγξτε την προτροπή έγκρισης και έπειτα επιτρέψτε στην εγκεκριμένη ενέργεια να συνεχιστεί. Για επαναλαμβανόμενες υπενθυμίσεις, διατηρήστε ενεργοποιημένη την αποδιπλοποίηση υπενθυμίσεων ώστε το ίδιο συμβάν ή η ίδια επαφή να μην ειδοποιείται επανειλημμένα.

## Προσαρμογή πρακτόρων

Κάθε ενσωματωμένος πράκτορας μπορεί να επεκταθεί ή να αντικατασταθεί μέσω του φίλτρου `gratis_ai_agent_agents`.

### Προσθήκη προσαρμοσμένου system prompt σε υπάρχοντα πράκτορα

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### Καταχώριση νέου πράκτορα

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

Ο νέος πράκτορας εμφανίζεται στον Επιλογέα πρακτόρων αμέσως μετά την εκτέλεση του φίλτρου.

### Αφαίρεση ενσωματωμένου πράκτορα

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
