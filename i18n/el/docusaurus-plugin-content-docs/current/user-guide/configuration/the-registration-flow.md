# Η Ροή Εγγραφής (v2) {#the-registration-flow-v2}

_**ΣΗΜΑΝΤΙΚΗ ΣΗΜΕΙΩΣΗ: Αυτό το άρθρο αναφέρεται στην έκδοση 2.x του Ultimate Multisite.**_

Οι χρήστες μπορούν να εγγραφούν στο δίκτυό σας με διαφορετικούς τρόπους. Μπορούν να χρησιμοποιήσουν τη φόρμα εγγραφής σας ή έναν κοινοποιήσιμο σύνδεσμο προς ένα προεπιλεγμένο πλάνο. Εδώ θα σας δείξουμε πώς οι πελάτες σας μπορούν να εγγραφούν στο δίκτυό σας χρησιμοποιώντας τις διαθέσιμες διαδρομές και τι συμβαίνει αφού εγγραφούν στο δίκτυό σας.

## Χρήση της Φόρμας Εγγραφής: {#using-the-registration-form}

Αυτή είναι η τυπική διαδικασία εγγραφής. Δημιουργείτε μια σελίδα εγγραφής με μια **φόρμα checkout** και αυτό θα είναι το σημείο όπου οι πελάτες σας θα μεταβαίνουν για να εγγραφούν στο δίκτυό σας και να εγγραφούν σε ένα πλάνο. Μπορείτε να έχετε πολλές σελίδες εγγραφής, καθεμία με διαφορετική φόρμα εγγραφής αν θέλετε.

Η προεπιλεγμένη σελίδα για την εγγραφή είναι η [_**yourdomain.com/register**_](http://yourdomain.com/register), αλλά μπορείτε να την αλλάξετε οποιαδήποτε στιγμή στο **Ultimate Multisite > Settings > Login & Registration > Default Registration Page**.

Αφού ένας χρήστης φτάσει στη σελίδα εγγραφής σας (συνήθως κάνοντας κλικ σε ένα κουμπί **Σύνδεση** ή **Αγορά τώρα**), θα δει εκεί τη φόρμα εγγραφής σας.

![Φόρμα εγγραφής που εμφανίζεται στη σελίδα εγγραφής](/img/frontend/registration-form.png)

Ακολουθεί ένα παράδειγμα της φόρμας checkout όπως εμφανίζεται στο frontend:

![Φόρμα εγγραφής checkout στο frontend](/img/config/checkout-frontend-registration.png)

Το μόνο που πρέπει να κάνουν είναι να συμπληρώσουν όλα τα υποχρεωτικά πεδία - email, όνομα χρήστη, κωδικό πρόσβασης, κ.λπ... - και να πληρώσουν για το πλάνο ή να επιβεβαιώσουν τη διεύθυνση email τους αν εγγράφονται για ένα δωρεάν πλάνο ή για ένα πληρωμένο πλάνο με δοκιμαστική περίοδο χωρίς πληροφορίες πληρωμής.

Στη σελίδα "Ευχαριστούμε", θα δουν ένα μήνυμα που τους ενημερώνει αν χρειάζεται να επιβεβαιώσουν τη διεύθυνση email τους ή αν ο ιστότοπός τους έχει ήδη ενεργοποιηθεί και μπορούν να αρχίσουν να τον χρησιμοποιούν.

![Σελίδα Ευχαριστούμε μετά την εγγραφή](/img/frontend/registration-thank-you.png)

Αν απαιτείται επιβεβαίωση διεύθυνσης email, θα πρέπει να μεταβούν στα εισερχόμενα του email τους και να κάνουν κλικ στον σύνδεσμο επαλήθευσης. Ο ιστότοπός τους δεν θα ενεργοποιηθεί αν η διεύθυνση email τους δεν επαληθευτεί.

Αν έχουν εγγραφεί σε ένα πληρωμένο πλάνο ή η επαλήθευση email δεν είναι υποχρεωτική στο δίκτυό σας, ο ιστότοπός τους θα ενεργοποιηθεί αμέσως μετά το checkout και θα εμφανιστεί ένας σύνδεσμος για να συνδεθούν στο dashboard τους.

![Ιστότοπος ενεργοποιημένος με σύνδεσμο για σύνδεση στο dashboard](/img/frontend/site-activated.png)

## Χρήση Κοινοποιήσιμου Συνδέσμου: {#using-a-shareable-link}

Η διαδικασία εγγραφής χρησιμοποιώντας έναν κοινοποιήσιμο σύνδεσμο είναι βασικά η ίδια με τη φόρμα εγγραφής, με τη μόνη διαφορά ότι χρησιμοποιώντας έναν κοινοποιήσιμο σύνδεσμο, οι πελάτες σας μπορούν να έχουν ένα προϊόν ή πρότυπο ιστότοπου προεπιλεγμένο στη φόρμα checkout (ανατρέξτε στην ενότητα Προεπιλογή προϊόντων και προτύπων μέσω παραμέτρων URL) ή ίσως να έχει προστεθεί ένας κωδικός κουπονιού (ανατρέξτε στην ενότητα Χρήση Παραμέτρων URL).

Η διαδικασία εγγραφής θα είναι η ίδια: θα πρέπει να συμπληρώσουν το όνομά τους, το όνομα χρήστη, τη διεύθυνση email, το όνομα και τον τίτλο του ιστότοπου, κ.λπ... αλλά το πλάνο ή το πρότυπο ιστότοπου θα είναι ήδη προεπιλεγμένο για αυτούς.

### Εγγραφή με Χρήση Χειροκίνητων Πληρωμών: {#registering-using-manual-payments}

Αν δεν θέλετε να χρησιμοποιήσετε PayPal, Stripe ή οποιαδήποτε άλλη πύλη πληρωμών που προσφέρεται από το Ultimate Multisite ή τις ενσωματώσεις add-on του, μπορείτε να χρησιμοποιήσετε χειροκίνητες πληρωμές για τους πελάτες σας. Με αυτόν τον τρόπο, μπορείτε να δημιουργήσετε ένα τιμολόγιο για να το πληρώσουν στον προτιμώμενο επεξεργαστή πληρωμών σας αφού εγγραφούν στο δίκτυό σας.

Η διαδικασία εγγραφής θα είναι ακριβώς η ίδια όπως παραπάνω, αλλά στη σελίδα εγγραφής οι πελάτες σας θα δουν ένα μήνυμα που δηλώνει ότι θα λάβουν ένα email με περαιτέρω οδηγίες για να ολοκληρώσουν την πληρωμή.

![Μήνυμα χειροκίνητης πληρωμής κατά την εγγραφή](/img/frontend/registration-manual-notice.png)

Και αφού ολοκληρωθεί η εγγραφή, θα δουν τις οδηγίες πληρωμής που έχετε ορίσει (και θα τις λάβουν επίσης στο email τους).

![Οδηγίες πληρωμής που εμφανίζονται μετά την εγγραφή](/img/frontend/registration-payment-instructions.png)

Οι οδηγίες πληρωμής μπορούν να αλλάξουν στο **Ultimate Multisite > Settings > Payments** αφού ενεργοποιήσετε την επιλογή πληρωμής **Manual**:

![Εναλλαγή χειροκίνητης πληρωμής με πεδίο οδηγιών πληρωμής](/img/config/manual-gateway-settings.png)

Αφού οι πελάτες σας ολοκληρώσουν τη χειροκίνητη πληρωμή και σας στείλουν την επιβεβαίωση, πρέπει να **επιβεβαιώσετε χειροκίνητα την πληρωμή** για να ενεργοποιήσετε τη συνδρομή και τον ιστότοπο του πελάτη.

Για να το κάνετε αυτό, μεταβείτε στο **Ultimate Multisite > Payments** και βρείτε την πληρωμή του πελάτη. Θα πρέπει ακόμη να εμφανίζει κατάσταση **Pending**.

![Λίστα πληρωμών με εκκρεμή χειροκίνητη πληρωμή](/img/admin/payments-list.png)

Κάντε κλικ στον αριθμό πληρωμής και θα μπορείτε να αλλάξετε την κατάστασή της σε **Completed**.

![Σελίδα λεπτομερειών πληρωμής](/img/admin/payment-edit.png)

![Αλλαγή κατάστασης πληρωμής σε Completed](/img/admin/payment-status-completed.png)

Αφού αλλάξετε την κατάστασή της σε **Completed** , θα πρέπει να δείτε ένα μήνυμα **Activate membership**. Ενεργοποιήστε αυτήν την επιλογή **on** για να ενεργοποιήσετε τη συνδρομή και τον ιστότοπο που σχετίζονται με αυτόν τον πελάτη. Στη συνέχεια, κάντε κλικ στο **Save Payment**.

![Εναλλαγή ενεργοποίησης συνδρομής και κουμπί Save Payment](/img/admin/payment-activate-membership.png)

Ο πελάτης σας θα πρέπει τώρα να μπορεί να έχει πρόσβαση στο dashboard και σε όλες τις λειτουργίες στις οποίες έχει εγγραφεί.
