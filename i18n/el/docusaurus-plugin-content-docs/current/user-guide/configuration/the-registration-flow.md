# Η Διαδικασία Εγγραφής (v2)

_**ΣΗΜΑΝΤΙΚΗ ΣΗΜΕΙΩΣΗ: Αυτό το άρθρο αναφέρεται στην έκδοση 2.x του Ultimate Multisite.**_

Οι χρήστες μπορούν να εγγραφούν στο δίκτυό σας με διάφορους τρόπους. Μπορούν να χρησιμοποιήσουν τη φόρμα εγγραφής σας ή έναν σύνδεσμο κοινοποίησης για ένα προεπιλεγμένο πλάνο. Εδώ θα σας δείξουμε πώς μπορούν οι πελάτες σας να εγγραφούν στο δίκτυό σας χρησιμοποιώντας τις διαθέσιμες διαδρομές και τι συμβαίνει μετά την εγγραφή τους.

## Χρήση της Φόρμας Εγγραφής:

Αυτή είναι η τυπική διαδικασία εγγραφής. Δημιουργείτε μια σελίδα εγγραφής με μια **φόρμα checkout** και εκεί θα πηγαίνουν οι πελάτες σας για να εγγραφούν στο δίκτυό σας και να κάνουν συνδρομή σε ένα πλάνο. Μπορείτε να έχετε πολλαπλές σελίδες εγγραφής, κάθε μία με διαφορετική φόρμα εγγραφής αν θέλετε.

Η προεπιλεγμένη σελίδα εγγραφής είναι η [_**yourdomain.com/register**_](http://yourdomain.com/register), αλλά μπορείτε να την αλλάξετε ανά πάσα στιγμή στο **Ultimate Multisite > Settings > Login & Registration > Default Registration Page**.

Αφού ο χρήστης φτάσει στη σελίδα εγγραφής σας (συνήθως κάνοντας κλικ σε ένα κουμπί **Σύνδεση** ή **Αγορά τώρα**), θα δει εκεί τη φόρμα εγγραφής σας.

![Φόρμα εγγραφής που εμφανίζεται στη σελίδα εγγραφής](/img/config/checkout-forms-list.png)

Το μόνο που χρειάζεται να κάνουν είναι να συμπληρώσουν όλα τα υποχρεωτικά πεδία - email, όνομα χρήστη, κωδικό πρόσβασης, κ.λπ... - και να πληρώσουν για το πλάνο ή να επιβεβαιώσουν τη διεύθυνση email τους αν εγγράφονται σε δωρεάν πλάνο ή σε πλάνο με δοκιμαστική περίοδο χωρίς στοιχεία πληρωμής.

Στη σελίδα "Ευχαριστούμε", θα δουν ένα μήνυμα που τους ενημερώνει αν χρειάζεται να επιβεβαιώσουν τη διεύθυνση email τους ή αν ο ιστότοπός τους είναι ήδη ενεργοποιημένος και μπορούν να αρχίσουν να τον χρησιμοποιούν.

![Σελίδα Ευχαριστούμε μετά την εγγραφή](/img/config/checkout-form-editor.png)

Αν απαιτείται επιβεβαίωση διεύθυνσης email, θα χρειαστεί να μεταβούν στα εισερχόμενά τους και να κάνουν κλικ στον σύνδεσμο επαλήθευσης. Ο ιστότοπός τους δεν θα ενεργοποιηθεί αν δεν επαληθευτεί η διεύθυνση email τους.

Αν έχουν εγγραφεί σε πλάνο επί πληρωμή ή η επαλήθευση email δεν είναι υποχρεωτική στο δίκτυό σας, ο ιστότοπός τους θα ενεργοποιηθεί αμέσως μετά το checkout και θα τους εμφανιστεί ένας σύνδεσμος για να συνδεθούν στον πίνακα ελέγχου τους.

![Ιστότοπος ενεργοποιημένος με σύνδεσμο σύνδεσης στον πίνακα ελέγχου](/img/config/checkout-form-editor.png)

## Χρήση Συνδέσμου Κοινοποίησης:

Η διαδικασία εγγραφής με σύνδεσμο κοινοποίησης είναι ουσιαστικά η ίδια με τη φόρμα εγγραφής, με τη μόνη διαφορά ότι χρησιμοποιώντας σύνδεσμο κοινοποίησης, οι πελάτες σας μπορούν να έχουν ένα προϊόν ή πρότυπο ιστότοπου προεπιλεγμένο στη φόρμα checkout (ανατρέξτε στην ενότητα Προεπιλογή προϊόντων και προτύπων μέσω παραμέτρων URL) ή ίσως έναν κωδικό κουπονιού ήδη προσθεμένο (ανατρέξτε στην ενότητα Χρήση Παραμέτρων URL).

Η διαδικασία εγγραφής θα είναι η ίδια: θα χρειαστεί να συμπληρώσουν το όνομά τους, το όνομα χρήστη, τη διεύθυνση email, το όνομα και τον τίτλο του ιστότοπου, κ.λπ... αλλά το πλάνο ή το πρότυπο ιστότοπου θα είναι ήδη προεπιλεγμένα για αυτούς.

### Εγγραφή με Χειροκίνητες Πληρωμές:

Αν δεν θέλετε να χρησιμοποιήσετε PayPal, Stripe ή οποιαδήποτε άλλη πύλη πληρωμών που προσφέρεται από το Ultimate Multisite ή τα πρόσθετα ενσωματώσεών του, μπορείτε να χρησιμοποιήσετε χειροκίνητες πληρωμές για τους πελάτες σας. Με αυτόν τον τρόπο, μπορείτε να δημιουργήσετε ένα τιμολόγιο για να πληρώσουν στον πάροχο πληρωμών της προτίμησής σας μετά την εγγραφή τους στο δίκτυό σας.

Η διαδικασία εγγραφής θα είναι ακριβώς η ίδια με παραπάνω, αλλά στη σελίδα εγγραφής οι πελάτες σας θα δουν ένα μήνυμα που αναφέρει ότι θα λάβουν ένα email με περαιτέρω οδηγίες για την ολοκλήρωση της πληρωμής.

![Μήνυμα χειροκίνητης πληρωμής κατά την εγγραφή](/img/config/settings-payment-gateways.png)

Και μετά την ολοκλήρωση της εγγραφής, θα δουν τις οδηγίες πληρωμής που ορίσατε (και θα τις λάβουν επίσης στο email τους).

![Οδηγίες πληρωμής που εμφανίζονται μετά την εγγραφή](/img/config/settings-payment-gateways.png)

Οι οδηγίες πληρωμής μπορούν να αλλάξουν στο **Ultimate Multisite > Settings > Payments** αφού ενεργοποιήσετε την επιλογή πληρωμής **Manual**:

![Διακόπτης χειροκίνητης πληρωμής με πεδίο οδηγιών πληρωμής](/img/config/settings-payment-gateways.png)

Αφού οι πελάτες σας ολοκληρώσουν τη χειροκίνητη πληρωμή και σας στείλουν την επιβεβαίωση, πρέπει να **επιβεβαιώσετε χειροκίνητα την πληρωμή** για να ενεργοποιήσετε τη συνδρομή και τον ιστότοπο του πελάτη.

Για να το κάνετε αυτό, μεταβείτε στο **Ultimate Multisite > Payments** και βρείτε την πληρωμή του πελάτη. Θα πρέπει να εμφανίζει ακόμα κατάσταση **Pending**.

![Λίστα πληρωμών με εκκρεμή χειροκίνητη πληρωμή](/img/admin/payments-list.png)

Κάντε κλικ στον αριθμό πληρωμής και θα μπορείτε να αλλάξετε την κατάστασή της σε **Completed**.

![Σελίδα λεπτομερειών πληρωμής](/img/admin/payments-list.png)

![Αλλαγή κατάστασης πληρωμής σε Completed](/img/admin/payments-list.png)

Μετά την αλλαγή της κατάστασης σε **Completed**, θα δείτε ένα μήνυμα **Activate membership**. Ενεργοποιήστε αυτή την επιλογή για να ενεργοποιήσετε τη συνδρομή και τον ιστότοπο που σχετίζεται με αυτόν τον πελάτη. Στη συνέχεια, κάντε κλικ στο **Save Payment**.

![Διακόπτης ενεργοποίησης συνδρομής και κουμπί Save Payment](/img/admin/payments-list.png)

Ο πελάτης σας θα πρέπει τώρα να μπορεί να έχει πρόσβαση στον πίνακα ελέγχου και σε όλες τις δυνατότητες στις οποίες έχει εγγραφεί.
