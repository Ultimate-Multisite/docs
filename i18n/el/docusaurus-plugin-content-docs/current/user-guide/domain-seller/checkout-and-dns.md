# Διαχείριση πεδίου checkout και DNS των πελατών

## Το πεδίο επιλογής ονόματος χώρου (Domain Selection)

Το πεδίο **Domain Selection** είναι ένα στοιχείο checkout που δίνει στους πελάτες την επιλογή για το πώς θέλουν να αποκτήσουν το όνομα χώρου (domain) για την ιστοσελίδα τους. Προσθέστε το σε οποιαδήποτε φόρμα checkout για να ενεργοποιήσετε την πώληση ονομάτων χώρου.

### Προσθήκη του πεδίου σε μια φόρμα checkout

1. Πηγαίνετε στο **Network Admin › Ultimate Multisite › Checkout Forms**
2. Ανοίξτε ή δημιουργήστε μια φόρμα checkout
3. Στο editor της checkout, κάντε κλικ στο **Add Field**
4. Επιλέξτε **Domain Selection** από τη λίστα των πεδίων
5. Διαμορφώστε τις επιλογές του πεδίου (βλ. παρακάτω)
6. Αποθηκεύστε τη φόρμα

### Επιλογές πεδίου

**Domain modes** — Επιλέξτε ποια καρτέλα (tab) θα δει ο πελάτης. Κάθε λειτουργία μπορεί να ενεργοποιηθεί ή να απενεργοποιηθεί ανεξάρτητα:

| Mode | Τι κάνει |
|---|---|
| **Subdomain** | Ο πελάτης χρησιμοποιεί ένα δωρεάν subdomain στο δικό σας network (π.χ., `mysite.yournetwork.com`). Δεν απαιτείται πληρωμή. |
| **Register New Domain** | Ο πελάτης αναζητά ένα νέο όνομα χώρου και το καταχωρεί μέσω του παρόχου που έχετε ρυθμίσει. Χρησιμοποιεί το προϊόν που αντιστοιχεί στο όνομα χώρου για την τιμολόγηση. |
| **Existing Domain** | Ο πελάτης συνδέει ένα όνομα χώρου που διαθέτει ήδη. Δεν υπάρχει τέλος καταχώρισης. Το όνομα χώρου συνδέεται αυτόματα με την ιστοσελίδα του. |

**Default mode** — Όταν ενεργοποιούνται και οι τρεις λειτουργίες, ποια καρτέλα ανοίγει πρώτη. Ρυστεί στο **Subdomain** για να παραμείνει η καταχώριση ονόματος χώρου προαιρετική, ή στο **Register New Domain** για να ενθαρρύνετε τις αγορές.

**Domain product** — Επιλέξτε προαιρετικά να συνδέσετε αυτό το πεδίο με ένα συγκεκριμένο προϊόν ονόματος χώρου. Αν δεν ρυθμιστεί, το addon επιλέγει αυτόματα το αντιστοίλειο προϊόν με βάση το TLD που αναζητά ο πελάτης.

### Πεδία επικοινωνίας του καταχωρητή (Registrant contact fields)

Όταν ο πελάτης επιλέγει την καρτέλα **Register New Domain**, η φόρμα checkout προσθέτει inline πεδία επικοινωνίας του καταχωρητή:

- Πρωτοεмя / Επώνυμο
- Διεύθυνση email
- Διεύθυνση (γραμμή 1, πόλη, πολιτεία/περιοχή, ταχυδρομικός κώδικας, χώρα)
- Τηλέφωνο

Αυτά είναι υποχρεωτικά από όλους τους καταχωρητές και επικυρώνονται πριν γίνει η κλήση API καταχώρισης. Τα τηλέφωνα μορφοποιούνται αυτόματα στο διεθνές φορμά `+CC.NNN` που αναμένεται από τους καταχωρητές.

### Αυτογεννημένη URL της ιστοσελίδας

Όταν ένας πελάτης καταχωρεί ή συνδέει ένα όνομα χώρου, το πεδίο URL της ιστοσελίδας συμπληρώνεται αυτόματα από το επιλεγμένο όνομα χώρου. Οι πελάτες δεν χρειάζεται να συμπληρώσουν ένα ξεχωριστό πεδίο URL.

### Συμπεριφορά αναζήτησης

- Η διαθεσιμότητα του ονόματος χώρου ελέγχεται σε πραγματικό χρόνο με AJAX καθώς ο πελάτης πληκτρολογεί
- Προτείνονται εναλλακτικές επιλογές TLD όταν το προτιμώμενο όνομα χώρου δεν είναι διαθέσιμο
- Η τιμολόγηση λαμβάνεται ζωντανά και εμφανίζεται με σαφήνεια (τιμή καταχώρισης, τιμή ανανέωσης, προαιρετικό τέλος WHOIS privacy)
- Οι κωδικοί κουπονιών εφαρμόζονται στα προϊόντα ονομάτων χώρου το ίδιο με οποιοδήποτε άλλο προϊόν

**Βελτιστοποίηση της απόκρισης της αναζήτησης:**

```php
// Αυξήστε το debounce delay (milliseconds) για μείωση των κλήσεων API σε αργές συνδέσεις
add_filter('wu_domain_seller_search_delay', function($delay) {
    return 800; // default: 500
});
```

**Προσθήκη προσαρμοσμένων πεδίων στη φόρμα αναζήτησης ονόματος χώρου:**

```php
add_filter('wu_checkout_form_register_domain_form_fields', function($fields) {
    $fields['custom_note'] = [
        'type'  => 'text',
        'label' => 'Additional notes',
    ];
    return $fields;
});
```

## Διαχείριση DNS των πελατών

Οι πελάτες μπορούν να διαχειριστούν τα DNS records για τα καταχωρημένα ονόματα χώρου τους από τη σελίδα **My Account**, κάτω από την καταχώριση του ονόματος χώρου τους.

### Υποστηριζόμενοι τύποι records

| Type | Χρήση |
|---|---|
| **A** | Συνδέει το όνομα host με διεύθυνση IPv4 |
| **AAAA** | Συνδέει το όνομα host με διεύθυνση IPv6 |
| **CNAME** | Δημιουργεί ένα alias που δείχνει σε άλλο όνομα host |
| **MX** | Ρυθμίζει τον εξμεταλλακτικό διακομιστή email (mail exchange server) |
| **TXT** | Προσθέτει records κειμένου SPF, DMARC, επαλήθευσης ή άλλες |

### Ποιοι πάροχοι υποστηρίζουν τη διαχείριση DNS;

Η διαχείριση DNS (προσθήκη, επεξεργασία, διαγραφή records) είναι διαθέσιμη με το **OpenSRS**, το **ResellerClub** και το **Enom**. Τα ονόματα χώρου Namecheap, GoDaddy και NameSilo εμφανίζουν πληροφορίες κατάστασης και λήξης, αλλά τα DNS πρέπει να διαχειρίζονται απευθείας στο control panel του καταχωρητή.

### Προεπιλεγμένα DNS records

Μπορείτε να ρυθμίσετε προεπιλεγμένα DNS records που εφαρμόζονται αυτόματα όταν καταχωρείται ένα όνομα χώρου. Πηγαίνετε στο **Settings › Domain Seller › Default DNS Records**.

Οι προεπιλεγμένες τιμές των records υποστηρίζουν δύο tokens:

| Token | Αντικαθίσταται με |
|---|---|
| `{DOMAIN}` | Το καταχωρημένο όνομα ονόματος χώρου (π.χ., `example.com`) |
| `{SITE_URL}` | Η URL της ιστοσελίδας WordPress του πελάτη |

**Παράδειγμα — σύνδεση του apex domain και του www στον IP του server σας:**

```
Type: A
Name: {DOMAIN}
Value: 203.0.113.10

Type: CNAME
Name: www.{DOMAIN}
Value: {DOMAIN}
```

### Admin: προβολή και επεξεργασία DNS

Οι network admins μπορούν να προсмотрят και να επεξεργαστούν τα DNS records για οποιοδήποτε όνομα χώρου πελάτη από τη σελίδα επεξεργασίας του ονόματος χώρου στο **Network Admin › Ultimate Multisite › Domains**.
