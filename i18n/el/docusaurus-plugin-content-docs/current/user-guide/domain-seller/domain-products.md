# Προϊόντα και Τιμολόγηση Domain {#domain-products-and-pricing}

Τα προϊόντα domain είναι ο τρόπος με τον οποίο ελέγχετε τις τιμές, τις TLDs, την ιδιωτικότητα WHOIS και την επιλογή παρόχου. Κάθε προϊόν domain είναι ένα τυπικό προϊόν Multisite Ultimate με μια επιπλέον καρτέλα **Domain Settings**.

## Δημιουργία προϊόντος domain {#creating-a-domain-product}

1. Πηγαίνετε στο **Network Admin › Ultimate Multisite › Products**
2. Κάντε κλικ στο **Add New**
3. Ρυθμίστε τον τύπο του προϊόντος σε **Domain**
4. Διαμορφώστε την καρτέλα **Domain Settings** (βλ. παρακάτω)
5. Αποθήκευση

Τα προϊόντα domain εμφανίζονται με ένα μωβτα **Domain** badge στη λίστα προϊόντων και μπορούν να φιλτριστούν χρησιμοποιώντας την καρτέλα **Domain Products**.

## Καρτέλα ρυθμίσεων domain {#domain-settings-tab}

### Provider {#provider}

Επιλέξτε ποιος καταχωρητής (registrar) χειρίζεται την εγγραφή για αυτό το προϊόν. Το προεπιλεγμένο είναι το παγκόσμιο **Default provider** που ρυθμίστηκε στις ρυθμίσεις Domain Seller.

### Supported TLDs {#supported-tlds}

Αφήστε κενό για να δημιουργήσετε ένα **catch-all product** που εφαρμόζεται σε όλες τις TLDs που δεν ταιριάζουν με κανένα άλλο προϊόν.

Εισάγετε μια λίστα TLDs που χωρίζονται με κόμμα (π.χ., `.com, .net, .org`) για να δημιουργήσετε ένα **TLD-specific product** που εφαρμόζεται μόνο σε αυτές τις επεκτάσεις.

**Πώς λειτουργεί η αντιστοίχιση προϊόντων:** Όταν ένας πελάτης αναζητά ένα domain, το add-on επιλέγει το πιο συγκεκριμένο προϊόν που ταιριάζει. Ένα προϊόν με `.com` στη λίστα TLDs του έχει προτεραιότητα έναντι ενός catch-all product. Εάν κανένα TLD-specific product δεν ταιριάζει, χρησιμοποιείται το catch-all. Εάν δεν υπάρχουν προϊόντα, δεν εμφανίζεται η αναζήτηση domain.

### Markup type {#markup-type}

Τρεις λειτουργίες ελέγχουν πώς υπολογίζεται η λιανική σας τιμή από το λιανικό κόστος:

| Mode | How it works |
|---|---|
| **Percentage** | Προσθέτει μια ποσοστιαία αύξηση στο λιανικό κόστος. Μια αύξηση 20% σε ένα domain λιανικού κόστους $10 δίνει $12. |
| **Fixed markup** | Προσθέτει ένα σταθερό ποσό σε δολάρια. Μια αύξηση $5 σε ένα domain $10 δίνει $15. |
| **Fixed price** | Αποτυγχάνει εντελώς το λιανικό κόστος. Χρεώνει πάντα το ποσό που εισάγετε. |

### Introductory pricing {#introductory-pricing}

Ενεργοποιήστε το για να προσφέρετε μια μειωμένη τιμή για το πρώτο έτος. Ρυθμίστε μια ξεχωριστή **Introductory price** (η τιμή του έτους 1) μαζί με την κανονική **Renewal price** (έτος 2+). Ο πελάτης βλέπει και τις δύο τιμές κατά την ολοκλήρωση της αγοράς, ώστε να γνωρίζει τι να περιμένει κατά την ανανέωση.

### WHOIS privacy {#whois-privacy}

Ελέγχει εάν προσφέρεται προστασία ιδιωτικότητας WHOIS για τα domains που καταχωρούνται μέσω αυτού του προϊόντος.

| Setting | Behaviour |
|---|---|
| **Disabled** | Η ιδιωτικότητα WHOIS δεν προσφέρεται ή ενεργοποιείται ποτέ. |
| **Always Included** | Η ιδιωτικότητα WHOIS ενεργοποιείται αυτόματα κατά την εγγραφή χωρίς χρέωση. |
| **Customer Choice** | Εμφανίζεται ένα πλαίσιο επιλογής κατά την ολοκλήρωση της αγοράς. Ρυθμίστε την **Privacy price** για να χρεώνεται ανά έτος, ή αφήστε το στο $0 για να το προσφέρετε δωρεάν. |

Για το Namecheap, η ιδιωτικότητα WHOIS χρησιμοποιεί το WhoisGuard (πάντα δωρεάν). Για το OpenSRS, χρησιμοποιεί την υπηρεσία ιδιωτικότητας OpenSRS (μπορεί να έχει κόστος στο λιανικό).

## Εισαγωγή και συγχρονισμός TLD {#tld-import-and-sync}

Τα προϊόντα domain εμφανίζουν τιμές λιανικού κόστους σε πραγματικό χρόνο που λαμβάνονται από τον συνδεδεμένο πάροχο. Για να λειτουργήσει αυτό, πρέπει να εισαχθούν οι TLDs.

- **Manual sync:** Settings › Domain Seller › Sync TLDs (per provider)
- **Automatic sync:** Runs daily via a scheduled cron job across all configured providers

Μετά τον συγχρονισμό, μεταβείτε στην καρτέλα Domain Settings οποιουδήποτε προϊόντος domain και χρησιμοποιήστε τον επιλογέα TLD για να δείτε τις διαθέσιμες TLDs με τις τρέχουσες τιμές λιανικού κόστους τους.

## Αυτόματη ανανέωση {#auto-renewal}

Οι ανανεώσεις domain συνδέονται με την κατάσταση ταμείου (membership status) του πελάτη:

- Όταν ένα ταμείο ανανεώνεται και ένα domain είναι συνδεδεμένο, η ανανέωση του domain τίθεται αυτόματα σε cola.
- Οι προσπάθειες ανανέωσης χρησιμοποιούν την ενεργή πύλη πληρωμής του πελάτη.
- Οι αποτυχημένες ανανεώσεις επαναπροσπαθούνται αυτόματα με εκθετική υποπίεση (exponential backoff).
- Στέλνονται email ειδοποιήσεις για επιτυχημένες ανανεώσεις, αποτυχίες και επερχόμενες λήξεις.

Ταżs]<Email template IDs> για τα γεγονότα κύκλου ζωής του domain:

| Event | Template ID |
|---|---|
| Domain registered | `domain_registered` |
| Domain renewed | `domain_renewed` |
| Renewal failed | `domain_renewal_failed` |
| Domain expiring soon | `domain_expiring_soon` |

## Διαχειριστής: Χειροκίνητη εγγραφή domain {#admin-manual-domain-registration}

Για να καταχωρήσετε ένα domain για λογαριασμό ενός πελάτη χωρίς να περάσει από την ολοκλήρωση της αγοράς:

1. Πηγαίνετε στο **Network Admin › Ultimate Multisite › Register Domain**
2. Επιλέξτε τον πελάτη και εισάγετε το όνομα domain
3. Συμπληρώστε τα στοιχεία επικοινωνίας του καταχωρητή (όνομα, διεύθυνση, τηλέφωνο)
4. Κάντε κλικ στο **Register**

Η καταχώριση του domain δημιουργείται και συνδέεται με τον λογαριασμό του πελάτη.
