# Domain Seller: Ρύθμιση και διαμόρφωση παρόχου {#domain-seller-setup-and-provider-configuration}

Το πρόσθετο Domain Seller περιλαμβάνει έναν καθοδηγούμενο οδηγό ρύθμισης που σας καθοδηγεί σε κάθε απαιτούμενο βήμα. Αυτή η σελίδα καλύπτει τη ροή του οδηγού και το πώς να διαμορφώσετε ή να επαναδιαμορφώσετε παρόχους στη συνέχεια.

## Απαιτήσεις {#requirements}

- **Multisite Ultimate** v2.4.12 ή νεότερη, ενεργοποιημένο στο δίκτυο
- **PHP** 7.4+
- Διαπιστευτήρια API για τουλάχιστον έναν υποστηριζόμενο καταχωρητή

## Οδηγός ρύθμισης πρώτης εκτέλεσης {#first-run-setup-wizard}

Ο οδηγός ρύθμισης ξεκινά αυτόματα την πρώτη φορά που ενεργοποιείτε το πρόσθετο στο δίκτυο. Είναι επίσης διαθέσιμος ανά πάσα στιγμή από **Network Admin › Ultimate Multisite › Domain Seller Setup**.

### Βήμα 1 — Επιλογή παρόχου {#step-1--choose-a-provider}

Επιλέξτε τον καταχωρητή που θέλετε να συνδέσετε. Υποστηριζόμενες επιλογές:

| Πάροχος | Διαχείριση DNS | Απόρρητο WHOIS |
|---|---|---|
| OpenSRS | Ναι | Ναι |
| Namecheap | Όχι | Ναι (WhoisGuard, δωρεάν) |
| HostAfrica | Ναι | Ναι (προστασία ID) |
| Openprovider | Ναι | Ναι |
| Hostinger | Μέσω της βασικής αντιστοίχισης domain του Hostinger για φιλοξενούμενα domain | Ναι |
| GoDaddy | Όχι | Όχι |
| ResellerClub | Ναι | Όχι |
| NameSilo | Όχι | Όχι |
| Enom | Ναι | Όχι |

### Βήμα 2 — Εισαγωγή διαπιστευτηρίων {#step-2--enter-credentials}

Κάθε πάροχος έχει διαφορετικά πεδία διαπιστευτηρίων:

**OpenSRS** — Όνομα χρήστη και ιδιωτικό κλειδί (από το OpenSRS Reseller Control Panel)

**Namecheap** — Όνομα χρήστη και API key (από Account › Tools › API Access)

**HostAfrica** — Domains Reseller API endpoint και διαπιστευτήρια από το module μεταπωλητή του HostAfrica. Δεν τεκμηριώνεται προς το παρόν ξεχωριστό sandbox endpoint· δοκιμάστε με ασφαλείς ελέγχους μόνο για ανάγνωση πριν εκτελέσετε ζωντανές καταχωρίσεις.

**Openprovider** — Όνομα χρήστη και κωδικός πρόσβασης με ενεργοποιημένη πρόσβαση API. Η προαιρετική λειτουργία sandbox χρησιμοποιεί το Openprovider sandbox API, και ένα προαιρετικό προεπιλεγμένο customer handle μπορεί να επαναχρησιμοποιηθεί για καταχωρίσεις.

**Hostinger** — Το κοινόχρηστο Hostinger hPanel API token από τη βασική ενσωμάτωση Hostinger. Το ίδιο token τροφοδοτεί τη βασική αντιστοίχιση domain και τις λειτουργίες καταχώρισης του Domain Seller.

**GoDaddy** — API key και secret (από developer.godaddy.com)

**ResellerClub** — Reseller ID και API key (από τον πίνακα ελέγχου ResellerClub)

**NameSilo** — API key (από namesilo.com › Account › API Manager)

**Enom** — Account ID και API token

Επιλέξτε **Λειτουργία sandbox** όπου είναι διαθέσιμη, για να κάνετε δοκιμές με το περιβάλλον δοκιμών του παρόχου πριν βγείτε σε παραγωγή.

### Βήμα 3 — Δοκιμή της σύνδεσης {#step-3--test-the-connection}

Κάντε κλικ στο **Δοκιμή σύνδεσης**. Ο οδηγός στέλνει μια ελαφριά κλήση API για να επαληθεύσει τα διαπιστευτήρια και τη συνδεσιμότητα. Διορθώστε τυχόν προβλήματα διαπιστευτηρίων πριν συνεχίσετε.

### Βήμα 4 — Εισαγωγή TLDs {#step-4--import-tlds}

Κάντε κλικ στο **Εισαγωγή TLDs** για να αντλήσετε όλα τα διαθέσιμα TLDs και την τιμολόγηση χονδρικής από τον συνδεδεμένο πάροχο. Αυτό συμπληρώνει τη λίστα TLD που χρησιμοποιείται από τα προϊόντα domain. Η εισαγωγή μπορεί να διαρκέσει 30–60 δευτερόλεπτα για παρόχους με μεγάλους καταλόγους TLD.

Τα TLDs συγχρονίζονται επίσης ξανά αυτόματα μία φορά την ημέρα μέσω προγραμματισμένης εργασίας cron.

### Βήμα 5 — Δημιουργία προϊόντος domain {#step-5--create-a-domain-product}

Ο οδηγός δημιουργεί ένα προεπιλεγμένο καθολικό προϊόν domain με προσαύξηση 10%. Μπορείτε να επεξεργαστείτε αυτό το προϊόν αμέσως ή να το παραλείψετε και να δημιουργήσετε προϊόντα χειροκίνητα από **Ultimate Multisite › Products**.

Δείτε [Προϊόντα domain και τιμολόγηση](./domain-products) για τον πλήρη οδηγό διαμόρφωσης προϊόντος.

## Επαναδιαμόρφωση παρόχου {#reconfiguring-a-provider}

Μεταβείτε στο **Network Admin › Ultimate Multisite › Settings › Domain Seller** (ή κάντε κλικ στο **Settings** στη λίστα πρόσθετων).

Η σελίδα ρυθμίσεων περιέχει:

- **Ενεργοποίηση πώλησης domain** — ενεργοποιήστε/απενεργοποιήστε ολόκληρη τη λειτουργία
- **Προεπιλεγμένος πάροχος** — ο πάροχος που χρησιμοποιείται για αναζητήσεις domain και νέα προϊόντα
- **Μέγιστος αριθμός TLDs ανά αναζήτηση** — πόσα TLDs να ελέγχονται όταν ένας πελάτης κάνει αναζήτηση· οι υψηλότερες τιμές εμφανίζουν περισσότερες επιλογές αλλά είναι πιο αργές
- **Διάρκεια cache διαθεσιμότητας** — για πόσο χρόνο να αποθηκεύονται προσωρινά τα αποτελέσματα διαθεσιμότητας και τιμολόγησης· οι χαμηλότερες τιμές είναι πιο ακριβείς αλλά αυξάνουν τις κλήσεις API
- **Διαχείριση προϊόντων domain** — γρήγορος σύνδεσμος προς τη λίστα Products
- **Διαμόρφωση παρόχων** — ανοίγει τον Integration Wizard για προσθήκη ή επαναδιαμόρφωση παρόχων

### Προσθήκη δεύτερου παρόχου {#adding-a-second-provider}

Κάντε κλικ στο **Διαμόρφωση παρόχων** και εκτελέστε ξανά τον οδηγό για τον νέο καταχωρητή. Μπορείτε να έχετε πολλούς παρόχους διαμορφωμένους ταυτόχρονα. Αναθέστε κάθε προϊόν domain σε έναν συγκεκριμένο πάροχο ή αφήστε το στον προεπιλεγμένο.

### Χειροκίνητος συγχρονισμός TLDs {#syncing-tlds-manually}

Στη σελίδα ρυθμίσεων, κάντε κλικ στο **Συγχρονισμός TLDs** δίπλα σε οποιονδήποτε διαμορφωμένο πάροχο για να αντλήσετε την πιο πρόσφατη τιμολόγηση. Αυτό είναι χρήσιμο αφού ένας πάροχος ενημερώσει την τιμολόγηση χονδρικής ή προσθέσει νέα TLDs.

| `domain-seller-registration` | Όλες οι προσπάθειες καταχώρισης (επιτυχίες και αποτυχίες) |
| `domain-seller-renewal` | Αποτελέσματα εργασιών ανανέωσης |
| `domain-seller-opensrs` | Ακατέργαστη δραστηριότητα OpenSRS API |
| `domain-seller-namecheap` | Ακατέργαστη δραστηριότητα Namecheap API |
| `domain-seller-hostafrica` | Ακατέργαστη δραστηριότητα HostAfrica API |
| `domain-seller-openprovider` | Ακατέργαστη δραστηριότητα Openprovider API |
| `domain-seller-hostinger` | Ακατέργαστη δραστηριότητα Hostinger API |
| `domain-seller-godaddy` | Ακατέργαστη δραστηριότητα GoDaddy API |
| `domain-seller-resellerclub` | Ακατέργαστη δραστηριότητα ResellerClub API |
| `domain-seller-namesilo` | Ακατέργαστη δραστηριότητα NameSilo API |
| `domain-seller-enom` | Ακατέργαστη δραστηριότητα Enom API |

## Σημειώσεις δυνατοτήτων παρόχων {#provider-capability-notes}

Δεν εκθέτει κάθε API καταχωρητή τις ίδιες λειτουργίες. Το πρόσθετο εμφανίζει μη υποστηριζόμενες λειτουργίες με σαφή σφάλματα προς τον διαχειριστή, αντί να αποτυγχάνει σιωπηλά.

- **HostAfrica** υποστηρίζει την πιο εκτεταμένη ζωντανή ροή εργασίας μεταπωλητή, συμπεριλαμβανομένων αναζήτησης, συγχρονισμού TLD/τιμολόγησης, εγγραφής, ανανέωσης, μεταφοράς, ενημερώσεων nameserver, εγγραφών DNS, κωδικών EPP, κλειδώματος καταχωρητή και προστασίας ID.
- **Openprovider** υποστηρίζει συγχρονισμό TLD σε τιμές μεταπωλητή, εγγραφή, ανανέωση, μεταφορές, ενημερώσεις nameserver, ζώνες DNS, κωδικούς EPP, κλείδωμα καταχωρητή και απόρρητο WHOIS. Πραγματοποιεί έλεγχο ταυτότητας με ένα βραχύβιο bearer token που το addon ανανεώνει αυτόματα.
- **Hostinger** υποστηρίζει αναζήτηση διαθεσιμότητας, εγγραφή, αναζήτηση χαρτοφυλακίου, ενημερώσεις nameserver, κλείδωμα καταχωρητή και απόρρητο WHOIS μέσω του κοινόχρηστου hPanel API token. Το δημόσιο Domains API της Hostinger δεν εκθέτει τιμολόγηση μεταπωλητή/χονδρικής, εισερχόμενη μεταφορά, ρητή ανανέωση ή ανάκτηση κωδικού EPP· οι ανανεώσεις γίνονται μόνο με αυτόματη ανανέωση.
