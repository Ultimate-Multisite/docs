# Οι ιστότοποι του πελάτη σας (v2)

Μετά την εγγραφή στο δίκτυό σας, οι πελάτες σας μπορούν να αποκτήσουν πρόσβαση στο Dashboard τους.

Εκτός από τη σελίδα **Account**, όπου μπορούν να δουν σημαντικές πληροφορίες σχετικά με τις πληρωμές τους, τις συνδρομές, τους τομείς, τους περιορισμούς πλάνου και άλλα, έχουν επίσης πρόσβαση στη σελίδα **Οι ιστότοποί μου**, όπου μπορούν να δουν και να διαχειριστούν όλους τους ιστότοπούς τους. Το μόνο που χρειάζεται να κάνουν είναι να κάνουν κλικ στο Οι ιστότοποί μου στο Dashboard τους.

![Σύνδεσμος Οι ιστότοποί μου στο Dashboard του πελάτη](/img/account-page/my-sites-link.png)

Θα εμφανιστούν όλοι οι ιστότοποί τους, με ένα κουμπί **Διαχείριση** ακριβώς κάτω από τον καθέναν από αυτούς. Αν κάνουν κλικ σε αυτό, θα ανακατευθυνθούν στο **Dashboard του συγκεκριμένου ιστότοπου**.

Μπορούν επίσης να δουν ποιος από τους ιστότοπούς τους είναι ο **κύριος** για πλάνα που τους επιτρέπουν να έχουν περισσότερους από έναν ιστότοπο.

![Σελίδα Οι ιστότοποί μου που εμφανίζει τους ιστότοπους πελατών με κουμπιά Διαχείριση](/img/account-page/my-sites-page.png)

Ακολουθεί μια πλήρης προβολή του περιβάλλοντος επεξεργασίας ιστότοπου:

![Πλήρες περιβάλλον επεξεργασίας ιστότοπου](/img/admin/site-edit-full.png)

Αν θέλουν να αλλάξουν τον κύριο ιστότοπό τους, πρέπει να μεταβούν στη σελίδα Account κάνοντας κλικ στο μενού **Account**.

![Σύνδεσμος μενού Account στο Dashboard πελάτη](/img/account-page/account-menu.png)

Μπορείτε επίσης να προβάλετε και να επεξεργαστείτε τα στοιχεία πελάτη:

![Περιβάλλον επεξεργασίας πελάτη](/img/admin/customer-edit.png)

Στη συνέχεια, θα πρέπει να κάνουν κλικ στο **Αλλαγή προεπιλεγμένου ιστότοπου**.

![Κουμπί Αλλαγή προεπιλεγμένου ιστότοπου στη σελίδα Account](/img/account-page/change-default-site.png)

Θα εμφανιστεί ένα αναδυόμενο παράθυρο. Κάντε κλικ στην αναπτυσσόμενη λίστα και επιλέξτε ποιον ιστότοπο θέλετε να ορίσετε ως κύριο ιστότοπο.

![Επιλογή αναπτυσσόμενης λίστας Αλλαγή προεπιλεγμένου ιστότοπου](/img/account-page/change-default-site-dropdown.png)

Στη συνέχεια, απλώς κάντε κλικ στο **Αλλαγή προεπιλεγμένου ιστότοπου** και τελειώσατε!

![Επιβεβαίωση της αλλαγής προεπιλεγμένου ιστότοπου](/img/account-page/change-default-site-confirm.png)
