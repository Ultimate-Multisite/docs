# Οι Ιστοσελίδες των Πελατών σας (v2)

Μετά την εγγραφή τους στο δίκτυό σας, οι πελάτες σας μπορούν να έχουν πρόσβαση στον πίνακα ελέγχου τους.

Εκτός από τη σελίδα **Λογαριασμός**, όπου μπορούν να δουν σημαντικές πληροφορίες σχετικά με τις πληρωμές, τις συνδρομές, τα domains, τους περιορισμούς του πλάνου τους και άλλα, έχουν επίσης πρόσβαση στη σελίδα **Οι Ιστοσελίδες μου**, όπου μπορούν να δουν και να διαχειριστούν όλες τις ιστοσελίδες τους. Το μόνο που χρειάζεται να κάνουν είναι να κάνουν κλικ στο Οι Ιστοσελίδες μου στον πίνακα ελέγχου τους.

![Σύνδεσμος Οι Ιστοσελίδες μου στον πίνακα ελέγχου πελάτη](/img/admin/customers-list.png)

Θα τους εμφανιστούν όλες οι ιστοσελίδες τους, με ένα κουμπί **Διαχείριση** ακριβώς κάτω από κάθε μία. Αν κάνουν κλικ σε αυτό, θα μεταφερθούν στον **πίνακα ελέγχου της συγκεκριμένης ιστοσελίδας**.

Μπορούν επίσης να δουν ποια από τις ιστοσελίδες τους είναι η **κύρια**, για πλάνα που τους επιτρέπουν να έχουν περισσότερες από μία ιστοσελίδες.

![Σελίδα Οι Ιστοσελίδες μου που δείχνει τις ιστοσελίδες του πελάτη με κουμπιά Διαχείριση](/img/admin/customers-list.png)

Αν θέλουν να αλλάξουν την κύρια ιστοσελίδα τους, πρέπει να μεταβούν στη σελίδα Λογαριασμός κάνοντας κλικ στο μενού **Λογαριασμός**.

![Σύνδεσμος μενού Λογαριασμός στον πίνακα ελέγχου πελάτη](/img/admin/customers-list.png)

Στη συνέχεια, πρέπει να κάνουν κλικ στο **Αλλαγή Προεπιλεγμένης Ιστοσελίδας**.

![Κουμπί Αλλαγή Προεπιλεγμένης Ιστοσελίδας στη σελίδα λογαριασμού](/img/admin/customers-list.png)

Θα εμφανιστεί ένα αναδυόμενο παράθυρο. Κάντε κλικ στην αναπτυσσόμενη λίστα και επιλέξτε ποια ιστοσελίδα θέλετε να ορίσετε ως κύρια.

![Επιλογή από αναπτυσσόμενη λίστα για Αλλαγή Προεπιλεγμένης Ιστοσελίδας](/img/admin/customers-list.png)

Μετά, απλά κάντε κλικ στο **Αλλαγή Προεπιλεγμένης Ιστοσελίδας** και είστε έτοιμοι!

![Επιβεβαίωση αλλαγής προεπιλεγμένης ιστοσελίδας](/img/admin/customers-list.png)
