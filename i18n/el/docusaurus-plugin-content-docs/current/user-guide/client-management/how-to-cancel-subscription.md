# Πώς μπορώ να ακυρώσω τη συνδρομή μου;

Αν έχετε άδεια Ultimate Multisite, μπορείτε να ακυρώσετε την ανανέωσή της όποτε θέλετε. Απλά ακολουθήστε τα παρακάτω βήματα:


<!-- Screenshot unavailable: Account page email with access link -->

Χρησιμοποιήστε το email που δώσατε κατά την αγορά και τον κωδικό σας.

<!-- Screenshot unavailable: Login page with email and password fields -->

Αυτή είναι η αρχική οθόνη της Σελίδας Λογαριασμού σας:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

Για να ακυρώσετε τη συνδρομή σας, από το μενού στα αριστερά, μεταβείτε στην επιλογή _Ανανέωση & Χρέωση._

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

Κάντε κλικ στο βέλος στα δεξιά για να ανοίξει ένα πλαϊνό παράθυρο. Στη συνέχεια, επιλέξτε την επιλογή που θα ακυρώσει την **αυτόματη ανανέωση**.

Το σύστημα θα σας εμφανίσει ένα μήνυμα επιβεβαίωσης.

Αν είστε σίγουροι ότι θέλετε να ακυρώσετε τη συνδρομή σας, απλά κάντε κλικ στο κουμπί _**Ακύρωση Ανανεώσεων**_. Μετά από αυτή την ενέργεια, θα σας ζητηθεί να απαντήσετε σε μια σύντομη έρευνα.

Έτοιμο! Η συνδρομή σας δεν θα ανανεωθεί αυτόματα.

**Θα έχετε έγκυρο κλειδί μέχρι να λήξει η συνδρομή σας**. Σε περίπτωση που θέλετε να επανενεργοποιήσετε τη συνδρομή σας, θα πρέπει να το κάνετε χειροκίνητα.
