# Ακύρωση Συνδρομής (v2)

_**ΣΗΜΑΝΤΙΚΗ ΣΗΜΕΙΩΣΗ: Αυτό το άρθρο αναφέρεται στην έκδοση 2.x του Ultimate Multisite.**_

Η ακύρωση μιας συνδρομής από την πλευρά των πελατών σας είναι αρκετά εύκολη.

Αν ο χρήστης σας θέλει να ακυρώσει τη συνδρομή του, θα πρέπει πρώτα να μεταβεί στη σελίδα **Λογαριασμός**.

Εκεί, θα πρέπει να κατευθυνθεί στην ενότητα **Danger Zone** και να κάνει κλικ στο **Delete Account**.

![Ενότητα Danger Zone στη σελίδα λογαριασμού με το κουμπί Delete Account](/img/admin/memberships-list.png)

Θα ανοίξει ένα παράθυρο που θα ζητά επιβεβαίωση. Θα πρέπει να ενεργοποιήσει την επιλογή **Confirm Account Deletion** και να κάνει κλικ στο **Delete Account**.

![Παράθυρο επιβεβαίωσης διαγραφής λογαριασμού](/img/admin/memberships-list.png)

Αυτό θα τον αποσυνδέσει από τον ιστότοπο και θα ακυρώσει τη συνδρομή του στο δίκτυό σας, ενώ παράλληλα θα διαγράψει τον ιστότοπό του από το δίκτυό σας.

![Αποσύνδεση μετά τη διαγραφή λογαριασμού](/img/admin/memberships-list.png)

![Επιβεβαίωση ακύρωσης συνδρομής](/img/admin/memberships-list.png)

**Σημαντικό:** Σημειώστε ότι αυτή η ενέργεια δεν θα αφαιρέσει τον πελάτη από το δίκτυό σας, καθώς ένας πελάτης μπορεί να έχει περισσότερες από μία συνδρομές ή καμία συνδρομή συνδεδεμένη μαζί του.

![Ο πελάτης εξακολουθεί να υπάρχει στο δίκτυο μετά την ακύρωση](/img/admin/memberships-list.png) | ![Πελάτης χωρίς ενεργές συνδρομές](/img/admin/memberships-list.png)
