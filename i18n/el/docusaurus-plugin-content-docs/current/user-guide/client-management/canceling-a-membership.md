# Ακύρωση συνδρομής μέλους (v2) {#canceling-a-membership-v2}

_**ΣΗΜΑΝΤΙΚΗ ΣΗΜΕΙΩΣΗ: Αυτό το άρθρο αναφέρεται στο Ultimate Multisite έκδοση 2.x.**_

Η ακύρωση μιας συνδρομής μέλους από την πλευρά των πελατών σας είναι αρκετά εύκολη.

Αν ο χρήστης σας θέλει να ακυρώσει τη συνδρομή μέλους του, πρέπει πρώτα να αποκτήσει πρόσβαση στη σελίδα ****Account**** του.

Εκεί, πρέπει να μεταβεί στη **Ζώνη Κινδύνου** και να κάνει κλικ στο **Διαγραφή Account**.

![Ζώνη Κινδύνου της σελίδας Account με κουμπί Διαγραφή Account](/img/account-page/danger-zone.png)

Θα ανοίξει ένα module που θα ζητά την επιβεβαίωσή του. Πρέπει να ενεργοποιήσει την επιλογή **Επιβεβαίωση Διαγραφής Account** και να κάνει κλικ στο **Διαγραφή Account**.

![Παράθυρο διαλόγου Επιβεβαίωσης Διαγραφής Account](/img/account-page/delete-account-confirm.png)

Αυτό θα τον αποσυνδέσει από τον ιστότοπο και θα ακυρώσει τη συνδρομή μέλους του στο δίκτυό σας, καθώς και θα διαγράψει τον ιστότοπό του από το δίκτυό σας.

![Αποσύνδεση μετά τη διαγραφή λογαριασμού](/img/account-page/logged-out-after-delete.png)

![Επιβεβαίωση ακύρωσης συνδρομής μέλους](/img/account-page/membership-canceled.png)

**Σημαντικό:** Σημειώστε ότι αυτή η ενέργεια δεν θα εξαιρέσει τον πελάτη από το δίκτυό σας, καθώς ένας πελάτης μπορεί να έχει περισσότερες από μία συνδρομές μέλους ή καμία συνδρομή μέλους συνδεδεμένη με αυτόν.

![Ο πελάτης εξακολουθεί να υπάρχει στο δίκτυο μετά την ακύρωση](/img/admin/customer-after-cancel.png)

![Πελάτης χωρίς ενεργές συνδρομές μέλους](/img/admin/memberships-customer-empty.png)
