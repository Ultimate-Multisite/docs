# Αναβάθμιση Πακέτου (v2)

_**ΣΗΜΑΝΤΙΚΗ ΣΗΜΕΙΩΣΗ: Αυτό το άρθρο αφορά την έκδοση 2.x του Ultimate Multisite.**_

Οι πελάτες σας μπορούν να αναβαθμίσουν το πακέτο τους οποιαδήποτε στιγμή. Μπορούν είτε να αναβαθμίσουν σε άλλο πακέτο είτε να αγοράσουν πρόσθετες υπηρεσίες ή πακέτα που προσφέρετε στο δίκτυό σας.

Σε αυτόν τον οδηγό θα δούμε πώς μπορούν να αναβαθμίσουν το πακέτο τους και τι συμβαίνει μετά τη διαδικασία αναβάθμισης.

Για να αναβαθμίσουν το πακέτο τους, οι πελάτες σας πρέπει να συνδεθούν στον πίνακα ελέγχου τους και να μεταβούν στη σελίδα **Λογαριασμός**.

![Πίνακας ελέγχου πελάτη με σύνδεσμο προς τη σελίδα Λογαριασμός](/img/admin/memberships-list.png)

Στη σελίδα Λογαριασμός, θα δουν την τρέχουσα συνδρομή τους και το πακέτο που σχετίζεται με αυτήν. Για να αναβαθμίσουν σε άλλο πακέτο, πρέπει να κάνουν κλικ στο **Αλλαγή** στην επάνω δεξιά γωνία της ενότητας **Η Συνδρομή σας**.

![Ενότητα Η Συνδρομή σας με κουμπί Αλλαγή](/img/admin/memberships-list.png)

Θα ανακατευθυνθούν σε μια φόρμα ολοκλήρωσης αγοράς όπου θα εμφανίζονται όλα τα διαθέσιμα πακέτα.

Θα μπορούν επίσης να δουν τις **υπηρεσίες και τα πακέτα που είναι διαθέσιμα για το τρέχον πακέτο τους**, σε περίπτωση που θέλουν απλώς να αγοράσουν μια συγκεκριμένη υπηρεσία ή πακέτο (όπως απεριόριστες επισκέψεις ή χώρο αποθήκευσης στο παράδειγμά μας), χωρίς να αναβαθμίσουν το πακέτο.

![Φόρμα ολοκλήρωσης αγοράς με τα διαθέσιμα πακέτα και υπηρεσίες](/img/admin/memberships-list.png)

Αφού επιλέξουν το προϊόν που θέλουν να αγοράσουν, θα δουν πόσο πρέπει να πληρώσουν τώρα - χωρίς να συμπεριλαμβάνεται τυχόν υπάρχον πιστωτικό υπόλοιπο - και πόσο θα χρεωθούν στην επόμενη ημερομηνία χρέωσης.

Συνήθως, αν το προϊόν είναι άλλο πακέτο και η πληρωμή γίνεται ενδιάμεσα μιας χρέωσης συνδρομής, θα λάβουν πιστωτικό υπόλοιπο για το ποσό που πλήρωσαν στο πρώτο πακέτο.

![Σύνοψη πληρωμής αναβάθμισης με πιστωτικό υπόλοιπο και ποσό επόμενης χρέωσης](/img/admin/memberships-list.png)

Αν επιλέξουν ένα πακέτο ή υπηρεσία που δεν αλλάζει τίποτα από την τρέχουσα συνδρομή, θα δουν ένα μήνυμα που το εξηγεί.

![Μήνυμα όταν το επιλεγμένο πακέτο δεν αλλάζει τη συνδρομή](/img/admin/memberships-list.png)

Μετά την ολοκλήρωση της αγοράς, τα νέα προϊόντα θα προστεθούν στον λογαριασμό των πελατών σας και όλα τα όρια ή χαρακτηριστικά των νέων προϊόντων θα ενεργοποιηθούν άμεσα: επισκέψεις, χώρος αποθήκευσης, άρθρα κ.λπ.

## Διαδρομές Αναβάθμισης και Υποβάθμισης

Σε κάθε προϊόν σας, θα υπάρχει μια καρτέλα **Αναβαθμίσεις & Υποβαθμίσεις**. Η πρώτη επιλογή σε αυτή την καρτέλα είναι ένα πεδίο που ονομάζεται **Ομάδα Πακέτων**.

Οι **ομάδες πακέτων** επιτρέπουν στο Ultimate Multisite να γνωρίζει ότι ορισμένα πακέτα ανήκουν στην ίδια "οικογένεια" και επομένως πρέπει να χρησιμοποιούνται για τη δημιουργία επιλογών διαδρομής αναβάθμισης/υποβάθμισης.

![Καρτέλα Αναβαθμίσεις & Υποβαθμίσεις με πεδίο Ομάδα Πακέτων](/img/config/product-upgrades.png)

Για παράδειγμα, έχετε διαθέσιμο ένα **Δωρεάν πακέτο**, ένα **Βασικό Πακέτο** και ένα **Premium Πακέτο**. Θέλετε οι χρήστες που είναι εγγεγραμμένοι στο **Δωρεάν Πακέτο** να μπορούν να αναβαθμίσουν μόνο στο **Premium Πακέτο** και δεν θέλετε να βλέπουν το "Βασικό Πακέτο" ως επιλογή αναβάθμισης. Το μόνο που χρειάζεται να κάνετε είναι να αντιστοιχίσετε το ίδιο όνομα ομάδας πακέτων τόσο στο Δωρεάν όσο και στο Premium πακέτο, όπως φαίνεται στα παρακάτω στιγμιότυπα.

![Δωρεάν Πακέτο με ομάδα πακέτων High End](/img/config/product-upgrades.png)

![Premium Πακέτο με ομάδα πακέτων High End](/img/config/product-upgrades.png)

Αυτό ενημερώνει το Ultimate Multisite ότι υπάρχει μια "οικογένεια" πακέτων στο δίκτυο που ονομάζεται **High End**. Κατά την προσφορά αναβαθμίσεων ή υποβαθμίσεων, μόνο πακέτα από την ίδια οικογένεια θα εμφανίζονται ως επιλογή στον χρήστη.
