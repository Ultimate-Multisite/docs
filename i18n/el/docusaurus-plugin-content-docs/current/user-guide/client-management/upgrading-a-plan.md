# Αναβάθμιση πλάνου (v2)

_**ΣΗΜΑΝΤΙΚΗ ΣΗΜΕΙΩΣΗ: Αυτό το άρθρο αναφέρεται στην έκδοση 2.x του Ultimate Multisite.**_

Οι πελάτες σας μπορούν να αναβαθμίσουν τα πλάνα τους ανά πάσα στιγμή. Μπορούν είτε να αναβαθμίσουν σε άλλο πλάνο είτε να αγοράσουν οποιεσδήποτε επιπλέον υπηρεσίες ή πακέτα που προσφέρετε στο δίκτυό σας.

Σε αυτόν τον οδηγό θα καλύψουμε πώς μπορούν να αναβαθμίσουν το πλάνο τους και τι συμβαίνει μετά τη διαδικασία αναβάθμισης.

Για να αναβαθμίσουν το πλάνο τους, οι πελάτες σας θα πρέπει να αποκτήσουν πρόσβαση στο dashboard τους και να μεταβούν στη σελίδα **Account**.

![Dashboard υπο-ιστότοπου πελάτη με ορατό σύνδεσμο μενού Account](/img/account-page/account-menu.png)

Στη σελίδα Account, θα εμφανιστεί η τρέχουσα συνδρομή τους και το πλάνο που σχετίζεται με αυτή. Για να αναβαθμίσουν σε άλλο πλάνο, πρέπει να κάνουν κλικ στο **Αλλαγή** στην επάνω δεξιά γωνία της ενότητας **Η συνδρομή σας**.

![Σελίδα Account, κάρτα Η συνδρομή σας με κουμπί Αλλαγή](/img/account-page/membership-change-button.png)

Θα ανακατευθυνθούν σε μια φόρμα checkout όπου θα εμφανίζονται όλα τα διαθέσιμα πλάνα.

Θα μπορούν επίσης να δουν τις **υπηρεσίες και τα πακέτα που είναι διαθέσιμα για το τρέχον πλάνο τους**, σε περίπτωση που θέλουν απλώς να αγοράσουν μια συγκεκριμένη υπηρεσία ή ένα πακέτο (όπως απεριόριστες επισκέψεις ή χώρο δίσκου στο παράδειγμά μας εδώ), και όχι να αναβαθμίσουν το πλάνο.

![Επιλογέας αναβάθμισης που εμφανίζει διαθέσιμα πλάνα και πακέτα στην πλευρά του πελάτη](/img/account-page/upgrade-picker.png)

Αφού επιλέξουν το προϊόν που θέλουν να αγοράσουν, θα δουν πόσα θα χρειαστεί να πληρώσουν τώρα - εξαιρουμένης οποιασδήποτε υπάρχουσας πίστωσης - και πόσα θα χρεωθούν στην επόμενη ημερομηνία χρέωσης.

Συνήθως, αν το προϊόν είναι άλλο πλάνο και η πληρωμή πρόκειται να γίνει μεταξύ χρεώσεων συνδρομής, θα λάβουν πίστωση για το ποσό που πληρώθηκε στο πρώτο πλάνο.

![Σύνοψη πληρωμής αναβάθμισης που εμφανίζει εφαρμοσμένη πίστωση και ποσό επόμενης χρέωσης](/img/account-page/upgrade-summary.png)

Αν επιλέξουν ένα πλάνο ή πακέτο που δεν θα αλλάξει τίποτα από την τρέχουσα συνδρομή, θα δουν ένα μήνυμα που το εξηγεί αυτό.

![Ειδοποίηση όταν το επιλεγμένο πλάνο δεν αλλάζει τη συνδρομή](/img/account-page/upgrade-no-change.png)

Αφού ολοκληρωθεί το checkout, τα νέα προϊόντα θα προστεθούν στον λογαριασμό των πελατών σας και όλα τα όρια ή οι δυνατότητες των νέων προϊόντων θα προστεθούν άμεσα σε αυτόν: επισκέψεις, χώρος δίσκου, αναρτήσεις, κ.λπ...

## Διαδρομές αναβάθμισης και υποβάθμισης

Σε καθένα από τα προϊόντα σας, θα έχετε μια καρτέλα **Up & Downgrades**. Η πρώτη επιλογή σε αυτήν την καρτέλα είναι ένα πεδίο που ονομάζεται **Ομάδα πλάνων**.

Οι **ομάδες πλάνων** είναι αυτό που σας επιτρέπει να ενημερώσετε το Ultimate Multisite ότι ορισμένα πλάνα ανήκουν στην ίδια «οικογένεια», και επομένως θα πρέπει να χρησιμοποιούνται για τη δημιουργία επιλογών διαδρομών αναβάθμισης/υποβάθμισης.

![Καρτέλα Up and Downgrades επεξεργασίας προϊόντος με πεδίο Ομάδα πλάνων](/img/config/product-upgrades-plan-group.png)

Για παράδειγμα, έχετε ένα **δωρεάν πλάνο**, ένα **Basic Plan** και ένα **Premium Plan** διαθέσιμα. Θέλετε οι χρήστες που είναι εγγεγραμμένοι στο **Free Plan** να μπορούν να αναβαθμίσουν μόνο στο **Premium Plan** και δεν θέλετε να βλέπουν το «Basic Plan» ως επιλογή αναβάθμισης. Το μόνο που χρειάζεται να κάνετε είναι να εκχωρήσετε το ίδιο όνομα ομάδας πλάνου τόσο για τα πλάνα Free όσο και Premium, όπως φαίνεται στα παρακάτω στιγμιότυπα οθόνης.

![Σελίδα προϊόντος Free Plan με εκχωρημένη ομάδα πλάνων High End](/img/config/product-upgrades-free.png)

![Σελίδα προϊόντος Premium Plan με εκχωρημένη ομάδα πλάνων High End](/img/config/product-upgrades-premium.png)

Αυτό θα πρέπει να ενημερώσει το Ultimate Multisite ότι υπάρχει μια «οικογένεια» πλάνων στο δίκτυο που ονομάζεται **High End**. Όταν προσφέρονται αναβαθμίσεις ή υποβαθμίσεις, μόνο πλάνα από την ίδια οικογένεια θα παρουσιάζονται ως επιλογή στον χρήστη.
