# Πώς μπορώ να αποκτήσω πρόσβαση στη σελίδα του λογαριασμού μου για πρώτη φορά;

Καλώς ήρθατε στο Ultimate Multisite!

Αποκτήστε πρόσβαση στη σελίδα του λογαριασμού σας μέσω του συνδέσμου που λάβατε στο e-mail σας μετά την αγορά του plugin:

<!-- Screenshot unavailable: Account page email with access link -->

Χρησιμοποιήστε το email που δώσατε κατά την αγορά και τον κωδικό πρόσβασης που λάβατε στο email σας.

<!-- Screenshot unavailable: Login page with email and password fields -->

Επειδή αυτή είναι η πρώτη φορά που συνδέεστε σε αυτόν τον λογαριασμό, θα σας ζητηθεί να αλλάξετε τον κωδικό πρόσβασής σας. Αλλάξτε τον και κάντε κλικ στο κουμπί _Αποθήκευση_.

<!-- Screenshot unavailable: Change password form with Save button -->

Στη σελίδα του λογαριασμού σας, μπορείτε να διαχειριστείτε όλες τις λεπτομέρειες του λογαριασμού σας, όπως χρεώσεις, ανανεώσεις, ακυρώσεις και πολλά άλλα.
