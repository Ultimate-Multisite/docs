# Υποβάθμιση πλάνου (v2)

_**ΣΗΜΑΝΤΙΚΗ ΣΗΜΕΙΩΣΗ: Αυτό το άρθρο αναφέρεται στην έκδοση 2.x του Ultimate Multisite.**_

Η υποβάθμιση ενός πλάνου ή συνδρομής είναι μια συνηθισμένη ενέργεια που μπορεί να κάνουν οι πελάτες σας αν έχουν περιορισμένο budget ή αποφάσισαν ότι δεν θα χρειαστούν πολλούς πόρους για να λειτουργήσουν το subsite τους.

## Πώς να υποβαθμίσετε ένα πλάνο

Οι πελάτες σας μπορούν να υποβαθμίσουν το πλάνο τους οποιαδήποτε στιγμή, συνδεόμενοι στο admin dashboard του subsite τους και κάνοντας κλικ στο **Change** στη σελίδα του λογαριασμού τους.

![Σελίδα λογαριασμού με κουμπί Change κάτω από τη συνδρομή](/img/admin/memberships-list.png)

Κάνοντας κλικ στο κουμπί **Change**, ο χρήστης/πελάτης θα ανακατευθυνθεί στη σελίδα checkout όπου μπορεί να επιλέξει το πλάνο στο οποίο θέλει να αλλάξει τη συνδρομή του.

![Σελίδα checkout που εμφανίζει επιλογές πλάνων για υποβάθμιση](/img/admin/memberships-list.png)

Σε αυτό το παράδειγμα, υποβαθμίζουμε το πλάνο από **Premium** σε **Free**.

Για να προχωρήσει, ο χρήστης απλά πρέπει να κάνει κλικ στο κουμπί **Complete Checkout**. Στη συνέχεια θα επιστρέψει στη σελίδα λογαριασμού όπου θα εμφανίζεται ένα μήνυμα για την εκκρεμή αλλαγή της συνδρομής. Οι αλλαγές θα τεθούν σε ισχύ στον **επόμενο κύκλο χρέωσης** του πελάτη.

![Σελίδα λογαριασμού που εμφανίζει μήνυμα εκκρεμούς αλλαγής συνδρομής](/img/admin/memberships-list.png)

### Τι συμβαίνει όταν ένας χρήστης υποβαθμίζει το πλάνο του

Είναι σημαντικό να σημειωθεί ότι η υποβάθμιση του πλάνου δεν τροποποιεί την υπάρχουσα διαμόρφωση στο subsite του χρήστη.

Δεν αλλάζει αυτόματα το template του site, καθώς η αλλαγή του template θα διαγράψει και θα επαναφέρει πλήρως το subsite. Αυτό γίνεται για να αποφευχθεί η περιττή απώλεια δεδομένων. Επομένως, ο χώρος αποθήκευσης, τα themes, τα plugins κ.λπ. θα παραμείνουν ανέπαφα, εκτός από τα άρθρα.

Κατανοούμε ότι η κύρια ανησυχία σας θα είναι τα όρια και οι ποσοστώσεις που έχετε ορίσει σε κάθε πλάνο, αλλά πρέπει να λάβουμε υπόψη τη ζημιά που θα προκαλούσε στο subsite του χρήστη αν διαγράφαμε ή αλλάζαμε οποιαδήποτε από τις ρυθμίσεις του.

Για τα άρθρα που υπερβαίνουν το όριο που έχει οριστεί στο πλάνο, έχετε 3 διαφορετικές επιλογές: **Διατήρηση των άρθρων ως έχουν***,* **Μετακίνηση των άρθρων στον κάδο απορριμμάτων***,* ή **Μετακίνηση των άρθρων σε πρόχειρα***.* Μπορείτε να το ρυθμίσετε αυτό στις ρυθμίσεις του Ultimate Multisite.

![Επιλογές υπέρβασης ορίου άρθρων στις ρυθμίσεις του Ultimate Multisite](/img/config/settings-sites.png)

### Τι συμβαίνει με την πληρωμή

Στην έκδοση 2.0, δεν απαιτούνται πλέον προσαρμογές στην πληρωμή όσον αφορά την αναλογική χρέωση.

Αυτό συμβαίνει επειδή το σύστημα θα περιμένει να **ολοκληρωθεί ο κύκλος χρέωσης** της υπάρχουσας συνδρομής προτού τεθεί σε ισχύ το νέο πλάνο/συνδρομή. Το νέο ποσό χρέωσης για τη νέα συνδρομή θα εφαρμοστεί και θα χρεωθεί αυτόματα στον επόμενο κύκλο χρέωσης.
