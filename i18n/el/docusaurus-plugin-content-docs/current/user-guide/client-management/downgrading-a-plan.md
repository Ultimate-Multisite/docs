# Υποβάθμιση ενός πλάνου (v2)

_**ΣΗΜΑΝΤΙΚΗ ΣΗΜΕΙΩΣΗ: Αυτό το άρθρο αναφέρεται στην έκδοση Ultimate Multisite 2.x.**_

Η υποβάθμιση ενός πλάνου ή μιας συνδρομής είναι μια συνηθισμένη ενέργεια που μπορεί να κάνουν οι πελάτες σας αν έχουν περιορισμένο προϋπολογισμό ή αποφάσισαν ότι δεν θα χρειαστούν πολλούς πόρους για να λειτουργήσουν τον υπο-ιστότοπό τους.

## Πώς να υποβαθμίσετε ένα πλάνο

Οι πελάτες σας μπορούν να υποβαθμίσουν το πλάνο τους οποιαδήποτε στιγμή κάνοντας σύνδεση στο dashboard διαχείρισης του υπο-ιστότοπού τους και κάνοντας κλικ στο **Αλλαγή** κάτω από τη σελίδα account τους.

![Σελίδα account πελάτη με κάρτα Η συνδρομή σας και κουμπί Αλλαγή](/img/account-page/membership-change-button.png)

Μόλις κάνουν κλικ στο κουμπί **Αλλαγή**, ο χρήστης/πελάτης θα ανακατευθυνθεί στη σελίδα ολοκλήρωσης αγοράς όπου μπορεί να επιλέξει το πλάνο στο οποίο θέλει να αλλάξει τη συνδρομή του.

![Σελίδα επιλογών υποβάθμισης πλάνου από την πλευρά του πελάτη](/img/account-page/downgrade-picker.png)

Σε αυτό το παράδειγμα, υποβαθμίζουμε το πλάνο από **Premium** σε **Free**.

Για να συνεχίσει, ο χρήστης χρειάζεται απλώς να κάνει κλικ στο κουμπί **Complete Checkout**. Στη συνέχεια, θα επιστρέψει στη σελίδα account, όπου θα εμφανίζεται ένα μήνυμα σχετικά με την εκκρεμή αλλαγή για τη συνδρομή. Οι αλλαγές θα τεθούν σε ισχύ στον **επόμενο κύκλο χρέωσης** του πελάτη.

![Σελίδα account που εμφανίζει banner εκκρεμούς αλλαγής συνδρομής](/img/account-page/pending-change.png)

### Τι συμβαίνει όταν ένας χρήστης υποβαθμίζει το πλάνο του

Είναι σημαντικό να σημειωθεί ότι η υποβάθμιση του πλάνου δεν αλλάζει την υπάρχουσα διαμόρφωση στον υπο-ιστότοπο του χρήστη.

Δεν αλλάζει αυτόματα το πρότυπο ιστότοπου, καθώς η αλλαγή του προτύπου ιστότοπου θα διαγράψει πλήρως και θα επαναφέρει τον υπο-ιστότοπο. Αυτό γίνεται για να αποφευχθεί η περιττή απώλεια δεδομένων. Έτσι, ο χώρος δίσκου, τα θέματα, τα plugins κ.λπ. θα παραμείνουν ανέπαφα, εκτός από τις αναρτήσεις.

Κατανοούμε ότι η κύρια ανησυχία σας θα ήταν τα όρια και οι ποσοστώσεις που ορίζετε κάτω από κάθε πλάνο, αλλά πρέπει να λάβουμε υπόψη τη ζημιά που θα προκαλούσε στον υπο-ιστότοπο του χρήστη αν διαγράφαμε ή αλλάζαμε οποιαδήποτε από τις διαμορφώσεις του.

Για τις αναρτήσεις που υπερβαίνουν το όριο που έχει οριστεί στο πλάνο, έχετε 3 διαφορετικές επιλογές: **Διατήρηση των αναρτήσεων ως έχουν** *,* **Μετακίνηση των αναρτήσεων στον κάδο απορριμμάτων** *,* ή **Μετακίνηση των αναρτήσεων σε πρόχειρο** *.* Μπορείτε να το διαμορφώσετε αυτό στις ρυθμίσεις του Ultimate Multisite.

![Σελίδα Ιστότοποι στις Ρυθμίσεις Διαχείρισης Δικτύου που εμφανίζει επιλογές συμπεριφοράς υπέρβασης ορίου αναρτήσεων](/img/account-page/settings-sites-post-limit.png)

### Τι συμβαίνει με την πληρωμή

Στην έκδοση 2.0, δεν απαιτούνται πλέον προσαρμογές στην πληρωμή όσον αφορά την αναλογική χρέωση.

Αυτό συμβαίνει επειδή το σύστημα θα περιμένει την υπάρχουσα συνδρομή να **ολοκληρώσει τον κύκλο χρέωσής της πριν** τεθεί σε ισχύ το νέο πλάνο/συνδρομή. Το νέο ποσό χρέωσης για τη νέα συνδρομή θα εφαρμοστεί και θα χρεωθεί αυτόματα στον επόμενο κύκλο χρέωσης.
