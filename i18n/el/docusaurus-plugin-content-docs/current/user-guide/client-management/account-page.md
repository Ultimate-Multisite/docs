# Η Σελίδα Λογαριασμού του Πελάτη σας (v2)

_**ΣΗΜΑΝΤΙΚΗ ΣΗΜΕΙΩΣΗ: Αυτό το άρθρο αναφέρεται στην έκδοση 2.x του Ultimate Multisite.**_

Όταν οι πελάτες εγγράφονται σε ένα πλάνο στο δίκτυό σας, αποκτούν πρόσβαση σε μια ιστοσελίδα και στο dashboard της με σημαντικές πληροφορίες σχετικά με τις πληρωμές τους, τις συνδρομές τους, τα domains, τους περιορισμούς του πλάνου τους κ.λπ.

Σε αυτόν τον οδηγό, θα σας ξεναγήσουμε στη σελίδα λογαριασμού του πελάτη και θα δείτε τι μπορούν να δουν και να κάνουν οι πελάτες σας μέσα σε αυτήν.

## Η Σελίδα Λογαριασμού

Η σελίδα λογαριασμού είναι προσβάσιμη κάνοντας κλικ στο **Account** μέσα στο dashboard του πελάτη σας.

![Μενού Account στο dashboard του πελάτη](/img/admin/dashboard.png)

![Επισκόπηση της σελίδας λογαριασμού πελάτη](/img/admin/dashboard.png)

Αφού ο πελάτης κάνει κλικ σε αυτό, θα δει μια επισκόπηση της συνδρομής του, της διεύθυνσης χρέωσης, των τιμολογίων, των domains, των περιορισμών της ιστοσελίδας, και θα μπορεί επίσης να αλλάξει το **Site Template** (αν επιτρέπεται στο δίκτυό σας).

Μπορούν επίσης να αλλάξουν τη συνδρομή σε άλλο πλάνο ή να αγοράσουν κάποιο άλλο πακέτο ή υπηρεσία που προσφέρετε. Ας δούμε κάθε ενότητα ξεχωριστά.

### Επισκόπηση της Συνδρομής σας:

Το πρώτο μπλοκ ακριβώς κάτω από το όνομα της ιστοσελίδας των πελατών σας δείχνει μια επισκόπηση του τρέχοντος πλάνου τους και των υπηρεσιών/πακέτων που αγοράστηκαν μαζί με αυτό. Το μπλοκ δείχνει επίσης τον αριθμό συνδρομής, το αρχικό ποσό που πληρώθηκε, πόσο κοστίζει το πλάνο και οποιαδήποτε υπηρεσία/πακέτο, καθώς και πόσες φορές έχουν χρεωθεί για αυτή τη συνδρομή. Μπορούν επίσης να δουν αν η συνδρομή είναι **Active**, **Expired** ή **Canceled**.

![Επισκόπηση συνδρομής που δείχνει το πλάνο, το ποσό και τις λεπτομέρειες χρέωσης](/img/admin/memberships-list.png)

Ακριβώς κάτω από αυτό το μπλοκ, οι πελάτες σας μπορούν να δουν τα μπλοκ **About This Site** και **Site Limits**. Αυτά τα μπλοκ τους δείχνουν όλους τους περιορισμούς που συνοδεύουν το πλάνο τους: χώρο αποθήκευσης, άρθρα, σελίδες, επισκέψεις κ.λπ. Αυτοί οι περιορισμοί μπορούν να ρυθμιστούν σε κάθε σελίδα πλάνου στο **Ultimate Multisite > Products**.

![Τα μπλοκ About This Site και Site Limits που δείχνουν τους περιορισμούς του πλάνου](/img/admin/dashboard.png)

Στη δεξιά πλευρά του **Your Membership**, οι πελάτες μπορούν να κάνουν κλικ στο **Change**. Αυτό θα τους δείξει όλα τα διαθέσιμα πλάνα και πακέτα/υπηρεσίες. Αν επιλέξουν άλλο πλάνο, οι περιορισμοί του νέου πλάνου θα αντικαταστήσουν τους τρέχοντες περιορισμούς της συνδρομής - ανεξάρτητα από το αν κάνουν υποβάθμιση ή αναβάθμιση.

Τώρα, αν οι πελάτες σας επιλέξουν να αγοράσουν πακέτα ή υπηρεσίες για αυτή την τρέχουσα συνδρομή - όπως περισσότερο χώρο αποθήκευσης ή επισκέψεις - η τρέχουσα συνδρομή δεν θα αλλάξει, αλλά μόνο τα νέα πακέτα θα προστεθούν σε αυτήν.

Σημειώστε ότι οι κωδικοί κουπονιών δεν μπορούν να προστεθούν σε αυτή τη σελίδα αλλαγής συνδρομής. Αν ο πελάτης χρησιμοποίησε έναν κωδικό κουπονιού στην πρώτη αγορά συνδρομής, ο κωδικός θα εφαρμοστεί και στη νέα συνδρομή.

### Ενημέρωση της Διεύθυνσης Χρέωσης:

Στη σελίδα λογαριασμού, οι πελάτες σας μπορούν επίσης να ενημερώσουν τη διεύθυνση χρέωσής τους. Απλά χρειάζεται να κάνουν κλικ στο **Update** δίπλα στο _Billing Address_.

![Ενότητα Billing Address με κουμπί Update](/img/admin/customers-list.png)

Θα εμφανιστεί ένα νέο παράθυρο στον πελάτη σας. Το μόνο που χρειάζεται να κάνει είναι να συμπληρώσει τη νέα διεύθυνση και να κάνει κλικ στο _Save Changes_.

![Φόρμα ενημέρωσης διεύθυνσης χρέωσης](/img/admin/customers-list.png)

### Αλλαγή του Site Template:

Για να επιτρέψετε στους πελάτες σας να αλλάζουν τα site templates τους, πρέπει να πάτε στο **Ultimate Multisite > Settings > Sites** και να ενεργοποιήσετε την επιλογή **Allow Template Switching**.

Επίσης, στο **Ultimate Multisite > Products**, επιλέξτε τα πλάνα σας και πηγαίνετε στην καρτέλα **Site Templates**. Βεβαιωθείτε ότι η επιλογή **Allow Site Templates** είναι ενεργοποιημένη και στο **Site Template Selection Mode** είναι επιλεγμένο το **Choose Available Site Templates**.

![Καρτέλα Site Templates του προϊόντος με τη λειτουργία επιλογής template](/img/config/product-site-templates.png)

Θα μπορείτε να δείτε όλα τα διαθέσιμα site templates στην ιστοσελίδα σας. Επιλέξτε ποια θέλετε να κάνετε διαθέσιμα και ποια θέλετε να μην είναι διαθέσιμα στους πελάτες σας που είναι εγγεγραμμένοι σε αυτό το πλάνο. Σημειώστε ότι αυτές οι επιλογές επηρεάζουν επίσης τη φόρμα ολοκλήρωσης αγοράς, οπότε οποιοδήποτε template επιλεγεί ως **Not Available** δεν θα εμφανίζεται στη σελίδα εγγραφής για αυτό το πλάνο.

Τώρα οι πελάτες σας μπορούν να κάνουν κλικ στο **Change Site Template** μέσα στη σελίδα λογαριασμού τους.

![Κουμπί Change Site Template στη σελίδα λογαριασμού](/img/admin/dashboard.png)  
Θα εμφανιστεί στον πελάτη σας μια λίστα με όλα τα διαθέσιμα Site Templates για αυτό το πλάνο.

![Λίστα διαθέσιμων site templates για το πλάνο](/img/config/site-templates-list.png)

Αφού επιλέξουν αυτό που θέλουν, θα τους ζητηθεί να επιβεβαιώσουν την αλλαγή.

![Παράθυρο επιβεβαίωσης αλλαγής site template](/img/admin/dashboard.png)

Αφού ενεργοποιήσουν την επιβεβαίωση και κάνουν κλικ στο **Process Switch**, το νέο site template θα χρησιμοποιηθεί στην ιστοσελίδα του πελάτη σας.

### Προσθήκη Custom Domains:

Οι πελάτες σας θα έχουν επίσης τη δυνατότητα να προσθέσουν ένα custom domain για αυτό το πλάνο στη σελίδα λογαριασμού τους. Για να επιτρέψετε στους πελάτες σας να χρησιμοποιούν custom domains, πηγαίνετε στο **Ultimate Multisite > Settings >** **Domain Mapping**.

Ενεργοποιήστε την επιλογή **Enable Domain Mapping**. Αυτό θα επιτρέψει στους πελάτες σας να χρησιμοποιούν custom domains σε επίπεδο δικτύου.

Μην ξεχάσετε επίσης να ελέγξετε αν το domain mapping είναι ενεργοποιημένο σε επίπεδο προϊόντος - επειδή μπορείτε να περιορίσετε ένα προϊόν ώστε να μην επιτρέπει στους πελάτες σας να χρησιμοποιούν custom domains.

Πηγαίνετε στο **Ultimate Multisite > Products**. Επιλέξτε το πλάνο της επιλογής σας και πηγαίνετε στην καρτέλα **Custom Domains**. Ενεργοποιήστε την επιλογή **Allow Custom Domains**.

![Καρτέλα Custom Domains με το διακόπτη Allow Custom Domains](/img/config/product-custom-domains.png)

Αυτό θα επιτρέψει σε όλους τους πελάτες που είναι εγγεγραμμένοι σε αυτό το συγκεκριμένο πλάνο να χρησιμοποιούν custom domains. Τώρα, στη σελίδα Λογαριασμού, οι πελάτες σας μπορούν να προσθέσουν ένα custom domain κάνοντας κλικ στο **Add Domain**.

![Κουμπί Add Domain στη σελίδα λογαριασμού](/img/admin/domains-list.png)

Το πρώτο παράθυρο που ανοίγει θα δείξει στους πελάτες σας ένα μήνυμα με οδηγίες για το πώς να ενημερώσουν τις εγγραφές DNS τους ώστε αυτό το custom domain να λειτουργήσει στο δίκτυό σας.

![Οδηγίες DNS που εμφανίζονται κατά την προσθήκη custom domain](/img/admin/domains-list.png)

Αυτό το μήνυμα μπορεί να επεξεργαστεί (από εσάς) στο **Ultimate Multisite > Settings > Domain Mapping > Add New Domain Instructions**.

![Ρύθμιση Add New Domain Instructions στο Domain Mapping](/img/config/settings-domain-mapping.png)

Αφού κάνουν κλικ στο **Next Step**, οι πελάτες σας μπορούν να προσθέσουν το όνομα του custom domain τους και να επιλέξουν αν αυτό το custom domain θα είναι το κύριο. Σημειώστε ότι οι πελάτες σας μπορούν να χρησιμοποιούν περισσότερα από ένα custom domains για τις ιστοσελίδες τους, οπότε μπορούν να επιλέξουν ποιο θα είναι το κύριο.

![Εισαγωγή ονόματος custom domain με επιλογή κύριου domain](/img/admin/domains-list.png)

Αφού κάνουν κλικ στο **Add Domain**, το domain θα προστεθεί στον λογαριασμό του πελάτη σας. Το μόνο που χρειάζεται να κάνουν τώρα είναι να αλλάξουν τις εγγραφές DNS αυτού του custom domain στον καταχωρητή domain τους.

### Αλλαγή Κωδικού Πρόσβασης:

Μέσα στο dashboard του λογαριασμού, οι πελάτες σας μπορούν επίσης να αλλάξουν τον κωδικό πρόσβασής τους κάνοντας κλικ στο **Change Password**.

![Κουμπί Change Password στη σελίδα λογαριασμού](/img/admin/dashboard.png)

Αυτό θα εμφανίσει ένα νέο παράθυρο όπου οι πελάτες σας θα πρέπει να συμπληρώσουν τον τρέχοντα κωδικό πρόσβασής τους και στη συνέχεια να συμπληρώσουν τον νέο κωδικό πρόσβασης που θέλουν να χρησιμοποιήσουν.

![Φόρμα αλλαγής κωδικού πρόσβασης με πεδία τρέχοντος και νέου κωδικού](/img/admin/dashboard.png)

### Επικίνδυνη Ζώνη:

Έχουμε επίσης δύο επιλογές που εμφανίζονται στο τμήμα **Danger Zone**: **Delete Site** και **Delete Account**. Βρίσκονται και οι δύο στην Επικίνδυνη Ζώνη επειδή αυτές οι δύο ενέργειες είναι μη αναστρέψιμες. Αν οι πελάτες σας διαγράψουν την ιστοσελίδα ή τον λογαριασμό τους, δεν μπορούν να τα ανακτήσουν.

![Danger Zone με τις επιλογές Delete Site και Delete Account](/img/admin/dashboard.png)

Αν οι πελάτες σας κάνουν κλικ σε οποιαδήποτε από αυτές τις δύο επιλογές, θα τους εμφανιστεί ένα παράθυρο όπου θα πρέπει να ενεργοποιήσουν την επιλογή για διαγραφή της ιστοσελίδας ή του λογαριασμού και θα προειδοποιηθούν ότι αυτή η ενέργεια δεν μπορεί να αναιρεθεί.

![Παράθυρο επιβεβαίωσης διαγραφής ιστοσελίδας](/img/admin/dashboard.png)

![Παράθυρο επιβεβαίωσης διαγραφής λογαριασμού](/img/admin/dashboard.png)

Αν διαγράψουν την ιστοσελίδα τους, ο λογαριασμός και η συνδρομή τους θα παραμείνουν ανέπαφα. Απλά θα χάσουν όλο το περιεχόμενο της ιστοσελίδας τους. Αν διαγράψουν τον λογαριασμό τους, όλες οι ιστοσελίδες, οι συνδρομές και οι πληροφορίες που σχετίζονται με αυτόν τον λογαριασμό θα χαθούν.
