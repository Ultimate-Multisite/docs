# Συμβάν: Εγγραφή λογαριασμού μέσω Zapier

Στο άρθρο [Ενσωμάτωση του Ultimate Multisite με το Zapier](zapier.md), εξηγήσαμε πώς να χρησιμοποιείτε το Zapier για να εκτελείτε διάφορες ενέργειες στο Ultimate Multisite με βάση triggers και συμβάντα. Σε αυτό το άρθρο, θα δείξουμε πώς μπορείτε να ενσωματώσετε εφαρμογές τρίτων. Θα χρησιμοποιήσουμε το Google Sheets ως πηγή δεδομένων και θα στείλουμε τις πληροφορίες στο Ultimate Multisite για να εγγράψουμε έναν λογαριασμό.

Αρχικά, πρέπει να δημιουργήσετε ένα **Google Sheet** στο Google Drive σας. Βεβαιωθείτε ότι έχετε ορίσει σωστά κάθε στήλη, ώστε να μπορείτε εύκολα να αντιστοιχίσετε τα δεδομένα αργότερα.

![Google Sheet με στήλες για δεδομένα πελατών](/img/admin/webhooks-list.png)Αφού δημιουργήσετε το Google Sheet, μπορείτε να συνδεθείτε στον λογαριασμό σας στο Zapier και να ξεκινήσετε τη δημιουργία ενός zap.

![Πίνακας ελέγχου Zapier για τη δημιουργία ενός zap](/img/admin/webhooks-list.png)Στο πεδίο αναζήτησης για **"App event"** επιλέξτε **"Google Sheets"**

![Επιλογή Google Sheets ως app event](/img/admin/webhooks-list.png)

Στη συνέχεια, στο πεδίο "**Event**" επιλέξτε "**New spreadsheet row**" και πατήστε "**Continue**"

![Επιλογή συμβάντος New spreadsheet row στο Zapier](/img/admin/webhooks-list.png)Το επόμενο βήμα θα σας ζητήσει να επιλέξετε τον **Λογαριασμό Google** όπου είναι αποθηκευμένο το **Google Sheet**. Απλώς βεβαιωθείτε ότι έχετε επιλέξει τον σωστό λογαριασμό Google.

![Επιλογή Λογαριασμού Google για το Google Sheet](/img/admin/webhooks-list.png)

Στην ενότητα "**Set up trigger**", θα χρειαστεί να επιλέξετε και να καθορίσετε το υπολογιστικό φύλλο και το φύλλο εργασίας από όπου θα προέρχονται τα δεδομένα. Συμπληρώστε αυτά τα πεδία και πατήστε "**Continue**"

![Ρύθμιση trigger με επιλογή υπολογιστικού φύλλου και φύλλου εργασίας](/img/admin/webhooks-list.png)Το επόμενο βήμα είναι να "**test your trigger**" για να βεβαιωθείτε ότι το Google Sheet σας είναι σωστά συνδεδεμένο.

![Βήμα δοκιμής trigger στο Zapier](/img/admin/webhooks-list.png)Αν η δοκιμή είναι επιτυχής, θα δείτε το αποτέλεσμα να εμφανίζει κάποιες τιμές από τα υπολογιστικά φύλλα σας. Κάντε κλικ στο "**Continue**" για να συνεχίσετε.

![Επιτυχής δοκιμή trigger που εμφανίζει τιμές υπολογιστικού φύλλου](/img/admin/webhooks-list.png)Το επόμενο βήμα είναι να ρυθμίσετε τη δεύτερη ενέργεια που θα δημιουργήσει ή θα εγγράψει έναν λογαριασμό στο Ultimate Multisite. Στο πεδίο αναζήτησης επιλέξτε "**Ultimate Multisite(2.0.2)**"

![Επιλογή Ultimate Multisite ως εφαρμογή ενέργειας](/img/admin/webhooks-list.png)

Στο πεδίο "**Event**", επιλέξτε "**Register an Account in Ultimate Multisite**" και μετά κάντε κλικ στο κουμπί "**Continue**".

![Συμβάν ενέργειας Register an Account in Ultimate Multisite](/img/admin/webhooks-list.png)Στην ενότητα "**Set up an action**", θα δείτε διάφορα πεδία διαθέσιμα για δεδομένα πελατών, συνδρομές, προϊόντα κ.λπ. Μπορείτε να αντιστοιχίσετε τις τιμές από το Google Sheet σας και να τις αναθέσετε στο κατάλληλο πεδίο όπου πρέπει να συμπληρωθούν, όπως φαίνεται στο παρακάτω στιγμιότυπο οθόνης.

![Αντιστοίχιση τιμών Google Sheet σε πεδία Ultimate Multisite](/img/admin/webhooks-list.png)

Αφού αντιστοιχίσετε τις τιμές, μπορείτε να δοκιμάσετε την ενέργεια.

![Δοκιμή της ενέργειας εγγραφής λογαριασμού στο Zapier](/img/admin/webhooks-list.png)
