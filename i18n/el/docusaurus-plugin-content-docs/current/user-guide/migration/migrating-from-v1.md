# Μετάβαση από την Έκδοση 1

## Το Ultimate Multisite έχει αλλάξει από την αρχική σειρά εκδόσεων 1.x στη σειρά εκδόσεων 2.x.

Η έκδοση 2.0 και οι νεότερες του Ultimate Multisite αποτελούν πλήρη επανεγγραφή του κώδικα, πράγμα που σημαίνει ότι υπάρχουν ελάχιστα κοινά στοιχεία μεταξύ της παλιάς και της νέας έκδοσης. Για αυτόν τον λόγο, κατά την αναβάθμιση από την 1.x στην 2.x, τα δεδομένα σας θα πρέπει να μεταφερθούν σε μια μορφή που μπορούν να κατανοήσουν οι νέες εκδόσεις.

Ευτυχώς, το Ultimate Multisite 2.0+ **διαθέτει ένα ενσωματωμένο εργαλείο μετάβασης** στον πυρήνα του, το οποίο μπορεί να εντοπίσει δεδομένα από την παλιά έκδοση και να τα μετατρέψει στη νέα μορφή. Αυτή η μετάβαση πραγματοποιείται κατά τη διάρκεια του **Οδηγού Εγκατάστασης** της έκδοσης 2.0+.

Αυτό το μάθημα καλύπτει τον τρόπο λειτουργίας του εργαλείου μετάβασης, τι να κάνετε σε περίπτωση αποτυχίας και πώς να αντιμετωπίσετε προβλήματα που ενδέχεται να προκύψουν κατά τη διαδικασία.

_**ΣΗΜΑΝΤΙΚΟ: Πριν ξεκινήσετε την αναβάθμιση από την έκδοση 1.x στην έκδοση 2.0, βεβαιωθείτε ότι έχετε δημιουργήσει αντίγραφο ασφαλείας της βάσης δεδομένων του ιστότοπού σας**_

## Πρώτα βήματα

Το πρώτο βήμα είναι να κατεβάσετε το αρχείο .zip του plugin και να εγκαταστήσετε την έκδοση 2.0 στον πίνακα διαχείρισης του δικτύου σας.

Αφού [εγκαταστήσετε και ενεργοποιήσετε την έκδοση 2.0](../getting-started/installing-ultimate-multisite.md), το σύστημα θα εντοπίσει αυτόματα ότι το Multisite σας τρέχει στην παλαιότερη έκδοση και θα δείτε αυτό το μήνυμα στην κορυφή της σελίδας του plugin.

_**ΣΗΜΕΙΩΣΗ:** Αν έχετε εγκατεστημένο το Ultimate Multisite 1.x στο Multisite σας, θα έχετε την επιλογή να αντικαταστήσετε το plugin με την έκδοση που μόλις κατεβάσατε. Παρακαλούμε, προχωρήστε και κάντε κλικ στο **Replace current with uploaded**._

<!-- Screenshot unavailable: Plugin page showing option to replace v1 with uploaded v2 version -->

Η επόμενη σελίδα θα σας ενημερώσει ποια παλαιότερα add-ons έχετε εγκατεστημένα μαζί με την έκδοση 1.x. Θα περιέχει οδηγίες σχετικά με το αν η έκδοση που χρησιμοποιείτε είναι συμβατή με την έκδοση 2.0 ή αν χρειάζεται να εγκαταστήσετε μια αναβαθμισμένη έκδοση του add-on μετά τη μετάβαση.

<!-- Screenshot unavailable: Message on the top of the plugins page about updating to v2.0, with list of add-ons that need updating -->

Όταν είστε έτοιμοι να προχωρήσετε, μπορείτε να κάνετε κλικ στο κουμπί που λέει **Visit the Installer to finish the upgrade**.

<!-- Screenshot unavailable: Button saying Visit the Installer to finish the upgrade -->

Στη συνέχεια θα μεταφερθείτε στη σελίδα του οδηγού εγκατάστασης με μερικά μηνύματα καλωσορίσματος. Απλά χρειάζεται να κάνετε κλικ στο **Get Started** για να μεταβείτε στην επόμενη σελίδα.

<!-- Screenshot unavailable: Setup Wizard welcoming page with Get Started button -->

Αφού κάνετε κλικ στο **Get Started**, θα ανακατευθυνθείτε στους Προ-εγκατάστασης Ελέγχους_._ Εδώ θα δείτε τις Πληροφορίες Συστήματος και της εγκατάστασης WordPress και θα ενημερωθείτε αν πληρούν τις **απαιτήσεις του Ultimate Multisite**.

<!-- Screenshot unavailable: Pre-install Checks page showing system requirements confirmation -->

Το επόμενο βήμα είναι να εισάγετε το κλειδί άδειας του Ultimate Multisite και να ενεργοποιήσετε το plugin. Αυτό θα διασφαλίσει ότι όλες οι λειτουργίες, συμπεριλαμβανομένων των add-ons, θα είναι διαθέσιμες στον ιστότοπό σας.

<!-- Screenshot unavailable: License activation page with license key input and Agree and Activate button -->

Αφού εισάγετε το κλειδί σας, κάντε κλικ στο **Agree & Activate**.

Μετά την ενεργοποίηση της άδειας, μπορείτε να ξεκινήσετε την πραγματική εγκατάσταση κάνοντας κλικ στο **Install** στην επόμενη σελίδα. Αυτό θα δημιουργήσει αυτόματα τα απαραίτητα αρχεία και τη βάση δεδομένων που χρειάζεται η έκδοση 2.0 για να λειτουργήσει.

<!-- Screenshot unavailable: Installation page showing what will be updated with Install button -->

## Τώρα, η μετάβαση

Το εργαλείο μετάβασης διαθέτει ενσωματωμένο μηχανισμό ασφαλείας που ελέγχει ολόκληρο το multisite σας για να βεβαιωθεί ότι όλα τα δεδομένα του Ultimate Multisite μπορούν να μεταφερθούν χωρίς προβλήματα. Κάντε κλικ στο κουμπί **Run Check** για να ξεκινήσει η διαδικασία.

<!-- Screenshot unavailable: Migration page with Run Check button to verify data can be converted -->

Μετά την εκτέλεση του ελέγχου, υπάρχουν δύο πιθανά αποτελέσματα: το αποτέλεσμα μπορεί να είναι είτε **με σφάλμα** είτε **χωρίς σφάλμα**.

### Με Σφάλμα

Αν λάβετε μήνυμα σφάλματος, θα χρειαστεί να επικοινωνήσετε με την ομάδα υποστήριξής μας για να σας βοηθήσουν να διορθώσετε το σφάλμα. Βεβαιωθείτε ότι **παρέχετε το αρχείο καταγραφής σφαλμάτων** όταν δημιουργείτε ένα αίτημα υποστήριξης. Μπορείτε να κατεβάσετε το αρχείο καταγραφής ή να κάνετε κλικ στον σύνδεσμο που λέει contact our support team. Θα ανοίξει το widget βοήθειας στη δεξιά πλευρά της σελίδας σας με τα πεδία προ-συμπληρωμένα, συμπεριλαμβανομένων των αρχείων καταγραφής σφαλμάτων στην περιγραφή.

_**Επειδή το σύστημα εντόπισε σφάλμα, δεν θα μπορέσετε να προχωρήσετε στη μετάβαση στην έκδοση 2.0. Μπορείτε τότε να επιστρέψετε στην έκδοση 1.x για να συνεχίσετε να λειτουργεί το δίκτυό σας μέχρι να διορθωθεί το σφάλμα.**_

### Χωρίς Σφάλμα

Αν το σύστημα δεν εντοπίσει κανένα σφάλμα, θα δείτε ένα μήνυμα επιτυχίας και ένα κουμπί **Migrate** στο κάτω μέρος που θα σας επιτρέψει να προχωρήσετε με τη μετάβαση. Σε αυτήν τη σελίδα, θα σας υπενθυμιστεί να δημιουργήσετε αντίγραφο ασφαλείας της βάσης δεδομένων σας πριν προχωρήσετε, κάτι που συνιστούμε ανεπιφύλακτα. Πατήστε **Migrate** αν έχετε ήδη δημιουργήσει αντίγραφο ασφαλείας.
