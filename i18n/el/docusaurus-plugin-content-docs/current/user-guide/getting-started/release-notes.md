# Σημειώσεις έκδοσης

## Έκδοση 2.13.0 — Κυκλοφόρησε στις 2026-06-05 {#version-2130--released-on-2026-06-05}

- Νέο: Προστέθηκε υποστήριξη sovereign-tenant για ροές Account πελάτη, checkout, χρέωσης, ιστοτόπου, τιμολογίου, εναλλαγής προτύπων και αντιστοίχισης domain, ώστε τα δίκτυα tenant να μπορούν να κατευθύνουν τους πελάτες πίσω στον κύριο ιστότοπο για διαχειριζόμενες ενέργειες.
- Νέο: Προστέθηκαν έλεγχοι διαπιστευτηρίων ανανέωσης για επαναλαμβανόμενες συνδρομές μέλους, ώστε οι πύλες να μπορούν να απενεργοποιούν την αυτόματη ανανέωση όταν λείπει μια αποθηκευμένη συμφωνία χρέωσης, συνδρομή ή vault token.
- Νέο: Προστέθηκε δημοσίευση loopback επαληθευμένη με HMAC για εκκρεμή δημιουργία ιστοτόπου, ώστε η παροχή από checkout σε ιστότοπο να είναι πιο αξιόπιστη σε hosts όπου οι εργασίες παρασκηνίου καθυστερούν.
- Νέο: Προστέθηκαν σημεία επέκτασης για προγραμματιστές για SSO URL, βασικά domain φορμών checkout και αυτόματη δημιουργία εγγραφών domain.
- Διόρθωση: Το SSO είναι πιο αξιόπιστο σε αντιστοιχισμένα domain, ανώνυμες επισκέψεις broker, αποσύνδεση και συγκρούσεις εξαρτήσεων μεταξύ plugin.
- Διόρθωση: Η εκκρεμής δημιουργία ιστοτόπου πλέον ανακάμπτει από παρωχημένες σημαίες δημοσίευσης και αποφεύγει να αφήνει πελάτες κολλημένους στην οθόνη δημιουργίας ιστοτόπου.
- Διόρθωση: Οι εγγραφές domain δεν δημιουργούνται πλέον για κοινόχρηστα βασικά domain φορμών checkout, και οι αχρησιμοποίητες εργασίες παρασκηνίου παρόχου host παραλείπονται όταν δεν είναι ενεργή καμία ενσωμάτωση.
- Διόρθωση: Ακραίες περιπτώσεις στο checkout, στη διεύθυνση χρέωσης, στην επαναφορά κωδικού πρόσβασης, στην επαλήθευση email, στην εναλλαγή προτύπων, στις περιηγήσεις και στο Dashboard πελάτη δεν μπλοκάρουν πλέον τις κανονικές ροές πελατών.
- Διόρθωση: Τα email broadcast πλέον διατηρούν τους παραλήπτες ιδιωτικούς, αποφεύγοντας παράλληλα κρίσιμα σφάλματα SMTP/plugin όταν αποτυγχάνουν οι λίστες παραληπτών ή οι μεταφορές αλληλογραφίας.
- Διόρθωση: Ακραίες περιπτώσεις ανανεώσεων συνδρομών μέλους, εμφάνισης λήξης και είσπραξης πληρωμών πλέον αποφεύγουν άμεσες λήξεις, καταρρεύσεις ή χαμένες απαιτούμενες πληρωμές.
- Βελτιωμένο: Η συμβατότητα με WordPress δοκιμάζεται έως την έκδοση 7.0, τα παραγωγικά Vue assets αναδημιουργούνται από πηγές npm, και η κάλυψη end-to-end του Cypress πλέον δοκιμάζει περισσότερες ροές checkout, setup, SSO και πυλών.

## Έκδοση 2.12.0 — Κυκλοφόρησε στις 2026-05-15 {#version-2120--released-on-2026-05-15}

- Νέο: Προστέθηκε το Hostinger (hPanel) ως υποστηριζόμενος πάροχος host με ενσωμάτωση αντιστοίχισης domain
- Νέο: Το Site Exporter πλέον χειρίζεται πακέτα εισαγωγής δικτύου για απλοποιημένη επαναφορά ιστοτόπων σε όλο το δίκτυο
- Διόρθωση: Τα BCC email broadcast πλέον χρησιμοποιούν κεφαλίδα undisclosed-recipients για να αποτρέπεται η έκθεση διευθύνσεων παραληπτών
- Διόρθωση: Η ημερομηνία λήξης συνδρομής μέλους δεν αλλοιώνεται πλέον όταν αποθηκεύεται με μη ημερομηνιακή τιμή
- Διόρθωση: Οι ενημερώσεις συνδρομής μέλους Stripe πλέον εκκαθαρίζουν σωστά τις εκπτώσεις χωρίς να καλούν το παρωχημένο deleteDiscount API
- Διόρθωση: Οι ανακατευθύνσεις SSO σε ιστοτόπους με αντιστοιχισμένο domain πλέον περιορίζονται για να αποτρέπονται άπειροι βρόχοι ανακατεύθυνσης
- Διόρθωση: Η επιλογή επιλογέα εικόνας του οδηγού setup πλέον ενημερώνει σωστά το υποκείμενο μοντέλο δεδομένων
- Διόρθωση: Το Site Exporter CLI πλέον διατηρεί τη σωστή προεπιλεγμένη επιλογή ιστοτόπου δικτύου
- Βελτιωμένο: Αφαιρέθηκε το πακέτο wp-cli που ήταν ενσωματωμένο στο πακέτο plugin, μειώνοντας το μέγεθος του plugin

## Έκδοση 2.11.0 — Κυκλοφόρησε στις 2026-05-11 {#version-2110--released-on-2026-05-11}

- Νέο: Οι εξαγωγές ιστοτόπων πλέον περιλαμβάνουν ένα αυτοεκκινούμενο `index.php`, ώστε το ZIP να μπορεί να εγκατασταθεί σε νέο host χωρίς ξεχωριστή εγκατάσταση plugin.
- Νέο: Η εξαγωγή δικτύου επιτρέπει στους διαχειριστές να εξάγουν όλους τους υποϊστοτόπους σε ένα ενιαίο αρχείο από τη σελίδα διαχείρισης Site Export.
- Νέο: Ο διακόπτης πλάνου Allow Site Templates πλέον επιβάλλεται μέσω αλυσίδας εφεδρείας, περιορίζοντας σωστά τη διαθεσιμότητα προτύπων για όρια πλάνου.
- Νέο: Ο επεξεργαστής φόρμας checkout προειδοποιεί όταν προστίθεται ένα προϊόν χωρίς διαμορφωμένο απαιτούμενο πεδίο.
- Νέο: Η καρτέλα ρυθμίσεων Import/Export πλέον περιγράφει καθαρά το εύρος της και συνδέεται απευθείας με το εργαλείο Site Export.

## Έκδοση 2.10.0 — Κυκλοφόρησε στις 2026-05-05 {#version-2100--released-on-2026-05-05}

- Νέο: Καθοδηγούμενος οδηγός setup PayPal για χειροκίνητη εισαγωγή διαπιστευτηρίων με πύλη σημαίας OAuth για απρόσκοπτη διαμόρφωση πύλης.
- Νέο: Ο πίνακας πελάτη εναλλαγής προτύπων επανασχεδιάστηκε με κάρτα τρέχοντος προτύπου, μόνιμο πλέγμα και κουμπί **Επαναφορά τρέχοντος προτύπου**.
- Διόρθωση: Η εναλλαγή προτύπων δεν κολλάει πλέον το UI σε αποτυχία AJAX.
- Διόρθωση: Οι καταστάσεις δικαιωμάτων εναλλαγής προτύπων ασφαλίστηκαν έναντι μη εξουσιοδοτημένης πρόσβασης.
- Διόρθωση: Οι είσοδοι παράκαμψης ιστοτόπου επικυρώνονται πριν από την αποθήκευση.
- Διόρθωση: Η προτροπή διεύθυνσης χρέωσης πλέον εμφανίζεται όταν η διεύθυνση είναι κενή.
- Διόρθωση: Επιλύθηκαν οι ειδοποιήσεις παρωχημένης χρήσης null-to-string στο PHP 8.1.
- Διόρθωση: Τα Currents φορτώνονται τεμπέλικα πριν από το init hook για να αποτρέπονται προβλήματα χρονισμού.
- Διόρθωση: Η φιλτραρισμένη διαδρομή SSO γίνεται σεβαστή σε όλες τις ροές σύνδεσης.
- Διόρθωση: Οι κενές επιλογές ταυτότητας ιστοτόπου διατηρούνται κατά την αποθήκευση.

- Διόρθωση: Ο εκκρεμής ιστότοπος δεν δημοσιευόταν όταν ο διαχειριστής επαλήθευε χειροκίνητα το email πελάτη.
- Διόρθωση: Ορφανές εγγραφές pending_site καθαρίζονται όταν λείπει η συνδρομή μέλους.

- Διόρθωση: Οι εκκρεμείς ιστότοποι πλέον εμφανίζονται πρώτοι στην προβολή Όλοι οι Ιστότοποι.
- Διόρθωση: Προστέθηκε κεφαλίδα User-Agent παρόχου στιγμιότυπων οθόνης (mShots) για την αποτροπή σφαλμάτων 403.
- Διόρθωση: Επιλύθηκε κυκλική εξάρτηση του χρονοπρογραμματισμού cron εισαγωγής.
- Διόρθωση: Τα αναγνωριστικά περιήγησης κανονικοποιήθηκαν σε κάτω παύλες στα κλειδιά ρυθμίσεων χρήστη.
