# Σημειώσεις Έκδοσης

## Έκδοση 2.12.0 — Έκδοση στις 2026-05-15

- Νέο: Προστέθηκε το Hostinger (hPanel) ως υποστηριζόμενος πάροχος hosting με ενσωμάτωση χαρτογράφησης ονομάτων πεδίου (domain mapping).
- Νέο: Το Site Exporter τώρα χειρίζεται τα πακέτα εισαγωγής δικτύου (network import bundles) για μια απλοποιημένη αποκατάσταση σε όλο το δίκτυο.
- Διόρθωση: Τα emails μετάδοσης BCC χρησιμοποιούν πλέον ένα πεδίο `undisclosed-recipients` για να αποτρέψουν την έκθεση των διευθύνσεων των παραλήπτων.
- Διόρθωση: Η ημερομηνία λήξης της सदस्यता δεν είναι πλέον κατεστραμμένη κατά την αποθήκευση με μια τιμή που δεν είναι ημερομηλόγυρο.
- Διόρθωση: Οι ενημερώσεις συνδρομής του Stripe αφαιρούν πλέον σωστά τις εκπτώσεις χωρίς να καλούν την αποθησαρτωμένη (deprecated) API `deleteDiscount`.
- Διόρθωση: Οι ανακατευθύνσεις SSO σε sites με domain mapping περιορίζονται τώρα για να αποτραπούν οι άπειρες βρόχνοι ανακατεύθυνσης.
- Διόρθωση: Η επιλογή εικόνας του setup wizard ενημερώνει πλέον σωστά το υποκείμενο μοντέλο δεδομένων.
- Διόρθωση: Το Site Exporter CLI διατηρεί πλέον την σωστή προεπιλεγμένη επιλογή site του δικτύου.
- Βελτίωση: Αφαιρέθηκε το πακέτο `wp-cli` που ήταν ενσωματωμένο στο plugin, μειώνοντας το μέγεθος του plugin.

## Έκδοση 2.11.0 — Έκδοση στις 2026-05-11

- Νέο: Οι εξαγωγές sites τώρα ενσωματώνουν ένα `index.php` με αυτό-εκκίνηση (self-booting) ώστε το ZIP να μπορεί να εγκατασταθεί σε έναν νέο host χωρίς ξεχωριστή εγκατάσταση plugin.
- Νέο: Η εξαγωγή δικτύου επιτρέπει στους διαχειριστές να εξάγουν όλα τα υπο-sites σε ένα μόνο αρχείο από τη σελίδα διαχείρισης Site Export.
- Νέο: Η δυνατότητα ενεργοποίησης/απενεργοποίησης Site Templates επιβάλλεται πλέον μέσω μιας αλυσίδας fallback, περιορίζοντας σωστά τη διαθεσιμότητα των templates ανάλογα με τα όρια του plan.
- Νέο: Ο editor του checkout form προειδοποιεί όταν προστίθεται ένα προϊόν χωρίς να έχει ρυθμιστεί ένα υποχρεωτικό πεδίο.
- Νέο: Η καρτέλα ρυθμίσεων Import/Export περιγράφει πλέον ξεκάθαρα το πεδίο εφαρμογής της και συνδέεται απευθείας με το εργαλείο Site Export.

## Έκδοση 2.10.0 — Έκδοση στις 2026-05-05

- Νέο: Guided setup wizard για PayPal με χειροκίνητη εισαγωγή διαπιστευτηρίων και gate OAuth για απρόσκοπτη ρύθμιση gateway.
- Νέο: Το customer panel για αλλαγή templates αναδιαμορφώθηκε με κάρτα τρέχοντος template, διατηρητικό grid και κουμπί **Reset current template**.
- Διόρθωση: Η αλλαγή templates δεν κ้างεί πλέον το UI σε περίπτωση αποτυχίας AJAX.
- Διόρθωση: Οι καταστάσεις δικαιωμάτων για αλλαγή templates εξασφαλίστηκαν ενάντια σε μη εξουσιοδοτημένη πρόσβαση.
- Διόρθωση: Οι εισαγωγές site override επικυρώνονται πριν από την αποθήκευση.
- Διόρθωση: Η ειδοποίηση για τη διεύθυνση χρέωσης εμφανίζεται πλέον όταν η διεύθυνση είναι κενή.
- Διόρθωση: Λύνθηκαν οι ειδοποιήσεις αποθησαρτωμένων (deprecation notices) `null-to-string` του PHP 8.1.
- Διόρθωση: Το Currents φορτώνεται lazy-loaded πριν το hook `init` για να αποτραπούν τα προβλήματα χρονισμού.
- Διόρθωση: Ο φιλτραρισμένος (filtered) δρόμος SSO τηρείται σε όλες τις ροές σύνδεσης.
- Διόρθωση: Οι επιλογές κενής ταυτότητας site διατηρούνται κατά την αποθήκευση.

## Έκδοση 2.9.0 — Έκδοση στις 2026-04-30

- Νέο: Προστέθηκαν single-site export και import κάτω από **Tools > Export & Import**.
- Διόρθωση: Τα αρχεία ZIP export εξυπηρετούνται πλέον μέσω ενός τελικού σημείου λήψης που απαιτεί ταυτοποίηση (authenticated download endpoint).
- Διόρθωση: Διορθώθηκαν οι κίνδυνοι SQL injection και τα προβλήματα ερωτημάτων στις αναμενόμενες (pending) λειτουργίες export/import.
- Διόρθωση: Το site δεν δημοσιεύεται (not published) όταν ο διαχειριστής επαληθεύει χειροκίνητα το email του πελάτη.
- Διόρθωση: Τα καταγεγραμμένα records `pending_site` καθαρίζονται όταν λείπει η सदस्यता.
- Διόρθωση: Διορθώθηκαν οι αποστάσεις (padding) της πλοήγησης ρυθμίσεων και η πλοήγή του anchor search.
- Διόρθωση: Τα pending sites εμφανίζονται πλέον πρώτα στη θέα All Sites.
- Διόρθωση: Προστέθηκε το User-Agent header του παρόχου screenshots (mShots) για να αποτραπούν τα σφάλματα 403.
- Διόρθωση: Λύνεται η κυκλική εξάρτηση του cron schedule εισαγωγής.
- Διόρθωση: Οι αναγνωριστικά (IDs) των tours κανονικοποιούνται σε underscores στις κλειδιά ρυθμίσεων χρήστη.
- Βελτίωση: Χρησιμοποιείται πλέον το ZipArchive αντί για το Alchemy/Zippy για καλύτερη συμβατότητα.
