# Βασικές Έννοιες

Για κάποιον που μόλις ξεκινάει με το WordPress Multisite και το Ultimate Multisite, υπάρχουν πολλοί νέοι όροι και φράσεις που πρέπει να κατανοήσει. Η εκμάθησή τους είναι σημαντική, γιατί θα σας βοηθήσει να καταλάβετε την πλατφόρμα και τον τρόπο λειτουργίας της συνολικά.

Σε αυτό το άρθρο, θα προσπαθήσουμε να ορίσουμε και να εξηγήσουμε μερικές από τις βασικές έννοιες του WordPress. Κάποιες από αυτές αφορούν περισσότερο τους χρήστες, άλλες τους προγραμματιστές και κάποιες και τις δύο κατηγορίες.

## WordPress Multisite

Το WordPress **Multisite** είναι ένας τύπος εγκατάστασης WordPress που σας επιτρέπει να δημιουργείτε και να διαχειρίζεστε ένα δίκτυο πολλαπλών ιστοσελίδων από έναν ενιαίο πίνακα ελέγχου WordPress. Μπορείτε να διαχειριστείτε τα πάντα, συμπεριλαμβανομένου του αριθμού των ιστοσελίδων, των λειτουργιών, των θεμάτων και των ρόλων χρηστών. Είναι δυνατή η διαχείριση εκατοντάδων ή και χιλιάδων ιστοσελίδων.

## Δίκτυο

Στο πλαίσιο του WordPress, ένα δίκτυο multisite είναι το περιβάλλον όπου μπορείτε να διαχειρίζεστε πολλές **υποϊστοσελίδες** από έναν ενιαίο πίνακα ελέγχου. Αν και η δημιουργία ενός δικτύου multisite διαφέρει ανάλογα με τον πάροχο φιλοξενίας, το αποτέλεσμα είναι συνήθως μερικές επιπλέον οδηγίες στο αρχείο **wp-config.php** που ενημερώνουν το WordPress ότι λειτουργεί σε αυτή τη συγκεκριμένη λειτουργία.

Υπάρχουν αρκετές σημαντικές διαφορές μεταξύ ενός δικτύου multisite και μιας αυτόνομης εγκατάστασης WordPress, τις οποίες θα αναλύσουμε σύντομα.

## Βάση Δεδομένων

Μια βάση δεδομένων είναι ένα δομημένο, οργανωμένο σύνολο πληροφοριών. Στην ορολογία της πληροφορικής, αναφέρεται σε λογισμικό που χρησιμοποιείται για την αποθήκευση και οργάνωση δεδομένων. Σκεφτείτε το σαν ένα αρχειοθήκη όπου αποθηκεύετε δεδομένα σε διαφορετικές ενότητες που ονομάζονται πίνακες.

Το WordPress Multisite χρησιμοποιεί μία βάση δεδομένων και κάθε υποϊστοσελίδα αποκτά τους δικούς της πίνακες με το blog id στο πρόθεμα. Έτσι, μόλις εγκαταστήσετε ένα δίκτυο και δημιουργήσετε μια υποϊστοσελίδα, θα πρέπει να έχετε αυτούς τους πίνακες:

_wp_1_options_ \- πίνακας επιλογών για την πρώτη υποϊστοσελίδα

_wp_2_options_ \- πίνακας επιλογών για τη δεύτερη υποϊστοσελίδα

## Πάροχος Φιλοξενίας

Ο πάροχος φιλοξενίας είναι μια εταιρεία που επιτρέπει σε επιχειρήσεις και ιδιώτες να κάνουν τις ιστοσελίδες τους διαθέσιμες μέσω του Παγκόσμιου Ιστού. Οι υπηρεσίες που προσφέρουν οι πάροχοι φιλοξενίας ποικίλλουν, αλλά συνήθως περιλαμβάνουν σχεδιασμό ιστοσελίδων, αποθηκευτικό χώρο σε διακομιστή και σύνδεση στο διαδίκτυο.

## Domain

Το domain name (όνομα τομέα) είναι η διεύθυνση που χρησιμοποιούν οι άνθρωποι για να επισκεφθούν την ιστοσελίδα σας. Υποδεικνύει στον περιηγητή πού να αναζητήσει την ιστοσελίδα σας. Όπως ακριβώς μια ταχυδρομική διεύθυνση, έτσι και το domain είναι ο τρόπος με τον οποίο οι άνθρωποι επισκέπτονται την ιστοσελίδα σας στο διαδίκτυο. Είναι σαν να έχετε μια πινακίδα μπροστά από το κατάστημά σας. Αν θέλετε να επισκεφθείτε την ιστοσελίδα μας, θα πρέπει να πληκτρολογήσετε τη διεύθυνσή μας στη γραμμή διευθύνσεων του περιηγητή σας, που είναι [_www.ultimatemultisite.com_](http://www.ultimatemultisite.com) _,_ όπου το [**ultimatemultisite.com**](http://ultimatemultisite.com) είναι το domain name.

## Subdomain

Το subdomain είναι ένας τύπος ιεραρχίας ιστοσελίδας κάτω από το κύριο domain, αλλά αντί να χρησιμοποιεί φακέλους για την οργάνωση περιεχομένου, αποκτά ουσιαστικά τη δική του ιστοσελίδα. Παρουσιάζεται ως [**https://site1.domain.com/**](https://site1.domain.com/) όπου το _site1_ είναι το όνομα του subdomain και το [_domain.com_](http://domain.com) είναι το κύριο domain.

## Υποκατάλογος

Ο υποκατάλογος είναι ένας τύπος ιεραρχίας ιστοσελίδας κάτω από ένα βασικό domain που χρησιμοποιεί φακέλους για την οργάνωση περιεχομένου. Ο υποκατάλογος είναι το ίδιο με τον υποφάκελο και οι όροι μπορούν να χρησιμοποιηθούν εναλλακτικά. Παρουσιάζεται ως [**https://domain.com/site1**](https://domain.com/site1) όπου το _site1_ είναι το όνομα του υποκαταλόγου και το [_domain.com_](http://domain.com) είναι το κύριο domain.

## Υποϊστοσελίδα

Η υποϊστοσελίδα είναι μια θυγατρική ιστοσελίδα που δημιουργείτε σε ένα δίκτυο Multisite. Μπορεί να είναι είτε **subdomain** είτε **υποκατάλογος**, ανάλογα με τον τρόπο που έχει ρυθμιστεί η εγκατάσταση WordPress Multisite.

## Super Admin

Ο Super Admin του WordPress είναι ένας ρόλος χρήστη με πλήρεις δυνατότητες διαχείρισης όλων των υποϊστοσελίδων σε ένα δίκτυο Multisite. Για τους χρήστες Multisite, είναι το **υψηλότερο επίπεδο πρόσβασης** που μπορείτε να παρέχετε στην εγκατάσταση WordPress.

## Plugin

Γενικά, ένα plugin είναι ένα σύνολο κώδικα που προσθέτει επιπλέον λειτουργικότητα στην ιστοσελίδα σας WordPress. Αυτό μπορεί να είναι κάτι απλό όπως η αλλαγή του λογότυπου σύνδεσης ή κάτι πιο σύνθετο όπως η προσθήκη λειτουργιών ηλεκτρονικού εμπορίου. Τα _Woocommerce και Contact Form_ είναι παραδείγματα plugin.

Σε ένα WordPress Multisite, τα plugins μπορούν να εγκατασταθούν μόνο από τον πίνακα διαχείρισης δικτύου από τον Super Admin. Οι διαχειριστές υποϊστοσελίδων μπορούν μόνο να ενεργοποιούν και να απενεργοποιούν plugins μέσα στην υποϊστοσελίδα τους.

## Θέματα

Ένα θέμα WordPress είναι μια ομάδα αρχείων (_γραφικά, φύλλα στυλ και κώδικας_) που καθορίζει τη συνολική εμφάνιση της ιστοσελίδας. Παρέχει όλη τη μορφοποίηση της διεπαφής, όπως στυλ γραμματοσειράς, διάταξη σελίδας, χρώματα κ.λπ.

Όπως και τα plugins, τα θέματα στο WordPress Multisite μπορούν να εγκατασταθούν μόνο από τον Super Admin και μπορούν να ενεργοποιηθούν σε επίπεδο υποϊστοσελίδας από τους διαχειριστές υποϊστοσελίδων.

## Πρότυπο Ιστοσελίδας

Το **Πρότυπο Ιστοσελίδας** είναι μια προκατασκευασμένη ιστοσελίδα που μπορεί να χρησιμοποιηθεί ως βάση κατά τη δημιουργία νέων ιστοσελίδων στο δίκτυό σας.

Αυτό σημαίνει ότι μπορείτε να δημιουργήσετε μια βασική ιστοσελίδα, να ενεργοποιήσετε διάφορα plugins, να ορίσετε ένα ενεργό θέμα και να την προσαρμόσετε όπως επιθυμείτε. Στη συνέχεια, όταν ο πελάτης σας δημιουργεί έναν νέο λογαριασμό, αντί να λαμβάνει μια προεπιλεγμένη ιστοσελίδα WordPress χωρίς ουσιαστικό περιεχόμενο, θα λαμβάνει ένα αντίγραφο της βασικής σας ιστοσελίδας με όλες τις προσαρμογές και το περιεχόμενο ήδη στη θέση τους.

## Domain Mapping

Το **Domain mapping** στο WordPress είναι ένας τρόπος ανακατεύθυνσης των χρηστών στον σωστό διακομιστή, μέσω της διεύθυνσης μιας ιστοσελίδας. Σε ένα WordPress Multisite, οι υποϊστοσελίδες δημιουργούνται χρησιμοποιώντας είτε υποκατάλογο είτε subdomain. Αυτό που κάνει το domain mapping είναι να επιτρέπει στους χρήστες υποϊστοσελίδων να χρησιμοποιούν ένα domain ανώτατου επιπέδου όπως το [**joesbikeshop.com**](http://joesbikeshop.com) για να κάνουν τη διεύθυνση της ιστοσελίδας τους να φαίνεται πιο επαγγελματική.

## SSL

Το SSL σημαίνει **Secure Sockets Layer** (Επίπεδο Ασφαλών Υποδοχών). Είναι ένα ψηφιακό πιστοποιητικό που επαληθεύει την ταυτότητα μιας ιστοσελίδας και ενεργοποιεί μια κρυπτογραφημένη σύνδεση. Σήμερα χρησιμοποιείται ως η τυπική τεχνολογία για τη διατήρηση μιας ασφαλούς σύνδεσης στο διαδίκτυο και την προστασία ευαίσθητων δεδομένων που αποστέλλονται μεταξύ δύο συστημάτων, εμποδίζοντας εγκληματίες να διαβάσουν και να τροποποιήσουν οποιαδήποτε πληροφορία μεταφέρεται, συμπεριλαμβανομένων πιθανών προσωπικών στοιχείων. Οι σύγχρονοι περιηγητές απαιτούν SSL, καθιστώντας το απαραίτητο κατά τη δημιουργία και λειτουργία μιας ιστοσελίδας.

## Πολυμέσα

Τα πολυμέσα είναι εικόνες, ήχος, βίντεο και άλλα αρχεία που απαρτίζουν μια ιστοσελίδα.

Οι ιστοσελίδες του δικτύου μοιράζονται μία κοινή βάση δεδομένων σε ένα WordPress Multisite, αλλά διατηρούν ξεχωριστές διαδρομές στο σύστημα αρχείων για τα αρχεία πολυμέσων.

Η τυπική τοποθεσία του WordPress (wp-content/uploads) παραμένει· ωστόσο, η διαδρομή της τροποποιείται για να αντικατοπτρίζει το μοναδικό ID της ιστοσελίδας του δικτύου. Κατά συνέπεια, τα αρχεία πολυμέσων για μια ιστοσελίδα δικτύου εμφανίζονται ως wp-contents/uploads/site/[id].

## Μόνιμοι Σύνδεσμοι

Οι μόνιμοι σύνδεσμοι είναι οι μόνιμες διευθύνσεις URL των μεμονωμένων αναρτήσεων ή σελίδων του ιστολογίου σας μέσα στην ιστοσελίδα σας. Οι μόνιμοι σύνδεσμοι αναφέρονται επίσης ως **pretty links**. Από προεπιλογή, τα URL του WordPress χρησιμοποιούν τη μορφή query string που μοιάζει κάπως έτσι:

[**http://www.example.com/registration**](http://www.example.com/registration)

## Ultimate Multisite

Το Ultimate Multisite είναι ένα plugin για WordPress, σχεδιασμένο για εγκαταστάσεις WordPress Multisite, που μετατρέπει την εγκατάστασή σας WordPress σε ένα premium δίκτυο ιστοσελίδων – όπως το [WordPress.com](https://WordPress.com) – επιτρέποντας στους πελάτες να δημιουργούν ιστοσελίδες με μηνιαίες, τριμηνιαίες ή ετήσιες χρεώσεις (μπορείτε επίσης να δημιουργήσετε δωρεάν πακέτα).

## Φόρμα Ολοκλήρωσης Αγοράς

Η Φόρμα Ολοκλήρωσης Αγοράς είναι μια φόρμα παραγγελίας ενός ή πολλαπλών βημάτων που περιλαμβάνει τη δημιουργία υποϊστοσελίδας, συνδρομής και λογαριασμών χρηστών μέσω της εγγραφής στο Ultimate Multisite. Αποτελείται από διάφορα πεδία και φόρμες πληρωμής που πρέπει να υποβάλει ο χρήστης κατά τη διαδικασία εγγραφής.

## Webhook

Ένα webhook (γνωστό επίσης ως web callback ή HTTP push API) είναι ένας τρόπος για μια εφαρμογή να παρέχει σε άλλες εφαρμογές πληροφορίες σε πραγματικό χρόνο. Ένα webhook παραδίδει δεδομένα σε άλλες εφαρμογές τη στιγμή που συμβαίνουν, πράγμα που σημαίνει ότι λαμβάνετε τα δεδομένα αμέσως.

Τα **webhooks του Ultimate Multisite** ανοίγουν απεριόριστες δυνατότητες, επιτρέποντας στους διαχειριστές δικτύου να κάνουν κάθε είδους πρωτότυπες αλλά χρήσιμες ενοποιήσεις, ειδικά σε συνδυασμό με υπηρεσίες όπως το _Zapier και το IFTTT_.

## Συμβάντα

Ένα Συμβάν είναι μια ενέργεια που συμβαίνει ως αποτέλεσμα μιας ενέργειας του χρήστη ή άλλης πηγής, όπως ένα κλικ του ποντικιού. Το Ultimate Multisite διατηρεί αρχείο όλων των συμβάντων και καταγραφών που συμβαίνουν σε ολόκληρο το δίκτυό σας. Παρακολουθεί διάφορες δραστηριότητες που συμβαίνουν στο multisite σας, όπως αλλαγές πακέτων.
