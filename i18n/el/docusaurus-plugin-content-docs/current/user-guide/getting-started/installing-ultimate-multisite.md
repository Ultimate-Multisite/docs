# Εγκατάσταση του Ultimate Multisite

:::note
Αυτό το εκπαιδευτικό πρόγραμμα υποθέτει ότι έχετε ήδη εγκαταστήσει και διαμορφώσει το WordPress Multisite. Για να μάθετε πώς, δείτε αυτό το [εκπαιδευτικό πρόγραμμα](https://www.wpbeginner.com/glossary/multisite/) από το WP Beginner.
:::

## Εγκατάσταση του Προσθέματος (Plugin)

Το Ultimate Multisite είναι διαθέσιμο δωρεάν στο [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/).

Από τον **Πίνακα Ελέγχου Δικτύου** σας, μεταβείτε στην ενότητα **Προσθήματα → Προσθήκη Νέου Προσθέματος**.

![Σελίδα "Προσθήκη Νέου Προσθέματος" στον Πίνακα Ελέγχου Δικτύου](/img/installation/add-new-plugin.png)

Αναζητήστε το **"Ultimate Multisite"** (με εισαγωγικά για ακριβή αντιστοιχία) και θα εμφανιστεί ως το πρώτο αποτέλεσμα. Κάντε κλικ στο **Εγκατάσταση Τώρα**.

![Αποτελέσματα αναζήτησης που δείχνουν το Ultimate Multisite](/img/installation/search-ultimate-multisite.png)

Μόλις εγκατασταθεί, κάντε κλικ στο **Ενεργοποίηση σε Δίκτυο** για να ενεργοποιήσετε το πρόσθεμα σε ολόκληρο το δίκτυό σας.

![Πρόσθεμα εγκατεστημένο με κουμπί "Ενεργοποίηση σε Δίκτυο"](/img/installation/plugin-installed.png)

Μετά την ενεργοποίηση, θα ανακατευθυνθείτε αυτόματα στον Οδηγό Ρύθμισης (Setup Wizard).

![Πρόσθεμα ενεργοποιημένο και ανακατεύθυνση στον οδηγό](/img/installation/plugin-activated.png)

## Οδηγός Ρύθμισης

Ο Οδηγός Ρύθμισης θα σας καθοδηγήσει στη διαμόρφωση του Ultimate Multisite σε περίπου 10 λεπτά.

### Καλώς ήρθατε

Κάντε κλικ στο **Ξεκινήστε** για να ξεκινήσετε.

![Αρχική οθόνη του Οδηγού Ρύθμισης](/img/installation/wizard-welcome.png)

### Έλεγχοι Προεγκατάστασης

Αυτό το βήμα ελέγχει τις πληροφορίες του συστήματός σας και την εγκατάσταση του WordPress για να διασφαλίσει ότι πληροί τις απαιτήσεις του Ultimate Multisite. Εάν όλα φαίνονται καλά, κάντε κλικ στο **Μεταβείτε στο Επόμενο Βήμα**.

![Έλεγχοι προεγκατάστασης που δείχνουν τις απαιτήσεις του συστήματος](/img/installation/wizard-pre-install-checks.png)

:::note Κουμπί "Ενεργοποίηση σε Δίκτυο" (v2.6.1+)
Εάν το Ultimate Multisite έχει εγκατασταθεί αλλά **δεν έχει ακόμη ενεργοποιηθεί σε επίπεδο δικτύου** — για παράδειγμα, εάν κάνατε κλικ στο **Ενεργοποίηση** (για μία μόνο ιστοσελίδα) αντί για το **Ενεργοποίηση σε Δίκτυο** από την οθόνη των προσθέτων του δικτύου — το βήμα "Έλεγχοι Προεγκατάστασης" θα το ανιχνεύσει και θα εμφανίσει ένα κουμπί **Ενεργοποίηση σε Δίκτυο**.

Κάνοντας κλικ στο **Ενεργοποίηση σε Δίκτυο**, ενεργοποιείται αυτόματα το πρόσθεμα σε ολόκληρο το δίκτυό σας. Μόλις ενεργοποιηθεί, ο οδηγός συνεχίζει κανονικά στο βήμα εγκατάστασης. Δεν χρειάζεται να αποχωρήσετε από τον οδηγό για να διορθώσετε την κατάσταση ενεργοποίησης.
:::

### Εγκατάσταση

Το πρόγραμμα εγκατάστασης θα δημιουργήσει τις απαραίτητες πίνακες βάσης δεδομένων και θα εγκαταστήσει το αρχείο `sunrise.php` που χρειάζεται το Ultimate Multisite για να λειτουργήσει. Κάντε κλικ στο **Εγκατάσταση** για να συνεχίσετε.

![Βήμα εγκατάστασης που δείχνει τις πίνακες βάσης δεδομένων και το sunrise.php](/img/installation/wizard-installation.png)

### Η Εταιρεία σας

Συμπληρώστε τις πληροφορίες της εταιρείας σας και ορίστε το προεπιλεγμένο νόμισμά σας. Αυτές οι πληροφορίες θα χρησιμοποιηθούν σε όλη την πλατφόρμα WaaS (Website as a Service) που διαθέτετε. Κάντε κλικ στο **Συνέχεια** όταν τελειώσετε.

![Βήμα διαμόρφωσης "Η Εταιρεία σας"](/img/installation/wizard-your-company.png)

### Προεπιλεγμένο Περιεχόμενο

Αυτό το βήμα σάς επιτρέπει να εγκαταστήσετε προκαθορισμένα πρότυπα, προϊόντα και άλλο περιεχόμενο εκκίνησης. Αυτός είναι ένας πολύ καλός τρόπος για να εξοικειωθείτε με τις δυνατότητες του Ultimate Multisite. Κάντε κλικ στο **Εγκατάσταση** για να προσθέσετε το προεπιλεγμένο περιεχόμενο ή παραλείψτε αυτό το βήμα εάν προτιμάτε να ξεκινήσετε από την αρχή.

![Βήμα εγκατάστασης προεπιλεγμένου περιεχομένου](/img/installation/wizard-default-content.png)

### Συνιστώμενα Προσθήματα

Εγκαταστήστε προαιρετικά συνιστώμενα πρόσθετα συνοδευτικά. Κάντε κλικ στο **Εγκατάσταση** για να τα προσθέσετε ή παραλείψτε το για να συνεχίσετε.

![Βήμα των συνιστώμενων προσθέτων](/img/installation/wizard-recommended-plugins.png)

### Έτοιμο!

Αυτό είναι όλο! Η εγκατάσταση του Ultimate Multisite έχει ολοκληρωθεί. Τώρα μπορείτε να αρχίσετε να δημιουργείτε την πλατφόρμα Website as a Service από τον **Πίνακα Ελέγχου Δικτύου**.

![Οδηγός Ρύθμισης - Ολοκλήρωση](/img/installation/wizard-ready.png)

![Πίνακας Ελέγχου Δικτύου με ενεργοποιημένο το Ultimate Multisite](/img/installation/network-dashboard.png)

Ξεκινήστε και διασκεδάστε!
