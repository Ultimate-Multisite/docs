# Εγκατάσταση του Ultimate Multisite

:::note
Αυτός ο οδηγός προϋποθέτει ότι έχετε ήδη εγκαταστήσει και ρυθμίσει το WordPress Multisite. Για να μάθετε πώς, δείτε [αυτόν τον οδηγό](https://www.wpbeginner.com/glossary/multisite/) από το WP Beginner.
:::

## Εγκατάσταση του Plugin

Το Ultimate Multisite είναι διαθέσιμο δωρεάν στο [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/).

Από τον **Πίνακα Διαχείρισης Δικτύου**, μεταβείτε στο **Πρόσθετα → Προσθήκη Νέου**.

![Σελίδα Προσθήκη Νέου Προσθέτου στη Διαχείριση Δικτύου](/img/installation/add-new-plugin.png)

Αναζητήστε **"Ultimate Multisite"** (με εισαγωγικά για ακριβή αντιστοίχιση) και θα εμφανιστεί ως πρώτο αποτέλεσμα. Κάντε κλικ στο **Εγκατάσταση Τώρα**.

![Αποτελέσματα αναζήτησης που δείχνουν το Ultimate Multisite](/img/installation/search-ultimate-multisite.png)

Μετά την εγκατάσταση, κάντε κλικ στο **Ενεργοποίηση Δικτύου** για να ενεργοποιήσετε το plugin σε ολόκληρο το δίκτυό σας.

![Plugin εγκατεστημένο με κουμπί Ενεργοποίησης Δικτύου](/img/installation/plugin-installed.png)

Μετά την ενεργοποίηση, θα μεταφερθείτε αυτόματα στον Οδηγό Εγκατάστασης.

![Plugin ενεργοποιημένο και ανακατεύθυνση στον οδηγό](/img/installation/plugin-activated.png)

## Οδηγός Εγκατάστασης

Ο Οδηγός Εγκατάστασης θα σας καθοδηγήσει στη ρύθμιση του Ultimate Multisite σε περίπου 10 λεπτά.

### Καλωσόρισμα

Κάντε κλικ στο **Ξεκινήστε** για να αρχίσετε.

![Οθόνη καλωσορίσματος του Οδηγού Εγκατάστασης](/img/installation/wizard-welcome.png)

### Έλεγχοι Προεγκατάστασης

Αυτό το βήμα ελέγχει τις πληροφορίες του συστήματός σας και την εγκατάσταση του WordPress για να βεβαιωθεί ότι πληρούν τις απαιτήσεις του Ultimate Multisite. Αν όλα φαίνονται εντάξει, κάντε κλικ στο **Μετάβαση στο Επόμενο Βήμα**.

![Έλεγχοι προεγκατάστασης που δείχνουν τις απαιτήσεις συστήματος](/img/installation/wizard-pre-install-checks.png)

### Εγκατάσταση

Το πρόγραμμα εγκατάστασης θα δημιουργήσει τους απαραίτητους πίνακες βάσης δεδομένων και θα εγκαταστήσει το αρχείο `sunrise.php` που χρειάζεται το Ultimate Multisite για να λειτουργήσει. Κάντε κλικ στο **Εγκατάσταση** για να συνεχίσετε.

![Βήμα εγκατάστασης που δείχνει πίνακες βάσης δεδομένων και sunrise.php](/img/installation/wizard-installation.png)

### Η Εταιρεία σας

Συμπληρώστε τα στοιχεία της εταιρείας σας και ορίστε το προεπιλεγμένο νόμισμα. Αυτές οι πληροφορίες θα χρησιμοποιούνται σε όλη την πλατφόρμα WaaS σας. Κάντε κλικ στο **Συνέχεια** όταν τελειώσετε.

![Βήμα ρύθμισης Η Εταιρεία σας](/img/installation/wizard-your-company.png)

### Προεπιλεγμένο Περιεχόμενο

Αυτό το βήμα σας επιτρέπει να εγκαταστήσετε προκαθορισμένα πρότυπα, προϊόντα και άλλο αρχικό περιεχόμενο. Είναι ένας εξαιρετικός τρόπος να εξοικειωθείτε με τις δυνατότητες του Ultimate Multisite. Κάντε κλικ στο **Εγκατάσταση** για να προσθέσετε το προεπιλεγμένο περιεχόμενο, ή παραλείψτε αυτό το βήμα αν προτιμάτε να ξεκινήσετε από την αρχή.

![Βήμα εγκατάστασης προεπιλεγμένου περιεχομένου](/img/installation/wizard-default-content.png)

### Προτεινόμενα Plugins

Προαιρετικά εγκαταστήστε προτεινόμενα συμπληρωματικά plugins. Κάντε κλικ στο **Εγκατάσταση** για να τα προσθέσετε ή παραλείψτε για να συνεχίσετε.

![Βήμα προτεινόμενων plugins](/img/installation/wizard-recommended-plugins.png)

### Έτοιμο!

Αυτό ήταν! Η εγκατάσταση του Ultimate Multisite ολοκληρώθηκε. Μπορείτε τώρα να αρχίσετε να χτίζετε την πλατφόρμα Website as a Service από τον **Πίνακα Διαχείρισης Δικτύου**.

![Εγκατάσταση ολοκληρώθηκε - Οθόνη Έτοιμο](/img/installation/wizard-ready.png)

![Πίνακας Διαχείρισης Δικτύου με το Ultimate Multisite ενεργό](/img/installation/network-dashboard.png)

Προχωρήστε και διασκεδάστε!
