# Απαιτήσεις του Ultimate Multisite

Το πρώτο πράγμα που πιθανότατα αναρωτιέστε πριν αγοράσετε το Ultimate Multisite είναι ποιες είναι οι απαιτήσεις για να λειτουργήσει το plugin μας. Ευτυχώς, υπάρχουν μόνο λίγες βασικές απαιτήσεις για να τρέξετε το Ultimate Multisite: ένα **όνομα τομέα (domain name)**, έναν καλό **πάροχο φιλοξενίας (hosting)** και μια εγκατάσταση WordPress **Multisite**. Αυτά είναι όλα όσα χρειάζεστε!

## Όνομα Τομέα

Αρχικά, ας μιλήσουμε για το όνομα τομέα.

Όταν επιλέγετε ένα όνομα τομέα, σκεφτείτε ποια είναι η συγκεκριμένη σας εξειδίκευση και τι θα δημιουργήσετε με το Ultimate Multisite: Θέλετε να δημιουργήσετε μια εταιρεία κατασκευής ιστοσελίδων; Να πουλάτε ιστοσελίδες τύπου ηλεκτρονικού καταστήματος; Ή ίσως υπηρεσίες φιλοξενίας WordPress; Όλα είναι εφικτά με το Ultimate Multisite.

Το plugin μας παρέχει υπηρεσίες συνδρομής, είσπραξη πληρωμών, φόρμες ολοκλήρωσης αγοράς, κουπόνια έκπτωσης και επικοινωνία με πελάτες. Μεγάλο μέρος της εργασίας που απαιτείται για την εγκατάσταση, ρύθμιση και συντήρηση ενός WordPress Multisite διευκολύνεται από το Ultimate Multisite.

Οποιαδήποτε εξειδίκευση επιλέξετε μπορεί να διαχειριστεί με το plugin μας, αρκεί να πουλάτε ιστοσελίδες ως υπηρεσία! Φροντίστε να επιλέξετε ένα όνομα τομέα που ταιριάζει με το επιχειρηματικό σας πλάνο.

## Πάροχος Φιλοξενίας

Τώρα που έχετε ένα ελκυστικό όνομα τομέα που αντιπροσωπεύει την εξειδίκευση και την επιχείρησή σας, θα χρειαστεί να επιλέξετε μια καλή εταιρεία φιλοξενίας για να φιλοξενήσει το δίκτυό σας με το Ultimate Multisite. Υπάρχουν αρκετές καλές επιλογές στην αγορά. Ορισμένοι πάροχοι φιλοξενίας ενσωματώνονται καλά με το Ultimate Multisite, κάτι που μπορεί να αποτελέσει καθοριστικό παράγοντα ανάλογα με το τι σκοπεύετε να δημιουργήσετε.

Το Ultimate Multisite θα πρέπει να λειτουργεί σε οποιονδήποτε πάροχο φιλοξενίας. Η μόνη διαφορά είναι ότι ορισμένες εταιρείες φιλοξενίας όπως η _WPMU DEV Hosting_, η _WP Engine_, η _Closte_, η _Cloudways_, η _Gridpane_ και η _Runcloud_ έχουν πιο στενή ενσωμάτωση για την αντιστοίχιση τομέων (domain mapping). Για άλλους παρόχους φιλοξενίας, ίσως χρειαστεί να αντιστοιχίσετε τους προσαρμοσμένους τομείς χειροκίνητα. Είναι επίσης καλή ιδέα να επιλέξετε έναν πάροχο φιλοξενίας που υποστηρίζει AutoSSL. Η _Closte_, η _Runcloud_ και το _cPanel_ το υποστηρίζουν.

Όσον αφορά τις προδιαγραφές του διακομιστή, δεν υπάρχει συγκεκριμένη φόρμουλα, καθώς εξαρτάται από τις δυνατότητες και λειτουργίες που θα προσφέρετε και το μέγεθος του δικτύου σας. Μια κοινόχρηστη φιλοξενία (shared hosting) μπορεί να είναι μια καλή αρχή, αλλά μόλις το δίκτυό σας αρχίσει να μεγαλώνει, πιθανότατα θα χρειαστείτε έναν αποκλειστικό VPS.

## WordPress Multisite

Τώρα, ας μιλήσουμε λίγο για την εγκατάσταση WordPress Multisite.

Το WordPress Multisite σας επιτρέπει να έχετε ένα δίκτυο ιστοσελίδων σε μία μόνο εγκατάσταση. Αυτή είναι μια ενσωματωμένη δυνατότητα, αλλά δεν είναι ενεργοποιημένη από προεπιλογή. Πρέπει να ενεργοποιήσετε αυτή τη δυνατότητα μέσα από την εγκατάσταση WordPress και τον πίνακα ελέγχου της φιλοξενίας σας.

Η ύπαρξη WordPress με ενεργοποιημένο το Multisite **αποτελεί τη βάση για το Ultimate Multisite**. Δεδομένου ότι η κύρια χρήση του Ultimate Multisite είναι να βοηθήσει τους χρήστες να διαχειρίζονται και να πουλούν ιστοσελίδες ως υπηρεσία, αυτό πρέπει να γίνεται σε WordPress με ενεργοποιημένο το Multisite.

Έχουμε **ένα αναλυτικό άρθρο** που εξηγεί βήμα προς βήμα τι χρειάζεται να γίνει για να ενεργοποιήσετε τη δυνατότητα Multisite στην εγκατάσταση WordPress σας.

Αφού ενεργοποιήσετε το Multisite στο WordPress, ήρθε η ώρα να εγκαταστήσετε το Ultimate Multisite. Για να εγκαταστήσετε σωστά το Ultimate Multisite, πρέπει να έχετε:

  * WordPress: v5.3+ (Συνιστάται: Η τελευταία σταθερή έκδοση)

  * PHP: 7.4.x (η υποστήριξη για > 8.0 έρχεται σύντομα)

  * MySQL: v5+ (Συνιστάται: 5.6, σε περίπτωση που δεν έχετε διαθέσιμη την 8.0 στον πάροχο φιλοξενίας σας)

Αυτές είναι οι μόνες απαιτήσεις λογισμικού για το Ultimate Multisite.

Να έχετε υπόψη ότι το Ultimate Multisite μπορεί να λειτουργήσει με PHP 8.0, αλλά συνιστούμε να το τρέχετε με PHP 7.4.x.

Επίσης, συνιστούμε να το εγκαταστήσετε σε **κύριους τομείς**, όχι σε subdomains ή υποκαταλόγους. Μπορεί να λειτουργήσει σε subdomains/υποκαταλόγους, αλλά ενδέχεται να εμφανίσει κάποια σφάλματα.

Για έναν αναλυτικό οδηγό σχετικά με τον τρόπο εγκατάστασης του Ultimate Multisite, **κάντε κλικ εδώ**.
