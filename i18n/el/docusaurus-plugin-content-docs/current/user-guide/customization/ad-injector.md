# Τοποθέτηση διαφημίσεων στο δίκτυό σας με το Ultimate Multisite: Ad Injector

Το Ultimate Multisite: Ad Injector είναι ένα **δωρεάν πρόσθετο** που σας δίνει τα εργαλεία για να τοποθετείτε διαφημίσεις στα πλάνα σας απευθείας από τη σελίδα επεξεργασίας τους.

Αφού εγκαταστήσετε το Ultimate Multisite: Ad Injector, μεταβείτε στη σελίδα _Πρόσθετα_ για να το ενεργοποιήσετε.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Κάντε κλικ στο κουμπί Ενεργοποίηση Δικτύου_

Μόλις το πρόσθετο είναι ενεργό, το πλαίσιο _Προχωρημένες Επιλογές_ στην οθόνη επεξεργασίας πλάνου θα έχει μια νέα επιλογή που ονομάζεται _Ρυθμίσεις Διαφημίσεων_.

![Ρυθμίσεις Διαφημίσεων στην οθόνη επεξεργασίας πλάνου](/img/config/product-edit-full.png)

_Τώρα μπορείτε να επεξεργάζεστε τις ρυθμίσεις διαφημίσεων για κάθε πλάνο σας_

Σε αυτήν την καρτέλα, θα μπορείτε να προσθέσετε τους κωδικούς διαφημίσεων για άρθρα και σελίδες στο front-end, πριν και μετά το περιεχόμενο.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Αφού εισάγετε τον κωδικό της διαφήμισής σας, θα εμφανίζεται στα άρθρα και τις σελίδες του συγκεκριμένου πλάνου._

Μπορείτε επίσης να έχετε διαφημίσεις στην κεφαλίδα των σελίδων διαχείρισης για τους χρήστες αυτού του πλάνου.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Μεταβείτε στην επιλογή Κωδικός Διαφήμισης Πίνακα Διαχείρισης αν θέλετε να εμφανίζετε διαφημίσεις στο back-end των χρηστών σας_

### Πέρα από τη Διαφήμιση

Μπορείτε επίσης να χρησιμοποιήσετε το Ultimate Multisite: Ad Injector για να προσφέρετε υποστήριξη σε πλάνα υψηλότερης βαθμίδας.

Απλά μεταβείτε στην επιλογή _Διαφημίσεις Πίνακα Διαχείρισης_ και προσθέστε τον κώδικα HTML για το εργαλείο υποστήριξής σας.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Προσθέστε υποστήριξη πελατών σε συγκεκριμένα πλάνα_

Με αυτόν τον τρόπο οι χρήστες μπορούν να επικοινωνήσουν μαζί σας απευθείας από τον Πίνακα Διαχείρισής τους!
