# Εξαγωγή & Εισαγωγή

Το Ultimate Multisite 2.9.0 προσθέτει ένα εργαλείο **Εξαγωγής & Εισαγωγής** για αυτόνομη ιστοσελίδα κάτω από **Tools > Export & Import**. Χρησιμοποιήστε το όταν χρειάζεται να συσκευάσετε μια ιστοσελίδα WordPress σε αρχείο ZIP, να αποκαταστήσετε αυτό το ZIP, ή να μετακινήσετε μια ιστοσελίδα μεταξύ συμβατών εγκαταστάσεων Ultimate Multisite και WordPress για αυτόνομη ιστοσελίδα.

## Απαιτούμενα δικαιώματα

Πρέπει να συνδεθείτε ως διαχειριστής που έχει πρόσβαση στο μενού **Tools** του WordPress στην ιστοσελίδα που πρόκει να εξαχθεί ή να εισαχθεί. Σε ένα multisite network, χρησιμοποιήστε έναν λογαριασμό διαχειριστή του network όταν εξάγετε ή εισάγετε υπο-ιστοσελίδες από τα εργαλεία Ultimate Multisite σε επίπεδο network.

Οι λήψεις ZIP εξαγωγής εξυπηρετούν μέσω ενός προστατευμένου endpoint λήψης, οπότε διατηρήστε ενεργή τη συνεδρία του διαχειριστή μέχρι να ολοκληρωθεί η λήψη και μην μοιραστείτε ποτέ δημόσια τα δημιουργημένα URL λήψης.

## Εξαγωγή μιας ιστοσελίδας σε ZIP

1. Στο διαχειριστικό πάνελ του WordPress της ιστοσελίδας που θέλετε να αντιγράψετε, μεταβείτε στο **Tools > Export & Import**.
2. Ανοίξτε την περιοχή εξαγωγής και επιλέξτε την ιστοσελίδα που θέλετε να συσκευάσετε.
3. Επιλέξτε τα προαιρετικά περιεholdύγια για συμπερίληψη, όπως οι uploads, τα plugins και τα themes, όταν αυτές οι επιλογές είναι διαθέσιμες.
4. Ξεκινήστε την εξαγωγή και περιμένετε να ολοκληρωθεί η διαδικασία. Οι μεγάλες ιστοσελίδες μπορεί να χρειαστούν να ολοκληρωθούν στο παρασκήνιο πριν είναι έτοιμο το ZIP.
5. Κατεβάστε το ολοκληρωμένο ZIP από τη λίστα εξαγωγών.

Κρατήστε το ZIP σε μια ασφαλή τοποθεσία. Μπορεί να περιέχει περιεχόμενο ιστοσελίδας, ρυθμίσεις, αρχεία media και επιλεγμένα στοιχεία κώδικα.

## Τι περιλαμβάνει η εξαγωγή

Ένα ZIP εξαγωγής μπορεί να περιέχει:

- Το περιεχόμενο της βάσης δεδομένων και τη διαμόρφωση της επιλεγμένης ιστοσελίδας.
- Τα αρχεία media που έχουν ανεβαστεί, όταν συμπεριλαμβάνονται τα uploads.
- Plugins και themes, όταν επιλέγονται αυτές οι επιλογές.
- Μεταδεδομένα εισαγωγής που χρησιμοποιούνται από το εργαλείο Export & Import για την αναδόμηση της ιστοσελίδας στην τελική εγκατάσταση.

Το ακριβές μέγεθος του ZIP εξαρτάται από την ποσότητα των media, τα επιλεγμένα plugins και themes, και το μέγεθος των πινάκων της βάσης δεδομένων της ιστοσελίδας.

## Εισαγωγή μιας ιστοσελίδας από ZIP

1. Μεταβείτε στο **Tools > Export & Import** στην τελική ιστοσελίδα WordPress.
2. Ανοίξτε την περιοχή εισαγωγής και ανεβάστε το ZIP που δημιουργήθηκε από το εργαλείο Export & Import.
3. Εισάγετε ένα URL αντικατάστασης αν η ιστοσελίδα πρέπει να χρησιμοποιήσει μια νέα διεύθυνση, ή αφήστε το πεδίο κενό για να διατηρήσετε το αρχικό URL.
4. Επιλέξτε αν θέλετε να διαγραφεί το ανεβασμένο ZIP αφού ολοκληρωθεί η εισαγωγή.
5. Κάντε κλικ στο **Begin Import**.
6. Παρακολουθήστε την ανα挂λημένη εισαγωγή μέχρι να ολοκληρωθεί. Χρησιμοποιήστε το **Cancel Import** μόνο αν χρειάζεται να σταματήσετε τη διαδικασία πριν την ολοκλήρωση.
7. Ελέγξτε την εισαχθείσα ιστοσελίδα πριν επιτρέψετε την κανονική κίνηση ή την πρόσβαση των πελατών.

Σε μια εγκατάσταση WordPress για αυτόνομη ιστοσελίδα, η εισαγωγή ενός ZIP αντικαθιστά την τρέχουσα ιστοσελίδα με την εισαχθείσα. Δημιουργήστε πλήρη backup της στόχευσης ιστοσελίδας πριν ξεκινήσετε, και αποφύγετε να ξεκινάτε πολλαπλές εισαγωγές για την ίδια ιστοσελίδα ταυτόχρονα.

## Περιορισμοί και σημειώσεις συμβατότητας

- Πολύ μεγάλοι καταλόγους uploads ή βιβλιοθήκες media μπορούν να δημιουργήσουν μεγάλα αρχεία ZIP. Επιβεβαιώστε τα όρια upload του PHP, τα όρια εκτέλεσης, τον χώρο δίσκου, τη μνήμη και τις ρυθμίσεις timeout του server πριν εξάγετε ή εισάγετε μεγάλες ιστοσελίδες.
- Οι πολύ μεγάλες βιβλιοθήκες media ενδέχεται να χρειαστούν μετακίνηση κατά τη διάρκεια μιας συντήρησης με χαμηλή κίνηση.
- Η στόχευση εγκατάσταση WordPress πρέπει να τρέχει συμβατές εκδόσεις WordPress, PHP, Ultimate Multisite, plugin και theme.
- Μια εισαγωγή για αυτόνομη ιστοσελίδα αντικαθιστά την στόχευση ιστοσελίδα. Δεν είναι εργαλείο συγχώνευσης.
- Οι μετακινήσεις από multisite σε αυτόνομη ιστοσελίδα και από αυτόνομη ιστοσελίδα σε multisite υποστηρίζονται μόνο όταν το στόχευση περιβάλλον μπορεί να τρέξει τα plugins, themes, URLs και τα απαιτούμενα στοιχεία Ultimate Multisite της εξαγόμενης ιστοσελίδας.
- Κρατήστε το ZIP ιδιωτικό. Μπορεί να περιέχει περιεχόμενο βάσης δεδομένων, ανεβασμένα αρχεία και λεπτομέρειες διαμόρφωσης από την εξαγόμενη ιστοσελίδα.

Για παλαιότερες ροές εργασίας εξαγωγής σε επίπεδο network, δείτε [Site Export](/user-guide/administration/site-export).
