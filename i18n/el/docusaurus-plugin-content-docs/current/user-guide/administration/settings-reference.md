# Αναφορά Ρυθμίσεων

Αυτή η σελίδα παρακολουθεί τις ρυθμίσεις που επηρεάζουν τη καθημερινή διαχείριση και τις πρόσφατες αλλαγές συμπεριφοράς στο Ultimate Multisite.

## Άλλες Επιλογές (Other Options)

Η περιοχή **Άλλες Επιλογές (Other Options)** εμφανίζεται κάτω από **Ultimate Multisite > Settings > Login & Registration**.

| Ρύθμιση | Περιγραφή |
|---|---|
| **Enable Jumper** | Εμφανίζει το εργαλείο γρήγορης πλοήγησης Jumper στην περιοχή διαχείρισης. Χρησιμοποιήστε το για να μεταπηδήσετε απευθείας στις οθόνες του Ultimate Multisite, στους αντικείμενα του network και στις υποστηριζόμενες προορισμούς διαχείρισης. Απενεργοποιήστε το αν δεν θέλετε αυτό το shortcut να είναι ορατό. |

## Αναφορά Σφαλμάτων και Τηλεμετρία (Error reporting and telemetry)

Η προηγούμενη ρύθμιση opt-in για αναφορά σφαλμάτων έχει αφαιρεθεί από τη σελίδα ρυθμίσεων. Η ανωνυμική τηλεμετρία είναι απενεργοποιημένη και δεν υπάρχει διακόπτης UI για να την ενεργοποιήσετε.

Εάν διατηρείτε εσωτερικά runbooks ή στιγμιότυπα οθόνης για τη σελίδα ρυθμίσεων, αφαιρέστε τις αναφορές για το παλιό πεδίο opt-in αναφοράς σφαλμάτων, ώστε οι διαχειριστές να μην ψάχνουν για μια ρύθμιση που δεν υπάρχει πλέον.

## Ρυθμίσεις Εισαγωγής/Εξαγωγής (Import/Export settings)

Η καρτέλα **Ρυθμίσεις Εισαγωγής/Εξαγωγής (Import/Export)** περιγράφει ποιες ρυθμίσεις ελέγχει και συνδέεται απευθείας στο **Ultimate Multisite > Site Export** για αρχεία καταγραφής (archives) σε επίπεδο site και network. Χρησιμοποιήστε την καρτέλα ρυθμίσεων για τη διαμόρφωση εισαγωγής/εξαγωγής, χρησιμοποιήστε το **Tools > Export & Import** για τη ροή εργασίας εξαγωγής/εισαγωγής ενός μόνο site, και χρησιμοποιήστε το εργαλείο Site Export όταν χρειάζεστε ένα πλήρες αρχείο καταγραφής Network Export.
