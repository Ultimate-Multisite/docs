# Φόρμες Επικοινωνίας με Site Builder

Το Superdav AI Agent v1.10.0 προσθέτει την δυνατότητα δημιουργίας φόρமς επικοινωνίας απευθείας από τη διεπαφή chat, χρησιμοποιώντας το Site Builder agent. Αυτό σας επιτρέπει να προσθέτετε πλήρως λειτουργικές φόρμες επικοινωνίας σε οποιαδήποτε σελίδα χωρίς να χρειαστεί να απομακρυνθείτε από το chat.

## Γενική Επισκόπηση {#overview}

Η δυνατότητα φόρμας επικοινωνίας του Site Builder (`create_contact_form`) ανιχνεύει αυτόματα το plugin φόρμας που έχετε εγκαταστήσει και δημιουργεί μια φόρμα επικοινωνίας χρησιμοποιώντας τις εγγενείς δυνατότητες αυτού του plugin. Τα υποστηριζόμενα plugins φόρμας περιλαμβάνουν:

- Contact Form 7
- WPForms
- Fluent Forms
- Gravity Forms

Το βοηθητικό πρόγραμμα επιλέγει αυτόματα το καλύτερο διαθέσιμο plugin φόρμας στην ιστοσελίδα σας και δημιουργεί μια φόρμα επικοινωνίας προσαρμοσμένη σε αυτό το plugin.

## Δημιουργία Φόρμας Επικοινωνίας {#creating-a-contact-form}

Για να δημιουργήσετε μια φόρμα επικοινωνίας χρησιμοποιώντας το Site Builder:

1. Ανοίξτε το παράθυρο chat του **Gratis AI Agent** στο dashboard του WordPress.
2. Μεταβείτε στο **Site Builder** agent κάνοντας κλικ στο εικονίδιο του agent στην κεφαλίδα του chat.
3. Περιγράψτε τη φόρμα επικοινωνίας που θέλετε να δημιουργήσετε. Για παράδειγμα:

   > "Πρόστε μια φόρμα επικοινωνίας στη σελίδα Επικοινωνίας με πεδία για όνομα, email, μήνυμα και τηλέφωνο."

   Ή απλά:

   > "Δημιούργησε μια φόρμα επικοινωνίας για τη σελίδα Επικοινωνίας."

4. Το Site Builder θα δημιουργήσει τη φόρμα επικοινωνίας και θα επιστρέψει ένα shortcode έτοιμο για ενσωμάτωση.

## Χρήση του Δημιουργημένου Shortcode {#using-the-generated-shortcode}

Μετά τη δημιουργία της φόρμας επικοινωνίας από το Site Builder, αυτό επιστρέφει ένα shortcode (π.χ., `[contact-form-7 id="123"]`). Μπορείτε να:

1. **Το ενσωματώσετε σε μια σελίδα ή ένα post** — Αντιγράψτε το shortcode και επικολλήστε το σε οποιαδήποτε σελίδα ή post χρησιμοποιώντας τον block editor ή τον classic editor.
2. **Το προσθέσετε μέσω του Site Builder** — Ζητήστε από το Site Builder να προσθέσει τη φόρμα σε μια συγκεκριμένη σελίδα αυτόματα:

   > "Πρόσθεσε τη φόρμα επικοινωνίας στη σελίδα Επικοινωνίας."

3. **Το χρησιμοποιήσετε σε ένα template** — Αν είστε εξοικειωμένοι με το PHP, μπορείτε να προσθέσετε το shortcode σε ένα αρχείο template του theme.

## Προσαρμογή της Φόρμας Επικοινωνίας {#customizing-the-contact-form}

Μετά τη δημιουργία της φόρμας επικοινωνίας από το Site Builder, μπορείτε να την προσαρμόσετε περαιτέρω:

### Μέσω της Διεπαφής Chat {#via-the-chat-interface}

Ζητήστε από το Site Builder να τροποποιήσει τη φόρμα:

> "Επέκτεινε τη φόρμα επικοινωνίας για να προσθέσει ένα πεδίο θέματος και να κάνει το πεδίο μηνύματος υποχρεωτικό."

Το Site Builder θα ενημερώσει τη φόρμα και θα επιστρέψει το ενημερωμένο shortcode.

### Μέσω της Διαchnittήματος Διαχείρισης του Plugin Φόρμας {#via-the-form-plugins-admin-interface}

Μπορείτε επίσης να προσαρμόσετε τη φόρμα απευθείας στις ρυθμίσεις του plugin φόρμας σας:

1. Πηγαίνετε στο **Contact Form 7** (ή το εγκατεστημένο plugin φόρμας σας) στο dashboard του WordPress.
2. Βρείτε τη φόρμα που δημιουργήθηκε από το Site Builder.
3. Επεξεργαστείτε τα πεδία της φόρμας, τους κανόνες επικύρωσης και τις ειδοποιήσεις email όπως χρειάζεται.

## Plugins Φόρμας και Συμβατότητα {#form-plugins-and-compatibility}

Το Site Builder ανιχνεύει αυτόματα ποιο plugin φόρμας είναι εγκατεστημένο στην ιστοσελίδα σας και το χρησιμοποιεί για τη δημιουργία της φόρμας επικοινωνίας. Εάν είναι εγκατεστημένα πολλαπλά plugins φόρμας, το Site Builder δίνει προτεραιότητα σε αυτά με αυτή τη σειρά:

1. Contact Form 7
2. WPForms
3. Fluent Forms
4. Gravity Forms

Εάν κανένα από αυτά τα plugins δεν είναι εγκατεστημένο, το Site Builder θα σας συστήσει να εγκαταστήσετε ένα πριν δημιουργήσετε τη φόρμα επικοινωνίας.

## Ειδοποιήσεις Email {#email-notifications}

Οι φόρμες επικοινωνίας που δημιουργούνται από το Site Builder ρυθμίζονται από προεπιλογή για να αποστέλουν ειδοποιήσεις email στον διαχειριστή της ιστοσελίδας. Μπορείτε να προσαρμόσετε τη διεύθυνση email του παραλήπτη και το μήνυμα ειδοποίησης:

1. Πηγαίνετε στο διαchnittίμα διαχείρισης του plugin φόρμας σας.
2. Βρείτε τη φόρμα που δημιουργήθηκε από το Site Builder.
3. Επεξεργαστείτε τις ρυθμίσεις ειδοποίησης email.

Για λεπτομερείς οδηγίες, ανατρέξτε στην τεκμηρίωση του plugin φόρμας σας:

- [Contact Form 7 Email Notifications](https://contactform7.com/docs/)
- [WPForms Email Notifications](https://wpforms.com/docs/)
- [Fluent Forms Email Notifications](https://fluentforms.com/docs/)
- [Gravity Forms Email Notifications](https://docs.gravityforms.com/)
