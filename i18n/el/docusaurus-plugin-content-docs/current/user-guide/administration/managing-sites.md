# Διαχείριση Ιστοτόπων

Οι ιστότοποι (ή υπο-ιστότοποι) αποτελούν τον πυρήνα της WaaS επιχείρησής σας. Το Ultimate Multisite διαθέτει 3 τύπους ιστοτόπων:

- **Ανήκουν σε πελάτη** — Ιστότοποι που ανήκουν σε συγκεκριμένους πελάτες
- **Πρότυπα Ιστοτόπων** — Προκαθορισμένοι ιστότοποι που οι πελάτες μπορούν να επιλέξουν ως σημείο εκκίνησης
- **Κύριος Ιστότοπος** — Ο κεντρικός ιστότοπος του δικτύου σας

## Προβολή Ιστοτόπων

Μεταβείτε στο **Ultimate Multisite → Sites** για να δείτε όλους τους υπο-ιστότοπους στο δίκτυό σας. Κάθε ιστότοπος έχει ετικέτα ανάλογα με τον τύπο του και μπορείτε να φιλτράρετε ανά Όλοι οι Ιστότοποι, Ανήκουν σε Πελάτη, Πρότυπα ή Σε Αναμονή.

![Sites list page](/img/admin/sites-list.png)

## Προσθήκη Νέου Ιστοτόπου

Κάντε κλικ στο κουμπί **Add Site** για να δημιουργήσετε έναν νέο ιστότοπο. Θα χρειαστεί να συμπληρώσετε:

- **Site Title** — Το όνομα του νέου ιστοτόπου
- **Site URL** — Το domain/διαδρομή για τον ιστότοπο
- **Site Type** — Αν πρόκειται για ιστότοπο πελάτη, πρότυπο ή κανονικό ιστότοπο

Η επιλογή **Copy Site** σας επιτρέπει να δημιουργήσετε έναν νέο ιστότοπο βασισμένο σε υπάρχον πρότυπο. Όταν είναι ενεργοποιημένη, μπορείτε να επιλέξετε ποιο πρότυπο θα χρησιμοποιηθεί ως σημείο εκκίνησης. Βεβαιωθείτε ότι η επιλογή **Copy Media on Duplication** είναι ενεργοποιημένη για να συμπεριληφθούν και τα αρχεία πολυμέσων.

## Διαχείριση Υπάρχοντος Ιστοτόπου

Κάντε κλικ στο **Manage** σε οποιονδήποτε ιστότοπο για να ανοίξετε τη σελίδα **Edit Site**. Εδώ θα βρείτε:

### Βασικές Πληροφορίες

Το όνομα του ιστοτόπου, τον τύπο, το αναγνωριστικό (site ID) και την περιγραφή. Θα δείτε επίσης το συνδεδεμένο domain, τη συσχετισμένη συνδρομή και τον λογαριασμό πελάτη στον οποίο ανήκει ο ιστότοπος.

### Επιλογές Ιστοτόπου

Ρυθμίστε τις δυνατότητες και τα όρια του ιστοτόπου:

- **Όρια επισκέψεων** — Μέγιστος αριθμός επισκέψεων στον ιστότοπο
- **Όρια λογαριασμών χρηστών** — Όρια ανά ρόλο χρήστη
- **Χώρος αποθήκευσης** — Όριο αποθηκευτικού χώρου για τον ιστότοπο
- **Προσαρμοσμένο domain** — Ενεργοποίηση αντιστοίχισης domain για αυτόν τον ιστότοπο
- **Ορατότητα plugins και θεμάτων** — Έλεγχος για το ποια plugins και θέματα είναι ορατά, κρυφά ή προ-ενεργοποιημένα

Από προεπιλογή, οι ιστότοποι ακολουθούν τα όρια που έχουν οριστεί σε επίπεδο συνδρομής. Ο ορισμός ορίων σε επίπεδο ιστοτόπου θα υπερισχύσει των ρυθμίσεων συνδρομής.

### Συσχετίσεις

Κάτω από τις επιλογές ιστοτόπου θα βρείτε πληροφορίες για:

- **Αντιστοιχισμένα domains** που σχετίζονται με τον ιστότοπο
- **Συνδρομή** στην οποία υπάγεται ο ιστότοπος
- **Λογαριασμό πελάτη** που είναι συνδεδεμένος με τον ιστότοπο

### Δεξιά Πλαϊνή Στήλη

Στη δεξιά πλευρά μπορείτε να:

- **Ενεργοποιήσετε/απενεργοποιήσετε τον ιστότοπο** με ένα κουμπί εναλλαγής
- **Αλλάξετε τον τύπο του ιστοτόπου** ή να αναθέσετε εκ νέου την ιδιοκτησία
- **Ορίσετε εικόνα/μικρογραφία ιστοτόπου** (εμφανίζεται στο frontend για τα πρότυπα ιστοτόπων)
- **Διαγράψετε τον ιστότοπο** οριστικά

:::warning
Η διαγραφή ενός ιστοτόπου είναι μη αναστρέψιμη. Ο ιστότοπος και όλο το περιεχόμενό του θα αφαιρεθούν οριστικά.
:::
