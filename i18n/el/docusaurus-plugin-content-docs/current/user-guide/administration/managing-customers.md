# Διαχείριση Πελατών

Η σελίδα **Πελάτες** εμφανίζει τη λίστα των μελών που έχουν εγγραφεί στο δίκτυό σας. Σε αυτή τη σελίδα βλέπετε βασικές πληροφορίες χρηστών, όπως όνομα, διεύθυνση email, τελευταία σύνδεση, καθώς και έναν σύνδεσμο **εναλλαγή σε** που σας επιτρέπει να συνδεθείτε στην ιστοσελίδα του χρήστη ως διαχειριστής.

![Σελίδα λίστας πελατών](/img/admin/customers-list.png)

Κάντε κλικ στο **Διαχείριση** κάτω από την κάρτα ενός πελάτη για να δείτε αναλυτικές πληροφορίες για αυτόν.

Στη σελίδα λεπτομερειών του πελάτη, το μπλοκ **Με μια Ματιά** εμφανίζει το αναγνωριστικό πελάτη, την τελευταία σύνδεση και το συνολικό ποσό που έχει ξοδέψει ο πελάτης.

Το μπλοκ **Επιλογές Πελάτη** σας επιτρέπει να ορίσετε τον πελάτη ως **VIP** (μια ετικέτα για να τον βρίσκετε πιο εύκολα). Η καρτέλα **Στοιχεία Χρέωσης** εμφανίζει πληροφορίες χρέωσης όπως email, διεύθυνση και χώρα.

Το μπλοκ **Πεδία** εμφανίζει το όνομα χρήστη, το email και σας επιτρέπει να αλλάξετε την κατάσταση **Επαλήθευσης Email**.

Κάτω από το μπλοκ Επιλογές Πελάτη θα βρείτε τέσσερα επιπλέον μπλοκ:

- **Συνδρομές** — Οι τρέχουσες συνδρομές στις οποίες είναι εγγεγραμμένος ο πελάτης
- **Πληρωμές** — Το ιστορικό πληρωμών του πελάτη
- **Ιστοσελίδες** — Οι ιστοσελίδες που ανήκουν σε αυτόν τον πελάτη
- **Συμβάντα** — Συμβάντα που σχετίζονται με αυτόν τον πελάτη (επαλήθευση email, αλλαγές χρέωσης κ.λπ.)

Κάντε κλικ στο εικονίδιο βέλους (→) δίπλα σε οποιοδήποτε στοιχείο για να μεταβείτε στη σελίδα λεπτομερειών του. Για παράδειγμα, κάνοντας κλικ στο βέλος δίπλα σε μια ιστοσελίδα μεταφέρεστε στη σελίδα επεξεργασίας της, όπου μπορείτε να τροποποιήσετε τη συσχέτιση συνδρομής, τον τύπο ιστοσελίδας και την κατάσταση ενεργοποίησης.

Το ίδιο ισχύει για τις συνδρομές και τις πληρωμές — κάθε μία συνδέεται με σελίδες λεπτομερειών όπου μπορείτε να επεξεργαστείτε την κατάσταση συνδρομής, τον συσχετισμένο πελάτη, τα προϊόντα, τις ρυθμίσεις της πύλης πληρωμών, ακόμα και να επεξεργαστείτε επιστροφές χρημάτων.

Στη δεξιά πλευρά της σελίδας πελάτη θα βρείτε:

- **Τελευταία Σύνδεση & IP** — Τελευταία σύνδεση, διεύθυνση IP και χώρα
- **Χρονικές Σημάνσεις** — Πότε δημιουργήθηκε και πότε τροποποιήθηκε τελευταία ο λογαριασμός πελάτη
- **Διαγραφή** — Επιλογή για μόνιμη διαγραφή του πελάτη

:::warning
Η διαγραφή ενός πελάτη είναι μόνιμη και δεν μπορεί να αναιρεθεί.
:::
