# Gratis AI Agent Ρυθμίσεις {#gratis-ai-agent-settings}

Η οθόνη **Settings → Advanced** στο Gratis AI Agent παρέχει ρύθμιση επιπέδου διαχειριστή για ενσωματώσεις backend. Αυτή η σελίδα τεκμηριώνει την προώθηση σχολίων, τα κλειδιά παρόχων αναζήτησης, τη ρύθμιση διαχειριζόμενης υπηρεσίας Superdav, τα στοιχεία ελέγχου Google Calendar, τις ρυθμίσεις SMS TextBee και τις σημάνσεις λειτουργιών σε όλο το δίκτυο.

## Πρόσβαση στις ρυθμίσεις {#accessing-settings}

1. Στη διαχείριση WordPress, μεταβείτε στο **Gratis AI Agent → Settings**.
2. Κάντε κλικ στην καρτέλα **Advanced**.

## Ρύθμιση παραμέτρων endpoint σχολίων {#feedback-endpoint-configuration}

Το endpoint σχολίων λαμβάνει αιτήματα POST από τον AI agent κάθε φορά που ένας χρήστης υποβάλλει σχόλια μέσω του κουμπιού αντίχειρα προς τα κάτω, του banner αυτόματης προτροπής ή της εντολής `/report-issue`.

| Πεδίο | Περιγραφή |
|---|---|
| **Feedback Endpoint URL** | Το URL που λαμβάνει υποβολές σχολίων ως αιτήματα HTTP POST με σώμα JSON. |
| **Feedback API Key** | Ένα bearer token που αποστέλλεται στο `Authorization` header κάθε αιτήματος σχολίων. Αφήστε το κενό αν το endpoint σας δεν απαιτεί έλεγχο ταυτότητας. |

### Αναμενόμενο JSON payload {#expected-json-payload}

Το endpoint σχολίων σας πρέπει να αποδέχεται σώμα JSON με τουλάχιστον τα ακόλουθα πεδία:

```json
{
  "message_id": "msg_abc123",
  "conversation_id": "conv_xyz789",
  "feedback_text": "The answer was incorrect about pricing.",
  "triage_category": "factual_error"
}
```

Ενδέχεται να υπάρχουν πρόσθετα πεδία στο payload, ανάλογα με το πλαίσιο της συνομιλίας.

### Τιμές `triage_category` {#triagecategory-values}

Το επίπεδο διαλογής AI εκχωρεί μία από τις ακόλουθες τιμές στο `triage_category` πριν προωθήσει το payload:

| Τιμή | Σημασία |
|---|---|
| `factual_error` | Ο βοηθός παρείχε εσφαλμένες πραγματολογικές πληροφορίες. |
| `unhelpful_answer` | Η απάντηση ήταν τεχνικά σωστή αλλά όχι χρήσιμη. |
| `inappropriate_content` | Η απάντηση περιείχε περιεχόμενο που δεν πρέπει να εμφανίζεται στους χρήστες. |
| `other` | Τα σχόλια δεν ταίριαξαν με γνωστή κατηγορία. |

### Έλεγχος ταυτότητας {#authentication}

Αν το endpoint σας απαιτεί έλεγχο ταυτότητας, ορίστε το πεδίο **Feedback API Key** στο bearer token σας. Ο agent στέλνει:

```
Authorization: Bearer <your-api-key>
```

Αν το πεδίο **Feedback API Key** είναι κενό, δεν αποστέλλεται `Authorization` header.

### Απενεργοποίηση συλλογής σχολίων {#disabling-feedback-collection}

Αφήστε κενά και τα δύο πεδία **Feedback Endpoint URL** και **Feedback API Key**. Το κουμπί αντίχειρα προς τα κάτω και το UI σχολίων παραμένουν ορατά στους χρήστες, αλλά οι υποβολές δεν προωθούνται σε καμία εξωτερική υπηρεσία.

## Brave Search API Key {#brave-search-api-key}

Επίσης στην καρτέλα **Advanced**, το πεδίο **Brave Search API Key** ενεργοποιεί τη δυνατότητα [Αναζήτησης στο διαδίκτυο](../configuration/internet-search).

| Πεδίο | Περιγραφή |
|---|---|
| **Brave Search API Key** | Το API key σας από το dashboard προγραμματιστών Brave Search. Απαιτείται για την ενεργοποίηση αναζήτησης στο διαδίκτυο στον AI assistant. |

Η ετικέτα του πεδίου περιλαμβάνει σύνδεσμο με δυνατότητα κλικ προς τη σελίδα εγγραφής Brave Search API. Αφήστε το κενό για να απενεργοποιήσετε την αναζήτηση στο διαδίκτυο.

Δείτε την [Αναζήτηση στο διαδίκτυο](../configuration/internet-search) για τεκμηρίωση τελικού χρήστη σχετικά με αυτήν τη λειτουργία.

## Διαχειριζόμενη υπηρεσία Superdav {#managed-superdav-service}

Το Superdav AI Agent v1.18.0 προσθέτει διαχειριζόμενα endpoint υπηρεσίας Superdav και αυτόματη παροχή σύνδεσης για υποστηριζόμενους ιστότοπους. Χρησιμοποιήστε αυτά τα στοιχεία ελέγχου όταν ο ιστότοπός σας πρέπει να συνδεθεί στον φιλοξενούμενο πάροχο αντί για ένα μη αυτόματα ρυθμισμένο endpoint υπηρεσίας.

| Πεδίο | Περιγραφή |
|---|---|
| **Managed Superdav Service** | Ενεργοποιεί τη σύνδεση φιλοξενούμενης υπηρεσίας Superdav για υποστηριζόμενους ιστότοπους. |
| **Provision Connection** | Ξεκινά την αυτόματη παροχή endpoint και διαπιστευτηρίων. Χρησιμοποιήστε το αφού επιβεβαιώσετε ότι ο ιστότοπος πρέπει να χρησιμοποιεί τον διαχειριζόμενο πάροχο. |
| **Service Endpoint / Connection Status** | Εμφανίζει το τρέχον endpoint ή την κατάσταση σύνδεσης μετά την παροχή. |

Μετά την παροχή, αποθηκεύστε τις ρυθμίσεις και επαληθεύστε την κατάσταση σύνδεσης πριν βασιστείτε σε ροές εργασίας διαχειριζόμενης υπηρεσίας. Αν η παροχή αποτύχει, ελέγξτε τις εμφανιζόμενες οδηγίες επανάληψης και επιβεβαιώστε ότι ο ιστότοπος έχει άδεια να χρησιμοποιήσει τον φιλοξενούμενο πάροχο.

## Ρύθμιση παραμέτρων Google Calendar {#google-calendar-configuration}

Όταν είναι ενεργοποιημένες οι λειτουργίες ημερολογίου του Superdav AI Agent v1.18.0, ο agent μπορεί να διαβάζει ρυθμισμένα ημερολόγια και λεπτομέρειες συμβάντων. Τα εργαλεία ημερολογίου είναι προσανατολισμένα στην ανάγνωση και είναι χρήσιμα για υπενθυμίσεις με επίγνωση προγράμματος, παρακολούθηση συμμετεχόντων και αντιστοίχιση επαφών.

| Πεδίο | Περιγραφή |
|---|---|
| **Google Calendar Credentials** | Αποθηκεύει τα διαπιστευτήρια ή τη σύνδεση token που απαιτείται για την ανάγνωση δεδομένων ημερολογίου. |
| **Calendar Selection** | Περιορίζει ποια ρυθμισμένα ημερολόγια μπορεί να επιθεωρεί ο agent. |
| **Calendar Connection Status** | Επιβεβαιώνει αν τα τρέχοντα διαπιστευτήρια μπορούν να διαβάζουν ημερολόγια και συμβάντα. |

Περιορίστε τα διαπιστευτήρια ημερολογίου στα ημερολόγια που χρειάζεται ο agent. Επανασυνδέστε ή ανανεώστε τα διαπιστευτήρια αν η κατάσταση υποδεικνύει ληγμένο token.

## Ειδοποιήσεις SMS TextBee {#textbee-sms-notifications}

Το Superdav AI Agent v1.18.0 προσθέτει το TextBee ως πάροχο SMS για ρυθμισμένες ροές εργασίας ειδοποιήσεων. Οι ειδοποιήσεις SMS πρέπει να συνδυάζονται με πύλες ανθρώπινης έγκρισης για ευαίσθητα μηνύματα ή μηνύματα που απευθύνονται σε χρήστες.

| Πεδίο | Περιγραφή |
|---|---|
| **TextBee API Key** | Ελέγχει την ταυτότητα αιτημάτων προς τον πάροχο SMS TextBee. |
| **TextBee Device / Sender** | Επιλέγει τον αποστολέα ή τη συσκευή TextBee που χρησιμοποιείται για εξερχόμενα μηνύματα, όταν απαιτείται από τον πάροχο. |
| **SMS Notifications Enabled** | Επιτρέπει σε εγκεκριμένες ροές εργασίας να στέλνουν ειδοποιήσεις μηνυμάτων κειμένου. Αφήστε το απενεργοποιημένο για να αποτρέψετε αποστολές SMS. |

Στείλτε ένα δοκιμαστικό μήνυμα μόνο σε αριθμό που ανήκει σε διαχειριστή και, στη συνέχεια, επιβεβαιώστε τη συμπεριφορά της πύλης έγκρισης πριν ενεργοποιήσετε προγραμματισμένες υπενθυμίσεις ή υπενθυμίσεις προς συμμετέχοντες.

## Σημάνσεις λειτουργιών {#feature-flags}

Επίσης εισαγόμενη στην v1.9.0, η καρτέλα **Settings → Feature Flags** παρέχει διακόπτες εναλλαγής για προαιρετική λειτουργικότητα. Κάθε σήμανση είναι είτε ενεργοποιημένη είτε απενεργοποιημένη σε όλο το δίκτυο· δεν υπάρχει παράκαμψη ανά ιστότοπο αυτήν τη στιγμή.

### Πρόσβαση στις σημάνσεις λειτουργιών {#accessing-feature-flags}

1. Στη διαχείριση WordPress, μεταβείτε στο **Gratis AI Agent → Settings**.
2. Κάντε κλικ στην καρτέλα **Feature Flags**.

### Σημάνσεις ελέγχου πρόσβασης {#access-control-flags}

| Σημαία | Προεπιλογή | Περιγραφή |
|---|---|---|
| **Περιορισμός σε διαχειριστές** | Ανενεργό | Όταν είναι ενεργό, μόνο χρήστες με τον ρόλο `administrator` μπορούν να ανοίξουν το πλαίσιο συνομιλίας του AI Agent. Όλοι οι άλλοι ρόλοι βλέπουν αντ’ αυτού ένα μήνυμα "Επικοινωνήστε με τον διαχειριστή σας". |
| **Περιορισμός σε διαχειριστές δικτύου** | Ανενεργό | Όταν είναι ενεργό σε ένα multisite δίκτυο, μόνο οι Super Admins μπορούν να χρησιμοποιήσουν τον agent. Οι διαχειριστές μεμονωμένων site αποκλείονται. Υπερισχύει του "Περιορισμός σε διαχειριστές" αν και τα δύο είναι ενεργά. |
| **Να επιτρέπεται πρόσβαση σε συνδρομητές** | Ανενεργό | Όταν είναι ενεργό, χρήστες με τον ρόλο `subscriber` μπορούν να χρησιμοποιούν τη διεπαφή συνομιλίας, αλλά περιορίζονται σε δυνατότητες μόνο για ανάγνωση (χωρίς δημιουργία αναρτήσεων ή αλλαγές ρυθμίσεων). |
| **Απενεργοποίηση για μη μέλη** | Ανενεργό | Ενσωματώνεται με την κατάσταση συνδρομής του Ultimate Multisite. Όταν είναι ενεργό, η συνομιλία αποκρύπτεται για site που δεν έχουν ενεργή συνδρομή. |

### Σημαίες branding {#branding-flags}

| Σημαία | Προεπιλογή | Περιγραφή |
|---|---|---|
| **Απόκρυψη υποσέλιδου "Powered by Gratis AI Agent"** | Ανενεργό | Αφαιρεί τη γραμμή απόδοσης branding που εμφανίζεται στο κάτω μέρος του widget συνομιλίας. Συνιστάται για υλοποιήσεις white-label. |
| **Προσαρμοσμένο όνομα agent** | *(κενό)* | Αντικαθιστά την προεπιλεγμένη ετικέτα "Gratis AI Agent" στην κεφαλίδα συνομιλίας και στο μενού διαχείρισης με το δικό σας όνομα προϊόντος. Αφήστε το κενό για να χρησιμοποιηθεί η προεπιλογή. |
| **Απόκρυψη επιλογέα agent** | Ανενεργό | Όταν είναι ενεργό, οι χρήστες δεν μπορούν να εναλλάσσονται μεταξύ των πέντε ενσωματωμένων agent. Ο τρέχων agent είναι σταθερός σε ό,τι έχει διαμορφωθεί ως προεπιλογή στις Ρυθμίσεις → Γενικά. |
| **Χρήση εικονιδίου site ως avatar συνομιλίας** | Ανενεργό | Αντικαθιστά το προεπιλεγμένο εικονίδιο AI στην κεφαλίδα του widget συνομιλίας με το εικονίδιο site του WordPress (ορίζεται στο Εμφάνιση → Προσαρμογή → Ταυτότητα site). |

### Σημαίες ασφάλειας αυτοματισμών {#automation-safety-flags}

Το Superdav AI Agent v1.18.0 εισάγει πύλες ανθρώπινης έγκρισης και εγγραφές υπενθυμίσεων για ασφαλέστερες εκτελέσεις αυτοματισμών. Αυτά τα στοιχεία ελέγχου μπορεί να εμφανίζονται στις σημαίες λειτουργιών ή στις προηγμένες ρυθμίσεις αυτοματισμών, ανάλογα με το εγκατεστημένο πακέτο.

| Σημαία | Προεπιλογή | Περιγραφή |
|---|---|---|
| **Απαίτηση ανθρώπινης έγκρισης** | Συνιστάται ενεργό | Παύει ευαίσθητους αυτοματισμούς μέχρι ένας εξουσιοδοτημένος χρήστης να ελέγξει και να επιβεβαιώσει την προτεινόμενη ενέργεια. |
| **Αποδιπλοποίηση υπενθυμίσεων** | Ενεργό | Καταγράφει τις απεσταλμένες υπενθυμίσεις, ώστε οι επαναλήψεις ή οι προγραμματισμένες εκτελέσεις να μην στέλνουν διπλές ειδοποιήσεις. |
| **Ενεργοποίηση εργαλείων ημερολογίου** | Ανενεργό μέχρι να διαμορφωθεί | Επιτρέπει στον agent να διαβάζει διαμορφωμένα ημερολόγια και συμβάντα Google. |
| **Ενεργοποίηση ειδοποιήσεων SMS** | Ανενεργό μέχρι να διαμορφωθεί | Επιτρέπει σε εγκεκριμένες ροές εργασίας να στέλνουν ειδοποιήσεις SMS TextBee αφού αποθηκευτούν τα διαπιστευτήρια. |

### Εφαρμογή αλλαγών {#applying-changes}

Κάντε κλικ στο **Αποθήκευση ρυθμίσεων** αφού αλλάξετε οποιαδήποτε σημαία. Οι αλλαγές τίθενται σε ισχύ αμέσως — δεν απαιτείται εκκαθάριση cache ή επανενεργοποίηση plugin.
