# Διαχείριση πληρωμών και τιμολογίων {#managing-payments-and-invoices}

## Ρυθμίσεις πληρωμών {#payment-settings}

Πριν αρχίσετε να λαμβάνετε πληρωμές, θα χρειαστεί να διαμορφώσετε τις ρυθμίσεις που σχετίζονται με τις πληρωμές. Μεταβείτε στο **Ultimate Multisite → Settings** και κάντε κλικ στην καρτέλα **Payment**.

![Καρτέλα ρυθμίσεων πληρωμών](/img/admin/settings-payments-top.png)

Ακολουθεί μια πλήρης προβολή της σελίδας ρυθμίσεων πληρωμών:

![Πλήρης σελίδα ρυθμίσεων πληρωμών](/img/admin/settings-payments-full.png)

### Γενικές επιλογές πληρωμών {#general-payment-options}

Στις γενικές ρυθμίσεις μπορείτε να διαμορφώσετε:

- **Νόμισμα** — Το προεπιλεγμένο νόμισμα που χρησιμοποιείται για συναλλαγές
- **Θέση νομίσματος** — Πού εμφανίζεται το σύμβολο του νομίσματος (πριν/μετά το ποσό)

![Ρυθμίσεις πύλης πληρωμών](/img/admin/settings-payments-options.png)

### Πύλες πληρωμών {#payment-gateways}

Το Ultimate Multisite υποστηρίζει πολλαπλές πύλες πληρωμών. Μπορείτε να ενεργοποιήσετε και να διαμορφώσετε καθεμία από την καρτέλα ρυθμίσεων Payment.

![Διαμόρφωση πύλης πληρωμών](/img/admin/settings-payments-gateways.png)

Οι διαθέσιμες πύλες περιλαμβάνουν:

- **Stripe** — Πληρωμές με πιστωτική κάρτα μέσω Stripe
- **PayPal** — Πληρωμές PayPal
- **Manual** — Για επεξεργασία πληρωμών εκτός σύνδεσης ή προσαρμοσμένη επεξεργασία πληρωμών

Κάθε πύλη έχει τη δική της ενότητα διαμόρφωσης όπου εισάγετε API keys και άλλες ρυθμίσεις.

![Πρόσθετες ρυθμίσεις πύλης](/img/admin/settings-payments-gateways-2.png)

### Λειτουργία Sandbox {#sandbox-mode}

Μπορείτε να ενεργοποιήσετε τη **Λειτουργία Sandbox** για να δοκιμάσετε την ενσωμάτωση πληρωμών σας πριν τη δημοσιεύσετε. Όταν η λειτουργία sandbox είναι ενεργή, δεν θα πραγματοποιηθούν πραγματικές χρεώσεις.

## Προβολή πληρωμών {#viewing-payments}

Μεταβείτε στη σελίδα **Payments** κάτω από το Ultimate Multisite για να δείτε όλες τις συναλλαγές σε όλο το δίκτυό σας.

![Λίστα πληρωμών](/img/admin/payments-list.png)

Μπορείτε να φιλτράρετε πληρωμές κατά κατάσταση (ολοκληρωμένη, σε εκκρεμότητα, αποτυχημένη, επιστραφείσα) και να αναζητήσετε συγκεκριμένες συναλλαγές.

Κάντε κλικ σε μια πληρωμή για να δείτε τα πλήρη στοιχεία της, συμπεριλαμβανομένων των στοιχείων γραμμής, της σχετικής συνδρομής, των πληροφοριών πελάτη και των δεδομένων της πύλης πληρωμών.

## Τιμολόγια {#invoices}

Το Ultimate Multisite μπορεί να δημιουργεί αυτόματα τιμολόγια για πληρωμές. Μπορείτε να προσαρμόσετε το πρότυπο τιμολογίου και τη μορφή αρίθμησης από τις ρυθμίσεις Payment.

Οι επιλογές προσαρμογής τιμολογίων περιλαμβάνουν:

- **Όνομα και διεύθυνση εταιρείας** που εμφανίζονται στα τιμολόγια
- **Αρίθμηση τιμολογίων** μορφή και ακολουθία
- **Λογότυπο** που εμφανίζεται στην κεφαλίδα του τιμολογίου
- **Προσαρμοσμένο κείμενο υποσέλιδου** για όρους, σημειώσεις ή νομικές πληροφορίες

Για να προσαρμόσετε το πρότυπο τιμολογίου, μεταβείτε στο **Ultimate Multisite → Settings → Payment** και αναζητήστε τις ρυθμίσεις που σχετίζονται με τα τιμολόγια.
