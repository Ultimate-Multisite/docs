# Διαχείριση Πληρωμών και Τιμολογίων

## Ρυθμίσεις Πληρωμών

Πριν ξεκινήσετε να λαμβάνετε πληρωμές, θα χρειαστεί να διαμορφώσετε τις ρυθμίσεις που σχετίζονται με τις πληρωμές. Μεταβείτε στο **Ultimate Multisite → Ρυθμίσεις** και κάντε κλικ στην καρτέλα **Πληρωμές**.

![Καρτέλα ρυθμίσεων πληρωμών](/img/admin/settings-payments-top.png)

### Γενικές Επιλογές Πληρωμών

Στις γενικές ρυθμίσεις μπορείτε να διαμορφώσετε:

- **Νόμισμα** — Το προεπιλεγμένο νόμισμα για τις συναλλαγές
- **Θέση Νομίσματος** — Πού εμφανίζεται το σύμβολο του νομίσματος (πριν/μετά το ποσό)

![Ρυθμίσεις πύλης πληρωμών](/img/admin/settings-payments-options.png)

### Πύλες Πληρωμών

Το Ultimate Multisite υποστηρίζει πολλαπλές πύλες πληρωμών. Μπορείτε να ενεργοποιήσετε και να διαμορφώσετε την καθεμία από την καρτέλα ρυθμίσεων Πληρωμών.

![Διαμόρφωση πύλης πληρωμών](/img/admin/settings-payments-gateways.png)

Οι διαθέσιμες πύλες περιλαμβάνουν:

- **Stripe** — Πληρωμές με πιστωτική κάρτα μέσω Stripe
- **PayPal** — Πληρωμές PayPal
- **Χειροκίνητη** — Για offline ή προσαρμοσμένη επεξεργασία πληρωμών

Κάθε πύλη έχει τη δική της ενότητα διαμόρφωσης όπου εισάγετε τα API keys και άλλες ρυθμίσεις.

![Πρόσθετες ρυθμίσεις πύλης](/img/admin/settings-payments-gateways-2.png)

### Λειτουργία Sandbox

Μπορείτε να ενεργοποιήσετε τη **Λειτουργία Sandbox** για να δοκιμάσετε την ενσωμάτωση πληρωμών πριν ξεκινήσετε κανονικά. Όταν είναι ενεργή η λειτουργία sandbox, δεν θα γίνονται πραγματικές χρεώσεις.

## Προβολή Πληρωμών

Μεταβείτε στη σελίδα **Πληρωμές** κάτω από το Ultimate Multisite για να δείτε όλες τις συναλλαγές σε ολόκληρο το δίκτυό σας.

![Λίστα πληρωμών](/img/admin/payments-list.png)

Μπορείτε να φιλτράρετε τις πληρωμές ανά κατάσταση (ολοκληρωμένες, σε εκκρεμότητα, αποτυχημένες, επιστροφές) και να αναζητήσετε συγκεκριμένες συναλλαγές.

Κάντε κλικ σε μια πληρωμή για να δείτε τις πλήρεις λεπτομέρειές της, συμπεριλαμβανομένων των στοιχείων γραμμής, της συσχετισμένης συνδρομής, των πληροφοριών πελάτη και των δεδομένων της πύλης πληρωμών.

## Τιμολόγια

Το Ultimate Multisite μπορεί να δημιουργεί αυτόματα τιμολόγια για τις πληρωμές. Μπορείτε να προσαρμόσετε το πρότυπο τιμολογίου και τη μορφή αρίθμησης από τις ρυθμίσεις Πληρωμών.

Οι επιλογές προσαρμογής τιμολογίου περιλαμβάνουν:

- **Επωνυμία και διεύθυνση εταιρείας** που εμφανίζονται στα τιμολόγια
- **Μορφή αρίθμησης τιμολογίων** και ακολουθία
- **Λογότυπο** που εμφανίζεται στην κεφαλίδα του τιμολογίου
- **Προσαρμοσμένο κείμενο υποσέλιδου** για όρους, σημειώσεις ή νομικές πληροφορίες

Για να προσαρμόσετε το πρότυπο τιμολογίου, μεταβείτε στο **Ultimate Multisite → Ρυθμίσεις → Πληρωμές** και αναζητήστε τις ρυθμίσεις που σχετίζονται με τα τιμολόγια.
