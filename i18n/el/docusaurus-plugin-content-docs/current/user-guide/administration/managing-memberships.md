# Διαχείριση Συνδρομών

Στο Ultimate Multisite μπορείτε να διαχειρίζεστε τις συνδρομές και να έχετε πρόσβαση σε όλες τις συνδρομές του δικτύου σας. Ελέγξτε και επεξεργαστείτε τα στοιχεία της συνδρομής, όπως πλάνα και προϊόντα, τον αριθμό ιστοτόπων, το ποσό και τη συχνότητα χρέωσης, το ιστορικό πληρωμών και τις χρονικές σημάνσεις.

Για να αποκτήσετε πρόσβαση σε αυτή τη σελίδα, μεταβείτε στην καρτέλα **Memberships** στο μενού του Ultimate Multisite.

![Σελίδα λίστας συνδρομών](/img/admin/memberships-list.png)

Στη σελίδα Memberships μπορείτε να δείτε μια λίστα με τους πελάτες σας και τα προϊόντα που σχετίζονται με τους λογαριασμούς τους. Μπορείτε να φιλτράρετε τις συνδρομές ανά κατάσταση: ενεργές, σε αναμονή, σε παύση, ληγμένες ή ακυρωμένες.

Κάντε κλικ σε μια συνδρομή για να αποκτήσετε πρόσβαση στη σελίδα όπου μπορείτε να επεξεργαστείτε τα στοιχεία της συνδρομής — πληροφορίες χρέωσης, όρια ιστοτόπων ή επισκέψεων και άλλα.

Μπορείτε επίσης να προσθέσετε νέες συνδρομές που συνδέονται με χρήστες που έχουν ήδη εγγραφεί στο δίκτυό σας. Κάντε κλικ στο **Add Membership** και διαμορφώστε τα στοιχεία.

![Παράθυρο προσθήκης συνδρομής](/img/admin/membership-add-modal.png)

Στη συνέχεια κάντε κλικ στο **Create Membership**. Η νέα συνδρομή θα εμφανιστεί στη λίστα συνδρομών.
