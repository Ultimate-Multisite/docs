# Διαχείριση συνδρομών μέλους {#managing-memberships}

Στο Ultimate Multisite μπορείτε να διαχειρίζεστε συνδρομές μέλους και να έχετε πρόσβαση σε όλες τις συνδρομές στο δίκτυό σας. Ελέγξτε και επεξεργαστείτε λεπτομέρειες συνδρομής, όπως προγράμματα και προϊόντα, τον αριθμό των ιστότοπων, το ποσό και τη συχνότητα χρέωσης, το ιστορικό πληρωμών και τις χρονικές σημάνσεις.

Για να αποκτήσετε πρόσβαση σε αυτήν τη σελίδα, μεταβείτε στην καρτέλα **Συνδρομές μέλους** στο μενού Ultimate Multisite.

![Σελίδα λίστας συνδρομών μέλους](/img/admin/memberships-list.png)

Στη σελίδα Συνδρομές μέλους μπορείτε να δείτε μια λίστα με τους πελάτες σας και τα προϊόντα που σχετίζονται με τους λογαριασμούς τους. Μπορείτε να φιλτράρετε τις συνδρομές μέλους κατά κατάσταση: ενεργή, σε εκκρεμότητα, σε αναμονή, ληγμένη ή ακυρωμένη.

Οι δωρεάν συνδρομές μέλους αντιμετωπίζονται ως **ισόβιες** συνδρομές μέλους. Δεν λαμβάνουν ημερομηνία λήξης και δεν θα λήξουν αυτόματα. Αν ένα δωρεάν πρόγραμμα πρέπει να τελειώνει μετά από μια δοκιμαστική ή καθορισμένη περίοδο, χρησιμοποιήστε ένα επί πληρωμή προϊόν με δοκιμαστική περίοδο, ακυρώστε χειροκίνητα τη συνδρομή μέλους ή μετακινήστε τον πελάτη σε άλλο πρόγραμμα όταν θα πρέπει να αλλάξει η πρόσβασή του.

Κάντε κλικ σε μια συνδρομή μέλους για να αποκτήσετε πρόσβαση στη σελίδα όπου μπορείτε να επεξεργαστείτε τις λεπτομέρειες της συνδρομής μέλους — στοιχεία χρέωσης, όρια ιστότοπων ή επισκέψεων και άλλα.

![Διεπαφή επεξεργασίας συνδρομής μέλους](/img/admin/membership-edit.png)

Ακολουθεί πλήρης προβολή της σελίδας επεξεργασίας συνδρομής μέλους:

![Πλήρης διεπαφή επεξεργασίας συνδρομής μέλους](/img/admin/membership-edit-full.png)

Μπορείτε επίσης να προσθέσετε νέες συνδρομές μέλους που σχετίζονται με χρήστες που είχαν εγγραφεί προηγουμένως στο δίκτυό σας. Κάντε κλικ στο **Προσθήκη συνδρομής μέλους** και διαμορφώστε τις λεπτομέρειες.

![Παράθυρο προσθήκης συνδρομής μέλους](/img/admin/membership-add-modal.png)

Στη συνέχεια, κάντε κλικ στο **Δημιουργία συνδρομής μέλους**. Η νέα συνδρομή μέλους θα εμφανιστεί στη λίστα συνδρομών μέλους.
