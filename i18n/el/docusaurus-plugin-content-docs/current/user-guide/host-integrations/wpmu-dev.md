# Ενσωμάτωση WPMU DEV

## Επισκόπηση
Το WPMU DEV είναι μια ολοκληρωμένη πλατφόρμα WordPress που προσφέρει φιλοξενία, plugins και υπηρεσίες για ιστοσελίδες WordPress. Αυτή η ενσωμάτωση επιτρέπει τον αυτόματο συγχρονισμό domains και τη διαχείριση πιστοποιητικών SSL μεταξύ του Ultimate Multisite και της φιλοξενίας WPMU DEV.

## Δυνατότητες
- Αυτόματος συγχρονισμός domains
- Διαχείριση πιστοποιητικών SSL
- Εκτεταμένες προσπάθειες επαλήθευσης πιστοποιητικών SSL

## Απαιτήσεις
Η ενσωμάτωση ανιχνεύει αυτόματα αν φιλοξενείστε στο WPMU DEV και χρησιμοποιεί το ενσωματωμένο API. Δεν απαιτείται επιπλέον ρύθμιση αν φιλοξενείστε στο WPMU DEV.

Η ενσωμάτωση ελέγχει την ύπαρξη της σταθεράς `WPMUDEV_HOSTING_SITE_ID`, η οποία ορίζεται αυτόματα όταν φιλοξενείστε στο WPMU DEV.

## Οδηγίες Ρύθμισης

### 1. Επαλήθευση Φιλοξενίας WPMU DEV

Αν φιλοξενείστε στο WPMU DEV, οι απαραίτητες σταθερές θα πρέπει να είναι ήδη ορισμένες. Βεβαιωθείτε ότι:

1. Η σταθερά `WPMUDEV_HOSTING_SITE_ID` είναι ορισμένη στο περιβάλλον σας
2. Έχετε ενεργή συνδρομή WPMU DEV με πρόσβαση στο API

### 2. Ενεργοποίηση της Ενσωμάτωσης

1. Στη διαχείριση του WordPress, μεταβείτε στο Ultimate Multisite > Settings
2. Πλοηγηθείτε στην καρτέλα "Domain Mapping"
3. Κατεβείτε στην ενότητα "Host Integrations"
4. Ενεργοποιήστε την ενσωμάτωση WPMU DEV
5. Κάντε κλικ στο "Save Changes"

## Πώς Λειτουργεί

### Συγχρονισμός Domains

Όταν ένα domain αντιστοιχίζεται στο Ultimate Multisite:

1. Η ενσωμάτωση χρησιμοποιεί το API του WPMU DEV για να προσθέσει το domain στον λογαριασμό φιλοξενίας σας
2. Προσθέτει επίσης αυτόματα την έκδοση www του domain
3. Το WPMU DEV αναλαμβάνει τη ρύθμιση του domain και την έκδοση του πιστοποιητικού SSL

### Διαχείριση Πιστοποιητικών SSL

Η ενσωμάτωση είναι ρυθμισμένη να αυξάνει τον αριθμό των προσπαθειών επαλήθευσης πιστοποιητικών SSL για τη φιλοξενία WPMU DEV, καθώς μπορεί να χρειαστεί κάποιος χρόνος για την έκδοση και εγκατάσταση των πιστοποιητικών SSL. Από προεπιλογή, θα δοκιμάσει έως 10 φορές για την επαλήθευση του πιστοποιητικού SSL, σε σύγκριση με τις συνήθεις 5 προσπάθειες.

## Σημαντικές Σημειώσεις

### Αφαίρεση Domain

Προς το παρόν, το API του WPMU DEV δεν παρέχει τρόπο αφαίρεσης domains. Όταν αφαιρείται μια αντιστοίχιση domain στο Ultimate Multisite, το domain θα παραμείνει στον λογαριασμό φιλοξενίας WPMU DEV. Θα χρειαστεί να το αφαιρέσετε χειροκίνητα από τον πίνακα ελέγχου φιλοξενίας WPMU DEV αν είναι απαραίτητο.

### Ταυτοποίηση API

Η ενσωμάτωση χρησιμοποιεί το κλειδί API του WPMU DEV που είναι αποθηκευμένο στη βάση δεδομένων του WordPress ως επιλογή `wpmudev_apikey`. Αυτό ρυθμίζεται αυτόματα όταν συνδέετε την ιστοσελίδα σας στο WPMU DEV.

## Αντιμετώπιση Προβλημάτων

### Προβλήματα Σύνδεσης API
- Βεβαιωθείτε ότι η ιστοσελίδα σας είναι σωστά συνδεδεμένη στο WPMU DEV
- Ελέγξτε ότι η επιλογή `wpmudev_apikey` είναι ορισμένη στη βάση δεδομένων του WordPress
- Βεβαιωθείτε ότι η συνδρομή σας στο WPMU DEV είναι ενεργή

### Προβλήματα Πιστοποιητικών SSL
- Το WPMU DEV μπορεί να χρειαστεί κάποιο χρόνο για την έκδοση πιστοποιητικών SSL (συνήθως 5-15 λεπτά)
- Η ενσωμάτωση είναι ρυθμισμένη να ελέγχει έως 10 φορές για πιστοποιητικά SSL
- Αν τα πιστοποιητικά SSL εξακολουθούν να μην εκδίδονται μετά από πολλές προσπάθειες, επικοινωνήστε με την υποστήριξη του WPMU DEV

### Το Domain Δεν Προστέθηκε
- Ελέγξτε τα αρχεία καταγραφής του Ultimate Multisite για τυχόν μηνύματα σφάλματος
- Βεβαιωθείτε ότι το domain δεν έχει ήδη προστεθεί στο WPMU DEV
- Βεβαιωθείτε ότι το πακέτο φιλοξενίας WPMU DEV υποστηρίζει τον αριθμό των domains που προσθέτετε
