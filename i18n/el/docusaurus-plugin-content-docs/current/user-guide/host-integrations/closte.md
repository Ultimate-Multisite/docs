# Ενσωμάτωση Closte

## Επισκόπηση
Το Closte είναι μια πλατφόρμα διαχειριζόμενης φιλοξενίας WordPress, χτισμένη πάνω στην υποδομή του Google Cloud. Αυτή η ενσωμάτωση επιτρέπει τον αυτόματο συγχρονισμό domains και τη διαχείριση πιστοποιητικών SSL μεταξύ του Ultimate Multisite και του Closte.

## Χαρακτηριστικά
- Αυτόματος συγχρονισμός domains
- Διαχείριση πιστοποιητικών SSL
- Υποστήριξη wildcard domains
- Δεν απαιτείται ρύθμιση αν τρέχετε στο Closte

## Απαιτήσεις
Η παρακάτω σταθερά πρέπει να οριστεί στο αρχείο `wp-config.php` αν χρησιμοποιείτε το Closte:

```php
define('CLOSTE_CLIENT_API_KEY', 'your_api_key');
```

Αυτή η σταθερά συνήθως είναι ήδη ορισμένη αν φιλοξενείστε στο Closte.

## Οδηγίες Εγκατάστασης

### 1. Επαληθεύστε το API Key του Closte

Αν φιλοξενείστε στο Closte, η σταθερά `CLOSTE_CLIENT_API_KEY` θα πρέπει να είναι ήδη ορισμένη στο αρχείο `wp-config.php`. Μπορείτε να το επαληθεύσετε ελέγχοντας το αρχείο `wp-config.php`.

### 2. Ενεργοποιήστε την Ενσωμάτωση

1. Στη διαχείριση του WordPress, μεταβείτε στο Ultimate Multisite > Settings
2. Πλοηγηθείτε στην καρτέλα "Domain Mapping"
3. Κατεβείτε στην ενότητα "Host Integrations"
4. Ενεργοποιήστε την ενσωμάτωση Closte
5. Κάντε κλικ στο "Save Changes"

## Πώς Λειτουργεί

Όταν γίνεται αντιστοίχιση ενός domain στο Ultimate Multisite:

1. Η ενσωμάτωση στέλνει ένα αίτημα στο API του Closte για να προσθέσει το domain στην εφαρμογή σας
2. Το Closte χειρίζεται αυτόματα την έκδοση πιστοποιητικού SSL
3. Όταν αφαιρείται μια αντιστοίχιση domain, η ενσωμάτωση αφαιρεί το domain από το Closte

Η ενσωμάτωση λειτουργεί επίσης με τη ρύθμιση χρονικού διαστήματος ελέγχου DNS στο Ultimate Multisite, επιτρέποντάς σας να ρυθμίσετε πόσο συχνά το σύστημα ελέγχει για διάδοση DNS και έκδοση πιστοποιητικού SSL.

## Δημιουργία Εγγραφής Domain

Αυτή η ενσωμάτωση διασφαλίζει ότι όταν δημιουργείται ή αντιγράφεται ένας ιστότοπος, δημιουργείται αυτόματα μια εγγραφή domain. Αυτό είναι ιδιαίτερα σημαντικό για την ενσωμάτωση με το Closte, καθώς η δημιουργία εγγραφής domain ενεργοποιεί το API του Closte για να δημιουργήσει το domain και το πιστοποιητικό SSL.

## Αντιμετώπιση Προβλημάτων

### Προβλήματα Σύνδεσης API
- Βεβαιωθείτε ότι το API key του Closte είναι σωστό
- Ελέγξτε ότι ο λογαριασμός σας στο Closte έχει τα απαραίτητα δικαιώματα

### Προβλήματα Πιστοποιητικού SSL
- Το Closte μπορεί να χρειαστεί λίγο χρόνο για να εκδώσει πιστοποιητικά SSL (συνήθως 5-10 λεπτά)
- Βεβαιωθείτε ότι τα domains σας δείχνουν σωστά στη διεύθυνση IP του server σας στο Closte
- Ελέγξτε τις εγγραφές DNS του domain σας για να βεβαιωθείτε ότι είναι σωστά ρυθμισμένες

### Το Domain Δεν Προστέθηκε
- Ελέγξτε τα logs του Ultimate Multisite για τυχόν μηνύματα σφάλματος
- Βεβαιωθείτε ότι το domain δεν έχει ήδη προστεθεί στο Closte
- Ελέγξτε ότι οι εγγραφές DNS του domain σας είναι σωστά ρυθμισμένες

### Χρονικό Διάστημα Ελέγχου DNS
- Αν τα πιστοποιητικά SSL αργούν να εκδοθούν, μπορείτε να προσαρμόσετε το χρονικό διάστημα ελέγχου DNS στις ρυθμίσεις Domain Mapping
- Το προεπιλεγμένο διάστημα είναι 300 δευτερόλεπτα (5 λεπτά), αλλά μπορείτε να το ορίσετε μέχρι και 10 δευτερόλεπτα για ταχύτερο έλεγχο κατά τη διάρκεια δοκιμών
