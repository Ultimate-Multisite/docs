# Hooks και Filters του Ultimate Multisite

Το Ultimate Multisite αποτελείται από ένα μεγάλο σύνολο συναρτήσεων και ερωτημάτων βάσης δεδομένων. Διαθέτει hooks και filters που σας επιτρέπουν να τροποποιήσετε ή να προσθέσετε νέο κώδικα σε υπάρχουσες συναρτήσεις.

Δημιουργήσαμε μια λίστα με αυτά τα hooks και filters ως αναφορά, για να βοηθήσουμε τους προχωρημένους χρήστες να προσαρμόσουν περαιτέρω το δίκτυό τους. Μπορείτε να αποκτήσετε πρόσβαση στη λίστα από το Github κάνοντας κλικ στον παρακάτω σύνδεσμο.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
