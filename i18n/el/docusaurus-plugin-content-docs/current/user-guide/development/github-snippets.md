# Πώς να χρησιμοποιήσετε τα snippets του Ultimate Multisite από το αποθετήριο GitHub

Υπάρχουν διαθέσιμα code snippets στο αποθετήριο GitHub που ζητούνται συχνά από χρήστες του Ultimate Multisite που θέλουν να προσθέσουν μικρές λειτουργίες, όπως την προσθήκη script του Google Analytics στις σελίδες εγγραφής ή την απόκρυψη ενός meta box από τον πίνακα διαχείρισης.

Αυτό το άρθρο θα σας δείξει πώς να χρησιμοποιήσετε αυτούς τους κώδικες και πιο συγκεκριμένα πού να τους τοποθετήσετε.

Μπορείτε να βρείτε τα snippets στον παρακάτω σύνδεσμο.

https://github.com/next-press/wp-ultimo-snippets/

Υπάρχουν 2 τρόποι για να προσθέσετε τον κώδικα

  1. Στο αρχείο functions.php του θέματός σας.

  2. Must-Use Plugins (mu-plugins)

# Πώς να προσθέσετε το snippet στο αρχείο functions.php του θέματός σας

  1. Συνδεθείτε στον πίνακα διαχείρισης του WordPress Network και μεταβείτε στο Θέματα > Επεξεργαστής Θέματος (Δείτε το στιγμιότυπο οθόνης παρακάτω).

  2. Στη σελίδα "Επεξεργασία Θεμάτων", βεβαιωθείτε ότι έχετε επιλέξει το ενεργό θέμα σας στο αναπτυσσόμενο πεδίο που βρίσκεται στην επάνω δεξιά πλευρά της οθόνης σας (#3 στο στιγμιότυπο οθόνης παρακάτω).

  3. Κάντε κλικ στο αρχείο functions.php κάτω από την ενότητα "Αρχεία Θέματος" για να φορτώσετε το αρχείο. Μετακινηθείτε προς τα κάτω και επικολλήστε το snippet του Ultimate Multisite που πήρατε από το αποθετήριο GitHub.

# Πώς να δημιουργήσετε Must-Use Plugins (mu-plugins)

Το WordPress διαθέτει μια λειτουργία που σας επιτρέπει να φορτώνετε προσαρμοσμένη λειτουργικότητα που ονομάζεται "Must-Use Plugins", ή "mu-plugins" για συντομία.

Αυτά τα ειδικά mu-plugins φορτώνονται πριν από όλα τα άλλα κανονικά plugins και δεν μπορούν να απενεργοποιηθούν. Σε ένα multisite network, ο κώδικας σε αυτά τα mu-plugins θα φορτωθεί σε όλους τους ιστότοπους της εγκατάστασής σας.

1\. Χρησιμοποιήστε FTP ή SSH για να αποκτήσετε πρόσβαση στο σύστημα αρχείων της εγκατάστασης WordPress σας.

2\. Μέσα στον κατάλογο wp-content της εγκατάστασης WordPress σας, δημιουργήστε έναν νέο κατάλογο με όνομα: mu-plugins.

<!-- Screenshot unavailable: File manager showing wp-content directory with mu-plugins folder -->

3\. Δημιουργήστε ένα νέο αρχείο PHP στον υπολογιστή σας με όνομα wu-snippet.php χρησιμοποιώντας το Notepad ή οποιοδήποτε πρόγραμμα επεξεργασίας κώδικα.

4\. Τοποθετήστε το code snippet του Ultimate Multisite που πήρατε από το αποθετήριο GitHub μέσα στο αρχείο και αποθηκεύστε το. Μπορείτε επίσης να προσθέσετε αυτόν τον κώδικα στην κορυφή του snippet για να επισημάνετε το mu plugin σας.
