# Γνωστές ασυμβατότητες με άλλα Plugins

Μετά την εγκατάσταση και ενεργοποίηση του Ultimate Multisite, ενδέχεται να αντιμετωπίσετε προβλήματα λόγω ασυμβατοτήτων με άλλα plugins που δεν έχουν ακόμη επιλυθεί.

Εργαζόμαστε ενεργά για να μειώσουμε αυτή τη λίστα με κάθε νέα έκδοση. Ωστόσο, αν κάποιο plugin αναφέρεται εδώ, ίσως χρειαστεί να το απενεργοποιήσετε για να έχετε ένα πλήρως λειτουργικό Ultimate Multisite Network.

Ακολουθεί η λίστα με αυτά τα plugins και τι πρέπει να κάνετε σε κάθε περίπτωση:

**Ορισμένα plugins της WPMU DEV** Κάποια plugins που _δεν_ συντηρούνται πλέον από την WPMU DEV είναι γνωστό ότι προκαλούν διακοπή της ροής εγγραφής του Ultimate Multisite στο τελευταίο βήμα (ο χρήστης δεν μπορεί να δημιουργήσει λογαριασμό). Ακολουθεί η λίστα με τα plugins που προκαλούν αυτά τα προβλήματα:

  * Signup Language: _Το Ultimate Multisite διαθέτει ένα add-on που προσφέρει την ίδια λειτουργικότητα, το Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Αυτή η δυνατότητα είναι ενσωματωμένη στο Ultimate Multisite, οπότε δεν χρειάζεται επιπλέον plugin._

  * New Blog Templates: _Αυτή η δυνατότητα είναι ενσωματωμένη στο Ultimate Multisite, οπότε δεν χρειάζεται επιπλέον plugin._

  * Set Password on Multisite Blog Creation: _Αυτή η δυνατότητα είναι ενσωματωμένη στο Ultimate Multisite, οπότε δεν χρειάζεται επιπλέον plugin._

  * Signup TOS: _Αυτή η δυνατότητα είναι ενσωματωμένη στο Ultimate Multisite, οπότε δεν χρειάζεται επιπλέον plugin._

  * Anti-Splog: Αν και με λιγότερες δυνατότητες, αυτή η λειτουργία είναι επίσης _ενσωματωμένη στο Ultimate Multisite._

**Σημείωση** : Όλα τα υπόλοιπα plugins της WPMU Dev μπορούν να χρησιμοποιηθούν κανονικά μαζί με το Ultimate Multisite. Παραδείγματα περιλαμβάνουν τα _Smush_ , _Forminator_ , _Defender,_ κ.ά.
