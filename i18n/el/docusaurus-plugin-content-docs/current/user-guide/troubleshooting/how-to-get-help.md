# Πώς να λάβετε βοήθεια

Έχετε το έγκυρο κλειδί άδειας χρήσης σας, και μαζί με αυτό έχετε πρόσβαση στην ομάδα μας για να σας βοηθήσει να λύσετε τυχόν προβλήματα που μπορεί να αντιμετωπίσετε κατά τη ρύθμιση του Ultimate Multisite.

Αυτό το μάθημα καλύπτει τα διαθέσιμα κανάλια υποστήριξης για τους κατόχους άδειας και τι είδους πληροφορίες θα χρειαστεί να έχετε στα χέρια σας για να επικοινωνήσετε με την ομάδα υποστήριξης.

## Ιστότοπος Βοήθειας

Μπορείτε να έχετε πρόσβαση σε διάφορα άρθρα και οδηγούς από τον ιστότοπο βοήθειας, τον οποίο θα βρείτε στον παρακάτω σύνδεσμο.

[)

Ενθαρρύνουμε τους χρήστες μας να εξοικειωθούν με τις βασικές λειτουργίες του Ultimate Multisite. Υπάρχουν άρθρα "Ξεκινώντας" διαθέσιμα στον ιστότοπο βοήθειας, καθώς και απαντήσεις σε συχνές ερωτήσεις, όπως πώς να εγκαταστήσετε πρόσθετα, πώς να ενσωματώσετε πύλες πληρωμών, κ.λπ.

## Επικοινωνία με την Υποστήριξη

Αν κολλήσετε και δεν μπορείτε να βρείτε την απάντηση στον ιστότοπο βοήθειας, μπορείτε να επικοινωνήσετε με την ομάδα υποστήριξης είτε κάνοντας κλικ στο εικονίδιο με το συννεφάκι ομιλίας που βρίσκεται στην κάτω δεξιά πλευρά του ιστότοπού μας, ΕΊΤΕ στέλνοντας email απευθείας στο support@ultimatemultisite.com.

Όταν επικοινωνείτε με την υποστήριξη για τεχνικά ζητήματα, παρακαλούμε βεβαιωθείτε ότι παρέχετε μια λεπτομερή περιγραφή του προβλήματός σας μαζί με στιγμιότυπο οθόνης, μηνύματα σφάλματος ή οποιαδήποτε πληροφορία που θα βοηθήσει την ομάδα υποστήριξης να κατανοήσει και να αξιολογήσει εύκολα το πρόβλημά σας. Θα χρειαστεί επίσης να παρέχετε το κλειδί άδειας χρήσης σας. Μόνο οι ενεργοί κάτοχοι άδειας δικαιούνται υποστήριξη.

## Ιστότοπος Κοινότητας

Ο [ιστότοπος κοινότητας](https://community.ultimatemultisite.com/) είναι ένας χώρος όπου οι χρήστες του Ultimate Multisite μπορούν να αλληλεπιδρούν μεταξύ τους. Λειτουργεί ως σύστημα πληροφοριών όπου τα μέλη μπορούν να δημοσιεύουν, να σχολιάζουν σε συζητήσεις, να δίνουν συμβουλές, να συνεργάζονται ή να ανταλλάσσουν ιδέες. Θα σας ζητηθεί να παρέχετε το κλειδί άδειας Ultimate Multisite για να ξεκλειδώσετε ιδιωτικούς χώρους/σελίδες.
