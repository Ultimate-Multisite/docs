# Διαχείριση φόρων

Το Ultimate Multisite διαθέτει ενσωματωμένη μονάδα είσπραξης φόρων στο βασικό μας plugin, οπότε αν χρειάζεται να εισπράττετε φόρους πωλήσεων στα πλάνα, τα πακέτα και τις υπηρεσίες σας, μπορείτε να το κάνετε εύκολα χωρίς να καταφύγετε στην εγκατάσταση πρόσθετων.

Για εταιρείες που βρίσκονται στην Ευρώπη, προσφέρουμε ένα **πρόσθετο** που προσθέτει εργαλεία και λειτουργίες για καλύτερη **υποστήριξη συμμόρφωσης με τον ΦΠΑ**.

Το Ultimate Multisite δεν υποβάλλει ούτε αποδίδει φόρους εκ μέρους σας στην κυβέρνηση· απλώς σας βοηθάμε να εισπράξετε τους κατάλληλους φόρους κατά τη στιγμή της συναλλαγής. **Θα εξακολουθείτε να χρειάζεται να αποδίδετε τους φόρους μόνοι σας.**

## Ενεργοποίηση είσπραξης φόρων {#enabling-tax-collection}

Η είσπραξη φόρων δεν είναι ενεργοποιημένη από προεπιλογή. Για να την ενεργοποιήσετε, πρέπει να μεταβείτε στο **Ultimate Multisite > Ρυθμίσεις > Φόροι** και να ενεργοποιήσετε τη ρύθμιση Ενεργοποίηση φόρων.

![Διακόπτης Ενεργοποίησης φόρων στο επάνω μέρος της σελίδας ρυθμίσεων Φόρων](/img/config/settings-taxes-enable.png)

Ακολουθεί μια πλήρης προβολή της σελίδας ρυθμίσεων φόρων:

![Πλήρης σελίδα ρυθμίσεων φόρων](/img/config/settings-taxes-full.png)

Μπορείτε επίσης να δείτε τις ρυθμίσεις φόρων για μεμονωμένα προϊόντα:

![Ρυθμίσεις φόρων για προϊόντα](/img/config/settings-taxes.png)

### Φόρος εκτός τιμής έναντι φόρου εντός τιμής {#tax-excluded-vs-tax-included}

Από προεπιλογή, όλες οι τιμές των προϊόντων σας είναι χωρίς φόρο, που σημαίνει ότι οι φόροι **δεν περιλαμβάνονται** στην τιμή του προϊόντος. Αν διαπιστώσουμε ότι ένας πελάτης πρέπει να πληρώσει φόρους για μια συγκεκριμένη αγορά, θα προσθέσουμε τους φόρους **επιπλέον** του μερικού συνόλου.

Αν προτιμάτε οι φόροι να περιλαμβάνονται στην τιμή του προϊόντος σας, μπορείτε να το κάνετε ενεργοποιώντας τη ρύθμιση **Φόρος συμπεριλαμβανόμενος**.

![Σειρά διακόπτη Φόρος συμπεριλαμβανόμενος κάτω από τη ρύθμιση Ενεργοποίηση φόρων](/img/config/settings-taxes-inclusive.png)

Μην ξεχάσετε να **αποθηκεύσετε** τις αλλαγές που κάνατε.

###

## Δημιουργία φορολογικών συντελεστών {#creating-tax-rates}

Αφού ενεργοποιήσετε την Είσπραξη φόρων, θα χρειαστεί να δημιουργήσετε φορολογικούς συντελεστές για συγκεκριμένες τοποθεσίες χρησιμοποιώντας τον επεξεργαστή φορολογικών συντελεστών μας.

Μπορείτε να αποκτήσετε πρόσβαση στον επεξεργαστή κάνοντας κλικ στο κουμπί **Διαχείριση φορολογικών συντελεστών** στην πλαϊνή γραμμή της σελίδας ρυθμίσεων Φόρων.

![Σύνδεσμος Διαχείριση φορολογικών συντελεστών στο πλαίσιο Φορολογικοί συντελεστές στη σελίδα ρυθμίσεων](/img/config/settings-taxes-manage-rates.png)

Στη σελίδα επεξεργαστή φορολογικών συντελεστών, μπορείτε να προσθέσετε νέους Φορολογικούς συντελεστές κάνοντας κλικ στο κουμπί **Προσθήκη νέας γραμμής**.

![Πίνακας επεξεργαστή φορολογικών συντελεστών με κουμπί Προσθήκη νέας γραμμής στο επάνω μέρος](/img/config/tax-rates-editor.png)

Θα χρειαστεί να δώσετε σε κάθε φορολογικό συντελεστή έναν **τίτλο** (χρησιμοποιείται στα τιμολόγια). Έπειτα μπορείτε να επιλέξετε τη **χώρα** (απαιτείται), την **πολιτεία/περιφέρεια** και την **πόλη** (και τα δύο προαιρετικά) όπου θα χρεώνεται αυτός ο φόρος. Τέλος, προσθέστε τον **φορολογικό συντελεστή σε ποσοστό**.

### Φορολογικές κατηγορίες {#tax-categories}

Μπορείτε επίσης να δημιουργήσετε πολλαπλές Φορολογικές κατηγορίες, για να προσθέσετε διαφορετικούς φορολογικούς συντελεστές για διαφορετικά είδη προϊόντων.

Κάντε κλικ στο **Προσθήκη νέας φορολογικής κατηγορίας** και έπειτα γράψτε το όνομα της κατηγορίας σας και πατήστε **Δημιουργία**.

![Κουμπί Προσθήκη νέας φορολογικής κατηγορίας στο επάνω μέρος του επεξεργαστή φορολογικών συντελεστών](/img/config/tax-categories-add.png)

![Πεδίο εισαγωγής Όνομα φορολογικής κατηγορίας στο παράθυρο δημιουργίας κατηγορίας](/img/config/tax-categories-create-modal.png)

Για να περιηγηθείτε στις κατηγορίες, κάντε κλικ στο **Εναλλαγή** και επιλέξτε την κατηγορία στην οποία θέλετε να προσθέσετε νέους φόρους.

![Κουμπί αναπτυσσόμενου μενού Εναλλαγή για αλλαγή μεταξύ φορολογικών κατηγοριών](/img/config/tax-categories-switch.png)

![Αναπτυσσόμενο μενού επιλογής φορολογικής κατηγορίας που εμφανίζει τις διαθέσιμες κατηγορίες](/img/config/tax-categories-select.png)

Μπορείτε να ορίσετε τη φορολογική κατηγορία για ένα συγκεκριμένο προϊόν μεταβαίνοντας στη **σελίδα επεξεργασίας προϊόντος** και έπειτα στην καρτέλα Φόροι.

![Καρτέλα φόρων προϊόντος με φορολογική κατηγορία και διακόπτη φορολόγησης](/img/config/product-taxes.png)

Στην ίδια οθόνη, μπορείτε να απενεργοποιήσετε τον διακόπτη **Είναι φορολογητέο;** για να ενημερώσετε το Ultimate Multisite ότι δεν πρέπει να εισπράττει φόρους για το συγκεκριμένο προϊόν.

## Υποστήριξη ευρωπαϊκού ΦΠΑ {#european-vat-support}

Όπως αναφέρθηκε προηγουμένως, διαθέτουμε ένα πρόσθετο για πελάτες στην ΕΕ που έχουν πρόσθετες απαιτήσεις λόγω των ευρωπαϊκών κανονισμών ΦΠΑ.

Τα εργαλεία ΦΠΑ μας βοηθούν με μερικά σημαντικά πράγματα:

  * Εύκολη φόρτωση των συντελεστών ΦΠΑ της ΕΕ;

  * Συλλογή και επικύρωση αριθμού ΦΠΑ - και αντίστροφη χρέωση για οντότητες που απαλλάσσονται από τον ΦΠΑ (όπως εταιρείες με έγκυρους αριθμούς ΦΠΑ);

Για να εγκαταστήσετε αυτό το πρόσθετο, μεταβείτε στο **Ultimate Multisite > Ρυθμίσεις** και έπειτα κάντε κλικ στον σύνδεσμο της πλαϊνής γραμμής **Ελέγξτε τα πρόσθετά μας**.

![Πλαϊνή γραμμή σελίδας ρυθμίσεων με σύνδεσμο Ελέγξτε τα πρόσθετά μας](/img/config/settings-taxes-addons-link.png)

Θα ανακατευθυνθείτε στη σελίδα πρόσθετών μας. Εκεί, μπορείτε να αναζητήσετε το **πρόσθετο ΦΠΑ Ultimate Multisite** και να το εγκαταστήσετε.

<!-- Screenshot unavailable: VAT add-on tile on the Add-ons page -->

<!-- Screenshot unavailable: VAT add-on Install Now dialog -->

Έπειτα, μεταβείτε στο **Network Admin > Plugins** και ενεργοποιήστε αυτό το πρόσθετο σε όλο το δίκτυο.

<!-- Screenshot unavailable: Network Activate action for the VAT add-on on the Plugins page -->

Αν επιστρέψετε στην **καρτέλα Ρυθμίσεις φόρων**, θα δείτε νέες διαθέσιμες επιλογές. Ενεργοποιήστε την επιλογή **Ενεργοποίηση υποστήριξης ΦΠΑ** για να ενεργοποιήσετε τα νέα εργαλεία ΦΠΑ. Μην ξεχάσετε να **αποθηκεύσετε** τις ρυθμίσεις σας!

### Ανάκτηση φορολογικών συντελεστών ΦΠΑ {#pulling-on-vat-tax-rates}

Ένα από τα εργαλεία που προσθέτει η ενσωμάτωσή μας είναι η δυνατότητα φόρτωσης των φορολογικών συντελεστών για τα κράτη μέλη της ΕΕ. Αυτό μπορεί να γίνει επισκεπτόμενοι τη σελίδα επεξεργαστή φορολογικών συντελεστών αφού ενεργοποιήσετε την υποστήριξη ΦΠΑ ΕΕ.

Στο κάτω μέρος της σελίδας, θα δείτε τις επιλογές ανάκτησης ΦΠΑ. Επιλέγοντας έναν τύπο συντελεστή και κάνοντας κλικ στο κουμπί **Ενημέρωση συντελεστών ΦΠΑ ΕΕ**, θα ανακτηθεί και θα συμπληρωθεί αυτόματα ο πίνακας με τους φορολογικούς συντελεστές για κάθε κράτος μέλος της ΕΕ. Έπειτα, χρειάζεται μόνο να τον αποθηκεύσετε.

![Κουμπί Ενημέρωση συντελεστών ΦΠΑ ΕΕ στο κάτω μέρος του επεξεργαστή φορολογικών συντελεστών](/img/config/tax-rates-vat-pull.png)

Μπορείτε επίσης να επεξεργαστείτε τις τιμές αφού τις ανακτήσετε. Για να το κάνετε, απλώς επεξεργαστείτε τη γραμμή του πίνακα που χρειάζεστε και κάντε κλικ για να αποθηκεύσετε τις νέες τιμές.

### Επικύρωση ΦΠΑ {#vat-validation}

Όταν η υποστήριξη ΦΠΑ είναι ενεργοποιημένη, το Ultimate Multisite θα προσθέσει ένα πρόσθετο πεδίο στη φόρμα checkout, κάτω από το πεδίο διεύθυνσης χρέωσης. Το πεδίο θα εμφανίζεται μόνο για πελάτες που βρίσκονται στην ΕΕ.

<!-- Screenshot unavailable: VAT Number field on the frontend checkout form below the billing address -->

Το Ultimate Multisite θα επικυρώσει τότε τον ΑΦΜ και, αν επιστρέψει ως έγκυρος, εφαρμόζεται ο μηχανισμός αντίστροφης χρέωσης και ο φορολογικός συντελεστής ορίζεται σε 0% σε αυτήν την παραγγελία.
