# Διαχείριση Φόρων

Το Ultimate Multisite διαθέτει μια ενσωματωμένη λειτουργία είσπραξης φόρων στο βασικό plugin, οπότε αν χρειάζεται να εισπράττετε φόρους πωλήσεων στα πλάνα, τα πακέτα και τις υπηρεσίες σας, μπορείτε να το κάνετε εύκολα χωρίς να χρειαστεί να εγκαταστήσετε πρόσθετα.

Για εταιρείες που εδρεύουν στην Ευρώπη, προσφέρουμε ένα **add-on** που παρέχει εργαλεία και λειτουργίες για καλύτερη **υποστήριξη συμμόρφωσης με τον ΦΠΑ**.

Το Ultimate Multisite δεν υποβάλλει ούτε αποδίδει φόρους για λογαριασμό σας στο κράτος· απλώς σας βοηθάμε να εισπράττετε τους κατάλληλους φόρους κατά τη στιγμή της συναλλαγής. **Θα χρειαστεί να αποδώσετε τους φόρους μόνοι σας.**

## Ενεργοποίηση Είσπραξης Φόρων

Η είσπραξη φόρων δεν είναι ενεργοποιημένη από προεπιλογή. Για να την ενεργοποιήσετε, πηγαίνετε στο **Ultimate Multisite > Ρυθμίσεις > Φόροι** και ενεργοποιήστε τη ρύθμιση Ενεργοποίηση Φόρων.

![Διακόπτης Ενεργοποίησης Φόρων στις ρυθμίσεις φόρων](/img/config/settings-taxes.png)

### Τιμές χωρίς φόρο vs. Τιμές με φόρο

Από προεπιλογή, όλες οι τιμές των προϊόντων σας είναι χωρίς φόρο, που σημαίνει ότι οι φόροι **δεν περιλαμβάνονται** στην τιμή του προϊόντος. Αν διαπιστώσουμε ότι ένας πελάτης πρέπει να πληρώσει φόρους για μια συγκεκριμένη αγορά, θα προσθέσουμε τους φόρους **επιπλέον** του υποσυνόλου.

Αν προτιμάτε οι φόροι να περιλαμβάνονται στην τιμή του προϊόντος σας, μπορείτε να το κάνετε ενεργοποιώντας τη ρύθμιση **Συμπερίληψη Φόρου**.

![Διακόπτης ρύθμισης Συμπερίληψης Φόρου](/img/config/settings-taxes.png)

Μην ξεχάσετε να **αποθηκεύσετε** τις αλλαγές που κάνατε.

## Δημιουργία Φορολογικών Συντελεστών

Αφού ενεργοποιήσετε την Είσπραξη Φόρων, θα χρειαστεί να δημιουργήσετε φορολογικούς συντελεστές για συγκεκριμένες τοποθεσίες χρησιμοποιώντας τον επεξεργαστή φορολογικών συντελεστών.

Μπορείτε να αποκτήσετε πρόσβαση στον επεξεργαστή κάνοντας κλικ στο κουμπί **Διαχείριση Φορολογικών Συντελεστών** στην πλαϊνή στήλη της σελίδας ρυθμίσεων Φόρων.

![Κουμπί Διαχείρισης Φορολογικών Συντελεστών στην πλαϊνή στήλη](/img/config/settings-taxes.png)

Στη σελίδα του επεξεργαστή φορολογικών συντελεστών, μπορείτε να προσθέσετε νέους Φορολογικούς Συντελεστές κάνοντας κλικ στο κουμπί **Προσθήκη νέας Γραμμής**.

![Επεξεργαστής φορολογικών συντελεστών με κουμπί Προσθήκης νέας Γραμμής](/img/config/settings-taxes.png)

Θα χρειαστεί να δώσετε σε κάθε φορολογικό συντελεστή έναν **τίτλο** (που χρησιμοποιείται στα τιμολόγια). Στη συνέχεια μπορείτε να επιλέξετε τη **χώρα** (υποχρεωτικό), την **πολιτεία** και την **πόλη** (και τα δύο προαιρετικά) όπου θα χρεώνεται αυτός ο φόρος. Τέλος, προσθέστε τον **φορολογικό συντελεστή σε ποσοστό**.

### Κατηγορίες Φόρων

Μπορείτε επίσης να δημιουργήσετε πολλαπλές Κατηγορίες Φόρων, για να προσθέσετε διαφορετικούς φορολογικούς συντελεστές για διαφορετικά είδη προϊόντων.

Κάντε κλικ στο **Προσθήκη νέας Κατηγορίας Φόρου**, στη συνέχεια γράψτε το όνομα της κατηγορίας σας και πατήστε **Δημιουργία**.

![Κουμπί Προσθήκης νέας Κατηγορίας Φόρου](/img/config/settings-taxes.png)

![Δημιουργία νέας κατηγορίας φόρου](/img/config/settings-taxes.png)

Για να περιηγηθείτε στις κατηγορίες, κάντε κλικ στο **Εναλλαγή** και επιλέξτε την κατηγορία στην οποία θέλετε να προσθέσετε νέους φόρους.

![Εναλλαγή μεταξύ κατηγοριών φόρων](/img/config/settings-taxes.png)

![Επιλογή κατηγορίας φόρου για προβολή](/img/config/settings-taxes.png)

Μπορείτε να ορίσετε την κατηγορία φόρου για ένα συγκεκριμένο προϊόν πηγαίνοντας στη **σελίδα επεξεργασίας Προϊόντος** και στη συνέχεια στην καρτέλα Φόροι.

![Καρτέλα φόρων προϊόντος με κατηγορία φόρου και διακόπτη φορολόγησης](/img/config/product-taxes.png)

Στην ίδια οθόνη, μπορείτε να απενεργοποιήσετε τον διακόπτη **Είναι Φορολογήσιμο;** για να ενημερώσετε το Ultimate Multisite ότι δεν πρέπει να εισπράττει φόρους για το συγκεκριμένο προϊόν.

## Υποστήριξη Ευρωπαϊκού ΦΠΑ

Όπως αναφέρθηκε προηγουμένως, διαθέτουμε ένα add-on για πελάτες στην ΕΕ που έχουν πρόσθετες απαιτήσεις λόγω των ευρωπαϊκών κανονισμών ΦΠΑ.

Τα εργαλεία ΦΠΑ μας βοηθούν με μερικά σημαντικά θέματα:

  * Εύκολη φόρτωση συντελεστών ΦΠΑ της ΕΕ·

  * Συλλογή και επικύρωση Αριθμού ΦΠΑ - και αντιστροφή της επιβάρυνσης για οντότητες απαλλαγμένες από ΦΠΑ (όπως εταιρείες με έγκυρους αριθμούς ΦΠΑ)·

Για να εγκαταστήσετε αυτό το add-on, πηγαίνετε στο **Ultimate Multisite > Ρυθμίσεις** και μετά κάντε κλικ στον σύνδεσμο **Δείτε τα Add-ons μας** στην πλαϊνή στήλη.

![Σελίδα ρυθμίσεων με σύνδεσμο add-ons στην πλαϊνή στήλη](/img/config/settings-taxes.png)

Θα ανακατευθυνθείτε στη σελίδα add-ons μας. Εκεί, μπορείτε να αναζητήσετε το **Ultimate Multisite VAT add-on** και να το εγκαταστήσετε.

![VAT add-on στη σελίδα add-ons](/img/config/settings-taxes.png)

![Παράθυρο εγκατάστασης VAT add-on](/img/config/settings-taxes.png)

Στη συνέχεια, πηγαίνετε στο **Network Admin > Plugins** και ενεργοποιήστε αυτό το add-on σε όλο το δίκτυο.

![Ενεργοποίηση του VAT add-on σε επίπεδο δικτύου](/img/config/settings-taxes.png)

Αν επιστρέψετε στην **καρτέλα Ρυθμίσεων Φόρων**, θα δείτε νέες διαθέσιμες επιλογές. Ενεργοποιήστε την επιλογή **Ενεργοποίηση Υποστήριξης ΦΠΑ** για να ενεργοποιήσετε τα νέα εργαλεία ΦΠΑ. Μην ξεχάσετε να **αποθηκεύσετε** τις ρυθμίσεις σας!

![Διακόπτης Ενεργοποίησης Υποστήριξης ΦΠΑ στις ρυθμίσεις φόρων](/img/config/settings-taxes.png)

### Άντληση Συντελεστών ΦΠΑ

Ένα από τα εργαλεία που προσθέτει η ενσωμάτωσή μας είναι η δυνατότητα φόρτωσης των φορολογικών συντελεστών για τα κράτη μέλη της ΕΕ. Αυτό μπορεί να γίνει επισκεπτόμενοι τη σελίδα του επεξεργαστή φορολογικών συντελεστών αφού ενεργοποιήσετε την υποστήριξη ΦΠΑ της ΕΕ.

Στο κάτω μέρος της σελίδας, θα δείτε τις επιλογές άντλησης ΦΠΑ. Επιλέγοντας έναν τύπο συντελεστή και κάνοντας κλικ στο κουμπί **Ενημέρωση Συντελεστών ΦΠΑ ΕΕ** θα αντλήσει και θα συμπληρώσει αυτόματα τον πίνακα με τους φορολογικούς συντελεστές για κάθε κράτος μέλος της ΕΕ. Στη συνέχεια, χρειάζεται μόνο να τον αποθηκεύσετε.

### ![Επιλογές άντλησης συντελεστών ΦΠΑ με κουμπί Ενημέρωσης Συντελεστών ΦΠΑ ΕΕ](/img/config/settings-taxes.png)

Μπορείτε επίσης να επεξεργαστείτε τις τιμές μετά την άντλησή τους. Για να το κάνετε, απλώς επεξεργαστείτε τη γραμμή του πίνακα που χρειάζεστε και κάντε κλικ για να αποθηκεύσετε τις νέες τιμές.

### Επικύρωση ΦΠΑ

Όταν είναι ενεργοποιημένη η υποστήριξη ΦΠΑ, το Ultimate Multisite θα προσθέσει ένα επιπλέον πεδίο στη φόρμα ολοκλήρωσης αγοράς, κάτω από το πεδίο διεύθυνσης χρέωσης. Το πεδίο θα εμφανίζεται μόνο σε πελάτες που βρίσκονται στην ΕΕ.

![Πεδίο αριθμού ΦΠΑ στη φόρμα ολοκλήρωσης αγοράς](/img/config/settings-taxes.png)

Το Ultimate Multisite στη συνέχεια θα επικυρώσει τον Αριθμό ΦΠΑ και αν επιστρέψει ως έγκυρος, εφαρμόζεται ο μηχανισμός αντιστροφής της επιβάρυνσης και ο φορολογικός συντελεστής ορίζεται στο 0% για αυτή την παραγγελία.
