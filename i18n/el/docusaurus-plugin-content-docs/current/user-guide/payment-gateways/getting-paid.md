# Πληρωμές (v2)

_**ΣΗΜΑΝΤΙΚΗ ΣΗΜΕΙΩΣΗ: Αυτό το άρθρο αναφέρεται στην έκδοση 2.x του Ultimate Multisite.**_

Το Ultimate Multisite διαθέτει ενσωματωμένο σύστημα συνδρομών και χρεώσεων. Για να λειτουργήσει το σύστημα χρεώσεων, έχουμε ενσωματώσει τις πιο δημοφιλείς πύλες πληρωμών που χρησιμοποιούνται στο ηλεκτρονικό εμπόριο. Οι προεπιλεγμένες πύλες πληρωμών στο Ultimate Multisite είναι το _Stripe_, το _PayPal_ και η Χειροκίνητη Πληρωμή. Μπορείτε επίσης να χρησιμοποιήσετε το _WooCommerce_, το _GoCardless_ και το _Payfast_ για να λαμβάνετε πληρωμές, εγκαθιστώντας τα αντίστοιχα πρόσθετα (add-ons).

## Βασικές Ρυθμίσεις

Μπορείτε να ρυθμίσετε οποιαδήποτε από αυτές τις πύλες πληρωμών στις ρυθμίσεις πληρωμών του Ultimate Multisite. Θα τις βρείτε πηγαίνοντας στο **μενού Ultimate Multisite > Ρυθμίσεις > Πληρωμές.**

![Σελίδα ρυθμίσεων πληρωμών στο Ultimate Multisite](/img/config/settings-payment-gateways.png)

Πριν ρυθμίσετε την πύλη πληρωμών σας, ρίξτε μια ματιά στις βασικές ρυθμίσεις πληρωμών που μπορείτε να διαμορφώσετε:

**Υποχρεωτική αυτόματη ανανέωση:** Αυτή η επιλογή διασφαλίζει ότι η πληρωμή θα επαναλαμβάνεται αυτόματα στο τέλος κάθε κύκλου χρέωσης, ανάλογα με τη συχνότητα χρέωσης που επέλεξε ο χρήστης.

![Ρύθμιση εναλλαγής υποχρεωτικής αυτόματης ανανέωσης](/img/config/settings-payment-gateways.png)

**Δοκιμαστική περίοδος χωρίς τρόπο πληρωμής:** Με αυτή την επιλογή ενεργοποιημένη, ο πελάτης σας δεν θα χρειαστεί να προσθέσει οικονομικά στοιχεία κατά τη διαδικασία εγγραφής. Αυτό θα απαιτηθεί μόνο όταν λήξει η δοκιμαστική περίοδος.

![Εναλλαγή επιτρεπόμενης δοκιμαστικής περιόδου χωρίς τρόπο πληρωμής](/img/config/settings-payment-gateways.png)

**Αποστολή τιμολογίου μετά την επιβεβαίωση πληρωμής:** Αυτή η επιλογή σας δίνει τη δυνατότητα να επιλέξετε αν θα αποστέλλεται τιμολόγιο μετά την πληρωμή ή όχι. Σημειώστε ότι οι χρήστες θα έχουν πρόσβαση στο ιστορικό πληρωμών τους μέσα από τον πίνακα ελέγχου του υποϊστότοπού τους. Αυτή η επιλογή δεν ισχύει για τη Χειροκίνητη Πύλη.

![Εναλλαγή αποστολής τιμολογίου μετά την επιβεβαίωση πληρωμής](/img/config/settings-payment-gateways.png)

**Σύστημα αρίθμησης τιμολογίων:** Εδώ μπορείτε να επιλέξετε είτε κωδικό αναφοράς πληρωμής είτε σύστημα αύξουσας αρίθμησης. Αν επιλέξετε να χρησιμοποιείτε κωδικό αναφοράς πληρωμής για τα τιμολόγιά σας, δεν χρειάζεται να ρυθμίσετε τίποτα. Αν επιλέξετε σύστημα αύξουσας αρίθμησης, θα πρέπει να ρυθμίσετε τον **επόμενο αριθμό τιμολογίου** (Αυτός ο αριθμός θα χρησιμοποιηθεί ως αριθμός τιμολογίου για το επόμενο τιμολόγιο που θα δημιουργηθεί στο σύστημα. Αυξάνεται κατά ένα κάθε φορά που δημιουργείται νέο τιμολόγιο. Μπορείτε να τον αλλάξετε και να τον αποθηκεύσετε για να επαναφέρετε τον αύξοντα αριθμό τιμολογίου σε μια συγκεκριμένη τιμή) και το **πρόθεμα αριθμού τιμολογίου.**

![Επιλογή συστήματος αρίθμησης τιμολογίων](/img/config/settings-payment-gateways.png)

![Ρυθμίσεις αύξοντα αριθμού τιμολογίου και προθέματος](/img/config/settings-payment-gateways.png)

## Πού θα βρείτε τις πύλες πληρωμών:

Μπορείτε να ρυθμίσετε τις πύλες πληρωμών στην ίδια σελίδα (**Ultimate Multisite > Ρυθμίσεις > Πληρωμές**). Ακριβώς κάτω από τις **ενεργές πύλες πληρωμών**, θα δείτε: _Stripe_, _Stripe Checkout_, _PayPal_ και _Χειροκίνητη_.

![Λίστα ενεργών πυλών πληρωμών](/img/config/settings-payment-gateways.png)

Έχουμε ένα αποκλειστικό άρθρο για κάθε πύλη πληρωμών που θα σας καθοδηγήσει στα βήματα ρύθμισής της, τα οποία μπορείτε να βρείτε στους παρακάτω συνδέσμους.

**Ρύθμιση της πύλης Stripe**

**Ρύθμιση της πύλης PayPal**

**Ρύθμιση χειροκίνητων πληρωμών**

Τώρα, αν θέλετε να χρησιμοποιήσετε το _WooCommerce_, το _GoCardless_ ή το _Payfast_ ως πύλη πληρωμών, θα πρέπει να **εγκαταστήσετε και να ρυθμίσετε τα αντίστοιχα πρόσθετα**.

### Πώς να εγκαταστήσετε το πρόσθετο WooCommerce:

Κατανοούμε ότι το _Stripe_ και το _PayPal_ δεν είναι διαθέσιμα σε ορισμένες χώρες, γεγονός που περιορίζει ή εμποδίζει τους χρήστες του Ultimate Multisite να χρησιμοποιήσουν αποτελεσματικά το plugin μας. Γι' αυτό δημιουργήσαμε ένα πρόσθετο για ενσωμάτωση με το _WooCommerce_, το οποίο είναι ένα πολύ δημοφιλές plugin ηλεκτρονικού εμπορίου. Προγραμματιστές σε όλο τον κόσμο έχουν δημιουργήσει πρόσθετα για να ενσωματώσουν διάφορες πύλες πληρωμών σε αυτό. Εκμεταλλευτήκαμε αυτό για να επεκτείνουμε τις πύλες πληρωμών που μπορείτε να χρησιμοποιήσετε με το σύστημα χρεώσεων του Ultimate Multisite.

_**ΣΗΜΑΝΤΙΚΟ:** Το Ultimate Multisite: WooCommerce Integration απαιτεί να είναι ενεργοποιημένο το WooCommerce τουλάχιστον στον κύριο ιστότοπό σας._

Πρώτα, πηγαίνετε στη σελίδα πρόσθετων. Θα τη βρείτε πηγαίνοντας στο **Ultimate Multisite > Ρυθμίσεις**. Θα δείτε τον πίνακα **Πρόσθετα**. Κάντε κλικ στο **Δείτε τα Πρόσθετά μας**.

![Σελίδα ρυθμίσεων με την ενότητα πρόσθετων](/img/config/settings-general.png)

Αφού κάνετε κλικ στο **Δείτε τα Πρόσθετά μας**, θα ανακατευθυνθείτε στη σελίδα πρόσθετων. Εδώ μπορείτε να βρείτε όλα τα πρόσθετα του Ultimate Multisite. Κάντε κλικ στο πρόσθετο **Ultimate Multisite: WooCommerce Integration**.

![Σελίδα πρόσθετων με τη λίστα διαθέσιμων πρόσθετων](/img/config/settings-general.png)

Θα εμφανιστεί ένα παράθυρο με τις λεπτομέρειες του πρόσθετου. Απλά κάντε κλικ στο **Εγκατάσταση τώρα**.

![Παράθυρο εγκατάστασης πρόσθετου WooCommerce](/img/config/settings-general.png)

Μετά την ολοκλήρωση της εγκατάστασης, θα ανακατευθυνθείτε στη σελίδα plugins. Εδώ, απλά κάντε κλικ στο **Ενεργοποίηση Δικτύου** και το πρόσθετο WooCommerce θα ενεργοποιηθεί στο δίκτυό σας.

![Ενεργοποίηση Δικτύου για το πρόσθετο WooCommerce](/img/config/settings-general.png)

Μετά την ενεργοποίηση, αν δεν έχετε ακόμα εγκαταστήσει και ενεργοποιήσει το plugin WooCommerce στον ιστότοπό σας, θα λάβετε μια υπενθύμιση.

![Ειδοποίηση υπενθύμισης ενεργοποίησης WooCommerce](/img/config/settings-general.png)

Για να διαβάσετε περισσότερα σχετικά με το πρόσθετο WooCommerce Integration, **κάντε κλικ εδώ**.

### Πώς να εγκαταστήσετε το πρόσθετο GoCardless:

Τα βήματα για την εγκατάσταση του πρόσθετου _GoCardless_ είναι σχεδόν ίδια με αυτά του πρόσθετου _WooCommerce_. Πηγαίνετε στη σελίδα πρόσθετων και επιλέξτε το πρόσθετο **Ultimate Multisite: GoCardless Gateway**.

![Σελίδα πρόσθετων με τη λίστα διαθέσιμων πρόσθετων](/img/config/settings-general.png)

Θα εμφανιστεί το παράθυρο του πρόσθετου. Κάντε κλικ στο **Εγκατάσταση τώρα**.

![Παράθυρο εγκατάστασης πρόσθετου GoCardless](/img/config/settings-general.png)

Μετά την ολοκλήρωση της εγκατάστασης, θα ανακατευθυνθείτε στη σελίδα plugins. Εδώ, απλά κάντε κλικ στο **Ενεργοποίηση Δικτύου** και το πρόσθετο _GoCardless_ θα ενεργοποιηθεί στο δίκτυό σας.

![Ενεργοποίηση Δικτύου για το πρόσθετο GoCardless](/img/config/settings-general.png)

Για να μάθετε πώς να ξεκινήσετε με την πύλη _GoCardless_, **διαβάστε αυτό το άρθρο**.

### Πώς να εγκαταστήσετε το πρόσθετο Payfast:

Πηγαίνετε στη σελίδα πρόσθετων και επιλέξτε το πρόσθετο **Ultimate Multisite: Payfast Gateway**.

![Σελίδα πρόσθετων με τη λίστα διαθέσιμων πρόσθετων](/img/config/settings-general.png)

Θα εμφανιστεί το παράθυρο του πρόσθετου. Κάντε κλικ στο **Εγκατάσταση τώρα.**

![Παράθυρο εγκατάστασης πρόσθετου Payfast](/img/config/settings-general.png)

Μετά την ολοκλήρωση της εγκατάστασης, θα ανακατευθυνθείτε στη σελίδα plugins. Εδώ, απλά κάντε κλικ στο **Ενεργοποίηση Δικτύου** και το πρόσθετο _Payfast_ θα ενεργοποιηθεί στο δίκτυό σας.

![Ενεργοποίηση Δικτύου για το πρόσθετο Payfast](/img/config/settings-general.png)
