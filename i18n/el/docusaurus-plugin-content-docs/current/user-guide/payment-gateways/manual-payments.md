# Ρύθμιση χειροκίνητων πληρωμών (v2)

_**ΣΗΜΑΝΤΙΚΗ ΣΗΜΕΙΩΣΗ: Αυτό το άρθρο αναφέρεται στην έκδοση Ultimate Multisite 2.x.**_

Οι χειροκίνητες πληρωμές είναι ένας τρόπος να προσφέρετε άλλες μεθόδους πληρωμής, σε περίπτωση που το **Stripe** ή το **PayPal** δεν είναι διαθέσιμα για τους χρήστες σας. Μπορεί να είναι έμβασμα ή τραπεζική μεταφορά ή οποιαδήποτε άλλη μέθοδος πληρωμής που είναι διαθέσιμη τοπικά στους χρήστες σας.

## Πώς να ενεργοποιήσετε τις χειροκίνητες πληρωμές

Η ρύθμιση της χειροκίνητης πληρωμής είναι πολύ εύκολη. Απλώς πρέπει να την ενεργοποιήσετε στις πύλες πληρωμών και να προσθέσετε λεπτομερείς οδηγίες για το πώς πρέπει ο χρήστης να στείλει την πληρωμή.

Πρώτα, μεταβείτε στο **Ultimate Multisite > Settings > Payments**. Κάτω από το **Payment Gateways** , ενεργοποιήστε το **Manual**. Θα δείτε ότι θα εμφανιστεί για εσάς ένα πλαίσιο **Payment Instructions**.

Προσθέστε σε αυτό το πλαίσιο τις πληροφορίες που θα χρειαστεί ο πελάτης σας για να πραγματοποιήσει την πληρωμή. Μπορεί να είναι τα στοιχεία του τραπεζικού σας λογαριασμού και το email σας, ώστε ο πελάτης να μπορεί να σας στείλει την επιβεβαίωση της πληρωμής, για παράδειγμα.

![Εναλλαγή πύλης χειροκίνητης πληρωμής με περιοχή κειμένου Payment Instructions](/img/config/manual-gateway-expanded.png)

Ακολουθεί η διεπαφή ρυθμίσεων της χειροκίνητης πύλης:

![Ρυθμίσεις χειροκίνητης πύλης](/img/config/manual-gateway-settings.png)

Αφού τη ρυθμίσετε, απλώς κάντε κλικ στο **Save Settings** και ολοκληρώθηκε. Όταν οι χρήστες εγγράφονται στο δίκτυό σας, θα βλέπουν ένα μήνυμα που τους ενημερώνει ότι θα λάβουν τις οδηγίες σας για να ολοκληρώσουν την αγορά.

![Μήνυμα επιβεβαίωσης εγγραφής που ενημερώνει τον χρήστη ότι θα λάβει οδηγίες πληρωμής](/img/frontend/registration-manual-notice.png)

Και θα λάβουν επίσης ένα μήνυμα στη σελίδα **Thank You** με τις οδηγίες πληρωμής σας.

## Επιβεβαίωση χειροκίνητων πληρωμών

Για να επιβεβαιώσετε μια χειροκίνητη πληρωμή, μεταβείτε στο μενού **Payments** στην αριστερή μπάρα. Εκεί μπορείτε να δείτε όλες τις πληρωμές στο δίκτυό σας και τις λεπτομέρειές τους, συμπεριλαμβανομένης της **κατάστασής** τους. Μια χειροκίνητη πληρωμή θα έχει πάντα κατάσταση **Pending** μέχρι να την αλλάξετε χειροκίνητα.

![Λίστα πληρωμών που εμφανίζει εκκρεμή χειροκίνητη πληρωμή](/img/admin/payments-list.png)

Μπείτε στη σελίδα πληρωμής κάνοντας κλικ στον **κωδικό αναφοράς**. Σε αυτή τη σελίδα έχετε όλες τις λεπτομέρειες της εκκρεμούς πληρωμής, όπως αναγνωριστικό αναφοράς, προϊόντα, χρονικές σημάνσεις και άλλα.

![Σελίδα λεπτομερειών πληρωμής που εμφανίζει κωδικό αναφοράς, προϊόντα και σύνολα](/img/admin/payment-edit.png)

Στη δεξιά στήλη, μπορείτε να αλλάξετε την κατάσταση της πληρωμής. Η αλλαγή της σε **Completed** και η **ενεργοποίηση της επιλογής Activate Membership** θα ενεργοποιήσουν τον ιστότοπο του πελάτη σας και η συνδρομή του θα είναι ενεργή.

![Σελίδα επεξεργασίας πληρωμής με Status ορισμένο σε Completed και εναλλαγή Activate Membership](/img/admin/payment-activate-membership.png)
