# Ρύθμιση Χειροκίνητων Πληρωμών (v2)

_**ΣΗΜΑΝΤΙΚΗ ΣΗΜΕΙΩΣΗ: Αυτό το άρθρο αναφέρεται στην έκδοση 2.x του Ultimate Multisite.**_

Οι χειροκίνητες πληρωμές σας επιτρέπουν να προσφέρετε εναλλακτικούς τρόπους πληρωμής σε περίπτωση που το **Stripe** ή το **PayPal** δεν είναι διαθέσιμα για τους χρήστες σας. Μπορεί να είναι τραπεζικό έμβασμα ή οποιοσδήποτε άλλος τρόπος πληρωμής διαθέσιμος τοπικά στους χρήστες σας.

## Πώς να ενεργοποιήσετε τις Χειροκίνητες Πληρωμές

Η ρύθμιση των χειροκίνητων πληρωμών είναι πολύ εύκολη. Αρκεί να τις ενεργοποιήσετε από τις πύλες πληρωμών και να προσθέσετε αναλυτικές οδηγίες για το πώς ο χρήστης πρέπει να στείλει την πληρωμή.

Αρχικά, μεταβείτε στο **Ultimate Multisite > Settings > Payments**. Κάτω από το **Payment Gateways**, ενεργοποιήστε την επιλογή **Manual**. Θα δείτε να εμφανίζεται ένα πλαίσιο **Payment Instructions**.

Προσθέστε σε αυτό το πλαίσιο τις πληροφορίες που θα χρειαστεί ο πελάτης σας για να ολοκληρώσει την πληρωμή. Για παράδειγμα, μπορείτε να συμπεριλάβετε τα στοιχεία του τραπεζικού σας λογαριασμού και το email σας, ώστε ο πελάτης να σας στείλει την επιβεβαίωση πληρωμής.

![Διακόπτης ενεργοποίησης χειροκίνητης πύλης πληρωμών και πλαίσιο οδηγιών πληρωμής](/img/config/settings-payment-gateways.png)

Αφού ολοκληρώσετε τη ρύθμιση, απλά κάντε κλικ στο **Save Settings** και είστε έτοιμοι. Όταν οι χρήστες εγγράφονται στο δίκτυό σας, θα βλέπουν ένα μήνυμα που τους ενημερώνει ότι θα λάβουν τις οδηγίες σας για να ολοκληρώσουν την αγορά.

![Μήνυμα χειροκίνητης πληρωμής κατά την εγγραφή](/img/config/settings-payment-gateways.png)

Επίσης, θα λάβουν ένα μήνυμα στη σελίδα **Thank You** με τις οδηγίες πληρωμής σας.

![Σελίδα Thank You με οδηγίες πληρωμής](/img/config/settings-payment-gateways.png)

## Επιβεβαίωση χειροκίνητων πληρωμών

Για να επιβεβαιώσετε μια χειροκίνητη πληρωμή, μεταβείτε στο μενού **Payments** στην αριστερή στήλη. Εκεί μπορείτε να δείτε όλες τις πληρωμές στο δίκτυό σας και τις λεπτομέρειές τους, συμπεριλαμβανομένης της **κατάστασής** τους. Μια χειροκίνητη πληρωμή θα έχει πάντα κατάσταση **Pending** μέχρι να την αλλάξετε χειροκίνητα.

![Λίστα πληρωμών με εκκρεμή χειροκίνητη πληρωμή](/img/admin/payments-list.png)

Μπείτε στη σελίδα της πληρωμής κάνοντας κλικ στον **κωδικό αναφοράς**. Σε αυτή τη σελίδα έχετε όλες τις λεπτομέρειες της εκκρεμούς πληρωμής, όπως αναγνωριστικό αναφοράς, προϊόντα, χρονικές σημάνσεις και άλλα.

![Σελίδα λεπτομερειών πληρωμής με κωδικό αναφοράς και προϊόντα](/img/admin/payments-list.png)

Στη δεξιά στήλη, μπορείτε να αλλάξετε την κατάσταση της πληρωμής. Αλλάζοντάς την σε **Completed** και ενεργοποιώντας την επιλογή **Activate Membership**, θα ενεργοποιήσετε τον ιστότοπο του πελάτη σας και η συνδρομή του θα είναι ενεργή.

![Αλλαγή κατάστασης πληρωμής σε Completed με διακόπτη Activate Membership](/img/admin/payments-list.png)
