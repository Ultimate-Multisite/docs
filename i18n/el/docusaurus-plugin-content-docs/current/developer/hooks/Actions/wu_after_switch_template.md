# Action: wu_after_switch_template {#action-wuafterswitchtemplate}

Επιτρέπει στους προγραμματιστές plugins να συνδέσουν (hook) συναρτήσεις αφού ένας χρήστης ή super admin αλλάξει το template του site.

| Όνομα | Τύπος | Περιγραφή |
|------|------|-------------|
| $id | `int` | Site ID |
