# Action: wu_dashboard_widgets

Επιτρέπει στους developers των plugins να προσθέτουν widgets στο πάνελ του Dashboard του Network.

## Παράμετροι {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $tab | `string` | Η τρέχουσα καρτέλα. |
| $screen | `\WP_Screen` | Το αντικείμενο της οθόνης. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Αντίγραφο της διαχειριστικής σελίδας Ultimate Multisite. |

### Από {#since}

- 2.0.0

Ορίζεται στο [`inc/admin-pages/class-dashboard-admin-page.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/admin-pages/class-dashboard-admin-page.php#L225) στην γραμμή 225.
