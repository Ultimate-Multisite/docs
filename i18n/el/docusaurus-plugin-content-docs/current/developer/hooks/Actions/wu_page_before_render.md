# Action: wu_page_before_render

Επιτρέπει στους developers plugins να προσθέτουν επιπ nội dung πριν από το rendering της σελίδας.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $page_id | `string` | Ο ID της σελίδας αυτής. |
| $page | `object` | Το αντικείμενο της σελίδας. |
