# Action: wu_page_{$this->id}_after_render

Επιτρέπει στους developers των plugins να προσθέτουν επιπλέον περιεχόμενο αφού εκτυπώσουμε τη σελίδα

## Παράμετροι

| Όνομα | Τύπος | Περιγραφή |
|------|------|-------------|
| $page_id | `string` | Η ID αυτής της σελίδας |
| $page | `object` | Το αντικείμενο της σελίδας. |
