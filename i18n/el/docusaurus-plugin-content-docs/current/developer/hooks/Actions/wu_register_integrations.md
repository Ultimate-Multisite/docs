# Action: wu_register_integrations {#action-wuregisterintegrations}

Εκτελείται για να επιτρέψει στις ενσωματώσεις (integrations) να καταχωρηθούν.

Το Core καταχωρεί εδώ τις ενσωματώσεις του.

## Παράμετροι {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $registry | `\Integration_Registry` | Η οντότητα (instance) του καταχώρισης. |

### Since {#since}

- 2.5.0
