# Action: wu_mcp_adapter_initialized

Εκτελείται όταν ο MCP adapter έχει ολοκληρώσει την 초기ποίηση.

Επιτρέπει σε άλλα plugins και themes να καταχωρήσουν τις δικές τους ικανότητες.

## Παράμετροι

| Όνομα | Τύπος | Περιγραφή |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | Το αντίγραφο του MCP adapter. |

### Από την έκδοση

- 2.5.0
### Πηγή

Ορίζεται στο [`inc/class-mcp-adapter.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-mcp-adapter.php#L98) στην γραμμή 98
