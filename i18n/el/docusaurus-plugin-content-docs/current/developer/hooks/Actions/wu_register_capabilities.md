# Action: wu_register_capabilities

Εκτελείται για να επιτρέψει στους add-ons να συνδέουν modules ικανοτήτων (capability modules) με τις ενσωματώσεις (integrations).

## Παράμετροι {#parameters}

| Όνομα | Τύπος | Περιγραφή |
|------|------|-------------|
| $registry | `\Integration_Registry` | Η εντολή (instance) του registry. |

### Από {#since}

- 2.5.0

Ορίζεται στο [`inc/integrations/class-integration-registry.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/integrations/class-integration-registry.php#L148) στην γραμμή 148
