# Action: wu_magic_link_login

Εκτελείται όταν ένας χρήστης συνδεθεί μέσω μαγικού συνδέσμου.

## Παράμετροι

| Όνομα | Τύπος | Περιγραφή |
|------|------|-------------|
| $user_id | `int` | Ο αναγνωριστικός αριθμός χρήστη. |
| $site_id | `int` | Ο αναγνωριστικός αριθμός του site. |
