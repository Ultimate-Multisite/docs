# Action: wu_before_light_ajax

Σε ορισμένες περιπτώσεις, θα χρειαστούμε να φορτώσουμε επιπλέον λειτουργικότητα για να διαχειριστούμε τις δράσεις.

Αυτό το action μπορεί να χρησιμοποιηθεί για το φόρτωση επιπλέον εξαρτήσεων όταν είναι απαραίτητο.

- 2.0.0
