# Filter: wu_upgrade_to_unlock_url {#filter-wuupgradetounlockurl}

Επιτρέπει στους developers να αλλάξουν τη URL που χρησιμοποιείται για την αποκλειδώσα επιβίωση (upgrade to unlock URL).

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $url | `string` | Η URL που χρησιμοποιείται για την επιβίωση. |
| $args | `array` | Το module και ο τύπος περιορισμού που χρειάζεται επιβίωση. |

### Source {#source}

Ορίζεται στο [`inc/functions/limitations.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/limitations.php#L177) στην γραμμή 177.
