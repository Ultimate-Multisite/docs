# Φίλτρο: wu_current_set_customer

Επιτρέπει στους προγραμματιστές να τροποποιήσουν τη βασική συμπεριφορά και να ορίσουν διαφορετικά τον τρέχοντα πελάτη.

## Παράμετροι

| Όνομα | Τύπος | Περιγραφή |
|------|------|-------------|
| $customer | `\WP_Ultimo\Models\Customer` | Ο τρέχων πελάτης που θέλετε να ορίσετε. |
| $ | `self` | Το τρέχον ίδιο στιγμιότυπο (instance) της κλάσης. |

### Από

- 2.0.9
### Πηγή

Ορίζεται στο [`inc/class-current.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-current.php#L377) στην γραμμή 377
