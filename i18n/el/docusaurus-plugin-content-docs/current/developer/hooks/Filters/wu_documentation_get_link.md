# Φίλτρο: wu_documentation_get_link {#filter-wudocumentationgetlink}

Επιτρέπει στους προγραμματιστές plugins να φιλτράρουν τους συνδέσμους.

Δεν είμαι σίγουρος πώς θα είναι χρήσιμο, αλλά δεν βλάπτει να το έχουμε.

## Παράμετροι {#parameters}

| Όνομα | Τύπος | Περιγραφή |
|------|------|-------------|
| $link | `string` | Ο σύνδεσμος που καταχωρήθηκε |
| $slug | `string` | Το slug που χρησιμοποιείται για την ανάκτηση του συνδέσμου |
| $default_link | `string` | Ο προεπιλεγμένος καταχωρημένος σύνδεσμος |

### Από {#since}

- 1.7.0
### Πηγή {#source}

Ορίζεται στο [`inc/class-documentation.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-documentation.php#L125) στην γραμμή 125
