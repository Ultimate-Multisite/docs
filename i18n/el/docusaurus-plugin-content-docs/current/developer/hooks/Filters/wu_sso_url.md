# Φίλτρο: wu_sso_url

Φιλτράρει τα παραγόμενα SSO URLs πριν επιστραφούν για ενέργειες πελατών μεταξύ τομέων.

Χρησιμοποιήστε αυτό το φίλτρο όταν μια ενσωμάτωση χρειάζεται να προσθέσει αξιόπιστο πλαίσιο σε έναν SSO σύνδεσμο κυρίαρχου tenant ή να αντικαταστήσει το URL μεσολαβητή, διατηρώντας παράλληλα την επικύρωση token του Ultimate Multisite.

## Παράμετροι {#parameters}

| Όνομα | Τύπος | Περιγραφή |
|------|------|-------------|
| $sso_url | `string` | Παραγόμενο SSO URL. |
| $user | `WP_User` | Χρήστης που θα πιστοποιηθεί από την επίσκεψη SSO. |
| $site_id | `int` | ID ιστότοπου-στόχου για την επίσκεψη. |
| $redirect_to | `string` | URL προορισμού μετά την επιτυχημένη επικύρωση SSO. |

### Από την έκδοση {#since}

- 2.13.0

### Πηγή {#source}

Ορίζεται στο `inc/sso/class-sso.php`.


## Επιστρέφει {#returns}

Φιλτραρισμένο SSO URL.
