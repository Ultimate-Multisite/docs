# Filter: wu_current_set_site {#filter-wucurrentsetsite}

## Παράμετροι {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $site | `\WP_Ultimo\Models\Site` | Το τρέχον site που θέλετε να ορίσετε. |
| $ | `self` | Η στιγμιαία (instance) της κλάσης Current. |
