# Filter: wu_password_require_number

Ελέγχει αν απαιτούνται αριθμοί στα κωδικούς πρόσβασης.

## Παράμετροι

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | Αν απαιτούνται αριθμοί. Προεπιλεγμένο `true` όταν επιβάλλονται κανόνες. |
| $defender_active | `bool` | Εάν είναι ενεργό το Defender Pro Strong Password. |

### Από

- 2.4.0
### Πηγή

Ορίζεται στο [`inc/class-scripts.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-scripts.php#L573) στην γραμμή 573.
