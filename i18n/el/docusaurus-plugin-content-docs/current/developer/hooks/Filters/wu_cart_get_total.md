# Filter: wu_cart_get_total

Φιλτράρει το συνολικό ποσό του «αρχικού ποσού».

## Παράμετροι {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $total | `float` | Το συνολικό ποσό που πρέπει να πληρωθεί σήμερα. |
| $this | `\WP_Ultimo\Checkout\Cart` | Το αντικείμενο του καλαθιού. |

### Πηγή {#source}

Ορίζεται στο [`inc/checkout/class-cart.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-cart.php#L2141) στην γραμμή 2141.
