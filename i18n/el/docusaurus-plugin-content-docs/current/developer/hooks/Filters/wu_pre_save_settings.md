# Filter: wu_pre_save_settings

Επιτρέπει στους developers να φιλτράρουν τις ρυθμίσεις πριν από την αποθήκευση, μέσω του Ultimate Multisite.

## Παράμετροι

| Name | Type | Description |
|------|------|-------------|
| $settings | `array` | Οι ρυθμίσεις που πρόκειται να αποθηκευτούν. |
| $settings_to_save | `array` | Οι νέες ρυθμίσεις που θα προστεθούν. |
| $saved_settings | `array` | Οι τρέχουσες αποθηκευμένες ρυθμίσεις. |

### Από

- 2.0.18
### Πηγή

Ορίζεται στο [`inc/class-settings.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-settings.php#L296) στην γραμμή 296
