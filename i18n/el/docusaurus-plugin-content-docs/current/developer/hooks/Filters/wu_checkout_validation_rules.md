# Φίλτρο: wu_checkout_validation_rules

Επιτρέπει στους developers plugins να φιλτράρουν τους κανόνες επικύρωσης.

## Παράμετροι {#parameters}

| Όνομα | Τύπος | Περιγραφή |
|------|------|-------------|
| $validation_rules | `array` | Οι κανόνες επικύρωσης που θα χρησιμοποιηθούν. |
| $checkout | `\Checkout` | Η κλάση checkout. |
