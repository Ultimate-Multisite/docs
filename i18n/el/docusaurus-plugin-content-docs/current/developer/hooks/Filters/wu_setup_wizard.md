# Φίλτρο: wu_setup_wizard

Επιτρέπει στους developers να προσθέτουν επιπλέον βήματα στον setup wizard.

## Παράμετροι

| Όνομα | Τύπος | Περιγραφή |
|------|------|-------------|
| $sections | `array` | Οι τρέχουσες ενότητες. |
| $is_migration | `bool` | Αν πρόκειται για μια μεταφορά (migration) ή όχι. |
| $wizard | `object` | Η τρέχουσα οντότητα (instance). |

### Από

- 2.0.0
### Πηγή

Ορίζεται στο [`inc/admin-pages/class-setup-wizard-admin-page.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/admin-pages/class-setup-wizard-admin-page.php#L525) στην γραμμή 525
