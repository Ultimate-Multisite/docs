# Filter: wu_add_new_domain_modal_fields {#filter-wuaddnewdomainmodalfields}

Φιλτράρει τα πεδία που εμφανίζονται στο modal για την προσθήκη νέου domain.

Επιτρέπει στους add-ons (π.χ. Domain Seller) να τροποποιήσουν ή να αντικαταστήσουν τα πεδία εγγραφής domain.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $fields | `array` | Τα πεδία του φόρμας. |

### Since {#since}

- 2.1.0
