# Filter: wu_current_set_membership

Επιτρέπει στους developers να τροποποιήσουν τη βασική συμπεριφορά και να ορίσουν τη συνδρομή (membership) που είναι ενεργή με διαφορετικό τρόπο.

## Παράμετροι {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $membership | `\WP_Ultimo\Models\Membership` | Η συνδρομή που θέλετε να οριστεί ως ενεργή. |
| $ | `self` | Το στιγμιότυπο (instance) της κλάσης Current. |

- 2.0.18
