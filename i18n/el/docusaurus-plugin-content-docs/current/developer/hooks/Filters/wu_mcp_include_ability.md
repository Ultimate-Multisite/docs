# Φίλτρο: wu_mcp_include_ability {#filter-wumcpincludeability}

Φίλτρο για να υπερρίψετε την απόφαση συμπερίληψης ικανότητας (ability inclusion decision) του MCP.

## Παράμετροι {#parameters}

| Όνομα | Τύπος | Περιγραφή |
|------|------|-------------|
| $include | `bool` | Αν πρέπει να συμπεριληφθεί η ικανότητα. |
| $ability_id | `string` | Ο αναγνωριστικός κωδικός (ID) της ικανότητας. |

### Από έκδοση {#since}

- 2.4.8
