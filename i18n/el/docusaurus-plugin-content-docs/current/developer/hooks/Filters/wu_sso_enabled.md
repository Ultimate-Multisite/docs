# Φίλτρο: wu_sso_enabled

Ενεργοποιεί/Απενεργοποιεί τη δυνατότητα single-sign-on (SSO) σε διαφορετικά domains.

Μπορείτε να χρησιμοποιήσετε αυτό το φίλτρο για να απενεργοποιήσετε εντελώς το single-sign-on, ή να το ενεργοποιήσετε μόνο υπό συγκεκριμένες συνθήκες.

## Παράμετροι

| Όνομα | Τύπος | Περιγραφή |
|------|------|-------------|
| $enabled | `bool` | Θα ενεργοποιηθεί το SSO; True για ενεργοποίηση, false για απενεργοποίηση. |

### Από

- 2.0.11
### Πηγή

Ορίζεται στο [`inc/sso/class-sso.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/sso/class-sso.php#L110) στην γραμμή 110


## Επιστρέφει
Αν το SSO είναι ενεργοποιημένο ή όχι.
