# Φίλτρο: wu_checkout_validation_aliases

Επιτρέπει στους developers plugins να προσθέτουν προσαρμοσμένα aliases στον επικυρωτή φόρμας.

## Παράμετροι {#parameters}

| Όνομα | Τύπος | Περιγραφή |
|------|------|-------------|
| $validation_aliases | `array` | Το array με το id => alias. |
| $checkout | `\Checkout` | Η κλάση checkout. |

- 2.1
