# Filter: wp_ultimo_redirect_url_after_signup {#filter-wpultimoredirecturlaftersignup}

Ορισμός της URL που ανακατευθύνει.

Πρόκειται για ένα παρωχημένο (legacy) filter. Μερικά από τα παραμέτρων που περνούν δεν είναι πλέον διαθέσιμα, όπως το $site_id.

### Από το {#since}

- 1.1.3: Επιτρέπει στους developers να φιλτράρουν την URL που ανακατευθύνει.
