# Φίλτρο: wu_password_require_uppercase

Ελέγχει αν απαιτούνται κεφαλαία γράμματα στις κωδικούς πρόσβασης.

## Παράμετροι

| Όνομα | Τύπος | Περιγραφή |
|------|------|-------------|
| $require | `bool` | Αν απαιτούνται κεφαλαία. Το προεπιλεγμένο είναι `true` όταν επιβάλλονται κανόνες. |
| $defender_active | `bool` | Αν είναι ενεργό το Defender Pro Strong Password. |

### Από

- 2.4.0
### Πηγή

Ορίζεται στο [`inc/class-scripts.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-scripts.php#L553) στην γραμμή 553
