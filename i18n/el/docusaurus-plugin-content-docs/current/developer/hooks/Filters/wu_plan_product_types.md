# Filter: wu_plan_product_types

Φιλτράρει τους τύπους προϊόντων που πρέπει να αντιμετωπίζονται ως πακέτα (plans).

Αυτό το φίλτρο επιτρέπει στους add-ons να καταχωρούν επιπλέον τύπους προϊόντων που πρέπει να αναγνωρίζονται ως πακέτα (plans) κατά τη διαδικασία επικύρωσης και διαχωρισμού.

## Παράμετροι {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | Πίνακας (Array) των τύπων προϊόντων που πρέπει να αντιμετωπίζονται ως πακέτα. |

### Από {#since}

- 2.3.0

Ορίζεται στο [`inc/functions/product.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/product.php#L196) στην γραμμή 196
