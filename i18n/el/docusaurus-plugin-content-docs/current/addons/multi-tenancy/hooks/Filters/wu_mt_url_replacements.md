# Φίλτρο: wu_mt_url_replacements

Φιλτράρει τα ζεύγη αντικατάστασης URL.

## Παράμετροι {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $replacements | `array` | Τα τρέχοντα ζεύγη αντικατάστασης. |
| $old_url | `string` | Το παλιό URL. |
| $new_url | `string` | Το νέο URL. |
| $blog_id | `int` | Το ID του blog. |
