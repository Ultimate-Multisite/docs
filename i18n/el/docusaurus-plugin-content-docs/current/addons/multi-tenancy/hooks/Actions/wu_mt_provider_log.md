# Action: wu_mt_provider_log

Εκτρέπεται αυτό το action όταν ένας πάροχος καταγράφει ένα μήνυμα.

## Παράμετροι

| Όνομα | Τύπος | Περιγραφή |
|------|------|-------------|
| $message | `string` | Το μήνυμα καταγραφής. |
| $level | `string` | Το επίπεδο καταγραφής. |
| $context | `array` | Επιπλέον πλαίσιο/πληροφορίες. |

### Από

- 1.0.0
### Πηγή

Ορίζεται στο [`inc/providers/class-base-provider.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-multi-tenancy/blob/main/inc/providers/class-base-provider.php#L400) στην γραμμή 400
