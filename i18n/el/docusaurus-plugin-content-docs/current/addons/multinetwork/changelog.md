# Κατάλογος Αλλαγών Multi-Network

### 1.3.0 {#130}
* Νέο: Προεπισκόπηση προτύπων δικτύου — περιηγηθείτε και προεπισκόπηση τα πρότυπα δικτύου με ένα ζωντανό πάνελ πριν την αγορά.
* Νέο: Επιλογή προτύπου δικτύου στο checkout — οι πελάτες μπορούν να επιλέξουν ένα πρότυπο δικτύου κατά τη διαδικασία ολοκλήρωσης (checkout).
* Νέο: Magic link SSO μεταξύ δικτύων — οι πελάτες μπορούν να έχουν πρόσβαση σε υποδίκτυα αδιάλειπτα μέσω ταυτοποίησης magic-link.
* Νέο: Αντιγραφή μέσων δικτύου — τα μέσα του site αντιγράφονται σωστά όταν αντιγράφετε δίκτυα από ένα πρότυπο.
* Διόρθωση: Η αντιγραφή δικτύου λειτουργεί τώρα χωρίς το add-on πολυενοικιακότητας (multi-tenancy); τα προνόμια super admin αποκατασταλούν μετά την αντιγραφή.
* Διόρθωση: Η δημιουργία δικτύου είναι τώρα συμβατή με το WooCommerce και όλες τις πύλες πληρωμής.
* Διόρθωση: Ο αριθμός θύρας περιλαμβάνεται τώρα στη δημιουργία fallback domain.
* Διόρθωση: Το CSS του επιλογέα προτύπων μεταφέρθηκε σε ένα enqueued stylesheet για σωστή εμφάνιση σε όλα τα θέματα.
* Διόρθωση: Υπάρχει προστασία ενάντια στην κλειδί `network_id` που είναι μη καθορισμένη στο scope της site query.
* Διόρθωση: Η επαλήθευση δικαιωμάτων δικτύου πριν από το context switching αποτρέπει τα σφάλματα ιδιοτήτων πίνακα `wpdb`.
