# Action: add_network

Εκτελείται αφού έχει προστεθεί ένα νέο network.

## Παράμετροι

| Όνομα | Τύπος | Περιγραφή |
|------|------|-------------|
| $new_network_id | `int` | Το ID του προστιθέμενου network. |
| $r | `array` | Το πλήρες συνδεδεμένο array με τα arguments του network. |

### Από

- 1.3.0
### Πηγή

Ορίζεται στο [`inc/functions/network.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-multinetwork/blob/main/inc/functions/network.php#L1304) στην γραμμή 1304
