# Action: update_network

Εκτελείται αφού ενημερωθεί ένα υπάρχον δίκτυο.

## Παράμετροι {#parameters}

| Όνομα | Τύπος | Περιγραφή |
|------|------|-------------|
| $network_id | `int` | Το ID του προστιθέμενου δικτύου. |
| $args | `array` | Συσχετιστικό πίνακα με τα επιχειρήματα του δικτύου. |
