# Multi-Network

Το Ultimate Multisite: Multinetwork addon επεκτείνει την πλατφόρμα Ultimate Multisite σας με ισχυρές λειτουργίες multi-network. Αυτό το addon επιτρέπει στους πελάτες σας να αγοράζουν και να διαχειρίζονται ολόκληρες شبكة WordPress multisite, και όχι μόνο μεμονωμένα sites.

## Βασικά Χαρακτηριστικά

* **Network Provisioning**: Δημιουργία και αυτόματη διαμόρφωση ολόκληρων شبكة WordPress multisite για τους πελάτες σας.
* **Network Templates**: Επιτρέπετε στους πελάτες να περιηγούνται, να προεπισκόπηση και να επιλέγουν μια έτοιμη network template πριν την εγγραφή.
* **Network Management**: Πλήρεις δυνατότητες διαχείρισης για περιβάλλοντα multi-network.
* **Isolated Networks**: Κάθε network πελάτη λειτουργεί ανεξάρτητα με τα δικά του جداول βάσεων δεδομένων και τη δική του διαμόρφωση.
* **Cross-Network SSO**: Παρέχετε στους πελάτες απρόσκοπτη πρόσβαση στις υπο-شبකές τους με ταυτοποίηση magic-link.
* **Scalable Architecture**: Διαχειρίζεστε πολλαπλές شبකές αποτελεσματικά με βελτιστοποιημένη κατανομή πόρων.
* **Integration**: Ενσωματώνεται απρόσκοπτα με τα συστήματα συνδρομών και χρέωσης του Ultimate Multisite.

## Περιπτώσεις Χρήσης

* **White Label Hosting**: Παρέχετε στους πελάτες τους δικούς τους شبكة multisite με τη δική τους επωνυμία.
* **Agency Solutions**: Επιτρέπετε στις εταιρείες να διαχειρίζονται πολλαπλές شبකές πελατών από μια ενιαία πλατφόρμα.
* **Enterprise Deployments**: Υποστηρίζετε μεγάλες οργανώσεις με σύνθετες απαιτήσεις multi-network.
* **Education Platforms**: Δημιουργείτε ξεχωριστές شبකές για διαφορετικά σχολεία ή τμήματα.

## Απαιτήσεις

* Ultimate Multisite 2.0.0 ή νεότερη
* Εγκατάσταση WordPress Multisite
* PHP 7.4 ή νεότερο
* MySQL 5.6 ή νεότερο

## Εγκατάσταση

1. Ανεβάστε τα αρχεία του plugin στον κατάλογο `/wp-content/plugins/ultimate-multisite-multinetwork`
2. Ενεργοποιήστε το plugin μέσω της οθόνης 'Plugins' στο WordPress
3. Διαμορφώστε τις ρυθμίσεις multi-network μέσω Ultimate Multisite → Settings

## Network Templates

Η έκδοση 1.3.0 προσθέτει μια ροή ανακάλυψης και επιλογής templates για τους πελάτες που αγοράζουν νέες شبකές. Χρησιμοποιήστε τα network templates όταν θέλετε κάθε νέα network πελάτη να ξεκινήσει από μια προετοιμασμένη δομή, σχεδιασμό, βιβλιοθήκη περιεχομένου ή ρύθμιση ειδική για το vertical.

### Template previewer

Ο προεπισκοπιζέας network templates επιτρέπει στους πελάτες να περιηγούνται στις διαθέσιμες templates πριν την ολοκλήρωση της αγοράς. Κάθε template μπορεί να ανοίξει σε ένα live preview panel, ώστε οι πελάτες να συγκρίνουν τα layouts και το περιεχόμενο πριν αγοράσουν μια network.

Χρησιμοποιήστε τον προεπισκοπιζέα για να βοηθήσετε τους πελάτες:

* Να εξετάσουν το görünüm και τη δομή κάθε διαθέσιμη network template.
* Να συγκρίνουν πολλαπλές templates χωρίς να απομακρύνουν από τη ροή αγοράς.
* Να επιλέξουν το template που ταιριάζει καλύτερα στην περίπτωση χρήσης τους πριν την εγγραφή.

Κρατήστε τα ονόματα και τις περιγραφές των templates φιλικά προς τον πελάτη. Ο προεπισκοπιζέας είναι πιο χρήσιμος όταν οι πελάτες μπορούν να καταλάβουν την προορισμένη περίπτωση χρήσης, όπως شبකές εταιρειών, شبකές σχολείων, شبකές franchise ή πακέτα white-label hosting.

### Template picker at checkout

Το template picker μπορεί να προστεθεί στη ροή checkout εγγραφής ως πεδίο. Όταν ρυθμιστεί, οι πελάτες επιλέγουν το network template που θέλουν κατά την ολοκλήρωση της αγοράς, και το επιλεγμένο template χρησιμοποιείται όταν δημιουργείται η νέα τους network.

Για να χρησιμοποιήσετε το picker σε μια ροή checkout:

1. Προετοιμάστε τις شبකές που θέλετε να προσφέρετε ως templates.
2. Ενεργοποιήστε το πεδίο επιλογής Multinetwork template στη σχετική φόρμα εγγραφής ή checkout.
3. Επιλέξτε ποιες templates πρέπει να είναι διαθέσιμες για αυτή τη ροή checkout.
4. Δημοσιεύστε τη ροή checkout και δοκιμάστε μια εγγραφή για να επιβεβαιώσετε ότι το επιλεγμένο template εφαρμόζεται στη νέα network.

Εάν μια ροή checkout προσφέρει μόνο ένα προϊόν network, το picker μπορεί ακόμα να είναι χρήσιμο ως βήμα καθοδήγησης onboarding. Για multi-plan funnels, εμφανίστε μόνο τα templates που είναι συμβατά με το επιλεγμένο προϊόν ή plan.

## Cross-Network Magic Link SSO

Το Cross-network magic link SSO επιτρέπει στους πελάτες να μετακινούν μεταξύ της κύριας περιοχής λογαριασμού τους και των υπο-شبකών χωρίς να χρειάζεται να συνδεθούν ξανά χειροκίνητα. Όταν ένας πελάτης ακολουθεί έναν cross-network access link, το Ultimate Multisite δημιουργεί ένα magic link με περιορισμένο χρόνο και ταυτοποιεί τον πελάτη στην προοριστική network.

Αυτό βελτιώνει την εμπειρία για τους πελάτες που διαχειρίζονται περισσότερες από μία شبكة ή που χρειάζεται να έχουν πρόσβαση σε νέες κλωνοποιημένες network templates αμέσως μετά την αγορά.

Ελέγχος για τον διαχειριστή:

* Επιβεβαιώστε ότι το Multinetwork addon είναι ενεργό σε όλη την εγκατάσταση WordPress multisite.
* Βεβαιωθείτε ότι οι πελάτες έχουν την σωστή πρόσβαση στη network πριν τους προσφερθούν cross-network links.
* Διατηρήστε τις network domains και τα mapped domains σωστά ρυθμισμένα ώστε τα magic links να οδηγούν στην προοριστική network.
* Δοκιμάστε το dashboard του πελάτη και οποιονδήποτε κουμπί πρόσβασης μετά το checkout μετά την αλλαγή των ρυθμίσεων network, domain ή checkout.

Τα magic links πρέπει να αντιμετωπίζονται ως links σύνδεσης. Είναι σχεδιασμένα για απρόσκοπτη πρόσβαση του πελάτη και δεν πρέπει να μεταφέρονται ή να επαναχρησιμοποιούνται εκτός της προορισμένης συνεδρίας.

## Συχνές Ερωτήσεις

### Τι είναι το multinetwork;

Το Multinetwork είναι ένα χαρακτηριστικό του WordPress που σας επιτρέπει να εκτελείτε πολλαπλές ανεξάρτητες شبකές multisite σε μια ενιαία εγκατάσταση WordPress.

### Πώς διαφέρει αυτό από το κανονικό multisite;

Το κανονικό multisite επιτρέπει πολλαπλά sites εντός μιας network. Το Multinetwork επιτρέπει πολλαπλές networks, κάθε μία με τα δικά της sites, όλα εντός μιας εγκατάστασης WordPress.

### Απαιτείται ειδική ρύθμιση server;

Ναι, το multinetwork απαιτεί συγκεκριμένη ρύθμιση και διαμόρφωση server. Παρακαλούμε συμβουλευτείτε την τεκμηρίωση για λεπτομερείς οδηγίες ρύθμισης.
