# Loco Translate

Το Ultimate Multisite: Loco Translate Integration addon φέρνει δυνατότητες μετάφρασης ειδικές για το site στο δίκτυο WordPress multisite σας. Αν χρειάστηκαν ποτέ διαφορετικά sites στο δίκτυό σας να έχουν διαφορετικές μεταφράσεις του ίδιου plugin ή theme, αυτό το addon είναι η τέλεια λύση.

**Τι κάνει αυτό το Addon**

Αυτό το addon ενσωματώνει απρόσκοπτα το Ultimate Multisite με το δημοφιλές plugin Loco Translate, επιτρέποντας σε κάθε υποσite του δικτύου σας να διατηρεί τις δικές του προσαρμοσμένες μεταφράσεις. Είτε διατηρείτε μια πλατφόρμα με πολλαπές περιοχές, εξυπηρετώντας διαφορετικές κοινότητες języków, είτε απλώς χρειάζεστε ευελιξία μετάφρασης σε όλο το δίκτυό σας, αυτό το addon το κάνει εύκολο.

**Ιδανικό για:**

* Επιχειρήσεις με πολλαπλές περιοχές και διαφορετικές ανάγκες τοπικοποίησης
* Εκπαιδευτικά ιδρύματα που εξυπηρετούν ποικίλες κοινότητες
* Πλατφόρμες SaaS που προσφέρουν υποστήριξη πολλαπλών γλωσσών
* Web agencies που διαχειρίζονται πολλαπλά sites πελατών
* Οποιοδήποτε δίκτυο multisite απαιτεί ευελιξία μετάφρασης

## Εγκατάσταση {#installation}

1. Βεβαιωθείτε ότι το Ultimate Multisite είναι εγκατεστημένο και ενεργοποιημένο σε επίπεδο δικτύου.
2. Εγκαταστήστε και ενεργοποιήστε το plugin Loco Translate.
3. Ανεβάστε αυτό το addon στον φάκελο plugins σας.
4. Ενεργοποιήστε το addon Loco Translate Integration σε επίπεδο δικτύου.
5. Ξεκινήστε να δημιουργείτε μεταφράσεις ειδικές για το site μέσω του Loco Translate σε οποιοδήποτε υποσite.

Το μόνο αυτό! Η ενσωμάτωση λειτουργεί αυτόματα χωρίς οποιαδήποτε επιπλέον ρύθμιση.

## Συχνές Ερωτήσεις {#frequently-asked-questions}

## Πρέπει να ρυθμίσω κάτι μετά την εγκατάσταση; {#do-i-need-to-configure-anything-after-installation}

Όχι! Η ενσωμάτωση είναι αυτόματη. Μόλις ενεργοποιηθούν τόσο το addon όσο και το Loco Translate, τα υποσites μπορούν αμέσως να αρχίσουν να δημιουργούν τις δικές τους προσαρμοσμένες μεταφράσεις.

## Τι συμβαίνει αν ένα υποσite δεν δημιουργήσει προσαρμοσμένες μεταφράσεις; {#what-happens-if-a-subsite-doesnt-create-custom-translations}

Το site θα χρησιμοποιήσει αυτόματα τις μεταφράσεις σε επίπεδο δικτύου ως εφεδρική λύση (fallback). Τίποτα δεν καταστρέφεται και όλα λειτουργούν απρόσκοπτα.

## Μπορούν τα υποσites να έχουν πρόσβαση στις μεταφράσεις σε επίπεδο δικτύου; {#can-subsites-access-network-wide-translations}

Οι απλοί site admins βλέπουν και επεξεργάζονται μόνο τις μεταφράσεις του δικού τους site. Οι super admins μπορούν να έχουν πρόσβαση τόσο στις μεταφράσεις σε επίπεδο δικτύου όσο και στις μεταφράσεις ειδικές για το site, ανάλογα με το site που βλέπουν.

## Λειτουργεί αυτό τόσο με plugins όσο και με themes; {#does-this-work-with-both-plugins-and-themes}

Ναι! Οι μεταφράσεις ειδικές για το site λειτουργούν τόσο για τα plugins όσο και για τα themes του WordPress.

## Θα επιβραδύνει αυτό το δίκτυο multisite μου; {#will-this-slow-down-my-multisite-network}

Όχι. Το addon χρησιμοποιεί αποδοτική φόρτωση αρχείων που ελέγχει για μεταφράσεις ειδικές για το site μόνο όταν είναι απαραίτητο. Δεν υπάρχει εμφανής επιβάρυνση στην απόδοση.

## Μπορώ να αντιγράψω μεταφράσεις σε επίπεδο δικτύου σε ένα συγκεκριμένο site ως σημείο εκκίνησης; {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

Ναι! Ενώ επεξεργάζεστε τις μεταφράσεις στο Loco Translate, μπορείτε εύκολα να ξεκινήσετε με τις μεταφράσεις σε επίπεδο δικτύου και να τις προσαρμόσετε για κάθε site.

## Λειτουργεί αυτό με το WooCommerce και άλλα plugins; {#does-this-work-with-woocommerce-and-other-plugins}

Απολύτως! Κάποιοδήποτε plugin ή theme χρησιμοποιεί τις τυπικές συναρτήσεις μετάφρασης του WordPress θα λειτουργήσει με τις μεταφράσεις ειδικές για το site.
