# Ενσωμάτωση WooCommerce

Η Ενσωμάτωση WooCommerce λειτουργεί με πολύ παρόμοιο τρόπο με τον τρόπο που λειτουργεί η Μη αυτόματη πύλη μας. Προσφέρει μεγάλη ευελιξία στους χρήστες σας, επιτρέποντάς τους να επιλέξουν οποιαδήποτε από τις φόρμες πληρωμής που είναι διαθέσιμες στην εγκατάσταση WooCommerce σας. Και είναι απλό: μόλις λήξει η περίοδος χρέωσης του χρήστη, η συνδρομή του χρήστη τίθεται σε αναστολή. Ταυτόχρονα, θα δημιουργηθεί μια παραγγελία στην εγκατάσταση WooCommerce του κύριου ιστότοπου του δικτύου. Στη συνέχεια, ο χρήστης λαμβάνει ένα τιμολόγιο στο email του, με έναν σύνδεσμο για να πραγματοποιήσει την πληρωμή. Μόλις ληφθεί η πληρωμή, η συνδρομή ανανεώνεται.

## Δυνατότητες

- Εύκολη ενσωμάτωση μεταξύ Ultimate Multisite και WooCommerce
- Διαθέσιμες αρκετές μέθοδοι πληρωμής
- Αυτοματοποιημένη επαναλαμβανόμενη χρέωση με χρήση του ενσωματωμένου πυρήνα συνδρομών, με προαιρετικές βελτιώσεις από το WooCommerce Subscriptions plugin
- Τα σύνολα του checkout περιλαμβάνουν τους φόρους του Ultimate Multisite ως ξεχωριστές, μη φορολογητέες γραμμές χρέωσης WooCommerce, ώστε οι πελάτες να μπορούν να ελέγχουν τα ποσά φόρων πριν από την πληρωμή

## Τεκμηρίωση

**Ρύθμιση της Ενσωμάτωσης WooCommerce**

**Επέκταση των Επιλογών Πληρωμής σας με το Ultimate Multisite: Ενσωμάτωση WooCommerce**

## Εγκατάσταση

1. Ανεβάστε το 'ultimate-multisite-woocommerce' στον κατάλογο '/wp-content/plugins/'
2. Ενεργοποιήστε το plugin μέσω του μενού 'Plugins' στο WordPress
3. Ακολουθήστε τον οδηγό βήμα προς βήμα για να ρυθμίσετε το plugin

## Οδηγός Χρήσης

_**ΣΗΜΑΝΤΙΚΗ ΣΗΜΕΙΩΣΗ: Αυτό το άρθρο αναφέρεται στην έκδοση 2.x του Ultimate Multisite.**_

_**ΠΡΟΣΟΧΗ:** Ultimate Multisite: Η Ενσωμάτωση WooCommerce απαιτεί το WooCommerce να είναι ενεργοποιημένο τουλάχιστον στον κύριο ιστότοπό σας._

Κατανοούμε ότι το _Stripe_ και το _PayPal_ δεν είναι διαθέσιμα σε ορισμένες χώρες, κάτι που περιορίζει ή εμποδίζει τους χρήστες του Ultimate Multisite να χρησιμοποιούν αποτελεσματικά το plugin μας. Έτσι δημιουργήσαμε ένα **add-on για ενσωμάτωση του _WooCommerce_**.

Το _WooCommerce_ είναι ένα πολύ δημοφιλές eCommerce plugin. Προγραμματιστές σε όλο τον κόσμο δημιούργησαν add-ons για να ενσωματώσουν διαφορετικές πύλες πληρωμών σε αυτό. Το αξιοποιήσαμε για να επεκτείνουμε τις πύλες πληρωμών που μπορείτε να χρησιμοποιήσετε στο σύστημα χρέωσης του Ultimate Multisite.

## Ρύθμισή του

Πρώτα, πρέπει να εγκαταστήσετε το add-on Ενσωμάτωσης WooCommerce, το οποίο είναι **δωρεάν για αδειοδοτημένους χρήστες**. Για να το εγκαταστήσετε, κάντε κλικ στο Ultimate Multisite στο Dashboard του υπερδιαχειριστή σας και επιλέξτε **Add-ons**.

![Σελίδα Add-ons](/img/admin/addons-page.png)

Εδώ μπορείτε να βρείτε όλα τα add-ons του Ultimate Multisite. Κάντε κλικ στο add-on **Ultimate Multisite: WooCommerce Integration**.

![Σελίδα Add-ons που εμφανίζει την Ενσωμάτωση WooCommerce](/img/admin/addons-page.png)

Θα εμφανιστεί ένα παράθυρο με τις λεπτομέρειες του add-on. Απλώς κάντε κλικ στο **Install Now** *.*

<!-- Screenshot unavailable: WooCommerce Integration addon install popup with Install Now button -->

Αφού ολοκληρωθεί η εγκατάσταση, θα ανακατευθυνθείτε στη σελίδα plugins. Εδώ, απλώς κάντε κλικ στο **Network Activate** και το add-on _WooCommerce_ θα ενεργοποιηθεί στο δίκτυό σας.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for WooCommerce Integration -->

Αφού το ενεργοποιήσετε, αν ακόμη δεν έχετε εγκαταστήσει και ενεργοποιήσει το plugin WooCommerce στον ιστότοπό σας, **θα λάβετε μια υπενθύμιση**.

<!-- Screenshot unavailable: Admin notice reminding to install and activate WooCommerce plugin -->

Μόλις εγκατασταθεί, θα δείτε μια πρόσθετη επιλογή κάτω από τις ρυθμίσεις πληρωμών του Ultimate Multisite. Στη συνέχεια, μπορείτε να **ενεργοποιήσετε** την επιλογή _WooCommerce_ από εκεί.

![Ρυθμίσεις πύλης πληρωμών Ultimate Multisite με ενεργοποιημένο το WooCommerce](/img/addons/woocommerce-payment-gateway.png)

Μπορείτε να διαμορφώσετε ορισμένες βασικές ρυθμίσεις, όπως το εμφανιζόμενο όνομα που θα εμφανίζεται στο front end και οδηγίες για να καθοδηγήσετε τους χρήστες σας στη ροή checkout τους. Έπειτα μπορείτε να μεταβείτε στις ρυθμίσεις _WooCommerce_ του **Κύριου ιστότοπού σας** για να ενεργοποιήσετε και να διαμορφώσετε την πύλη πληρωμών που προτιμάτε να χρησιμοποιήσετε.

### Γραμμές χρέωσης φόρου checkout

Όταν ένα checkout του Ultimate Multisite περιλαμβάνει φόρους, η ενσωμάτωση προσθέτει κάθε ποσό φόρου του Ultimate Multisite στο checkout του WooCommerce ως ξεχωριστή, μη φορολογητέα γραμμή χρέωσης. Αυτό διατηρεί το συνολικό ποσό φόρου ορατό πριν από την πληρωμή, αποτρέποντας παράλληλα το WooCommerce από το να φορολογήσει ξανά τη χρέωση φόρου.

## Πώς να διαχειριστείτε την πληρωμή

Η Ενσωμάτωση WooCommerce λειτουργεί με πολύ παρόμοιο τρόπο με τον τρόπο που λειτουργεί η **Μη αυτόματη πύλη** μας. Το μειονέκτημα είναι ότι κάθε πληρωμή πρέπει να πληρώνεται χειροκίνητα στο τέλος της περιόδου χρέωσης. Δείτε πώς λειτουργεί η ροή:

  * Μόλις λήξει ο κύκλος χρέωσης του χρήστη, η συνδρομή τίθεται **σε αναστολή**. Ταυτόχρονα, θα δημιουργηθεί μια παραγγελία στην εγκατάσταση _WooCommerce_ στον κύριο ιστότοπο του δικτύου σας.

  * Οι χρήστες στη συνέχεια λαμβάνουν ένα τιμολόγιο στο email τους, με έναν σύνδεσμο για να πραγματοποιήσουν την πληρωμή (ο σύνδεσμος εμφανίζεται επίσης στο ιστορικό χρεώσεών τους από το Dashboard του υποϊστότοπού τους). Αυτός ο σύνδεσμος ανακατευθύνει τον χρήστη στο _**WooCommerce**_ **checkout** , όπου οι χρήστες θα μπορούν να επιλέξουν μία από τις **διαθέσιμες πύλες _WooCommerce_** προκειμένου να επεξεργαστούν την πληρωμή τους.

  * Μόλις ληφθεί η πληρωμή, η συνδρομή **ανανεώνεται** , η πληρωμή καταγράφεται στο Ultimate Multisite και η παραγγελία ορίζεται ως ολοκληρωμένη.

### Μερικές Σημαντικές Σημειώσεις:

  * Η προεπιλεγμένη λειτουργικότητα του Ultimate Multisite εξακολουθεί να λειτουργεί με αυτή τη μορφή ενσωμάτωσης. Για παράδειγμα, η **έκδοση επιστροφών χρημάτων** από την οθόνη **Διαχείριση Συνδρομής** θα επικοινωνήσει με το WooCommerce για να εκδώσει την επιστροφή χρημάτων χρησιμοποιώντας την επιλεγμένη φόρμα πληρωμής.

  * Ultimate Multisite: Η Ενσωμάτωση WooCommerce βασίζεται στο WP-Cron για να λειτουργήσει. Αυτό λειτουργεί πολύ καλά τις περισσότερες φορές, αλλά αν ο ιστότοπος έχει χαμηλή επισκεψιμότητα, τότε το cron δεν ενεργοποιείται για να ελέγξει προγραμματισμένα συμβάντα και το συμβάν χάνεται, κάτι που μπορεί να προκαλέσει καθυστερήσεις εκτέλεσης.

## WooCommerce Subscriptions Plugin

Η χειροκίνητη έναρξη μιας πληρωμής κάθε μήνα δεν είναι ιδανική για τους τελικούς χρήστες, επειδή οι συνδρομές αναμένεται να επαναλαμβάνονται αυτόματα σε κάθε κύκλο χρέωσης. Το WooCommerce Integration add-on πλέον περιλαμβάνει τον βασικό κώδικα συνδρομών, επομένως τα επαναλαμβανόμενα προϊόντα χρησιμοποιούν πάντα συνδρομές για επαναλαμβανόμενη χρέωση.

**Δεν** χρειάζεται να αγοράσετε ή να εγκαταστήσετε το ξεχωριστό [WooCommerce Subscriptions plugin](https://woocommerce.com/products/woocommerce-subscriptions/) για να πουλάτε επαναλαμβανόμενα προϊόντα μέσω αυτής της ενσωμάτωσης. Αν αυτό το plugin αγοραστεί, εγκατασταθεί και είναι διαθέσιμο στον κύριο ιστότοπό σας, το Ultimate Multisite θα το χρησιμοποιήσει και το WooCommerce θα εμφανίσει πρόσθετες δυνατότητες διαχείρισης συνδρομών. Όταν δεν είναι διαθέσιμο, ο ενσωματωμένος πυρήνας συνδρομών εξακολουθεί να διαχειρίζεται τα επαναλαμβανόμενα προϊόντα.

![Ρυθμίσεις ενσωμάτωσης συνδρομής WooCommerce](/img/addons/woocommerce-subscription-integration.png)

Και αυτό είναι όλο. Οι τελικοί χρήστες σας δεν θα χρειάζεται πλέον να πραγματοποιούν χειροκίνητα μια πληρωμή στο τέλος του κύκλου χρέωσής τους για επαναλαμβανόμενα προϊόντα.

Εκτός από τη δημιουργία μιας παραγγελίας στον κύριο ιστότοπό σας κάτω από το _WooCommerce_, θα δημιουργηθεί επίσης μια συνδρομή για κάθε λογαριασμό.

### Συμπεριφορά αναστολής αποτυχημένης ανανέωσης

Από προεπιλογή, οι συνδρομές μέλους παραμένουν ενεργές ενώ το WooCommerce Subscriptions επεξεργάζεται το πρόγραμμα επαναλήψεων για αποτυχημένες πληρωμές. Αυτό διατηρεί το κανονικό παράθυρο χάριτος του WooCommerce Subscriptions, ώστε μια ανακτημένη ανανέωση να μπορεί να επαναφέρει τη συνδρομή και τη συνδρομή μέλους σε συγχρονισμό χωρίς να διακόψει πρόωρα την πρόσβαση.

Οι διαχειριστές που θέλουν αυστηρότερο έλεγχο πρόσβασης μπορούν να ενεργοποιήσουν το **Άμεση αναστολή συνδρομών μέλους σε αποτυχημένες ανανεώσεις** στις ρυθμίσεις WooCommerce Integration. Όταν αυτή η προαιρετική ρύθμιση είναι ενεργοποιημένη, το Ultimate Multisite αναστέλλει τη σχετική συνδρομή μέλους μόλις αποτύχει μια πληρωμή ανανέωσης, αντί να περιμένει να ολοκληρωθεί ο χειρισμός επαναλήψεων του WooCommerce Subscriptions.

Οι προγραμματιστές μπορούν να παρακάμψουν την ίδια απόφαση με το φίλτρο `wu_woo_suspend_on_payment_failure`. Οι σελίδες αναφοράς hook δημιουργούνται από τον πηγαίο κώδικα του add-on, επομένως αυτή η επισκόπηση αναφέρει μόνο το όνομα του φίλτρου για προσαρμοσμένες πολιτικές αναστολής.
