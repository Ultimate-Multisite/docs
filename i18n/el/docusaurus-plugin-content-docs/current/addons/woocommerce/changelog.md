# WooCommerce Integration Changelog

Version 2.0.6 - Released on 2026-01-16
* Βελτίωση: Περιλαμβάνει τα βασικά συνδρομητικά πακέτα (core subscriptions) στο addon. Δεν απαιτεί πλέον την επέκταση WooCommerce Subscriptions.

Version 2.0.5 - Released on 2026-01-09
* Βελτίωση: Φόρτωση μεταφράσεων από το glotpress API.
* Διόρθωση: Συμβαίνει ένα καταστροφικό σφάλμα (fatal error) σε ορισμένα page builders.
* Διόρθωση: Άπειρη ανακατεύθυνση (infinite redirect) όταν ο πελάτης είναι μέλος της κύριας ιστοσελίδας.

Version 2.0.4 - Released on 2025-11-14
* Προστέθηκε: Μεταφράσεις για πολύ περισσότερες γλώσσες.
* Αλλάχθηκε: Μετονομάστηκε σε Ultimate Multisite: Woocommerce Integration.
* Προστέθηκε: Συμβατότητα με Woocommerce 10.2.1.
* Προστέθηκε: Συμβατότητα με Woocommerce Subscriptions 7.7.0.
* Διόρθωση: Συμβατότητα με PHP 8.4
* Διόρθωση: Ανακατεύθυνση, αλλά όταν δεν υπάρχει σελίδα λογαρκαρίου WC.

Version 2.0.3 - Released on 2025-08-13
* Αλλάχθηκε: Ενεργοποιήθηκαν οι αυτόματες ενημερώσεις με το νέο marketplace.

Version 2.0.2 - Released on 2025-07-05
* Αλλάχθηκε: Μετονομάστηκε σε Multisite Ultimate: Woocommerce Integration.
* Προστέθηκε: Συμβατότητα με Woocommerce 9.8.1.
* Προστέθηκε: Συμβατότητα με Woocommerce Subscriptions 7.3.0.
* Διόρθωση: Ακύρωση συνδρομής από τον πελάτη.
* Διόρθωση: Καταστροφικό σφάλμα (fatal error) κατά τη χρήση του checkout block.
* Βελτίωση: Τώρα συμβατό με τα προσαρμοσμένα πίνακες παραγγεμιών υψηλής απόδοσης του Woocommerce.
* Διόρθωση: Η ακύρωση στο checkout του WooCommerce μπορεί ακόμα να κάνει upgrade σε μια सदस्यता.

Version 2.0.1 - Released on 2023-08-09

* Προστέθηκε: Συμβατότητα με Woocommerce 7.9.0.
* Προστέθηκε: Συμβατότητα με Woocommerce Subscriptions 5.3.0.
* Προστέθηκε: Υποστήριξη για ενημερώσεις μελών.
* Προστέθηκε: Ειδοποιήσεις για δοκιμαστικές περιόδους (trials) και τέλη ρύθμισης (setup fees) στο Woocommerce.
* Προστέθηκε: Αναγνώριση των προϊόντων Ultimate Multisite Woocommerce με ένα meta value.
* Προστέθηκε: Ενσωματώθηκε μια εφάπαξ διόρθωση για την επισήμανση όλων των προϊόντων Woocommerce που σχετίζονται με το Ultimate Multisite.
* Προστέθηκε: Αφαίρεση των προϊόντων που δημιουργήθηκαν από το Ultimate Multisite από τη λίστα του Woocommerce.
* Βελτίωση: Δημιουργήθηκε μια μη επαναλαμβανόμενη έκπτωση Woocommerce για εφαρμογή στο καλάθι.
* Βελτίωση: Επαναφορά της επαναλαμβανόμενης έκπτωσης στο προϊόν Woocommerce.
* Βελτίωση: Προστέθηκε μια ένδειξη επαναλαμβανόμενης έκπτωσης στο προϊόν Woocommerce.
* Βελτίωση: Εξασφάλιση του τύπου προϊόντος στο checkout.
* Διόρθωση: Διατήρηση της κατάστασης μέλους κατά τη διαδικασία downgrade.
* Διόρθωση: Έλεγχος αν υπάρχει συνδρομή για αποφυγή σφαλμάτων κατά τη διαδικασία ακύρωσης.
* Διόρθωση: Προσθήκη της ημερομηνίας έναρξης συνδρομής για χρήση στις συνδρομές του Woocommerce.
* Εσωτερικό: Υλοποιήθηκε μια νέα διαδικασία build για PHP 8.1.

Version 2.0.0 - Complete rewrite.

* Προστέθηκε: Μέθοδος διακόπησης της διαδικασίας (process cancellation method) για αφαίρεση της συνδρομής woo κατά την αλλαγή του gateway ή την ακύρωση της सदस्यता;
* Προστέθηκε: Handler για downgrade και upgrade των μελών;
* Βελτίωση: Φόρτωση εξαρτήσεων του woocommerce στο form ενημέρωσης πελάτη στα subsites για να επιτρέπεται η ενημέρωση του λογαρκαρίου;
* Βελτίωση: Διόρθωση φόρτωσης του Woocommerce cart αν δεν υπάρχει;
* Βελτίωση: Βεβαιωθεί ότι είμαστε στους πίνακες της κύριας ιστοσελίδας όταν επεξεργαζόμαστε ένα checkout;
* Βελτίωση: Κάντε το αίτημα ανανέωσης Ultimo βασισμένο στην αξία της παραγγελίας συνδρομής του Woocommerce και όχι στην τελευταία πληρωμή;
* Διόρθωση: Πλοήγιση στο link κουμπιού WU Membership;
* Διόρθωση: Στήσιμο της παραγγελίας Ultimo ως πληρωμένη όταν η ανανέωση της συνδρομής του Woocommerce έχει πληρωθεί;
* Build: Προσθήκη MPB ως builder;

* Διόρθωση: Αποτροπή δημιουργίας πολλαπλών προϊόντων όταν δεν είναι απαραίτητο;
