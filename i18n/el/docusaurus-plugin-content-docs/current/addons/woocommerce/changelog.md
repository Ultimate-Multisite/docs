# Αρχείο αλλαγών WooCommerce Integration {#woocommerce-integration-changelog}

Έκδοση 2.2.0 - Κυκλοφόρησε στις 2026-07-01
* Νέο: Τα ποσά φόρων του Ultimate Multisite εμφανίζονται πλέον ως ξεχωριστές γραμμές χρέωσης WooCommerce στο checkout, κάνοντας τα σύνολα φόρων πιο σαφή πριν από την πληρωμή.
* Νέο: Προστέθηκε προαιρετική ρύθμιση "Άμεση αναστολή συνδρομών σε αποτυχημένες ανανεώσεις" και το φίλτρο `wu_woo_suspend_on_payment_failure` για ιστότοπους που θέλουν αναστολή λόγω αποτυχημένης ανανέωσης κατά το παράθυρο επανάληψης WooCommerce Subscriptions.
* Διόρθωση: Συμφιλιώθηκαν οι καταστάσεις WooCommerce Subscription που μπορούσαν να παραμείνουν εκτός συγχρονισμού με τις συνδρομές Ultimate Multisite μετά από αποτυχημένες ή ανακτημένες ανανεώσεις.
* Διόρθωση: Προστέθηκε το νόμισμα του καταστήματος WooCommerce στη λίστα νομισμάτων του Ultimate Multisite όταν έλειπε.
* Διόρθωση: Διατηρήθηκαν τα στοιχεία χρέωσης πελάτη κατά την ανακατεύθυνση συνδρομητών στο WooCommerce checkout.
* Βελτίωση: Προστέθηκε συμβατότητα με Jetpack Autoloader 5.
* Βελτίωση: Καθαρίστηκε η δημιουργία πακέτου έκδοσης, ώστε τα αρχεία zip του GitHub και του marketplace να αποφεύγουν ένθετους καταλόγους προετοιμασίας και αρχεία ανάπτυξης.

Έκδοση 2.0.6 - Κυκλοφόρησε στις 2026-01-16
* Βελτίωση: Συμπερίληψη βασικών συνδρομών στο πρόσθετο. Δεν απαιτεί πλέον την επέκταση Woocommerce Subscriptinos.

Έκδοση 2.0.5 - Κυκλοφόρησε στις 2026-01-09
* Βελτίωση: Φόρτωση μεταφράσεων από το glotpress API.
* Διόρθωση: Μοιραίο σφάλμα σε ορισμένους κατασκευαστές σελίδων.
* Διόρθωση: Άπειρη ανακατεύθυνση όταν ο πελάτης είναι μέλος του κύριου ιστότοπου.

Έκδοση 2.0.4 - Κυκλοφόρησε στις 2025-11-14

* Αλλαγή: Μετονομάστηκε σε Ultimate Multisite: Woocommerce Integration.
* Προστέθηκε: Συμβατότητα με Woocommerce 10.2.1.
* Προστέθηκε: Συμβατότητα με Woocommerce Subscriptions 7.7.0.
* Διόρθωση: Συμβατότητα με PHP 8.4
* Διόρθωση: Ανακατεύθυνση όταν δεν υπάρχει σελίδα WC Account.

Έκδοση 2.0.3 - Κυκλοφόρησε στις 2025-08-13
* Αλλαγή: Ενεργοποιήθηκαν αυτόματες ενημερώσεις με νέο marketplace.

* Διόρθωση: Προστέθηκε η ημερομηνία έναρξης συνδρομής για χρήση στις συνδρομές Woocommerce.
