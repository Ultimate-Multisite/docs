# Gratis AI Agent {#gratis-ai-agent}

Το Gratis AI Agent είναι ένας αυτονομικός βοηθός AI για WordPress που μπορεί να σχεδιάζει, να κατασκευάζει και να διαχειρίζεται ολόκληρες ιστοσελίδες WordPress μέσω φυσικής γλώσσας. Εκθέτει ένα δομημένο σύνολο **ικανοτήτων (abilities)** — διακριτές, δοκιμαστέες ενέργειες που μπορεί να καλέσει ο βοηθός — καλύπτοντας τα πάντα, από τη διαχείριση περιεχομένου μέχρι την προσαρμογή του design system. Η έκδοση 1.9.0 προσθέτει ικανότητες δημιουργίας περιεχομένου, οπτική αναθεώρηση σελίδων μέσω client-side screenshots, πέντε ενσωματωμένους προφίλ βοηθού, και feature flags για τον έλεγχο πρόσβασης και το branding.

## Βασικά Χαρακτηριστικά {#key-features}

- **System των Ικανοτήτων (Abilities system)** — ένα μοντέλλο, επεκτάσιμο καταλόγιο ενεργειών που μπορεί να καλέσει ο βοηθός στην εγκατάστασή σας του WordPress
- **Διαχείριση Περιεχομένου (Content Management)** — δημιουργία άρθρων και σελίδων (συμπεριλαμβανομένης της καταχώρισης page templates), μαζική δημιουργία πολλαπλών άρθρων με μία κλήση, ρύθμιση featured images και κατασκευή contact forms
- **Οπτική Αναθεώρηση (Visual Review)** — καταγραφή screenshots ζωντανών σελίδων, σύγκριση καταστάσεων πριν/μετά, και λήψη μιας αναθεώρησης σχεδιασμού που παράγεται από AI, καλύπτοντας layout, typograpy, χρώμα και προσβασιμότητα
- **Πέντε ενσωματωμένοι βοηθοί (Five built-in agents)** — Content Writer, Site Builder, Design Studio, Plugin Manager και Support Assistant· μπορούν να αλλάξονται μέσω του Agent Picker στην κεφαλίδα του chat
- **Feature flags** — διακόπτες για τον έλεγχο πρόσβασης και το branding στις Settings → Feature Flags για περιορισμούς ρόλων, ονόματα white-label και προσαρμογή widgets
- **Διαχείριση Custom Post Type** — καταχώριση, λίστα και διαγραφή custom post types με διατήρηση μετά από επανεκκλήσεις
- **Διαχείριση Custom Taxonomy** — καταχώριση, λίστα και διαγραφή custom taxonomies με διατήρηση
- **Ικανότητες Design System** — εισαγωγή custom CSS, διαχείριση επιλεγμένων block patterns, ρύθμιση του site logo και εφαρμογή theme.json presets
- **Διαχείριση Global Styles** — ανάγνωση και κتابική των τιμών theme.json για παλέτες χρωμάτων, typograpy, αποστάσεων και άλλα
- **Διαχείριση Πλοήγησης (Navigation Menu management)** — δημιουργία και ενημέρωση πλοήγησης και των στοιχείων της προγραμματικά
- **Διαχείριση Επιλογών (Options Management)** — ανάγνωση και κتابική των επιλογών του WordPress με μια ενσωματωμένη λίστα ασφαλείας (blocklist) για προστασία κρίσιμων ρυθμίσεων
- **Εγκατάστασιμη Αρχή Ικανοτήτων (Installable Abilities Registry)** — ανακάλυψη και ενεργοποίηση επιπλέον πακέτων ικανοτήτων χωρίς να τροποποιείται το core plugin files
- **Site Builder Orchestration v2** — σχέδια κατασκευής σελίδων με πολλαπλά βήματα, παρακολούθηση προόδου και ανάκτηση σφαλμάτων
- **WP-CLI benchmark command** — εκτέλεση του benchmark suite των ικανοτήτων του βοηθού από την command line για CI και αξιολόγηση μοντέλου

## Απαιτήσεις {#requirements}

- WordPress 6.0 ή νεότερη
- PHP 8.1 ή νεότερη
- WordPress Multisite (ενεργοποιημένο στο network) ή single-site εγκατάσταση
- Ένα ενεργό plugin connector AI provider (Anthropic, OpenAI ή συμβατό)

## Εγκατάσταση {#installation}

1. Ανέβασμα των αρχείων του plugin στο `/wp-content/plugins/gratis-ai-agent`
2. Ενεργοποίηση του plugin από την οθόνη **Plugins** στο WordPress
3. Πλοήγηση στο **Gratis AI Agent → Settings** και σύνδεση του AI provider σας
4. (Προαιρετικό) Περιήγηση στο **Abilities Registry** για εγκατάσταση επιπλέον πακέτων ικανοτήτων

## Γρήγορη Εκκίνηση {#quick-start}

Μόλις ενεργοποιηθεί και συνδεθεί σε έναν AI provider, ανοίξτε το πάνελ **Gratis AI Agent** στην πλαϊνή γραμμή του WordPress admin. Πληκτρολογήστε μια εντολή φυσικής γλώσσας και ο βοηθός θα σχεδιάσει και θα εκτελέσει τα απαιτούμενα βήματα χρησιμοποιώντας τις διαθέσιμες ικανότητές του.

**Παραδείγματα εντολών:**

```
Create a "Portfolio" custom post type with a "Project Category" taxonomy.
```

```
Inject a dark-mode colour palette into the theme and set the primary colour to #1a1a2e.
```

```
Build a restaurant website with an online menu, booking form, and contact page.
```

## Επισκόπηση Ικανοτήτων (Abilities Overview) {#abilities-overview}

Οι ικανότητες είναι οι ατομικές ενέργειες που μπορεί να εκτελέσει ο βοηθός. Κάθε ικανότητα είναι μια καταχωρημένη PHP class που εκθέτει ένα schema (παράμετροι και τύπους επιστροφής) που διαβάζει ο βοηθός κατά τη λειτουργία. Οι ικανότητες είναι ομαδοποιημένες σε λειτουργικές περιοχές:

| Περιοχή | Ικανότητες |
|---|---|
| **Διαχείριση Περιεχομένου** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **Οπτική Αναθεώρηση** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **Custom Post Types** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **Custom Taxonomies** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **Design System** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **Global Styles** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **Πλοήγησης** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **Διαχείριση Επιλογών** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **Εγκατάστασιμες Ικανότητες** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **Site Builder** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

Δείτε την [Abilities Reference](./abilities) για πλήρη τεκμηρίωση παραμέτρων και παραδείγματα χρήσης.

## WP-CLI {#wp-cli}

Το Gratis AI Agent παρέχει μια οικογένεια εντολών `wp gratis-ai-agent` για την εκτέλεση benchmarks και τη διαχείριση ικανοτήτων από την command line. Δείτε την [WP-CLI Reference](./wp-cli) για την πλήρη λίστα εντολών.

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## Ρύθμιση (Configuration) {#configuration}

Όλες οι ρυθμίσεις βρίσκονται στο **Gratis AI Agent → Settings** στο WordPress admin:

| Ρύθμιση | Περιγραφή |
|---|---|
| **AI Provider** | Επιλέξτε και ρυθμίστε τον ενεργό AI provider connector |
| **Model** | Επιλέξτε το language model που θα χρησιμοποιηθεί για τις συνεδρίες του βοηθού |
| **Max Tokens** | Προϋπολογισμός tokens ανά αίτημα (default 4096) |
| **Ability Timeout** | Μέγιστος χρόνος σε δευτερόλεπτα που μπορεί να τρέξει μια μεμονωμένη ικανότητα (default 30) |
| **Debug Logging** | Καταγραφή των κλήσεων ικανοτήτων και της λογικής του βοηθού στο debug log του WordPress |

## Λίστα Ασφαλείας Διαχείρισης Επιλογών (Options Management Safety Blocklist) {#options-management-safety-blocklist}

Η ικανότητα Options Management περιλαμβάνει μια ενσωματωμένη λίστα ασφαλείας που αποτρέπει από τον βοηθό να διαβάζει ή να γράφει ευαίσθης επιλογές του WordPress. Η προεπιλεγμένη λίστα ασφαλείας καλύπτει:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (επιλογές URL — χρησιμοποιήστε ένα ειδικό εργαλείο migration για αυτές)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (κρυφές κλειδιά)
- `active_plugins`, `template` (ενεργοποίηση plugin/theme — χρησιμοποιήστε ειδικές ικανότητες)
- Τα στοιχεία σύνδεσης (credentials) και τα API keys των payment gateways που καταχωρούνται από τα εγκατεστημένα plugins

Πρόσθετες καταχωρίσεις μπορούν να προστεθούν μέσω του filter `gratis_ai_agent_options_blocklist`.

## Hooks και Filters {#hooks-and-filters}

### Actions {#actions}

- `gratis_ai_agent_ability_registered` — εκδίδεται μετά την καταχώριση μιας ικανότητας, λαμβάνει το όνομα class της ικανότητας
- `gratis_ai_agent_plan_started` — εκδίδεται όταν ξεκινά η εκτέλεση ενός plan site builder
- `gratis_ai_agent_plan_completed` — εκδίδεται όταν ολοκληρώνεται επιτυχώς ένα plan
- `gratis_ai_agent_plan_failed` — εκδίδεται όταν ένα plan συναντά ένα μη ανακτήσιμο σφάλμα

### Filters {#filters}

- `gratis_ai_agent_abilities` — τροποποιεί τη καταχωρημένη λίστα ικανοτήτων πριν φορτωθεί από τον βοηθό
- `gratis_ai_agent_options_blocklist` — επεκτείνει ή μειώνει τη λίστα ασφαλείας των επιλογών
- `gratis_ai_agent_benchmark_questions` — προσθέτει προσαρμοστικά ερωτήματα benchmark στο suite αξιολόγησης
- `gratis_ai_agent_plan_plugins` — τροποποιεί τη λίστα ανακάλυψης plugins κατά τη δημιουργία του plan site
