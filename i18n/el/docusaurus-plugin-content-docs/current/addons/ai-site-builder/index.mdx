# AI Site Builder

ultimate-multisite-ai-site-builder

## Απαιτήσεις {#requirements}

- WordPress 5.3 ή νεότερη έκδοση
- PHP 7.4 ή νεότερη έκδοση
- Multisite Ultimate plugin (ενεργοποιημένο)

## Εγκατάσταση {#installation}

1. Ανεβάστε τα αρχεία του addon στον κατάλογο `/wp-content/plugins/`
2. Ενεργοποιήστε το plugin μέσω του μενού 'Plugins' στο WordPress
3. Ρυθμίστε τις ρυθμίσεις του addon στον διαχειριστή του network

## Ανάπτυξη {#development}

### Ρύθμιση {#setup}

```bash
# Εγκατάσταση εξαρτήσεων
composer install

# Διόρθωση προβλημάτων στυλ κώδικα
vendor/bin/phpcbf
```

### Δοκιμές {#testing}

Το addon περιλαμβάνει ένα ολοκληρωμένο πλαίσιο δοκιμών:

- **Unit Tests**: Δοκιμές βασισμένες στο PHPUnit με το WordPress test suite
- **Base Test Class**: Παρέχει βοηθητικές μεθόδους για συνηθισμένες περιπτώσεις δοκιμών
- **Test Coverage**: Ρυθμισμένο για παρακολούθηση της κάλυψης του κώδικα
- **Multisite Testing**: Οι δοκιμές εκτελούνται σε περιβάλλον multisite

### Κανόνες Κωδικοποίησης {#code-standards}

- **WordPress Coding Standards**: Εφαρμόζεται μέσω PHPCS
- **PHP 7.4+ Compatibility**: Υποστηρίζονται σύγχρονα χαρακτηριστικά PHP
- **Static Analysis**: Ενσωμάτωση PHPStan για έλεγχο τύπων (type checking)
- **Code Modernization**: Κανόνες Rector για αναβάθμιση PHP

### Σύστημα Build {#build-system}

- **Asset Processing**: Συμπίεση (Minification) για αρχεία CSS/JS
- **Translation**: Δημιουργία αρχείου POT
- **Packaging**: Δημιουργία αρχείου archive για διανομή
- **Development Mode**: Μη συμπιεσμένα assets για debugging

## Ρύθμιση {#configuration}

Το addon υποστηρίζει διάφορες επιλογές ρύθμισης που είναι προσβάσιμες μέσω της διεπαφής του network admin.

## Hooks και Filters {#hooks-and-filters}

### Actions {#actions}

- `ultimate-multisite-ai-site-builder_init` - Εκτελείται μετά την αρχικοποίηση του addon
- `ultimate-multisite-ai-site-builder_loaded` - Εκτελείται μετά το φόρτωση όλων των αρχείων του addon

### Filters {#filters}

- `ultimate-multisite-ai-site-builder_settings` - Για τη τροποποίηση των ρυθμίσεων του addon
- `ultimate-multisite-ai-site-builder_enabled` - Για την παρακάνωση της κατάστασης ενεργοποίησης/απενεργοποίησης του addon
