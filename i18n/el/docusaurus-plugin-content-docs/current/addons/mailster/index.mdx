# Mailster {#mailster}

Μετατρέψτε το multisite network σας σε μια ισχυρή μηχανή κατασκευής email λιστών με το Ultimate Multisite Mailster Integration addon. Προσθέστε αυτόματα νέους πελάτες στις λίστες email του Mailster σας όταν κάνουν εγγραφή ή αγορές, δημιουργώντας μια άπταιστη σύνδεση μεταξύ της πλατφόρμας μελών σας και της στρατηγικής σας email marketing.

## Βασικά Χαρακτηριστικά {#key-features}

**Αυτόμανη Συγχρονισμός Συνδρομητών**
* Προσθέτει αυτόματα τους πελάτες στις λίστες Mailster όταν κάνουν εγγραφή
* Επιλέξτε αν θέλετε να συγχρονιστεί αμέσως κατά την εγγραφή ή να περιμένετε την επιβεβαίωση πληρωμής
* Διατηρεί τις λίστες email σας φρέσκες και ενημερωμένες χωρίς χειροκίνηczną παρέμβαση

**Ευέλικτη Διαχείριση Λιστών**
* Ρυθμίστε τις παγκόσμιες προεπιλεγμένες λίστες για όλους τους νέους πελάτες
* Υπερρίστε τις προεπιλεγές με λίστες ειδικές για προϊόντα για στοχευμένες καμπάνιες
* Διαχειριστείτε εύκολα τις καταχωρίσεις λιστών με την quen thuộc interface του Mailster

**Επιλογές Opt-In Συμμορφούμενες με το GDPR**
* Επιλέξτε μεταξύ αυτόματης εγγραφής ή συναίνεσης με πλαίσιο (checkbox)
* Ενεργοποιήστε το double opt-in για επιβεβαιωμένες συνδρομές
* Εμφανίζετε/απεμφανίζετε το πλαίσιο opt-in ανάλογα με τις απαιτήσεις συμμόρφωσης σας

**Έξυπνη Αντιστοίχιση Δεδομένων (Smart Data Mapping)**
* Συγχρονίζει αυτόματα τα ονόματα και τα στοιχεία επικοινωνίας των πελατών
* Αντιστοιχίζει τα στοιχεία χρέωσης στα πεδία των συνδρομητών
* Παρακολουθεί τις ημερομηνίες εγγραφής για στοχευμένες καμπάνιες

**Έλεγχος σε Επίπεδο Προϊόντος**
* Καταχωρίστε διαφορετικές λίστες σε διαφορετικά προϊόντα
* Δημιουργήστε στοχευμένες καμπάνιες με βάση αυτά που αγοράζουν οι πελάτες
* Απενεργοποιήστε το Mailster για συγκεκριμένα προϊόντα όταν χρειαστεί

**Επαγγελματική Εμπειρία**
* Ενσωμάτωση native Mailster UI για επιλογή λίστας
* Αναλυτικό logging για απο₂)troubleshooting και επαλήθευση
* Άπταιστη ενημέρωση των υπαρχόντων συνδρομητών

## Ιδανικό Για {#perfect-for}

* Ιδιοκτήτες sites μελών που κατασκευάζουν email λίστες
* Πλατφόρμες SaaS που θέλουν να φροντίζουν τους νέους πελάτες
* Δημιουργούς μαθημάτων που κάνουν segmenting των φοιτητών βάσει εγγραφής
* Το άτομο που τρέχει το Ultimate Multisite με το Mailster

## Πώς Λειτουργεί {#how-it-works}

1. Εγκαταστήστε και ενεργοποιήστε το addon στο Ultimate Multisite network σας
2. Ρυθμίστε τις παγκόσμιες προεπιλεγμένες λίστες στις ρυθμίσεις του Ultimate Multisite
3. Προαιρετικά, υπερρίστε με λίστες ειδικές για προϊόντα
4. Επιλέξτε τον τρόπο opt-in και το timing της συνδρομής
5. Παρακολουθήστε τις λίστες Mailster σας να μεγαλώνουν αυτόματα!

## Εγκατάσταση {#installation}

1. Ανεβάστε τα αρχεία του plugin στο directory των addons του Ultimate Multisite σας
2. Βεβαιωθείτε ότι το Mailster είναι εγκατεστημένο και ενεργό στο κύριο site σας
3. Ενεργοποιήστε το addon μέσω της σελίδας addons του Ultimate Multisite
4. Πλοηγηθείτε στις Settings > Mailster Integration για να κάνετε τις ρυθμίσεις
5. Επιλέξτε τις προεπιλεγμένες λίστες σας και ρυθμίστε τις προτιμήσεις opt-in

## Συχνές Ερωτήσεις {#frequently-asked-questions}

## Αυτό το addon λειτουργεί χωρίς το Ultimate Multisite; {#does-this-addon-work-without-ultimate-multisite}

Όχι, αυτό το addon απαιτεί το Ultimate Multisite για να λειτουργήσει. Είναι σχεδιασμένο ειδικά για να ενσωματώνει το Mailster με το σύστημα πελατών και προϊόντων του Ultimate Multisite.

## Χρειάζομαι να είναι το Mailster εγκατεστημένο σε όλα τα sites; {#do-i-need-mailster-installed-on-all-sites}

Όχι, το Mailster χρειάζεται να είναι εγκατεστημένο και ενεργό μόνο στο κύριο site του network σας. Το addon θα συνδεθεί αυτόματα σε αυτό από όλα τα subsites.

## Μπορώ να καταχωρίσω διαφορετικές λίστες σε διαφορετικά προϊόντα; {#can-i-assign-different-lists-to-different-products}

Ναι! Μπορείτε να ρυθμίσετε παγκόσμιες προεπιλεγμένες λίστες, και μετά να τις υπερρίσετε σε βάση προϊόντος. Αυτό σας επιτρέπει να δημιουργήσετε εξαιρετικά στοχευμένες email καμπάνιες με βάση αυτά που αγοράζουν οι πελάτες.

## Τι συμβαίνει αν ένας πελάτης υπάρχει ήδη στο Mailster; {#what-happens-if-a-customer-already-exists-in-mailster}

Με προεπιλογή, το addon θα ενημερώνει τους υπάρχοντες συνδρομητές και θα τους προσθέτει σε νέες λίστες. Μπορείτε να απενεργοποιήσετε αυτό το χαρακτηριστικό στις ρυθμίσεις αν προτιμάτε να παραλείψετε τους υπάρχοντες συνδρομητές.

## Είναι αυτό GDPR compliant; {#is-this-gdpr-compliant}

Ναι! Το addon περιλαμβάνει λειτουργία opt-in με πλαίσιο (checkbox) όπου οι πελάτες πρέπει να δώσουν ρητή συγκατάθεση για να λαμβάνουν emails. Αυτό πληροί τις απαιτήσεις του GDPR όταν συνδυάζεται με τις κατάλληλες πολιτικές απορρήτου.

## Μπορώ να χρησιμοποιήσω double opt-in; {#can-i-use-double-opt-in}

Απολύτως! Ενεργοποιήστε τη ρύθμιση double opt-in και το Mailster θα στείλει emails επιβεβαίωσης πριν ενεργοποιήσει τις συνδρομές.
