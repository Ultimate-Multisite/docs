# Filter: wu_apc_should_display_admin_menu {#filter-wuapcshoulddisplayadminmenu}

Επιτρέπει στους διαχειριστές να κρύβουν τις σελίδες του μενού διαχειριστή, όταν έχουν ολοκληρώσει τη διαδικασία προσαρμογής.

## Παράμετροι {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $should_display_admin_menu | `bool` | Αν πρέπει να εμφανιστούν οι σελίδες. |
| $network_active | `bool` | Ελέγχει αν το plugin είναι ενεργό για τον διαχειριστή του δικτύου. |
