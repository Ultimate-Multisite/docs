# Action: wu_page_enqueue_scripts

Προσθέτουμε μια προσαρμοσμένη action εδώ ώστε να έχουμε ένα σημείο για να κάνουμε το `enqueue` των scripts και των styles gerekti και να διασφαλίσουμε ότι φορτώνονται μόνο στις σωστές σελίδες.

## Παράμετροι {#parameters}

| Όνομα | Τύπος | Περιγραφή |
|------|------|-------------|
| $ | `\WU_Page` | Η τρέχουσα σελίδα διαχείρισης (admin page) |
