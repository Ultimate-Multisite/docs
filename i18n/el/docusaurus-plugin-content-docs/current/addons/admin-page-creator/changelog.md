# Κατάλογος Αλλαγών Admin Page Creator

### Έκδοση 1.9.0 - Έκδοση στις 2026-01-18 {#version-190---released-on-2026-01-18}

* Διορθώσεις: Καταστροφικά σφάλματα κατά την ενεργοποίηση.
* Αναδιαμόρφωση: Οι κλάσεις πηγής περιεχομένου χρησιμοποιούν τώρα το Singleton trait με σωστή αρχικοποίηση του κλάσης-γονέα.
* Διορθώσεις: Σφάλματα μεταγλώττισης template Vue.js που αποτρέπουν την εμφάνιση του επιλεκτή τύπου περιεχομένου.
* Διορθώσεις: Αποκλειόμενες ιδιότητες δεδομένων Vue (template_id, external_link_url_embedable, external_link_url_checking).
* Διορθώσεις: Ο editor TinyMCE δεν αρχικοποιούνταν σωστά κατά το φόρτωση της σελίδας.
* Διορθώσεις: Το αρχείο υποστήριξης Divi δεν φορτωνόταν αυτόματα μέσω Composer.
* Διορθώσεις: Κατεστραμμένη σύνταξη PHP στην κλάση υποστήριξης Divi.
* Βελτιώσεις: Οργάνωση και minification των JavaScript assets.
* Βελτιώσεις: Συμβατότητα με τις τελευταίες εκδόσεις του WordPress και των page builder.
