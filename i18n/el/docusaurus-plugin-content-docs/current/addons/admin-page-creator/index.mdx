# Admin Page Creator

## Περιεχόμενο {#content}
Το Admin Page Creator θα σας επιτρέψει να προσθέτετε εύκολα νέες admin σελίδες και υποσελίδες στο WordPress admin dashboard. Μπορεί να χρησιμοποιηθεί για τη δημιουργία μιας σελίδας καλωσορίσματος ή μιας σελίδας εκπαιδευτικού υλικού που μπορεί να λειτουργήσει ως landing page για τους νέους χρήστες. Μπορείτε να χρησιμοποιήσετε page builders όπως το Beaver Builder, Elementor, Brizy και το Oxygen για να σχεδιάσετε τις σελίδες χωρίς κόπο.

Παρέχει επίσης υποστήριξη για custom HTML, CSS και JavaScript, καθώς και έλεγχο πρόσβασης βάσει plan και user role!

## Χαρακτηριστικά {#features}
* Υποστήριξη Custom HTML και CSS
* Υποστήριξη JavaScript
* PHP Content Mode
* Άδεια βάσει User role
* Υποστήριξη για page builders

## Εγκατάσταση {#installation}

1. Ανεβάστε το **ultimate-multisite-admin-page-creator** στον κατάλογο **/wp-content/plugins/**
2. Ενεργοποιήστε το plugin μέσω του μενού **Plugins** στο WordPress Network Admin σας

## Οδηγός Χρήσης {#usage-guide}

Ενώ το WordPress κάνει πολύ εύκολο το προσθέτον κειμένου στο front end των sites, η δημιουργία σελίδων και υποσελίδων στην admin πλευρά απαιτεί κάποιες βασικές γνώσεις κωδικοποίησης. Το Ultimate Multisite: Admin Page Creator στοχεύει να φέρει αυτή την ίδια ευκολία χρήσης του front end στο back end των sites του network σας.

## Γιατί να χρησιμοποιήσετε το Admin Page Creator; {#why-use-admin-page-creator}

Ίσως θέλετε να προσφέρετε μια custom σελίδα στο admin panel των πελατών σας με tutorials, ίσως θέλετε να προσθέσετε μια custom σελίδα με FAQs και ένα contact widget. Οι δυνατότητες είναι απεριόριστες με το Admin Page Creator, και αυτό είναι αυτό που θα εξερευνήσουμε στο σημερινό tutorial.

## Δημιουργία της πρώτης Custom Admin Σελίδας {#creating-your-first-custom-admin-page}

Μετά την εγκατάσταση και την ενεργοποίηση του Ultimate Multisite: Admin Page Creator, μεταβείτε στο μενού network admin και επιλέξτε Admin Pages.

![Custom Admin Pages list on Network Admin](/img/admin/admin-pages-list.png)

_Λίστα Custom Admin Pages στο Network Admin_

Θα δείτε μια λίστα με όλες τις admin σελίδες που έχετε δημιουργήσει μέχρι στιγμής (επειδή είναι η πρώτη σας επίσκεψη, η λίστα θα είναι κενή). Πατήστε **Add new Admin Page**. Θα μετατραπεί στο editor της Admin Page.

## Τα στοιχεία μιας Custom Admin Σελίδας {#the-elements-of-a-custom-admin-page}

Στο Admin Page editor, θα έχετε μια σειρά από διαφορετικές επιλογές για να προσαρμόσετε τη σελίδα σας.

### Τίτλος Σελίδας (Page Title) {#page-title}

Αυτός είναι ο τίτλος που θα εμφανίζεται στην κορυφή της σελίδας, και όχι η ετικέτα του μενού admin.

<!-- Screenshot unavailable: Admin Page editor showing Page Title field -->

_Εισάγετε έναν Τίτλο Σελίδας_

### Επιλογή Τύπου Περιεχομένου (Content Type Selector) {#content-type-selector}

Το Ultimate Multisite: Admin Page Creator σας δίνει δύο επιλογές όσον αφορά το περιεχόμενο της σελίδας: είτε μπορείτε να χρησιμοποιήσετε το default WordPress WYSIWYG editor είτε να εναλλάσσετε σε έναν HTML editor.

Εάν δεν είστε εξοικेडiados με το HTML, παραμείνετε στο default WordPress editor. Θα μπορείτε να επεξεργαστείτε το περιεχόμενο της σελίδας ακριβώς όπως θα το κάνατε με ένα normal Post ή Page.

## Επιλογές Μενού {#menu-options}

Στα πλάγια, στο meta-box General Options, θα μπορείτε να ρυθμίσετε τον τύπο μενού, την ετικέτα και πολύ περισσότερα.

### Τύποι Μενού (Menu Types) {#menu-types}

Η νέα custom admin σελίδα σας μπορεί να έχει δύο διαφορετικούς τύπους μενού: top-level menu και submenu.

Μια σελίδα μενού top-level, όπως υποδηλώνει το όνομά της, θα προσθέσει τη newly created σελίδα σας στο admin menu μαζί με τις default WordPress σελίδες.

Εάν επιλέξετε την επιλογή top-level, θα μπορείτε επίσης να επιλέξετε τη σειρά του μενού και ένα icon μενού.

Εάν θέλετε να κάνετε αυτή τη σελίδα μια σελίδα υπο-μενού, θα μπορείτε να επιλέξετε μια parent page στην οποία θα συνδέσετε αυτή τη σελίδα. Μπορείτε να προσθέσετε τη custom σελίδα σας στις default WordPress σελίδες ή στις top-level custom admin σελίδες σας.

## Προηγμένες Επιλογές (Advanced Options) {#advanced-options}

Θα πρέπει να μπορείτε να προσθέσετε όλα τα είδη περιεχομένου στις custom admin σελίδες σας. Αυτό περιλαμβάνει κανόνες και αρχεία CSS, JavaScript και εξωτερικές βιβλιοθήκες. Μπορείτε να το κάνετε χρησιμοποιώντας την Advanced Options Tab.

### Custom CSS {#custom-css}

Στην tab CSS, μπορείτε να προσθέσετε τον δικό σας custom CSS κώδικα, καθώς και να εισάγετε CSS αρχεία από remote sites (όπως CDNs, Google Fonts, κ.λπ).

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_Υποστηρίζεται και το Custom CSS. Μπορείτε επίσης να συμπεριλάβετε εξωτερικά style files!_

### Custom JavaScript {#custom-javascript}

Το ίδιο ισχύει και για το JavaScript. Μπορείτε να προσθέσετε εξωτερικές βιβλιοθήκες για να χρησιμοποιήσετε στον custom JavaScript κώδικά σας.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Προσθέστε τον δικό σας JavaScript κώδικα!_

## Άδειες (Permissions) {#permissions}

Επίσης στις Advanced Options, θα μπορείτε να επιλέξετε ποια sub-sites θα έχουν πρόσβαση σε αυτή τη σελίδα. Αυτό είναι χρήσιμο εάν σχεδιάζετε να εξυπηρετήσετε διαφορετικό αποκλειστικό περιεχόμενο σε διαφορετικά plan tiers, για παράδειγμα.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Έχετε πλήρη έλεγχο για το πού πρόκειται να εμφανιστεί αυτή η σελίδα_

**Σημαντικό**: Και οι δύο προϋποθέσεις πρέπει να πληρούνται για να εμφανιστεί η σελίδα. Αυτό σημαίνει ότι αν ο χρήστης είναι πελάτης ενός συγκεκριμένου plan, αλλά δεν έχει έναν από τους επιλεγμένους ρόλους (ή το αντίστροφο), η σελίδα δεν θα εμφανιστεί στο panel του.

## Ενεργό (Active) {#active}

Εάν θέλετε να απενεργοποιήσετε μια custom admin σελίδα χωρίς να την διαγράψετε ή να παίξετε με τις άδειες, απλώς αφαιρέστε την επιλογή Active.

Η απενεργοποίηση μιας top-level σελίδας θα απενεργοποιήσει αυτόματα όλες τις υποσελίδες που είναι συνδεδεμένες σε αυτή τη parent page.
