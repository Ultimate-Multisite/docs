# Filter: wu_cap_captcha_config

Φιλτράρει τη ρύθμιση του Cap Captcha.

Επιτρέπει στους developers να προσαρμόσουν τις ρυθμίσεις του Cap Captcha.

## Παράμετροι {#parameters}

| Όνομα | Τύπος | Περιγραφή |
|------|------|-------------|
| $config | `array` | Το array της ρύθμισης του Cap. |
| $level | `string` | Το επίπεδο ασφαλείας (fast, medium, max). |

### Από {#since}

- 1.2.0
