# Language Selector {#language-selector}

Ultimate Multisite: Language Selector κατά την Εγγραφή

Προσθέστε ένα πεδίο επιλογής γλώσσας στη ροή εγγραφής.

## Εγκατάσταση {#installation}

1. Ανεβάστε το 'ultimate-multisite-language-selector' στον κατάλογο '/wp-content/plugins/'
2. Ενεργοποιήστε το plugin μέσω του μενού 'Plugins' στο Network Admin του WordPress σας

## Οδηγός Χρήσης {#usage-guide}

Με αυτό το δωρεάν add-on, μπορείτε να επιτρέψετε στους χρήστες σας να επιλέγουν ποια γλώσσα θα θέλουν να δουν τόσο στη ροή εγγραφής όσο και στη σελίδα σύνδεσης.

Μετά την [ενεργοποίηση της άδειάς σας](../../user-guide/getting-started/installing-ultimate-multisite.md), θα μπορείτε να εγκαταστήσετε τα add-ons του Ultimate Multisite που σας επιτρέπει η άδειά σας.

Μεταβείτε στη σελίδα ρυθμίσεων του Ultimate Multisite και στη συνέχεια, στο μενού αριστερά, κάντε κλικ στην ένδειξη _Add-ons_.

![Add-ons page](/img/admin/addons-page.png)

_Μεταβείτε στη σελίδα Add-ons για να ξεκινήσετε την εγκατάσταση του Language Selector_

Εκεί θα βρείτε όλα τα add-ons που προσφέρουμε.

Τα δωρεάν add-ons είναι διαθέσιμα σε όλους τους πελάτες μας, ανεξαρτήτως του πακέτου συνδρομής που έχουν υπογράψει.

Κάντε κλικ στο κουμπί _Install it Now!_ κάτω από την εικόνα του add-on. Στη συνέχεια, ενεργοποιήστε το στη σελίδα Plugins.

![Add-ons page with install button](/img/admin/addons-page.png)

_Κάντε κλικ στο κουμπί για την εγκατάσταση του add-on_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_Ενεργοποιήστε το στη σελίδα Plugins σας_

Τώρα, μεταβείτε στο μενού Ρυθμίσεων του Ultimate Multisite και εισάγετε την καρτέλα _Network Settings_. Εκεί, στο κάτω μέρος της σελίδας, θα βρείτε μια νέα ενότητα που ονομάζεται _Languages_.

![Settings page](/img/config/settings-general.png)

_Εμφανίζεται μια εντελώς νέα ενότητα!_

Μπορείτε να την αφήσετε κενή για να εμφανιστούν όλες οι γλώσσες ως επιλογές ή να επιλέξετε ποιες θέλετε να είναι διαθέσιμες. Οι γλώσσες είναι αυτές που έχετε εγκαταστήσει στο WordPress setup σας.

Μετά την αποθήκευση των αλλαγών που κάνατε, ένα μενού επιλογών με τις γλώσσες που επιλέξατε θα εμφανίζεται τόσο στη σελίδα Σύνδεσης όσο και στη Ροή Εγγραφής.

## Ρύθμιση της Γλώσσας μέσω της URL {#setting-the-language-via-the-url}

Μπορείτε επίσης να αλλάξετε τη γλώσσα μιας σελίδας παρακάμπτοντας το παράμετρο locale στην URL. Αν θέλετε να κάνετε redirect σε μια σελίδα εγγραφής στα ιταλικά, για παράδειγμα, προσθέστε το ?locale=it_IT στην URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Η γλώσσα πρέπει να είναι εγκατεστημένη στο WordPress network σας για να λειτουργήσει αυτό.
