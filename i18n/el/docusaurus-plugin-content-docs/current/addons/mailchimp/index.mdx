# Mailchimp

Ultimate Multisite: Ενσωμάτωση με Mailchimp

Συνδέστε το δίκτυο Ultimate Multisite σας με το Mailchimp. Μπορείτε να συνδέσετε συγκεκριμένα πακέτα με τις λίστες του Mailchimp και να προσθέτετε αυτόματα τους συνδρομητές σε αυτές τις λίστες κατά την εγγραφή τους.

## Εγκατάσταση

1. Ανεβάστε το 'ultimate-multisite-mailchimp' στον κατάλογο '/wp-content/plugins/'
2. Ενεργοποιήστε το plugin μέσω του μενού 'Plugins' στο WordPress Network Admin σας

## Οδηγός Χρήσης

Η 동 bộ (synchronization) των email των συνδρομητών με τις λίστες MailChimp είναι πολύ εύκολη με το Add-on Ultimate Multisite: MailChimp Integration. Αυτό το tutorial έχει σχεδιαστεί για να σας βοηθήσει να ρυθμίσετε όλα τα πράγματα!

## Λήψη ενός API key

Πριν ξεκινήσουμε να εξετάζουμε τις διάφορες ρυθμίσεις που είναι διαθέσιμες, χρειάζεστε έναν API key του MailChimp. Αυτό θα επιτρέψει στο Ultimate Multisite να επικοινωνεί με το MailChimp API για να ανακτήσει λίστες, ομάδες και να προσθέσει διευθύνσεις email σε αυτές τις λίστες και ομάδες όταν οι χρήστες κάνουν εγγραφή.

Ακολουθήστε αυτό το tutorial για να λάβετε τον δικό σας MailChimp API key: [About API Keys | Mailchimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

_Προσθέστε το MailChimp API key σας στις_ _**Ρυθμίσεις Ultimate Multisite - > Ρυθμίσεις Add-on -> MailChimp**_

## Λειτουργία Ενσωμάτωσης (Integration Mode)

Το add-on MailChimp Integration προσφέρει δύο «λειτουργίες ενσωμάτωσης», που σημαίνει ότι υπάρχουν δύο διαφορετικοί τρόποι με τους οποίους μπορείτε να χρησιμοποιήσετε το add-on για να 동 bộ (synchronize) τις διευθύνσεις email.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_Δύο λειτουργίες ενσωμάτωσης για μεγαλύτερη ευελιξία!_

### Λειτουργία 1: Πολλαπλές Λίστες (Multiple Lists)

Η προεπιλεγμένη λειτουργία χρησιμοποιεί **Πολλαπλές Λίστες (Multiple Lists)**. Αυτή η λειτουργία θα σας επιτρέψει να επιλέξετε πολλαπλές λίστες MailChimp για κάθε πακέτο. Αυτή είναι η καλύτερη επιλογή αν κατηγοριοποιείτε τους χρήστες σας σε διαφορετικές λίστες.

Για παράδειγμα, υποθέστε ότι έχετε 3 διαφορετικές λίστες στον λογαριασμό σας MailChimp: **Newsletter**, **Plan A** και **Plan B**. Θέλετε όλοι οι χρήστες, ανεξάρτητα από το πακέτο τους, να περιλαμβάνονται και στη λίστα Newsletter, ώστε να μην χάνουν τις ενημερώσεις σας.

Για να το πετύχετε, μπορείτε να χρησιμοποιήσετε τη λειτουργία Multiple Lists για να προσθέσετε τόσο τις λίστες **Newsletter** όσο και **Plan A** στις Ρυθμίσεις MailChimp του Plan A, και τις **Newsletter** και **Plan B** στις Ρυθμίσεις MailChimp του Plan B.

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_Επιλέξτε τις προεπιλεγμένες λίστες για νέους και για ακυρωμένους λογαριασμούς_

### Λειτουργία 2: Μία Λίστα + Πολλαπλές Ομάδες (Single List + Multiple Groups)

Το MailChimp σας επιτρέπει να χωρίσετε τη λίστα σας σε μικρότερες ομάδες χρηστών που ονομάζονται **Ομάδες (Groups)**. Αυτό σας επιτρέπει να έχετε περισσότερο έλεγχο, καθώς μπορείτε να στείλετε μια καμπάνια σε ένα συγκεκριμένο υποσύνολο των συνδρομητών σας χωρίς να χρειάζεται να δημιουργήσετε πολλαπλές αντιγραφές της ίδιας καμπάνιας στο MailChimp (κάτι που δυσκολεύει την ανάλυση των μετρήσεων αυτής της καμπάνιας αργότερα).

Για παράδειγμα, μπορείτε να έχετε μία λίστα που ονομάζεται **My Network Subscribers** και μέσα σε αυτήν να έχετε ομάδες για **Plan A** και **Plan B**.

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_Επιλέξτε τις προεπιλεγμένες λίστες και ομάδες, αν χρησιμοποιείτε τη λειτουργία ενσωμάτωσης «μία λίστα + πολλαπλές ομάδες»_

## Προεπιλεγμένες Λίστες και Ομάδες

Αφού επιλέξετε ποια λειτουργία ενσωμάτωσης έχει περισσότερο νόημα για τη συγκεκριμένη χρήση σας, είναι ώρα να επιλέξετε τις προεπιλεγμένες λίστες (και ομάδες, ανάλογα με τη λειτουργία). Η προεπιλεγμένη λίστα θα χρησιμοποιηθεί όταν ένα συγκεκριμένο πακέτο δεν έχει επιλεγμένη λίστα.

Μπορείτε επίσης να επιλέξετε τη προεπιλεγμένη λίστα για τους ακυρωμένους λογαριασμούς. Όποτε ένας χρήστης καταργεί τη συνδρομή του και αφαιρεί τον λογαριασμό του από το δίκτυό σας, η διεύθυνση email του θα μεταφερθεί στη λίστα που έχει ρυθμιστεί για τους ακυρωμένους λογαριασμούς.

## Ενεργοποίηση Διπλής Συνδρομής (Double Subscription)

Στην ίδια σελίδα ρυθμίσεων, θα μπορείτε να ενεργοποιήσετε τη **Διπλή Συνδρομή (Double Subscription)**. Αυτή η επιλογή θα προσθέσει τον χρήστη τόσο στη προεπιλεγμένη λίστα που επιλέχθηκε όσο και στη λίστα που επιλέχθηκε για το πακέτο του χρήστη. Αν αυτή η επιλογή δεν είναι ενεργή, ο χρήστης θα προστεθεί μόνο στη λίστα που επιλέχθηκε για το πακέτο του χρήστη. Αν δεν έχει καθόλου λίστα για αυτό το συγκεκριμένο πακέτο, τότε ο χρήστης θα προστεθεί στη προεπιλεγμένη λίστα που επιλέχθηκε.

_Μπορείτε να έχετε τους χρήστες σας συνδρομητικούς τόσο στη προεπιλεγμένη λίστα όσο και στη λίστα του πακέτου κατά την εγγραφή_

## Επιλογή Λιστών και Ομάδων για κάθε Πακέτο

Μεταβείτε στα **Plans** σας και επιλέξτε το πακέτο που θέλετε να επεξεργαστείτε. Σημειώστε ότι στο meta-box **Advanced Options**, θα είναι διαθέσιμη μια νέα καρτέλα με τίτλο MailChimp.

Σε αυτήν την καρτέλα, θα μπορείτε να επιλέξετε πολλαπλές λίστες ή μία λίστα και πολλαπλές ομάδες (ανάλογα με τη λειτουργία ενσωμάτωσης) για αυτό το πακέτο.

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_Επιλέξτε τις Λίστες και τις Ομάδες που θέλετε να χρησιμοποιήσετε για κάθε ένα από τα πακέτα σας_

Όταν ένας νέος χρήστης εγγράφεται για αυτό το πακέτο, η διεύθυνση email του θα προστεθεί στις επιλεγμένες λίστες και ομάδες!
