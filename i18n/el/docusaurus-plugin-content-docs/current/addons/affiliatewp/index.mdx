# AffiliateWP

Multisite Ultimate: Ενσωμάτωση με AffiliateWP

Χρησιμοποιήστε το ισχυρό AffiliateWP για να αναπτύξετε τη βάση πελατών του Ultimo Network σας!

## Εγκατάσταση {#installation}

1. Ανεβάστε το 'ultimate-multisite-git ' στον κατάλογο '/wp-content/plugins/'
2. Ενεργοποιήστε το plugin μέσω του μενού 'Plugins' στο WordPress
3. Ακολουθήστε τον οδηγό βήμα προς βήμα για να ρυθμίσετε το plugin

## Οδηγός Χρήσης {#usage-guide}

[AffiliateWP](https://affiliatewp.com/) έχει γίνει η ιδανική λύση όταν δημιουργείτε ένα πρόγραμμα συνεργαστών (affiliate program) για την online επιχείρησή σας. Το Ultimate Multisite: AffiliateWP είναι το γέφυρα που θα σας επιτρέψει να δημιουργήσετε το ίδιο για το premium network σας.

## Ξεκίνημα {#getting-started}

Πρώτα απ' όλα, **πρέπει να έχετε** ενεργό το [**AffiliateWP**](https://affiliatewp.com/)! Δεν θα μπορείτε να χρησιμοποιήσετε το add-on μας αν δεν έχετε **το plugin που ενσωματώνεται μαζί του**. Σε περίπτωση που δεν σκοπεύετε να το επιτρέψετε στους πελάτες σας να το χρησιμοποιούν και στα δικά τους sites, το AffiliateWP πρέπει να είναι ενεργό μόνο στο κύριο site σας.

Επίσης, θα χρειαστείτε και τα δύο, το **Ultimate Multisite** και το **Ultimate Multisite: AffiliateWP**, **ενεργοποιημένα** **σε όλο το network**.

## Προσθήκη του Ultimate Multisite ως Ενσωμάτωση AffiliateWP {#adding-ultimate-multisite-as-an-affiliatewp-integration}

Στο κύριο site σας, μεταβείτε στις Ρυθμίσεις Συνεργατών (Affiliates Settings) και στην καρτέλα Ενσωματώσεις (Integrations), επιλέξτε το Ultimate Multisite. Αυτό θα ενημερώσει το [AffiliateWP](https://affiliatewp.com/) να αρχίσει να παρακολουθεί τις επισκέψεις και τις μετατροπές (conversions) από τη ροή εγγραφής του Ultimate Multisite.

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_Ελέγξτε το κουτί Ultimate Multisite στο κάτω μέρος και αποθηκεύστε_

Πρακτικά, αυτό είναι όλα όσα χρειάζεστε. Οι επισκέψεις που έρχονται από τα referral links θα παρακολουθούνται και όταν ο χρήστης κάνει αργότερα την πρώτη του πληρωμή, θα δημιουργηθεί η μετατροπή και θα κρατηθεί μια προμήθεια για τον συνεργάτη που είναι υπεύθυνος για τη μετατροπή.

Ωστόσο, θα παρακολουθείται μόνο η πρώτη πληρωμή που λαμβάνεται από τον νέο χρήστη. Για να ενεργοποιήσετε τις επαναλαμβανόμενες πληρωμές από referrals, θα χρειαστείτε το [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## Ενεργοποίηση Επαναλαμβανόμενων Referrals {#enabling-recurring-referrals}

Αφού εγκαταστήσετε το [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/), μεταβείτε στην καρτέλα Recurring Referrals στις Ρυθμίσεις Συνεργατών (Affiliates Settings) και ενεργοποιήστε τη χρήση των επαναλαμβανόμενων referrals.

<!-- Screenshot unavailable: AffiliateWP Recurring Referrals settings with Enable Recurring Referrals toggle -->

_Όσο απλό που μπορεί να είναι: Ενεργοποιήστε τα Recurring Referrals και είστε έτοιμοι!_

Μάλιστα, αυτό είναι όλα όσα χρειάζεστε!

Τώρα, κάθε φορά που λαμβάνεται μια νέα πληρωμή σε έναν παραπομπμένο λογαριασμό, ο υπεύθυνος συνεργάτης θα λάβει αυτόματα τη δική του μερίδα!
