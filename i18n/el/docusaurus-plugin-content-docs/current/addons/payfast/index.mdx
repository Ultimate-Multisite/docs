# PayFast

Multisite Ultimate: Payfast Gateway

Προσθέτει το Payfast – μια γρήγορη και ασφαλής πύλη διαδικτυακών πληρωμών στην Νότια Αφρική – ως επιλογή πληρωμής για το Multisite Ultimate.

## Εγκατάσταση {#installation}

1. Ανεβάστε το 'ultimate-multisite-payfast' στον κατάλογο '/wp-content/plugins/'
2. Ενεργοποιήστε το plugin μέσω του μενού 'Plugins' στο WordPress
3. Ακολουθήστε τον βήμα προς βήμα Wizard για να ρυθμίσετε το plugin

## Οδηγός Χρήσης {#usage-guide}

Το PayFast είναι ένας από τους κορυφαίους διαδικτυακούς επεξεργαστές πληρωμών για τις επιχειρήσεις στη Νότια Αφρική. Είναι αφιερωμένο στη βοήθεια των διαδικτυακών εμπόρων να αναπτυχθούν, κάνοντας τις συναλλαγές απρόσκοπτες και ασφαλείς. Κατόπιν αιτήματος των χρηστών του Ultimate Multisite, αποφασίσαμε να δημιουργήσουμε ένα add-on που θα επιτρέψει την ενσωμάτωση με το PayFast. Αυτός ο άρθρο δείχνει πώς να ρυθμίσετε το PayFast ως την πύλη πληρωμών σας για να λαμβάνετε πληρωμές στο δίκτυο Ultimate Multisite σας.

Το πρώτο βήμα είναι η εγκατάσταση του add-on.

# Πώς να εγκαταστήσετε το add-on Payfast:

Μεταβείτε στη σελίδα των add-ons και επιλέξτε το add-on Ultimate Multisite: Payfast Gateway.

![Add-ons page](/img/admin/addons-page.png)

Θα εμφανιστεί ένα παράθυρο για το add-on. Κάντε κλικ στο Install Now (Εγκατάσταση Τώρα).

![Add-ons page with install option](/img/admin/addons-page.png)

Μετά την ολοκλήρωση της εγκατάστασης, θα ανακατευθυνθείτε στη σελίδα των plugins. Εδώ, απλώς κάντε κλικ στο Network Activate και το add-on _Payfast_ θα ενεργοποιηθεί στο δίκτυό σας.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# Ρύθμιση του PayFast

Μετά την εγκατάσταση του add-on, μπορείτε τώρα να μεταβείτε στις ρυθμίσεις του Ultimate Multisite και να αρχίσετε να ρυθμίζετε το PayFast.

Στη σελίδα **Ultimate Multisite settings**, μπορείτε να βρείτε την καρτέλα "Payments" (Πληρωμές). Σε αυτήν την καρτέλα, θα πρέπει να βρείτε το "**Payfast**" μαζί με ένα κουμπί αναπήδησης (toggle button) για να ενεργοποιήσετε ή να απενεργοποιήσετε αυτήν την πύλη πληρωμών.

![Payment gateway settings](/img/config/settings-payment-gateways.png)Συνεχίστε και κάντε κλικ στο κουμπί αναπήδησης για να το ενεργοποιήσετε. Τότε θα επεκταθεί το πλαίσιο για να εμφανίσει τα διαφορετικά πεδία που πρέπει να συμπληρώσετε, όπως το **Merchant ID, Merchant Key και PassPhrase.** Αυτές τις πληροφορίες μπορείτε να τις βρείτε στον λογαριασμό σας στο **Payfast**

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Το επόμενο βήμα είναι να συνδεθείτε στον λογαριασμό σας στο PayFast για να λάβετε τις πληροφορίες που χρειάζεστε.

Μετά τη σύνδεση, μπορείτε να δείτε αμέσως τα στοιχεία που χρειάζεστε στο **dashboard** του λογαριασμού σας στο **Payfast**.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->Μπορείτε απλώς να αντιγράψετε και να επικολλήσετε τα στοιχεία και να συμπληρώσετε τις λεπτομέρειες στις ρυθμίσεις του Ultimate Multisite σας. Στη συνέχεια, κάντε κλικ στο κουμπί **"Save settings"** για να αποθηκεύσετε τις αλλαγές.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Το αυτό είναι! Η ενσωμάτωση πληρωμών PayFast ολοκληρώθηκε!
