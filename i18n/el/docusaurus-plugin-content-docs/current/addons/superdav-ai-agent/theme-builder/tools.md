# Εργαλεία και Ρύθμιση

Το Theme Builder χρησιμοποιεί ένα σύνολο εργαλείων για να αναλύσει, να σχεδιάσει και να κατασκευάσει το WordPress site σας. Στην έκδοση 1.16.0, το **sd-ai-agent/site-scrape είναι πλέον ένα εργαλείο Tier 1**, καθιστώντας το διαθέσιμο εξ ορισμού στο Theme Builder.

## Διαθέσιμα Εργαλεία

### Εργαλεία Tier 1 (πάντα διαθέσιμα)

Τα εργαλεία Tier 1 είναι διαθέσιμα εξ ορισμού στο Theme Builder χωρίς οποιαδήποτε επιπλέον ρύθμιση.

#### sd-ai-agent/site-scrape

**Σκοπός:** Ανάλυση υπαρχόντων ιστοσελίδων για την εξαγωγή πηγών έμπνευσης σχεδιασμού, δομής περιεχομένου και λειτουργικότητας.

**Ικανότητες:**

- **Ανάλυση ιστοσελίδων** — συλλέγει και αναλύει ιστοσελίδες ανταγωνιστών ή πηγές έμπνευσης
- **Εξαγωγή σχεδιασμού** — αναγνωρίζει χρώματα, γραμματοσειρές και μοτίβα διάταξης
- **Δομή περιεχομένου** — κατανοεί την οργάνωση και τη ιεραρχία των σελίδων
- **Ανίχνευση λειτουργιών** — αναγνωρίζει plugins και λειτουργικότητα
- **Ανάλυση απόδοσης** — ελέγχει την ταχύτητα και τη βελτιστοποίηση της σελίδας
- **SEO ανάλυση** — αξιολογεί meta tags και δομημένα δεδομένα

**Χρήση:**

```
Analyze the design of example.com and use it as inspiration for my site.
```

**Τι εξάγει:**

- Παλίτα χρωμάτων και χρήση χρωμάτων
- Τυπογραφία (γραφικές απολήξεις και μεγέθη)
- Δομή διάταξης και αποστάσεις
- Μοτίβα πλοήγησης
- Οργάνωση περιεχομένου
- Εικόνες και χρήση μέσων
- Διαδραστικά στοιχεία
- Mobile responsiveness

**Περιορισμοί:**

- Δεν μπορεί να συλλέξει δεδομένα από ιστοσελίδες με προστασία κωδικού πρόσβασης
- Σεβόμαστε το robots.txt και τις πολιτικές των ιστοσελίδων
- Μπορεί να μην καταγράψει δυναμικό περιεχόμενο
- Οι ιστοσελίδες με πολύ JavaScript μπορεί να έχουν περιορισμένη ανάλυση
- Οι μεγάλες ιστοσελίδες μπορεί να χρειαστούν περισσότερο χρόνο για ανάλυση

### Εργαλεία Tier 2 (προαιρετικά)

Τα εργαλεία Tier 2 είναι διαθέσιμα όταν ενεργοποιούνται ρητά στις ρυθμίσεις του Theme Builder.

#### Advanced Analytics

Παρέχει λεπτομερείς μετρήσεις απόδοσης:

- Χρόνοι φόρτωσης σελίδων
- Core Web Vitals
- Βαθμολογία SEO
- Βαθμολογία προσβασιμότητας
- Βαθμολογία βέλτιστων πρακτικών

#### Content Optimizer

Αναλύει και προτείνει βελτιώσεις για:

- Διαβασιμότητα
- Βελτιστοποίηση SEO
- Χρήση λέξεων-κλειδιών
- Δομή περιεχομένου
- Τοποθέτηση calls-to-action

### Εργαλεία Tier 3 (Premium)

Τα εργαλεία Tier 3 απαιτούν επιπλέον ρύθμιση ή premium λειτουργίες.

#### AI Content Generator

Δημιουργεί περιεχόμενο για:

- Περιγραφές προϊόντων
- Σελίδες υπηρεσιών
- BlogPost
- Meta descriptions
- Calls-to-action copy

#### Advanced Design Tools

Παρέχει προηγμένες δυνατότητες σχεδιασμού:

- Δημιουργία Custom CSS
- Δημιουργία κινούμενων εικόνων (Animation)
- Σχεδιασμός διαδραστικών στοιχείων
- Προηγμένη θεωρία χρώματος
- Βελτιστοποίηση γραμματοσειρών (Typography)

## Ρύθμιση Εργαλείων

### Ενεργοποίηση Εργαλείων

Για να ενεργοποιήσετε επιπλέον εργαλεία στο Theme Builder:

1. Μεταβείτε στο **Gratis AI Agent → Settings**
2. Μεταβείτε στο **Theme Builder → Tools**
3. Ενεργοποιήστε/απενεργοποιήστε τα εργαλεία ανάλογα με τις ανάγκες
4. Αποθηκεύστε τις ρυθμίσεις

### Άδειες Εργαλείων

Ορισμένα εργαλεία απαιτούν άδειες:

- **Site scraping** — απαιτεί πρόσβαση στο internet
- **Analytics** — απαιτεί ενσωμάτωση Google Analytics
- **Content generation** — απαιτεί API keys
- **Advanced features** — μπορεί να απαιτεί premium subscription

### API Keys και Credentials

Ρυθμίστε τα API keys για τα εργαλεία που τα απαιτούν:

1. Μεταβείτε στο **Gratis AI Agent → Settings → API Keys**
2. Εισάγετε τα credentials για κάθε εργαλείο
3. Δοκιμάστε τη σύνδεση
4. Αποθηκεύστε με ασφάλεια

## Χρήση sd-ai-agent/site-scrape

### Βασική Χρήση

Ζητήστε από το Theme Builder να αναλύσει μια ιστοσελίδα:

```
Analyze the design of my competitor's site at competitor.com
and suggest design improvements for my site.
```

### Συγκεκριμένη Ανάλυση

Ζητήστε συγκεκριμένους τύπους ανάλυσης:

```
Extract the color palette from example.com and use it as inspiration.
```

```
Analyze the navigation structure of example.com and apply similar
organization to my site.
```

```
Check the mobile responsiveness of example.com and ensure my site
is equally responsive.
```

### Ανάλυση Σύγκρισης

Συγκρίνετε πολλαπλές ιστοσελίδες:

```
Compare the designs of site1.com and site2.com and create a design
that combines the best elements of both.
```

## Περιορισμοί και Σκέψεις για τα Εργαλεία

### Rate Limiting (Περιορισμός Ρυθμού)

- Το scraping περιορίζεται σε ρυθμό για να αποτραπεί η υπερφόρτωση του server
- Οι πολλαπλές αιτήσεις στην ίδια ιστοσελίδα μπορεί να περιοριστούν (throttled)
- Οι μεγάλες ιστοσελίδες μπορεί να χρειαστούν περισσότερο χρόνο για ανάλυση

### Ακρίβεια Περιεχομένου

- Το δυναμικό περιεχόμενο μπορεί να μην καταγραφεί πλήρως
- Το περιεχόμενο που δημιουργείται με JavaScript μπορεί να είναι ατελές
- Τα δεδομένα σε πραγματικό χρόνο μπορεί να μην αντανακλούνται
- Μερικό περιεχόμενο μπορεί να είναι πίσω από paywalls

### Νομικές και Ηθικές Σκέψεις

- Σεβαστείτε το robots.txt και τις πολιτικές των ιστοσελίδων
- Μην κάνετε scraping περιεχομένου που έχει πνευματικά δικαιώματα για αναπαραγωγή
- Χρησιμοποιήστε την ανάλυση για έμπνευση, όχι για αντιγραφή
- Βεβαιωθείτε ότι έχετε τα δικαιώματα χρήσης του εξαγόμενου περιεχομένου
- Τηρείτε τους όρους χρήσης των αναλυόμενων ιστοσελίδων

### Αντίκτυπος Απόδοσης

- Το scraping μεγάλων ιστοσελίδων μπορεί να διαρκέσει
- Οι πολλαπλές ταυτόχρονες συλλεções δεδομένων μπορεί να είναι πιο αργές
- Η σύνδεση στο δίκτυο επηρεάζει την ταχύτητα
- Τα μεγάλα datasets μπορεί να απαιτούν περισσότερη επεξεργασία

## Βέλτιστες Πρακτικές

### Χρήση Ανάλυσης Ιστοσελίδων

1. **Αναλύστε πολλαπλές ιστοσελίδες** — συλλέξτε έμπνευση από πολλαπλές πηγές
2. **Εστιάστε στη δομή** — μάθετε μοτίβα διάταξης και οργάνωσης
3. **Εξάγετε χρώματα** — χρησιμοποιήστε παλέτες χρωμάτων ως σημείο εκκίνησης
4. **Μελετήστε τη γραμματοσειρά** — αναγνωρίστε συνδυασμούς γραμματοσειρών που σας αρέσουν
5. **Ελέγξτε την πλοήγηση** — κατανοήστε την οργάνωση του μενού

### Ηθικό Scraping

1. **Σεβαστείτε το robots.txt** — ακολουθήστε τις οδηγίες της ιστοσελίδας
2. **Μην αντιγράφετε περιεχόμενο** — χρησιμοποιήστε την ανάλυση μόνο για έμπνευση
3. **Επιβεβαιώστε τα δικαιώματα** — βεβαιωθείτε ότι μπορείτε να χρησιμοποιήσετε το εξαγόμενο περιεχόμενο
4. **Αναφέρετε τις πηγές** — δώστε πίστωση στις πηγές έμπνευσης
5. **Τηρείτε τους όρους** — συμμορφωθείτε με τους όρους χρήσης της ιστοσελίδας

### Μέγιστο Κάννα τα Εργαλεία

1. **Είστε συγκεκριμένοι** — ζητήστε συγκεκριμένους τύπους ανάλυσης
2. **Παρέχετε πλαίσιο** — εξηγήστε τον σκοπό της ιστοσελίδας σας
3. **Ορίστε προσδοκίες** — περιγράψτε τους στόχους σχεδιασμού σας
4. **Επαναλάβετε** — βελτιώστε με βάση τα αποτελέσματα
5. **Συνδυάστε εργαλεία** — χρησιμοποιήστε πολλαπλά εργαλεία για ολοκληρωμένη ανάλυση

## Αντιμετώπιση Προβλημάτων

### Η Ιστοσελίδα Δεν Συλλέγει Δεδομένα

- Ελέγξτε αν η ιστοσελίδα είναι δημόσια προσβάσιμη
- Επιβεβαιώστε ότι το robots.txt επιτρέπει το scraping
- Δοκιμάστε μια διαφορετική ιστοσελίδα
- Ελέγξτε τη σύνδεση στο internet
- Επικοινωνήστε με την υποστήριξη αν το πρόβλημα συνεχίζεται

### Ατελής Ανάλυση

- Η ιστοσελίδα μπορεί να έχει δυναμικό περιεχόμενο
- Το JavaScript μπορεί να μην έχει ρυθμιστεί πλήρως
- Οι μεγάλες ιστοσελίδες μπορεί να λήξουν (timeout)
- Δοκιμάστε να αναλύσετε συγκεκριμένες σελίδες αντίθετα
- Ζητήστε συγκεκριμένους τύπους ανάλυσης

### Αργή Απόδοση

- Οι μεγάλες ιστοσελίδες χρειάζονται περισσότερο χρόνο για ανάλυση
- Οι πολλαπλές ταυτόχρονες συλλεções δεδομένων είναι πιο αργές
- Η σύνδεση στο δίκτυο επηρεάζει την ταχύτητα
- Δοκιμάστε σε ώρες χαμηλής περιόδου χρήσης
- Αναλύστε πρώτα τα μικρότερα τμήματα
