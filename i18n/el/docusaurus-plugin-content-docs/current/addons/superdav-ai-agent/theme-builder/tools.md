# Εργαλεία και Ρύθμιση Παραμέτρων {#tools-and-configuration}

Το Theme Builder χρησιμοποιεί ένα σύνολο εργαλείων για να αναλύει, να σχεδιάζει και να δημιουργεί τον WordPress ιστότοπό σας. Στην έκδοση 1.16.0, **το sd-ai-agent/site-scrape είναι πλέον εργαλείο Βαθμίδας 1**, καθιστώντας το διαθέσιμο από προεπιλογή στο Theme Builder. Το Superdav AI Agent v1.18.0 εισάγει επίσης ένα ξεχωριστά διανεμόμενο πακέτο συνοδευτικών Advanced για αξιόπιστες ροές εργασίας προγραμματιστών· αυτά τα εργαλεία δεν αποτελούν μέρος του βασικού πακέτου WordPress.org και πρέπει να εγκατασταθούν και να εξουσιοδοτηθούν ξεχωριστά.

## Διαθέσιμα Εργαλεία {#available-tools}

### Εργαλεία Βαθμίδας 1 (Πάντα Διαθέσιμα) {#tier-1-tools-always-available}

Τα εργαλεία Βαθμίδας 1 είναι διαθέσιμα από προεπιλογή στο Theme Builder χωρίς καμία πρόσθετη ρύθμιση παραμέτρων.

#### sd-ai-agent/site-scrape {#sd-ai-agentsite-scrape}

**Σκοπός:** Ανάλυση υπαρχόντων ιστοτόπων για εξαγωγή σχεδιαστικής έμπνευσης, δομής περιεχομένου και λειτουργικότητας.

**Δυνατότητες:**

- **Ανάλυση ιστοτόπου** — συλλογή και ανάλυση ανταγωνιστικών ιστοτόπων ή ιστοτόπων έμπνευσης
- **Εξαγωγή σχεδιασμού** — εντοπισμός χρωμάτων, γραμματοσειρών και μοτίβων διάταξης
- **Δομή περιεχομένου** — κατανόηση της οργάνωσης και ιεραρχίας σελίδων
- **Εντοπισμός χαρακτηριστικών** — εντοπισμός plugins και λειτουργικότητας
- **Ανάλυση απόδοσης** — έλεγχος ταχύτητας σελίδας και βελτιστοποίησης
- **Ανάλυση SEO** — έλεγχος meta tags και δομημένων δεδομένων

**Χρήση:**

```
Ανάλυσε τον σχεδιασμό του example.com και χρησιμοποίησέ τον ως έμπνευση για τον ιστότοπό μου.
```

**Τι Εξάγει:**

- Παλέτα χρωμάτων και χρήση χρωμάτων
- Τυπογραφία (γραμματοσειρές και μεγέθη)
- Δομή διάταξης και αποστάσεις
- Μοτίβα πλοήγησης
- Οργάνωση περιεχομένου
- Χρήση εικόνων και πολυμέσων
- Διαδραστικά στοιχεία
- Απόκριση σε κινητές συσκευές

**Περιορισμοί:**

- Δεν μπορεί να συλλέξει δεδομένα από ιστοτόπους προστατευμένους με κωδικό πρόσβασης
- Σέβεται το robots.txt και τις πολιτικές του ιστοτόπου
- Ενδέχεται να μην καταγράψει δυναμικό περιεχόμενο
- Ιστότοποι με έντονη χρήση JavaScript ενδέχεται να έχουν περιορισμένη ανάλυση
- Μεγάλοι ιστότοποι ενδέχεται να χρειαστούν περισσότερο χρόνο για ανάλυση

### Εργαλεία Βαθμίδας 2 (Προαιρετικά) {#tier-2-tools-optional}

Τα εργαλεία Βαθμίδας 2 είναι διαθέσιμα όταν ενεργοποιούνται ρητά στις ρυθμίσεις του Theme Builder.

#### Προηγμένα Analytics {#advanced-analytics}

Παρέχει λεπτομερείς μετρήσεις απόδοσης:

- Χρόνοι φόρτωσης σελίδας
- Core Web Vitals
- Βαθμολογία SEO
- Βαθμολογία προσβασιμότητας
- Βαθμολογία βέλτιστων πρακτικών

#### Βελτιστοποιητής Περιεχομένου {#content-optimizer}

Αναλύει και προτείνει βελτιώσεις για:

- Αναγνωσιμότητα
- Βελτιστοποίηση SEO
- Χρήση λέξεων-κλειδιών
- Δομή περιεχομένου
- Τοποθέτηση call-to-action

### Εργαλεία Βαθμίδας 3 (Premium) {#tier-3-tools-premium}

Τα εργαλεία Βαθμίδας 3 απαιτούν πρόσθετη ρύθμιση παραμέτρων ή premium χαρακτηριστικά.

#### Γεννήτρια Περιεχομένου AI {#ai-content-generator}

Δημιουργεί περιεχόμενο για:

- Περιγραφές προϊόντων
- Σελίδες υπηρεσιών
- Αναρτήσεις ιστολογίου
- Meta περιγραφές
- Κείμενο call-to-action

#### Προηγμένα Εργαλεία Σχεδιασμού {#advanced-design-tools}

Παρέχει προηγμένες δυνατότητες σχεδιασμού:

- Δημιουργία προσαρμοσμένου CSS
- Δημιουργία animation
- Σχεδιασμός διαδραστικών στοιχείων
- Προηγμένη θεωρία χρωμάτων
- Βελτιστοποίηση τυπογραφίας

### Προηγμένα Συνοδευτικά Εργαλεία (Ξεχωριστό Πακέτο) {#advanced-companion-tools-separate-package}

Το πακέτο συνοδευτικών Advanced διανέμεται ξεχωριστά από τη βασική έκδοση του Superdav AI Agent. Προορίζεται για αξιόπιστα περιβάλλοντα προγραμματιστών και ιδιοκτητών ιστοτόπων, όπου οι διαχειριστές επιτρέπουν ρητά στον agent να χρησιμοποιεί εργαλεία υψηλού αντίκτυπου.

Οι δυνατότητες των συνοδευτικών Advanced μπορούν να περιλαμβάνουν:

- **Εργαλεία συστήματος αρχείων** — επιθεώρηση και τροποποίηση εγκεκριμένων αρχείων σε αξιόπιστα περιβάλλοντα
- **Εργαλεία βάσης δεδομένων** — υποβολή ερωτημάτων ή ενημέρωση δεδομένων ιστοτόπου όταν έχει δοθεί ρητή εξουσιοδότηση
- **Εργαλεία WP-CLI** — εκτέλεση εντολών συντήρησης και επιθεώρησης WordPress
- **Εργαλεία REST dispatcher** — κλήση καταχωρισμένων REST endpoints μέσω ελεγχόμενων ροών εργασίας dispatch
- **Εργαλεία δημιουργίας plugin** — σκαλωσιά και επαναληπτική επεξεργασία κώδικα plugin
- **Εργαλεία στιγμιότυπων Git** — δημιουργία στιγμιότυπων πριν από ριψοκίνδυνες λειτουργίες προγραμματιστών
- **Εργαλεία διαχείρισης χρηστών** — βοήθεια στον έλεγχο ή την προσαρμογή εγγραφών χρηστών όπου επιτρέπεται
- **Εργαλεία benchmark** — εκτέλεση benchmarks απόδοσης ή δυνατοτήτων για έλεγχο ανάπτυξης

Μην τα τεκμηριώνετε ως γενικά διαθέσιμα εργαλεία Theme Builder για κάθε εγκατάσταση. Είναι διαθέσιμα μόνο όταν το πακέτο συνοδευτικών Advanced έχει εγκατασταθεί, ενεργοποιηθεί και περιοριστεί σε αξιόπιστους διαχειριστές.

## Ρύθμιση Παραμέτρων Εργαλείων {#tool-configuration}

### Ενεργοποίηση Εργαλείων {#enabling-tools}

Για να ενεργοποιήσετε πρόσθετα εργαλεία στο Theme Builder:

1. Μεταβείτε στο **Gratis AI Agent → Ρυθμίσεις**
2. Πηγαίνετε στο **Theme Builder → Εργαλεία**
3. Ενεργοποιήστε/απενεργοποιήστε εργαλεία όπως χρειάζεται
4. Αποθηκεύστε τις ρυθμίσεις

### Δικαιώματα Εργαλείων {#tool-permissions}

Ορισμένα εργαλεία απαιτούν δικαιώματα:

- **Συλλογή δεδομένων ιστοτόπου** — απαιτεί πρόσβαση στο διαδίκτυο
- **Analytics** — απαιτεί ενσωμάτωση Google Analytics
- **Δημιουργία περιεχομένου** — απαιτεί API keys
- **Προηγμένα χαρακτηριστικά** — ενδέχεται να απαιτούν premium συνδρομή
- **Συνοδευτικά εργαλεία Advanced** — απαιτούν το ξεχωριστά διανεμόμενο πακέτο Advanced και ρητή εμπιστοσύνη διαχειριστή

### API Keys και Διαπιστευτήρια {#api-keys-and-credentials}

Ρυθμίστε API keys για εργαλεία που τα απαιτούν:

1. Μεταβείτε στο **Gratis AI Agent → Ρυθμίσεις → API Keys**
2. Εισαγάγετε διαπιστευτήρια για κάθε εργαλείο
3. Δοκιμάστε τη σύνδεση
4. Αποθηκεύστε με ασφάλεια

### Εγκατάσταση του Πακέτου Συνοδευτικών Advanced {#installing-the-advanced-companion-package}

Το πακέτο συνοδευτικών Advanced κυκλοφορεί ξεχωριστά από το βασικό ZIP του WordPress.org. Εγκαταστήστε το μόνο από το επίσημο κανάλι διανομής του έργου για την έκδοση και, στη συνέχεια, περιορίστε την πρόσβαση σε αξιόπιστους διαχειριστές. Ελέγξτε τις πύλες ανθρώπινης έγκρισης και τη συμπεριφορά στιγμιότυπων git πριν ενεργοποιήσετε εργαλεία συστήματος αρχείων, βάσης δεδομένων, WP-CLI, REST dispatcher, δημιουργίας plugin, διαχείρισης χρηστών ή benchmark σε έναν παραγωγικό ιστότοπο.

## Περιορισμοί και Παράμετροι Εργαλείων {#tool-limitations-and-considerations}

### Περιορισμός Ρυθμούλήψεων {#rate-limiting}

- Το scraping έχει περιορισμό ρυθμού για την αποφυγή υπερφόρτωσης του server
- Πολλαπλά αιτήματα προς τον ίδιο ιστότοπο μπορεί να περιοριστούν
- Οι μεγάλοι ιστότοποι μπορεί να χρειαστούν περισσότερο χρόνο για ανάλυση

### Ακρίβεια περιεχομένου {#content-accuracy}

- Το δυναμικό περιεχόμενο μπορεί να μην καταγραφεί πλήρως
- Το περιεχόμενο που αποδίδεται με JavaScript μπορεί να είναι ελλιπές
- Τα δεδομένα σε πραγματικό χρόνο μπορεί να μην αντικατοπτρίζονται
- Κάποιο περιεχόμενο μπορεί να βρίσκεται πίσω από paywalls

### Νομικές και ηθικές παράμετροι {#legal-and-ethical-considerations}

- Σεβαστείτε το robots.txt και τις πολιτικές του ιστότοπου
- Μην κάνετε scraping περιεχομένου που προστατεύεται από πνευματικά δικαιώματα για αναπαραγωγή
- Χρησιμοποιήστε την ανάλυση για έμπνευση, όχι για αντιγραφή
- Επαληθεύστε ότι έχετε δικαιώματα χρήσης του εξαγόμενου περιεχομένου
- Ακολουθήστε τους όρους υπηρεσίας των αναλυόμενων ιστότοπων

### Επίδραση στην απόδοση {#performance-impact}

- Το scraping μεγάλων ιστότοπων μπορεί να πάρει χρόνο
- Πολλαπλά ταυτόχρονα scrapes μπορεί να είναι πιο αργά
- Η συνδεσιμότητα δικτύου επηρεάζει την ταχύτητα
- Τα μεγάλα σύνολα δεδομένων μπορεί να απαιτούν περισσότερη επεξεργασία

## Βέλτιστες πρακτικές {#best-practices}

### Χρήση ανάλυσης ιστότοπου {#using-site-analysis}

1. **Αναλύστε πολλούς ιστότοπους** — συλλέξτε έμπνευση από διάφορες πηγές
2. **Εστιάστε στη δομή** — μάθετε μοτίβα διάταξης και οργάνωσης
3. **Εξαγάγετε χρώματα** — χρησιμοποιήστε παλέτες χρωμάτων ως σημεία εκκίνησης
4. **Μελετήστε την τυπογραφία** — εντοπίστε συνδυασμούς γραμματοσειρών που σας αρέσουν
5. **Ελέγξτε την πλοήγηση** — κατανοήστε την οργάνωση του μενού

### Ηθικό scraping {#ethical-scraping}

1. **Σεβαστείτε το robots.txt** — ακολουθήστε τις οδηγίες του ιστότοπου
2. **Μην αντιγράφετε περιεχόμενο** — χρησιμοποιήστε την ανάλυση μόνο για έμπνευση
3. **Επαληθεύστε τα δικαιώματα** — βεβαιωθείτε ότι μπορείτε να χρησιμοποιήσετε το εξαγόμενο περιεχόμενο
4. **Αναφέρετε τις πηγές** — αποδώστε εύσημα στις πηγές έμπνευσης
5. **Ακολουθήστε τους όρους** — συμμορφωθείτε με τους όρους υπηρεσίας του ιστότοπου

### Μεγιστοποίηση της αποτελεσματικότητας του εργαλείου {#maximizing-tool-effectiveness}

1. **Να είστε συγκεκριμένοι** — ζητήστε συγκεκριμένους τύπους ανάλυσης
2. **Παρέχετε πλαίσιο** — εξηγήστε τον σκοπό του ιστότοπού σας
3. **Ορίστε προσδοκίες** — περιγράψτε τους στόχους σχεδιασμού σας
4. **Επαναλάβετε** — βελτιώστε με βάση τα αποτελέσματα
5. **Συνδυάστε εργαλεία** — χρησιμοποιήστε πολλαπλά εργαλεία για ολοκληρωμένη ανάλυση

## Αντιμετώπιση προβλημάτων {#troubleshooting}

### Ο ιστότοπος δεν μπορεί να γίνει scrape {#site-wont-scrape}

- Ελέγξτε αν ο ιστότοπος είναι δημόσια προσβάσιμος
- Επαληθεύστε ότι το robots.txt επιτρέπει το scraping
- Δοκιμάστε διαφορετικό ιστότοπο
- Ελέγξτε τη σύνδεση στο διαδίκτυο
- Επικοινωνήστε με την υποστήριξη αν το πρόβλημα παραμένει

### Ελλιπής ανάλυση {#incomplete-analysis}

- Ο ιστότοπος μπορεί να έχει δυναμικό περιεχόμενο
- Η JavaScript μπορεί να μην αποδίδεται πλήρως
- Οι μεγάλοι ιστότοποι μπορεί να λήξουν λόγω χρονικού ορίου
- Δοκιμάστε να αναλύσετε συγκεκριμένες σελίδες αντί για αυτό
- Ζητήστε συγκεκριμένους τύπους ανάλυσης

### Αργή απόδοση {#slow-performance}

- Οι μεγάλοι ιστότοποι χρειάζονται περισσότερο χρόνο για ανάλυση
- Πολλαπλά ταυτόχρονα scrapes είναι πιο αργά
- Η συνδεσιμότητα δικτύου επηρεάζει την ταχύτητα
- Δοκιμάστε σε ώρες χαμηλής αιχμής
- Αναλύστε πρώτα μικρότερες ενότητες
