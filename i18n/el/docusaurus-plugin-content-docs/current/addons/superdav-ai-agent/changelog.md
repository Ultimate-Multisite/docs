# Αρχείο αλλαγών {#changelog}

## 1.18.0 — Κυκλοφόρησε στις 2026-06-29 {#1180--released-on-2026-06-29}

### Νέα {#new}

- **Εργαλεία Google Calendar** — ανάγνωση ρυθμισμένων ημερολογίων και συμβάντων για αυτοματισμούς που λαμβάνουν υπόψη το πρόγραμμα
- **Αντιστοίχιση επαφών και βοηθοί συμμετεχόντων** — αντιστοίχιση συμμετεχόντων ημερολογίου με χρήστες και επαφές του ιστότοπου
- **Πύλες ανθρώπινης έγκρισης και εγγραφές υπενθυμίσεων** — παύση αυτοματισμών για έλεγχο και αποφυγή διπλών ειδοποιήσεων
- **Πάροχος SMS TextBee** — αποστολή ρυθμισμένων ειδοποιήσεων μηνυμάτων κειμένου μέσω TextBee
- **Προηγμένο συνοδευτικό πακέτο** — προσθήκη εργαλείων συστήματος αρχείων αξιόπιστου προγραμματιστή, βάσης δεδομένων, WP-CLI, REST dispatcher, δημιουργού plugin, στιγμιότυπου git, διαχείρισης χρηστών και benchmark, που διανέμονται ξεχωριστά από την έκδοση του WordPress.org

### Βελτιώθηκε {#improved}

- **Ρύθμιση διαχειριζόμενης υπηρεσίας Superdav** — προσθήκη endpoint φιλοξενούμενης υπηρεσίας και αυτόματης παροχής σύνδεσης για υποστηριζόμενους ιστότοπους
- **Πακετάρισμα έκδοσης** — δημιουργία ξεχωριστών ZIP για τον πυρήνα και το Advanced, δημοσίευση και των δύο στο GitHub και αποστολή μόνο του πακέτου πυρήνα στο WordPress.org

### Διορθώθηκε {#fixed}

- **Αξιοπιστία αιτημάτων AI** — βελτίωση επιλογής μοντέλου, χρονικών ορίων αιτημάτων, προεπιλεγμένου μοντέλου ρύθμισης, χειρισμού κειμένου συλλογισμού και καθοδήγησης επανάληψης για μη έγκυρες κλήσεις εργαλείων
- **Ενίσχυση ημερολογίου και υπενθυμίσεων** — ενίσχυση token του Google Calendar και αποδιπλοποίηση υπενθυμίσεων
- **Συνέχιση εισαγωγής και εγκρίσεων** — διόρθωση εκκινήσεων εισαγωγής στο frontend και συνέχισης επιβεβαιωμένων δυνατοτήτων
- **Ζητήματα ελέγχου πακεταρίσματος WordPress.org** — αντιμετώπιση σχολίων ελέγχου πακεταρίσματος για την έκδοση πυρήνα

## 1.16.0 — Κυκλοφόρησε στις 2026-05-20 {#1160--released-on-2026-05-20}

### Νέα {#new-1}

- **Δυνατότητα δημιουργίας Logo SVG** — το Theme Builder μπορεί πλέον να δημιουργεί και να ενσωματώνει προσαρμοσμένα SVG λογότυπων με απολύμανση ασφαλή ως προς τα namespace
- **Μεταφόρτωση φωτογραφίας στη συνέντευξη ανακάλυψης** — η συνέντευξη ανακάλυψης του Theme Builder περιλαμβάνει πλέον βήμα μεταφόρτωσης φωτογραφίας για πλουσιότερο πλαίσιο σχεδιασμού
- **Δυνατότητα επικύρωσης αντίθεσης παλέτας** — έλεγχος ζευγών χρωμάτων για συμμόρφωση με WCAG πριν από την εφαρμογή σε ένα θέμα
- **Μενού φιλοξενίας** — το Theme Builder μπορεί πλέον να δημιουργεί δομημένες σελίδες μενού φαγητού και ποτών για επιχειρήσεις φιλοξενίας
- **Απόδοση προεπισκόπησης σε υπολογιστή και κινητό** — προεπισκόπηση του σχεδιασμού σας σε υπολογιστές και κινητές συσκευές κατά την επιλογή κατεύθυνσης σχεδιασμού
- **Παράμετρος ετικέτας πλοήγησης** — η δυνατότητα Create Menu υποστηρίζει πλέον ξεχωριστό `navigation_label` ανεξάρτητο από τον τίτλο της σελίδας
- **Διαθεσιμότητα εργαλείου Tier 1** — το sd-ai-agent/site-scrape είναι πλέον εργαλείο Tier 1 διαθέσιμο από προεπιλογή στο Theme Builder

### Διορθώθηκε {#fixed-1}

- **Cache AI Client** — πλέον υποστηρίζεται από transients για διατήρηση μεταξύ αιτημάτων, αποτρέποντας απώλεια δεδομένων σε μακροχρόνιες εργασίες agent
- **Σύνδεσμοι ενεργειών γραμμής plugin** — διορθώθηκαν και μετονομάστηκαν για σαφήνεια

## 1.10.0 — Κυκλοφόρησε στις 2026-05-05 {#1100--released-on-2026-05-05}

### Νέα {#new-2}

- **Αναζήτηση διαδικτύου Tavily** — προσθήκη του Tavily ως παρόχου αναζήτησης για πλουσιότερα αποτελέσματα αναζήτησης στο διαδίκτυο παράλληλα με το Brave Search
- **Ενσωματωμένες δεξιότητες με επίγνωση θέματος** — οδηγοί δεξιοτήτων για Block Themes, Classic Themes, Kadence Blocks και Kadence Theme πλέον περιλαμβάνονται με το plugin
- **Δυνατότητα φόρμας επικοινωνίας κατασκευαστή ιστότοπου** — προσθήκη φόρμας επικοινωνίας σε οποιαδήποτε σελίδα απευθείας από τη διεπαφή συνομιλίας

### Βελτιώθηκε {#improved-1}

- **Ανασχεδιασμός ενσωμάτωσης WooCommerce** — πλέον χρησιμοποιεί εγγενή WooCommerce API για καλύτερη αξιοπιστία και συμβατότητα
- **Η λίστα παρόχων ανανεώνεται αυτόματα** — όταν οποιοδήποτε plugin ενεργοποιείται ή απενεργοποιείται

### Διορθώθηκε {#fixed-2}

- **Δυνατότητα navigate-to** — διορθώθηκε ατελείωτος βρόχος επαναφόρτωσης σε ορισμένες σελίδες διαχείρισης
- **Δυνατότητα list-posts** — πλέον επιλύει σωστά ονόματα κατηγοριών και ετικετών σε slugs
- **Εντολές WP-CLI** — αποκαταστάθηκαν ψευδώνυμα namespace που έλειπαν μετά από προηγούμενο ανασχεδιασμό
- **Αυτοματισμός συμβάντων** — χειρίζεται ομαλά ιστότοπους όπου οι πίνακες αυτοματισμού δεν έχουν ακόμη δημιουργηθεί
- **Δυνατότητα memory-save** — πλέον χρησιμοποιεί το σωστό πρόθεμα namespace στον δημιουργό οδηγιών συστήματος
- **Αποτελέσματα εργαλείων scalar** — πλέον περιτυλίγονται σωστά πριν επιστραφούν στο AI
- **Στατιστικά χρήσης** — πλέον χειρίζονται σωστά τη μορφή κλειδιού παλαιού τύπου για δυνατότητες κατά την αναβάθμιση από παλαιότερες εκδόσεις
