# Changelog

## 1.16.0 — Released on 2026-05-20

### Νέες λειτουργίες (New)

- **Δυνατότητα δημιουργίας SVG Logo** — Το Theme Builder μπορεί πλέον να δημιουργεί και να ενσωματώνει προσαρμοσμένα SVG logos με ασφαλή αποστείρωση (sanitisation) με namespace.
- **Επισύναψη φωτογραφίας στην συνέντευξη ανακάλυψης (discovery interview)** — Η συνέντευξη ανακάλυψης του Theme Builder περιλαμβάνει πλέον ένα βήμα για την επισύναψη φωτογραφίας, επιτρέποντας ένα πλουσιότερο πλαίσιο σχεδιασμού.
- **Δυνατότητα επικύρωσης αντίθεσης παλέτας (Validate Palette Contrast)** — Ελέγξτε τις ζεύξεις χρωμάτων για συμμόρφωση με τα WCAG πριν τα εφαρμόσετε σε ένα θέμα.
- **Μενύα για τοποθεστικό/καλυμμένο χώρο (Hospitality menus)** — Το Theme Builder μπορεί πλέον να δημιουργεί δομημένα σελίδες με μενού φαγητού και ποτών για επιχειρήσεις του τομέα του hospitality.
- **Προεπισκόπηση για desktop και mobile** — Προεπισκόprite τον σχεδιασμό σας σε συσκευές desktop και mobile κατά την επιλογή της κατεύθυνσης σχεδιασμού.
- **Παράμετρος ετικέτας πλοήγησης (Navigation label)** — Η λειτουργία «Δημιουργία Μενού» (Create Menu) υποστηρίζει πλέον μια ξεχωριστή `navigation_label` που διαφέρει από τον τίτλο της σελίδας.
- **Διαθεσιμότητα εργαλείου Tier 1** — Το sd-ai-agent/site-scrape είναι πλέον εργαλείο Tier 1 και είναι διαθέσιμο εξ ορισμού στο Theme Builder.

### Διορθώσεις (Fixed)

- **Cache του AI Client** — Τώρα υποστηρίζεται από transients για διατήρηση μεταξύ αιτημάτων (cross-request persistence), αποτρέποντας απώλεια δεδομένων σε εργασίες agent που εκτελούνται για μεγάλο διάστημα.
- **Σύνδεσμοι δράσης γραμμής plugin** — Διορθώθηκαν και ανανεώθηκαν ονόματα για μεγαλύτερη σαφήνεια.

## 1.10.0 — Released on 2026-05-05

### Νέες λειτουργίες (New)

- **Αναζήτηση στο internet με Tavily** — Προστέθηκε το Tavily ως πάροχος αναζήτησης για πλουσιότερες תוצאες αναζήτησης στο internet, εκτός από το Brave Search.
- **Ενσωματωμένες δεξιότητες ευαίσθητες στο θέμα (Theme-aware built-in skills)** — Τα Block Themes, Classic Themes, Kadence Blocks και τα οδηγούς δεξιοτήτων του Kadence Theme έρχονται πλέον μαζί με το plugin.
- **Δυνατότητα φόρμας επικοινωνίας στο site builder** — Προσθέστε μια φόρμα επικοινωνίας σε οποιαδήποτε σελίδα απευθείας από τη διεπαφή chat.

### Βελτιώσεις (Improved)

- **Refactored ενσωμάτωση WooCommerce** — Χρησιμοποιεί πλέον τις native WooCommerce APIs για καλύτερη αξιοπιστία και συμβατότητα.
- **Αυτόματη ανανέωση λίστας παρόχων** — Όταν ενεργοποιείται ή απενεργοποιείται οποιοδήποτε plugin.

### Διορθώσεις (Fixed)

- **λειτουργία navigate-to** — Διορθώθηκε η άπειρη επανεκκίνηση (infinite reload loop) σε ορισμένες σελίδες διαχείρισης (admin pages).
- **λειτουργία list-posts** — Τώρα αναλύει σωστά τα ονόματα κατηγοριών και ετικετών σε slugs.
- **Εντολές WP-CLI** — Επαναφέρθηκαν οι χαμένοι aliases namespace μετά από μια προηγούμενη αναδιάρθρωση (refactor).
- **Ευ자동ποίηση γεγονότων (Event automation)** — Διαχειρίζεται ομαλά τα sites όπου οι πίνακες αυτοματοποίησης δεν έχουν ακόμη δημιουργηθεί.
- **λειτουργία memory-save** — Χρησιμοποιεί πλέον το σωστό prefix namespace στον δημιουργό οδηγιών συστήματος (system instruction builder).
- **Αποτελέσματα εργαλείου Scalar** — Τώρα συσκευάζονται σωστά πριν επιστραφούν στο AI.
- **Στατιστικά χρήσης (Usage statistics)** — Διαχειρίζονται σωστά το παλαιότερο μορφότυπο κλειδιού λειτουργίας (legacy ability key format) κατά την αναβάθμιση από παλαιότερες εκδόσεις.
