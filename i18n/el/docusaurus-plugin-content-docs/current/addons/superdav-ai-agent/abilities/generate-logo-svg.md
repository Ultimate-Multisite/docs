# Δημιουργία SVG Λογότυπου

Η δυνατότητα **Δημιουργία SVG Λογότυπου** επιτρέπει στο Theme Builder να δημιουργεί και να ενσωματώνει προσαρμοσμένα SVG λογότυπα απευθείας στην ιστοσελίδα σας WordPress, με αυτόμαλο και ασφαλές καθαρισμό (sanitisation) του namespace.

## Επισκόπηση {#overview}

Αυτή η δυνατότητα δημιουργεί λογότυπα σε μορφή διανυσματικών γραφικών (SVG) που βασίζονται στην κατεύθυνση branding και τις σχεδιαστικές προτιμήσεις της ιστοσελίδας σας. Τα δημιουργημένα SVG καθαρίζονται αυτόματα για να διασφαλιστεί ότι είναι ασφαλή για χρήση στο WordPress, διατηρώντας παράλληλα την οπτική ακεραιότητα.

## Παράμετροι {#parameters}

| Παράμετρος | Τύπος | Υποχρεωτική | Περιγραφή |
|-----------|------|----------|-------------|
| `site_name` | string | Ναι | Το όνομα της ιστοσελίδας για την οποία θα δημιουργηθεί το λογότυπο |
| `style` | string | Ναι | Το στυλ σχεδιασμού (π.χ., "modern", "classic", "minimalist", "playful") |
| `colors` | array | Όχι | Πίνακας κωδικών χρώματος hex για χρήση στο λογότυπο (π.χ., `["#678233", "#ffffff"]`) |
| `width` | number | Όχι | Πλάτος SVG σε pixels (προεπιλεγμένο: 200) |
| `height` | number | Όχι | Ύψος SVG σε pixels (προεπιλεγμένο: 200) |
| `include_text` | boolean | Όχι | Αν πρέπει να συμπεριληφθεί το όνομα της ιστοσελίδας ως κείμενο στο λογότυπο (προεπιλεγμένο: true) |

## Μορφή Εξόδου {#output-format}

Η δυνατότητα επιστρέφει μια συμβολοσειρά SVG με την ακόλουθη δομή:

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## Συμπεριφορά Καθαρισμού SVG {#svg-sanitisation-behaviour}

Τα δημιουργημένα SVG υποβάλλονται αυτόμαλο και ασφαλής καθαρισμός namespace για να:

- **Αφαιρούν μη ασφαλή χαρακτηριστικά** — αφαιρούν event handlers, scripts και δυνητικά επικίνδυνα χαρακτηριστικά
- **Διατηρούν namespaces** — διατηρούν τα namespaces SVG (xmlns, xlink) για σωστή εμφάνιση
- **Ελέγχουν τη δομή** — διασφαλίζουν ότι το SVG συμμορφώνεται με τα πρότυπα W3C
- **Κωδικοποιούν οντότητες** — κωδικοποιούν σωστά τα ειδικά χαρακτήρες στο περιεχόμενο κειμένου
- **Αφαιρούν εξωτερικές αναφορές** — αφαιρούν εξωτερικά style sheets και αναφορές εικόνων

Αυτό διασφαλίζει ότι το SVG είναι ασφαλές για ενσωμάτωση απευθείας στο WordPress χωρίς να απαιτείται επιπλέον καθαρισμός.

## Παράδειγμα Χρήσης {#usage-example}

**Prompt:**
```
Generate a modern logo for my tech startup called "CloudSync" using blue and white colors.
```

**Αποτέλεσμα:**
Η δυνατότητα δημιουργεί ένα SVG λογότυπο που:
- Περιλαμβάνει το όνομα της ιστοσελίδας "CloudSync"
- Χρησιμοποιεί το καθορισμένο μπλε και λευκό χρώμα
- Τηλεπάτη από σύγχρονα σχέδια
- Είναι αυτόματα καθαρισμένο και έτοιμο προς χρήση

## Ενσωμάτωση με το Theme Builder {#integration-with-theme-builder}

Όταν χρησιμοποιείτε την επιλογή κατεύθυνσης σχεδιασμού του Theme Builder, η δυνατότητα Generate Logo SVG:

1. Ανาวิεάζει την κατεύθυνση σχεδιασμού και τη παλέτα χρώματος.
2. Δημιουργεί ένα προσαρμοσμένο SVG λογότυπο που ταιριάζει στις προτιμήσεις σας.
3. Ενσωματώνει αυτόματα το λογότυπο στην περιοχή branding/header της ιστοσελίδας σας.
4. Αποθηκεύει το SVG ως προσαρμοσμένο λογότυπο στα μέσα του WordPress.

## Βέλτιστες Πρακτικές {#best-practices}

- **Παρέχετε σαφείς προτιμήσεις στυλ** — περιγράψτε το στυλ σχεδιασμού που θέλετε (modern, classic, playful κ.λπ.).
- **Καθορίστε χρώματα** — συμπεριλάβετε τα brand colors σας για συνέπεια.
- **Ελέγξτε την εμφάνιση** — επιβεβαιώστε ότι το λογότυπο εμφανίζεται σωστά σε διαφορετικά μεγέθη οθόνης.
- **Προσαρμόστε περαιτέρω** — χρησιμοποιήστε τα εργαλεία προσαρμογής λογότυπου του WordPress για να ρυθμίσετε το μέγεθος και τη τοποθέτηση.

## Περιορισμοί {#limitations}

- Τα λογότυπα SVG δημιουργούνται ως στατικά γραφικά (όχι κινούμενα).
- Τα πολύπλοκα λογότυπα με πολλά στοιχεία μπορεί να απαιτήσουν χειροκίνη %>% refinement.
- Δεν υποστηρίζονται προσαρμοσμένα γραμματοσειρές (fonts); το κείμενο χρησιμοποιεί τα system fonts.
- Πολύ μεγάλα ή πολύ μικρά διαστάσεις μπορεί να επηρεάσουν την ποιότητα.
