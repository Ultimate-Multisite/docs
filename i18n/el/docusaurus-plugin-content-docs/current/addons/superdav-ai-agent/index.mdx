# Superdav AI Agent {#superdav-ai-agent}

Το Superdav AI Agent είναι ένας προηγμένος βοηθός AI για WordPress που επεκτείνει το Gratis AI Agent με δεξιότητες που λαμβάνουν υπόψη το θέμα, βελτιωμένες δυνατότητες αναζήτησης στο διαδίκτυο, λειτουργίες δημιουργίας ιστοτόπων και προαιρετικές ενσωματώσεις αυτοματοποίησης. Η έκδοση 1.18.0 προσθέτει εργαλεία ανάγνωσης Google Calendar, αντιστοίχιση επαφών και συμμετεχόντων, πύλες ανθρώπινης έγκρισης, αποδιπλοποίηση υπενθυμίσεων, ειδοποιήσεις SMS TextBee, παροχή διαχειριζόμενης υπηρεσίας Superdav και ένα ξεχωριστά διανεμόμενο συνοδευτικό πακέτο Advanced για ροές εργασίας αξιόπιστων προγραμματιστών.

## Κύρια χαρακτηριστικά {#key-features}

- **Δεξιότητες που λαμβάνουν υπόψη το θέμα** — προσαρμόζει αυτόματα την καθοδήγηση και τις δυνατότητες με βάση το ενεργό θέμα WordPress και τα εγκατεστημένα πρόσθετά σας
- **Πολλαπλοί πάροχοι αναζήτησης στο διαδίκτυο** — επιλέξτε μεταξύ Brave Search και Tavily για πλουσιότερα αποτελέσματα αναζήτησης και προηγμένες δυνατότητες έρευνας
- **Φόρμες επικοινωνίας δημιουργίας ιστοτόπων** — δημιουργήστε πλήρως λειτουργικές φόρμες επικοινωνίας απευθείας από τη διεπαφή συνομιλίας χρησιμοποιώντας το εγκατεστημένο πρόσθετο φορμών σας
- **Βελτιωμένο εργαλείο δημιουργίας ιστοτόπων** — βελτιωμένα πολυφασικά πλάνα δημιουργίας ιστοτόπων με καλύτερο εντοπισμό πρόσθετων και ανάκτηση από σφάλματα
- **Ενσωματωμένες δεξιότητες θεμάτων** — εξειδικευμένη καθοδήγηση για Block Themes, Classic Themes, Kadence Blocks και Kadence Theme
- **Βελτιωμένη ενσωμάτωση WooCommerce** — ανασχεδιάστηκε ώστε να χρησιμοποιεί εγγενή WooCommerce API για καλύτερη αξιοπιστία και συμβατότητα
- **Αυτόματος εντοπισμός παρόχων** — εντοπίζει και ανανεώνει αυτόματα διαθέσιμα πρόσθετα και θέματα όταν ενεργοποιούνται ή απενεργοποιούνται
- **Δημιουργία προσαρμοσμένου λογοτύπου** — δημιουργήστε και ενσωματώστε προσαρμοσμένα λογότυπα SVG με αυτόματη απολύμανση ασφαλή ως προς τα namespaces
- **Επικύρωση αντίθεσης παλέτας** — ελέγξτε ζεύγη χρωμάτων για συμμόρφωση με την προσβασιμότητα WCAG πριν τα εφαρμόσετε στο θέμα σας
- **Ανακάλυψη ενισχυμένη με φωτογραφίες** — ανεβάστε φωτογραφίες κατά τη συνέντευξη ανακάλυψης του Theme Builder για να επηρεάσετε την κατεύθυνση σχεδιασμού
- **Μενού φιλοξενίας** — δημιουργήστε δομημένες σελίδες μενού φαγητών και ποτών για εστιατόρια, καφέ και επιχειρήσεις φιλοξενίας
- **Προεπισκοπήσεις σε υπολογιστή και κινητό** — κάντε προεπισκόπηση του σχεδίου σας σε υπολογιστή και κινητές συσκευές κατά την επιλογή κατεύθυνσης σχεδιασμού
- **Tier 1 απόξεση ιστοτόπων** — το sd-ai-agent/site-scrape είναι πλέον διαθέσιμο από προεπιλογή για ανάλυση ανταγωνιστικών και εμπνευστικών ιστοτόπων
- **Εργαλεία ανάγνωσης Google Calendar** — διαβάστε διαμορφωμένα ημερολόγια και συμβάντα, ώστε οι αυτοματισμοί να μπορούν να συλλογίζονται σχετικά με το επερχόμενο πλαίσιο προγράμματος
- **Αντιστοίχιση επαφών και συμμετεχόντων** — αντιστοιχίστε συμμετέχοντες ημερολογίου με γνωστούς χρήστες και επαφές του ιστοτόπου πριν δημιουργήσετε επακόλουθη εργασία
- **Πύλες ανθρώπινης έγκρισης** — παύστε ευαίσθητους αυτοματισμούς για έλεγχο πριν από την αποστολή μηνυμάτων, τη συνέχιση εργασίας ή την πραγματοποίηση αλλαγών
- **Εγγραφές υπενθυμίσεων και αποδιπλοποίηση** — παρακολουθήστε υπενθυμίσεις που έχουν ήδη σταλεί, ώστε οι επαναλαμβανόμενες εκτελέσεις αυτοματοποίησης να μην ειδοποιούν τον ίδιο παραλήπτη δύο φορές
- **Ειδοποιήσεις SMS TextBee** — στείλτε διαμορφωμένες ειδοποιήσεις γραπτών μηνυμάτων μέσω TextBee όταν είναι ενεργοποιημένος ένας πάροχος SMS
- **Παροχή διαχειριζόμενης υπηρεσίας Superdav** — συνδέστε υποστηριζόμενους ιστοτόπους με φιλοξενούμενα Superdav endpoints με αυτόματη ρύθμιση αντί για χειροκίνητη διαμόρφωση endpoint
- **Προηγμένο συνοδευτικό πακέτο** — ξεχωριστά διανεμόμενα εργαλεία αξιόπιστων προγραμματιστών για ροές εργασίας συστήματος αρχείων, βάσης δεδομένων, WP-CLI, REST dispatcher, δημιουργίας πρόσθετων, στιγμιότυπου git, διαχείρισης χρηστών και συγκριτικών δοκιμών

## Απαιτήσεις {#requirements}

- WordPress 6.0 ή νεότερο
- PHP 8.1 ή νεότερο
- WordPress Multisite (ενεργοποιημένο σε δίκτυο) ή εγκατάσταση ενός ιστοτόπου
- Ένα ενεργό πρόσθετο σύνδεσης παρόχου AI (Anthropic, OpenAI ή συμβατό)
- Gratis AI Agent v1.9.0 ή νεότερο (το Superdav AI Agent επεκτείνει το Gratis AI Agent)

## Εγκατάσταση {#installation}

1. Ανεβάστε τα αρχεία του πρόσθετου στο `/wp-content/plugins/superdav-ai-agent`
2. Ενεργοποιήστε το πρόσθετο μέσω της οθόνης **Πρόσθετα** στο WordPress
3. Βεβαιωθείτε ότι το Gratis AI Agent είναι επίσης ενεργοποιημένο
4. Μεταβείτε στο **Gratis AI Agent → Ρυθμίσεις** και διαμορφώστε τον πάροχο αναζήτησης στο διαδίκτυο, τη σύνδεση διαχειριζόμενης υπηρεσίας Superdav, το Google Calendar ή τα διαπιστευτήρια TextBee όπως απαιτείται

Το πακέτο WordPress.org περιέχει τις βασικές λειτουργίες του Superdav AI Agent. Το συνοδευτικό πακέτο Advanced δημιουργείται και διανέμεται ξεχωριστά για περιβάλλοντα όπου τα εργαλεία αξιόπιστων προγραμματιστών εγκαθίστανται και εξουσιοδοτούνται ρητά.

## Δεξιότητες που λαμβάνουν υπόψη το θέμα {#theme-aware-skills}

Το Superdav AI Agent εντοπίζει αυτόματα το ενεργό θέμα και τα εγκατεστημένα πρόσθετά σας, φορτώνοντας την κατάλληλη δεξιότητα που λαμβάνει υπόψη το θέμα στο πλαίσιο του βοηθού. Οι διαθέσιμες δεξιότητες περιλαμβάνουν:

| Δεξιότητα | Εφαρμόζεται σε |
|---|---|
| **Block Themes** | Θέματα που χρησιμοποιούν την αρχιτεκτονική του WordPress που βασίζεται σε μπλοκ (Full Site Editing) |
| **Classic Themes** | Παραδοσιακά θέματα WordPress που χρησιμοποιούν πρότυπα PHP και τον κλασικό επεξεργαστή |
| **Kadence Blocks** | Ιστότοποι με ενεργό το πρόσθετο Kadence Blocks |
| **Kadence Theme** | Ιστότοποι με το θέμα Kadence ως ενεργό θέμα |

Δείτε τις [Δεξιότητες που λαμβάνουν υπόψη το θέμα](../../user-guide/configuration/theme-aware-skills) για λεπτομερή καθοδήγηση σχετικά με κάθε δεξιότητα.

## Πάροχοι αναζήτησης στο διαδίκτυο {#internet-search-providers}

Το Superdav AI Agent v1.10.0 προσθέτει το **Tavily** ως εναλλακτική του Brave Search για ερωτήματα αναζήτησης στο διαδίκτυο. Και οι δύο πάροχοι προσφέρουν αποτελέσματα αναζήτησης σε πραγματικό χρόνο και μπορούν να διαμορφωθούν στο **Gratis AI Agent → Ρυθμίσεις → Προηγμένα**.

| Πάροχος | Χαρακτηριστικά | Ρύθμιση |
|---|---|---|
| **Brave Search** | Γρήγορα αποτελέσματα αναζήτησης με έμφαση στην ιδιωτικότητα | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Πλουσιότερα αποτελέσματα αναζήτησης, προηγμένες δυνατότητες έρευνας | [Tavily API](https://tavily.com/) |

Δείτε την [Αναζήτηση στο διαδίκτυο](../../user-guide/configuration/internet-search) για οδηγίες διαμόρφωσης.

## Ροές εργασίας ημερολογίου, έγκρισης και ειδοποιήσεων {#calendar-approval-and-notification-workflows}

Το Superdav AI Agent v1.18.0 επεκτείνει τις ροές εργασίας αυτοματισμού πέρα από τις εργασίες περιεχομένου και δημιουργίας ιστότοπων:

- **Τα εργαλεία Google Calendar** επιτρέπουν στον agent να διαβάζει διαμορφωμένα ημερολόγια και λεπτομέρειες εκδηλώσεων όταν είναι διαθέσιμα τα διαπιστευτήρια ημερολογίου.
- **Οι βοηθοί επαφών και συμμετεχόντων** αντιστοιχίζουν τους συμμετέχοντες εκδηλώσεων με χρήστες WordPress ή γνωστές επαφές πριν από την προετοιμασία υπενθυμίσεων ή εργασιών παρακολούθησης.
- **Οι πύλες ανθρώπινης έγκρισης** επιτρέπουν σε έναν αυτοματισμό να κάνει παύση έως ότου ένας εξουσιοδοτημένος χρήστης ελέγξει την προτεινόμενη ενέργεια.
- **Οι εγγραφές υπενθυμίσεων** αποτρέπουν διπλές υπενθυμίσεις ημερολογίου ή ειδοποιήσεων όταν ένας αυτοματισμός επαναλαμβάνεται ή προγραμματίζεται επανειλημμένα.
- **Η υποστήριξη TextBee SMS** ενεργοποιεί ειδοποιήσεις με γραπτά μηνύματα όταν ο πάροχος TextBee έχει διαμορφωθεί.

Χρησιμοποιήστε αυτά τα εργαλεία για βοήθεια με επίγνωση προγράμματος, παρακολούθηση συμμετεχόντων και ροές εργασίας υπενθυμίσεων που εξακολουθούν να απαιτούν ανθρώπινο έλεγχο πριν σταλούν ευαίσθητα μηνύματα.

## Διαμόρφωση {#configuration}

Όλες οι ρυθμίσεις βρίσκονται κάτω από **Gratis AI Agent → Ρυθμίσεις** στη διαχείριση WordPress. Το Superdav AI Agent προσθέτει τις ακόλουθες επιλογές διαμόρφωσης:

| Ρύθμιση | Περιγραφή |
|---|---|
| **Πάροχος αναζήτησης στο διαδίκτυο** | Επιλέξτε μεταξύ Brave Search και Tavily (προαιρετικό) |
| **Tavily API Key** | API key για αναζήτηση στο διαδίκτυο με Tavily (αν έχει επιλεγεί το Tavily) |
| **Διαχειριζόμενη υπηρεσία Superdav** | Συνδέστε υποστηριζόμενους ιστότοπους με τη φιλοξενούμενη υπηρεσία Superdav με αυτόματη παροχή endpoint |
| **Διαπιστευτήρια Google Calendar** | Διαμορφώστε την πρόσβαση ημερολογίου ώστε ο agent να μπορεί να διαβάζει ημερολόγια και εκδηλώσεις |
| **Πάροχος TextBee SMS** | Διαμορφώστε τα διαπιστευτήρια TextBee για εγκεκριμένες ροές εργασίας ειδοποιήσεων SMS |

## Προηγμένο συνοδευτικό πακέτο {#advanced-companion-package}

Το Superdav AI Agent v1.18.0 προσθέτει ένα ξεχωριστά διανεμόμενο προηγμένο συνοδευτικό πακέτο για αξιόπιστες ροές εργασίας προγραμματιστών. Αυτές οι δυνατότητες δεν περιλαμβάνονται στο βασικό πακέτο WordPress.org. Εγκαταστήστε το συνοδευτικό πακέτο μόνο σε ιστότοπους όπου οι διαχειριστές εμπιστεύονται ρητά τον agent για πρόσβαση σε εργαλεία προγραμματιστών.

Τα προηγμένα εργαλεία περιλαμβάνουν λειτουργίες συστήματος αρχείων, πρόσβαση σε βάση δεδομένων, WP-CLI, ροές εργασίας REST dispatcher, βοηθητικά εργαλεία δημιουργού plugin, στιγμιότυπα git, βοηθούς διαχείρισης χρηστών και εργαλεία benchmark. Διατηρήστε αυτά τα εργαλεία περιορισμένα σε αξιόπιστους διαχειριστές και ελέγξτε τη διαμόρφωση πύλης έγκρισης πριν τα ενεργοποιήσετε σε παραγωγικούς ιστότοπους.
