# Superdav AI Agent

Το Superdav AI Agent είναι ένας προηγμένος βοηθός AI για το WordPress που επεκτείνει το Gratis AI Agent με ικανότητες συντονισμένες με το θέμα (theme-aware skills), βελτιωμένες δυνατότητες αναζήτησης στο διαδίκτυο και βελτιωμένες λειτουργίες site builder. Η έκδοση 1.16.0 προσθέτει τη δημιουργία προσαρμοσμένου λογότυπου, τη επικύρωση αντίθεσης παλέτας, την ανέβασμα φωτογραφιών στην συνέντευξη ανακάλυψης, τη δημιουργία δομημένων μενού φιλοξενίας και την προεπισκόπηση desktop/mobile στο Theme Builder.

## Βασικά Χαρακτηριστικά

- **Ικανότητες συντονισμένες με το θέμα** — προσαρμόζεται αυτόματα η καθοδήγηση και οι δυνατότητες με βάση το ενεργό θέμα WordPress και τα εγκατεστημένα plugins.
- **Πολλαπλοί πάροχοι αναζήτησης στο διαδίκτυο** — επιλέξτε μεταξύ Brave Search και Tavily για πιο πλούσιμα αποτελέσματα αναζήτησης και προηγμένες δυνατότητες έρευνας.
- **Επαφικές φόρμες στο site builder** — δημιουργήστε πλήρως λειτουργικές φόρμες επικοινωνίας απευθείας από τη διεπαφή chat, χρησιμοποιώντας το εγκατεστημένο plugin φόρμας σας.
- **Βελτιωμένος site builder** — βελτιωμένα σχέδια site build σε πολλαπλές φάσεις με καλύτερη ανακάλυψη plugins και ανάκαμψη σφαλμάτων.
- **Ενσωματωμένες ικανότητες θέματος** — εξειδικευμένη καθοδήγηση για Block Themes, Classic Themes, Kadence Blocks και Kadence Theme.
- **Βελτιωμένη ενσωμάτωση WooCommerce** — αναδιαμορφώθηκε για να χρησιμοποιεί τις native WooCommerce APIs για καλύτερη αξιοπιστία και συμβατότητα.
- **Αυτόματη ανίχνευση παρόχου** — ανιχνεύει και ανανεώνει αυτόματα τα διαθέσιμα plugins και θέματα όταν ενεργοποιούνται ή απενεργοποιούνται.
- **Δημιουργία προσαρμοσμένου λογότυπου** — δημιουργήστε και ενσωματώστε προσαρμοσμένα SVG λογότυπα με αυτόματη αποστείρωση (sanitisation) του namespace.
- **Επικύρωση αντίθεσης παλέτας** — ελέγχετε τις ζεύξεις χρωμάτων για συμμόρφωση με την προσβασιμότητα WCAG πριν τα εφαρμόσετε στο θέμα σας.
- **Ανακάλυψη με φωτογραφίες** — ανεβάζετε φωτογραφίες κατά τη διάρκεια της συνέντευξης ανακάλυψης του Theme Builder για να επηρεάσετε την κατεύθυνση του σχεδιασμού.
- **Μενού φιλοξενίας** — δημιουργείτε δομημένα σελίδες μενού φαγητού και ποτών για εστιατόρια, καφέ και επιχειρήσεις φιλοξενίας.
- **Προεπισκοπή desktop και mobile** — προεπισκοπείτε τον σχεδιασμό σας σε συσκευές desktop και mobile κατά την επιλογή κατεύθυνσης σχεδιασμού.
- **Scraping σε επίπεδο 1 (Tier 1 site scraping)** — το sd-ai-agent/site-scrape είναι πλέον διαθέσιμο εξ ορισμού για την ανάλυση ιστοσελίδων ανταγωνιστών και πηγών έμπνευσης.

## Απαιτήσεις

- WordPress 6.0 ή νεότερη
- PHP 8.1 ή νεότερη
- WordPress Multisite (ενεργοποιημένο στο network) ή εγκατάσταση single-site
- Ένα ενεργό plugin συνδέσμου AI provider (Anthropic, OpenAI ή συμβατό)
- Gratis AI Agent v1.9.0 ή νεότερη (Το Superdav AI Agent επεκτείνει το Gratis AI Agent)

## Εγκατάσταση

1. Ανέβασμα των αρχείων του plugin στο `/wp-content/plugins/superdav-ai-agent`
2. Ενεργοποίηση του plugin από την οθόνη **Plugins** στο WordPress
3. Βεβαιωθείτε ότι το Gratis AI Agent είναι επίσης ενεργοποιημένο
4. Πλοηγηθείτε στο **Gratis AI Agent → Settings** και ρυθμίστε τον πάροχο αναζήτησης στο διαδίκτυο (προαιρετικό)

## Γρήγορη Εκκίνηση

Μόλις ενεργοποιηθεί, το Superdav AI Agent βελτιώνει αυτόματα την εγκατάστασή σας του Gratis AI Agent με ικανότητες συντονισμένες με το θέμα και επιπλέον δυνατότητες. Ανοίξτε το πάνελ **Gratis AI Agent** στην πλαϊνή γραμμή του διαχειριστή WordPress και ξεκινήστε τη χρήση των νέων λειτουργιών.

**Παραδείγματα οδηγιών:**

```
Add a contact form to the Contact page.
```

```
Search the web for the latest WordPress security best practices.
```

```
Create a hero section using the active theme's block patterns.
```

## Ικανότητες Συντονισμένες με το Θέμα (Theme-Aware Skills)

Το Superdav AI Agent ανιχνεύει αυτόματα το ενεργό θέμα και τα εγκατεστημένα plugins, φορτώνοντας την κατάλληλη ικανότητα συντονισμένη με το θέμα στο πλαίσιο του βοηθού. Οι διαθέσιμες ικανότητες περιλαμβάνουν:

| Ικανότητα | Εφαρμόζεται σε |
|---|---|
| **Block Themes** | Θέματα που χρησιμοποιούν την αρχιτεκτονική βασισμένη σε blocks του WordPress (Full Site Editing) |
| **Classic Themes** | Πα παραδοσιακά θέματα WordPress που χρησιμοποιούν PHP templates και τον κλασικό editor |
| **Kadence Blocks** | Ιστοσελίδες με το plugin Kadence Blocks ενεργοποιημένο |
| **Kadence Theme** | Ιστοσελίδες με το Kadence theme ως ενεργό θέμα |

Για λεπτομερή καθοδήγηση για κάθε ικανότητα, δείτε [Theme-Aware Skills](../../user-guide/configuration/theme-aware-skills).

## Πάροχοι Αναζήτησης στο Διαδίκτυο (Internet Search Providers)

Το Superdav AI Agent v1.10.0 προσθέτει το **Tavily** ως εναλλακτική του Brave Search για ερωτήσεις αναζήτησης στο διαδίκτυο. Και οι δύο πάροχοι προσφέρουν αποτελέσματα αναζήτησης σε πραγματικό χρόνο και μπορούν να ρυθμιστούν στο **Gratis AI Agent → Settings → Advanced**.

| Πάροχος | Χαρακτηριστικά | Ρύθμιση |
|---|---|---|
| **Brave Search** | Γρήγορα αποτελέσματα αναζήτησης με έμφαση στην ιδιωτικότητα | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Πιο πλούσια αποτελέσματα αναζήτησης, προηγμένες δυνατότητες έρευνας | [Tavily API](https://tavily.com/) |

Για οδηγίες ρύθμισης, δείτε [Internet Search](../../user-guide/configuration/internet-search).

## Επαφικές Φόρμες στο Site Builder

Ο agent Site Builder μπορεί πλέον να δημιουργεί φόρμες επικοινωνίας απευθείας από τη διεπαφή chat. Απλώς ζητήστε από τον Site Builder να δημιουργήσει μια φόρμα επικοινωνίας και αυτό θα ανιχνεύσει αυτόματα το εγκατεστημένο plugin φόρμας σας (Contact Form 7, WPForms, Fluent Forms ή Gravity Forms) και θα δημιουργήσει μια έτοιμη προς χρήση φόρμα.

Για λεπτομερή καθοδήγηση, δείτε [Site Builder Contact Forms](../../user-guide/administration/site-builder-contact-forms).

## Χαρακτηριστικά Theme Builder

Το Theme Builder στο Superdav AI Agent v1.16.0 περιλαμβάνει ισχυρές νέες δυνατότητες:

### Συνέντευξη Ανακάλυψης με Ανέβασμα Φωτογραφιών

Η συνέντευξη ανακάλυψης του Theme Builder περιλαμβάνει πλέον ένα βήμα ανέβασμα φωτογραφιών όπου μπορείτε να παρέχετε οπτική έμπνευση. Ανέβασμα φωτογραφιών της μάρκας σας, ιστοσελίδων ανταγωνιστών ή σχεδιαστικής έμπνευσης για να βοηθήσετε το Theme Builder να καταλάβει την κατεύθυνση του σχεδιασμού σας.

Για λεπτομερή καθοδήγηση, δείτε [Discovery Interview](./theme-builder/discovery-interview.md).

### Κατεύθυνση Σχεδιασμού με Προεπισκοπή Desktop και Mobile

Ελέγξτε τον σχεδιασμό σας τόσο σε συσκευές desktop όσο και mobile πριν το Theme Builder δημιουργήσει το πλήρες θέμα σας. Δείτε προεπισκοπή σε πραγματικό χρόνο σε πολλαπλά responsive breakpoints.

Για λεπτομερή καθοδήγηση, δείτε [Design Direction](./theme-builder/design-direction.md).

### Μενού Φιλοξενίας

Το Theme Builder μπορεί πλέον να δημιουργεί δομημένα σελίδες μενού φαγητού και ποτών για εστιατόρια, καφέ, μπαρ και άλλες επιχειρήσεις φιλοξενίας. Τα μενού περιλαμβάνουν τιμές, περιγραφές, πληροφορίες διατροφικής αξίας και προειδοποιήσεις για αλλεργιογόνα.

Για λεπτομερή καθοδήγηση, δείτε [Hospitality Menus](./theme-builder/hospitality-menus.md).

### Νέες Ικανότητες

- **[Generate Logo SVG](./abilities/generate-logo-svg.md)** — δημιουργήστε προσαρμοσμένα SVG λογότυπα με αυτόματη αποστείρωση
- **[Validate Palette Contrast](./abilities/validate-palette-contrast.md)** — ελέγχετε τις ζεύξεις χρωμάτων για συμμόρφωση WCAG
- **[Create Menu](./abilities/create-menu.md)** — δημιουργείτε πλοήγητικά μενού με ξεχωριστές ετικέτες πλοήγησης

Για πληροφορίες σχετικά με τις διαθέσιμες εργαλεία και το εργαλείο site-scrape σε επίπεδο 1, δείτε [Tools and Configuration](./theme-builder/tools.md).

## Ρύθμιση (Configuration)

Όλες οι ρυθμίσεις βρίσκονται στο **Gratis AI Agent → Settings** στο admin του WordPress. Το Superdav AI Agent προσθέτει τις ακόλουθες επιλογές ρύθμισης:

| Ρύθμιση | Περιγραφή |
|---|---|
| **Internet Search Provider** | Επιλέξτε μεταξύ Brave Search και Tavily (προαιρετικό) |
| **Tavily API Key** | API key για αναζήτηση στο διαδίκτυο της Tavily (εάν επιλεγεί η Tavily) |

## Βελτιώσεις στην v1.16.0

- **AI Client cache** — τώρα υποστηρίζεται από transients για διατήρηση μεταξύ αιτημάτων, αποτρέποντας απώλεια δεδομένων σε εργασίες agent που εκτελούνται για μεγάλο διάστημα.
- **Εργαλείο site-scrape σε επίπεδο 1 (Tier 1 site-scrape)** — το sd-ai-agent/site-scrape είναι πλέον διαθέσιμο εξ ορισμού στο Theme Builder για την ανάλυση ιστοσελίδων ανταγωνιστών και πηγών έμπνευσης.
- **Σύνδεσμοι δράσης γραμμής plugin** — διορθώθηκαν και αναδόθηκαν ονόματα για μεγαλύτερη σαφήνεια.

## Βελτιώσεις στην v1.10.0

- **Αναδιαμορφωμένη ενσωμάτωση WooCommerce** — χρησιμοποιεί πλέον τις native WooCommerce APIs για καλύτερη αξιοπιστία και συμβατότητα.
- **Αυτόματη ανανέωση λίστας παρόχων** — ανιχνεύει αυτόματα όταν ενεργοποιούνται ή απενεργοποιούνται τα plugins και ενημερώνει τη διαθέσιμη λίστα παρόχων.
- **Βελτιωμένος site builder** — βελτιωμένα σχέδια site build σε πολλαπλές φάσεις με καλύτερη ανάκαμψη σφαλμάτων.
- **Ικανότητες συντονισμένες με το θέμα** — τέσσερις νέες ενσωματωμένες ικανότητες που προσαρμόζονται αυτόματα στο ενεργό θέμα σας.

## Διορθώσεις Σφαλμάτων στην v1.10.0

- Διορθώθηκε η ικανότητα `navigate-to` που προκαλούσε άπειρες επαναφόρτωση σε ορισμένες σελίδες διαχειριστή.
- Διορθώθηκε η ικανότητα `list-posts` που δεν αν rozwiązyμαζε σωστά τα ονόματα κατηγορίας και tag σε slugs.
- Διορθώθηκαν οι εντολές WP-CLI που έλειπαν namespace aliases μετά από μια προηγούμενη αναδιαμόρφωση.
- Διορθώθηκε η αυτοματοποίηση γεγονότων (event automation) που χειρίζεται ομαλά τις ιστοσελίδες όπου τα جداول αυτοματοποίησης δεν έχουν ακόμη δημιουργηθεί.
- Διορθώθηκε η ικανότητα `memory-save` που χρησιμοποιεί το σωστό prefix namespace στον builder των system instructions.
- Διορθώθηκε το αποτέλεσμα του scalar tool ώστε να συσκευάζεται σωστά πριν επιστραφεί στο AI.
- Διορθώθηκαν τα στατιστικά χρήσης που χειρίζονται σωστά το παλαιό μορφότυπο κλειδιού ικανότητας κατά την αναβάθμιση από παλαιότερες εκδόσεις.
