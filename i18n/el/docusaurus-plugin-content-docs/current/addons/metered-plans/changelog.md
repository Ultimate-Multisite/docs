# Κατάλογος Αλλαγών Metered Plans

Έκδοση 1.1.0 - Έκδοση στις 2026-05-05
- Νέο: Χρέωση AI tokens για υπο-σάιτ σε multisite — Παρακολουθείτε και χρεώνετε τη χρήση AI tokens σε πολλαπλούς καταστήματα πελατών με ρυθμιζόμενες τιμές ανά token.
- Νέο: Η εφαρμογή του Connector αναβδόμησε με δυναμική ανακάλυψη ορίων (limit discovery) και write-through, εξασφαλίζοντας ακρίβεια σε πραγματικό χρόνο σε όλους τους connectors.
- Διόρθωση: Οι αναβαθμίσεις των πινάκων βάσης δεδομένων ευθυγραμμίζονται πλέον σωστά με τις οριστικές δομές (schema) της BerlinDB, αποτρέποντας αποτυχίες αναβάθμισης σε νέες εγκαταστάσεις.
- Διόρθωση: Οι callbacks αναβάθμισης βάσης δεδομένων μετατρήθηκαν στο σωστό format, επιλύοντας τις αποτυχίες αναβάθμισης που δεν εμφανίζονται.
- Διόρθωση: Τώρα αποδέχεστε κλασματικές τιμές στο πεδίο εισαγωγής για την επιπλέον χρέωση χρήσης AI (AI Usage Overage Markup).
- Διόρθωση: Λύθηκαν 치νικές (fatal) και προβλήματα διπλής αρχικοποίησης κατά το φόρτ του plugin.
- Βελτιωμένο: Προστέθηκε το npm script check-env, ώστε τα developer environments να αυτο-διαμορφώνουν κατά την πρώτη εκτέλεση.
