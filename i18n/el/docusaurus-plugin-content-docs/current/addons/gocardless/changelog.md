# Κατάλογος Αλλαγών GoCardless {#gocardless-changelog}

Έκδοση: 1.0.3 - Έκβηκε στις 2026-05-05
* Διόρθωση: Τα αποκλεισμένα sites σε αναμονή καθαρίζονται τώρα όταν μια सदस्यता GoCardless ακυρώνεται ενώ βρίσκεται σε κατάσταση αναμονής ή παύσης.
* Βελτίωση: Ο αυτόματος φορτωτής plugins παραλείπεται όταν ο αυτόματος φορτωτής του root του Bedrock έχει ήδη φορτώσει τις εξαρτήσεις.

Έκδοση: 1.0.1 - Έκβηκε στις 2025-09-28

* Μετονομασία προεπίγραφης σε ultimate-multisite· ενημέρωση πεδίου κειμένου· αύξηση αριθμού έκδοσης.


2021-09-24 - Έκδοση 1.0.0-beta.4

* Προστέθηκε: ο φίλτρο `wp_ultimo_skip_network_active_check` για ρυθμίσεις με βάση τα mu-plugins·

2021-06-16 - Έκδοση 1.0.0-beta.3

* Διόρθωση: Οι διαχειριστές webhook που αποτύχουν να επιβεβαιώσουν τις υπογραφές·

2021-06-14 - Έκδοση 1.0.0-beta.2

* Βελτίωση: Η δημοσίευση του site μετά τη δημιουργία μιας διαταγής GoCardless·

Έκδοση 1.0.0-beta.1 - Αρχική Έκδοση
