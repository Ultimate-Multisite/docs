# Multi-Currency

Το Ultimate Multisite: Multi-Currency επιτρέπει στο network σας να δέχεται πληρωμές σε πολλαπλές νομίσματα. Μετατρέπει αυτόματα τις τιμές χρησιμοποιώντας ρυθμούς συναλλαγών (live ή χειροκίνητους), σας επιτρέπει να ορίσετε ακριβείς τιμές ανά προϊόν για οποιοδήποτε νόμισμα και ανιχνεύει το προτιμώμενο νόμισμα κάθε επισκέπτη μέσω γεωγραφικής τοποθεσίας — όλα διαχειριζόμενα από ένα ενιαίο admin dashboard.

## Βασικά Χαρακτηριστικά

- **Αυτόματη μετατροπή ρυθμών συναλλαγών** — οι τιμές μετατρέπονται από τη βασική σας νομισματική μονάδα χρησιμοποιώντας ρυθμούς που λαμβάνονται από τον επιλεγμένο πάροχο.
- **Υπερρύθμιση τιμών ανά προϊόν** — ορίστε ακριβείς τιμές για συγκεκριμένα νομίσματα σε οποιοδήποτε προϊόν για να παρακάμψετε την αυτόματη μετατροπή.
- **Πεδίο επιλογής νομίσματος κατά την εγγραφή** — επιτρέπει στους πελάτες να επιλέγουν το προτιμώμενο νόμισμά τους κατά την ολοκλήρωση της αγοράς/εγγραφή.
- **Πολλαπλοί πάροχοι ρυθμών συναλλαγών** — ECB, Frankfurter, Stripe ή πλήρως χειροκίνητοι ρυθμοί.
- **Ανίχνευση νομίσματος βάσει γεωγραφικής τοποθεσίας** — επιλέγει αυτόματα το νόμισμα που ταιριάζει με τη χώρα του επισκέπτη.
- **Admin dashboard** — διαχειριστείτε τα ενεργά νομίσματα, τους ρυθμούς συναλλαγών και τις ρυθμίσεις των παρόχων από ένα μέρος.

## Απαιτήσεις

- Ultimate Multisite 2.0.0 ή νεότερη
- Εγκατάσταση WordPress Multisite
- PHP 7.4 ή νεότερο

## Εγκατάσταση

1. Ανεβάστε τα αρχεία του plugin στο `/wp-content/plugins/ultimate-multisite-multi-currency`
2. Ενεργοποιήστε το plugin από την οθόνη **Plugins** στο WordPress (Network Activate)
3. Πλοηγηθείτε στο **Ultimate Multisite → Multi-Currency** για να ανοίξετε το admin dashboard
4. Προσθέστε τα νομίσματα που θέλετε να δέχεστε και ρυθμίστε τον πάροχο ρυθμών συναλλαγών σας.

## Ρύθμιση (Configuration)

### Admin Dashboard

Το admin dashboard Multi-Currency (**Ultimate Multisite → Multi-Currency**) είναι το κεντρικό σημείο για να:

- Προσθέτετε, ενεργοποιείτε ή απενεργοποιείτε νομίσματα.
- Επιλέγετε έναν πάροχο ρυθμών συναλλαγών και ανανεώνετε τους ρυθμούς βάσει της ανάγκης.
- Ελέγχετε τους τρέχοντες ρυθμούς για όλα τα ενεργά νομίσματα.
- Ορίζετε χειροκίνητους ρυθμούς όταν οι αυτόματοι πάροχοι δεν είναι κατάλληλοι.

### Πάροχοι Ρυθμών Συναλλαγών

| Provider | Description |
|---|---|
| **ECB** | European Central Bank — δωρεάν καθημερινούς ρυθμούς για networks βασισμένα στο EUR |
| **Frankfurter** | Open-source ECB mirror API — δωρεάν, δεν απαιτείται API key |
| **Stripe** | Χρησιμοποιεί τους live ρυθμούς FX της Stripe — απαιτεί εν ενεργεία ολοκλήρωση Stripe |
| **Manual** | Εισάγετε τους ρυθμούς απευθείας· δεν απαιτούνται εξωτερικές κλήσεις API |

Επιλέξτε τον πάροχο σας κάτω από **Ultimate Multisite → Multi-Currency → Settings → Exchange Rate Provider**. Οι ρυθμοί μπορούν να ανανεωθούν χειροκίνητα οποιαδήποτε στιγμή ή να προγραμματιστούν για αυτόματη ενημέρωση.

### Υπερρύθμιση Τιμών ανά Προϊόν

Όταν η αυτόματη μετατροπή δεν είναι αρκετά ακριβής, μπορείτε να ορίσετε μια ακριβή τιμή για ένα συγκεκριμένο νόμισμα σε οποιοδήποτε προϊόν:

1. Ανοίξτε τον επεξεργαστή προϊόντος (**Ultimate Multisite → Products → Edit**)
2. Κάντε κύλιση στην ενότητα **Currency Pricing**
3. Εισάγετε την ακριβή τιμή για κάθε νόμισμα που θέλετε να υπερρυθμίσετε.
4. Αποθηκεύστε το προϊόν.

Τα προϊόντα χωρίς υπερρύθμιση χρησιμοποιούν την μετατρεπόμενη τιμή από τη βασική σας νομισματική μονάδα.

### Πεδίο Επιλογής Νομίσματος κατά την Εγγραφή

Το πεδίο επιλογής νομίσματος μπορεί να προστεθεί στη ροή εγγραφής/checkout σας, ώστε οι πελάτες να επιλέγουν το προτιμώμενο νόμισμά τους κατά την εγγραφή:

1. Πηγαίνετε στο **Ultimate Multisite → Multi-Currency → Settings**
2. Ενεργοποιήστε **Show currency selector on signup**
3. Προαιρετικά, ορίστε ένα προεπιλεγμένο νόμισμα ή αφήστε τη γεωγραφική τοποθεσία να καθορίσει την προεπιλεγμένη τιμή.

Το επιλεγμένο νόμισμα αποθηκεύεται μαζί με τη συνδρομή και χρησιμοποιείται για όλες τις μελλοντικές χρεώσεις.

### Ανίχνευση Νομίσματος βάσει Γεωγραφικής Τοποθεσίας

Όταν είναι ενεργή η ανίχνευση γεωγραφικής τοποθεσίας, το addon αντιστοιχίζει τη χώρα του επισκέπτη σε ένα νόμισμα και το επιλέγει αυτό προεπιλεγμένο στο πεδίο επιλογής νομίσματος. Για να ενεργοποιήσετε:

1. Πηγαίνετε στο **Ultimate Multisite → Multi-Currency → Settings**
2. Ενεργοποιήστε **Detect currency from visitor location**
3. Αντιστοιχίστε χώρες σε νομίσματα στον πίνακα **Geolocation Mapping**.

Εάν δεν υπάρχει αντιστοίχιση για τη χώρα του επισκέπτη, χρησιμοποιείται η βασική νομισματική μονάδα του network ως εφεδρικό.

## Συχνές Ερωτήσεις

### Τι συμβαίνει αν ένας πάροχος ρυθμών συναλλαγών είναι недоступός;

Το addon επιστρέφει στους τελευταίους επιτυχώς ληφθέντες ρυθμούς. Εάν ποτέ δεν έχουν ληφθεί ρυθμοί, επιστρέφει στους χειροκίνητους ρυθμούς που έχετε ρυθμίσει. Συνιστάται να ρυθμίσετε χειροκίνητους ρυθμούς ως ασφάλεια.

### Μπορώ να χρησιμοποιήσω διαφορετικούς παρόχους για διαφορετικά νομίσματα;

Όχι — ένας πάροχος είναι ενεργός ταυτόχρονα για όλα τα νομίσματα. Χρησιμοποιήστε το **Manual** εάν χρειάζεστε πλήρη έλεγχο των μεμονωμένων ρυθμών.

### Απαιτείται το geolocation ένα ξεχωριστό plugin;

Δεν απαιτείται εξωτερικό plugin geolocation. Το addon χρησιμοποιεί την ενσωματωμένη ανίχνευση IP-σε-χώρα του WordPress. Για υψηλότερη ακρίβεια, μπορεί να ενσωματωθεί μια εξυπηρεσία geolocation αφιερωμένη μέσω των παρεχόμενων hook.

### Οι υπερρυθμίσεις ανά προϊόν εφαρμόζονται πριν ή μετά τη φορολογία;

Οι υπερρυθμίσεις ανά προϊόν ορίζουν την τιμή πριν τη φορολογία στο στόχο νόμισμα. Η φορολογία υπολογίζεται πάνω από την τιμή της υπερρύθμισης χρησιμοποιώντας την υπάρχουσα ρύθμιση φορολογίας σας.
