# Fluent Forms

Αυτό το πρόσθετο επιτρέπει την προσθήκη περιορισμών στο Fluent Forms σε περιβάλλον Multisite Ultimate, είτε πρόκειται για προϊόντα, συνδρομές είτε για ολόκληρες ιστοσελίδες.
Το Fluent Forms χρησιμοποιεί προσαρμοσμένα جداول αντί για προσαρμοσμένα τύπους καταχώρισης, οπότε οι κανονικές επιλογές περιορισμού δεν λειτουργούν μαζί του.

## Χαρακτηριστικά {#features}

- Προσθήκη περιορισμών για το Fluent Forms ή το Conversational Fluent Forms.
- Οι πελάτες λαμβάνουν ένα φιλικό μήνυμα που τους υπενθυ ánει να αναβαθμίσουν το πακέτο τους όταν φτάσουν στον όριο.
