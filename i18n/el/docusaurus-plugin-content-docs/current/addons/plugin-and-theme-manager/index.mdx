# Plugin & Theme Manager

Multisite Ultimate: Plugin and Theme Manager

Επιτρέπετε την επεξεργασία ή την κρυφήκωση των μεταδεδομένων (τίτλος, συγγραφέας, miniatura και περιγραφή) των εγκατεστημένων plugins και themes. Δημιουργείτε κατηγορίες για τα plugins και τα themes, επιτρέποντας στους χρήστες σας να τα φιλτράρουν και προσθέτοντας μια όμορφη, προσαρμοσμένη σελίδα Plugins στο πάνελ των χρηστών σας.

Για να χρησιμοποιήσετε το plugin, απλώς μεταβείτε στις σελίδες Plugins ή Themes στο network admin σας και κάντε κλικ στο "Edit Plugin/Theme info".

## Εγκατάσταση {#installation}

1. Ανεβάστε το 'ultimate-multisite-plugin-and-theme-manager' στον κατάλογο '/wp-content/plugins/'
2. Ενεργοποιήστε το plugin μέσω του μενού 'Plugins' στο WordPress Network Admin σας

## Οδηγός Χρήσης {#usage-guide}

Το Ultimate Multisite: Plugin and Theme Manager είναι ένα ισχυρό **premium** add-on για το Ultimate Multisite που σας δίνει πλήρη έλεγχο στον τρόπο που οι χρήστες του network σας βλέπουν και αλληλεπιδρούν με τις επιλογές των plugins και themes στο network σας.

Αυτό το add-on εξυπηρετεί τρεις σκοπούς:

  1. Επιτρέπει να αλλάξετε πλήρως τα μεταδεδομένα τόσο των plugins όσο και των themes. Αυτό περιλαμβάνει το όνομα, την περιγραφή, το όνομα συγγραφέα και ακόμη και τις miniaturas!

  2. Επιτρέπει να κατηγοριοποιήσετε τα plugins και τα themes, επιτρέποντας στους χρήστες σας να τα ταξινομήσουν εύκολα.

  3. Σας δίνει την επιλογή να αντικαταστήσετε τη vecchia, βαρετή σελίδα plugins με μια νέα σελίδα που μοιάζει με themes, με ταξινομήσεις βάσει κατηγοριών.

Μετά το download και την εγκατάσταση, μια νέα καρτέλα – _Add-on Settings_ – θα εμφανιστεί στη σελίδα Ρυθμίσεων του Ultimate Multisite.
