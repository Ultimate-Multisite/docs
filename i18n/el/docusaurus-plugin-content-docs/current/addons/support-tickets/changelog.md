# Κατάλογος Αλλαγών Τικετών Υποστήριξης

### 1.0.4 - 2026-05-05 {#104---2026-05-05}
* Βελτιωμένο: Αφαιρέθηκε το directory `vendor/` από την παρακολούθηση του Git (το οποίο ήδη καλύπτεται από το `.gitignore`), μειώνοντας το μέγεθος του αποθετηρίου.
* Βελτιωμένο: Δοκιμάστηκε μέχρι το WordPress 7.0.

### 1.0.3 - 2026-05-01 {#103---2026-05-01}
* Διορθώθηκε: Επιτρέπεται η αποδοχή τιμών null στους setters του μοντέλου των τικετών που επιτρέπουν null.
* Διορθώθηκε: Όλα τα ³τικετάτα αποθηκεύονται σε παγκόᱢα, σε πίνακες ολόκληρου του δικτύου, για σωστή υποστήριξη multisite.
* Διορθώθηκε: Κρύβονται τα πεδία του προσωπικού και διορθώνεται η ανακατεύθυνση σε κενό σελίδα στο φόρμα νέου ³τικετού του πελάτη.
* Διορθώθηκε: Αντικαταστάθηκαν οι κλήσεις `add_meta()` που δεν ορίζονται με τη σωστή `update_meta()` για την αποθήκευση μεταδεδομένων.
* Διορθώθηκε: Αντικαταστάθηκε η έλεγχος δυνατότητας (capability) που δεν καταχωρήθηκε με τη σωστή `wu_view_all_support_tickets`.
* Διορθώθηκε: Προστέθηκαν λείπουν AJAX handlers για τις ενέργειες κατάστασης ³τικετού, εκκα시아σμού και γρήγορης επεξεργασίας.
* Διορθώθηκε: Η ονομασία της μεθόδου για την ανίχνευση της απάντησης του προσωπικού στις προβολές των ³τικετών.
* Διορθώθηκε: Συσυ集ώθηκαν τα διπλότυπα handlers απάντησης και ευθυγραμμίστηκαν τα ονόματα δράσης nonce.
* Διορθώθηκε: Προστέθηκε λείπουν frontend view για το shortcode `[wu_submit_ticket]`.
* Διορθώθηκε: Προστέθηκε η λείπουν στήλη `user_id` και διορθώθηκε η μέθοδος `Support_Ticket::get_user_id()`.
* Διορθώθηκε: Αφαιρέθηκε το επιπλέον διπλό εισαγωγικό στο tag επιλογής φίλτρου προτεραιότητας.
* Διορθώθηκε: Προστέθηκε το πάνελ διαχείρισης ³τικετών για το προσωπικό διαχειριστές (super admins) στο επίπεδο του δικτύου.
* Βελτιωμένο: Συσυ集ώθηκαν τα CSS του διαχειριστή σε ένα μόνο εξωτερικό stylesheet.
* Βελτιωμένο: Αφαιρέθηκε το υπομενού Ρυθμίσεις από το μενού διαχειριστή του υπο-site.
* Βελτιωμένο: Φόρτωση frontend assets μόνο υπό συνθήκη σε σελίδες ³τικετών υποστήριξης.
* Βελτιωμένο: Παραλείπεται ο plugin autoloader όταν ο root autoloader του Bedrock έχει ήδη φορτώσει τις εξαρτήσεις.

### 1.0.2 - 2025-12-11 {#102---2025-12-11}
* Προστέθηκε: Πλήρης λειτουργικότητα προβολής ³τικετών για τον διαχειριστή (admin) και το frontend.
* Προστέθηκε: AJAX handler για τις υποβολές απαντήσεων ³τικετών.
* Προστέθηκε: Υποστήριξη για τη λειτουργικότητα απάντησης ³τικετών με σωστή διαχείριση φόρμας.
* Προστέθηκε: Σωστή εμφάνιση ειδοποιήσεων για τις υποβολές και τις απαντήσεις ³τικετών στο admin.
* Προστέθηκε: Αυτόματη σύνδεση του ³τικετού με τον τρέχοντα χρήστη για τις αυτο-υποβολές των πελατών.
* Προστέθηκε: Βελτίωση ασφαλείας για την αποτροπή από πελάτες να παρακάμπτουν την ιδιοκτησία του ³τικετού.
* Προστέθηκαν: Λείπουν βοηθητικές συναρτήσεις (wu_format_date, wu_user_can_view_ticket, κ.λπ.).
* Προστέθηκε: Σωστή λήψη και διαχείριση συνημμένων αρχείων.
* Προστέθηκε: Σύστημα ειδοποιήσεων μέσω email για απαντήσεις και αλλαγές κατάστασης ³τικετών.
* Διορθώθηκε: Το φόρμα απάντησης ³τικετού περιλαμβάνει τώρα το απαραίτητο ID του ³τικετού.
* Διορθώθηκε: Ζητήματα ορατότητας ειδοποιήσεων στις προβολές ³τικετών του admin.
* Διορθώθηκε: Σφάλματα σύνταξης στις συναρτήσεις ³τικετών υποστήριξης.
* Βελτιωμένο: Δομή βάσης δεδομένων με σωστές κλάσεις ερωτήσεων και σχήματος για απαντήσεις και συνημμένα.
* Βελτιωμένο: Μετανάστευση σε πραγματικές ιδιότητες (real properties) αντί για χαρακτηριστικά (attributes) για τα μοντέλα ³τικετών.
