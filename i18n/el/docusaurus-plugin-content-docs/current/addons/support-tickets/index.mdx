# Support Tickets {#support-tickets}

**Support Tickets** είναι ένα ισχυρό addon για Ultimate Multisite που προσθέτει ένα ολοκληρωμένο σύστημα υποστήριξης αιτημάτων (support ticket system) στο WordPress network σας. Είναι ιδανικό για πλατφόρμες SaaS, παρόχους WaaS (Website as a Service) και multisite networks που χρειάζονται να παρέχουν υποστήριξη πελατών τόσο σε επίπεδο network όσο και σε επίπεδο μεμονωμένου site.

## Γιατί να επιλέξετε Support Tickets; {#why-choose-support-tickets}

* **Υποστήριξη σε πολλά επίπεδα (Multi-Level Support)**: Οι διαχειριστές του network διαχειρίζονται τα αιτήματα σε όλο το network, ενώ οι ιδιοκτήτες των sites διαχειρίζονται την υποστήριξη των δικών τους πελατών.
* **Επαγγελματική Ροή Εργασίας (Professional Workflow)**: Πλήρης διαχείριση του κύκλου ζωής των αιτημάτων, με προτεραιότητες, καταστάσεις και ανάθεση.
* **Εμπειρία Πελάτη (Customer Experience)**: ένα όμορφο, responsive interface που κάνει εύκολο για τους πελάτες να βρουν βοήθεια.
* **Ασφαλές και Надежное**: Κατασκευασμένο με τις βέλτιστες πρακτικές ασφαλείας του WordPress και βελτιστοποιημένη απόδοση της βάσης δεδομένων.
* **Αδιάλειπτη Ενσωμάτωση (Seamless Integration)**: Λειτουργεί τέλεια με τη διαχείριση πελατών και sites του Ultimate Multisite.

## Βασικά Χαρακτηριστικά {#key-features}

**Ολοκληρωμένη Διαχείριση Αιτημάτων (Complete Ticket Management)**

* Σύστημα μοναδικής αρίθρωσης αιτημάτων για εύκολη αναφορά.
* Προσαρμόσιμα επίπεδα προτεραιότητας (Χαμηλή, Κανονική, Υψηλή, Κρίσιμη).
* Ολοκληρωμένη ροή εργασίας καταστάσεων (Ανοιχτό, Σε Εξέλιξη, Αναμονή, Επίλυση, Κλειστό).
* Κατηγοριοποίηση αιτημάτων (Γενικό, Τεχνικό, Λογιστικό, Αίτημα Χαρακτηριστικού, Αναφορά Σφάλματος).
* Ανάθεση αιτημάτων σε πράκτορες υποστήριξης με επιλογή αυτόματης ανάθεσης.

**Συρραχιαίες Συνομιλίες (Threaded Conversations)**

* Πλήρης συρραχιαία συνομιλία για τις απαντήσεις στα αιτήματα.
* Διακριτική εμφάνιση για τις απαντήσεις του προσωπικού και του πελάτη.
* Ιδιωτικά σχόλια προσωπικού που δεν μπορούν να δουν οι πελάτες.
* Interface σε πραγματικό χρόνο με AJAX.
* Σύγχρονο επεξεργαστή κειμένου με επιλογές μορφοποίησης.

**Επισύλληψη Αρχείων (File Attachments)**

* Ασφαλής ανέβασμα αρχείων με υποστήριξη drag-and-drop.
* Προσαρμόσιμα περιοριστικά τύπου και μεγέθους αρχείου.
* Προηγμένη ασφάλεια με ανίχνευση κακόβουλου περιεχομένου.
* Προστατευμένα downloads με ελέγχους πρόσβασης.

**Έξυπνες Ειδοποιήσεις (Smart Notifications)**

* Αυτόματες ειδοποιήσεις μέσω email για νέα αιτήματα, απαντήσεις και αλλαγές κατάστασης.
* Προσαρμόσιμα email templates.
* Ειδοποιήσεις βάσει ρόλων για το προσωπικό και τους πελάτες.
* Δυναμικά μεταβλητά templates για εξατομικευμένα μηνύματα.

**Ισχυρό Admin Interface**

* Προηγμένη λίστα αιτημάτων με ταξινόμηση και φιλτράρισμα.
* Γρήγορη επεξεργασία για κατάσταση, προτεραιότητα και ανάθεση.
* Συλλογικές ενέργειες (Bulk actions) για αποδοτική διαχείριση.
* Dashboard στατιστικών με μετρήσεις απόδοσης.
* Ολοκληρωμένο πάνελ ρυθμίσεων.

**Όμορφος Πόρταλ Πελάτη (Beautiful Customer Portal)**

* Καθαρό, responsive interface σε όλες τις συσκευές.
* Σχεδιασμός με έμφαση στο mobile (Mobile-first design).
* Προηγμένη αναζήτηση και φιλτράρισμα.
* Σύγχρονος editor με thanh εργαλεία μορφοποίησης.
* Εύκολα στη χρήση φόρμες υποβολής αιτημάτων.

**Ευέλικτη Ενσωμάτωση (Flexible Integration)**

* Χρησιμοποιήστε shortcodes ή στοιχεία UI του Ultimate Multisite.
* Εμφανίστε φόρμες και λίστες αιτημάτων οπουδήποτε στο site σας.
* Λειτουργεί απρόσκοπτα με τη δομή του multisite network.
* Επεκτάσιμο με hooks και filters.

## Ιδανικό Για {#perfect-for}

* Πλατφόρμες SaaS που χρειάζονται υποστήριξη πελατών.
* Παρόχους WaaS που διαχειρίζονται πολλαπλά sites πελατών.
* Sites με συνδρομητικό μοντέλο (Membership sites) με διαβαθμισμένη υποστήριξη.
* Διαχειριστές network που διαχειρίζονται πολλαπλές marcas.
* Ιδιοκτήτες sites που παρέχουν εξυπηρέτηση πελατών.

## Εγκατάσταση {#installation}

1. Βεβαιωθείτε ότι το plugin Ultimate Multisite είναι εγκατεστημένο και ενεργοποιημένο στο WordPress network σας.
2. Ανέβασμα των αρχείων του addon στον κατάλογο `/wp-content/plugins/`.
3. Ενεργοποιήστε το plugin σε επίπεδο Network μέσω του μενού 'Plugins' στο WordPress.
4. Πλοηγηθείτε στο Network Admin > Support Tickets για να ρυθμίσετε τις επιλογές σας.

## Συχνές Ερωτήσεις {#frequently-asked-questions}

## Απαιτείται το Ultimate Multisite; {#does-this-require-ultimate-multisite}

Ναι, αυτό το addon απαιτεί το Ultimate Multisite να είναι εγκατεστημένο και ενεργό. Ενσωματώνεται βαθιά με τα χαρακτηριστικά διαχείρισης πελατών και sites του Ultimate Multisite.

## Οι ιδιοκτήτες sites μπορούν να διαχειρίζονται τη δική τους υποστήριξη πελατών; {#can-site-owners-handle-their-own-customer-support}

Απολύτως! Οι ιδιοκτήτες sites μπορούν να χρησιμοποιήσουν το σύστημα αιτημάτων για να παρέχουν υποστήριξη στους δικούς τους πελάτες, ενώ οι διαχειριστές του network μπορούν να διαχειρίζονται τα αιτήματα σε όλο το network.

## Πώς υποβάλλουν τα αιτήματα οι πελάτες; {#how-do-customers-submit-tickets}

Μπορείτε να προσθέσετε φόρμες υποβολής αιτημάτων οπουδήποτε στο site σας χρησιμοποιώντας shortcodes ή στοιχεία UI του Ultimate Multisite. Οι πελάτες απλώς συμπληρώνουν τη φόρμα και υποβάλλουν το αίτημά τους.

## Μπορώ να προσαρμόσω τις καταστάσεις και τις προτεραιότητες των αιτημάτων; {#can-i-customize-the-ticket-statuses-and-priorities}

Ναι! Μπορείτε να προσαρμόσετε πλήρως τα επίπεδα προτεραιότητας, τις καταστάσεις και τους τύπους αιτημάτων στις ρυθμίσεις του network admin ώστε να ταιριάζουν με τη ροή εργασίας υποστήριξης που θέλετε.

## Υποστηρίζει επισύλληψη αρχείων; {#does-it-support-file-attachments}

Ναι, τόσο οι πελάτες όσο και το προσωπικό μπορούν να επισύλλουν αρχεία στα αιτήματα και στις απαντήσεις. Έχετε πλήρη έλεγχο για τους επιτρεπόμενους τύπους αρχείων και τα όρια μεγέθους για λόγους ασφαλείας.

## Πώς διαχειρίζονται οι ειδοποιήσεις; {#how-are-notifications-handled}

Το σύστημα στέλνει αυτόματα ειδοποιήσεις μέσω email για νέα αιτήματα, απαντήσεις και αλλαγές κατάστασης. Μπορείτε να προσαρμόσετε όλα τα email templates στις ρυθμίσεις.

## Μπορώ να αναθέσω τα αιτήματα σε συγκεκριμένους πράκτορες υποστήριξης; {#can-i-assign-tickets-to-specific-support-agents}

Ναι, μπορείτε να αναθέσετε χειροκίνητα τα αιτήματα σε πράκτορες ή να ενεργοποιήσετε την αυτόματη ανάθεση για να διανεμηθούν τα αιτήματα αυτόματα.

## Υπάρχουν ιδιωτικά σχόλια για το προσωπικό; {#are-there-private-notes-for-staff}

Ναι, τα μέλη του προσωπικού μπορούν να προσθέσουν ιδιωτικές απαντήσεις που είναι ορατές μόνο σε άλλα μέλη του προσωπικού, και όχι στους πελάτες.

## Είναι φιλικό προς τα mobile; {#is-it-mobile-friendly}

Απολύτως! Ολόκληρο το interface είναι κατασκευασμένο με μια προσέγγιση mobile-first και λειτουργεί υπέροχα σε όλες τις συσκευές.

## Μπορώ να εμφανίσω λίστες και φόρμες αιτημάτων χρησιμοποιώντας shortcodes; {#can-i-display-ticket-lists-and-forms-using-shortcodes}

Ναι! Διαθέσιμα shortcodes:
* `[wu_support_tickets]` - Εμφανίζει τη λίστα αιτημάτων των πελατών.
* `[wu_support_ticket_form]` - Εμφανίζει τη φόρμα υποβολής νέου αιτήματος.
* `[wu_support_ticket_view ticket_id="123"]` - Εμφανίζει ένα συγκεκριμένο αίτημα.

## Τι τύποι αρχείων μπορούν να ανέβουν; {#what-file-types-can-be-uploaded}

Μπορείτε να ρυθμίσετε τους επιτρεπόμενους τύπους αρχείων στις ρυθμίσεις. Το σύστημα περιλαμβάνει προηγμένη σάρωση ασφαλείας για προστασία από κακόβουλα αρχεία.

## Μπορώ να εξάγω τα δεδομένα των αιτημάτων; {#can-i-export-ticket-data}

Ναι, το addon περιλαμβάνει δυνατότητες εξαγωγής για τα δεδομένα και τα στατιστικά των αιτημάτων.
