# Κατάλογος Αλλαγών ΦΠΑ

Έκδοση 1.0.7 - Έκδοση στις 2026-02-03

* Διόρθωση: Οι αριθμοί ΦΠΑ της Γερμανίας (DE) αποτύγουν την επικύρωση λόγω λαθών MS_MAX_CONCURRENT_REQ της υπηρεσίας VIES SOAP. Μεταπήδηση από την παλαιότερη SOAP API στην επίσημη EU VIES REST API για πιο αξιόπιστη επικύρωση.
* Διόρθωση: Οι αριθμοί ΦΠΑ αποδέχεστεί τώρα ακόμα και όταν η υπηρεσία VIES είναι προσωρινά μη διαθέσιμη, αντί να απορρίπτονται λανθασμένα. Δημιουργείται μια καταχώριση καταγραφής για περαιτέρω εξέταση.
* Διόρθωση: Το πεδίο ΦΠΑ/Αριθμός Φορολογικού Ταυτοποιητή είναι τώρα ορατό για όλες τις χώρες, και όχι μόνο για τα μέλη της ΕΕ. Οι αριθμοί ΦΠΑ μη-ΕΕ (π.χ. αριθμοί CHE της Ελβετίας) αποθηκεύονται για εμφάνιση τιμολογίων χωρίς επικύρωση VIES.
* Διόρθωση: Η αναζήτηση προεπίγραφης ΦΠΑ για την Ελλάδα (GR), το Μονακό (MC) και το νησί του Μαν (IM) ήταν κατεστραμμένη λόγω λανθασικής λογικής αναζήτησης πίνακα.
* Βελτίωση: Αφαιρέθηκε η περιττή εξάρτηση εκάστοτε `wp-cli-bundle` dev που μπορούσε να προκαλέσει 치ντικά λάθη όταν χρησιμοποιούνταν μαζί με το addon Site Exporter.
* Βελτίωση: Προστέθηκε πλήρες σύνολο δοκιμών μονάδων (53 δοκιμές).

Έκδοση 1.0.6 - Έκδοση στις 2026-01-25

* Διόρθωση: Οι εγχώριες συναλλαγές B2B χρεώνουν σωστά ΦΠΑ. Σύμφωνα με τους κανόνες ΦΠΑ της ΕΕ, η αντίστροφη χρέωση (reverse charge) εφαρμόζεται μόνο σε μετασυνοριακές συναλλαγές B2B, και όχι όταν η χώρα του πελάτη ταιριάζει με τη χώρα της εταιρείας.

Έκδοση 1.0.5 - Έκδοση στις 2026-01-22

* Διόρθωση: Το addon δεν φορτώνει ή δεν λειτουργεί σωστά.
* Αλλαγή: Μεταπήδηση της πηγής δεδομένων ρυθμών ΦΠΑ από euvatrates.com στο repository ibericode/vat-rates για πιο αξιόπιστα και ενεργά συντηρούμενα δεδομένα.
* Διόρθωση: Διορθώθηκε το λάθος στην τιμή της επιλογής `super_reduced_rates`.
* Βελτίωση: Προστέθηκαν ελέγχους null για τη διαχείριση κωδικών χωρών για την αποτροπή λαθών.
* Αλλαγή: Αφαιρέθηκαν τα πακέτα μεταφράσεων υπέρ των αυτόματων ενημερώσεων μέσω Traduttore.
