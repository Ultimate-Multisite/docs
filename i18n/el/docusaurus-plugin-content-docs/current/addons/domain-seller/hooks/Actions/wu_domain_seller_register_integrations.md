# Action: wu_domain_seller_register_integrations

Επιτρέπει σε άλλα plugins να καταχωρήσουν επιπλέον ενσωματώσεις (integrations) για πωλητές domains.

## Παράμετροι

| Name | Type | Description |
|------|------|-------------|
| $registry | `\WP_Ultimo\Integrations\Integration_Registry` | Το μητρώο ενσωματώσεων (integration registry). |

### Από

- 2.1.0
### Πηγή

Ορίζεται στο [`inc/class-domain-seller.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-domain-seller/blob/main/inc/class-domain-seller.php#L293) στην γραμμή 293
