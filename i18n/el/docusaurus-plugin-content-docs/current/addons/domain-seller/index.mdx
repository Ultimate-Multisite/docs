# Domain Seller

**Multisite Ultimate: Domain Seller** μετατρέπει το WordPress Multisite δίκτυό σας σε μια ισχυρή πλατφόρμα πώλησης domain. Επιτρέψτε στους πελάτες να καταχωρούν προσαρμοσμένα domain απευθείας κατά την εγγραφή ή από το Dashboard τους, με έλεγχο διαθεσιμότητας σε πραγματικό χρόνο και διαφανή τιμολόγηση.

### Γιατί Domain Seller; {#why-domain-seller}

Η καταχώρηση domain είναι ένα από τα πιο ζητούμενα upsells για οποιαδήποτε πλατφόρμα WaaS (Website as a Service). Αντί να στέλνετε τους πελάτες σας σε έναν τρίτο καταχωρητή, μπορείτε να προσφέρετε domain απευθείας — κρατώντας περισσότερα έσοδα στο δίκτυό σας ενώ παρέχετε στους πελάτες μια εμπειρία χωρίς τριβές.

### Βασικά χαρακτηριστικά {#key-features}

#### Υποστήριξη πολλαπλών καταχωρητών domain {#multiple-domain-registrar-support}
* **OpenSRS** — Πλήρης καταχώρηση, ανανέωση και διαχείριση
* **Namecheap** — Πλήρης ενσωμάτωση με WhoisGuard και δοκιμές sandbox
* **HostAfrica** — Αναζήτηση domain, τιμολόγηση, καταχώρηση, μεταφορά, DNS και προστασία ID μέσω του Domains Reseller API
* **Openprovider** — Τιμολόγηση μεταπωλητή, καταχώρηση, ανανέωση, μεταφορές, DNS, απόρρητο WHOIS και συγχρονισμός TLD
* **Hostinger** — Αναζήτηση και καταχώρηση domain χρησιμοποιώντας το κοινό Hostinger integration token
* **Επεκτάσιμη αρχιτεκτονική** — Βασισμένη στο μοτίβο Integration Registry, καθιστώντας εύκολη την προσθήκη περισσότερων παρόχων
* **Ενσωματωμένος έλεγχος σύνδεσης** — Επαληθεύστε τα διαπιστευτήρια παρόχου και τη συνδεσιμότητα πριν βγείτε ζωντανά

#### Καθοδηγούμενος οδηγός ρύθμισης {#guided-setup-wizard}
* **Ένταξη με ένα κλικ** — Σας καθοδηγεί στη διαμόρφωση παρόχου, την εισαγωγή TLD και τη δημιουργία προϊόντος domain
* **Αυτόματος συγχρονισμός TLD** — Εισάγει όλα τα διαθέσιμα TLD και την τιμολόγηση χονδρικής από τους συνδεδεμένους παρόχους σας
* **Δημιουργία προεπιλεγμένου προϊόντος** — Δημιουργεί ένα έτοιμο προς χρήση προϊόν domain με λογικές προεπιλογές προσαύξησης

#### Ευέλικτη τιμολόγηση domain {#flexible-domain-pricing}
* **Τρεις λειτουργίες προσαύξησης** — Ποσοστιαία προσαύξηση, σταθερή προσαύξηση ή σταθερή τιμή ανά TLD
* **Προϊόντα ειδικά ανά TLD** — Δημιουργήστε ξεχωριστά προϊόντα domain για διαφορετικές ομάδες TLD (.com/.net, .io, κωδικούς χωρών κ.λπ.)
* **Εισαγωγική τιμολόγηση** — Προσφέρετε μειωμένη τιμή πρώτου έτους για να προσελκύσετε νέες καταχωρήσεις
* **Τιμολόγηση σε πραγματικό χρόνο** — Ζωντανή τιμολόγηση χονδρικής από παρόχους με έξυπνη προσωρινή αποθήκευση
* **Υποστήριξη κουπονιών** — Πλήρης συμβατότητα με κωδικούς έκπτωσης Multisite Ultimate

#### Προστασία απορρήτου WHOIS {#whois-privacy-protection}
* **Διαμόρφωση ανά προϊόν** — Ορίστε το απόρρητο WHOIS σε Απενεργοποιημένο, Πάντα συμπεριλαμβανόμενο ή Επιλογή πελάτη
* **Προαιρετική συμμετοχή πελάτη** — Όταν έχει οριστεί σε Επιλογή πελάτη, ένα πλαίσιο ελέγχου στο checkout επιτρέπει στους πελάτες να προσθέσουν προστασία απορρήτου
* **Διαμορφώσιμη τιμολόγηση** — Ορίστε τιμή ανά έτος για το πρόσθετο απορρήτου ή προσφέρετέ το δωρεάν
* **Ενσωμάτωση καταχωρητή** — Ενεργοποιεί αυτόματα το WhoisGuard (Namecheap) ή το απόρρητο WHOIS (OpenSRS) κατά την καταχώρηση

#### Ενιαία εμπειρία checkout {#unified-checkout-experience}
* **Τρεις λειτουργίες domain** — Οι πελάτες επιλέγουν ανάμεσα σε ένα subdomain (δωρεάν), την καταχώρηση ενός νέου domain ή τη χαρτογράφηση ενός υπάρχοντος domain που ήδη κατέχουν
* **Αναζήτηση domain σε πραγματικό χρόνο** — Έλεγχος διαθεσιμότητας με AJAX χωρίς επαναφορτώσεις σελίδας
* **Έξυπνες προτάσεις** — Εναλλακτικές προτάσεις TLD όταν το προτιμώμενο domain είναι κατειλημμένο
* **Διαφανής τιμολόγηση** — Σαφής εμφάνιση της τιμής καταχώρησης, της τιμής ανανέωσης και της προαιρετικής χρέωσης απορρήτου WHOIS
* **Πεδία επικοινωνίας καταχωρίζοντος** — Συλλέγει τις απαιτούμενες πληροφορίες καταχωρίζοντος (όνομα, διεύθυνση, τηλέφωνο) εντός της ροής κατά το checkout
* **Αυτόματα δημιουργημένο URL ιστότοπου** — Η διεύθυνση ιστότοπου παράγεται αυτόματα από το επιλεγμένο domain

#### Σύστημα αυτόματης ανανέωσης {#auto-renewal-system}
* **Ανανεώσεις συνδεδεμένες με συνδρομή μέλους** — Τα domain ανανεώνονται αυτόματα όσο η συνδρομή μέλους του πελάτη είναι ενεργή
* **Έξυπνος προγραμματισμός** — Οι εργασίες ανανέωσης μπαίνουν σε ουρά με βάση τις ημερομηνίες λήξης των domain
* **Λογική επανάληψης** — Έξυπνες επαναλήψεις για αποτυχημένες προσπάθειες ανανέωσης
* **Ειδοποιήσεις email** — Διαμορφώσιμες ειδοποιήσεις για επερχόμενες ανανεώσεις, επιτυχημένες ανανεώσεις, αποτυχίες και λήξεις

#### Διαχείριση DNS από πελάτες {#customer-dns-management}
* **Αυτοεξυπηρέτηση DNS** — Οι πελάτες μπορούν να προσθέτουν, να επεξεργάζονται και να διαγράφουν εγγραφές DNS (A, AAAA, CNAME, MX, TXT) από το Dashboard τους
* **Κατάσταση domain** — Κατάσταση καταχώρησης, ημερομηνίες λήξης και πληροφορίες nameserver σε πραγματικό χρόνο
* **Αυτόματη χαρτογράφηση domain** — Τα καταχωρημένα ή υπάρχοντα domain χαρτογραφούνται αυτόματα στον ιστότοπο του πελάτη

#### Εργαλεία διαχειριστή {#admin-tools}
* **Χειροκίνητη καταχώρηση domain** — Καταχωρήστε domain εκ μέρους πελατών από τον πίνακα διαχείρισης
* **Παρακολούθηση domain** — Δείτε όλα τα domain πελατών, τις καταστάσεις και τις ημερομηνίες λήξης από μία σελίδα
* **Καταγραφή ειδική ανά πάροχο** — Ξεχωριστά κανάλια καταγραφής για κάθε καταχωρητή (π.χ., domain-seller-namecheap.log)
* **Λεπτομέρειες καταχώρησης** — Δείτε πληροφορίες καταχωρίζοντος και δεδομένα καταχώρησης στη βασική σελίδα επεξεργασίας domain

### Υποστηριζόμενοι πάροχοι {#supported-providers}

* **OpenSRS** — Πλήρης ενσωμάτωση με καταχώρηση, ανανέωση, διαχείριση DNS και απόρρητο WHOIS
* **Namecheap** — Πλήρης ενσωμάτωση με καταχώρηση, ανανέωση, μαζικό έλεγχο διαθεσιμότητας και δωρεάν WhoisGuard
* **HostAfrica** — Καταχώρηση, ανανέωση, μεταφορές, διαχείριση DNS, ανάκτηση κωδικού EPP, κλείδωμα καταχωρητή και προστασία ID
* **Openprovider** — Καταχώρηση, ανανέωση, μεταφορές, διαχείριση DNS, ανάκτηση κωδικού EPP, κλείδωμα καταχωρητή, απόρρητο WHOIS και συγχρονισμός TLD μεταπωλητή
* **Hostinger** — Καταχώρηση, αναζήτηση domain, ενημερώσεις nameserver, κλείδωμα καταχωρητή και απόρρητο WHOIS· οι ανανεώσεις χειρίζονται από την αυτόματη ανανέωση της Hostinger
* **GoDaddy** — Καταχώρηση και ανανέωση
* **ResellerClub** — Καταχώρηση, ανανέωση και διαχείριση DNS
* **NameSilo** — Καταχώρηση και ανανέωση
* **Enom** — Καταχώρηση, ανανέωση και διαχείριση DNS

### Τεχνικές λεπτομέρειες {#technical-details}

* **Απαιτεί:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **Αρχιτεκτονική:** Μοτίβο Integration Registry με interface Domain_Selling_Capability
* **Autoloading:** Jetpack autoloader με αντιστοίχιση κλάσεων PSR-4
* **Πρότυπα:** Πρότυπα κώδικα WordPress, αυστηρή τυποποίηση, prepared statements

## Εγκατάσταση {#installation}

1. Βεβαιωθείτε ότι το **Multisite Ultimate v2.4.12+** είναι εγκατεστημένο και ενεργοποιημένο σε επίπεδο δικτύου
2. Ανεβάστε το plugin στο `/wp-content/plugins/ultimate-multisite-domain-seller/`
3. Ενεργοποιήστε το plugin σε επίπεδο δικτύου από τα **Plugins** στο WordPress Network Admin
4. Το **Setup Wizard** εκκινείται αυτόματα κατά την πρώτη ενεργοποίηση — ακολουθήστε τα βήματα για να:
   * Συνδέσετε τον καταχωρητή domain σας
   * Δοκιμάσετε τη σύνδεση
   * Εισαγάγετε διαθέσιμα TLD και τιμές χονδρικής
   * Δημιουργήσετε το πρώτο σας προϊόν domain με προσαύξηση τιμής
5. Προσθέστε το πεδίο **Επιλογή Domain** στη φόρμα checkout σας στο **Ultimate Multisite > Φόρμες Checkout**

## Συχνές Ερωτήσεις {#frequently-asked-questions}

### Ποια έκδοση του Multisite Ultimate απαιτείται; {#what-version-of-multisite-ultimate-is-required}

Έκδοση 2.4.12 ή νεότερη. Το addon δεν είναι συμβατό με WP Ultimo v1.x ή εκδόσεις Ultimate Multisite κάτω από 2.4.12.

### Ποιοι πάροχοι domain υποστηρίζονται; {#which-domain-providers-are-supported}

Υποστηρίζονται εννέα καταχωρητές: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo** και **Enom**. Οι δυνατότητες διαφέρουν ανά πάροχο, επειδή τα API των καταχωρητών εκθέτουν διαφορετικές λειτουργίες· δείτε τον οδηγό εγκατάστασης για μια επισκόπηση ανά πάροχο. Η αρχιτεκτονική Integration Registry του addon καθιστά απλή την προσθήκη προσαρμοσμένων παρόχων — δείτε τον [οδηγό για προγραμματιστές](/developer/integration-guide/custom-registrar).

### Μπορώ να προσφέρω προστασία απορρήτου WHOIS; {#can-i-offer-whois-privacy-protection}

Ναι. Κάθε προϊόν domain έχει μια ρύθμιση Απορρήτου WHOIS με τρεις λειτουργίες: Απενεργοποιημένο, Πάντα Συμπεριλαμβάνεται ή Επιλογή Πελάτη. Όταν έχει οριστεί σε Επιλογή Πελάτη, οι πελάτες βλέπουν ένα πλαίσιο επιλογής κατά το checkout με την τιμή (ή "Δωρεάν" για το WhoisGuard της Namecheap). Η σημαία απορρήτου μεταβιβάζεται αυτόματα στον καταχωρητή κατά την εγγραφή.

### Μπορώ να ορίσω διαφορετική τιμολόγηση για διαφορετικά TLD; {#can-i-set-different-pricing-for-different-tlds}

Ναι. Δημιουργήστε ξεχωριστά προϊόντα domain για διαφορετικές ομάδες TLD. Κάθε προϊόν μπορεί να έχει τον δικό του τύπο προσαύξησης (ποσοστό, σταθερή προσαύξηση ή σταθερή τιμή), υποστηριζόμενα TLD και εισαγωγική τιμολόγηση.

### Ποιες επιλογές checkout βλέπουν οι πελάτες; {#what-checkout-options-do-customers-see}

Το πεδίο Επιλογή Domain δίνει στους πελάτες τρεις καρτέλες: **Subdomain** (χρήση ενός δωρεάν subdomain στο δίκτυό σας), **Εγγραφή Νέου Domain** (αναζήτηση και εγγραφή νέου domain) και **Υφιστάμενο Domain** (αντιστοίχιση ενός domain που ήδη κατέχουν). Θα εμφανίζονται μόνο οι λειτουργίες που ενεργοποιείτε.

### Υποστηρίζει αυτόματες ανανεώσεις; {#does-it-support-automatic-renewals}

Ναι. Τα domains μπορούν να ρυθμιστούν για αυτόματη ανανέωση και συνδέονται με την κατάσταση συνδρομής του πελάτη. Οι πληρωμές ανανέωσης επεξεργάζονται μέσω της υπάρχουσας πύλης πληρωμών του πελάτη.

### Μπορούν οι πελάτες να διαχειρίζονται τις εγγραφές DNS τους; {#can-customers-manage-their-dns-records}

Ναι. Οι πελάτες αποκτούν μια διεπαφή διαχείρισης DNS στο dashboard τους για να προσθέτουν, να επεξεργάζονται και να διαγράφουν εγγραφές A, AAAA, CNAME, MX και TXT. Η διαχείριση DNS είναι διαθέσιμη με OpenSRS, ResellerClub, Enom, HostAfrica και Openprovider. Το DNS της Hostinger για φιλοξενούμενα domains διαχειρίζεται από την κύρια ενσωμάτωση αντιστοίχισης domain της Hostinger.

### Υπάρχει setup wizard; {#is-there-a-setup-wizard}

Ναι. Το setup wizard εκτελείται αυτόματα κατά την πρώτη ενεργοποίηση και σας καθοδηγεί στη σύνδεση ενός καταχωρητή, την εισαγωγή TLD και τη δημιουργία του πρώτου σας προϊόντος domain. Μπορείτε επίσης να εκτελέσετε ξανά τον wizard από τις ρυθμίσεις του addon.
