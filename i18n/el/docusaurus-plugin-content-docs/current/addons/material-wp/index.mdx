# Material WP {#material-wp}

Το περιβάλλον διαχείρισης του WordPress έχει παραμείνει το ίδιο εδώ και χρόνια. Αλλά νομίζατε ποτέ πώς θα έμοιαζε το περιβάλλον διαχείρισης με ένα Material Design;

Το Material WP είναι ένα theme διαχείρισης που αναπτύχθηκε για να σας δώσει αυτή τη εμφάνιση και αίσθηση, βοηθώντας σας να παρέχετε στους πελάτες σας ένα πιο προσαρμοσμένο Dashboard.

## Χαρακτηριστικά {#features}

- Υποστήριξη προσαρμοσμένου CSS
- Εύκολα ρυθμίσεις στυλ
- Αναδιάταξη μενού
- Εύκολη εισαγωγή/εξαγωγή

## Τεκμηρίωση {#documentation}

**Προσαρμολόγηση του Πάνελ Διαχείρισης σας με το Material WP**

## Εγκατάσταση {#installation}

1. Ανεβάστε το 'material-wp' στον κατάλογο '/wp-content/plugins/'
2. Ενεργοποιήστε το plugin μέσω του μενού 'Plugins' στο WordPress

## Οδηγός Χρήσης {#usage-guide}

Το περιβάλλον διαχείρισης του WordPress έχει παραμείνει το ίδιο εδώ και χρόνια. Αλλά νομίζατε ποτέ πώς θα έμοιαζε το περιβάλλον διαχείρισης με ένα Material Design;

Το Material WP είναι ένα theme διαχείρισης που αναπτύχθηκε για να σας δώσει αυτή τη εμφάνιση και αίσθηση, βοηθώντας σας να παρέχετε στους πελάτες σας ένα πιο προσαρμοσμένο Dashboard. Το Material WP αποτελεί επίσης μέρος του core premium plugins της Ultimate Multisite.

Μετά την **εγκατάστασή του**, μεταβείτε στη σελίδα σας _Plugins_ και κάντε κλικ για να το ενεργοποιήσετε στο Network σας.

Τώρα, έχετε επανασχεδιαστεί πλήρως το περιβάλλον διαχείρισης! Έχετε επίσης ένα νέο μενού στην αριστερή στήλη που ονομάζεται Material WP
