# Κατάλογος Αλλαγών Content Sync {#content-sync-changelog}

### Έκδοση 1.1.0 - Έκδοση στις 2025-12-31 {#version-110---released-on-2025-12-31}
* Νέο: Υποστήριξη συγχρονισμού όλων των τύπων περιεχομένου (posts, pages, custom post types)
* Νέο: Φιλτράρισμα τύπων περιεχομένου στο βήμα επιλογής περιεχομένου
* Νέο: Βήμα επιλογής πηγής site - συγχρονισμός από οποιοδήποτε site, και όχι μόνο από το κύριο site
* Νέο: Προ-επιλογή υπο-sites για τα template sites
* Νέο: Εξαγωγή μέσων από το post_content για όλους τους τύπους περιεχομένου
* Νέο: Συγχρονισμός ταξινομικών (κατηγορίες, tags, custom taxonomies)
* Αλλαγή: Μετονομάστηκε από "Elementor Template Sync" σε "Content Sync"
* Αλλαγή: Το βήμα "Select Templates" μετονομάστηκε σε "Select Data"
* Βελτίωση: Ο κώδικας που σχετίζεται με το Elementor τρέχει πλέον μόνο για τον τύπο περιεχομένου elementor_library

### Έκδοση 1.0.0 - Έκδοση στις 2025-12-19 {#version-100---released-on-2025-12-19}
* Αρχική έκδοση
* Πολυδιάστατο wizard για επιλογή template και site
* Διαδικασία επεξεργασίας σε παρτίδες με ανατροφοδότηση προόδου σε πραγματικό χρόνο
* Πλήρης συγχρονισμός μέσων με αυτόματη αναназнаίωση ID
* Ενσωμάτωση ρυθμίσεων με το Ultimate Multisite
* Υποστήριξη αποκλεισμού sites μέσω των ρυθμίσεων του network
* Καθάρισμα cache του Elementor μετά τον συγχρονισμό
* Διατήρηση ταξινομικών των templates
