# Σύγχρονιση Περιεχομένου {#content-sync}

Multisite Ultimate: Σύγχρονιση Περιεχομένου

Αυτό το add-on για το Multisite Ultimate επιτρέπει στους διαχειριστές του network να συγχρονίζουν περιεχόμενο από οποιοδήποτε site σε άλλα sites του network. Υποστηρίζει όλα τα post types, συμπεριλαμβανομένων των posts, pages, custom post types και Elementor templates. Το add-on περιλαμβάνει μια διεπαφή wizard με πολλά βήματα για την επιλογή περιεχομένου και στόχων sites, καθώς και batch processing και feedback της πρόοδου σε πραγματικό χρόνο.

## Χαρακτηριστικά {#features}

* Επιλογή Περιεχομένου: Οπτική πλέγμα καρτών με μικρογραφίες και φιλτράρισμα τύπου post.
* Όλοι οι Τύποι Post: Συγχρονίζει posts, pages, προϊόντα, Elementor templates και οποιονδήποτε custom post type.
* Επιλογή Sites: Επιλέγει σε ποια sites να συγχρονιστεί το περιεχόμενο, με εξαίρεση το source site και προ-επιλογή των child sites.
* Batch Processing: Συγχρονίζει σε 10 sites ταυτόχρονα με ενημερώσεις πρόοδου.
* Συγχρονισμός Media: Αντιγράφει και ανακατευθύνει αυτόματα τα media files (εικόνες, βίντεο) από το περιεχόμενο των posts στα target sites.
* Συγχρονισμός Ταξινομικών: Διατηρεί τις κατηγορίες, τα tags και τα custom taxonomies.
* Υποστήριξη Elementor: Πλήρης υποστήριξη για Elementor templates με αναγέννηση CSS και ανακατεύθυνση ID templates.
* Ανάλυση Συγκρούσεων: Επιλογή αντικατάστασης του υπάρχοντος περιεχομένου ή παραλείpx του.
* Διαχείριση Cache: Καθαρίζει αυτόματα το cache του Elementor στα target sites μετά τη συγχρονιστική διαδικασία.

## Απαιτήσεις {#requirements}

* WordPress 5.3 ή νεότερη
* PHP 7.4 ή νεότερη
* Multisite Ultimate plugin (ενεργοποιημένο)
* Elementor plugin (προαιρετικό, απαιτείται για συγχρονισμό Elementor templates)

## Εγκατάσταση {#installation}

1. Ανεβάστε το 'ultimate-multisite-content-sync' στον κατάλογο '/wp-content/plugins/'
2. Ενεργοποιήστε το plugin μέσω του μενού 'Plugins' στο WordPress Network Admin σας.
3. Ρυθμίστε τις ρυθμίσεις στο Ultimate Multisite → Settings → Content Sync.
4. Πλοηγηθείτε στο Ultimate Multisite → Content Sync για να ξεκινήσετε τη συγχρονιστική διαδικασία.
