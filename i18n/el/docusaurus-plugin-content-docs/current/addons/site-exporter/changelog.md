# Κατάλογος Αλλαγών Site Exporter

Έκδοση: 1.0.6 - Έκβηκε στις 2026-05-11
* Νέο: Οι εξαγωγές ιστοσελίδων περιλαμβάνουν τώρα ένα index.php που εκκινείται αυτόματα (self-booting), ώστε το ZIP να μπορεί να εγκατασταθεί σε ένα νέο hosting χωρίς να χρειάζεται ξεχωριστή εγκατάσταση plugin.
* Νέο: Η εξαγωγή δικτύου (Network export) επιτρέπει στους διαχειριστές να εξάγουν όλα τα υποσυστήματα (subsites) σε ένα μόνο αρχείο από τη σελίδα διαχειριστή Site Export.

Έκδοση: 1.0.5 - Έκβηκε στις 2026-05-05
* Διόρθωση: Προστέθηκε το __callStatic στο WP_CLI polyfill stub για να χειριστεί μεθόδους που δεν είναι mock-owane, όπως το get_config, αποτρέποντας έτσι αποτυχίες εκκίνησης δοκιμών (test bootstrap failures).

Έκδοση: 1.0.4 - Έκβηκε στις 2025-11-25

* Διόρθωση: Διόρθωση συμβατότητας με το Ultimate Multisite 2.4.

Έκδοση: 1.0.2 - Έκβηκε στις 2025-09-28

* Μετονομασία prefix σε ultimate-multisite; ενημέρωση text domain; αύξηση έκδοσης.

### Έκδοση 1.0.1 - Έκβηκε στις 2023-08-09

* Βελτιωμένο: Τώρα διασφαλίζει ότι λαμβάνεται ο σωστός δρόμος αρχείου (file path) του αρχείου εισαγωγής (importer).
* Διόρθωση: Απενεργοποιήθηκε το object cache για την αποφυγή σφαλμάτων κατά την εισαγωγή.
* Διόρθωση: Διόρθωση του προβλήματος με τον σύνδεσμο λήψης του plugin εισαγωγής που δεν λειτουργούσε.

### Έκδοση 1.0.0 - Έκβηκε στις 2022-12-23

* Διόρθωση: Το popup του φόρμας εξαγωγής δεν φορτωνόταν;
* Διόρθωση: Τα αρχεία Javascript δεν φορτωνόταν στη σελίδα εξαγωγής;
* Εσωτερικό: Νέο διαδικασία κατασκευής (build process);

### Έκδοση 1.0.0-beta.4 - Έκβηκε στις 2022-01-13

* Εσωτερικό: Προστέθηκε ο γεννήτρια hooks και filters;
* Εσωτερικό: Προστέθηκαν τα stubs του WP Ultimo για τη βελτίωση της ποιότητας ζωής των developers;
