# Filter: wp_ultimo_site_exporter_url

Επιτρέπει στους developers να διαχειριστούν τον σχετικό δρόμο (relative path) προς τη ρίζα του plugin.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $dir | `string` | Ο δρόμος σχετικό με τη ρίζα του plugin που θέλετε να αποκτήσετε πρόσβαση. |

### Since

- 0.0.1
### Source

Ορίζεται στο [`inc/class-helper.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-site-exporter/blob/main/inc/class-helper.php#L52) στην γραμμή 52


## Returns
Ο δρόμος σχετικό με τη ρίζα του plugin.
