# Tātari: networks_user_is_network_admin

Ka tātari i ngā network e whakahaerehia ana e tētahi kaiwhakamahi hei kaiwhakahaere.

## Ngā Tawhā {#parameters}

| Ingoa | Momo | Whakaahuatanga |
|------|------|-------------|
| $ | `array\|bool` | Rārangi o ngā ID network, he false rānei mēnā kāore he network mō te kaiwhakamahi. |
| $ | `int` | ID kaiwhakamahi e tika ana kia whakahokia mai ngā network mōna. |
