# Hohenga: switch_network

Ka whakakāngia ina whakawhitia te horopaki network o nāianei.

## Tawhā {#parameters}

| Ingoa | Momo | Whakaahuatanga |
|------|------|-------------|
| $new_network_id | `int` | ID o te network e whakawhitia atu ana. |
| $old_network_id | `int` | ID o te network o mua i noho hei network o nāianei. |
