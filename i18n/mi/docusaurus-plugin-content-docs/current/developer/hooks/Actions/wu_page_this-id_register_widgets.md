# Action: wu_page_\{$this->id\}_register_widgets

Ka whakahohehia i muri i te rēhitatanga o ngā widget mō tēnei whārangi.

The dynamic portion of the hook name, `$this-&gt;id`, refers to the page id.

## Ngā Tawhā

| Ingoa | Momo | Whakaahuatanga |
|------|------|-------------|
| $id | `string` | Te id o te whārangi. |
| $page_hook | `string` | Te hook o te whārangi. |
| $page | `object` | Te ahanoa whārangi. |
