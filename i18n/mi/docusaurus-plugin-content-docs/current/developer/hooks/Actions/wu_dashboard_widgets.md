# Hohenga: wu_dashboard_widgets {#action-wudashboardwidgets}

Tukua ngā kaiwhakawhanake plugin kia tāpiri widget ki te Paewhiri Network Dashboard.

## Tawhā {#parameters}

| Ingoa | Momo | Whakaahuatanga |
|------|------|-------------|
| $tab | `string` | Te ripa o nāianei. |
| $screen | `\WP_Screen` | Te ahanoa mata. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Te tauira whārangi kaiwhakahaere Ultimate Multisite. |
