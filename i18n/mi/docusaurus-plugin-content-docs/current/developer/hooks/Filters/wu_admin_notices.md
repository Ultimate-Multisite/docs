# Tātari: wu_admin_notices

Tukua ngā kaiwhakawhanake kia tātari i ngā pānui admin kua tāpirihia e Ultimate Multisite.

## Tawhā {#parameters}

| Ingoa | Momo | Whakaahuatanga |
|------|------|-------------|
| $notices | `array` | Rārangi o ngā pānui mō taua paewhiri motuhake. |
| $all_notices | `array` | Rārangi o ngā pānui kua tāpirihia, kua wehea mā te paewhiri. |
| $panel | `string` | Paewhiri hei tiki i ngā pānui. |
| $filter | `string` | Mēnā kua tātaria atu ngā pānui ka taea te whakakore. |
| $dismissed_messages | `array` | Rārangi o ngā kī pānui kua whakakorea. |

### Mai i {#since}

- 2.0.0
