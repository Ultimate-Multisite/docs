# Ìwé ìtọ́nisọ́nà Ultimate Multisite

Káàbọ̀ sí Ultimate Multisite documentation. Ultimate Multisite ń yí WordPress Multisite padà sí pẹpẹ Website as a Service (WaaS) pípé.

## Àwọn abala ìwé ìtọ́nisọ́nà {#documentation-sections}

### [Ìtọnisọna Olùmúlò](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

Gbogbo ohun tí o nílò láti ṣètò, ṣàgbékalẹ̀, àti ṣàkóso pẹpẹ WaaS rẹ. Ó kàn fifi sílẹ̀, àwọn payment gateway, domain mapping, ìṣàkóso client, àti bẹ́ẹ̀ bẹ́ẹ̀ lọ.

### [Ìwé ìtọ́nisọ́nà fún olùgbéejáde](/developer/hooks) {#developer-documentation}

Ìtọ́kasí imọ̀-ẹrọ fún àwọn developer tí ń kọ́ lórí Ultimate Multisite. Ó ní hooks reference, REST API documentation, àti àwọn ìtọnisọna ìdàgbàsókè addon.

### [Àfikún](/addons) {#addons}

Documentation fún àwọn addon plugins tí a ń san owó fún tí ń faagun Ultimate Multisite pẹ̀lú àwọn ẹya àfikún bíi ìṣepọ̀ WooCommerce, analytics, títà domain, àti bẹ́ẹ̀ bẹ́ẹ̀ lọ.

- [GitHub Issues](https://github.com/Ultimate-Multisite/ultimate-multisite/issues) -- Jábọ̀ àwọn àṣìṣe
- [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) -- Ṣe ìgbàsílẹ̀ plugin pàtàkì
