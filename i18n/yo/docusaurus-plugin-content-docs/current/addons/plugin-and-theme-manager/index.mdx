# Oluṣakoso Afikun & Akori

Ultimate Multisite: Oluṣakoso Afikun ati Akori

Ṣàtúnṣe tàbí fi alaye meta pamọ́ (akọle, onkọwe, aworan kekere àti àpejuwe) ti àwọn afikun àti akori tí a ti fi sori ẹrọ. Ṣẹda àwọn ẹ̀ka fún àwọn afikun àti akori láti jẹ́ kí àwọn olumulo rẹ lè ṣe àlẹmọ wọn, kí o sì ṣafikun oju-iwe Àwọn Afikun aṣa tó lẹ́wa sí paneli àwọn olumulo rẹ.

Láti lo afikun náà, o kan lè lọ sí àwọn oju-iwe Àwọn Afikun tàbí Àwọn Akori rẹ ní network admin rẹ, kí o sì tẹ "Ṣàtúnṣe alaye Afikun/Akori".

## Fífi sori ẹrọ

1. Ṣe ìkójọpọ̀ 'ultimate-multisite-plugin-and-theme-manager' sí itọsọna '/wp-content/plugins/'
2. Mú afikun náà ṣiṣẹ́ nípasẹ̀ akojọ 'Àwọn Afikun' nínú WordPress Network Admin rẹ

## Ìtọ́sọ́nà Lílo

Ultimate Multisite: Oluṣakoso Afikun ati Akori jẹ́ afikun Ultimate Multisite **premium** alágbára tí ó fún ọ ní ìṣàkóso kíkún lórí bí àwọn olumulo nẹ́tíwọ́ọ̀kì rẹ ṣe ń rí àti ṣe ìbáṣepọ̀ pẹ̀lú àwọn aṣayan afikun àti akori lórí nẹ́tíwọ́ọ̀kì rẹ.

Afikun yìí ń ṣiṣẹ́ fún ìdí mẹ́ta:

  1. Ó jẹ́ kí o yí metadata àwọn afikun àti akori méjèèjì padà pátápátá. Ìyẹn pẹlu orúkọ, àpejuwe, orúkọ onkọwe, àti àwọn aworan kekere pàápàá!

  2. Ó jẹ́ kí o pín àwọn afikun àti akori sí ẹ̀ka, láti jẹ́ kí àwọn olumulo rẹ lè ṣètò wọn ní rọrùn.

  3. Ó fún ọ ní aṣayan láti rọ́pò oju-iwe àwọn afikun àtijọ́, aláìnífẹ̀ẹ́, pẹ̀lú oju-iwe tuntun tó dà bí ti àwọn akori, pẹ̀lú ṣíṣe ìtòlẹ́sẹẹsẹ nípasẹ̀ àwọn ẹ̀ka.

Lẹ́yìn gbígbà á sílẹ̀ àti fífi sori ẹrọ, taabu tuntun kan - _Ètò Afikun_ \- yóò hàn lórí oju-iwe Ètò Ultimate Multisite.
