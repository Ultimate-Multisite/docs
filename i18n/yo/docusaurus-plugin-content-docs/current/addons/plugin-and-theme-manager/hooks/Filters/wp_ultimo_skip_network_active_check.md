# Àlẹmọ: wp_ultimo_skip_network_active_check {#filter-wpultimoskipnetworkactivecheck}

Fàyè gba àwọn developer láti short-circuit ìṣàyẹ̀wò yìí.

Èyí wúlò nígbà tí a bá ń lo àwọn ìṣètò tó dá lórí composer àti àwọn ìṣètò àkànṣe míì, bíi Bedrock, fún àpẹẹrẹ, níbi tí lílo àwọn plugin gẹ́gẹ́ bí mu-plugins ti jẹ́ ohun tí ó wọ́pọ̀.

### Láti {#since}

- 2.0.0
