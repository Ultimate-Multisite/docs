# Àwọn Ètò Tí a Nwọ̀n {#metered-plans}

Ultimate Multisite: addon Àwọn Ètò Tí a Nwọ̀n yí bí o ṣe ń gba owó lọ́wọ́ àwọn oníbàárà rẹ padà nípa fífi agbára ìsanwó tó rọrùn gẹ́gẹ́ bí lílò kún un. Dípò àwọn ààlà líle tí kò rọrùn tí ń dí àwọn aṣàmúlò dúró nígbà tí wọ́n bá kọjá wọn, jẹ́ kí àwọn owó àfikún lílò ṣiṣẹ́ tí yóò gba owó fún lílò àfikún láìfọwọ́sí.

## Àwọn Àmì Pàtàkì {#key-features}

* **Ìsanwó Gẹ́gẹ́ Bí Lílò**: Gba owó lọ́wọ́ àwọn oníbàárà dá lórí lílò gidi dípò àwọn ìpele tó dájú
* **Ìṣàkóso Àfikún Lílò**: Gba owó fún lílò tó kọjá ààlà ètò láìfọwọ́sí
* **Títọpa Lílò**: Ṣàbójútó bí oníbàárà ṣe ń lo ohun àmúṣọrọ ní àkókò gidi
* **Ìdíyelé Tó Rọrùn**: Ṣètò iye owó àkànṣe fún àwọn ohun àmúṣọrọ ọ̀tọ̀ọ̀tọ̀ (ibi ìpamọ́, bandwidth, àwọn site, àti bẹ́ẹ̀ bẹ́ẹ̀ lọ)
* **Ìsanwó Token AI**: Tọpa kí o sì gba owó fún àwọn token ìwọlé àti ìjáde fún lílò AI káàkiri àwọn subsite oníbàárà
* **Ṣíṣe Ìwé-ìdíyelé Láìfọwọ́sí**: Ṣẹ̀dá àwọn ìwé-ìdíyelé láìfọwọ́sí nígbà tí àwọn ìpele ààlà lílò bá kọjá
* **Àwọn Ìròyìn Lílò**: Ìtúpalẹ̀ lílò àti ìròyìn kíkún fún àwọn oníbàárà àti àwọn alábójútó

## Ó Pé Fún {#perfect-for}

* **Àwọn Olùpèsè Hosting**: Pèsè àwọn ètò tó lè gbooro pẹ̀lú àìní oníbàárà
* **Àwọn Pẹpẹ SaaS**: Gba owó dá lórí lílò ohun àmúṣọrọ gidi
* **Àwọn Nẹ́tíwọ̀ọ̀kì Agency**: Àwọn àwòṣe ìdíyelé tó rọrùn fún àwọn iṣẹ́ akanṣe client
* **Reseller Hosting**: Jẹ́ kí àwọn alábàáṣiṣẹ́ lè pèsè ìdíyelé tó dá lórí lílò

## Bí Ó Ṣe Ń Ṣiṣẹ́ {#how-it-works}

1. Ṣètò àwọn ohun àmúṣọrọ tí a nwọ̀n (ààyè disk, bandwidth, iye àwọn site, àti bẹ́ẹ̀ bẹ́ẹ̀ lọ)
2. Ṣètò àwọn ààlà ètò ìpilẹ̀ àti àwọn iye owó àfikún lílò
3. Ẹ̀rọ náà yóò tọpa lílò káàkiri nẹ́tíwọ̀ọ̀kì rẹ láìfọwọ́sí
4. Nígbà tí àwọn oníbàárà bá kọjá àwọn ààlà, a yóò ṣírò àwọn owó àfikún lílò
5. A yóò ṣẹ̀dá àwọn ìwé-ìdíyelé láìfọwọ́sí dá lórí iyípo ìsanwó
6. Àwọn oníbàárà lè wo lílò wọn àti àwọn owó tó ń bọ̀ ní àkókò gidi

## Ìsanwó Token AI {#ai-token-billing}

Ẹ̀yà 1.1.0 ṣàfikún ìsanwó token AI fún àwọn nẹ́tíwọ̀ọ̀kì tí ń pèsè àwọn àfikún AI nípasẹ̀ WordPress AI Client SDK. Àwọn Ètò Tí a Nwọ̀n ń fetí sí àwọn ìbéèrè ìṣẹ̀dá AI tí ó parí, ń ṣàkọsílẹ̀ iye token ìwọlé àti ìjáde fún subsite lọwọlọwọ, ó sì ń yí àwọn token wọ̀nyẹn padà sí ináwó AI tí a lè gba owó fún ní àkókò ìsanwó lọwọlọwọ ti oníbàárà.

Lo ìsanwó token AI nígbà tí o bá fẹ́ pèsè àwọn irinṣẹ́ AI lórí àwọn site oníbàárà láì gba àwọn iye owó model tí kò ṣeé sọ tẹ́lẹ̀ sí ara rẹ. Ìbéèrè kọ̀ọ̀kan ni a so mọ́ subsite, oníbàárà, membership, provider, àti model kí àwọn alábójútó lè ṣàyẹ̀wò lílò nígbà tí àwọn oníbàárà lè rí bí lílò AI wọn ṣe ń kan Account wọn.

### Ṣètò Àwọn Iye Owó Fún Token Kọ̀ọ̀kan {#configure-per-token-rates}

A ń ṣètò àwọn iye owó AI fún model kọ̀ọ̀kan gẹ́gẹ́ bí àwọn iye owó ìwọlé àti ìjáde tó yàtọ̀. A ń ṣírò àwọn iye owó fún token mílíọ̀nù 1, èyí tó bá àwọn ojú-ewé ìdíyelé provider AI wọ́pọ̀ mu.

1. Nínú alábójútó nẹ́tíwọ̀ọ̀kì, ṣí **Multisite Ultimate → Settings → Metered Plans**.
2. Mú kí ìwọ̀n ináwó AI ṣiṣẹ́ fún product tàbí membership tí ó yẹ kí ó gba owó fún lílò AI.
3. Ṣàfikún àwọn ìwọlé iye owó model ní lílo àwọn ID model tí connector rẹ dá padà, bíi `gpt-4o`, `gpt-4o-mini`, tàbí ìdánimọ̀ model míì tó bá OpenAI mu.
4. Tẹ àwọn iye **Iye Owó Ìwọlé** àti **Iye Owó Ìjáde** tó yàtọ̀ sílẹ̀ nínú owó site rẹ fún token mílíọ̀nù 1.
5. Ṣètò iye ààyè ináwó AI tó wà nínú ètò oníbàárà àti ìhùwàsí àfikún lílò fún plan náà.

Nígbà tí ID model gangan kò bá ṣètò, Àwọn Ètò Tí a Nwọ̀n lè bá a mu nípasẹ̀ ìbẹ̀rẹ̀ model. Fún àpẹẹrẹ, iye owó tí a ṣètò fún `claude-sonnet-4` lè kan àwọn ID model pẹ̀lú àmì àkókò tí ó bẹ̀rẹ̀ pẹ̀lú ìbẹ̀rẹ̀ yẹn. Àwọn olùdàgbàsókè tún lè ṣàtúnṣe tabili iye owó ìkẹyìn pẹ̀lú filter [`wu_ai_token_rates`](./hooks/Filters/wu_ai_token_rates).

### Àwọn Connector AI Tí a Ṣe Àtìlẹ́yìn Fún {#supported-ai-connectors}

Ìsanwó token AI ń ṣiṣẹ́ pẹ̀lú àwọn connector tí ń tú àwọn ìṣẹ̀lẹ̀ ìṣẹ̀dá WordPress AI Client SDK jáde tí wọ́n sì ní lílò token nínú metadata èsì. Èyí ní ẹbí connector Ultimate Multisite AI tí a ṣàkọsílẹ̀ níbí:

* [Ultimate AI Connector fún Àwọn Ojúami-ipari Tí Ó Bá Mu](../ultimate-ai-connector-compatible-endpoints/) fún àwọn API tó bá OpenAI mu, pẹ̀lú OpenAI, Mistral, àwọn gateway tó bá Ollama mu, àti àwọn provider míì tí ń ṣàfihàn metadata lílò token tó bá mu
* [Ultimate AI Connector fún WebLLM](../ultimate-ai-connector-webllm/) fún lílò WebLLM abínibí browser nígbà tí àwọn ìbéèrè bá ń gba inú sísàn ìṣẹ̀lẹ̀ AI Client SDK kọjá
* Àwọn connector provider AI tí [Gratis AI Agent](../gratis-ai-agent/) ń lo nígbà tí provider tí a yan bá ń jábọ̀ iye token prompt àti completion nípasẹ̀ SDK

Àwọn connector àkànṣe lè kópa nípa jíjẹ́ kí àwọn ìṣẹ̀lẹ̀ boṣewa `wp_ai_client_before_generate_result` àti `wp_ai_client_after_generate_result` ṣiṣẹ́ pẹ̀lú model, provider, àti metadata lílò token. A ń fi àwọn ètò connector múlẹ̀ láti site àkọ́kọ́ sí àwọn subsite kí àwọn site oníbàárà lè lo àwọn provider àti ààlà tí nẹ́tíwọ̀ọ̀kì fọwọ́ sí.

### Lílò Account Oníbàárà {#customer-account-usage}

Àwọn oníbàárà rí lílò AI lẹ́gbẹ̀ẹ́ àwọn ohun àmúṣọrọ tí a nwọ̀n míì nínú agbègbè Account wọn. Ìlà ináwó AI ń fi apapọ̀ àkókò ìsanwó lọwọlọwọ hàn tí a yọ láti àwọn token prompt àti completion tí a ṣàkọsílẹ̀, àwọn iye owó model tí a ṣètò, àti iye ààyè èyíkéyìí tó wà nínú plan náà.

Bí oníbàárà bá ní ọ̀nà ìsanwó lórí fáìlì, lílò tó ju iye ààyè tó wà nínú ètò lọ ni a máa kà sí àfikún lílò tí a lè gba owó fún. Bí plan náà bá ṣètò fún dídá líle dúró tí oníbàárà kò sì ní ọ̀nà ìsanwó tó wà, a lè dí àwọn ìbéèrè AI dúró lẹ́yìn tí iye ààyè bá dé, a ó sì rán oníbàárà létí láti ṣàfikún àwọn àlàyé ìsanwó kí ó tó tẹ̀síwájú.

## Àwọn Ohun Àmúṣọrọ Tí a Ṣe Àtìlẹ́yìn Fún {#supported-resources}

* Ìpamọ́ disk
* Bandwidth/Gbigbé data
* Iye àwọn site
* Iye àwọn aṣàmúlò
* Lílò token AI àti ináwó AI
* Àwọn irú ohun àmúṣọrọ àkànṣe

## Àwọn Ohun Tí a Nílò {#requirements}

* Multisite Ultimate 2.0.0 tàbí tó ga ju bẹ́ẹ̀ lọ
* Fífì WordPress Multisite sílẹ̀
* PHP 7.4 tàbí tó ga ju bẹ́ẹ̀ lọ

## Fífì Sílẹ̀ {#installation}

1. Gbe àwọn fáìlì plugin sí `/wp-content/plugins/ultimate-multisite-metered-plans`
2. Mú plugin náà ṣiṣẹ́ nípasẹ̀ iboju 'Plugins' nínú WordPress
3. Lọ sí Multisite Ultimate → Settings → Metered Plans
4. Ṣètò àwọn ohun àmúṣọrọ tí a nwọ̀n rẹ àti ìdíyelé
5. Mú ìsanwó gẹ́gẹ́ bí lílò ṣiṣẹ́ fún àwọn product nínú olootu product

## Àwọn Ìbéèrè Tí a Máa Ń Béèrè Lọ́pọ̀ Ìgbà {#frequently-asked-questions}

## Kí ló ṣẹlẹ̀ nígbà tí oníbàárà bá kọjá ààlà rẹ? {#what-happens-when-a-customer-exceeds-their-limit}

Dípò dídí ìwọlé dúró, ẹ̀rọ náà ń tọpa àfikún lílò náà, ó sì ń ṣàfikún un sí ìwé-ìdíyelé wọn tó ń bọ̀ dá lórí àwọn iye owó tí o ti ṣètò.

## Ṣé mo lè ṣètò àwọn iye owó àfikún lílò tó yàtọ̀ fún àwọn product tó yàtọ̀? {#can-i-set-different-overage-rates-for-different-products}

Bẹ́ẹ̀ ni, product kọ̀ọ̀kan lè ní àwọn ohun àmúṣọrọ tí a nwọ̀n àti ìdíyelé àfikún lílò tirẹ̀.

## Báwo ni a ṣe ń fi tó àwọn oníbàárà létí nípa lílò? {#how-are-customers-notified-about-usage}

Àwọn oníbàárà lè wo lílò wọn lọwọlọwọ nínú Dashboard wọn, wọ́n sì lè gba àwọn ìfitónilétí àṣàyàn nígbà tí wọ́n bá ń sún mọ́ tàbí kọjá àwọn ààlà.

## Báwo ni a ṣe ń díyelé lílò token AI? {#how-is-ai-token-usage-priced}

A ń díyelé lílò token AI láti inú àwọn iye owó ìwọlé àti ìjáde tí a ṣètò fún model fún token mílíọ̀nù 1. Àwọn Ètò Tí a Nwọ̀n ń sọ iye token prompt àti completion ìbéèrè kọ̀ọ̀kan di púpọ̀ pẹ̀lú àwọn iye owó wọ̀nyẹn, ó ń ṣàfikún èsì náà sí ináwó AI àkókò ìsanwó lọwọlọwọ ti oníbàárà, ó sì ń lo iye ààyè plan náà tàbí àwọn òfin àfikún lílò.

## Ṣé mo tún lè lo àwọn ààlà líle? {#can-i-still-use-hard-limits}

Bẹ́ẹ̀ ni, ìsanwó gẹ́gẹ́ bí lílò jẹ́ àṣàyàn fún product kọ̀ọ̀kan. O lè dapọ̀ àwọn product ààlà-líle ìbílẹ̀ mọ́ àwọn product tí a nwọ̀n.
