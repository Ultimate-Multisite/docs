# Ìwé àkọsílẹ̀ ìyípadà àwọn Plan Tí a Díwọ̀n

Ẹ̀yà 1.1.0 - Tí a tu jáde ní 2026-05-05
- Tuntun: ìsanwó AI token fún àwọn multisite subsite — tọ́pa kí o sì gba owó fún lílo AI token kọjá àwọn site oníbàárà pẹ̀lú àwọn iye owó fún token kọ̀ọ̀kan tí a lè ṣètò
- Tuntun: A tún àfipámú connector kọ́ pẹ̀lú ìṣàwárí ààlà alágbára àti write-through, tó ń rí i dájú pé ìpéye wà ní àkókò gidi kọjá gbogbo àwọn connector
- Àtúnṣe: Àwọn ìgbésókè tábìlì Database ti bá àwọn ìtumọ̀ schema BerlinDB mu dáadáa báyìí, tí ń dènà ìkùnà ìgbésókè lórí àwọn ìfìdímúlẹ̀ tuntun
- Àtúnṣe: A ti yí àwọn callbacks ìgbésókè Database padà sí fọ́ọ̀mù tó tọ́, tí ń yanjú àwọn ìkùnà ìgbésókè tí kò fi ìkìlọ̀ hàn
- Àtúnṣe: A ti ń gba àwọn iye aláìkún-odidi báyìí nínú pápá ìfọwọ́sí AI Usage Overage Markup
- Àtúnṣe: A ti yanjú àwọn àṣìṣe tó lèwu àti àwọn ìṣòro ìbẹ̀rẹ̀-lẹ́ẹ̀mejì nígbà tí plugin bá ń kojú
- Tí a mú dára síi: A fi script npm check-env kún un kí àwọn àyíká developer lè ṣètò ara wọn nígbà ìṣiṣẹ́ àkọ́kọ́
