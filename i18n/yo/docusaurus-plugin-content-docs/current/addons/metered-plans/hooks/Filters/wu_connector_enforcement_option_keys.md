# Àlẹmọ: wu_connector_enforcement_option_keys

Ṣe àlẹmọ àtòkọ àwọn option key ti olupèsè AI tí a fi múlẹ̀ láti ojúlé àkọ́kọ́.

Ṣàfikún àwọn option key fún àwọn plugin olupèsè AI àdáṣe tàbí ti ẹni-kẹta kí àwọn ètò wọn lè tún jogún láti ojúlé àkọ́kọ́ lórí àwọn ojúlé abẹ́.

## Àwọn Parameter

| Orúkọ | Iru | Àpèjúwe |
|------|------|-------------|
| $keys | `string[]` | Àwọn orúkọ option key láti fi múlẹ̀ (ó ti ní àwọn connector key tí a ṣàwárí ní ìmúdàgba àti EXTRA_PROVIDER_OPTIONS). |

### Láti ìgbà

- 1.2.0
