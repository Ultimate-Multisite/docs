# PayFast

Multisite Ultimate: Ẹnu-ọ̀nà Payfast

Ṣe àfikún Payfast - ẹnu-ọ̀nà ìsanwó ori ayélujára tó yara àti tó ní ààbò ní South Africa - gẹ́gẹ́ bí aṣayan ìsanwó Multisite Ultimate.

## Fífi sílẹ̀

1. Ṣàgbékalẹ̀ 'ultimate-multisite-payfast' sí inú itọ́kasí '/wp-content/plugins/'
2. Mú àfikún náà ṣiṣẹ́ nípasẹ̀ akojọ 'Plugins' nínú WordPress
3. Tẹ̀lé Wizard ìgbésẹ̀-ní-ìgbésẹ̀ láti ṣètò àfikún náà

## Ìtọ́nisọ́nà Ìlò

PayFast jẹ́ olùṣètò ìsanwó ori ayélujára tó gbajúmọ̀ fún àwọn ilé-iṣẹ́ ní South Africa. Ó yà sí mímú kí àwọn oníṣòwò ori ayélujára dàgbà nípa jíjẹ́ kí àwọn ìdúnàádúrà rọrùn àti ní ààbò. Gẹ́gẹ́ bí ìbéèrè olumulo Ultimate Multisite, a pinnu láti dá àfikún kan tí yóò jẹ́ kí ìṣàkóso pọ̀ mọ́ PayFast ṣeé ṣe. Àpilẹ̀kọ yìí yóò fi hàn bí a ṣe lè ṣètò PayFast gẹ́gẹ́ bí ẹnu-ọ̀nà ìsanwó rẹ láti gba ìsanwó nínú nẹ́tíwọ́ọ̀kì Ultimate Multisite rẹ.

Ìgbésẹ̀ àkọ́kọ́ ni láti fi àfikún náà sílẹ̀.

# Bí a ṣe lè fi àfikún Payfast sílẹ̀:

Lọ sí ojú-ewé àwọn àfikún kí o sì yan àfikún Ultimate Multisite: Payfast Gateway.

![Ojú-ewé àwọn àfikún](/img/admin/addons-page.png)

Fèrèsé àfikún náà yóò yọ jáde. Tẹ Install Now.

![Ojú-ewé àwọn àfikún pẹ̀lú aṣayan fífi sílẹ̀](/img/admin/addons-page.png)

Lẹ́yìn tí fífi sílẹ̀ bá ti parí, a ó darí rẹ padà sí ojú-ewé àwọn àfikún. Níhìn-ín, kan tẹ Network Activate, àfikún _Payfast_ náà yóò sì ṣiṣẹ́ lórí nẹ́tíwọ́ọ̀kì rẹ.

# Ṣíṣètò PayFast

Lẹ́yìn fífi àfikún náà sílẹ̀, o lè lọ sí àwọn ètò Ultimate Multisite báyìí kí o sì bẹ̀rẹ̀ sí í ṣàtúnṣe PayFast.

Lábẹ́ ojú-ewé **àwọn ètò Ultimate Multisite**, o lè rí taabu "Ìsanwó". Lábẹ́ taabu yìí, o yẹ kí o rí "**Payfast** " pẹ̀lú bọ́tìnì yíyí láti mú ẹnu-ọ̀nà ìsanwó yìí ṣiṣẹ́ tàbí láti pa á.

![Àwọn ètò ẹnu-ọ̀nà ìsanwó](/img/config/settings-payment-gateways.png)Tẹ̀síwájú kí o sì tẹ bọ́tìnì yíyí náà láti mú un ṣiṣẹ́. Lẹ́yìn náà, ó máa gbooro apótí náà láti fi àwọn ààyè ọ̀tọ̀ọ̀tọ̀ hàn tí o nílò láti kún, bí **ID Oníṣòwò, Kọ́kọ́rọ́ Oníṣòwò àti Gbólóhùn-ìgbaniwọlé.** O lè gba ìwífún yìí láti inú **Payfast** Account rẹ

<!-- Àwòrán iboju kò sí: àwọn ètò PayFast tí ó gbooro tó ń fi àwọn ààyè ID Oníṣòwò, Kọ́kọ́rọ́ Oníṣòwò, àti Gbólóhùn-ìgbaniwọlé hàn -->

Ìgbésẹ̀ tó kàn ni láti wọlé sí Account PayFast rẹ láti gba ìwífún tí o nílò.

Nígbà tí o bá wọlé, o lè rí àwọn ẹ̀rí ìfọwọ́sowọ́pọ̀ tí o nílò lẹ́sẹ̀kẹsẹ̀ láti inú dashboard Account **Payfast** rẹ.

Ó ti yá, ìṣàkóso ìsanwó PayFast rẹ ti pé!
