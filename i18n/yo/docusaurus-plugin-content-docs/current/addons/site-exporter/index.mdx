# Olùtajàde Ojúlé

## Àkóónú {#content}

A lóye àìní láti lè gbe àwọn ojúlé-kéékèèké nínú nẹ́tíwọ́ọ̀kì rẹ lọ sí fifi sori ojúlé kan ṣoṣo, àti ní òdìkejì, bí oníbàárà rẹ bá pinnu láti ṣí ojúlé wẹ́ẹ̀bù wọn lọ. Àfikún olùtajàde àti olùgbéwọlé ojúlé mú iṣẹ́ yìí rọrùn síi. Irinṣẹ́ yìí ń tajàde àti ń gbéwọlé àwọn ojúlé ní lílo àwọn fáìlì zip, èyí tó tún lè ṣiṣẹ́ gẹ́gẹ́ bí ìpamọ́ àfikún.

## Àwọn Àmúyẹ {#features}

* Tajàde àti Gbéwọlé Àwọn Ojúlé

* Ìbáṣepọ̀ tó rọrùn láti lò

* Ṣẹ̀dá ìpamọ́ àfikún ojúlé

## Ìwé Ìtọ́nisọ́nà {#documentation}
**Bí a ṣe ń Tajàde àti Gbéwọlé Ojúlé kan**

## Fífi sori ẹrọ {#installation}

1. Gbé **ultimate-multisite-site-exporter** sí àkójọpọ̀ **/wp-content/plugins/**
2. Mú àfikún náà ṣiṣẹ́ nípasẹ̀ àkójọ **Àwọn Àfikún** nínú WordPress
3. Tẹ̀lé Wizard ní ìgbésẹ̀-nípasẹ̀-ìgbésẹ̀ láti ṣètò àfikún náà

## Ìtọ́nisọ́nà Lílo {#usage-guide}

Pẹ̀lú Ultimate Multisite o lè tajàde ojúlé kan láti nẹ́tíwọ́ọ̀kì rẹ tàbí gbéwọlé ojúlé kan sí fifi sori multisite tàbí ojúlé-kan ṣoṣo rẹ.

Láti bẹ̀rẹ̀ ìlànà náà, lọ sí ojú-ewé àwọn àfikún wa kí o sì fi àfikún Ultimate Multisite: Olùtajàde & Olùgbéwọlé Ojúlé sori ẹrọ.![Ojú-ewé àwọn àfikún](/img/admin/addons-page.png)

![Ojú-ewé àwọn àfikún pẹ̀lú aṣàyàn fífi sori ẹrọ](/img/admin/addons-page.png)

Lọ sí ojú-ewé Àwọn Àfikún rẹ kí o tẹ láti **Mú ṣiṣẹ́ lórí Nẹ́tíwọ́ọ̀kì** àfikún tí o ṣẹ̀ṣẹ̀ fi sori ẹrọ náà.<!-- Àwòrán-iboju kò sí: Ojú-ewé Àwọn Àfikún WP tó ń fi bọ́tìnì Mú ṣiṣẹ́ lórí Nẹ́tíwọ́ọ̀kì hàn fún àfikún Olùtajàde Ojúlé -->

Aṣàyàn **Ìtajàde & Ìgbéwọlé** wà lórí àkójọ Ultimate Multisite, ní apá òsì.

## Bí a ṣe ń tajàde ojúlé kan {#how-to-export-a-site}

Láti tajàde ojúlé kan láti nẹ́tíwọ́ọ̀kì rẹ, lọ sí **Ìtajàde & Ìgbéwọlé** lórí àkójọ Ultimate Multisite àti, lórí taabu **Tajàde Àwọn Ojúlé**, tẹ láti **Ṣẹ̀dá Ìtajàde Ojúlé tuntun**.

<!-- Àwòrán-iboju kò sí: Taabu Tajàde Àwọn Ojúlé pẹ̀lú bọ́tìnì Ṣẹ̀dá Ìtajàde Ojúlé tuntun -->

Wá ojúlé tí o fẹ́ tajàde kí o sì yan àwọn ohun tí o fẹ́ fi sínú fáìlì tí a tajàde. O lè fi àwọn theme, àfikún, àti àwọn ìgbé-sókè kún un. Bí ojúlé rẹ bá tóbi jù, o lè yàn láti ṣiṣẹ́ ìlànà ìtajàde náà ní abẹ́lẹ̀.<!-- Àwòrán-iboju kò sí: Àwọn aṣàyàn ìtajàde ojúlé pẹ̀lú ìwádìí ojúlé, àwọn àpótí àyẹ̀wò láti fi àwọn theme/àfikún/ìgbé-sókè kún un -->

Tẹ bọ́tìnì **Tajàde Ojúlé**. Ojúlé tí o tajàde yóò wà lórí àtòkọ **Àwọn Ìtajàde Tó Wà Tẹ́lẹ̀**. Tẹ bọ́tìnì **Ṣe Ìgbàsílẹ̀ ZIP** láti gba fáìlì náà sílẹ̀. Àwọn ZIP ìtajàde ní `index.php` tó ń bẹ̀rẹ̀ fúnra rẹ àti `readme.txt`, nítorí náà a lè fi wọ́n sori host tuntun láì kọ́kọ́ fi àfikún olùgbéwọlé lọ́tọ̀ sori ẹrọ.<!-- Àwòrán-iboju kò sí: Àtòkọ Àwọn Ìtajàde Tó Wà Tẹ́lẹ̀ pẹ̀lú bọ́tìnì Ṣe Ìgbàsílẹ̀ ZIP -->

## Bí a ṣe ń tajàde gbogbo nẹ́tíwọ́ọ̀kì {#how-to-export-the-whole-network}

Láti tajàde gbogbo àwọn ojúlé-kéékèèké pọ̀, lọ sí **Ìtajàde & Ìgbéwọlé** lórí àkójọ Ultimate Multisite kí o sì lo aṣàyàn **Ìtajàde Nẹ́tíwọ́ọ̀kì** lórí iboju ìtajàde. Àkójọ-fáìlì tí a dá jáde ní gbogbo àwọn ojúlé-kéékèèké nínú ZIP kan ṣoṣo fún ìṣípadà host, àtúnkọ́ staging, tàbí àwọn ìpamọ́ àfikún ipele-nẹ́tíwọ́ọ̀kì.

Àwọn ìtajàde nẹ́tíwọ́ọ̀kì lè tóbi. Ṣiṣẹ́ wọn nígbà tí ìrìn-àjò kò pọ̀, kí o sì jẹ́rìí pé server ní ibi ìpamọ́ tó tó fún àkójọ-fáìlì náà àti àwọn ìgbé-sókè, àfikún, àti theme èyíkéyìí tí a fi kún un.

## Bí a ṣe ń gbéwọlé ojúlé kan {#how-to-import-a-site}

Láti gbéwọlé ojúlé kan, lọ sí taabu **Gbéwọlé Ojúlé kan** kí o sì gbé fáìlì .zip ti ojúlé tí o fẹ́ gbéwọlé sókè. O lè yí URL padà sí tuntun tàbí pa URL atijọ́ mọ́ nípa fífi ààyè náà sílẹ̀ ní òfo. Bákan náà, o lè ṣàyẹ̀wò àpótí náà láti pa fáìlì .zip rẹ́ kúrò nínú fifi sori ẹrọ rẹ lẹ́yìn ìgbéwọlé. Lẹ́yìn náà tẹ láti **Bẹ̀rẹ̀ Ìgbéwọlé**.

<!-- Àwòrán-iboju kò sí: Taabu Gbéwọlé Ojúlé kan pẹ̀lú pápá ìgbé-sókè zip àti bọ́tìnì Bẹ̀rẹ̀ Ìgbéwọlé -->

Ìlànà náà lè gba àkókò díẹ̀. Wàá rí ìgbéwọlé tó ń dúró nínú àtòkọ tó wà ní ìsàlẹ̀ olùgbéwọlé.

<!-- Àwòrán-iboju kò sí: Ìgbéwọlé tó ń dúró nínú àtòkọ ní ìsàlẹ̀ olùgbéwọlé -->

O lè dá ìlànà náà dúró nípa títẹ **Fagilé Ìgbéwọlé**.

<!-- Àwòrán-iboju kò sí: Bọ́tìnì Fagilé Ìgbéwọlé láti dá ìlànà ìgbéwọlé dúró -->

Lẹ́yìn tí ìlànà náà bá parí, ojúlé rẹ ti wà lórí nẹ́tíwọ́ọ̀kì rẹ.

## Bí a ṣe ń gbéwọlé ojúlé kan lórí fifi sori ẹrọ kan ṣoṣo {#how-to-import-a-site-on-a-single-install}

Láti gbéwọlé ojúlé kan sí fifi sori ẹrọ rẹ kan ṣoṣo, kọ́kọ́ fi olùgbéwọlé sori ẹrọ kí o sì mú un ṣiṣẹ́ lórí ojúlé rẹ. O lè rí ìjápọ̀ láti gba a sílẹ̀ lórí taabu olùtajàde.

<!-- Àwòrán-iboju kò sí: Taabu olùtajàde pẹ̀lú ìjápọ̀ láti gba olùgbéwọlé ojúlé-kan ṣoṣo sílẹ̀ -->

O lè rí olùgbéwọlé lórí àkójọ **Àwọn Irinṣẹ́**, ní apá òsì, lórí ohun-abẹ́ **Ìtajàde & Ìgbéwọlé**.

<!-- Àwòrán-iboju kò sí: Àkójọ Àwọn Irinṣẹ́ ojúlé-kan ṣoṣo tó ń fi àkójọ-abẹ́ Ìtajàde àti Ìgbéwọlé hàn -->

Gbé fáìlì .zip ti ojúlé tí o fẹ́ gbéwọlé sókè. O lè yí URL padà sí tuntun tàbí pa URL atijọ́ mọ́ nípa fífi ààyè náà sílẹ̀ ní òfo. Bákan náà, o lè ṣàyẹ̀wò àpótí náà láti pa fáìlì .zip rẹ́ kúrò nínú fifi sori ẹrọ rẹ lẹ́yìn ìgbéwọlé. Lẹ́yìn náà tẹ láti **Bẹ̀rẹ̀ Ìgbéwọlé**.

<!-- Àwòrán-iboju kò sí: Fọ́ọ̀mù ìgbéwọlé ojúlé-kan ṣoṣo pẹ̀lú ìgbé-sókè zip àti bọ́tìnì Bẹ̀rẹ̀ Ìgbéwọlé -->

Ìlànà náà lè gba àkókò díẹ̀. Wàá rí ìgbéwọlé tó ń dúró nínú àtòkọ tó wà ní ìsàlẹ̀ olùgbéwọlé.

<!-- Àwòrán-iboju kò sí: Ìgbéwọlé ojúlé-kan ṣoṣo tó ń dúró nínú àtòkọ -->

O lè dá ìlànà náà dúró nípa títẹ **Fagilé Ìgbéwọlé**.

<!-- Àwòrán-iboju kò sí: Bọ́tìnì Fagilé Ìgbéwọlé láti dá ìlànà ìgbéwọlé dúró -->

Lẹ́yìn tí ìlànà náà bá parí, ojúlé rẹ yóò rọ́pò pẹ̀lú ojúlé tí o ṣẹ̀ṣẹ̀ gbéwọlé.
