# Ìṣe: wu_page_added

Gba àwọn olùdàgbàsókè plugin láàyè láti ṣe àwọn ohun àfikún nígbà tí a bá forúkọsílẹ̀ àwọn oju-iwe.

Kò dàbí wu_page_load, tí ó máa ń ṣiṣẹ́ nikan nígbà tí a bá ń wo oju-iwe kan pàtó, hook yìí ń ṣiṣẹ́ nígbà ìforúkọsílẹ̀ fún gbogbo oju-iwe alábòójútó tí a ń ṣàfikún nípa lílo code WP Ultimo.

## Àwọn Paramita

| Orúkọ | Iru | Àpèjúwe |
|------|------|-------------|
| $page_id | `string\|\WP_Ultimo_Site_Exporter\Admin_Pages\Base_Admin_Page` | ID ti oju-iwe yìí. |
| $page_hook | `string\|\WP_Ultimo_Site_Exporter\Admin_Pages\Base_Admin_Page` | Hook tí a ṣe fún oju-iwe yìí. |
