# Fluent Forms

Àfikún yìí jẹ́ kí ó ṣeé ṣe láti fi àwọn ààlà kún Fluent Forms lórí Multisite Ultimate nínú àwọn ọjà, àwọn ìjẹ́mọ́ọ̀mọ́ tàbí àwọn sáìtì.
Fluent Forms máa ń lo àwọn tábìlì àdáni dípò àwọn irú ìfìwéránṣẹ́ àdáni, nítorí náà àwọn aṣàyàn ìdíwọ̀n deede kò ṣiṣẹ́ pẹ̀lú Fluent Forms.

## Àwọn ẹ̀yà ara

- Fi àwọn ààlà kún Fluent Forms tàbí Conversational Fluent Forms
- Àwọn oníbàárà máa gba ìfiránṣẹ́ onínúure tí ń rọ̀ wọ́n láti ṣe ìgbésókè plan wọn nígbà tí wọ́n bá dé ààlà náà
