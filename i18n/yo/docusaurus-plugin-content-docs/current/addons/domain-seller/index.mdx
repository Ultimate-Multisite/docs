# Domain Seller {#domain-seller}

**Multisite Ultimate: Domain Seller** yí nẹ́tíwọ́ọ̀kì WordPress Multisite rẹ padà sí pẹpẹ títà domain tó lágbára. Jẹ́ kí àwọn oníbàárà forúkọsílẹ̀ fún domain àdáni taara nígbà ìforúkọsílẹ̀ tàbí láti dashboard wọn, pẹ̀lú ṣíṣe àyẹ̀wò ìwàníní ní àkókò gidi àti ìdíyelé tó hàn gbangba.

### Kí nìdí Domain Seller? {#why-domain-seller}

Ìforúkọsílẹ̀ domain jẹ́ ọ̀kan lára àwọn àfikún títà tí a máa ń béèrè jù lọ fún pẹpẹ WaaS (Website as a Service) èyíkéyìí. Dípò fífi àwọn oníbàárà rẹ ránṣẹ́ sí olùforúkọsílẹ̀ ẹni-kẹta, o lè pèsè domain taara — kí ìwọ lè pa owó-wiwọlé púpọ̀ síi mọ́ nínú nẹ́tíwọ́ọ̀kì rẹ, nígbà tí o tún ń fún àwọn oníbàárà ní ìrírí tí kò ní ìdènà.

### Àwọn Ẹ̀yà Pàtàkì {#key-features}

#### Àtìlẹ́yìn fún Ọ̀pọ̀ Olùforúkọsílẹ̀ Domain {#multiple-domain-registrar-support}
* **OpenSRS** — Ìforúkọsílẹ̀, ìtúnse, àti ìṣàkóso kíkún
* **Namecheap** — Ìṣepọ̀ pípé pẹ̀lú WhoisGuard àti ìdánwò sandbox
* **HostAfrica** — Ìwádìí domain, ìdíyelé, ìforúkọsílẹ̀, ìgbéwọlé, DNS, àti ààbò ID nípasẹ̀ Domains Reseller API
* **Openprovider** — Ìdíyelé reseller, ìforúkọsílẹ̀, ìtúnse, ìgbéwọlé, DNS, aṣírí WHOIS, àti ìmúdọ́gba TLD
* **Hostinger** — Ìwádìí domain àti ìforúkọsílẹ̀ nípa lílo token ìṣepọ̀ Hostinger tí a pín
* **Àkọ́lé-ìṣètò tó ṣeé faagun** — A kọ́ ọ lórí àpẹẹrẹ Integration Registry, tó mú kí ó rọrùn láti ṣàfikún àwọn olupèsè míì
* **Ìdánwò Ìsopọ̀ Tí a Kọ́ Sínú rẹ** — Ṣàyẹ̀wò ẹ̀rí olupèsè àti ìsopọ̀ kí o tó lọ sí ìmúlò gidi

#### Wizard Ìṣètò Pẹ̀lú Ìtọ́sọ́nà {#guided-setup-wizard}
* **Ìbẹ̀rẹ̀-lílò pẹ̀lú títẹ ẹ̀kan** — Ó máa tọ́ ọ kọjá ìṣètò olupèsè, ìgbéwọlé TLD, àti dídá ọjà domain
* **Ìmúdọ́gba TLD aládàáṣiṣẹ́** — Ó ń gbe gbogbo TLD tó wà àti ìdíyelé wholesale wọlé láti ọ̀dọ̀ àwọn olupèsè tí o so pọ̀
* **Dídá ọjà aiyipada** — Ó dá ọjà domain tó ti ṣetán fún lílò pẹ̀lú àwọn aiyipada markup tó bójú mu

#### Ìdíyelé Domain Tó Rọrùn {#flexible-domain-pricing}
* **Àwọn ipo markup mẹ́ta** — Markup ìdá-ogorun, markup tó dúró, tàbí iye tó dúró fún TLD kọ̀ọ̀kan
* **Àwọn ọjà tó dá lórí TLD** — Dá àwọn ọjà domain ọ̀tọ̀ fún àwọn ẹgbẹ́ TLD ọ̀tọ̀ (.com/.net, .io, kóòdù orílẹ̀-èdè, àti bẹ́ẹ̀ bẹ́ẹ̀ lọ)
* **Ìdíyelé ìbẹ̀rẹ̀** — Pèsè iye ọdún àkọ́kọ́ tí a dínkù láti fa àwọn ìforúkọsílẹ̀ tuntun
* **Ìdíyelé àkókò gidi** — Ìdíyelé wholesale láàyè láti ọ̀dọ̀ àwọn olupèsè pẹ̀lú caching olóye
* **Àtìlẹ́yìn coupon** — Ìbámu kíkún pẹ̀lú àwọn kóòdù ẹ̀dinwó Multisite Ultimate

#### Ààbò Aṣírí WHOIS {#whois-privacy-protection}
* **Ìṣètò fún ọjà kọ̀ọ̀kan** — Ṣètò aṣírí WHOIS sí Disabled, Always Included, tàbí Customer Choice
* **Ìwọlé-oníbàárà** — Nígbà tí a bá ṣètò sí Customer Choice, checkbox checkout yóò jẹ́ kí àwọn oníbàárà ṣàfikún ààbò aṣírí
* **Ìdíyelé tó ṣeé ṣètò** — Ṣètò iye ọdún kan fún àfikún aṣírí náà, tàbí pèsè rẹ lọ́fẹ̀ẹ́
* **Ìṣepọ̀ olùforúkọsílẹ̀** — Ó ń mu WhoisGuard (Namecheap) tàbí aṣírí WHOIS (OpenSRS) ṣiṣẹ́ laifọwọyi nígbà ìforúkọsílẹ̀

#### Ìrírí Checkout Tó Dàpọ̀ {#unified-checkout-experience}
* **Àwọn ipo domain mẹ́ta** — Àwọn oníbàárà yàn láàárín subdomain (ọfẹ), fíforúkọsílẹ̀ domain tuntun, tàbí mapping domain tó wà tẹ́lẹ̀ tí wọ́n ní
* **Ìwádìí domain àkókò gidi** — Ṣíṣe àyẹ̀wò ìwàníní tí AJAX ń ṣiṣẹ́, láì tún ojúewé kó
* **Àwọn àbá ọlọ́gbọ́n** — Àwọn àbá TLD míì nígbà tí domain tí a fẹ́ bá ti gba
* **Ìdíyelé tó hàn gbangba** — Ìfihàn kedere ti iye ìforúkọsílẹ̀, iye ìtúnse, àti owó aṣírí WHOIS tí ó jẹ́ aṣàyàn
* **Àwọn pápá ìkànìyàn registrant** — Ó ń gba ìwífún registrant tó yẹ (orúkọ, àdírẹ́sì, fóònù) nítaara nígbà checkout
* **URL site tí a dá laifọwọyi** — Àdírẹ́sì site ni a máa yọ laifọwọyi láti domain tí a yàn

#### Eto Ìtúnse-Aládàáṣiṣẹ́ {#auto-renewal-system}
* **Àwọn ìtúnse tí a so mọ́ membership** — Àwọn domain máa ń túnṣe laifọwọyi nígbà tí membership oníbàárà bá ṣi ń ṣiṣẹ́
* **Ìṣètò àkókò ọlọ́gbọ́n** — A máa to àwọn iṣẹ́ ìtúnse sílẹ̀ dá lórí àwọn ọjọ́ ìparí domain
* **Ìmọ̀ràn ìtun-gbìyànjú** — Àwọn ìtun-gbìyànjú ọlọ́gbọ́n fún àwọn ìgbìyànjú ìtúnse tó kùnà
* **Àwọn ìfitónilétí email** — Àwọn ìkìlọ̀ tó ṣeé ṣètò fún àwọn ìtúnse tó ń bọ̀, àwọn ìtúnse tó ṣàṣeyọrí, ìkùnà, àti ìparí

#### Ìṣàkóso DNS fún Oníbàárà {#customer-dns-management}
* **DNS iṣẹ́-ara-ẹni** — Àwọn oníbàárà lè ṣàfikún, ṣàtúnṣe, àti pa àwọn record DNS (A, AAAA, CNAME, MX, TXT) rẹ́ láti dashboard wọn
* **Ipò domain** — Ipò ìforúkọsílẹ̀ àkókò gidi, àwọn ọjọ́ ìparí, àti ìwífún nameserver
* **Mapping domain aládàáṣiṣẹ́** — Àwọn domain tí a forúkọsílẹ̀ tàbí tó wà tẹ́lẹ̀ ni a máa map laifọwọyi sí site oníbàárà

#### Àwọn Irinṣẹ́ Admin {#admin-tools}
* **Ìforúkọsílẹ̀ domain lọ́wọ́** — Forúkọsílẹ̀ àwọn domain dípò àwọn oníbàárà láti admin panel
* **Ìbojútó domain** — Wo gbogbo àwọn domain oníbàárà, àwọn ipò, àti àwọn ọjọ́ ìparí láti ojúewé kan
* **Logging tó dá lórí olupèsè** — Àwọn channel log ọ̀tọ̀ fún olùforúkọsílẹ̀ kọ̀ọ̀kan (fún àpẹẹrẹ, domain-seller-namecheap.log)
* **Àwọn ẹ̀kúnrẹ́rẹ́ ìforúkọsílẹ̀** — Wo ìwífún registrant àti data ìforúkọsílẹ̀ lórí ojúewé àtúnṣe domain pataki

### Àwọn Olupèsè Tí a Ṣe Àtìlẹ́yìn fún {#supported-providers}

* **OpenSRS** — Ìṣepọ̀ kíkún pẹ̀lú ìforúkọsílẹ̀, ìtúnse, ìṣàkóso DNS, àti aṣírí WHOIS
* **Namecheap** — Ìṣepọ̀ kíkún pẹ̀lú ìforúkọsílẹ̀, ìtúnse, ṣíṣe àyẹ̀wò ìwàníní lọ́pọ̀, àti WhoisGuard ọfẹ
* **HostAfrica** — Ìforúkọsílẹ̀, ìtúnse, ìgbéwọlé, ìṣàkóso DNS, gbígba kóòdù EPP, registrar lock, àti ààbò ID
* **Openprovider** — Ìforúkọsílẹ̀, ìtúnse, ìgbéwọlé, ìṣàkóso DNS, gbígba kóòdù EPP, registrar lock, aṣírí WHOIS, àti ìmúdọ́gba TLD reseller
* **Hostinger** — Ìforúkọsílẹ̀, wíwá domain, àwọn ìmúdójúìwọ̀n nameserver, registrar lock, àti aṣírí WHOIS; Hostinger auto-renew ló ń bójú tó àwọn ìtúnse
* **GoDaddy** — Ìforúkọsílẹ̀ àti ìtúnse
* **ResellerClub** — Ìforúkọsílẹ̀, ìtúnse, àti ìṣàkóso DNS
* **NameSilo** — Ìforúkọsílẹ̀ àti ìtúnse
* **Enom** — Ìforúkọsílẹ̀, ìtúnse, àti ìṣàkóso DNS

### Àwọn Ẹ̀kúnrẹ́rẹ́ Imọ̀-Ẹ̀rọ {#technical-details}

* **Ń béèrè:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **Àkọ́lé-ìṣètò:** Àpẹẹrẹ Integration Registry pẹ̀lú Domain_Selling_Capability interface
* **Autoloading:** Jetpack autoloader pẹ̀lú PSR-4 class mapping
* **Àwọn ìlànà:** Àwọn ìlànà coding WordPress, strict typing, prepared statements

## Fífi Sórí Ẹ̀rọ {#installation}

1. Rí i dájú pé **Multisite Ultimate v2.4.12+** ti fi síi tí a sì ti network-activate rẹ
2. Gbe plugin náà sí `/wp-content/plugins/ultimate-multisite-domain-seller/`
3. Network-activate plugin náà láti **Àwọn Plugin** nínú WordPress Network Admin
4. **Wizard Ìṣètò** yóò bẹ̀rẹ̀ laifọwọyi ní ìṣiṣẹ́ àkọ́kọ́ — tẹ̀lé àwọn ìgbésẹ̀ láti:
   * So olùforúkọsílẹ̀ domain rẹ pọ̀
   * Dán ìsopọ̀ náà wò
   * Gbe àwọn TLD tó wà àti ìdíyelé wholesale wọlé
   * Dá ọjà domain àkọ́kọ́ rẹ pẹ̀lú markup ìdíyelé
5. Ṣàfikún pápá **Yíyan Domain** sí fọ́ọ̀mù checkout rẹ nínú **Ultimate Multisite > Checkout Forms**

## Àwọn Ìbéèrè Tí a Máa Ń Béèrè Lọ́pọ̀ {#frequently-asked-questions}

### Ẹ̀yà Multisite Ultimate wo ni a nílò? {#what-version-of-multisite-ultimate-is-required}

Ẹ̀yà 2.4.12 tàbí tó ga ju bẹ́ẹ̀ lọ. Addon náà kò bá WP Ultimo v1.x tàbí àwọn ẹ̀yà Ultimate Multisite tó wà ní ìsàlẹ̀ 2.4.12 ṣiṣẹ́.

### Àwọn olupèsè domain wo ni a ṣe àtìlẹ́yìn fún? {#which-domain-providers-are-supported}

Olùforúkọsílẹ̀ mẹ́sàn-án ni a ṣe àtìlẹ́yìn fún: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo**, àti **Enom**. Àwọn agbára yàtọ̀ síra nípa olupèsè nítorí registrar APIs ń fi àwọn iṣẹ́ ọ̀tọ̀ hàn; wo ìtọ́sọ́nà ìṣètò fún àkótán olupèsè-kọ̀ọ̀kan. Àkọ́lé-ìṣètò Integration Registry addon náà mú kí ó rọrùn láti ṣàfikún àwọn olupèsè àdáni — wo [ìtọ́sọ́nà olùgbéejáde](/developer/integration-guide/custom-registrar).

### Ṣé mo lè pèsè ààbò aṣírí WHOIS? {#can-i-offer-whois-privacy-protection}

Bẹ́ẹ̀ ni. Ọjà domain kọ̀ọ̀kan ní ètò Aṣírí WHOIS pẹ̀lú ipo mẹ́ta: Disabled, Always Included, tàbí Customer Choice. Nígbà tí a bá ṣètò sí Customer Choice, àwọn oníbàárà máa rí checkbox nígbà checkout pẹ̀lú iye náà (tàbí "Ọfẹ" fún WhoisGuard ti Namecheap). Àmì aṣírí náà ni a máa ránṣẹ́ laifọwọyi sí olùforúkọsílẹ̀ nígbà ìforúkọsílẹ̀.

### Ṣé mo lè ṣètò ìdíyelé ọ̀tọ̀ fún àwọn TLD ọ̀tọ̀? {#can-i-set-different-pricing-for-different-tlds}

Bẹ́ẹ̀ ni. Dá àwọn ọjà domain ọ̀tọ̀ fún àwọn ẹgbẹ́ TLD ọ̀tọ̀. Ọjà kọ̀ọ̀kan lè ní irú markup tirẹ̀ (ìdá-ogorun, markup tó dúró, tàbí iye tó dúró), àwọn TLD tí a ṣe àtìlẹ́yìn fún, àti ìdíyelé ìbẹ̀rẹ̀.

### Àwọn aṣàyàn checkout wo ni àwọn oníbàárà máa rí? {#what-checkout-options-do-customers-see}

Pápá Yíyan Domain ń fún àwọn oníbàárà ní tab mẹ́ta: **Subdomain** (lo subdomain ọfẹ lórí nẹ́tíwọ́ọ̀kì rẹ), **Forúkọsílẹ̀ Domain Tuntun** (wá kí o sì forúkọsílẹ̀ domain tuntun), àti **Domain Tó Wà Tẹ́lẹ̀** (map domain kan tí wọ́n ti ní tẹ́lẹ̀). Àwọn ipo tí o bá muu ṣiṣẹ́ nìkan ni yóò farahàn.

### Ṣé ó ṣe àtìlẹ́yìn fún àwọn ìtúnse aládàáṣiṣẹ́? {#does-it-support-automatic-renewals}

Bẹ́ẹ̀ ni. A lè ṣètò àwọn domain sí auto-renew, wọ́n sì ní ìbáṣepọ̀ pẹ̀lú ipò membership oníbàárà. A máa ṣiṣẹ́ lórí àwọn ìsanwó ìtúnse nípasẹ̀ payment gateway tí oníbàárà ní tẹ́lẹ̀.

### Ṣé àwọn oníbàárà lè ṣàkóso àwọn record DNS wọn? {#can-customers-manage-their-dns-records}

Bẹ́ẹ̀ ni. Àwọn oníbàárà gba interface ìṣàkóso DNS nínú dashboard wọn láti ṣàfikún, ṣàtúnṣe, àti pa àwọn record A, AAAA, CNAME, MX, àti TXT rẹ́. Ìṣàkóso DNS wà pẹ̀lú OpenSRS, ResellerClub, Enom, HostAfrica, àti Openprovider. DNS Hostinger fún àwọn domain tí a gbalejo ni ìṣepọ̀ domain-mapping Hostinger pataki ń bójú tó.

### Ṣé wizard ìṣètò wà? {#is-there-a-setup-wizard}

Bẹ́ẹ̀ ni. Wizard ìṣètò máa ń ṣiṣẹ́ laifọwọyi ní ìṣiṣẹ́ àkọ́kọ́, ó sì máa tọ́ ọ kọjá sisopọ̀ olùforúkọsílẹ̀, gbígbé àwọn TLD wọlé, àti dídá ọjà domain àkọ́kọ́ rẹ. O tún lè tún wizard náà ṣiṣẹ́ láti àwọn ètò addon.
