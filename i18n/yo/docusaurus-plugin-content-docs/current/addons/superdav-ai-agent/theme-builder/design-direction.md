# Ìtọ́sọ́nà Àpẹrẹ {#design-direction}

Ìgbésẹ̀ **Ìtọ́sọ́nà Àpẹrẹ** jẹ́ kí o ṣàgbéyẹ̀wò, túnṣe, kí o sì parí àpẹrẹ ojú-ìwòye theme rẹ kí Theme Builder tó dá theme pípé náà jáde.

## Àkótán {#overview}

Lẹ́yìn tí o bá parí Ìfọ̀rọ̀wánilẹ́nuwò Ìṣàwárí, Theme Builder máa fi ìtọ́sọ́nà àpẹrẹ rẹ hàn pẹ̀lú **ìṣàfihàn desktop àti mobile**. Èyí jẹ́ kí o rí gan-an bí àpẹrẹ rẹ yóò ṣe rí lórí àwọn ẹ̀rọ oriṣiriṣi kí o tó fi ara mọ́ theme náà.

## Àwọn Ẹ̀yà Ìtọ́sọ́nà Àpẹrẹ {#design-direction-components}

### 1. Àkójọpọ̀ Àwọ̀ {#1-color-palette}

Ètò àwọ̀ theme rẹ, pẹ̀lú:

- **Àwọ̀ àkọ́kọ́** — àwọ̀ brand pàtàkì tí a ń lò fún àwọn bọ́tìnì, àwọn ọna asopọ, àti àwọn àmì ìtẹnumọ́
- **Àwọ̀ kejì** — àwọ̀ àfikún fún onírúurú
- **Àwọ̀ ìtẹnumọ́** — àwọ̀ ìfihàn fún àwọn èròjà pàtàkì
- **Àwọn àwọ̀ neutral** — grẹ́yì àti funfun fún abẹ́lẹ̀ àti ọ̀rọ̀
- **Àwọ̀ ọ̀rọ̀** — àwọ̀ ọ̀rọ̀ àkọ́kọ́ fún ìrọ̀rùn kíkà

### 2. Typography {#2-typography}

Àwọn yíyan font fún:

- **Font akọlé** — tí a ń lò fún àwọn akọlé ojú-ewé àti akọlé apá
- **Font ara** — tí a ń lò fún ọ̀rọ̀ paragirafi àti àkóónú
- **Ìwọn font** — ìwọn tó ń dahùn sí ìwọn iboju oriṣiriṣi
- **Gíga laini** — ààyè láàárín àwọn laini fún ìrọ̀rùn kíkà
- **Ìwuwo font** — àwọn ìyàtọ̀ bold, regular, àti light

### 3. Ìfilélẹ̀ àti Ààyè {#3-layout-and-spacing}

- **Ìwọ̀n container** — ìwọ̀n tó pọ̀ jù fún àkóónú
- **Padding àti margins** — ààyè yí àwọn èròjà ká
- **Ètò grid** — ìfilélẹ̀ column fún àpẹrẹ tó ń dahùn sí iboju
- **Ààyè component** — ìjìnnà láàárín àwọn èròjà UI

### 4. Àwọn Èròjà Ojú-ìwòye {#4-visual-elements}

- **Àwọn ara bọ́tìnì** — àwọn àpẹrẹ bọ́tìnì àkọ́kọ́, kejì, àti kẹta
- **Àwọn àpẹrẹ kaadi** — àwọn ìfilélẹ̀ fún àwọn bulọọki àkóónú
- **Àwọn icon** — ara icon àti ìwọn rẹ
- **Àwọn aàlà àti òjìji** — ìjìnlẹ̀ ojú-ìwòye kékeré
- **Àwọn ipa hover** — ṣíṣe ara ipo ìbáṣepọ̀

## Ìṣàfihàn Preview {#preview-rendering}

### Preview Desktop {#desktop-preview}

Preview desktop fi àpẹrẹ rẹ hàn ní ìwọ̀n kíkún (ní ọ̀pọ̀ ìgbà 1200px tàbí tó gbòòrò ju bẹ́ẹ̀ lọ):

- Menu ìtọ́sọ́nà kíkún
- Ìfilélẹ̀ pípé pẹ̀lú gbogbo columns
- Typography àti ààyè ńlá
- Gbogbo àwọn èròjà ojú-ìwòye ní ìwọn kíkún

### Preview Mobile {#mobile-preview}

Preview mobile fi àpẹrẹ rẹ hàn ní ìwọ̀n mobile (ní ọ̀pọ̀ ìgbà 375px):

- Ìtọ́sọ́nà tó ń dahùn sí iboju (hamburger menu)
- Ìfilélẹ̀ column kan ṣoṣo
- Typography àti ààyè tí a túnṣe
- Ìwọn bọ́tìnì tó rọrùn fún fífi ọwọ́ kan

### Àwọn Breakpoint Tó Ń Dahùn sí Iboju {#responsive-breakpoints}

Theme Builder ń ṣe ìfihàn previews ní àwọn breakpoint wọ̀nyí:

| Ẹ̀rọ | Ìwọ̀n | Preview |
|--------|-------|---------|
| **Mobile** | 375px | Fóònù portrait |
| **Tablet** | 768px | Tablet landscape |
| **Desktop** | 1200px | Desktop ìwọ̀n-kíkún |
| **Large Desktop** | 1920px | Ìfihàn tó gbòòrò gan-an |

## Títún Àpẹrẹ Rẹ Ṣe {#refining-your-design}

### Ṣíṣe Àwọn Àtúnṣe {#making-adjustments}

O lè tún ìtọ́sọ́nà àpẹrẹ rẹ ṣe nípa:

1. **Ṣíṣe àtúnṣe àwọn àwọ̀** — yí àwọ̀ èyíkéyìí nínú àkójọpọ̀ àwọ̀ padà
2. **Yíyí fonts padà** — yan àwọn typefaces míì
3. **Ṣíṣe àtúnṣe ààyè** — tún padding àti margins ṣe
4. **Ṣíṣe àfikún ìfilélẹ̀** — yí ìwọ̀n container àti àwọn grid columns padà
5. **Ṣíṣe àwọn èròjà ní ti ara ẹni** — tún àwọn ara bọ́tìnì, àwọn àpẹrẹ kaadi, àti bẹ́ẹ̀ bẹ́ẹ̀ lọ ṣe.

### Àwọn Ìmúdójúìwọ̀n Preview {#preview-updates}

Àwọn ìyípadà máa hàn ní àkókò gidi:

- Preview desktop máa tún ara rẹ ṣe lẹ́sẹ̀kẹsẹ̀
- Preview mobile máa tún ara rẹ ṣe lẹ́sẹ̀kẹsẹ̀
- Gbogbo àwọn breakpoint tó ń dahùn sí iboju ni a máa túnṣe
- O lè yí padà láàárín previews láti jẹ́risi àwọn ìyípadà

### Ìwòye Ìfiwé {#comparison-view}

Fi ìtọ́sọ́nà àpẹrẹ rẹ wé:

- **Àpẹrẹ ìbẹ̀rẹ̀** — rí ohun tí ó yí padà
- **Àwọn àpẹrẹ olùdíje** — fi wé àwọn site ìmísí
- **Àwọn ẹ̀dà ṣáájú** — padà sí àwọn ìtọ́sọ́nà àpẹrẹ àtijọ́

## Ìfọwọ́sí Ìtọ́sọ́nà Àpẹrẹ {#design-direction-approval}

Nígbà tí o bá ní ìtẹ́lọ́run pẹ̀lú àpẹrẹ rẹ:

1. **Ṣàgbéyẹ̀wò previews méjèèjì** — jẹ́risi pé desktop àti mobile dáa
2. **Ṣàyẹ̀wò gbogbo àwọn àwọ̀** — rí i dájú pé contrast àti accessibility wà
3. **Dán typography wò** — jẹ́risi ìrọ̀rùn kíkà ní gbogbo ìwọn
4. **Jẹ́risi ìfilélẹ̀** — ṣàyẹ̀wò ààyè àti ìtòsí
5. **Fọwọ́sí àpẹrẹ** — tẹ̀síwájú sí theme generation

## Àwọn Ìgbésẹ̀ Tó Kàn {#next-steps}

Lẹ́yìn fífi ìtọ́sọ́nà àpẹrẹ rẹ sílẹ̀:

1. Theme Builder máa dá theme pípé rẹ jáde
2. A máa fi theme náà sí WordPress site rẹ
3. O lè tún un ṣe síi ní lílo WordPress customizer
4. Tẹ̀síwájú sí [Àwọn Menu Hospitality](./hospitality-menus.md) tàbí àwọn feature míì

## Àwọn Ìlànà Tó Dára Jù {#best-practices}

- **Dán an wò lórí àwọn ẹ̀rọ gidi** — lo àwọn fóònù àti tablet gidi bí ó bá ṣeé ṣe
- **Ṣàyẹ̀wò ìrọ̀rùn kíkà** — rí i dájú pé ọ̀rọ̀ ṣeé kà ní gbogbo ìwọn
- **Jẹ́risi contrast** — lo [Jẹ́risi Contrast Àkójọpọ̀ Àwọ̀](../abilities/validate-palette-contrast.md) fún accessibility
- **Ronú nípa iṣẹ́ṣe** — ṣe optimization àwọn àwòrán àti fonts fún iyára
- **Ṣètò fún àkóónú** — rí i dájú pé ìfilélẹ̀ ń ṣiṣẹ́ pẹ̀lú àkóónú gidi rẹ

## Ìṣòro àti Ìtúnṣe {#troubleshooting}

### Preview Kò Ṣe Ìmúdójúìwọ̀n {#preview-not-updating}

- Refresh ojú-ewé náà
- Pa cache browser rẹ rẹ́
- Gbìyànjú browser míì
- Ṣàyẹ̀wò asopọ internet

### Àwọn Àwọ̀ Dà Bí Ẹni Pé Wọ́n Yàtọ̀ {#colors-look-different}

- Ṣàyẹ̀wò àwọn color settings monitor
- Gbìyànjú lórí àwọn ẹ̀rọ oriṣiriṣi
- Jẹ́risi color contrast pẹ̀lú àwọn irinṣẹ́ accessibility
- Ronú nípa àwọn simulators color blindness

### Àwọn Ìṣòro Typography {#typography-issues}

- Jẹ́risi pé àwọn faili font ń kojọpọ̀
- Ṣàyẹ̀wò ìwọn font ní àwọn breakpoint oriṣiriṣi
- Dán an wò pẹ̀lú àkóónú gidi
- Ronú nípa gígùn laini fún ìrọ̀rùn kíkà
