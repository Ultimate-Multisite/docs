# Àwọn Irinṣẹ́ àti Ìṣètò

Theme Builder ń lo àkójọpọ̀ àwọn irinṣẹ́ láti ṣe àyẹ̀wò, ṣe àpẹrẹ, àti kọ́ ojúlé WordPress rẹ. Ní ẹ̀yà 1.16.0, **sd-ai-agent/site-scrape ti di irinṣẹ́ Tier 1 báyìí**, ó sì wà fún lílò ní aiyipada nínú Theme Builder. Superdav AI Agent v1.18.0 tún mú àkójọpọ̀ alábàákẹ́gbẹ́ Advanced kan wá tí a pín ká lọ́tọ̀ fún ìṣàn iṣẹ́ olùgbéejáde tí a fọkàn tán; àwọn irinṣẹ́ wọ̀nyẹn kì í ṣe apá kan àkójọpọ̀ pàtàkì WordPress.org, wọ́n sì gbọ́dọ̀ fi sílẹ̀ àti fún ní àṣẹ lọ́tọ̀.

## Àwọn Irinṣẹ́ Tí Ó Wà

### Àwọn Irinṣẹ́ Tier 1 (Wà Nígbà Gbogbo)

Àwọn irinṣẹ́ Tier 1 wà ní aiyipada nínú Theme Builder láìsí ìṣètò àfikún kankan.

#### sd-ai-agent/site-scrape

**Ète:** Ṣe àyẹ̀wò àwọn wẹ́ẹ̀bùsáìtì tó wà tẹ́lẹ̀ láti yọ ìmísí àpẹrẹ, ìṣètò àkóónú, àti iṣẹ́ jáde.

**Àwọn Agbára:**

- **Àyẹ̀wò wẹ́ẹ̀bùsáìtì** — fa àti ṣe àyẹ̀wò wẹ́ẹ̀bùsáìtì olùdíje tàbí ti ìmísí
- **Yíyọ àpẹrẹ jáde** — dá àwọn àwọ̀, fọ́ńtì, àti àwọn àpẹẹrẹ ìtòlẹ́sẹẹsẹ mọ̀
- **Ìṣètò àkóónú** — lóye ìṣètò ojúewé àti ìpele-ńlá-sí-kékeré
- **Ìdámọ̀ àwọn àbùdá** — dá àwọn plugin àti iṣẹ́ mọ̀
- **Àyẹ̀wò iṣẹ́ ṣiṣe** — ṣàyẹ̀wò iyára ojúewé àti ìmúdára
- **Àyẹ̀wò SEO** — ṣàgbéyẹ̀wò àwọn meta tag àti data tí a ṣètò

**Lílò:**

```
Ṣe àyẹ̀wò àpẹrẹ example.com kí o sì lò ó gẹ́gẹ́ bí ìmísí fún ojúlé mi.
```

**Ohun Tí Ó Máa Ń Yọ Jáde:**

- Àkójọpọ̀ àwọ̀ àti lílo àwọ̀
- Typography (àwọn fọ́ńtì àti ìwọ̀n)
- Ìṣètò ìtòlẹ́sẹẹsẹ àti ààlà àárín
- Àwọn àpẹẹrẹ ìtọ́sọ́nà
- Ìṣètò àkóónú
- Lílo àwọn àwòrán àti media
- Àwọn èròjà ìbáṣepọ̀
- Ìdáhùn tó bá alágbèéká mu

**Àwọn Ààlà:**

- Kò lè fa àwọn ojúlé tí ọ̀rọ̀ aṣínà dáàbò bò
- Ó bọ̀wọ̀ fún robots.txt àti àwọn ìlànà ojúlé
- Ó lè má gba àkóónú dainamiki
- Àwọn ojúlé tí JavaScript pọ̀ lè ní àyẹ̀wò tó lopin
- Àwọn ojúlé ńlá lè gba àkókò pípẹ́ síi láti ṣe àyẹ̀wò

### Àwọn Irinṣẹ́ Tier 2 (Yíyàn)

Àwọn irinṣẹ́ Tier 2 wà fún lílò nígbà tí a bá mú wọn ṣiṣẹ́ ní kedere nínú àwọn ìṣètò Theme Builder.

#### Ìtúpalẹ̀ Onígàárí

Ó pèsè àwọn ìwọ̀n iṣẹ́ ṣiṣe pẹ̀lú àlàyé:

- Àwọn àkókò ìrùsókè ojúewé
- Core Web Vitals
- Ìdíwọ̀n SEO
- Ìdíwọ̀n ìráyè-wọlé
- Ìdíwọ̀n àwọn ìlànà tó dára jù

#### Olùmúdára Àkóónú

Ó ń ṣe àyẹ̀wò, ó sì ń dábàá àwọn ìmúgbòrò fún:

- Ìrọ̀rùn kíkà
- Ìmúdára SEO
- Lílo ọ̀rọ̀ pàtàkì
- Ìṣètò àkóónú
- Ìgbékalẹ̀ ìpè-sí-ìgbésẹ̀

### Àwọn Irinṣẹ́ Tier 3 (Premium)

Àwọn irinṣẹ́ Tier 3 nílò ìṣètò àfikún tàbí àwọn àbùdá premium.

#### Olùdá Àkóónú AI

Ó ń dá àkóónú fún:

- Àwọn àpèjúwe ọjà
- Àwọn ojúewé iṣẹ́
- Àwọn àpilẹ̀kọ blog
- Àwọn àpèjúwe meta
- Ẹ̀dà ìpè-sí-ìgbésẹ̀

#### Àwọn Irinṣẹ́ Àpẹrẹ Onígàárí

Ó pèsè àwọn agbára àpẹrẹ onígàárí:

- Ṣíṣe CSS àdáni
- Ṣíṣe animation
- Àpẹrẹ èròjà ìbáṣepọ̀
- Ẹ̀kọ́ àwọ̀ onígàárí
- Ìmúdára typography

### Àwọn Irinṣẹ́ Alábàákẹ́gbẹ́ Advanced (Àkójọpọ̀ Lọ́tọ̀)

A pín àkójọpọ̀ alábàákẹ́gbẹ́ Advanced ká lọ́tọ̀ sí ìtúsílẹ̀ pàtàkì Superdav AI Agent. A pinnu rẹ̀ fún àwọn àyíká olùgbéejáde àti onílé-ojúlé tí a fọkàn tán níbi tí àwọn alábòójútó ti jẹ́ kí agent lo irinṣẹ́ tó ní ipa gíga ní kedere.

Àwọn agbára alábàákẹ́gbẹ́ Advanced lè ní:

- **Àwọn irinṣẹ́ filesystem** — ṣàyẹ̀wò àti ṣàtúnṣe àwọn fáìlì tí a fọwọ́ sí nínú àwọn àyíká tí a fọkàn tán
- **Àwọn irinṣẹ́ database** — béèrè tàbí ṣe àfikún data ojúlé nígbà tí a bá fún ní àṣẹ ní kedere
- **Àwọn irinṣẹ́ WP-CLI** — ṣiṣẹ́ àwọn àṣẹ ìtọju àti ìyẹ̀wò WordPress
- **Àwọn irinṣẹ́ REST dispatcher** — pe àwọn endpoint REST tí a forúkọsílẹ̀ nípasẹ̀ àwọn ìṣàn iṣẹ́ dispatch tí a ṣàkóso
- **Àwọn irinṣẹ́ akọ́-plugin** — dá ìpilẹ̀ àti tún kóòdù plugin ṣe díẹ̀díẹ̀
- **Àwọn irinṣẹ́ Git snapshot** — dá àwọn snapshot ṣáájú àwọn iṣẹ́ olùgbéejáde tó léwu
- **Àwọn irinṣẹ́ ìṣàkóso oníṣe** — ràn wá lọ́wọ́ láti ṣàgbéyẹ̀wò tàbí ṣàtúnṣe àwọn àkọsílẹ̀ oníṣe níbi tí a ti gba láàyè
- **Àwọn irinṣẹ́ benchmark** — ṣiṣẹ́ àwọn benchmark iṣẹ́ ṣiṣe tàbí agbára fún àgbéyẹ̀wò ìdàgbàsókè

Má ṣe kọ àwọn wọ̀nyí sílẹ̀ gẹ́gẹ́ bí irinṣẹ́ Theme Builder tí ó wà ní gbogbogbò fún gbogbo fifi-sílẹ̀. Wọ́n wà fún lílò nìkan nígbà tí àkójọpọ̀ alábàákẹ́gbẹ́ Advanced bá ti fi sílẹ̀, ti ṣiṣẹ́, tí a sì ti fi mọ́ àwọn alábòójútó tí a fọkàn tán.

## Ìṣètò Irinṣẹ́

### Mímú Àwọn Irinṣẹ́ Ṣiṣẹ́

Láti mú àwọn irinṣẹ́ àfikún ṣiṣẹ́ nínú Theme Builder:

1. Lọ sí **Gratis AI Agent → Settings**
2. Lọ sí **Theme Builder → Tools**
3. Yí àwọn irinṣẹ́ sí on/off bí ó ṣe nílò
4. Fi àwọn ìṣètò pamọ́

### Àwọn Ìyọ̀nda Irinṣẹ́

Àwọn irinṣẹ́ kan nílò ìyọ̀nda:

- **Fífà ojúlé** — nílò ìráyè sí ìńtánẹ́ẹ̀tì
- **Ìtúpalẹ̀** — nílò ìsọ̀kan Google Analytics
- **Ṣíṣe àkóónú** — nílò àwọn API keys
- **Àwọn àbùdá onígàárí** — lè nílò subscription premium
- **Àwọn irinṣẹ́ alábàákẹ́gbẹ́ Advanced** — nílò àkójọpọ̀ Advanced tí a pín ká lọ́tọ̀ àti ìgbẹ́kẹ̀lé alábòójútó ní kedere

### Àwọn API Keys àti Ẹ̀rí Ìwọlé

Ṣètò àwọn API keys fún àwọn irinṣẹ́ tí ó nílò wọn:

1. Lọ sí **Gratis AI Agent → Settings → API Keys**
2. Tẹ ẹ̀rí ìwọlé fún irinṣẹ́ kọ̀ọ̀kan
3. Dán ìsopọ̀ wò
4. Fi pamọ́ lọ́nà ààbò

### Fífì Àkójọpọ̀ Alábàákẹ́gbẹ́ Advanced Sílẹ̀

A ń tu àkójọpọ̀ alábàákẹ́gbẹ́ Advanced sílẹ̀ lọ́tọ̀ sí ZIP pàtàkì WordPress.org. Fi sílẹ̀ nìkan láti ikanni ìpínkiri iṣẹ́ akanṣe osìsẹ̀ fún ìtúsílẹ̀ náà, lẹ́yìn náà fi ìráyè mọ́ àwọn alábòójútó tí a fọkàn tán. Ṣàgbéyẹ̀wò àwọn ẹnubodè ìfọwọ́sí ènìyàn àti ìhùwàsí git snapshot ṣáájú mímú filesystem, database, WP-CLI, REST dispatcher, akọ́-plugin, ìṣàkóso oníṣe, tàbí àwọn irinṣẹ́ benchmark ṣiṣẹ́ lórí ojúlé production.

## Lílo sd-ai-agent/site-scrape

### Lílò Ìpìlẹ̀

Béèrè kí Theme Builder ṣe àyẹ̀wò wẹ́ẹ̀bùsáìtì kan:

```
Ṣe àyẹ̀wò àpẹrẹ ojúlé olùdíje mi ní competitor.com
kí o sì dábàá àwọn ìmúgbòrò àpẹrẹ fún ojúlé mi.
```

### Àyẹ̀wò Pàtó

Béèrè fún àwọn irú àyẹ̀wò pàtó:

```
Yọ àkójọpọ̀ àwọ̀ jáde láti example.com kí o sì lò ó gẹ́gẹ́ bí ìmísí.
```

```
Ṣe àyẹ̀wò ìṣètò ìtọ́sọ́nà example.com kí o sì fi ìṣètò tó jọ bẹ́ẹ̀
sí ojúlé mi.
```

```
Ṣàyẹ̀wò ìdáhùn sí alágbèéká ti example.com kí o sì rí dájú pé ojúlé mi
dáhùn bákan náà.
```

### Àyẹ̀wò Ìfiwéra

Fi àwọn ojúlé púpọ̀ wéra:

```
Fi àwọn àpẹrẹ site1.com àti site2.com wéra kí o sì dá àpẹrẹ kan
tí ó darapọ̀ àwọn èròjà tó dára jù nínú méjèèjì.
```

## Àwọn Ààlà Irinṣẹ́ àti Àwọn Ohun Tó Yẹ Kí A Ronú Lé

### Dídiwọ̀n Ìwọ̀n Lílò

- Scraping ní ààlà ìyára láti dènà kó server má bàjẹ́ nítorí ẹrù púpọ̀
- Ọ̀pọ̀lọpọ̀ ìbéèrè sí site kan náà lè jẹ́ dín ní ìyára
- Àwọn site ńlá lè gba àkókò pípẹ́ síi láti ṣe àyẹ̀wò

### Ìpéye Àkóónú

- Àkóónú tó ń yí padà lè má jẹ́ gbà ní kíkún
- Àkóónú tí JavaScript ń ṣàfihàn lè má pé
- Dátà àsìkò-gidi lè má hàn
- Díẹ̀ lára àkóónú lè wà lẹ́yìn paywalls

### Àwọn Ìgbéyẹ̀wò Òfin àti Ìwà Tó Tọ́

- Bọ̀wọ̀ fún robots.txt àti àwọn ìlànà site
- Má ṣe scrape àkóónú tó ní copyright fún àtúnṣe
- Lo àyẹ̀wò fún ìmísí, kì í ṣe fún ṣíṣe ẹ̀dà
- Jẹ́rìí pé o ní ẹ̀tọ́ láti lo àkóónú tí a yọ jáde
- Tẹ̀lé terms of service ti àwọn site tí a ṣe àyẹ̀wò

### Ipa Lórí Performance

- Scraping àwọn site ńlá lè gba àkókò
- Ọ̀pọ̀ scraping ní àkókò kan náà lè lọra síi
- Ìsopọ̀ nẹ́tíwọ́ọ̀kì ní ipa lórí ìyára
- Àwọn dataset ńlá lè nílò ìṣiṣẹ́pọ̀ síi

## Àwọn Ìlànà Tó Dára Jù

### Lílo Àyẹ̀wò Site

1. **Ṣe àyẹ̀wò ọ̀pọ̀ site** — kó ìmísí jọ láti ọ̀pọ̀ orísun
2. **Dojú kọ ìgbékalẹ̀** — kọ́ àwọn àpẹẹrẹ layout àti ètò
3. **Yọ àwọn àwọ̀ jáde** — lo palettes àwọ̀ gẹ́gẹ́ bí ibẹrẹ
4. **Kẹ́kọ̀ọ́ typography** — mọ̀ àwọn àpapọ̀ font tí o fẹ́ràn
5. **Ṣàtúnyẹ̀wò ìrìnàjò** — lóye ètò menu

### Scraping Tó Ní Ìwà Tó Tọ́

1. **Bọ̀wọ̀ fún robots.txt** — tẹ̀lé ìtọ́sọ́nà site
2. **Má ṣe da àkóónú kọ́pí** — lo àyẹ̀wò fún ìmísí nìkan
3. **Jẹ́rìí ẹ̀tọ́** — rí dájú pé o lè lo àkóónú tí a yọ jáde
4. **Dárúkọ àwọn orísun** — fún àwọn orísun ìmísí ní ìyìn
5. **Tẹ̀lé terms** — tẹ̀lé terms of service ti site

### Mímú Ìmunadoko Irinṣẹ́ pọ̀ si

1. **Jẹ́ kedere** — béèrè fún àwọn irú àyẹ̀wò pàtó
2. **Pèsè àyíká** — ṣàlàyé ète site rẹ
3. **Ṣètò ìreti** — ṣàpèjúwe àwọn ibi-afẹ́ design rẹ
4. **Ṣe àtúnṣe leralera** — túnṣe dá lórí àwọn èsì
5. **Darapọ̀ àwọn irinṣẹ́** — lo ọ̀pọ̀ irinṣẹ́ fún àyẹ̀wò tó péye

## Yíyanjú Ìṣòro

### Site Kò Ṣe Scrape

- Ṣàyẹ̀wò bóyá site wà fún gbogbo ènìyàn láti wọlé
- Jẹ́rìí pé robots.txt gba scraping láàyè
- Gbìyànjú site míì
- Ṣàyẹ̀wò ìsopọ̀ intanẹẹti
- Kàn sí support bí ìṣòro bá ń tẹ̀síwájú

### Àyẹ̀wò Tí Kò Pé

- Site lè ní àkóónú tó ń yí padà
- JavaScript lè má jẹ́ ṣàfihàn ní kíkún
- Àwọn site ńlá lè timeout
- Gbìyànjú ṣíṣe àyẹ̀wò àwọn ojúewé pàtó dípò
- Béèrè fún àwọn irú àyẹ̀wò pàtó

### Performance Tó Lọra

- Àwọn site ńlá máa ń gba àkókò pípẹ́ síi láti ṣe àyẹ̀wò
- Ọ̀pọ̀ scraping ní àkókò kan náà máa ń lọra síi
- Ìsopọ̀ nẹ́tíwọ́ọ̀kì ní ipa lórí ìyára
- Gbìyànjú nígbà tí lílo kò pọ̀
- Ṣe àyẹ̀wò àwọn apá kékeré kọ́kọ́
