# Àwọn Mẹ́nu Hospitality

Ẹ̀ya **Àwọn Mẹ́nu Hospitality** jẹ́ kí Theme Builder lè ṣẹ̀dá kí ó sì fi àwọn ojúewé mẹ́nu oúnjẹ àti ohun mímu tó ní ìṣètò sínú site WordPress rẹ taara.

## Àkótán

Theme Builder lè ṣẹ̀dá àwọn ojúewé mẹ́nu amọ̀ja tó ní ìṣètò fún àwọn ilé-iṣẹ́ hospitality, pẹ̀lú àwọn ilé oúnjẹ, cafes, bars, àti àwọn iṣẹ́ catering. Àwọn mẹ́nu wọ̀nyí darapọ̀ pátápátá mọ́ design site rẹ, a sì lè ṣe imudojuiwọn àti ṣàkóso wọn ní rọrùn.

## Àwọn Irú Hospitality Tí A Ṣe Àtìlẹ́yìn Fún

- **Àwọn Ilé Oúnjẹ** — àwọn mẹ́nu ìjẹun iṣẹ́-kíkún
- **Cafes** — àwọn mẹ́nu kọ́fí àti oúnjẹ fẹ́rẹ́fẹ́
- **Bars àti Lounges** — àwọn mẹ́nu ohun mímu àti appetizer
- **Àwọn Bakery** — àwọn mẹ́nu pastry àti búrẹ́dì
- **Àwọn Iṣẹ́ Catering** — àwọn aṣayan mẹ́nu ìṣẹ̀lẹ̀
- **Àwọn Food Trucks** — àwọn mẹ́nu iṣẹ́ oúnjẹ alágbèéká
- **Àwọn Brewery àti Winery** — àwọn mẹ́nu ohun mímu pẹ̀lú àlàyé

## Ìṣètò Mẹ́nu

### Àwọn Ẹ̀ka Mẹ́nu

A ṣètò àwọn mẹ́nu sí àwọn ẹ̀ka:

- **Appetizers** — àwọn ohun ìbẹ̀rẹ̀ àti àwọn awo kéékèèké
- **Entrees** — àwọn oúnjẹ àkọ́kọ́
- **Sides** — àwọn ìbáṣepọ̀ oúnjẹ àti ẹ̀fọ́
- **Desserts** — àwọn ìpanu dídùn
- **Beverages** — àwọn ohun mímu (tí ó ní ọtí àti tí kò ní ọtí)
- **Specials** — àwọn àkànṣe ojoojúmọ́ tàbí ti àkókò

### Fọ́ọ̀mù Ohun Mẹ́nu

Ohun mẹ́nu kọọkan ní:

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### Àwọn Field Ohun

| Field | Type | Description |
|-------|------|-------------|
| `name` | string | Orúkọ oúnjẹ tàbí ohun mímu |
| `description` | string | Àlàyé kíkún nípa ohun náà |
| `price` | string | Iye owó (tí a ṣe fọ́ọ̀mù rẹ pẹ̀lú owó orílẹ̀-èdè) |
| `dietary_info` | array | Àwọn ànímọ́ oúnjẹ (Vegan, Gluten-Free, àti bẹ́ẹ̀ bẹ́ẹ̀ lọ) |
| `allergens` | array | Àwọn nkan tí ń fa allergy tí ó wọ́pọ̀ (Nuts, Shellfish, àti bẹ́ẹ̀ bẹ́ẹ̀ lọ) |
| `availability` | string | Ìgbà tí ó wà (Daily, Seasonal, àti bẹ́ẹ̀ bẹ́ẹ̀ lọ) |

## Ṣíṣe Àwọn Mẹ́nu Hospitality

### Ìgbésẹ̀ 1: Pèsè Ìwífún Mẹ́nu

Sọ fún Theme Builder nípa mẹ́nu rẹ:

```
Create a restaurant menu for my Italian restaurant with appetizers,
pasta dishes, main courses, and desserts. Include prices and descriptions.
```

### Ìgbésẹ̀ 2: Theme Builder Ṣẹ̀dá Mẹ́nu

Theme Builder:

1. Ṣẹ̀dá ojúewé mẹ́nu tó ní ìṣètò
2. Ṣe design rẹ kí ó bá theme site rẹ mu
3. Ṣètò àwọn ohun sí àwọn ẹ̀ka
4. Ṣe fọ́ọ̀mù àwọn iye owó àti àwọn àlàyé
5. Ṣàfikún ìwífún oúnjẹ àti allergen

### Ìgbésẹ̀ 3: Ṣàyẹ̀wò àti Ṣe Àtúnṣe

O lè:

1. Ṣàtúnṣe àwọn ohun mẹ́nu àti iye owó
2. Ṣàfikún tàbí yọ àwọn ẹ̀ka kúrò
3. Tún àwọn ohun tó wà nínú ẹ̀ka ṣètò
4. Ṣe imudojuiwọn àwọn àlàyé àti ìwífún oúnjẹ
5. Ṣàtúnṣe styling àti layout

## Àwọn Aṣayan Ìfihàn Mẹ́nu

### Ojúewé Mẹ́nu Kíkún

Ojúewé pàtàkì kan tí ń ṣàfihàn mẹ́nu rẹ pípé:

- A ṣètò rẹ nípasẹ̀ ẹ̀ka
- Ó ṣeé wá, ó sì ṣeé filter
- Layout tó dára fún titẹ̀ jáde
- Design tó dáhùn sí mobile

### Widget Mẹ́nu

Fi àwọn abala mẹ́nu sínú àwọn ojúewé mìíràn:

- Àwọn ohun tí a ṣe àfihàn lórí homepage
- Àwọn àkànṣe ojoojúmọ́ lórí sidebar
- Mẹ́nu ohun mímu lórí ojúewé bar
- Àfihàn dessert lórí footer

### PDF Mẹ́nu

Ṣẹ̀dá mẹ́nu PDF tí a lè ṣe igbasilẹ:

- Fọ́ọ̀mù amọ̀ja
- Didara tó ṣetán fún titẹ̀ jáde
- Ó ní àwọn àwòrán àti àlàyé nínú
- Rọrùn láti pín àti fi imeeli ránṣẹ́

## Ìwífún Oúnjẹ àti Allergen

### Àwọn Ànímọ́ Oúnjẹ

Samisi àwọn ohun pẹ̀lú ìwífún oúnjẹ:

- **Vegan** — kò sí ọja ẹranko
- **Vegetarian** — kò sí ẹran
- **Gluten-Free** — dájú fún àrùn celiac
- **Dairy-Free** — kò sí ọja wàrà
- **Nut-Free** — kò sí èso igi tàbí peanuts
- **Low-Carb** — carbohydrate tí a dín kù
- **High-Protein** — ọlọ́rọ̀ ní protein

### Àwọn Ìkìlọ̀ Allergen

Fi àwọn allergen tí ó wọ́pọ̀ sínú:

- **Nuts** — àwọn èso igi àti peanuts
- **Shellfish** — crustaceans àti mollusks
- **Fish** — gbogbo ẹ̀yà ẹja
- **Dairy** — wàrà àti àwọn ọja wàrà
- **Eggs** — ẹyin adìẹ
- **Soy** — àwọn ọja soy
- **Gluten** — alikama àti àwọn irúgbìn tó ní gluten
- **Sesame** — irúgbìn sesame àti òróró

## Ìṣàkóso Mẹ́nu

### Ṣíṣe Imudojuiwọn Àwọn Iye Owó

Ṣe imudojuiwọn àwọn iye owó mẹ́nu ní rọrùn:

1. Lọ sí ojúewé mẹ́nu náà
2. Tẹ "Ṣàtúnṣe Mẹ́nu"
3. Ṣe imudojuiwọn iye owó fún àwọn ohun
4. Fi àwọn ayípadà pamọ́
5. Àwọn ayípadà yóò hàn lẹ́sẹ̀kẹsẹ̀ lórí site rẹ

### Ṣíṣe Àfikún Àwọn Ohun Ti Àkókò

Ṣẹ̀dá àwọn ìyípadà mẹ́nu ti àkókò:

1. Ṣẹ̀dá ẹ̀yà mẹ́nu tuntun
2. Ṣàfikún àwọn ohun ti àkókò
3. Samisi àwọn ohun gẹ́gẹ́ bí "Seasonal"
4. Ṣètò àwọn ọjọ́ availability
5. Ṣàfihàn laifọwọyi nígbà àkókò náà

### Ṣíṣàkóso Àwọn Specials

Ṣàfihàn àwọn àkànṣe ojoojúmọ́ tàbí ọ̀sọ̀ọ̀sẹ̀:

1. Ṣẹ̀dá ẹ̀ka "Specials"
2. Ṣàfikún àwọn ohun pẹ̀lú àwọn ọjọ́ availability
3. Ṣe ìtẹnumọ́ àwọn specials lórí homepage
4. Ṣe imudojuiwọn lojoojúmọ́ tàbí lọ́sọ̀ọ̀sẹ̀
5. Fi àwọn specials àtijọ́ sí archive

## Ìṣepọ̀ pẹ̀lú Theme Builder

Nígbà tí o bá ń lo Theme Builder fún àwọn site hospitality:

1. **Ìdámọ̀ mẹ́nu laifọwọyi** — ń dá a mọ̀ bóyá o jẹ́ ilé-iṣẹ́ oúnjẹ/ohun mímu
2. **Ṣíṣẹ̀dá ojúewé mẹ́nu** — ń ṣẹ̀dá àwọn ojúewé mẹ́nu amọ̀ja
3. **Ìbámu design** — àwọn mẹ́nu bá theme site rẹ mu
4. **Ìmúṣẹ̀dára fún mobile** — àwọn mẹ́nu hàn dáadáa lórí fóònù
5. **Ìmúṣẹ̀dára SEO** — àwọn mẹ́nu dára fún ẹ̀rọ ìwádìí

## Àwọn Ìlànà Tó Dára Jù

### Design Mẹ́nu

- **Ìṣètò tó ṣe kedere** — ìṣètò ẹ̀ka tó ní ọgbọ́n
- **Àwọn àlàyé tó ṣeé kà** — tó ń mú ìfẹ́ oúnjẹ wá, tó sì ní ìwífún
- **Iye owó tó bá ara mu** — owó orílẹ̀-èdè àti fọ́ọ̀mù tó ṣe kedere
- **Àwọn fọ́tò amọ̀ja** — àwọn àwòrán oúnjẹ didara-gíga
- **Ààyè funfun** — má ṣe kún ojúewé ju

### Àkóónú

- **Àwọn àlàyé tó péye** — ṣàlàyé àwọn ohun ní deede
- **Ṣe ìtẹnumọ́ àwọn specials** — jẹ́ kí àwọn specials yàtọ̀ síta
- **Fi àwọn allergen sínú** — máa ṣe atokọ àwọn allergen nígbà gbogbo
- **Ṣe imudojuiwọn déédéé** — jẹ́ kí iye owó àti àwọn ohun jẹ́ ti báyìí
- **Lo èdè tó ń mú ìfẹ́ oúnjẹ wá** — jẹ́ kí àwọn ohun dun bíi pé wọ́n dùn gan-an

### Ìráàyèsí

- **Àwọn font tó ṣeé kà** — lo àwọn typeface tó ṣe kedere, tó sì ṣeé kà
- **Contrast tó tó** — rí i dájú pé text ṣeé kà
- **Àwọn label oúnjẹ** — samisi àwọn aṣayan oúnjẹ ní kedere
- **Àwọn ìkìlọ̀ allergen** — ṣàfihàn àwọn allergen ní ibi tí ó hàn gbangba
- **Ó dára fún mobile** — dán wò lórí gbogbo ẹrọ
