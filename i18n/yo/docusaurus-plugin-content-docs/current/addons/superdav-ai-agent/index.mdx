# Superdav AI Agent

Superdav AI Agent jẹ olùrànlọwọ AI WordPress tó ti ni ilọsiwaju tí ó fa Gratis AI Agent gbooro pẹ̀lú àwọn ọgbọ́n tó mọ theme, agbára ìwádìí ìntánẹ́ẹ̀tì tó dára síi, àwọn ẹ̀ya site builder, àti àwọn ìsopọ̀ automation tí a lè yan. Ẹ̀dà 1.18.0 ṣàfikún àwọn irinṣẹ́ kíkà Google Calendar, mímápù contact àti attendee, àwọn ẹnu-ọ̀nà ìfọwọ́sí ènìyàn, yíyọ reminder àtúnṣe kúrò, àwọn ìfitónilétí TextBee SMS, pípèsè iṣẹ́ Superdav tí a ń ṣakoso, àti package Advanced ẹlẹgbẹ́ tí a pín ká lọ́tọ̀ fún àwọn developer workflow tí a gbẹ́kẹ̀ lé.

## Àwọn Ẹ̀ya Pàtàkì

- **Àwọn ọgbọ́n tó mọ theme** — ó máa ń ṣe àtúnṣe ìtọ́sọ́nà àti agbára laifọwọyi dá lórí WordPress theme rẹ tó ń ṣiṣẹ́ àti àwọn plugin tí a ti fi sí
- **Ọ̀pọ̀lọpọ̀ provider ìwádìí ìntánẹ́ẹ̀tì** — yan láàárín Brave Search àti Tavily fún àwọn èsì ìwádìí tó jinlẹ̀ síi àti agbára ìwádìí tó ti ni ilọsiwaju
- **Fọ́ọ̀mù contact site builder** — dá àwọn fọ́ọ̀mù contact tó ń ṣiṣẹ́ ní kikún taara láti inú chat interface nípa lílo plugin fọ́ọ̀mù tí o ti fi sí
- **Site builder tó ti ni ilọsiwaju** — àwọn plan ìkọ́ site ọ̀pọ̀ ìpele tí a mú dara síi pẹ̀lú ìṣàwárí plugin tó dára jù àti ìmúpadàbọ̀ láti aṣìṣe
- **Àwọn ọgbọ́n theme inú rẹ** — ìtọ́sọ́nà amọ̀ja fún Block Themes, Classic Themes, Kadence Blocks, àti Kadence Theme
- **Ìsopọ̀ WooCommerce tó dara síi** — a tún un ṣe láti lo àwọn WooCommerce API abinibi fún igbẹ́kẹ̀lé àti ìbámu tó dara síi
- **Ìṣàwárí provider laifọwọyi** — ó máa ń ṣàwárí àti tún àwọn plugin àti theme tó wà ṣe laifọwọyi nígbà tí a bá mú wọn ṣiṣẹ́ tàbí pa wọ́n
- **Ṣíṣe logo àdáni** — ṣe kí o sì fi àwọn logo SVG àdáni sínú pẹ̀lú ìwẹ̀nù namespace-safe laifọwọyi
- **Ìfọwọ́sí ìyàtọ̀ palette** — ṣàyẹ̀wò àwọn tọkọtaya colour fún ìbámu accessibility WCAG kí o tó fi sí theme rẹ
- **Ìṣàwárí tó ni fọ́tò** — gbe àwọn fọ́tò sórí nígbà ìfọ̀rọ̀wánilẹ́nuwò ìṣàwárí Theme Builder láti ní ipa lórí ìtọ́sọ́nà design
- **Àwọn menu hospitality** — dá àwọn ojúewé menu oúnjẹ àti ohun mímu tó ní ìgbékalẹ̀ fún ilé oúnjẹ, kafe, àti àwọn iṣẹ́ hospitality
- **Àwọn àwòkọ desktop àti mobile** — wo design rẹ ṣáájú lórí desktop àti ẹ̀rọ mobile nígbà yíyan ìtọ́sọ́nà design
- **Tier 1 site scraping** — sd-ai-agent/site-scrape wà báyìí ní aiyélujára fún ṣíṣe àyẹ̀wò àwọn website olùdíje àti ìmísí
- **Àwọn irinṣẹ́ kíkà Google Calendar** — ka àwọn calendar àti event tí a ti ṣètò kí àwọn automation lè rántí ipo schedule tó ń bọ̀
- **Mímápù contact àti attendee** — bá àwọn olukópa calendar mu mọ́ àwọn olumulo site àti contact tí a mọ̀ kí a tó dá iṣẹ́ follow-up sílẹ̀
- **Àwọn ẹnu-ọ̀nà ìfọwọ́sí ènìyàn** — dá àwọn automation ẹlẹ́gẹ́ dúró fún àyẹ̀wò kí a tó fi ránṣẹ́, tẹ̀síwájú iṣẹ́, tàbí ṣe àwọn ayipada
- **Àwọn àkọọ́lẹ̀ reminder àti yíyọ àtúnṣe kúrò** — tọpinpin àwọn reminder tí a ti rán tẹ́lẹ̀ kí àwọn automation tí ń ṣiṣẹ́ lẹ́ẹ̀kansi má ṣe fi tó olùgbà kan náà létí lẹ́ẹ̀mejì
- **Àwọn ìfitónilétí TextBee SMS** — fi àwọn ìfitónilétí text-message tí a ti ṣètò ránṣẹ́ nípasẹ̀ TextBee nígbà tí SMS provider bá ṣiṣẹ́
- **Pípèsè iṣẹ́ Superdav tí a ń ṣakoso** — so àwọn site tí a ṣe àtìlẹ́yìn pọ̀ mọ́ àwọn endpoint Superdav tí a gbalejo pẹ̀lú ìṣètò laifọwọyi dípò fífi endpoint ṣe àtúnṣe lọ́wọ́
- **Package Advanced ẹlẹgbẹ́** — àwọn irinṣẹ́ trusted-developer tí a pín ká lọ́tọ̀ fún filesystem, database, WP-CLI, REST dispatcher, plugin builder, git snapshot, user-management, àti benchmark workflows

## Àwọn Ìbéèrè

- WordPress 6.0 tàbí tó ga jù bẹ́ẹ̀ lọ
- PHP 8.1 tàbí tó ga jù bẹ́ẹ̀ lọ
- WordPress Multisite (network-activated) tàbí fifi sori single-site
- Plugin connector AI provider tó ń ṣiṣẹ́ (Anthropic, OpenAI, tàbí tó bá mu)
- Gratis AI Agent v1.9.0 tàbí tó ga jù bẹ́ẹ̀ lọ (Superdav AI Agent fa Gratis AI Agent gbooro)

## Fifi Sori

1. Gbe àwọn fáìlì plugin sí `/wp-content/plugins/superdav-ai-agent`
2. Mú plugin náà ṣiṣẹ́ nípasẹ̀ iboju **Plugins** ní WordPress
3. Rí i dájú pé Gratis AI Agent náà ti ṣiṣẹ́
4. Lọ sí **Gratis AI Agent → Settings** kí o sì ṣètò provider ìwádìí ìntánẹ́ẹ̀tì rẹ, ìsopọ̀ iṣẹ́ Superdav tí a ń ṣakoso, Google Calendar, tàbí àwọn credential TextBee bí ó ṣe nílò

Package WordPress.org ní àwọn ẹ̀ya mojuto Superdav AI Agent nínú. A kọ package Advanced ẹlẹgbẹ́, a sì pín un ká lọ́tọ̀ fún àwọn ayika níbi tí a ti fi àwọn irinṣẹ́ developer tí a gbẹ́kẹ̀ lé sori àti tí a fún ní àṣẹ ní kedere.

## Ìbẹ̀rẹ̀ Kíákíá

## Àwọn Ọgbọ́n Tó Mọ Theme

Superdav AI Agent máa ń ṣàwárí theme rẹ tó ń ṣiṣẹ́ àti àwọn plugin tí a ti fi sí laifọwọyi, ó sì ń kó ọgbọ́n tó mọ theme tó yẹ sínú context olùrànlọwọ. Àwọn ọgbọ́n tó wà ni:

| Ọgbọ́n | Wúlò fún |
|---|---|
| **Block Themes** | Àwọn theme tó ń lo faaji WordPress tó dá lórí block (Full Site Editing) |
| **Classic Themes** | Àwọn WordPress theme àtọwọ́dọwọ́ tó ń lo àwọn template PHP àti classic editor |
| **Kadence Blocks** | Àwọn site tí plugin Kadence Blocks ń ṣiṣẹ́ lórí |
| **Kadence Theme** | Àwọn site tí theme Kadence jẹ́ theme tó ń ṣiṣẹ́ |

Wo [Àwọn Ọgbọ́n Tó Mọ Theme](../../user-guide/configuration/theme-aware-skills) fún ìtọ́sọ́nà kíkún lórí ọgbọ́n kọ̀ọ̀kan.

## Àwọn Provider Ìwádìí Ìntánẹ́ẹ̀tì

Superdav AI Agent v1.10.0 ṣàfikún **Tavily** gẹ́gẹ́ bí yíyàn mìíràn sí Brave Search fún àwọn ìbéèrè ìwádìí ìntánẹ́ẹ̀tì. Àwọn provider méjèèjì ń fúnni ní àwọn èsì ìwádìí akoko-gidi, a sì lè ṣètò wọn ní **Gratis AI Agent → Settings → Advanced**.

| Provider | Àwọn Ẹ̀ya | Ìṣètò |
|---|---|---|
| **Brave Search** | Àwọn èsì ìwádìí tó yára, tó dojúkọ privacy | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Àwọn èsì ìwádìí tó jinlẹ̀ síi, agbára ìwádìí tó ti ni ilọsiwaju | [Tavily API](https://tavily.com/) |

Wo [Ìwádìí Ìntánẹ́ẹ̀tì](../../user-guide/configuration/internet-search) fún àwọn ìtọ́ni ìṣètò.

## Calendar, Ìfọwọ́sí, àti Àwọn Workflow Ìfitónilétí

Superdav AI Agent v1.18.0 fa àwọn workflow automation kọjá àwọn iṣẹ́ content àti ìkọ́ site:

- **Àwọn irinṣẹ́ Google Calendar** jẹ́ kí agent ka àwọn calendar tí a ti ṣètò àti àwọn àlàyé event nígbà tí àwọn credential calendar bá wà.
- **Àwọn olùrànlọwọ contact àti attendee** máa ń mápù àwọn olukópa event sí àwọn olumulo WordPress tàbí àwọn contact tí a mọ̀ kí a tó pèsè àwọn reminder tàbí iṣẹ́ follow-up.
- **Àwọn ẹnu-ọ̀nà ìfọwọ́sí ènìyàn** jẹ́ kí automation dá dúró títí olùmúlò tí a fún ní àṣẹ yóò fi ṣàyẹ̀wò ìṣe tí a dabaa.
- **Àwọn àkọọ́lẹ̀ reminder** dènà calendar tàbí notification reminder méjì nígbà tí a bá tún automation gbìyànjú tàbí ṣètò rẹ léraléra.
- **Àtìlẹ́yìn TextBee SMS** jẹ́ kí àwọn ìfitónilétí text-message ṣiṣẹ́ nígbà tí a bá ṣètò provider TextBee.

Lo àwọn irinṣẹ́ wọ̀nyí fún ìrànwọ́ tó mọ schedule, follow-up attendee, àti àwọn workflow reminder tí ó ṣì nílò àyẹ̀wò ènìyàn kí a tó fi àwọn message ẹlẹ́gẹ́ ránṣẹ́.

## Fọ́ọ̀mù Contact Site Builder

Agent Site Builder lè dá àwọn fọ́ọ̀mù contact sílẹ̀ taara láti inú chat interface báyìí. Kan ní kí Site Builder dá fọ́ọ̀mù contact sílẹ̀, yóò sì ṣàwárí plugin fọ́ọ̀mù tí o ti fi sí laifọwọyi (Contact Form 7, WPForms, Fluent Forms, tàbí Gravity Forms) kí o sì dá fọ́ọ̀mù tó ṣetán-láti-lò sílẹ̀.

Wo [Fọ́ọ̀mù Contact Site Builder](../../user-guide/administration/site-builder-contact-forms) fún ìtọ́sọ́nà kíkún.

## Ìṣètò

Gbogbo settings wà lábẹ́ **Gratis AI Agent → Settings** nínú admin WordPress. Superdav AI Agent ṣàfikún àwọn aṣàyàn ìṣètò wọ̀nyí:

| Setting | Àpèjúwe |
|---|---|
| **Internet Search Provider** | Yan láàárín Brave Search àti Tavily (àṣàyàn) |
| **Tavily API Key** | API key fún ìwádìí ìntánẹ́ẹ̀tì Tavily (bí Tavily bá jẹ́ èyí tí a yan) |
| **Managed Superdav Service** | So àwọn site tí a ṣe àtìlẹ́yìn pọ̀ mọ́ iṣẹ́ Superdav tí a gbalejo pẹ̀lú pípèsè endpoint laifọwọyi |
| **Google Calendar Credentials** | Ṣètò access calendar kí agent lè ka àwọn calendar àti event |
| **TextBee SMS Provider** | Ṣètò àwọn credential TextBee fún àwọn workflow ìfitónilétí SMS tí a fọwọ́sí |

## Package Advanced Ẹlẹgbẹ́

Superdav AI Agent v1.18.0 ṣàfikún package Advanced ẹlẹgbẹ́ tí a pín ká lọ́tọ̀ fún àwọn developer workflow tí a gbẹ́kẹ̀ lé. Àwọn agbára wọ̀nyí kò sí nínú package mojuto WordPress.org. Fi package ẹlẹgbẹ́ sori àwọn site nìkan níbi tí administrators ti gbẹ́kẹ̀ lé agent ní kedere láti wọlé sí developer tooling.

Àwọn irinṣẹ́ Advanced ní àwọn iṣẹ́ filesystem, access database, WP-CLI, REST dispatcher workflows, àwọn ohun èlò plugin builder, git snapshots, àwọn olùrànlọwọ user-management, àti benchmark tooling. Jẹ́ kí àwọn irinṣẹ́ wọ̀nyí mọ́ sí administrators tí a gbẹ́kẹ̀ lé nìkan, kí o sì ṣàyẹ̀wò ìṣètò approval-gate kí o tó mú wọn ṣiṣẹ́ lórí àwọn site production.

## Àwọn Àtúnṣe Àṣìṣe nínú v1.10.0

- A tún agbára `navigate-to` ṣe tó ń fa àwọn ìyípadà ìgbéyàpadà aláìlópin lórí díẹ̀ lára àwọn ojúewé admin
- A tún agbára `list-posts` ṣe tí kò ń yanjú àwọn orúkọ ẹ̀ka àti tag sí slugs dáadáa
- A tún àwọn àṣẹ WP-CLI ṣe tí àwọn alias namespace sọnù lẹ́yìn refactor ṣáájú
- A tún event automation ṣe kí ó lè bójú tó àwọn ojúlé dáadáa níbi tí a kò tíì dá àwọn tábìlì automation sílẹ̀
- A tún agbára `memory-save` ṣe kí ó máa lo prefix namespace tó tọ́ nínú olùkọ́ ìtọ́nisọ́nà eto
- A tún àwọn abajade irinṣẹ́ scalar ṣe kí a lè dì wọ́n dáadáa kí a tó dá wọn padà sí AI
- A tún àwọn iṣirò lílò ṣe kí wọ́n lè bójú tó format key agbára àtijọ́ dáadáa nígbà ìgbésókè láti àwọn ẹ̀yà àtijọ́

## Àtìlẹ́yìn

Fún àtìlẹ́yìn, ṣàbẹ̀wò sí [ultimatemultisite.com](https://ultimatemultisite.com) tàbí ṣí issue kan nínú ibi ìpamọ́ afikun náà.
