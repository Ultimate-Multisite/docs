# Àkọsílẹ̀ ìyípadà {#changelog}

## 1.18.0 — Tú síta ní 2026-06-29 {#1180--released-on-2026-06-29}

### Tuntun {#new}

- **Àwọn irinṣẹ́ Google Calendar** — ka àwọn kálẹ́ńdà àti ìṣẹ̀lẹ̀ tí a ti ṣètò fún àwọn àfọwọ́ṣe tó mọ ìṣètò-àkókò
- **Ìbámu olubásọrọ̀ àti àwọn olùrànlọ́wọ́ olukópa** — bá àwọn olukópa kálẹ́ńdà mu pẹ̀lú àwọn aṣàmúlò àti olubásọrọ̀ ojúlé
- **Àwọn ẹnubodè ìfọwọ́sí ènìyàn àti àwọn àkọsílẹ̀ ìránnilétí** — dá àwọn àfọwọ́ṣe dúró fún àyẹ̀wò kí o sì yàgò fún àwọn ìfitónilétí méjì
- **Olùpèsè SMS TextBee** — fi àwọn ìfitónilétí ìfiránṣẹ́-ọ̀rọ̀ tí a ti ṣètò ránṣẹ́ nípasẹ̀ TextBee
- **Àpò ìbákẹ́gbẹ́ gíga** — ṣàfikún filesystem olùdàgbàsókè-tí-a-gbẹ́kẹ̀lé, ibi ìpamọ́ dátà, WP-CLI, REST dispatcher, olùkọ́ plugin, git snapshot, ìṣàkóso-aṣàmúlò, àti àwọn irinṣẹ́ benchmark tí a pín lọ́tọ̀ sí build WordPress.org

### Ti mú dara síi {#improved}

- **Ìṣètò iṣẹ́ Superdav tí a ń ṣàkóso** — ṣàfikún àwọn endpoint iṣẹ́ tí a gbàlejò àti pípèsè ìsopọ̀ aládàáṣiṣẹ́ fún àwọn ojúlé tí a ṣe àtìlẹ́yìn fún
- **Ìṣàkójọ ìtújáde** — kọ́ core àti àwọn ZIP Advanced lọ́tọ̀, tẹ àwọn méjèèjì jáde lórí GitHub, kí o sì fi àpò core nìkan ránṣẹ́ sí WordPress.org

### Ti ṣe atunṣe {#fixed}

- **Ìgbẹ́kẹ̀lé ìbéèrè AI** — mú yíyan model, àwọn àkókò-ìparí ìbéèrè, model ìṣètò àiyépadà, ìmúlò reasoning-text, àti ìtọ́nisọ́nà ìgbìyànjú lẹ́ẹ̀kansi fún àwọn ìpè irinṣẹ́ tí kò tọ́ dara síi
- **Ìmúlágbára kálẹ́ńdà àti ìránnilétí** — fún àwọn token Google Calendar àti ìyọkúrò àtúnṣe ìránnilétí ní agbára
- **Ìbẹ̀rẹ̀-lílò àti ìpadàbọ̀ ìfọwọ́sí** — ṣe atunṣe àwọn ìfilọlẹ̀ ìbẹ̀rẹ̀-lílò frontend àti àwọn ìpadàbọ̀ agbára tí a ti jẹ́rìí sí
- **Àwọn ọ̀ràn àyẹ̀wò ìṣàkójọ WordPress.org** — dáhùn sí èsì àyẹ̀wò ìṣàkójọ fún ìtújáde core

## 1.16.0 — Tú síta ní 2026-05-20 {#1160--released-on-2026-05-20}

### Tuntun {#new-1}

- **Agbára Generate Logo SVG** — Theme Builder lè dá àti fi àwọn SVG àmì-àmì àdáni sínú báyìí pẹ̀lú ìsọ̀dímímọ́ tó dáàbò bo namespace
- **Ìkojọpọ̀ fọ́tò nínú ìfọ̀rọ̀wánilẹ́nuwò ìṣàwárí** — ìfọ̀rọ̀wánilẹ́nuwò ìṣàwárí Theme Builder ní ìgbésẹ̀ ìkojọpọ̀ fọ́tò báyìí fún àyíká ọ̀nà-àpẹrẹ tó jinlẹ̀ síi
- **Agbára Validate Palette Contrast** — ṣàyẹ̀wò àwọn méjì àwọ̀ fún ìbámu WCAG kí o tó lò ó sí theme
- **Àwọn menu ilé-àlejò** — Theme Builder lè dá àwọn ojú-ewé menu oúnjẹ àti ohun mímu tó ní ìṣètò fún àwọn ilé-iṣẹ́ àlejò báyìí
- **Ìfihàn àyẹ̀wò desktop àti mobile** — ṣàyẹ̀wò àpẹrẹ rẹ lórí desktop àti àwọn ẹ̀rọ mobile nígbà yíyan ìtọ́sọ́nà-àpẹrẹ
- **Paramita àkọlé ìtọ́sọ́nà** — agbára Create Menu ṣe àtìlẹ́yìn fún `navigation_label` tó yàtọ̀ sí àkọlé ojú-ewé báyìí
- **Wíwà irinṣẹ́ Tier 1** — sd-ai-agent/site-scrape jẹ́ irinṣẹ́ Tier 1 tó wà ní àiyépadà nínú Theme Builder báyìí

### Ti ṣe atunṣe {#fixed-1}

- **Cache AI Client** — a ti ṣe àtìlẹ́yìn rẹ̀ pẹ̀lú transients fún ìdúróṣinṣin kọjá-ìbéèrè, tó ń dènà ìsonù dátà lórí àwọn iṣẹ́ agent tó ń ṣiṣẹ́ pẹ́
- **Àwọn link ìṣe ẹsẹ̀ plugin** — a ṣe atunṣe wọn, a sì tún sọ wọ́n lórúkọ fún ìmọ́ kedere

## 1.10.0 — Tú síta ní 2026-05-05 {#1100--released-on-2026-05-05}

### Tuntun {#new-2}

- **Ìṣàwárí ìńtánẹ́ẹ̀tì Tavily** — ṣàfikún Tavily gẹ́gẹ́ bí olùpèsè ìṣàwárí fún àwọn èsì ìṣàwárí ìńtánẹ́ẹ̀tì tó jinlẹ̀ síi lẹ́gbẹ̀ẹ́ Brave Search
- **Àwọn ọgbọ́n inú rẹ̀ tí ó mọ theme** — àwọn ìtọ́nisọ́nà ọgbọ́n Block Themes, Classic Themes, Kadence Blocks, àti Kadence Theme wà pẹ̀lú plugin báyìí
- **Agbára fọ́ọ̀mù olubásọrọ̀ olùkọ́ ojúlé** — ṣàfikún fọ́ọ̀mù olubásọrọ̀ sí ojú-ewé èyíkéyìí tààrà láti interface ìfọ̀rọ̀wérọ̀

### Ti mú dara síi {#improved-1}

- **Ìṣepọ̀ WooCommerce tí a tún ṣètò** — ó ń lo àwọn API WooCommerce abinibi báyìí fún ìgbẹ́kẹ̀lé àti ìbáramu tó dára síi
- **Àtòkọ olùpèsè ń tún ara rẹ̀ ṣe ní aládàáṣiṣẹ́** — nígbà tí plugin èyíkéyìí bá ti muu ṣiṣẹ́ tàbí pa á

### Ti ṣe atunṣe {#fixed-2}

- **Agbára navigate-to** — ṣe atunṣe yíyí-po ìtúnkojọpọ̀ àìlópin lórí àwọn ojú-ewé admin kan
- **Agbára list-posts** — ó ń yanjú àwọn orúkọ ẹ̀ka àti tag sí slugs dáadáa báyìí
- **Àwọn àṣẹ WP-CLI** — dá àwọn alias namespace tí ó sọnù padà lẹ́yìn refactor kan tẹ́lẹ̀
- **Àfọwọ́ṣe ìṣẹ̀lẹ̀** — ń bójú tó àwọn ojúlé tí a kò tíì dá àwọn tábìlì àfọwọ́ṣe sílẹ̀ láìní ìṣòro
- **Agbára memory-save** — ó ń lo prefix namespace tó tọ́ nínú olùkọ́ ìtọ́nisọ́nà system báyìí
- **Àwọn èsì irinṣẹ́ Scalar** — a ti fi wọ́n sínú ìdìpọ̀ dáadáa báyìí kí a tó dá wọn padà sí AI
- **Àwọn ìṣirò lílò** — ó ń bójú tó fọ́ọ̀mátì bọ́tìnì agbára legacy dáadáa báyìí nígbà ìgbésókè láti àwọn ẹ̀dà àtijọ́
