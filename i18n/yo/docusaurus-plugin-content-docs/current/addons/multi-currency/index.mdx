# Ọ̀pọ̀ Owó

Ultimate Multisite: Ọ̀pọ̀ Owó jẹ́ kí nẹ́tíwọ́ọ̀kì rẹ gba ìsanwó ní ọ̀pọ̀lọpọ̀ owó. Ó ń yí àwọn iye owó padà laifọwọyi nípa lílo oṣùwọ̀n pàṣípààrọ̀ tí ó wà láàyè tàbí ti ọwọ́, ó jẹ́ kí o ṣètò iye owó gangan fún ọjà kọọkan fún owó èyíkéyìí, ó sì ń ṣàwárí owó tí alejo kọọkan fẹ́ràn nípasẹ̀ geolocation — gbogbo rẹ̀ ni a ń ṣàkóso láti Dashboard alábójútó kan ṣoṣo.

## Àwọn Ẹ̀yà Pàtàkì

- **Ìyípadà oṣùwọ̀n pàṣípààrọ̀ laifọwọyi** — a ń yí àwọn iye owó padà láti owó ìpìlẹ̀ rẹ ní lílo àwọn oṣùwọ̀n tí a gba láti ọdọ olùpèsè tí o yàn
- **Ìforúkọsílẹ̀ iye owó ọjà kọọkan** — ṣètò àwọn iye owó pàtó fún owó kan lori ọjà èyíkéyìí láti kọjá ìyípadà laifọwọyi
- **Pápá ìforúkọsílẹ̀ aṣàyàn owó** — jẹ́ kí àwọn oníbàárà yan owó tí wọ́n fẹ́ràn nígbà ìsanwó/ìforúkọsílẹ̀
- **Ọ̀pọ̀lọpọ̀ olùpèsè oṣùwọ̀n pàṣípààrọ̀** — ECB, Frankfurter, Stripe, tàbí àwọn oṣùwọ̀n ọwọ́ pátápátá
- **Ìṣàwárí owó tó dá lórí geolocation** — ó máa ń yan owó tí ó bá orílẹ̀-èdè alejo mu ṣáájú laifọwọyi
- **Dashboard alábójútó** — ṣàkóso àwọn owó tí ń ṣiṣẹ́, àwọn oṣùwọ̀n pàṣípààrọ̀, àti àwọn ìṣètò olùpèsè láti ibi kan

## Àwọn Ohun Tí A Nílò

- Ultimate Multisite 2.0.0 tàbí tó ga jù bẹ́ẹ̀ lọ
- Fífi WordPress Multisite sílẹ̀
- PHP 7.4 tàbí tó ga jù bẹ́ẹ̀ lọ

## Fífi Sílẹ̀

1. Gbé àwọn fáìlì plugin sí `/wp-content/plugins/ultimate-multisite-multi-currency`
2. Mú plugin ṣiṣẹ́ nípasẹ̀ ojú-ìbojú **Àwọn Plugin** nínú WordPress (Mú ṣiṣẹ́ fún Nẹ́tíwọ́ọ̀kì)
3. Lọ sí **Ultimate Multisite → Ọ̀pọ̀ Owó** láti ṣí Dashboard alábójútó
4. Ṣàfikún àwọn owó tí o fẹ́ gba, kí o sì ṣètò olùpèsè oṣùwọ̀n pàṣípààrọ̀ rẹ

## Ìṣètò

### Dashboard Alábójútó

Dashboard alábójútó Ọ̀pọ̀ Owó (**Ultimate Multisite → Ọ̀pọ̀ Owó**) ni ibi àárín gbùngbùn láti:

- Ṣàfikún, mú ṣiṣẹ́, tàbí pa àwọn owó
- Yan olùpèsè oṣùwọ̀n pàṣípààrọ̀ kan, kí o sì tún àwọn oṣùwọ̀n ṣe nígbà tí o bá fẹ́
- Ṣàyẹ̀wò àwọn oṣùwọ̀n lọwọlọwọ fún gbogbo àwọn owó tí ń ṣiṣẹ́
- Ṣètò àwọn oṣùwọ̀n ọwọ́ nígbà tí àwọn olùpèsè laifọwọyi kò bá yẹ

### Àwọn Olùpèsè Oṣùwọ̀n Pàṣípààrọ̀

| Olùpèsè | Àpèjúwe |
|---|---|
| **ECB** | European Central Bank — àwọn oṣùwọ̀n ojoojúmọ́ ọ̀fẹ́ fún àwọn nẹ́tíwọ́ọ̀kì tó dá lórí EUR |
| **Frankfurter** | API dígí ECB onísọ́fitiwia-ọ̀fẹ́ — ọ̀fẹ́, kò nílò kọ́kọ́rọ́ API |
| **Stripe** | Ó ń lo àwọn oṣùwọ̀n FX láàyè ti Stripe — ó nílò ìsopọ̀ Stripe tí ń ṣiṣẹ́ |
| **Ọwọ́** | Ìwọ ló ń tẹ àwọn oṣùwọ̀n wọlé taara; kò sí ìpè API òde |

Yan olùpèsè rẹ lábẹ́ **Ultimate Multisite → Ọ̀pọ̀ Owó → Ìṣètò → Olùpèsè Oṣùwọ̀n Pàṣípààrọ̀**. A lè tún àwọn oṣùwọ̀n ṣe lọ́wọ́ nígbàkigbà tàbí ṣètò wọn kí wọ́n máa ṣe imudojuiwọn laifọwọyi.

### Ìforúkọsílẹ̀ Iye Owó Ọjà Kọọkan

Nígbà tí ìyípadà laifọwọyi kò bá pé tó, o lè ṣètò iye owó gangan fún owó pàtó kan lori ọjà èyíkéyìí:

1. Ṣí olóòtú ọjà (**Ultimate Multisite → Àwọn Ọjà → Ṣàtúnṣe**)
2. Yi lọ sí apá **Ìdíyelé Owó**
3. Tẹ iye owó gangan wọlé fún owó kọọkan tí o fẹ́ forúkọsílẹ̀ dípò
4. Fi ọjà náà pamọ́

Àwọn ọjà tí kò ní ìforúkọsílẹ̀ dípò ń lo iye owó tí a yí padà láti owó ìpìlẹ̀ rẹ.

### Pápá Ìforúkọsílẹ̀ Aṣàyàn Owó

A lè ṣàfikún pápá aṣàyàn owó sí ìṣàn ìforúkọsílẹ̀/ìsanwó rẹ kí àwọn oníbàárà lè yan owó tí wọ́n fẹ́ràn nígbà ìforúkọsílẹ̀:

1. Lọ sí **Ultimate Multisite → Ọ̀pọ̀ Owó → Ìṣètò**
2. Mú **Fi aṣàyàn owó hàn ní ìforúkọsílẹ̀** ṣiṣẹ́
3. Bí o bá fẹ́, ṣètò owó aiyipada tàbí jẹ́ kí geolocation pinnu iye tí a yan ṣáájú

Owó tí a yan ni a ń tọ́jú pẹ̀lú subscription, a sì ń lo ó fún gbogbo ìdíyelé tó tẹ̀lé.

### Ìṣàwárí Owó Tó Dá Lórí Geolocation

Nígbà tí ìṣàwárí geolocation bá ṣiṣẹ́, addon náà ń bá orílẹ̀-èdè alejo mu pẹ̀lú owó kan, ó sì ń yan án ṣáájú nínú pápá aṣàyàn owó. Láti mú un ṣiṣẹ́:

1. Lọ sí **Ultimate Multisite → Ọ̀pọ̀ Owó → Ìṣètò**
2. Mú **Ṣàwárí owó láti ipo alejo** ṣiṣẹ́
3. Ṣe maapu àwọn orílẹ̀-èdè sí àwọn owó nínú tábìlì **Maapu Geolocation**

Bí kò bá sí maapu fún orílẹ̀-èdè alejo kan, owó ìpìlẹ̀ nẹ́tíwọ́ọ̀kì ni a máa lo gẹ́gẹ́ bí àṣàyàn ìpẹ̀yà.

## Àwọn Ìbéèrè Tí A Máa Ń Béèrè Lọ́pọ̀ Ìgbà

### Kí ló máa ṣẹlẹ̀ bí olùpèsè oṣùwọ̀n pàṣípààrọ̀ kò bá sí nílẹ̀?

Addon náà máa padà sí àwọn oṣùwọ̀n tí a gba ní aṣeyọrí kẹ́yìn. Bí a kò bá tíì gba oṣùwọ̀n kankan rí, yóò padà sí àwọn oṣùwọ̀n ọwọ́ tí o ti ṣètò. A ṣàbẹ̀wò pé kí o ṣètò àwọn oṣùwọ̀n ọwọ́ gẹ́gẹ́ bí ààbò àfikún.

### Ṣé mo lè lo àwọn olùpèsè tó yàtọ̀ fún àwọn owó tó yàtọ̀?

Rárá — olùpèsè kan ṣoṣo ni ó ń ṣiṣẹ́ nígbà kan fún gbogbo àwọn owó. Lo **Ọwọ́** bí o bá nílò ìṣàkóso pátápátá lori àwọn oṣùwọ̀n kọọkan.

### Ṣé geolocation nílò plugin míì lọ́tọ̀?

Kò sí plugin geolocation òde tí a nílò. Addon náà ń lo ìṣàwárí IP-sí-orílẹ̀-èdè tí WordPress ní sínú. Fún ìpéye tó ga, a lè so iṣẹ́ geolocation pàtó pọ̀ nípasẹ̀ àwọn hook àlẹmọ tí a pèsè.

### Ṣé a ń lo ìforúkọsílẹ̀ dípò ọjà kọọkan ṣáájú tàbí lẹ́yìn owó-orí?

Ìforúkọsílẹ̀ dípò ọjà kọọkan ń ṣètò iye owó ṣáájú owó-orí nínú owó àfojúsùn. A ń ṣírò owó-orí lórí iye owó ìforúkọsílẹ̀ dípò ní lílo ìṣètò owó-orí tó wà lọwọlọwọ rẹ.
