# Àwọn Aṣojú Ìtìlẹ́yìn

Ṣíṣàkóso gbogbo nẹ́tíwọ́ọ̀kì kan fúnra rẹ kì í ṣe iṣẹ́ rọrùn. Bí nẹ́tíwọ́ọ̀kì rẹ ṣe ń bẹ̀rẹ̀ sí í dàgbà, ìwọ yóò bẹ̀rẹ̀ sí í rí ìbéèrè ìtìlẹ́yìn tó ń pọ̀ sí i. Yóò jẹ́ ìmọ̀ràn rere láti mú àwọn ènìyàn wá tí yóò ràn ọ́ lọ́wọ́ láti dín ẹrù iṣẹ́ rẹ kù. Síbẹ̀, fífúnni ní ìwọlé/àṣẹ kíkún lewu, ó sì lè ba nẹ́tíwọ́ọ̀kì rẹ jẹ́. Ibẹ̀ ni add-on Àwọn Aṣojú Ìtìlẹ́yìn yìí ti wúlò.

Add-on yìí jẹ́ kí o lè dá irú olumulo WordPress tuntun kan sílẹ̀ tí a ń pè ní Aṣojú Ìtìlẹ́yìn, ó sì fún ọ ní àkóso pátápátá lórí àwọn àṣẹ tí olumulo náà ní. Fífi àṣẹ fúnni rọrùn bí ṣíṣi bọtini yípadà sílẹ̀, àti pípa á jẹ́ pé o pinnu láti yọ ọ́ kúrò.

## Àwọn Ẹ̀yà

- Àtòjọ ohun kọọkan ti àṣẹ WordPress & Ultimate Multisite
- Mú ìwọlé ṣiṣẹ́ & pa á pẹ̀lú ìrọ̀rùn
- Dá iye àwọn Account Aṣojú tí kò lopin sílẹ̀

## Fífìsílẹ̀

1. Ṣàgbéjáde 'ultimate-multisite-support-agents' sí directory '/wp-content/plugins/'
2. Mú àfikún náà ṣiṣẹ́ nípasẹ̀ menu 'Àwọn Àfikún' ninu WordPress
3. Tẹ̀lé olùrànlọ́wọ́ ìgbésẹ̀-ní-ìgbésẹ̀ láti ṣètò àfikún náà
