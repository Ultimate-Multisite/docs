# Loco Translate

Afikun ìṣepọ̀ Ultimate Multisite: Loco Translate ń mú agbára ìtumọ̀ tó jẹ mọ́ oju opo kọọkan wá sí nẹtiwọọki multisite WordPress rẹ. Tí o bá ti nílò kí àwọn oju opo oriṣiriṣi nínú nẹtiwọọki rẹ ní àwọn ìtumọ̀ oriṣiriṣi fún afikun tàbí akori kan náà rí, afikun yìí ni ojútùú tó péye.

**Ohun Tí Afikun Yìí Ṣe**

Afikun yìí ń ṣepọ̀ Ultimate Multisite pẹ̀lú afikun Loco Translate tó gbajúmọ̀ lọ́nà aláìní ìdíwọ́, ó sì ń jẹ́ kí oju opo abẹ́ kọọkan nínú nẹtiwọọki rẹ lè pa àwọn ìtumọ̀ àṣà tirẹ̀ mọ́. Bóyá o ń ṣiṣẹ́ pẹpẹ agbègbè-púpọ̀, o ń sin àwọn àwùjọ èdè oriṣiriṣi, tàbí o kan nílò ìrọ̀rùn ìtumọ̀ káàkiri nẹtiwọọki rẹ, afikun yìí mú un rọrùn.

**Ó Péye Fún:**

* Àwọn iṣowo agbègbè-púpọ̀ pẹ̀lú àwọn àìní ìṣe-àdágbè tó yàtọ̀
* Àwọn ilé-ẹ̀kọ́ tó ń sin àwọn àwùjọ onírúurú
* Àwọn pẹpẹ SaaS tó ń fúnni ní àtìlẹ́yìn ọ̀pọ̀ èdè
* Àwọn ajọ wẹ́ẹ̀bù tó ń ṣàkóso ọ̀pọ̀ oju opo oníbàárà
* Eyikeyi nẹtiwọọki multisite tó nílò ìrọ̀rùn ìtumọ̀

## Fífì Sọ́ọ̀lù

1. Rí dájú pé Ultimate Multisite ti fi sílẹ̀, ó sì ti ṣiṣẹ́ lórí nẹtiwọọki
2. Fi afikun Loco Translate sílẹ̀, kí o sì mú un ṣiṣẹ́
3. Gbé afikun yìí sí àkójọ plugins rẹ
4. Mú afikun ìṣepọ̀ Loco Translate ṣiṣẹ́ lórí nẹtiwọọki
5. Bẹ̀rẹ̀ sí í dá àwọn ìtumọ̀ tó jẹ mọ́ oju opo kọọkan ṣẹ nípasẹ̀ Loco Translate lórí eyikeyi oju opo abẹ́

Ìyẹn nìyẹn! Ìṣepọ̀ náà ń ṣiṣẹ́ fúnra rẹ̀ láìsí àtúnṣe míì kankan.

## Àwọn Ìbéèrè Tí A Máa ń Béèrè Léraléra

## Ṣé mo nílò láti tún ohunkóhun ṣe lẹ́yìn fífì sílẹ̀?

Rárá! Ìṣepọ̀ náà jẹ́ aládàáṣiṣẹ́. Nígbà tí afikun náà àti Loco Translate bá ti ṣiṣẹ́, àwọn oju opo abẹ́ lè bẹ̀rẹ̀ lẹ́sẹ̀kẹsẹ̀ sí í dá àwọn ìtumọ̀ àṣà tiwọn ṣẹ.

## Kí ló máa ṣẹlẹ̀ tí oju opo abẹ́ kan kò bá dá àwọn ìtumọ̀ àṣà ṣẹ?

Oju opo náà yóò lo àwọn ìtumọ̀ gbogbo-nẹtiwọọki fúnra rẹ̀ gẹ́gẹ́ bí ìdápadà. Kò sí ohun tó máa bàjẹ́, ohun gbogbo sì máa ṣiṣẹ́ lọ́nà aláìní ìdíwọ́.

## Ṣé àwọn oju opo abẹ́ lè wọlé sí àwọn ìtumọ̀ gbogbo-nẹtiwọọki?

Àwọn alákóso oju opo deede máa ń rí, wọ́n sì máa ń ṣàtúnṣe àwọn ìtumọ̀ oju opo tiwọn nìkan. Àwọn super admins lè wọlé sí ìtumọ̀ gbogbo-nẹtiwọọki àti ìtumọ̀ tó jẹ mọ́ oju opo kọọkan, gẹ́gẹ́ bí oju opo tí wọ́n ń wò.

## Ṣé èyí ń ṣiṣẹ́ pẹ̀lú àwọn afikun àti àwọn akori méjèèjì?

Bẹ́ẹ̀ni! Àwọn ìtumọ̀ tó jẹ mọ́ oju opo kọọkan ń ṣiṣẹ́ fún àwọn afikun WordPress àti àwọn akori méjèèjì.

## Ṣé èyí yóò fa fifalẹ nẹtiwọọki multisite mi?

Rárá. Afikun náà ń lo ìkójọpọ̀ fáìlì tó munadoko, èyí tó máa ń ṣàyẹ̀wò àwọn ìtumọ̀ tó jẹ mọ́ oju opo kọọkan nígbà tí a bá nílò wọn nìkan. Kò sí ipa kankan tó ṣeé fojú rí lórí iṣẹ́ ṣiṣe.

## Ṣé mo lè da àwọn ìtumọ̀ nẹtiwọọki sí oju opo kan pàtó gẹ́gẹ́ bí ibi ìbẹ̀rẹ̀?

Bẹ́ẹ̀ni! Nígbà tí o bá ń ṣàtúnṣe àwọn ìtumọ̀ nínú Loco Translate, o lè bẹ̀rẹ̀ pẹ̀lú àwọn ìtumọ̀ nẹtiwọọki ní rọrùn, kí o sì ṣe wọn ní àṣà fún oju opo kọọkan.

## Ṣé èyí ń ṣiṣẹ́ pẹ̀lú WooCommerce àti àwọn afikun míì?

Dájúdájú! Eyikeyi afikun tàbí akori tó ń lo àwọn iṣẹ́ ìtumọ̀ boṣewa WordPress yóò ṣiṣẹ́ pẹ̀lú àwọn ìtumọ̀ tó jẹ mọ́ oju opo kọọkan.
