# Filter: wp_ultimo_woocommerce_checkout_fields_to_clean

Ó jẹ́ kí àwọn olùgbéejáde lè filter àwọn field àdírẹ́ẹ̀sì ìsanwó WooCommerce aiyipada.

Nítorí Ultimate Multisite ní àwọn field àdírẹ́ẹ̀sì ìsanwó, ó lè bọ́gbọ́n mu láti yọ àwọn field aiyipada tí WooCommerce fi hàn lórí fọ́ọ̀mù checkout kúrò. Ní aiyipada, kò sí iye tí a fi ránṣẹ́ sí àkójọ yìí, nítorí náà gbogbo field tí a ṣètò lórí WooCommerce láti fi hàn ni a pa mọ́.

### Láti

- 2.0.0
### Orísun

A ṣàlàyé rẹ̀ nínú [`inc/gateways/class-woocommerce-gateway.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-woocommerce/blob/main/inc/gateways/class-woocommerce-gateway.php#L277) ní ìlà 277


## Àwọn ìpadà
Àkójọ àwọn field láti yọ kúrò. Àwọn iye tó wà nílẹ̀ pẹ̀lú billing_first_name, billing_last_name, billing_company, billing_country, billing_address_1, billing_address_2, billing_city, billing_state, billing_postcode, billing_email, àti billing_phone.
