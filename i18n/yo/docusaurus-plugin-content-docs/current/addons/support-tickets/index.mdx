# Tikẹti Atilẹyin {#support-tickets}

**Tikẹti Atilẹyin** jẹ addon alágbára fún Ultimate Multisite tó ń ṣàfikún eto tikẹti atilẹyin tó péye sí nẹ́tíwọọki WordPress rẹ. Ó dára gan-an fún pẹpẹ SaaS, àwọn olùpèsè WaaS (Website as a Service), àti àwọn nẹ́tíwọọki multisite tó nílò láti pèsè atilẹyin oníbàárà ní ipele nẹ́tíwọọki àti ní ipele site kọ̀ọ̀kan.

## Kí Ló Dé Tí Wàá Fi Yan Tikẹti Atilẹyin? {#why-choose-support-tickets}

* **Atilẹyin Ipele-Púpọ̀**: Àwọn alábòójútó nẹ́tíwọọki ń bójú tó àwọn tikẹti tó kan gbogbo nẹ́tíwọọki, nígbà tí àwọn oní site ń ṣàkóso atilẹyin oníbàárà tiwọn
* **Ìṣàn Iṣẹ́ Amọ̀ṣẹ́dájú**: Ìṣàkóso ìgbésí-ayé tikẹti ní kíkún pẹ̀lú àwọn àkọ́kọ́ṣe, àwọn ipò, àti àwọn ìpín iṣẹ́
* **Ìrírí Oníbàárà**: Ojú-ìwòye tó rẹwà, tó sì ń dáhùn dáadáa, tó mú kó rọrùn fún àwọn oníbàárà láti gba ìrànlọ́wọ́
* **Ailewu & Gbẹ́kẹ̀lé**: A kọ́ ọ pẹ̀lú àwọn ìlànà ààbò WordPress tó dára jù lọ, a sì ṣe àtúnṣe fún iṣẹ́ database tó péye
* **Ìṣepọ̀ Láìní Ìdíwọ́**: Ó ń ṣiṣẹ́ pẹ̀lú ìṣàkóso oníbàárà àti site ti Ultimate Multisite láìsí ìṣòro

## Àwọn Ẹ̀yà Pàtàkì {#key-features}

**Ìṣàkóso Tikẹti Ní Kíkún**

* Eto nọ́mbà tikẹti aláìlẹgbẹ́ fún ìtọ́kasí tó rọrùn
* Àwọn ipele àkọ́kọ́ṣe tí a lè ṣètò (Kéré, Déédéé, Gíga, Pàtàkì Gan-an)
* Ìṣàn iṣẹ́ ipò tó péye (Ṣí, Ń lọ lọ́wọ́, Ń dúró, Ti yanjú, Ti pa)
* Ìsọ̀rí tikẹti (Gbogbogbò, Ìmọ̀-ẹ̀rọ, Ìsanwó, Ìbéèrè Ẹ̀yà, Ìròyìn Àṣìṣe)
* Pín àwọn tikẹti sí àwọn aṣojú atilẹyin pẹ̀lú àṣàyàn ìpín-láìfọwọ́wọ́

**Àwọn Ìjíròrò Olóògùn**

* Ìlànà ìjíròrò ní kíkún fún àwọn ìdáhùn tikẹti
* Ìṣàfihàn tó yàtọ̀ fún àwọn ìdáhùn òṣìṣẹ́ àti oníbàárà
* Àwọn akọsilẹ̀ òṣìṣẹ́ aládáni tí àwọn oníbàárà kò lè rí
* Ojú-ìwòye tó ń lo AJAX ní àkókò gidi
* Olootu ọ̀rọ̀ òde-òní pẹ̀lú àwọn àṣàyàn ìṣàtúnṣe ìrísí

**Àwọn Àfikún Fáìlì**

* Ìgbéjáde fáìlì tó ní ààbò pẹ̀lú atilẹyin fífà-ati-fí-sílẹ̀
* Àwọn ìhámọ́ irú fáìlì àti ìwọ̀n fáìlì tí a lè ṣètò
* Ààbò tó gíga pẹ̀lú ìwárí fáìlì oníbàjẹ́
* Àwọn ìgbàsílẹ̀ tó ni ààbò pẹ̀lú ìṣàkóso ìwọlé

**Àwọn Ìfitónilétí Ọlọ́gbọ́n**

* Àwọn ìkìlọ̀ email aládàáṣiṣẹ́ fún àwọn tikẹti tuntun, àwọn ìdáhùn, àti àwọn ayípadà ipò
* Àwọn template email tí a lè ṣàtúnṣe
* Àwọn ìfitónilétí tó dá lórí ipa fún òṣìṣẹ́ àti àwọn oníbàárà
* Àwọn oniyí template aláyípadà fún àwọn ìfiránṣẹ́ ẹni-kọ̀ọ̀kan

**Ojú-ìwòye Alábòójútó Alágbára**

* Àtòkọ tikẹti tó gíga pẹ̀lú títò àti fífi àlẹ́mọ́
* Ìṣàtúnṣe kíákíá fún ipò, àkọ́kọ́ṣe, àti àwọn ìpín iṣẹ́
* Àwọn ìgbésẹ̀ ọ̀pọ̀ fún ìṣàkóso tó munadoko
* Dashboard ìṣirò pẹ̀lú àwọn ìwọ̀n iṣẹ́
* Panel eto tó péye

**Portal Oníbàárà Tó Rẹwà**

* Ojú-ìwòye tó mọ́, tó ń dáhùn dáadáa lórí gbogbo àwọn ẹrọ
* Àpẹrẹ tó kọ́kọ́ ráyè fún mobile
* Ìwádìí àti fífi àlẹ́mọ́ tó gíga
* Olootu òde-òní pẹ̀lú ọpa irinṣẹ́ ìṣàtúnṣe ìrísí
* Àwọn fọ́ọ̀mù fífi tikẹti ránṣẹ́ tó rọrùn láti lò

**Ìṣepọ̀ Tó Rọrùn**

* Lo shortcodes tàbí àwọn èròjà UI Ultimate Multisite
* Ṣàfihàn àwọn fọ́ọ̀mù tikẹti àti àwọn àtòkọ ní ibikíbi lórí site rẹ
* Ó ń ṣiṣẹ́ láìsí ìṣòro pẹ̀lú ìkọ̀lé nẹ́tíwọọki multisite
* Ó lè gbòòrò pẹ̀lú hooks àti filters

## Ó Dára Fún {#perfect-for}

* Àwọn pẹpẹ SaaS tó nílò atilẹyin oníbàárà
* Àwọn olùpèsè WaaS tó ń ṣàkóso ọ̀pọ̀ site oníbàárà
* Àwọn site membership pẹ̀lú atilẹyin ìpele
* Àwọn alábòójútó nẹ́tíwọọki tó ń ṣàkóso ọ̀pọ̀ brand
* Àwọn oní site tó ń pèsè iṣẹ́ oníbàárà

## Ìfíṣípò {#installation}

1. Rí i dájú pé plugin Ultimate Multisite ti wà ní fífiṣípò, ó sì ti ṣiṣẹ́ lórí nẹ́tíwọọki WordPress rẹ
2. Gbé àwọn fáìlì addon náà sí directory `/wp-content/plugins/` rẹ
3. Mú plugin náà ṣiṣẹ́ fún nẹ́tíwọọki nípasẹ̀ menu 'Plugins' nínú WordPress
4. Lọ sí Network Admin > Support Tickets láti ṣètò àwọn eto rẹ

## Àwọn Ìbéèrè Tí A Máa Ń Béèrè Léraléra {#frequently-asked-questions}

## Ṣé èyí nílò Ultimate Multisite? {#does-this-require-ultimate-multisite}

Bẹ́ẹ̀ni, addon yìí nílò kí Ultimate Multisite wà ní fífiṣípò, kí ó sì máa ṣiṣẹ́. Ó ní ìṣepọ̀ jinlẹ̀ pẹ̀lú àwọn ẹ̀yà ìṣàkóso oníbàárà àti site ti Ultimate Multisite.

## Ṣé àwọn oní site lè bójú tó atilẹyin oníbàárà tiwọn? {#can-site-owners-handle-their-own-customer-support}

Dájúdájú! Àwọn oní site lè lo eto tikẹti náà láti pèsè atilẹyin fún àwọn oníbàárà tiwọn, nígbà tí àwọn alábòójútó nẹ́tíwọọki lè bójú tó àwọn tikẹti atilẹyin tó kan gbogbo nẹ́tíwọọki.

## Báwo ni àwọn oníbàárà ṣe ń fi tikẹti ránṣẹ́? {#how-do-customers-submit-tickets}

O lè ṣàfikún àwọn fọ́ọ̀mù fífi tikẹti ránṣẹ́ ní ibikíbi lórí site rẹ ní lílo shortcodes tàbí àwọn èròjà UI Ultimate Multisite. Àwọn oníbàárà kan máa kún fọ́ọ̀mù náà, wọ́n á sì fi ìbéèrè wọn ránṣẹ́.

## Ṣé mo lè ṣàtúnṣe àwọn ipò tikẹti àti àwọn àkọ́kọ́ṣe? {#can-i-customize-the-ticket-statuses-and-priorities}

Bẹ́ẹ̀ni! O lè ṣàtúnṣe àwọn ipele àkọ́kọ́ṣe, àwọn ipò, àti àwọn irú tikẹti ní kíkún nínú àwọn eto alábòójútó nẹ́tíwọọki láti bá ìṣàn iṣẹ́ atilẹyin rẹ mu.

## Ṣé ó ṣe atilẹyin àwọn àfikún fáìlì? {#does-it-support-file-attachments}

Bẹ́ẹ̀ni, àwọn oníbàárà àti òṣìṣẹ́ lè so àwọn fáìlì mọ́ àwọn tikẹti àti àwọn ìdáhùn. O ní ìṣàkóso kíkún lórí àwọn irú fáìlì tí a gbà láàyè àti àwọn ààlà ìwọ̀n fún ààbò.

## Báwo ni a ṣe ń bójú tó àwọn ìfitónilétí? {#how-are-notifications-handled}

Eto náà máa ń fi àwọn ìfitónilétí email ránṣẹ́ laifọwọ́wọ́ fún àwọn tikẹti tuntun, àwọn ìdáhùn, àti àwọn ayípadà ipò. O lè ṣàtúnṣe gbogbo àwọn template email nínú àwọn eto.

## Ṣé mo lè pín àwọn tikẹti sí àwọn aṣojú atilẹyin kan pàtó? {#can-i-assign-tickets-to-specific-support-agents}

Bẹ́ẹ̀ni, o lè fi ọwọ́ pín àwọn tikẹti sí àwọn aṣojú tàbí mu ìpín-láìfọwọ́wọ́ ṣiṣẹ́ láti pín àwọn tikẹti ká laifọwọ́wọ́.

## Ṣé àwọn akọsilẹ̀ aládáni wà fún òṣìṣẹ́? {#are-there-private-notes-for-staff}

Bẹ́ẹ̀ni, àwọn ọmọ ẹgbẹ́ òṣìṣẹ́ lè ṣàfikún àwọn ìdáhùn aládáni tí àwọn ọmọ ẹgbẹ́ òṣìṣẹ́ míì nìkan lè rí, kì í ṣe àwọn oníbàárà.

## Ṣé ó bá mobile mu? {#is-it-mobile-friendly}

Dájúdájú! Gbogbo ojú-ìwòye náà ni a kọ́ pẹ̀lú ọ̀nà tí ó kọ́kọ́ ráyè fún mobile, ó sì ń ṣiṣẹ́ dáadáa lórí gbogbo àwọn ẹrọ.

## Ṣé mo lè ṣàfihàn àwọn àtòkọ tikẹti àti àwọn fọ́ọ̀mù ní lílo shortcodes? {#can-i-display-ticket-lists-and-forms-using-shortcodes}

Bẹ́ẹ̀ni! Àwọn shortcodes tó wà:
* `[wu_support_tickets]` - Ṣàfihàn àtòkọ tikẹti oníbàárà
* `[wu_support_ticket_form]` - Ṣàfihàn fọ́ọ̀mù fífi tikẹti tuntun ránṣẹ́
* `[wu_support_ticket_view ticket_id="123"]` - Ṣàfihàn tikẹti kan pàtó

## Irú fáìlì wo ni a lè gbé jáde? {#what-file-types-can-be-uploaded}

O lè ṣètò àwọn irú fáìlì tí a gbà láàyè nínú àwọn eto. Eto náà ní ìṣàyẹ̀wò ààbò tó gíga láti dáàbò bo lòdì sí àwọn fáìlì oníbàjẹ́.

## Ṣé mo lè export data tikẹti? {#can-i-export-ticket-data}

Bẹ́ẹ̀ni, addon náà ní agbára export fún data tikẹti àti àwọn ìṣirò.
