# Material WP {#material-wp}

Àwòṣe ìṣàkóso WordPress ti jẹ́ bakan náà fún ìgbà pípẹ́. Ṣùgbọ́n ṣé o ti rò rí bí ẹ̀yà Material Design ti àwòṣe ìṣàkóso yóò ṣe rí? 

Material WP jẹ́ theme ìṣàkóso tí a dá sílẹ̀ láti fún ọ ní ìrísí àti ìmọ̀lára yẹn, tí ń ràn ọ́ lọ́wọ́ láti pèsè Dashboard tí a ṣe àdáṣe síi fún àwọn oníbàárà rẹ.

## Àwọn Ẹ̀ya {#features}

- Àtìlẹ́yìn CSS àdáṣe
- Àwọn ètò ìṣàṣà rọrùn
- Títún àtòjọ menu ṣe
- Ìgbéwọlé/ìgbéjáde rọrùn

## Ìwé Ìtọ́nisọ́nà {#documentation}

**Ṣíṣe Admin Panel rẹ ní àdáṣe pẹ̀lú Material WP**

## Fífi sílẹ̀ {#installation}

1. Gbé 'material-wp' sí directory '/wp-content/plugins/'
2. Mú plugin náà ṣiṣẹ́ nípasẹ̀ menu 'Àwọn Plugin' nínú WordPress
