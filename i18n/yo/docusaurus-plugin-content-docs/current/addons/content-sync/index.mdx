# Amuṣiṣẹpọ Akoonu

Multisite Ultimate: Amuṣiṣẹpọ Akoonu

Àfikún yìí fún Multisite Ultimate ń jẹ́ kí àwọn alábòójútó nẹ́tíwọ́ọ̀kì lè mú akoonu ṣiṣẹ́ pọ̀ láti ojúlé èyíkéyìí sí àwọn ojúlé míràn nínú nẹ́tíwọ́ọ̀kì. Ó ṣe àtìlẹ́yìn fún gbogbo irú àtẹ̀jáde, pẹ̀lú àwọn àtẹ̀jáde, àwọn ojúewé, àwọn irú àtẹ̀jáde àdáni, àti àwọn template Elementor. Àfikún náà ní atọ́nà wizard alá-ìgbésẹ̀-púpọ̀ fún yíyan akoonu àti àwọn ojúlé àfojúsùn, pẹ̀lú ìṣiṣẹ́ batch àti ìfèsì ìlọsíwájú ní àkókò gidi.

## Àwọn Ẹ̀yà

* Yíyan Akoonu: Grid kaadi ojúran pẹ̀lú àwọn thumbnail àti fíltà irú àtẹ̀jáde
* Gbogbo Irú Àtẹ̀jáde: Mú àwọn àtẹ̀jáde, àwọn ojúewé, àwọn ọjà, àwọn template Elementor, àti irú àtẹ̀jáde àdáni èyíkéyìí ṣiṣẹ́ pọ̀
* Yíyan Ojúlé: Yan àwọn ojúlé tí a ó mú akoonu ṣiṣẹ́ pọ̀ sí, pẹ̀lú yíyọ ojúlé orísun kúrò àti yíyan àwọn ojúlé ọmọ ṣáájú
* Ìṣiṣẹ́ Batch: Ń mú ṣiṣẹ́ pọ̀ sí ojúlé 10 ní ẹ̀ẹ̀kan pẹ̀lú àwọn ìmúdójúìwọ̀n ìlọsíwájú
* Amuṣiṣẹpọ Media: Ń daakọ àti tún maapu àwọn fáìlì media (àwọn àwòrán, àwọn fídíò) láti inú akoonu àtẹ̀jáde sí àwọn ojúlé àfojúsùn laifọwọyi
* Amuṣiṣẹpọ Taxonomy: Ń pa àwọn ẹ̀ka, àwọn àmì, àti àwọn taxonomy àdáni mọ́
* Àtìlẹ́yìn Elementor: Àtìlẹ́yìn kíkún fún àwọn template Elementor pẹ̀lú àtúndá CSS àti àtúnmaapu ID template
* Yíyanjú Ìkọlù: Àṣàyàn láti rọ́pò akoonu tó wà tẹ́lẹ̀ tàbí láti fò ó kọjá
* Ìṣàkóso Cache: Ń pa cache Elementor rẹ́ laifọwọyi lórí àwọn ojúlé àfojúsùn lẹ́yìn amuṣiṣẹpọ

## Àwọn Ohun Tí A Nílò

* WordPress 5.3 tàbí ju bẹ́ẹ̀ lọ
* PHP 7.4 tàbí ju bẹ́ẹ̀ lọ
* plugin Multisite Ultimate (tó ń ṣiṣẹ́)
* plugin Elementor (àṣàyàn, a nílò rẹ̀ fún amuṣiṣẹpọ template Elementor)

## Fífi Sọ́wọ́

1. Gbé 'ultimate-multisite-content-sync' sí directory '/wp-content/plugins/'
2. Mú plugin náà ṣiṣẹ́ nípasẹ̀ menu 'Àwọn Plugin' nínú Alábòójútó Nẹ́tíwọ́ọ̀kì WordPress rẹ
3. Ṣàtúnṣe àwọn ètò nínú Ultimate Multisite → Àwọn Ètò → Amuṣiṣẹpọ Akoonu
4. Lọ sí Ultimate Multisite → Amuṣiṣẹpọ Akoonu láti bẹ̀rẹ̀ amuṣiṣẹpọ
