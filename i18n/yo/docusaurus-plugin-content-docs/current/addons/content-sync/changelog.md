# Àkọsílẹ̀ Àyípadà Content Sync

### Ẹ̀yà 1.1.0 - Tú jáde ní 2025-12-31 {#version-110---released-on-2025-12-31}
* Tuntun: Àtìlẹ́yìn fún mímú gbogbo irú ìfìwéránṣẹ́ bá ara wọn mu (àwọn ìfìwéránṣẹ́, àwọn ojúewé, àwọn irú ìfìwéránṣẹ́ àkànṣe)
* Tuntun: Ìyàn irú ìfìwéránṣẹ́ nínú ìgbésẹ̀ yíyan àkóónú
* Tuntun: Ìgbésẹ̀ yíyan site orísun - mú bá ara wọn mu láti site èyíkéyìí, kì í ṣe site àkọ́kọ́ nìkan
* Tuntun: Yíyan site ọmọ ṣáájú fún àwọn site àdàkọ
* Tuntun: Yíyọ mídíà jáde láti post_content fún gbogbo irú ìfìwéránṣẹ́
* Tuntun: Mímú taxonomy bá ara wọn mu (àwọn ẹ̀ka, àwọn táàgì, àwọn taxonomy àkànṣe)
* Yípadà: Tún orúkọ rẹ̀ ṣe láti "Elementor Template Sync" sí "Content Sync"
* Yípadà: Ìgbésẹ̀ "Yan Templates" ni a tún sọ orúkọ rẹ̀ di "Yan Data"
* Dára síi: Kóòdù tó jẹ́ ti Elementor pàtó báyìí ń ṣiṣẹ́ fún irú ìfìwéránṣẹ́ elementor_library nìkan

### Ẹ̀yà 1.0.0 - Tú jáde ní 2025-12-19 {#version-100---released-on-2025-12-19}
* Ìtúsílẹ̀ àkọ́kọ́
* Wizard oní-ìgbésẹ̀ púpọ̀ fún yíyan template àti site
* Ṣíṣe ní ìpele pẹ̀lú ìdáhùn ìtẹ̀síwájú ní àkókò gidi
* Mímú mídíà kíkún bá ara wọn mu pẹ̀lú àtúntò ID aládàáṣiṣẹ́
* Ìṣepọ̀ àwọn settings pẹ̀lú Ultimate Multisite
* Àtìlẹ́yìn fún yíyọ site kúrò nípasẹ̀ àwọn settings network
* Pípa cache Elementor mọ́ lẹ́yìn mímú bá ara wọn mu
* Ìtọ́jú taxonomy template
