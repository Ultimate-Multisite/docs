# Ẹlẹda Oju-iwe Admin

## Àkóónú {#content}
Ẹlẹda oju-iwe admin yoo jẹ ki o le fi awọn oju-iwe admin tuntun ati awọn oju-iwe abẹ́ kun ni irọrun laarin WordPress admin dashboard. A le lo o lati ṣẹda oju-iwe kaabọ tabi oju-iwe ẹkọ ti o le ṣiṣẹ gẹgẹ bi oju-iwe ibalẹ fun awọn olumulo tuntun. O le lo akọ́ oju-iwe bii Beaver Builder, Elementor, Brizy, ati Oxygen lati ṣe apẹrẹ awọn oju-iwe naa laisi wahala.

Ó tún nfunni ni atilẹyin HTML, CSS ati JavaScript ti a ṣe adani, bakanna pẹlu iṣakoso wiwọle ti o da lori eto ati ipa olumulo!

## Àwọn Àmúyẹ {#features}
* Ṣe atilẹyin HTML ati CSS ti a ṣe adani
* Atilẹyin JavaScript
* Ipo Àkóónú PHP
* Ìyọ̀nda ti o da lori ipa Olumulo
* Atilẹyin fun awọn akọ́ oju-iwe

## Fifi sori ẹrọ {#installation}

1. Gbe **ultimate-multisite-admin-page-creator** si itọsọna **/wp-content/plugins/**
2. Mu afikun naa ṣiṣẹ nipasẹ akojọ **Àwọn Afikun** ninu WordPress Network Admin rẹ

## Ìtọ́nisọ́nà Lilo {#usage-guide}

Lakoko ti WordPress jẹ ki o rọrun gan lati fi àkóónú kun si apakan iwaju awọn aaye, ṣiṣẹda awọn oju-iwe ati awọn oju-iwe abẹ́ ni ẹgbẹ admin nilo diẹ ninu awọn ọgbọn kóòdù ipilẹ. Ultimate Multisite: Ẹlẹda Oju-iwe Admin ni ero lati mu irọrun lilo kanna ti apakan iwaju wa si apakan ẹhin awọn aaye nẹtiwọọki rẹ.

## Kí nìdí tí a fi máa lo Ẹlẹda Oju-iwe Admin? {#why-use-admin-page-creator}

Boya o fẹ lati funni ni oju-iwe ti a ṣe adani lori panẹli admin awọn alabara rẹ pẹlu awọn ẹkọ, boya o fẹ lati fi oju-iwe ti a ṣe adani kun pẹlu awọn FAQ ati ẹrọ olubasọrọ. Awọn aye ṣeeṣe ko lopin pẹlu Ẹlẹda Oju-iwe Admin, ati pe iyẹn ni ohun ti a yoo ṣawari ninu ẹkọ oni.

## Àwọn Aṣayan Akojọ {#menu-options}

Ni ẹgbẹ, lori apoti meta Awọn Aṣayan Gbogbogbo, iwọ yoo le ṣatunto iru akojọ, aami, ati pupọ diẹ sii.

### Àwọn Iru Akojọ {#menu-types}

Oju-iwe admin tuntun ti a ṣe adani rẹ le ni iru akojọ meji oriṣiriṣi: akojọ ipele-oke ati akojọ abẹ́.

Oju-iwe akojọ ipele-oke, gẹgẹ bi orukọ ṣe tọka, yoo fi oju-iwe tuntun ti o ṣẹda kun si akojọ admin lẹgbẹẹ awọn oju-iwe WordPress aiyipada.

Ti o ba yan aṣayan ipele-oke, iwọ yoo le yan ilana akojọ ati aami akojọ pẹlu.

Ti o ba fẹ ṣe oju-iwe yii gẹgẹ bi oju-iwe akojọ abẹ́, iwọ yoo le yan oju-iwe obi lati so oju-iwe yii mọ. O le fi oju-iwe ti a ṣe adani rẹ kun si awọn oju-iwe WordPress aiyipada tabi si awọn oju-iwe admin ti a ṣe adani ipele-oke tirẹ.

## Àwọn Aṣayan To ti ni Ilọsiwaju {#advanced-options}

O yẹ ki o le fi gbogbo iru àkóónú kun si awọn oju-iwe admin ti a ṣe adani rẹ. Iyẹn pẹlu awọn ofin ati faili CSS, JavaScript, ati awọn ibi ìkàwé ita pẹlu. O le ṣe iyẹn nipa lilo Taabu Awọn Aṣayan To ti ni Ilọsiwaju.

### CSS Ti a Ṣe Adani {#custom-css}

Lori taabu CSS, o le fi kóòdù CSS ti a ṣe adani tirẹ kun, bakanna bi gbigbe awọn faili CSS wọle lati awọn aaye jijin (bii CDNs, Google Fonts, abbl).

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_CSS ti a ṣe adani tun ni atilẹyin. O tun le ṣafikun awọn faili ara ita!_

### JavaScript Ti a Ṣe Adani {#custom-javascript}

Ohun kanna wulo fun JavaScript. O le fi awọn ibi ìkàwé ita kun lati lo lori kóòdù JavaScript ti a ṣe adani rẹ.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Fi kóòdù JavaScript tirẹ kun!_

## Àwọn Ìyọ̀nda {#permissions}

Bakannaa ninu Awọn Aṣayan To ti ni Ilọsiwaju, iwọ yoo le yan iru awọn aaye abẹ́ wo ni yoo ni iwọle si oju-iwe yii. Eyi wulo ti o ba n gbero lati pese àkóónú iyasọtọ oriṣiriṣi si awọn ipele eto oriṣiriṣi, fun apẹẹrẹ.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_O ni iṣakoso lapapọ lori ibiti oju-iwe yii yoo han_

**Pataki** : Awọn ipo mejeeji gbọdọ ṣẹ fun oju-iwe naa lati han. Iyẹn tumọ si pe ti olumulo ba jẹ alabara ti eto kan pato, ṣugbọn ko ni ọkan ninu awọn ipa ti a yan (tabi idakeji) oju-iwe naa kii yoo han lori panẹli wọn.

## Nṣiṣẹ {#active}

Ti o ba fẹ mu oju-iwe admin ti a ṣe adani kan kuro lai paarẹ rẹ tabi lai ba awọn ìyọ̀nda jẹ, kan yọ àmì ṣíṣe kuro ni aṣayan Nṣiṣẹ.

Mimu oju-iwe ipele-oke kan kuro yoo mu gbogbo awọn oju-iwe abẹ́ ti a so mọ oju-iwe obi yẹn kuro laifọwọyi.
