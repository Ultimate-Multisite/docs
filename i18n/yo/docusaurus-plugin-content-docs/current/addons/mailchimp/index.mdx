# Mailchimp

Multisite Ultimate: Ìṣepọ̀ Mailchimp

Ṣe ìṣepọ̀ nẹ́tíwọ́ọ̀kì Multisite Ultimate rẹ mọ́ Mailchimp rẹ. O lè so àwọn ètò kan pọ̀ mọ́ àwọn àtòjọ Mailchimp kí o sì fi àwọn alábàápín kún àwọn àtòjọ wọ̀nyẹn laifọwọyi nígbà ìforúkọsílẹ̀.

## Fífì Sórí Ẹ̀rọ

1. Gbé 'ultimate-multisite-mailchimp' sí inú ìkànnì '/wp-content/plugins/'
2. Mu plugin náà ṣiṣẹ́ nípasẹ̀ àkójọ 'Plugins' nínú WordPress Network Admin rẹ

## Ìtọ́nisọ́nà Lílo

Ṣíṣe amuṣiṣẹpọ̀ àwọn ímeèlì alábàápín pẹ̀lú àwọn àtòjọ MailChimp rẹ rọrùn gan-an pẹ̀lú àfikún Ultimate Multisite: MailChimp Integration wa. Ẹ̀kọ́ yìí ní ète láti ràn ọ́ lọ́wọ́ láti ṣètò gbogbo nǹkan!

## Gbigba API key

Kí a tó bẹ̀rẹ̀ sí í wọ inú àwọn ètò ọ̀tọ̀ọ̀tọ̀ tó wà, a nílò MailChimp API key kan. Èyí yóò jẹ́ kí Ultimate Multisite lè bá MailChimp API sọ̀rọ̀ láti gba àwọn àtòjọ, àwọn ẹgbẹ́, àti láti fi àwọn àdírẹ́sì ímeèlì kún àwọn àtòjọ àti ẹgbẹ́ wọ̀nyẹn nígbà tí àwọn aṣàmúlò bá forúkọ sílẹ̀.

Tẹ̀lé ẹ̀kọ́ yìí láti gba MailChimp API key tirẹ: [Nípa API Keys | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

## Móòdù Ìṣepọ̀

Àfikún MailChimp Integration ń pèsè “àwọn móòdù ìṣepọ̀” méjì, èyí túmọ̀ sí pé ọ̀nà méjì ọ̀tọ̀ọ̀tọ̀ ló wà tí o lè fi lo àfikún náà láti ṣe amuṣiṣẹpọ̀ àwọn àdírẹ́sì ímeèlì.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_Àwọn móòdù ìṣepọ̀ méjì fún ìrọ̀rùn tó pọ̀ síi!_

### Móòdù 1: Ọ̀pọ̀ Àtòjọ

Móòdù àiyipada máa ń lo **Ọ̀pọ̀ Àtòjọ**. Móòdù yìí yóò jẹ́ kí o lè yan ọ̀pọ̀ àtòjọ MailChimp fún ọ̀kọ̀ọ̀kan àwọn ètò. Èyí ni àṣàyàn tó dára jùlọ bí o bá pín àwọn aṣàmúlò rẹ sí àwọn àtòjọ ọ̀tọ̀ọ̀tọ̀.

Fún àpẹẹrẹ, ẹ jẹ́ ká sọ pé o ní àtòjọ ọ̀tọ̀ọ̀tọ̀ mẹ́ta lórí Account MailChimp rẹ: **Ìwé Ìròyìn** , **Ètò A** , àti **Ètò B**. O fẹ́ kí gbogbo aṣàmúlò, láìka ètò wọn sí, tún wà nínú àtòjọ Ìwé Ìròyìn, kí wọ́n má bàa padà sẹ́yìn nínú àwọn ìmúdójúìwọ̀n rẹ.

Láti ṣàṣeparí èyí, o lè lo móòdù Ọ̀pọ̀ Àtòjọ láti fi àwọn àtòjọ méjèèjì **Ìwé Ìròyìn** àti **Ètò A** kún Àwọn Ètò MailChimp Ètò A, àti **Ìwé Ìròyìn** àti **Ètò B** sí Àwọn Ètò MailChimp Ètò B.

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_Yan àwọn àtòjọ àiyipada fún àwọn Account tuntun àti fún àwọn Account tí a fagilé_

### Móòdù 2: Àtòjọ Kan + Ọ̀pọ̀ Ẹgbẹ́

MailChimp jẹ́ kí o lè pín àtòjọ rẹ sí àwọn ẹgbẹ́ kéékèèké ti àwọn aṣàmúlò tí a ń pè ní **Àwọn Ẹgbẹ́**. Èyí jẹ́ kí o ní ìṣàkóso púpọ̀ síi nítorí o lè fi ìpolongo ránṣẹ́ sí apá pàtó kan nínú àwọn alábàápín rẹ láìní láti dá ọ̀pọ̀ ẹ̀dà ìpolongo kan náà sílẹ̀ lórí MailChimp (èyí tó máa ń jẹ́ kí ó ṣòro láti ṣe àyẹ̀wò àwọn òṣùwọ̀n ìpolongo yẹn lẹ́yìn náà).

Fún àpẹẹrẹ, o lè ní àtòjọ kan ṣoṣo tí a ń pè ní **Àwọn Alábàápín Nẹ́tíwọ́ọ̀kì Mi,** àti nínú rẹ̀ kí o ní àwọn ẹgbẹ́ fún **Ètò A** àti **Ètò B**.

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_Yan àwọn àtòjọ àti ẹgbẹ́ àiyipada, bí o bá ń lo móòdù ìṣepọ̀ “àtòjọ kan + ọ̀pọ̀ ẹgbẹ́”_

## Àwọn Àtòjọ àti Ẹgbẹ́ Àiyipada

Lẹ́yìn tí o bá yan móòdù ìṣepọ̀ wo ló bá ọ̀rọ̀ lílo rẹ pàtó mu jù, ó tó àkókò láti yan àwọn àtòjọ àiyipada (àti àwọn ẹgbẹ́, da lórí móòdù náà). Àtòjọ àiyipada ni a óò lo nígbà tí ètò kan kò bá ní Àtòjọ pàtó tí a yàn.

O tún lè yan àtòjọ àiyipada fún àwọn Account tí a fagilé. Nígbàkigbà tí aṣàmúlò bá parí subscription rẹ̀ tí ó sì yọ Account rẹ̀ kúrò nínú nẹ́tíwọ́ọ̀kì rẹ, a óò gbe àdírẹ́sì ímeèlì rẹ̀ lọ sí àtòjọ tí a ṣètò fún àwọn Account tí a fagilé.

## Mímu Double Subscription Ṣiṣẹ́

Lórí ojú-ìwé ètò kan náà, ìwọ yóò lè mu **Double Subscription** ṣiṣẹ́. Àṣàyàn yìí yóò fi aṣàmúlò náà kún àtòjọ àiyipada tí a yàn ÀTI àtòjọ tí a yàn fún ètò aṣàmúlò yẹn. Bí a kò bá mu àṣàyàn yìí ṣiṣẹ́, aṣàmúlò náà yóò wà nínú àtòjọ tí a yàn fún ètò aṣàmúlò náà nìkan. Bí kò bá sí èyí tí a ṣètò fún ètò pàtó yẹn, nígbà náà a óò fi aṣàmúlò náà kún àtòjọ àiyipada tí a yàn.

_O lè jẹ́ kí àwọn aṣàmúlò rẹ di alábàápín sí àtòjọ àiyipada àti àtòjọ ètò náà nígbà ìforúkọsílẹ̀_

## Yíyan Àwọn Àtòjọ àti Ẹgbẹ́ fún ọ̀kọ̀ọ̀kan Àwọn Ètò

Lọ sí **Àwọn Ètò** rẹ kí o sì yan ètò tí o fẹ́ ṣàtúnṣe. Ṣàkíyèsí pé lórí meta-box **Àwọn Àṣàyàn Onílọsíwájú**, taabu tuntun kan tí a ń pè ní MailChimp yóò wà.

Lórí taabu yẹn, ìwọ yóò lè yan ọ̀pọ̀ àtòjọ tàbí àtòjọ kan ṣoṣo àti ọ̀pọ̀ ẹgbẹ́ (da lórí móòdù ìṣepọ̀) fún ètò yẹn.

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_Yan Àwọn Àtòjọ àti Ẹgbẹ́ tí o fẹ́ lo fún ọ̀kọ̀ọ̀kan àwọn ètò rẹ_

Nígbà tí aṣàmúlò tuntun bá forúkọ sílẹ̀ fún ètò yẹn, a óò fi àdírẹ́sì ímeèlì rẹ̀ kún àwọn àtòjọ àti ẹgbẹ́ tí a yàn!
