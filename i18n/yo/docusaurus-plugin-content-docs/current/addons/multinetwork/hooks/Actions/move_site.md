# Ìṣe: move_site

Ó máa ń ṣiṣẹ́ lẹ́yìn tí a bá ti gbe ààyè kan lọ sí network tuntun.

## Àwọn Parameter {#parameters}

| Orúkọ | Iru | Àpèjúwe |
|------|------|-------------|
| $site_id | `int` | ID ti ààyè tí a ti gbe. |
| $old_network_id | `int` | ID ti network ìbẹ̀rẹ̀ fún ààyè náà. |
| $new_network_id | `int` | ID ti network tí a ti gbe ààyè náà lọ sí. |

### Láti {#since}

- 1.3.0
