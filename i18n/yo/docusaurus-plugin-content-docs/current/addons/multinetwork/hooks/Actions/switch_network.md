# Iṣe: switch_network

Ó máa ń ṣiṣẹ́ nígbà tí a bá yí àyíká network lọwọlọwọ padà.

## Àwọn parameter {#parameters}

| Orúkọ | Iru | Apejuwe |
|------|------|-------------|
| $new_network_id | `int` | ID ti network tí a ń yí padà sí. |
| $old_network_id | `int` | ID ti network lọwọlọwọ tẹ́lẹ̀. |
