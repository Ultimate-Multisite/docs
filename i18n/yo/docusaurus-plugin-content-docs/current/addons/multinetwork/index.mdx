# Ọ̀pọ̀-Nẹ́tíwọ́ọ̀kì

Àfikún Ultimate Multisite: Multinetwork ń faagun platform Ultimate Multisite rẹ pẹ̀lú iṣẹ́ Multinetwork alágbára. Àfikún yìí jẹ́ kí àwọn oníbàárà rẹ lè ra kí wọ́n sì ṣàkóso gbogbo àwọn nẹ́tíwọ́ọ̀kì multisite WordPress, kì í ṣe àwọn ojúlé kọ̀ọ̀kan nìkan.

## Àwọn Ẹ̀yà Pàtàkì {#key-features}

* **Ìpèsè Nẹ́tíwọ́ọ̀kì**: Ṣẹ̀dá kí o sì ṣètò gbogbo àwọn nẹ́tíwọ́ọ̀kì multisite WordPress fún àwọn oníbàárà rẹ láìfọwọ́ṣe
* **Àwọn Àwoṣe Nẹ́tíwọ́ọ̀kì**: Jẹ́ kí àwọn oníbàárà ṣàwárí, wo àwòtẹ́lẹ̀, kí wọ́n sì yan àwoṣe nẹ́tíwọ́ọ̀kì tí a ti kọ́ tẹ́lẹ̀ kí wọ́n tó forúkọsílẹ̀
* **Ìṣàkóso Nẹ́tíwọ́ọ̀kì**: Àwọn agbára ìṣàkóso kíkún fún àwọn àyíká Multinetwork
* **Àwọn Nẹ́tíwọ́ọ̀kì Tí a Ya Sọ́tọ̀**: Nẹ́tíwọ́ọ̀kì oníbàárà kọ̀ọ̀kan ń ṣiṣẹ́ lọ́tọ̀ pẹ̀lú àwọn tábìlì database àti ìṣètò tirẹ̀
* **SSO Láàárín-Nẹ́tíwọ́ọ̀kì**: Fún àwọn oníbàárà ní ìwọlé aláìdènà sí àwọn nẹ́tíwọ́ọ̀kì abẹ́ wọn pẹ̀lú ìjẹ́rìí magic-link
* **Àkópọ̀ Ilé-iṣẹ́ Tó Lè Gbòòrò**: Ṣàkóso ọ̀pọ̀ nẹ́tíwọ́ọ̀kì dáadáa pẹ̀lú pípín àwọn ohun àmúlò tí a ti mú ṣe dáadáa
* **Ìṣàkópọ̀**: Ó ṣàkópọ̀ láìṣòro pẹ̀lú àwọn eto ìforúkọsílẹ̀ àti ìsanwó ti Ultimate Multisite

## Àwọn Ọ̀nà Lílò {#use-cases}

* **Hosting White Label**: Pèsè àwọn nẹ́tíwọ́ọ̀kì multisite tí wọ́n ní brand tirẹ̀ fún àwọn oníbàárà
* **Àwọn Ojútùú Fún Agency**: Jẹ́ kí àwọn agency lè ṣàkóso ọ̀pọ̀ nẹ́tíwọ́ọ̀kì oníbàárà láti platform kan ṣoṣo
* **Àwọn Ìmúlò Enterprise**: Ṣe àtìlẹ́yìn fún àwọn àjọ ńlá pẹ̀lú àwọn àìní ọ̀pọ̀-nẹ́tíwọ́ọ̀kì tó díjú
* **Àwọn Platform Ẹ̀kọ́**: Ṣẹ̀dá àwọn nẹ́tíwọ́ọ̀kì lọ́tọ̀ fún àwọn ilé-ẹ̀kọ́ tàbí ẹ̀ka ọ̀tọ̀ọ̀tọ̀

## Àwọn Ohun Tí a Nílò {#requirements}

* Ultimate Multisite 2.0.0 tàbí tó ga jù bẹ́ẹ̀ lọ
* Ìfisílẹ̀ WordPress Multisite
* PHP 7.4 tàbí tó ga jù bẹ́ẹ̀ lọ
* MySQL 5.6 tàbí tó ga jù bẹ́ẹ̀ lọ

## Ìfisílẹ̀ {#installation}

1. Gbe àwọn fáìlì àfikún sí directory `/wp-content/plugins/ultimate-multisite-multinetwork`
2. Mu àfikún náà ṣiṣẹ́ nípasẹ̀ iboju 'Àwọn Àfikún' nínú WordPress
3. Ṣètò àwọn ìṣètò Multinetwork nípasẹ̀ Ultimate Multisite → Ìṣètò

## Àwọn Àwoṣe Nẹ́tíwọ́ọ̀kì {#network-templates}

Ẹ̀dà 1.3.0 ṣàfikún ìṣàn ṣíṣàwárí àti yíyan àwoṣe fún àwọn oníbàárà tí ń ra àwọn nẹ́tíwọ́ọ̀kì tuntun. Lo àwọn àwoṣe nẹ́tíwọ́ọ̀kì nígbà tí o bá fẹ́ kí nẹ́tíwọ́ọ̀kì oníbàárà tuntun kọ̀ọ̀kan bẹ̀rẹ̀ láti inú àkópọ̀ tí a ti pèsè, àpẹrẹ, ibi ìkàwé àkóónú, tàbí ìṣètò pàtó-fún-ẹ̀ka.

### Olùwo àwòtẹ́lẹ̀ àwoṣe {#template-previewer}

Olùwo àwòtẹ́lẹ̀ àwoṣe nẹ́tíwọ́ọ̀kì jẹ́ kí àwọn oníbàárà ṣàwárí àwọn àwoṣe tó wà kí wọ́n tó checkout. Àwoṣe kọ̀ọ̀kan lè ṣí sílẹ̀ nínú panel àwòtẹ́lẹ̀ alààyè, kí àwọn oníbàárà lè fi àwọn layouts àti àkóónú wéra kí wọ́n tó ra nẹ́tíwọ́ọ̀kì kan.

Lo olùwo àwòtẹ́lẹ̀ láti ran àwọn oníbàárà lọ́wọ́ láti:

* Ṣàyẹ̀wò ìrísí àti àkópọ̀ àwoṣe nẹ́tíwọ́ọ̀kì kọ̀ọ̀kan tó wà
* Fi ọ̀pọ̀ àwoṣe wéra láìfi ìṣàn rírà sílẹ̀
* Yan àwoṣe tí ó bá ọ̀nà lílò wọn mu jù lọ kí wọ́n tó forúkọsílẹ̀

Jẹ́ kí àwọn orúkọ àti àpèjúwe àwoṣe rọrùn fún oníbàárà. Olùwo àwòtẹ́lẹ̀ wúlò jù lọ nígbà tí àwọn oníbàárà lè lóye ọ̀nà lílò tí a pinnu, bíi àwọn nẹ́tíwọ́ọ̀kì agency, àwọn nẹ́tíwọ́ọ̀kì ilé-ẹ̀kọ́, àwọn nẹ́tíwọ́ọ̀kì franchise, tàbí àwọn àkójọpọ̀ hosting white-label.

### Olùyan àwoṣe ní checkout {#template-picker-at-checkout}

Olùyan àwoṣe lè ṣàfikún sí ìṣàn checkout ìforúkọsílẹ̀ gẹ́gẹ́ bí field kan. Nígbà tí a bá ṣètò rẹ̀, àwọn oníbàárà yan àwoṣe nẹ́tíwọ́ọ̀kì tí wọ́n fẹ́ nígbà checkout, a ó sì lo àwoṣe tí a yan nígbà tí a bá pèsè nẹ́tíwọ́ọ̀kì tuntun wọn.

Láti lo olùyan náà nínú ìṣàn checkout:

1. Pèsè àwọn nẹ́tíwọ́ọ̀kì tí o fẹ́ fúnni gẹ́gẹ́ bí àwọn àwoṣe.
2. Mu field yíyan àwoṣe Multinetwork ṣiṣẹ́ nínú fọọmu ìforúkọsílẹ̀ tàbí checkout tó yẹ.
3. Yan àwọn àwoṣe wo ló yẹ kí ó wà fún ìṣàn checkout yẹn.
4. Ṣàtẹ̀jáde ìṣàn checkout náà kí o sì dán ìforúkọsílẹ̀ kan wò láti jẹ́rìí pé àwoṣe tí a yan ni a fi sí nẹ́tíwọ́ọ̀kì tuntun náà.

Bí ìṣàn checkout kan bá ń fúnni ní ọja nẹ́tíwọ́ọ̀kì kan ṣoṣo, olùyan náà ṣì lè wúlò gẹ́gẹ́ bí ìgbésẹ̀ onboarding tí a tọ́sọ́nà. Fún àwọn funnel ọ̀pọ̀-plan, fi hàn àwọn àwoṣe nìkan tí ó bá ọja tàbí plan tí a yan mu.

## Magic Link SSO Láàárín-Nẹ́tíwọ́ọ̀kì {#cross-network-magic-link-sso}

Magic link SSO láàárín-nẹ́tíwọ́ọ̀kì jẹ́ kí àwọn oníbàárà lè rìn láàárín agbègbè Account àkọ́kọ́ wọn àti àwọn nẹ́tíwọ́ọ̀kì abẹ́ láìwọlé pẹ̀lú ọwọ́ lẹ́ẹ̀kansi. Nígbà tí oníbàárà bá tẹ̀lé ọna asopọ ìwọlé láàárín-nẹ́tíwọ́ọ̀kì, Ultimate Multisite ń ṣe agbejade magic link tí ó ní àkókò díẹ̀, ó sì ń jẹ́rìí oníbàárà wọlé sí nẹ́tíwọ́ọ̀kì ibi tí wọ́n ń lọ.

Èyí ń mú ìrírí dara sí fún àwọn oníbàárà tí ń ṣàkóso ju nẹ́tíwọ́ọ̀kì kan lọ tàbí tí wọ́n nílò láti wọlé sí àwọn nẹ́tíwọ́ọ̀kì àwoṣe tí a ṣẹ̀ṣẹ̀ clone lẹ́sẹ̀kẹsẹ̀ lẹ́yìn rírà.

Àkójọ ìṣàyẹ̀wò admin:

* Jẹ́rìí pé àfikún Multinetwork ṣiṣẹ́ káàkiri ìfisílẹ̀ WordPress multisite.
* Rí i dájú pé àwọn oníbàárà ní ìwọlé nẹ́tíwọ́ọ̀kì tó tọ́ kí a tó fún wọn ní àwọn ọna asopọ láàárín-nẹ́tíwọ́ọ̀kì.
* Jẹ́ kí àwọn domain nẹ́tíwọ́ọ̀kì àti àwọn domain tí a mapped jẹ́ títò dáadáa kí magic links lè yọrí sí nẹ́tíwọ́ọ̀kì ibi tí a pinnu.
* Dán dashboard oníbàárà àti àwọn bọ́tìnì ìwọlé lẹ́yìn-checkout wò lẹ́yìn tí o bá yí nẹ́tíwọ́ọ̀kì, domain, tàbí àwọn ìṣètò checkout padà.

Ó yẹ kí a ka magic links sí àwọn ọna asopọ ìwọlé. A ṣe wọ́n fún ìwọlé oníbàárà aláìdènà, kò sì yẹ kí a fi ránṣẹ́ síwájú tàbí tún lo níta session tí a pinnu.

## Àwọn Ìbéèrè Tí a Máa ń Béèrè Lọ́pọ̀ Ìgbà {#frequently-asked-questions}

### Kí ni Multinetwork? {#what-is-multinetwork}

Multinetwork jẹ́ ẹ̀yà WordPress kan tí ó jẹ́ kí o lè ṣiṣẹ́ ọ̀pọ̀ nẹ́tíwọ́ọ̀kì multisite olómìnira lórí ìfisílẹ̀ WordPress kan ṣoṣo.

### Báwo ni èyí ṣe yàtọ̀ sí multisite àtẹ̀yìnwá? {#how-is-this-different-from-regular-multisite}

Multisite àtẹ̀yìnwá jẹ́ kí ọ̀pọ̀ ojúlé wà nínú nẹ́tíwọ́ọ̀kì kan. Multinetwork jẹ́ kí ọ̀pọ̀ nẹ́tíwọ́ọ̀kì wà, ọkọ̀ọ̀kan pẹ̀lú àwọn ojúlé tirẹ̀, gbogbo rẹ̀ nínú ìfisílẹ̀ WordPress kan ṣoṣo.

### Ṣé èyí nílò ìṣètò server pàtó? {#does-this-require-special-server-configuration}

Bẹ́ẹ̀ni, Multinetwork nílò ìgbékalẹ̀ àti ìṣètò server pàtó. Jọ̀wọ́ wo documentation fún àwọn ìtọ́nisọ́nà ìgbékalẹ̀ kíkún.
