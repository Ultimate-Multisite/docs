# Olùyan Èdè {#language-selector}

Ultimate Multisite: Olùyan Èdè nígbà Ìforúkọsílẹ̀

Ṣàfikún pápá olùyan èdè sí ìṣàn ìforúkọsílẹ̀.

## Fífí sori ẹrọ {#installation}

1. Ṣe ìgbéjáde 'ultimate-multisite-language-selector' sí àkójọ fáìlì '/wp-content/plugins/'
2. Mú plugin náà ṣiṣẹ́ nípasẹ̀ àkójọ 'Plugins' nínú WordPress Network Admin rẹ

## Ìtọ́nisọ́nà Lílo {#usage-guide}

Pẹ̀lú add-on ọ̀fẹ́ yìí, o lè jẹ́ kí àwọn aṣàmúlò rẹ yan èdè tí wọ́n fẹ́ rí lórí ìṣàn ìforúkọsílẹ̀ àti lórí ojú-ewé ìwọlé.

Lẹ́yìn [mímu license rẹ ṣiṣẹ́](../../user-guide/getting-started/installing-ultimate-multisite.md), o máa lè fi àwọn add-on Ultimate Multisite tí license rẹ fún ọ ní ààyè sí sori ẹrọ.

Lọ sí ojú-ewé àwọn settings Ultimate Multisite rẹ, lẹ́yìn náà, lórí àkójọ ní apá òsì, tẹ àmì _Add-ons_.

![Ojú-ewé Add-ons](/img/admin/addons-page.png)

_Lọ sí ojú-ewé Add-ons láti bẹ̀rẹ̀ fífi Language Selector sori ẹrọ_

Ní bẹ̀, ìwọ yóò rí gbogbo add-on tí a nṣe.

Àwọn add-on ọ̀fẹ́ wà fún gbogbo àwọn oníbàárà wa, láìka plan subscription tí wọ́n forúkọsílẹ̀ sí.

Tẹ bọ́tìnnì _Install it Now!_ ní ìsàlẹ̀ àwòrán kékeré add-on náà. Lẹ́yìn náà, mú un ṣiṣẹ́ lórí ojú-ewé Plugins.

![Ojú-ewé Add-ons pẹ̀lú bọ́tìnnì fífi sori ẹrọ](/img/admin/addons-page.png)

_Tẹ bọ́tìnnì náà láti fi add-on náà sori ẹrọ_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_Mú un ṣiṣẹ́ lórí ojú-ewé Plugins rẹ_

Báyìí, lọ sí àkójọ Settings Ultimate Multisite, kí o sì wọ taabu _Network Settings_. Ní bẹ̀, ní ìsàlẹ̀ ojú-ewé náà, ìwọ yóò rí apá tuntun kan tí a ń pè ní _Languages_.

![Ojú-ewé Settings](/img/config/settings-general.png)

_Apá tuntun pátápátá hàn!_

O lè fi sílẹ̀ ní òfo láti ṣàfihàn gbogbo àwọn èdè gẹ́gẹ́ bí àwọn àṣàyàn, tàbí yan èyí tí o fẹ́ jẹ́ kí ó wà fún lílo. Àwọn èdè náà ni àwọn tí o ti fi sori ẹrọ nínú ìṣètò WordPress rẹ.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_Àtòkọ àwọn èdè tó wà lórí fífi WP yìí sori ẹrọ_

Lẹ́yìn fífi àwọn àyípadà tí o ṣe pamọ́, àkójọ yíyan pẹ̀lú àwọn èdè tí o yàn yóò hàn lórí ojú-ewé Login àti lórí Ìṣàn Ìforúkọsílẹ̀.

![Àwọn settings Login àti Registration](/img/config/settings-login-registration.png)

_Language Selector lórí Ojú-ewé Login_

![Ojú-ewé Registration](/img/config/registration-page.png)

_Language Selector lórí Ìṣàn Ìforúkọsílẹ̀_

## Ṣíṣètò Èdè nípasẹ̀ URL {#setting-the-language-via-the-url}

O tún lè yí èdè ojú-ewé kan padà nípa fífi parâmità locale sí URL. Bí o bá fẹ́ dari lọ sí ojú-ewé ìforúkọsílẹ̀ ní èdè Ítálì, fún àpẹẹrẹ, ṣàfikún ?locale=it_IT sí URL náà.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Èdè náà gbọ́dọ̀ ti wà lórí network WordPress rẹ kí èyí tó lè ṣiṣẹ́.
