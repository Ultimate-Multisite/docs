# Gratis AI Agent

Gratis AI Agent jẹ́ olùrànlọ́wọ́ AI aládàáṣiṣẹ́ fún WordPress tí ó lè gbero, kọ́, àti ṣàkóso àwọn sáịtị WordPress pípé nípasẹ̀ àwọn ìtọ́nisọ́nà èdè àdánidá. Ó ṣí àkójọpọ̀ **àwọn agbára** tó ní ètò síta — àwọn ìṣe ọ̀tọ̀ọ̀tọ̀ tí a lè dán wò tí aṣojú náà lè pè — tí ó bo gbogbo nǹkan láti ìṣàkóso àkóónú dé sí ṣíṣe àkànṣe ètò àpẹrẹ. Ẹ̀yà 1.9.0 fi àwọn agbára ṣíṣẹ̀dá àkóónú kún un, àyẹ̀wò ojúewé àwòrán nípasẹ̀ àwọn sikírínsọ́ọ̀tì ẹgbẹ́ client, àwọn profaili aṣojú márùn-ún tí a ti kọ́ sínú rẹ̀, àti àwọn asia ẹya fún ìṣàkóso ìráyè àti àmì-ọjà.

## Àwọn Ẹ̀ya Pàtàkì

- **Ètò àwọn agbára** — àkójọ ìṣe onímódúlù, tí a lè faagun, tí aṣojú náà lè pè lórí ìfikún WordPress rẹ
- **Ìṣàkóso Àkóónú** — dá àwọn àtẹ̀jáde àti ojúewé sílẹ̀ (pẹ̀lú yíyan àwọn àdàkọ ojúewé), dá ọ̀pọ̀ àtẹ̀jáde sílẹ̀ ní ìpè kan ṣoṣo, ṣètò àwọn àwòrán àfihàn, kí o sì kọ́ àwọn fọ́ọ̀mù ìbánisọ̀rọ̀
- **Àyẹ̀wò Àwòrán** — ya àwọn sikírínsọ́ọ̀tì àwọn ojúewé tó wà láàyè, ṣe ìyàtọ̀ àwọn ipò ṣáájú/lẹ́yìn, kí o sì gba àyẹ̀wò àpẹrẹ tí AI ṣe tí ó bo ìtòlẹ́sẹẹsẹ, typography, àwọ̀, àti ìrọ̀rùn lílò
- **Àwọn aṣojú márùn-ún tí a ti kọ́ sínú rẹ̀** — Olùkọ Àkóónú, Olùkọ́ Sáịtị, Ilé Ìṣẹ́ Àpẹrẹ, Alákòóso Àfikún, àti Olùrànlọ́wọ́ Ìtìlẹ́yìn; a lè yí wọn padà nípasẹ̀ Agent Picker nínú akọlé ìfọ̀rọ̀wérọ̀
- **Àwọn asia ẹya** — àwọn ìyípadà ìṣàkóso ìráyè àti àmì-ọjà nínú Settings → Feature Flags fún àwọn ìdíwọ́ ipa, orúkọ white-label, àti ṣíṣe àkànṣe widget
- **Ìṣàkóso Custom Post Type** — forúkọsílẹ̀, ṣe àtòjọ, àti pa àwọn custom post types rẹ́ pẹ̀lú ìfaradà kọjá àwọn ìbẹ̀rẹ̀ padà
- **Ìṣàkóso Custom Taxonomy** — forúkọsílẹ̀, ṣe àtòjọ, àti pa àwọn custom taxonomies rẹ́ pẹ̀lú ìfaradà
- **Àwọn agbára Ètò Àpẹrẹ** — fi CSS àkànṣe sínú rẹ̀, ṣàkóso àwọn block patterns tí a ti yan, ṣètò àmì ilé-iṣẹ́ sáịtị, kí o sì lo àwọn àtòjọ theme.json
- **Ìṣàkóso Global Styles** — ka àti kọ àwọn iye theme.json fún àwọn àkójọpọ̀ àwọ̀, typography, ààlà ààyè, àti bẹ́ẹ̀ bẹ́ẹ̀ lọ
- **Ìṣàkóso Àkójọ Ìtọ́sọ́nà** — dá àti ṣe ìmúdójúìwọ̀n àwọn àkójọ ìtọ́sọ́nà àti àwọn ohun wọn nípasẹ̀ eto
- **Ìṣàkóso Àwọn Àṣàyàn** — ka àti kọ àwọn àṣàyàn WordPress pẹ̀lú blocklist ààbò tí a ti kọ́ sínú rẹ̀ láti dáàbò bo àwọn ètò pàtàkì
- **Ìforúkọsílẹ̀ Àwọn Agbára Tí A Lè Fi Sílẹ̀** — ṣàwárí àti mú àwọn ìdìpọ̀ agbára míì ṣiṣẹ́ láì ṣe àtúnṣe àwọn fáìlì àfikún pàtàkì
- **Site Builder Orchestration v2** — àwọn ètò ìkọ́ sáịtị onígbésẹ̀ púpọ̀ pẹ̀lú ìṣàwárí àfikún, títọpa ìlọsíwájú, àti ìmúpadàbọ̀sípò àṣìṣe
- **Àṣẹ ìdánwò ìṣe WP-CLI** — ṣiṣẹ́ àkójọpọ̀ ìdánwò ìṣe agbára aṣojú láti command line fún CI àti ìdánilójú model

## Àwọn Ohun Tó Nílò

- WordPress 6.0 tàbí èyí tó ga jù
- PHP 8.1 tàbí èyí tó ga jù
- WordPress Multisite (tí a mú ṣiṣẹ́ lórí nẹ́tíwọ́ọ̀kì) tàbí ìfikún sáịtị kan ṣoṣo
- Àfikún asopọ̀ olupèsè AI tó ń ṣiṣẹ́ (Anthropic, OpenAI, tàbí tó bá a mu)

## Fífí Sílẹ̀

1. Po àwọn fáìlì àfikún sí `/wp-content/plugins/gratis-ai-agent`
2. Mú àfikún náà ṣiṣẹ́ nípasẹ̀ iboju **Àwọn Àfikún** nínú WordPress
3. Lọ sí **Gratis AI Agent → Settings** kí o sì so olupèsè AI rẹ pọ̀
4. (Àṣàyàn) Ṣàwárí **Ìforúkọsílẹ̀ Àwọn Agbára** láti fi àwọn ìdìpọ̀ agbára míì sílẹ̀

## Ìbẹ̀rẹ̀ Kíákíá

Nígbà tí a bá ti mú un ṣiṣẹ́ tí a sì ti so mọ́ olupèsè AI, ṣí pánẹ́ẹ̀lì **Gratis AI Agent** nínú pẹpẹ ẹgbẹ́ alákóso WordPress. Tẹ ìtọ́nisọ́nà èdè àdánidá kan, aṣojú náà yóò sì gbero àti ṣe àwọn ìgbésẹ̀ tí a nílò ní lílo àwọn agbára tó wà fún un.

**Àwọn àpẹẹrẹ ìtọ́nisọ́nà:**

```
Dá custom post type "Portfolio" kan sílẹ̀ pẹ̀lú taxonomy "Project Category".
```

```
Fi àkójọpọ̀ àwọ̀ dark-mode sínú theme náà kí o sì ṣètò àwọ̀ àkọ́kọ́ sí #1a1a2e.
```

```
Kọ́ wẹ́ẹ̀bùsáịtị ilé oúnjẹ pẹ̀lú àkójọ oúnjẹ lórí ayélujára, fọ́ọ̀mù ìfọwọ́sowọ́pọ̀ ìpamọ́ àyè, àti ojúewé ìbánisọ̀rọ̀.
```

## Àkótán Àwọn Agbára

Àwọn agbára ni àwọn ìṣe atomiki tí aṣojú náà lè ṣe. Agbára kọ̀ọ̀kan jẹ́ kilasi PHP tí a forúkọsílẹ̀ tí ó ṣí schema (àwọn paramita àti àwọn iru ìpadà) síta tí aṣojú náà máa ń kà nígbà ìṣiṣẹ́. A pín àwọn agbára sí àwọn agbègbè iṣẹ́:

| Agbègbè | Àwọn Agbára |
|---|---|
| **Ìṣàkóso Àkóónú** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **Àyẹ̀wò Àwòrán** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **Custom Post Types** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **Custom Taxonomies** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **Ètò Àpẹrẹ** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **Global Styles** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **Àwọn Àkójọ Ìtọ́sọ́nà** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **Ìṣàkóso Àwọn Àṣàyàn** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **Àwọn Agbára Tí A Lè Fi Sílẹ̀** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **Olùkọ́ Sáịtị** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

Wo [Ìtọ́kasí Àwọn Agbára](./abilities) fún ìwé àkọsílẹ̀ paramita pípé àti àwọn àpẹẹrẹ lílò.

## WP-CLI

Gratis AI Agent wá pẹ̀lú ẹbí àṣẹ `wp gratis-ai-agent` fún ṣiṣẹ́ àwọn ìdánwò ìṣe àti ṣíṣàkóso àwọn agbára láti command line. Wo [Ìtọ́kasí WP-CLI](./wp-cli) fún àtòjọ àṣẹ pípé.

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## Ìṣètò

Gbogbo àwọn ètò wà lábẹ́ **Gratis AI Agent → Settings** nínú alákóso WordPress:

| Ètò | Àpèjúwe |
|---|---|
| **Olupèsè AI** | Yan kí o sì ṣètò asopọ̀ olupèsè AI tó ń ṣiṣẹ́ |
| **Model** | Yan model èdè tí a ó lò fún àwọn ìpàdé aṣojú |
| **Àwọn Token Tó Pọ̀ Jù** | Ìṣúná token fún ìbéèrè kọ̀ọ̀kan (àìyípadà 4096) |
| **Àkókò Ìparí Agbára** | Àwọn ìṣẹ́jú-àáyá tó pọ̀ jù tí agbára ẹyọ kọ̀ọ̀kan lè ṣiṣẹ́ (àìyípadà 30) |
| **Ìkọ̀wé Debug** | Kọ àwọn ìpè agbára àti ìrònú aṣojú sí WordPress debug log |

## Blocklist Ààbò Ìṣàkóso Àwọn Àṣàyàn

Agbára Ìṣàkóso Àwọn Àṣàyàn ní blocklist tí a ti kọ́ sínú rẹ̀ tí ó ń dá aṣojú náà dúró láti ka tàbí kọ àwọn àṣàyàn WordPress tó ní ìmọ̀lára. Blocklist àìyípadà bo:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (àwọn àṣàyàn URL — lo irinṣẹ́ ìṣípadà pàtàkì fún wọ̀nyí)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (àwọn kọ́kọ́rọ́ aṣírí)
- `active_plugins`, `template` (mímú àfikún/theme ṣiṣẹ́ — lo àwọn agbára pàtàkì)
- Àwọn ẹ̀rí ẹnubodè ìsanwó àti àwọn kọ́kọ́rọ́ API tí àwọn àfikún tí a fi sílẹ̀ forúkọsílẹ̀

A lè fi àwọn ìwọlé míì kún un nípasẹ̀ filter `gratis_ai_agent_options_blocklist`.

## Àwọn Hook àti Filter

### Àwọn Action

- `gratis_ai_agent_ability_registered` — a máa ń tan án lẹ́yìn tí a bá forúkọsílẹ̀ agbára kan, ó gba orúkọ kilasi agbára náà
- `gratis_ai_agent_plan_started` — a máa ń tan án nígbà tí ètò site builder kan bẹ̀rẹ̀ ìmúṣẹ
- `gratis_ai_agent_plan_completed` — a máa ń tan án nígbà tí ètò kan bá parí ní àṣeyọrí
- `gratis_ai_agent_plan_failed` — a máa ń tan án nígbà tí ètò kan bá pàdé àṣìṣe tí a kò lè mú padà bọ̀ sípò

### Àwọn Filter

- `gratis_ai_agent_abilities` — ṣe àtúnṣe àtòjọ agbára tí a forúkọsílẹ̀ kí aṣojú náà tó kó o
- `gratis_ai_agent_options_blocklist` — faagun tàbí dín blocklist ààbò àwọn àṣàyàn kù
- `gratis_ai_agent_benchmark_questions` — fi àwọn ìbéèrè ìdánwò ìṣe àkànṣe kún àkójọpọ̀ ìdánilójú
- `gratis_ai_agent_plan_plugins` — ṣe àtúnṣe àtòjọ ìṣàwárí àfikún nígbà ìṣẹ̀dá ètò sáịtị
