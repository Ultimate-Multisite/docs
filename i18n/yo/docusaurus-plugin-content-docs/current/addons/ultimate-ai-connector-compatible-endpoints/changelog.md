# Ultimate AI Connector fún Compatible Endpoints Changelog

## Ẹya 2.0.0 — Tí a tu sílẹ̀ ní 2026-04-24 {#version-200--released-on-2026-04-24}

* Tuntun: Àtìlẹ́yìn olùpèsè-púpọ̀ — ṣètò ọ̀pọ̀ AI endpoints kí o sì tọ́ àwọn ìbéèrè pẹ̀lú automatic fallback kọjá àwọn olùpèsè.
* Àtúnṣe: Ìṣepọ̀ SDK olùpèsè-púpọ̀ pẹ̀lú provider IDs tó tọ́, registration URLs, àti HTTP filter scoping fún olùpèsè kọ̀ọ̀kan.
* Àtúnṣe: Káàdì olùpèsè tuntun ń auto-expand nísinsìnyí nígbà fífi kún; script cache busting nígbà ìmúdójúìwọ̀n plugin.
* Àtúnṣe: Dynamic provider class namespace fún eval() àti ìhùwàsí auto-expand fún àwọn olùpèsè tí a ṣẹ̀ṣẹ̀ fi kún.
* Àtúnṣe: Ó ń lo àwọn components Card/CardBody/CardHeader/CardDivider tó dúró ṣinṣin (kì í ṣe experimental mọ́) fún ìbámu WordPress 6.9+.
* Àtúnṣe: A rọ́pò DragHandle tí a kò ṣàlàyé pẹ̀lú unicode grip icon fún olùpèsè drag-to-reorder.
* Àtúnṣe: A mú blocking HTTP request tí ń ṣiṣẹ́ ní gbogbo ìgbà tí ojú-ewé bá kojú kúrò.
* Ti dára síi: GitHub Actions workflows ti gbéga sí Node.js 24.
