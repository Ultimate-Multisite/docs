# Ẹ̀kọ́ 7: Ṣíṣe É Di Tìrẹ

Kí àwọn oníbàárà rẹ má ṣe ní ìmọ̀lára pé wọ́n ń lo "plugin WordPress kan." Kí wọ́n ní ìmọ̀lára pé wọ́n ń lo FitSite -- pẹpẹ kan tí a kọ́ fún ilé-iṣẹ́ wọn. Ẹ̀kọ́ yìí bo ìṣàmì-ọjà, white-labeling, àti bí a ṣe ń mú kí pẹpẹ náà dà bí ọjà gidi.

## Ibi Tí A Dúró Sí

FitSite ti ní checkout flow tó ń ṣiṣẹ́ tí ó máa ń mú àwọn onílé studio amọdájú ara láti yíyan plan dé site tó wà láàyè. Bayìí a ó mú gbogbo ìrírí náà dà bí ọjà tó ṣọ̀kan, tó sì ní brand.

## Domain Pẹpẹ Rẹ

Ìpìlẹ̀ brand rẹ ni domain rẹ. Fún FitSite:

- **Domain àkọ́kọ́**: `fitsite.com` (site ìpolówó rẹ àti gbòǹgbò network)
- **Àwọn site oníbàárà**: `studioname.fitsite.com` (subdomains)
- **Àwọn domain àdáni**: Àwọn oníbàárà lórí Growth àti Pro plans lè map domain tiwọn

### Ṣíṣètò Domain Rẹ

1. Forúkọsílẹ̀ domain pẹpẹ rẹ
2. Darí rẹ sí hosting provider rẹ
3. Ṣètò wildcard DNS (`*.fitsite.com`) fún àwọn subdomain oníbàárà
4. Rí i dájú pé wildcard SSL ń ṣiṣẹ́

Wo [Bí a Ṣe Lè Ṣètò Domain Mapping](/user-guide/domain-mapping/how-to-configure-domain-mapping) fún ìtọ́nisọ́nà kíkún.

## White-Labeling Ìrírí Admin

Nígbà tí onílé studio amọdájú ara bá wọlé sí site dashboard wọn, kí wọ́n rí brand rẹ, kì í ṣe branding WordPress tàbí Ultimate Multisite.

### Ojú-ìwé Login Àdáni

Ṣe àtúnṣe ojú-ìwé login WordPress láti fi hàn:

- Logo FitSite rẹ
- Àwòrán abẹ́lẹ̀ tó yẹ fún amọdájú ara
- Àwọn àwọ̀ brand rẹ

### Branding Dashboard

Lo addon [Admin Page Creator](/addons/admin-page-creator) tàbí CSS àdáni láti:

- Rọ́pò logo WordPress pẹ̀lú logo FitSite rẹ
- Ṣe àtúnṣe ètò àwọ̀ admin kí ó bá brand rẹ mu
- Ṣàfikún widget dashboard àdáni pẹ̀lú àwọn quick links àti help resources tó dá lórí amọdájú ara

### Àwọn Ojú-ìwé Admin Àdáni

Ronú nípa dídá àwọn ojú-ìwé admin àdáni tí ó máa ń fi àwọn ìgbésẹ̀ tó ṣe pàtàkì jù hàn fún àwọn onílé studio amọdájú ara:

- "Ṣàtúnṣe Àkókò Kíláàsì Rẹ"
- "Ṣe Ìmúdójúìwọ̀n Profaili Àwọn Trainer"
- "Wo Site Rẹ"

Èyí dín learning curve kù nípa fífi àwọn ìgbésẹ̀ tó bá niche mu sí iwájú àti àárín dípò kí wọ́n sin wọ́n sínú menu WordPress boṣewa.

## Ṣíṣe Branding fún Àwọn Ìbánisọ̀rọ̀ Rẹ

Gbogbo email, invoice, àti ìfitónilétí gbọ́dọ̀ mú brand rẹ lágbára.

### Àwọn System Emails

Lọ sí **Ultimate Multisite > Settings > Emails** kí o sì ṣe àtúnṣe gbogbo system emails:

- **From name**: FitSite
- **From email**: hello@fitsite.com
- **Àwọn email templates**: Lo àwọn àwọ̀ brand rẹ àti logo rẹ
- **Èdè**: Kí ó jẹ́ ti amọdájú ara ní gbogbo ibìkan

Àwọn email pàtàkì láti ṣe àtúnṣe:

| Email | Ẹ̀yà Gbogboogbo | Ẹ̀yà FitSite |
|-------|----------------|--------------|
| Ìkíni káàbọ̀ | "Site tuntun rẹ ti ṣetán" | "Website studio amọdájú ara rẹ ti wà láàyè" |
| Rísítì sísanwo | "A ti gba sísanwo" | "A ti jẹ́rìí sísanwo subscription FitSite" |
| Trial ń parí | "Trial rẹ máa parí láìpẹ́" | "Trial FitSite rẹ máa parí ní ọjọ́ 3 -- jẹ́ kí website studio rẹ wà láàyè" |

### Àwọn Invoices

Ṣe àtúnṣe àwọn invoice templates pẹ̀lú:

- Logo FitSite rẹ àti àwọn àwọ̀ brand
- Àwọn àlàyé business rẹ
- Àwọn orúkọ ọjà tó dá lórí amọdájú ara (kì í ṣe generic plan IDs)

## Site Tí Oníbàárà Máa Rí

Domain àkọ́kọ́ rẹ (`fitsite.com`) nílò site ìpolówó tó ń tà pẹpẹ náà. Èyí yàtọ̀ sí admin network Ultimate Multisite -- ó jẹ́ ojú business rẹ fún gbogbo ènìyàn.

Àwọn ojú-ìwé pàtàkì:

- **Homepage**: Ìlérí iye tó ṣe kedere fún àwọn business amọdájú ara
- **Features**: Ohun tí FitSite ń ṣe, ní ọ̀rọ̀ amọdájú ara
- **Pricing**: Àwọn plan mẹ́ta rẹ pẹ̀lú àfiwé feature tó dá lórí niche
- **Àwọn Àpẹẹrẹ**: Ṣàfihàn àwọn site tí a kọ́ lórí pẹpẹ náà
- **Sign Up**: Àwọn link sí checkout form rẹ

:::tip Site Ìpolówó Rẹ Lè Jẹ́ Site Network
Dá site ìpolówó rẹ gẹ́gẹ́ bí site kan nínú network tirẹ. Èyí jẹ́ kí o lè ṣàkóso rẹ láti dashboard kan náà, ó sì ń fi agbára pẹpẹ tirẹ hàn.
:::

## Domain Àdáni fún Àwọn Oníbàárà

Fún àwọn oníbàárà lórí plans tí ó ní custom domains, kọ ìlànà náà sílẹ̀ kedere:

1. Oníbàárà forúkọsílẹ̀ tàbí gbe domain wọn lọ sí registrar
2. Oníbàárà ṣe ìmúdójúìwọ̀n DNS láti tọ́ka sí pẹpẹ rẹ (pèsè records gangan)
3. Ultimate Multisite ń bójú tó domain mapping àti SSL

Dá help article tàbí knowledge base entry kan sílẹ̀ pàápàá fún ìlànà yìí, tí a kọ fún àwọn onílé studio amọdájú ara tí kì í ṣe onímọ̀ technical.

## Network FitSite Títí Di Báyìí

```
Network FitSite
├── WordPress Multisite (ipo subdomain)
├── Ultimate Multisite (tí a ti ṣètò + tí a ti ṣe branding)
├── Domain Pẹpẹ (fitsite.com + wildcard SSL)
├── Àwọn Site Templates (Studio Essential, Gym Pro, Fitness Chain)
├── Àwọn Ọjà (Starter, Growth, Pro + Order Bumps)
├── Checkout Flow (tó dá lórí niche, tí a ti dán wò)
├── Branding
│   ├── Ojú-ìwé login àdáni
│   ├── Admin dashboard tó ní brand
│   ├── Àwọn system emails tó dá lórí niche
│   ├── Àwọn invoices tó ní brand
│   └── Site ìpolówó lórí fitsite.com
└── Ṣetán fún onboarding flow (ẹ̀kọ́ tó kàn)
```

## Ohun Tí A Kọ́ Nínú Ẹ̀kọ́ Yìí

- **Domain pẹpẹ àti DNS** tí a ṣètò fún ìrírí tó ní brand
- **Admin white-labeled** pẹ̀lú branding FitSite ní gbogbo ibìkan
- **Àwọn ìbánisọ̀rọ̀ tí a ṣe àtúnṣe** -- emails, invoices, àti notifications gbogbo wọn bá brand mu
- **Site ìpolówó kan** tó ń tà FitSite fún àwọn onílé studio amọdájú ara
- **Ìwé ìtọ́nisọ́nà domain àdáni** fún àwọn oníbàárà tó fẹ́ domain tiwọn
