# Ẹ̀kọ́ 8: Ìtọ́sọ́nà Oníbàárà Tuntun {#lesson-8-customer-onboarding}

Mímú oníbàárà forúkọsílẹ̀ jẹ́ ìdajì iṣẹ́ nìkan. Bí wọ́n bá wọlé, tí ohun gbogbo bá dà bí ó ti pọ̀ jù fún wọn, tí wọn kò sì padà mọ́, o ti pàdánù wọn. Ẹ̀kọ́ yìí ṣe àpẹrẹ ìrírí tí yóò yí ìforúkọsílẹ̀ tuntun padà sí oníbàárà tó ń ṣiṣẹ́, tó sì ní ìfarapa.

## Ibi Tí A Dúró Sí {#where-we-left-off}

FitSite ti ní àmì-ẹ̀yà pátápátá pẹ̀lú ìṣàn checkout tó ń ṣiṣẹ́. Ní báyìí a dojú kọ ohun tó máa ṣẹlẹ̀ lẹ́yìn tí onílé studio amọdájú ara bá parí ìforúkọsílẹ̀, tí ó sì dé ojúlé tuntun rẹ fún ìgbà àkọ́kọ́.

## Ìdí Tí Ìtọ́sọ́nà Oníbàárà Tuntun Fi Ṣe Pàtàkì {#why-onboarding-matters}

Ìṣẹ́jú 30 àkọ́kọ́ lẹ́yìn ìforúkọsílẹ̀ ló ń pinnu bóyá oníbàárà máa dúró tàbí máa lọ. Onílé studio amọdájú ara tí ó:

- Wọlé, tí ó sì rí ojúlé tó ti dà bí oju opo wẹẹbu amọdájú ara → dúró
- Mọ ohun tí ó yẹ kí ó ṣe lẹ́sẹ̀kẹsẹ̀ → dúró
- Ní ìmọ̀lára pé ó sọnù nínú WordPress Dashboard gbogbogbò → lọ

Àwọn template niche rẹ (Ẹ̀kọ́ 4) ń bójú tó kókó àkọ́kọ́. Ẹ̀kọ́ yìí ń bójú tó èkejì.

## Ìrírí Wíwọlé Àkọ́kọ́ {#the-first-login-experience}

### Widget Dashboard Ìkáàbọ̀ {#welcome-dashboard-widget}

Dá widget Dashboard àdáni sílẹ̀ tí yóò kí àwọn oníbàárà tuntun, tí yóò sì tọ́ wọn nípasẹ̀ setup. Èyí yẹ kí ó hàn gbangba nígbà tí wọ́n bá kọ́kọ́ wọlé.

**Ìbẹ̀rẹ̀ Kánkán FitSite:**

1. **Fi orúkọ studio rẹ àti logo kún un** -- Ọ̀nà asopọ̀ sí àwọn ètò Customizer tàbí Site Identity
2. **Ṣe imudójúìwọ̀n sí ìṣètò kíláàsì rẹ** -- Ọ̀nà asopọ̀ taara sí editor ojúewé Classes
3. **Fi àwọn trainer rẹ kún un** -- Ọ̀nà asopọ̀ sí editor ojúewé Trainers
4. **Ṣètò àwọn àlàyé ìkànìyàn rẹ** -- Ọ̀nà asopọ̀ sí editor ojúewé Contact
5. **Wo àyẹ̀wò ojúlé rẹ** -- Ọ̀nà asopọ̀ sí frontend

Ìgbésẹ̀ kọ̀ọ̀kan ní ọ̀nà asopọ̀ taara sí ojúewé tàbí ètò tó yẹ. Kò sí wíwá kiri nínú àwọn menu.

### Rọrùn Dashboard {#simplify-the-dashboard}

Àwọn oníbàárà tuntun kò nílò láti rí gbogbo ohun menu WordPress. Ronú nípa:

- Fífi àwọn ohun menu tí kò ṣe pàtàkì sí ìṣàkóso ojúlé amọdájú ara pamọ́ (fún àpẹẹrẹ, Comments bí a kò bá lò ó)
- Ṣíṣètò menu padà láti fi àwọn ohun tí a máa ń lò jù síwájú
- Fífi àwọn label menu àdáni kún un tí ó ní ìtumọ̀ fún niche náà ("Studio Rẹ" dípò "Appearance")

Addon [Alákóso Plugin & Theme](/addons/plugin-and-theme-manager) lè ràn ọ́ lọ́wọ́ láti ṣàkóso ohun tí àwọn oníbàárà ń rí.

## Ọ̀wọ́ Imeeli Ìkáàbọ̀ {#welcome-email-sequence}

Imeeli ìkáàbọ̀ kan ṣoṣo kò tó. Ṣètò ọ̀wọ́ kan tí yóò tọ́ àwọn oníbàárà nípasẹ̀ ọ̀sẹ̀ àkọ́kọ́ wọn:

### Imeeli 1: Ìkáàbọ̀ (Lẹ́sẹ̀kẹsẹ̀ lẹ́yìn ìforúkọsílẹ̀) {#email-1-welcome-immediately-after-signup}

- Kókó: "Káàbọ̀ sí FitSite -- oju opo wẹẹbu studio rẹ ti wà lórí afẹ́fẹ́"
- Àkóónú: Ọ̀nà asopọ̀ wíwọlé, àwọn ìgbésẹ̀ ìbẹ̀rẹ̀ kánkán, ọ̀nà asopọ̀ sí àwọn ohun èlò ìrànlọ́wọ́
- Ohùn: Ayọ̀, ìgbani níyànjú, pàtó sí amọdájú ara

### Imeeli 2: Àwọn Àṣeyọrí Kánkán (Ọjọ́ 1) {#email-2-quick-wins-day-1}

- Kókó: "Ohun 3 láti ṣe kọ́kọ́ lórí FitSite rẹ"
- Àkóónú: Fi logo rẹ kún un, ṣe imudójúìwọ̀n sí àwòrán hero ojúewé ilé, fi ìṣètò kíláàsì rẹ kún un
- Fi àwọn screenshot tó fi ibi tí a gbọ́dọ̀ tẹ̀ hàn gẹ́gẹ́ bí ó ti rí kún un

### Imeeli 3: Ṣe É Ní Tirẹ (Ọjọ́ 3) {#email-3-make-it-yours-day-3}

- Kókó: "Jẹ́ kí ojúlé amọdájú ara rẹ yàtọ̀"
- Àkóónú: Ṣe àdáni àwọn àwọ̀, fi àwọn fọ́tò trainer kún un, kọ ìtàn studio rẹ
- Ọ̀nà asopọ̀ sí àwọn àpẹẹrẹ ojúlé amọdájú ara tó dára lórí platform náà

### Imeeli 4: Lọ Live (Ọjọ́ 7) {#email-4-go-live-day-7}

- Kókó: "Ṣé o ti ṣetán láti pín FitSite rẹ pẹ̀lú ayé?"
- Àkóónú: Àtòjọ ohun láti jẹ́risi kí o tó pín in, bí a ṣe lè so domain àdáni pọ̀ (bí ó bá wà lórí Growth/Pro), àwọn ìmòràn pínpín lórí social

:::tip Ìmúlò Imeeli Aládàáṣiṣẹ́
Lo [Webhooks](/user-guide/integrations/webhooks) tàbí [Zapier](/user-guide/integrations/zapier) láti fa àwọn imeeli wọ̀nyí nípasẹ̀ platform ìtajà imeeli rẹ. Èyí fún ọ ní àkóso púpọ̀ síi lórí àsìkò, ó sì jẹ́ kí o lè tọ́pa ìfarapa.
:::

## Àwọn Ohun Èlò Ìrànlọ́wọ́ {#help-resources}

Dá àkóónú ìrànlọ́wọ́ pàtó sí niche sílẹ̀ tí yóò dáhùn àwọn ìbéèrè tí àwọn onílé studio amọdájú ara máa ń béèrè gan-an:

### Àwọn Àpilẹ̀kọ Knowledge Base {#knowledge-base-articles}

- "Bí a ṣe lè ṣe imudójúìwọ̀n sí ìṣètò kíláàsì rẹ"
- "Fífi àti ṣíṣàtúnṣe àwọn profaili trainer"
- "Yíyí logo àti àwọn àwọ̀ studio rẹ padà"
- "Sísopọ̀ orúkọ domain tirẹ" (fún àwọn oníbàárà Growth/Pro)
- "Fífi widget ìfipamọ́ àkókò kún ojúlé rẹ"

Kọ ìwọ̀nyí fún àwọn olumulo tí kì í ṣe onímọ̀ imọ̀-ẹ̀rọ. Lo àwọn screenshot. Yẹra fún jargon WordPress.

### Àwọn Video Walkthrough {#video-walkthroughs}

Àwọn ìgbasilẹ iboju kúkúrú (ìṣẹ́jú 2-3) tó ń fi hàn:

- Wíwọlé àkọ́kọ́ àti ìtọ́nisọ́nà
- Ṣíṣàtúnṣe ojúewé ilé
- Ṣíṣe imudójúìwọ̀n sí ìṣètò kíláàsì
- Fífi trainer tuntun kún un

Ìwọ̀nyí kò nílò láti jẹ́ àwọn iṣelọpọ tó péye gan-an. Ohun tó ṣe pàtàkì ni kedere, ìrànlọ́wọ́, àti jíjẹ́ pàtó sí niche.

## Ojúewé Account {#the-account-page}

Ultimate Multisite ní [Ojúewé Account](/user-guide/client-management/account-page) tí ó dojú kọ oníbàárà níbi tí àwọn oníbàárà ti ń ṣàkóso subscription wọn. Ṣe é ní àdáni láti:

- Fi plan FitSite lọwọlọwọ wọn hàn
- Ṣàfihàn àwọn aṣayan upgrade pẹ̀lú àwọn ànfààní pàtó sí amọdájú ara
- Pèsè ìtàn billing àti àwọn gbigba invoice sílẹ̀
- Sọ pọ̀ mọ́ àwọn ohun èlò ìrànlọ́wọ́

## Dídiwọ̀n Àṣeyọrí Ìtọ́sọ́nà Oníbàárà Tuntun {#measuring-onboarding-success}

Tọ́pa àwọn àmì wọ̀nyí láti mọ bóyá ìtọ́sọ́nà oníbàárà tuntun rẹ ń ṣiṣẹ́:

- **Oṣùwọ̀n ìmúlò**: Iwọn ogorun wo nínú àwọn ìforúkọsílẹ̀ ló ṣe àdáni ojúlé wọn gan-an láàárín ọ̀sẹ̀ àkọ́kọ́?
- **Àwọn wíwọlé ọ̀sẹ̀ àkọ́kọ́**: Ìgbà mélòó ni oníbàárà tuntun ń wọlé ní ọ̀sẹ̀ àkọ́kọ́ wọn?
- **Àwọn tikẹ́ẹ̀tì support láti ọdọ àwọn oníbàárà tuntun**: Ìwọ̀n tó ga túmọ̀ sí pé ìtọ́sọ́nà oníbàárà tuntun rẹ ní àwọn ààlà
- **Ìyípadà láti trial sí paid**: Bí o bá ń fúnni ní trial, iwọn ogorun mélòó ló yípadà?
