# Ẹ̀kọ́ 10: Ọjọ́ Ìfilọlẹ̀

Gbogbo nkan ti kọ́ tán. Kí o tó ṣí ilẹ̀kùn, ẹ̀kọ́ yìí máa rìn ọ kọjá àtòjọ ohun tí a gbọ́dọ̀ ṣàyẹ̀wò ṣáájú ìfilọlẹ̀ láti rí i dájú pé kò sí ohun tó bàjẹ́, tó sọnù, tàbí tó lè mú ìtìjú wá.

## Ibi Tá A Dúró Sí

FitSite ti ní àwọn template, plan, checkout, branding, onboarding, àti pricing tí a ti ṣètò pátápátá. Ní báyìí a máa jẹ́risi pé gbogbo nkan ń ṣiṣẹ́, a sì máa lọ live.

## Àtòjọ Ṣíṣe Àyẹ̀wò Ṣáájú Ìfilọlẹ̀

Ṣiṣẹ́ lórí gbogbo nkan. Má ṣe fo èyíkéyìí.

### Amáyédẹrùn Platform

- [ ] Hosting dúró ṣinṣin, ó sì ń ṣiṣẹ́ dáadáa lábẹ́ ìrù iṣẹ́ púpọ̀
- [ ] Wildcard SSL ń ṣiṣẹ́, gbogbo subdomain sì ń ṣiṣẹ́ lórí HTTPS
- [ ] Domain mapping ń ṣiṣẹ́ -- dán an wò nípa dídá site kan sílẹ̀ àti mapping domain àkànṣe kan
- [ ] A ti ṣètò backups, a sì ti dán wọn wò (restore ó kere tán ọ̀kan láti jẹ́risi)
- [ ] Monitoring wà nílẹ̀ -- ìwọ yóò mọ̀ bí platform bá lọ silẹ̀

### Àwọn Template

- [ ] Gbogbo template mẹ́ta ń ṣí dáadáa lórí àwọn site tuntun
- [ ] Placeholder content wúlò, kò sì ní àṣìṣe ìkọ̀wé
- [ ] Gbogbo àwòrán ní ìwé-àṣẹ tó yẹ (kò sí stock photo watermark)
- [ ] Mobile responsiveness ń ṣiṣẹ́ lórí gbogbo ojúewé template
- [ ] Iyára ìkójọpọ̀ ojúewé jẹ́ itẹ́wọ́gbà (dán an wò pẹ̀lú irinṣẹ́ bí GTmetrix tàbí PageSpeed Insights)
- [ ] Kò sí link tó bàjẹ́ tàbí asset tó sọnù lórí template kankan

### Àwọn Plan àti Ọjà

- [ ] Gbogbo plan mẹ́ta ń ṣiṣẹ́, a sì lè rí wọn
- [ ] Àpejuwe plan péye, ó sì bá niche mu
- [ ] Pricing péye (osù-osù àti ọdọọdún)
- [ ] A ti ṣètò setup fees lórí àwọn plan tó tọ́
- [ ] Trial period ń ṣiṣẹ́ lórí Starter plan
- [ ] Order bumps hàn dáadáa nígbà checkout
- [ ] Àwọn ìdíwọ̀n plugin àti theme jẹ́ fífi múlẹ̀ dáadáa gẹ́gẹ́ bí plan kọọkan

### Ìṣàn Checkout

- [ ] Pari ìforúkọsílẹ̀ ìdánwò kikún lórí plan kọọkan (lo test payment mode)
- [ ] Yíyan template fi àwọn template tó tọ́ hàn gẹ́gẹ́ bí plan
- [ ] Ìsanwó kọjá ní aṣeyọrí
- [ ] Oníbàárà gba email ìkáàbọ̀ lẹ́yìn ìforúkọsílẹ̀
- [ ] A dá site tuntun sílẹ̀ pẹ̀lú template tó tọ́
- [ ] Oníbàárà lè log in sí site tuntun wọn lẹ́sẹ̀kẹsẹ̀
- [ ] Discount codes ń ṣiṣẹ́ dáadáa

### Branding

- [ ] Ojúewé login fi branding FitSite hàn
- [ ] Admin dashboard fi branding FitSite hàn
- [ ] Gbogbo system emails lo branding FitSite àti èdè tó dá lórí amọdaju ara
- [ ] Invoices hàn dáadáa pẹ̀lú alaye ilé-iṣẹ́ rẹ
- [ ] Site marketing wà live, ó sì ní link sí fọọmu checkout

### Onboarding

- [ ] Quick Start widget hàn lórí Dashboard àwọn oníbàárà tuntun
- [ ] Gbogbo link Quick Start tọ́ka sí àwọn ojúewé tó tọ́
- [ ] A ti ṣètò ìtẹ̀lé email ìkáàbọ̀, a sì ti dán an wò
- [ ] A ti tẹ àwọn àpilẹ̀kọ knowledge base jáde, wọ́n sì ṣeé wọlé sí
- [ ] Ojúewé Account fi alaye plan tó tọ́ àti àwọn aṣayan upgrade hàn

### Òfin àti Iṣòwò

- [ ] Terms of service ti jẹ́ títẹ̀jáde, a sì ti ṣe link rẹ láti checkout
- [ ] Privacy policy ti jẹ́ títẹ̀jáde, ó sì ṣeé wọlé sí
- [ ] Refund policy ti jẹ́ dídéfin, a sì ti kọ ọ sílẹ̀
- [ ] A ti ṣètò business entity fún gbigba ìsanwó
- [ ] Payment gateway wà ní live mode (kì í ṣe test mode)
- [ ] Tax configuration tọ́ fún agbègbè òfin rẹ

## Soft Launch vs. Hard Launch

Ronú nípa ìfilọlẹ̀ ìpele méjì:

### Ìpele 1: Soft Launch

Pe àwọn onílé fitness studio 5-10 kí wọ́n forúkọ sílẹ̀ ṣáájú ìfilọlẹ̀ fún gbogbogbò. Ìwọ̀nyí ni àwọn oníbàárà beta rẹ. Fún wọn ní ẹdinwo pàtàkì (50% off fún gbogbo ayé, tàbí oṣù 3 ọ̀fẹ́) ní pàṣípààrọ̀ fún:

- Àbá òtítọ́ lórí ìrírí ìforúkọsílẹ̀ àti onboarding
- Ìyọ̀nda láti lo site wọn gẹ́gẹ́ bí àpẹẹrẹ ìfihàn
- Fífìròyìn eyikeyi bug tàbí ìṣòro tí wọ́n bá pàdé

Èyí fún ọ ní àbá oníbàárà gidi àti àwọn site live láti fi hàn ṣáájú kí o tó ṣí sí gbogbo ènìyàn.

### Ìpele 2: Ìfilọlẹ̀ Fún Gbogbo Ènìyàn

Nígbà tí a bá ti fi àbá soft launch kún un:

- Yí payment gateway padà sí live mode
- Tẹ site marketing rẹ jáde
- Bẹ̀rẹ̀ gbigba àwọn oníbàárà (Ẹ̀kọ́ 11)
- Kéde lórí àwọn ikanni ilé-iṣẹ́ fitness tó yẹ

## Àwọn Ìgbésẹ̀ Ọjọ́ Ìfilọlẹ̀

Ní ọjọ́ tí o bá lọ síta fún gbogbo ènìyàn:

1. **Yí padà sí live payments** -- pa test mode lórí payment gateway rẹ
2. **Jẹ́risi lẹ́ẹ̀kan sí i** -- ṣe ìforúkọsílẹ̀ ìdánwò pípé pẹ̀lú ìsanwó gidi (refund fún ara rẹ lẹ́yìn náà)
3. **Ṣọ́ra kí o sì máa tọ́pinpin** -- wo àwọn errors, ìforúkọsílẹ̀ tó kùnà, tàbí ìṣòro ìsanwó
4. **Wà nílẹ̀** -- àwọn oníbàárà gidi àkọ́kọ́ rẹ lè nílò ìrànlọ́wọ́, ìdáhùn yára sì ń kọ́ ìgbẹ́kẹ̀lé
5. **Ṣe ayẹyẹ díẹ̀** -- lẹ́yìn náà padà sí iṣẹ́

## Ohun Tó Lè Ṣẹlẹ̀ Lọ́nà Tí Kò Tọ́

Múra sílẹ̀ fún:

- **Àwọn ìṣòro payment gateway**: Ní contact support ti gateway provider rẹ ní imurasilẹ
- **Àwọn ìṣòro SSL certificate**: Mọ bí a ṣe ń ṣàyẹ̀wò àti renew certificates
- **Ìkùnà fífi email ránṣẹ́**: Dán an wò pé emails dé ní tòótọ́ (ṣàyẹ̀wò spam folders)
- **Performance lábẹ́ ìrù iṣẹ́ púpọ̀**: Bí o bá ní traffic spike, mọ bí o ṣe lè scale hosting rẹ
- **Ìdàrúdàpọ̀ oníbàárà**: Ní knowledge base àti àwọn ikanni support rẹ ní imurasilẹ

## Ohun Tá A Kọ́ Nínú Ẹ̀kọ́ Yìí

- **Àtòjọ ṣíṣe àyẹ̀wò ṣáájú ìfilọlẹ̀ tó péye** tó bo amáyédẹrùn, content, ìsanwó, àti òfin

- **Àwọn ìlànà ọjọ́ ìfilọlẹ̀** láti lọ live pẹ̀lú ìdánilójú
- **Ètò ìmúra sí ìṣòro** fún àwọn ìṣòro ọjọ́ ìfilọlẹ̀ tó wọ́pọ̀
