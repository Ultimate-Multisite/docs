# Ojú-ìwé Account ti Oníbàárà Rẹ (v2)

_**ÀKÍYÈSÍ PÀTÀKÌ: Àpilẹ̀kọ yìí tọ́ka sí Ultimate Multisite version 2.x.**_

Nígbà tí àwọn oníbàárà bá ṣe alabapin sí plan kan lórí nẹtiwọọki rẹ, wọ́n máa ní àyè sí aaye wẹẹbu kan àti dashboard rẹ̀ pẹ̀lú ìwífún pàtàkì nípa àwọn ìsanwó wọn, àwọn membership, àwọn domain, àwọn ààlà plan, abbl...

Nínú ìtọ́nisọ́nà yìí, a ó tọ́ ọ kọjá ojú-ìwé account oníbàárà náà, iwọ yóò sì rí ohun tí àwọn oníbàárà rẹ lè rí àti ṣe nínú rẹ̀.

## Ojú-ìwé Account

Ojú-ìwé account lè wọlé sí nípa títẹ **Account** nínú dashboard oníbàárà rẹ.

![Mẹ́nù Account nínú dashboard oníbàárà](/img/account-page/account-menu.png)

Lórí àwọn nẹtiwọọki tenant aláṣẹ-ara, Ultimate Multisite v2.13.0 pa ìrírí ìṣàkóso oníbàárà yìí mọ́ lórí aaye àkọ́kọ́. Bí oníbàárà bá ṣí account, checkout, billing, invoice, ìṣàkóso-aaye, yíyí-àwòṣe-padà, tàbí àwọn ìṣe ìmápù-domain láti tenant aláṣẹ-ara, ìṣe náà máa tọ́ka padà sí panel oníbàárà ti aaye-àkọ́kọ́ kí àwọn àkọsílẹ̀ billing àti membership ti nẹtiwọọki lè dúró gẹ́gẹ́ bí orísun tó láṣẹ.

Nígbà tí oníbàárà bá dé láti tenant aláṣẹ-ara, panel oníbàárà ti aaye-àkọ́kọ́ lè ní link ìpadà sí aaye tenant náà. Link ìpadà náà ni a máa fi hàn nìkan nígbà tí Ultimate Multisite bá lè jẹ́rìí pé ibi ìpadà náà jẹ́ ọ̀kan lára àwọn aaye oníbàárà náà, èyí tó dènà àwọn redirect aláìlábòójútó nígbà tó ń pa workflow tenant mọ́.

![Àkótán ojú-ìwé account oníbàárà](/img/account-page/overview.png)

Lẹ́yìn tí oníbàárà bá tẹ ẹ́, wọ́n máa rí àkótán membership wọn, àdírẹ́sì billing, àwọn invoice, àwọn domain, àwọn ààlà aaye, wọ́n yóò sì tún lè yí ****Àwòṣe Aaye** padà (bí ó bá jẹ́ pé a gbà á láàyè nínú nẹtiwọọki rẹ)**.

Wọ́n tún lè yí membership padà sí plan míì, tàbí ra package tàbí iṣẹ́ míì tí o ń pèsè. Ẹ jẹ́ ká wo apá kọ̀ọ̀kan lọ́tọ̀ọ̀tọ̀.

### Àkótán Membership Rẹ:

Block àkọ́kọ́ ní ìsàlẹ̀ orúkọ aaye wẹẹbu àwọn oníbàárà rẹ fi àkótán plan lọwọlọwọ wọn àti àwọn iṣẹ́/package tí a rà pẹ̀lú rẹ̀ hàn. Block náà tún fi nọ́mbà membership hàn, iye àkọ́kọ́ tí a san fún un, iye tí plan àti iṣẹ́/package kankan ń ná, àti iye ìgbà tí a gba wọn ní owó fún membership yìí. Wọ́n tún lè rí bóyá membership náà jẹ́ **Active** , **Expired** tàbí **Canceled**.

![Àkótán membership tó ń fi plan, iye owó, àti alaye billing hàn](/img/account-page/membership-card.png)

Ní ìsàlẹ̀ block yìí, àwọn oníbàárà rẹ lè rí àwọn block **Nípa Aaye Yìí** àti **Àwọn Ààlà Aaye**. Àwọn block wọ̀nyí fi gbogbo àwọn ààlà tó wà pẹ̀lú plan wọn hàn fún wọn: ààyè disk, àwọn post, àwọn ojú-ìwé, àwọn ìbẹ̀wò, abbl... A lè ṣàtúnṣe àwọn ààlà wọ̀nyí lórí ojú-ìwé plan kọ̀ọ̀kan ní **Ultimate Multisite > Àwọn Ọjà**.

![Àwọn block Nípa Aaye Yìí àti Àwọn Ààlà Aaye tó ń fi àwọn ààlà plan hàn](/img/account-page/site-limits.png)

Ní apá ọ̀tún **Membership Rẹ** , àwọn oníbàárà lè tẹ **Yí Padà**. Èyí yóò fi gbogbo àwọn plan àti package/ìṣẹ́ tó wà hàn fún wọn. Bí wọ́n bá yan plan míì, àwọn ààlà fún plan náà yóò bẹ̀rẹ̀ sí í ṣiṣẹ́ dípò àwọn ààlà lọwọlọwọ ti membership náà - kò ṣe pàtàkì bóyá wọ́n ń dín un kù tàbí ń gbé e ga.

Ní báyìí, bí àwọn oníbàárà rẹ bá yan láti ra package tàbí iṣẹ́ fún membership lọwọlọwọ yìí - bíi ààyè disk tàbí ìbẹ̀wò púpọ̀ síi - membership lọwọlọwọ náà kò ní yí padà, ṣùgbọ́n àwọn package tuntun nìkan ni a ó fi kún un.

Ṣàkíyèsí pé a kò lè fi coupon code kún ojú-ìwé ìyípadà membership yìí. Bí oníbàárà bá lo coupon code ní ìrà membership àkọ́kọ́, code náà yóò tún kan membership tuntun yìí.

### Ṣíṣe Àfikún sí Àdírẹ́sì Billing:

Lórí ojú-ìwé account, àwọn oníbàárà rẹ tún lè ṣe àfikún sí àdírẹ́sì billing wọn. Wọ́n kan nílò láti tẹ **Ṣe Àfikún** lẹ́gbẹ̀ẹ́ _Àdírẹ́sì Billing_.

![Apá Àdírẹ́sì Billing pẹ̀lú bọ́tìnì Ṣe Àfikún](/img/account-page/billing-address.png)

Fèrèsé tuntun yóò farahàn sí oníbàárà rẹ. Gbogbo ohun tó nílò láti ṣe ni láti kún àdírẹ́sì tuntun náà kí ó sì tẹ _Fi Àwọn Àyípadà Pamọ́_.

![Fọ́ọ̀mù ṣíṣe àfikún sí àdírẹ́sì billing](/img/account-page/billing-address-form.png)

### Yíyí Àwòṣe Aaye Padà:

Láti gba àwọn oníbàárà rẹ láàyè láti yí àwọn àwòṣe aaye wọn padà, o nílò láti lọ sí **Ultimate Multisite > Ètò > Àwọn Aaye** kí o sì tan aṣayan **Gba Yíyí Àwòṣe Padà Láàyè**.

Bákannáà, lórí **Ultimate Multisite > Àwọn Ọjà**, yan àwọn plan rẹ kí o sì lọ sí tab **Àwọn Àwòṣe Aaye**. Rí i dájú pé aṣayan **Gba Àwọn Àwòṣe Aaye Láàyè** ti tan, àti lórí **Ipo Yíyan Àwòṣe Aaye** , aṣayan **Yan Àwọn Àwòṣe Aaye Tó Wà** ni a yan.

![Tab àwọn àwòṣe aaye ọjà pẹ̀lú ipo yíyan àwòṣe](/img/config/product-site-templates.png)

Ìwọ yóò lè rí gbogbo àwọn àwòṣe aaye tó wà lórí aaye wẹẹbu rẹ. Yan èyí tí o fẹ́ kí ó wà fún àwọn oníbàárà rẹ àti èyí tí o kò fẹ́ kí ó wà fún àwọn oníbàárà tó ṣe alabapin lábẹ́ plan yìí. Ṣàkíyèsí pé àwọn aṣayan yìí tún kan fọ́ọ̀mù checkout, nítorí náà àwòṣe kankan tí a yan gẹ́gẹ́ bí **Kò Sí Nílẹ̀** kò ní hàn lórí ojú-ìwé ìforúkọsílẹ̀ fún plan yìí.

Ní báyìí àwọn oníbàárà rẹ lè tẹ **Yí Àwòṣe Aaye Padà** nínú ojú-ìwé account wọn.

![Bọ́tìnì Yí Àwòṣe Aaye Padà lórí ojú-ìwé account](/img/account-page/change-template-button.png)

Ultimate Multisite 2.10.0 fi panel yíyí àwòṣe padà tí a tún ṣe hàn. Panel náà bẹ̀rẹ̀ pẹ̀lú **káàdì-àwòṣe-lọwọlọwọ** kí àwọn oníbàárà lè rí àwòṣe wo ni ó ń ṣiṣẹ́ kí wọ́n tó yan rirọ́pò.

Grid tí ń dúró pẹ́ ti àwọn àwòṣe aaye tó wà máa ń hàn nígbà tí àwọn oníbàárà ń ṣàyẹ̀wò àwọn aṣayan wọn. Èyí ràn wọ́n lọ́wọ́ láti fi àwọn àwòṣe tí a gbà láàyè fún plan wọn wé ara wọn láì padanu ìfojúsọ́nà yíyan lọwọlọwọ.

![Àtòjọ àwọn àwòṣe aaye tó wà fún plan náà](/img/config/site-templates-list.png)

Lẹ́yìn yíyan èyí tí wọ́n fẹ́ yí padà sí, a ó beere lọwọ wọn láti jẹ́rìí ìyípadà náà.

![Dialog ìjẹ́rìí yíyí àwòṣe aaye padà](/img/account-page/template-switch-confirm.png)

Lẹ́yìn títan ìjẹ́rìí náà àti títẹ **Ṣe Switch** , àwòṣe aaye tuntun náà ni a ó lò lórí aaye wẹẹbu oníbàárà rẹ.

Àwọn oníbàárà tún lè lo **Reset current template** láti inú panel yìí nígbà tí wọ́n bá fẹ́ tún site náà padà sí template tí a yàn fún un lọ́wọ́lọ́wọ́. Bíi yíyí padà sí template míì, títún template ṣe lè kọ́ àkóónú site lórí, nítorí náà àwọn oníbàárà yẹ kí wọ́n jẹ́rìí sí i nìkan nígbà tí wọ́n bá lóye ìṣe ìtúnṣe náà.

### Fífi Àwọn Domain Àkànṣe Kún Un:

Àwọn oníbàárà rẹ tún máa ní àṣàyàn láti fi domain àkànṣe kún un fún plan yìí lórí ojú ewé account wọn. Láti jẹ́ kí àwọn oníbàárà rẹ lè lo àwọn domain àkànṣe, lọ sí **Ultimate Multisite > Settings >** **Domain Mapping**.

Tan àṣàyàn **Enable Domain Mapping**. Èyí yóò jẹ́ kí àwọn oníbàárà rẹ lè lo àwọn domain àkànṣe ní ìpele network.

Má ṣe gbàgbé láti tún ṣàyẹ̀wò bóyá domain mapping ti ṣiṣẹ́ lórí ìpìlẹ̀ product - nítorí pé o lè dín product kan kù kí ó má jẹ́ kí àwọn oníbàárà rẹ lo àwọn domain àkànṣe.

Lọ sí **Ultimate Multisite > Products**. Yan plan tí o fẹ́, kí o sì lọ sí tab **Custom Domains**. Tan àṣàyàn **Allow Custom Domains**.

![Tab Custom Domains pẹ̀lú toggle Allow Custom Domains](/img/config/product-custom-domains.png)

Èyí yóò jẹ́ kí gbogbo àwọn oníbàárà tí wọ́n ṣe alabapin sí plan pàtó yìí lè lo àwọn domain àkànṣe. Ní báyìí, lórí ojú ewé Account, àwọn oníbàárà rẹ lè fi domain àkànṣe kún un nípa títẹ **Add Domain**.

![Bọ́tìnì Add Domain lórí ojú ewé account](/img/account-page/add-domain-button.png)

Window àkọ́kọ́ tí ó ṣí yóò fi ìfiránṣẹ́ hàn sí àwọn oníbàárà rẹ tí yóò kọ́ wọn nípa bí wọ́n ṣe lè ṣe àtúnṣe àwọn àkọsílẹ̀ DNS wọn kí domain àkànṣe yìí lè ṣiṣẹ́ lórí network rẹ.

![Àwọn ìtọnisọna DNS tí a fi hàn nígbà fífi domain àkànṣe kún un](/img/account-page/add-domain-dns.png)

Ìfiránṣẹ́ yìí lè ṣe àtúnṣe (láti ọwọ́ rẹ) lórí **Ultimate Multisite > Settings > Domain Mapping > Add New Domain Instructions**.

![Ètò Add New Domain Instructions nínú Domain Mapping](/img/config/settings-domain-mapping.png)

Èyí ni ìwo kíkún ti ojú ewé àwọn ètò domain mapping:

![Ojú ewé kíkún àwọn ètò domain mapping](/img/config/settings-domain-mapping-full.png)

Lẹ́yìn títẹ **Next Step** , àwọn oníbàárà rẹ lè fi orúkọ domain àkànṣe wọn kún un, kí wọ́n sì yan bóyá domain àkànṣe yìí ni yóò jẹ́ primary. Ṣàkíyèsí pé àwọn oníbàárà rẹ lè lo ju domain àkànṣe kan lọ fún àwọn website wọn, nítorí náà wọ́n lè yan èyí tí yóò jẹ́ primary.

![Ìwọlé orúkọ domain àkànṣe pẹ̀lú àṣàyàn domain primary](/img/account-page/add-domain-primary.png)

Lẹ́yìn títẹ **Add Domain** , domain náà yóò fi kún account oníbàárà rẹ. Ohun kan ṣoṣo tí wọ́n nílò láti ṣe báyìí ni yíyí àwọn àkọsílẹ̀ DNS ti domain àkànṣe yìí padà ní registrar domain wọn.

### Yíyí Ọ̀rọ̀ìwọlé Padà:

Nínú account dashboard, àwọn oníbàárà rẹ tún lè yí password wọn padà nípa títẹ **Change Password**.

![Bọ́tìnì Change Password lórí ojú ewé account](/img/account-page/change-password-button.png)

Èyí yóò fi window tuntun hàn níbi tí àwọn oníbàárà rẹ yóò ti ní láti fọwọ́ sí password lọwọlọwọ wọn, lẹ́yìn náà kí wọ́n fọwọ́ sí password tuntun tí wọ́n fẹ́ lo.

![Fọ́ọ̀mù yíyí password padà pẹ̀lú àwọn field password lọwọlọwọ àti tuntun](/img/account-page/change-password-form.png)

### Àgbègbè Èwu:

A tún ní àṣàyàn méjì tí a fi hàn ní apá **Danger Zone**: **Delete Site** àti **Delete Account**. Méjèèjì wà ní apá Danger Zone nítorí pé àwọn ìṣe méjì yìí kò ṣeé yí padà. Bí àwọn oníbàárà rẹ bá pa website wọn tàbí account wọn rẹ́, wọn kò lè gba wọ́n padà.

![Danger Zone pẹ̀lú àwọn àṣàyàn Delete Site àti Delete Account](/img/account-page/danger-zone.png)

Bí àwọn oníbàárà rẹ bá tẹ èyíkéyìí nínú àwọn àṣàyàn méjì yìí, a ó fi window hàn fún wọn níbi tí wọn yóò ti ní láti tan àṣàyàn láti yọ website tàbí account náà kúrò, a ó sì kìlọ̀ fún wọn pé ìṣe yìí kò ṣeé yí padà.

![Àpótí ìjẹ́rìí Delete Site](/img/account-page/delete-site-confirm.png)

![Àpótí ìjẹ́rìí Delete Account](/img/account-page/delete-account-confirm.png)

Bí wọ́n bá pa website wọn rẹ́, account àti membership wọn yóò ṣi wà láìfọwọ́kàn. Wọ́n yóò kan pàdánù gbogbo àkóónú lórí website wọn. Bí wọ́n bá pa account wọn rẹ́, gbogbo àwọn website, membership àti ìwífún tó ní í ṣe pẹ̀lú account yìí yóò sọnù.
