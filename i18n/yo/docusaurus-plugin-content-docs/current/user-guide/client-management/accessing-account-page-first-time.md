# Báwo ni mo ṣe lè wọlé sí ojú-ewé Account mi fún ìgbà àkọ́kọ́?

Káàbọ̀ sí Ultimate Multisite!

Wọlé sí ojú-ewé Account rẹ nípasẹ̀ ìjápọ̀ tí o gba nínú e-mail rẹ lẹ́yìn tí o rà plugin wa:

<!-- Screenshot unavailable: Account page email with access link -->

Lo email tí o pèsè nígbà rírà àti password tí o gba nínú email rẹ.

<!-- Screenshot unavailable: Login page with email and password fields -->

Níwọ̀n bí èyí ṣe jẹ́ ìgbà àkọ́kọ́ tí o ń wọlé sí Account yìí, a ó ní kí o yí password rẹ padà. Yí i padà kí o sì tẹ bọ́tìnì _Save_.

<!-- Screenshot unavailable: Change password form with Save button -->

Lórí ojú-ewé Account rẹ, o lè ṣàkóso gbogbo àlàyé Account rẹ bíi billings, renewals, cancelations, àti ọ̀pọ̀ nǹkan míì.
