# Fifagilé Ọmọ-ẹgbẹ kan (v2) {#canceling-a-membership-v2}

_**AKIYESI PATAKI: Àpilẹ̀kọ yìí tọ́ka sí Ultimate Multisite ẹ̀yà 2.x.**_

Fifagilé ọmọ-ẹgbẹ ní ẹgbẹ́ àwọn oníbàárà rẹ rọrùn gan-an.

Tí aṣàmúlò rẹ bá fẹ́ fagilé ọmọ-ẹgbẹ wọn, wọ́n gbọ́dọ̀ kọ́kọ́ wọlé sí ojú-ìwé ****Account**** wọn.

Ní bẹ̀, wọ́n gbọ́dọ̀ tẹ lọ sí **Agbègbè Ewu** , kí wọ́n sì tẹ **Pa Account rẹ́**.

![Ojú-ìwé Account Agbègbè Ewu pẹ̀lú bọ́tìnì Pa Account rẹ́](/img/account-page/danger-zone.png)

Módùùlù kan yóò ṣí tí yóò béèrè ìmúdájú wọn. Wọ́n gbọ́dọ̀ tan aṣàyàn **Jẹ́rìí ìpamọ́ Account rẹ́** sílẹ̀ kí wọ́n sì tẹ **Pa Account rẹ́**.

![Dáyálọ́ọ̀gì Jẹ́rìí ìpamọ́ Account rẹ́](/img/account-page/delete-account-confirm.png)

Èyí yóò mú wọn jáde kúrò nínú ojúlé náà, yóò sì fagilé ọmọ-ẹgbẹ wọn lórí nẹ́tíwọ́ọ̀kì rẹ, bẹ́ẹ̀ náà ni yóò pa ojúlé wọn rẹ́ kúrò nínú nẹ́tíwọ́ọ̀kì rẹ.

![A ti mú un jáde lẹ́yìn pípaarẹ Account](/img/account-page/logged-out-after-delete.png)

![Ìmúdájú pé a ti fagilé ọmọ-ẹgbẹ](/img/account-page/membership-canceled.png)

**Pàtàkì:** Ṣàkíyèsí pé ìṣe yìí kì yóò yọ oníbàárà náà kúrò nínú nẹ́tíwọ́ọ̀kì rẹ, nítorí oníbàárà lè ní ju ọmọ-ẹgbẹ kan lọ tàbí kò ní ọmọ-ẹgbẹ kankan tí a so mọ́ wọn.

![Oníbàárà ṣì wà nínú nẹ́tíwọ́ọ̀kì lẹ́yìn fífagilé](/img/admin/customer-after-cancel.png)

![Oníbàárà tí kò ní ọmọ-ẹgbẹ tó ń ṣiṣẹ́](/img/admin/memberships-customer-empty.png)
