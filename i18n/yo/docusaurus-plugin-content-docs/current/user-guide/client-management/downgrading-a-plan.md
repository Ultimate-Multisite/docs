# Bí a ṣe ń sọ plan kan di kekere (v2)

_**ÀKÍYÈSÍ PÀTÀKÌ: Àpilẹ̀kọ yìí ń tọ́ka sí Ultimate Multisite version 2.x.**_

Sísọ plan tàbí subscription di kekere jẹ́ ìgbésẹ̀ tí ó wọ́pọ̀ tí àwọn client rẹ lè ṣe bí wọ́n bá ní ìnáwó tó lopin tàbí bí wọ́n bá pinnu pé wọn kò ní nílò ọ̀pọ̀ resources láti ṣiṣẹ́ subsite wọn.

## Bí a ṣe ń sọ plan kan di kekere

Àwọn client rẹ lè sọ plan wọn di kekere nígbàkigbà nípa wíwọlé sí admin dashboard subsite wọn àti títẹ **Change** lábẹ́ ojú-ewé account wọn.

![Ojú-ewé account oníbàárà pẹ̀lú káàdì Your Membership àti bọ́tìnì Change](/img/account-page/membership-change-button.png)

Nígbà tí a bá tẹ bọ́tìnì **Change**, user/client náà yóò jẹ́ darípadà sí ojú-ewé checkout níbi tí wọ́n ti lè yan plan tí wọ́n fẹ́ yí subscription wọn sí.

![Ojú-ewé àwọn àṣàyàn plan dídi kekere ní apá oníbàárà](/img/account-page/downgrade-picker.png)

Nínú àpẹẹrẹ yìí, a ń sọ plan náà di kekere láti **Premium** sí **Free**.

Láti tẹ̀síwájú, user náà kan nílò láti tẹ bọ́tìnì **Complete Checkout**. Lẹ́yìn náà, yóò mú wọn padà sí ojú-ewé account tí yóò fi ìfiránṣẹ́ kan hàn nípa ìyípadà tó ń dúró fún membership náà. Àwọn ìyípadà náà yóò bẹ̀rẹ̀ sí ní ṣiṣẹ́ ní **next billing cycle** ti oníbàárà.

![Ojú-ewé Account tó ń fi asia ìyípadà membership tó ń dúró hàn](/img/account-page/pending-change.png)

### Ohun tó máa ṣẹlẹ̀ nígbà tí user kan bá sọ plan wọn di kekere

Ó ṣe pàtàkì láti ṣe àkíyèsí pé sísọ plan di kekere kì í yí configuration tó wà tẹ́lẹ̀ nínú subsite user náà padà.

Kò ní yí site template padà laifọwọ́yi nítorí pé yíyí site template padà yóò pa subsite náà rẹ́ pátápátá, yóò sì tún un ṣe. Èyí jẹ́ láti yàgò fún pípàdánù data tí kò ṣe dandan. Nítorí náà, disk space, themes, plugins àti bẹ́ẹ̀ bẹ́ẹ̀ lọ yóò wà bí wọ́n ṣe wà, àyàfi fún àwọn posts.

A lóye pé àníyàn pàtàkì rẹ yóò jẹ́ àwọn limits àti quotas tí o ṣètò lábẹ́ plan kọ̀ọ̀kan, ṣùgbọ́n a ní láti ronú nípa ìpalára tí yóò fa sí subsite user náà bí a bá pa tàbí yí èyíkéyìí nínú configurations rẹ padà.

Fún àwọn posts tó kọjá limit tí a ṣètò lórí plan náà, o ní àwọn àṣàyàn mẹ́ta ọ̀tọ̀ọ̀tọ̀: **Fi àwọn posts náà sílẹ̀ bí wọ́n ṣe wà** *,* **Gbé àwọn posts náà lọ sí trash** *,* tàbí **Gbé àwọn posts náà lọ sí draft** *.* O lè ṣètò èyí lábẹ́ settings Ultimate Multisite.

![Ojú-ewé Network Admin Settings Sites tó ń fi àwọn àṣàyàn ìhùwàsí post-limit-exceeded hàn](/img/account-page/settings-sites-post-limit.png)

### Ohun tó máa ṣẹlẹ̀ sí payment

Nínú version 2.0, kò tún nílò àwọn àtúnṣe kankan lórí payment ní ti proration mọ́.

Èyí jẹ́ nítorí pé system náà yóò dúró de membership tó wà tẹ́lẹ̀ láti **parí billing cycle rẹ kí** plan/membership tuntun tó bẹ̀rẹ̀ sí ní ṣiṣẹ́. Iye billing tuntun fún membership tuntun yóò jẹ́ fífi sílẹ̀ laifọwọ́yi, a ó sì gba owó náà ní billing cycle tó tẹ̀lé.
