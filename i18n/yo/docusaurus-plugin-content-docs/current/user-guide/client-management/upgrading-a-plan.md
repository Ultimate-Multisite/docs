# Ìgbésókè Eto kan (v2)

_**ÀKÍYÈSÍ PÀTÀKÌ: Àpilẹ̀kọ yìí tọ́ka sí Ultimate Multisite ẹ̀yà 2.x.**_

Àwọn oníbàárà rẹ lè ṣe ìgbésókè àwọn ètò wọn nígbàkigbà. Wọ́n lè ṣe ìgbésókè sí ètò mìíràn tàbí ra àwọn iṣẹ́ àfikún tàbí àwọn àpò iṣẹ́ èyíkéyìí tí o ń pèsè lórí nẹ́tíwọ́ọ̀kì rẹ.

Nínú ìtọ́sọ́nà yìí, a ó ṣàlàyé bí wọ́n ṣe lè ṣe ìgbésókè ètò wọn àti ohun tó máa ṣẹlẹ̀ lẹ́yìn ìlànà ìgbésókè náà.

Láti ṣe ìgbésókè ètò wọn, àwọn oníbàárà rẹ gbọ́dọ̀ wọlé sí Dashboard wọn kí wọ́n sì lọ sí ojú-ewé **Account**.

![Dashboard abẹ́-sáìtì oníbàárà pẹ̀lú ìjápọ̀ akojọ Account tó hàn](/img/account-page/account-menu.png)

Lórí ojú-ewé Account, a ó fi membership wọn lọwọlọwọ àti ètò tó ní í ṣe pẹ̀lú rẹ hàn wọ́n. Láti ṣe ìgbésókè sí ètò mìíràn, wọ́n gbọ́dọ̀ tẹ **Yí Padà** ní igun òkè-ọtún apá **Membership Rẹ**.

![Ojú-ewé Account káàdì Membership Rẹ pẹ̀lú bọtìnì Yí Padà](/img/account-page/membership-change-button.png)

A ó darí wọn lọ sí fọ́ọ̀mù ìsanwó níbi tí gbogbo àwọn ètò tó wà nílẹ̀ yóò ti hàn.

Wọ́n tún máa lè rí **àwọn iṣẹ́ àti àpò iṣẹ́ tó wà fún ètò wọn lọwọlọwọ** , bí wọ́n bá kan fẹ́ ra iṣẹ́ tàbí àpò iṣẹ́ kan pàtó (gẹ́gẹ́ bí ìbẹ̀wò aláìlópin tàbí ààyè disiki nínú àpẹẹrẹ wa níbí), tí wọn kò sì fẹ́ ṣe ìgbésókè ètò náà.

![Olùyan ìgbésókè tó ń fi àwọn ètò àti àpò iṣẹ́ tó wà nílẹ̀ hàn ní ẹ̀gbẹ́ oníbàárà](/img/account-page/upgrade-picker.png)

Lẹ́yìn tí wọ́n bá yan ọjà tí wọ́n fẹ́ rà, wọ́n yóò rí iye tí wọ́n máa nílò láti san ní báyìí - láìka kirẹditi tó wà tẹ́lẹ̀ sí - àti iye tí a ó gba lọ́wọ́ wọn ní ọjọ́ ìsanwó tó kàn.

Ní ọ̀pọ̀ ìgbà, bí ọjà náà bá jẹ́ ètò mìíràn, tí ìsanwó náà sì máa wáyé láàárín ìgba owó membership kan, wọ́n yóò gba kirẹditi fún iye owó tí wọ́n san lórí ètò àkọ́kọ́.

![Àkópọ̀ ìsanwó ìgbésókè tó ń fi kirẹditi tí a fi sí i àti iye ìsanwó tó kàn hàn](/img/account-page/upgrade-summary.png)

Bí wọ́n bá yan ètò tàbí àpò iṣẹ́ kan tí kò ní yí ohunkóhun padà lára ìforúkọsílẹ̀ lọwọlọwọ, wọ́n yóò rí ìfiránṣẹ́ kan tó ń ṣàlàyé ìyẹn.

![Àkíyèsí nígbà tí ètò tí a yan kò yí ìforúkọsílẹ̀ náà padà](/img/account-page/upgrade-no-change.png)

Lẹ́yìn tí ìsanwó bá parí, a ó fi ọjà tuntun náà/àwọn ọjà tuntun náà kún Account àwọn oníbàárà rẹ, gbogbo àwọn ààlà tàbí àwọn ànfààní ọjà tuntun náà/àwọn ọjà tuntun náà yóò sì wáyé lójú ẹsẹ̀ nínú rẹ: ìbẹ̀wò, ààyè disiki, àwọn àròkọ, abbl...

## Àwọn Ọ̀nà Ìgbésókè àti Ìsọdipalẹ̀ {#upgrade-and-downgrade-paths}

Lórí ọ̀kọ̀ọ̀kan àwọn ọjà rẹ, ìwọ yóò ní táàbù **Ìgbésókè & Ìsọdipalẹ̀**. Àṣàyàn àkọ́kọ́ lórí táàbù yẹn ni pápá kan tí a ń pè ní **Ẹgbẹ́ Eto**.

**Àwọn ẹgbẹ́ ètò** ni ohun tó jẹ́ kí o lè jẹ́ kí Ultimate Multisite mọ̀ pé àwọn ètò kan jẹ́ ti “ẹbí” kan náà, nítorí náà kí a sì lò wọ́n láti kọ àwọn àṣàyàn ọ̀nà ìgbésókè/ìsọdipalẹ̀.

![Táàbù Ìgbésókè àti Ìsọdipalẹ̀ ìṣàtúnṣe ọjà pẹ̀lú pápá Ẹgbẹ́ Eto](/img/config/product-upgrades-plan-group.png)

Fún àpẹẹrẹ, o ní **Ètò Ọfẹ́** , **Ètò Ipilẹ̀** àti **Ètò Ààyò** tó wà nílẹ̀. O fẹ́ kí àwọn aṣàmúlò tí wọ́n forúkọsílẹ̀ lábẹ́ **Ètò Ọfẹ́** lè ṣe ìgbésókè sí **Ètò Ààyò** nìkan, o kò sì fẹ́ kí wọ́n rí “Ètò Ipilẹ̀” gẹ́gẹ́ bí àṣàyàn ìgbésókè. Gbogbo ohun tí o nílò láti ṣe ni láti yan orúkọ ẹgbẹ́ ètò kan náà fún Ètò Ọfẹ́ àti Ètò Ààyò, gẹ́gẹ́ bí a ṣe fi hàn nínú àwọn àwòrán-iboju ní ìsàlẹ̀.

![Ojú-ewé ọjà Ètò Ọfẹ́ pẹ̀lú ẹgbẹ́ ètò Ìpele Gíga tí a yàn](/img/config/product-upgrades-free.png)

![Ojú-ewé ọjà Ètò Ààyò pẹ̀lú ẹgbẹ́ ètò Ìpele Gíga tí a yàn](/img/config/product-upgrades-premium.png)

Ohun tí èyí yẹ kí ó ṣe ni láti sọ fún Ultimate Multisite pé “ẹbí” àwọn ètò kan wà nínú nẹ́tíwọ́ọ̀kì tí a ń pè ní **Ìpele Gíga**. Nígbà tí a bá ń pèsè àwọn ìgbésókè tàbí ìsọdipalẹ̀, àwọn ètò láti inú ẹbí kan náà nìkan ni a ó fi hàn gẹ́gẹ́ bí àṣàyàn fún aṣàmúlò.
