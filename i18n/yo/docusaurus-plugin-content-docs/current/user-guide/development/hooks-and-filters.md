# Ultimate Multisite Hooks àti filters {#ultimate-multisite-hooks-and-filters}

Ultimate Multisite jẹ́ àkójọpọ̀ ọ̀pọ̀lọpọ̀ functions àti database queries. Ó ní hooks àti filters tí yóò jẹ́ kí o lè ṣàtúnṣe tàbí ṣàfikún code tuntun sí functions tó wà tẹ́lẹ̀.

A dá àtòjọ hooks àti filters wọ̀nyí sílẹ̀ gẹ́gẹ́ bí ìtọ́kasí láti ran àwọn aṣàmúlò tó ní ìmọ̀ jinlẹ̀ lọ́wọ́ láti tún network wọn ṣe bó ṣe wù wọ́n sí i. O lè wọlé sí àtòjọ náà láti Github nípa títẹ link tó wà nísàlẹ̀.
