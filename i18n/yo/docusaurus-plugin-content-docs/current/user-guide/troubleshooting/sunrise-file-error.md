# Àṣìṣe Nígbà Fífí Fáìlì Sunrise Sílẹ̀ {#error-installing-the-sunrise-file}

Fáìlì sunrise.php jẹ́ fáìlì pàtàkì kan tí WordPress máa ń wá nígbà tí ó ń bẹ̀rẹ̀ ara rẹ̀. Kí WordPress lè rí fáìlì sunrise.php, ó ní láti wà nínú **folda wp-content**.

Nígbà tí o bá mú Ultimate Multisite ṣiṣẹ́ tí o sì kọjá inú olùrànlọ́wọ́ ìṣètò bí èyí tí o ní lórí àwòrán iboju, Ultimate Multisite máa ń gbìyànjú láti daakọ fáìlì sunrise.php wa sínú folda wp-content.

<!-- Àwòrán iboju kò sí: Ojú-ewé olùrànlọ́wọ́ ìṣètò tó ń fi ìgbésẹ̀ fífí sunrise.php sílẹ̀ hàn -->

Ní ọ̀pọ̀ ìgbà, a máa ń lè daakọ fáìlì náà ní àṣeyọrí, ohun gbogbo a sì ṣiṣẹ́. Ṣùgbọ́n, bí nkan kan kò bá ṣe é ṣètò dáadáa (àwọn ìyọ̀nda folda, fún àpẹẹrẹ), o lè bá ipò kan pàdé níbi tí Ultimate Multisite kò ti lè daakọ fáìlì náà.

Bí o bá ka ìfiránṣẹ́ àṣìṣe tí Ultimo fún ọ, wàá rí i pé ohun tí ó ṣẹlẹ̀ níbí gan-an nìyẹn: **Dídàakọ Sunrise kùnà**.

<!-- Àwòrán iboju kò sí: Ìfiránṣẹ́ àṣìṣe tó ń fi hàn pé dídàakọ Sunrise kùnà -->

Láti tún un ṣe, o lè kan daakọ fáìlì sunrise.php láti inú folda afikun wp-ultimo kí o sì lẹ̀ ẹ́ mọ́ inú folda wp-content rẹ. Lẹ́yìn tí o bá ṣe bẹ́ẹ̀, tún ojú-ewé olùrànlọ́wọ́ náà gbé kalẹ̀, àwọn àyẹ̀wò náà sì yẹ kí ó kọjá.

<!-- Àwòrán iboju kò sí: Olùṣàkóso fáìlì tó ń fi sunrise.php hàn nínú folda afikun wp-ultimo --> Ní gbogbo ọ̀nà, èyí lè jẹ́ kí ó yẹ kí o ṣe àyẹ̀wò gbogbogbòò àwọn ìyọ̀nda folda rẹ láti yàgò fún níní ìṣòro lọ́jọ́ iwájú (kì í ṣe pẹ̀lú Ultimate Multisite nìkan, ṣùgbọ́n pẹ̀lú àwọn afikun àti àwọn theme míì pẹ̀lú).

**Irinṣẹ́ Ayẹwo Ilera** tí ó jẹ́ apá kan ti WordPress (o lè wọlé sí i nípasẹ̀ **pánẹ́ẹ̀lì alákóso > Àwọn Irinṣẹ́ > Ayẹwo Ilera** ti aaye àkọ́kọ́ rẹ) lè jẹ́ kí o mọ̀ bí o bá ti ṣètò àwọn ìyọ̀nda folda sí iye tí ó lè fa ìṣòro pẹ̀lú WordPress.

<!-- Àwòrán iboju kò sí: Irinṣẹ́ Ayẹwo Ilera WordPress tó ń fi ipò àwọn ìyọ̀nda folda hàn -->
