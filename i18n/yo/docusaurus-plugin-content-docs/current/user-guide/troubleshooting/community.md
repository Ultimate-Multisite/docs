# Bii a ṣe le darapọ mọ Ultimate Multisite Community

Ultimate Multisite Community jẹ aaye kan nibiti o ti le kan si awọn olumulo Ultimate Multisite miiran ki o pin awọn imọran, iriri ati diẹ sii. Lati wọle si Ultimate Multisite Community lọ si https://community.ultimatemultisite.com/home.

Lori oju-iwe ìkáàbọ̀ lọ si ọna asopọ “**creating your account here** ”.

<!-- Screenshot unavailable: Community welcome page with creating your account link -->

A ó darí rẹ sí oju-iwe ìforúkọsílẹ̀ wa. Bayi o nilo lati tẹ orúkọ olumulo, imeeli rẹ ati ọ̀rọ̀ aṣínà kan sii. Ṣe Captcha ki o tẹ bọtini **Register**.

<!-- Screenshot unavailable: Community register page with username, email, password fields and Captcha -->

Lẹhin ìforúkọsílẹ̀, o to akoko lati ṣẹda profaili rẹ. Lori oju-iwe yii o le gbe fọto rẹ soke, tẹ orúkọ kikun rẹ sii, kọ bio kekere kan ati bẹ́ẹ̀ bẹ́ẹ̀ lọ. Pẹlupẹlu, ṣafikun awọn ọna asopọ media awujọ rẹ ki o ṣeto awọn igbanilaaye fun Account rẹ.

<!-- Screenshot unavailable: Community profile creation page with photo upload and bio fields -->

Bayi o wa lori ikanni “Introductions”, nibiti o ti le ṣafihan ara rẹ ki o mọ diẹ nipa awọn ọmọ ẹgbẹ miiran. O le ṣẹda ifiweranṣẹ rẹ nipa titẹ bọtini **New Post**.

<!-- Screenshot unavailable: Community Introductions channel with New Post button -->

Awọn ikanni lori apakan Lounge ati ikanni “Dev Logs” ṣí sí gbogbo eniyan. Sibẹsibẹ iwọ yoo nilo igbanilaaye pataki lati wọle si awọn ikanni miiran ninu agbegbe naa.

<!-- Screenshot unavailable: Community channels showing Lounge section and Dev Logs channel -->

Wiwọle yii yoo jẹ fifun ni kete ti o ba pese bọtini iwe-aṣẹ Ultimate Multisite ti n ṣiṣẹ fun wa. Lati ṣe e, tẹ ọna asopọ ikanni aládàáni eyikeyi, lẹ́yìn náà tẹ bọtini **Unlock**.

<!-- Screenshot unavailable: Community private channel with Unlock button -->

A ó darí rẹ sí oju-iwe ṣiṣi wiwọle wa. Tẹ bọtini iwe-aṣẹ Ultimate Multisite rẹ sii ki o tẹ bọtini Unlock Access:

<!-- Screenshot unavailable: Community unlock access page with license key input -->

Iwe-aṣẹ rẹ yoo jẹrisi, iwọ yoo sì le wọle si awọn ikanni aládàáni. Ilana yii le gba to wakati 24, nítorí náà tẹ ọna asopọ naa lati pada si agbegbe naa ki o gbadun awọn aaye miiran nigba ti a n ṣiṣẹ lori ìbéèrè rẹ.

<!-- Screenshot unavailable: Community license validation confirmation page -->
