# Bí o ṣe lè gba ìrànlọ́wọ́

O ti gba License key rẹ tó wúlò, àti pé ohun tí ó so mọ́ ọn ni ìráyè sí ẹgbẹ́ wa láti ràn ọ́ lọ́wọ́ láti yanjú àwọn ìṣòro tí o lè bá pàdé nígbà tí o ń ṣètò Ultimate Multisite.

Ẹ̀kọ́ yìí ṣàlàyé àwọn ọ̀nà ìtìlẹ́yìn tó wà fún àwọn tó ní license àti irú ìwífún wo ni o máa nílò láti ní lọ́wọ́ kí o tó kan sí ẹgbẹ́ ìtìlẹ́yìn wa.

## Ojúlé Ìrànlọ́wọ́

O lè wọlé sí oríṣiríṣi àwọn àpilẹ̀kọ àti àwọn ìtọ́sọ́nà láti ojúlé ìrànlọ́wọ́ wa, èyí tí o lè rí nínú ìjápọ̀ tó wà nísàlẹ̀.

[)

A gba àwọn aṣàmúlò wa níyànjú láti mọ àwọn ẹ̀ya àti iṣẹ́ ìpilẹ̀ Ultimate Multisite dáadáa. Àwọn àpilẹ̀kọ "Getting Started" wà lórí ojúlé ìrànlọ́wọ́ wa, àwọn ìdáhùn sí àwọn ìbéèrè tí ó wọ́pọ̀ sì tún lè rí níbẹ̀ pẹ̀lú, bíi bí a ṣe ń fi add-ons sílẹ̀, bí a ṣe ń so payment gateways pọ̀, àti bẹ́ẹ̀ bẹ́ẹ̀ lọ.

## Kan Sí Ìtìlẹ́yìn

Tí o bá di mọ́ ibi kan, tí o kò sì lè rí ìdáhùn láti ojúlé ìrànlọ́wọ́ wa, o lè kan sí ẹgbẹ́ ìtìlẹ́yìn wa bóyá nípa títẹ aami speech bubble tó wà ní apá ọ̀tún ìsàlẹ̀ ojúlé wẹẹbu wa TÀBÍ nípa fífi imeeli ránṣẹ́ tààrà sí support@ultimatemultisite.com.

Nígbà tí o bá ń kan sí ìtìlẹ́yìn fún àwọn ìṣòro imọ̀-ẹ̀rọ, jọ̀wọ́ rí i dájú pé o pèsè àpèjúwe kíkún nípa ìṣòro rẹ pẹ̀lú àwòrán sikirinisọọti, àwọn ìfiránṣẹ́ àṣìṣe, tàbí ìwífún èyíkéyìí tí yóò ràn àwọn ènìyàn ìtìlẹ́yìn wa lọ́wọ́ láti lóye àti ṣe àyẹ̀wò ìṣòro rẹ ní rọrùn. O tún nílò láti pèsè license key rẹ. Àwọn tó ní license tó ń ṣiṣẹ́ nìkan ni ó lẹ́tọ̀ọ́ sí ìtìlẹ́yìn.

## Ojúlé Àwùjọ

[Ojúlé àwùjọ](https://community.ultimatemultisite.com/) wa jẹ́ ibi fún àwọn aṣàmúlò Ultimate Multisite láti bá ara wọn ṣe ìbáṣepọ̀. Ó ń ṣiṣẹ́ gẹ́gẹ́ bí ètò ìwífún níbi tí àwọn ọmọ ẹgbẹ́ ti lè fi ìfìwéránṣẹ́ sílẹ̀, sọ̀rọ̀ lórí àwọn ìjíròrò, fúnni ní ìmọ̀ràn, ṣiṣẹ́ pọ̀, tàbí gba ìrànlọ́wọ́ ọ̀pọ̀ ènìyàn. A óò béèrè pé kí o pèsè Ultimate Multisite license key rẹ láti ṣí àwọn ààyè/ojú-ìwé aládàáni.
