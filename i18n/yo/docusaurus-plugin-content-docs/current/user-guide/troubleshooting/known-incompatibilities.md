# Àwọn àìbáramu tí a mọ̀ pẹlu àwọn Àfikún míì

Lẹ́yìn fifi Ultimate Multisite sílẹ̀ àti mímú un ṣiṣẹ́, o lè pàdé àwọn ìṣòro nítorí àwọn àìbáramu tí a kò tíì yanjú pẹlu àwọn àfikún míì.

A ń ṣiṣẹ́ takuntakun láti dín àtòkọ yìí kù pẹlu gbogbo ìtújáde tuntun kọọkan, ṣùgbọ́n tí àfikún kan bá wà níbí, o lè nílò láti dá a dúró kí Ultimate Multisite Network lè ṣiṣẹ́ ní kíkún.

Èyí ni àtòkọ wọn àti ohun tí ó yẹ kí o ṣe ní àwọn ọ̀ràn wọ̀nyí:

**Àwọn àfikún WPMU DEV kan** Díẹ̀ lára àwọn àfikún tí WPMU DEV _kò tún_ ń ṣètọju mọ́ ni a mọ̀ pé wọ́n lè mú kí ìṣàn ìforúkọsílẹ̀ Ultimate Multisite dúró ní ìgbésẹ̀ ikẹyìn (olùmúlò kò lè dá Account sílẹ̀). Èyí ni àtòkọ àwọn àfikún tí ń fa àwọn ìṣòro wọ̀nyẹn:

  * Signup Language: _Ultimate Multisite ní àfikún kan tí ó ń pèsè iṣẹ́ kan náà, tí a ń pè ní Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Èyí wà nínú Ultimate Multisite, nítorí náà kò sí ìdí fún àfikún míì._

  * New Blog Templates: _Èyí wà nínú Ultimate Multisite, nítorí náà kò sí ìdí fún àfikún míì._

  * Set Password on Multisite Blog Creation: _Èyí wà nínú Ultimate Multisite, nítorí náà kò sí ìdí fún àfikún míì._

  * Signup TOS: _Èyí wà nínú Ultimate Multisite, nítorí náà kò sí ìdí fún àfikún míì._

  * Anti-Splog: Bí ó tilẹ̀ jẹ́ pé agbára rẹ̀ kò tó bẹ́ẹ̀, èyí náà _wà nínú Ultimate Multisite._

**Àkíyèsí** : Gbogbo àwọn àfikún WPMU Dev yòókù lè ṣee lò bí ó ti yẹ lẹ́gbẹ̀ẹ́ Ultimate Multisite. Àwọn àpẹẹrẹ ni _Smush_ , _Forminator_ , _Defender,_ àti bẹ́ẹ̀ bẹ́ẹ̀ lọ.
