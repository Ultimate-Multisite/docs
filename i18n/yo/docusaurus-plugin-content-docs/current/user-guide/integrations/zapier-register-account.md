# Ìṣẹ̀lẹ̀: Forúkọsílẹ̀ Account nípasẹ̀ Zapier

Nínú àpilẹ̀kọ [Ṣíṣe ìṣàkóso Ultimate Multisite pẹ̀lú Zapier](zapier.md), a jíròrò bí a ṣe lè lo Zapier láti ṣe àwọn ìṣe ọ̀tọ̀ọ̀tọ̀ nínú Ultimate Multisite dá lórí àwọn ohun tí ń ru ìṣe àti àwọn ìṣẹ̀lẹ̀. Nínú àpilẹ̀kọ yìí, a ó fi hàn bí o ṣe lè ṣàkóso àwọn ohun elo ẹgbẹ́ kẹta. A ó lo Google Sheets gẹ́gẹ́ bí orísun dátà, a ó sì fi ìwífún náà ránṣẹ́ sí Ultimate Multisite láti forúkọsílẹ̀ Account kan.

Ṣáájú kí o tó so Zapier pọ̀, lọ sí **Ultimate Multisite > Settings > API & Webhooks** kí o sì jẹ́rìí pé API ti ṣiṣẹ́. Daakọ API Key àti API Secret láti ojú iboju yìí nígbà tí Zapier bá béèrè fún àwọn ẹ̀rí Account Ultimate Multisite.

![Àwọn eto API àti Webhooks pẹ̀lú API Key, API Secret, àti àwọn aṣayan Enable API](/img/admin/settings-api-webhooks.png)

Kọ́kọ́, o ní láti ṣẹ̀dá **Google Sheet** kan lábẹ́ Google Drive rẹ. Rí i dájú pé o ṣàlàyé kólọ́ọ̀mù kọọkan dáadáa kí o lè ya dátà náà sípò rọrùn nígbà tó bá yá.

Lẹ́yìn tí o bá ti ṣẹ̀dá Google sheet, o lè wọlé sí Account Zapier rẹ kí o sì bẹ̀rẹ̀ sí í ṣẹ̀dá zap kan.

Lábẹ́ pápá ìwádìí fún **"Ìṣẹ̀lẹ̀ app"** yan **"Google Sheets"**


Lẹ́yìn náà fún pápá "**Ìṣẹ̀lẹ̀** " yan "**Ìlà spreadsheet tuntun** " kí o sì tẹ "**Tẹ̀síwájú** "

Ìgbésẹ̀ tó kàn yóò béèrè kí o yan **Google Account** kan níbi tí a ti fi **Google Sheet** pamọ́ sí. Nítorí náà, rí i dájú pé Account Google tó tọ́ ni a sọ pàtó.


Lábẹ́ **"Ṣètò ohun tí ń ru ìṣe** ", o ní láti yan kí o sì sọ pàtó spreadsheet àti worksheet Google tí ìwọ yóò lò, níbi tí dátà yóò ti máa wá. Kan tẹ̀síwájú kí o kún wọn, kí o sì tẹ "**Tẹ̀síwájú** "

Ohun tó kàn ni láti "**dán ohun tí ń ru ìṣe rẹ wò** " láti rí i dájú pé Google sheet rẹ ti sopọ̀ dáadáa.

Tí ìdánwò rẹ bá ṣàṣeyọrí, o yẹ kí o rí èsì tó ń fi díẹ̀ lára àwọn iye hàn láti inú àwọn spreadsheets rẹ. Tẹ "**Tẹ̀síwájú** " láti tẹ̀síwájú.

Ìgbésẹ̀ tó kàn ni láti ṣètò ìṣe kejì tí yóò ṣẹ̀dá tàbí forúkọsílẹ̀ Account kan nínú Ultimate Multisite. Lórí pápá ìwádìí yan "**Ultimate Multisite(2.0.2)** "


Lábẹ́ pápá "**Ìṣẹ̀lẹ̀** ", yan "**Forúkọsílẹ̀ Account kan nínú Ultimate Multisite** " lẹ́yìn náà tẹ bọ́tìnì "**Tẹ̀síwájú** ".

Lábẹ́ "**Ṣètò ìṣe kan** ", ìwọ yóò rí àwọn pápá ọ̀tọ̀ọ̀tọ̀ tó wà fún dátà oníbàárà, àwọn memberships, àwọn ọjà, àti bẹ́ẹ̀ bẹ́ẹ̀ lọ. O lè ya àwọn iye lábẹ́ Google sheet rẹ sípò kí o sì fi wọ́n sí pápá tó yẹ níbi tí wọ́n gbọ́dọ̀ kun sí gẹ́gẹ́ bí a ṣe fi hàn nínú àwòrán iboju ní ìsàlẹ̀.


Lẹ́yìn tí o bá ti ya àwọn iye sípò, o lè dán ìṣe náà wò.
