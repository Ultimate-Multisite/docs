# Site Templates (v2)

_**AKIYESI: Àpilẹ̀kọ yìí tọ́ka sí Ultimate Multisite ẹ̀yà 2.x. Tí o bá ń lo ẹ̀yà 1.x,**_ **wo àpilẹ̀kọ yìí** _**.**_

Ìlépa wa nígbà tí a bá ń ṣẹ̀dá nẹ́tíwọ́ọ̀kì premium pẹ̀lú Ultimate Multisite ni láti ṣe àdáṣiṣẹ́ fún ọ̀pọ̀lọpọ̀ ìlànà bí ó ti ṣeé ṣe tó, nígbà tí a tún ń fún àwọn oníbàárà wa ní ìrọ̀rùn àti àwọn àṣàyàn onírúurú láti yan lára nígbà tí wọ́n bá ń ṣẹ̀dá àwọn wẹ́ẹ̀bù wọn. Ọ̀nà rọrùn kan láti ṣàṣeyọrí ìwòntúnwònsì yìí ni lílo ẹ̀yà Ultimate Multisite Site Templates.

## Kí ni Site Template? {#what-is-a-site-template}

Gẹ́gẹ́ bí orúkọ ṣe fi hàn, Site Template jẹ́ wẹ́ẹ̀bù boilerplate tí a lè lò gẹ́gẹ́ bí ìpìlẹ̀ nígbà tí a bá ń ṣẹ̀dá àwọn wẹ́ẹ̀bù tuntun nínú nẹ́tíwọ́ọ̀kì rẹ.

Èyí túmọ̀ sí pé o lè ṣẹ̀dá wẹ́ẹ̀bù ìpìlẹ̀ kan, mú àwọn plugin onírúurú ṣiṣẹ́, ṣètò theme tí ń ṣiṣẹ́, kí o sì ṣe àṣàṣe rẹ ní ọ̀nà èyíkéyìí tí o fẹ́. Lẹ́yìn náà, nígbà tí oníbàárà rẹ bá ṣẹ̀dá Account tuntun, dípò kí wọ́n gba wẹ́ẹ̀bù WordPress àiyédẹ̀rùn tí kò ní àkóónú tó ní ìtumọ̀ nínú rẹ, wọ́n yóò gba ẹ̀dà wẹ́ẹ̀bù ìpìlẹ̀ rẹ pẹ̀lú gbogbo àwọn àṣàṣe àti àkóónú tí ó ti wà ní ipò.

Ìyẹn dùn gan-an, ṣùgbọ́n báwo ni mo ṣe lè ṣẹ̀dá site template tuntun? Ó rọrùn bí ó ṣe lè rí.

## Ṣíṣẹ̀dá àti Ṣíṣàtúnṣe Site Template tuntun {#creating-and-editing-a-new-site-template}

Site Templates jẹ́ àwọn wẹ́ẹ̀bù àtẹ̀yìnwá lórí nẹ́tíwọ́ọ̀kì rẹ. Láti ṣẹ̀dá template tuntun, o lè lọ sí **Network Admin > Ultimate Multisite > Sites > Add Site.**

**![Bọ́tìnì fi wẹ́ẹ̀bù kún un ní ojúewé àtòkọ Sites](/img/config/site-templates-list.png)**

Èyí yóò ṣí fèrèsé modal kan níbi tí yóò ti béèrè fún **Àkọlé wẹ́ẹ̀bù, Domain/ọ̀nà wẹ́ẹ̀bù,** àti **Iru wẹ́ẹ̀bù**. Lábẹ́ pápá ìṣàlẹ̀ **Site Type**, rí i dájú pé o yan **Site Template** *.*

_![Modal fífi site template kún un pẹ̀lú akojọ aṣayan isalẹ iru wẹ́ẹ̀bù](/img/config/site-templates-list.png)_

O tún lè fi àpèjúwe tí oníbàárà yóò rí kún un láti ṣàlàyé ohun tí template náà ní:

![Pápá àpèjúwe site template lórí iboju àtúnṣe site template](/img/config/site-template-edit-description.png)

Ní ìsàlẹ̀ fọ́ọ̀mù náà, ìwọ yóò rí yíyí **Copy Site**. Èyí yóò jẹ́ kí o ṣẹ̀dá site template tuntun tí ó dá lórí site template tó wà tẹ́lẹ̀ gẹ́gẹ́ bí ibi ìbẹ̀rẹ̀ rẹ láti ràn ọ́ lọ́wọ́ láti fi àkókò pamọ́ dípò kí o ṣẹ̀dá site template láti ìbẹ̀rẹ̀ pátápátá.

![Modal fífi site template kún un pẹ̀lú yíyí copy site](/img/config/site-templates-list.png)

### Ṣíṣe àṣàṣe àwọn àkóónú Site Template {#customizing-the-contents-of-a-site-template}

Láti ṣe àṣàṣe site template rẹ, kan lọ sí pánẹ́ẹ̀lì Dashboard rẹ kí o ṣe àwọn ayípadà tí o nílò. O lè ṣẹ̀dá àwọn ìfìwéránṣẹ́ tuntun, àwọn ojúewé, mú àwọn plugin ṣiṣẹ́, kí o sì yí theme tí ń ṣiṣẹ́ padà. O tilẹ̀ lè lọ sí Customizer kí o sì yí gbogbo irú àwọn àṣàyàn àṣàṣe padà.

![Ìbánisọ̀rọ̀ àtúnṣe site template](/img/config/site-template-edit.png)

Gbogbo dátà yẹn ni a ó dà kọjá nígbà tí oníbàárà bá ṣẹ̀dá wẹ́ẹ̀bù tuntun tí ó dá lórí Site Template yẹn.

### Àwọn Àṣàyàn Tí ó Gíga {#advanced-options}

Tí o bá mọ ọ̀nà rẹ nínú díẹ̀ ninu coding àṣà, o lè lo API Wa àti Rọ́pò wa láti rọ́pò ìwífún laifọwọyi lórí wẹ́ẹ̀bù tuntun lẹ́yìn tí a bá ti ṣẹ̀dá rẹ. Èyí wúlò fún àwọn nǹkan bíi rírọ́pò àwọn orúkọ ilé-iṣẹ́ lórí ojúewé About, rírọ́pò ímeèlì ìbánisọ̀rọ̀ lórí ojúewé Contact, àti bẹ́ẹ̀ bẹ́ẹ̀ lọ.

### Lílo Site Templates {#using-site-templates}

Ó dáa, nítorí náà o ti ṣẹ̀dá ọ̀pọ̀ Site Templates onírúurú pẹ̀lú àwọn dísáìnì, themes àti àwọn ètò onírúurú. Báwo ni o ṣe mú kí wọ́n wúlò lórí nẹ́tíwọ́ọ̀kì rẹ báyìí?

Ní pàtàkì, ọ̀nà méjì ló wà tí o lè lò báyìí (kì í ṣe ní àkókò kan náà):

  * Lílo Site Template kan mọ́ ọ̀kọ̀ọ̀kan àwọn Plans rẹ

**TÀBÍ**

  * Jíjẹ́ kí àwọn oníbàárà rẹ yan àwọn site templates fúnra wọn nígbà ìforúkọsílẹ̀.

#### Ipo 1: Yan Site Template fún un {#mode-1-assign-site-template}

Ní ipo yìí, àwọn oníbàárà rẹ kò ní lè yan template nígbà tí wọ́n bá ṣẹ̀dá Account, ṣùgbọ́n dípò bẹ́ẹ̀ ìwọ yóò ṣàlàyé template wo ni ó yẹ kí a lò lórí ọ̀kọ̀ọ̀kan àwọn Plans rẹ.

Láti ṣe bẹ́ẹ̀, ìwọ yóò ní láti lọ sí **Ultimate Multisite > Products > Edit**.

![Ṣàtúnṣe ọjà láti yan site template fún un](/img/config/product-site-templates.png)

Èyí yóò mú ọ lọ sí ojúewé **Edit Product**. Lábẹ́ abala **Product Options**, wá taabu **Site template** kí o sì yan àṣàyàn **Assign Site Template** láti pápá ìṣàlẹ̀. Èyí yóò mú àtòkọ àwọn site templates tó wà jáde, yóò sì jẹ́ kí o yan site template kan ṣoṣo tí a yà sọ́tọ̀ fún ọjà náà.

![Taabu site template ọjà pẹ̀lú àṣàyàn assign site template](/img/config/product-site-templates.png)

#### Ipo 2: Yan Site Template Tó Wà {#mode-2-choose-available-site-template}

Ní ipo yìí, ìwọ yóò fún àwọn oníbàárà rẹ ní àṣàyàn nígbà ìlànà ìforúkọsílẹ̀. Wọ́n yóò lè yan lára àwọn site templates onírúurú tí o ṣàlàyé lábẹ́ àwọn ètò ọjà. O ní àṣàyàn láti fi ààlà sí site template tí wọ́n lè yan lábẹ́ ọjà tí a yàn. Èyí yóò jẹ́ kí o ní àwọn akojọpọ̀ site templates onírúurú lábẹ́ ọ̀kọ̀ọ̀kan ọjà, èyí tí ó péye láti tẹnumọ́ àwọn iṣẹ́ àti àwọn ẹ̀yà ara onírúurú fún ọjà tí iye rẹ ga jù.

Lórí ojúewé **Edit Product**. Lábẹ́ abala **Product Options**, wá taabu **Site template** kí o sì yan àṣàyàn **Choose Available Site Template** láti pápá ìṣàlẹ̀. Èyí yóò mú àtòkọ àwọn site templates tó wà jáde, yóò sì jẹ́ kí o yan site template tí o fẹ́ kí ó wà. O lè ṣe èyí nípa yíyan Behavior rẹ: **Available** tí o bá fẹ́ kí site template náà wà nínú àtòkọ. _**Not Available**_ tí o bá fẹ́ kí site template náà má ṣe hàn gẹ́gẹ́ bí àṣàyàn. Àti **Pre-selected** tí o bá fẹ́ kí ọ̀kan lára àwọn site templates tí a ṣe àtòkọ rẹ jẹ́ eyi tí a yàn ní aiyédẹ̀rùn.

![Yan site templates tó wà pẹ̀lú àwọn àṣàyàn ìhùwàsí](/img/config/product-site-templates.png)

### Ipo Aiyédẹ̀rùn: Yíyan site template lórí fọ́ọ̀mù Checkout {#default-mode-site-template-selection-on-the-checkout-form}

Tí o bá fẹ́ kí gbogbo site templates rẹ wà nígbà ìforúkọsílẹ̀, tàbí bóyá o kò fẹ́ ṣe iṣẹ́ àfikún ti yíyan tàbí ṣíṣe pàtó site templates lábẹ́ ọ̀kọ̀ọ̀kan ọjà tí o ṣẹ̀dá. Nígbà náà o lè kan ṣètò yíyan site template lábẹ́ **Checkout Form** rẹ. Láti ṣe èyí, lọ sí **Ultimate Multisite > Checkout Forms** kí o tẹ **Edit** lábẹ́ fọ́ọ̀mù tí o fẹ́ ṣètò.

Èyí yóò mú ojúewé **Edit Checkout Form** jáde. Wá pápá **Template Selection** kí o sì tẹ **Edit** lábẹ́ rẹ.

Ferese modal kan yoo han. Labẹ aaye **Template Sites** o le yan ki o si ṣe atokọ gbogbo àwọn template aaye tí o fẹ́ kí wọn wà nígbà ìforúkọsílẹ̀. Àwọn template aaye tí o ṣàlàyé láti ibi yìí yoo wà láìka ọja èyíkéyìí tí olumulo yan sí.

![Aaye yíyan template ninu olootu fọọmu checkout](/img/config/checkout-form-template-step.png)

Lori frontend, àwọn oníbàárà máa rí olùyan template nígbà checkout, wọn sì le yan àwòrán ìbẹ̀rẹ̀ fún aaye tuntun wọn.

![Olùyan template frontend nígbà ìforúkọsílẹ̀](/img/config/frontend-template-chooser.png)

### Àwọn Àṣàyàn Template Aaye {#site-template-options}

Àwọn iṣẹ́ template aaye mìíràn wà tí o le tan-an tàbí pa lábẹ́ àwọn ètò Ultimate Multisite.

![Àwọn aṣàyàn template aaye ninu àwọn ètò Ultimate Multisite](/img/config/settings-sites-templates-section.png)

#### Gba Yíyí Template Padà {#allow-template-switching}

Mímú aṣàyàn yìí ṣiṣẹ́ yoo jẹ́ kí àwọn client rẹ yí template tí wọn yan nígbà ilana ìforúkọsílẹ̀ padà lẹ́yìn tí Account àti aaye ti dá. Èyí wúlò láti ojú ìwòye client nítorí ó jẹ́ kí wọn tún yan template bí wọn bá rí i nigbamii pé yíyan àkọ́kọ́ wọn kò dara jùlọ fún àwọn àìní pàtó wọn.

#### Gba Àwọn Olumulo Láyè láti lo Aaye wọn gẹ́gẹ́ bí templates {#allow-users-to-use-their-site-as-templates}

Nítorí àwọn olumulo subsite lo akoko láti kọ́ àti ṣe àpẹẹrẹ aaye tiwọn, wọ́n lè fẹ́ ṣe ẹ̀dà rẹ ki wọn sì lo ó gẹ́gẹ́ bí ọ̀kan nínú àwọn template aaye tó wà nígbà tí wọ́n ń dá subsite míì sílẹ̀ lori nẹ́tíwọ́ọ̀kì rẹ. Aṣàyàn yìí yoo jẹ́ kí wọ́n lè ṣe bẹ́ẹ̀.

#### Da Media Kọ nígbà Ìṣàdáwòkọ Template {#copy-media-on-template-duplication}

Ṣíṣàyẹ̀wò aṣàyàn yìí yoo da media tí a gbé sí ori aaye template kọ sí aaye tuntun tí a ṣẹ̀ṣẹ̀ dá. Èyí le jẹ́ kí a bori rẹ lori ọkọọkan àwọn plan.

#### **Dènà Àwọn Ẹ̀rọ Ìṣàwárí láti indexing Àwọn Template Aaye** {#prevent-search-engines-from-indexing-site-templates}

Àwọn template aaye gẹ́gẹ́ bí a ti jiroro ninu àpilẹ̀kọ yìí jẹ́ boilerplate ṣùgbọ́n wọ́n ṣì jẹ́ apá kan ti nẹ́tíwọ́ọ̀kì rẹ, èyí túmọ̀ sí pé ó ṣì wà fún àwọn ẹ̀rọ ìṣàwárí láti rí. Aṣàyàn yìí yoo jẹ́ kí o fi àwọn template aaye pamọ́ kí àwọn ẹ̀rọ ìṣàwárí lè index wọn.

## Ṣíṣe ìkún-tẹ́lẹ̀ Àwọn Template Aaye pẹ̀lú auto search-and-replace {#pre-populating-site-templates-with-auto-search-and-replace}

Ọ̀kan lára àwọn ẹya tó lágbára jùlọ ti Ultimate Multisite ni agbára láti ṣàfikún ọ̀rọ̀, awọ, àti àwọn aaye yíyan tó jẹ́ tùfẹ́ sí fọọmu ìforúkọsílẹ̀. Nígbà tí a bá ti gba data yẹn, a le lo o láti fi kún àkóónú tẹ́lẹ̀ ninu àwọn apá kan ti template aaye tí a yan. Lẹ́yìn náà, nígbà tí a bá ń tẹ aaye tuntun jáde, Ultimate Multisite yoo rọ́pò àwọn placeholder pẹ̀lú ìwífún gidi tí a tẹ̀ wọlé nígbà ìforúkọsílẹ̀.

Fún àpẹẹrẹ, bí o bá fẹ́ gba orúkọ ilé-iṣẹ́ ti end-user rẹ nígbà ìforúkọsílẹ̀ ki o sì fi orúkọ ilé-iṣẹ́ sí ojú-ìwé ile laifọwọyi. Lori ojú-ìwé ile aaye template rẹ, o ní láti ṣàfikún àwọn placeholder, bí ninu àwòrán ni isalẹ (a gbọ́dọ̀ ṣàfikún àwọn placeholder tí a yí ká pẹ̀lú àmì ìdábọ̀ oníkurili méjì - {{placeholder_name}}).

![Ojú-ìwé ile pẹ̀lú ọ̀rọ̀ placeholder ninu àmì ìdábọ̀ kurili](/img/config/site-templates-list.png)

Lẹ́yìn náà, o le ṣàfikún aaye ìforúkọsílẹ̀ tó bá a mu lori fọọmu checkout rẹ láti gba data yẹn. Olootu fọọmu checkout kan náà tí a lo fún yíyan template jẹ́ kí o gbe àwọn aaye àdáni lẹ́gbẹ̀ẹ́ olùyan template:

![Yíyan template àti àwọn aaye ìforúkọsílẹ̀ ninu olootu fọọmu checkout](/img/config/checkout-form-template-step.png)

Oníbàárà rẹ yoo lè fọwọ́si aaye yẹn nígbà ìforúkọsílẹ̀.

![Fọọmu checkout frontend pẹ̀lú olùyan template](/img/config/frontend-template-chooser.png)

Ultimate Multisite yoo wá rọ́pò àwọn placeholder pẹ̀lú data tí oníbàárà pèsè laifọwọyi.

![Àwọn placeholder tí a rọ́pò pẹ̀lú data oníbàárà lori aaye](/img/config/site-templates-list.png)

### Yíyanjú ìṣòro "template full of placeholders" {#solving-the-template-full-of-placeholders-problem}

Gbogbo èyí dára, ṣùgbọ́n a máa ń dojú kọ ìṣòro ẹlẹ́gbin kan: báyìí àwọn template aaye wa - tí àwọn oníbàárà wa le ṣàbẹ̀wò sí - kún fún àwọn placeholder ẹlẹ́gbin tí kò sọ púpọ̀.

Láti yíyanjú èyí, a nfunni ní aṣàyàn láti ṣètò àwọn iye àròsọ fún àwọn placeholder, a sì máa ń lo àwọn iye yẹn láti wá àti rọ́pò àkóónú wọn lori àwọn aaye template nígbà tí àwọn oníbàárà rẹ ń ṣàbẹ̀wò.

O le ní ìwọlé sí olootu placeholder template nípa lílọ sí **Ultimate Multisite > Settings > Sites**, lẹ́yìn náà, lori sidebar, tẹ ọna asopọ **Ṣàtúnṣe Àwọn Placeholder**.

![Àwọn ètò template aaye labẹ ojú-ìwé ètò Sites](/img/config/settings-sites-templates-section.png)

Èyí yoo mú ọ lọ sí olootu àkóónú àwọn placeholder, níbi tí o ti le ṣàfikún àwọn placeholder àti àkóónú tó bá wọn mu.

![Oju-ọ̀nà wọlé olootu àkóónú àwọn placeholder template](/img/config/settings-sites-templates-section.png)
