# Àwọn Ọgbọn Tí Ó Mọ Akori {#theme-aware-skills}

Superdav AI Agent v1.10.0 wa pẹlu àwọn ọgbọn inú-rẹ tuntun mẹrin tí ó mọ akori, tí wọ́n sì ń bá akori WordPress rẹ tí ó n ṣiṣẹ mu laifọwọyi. Àwọn ọgbọn wọ̀nyí ń pèsè ìtọ́nisọ́nà àti agbára amọ̀ja tí a ṣe pàtàkì fún faaji àti àwọn ẹya akori rẹ.

## Kí Ni Àwọn Ọgbọn Tí Ó Mọ Akori? {#what-are-theme-aware-skills}

Àwọn ọgbọn tí ó mọ akori jẹ́ àwọn ibi ìmọ̀ àti àwọn akojọpọ irinṣẹ́ tí a ti ṣètò tẹ́lẹ̀, tí olùrànlọ́wọ́ AI máa ń yan laifọwọyi gẹ́gẹ́ bí akori WordPress tí ó n ṣiṣẹ lọwọlọwọ lórí ojúlé rẹ. Nígbà tí o bá yí àwọn akori padà, àwọn ọgbọn tó wà fún olùrànlọ́wọ́ náà máa ń ṣe imudojuiwọn laifọwọyi — kò nílò ìṣètò ọwọ́.

Ọgbọn kọ̀ọ̀kan ní:

- **Ìwé ìtọ́nisọ́nà pàtàkì fún akori** — ìtọ́nisọ́nà lórí lílo àti ṣíṣe àṣàṣe akori náà
- **Àwọn ìtọ́ka bulọọki àti àwòṣe** — àwọn bulọọki, àwòṣe, àti àwọn aṣayan apẹrẹ tó wà
- **Àwọn àpẹẹrẹ àṣàṣe** — àwọn àpẹẹrẹ kóòdù àti àwọn àwòṣe ìṣètò fún àwọn iṣẹ́ tó wọ́pọ̀
- **Àwọn ìlànà tó dára jù** — àwọn àbá fún faaji àti ìṣàn-iṣẹ́ akori náà

## Àwọn Ọgbọn Tí Ó Mọ Akori Tó Wà {#available-theme-aware-skills}

### Àwọn Akori Bulọọki {#block-themes}

**Ó kan:** Àwọn akori tí ń lo faaji WordPress tó dá lórí bulọọki (Ìṣàtúnṣe Gbogbo Ojúlé).

Ọgbọn Àwọn Akori Bulọọki ń pèsè ìtọ́nisọ́nà lórí:

- Ṣíṣe àti ṣíṣàtúnṣe àwọn awoṣe ní lílo olóòtú bulọọki
- Ṣíṣiṣẹ́ pẹ̀lú àwọn àwòṣe bulọọki àti àwọn bulọọki tí a tún lè lò
- Ṣíṣe àṣàṣe àwọn ara gbogbogbo àti àwọn ìṣètò theme.json
- Lílo àwọn bulọọki akori àti àwọn iyatọ̀
- Kíkọ́ àwọn àwòṣe bulọọki àṣàṣe fún ojúlé rẹ

**A máa muu ṣiṣẹ laifọwọyi nígbà tí:** Akori rẹ tí ó n ṣiṣẹ jẹ́ akori bulọọki (ó ṣe àtìlẹ́yìn fún ẹya `block-templates`).

### Àwọn Akori Kílásíkì {#classic-themes}

**Ó kan:** Àwọn akori WordPress àtọwọ́dọwọ́ tí ń lo àwọn awoṣe PHP àti olóòtú kílásíkì.

Ọgbọn Àwọn Akori Kílásíkì ń pèsè ìtọ́nisọ́nà lórí:

- Ṣíṣiṣẹ́ pẹ̀lú àwọn fáìlì awoṣe PHP àti hooks
- Ṣíṣe àṣàṣe ìrísí akori nípasẹ̀ Customizer
- Lílo àwọn agbègbè widget àti àwọn sidebar
- Ṣíṣe àtúnṣe CSS àti ìdàgbàsókè akori ọmọ
- Lílóye ìpele akori àti àwọn àmì awoṣe

**A máa muu ṣiṣẹ laifọwọyi nígbà tí:** Akori rẹ tí ó n ṣiṣẹ jẹ́ akori kílásíkì (kì í ṣe bulọọki).

### Kadence Blocks {#kadence-blocks}

**Ó kan:** Àwọn ojúlé tí ń lo afikun Kadence Blocks fún apẹrẹ tó dá lórí bulọọki tó ti ni ilọsiwaju.

Ọgbọn Kadence Blocks ń pèsè ìtọ́nisọ́nà lórí:

- Lílo ilé-ìkàwé bulọọki tó ti ni ilọsiwaju ti Kadence (Hero, Testimonials, Pricing, abbl.)
- Ṣíṣètò àwọn ìṣètò bulọọki Kadence àti àwọn aṣayan tó ń fèsì sí ìwọ̀n ẹrọ
- Kíkọ́ àwọn ìtòlẹ́sẹẹsẹ àṣàṣe pẹ̀lú grid àti àwọn bulọọki container ti Kadence
- Ṣíṣepọ̀ àwọn bulọọki Kadence pẹ̀lú akori rẹ
- Lílo ètò apẹrẹ àti àwọn ìṣètò-tẹ́lẹ̀ ti Kadence

**A máa muu ṣiṣẹ laifọwọyi nígbà tí:** Afikun Kadence Blocks bá n ṣiṣẹ lórí ojúlé rẹ.

### Akori Kadence {#kadence-theme}

**Ó kan:** Àwọn ojúlé tí ń lo akori Kadence fún apẹrẹ àti àṣàṣe tó dá lórí bulọọki.

Ọgbọn Akori Kadence ń pèsè ìtọ́nisọ́nà lórí:

- Ṣíṣe àṣàṣe akori Kadence nípasẹ̀ àwọn ara gbogbogbo àti theme.json
- Lílo àwọn àwòṣe bulọọki àti awoṣe inú-rẹ ti Kadence
- Ṣíṣètò àwọn ìṣètò àti aṣayan akori Kadence
- Kíkọ́ àwọn apẹrẹ àṣàṣe pẹ̀lú ètò apẹrẹ Kadence
- Ṣíṣepọ̀ Kadence pẹ̀lú àwọn afikun àti irinṣẹ́ tó gbajúmọ̀

**A máa muu ṣiṣẹ laifọwọyi nígbà tí:** Akori Kadence jẹ́ akori rẹ tí ó n ṣiṣẹ.

## Bí A Ṣe Ń Yan Àwọn Ọgbọn {#how-skills-are-selected}

Olùrànlọ́wọ́ náà máa ń ṣàwárí akori rẹ tí ó n ṣiṣẹ àti àwọn afikun tí a fi sori ẹrọ laifọwọyi lórí ìfiránṣẹ́ kọ̀ọ̀kan. Bí ọgbọn tí ó mọ akori tó báamu bá wà, a máa kó o sínú àyíká olùrànlọ́wọ́ náà laifọwọyi. O kò nílò láti muu àwọn ọgbọn ṣiṣẹ tàbí yí wọn padà pẹ̀lú ọwọ́.

### Ọ̀pọ̀ Ọgbọn {#multiple-skills}

Bí ọ̀pọ̀ ọgbọn bá kan ojúlé rẹ (fún àpẹẹrẹ, bí Kadence Blocks àti Akori Kadence méjèèjì bá n ṣiṣẹ), olùrànlọ́wọ́ náà ní ààyè sí gbogbo àwọn ọgbọn tó yẹ, ó sì lè tọ́ka sí ìtọ́nisọ́nà láti ọ̀dọ̀ kọ̀ọ̀kan.

### Yíyí Àwọn Akori Padà {#switching-themes}

Nígbà tí o bá yí akori rẹ tí ó n ṣiṣẹ padà, àwọn ọgbọn tó wà fún olùrànlọ́wọ́ náà máa ń ṣe imudojuiwọn laifọwọyi lórí ìfiránṣẹ́ tó tẹ̀lé. Fún àpẹẹrẹ:

1. O ń lo akori bulọọki pẹ̀lú ọgbọn **Àwọn Akori Bulọọki** tí ó n ṣiṣẹ.
2. O yí padà sí akori kílásíkì.
3. Lórí ìfiránṣẹ́ rẹ tó tẹ̀lé, ọgbọn **Àwọn Akori Kílásíkì** ni a máa kó wọlé laifọwọyi, ọgbọn **Àwọn Akori Bulọọki** kò sì ní sí mọ́.

## Lílo Àwọn Ọgbọn Tí Ó Mọ Akori {#using-theme-aware-skills}

Láti lo ọgbọn tí ó mọ akori, ṣàpèjúwe ohun tí o fẹ́ ṣe nínú àgbékalẹ̀ ìfọ̀rọ̀wérọ̀. Olùrànlọ́wọ́ náà yóò tọ́ka sí ìtọ́nisọ́nà ọgbọn tó yẹ laifọwọyi.

### Àwọn Àpẹẹrẹ Ìbéèrè {#example-prompts}

**Fún Àwọn Akori Bulọọki:**
> "Ṣẹ̀dá apá hero kan pẹ̀lú àwòrán abẹ́lẹ̀ àti ọ̀rọ̀ tó wà ní àárín ní lílo àwọn àwòṣe bulọọki."

**Fún Àwọn Akori Kílásíkì:**
> "Ṣàfikún agbègbè widget àṣàṣe sí sidebar ní lílo akori ọmọ."

**Fún Kadence Blocks:**
> "Kọ́ apá testimonials kan ní lílo bulọọki Kadence Testimonials."

**Fún Akori Kadence:**
> "Ṣe àṣàṣe ìtòlẹ́sẹẹsẹ header àti ara àkojọ aṣàyàn ìrìnàjò."

Olùrànlọ́wọ́ náà yóò pèsè ìtọ́nisọ́nà pàtàkì fún akori àti àwọn àpẹẹrẹ kóòdù tí a ṣe pàtàkì fún akori àti àwọn afikun rẹ tí ó n ṣiṣẹ.

:::note
Àwọn ọgbọn tí ó mọ akori wà laifọwọyi nínú Superdav AI Agent v1.10.0 àti àwọn ẹ̀yà tó tẹ̀lé. Kò sí ìṣètò tàbí àtúnṣe míì tí a nílò.
:::
