# Ìṣàn Ìforúkọsílẹ̀ (v2)

_**ÀKÍYÈSÍ PÀTÀKÌ: Àpilẹ̀kọ yìí tọ́ka sí Ultimate Multisite ẹ̀yà 2.x.**_

Àwọn aṣàmúlò lè forúkọsílẹ̀ sí nẹ́tíwọ́ọ̀kì rẹ ní ọ̀nà oríṣìíríṣìí. Wọ́n lè lo fọ́ọ̀mù ìforúkọsílẹ̀ rẹ tàbí link tí a lè pín sí plan tí a ti yan ṣáájú. Níhìn-ín a ó fi hàn ọ́ bí àwọn oníbàárà rẹ ṣe lè forúkọsílẹ̀ lórí nẹ́tíwọ́ọ̀kì rẹ ní lílo àwọn ọ̀nà tó wà àti ohun tó máa ṣẹlẹ̀ lẹ́yìn tí wọ́n bá forúkọsílẹ̀ lórí nẹ́tíwọ́ọ̀kì rẹ.

## Lílo Fọ́ọ̀mù Ìforúkọsílẹ̀: {#using-the-registration-form}

Èyí ni ìlànà ìforúkọsílẹ̀ àtìgbàdégbà. O dá ojú-ewé ìforúkọsílẹ̀ pẹ̀lú **fọ́ọ̀mù ìparí rira**, èyí yóò sì jẹ́ ibi tí àwọn oníbàárà rẹ yóò lọ láti forúkọsílẹ̀ lórí nẹ́tíwọ́ọ̀kì rẹ kí wọ́n sì ṣe alabapin sí plan kan. O lè ní ọ̀pọ̀ ojú-ewé ìforúkọsílẹ̀, ọ̀kọ̀ọ̀kan pẹ̀lú fọ́ọ̀mù ìforúkọsílẹ̀ tó yàtọ̀ bí o bá fẹ́.

Ojú-ewé àiyipada fún ìforúkọsílẹ̀ ni [_**yourdomain.com/register**_](http://yourdomain.com/register), ṣùgbọ́n o lè yí èyí padà nígbàkigbà lórí **Ultimate Multisite > Settings > Login & Registration > Default Registration Page**.

Lẹ́yìn tí aṣàmúlò kan bá dé ojú-ewé ìforúkọsílẹ̀ rẹ (nígbà púpọ̀ nípa títẹ bọ́tìnì **Wọlé** tàbí **Ra báyìí**), wọ́n yóò rí fọ́ọ̀mù ìforúkọsílẹ̀ rẹ níbẹ̀.

![Fọ́ọ̀mù ìforúkọsílẹ̀ tí a fi hàn lórí ojú-ewé ìforúkọsílẹ̀](/img/frontend/registration-form.png)

Èyí ni àpẹẹrẹ fọ́ọ̀mù ìparí rira bí ó ṣe hàn lórí frontend:

![Fọ́ọ̀mù ìforúkọsílẹ̀ ìparí rira frontend](/img/config/checkout-frontend-registration.png)

Gbogbo ohun tí wọ́n ní láti ṣe ni fífi gbogbo àwọn pápá dandan kún - ímeèlì, orúkọ aṣàmúlò, ọ̀rọ̀ aṣínà, àti bẹ́ẹ̀ bẹ́ẹ̀ lọ... - àti sanwó fún plan náà tàbí jẹ́rìí sí àdírẹ́ẹ̀sì ímeèlì wọn bí wọ́n bá ń forúkọsílẹ̀ fún plan ọ̀fẹ́ tàbí plan tí a sanwó fún pẹ̀lú àkókò ìdánwò láìsí ìwífún ìsanwó.

Lórí ojú-ewé "Ẹ ṣé", wọ́n yóò rí ìfiránṣẹ́ tó ń sọ fún wọn bóyá wọ́n nílò láti jẹ́rìí sí àdírẹ́ẹ̀sì ímeèlì wọn tàbí bóyá website wọn ti ṣiṣẹ́ tán, wọ́n sì lè bẹ̀rẹ̀ sí í lò ó.

![Ojú-ewé Ẹ ṣé lẹ́yìn ìforúkọsílẹ̀](/img/frontend/registration-thank-you.png)

Bí ìjẹ́rìí àdírẹ́ẹ̀sì ímeèlì bá jẹ́ dandan, wọ́n yóò nílò láti lọ sí àpótí ìwọlé ímeèlì wọn kí wọ́n sì tẹ link ìjẹ́rìí náà. Website wọn kò ní ṣiṣẹ́ bí àdírẹ́ẹ̀sì ímeèlì wọn kò bá jẹ́rìí.

Bí wọ́n bá ti forúkọsílẹ̀ lórí plan tí a sanwó fún tàbí bí ìjẹ́rìí ímeèlì kò bá jẹ́ dandan lórí nẹ́tíwọ́ọ̀kì rẹ, website wọn yóò ṣiṣẹ́ lẹ́sẹ̀kẹsẹ̀ lẹ́yìn ìparí rira, a ó sì fi link kan hàn wọ́n láti wọlé sí Dashboard wọn.

![Site ti ṣiṣẹ́ pẹ̀lú link láti wọlé sí Dashboard](/img/frontend/site-activated.png)

## Lílo Link Tí A Lè Pín: {#using-a-shareable-link}

Ìlànà fíforúkọsílẹ̀ ní lílo link tí a lè pín jẹ́ ohun kan náà pẹ̀lú fọ́ọ̀mù ìforúkọsílẹ̀ ní pàtàkì; ìyàtọ̀ kan ṣoṣo ni pé ní lílo link tí a lè pín, àwọn oníbàárà rẹ lè ní ọja tàbí àwòṣe website tí a ti yan ṣáájú lórí fọ́ọ̀mù ìparí rira (tọ́ka sí apá Yíyan àwọn ọja àti àwọn àwòṣe ṣáájú nípasẹ̀ àwọn parameter URL) tàbí bóyá coupon code kan tí a ti ṣàfikún (tọ́ka sí apá Lílo Àwọn Parameter URL).

Ìlànà ìforúkọsílẹ̀ yóò jẹ́ ohun kan náà: wọ́n yóò nílò láti kun orúkọ wọn, orúkọ aṣàmúlò, àdírẹ́ẹ̀sì ímeèlì, orúkọ website àti àkọlé, àti bẹ́ẹ̀ bẹ́ẹ̀ lọ... ṣùgbọ́n plan tàbí àwòṣe site yóò ti jẹ́ èyí tí a ti yan ṣáájú fún wọn.

### Fíforúkọsílẹ̀ Ní Lílo Àwọn Ìsanwó Ọwọ́: {#registering-using-manual-payments}

Bí o kò bá fẹ́ lo PayPal, Stripe tàbí ẹnu-ọ̀nà ìsanwó mìíràn tí Ultimate Multisite tàbí àwọn ìṣepọ̀ add-on rẹ ń pèsè, o lè lo àwọn ìsanwó ọwọ́ fún àwọn oníbàárà rẹ. Ní ọ̀nà yìí, o lè ṣẹ̀dá invoice fún wọn láti san lórí payment processor tí o fẹ́ lẹ́yìn tí wọ́n bá forúkọsílẹ̀ lórí nẹ́tíwọ́ọ̀kì rẹ.

Ìlànà ìforúkọsílẹ̀ yóò jẹ́ ohun kan náà gẹ́gẹ́ bí lókè, ṣùgbọ́n lórí ojú-ewé ìforúkọsílẹ̀ àwọn oníbàárà rẹ yóò rí ìfiránṣẹ́ kan tó sọ pé wọ́n yóò gba ímeèlì kan pẹ̀lú àwọn ìtọ́nisọ́nà síwájú síi láti parí ìsanwó náà.

![Ìfiránṣẹ́ ìsanwó ọwọ́ nígbà ìforúkọsílẹ̀](/img/frontend/registration-manual-notice.png)

Àti lẹ́yìn tí ìforúkọsílẹ̀ bá ti parí, wọ́n yóò rí àwọn ìtọ́nisọ́nà ìsanwó tí o ṣètò (wọn yóò sì tún gbà á lórí ímeèlì wọn).

![Àwọn ìtọ́nisọ́nà ìsanwó tí a fi hàn lẹ́yìn ìforúkọsílẹ̀](/img/frontend/registration-payment-instructions.png)

Àwọn ìtọ́nisọ́nà ìsanwó lè yí padà lórí **Ultimate Multisite > Settings > Payments** lẹ́yìn tí o bá tan aṣayan ìsanwó **Ọwọ́**:

![Toggle ìsanwó ọwọ́ pẹ̀lú pápá àwọn ìtọ́nisọ́nà ìsanwó](/img/config/manual-gateway-settings.png)

Lẹ́yìn tí àwọn oníbàárà rẹ bá parí ìsanwó ọwọ́ tí wọ́n sì fi ìjẹ́rìí ránṣẹ́ sí ọ, o nílò láti **jẹ́rìí sí ìsanwó náà lọ́wọ́** láti mú membership àti website oníbàárà náà ṣiṣẹ́.

Láti ṣe èyí, lọ sí **Ultimate Multisite > Payments** kí o sì wá ìsanwó oníbàárà náà. Ó yẹ kí ó ṣì fi ipò **Ní ìdúró** hàn.

![Àtòkọ àwọn ìsanwó pẹ̀lú ìsanwó ọwọ́ tí ó wà ní ìdúró](/img/admin/payments-list.png)

Tẹ nọ́ńbà ìsanwó náà, ìwọ yóò sì lè yí ipò rẹ padà sí **Ti parí**.

![Ojú-ewé àwọn àlàyé ìsanwó](/img/admin/payment-edit.png)

![Yíyí ipò ìsanwó padà sí Ti parí](/img/admin/payment-status-completed.png)

Lẹ́yìn yíyí ipò rẹ padà sí **Ti parí** , ó yẹ kí o rí ìfiránṣẹ́ **Mú membership ṣiṣẹ́**. Tan aṣayan yìí **sí on** láti mú membership àti website tó ní í ṣe pẹ̀lú oníbàárà yìí ṣiṣẹ́. Lẹ́yìn náà, tẹ láti **Fi Ìsanwó Pamọ́**.

![Toggle Mú membership ṣiṣẹ́ àti bọ́tìnì Fi Ìsanwó Pamọ́](/img/admin/payment-activate-membership.png)

Oníbàárà rẹ yẹ kí ó lè wọlé sí Dashboard àti gbogbo àwọn ẹ̀yà iṣẹ́ tí wọ́n ṣe alabapin sí báyìí.
