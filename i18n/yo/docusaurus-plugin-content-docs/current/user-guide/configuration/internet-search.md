# Ìṣàwárí Íńtánẹ́ẹ̀tì

Gratis AI Agent v1.5.0 ṣàfikún agbára **Ìṣàwárí Íńtánẹ́ẹ̀tì** tó jẹ́ kí olùrànlọ́wọ́ AI gba ìwífún tuntun láti orí wẹ́ẹ̀bù nígbà ìjíròrò. Èyí ń ṣiṣẹ́ pẹ̀lú [Brave Search API](https://brave.com/search/api/) tàbí [Tavily API](https://tavily.com/).

## Bí Ó Ṣe Ń Ṣiṣẹ́

Nígbà tí ìṣàwárí íńtánẹ́ẹ̀tì bá ṣiṣẹ́, olùrànlọ́wọ́ lè béèrè lọ́wọ́ olupèsè ìṣàwárí tí o ti ṣètò ní aládàáṣiṣẹ́ nígbà tó bá pinnu pé ìbéèrè kan nílò ìwífún tuntun tàbí ti òde — fún àpẹẹrẹ, ìròyìn tuntun, owó tí ń lọ lọwọ, tàbí ìwé ìtọ́nisọ́nà tó lè ti yí padà lẹ́yìn àkókò ìdádúró ìkẹ́kọ̀ọ́ awoṣe náà.

A máa gba àwọn èsì ní àsìkò gidi, a ó sì fi wọ́n sínú àyíká ọ̀rọ̀ olùrànlọ́wọ́ kí ó tó dá èsì sílẹ̀. Olùrànlọ́wọ́ máa tọ́ka sí i nígbà tó bá ti lo àwọn èsì ìṣàwárí láti dáhùn ìbéèrè kan.

## Mímú Ìṣàwárí Íńtánẹ́ẹ̀tì Ṣiṣẹ́

Ìṣàwárí íńtánẹ́ẹ̀tì nílò API key láti ọ̀dọ̀ olupèsè ìṣàwárí tí o yàn. Láti ṣètò rẹ̀:

1. Lọ sí **Gratis AI Agent → Ètò → Ìlọsíwájú**.
2. Wá àṣàyàn-sílẹ̀ **Olùpèsè Ìṣàwárí Íńtánẹ́ẹ̀tì** kí o sì yan bóyá **Brave Search** tàbí **Tavily**.
3. Tẹ API key rẹ sínú pápá tó bá a mu. Àwọn ìjápọ̀ ìforúkọsílẹ̀ wà lẹ́gbẹ̀ẹ́ pápá kọ̀ọ̀kan.
4. Tẹ **Fi Ètò Pamọ́**.

Lẹ́yìn tí a bá ti fi kọ́kọ́rọ́ náà pamọ́, agbára Ìṣàwárí Íńtánẹ́ẹ̀tì yóò wà fún olùrànlọ́wọ́ ní aládàáṣiṣẹ́.

## Brave Search

### Gbigba Brave Search API Key

1. Ṣàbẹ̀wò sí [ojúewé Brave Search API](https://brave.com/search/api/).
2. Forúkọ sílẹ̀ fún ètò kan. Ìpele ọ̀fẹ́ wà pẹ̀lú ìdíwọ̀n ìbéèrè oṣooṣù.
3. Ṣàdàkọ API key rẹ láti Brave Search developer dashboard.
4. Lẹ̀ ẹ́ sínú pápá **Brave Search API Key** nínú àwọn ètò Gratis AI Agent.

### Àwọn Ìdíwọ̀n Lílò

Brave Search máa ń gba owó lílò gẹ́gẹ́ bí iye àwọn ìbéèrè tí a ṣe. Èsì AI kọ̀ọ̀kan tó bá fa ìṣàwárí jẹ́ ìbéèrè kan. Ṣàbójútó lílò rẹ nínú [Brave Search developer dashboard](https://brave.com/search/api/) láti yàgò fún àwọn ìnáwó àìròtẹ́lẹ̀.

## Tavily

Superdav AI Agent v1.10.0 ṣàfikún **Tavily** gẹ́gẹ́ bí olupèsè ìṣàwárí íńtánẹ́ẹ̀tì mìíràn, tó ń fúnni ní àwọn èsì ìṣàwárí tó jinlẹ̀ sí i àti àwọn agbára ìwádìí tó ti ni ilọsíwájú.

### Gbigba Tavily API Key

1. Ṣàbẹ̀wò sí [ojúewé Tavily API](https://tavily.com/).
2. Forúkọ sílẹ̀ fún Account kan. Ìpele ọ̀fẹ́ wà pẹ̀lú ìdíwọ̀n ìbéèrè oṣooṣù.
3. Ṣàdàkọ API key rẹ láti Tavily dashboard.
4. Lẹ̀ ẹ́ sínú pápá **Tavily API Key** nínú àwọn ètò Gratis AI Agent.

### Àwọn Ìdíwọ̀n Lílò

Tavily máa ń gba owó lílò gẹ́gẹ́ bí iye àwọn ìpè API tí a ṣe. Èsì AI kọ̀ọ̀kan tó bá fa ìṣàwárí jẹ́ ìpè kan. Ṣàbójútó lílò rẹ nínú [Tavily dashboard](https://tavily.com/) láti yàgò fún àwọn ìnáwó àìròtẹ́lẹ̀.

## Pípa Ìṣàwárí Íńtánẹ́ẹ̀tì

Yọ API key kúrò nínú pápá olupèsè ìṣàwárí tó ń ṣiṣẹ́, kí o sì fi pamọ́. A kò ní tún fi agbára Ìṣàwárí Íńtánẹ́ẹ̀tì fún olùrànlọ́wọ́ mọ́.

:::note
Ìṣàwárí íńtánẹ́ẹ̀tì ń ṣàfikún ìdádúró sí àwọn èsì nítorí olùrànlọ́wọ́ gbọ́dọ̀ dúró de àwọn èsì ìṣàwárí kí ó tó dáhùn. Fún àwọn ọ̀nà lílò tí àsìkò ṣe pàtàkì, ronú bóyá ìṣàwárí àsìkò gidi ṣe pàtàkì tàbí bóyá ìmọ̀ inú olùrànlọ́wọ́ fúnra rẹ̀ ti tó.
:::
