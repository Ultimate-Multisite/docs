# Ṣíṣe Àwárí àti Ríròpo Laifọwọyi nínú Àwọn Template Site (v2) {#auto-search-and-replace-in-site-templates-v2}

_**Ìkọ́ni yìí nílò WP UItimo ẹ̀yà 2.x.**_

Ọ̀kan lára àwọn ànfààní tó lágbára jùlọ ti Ultimate Multisite ni agbára láti ṣàfikún ọ̀rọ̀, àwọ̀, àti àwọn field yíyan bí ó ṣe fẹ́ sí fọ́ọ̀mù ìforúkọsílẹ̀. Nígbà tí a bá ti gba data yẹn, a lè lò ó láti kọ́kọ́ kún àkóónú ní àwọn apá kan nínú template site tí a yàn. Lẹ́yìn náà, nígbà tí site tuntun bá ti jẹ́ títẹ̀jáde, Ultimate Multisite yóò rọ́pò àwọn placeholder pẹ̀lú ìwífún gangan tí a tẹ nígbà ìforúkọsílẹ̀.

Fún àpẹẹrẹ, o lè ṣe àwọn site template rẹ pẹ̀lú àwọn placeholder. Ó yẹ kí a ṣàfikún àwọn placeholder ní àyíká àmì akọ́mọ́ oníyí méjì - {{placeholder_name}}.

Lẹ́yìn náà, o lè kàn ṣàfikún field ìforúkọsílẹ̀ tó bá a mu láti gba data yẹn

Oníbàárà rẹ yóò sì lè kún field yẹn nígbà ìforúkọsílẹ̀.

Ultimate Multisite yóò sì rọ́pò àwọn placeholder pẹ̀lú data tí oníbàárà pèsè laifọwọyi.

## **Yíyanjú ìṣòro "template tí ó kún fún àwọn placeholder"** {#solving-the-template-full-of-placeholders-problem}

Gbogbo èyí dára, ṣùgbọ́n a máa ń bá ìṣòro búburú kan pàdé: ní báyìí, àwọn site template wa - tí àwọn oníbàárà wa lè ṣèbẹ̀wò sí - kún fún àwọn placeholder búburú tí kò sọ púpọ̀.

Láti yanju èyí, a pèsè àṣàyàn láti ṣètò àwọn iye àfọ̀mọ́ fún àwọn placeholder, a sì máa ń lo àwọn iye wọ̀nyẹn láti wá àti rọ́pò àkóónú wọn lórí àwọn site template nígbà tí àwọn oníbàárà rẹ bá ń ṣèbẹ̀wò.

O lè ní ìráyè sí editor àwọn template placeholder nípa lílọ sí **Ultimate Multisite > Settings > Sites**, yíyí lọ sí agbègbè Site Template Options, lẹ́yìn náà kí o tẹ link **Edit Placeholders**.

![Agbègbè Site Template Options nínú ojúewé Settings Sites](/img/config/settings-sites-templates-section.png)

Ìyẹn yóò mú ọ lọ sí editor àkóónú àwọn placeholder, níbi tí o ti lè ṣàfikún àwọn placeholder àti àkóónú wọn tó yẹ.

![Ojútùú ìwọlé editor template placeholders](/img/config/settings-sites-templates-section.png)
