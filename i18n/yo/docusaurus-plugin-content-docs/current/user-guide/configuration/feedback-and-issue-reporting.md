# Esi Oníbàárà & Ìjabọ Ìṣòro

Gratis AI Agent v1.5.0 ṣàgbékalẹ̀ ètò esi àti ìjabọ ìṣòro tí a kọ sínú rẹ̀, tí ó jẹ́ kí àwọn olumulo opin lè tọ́ka sí àwọn ìdáhùn tí kò wúlò kí wọ́n sì jabo àwọn ìṣòro tààrà láti inú wiwo ìfọ̀rọ̀wérọ̀. Ètò yìí ní ìṣàkóso ìfọwọ́sí, àṣẹ ìjabọ aládàáṣiṣẹ́, àti ìtúpalẹ̀ ìpele-àkọ́kọ́ tí AI ràn lọ́wọ́ ní backend.

## Bọ́tìnì Àtàǹpàkò-sílẹ̀

Gbogbo ìfiránṣẹ́ tí olùrànlọ́wọ́ AI bá rán ń fi bọ́tìnì **àtàǹpàkò-sílẹ̀** (👎) hàn. Nígbà tí olumulo bá tẹ ẹ, wọ́n lè samisi ìdáhùn kan gẹ́gẹ́ bí èyí tí kò wúlò tàbí tí kò tọ́.

- Bọ́tìnì náà máa hàn nígbà tí a bá gbe kọ́sọ̀ sórí rẹ̀ lẹ́gbẹ̀ẹ́ ìfiránṣẹ́ olùrànlọ́wọ́ kọ̀ọ̀kan.
- Títẹ ẹ máa ṣí **Ferese Ìfọwọ́sí Esi**.
- Esi náà ní ìbáṣepọ̀ pẹ̀lú okun ìfọ̀rọ̀wérọ̀ àti ìfiránṣẹ́ pàtó náà.

## Ferese Ìfọwọ́sí Esi

Nígbà tí olumulo bá tẹ bọ́tìnì àtàǹpàkò-sílẹ̀, ferese ìfọwọ́sí kan máa hàn kí a tó fi data kankan ránṣẹ́. Ferese náà:

- Ṣàlàyé irú ìwífún wo ni a máa kó jọ (àyọkà ìfọ̀rọ̀wérọ̀, àyíká browser).
- Béèrè lọ́wọ́ olumulo láti jẹ́rìí pé wọ́n fọwọ́ sí pípín data yìí.
- Pèsè pápá ọ̀rọ̀-ọ̀fẹ́ fún olumulo láti ṣàpèjúwe ohun tó ṣẹlẹ̀ ní àìtọ́.
- Pèsè aṣayan **Fagilé** láti pa á tì láìfi ránṣẹ́.

A kò ní gba esi kankan sílẹ̀ títí olumulo yóò fi jẹ́rìí ní kedere.

## Bánà Esi Ìbéèrè-Aládàáṣiṣẹ́

Ní ìparí ìfọ̀rọ̀wérọ̀ kan, olùrànlọ́wọ́ le fi **bánà esi ìbéèrè-aládàáṣiṣẹ́** hàn — ìfiránṣẹ́ tí kò díni lọ́wọ́ tí ń béèrè bóyá ìpàdé náà wúlò.

Bánà yìí máa hàn ní aládàáṣiṣẹ́ gẹ́gẹ́ bí gígùn ìfọ̀rọ̀wérọ̀ àti àwọn ìlànà ìtẹ̀síwájú abajade. Ó sopọ̀ mọ́ ìṣàn esi kan náà bí bọ́tìnì àtàǹpàkò-sílẹ̀. Àwọn olumulo lè pa bánà náà tì láìpèsè esi.

## Àṣẹ Slash /report-issue

Àwọn olumulo lè fa ìṣàn esi jáde tààrà nípa títẹ `/report-issue` sínú ààyè ìwọlé ìfọ̀rọ̀wérọ̀. Àṣẹ yìí:

- Ṣí Ferese Ìfọwọ́sí Esi lẹ́sẹ̀kẹsẹ̀.
- Fọwọ́si pápá àpèjúwe ṣáájú pẹ̀lú àyíká nípa ìfọ̀rọ̀wérọ̀ lọ́wọ́lọ́wọ́.
- Gba àwọn olumulo láyè láti ṣàfikún àlàyé míì ṣáájú fífi ránṣẹ́.

Àṣẹ `/report-issue` wà ní gbogbo àwọn ipo ìfọ̀rọ̀wérọ̀ (nínú ìlà, widget tí ń fò, iboju-kíkún).

## Ìtúpalẹ̀ Ìpele-àkọ́kọ́ tí AI Ràn Lọ́wọ́

Esi tí a fi ránṣẹ́ ni a máa darí sí fẹ́lẹ́fẹ́lẹ̀ ìtúpalẹ̀ ìpele-àkọ́kọ́ AI tí ó:

- Pín ìjabọ náà sí ẹ̀ka (àṣìṣe òtítọ́, ìdáhùn tí kò wúlò, àkóónú tí kò yẹ, àti bẹ́ẹ̀ bẹ́ẹ̀ lọ).
- So metadata ìfọ̀rọ̀wérọ̀ tó yẹ mọ́ ọn.
- Dari àkópọ̀ ìtúpalẹ̀ náà sí **endpoint Esi** tí a ti ṣètò (wo [Ètò > To ti ni ilọsiwaju](#settings-advanced)).

Èyí dín àkókò àyẹ̀wò ọwọ́ kù nípa fífi àwọn ìṣòro tó ṣe pàtàkì jù lọ hàn ní àkọ́kọ́.

## Ètò > To ti ni ilọsiwaju {#settings-advanced}

Láti mú backend esi ṣiṣẹ́, ṣètò àwọn pápá wọ̀nyí lábẹ́ **Gratis AI Agent → Ètò → To ti ni ilọsiwaju**:

| Pápá | Àpèjúwe |
|---|---|
| **URL endpoint Esi** | URL tí ń gba àwọn ìbéèrè POST pẹ̀lú payloads esi (JSON). |
| **Feedback API Key** | Bearer token tí a fi ránṣẹ́ nínú `Authorization` header fún ìfìdí-múlẹ̀ àwọn fífi esi ránṣẹ́. |

Fi àwọn pápá méjèèjì sílẹ̀ ní òfo láti pa kíkó esi jọ. Àwọn bọ́tìnì esi kọ̀ọ̀kan yóò ṣì hàn sí àwọn olumulo, ṣùgbọ́n a kò ní dari àwọn fífi ránṣẹ́ náà síwájú.

:::tip
endpoint esi náà gbọ́dọ̀ gba ara JSON pẹ̀lú ó kéré tán àwọn pápá `message_id`, `conversation_id`, `feedback_text`, àti `triage_category`. Wo ìwé ìtọ́nisọ́nà olùpèsè endpoint rẹ fún schema tí a ń retí.
:::
