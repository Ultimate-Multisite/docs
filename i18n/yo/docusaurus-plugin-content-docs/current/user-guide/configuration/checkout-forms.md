# Àwọn Fọọmu Checkout {#checkout-forms}

Àwọn Fọọmu Checkout jẹ́ ọ̀nà rọrùn àti tó rọ láti dán àwọn ọ̀nà míì wò nígbà tí o bá ń gbìyànjú láti yí àwọn oníbàárà tuntun padà sí olùrà.

Ultimate Multisite 2.0 ń pèsè olóòtú Fọọmu Checkout kan tí ó jẹ́ kí o dá fọọmu púpọ̀ tó bí o ṣe fẹ́, pẹ̀lú àwọn pápá ọ̀tọ̀ọ̀tọ̀, àwọn ọjà tí a ń fúnni, àti bẹ́ẹ̀ bẹ́ẹ̀ lọ.

Láti wọlé sí ẹ̀yà yìí, lọ sí àkójọ aṣàyàn Àwọn Fọọmu Checkout, ní ẹgbẹ́-ìkànnì òsì.

![Àkójọ àwọn Fọọmu Checkout](/img/config/checkout-forms-list.png)

Lórí ojú-ìwé yìí, o lè rí gbogbo àwọn fọọmu checkout tí o ní.

Tábìlì àkójọ náà ní kọ́lọ̀mù **Ipò** kí o lè jẹ́risi bóyá fọọmu kọ̀ọ̀kan wà fún àwọn oníbàárà lọ́wọ́lọ́wọ́:

| Ipò | Ìtumọ̀ |
|---|---|
| **Ń ṣiṣẹ́** | A lè lo fọọmu náà ní ibikíbi tí shortcode rẹ̀ tàbí ojú-ìwé ìforúkọsílẹ̀ rẹ̀ ti jẹ́ títẹ̀jáde. |
| **Kò ṣiṣẹ́** | A ti fi fọọmu náà pamọ́ ṣùgbọ́n a ti pa á. Àwọn oníbàárà kò lè parí checkout pẹ̀lú rẹ̀ títí tí o fi tún muu ṣiṣẹ́. |

Lo kọ́lọ̀mù ipò náà kí o tó ṣàtúnṣe ìṣàn ìforúkọsílẹ̀ gbangba, pàápàá nígbà tí o bá ń pa àwọn fọọmu checkout àkọ́kọ́ tàbí ti àsìkò mọ́ lẹ́gbẹ̀ẹ́ àwọn fọọmu rẹ tí ń ṣiṣẹ́.

Tí o bá fẹ́ dá tuntun kan, kan tẹ Fi Fọọmu Checkout Kún un lókè ojú-ìwé náà.

O lè yan ọ̀kan nínú àwọn aṣàyàn mẹ́ta wọ̀nyí gẹ́gẹ́ bí ìbẹ̀rẹ̀ rẹ: ìgbésẹ̀ kan ṣoṣo, ọ̀pọ̀ ìgbésẹ̀ tàbí òfo. Lẹ́yìn náà, tẹ láti Lọ sí Olóòtú.

Nígbà tí o bá yan **ìgbésẹ̀ kan ṣoṣo** tàbí **ọ̀pọ̀ ìgbésẹ̀** gẹ́gẹ́ bí ìbẹ̀rẹ̀ rẹ, àdàkọ fọọmu náà ní pápá **Yíyan Àdàkọ** ní aiyipada báyìí. Pápá yìí jẹ́ kí àwọn oníbàárà rẹ yan àdàkọ site kan nígbà ìlànà ìforúkọsílẹ̀. O lè fi síbẹ̀, yọ ọ́ kúrò, tàbí tún ipò rẹ̀ ṣe bí pápá míìkí míì nínú olóòtú náà.

![Olóòtú Fọọmu Checkout](/img/config/checkout-form-editor.png)

Ní ọ̀nà míì, o lè ṣàtúnṣe tàbí ṣe ẹ̀dà àwọn fọọmu tí o ti ní tẹ́lẹ̀ nípa títẹ àwọn aṣàyàn tó wà ní ìsàlẹ̀ orúkọ rẹ̀. Ní bẹ̀, ìwọ yóò tún rí àwọn aṣàyàn láti daakọ shortcode fọọmu náà tàbí láti pa fọọmu náà rẹ́.

![Àwọn ìṣe hover fọọmu checkout](/img/config/checkout-form-hover-actions.png)

#### Àkíyèsí Gbogbogbò Olóòtú Fọọmu Checkout {#checkout-form-editor-overview}

Olóòtú fọọmu checkout ń pèsè ojúṣe tó péye fún kíkọ àwọn fọọmu ìforúkọsílẹ̀ rẹ. Èyí ni àkíyèsí gbogbogbò ìṣètò olóòtú náà:

![Àkíyèsí gbogbogbò olóòtú Fọọmu Checkout](/img/config/checkout-form-editor-overview.png)

### Ṣíṣàtúnṣe Fọọmu Checkout kan {#editing-a-checkout-form}

O lè dá àwọn fọọmu checkout fún àwọn ète ọ̀tọ̀ọ̀tọ̀. Nínú àpẹẹrẹ yìí a máa ṣiṣẹ́ lórí fọọmu ìforúkọsílẹ̀ kan.

Lẹ́yìn tí o bá ti lọ sí olóòtú fọọmu checkout, fún fọọmu rẹ ní orúkọ kan (tí a óò lò fún ìtọ́kasí inú nìkan) àti slug kan (tí a lò láti dá shortcakes, fún àpẹẹrẹ).

![Olóòtú Fọọmu Checkout](/img/config/checkout-form-editor.png)

Àwọn fọọmu jẹ́ ti àwọn ìgbésẹ̀ àti pápá. O lè fi ìgbésẹ̀ tuntun kún un nípa títẹ Fi Ìgbésẹ̀ Checkout Tuntun Kún un.

![Fi Ìgbésẹ̀ Checkout Tuntun Kún un](/img/config/checkout-form-add-step.png)

Lórí taabu àkọ́kọ́ ti fèrèsé modal, kún àkóónú ìgbésẹ̀ fọọmu rẹ. Fún un ní ID, orúkọ àti àpèjúwe. Àwọn nǹkan wọ̀nyí ni a máa ń lò jùlọ ní inú eto.

![Taabu àkóónú ìgbésẹ̀ fọọmu checkout](/img/config/checkout-form-step-content.png)

Tó kàn, ṣètò ìfarahàn ìgbésẹ̀ náà. O lè yan láàárín "Máa fi hàn nígbà gbogbo", "Fi hàn fún àwọn oníṣe tí wọlé nìkan" tàbí "Fi hàn fún àwọn àlejò nìkan".

![Taabu ìfarahàn ìgbésẹ̀ fọọmu checkout](/img/config/checkout-form-step-visibility.png)

Ní ìkẹyìn, ṣètò ara ìgbésẹ̀ náà. Àwọn wọ̀nyí jẹ́ pápá àṣàyàn.

![Taabu ara ìgbésẹ̀ fọọmu checkout](/img/config/checkout-form-step-style.png)

Báyìí, àsìkò ti tó láti fi àwọn pápá kún ìgbésẹ̀ àkọ́kọ́ wa. Kan tẹ láti Fi Pápá Tuntun Kún un kí o sì yan irú abala tí o fẹ́.

![Ìgbésẹ̀ fọọmu checkout pẹ̀lú àwọn pápá](/img/config/checkout-form-step-with-fields.png)

Pápá kọ̀ọ̀kan ní àwọn parâmità ọ̀tọ̀ọ̀tọ̀ tí a gbọ́dọ̀ kún. Fún ìwọlé àkọ́kọ́ yìí, a óò yan pápá "Orúkọ oníṣe".

![Àṣàyàn irú pápá dropdown](/img/config/checkout-form-field-type-dropdown.png)

![Àwọn ètò àkóónú pápá Orúkọ oníṣe](/img/config/checkout-form-username-content.png)

![Àwọn ètò ìfarahàn pápá Orúkọ oníṣe](/img/config/checkout-form-username-visibility.png)

![Àwọn ètò ara pápá Orúkọ oníṣe](/img/config/checkout-form-username-style.png)

O lè fi ìgbésẹ̀ àti pápá pọ̀ tó bí o ṣe nílò. Láti fi àwọn ọjà rẹ hàn fún àwọn oníbàárà rẹ kí wọ́n lè yan ọ̀kan, lo pápá Tábìlì Iye. Tí o bá fẹ́ jẹ́ kí àwọn oníbàárà rẹ yan àdàkọ kan, fi pápá Yíyan Àdàkọ kún un. Bẹ́ẹ̀ lọ.

![Ìbánisọ̀rọ̀ Fi Pápá Tuntun Kún un](/img/config/checkout-form-add-field-dialog.png)

_**Àkíyèsí:** orúkọ oníṣe, email, ọ̀rọ̀ aṣínà, àkọlé site, URL site, àkótán ìbẹ̀rẹ̀, ìsanwó, àti bọ́tìnì fífi ránṣẹ́ jẹ́ àwọn pápá dandan láti dá fọọmu checkout._

Nígbà tí o bá ń ṣiṣẹ́ lórí fọọmu checkout rẹ, o lè máa lo bọ́tìnì Àyẹ̀wò ṣáájú nígbà gbogbo láti rí bí àwọn oníbàárà rẹ yóò ṣe rí fọọmu náà. O tún lè yí padà láàárín wíwo gẹ́gẹ́ bí oníṣe tó wà tẹ́lẹ̀ tàbí àlejò

![Bọ́tìnì àyẹ̀wò ṣáájú Fọọmu Checkout](/img/config/checkout-form-preview-button.png)

![Modal àyẹ̀wò ṣáájú Fọọmu Checkout](/img/config/checkout-form-preview-modal.png)

Ní ìkẹyìn, lórí Àwọn Aṣàyàn To ti Ní Ilọsíwájú o lè ṣètò ìfiránṣẹ́ fún ojú-ìwé "Ẹ Ṣé", fi snippets kún un láti tọpa àwọn ìyípadà, fi CSS àdáni kún fọọmu checkout rẹ tàbí dín ìwọlé rẹ̀ mọ́ àwọn orílẹ̀-èdè kan.

![Àwọn Aṣàyàn To ti Ní Ilọsíwájú](/img/config/checkout-form-advanced.png)

O tún lè fi ọwọ́ muu fọọmu checkout rẹ ṣiṣẹ́ tàbí pa á nípa yíyan aṣàyàn yìí lórí kọ́lọ̀mù ọ̀tún, tàbí pa fọọmu náà rẹ́ pátápátá.

![Toggle tó ń ṣiṣẹ́](/img/config/checkout-form-active.png)

Láti pa fọọmu kan rẹ́, tẹ aṣàyàn piparẹ́ nínú àwọn ìṣe fọọmu:

![Aṣàyàn piparẹ́ fọọmu checkout](/img/config/checkout-form-delete.png)

Má ṣe gbàgbé láti fi fọọmu checkout rẹ pamọ́!

![Bọ́tìnì ìpamọ́](/img/config/checkout-form-save.png)

Láti gba shortcode fọọmu rẹ tẹ láti Ṣẹ̀dá Shortcode kí o sì daakọ èsì tí a fi hàn lórí fèrèsé modal.

![Bọ́tìnì ìpamọ́ pẹ̀lú shortcode](/img/config/checkout-form-save.png)

### Pápá Tábìlì Iye {#the-pricing-table-field}

Pápá **Tábìlì Iye** ń fi àwọn ọjà rẹ hàn lórí fọọmu checkout kí àwọn oníbàárà lè yan plan kan. Nígbà tí o bá ṣàtúnṣe pápá yìí, o lè ṣètò àwọn aṣàyàn mélòó kan:

![Àwọn ètò pápá Tábìlì Iye](/img/config/pricing-table-field-settings.png)

Èyí ni bí tábìlì iye ṣe máa hàn lórí fọọmu ìforúkọsílẹ̀ frontend:

![Tabili iye owó ìsanwó frontend](/img/config/frontend-checkout-pricing-table.png)

  * **Àwọn Ọja**: Yan àwọn ọja tí a ó fi hàn àti ìtòlẹ́sẹẹsẹ tí wọ́n máa hàn.
  * **Fi Àwọn Ìpẹ́ Tó Yàtọ̀ Múlẹ̀**: Nígbà tí a bá ti mú un ṣiṣẹ́, gbogbo ọja ni a máa fi hàn láìka bóyá wọ́n ní ìyípadà iye owó tó bá akoko ìdíyelé tí a yan lọ́wọ́ mu tàbí rara. Nígbà tí a kò bá mú un ṣiṣẹ́ (àiyípadà), àwọn ọja tí kò ní ìyípadà fún akoko tí a yan ni a máa fi pamọ́.
    ![Àṣàyàn fífi àwọn ìpẹ́ múlẹ̀ fún tabili iye owó](/img/config/pricing-table-force-durations.png)
  * **Fi pamọ́ nígbà tí a bá ti Yan Ṣáájú**: Fi tabili iye owó pamọ́ nígbà tí a bá ti yan plan kan tẹ́lẹ̀ nípasẹ̀ URL (fún àpẹẹrẹ `/register/premium`).
  * **Àdàkọ Tabili Iye Owó**: Yan àdàkọ ìríran fún tabili iye owó (Àtòjọ Rọrùn, Legacy, àti bẹ́ẹ̀ bẹ́ẹ̀ lọ).

Tí o bá fi ọja kan kún Tabili Iye Owó kí fọ́ọ̀mù náà tó ní àwọn ààyè tí a nílò láti parí ìṣàn ìsanwó ọja yẹn, olootu náà yóò fi ìkìlọ̀ hàn báyìí. Lo ìkìlọ̀ náà láti fi ààyè dandan tó ṣòfò kún un kí o tó tẹ̀ jáde tàbí fi àwọn ayípadà pamọ́ fún fọ́ọ̀mù ìforúkọsílẹ̀ tó wà láàyè.

### Fífi Toggle Yíyan Akoko Kún Un {#adding-a-period-selection-toggle}

Tí o bá ti ṣètò [Àwọn Ìyípadà Iye Owó](creating-your-first-subscription-product#price-variations) lórí àwọn ọja rẹ (fún àpẹẹrẹ iye owó oṣooṣù àti ọdọọdún), o lè fi ààyè **Yíyan Akoko** kún fọ́ọ̀mù ìsanwó rẹ. Ààyè yìí ń fi toggle hàn tí ó jẹ́ kí àwọn oníbàárà yí padà láàárín àwọn akoko ìdíyelé, tabili iye owó sì ń ṣe àfikún lójú-ẹsẹ̀ ní àsìkò gidi.

#### Ìgbésẹ̀ 1: Ṣètò Àwọn Ìyípadà Iye Owó lórí Àwọn Ọja Rẹ {#step-1-set-up-price-variations-on-your-products}

Kí o tó fi ààyè Yíyan Akoko kún un, rí i dájú pé àwọn ọja rẹ ti ní àwọn ìyípadà iye owó tí a ti ṣètò. Lọ sí **Ultimate Multisite > Àwọn Ọja**, ṣàtúnṣe ọja kan, kí o sì lọ sí taabu **Àwọn Ìyípadà Iye Owó** láti fi àwọn akoko ìdíyelé mìíràn kún un (fún àpẹẹrẹ Ọdọọdún pẹ̀lú iye owó ẹ̀dinwó).

![Taabu Àwọn Ìyípadà Iye Owó lórí ọja kan](/img/config/product-price-variations-tab.png)

#### Ìgbésẹ̀ 2: Fi Ààyè Yíyan Akoko Kún Fọ́ọ̀mù Ìsanwó Rẹ {#step-2-add-the-period-selection-field-to-your-checkout-form}

1. Lọ sí **Ultimate Multisite > Àwọn Fọ́ọ̀mù Ìsanwó** kí o sì ṣàtúnṣe fọ́ọ̀mù ìsanwó rẹ.

2. Yi lọ sísàlẹ̀ sí ìgbésẹ̀ tí ó ní ààyè **Tabili Iye Owó** rẹ, kí o sì tẹ **Fi Ààyè tuntun kún un**.

3. Nínú dialog yíyan irú ààyè, tẹ **Yan Akoko**.

![Dialog Fi Ààyè Tuntun Kún Un tó ń fi Yan Akoko hàn](/img/config/checkout-form-add-field-dialog.png)

4. Ṣètò àwọn àṣàyàn akoko. Àṣàyàn kọ̀ọ̀kan nílò:
   * **Ìpẹ́**: Nọ́mbà náà (fún àpẹẹrẹ `1`)
   * **Ẹyọ ìpẹ́**: Irú akoko náà (Àwọn Ọjọ́, Àwọn Ọ̀sẹ̀, Àwọn Oṣù, tàbí Àwọn Ọdún)
   * **Àkọlé**: Ọ̀rọ̀ tí àwọn oníbàárà yóò rí (fún àpẹẹrẹ "Oṣooṣù", "Ọdọọdún")

5. Tẹ **+ Fi Àṣàyàn Kún Un** láti fi àwọn yíyàn akoko míì kún un. Àwọn àṣàyàn wọ̀nyí gbọ́dọ̀ bá àwọn ìyípadà iye owó tí o ṣètò lórí àwọn ọja rẹ mu.

![Àwọn ètò ààyè Yíyan Akoko](/img/config/period-selection-field-settings.png)

6. Yan **Àdàkọ Olùyan Akoko** kan (Clean ni àiyípadà, èyí tó ń ṣe olùyan tí a ṣe aṣa rẹ̀ ní rọrùn, tó sì ṣetán fún CSS àkànṣe).

7. Tẹ **Fi Ààyè Pamọ́**.

#### Ìgbésẹ̀ 3: Gbe Ààyè Náà Sókè Tabili Iye Owó {#step-3-position-the-field-above-the-pricing-table}

Fún ìrírí aṣàmúlò tó dára jù, rí i dájú pé ààyè Yíyan Akoko hàn **ṣáájú** ààyè Tabili Iye Owó nínú ìgbésẹ̀ ìsanwó rẹ. O lè fà àwọn ààyè láti tún wọn tò nínú olootu fọ́ọ̀mù ìsanwó. Ní ọ̀nà yìí, àwọn oníbàárà á kọ́kọ́ yan akoko ìdíyelé, lẹ́yìn náà wọ́n á rí àwọn iye owó fún akoko yẹn.

![Olootu fọ́ọ̀mù ìsanwó tó ń fi ìtòlẹ́sẹẹsẹ ààyè hàn](/img/config/checkout-form-editor-with-fields.png)

#### Bí Ó Ṣe Ń Ṣiṣẹ́ lórí Frontend {#how-it-works-on-the-frontend}

Nígbà tí a bá ti ṣètò rẹ̀, àwọn oníbàárà tó bẹ̀ wò ojúewé ìforúkọsílẹ̀ rẹ yóò rí olùyan akoko lókè tabili iye owó. Nígbà tí wọ́n bá tẹ akoko ìdíyelé mìíràn:

  * Tabili iye owó yóò ṣe àfikún lẹ́sẹ̀kẹsẹ̀ láti fi àwọn iye owó fún akoko tí a yan hàn (kò sí ìtúnrù ojúewé tí a nílò).
  * Tí **Fi Àwọn Ìpẹ́ Tó Yàtọ̀ Múlẹ̀** kò bá ṣiṣẹ́ lórí ààyè Tabili Iye Owó, àwọn ọja tí kò ní ìyípadà iye owó fún akoko tí a yan ni a ó fi pamọ́.
  * Tí **Fi Àwọn Ìpẹ́ Tó Yàtọ̀ Múlẹ̀** bá ṣiṣẹ́, gbogbo ọja yóò wà ní híhàn bí wọ́n kò tilẹ̀ ní ìyípadà fún akoko tí a yan (wọn yóò fi iye owó àiyípadà wọn hàn).

#### Yíyan Akoko Ìdíyelé Ṣáájú nípasẹ̀ URL {#pre-selecting-a-billing-period-via-url}

O tún lè yan ọja àti akoko ìdíyelé ṣáájú nípasẹ̀ URL. Ultimate Multisite ṣe àtìlẹ́yìn fún àwọn àpẹẹrẹ URL wọ̀nyí:

  * `/register/premium` — Yan ọja "Premium" ṣáájú nìkan
  * `/register/premium/12` — Yan ọja náà àti ìpẹ́ oṣù 12 ṣáájú
  * `/register/premium/1/year` — Yan ọja náà ṣáájú pẹ̀lú ìpẹ́ ọdún 1

### Ààyè Yíyan Àdàkọ {#the-template-selection-field}

Ààyè **Yíyan Àdàkọ** jẹ́ kí àwọn oníbàárà yan àdàkọ saịtị nígbà ìsanwó. Ó ti wà nínú àwọn àdàkọ fọ́ọ̀mù ìsanwó **ìgbésẹ̀ kan** àti **ọ̀pọ̀-ìgbésẹ̀** ní àiyípadà báyìí, tí a fi kún Ultimate Multisite v2.6.1.

#### Fífi ààyè náà kún un lọ́nà ọwọ́ {#adding-the-field-manually}

Tí o bá ń ṣiṣẹ́ pẹ̀lú fọ́ọ̀mù tí a dá ṣáájú v2.6.1, tàbí tí o bẹ̀rẹ̀ láti àdàkọ òfo:

1. Lọ sí **Ultimate Multisite > Àwọn Fọ́ọ̀mù Ìsanwó** kí o sì ṣàtúnṣe fọ́ọ̀mù ìsanwó rẹ.
2. Nínú ìgbésẹ̀ tí a ti ń gba àwọn alaye saịtị, tẹ **Fi Ààyè tuntun kún un**.
3. Yan **Yíyan Àdàkọ** láti inú dialog irú ààyè.
4. Ṣètò ààyè náà:
   - **Àkọlé** — Àkọlé tí àwọn oníbàárà rí lókè grid àdàkọ (fún àpẹẹrẹ "Yan àdàkọ saịtị kan").
   - **Dandan** — Bóyá àwọn oníbàárà gbọ́dọ̀ yan àdàkọ kí wọ́n tó tẹ̀síwájú.

#### Bí ó ṣe ń ṣiṣẹ́ {#how-it-works}

Nígbà tí oníbàárà bá yan àdàkọ nígbà ìsanwó, Ultimate Multisite máa ń lò ó nígbà tí ó ń pèsè saịtị tuntun wọn. Àwọn àdàkọ tí a fi hàn wá láti inú àtòjọ **Àwọn Àdàkọ Saịtị** rẹ (**Ultimate Multisite > Àwọn Àdàkọ Saịtị**). Àwọn àdàkọ tí a samisi pé ó wà fún àwọn oníbàárà nìkan ni yóò hàn níbí.

### Àwọn domain ìpìlẹ̀ fọ́ọ̀mù ìsanwó {#checkout-form-base-domains}

Ultimate Multisite v2.13.0 ka àwọn domain tí a ṣètò lórí àwọn ààyè **URL Saịtị** fọ́ọ̀mù ìsanwó sí àwọn domain ìpìlẹ̀ nẹ́tíwọ́ọ̀kì. Lo àwọn ètò domain tó wà fún ààyè náà nígbà tí o bá fẹ́ kí àwọn oníbàárà dá àwọn saịtị sílẹ̀ lábẹ́ ọ̀kan tàbí ju bẹ́ẹ̀ lọ nínú àwọn domain ìforúkọsílẹ̀ tí a pín, bíi `example.com` àti `sites.example.com`.

A kì í ka àwọn domain ìpìlẹ̀ fọ́ọ̀mù ìsanwó pínpín sí àwọn ìtọ́kasí domain àdáni fún ojúlé kọ̀ọ̀kan. Nígbà tí oníbàárà bá dá ojúlé subdirectory sílẹ̀ lórí ọ̀kan lára àwọn ìpìlẹ̀ wọ̀nyẹn, Ultimate Multisite kì í dá àkọsílẹ̀ mapped-domain sílẹ̀ tí yóò mú kí host pínpín náà jẹ́ ti ojúlé kan ṣoṣo yẹn. Host pínpín náà máa ń wà fún àwọn ojúlé arákùnrin tí wọ́n ń lo ìpìlẹ̀ fọ́ọ̀mù ìsanwó kan náà.

Pa àwọn domain àdáni mọ́ fún àwọn host tí a tọ́ka fún oníbàárà kọ̀ọ̀kan, bíi `customer-example.com`. Pa àwọn domain ìpìlẹ̀ fọ́ọ̀mù ìsanwó mọ́ fún àwọn host ìforúkọsílẹ̀ pínpín tí ọ̀pọ̀ ojúlé lè lò.

#### Yíyọ ààyè náà kúrò {#removing-the-field}

Tí o kò bá ń fúnni ní àwọn tẹ́ńpílẹ́ẹ̀tì ojúlé, yọ ààyè Yíyan Tẹ́ńpílẹ́ẹ̀tì kúrò nínú fọ́ọ̀mù rẹ. Nígbà náà, àwọn oníbàárà yóò gba tẹ́ńpílẹ́ẹ̀tì àiyẹ̀wù èyíkéyìí tí a ti ṣètò lábẹ́ **Ultimate Multisite > Àwọn Ètò > Àwọn Tẹ́ńpílẹ́ẹ̀tì Ojúlé**.
