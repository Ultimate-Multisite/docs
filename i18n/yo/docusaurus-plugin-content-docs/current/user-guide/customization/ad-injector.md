# Fífi Ìpolówó sí nẹ́tíwọ́ọ̀kì rẹ pẹ̀lú Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector jẹ́ **add-on ọ̀fẹ́** tí ó fún ọ ní àwọn irinṣẹ́ láti fi ìpolówó sí àwọn plan rẹ tààrà láti ojúewé àtúnṣe wọn.

Lẹ́yìn fífi Ultimate Multisite: Ad Injector sílẹ̀, lọ sí ojúewé _Plugins_ rẹ láti mú un ṣiṣẹ́.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Tẹ bọ́tìnì Network Activate_

Nígbà tí add-on náà bá ti ṣiṣẹ́, àpótí meta _Advanced Options_ lórí ìbojú àtúnṣe plan yóò ní aṣàyàn tuntun tí a ń pè ní _Ad Settings_.

![Ad Settings lórí ìbojú àtúnṣe plan](/img/config/product-edit-full.png)

_Nísinsin yìí o lè ṣàtúnṣe àwọn ad settings fún ọkọọkan àwọn plan rẹ_

Lórí taabu yẹn, ìwọ yóò lè ṣàfikún àwọn code ìpolówó fún àwọn posts àti pages ní front-end, ṣáájú àti lẹ́yìn àkóónú.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Lẹ́yìn fífi code ìpolówó rẹ sílẹ̀, yóò hàn lórí àwọn posts àti pages ti plan pàtó yẹn._

O tún lè jẹ́ kí ìpolówó wà nínú header ti Admin Pages fún àwọn olumulo plan yẹn.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Lọ sí aṣàyàn Admin Panel Ad Code bí o bá fẹ́ ṣàfihàn ìpolówó lórí back-end olumulo rẹ_

### Ju Ìpolówó Lọ

O tún lè lo Ultimate Multisite: Ad Injector láti pèsè ìtìlẹ́yìn fún àwọn plan ipele-gíga.

Kan lọ sí aṣàyàn _Admin Panel Ads_ kí o sì ṣàfikún code HTML fún irinṣẹ́ ìtìlẹ́yìn rẹ.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Ṣàfikún ìtìlẹ́yìn oníbàárà sí àwọn plan pàtó_

Ní ọ̀nà yìí, àwọn ènìyàn lè dé ọ́ lọ́dọ̀ tààrà láti Admin Panel wọn!
