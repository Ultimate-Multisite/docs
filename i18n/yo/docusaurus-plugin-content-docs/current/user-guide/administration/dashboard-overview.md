# Akopọ Dashboard

Ultimate Multisite ní **Dashboard** tirẹ̀ lábẹ́ akojọ Ultimate Multisite, ó sì tún ń fi àwọn widgets kún Network Dashboard. Àpilẹ̀kọ yìí ṣàlàyé ohun tí ìwọ yóò rí ní ibi méjèèjì.

## Network Dashboard {#the-network-dashboard}

Ultimate Multisite ń fi àwọn widgets tó wúlò sínú dashboard ìpele nẹ́tíwọ́ọ̀kì tí ó wà ní **My Sites → Network Admin → Dashboard**.

![Network Dashboard pẹ̀lú àwọn widgets Ultimate Multisite](/img/admin/network-dashboard-full.png)

Èyí ni ohun tí widget kọ̀ọ̀kan ń pèsè:

**Àwọn Ìgbésẹ̀ Àkọ́kọ́:** Ó máa ń hàn lórí ìfìdásílẹ̀ tuntun, ó sì ń rán ọ létí láti parí àwọn iṣẹ́ ìṣètò bíi ṣíṣe ìṣètò ọ̀nà ìsanwó àti ṣíṣẹ̀dá account ìdánwò kan.

![Widget Àwọn Ìgbésẹ̀ Àkọ́kọ́](/img/admin/network-dashboard-first-steps.png)

**Akopọ:** Ó ń jabo iye àwọn ìforúkọsílẹ̀ àti owó-wíwọlé fún ọjọ́ náà. MRR (Monthly Recurring Revenue) ń tọ́ka sí àpapọ̀ tí a ṣe àfojúsùn kọjá àwọn oníbàárà pẹ̀lú memberships tó ń mú owó-wíwọlé wá.

![Widget Akopọ](/img/admin/network-dashboard-summary.png)

**Ìṣàn Ìṣẹ̀lẹ̀:** Ó ń jabo lórí àwọn ìṣẹ̀lẹ̀ káàkiri Ultimate Multisite — àwọn ìforúkọsílẹ̀, àwọn ìfagilé, àwọn ìṣẹ̀dá site, àwọn ìparẹ́, àti àwọn ìgbòkègbodò míì.

![Widget Ìṣàn Ìṣẹ̀lẹ̀](/img/admin/network-dashboard-activity-stream.png)

**Ní Báyìí:** Ó ń ṣàfihàn akopọ kúkúrú ti àwọn users àti sites nínú nẹ́tíwọ́ọ̀kì pẹ̀lú àwọn ọna abuja láti ṣẹ̀dá sites tàbí users tuntun.

![Widget Ní Báyìí](/img/admin/network-dashboard-right-now.png)

**Ìròyìn & Àwọn Ìjíròrò:** Ó ń mú wá, ó sì ń ṣàfihàn ìròyìn Ultimate Multisite tuntun jùlọ, pẹ̀lú àwọn imudójúìwọ̀n, àwọn àṣìṣe pàtàkì, àti àwọn àtúnṣe ààbò.

![Widget Ìròyìn](/img/admin/network-dashboard-news.png)

## Dashboard Ultimate Multisite {#the-ultimate-multisite-dashboard}

Nígbà tí network dashboard ń ṣàfihàn ìmọ̀ àpapọ̀ nípa nẹ́tíwọ́ọ̀kì, Dashboard Ultimate Multisite (ní ìpele òkè ti akojọ Ultimate Multisite) ń ṣàfihàn ìmọ̀ nípa iṣẹ́ náà.

![Dashboard Ultimate Multisite](/img/admin/um-dashboard-top.png)

**Ìdàgbàsókè Monthly Recurring Revenue:** Ó ń ṣàfihàn ìpínpín owó-wíwọlé oṣooṣù fún àkókò kàlẹ́ńdà oṣù 12. Owó-wíwọlé tuntun ni a fi buluu hàn, nígbà tí àwọn ìfagilé ń hàn ní pupa.

![Àwòrán MRR Growth](/img/admin/um-dashboard-mrr.png)

**Àtìlẹ́yìn Ìbùdó Ọjọ́:** Ṣe àlẹmọ data dashboard nípasẹ̀ àkókò láti dojú kọ ìmọ̀ tí o nílò. Ìbùdó ọjọ́ ló ń ṣàkóso ohun tí a fi hàn nínú àwọn widgets tó wà nísàlẹ̀.

![Ìbùdó ọjọ́ àti àwọn iṣirò dashboard](/img/admin/um-dashboard-date-range.png)

Àwọn widgets dashboard náà ní:

- **Owó-wíwọlé** — Àpapọ̀ owó-wíwọlé fún àkókò náà pẹ̀lú MRR àti ìpínpín gẹ́gẹ́ bí ọja kọ̀ọ̀kan
- **Memberships Tuntun** — Iye memberships tuntun tí a ṣẹ̀dá nínú àkókò tí a yàn
- **Àwọn Ìforúkọsílẹ̀ gẹ́gẹ́ bí Orílẹ̀-èdè** — Ìpínpín ilẹ̀-ayé ti àwọn ìforúkọsílẹ̀
- **Àwọn Ìforúkọsílẹ̀ gẹ́gẹ́ bí Fọ́ọ̀mù** — Iye àwọn oníbàárà tí wọ́n forúkọsílẹ̀ nípasẹ̀ checkout form kọ̀ọ̀kan ní ìfiwéra pẹ̀lú àwọn tí admins ṣàfikún
- **Àwọn Sites tí a Ṣèbẹ̀wò Sí Jùlọ** — Àwọn sites tí a ṣèbẹ̀wò sí jùlọ lórí nẹ́tíwọ́ọ̀kì rẹ (ó nílò kí o mu kíkà ìbẹ̀wò ṣiṣẹ́ ní **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**)

![Àwọn iṣirò dashboard](/img/admin/um-dashboard-stats.png)
