# Ṣíṣàbẹ̀wò Panel Admin

Ultimate Multisite fẹ́ jẹ́ rọrùn àti kedere bí ó ti ṣeé ṣe tó, ṣùgbọ́n pẹ̀lú irinṣẹ́ tó lágbára báyìí, ìrìnàjò ìtọ́nisọ́nà lè ràn ọ́ lọ́wọ́. Ẹ jẹ́ ká rìn kọjá àwọn ojúewé admin láti ràn ọ́ lọ́wọ́ láti mọ ibi tí o wà.

## Dashboard {#dashboard}

Ultimate Multisite **Dashboard** ń fi àwọn ìròyìn àti ìtúpalẹ̀ ìpìlẹ̀ hàn, pẹ̀lú owó tí wọlé, àwọn ìgbòkègbodò ojúlé, ìdàgbàsókè ọmọ ẹgbẹ́, iye àwọn àlejò, àti dátà àgbègbè.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

Èyí ni ìwòye kíkún ti dashboard:

![Ultimate Multisite Dashboard ojúewé kíkún](/img/admin/um-dashboard-full.png)

Àti apá ìsàlẹ̀ ti dashboard:

![Ultimate Multisite Dashboard apá ìsàlẹ̀](/img/admin/um-dashboard-bottom.png)

Èyí tún ni ìwòye kíkún ti dashboard nẹ́tíwọ́ọ̀kì:

![Network Dashboard ojúewé kíkún](/img/admin/network-dashboard-full-page.png)

## Àwọn Fọ́ọ̀mù Ìparí Rira {#checkout-forms}

**Àwọn Fọ́ọ̀mù Ìparí Rira** fún ọ ní ìrọ̀rùn láti dá àwọn ojúewé ìforúkọsílẹ̀ àdáni ṣẹ̀dá. O lè ṣe àwọn field ní àdáni, fi àwọn class àti script àdáni sínú, kí o sì dín ààyè wọlé kù dá lórí orílẹ̀-èdè tàbí ipò àlejò.

![Àtòkọ Àwọn Fọ́ọ̀mù Ìparí Rira](/img/admin/checkout-forms-list.png)

## Àwọn Ọjà {#products}

Apá **Àwọn Ọjà** ni ibi tí o ti ń dá àwọn plan, package, àti iṣẹ́ oriṣiriṣi ṣẹ̀dá fún nẹ́tíwọ́ọ̀kì rẹ. Ṣàlàyé àwọn owó, ìgbà ìsanwó, àti àwọn ààlà & quota ní ipele ọjà.

![Àtòkọ Àwọn Ọjà](/img/admin/products-list.png)

## Àwọn Ọmọ Ẹgbẹ́ {#memberships}

Ojúewé **Àwọn Ọmọ Ẹgbẹ́** ń fi gbogbo àwọn subscription hàn nínú nẹ́tíwọ́ọ̀kì rẹ. Ṣàyẹ̀wò kí o sì ṣàtúnṣe àwọn alaye subscription, pẹ̀lú àwọn plan, àwọn ọjà, iye àti ìgbà ìsanwó, ìtàn ìsanwó, àti timestamp.

![Àtòkọ Àwọn Ọmọ Ẹgbẹ́](/img/admin/memberships-list.png)

## Àwọn Ìsanwó {#payments}

Ojúewé **Àwọn Ìsanwó** ń pèsè ìwòye yára ti ìtàn ìsanwó káàkiri gbogbo nẹ́tíwọ́ọ̀kì rẹ, pẹ̀lú alaye transaction ní kíkún, títí kan àwọn ọjà pàtó àti iye owó.

![Àtòkọ Àwọn Ìsanwó](/img/admin/payments-list.png)

## Àwọn Oníbàárà {#customers}

Ojúewé **Àwọn Oníbàárà** ń fi gbogbo àwọn ọmọ ẹgbẹ́ tí a forúkọsílẹ̀ nínú nẹ́tíwọ́ọ̀kì rẹ hàn, pẹ̀lú alaye olumulo ìpìlẹ̀ bíi orúkọ, àdírẹ́sì email, ìwọlé tó kẹ́yìn, àti link "yí padà sí" fún wíwọlé sínú ojúlé abẹ́ wọn.

![Àtòkọ Àwọn Oníbàárà](/img/admin/customers-list.png)

## Àwọn Ojúlé {#sites}

Ojúewé **Àwọn Ojúlé** ń to gbogbo àwọn ojúlé abẹ́ nínú nẹ́tíwọ́ọ̀kì rẹ, pẹ̀lú àwọn template ojúlé àti àwọn ojúlé tí oníbàárà ní. Ṣàkóso àwọn domain tí a mapped, àwọn ààlà & quota, àwọn àfikún àti àkòrí, àti àwọn thumbnail ojúlé ní rọrùn.

![Àtòkọ Àwọn Ojúlé](/img/admin/sites-list.png)

## Àwọn Domain {#domains}

Ojúewé **Àwọn Domain** jẹ́ fún àwọn domain àdáni tí a mapped sí àwọn ojúlé abẹ́. Gẹ́gẹ́ bí super admin, o lè fi ọwọ́ ṣàfikún tàbí map àwọn orúkọ domain àdáni, kí o sì wo àwọn DNS record àti log.

![Àtòkọ Àwọn Domain](/img/admin/domains-list.png)

## Àwọn Kóòdù Ẹdinwó {#discount-codes}

**Àwọn Kóòdù Ẹdinwó** jẹ́ kí o dá àwọn kóòdù coupon ṣẹ̀dá láti fún àwọn olumulo ìkẹyìn rẹ ní ẹdinwó. Ṣàtọ́ka iye ẹdinwó àti àwọn olumulo tàbí ẹgbẹ́ pàtó tí ó kan.

![Àtòkọ Àwọn Kóòdù Ẹdinwó](/img/admin/discount-codes-list.png)

## Àwọn Ìkéde {#broadcasts}

**Àwọn Ìkéde** jẹ́ irinṣẹ́ fún títú àwọn ìkéde jáde tàbí fífi àwọn ìfiranṣẹ́ aládàáni ránṣẹ́ sí àwọn olumulo ojúlé abẹ́ rẹ. Ṣe àfọ́kànsí sí àwọn ẹgbẹ́ olumulo pàtó dá lórí plan tàbí ọjà tí wọ́n ṣe alabapin sí.

![Àtòkọ Àwọn Ìkéde](/img/admin/broadcasts-list.png)

## Ètò {#settings}

Ojúewé **Ètò** ni ibi tí o ti ń ṣètò Ultimate Multisite — àwọn ètò ìforúkọsílẹ̀, àwọn ìsanwó, API àti webhooks, domain mapping, àti àwọn integration míì.

Àtòkọ ètò connector AI ń fi àwọn pool olupèsè OAuth tí a ń ṣe àtìlẹ́yìn fún lọ́wọ́lọ́wọ́ hàn: Anthropic Max, OpenAI ChatGPT/Codex, àti Google AI Pro. Card olupèsè kọ̀ọ̀kan jẹ́ kí àwọn super admin so accounts pọ̀, refresh àwọn accounts tí a fipamọ́, yọ accounts kúrò nípasẹ̀ email, kí wọ́n sì lo manual OAuth fallback nígbà tí àyíká sandboxed bá dí browser redirect. A ti yọ àwọn àṣàyàn ìṣètò Cursor Pro kúrò nínú panel admin.

Àwọn accounts ChatGPT/Codex ń ṣe àtìlẹ́yìn fún lílo irinṣẹ́ tí connector ń ṣe àtìlẹ́yìn fún níbi tí iṣẹ́ náà ti gba àwọn irinṣẹ́ láàyè, nítorí náà àwọn workflow admin tí ó gbẹ́kẹ̀lé àwọn iṣẹ́ tí connector ṣe àtìlẹ́yìn fún lè lo ìhùwàsí irinṣẹ́ Codex lẹ́yìn tí OpenAI account bá ti sopọ̀.

## Àwọn Ìṣẹ̀lẹ̀ {#events}

Ojúewé **Àwọn Ìṣẹ̀lẹ̀** ń tọ́jú àkọsílẹ̀ gbogbo àwọn ìṣẹ̀lẹ̀ àti log nínú nẹ́tíwọ́ọ̀kì rẹ. Ó ń tọpinpin àwọn ìgbòkègbodò bíi àwọn ìyípadà plan, àwọn ìforúkọsílẹ̀, àti àwọn ìṣipopada míì — ó wúlò fún mímójútó multisite rẹ.

![Àtòkọ Àwọn Ìṣẹ̀lẹ̀](/img/admin/events-list.png)

## Webhooks {#webhooks}

**Webhooks** jẹ́ kí o fi dátà ránṣẹ́ sí application mìíràn. Ó wúlò fún fífi dátà ránṣẹ́ láti Ultimate Multisite sí àwọn platform bíi Zapier.

![Àtòkọ Webhooks](/img/admin/webhooks-list.png)
