# Ṣíṣàkóso Àwọn Oníbàárà {#managing-customers}

Ojú-ewé **Customers** ń fi àtòjọ àwọn ọmọ ẹgbẹ́ tí a forúkọ sílẹ̀ nínú nẹ́tíwọ́ọ̀kì rẹ hàn. Ojú-ewé yìí ń ṣàfihàn ìwífún ìpìlẹ̀ ti aṣàmúlò, pẹ̀lú orúkọ, àdírẹ́sì ímeèlì, ìwọlé tó kẹ́yìn, àti ìjápọ̀ **switch to** tí ó jẹ́ kí o wọlé sí subsite aṣàmúlò gẹ́gẹ́ bí alábòójútó site.

![Ojú-ewé àtòjọ oníbàárà](/img/admin/customers-list.png)

Tẹ **Manage** ní ìsàlẹ̀ káàdì oníbàárà kan láti rí ìwífún kíkún nípa oníbàárà yẹn.

Lórí ojú-ewé àlàyé oníbàárà, bulọọki **At a Glance** ń fi ID oníbàárà, àsìkò ìwọlé tó kẹ́yìn, àti àpapọ̀ iye owó tí oníbàárà yìí ti ná hàn.

Bulọọki **Customer Options** jẹ́ kí o ṣètò oníbàárà yìí gẹ́gẹ́ bí **VIP** (tag kan láti jẹ́ kí ó rọrùn láti rí wọn). Taabu **Billing Info** ń fi ìwífún ìsanwó hàn bí ímeèlì, àdírẹ́sì, àti orílẹ̀-èdè.

Bulọọki **Fields** ń fi orúkọ aṣàmúlò oníbàárà, ímeèlì hàn, ó sì jẹ́ kí o yí ipò **Email Verification** padà.

Ní ìsàlẹ̀ bulọọki Customer Options, ìwọ yóò rí àwọn bulọọki àfikún mẹ́rin:

- **Memberships** — Àwọn subscription lọwọlọwọ tí oníbàárà ti forúkọ sí
- **Payments** — Ìtàn ìsanwó oníbàárà
- **Sites** — Àwọn site tí oníbàárà yìí ní
- **Events** — Àwọn ìṣẹ̀lẹ̀ tó ní í ṣe pẹ̀lú oníbàárà yìí (ìmúdájú ímeèlì, àwọn ìyípadà ìdíyelé, bẹ́ẹ̀ bẹ́ẹ̀ lọ)

Tẹ aami ọfà (→) lẹ́gbẹ̀ẹ́ ohun èyíkéyìí láti lọ sí ojú-ewé àlàyé rẹ. Fún àpẹẹrẹ, títẹ ọfà lẹ́gbẹ̀ẹ́ site kan máa mú ọ lọ sí ojú-ewé àtúnṣe site níbi tí o ti lè ṣàtúnṣe ìbáṣepọ̀ membership, irú site, àti ipò ìṣiṣẹ́.

Ohun kan náà ló kan memberships àti payments — ọ̀kọ̀ọ̀kan ní ìjápọ̀ sí àwọn ojú-ewé àlàyé níbi tí o ti lè ṣàtúnṣe ipò membership, oníbàárà tó ní ìbáṣepọ̀, àwọn product, àwọn ètò payment gateway, àti tún ṣe refunds.

Ní ẹ̀gbẹ́ ọ̀tún ojú-ewé oníbàárà, ìwọ yóò rí:

- **Last Login & IPs** — Àsìkò ìwọlé tó kẹ́yìn, àdírẹ́sì IP, àti orílẹ̀-èdè
- **Timestamps** — Nígbà tí Account oníbàárà dá sílẹ̀ àti nígbà tí a ṣe àtúnṣe sí i kẹ́yìn
- **Delete** — Àṣàyàn láti pa oníbàárà rẹ́ pátápátá

:::warning
Pípa oníbàárà rẹ́ jẹ́ ohun títí láé, kò sì lè ṣeé yí padà.
:::
