# Lílóye Ultimate Multisite Dashboard {#understanding-the-ultimate-multisite-dashboard}

Kí a tó rì sínú àwọn aṣàyàn ìṣètò tó ti ni ìlọsíwájú síi àti àwọn apá iṣẹ́ ti Ultimate Multisite, ẹ jẹ́ ká wo dashboard náà.

Gẹ́gẹ́ bí ọ̀pọ̀ WordPress plugins, Ultimate Multisite dá àkọsílẹ̀ menu ìpele nẹ́tíwọ́ọ̀kì kan sílẹ̀ tí a pè ní **Ultimate Multisite** pẹ̀lú ọ̀pọ̀ àkọsílẹ̀ sub-menu ní abẹ́ rẹ̀.

## Àwọn Widget Dashboard Nẹ́tíwọ́ọ̀kì {#network-dashboard-widgets}

Ultimate Multisite fi díẹ̀ nínú àwọn widget tó wúlò sínú dashboard ìpele nẹ́tíwọ́ọ̀kì. A rí dashboard yìí nípa lílọ sí **My Sites → Network Admin → Dashboard**. A ṣe àwọn widget náà láti pèsè ìráyè rọrùn sí ìwífún àti àwọn iṣẹ́ tí wọ́n wọ́pọ̀ fún alábòójútó nẹ́tíwọ́ọ̀kì.

![Dashboard Nẹ́tíwọ́ọ̀kì pẹ̀lú àwọn widget Ultimate Multisite](/img/admin/network-dashboard-full.png)

### Àwọn Ìgbésẹ̀ Àkọ́kọ́ {#first-steps}

![Widget Àwọn Ìgbésẹ̀ Àkọ́kọ́](/img/admin/network-dashboard-first-steps.png)

Widget yìí máa ń hàn lórí ìfìdásílẹ̀ tuntun ti Ultimate Multisite, ó sì máa ń rántí alábòójútó nẹ́tíwọ́ọ̀kì láti parí àwọn iṣẹ́ àfikún bí ṣíṣe ìṣètò ọ̀nà ìsanwó àti dídá account ìdánwò sílẹ̀.

### Àkótán {#summary}

![Widget Àkótán](/img/admin/network-dashboard-summary.png)

Widget Àkótán náà ń jábọ̀ lórí iye àwọn ìforúkọsílẹ̀ àti owó wọlé wọn fún ọjọ́ náà. MRR (Monthly Recurring Revenue) fi àpapọ̀ tí a ṣe àfojúsùn hàn láàárín àwọn alabara tí wọ́n ní membership tó ń mú owó wọlé.

### Ìṣàn Ìṣe {#activity-stream}

![Widget Ìṣàn Ìṣe](/img/admin/network-dashboard-activity-stream.png)

Widget yìí ń jábọ̀ lórí àwọn ìṣẹ̀lẹ̀ káàkiri Ultimate Multisite. Àwọn ìṣẹ̀lẹ̀ náà ní ìforúkọsílẹ̀, fífi iṣẹ́ sílẹ̀, dídá site sílẹ̀, píparẹ́, àti àwọn ìṣe míì.

Jọ̀wọ́ tọ́ka sí apá Àwọn Ìṣẹ̀lẹ̀ nínú ìwé ìtọ́nisọ́nà fún àpèjúwe tó kúnrẹ́rẹ́ síi.

### Ní Báyìí {#right-now}

![Widget Ní Báyìí](/img/admin/network-dashboard-right-now.png)

Widget irinṣẹ́ yìí ń ṣàfihàn àkótán kúkúrú ti àwọn aṣàmúlò àti sites nínú nẹ́tíwọ́ọ̀kì. Àwọn ọna abuja tó rọrùn ní òkè gba ọ́ láyè láti wọlé pẹ̀lú ìtẹ̀ kan ṣoṣo láti dá sites tàbí aṣàmúlò tuntun sílẹ̀.

### Ìròyìn àti Ìjíròrò {#news-and-discussions}

![Widget Ìròyìn àti Ìjíròrò](/img/admin/network-dashboard-news.png)

Widget yìí ń gba ó sì ń ṣàfihàn ìròyìn tuntun jùlọ ti Ultimate Multisite. Máa ṣọ́ widget yìí fún ìwífún nípa àwọn ìmúdójúìwọ̀n, àwọn àṣìṣe pàtàkì, àti àwọn àtúnṣe ààbò.

## Ultimate Multisite Dashboard {#ultimate-multisite-dashboard}

Níbi tí dashboard ìpele nẹ́tíwọ́ọ̀kì ti ń fi ìwífún gbogbo-gbòò nípa nẹ́tíwọ́ọ̀kì hàn, Ultimate Multisite dashboard (tó wà ní ìpele òkè menu Ultimate Multisite) ń fi ìwífún nípa iṣẹ́ náà hàn.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

### Ìdàgbàsókè Monthly Recurring Revenue {#monthly-recurring-revenue-growth}

Àtẹ MRR (Monthly Recurring Revenue) ń ṣàfihàn pípín owó wọlé oṣooṣù lórí àkókò kálẹ́ńdà oṣù 12.

![Àtẹ Ìdàgbàsókè MRR](/img/admin/um-dashboard-mrr.png)

Owó wọlé tuntun ni a ń tọpa ní àwọ̀ buluu nígbà tí fífi iṣẹ́ sílẹ̀ hàn ní àwọ̀ pupa.

### Àtìlẹ́yìn Ààlà Ọjọ́ {#date-range-support}

Da lórí iye àwọn alabara àti ìṣe, àwọn ìṣirò lè di púpọ̀ ju. Àtìlẹ́yìn ààlà ọjọ́ tí a kọ sínú dashboard gba alábòójútó nẹ́tíwọ́ọ̀kì láyè láti dojú kọ ìwífún àti àkókò tó nílò nìkan.

![Olùyan ààlà ọjọ́ àti stats](/img/admin/um-dashboard-date-range.png)

Ààlà ọjọ́ náà ń ṣàkóso àkókò ìwífún tí àwọn widget tó wà lẹ́sẹ̀kẹsẹ̀ ní ìsàlẹ̀ ń ṣàfihàn, pẹ̀lú owó wọlé, àwọn membership tuntun, ìforúkọsílẹ̀ nípa orílẹ̀-èdè, ìforúkọsílẹ̀ nípa fọọmu, àti sites tí a bẹ̀ wò jùlọ.

![Àwọn widget ìṣirò Dashboard](/img/admin/um-dashboard-stats.png)

:::tip
Láti mú kí widget **Sites Tí A Bẹ̀ Wò Jùlọ** ṣiṣẹ́, jẹ́ kí kíkà ìbẹ̀wò ṣiṣẹ́ ní **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**.
:::
