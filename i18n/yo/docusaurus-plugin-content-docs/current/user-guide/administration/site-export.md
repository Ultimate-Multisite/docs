# Gbigbe Oju-ìlé Jáde {#site-export}

Ojú-ewé abojútó **Gbigbe Oju-ìlé Jáde** jẹ́ kí àwọn alábòójútó nẹ́tíwọ́ọ̀kì kó oju-ìlé kan, tàbí gbogbo nẹ́tíwọ́ọ̀kì, sínú àkójọpọ̀ fáìlì tí a lè ṣe igbasilẹ fún ìṣílọ, àfipamọ́, tàbí iṣẹ́ fífi lé ẹlòmíràn lọ́wọ́.

## Gbigbe oju-ìlé kan jáde {#exporting-one-site}

Lọ sí **Ultimate Multisite > Gbigbe Oju-ìlé Jáde** kí o sì yan **Ṣẹ̀dá Gbigbe Oju-ìlé Jáde tuntun**. Yan oju-ìlé abẹ́ tí o fẹ́ gbe jáde, lẹ́yìn náà yan bóyá àkójọpọ̀ fáìlì náà yẹ kí ó ní àwọn ìkójọpọ̀ tí a gbé sókè, àwọn àfikún, àti àwọn àpẹrẹ nínú.

Nígbà tí gbigbé jáde bá parí, ṣe igbasilẹ ZIP láti inú àtòkọ **Àwọn Gbigbé Jáde Tí Ó Wà**. Àwọn ZIP gbigbé jáde ní `index.php` tí ó lè bẹ̀rẹ̀ fúnra rẹ̀ àti `readme.txt` báyìí, nítorí náà a lè gbe àkójọpọ̀ fáìlì náà sí host tuntun kí a sì bẹ̀rẹ̀ láì kọ́kọ́ fi plugin amugbewọlé míràn sílẹ̀.

## Gbigbe gbogbo nẹ́tíwọ́ọ̀kì jáde {#exporting-the-whole-network}

Lo **Gbigbe Nẹ́tíwọ́ọ̀kì Jáde** lórí ojú-ewé Gbigbe Oju-ìlé Jáde nígbà tí o bá nílò àkójọpọ̀ fáìlì kan ṣoṣo tí ó ní gbogbo àwọn oju-ìlé abẹ́ nínú nẹ́tíwọ́ọ̀kì. Èyí wúlò ṣáájú àwọn ìṣílọ host, àwọn ìdánwò ìmúpadàbọ̀ láti ìjàmbá, tàbí àwọn àtúnkọ́ staging níbi tí oju-ìlé abẹ́ kọ̀ọ̀kan ti nílò láti rìn papọ̀.

Nítorí gbigbé nẹ́tíwọ́ọ̀kì jáde lè tóbi ju gbigbé oju-ìlé kan jáde lọ, ṣiṣẹ́ rẹ̀ ní àsìkò tí ìrìnàjò kéré, kí o sì jẹ́rìí pé ibi ìpamọ́ ìfojúsùn ní ààyè ọ̀fẹ́ tó fún àwọn ìkójọpọ̀ tí a gbé sókè, àwọn àfikún, àwọn àpẹrẹ, àti àwọn fáìlì ZIP tí a ṣẹ̀dá.

### Àwọn Àkójọpọ̀ Amugbewọlé Nẹ́tíwọ́ọ̀kì {#network-import-bundles}

Bẹ̀rẹ̀ pẹ̀lú Ultimate Multisite 2.12.0, Olùgbé Oju-ìlé Jáde lè ṣẹ̀dá **àwọn àkójọpọ̀ amugbewọlé nẹ́tíwọ́ọ̀kì** — àwọn àkójọpọ̀ fáìlì amọ̀ja tí a ṣe fún ìmúpadàbọ̀ oju-ìlé káàkiri nẹ́tíwọ́ọ̀kì ní ọ̀nà tó rọrùn. Àkójọpọ̀ amugbewọlé nẹ́tíwọ́ọ̀kì ní gbogbo àwọn fáìlì àti metadata tó ṣe pàtàkì láti mú ọ̀pọ̀ oju-ìlé padà sí fifi nẹ́tíwọ́ọ̀kì tuntun sílẹ̀.

#### Ṣíṣẹ̀dá Àkójọpọ̀ Amugbewọlé Nẹ́tíwọ́ọ̀kì {#generating-a-network-import-bundle}

1. Lọ sí **Ultimate Multisite > Gbigbe Oju-ìlé Jáde**
2. Tẹ **Ṣẹ̀dá Gbigbe Nẹ́tíwọ́ọ̀kì Jáde tuntun**
3. Yan **Àkójọpọ̀ Amugbewọlé Nẹ́tíwọ́ọ̀kì** gẹ́gẹ́ bí irú gbigbé jáde
4. Yan bóyá kí o fi àwọn ìkójọpọ̀ tí a gbé sókè, àwọn àfikún, àti àwọn àpẹrẹ kún un
5. Tẹ **Ṣẹ̀dá**
6. Ṣe igbasilẹ ZIP àkójọpọ̀ náà láti inú àtòkọ **Àwọn Gbigbé Jáde Tí Ó Wà**

#### Mímúpadàbọ̀ láti inú Àkójọpọ̀ Amugbewọlé Nẹ́tíwọ́ọ̀kì {#restoring-from-a-network-import-bundle}

Láti mú àwọn oju-ìlé padà láti inú àkójọpọ̀ amugbewọlé nẹ́tíwọ́ọ̀kì:

1. Fi Ultimate Multisite sí host ìfojúsùn rẹ
2. Parí wizard ìṣètò multisite
3. Lọ sí **Ultimate Multisite > Gbigbe Oju-ìlé Jáde** lórí nẹ́tíwọ́ọ̀kì tuntun náà
4. Tẹ **Gbe Àkójọpọ̀ Nẹ́tíwọ́ọ̀kì Wọlé**
5. Gbe fáìlì ZIP àkójọpọ̀ amugbewọlé nẹ́tíwọ́ọ̀kì sókè
6. Tẹ̀lé àwọn ìtọ́nisọ́nà amugbewọlé lórí-ìbojú
7. Ìlànà amugbewọlé náà yóò mú gbogbo àwọn oju-ìlé, àkóónú wọn, àti àwọn ìṣètò wọn padà

Àwọn àkójọpọ̀ amugbewọlé nẹ́tíwọ́ọ̀kì máa ń pa:
- Àkóónú oju-ìlé (àwọn àtẹ̀jáde, àwọn ojú-ewé, àwọn irú àtẹ̀jáde àdáṣe)
- Àwọn ìṣètò àti àwọn aṣayan oju-ìlé
- Àwọn ipa àti àwọn àṣẹ olùmúlò
- Àwọn àfikún àti àwọn àpẹrẹ (bí wọ́n bá wà nínú àkójọpọ̀ náà)
- Àwọn ìkójọpọ̀ media tí a gbé sókè (bí wọ́n bá wà nínú àkójọpọ̀ náà)
- Àwọn tábìlì database àdáṣe àti data

## Fífi ZIP gbigbé jáde tí ó lè bẹ̀rẹ̀ fúnra rẹ̀ sílẹ̀ {#installing-a-self-booting-export-zip}

Láti mú ZIP tí ó lè bẹ̀rẹ̀ fúnra rẹ̀ padà lórí host tuntun:

1. Gbe àkóónú ZIP tí a gbé jáde sí gbòǹgbò wẹ́ẹ̀bù ìfojúsùn.
2. Ṣí `index.php` tí a gbé sókè nínú browser.
3. Tẹ̀lé àwọn ìtọ́nisọ́nà olùfìṣàgbékalẹ̀ lórí-ìbojú láti inú package gbigbé jáde tí a so pọ̀.
4. Ṣàyẹ̀wò `readme.txt` tí a so pọ̀ fún àwọn àkíyèsí pàtó sí gbigbé jáde ṣáájú kí o tó yọ àwọn fáìlì ìgbà díẹ̀ kúrò.

Fún fifi sílẹ̀ àti àwọn àlàyé amugbewọlé pàtó sí àfikún, wo [ìwé àkọsílẹ̀ àfikún Olùgbé Oju-ìlé Jáde](/addons/site-exporter).

Fún irinṣẹ́ oju-ìlé kan ṣoṣo tí a fi kún un nínú Ultimate Multisite 2.9.0, wo [Gbigbe Jáde & Gbigbe Wọlé](/user-guide/administration/export-import).
