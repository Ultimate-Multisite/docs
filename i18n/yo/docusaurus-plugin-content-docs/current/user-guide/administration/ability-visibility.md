# Hihan Agbara

Superdav AI Agent v1.12.0 ṣafihan awọn iṣakoso **Hihan Agbara** ti o n ṣakoso iru awọn oju-ilẹ wo ni o fi agbara kọọkan han. Eyi jẹ ki awọn alakoso le ṣatunṣe pẹkipẹki iru awọn agbara agent wo ni o wa nipasẹ REST APIs, MCP servers, awọn iṣepọ WooCommerce, ati awọn atọkun miiran.

## Kini Hihan Agbara?

Hihan Agbara jẹ eto igbanilaaye ti o n ṣakoso:

- **Iru awọn agbara wo** ni o wa fun awọn agent
- **Ibi ti wọn ti han** (REST API, MCP, WooCommerce, abbl.)
- **Tani o le wọle si wọn** (nipasẹ awọn atokọ-igbanilaaye alabaṣepọ)
- **Bii a ṣe pin wọn si ẹka** (ti a mọ vs. ti a ko pin si ẹka)

Agbara kọọkan ni ipele hihan kan ti o pinnu wiwa rẹ kọja awọn oju-ilẹ oriṣiriṣi.

## Awọn Ipele Hihan

### Gbangba

**Awọn agbara Gbangba** wa nibi gbogbo:

- Awọn endpoint REST API
- MCP servers
- Awọn iṣepọ WooCommerce
- Awọn atọkun iwiregbe
- Gbogbo awọn ipa olumulo (pẹlu awọn igbanilaaye to yẹ)

Apeere: `scaffold-block-theme`, `activate-theme`, `send-email`

### Inu

**Awọn agbara Inu** wa nikan laarin fifi sori WordPress rẹ:

- Awọn atọkun iwiregbe
- Awọn panel alakoso
- Awọn olumulo ti o wọle nikan
- Ko han nipasẹ REST API tabi awọn iṣepọ ita

Apeere: `manage-settings`, `view-analytics`, `export-data`

### Alabaṣepọ

**Awọn agbara Alabaṣepọ** wa nikan fun awọn alabaṣepọ ti a fi si atokọ funfun:

- Nilo titẹsi atokọ-igbanilaaye alabaṣepọ kedere
- Wa nipasẹ REST API pẹlu awọn iwe-ẹri alabaṣepọ
- Wa nipasẹ MCP pẹlu ijẹrisi alabaṣepọ
- Ko wa fun awọn olumulo deede

Apeere: `bulk-import-users`, `modify-billing`, `access-analytics`

### Ti pa

**Awọn agbara Ti pa** ko wa nibikibi:

- Ko han nipasẹ oju-ilẹ kankan
- Ko wa ninu iwiregbe
- Ko wa ninu awọn panel alakoso
- Wulo fun awọn agbara ti a ti dawọ duro tabi ti adanwo

## Ṣiṣakoso Hihan Agbara

### Wiwọle si Awọn Eto Hihan Agbara

1. Lọ si **WordPress Admin** → **Superdav AI Agent** → **Settings**
2. Tẹ taabu **Awọn agbara**
3. Iwọ yoo ri atokọ gbogbo awọn agbara ti a ti fi sori ẹrọ pẹlu awọn ipele hihan wọn

### Wiwo Awọn Alaye Agbara

Tẹ lori agbara eyikeyi lati ri:

- **Orukọ**: Olùdánimọ̀ agbara naa
- **Apejuwe**: Ohun ti agbara naa n ṣe
- **Hihan lọwọlọwọ**: Gbangba, Inu, Alabaṣepọ, tabi Ti pa
- **Atokọ-igbanilaaye alabaṣepọ**: Iru awọn alabaṣepọ wo ni o le wọle si (ti hihan Alabaṣepọ ba jẹ)
- **Imudojuiwọn ikẹhin**: Nigba ti a yipada hihan kẹhin
- **Ipo**: Ti a mọ tabi Ti a ko pin si ẹka

### Yiyipada Awọn Ipele Hihan

Lati yi hihan agbara kan pada:

1. Tẹ agbara naa ninu atokọ
2. Yan ipele hihan tuntun lati dropdown
3. Ti o ba n yan "Alabaṣepọ", ṣafikun awọn olùdánimọ̀ alabaṣepọ si atokọ-igbanilaaye
4. Tẹ **Fipamọ**

Apeere:

```
Ability: bulk-import-users
Current visibility: Public
Change to: Partner
Partner allow-list: partner-123, partner-456
```

### Awọn Iṣẹ Ọpọ

Lati yi hihan pada fun awọn agbara pupọ:

1. Ṣayẹwo awọn apoti lẹgbẹẹ awọn agbara naa
2. Yan ipele hihan tuntun lati dropdown iṣẹ ọpọ
3. Tẹ **Waye**

## Atokọ-Igbanilaaye Alabaṣepọ

**Atokọ-igbanilaaye alabaṣepọ** n ṣakoso iru awọn alabaṣepọ ita wo ni o le wọle si awọn agbara ipele Alabaṣepọ.

### Ṣafikun Awọn Alabaṣepọ

1. Lọ si **Superdav AI Agent** → **Settings** → **Awọn Alabaṣepọ**
2. Tẹ **Ṣafikun Alabaṣepọ**
3. Tẹ olùdánimọ̀ alabaṣepọ sii (nigbagbogbo API key tabi ID ajo)
4. Ni yiyan, ṣafikun orukọ alabaṣepọ ati apejuwe
5. Tẹ **Fipamọ**

### Fifi Awọn Agbara si Awọn Alabaṣepọ

Lẹhin fifi alabaṣepọ kun:

1. Lọ si taabu **Awọn agbara**
2. Wa agbara kan pẹlu hihan Alabaṣepọ
3. Tẹ lati ṣatunkọ
4. Ninu abala "Atokọ-igbanilaaye alabaṣepọ", ṣayẹwo awọn apoti fun awọn alabaṣepọ ti o yẹ ki wọn ni wiwọle
5. Tẹ **Fipamọ**

### Yiyọ Wiwọle Alabaṣepọ Kuro

Lati yọ wiwọle alabaṣepọ kuro:

1. Ṣatunkọ agbara naa
2. Yọ ami ayẹwo kuro ninu apoti alabaṣepọ ninu atokọ-igbanilaaye
3. Tẹ **Fipamọ**

Alabaṣepọ naa yoo padanu wiwọle si agbara yẹn lẹsẹkẹsẹ.

## Awọn Agbara Ti a Ko Pin si Ẹka

Nigbati o ba fi agbara ẹni-kẹta sori ẹrọ ti Superdav AI Agent ko mọ, a samisi rẹ bi **Ti a ko pin si ẹka**.

### Ifitonileti Alakoso fun Awọn Agbara Ti a Ko Pin si Ẹka

Iwọ yoo ri ifitonileti alakoso kan:

```
⚠️ Unclassified Abilities Detected

The following abilities were installed but are not recognized:
- custom-import-tool
- external-api-wrapper

These abilities may pose security or compatibility risks.
Please review and classify them.

[Review Abilities] [Dismiss]
```

### Atunyẹwo Awọn Agbara Ti a Ko Pin si Ẹka

1. Tẹ **Ṣayẹwo Awọn Agbara** ninu ifitonileti naa
2. Fun agbara kọọkan ti a ko pin si ẹka, iwọ yoo ri:
   - Orukọ agbara ati apejuwe
   - Orisun (plugin/addon wo ni o fi sii)
   - Ipele hihan ti a daba
   - Ayẹwo aabo

3. Yan ipele hihan kan:
   - **Gbangba**: Gbẹkẹle agbara naa; fi han nibi gbogbo
   - **Inu**: Fi opin si lilo inu nikan
   - **Alabaṣepọ**: Fi opin si awọn alabaṣepọ kan pato
   - **Ti pa**: Maṣe lo agbara yii

4. Tẹ **Pin si ẹka** lati fipamọ ipinnu rẹ

### Kí Ló Dé Tí A Fi Yẹ Kí A Pin Awọn Agbara Ti a Ko Pin si Ẹka Si Ẹka?

Pipin awọn agbara si ẹka:

- **Mu aabo dara si**: O fọwọsi kedere ohun ti a fi han
- **Dena awọn ijamba**: Awọn agbara ti a ko pin si ẹka ti wa ni pipa nipa aiyipada
- **Mu awọn ẹya ṣiṣẹ**: Ni kete ti a ba pin si ẹka, agbara naa di wiwa
- **Ṣe igbasilẹ awọn ipinnu**: A kọ awọn yiyan rẹ sinu log fun awọn idi audit

## Hihan ninu Awọn Oju-ilẹ Oriṣiriṣi

### REST API

**Awọn agbara Gbangba** wa nipasẹ awọn endpoint REST:

```bash
curl -X POST https://yoursite.com/wp-json/superdav/v1/abilities/scaffold-block-theme \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{"theme_name": "My Theme"}'
```

**Awọn agbara Inu ati Alabaṣepọ** ko wa nipasẹ REST API.

### MCP Servers

**Awọn agbara Gbangba** wa nipasẹ MCP:

```
MCP Server: Superdav AI Agent
Available abilities:
- scaffold-block-theme
- activate-theme
- send-email
```

**Awọn agbara Inu** ko han nipasẹ MCP.

**Awọn agbara Alabaṣepọ** wa nikan pẹlu awọn iwe-ẹri alabaṣepọ.

### Iṣepọ WooCommerce

**Awọn agbara Gbangba** ti o ni ibatan si WooCommerce wa:

- Ṣiṣakoso ọja
- Ṣiṣe aṣẹ
- Ibaraẹnisọrọ alabara

**Awọn agbara Inu** ko han si WooCommerce.

### Atọkun Iwiregbe

**Gbogbo awọn agbara** (Gbangba, Inu, Alabaṣepọ) wa ninu iwiregbe, da lori awọn igbanilaaye olumulo:

- Awọn alakoso ri gbogbo awọn agbara
- Awọn olumulo deede ri awọn agbara Gbangba nikan
- Awọn alabaṣepọ ri Gbangba + awọn agbara Alabaṣepọ (ti wọn ba wa ninu atokọ funfun)

## Awọn Iṣe Aabo Ti o Dara Julọ

### Ilana Anfaani Ti o Kere Julọ

- Ṣeto awọn agbara si hihan ti o ni ihamọ julọ ti o tun gba lilo wọn laaye
- Lo hihan Alabaṣepọ fun awọn iṣẹ ti o ni ifamọ
- Pa awọn agbara ti o ko lo

### Awọn Audit Deede

- Ṣayẹwo hihan agbara ni gbogbo oṣu
- Ṣayẹwo fun awọn agbara ti a ko pin si ẹka
- Yọ wiwọle kuro fun awọn alabaṣepọ ti a ko lo

### Gbigbasilẹ Log ati Abojuto

- Ṣakiyesi iru awọn agbara wo ni a n lo julọ
- Tọpinpin awọn ilana wiwọle alabaṣepọ
- Fun itaniji lori lilo agbara ti ko wọpọ

### Awọn Agbara Ẹni-Kẹta

- Ṣayẹwo awọn agbara ẹni-kẹta ṣaaju mimu wọn ṣiṣẹ
- Pin wọn si ẹka kedere
- Bẹrẹ pẹlu hihan Inu tabi Alabaṣepọ
- Gbe si Gbangba nikan lẹhin ayẹwo

## Ṣiṣatunṣe Iṣoro

**Agbara kan ko han ninu atokọ**
- Jẹrisi pe agbara naa ti fi sori ẹrọ ati pe o n ṣiṣẹ
- Ṣayẹwo pe plugin/addon ti ṣiṣẹ
- Tuntun oju-iwe naa

**Mi ò le yi hihan agbara kan pada**
- Jẹrisi pe o ni awọn igbanilaaye alakoso
- Ṣayẹwo pe plugin kan ko tii agbara naa mọ
- Gbiyanju pipa awọn plugin ti o n koju ara wọn

**Alabaṣepọ kan ko le wọle si agbara kan**
- Jẹrisi pe alabaṣepọ naa wa ninu atokọ-igbanilaaye
- Ṣayẹwo pe hihan agbara naa ti ṣeto si Alabaṣepọ
- Jẹrisi pe awọn iwe-ẹri alabaṣepọ naa tọ
- Ṣayẹwo awọn log API fun awọn aṣiṣe ijẹrisi

**Awọn agbara ti a ko pin si ẹka n tẹsiwaju lati han**
- Ṣayẹwo ki o pin wọn si ẹka ninu ifitonileti alakoso
- Ṣayẹwo pe a ti fipamọ ipin-ẹka rẹ
- Jẹrisi pe plugin ti o pese agbara naa wa ni imudojuiwọn

## Gbigbe Kuro lati Ipo Ogún

Ti o ba n ṣe igbesoke lati ẹya atijọ ti Superdav AI Agent:

- Gbogbo awọn agbara to wa tẹlẹ ni a pin si ẹka laifọwọyi bi Gbangba
- Ṣayẹwo ki o ṣatunṣe hihan bi o ṣe nilo
- Ko si igbesẹ ti a nilo ayafi ti o ba fẹ fi opin si wiwọle

Wo **Gbigbe Ipo Ẹni-Kẹta** fun awọn alaye diẹ sii lori iyipada si iṣepọ Abilities API abinibi.

## Awọn Igbesẹ Tó Kàn

Lẹhin atunto hihan agbara:

1. **Ṣayẹwo awọn agbara rẹ**: Ṣe audit gbogbo awọn agbara ti a fi sori ẹrọ
2. **Pin awọn agbara ti a ko pin si ẹka si ẹka**: Dahun si awọn ifitonileti alakoso
3. **Ṣeto awọn alabaṣepọ**: Ṣafikun awọn alabaṣepọ ti o ba n lo hihan Alabaṣepọ
4. **Ṣakiyesi lilo**: Tọpinpin iru awọn agbara wo ni a n lo julọ
5. **Ṣe igbasilẹ awọn ipinnu**: Pa awọn akọsilẹ mọ lori idi ti o fi yan ipele hihan kọọkan
