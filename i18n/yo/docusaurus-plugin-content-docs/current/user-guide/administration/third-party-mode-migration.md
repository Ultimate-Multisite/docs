# Ìṣíkiri Ipo Ẹni-kẹta

Superdav AI Agent v1.12.0 yí bí a ṣe ń bójú tó àwọn agbára ẹni-kẹta padà. **Ipo ẹni-kẹta báyìí ń lo auto gẹ́gẹ́ bí ìpìlẹ̀**, ó ń jẹ́ kí ìṣepọ̀ WordPress Abilities API abinibi ṣiṣẹ́ lórí WordPress 7.0+ láìsí iṣètò ọwọ́.

## Kí Ni Ó Yí Padà?

### Ṣáájú v1.12.0

Àwọn agbára ẹni-kẹta nílò iṣètò ọwọ́:

- O ní láti tan "ipo ẹni-kẹta" ní kedere
- A máa ń kó àwọn agbára láti inú ìforúkọsílẹ̀ àdáṣe kan
- Ìṣepọ̀ pẹ̀lú WordPress Abilities API jẹ́ aṣàyàn
- Ipo ogún ni ìpìlẹ̀

### Lẹ́yìn v1.12.0

Àwọn agbára ẹni-kẹta ń ṣiṣẹ́ laifọwọyi:

- Ipo ẹni-kẹta ń lo "auto" gẹ́gẹ́ bí ìpìlẹ̀
- Àwọn agbára ń ṣepọ̀ ní abinibi pẹ̀lú WordPress Abilities API
- Kò sí iṣètò ọwọ́ tí a nílò lórí WordPress 7.0+
- Ipo ogún ṣì wà fún àwọn ẹ̀dà WordPress àgbà

## Ta Ni Ó Kan?

### Àwọn Ìfisípò Tuntun (WordPress 7.0+)

**Kò sí ìgbésẹ̀ tí a nílò.** A ṣètò ipo ẹni-kẹta laifọwọyi sí "auto", àwọn agbára sì ń ṣiṣẹ́ lẹ́sẹ̀kẹsẹ̀.

### Àwọn Ìfisípò Tó Wà Tẹ́lẹ̀

**Àwọn ìṣètò rẹ ni a pa mọ́.** Tí o bá ń lo:

- **Ipo ogún**: O dúró nínú ipo ogún (kò sí ìyípadà)
- **Ipo ẹni-kẹta ọwọ́**: O dúró nínú ipo ọwọ́ (kò sí ìyípadà)
- **Ipo auto**: O ń tẹ̀síwájú pẹ̀lú ipo auto (kò sí ìyípadà)

### Àwọn Ẹ̀dà WordPress Ṣáájú 7.0

**Ipo ogún ṣì wà.** Tí o bá wà lórí WordPress 6.x tàbí ṣáájú:

- Ipo ẹni-kẹta ń lo "legacy" gẹ́gẹ́ bí ìpìlẹ̀
- O lè tan ipo ẹni-kẹta ní ọwọ́ tí o bá fẹ́
- Ṣe ìgbéléwọn sí WordPress 7.0+ láti lo Abilities API abinibi

## Líloye Àwọn Ipo Náà

### Ipo Auto (Ìpìlẹ̀ Tuntun)

**Ipo Auto** ń lo ìṣepọ̀ WordPress Abilities API abinibi:

- A forúkọsílẹ̀ àwọn agbára nípasẹ̀ WordPress hooks
- Ìbámu kikún pẹ̀lú WordPress 7.0+ Abilities API
- Ìṣàwárí laifọwọyi fún àwọn agbára ẹni-kẹta
- Kò sí iṣètò ọwọ́ tí a nílò

**Ìgbà láti lò ó**: WordPress 7.0+ pẹ̀lú àwọn agbára ẹni-kẹta

### Ipo Ọwọ́

**Ipo ọwọ́** nílò iṣètò kedere:

- O sọ àwọn agbára ẹni-kẹta tí yóò kó wọlé
- Ó wúlò fún ìdánwò tàbí kíkó agbára ní yíyan
- Ó nílò ṣíṣàtúnṣe àwọn fáìlì iṣètò
- Ìṣàkóso púpọ̀ síi, ṣùgbọ́n iṣètò púpọ̀ síi

**Ìgbà láti lò ó**: Ìdánwò, kíkó agbára ní yíyan, tàbí àwọn iṣètò àdáṣe

### Ipo Ogún

**Ipo ogún** ń lo ètò agbára ẹni-kẹta àtijọ́:

- Ìforúkọsílẹ̀ agbára àdáṣe (kì í ṣe WordPress Abilities API)
- Ó bá àwọn ẹ̀dà WordPress àgbà ṣiṣẹ́ sẹ́yìn
- Kò sí ìṣepọ̀ WordPress abinibi
- A ti samisi rẹ̀ gẹ́gẹ́ bí èyí tí a kò tún gbà níyànjú, ṣùgbọ́n a ṣì ń ṣe àtìlẹ́yìn fún un

**Ìgbà láti lò ó**: WordPress 6.x tàbí ṣáájú, tàbí nígbà tí o nílò ìbámu ogún

## Ṣíṣàyẹ̀wò Ipo Rẹ Lọ́wọ́lọ́wọ́

### Nípasẹ̀ Pánẹ́ẹ̀lì Alákóso

1. Lọ sí **WordPress Admin** → **Superdav AI Agent** → **Settings**
2. Wá ìṣètò **Ipo Ẹni-kẹta**
3. Wàá rí ipo rẹ lọ́wọ́lọ́wọ́ àti àwọn aṣàyàn láti yí i padà

### Nípasẹ̀ Code

```php
$mode = get_option( 'superdav_third_party_mode' );
echo $mode; // 'auto', 'manual', or 'legacy'
```

## Yíyí Ipo Rẹ Padà

### Yí Padà sí Ipo Auto

Tí o bá wà lórí WordPress 7.0+ tí o sì fẹ́ lo ipo auto:

1. Lọ sí **Superdav AI Agent** → **Settings**
2. Wá **Ipo Ẹni-kẹta**
3. Yan **Auto (WordPress Abilities API)**
4. Tẹ **Save**

Superdav AI Agent yóò ṣàwárí àti forúkọsílẹ̀ àwọn agbára ẹni-kẹta laifọwọyi.

### Yí Padà sí Ipo Ọwọ́

Tí o bá fẹ́ ṣàkóso àwọn agbára tí ń kó wọlé ní ọwọ́:

1. Lọ sí **Superdav AI Agent** → **Settings**
2. Wá **Ipo Ẹni-kẹta**
3. Yan **Manual**
4. Tẹ **Save**
5. Ṣàtúnṣe fáìlì iṣètò rẹ láti sọ àwọn agbára tí yóò kó wọlé

### Yí Padà sí Ipo Ogún

Tí o bá nílò ìbámu ogún:

1. Lọ sí **Superdav AI Agent** → **Settings**
2. Wá **Ipo Ẹni-kẹta**
3. Yan **Legacy**
4. Tẹ **Save**

## Àwọn Àǹfààní Ipo Auto

### Ìṣàwárí Laifọwọyi

A máa ń ṣàwárí àwọn agbára laifọwọyi láti inú:

- Àwọn plugin tí a ti fi sílẹ̀
- Theme tí ń ṣiṣẹ́
- Àwọn plugin tí a gbọ́dọ̀ lò
- Àwọn plugin drop-in

Kò sí ìforúkọsílẹ̀ ọwọ́ tí a nílò.

### Ìṣepọ̀ Abinibi

Àwọn agbára ń ṣepọ̀ pẹ̀lú WordPress Abilities API:

- Ó bá WordPress core mu
- Ó ń ṣiṣẹ́ pẹ̀lú WordPress admin
- Ó bá àwọn plugin mìíràn tí ń lo Abilities API mu
- Ó ti ṣetan fún ọjọ́ iwájú bí WordPress ṣe ń dàgbà

### Ìṣàkóso Tó Rọrùn

- Kò sí àwọn fáìlì iṣètò láti ṣàtúnṣe
- Kò sí ìforúkọsílẹ̀ agbára ọwọ́
- Àwọn ìṣàkóso Híhàn Agbára ń ṣiṣẹ́ laifọwọyi
- Àwọn ìkìlọ̀ alákóso ń kéde àwọn agbára tí a kò tíì pín sí ẹ̀ka fún ọ

### Ìṣe Tó Dáa Síi

- A máa ń fi àwọn agbára sínú cache
- A máa ń kó wọn wọlé nígbà tí a bá nílò wọn
- A ṣe àtúnṣe rẹ̀ fún WordPress 7.0+

## Ọ̀nà Ìṣíkiri

### Tí O Bá Wà Lórí WordPress 6.x

1. **Ṣe ìgbéléwọn sí WordPress 7.0+** (nígbà tí o bá ṣetan)
2. **Ṣe ìmúdójúìwọ̀n Superdav AI Agent** sí v1.12.0+
3. **Yí ipo ẹni-kẹta padà sí Auto** (aṣàyàn; ipo ogún ṣì ń ṣiṣẹ́)
4. **Ṣàyẹ̀wò híhàn agbára** láti rí dájú pé àwọn ìṣàkóso ìwọlé tọ́

### Tí O Bá Wà Lórí WordPress 7.0+

1. **Ṣe ìmúdójúìwọ̀n Superdav AI Agent** sí v1.12.0+
2. **Jẹ́rìí pé a ṣètò ipo ẹni-kẹta sí Auto** (ó yẹ kí ó rí bẹ́ẹ̀ ní ìpìlẹ̀)
3. **Ṣàyẹ̀wò híhàn agbára** láti rí dájú pé àwọn ìṣàkóso ìwọlé tọ́
4. **Dán àwọn agbára ẹni-kẹta wò** láti jẹ́rìí pé wọ́n ń ṣiṣẹ́

## Yíyanjú Ìṣòro

### Àwọn agbára kò ń kó wọlé nínú ipo auto

- Jẹ́rìí pé o wà lórí WordPress 7.0+
- Ṣàyẹ̀wò pé a ṣètò ipo ẹni-kẹta sí "Auto"
- Jẹ́rìí pé plugin tí ń pèsè agbára náà ń ṣiṣẹ́
- Ṣàyẹ̀wò àwọn àkọsílẹ̀ aṣìṣe WordPress fún àwọn aṣìṣe ìforúkọsílẹ̀

### Mo fẹ́ pa ipo ogún mọ́

- Lọ sí **Settings** → **Ipo Ẹni-kẹta**
- Yan **Legacy**
- Tẹ **Save**
- Ipo ogún yóò tẹ̀síwájú láti ṣiṣẹ́

### Àwọn agbára àdáṣe mi kò hàn

- Jẹ́rìí pé a forúkọsílẹ̀ wọn nípasẹ̀ WordPress hooks
- Ṣàyẹ̀wò pé wọ́n ṣe Abilities API lọ́nà tó tọ́
- Ṣàyẹ̀wò àwọn àkọsílẹ̀ aṣìṣe WordPress
- Lo ojúewé alákóso **Híhàn Agbára** láti rí gbogbo àwọn agbára tí a forúkọsílẹ̀

### Mo ń gba àwọn ìkìlọ̀ "agbára tí a kò pín sí ẹ̀ka"

- Èyí jẹ́ ohun tí ó ṣe déédéé fún àwọn agbára ẹni-kẹta tuntun
- Ṣàyẹ̀wò kí o sì pín wọn sí ẹ̀ka nínú ìkìlọ̀ alákóso
- Wo **Híhàn Agbára** fún àwọn ẹ̀kúnrẹ́rẹ́ lórí pípín sí ẹ̀ka

## Ìbámu Sẹ́yìn

### Àwọn Iṣètò Tó Wà Tẹ́lẹ̀

Tí o bá ní àwọn iṣètò agbára ẹni-kẹta tó wà tẹ́lẹ̀:

- **Ipo Legacy**: Ìṣètò rẹ ń bá a lọ láti ṣiṣẹ́
- **Ipo Manual**: Ìṣètò rẹ ń bá a lọ láti ṣiṣẹ́
- **Ipo Auto**: A kò ka ìṣètò rẹ sí (ipo Auto ló gba àkóso)

Láti pa ìṣètò àdáṣe tirẹ mọ́, dúró ní ipo Manual tàbí Legacy.

### Ìlànà Àkókò Ìparí Lílò

- **v1.12.0**: Àwọn ipo Legacy àti Manual ṣì ní àtìlẹ́yìn kíkún
- **v1.13.0+**: Ipo Legacy lè fi àwọn ìkìlọ̀ ìparí lílò hàn
- **v2.0.0**: Ipo Legacy lè yọ kúrò (TBD)

## Àwọn Ìlànà Tó Dára Jù

### Fún Àwọn Ìfiṣètò Tuntun

- Lo ipo Auto (òun ni àiyipada)
- Jẹ́ kí Superdav AI Agent ṣàwárí àwọn agbára laifọwọyi
- Lo Híhàn Agbára láti ṣàkóso ìwọlé

### Fún Àwọn Ìfiṣètò Tó Wà Tẹ́lẹ̀

- Ṣe àgbéga sí WordPress 7.0+ nígbà tó bá ṣeé ṣe
- Yí padà sí ipo Auto fún ìṣàkóso tó rọrùn
- Ṣàyẹ̀wò kí o sì ṣàtòjọ àwọn agbára nípa lílo Híhàn Agbára

### Fún Àwọn Agbára Àdáṣe

- Forúkọsílẹ̀ àwọn agbára nípasẹ̀ WordPress hooks (Abilities API)
- Yẹra fún àwọn àkọsílẹ̀ agbára àdáṣe
- Ṣe ìdánwò lórí WordPress 7.0+ pẹ̀lú ipo Auto

## Àwọn Ìgbésẹ̀ Tó Kàn

1. **Ṣàyẹ̀wò ẹ̀yà WordPress rẹ**: Jẹ́rìí pé o wà lórí 7.0+ fún ipo Auto
2. **Ṣàyẹ̀wò ipo ẹgbẹ́-kẹta rẹ**: Lọ sí Settings kí o sì ṣàyẹ̀wò ipo rẹ lọwọlọwọ
3. **Ṣe ìmúdójúìwọ̀n bí ó bá nílò**: Yí padà sí ipo Auto bí o bá wà lórí WordPress 7.0+
4. **Ṣàtòjọ àwọn agbára**: Ṣàyẹ̀wò kí o sì ṣàtòjọ eyikeyi agbára tí a kò tíì ṣàtòjọ
5. **Ṣe ìdánwò**: Jẹ́rìí pé àwọn agbára ẹgbẹ́-kẹta rẹ ń ṣiṣẹ́ dáadáa

## Àwọn Kókó Tó Jọmọ́

- **Híhàn Agbára**: Ṣàkóso àwọn agbára wo ni a fi hàn ní ibi wo
- **WordPress Abilities API**: Kọ́ nípa ìforúkọsílẹ̀ agbára abínibí WordPress
- **Ìdàgbàsókè Agbára Ẹgbẹ́-kẹta**: Ṣẹ̀dá àwọn agbára tó ń ṣiṣẹ́ pẹ̀lú ipo Auto
