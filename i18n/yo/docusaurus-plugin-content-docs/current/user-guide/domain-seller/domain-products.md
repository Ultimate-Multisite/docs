# Ọjà Domain àti Ìdíyelé

Ọjà domain ni bí o ṣe ń ṣàkóso ìdíyelé, TLDs, àṣírí WHOIS, àti yíyan olupese. Ọjà domain kọọkan jẹ́ ọjà Multisite Ultimate boṣewa pẹ̀lú taabu **Ètò Domain** àfikún.

## Ṣíṣẹ̀dá ọjà domain

1. Lọ sí **Network Admin › Ultimate Multisite › Ọjà**
2. Tẹ **Ṣàfikún Tuntun**
3. Ṣètò irú ọjà sí **Domain**
4. Ṣàtòpọ̀ taabu **Ètò Domain** (wo ìsàlẹ̀)
5. Fipamọ́

Ọjà domain máa ń hàn pẹ̀lú àmì **Domain** aláwọ̀ púpù ní inú àtòjọ ọjà, a sì lè yọ wọn jáde nípa lílo taabu **Ọjà Domain**.

## Taabu ètò domain

### Olupese

Yan registrar tí yóò bójú tó ìforúkọsílẹ̀ fún ọjà yìí. Aiyipada ni **Olupese aiyipada** àgbáyé tí a ṣètò nínú ètò Domain Seller.

### TLDs tí a ṣe àtìlẹ́yìn fún

Fi sílẹ̀ ní òfo láti dá **ọjà tó mú gbogbo rẹ̀** ṣẹ̀, èyí tí yóò kan gbogbo TLDs tí ọjà míì kò bá mu.

Tẹ àtòjọ TLDs tí a fi kómà yà (fún àpẹẹrẹ, `.com, .net, .org`) láti dá **ọjà pàtó fún TLD** ṣẹ̀, èyí tí yóò kan àwọn ìfikún wọ̀nyẹn nìkan.

**Bí ìbámu ọjà ṣe ń ṣiṣẹ́:** Nígbà tí oníbàárà bá wá domain kan, addon yóò yan ọjà tó bá a mu jù lọ. Ọjà kan tó ní `.com` nínú àtòjọ TLD rẹ̀ ní àkọ́kọ́ ju ọjà tó mú gbogbo rẹ̀ lọ. Tí kò bá sí ọjà pàtó fún TLD tó bá a mu, a ó lo ọjà tó mú gbogbo rẹ̀. Tí kò bá sí ọjà kankan, ìwá domain kò ní hàn.

### Irú àfikún èrè

Ọ̀nà mẹ́ta ló ń ṣàkóso bí a ṣe ń ṣe ìṣirò iye owó títà rẹ láti inú iye owó osunwon:

| Ọ̀nà | Bí ó ṣe ń ṣiṣẹ́ |
|---|---|
| **Ìdá ogorun** | Ṣàfikún ìdá ogorun kan sórí iye owó osunwon. Àfikún èrè 20% lórí domain osunwon $10 máa jẹ́ $12. |
| **Àfikún èrè títọ́** | Ṣàfikún iye dọ́là títọ́ kan. Àfikún èrè $5 lórí domain $10 máa jẹ́ $15. |
| **Iye owó títọ́** | Kò ka iye owó osunwon rárá. Ó máa ń gba iye tí o tẹ sílẹ̀ nígbà gbogbo. |

### Ìdíyelé ìbẹ̀rẹ̀

Mú un ṣiṣẹ́ láti fúnni ní iye owó ọdún àkọ́kọ́ tó ní ẹdinwo. Ṣètò **Iye owó ìbẹ̀rẹ̀** lọ́tọ̀ (iye owó ọdún 1) lẹ́gbẹ̀ẹ́ **Iye owó ìtúnṣe** déédéé (ọdún 2+). Oníbàárà yóò rí iye owó méjèèjì tí a fi hàn nígbà checkout kí wọ́n lè mọ ohun tí wọ́n máa retí nígbà ìtúnṣe.

### Àṣírí WHOIS

Ó ń ṣàkóso bóyá a ó fúnni ní ààbò àṣírí WHOIS fún domains tí a forúkọ sílẹ̀ nípasẹ̀ ọjà yìí.

| Ètò | Ìhùwàsí |
|---|---|
| **Paa** | A kì í fúnni ní àṣírí WHOIS tàbí mú un ṣiṣẹ́ rí. |
| **Wà Pẹ̀lú Rẹ̀ Nígbà Gbogbo** | Àṣírí WHOIS máa ń ṣiṣẹ́ laifọwọyi nígbà ìforúkọsílẹ̀ láìsí ìsanwó. |
| **Yíyan Oníbàárà** | Àpótí àmì kan máa ń hàn nígbà checkout. Ṣètò **Iye owó àṣírí** láti gba owó lọ́dọọdún, tàbí fi sílẹ̀ ní $0 láti fúnni ní ọ̀fẹ́. |

Fún Namecheap, àṣírí WHOIS ń lo WhoisGuard (ọ̀fẹ́ nígbà gbogbo). Fún OpenSRS, ó ń lo iṣẹ́ àṣírí OpenSRS (ó lè ní iye owó ní osunwon).

## Gbigbewọlé àti ìmúdọ́gba TLD

Ọjà domain ń fi ìdíyelé osunwon àkókò-gidi hàn tí a mú wá láti ọdọ olupese tí a so pọ̀. Kí èyí lè ṣiṣẹ́, a gbọ́dọ̀ gbé TLDs wọlé.

- **Ìmúdọ́gba ọwọ́:** Settings › Domain Seller › Sync TLDs (fún olupese kọọkan)
- **Ìmúdọ́gba laifọwọyi:** Ń ṣiṣẹ́ lojoojúmọ́ nípasẹ̀ iṣẹ́ cron tí a ṣètò jákèjádò gbogbo àwọn olupese tí a ti ṣàtòpọ̀

Lẹ́yìn ìmúdọ́gba, lọ sí taabu Ètò Domain ti ọjà domain èyíkéyìí, kí o sì lo olùyan TLD láti rí TLDs tó wà pẹ̀lú iye owó osunwon wọn lọwọlọwọ.

## Ìtúnṣe laifọwọyi

Àwọn ìtúnṣe domain so mọ́ ipò membership oníbàárà:

- Nígbà tí membership kan bá túnṣe, tí domain kan sì so mọ́ ọn, ìtúnṣe domain náà máa ń wọ ìlà laifọwọyi
- Àwọn ìgbìyànjú ìtúnṣe ń lo ẹnu-ọ̀nà ìsanwó oníbàárà tó ń ṣiṣẹ́
- Àwọn ìtúnṣe tó kùnà ni a máa tún gbìyànjú laifọwọyi pẹ̀lú exponential backoff
- A máa fi àwọn ìfitónilétí ímeèlì ránṣẹ́ fún ìtúnṣe tó ṣàṣeyọrí, ìkùnà, àti ìparí tó ń bọ̀

Àwọn ID àdàkọ ímeèlì fún àwọn ìṣẹ̀lẹ̀ ìgbésíayé domain:

| Ìṣẹ̀lẹ̀ | ID Àdàkọ |
|---|---|
| Domain tí a forúkọ sílẹ̀ | `domain_registered` |
| Domain tí a túnṣe | `domain_renewed` |
| Ìtúnṣe kùnà | `domain_renewal_failed` |
| Domain ń parí láìpẹ́ | `domain_expiring_soon` |

## Admin: Ìforúkọsílẹ̀ domain ọwọ́

Láti forúkọ domain kan sílẹ̀ ní orúkọ oníbàárà láì jẹ́ kí wọ́n gba checkout kọjá:

1. Lọ sí **Network Admin › Ultimate Multisite › Forúkọ Domain Sílẹ̀**
2. Yan oníbàárà náà, kí o sì tẹ orúkọ domain náà sílẹ̀
3. Kún àwọn àlàyé ìbásọrọ̀ registrant (orúkọ, àdírẹ́sì, fóònù)
4. Tẹ **Forúkọ Sílẹ̀**

A ó dá ìgbasilẹ domain ṣẹ̀, a ó sì so ó mọ́ Account oníbàárà náà.
