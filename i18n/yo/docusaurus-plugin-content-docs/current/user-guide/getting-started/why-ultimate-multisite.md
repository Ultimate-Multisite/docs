# Kí nìdí Ultimate Multisite

Tí o bá dé àpilẹ̀kọ yìí, ó ṣeé ṣe kí o ti gbọ́ pé plugin wa lè jẹ́ ojútùú láti mú èrò ìṣòwò rẹ ṣẹ. A nírètí pé ìwọ yóò ní àwọn ìdáhùn tí o nílò ní òpin àpilẹ̀kọ yìí.

## Kí ni Ultimate Multisite?

Ultimate Multisite jẹ́ WordPress Multisite plugin tí ó jẹ́ kí o lè fún àwọn oníbàárà ní WaaS (ìyẹn Website as a Service). Pẹ̀lú plugin wa, o lè ní oríṣiríṣi ìpele subscription kí àwọn oníbàárà sì máa san owó àtúnṣe sí ọ láti ní site tí a gbàlejò nínú nẹ́tíwọ́ọ̀kì Multisite rẹ.

Ultimate Multisite jẹ́ kí o lè gba àwọn ìsanwó rẹ pẹ̀lú oríṣiríṣi gateways (bíi WooCommerce tàbí Stripe), ṣẹ̀dá àwọn àwòṣe site láti jẹ́ kí ìgbésí ayé rẹ rọrùn púpọ̀ nígbà tí olùmúlò tuntun bá forúkọsílẹ̀ lórí nẹ́tíwọ́ọ̀kì rẹ, so àwọn domains mọ́, kí o sì ṣe àdáni àwọn fọ́ọ̀mù ìforúkọsílẹ̀ rẹ láti ṣẹ̀dá ìrírí tí a ṣe pàtàkì fún àwọn oníbàárà rẹ.

Èyí sì jẹ́ ìbẹ̀rẹ̀ nìkan nítorí pé o lè faagun àwọn iṣẹ́ rẹ pẹ̀lú add-ons àti webhooks.

## Àwọn àbùdá míì

Ultimate Multisite tún ń pèsè:

  * Ìṣọ̀kan Domain Mapping pẹ̀lú àwọn olùpèsè gbígbàlejò Managed WordPress: Ultimate Multisite máa ń ṣàfikún àwọn domains tí a ti map laifọwọ́yi sí àwọn pánẹ́ẹ̀lì admin ti WP Engine, Cloudways, cPanel, RunCloud, Closte, àti míì nígbà tí domain tuntun bá jẹ́ mapped, kí o má bàa ní láti ṣe é pẹ̀lú ọwọ́. Ní àwọn ọ̀ràn kan, a ó tilẹ̀ bẹ̀rẹ̀ gbígba ìwé-ẹ̀rí SSL fún ọ.

  * Webhooks: Ultimate Multisite ní ètò Webhook alágbára tí ó jẹ́ kí o lè ṣepọ̀ nẹ́tíwọ́ọ̀kì rẹ pẹ̀lú ọ̀pọ̀lọpọ̀ iṣẹ́ oríṣiríṣi. A tún ń pèsè ìṣọ̀kan tààrà pẹ̀lú Zapier, èyí tí ó ṣí nẹ́tíwọ́ọ̀kì rẹ sí díẹ̀ lára àwọn 1000+ apps àti iṣẹ́ tí ó wà lórí pẹpẹ wọn.

  * Àwọn Ojúewé Admin alágbára láti ran àwọn oníbàárà rẹ lọ́wọ́ nípa lílo page builder tí o fẹ́ràn: Ultimate Multisite ní add-on tó wúlò tí a ń pè ní Admin Pages Creator, èyí tí ní pàtàkì jẹ́ kí o lè ṣàfikún àwọn Ojúewé Admin tí a ṣe àdáni pátápátá sí dashboard àwọn oníbàárà rẹ ní lílo àwọn àwòṣe Elementor, Beaver Builder, àti Brizy.

  * Admin Themes méjì ọ̀tọ̀ọ̀tọ̀ láti ṣe àdáni ìrírí dashboard àwọn oníbàárà rẹ: Wo Material WP àti PRO.

Àti àwọn ohun tó ti ni ìlọsíwájú síi:

  * Àtìlẹ́yìn fún Template Overrides: Pẹ̀lú Ultimate Multisite, o lè rọ́pò àwọn àwòṣe HTML àiyipada fún ìṣàn ìforúkọsílẹ̀ pẹ̀lú àwọn ẹ̀dà tirẹ láìní láti ṣàníyàn nípa yíyí àwọn fáìlì core plugin padà.

  * Search and Replace API: O tún lè fi àmi sí àwọn àwòṣe site pẹ̀lú placeholders bíi {{site_name}} kí o sì lo Search and Replace API láti fi ìwífún tí oníbàárà rẹ pèsè kún àwọn placeholders wọ̀nyẹn laifọwọ́yi lẹ́sẹ̀kẹsẹ̀ lẹ́yìn ìforúkọsílẹ̀.

## Àwọn ọ̀nà lílò

Báyìí àkókò ti tó láti rí díẹ̀ lára àwọn ọ̀nà lílò tí ó ṣeé ṣe fún Ultimate Multisite.

### Ọ̀ràn 1: Ilé-iṣẹ́ Aṣojú kan

Ní gbogbogbò, àwọn ọgbọ́n pàtàkì ti ilé-iṣẹ́ aṣojú kan wà nínú àpẹrẹ àwọn wẹ́ẹ̀bù, pẹ̀lú àwọn abala bíi gbígbàlejò wọn tàbí títàjà wọn tí a ń kà sí àwọn iṣẹ́ àfikún.

Fún àwọn ilé-iṣẹ́ aṣojú, Ultimate Multisite ń pèsè ìfilọ́ iye tí ó tayọ ní agbára rẹ láti gbàlejò àti ṣàkóso ọ̀pọ̀ wẹ́ẹ̀bù lórí pẹpẹ kan ṣoṣo. Jù bẹ́ẹ̀ lọ, àwọn ilé-iṣẹ́ aṣojú tí wọ́n ń ṣe àtòṣe àwọn àpẹrẹ wọn lórí àwọn themes kan pàtó bíi GeneratePress, Astra, OceanWP, tàbí àwọn míì lè lo àwọn agbára Ultimate Multisite láti mú àwọn themes wọ̀nyí ṣiṣẹ́ laifọwọ́yi fún site tuntun kọọkan.

Ìṣàkóso tó muna lórí plugins àti themes ni a ń pa mọ́ lórí ìpìlẹ̀ product kọọkan nípasẹ̀ àwọn ètò Ultimate Multisite, tí ó jẹ́ kí plugins àti themes lè jẹ́ kí ó wà fún lílò tàbí farapamọ́, bakanna bí ipò ìmúṣiṣẹ́ wọn nígbà tí a bá dá a sílẹ̀ fún site tuntun.

Àwọn ilé-iṣẹ́ aṣojú yóò rí ìsinmi ọkàn pẹ̀lú Ultimate Multisite tí ó jẹ́ kí wọ́n ṣe ohun tí wọ́n mọ̀ dáadáa jù - ṣíṣe àpẹrẹ àwọn wẹ́ẹ̀bù alárinrin.

### Ọ̀ràn 2: Olùpèsè Niche

Òwe àtijọ́ kan wà tí ó sọ pé, “Ṣe ohun kan, kí o sì ṣe é dáadáa”. Fún ọ̀pọ̀ àwọn amọ̀ja, èyí túmọ̀ sí ṣíṣẹ̀dá product tàbí iṣẹ́ kan yí ká èrò pàtàkì kan ṣoṣo.

Tí o bá dojú kọ ìdàgbàsókè nẹ́tíwọ́ọ̀kì niche ti àwọn sites, ó ṣeé ṣe kí o fẹ́ pèsè àwọn iṣẹ́ tí ó dá lórí framework àti pẹpẹ kan náà. Ìgbà yẹn ni àwọn àwòṣe site lè ràn ọ́ lọ́wọ́!

Ọ̀kan lára àwọn àbùdá tuntun Ultimate Multisite, template site jẹ́ ibi tí a ti fi theme sí, tí a sì ti mú un ṣiṣẹ́, àwọn plugins tó ṣe pàtàkì ti fi sí, tí a sì ti mú wọn ṣiṣẹ́, àti àwọn àpẹẹrẹ posts tàbí ojúewé tí a ti ṣẹ̀dá. Nígbà tí oníbàárà bá ṣẹ̀dá site tuntun tí ó dá lórí àwòṣe náà, àwọn akoonu àti ètò àwòṣe náà ni a máa daakọ sí site tuntun tí a ṣẹ̀dá.

Fún olùpèsè àwọn sites àti iṣẹ́ niche, èyí ń pèsè àǹfààní tí kò lẹ́gbẹ́ nínú agbára láti ṣẹ̀dá site lẹ́sẹ̀kẹsẹ̀ tí ó ti ṣetán láti ṣiṣẹ́ pẹ̀lú plugins àdáni àti àpẹrẹ. Oníbàárà nílò láti pèsè ìfọwọ́sowọ́pọ̀ kékeré nìkan láti parí iṣẹ́ náà.

### Ọ̀ràn 3: Gbígbàlejò Wẹ́ẹ̀bù WordPress

Ọ̀pọ̀lọpọ̀ ọ̀nà ló wà láti gbàlejò àwọn WordPress sites ṣùgbọ́n kì í sábà rọrùn tó bí pípèsè àyè wẹ́ẹ̀bù fún oníbàárà pẹ̀lú ẹ̀dà WordPress tí a ti fi sí tẹ́lẹ̀. Èyí jẹ́ nítorí pé ọ̀pọ̀ ìpinnu àti ìrònú ló nílò láti wá pọ̀ láti pèsè iṣẹ́ tó ní ìtumọ̀.

Ultimate Multisite tayọ ní agbègbè yìí nípa pípèsè ojútùú turnkey pípé fún gbígbàlejò àwọn WordPress sites. Ohun tí ó wà nínú ojútùú náà ni àwọn ẹ̀rọ pàtàkì láti pèsè àwọn iṣẹ́ subscription, gbigba ìsanwó, àwọn fọ́ọ̀mù checkout, àwọn voucher ẹdinwo, àti ìbánisọ̀rọ̀ oníbàárà.

Ọ̀pọ̀ iṣẹ́ pàtàkì tí ó yẹ láti fi WordPress Multisite sí dáadáa, ṣe àtúnṣe rẹ, àti tọ́jú rẹ ni Ultimate Multisite ń mú rọrùn dé ibi tí àwọn alákóso nẹ́tíwọ́ọ̀kì nílò láti ronú nípa àwọn abala gẹ́gẹ́ bí wọ́n ṣe kan iṣẹ́ wọn tàbí niche wọn nìkan, bíi àwọn ìpele product, ìdíyelé, àti àwọn ìfilọ́ iṣẹ́.

Fún àwọn olùdàgbàsókè tí wọ́n fẹ́ ṣepọ̀ pẹ̀lú Ultimate Multisite, ojútùú náà tún ń pèsè RESTful API pípé àti Webhooks fún ìfitónilétí ìṣẹ̀lẹ̀.

Láì gbára lé ọ̀pọ̀lọpọ̀ plugins àti licenses òde, Ultimate Multisite ń pèsè ojútùú tí ó kún fún àbùdá àti tí a lè fi wé ti Wix, Squarespace, [WordPress.com](https://WordPress.com), àti àwọn míì.
