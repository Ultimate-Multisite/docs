# Àwọn ohun tí Ultimate Multisite nílò

Ohun àkọ́kọ́ tí o lè bi ara rẹ kí o tó ra Ultimate Multisite ni àwọn ohun tí a nílò láti ṣiṣẹ́ plugin wa. Ó dáa, láyọ̀, àwọn ohun ìpilẹ̀ díẹ̀ pere ni a nílò fún mímú Ultimate Multisite ṣiṣẹ́: **orúkọ domain** , **olùpèsè hosting** tó dáa àti fifi WordPress **Multisite** sílẹ̀. Ìyẹn ni gbogbo ohun tí o nílò!

## Orúkọ Domain

Lákọ̀ọ́kọ́, ẹ jẹ́ ká sọ̀rọ̀ nípa orúkọ domain.

Nígbà tí o bá ń yan orúkọ domain, rántí ohun tí niche pàtó rẹ jẹ́ àti ohun tí ìwọ yóò dá pẹ̀lú Ultimate Multisite: Ṣé o fẹ́ dá agency kan? Ta àwọn website tó dàbí ilé ìtajà? Tàbí bóyá WordPress web hosting? Gbogbo nǹkan ṣeé ṣe pẹ̀lú Ultimate Multisite.

Plugin wa ń pèsè àwọn iṣẹ́ subscription, gbígba owó sísan, àwọn fọ́ọ̀mù checkout, àwọn voucher ẹdinwo àti ìbánisọ̀rọ̀ pẹ̀lú oníbàárà. Ultimate Multisite ń mú ọ̀pọ̀ iṣẹ́ tí a nílò láti fi WordPress Multisite sílẹ̀, ṣètò rẹ̀, àti tọ́jú rẹ̀ rọrùn.

Niche èyíkéyìí tí o bá yan ni a lè ṣàkóso ní lílo plugin wa, níwọ̀n ìgbà tí o bá ń ta àwọn website gẹ́gẹ́ bí iṣẹ́! Rí i dájú pé o yan orúkọ domain tí ó bá ètò business rẹ mu.

## Olùpèsè Hosting

Ní báyìí tí o ti ní orúkọ domain tó ń fa àkíyèsí, tí ó sì ṣàpẹẹrẹ niche àti business rẹ, o ní láti yan ilé iṣẹ́ web hosting tó dáa láti host network rẹ pẹ̀lú Ultimate Multisite. Àwọn aṣàyàn tó dáa wà lórí ọjà. Díẹ̀ lára àwọn olùpèsè hosting ń ṣepọ̀ dáadáa pẹ̀lú Ultimate Multisite, èyí tí ó lè jẹ́ kókó ìpinnu, dá lórí ohun tí o ń gbero láti kọ́.

Ultimate Multisite yẹ kí ó ṣiṣẹ́ lórí olùpèsè hosting èyíkéyìí. Ìyàtọ̀ kan ṣoṣo ni pé àwọn ilé iṣẹ́ hosting kan bí _WPMU DEV Hosting_ , _WP Engine_ , _Closte_ , _Cloudways_ , _Gridpane_ àti _Runcloud_ ní ìṣepọ̀ tó jinlẹ̀ síi fún domain mapping. Fún hosting míràn, o lè nílò láti map àwọn custom domains pẹ̀lú ọwọ́. Ó tún jẹ́ ìmọ̀ràn tó dáa láti gba olùpèsè hosting tí ó ń ṣe àtìlẹ́yìn fún AutoSSL. _Closte_ , _Runcloud_ àti _cPanel_ ń ṣe àtìlẹ́yìn rẹ̀.

Nípa àwọn àlàyé server, kò sí fọ́ọ̀múlà pàtó nítorí ó máa dá lórí àwọn feature àti function tí o máa pèsè àti ìwọn network rẹ. Shared hosting lè jẹ́ ìbẹ̀rẹ̀ tó dáa, ṣùgbọ́n ní kété tí network rẹ bá bẹ̀rẹ̀ sí í dàgbà, ó ṣeé ṣe kí o nílò VPS tí a yà sọ́tọ̀.

## WordPress Multisite

Ní báyìí, ẹ jẹ́ ká sọ̀rọ̀ díẹ̀ nípa fifi WordPress Multisite sílẹ̀.

WordPress Multisite jẹ́ kí o ní network àwọn site lórí installation kan ṣoṣo. Èyí jẹ́ feature tí a kọ sínú rẹ̀, ṣùgbọ́n kò ṣiṣẹ́ láìyípadà. O nílò láti jẹ́ kí feature yìí ṣiṣẹ́ nínú installation WordPress rẹ àti control panel web hosting.

Níní WordPress tí Multisite ti ṣiṣẹ́ **ni ìpìlẹ̀ fún Ultimate Multisite**. Níwọ̀n bí lílo pàtàkì ti Ultimate Multisite ṣe jẹ́ láti ran àwọn olumulo lọ́wọ́ láti ṣàkóso àti ta àwọn website gẹ́gẹ́ bí iṣẹ́, èyí gbọ́dọ̀ ṣe lórí WordPress tí Multisite ti ṣiṣẹ́.

A ní **àpilẹ̀kọ tó jinlẹ̀** tí ń kọ́ni ní ìgbésẹ̀-ní-ìgbésẹ̀ ohun tí ó yẹ kí a ṣe láti jẹ́ kí feature Multisite ṣiṣẹ́ lórí installation WordPress rẹ.

Lẹ́yìn tí o bá ti jẹ́ kí Multisite ṣiṣẹ́ lórí WordPress, ó tó àkókò láti fi Ultimate Multisite sílẹ̀. Láti fi Ultimate Multisite sílẹ̀ lọ́nà tó tọ́, o gbọ́dọ̀ ní:

  * WordPress: v5.3+ (A dámọ̀ràn: Ẹ̀dà stable tuntun jùlọ)

  * PHP: 7.4.x (àtìlẹ́yìn fún > 8.0 ń bọ̀ láìpẹ́)

  * MySQL: v5+ (A dámọ̀ràn: 5.6, bí o kò bá ní 8.0 nílẹ̀ lórí olùpèsè hosting rẹ)

Ìwọ̀nyí nìkan ni àwọn ohun software tí Ultimate Multisite nílò.

Rántí pé Ultimate Multisite lè ṣiṣẹ́ pẹ̀lú PHP 8.0 ṣùgbọ́n a dámọ̀ràn kí o ṣiṣẹ́ ọ́ pẹ̀lú PHP 7.4.x.

Pẹ̀lú èyí, a dámọ̀ràn fífi sílẹ̀ lórí **main domains** , kì í ṣe subdomains tàbí subdirectories. Ó lè ṣiṣẹ́ lórí subdomains/subdirectories, ṣùgbọ́n ó lè fi àwọn aṣìṣe kan hàn.

Fún ìtọ́sọ́nà pẹ̀lú àlàyé lórí bí a ṣe ń fi Ultimate Multisite sílẹ̀, **tẹ̀ ibi**.
