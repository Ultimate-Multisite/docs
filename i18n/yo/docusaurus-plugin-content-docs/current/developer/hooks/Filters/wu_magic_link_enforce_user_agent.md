# Àlẹmọ: wu_magic_link_enforce_user_agent

Ṣe àlẹmọ bóyá kí a fi ìjẹrisi user agent múlẹ̀.

## Àwọn Parameters

| Orúkọ | Iru | Àpèjúwe |
|------|------|-------------|
| $enforce | `bool` | Bóyá kí a fi ìbámu user agent múlẹ̀. |
