# Filter: wu_country_get_cities {#filter-wucountrygetcities}

Ó dá àtòjọ àwọn ìlú fún ipinlẹ kan nínú orilẹ-ede kan padà.

## Àwọn Parameter {#parameters}

| Orúkọ | Iru | Àpèjúwe |
|------|------|-------------|
| $cities | `array` | Àtòjọ àwọn orúkọ ìlú ipinlẹ. Kò sí àwọn key. |
| $country_code | `string` | Kóòdù ISO oní-lẹ́tà-méjì fún orilẹ-ede náà. |
| $state_code | `string` | Kóòdù ISO oní-lẹ́tà-méjì fún ipinlẹ náà. |
| $current_country | `\WP_Ultimo\Country\Country` | Instance ti kilasi lọwọlọwọ. |
