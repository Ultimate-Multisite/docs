# Àlẹmọ: wu_password_require_uppercase {#filter-wupasswordrequireuppercase}

Àlẹmọ bóyá kí ó jẹ́ dandan láti ní àwọn lẹ́tà uppercase nínú àwọn ọ̀rọ̀ìpamọ́.

## Àwọn Párámítà {#parameters}

| Orúkọ | Iru | Àpèjúwe |
|------|------|-------------|
| $require | `bool` | Bóyá kí uppercase jẹ́ dandan. Ìbẹ̀rẹ̀ jẹ́ true nígbà tí a bá fi àwọn òfin múlẹ̀. |
| $defender_active | `bool` | Bóyá Defender Pro Strong Password ń ṣiṣẹ́. |

### Láti ìgbà {#since}

- 2.4.0
