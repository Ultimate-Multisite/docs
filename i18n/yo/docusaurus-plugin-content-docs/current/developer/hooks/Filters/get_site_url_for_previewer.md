# Àlẹmọ: get_site_url_for_previewer {#filter-getsiteurlforpreviewer}

## Àwọn Parameter {#parameters}

| Orúkọ | Iru | Àpèjúwe |
|------|------|-------------|
| $domain | `string` | Domain àiyéṣe tí a ń lò ní báyìí, ó wúlò fún àwọn ìyípadà |
| $domain_options | `array` | Àkójọ gbogbo àwọn àṣàyàn domain tí a tẹ sí Ultimate Multisite Ètò -&gt; Ètò Nẹ́tíwọ́ọ̀kì -&gt; Àwọn Àṣàyàn Domain |
