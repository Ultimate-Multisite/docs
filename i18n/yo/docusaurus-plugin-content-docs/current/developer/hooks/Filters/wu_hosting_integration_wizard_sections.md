# Àlẹmọ: wu_hosting_integration_wizard_sections

Ó ń ṣàlẹmọ àwọn apá olùtọ́sọ́nà fún ìṣètò ìṣepọ̀ hosting.

Ó gba àwọn àfikún láàyè láti ṣàfikún, yọkúrò, tàbí ṣàtúnṣe àwọn apá olùtọ́sọ́nà.

## Àwọn Parameter

| Orúkọ | Iru | Àpèjúwe |
|------|------|-------------|
| $sections | `array` | Àwọn apá olùtọ́sọ́nà. |
| $integration | `\WP_Ultimo\Integrations\Host_Providers\Base_Host_Provider\|\WP_Ultimo\Integrations\Integration` | Ìṣepọ̀ tí a ń ṣètò. |
| $page | `\Hosting_Integration_Wizard_Admin_Page` | Ẹ̀dá ojú-ewé olùtọ́sọ́nà. |

### Láti

- 2.5.0
