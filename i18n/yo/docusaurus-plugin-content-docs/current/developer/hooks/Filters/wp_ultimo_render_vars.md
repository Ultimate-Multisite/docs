# Àlẹmọ: wp_ultimo_render_vars {#filter-wpultimorendervars}

Gba awọn olùdàgbàsókè àfikún láàyè láti ṣàfikún variable mìíràn sí render context káàkiri.

## Àwọn parameter {#parameters}

| Orúkọ | Irú | Àpèjúwe |
|------|------|-------------|
| $args | `array` | Array tí ó ní àwọn variable tí render call fi ránṣẹ́. |
| $view | `string` | Orúkọ view tí a ó render. |
| $default_view | `string` | Orúkọ fallback_view |
