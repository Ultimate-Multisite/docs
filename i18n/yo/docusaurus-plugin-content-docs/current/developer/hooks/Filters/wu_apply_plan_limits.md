# Filter: wu_apply_plan_limits

Gba àwọn olùdàgbàsókè plugin laaye láti yí àwọn ìdíwọ̀n náà kọjá ní kíákíá.

O lè lo filter yìí láti ṣiṣẹ́ code èyíkéyìí kí ìkankan nínú àwọn ìdíwọ̀n náà tó bẹ̀rẹ̀. Tí filter rẹ bá dá iye truthy padà, ìlànà náà yóò tẹ̀síwájú; tí ó bá dá iye falsy padà, code náà yóò dá padà, kò sì sí ọ̀kan lára àwọn hook tó wà ní ìsàlẹ̀ tí yóò ṣiṣẹ́.

### Láti

- 1.7.0
