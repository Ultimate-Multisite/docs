# Àlẹmọ: wu_use_domain_mapping

Pinnu bóyá mapping kan yẹ kí a lò

Ní gbogbogbò, o máa fẹ́ kí àwọn mapping tó ń ṣiṣẹ́ nìkan ni a gba láyè láti lò. Síbẹ̀, bí o bá fẹ́ lo logic tó ti ni ìlọsíwájú síi, tàbí gba àwọn domain tí kò ṣiṣẹ́ láyè láti jẹ́ mapped pẹ̀lú, kan ṣe àlẹmọ níbí.

## Àwọn Parameter {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $is_active | `bool` | Ṣé mapping náà yẹ kí a kà sí ohun tó ń ṣiṣẹ́? |
| $mapping | `\Domain` | Mapping tí a ń ṣàyẹ̀wò |
| $domain | `string` |  |
