# Àlẹmọ: wu_get_checkout_variables

Gba àwọn olùdàgbàsókè plugin láàyè láti ṣe àlẹmọ àwọn ìṣètò-tẹ́lẹ̀ ti ojúewé ìsanwó.

Ṣọ́ra, àwọn kọ́kọ́rọ́ tí ó sọnù lè bà ìsanwó jẹ́ pátápátá ní front-end.

## Àwọn Paramita

| Orúkọ | Iru | Àpèjúwe |
|------|------|-------------|
| $variables | `array` | Àwọn oníyí tí a ti sọ di ti agbègbè. |
| $checkout | `\Checkout` | Kíláàsì ìsanwó náà. |

### Láti

- 2.0.0
### Orísun

A ṣàlàyé rẹ̀ nínú [`inc/checkout/class-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout.php#L1970) ní ìlà 1970


## Àwọn Ìpadàbọ̀
Àkójọpọ̀ àwọn oníyí tuntun.
