# Àlẹmọ: wu_admin_notices {#filter-wuadminnotices}

Gba àwọn olùdàgbàsókè láàyè láti ṣe àlẹmọ àwọn ìfitónilétí admin tí Ultimate Multisite fi kun.

## Àwọn paramètres {#parameters}

| Orúkọ | Irú | Àpèjúwe |
|------|------|-------------|
| $notices | `array` | Àtòjọ àwọn ìfitónilétí fún panel pàtó yẹn. |
| $all_notices | `array` | Àtòjọ àwọn ìfitónilétí tí a fi kun, tí a yà sọ́tọ̀ nípasẹ̀ panel. |
| $panel | `string` | Panel láti gba àwọn ìfitónilétí náà padà. |
| $filter | `string` | Bí a bá ti yọ àwọn ìfitónilétí tí a lè kọ̀ kúrò. |
| $dismissed_messages | `array` | Àtòjọ àwọn kọ́kọ́rọ́ ìfitónilétí tí a ti kọ̀. |

### Láti {#since}

- 2.0.0
