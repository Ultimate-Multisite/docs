# Àlẹmọ: wu_minimum_password_strength {#filter-wuminimumpasswordstrength}

Ṣe àlẹmọ agbára ọ̀rọ̀ aṣínà tó kéré jù lọ tí a nílò (àmì zxcvbn).

Àwọn ìpele agbára: - 0, 1: Kò lágbára rárá - 2: Kò lágbára - 3: Àárín - 4: Lágbára (àiyipada)

## Àwọn Paramita {#parameters}

| Orúkọ | Iru | Àpèjúwe |
|------|------|-------------|
| $min_strength | `int` | Ìpele agbára tó kéré jù lọ tí a nílò. |
| $strength_setting | `string` | Iye ìṣètò admin (medium, strong, super_strong). |

### Láti {#since}

- 2.4.0
