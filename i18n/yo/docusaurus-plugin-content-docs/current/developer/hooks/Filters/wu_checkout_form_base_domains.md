# Àlẹmọ: wu_checkout_form_base_domains

Àlẹmọ àwọn domain ìpilẹ̀ fọ́ọ̀mù checkout tí a pín tí kò yẹ kí wọ́n di àwọn ìgbasilẹ domain tí a so mọ́.

Lo àlẹmọ yìí nígbà tí integration kan bá pèsè àwọn domain ìpilẹ̀ àfikún fún àwọn pápá fọ́ọ̀mù checkout **Site URL**. Àwọn domain tí àlẹmọ yìí dá padà ni a máa kà sí àwọn host ìforúkọsílẹ̀ tí a pín dípò àwọn domain àkànṣe fún site kọ̀ọ̀kan.

## Àwọn Paramita

| Orúkọ | Iru | Àpèjúwe |
|------|------|-------------|
| $domains | `array` | Àwọn domain ìpilẹ̀ tí a pín tí a kó jọ láti configuration fọ́ọ̀mù checkout. |

### Láti

- 2.13.0

### Orísun

A ṣàlàyé rẹ̀ nínú `inc/functions/domain.php`.


## Àwọn Ìpadàbọ̀

Array ti àwọn domain ìpilẹ̀ fọ́ọ̀mù checkout tí a ṣe normalize.
