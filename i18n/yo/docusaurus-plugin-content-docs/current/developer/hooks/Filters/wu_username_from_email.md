# Àlẹmọ: wu_username_from_email

Ṣe àlẹmọ orúkọ aṣàmúlò oníbàárà tuntun.

## Àwọn Paramita {#parameters}

| Orúkọ | Iru | Àpèjúwe |
|------|------|-------------|
| $username | `string` | Orúkọ aṣàmúlò oníbàárà. |
| $email | `string` | Àdírẹ́sì ímeèlì oníbàárà tuntun. |
| $new_user_args | `array` | Àkójọpọ̀ àwọn args aṣàmúlò tuntun, ó ṣeé ṣe kí ó ní orúkọ àkọ́kọ́ àti orúkọ ìdílé. |
| $suffix | `string` | Ṣàfikún string sí orúkọ aṣàmúlò láti jẹ́ kí ó jẹ́ aláìlẹ́gbẹ́. |
