# Àlẹmọ: wu_minimum_password_length

Ṣe àlẹmọ gígùn ọ̀rọ̀ aṣínà tó kere jù.

A máa fi ipa mú un nìkan nígbà tí wu_enforce_password_rules jẹ́ true.

## Àwọn Parameter

| Orúkọ | Type | Àpèjúwe |
|------|------|-------------|
| $min_length | `int` | Gígùn ọ̀rọ̀ aṣínà tó kere jù. Aiyipada 12 (bá Defender Pro mu). |
| $defender_active | `bool` | Bóyá Defender Pro Strong Password ń ṣiṣẹ́. |

### Láti ìgbà

- 2.4.0
### Orísun

A ṣàlàyé rẹ̀ nínú [`inc/class-scripts.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-scripts.php#L543) ní laini 543
