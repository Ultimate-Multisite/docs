# Àlẹmọ: wu_add_new_domain_modal_fields

Ṣe àlẹmọ àwọn pápá fún modal fífi domain tuntun kún.

Ó jẹ́ kí àwọn àfikún (fún àpẹẹrẹ Domain Seller) lè ṣàtúnṣe tàbí rọ́pò àwọn pápá ìforúkọsílẹ̀ domain.

## Àwọn Paramita

| Orúkọ | Irú | Àpèjúwe |
|------|------|-------------|
| $fields | `array` | Àwọn pápá fọ́ọ̀mù náà. |

### Láti

- 2.1.0
