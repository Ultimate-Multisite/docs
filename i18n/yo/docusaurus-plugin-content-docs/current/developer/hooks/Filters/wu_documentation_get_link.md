# Àlẹmọ: wu_documentation_get_link {#filter-wudocumentationgetlink}

Gba àwọn olùdàgbàsókè àfikún láàyè láti ṣe àlẹmọ àwọn ìjápọ̀ náà.

Kò dá mi lójú bí èyí ṣe lè wúlò, ṣùgbọ́n kò burú láti ní i

## Àwọn Pàrámità {#parameters}

| Orúkọ | Iru | Apejuwe |
|------|------|-------------|
| $link | `string` | Ìjápọ̀ tí a forúkọsílẹ̀ |
| $slug | `string` | Slug tí a lò láti gba ìjápọ̀ náà |
| $default_link | `string` | Ìjápọ̀ aiyipada tí a forúkọsílẹ̀ |
