# Àlẹmọ: wu_contains_element {#filter-wucontainselement}

Gba àwọn olùgbéejáde láàyè láti yí àwọn àbájáde ìṣàwárí àkọ́kọ́ padà.

Èyí wúlò fún àwọn olùkọ́ nkan ẹni-kẹta àti irú bẹ́ẹ̀.

## Àwọn Parameter {#parameters}

| Orúkọ | Iru | Àpèjúwe |
|------|------|-------------|
| $contains_elements | `bool` | Bí element náà bá wà nínú content náà. |
| $content | `string` | Content tí a ń ṣàyẹ̀wò. |
| $element | `self` | Element lọwọlọwọ. |
| $post | `null\|\WP_Post` | ìfiranṣẹ́ láti ṣàyẹ̀wò. |
