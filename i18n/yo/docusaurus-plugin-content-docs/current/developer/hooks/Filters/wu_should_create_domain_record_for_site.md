# Ajọ: wu_should_create_domain_record_for_site

Ó ń ṣàjọ bóyá Ultimate Multisite yẹ kí ó dá igbasilẹ domain fún site tí a ṣẹ̀ṣẹ̀ dá.

Lo ajọ yìí láti dènà tàbí sún dídá igbasilẹ domain aládàáṣiṣẹ́ síwájú fún àwọn site tó ń lo domain ipilẹ fọọmu isanwo ipari tí a pín, host inú, tàbí domain tí ìṣepọ̀ míì yóò ṣàkóso lọ́tọ̀.

## Àwọn Parameter {#parameters}

| Orúkọ | Iru | Àpèjúwe |
|------|------|-------------|
| $create | `bool` | Bóyá ó yẹ kí a dá igbasilẹ domain náà. |
| $site | `WP_Site` | Object site tí a ṣẹ̀ṣẹ̀ dá. |

### Láti {#since}

- 2.13.0

### Orísun {#source}

A ṣàlàyé rẹ̀ nínú `inc/functions/domain.php`.


## Ohun Tí Ó Dá Padà {#returns}

Boolean tó ń tọ́ka bóyá kí a dá igbasilẹ domain náà.
