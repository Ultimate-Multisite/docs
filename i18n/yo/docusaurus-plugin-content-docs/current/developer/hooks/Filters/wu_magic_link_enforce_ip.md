# Àlẹmọ: wu_magic_link_enforce_ip

Ṣe àlẹmọ bóyá kí ìjẹ́rìísí IP address jẹ́ dandan.

Ṣètò sí false láti jẹ́ kí tokens ṣiṣẹ́ láti orí networks tó yàtọ̀. Èyí dín ààbò kù ṣùgbọ́n ó mú ìrọ̀rùn lílò pọ̀ sí i (fún àpẹẹrẹ, fún àwọn oníṣe mobile tí ń yí networks padà).

## Àwọn Parameters {#parameters}

| Orúkọ | Iru | Àpejuwe |
|------|------|-------------|
| $enforce | `bool` | Bóyá kí ìbámu IP address jẹ́ dandan. |

### Láti {#since}

- 2.0.0
