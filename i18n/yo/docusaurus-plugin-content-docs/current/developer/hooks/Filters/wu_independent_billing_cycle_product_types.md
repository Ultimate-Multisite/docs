# Àlẹmọ: wu_independent_billing_cycle_product_types {#filter-wuindependentbillingcycleproducttypes}

Àlẹmọ àwọn iru ọja tí ó ní àwọn billing cycles olómìnira.

Àwọn ọja pẹ̀lú billing cycles olómìnira kì í jẹ́ kí wọ́n fi ipa mú wọn bá àkókò ìsanwó plan mu nínú cart. Èyí wúlò fún àwọn ọja bíi ìforúkọsílẹ̀ domain tí ó máa ń sanwó lọ́dọọdún nígbà gbogbo láìka bóyá plan jẹ́ oṣooṣù tàbí ọdún-ọdún.

## Àwọn Parameter {#parameters}

| Orúkọ | Iru | Àpèjúwe |
|------|------|-------------|
| $types | `array` | Array ti àwọn iru ọja pẹ̀lú billing cycles olómìnira. |

### Láti ìgbà {#since}

- 2.5.0
