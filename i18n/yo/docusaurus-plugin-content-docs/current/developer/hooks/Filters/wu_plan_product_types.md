# Àlẹmọ: wu_plan_product_types

Ṣe àlẹmọ àwọn irú ọja tí ó yẹ kí a kà sí àwọn plan.

Àlẹmọ yìí gba àwọn addon láyè láti forúkọsílẹ̀ àwọn irú ọja àfikún tí ó yẹ kí a mọ̀ sí àwọn plan nínú ìfọwọ́sí àti ìyànsọ́tọ̀.

## Àwọn Parameter {#parameters}

| Name | Type | Àpèjúwe |
|------|------|-------------|
| $plan_types | `array` | Array àwọn irú ọja láti kà sí àwọn plan. |
