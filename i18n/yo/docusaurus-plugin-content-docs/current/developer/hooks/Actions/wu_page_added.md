# Ìṣe: wu_page_added

Gba àwọn olùdàgbàsókè plugin láàyè láti ṣiṣẹ́ àwọn ohun àfikún nígbà tí a bá forúkọsílẹ̀ àwọn ojú-iwe.

Kò dà bí wu_page_load, èyí tí ó máa ń ṣiṣẹ́ nígbà tí a bá ń wo ojú-iwe kan pàtó nìkan, hook yìí máa ń ṣiṣẹ́ nígbà ìforúkọsílẹ̀ fún gbogbo ojú-iwe alábòójútó tí a ń fi kóòdù Ultimate Multisite ṣàfikún.

## Àwọn parameter

| Orúkọ | Iru | Àpèjúwe |
|------|------|-------------|
| $page_id | `string` | ID ojú-iwe yìí. |
| $page_hook | `string` | Orúkọ hook ojú-iwe yìí. |

### Láti

- 2.0.0
