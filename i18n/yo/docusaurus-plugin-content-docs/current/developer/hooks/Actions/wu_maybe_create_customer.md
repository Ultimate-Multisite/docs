# Ìṣe: wu_maybe_create_customer {#action-wumaybecreatecustomer}

Gba àwọn olùdàgbàsókè àfikún láàyè láti ṣe àwọn ohun míì nígbà tí a bá fi oníbàárà kún un.

Ibí nìyí tí a ti fi àwọn hook kún un fún fífi customer-&gt;user kún ojúlé àkọ́kọ́ náà pẹ̀lú, fún àpẹẹrẹ.

## Àwọn Párámítà {#parameters}

| Orúkọ | Iru | Àpèjúwe |
|------|------|-------------|
| $customer | `\Customer` | Oníbàárà tí ó ṣeé ṣe kí a ṣẹ̀dá. |
| $checkout | `\Checkout` | Kilasi ìsanwó-parí lọwọlọwọ. |
