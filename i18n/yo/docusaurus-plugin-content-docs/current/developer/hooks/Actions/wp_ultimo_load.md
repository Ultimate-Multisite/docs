# Iṣe: wp_ultimo_load {#action-wpultimoload}

Ó máa ń ṣẹlẹ̀ nígbà tí gbogbo àwọn ìgbẹ́kẹ̀lé bá ti kojọpọ̀

Ó gba àwọn olùdàgbàsókè àfikún láàyè láti ṣàfikún iṣẹ́ tuntun. Fún àpẹẹrẹ, ìtìlẹ́yìn fún àwọn olupèsè ìgbalejo tuntun, àti bẹ́ẹ̀ bẹ́ẹ̀ lọ.
