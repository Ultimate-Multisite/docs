# Ìṣe: wu_domain_mapping_register_filters

Àwọn àfikún kan máa fi URL pamọ́ kí mapping tó ṣiṣẹ́ tàbí wọn máa dá àwọn URL kalẹ̀ ní ọ̀nà míì tí kò wà nínú àwọn àlẹmọ lókè yìí.

Ní àwọn ọ̀ràn bẹ́ẹ̀, a fẹ́ fi àwọn àlẹmọ àfikún kún un. Párámítà kejì tí a fi ránṣẹ́ ni ìpè-padà mangle_url. A ṣàbẹ̀wò lòdì sí lílo àlẹmọ yìí taara. Dípò bẹ́ẹ̀, lo ọ̀nà Domain_Mapping::apply_mapping_to_url.

## Àwọn Párámítà

| Orúkọ | Iru | Àpèjúwe |
|------|------|-------------|
| $mangle_url | `callable` | callable mangle náà. |
| $domain_mapper | `self` | Nǹkan yìí. |

- 2.0.0
