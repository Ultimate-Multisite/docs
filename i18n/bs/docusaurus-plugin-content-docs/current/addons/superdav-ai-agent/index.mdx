# Superdav AI Agent

Superdav AI Agent je napredni WordPress AI asistent koji proširuje Gratis AI Agent vještinama svjesnim teme, poboljšanim mogućnostima internet pretrage, funkcijama graditelja stranice i opcionalnim integracijama automatizacije. Verzija 1.18.0 dodaje alate za čitanje Google Calendar, mapiranje kontakata i učesnika, kapije za ljudsko odobrenje, deduplikaciju podsjetnika, TextBee SMS obavještenja, upravljano obezbjeđivanje Superdav usluge i odvojeno distribuirani Advanced prateći paket za pouzdane developerske tokove rada.

## Ključne funkcije {#key-features}

- **Vještine svjesne teme** — automatski prilagođava smjernice i mogućnosti na osnovu vaše aktivne WordPress teme i instaliranih pluginova
- **Više pružalaca internet pretrage** — birajte između Brave Search i Tavily za bogatije rezultate pretrage i napredne istraživačke mogućnosti
- **Kontakt forme graditelja stranice** — kreirajte potpuno funkcionalne kontakt forme direktno iz interfejsa chata koristeći instalirani plugin za forme
- **Poboljšani graditelj stranice** — unaprijeđeni višefazni planovi izgradnje stranice s boljim otkrivanjem pluginova i oporavkom od grešaka
- **Ugrađene vještine za teme** — specijalizirane smjernice za Block Themes, Classic Themes, Kadence Blocks i Kadence Theme
- **Poboljšana WooCommerce integracija** — refaktorisana za korištenje izvornih WooCommerce API-ja radi bolje pouzdanosti i kompatibilnosti
- **Automatsko otkrivanje pružalaca** — automatski otkriva i osvježava dostupne pluginove i teme kada se aktiviraju ili deaktiviraju
- **Generisanje prilagođenog logotipa** — generišite i ugradite prilagođene SVG logotipe s automatskom sanitizacijom sigurnom za namespace
- **Validacija kontrasta palete** — provjerite parove boja za usklađenost s WCAG pristupačnošću prije primjene na vašu temu
- **Otkrivanje poboljšano fotografijama** — otpremite fotografije tokom intervjua za otkrivanje u Theme Builder kako biste uticali na smjer dizajna
- **Ugostiteljski meniji** — generišite strukturirane stranice menija hrane i pića za restorane, kafiće i ugostiteljske biznise
- **Desktop i mobilni pregledi** — pregledajte svoj dizajn na desktopu i mobilnim uređajima tokom odabira smjera dizajna
- **Scraping stranica nivoa 1** — sd-ai-agent/site-scrape je sada podrazumijevano dostupan za analizu konkurentskih i inspirativnih web stranica
- **Alati za čitanje Google Calendar** — čitajte konfigurisane kalendare i događaje kako bi automatizacije mogle zaključivati o kontekstu nadolazećeg rasporeda
- **Mapiranje kontakata i učesnika** — povežite učesnike kalendara s poznatim korisnicima stranice i kontaktima prije generisanja naknadnog rada
- **Kapije za ljudsko odobrenje** — pauzirajte osjetljive automatizacije radi pregleda prije slanja poruka, nastavka rada ili pravljenja izmjena
- **Zapisi podsjetnika i deduplikacija** — pratite podsjetnike koji su već poslani kako ponovljena pokretanja automatizacije ne bi dvaput obavijestila istog primaoca
- **TextBee SMS obavještenja** — šaljite konfigurisana tekstualna obavještenja putem TextBee kada je SMS pružalac omogućen
- **Upravljano obezbjeđivanje Superdav usluge** — povežite podržane stranice s hostovanim Superdav endpointima uz automatsko podešavanje umjesto ručne konfiguracije endpointa
- **Advanced prateći paket** — odvojeno distribuirani alati za pouzdane developere za fajl sistem, bazu podataka, WP-CLI, REST dispatcher, graditelj pluginova, git snapshot, upravljanje korisnicima i benchmark tokove rada

## Zahtjevi {#requirements}

- WordPress 6.0 ili noviji
- PHP 8.1 ili noviji
- WordPress Multisite (aktiviran na mreži) ili instalacija jedne stranice
- Aktivan plugin konektora AI pružaoca (Anthropic, OpenAI ili kompatibilan)
- Gratis AI Agent v1.9.0 ili noviji (Superdav AI Agent proširuje Gratis AI Agent)

## Instalacija {#installation}

1. Otpremite fajlove plugina u `/wp-content/plugins/superdav-ai-agent`
2. Aktivirajte plugin kroz ekran **Plugins** u WordPressu
3. Osigurajte da je Gratis AI Agent također aktiviran
4. Idite na **Gratis AI Agent → Settings** i konfigurišite svog pružaoca internet pretrage, vezu s upravljanom Superdav uslugom, Google Calendar ili TextBee akreditive prema potrebi

WordPress.org paket sadrži osnovne funkcije Superdav AI Agent. Advanced prateći paket se gradi i distribuira odvojeno za okruženja gdje su alati za pouzdane developere izričito instalirani i odobreni.

## Brzi početak {#quick-start}

Nakon aktivacije, Superdav AI Agent automatski poboljšava vašu Gratis AI Agent instalaciju vještinama svjesnim teme i dodatnim mogućnostima. Otvorite panel **Gratis AI Agent** u WordPress administratorskoj bočnoj traci i počnite koristiti nove funkcije.

**Primjeri uputa:**

```
Dodaj kontakt formu na stranicu Kontakt.
```

```
Pretraži web za najnovije najbolje prakse sigurnosti za WordPress.
```

```
Kreiraj hero sekciju koristeći blok obrasce aktivne teme.
```

```
Pregledaj sutrašnje konfigurisane događaje u Google Calendar i pripremi nacrte podsjetnika za učesnike.
```

```
Pošalji SMS podsjetnik nakon što odobrim poruku.
```

## Vještine svjesne teme {#theme-aware-skills}

Superdav AI Agent automatski otkriva vašu aktivnu temu i instalirane pluginove, učitavajući odgovarajuću vještinu svjesnu teme u kontekst asistenta. Dostupne vještine uključuju:

| Vještina | Primjenjuje se na |
|---|---|
| **Block Themes** | Teme koje koriste WordPress arhitekturu zasnovanu na blokovima (Full Site Editing) |
| **Classic Themes** | Tradicionalne WordPress teme koje koriste PHP šablone i klasični editor |
| **Kadence Blocks** | Stranice s aktivnim pluginom Kadence Blocks |
| **Kadence Theme** | Stranice s Kadence temom kao aktivnom temom |

Pogledajte [Vještine svjesne teme](../../user-guide/configuration/theme-aware-skills) za detaljne smjernice o svakoj vještini.

## Pružaoci internet pretrage {#internet-search-providers}

Superdav AI Agent v1.10.0 dodaje **Tavily** kao alternativu za Brave Search za upite internet pretrage. Oba pružaoca nude rezultate pretrage u stvarnom vremenu i mogu se konfigurisati u **Gratis AI Agent → Settings → Advanced**.

| Pružatelj | Funkcije | Podešavanje |
|---|---|---|
| **Brave Search** | Brzi rezultati pretrage usmjereni na privatnost | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Bogatiji rezultati pretrage, napredne mogućnosti istraživanja | [Tavily API](https://tavily.com/) |

Pogledajte [Internet pretraga](../../user-guide/configuration/internet-search) za upute za konfiguraciju.

## Tokovi rada za kalendar, odobrenje i obavještenja {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 proširuje automatizacijske tokove rada izvan zadataka sadržaja i izrade stranica:

- **Google Calendar alati** omogućavaju agentu da čita konfigurirane kalendare i detalje događaja kada su dostupne vjerodajnice kalendara.
- **Pomoćnici za kontakte i učesnike** mapiraju učesnike događaja na WordPress korisnike ili poznate kontakte prije pripreme podsjetnika ili naknadnih zadataka.
- **Kapije za ljudsko odobrenje** omogućavaju automatizaciji da se pauzira dok ovlašteni korisnik ne pregleda predloženu radnju.
- **Zapisi podsjetnika** sprečavaju duple kalendarske podsjetnike ili podsjetnike obavještenja kada se automatizacija ponovo pokuša ili ponavljano zakazuje.
- **TextBee SMS podrška** omogućava obavještenja tekstualnim porukama kada je TextBee pružatelj konfiguriran.

Koristite ove alate za pomoć svjesnu rasporeda, naknadno praćenje učesnika i tokove rada podsjetnika koji i dalje zahtijevaju ljudski pregled prije slanja osjetljivih poruka.

## Kontakt forme izrađivača stranica {#site-builder-contact-forms}

Agent za izradu stranica sada može kreirati kontakt forme direktno iz interfejsa za razgovor. Jednostavno zatražite od izrađivača stranica da kreira kontakt formu, i on će automatski otkriti vaš instalirani plugin za forme (Contact Form 7, WPForms, Fluent Forms ili Gravity Forms) i generirati formu spremnu za upotrebu.

Pogledajte [Kontakt forme izrađivača stranica](../../user-guide/administration/site-builder-contact-forms) za detaljne smjernice.

## Konfiguracija {#configuration}

Sve postavke su pod **Gratis AI Agent → Settings** u WordPress administraciji. Superdav AI Agent dodaje sljedeće opcije konfiguracije:

| Postavka | Opis |
|---|---|
| **Pružatelj internet pretrage** | Odaberite između Brave Search i Tavily (opcionalno) |
| **Tavily API Key** | API ključ za Tavily internet pretragu (ako je Tavily odabran) |
| **Upravljana Superdav usluga** | Povežite podržane stranice s hostovanom Superdav uslugom uz automatsko provisioniranje endpointa |
| **Google Calendar vjerodajnice** | Konfigurirajte pristup kalendaru kako bi agent mogao čitati kalendare i događaje |
| **TextBee SMS pružatelj** | Konfigurirajte TextBee vjerodajnice za odobrene tokove rada SMS obavještenja |

## Napredni prateći paket {#advanced-companion-package}

Superdav AI Agent v1.18.0 dodaje zasebno distribuiran napredni prateći paket za pouzdane developerske tokove rada. Ove mogućnosti nisu uključene u osnovni WordPress.org paket. Instalirajte prateći paket samo na stranicama gdje administratori izričito vjeruju agentu da pristupa developerskim alatima.

Napredni alati uključuju operacije nad datotečnim sistemom, pristup bazi podataka, WP-CLI, tokove rada REST dispečera, pomoćne alate za izradu pluginova, git snimke stanja, pomoćnike za upravljanje korisnicima i alate za benchmark. Ograničite ove alate na pouzdane administratore i pregledajte konfiguraciju kapije odobrenja prije nego što ih omogućite na produkcijskim stranicama.

- Ispravljena automatizacija događaja tako da elegantno rukuje web stranicama gdje tabele automatizacije još nisu kreirane
- Ispravljena mogućnost `memory-save` koja koristi ispravan prefiks imenskog prostora u graditelju sistemskih uputa
- Ispravljeni skalarni rezultati alata tako da se ispravno umotaju prije vraćanja AI-ju
