# Admin Page Creator

## Content
Admin Page Creator vam omogućit će da lako dodate nove admin stranice i podstranice unutar WordPress admin dashboarda. Može se koristiti za kreiranje stranice dobrodošlice ili stranice s uputstvima, koja može poslužiti kao početna stranica za nove korisnike. Možete koristiti page builder kao što su Beaver Builder, Elementor, Brizy i Oxygen za dizajniranje stranica bez komplikacija.

Također nudi podršku za custom HTML, CSS i JavaScript, kao i kontrolu pristupa zasnovanu na planu i ulozi korisnika!

## Features
* Podrška za Custom HTML i CSS
* Podrška za JavaScript
* PHP Content Mode
* Dopuštenje zasnovano na ulozi korisnika
* Podrška za page builders

[Creating Custom Admin Pages and Sub-pages using Multisite Ultimate: Admin Page Creator](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## Installation

1. Upload **ultimate-multisite-admin-page-creator** u direktorij **/wp-content/plugins/**
2. Aktivizirajte plugin kroz meni **Plugins** u vašem WordPress Network Adminu

## Usage Guide

Iako WordPress čini izuzetno lako dodavanje sadržaja na prednji kraj sajtova, kreiranje stranica i podstranica na admin strani zahtijeva neke osnovne vještine kodiranja. Ultimate Multisite: Admin Page Creator teži da donese istu lakoću korištenja sa prednjeg kraja na stražnji kraj vaših mrežnih sajtova.

## Why use Admin Page Creator?

Možda želite ponuditi prilagođenu stranicu na admin panelu vaših klijenata sa uputstvima, možda želite dodati prilagođenu stranicu sa FAQ-ovima i widgetom za kontakt. Mogućnosti su neograničene sa Admin Page Creatorom, i to ćemo istražiti u današnjem tutorijalu.

## Creating your first Custom Admin Page

Nakon instaliranja i aktiviranja Ultimate Multisite: Admin Page Creator, idite na meni vašeg network admina i odaberite Admin Pages.

![Custom Admin Pages list on Network Admin](/img/admin/admin-pages-list.png)

_Lista prilagođenih admin stranica na Network Adminu_

Vidjet ćete listu svih admin stranica koje ste do sada kreirali (jer je ovo vaš prvi posjet, lista će biti prazna). Slobodno kliknite na **Add new Admin Page**. Preusmjerat ćete se na editor Admin Page.

## The elements of a Custom Admin Page

Na editoru Admin Page, imaćete niz različitih opcija za prilagođavanje vaše stranice.

### Page Title

Ovo je naslov koji će biti prikazan na vrhu stranice, a ne oznaka u admin meniju.

### Content Type Selector

Ultimate Multisite: Admin Page Creator vam nudi dvije opcije kada je riječ o sadržaju stranice: možete koristiti podrazumjetveni WordPress WYSIWYG editor ili prebaciti na HTML editor.

Ako niste upoznati sa HTML-om, držite se podrazumjetevnog WordPress editora. Bit ćete u stanju uređivati sadržaj stranice baš kao što biste to učinili sa normalnim Postom ili Page-om.

## Menu Options

Na bočnoj strani, u meta-boxu General Options, bit ćete u stanju konfigurirati tip menija, oznaku i mnogo više.

### Menu Types

Vaša nova prilagođena admin stranica može imati dva različita tipa menija: glavni meni (top-level menu) i podmeni (submenu).

Glavni meni, kako ime sugeriše, dodavati će vašu novokreiranu stranicu u admin meni pored podrazumjetvenih WordPress stranica.

Ako odaberete opciju glavnog menija, bit ćete u stanju odabrati redoslijed menija i ikonu menija.

Ako želite da ova stranica bude podmeni, bit ćete u stanju odabrati roditeljsku stranicu na koju ćete priložiti ovu stranicu. Možete dodati svoju prilagođenu stranicu na podrazumjetvene WordPress stranice ili na vaše glavne prilagođene admin stranice.

## Advanced Options

Trebali biste biti u stanju dodati sve vrste sadržaja na vaše prilagođene admin stranice. To uključuje CSS pravila i fajlove, JavaScript i eksterne biblioteke. To možete učiniti koristeći karticu Advanced Options.

### Custom CSS

Na CSS kartici, možete dodati svoj vlastiti custom CSS kod, kao i uvesti CSS fajlove sa udaljenih sajtova (poput CDN-ova, Google Fonts, itd).

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

### Custom JavaScript

Isto važi za JavaScript. Možete dodati eksterne biblioteke koje želite koristiti u vašem custom JavaScript kodu.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Dodajte svoj vlastiti JavaScript kod!_

## Permissions

Takođe u Advanced Options, bit ćete u stanju odabrati koji će pod-sajtovi imati pristup ovoj stranici. Ovo je korisno ako planirate da servirate različit ekskluzivan sadržaj različitim nivoima planova, na primjer.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Imate potpunu kontrolu nad onako gdje će se ova stranica prikazati_

**Važno**: Oba uslova moraju biti ispunjena da bi stranica bila prikazana. To znači da ako je korisnik klijent određenog plana, ali nema jednu od odabranih uloga (ili obrnuto), stranica neće biti prikazana na njegovom panelu.

## Active

Ako želite da onemogućite custom admin stranicu bez brisanja ili miješanja sa dopuštenjima, samo skine označavanje sa opcije Active.

Deaktiviranje glavne stranice automatski će deaktivirati sve podstranice pridružene toj roditeljskoj stranici.
