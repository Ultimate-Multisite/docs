# Material WP

WordPress admin interfejs je izgledao isto godinama. Ali jesu li se ikada pitali kako bi izgledala Material Design verzija tog interfejsa?

Material WP je admin tema razvijena da vam pruži taj izgled i osjećaj, pomažući vam da vašim klijentima ponudite prilagođeniji Dashboard.

## Funkcionalnosti {#features}

- Podrška za Custom CSS
- Lako podešavanje stila
- Preuređivanje menija
- Lako uvoz/izvoz

## Dokumentacija {#documentation}

**Prilagođavanje vašeg Admin Panela sa Material WP**

## Instalacija {#installation}

1. Učitajte 'material-wp' u direktorij `/wp-content/plugins/`
2. Aktivizirajte plugin putem menija 'Plugins' u WordPressu

## Vodič za korištenje {#usage-guide}

WordPress admin interfejs je izgledao isto godinama. Ali jesu li se ikada pitali kako bi izgledala Material Design verzija tog interfejsa?

Nakon **instalacije**, idite na stranicu _Plugins_ i kliknite da ga aktivirate na vašoj Mreži (Network).
