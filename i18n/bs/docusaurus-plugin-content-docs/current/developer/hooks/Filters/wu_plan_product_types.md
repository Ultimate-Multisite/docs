# Filter: wu_plan_product_types {#filter-wuplanproducttypes}

Filtrira tipove proizvoda koji bi trebali biti tretirani kao planovi.

Ovaj filter omogućava dodacima (addons) da registruju dodatne tipove proizvoda koji bi trebali biti prepoznati kao planovi tokom validacije i segregacije.

## Parametri {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | Niz tipova proizvoda koji treba tretirati kao planove. |

- 2.3.0
