# Filter: wu_stripe_existing_product_id

Filtrira ID proizvoda koji se provjerava. Ako postoji, nova pretplata će koristiti ovaj proizvod.

## Parametri {#parameters}

| Name | Type | Opis |
|------|------|-------------|
| $product_id | `string` | ID proizvoda Stripe-a koji se provjerava. |
| $name | `string` | Ime proizvoda. |
