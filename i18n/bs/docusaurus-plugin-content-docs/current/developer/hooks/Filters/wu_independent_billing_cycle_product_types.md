# Filter: wu_independent_billing_cycle_product_types {#filter-wuindependentbillingcycleproducttypes}

Filtrira proizvode čije imaju nezavisne periode naplate.

Proizvodi sa nezavisnim periodima naplate ne moraju se slagati sa periodom naplate plana u korpi. Ovo je korisno za proizvode poput registracija domena koji se uvijek naplaćuju godišnje, bez obzira na to je plan mjesečni ili godišnji.

## Parametri {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | Niz (array) tipova proizvoda koji imaju nezavisne periode naplate. |

- 2.5.0
