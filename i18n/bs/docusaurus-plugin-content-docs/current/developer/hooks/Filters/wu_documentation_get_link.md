# Filter: wu_documentation_get_link

Omogućava programerima pluginova da filtriraju linkove.

Nisam siguran kako bi to moglo biti korisno, ali ne škodi imati ga.

## Parametri

| Ime | Tip | Opis |
|------|------|-------------|
| $link | `string` | Registrovani link |
| $slug | `string` | Slug koji se koristi za dohvaćanje linka |
| $default_link | `string` | Registrovani podrazumevani link |
