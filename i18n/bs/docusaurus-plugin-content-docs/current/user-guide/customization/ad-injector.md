# Postavljanje oglasa na vašu mrežu s Ultimate Multisite: Ad Injector {#placing-ads-on-your-network-with-ultimate-multisite-ad-injector}

Ultimate Multisite: Ad Injector je **besplatni dodatak** koji vam daje alate za postavljanje oglasa na vaše planove, direktno sa njihove stranice za uređivanje.

Nakon instaliranja Ultimate Multisite: Ad Injector, idite na stranicu _Plugins_ da ga aktivirate.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Kliknite na gumb Network Activate_

Kada je dodatak aktivan, u okviru _Advanced Options_ na ekranu za uređivanje plana pojavit će se nova opcija nazvana _Ad Settings_.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_Sada možete uređivati postavke oglasa za svaki od vaših planova_

Na toj kartici, moći ćete dodati kodove oglasa za objave (posts) i stranice (pages) na prednjoj strani (front-end), prije i poslije sadržaja.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Nakon umetanje vašeg ad koda, on će se prikazati na objavama i stranicama tog specifičnog plana._

Također možete uključiti oglase u zaglavlje Admin stranica za korisnike tog plana.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Idite na opciju Admin Panel Ad Code ako želite prikazati oglase na korisnikovom backendu_

### Izvan oglašavanja {#beyond-advertising}

Također možete koristiti Ultimate Multisite: Ad Injector za pružanje podrške premium planovima.

Samo idite na opciju _Admin Panel Ads_ i dodajte HTML kod za vaš alat za podršku.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Dodajte korisničku podršku specifičnim planovima_

Na ovaj način ljudi vam mogu doći do direktno sa svog Admin Panela!
