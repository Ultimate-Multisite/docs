# Ultimate Multisite Hooks i filters

Ultimate Multisite sastoji se od ogromne količine funkcija i upita baze podataka. Ima hooks i filters koji će vam omogućiti da modifikujete ili dodate novi kod postojećim funkcijama.

Napravili smo listu ovih hooks-a i filtersa kao referencu kako bismo pomogli naprednim korisnicima da još više prilagode svoju mrežu. Listu možete pronaći na Githubu klikom na link ispod.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
