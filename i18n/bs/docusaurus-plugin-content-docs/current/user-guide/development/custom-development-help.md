# Kako dobiti prilagođenu razvojnu pomoć od Codeable-a?

Ponekad nam je zaista potrebna neka odlična prilagodba kako bismo našu mrežu učinili tačno onako kako želimo. Ako tražite nešto slično, preporučujemo [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), mjesto gdje možete pronaći iskusne developere sa ekspertizom za Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) je platforma koja povezuje klijente sa kvalifikovanim WordPress freelance developerima, sposobnim za rad na projektima bilo koje veličine. Ima pažljiv proces prijave kako bi osigurala da vas upare sa stručnjacima koji su specifično kvalifikovani za zadatak koji imate na umu.

### Kako kreirati projekat u Codeable-u? {#how-to-create-a-project-in-codeable}

Prvo, idite na [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) i kliknite na dugme **Start a Project**.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Sada, da biste podnijeli projekat, kliknite na lijevo na **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

Pod "I need help with", izaberite **Customizing** :

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

Pod "My", izaberite **Plugin** :

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Zatim kliknite na dugme **Continue**.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Sada je vrijeme da opišete svoj projekat i potrebe na najbolji mogući način. Pokušajte da date što više informacija!

<!-- Screenshot unavailable: Codeable project description form -->

Odgovorite na sljedeća pitanja kako bi se pomoglo sa određivanjem cijene.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Nakon toga, ako još nemate [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) nalog, kreirajte ga i kliknite na dugme **Publish My Project**.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

Nakon što objavite svoj [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) projekat, platforma će vas povezati sa developerima koji odgovaraju setu vještina potrebnih za vaš projekat.

Ovaj proces će pomoći da se pronađe prava osoba za vaš posao, koja ima iskustva sa WordPressom i Ultimate Multisite-om!

Minimalna tarifa Codeable-a je od 70–120 USD po satu. Procjene će se mijenjati u zavisnosti od obima posla, složenosti i hitnosti projekta.

Kada dobijete procjenu, moći ćete da angažujete developera koji najbolje odgovara vašim potrebama i započnete svoj projekat zajedno!
