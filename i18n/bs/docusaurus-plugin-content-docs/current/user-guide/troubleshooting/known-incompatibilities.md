# Poznate nekompatibilnosti s drugim pluginovima

Nakon instaliranja i aktiviranja Ultimate Multisite-a, možete naići na probleme zbog neadresiranih nekompatibilnosti s drugim pluginovima.

Aktivno radimo na smanjenju ove liste sa svakim novim izdanjem, ali ako je plugin naveden ovdje, možda ćete morati da ga deaktivirate kako biste imali potpuno funkcionalnu Ultimate Multisite mrežu.

Evo liste tih pluginova i šta trebate učiniti u ovim slučajevima:

**Neki WPMU DEV pluginovi** Neki pluginovi koje WPMU DEV _više ne_ održava poznato je da uzrokuju zaustavljanje procesa registracije Ultimate Multisite-a na zadnjem koraku (korisnik nije u stanju kreirati račun). Evo liste pluginova koji uzrokuju te probleme:

  * Signup Language: _Ultimate Multisite ima dodatak koji nudi istu funkcionalnost, nazvan Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Ovo je ugrađeno u Ultimate Multisite, pa nije potreban dodatni plugin._

  * New Blog Templates: _Ovo je ugrađeno u Ultimate Multisite, pa nije potreban dodatni plugin._

  * Set Password on Multisite Blog Creation: _Ovo je ugrađeno u Ultimate Multisite, pa nije potreban dodatni plugin._

  * Signup TOS: _Ovo je ugrađeno u Ultimate Multisite, pa nije potreban dodatni plugin._

  * Anti-Splog: Iako je manje moćan, on je takođe _ugrađen u Ultimate Multisite._

**Napomena** : Sve ostale WPMU Dev pluginove možete koristiti normalno uz Ultimate Multisite. Primjeri uključuju _Smush_ , _Forminator_ , _Defender,_ itd.
