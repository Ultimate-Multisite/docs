# Kako mogu pristupiti stranici svog računa prvi put? {#how-can-i-access-my-account-page-for-the-first-time}

Dobrodošli u Ultimate Multisite!

Pristup stranici svog računa putem linka koji ste primili u e-mailu nakon kupovine našeg plugin-a:

<!-- Screenshot unavailable: Account page email with access link -->

Koristite e-mail adresu koju ste naveli prilikom kupovine i lozinku koju ste primili u e-mailu.

<!-- Screenshot unavailable: Login page with email and password fields -->

Budući da je ovo vaš prvi prijavni pokušaj na ovaj račun, bit ćete traženi da promijenite lozinku. Promijenite je i kliknite na gumb _Save_.

<!-- Screenshot unavailable: Change password form with Save button -->

Na stranici svog računa, možete upravljati svim detaljima vašeg računa, kao što su naplata, obnove, otkazivanja i mnogo drugih stvari.
