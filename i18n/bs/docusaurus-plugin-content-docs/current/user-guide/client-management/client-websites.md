# Web sajtovi vašeg klijenta (v2) {#your-clients-websites-v2}

Nakon registracije na vašoj mreži, vaši klijenti mogu pristupiti svom dashboardu.

Pored stranice **Account**, gdje mogu vidjeti važne informacije o plaćanjima, članstvima, domenima, ograničenjima plana i više, imaju pristup i stranici **My Sites**, gdje mogu vidjeti i upravljati svim svojim web sajtovima. Sve što moraju učiniti je kliknuti na My Sites na svom dashboardu.

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

Tu će biti prikazani svi njihovi web sajtovi, sa gumnom **Manage** odmah ispod svakog od njih. Ako kliknu na tu gumnu, bit će preusmjereni na **dashboard tog specifičnog web sajta**.

Također mogu vidjeti koji je njihov web sajt **primarni** za planove koji im dozvoljavaju više od jednog web sajta.

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

Evo punog prikaza interfejsa za uređivanje sajta:

![Site edit full interface](/img/admin/site-edit-full.png)

Ako žele promijeniti svoj primarni web sajt, moraju otići na stranicu **Account** klikom na meni **Account**.

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

Također možete pregledati i uređivati podatke o klijentima:

![Customer edit interface](/img/admin/customer-edit.png)

Zatim, trebali bi kliknuti na **Change Default Site**.

![Change Default Site button on account page](/img/account-page/change-default-site.png)

Pojavit će se pop-up prozor. Kliknite na padajući izbornik i odaberite koji web sajt želite da učinite primarnim.

![Change Default Site dropdown selection](/img/account-page/change-default-site-dropdown.png)

Zatim, samo kliknite na **Change Default Site** i gotovo!

![Confirming the default site change](/img/account-page/change-default-site-confirm.png)
