# Auto Search and Replace in Site Templates (v2)

_**Ez dago WP UItimo bertsio 2.x behar da.**_

Ultimate Multisite-aren oso potentziala duen funtuari bat da, registratzeko formularioan testu, kolor eta lapurriak erabilera emateko. Horrek datua hartuta gabe, site template-aren bestelako parteetan kontuentu ahalegin pre-poblatzeko benutzarekin jartzen dugu. Horrek, leku berria publikatu ondoren, Ultimate Multisite-ak placeholder-ek (lekuak) erabat informazioa hartuta iragilduko izango dira.

Adibidez, template-ek placeholder-ein egin dezakezu. Placeholder-ek doble curly braces-ek aurrera egin behar dira - {{placeholder_name}}.

Horren ondoren, hori datua hartzeko ekuipot bat erabiliz bere registratzeko lapurialdia gehiago emateko da.

Gero, klienteak registrazioan hori lekua amaitu dezaketak.

Ultimate Multisite-ak hori datuak klientearen aurretik ematen duen informazioarekin otomatikoki placeholder-ek erabat hartuko dira.

## **"placeholder-ek ematuta template" problemazia iradokizuna**

Hau guztia da amaiginta, baina ez du gertatzen: site template-ek - klienteak bisitatzen dituztenak - placeholder-ek oso horriak dago, eta ez dute askotan informazioa ematen.

Horri konponduko, placeholder-ek artean falsu jarduerak emateko opszioa ematen dugu, eta horiek datuak erabiliz template site-etan placeholder-ek kontuentuak bilatu eta erabat hartuko da klienteak bisitatzen dituzten bitelako.

Template placeholder-ek editorebi zuzenean joan duzu **Ultimate Multisite > Settings > Sites** funtuationera, Site Template Options (Site Template Opzioak) zeisten, eta horrek klik egin behar duzu **Edit Placeholders** (Placeholder Editatu) linkuaren.

![Site Template Options area in the Sites settings page](/img/config/settings-sites-templates-section.png)

Horrek placeholder-ek kontuentu editorebi jarraituko duzu, horretan placeholder-ek eta horiek erabilera ematen kontuentuak gehiago emateko da.

![Template placeholders editor entry point](/img/config/settings-sites-templates-section.png)
