# Planak Linkakatuaren Eskaintza Kontrolatzea {#managing-shareable-links-for-plans}

Ultimate Multisite-ek planak linkak direktamente client-ekin iritsi dezake. Horrek egin dezakezu dituzteko, plan bat eduki editatzia eskuratu eta "_Shareable Link kopioatzeko klikatu_ ("Click to copy the Shareable Link") botoia klikatu behar duzu.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_Planaren Editatzian, shareable linkak sortzeko botoia aurkitu ezazu._

Ez dutela mekanismoa default billing frequencya (sortu-gaindoko iraupenderio) erabiltzen du linka sortzeko. Horrek esan nahi du, batezian default billinga mensual dela, adibidez, azpiegitura, linka araua izango da: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Shareable linkak quarterly (kuartaliko) gisa defaultatzea nahi du, 3 leizten apenderri behar duzu.

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Plan _premium_a yearly (labirrialdia) erabiliteko, 1 eta horrek da leitzia gehitusi:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Ez dutela bat linka-kodiak (billing frequencies) ezberdinetik bat izango du. Billing frequency baten artean, batezian, linka zehatua bat emate behar duzu.

Linka-kodiak client-ekin iritsi denean, horrek sizista-ren lehen urtera irekituko da.
