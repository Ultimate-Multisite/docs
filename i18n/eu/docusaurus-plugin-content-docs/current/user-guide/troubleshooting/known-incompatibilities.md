# Ezpotikak ondoriozko beste Plugin-ek direne {#known-incompatibilities-with-other-plugins}

Ultimate Multisite instalatu eta aktibatu egin ondoren, beste plugin-ekin ez dagoen ondoezpenak gertatzen dira.

Beste edukiak ebazpenak azaltzeko prozesuarekin, lehen daura hasi duena, baina plugin bat listan dagoenean, Ultimate Multisite Network guztiz funtzionatzeko deia izango da hori desaktibatu behar izan dugu.

Hona hemen lista bat eta horiek gertatzen dutelako kasuak zer egin behar duen:

**Bazun WPMU DEV plugin-ek** Plugin-ek batzuk, zehatik ez dagoen WPMU DEV-eko manutenzionea eskatzen ditelako, Ultimate Multisite-aren registro prozesua lehen edo beste irudiari (testu edo kontu emateko) atalarekin arazoak sortzen du. Horrek problema hauek sortzen duten plugin-ek lista bat da:

  * Signup Language: _Ultimate Multisite-ek hori bere funtzioa eskaintzen duen add-on bat dago, eta hori Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Hori Ultimate Multisite-aren parte batean dago, gaur egun beste plugin bat behar ez du._

  * New Blog Templates: _Hori Ultimate Multisite-aren parte batean dago, gaur egun beste plugin bat behar ez du._

  * Set Password on Multisite Blog Creation: _Hori Ultimate Multisite-aren parte batean dago, gaur egun beste plugin bat behar ez du._

  * Signup TOS: _Hori Ultimate Multisite-aren parte batean dago, gaur egun beste plugin bat behar ez du._

  * Anti-Splog: Maliagoa txikiago da, baina hori ere _Ultimate Multisite-aren parte batean dago._

**Nota**: Beste WPMU Dev plugin-ek normalmentari Ultimate Multisite-arekin egiteko gabe dira. Oinarriak hauek dira: _Smush_, _Forminator_, _Defender_, eta beste batzuk.
