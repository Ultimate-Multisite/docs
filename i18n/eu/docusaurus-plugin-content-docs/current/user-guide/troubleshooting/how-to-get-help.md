# Zerriz laguntza esan {#how-to-get-help}

Ezinbestalde, lehen erabilgarri License key-ak dut, eta horrekin ekartu da moduirean ekipua berri ezarritzean Ultimate Multisite erabiltzen duen galera eta problema hauek konpondu eta iradokizuneko aurretik gertatzen dituzteko laguntzeko.

Hauko lekurriaren eskolapenari, license holder-ek jasotzen dituzten suporta kanaletak eta ekipua kontaktu dezakekin beharrezkoa izango da informazio zehat ezagutzea.

## Laguntza Saila (Help Site) {#help-site}

Linku honetan aurkitzen dituen laguntza sailan artikulu eta tutorial askore erabil dezakezu.

[)

Erabiltzaile-ek Ultimate Multisite-aren espezial funtzioak eta funtzioak ezagutzea gogoratzen dugu. Laguntza sailan "Getting Started" (Zerriz hasi) artikulu daude, eta besteak osi, add-onak instalatzeko moduan edo payment gateway-ak integratzeko moduan nola egin behar da, bezala eskaera berehala aurkitu nahi duzu.

## Suporta Kontatu {#contact-support}

Laguntza sailan erantzuna ez aurkitu eta horrek ez dutenean, ekipua kontaktu dezakezu web-sailaren esku-laporta (lower right side) aukera ikona klikizinean edo direktamente support@ultimatemultisite.com emaila bidali.

Tekniko problema bat eskaintzeko suportara kontaktatzen duzenean, problema zehat deskribatu eta screenshotak, error messageak edo laguntzak gisa informazioak emate zai behar duzu, hau da, ekipua lehtëna jakiteko eta balioa jokatzeko. Bere license key-ak ere ematea behar duzu. Aktibo license holder-ek dira suportarekin eskatzen dituztenak.

## Komunitate Saila (Community Site) {#community-site}

Ultimate Multisite erabiltzaile-ek besteekin interakzio egiteko leku da [community site](https://community.ultimatemultisite.com/). Horra informazio sistema bat dela, aholatutako diskutiak postatu, komentatu, aholkatzen, berain kolaboratzen edo crowdsource egin dezakezu. Itxarren pribatu espazio/pagine desblozkatzeko Ultimate Multisite license key-ak ematea aurrerateko behar duzu.
