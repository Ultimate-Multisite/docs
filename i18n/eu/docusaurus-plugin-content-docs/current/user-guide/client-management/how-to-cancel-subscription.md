# Ondo ez dauden abonazioa cancelatu? {#how-can-i-cancel-my-subscription}

Ultimate Multisite lisentzia duzuz, hori zehar gurean jakin duen abantzukiz cancelatuta ditzake. Lehen daude askotan:

<!-- Screenshot unavailable: Account page email with access link -->

Erabiltu email-a ematen dut irakurriaren aurretik eta password-a.

<!-- Screenshot unavailable: Login page with email and password fields -->

Hau da zure Account Page-aren lehen ekrania:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

Zure abonazioa cancelatuz, eskurriari menukuan (izquierdaan), _Renewing & Billing._ funtziara jarraitu.

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

Adiaren (handiaren) partean dagoen arrow-ra klikatu eta lapurtu ikusi finestra bat lortu duzu. Horrek, **auto-renew** cancelatzea aukera duen opsioa ematen duzu.

Sistemaa leku bat konfirmazio mezu bat erakituo duzu.

Abonazioa cancelatuta jakin duzu nahi du, _**Cancel Renewals**_ botoia klikatu. Horren ondoren, lapurtu iruzte bat ematen dituzte.

Maiz! Abonazioa auto-renew ez daude.

**Zure clavea abonazioa geratzen duen erabilera hasi egiten du**. Abonazioa berriro aktiboa egin nahi du, handiaren partean manuaz egin behar duzu.
