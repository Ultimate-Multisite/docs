# Zerren ez daite akun-lekurarekin irki? {#how-can-i-access-my-account-page-for-the-first-time}

Ultimate Multisite-r gorri etorri!

Plugin-a erakusten duen e-mailan ematen linkuaren bidez akun-lekurara jartzen dituzte:

<!-- Screenshot unavailable: Account page email with access link -->

Erabiltzen duzu irakusteko e-maila eta e-mailan ematen pasordea.

<!-- Screenshot unavailable: Login page with email and password fields -->

Hau akun-lekurara lehen aldiz jartzen ari zaitu, pasordea aldatzea gautu dute. Aldatu eta _Save_ (Salbatu) botoia klikatu.

<!-- Screenshot unavailable: Change password form with Save button -->

Akut-lekuraren daitean, fakturazioa, berri edinenak, cancelariak eta gehiago beste guztietari jakinarazteko akun-lekurako datu horiek eskaintzen dituzte.
