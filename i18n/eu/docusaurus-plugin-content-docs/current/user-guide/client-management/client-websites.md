# Itskiz Lehenaren Kontseilak (v2) {#your-clients-websites-v2}

Sie networkan egin ondoren, kliente bere dashboard-era erabilariak jartzen dira.

**Account** lapuriko edozein informazio espero jakinarazten dituzte, galdirek, domain-ek, planaren limitazioak eta gehiago ezagutzeko, **My Sites** lapuriko ere jartzen dute, horiek guztietan bere websteileiak ikusi eta eskaintzea posible da. Horrek egin behar duena da dashboard-ean My Sites-era klik egin.

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

Websteileia guztiek erakusten dira, eta horiek guztietan **Manage** (Eskaintzea) botoi dago. Horrek klik egin ondoren, hori bere websteilearen **dashboard-era** irekitzen da.

Gehiago websteile batzuk dugu planak erabiltzeko aukera duen planetan, zein websteilea **primary** (garrantzitsuena) dela ere ikusi dezatat.

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

Hau dago websteile editatzeko interfazaren guztia:

![Site edit full interface](/img/admin/site-edit-full.png)

Bere websteilea primary bat gehitzen nahi du, **Account** menukoa klik egin behar du.

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

Klientearen datuak ere ikusi eta editatzea posible da:

![Customer edit interface](/img/admin/customer-edit.png)

Gero, **Change Default Site** (Default Websteilea Gehitzea)-era klik egin behar du.

![Change Default Site button on account page](/img/account-page/change-default-site.png)

Pop-up ikunari bat erakusten da. Drop-down listan klik egin eta zein websteileak primary websteilea egin nahi duela aukeratu.

![Change Default Site dropdown selection](/img/account-page/change-default-site-dropdown.png)

Gero, **Change Default Site** (Default Websteilea Gehitzea)-era klik egin eta gertuta da!

![Default sitio haseraren bidez konformatu](/img/account-page/change-default-site-confirm.png)
