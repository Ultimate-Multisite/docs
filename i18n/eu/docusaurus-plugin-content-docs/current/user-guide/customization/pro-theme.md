# Administrazioa irudiari PRO Theme-ekin

Klient ziurtak egiten duenean, WordPress-en administrazio panelan (dashboard) oso espezialbidearen eragin bat emateko beharrak jartzen dugu. Hau lehen aurrera, Ultimate Multisite-ren premium plugin-ekaren parte bat da PRO Theme.

Install egin ondoren, _Plugins_ zehalan jarraitu eta Network-an aktibatu (activate) egin.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Network-an PRO Theme-a aktibatu_

_**NOTIZIA:**_ _Hau lehen aurrera installatuta dutenean, client-ekari espezialitate gisa erakusten da, eta hori erabiltzaile batzuk-k bitartean theme-a hasterain emateko (markatzeko) konditionala da._

Arriba, administrazio interface-a guztiz diren rediseñatu dira! Lezte kolona eskurri duzu PRO Theme.

<!-- Screenshot unavailable: PRO Theme redesigned admin panel interface -->

_Administrazio Panel-eko berria irudi bat_

Hau guztietan, estilaren elementuak aktibatu edo desaktibatu dezakezu, funksionalitateak emate edo blokea egin dezakezu, eta client-ekari espezialbidearen eragin bat emateko konditionala aukerak emateke dituztezu network-an dagoen situak.
