# Reklamak leku networkanare Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector da **plugina libre** daiko dio, zein daiko planekonaren edit maila ere reklamak egindatzeko herramientaek ematen du.

Ultimate Multisite: Ad Injector instalatu ondoren, _Plugins_ mailari joan eta aktibatuu behar da.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Click the Network Activate button_

Pluginak aktibatuta jakinai duenean, plan edit ekraneko _Advanced Options_ meta box-ean "Ad Settings" (Reklama Kontrolak) dela eta daiko leku bat da.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_Aurreratuta, zein planekari ere reklamaren kontroldunak editu dezakezu._

Horren tabean, front-endeko post eta page-etan ad codeak (reklamaren kodeak) contentaren aurrera, aurrera eta ondoren ezartzeko lekuak dira.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Ad codeak egindatu ondoren, hori planaren post eta page-etan erabilizteko da._

Bere lekuen erabiltzaileekara Admin Pages-en header-ean ere reklamak hartzeko aukera ditu.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Erabiltzailearen back-end-an (admin panel) reklamak erabilizte nahi duzu, Admin Panel Ad Code lekuan joan._

### Reklamazko Gainara

Ultimate Multisite: Ad Injector daiko planak oso garrantzitsuakari suportu emateko ere erabilizte dituz.

_Admin Panel Ads_ lekuan joan eta suporta tool-eko HTML kodeak gehitu behar duzu.
