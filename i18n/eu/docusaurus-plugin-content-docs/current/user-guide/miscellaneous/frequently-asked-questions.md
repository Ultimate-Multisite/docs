# Aipatzen Dutilaketa {#frequently-asked-questions}

Hona da **Aipatzen Dutilaketa** (Frequently Asked Questions) hauek, zein irudien da hautatzen dituz beti.

  * **Zer da Ultimate Multisite-a?**

Ultimate Multisite da WordPress plugin bat da, WordPress multisite instalazioetara eginez gero, zure WordPress instalazioa leku bat premium network of sites (siteak) batean transformatzen du – [WordPress.com](https://WordPress.com) bezala –, zein irudien da client-ek erabiliziek siteak sortu dezaten (monthly, quarterly edo yearly fees bidez, eta Free plans ere sortu dezakezu).

  * **Ultimate Multisite-a Multisite gabe jakin du?**

Ez. Ultimate Multisite-ak WordPress multisite, subdirectory edo subdomain behar du.

  * **Ultimate Multisite-aren eskatzen dituen erabilgarriak (requirements) zer da?**

Ultimate Multisite-ak hauek behar dute:

**WordPress Version** : v5.3+ (Reikinduta: Ustezuna lehen estable versiona)
**PHP Version** : 7.4.x (8.0-ren gain eta irudien eskaintza hasten da)
**MySQL Version** : v5+ (Reikinduta: Versio 5.6, hostearen prozesi zehatzu ez dutenean 8.0 ez duzena)

_Multisiteak (subdomain edo subdirectory) ere aktibatu behar da._ [WordPress Multisite-a instalatzeko gaitasuna](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **Ultimate Multisite-ak Subdirectories-ek duen Multisitearekin jakin du?**

Bai. Ultimate Multisite-ak subdirectory instalazio eta subdirectory instalazio arte guztiz suportatzen du.

  * **Ultimate Multisite-ak Subdomains-ek duen Multisitearekin jakin du?**

Bai. Ultimate Multisite-ak subdirectory instalazio eta subdomain instalazio arte guztiz suportatzen du.

  * **Plugin eta Themes zein irudien da erabiltzaileei limitatuz Ultimate Multisite-a erabil dezakezu?**

Bai. Ultimate Multisite-arekin, zure network-eko plan baten artean zein plugin eta themesak erabilera dagoela aukeratu dezakezu.

  * **Erabiltzaile bat site batean sortuedu post eguntezko irudien da limitatuz Ultimate Multisite-a erabil dezakezu?**

Bai. Ultimate Multisite-arekin post quota bat ematen guztietan duzun. Horrek adminak invite egin dezatatzen erabiltzailearen koilua, disko espazioa eta baino gehiago ezarritzen dituzte.

  * **Ultimate Multisite-arekin Site Templates (Modelu) suportatzen dira?**

Bai. Ultimate Multisite-arekin zerbait Site Template bat ere ezartu dezakezu. Site Templates hauek themes (tema) ez dira, baireko koilua da. Horrek network-an normal sitesak dira. Pluginak, themesak eta horretan Site Templatea jakinarazte dituztezu jakinarazte ditzakezu. Erabiltzaile bat horretan templatea aukeratu beharko du, horrekin horren sitea hori kopia gisa sortuko da, guztizko jakinarazteak hartuta.

Mende, plan batako template bat ezarri dezakezu. Horrek egin ondoren, horren planera emateko erabiltzaile guztiek hor Site Templatearen duplikatua sitea sortuko dute.

  * **Ultimate Multisite-arekin WooCommerce (WooCommerce) integratzen da?**

Bai!

Network-an erabiltzaileek WooCommerce erabiliz dutelako, negozioak sortzeko aukera emateko nahi duzu, Ultimate Multisite-arekin hori posible da. Plan baten artean clientak sortzeko produktu guztia ezarritzen ere limitatzen dituztezu.

Ultimate Multisite-arekin abonazio koilua eskatzeko, [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) erabiliz beste Ultimate Multisite clientak main sitean WooCommerce gateway batzuk erabiltzen jakinarazteko aukera emate ditzakezu. Horrek, WooCommerce-eko developatu library handiaren parte egin dezakezu eta horiek erabiliz Ultimate Multisite clientak bilatzea posible da.

  * **WPMUDEV Pro Sites-ek abonazio koilua migra dezakezu?**

Bai, Ultimate Multisite-ek indiz da ematen du [bideaiaren herramienta migrazioa](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) Pro Sites network-era.

  * **Nola da dago "Allowed Plugins" tab-ek ez dago Plan Edit Pagean?**

Ultimate Multisite-ek saldunea pluginak ematen edo ez ematen irizten duena, eta horiek saldunean aktibatu gabeago dira. Plugin bat network-aktibatuta jakin dela, bere naturko arabera, saldunearen guztietan default-ean cargatuko da.

"Allowed Plugins" taban plugin bat ez dago listatzen duen ikusi nahi du, plugin denak network-aktibatuta edo ez dagoela ikustu.

  * **Mogu plugin batekin designaia erabiltzen dezake?**

Bai. 1.4.0 bertsioaren partirako, Ultimate Multisite-ek template overrides (template-ak gehiago modifikatu) suportatzen du.

  * **Ultimate Multisite-ek WP Enginearekin lan egiten du?**

Bai. Ultimate Multisite-ek dela WP Engine-ek ezarrituta dagoela, baina domain mappinga (domain-ak jartzen) funtzionatzeko adminaren parte bat behar zuela (domain-ak WP Engine panelan gehitzea). **Hau ez da horren kasuan baten, 1.6.0 bertsioaren partirako.** Ultimate Multisite-ek WP Engine API-arekin direktu egiten du eta mapping egin ondoren domain-ak panelan otomatikoki gehitzen dira.

  * **Zer hostuingtain ematen dituz?**

Ultimate Multisite-ek honelaire guztietan lan egiten du. Arlozkatu bat da, Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane eta [Runcloud.io](http://Runcloud.io) bezala hostuingileen artean domain mappingak erabiliz duten gehiago integrazioa dago, zehatzago lan egiten dute "out of the box". Beste hostuingileen artean, domain mappingak manuel konpondu behar zabaltu daitezke.
