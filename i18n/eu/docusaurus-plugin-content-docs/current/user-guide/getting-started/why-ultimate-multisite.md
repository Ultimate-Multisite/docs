# Zerbi Ultimate Multisite-aren {#why-ultimate-multisite}

Hau aurki daura, gure artikuluan aurkitu zutuz, pluginari zure negozio ideak errealizatzeko aukera izango dela eskatzen duzu. Horrek guztia arau behar duten zehalak iragatik espero dugu.

## Zer da Ultimate Multisite-aren? {#what-is-ultimate-multisite}

Ultimate Multisite da WordPress Multisite plugin bat da, eta horrek leku hartzeko WaaS (Website as a Service) teklatzea gure erabiltzaileei konpondu egiten du. Gure pluginarekin, abonazio zehalak onartu dezakezu eta erabiltzaileek zure Multisite-aren webgune bat hostea duen bitartean iragatik errespetu edo abonazio laburua ordaintzen dituz.

Ultimate Multisite arau duzu errendatutako metodo batzuekin (adibidez, WooCommerce edo Stripe) ordainak hartzea, erabiltzaile berri bat zure network-ean ematean webgune moldak sortzea, dominiak mapatzea eta zure respertazio formulak pertsonalizatzea duzu erabilera konpondu dezakezu.

Eta horra lehen partea da, gure funtzioak add-ons eta webhook-ekin bidez hobetzeko aukera oso handia da.

## Beste funtunaldiak {#other-features}

Ultimate Multisite hauek ere eskaintzen du:

  * Domini mapatzea (Domain Mapping) WordPress hostearen eragiketa mezuarekin integrazioa: Ultimate Multisite-aren bidez, erabiltzaile berri bat dominiu mapatzen duenean, WP Engine, Cloudways, cPanel, RunCloud, Closte eta gehiago pluginak admin paneletan otomatikoki haserazten ditu, gaur egun ez dutelako handia. Bestari artean, SSL sertifikatua hartzeko prozesua ere gure parte batez hasi dugu.

  * Webhooks: Ultimate Multisite-ek sistema oso potentzia duen Webhook bat du, eta horrek zure network-a erabilera dauden serbisuak edo testuinguruan integratzea lortzen duzu. Zapierarekin direkta integrazioa ere eskaintzen dugu, eta horrek zure network-a hori plataformaan dagoen 1000+ aplikazio eta serbizio guztietan erabilera ematen du.

* Potent Admin Pages funtziak eta gehiago erabiltzeko: Ultimate Multisite-ek hasiera daiko plugin bat du, Admin Pages Creator, hau da, elementor, Beaver Builder eta Brizy template-ak erabiliz customer-innen dashboard-etan pertsonalizatutako Admin Pages leitzateko.

* Dashboard-aren erpaldi zehatzeko bi tema ondinet: Check Material WP eta PRO.

Eta gehiago advanceda da:

* Template Overrides (Template-ak superatu): Ultimate Multisite-rekin, pluginen edo core pluginen jarduerak aldatzen ez du interes gabe, signup flow-aren default HTML template-ak bere versionekin erpotik egin dezakezu.

* Search and Replace API: Leitzateak {{site_name}} bezala placeholder-ek markatuta ere egin dezakezu eta customer-ek signatu egin ondoren ematen informazioa Search and Replace API-aren erabiliz, automatikoki horiek placeholder-ek hartzen dituz.

## Erabili edukiak (Use cases) {#use-cases}

Ultimate Multisite-ren posible use cases bat begiratzeko ordu da.

### Kasu 1: Agenzia {#case-1-an-agency}

Geroan, agenziako jardueraren espezialitatea webstei diseinatzen ari dira, zehatzko hosting edo marketinga jatorrizko serbisuak gisa listatuta jakinarazten beste aspektuak eta.

Agenzie-ek, Ultimate Multisite-rekin bat plataforma-an webstei arte eta administrazio dezakearen potensiala berragarri eragikizten du. Bere diseinazioak GeneratePress, Astra, OceanWP edo beste batzuk bezala tema batzuken arabera standardizatzen dituzten agenzieek, Ultimate Multisite-ren jarduerak erabiliz leitzateko dira, eta horiek guztietan webstei berrakoetara automatikoki aktiboa egin dezakezu.

Ultimate Multisite-ren aukerak plugin eta tema batzuk pertsonalizatuta administrazio egiten du, hau da, plugin eta temak erabilera edo ez erabilera emateko eta bere aktibazio estadoa berrako webstei berrakoetara instanciatuta.

Agencias will find peace of mind with Ultimate Multisite allowing them to do what they do best - design exceptional websites.

### Kasu 2: Provedor Nichea (Niche Provider) {#case-2-niche-provider}

Dau da ezberdune dagoela irudi hau: "Bat beste gauza egin eta hori on egiten duzu". Orain, espezialista askorengari, horrek bat ideakaren edo serbisua bat sentazioa aldatzeko gabe.

Hau jakin baduzu site-etegiak (sites) bat networka dezakez, gogoratzen duzu zein irudiak eta plataforma bat ematen dituzteko serbisuak. Hau da horrek template-ek (site templates) laguntzen duen testuinguru hau!

Ultimate Multisite-aren inobazioa bat da, site template bat da horrek tema (theme) instalatuta eta aktibatuta dagoena, beharrezko pluginak instalatuta eta aktibatuta dagoena, eta proba postak edo zehalak (pages) ezarrituta dagoena. Hau da horrek kliantzak leku bat template-aren alde egin denean, template-aren kontuentuak eta aukerak (settings) site berri erdatuaren kopurretan kopiatzen duela.

Nichea site eta serbisuek ematen duen provedor bat bilera, hau da ezberdune bat: custom pluginak eta diseinazioa eskatzen sansaltzeko, site batean lehenbait input (input) emate behar da. Kliantzak serbisuak amaitzen duela minimal input bat ematea dugu.

### Kasu 3: WordPress Web Hosting {#case-3-wordpress-web-hosting}

WordPress site-ak hostatzea artean pertsona askorena erantzuna dago, baina WordPress bertsio bat instalatuta kliantzak web espazio emateko garrantzia ez da. Hau da horrek, espezialbidea serbisu on bat emateko beharrezkoa da erabaki eta ikuspenak (considerations) batzuk batatu behar dira.

Ultimate Multisite-aren alde hau da, WordPress site-ak hostatzea beste batzuk ematen duen konponbidea (turnkey solution). Erantzunaren parte batean dago pluginak, abonazio serbisuak emateko mekanismoak, mugikorrak (checkout forms), eskaintzak (discount vouchers) eta kliantzailearekin harremanak.

WordPress Multisite-a leizkilean instalatzeko, konfiguratzeko eta jarraitu behar duen garrantzitsuak parteitzea Ultimate Multisite-ek eragiten du, nahiz eta network administratorrak horiek ez dute gehiago ikusten behar; hori erabiltzen dituen serbisua edo niche-a (produktuen nivåak, prezioak eta serbizioak) nola betetzeko.

Ultimate Multisite-arekin integratzea buskitzen duen developer-ek, soluzioak RESTful API eta Webhooks ere ematen du esgia-eragiketa iragatik.

Ez aurkitu behar beste plugin eta lisentziak ez dute, Ultimate Multisite-k Wix, Squarespace, [WordPress.com](https://WordPress.com) eta beste batzukaren erabilera pareko eta funtzioa oso adibideak ematen du.
