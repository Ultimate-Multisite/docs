# Ultimate Multisite Hooks eta filters

Ultimate Multisite funtzioak eta database query-ak bat da. Funktio eta filter-ek du, zein modifikazio edo kode berria ezartzeko aukera ematen dute dagoen funtizi buruz.

Adalu behar dituen erabiltzaile guztiek ereduarekin gehiago personalizatu dezaten dituzteko, hook eta filters bat listak gureme honetan. Listari aurkitu ditzake Github-an, aipatutako linkera klik egin dezakezu.
