# Codeable-etik eskaintan modalea laguntzen dituz?

Mundiaren edo network-ek jakin dituen modualak ez duzun, modu ondo egiten dituzten espezialbideak behar dituzu. Hau da zure eskatutakoa dela, [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) iradokituz duten, Ultimate Multisite-aren esperientzia dituzten desarrollador bat aurkitu dituen leku bat.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) da kostaldeak kwalifikatuak WordPress freelance developaritorik ezartzen dituzten plataforma bat, zehat espezialbideak dituzten proyekte-ei lan egin dezaten. Zure irudien edo eskatutakoa egokitzen duen ekspertegitorekin jotasera aurkitzeko prozesua oso adibideago dago.

### Codeable-an proiektu bat zer batean ezartzen? {#how-to-create-a-project-in-codeable}

Lehenik, [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) iradokituz, **Start a Project** (Proiektu bat hasten) botoia klikatu.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Aurrera, proiektu bat emateko, eskuratu eskuratu laualaren "Start a Project" (Proiektu bat hasten) lekuan klikatu.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

"I need help with" (Ez aurrez ikusten dut) lekuan, **Customizing** (Modala egokitzen) aukeratu duzu :

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

"My" (Mea) lekuan, **Plugin** (Plugin) aukeratu duzu :

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Gero klikatu **Continue** (Aurrera egin) botoia.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Aurrera, proiektuaren eta beharrak descriptiona emateko prozesua da, lehenik gure beste moduan besteko bidea eratu. Posiblean gure informazioa guztiz ematen dituzu!

<!-- Screenshot unavailable: Codeable project description form -->

Prezioak egokitzen lagitzake, hon batzuk erantzute.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Horren, eta gertatzen da, ba [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) kontua ez duzu? Ez dago, benitzu bat sortuți eta **Publish My Project** botoiari klikatu.

<!-- Ekinak ez dago: Codeable kontua sortzea eta Publish My Project botoia -->

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) proiektuaren publiko egin ondoren, plataforma-k sizkietarekin bat konektatuko da irudien aurkezpenko proiektua eskatzen dituen taldearekin.

Hauprozesuak ziurtatzen duzu zure lanbidearekin ondoan jakin duen pertsona aurkitzea, WordPress eta Ultimate Multisite-ekin esperientzia duten!

Codeablearen minimi tasa da $70–$120 USD gal/ora. Estimazioak lanbidearen erabilera, kompleksitatea eta proiektuaren apurtu honearen arabera aldatuko dira.

Estimazioa aurkitu ondoren, zure beharrezko pertsona hiria sortuti eta proiektua berri egiten!
