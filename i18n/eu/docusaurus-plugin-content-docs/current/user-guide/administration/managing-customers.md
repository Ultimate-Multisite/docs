# Kudeak Barkapen Gestiona {#managing-customers}

Hau **Kudeak** lapiaren da landetara emateko taldeko testiak erlistatu da. Hau erabiltzailearen batzu informazio eskatzen du, zehazki, maila, lehen datu-erregistroa eta erabiltzailearen subsitean administrali gisa logiatzeko linkua.

![Customers list page](/img/admin/customers-list.png)

Erabiltzaile batkaren kartelaren ondoren **Gestiona** klikatu duzu horrek kudeakari informazio erdetik ikusteko.

Kudeakari informazio lapiaren da, **At a Glance** blokoa erabiltzailearen ID-a, lehen datu-erregistroaren testua eta erabiltzailearen guztizko irudia erakusten du.

**Customer Options** blokoa erabiltzailearen VIP (horrek lehiagarri aurkitzeko tagia da) gisa emateko funtuna du. **Billing Info** tab-a maila, sistemak eta país bezala informazio finantziari erakusten du.

**Fields** blokoa erabiltzailearen erabiltzailearen pertsonalizazioa, maila eta **Email Verification** statusa aldatzeko funtuna du.

Customer Options blokoaren ondoren hiru bloko gehiago aurkitu duzu:

- **Memberships** — Erabiltzaileak abonatu dauden abonamenduak
- **Payments** — Erabiltzailearen irudia historia
- **Sites** — Erabiltzailearen bonean duten siteak
- **Events** — Erabiltzailei erakundutako gaitasun-erakunde (email bideratzea, finantziari aldatu eta teta)

Gure item batzueko ondoreko lekuan (→) klikatu duzu horrek zehaztu lapiaren da. Adibidez, site batzueko ondoreko lekuan klikatu duzu sitearen editazio lapiaren da, horretan abonamenduak, sitearen bideak eta aktibo statusa aldatu dezakezu.

Amoin abonamenduak eta irudiekari eskatzen du — zehaztu lapiaren da erabilera statusa, erabiltzaile batzua, produkzioak, irudia-eragailuaren konfiguratura eta baino refundak ere procesatu dezakezu.

Kudeakari lapiaren handi-batean aurkitu duzu:

- **Dugutzeko egia eta IP-ak** — Lehen login-aren testua, IP-aren jarduera eta erdatu landetxea
- **Ordea-testuak (Timestamps)** — Kudeatzailearen kontua zeran sortutako eta lehen aldiz aldatutako testuak
- **Dugutzea** — Kudeatzailearen permanente dugutzeko opszioa

:::warning
Kudeatzaile bat dugutzea permanente da eta ez da aurreratu dezake.
:::
