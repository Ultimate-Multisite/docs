# Dashboard Deskurtso

Ultimate Multisite-ek **Dashboard** daingoa menuku Ultimate Multisite-an ondare dago, eta Network Dashboard-ra widget-ak gehiago ez du. Hau artikulu horiek bat daude daude erabiliko horietan.

## Network Dashboard {#the-network-dashboard}

Ultimate Multisite-k network-lekin dagoen dashboard-ean (My Sites → Network Admin → Dashboard) widget-ak garrantzitsuak jartzen ditu.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

Hau daude widget lehen erabiliko:

**First Steps:** Instalazio berri bat daingoa dagoenean erabilizten, metodo koekonatu eta test kontua egokitzeko bezalako jarduerak amaitzen dituenko iradokizuna ematen du.

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Summary:** Eguneko inscrepsioak eta merkataritza zehatza informatzen du. MRR (Monthly Recurring Revenue) daude, merkataritza-gune membership-ek duten kontsumarietar arteko iradokizuna proiektatuta ematen du.

![Summary widget](/img/admin/network-dashboard-summary.png)

**Activity Stream:** Ultimate Multisite-an dagoen jarduerak informatzen du — inscrepsioak, cancelazioak, site-ak lekuak, borratzea eta beste jarduerak.

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** Network-an dagoen erabiltzaile eta site-ekin buruzko laburra zehaztasun ematen du, eta site-ak edo erabiltzaile-ak berri lekuak sortzeko aukera ematen du.

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** Ultimate Multisite-aren lehen bidea informazioa jartzen eta eragiten du, horrek update-ak, bug-ak kritikoak eta segurtasun-irudinetarako hasierak barne ditu.

![News widget](/img/admin/network-dashboard-news.png)

## Ultimate Multisite Dashboard {#the-ultimate-multisite-dashboard}

Network dashboard-ek network-aren buruzko informazioa guztiz ematen du, baina Ultimate Multisite dashboard-ek (Ultimate Multisite menuku lehen zerean) serbisioaren buruzko informazioa ematen du.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Mensual Recurring Revenue Growth:** Ekinakizki iragitziaren (12-langile kalendaria periedo) merkataritza dauna mensulari batean eratu egiten du. Iragitzia berria dauriaurreko lekuan (azul) eta cancelazioak (guzki) gaurkiz iragitzia dauriaurreko lekuan (guzki) eratu dira.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**Testu-eragilearen (Date-range) suporta:** Dashboard datuak testu-eragilearen edo itzultzeko prozesuaren ondorioz, beharrezko informazioan onartzeko zehaztekin iragitziaren dauraztekin filtrazio egin dezakezu. Testu-eragilearen dauraztea aipatu widget hauek dauratsailatzen duen informazioa kontrolatzen du.

![Date range and dashboard stats](/img/admin/um-dashboard-date-range.png)

Dashboard widget hauek barne dago:

- **Revenue (Iragitzia)** — Aportuaren guztira eragilearen dauraztekin eta produkzio pertsonal batzako dauratsailak datuz.
- **New Memberships (Membresariak berriak)** — Iragitziaren aipatutako testu-eragilean ezarritutako iragitzia berriak zer da.
- **Signups by Countries (Iragitziaren zehatzeko país pertsonalak)** — Iragitziaren geografikoa datuzailakoa.
- **Signups by Forms (Formakaren ondorioz iragitziaren zehatzeko pertsonalak)** — Kontrolarekin iragitziaren bidez zerko kliantzak iragitzia da, eta administurietik ezarritutakoak zerko da.
- **Most Visited Sites (Iragitziaren lehen bisitatutako lekuak)** — Iragitziaren arloaren lehen bisitatutako lekuak (Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting-an bisitatzen kontuamendu aktiboa emate behar da).

![Dashboard statistics](/img/admin/um-dashboard-stats.png)
