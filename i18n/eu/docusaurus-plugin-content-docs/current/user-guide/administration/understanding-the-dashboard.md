# Ultimate Multisite Dashboardaren Errazten

Ultimate Multisite-aren gehi da konfiguerazio opsioak eta Ultimate Multisite-aren operazio aspetuak ez adauzu ditzakei, lehenik, dashboard-aren ikusituzu barru egin dezaten.

WordPress plugin batzuk garrantzia bezala, Ultimate Multisite-k **Ultimate Multisite** titulu itzultu menu entry bat jartzen du, eta horrekin ondarekin sub-menu entry bat daude.

## Network Dashboard Widgetak {#network-dashboard-widgets}

Ultimate Multisite-k network-level dashboard-aren widget batzuk eragiten du. Horrek **My Sites → Network Admin → Dashboard** iruditzen ditzake. Widget hauek network administrator-ari informazio eta operazio commonekoak lehenik emateko diseinatuta dira.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

### Lehen Oinarriak {#first-steps}

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

Widget hau Ultimate Multisite-aren berri instalazioan erakusten du eta network administrator-ari informazio metodo bat konfigueratzea eta test kontu bat leku ezarria emateko bezala, jarduera gehiago amaitzen dituen alkartea ematen du.

### Sumarioa {#summary}

![Summary widget](/img/admin/network-dashboard-summary.png)

Sumarioa widget-ak eguneko inscrepsioaren eta horien merkataritza (revenue) ezarria informatzen du. MRR (Monthly Recurring Revenue) dauden kontsumarietar arteko proiektu totala ematen du, merkataritza generatzean duten membership-ek parte hartzen duen.

### Jarduera Fluxuua (Activity Stream) {#activity-stream}

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

Widget hau Ultimate Multisite-aren arteko jarduerak informatzen du. Jarduera hauek inscrepsioak, cancelazioak, site-ak leku ezarria emateak, borratzea eta beste jarduerak barne ditu.

Mehatzeko erantzun mendazko dokumentazioaren Events seksioan informazio gehiago hartu dezakezu.

### Ora {#right-now}

![Right Now widget](/img/admin/network-dashboard-right-now.png)

Ezaren utilitatea widget-ak ez da, erabiltzaile eta webgune bat-etan dagoen txiki sumarioa erakusten du. Gainara, itxura hautatutako lehen partean, webgune edo erabiltzaile juelasak lehiak sortzeko kliku batekin zehaztasuna ematen du.

### Notiziak eta Diskusiak {#news-and-discussions}

![Notiziak eta Diskusiak widget](/img/admin/network-dashboard-news.png)

Ez dago widget hau Ultimate Multisite-aren dauden lehen notiziak hartzen eta erakusteko. Update-ak, errorek kritikoak eta segurtasun aukerak buruzko informazioa ezagutzeko, honen widget-a jarraitu.

## Ultimate Multisite Dashboard {#ultimate-multisite-dashboard}

Hau network-lekin dagoen dashboard-ek network-aren holistikoa informazioa ematen du, baina Ultimate Multisite dashboard-ek (Ultimate Multisite menuko lehen zahar partean dagoena) serbisua buruzko informazioa erakusten du.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

### Mensual Recurring Revenue Growth (MRR) {#monthly-recurring-revenue-growth}

MRR (Mensual Recurring Revenue) grafikoa 12-langile kalendariaren edozein periodoaren merkataritza eskaintzen iragailu du.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

Hobitzenak (new revenue) lurroko da, baina cancelazioak (cancellations) garbia (red) erakusten dira.

### Data-erren Periodoa Udaldu {#date-range-support}

Kunde eta aktibitatea ezagutzeko, statistika oso ezberdina izan daiteke. Dashboard-aren itxura dagoen data-erren periodoa udalu egiten du network administratorari, hori beharrezkoa den informazio eta testu-periodioan jarraitzea posible egiten du.

![Date range selector and stats](/img/admin/um-dashboard-date-range.png)

Data-erren periodoak widget-ek aurrera erakusten duen periodoa udaltzen du, merkataritza, new membership-ak, eta país edo forma-ean egin aukera eginak, eta lehen bisitatutako webguneak barne.

![Dashboard statistics widgets](/img/admin/um-dashboard-stats.png)

:::tip
**Most Visited Sites** widgeta erabilera egiteko, **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** da ezarri behar du bisitatzen kontuaren aktibitatea.
:::
