# Memendua (Memberships) Ustez

Ultimate Multisite-an daude memenduek eta zehatzatu duzu irudiak eta abonamendu guztiek network-an. Abonamenduaren detallak, gure planak eta produkzioak, sitiaren kopurua, labenguen aurrekontu eta frekwentzia, mugimenduaren historia eta horrekiko datuak editeatu eta jartzeko.

Horarekin aurkitu dituztezu **Memberships** tab-a Ultimate Multisite menukuan.

![Memberships list page](/img/admin/memberships-list.png)

Memberships lapurialan, kliente guztiek eta horiek erabiliko produkzioak ikusten duzu. Memenduek statusaren arabera filtrazio dezakezu: aktibo (active), eskatuta (pending), mugituan (on hold), amaituta (expired) edo canceldatu (cancelled).

Libertateko memendueak **biurte bat** (lifetime) membruntzak gisa tratatzen dira. Ez dute amaieraren datua eta auto-amaitzen dira. Libertateko plan bat trial edo periodo bat gaur egin behar bada, trialarekin duen produkzioa erabiltu, memenduea manuaz canceldu, edo zehatzatu daitekean erabilera aldatzeko klientea beste plan batean jarri behar bada.

Memendue bat klikatu duzu eta membruntzakaren detallak editeatzea jartzen dituzteko lapurialan aurkitu duzu — billing informazioa, siti edo bisitat mugimenduak eta gehiago.

![Membership edit interface](/img/admin/membership-edit.png)

Hau da membruntzakaren editeze lapurialaren guztizko ikuspen:

![Membership edit full interface](/img/admin/membership-edit-full.png)

Gure networkan irregistraturik dituzten erabiltzaileei lotutako membruntzak berriak gehitzen ereiteko dituztezu. **Add Membership** klikatu eta detallak konfiguraatu duzu.

![Add Membership modal](/img/admin/membership-add-modal.png)

Gero, **Create Membership** klikatu duzu. Memendue berria membruntzakaren lapurialan erabilera geratu daiteke.
