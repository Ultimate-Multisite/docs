# Lehen 10: Eguna

Gure guztia dago. Puertasak lortzeko, lehen aurrera egin baino lehen, hori ez duzu pre-launch checklist-a ikusten duzu, eta ez duzu ez dauden, ez duzu faltatu edo ez duzu inbildatzen duen eduki.

## Zer dauden geratu ditugu

FitSite-ek templateak, planak, checkout, branding, onboarding eta prezioak konfiguratuta dira. Arriba, gure guztia funtzionatzen duen jakin dugu eta iraupizten duzu.

## Pre-Launch Checklist

Arritzen duzu guztia. Ez faltatu ez duzu.

### Plataforma Infrastruktura

- [ ] Hostingak estabal dago eta zahar edo oso eragokoan ondo dago
- [ ] Wildcard SSLak aktibo dira eta subdomain guztiak HTTPS-ean iraupizten dute
- [ ] Domain mappingak funtzionatzen du -- web hau sortu eta domaina osoa mapatzea jartzen duzu
- [ ] Backupak konfiguratuta eta probatu dira (ezondrik bat iraupizteko da)
- [ ] Monitoringak dago -- plataformak ez dutenean jakin dugu

### Templateak

- [ ] Template guztiek berri web hauetan lehen eragiten du
- [ ] Placeholder kontentuak laguntzen du eta ez duzu errorek
- [ ] Bildiak guztiek zuzen betelako (stock photo-aren agua markak ez dut)
- [ ] Mobile responsivenessk guzti template laperean funtzionatzen du
- [ ] Paguen iraupiztzeko gaitasuna ondo dago (GTmetrix edo PageSpeed Insights gabe probatu)
- [ ] Template bat ez du broken links edo faltatu assetak

### Planak eta Produkzioak

- [ ] Hau guztizko treskiak aktibo eta erabilgarriak dira
- [ ] Treskia deskribioak onartu eta espezifikoari dira
- [ ] Prezioak barne (mensual eta anual) batezkoa da
- [ ] Instalazio-ezpenak kanan treskietan konfiguratuta dira
- [ ] Probar egintzaa funtzionatzen du Starter treskiarekin
- [ ] Orden-ezpenak checkout-ean barne eragiten dira
- [ ] Plugin eta theme-aren limitazioak kanpoan onartu dira

### Checkout Flow (Checkout-aren bilakidetze)

- [ ] Harri guztietan proba inscrepsio bat egin (proba-pagmento modua erabiliz)
- [ ] Template-aren aukeraak treskiare honako urtera barne jakin dituzte
- [ ] Pagmentazio prosesuak onartu dira
- [ ] Inskribapena gaur ondoren iragailu emaila lortzen du
- [ ] Irudi berria situaile bat template-aren barne ere itxera daiteke
- [ ] Harriberak bere situailean hemen egin dezake
- [ ] Ezpen-kodeak onartu dira

### Branding (Markinguea)

- [ ] Login-aren lapurtia FitSite brandinga eragiten du
- [ ] Admin dashboard-aren FitSite brandinga eragiten du
- [ ] Hau guztizko email-ek FitSite branding eta fitness-spesifik hizkuntza erabiltzen dute
- [ ] Faktura hauek zure negozio-detalioakarekin barne eragiten dira
- [ ] Marketing situaileak aktibo da eta checkout formara link egiten du

### Onboarding (Inizioa)

- [ ] Irudi berri klientzia dashboard-etan Quick Start widgeta eragiten du
- [ ] Hau guztizko Quick Start link hauek irudi ezarritzen dituzte
- [ ] Welcome email-aren sersioak konfiguratuta eta probatuta daitezke
- [ ] Knowledge base artikuluak publikatu dira eta erabilera daiteke
- [ ] Kontua-aren lapurtia treskiaren barne informazioa eta upgrade aukerak eragiten du

### Legal and Business (Legea eta Negozia)

- [ ] Kontuakontratu-er datuen publikoa eta checkout-etik linkatu daiteke.
- [ ] Gizurpen politika (Privacy policy) publikoa eta erabilera daiteke.
- [ ] Iragailuaren politika (Refund policy) definitua eta dokumentatua daiteke.
- [ ] Biznes entidad bat ematen daingoa iradokizunak hartzeko.
- [ ] Pagamentu gaitasuna (Payment gateway) live moduan dago (test moduan ez du).
- [ ] Tax konfiguratazioa (Tax configuration) zure erdatzeko ekaintzari (jurisdicción) erabilera ondo dago.

## Soft Launch vs. Hard Launch (Lehen fasea vs. Publikoa lanpokoa)

Lanpokoa iradokizuna bat ikusituzu:

### Fase 1: Soft Launch (Lehen faza)

Publikoa lanpokoaren aurian, 5-10 fitness studio dueñoak ematen dituzteko kontuak hartu. Hau dira zure beta kliuntzak. Adierazten dituzteko eskaintza egin dezakezu (bizitza langilean 50% eskaintza, edo 3 hiligoa jarduera gabe) berriaren kontratu eta erabilera ezagutzeko honesta itxarpenak, site-a erekizgunea gisa erabiltzeko izin, eta jartzen dituen bugak edo problema horiek eskatzea.

Hau da zure geraketa kliuntzak eta publikoa irekitu baino lehen erekizguna gisa erekituko live site hauek ematen duela.

### Fase 2: Public Launch (Publikoa lanpokoa)

Soft launch-aren itxarpenak inguratu gabe egin ondoren:

- Pagamentu gaitasuna (payment gateway) live moduan jarri.
- Zure marketing site-a publikatu.
- Kliuntzak hartzeko prozesua hasten dena (Lesson 11).
- Fitness industria erabilera eskatzen dituen kanaletaketan jaurteko.

## Launch Day Actions (Lanpokoaren egunean egin behar ditu mga jarduerak)

Publikoa lanpokoa egiten duen egunean:

1. **Orri bizitza-pagaiak aktiboa** -- test modua deshabilitatu payment gateway-an
2. **Beste barne eratu** -- berezko pagamentuarekin (berri eragikitzeko) test signup bat egin (berri eragikizeko ondoren refundatu)
3. **Garrantzitsuz eratu** -- gaiak, signup-ek ez duña edo bizitza-pagaiak problema hauek ikusi
4. **Aurreratu** -- bere lehen kliuntzak laguntza behar izan dezatat, eta azkar erantzuna jardutza sortzen du

## Zer da iragailu nahi?

Hauzuk espero duzu:

- **Problema payment gateway-an**: Payment gateway-ren suporta kontaktoak gehiago ezarri
- **Problema SSL certificate-an**: Zertifikatak ikusten eta berri eduki jakin dutela
- **Email delivery-n problema**: Emailak berezko iragaildu daitezkeen jakin (spam folderrak ikusi)
- **Performans load-an**: Trafik aukeratzen du, hosting-a eskala dezake jakin
