# PayFast

Ultimate Multisite: Payfast Gateway

Payfast — Hegoafrikan lineako ordainketetarako gateway azkar eta segurua — gehitzen du Ultimate Multisite ordainketa-aukera gisa.

## Instalazioa {#installation}

1. Kargatu 'ultimate-multisite-payfast' '/wp-content/plugins/' direktorioan
2. Aktibatu plugina WordPress-eko 'Pluginak' menuaren bidez
3. Jarraitu urratsez urratseko morroiari plugina konfiguratzeko

## Erabilera-gida {#usage-guide}

PayFast Hegoafrikako enpresentzako lineako ordainketa-prozesadore nagusietako bat da. Lineako merkatariei hazten laguntzera bideratuta dago, transakzioak errazak eta seguruak eginez. Ultimate Multisite erabiltzailearen eskaerari jarraituz, PayFast-ekin integrazioa posible egingo duen gehigarri bat sortzea erabaki genuen. Artikulu honek erakutsiko du nola konfiguratu PayFast zure ordainketa-gateway gisa, zure Ultimate Multisite sarean ordainketak jasotzeko.

Lehen urratsa gehigarria instalatzea da.

# Nola instalatu Payfast gehigarria:

Joan gehigarrien orrira eta hautatu Ultimate Multisite: Payfast Gateway gehigarria.

![Gehigarrien orria](/img/admin/addons-page.png)

Gehigarriaren leihoa agertuko da. Egin klik Instalatu orain aukeran.

![Instalatzeko aukera duen gehigarrien orria](/img/admin/addons-page.png)

Instalazioa amaitu ondoren, pluginen orrira birbideratuko zaituzte. Hemen, egin klik Network Activate aukeran eta _Payfast_ gehigarria zure sarean aktibatuko da.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# PayFast konfiguratzea

Gehigarria instalatu ondoren, Ultimate Multisite ezarpenetara joan zaitezke eta PayFast konfiguratzen hasi.

**Ultimate Multisite ezarpenak** orrian, "Ordainketak" fitxa aurki dezakezu. Fitxa honen azpian, "**Payfast** " ikusiko duzu, ordainketa-gateway hau gaitzeko edo desgaitzeko txandakatzeko botoi batekin batera.

![Ordainketa-gatewayaren ezarpenak](/img/config/settings-payment-gateways.png)Aurrera eta egin klik txandakatzeko botoian gaitzeko. Ondoren, koadroa zabalduko da bete behar dituzun hainbat eremu erakusteko, hala nola **Mechant ID, Merchant Key eta PassPhrase.** Informazio hori zure **Payfast** kontutik lor dezakezu

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Hurrengo urratsa zure PayFast Account-en saioa hastea da, behar duzun informazioa lortzeko.

Saioa hasi ondoren, behar dituzun kredentzialak berehala ikus ditzakezu zure **Payfast** Account Dashboard-ean.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->Kredentziala kopiatu eta itsatsi besterik ez duzu egin behar, eta xehetasunak bete zure Ultimate Multisite ezarpenetan. Ondoren, egin klik **"Gorde ezarpenak** " botoian aldaketak gordetzeko.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Hori da dena, zure PayFast ordainketa-integrazioa osatuta dago!
