# Iragazkia: wu_woocommerce_line_item_params

Iragazi WooCommerce Cart-en lerro-elementuak sortzeko erabilitako parametroak, bezeroa WooCommerce ordainketara birbideratu aurretik.

## Parametroak

| Izena | Mota | Deskribapena |
|------|------|-------------|
| $line_item_params | `array` | Saskiko lerro-elementuaren parametroak. |
| $wu_line_item | `\WP_Ultimo\Checkout\Line_Item` | Ultimate Multisite lerro-elementuaren instantzia. |
| $wc_product | `\WC_Product` | WooCommerce produktua. |

### Noiztik
