# Fluent Forms

Gehigarri honek Multisite Ultimate-n Fluent Forms-i mugak gehitzea ahalbidetzen du produktuetan, kidetzetan edo guneetan.
Fluent Forms-ek taula pertsonalizatuak erabiltzen ditu argitalpen mota pertsonalizatuen ordez; beraz, ohiko murrizketa aukerek ez dute funtzionatzen Fluent Forms-ekin.

- Gehitu mugak Fluent Forms edo Conversational Fluent Forms-erako
