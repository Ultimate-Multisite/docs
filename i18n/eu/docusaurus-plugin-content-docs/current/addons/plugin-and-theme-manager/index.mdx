# Pluginen eta gaien kudeatzailea {#plugin--theme-manager}

Multisite Ultimate: Pluginen eta gaien kudeatzailea

Editatu edo ezkutatu instalatutako pluginen eta gaien meta-informazioa (izenburua, egilea, irudi txikia eta deskribapena). Sortu kategoriak plugin eta gaietarako, zure erabiltzaileek iragazi ahal izan ditzaten, eta gehitu Pluginen orri pertsonalizatu eder bat zure erabiltzaileen panelean.

Plugina erabiltzeko, zure sareko administrazioko Pluginak edo Gaiak orrietara joan eta "Editatu pluginaren/gaiaren informazioa" sakatu besterik ez duzu.

## Instalazioa {#installation}

1. Kargatu 'ultimate-multisite-plugin-and-theme-manager' '/wp-content/plugins/' direktoriora
2. Aktibatu plugina WordPress sareko administrazioko 'Pluginak' menuaren bidez

## Erabilera-gida {#usage-guide}

Ultimate Multisite: Pluginen eta gaien kudeatzailea Ultimate Multisite-ren **premium** gehigarri indartsu bat da, eta zure sareko erabiltzaileek zure sareko pluginen eta gaien aukerak nola ikusten eta haiekin nola elkarreragiten duten erabat kontrolatzeko aukera ematen dizu.

Gehigarri honek hiru helburu ditu:

  1. Pluginen eta gaien metadatuak erabat aldatzeko aukera ematen dizu. Hor sartzen dira izena, deskribapena, egilearen izena eta baita irudi txikiak ere!

  2. Pluginak eta gaiak kategorizatzeko aukera ematen dizu, zure erabiltzaileek erraz ordenatu ahal izan ditzaten.

  3. Pluginen orri zahar eta aspergarria gai-itxurako orri berri batekin ordezkatzeko aukera ematen dizu, kategorien araberako ordenazioarekin.

Deskargatu eta instalatu ondoren, fitxa berri bat - _Gehigarriaren ezarpenak_ \- agertuko da Ultimate Multisite ezarpenen orrian.
