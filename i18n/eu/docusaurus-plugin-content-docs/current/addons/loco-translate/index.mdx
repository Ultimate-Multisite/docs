# Loco Translate {#loco-translate}

Ultimate Multisite: Loco Translate Integration gehigarriak gunearen araberako itzulpen-gaitasunak ekartzen dizkio zure WordPress multisite sareari. Zure sareko gune desberdinek plugin edo theme beraren itzulpen desberdinak izatea inoiz behar izan baduzu, gehigarri hau irtenbide ezin hobea da.

**Gehigarri Honek Zer Egiten Duen**

Gehigarri honek Ultimate Multisite modu bateratuan integratzen du Loco Translate plugin ezagunarekin, zure sareko azpigune bakoitzak bere itzulpen pertsonalizatuak mantentzeko aukera izan dezan. Eskualde anitzeko plataforma bat kudeatzen ari bazara, hizkuntza-komunitate desberdinei zerbitzua ematen badiezu, edo zure sare osoan itzulpen-malgutasuna besterik gabe behar baduzu, gehigarri honek oso erraza bihurtzen du.

**Ezin Hobea Honetarako:**

* Lokalizazio-behar desberdinak dituzten eskualde anitzeko enpresak
* Komunitate anitzei zerbitzua ematen dieten hezkuntza-erakundeak
* Hizkuntza anitzeko euskarria eskaintzen duten SaaS plataformak
* Bezeroen gune anitz kudeatzen dituzten web agentziak
* Itzulpen-malgutasuna behar duen edozein multisite sare

## Instalazioa {#installation}

1. Ziurtatu Ultimate Multisite instalatuta eta sarean aktibatuta dagoela
2. Instalatu eta aktibatu Loco Translate plugin-a
3. Kargatu gehigarri hau zure pluginen direktoriora
4. Aktibatu sarean Loco Translate Integration gehigarria
5. Hasi gune bakoitzerako berariazko itzulpenak sortzen Loco Translate bidez edozein azpigunetan

Hori da dena! Integrazioak automatikoki funtzionatzen du konfigurazio gehigarririk gabe.

## Maiz Egindako Galderak {#frequently-asked-questions}

## Instalazioaren ondoren zerbait konfiguratu behar al dut? {#do-i-need-to-configure-anything-after-installation}

Ez! Integrazioa automatikoa da. Gehigarria eta Loco Translate aktibo daudenean, azpiguneek berehala hasi ahal izango dute beren itzulpen pertsonalizatuak sortzen.

## Zer gertatzen da azpigune batek itzulpen pertsonalizaturik sortzen ez badu? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

Guneak automatikoki erabiliko ditu sare osoko itzulpenak ordezko gisa. Ez da ezer apurtzen, eta denak modu bateratuan funtzionatzen du.

## Azpiguneek sare osoko itzulpenetara sarbidea izan dezakete? {#can-subsites-access-network-wide-translations}

Gune-administratzaile arruntek beren guneko itzulpenak bakarrik ikusi eta editatzen dituzte. Super administratzaileek sare osoko itzulpenetara zein gune bakoitzerako berariazko itzulpenetara sar daitezke, ikusten ari diren gunearen arabera.

## Honek pluginekin eta theme-ekin funtzionatzen al du? {#does-this-work-with-both-plugins-and-themes}

Bai! Gune bakoitzerako berariazko itzulpenek WordPress plugin eta theme-etarako funtzionatzen dute.

## Honek nire multisite sarea motelduko al du? {#will-this-slow-down-my-multisite-network}

Ez. Gehigarriak fitxategi-kargatze eraginkorra erabiltzen du, behar denean bakarrik egiaztatzen duena gune bakoitzerako berariazko itzulpenik dagoen. Ez dago errendimenduan nabaritzeko moduko eraginik.

## Kopiatu al ditzaket sareko itzulpenak gune jakin batera abiapuntu gisa? {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

Bai! Loco Translate-n itzulpenak editatzean, erraz has zaitezke sareko itzulpenetatik eta gune bakoitzerako pertsonaliza ditzakezu.

## Honek WooCommerce-rekin eta beste plugin batzuekin funtzionatzen al du? {#does-this-work-with-woocommerce-and-other-plugins}

Erabat! WordPress-en itzulpen-funtzio estandarrak erabiltzen dituen edozein plugin edo theme-ek funtzionatuko du gune bakoitzerako berariazko itzulpenekin.
