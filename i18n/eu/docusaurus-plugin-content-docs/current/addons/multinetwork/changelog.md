# Sare anitzeko aldaketen erregistroa

### 1.3.0 {#130}
* Berria: Sare-txantiloien aurreikustailea — arakatu eta aurreikusi sare-txantiloiak zuzeneko panel batekin erosi aurretik.
* Berria: Sare-txantiloi hautatzailea ordainketan — bezeroek sare-txantiloi bat hauta dezakete erregistroko checkout fluxuan.
* Berria: Sareen arteko magic link SSO — bezeroek azpisareetara modu jarraituan sar daitezke magic-link autentifikazioaren bidez.
* Berria: Sareko media klonatzea — guneko media behar bezala bikoizten da sareak txantiloi batetik klonatzean.
* Konponketa: Sareen klonatzeak orain multi-tenancy addon gabe funtzionatzen du; super admin pribilegioak berriro ematen dira klonatu ondoren.
* Konponketa: Sarearen sorrera orain bateragarria da WooCommerce-rekin eta ordainketa-pasabide guztiekin.
* Konponketa: Ataka-zenbakia orain ordezko domeinuaren sorreran sartzen da.
* Konponketa: Txantiloi-hautatzailearen CSSa ilaran jarritako estilo-orri batera mugitu da, theme guztietan behar bezala errendatzeko.
* Konponketa: `network_id` gako definitu gabea saihesten da gunearen kontsulta-esparruan.
* Konponketa: Testuingurua aldatu aurreko sare-baimenen egiaztapenak `wpdb` taulen propietate-erroreak saihesten ditu.
