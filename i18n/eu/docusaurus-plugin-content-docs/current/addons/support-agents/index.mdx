# Laguntza-agenteak

Sare oso bat zeure kabuz kudeatzea ez da lan erraza. Zure sarea hazten hasten den heinean, laguntza-eskaera gero eta handiagoa ikusten hasiko zara. Ideia ona izango da zure lan-karga arintzen lagunduko dizuten pertsonak ekartzea. Hala ere, sarbide/baimen osoa ematea arriskutsua da eta zure sarea arriskuan jar dezake. Hor sartzen da Support Agent gehigarri hau.

Gehigarri honek Support Agent izeneko WordPress erabiltzaile mota berri bat sortzeko aukera ematen dizu, eta erabiltzaile horrek dituen baimenen gaineko kontrol osoa ematen dizu. Baimena ematea etengailu-botoi bat aktibatzea bezain erraza da, eta kentzea erabakitzen baduzu, desaktibatzea.

## Ezaugarriak

- WordPress eta Ultimate Multisite baimenen zerrenda xehatua
- Sarbidea erraz gaitu eta desgaitu
- Sortu Agent kontu kopuru mugagabea

## Instalazioa

1. Kargatu 'ultimate-multisite-support-agents' '/wp-content/plugins/' direktoriora
2. Aktibatu plugina WordPress-eko 'Pluginak' menuaren bidez
3. Jarraitu urratsez urratseko Wizard-ari plugina konfiguratzeko
