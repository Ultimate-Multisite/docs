# Iragazkia: signup_get_available_languages

Front-end guneko izen-emateetarako erabilgarri dauden hizkuntzen zerrenda iragazten du.

Hook honi array huts bat pasatzeak ezarpenaren irteera desgaituko du izen-emate inprimakian, eta hizkuntza lehenetsia erabiliko da gunea sortzean. Jada instalatuta ez dauden hizkuntzak kenduko dira.

## Parametroak

| Izena | Mota | Deskribapena |
|------|------|-------------|
| $available_languages | `array` | Erabilgarri dauden hizkuntzak. |

- 4.4.0
