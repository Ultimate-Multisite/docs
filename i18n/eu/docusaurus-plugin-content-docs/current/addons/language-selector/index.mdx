# Hizkuntza-hautatzailea

Ultimate Multisite: hizkuntza-hautatzailea erregistratzean

Gehitu hizkuntza-hautapeneko eremu bat erregistratzeko fluxuari.

## Instalazioa

1. Igo 'ultimate-multisite-language-selector' '/wp-content/plugins/' direktoriora
2. Aktibatu plugin-a WordPress Network Admin-eko 'Plugins' menuaren bidez

## Erabilera-gida

Doako gehigarri honekin, zure erabiltzaileei erregistratzeko fluxuan eta saioa hasteko orrian zein hizkuntza ikusi nahi duten aukeratzen utz diezaiekezu.

[Zure lizentzia aktibatu](../../user-guide/getting-started/installing-ultimate-multisite.md) ondoren, zure lizentziak sarbidea ematen dizun Ultimate Multisite gehigarriak instalatu ahal izango dituzu.

Joan zure Ultimate Multisite ezarpenen orrira eta, ondoren, ezkerreko menuan, egin klik _Gehigarriak_ etiketaren gainean.

![Gehigarrien orria](/img/admin/addons-page.png)

_Joan Gehigarrien orrira Language Selector instalatzen hasteko_

Han, eskaintzen ditugun gehigarri guztiak aurkituko dituzu.

Doako gehigarriak gure bezero guztientzat daude eskuragarri, harpidetu diren harpidetza-plana edozein dela ere.

Egin klik gehigarriaren miniaturaren azpiko _Install it Now!_ botoian. Ondoren, aktibatu Plugins orrian.

![Instalatzeko botoia duen gehigarrien orria](/img/admin/addons-page.png)

_Egin klik botoian gehigarria instalatzeko_

<!-- Pantaila-argazkia ez dago eskuragarri: WP Plugins orria, Language Selector gehigarrirako Network Activate botoia erakutsiz -->

_Aktibatu zure Plugins orrian_

Orain, joan Ultimate Multisite-ren Ezarpenen menura eta sartu _Network Settings_ fitxan. Han, orriaren behealdean, _Languages_ izeneko atal berri bat aurkituko duzu.

![Ezarpenen orria](/img/config/settings-general.png)

_Atal guztiz berri bat agertzen da!_

Hutsik utz dezakezu hizkuntza guztiak aukera gisa bistaratzeko, edo eskuragarri jarri nahi dituzunak hauta ditzakezu. Hizkuntzak zure WordPress konfigurazioan instalatuta dituzunak dira.

<!-- Pantaila-argazkia ez dago eskuragarri: WP instalazioan eskuragarri dauden hizkuntzak erakusten dituen hizkuntza-hautapeneko goitibeherako menua -->

_WP instalazio honetan eskuragarri dauden hizkuntzen zerrenda_

Egin dituzun aldaketak gorde ondoren, aukeratu dituzun hizkuntzak dituen hautapen-menu bat bistaratuko da Saioa hasteko orrian, baita Erregistratzeko fluxuan ere.

![Saioa hasteko eta erregistratzeko ezarpenak](/img/config/settings-login-registration.png)

_Language Selector Saioa hasteko orrian_

![Erregistratzeko orria](/img/config/registration-page.png)

_Language Selector Erregistratzeko fluxuan_

## Hizkuntza URLaren bidez ezartzea

Orri baten hizkuntza ere alda dezakezu locale parametroa URLari pasatuz. Adibidez, italierazko erregistro-orri batera birbideratu nahi baduzu, gehitu ?locale=it_IT URLari.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Honek funtzionatzeko, hizkuntza zure WordPress sarean instalatuta egon behar da.
