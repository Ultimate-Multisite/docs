# Aldaketen erregistroa

## 1.18.0 — 2026-06-29an argitaratua

### Berria

- **Google Calendar tresnak** — konfiguratutako egutegiak eta ekitaldiak irakurri, ordutegiaren jakitun diren automatizazioetarako
- **Kontaktuen mapaketa eta parte-hartzaileen laguntzaileak** — egutegiko parte-hartzaileak guneko erabiltzaileekin eta kontaktuekin lotu
- **Gizakien onarpen-ateak eta oroigarri-erregistroak** — automatizazioak berrikuspenerako pausatu eta jakinarazpen bikoiztuak saihestu
- **TextBee SMS hornitzailea** — konfiguratutako testu-mezuen jakinarazpenak bidali TextBee bidez
- **Laguntza-pakete aurreratua** — garatzaile fidagarrien fitxategi-sistema, datu-basea, WP-CLI, REST banatzailea, plugin eraikitzailea, git snapshot, erabiltzaile-kudeaketa eta errendimendu-probako tresnak gehitu, WordPress.org eraikuntzatik bereizita banatuak

### Hobetua

- **Kudeatutako Superdav zerbitzuaren konfigurazioa** — ostatatutako zerbitzu-endpointak eta konexio-hornidura automatikoa gehitu onartutako guneetarako
- **Argitalpen-paketatzea** — core eta Advanced ZIP bereiziak eraiki, biak GitHub-en argitaratu, eta core paketea soilik bidali WordPress.org-era

### Konponduta

- **AI eskaeren fidagarritasuna** — hobetu eredu-hautaketa, eskaeren denbora-mugak, konfigurazio lehenetsiko eredua, arrazoiketa-testuaren tratamendua eta tresna-dei baliogabeetarako berriro saiatzeko jarraibideak
- **Egutegiaren eta oroigarrien sendotzea** — Google Calendar tokenak eta oroigarrien deduplikazioa indartu
- **Hastapena eta onarpenen berrekiteak** — frontend-eko hastapen-abiarazteak eta berretsitako gaitasunen berrekiteak konpondu
- **WordPress.org paketatze-berrikuspeneko arazoak** — core argitalpenerako paketatze-berrikuspenaren iritziak landu

## 1.16.0 — 2026-05-20an argitaratua

### Berria

- **Generate Logo SVG gaitasuna** — Theme Builder-ek orain logo SVG pertsonalizatuak sor eta txerta ditzake, izen-espazioetarako segurua den garbiketarekin
- **Argazki-igoera aurkikuntza-elkarrizketan** — Theme Builder-en aurkikuntza-elkarrizketak orain argazkiak igotzeko urrats bat dauka, diseinu-testuinguru aberatsagoa izateko
- **Validate Palette Contrast gaitasuna** — kolore-bikoteak WCAG betetzea egiaztatu, theme bati aplikatu aurretik
- **Ostalaritzako menuak** — Theme Builder-ek orain janari eta edarien menu-orri egituratuak sor ditzake ostalaritza-enpresentzat
- **Mahaigaineko eta mugikorreko aurrebistaren errendatzea** — zure diseinua aurreikusi mahaigaineko eta mugikorreko gailuetan, diseinu-norabidea hautatzean
- **Nabigazio-etiketaren parametroa** — Create Menu gaitasunak orain `navigation_label` bereizi bat onartzen du, orriaren izenburutik aparte
- **Tier 1 tresnen erabilgarritasuna** — sd-ai-agent/site-scrape orain Tier 1 tresna da, lehenespenez erabilgarri Theme Builder-en

### Konponduta

- **AI Client cachea** — orain transientek babesten dute eskaeren arteko iraunkortasunerako, iraupen luzeko agente-zereginetan datu-galera saihestuz
- **Plugin errenkadako ekintza-estekak** — zuzendu eta berrizendatu dira argitasunerako

## 1.10.0 — 2026-05-05ean argitaratua

### Berria

- **Tavily interneteko bilaketa** — Tavily bilaketa-hornitzaile gisa gehitu, Brave Search-ekin batera interneteko bilaketa-emaitza aberatsagoak lortzeko
- **Theme-az jabetutako trebetasun integratuak** — Block Themes, Classic Themes, Kadence Blocks eta Kadence Theme trebetasun-gidak orain plugin-arekin batera datoz
- **Gune-eraikitzailearen kontaktu-inprimakiaren gaitasuna** — kontaktu-inprimaki bat gehitu edozein orritara zuzenean txat-interfazetik

### Hobetua

- **WooCommerce integrazioa berrantolatua** — orain WooCommerce API natiboak erabiltzen ditu fidagarritasun eta bateragarritasun hobea lortzeko
- **Hornitzaileen zerrenda automatikoki freskatzen da** — edozein plugin aktibatzen edo desaktibatzen denean

### Konponduta

- **navigate-to gaitasuna** — administrazio-orri batzuetan berriro kargatzeko begizta infinitua konpondu da
- **list-posts gaitasuna** — orain kategoriaren eta etiketaren izenak slug bihurtzen ditu behar bezala
- **WP-CLI komandoak** — aurreko berrantolaketa baten ondoren falta ziren izen-espazioen aliasak leheneratu dira
- **Ekitaldi-automatizazioa** — automatizazio-taulak oraindik sortu ez diren guneak modu egokian kudeatzen ditu
- **memory-save gaitasuna** — orain izen-espazioaren aurrizki zuzena erabiltzen du sistemaren jarraibideen eraikitzailean
- **Tresnen emaitza eskalarrak** — orain behar bezala biltzen dira AI-ra itzuli aurretik
- **Erabilera-estatistikak** — orain gaitasun-gakoaren formatu zaharra behar bezala kudeatzen dute bertsio zaharragoetatik eguneratzean
