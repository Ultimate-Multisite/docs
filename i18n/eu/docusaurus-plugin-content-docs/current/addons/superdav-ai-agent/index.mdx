# Superdav AI Agent {#superdav-ai-agent}

Superdav AI Agent WordPress-erako AI laguntzaile aurreratua da, Gratis AI Agent hedatzen duena gaiaren araberako trebetasunekin, interneteko bilaketa-gaitasun hobetuak, webgune-eraikitzailearen eginbideak eta aukerako automatizazio-integrazioak gehituz. 1.18.0 bertsioak Google Calendar irakurtzeko tresnak, kontaktuen eta parte-hartzaileen mapaketa, giza onarpen-ateak, oroigarrien deduplikazioa, TextBee SMS jakinarazpenak, kudeatutako Superdav zerbitzuaren hornidura eta garatzaile fidagarrien lan-fluxuetarako bereiz banatutako Advanced companion pakete bat gehitzen ditu.

## Eginbide nagusiak {#key-features}

- **Gaiaren araberako trebetasunak** — automatikoki egokitzen ditu jarraibideak eta gaitasunak zure WordPress gai aktiboaren eta instalatutako pluginen arabera
- **Interneteko bilaketa-hornitzaile anitz** — aukeratu Brave Search eta Tavily artean bilaketa-emaitza aberatsagoak eta ikerketa-gaitasun aurreratuak lortzeko
- **Webgune-eraikitzailearen kontaktu-inprimakiak** — sortu guztiz funtzionalak diren kontaktu-inprimakiak zuzenean txat-interfazetik, instalatuta duzun inprimaki-plugina erabiliz
- **Webgune-eraikitzaile hobetua** — fase anitzeko webgune-eraikuntza plan hobetuak, pluginen aurkikuntza hobearekin eta erroreen berreskuratzearekin
- **Gai-trebetasun integratuak** — Block Themes, Classic Themes, Kadence Blocks eta Kadence Theme-rako jarraibide espezializatuak
- **WooCommerce integrazio hobetua** — berrantolatua WooCommerce API natiboak erabiltzeko fidagarritasun eta bateragarritasun hobea lortzeko
- **Hornitzaileen detekzio automatikoa** — automatikoki detektatzen eta freskatzen ditu erabilgarri dauden pluginak eta gaiak aktibatzen edo desaktibatzen direnean
- **Logotipo pertsonalizatuen sorrera** — sortu eta txertatu SVG logotipo pertsonalizatuak, izen-espazioetarako segurua den saneamendu automatikoarekin
- **Paletaren kontrastearen baliozkotzea** — egiaztatu kolore-pareak WCAG irisgarritasun-betetzerako, zure gaian aplikatu aurretik
- **Argazkiekin hobetutako aurkikuntza** — igo argazkiak Theme Builder aurkikuntza-elkarrizketan diseinuaren norabidean eragiteko
- **Ostalaritzako menuak** — sortu janari eta edarien menu-orri egituratuak jatetxe, kafetegi eta ostalaritza-enpresetarako
- **Mahaigaineko eta mugikorreko aurrebistak** — aurreikusi zure diseinua mahaigaineko eta mugikorreko gailuetan diseinu-norabidea hautatzean
- **1. mailako webguneen scraping-a** — sd-ai-agent/site-scrape orain lehenespenez dago erabilgarri lehiakideen eta inspirazio-webguneen analisia egiteko
- **Google Calendar irakurtzeko tresnak** — irakurri konfiguratutako egutegiak eta gertaerak, automatizazioek datozen ordutegi-testuingurua arrazoitu ahal izan dezaten
- **Kontaktuen eta parte-hartzaileen mapaketa** — lotu egutegiko parte-hartzaileak ezagutzen diren webguneko erabiltzaileekin eta kontaktuekin, jarraipen-lana sortu aurretik
- **Giza onarpen-ateak** — eten automatizazio sentikorrak berrikusteko, mezuak bidali, lana berrabiarazi edo aldaketak egin aurretik
- **Oroigarri-erregistroak eta deduplikazioa** — jarraitu dagoeneko bidali diren oroigarriei, automatizazioaren errepikapenek hartzaile berari bi aldiz jakinarazi ez diezaioten
- **TextBee SMS jakinarazpenak** — bidali konfiguratutako testu-mezuen jakinarazpenak TextBee bidez SMS hornitzaile bat gaituta dagoenean
- **Kudeatutako Superdav zerbitzuaren hornidura** — konektatu onartutako webguneak ostatatutako Superdav endpoint-etara konfigurazio automatikoarekin, eskuzko endpoint konfigurazioaren ordez
- **Advanced companion paketea** — bereiz banatutako garatzaile fidagarrientzako tresnak fitxategi-sistema, datu-base, WP-CLI, REST dispatcher, plugin-eraikitzaile, git snapshot, erabiltzaile-kudeaketa eta benchmark lan-fluxuetarako

## Eskakizunak {#requirements}

- WordPress 6.0 edo berriagoa
- PHP 8.1 edo berriagoa
- WordPress Multisite (sarean aktibatua) edo webgune bakarreko instalazioa
- AI hornitzaile-konektore plugin aktibo bat (Anthropic, OpenAI edo bateragarria)
- Gratis AI Agent v1.9.0 edo berriagoa (Superdav AI Agent-ek Gratis AI Agent hedatzen du)

## Instalazioa {#installation}

1. Igo plugin fitxategiak `/wp-content/plugins/superdav-ai-agent`-era
2. Aktibatu plugina WordPress-eko **Pluginak** pantailaren bidez
3. Ziurtatu Gratis AI Agent ere aktibatuta dagoela
4. Joan **Gratis AI Agent → Ezarpenak** atalera eta konfiguratu zure interneteko bilaketa-hornitzailea, kudeatutako Superdav zerbitzu-konexioa, Google Calendar edo TextBee kredentzialak behar bezala

WordPress.org paketeak Superdav AI Agent-en oinarrizko eginbideak ditu. Advanced companion paketea bereiz eraiki eta banatzen da garatzaile fidagarrien tresnak esplizituki instalatu eta baimenduta dauden inguruneetarako.

## Gaiaren araberako trebetasunak {#theme-aware-skills}

Superdav AI Agent-ek automatikoki detektatzen ditu zure gai aktiboa eta instalatutako pluginak, eta dagokion gaiaren araberako trebetasuna kargatzen du laguntzailearen testuinguruan. Eskuragarri dauden trebetasunak hauek dira:

| Trebetasuna | Honi aplikatzen zaio |
|---|---|
| **Block Themes** | WordPress-en blokeetan oinarritutako arkitektura (Gune osoaren edizioa) erabiltzen duten gaiak |
| **Classic Themes** | PHP txantiloiak eta editore klasikoa erabiltzen dituzten WordPress gai tradizionalak |
| **Kadence Blocks** | Kadence Blocks plugina aktibo duten webguneak |
| **Kadence Theme** | Kadence gaia gai aktibo gisa duten webguneak |

Ikusi [Gaiaren araberako trebetasunak](../../user-guide/configuration/theme-aware-skills) trebetasun bakoitzari buruzko jarraibide zehatzetarako.

## Interneteko bilaketa-hornitzaileak {#internet-search-providers}

Superdav AI Agent v1.10.0 bertsioak **Tavily** gehitzen du Brave Search-en alternatiba gisa interneteko bilaketa-kontsultetarako. Bi hornitzaileek denbora errealeko bilaketa-emaitzak eskaintzen dituzte eta **Gratis AI Agent → Ezarpenak → Aurreratua** atalean konfigura daitezke.

| Hornitzailea | Ezaugarriak | Konfigurazioa |
|---|---|---|
| **Brave Search** | Bilaketa-emaitza azkarrak, pribatutasunean oinarrituak | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Bilaketa-emaitza aberatsagoak, ikerketa-gaitasun aurreratuak | [Tavily API](https://tavily.com/) |

Ikusi [Interneteko bilaketa](../../user-guide/configuration/internet-search) konfigurazio-jarraibideetarako.

## Egutegi-, onespen- eta jakinarazpen-lan-fluxuak {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0-k automatizazio-lan-fluxuak edukitik eta guneak eraikitzeko zereginetatik harago zabaltzen ditu:

- **Google Calendar tresnek** agenteari konfiguratutako egutegiak eta gertaeren xehetasunak irakurtzeko aukera ematen diote, egutegiaren kredentzialak erabilgarri daudenean.
- **Kontaktuen eta bertaratuen laguntzaileek** gertaerako parte-hartzaileak WordPress erabiltzaileekin edo kontaktu ezagunekin mapatzen dituzte, oroigarriak edo jarraipen-zereginak prestatu aurretik.
- **Gizakiaren onespen-ateek** automatizazio bati pausatzeko aukera ematen diote, baimendutako erabiltzaile batek proposatutako ekintza berrikusi arte.
- **Oroigarri-erregistroek** egutegi- edo jakinarazpen-oroigarri bikoiztuak saihesten dituzte automatizazio bat berriro saiatzen denean edo behin eta berriz programatzen denean.
- **TextBee SMS euskarriek** testu-mezu bidezko jakinarazpenak gaitzen dituzte TextBee hornitzailea konfiguratuta dagoenean.

Erabili tresna hauek ordutegia kontuan hartzen duen laguntzarako, bertaratuen jarraipenerako eta oroigarri-lan-fluxuetarako, mezu sentikorrak bidali aurretik oraindik gizakiaren berrikuspena behar dutenean.

## Gune-eraikitzailearen kontaktu-formularioak {#site-builder-contact-forms}

Gune-eraikitzailearen agenteak orain kontaktu-formularioak sor ditzake zuzenean txat-interfazetik. Eskatu besterik ez gune-eraikitzaileari kontaktu-formulario bat sortzeko, eta automatikoki detektatuko du instalatuta duzun formulario plugin-a (Contact Form 7, WPForms, Fluent Forms edo Gravity Forms) eta erabiltzeko prest dagoen formulario bat sortuko du.

Ikusi [Gune-eraikitzailearen kontaktu-formularioak](../../user-guide/administration/site-builder-contact-forms) jarraibide xeheetarako.

## Theme Builder ezaugarriak {#theme-builder-features}

Superdav AI Agent v1.16.0-ko Theme Builder-ek gaitasun berri indartsuak ditu:

### Aurkikuntza-elkarrizketa argazki-igoerarekin {#discovery-interview-with-photo-upload}

Theme Builder-en aurkikuntza-elkarrizketak orain argazkiak igotzeko urrats bat dauka, non inspirazio bisuala eman dezakezun. Igo zure markaren, lehiakideen guneen edo diseinu-inspirazioaren argazkiak, Theme Builder-ek zure diseinu-norabidea uler dezan.

Ikusi [Aurkikuntza-elkarrizketa](./theme-builder/discovery-interview.md) jarraibide xeheetarako.

### Diseinu-norabidea mahaigaineko eta mugikorreko aurrebistekin {#design-direction-with-desktop-and-mobile-previews}

Berrikusi zure diseinua mahaigaineko zein mugikorreko gailuetan Theme Builder-ek zure theme osoa sortu aurretik. Ikusi denbora errealeko aurrebistak hainbat breakpoint moldakorretan.

Ikusi [Diseinu-norabidea](./theme-builder/design-direction.md) jarraibide xeheetarako.

### Ostalaritzako menuak {#hospitality-menus}

Theme Builder-ek orain janari- eta edari-menuen orri egituratuak sor ditzake jatetxe, kafetegi, taberna eta beste ostalaritza-negozio batzuetarako. Menuek prezioak, deskribapenak, dieta-informazioa eta alergeno-oharrak biltzen dituzte.

Ikusi [Ostalaritzako menuak](./theme-builder/hospitality-menus.md) jarraibide xeheetarako.

### Gaitasun berriak {#new-abilities}

- **[Sortu Logo SVG](./abilities/generate-logo-svg.md)** — sortu SVG logo pertsonalizatuak sanitizazio automatikoarekin
- **[Baliozkotu paletaren kontrastea](./abilities/validate-palette-contrast.md)** — egiaztatu kolore-pareak WCAG betetzeko
- **[Sortu menua](./abilities/create-menu.md)** — sortu nabigazio-menuak nabigazio-etiketa bereiziekin

Ikusi [Tresnak eta konfigurazioa](./theme-builder/tools.md) tresna erabilgarriei eta Tier 1 gune-scrape tresnari buruzko informaziorako.

## Konfigurazioa {#configuration}

Ezarpen guztiak **Gratis AI Agent → Settings** atalean daude WordPress administrazioan. Superdav AI Agent-ek konfigurazio-aukera hauek gehitzen ditu:

| Ezarpena | Deskribapena |
|---|---|
| **Internet Search Provider** | Aukeratu Brave Search eta Tavily artean (aukerakoa) |
| **Tavily API Key** | Tavily interneteko bilaketarako API gakoa (Tavily hautatuta badago) |
| **Managed Superdav Service** | Konektatu onartutako guneak ostatatutako Superdav zerbitzura endpoint hornidura automatikoarekin |
| **Google Calendar Credentials** | Konfiguratu egutegi-sarbidea, agenteak egutegiak eta gertaerak irakur ditzan |
| **TextBee SMS Provider** | Konfiguratu TextBee kredentzialak onartutako SMS jakinarazpen-lan-fluxuetarako |

## Advanced companion paketea {#advanced-companion-package}

Superdav AI Agent v1.18.0-k bereiz banatutako Advanced companion pakete bat gehitzen du garatzaile fidagarrien lan-fluxuetarako. Gaitasun hauek ez daude WordPress.org core paketean bilduta. Instalatu companion paketea administratzaileek esplizituki agenteak garatzaile-tresnetara sarbidea izatea fidatzen duten guneetan soilik.

Advanced tresnek fitxategi-sistemako eragiketak, datu-basearen sarbidea, WP-CLI, REST dispatcher lan-fluxuak, plugin eraikitzailearen utilitateak, git snapshots, erabiltzaile-kudeaketarako laguntzaileak eta benchmark tresnak barne hartzen dituzte. Mantendu tresna hauek administratzaile fidagarrientzat mugatuta, eta berrikusi onespen-ateen konfigurazioa produkzioko guneetan gaitu aurretik.
