# VAT {#vat}

Multisite Ultimate: Europako VAT

Multisite Ultimate-ren zerga-kudeaketa lehenetsiari eginbide eta tresna gehigarriak gehitzen dizkio, Europako VAT hobeto onartzeko.
Automatikoki gehitzen ditu EU VAT tasak Zerga-tasen orrian, hemendik

## Instalazioa {#installation}

1. Igo 'ultimate-multisite-vat' '/wp-content/plugins/' direktoriora
2. Aktibatu plugina WordPress-eko 'Pluginak' menuaren bidez
3. Joan Ultimate Multisite > Ezarpenak > Zergak atalera eta gaitu EU VAT.
4. Egin klik Zerga-tasetan, eguneratu EU VAT tasak behealdean, eta gorde zerga-tasak.
5. Eginda!
