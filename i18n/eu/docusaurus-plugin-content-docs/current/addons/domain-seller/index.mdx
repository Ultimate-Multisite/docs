# Domain Seller

**Multisite Ultimate: Domain Seller**-ek zure WordPress Multisite sarea domeinuak saltzeko plataforma indartsu bihurtzen du. Utzi bezeroei domeinu pertsonalizatuak zuzenean erregistratzen izena ematean edo beren dashboardetik, erabilgarritasuna denbora errealean egiaztatuz eta prezio gardenekin.

### Zergatik Domain Seller? {#why-domain-seller}

Domeinu-erregistroa edozein WaaS (Website as a Service) plataformatan gehien eskatzen diren salmenta gehigarrietako bat da. Zure bezeroak hirugarrenen erregistratzaile batera bidali beharrean, domeinuak zuzenean eskain ditzakezu — diru-sarrera gehiago zure sarean mantenduz eta bezeroei esperientzia oztoporik gabea emanez.

### Ezaugarri nagusiak {#key-features}

#### Domeinu-erregistratzaile anitzen euskarria {#multiple-domain-registrar-support}
* **OpenSRS** — Erregistro, berritze eta kudeaketa osoa
* **Namecheap** — WhoisGuard-ekin eta sandbox probekin integrazio osoa
* **HostAfrica** — Domeinu-bilaketa, prezioak, erregistroa, transferentzia, DNS eta ID babesa Domains Reseller API bidez
* **Openprovider** — Saltzaileentzako prezioak, erregistroa, berritzea, transferentziak, DNS, WHOIS pribatutasuna eta TLD sinkronizazioa
* **Hostinger** — Domeinu-bilaketa eta erregistroa partekatutako Hostinger integrazio-tokena erabiliz
* **Arkitektura hedagarria** — Integration Registry ereduan eraikia, hornitzaile gehiago gehitzea erraztuz
* **Konexio-probak integratuta** — Balioztatu hornitzaileen kredentzialak eta konektagarritasuna martxan jarri aurretik

#### Konfigurazio morroi gidatua {#guided-setup-wizard}
* **Klik bakarreko hasierako konfigurazioa** — Hornitzailearen konfigurazioan, TLD inportazioan eta domeinu-produktuaren sorkuntzan gidatzen zaitu
* **TLD sinkronizazio automatikoa** — Eskuragarri dauden TLD guztiak eta handizkako prezioak inportatzen ditu konektatutako hornitzaileetatik
* **Produktu lehenetsiaren sorkuntza** — Erabiltzeko prest dagoen domeinu-produktu bat sortzen du zentzuzko marjina lehenetsiekin

#### Domeinu-prezio malguak {#flexible-domain-pricing}
* **Hiru marjina modu** — Ehuneko bidezko marjina, marjina finkoa edo TLD bakoitzeko prezio finkoa
* **TLD espezifikoko produktuak** — Sortu domeinu-produktu bereiziak TLD talde desberdinetarako (.com/.net, .io, herrialde-kodeak, etab.)
* **Sarrerako prezioak** — Eskaini lehen urteko prezio deskontatua erregistro berriak erakartzeko
* **Denbora errealeko prezioak** — Hornitzaileen zuzeneko handizkako prezioak cache adimendunarekin
* **Kupoien euskarria** — Multisite Ultimate deskontu-kodeekin bateragarritasun osoa

#### WHOIS pribatutasun-babesa {#whois-privacy-protection}
* **Produktu bakoitzeko konfigurazioa** — Ezarri WHOIS pribatutasuna Desgaituta, Beti barne edo Bezeroaren aukera gisa
* **Bezeroaren baimen esplizitua** — Bezeroaren aukera gisa ezarrita dagoenean, checkout-eko kontrol-lauki batek bezeroei pribatutasun-babesa gehitzeko aukera ematen die
* **Prezio konfiguragarria** — Ezarri urteko prezio bat pribatutasun gehigarriarentzat, edo eskaini doan
* **Erregistratzailearen integrazioa** — Automatikoki gaitzen du WhoisGuard (Namecheap) edo WHOIS pribatutasuna (OpenSRS) erregistro-unean

#### Checkout esperientzia bateratua {#unified-checkout-experience}
* **Hiru domeinu modu** — Bezeroek azpidomeinu baten (doan), domeinu berri bat erregistratzearen edo dagoeneko jabetzan duten domeinu bat mapatzearen artean aukeratzen dute
* **Denbora errealeko domeinu-bilaketa** — AJAX bidezko erabilgarritasun-egiaztapena, orria berriro kargatu gabe
* **Iradokizun adimendunak** — TLD alternatiboen iradokizunak nahi den domeinua hartuta dagoenean
* **Prezio gardenak** — Erregistro-prezioaren, berritze-prezioaren eta aukerako WHOIS pribatutasun-kuotaren bistaratze argia
* **Erregistratzailearen kontaktu-eremuak** — Beharrezko erregistratzaile-informazioa (izena, helbidea, telefonoa) inline biltzen du checkout-ean
* **Automatikoki sortutako gunearen URLa** — Gunearen helbidea automatikoki eratortzen da aukeratutako domeinutik

#### Berritze automatikoaren sistema {#auto-renewal-system}
* **Membership-i lotutako berritzeak** — Domeinuak automatikoki berritzen dira bezeroaren membership aktibo dagoen bitartean
* **Programazio adimenduna** — Berritze-lanak ilaran jartzen dira domeinuen iraungitze-daten arabera
* **Berriro saiatzeko logika** — Huts egindako berritze-saiakeretarako berriro saiakera adimendunak
* **Email jakinarazpenak** — Konfiguragarriak diren alertak hurrengo berritzeetarako, berritze arrakastatsuetarako, hutsegiteetarako eta iraungitzeetarako

#### Bezeroaren DNS kudeaketa {#customer-dns-management}
* **Autozerbitzuko DNS** — Bezeroek DNS erregistroak (A, AAAA, CNAME, MX, TXT) gehitu, editatu eta ezabatu ditzakete beren dashboardetik
* **Domeinuaren egoera** — Denbora errealeko erregistro-egoera, iraungitze-datak eta nameserver informazioa
* **Domeinu-mapaketa automatikoa** — Erregistratutako edo lehendik dauden domeinuak automatikoki mapatzen dira bezeroaren gunera

#### Administrazio-tresnak {#admin-tools}
* **Eskuzko domeinu-erregistroa** — Erregistratu domeinuak bezeroen izenean administrazio-paneletik
* **Domeinuen monitorizazioa** — Ikusi bezero guztien domeinuak, egoerak eta iraungitze-datak orri bakar batean
* **Hornitzaile espezifikoko erregistroak** — Erregistro-kanal bereiziak erregistratzaile bakoitzarentzat (adib., domain-seller-namecheap.log)
* **Erregistro-xehetasunak** — Ikusi erregistratzailearen informazioa eta erregistro-datuak oinarrizko domeinuaren edizio-orrian

### Onartutako hornitzaileak {#supported-providers}

* **OpenSRS** — Integrazio osoa erregistroarekin, berritzearekin, DNS kudeaketarekin eta WHOIS pribatutasunarekin
* **Namecheap** — Integrazio osoa erregistroarekin, berritzearekin, erabilgarritasunaren lote-egiaztapenarekin eta doako WhoisGuard-ekin
* **HostAfrica** — Erregistroa, berritzea, transferentziak, DNS kudeaketa, EPP kodea eskuratzea, erregistratzaile-blokeoa eta ID babesa
* **Openprovider** — Erregistroa, berritzea, transferentziak, DNS kudeaketa, EPP kodea eskuratzea, erregistratzaile-blokeoa, WHOIS pribatutasuna eta saltzaileentzako TLD sinkronizazioa
* **Hostinger** — Erregistroa, domeinu-bilaketa, nameserver eguneraketak, erregistratzaile-blokeoa eta WHOIS pribatutasuna; berritzeak Hostinger auto-renew-ek kudeatzen ditu
* **GoDaddy** — Erregistroa eta berritzea
* **ResellerClub** — Erregistroa, berritzea eta DNS kudeaketa
* **NameSilo** — Erregistroa eta berritzea
* **Enom** — Erregistroa, berritzea eta DNS kudeaketa

### Xehetasun teknikoak {#technical-details}

* **Behar du:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **Arkitektura:** Integration Registry eredua Domain_Selling_Capability interfazearekin
* **Autoloading:** Jetpack autoloader PSR-4 klase-mapaketarekin
* **Estandarrak:** WordPress kode-estandarrak, tipatze zorrotza, prestatutako adierazpenak

## Instalazioa {#installation}

1. Ziurtatu **Multisite Ultimate v2.4.12+** instalatuta eta sarean aktibatuta dagoela
2. Kargatu plugina hona: `/wp-content/plugins/ultimate-multisite-domain-seller/`
3. Aktibatu plugina sarean WordPress Sareko Administrazioko **Pluginak** ataletik
4. **Konfigurazio morroia** automatikoki abiarazten da lehen aktibazioan — jarraitu urratsei hau egiteko:
   * Konektatu zure domeinu-erregistratzailea
   * Probatu konexioa
   * Inportatu eskuragarri dauden TLDak eta handizkako prezioak
   * Sortu zure lehen domeinu-produktua prezio-gainkargarekin
5. Gehitu **Domeinu-hautaketa** eremua zure ordainketa-inprimakira **Ultimate Multisite > Ordainketa-inprimakiak** atalean

## Maiz egiten diren galderak {#frequently-asked-questions}

### Multisite Ultimate-ren zein bertsio behar da? {#what-version-of-multisite-ultimate-is-required}

2.4.12 bertsioa edo berriagoa. Gehigarria ez da bateragarria WP Ultimo v1.x-rekin edo 2.4.12 baino lehenagoko Ultimate Multisite bertsioekin.

### Zein domeinu-hornitzaile onartzen dira? {#which-domain-providers-are-supported}

Bederatzi erregistratzaile onartzen dira: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo** eta **Enom**. Gaitasunak hornitzailearen arabera aldatzen dira, erregistratzaileen APIek eragiketa desberdinak eskaintzen dituztelako; ikusi konfigurazio-gida hornitzailez hornitzaileko ikuspegi orokorrerako. Gehigarriaren Integration Registry arkitekturak hornitzaile pertsonalizatuak gehitzea errazten du — ikusi [garatzaileentzako gida](/developer/integration-guide/custom-registrar).

### WHOIS pribatutasun-babesa eskain dezaket? {#can-i-offer-whois-privacy-protection}

Bai. Domeinu-produktu bakoitzak WHOIS Pribatutasun ezarpen bat du hiru modurekin: Desgaituta, Beti barne, edo Bezeroaren aukera. Bezeroaren aukera gisa ezarrita dagoenean, bezeroek kontrol-lauki bat ikusiko dute ordainketan prezioarekin (edo "Doan" Namecheap-en WhoisGuard-entzat). Pribatutasun-marka automatikoki bidaltzen zaio erregistratzaileari erregistro-unean.

### TLD desberdinetarako prezio desberdinak ezar ditzaket? {#can-i-set-different-pricing-for-different-tlds}

Bai. Sortu domeinu-produktu bereiziak TLD talde desberdinetarako. Produktu bakoitzak bere gainkarga mota (ehunekoa, gainkarga finkoa edo prezio finkoa), onartutako TLDak eta sarrerako prezioak izan ditzake.

### Zer ordainketa-aukera ikusten dituzte bezeroek? {#what-checkout-options-do-customers-see}

Domeinu-hautaketa eremuak hiru fitxa ematen dizkie bezeroei: **Azpidomeinua** (erabili doako azpidomeinu bat zure sarean), **Erregistratu domeinu berria** (bilatu eta erregistratu domeinu berri bat), eta **Lehendik dagoen domeinua** (mapatu jada baduten domeinu bat). Gaitzen dituzun moduak bakarrik agertuko dira.

### Berritze automatikoak onartzen ditu? {#does-it-support-automatic-renewals}

Bai. Domeinuak automatikoki berritzeko ezar daitezke eta bezeroaren kidetzaren egoerari lotuta daude. Berritze-ordainketak bezeroaren lehendik dagoen ordainketa-pasabidearen bidez prozesatzen dira.

### Bezeroek beren DNS erregistroak kudea ditzakete? {#can-customers-manage-their-dns-records}

Bai. Bezeroek DNS kudeaketa-interfaze bat dute beren arbelean, A, AAAA, CNAME, MX eta TXT erregistroak gehitzeko, editatzeko eta ezabatzeko. DNS kudeaketa eskuragarri dago OpenSRS, ResellerClub, Enom, HostAfrica eta Openprovider-ekin. Hostinger DNS ostatatutako domeinuetarako Hostinger-en oinarrizko domeinu-mapatze integrazioak kudeatzen du.

### Ba al dago konfigurazio-morroirik? {#is-there-a-setup-wizard}

Bai. Konfigurazio-morroia automatikoki exekutatzen da lehen aktibazioan eta erregistratzaile bat konektatzeko, TLDak inportatzeko eta zure lehen domeinu-produktua sortzeko prozesuan gidatzen zaitu. Morroia berriro exekuta dezakezu gehigarriaren ezarpenetatik.
