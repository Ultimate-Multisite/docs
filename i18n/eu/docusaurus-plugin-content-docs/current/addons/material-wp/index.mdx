# Material WP {#material-wp}

WordPress admin interfazea berdina izan da urte luzez. Baina inoiz galdetu al diozu zeure buruari nolakoa izango litzatekeen admin interfazearen Material Design bertsio bat? 

Material WP admin theme bat da, itxura eta sentsazio hori emateko garatua, zure bezeroei Dashboard pertsonalizatuago bat eskaintzen lagunduz.

## Ezaugarriak {#features}

- CSS pertsonalizaturako euskarria
- Estilo-ezarpen errazak
- Menua berrantolatzea
- Inportazio/esportazio erraza

## Dokumentazioa {#documentation}

**Zure admin panela Material WP-rekin pertsonalizatzea**

## Instalazioa {#installation}

1. Igo 'material-wp' '/wp-content/plugins/' direktoriora
2. Aktibatu plugin-a WordPress-eko 'Pluginak' menuaren bidez
