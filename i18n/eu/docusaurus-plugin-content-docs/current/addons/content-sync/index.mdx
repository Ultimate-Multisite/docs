# Eduki-sinkronizazioa

Multisite Ultimate: Eduki-sinkronizazioa

Multisite Ultimate-rako addon honek sareko administratzaileei edukia edozein gunetatik sareko beste gune batzuetara sinkronizatzeko aukera ematen die. Argitalpen mota guztiak onartzen ditu, besteak beste argitalpenak, orriak, pertsonalizatutako argitalpen motak eta Elementor txantiloiak. Addonak urrats anitzeko morroi-interfaze bat dauka edukia eta helburuko guneak hautatzeko, batch prozesamenduarekin eta denbora errealeko aurrerapen-feedbackarekin.

## Ezaugarriak {#features}

* Eduki-hautaketa: Txartel-sare bisuala miniaturak eta argitalpen motaren araberako iragazketa dituena
* Argitalpen mota guztiak: Sinkronizatu argitalpenak, orriak, produktuak, Elementor txantiloiak eta edozein pertsonalizatutako argitalpen mota
* Gune-hautaketa: Hautatu edukia zein gunetara sinkronizatu, iturburu-gunea baztertuta eta haur-guneak aurrez hautatuta
* Batch prozesamendua: Aldi bakoitzean 10 gunetara sinkronizatzen du, aurrerapen-eguneraketekin
* Media sinkronizazioa: Automatikoki kopiatzen eta berriro mapatzen ditu media fitxategiak (irudiak, bideoak) argitalpenaren edukitik helburuko guneetara
* Taxonomia-sinkronizazioa: Kategoriak, etiketak eta pertsonalizatutako taxonomiak mantentzen ditu
* Elementor euskarria: Elementor txantiloientzako euskarri osoa, CSS birsortzearekin eta txantiloi-IDen berriro mapatzearekin
* Gatazken ebazpena: Lehendik dagoen edukia ordezkatzeko edo saltatzeko aukera
* Cache-kudeaketa: Automatikoki garbitzen du Elementor cachea helburuko guneetan sinkronizazioaren ondoren

## Eskakizunak {#requirements}

* WordPress 5.3 edo berriagoa
* PHP 7.4 edo berriagoa
* Multisite Ultimate plugina (aktibo)
* Elementor plugina (aukerakoa, beharrezkoa Elementor txantiloiak sinkronizatzeko)

## Instalazioa {#installation}

1. Kargatu 'ultimate-multisite-content-sync' '/wp-content/plugins/' direktoriora
2. Aktibatu plugina zure WordPress Network Admin-eko 'Pluginak' menuaren bidez
3. Konfiguratu ezarpenak Ultimate Multisite → Ezarpenak → Eduki-sinkronizazioa atalean
4. Joan Ultimate Multisite → Eduki-sinkronizazioa atalera sinkronizatzen hasteko
