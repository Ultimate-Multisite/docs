# Content Sync aldaketen erregistroa {#content-sync-changelog}

### 1.1.0 bertsioa - 2025-12-31n argitaratua {#version-110---released-on-2025-12-31}
* Berria: post mota guztiak sinkronizatzeko euskarria (postak, orriak, post mota pertsonalizatuak)
* Berria: post motaren iragazketa eduki-hautapenaren urratsean
* Berria: iturburu-gunea hautatzeko urratsa - edozein gunetatik sinkronizatu, ez soilik gune nagusitik
* Berria: gune semeak aurrez hautatzea txantiloi-guneetarako
* Berria: multimedia-erauzketa `post_content`-etik post mota guztietarako
* Berria: taxonomien sinkronizazioa (kategoriak, etiketak, taxonomia pertsonalizatuak)
* Aldatua: "Elementor Template Sync" izenetik "Content Sync" izenera aldatu da
* Aldatua: "Select Templates" urratsari "Select Data" izena jarri zaio
* Hobetua: Elementor-eko kode espezifikoa orain `elementor_library` post motarako bakarrik exekutatzen da

* Ezarpenen integrazioa Ultimate Multisite-rekin
* Guneak baztertzeko euskarria sare-ezarpenen bidez
* Elementor cachea garbitzea sinkronizazioaren ondoren
* Txantiloi-taxonomiaren mantentzea
