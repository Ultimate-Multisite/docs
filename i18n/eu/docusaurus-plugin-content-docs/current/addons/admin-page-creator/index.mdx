# Admin Page Creator

## Edukia {#content}
Admin Page Creator-ek aukera emango dizu WordPress admin dashboard barruan admin orri eta azpiorri berriak erraz gehitzeko. Ongietorri-orri bat edo tutorial-orri bat sortzeko erabil daiteke, erabiltzaile berrientzako helmuga-orri gisa balio dezakeena. Beaver Builder, Elementor, Brizy eta Oxygen bezalako orri-eraikitzaileak erabil ditzakezu orriak arazorik gabe diseinatzeko.

HTML, CSS eta JavaScript pertsonalizatuaren euskarria ere eskaintzen du, baita planaren eta erabiltzaile-rolaren araberako sarbide-kontrola ere!

## Ezaugarriak {#features}
* HTML eta CSS pertsonalizatuaren euskarria
* JavaScript euskarria
* PHP eduki modua
* Erabiltzaile-rolaren araberako baimena
* Orri-eraikitzaileentzako euskarria

## Instalazioa {#installation}

1. Kargatu **ultimate-multisite-admin-page-creator** **/wp-content/plugins/** direktoriora
2. Aktibatu plugina zure WordPress Network Admin-eko **Plugins** menuaren bidez

## Erabilera-gida {#usage-guide}

WordPress-ek guneen aurrealdean edukia gehitzea oso erraza egiten duen arren, admin aldean orriak eta azpiorriak sortzeko oinarrizko kodeketa-trebetasun batzuk behar dira. Ultimate Multisite: Admin Page Creator-en helburua da aurrealdeko erabilera-erraztasun bera zure sareko guneen atzealdera ekartzea.

## Zergatik erabili Admin Page Creator? {#why-use-admin-page-creator}

Agian zure bezeroen admin panelean tutorialak dituen orri pertsonalizatu bat eskaini nahi duzu, edo agian FAQak eta kontaktu-widget bat dituen orri pertsonalizatu bat gehitu nahi duzu. Aukerak amaigabeak dira Admin Page Creator-ekin, eta hori aztertuko dugu gaurko tutorialean.

## Zure lehen admin orri pertsonalizatua sortzea {#creating-your-first-custom-admin-page}

Ultimate Multisite: Admin Page Creator instalatu eta aktibatu ondoren, joan zure sareko admin menura eta sartu Admin Pages atalera.

![Admin orri pertsonalizatuen zerrenda Network Admin-en](/img/admin/admin-pages-list.png)

_Admin orri pertsonalizatuen zerrenda Network Admin-en_

Orain arte sortu dituzun admin orri guztien zerrenda ikusiko duzu (zure lehen bisita denez, zerrenda hutsik egongo da). Egin klik **Gehitu admin orri berria** botoian. Admin Page editorera birbideratuko zaituzte.

## Menu-aukerak {#menu-options}

Alboan, General Options meta-boxean, menu mota, etiketa eta askoz gehiago konfiguratu ahal izango dituzu.

### Menu motak {#menu-types}

Zure admin orri pertsonalizatu berriak bi menu mota izan ditzake: goi-mailako menua eta azpimenua.

Goi-mailako menu-orri batek, izenak dioen bezala, sortu berri duzun orria admin menura gehituko du WordPress-en orri lehenetsien ondoan.

Goi-mailako aukera hautatzen baduzu, menuaren ordena eta menu-ikonoa ere hautatu ahal izango dituzu.

Orri hau azpimenu-orri bihurtu nahi baduzu, orri nagusi bat hautatu ahal izango duzu orri hau hari lotzeko. Zure orri pertsonalizatua WordPress-en orri lehenetsietara edo zure goi-mailako admin orri pertsonalizatuetara gehitu dezakezu.

_Pertsonalizatu erabat menu-elementua orri pertsonalizatu honetarako_

_Orri hau beste menu-orri batzuen azpimenu gisa ere gehi dezakezu, zure goi-mailako pertsonalizatuak barne._

## Aukera aurreratuak {#advanced-options}

Zure admin orri pertsonalizatuetan era guztietako edukia gehitzeko gai izan beharko zenuke. Horrek CSS arauak eta fitxategiak, JavaScript eta kanpoko liburutegiak ere barne hartzen ditu. Hori Advanced Options fitxa erabiliz egin dezakezu.

### CSS pertsonalizatua {#custom-css}

CSS fitxan, zure CSS kode pertsonalizatua gehi dezakezu, baita urruneko guneetatik CSS fitxategiak inportatu ere (CDNak, Google Fonts, etab.).

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_CSS pertsonalizatua ere onartzen da. Kanpoko estilo-fitxategiak ere sar ditzakezu!_

### JavaScript pertsonalizatua {#custom-javascript}

Gauza bera balio du JavaScript-entzat. Kanpoko liburutegiak gehitu ditzakezu zure JavaScript kode pertsonalizatuan erabiltzeko.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Gehitu zure JavaScript kodea!_

## Baimenak {#permissions}

Advanced Options-en ere, orri honetara sarbidea izango duten azpiguneak hautatu ahal izango dituzu. Hau erabilgarria da plan-maila desberdinei eduki esklusibo desberdina eskaini nahi badiezu, adibidez.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Kontrol osoa duzu orri hau non agertuko den zehazteko_

**Garrantzitsua** : Bi baldintzak bete behar dira orria bistaratzeko. Horrek esan nahi du erabiltzailea plan jakin bateko bezeroa bada, baina hautatutako roletako bat ez badu (edo alderantziz), orria ez dela bere panelean agertuko.

## Aktibo {#active}

Administrazio-orri pertsonalizatu bat ezabatu gabe edo baimenekin nahastu gabe desgaitu nahi baduzu, desmarkatu Active aukera.

Goi-mailako orri bat desaktibatzeak automatikoki desaktibatuko ditu guraso-orri horri lotutako azpiorri guztiak.
