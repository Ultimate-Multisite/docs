# Iragazkia: wu_minimum_password_strength {#filter-wuminimumpasswordstrength}

Iragazi eskatzen den gutxieneko pasahitzaren sendotasuna (zxcvbn puntuazioa).

Sendotasun-mailak: - 0, 1: Oso ahula - 2: Ahula - 3: Ertaina - 4: Sendoa (lehenetsia)

## Parametroak {#parameters}

| Izena | Mota | Deskribapena |
|------|------|-------------|
| $min_strength | `int` | Eskatzen den gutxieneko sendotasun-maila. |
| $strength_setting | `string` | Admin ezarpenaren balioa (medium, strong, super_strong). |

- 2.4.0
