# Iragazkia: wp_ultimo_render_vars {#filter-wpultimorendervars}

## Parametroak {#parameters}

| Izena | Mota | Deskribapena |
|------|------|-------------|
| $args | `array` | Errendatze-deiak pasatutako aldagaiak dituen arraya. |
| $view | `string` | Errendatu beharreko ikuspegiaren izena. |
| $default_view | `string` | fallback_view-ren izena |
