# Ekintza: wu_maybe_create_customer {#action-wumaybecreatecustomer}

Gehigarrien garatzaileei aukera ematen die bezeroa gehitzen denean gauza gehigarriak egiteko.

Hemen gehitzen ditugu bezeroa-&gt;erabiltzailea gune nagusira ere gehitzeko hook-ak, adibidez.

## Parametroak {#parameters}

| Izena | Mota | Deskribapena |
|------|------|-------------|
| $customer | `\Customer` | Agian sortu den bezeroa. |
| $checkout | `\Checkout` | Ordainketa-prozesuaren uneko klasea. |
