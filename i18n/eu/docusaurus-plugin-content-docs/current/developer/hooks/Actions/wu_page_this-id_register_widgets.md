# Action: wu_page_\{$this->id\}_register_widgets

Orri honetarako widgetak erregistratu ondoren abiarazten da.

The dynamic portion of the hook name, `$this-&gt;id`, refers to the page id.

## Parametroak

| Izena | Mota | Deskribapena |
|------|------|-------------|
| $id | `string` | Orriaren id-a. |
| $page_hook | `string` | Orriaren hook-a. |
| $page | `object` | Orriaren objektua. |
