# Gratis AI Agent {#gratis-ai-agent}

Gratis AI Agent é um assistente de IA autônomo para WordPress que pode planejar, construir e gerenciar sites WordPress completos usando instruções em linguagem natural. Ele expõe um conjunto estruturado de **habilidades** — ações discretas e testáveis que o agente pode invocar — cobrindo desde gerenciamento de conteúdo até customização do sistema de design. A Versão 1.9.0 adiciona habilidades de criação de conteúdo, revisão visual de páginas via capturas de tela do lado do cliente, cinco perfis de agente embutidos e flags de recurso para controle de acesso e branding.

## Principais Recursos {#key-features}

- **Sistema de Habilidades** — um catálogo modular e extensível de ações que o agente pode invocar na sua instalação WordPress
- **Gerenciamento de Conteúdo** — criar posts e páginas (incluindo atribuição de templates de página), criar vários posts em lote em uma única chamada, definir imagens em destaque e construir formulários de contato
- **Revisão Visual** — capturar capturas de tela de páginas ativas, comparar estados antes/depois e obter uma revisão de design gerada por IA, cobrindo layout, tipografia, cor e acessibilidade
- **Cinco Agentes Embutidos** — Content Writer, Site Builder, Design Studio, Plugin Manager e Support Assistant; alternáveis via Seletor de Agente no cabeçalho do chat
- **Flags de Recurso** — toggles de controle de acesso e branding em Configurações → Flags de Recurso para restrições de função, nome de marca (white-label) e customização de widgets
- **Gerenciamento de Tipos de Conteúdo Personalizados** — registrar, listar e excluir tipos de conteúdo personalizados com persistência após reinicializações
- **Gerenciamento de Taxonomias Personalizadas** — registrar, listar e excluir taxonomias personalizadas com persistência
- **Habilidades de Sistema de Design** — injetar CSS personalizado, gerenciar padrões de blocos curados, definir o logo do site e aplicar presets theme.json
- **Gerenciamento de Estilos Globais** — ler e escrever valores theme.json para paletas de cores, tipografia, espaçamento e muito mais
- **Gerenciamento de Menu de Navegação** — criar e atualizar menus de navegação e seus itens programaticamente
- **Gerenciamento de Opções** — ler e escrever opções do WordPress com uma lista de bloqueio de segurança embutida para proteger configurações críticas
- **Registro de Habilidades Instaláveis** — descobrir e ativar pacotes de habilidades adicionais sem modificar os arquivos principais do plugin
- **Orquestração do Site Builder v2** — planos de construção de site em múltiplas etapas com descoberta de plugins, rastreamento de progresso e recuperação de erros
- **Comando de Benchmark WP-CLI** — executar o conjunto de testes de benchmark da capacidade do agente a partir da linha de comando para CI e avaliação de modelos

## Requisitos {#requirements}

- WordPress 6.0 ou superior
- PHP 8.1 ou superior
- WordPress Multisite (ativado na rede) ou instalação de site único
- Um plugin conector de IA ativo (Anthropic, OpenAI ou compatível)

## Instalação {#installation}

1. Faça o upload dos arquivos do plugin para `/wp-content/plugins/gratis-ai-agent`
2. Ative o plugin na tela **Plugins** do WordPress
3. Navegue até **Gratis AI Agent → Configurações** e conecte seu provedor de IA
4. (Opcional) Navegue pelo **Registro de Habilidades** para instalar pacotes de habilidades adicionais

## Início Rápido {#quick-start}

Após a ativação e conexão com um provedor de IA, abra o painel **Gratis AI Agent** na barra lateral do administrador do WordPress. Digite uma instrução em linguagem natural e o agente planejará e executará as etapas necessárias usando suas habilidades disponíveis.

**Instruções de exemplo:**

```
Crie um tipo de conteúdo personalizado "Portfólio" com uma taxonomia "Categoria de Projeto".
```

```
Injete uma paleta de cores de modo escuro no tema e defina a cor primária como #1a1a2e.
```

```
Construa um site de restaurante com um menu online, formulário de reserva e página de contato.
```

## Visão Geral das Habilidades {#abilities-overview}

Habilidades são as ações atômicas que o agente pode executar. Cada habilidade é uma classe PHP registrada que expõe um esquema (parâmetros e tipos de retorno) que o agente lê em tempo de execução. As habilidades são agrupadas em áreas funcionais:

| Área | Habilidades |
|---|---|
| **Gerenciamento de Conteúdo** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **Revisão Visual** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **Tipos de Conteúdo Personalizados** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **Taxonomias Personalizadas** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **Sistema de Design** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **Estilos Globais** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **Menus de Navegação** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **Gerenciamento de Opções** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **Habilidades Instaláveis** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **Site Builder** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

Consulte a [Referência de Habilidades](./abilities) para documentação completa de parâmetros e exemplos de uso.

## WP-CLI {#wp-cli}

Gratis AI Agent inclui uma família de comandos `wp gratis-ai-agent` para executar benchmarks e gerenciar habilidades a partir da linha de comando. Consulte a [Referência WP-CLI](./wp-cli) para a lista completa de comandos.

```bash
# Executa o conjunto completo de benchmarks
wp gratis-ai-agent benchmark

# Lista as habilidades instaladas
wp gratis-ai-agent abilities list

# Instala um pacote de habilidades
wp gratis-ai-agent abilities install <slug>
```

## Configuração {#configuration}

Todas as configurações estão em **Gratis AI Agent → Configurações** no painel do WordPress:

| Configuração | Descrição |
|---|---|
| **Provedor de IA** | Selecione e configure o conector de provedor de IA ativo |
| **Modelo** | Escolha o modelo de linguagem a ser usado para as sessões do agente |
| **Max Tokens** | Orçamento de tokens por solicitação (padrão 4096) |
| **Timeout da Habilidade** | Tempo máximo em segundos que uma habilidade individual pode rodar (padrão 30) |
| **Log de Debug** | Registra chamadas de habilidades e raciocínio do agente no log de debug do WordPress |

## Lista de Bloqueio de Segurança de Gerenciamento de Opções {#options-management-safety-blocklist}

A habilidade de Gerenciamento de Opções inclui uma lista de bloqueio embutida que impede o agente de ler ou escrever opções sensíveis do WordPress. A lista de bloqueio padrão cobre:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (opções de URL — use uma ferramenta de migração dedicada para estas)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (chaves secretas)
- `active_plugins`, `template` (ativação de plugin/tema — use habilidades dedicadas)
- Credenciais de gateway de pagamento e chaves de API registradas por plugins instalados

Entradas adicionais podem ser adicionadas via filtro `gratis_ai_agent_options_blocklist`.

## Hooks e Filtros {#hooks-and-filters}

### Actions {#actions}

- `gratis_ai_agent_ability_registered` — disparado após o registro de uma habilidade, recebe o nome da classe da habilidade
- `gratis_ai_agent_plan_started` — disparado quando um plano de site builder começa a execução
- `gratis_ai_agent_plan_completed` — disparado quando um plano é concluído com sucesso
- `gratis_ai_agent_plan_failed` — disparado quando um plano encontra um erro irrecuperável

### Filters {#filters}

- `gratis_ai_agent_abilities` — modifica a lista de habilidades registradas antes que o agente a carregue
- `gratis_ai_agent_options_blocklist` — estende ou reduz a lista de bloqueio de segurança de opções
- `gratis_ai_agent_benchmark_questions` — adiciona perguntas de benchmark personalizadas ao conjunto de avaliação
- `gratis_ai_agent_plan_plugins` — modifica a lista de descoberta de plugins durante a geração do plano do site
