# Admin Page Creator

## Content
O Admin Page Creator permitirá que você adicione facilmente novas páginas de administração e subpáginas dentro do painel de administração do WordPress. Ele pode ser usado para criar uma página de boas-vindas ou página de tutorial que pode servir como página de destino para novos usuários. Você pode usar construtores de página como Beaver Builder, Elementor, Brizy e Oxygen para projetar as páginas sem complicações.

Ele também oferece suporte a HTML, CSS e JavaScript personalizados, além de controle de acesso baseado no plano e na função do usuário!

## Features
* Suporte a HTML e CSS personalizados
* Suporte a JavaScript
* Modo de Conteúdo PHP
* Permissão baseada na função do usuário
* Suporte a construtores de página

## Installation

1. Faça upload do **ultimate-multisite-admin-page-creator** para o diretório **/wp-content/plugins/**
2. Ative o plugin através do menu **Plugins** no seu Admin de Rede WordPress

## Usage Guide

Embora o WordPress facilite muito a adição de conteúdo ao front-end dos sites, criar páginas e subpáginas no lado administrativo requer algumas habilidades básicas de codificação. O Ultimate Multisite: Admin Page Creator tem como objetivo trazer essa mesma facilidade de uso do front-end para o back-end dos seus sites em rede.

## Why use Admin Page Creator?

Talvez você queira oferecer uma página personalizada no painel de administração dos seus clientes com tutoriais, talvez queira adicionar uma página personalizada com FAQs e um widget de contato. As possibilidades são infinitas com o Admin Page Creator, e é isso que exploraremos neste tutorial.

## Creating your first Custom Admin Page

Após instalar e ativar o Ultimate Multisite: Admin Page Creator, vá ao menu de administração da rede e acesse Páginas de Administração.

![Custom Admin Pages list on Network Admin](/img/admin/admin-pages-list.png)

_Lista de Páginas de Administração Personalizadas no Admin de Rede_

Você verá uma lista de todas as páginas de administração que criou até agora (como esta é sua primeira visita, a lista ficará vazia). Clique em **Add new Admin Page**. Você será redirecionado para o editor de Páginas de Administração.

### Page Title

Este é o título que será exibido no topo da página, não o rótulo do menu de administração.

<!-- Screenshot unavailable: Admin Page editor showing Page Title field -->

### Content Type Selector

O Ultimate Multisite: Admin Page Creator oferece duas opções para o conteúdo da página: você pode usar o editor WYSIWYG padrão do WordPress ou mudar para um editor HTML.

Se você não está familiarizado com HTML, mantenha o editor padrão do WordPress. Você poderá editar o conteúdo da página como faria com um Post ou Página normal.

## Menu Options

No lado, na meta-box Opções Gerais, você poderá configurar o tipo de menu, rótulo e muito mais.

### Menu Types

Sua nova página de administração personalizada pode ter dois tipos de menu diferentes: menu de nível superior e submenu.

Uma página de menu de nível superior, como o nome indica, adicionará sua página recém-criada ao menu de administração ao lado das páginas padrão do WordPress.

Se você selecionar a opção de nível superior, poderá escolher a ordem do menu e um ícone de menu também.

Se você quiser tornar esta página um submenu, poderá selecionar uma página pai para anexar esta página. Você pode adicionar sua página personalizada às páginas padrão do WordPress ou às suas páginas de administração personalizadas de nível superior.

## Advanced Options

Você deve ser capaz de adicionar todo tipo de conteúdo às suas páginas de administração personalizadas. Isso inclui regras e arquivos CSS, JavaScript e bibliotecas externas também. Você pode fazer isso usando a aba Opções Avançadas.

### Custom CSS

Na aba CSS, você pode adicionar seu próprio código CSS personalizado, bem como importar arquivos CSS de sites remotos (como CDNs, Google Fonts, etc.).

_CSS personalizado também é suportado. Você também pode incluir arquivos de estilo externos!_

### Custom JavaScript

O mesmo vale para JavaScript. Você pode adicionar bibliotecas externas para usar em seu código JavaScript personalizado.

_Adicione seu próprio código JavaScript!_

## Permissions

Também nas Opções Avançadas, você poderá selecionar quais sub-sites terão acesso a esta página. Isso é útil se você planeja oferecer conteúdo exclusivo diferente para diferentes níveis de plano, por exemplo.

_Você tem controle total sobre onde esta página aparecerá_

**Importante** : Ambas as condições devem ser atendidas para que a página seja exibida. Isso significa que, se o usuário for cliente de um determinado plano, mas não tiver uma das funções selecionadas (ou vice-versa), a página não aparecerá em seu painel.

## Active

Se você quiser desativar uma página de administração personalizada sem excluí-la ou mexer nas permissões, basta desmarcar a opção Ativo.

Desativar uma página de nível superior desativará automaticamente todas as subpáginas anexadas a essa página pai.
