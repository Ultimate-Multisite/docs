## 🚦 Como funciona {#-how-it-works}

### 1. **Projete seu formulário** {#1-design-your-form}

Use o construtor de arrastar e soltar do Gravity Forms para criar a experiência de registro perfeita.

### 2. **Mapeie seus campos** {#2-map-your-fields}

Conecte os campos do formulário aos pontos de dados do Multisite Ultimate com nossa interface de mapeamento visual.

### 3. **Configure o pagamento** {#3-configure-payment}

Escolha entre redirecionamento de checkout ou integração com WooCommerce para processamento de pagamento.

### 4. **Coloque em produção** {#4-go-live}

Publique seu formulário e comece a converter visitantes em clientes pagantes!
