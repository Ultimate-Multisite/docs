# Mailchimp

Multisite Ultimate: integração com Mailchimp

Integre sua rede Multisite Ultimate ao seu Mailchimp. Você pode vincular determinados planos a listas do Mailchimp e adicionar automaticamente assinantes a essas listas no cadastro.

## Instalação

1. Envie 'ultimate-multisite-mailchimp' para o diretório '/wp-content/plugins/'
2. Ative o plugin pelo menu 'Plugins' no seu WordPress Network Admin

## Guia de uso

Sincronizar os e-mails dos assinantes com suas listas do MailChimp é muito fácil com nosso complemento Ultimate Multisite: MailChimp Integration. Este tutorial tem como objetivo ajudar você a configurar tudo!

## Como obter uma chave de API

Antes de começarmos a ver as diferentes configurações disponíveis, precisamos de uma chave de API do MailChimp. Isso permitirá que o Ultimate Multisite se comunique com a API do MailChimp para recuperar listas, grupos e adicionar endereços de e-mail a essas listas e grupos quando os usuários se cadastrarem.

Siga este tutorial para obter sua própria chave de API do MailChimp: [Sobre chaves de API | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

_Adicione sua chave de API do MailChimp em_ _**Ultimate Multisite Settings - > Add-on Settings -> MailChimp**_

## Modo de integração

O complemento de integração do MailChimp oferece dois “modos de integração”, o que significa que há duas maneiras diferentes de usar o complemento para sincronizar endereços de e-mail.

<!-- Captura de tela indisponível: seleção do modo de integração do MailChimp com opções de múltiplas listas e lista única -->

_Dois modos de integração para maior flexibilidade!_

### Modo 1: Múltiplas listas

O modo padrão usa **Múltiplas listas**. Este modo permitirá que você selecione várias listas do MailChimp para cada um dos planos. Esta é a melhor opção se você segmenta seus usuários em listas diferentes.

Por exemplo, vamos supor que você tenha 3 listas diferentes na sua Account do MailChimp: **Newsletter** , **Plano A** , e **Plano B**. Você quer que todos os usuários, independentemente do plano, também sejam incluídos na lista de Newsletter, para que não percam suas atualizações.

Para conseguir isso, você pode usar o modo Múltiplas listas para adicionar as listas **Newsletter** e **Plano A** às configurações do MailChimp do Plano A, e **Newsletter** e **Plano B** às configurações do MailChimp do Plano B.

<!-- Captura de tela indisponível: seleção de listas padrão do MailChimp para novas contas e contas canceladas -->

_Selecione as listas padrão para novas contas e para contas canceladas_

### Modo 2: Lista única + múltiplos grupos

O MailChimp permite segmentar sua lista em grupos menores de usuários chamados **Grupos**. Isso permite que você tenha mais controle, pois pode enviar uma campanha para um subconjunto específico dos seus assinantes sem precisar criar várias cópias da mesma campanha no MailChimp (o que dificulta analisar as métricas dessa campanha depois).

Por exemplo, você pode ter uma única lista chamada **Assinantes da minha rede,** e dentro dela ter grupos para **Plano A** e **Plano B**.

<!-- Captura de tela indisponível: seleção de lista única e grupos do MailChimp para listas padrão -->

_Selecione as listas e os grupos padrão, se estiver usando o modo de integração “lista única + múltiplos grupos”_

## Listas e grupos padrão

Depois de selecionar qual modo de integração faz mais sentido para seu caso de uso específico, é hora de selecionar as listas padrão (e grupos, dependendo do modo). A lista padrão será usada quando um determinado plano não tiver uma lista específica selecionada.

Você também pode selecionar a lista padrão para contas canceladas. Sempre que um usuário encerrar sua assinatura e remover sua conta da sua rede, o endereço de e-mail dele será movido para a lista definida para contas canceladas.

## Como ativar a assinatura dupla

Na mesma página de configurações, você poderá ativar a **Assinatura dupla**. Esta opção adicionará o usuário tanto à lista padrão selecionada QUANTO à lista selecionada para o plano desse usuário. Se esta opção não estiver ativada, o usuário será adicionado apenas à lista selecionada para o plano do usuário. Se nenhuma estiver definida para esse plano específico, então o usuário será adicionado à lista padrão selecionada.

<!-- Captura de tela indisponível: configuração de alternância de assinatura dupla do MailChimp -->

_Você pode fazer com que seus usuários sejam assinados tanto na lista padrão quanto na lista do plano no cadastro_

## Como selecionar listas e grupos para cada um dos planos

Vá para seus **Planos** e selecione o plano que deseja editar. Observe que, na meta-box **Opções avançadas**, uma nova aba chamada MailChimp estará disponível.

Nessa aba, você poderá selecionar várias listas ou uma única lista e múltiplos grupos (dependendo do modo de integração) para esse plano.

<!-- Captura de tela indisponível: aba do MailChimp na página de edição do plano mostrando a seleção de listas e grupos -->

_Selecione as listas e grupos que você quer usar para cada um dos seus planos_

Quando um novo usuário se cadastrar nesse plano, o endereço de e-mail dele será adicionado às listas e grupos selecionados!
