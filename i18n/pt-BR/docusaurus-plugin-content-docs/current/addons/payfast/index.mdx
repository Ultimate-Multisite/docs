# PayFast

Multisite Ultimate: Payfast Gateway

Adiciona Payfast - um gateway de pagamentos online rápido e seguro na África do Sul - como uma opção de pagamento do Multisite Ultimate.

## Instalação

1. Faça upload de 'ultimate-multisite-payfast' para o diretório '/wp-content/plugins/'
2. Ative o plugin através do menu 'Plugins' no WordPress
3. Siga o assistente passo a passo para configurar o plugin

## Guia de Uso

PayFast é um processador de pagamentos online líder para empresas na África do Sul. Ele se dedica a ajudar os comerciantes online a crescer, tornando as transações fluídas e seguras. Conforme o pedido de um usuário do Ultimate Multisite, decidimos criar um addon que possibilite a integração com o PayFast. Este artigo mostra como configurar o PayFast como seu gateway de pagamento para receber pagamentos em sua rede Ultimate Multisite.

O primeiro passo é instalar o addon.

# Como instalar o addon Payfast:

Vá para a página de addons e selecione o addon Ultimate Multisite: Payfast Gateway.

![Add-ons page](/img/admin/addons-page.png)

A janela do addon aparecerá. Clique em Instalar Agora.

![Add-ons page with install option](/img/admin/addons-page.png)

Após a instalação ser concluída, você será redirecionado para a página de plugins. Aqui, basta clicar em Ativar na Rede e o addon _Payfast_ será ativado em sua rede.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# Configurando o PayFast

Após instalar o addon, você pode agora acessar as configurações do Ultimate Multisite e começar a configurar o PayFast.

Na página **Ultimate Multisite settings**, você pode encontrar a aba 'Payments'. Nesta aba, você deve ver '**Payfast**' junto com um botão de alternância para habilitar ou desabilitar este gateway de pagamento.

![Payment gateway settings](/img/config/settings-payment-gateways.png)Go ahead and click the toggle button to enable it. It will then expand the box to show different fields that you need to populate such as **Mechant ID, Merchant Key and PassPhrase.** You can get this information from your **Payfast** account

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

O próximo passo é fazer login na sua conta PayFast para obter as informações necessárias.

Ao fazer login, você pode ver imediatamente as credenciais necessárias no painel da sua conta **Payfast**.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->You can simply copy and paste the credential and populate the details on your Ultimate Multisite settings. Then click the **"Save settings** " button to save the changes.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Pronto, sua integração de pagamento PayFast está concluída!
