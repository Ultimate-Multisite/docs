# Plugin & Theme Manager

Multisite Ultimate: Plugin and Theme Manager

Edite ou oculte as informações meta (título, autor, miniatura e descrição) dos plugins e temas instalados. Crie categorias para plugins e temas para permitir que seus usuários filtrem e adicione uma página de Plugins personalizada e bonita ao painel dos seus usuários.

Para usar o plugin, basta navegar até suas páginas de Plugins ou Temas no administrador da sua rede e clicar em "Editar informações do Plugin/Theme".

## Installation

1. Faça upload de 'ultimate-multisite-plugin-and-theme-manager' para o diretório '/wp-content/plugins/'.
2. Ative o plugin através do menu 'Plugins' no Administrador da Rede WordPress.

## Usage Guide

Ultimate Multisite: Plugin and Theme Manager é um poderoso add-on **premium** Ultimate Multisite que lhe dá controle total sobre a forma como os usuários da sua rede veem e interagem com as opções de plugin e tema na sua rede.

Este add-on tem três propósitos:

  1. Ele permite que você altere completamente os metadados de plugins e temas. Isso inclui nome, descrição, nome do autor e até miniaturas!

  2. Ele permite categorizar plugins e temas, permitindo que seus usuários os organizem facilmente.

  3. Ele oferece a opção de substituir a antiga página de plugins entediante por uma nova página semelhante a temas, com ordenação por categorias.

Após baixar e instalar, uma nova aba - _Add-on Settings_ \- aparecerá na página de Configurações do Ultimate Multisite.
