# Fluent Forms

Este complemento permite adicionar limites ao Fluent Forms no Multisite Ultimate em produtos, assinaturas ou sites. O Fluent Forms usa tabelas personalizadas em vez de tipos de post personalizados, então as opções de limitação normais não funcionam com o Fluent Forms.

## Features

- Adicionar limites para Fluent Forms ou Conversational Fluent Forms
- Os clientes recebem uma mensagem amigável solicitando que atualizem seu plano quando atingirem o limite
