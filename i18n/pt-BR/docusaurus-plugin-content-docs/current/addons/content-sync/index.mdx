# Content Sync

Multisite Ultimate: Content Sync

Este addon para Multisite Ultimate permite que administradores de rede sincronizem conteúdo de qualquer site para outros sites na rede. Ele suporta todos os tipos de postagem, incluindo posts, páginas, tipos de postagem personalizados e templates Elementor. O addon inclui uma interface de assistente de vários passos para selecionar conteúdo e sites de destino, com processamento em lote e feedback de progresso em tempo real.

## Funcionalidades

* Seleção de Conteúdo: Grade de cartões visuais com miniaturas e filtragem por tipo de postagem
* Todos os Tipos de Postagem: Sincronizar posts, páginas, produtos, templates Elementor e qualquer tipo de postagem personalizado
* Seleção de Site: Selecione quais sites sincronizar o conteúdo, com exclusão do site de origem e pré-seleção de sites filhos
* Processamento em Lote: Sincroniza com 10 sites por vez com atualizações de progresso
* Sincronização de Mídia: Copia e remapeia automaticamente arquivos de mídia (imagens, vídeos) do conteúdo do post para os sites de destino
* Sincronização de Taxonomia: Preserva categorias, tags e taxonomias personalizadas
* Suporte ao Elementor: Suporte completo para templates Elementor com regeneração de CSS e remapeamento de ID de template
* Resolução de Conflitos: Opção de substituir conteúdo existente ou ignorá-lo
* Gerenciamento de Cache: Limpa automaticamente o cache do Elementor nos sites de destino após a sincronização

## Requisitos

* WordPress 5.3 ou superior
* PHP 7.4 ou superior
* Plugin Multisite Ultimate (ativo)
* Plugin Elementor (opcional, necessário para sincronização de templates Elementor)

## Instalação

1. Faça upload de 'ultimate-multisite-content-sync' para o diretório '/wp-content/plugins/'
2. Ative o plugin através do menu 'Plugins' no Administrador de Rede do WordPress
3. Configure as configurações em Ultimate Multisite → Configurações → Content Sync
4. Navegue até Ultimate Multisite → Content Sync para iniciar a sincronização
