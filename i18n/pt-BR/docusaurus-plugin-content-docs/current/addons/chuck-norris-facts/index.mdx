# Chuck Norris Facts

Adicione um pouco de humor às suas tarefas diárias de administração do WordPress com o addon **Ultimate Multisite - Chuck Norris Facts**. Quando ativado, você verá um fato aleatório sobre Chuck Norris exibido no canto superior direito de cada página administrativa.

Perfeito para trazer sorrisos à sua equipe, aliviar o clima durante longas sessões de administração ou simplesmente celebrar a incrível lenda que é Chuck Norris.

## Why Chuck Norris Facts?

* **Boost Team Morale**: Adicione um elemento divertido à sua administração de rede
* **Break the Ice**: Ótimo iniciador de conversas para novos membros da equipe
* **Productivity Boost**: Uma risada rápida pode melhorar o foco e a produtividade
* **Timeless Humor**: Fatos sobre Chuck Norris nunca envelhecem

Uma coleção abrangente de piadas e fatos clássicos sobre Chuck Norris que celebram a própria lenda.

## Instalação

1. Certifique-se de que o plugin Ultimate Multisite está instalado e ativado na sua rede WordPress
2. Faça upload dos arquivos do addon para o diretório `/wp-content/plugins/`
3. Ative o plugin em rede através do menu 'Plugins' no WordPress
4. Aproveite fatos aleatórios de Chuck Norris em cada página administrativa!

Nenhuma configuração necessária - basta ativar e sorrir!

## Perguntas Frequentes

## Isso requer o Ultimate Multisite?

Embora seja designado como um addon do Ultimate Multisite, o plugin funciona de forma independente e pode ser executado em qualquer instalação do WordPress.

## Posso adicionar meus próprios fatos?

Sim! Os fatos estão prontos para tradução. Você pode criar seu próprio arquivo `.po` com fatos personalizados e colocá-lo no diretório `/lang`. Ou você pode filtrar o array de fatos programaticamente.

## Isso atrasa minha administração?

De jeito nenhum! O plugin é extremamente leve e não tem impacto no desempenho. Ele apenas exibe texto - sem chamadas externas de API ou processamento pesado.

## Posso desativá-lo em determinadas páginas?

Os fatos são ocultados automaticamente em páginas onde podem interferir (como o editor de blocos). Você também pode usar hooks do WordPress para personalizar onde os fatos aparecem.

## Os fatos são adequados para todos os públicos?

Sim! Todos os fatos são humor limpo e adequado para a família, celebrando o status lendário de Chuck Norris.

## Posso traduzir os fatos para outro idioma?

Absolutamente! Todos os fatos usam funções de tradução do WordPress. Crie um arquivo de tradução para o seu idioma e coloque-o no diretório `/lang`. Domínio de texto: `ultimate-multisite-chuck-norris-facts`

## Funciona com multisite?

Sim! Foi projetado especificamente para redes multisite e pode ser ativado em rede.
