# Superdav AI Agent

Superdav AI Agent é um assistente de IA avançado para WordPress que estende o Gratis AI Agent com habilidades cientes do theme, recursos aprimorados de busca na internet, funcionalidades de construtor de site e integrações opcionais de automação. A versão 1.18.0 adiciona ferramentas de leitura do Google Calendar, mapeamento de contatos e participantes, etapas de aprovação humana, deduplicação de lembretes, notificações por SMS via TextBee, provisionamento gerenciado do serviço Superdav e um pacote complementar Advanced distribuído separadamente para fluxos de trabalho de desenvolvedores confiáveis.

## Principais recursos

- **Habilidades cientes do theme** — adapta automaticamente orientações e recursos com base no seu theme WordPress ativo e nos plugins instalados
- **Vários provedores de busca na internet** — escolha entre Brave Search e Tavily para resultados de busca mais ricos e recursos avançados de pesquisa
- **Formulários de contato do construtor de site** — crie formulários de contato totalmente funcionais diretamente pela interface de chat usando seu plugin de formulário instalado
- **Construtor de site aprimorado** — planos de criação de site em várias fases melhorados, com melhor descoberta de plugins e recuperação de erros
- **Habilidades de theme integradas** — orientação especializada para Block Themes, Classic Themes, Kadence Blocks e Kadence Theme
- **Integração aprimorada com WooCommerce** — refatorada para usar APIs nativas do WooCommerce para melhor confiabilidade e compatibilidade
- **Detecção automática de provedor** — detecta e atualiza automaticamente plugins e themes disponíveis quando são ativados ou desativados
- **Geração de logo personalizado** — gere e incorpore logos SVG personalizados com sanitização automática segura para namespaces
- **Validação de contraste de paleta** — verifique pares de cores quanto à conformidade de acessibilidade WCAG antes de aplicar ao seu theme
- **Descoberta aprimorada por fotos** — envie fotos durante a entrevista de descoberta do Theme Builder para influenciar a direção do design
- **Cardápios de hospitalidade** — gere páginas estruturadas de cardápios de comidas e bebidas para restaurantes, cafés e negócios de hospitalidade
- **Pré-visualizações em desktop e mobile** — pré-visualize seu design em dispositivos desktop e mobile durante a seleção da direção de design
- **Raspagem de site de Tier 1** — sd-ai-agent/site-scrape agora está disponível por padrão para analisar sites de concorrentes e de inspiração
- **Ferramentas de leitura do Google Calendar** — leia calendários e eventos configurados para que automações possam raciocinar sobre o contexto da agenda futura
- **Mapeamento de contatos e participantes** — associe participantes do calendário a usuários e contatos conhecidos do site antes de gerar trabalho de acompanhamento
- **Etapas de aprovação humana** — pause automações sensíveis para revisão antes de enviar mensagens, retomar o trabalho ou fazer alterações
- **Registros de lembretes e deduplicação** — acompanhe lembretes que já foram enviados para que execuções repetidas de automação não notifiquem o mesmo destinatário duas vezes
- **Notificações por SMS via TextBee** — envie notificações configuradas por mensagem de texto através do TextBee quando um provedor de SMS estiver habilitado
- **Provisionamento gerenciado do serviço Superdav** — conecte sites compatíveis a endpoints Superdav hospedados com configuração automática em vez de configuração manual de endpoint
- **Pacote complementar Advanced** — ferramentas para desenvolvedores confiáveis distribuídas separadamente para fluxos de trabalho de sistema de arquivos, banco de dados, WP-CLI, despachante REST, construtor de plugin, snapshot de git, gerenciamento de usuários e benchmarks

## Requisitos

- WordPress 6.0 ou superior
- PHP 8.1 ou superior
- WordPress Multisite (ativado na rede) ou instalação de site único
- Um plugin conector de provedor de IA ativo (Anthropic, OpenAI ou compatível)
- Gratis AI Agent v1.9.0 ou superior (Superdav AI Agent estende Gratis AI Agent)

## Instalação

1. Envie os arquivos do plugin para `/wp-content/plugins/superdav-ai-agent`
2. Ative o plugin pela tela **Plugins** no WordPress
3. Certifique-se de que o Gratis AI Agent também esteja ativado
4. Navegue até **Gratis AI Agent → Settings** e configure seu provedor de busca na internet, conexão gerenciada do serviço Superdav, Google Calendar ou credenciais do TextBee conforme necessário

O pacote do WordPress.org contém os principais recursos do Superdav AI Agent. O pacote complementar Advanced é criado e distribuído separadamente para ambientes em que ferramentas de desenvolvedores confiáveis são explicitamente instaladas e autorizadas.

## Início rápido

Depois de ativado, o Superdav AI Agent aprimora automaticamente sua instalação do Gratis AI Agent com habilidades cientes do theme e recursos adicionais. Abra o painel **Gratis AI Agent** na barra lateral de administração do WordPress e comece a usar os novos recursos.

## Habilidades cientes do theme

Superdav AI Agent detecta automaticamente seu theme ativo e os plugins instalados, carregando a habilidade ciente do theme apropriada no contexto do assistente. As habilidades disponíveis incluem:

| Habilidade | Aplica-se a |
|---|---|
| **Block Themes** | Themes que usam a arquitetura baseada em blocos do WordPress (Full Site Editing) |
| **Classic Themes** | Themes WordPress tradicionais que usam templates PHP e o editor clássico |
| **Kadence Blocks** | Sites com o plugin Kadence Blocks ativo |
| **Kadence Theme** | Sites com o theme Kadence como theme ativo |

Consulte [Habilidades cientes do theme](../../user-guide/configuration/theme-aware-skills) para orientações detalhadas sobre cada habilidade.

## Provedores de busca na internet

Superdav AI Agent v1.10.0 adiciona **Tavily** como alternativa ao Brave Search para consultas de busca na internet. Ambos os provedores oferecem resultados de busca em tempo real e podem ser configurados em **Gratis AI Agent → Settings → Advanced**.

| Provedor | Recursos | Configuração |
|---|---|---|
| **Brave Search** | Resultados de busca rápidos e com foco em privacidade | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Resultados de busca mais ricos, recursos avançados de pesquisa | [Tavily API](https://tavily.com/) |

Consulte [Busca na Internet](../../user-guide/configuration/internet-search) para obter instruções de configuração.

## Fluxos de Trabalho de Calendário, Aprovação e Notificação

Superdav AI Agent v1.18.0 expande os fluxos de trabalho de automação para além de tarefas de conteúdo e criação de sites:

- **Ferramentas do Google Calendar** permitem que o agente leia calendários configurados e detalhes de eventos quando as credenciais de calendário estão disponíveis.
- **Auxiliares de contatos e participantes** mapeiam participantes de eventos para usuários do WordPress ou contatos conhecidos antes de preparar lembretes ou tarefas de acompanhamento.
- **Portões de aprovação humana** permitem que uma automação seja pausada até que um usuário autorizado revise a ação proposta.
- **Registros de lembretes** evitam lembretes de calendário ou notificação duplicados quando uma automação é tentada novamente ou agendada repetidamente.
- **Suporte a SMS do TextBee** habilita notificações por mensagem de texto quando o provedor TextBee está configurado.

Use essas ferramentas para assistência sensível à agenda, acompanhamento de participantes e fluxos de trabalho de lembretes que ainda exigem revisão humana antes que mensagens sensíveis sejam enviadas.

## Formulários de Contato do Construtor de Sites

O agente Construtor de Sites agora pode criar formulários de contato diretamente pela interface de chat. Basta pedir ao Construtor de Sites para criar um formulário de contato, e ele detectará automaticamente o plugin de formulários instalado (Contact Form 7, WPForms, Fluent Forms ou Gravity Forms) e gerará um formulário pronto para uso.

Consulte [Formulários de Contato do Construtor de Sites](../../user-guide/administration/site-builder-contact-forms) para orientações detalhadas.

## Configuração

Todas as configurações ficam em **Gratis AI Agent → Settings** no administrador do WordPress. Superdav AI Agent adiciona as seguintes opções de configuração:

| Configuração | Descrição |
|---|---|
| **Internet Search Provider** | Escolha entre Brave Search e Tavily (opcional) |
| **Tavily API Key** | Chave de API para busca na internet do Tavily (se Tavily estiver selecionado) |
| **Managed Superdav Service** | Conecte sites compatíveis ao serviço Superdav hospedado com provisionamento automático de pontos de extremidade |
| **Google Calendar Credentials** | Configure o acesso ao calendário para que o agente possa ler calendários e eventos |
| **TextBee SMS Provider** | Configure credenciais do TextBee para fluxos de trabalho aprovados de notificação por SMS |

## Pacote Complementar Advanced

Superdav AI Agent v1.18.0 adiciona um pacote complementar Advanced distribuído separadamente para fluxos de trabalho de desenvolvedores confiáveis. Esses recursos não são incluídos no pacote principal do WordPress.org. Instale o pacote complementar apenas em sites onde os administradores confiam explicitamente que o agente acesse ferramentas de desenvolvedor.

As ferramentas Advanced incluem operações no sistema de arquivos, acesso ao banco de dados, WP-CLI, fluxos de trabalho do despachante REST, utilitários de criador de plugins, snapshots do git, auxiliares de gerenciamento de usuários e ferramentas de benchmark. Mantenha essas ferramentas limitadas a administradores confiáveis e revise a configuração do portão de aprovação antes de habilitá-las em sites de produção.
