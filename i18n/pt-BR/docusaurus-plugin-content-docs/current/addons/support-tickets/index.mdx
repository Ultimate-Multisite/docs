# Support Tickets

**Support Tickets** é um addon poderoso para Ultimate Multisite que adiciona um sistema completo de tickets de suporte à sua rede WordPress. Perfeito para plataformas SaaS, provedores WaaS (Website as a Service) e redes multisite que precisam oferecer suporte ao cliente tanto na rede quanto nos sites individuais.

## Why Choose Support Tickets?

* **Multi-Level Support**: Administradores de rede lidam com tickets em toda a rede enquanto os proprietários de sites gerenciam seu próprio suporte ao cliente
* **Professional Workflow**: Gerenciamento completo do ciclo de vida do ticket com prioridades, status e atribuições
* **Customer Experience**: Interface bonita e responsiva que facilita para os clientes obter ajuda
* **Secure & Reliable**: Construído com as melhores práticas de segurança do WordPress e desempenho otimizado de banco de dados
* **Seamless Integration**: Funciona perfeitamente com a gestão de clientes e sites do Ultimate Multisite

## Key Features

**Complete Ticket Management**

* Sistema único de numeração de tickets para referência fácil
* Níveis de prioridade configuráveis (Baixa, Normal, Alta, Crítica)
* Fluxo de status abrangente (Aberto, Em andamento, Aguardando, Resolvido, Fechado)
* Classificação de tickets (Geral, Técnico, Faturamento, Solicitação de Funcionalidade, Relatório de Erro)
* Atribuir tickets a agentes de suporte com autoatribuição opcional

**Threaded Conversations**

* Conversação em thread completa para respostas de tickets
* Estilo distinto para respostas de equipe e cliente
* Notas privadas de equipe que os clientes não podem ver
* Interface em tempo real com AJAX
* Editor de texto moderno com opções de formatação

**File Attachments**

* Segurança avançada com detecção de arquivos maliciosos

**Smart Notifications**

* Alertas de e‑mail automáticos para novos tickets, respostas e alterações de status

* Variáveis dinâmicas de modelo para mensagens personalizadas

**Powerful Admin Interface**

* Lista avançada de tickets com ordenação e filtragem
* Edição rápida de status, prioridade e atribuições
* Ações em massa para gerenciamento eficiente
* Painel de estatísticas com métricas de desempenho
* Painel de configurações abrangente

**Beautiful Customer Portal**

* Interface limpa e responsiva em todos os dispositivos
* Design mobile‑first
* Busca avançada e filtragem
* Editor moderno com barra de ferramentas de formatação
* Formulários de envio de tickets fáceis de usar

**Flexible Integration**

## Installation

1. Certifique-se de que o plugin Ultimate Multisite esteja instalado e ativado na sua rede WordPress
2. Faça upload dos arquivos do addon para o seu diretório `/wp-content/plugins/`
3. Ative o plugin na rede através do menu 'Plugins' no WordPress
4. Navegue até Network Admin > Support Tickets para configurar suas opções

## Frequently Asked Questions

## Does this require Ultimate Multisite?

Sim, este addon requer que o Ultimate Multisite esteja instalado e ativo. Ele se integra profundamente com os recursos de gerenciamento de clientes e sites do Ultimate Multisite.

## Can site owners handle their own customer support?

Absolutamente! Os proprietários de sites podem usar o sistema de tickets para oferecer suporte aos seus próprios clientes, enquanto os administradores de rede podem lidar com tickets de suporte em toda a rede.

## How do customers submit tickets?

Você pode adicionar formulários de envio de tickets em qualquer lugar do seu site usando shortcodes ou elementos de UI do Ultimate Multisite. Os clientes simplesmente preenchem o formulário e enviam sua solicitação.

## Can I customize the ticket statuses and priorities?

Sim! Você pode personalizar totalmente os níveis de prioridade, status e tipos de ticket nas configurações do administrador da rede para combinar com seu fluxo de trabalho de suporte.

## Does it support file attachments?

Sim, tanto os clientes quanto a equipe podem anexar arquivos a tickets e respostas. Você tem controle total sobre os tipos de arquivo permitidos e os limites de tamanho para segurança.

## How are notifications handled?

O sistema envia automaticamente notificações por e‑mail para novos tickets, respostas e alterações de status. Você pode personalizar todos os modelos de e‑mail nas configurações.

## Can I assign tickets to specific support agents?

Sim, você pode atribuir manualmente tickets a agentes ou habilitar a autoatribuição para distribuir tickets automaticamente.

## Are there private notes for staff?

Sim, os membros da equipe podem adicionar respostas privadas que são visíveis apenas para outros membros da equipe, não para os clientes.

## Is it mobile-friendly?

Absolutamente! A interface inteira foi construída com abordagem mobile‑first e funciona maravilhosamente em todos os dispositivos.

## Can I display ticket lists and forms using shortcodes?

Sim! Shortcodes disponíveis:

* `[wu_support_tickets]` - Exibir lista de tickets de clientes
* `[wu_support_ticket_form]` - Exibir formulário de envio de novo ticket
* `[wu_support_ticket_view ticket_id="123"]` - Exibir ticket específico

## What file types can be uploaded?

Você pode configurar os tipos de arquivo permitidos nas configurações. O sistema inclui varredura de segurança avançada para proteger contra arquivos maliciosos.

## Can I export ticket data?

Sim, o addon inclui recursos de exportação de dados e estatísticas de tickets.
