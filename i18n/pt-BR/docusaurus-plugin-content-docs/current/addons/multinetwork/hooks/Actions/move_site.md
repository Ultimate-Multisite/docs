# Action: move_site

Dispara após um site ter sido movido para uma nova rede.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $site_id | `int` | ID do site que foi movido. |
| $old_network_id | `int` | ID da rede original do site. |
| $new_network_id | `int` | ID da rede para a qual o site foi movido. |
