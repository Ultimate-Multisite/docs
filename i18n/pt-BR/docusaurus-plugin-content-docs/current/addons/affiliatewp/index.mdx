# AffiliateWP

Multisite Ultimate: AffiliateWP Integração

Use o poderoso AffiliateWP para aumentar a base de clientes da sua Ultimo Network!

## Installation

1. Faça upload de 'ultimate-multisite-git' para o diretório '/wp-content/plugins/'
2. Ative o plugin através do menu 'Plugins' no WordPress
3. Siga o assistente passo a passo para configurar o plugin

## Usage Guide

O [AffiliateWP](https://affiliatewp.com/) tornou-se a solução padrão ao criar um programa de afiliados para o seu negócio online. Ultimate Multisite: AffiliateWP é a ponte que permitirá que você crie a mesma coisa para sua rede premium.

## Getting Started

Primeiro, você **precisa ter** [**AffiliateWP**](https://affiliatewp.com/) **ativo**! Você não poderá usar nosso add-on se não tiver **o plugin que se integra a ele**. Caso não pretenda permitir que seus clientes usem no seus sites também, o AffiliateWP deve estar ativo apenas no seu site principal.

Além disso, você precisará tanto do **Ultimate Multisite** quanto do **Ultimate Multisite: AffiliateWP** **ativados** **em toda a rede**.

## Adding Ultimate Multisite as an AffiliateWP Integration

No seu site principal, vá para Configurações de Afiliados, e na aba Integrações, marque Ultimate Multisite. Isso informará ao [AffiliateWP](https://affiliatewp.com/) para começar a rastrear visitas e conversões a partir do fluxo de cadastro do Ultimate Multisite.

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_Marque a caixa Ultimate Multisite na parte inferior e salve_

Isso é praticamente tudo que é necessário. Visitas provenientes de links de referência serão rastreadas e, quando esse usuário fizer seu primeiro pagamento, a conversão será criada e uma comissão será reservada para o afiliado responsável pela conversão.

Somente o primeiro pagamento recebido do usuário recém‑criado será rastreado. Para habilitar pagamentos recorrentes de referência, você precisará do [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## Enabling Recurring Referrals

Após instalar o [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/), vá para a aba Configurações de Afiliados > Referências Recorrentes, e habilite o uso de referências recorrentes.

<!-- Screenshot unavailable: AffiliateWP Recurring Referrals settings with Enable Recurring Referrals toggle -->

_Tão simples quanto parece: Habilite Referências Recorrentes e você está pronto!_

Novamente, isso é tudo que é necessário!

Agora, toda vez que um novo pagamento for recebido em uma conta referenciada, o afiliado responsável receberá sua comissão automaticamente!
