# AffiliateWP {#affiliatewp}

Integração do Multisite Ultimate com AffiliateWP

Use o poderoso AffiliateWP para aumentar a base de clientes da sua rede Ultimo!

## Instalação {#installation}

1. Envie 'ultimate-multisite-git ' para o diretório '/wp-content/plugins/'
2. Ative o plugin pelo menu 'Plugins' no WordPress
3. Siga o Assistente passo a passo para configurar o plugin

## Guia de uso {#usage-guide}

[AffiliateWP](https://affiliatewp.com/) se tornou a solução preferida ao criar um programa de afiliados para o seu negócio online. Ultimate Multisite: AffiliateWP é a ponte que permitirá que você crie a mesma coisa para a sua rede premium.

## Primeiros passos {#getting-started}

Antes de tudo, você **precisa ter** [**AffiliateWP**](https://affiliatewp.com/) **ativo**! Você não conseguirá usar nosso add-on se não tiver **o plugin que se integra a ele**. Caso você não planeje permitir que seus clientes também o usem nos sites deles, AffiliateWP deve estar ativo apenas no seu site principal.

Além disso, você precisará ter tanto **Ultimate Multisite** quanto **Ultimate Multisite: AffiliateWP** **ativados** **em toda a rede**.

## Adicionando Ultimate Multisite como uma integração do AffiliateWP {#adding-ultimate-multisite-as-an-affiliatewp-integration}

No seu site principal, acesse Configurações de Afiliados e, na aba Integrações, marque Ultimate Multisite. Isso dirá ao [AffiliateWP](https://affiliatewp.com/) para começar a rastrear visitas e conversões do fluxo de cadastro do Ultimate Multisite.

<!-- Captura de tela indisponível: aba de configurações de Integrações do AffiliateWP com caixa de seleção do Ultimate Multisite -->

_Marque a caixa Ultimate Multisite na parte inferior e salve_

Isso é praticamente tudo o que é necessário. Visitas vindas de links de indicação serão rastreadas e, quando esse usuário fizer seu primeiro pagamento posteriormente, a conversão será criada e uma comissão será reservada para o afiliado responsável pela conversão.

No entanto, apenas o primeiro pagamento recebido do usuário recém-criado será rastreado. Para habilitar pagamentos recorrentes por indicação, você precisará do [add-on AffiliateWP Recurring Referrals](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## Habilitando indicações recorrentes {#enabling-recurring-referrals}

Depois de instalar o [add-on AffiliateWP Recurring Referrals](https://affiliatewp.com/add-ons/pro/recurring-referrals/), acesse a aba Indicações recorrentes nas Configurações de Afiliados e habilite o uso de indicações recorrentes.

<!-- Captura de tela indisponível: configurações de Indicações recorrentes do AffiliateWP com alternância Habilitar indicações recorrentes -->

_Tão simples quanto possível: habilite Indicações recorrentes e está tudo pronto!_

Novamente, isso é tudo o que é necessário!

Agora, toda vez que um novo pagamento for recebido em uma Account indicada, o afiliado responsável receberá automaticamente sua própria parte!
