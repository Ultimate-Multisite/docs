# Metered Plans

O addon Metered Plans do Ultimate Multisite transforma a forma como você cobra seus clientes, introduzindo recursos de faturamento por uso flexíveis. Em vez de limites rígidos que bloqueiam os usuários quando excedidos, habilite cobranças por excesso que faturam automaticamente os clientes por uso adicional.

## Funcionalidades Principais

* **Faturamento por Uso**: Cobrar os clientes com base no uso real em vez de faixas fixas
* **Gestão de Excesso**: Faturar automaticamente por uso que excede os limites do plano
* **Monitoramento de Uso**: Monitorar o consumo de recursos do cliente em tempo real
* **Precificação Flexível**: Definir tarifas personalizadas para diferentes recursos (armazenamento, banda larga, sites, etc.)
* **Faturamento Automático**: Gerar faturas automaticamente quando os limites de uso são excedidos
* **Relatórios de Uso**: Análises detalhadas de uso e relatórios para clientes e administradores

## Ideal Para

* **Provedores de Hospedagem**: Oferecer planos escaláveis que crescem com as necessidades do cliente
* **Plataformas SaaS**: Cobrar com base no consumo real de recursos
* **Redes de Agências**: Modelos de precificação flexíveis para projetos de clientes
* **Hospedagem por Revenda**: Permitir que parceiros ofereçam precificação baseada em uso

## Como Funciona

1. Configure recursos medidos (espaço em disco, largura de banda, número de sites, etc.)
2. Definir limites base do plano e tarifas de excesso
3. O sistema rastreia automaticamente o uso em toda a sua rede
4. Quando os clientes excedem os limites, as cobranças por excesso são calculadas
5. Faturas são geradas automaticamente com base no ciclo de faturamento
6. Os clientes podem visualizar seu uso e cobranças futuras em tempo real

## Recursos Suportados

* Armazenamento em disco
* Largura de banda/Transferência de dados
* Número de sites
* Número de usuários
* Tipos de recursos personalizados

## Requisitos

* Multisite Ultimate 2.0.0 ou superior
* Instalação do WordPress Multisite
* PHP 7.4 ou superior

## Instalação

1. Faça upload dos arquivos do plugin para `/wp-content/plugins/ultimate-multisite-metered-plans`
2. Ative o plugin através da tela 'Plugins' no WordPress
3. Navegue até Multisite Ultimate → Configurações → Metered Plans
4. Configure seus recursos medidos e precificação
5. Habilite o faturamento por uso para produtos no editor de produtos

## Perguntas Frequentes

## O que acontece quando um cliente excede seu limite?

Em vez de bloquear o acesso, o sistema rastreia o excesso e adiciona à próxima fatura com base nas tarifas que você configurou.

## Posso definir tarifas de excesso diferentes para produtos diferentes?

Sim, cada produto pode ter recursos medidos e precificação de excesso únicos.

## Como os clientes são notificados sobre o uso?

Os clientes podem visualizar seu uso atual em seu painel e receber notificações opcionais ao se aproximarem ou excederem os limites.

## Ainda posso usar limites rígidos?

Sim, o faturamento por uso é opcional por produto. Você pode combinar produtos com limites rígidos tradicionais com produtos por uso.
