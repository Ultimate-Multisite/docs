# Registro de alterações dos Agentes de Suporte

Versão: 1.0.13 - Lançado em 2025-09-28

* Renomear prefixo para ultimate-multisite; atualizar domínio de texto; atualizar versão.

Versão 1.0.12 - Lançado em 2025-09-17

* Corrigido: compatibilidade com Ultimate Multisite 2.4.4

Versão 1.0.11 - Lançado em 2023-11-21

* Corrigido: compatibilidade com Ultimate Multisite 2.3.0

Versão 1.0.10 - Lançado em 2023-08-30

* Corrigido: remoção em massa de Support Agents

Versão 1.0.9 - Lançado em 2023-08-09

* Adicionado: adicionada a capacidade de ler formulários de checkout.
* Corrigido: layout do formulário de permissão de Support Agents;
* Corrigido: mensagem de aviso no formulário de exclusão de support agent;
* Corrigido: ocultar widgets do painel de rede para usuários não autorizados.
* Corrigido: corrigidas as capacidades de edição e exclusão de Support Agents.
* Corrigido: corrigido o processo de criação de Support Agents.
* Corrigido: impedido adicionar um cliente como support agent.
* Corrigido: redirecionados usuários não autorizados para a página de edição de support agent no admin.
* Corrigido: melhorado o layout do formulário de permissão de Support Agents.
* Interno: implementado um novo processo de build.

Versão 1.0.8 - Lançado em 2022-01-21

* Interno: adicionado gerador de hooks e filtros;
* Interno: adicionados stubs do Ultimate Multisite para qualidade de vida do desenvolvedor;
* Adicionado: capacidades de Checkout Form;
* Corrigido: incompatibilidade de nome de capacidade entre formulários do Ultimate Multisite e opções deste add-on;

Versão 1.0.7 - Lançado em 2021-12-02

* Corrigido: exclusão de Support Agents não funcionando;
* Corrigido: exclusão em massa de Support Agents não funcionando;

Versão 1.0.6 - Lançado em 2021-11-22

* Corrigido: modal de adição não funciona após otimizações de desempenho no core;
* Corrigido: visualização de grid-item presente no core e não no add-on;

Versão 1.0.5 - Lançado em 2021-10-31

* Corrigido: pequenas incompatibilidades com PHP 8;
* Melhorado: wp_ultimo_skip_network_active_check agora usa wu_is_must_use();

Versão 1.0.4 - 2021-09-24

* Adicionado: filtro wp_ultimo_skip_network_active_check para configurações baseadas em mu-plugins;

Versão 1.0.3 - Lançado em 2021-08-29

* Adicionado: suporte à REST API - support agents podem ser criados via REST;

Versão 1.0.2 - Lançado em 2021-07-26

* Corrigido: atualização de permissões de agente não funciona;

Versão 1.0.1 - Lançado em 2021-05-28

* Adicionado: aviso quando o usuário já é super admin;
* Adicionado: permissão para permitir troca de usuário;
* Melhorado: campos de formulário melhores;

Versão 1.0.0 - Lançamento inicial
