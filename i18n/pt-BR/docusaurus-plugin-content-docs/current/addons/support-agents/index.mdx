# Support Agents

Gerenciar toda uma rede sozinho não é uma tarefa fácil. À medida que sua rede começa a crescer, você perceberá uma demanda crescente por suporte. Será uma boa ideia trazer pessoas que ajudarão a aliviar sua carga de trabalho. No entanto, conceder acesso/permissão total é arriscado e pode comprometer sua rede. É aí que entra este Support Agent add-on.

## Features

- Lista detalhada de permissões do WordPress & Ultimate Multisite
- Ativar e desativar acesso com facilidade
- Criar número ilimitado de contas de Agente

## Installation

1. Faça upload de 'ultimate-multisite-support-agents' para o diretório '/wp-content/plugins/'
2. Ative o plugin através do menu 'Plugins' no WordPress
3. Siga o assistente passo a passo para configurar o plugin
