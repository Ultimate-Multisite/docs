# Material WP

A interface de administração do WordPress é a mesma há muito tempo. Mas você já se perguntou como seria uma versão Material Design da interface de administração? 

Material WP é um tema de administração desenvolvido para oferecer esse visual e experiência, ajudando você a fornecer aos seus clientes um Dashboard mais personalizado.

## Recursos

- Suporte a CSS personalizado
- Configurações fáceis de estilo
- Reordenação de menu
- Importação/exportação fácil

## Documentação

**Personalizando seu painel de administração com Material WP**

## Instalação

1. Envie 'material-wp' para o diretório '/wp-content/plugins/'
2. Ative o plugin pelo menu 'Plugins' no WordPress

## Guia de uso

A interface de administração do WordPress é a mesma há muito tempo. Mas você já se perguntou como seria uma versão Material Design da interface de administração?

Material WP é um tema de administração desenvolvido para oferecer esse visual e sensação, ajudando você a fornecer aos seus clientes um Dashboard mais personalizado. Material WP também faz parte do núcleo de plugins premium do Ultimate Multisite.

Depois de **instalá-lo**, acesse sua página _Plugins_ e clique para ativá-lo na sua Rede.

_**OBSERVAÇÃO:**_ _Se você apenas ativá-lo, ele aparecerá para o seu cliente como uma opção que pode ser marcada para adicionar o tema somente a esse usuário._

Agora, você tem sua interface de administração completamente redesenhada! Você também tem um novo menu na coluna esquerda chamado Material WP
