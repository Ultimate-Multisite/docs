# Material WP

A interface administrativa do WordPress tem sido a mesma há muito tempo. Mas você já se perguntou como seria uma versão do Material Design da interface administrativa?

Material WP é um tema administrativo desenvolvido para lhe proporcionar essa aparência e sensação, ajudando você a oferecer aos seus clientes um Dashboard mais personalizado.

## Features

- Suporte a CSS personalizado
- Configurações de estilo fáceis
- Reordenação de menu
- Importação/exportação fáceis

## Documentation

**Personalizando seu Painel Administrativo com Material WP**

## Installation

1. Faça upload de 'material-wp' para o diretório '/wp-content/plugins/'
2. Ative o plugin através do menu 'Plugins' no WordPress

## Usage Guide

A interface administrativa do WordPress tem sido a mesma há muito tempo. Mas você já se perguntou como seria uma versão do Material Design da interface administrativa?

Material WP é um tema administrativo desenvolvido para lhe proporcionar essa aparência e sensação, ajudando você a oferecer aos seus clientes um Dashboard mais personalizado. Material WP também faz parte do núcleo de plugins premium do Ultimate Multisite.

Após **instalá-lo**, vá para a sua página _Plugins_ e clique para ativá-lo na sua Rede.
