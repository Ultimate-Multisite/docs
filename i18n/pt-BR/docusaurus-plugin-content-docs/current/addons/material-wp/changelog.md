Version 0.0.8

* GRANDE ATUALIZAÇÃO: O sistema de blocos usado no tema foi substituído de .wrap (que é recomendado pelo WordPress, mas os devs muitas vezes não usam — o que estava causando várias incompatibilidades com plugins) para #wpbody-content. Com essa mudança, o Material WP agora é virtualmente compatível com todos os plugins do WordPress. (Mas, se você notar algo estranho, basta nos enviar uma mensagem como sempre).
* Novo: alterar a altura da barra de administração, bem como seus subitens
