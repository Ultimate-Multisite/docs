# Exportador de Sites

## Conteúdo

Entendemos a necessidade de poder mover subsites dentro da sua rede para uma instalação de site único e vice-versa, caso seu cliente decida migrar seu site. O addon Exportador e Importador de Sites facilita essa tarefa. Esta ferramenta exporta e importa sites usando arquivos zip, que também podem servir como backup.

## Funcionalidades

* Exportar e Importar Sites  
* Interface de fácil uso  
* Criar backup do site  

## Documentação

**Como Exportar e Importar um Site**

## Instalação

1. Faça upload do **ultimate-multisite-site-exporter** para o diretório **/wp-content/plugins/**  
2. Ative o plugin através do menu **Plugins** no WordPress  
3. Siga o assistente passo a passo para configurar o plugin  

## Guia de Uso

Com o Ultimate Multisite, você pode exportar um site da sua rede ou importar um site para sua instalação multisite ou single-site.  

To start the process, go to our add-ons page and install the Ultimate Multisite: Site Exporter & Importer add-on.![Add-ons page](/img/admin/addons-page.png)

![Add-ons page with install option](/img/admin/addons-page.png)

Go to your Plugins page and click to **Network Activate** the add-on you have just installed.<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Site Exporter addon -->

A opção **Export & Import** pode ser encontrada no menu Ultimate Multisite, à esquerda.

<!-- Screenshot unavailable: Ultimate Multisite menu showing Export and Import option -->

## Como exportar um site

Para exportar um site da sua rede, vá para **Export & Import** no menu Ultimate Multisite e, na aba **Export Sites**, clique em **Generate new Site Export**.

<!-- Screenshot unavailable: Export Sites tab with Generate new Site Export button -->

Search for the site you want to export and select the items you want to include in the exported file. You can include themes, plugins, and uploads. If your site is too big, you can choose to run the export process in the background.<!-- Screenshot unavailable: Site export options with site search, include themes/plugins/uploads checkboxes -->

Click the **Export Site** button. Your exported site will be available on the **Existing Exports** list. Click the **Download ZIP** button to download the file.<!-- Screenshot unavailable: Existing Exports list with Download ZIP button -->

## Como importar um site

Para importar um site, vá para a aba **Import a Site** e faça upload do arquivo .zip do site que deseja importar. Você pode alterar a URL para uma nova ou manter a URL antiga deixando o campo em branco. Além disso, pode marcar a caixa para excluir o arquivo .zip da sua instalação após a importação. Em seguida, clique em **Begin Import**.

<!-- Screenshot unavailable: Import a Site tab with zip upload field and Begin Import button -->

O processo pode demorar um pouco. Você verá uma importação pendente na lista abaixo do importador.

<!-- Screenshot unavailable: Pending import in the list below the importer -->

Você pode interromper o processo clicando em **Cancel Import**.

<!-- Screenshot unavailable: Cancel Import button to stop the import process -->

Após o processo ser concluído, seu site já estará disponível na sua rede.

## Como importar um site em uma instalação única

Para importar um site para sua instalação única, primeiro instale e ative o importador no seu site. Você pode encontrar o link para baixá-lo na aba exporter.

<!-- Screenshot unavailable: Exporter tab with link to download single-site importer -->

Você pode encontrar o importador no menu **Tools**, à esquerda, no subitem **Export & Import**.

<!-- Screenshot unavailable: Single-site Tools menu showing Export and Import submenu -->

Faça upload do arquivo .zip do site que deseja importar. Você pode alterar a URL para uma nova ou manter a URL antiga deixando o campo em branco. Além disso, pode marcar a caixa para excluir o arquivo .zip da sua instalação após a importação. Em seguida, clique em **Begin Import**.

<!-- Screenshot unavailable: Single-site import form with zip upload and Begin Import button -->

O processo pode demorar um pouco. Você verá uma importação pendente na lista abaixo do importador.

<!-- Screenshot unavailable: Single-site pending import in the list -->

Você pode interromper o processo clicando em **Cancel Import**.

<!-- Screenshot unavailable: Cancel Import button to stop the import process -->

Após o processo terminar, seu site será substituído pelo site que você acabou de importar.
