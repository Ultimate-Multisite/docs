# Filtro: signup_get_available_languages

Filtra a lista de idiomas disponíveis para inscrições de sites no front-end.

Passar um array vazio para este gancho desativará a exibição da configuração no formulário de inscrição, e o idioma padrão será usado ao criar o site. Idiomas que ainda não estão instalados serão removidos.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $available_languages | `array` | Idiomas disponíveis. |

- 4.4.0
