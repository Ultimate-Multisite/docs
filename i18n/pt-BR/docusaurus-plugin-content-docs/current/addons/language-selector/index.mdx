# Seletor de idioma

Multisite Ultimate: Seletor de idioma no cadastro

Adicione um campo de seletor de idioma ao fluxo de cadastro.

## Instalação {#installation}

1. Envie 'ultimate-multisite-language-selector' para o diretório '/wp-content/plugins/'
2. Ative o plugin pelo menu 'Plugins' no Admin da rede do WordPress

## Guia de uso {#usage-guide}

Com este add-on gratuito, você pode permitir que seus usuários escolham qual idioma gostariam de ver no fluxo de cadastro e na página de login.

Depois de [ativar sua licença](../../user-guide/getting-started/installing-ultimate-multisite.md), você poderá instalar os add-ons do Ultimate Multisite aos quais sua licença dá acesso.

Vá para a página de configurações do Ultimate Multisite e então, no menu à esquerda, clique no rótulo _Add-ons_.

![Página de add-ons](/img/admin/addons-page.png)

_Vá para a página de Add-ons para começar a instalar o Seletor de idioma_

Lá, você encontrará todos os add-ons que oferecemos.

Add-ons gratuitos estão disponíveis para todos os nossos clientes, independentemente do plano de assinatura em que se cadastraram.

Clique no botão _Install it Now!_ abaixo da miniatura do add-on. Em seguida, ative-o na página Plugins.

![Página de add-ons com botão de instalação](/img/admin/addons-page.png)

_Clique no botão para instalar o add-on_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_Ative-o na sua página Plugins_

Agora, vá para o menu Configurações do Ultimate Multisite e acesse a aba _Network Settings_. Lá, na parte inferior da página, você encontrará uma nova seção chamada _Languages_.

![Página de configurações](/img/config/settings-general.png)

_Uma seção novinha em folha aparece!_

Você pode deixá-la em branco para exibir todos os idiomas como opções ou selecionar quais deseja disponibilizar. Os idiomas são aqueles que você instalou na sua configuração do WordPress.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_Lista de idiomas disponíveis nesta instalação do WP_

Depois de salvar as alterações feitas, um menu de seleção com os idiomas que você escolheu será exibido na página de Login, bem como no fluxo de cadastro.

![Configurações de login e registro](/img/config/settings-login-registration.png)

_Seletor de idioma na página de Login_

![Página de registro](/img/config/registration-page.png)

_Seletor de idioma no fluxo de cadastro_

## Definindo o idioma pela URL {#setting-the-language-via-the-url}

Você também pode alterar o idioma de uma página passando o parâmetro locale para a URL. Se quiser redirecionar para uma página de registro em italiano, por exemplo, adicione ?locale=it_IT à URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

O idioma deve estar instalado na sua rede WordPress para que isso funcione.
