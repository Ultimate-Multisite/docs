# Seletor de Idioma

Multisite Ultimate: Seletor de Idioma na Inscrição

Adicione um campo de seletor de idioma ao fluxo de inscrição.

## Instalação

1. Carregue 'ultimate-multisite-language-selector' para o diretório '/wp-content/plugins/'
2. Ative o plugin através do menu 'Plugins' no seu Administrador de Rede WordPress

## Guia de Uso

Com este complemento gratuito, você pode permitir que seus usuários escolham qual idioma desejam ver no fluxo de inscrição e na página de login.

Após [ativar sua licença](../../user-guide/getting-started/installing-ultimate-multisite.md), você poderá instalar os complementos do Ultimate Multisite aos quais sua licença lhe dá acesso.

Vá para a página de configurações do seu Ultimate Multisite e, em seguida, no menu à esquerda, clique no rótulo _Complementos_.

![Add-ons page](/img/admin/addons-page.png)

_Vá para a página de Complementos para começar a instalar o Seletor de Idioma_

Lá, você encontrará todos os complementos que oferecemos.

Complementos gratuitos estão disponíveis para todos os nossos clientes, independentemente do plano de assinatura que eles tenham escolhido.

Clique no botão _Instale agora!_ abaixo da miniatura do complemento. Em seguida, ative-o na página de Plugins.

![Add-ons page with install button](/img/admin/addons-page.png)

_Clique no botão para instalar o complemento_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_Ative-o na sua página de Plugins_

Agora, vá ao menu de Configurações do Ultimate Multisite e acesse a aba _Configurações da Rede_. Lá, na parte inferior da página, você encontrará uma nova seção chamada _Idiomas_.

![Settings page](/img/config/settings-general.png)

_Uma nova seção aparece!_

Você pode deixá-la em branco para exibir todos os idiomas como opções ou selecionar quais deseja disponibilizar. Os idiomas são aqueles que você instalou em sua configuração do WordPress.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_Lista de idiomas disponíveis nesta instalação do WP_

Após salvar as alterações feitas, um menu de seleção com os idiomas escolhidos será exibido na página de Login, bem como no Fluxo de Inscrição.

![Login and Registration settings](/img/config/settings-login-registration.png)

_Seletor de Idioma na Página de Login_

![Registration page](/img/config/registration-page.png)

_Seletor de Idioma no Fluxo de Inscrição_

## Definindo o Idioma via URL

Você também pode alterar o idioma de uma página ignorando o parâmetro locale na URL. Se quiser redirecionar para uma página de registro em italiano, por exemplo, adicione ?locale=it_IT à URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

O idioma deve estar instalado na sua rede WordPress para que isso funcione.
