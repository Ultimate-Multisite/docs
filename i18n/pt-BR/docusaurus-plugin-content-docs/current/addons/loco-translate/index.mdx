# Loco Translate

O addon Loco Translate Integration do Ultimate Multisite traz recursos de tradução específicos para cada site na sua rede multisite do WordPress. Se você já precisou que sites diferentes na sua rede tenham traduções distintas do mesmo plugin ou tema, este addon é a solução perfeita.

**O que este addon faz**

Este addon integra perfeitamente o Ultimate Multisite com o popular plugin Loco Translate, permitindo que cada subsite na sua rede mantenha suas próprias traduções personalizadas. Se você administra uma plataforma multi-regional, atende a diferentes comunidades linguísticas ou simplesmente precisa de flexibilidade de tradução em toda a sua rede, este addon facilita tudo.

**Ideal para:**

* Empresas multi-regional com necessidades de localização diferentes
* Instituições educacionais que atendem comunidades diversas
* Plataformas SaaS que oferecem suporte multilíngue
* Agências web que gerenciam múltiplos sites de clientes
* Qualquer rede multisite que exija flexibilidade de tradução

## Instalação

1. Certifique-se de que o Ultimate Multisite está instalado e ativado na rede
2. Instale e ative o plugin Loco Translate
3. Faça upload deste addon para o diretório de plugins
4. Ative na rede o addon Loco Translate Integration
5. Comece a criar traduções específicas para cada site através do Loco Translate em qualquer subsite

Pronto! A integração funciona automaticamente sem nenhuma configuração adicional.

## Perguntas Frequentes

## Preciso configurar algo após a instalação?

Não! A integração é automática. Assim que tanto o addon quanto o Loco Translate estiverem ativos, os subsites podem começar imediatamente a criar suas próprias traduções personalizadas.

## O que acontece se um subsite não criar traduções personalizadas?

O site usará automaticamente as traduções de toda a rede como fallback. Nada quebra e tudo funciona perfeitamente.

## Os subsites podem acessar traduções de toda a rede?

Administradores de sites regulares só veem e editam as traduções do próprio site. Superadministradores podem acessar tanto as traduções de toda a rede quanto as específicas do site, dependendo do site que estão visualizando.

## Isso funciona com plugins e temas?

Sim! Traduções específicas por site funcionam tanto para plugins quanto para temas do WordPress.

## Isso vai deixar minha rede multisite mais lenta?

Não. O addon utiliza carregamento de arquivos eficiente que só verifica traduções específicas do site quando necessário. Não há impacto perceptível no desempenho.

## Posso copiar traduções da rede para um site específico como ponto de partida?

Sim! Ao editar traduções no Loco Translate, você pode começar facilmente com as traduções da rede e personalizá-las para cada site.

## Isso funciona com WooCommerce e outros plugins?

Absolutamente! Qualquer plugin ou tema que use funções padrão de tradução do WordPress funcionará com traduções específicas por site.
