# Personalizando seu painel de administração com PRO Theme

Frequentemente, ao realizar trabalhos para clientes, surge a necessidade de oferecer a eles uma experiência personalizada no Dashboard de administração do WordPress. Se você está procurando uma aparência e experiência mais profissionais, o PRO Theme pode ajudar.

PRO Theme é um tema de administração desenvolvido para ajudar você a personalizar o Dashboard dos seus clientes. E ele também faz parte do núcleo de plugins premium do Ultimate Multisite.

Depois de instalá-lo, vá até sua página de _Plugins_ e clique para ativá-lo na sua Network.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Ative-o em toda a network na sua página de Plugins_

_**OBSERVAÇÃO:**_ _Se você apenas instalá-lo, ele será exibido para o seu cliente como uma opção que pode ser marcada para adicionar o tema somente a esse usuário._

Agora, você tem sua interface de administração completamente redesenhada! Você também tem um novo menu na coluna à esquerda chamado PRO Theme.

_Um visual totalmente novo para o seu painel de administração_
