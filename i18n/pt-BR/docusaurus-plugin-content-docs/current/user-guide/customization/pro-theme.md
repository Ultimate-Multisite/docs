# Personalizando seu Painel Administrativo com o Tema PRO

Frequentemente, ao trabalhar com clientes, surge a necessidade de oferecer a eles uma experiência personalizada no painel administrativo do WordPress. Se você procura um visual e uma sensação mais profissional, o Tema PRO pode ajudar.

O Tema PRO é um tema administrativo desenvolvido para auxiliá-lo na personalização do Painel dos seus clientes. E também faz parte do núcleo de plugins premium do Ultimate Multisite.

Depois de instalá-lo, vá até a sua página _Plugins_ e clique para ativá-lo na sua Rede.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Ative-o em toda a rede na sua página de Plugins_

_**NOTA:**_ _Se você apenas instalá-lo, ele aparecerá no seu cliente como uma opção que pode ser marcada para adicionar o tema apenas a esse usuário._

Agora, você tem sua interface administrativa totalmente redesenhada! Você também tem um novo menu na coluna esquerda chamado Tema PRO.

<!-- Screenshot unavailable: PRO Theme redesigned admin panel interface -->

_Um visual totalmente novo para seu Painel Administrativo_
