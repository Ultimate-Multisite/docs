# Gerenciando Clientes

A página **Clientes** mostra a lista de membros registrados na sua rede. Esta página exibe informações básicas do usuário, incluindo nome, endereço de e‑mail, último login e um **switch to** link que permite que você faça login no subsitio do usuário como administrador do site.

![Página de lista de clientes](/img/admin/customers-list.png)

Clique em **Gerenciar** abaixo do cartão de um cliente para ver informações detalhadas sobre esse cliente.

Na página de detalhes do cliente, o bloco **Visão Geral** mostra o ID do cliente, hora do último login e o valor total gasto por esse cliente.

O bloco **Opções do Cliente** permite que você defina este cliente como **VIP** (uma tag para facilitar a busca). A aba **Informações de Faturamento** mostra informações de faturamento, como e‑mail, endereço e país.

O bloco **Campos** mostra o nome de usuário do cliente, e‑mail e permite que você altere o status de **Verificação de E‑mail**.

Abaixo do bloco Opções do Cliente você encontrará quatro blocos adicionais:

- **Assinaturas** — Assinaturas atuais que o cliente possui
- **Pagamentos** — Histórico de pagamentos do cliente
- **Sites** — Sites que este cliente possui
- **Eventos** — Eventos relacionados a este cliente (verificação de e‑mail, alterações de faturamento, etc.)

Clique no ícone de seta (→) ao lado de qualquer item para navegar até a página de detalhes. Por exemplo, clicar na seta ao lado de um site leva você à página de edição do site, onde pode modificar a associação de assinatura, o tipo de site e o status ativo.

O mesmo se aplica a assinaturas e pagamentos — cada um vincula a páginas de detalhes onde você pode editar o status da assinatura, o cliente associado, os produtos, as configurações do gateway de pagamento e até processar reembolsos.

No lado direito da página do cliente, você encontrará:

- **Último Login & IPs** — Último horário de login, endereço IP e país
- **Timestamps** — Quando a conta do cliente foi criada e modificada pela última vez
- **Excluir** — Opção para excluir permanentemente o cliente

:::warning
Excluir um cliente é permanente e não pode ser desfeito.
:::
