# Incompatibilidades conhecidas com outros plugins

Após instalar e ativar o Ultimate Multisite, você pode encontrar problemas devido a incompatibilidades não resolvidas com outros plugins.

Estamos trabalhando ativamente para reduzir essa lista a cada nova versão, mas se um plugin estiver listado aqui, você pode precisar desativá-lo para ter uma Rede Ultimate Multisite totalmente funcional.

Aqui está uma lista deles e o que você deve fazer nesses casos:

**Alguns plugins da WPMU DEV** Alguns dos plugins _não mais_ mantidos pela WPMU DEV são conhecidos por causar a interrupção do fluxo de registro do Ultimate Multisite na última etapa (o usuário não consegue criar uma conta). Aqui está uma lista dos plugins que causam esses problemas:

  * Idioma de Registro: _Ultimate Multisite has an add-on that offers the same functionality, called Ultimate Multisite: Language Switcher._
  * Definir Descrição do Blog na Criação do Blog: _This is built into Ultimate Multisite, so there's no need for an extra plugin._
  * Novos Modelos de Blog: _This is built into Ultimate Multisite, so there's no need for an extra plugin._
  * Definir Senha na Criação de Blog Multisite: _This is built into Ultimate Multisite, so there's no need for an extra plugin._
  * Termos de Serviço de Registro: _This is built into Ultimate Multisite, so there's no need for an extra plugin._
  * Anti-Splog: Embora menos potente, isso também está _built into Ultimate Multisite._

**Observação** : Todos os outros plugins da WPMU Dev podem ser usados normalmente junto com o Ultimate Multisite. Exemplos incluem _Smush_ , _Forminator_ , _Defender,_ etc.
