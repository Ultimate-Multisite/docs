# Ganchos e Filtros do Ultimate Multisite

Ultimate Multisite é composto por uma série de funções e consultas ao banco de dados. Ele possui ganchos e filtros que permitem modificar ou adicionar novo código às funções existentes.

Criamos uma lista desses ganchos e filtros como referência para ajudar usuários avançados a personalizar ainda mais sua rede. Você pode acessar a lista no Github clicando no link abaixo.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
