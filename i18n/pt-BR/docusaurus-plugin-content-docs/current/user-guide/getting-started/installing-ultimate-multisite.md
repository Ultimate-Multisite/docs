# Instalando Ultimate Multisite

:::note
Este tutorial assume que você já tem o WordPress Multisite instalado e configurado. Para aprender como, confira [este tutorial](https://www.wpbeginner.com/glossary/multisite/) do WP Beginner.
:::

## Instalando o Plugin

Ultimate Multisite está disponível gratuitamente em [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/).

No seu **Painel de Administração de Rede**, vá para **Plugins → Adicionar Novo Plugin**.

![Página Adicionar Novo Plugin no Painel de Administração de Rede](/img/installation/add-new-plugin.png)

Pesquise por **"Ultimate Multisite"** (com aspas para uma correspondência exata) e aparecerá como o primeiro resultado. Clique em **Install Now**.

![Resultados da pesquisa mostrando Ultimate Multisite](/img/installation/search-ultimate-multisite.png)

Depois de instalado, clique em **Ativar Rede** para ativar o plugin em toda a sua rede.

![Plugin instalado com botão Ativar Rede](/img/installation/plugin-installed.png)

Após a ativação, você será redirecionado automaticamente para o Assistente de Configuração.

![Plugin ativado e redirecionado para o assistente](/img/installation/plugin-activated.png)

## Assistente de Configuração

O Assistente de Configuração irá guiá-lo na configuração do Ultimate Multisite em cerca de 10 minutos.

### Boas-vindas

Clique em **Começar** para iniciar.

![Tela de boas-vindas do Assistente de Configuração](/img/installation/wizard-welcome.png)

### Verificações Pré-Instalação

Esta etapa verifica as informações do seu sistema e a instalação do WordPress para garantir que atendam aos requisitos do Ultimate Multisite. Se tudo estiver bom, clique em **Ir para o Próximo Passo**.

![Verificações pré-instalação mostrando requisitos do sistema](/img/installation/wizard-pre-install-checks.png)

### Instalação

O instalador criará as tabelas de banco de dados necessárias e instalará o arquivo `sunrise.php` que o Ultimate Multisite precisa para funcionar. Clique em **Instalar** para prosseguir.

![Etapa de instalação mostrando tabelas de banco de dados e sunrise.php](/img/installation/wizard-installation.png)

### Sua Empresa

Preencha as informações da sua empresa e defina a moeda padrão. Essas informações serão usadas em toda a sua plataforma WaaS. Clique em **Continuar** quando terminar.

![Etapa de configuração da sua empresa](/img/installation/wizard-your-company.png)

### Conteúdo Padrão

Esta etapa permite instalar modelos, produtos e outros conteúdos iniciais predefinidos. É uma ótima maneira de se familiarizar com os recursos do Ultimate Multisite. Clique em **Instalar** para adicionar o conteúdo padrão, ou ignore esta etapa se preferir começar do zero.

![Etapa de instalação de conteúdo padrão](/img/installation/wizard-default-content.png)

### Plugins Recomendados

Opcionalmente, instale os plugins recomendados. Clique em **Instalar** para adicioná-los ou ignore para continuar.

![Etapa de plugins recomendados](/img/installation/wizard-recommended-plugins.png)

### Pronto!

Pronto! Sua instalação do Ultimate Multisite está concluída. Você pode agora começar a construir sua plataforma Website como Serviço a partir do **Painel de Administração de Rede**.

![Tela de configuração completa - Pronto](/img/installation/wizard-ready.png)

![Painel de Administração de Rede com Ultimate Multisite ativo](/img/installation/network-dashboard.png)

Vá em frente e divirta-se!
