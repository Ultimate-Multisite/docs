# Gerenciando Links Compartilháveis para Planos

Ultimate Multisite oferece a opção de compartilhar links de plano diretamente com seus clientes. Para fazer isso, acesse a página de edição de qualquer um dos seus planos e clique no botão _Clique para copiar o Link Compartilhável_.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-2DJoMvapXh.png)

_No Página de Edição do Plano, você pode encontrar o botão para criar links compartilháveis_

Observe que esse mecanismo usa a frequência de faturamento padrão para construir o link. Isso significa que, se o seu faturamento padrão for mensal, por exemplo, no final, você terá um link como este: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Se você deseja que o link compartilhável seja padrão para trimestral, terá que adicionar 3 no final

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Para selecionar o plano _premium_ em um cronograma anual, você adicionaria 1 e depois a palavra:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Tenha em mente que você não pode ter apenas um link para diferentes frequências de faturamento. Você deve compartilhar um link específico para cada uma das suas frequências de faturamento.

Assim que você compartilhar o link com seus clientes, ele os redirecionará para a primeira etapa do seu fluxo de cadastro.
