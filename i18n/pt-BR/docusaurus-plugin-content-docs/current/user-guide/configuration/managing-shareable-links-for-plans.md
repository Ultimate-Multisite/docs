# Gerenciando links compartilháveis para planos

Ultimate Multisite oferece a opção de compartilhar links de planos diretamente com seus clientes. Para fazer isso, vá para qualquer uma das páginas de edição dos seus planos e clique no botão _Clique para copiar o link compartilhável_.

![Página de edição de produto com botão de link compartilhável](/img/config/product-edit-full.png)

_Na página de edição do plano, você pode encontrar o botão para criar links compartilháveis_

Observe que esse mecanismo usa a frequência de cobrança padrão para criar o link. Isso significa que, se sua cobrança padrão for mensal, por exemplo, no final, você terá um link como este: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Se você quiser que o link compartilhável use trimestral como padrão, terá que adicionar 3 no final

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Para selecionar o plano _premium_ em uma programação anual, você adicionaria 1 e depois a palavra:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Tenha em mente que você não pode ter apenas um link para diferentes frequências de cobrança. Você deve compartilhar um link específico para cada uma das suas frequências de cobrança.

Depois que você compartilhar o link com seus clientes, ele os redirecionará para a primeira etapa do seu fluxo de cadastro.
