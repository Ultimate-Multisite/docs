# Traduzindo Ultimate Multisite

Ultimate Multisite é desenvolvido em inglês, mas o plugin é totalmente traduzível e nós acolhemos ativamente contribuições de tradução.

## Traduções do Plugin Principal

O plugin principal Ultimate Multisite é traduzido através da plataforma oficial de tradução **WordPress.org**:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

Este é o sistema padrão de tradução da comunidade WordPress. Quando as traduções atingem um limiar de aprovação suficiente, elas são automaticamente distribuídas a todos os usuários do Ultimate Multisite através das atualizações do plugin WordPress.

**Precisamos da sua ajuda!** Se você fala um idioma diferente do inglês, por favor considere contribuir com traduções. Mesmo traduzir algumas strings ajuda. Para começar:

1. Visite [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)
2. Selecione seu idioma
3. Clique em "Contribute Translation"
4. Você precisará de uma conta gratuita no WordPress.org

Se você gostaria de se tornar um editor de tradução para o seu idioma (capaz de aprovar traduções), entre em contato via [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions).

## Traduções de Addons

Os plugins de addon são traduzidos separadamente em:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

As traduções de addons são atualmente geradas por IA como ponto de partida. Elas cobrem todos os idiomas suportados, mas podem conter frases estranhas ou imprecisões. Acolhemos tradutores humanos para revisar e sugerir melhorias, tornando-as mais naturais e precisas.

## Traduções Locais

Se você prefere criar traduções localmente para o seu próprio projeto em vez de contribuir para o upstream, existem duas opções:

### Usando Poedit

Você pode usar o aplicativo de desktop **[Poedit](https://poedit.net)** para traduzir o arquivo `.pot`:

1. Baixe o Poedit em [poedit.net](https://poedit.net)
2. Abra o arquivo `.pot` em `wp-content/plugins/ultimate-multisite/lang/`
3. Traduza as strings na interface do Poedit
4. Salve -- o Poedit gera arquivos `.po` e `.mo`
5. Coloque os arquivos gerados na pasta `lang`

### Usando Loco Translate

O plugin WordPress **[Loco Translate](https://wordpress.org/plugins/loco-translate/)** permite que você traduza diretamente do painel administrativo sem nenhuma ferramenta de desktop. Ele cuida da criação e compilação de arquivos de tradução para você.
