Filtra se o cookie de autenticação deve ser enviado apenas via HTTPS.

## Parâmetros

| Nome | Tipo | Descrição |
|------|------|-----------|
| $secure | `bool` | Se o cookie deve ser enviado apenas via HTTPS. |
| $user_id | `int` | ID do usuário. |
