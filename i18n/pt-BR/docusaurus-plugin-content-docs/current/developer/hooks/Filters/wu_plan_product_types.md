# Filtro: wu_plan_product_types

Filtrar os tipos de produto que devem ser tratados como planos.

Este filtro permite que addons registrem tipos de produto adicionais que devem ser reconhecidos como planos na validação e segregação.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | Array de tipos de produto a serem tratados como planos. |

### Desde

- 2.3.0
