# Filtro: wu_add_new_domain_modal_fields {#filter-wuaddnewdomainmodalfields}

Filtra os campos para o modal de adicionar novo domínio.

Permite addons (por exemplo, Domain Seller) modificar ou substituir os campos de registro de domínio.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $fields | `array` | The form fields. |
