Filter whether to require uppercase letters in passwords.

## Parameters

| Nome | Tipo | Descrição |
|------|------|-------------|
| $require | `bool` | Se deve exigir letras maiúsculas. Padrão verdadeiro quando as regras são aplicadas. |
| $defender_active | `bool` | Se o Defender Pro Strong Password está ativo. |
