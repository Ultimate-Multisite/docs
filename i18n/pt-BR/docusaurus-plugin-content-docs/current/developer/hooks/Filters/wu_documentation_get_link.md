# Filtro: wu_documentation_get_link

Permite que desenvolvedores de plugins filtrem os links.

Não tenho certeza de como isso pode ser útil, mas não custa ter.

## Parâmetros

| Nome | Tipo | Descrição |
|------|------|-------------|
| $link | `string` | O link registrado |
| $slug | `string` | O slug usado para recuperar o link |
| $default_link | `string` | O link padrão registrado |

### Desde

- 1.7.0
