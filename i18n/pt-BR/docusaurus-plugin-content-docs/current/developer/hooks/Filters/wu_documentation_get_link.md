# Filter: wu_documentation_get_link

Permitir que desenvolvedores de plugins filtrem os links.

Não tenho certeza de como isso pode ser útil, mas não há mal em tê-lo.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $link | `string` | O link registrado |
| $slug | `string` | O slug usado para recuperar o link |
| $default_link | `string` | O link padrão registrado |
