# Filter: wu_maybe_redirect_to_admin_panel_exclusion_list {#filter-wumayberedirecttoadminpanelexclusionlist}

Crie uma lista de exclusão de parâmetros que impedem o redirecionamento automático.

Isso é necessário porque, caso contrário, o construtor de páginas não poderá editar a página de login depois que ela for definida.
