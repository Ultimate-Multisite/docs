# Filtrar: wu_independent_billing_cycle_product_types

Filtrar os tipos de produto que têm ciclos de faturamento independentes.

Produtos com ciclos de faturamento independentes não são obrigados a corresponder ao período de faturamento do plano no carrinho. Isso é útil para produtos como registros de domínio que sempre faturam anualmente, independentemente de o plano ser mensal ou anual.

## Parâmetros

| Nome | Tipo | Descrição |
|------|------|-----------|
| $types | `array` | Array de tipos de produto com ciclos de faturamento independentes. |
