# Filtro: wu_view_override_replaceable_views

Permite que desenvolvedores adicionem pastas adicionais à lista substituível.

Cuidado, pois permitir pastas adicionais pode fazer com que cópias desatualizadas sejam carregadas em vez das versões do Ultimate Multisite.

## Parâmetros

| Nome | Tipo | Descrição |
|------|------|-----------|
| $replaceable_views | `array` | Lista de pastas permitidas. |

### Desde

- 2.0.0

### Fonte

Definido em [`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L50) na linha 50
