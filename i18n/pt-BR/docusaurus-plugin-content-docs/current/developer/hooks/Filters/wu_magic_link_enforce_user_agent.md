# Filter: wu_magic_link_enforce_user_agent

Filtrar se deve impor a verificação do agente do usuário.

Defina como falso para permitir que os tokens funcionem em diferentes navegadores/dispositivos. Isso reduz a segurança, mas aumenta a usabilidade.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $enforce | `bool` | Se deve impor a correspondência do agente do usuário. |
