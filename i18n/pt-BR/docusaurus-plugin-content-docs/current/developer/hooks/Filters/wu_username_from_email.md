# Filtro: wu_username_from_email

Filtrar nome de usuário do novo cliente.

## Parâmetros

| Nome | Tipo | Descrição |
|------|------|-----------|
| $username | `string` | Nome de usuário do cliente. |
| $email | `string` | Endereço de e-mail do novo cliente. |
| $new_user_args | `array` | Array de argumentos de novo usuário, talvez incluindo nome e sobrenome. |
| $suffix | `string` | Anexar string ao nome de usuário para torná-lo único. |
