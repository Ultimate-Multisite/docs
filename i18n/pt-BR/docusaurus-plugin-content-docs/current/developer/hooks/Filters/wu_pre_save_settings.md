# Filtro: wu_pre_save_settings

Permitir que os desenvolvedores filtrem as configurações antes de salvar pelo Ultimate Multisite.

## Parâmetros

| Nome | Tipo | Descrição |
|------|------|-------------|
| $settings | `array` | As configurações a serem salvas. |
| $settings_to_save | `array` | As novas configurações a adicionar. |
| $saved_settings | `array` | As configurações atuais salvas. |

- 2.0.18
