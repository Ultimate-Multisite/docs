# Ação: wu_mcp_adapter_initialized

Dispara após o MCP adapter ser inicializado.

Permite que outros plugins e temas registrem suas próprias habilidades.

## Parâmetros

| Name | Type | Description |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | A instância do MCP adapter. |

### Desde

- 2.5.0
