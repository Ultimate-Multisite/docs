# Ação: wu_site_created

Dispara após um site ser criado pela primeira vez.

Não dispara se duplicado a partir de um modelo.

## Parameters

| Nome | Tipo | Descrição |
|------|------|-----------|
| $data | `array` | Os dados do objeto que serão armazenados. |
| $site | `\Site` | A instância do objeto. |
