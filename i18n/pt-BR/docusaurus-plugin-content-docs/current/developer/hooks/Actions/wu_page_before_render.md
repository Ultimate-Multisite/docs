# Ação: wu_page_before_render

Permitir que desenvolvedores de plugins adicionem conteúdo adicional antes de imprimirmos a página.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $page_id | `string` | O id desta página. |
| $page | `object` | O objeto da página. |
