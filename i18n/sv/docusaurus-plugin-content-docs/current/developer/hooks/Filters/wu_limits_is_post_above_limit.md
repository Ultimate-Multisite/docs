# Filter: wu_limits_is_post_above_limit

Kontrollerar om en given post type är tillåten på denna plan. Gör det möjligt för plugin-utvecklare att filtrera returvärdet.

## Parametrar {#parameters}

| Namn | Typ | Beskrivning |
|------|------|-------------|
| $value | `bool` | Om post typen är inaktiverad eller inte |
