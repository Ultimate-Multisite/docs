# Filter: wu_stripe_existing_product_id

Filtrerar ID:t för den produkt som ska kontrolleras. Om detta finns, kommer det nya abonnemanget att använda denna produkt.

## Parametrar

| Name | Type | Description |
|------|------|-------------|
| $product_id | `string` | ID för Stripe-produkten som ska kontrolleras. |
| $name | `string` | Produktnamn. |
