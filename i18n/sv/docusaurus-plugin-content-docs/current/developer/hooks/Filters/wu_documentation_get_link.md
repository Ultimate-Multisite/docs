# Filter: wu_documentation_get_link {#filter-wudocumentationgetlink}

Låter tillåta plugin-utvecklare att filtrera länkar.

Jag är inte säker på hur användbart det kan vara, men det skadar inte att ha det med.

## Parametrar {#parameters}

| Namn | Typ | Beskrivning |
|------|------|-------------|
| $link | `string` | Länken som registreras |
| $slug | `string` | Slugen som används för att hämta länken |
| $default_link | `string` | Den standardlänk som registreras |
