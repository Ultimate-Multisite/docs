# Filter: wu_independent_billing_cycle_product_types {#filter-wuindependentbillingcycleproducttypes}

Filtrerar produktkategorier som har oberoende faktureringscykler.

Produkter med oberoende faktureringscykler tvingas inte att matcha planens faktureringsperiod i varukorgen. Detta är användbart för produkter som domänregistreringar, som alltid faktureras årligen oavsett om planen är månatlig eller årlig.

## Parametrar {#parameters}

| Namn | Typ | Beskrivning |
|------|------|-------------|
| $types | `array` | Array med produktkategorier som har oberoende faktureringscykler. |

### Sedan {#since}

- 2.5.0
