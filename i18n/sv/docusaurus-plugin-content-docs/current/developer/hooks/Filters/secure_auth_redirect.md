# Filter: secure_auth_redirect

Bestämmer om man ska använda en säker autentiseringsomdirigering.

## Parametrar

| Name | Type | Description |
|------|------|-------------|
| $secure | `bool` | Bestämmer om en säker autentiseringsomdirigering ska användas. Standardvärde är false. |

- 3.1.0
