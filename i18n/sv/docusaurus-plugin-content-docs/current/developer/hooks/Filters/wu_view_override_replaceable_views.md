# Filter: wu_view_override_replaceable_views {#filter-wuviewoverridereplaceableviews}

Låter utvecklare lägga till ytterligare mappar i listan över ersättningsbara mappar.

Var försiktig, eftersom tillåtelse att lägga till extra mappar kan få systemet att ladda gamla kopior istället för de versioner som Ultimate Multisite använder.

## Parametrar {#parameters}

| Namn | Typ | Beskrivning |
|------|------|-------------|
| $replaceable_views | `array` | Lista över tillåtna mappar. |

### Sedan {#since}

- 2.0.0
### Källa {#source}

Definieras i [`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L50) på rad 50
