# Filter: wu_\{$id\}_form_fields

Filtrerar fälten på ett formulär. Formuläret identifieras av ID:t i filternamnet.

## Parametrar

| Namn | Typ | Beskrivning |
|------|------|-------------|
| $fields | `array` | En lista över formulärelement i formuläret. |
