# Filter: wu_plan_product_types

Filtrerar de produktkategorier som ska behandlas som planer.

Detta filter gör det möjligt för tillägg (addons) att registrera ytterligare produktkategorier som ska kännas igen som planer vid validering och separation.

## Parametrar

| Namn | Typ | Beskrivning |
|------|------|-------------|
| $plan_types | `array` | Array med produktkategorier som ska behandlas som planer. |

### Sedan

- 2.3.0
### Källa

Definieras i [`inc/functions/product.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/product.php#L196) på rad 196
