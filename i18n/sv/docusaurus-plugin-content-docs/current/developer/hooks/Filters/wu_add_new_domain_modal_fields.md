# Filter: wu_add_new_domain_modal_fields {#filter-wuaddnewdomainmodalfields}

Filtrerar fälten för modalen för att lägga till ett nytt domännamn.

Låter tillägg (t.ex. Domain Seller) modifiera eller ersätta fälten för domänregistrering.

## Parametrar {#parameters}

| Namn | Typ | Beskrivning |
|------|------|-------------|
| $fields | `array` | Formulärfälten. |
