# Filter: wp_ultimo_redirect_url_after_signup

Ställer in omdirigerings-URL:en.

Detta är ett föråldrat filter. Vissa av parametrarna som skickas är inte tillgängliga, till exempel `$site_id`.

### Sedan

- 1.1.3: Gör det möjligt för utvecklare att filtrera omdirigerings-URL:en.
