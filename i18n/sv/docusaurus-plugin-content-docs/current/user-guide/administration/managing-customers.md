# Hantera kunder

Sidan **Kunder** visar listan över medlemmar som är registrerade i ditt nätverk. Här visas grundläggande användarinformation som namn, e-postadress, senaste inloggning och en **byt till**-länk som låter dig logga in på användarens underwebbplats som webbplatsadministratör.

![Kundlistsida](/img/admin/customers-list.png)

Klicka på **Hantera** under en kunds kort för att se detaljerad information om den kunden.

På kundens detaljsida visar blocket **Översikt** kund-ID, senaste inloggningstid och totalt belopp som kunden har spenderat.

I blocket **Kundalternativ** kan du markera kunden som **VIP** (en tagg som gör det lättare att hitta dem). Fliken **Faktureringsinfo** visar faktureringsinformation som e-post, adress och land.

Blocket **Fält** visar kundens användarnamn, e-postadress och låter dig ändra status för **E-postverifiering**.

Under blocket Kundalternativ hittar du fyra ytterligare block:

- **Medlemskap** — Aktuella prenumerationer som kunden prenumererar på
- **Betalningar** — Kundens betalningshistorik
- **Webbplatser** — Webbplatser som kunden äger
- **Händelser** — Händelser relaterade till kunden (e-postverifiering, faktureringsändringar m.m.)

Klicka på pilikonen (→) bredvid ett objekt för att navigera till dess detaljsida. Om du till exempel klickar på pilen bredvid en webbplats kommer du till webbplatsens redigeringssida där du kan ändra medlemskapsassociation, webbplatstyp och aktiveringsstatus.

Samma sak gäller för medlemskap och betalningar — var och en länkar till detaljsidor där du kan redigera medlemskapsstatus, associerad kund, produkter, inställningar för betalningsgateway och till och med hantera återbetalningar.

På höger sida av kundsidan hittar du:

- **Senaste inloggning & IP-adresser** — Senaste inloggningstid, IP-adress och land
- **Tidsstämplar** — När kundkontot skapades och senast ändrades
- **Ta bort** — Möjlighet att permanent ta bort kunden

:::warning
Att ta bort en kund är permanent och kan inte ångras.
:::
