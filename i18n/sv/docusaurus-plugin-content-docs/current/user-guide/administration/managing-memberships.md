# Hantera medlemskap

I Ultimate Multisite kan du hantera medlemskap och få tillgång till alla prenumerationer i ditt nätverk. Granska och redigera prenumerationsdetaljer som planer och produkter, antal webbplatser, fakturabelopp och faktureringsfrekvens, betalningshistorik och tidsstämplar.

För att komma till den här sidan, gå till fliken **Memberships** i Ultimate Multisite-menyn.

![Medlemskapslista](/img/admin/memberships-list.png)

På sidan Memberships ser du en lista över dina kunder och de produkter som är kopplade till deras konton. Du kan filtrera medlemskap efter status: aktiva, väntande, pausade, utgångna eller avbrutna.

Klicka på ett medlemskap för att komma till sidan där du kan redigera medlemskapsdetaljerna — faktureringsinformation, begränsningar för webbplatser eller besök, med mera.

Du kan också lägga till nya medlemskap kopplade till användare som redan är registrerade i ditt nätverk. Klicka på **Add Membership** och konfigurera detaljerna.

![Modal för att lägga till medlemskap](/img/admin/membership-add-modal.png)

Klicka sedan på **Create Membership**. Det nya medlemskapet visas nu i medlemskapslistan.
