# Ultimate Multisite Hooks och filter

Ultimate Multisite är uppbyggt av en mängd funktioner och databasfrågor. Det har hooks och filter som låter dig modifiera befintliga funktioner eller lägga till ny kod.

Vi har sammanställt en lista över dessa hooks och filter som referens för avancerade användare som vill anpassa sitt nätverk ytterligare. Du kommer åt listan på Github via länken nedan.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
