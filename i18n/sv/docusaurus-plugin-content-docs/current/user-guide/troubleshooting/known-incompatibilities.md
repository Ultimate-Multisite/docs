# Kända inkompatibiliteter med andra tillägg

Efter att du har installerat och aktiverat Ultimate Multisite kan du stöta på problem på grund av olösta inkompatibiliteter med andra tillägg.

Vi arbetar aktivt med att minska den här listan i varje ny version, men om ett tillägg finns listat här kan du behöva inaktivera det för att få ett fullt fungerande Ultimate Multisite-nätverk.

Här är en lista över dessa tillägg och vad du bör göra i varje fall:

**Vissa WPMU DEV-tillägg** Några av tilläggen som _inte längre_ underhålls av WPMU DEV är kända för att orsaka att Ultimate Multisites registreringsflöde stannar på det sista steget (användaren kan inte skapa ett konto). Här är en lista över tilläggen som orsakar dessa problem:

  * Signup Language: _Ultimate Multisite har ett tillägg som erbjuder samma funktionalitet, kallat Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Det här är inbyggt i Ultimate Multisite, så det finns inget behov av ett extra tillägg._

  * New Blog Templates: _Det här är inbyggt i Ultimate Multisite, så det finns inget behov av ett extra tillägg._

  * Set Password on Multisite Blog Creation: _Det här är inbyggt i Ultimate Multisite, så det finns inget behov av ett extra tillägg._

  * Signup TOS: _Det här är inbyggt i Ultimate Multisite, så det finns inget behov av ett extra tillägg._

  * Anti-Splog: Även om det är mindre kraftfullt, är det här också _inbyggt i Ultimate Multisite._

**Observera**: Alla andra WPMU Dev-tillägg kan användas normalt tillsammans med Ultimate Multisite. Exempel inkluderar _Smush_, _Forminator_, _Defender_ med flera.
