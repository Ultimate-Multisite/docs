# Anpassa din adminpanel med PRO Theme

När vi jobbar med kunduppdrag behöver vi ofta ge våra kunder en skräddarsydd upplevelse i WordPress adminpanel. Om du letar efter ett mer professionellt utseende och känsla kan PRO Theme vara till hjälp.

PRO Theme är ett admintema som är utvecklat för att hjälpa dig anpassa dina kunders Dashboard. Det ingår också i Ultimate Multisites paket med premiumtillägg.

Efter installationen går du till sidan _Tillägg_ och klickar för att aktivera det på ditt nätverk.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Aktivera det nätverksövergripande på din Tilläggssida_

_**OBS:**_ _Om du bara installerar det kommer det visas som ett alternativ hos dina kunder, där de kan välja att lägga till temat endast för den användaren._

Nu har du ett helt nydesignat admingränssnitt! Du har också en ny meny i vänsterkolumnen som heter PRO Theme.
