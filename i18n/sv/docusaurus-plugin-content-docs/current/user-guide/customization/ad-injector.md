# Placera annonser på ditt nätverk med Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector är ett **gratis tillägg** som ger dig verktygen för att placera annonser på dina planer direkt från deras redigeringssida.

Efter att du har installerat Ultimate Multisite: Ad Injector, gå till din _Tillägg_-sida för att aktivera det.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Klicka på knappen Nätverksaktivera_

När tillägget är aktiverat kommer metarutan _Avancerade alternativ_ på planens redigeringsskärm att ha ett nytt alternativ som heter _Annonsinställningar_.

![Annonsinställningar på planens redigeringsskärm](/img/config/product-edit-full.png)

_Nu kan du redigera annonsinställningar för var och en av dina planer_

På den fliken kan du lägga till annonskoder för inlägg och sidor i front-end, före och efter innehållet.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Efter att du har infogat din annonskod kommer den att visas på inlägg och sidor för den specifika planen._

Du kan också ha annonser inkluderade i sidhuvudet på adminsidor för användare av den planen.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Gå till alternativet Adminpanel-annonskod om du vill visa annonser i din användares back-end_

### Mer än bara reklam

Du kan också använda Ultimate Multisite: Ad Injector för att erbjuda support till planer på högre nivå.

Gå bara till alternativet _Adminpanel-annonser_ och lägg till HTML-koden för ditt supportverktyg.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Lägg till kundsupport till specifika planer_

På så sätt kan folk nå dig direkt från sin adminpanel!
