# Hantera delbara länkar för planer

Ultimate Multisite ger dig möjlighet att dela planlänkar direkt med dina kunder. För att göra detta, gå till redigeringssidan för någon av dina planer och klicka på knappen _Klicka för att kopiera den delbara länken_.

![Produktredigeringssida med knapp för delbar länk](/img/config/product-edit-full.png)

_På planens redigeringssida hittar du knappen för att skapa delbara länkar_

Observera att denna mekanism använder standardfaktureringsfrekvensen för att bygga länken. Det betyder att om din standardfakturering är månadsvis, till exempel, kommer du i slutändan att få en länk som ser ut så här: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Om du vill att den delbara länken ska ha kvartalsvis fakturering som standard, behöver du lägga till 3 i slutet

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

För att välja planen _premium_ med årsschema lägger du till 1 följt av ordet:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Tänk på att du inte kan ha bara en länk för olika faktureringsfrekvenser. Du bör dela en specifik länk för var och en av dina faktureringsfrekvenser.

När du delar länken med dina kunder kommer den att dirigera dem till det första steget i ditt registreringsflöde.
