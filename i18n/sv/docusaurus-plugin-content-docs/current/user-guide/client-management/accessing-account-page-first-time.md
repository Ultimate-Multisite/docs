# Hur kommer jag åt mitt konto för första gången?

Välkommen till Ultimate Multisite!

Du når din kontosida via länken du fick i e-postmeddelandet efter att du köpte vårt plugin:

<!-- Screenshot unavailable: Account page email with access link -->

Använd e-postadressen du angav vid köpet och lösenordet du fick i e-postmeddelandet.

<!-- Screenshot unavailable: Login page with email and password fields -->

Eftersom det här är första gången du loggar in på det här kontot kommer du att bli ombedd att byta lösenord. Byt lösenord och klicka på knappen _Spara_.

<!-- Screenshot unavailable: Change password form with Save button -->

På din kontosida kan du hantera alla dina kontouppgifter som fakturering, förnyelser, uppsägningar och mycket annat.
