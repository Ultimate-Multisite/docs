# Hur avbryter jag mitt abonnemang?

Om du har en Ultimate Multisite-licens kan du när som helst avbryta förnyelsen. Följ bara stegen nedan:


<!-- Screenshot unavailable: Account page email with access link -->

Använd e-postadressen du angav vid köpet tillsammans med ditt lösenord.

<!-- Screenshot unavailable: Login page with email and password fields -->

Det här är startsidan för ditt konto:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

För att avbryta ditt abonnemang, gå till _Renewing & Billing_ i menyn till vänster.

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

Klicka på pilen till höger för att öppna ett sidofönster. Välj sedan alternativet som stänger av den **automatiska förnyelsen**.

Systemet visar ett bekräftelsemeddelande.

Om du är säker på att du vill avbryta ditt abonnemang klickar du på knappen _**Cancel Renewals**_. Efter detta ombeds du att svara på en kort enkät.

Klart! Ditt abonnemang förnyas inte automatiskt.

**Du har en giltig licensnyckel tills ditt abonnemang löper ut**. Om du vill återaktivera ditt abonnemang måste du göra det manuellt.
