# Ultimate Multisite Dokumentation

Välkommen till Ultimate Multisite dokumentationen. Ultimate Multisite förvandlar WordPress Multisite till en fullständig plattform för Webbplats som en tjänst (WaaS).

## Dokumentationssektioner {#documentation-sections}

### [Användarguide](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

Allt du behöver för att ställa in, konfigurera och hantera din WaaS-plattform. Täcker installation, betalningsgatewayar, domänmappning, klienthantering och mer.

### [Utvecklardokumentation](/developer/hooks) {#developer-documentation}

Teknisk referens för utvecklare som bygger på Ultimate Multisite. Innehåller referens för hooks, REST API-dokumentation och guider för addon-utveckling.

### [Tillägg](/addons) {#addons}

Dokumentation för betalda addon-plugins som utökar Ultimate Multisite med ytterligare funktioner som WooCommerce-integration, analys och domänförsäljning, och mer.
