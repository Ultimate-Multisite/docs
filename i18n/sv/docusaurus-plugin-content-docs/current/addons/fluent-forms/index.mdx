# Fluent Forms

Detta tillägg gör det möjligt att lägga till gränser för Fluent Forms i Ultimate Multisite, antingen i produkter, medlemskap eller på hela webbplatser.
Fluent Forms använder anpassade tabeller istället för anpassade posttyper, vilket innebär att de vanliga gränsinställningarna inte fungerar med Fluent Forms.

## Funktioner {#features}

- Lägg till gränser för Fluent Forms eller Conversational Fluent Forms
- Kunderna får ett vänligt meddelande som uppmanar dem att uppgradera sitt abonnemang när de når gränsen
