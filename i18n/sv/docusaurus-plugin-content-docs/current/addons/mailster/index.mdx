# Mailster

Förvandla ditt multisite-nätverk till en kraftfull maskin för att bygga e-postlistor med Ultimate Multisite Mailster Integration-addonet. Lägg automatiskt till nya kunder i dina Mailster-e-postlistor när de registrerar sig eller gör köp, vilket skapar en sömlös koppling mellan din medlemskapsplattform och din e-postmarknadsföringsstrategi.

## Huvudfunktioner {#key-features}

**Automatisk prenumerant-synkronisering**
* Lägger automatiskt till kunder i Mailster-listor när de registrerar sig
* Välj att synkronisera direkt vid registrering eller vänta på betalningsbekräftelse
* Håller dina e-postlistor aktuella utan manuell inblandning

**Flexibel listhantering**
* Ställ in globala standardlistor för alla nya kunder
* Överskugga standardinställningarna med produkt-specifika listor för riktade kampanjer
* Hantera listtilldelningar enkelt med Mailsters välbekanta gränssnitt

**GDPR-kompatibla opt-in-alternativ**
* Välj mellan automatisk prenumeration eller samtycke via kryssruta
* Aktivera dubbel opt-in för bekräftade prenumerationer
* Visa/dölj kryssrutan för opt-in baserat på dina krav på regelefterlevnad

**Smart datamappning**
* Synkroniserar automatiskt kundnamn och kontaktinformation
* Mappar faktureringsadresser till prenumerantfält
* Spårar registreringsdatum för riktade kampanjer

**Kontroll på produktnivå**
* Tilldela olika listor till olika produkter
* Skapa riktade kampanjer baserat på vad kunderna köper
* Stäng av Mailster för specifika produkter vid behov

**Professionell upplevelse**
* Inbyggd Mailster UI-integration för listval
* Detaljerad loggning för felsökning och verifiering
* Sömlösa uppdateringar av befintliga prenumeranter

## Perfekt för {#perfect-for}

* Ägare av medlemskapswebbplatser som bygger e-postlistor
* SaaS-plattformar som vill vårda nya kunder
* Kursupphovsmän som segmenterar studenter efter inskrivning
* Alla som använder Ultimate Multisite med Mailster

## Hur det fungerar {#how-it-works}

1. Installera och aktivera addonet på ditt Ultimate Multisite-nätverk
2. Konfigurera globala standardlistor i Ultimate Multisite-inställningarna
3. Överordna valfria produkt-specifika listor
4. Välj ditt opt-in-läge och prenumerationstidpunkt
5. Se dina Mailster-listor växa automatiskt!

## Installation {#installation}

1. Ladda upp plugin-filerna till din Ultimate Multisite addons-katalog
2. Se till att Mailster är installerat och aktivt på din huvudwebbplats
3. Aktivera addonet via Ultimate Multisite addons-sidan
4. Navigera till Inställningar > Mailster Integration för att konfigurera
5. Välj dina standardlistor och konfigurera opt-in-preferenser

## Vanliga frågor {#frequently-asked-questions}

## Fungerar detta add-on utan Ultimate Multisite? {#does-this-addon-work-without-ultimate-multisite}

Nej, detta add-on kräver Ultimate Multisite för att fungera. Det är specifikt utformat för att integrera Mailster med Ultimate Multisites kund- och produktsystem.

## Behöver jag ha Mailster installerat på alla webbplatser? {#do-i-need-mailster-installed-on-all-sites}

Nej, Mailster behöver endast vara installerat och aktiverat på din huvudnätverkswebbplats. Addonet kommer automatiskt att ansluta till den från alla sub-sidor.

## Kan jag tilldela olika listor till olika produkter? {#can-i-assign-different-lists-to-different-products}

Ja! Du kan ställa in globala standardlistor och sedan överskugga dem per produkt. Detta gör att du kan skapa mycket riktade e-postkampanjer baserat på vad kunderna köper.

## Vad händer om en kund redan finns i Mailster? {#what-happens-if-a-customer-already-exists-in-mailster}

Som standard kommer addonet att uppdatera befintliga prenumeranter och lägga till dem i nya listor. Du kan stänga av detta i inställningarna om du föredrar att hoppa över befintliga prenumeranter.

## Är detta GDPR-kompatibelt? {#is-this-gdpr-compliant}

Ja! Addonet inkluderar ett opt-in-läge baserat på kryssrutor där kunder måste ge uttryckligt samtycke till att ta emot e-post. Detta uppfyller GDPR-kraven i kombination med lämpliga integritetspolicyer.

## Kan jag använda dubbel opt-in? {#can-i-use-double-opt-in}

Absolut! Aktivera inställningen för dubbel opt-in, så skickar Mailster bekräftelse-e-postmeddelanden innan prenumerationer aktiveras.
