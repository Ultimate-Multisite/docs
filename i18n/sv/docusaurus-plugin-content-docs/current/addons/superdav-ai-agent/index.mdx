# Superdav AI Agent

Superdav AI Agent är en avancerad WordPress AI-assistent som utökar Gratis AI Agent med temamedvetna färdigheter, förbättrade internetsökningsfunktioner, funktioner för webbplatsbyggare och valfria automatiseringsintegrationer. Version 1.18.0 lägger till läsverktyg för Google Calendar, mappning av kontakter och deltagare, godkännandesteg för människor, deduplicering av påminnelser, TextBee SMS-aviseringar, provisionering av hanterad Superdav-tjänst och ett separat distribuerat Advanced-kompanjonpaket för betrodda utvecklararbetsflöden.

## Viktiga funktioner {#key-features}

- **Temamedvetna färdigheter** — anpassar automatiskt vägledning och funktioner baserat på ditt aktiva WordPress-tema och installerade tillägg
- **Flera internetsökningsleverantörer** — välj mellan Brave Search och Tavily för rikare sökresultat och avancerade forskningsfunktioner
- **Kontaktformulär för webbplatsbyggare** — skapa fullt fungerande kontaktformulär direkt från chattgränssnittet med ditt installerade formulärtillägg
- **Förbättrad webbplatsbyggare** — förbättrade flerfasiga byggplaner för webbplatser med bättre tilläggsupptäckt och felåterställning
- **Inbyggda temafärdigheter** — specialiserad vägledning för blockteman, klassiska teman, Kadence Blocks och Kadence Theme
- **Förbättrad WooCommerce-integration** — omarbetad för att använda inbyggda WooCommerce API:er för bättre tillförlitlighet och kompatibilitet
- **Automatisk leverantörsidentifiering** — identifierar och uppdaterar automatiskt tillgängliga tillägg och teman när de aktiveras eller inaktiveras
- **Generering av anpassad logotyp** — generera och bädda in anpassade SVG-logotyper med automatisk namnområdessäker sanering
- **Validering av palettkontrast** — kontrollera färgpar för WCAG-tillgänglighetsefterlevnad innan du tillämpar dem på ditt tema
- **Fotoförbättrad upptäckt** — ladda upp foton under Theme Builder-upptäcktsintervjun för att påverka designriktningen
- **Menyer för hotell- och restaurangbranschen** — generera strukturerade sidor för mat- och dryckesmenyer för restauranger, kaféer och företag inom hotell- och restaurangbranschen
- **Förhandsvisningar för dator och mobil** — förhandsvisa din design på datorer och mobila enheter under valet av designriktning
- **Nivå 1-webbplatsskrapning** — sd-ai-agent/site-scrape är nu tillgängligt som standard för analys av konkurrenters och inspirationswebbplatser
- **Läsverktyg för Google Calendar** — läs konfigurerade kalendrar och händelser så att automatiseringar kan resonera kring kommande schemakontext
- **Mappning av kontakter och deltagare** — matcha kalenderdeltagare med kända webbplatsanvändare och kontakter innan uppföljningsarbete genereras
- **Godkännandesteg för människor** — pausa känsliga automatiseringar för granskning innan meddelanden skickas, arbete återupptas eller ändringar görs
- **Påminnelseposter och deduplicering** — spåra påminnelser som redan har skickats så att upprepade automatiseringskörningar inte meddelar samma mottagare två gånger
- **TextBee SMS-aviseringar** — skicka konfigurerade textmeddelandeaviseringar via TextBee när en SMS-leverantör är aktiverad
- **Provisionering av hanterad Superdav-tjänst** — anslut webbplatser som stöds till hostade Superdav-slutpunkter med automatisk installation i stället för manuell slutpunktskonfiguration
- **Advanced-kompanjonpaket** — separat distribuerade verktyg för betrodda utvecklare för filsystem, databas, WP-CLI, REST-dispatcher, tilläggsbyggare, git-snapshot, användarhantering och benchmark-arbetsflöden

## Krav {#requirements}

- WordPress 6.0 eller högre
- PHP 8.1 eller högre
- WordPress Multisite (nätverksaktiverad) eller installation med enskild webbplats
- Ett aktivt anslutningstillägg för AI-leverantör (Anthropic, OpenAI eller kompatibelt)
- Gratis AI Agent v1.9.0 eller högre (Superdav AI Agent utökar Gratis AI Agent)

## Installation {#installation}

1. Ladda upp tilläggsfilerna till `/wp-content/plugins/superdav-ai-agent`
2. Aktivera tillägget via skärmen **Tillägg** i WordPress
3. Säkerställ att Gratis AI Agent också är aktiverad
4. Gå till **Gratis AI Agent → Inställningar** och konfigurera din internetsökningsleverantör, anslutning till hanterad Superdav-tjänst, Google Calendar eller TextBee-inloggningsuppgifter efter behov

WordPress.org-paketet innehåller kärnfunktionerna i Superdav AI Agent. Advanced-kompanjonpaketet byggs och distribueras separat för miljöer där betrodda utvecklarverktyg uttryckligen installeras och auktoriseras.

## Temamedvetna färdigheter {#theme-aware-skills}

Superdav AI Agent identifierar automatiskt ditt aktiva tema och installerade tillägg, och läser in den lämpliga temamedvetna färdigheten i assistentens kontext. Tillgängliga färdigheter inkluderar:

| Färdighet | Gäller för |
|---|---|
| **Blockteman** | Teman som använder WordPress blockbaserade arkitektur (Full Site Editing) |
| **Klassiska teman** | Traditionella WordPress-teman som använder PHP-mallar och den klassiska redigeraren |
| **Kadence Blocks** | Webbplatser med tillägget Kadence Blocks aktivt |
| **Kadence Theme** | Webbplatser med Kadence-temat som aktivt tema |

Se [Temamedvetna färdigheter](../../user-guide/configuration/theme-aware-skills) för detaljerad vägledning om varje färdighet.

## Internetsökningsleverantörer {#internet-search-providers}

Superdav AI Agent v1.10.0 lägger till **Tavily** som ett alternativ till Brave Search för internetsökfrågor. Båda leverantörerna erbjuder sökresultat i realtid och kan konfigureras i **Gratis AI Agent → Inställningar → Avancerat**.

| Leverantör | Funktioner | Konfiguration |
|---|---|---|
| **Brave Search** | Snabba, integritetsfokuserade sökresultat | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Rikare sökresultat, avancerade forskningsfunktioner | [Tavily API](https://tavily.com/) |

Se [Internetsökning](../../user-guide/configuration/internet-search) för konfigurationsinstruktioner.

## Kalender-, godkännande- och notifieringsarbetsflöden {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 utökar automationsarbetsflöden bortom innehålls- och webbplatsbyggnadsuppgifter:

- **Google Calendar-verktyg** låter agenten läsa konfigurerade kalendrar och händelsedetaljer när kalenderinloggningsuppgifter är tillgängliga.
- **Kontakt- och deltagarhjälpare** mappar händelsedeltagare till WordPress-användare eller kända kontakter innan påminnelser eller uppföljningsuppgifter förbereds.
- **Mänskliga godkännandegrindar** låter en automation pausa tills en auktoriserad användare granskar den föreslagna åtgärden.
- **Påminnelseposter** förhindrar dubbla kalender- eller notifieringspåminnelser när en automation körs igen eller schemaläggs upprepade gånger.
- **TextBee SMS-stöd** möjliggör SMS-notifieringar när TextBee-leverantören är konfigurerad.

Använd dessa verktyg för schemamedveten assistans, uppföljning av deltagare och påminnelsearbetsflöden som fortfarande kräver mänsklig granskning innan känsliga meddelanden skickas.

## Konfiguration {#configuration}

Alla inställningar finns under **Gratis AI Agent → Settings** i WordPress-admin. Superdav AI Agent lägger till följande konfigurationsalternativ:

| Inställning | Beskrivning |
|---|---|
| **Internet Search Provider** | Välj mellan Brave Search och Tavily (valfritt) |
| **Tavily API Key** | API-nyckel för Tavily internetsökning (om Tavily är valt) |
| **Managed Superdav Service** | Anslut stödda webbplatser till den hostade Superdav-tjänsten med automatisk endpoint-tilldelning |
| **Google Calendar Credentials** | Konfigurera kalenderåtkomst så att agenten kan läsa kalendrar och händelser |
| **TextBee SMS Provider** | Konfigurera TextBee-inloggningsuppgifter för godkända SMS-notifieringsarbetsflöden |

## Avancerat companion-paket {#advanced-companion-package}

Superdav AI Agent v1.18.0 lägger till ett separat distribuerat Advanced companion-paket för betrodda utvecklararbetsflöden. Dessa funktioner ingår inte i WordPress.org-kärnpaketet. Installera companion-paketet endast på webbplatser där administratörer uttryckligen litar på att agenten får åtkomst till utvecklarverktyg.

Avancerade verktyg inkluderar filsystemoperationer, databasåtkomst, WP-CLI, REST-dispatcherarbetsflöden, verktyg för pluginbyggare, git-snapshots, användarhanteringshjälpare och benchmark-verktyg. Begränsa dessa verktyg till betrodda administratörer och granska konfigurationen av godkännandegrindar innan du aktiverar dem på produktionswebbplatser.
