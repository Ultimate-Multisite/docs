# Filter: networks_user_is_network_admin

Filtrerar de nätverk som en användare är administratör för.

## Parametrar

| Namn | Typ | Beskrivning |
|------|------|-------------|
| $ | `array\|bool` | Lista över nätverks-ID:n, eller `false` om användaren inte har några nätverk. |
| $ | `int` | Användar-ID för vilket nätverken ska returneras. |
