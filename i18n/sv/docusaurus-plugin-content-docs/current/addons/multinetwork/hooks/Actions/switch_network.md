# Action: switch_network

Körs när det aktuella nätverkskontextet byts.

## Parametrar

| Namn | Typ | Beskrivning |
|------|------|-------------|
| $new_network_id | `int` | ID:t på det nätverk som byts till. |
| $old_network_id | `int` | ID:t på det tidigare aktiva nätverket. |

- 1.3.0
