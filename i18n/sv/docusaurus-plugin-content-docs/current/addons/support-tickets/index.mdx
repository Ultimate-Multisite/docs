# Supportärenden

**Supportärenden** är ett kraftfullt tillägg för Ultimate Multisite som lägger till ett omfattande system för supportärenden i ditt WordPress-nätverk. Perfekt för SaaS-plattformar, WaaS-leverantörer (Website as a Service) och multisite-nätverk som behöver erbjuda kundsupport både på nätverksnivå och på enskilda webbplatser.

## Varför välja Supportärenden? {#why-choose-support-tickets}

* **Support på flera nivåer**: Nätverksadministratörer hanterar ärenden som gäller hela nätverket, medan webbplatsägare hanterar sin egen kundsupport.
* **Professionellt arbetsflöde**: Hantera hela ärendets livscykel med prioriteringar, statusar och tilldelningar.
* **Kundupplevelse**: Ett vackert och responsivt gränssnitt som gör det enkelt för kunder att få hjälp.
* **Säkert och pålitligt**: Byggt med WordPress:s bästa säkerhetspraxis och optimerat för databasprestanda.
* **Smidig integration**: Fungerar perfekt med Ultimate Multisites hantering av kunder och webbplatser.

## Huvudfunktioner {#key-features}

**Fullständig ärendehantering**

* Unikt ärendenummereringssystem för enkel referens
* Konfigurerbara prioritetssättningar (Låg, Normal, Hög, Kritisk)
* Omfattande statusarbetsflöde (Öppet, Pågående, Väntar, Löst, Stängt)
* Ärendeindelning (Allmänt, Tekniskt, Fakturering, Funktionförfrågan, Buggrapport)
* Tilldela ärenden till supportagenter med valfri automatisk tilldelning

**Trådade konversationer**

* Fullständig trådad konversation för svar på ärenden
* Distinkt styling för svar från personal och kunder
* Privata anteckningar för personal som inte syns för kunden
* Realtidsgränssnitt drivet av AJAX
* Modern textredigerare med formateringsalternativ

**Filbilagor**

* Säkra uppladdningar med stöd för drag-and-drop
* Konfigurerbara begränsningar för filtyp och storlek
* Avancerad säkerhet med upptäckt av skadliga filer
* Skyddade nedladdningar med åtkomstkontroller

**Smarta notifikationer**

* Automatiska e-postvarningar vid nya ärenden, svar och statusändringar
* Anpassningsbara e-postmallar
* Rollbaserade notifikationer för personal och kunder
* Dynamiska mallvariabler för personliga meddelanden

**Kraftfullt administrationsgränssnitt**

* Avancerad ärendelista med sortering och filtrering
* Snabbt redigeringsläge för status, prioritet och tilldelningar
* Massåtgärder för effektiv hantering
* Statistik-dashboard med prestandamått
* Omfattande inställningspanel

**Vackert kundportal**

* Rent, responsivt gränssnitt på alla enheter
* Designat med mobil först (mobile-first)
* Avancerad sökning och filtrering
* Modern redigerare med formateringsfält
* Enkla att använda formulär för ärendesändning

**Flexibel integration**

* Använd shortcodes eller Ultimate Multisite UI-element
* Visa ärendeformulär och listor var som helst på din webbplats
* Fungerar sömlöst med multisite-nätverksstrukturen
* Utbyggbart med hooks och filters

## Perfekt för {#perfect-for}

* SaaS-plattformar som behöver kundsupport
* WaaS-leverantörer som hanterar flera kundwebbplatser
* Medlemswebbplatser med nivåindelad support
* Nätverksadministratörer som hanterar flera varumärken
* Webbplatsägare som tillhandahåller kundservice

## Installation {#installation}

1. Se till att Ultimate Multisite-plugin är installerat och aktiverat på ditt WordPress-nätverk.
2. Ladda upp addon-filerna till katalogen `/wp-content/plugins/`.
3. Aktivera pluginet på nätverksnivå via menyn 'Plugins' i WordPress.
4. Navigera till Network Admin > Support Tickets för att konfigurera dina inställningar.

## Vanliga frågor {#frequently-asked-questions}

## Krävs Ultimate Multisite? {#does-this-require-ultimate-multisite}

Ja, detta tillägg kräver att Ultimate Multisite är installerat och aktivt. Det integreras djupt med Ultimate Multisites funktioner för hantering av kunder och webbplatser.

## Kan webbplatsägare hantera sin egen kundsupport? {#can-site-owners-handle-their-own-customer-support}

Absolut! Webbplatsägare kan använda ärendesystemet för att ge support till sina egna kunder, medan nätverksadministratörer kan hantera ärenden som gäller hela nätverket.

## Hur skickar kunder in ärenden? {#how-do-customers-submit-tickets}

Du kan lägga till formulär för ärendesändning var som helst på din webbplats med hjälp av shortcodes eller Ultimate Multisite UI-element. Kunderna fyller bara i formuläret och skickar in sin begäran.

## Kan jag anpassa ärendestatusar och prioriteringar? {#can-i-customize-the-ticket-statuses-and-priorities}

Ja! Du kan fullständigt anpassa prioritetssättningar, statusar och ärendetyper i nätverksadministratörens inställningar för att matcha ditt supportarbetsflöde.

## Stöder det filbilagor? {#does-it-support-file-attachments}

Ja, både kunder och personal kan bifoga filer till ärenden och svar. Du har full kontroll över tillåtna filtyper och storleksbegränsningar för säkerheten.

## Hur hanteras notifikationer? {#how-are-notifications-handled}

Systemet skickar automatiskt e-postmeddelanden vid nya ärenden, svar och statusändringar. Du kan anpassa alla e-postmallar i inställningarna.

## Kan jag tilldela ärenden till specifika supportagenter? {#can-i-assign-tickets-to-specific-support-agents}

Ja, du kan manuellt tilldela ärenden till agenter eller aktivera automatisk tilldelning för att fördela ärendena automatiskt.

## Finns det privata anteckningar för personal? {#are-there-private-notes-for-staff}

Ja, personalmedlemmar kan lägga till privata svar som endast är synliga för annan personal, inte för kunden.

## Är det mobilanpassat? {#is-it-mobile-friendly}

Absolut! Hela gränssnittet är byggt med en mobil-först-approach och fungerar vackert på alla enheter.

## Kan jag visa ärendelistor och formulär med shortcodes? {#can-i-display-ticket-lists-and-forms-using-shortcodes}

Ja! Tillgängliga shortcodes:
* `[wu_support_tickets]` - Visar kundens ärendelista
* `[wu_support_ticket_form]` - Visar formulär för nytt ärende
* `[wu_support_ticket_view ticket_id="123"]` - Visar ett specifikt ärende

## Vilka filtyper kan laddas upp? {#what-file-types-can-be-uploaded}

Du kan konfigurera tillåtna filtyper i inställningarna. Systemet inkluderar avancerad säkerhetsskanning för att skydda mot skadliga filer.

## Kan jag exportera ärendedata? {#can-i-export-ticket-data}

Ja, tillägget inkluderar exportfunktioner för ärendedata och statistik.
