# Moms

Multisite Ultimate: Europeisk moms

Lägger till extra funktioner och verktyg till Multisite Ultimate:s standardhantering av skatter för att bättre stödja europeisk moms.
Lägger automatiskt till EU:s momssatser på sidan för skattesatser från

## Installation {#installation}

1. Ladda upp 'ultimate-multisite-vat' till katalogen '/wp-content/plugins/'
2. Aktivera pluginet via menyn 'Plugins' i WordPress
3. Gå till Ultimate Multisite > Inställningar > Skatter och aktivera EU Moms.
4. Klicka på Skattesatser och uppdatera EU:s momssatser längst ner och Spara skattesatser.
5. Klart!
