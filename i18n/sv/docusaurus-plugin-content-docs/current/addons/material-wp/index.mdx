# Material WP

WordPress:s administrativa gränssnitt har varit oförändrat i evigheter. Men har du någonsin undrat hur ett Material Design-version av gränssnittet skulle se ut?

Material WP är ett admin-tema som utvecklats för att ge dig det utseendet och den känsla, vilket hjälper dig att ge dina klienter ett mer anpassat Dashboard.

## Features

- Stöd för anpassad CSS
- Enkla inställningar för styling
- Omordning av menyer
- Enkel import/export

## Documentation

**Anpassa ditt Admin-panel med Material WP**

## Installation

1. Ladda upp 'material-wp' till katalogen '/wp-content/plugins/'
2. Aktivera pluginet via menyn 'Plugins' i WordPress

## Usage Guide

WordPress:s administrativa gränssnitt har varit oförändrat i evigheter. Men har du någonsin undrat hur ett Material Design-version av gränssnittet skulle se ut?

Efter **installation**, gå till din _Plugins_-sida och klicka för att aktivera det på ditt Network.
