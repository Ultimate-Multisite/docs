# AffiliateWP {#affiliatewp}

Multisite Ultimate: AffiliateWP-integration

Använd det kraftfulla AffiliateWP för att växa kundbasen i ditt Ultimo Network!

## Installation {#installation}

1. Ladda upp 'ultimate-multisite-git ' till katalogen '/wp-content/plugins/'
2. Aktivera pluginet via menyn 'Plugins' i WordPress
3. Följ guiden steg för steg för att ställa in pluginet

## Usage Guide {#usage-guide}

[AffiliateWP](https://affiliatewp.com/) har blivit den främsta lösningen när man bygger ett affiliateprogram för din onlineverksamhet. Ultimate Multisite: AffiliateWP är bron som gör det möjligt för dig att bygga samma sak för ditt premium-nätverk.

## Getting Started {#getting-started}

Först och främst **måste du ha** [**AffiliateWP**](https://affiliatewp.com/) **aktiverat**! Du kommer inte att kunna använda vårt tillägg om du inte har **pluginet som integreras med det**. Om du inte planerar att låta dina kunder använda det på sina egna webbplatser, bör AffiliateWP vara aktiverat endast på din huvudsida.

Dessutom måste både **Ultimate Multisite** och **Ultimate Multisite: AffiliateWP** vara **aktiverade** **över hela nätverket**.

## Adding Ultimate Multisite as an AffiliateWP Integration {#adding-ultimate-multisite-as-an-affiliatewp-integration}

På din huvudsida, gå till Affiliates Settings och markera Ultimate Multisite under fliken Integrations. Detta instruerar [AffiliateWP](https://affiliatewp.com/) att börja spåra besök och konverteringar från Ultimate Multisite-registreringsflödet.

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_Markera rutan för Ultimate Multisite längst ner och spara_

Det var nästan allt som behövdes. Besök som kommer från hänvisningslänkar kommer att spåras, och när användaren senare gör sin första betalning skapas konverteringen och en provision sätts av till den affiliate som ansvarar för konverteringen.

Endast den första betalningen som tas emot från den nyligen skapade användaren kommer att spåras. För att aktivera återkommande hänvisningsbetalningar behöver du [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## Enabling Recurring Referrals {#enabling-recurring-referrals}

Efter att ha installerat [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/), gå till fliken Recurring Referrals i Affiliates Settings och aktivera användningen av återkommande hänvisningar.

<!-- Screenshot unavailable: AffiliateWP Recurring Referrals settings with Enable Recurring Referrals toggle -->

_Så enkelt som det går: Aktivera Recurring Referrals och du är klara!_

Igen, det var allt som krävdes!

Nu får den ansvariga affilieraten sin egen andel automatiskt varje gång en ny betalning tas emot på ett hänvisat konto!
