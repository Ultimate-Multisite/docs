# Loco Translate

Ultimate Multisite: Loco Translate Integration-tillägget ger dig sidspecifika översättningsmöjligheter för ditt WordPress multisite-nätverk. Om du någonsin har behövt att olika webbplatser i ditt nätverk ska ha olika översättningar av samma plugin eller tema, är detta tillägg den perfekta lösningen.

**Vad detta tillägg gör**

Detta tillägg integrerar Ultimate Multisite sömlöst med det populära Loco Translate-pluginet, vilket gör att varje subsite i ditt nätverk kan behålla sina egna anpassade översättningar. Oavsett om du driver en flermarknadsplattform, betjänar olika språkgrupper, eller bara behöver översättningsflexibilitet över hela ditt nätverk, gör detta tillägg det enkelt.

**Perfekt för:**

* Företag i flera regioner med olika lokaliseringstbehov
* Utbildningsinstitutioner som betjänar olika samhällen
* SaaS-plattformar som erbjuder flerspråkigt stöd
* Webbyråer som hanterar flera kundwebbplatser
* Ett multisite-nätverk som kräver översättningsflexibilitet

## Installation

1. Se till att Ultimate Multisite är installerat och att nätverket är aktiverat
2. Installera och aktivera Loco Translate-pluginet
3. Ladda upp detta tillägg till din plugins-mapp
4. Aktivera Loco Translate Integration-tillägget på nätverksnivå
5. Börja skapa sidspecifika översättningar via Loco Translate på vilken subsite som helst

Det var allt! Integrationen fungerar automatiskt utan någon ytterligare konfiguration.

## Vanliga frågor

## Behöver jag konfigurera något efter installationen?

Nej! Integrationen är automatisk. Så snart både tillägget och Loco Translate är aktiva kan subsites omedelbart börja skapa sina egna anpassade översättningar.

## Vad händer om en subsite inte skapar anpassade översättningar?

Webbplatsen kommer automatiskt att använda nätverksövergripande översättningar som fallback. Ingenting går sönder, och allt fungerar sömlöst.

## Kan subsites komma åt nätverksövergripande översättningar?

Vanliga site-administratörer ser och redigerar endast översättningarna för sin egen webbplats. Superadministratörer kan komma åt både nätverksövergripande och sitespecifika översättningar, beroende på vilken webbplats de tittar på.

## Fungerar detta med både plugins och teman?

Ja! Sitespecifika översättningar fungerar för både WordPress-plugins och teman.

## Kommer detta att sakta ner mitt multisite-nätverk?

Nej. Tillägget använder effektiv filhämtning som bara kontrollerar efter sitespecifika översättningar när det behövs. Det finns ingen märkbar prestandapåverkan.

## Kan jag kopiera nätverksöversättningar till en specifik webbplats som startpunkt?

Ja! När du redigerar översättningar i Loco Translate kan du enkelt börja med nätverksöversättningarna och anpassa dem för varje webbplats.

## Fungerar detta med WooCommerce och andra plugins?

Absolut! Ett plugin eller ett tema som använder standard WordPress-översättningsfunktioner kommer att fungera med sitespecifika översättningar.
