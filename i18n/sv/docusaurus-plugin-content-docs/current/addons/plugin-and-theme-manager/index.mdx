# Plugin & Theme Manager {#plugin--theme-manager}

Multisite Ultimate: Plugin and Theme Manager

Redigera eller dölja metadata (titel, författare, miniatyrbild och beskrivning) för installerade plugins och teman. Skapa kategorier för plugins och teman så att dina användare kan filtrera dem och lägga till en vacker, anpassad Plugins-sida i användarnas panel.

För att använda pluginet kan du bara navigera till dina sidor för Plugins eller Teman i din nätverksadmin och klicka på "Edit Plugin/Theme info".

## Installation {#installation}

1. Ladda upp 'ultimate-multisite-plugin-and-theme-manager' till katalogen '/wp-content/plugins/'
2. Aktivera pluginet via menyn 'Plugins' i din WordPress Network Admin

## Användningsguide {#usage-guide}

Ultimate Multisite: Plugin and Theme Manager är ett kraftfullt **premium** Ultimate Multisite-tillägg som ger dig full kontroll över hur dina nätverksanvändare ser och interagerar med plugin- och temaalternativen på ditt nätverk.

Detta tillägg tjänar tre syften:

  1. Det låter dig ändra metadata för både plugins och teman helt. Det inkluderar namn, beskrivning, författarnamn och till och med miniatyrbilder!

  2. Det låter dig kategorisera plugins och teman, vilket gör det enkelt för dina användare att sortera dem.

  3. Det ger dig möjlighet att ersätta den gamla, tråkiga plugins-sidan med en ny, temaliknande sida med sortering efter kategorier.

Efter att du har laddat ner och installerat det kommer en ny flik – _Add-on Settings_ – att visas på Ultimate Multisite Settings-sidan.
