# Ultimate AI Connector for Compatible Endpoints Ändringslogg

## Version 2.0.0 — Släpptes den 2026-04-24 {#version-200--released-on-2026-04-24}

* Nytt: Stöd för flera leverantörer – konfigurera flera AI-endpoints och dirigera förfrågningar med automatisk återhämtning mellan leverantörer.
* Fix: Multi-provider SDK-integration med korrekta leverantörs-ID, registrerings-URL:er och HTTP-filteromfång per leverantör.
* Fix: Den nya leverantörskortet expanderar nu automatiskt vid tillägg; script cache-brytning vid plugin-uppdatering.
* Fix: Dynamisk leverantörsklassnamnrymd för eval() och automatisk expansion för nyligen tillagda leverantörer.
* Fix: Använder stabila komponenter (inte längre experimentella) för Card/CardBody/CardHeader/CardDivider för kompatibilitet med WordPress 6.9+.
* Fix: Ersatte det odefinierade DragHandle med ett Unicode-greppikon för att dra och omordna leverantörer.
* Fix: Eliminerade den blockering av HTTP-förfrågan som utlöstes vid varje sidladdning.
* Förbättrat: GitHub Actions-arbetsflöden uppgraderade till Node.js 24.
