# Innehållssynkronisering

Multisite Ultimate: Content Sync

Detta tillägg för Multisite Ultimate gör det möjligt för nätverksadministratörer att synkronisera innehåll från vilken sida som helst till andra sidor i nätverket. Det stöder alla inläggstyper, inklusive inlägg, sidor, anpassade inläggstyper och Elementor-mallar. Tillägget inkluderar ett flerstegs-wizard-gränssnitt för att välja innehåll och målsidor, med batchbearbetning och realtidsåterkoppling om framsteg.

## Funktioner

* Innehållsval: Visuellt kortrutnät med miniatyrbilder och filtrering efter inläggstyp
* Alla inläggstyper: Synkroniserar inlägg, sidor, produkter, Elementor-mallar och alla anpassade inläggstyper
* Sidval: Välj vilka sidor innehållet ska synkroniseras till, med exkludering av källsidan och förvalda barnsidor
* Batchbearbetning: Synkroniserar till 10 sidor åt gången med uppdateringar av framsteg
* Mediasynk: Kopierar och ommappar automatiskt mediefiler (bilder, videor) från inläggsinnehållet till målsidorna
* Taxonomi-synk: Bevarar kategorier, taggar och anpassade taxinomier
* Elementor-stöd: Fullt stöd för Elementor-mallar med CSS-regenerering och ommappning av mall-ID
* Konfliktlösning: Alternativ att ersätta befintligt innehåll eller hoppa över det
* Cachehantering: Rensa automatiskt Elementor-cache på målsidorna efter synkronisering

## Krav

* WordPress 5.3 eller högre
* PHP 7.4 eller högre
* Multisite Ultimate plugin (aktivt)
* Elementor plugin (valfritt, krävs för synkronisering av Elementor-mallar)

## Installation

1. Ladda upp 'ultimate-multisite-content-sync' till katalogen '/wp-content/plugins/'
2. Aktivera pluginet via menyn 'Plugins' i din WordPress Network Admin
3. Konfigurera inställningarna under Ultimate Multisite → Settings → Content Sync
4. Gå till Ultimate Multisite → Content Sync för att starta synkroniseringen
