# Admin Page Creator

## Innehåll {#content}
Admin Page Creator låter dig enkelt lägga till nya administrationssidor och under-sidor i WordPress-administratörspanelen. Det kan användas för att skapa en välkomstsida eller en handledningssida som fungerar som en landningssida för nya användare. Du kan använda sidbyggare som Beaver Builder, Elementor, Brizy och Oxygen för att designa sidorna utan krångel.

Det erbjuder också stöd för anpassad HTML, CSS och JavaScript, samt åtkomstkontroll baserat på plan och användarroll!

## Funktioner {#features}
* Stöd för anpassad HTML och CSS
* JavaScript-stöd
* PHP-innehållsläge
* Behörigheter baserat på användarroll
* Stöd för sidbyggare

## Installation {#installation}

1. Ladda upp **ultimate-multisite-admin-page-creator** till katalogen **/wp-content/plugins/**
2. Aktivera pluginet via menyn **Plugins** i din WordPress Network Admin

## Guide för användning {#usage-guide}

Medan WordPress gör det väldigt enkelt att lägga till innehåll på sidorna (front end), kräver det att skapa sidor och under-sidor på administrationssidan (admin side) viss grundläggande kodkunskap. Ultimate Multisite: Admin Page Creator syftar till att föra över samma användarvänlighet som front end till back end av dina nätverkswebbplatser.

## Varför använda Admin Page Creator? {#why-use-admin-page-creator}

Kanske vill du erbjuda en anpassad sida på dina kunders administrationspanel med handledningar, eller så vill du lägga till en anpassad sida med FAQ och ett kontaktwidget. Möjligheterna är oändliga med Admin Page Creator, och det är vad vi ska utforska i dagens guide.

## Skapa din första anpassade administrationssidan {#creating-your-first-custom-admin-page}

Efter att du har installerat och aktiverat Ultimate Multisite: Admin Page Creator, gå till din nätverksadministratörsmeny och klicka på Admin Pages.

![Lista över anpassade administrationssidor i Network Admin](/img/admin/admin-pages-list.png)

_Lista över anpassade administrationssidor i Network Admin_

Du kommer att se en lista över alla de administrationssidor du hittills har skapat (eftersom det är ditt första besök kommer listan att vara tom). Klicka på **Add new Admin Page**. Du kommer att omdirigeras till redigeraren för administrationssidan.

I redigeraren för administrationssidan har du ett antal olika alternativ för att anpassa din sida.

Detta är titeln som visas högst upp på sidan, inte etiketten i administrationsmenyn.

## Menyalternativ {#menu-options}

Längre över, i meta-rutan för Allmänna alternativ (General Options), kan du konfigurera menytyp, etikett och mycket mer.

### Menutyper {#menu-types}

Din nya anpassade administrationssida kan ha två olika menutyper: överordnat menyalternativ och underordnat menyalternativ.

En överordnad meny sida, som namnet antyder, lägger till din nyskapade sida i administrationsmenyn tillsammans med de standard WordPress-sidorna.

Om du väljer överordnat alternativ kan du också välja menyordning och en menyikon.

Om du vill göra denna sida till ett underordnat menyalternativ kan du välja en föräldrasida som denna sida ska kopplas till. Du kan lägga till din anpassade sida till standard WordPress-sidor eller till dina överordnade anpassade administrationssidor.

<!-- Screenshot unavailable: Admin Page Creator menu type options with label, order, and icon settings -->

_Anpassa menyalternativet för denna anpassade sidan_

<!-- Screenshot unavailable: Admin Page Creator sub-menu parent page selection -->

_Du kan också lägga till denna sida som ett underordnat menyalternativ till andra menyalternativ, inklusive dina egna överordnade anpassade alternativ._

## Avancerade alternativ {#advanced-options}

Du bör kunna lägga till alla typer av innehåll i dina anpassade administrationssidor. Det inkluderar CSS-regler och filer, JavaScript och externa bibliotek. Du kan göra detta med fliken Avancerade alternativ (Advanced Options Tab).

### Anpassad CSS {#custom-css}

På fliken CSS kan du lägga till din egen anpassade CSS-kod, samt importera CSS-filer från externa platser (som CDN:er, Google Fonts, etc.).

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_Anpassad CSS stöds också. Du kan också inkludera externa stilfiler!_

### Anpassat JavaScript {#custom-javascript}

Samma gäller för JavaScript. Du kan lägga till externa bibliotek som ska användas i din anpassade JavaScript-kod.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Lägg till din egen JavaScript-kod!_

## Behörigheter {#permissions}

Även i Avancerade alternativ kan du välja vilka underwebbplatser som ska ha tillgång till denna sida. Detta är användbart om du planerar att visa olika exklusivt innehåll till olika planer, till exempel.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Du har total kontroll över var denna sida ska visas_

**Viktigt**: Båda villkoren måste vara uppfyllda för att sidan ska visas. Det innebär att om användaren är en kund på en given plan, men inte har någon av de valda rollerna (eller vice versa), kommer sidan inte att visas på deras panel.

## Aktiv {#active}

Om du vill inaktivera en anpassad administrationssida utan att ta bort den eller röra vid behörigheterna, bara avmarkera alternativet Aktiv (Active).

Att inaktivera en överordnad sida kommer automatiskt att inaktivera alla underordnade sidor som är kopplade till den föräldrasidan.
