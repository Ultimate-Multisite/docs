# Support Agents

Att hantera ett helt nätverk själv är ingen enkel uppgift. När ditt nätverk börjar växa kommer du att märka en ökande efterfrågan på support. Det kan vara en bra idé att anlita personer som kan hjälpa dig att lätta på arbetsbördan. Att ge full tillgång/behörighet är dock riskabelt och kan kompromettera ditt nätverk. Det är här Support Agent add-on kommer in.

Detta tillägg låter dig skapa en ny typ av WordPress-användare som kallas Support Agent, och ger dig total kontroll över vilka behörigheter den användaren har. Att ge behörighet är lika enkelt som att slå på en växlingsknapp, och du stänger av den om du bestämmer dig för att ta bort den.

## Funktioner {#features}

*   Detaljerad lista över behörigheter för WordPress och Ultimate Multisite
*   Aktivera och inaktivera åtkomst enkelt
*   Skapa ett obegränsat antal Agent-konton

## Installation {#installation}

1.  Ladda upp 'ultimate-multisite-support-agents' till katalogen '/wp-content/plugins/'
2.  Aktivera pluginet via menyn 'Plugins' i WordPress
3.  Följ guiden steg för steg för att ställa in pluginet
