# ಪಾಠ 7: ನಿಮ್ಮದೇ ಆಗಿ ಮಾಡಿಕೊಳ್ಳುವುದು

ನಿಮ್ಮ ಗ್ರಾಹಕರು ಎಂದಿಗೂ "ಯಾವ WordPress plugin" ಬಳಸುತ್ತಿದ್ದಾರೋ ಎಂಬ ಭಾವನೆ ಹೊಂದಬಾರದು. ಬದಲಿಗೆ, ಅವರು FitSite ಬಳಸುತ್ತಿದ್ದಾರೋ ಎಂಬ ಭಾವನೆ ಹೊಂದಬೇಕು—ಇದು ಅವರ ಉದ್ಯಮಕ್ಕಾಗಿ ನಿರ್ಮಿಸಲಾದ ಒಂದು ವೇದಿಕೆ. ಈ ಪಾಠದಲ್ಲಿ ಬ್ರ್ಯಾಂಡಿಂಗ್ (branding), ವೈಟ್-ಲೇಬಲಿಂಗ್ (white-labeling) ಮತ್ತು ಈ ವೇದಿಕೆಯನ್ನು ಒಂದು ಉತ್ಪನ್ನದಂತೆ (product) ಹೇಗೆ ಪ್ರಸ್ತುತಪಡಿಸುವುದು ಎಂಬುದನ್ನು ಕಲಿಯುತ್ತೇವೆ.

## ನಾವು ಎಲ್ಲಿ ನಿಲ್ಲಿಸಿದ್ದೆವು {#where-we-left-off}

FitSite ಗೆ ಒಂದು ಕಾರ್ಯನಿರ್ವಹಿಸುವ ಚೆಕ್‌ಔಟ್ ಫ್ಲೋ ಇದೆ. ಇದು ಫಿಟ್‌ನೆಸ್ ಸ್ಟುಡಿಯೋ ಮಾಲೀಕರನ್ನು ಯೋಜನೆ ಆಯ್ಕೆಯಿಂದ ನೇರವಾದ ಸೈಟ್‌ಗೆ ಕರೆದೊಯ್ಯುತ್ತದೆ. ಈಗ, ನಾವು ಸಂಪೂರ್ಣ ಅನುಭವವನ್ನು ಒಂದು ಸಮಗ್ರವಾದ, ಬ್ರ್ಯಾಂಡ್‌ ಹೊಂದಿರುವ ಉತ್ಪನ್ನದಂತೆ ಹೇಗೆ ಮಾಡುವುದು ಎಂಬುದನ್ನು ನೋಡೋಣ.

## ನಿಮ್ಮ ವೇದಿಕೆಯ ಡೊಮೇನ್ (Your Platform Domain) {#your-platform-domain}

ನಿಮ್ಮ ಬ್ರ್ಯಾಂಡ್‌ನ ಮೂಲಾಧಾರವೇ ನಿಮ್ಮ ಡೊಮೇನ್. FitSite ಗಾಗಿ:

- **ಪ್ರಮುಖ ಡೊಮೇನ್ (Main domain)**: `fitsite.com` (ನಿಮ್ಮ ಮಾರ್ಕೆಟಿಂಗ್ ಸೈಟ್ ಮತ್ತು ನೆಟ್‌ವರ್ಕ್ ಮೂಲ)
- **ಗ್ರಾಹಕರ ಸೈಟ್‌ಗಳು (Customer sites)**: `studioname.fitsite.com` (subdomains)
- **ಕಸ್ಟಮ್ ಡೊಮೇನ್‌ಗಳು (Custom domains)**: Growth ಮತ್ತು Pro ಯೋಜನೆಗಳಲ್ಲಿರುವ ಗ್ರಾಹಕರು ತಮ್ಮದೇ ಡೊಮೇನ್ ಅನ್ನು ಮ್ಯಾಪ್ ಮಾಡಬಹುದು.

### ನಿಮ್ಮ ಡೊಮೇನ್ ಅನ್ನು ಸೆಟಪ್ ಮಾಡುವುದು {#setting-up-your-domain}

1. ನಿಮ್ಮ ವೇದಿಕೆಯ ಡೊಮೇನ್ ಅನ್ನು ನೋಂದಾಯಿಸಿ
2. ಅದನ್ನು ನಿಮ್ಮ ಹೋಸ್ಟಿಂಗ್ ಪроваೈಡರ್‌ಗೆ ಕೊಂಡೊಯ್ಯಿರಿ (Point it)
3. ಗ್ರಾಹಕರ subdomains ಗಾಗಿ wildcard DNS (`*.fitsite.com`) ಅನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಿ
4. wildcard SSL ಸಕ್ರಿಯವಾಗಿದೆ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ

ವಿವರವಾದ ಸೂಚನೆಗಳಿಗಾಗಿ [How to Configure Domain Mapping](/user-guide/domain-mapping/how-to-configure-domain-mapping) ನೋಡಿ.

## ಅಡ್ಮಿನ್ ಅನುಭವಕ್ಕೆ ವೈಟ್-ಲೇಬಲಿಂಗ್ (White-Labeling the Admin Experience) {#white-labeling-the-admin-experience}

ಒಬ್ಬ ಫಿಟ್‌ನೆಸ್ ಸ್ಟುಡಿಯೋ ಮಾಲೀಕರು ತಮ್ಮ ಸೈಟ್ ಡ್ಯಾಶ್‌ಬೋರ್ಡ್‌ಗೆ ಲಾಗ್ ಇನ್ ಮಾಡಿದಾಗ, ಅವರಿಗೆ WordPress ಅಥವಾ Ultimate Multisite ಬ್ರ್ಯಾಂಡಿಂಗ್ ಬದಲು ನಿಮ್ಮ ಬ್ರ್ಯಾಂಡ್ ಕಾಣಬೇಕು.

### ಕಸ್ಟಮ್ ಲಾಗಿನ್ ಪುಟ (Custom Login Page) {#custom-login-page}

WordPress ಲಾಗಿನ್ ಪುಟವನ್ನು ಕಸ್ಟಮೈಸ್ ಮಾಡಿ, ಈ ಕೆಳಗಿನವುಗಳನ್ನು ತೋರಿಸಲು:

- ನಿಮ್ಮ FitSite ಲೋಗೋ
- ಫಿಟ್‌ನೆಸ್‌ಗೆ ಸೂಕ್ತವಾದ ಹಿನ್ನೆಲೆ ಚಿತ್ರಣ (Imagery)
- ನಿಮ್ಮ ಬ್ರ್ಯಾಂಡ್ ಬಣ್ಣಗಳು

### ಡ್ಯಾಶ್‌ಬೋರ್ಡ್ ಬ್ರ್ಯಾಂಡಿಂಗ್ (Dashboard Branding) {#dashboard-branding}

[Admin Page Creator](/addons/admin-page-creator) add-on ಅಥವಾ ಕಸ್ಟಮ್ CSS ಅನ್ನು ಬಳಸಿ:

- WordPress ಲೋಗೋವನ್ನು ನಿಮ್ಮ FitSite ಲೋಗೋದಿಂದ ಬದಲಾಯಿಸಿ
- ನಿಮ್ಮ ಬ್ರ್ಯಾಂಡ್‌ಗೆ ಹೊಂದಿಕೊಳ್ಳಲು ಅಡ್ಮಿನ್ ಬಣ್ಣದ ಸ್ಕೀಮ್ ಅನ್ನು ಕಸ್ಟಮೈಸ್ ಮಾಡಿ
- ಫಿಟ್‌ನೆಸ್‌ಗೆ ನಿರ್ದಿಷ್ಟವಾದ quick links ಮತ್ತು ಸಹಾಯ ಸಂಪನ್ಮೂಲಗಳೊಂದಿಗೆ ಕಸ್ಟಮ್ ಡ್ಯಾಶ್‌ಬೋರ್ಡ್ ವಿಜೆಟ್ ಅನ್ನು ಸೇರಿಸಿ

### ಕಸ್ಟಮ್ ಅಡ್ಮಿನ್ ಪುಟಗಳು (Custom Admin Pages) {#custom-admin-pages}

ಫಿಟ್‌ನೆಸ್ ಸ್ಟುಡಿಯೋ ಮಾಲೀಕರಿಗೆ ಅತ್ಯಂತ ಸಂಬಂಧಿತ ಕ್ರಿಯೆಗಳನ್ನು ಪ್ರದರ್ಶಿಸುವ ಕಸ್ಟಮ್ ಅಡ್ಮಿನ್ ಪುಟಗಳನ್ನು ರಚಿಸುವ ಬಗ್ಗೆ ಯೋಚಿಸಿ:

- "ನಿಮ್ಮ ಕ್ಲಾಸ್ ವೇಳಾಪಟ್ಟಿಯನ್ನು ಸಂಪಾದಿಸಿ" (Edit Your Class Schedule)
- "ಟ್ರೈನರ್ ಪ್ರೊಫೈಲ್‌ಗಳನ್ನು ಅಪ್ಡೇಟ್ ಮಾಡಿ" (Update Trainer Profiles)
- "ನಿಮ್ಮ ಸೈಟ್ ವೀಕ್ಷಿಸಿ" (View Your Site)

ಇದು standard WordPress ಮೆನುವಿನಲ್ಲಿ ಮುಳುಗಿಸುವ ಬದಲು, ಅಗತ್ಯವಾದ ಕ್ರಿಯೆಗಳನ್ನು ಮುಂಚೂಣಿಯಲ್ಲಿ ಇಡುವುದರಿಂದ ಕಲಿಕೆಯ ವಕ್ರರೇಖೆಯನ್ನು (learning curve) ಕಡಿಮೆ ಮಾಡುತ್ತದೆ.

## ನಿಮ್ಮ ಸಂವಹನಗಳಿಗೆ ಬ್ರ್ಯಾಂಡಿಂಗ್ ನೀಡುವುದು (Branding Your Communications) {#branding-your-communications}

ಪ್ರತಿಯೊಂದು ಇಮೇಲ್, ಇನ್‌ವಾಯ್ಸ್ ಮತ್ತು notification ಕೂಡ ನಿಮ್ಮ ಬ್ರ್ಯಾಂಡ್ ಅನ್ನು ಬಲಪಡಿಸಬೇಕು.

### ಸಿಸ್ಟಮ್ ಇಮೇಲ್‌ಗಳು (System Emails) {#system-emails}

**Ultimate Multisite > Settings > Emails** ಗೆ ಹೋಗಿ ಎಲ್ಲಾ ಸಿಸ್ಟಮ್ ಇಮೇಲ್‌ಗಳನ್ನು ಕಸ್ಟಮೈಸ್ ಮಾಡಿ:

- **ಇಂದ ಬರುವ ಹೆಸರು (From name)**: FitSite
- **ಇಂದ ಬರುವ ಇಮೇಲ್ (From email)**: hello@fitsite.com
- **ಇಮೇಲ್ ಟೆಂಪ್ಲೇಟ್‌ಗಳು (Email templates)**: ನಿಮ್ಮ ಬ್ರ್ಯಾಂಡ್ ಬಣ್ಣಗಳು ಮತ್ತು ಲೋಗೋ ಬಳಸಿ
- **ಭಾಷೆ (Language)**: ಫಿಟ್‌ನೆಸ್‌ಗೆ ನಿರ್ದಿಷ್ಟವಾದ ಭಾಷೆ ಇರಲಿ

ಕಸ್ಟಮೈಸ್ ಮಾಡಬೇಕಾದ ಪ್ರಮುಖ ಇಮೇಲ್‌ಗಳು:

| ಇಮೇಲ್ | ಸಾಮಾನ್ಯ ಆವೃತ್ತಿ (Generic Version) | FitSite ಆವೃತ್ತಿ (FitSite Version) |
|-------|-----------------------------------|-----------------------------------|
| ಸ್ವಾಗತ (Welcome) | "ನಿಮ್ಮ ಹೊಸ ಸೈಟ್ ಸಿದ್ಧವಾಗಿದೆ" | "ನಿಮ್ಮ ಫಿಟ್‌ನೆಸ್ ಸ್ಟುಡಿಯೋ ವೆಬ್‌ಸೈಟ್ ಲೈವ್ ಆಗಿದೆ" |
| ಪಾವತಿ ರಸೀದಿ (Payment receipt) | "ಪಾವತಿ ಸ್ವೀಕರಿಸಲಾಗಿದೆ" | "FitSite subscription payment ಖಚಿತಪಡಿಸಲಾಗಿದೆ" |
| ಟ್ರಯಲ್ ಮುಗಿಯುವುದು (Trial ending) | "ನಿಮ್ಮ ಟ್ರಯಲ್ ಶೀಘ್ರದಲ್ಲೇ ಮುಗಿಯುತ್ತಿದೆ" | "ನಿಮ್ಮ FitSite ಟ್ರಯಲ್ 3 ದಿನಗಳಲ್ಲಿ ಮುಗಿಯುತ್ತದೆ -- ನಿಮ್ಮ ಸ್ಟುಡಿಯೋ ಸೈಟ್ ಲೈವ್ ಆಗಿಡಲು ಉಳಿಸಿ" |

### ಇನ್‌ವಾಯ್ಸ್‌ಗಳು (Invoices) {#invoices}

ಈ ಕೆಳಗಿನವುಗಳೊಂದಿಗೆ ಇನ್‌ವಾಯ್ಸ್ ಟೆಂಪ್ಲೇಟ್‌ಗಳನ್ನು ಕಸ್ಟಮೈಸ್ ಮಾಡಿ:

- ನಿಮ್ಮ FitSite ಲೋಗೋ ಮತ್ತು ಬ್ರ್ಯಾಂಡ್ ಬಣ್ಣಗಳು
- ನಿಮ್ಮ ವ್ಯಾಪಾರ ವಿವರಗಳು
- ಫಿಟ್‌ನೆಸ್‌ಗೆ ನಿರ್ದಿಷ್ಟವಾದ ಉತ್ಪನ್ನದ ಹೆಸರುಗಳು (ಸಾಮಾನ್ಯ ಯೋಜನೆ ID ಗಳು ಅಲ್ಲ)

## ಗ್ರಾಹಕರಿಗೆ ಕಾಣುವ ಸೈಟ್ (The Customer-Facing Site) {#the-customer-facing-site}

ನಿಮ್ಮ ಪ್ರಮುಖ ಡೊಮೇನ್ (`fitsite.com`) ಒಂದು ಮಾರ್ಕೆಟಿಂಗ್ ಸೈಟ್ ಅನ್ನು ಹೊಂದಿರಬೇಕು, ಅದು ವೇದಿಕೆಯನ್ನು ಮಾರಾಟ ಮಾಡುತ್ತದೆ. ಇದು Ultimate Multisite ನೆಟ್‌ವರ್ಕ್ ಅಡ್ಮಿನ್‌ನಿಂದ ಪ್ರತ್ಯೇಕವಾಗಿದೆ—ಇದು ನಿಮ್ಮ ವ್ಯವಹಾರದ ಸಾರ್ವಜನಿಕ ಮುಖವಾಗಿದೆ.

ಪ್ರಮುಖ ಪುಟಗಳು:

- **ಹೋಮ್‌ಪೇಜ್ (Homepage)**: ಫಿಟ್‌ನೆಸ್ ವ್ಯವಹಾರಗಳಿಗೆ ಸ್ಪಷ್ಟ ಮೌಲ್ಯದ ಪ್ರತಿಪಾದನೆ
- **ఫీಚರ್‌ಗಳು (Features)**: FitSite ಏನು ಮಾಡುತ್ತದೆ, ಫಿಟ್‌ನೆಸ್ ಪದಗಳಲ್ಲಿ
- **ಬೆಲೆ (Pricing)**: ನಿಮ್ಮ ಮೂರು ಯೋಜನೆಗಳು, ಉದ್ಯಮಕ್ಕೆ ನಿರ್ದಿಷ್ಟವಾದ ವೈಶಿಷ್ಟ್ಯಗಳ ಹೋಲಿಕೆ
- **ಉದಾಹರಣೆಗಳು (Examples)**: ವೇದಿಕೆಯ ಮೇಲೆ ನಿರ್ಮಿಸಲಾದ ಸೈಟ್‌ಗಳನ್ನು ಪ್ರದರ್ಶಿಸಿ
- **ಸೈನ್ ಅಪ್ (Sign Up)**: ನಿಮ್ಮ ಚೆಕ್‌ಔಟ್ ಫಾರ್ಮ್‌ಗೆ ಲಿಂಕ್‌ಗಳು

:::tip ನಿಮ್ಮ ಮಾರ್ಕೆಟಿಂಗ್ ಸೈಟ್ ಒಂದು ನೆಟ್‌ವರ್ಕ್ ಸೈಟ್ ಆಗಿರಬಹುದು
ನಿಮ್ಮ ಮಾರ್ಕೆಟಿಂಗ್ ಸೈಟ್ ಅನ್ನು ನಿಮ್ಮದೇ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿರುವ ಒಂದು ಸೈಟ್ ಆಗಿ ರಚಿಸಿ. ಇದು ನಿಮಗೆ ಒಂದೇ ಡ್ಯಾಶ್‌ಬೋರ್ಡ್‌ನಿಂದ ಅದನ್ನು ನಿರ್ವಹಿಸಲು ಅನುಮತಿಸುತ್ತದೆ ಮತ್ತು ನಿಮ್ಮದೇ ವೇದಿಕೆಯ ಸಾಮರ್ಥ್ಯಗಳನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ.
:::

## ಗ್ರಾಹಕರಿಗಾಗಿ ಕಸ್ಟಮ್ ಡೊಮೇನ್ (Custom Domain for Customers) {#custom-domain-for-customers}

ಕಸ್ಟಮ್ ಡೊಮೇನ್‌ಗಳನ್ನು ಒಳಗೊಂಡಿರುವ ಯೋಜನೆಗಳಲ್ಲಿರುವ ಗ್ರಾಹಕರಿಗಾಗಿ, ಪ್ರಕ್ರಿಯೆಯನ್ನು ಸ್ಪಷ್ಟವಾಗಿ ದಾಖಲಿಸಿ:

1. ಗ್ರಾಹಕರು ತಮ್ಮ ಡೊಮೇನ್ ಅನ್ನು ನೋಂದಾಯಿಸುತ್ತಾರೆ ಅಥವಾ ಅದನ್ನು ಒಂದು ರಿಜಿಸ್ಟ್ರರ್‌ಗೆ ವರ್ಗಾಯಿಸುತ್ತಾರೆ
2. ಗ್ರಾಹಕರು DNS ಅನ್ನು ಅಪ್ಡೇಟ್ ಮಾಡಿ ನಿಮ್ಮ ವೇದಿಕೆಯ ಕಡೆಗೆ ಕೊಂಡೊಯ್ಯಲು (ಖಚಿತ ದಾಖಲೆಗಳನ್ನು ನೀಡಿ)
3. Ultimate Multisite ಡೊಮೇನ್ ಮ್ಯಾಪಿಂಗ್ ಮತ್ತು SSL ಅನ್ನು ನಿರ್ವಹಿಸುತ್ತದೆ

ಈ ಪ್ರಕ್ರಿಯೆಗಾಗಿ, ತಾಂತ್ರಿಕ ಜ್ಞಾನವಿಲ್ಲದ ಫಿಟ್‌ನೆಸ್ ಸ್ಟುಡಿಯೋ ಮಾಲೀಕರಿಗಾಗಿ ಬರೆದ ಸಹಾಯ ಲೇಖನ ಅಥವಾ knowledge base entry ಅನ್ನು ರಚಿಸಿ.

## ಇದುವರೆಗೆ FitSite ನೆಟ್‌ವರ್ಕ್ {#the-fitsite-network-so-far}

```
FitSite Network
├── WordPress Multisite (subdomain mode)
├── Ultimate Multisite (configured + branded)
├── Platform Domain (fitsite.com + wildcard SSL)
├── Site Templates (Studio Essential, Gym Pro, Fitness Chain)
├── Products (Starter, Growth, Pro + Order Bumps)
├── Checkout Flow (niche-specific, tested)
├── Branding
│   ├── Custom login page
│   ├── Branded admin dashboard
│   ├── Niche-specific system emails
│   ├── Branded invoices
│   └── Marketing site on fitsite.com
└── Ready for onboarding flow (next lesson)
```

## ಈ ಪಾಠದಲ್ಲಿ ನಾವು ಏನು ನಿರ್ಮಿಸಿದೆವು {#what-we-built-this-lesson}

- ಬ್ರ್ಯಾಂಡ್‌ ಹೊಂದಿರುವ ಅನುಭವಕ್ಕಾಗಿ **ವೇದಿಕೆ ಡೊಮೇನ್ ಮತ್ತು DNS** ಅನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಲಾಗಿದೆ
- FitSite ಬ್ರ್ಯಾಂಡಿಂಗ್‌ನೊಂದಿಗೆ **ವೈಟ್-ಲೇಬಲ್ಡ್ ಅಡ್ಮಿನ್**
- **ಕಸ್ಟಮೈಸ್ ಮಾಡಿದ ಸಂವಹನಗಳು** -- ಇಮೇಲ್‌ಗಳು, ಇನ್‌ವಾಯ್ಸ್‌ಗಳು ಮತ್ತು notifications ಎಲ್ಲವೂ ಬ್ರ್ಯಾಂಡ್‌ನಲ್ಲೇ ಇವೆ
- ಫಿಟ್‌ನೆಸ್ ಸ್ಟುಡಿಯೋ ಮಾಲೀಕರಿಗೆ FitSite ಅನ್ನು ಮಾರಾಟ ಮಾಡುವ **ಮಾರ್ಕೆಟಿಂಗ್ ಸೈಟ್**
- ತಮ್ಮದೇ ಡೊಮೇನ್ ಬೇಕಾದ ಗ್ರಾಹಕರಿಗಾಗಿ **ಕಸ್ಟಮ್ ಡೊಮೇನ್ ದಾಖಲಾತಿ**
