# PayFast

Multisite Ultimate: Payfast Gateway

ಇದು Payfast ಅನ್ನು ಸೇರಿಸುತ್ತದೆ. Payfast ಎಂದರೆ ದಕ್ಷಿಣ ಆಫ್ರಿಕಾದಲ್ಲಿರುವ ಒಂದು ವೇಗವಾದ ಮತ್ತು ಸುರಕ್ಷಿತ ಆನ್‌ಲೈನ್ ಪಾವತಿ 게ವೇ (gateway). ಇದನ್ನು Multisite Ultimate ಪಾವತಿ ಆಯ್ಕೆಯಾಗಿ ಬಳಸಬಹುದು.

## ಅಳವಡಿಕೆ (Installation) {#installation}

1. 'ultimate-multisite-payfast' ಅನ್ನು '/wp-content/plugins/' ಡೈರೆಕ್ಟರಿಗೆ ಅಪ್‌ಲೋಡ್ ಮಾಡಿ.
2. WordPress ನಲ್ಲಿರುವ 'Plugins' ಮೆನುವಿನ ಮೂಲಕ ಈ plugin ಅನ್ನು ಆಕ್ಟಿವೇಟ್ ಮಾಡಿ.
3. plugin ಅನ್ನು ಸೆಟ್ ಅಪ್ ಮಾಡಲು ಹಂತ ಹಂತದ Wizard ಅನ್ನು ಅನುಸರಿಸಿ.

## ಬಳಕೆಯ ಮಾರ್ಗದರ್ಶಿ (Usage Guide) {#usage-guide}

PayFast ಎಂದರೆ ದಕ್ಷಿಣ ಆಫ್ರಿಕಾದ ವ್ಯಾಪಾರ ಸಂಸ್ಥೆಗಳಿಗೆ ಪ್ರಮುಖ ಆನ್‌ಲೈನ್ ಪಾವತಿ ಪ್ರೊಸೆಸರ್ ಆಗಿದೆ. ಇದು ವಹಿವಾಟುಗಳನ್ನು ಸುಗಮ ಮತ್ತು ಸುರಕ್ಷಿತವಾಗಿಸುವ ಮೂಲಕ ಆನ್‌ಲೈನ್ ವ್ಯಾಪಾರಿಗಳು ಬೆಳೆಯಲು ಸಹಾಯ ಮಾಡುತ್ತದೆ. Ultimate Multisite ಬಳಕೆದಾರರ ವಿನಂತಿಯ ಮೇರೆಗೆ, PayFast ನೊಂದಿಗೆ ಇಂಟಿಗ್ರೇಷನ್ (integration) ಸಾಧ್ಯವಾಗಿಸುವ ಒಂದು add-on ಅನ್ನು ರಚಿಸಲು ನಾವು ನಿರ್ಧರಿಸಿದ್ದೇವೆ. ನಿಮ್ಮ Ultimate Multisite ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿ ಪಾವತಿ ಸ್ವೀಕರಿಸಲು PayFast ಅನ್ನು ನಿಮ್ಮ ಪಾವತಿ 게ವೇ (payment gateway) ಆಗಿ ಹೇಗೆ ಸೆಟ್ ಅಪ್ ಮಾಡುವುದು ಎಂದು ಈ ಲೇಖನದಲ್ಲಿ ತೋರಿಸಲಾಗಿದೆ.

ಮೊದಲ ಹಂತವೆಂದರೆ ಈ add-on ಅನ್ನು ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡುವುದು.

# Payfast add-on ಅನ್ನು ಹೇಗೆ ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡುವುದು:

add-ons ಪುಟಕ್ಕೆ ಹೋಗಿ ಮತ್ತು Ultimate Multisite: Payfast Gateway add-on ಅನ್ನು ಆಯ್ಕೆಮಾಡಿ.

![Add-ons page](/img/admin/addons-page.png)

add-on ವಿಂಡೋ ပေါ်လာುತ್ತದೆ. 'Install Now' ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ.

![Add-ons page with install option](/img/admin/addons-page.png)

ಇನ್‌ಸ್ಟಾಲ್ ಆದ ನಂತರ, ನೀವು plugins ಪುಟಕ್ಕೆ ಮರುನಿರ್ದೇಶನಗೊಳ್ಳುತ್ತೀರಿ. ಇಲ್ಲಿ, ಕೇವಲ 'Network Activate' ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು _Payfast_ add-on ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿ ಆಕ್ಟಿವೇಟ್ ಆಗುತ್ತದೆ.

# PayFast ಅನ್ನು ಸೆಟ್ ಅಪ್ ಮಾಡುವುದು

add-on ಅನ್ನು ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡಿದ ನಂತರ, ನೀವು Ultimate Multisite ಸೆಟ್ಟಿಂಗ್ಸ್‌ಗೆ ಹೋಗಿ PayFast ಅನ್ನು ಕಾನ್ಫಿಗರ್ (configure) ಮಾಡಲು ಪ್ರಾರಂಭಿಸಬಹುದು.

**Ultimate Multisite settings** ಪುಟದ ಅಡಿಯಲ್ಲಿ, ನಿಮಗೆ "Payments" ಟ್ಯಾಬ್ ಸಿಗುತ್ತದೆ. ಈ ಟ್ಯಾಬ್‌ನ ಅಡಿಯಲ್ಲಿ, ಈ ಪಾವತಿ 게ವೇ (payment gateway) ಅನ್ನು ಎನಬಲ್ (enable) ಅಥವಾ ಡಿಸೇಬಲ್ (disable) ಮಾಡಲು ಒಂದು ಟಾ글 ಬಟನ್‌ನೊಂದಿಗೆ "**Payfast** " ಇರಬೇಕು.

![Payment gateway settings](/img/config/settings-payment-gateways.png)ಮುಂದುವರಿದು, ಅದನ್ನು ಎನಬಲ್ ಮಾಡಲು ಟಾ글 ಬಟನ್ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ. ನಂತರ ಇದು ಬಾಕ್ಸ್ ಅನ್ನು ವಿಸ್ತರಿಸುತ್ತದೆ ಮತ್ತು ನೀವು ತುಂಬಬೇಕಾದ ವಿವಿಧ ಫೀಲ್ಡ್‌ಗಳನ್ನು ತೋರಿಸುತ್ತದೆ, ಉದಾಹರಣೆಗೆ **Mechant ID, Merchant Key ಮತ್ತು PassPhrase.** ಈ ಮಾಹಿತಿಯನ್ನು ನೀವು ನಿಮ್ಮ **Payfast** ಖಾತೆಯಿಂದ ಪಡೆಯಬಹುದು.

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

ಮುಂದಿನ ಹಂತವೆಂದರೆ ನಿಮಗೆ ಬೇಕಾದ ಮಾಹಿತಿಯನ್ನು ಪಡೆಯಲು ನಿಮ್ಮ PayFast ಖಾತೆಗೆ ಲಾಗಿನ್ ಮಾಡುವುದು.

Upon logging in, you can immediately see the credentials you need from your **Payfast** account dashboard.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->ನೀವು ಸರಳವಾಗಿ credentials ಅನ್ನು ಕಾಪಿ ಮತ್ತು ಪೇಸ್ಟ್ ಮಾಡಿ, ನಿಮ್ಮ Ultimate Multisite ಸೆಟ್ಟಿಂಗ್ಸ್‌ಗಳಲ್ಲಿ ವಿವರಗಳನ್ನು ತುಂಬಬಹುದು. ನಂತರ ಬದಲಾವಣೆಗಳನ್ನು ಉಳಿಸಲು **"Save settings** " ಬಟನ್ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

ಅಂತೆ, ನಿಮ್ಮ PayFast ಪಾವತಿ ಇಂಟಿಗ್ರೇಷನ್ (payment integration) ಪೂರ್ಣಗೊಂಡಿದೆ!
