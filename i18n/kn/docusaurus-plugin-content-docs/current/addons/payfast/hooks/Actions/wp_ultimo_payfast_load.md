# Action: wp_ultimo_payfast_load

ಎಲ್ಲಾ ಅವಲಂಬನೆಗಳು (dependencies) ಲೋಡ್ ಆದಾಗ ಇದು ಕಾರ್ಯಗತಗೊಳ್ಳುತ್ತದೆ.

ಪ್ಲಗಿನ್ developers-ಗಳು ಹೊಸ ಕಾರ್ಯಕ್ಷಮತೆಯನ್ನು (functionality) ಸೇರಿಸಲು ಇದನ್ನು ಬಳಸಬಹುದು. ಉದಾಹರಣೆಗೆ, ಹೊಸ ಹೋಸ್ಟಿಂಗ್ ಪроваೈಡರ್‌ಗಳಿಗೆ ಬೆಂಬಲ ನೀಡುವುದು ಇತ್ಯಾದಿ.
