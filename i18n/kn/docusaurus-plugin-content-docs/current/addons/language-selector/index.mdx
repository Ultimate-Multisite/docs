# ಭಾಷಾ ಆಯ್ಕೆಕಾರಕ (Language Selector) {#language-selector}

Multisite Ultimate: Sign-up ನಲ್ಲಿ ಭಾಷಾ ಆಯ್ಕೆಕಾರಕ

Sign-up ಪ್ರಕ್ರಿಯೆಗೆ (sign-up flow) ಭಾಷಾ ಆಯ್ಕೆಕಾರಕ ಫೀಲ್ಡ್ ಅನ್ನು ಸೇರಿಸಿ.

## ಅಳವಡಿಕೆ (Installation) {#installation}

1. 'ultimate-multisite-language-selector' ಅನ್ನು '/wp-content/plugins/' ಡೈರೆಕ್ಟರಿಗೆ ಅಪ್‌ಲೋಡ್ ಮಾಡಿ
2. ನಿಮ್ಮ WordPress Network Admin ನಲ್ಲಿರುವ 'Plugins' ಮೆನುವಿನ ಮೂಲಕ plugin ಅನ್ನು ಆಕ್ಟಿವೇಟ್ ಮಾಡಿ

## ಬಳಕೆಯ ಮಾರ್ಗದರ್ಶಿ (Usage Guide) {#usage-guide}

ಈ ಉಚಿತ add-on ಬಳಸಿ, ನಿಮ್ಮ ಬಳಕೆದಾರರು sign-up ಪ್ರಕ್ರಿಯೆಯಲ್ಲಿ ಮತ್ತು login ಪುಟದಲ್ಲಿ ಯಾವ ಭಾಷೆಯನ್ನು ನೋಡಲು ಬಯಸುತ್ತಾರೆ ಎಂಬುದನ್ನು ಆಯ್ಕೆ ಮಾಡಲು ನೀವು ಅವಕಾಶ ನೀಡಬಹುದು.

[ನಿಮ್ಮ ಲೈಸೆನ್스를 ಆಕ್ಟಿವೇಟ್ ಮಾಡಿದ ನಂತರ](../../user-guide/getting-started/installing-ultimate-multisite.md), ನಿಮ್ಮ ಲೈಸೆನ್ಸ್ ನಿಮಗೆ ಪ್ರವೇಶ ನೀಡುವ Ultimate Multisite add-ons ಅನ್ನು ನೀವು ಅಳವಡಿಸಿಕೊಳ್ಳಬಹುದು.

ನಿಮ್ಮ Ultimate Multisite ಸೆಟ್ಟಿಂಗ್ಸ್ ಪುಟಕ್ಕೆ ಹೋಗಿ ಮತ್ತು ನಂತರ, ಎಡಭಾಗದಲ್ಲಿರುವ ಮೆನುವಿನಲ್ಲಿ, _Add-ons_ ಲೇಬಲ್ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ.

![Add-ons page](/img/admin/addons-page.png)

_Language Selector ಅನ್ನು ಅಳವಡಿಸಲು Add-ons ಪುಟಕ್ಕೆ ಹೋಗಿ_

ಅಲ್ಲಿ, ನಾವು ನೀಡುವ ಎಲ್ಲಾ add-ons ಸಿಗುತ್ತವೆ.

ಯಾವ subscription plan ಗೆ ಅವರು ಸೇರಿದ್ದಾರೋ ಎಂಬುದನ್ನು ಲೆಕ್ಕಿಸದೆ, ಎಲ್ಲಾ add-ons ನಮ್ಮ ಎಲ್ಲಾ ಗ್ರಾಹಕರಿಗೆ ಲಭ್ಯವಿದೆ.

add-on ನ ಥಂಬ್‌ನೇಲ್ ಕೆಳಗೆ ಇರುವ _Install it Now!_ ಬಟನ್ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ. ನಂತರ Plugins ಪುಟದಲ್ಲಿ ಅದನ್ನು ಆಕ್ಟಿವೇಟ್ ಮಾಡಿ.

![Add-ons page with install button](/img/admin/addons-page.png)

_add-on ಅನ್ನು ಅಳವಡಿಸಲು ಬಟನ್ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_ನಿಮ್ಮ Plugins ಪುಟದಲ್ಲಿ ಅದನ್ನು ಆಕ್ಟಿವೇಟ್ ಮಾಡಿ_

ಈಗ, Ultimate Multisite ನ Settings ಮೆನುವಿಗೆ ಹೋಗಿ ಮತ್ತು _Network Settings_ ಟ್ಯಾಬ್ ಅನ್ನು ನಮೂದಿಸಿ. ಅಲ್ಲಿ, ಪುಟದ ಕೆಳಭಾಗದಲ್ಲಿ, ನೀವು _Languages_ ಎಂಬ ಹೊಸ ವಿಭಾಗವನ್ನು ಕಾಣುತ್ತೀರಿ.

![Settings page](/img/config/settings-general.png)

_ಒಂದು ಸಂಪೂರ್ಣ ಹೊಸ ವಿಭಾಗ ಕಾಣಿಸುತ್ತದೆ!_

ಎಲ್ಲಾ ಭಾಷೆಗಳನ್ನು ಆಯ್ಕೆಗಳಾಗಿ ತೋರಿಸಲು ನೀವು ಅದನ್ನು ಖಾಲಿ ಬಿಡಬಹುದು ಅಥವಾ ನೀವು ಲಭ್ಯವಾಗಿಸಲು ಬಯಸುವ ಭಾಷೆಗಳನ್ನು ಆಯ್ಕೆ ಮಾಡಬಹುದು. ಈ ಭಾಷೆಗಳು ನಿಮ್ಮ WordPress ಸೆಟಪ್‌ನಲ್ಲಿ ಅಳವಡಿಸಲಾದ ಭಾಷೆಗಳಾಗಿವೆ.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_ಈ WP ಇನ್‌ಸ್ಟಾಲ್‌ನಲ್ಲಿ ಲಭ್ಯವಿರುವ ಭಾಷೆಗಳ ಪಟ್ಟಿ_

ನೀವು ಮಾಡಿದ ಬದಲಾವಣೆಗಳನ್ನು ಉಳಿಸಿದ ನಂತರ, ನೀವು ಆರಿಸಿದ ಭಾಷೆಗಳೊಂದಿಗೆ ಒಂದು ಆಯ್ಕೆ ಮೆನು Login ಪುಟದಲ್ಲಿ ಮತ್ತು Sign-up ಪ್ರಕ್ರಿಯೆಯಲ್ಲಿ (Sign-up Flow) ಪ್ರದರ್ಶಿಸಲಾಗುತ್ತದೆ.

![Login and Registration settings](/img/config/settings-login-registration.png)

_Login ಪುಟದಲ್ಲಿ ಭಾಷಾ ಆಯ್ಕೆಕಾರಕ_

![Registration page](/img/config/registration-page.png)

_Sign-up ಪ್ರಕ್ರಿಯೆಯಲ್ಲಿ ಭಾಷಾ ಆಯ್ಕೆಕಾರಕ_

## URL ಮೂಲಕ ಭಾಷೆಯನ್ನು ಹೊಂದಿಸುವುದು {#setting-the-language-via-the-url}

ನೀವು URL ನಲ್ಲಿರುವ locale ಪ್ಯಾರಾಮೀಟರ್ ಅನ್ನು ಬೈಪಾಸ್ ಮಾಡುವ ಮೂಲಕವೂ ಒಂದು ಪುಟದ ಭಾಷೆಯನ್ನು ಬದಲಾಯಿಸಬಹುದು. ಉದಾಹರಣೆಗೆ, ನೀವು ಇಟಾಲಿಯನ್ ಭಾಷೆಯ ನೋಂದಣಿ ಪುಟಕ್ಕೆ (register page) ಮರುನಿರ್ದೇಶನ (redirect) ಮಾಡಲು ಬಯಸಿದರೆ, URL ಗೆ ?locale=it_IT ಸೇರಿಸಿ.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

ಇದು ಕೆಲಸ ಮಾಡಲು, ಆ ಭಾಷೆಯು ನಿಮ್ಮ WordPress ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿ ಅಳವಡಿಸಲ್ಪಟ್ಟಿರಬೇಕು.
