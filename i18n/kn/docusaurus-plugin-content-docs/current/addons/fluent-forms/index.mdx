# Fluent Forms

ಈ ಅಡ್-ಆನ್ (add-on) ಅನ್ನು ಬಳಸಿಕೊಂಡು, Multisite Ultimate ನಲ್ಲಿರುವ ಉತ್ಪನ್ನಗಳು (products), ಸದಸ್ಯತ್ವಗಳು (memberships) ಅಥವಾ ಸೈಟ್‌ಗಳಲ್ಲಿ Fluent Forms ಗೆ ಮಿತಿಗಳನ್ನು (limits) ಸೇರಿಸಲು ಸಾಧ್ಯವಾಗುತ್ತದೆ.
Fluent Forms ಕಸ್ಟಮ್ ಪೋಸ್ಟ್ ಟೈಪ್‌ಗಳ (custom post types) ಬದಲಿಗೆ ಕಸ್ಟಮ್ ಟೇಬಲ್‌ಗಳನ್ನು (custom tables) ಬಳಸುತ್ತದೆ. ಹಾಗಾಗಿ, ಸಾಮಾನ್ಯ ಮಿತಿ‌ಗಳ ಆಯ್ಕೆಗಳು (normal limiting options) Fluent Forms ಜೊತೆಗೆ ಕೆಲಸ ಮಾಡುವುದಿಲ್ಲ.

## Features {#features}

- Fluent Forms ಅಥವಾ Conversational Fluent Forms ಗೆ ಮಿತಿಗಳನ್ನು ಸೇರಿಸಿ
- ಬಳಕೆದಾರರು (Customers) ಮಿತಿಯನ್ನು ತಲುಪಿದಾಗ, ತಮ್ಮ ಯೋಜನೆಯನ್ನು (plan) ಅಪ್‌ಗ್ರೇಡ್ ಮಾಡಲು ಪ್ರೇರೇಪಿಸುವ ಸ್ನೇಹಪರ ಸಂದೇಶವನ್ನು ಪಡೆಯುತ್ತಾರೆ.
