# Action: wp_ultimo_plugin_and_theme_manager_load

ಎಲ್ಲಾ ಅವಲಂಬನೆಗಳು (dependencies) ಲೋಡ್ ಆದಾಗ ಇದು ಕಾರ್ಯಗತವಾಗುತ್ತದೆ.

ಪ್ಲಗ್ಇನ್ ಅಭಿವರ್ಧಕರು ಹೊಸ ಕಾರ್ಯಕ್ಷಮತೆಗಳನ್ನು (functionality) ಸೇರಿಸಲು ಇದು ಅನುಮತಿಸುತ್ತದೆ. ಉದಾಹರಣೆಗೆ, ಹೊಸ ಹೋಸ್ಟಿಂಗ್ (Hosting) ಪೂರೈಕೆದಾರರಿಗೆ ಬೆಂಬಲ (support) ಸೇರಿಸುವುದು ಇತ್ಯಾದಿ.

- 1.0.0
