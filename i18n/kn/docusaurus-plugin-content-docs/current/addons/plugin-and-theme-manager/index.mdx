# Plugin ಮತ್ತು Theme Manager

Multisite Ultimate: Plugin ಮತ್ತು Theme Manager

ಅಳವಡಿಸಲಾದ plugins ಮತ್ತು themes ನ meta information (ಶೀರ್ಷಿಕೆ, ಲೇಖಕ, ಥಂಬ್‌ನೇಲ್ ಮತ್ತು ವಿವರಣೆ) ಅನ್ನು ಸಂಪಾದಿಸಲು ಅಥವಾ ಮರೆಮಾಡಲು ಅನುವು ಮಾಡಿಕೊಡುತ್ತದೆ. ನಿಮ್ಮ ಬಳಕೆದಾರರು ಅವುಗಳನ್ನು ಫಿಲ್ಟರ್ ಮಾಡಲು plugins ಮತ್ತು themes ಗಾಗಿ ವರ್ಗಗಳನ್ನು (categories) ರಚಿಸಿ, ನಿಮ್ಮ ಬಳಕೆದಾರರ ಪ್ಯಾನೆಲ್‌ಗೆ ಸುಂದರವಾದ ಕಸ್ಟಮ್ Plugins ಪುಟವನ್ನು ಸೇರಿಸಬಹುದು.

ಈ plugin ಅನ್ನು ಬಳಸಲು, ನೀವು ನಿಮ್ಮ network admin ನಲ್ಲಿರುವ Plugins ಅಥವಾ Themes ಪುಟಗಳಿಗೆ ಹೋಗಿ "Edit Plugin/Theme info" ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಬಹುದು.

## Installation {#installation}

1. 'ultimate-multisite-plugin-and-theme-manager' ಅನ್ನು '/wp-content/plugins/' ಡೈರೆಕ್ಟರಿಗೆ ಅಪ್‌ಲೋಡ್ ಮಾಡಿ
2. ನಿಮ್ಮ WordPress Network Admin ನಲ್ಲಿರುವ 'Plugins' ಮೆನುವಿನ ಮೂಲಕ plugin ಅನ್ನು ಆಕ್ಟಿವೇಟ್ ಮಾಡಿ

## Usage Guide {#usage-guide}

Ultimate Multisite: Plugin and Theme Manager ಎಂಬುದು ಒಂದು ಶಕ್ತಿಶಾಲಿ **premium** Ultimate Multisite add-on ಆಗಿದೆ. ಇದು ನಿಮ್ಮ network ನಲ್ಲಿರುವ plugin ಮತ್ತು theme ಆಯ್ಕೆಗಳನ್ನು ನಿಮ್ಮ network ಬಳಕೆದಾರರು ಹೇಗೆ ನೋಡುತ್ತಾರೆ ಮತ್ತು ಅವುಗಳೊಂದಿಗೆ ಹೇಗೆ ಸಂವಹನ ನಡೆಸುತ್ತಾರೆ ಎಂಬುದರ ಮೇಲೆ ನಿಮಗೆ ಸಂಪೂರ್ಣ ನಿಯಂತ್ರಣ ನೀಡುತ್ತದೆ.

ಈ add-on ಮೂರು ಉದ್ದೇಶಗಳನ್ನು ಪೂರೈಸುತ್ತದೆ:

  1. ಇದು plugins ಮತ್ತು themes ಎರಡರ meta data ಅನ್ನು ಸಂಪೂರ್ಣವಾಗಿ ಬದಲಾಯಿಸಲು ನಿಮಗೆ ಅನುಮತಿಸುತ್ತದೆ. ಇದರಲ್ಲಿ ಹೆಸರು, ವಿವರಣೆ, ಲೇಖಕರ ಹೆಸರು ಮತ್ತು ಥಂಬ್‌ನೇಲ್‌ಗಳು ಕೂಡ ಸೇರಿವೆ!

  2. ಇದು plugins ಮತ್ತು themes ಅನ್ನು ವರ್ಗೀಕರಿಸಲು (categorize) ನಿಮಗೆ ಅನುಮತಿಸುತ್ತದೆ, ಇದರಿಂದ ನಿಮ್ಮ ಬಳಕೆದಾರರು ಅವುಗಳನ್ನು ಸುಲಭವಾಗಿ ವಿಂಗಡಿಸಬಹುದು.

  3. ಇದು ಹಳೆಯ, ನೀರಸವಾದ plugins ಪುಟವನ್ನು ವರ್ಗಗಳ ಮೂಲಕ ಸಾರ್ಟ್ ಮಾಡುವ (sorting) ಹೊಸ themes-ನಂತಹ ಪುಟದಿಂದ ಬದಲಾಯಿಸುವ ಆಯ್ಕೆಯನ್ನು ನೀಡುತ್ತದೆ.

ಇದನ್ನು ಡೌನ್‌ಲೋಡ್ ಮಾಡಿ ಮತ್ತು ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡಿದ ನಂತರ, Ultimate Multisite Settings ಪುಟದಲ್ಲಿ ಹೊಸ ಟ್ಯಾಬ್ - _Add-on Settings_ ಕಾಣಿಸುತ್ತದೆ.
