# Superdav AI Agent

Superdav AI Agent ಒಂದು ಉನ್ನತ WordPress AI ಸಹಾಯಕವಾಗಿದ್ದು, theme-aware ಕೌಶಲ್ಯಗಳು, ಸುಧಾರಿತ ಇಂಟರ್ನೆಟ್ ಹುಡುಕಾಟ ಸಾಮರ್ಥ್ಯಗಳು, ಸೈಟ್ ಬಿಲ್ಡರ್ ವೈಶಿಷ್ಟ್ಯಗಳು ಮತ್ತು ಐಚ್ಛಿಕ ಸ್ವಯಂಚಾಲನೆ ಸಂಯೋಜನೆಗಳೊಂದಿಗೆ Gratis AI Agent ಅನ್ನು ವಿಸ್ತರಿಸುತ್ತದೆ. ಆವೃತ್ತಿ 1.18.0 Google Calendar ಓದು ಪರಿಕರಗಳು, ಸಂಪರ್ಕ ಮತ್ತು ಪಾಲ್ಗೊಳ್ಳುವವರ ಮ್ಯಾಪಿಂಗ್, ಮಾನವ ಅನುಮೋದನಾ ಗೇಟ್‌ಗಳು, ಜ್ಞಾಪನೆ ಡಿಡ್ಯುಪ್ಲಿಕೇಶನ್, TextBee SMS ಸೂಚನೆಗಳು, ನಿರ್ವಹಿತ Superdav ಸೇವೆ ಒದಗಿಸುವಿಕೆ ಮತ್ತು ವಿಶ್ವಾಸಾರ್ಹ ಡೆವಲಪರ್ ಕಾರ್ಯಪ್ರವಾಹಗಳಿಗಾಗಿ ಪ್ರತ್ಯೇಕವಾಗಿ ವಿತರಿಸಲಾದ Advanced ಸಹಾಯಕ ಪ್ಯಾಕೇಜ್ ಅನ್ನು ಸೇರಿಸುತ್ತದೆ.

## ಪ್ರಮುಖ ವೈಶಿಷ್ಟ್ಯಗಳು {#key-features}

- **Theme-aware ಕೌಶಲ್ಯಗಳು** — ನಿಮ್ಮ ಸಕ್ರಿಯ WordPress ಥೀಮ್ ಮತ್ತು ಸ್ಥಾಪಿತ ಪ್ಲಗಿನ್‌ಗಳ ಆಧಾರದ ಮೇಲೆ ಮಾರ್ಗದರ್ಶನ ಮತ್ತು ಸಾಮರ್ಥ್ಯಗಳನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಹೊಂದಿಸುತ್ತದೆ
- **ಅನೇಕ ಇಂಟರ್ನೆಟ್ ಹುಡುಕಾಟ ಪೂರೈಕೆದಾರರು** — ಸಮೃದ್ಧ ಹುಡುಕಾಟ ಫಲಿತಾಂಶಗಳು ಮತ್ತು ಉನ್ನತ ಸಂಶೋಧನಾ ಸಾಮರ್ಥ್ಯಗಳಿಗಾಗಿ Brave Search ಮತ್ತು Tavily ನಡುವೆ ಆಯ್ಕೆಮಾಡಿ
- **ಸೈಟ್ ಬಿಲ್ಡರ್ ಸಂಪರ್ಕ ಫಾರ್ಮ್‌ಗಳು** — ನಿಮ್ಮ ಸ್ಥಾಪಿತ ಫಾರ್ಮ್ ಪ್ಲಗಿನ್ ಬಳಸಿ ಚಾಟ್ ಇಂಟರ್‌ಫೇಸ್‌ನಿಂದಲೇ ಪೂರ್ಣವಾಗಿ ಕಾರ್ಯನಿರ್ವಹಿಸುವ ಸಂಪರ್ಕ ಫಾರ್ಮ್‌ಗಳನ್ನು ರಚಿಸಿ
- **ಸುಧಾರಿತ ಸೈಟ್ ಬಿಲ್ಡರ್** — ಉತ್ತಮ ಪ್ಲಗಿನ್ ಪತ್ತೆ ಮತ್ತು ದೋಷ ಮರುಪಡೆಯುವಿಕೆಯೊಂದಿಗೆ ಸುಧಾರಿತ ಬಹು-ಹಂತದ ಸೈಟ್ ನಿರ್ಮಾಣ ಯೋಜನೆಗಳು
- **ಅಂತರ್ನಿರ್ಮಿತ ಥೀಮ್ ಕೌಶಲ್ಯಗಳು** — ಬ್ಲಾಕ್ ಥೀಮ್‌ಗಳು, ಕ್ಲಾಸಿಕ್ ಥೀಮ್‌ಗಳು, Kadence Blocks ಮತ್ತು Kadence Theme ಗಾಗಿ ವಿಶೇಷ ಮಾರ್ಗದರ್ಶನ
- **ಸುಧಾರಿತ WooCommerce ಸಂಯೋಜನೆ** — ಉತ್ತಮ ವಿಶ್ವಾಸಾರ್ಹತೆ ಮತ್ತು ಹೊಂದಾಣಿಕೆಗಾಗಿ ಸ್ಥಳೀಯ WooCommerce APIಗಳನ್ನು ಬಳಸುವಂತೆ ಮರುರಚಿಸಲಾಗಿದೆ
- **ಸ್ವಯಂಚಾಲಿತ ಪೂರೈಕೆದಾರ ಪತ್ತೆ** — ಪ್ಲಗಿನ್‌ಗಳು ಮತ್ತು ಥೀಮ್‌ಗಳನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿದಾಗ ಅಥವಾ ನಿಷ್ಕ್ರಿಯಗೊಳಿಸಿದಾಗ ಲಭ್ಯವಿರುವವುಗಳನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಪತ್ತೆಹಚ್ಚಿ ನವೀಕರಿಸುತ್ತದೆ
- **ಕಸ್ಟಮ್ ಲೋಗೋ ರಚನೆ** — ಸ್ವಯಂಚಾಲಿತ namespace-safe ಸ್ವಚ್ಛೀಕರಣದೊಂದಿಗೆ ಕಸ್ಟಮ್ SVG ಲೋಗೋಗಳನ್ನು ರಚಿಸಿ ಎಂಬೆಡ್ ಮಾಡಿ
- **ಪ್ಯಾಲೆಟ್ ಕಾಂಟ್ರಾಸ್ಟ್ ಪರಿಶೀಲನೆ** — ನಿಮ್ಮ ಥೀಮ್‌ಗೆ ಅನ್ವಯಿಸುವ ಮೊದಲು WCAG ಪ್ರವೇಶಸಾಧ್ಯತೆ ಅನುಸರಣೆಗೆ ಬಣ್ಣ ಜೋಡಿಗಳನ್ನು ಪರಿಶೀಲಿಸಿ
- **ಫೋಟೋ-ವರ್ಧಿತ ಅನ್ವೇಷಣೆ** — ವಿನ್ಯಾಸ ದಿಕ್ಕಿನ ಮೇಲೆ ಪ್ರಭಾವ ಬೀರುವಂತೆ Theme Builder ಅನ್ವೇಷಣಾ ಸಂದರ್ಶನದ ವೇಳೆ ಫೋಟೋಗಳನ್ನು ಅಪ್‌ಲೋಡ್ ಮಾಡಿ
- **ಆತಿಥ್ಯ ಮೆನುಗಳು** — ರೆಸ್ಟೋರೆಂಟ್‌ಗಳು, ಕಾಫೆಗಳು ಮತ್ತು ಆತಿಥ್ಯ ವ್ಯವಹಾರಗಳಿಗಾಗಿ ರಚನಾಬದ್ಧ ಆಹಾರ ಮತ್ತು ಪಾನೀಯ ಮೆನು ಪುಟಗಳನ್ನು ರಚಿಸಿ
- **ಡೆಸ್ಕ್‌ಟಾಪ್ ಮತ್ತು ಮೊಬೈಲ್ ಪೂರ್ವವೀಕ್ಷಣೆಗಳು** — ವಿನ್ಯಾಸ-ದಿಕ್ಕು ಆಯ್ಕೆ ಮಾಡುವಾಗ ಡೆಸ್ಕ್‌ಟಾಪ್ ಮತ್ತು ಮೊಬೈಲ್ ಸಾಧನಗಳಲ್ಲಿ ನಿಮ್ಮ ವಿನ್ಯಾಸವನ್ನು ಪೂರ್ವವೀಕ್ಷಿಸಿ
- **Tier 1 ಸೈಟ್ ಸ್ಕ್ರೇಪಿಂಗ್** — ಸ್ಪರ್ಧಿ ಮತ್ತು ಪ್ರೇರಣಾದಾಯಕ ವೆಬ್‌ಸೈಟ್‌ಗಳನ್ನು ವಿಶ್ಲೇಷಿಸಲು sd-ai-agent/site-scrape ಈಗ ಪೂರ್ವನಿಯೋಜಿತವಾಗಿ ಲಭ್ಯವಿದೆ
- **Google Calendar ಓದು ಪರಿಕರಗಳು** — ಮುಂಬರುವ ವೇಳಾಪಟ್ಟಿ ಸಂದರ್ಭವನ್ನು ಸ್ವಯಂಚಾಲನೆಗಳು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು ಸಂರಚಿಸಲಾದ ಕ್ಯಾಲೆಂಡರ್‌ಗಳು ಮತ್ತು ಈವೆಂಟ್‌ಗಳನ್ನು ಓದಿ
- **ಸಂಪರ್ಕ ಮತ್ತು ಪಾಲ್ಗೊಳ್ಳುವವರ ಮ್ಯಾಪಿಂಗ್** — ಮುಂದಿನ ಕೆಲಸವನ್ನು ರಚಿಸುವ ಮೊದಲು ಕ್ಯಾಲೆಂಡರ್ ಪಾಲ್ಗೊಳ್ಳುವವರನ್ನು ತಿಳಿದಿರುವ ಸೈಟ್ ಬಳಕೆದಾರರು ಮತ್ತು ಸಂಪರ್ಕಗಳಿಗೆ ಹೊಂದಿಸಿ
- **ಮಾನವ ಅನುಮೋದನಾ ಗೇಟ್‌ಗಳು** — ಸಂದೇಶಗಳನ್ನು ಕಳುಹಿಸುವುದು, ಕೆಲಸವನ್ನು ಮುಂದುವರಿಸುವುದು ಅಥವಾ ಬದಲಾವಣೆಗಳನ್ನು ಮಾಡುವ ಮೊದಲು ಪರಿಶೀಲನೆಗಾಗಿ ಸೂಕ್ಷ್ಮ ಸ್ವಯಂಚಾಲನೆಗಳನ್ನು ವಿರಾಮಗೊಳಿಸಿ
- **ಜ್ಞಾಪನೆ ದಾಖಲೆಗಳು ಮತ್ತು ಡಿಡ್ಯುಪ್ಲಿಕೇಶನ್** — ಈಗಾಗಲೇ ಕಳುಹಿಸಲಾದ ಜ್ಞಾಪನೆಗಳನ್ನು ಟ್ರ್ಯಾಕ್ ಮಾಡಿ, ಮರುಕಳಿಸುವ ಸ್ವಯಂಚಾಲನೆ ಚಾಲನೆಗಳು ಅದೇ ಸ್ವೀಕರಿಸುವವರಿಗೆ ಎರಡನೇ ಬಾರಿ ಸೂಚನೆ ಕಳುಹಿಸದಂತೆ ನೋಡಿಕೊಳ್ಳಿ
- **TextBee SMS ಸೂಚನೆಗಳು** — SMS ಪೂರೈಕೆದಾರ ಸಕ್ರಿಯವಾಗಿರುವಾಗ TextBee ಮೂಲಕ ಸಂರಚಿಸಲಾದ ಪಠ್ಯ-ಸಂದೇಶ ಸೂಚನೆಗಳನ್ನು ಕಳುಹಿಸಿ
- **ನಿರ್ವಹಿತ Superdav ಸೇವೆ ಒದಗಿಸುವಿಕೆ** — ಕೈಯಾರೆ ಎಂಡ್‌ಪಾಯಿಂಟ್ ಸಂರಚನೆಯ ಬದಲು ಸ್ವಯಂಚಾಲಿತ ಸೆಟಪ್‌ನೊಂದಿಗೆ ಬೆಂಬಲಿತ ಸೈಟ್‌ಗಳನ್ನು ಹೋಸ್ಟ್ ಮಾಡಲಾದ Superdav ಎಂಡ್‌ಪಾಯಿಂಟ್‌ಗಳಿಗೆ ಸಂಪರ್ಕಿಸಿ
- **Advanced ಸಹಾಯಕ ಪ್ಯಾಕೇಜ್** — ಫೈಲ್‌ಸಿಸ್ಟಮ್, ಡೇಟಾಬೇಸ್, WP-CLI, REST dispatcher, ಪ್ಲಗಿನ್ ಬಿಲ್ಡರ್, git snapshot, ಬಳಕೆದಾರ-ನಿರ್ವಹಣೆ ಮತ್ತು ಬೆಂಚ್‌ಮಾರ್ಕ್ ಕಾರ್ಯಪ್ರವಾಹಗಳಿಗಾಗಿ ಪ್ರತ್ಯೇಕವಾಗಿ ವಿತರಿಸಲಾದ ವಿಶ್ವಾಸಾರ್ಹ-ಡೆವಲಪರ್ ಪರಿಕರಗಳು

## ಅವಶ್ಯಕತೆಗಳು {#requirements}

- WordPress 6.0 ಅಥವಾ ಹೆಚ್ಚಿನದು
- PHP 8.1 ಅಥವಾ ಹೆಚ್ಚಿನದು
- WordPress Multisite (ನೆಟ್‌ವರ್ಕ್-ಸಕ್ರಿಯಗೊಳಿಸಿದ) ಅಥವಾ single-site ಸ್ಥಾಪನೆ
- ಸಕ್ರಿಯ AI ಪೂರೈಕೆದಾರ ಕನೆಕ್ಟರ್ ಪ್ಲಗಿನ್ (Anthropic, OpenAI ಅಥವಾ ಹೊಂದಾಣಿಕೆಯದು)
- Gratis AI Agent v1.9.0 ಅಥವಾ ಹೆಚ್ಚಿನದು (Superdav AI Agent, Gratis AI Agent ಅನ್ನು ವಿಸ್ತರಿಸುತ್ತದೆ)

## ಸ್ಥಾಪನೆ {#installation}

1. ಪ್ಲಗಿನ್ ಫೈಲ್‌ಗಳನ್ನು `/wp-content/plugins/superdav-ai-agent` ಗೆ ಅಪ್‌ಲೋಡ್ ಮಾಡಿ
2. WordPress ನಲ್ಲಿ **ಪ್ಲಗಿನ್‌ಗಳು** ಪರದೆಯ ಮೂಲಕ ಪ್ಲಗಿನ್ ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ
3. Gratis AI Agent ಕೂಡ ಸಕ್ರಿಯವಾಗಿದೆ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ
4. **Gratis AI Agent → Settings** ಗೆ ಹೋಗಿ, ಅಗತ್ಯವಿದ್ದಂತೆ ನಿಮ್ಮ ಇಂಟರ್ನೆಟ್ ಹುಡುಕಾಟ ಪೂರೈಕೆದಾರ, ನಿರ್ವಹಿತ Superdav ಸೇವೆ ಸಂಪರ್ಕ, Google Calendar ಅಥವಾ TextBee ರುಜುವಾತುಗಳನ್ನು ಸಂರಚಿಸಿ

WordPress.org ಪ್ಯಾಕೇಜ್‌ನಲ್ಲಿ ಮುಖ್ಯ Superdav AI Agent ವೈಶಿಷ್ಟ್ಯಗಳು ಸೇರಿವೆ. ವಿಶ್ವಾಸಾರ್ಹ ಡೆವಲಪರ್ ಪರಿಕರಗಳನ್ನು ಸ್ಪಷ್ಟವಾಗಿ ಸ್ಥಾಪಿಸಿ ಅನುಮೋದಿಸಿದ ಪರಿಸರಗಳಿಗಾಗಿ Advanced ಸಹಾಯಕ ಪ್ಯಾಕೇಜ್ ಅನ್ನು ಪ್ರತ್ಯೇಕವಾಗಿ ನಿರ್ಮಿಸಿ ವಿತರಿಸಲಾಗುತ್ತದೆ.

## ತ್ವರಿತ ಆರಂಭ {#quick-start}

ಸಕ್ರಿಯಗೊಳಿಸಿದ ನಂತರ, Superdav AI Agent ನಿಮ್ಮ Gratis AI Agent ಸ್ಥಾಪನೆಯನ್ನು theme-aware ಕೌಶಲ್ಯಗಳು ಮತ್ತು ಹೆಚ್ಚುವರಿ ಸಾಮರ್ಥ್ಯಗಳೊಂದಿಗೆ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸುಧಾರಿಸುತ್ತದೆ. WordPress ನಿರ್ವಹಣಾ ಸೈಡ್‌ಬಾರ್‌ನಲ್ಲಿ **Gratis AI Agent** ಪ್ಯಾನೆಲ್ ತೆರೆಯಿರಿ ಮತ್ತು ಹೊಸ ವೈಶಿಷ್ಟ್ಯಗಳನ್ನು ಬಳಸಲು ಪ್ರಾರಂಭಿಸಿ.

**ಉದಾಹರಣೆ ಸೂಚನೆಗಳು:**

```
ಸಂಪರ್ಕ ಪುಟಕ್ಕೆ ಸಂಪರ್ಕ ಫಾರ್ಮ್ ಸೇರಿಸಿ.
```

```
ಇತ್ತೀಚಿನ WordPress ಭದ್ರತಾ ಉತ್ತಮ ಕ್ರಮಗಳಿಗಾಗಿ ವೆಬ್‌ನಲ್ಲಿ ಹುಡುಕಿ.
```

```
ಸಕ್ರಿಯ ಥೀಮ್‌ನ ಬ್ಲಾಕ್ ಪ್ಯಾಟರ್ನ್‌ಗಳನ್ನು ಬಳಸಿ hero ವಿಭಾಗವನ್ನು ರಚಿಸಿ.
```

```
ನಾಳೆಯ ಸಂರಚಿಸಲಾದ Google Calendar ಈವೆಂಟ್‌ಗಳನ್ನು ಪರಿಶೀಲಿಸಿ ಮತ್ತು ಪಾಲ್ಗೊಳ್ಳುವವರಿಗಾಗಿ ಜ್ಞಾಪನೆ ಕರಡುಗಳನ್ನು ಸಿದ್ಧಪಡಿಸಿ.
```

```
ನಾನು ಸಂದೇಶವನ್ನು ಅನುಮೋದಿಸಿದ ನಂತರ SMS ಜ್ಞಾಪನೆಯನ್ನು ಕಳುಹಿಸಿ.
```

## Theme-Aware ಕೌಶಲ್ಯಗಳು {#theme-aware-skills}

Superdav AI Agent ನಿಮ್ಮ ಸಕ್ರಿಯ ಥೀಮ್ ಮತ್ತು ಸ್ಥಾಪಿತ ಪ್ಲಗಿನ್‌ಗಳನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಪತ್ತೆಹಚ್ಚಿ, ಸೂಕ್ತ theme-aware ಕೌಶಲ್ಯವನ್ನು ಸಹಾಯಕನ ಸಂದರ್ಭಕ್ಕೆ ಲೋಡ್ ಮಾಡುತ್ತದೆ. ಲಭ್ಯವಿರುವ ಕೌಶಲ್ಯಗಳಲ್ಲಿ ಇವು ಸೇರಿವೆ:

| ಕೌಶಲ್ಯ | ಅನ್ವಯಿಸುವುದು |
|---|---|
| **ಬ್ಲಾಕ್ ಥೀಮ್‌ಗಳು** | WordPress ಬ್ಲಾಕ್-ಆಧಾರಿತ (Full Site Editing) ಆರ್ಕಿಟೆಕ್ಚರ್ ಬಳಸುವ ಥೀಮ್‌ಗಳು |

| **Kadence Theme** | ಸಕ್ರಿಯ ಥೀಮ್ ಆಗಿ Kadence ಥೀಮ್ ಹೊಂದಿರುವ ಸೈಟ್‌ಗಳು |

ಪ್ರತಿ ಕೌಶಲ್ಯದ ವಿವರವಾದ ಮಾರ್ಗದರ್ಶನಕ್ಕಾಗಿ [Theme-Aware ಕೌಶಲ್ಯಗಳು](../../user-guide/configuration/theme-aware-skills) ನೋಡಿ.

## ಇಂಟರ್ನೆಟ್ ಹುಡುಕಾಟ ಪೂರೈಕೆದಾರರು {#internet-search-providers}

Superdav AI Agent v1.10.0 ಇಂಟರ್ನೆಟ್ ಹುಡುಕಾಟ ಪ್ರಶ್ನೆಗಳಿಗಾಗಿ Brave Search ಗೆ ಪರ್ಯಾಯವಾಗಿ **Tavily** ಅನ್ನು ಸೇರಿಸುತ್ತದೆ. ಎರಡೂ ಪೂರೈಕೆದಾರರು ರಿಯಲ್-ಟೈಮ್ ಹುಡುಕಾಟ ಫಲಿತಾಂಶಗಳನ್ನು ಒದಗಿಸುತ್ತಾರೆ ಮತ್ತು **Gratis AI Agent → Settings → Advanced** ನಲ್ಲಿ ಸಂರಚಿಸಬಹುದು.

| ಒದಗಿಸುವವರು | ವೈಶಿಷ್ಟ್ಯಗಳು | ಸೆಟಪ್ |
|---|---|---|
| **Brave Search** | ವೇಗವಾದ, ಗೌಪ್ಯತೆ-ಕೇಂದ್ರಿತ ಹುಡುಕಾಟ ಫಲಿತಾಂಶಗಳು | [Brave Search API](https://brave.com/search/api/) |

ಸಂರಚನಾ ಸೂಚನೆಗಳಿಗಾಗಿ [ಇಂಟರ್ನೆಟ್ ಹುಡುಕಾಟ](../../user-guide/configuration/internet-search) ನೋಡಿ.

## ಕ್ಯಾಲೆಂಡರ್, ಅನುಮೋದನೆ, ಮತ್ತು ಅಧಿಸೂಚನಾ ಕಾರ್ಯಪ್ರವಾಹಗಳು {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 ವಿಷಯ ಮತ್ತು site-ನಿರ್ಮಾಣ ಕಾರ್ಯಗಳಾಚೆಗೆ ಸ್ವಯಂಚಾಲಿತ ಕಾರ್ಯಪ್ರವಾಹಗಳನ್ನು ವಿಸ್ತರಿಸುತ್ತದೆ:

- **Google Calendar ಸಾಧನಗಳು** ಕ್ಯಾಲೆಂಡರ್ credentials ಲಭ್ಯವಿರುವಾಗ agent ಸಂರಚಿಸಲಾದ ಕ್ಯಾಲೆಂಡರ್‌ಗಳು ಮತ್ತು ಈವೆಂಟ್ ವಿವರಗಳನ್ನು ಓದಲು ಅವಕಾಶ ನೀಡುತ್ತವೆ.
- **ಸಂಪರ್ಕ ಮತ್ತು ಭಾಗವಹಿಸುವವರ ಸಹಾಯಕಗಳು** ಸ್ಮರಣಿಕೆಗಳು ಅಥವಾ ಮುಂದಿನ ಕ್ರಮಗಳನ್ನು ಸಿದ್ಧಪಡಿಸುವ ಮೊದಲು ಈವೆಂಟ್ ಭಾಗವಹಿಸುವವರನ್ನು WordPress ಬಳಕೆದಾರರು ಅಥವಾ ತಿಳಿದಿರುವ ಸಂಪರ್ಕಗಳಿಗೆ ಜೋಡಿಸುತ್ತವೆ.
- **ಮಾನವ ಅನುಮೋದನಾ ದ್ವಾರಗಳು** ಪ್ರಸ್ತಾವಿತ ಕ್ರಮವನ್ನು ಅಧಿಕೃತ ಬಳಕೆದಾರನು ಪರಿಶೀಲಿಸುವವರೆಗೆ automation ಅನ್ನು ವಿರಾಮಗೊಳಿಸಲು ಅವಕಾಶ ನೀಡುತ್ತವೆ.
- **ಸ್ಮರಣಿಕೆ ದಾಖಲೆಗಳು** automation ಅನ್ನು ಮರುಪ್ರಯತ್ನಿಸಿದಾಗ ಅಥವಾ ಮರುಮರು ನಿಗದಿಪಡಿಸಿದಾಗ ನಕಲಿ ಕ್ಯಾಲೆಂಡರ್ ಅಥವಾ ಅಧಿಸೂಚನಾ ಸ್ಮರಣಿಕೆಗಳನ್ನು ತಡೆಯುತ್ತವೆ.
- **TextBee SMS ಬೆಂಬಲ** TextBee provider ಸಂರಚಿಸಲಾದಾಗ ಪಠ್ಯ-ಸಂದೇಶ ಅಧಿಸೂಚನೆಗಳನ್ನು ಸಕ್ರಿಯಗೊಳಿಸುತ್ತದೆ.

ಸೂಕ್ಷ್ಮ ಸಂದೇಶಗಳನ್ನು ಕಳುಹಿಸುವ ಮೊದಲು ಇನ್ನೂ ಮಾನವ ಪರಿಶೀಲನೆ ಅಗತ್ಯವಿರುವ ವೇಳಾಪಟ್ಟಿ-ಜಾಗೃತ ಸಹಾಯ, ಭಾಗವಹಿಸುವವರ ಮುಂದಿನ ಕ್ರಮ, ಮತ್ತು ಸ್ಮರಣಿಕೆ ಕಾರ್ಯಪ್ರವಾಹಗಳಿಗೆ ಈ ಸಾಧನಗಳನ್ನು ಬಳಸಿ.

## Site Builder ಸಂಪರ್ಕ ಫಾರ್ಮ್‌ಗಳು {#site-builder-contact-forms}

## Theme Builder ವೈಶಿಷ್ಟ್ಯಗಳು {#theme-builder-features}

Superdav AI Agent v1.16.0 ನಲ್ಲಿನ Theme Builder ಶಕ್ತಿಶಾಲಿ ಹೊಸ ಸಾಮರ್ಥ್ಯಗಳನ್ನು ಒಳಗೊಂಡಿದೆ:

### ಫೋಟೋ ಅಪ್‌ಲೋಡ್‌ನೊಂದಿಗೆ ಅನ್ವೇಷಣಾ ಸಂದರ್ಶನ {#discovery-interview-with-photo-upload}

Theme Builder ಅನ್ವೇಷಣಾ ಸಂದರ್ಶನದಲ್ಲಿ ಈಗ ನೀವು ದೃಶ್ಯ ಪ್ರೇರಣೆಯನ್ನು ಒದಗಿಸಬಹುದಾದ ಫೋಟೋ ಅಪ್‌ಲೋಡ್ ಹಂತವಿದೆ. ನಿಮ್ಮ ವಿನ್ಯಾಸದ ದಿಕ್ಕನ್ನು Theme Builder ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು ಸಹಾಯವಾಗುವಂತೆ ನಿಮ್ಮ brand, ಸ್ಪರ್ಧಿ sites, ಅಥವಾ ವಿನ್ಯಾಸ ಪ್ರೇರಣೆಯ ಫೋಟೋಗಳನ್ನು ಅಪ್‌ಲೋಡ್ ಮಾಡಿ.

ವಿಸ್ತೃತ ಮಾರ್ಗದರ್ಶನಕ್ಕಾಗಿ [ಅನ್ವೇಷಣಾ ಸಂದರ್ಶನ](./theme-builder/discovery-interview.md) ನೋಡಿ.

### Desktop ಮತ್ತು Mobile ಪೂರ್ವವೀಕ್ಷಣೆಗಳೊಂದಿಗೆ ವಿನ್ಯಾಸದ ದಿಕ್ಕು {#design-direction-with-desktop-and-mobile-previews}

Theme Builder ನಿಮ್ಮ ಸಂಪೂರ್ಣ theme ಅನ್ನು ರಚಿಸುವ ಮೊದಲು desktop ಮತ್ತು mobile ಸಾಧನಗಳೆರಡರಲ್ಲೂ ನಿಮ್ಮ ವಿನ್ಯಾಸವನ್ನು ಪರಿಶೀಲಿಸಿ. ಅನೇಕ responsive breakpoints ನಲ್ಲಿ ತಕ್ಷಣದ ಪೂರ್ವವೀಕ್ಷಣೆಗಳನ್ನು ನೋಡಿ.

ವಿಸ್ತೃತ ಮಾರ್ಗದರ್ಶನಕ್ಕಾಗಿ [ವಿನ್ಯಾಸದ ದಿಕ್ಕು](./theme-builder/design-direction.md) ನೋಡಿ.

### ಆತಿಥ್ಯ ಮೆನುಗಳು {#hospitality-menus}

Theme Builder ಈಗ ರೆಸ್ಟೋರೆಂಟ್‌ಗಳು, ಕಫೆಗಳು, ಬಾರ್‌ಗಳು, ಮತ್ತು ಇತರ ಆತಿಥ್ಯ ವ್ಯವಹಾರಗಳಿಗೆ ರಚನಾತ್ಮಕ ಆಹಾರ ಮತ್ತು ಪಾನೀಯ menu ಪುಟಗಳನ್ನು ರಚಿಸಬಹುದು. ಮೆನುಗಳಲ್ಲಿ ಬೆಲೆ, ವಿವರಣೆಗಳು, ಆಹಾರ ಸಂಬಂಧಿತ ಮಾಹಿತಿ, ಮತ್ತು ಅಲರ್ಜಿ ಎಚ್ಚರಿಕೆಗಳು ಸೇರಿರುತ್ತವೆ.

ವಿಸ್ತೃತ ಮಾರ್ಗದರ್ಶನಕ್ಕಾಗಿ [ಆತಿಥ್ಯ ಮೆನುಗಳು](./theme-builder/hospitality-menus.md) ನೋಡಿ.

## ಸಂರಚನೆ {#configuration}

ಎಲ್ಲಾ settings ಗಳು WordPress admin ನಲ್ಲಿ **Gratis AI Agent → Settings** ಅಡಿಯಲ್ಲಿ ಇವೆ. Superdav AI Agent ಕೆಳಗಿನ ಸಂರಚನಾ ಆಯ್ಕೆಗಳನ್ನು ಸೇರಿಸುತ್ತದೆ:

|---|---|

## ಸುಧಾರಿತ Companion Package {#advanced-companion-package}

- **Tier 1 site-scrape ಸಾಧನ** — ಸ್ಪರ್ಧಿ ಮತ್ತು ಪ್ರೇರಣಾ ವೆಬ್‌ಸೈಟ್‌ಗಳನ್ನು ವಿಶ್ಲೇಷಿಸಲು sd-ai-agent/site-scrape ಈಗ Theme Builder‌ನಲ್ಲಿ ಪೂರ್ವನಿಯೋಜಿತವಾಗಿ ಲಭ್ಯವಿದೆ
- **ಪ್ಲಗಿನ್ ಸಾಲಿನ ಕ್ರಿಯಾ ಲಿಂಕ್‌ಗಳು** — ಸ್ಪಷ್ಟತೆಗೆ ಸರಿಪಡಿಸಿ ಮರುಹೆಸರಿಸಲಾಗಿದೆ

## v1.10.0 ನಲ್ಲಿ ಸುಧಾರಣೆಗಳು {#improvements-in-v1100}

- **WooCommerce ಏಕೀಕರಣವನ್ನು ಮರುರಚಿಸಲಾಗಿದೆ** — ಉತ್ತಮ ವಿಶ್ವಾಸಾರ್ಹತೆ ಮತ್ತು ಹೊಂದಾಣಿಕೆಗೆ ಈಗ ಸ್ಥಳೀಯ WooCommerce APIs ಬಳಸುತ್ತದೆ
- **ಪೂರೈಕೆದಾರ ಪಟ್ಟಿಯ ಸ್ವಯಂ-ರಿಫ್ರೆಶ್** — ಪ್ಲಗಿನ್‌ಗಳನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿದಾಗ ಅಥವಾ ನಿಷ್ಕ್ರಿಯಗೊಳಿಸಿದಾಗ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಗುರುತಿಸಿ ಲಭ್ಯವಿರುವ ಪೂರೈಕೆದಾರ ಪಟ್ಟಿಯನ್ನು ನವೀಕರಿಸುತ್ತದೆ
- **ವರ್ಧಿತ ಸೈಟ್ ಬಿಲ್ಡರ್** — ಉತ್ತಮ ದೋಷ ಚೇತರಿಕೆಯಿಂದ ಬಹು-ಹಂತದ ಸೈಟ್ ನಿರ್ಮಾಣ ಯೋಜನೆಗಳನ್ನು ಸುಧಾರಿಸಲಾಗಿದೆ
- **ಥೀಮ್‌ಗೆ ಹೊಂದಿಕೊಳ್ಳುವ ಕೌಶಲ್ಯಗಳು** — ನಿಮ್ಮ ಸಕ್ರಿಯ ಥೀಮ್‌ಗೆ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಹೊಂದಿಕೊಳ್ಳುವ ನಾಲ್ಕು ಹೊಸ ಅಂತರ್ನಿರ್ಮಿತ ಕೌಶಲ್ಯಗಳು

## v1.10.0 ನಲ್ಲಿ ದೋಷ ಸರಿಪಡಿಕೆಗಳು {#bug-fixes-in-v1100}

- ಕೆಲವು admin ಪುಟಗಳಲ್ಲಿ ಅನಂತ ಮರುಲೋಡ್ ಲೂಪ್‌ಗಳನ್ನು ಉಂಟುಮಾಡುತ್ತಿದ್ದ `navigate-to` ಸಾಮರ್ಥ್ಯವನ್ನು ಸರಿಪಡಿಸಲಾಗಿದೆ
- ವರ್ಗ ಮತ್ತು ಟ್ಯಾಗ್ ಹೆಸರುಗಳನ್ನು slugs ಆಗಿ ಸರಿಯಾಗಿ ಪರಿಹರಿಸದ `list-posts` ಸಾಮರ್ಥ್ಯವನ್ನು ಸರಿಪಡಿಸಲಾಗಿದೆ
- ಹಿಂದಿನ ಮರುರಚನೆಯ ನಂತರ namespace aliases ಕಳೆದುಕೊಂಡಿದ್ದ WP-CLI commands ಸರಿಪಡಿಸಲಾಗಿದೆ

- system instruction builder ನಲ್ಲಿ ಸರಿಯಾದ namespace prefix ಬಳಸುವಂತೆ `memory-save` ಸಾಮರ್ಥ್ಯವನ್ನು ಸರಿಪಡಿಸಲಾಗಿದೆ
- AIಗೆ ಹಿಂತಿರುಗಿಸುವ ಮೊದಲು scalar tool ಫಲಿತಾಂಶಗಳನ್ನು ಸರಿಯಾಗಿ wrap ಮಾಡುವಂತೆ ಸರಿಪಡಿಸಲಾಗಿದೆ
