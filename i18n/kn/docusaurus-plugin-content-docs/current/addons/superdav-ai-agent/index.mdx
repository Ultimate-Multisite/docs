# Superdav AI Agent

Superdav AI Agent ಎಂಬುದು ಒಂದು ಸುಧಾರಿತ WordPress AI ಸಹಾಯಕವಾಗಿದ್ದು, ಇದು Gratis AI Agent ಅನ್ನು ಥೀಮ್-ಅವೇರ್ ಸ್ಕಿಲ್ಸ್, ವರ್ಧಿತ ಇಂಟರ್ನೆಟ್ ಸರ್ಚ್ ಸಾಮರ್ಥ್ಯಗಳು ಮತ್ತು ಸುಧಾರಿತ ಸೈಟ್ ಬಿಲ್ಡರ್ ವೈಶಿಷ್ಟ್ಯಗಳೊಂದಿಗೆ ವಿಸ್ತರಿಸುತ್ತದೆ. Version 1.16.0 ರಲ್ಲಿ ಕಸ್ಟಮ್ ಲೋಗೋ ಜನರೇಷನ್, ಪ್ಯಾಲೆಟ್ ಕಾಂಟ್ರಾಸ್ಟ್ ವ್ಯಾಲಿಡೇಷನ್, ಡಿಸ್ಕವರಿ ಇಂಟರ್‌ವ್ಯೂನಲ್ಲಿ ಫೋಟೋ ಅಪ್‌ಲೋಡ್, ರಚನಾತ್ಮಕ ಹಾಸ್ಪಿಟಾಲಿಟಿ ಮೆನು ಜನರೇಷನ್ ಮತ್ತು Theme Builder ನಲ್ಲಿ ಡೆಸ್ಕ್‌ಟಾಪ್/ಮೊಬೈಲ್ ಪ್ರಿವ್ಯೂ ರೆಂಡರಿಂಗ್ ಸೇರಿಸಲಾಗಿದೆ.

## ಪ್ರಮುಖ ವೈಶಿಷ್ಟ್ಯಗಳು (Key Features)

- **ಥೀಮ್-ಅವೇರ್ ಸ್ಕಿಲ್ಸ್ (Theme-aware skills)** — ನಿಮ್ಮ ಸಕ್ರಿಯ WordPress ಥೀಮ್ ಮತ್ತು ಅಳವಡಿಸಲಾದ ಪ್ಲಗ್ಇನ್‌ಗಳ ಆಧಾರದ ಮೇಲೆ ಮಾರ್ಗದರ್ಶನ ಮತ್ತು ಸಾಮರ್ಥ್ಯಗಳನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಹೊಂದಿಸುತ್ತದೆ.
- **ಬಹು ಇಂಟರ್ನೆಟ್ ಸರ್ಚ್ ಪроваೈಡರ್‌ಗಳು (Multiple internet search providers)** — ಶ್ರೀಮಂತ ಸರ್ಚ್ ಫಲಿತಾಂಶಗಳು ಮತ್ತು ಸುಧಾರಿತ ಸಂಶೋಧನಾ ಸಾಮರ್ಥ್ಯಗಳಿಗಾಗಿ Brave Search ಮತ್ತು Tavily ನಡುವೆ ಆಯ್ಕೆಮಾಡಿ.
- **ಸೈಟ್ ಬಿಲ್ಡರ್ contact ಫಾರ್ಮ್‌ಗಳು (Site builder contact forms)** — ನಿಮ್ಮ ಅಳವಡಿಸಲಾದ ಫಾರ್ಮ್ ಪ್ಲಗ್ಇನ್ ಬಳಸಿ ಚಾಟ್ ಇಂಟರ್ಫೇಸ್‌ನಿಂದ ನೇರವಾಗಿ ಸಂಪೂರ್ಣ ಕ್ರಿಯಾತ್ಮಕ contact ಫಾರ್ಮ್‌ಗಳನ್ನು ರಚಿಸಿ.
- **ಸುಧಾರಿತ ಸೈಟ್ ಬಿಲ್ಡರ್ (Enhanced site builder)** — ಉತ್ತಮ ಪ್ಲಗ್ಇನ್ ಡಿಸ್ಕವರಿ ಮತ್ತು ಎರೆರ್ ರಿಕವರಿ ಜೊತೆಗೆ ಸುಧಾರಿತ ಬಹು-ಹಂತದ ಸೈಟ್ ಬಿಲ್ಡ್ ಯೋಜನೆಗಳು.
- **ಬिल्ಟ್-ಇನ್ ಥೀಮ್ ಸ್ಕಿಲ್ಸ್ (Built-in theme skills)** — Block Themes, Classic Themes, Kadence Blocks ಮತ್ತು Kadence Theme ಗಾಗಿ ವಿಶೇಷ ಮಾರ್ಗದರ್ಶನ.
- **ಸುಧಾರಿತ WooCommerce integration** — ಉತ್ತಮ ವಿಶ್ವಾಸಾರ್ಹತೆ ಮತ್ತು ಹೊಂದಾಣಿಕೆಗಾಗಿ native WooCommerce APIs ಬಳಸುವಂತೆ ಮರುರೂಪಿಸಲಾಗಿದೆ.
- **ಸ್ವಯಂಚಾಲಿತ ಪроваೈಡರ್ detection** — ಪ್ಲಗ್ಇನ್‌ಗಳು ಅಥವಾ ಥೀಮ್‌ಗಳನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿದಾಗ ಅಥವಾ ನಿಷ್ಕ್ರಿಯಗೊಳಿಸಿದಾಗ ಲಭ್ಯವಿರುವ ಪ್ಲಗ್ಇನ್‌ಗಳು ಮತ್ತು ಥೀಮ್‌ಗಳನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಪತ್ತೆಹಚ್ಚುತ್ತದೆ ಮತ್ತು ರಿಫ್ರೆಶ್ ಮಾಡುತ್ತದೆ.
- **ಕಸ್ಟಮ್ ಲೋಗೋ ಜನರೇಷನ್ (Custom logo generation)** — ಸ್ವಯಂಚಾಲಿತ namespace-safe sanitisation ಜೊತೆಗೆ ಕಸ್ಟಮ್ SVG ಲೋಗೋಗಳನ್ನು ರಚಿಸಿ ಮತ್ತು ಅಳವಡಿಸಿ.
- **ಪ್ಯಾಲೆಟ್ ಕಾಂಟ್ರಾಸ್ಟ್ ವ್ಯಾಲಿಡೇಷನ್ (Palette contrast validation)** — ನಿಮ್ಮ ಥೀಮ್‌ಗೆ ಅನ್ವಯಿಸುವ ಮೊದಲು WCAG ಪ್ರವೇಶಸಾಧ್ಯತೆ ಅನುಸರಣೆಗಾಗಿ ಬಣ್ಣ ಜೋಡಿಗಳನ್ನು ಪರಿಶೀಲಿಸಿ.
- **ಫೋಟೋ-enhanced discovery (Photo-enhanced discovery)** — ವಿನ್ಯಾಸದ ದಿಕ್ಕನ್ನು ಪ್ರಭಾವಿಸಲು Theme Builder ಡಿಸ್ಕವರಿ ಇಂಟರ್‌ವ್ಯೂ ಸಮಯದಲ್ಲಿ ಫೋಟೋಗಳನ್ನು ಅಪ್‌ಲೋಡ್ ಮಾಡಿ.
- **ಹಾಸ್ಪಿಟಾಲಿಟಿ ಮೆನುಗಳು (Hospitality menus)** — ರೆಸ್ಟೋರೆಂಟ್‌ಗಳು, ಕೆಫೆಗಳು ಮತ್ತು ಹಾಸ್ಪಿಟಾಲಿಟಿ ವ್ಯವಹಾರಗಳಿಗೆ ರಚನಾತ್ಮಕ ಆಹಾರ ಮತ್ತು ಪಾನೀಯ ಮೆನು ಪುಟಗಳನ್ನು ರಚಿಸಿ.
- **ಡೆಸ್ಕ್‌ಟಾಪ್ ಮತ್ತು ಮೊಬೈಲ್ ಪ್ರಿವ್ಯೂಗಳು (Desktop and mobile previews)** — ವಿನ್ಯಾಸದ ದಿಕ್ಕನ್ನು ಆಯ್ಕೆ ಮಾಡುವ ಸಮಯದಲ್ಲಿ ಡೆಸ್ಕ್‌ಟಾಪ್ ಮತ್ತು ಮೊಬೈಲ್ ಸಾಧನಗಳಲ್ಲಿ ನಿಮ್ಮ ವಿನ್ಯಾಸವನ್ನು ಪೂರ್ವವೀಕ್ಷಿಸಿ.
- **Tier 1 site scraping** — ಸ್ಪರ್ಧಾ ಮತ್ತು ಸ್ಫೂರ್ತಿ ವೆಬ್‌ಸೈಟ್‌ಗಳನ್ನು ವಿಶ್ಲೇಷಿಸಲು sd-ai-agent/site-scrape ಈಗ ಡೀಫಾಲ್ಟ್ ಆಗಿ ಲಭ್ಯವಿದೆ.

## ಅವಶ್ಯಕತೆಗಳು (Requirements)

- WordPress 6.0 ಅಥವಾ ಅದಕ್ಕಿಂತ ಹೆಚ್ಚಿನದು
- PHP 8.1 ಅಥವಾ ಅದಕ್ಕಿಂತ ಹೆಚ್ಚಿನದು
- WordPress Multisite (network-activated) ಅಥವಾ single-site installation
- ಸಕ್ರಿಯ AI ಪроваೈಡರ್ ಕನೆಕ್ಟರ್ ಪ್ಲಗ್ಇನ್ (Anthropic, OpenAI, ಅಥವಾ ಹೊಂದಾಣಿಕೆಯ)
- Gratis AI Agent v1.9.0 ಅಥವಾ ಅದಕ್ಕಿಂತ ಹೆಚ್ಚಿನದು (Superdav AI Agent, Gratis AI Agent ವಿಸ್ತರಣೆ)

## ಅಳವಡಿಕೆ (Installation)

1. ಪ್ಲಗ್ಇನ್ ಫೈಲ್‌ಗಳನ್ನು `/wp-content/plugins/superdav-ai-agent` ಗೆ ಅಪ್‌ಲೋಡ್ ಮಾಡಿ.
2. WordPress ನಲ್ಲಿರುವ **Plugins** ಪರದೆಯ ಮೂಲಕ ಪ್ಲಗ್ಇನ್ ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ.
3. Gratis AI Agent ಕೂಡ ಸಕ್ರಿಯಗೊಂಡಿದೆಯೇ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ.
4. **Gratis AI Agent → Settings** ಗೆ ನ್ಯಾವಿಗೇಟ್ ಮಾಡಿ ಮತ್ತು ನಿಮ್ಮ ಇಂಟರ್ನೆಟ್ ಸರ್ಚ್ ಪроваೈಡರ್ ಅನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಿ (ಐಚ್ಛಿಕ).

## ಥೀಮ್-ಅವೇರ್ ಸ್ಕಿಲ್ಸ್ (Theme-Aware Skills)

Superdav AI Agent ಸ್ವಯಂಚಾಲಿತವಾಗಿ ನಿಮ್ಮ ಸಕ್ರಿಯ ಥೀಮ್ ಮತ್ತು ಅಳವಡಿಸಲಾದ ಪ್ಲಗ್ಇನ್‌ಗಳನ್ನು ಪತ್ತೆಹಚ್ಚುತ್ತದೆ, ಮತ್ತು ಸೂಕ್ತವಾದ ಥೀಮ್-ಅವೇರ್ ಸ್ಕಿಲ್ ಅನ್ನು ಸಹಾಯಕನ ಸಂದರ್ಭಕ್ಕೆ (context) ಲೋಡ್ ಮಾಡುತ್ತದೆ. ಲಭ್ಯವಿರುವ ಸ್ಕಿಲ್ಗಳು:

| Skill | Applies To |
|---|---|
| **Block Themes** | WordPress block-based (Full Site Editing) architecture ಬಳಸುವ ಥೀಮ್‌ಗಳು |
| **Classic Themes** | PHP ಟೆಂಪ್ಲೇಟ್‌ಗಳು ಮತ್ತು classic editor ಬಳಸುವ ಸಾಂಪ್ರದಾಯಿಕ WordPress ಥೀಮ್‌ಗಳು |
| **Kadence Blocks** | Kadence Blocks ಪ್ಲಗ್ಇನ್ ಸಕ್ರಿಯವಾಗಿರುವ ಸೈಟ್‌ಗಳು |
| **Kadence Theme** | Kadence ಥೀಮ್ ಅನ್ನು ಸಕ್ರಿಯ ಥೀಮ್ ಆಗಿ ಬಳಸುವ ಸೈಟ್‌ಗಳು |

ಪ್ರತಿ ಸ್ಕಿಲ್ ಕುರಿತು ವಿವರವಾದ ಮಾರ್ಗದರ್ಶನಕ್ಕಾಗಿ [Theme-Aware Skills](../../user-guide/configuration/theme-aware-skills) ನೋಡಿ.

## ಇಂಟರ್ನೆಟ್ ಸರ್ಚ್ ಪроваೈಡರ್‌ಗಳು (Internet Search Providers)

Superdav AI Agent v1.10.0 ರಲ್ಲಿ ಇಂಟರ್ನೆಟ್ ಸರ್ಚ್ ಪ್ರಶ್ನೆಗಳಿಗೆ Brave Search ಗೆ ಪರ್ಯಾಯವಾಗಿ **Tavily** ಅನ್ನು ಸೇರಿಸಲಾಗಿದೆ. ಎರಡೂ ಪроваೈಡರ್‌ಗಳು ನೈಜ-ಸಮಯದ ಸರ್ಚ್ ಫಲಿತಾಂಶಗಳನ್ನು ನೀಡುತ್ತವೆ ಮತ್ತು ಅವುಗಳನ್ನು **Gratis AI Agent → Settings → Advanced** ನಲ್ಲಿ ಕಾನ್ಫಿಗರ್ ಮಾಡಬಹುದು.

| Provider | Features | Setup |
|---|---|---|
| **Brave Search** | ವೇಗದ, ಗೌಪ್ಯತೆ-ಕೇಂದ್ರಿತ ಸರ್ಚ್ ಫಲಿತಾಂಶಗಳು | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | ಶ್ರೀಮಂತ ಸರ್ಚ್ ಫಲಿತಾಂಶಗಳು, ಸುಧಾರಿತ ಸಂಶೋಧನಾ ಸಾಮರ್ಥ್ಯಗಳು | [Tavily API](https://tavily.com/) |

ಕಾನ್ಫಿಗರೇಶನ್ ಸೂಚನೆಗಳಿಗಾಗಿ [Internet Search](../../user-guide/configuration/internet-search) ನೋಡಿ.

## ಸೈಟ್ ಬಿಲ್ಡರ್ contact ಫಾರ್ಮ್‌ಗಳು (Site Builder Contact Forms)

ಸೈಟ್ ಬಿಲ್ಡರ್ ಏಜೆಂಟ್ ಈಗ ಚಾಟ್ ಇಂಟರ್ಫೇಸ್‌ನಿಂದ ನೇರವಾಗಿ contact ಫಾರ್ಮ್‌ಗಳನ್ನು ರಚಿಸಲು ಸಾಧ್ಯವಾಗಿದೆ. ಕೇವಲ ಸೈಟ್ ಬಿಲ್ಡರ್ ಅನ್ನು contact ಫಾರ್ಮ್ ರಚಿಸಲು ಕೇಳಿ, ಮತ್ತು ಅದು ನಿಮ್ಮ ಅಳವಡಿಸಲಾದ ಫಾರ್ಮ್ ಪ್ಲಗ್ಇನ್ (Contact Form 7, WPForms, Fluent Forms, ಅಥವಾ Gravity Forms) ಅನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಪತ್ತೆಹಚ್ಚಿ ಬಳಸಲು ಸಿದ್ಧವಾದ ಫಾರ್ಮ್ ಅನ್ನು ರಚಿಸುತ್ತದೆ.

ವಿವರವಾದ ಮಾರ್ಗದರ್ಶನಕ್ಕಾಗಿ [Site Builder Contact Forms](../../user-guide/administration/site-builder-contact-forms) ನೋಡಿ.

## Theme Builder ವೈಶಿಷ್ಟ್ಯಗಳು (Theme Builder Features)

Superdav AI Agent v1.16.0 ರಲ್ಲಿ Theme Builder ನಲ್ಲಿ ಶಕ್ತಿಶಾಲಿ ಹೊಸ ಸಾಮರ್ಥ್ಯಗಳು ಸೇರಿವೆ:

### ಫೋಟೋ ಅಪ್‌ಲೋಡ್‌ನೊಂದಿಗೆ ಡಿಸ್ಕವರಿ ಇಂಟರ್‌ವ್ಯೂ (Discovery Interview with Photo Upload)

Theme Builder ಡಿಸ್ಕವರಿ ಇಂಟರ್‌ವ್ಯೂ ಈಗ ಫೋಟೋ ಅಪ್‌ಲೋಡ್ ಹಂತವನ್ನು ಒಳಗೊಂಡಿದೆ, ಅಲ್ಲಿ ನೀವು ದೃಶ್ಯ ಸ್ಫೂರ್ತಿಯನ್ನು ಒದಗಿಸಬಹುದು. ನಿಮ್ಮ ಬ್ರ್ಯಾಂಡ್, ಸ್ಪರ್ಧಾ ಸೈಟ್‌ಗಳು ಅಥವಾ ವಿನ್ಯಾಸ ಸ್ಫೂರ್ತಿಯ ಫೋಟೋಗಳನ್ನು ಅಪ್‌ಲೋಡ್ ಮಾಡಿ, Theme Builder ಗೆ ನಿಮ್ಮ ವಿನ್ಯಾಸದ ದಿಕ್ಕನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು ಸಹಾಯ ಮಾಡಿ.

ವಿವರವಾದ ಮಾರ್ಗದರ್ಶನಕ್ಕಾಗಿ [Discovery Interview](./theme-builder/discovery-interview.md) ನೋಡಿ.

### ಡೆಸ್ಕ್‌ಟಾಪ್ ಮತ್ತು ಮೊಬೈಲ್ ಪ್ರಿವ್ಯೂಗಳೊಂದಿಗೆ ವಿನ್ಯಾಸದ ದಿಕ್ಕು (Design Direction with Desktop and Mobile Previews)

Theme Builder ನಿಮ್ಮ ಸಂಪೂರ್ಣ ಥೀಮ್ ಅನ್ನು ರಚಿಸುವ ಮೊದಲು ಡೆಸ್ಕ್‌ಟಾಪ್ ಮತ್ತು ಮೊಬೈಲ್ ಎರಡೂ ಸಾಧನಗಳಲ್ಲಿ ನಿಮ್ಮ ವಿನ್ಯಾಸವನ್ನು ಪರಿಶೀಲಿಸಿ. ಅನೇಕ responsive breakpoints ನಲ್ಲಿ ನೈಜ-ಸಮಯದ ಪೂರ್ವವೀಕ್ಷಣೆಗಳನ್ನು ನೋಡಿ.

ವಿವರವಾದ ಮಾರ್ಗದರ್ಶನಕ್ಕಾಗಿ [Design Direction](./theme-builder/design-direction.md) ನೋಡಿ.

### ಹಾಸ್ಪಿಟಾಲಿಟಿ ಮೆನುಗಳು (Hospitality Menus)

Theme Builder ಈಗ ರೆಸ್ಟೋರೆಂಟ್‌ಗಳು, ಕೆಫೆಗಳು, ಬಾರ್‌ಗಳು ಮತ್ತು ಇತರ ಹಾಸ್ಪಿಟಾಲಿಟಿ ವ್ಯವಹಾರಗಳಿಗೆ ರಚನಾತ್ಮಕ ಆಹಾರ ಮತ್ತು ಪಾನೀಯ ಮೆನು ಪುಟಗಳನ್ನು ರಚಿಸಲು ಸಾಧ್ಯವಾಗಿದೆ. ಮೆನುಗಳಲ್ಲಿ ಬೆಲೆ, ವಿವರಣೆಗಳು, ಆಹಾರದ ಮಾಹಿತಿ ಮತ್ತು ಅಲರ್ಜೆನ್ ಎಚ್ಚರಿಕೆಗಳು ಸೇರಿವೆ.

ವಿವರವಾದ ಮಾರ್ಗದರ್ಶನಕ್ಕಾಗಿ [Hospitality Menus](./theme-builder/hospitality-menus.md) ನೋಡಿ.

### ಹೊಸ ಸಾಮರ್ಥ್ಯಗಳು (New Abilities)

- **[Generate Logo SVG](./abilities/generate-logo-svg.md)** — ಸ್ವಯಂಚಾಲಿತ sanitisation ಜೊತೆಗೆ ಕಸ್ಟಮ್ SVG ಲೋಗೋಗಳನ್ನು ರಚಿಸಿ
- **[Validate Palette Contrast](./abilities/validate-palette-contrast.md)** — WCAG ಅನುಸರಣೆಗಾಗಿ ಬಣ್ಣ ಜೋಡಿಗಳನ್ನು ಪರಿಶೀಲಿಸಿ
- **[Create Menu](./abilities/create-menu.md)** — ವಿಭಿನ್ನ ನ್ಯಾವಿಗೇಷನ್ ಲೇಬಲ್‌ಗಳೊಂದಿಗೆ ನ್ಯಾವಿಗೇಷನ್ ಮೆನುಗಳನ್ನು ರಚಿಸಿ

ಲಭ್ಯವಿರುವ ಟೂಲ್‌ಗಳು ಮತ್ತು Tier 1 site-scrape ಟೂಲ್ ಕುರಿತು ಮಾಹಿತಿಗಾಗಿ [Tools and Configuration](./theme-builder/tools.md) ನೋಡಿ.

## ಕಾನ್ಫಿಗರೇಶನ್ (Configuration)

ಎಲ್ಲಾ ಸೆಟ್ಟಿಂಗ್‌ಗಳು WordPress ಆಡ್ಮಿನ್‌ನಲ್ಲಿ **Gratis AI Agent → Settings** ಅಡಿಯಲ್ಲಿವೆ. Superdav AI Agent ಈ ಕೆಳಗಿನ ಕಾನ್ಫಿಗರೇಶನ್ ಆಯ್ಕೆಗಳನ್ನು ಸೇರಿಸುತ್ತದೆ:

| Setting | Description |
|---|---|
| **Internet Search Provider** | Brave Search ಮತ್ತು Tavily ನಡುವೆ ಆಯ್ಕೆಮಾಡಿ (ಐಚ್ಛಿಕ) |
| **Tavily API Key** | Tavily ಇಂಟರ್ನೆಟ್ ಸರ್ಚ್ ಗಾಗಿ API key (Tavily ಆಯ್ಕೆ ಮಾಡಿದರೆ) |

## v1.16.0 ರಲ್ಲಿ ಸುಧಾರಣೆಗಳು (Improvements in v1.16.0)

- **AI Client cache** — ಈಗ cross-request persistence ಗಾಗಿ transients ನಿಂದ ಬೆಂಬಲಿತವಾಗಿದೆ, ಇದು ದೀರ್ಘಾವಧಿಯ ಏಜೆಂಟ್ ಕಾರ್ಯಗಳಲ್ಲಿ ಡೇಟಾ ನಷ್ಟವನ್ನು ತಡೆಯುತ್ತದೆ.
- **Tier 1 site-scrape tool** — ಸ್ಪರ್ಧಾ ಮತ್ತು ಸ್ಫೂರ್ತಿ ವೆಬ್‌ಸೈಟ್‌ಗಳನ್ನು ವಿಶ್ಲೇಷಿಸಲು Theme Builder ನಲ್ಲಿ sd-ai-agent/site-scrape ಈಗ ಡೀಫಾಲ್ಟ್ ಆಗಿ ಲಭ್ಯವಿದೆ.
- **Plugin row action links** — ಸ್ಪಷ್ಟತೆಗಾಗಿ ಸರಿಪಡಿಸಲಾಗಿದೆ ಮತ್ತು ಮರುನಾಮಕರಣ ಮಾಡಲಾಗಿದೆ.

## v1.10.0 ರಲ್ಲಿ ಸುಧಾರಣೆಗಳು (Improvements in v1.10.0)

- **WooCommerce integration refactored** — ಉತ್ತಮ ವಿಶ್ವಾಸಾರ್ಹತೆ ಮತ್ತು ಹೊಂದಾಣಿಕೆಗಾಗಿ ಈಗ native WooCommerce APIs ಬಳಸುತ್ತದೆ.
- **Provider list auto-refresh** — ಪ್ಲಗ್ಇನ್‌ಗಳನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿದಾಗ ಅಥವಾ ನಿಷ್ಕ್ರಿಯಗೊಳಿಸಿದಾಗ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಪತ್ತೆಹಚ್ಚುತ್ತದೆ ಮತ್ತು ಲಭ್ಯವಿರುವ ಪроваೈಡರ್ ಪಟ್ಟಿಯನ್ನು ನವೀಕರಿಸುತ್ತದೆ.
- **Enhanced site builder** — ಉತ್ತಮ ಎರೆರ್ ರಿಕವರಿ ಜೊತೆಗೆ ಸುಧಾರಿತ ಬಹು-ಹಂತದ ಸೈಟ್ ಬಿಲ್ಡ್ ಯೋಜನೆಗಳು.
- **Theme-aware skills** — ನಿಮ್ಮ ಸಕ್ರಿಯ ಥೀಮ್‌ಗೆ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಹೊಂದಿಕೊಳ್ಳುವ ನಾಲ್ಕು ಹೊಸ ಬिल्ಟ್-ಇನ್ ಸ್ಕಿಲ್ಸ್.

## v1.10.0 ರಲ್ಲಿ ಬಗ್ ಫಿಕ್ಸ್ (Bug Fixes in v1.10.0)

- ಕೆಲವು ಆಡ್ಮಿನ್ ಪುಟಗಳಲ್ಲಿ ಅನಂತ ರೀಲೋಡ್ ಲೂಪ್‌ಗಳನ್ನು ಉಂಟುಮಾಡುವ `navigate-to` ಸಾಮರ್ಥ್ಯವನ್ನು ಸರಿಪಡಿಸಲಾಗಿದೆ.
- ವರ್ಗ ಮತ್ತು ಟ್ಯಾಗ್ ಹೆಸರುಗಳನ್ನು ಸ್ಲಗ್‌ಗಳಿಗೆ ಸರಿಯಾಗಿ ಪರಿಹರಿಸದ `list-posts` ಸಾಮರ್ಥ್ಯವನ್ನು ಸರಿಪಡಿಸಲಾಗಿದೆ.
- ಹಿಂದಿನ refactor ನಂತರ WP-CLI ಕಮಾಂಡ್‌ಗಳು namespace aliases ಅನ್ನು ಕಳೆದುಕೊಳ್ಳುತ್ತಿದ್ದ ಸಮಸ್ಯೆಯನ್ನು ಸರಿಪಡಿಸಲಾಗಿದೆ.
- ಆಟೊಮೇಷನ್ ಟೇಬಲ್‌ಗಳು ಇನ್ನೂ ರಚಿಸದ ಸೈಟ್‌ಗಳನ್ನು gracefull ಆಗಿ ನಿರ್ವಹಿಸುವಂತೆ ಈವೆಂಟ್ ಆಟೊಮೇಷನ್ ಅನ್ನು ಸರಿಪಡಿಸಲಾಗಿದೆ.
- ಸಿಸ್ಟಮ್ ಇನ್‌ಸ್ಟ್ರಕ್ಷನ್ ಬಿಲ್ಡರ್‌ನಲ್ಲಿ ಸರಿಯಾದ namespace prefix ಅನ್ನು ಬಳಸುವ `memory-save` ಸಾಮರ್ಥ್ಯವನ್ನು ಸರಿಪಡಿಸಲಾಗಿದೆ.
- AI ಗೆ ಮರಳಿ ನೀಡುವ ಮೊದಲು ಸ್ಕೇಲರ್ ಟೂಲ್ ಫಲಿತಾಂಶಗಳನ್ನು ಸರಿಯಾಗಿ ಸುತ್ತುವರೆದಿರುವಂತೆ ಸರಿಪಡಿಸಲಾಗಿದೆ.
- ಹಳೆಯ ಆವร์ಸನ್‌ಗಳಿಂದ ಅಪ್‌ಗ್ರೇಡ್ ಮಾಡುವಾಗ usage statistics ಸರಿಯಾಗಿ legacy ability key format ಅನ್ನು ನಿರ್ವಹಿಸುವಂತೆ ಸರಿಪಡಿಸಲಾಗಿದೆ.
