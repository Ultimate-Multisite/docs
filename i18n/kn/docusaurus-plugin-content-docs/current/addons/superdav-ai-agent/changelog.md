# ಬದಲಾವಣೆಗಳ ದಾಖಲೆ

## 1.16.0 — Released on 2026-05-20

### ಹೊಸದಾಗಿ ಸೇರಿಸಿದ್ದು

- **ಲೋಗೋ SVG ರಚಿಸುವ ಸಾಮರ್ಥ್ಯ** — ಥೀಮ್ ಬಿಲ್ಡರ್ ಈಗ ನೇಮ್‌ಸ್ಪೇಸ್-ಸೇಫ್ ಸ್ಯಾನಿಟೈಸೇಶನ್‌ನೊಂದಿಗೆ ಕಸ್ಟಮ್ ಲೋಗೋ SVG ಗಳನ್ನು ರಚಿಸಲು ಮತ್ತು ಅಳವಡಿಸಲು ಸಾಧ್ಯವಾಗಿದೆ.
- **ಡಿಸ್ಕವರಿ ಇಂಟರ್‌ವ್ಯೂನಲ್ಲಿ ಫೋಟೋ ಅಪ್‌ಲೋಡ್** — ಶ್ರೀಮಂತ ವಿನ್ಯಾಸ ಸಂದರ್ಭಕ್ಕಾಗಿ (design context) ಥೀಮ್ ಬಿಲ್ಡರ್ ಡಿಸ್ಕವರಿ ಇಂಟರ್‌ವ್ಯೂನಲ್ಲಿ ಈಗ ಫೋಟೋ ಅಪ್‌ಲೋಡ್ ಹಂತವನ್ನು ಸೇರಿಸಲಾಗಿದೆ.
- **ಪ್ಯಾಲೆಟ್ ಕಾಂಟ್ರಾಸ್ಟ್ ಪರಿಶೀಲಿಸುವ ಸಾಮರ್ಥ್ಯ** — ಥೀಮ್‌ಗೆ ಅನ್ವಯಿಸುವ ಮೊದಲು ಬಣ್ಣದ ಜೋಡಿಗಳನ್ನು WCAG ಅನುಸರಣೆಗಾಗಿ ಪರಿಶೀಲಿಸಿ.
- **ಆತಿಥ್ಯ ಮೆನುಗಳು** — ಆತಿಥ್ಯ ವ್ಯವಹಾರಗಳಿಗೆ (hospitality businesses) ಥೀಮ್ ಬಿಲ್ಡರ್ ಈಗ ರಚನಾತ್ಮಕ ಆಹಾರ ಮತ್ತು ಪಾನೀಯ ಮೆನು ಪುಟಗಳನ್ನು ರಚಿಸಲು ಸಾಧ್ಯವಾಗಿದೆ.
- **ಡೆಸ್ಕ್‌ಟಾಪ್ ಮತ್ತು ಮೊಬೈಲ್ ಪ್ರಿವ್ಯೂ ರೆಂಡರಿಂಗ್** — ವಿನ್ಯಾಸದ ದಿಕ್ಕನ್ನು ಆಯ್ಕೆ ಮಾಡುವಾಗ ಡೆಸ್ಕ್‌ಟಾಪ್ ಮತ್ತು ಮೊಬೈಲ್ ಸಾಧನಗಳಲ್ಲಿ ನಿಮ್ಮ ವಿನ್ಯಾಸವನ್ನು ಪೂರ್ವವೀಕ್ಷಿಸಿ.
- **ನ್ಯಾವಿಗೇಷನ್ ಲೇಬಲ್ ಪ್ಯಾರಾಮೀಟರ್** — 'Create Menu' ಸಾಮರ್ಥ್ಯವು ಈಗ ಪುಟದ ಶೀರ್ಷಿಕೆಗಿಂತ ಪ್ರತ್ಯೇಕವಾದ `navigation_label` ಅನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ.
- **ಟೈರ್ 1 ಟೂಲ್ ಲಭ್ಯತೆ** — sd-ai-agent/site-scrape ಈಗ ಥೀಮ್ ಬಿಲ್ಡರ್‌ನಲ್ಲಿ ಡೀಫಾಲ್ಟ್ ಆಗಿ ಲಭ್ಯವಿರುವ ಟೈರ್ 1 ಟೂಲ್ ಆಗಿದೆ.

### ಸರಿಪಡಿಸಲಾದದ್ದು

- **AI ಕ್ಲೈಂಟ್ ಕ್ಯಾಶ್** — ಈಗ cross-request persistence ಗಾಗಿ ಟ್ರಾನ್ಸಿಯೆಂಟ್‌ಗಳಿಂದ (transients) ಬೆಂಬಲಿತವಾಗಿದೆ, ಇದು ದೀರ್ಘಾವಧಿಯ ಏಜೆಂಟ್ ಕಾರ್ಯಗಳಲ್ಲಿ ಡೇಟಾ ನಷ್ಟವನ್ನು ತಡೆಯುತ್ತದೆ.
- **ಪ್ಲಗ್ಇನ್ ರೋ ಆಕ್ಷನ್ ಲಿಂಕ್‌ಗಳು** — ಸ್ಪಷ್ಟತೆಗಾಗಿ ಸರಿಪಡಿಸಲಾಗಿದೆ ಮತ್ತು ಮರುನಾಮಕರಣ ಮಾಡಲಾಗಿದೆ.

## 1.10.0 — Released on 2026-05-05

### ಹೊಸದಾಗಿ ಸೇರಿಸಿದ್ದು

- **Tavily ಇಂಟರ್ನೆಟ್ ಸರ್ಚ್** — Brave Search ಜೊತೆಗೆ ಶ್ರೀಮಂತ ಇಂಟರ್ನೆಟ್ ಸರ್ಚ್ ಫಲಿತಾಂಶಗಳಿಗಾಗಿ Tavily ಅನ್ನು ಸರ್ಚ್ ಪ್ರೊವೈಡರ್ ಆಗಿ ಸೇರಿಸಿ.
- **ಥೀಮ್-ಅವೇರ್ ಬิลಟ್-ಇನ್ ಸ್ಕಿಲ್ಸ್** — ಬ್ಲಾಕ್ ಥೀಮ್ಸ್, ಕ್ಲಾಸಿಕ್ ಥೀಮ್ಸ್, Kadence Blocks, ಮತ್ತು Kadence Theme ಸ್ಕಿಲ್ ಗೈಡ್‌ಗಳು ಈಗ ಪ್ಲಗ್ಇನ್‌ನೊಂದಿಗೆ ಬರುತ್ತವೆ.
- **ಸೈಟ್ ಬಿಲ್ಡರ್ contact form ಸಾಮರ್ಥ್ಯ** — ಚಾಟ್ ಇಂಟರ್ಫೇಸ್‌ನಿಂದ ನೇರವಾಗಿ ಯಾವುದೇ ಪುಟಕ್ಕೆ contact form ಅನ್ನು ಸೇರಿಸಿ.

### ಸುಧಾರಣೆಗಳು

- **WooCommerce integration ಅನ್ನು ಮರುರೂಪಿಸಲಾಗಿದೆ** — ಉತ್ತಮ ವಿಶ್ವಾಸಾರ್ಹತೆ ಮತ್ತು compatibility ಗಾಗಿ ಈಗ native WooCommerce APIs ಅನ್ನು ಬಳಸುತ್ತದೆ.
- **ಪ್ರೊವೈಡರ್ ಪಟ್ಟಿ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ರಿಫ್ರೆಶ್ ಆಗುತ್ತದೆ** — ಯಾವುದೇ ಪ್ಲಗ್ಇನ್ ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿದಾಗ ಅಥವಾ ನಿಷ್ಕ್ರಿಯಗೊಳಿಸಿದಾಗ.

### ಸರಿಪಡಿಸಲಾದದ್ದು

- **navigate-to ಸಾಮರ್ಥ್ಯ** — ಕೆಲವು ಆಡ್ಮಿನ್ ಪುಟಗಳಲ್ಲಿನ ಅನಂತ ರೀಲೋಡ್ ಲೂಪ್ ಅನ್ನು ಸರಿಪಡಿಸಲಾಗಿದೆ.
- **list-posts ಸಾಮರ್ಥ್ಯ** — ಈಗ ವರ್ಗ ಮತ್ತು ಟ್ಯಾಗ್ ಹೆಸರುಗಳನ್ನು ಸರಿಯಾಗಿ ಸ್ಲಗ್‌ಗಳಿಗೆ (slugs) ಪರಿಹರಿಸುತ್ತದೆ.
- **WP-CLI commands** — ಹಿಂದಿನ refactor ನಂತರ ಕಳೆದುಹೋದ namespace aliases ಅನ್ನು ಮರುಸ್ಥಾಪಿಸಲಾಗಿದೆ.
- **Event automation** — automation tables ಇನ್ನೂ ರಚಿಸದ ಸೈಟ್‌ಗಳನ್ನು ಸುಗಮವಾಗಿ ನಿರ್ವಹಿಸುತ್ತದೆ.
- **memory-save ಸಾಮರ್ಥ್ಯ** — ಈಗ system instruction builder ನಲ್ಲಿ ಸರಿಯಾದ namespace prefix ಅನ್ನು ಬಳಸುತ್ತದೆ.
- **Scalar tool ಫಲಿತಾಂಶಗಳು** — ಈಗ AI ಗೆ ಮರಳಿ ನೀಡುವ ಮೊದಲು ಸರಿಯಾಗಿ ಸುತ್ತುವರೆದಿದೆ (wrapped).
- **Usage statistics** — ಹಳೆಯ ಆವร์ಸನ್‌ಗಳಿಂದ ಅಪ್‌ಗ್ರೇಡ್ ಮಾಡುವಾಗ legacy ability key format ಅನ್ನು ಈಗ ಸರಿಯಾಗಿ ನಿರ್ವಹಿಸುತ್ತದೆ.
