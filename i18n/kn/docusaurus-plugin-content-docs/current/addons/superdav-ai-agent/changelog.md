# ಬದಲಾವಣೆಗಳ ಪಟ್ಟಿ

## 1.18.0 — 2026-06-29 ರಂದು ಬಿಡುಗಡೆ {#1180--released-on-2026-06-29}

### ಹೊಸದು {#new}

- **Google Calendar ಉಪಕರಣಗಳು** — ವೇಳಾಪಟ್ಟಿ ಅರಿತ ಸ್ವಯಂಚಾಲನೆಗಳಿಗಾಗಿ ಸಂರಚಿಸಿದ ಕ್ಯಾಲೆಂಡರ್‌ಗಳು ಮತ್ತು ಈವೆಂಟ್‌ಗಳನ್ನು ಓದಿ
- **ಸಂಪರ್ಕ ಮ್ಯಾಪಿಂಗ್ ಮತ್ತು ಪಾಲ್ಗೊಳ್ಳುವವರ ಸಹಾಯಕಗಳು** — ಕ್ಯಾಲೆಂಡರ್ ಭಾಗವಹಿಸುವವರನ್ನು ಸೈಟ್ ಬಳಕೆದಾರರು ಮತ್ತು ಸಂಪರ್ಕಗಳೊಂದಿಗೆ ಹೊಂದಿಸಿ
- **ಮಾನವ ಅನುಮೋದನೆ ಗೇಟ್‌ಗಳು ಮತ್ತು ನೆನಪಿಸುವ ದಾಖಲೆಗಳು** — ಪರಿಶೀಲನೆಗಾಗಿ ಸ್ವಯಂಚಾಲನೆಗಳನ್ನು ವಿರಾಮಗೊಳಿಸಿ ಮತ್ತು ನಕಲಿ ಸೂಚನೆಗಳನ್ನು ತಪ್ಪಿಸಿ
- **TextBee SMS ಪೂರೈಕೆದಾರ** — ಸಂರಚಿಸಿದ ಪಠ್ಯ-ಸಂದೇಶ ಸೂಚನೆಗಳನ್ನು TextBee ಮೂಲಕ ಕಳುಹಿಸಿ
- **ಸುಧಾರಿತ ಸಹಾಯಕ ಪ್ಯಾಕೇಜ್** — WordPress.org ನಿರ್ಮಾಣದಿಂದ ಪ್ರತ್ಯೇಕವಾಗಿ ವಿತರಿಸಲಾದ ವಿಶ್ವಾಸಾರ್ಹ-ಡೆವಲಪರ್ ಫೈಲ್‌ಸಿಸ್ಟಮ್, ಡೇಟಾಬೇಸ್, WP-CLI, REST ಡಿಸ್ಪ್ಯಾಚರ್, plugin ಬಿಲ್ಡರ್, git ಸ್ನ್ಯಾಪ್‌ಶಾಟ್, ಬಳಕೆದಾರ-ನಿರ್ವಹಣೆ ಮತ್ತು ಬೆಂಚ್‌ಮಾರ್ಕ್ ಉಪಕರಣಗಳನ್ನು ಸೇರಿಸಿ

### ಸುಧಾರಿಸಲಾಗಿದೆ {#improved}

- **ನಿರ್ವಹಿತ Superdav ಸೇವೆ ಸೆಟಪ್** — ಬೆಂಬಲಿತ ಸೈಟ್‌ಗಳಿಗಾಗಿ ಹೋಸ್ಟ್ ಮಾಡಿದ ಸೇವಾ endpoint‌ಗಳು ಮತ್ತು ಸ್ವಯಂಚಾಲಿತ ಸಂಪರ್ಕ ಒದಗಿಸುವಿಕೆಯನ್ನು ಸೇರಿಸಿ
- **ಬಿಡುಗಡೆ ಪ್ಯಾಕೇಜಿಂಗ್** — ಪ್ರತ್ಯೇಕ ಕೋರ್ ಮತ್ತು ಸುಧಾರಿತ ZIP‌ಗಳನ್ನು ನಿರ್ಮಿಸಿ, ಎರಡನ್ನೂ GitHub ನಲ್ಲಿ ಪ್ರಕಟಿಸಿ, ಮತ್ತು ಕೇವಲ ಕೋರ್ ಪ್ಯಾಕೇಜ್ ಅನ್ನು WordPress.org ಗೆ ಕಳುಹಿಸಿ

### ಸರಿಪಡಿಸಲಾಗಿದೆ {#fixed}

- **AI ವಿನಂತಿ ವಿಶ್ವಾಸಾರ್ಹತೆ** — ಮಾದರಿ ಆಯ್ಕೆ, ವಿನಂತಿ ಸಮಯಮಿತಿಗಳು, ಡೀಫಾಲ್ಟ್ ಸೆಟಪ್ ಮಾದರಿ, reasoning-ಪಠ್ಯ ನಿರ್ವಹಣೆ, ಮತ್ತು ಅಮಾನ್ಯ ಉಪಕರಣ ಕರೆಗಳಿಗಾಗಿ ಮರುಪ್ರಯತ್ನ ಮಾರ್ಗದರ್ಶನವನ್ನು ಸುಧಾರಿಸಿ
- **ಕ್ಯಾಲೆಂಡರ್ ಮತ್ತು ನೆನಪಿಸುವಿಕೆ ಬಲಪಡಿಸುವಿಕೆ** — Google Calendar token‌ಗಳು ಮತ್ತು ನೆನಪಿಸುವಿಕೆ ನಕಲಿ ನಿವಾರಣೆಯನ್ನು ಬಲಪಡಿಸಿ
- **ಆನ್‌ಬೋರ್ಡಿಂಗ್ ಮತ್ತು ಅನುಮೋದನೆ ಮುಂದುವರಿಕೆಗಳು** — frontend ಆನ್‌ಬೋರ್ಡಿಂಗ್ ಆರಂಭಗಳನ್ನು ಮತ್ತು ದೃಢೀಕರಿಸಿದ ಸಾಮರ್ಥ್ಯ ಮುಂದುವರಿಕೆಗಳನ್ನು ಸರಿಪಡಿಸಿ
- **WordPress.org ಪ್ಯಾಕೇಜಿಂಗ್ ಪರಿಶೀಲನೆ ಸಮಸ್ಯೆಗಳು** — ಕೋರ್ ಬಿಡುಗಡೆಗಾಗಿ ಪ್ಯಾಕೇಜಿಂಗ್ ಪರಿಶೀಲನೆ ಪ್ರತಿಕ್ರಿಯೆಯನ್ನು ಪರಿಹರಿಸಿ

## 1.16.0 — 2026-05-20 ರಂದು ಬಿಡುಗಡೆ {#1160--released-on-2026-05-20}

### ಹೊಸದು {#new-1}

- **ಲೋಗೋ SVG ರಚಿಸುವ ಸಾಮರ್ಥ್ಯ** — Theme Builder ಈಗ namespace-ಸುರಕ್ಷಿತ ಶುದ್ಧೀಕರಣದೊಂದಿಗೆ ಕಸ್ಟಮ್ ಲೋಗೋ SVG‌ಗಳನ್ನು ರಚಿಸಿ ಎಂಬೆಡ್ ಮಾಡಬಹುದು
- **ಅನ್ವೇಷಣೆ ಸಂದರ್ಶನದಲ್ಲಿ ಫೋಟೋ ಅಪ್‌ಲೋಡ್** — Theme Builder ಅನ್ವೇಷಣೆ ಸಂದರ್ಶನದಲ್ಲಿ ಈಗ ಸಮೃದ್ಧ ವಿನ್ಯಾಸ ಸಂದರ್ಭಕ್ಕಾಗಿ ಫೋಟೋ ಅಪ್‌ಲೋಡ್ ಹಂತವಿದೆ
- **ಪ್ಯಾಲೆಟ್ ಕಾಂಟ್ರಾಸ್ಟ್ ಪರಿಶೀಲಿಸುವ ಸಾಮರ್ಥ್ಯ** — theme‌ಗೆ ಅನ್ವಯಿಸುವ ಮೊದಲು ಬಣ್ಣ ಜೋಡಿಗಳು WCAG ಅನುಸರಣೆಯಲ್ಲಿವೆಯೇ ಎಂದು ಪರಿಶೀಲಿಸಿ
- **ಆತಿಥ್ಯ ಮೆನುಗಳು** — Theme Builder ಈಗ ಆತಿಥ್ಯ ವ್ಯವಹಾರಗಳಿಗಾಗಿ ರಚನಾತ್ಮಕ ಆಹಾರ ಮತ್ತು ಪಾನೀಯ ಮೆನು ಪುಟಗಳನ್ನು ರಚಿಸಬಹುದು
- **ಡೆಸ್ಕ್‌ಟಾಪ್ ಮತ್ತು ಮೊಬೈಲ್ ಪೂರ್ವವೀಕ್ಷಣೆ ರೆಂಡರಿಂಗ್** — ವಿನ್ಯಾಸ-ದಿಕ್ಕು ಆಯ್ಕೆ ಮಾಡುವಾಗ ಡೆಸ್ಕ್‌ಟಾಪ್ ಮತ್ತು ಮೊಬೈಲ್ ಸಾಧನಗಳಲ್ಲಿ ನಿಮ್ಮ ವಿನ್ಯಾಸವನ್ನು ಪೂರ್ವವೀಕ್ಷಿಸಿ
- **ನ್ಯಾವಿಗೇಶನ್ ಲೇಬಲ್ ಪ್ಯಾರಾಮೀಟರ್** — Create Menu ಸಾಮರ್ಥ್ಯವು ಈಗ ಪುಟದ ಶೀರ್ಷಿಕೆಯಿಂದ ಪ್ರತ್ಯೇಕವಾದ ವಿಭಿನ್ನ `navigation_label` ಅನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ
- **Tier 1 ಉಪಕರಣ ಲಭ್ಯತೆ** — sd-ai-agent/site-scrape ಈಗ Theme Builder ನಲ್ಲಿ ಡೀಫಾಲ್ಟ್ ಆಗಿ ಲಭ್ಯವಿರುವ Tier 1 ಉಪಕರಣವಾಗಿದೆ

### ಸರಿಪಡಿಸಲಾಗಿದೆ {#fixed-1}

- **AI Client cache** — ಈಗ ವಿನಂತಿಗಳ ನಡುವೆ ಸ್ಥಿರತೆಗಾಗಿ transients ಮೂಲಕ ಬೆಂಬಲಿತವಾಗಿದೆ, ದೀರ್ಘಕಾಲ ನಡೆಯುವ agent ಕಾರ್ಯಗಳಲ್ಲಿ ಡೇಟಾ ನಷ್ಟವನ್ನು ತಡೆಯುತ್ತದೆ
- **Plugin ಸಾಲಿನ ಕ್ರಿಯಾ ಲಿಂಕ್‌ಗಳು** — ಸ್ಪಷ್ಟತೆಗಾಗಿ ಸರಿಪಡಿಸಿ ಮರುಹೆಸರಿಸಲಾಗಿದೆ

## 1.10.0 — 2026-05-05 ರಂದು ಬಿಡುಗಡೆ {#1100--released-on-2026-05-05}

### ಹೊಸದು {#new-2}

- **Tavily ಇಂಟರ್ನೆಟ್ ಹುಡುಕಾಟ** — Brave Search ಜೊತೆಗೆ ಸಮೃದ್ಧ ಇಂಟರ್ನೆಟ್ ಹುಡುಕಾಟ ಫಲಿತಾಂಶಗಳಿಗಾಗಿ Tavily ಅನ್ನು ಹುಡುಕಾಟ ಪೂರೈಕೆದಾರವಾಗಿ ಸೇರಿಸಿ
- **Theme-ಅರಿತ ಒಳನಿರ್ಮಿತ ಕೌಶಲ್ಯಗಳು** — Block Themes, Classic Themes, Kadence Blocks ಮತ್ತು Kadence Theme ಕೌಶಲ್ಯ ಮಾರ್ಗದರ್ಶಿಗಳು ಈಗ plugin ಜೊತೆಗೆ ಬರುತ್ತವೆ
- **ಸೈಟ್ ಬಿಲ್ಡರ್ ಸಂಪರ್ಕ ಫಾರ್ಮ್ ಸಾಮರ್ಥ್ಯ** — chat interface ನಿಂದಲೇ ಯಾವುದೇ ಪುಟಕ್ಕೆ ಸಂಪರ್ಕ ಫಾರ್ಮ್ ಸೇರಿಸಿ

### ಸುಧಾರಿಸಲಾಗಿದೆ {#improved-1}

- **WooCommerce ಏಕೀಕರಣ ಮರುರಚಿಸಲಾಗಿದೆ** — ಉತ್ತಮ ವಿಶ್ವಾಸಾರ್ಹತೆ ಮತ್ತು ಹೊಂದಾಣಿಕೆಗಾಗಿ ಈಗ native WooCommerce API‌ಗಳನ್ನು ಬಳಸುತ್ತದೆ
- **ಪೂರೈಕೆದಾರರ ಪಟ್ಟಿ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ರಿಫ್ರೆಶ್ ಆಗುತ್ತದೆ** — ಯಾವುದೇ plugin ಸಕ್ರಿಯಗೊಳಿಸಿದಾಗ ಅಥವಾ ನಿಷ್ಕ್ರಿಯಗೊಳಿಸಿದಾಗ

### ಸರಿಪಡಿಸಲಾಗಿದೆ {#fixed-2}

- **navigate-to ಸಾಮರ್ಥ್ಯ** — ಕೆಲವು admin ಪುಟಗಳಲ್ಲಿ ಅನಂತ ರೀಲೋಡ್ ಲೂಪ್ ಸರಿಪಡಿಸಲಾಗಿದೆ
- **list-posts ಸಾಮರ್ಥ್ಯ** — ಈಗ category ಮತ್ತು tag ಹೆಸರುಗಳನ್ನು slug‌ಗಳಿಗೆ ಸರಿಯಾಗಿ ಪರಿಹರಿಸುತ್ತದೆ
- **WP-CLI ಕಮಾಂಡ್‌ಗಳು** — ಹಿಂದಿನ ಮರುರಚನೆಯ ನಂತರ ಕಾಣೆಯಾಗಿದ್ದ namespace alias‌ಗಳನ್ನು ಮರುಸ್ಥಾಪಿಸಲಾಗಿದೆ
- **ಈವೆಂಟ್ ಸ್ವಯಂಚಾಲನೆ** — ಸ್ವಯಂಚಾಲನೆ ಟೇಬಲ್‌ಗಳನ್ನು ಇನ್ನೂ ರಚಿಸದಿರುವ ಸೈಟ್‌ಗಳನ್ನು ಸುಗಮವಾಗಿ ನಿರ್ವಹಿಸುತ್ತದೆ
- **memory-save ಸಾಮರ್ಥ್ಯ** — ಈಗ system instruction builder ನಲ್ಲಿ ಸರಿಯಾದ namespace prefix ಬಳಸುತ್ತದೆ
- **Scalar ಉಪಕರಣ ಫಲಿತಾಂಶಗಳು** — ಈಗ AI ಗೆ ಹಿಂತಿರುಗಿಸುವ ಮೊದಲು ಸರಿಯಾಗಿ wrap ಮಾಡಲಾಗುತ್ತದೆ
- **ಬಳಕೆ ಅಂಕಿಅಂಶಗಳು** — ಹಳೆಯ ಆವೃತ್ತಿಗಳಿಂದ upgrade ಮಾಡಿದಾಗ legacy ability key format ಅನ್ನು ಈಗ ಸರಿಯಾಗಿ ನಿರ್ವಹಿಸುತ್ತವೆ
