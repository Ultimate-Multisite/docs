# AffiliateWP

Multisite Ultimate: AffiliateWP Integration

ನಿಮ್ಮ Ultimo Network ನ client base ಅನ್ನು ಬೆಳೆಸಲು ಶಕ್ತಿಯುತ AffiliateWP ಅನ್ನು ಬಳಸಿ!

## Installation {#installation}

1. 'ultimate-multisite-git ' ಅನ್ನು '/wp-content/plugins/' ಡೈರೆಕ್ಟರಿಗೆ ಅಪ್‌ಲೋಡ್ ಮಾಡಿ
2. WordPress ನಲ್ಲಿರುವ 'Plugins' ಮೆನುವಿನ ಮೂಲಕ plugin ಅನ್ನು ಆಕ್ಟಿವೇಟ್ ಮಾಡಿ
3. plugin ಅನ್ನು ಸೆಟ್ ಅಪ್ ಮಾಡಲು ಹಂತ ಹಂತದ Wizard ಅನ್ನು ಅನುಸರಿಸಿ

## Usage Guide {#usage-guide}

[AffiliateWP](https://affiliatewp.com/) ಎಂಬುದು ನಿಮ್ಮ ಆನ್‌ಲೈನ್ ವ್ಯವಹಾರಕ್ಕಾಗಿ ಅಫಿಲಿಯೇಟ್ ಪ್ರೋಗ್ರಾಂ ಅನ್ನು ನಿರ್ಮಿಸುವಾಗ ಪ್ರಮುಖ ಪರಿಹಾರವಾಗಿ ಮಾರ್ಪಟ್ಟಿದೆ. Ultimate Multisite: AffiliateWP ಎಂಬುದು ನಿಮ್ಮ ಪ್ರೀಮಿಯಂ ನೆಟ್‌ವರ್ಕ್ ಗಾಗಿ ಇದೇ ರೀತಿಯದನ್ನು ನಿರ್ಮಿಸಲು ನಿಮಗೆ ಸಹಾಯ ಮಾಡುವ ಸೇತುವೆಯಾಗಿದೆ.

## Getting Started {#getting-started}

ಮೊದಲನೆಯದಾಗಿ, ನೀವು [**AffiliateWP**](https://affiliatewp.com/) ಅನ್ನು **ಆಕ್ಟಿವ್** ಮಾಡಬೇಕಾಗುತ್ತದೆ! ನೀವು **ಇದರೊಂದಿಗೆ ಇಂಟಿಗ್ರೇಟ್ ಆಗುವ plugin** ಇಲ್ಲದಿದ್ದರೆ, ನೀವು ನಮ್ಮ add-on ಅನ್ನು ಬಳಸಲು ಸಾಧ್ಯವಾಗುವುದಿಲ್ಲ. ನಿಮ್ಮ client ಗಳು ತಮ್ಮ সাইಟ್‌ಗಳಲ್ಲೂ ಇದನ್ನು ಬಳಸಲು ಯೋಜಿಸುತ್ತಿಲ್ಲದಿದ್ದರೆ, AffiliateWP ಅನ್ನು ನಿಮ್ಮ ಮುಖ್ಯ সাইಟ್‌ನಲ್ಲಿ ಮಾತ್ರ ಆಕ್ಟಿವ್ ಇರಬೇಕು.

ಅಲ್ಲದೆ, ನೀವು **Ultimate Multisite** ಮತ್ತು **Ultimate Multisite: AffiliateWP** ಎರಡನ್ನೂ **network-wide** ಆಕ್ಟಿವೇಟ್ ಮಾಡಬೇಕಾಗುತ್ತದೆ.

## Adding Ultimate Multisite as an AffiliateWP Integration {#adding-ultimate-multisite-as-an-affiliatewp-integration}

ನಿಮ್ಮ ಮುಖ್ಯ সাইಟ್‌ನಲ್ಲಿ, Affiliates Settings ಗೆ ಹೋಗಿ, ಮತ್ತು Integrations ಟ್ಯಾಬ್‌ನಲ್ಲಿ, Ultimate Multisite ಅನ್ನು ಚೆಕ್ ಮಾಡಿ. ಇದು [AffiliateWP](https://affiliatewp.com/) ಗೆ Ultimate Multisite sign-up flow ನಿಂದ ವೀಸಟ್‌ಗಳು ಮತ್ತು conversion ಗಳನ್ನು ಟ್ರ್ಯಾಕ್ ಮಾಡಲು ಪ್ರಾರಂಭಿಸಲು ಹೇಳುತ್ತದೆ.

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_ಕೆಳಭಾಗದಲ್ಲಿರುವ Ultimate Multisite ಬಾಕ್ಸ್ ಅನ್ನು ಚೆಕ್ ಮಾಡಿ ಮತ್ತು ಸೇವ್ ಮಾಡಿ_

ಅಷ್ಟೇ ಸಾಕು. ರೆಫರಲ್ ಲಿಂಕ್‌ಗಳಿಂದ ಬರುವ ವೀಸಟ್‌ಗಳನ್ನು ಟ್ರ್ಯಾಕ್ ಮಾಡಲಾಗುತ್ತದೆ ಮತ್ತು ಆ ಬಳಕೆದಾರರು ನಂತರ ತಮ್ಮ ಮೊದಲ ಪಾವತಿಯನ್ನು ಮಾಡಿದಾಗ, conversion ಅನ್ನು ರಚಿಸಲಾಗುತ್ತದೆ ಮತ್ತು conversion ಗಾಗಿ ಜವಾಬ್ದಾರರಾಗಿರುವ affiliate ಗೆ ಕಮಿಷನ್ ಇಡಲಾಗುತ್ತದೆ.

ಆದಾಗ್ಯೂ, ಹೊಸದಾಗಿ ರಚಿಸಲಾದ ಬಳಕೆದಾರರಿಂದ ಬಂದ ಮೊದಲ ಪಾವತಿಯನ್ನು ಮಾತ್ರ ಟ್ರ್ಯಾಕ್ ಮಾಡಲಾಗುತ್ತದೆ. ಪುನರಾವರ್ತಿತ ರೆಫರಲ್ ಪಾವತಿಗಳನ್ನು (recurring referral payments) ಸಕ್ರಿಯಗೊಳಿಸಲು, ನಿಮಗೆ [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) ಬೇಕಾಗುತ್ತದೆ.

## Enabling Recurring Referrals {#enabling-recurring-referrals}

[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) ಅನ್ನು ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡಿದ ನಂತರ, Affiliates Settings ನ Recurring Referrals ಟ್ಯಾಬ್‌ಗೆ ಹೋಗಿ, recurring referrals ಬಳಕೆಯನ್ನು ಆನ್ ಮಾಡಿ.

<!-- Screenshot unavailable: AffiliateWP Recurring Referrals settings with Enable Recurring Referrals toggle -->

_ಇಷ್ಟು ಸುಲಭ: Recurring Referrals ಅನ್ನು ಆನ್ ಮಾಡಿ ಮತ್ತು ನೀವು ಸಿದ್ಧರಿದ್ದೀರಿ!_

ಮತ್ತೊಮ್ಮೆ, ಅಷ್ಟೇ ಸಾಕು!

ಈಗ, ಯಾವುದೇ ರೆಫರ್ ಮಾಡಿದ ಖಾತೆಯ ಮೇಲೆ ಹೊಸ ಪಾವತಿಯನ್ನು ಸ್ವೀಕರಿಸಿದ ಪ್ರತಿ ಬಾರಿಯೂ, ಜವಾಬ್ದಾರರಾಗಿರುವ affiliate ಸ್ವಯಂಚಾಲಿತವಾಗಿ ತನ್ನದೇ ಆದ ಕಟ್ ಪಡೆಯುತ್ತದೆ!
