# Action: wp_ultimo_affiliatewp_load

ಎಲ್ಲಾ ಅವಲಂಬನೆಗಳು (dependencies) ಲೋಡ್ ಆದಾಗ ಇದು ಕಾರ್ಯಗತಗೊಳ್ಳುತ್ತದೆ.

ಪ್ಲಗ್ಇನ್ (plugin) developers ಗಳಿಗೆ ಹೊಸ ಕಾರ್ಯಚಟುವಟಿಕೆಗಳನ್ನು (functionality) ಸೇರಿಸಲು ಇದು ಅನುಮತಿಸುತ್ತದೆ. ಉದಾಹರಣೆಗೆ, ಹೊಸ ಹೋಸ್ಟಿಂಗ್ (Hosting) ಪроваೈಡರ್‌ಗಳಿಗೆ ಬೆಂಬಲ ನೀಡುವುದು ಇತ್ಯಾದಿ.
