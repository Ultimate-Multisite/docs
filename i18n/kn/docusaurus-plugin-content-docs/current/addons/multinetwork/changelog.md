# ಬಹು-ನೆಟ್‌ವರ್ಕ್ ಬದಲಾವಣೆಗಳ ವಿವರ

### 1.3.0 {#130}
* ಹೊಸದಾಗಿ: ನೆಟ್‌ವರ್ಕ್ ಟೆಂಪ್ಲೇಟ್ ಪೂರ್ವವೀಕ್ಷಣೆಕಾರ (Network template previewer) — ಖರೀದಿಸುವ ಮೊದಲು ಲೈವ್ ಪ್ಯಾನೆಲ್‌ನಲ್ಲಿ ನೆಟ್‌ವರ್ಕ್ ಟೆಂಪ್ಲೇಟ್‌ಗಳನ್ನು ಬ್ರೌಸ್ ಮಾಡಿ ಮತ್ತು ಪೂರ್ವವೀಕ್ಷಣೆ ಮಾಡಬಹುದು.
* ಹೊಸದಾಗಿ: ಚೆಕ್‌ಔಟ್‌ನಲ್ಲಿ ನೆಟ್‌ವರ್ಕ್ ಟೆಂಪ್ಲೇಟ್ ಪಿಕರ್ — ನೋಂದಣಿ ಚೆಕ್‌ಔಟ್ ಪ್ರಕ್ರಿಯೆಯ ಸಮಯದಲ್ಲಿ ಗ್ರಾಹಕರು ನೆಟ್‌ವರ್ಕ್ ಟೆಂಪ್ಲೇಟ್ ಅನ್ನು ಆಯ್ಕೆ ಮಾಡಬಹುದು.
* ಹೊಸದಾಗಿ: ನೆಟ್‌ವರ್ಕ್‌ಗಳ ನಡುವೆ ಮ್ಯಾಜಿಕ್ ಲಿಂಕ್ SSO — ಮ್ಯಾಜಿಕ್-ಲಿಂಕ್ प्रमाणीकरणದ ಮೂಲಕ ಗ್ರಾಹಕರು ಉಪ-ನೆಟ್‌ವರ್ಕ್‌ಗಳನ್ನು ಸುಗಮವಾಗಿ (seamlessly) அணுகಬಹುದು.
* ಹೊಸದಾಗಿ: ನೆಟ್‌ವರ್ಕ್ മീഡിയ ಕ್ಲೋನಿಂಗ್ — ಟೆಂಪ್ಲೇಟ್‌ನಿಂದ ನೆಟ್‌ವರ್ಕ್‌ಗಳನ್ನು ಕ್ಲೋನ್ ಮಾಡುವಾಗ ಸೈಟ್ മീഡിയ ಅನ್ನು ಸರಿಯಾಗಿ ನಕಲಿಸಲಾಗುತ್ತದೆ.
* ಸರಿಪಡಿಸಲಾಗಿದೆ: ನೆಟ್‌ವರ್ಕ್ ಕ್ಲೋನಿಂಗ್ ಈಗ ಮಲ್ಟಿ-ಟೆನನ್ಸಿ ಆಡ್-ಆನ್ ಇಲ್ಲದೆ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ; ಕ್ಲೋನ್ ಮಾಡಿದ ನಂತರ ಸೂಪರ್ ಅಡ್ಮಿನ್ ಪ್ರವ権ಗಳನ್ನು ಮರು-ನೀಡಲಾಗುತ್ತದೆ.
* ಸರಿಪಡಿಸಲಾಗಿದೆ: ನೆಟ್‌ವರ್ಕ್ ರಚನೆಯು ಈಗ WooCommerce ಮತ್ತು ಎಲ್ಲಾ ಪಾವತಿ 게ಟ್‌ವೇಯರ್‌ಗಳೊಂದಿಗೆ ಹೊಂದಾಣಿಕೆಯಾಗಿದೆ.
* ಸರಿಪಡಿಸಲಾಗಿದೆ: ಫಾಲ್‌ಬ್ಯಾಕ್ ಡೊಮೇನ್ ರಚನೆಯಲ್ಲಿ ಈಗ ಪೋರ್ಟ್ ಸಂಖ್ಯೆಯನ್ನು ಸೇರಿಸಲಾಗಿದೆ.
* ಸರಿಪಡಿಸಲಾಗಿದೆ: ಟೆಂಪ್ಲೇಟ್ ಪಿಕರ್ CSS ಅನ್ನು ಎಲ್ಲಾ ಥೀಮ್‌ಗಳಲ್ಲಿ ಸರಿಯಾದ ರೆಂಡರಿಂಗ್‌ಗಾಗಿ enqueued stylesheet ಗೆ ಸ್ಥಳಾಂತರಿಸಲಾಗಿದೆ.
* ಸರಿಪಡಿಸಲಾಗಿದೆ: ಸೈಟ್ ಕ್ವೆರಿ ಸ್ಕೋಪ್‌ನಲ್ಲಿ ಅಸಂಘಟಿತ (`undefined`) `network_id` ಕೀಗೆ ರಕ್ಷಣೆ ನೀಡಲಾಗಿದೆ.
* ಸರಿಪಡಿಸಲಾಗಿದೆ: ಸಂದರ್ಭ ಬದಲಾವಣೆಯ ಮೊದಲು ನೆಟ್‌ವರ್ಕ್ ಅನುಮತಿ ಪರಿಶೀಲನೆಯು `wpdb` ಟೇಬಲ್ ಆಸ್ತಿ ದೋಷಗಳನ್ನು ತಡೆಯುತ್ತದೆ.
