# Action: update_network {#action-updatenetwork}

| $args | `array` | ನೆಟ್‌ವರ್ಕ್ ಆರ್ಗ್ಯುಮೆಂಟ್‌ಗಳ ಅಸೋಸಿಯೇಟಿವ್ ಅರೇ. |
