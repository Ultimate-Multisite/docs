# Action: switch_network

ಪ್ರಸ್ತುತ ನೆಟ್‌ವರ್ಕ್ ಸಂದರ್ಭವನ್ನು (context) ಬದಲಾಯಿಸಿದಾಗ ಇದು ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $new_network_id | `int` | ಬದಲಾಯಿಸಲಾಗುತ್ತಿರುವ ನೆಟ್‌ವರ್ಕ್‌ನ ಐಡಿ. |
| $old_network_id | `int` | ಹಿಂದೆ ಪ್ರಸ್ತುತವಾಗಿದ್ದ ನೆಟ್‌ವರ್ಕ್‌ನ ಐಡಿ. |
