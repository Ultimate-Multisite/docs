# Action: move_site

ಒಂದು ಸೈಟ್ ಅನ್ನು ಹೊಸ ನೆಟ್‌ವರ್ಕ್‌ಗೆ ಸ್ಥಳಾಂತರಿಸಿದ ನಂತರ ಇದು ಕಾರ್ಯಗತಗೊಳ್ಳುತ್ತದೆ.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $site_id | `int` | ಸ್ಥಳಾಂತರಿಸಲಾದ ಸೈಟ್‌ನ ID. |
| $old_network_id | `int` | ಸೈಟ್‌ನ ಮೂಲ ನೆಟ್‌ವರ್ಕ್‌ನ ID. |
| $new_network_id | `int` | ಸೈಟ್ ಅನ್ನು ಸ್ಥಳಾಂತರಿಸಲಾದ ನೆಟ್‌ವರ್ಕ್‌ನ ID. |
