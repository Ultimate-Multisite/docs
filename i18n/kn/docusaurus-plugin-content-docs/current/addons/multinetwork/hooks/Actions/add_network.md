# Action: add_network

ಹೊಸ ನೆಟ್‌ವರ್ಕ್ ಅನ್ನು ಸೇರಿಸಿದ ನಂತರ ಈ ಆಕ್ಷನ್ ಕಾರ್ಯಗತವಾಗುತ್ತದೆ.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $new_network_id | `int` | ಸೇರಿಸಲಾದ ನೆಟ್‌ವರ್ಕ್‌ನ ಐಡಿ (ID). |
| $r | `array` | ನೆಟ್‌ವರ್ಕ್ ಆರ್ಗ್ಯುಮೆಂಟ್‌ಗಳ ಸಂಪೂರ್ಣ ಅಸೋಸಿಯೇಟಿವ್ ಆರೇ. |
