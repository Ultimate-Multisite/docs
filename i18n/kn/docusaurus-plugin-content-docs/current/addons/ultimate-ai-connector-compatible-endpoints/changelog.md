# Ultimate AI Connector for Compatible Endpoints ಬದಲಾವಣೆಗಳ ಪಟ್ಟಿ

## Version 2.0.0 — ಬಿಡುಗಡೆಯಾದ ದಿನಾಂಕ: 2026-04-24 {#version-200--released-on-2026-04-24}

* **ಹೊಸದಾಗಿ:** ಬಹು-ಪ್ರಾಡೈವರ್ ಬೆಂಬಲ — ನೀವು ಅನೇಕ AI ಎಂಡ್‌ಪಾಯಿಂಟ್‌ಗಳನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಬಹುದು ಮತ್ತು ಒದಗಿಸುವವರು (providers) ನಡುವೆ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಹಿಂತಿರುಗುವಿಕೆ (automatic fallback) ಯೊಂದಿಗೆ ವಿನಂತಿಗಳನ್ನು (requests) ರೂಟ್ ಮಾಡಬಹುದು.
* **ಸರಿಪಡಿಸಲಾಗಿದೆ:** ಪ್ರತಿ ಪ್ರಾಡೈವರ್‌ಗೆ ಸರಿಯಾದ ಪ್ರಾಡೈವರ್ IDಗಳು, ನೋಂದಣಿ URLಗಳು ಮತ್ತು HTTP ಫಿಲ್ಟರ್ ವ್ಯಾಪ್ತಿ (scope) ಯೊಂದಿಗೆ ಬಹು-ಪ್ರಾಡೈವರ್ SDK ಏಕೀಕರಣ.
* **ಸರಿಪಡಿಸಲಾಗಿದೆ:** ಹೊಸ ಪ್ರಾಡೈವರ್ ಕಾರ್ಡ್ ಈಗ ಸೇರಿಸಿದ ಕೂಡಲೇ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ವಿಸ್ತರಿಸುತ್ತದೆ; ಪ್ಲಗ್ಇನ್ ಅಪ್ಡೇಟ್ ಆದಾಗ ಸ್ಕ್ರಿಪ್ಟ್ ಕ್ಯಾಶ್ ಬಸ್ಟಿಂಗ್ (script cache busting) ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ.
* **ಸರಿಪಡಿಸಲಾಗಿದೆ:** `eval()` ಗಾಗಿ ಡೈನಾಮಿಕ್ ಪ್ರಾಡೈವರ್ ಕ್ಲಾಸ್ नेमಸ್ಪೇಸ್ (namespace) ಮತ್ತು ಹೊಸದಾಗಿ ಸೇರಿಸಲಾದ ಪ್ರಾಡೈವರ್‌ಗಳಿಗೆ ಆಟೋ-ಎಕ್ಪಾಂಡ್ ವರ್ತನೆ.
* **ಸರಿಪಡಿಸಲಾಗಿದೆ:** WordPress 6.9+ ಹೊಂದಾಣಿಕೆಗಾಗಿ ಸ್ಥಿರವಾದ Card/CardBody/CardHeader/CardDivider 컴ಪೋನೆಂಟ್‌ಗಳನ್ನು ಬಳಸಲಾಗಿದೆ (ಇವುಗಳು ಇನ್ನು ಮುಂದೆ ಪ್ರಾಯೋಗಿಕವಾಗಿಲ್ಲ).
* **ಸರಿಪಡಿಸಲಾಗಿದೆ:** ಪ್ರಾಡೈವರ್‌ಗಳನ್ನು ಎಳೆದು ಮರು-ಸರಿಹೊಂದಿಸಲು (drag-to-reorder) ಬಳಸುತ್ತಿದ್ದ ಅನಿರ್ದಿಷ್ಟ DragHandle ಅನ್ನು ಯುನಿಕೋಡ್ ಗ್ರಿಪ್ ಐಕಾನ್‌ನೊಂದಿಗೆ ಬದಲಾಯಿಸಲಾಗಿದೆ.
* **ಸರಿಪಡಿಸಲಾಗಿದೆ:** ಪ್ರತಿ ಪುಟದ ಲೋಡ್ ಆದಾಗ décliring ಆಗುತ್ತಿದ್ದ ಬ್ಲಾಕಿಂಗ್ HTTP ವಿನಂತಿಯನ್ನು ತೆಗೆದುಹಾಕಲಾಗಿದೆ.
* **ಸುಧಾರಿಸಲಾಗಿದೆ:** GitHub Actions ಕಾರ್ಯಪ್ರವಾಹಗಳನ್ನು (workflows) Node.js 24 ಗೆ ಅಪ್‌ಗ್ರೇಡ್ ಮಾಡಲಾಗಿದೆ.
