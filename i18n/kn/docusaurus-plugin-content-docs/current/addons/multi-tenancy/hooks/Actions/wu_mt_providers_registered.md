# Action: wu_mt_providers_registered

Default providers register ಆದ ನಂತರ ಇದು ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ (Fires).

ಕಸ್ಟಮ್ ಪ್ರೊವೈಡರ್‌ಗಳನ್ನು (custom providers) ನೋಂದಾಯಿಸಲು ಈ hook ಅನ್ನು ಬಳಸಿ.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $manager | `\Provider_Manager` | ಪ್ರೊವೈಡರ್ ಮ್ಯಾನೇಜರ್ ಇನ್‌ಸ್ಟಾನ್ಸ್ (provider manager instance). |

- 1.0.0
