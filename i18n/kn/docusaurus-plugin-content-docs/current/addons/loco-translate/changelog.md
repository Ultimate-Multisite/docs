# Loco Translate ಬದಲಾವಣೆಗಳ ವಿವರ

### 1.0.1 {#101}
* ಇತ್ತೀಚest Loco Translate ಆವೃತ್ತಿಯೊಂದಿಗೆ ಹೊಂದಾಣಿಕೆ ಸುಧಾರಣೆ
* ಉತ್ತಮ ಭದ್ರತೆಗಾಗಿ ಸುಧಾರಿತ ಪ್ರವೇಶ ನಿಯಂತ್ರಣ (Enhanced access control)
* ಕಾರ್ಯಕ್ಷಮತೆ ಆಪ್ಟಿಮೈಸೇಶನ್‌ಗಳು

### 1.0.0 {#100}
* ಮೊದಲ ಬಿಡುಗಡೆ
* ಸೈಟ್-ನಿರ್ದಿಷ್ಟ ಅನುವಾದ ಡೈರೆಕ್ಟರಿಗಳು (Site-specific translation directories)
* ಅನುವಾದ ಆದ್ಯತೆ ವ್ಯವಸ್ಥೆ (Translation priority system)
* ಪ್ರವೇಶ ನಿಯಂತ್ರಣದ ಅನುಷ್ಠಾನ (Access control implementation)
* Loco Translate ನೊಂದಿಗೆ ಸುಗಮ UI ಏಕೀಕರಣ (Seamless UI integration)
