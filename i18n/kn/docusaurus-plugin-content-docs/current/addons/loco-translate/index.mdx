# Loco Translate

The Ultimate Multisite: Loco Translate Integration addon ನಿಮ್ಮ WordPress multisite network ಗೆ site-specific translation ಸಾಮರ್ಥ್ಯಗಳನ್ನು ತರುತ್ತದೆ. ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿರುವ ವಿವಿಧ ಸೈಟ್‌ಗಳಿಗೆ ಒಂದೇ plugin ಅಥವಾ theme ನ ವಿಭಿನ್ನ ಅನುವಾದಗಳು ಬೇಕಾಗಿದ್ದರೆ, ಈ addon ಅತ್ಯುತ್ತಮ ಪರಿಹಾರವಾಗಿದೆ.

**ಈ Addon ಏನು ಮಾಡುತ್ತದೆ**

ಈ addon Ultimate Multisite ಅನ್ನು ಜನಪ್ರಿಯ Loco Translate plugin ನೊಂದಿಗೆ ಸುಲಭವಾಗಿ ಜೋಡಿಸುತ್ತದೆ. ಇದು ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿರುವ ಪ್ರತಿಯೊಂದು ಸಬ್‌ಸೈಟ್‌ಗೂ ತನ್ನದೇ ಆದ ಕಸ್ಟಮ್ ಅನುವಾದಗಳನ್ನು ನಿರ್ವಹಿಸಲು ಅನುವು ಮಾಡಿಕೊಡುತ್ತದೆ. ನೀವು ಬಹು-ಪ್ರಾದೇಶಿಕ ವೇದಿಕೆ (multi-regional platform) ನಡೆಸುತ್ತಿರಲಿ, ವಿಭಿನ್ನ ಭಾಷಾ ಸಮುದಾಯಗಳಿಗೆ ಸೇವೆ ಸಲ್ಲಿಸುತ್ತಿರಲಿ, ಅಥವಾ ಕೇವಲ ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿ ಅನುವಾದದ ಹೊಂದಾಣಿಕೆ ಬೇಕಾಗುತ್ತಿರಲಿ, ಈ addon ಅದನ್ನು ಸುಲಭವಾಗಿಸುತ್ತದೆ.

**ಯಾರಿಗೆ ಸೂಕ್ತ:**

* ವಿಭಿನ್ನ ಸ್ಥಳೀಕರಣ (localization) ಅಗತ್ಯತೆಗಳನ್ನು ಹೊಂದಿರುವ ಬಹು-ಪ್ರಾದೇಶಿಕ ವ್ಯವಹಾರಗಳು
* ವೈವಿಧ್ಯಮಯ ಸಮುದಾಯಗಳಿಗೆ ಸೇವೆ ಸಲ್ಲಿಸುವ ಶೈಕ್ಷಣಿಕ ಸಂಸ್ಥೆಗಳು
* ಬಹುಭಾಷಾ ಬೆಂಬಲ ನೀಡುವ SaaS ವೇದಿಕೆಗಳು
* ಅನೇಕ client ಸೈಟ್‌ಗಳನ್ನು ನಿರ್ವಹಿಸುವ ವೆಬ್ ಏಜೆನ್ಸಿಗಳು
* ಅನುವಾದದ ಹೊಂದಾಣಿಕೆ ಅಗತ್ಯವಿರುವ ಯಾವುದೇ multisite network

## Installation

1. Ultimate Multisite ಅನ್ನು ಅಳವಡಿಸಲಾಗಿದೆ ಮತ್ತು ನೆಟ್‌ವರ್ಕ್ ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಲಾಗಿದೆ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ
2. Loco Translate plugin ಅನ್ನು ಅಳವಡಿಸಿ ಮತ್ತು ಸಕ್ರಿಯಗೊಳಿಸಿ
3. ಈ addon ಅನ್ನು ನಿಮ್ಮ plugins directory ಗೆ ಅಪ್‌ಲೋಡ್ ಮಾಡಿ
4. Loco Translate Integration addon ಅನ್ನು ನೆಟ್‌ವರ್ಕ್ ಮಟ್ಟದಲ್ಲಿ ಸಕ್ರಿಯಗೊಳಿಸಿ
5. ಯಾವುದೇ ಸಬ್‌ಸೈಟ್‌ನಲ್ಲಿ Loco Translate ಮೂಲಕ site-specific ಅನುವಾದಗಳನ್ನು ರಚಿಸಲು ಪ್ರಾರಂಭಿಸಿ

ಅಷ್ಟೇ! ಯಾವುದೇ ಹೆಚ್ಚುವರಿ ಕಾನ್ಫಿಗರೇಶನ್ ಇಲ್ಲದೆ ಏಕೀಕರಣವು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ.

## Frequently Asked Questions

## ಅಳವಡಿಕೆಯ ನಂತರ ನಾನು ಏನಾದರೂ ಕಾನ್ಫಿಗರ್ ಮಾಡಬೇಕೇ?

ಇಲ್ಲ! ಏಕೀಕರಣವು ಸ್ವಯಂಚಾಲಿತವಾಗಿದೆ. addon ಮತ್ತು Loco Translate ಎರಡೂ ಸಕ್ರಿಯಗೊಂಡ ನಂತರ, ಸಬ್‌ಸೈಟ್‌ಗಳು ತಕ್ಷಣವೇ ತಮ್ಮದೇ ಆದ ಕಸ್ಟಮ್ ಅನುವಾದಗಳನ್ನು ರಚಿಸಲು ಪ್ರಾರಂಭಿಸಬಹುದು.

## ಒಂದು ಸಬ್‌ಸೈಟ್ ಕಸ್ಟಮ್ ಅನುವಾದಗಳನ್ನು ರಚಿಸದಿದ್ದರೆ ಏನಾಗುತ್ತದೆ?

ಸೈಟ್ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ನೆಟ್‌ವರ್ಕ್-ವ್ಯಾಪಿ ಅನುವಾದಗಳನ್ನು ಬದಲಿ (fallback) ಆಗಿ ಬಳಸುತ್ತದೆ. ಏನೂ ಕೆಡುವುದಿಲ್ಲ, ಮತ್ತು ಎಲ್ಲವೂ ಸುಗಮವಾಗಿ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ.

## ಸಬ್‌ಸೈಟ್‌ಗಳು ನೆಟ್‌ವರ್ಕ್-ವ್ಯಾಪಿ ಅನುವಾದಗಳನ್ನು ಪ್ರವೇಶಿಸಬಹುದೇ?

ಸಾಮಾನ್ಯ site adminಗಳು ತಮ್ಮದೇ ಸೈಟ್‌ನ ಅನುವಾದಗಳನ್ನು ಮಾತ್ರ ನೋಡಬಹುದು ಮತ್ತು ಸಂಪಾದಿಸಬಹುದು. Super adminಗಳು ಅವರು ಯಾವ ಸೈట్‌ವನ್ನು ನೋಡುತ್ತಿದ್ದಾರೆ ಎಂಬುದರ ಆಧಾರದ ಮೇಲೆ ನೆಟ್‌ವರ್ಕ್-ವ್ಯಾಪಿ ಮತ್ತು site-specific ಅನುವಾದಗಳನ್ನು ಪ್ರವೇಶಿಸಬಹುದು.

## ಇದು plugins ಮತ್ತು themes ಎರಡಕ್ಕೂ ಕೆಲಸ ಮಾಡುತ್ತದೆಯೇ?
