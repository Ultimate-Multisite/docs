# VAT ಬದಲಾವಣೆಗಳ ದಾಖಲೆ {#vat-changelog}

Version 1.0.7 - ಬಿಡುಗಡೆ ದಿನಾಂಕ 2026-02-03

* Fix: ಜರ್ಮನ್ (DE) VAT ಸಂಖ್ಯೆಗಳು VIES SOAP service MS_MAX_CONCURRENT_REQ ದೋಷಗಳಿಂದಾಗಿ ಪರಿಶೀಲನೆಯಲ್ಲಿ ವಿಫಲವಾಗುತ್ತಿದ್ದವು. ಹೆಚ್ಚು ವಿಶ್ವಾಸಾರ್ಹ ಪರಿಶೀಲನೆಗಾಗಿ ಹಳೆಯ SOAP API ಯಿಂದ ಅಧಿಕೃತ EU VIES REST API ಗೆ ಬದಲಾಯಿಸಲಾಗಿದೆ.
* Fix: VIES ಸೇವೆ ತಾತ್ಕಾಲಿಕವಾಗಿ ಲಭ್ಯವಿಲ್ಲದಿದ್ದಾಗಲೂ VAT ಸಂಖ್ಯೆಗಳನ್ನು ಈಗ ಸ್ವೀಕರಿಸಲಾಗುತ್ತದೆ, ಹಿಂದೆ ತಪ್ಪಾಗಿ ತಿರಸ್ಕರಿಸಲಾಗುತ್ತಿತ್ತು. ನಂತರ ಪರಿಶೀಲನೆಗಾಗಿ ಒಂದು ಲಾಗ್ ಎಂಟ್ರಿ ಅನ್ನು ರಚಿಸಲಾಗುತ್ತದೆ.
* Fix: VAT/Tax ID ಫೀಲ್ಡ್ ಅನ್ನು ಈಗ ಎಲ್ಲಾ ದೇಶಗಳಿಗೆ ಗೋಚರಿಸುವಂತೆ ಮಾಡಲಾಗಿದೆ, ಕೇವಲ EU ಸದಸ್ಯರಿಗೆ ಮಾತ್ರವಲ್ಲ. Non-EU tax ID ಗಳನ್ನು (ಉದಾಹರಣೆಗೆ ಸ್ವಿಟ್ಸರ್‌ನ CHE ಸಂಖ್ಯೆಗಳು) VIES ಪರಿಶೀಲನೆ ಇಲ್ಲದಿದ್ದರೂ ಇನ್‌ವಾಯ್ಸ್ ಪ್ರದರ್ಶನಕ್ಕಾಗಿ ಸಂಗ್ರಹಿಸಲಾಗುತ್ತದೆ.
* Fix: ಗ್ರೀಸ್ (GR), ಮೊನಕೋ (MC), ಮತ್ತು ಐಲ್ ಆಫ್ ಮ್ಯಾನ್ (IM) ಗಾಗಿ VAT ಪ್ರಿಫಿಕ್ಸ್ ಹುಡುಕಾಟವು ತಪ್ಪಾದ ಅರೇ ಲೂಕ್ಅಪ್ ತರ್ಕದಿಂದಾಗಿ ಕೆಡದಿತ್ತು.
* Improvement: Site Exporter addon ಜೊತೆಗೆ ಬಳಸಿದಾಗ ಫೇಟಲ್ ಎರೆರ್ಸ್‌ಗೆ ಕಾರಣವಾಗಬಹುದಾದ ಅನಗತ್ಯ wp-cli-bundle dev ಅವಲಂಬನೆ (dependency) ಅನ್ನು ತೆಗೆದುಹಾಕಲಾಗಿದೆ.
* Improvement: ಸಮಗ್ರ ಯೂನಿಟ್ ಟೆಸ್ಟ್ ಸೂಟ್ ಅನ್ನು (53 tests) ಸೇರಿಸಲಾಗಿದೆ.
