# Action: wp_ultimo_vat_load {#action-wpultimovatload}

ಎಲ್ಲಾ ಅವಲಂಬನೆಗಳು (dependencies) ಲೋಡ್ ಆದಾಗ ಇದು ಕಾರ್ಯಗತಗೊಳ್ಳುತ್ತದೆ.

ಪ್ಲಗ್ಇನ್ ಡೆವಲಪರ್‌ಗಳು ಹೊಸ ಕಾರ್ಯಕ್ಷಮತೆಯನ್ನು (functionality) ಸೇರಿಸಲು ಇದನ್ನು ಅನುಮತಿಸುತ್ತದೆ. ಉದಾಹರಣೆಗೆ, ಹೊಸ ಹೋಸ್ಟಿಂಗ್ ಪроваೈಡರ್‌ಗಳಿಗೆ ಬೆಂಬಲ ನೀಡುವುದು ಇತ್ಯಾದಿ.

- 1.0.0
