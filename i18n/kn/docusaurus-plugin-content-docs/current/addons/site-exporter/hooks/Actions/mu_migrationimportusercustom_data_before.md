# Action: mu_migration/import/user/custom_data_before

ಕಸ್ಟಮ್ ಬಳಕೆದಾರ ಡೇಟಾವನ್ನು ರಫ್ತು ಮಾಡುವ ಮೊದಲು ಇದು ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $user_data | `array` | ಬಳಕೆದಾರ ಡೇಟಾ ಇರುವ ಆರೇ (array). |
| $user | `\WP_User` | ಬಳಕೆದಾರ ಆಬ್ಜೆಕ್ಟ್ (user object). |
