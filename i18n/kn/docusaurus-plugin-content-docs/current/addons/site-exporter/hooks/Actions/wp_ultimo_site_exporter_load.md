# Action: wp_ultimo_site_exporter_load

ಎಲ್ಲಾ ಅವಲಂಬನೆಗಳು (dependencies) ಲೋಡ್ ಆದಾಗ ಇದು ಕಾರ್ಯಗತಗೊಳ್ಳುತ್ತದೆ.

ಇದು ಪ್ಲಗ್ಇನ್ ಅಭಿವರ್ಧಕರಿಗೆ ಹೊಸ ಕಾರ್ಯನಿರ್ವಹಣೆಯನ್ನು (functionality) ಸೇರಿಸಲು ಅವಕಾಶ ನೀಡುತ್ತದೆ. ಉದಾಹರಣೆಗೆ, ಹೊಸ ಹೋಸ್ಟಿಂಗ್ ಪೂರೈಕೆದಾರರಿಗೆ (Hosting providers) ಬೆಂಬಲ ನೀಡುವುದು ಇತ್ಯಾದಿ.
