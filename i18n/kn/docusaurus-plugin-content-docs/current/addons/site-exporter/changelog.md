# ಸೈಟ್ ಎಕ್ಸ್‌ಪೋರ್ಟರ್ ಬದಲಾವಣೆಗಳ ವಿವರ {#site-exporter-changelog}

Version: 1.0.6 - ಬಿಡುಗಡೆ ದಿನಾಂಕ: 2026-05-11
* ಹೊಸದಾಗಿ: ಸೈಟ್ ಎಕ್ಸ್‌ಪೋರ್ಟ್‌ಗಳು ಈಗ ಸ್ವಯಂ-ಬೂಟಿಂಗ್ index.php ಅನ್ನು ಒಟ್ಟಿಗೆ ಸೇರಿಸುತ್ತವೆ. ಇದರಿಂದಾಗಿ, ಪ್ರತ್ಯೇಕ plugin ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡದೆ ಹೊಸ ಹೋಸ್ಟ್‌ನಲ್ಲಿ ZIP ಅನ್ನು ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡಬಹುದು.
* ಹೊಸದಾಗಿ: ನೆಟ್‌ವರ್ಕ್ ಎಕ್ಸ್‌ಪೋರ್ಟ್ ಆಡ್ಮಿನ್ ಪುಟದಿಂದ ಎಲ್ಲಾ ಸಬ್‌ಸೈಟ್‌ಗಳನ್ನು ಒಂದೇ ಆರ್ಕೈವ್‌ನಲ್ಲಿ ರಫ್тувати ಅನುಮತಿಸುತ್ತದೆ.

Version: 1.0.5 - ಬಿಡುಗಡೆ ದಿನಾಂಕ: 2026-05-05
* ಸರಿಪಡಿಸಲಾಗಿದೆ: test bootstrap ವಿಫಲವಾಗುವುದನ್ನು ತಡೆಯಲು get_config ನಂತಹ unmocked ವಿಧಾನಗಳನ್ನು ನಿರ್ವಹಿಸಲು WP_CLI polyfill stub ಗೆ __callStatic ಅನ್ನು ಸೇರಿಸಲಾಗಿದೆ.

Version: 1.0.4 - ಬಿಡುಗಡೆ ದಿನಾಂಕ: 2025-11-25

* Ultimate Multisite 2.4 ರೊಂದಿಗೆ ಹೊಂದಾಣಿಕೆಯನ್ನು ಸರಿಪಡಿಸಲಾಗಿದೆ.

Version: 1.0.2 - ಬಿಡುಗಡೆ ದಿನಾಂಕ: 2025-09-28

* ಪ್ರಿಫಿಕ್ಸ್ ಅನ್ನು ultimate-multisite ಎಂದು ಮರುನಾಮಕರಣ ಮಾಡಲಾಗಿದೆ; ಟೆಕ್ಸ್ಟ್ ಡೊಮೇನ್ ಅನ್ನು ಅಪ್ಡೇಟ್ ಮಾಡಲಾಗಿದೆ; ಮತ್ತು ಆವೃತ್ತಿಯನ್ನು ಹೆಚ್ಚಿಸಲಾಗಿದೆ.

### Version 1.0.1 - ಬಿಡುಗಡೆ ದಿನಾಂಕ: 2023-08-09 {#version-101---released-on-2023-08-09}

* ಸುಧಾರಿಸಲಾಗಿದೆ: ಇಂಪೋರ್ಟರ್ ಫೈಲ್‌ನ ಸರಿಯಾದ ಫೈಲ್ ಪಥವನ್ನು ಪಡೆಯುವುದನ್ನು ಇದು ಈಗ ಖಚಿತಪಡಿಸುತ್ತದೆ.
* ಸರಿಪಡಿಸಲಾಗಿದೆ: ಇಂಪೋರ್ಟ್ ಮಾಡುವ ಸಮಯದಲ್ಲಿのエರರ್‌ಗಳನ್ನು ತಡೆಯಲು ಆಬ್ಜೆಕ್ಟ್ ಕ್ಯಾಶ್ ಅನ್ನು ನಿಷ್ಕ್ರಿಯಗೊಳಿಸಲಾಗಿದೆ.
* ಸರಿಪಡಿಸಲಾಗಿದೆ: ಇಂಪೋರ್ಟರ್ plugin ಡೌನ್‌ಲೋಡ್ ಲಿಂಕ್ ಕೆಲಸ ಮಾಡದ ಸಮಸ್ಯೆಯನ್ನು ಸರಿಪಡಿಸಲಾಗಿದೆ.
