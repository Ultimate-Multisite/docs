# Mailster

Ultimate Multisite Mailster Integration addon ಬಳಸಿ ನಿಮ್ಮ multisite network ಅನ್ನು ಶಕ್ತಿಶಾಲಿ ಇಮೇಲ್ ಲಿಸ್ಟ್ ನಿರ್ಮಿಸುವ ಯಂತ್ರವನ್ನಾಗಿ ಪರಿವರ್ತಿಸಿ. ಬಳಕೆದಾರರು ನೋಂದಾಯಿಸಿದಾಗ ಅಥವಾ ಖರೀದಿ ಮಾಡಿದಾಗ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಹೊಸ ಗ್ರಾಹಕರನ್ನು ನಿಮ್ಮ Mailster ಇಮೇಲ್ ಲಿಸ್ಟ್‌ಗಳಿಗೆ ಸೇರಿಸಿ, ನಿಮ್ಮ ಸದಸ್ಯತ್ವ ವೇದಿಕೆ ಮತ್ತು ಇಮೇಲ್ ಮಾರ್ಕೆಟಿಂಗ್ ಕಾರ್ಯತಂತ್ರದ ನಡುವೆ ಸುಗಮ ಸಂಪರ್ಕವನ್ನು ಸೃಷ್ಟಿಸಿ.

## ಪ್ರಮುಖ ವೈಶಿಷ್ಟ್ಯಗಳು (Key Features) {#key-features}

**ಸ್ವಯಂಚಾಲಿತ ಚಂದಾದಾರರ ಸಿಂಕ್ (Automatic Subscriber Sync)**
* ಬಳಕೆದಾರರು ನೋಂದಾಯಿಸಿದಾಗ ಸ್ವಯಂಚಾಲಿತವಾಗಿ Mailster ಲಿಸ್ಟ್‌ಗಳಿಗೆ ಗ್ರಾಹಕರನ್ನು ಸೇರಿಸಿ
* ನೋಂದಣಿಯ ಸಮಯದಲ್ಲಿ ತಕ್ಷಣ ಸಿಂಕ್ ಮಾಡಲು ಅಥವಾ ಪಾವತಿ confirmation ಕಾಯಲು ಆಯ್ಕೆಮಾಡಿ
* ಯಾವುದೇ ಕೈಯಾರೆ ಮಧ್ಯಸ್ಥಿಕೆ ಇಲ್ಲದೆ ನಿಮ್ಮ ಇಮೇಲ್ ಲಿಸ್ಟ್‌ಗಳನ್ನು ತಾಜಾ ಮತ್ತು ಅಪ್‌ಡೇಟ್ ಆಗಿರಿಸಿ

**ಸಮರ್ಥ ಲಿಸ್ಟ್ ನಿರ್ವಹಣೆ (Flexible List Management)**
* ಎಲ್ಲಾ ಹೊಸ ಗ್ರಾಹಕರಿಗಾಗಿ ಜಾಗತಿಕ ಡಿಫಾಲ್ಟ್ ಲಿಸ್ಟ್‌ಗಳನ್ನು ಹೊಂದಿಸಿ
* ನಿರ್ದಿಷ್ಟ ಪ್ರಚಾರಗಳಿಗಾಗಿ ಉತ್ಪನ್ನ-ನಿರ್ದಿಷ್ಟ ಲಿಸ್ಟ್‌ಗಳಿಂದ ಡಿಫಾಲ್ಟ್‌ಗಳನ್ನು ಬದಲಾಯಿಸಿ (Override)
* Mailster ನ ಪರಿಚಿತ ಇಂಟರ್ಫೇಸ್ ಬಳಸಿ ಲಿಸ್ಟ್ ನಿಯೋಜನೆಗಳನ್ನು ಸುಲಭವಾಗಿ ನಿರ್ವಹಿಸಿ

**GDPR-ಸಮ್ಮತ ಆಪ್ಟ್-ಇನ್ ಆಯ್ಕೆಗಳು (GDPR-Compliant Opt-In Options)**
* ಸ್ವಯಂಚಾಲಿತ ಚಂದಾದಾರಿಕೆ ಅಥವಾ ಚೆಕ್‌ಬಾಕ್ಸ್ ಆಧಾರಿತ ಸಮ್ಮತಿಯ ನಡುವೆ ಆಯ್ಕೆಮಾಡಿ
* ಖಚಿತಪಡಿಸಿದ ಚಂದಾದಾರಿಕೆಗಳಿಗಾಗಿ ಡಬಲ್ ಆಪ್ಟ್-ಇನ್ (double opt-in) ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ
* ನಿಮ್ಮ compliance ಅಗತ್ಯತೆಗಳ ಆಧಾರದ ಮೇಲೆ ಆಪ್ಟ್-ಇನ್ ಚೆಕ್‌ಬಾಕ್ಸ್ ಅನ್ನು ತೋರಿಸಿ/ಮರೆಮಾಡಿ

**ಸ್ಮಾರ್ಟ್ ಡೇಟಾ ಮ್ಯಾಪಿಂಗ್ (Smart Data Mapping)**
* ಗ್ರಾಹಕರ ಹೆಸರುಗಳು ಮತ್ತು ಸಂಪರ್ಕ ಮಾಹಿತಿಯನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸಿಂಕ್ ಮಾಡಿ
* ಬಿಲ್ಲಿಂಗ್ ವಿಳಾಸಗಳನ್ನು ಚಂದಾದಾರರ ಕ್ಷೇತ್ರಗಳಿಗೆ ಮ್ಯಾಪ್ ಮಾಡಿ
* ಗುರಿಪಡಿಸಿದ ಪ್ರಚಾರಗಳಿಗಾಗಿ ನೋಂದಣಿ ದಿನಾಂಕಗಳನ್ನು ಟ್ರ್ಯಾಕ್ ಮಾಡಿ

**ಉತ್ಪನ್ನ-ಮಟ್ಟದ ನಿಯಂತ್ರಣ (Product-Level Control)**
* ವಿವಿಧ ಉತ್ಪನ್ನಗಳಿಗೆ ವಿಭಿನ್ನ ಲಿಸ್ಟ್‌ಗಳನ್ನು ನಿಯೋಜಿಸಿ
* ಗ್ರಾಹಕರು ಏನು ಖರೀದಿಸುತ್ತಾರೆ ಅದರ ಆಧಾರದ ಮೇಲೆ ಗುರಿಪಡಿಸಿದ ಪ್ರಚಾರಗಳನ್ನು ರಚಿಸಿ
* ಅಗತ್ಯವಿದ್ದಾಗ ನಿರ್ದಿಷ್ಟ ಉತ್ಪನ್ನಗಳಿಗೆ Mailster ಅನ್ನು ನಿಷ್ಕ್ರಿಯಗೊಳಿಸಿ

**ವೃತ್ತಿಪರ ಅನುಭವ (Professional Experience)**
* ಲಿಸ್ಟ್ ಆಯ್ಕೆಗಾಗಿ native Mailster UI integration
* ಟ್ರಬಲ್ಶೂಟಿಂಗ್ ಮತ್ತು verification ಗಾಗಿ ವಿವರವಾದ ಲಾಗಿಂಗ್
* ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಚಂದಾದಾರರ ಸುಗಮ ಅಪ್‌ಡೇಟ್‌ಗಳು

## ಯಾರಿಗೇ ಸೂಕ್ತ (Perfect For) {#perfect-for}

* ಇಮೇಲ್ ಲಿಸ್ಟ್‌ಗಳನ್ನು ನಿರ್ಮಿಸುತ್ತಿರುವ ಸದಸ್ಯತ್ವ ಸೈಟ್ ಮಾಲೀಕರು
* ಹೊಸ ಗ್ರಾಹಕರನ್ನು ಪೋಷಿಸಲು ಬಯಸುವ SaaS ವೇದಿಕೆಗಳು
* ದಾಖಲಾತಿಯ ಮೂಲಕ ವಿದ್ಯಾರ್ಥಿಗಳನ್ನು ವಿಂಗಡಿಸುವ ಕೋರ್ಸ್ ರಚನೆಕಾರರು
* Mailster ಜೊತೆಗೆ Ultimate Multisite ಅನ್ನು ಬಳಸುವ ಪ್ರತಿಯೊಬ್ಬರಿಗೂ

## ಇದು ಹೇಗೆ ಕೆಲಸ ಮಾಡುತ್ತದೆ (How It Works) {#how-it-works}

1. ನಿಮ್ಮ Ultimate Multisite ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿ addon ಅನ್ನು ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡಿ ಮತ್ತು ಸಕ್ರಿಯಗೊಳಿಸಿ
2. Ultimate Multisite ಸೆಟ್ಟಿಂಗ್‌ಗಳಲ್ಲಿ ಜಾಗತಿಕ ಡಿಫಾಲ್ಟ್ ಲಿಸ್ಟ್‌ಗಳನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಿ
3. ಐಚ್ಛಿಕವಾಗಿ ಉತ್ಪನ್ನ-ನಿರ್ದಿಷ್ಟ ಲಿಸ್ಟ್‌ಗಳಿಂದ ಬದಲಾಯಿಸಿ (Override)
4. ನಿಮ್ಮ ಆಪ್ಟ್-ಇನ್ ಮೋಡ್ ಮತ್ತು ಚಂದಾದಾರಿಕೆ ಸಮಯವನ್ನು ಆರಿಸಿ
5. ನಿಮ್ಮ Mailster ಲಿಸ್ಟ್‌ಗಳು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಬೆಳೆಯುವುದನ್ನು ನೋಡಿ!

## ಇನ್‌ಸ್ಟಾಲೇಶನ್ (Installation) {#installation}

1. ಪ್ಲಗ್ಇನ್ ಫೈಲ್‌ಗಳನ್ನು ನಿಮ್ಮ Ultimate Multisite addons ಡೈರೆಕ್ಟರಿಯಲ್ಲಿ ಅಪ್‌ಲೋಡ್ ಮಾಡಿ
2. Mailster ನಿಮ್ಮ ಮುಖ್ಯ ಸೈಟ್‌ನಲ್ಲಿ ಇನ್‌ಸ್ಟಾಲ್ ಮತ್ತು ಸಕ್ರಿಯವಾಗಿದೆ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ
3. Ultimate Multisite addons ಪುಟದ ಮೂಲಕ addon ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ
4. ಕಾನ್ಫಿಗರ್ ಮಾಡಲು Settings > Mailster Integration ಗೆ ನ್ಯಾವಿಗೇಟ್ ಮಾಡಿ
5. ನಿಮ್ಮ ಡಿಫಾಲ್ಟ್ ಲಿಸ್ಟ್‌ಗಳನ್ನು ಆಯ್ಕೆಮಾಡಿ ಮತ್ತು ಆಪ್ಟ್-ಇನ್ ಆದ್ಯತೆಗಳನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಿ

## ಆಗಾಗ್ಗೆ ಕೇಳಲಾಗುವ ಪ್ರಶ್ನೆಗಳು (Frequently Asked Questions) {#frequently-asked-questions}

## ಈ addon Ultimate Multisite ಇಲ್ಲದೆ ಕೆಲಸ ಮಾಡುತ್ತದೆಯೇ? {#does-this-addon-work-without-ultimate-multisite}

ಇಲ್ಲ, ಈ addon ಕಾರ್ಯನಿರ್ವಹಿಸಲು Ultimate Multisite ಅಗತ್ಯವಿದೆ. ಇದು Mailster ಅನ್ನು Ultimate Multisite ನ ಗ್ರಾಹಕ ಮತ್ತು ಉತ್ಪನ್ನ ವ್ಯವಸ್ಥೆಯೊಂದಿಗೆ ಸಂಯೋಜಿಸಲು ನಿರ್ದಿಷ್ಟವಾಗಿ ವಿನ್ಯಾಸಗೊಳಿಸಲಾಗಿದೆ.

## ಎಲ್ಲಾ ಸೈಟ್‌ಗಳಲ್ಲಿ ನನಗೆ Mailster ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡಬೇಕೇ? {#do-i-need-mailster-installed-on-all-sites}

ಇಲ್ಲ, Mailster ಅನ್ನು ನಿಮ್ಮ ಮುಖ್ಯ ನೆಟ್‌ವರ್ಕ್ ಸೈಟ್‌ನಲ್ಲಿ ಇನ್‌ಸ್ಟಾಲ್ ಮತ್ತು ಸಕ್ರಿಯಗೊಳಿಸುವುದು ಮಾತ್ರ ಸಾಕು. addon ಎಲ್ಲಾ ಸಬ್‌ಸೈಟ್‌ಗಳಿಂದ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಅದಕ್ಕೆ ಸಂಪರ್ಕಿಸುತ್ತದೆ.

## ನಾನು ವಿಭಿನ್ನ ಉತ್ಪನ್ನಗಳಿಗೆ ವಿಭಿನ್ನ ಲಿಸ್ಟ್‌ಗಳನ್ನು ನಿಯೋಜಿಸಬಹುದೇ? {#can-i-assign-different-lists-to-different-products}

ಹೌದು! ನೀವು ಜಾಗತಿಕ ಡಿಫಾಲ್ಟ್ ಲಿಸ್ಟ್‌ಗಳನ್ನು ಹೊಂದಿಸಬಹುದು, ನಂತರ ಪ್ರತಿ ಉತ್ಪನ್ನದ ಆಧಾರದ ಮೇಲೆ ಅವುಗಳನ್ನು ಬದಲಾಯಿಸಬಹುದು (override). ಗ್ರಾಹಕರು ಏನು ಖರೀದಿಸುತ್ತಾರೆ ಅದರ ಆಧಾರದ ಮೇಲೆ ಅತ್ಯಂತ ಗುರಿಪಡಿಸಿದ ಇಮೇಲ್ ಪ್ರಚಾರಗಳನ್ನು ರಚಿಸಲು ಇದು ನಿಮಗೆ ಅನುಮತಿಸುತ್ತದೆ.

## ಗ್ರಾಹಕರು ಈಗಾಗಲೇ Mailster ನಲ್ಲಿ ಇದ್ದರೆ ಏನಾಗುತ್ತದೆ? {#what-happens-if-a-customer-already-exists-in-mailster}

ಡಿಫಾಲ್ಟ್ ಆಗಿ, addon ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಚಂದಾದಾರರನ್ನು ಅಪ್‌ಡೇಟ್ ಮಾಡುತ್ತದೆ ಮತ್ತು ಅವರನ್ನು ಹೊಸ ಲಿಸ್ಟ್‌ಗಳಿಗೆ ಸೇರಿಸುತ್ತದೆ. ನೀವು ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಚಂದಾದಾರರನ್ನು ಬಿಟ್ಟುಹೋಗಲು ಬಯಸಿದರೆ, ನೀವು ಸೆಟ್ಟಿಂಗ್‌ಗಳಲ್ಲಿ ಇದನ್ನು ನಿಷ್ಕ್ರಿಯಗೊಳಿಸಬಹುದು.

## ಇದು GDPR-ಸಮ್ಮತವೇ? {#is-this-gdpr-compliant}

ಹೌದು! addon ನಲ್ಲಿ ಚೆಕ್‌ಬಾಕ್ಸ್ ಆಧಾರಿತ ಆಪ್ಟ್-ಇನ್ ಮೋಡ್ ಇದೆ, ಅಲ್ಲಿ ಗ್ರಾಹಕರು ಇಮೇಲ್‌ಗಳನ್ನು ಸ್ವೀಕರಿಸಲು ಸ್ಪಷ್ಟವಾಗಿ ಸಮ್ಮತಿಸಬೇಕು. ಸರಿಯಾದ ಗೌಪ್ಯತಾ ನೀತಿಗಳೊಂದಿಗೆ ಇದನ್ನು ಸಂಯೋಜಿಸಿದಾಗ ಇದು GDPR ಅವಶ್ಯಕತೆಗಳನ್ನು ಪೂರೈಸುತ್ತದೆ.

## ನಾನು ಡಬಲ್ ಆಪ್ಟ್-ಇನ್ ಬಳಸಬಹುದೇ? {#can-i-use-double-opt-in}

ಖಂಡಿತ! ಡಬಲ್ ಆಪ್ಟ್-ಇನ್ ಸೆಟ್ಟಿಂಗ್ ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ ಮತ್ತು ಚಂದಾದಾರಿಕೆಗಳನ್ನು ಸಕ್ರಿಯಗೊಳಿಸುವ ಮೊದಲು Mailster confirmation ಇಮೇಲ್‌ಗಳನ್ನು ಕಳುಹಿಸುತ್ತದೆ.
