# Domain Seller {#domain-seller}

**Multisite Ultimate: Domain Seller** ನಿಮ್ಮ WordPress Multisite ನೆಟ್‌ವರ್ಕ್ ಅನ್ನು ಶಕ್ತಿಶಾಲಿ ಡೊಮೇನ್ ಮಾರಾಟ ವೇದಿಕೆಯಾಗಿಸುತ್ತದೆ. ಗ್ರಾಹಕರು ಸೈನ್-ಅಪ್ ಸಮಯದಲ್ಲೇ ಅಥವಾ ತಮ್ಮ Dashboard‌ನಿಂದಲೇ ಕಸ್ಟಮ್ ಡೊಮೇನ್‌ಗಳನ್ನು ನೋಂದಾಯಿಸಬಹುದು. ತಕ್ಷಣದ ಲಭ್ಯತೆ ಪರಿಶೀಲನೆ ಮತ್ತು ಸ್ಪಷ್ಟ ಬೆಲೆಯೊಂದಿಗೆ.

### Domain Seller ಏಕೆ? {#why-domain-seller}

ಯಾವುದೇ WaaS (Website as a Service) ವೇದಿಕೆಗೆ ಡೊಮೇನ್ ನೋಂದಣಿ ಅತ್ಯಂತ ಹೆಚ್ಚು ಕೇಳಲಾಗುವ ಅಪ್‌ಸೆಲ್‌ಗಳಲ್ಲಿ ಒಂದು. ನಿಮ್ಮ ಗ್ರಾಹಕರನ್ನು ಮೂರನೇ ಪಕ್ಷದ ರಿಜಿಸ್ಟ್ರಾರ್‌ಗೆ ಕಳುಹಿಸುವ ಬದಲು, ನೀವು ಡೊಮೇನ್‌ಗಳನ್ನು ನೇರವಾಗಿ ನೀಡಬಹುದು — ಹೀಗೆ ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿ ಹೆಚ್ಚು ಆದಾಯ ಉಳಿಸಿಕೊಂಡು, ಗ್ರಾಹಕರಿಗೆ ಅಡೆತಡೆರಹಿತ ಅನುಭವ ನೀಡಬಹುದು.

### ಪ್ರಮುಖ ವೈಶಿಷ್ಟ್ಯಗಳು {#key-features}

#### ಬಹು ಡೊಮೇನ್ ರಿಜಿಸ್ಟ್ರಾರ್ ಬೆಂಬಲ {#multiple-domain-registrar-support}
* **OpenSRS** — ಸಂಪೂರ್ಣ ನೋಂದಣಿ, ನವೀಕರಣ ಮತ್ತು ನಿರ್ವಹಣೆ
* **Namecheap** — WhoisGuard ಮತ್ತು ಸ್ಯಾಂಡ್‌ಬಾಕ್ಸ್ ಪರೀಕ್ಷೆಯೊಂದಿಗೆ ಸಂಪೂರ್ಣ ಏಕೀಕರಣ
* **HostAfrica** — Domains Reseller API ಮೂಲಕ ಡೊಮೇನ್ ಹುಡುಕಾಟ, ಬೆಲೆ, ನೋಂದಣಿ, ವರ್ಗಾವಣೆ, DNS ಮತ್ತು ID ರಕ್ಷಣೆ
* **Openprovider** — ರಿಸೆಲ್ಲರ್ ಬೆಲೆ, ನೋಂದಣಿ, ನವೀಕರಣ, ವರ್ಗಾವಣೆಗಳು, DNS, WHOIS ಗೌಪ್ಯತೆ ಮತ್ತು TLD ಸಿಂಕ್
* **Hostinger** — ಹಂಚಿಕೊಂಡ Hostinger ಏಕೀಕರಣ ಟೋಕನ್ ಬಳಸಿ ಡೊಮೇನ್ ಹುಡುಕಾಟ ಮತ್ತು ನೋಂದಣಿ
* **ವಿಸ್ತರಿಸಬಹುದಾದ ಆರ್ಕಿಟೆಕ್ಚರ್** — Integration Registry ಮಾದರಿಯ ಮೇಲೆ ನಿರ್ಮಿಸಲಾಗಿದೆ, ಇನ್ನಷ್ಟು ಪೂರೈಕೆದಾರರನ್ನು ಸೇರಿಸುವುದನ್ನು ಸುಲಭಗೊಳಿಸುತ್ತದೆ
* **ಒಳನಿರ್ಮಿತ ಸಂಪರ್ಕ ಪರೀಕ್ಷೆ** — ಲೈವ್‌ಗೆ ಹೋಗುವ ಮೊದಲು ಪೂರೈಕೆದಾರರ ರುಜುವಾತುಗಳು ಮತ್ತು ಸಂಪರ್ಕವನ್ನು ಪರಿಶೀಲಿಸಿ

#### ಮಾರ್ಗದರ್ಶಿತ ಸೆಟಪ್ ವಿಜಾರ್ಡ್ {#guided-setup-wizard}
* **ಒಂದು-ಕ್ಲಿಕ್ ಆನ್‌ಬೋರ್ಡಿಂಗ್** — ಪೂರೈಕೆದಾರ ಕಾನ್ಫಿಗರೇಶನ್, TLD ಆಮದು ಮತ್ತು ಡೊಮೇನ್ ಉತ್ಪನ್ನ ರಚನೆಯ ಮೂಲಕ ನಿಮಗೆ ಮಾರ್ಗದರ್ಶನ ನೀಡುತ್ತದೆ
* **ಸ್ವಯಂಚಾಲಿತ TLD ಸಿಂಕ್** — ನಿಮ್ಮ ಸಂಪರ್ಕಿತ ಪೂರೈಕೆದಾರರಿಂದ ಲಭ್ಯವಿರುವ ಎಲ್ಲಾ TLDಗಳು ಮತ್ತು ಹೋಲ್‌ಸೇಲ್ ಬೆಲೆಗಳನ್ನು ಆಮದು ಮಾಡುತ್ತದೆ
* **ಡೀಫಾಲ್ಟ್ ಉತ್ಪನ್ನ ರಚನೆ** — ಯುಕ್ತಿಯುಕ್ತ ಮಾರ್ಕಪ್ ಡೀಫಾಲ್ಟ್‌ಗಳೊಂದಿಗೆ ಬಳಸಲು ಸಿದ್ಧವಾದ ಡೊಮೇನ್ ಉತ್ಪನ್ನವನ್ನು ರಚಿಸುತ್ತದೆ

#### ಹೊಂದಿಕೊಳ್ಳುವ ಡೊಮೇನ್ ಬೆಲೆ {#flexible-domain-pricing}
* **ಮೂರು ಮಾರ್ಕಪ್ ಮೋಡ್‌ಗಳು** — ಶೇಕಡಾವಾರು ಮಾರ್ಕಪ್, ಸ್ಥಿರ ಮಾರ್ಕಪ್ ಅಥವಾ ಪ್ರತಿ TLDಗೆ ಸ್ಥಿರ ಬೆಲೆ
* **TLD-ನಿರ್ದಿಷ್ಟ ಉತ್ಪನ್ನಗಳು** — ಬೇರೆ ಬೇರೆ TLD ಗುಂಪುಗಳಿಗೆ (.com/.net, .io, ದೇಶ ಕೋಡ್‌ಗಳು ಇತ್ಯಾದಿ) ಪ್ರತ್ಯೇಕ ಡೊಮೇನ್ ಉತ್ಪನ್ನಗಳನ್ನು ರಚಿಸಿ
* **ಪರಿಚಯಾತ್ಮಕ ಬೆಲೆ** — ಹೊಸ ನೋಂದಣಿಗಳನ್ನು ಆಕರ್ಷಿಸಲು ರಿಯಾಯಿತಿಯ ಮೊದಲ ವರ್ಷದ ಬೆಲೆ ನೀಡಿ
* **ತಕ್ಷಣದ ಬೆಲೆ** — ಬುದ್ಧಿವಂತ ಕ್ಯಾಶಿಂಗ್‌ನೊಂದಿಗೆ ಪೂರೈಕೆದಾರರಿಂದ ಲೈವ್ ಹೋಲ್‌ಸೇಲ್ ಬೆಲೆ
* **ಕೂಪನ್ ಬೆಂಬಲ** — Multisite Ultimate ರಿಯಾಯಿತಿ ಕೋಡ್‌ಗಳೊಂದಿಗೆ ಸಂಪೂರ್ಣ ಹೊಂದಾಣಿಕೆ

#### WHOIS ಗೌಪ್ಯತಾ ರಕ್ಷಣೆ {#whois-privacy-protection}
* **ಪ್ರತಿ ಉತ್ಪನ್ನದ ಕಾನ್ಫಿಗರೇಶನ್** — WHOIS ಗೌಪ್ಯತೆಯನ್ನು ನಿಷ್ಕ್ರಿಯ, ಯಾವಾಗಲೂ ಒಳಗೊಂಡಿದೆ ಅಥವಾ ಗ್ರಾಹಕರ ಆಯ್ಕೆ ಎಂದು ಹೊಂದಿಸಿ
* **ಗ್ರಾಹಕರ ಆಯ್ಕೆ-ಸಮ್ಮತಿ** — ಗ್ರಾಹಕರ ಆಯ್ಕೆ ಎಂದು ಹೊಂದಿಸಿದಾಗ, ಚೆಕ್‌ಔಟ್ ಚೆಕ್‌ಬಾಕ್ಸ್ ಗ್ರಾಹಕರಿಗೆ ಗೌಪ್ಯತಾ ರಕ್ಷಣೆಯನ್ನು ಸೇರಿಸಲು ಅವಕಾಶ ನೀಡುತ್ತದೆ
* **ಕಾನ್ಫಿಗರ್ ಮಾಡಬಹುದಾದ ಬೆಲೆ** — ಗೌಪ್ಯತಾ ಆಡ್-ಆನ್‌ಗೆ ಪ್ರತಿ ವರ್ಷದ ಬೆಲೆ ಹೊಂದಿಸಿ ಅಥವಾ ಅದನ್ನು ಉಚಿತವಾಗಿ ನೀಡಿ
* **ರಿಜಿಸ್ಟ್ರಾರ್ ಏಕೀಕರಣ** — ನೋಂದಣಿ ಸಮಯದಲ್ಲಿ WhoisGuard (Namecheap) ಅಥವಾ WHOIS ಗೌಪ್ಯತೆ (OpenSRS) ಅನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸಕ್ರಿಯಗೊಳಿಸುತ್ತದೆ

#### ಏಕೀಕೃತ ಚೆಕ್‌ಔಟ್ ಅನುಭವ {#unified-checkout-experience}
* **ಮೂರು ಡೊಮೇನ್ ಮೋಡ್‌ಗಳು** — ಗ್ರಾಹಕರು ಉಪಡೊಮೇನ್ (ಉಚಿತ), ಹೊಸ ಡೊಮೇನ್ ನೋಂದಣಿ ಅಥವಾ ಈಗಾಗಲೇ ತಮ್ಮದೇ ಆದ ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಡೊಮೇನ್ ಮ್ಯಾಪಿಂಗ್ ಇವುಗಳಲ್ಲಿ ಒಂದನ್ನು ಆಯ್ಕೆಮಾಡುತ್ತಾರೆ
* **ತಕ್ಷಣದ ಡೊಮೇನ್ ಹುಡುಕಾಟ** — ಪುಟ ಮರುಲೋಡ್ ಇಲ್ಲದೆ AJAX-ಚಾಲಿತ ಲಭ್ಯತೆ ಪರಿಶೀಲನೆ
* **ಸ್ಮಾರ್ಟ್ ಸಲಹೆಗಳು** — ಆಯ್ಕೆ ಮಾಡಿದ ಡೊಮೇನ್ ಈಗಾಗಲೇ ತೆಗೆದುಕೊಳ್ಳಲ್ಪಟ್ಟಿದ್ದರೆ ಪರ್ಯಾಯ TLD ಸಲಹೆಗಳು
* **ಸ್ಪಷ್ಟ ಬೆಲೆ** — ನೋಂದಣಿ ಬೆಲೆ, ನವೀಕರಣ ಬೆಲೆ ಮತ್ತು ಐಚ್ಛಿಕ WHOIS ಗೌಪ್ಯತಾ ಶುಲ್ಕದ ಸ್ಪಷ್ಟ ಪ್ರದರ್ಶನ
* **ನೋಂದಣಿದಾರರ ಸಂಪರ್ಕ ಕ್ಷೇತ್ರಗಳು** — ಚೆಕ್‌ಔಟ್ ಸಮಯದಲ್ಲೇ ಅಗತ್ಯ ನೋಂದಣಿದಾರ ಮಾಹಿತಿ (ಹೆಸರು, ವಿಳಾಸ, ಫೋನ್) ಸಂಗ್ರಹಿಸುತ್ತದೆ
* **ಸ್ವಯಂ ರಚಿಸಲಾದ ಸೈಟ್ URL** — ಆಯ್ಕೆ ಮಾಡಿದ ಡೊಮೇನ್‌ನಿಂದ ಸೈಟ್ ವಿಳಾಸವನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಪಡೆಯಲಾಗುತ್ತದೆ

#### ಸ್ವಯಂ-ನವೀಕರಣ ವ್ಯವಸ್ಥೆ {#auto-renewal-system}
* **ಸದಸ್ಯತ್ವಕ್ಕೆ ಜೋಡಿಸಲಾದ ನವೀಕರಣಗಳು** — ಗ್ರಾಹಕರ ಸದಸ್ಯತ್ವ ಸಕ್ರಿಯವಾಗಿರುವವರೆಗೆ ಡೊಮೇನ್‌ಗಳು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ನವೀಕರಿಸುತ್ತವೆ
* **ಸ್ಮಾರ್ಟ್ ವೇಳಾಪಟ್ಟಿ** — ಡೊಮೇನ್ ಅವಧಿ ಮುಗಿಯುವ ದಿನಾಂಕಗಳ ಆಧಾರದ ಮೇಲೆ ನವೀಕರಣ ಕೆಲಸಗಳನ್ನು ಸರತಿಯಲ್ಲಿ ಇಡಲಾಗುತ್ತದೆ
* **ಮರುಪ್ರಯತ್ನ ಲಾಜಿಕ್** — ವಿಫಲವಾದ ನವೀಕರಣ ಪ್ರಯತ್ನಗಳಿಗೆ ಬುದ್ಧಿವಂತ ಮರುಪ್ರಯತ್ನಗಳು
* **ಇಮೇಲ್ ಅಧಿಸೂಚನೆಗಳು** — ಬರಲಿರುವ ನವೀಕರಣಗಳು, ಯಶಸ್ವಿ ನವೀಕರಣಗಳು, ವೈಫಲ್ಯಗಳು ಮತ್ತು ಅವಧಿ ಮುಕ್ತಾಯಗಳಿಗೆ ಕಾನ್ಫಿಗರ್ ಮಾಡಬಹುದಾದ ಎಚ್ಚರಿಕೆಗಳು

#### ಗ್ರಾಹಕರ DNS ನಿರ್ವಹಣೆ {#customer-dns-management}
* **ಸ್ವಯಂ-ಸೇವೆ DNS** — ಗ್ರಾಹಕರು ತಮ್ಮ Dashboard‌ನಿಂದ DNS ದಾಖಲೆಗಳನ್ನು (A, AAAA, CNAME, MX, TXT) ಸೇರಿಸಬಹುದು, ಸಂಪಾದಿಸಬಹುದು ಮತ್ತು ಅಳಿಸಬಹುದು
* **ಡೊಮೇನ್ ಸ್ಥಿತಿ** — ತಕ್ಷಣದ ನೋಂದಣಿ ಸ್ಥಿತಿ, ಅವಧಿ ಮುಗಿಯುವ ದಿನಾಂಕಗಳು ಮತ್ತು ನೇಮ್‌ಸರ್ವರ್ ಮಾಹಿತಿ
* **ಸ್ವಯಂಚಾಲಿತ ಡೊಮೇನ್ ಮ್ಯಾಪಿಂಗ್** — ನೋಂದಾಯಿತ ಅಥವಾ ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಡೊಮೇನ್‌ಗಳನ್ನು ಗ್ರಾಹಕರ ಸೈಟ್‌ಗೆ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಮ್ಯಾಪ್ ಮಾಡಲಾಗುತ್ತದೆ

#### ನಿರ್ವಾಹಕ ಸಾಧನಗಳು {#admin-tools}
* **ಹಸ್ತಚಾಲಿತ ಡೊಮೇನ್ ನೋಂದಣಿ** — ನಿರ್ವಾಹಕ ಪ್ಯಾನೆಲ್‌ನಿಂದ ಗ್ರಾಹಕರ ಪರವಾಗಿ ಡೊಮೇನ್‌ಗಳನ್ನು ನೋಂದಾಯಿಸಿ
* **ಡೊಮೇನ್ ಮೇಲ್ವಿಚಾರಣೆ** — ಎಲ್ಲಾ ಗ್ರಾಹಕರ ಡೊಮೇನ್‌ಗಳು, ಸ್ಥಿತಿಗಳು ಮತ್ತು ಅವಧಿ ಮುಗಿಯುವ ದಿನಾಂಕಗಳನ್ನು ಒಂದೇ ಪುಟದಿಂದ ನೋಡಿ
* **ಪೂರೈಕೆದಾರ-ನಿರ್ದಿಷ್ಟ ಲಾಗಿಂಗ್** — ಪ್ರತಿ ರಿಜಿಸ್ಟ್ರಾರ್‌ಗೆ ಪ್ರತ್ಯೇಕ ಲಾಗ್ ಚಾನಲ್‌ಗಳು (ಉದಾ., domain-seller-namecheap.log)
* **ನೋಂದಣಿ ವಿವರಗಳು** — ಮುಖ್ಯ ಡೊಮೇನ್ ಸಂಪಾದನೆ ಪುಟದಲ್ಲಿ ನೋಂದಣಿದಾರರ ಮಾಹಿತಿ ಮತ್ತು ನೋಂದಣಿ ಡೇಟಾವನ್ನು ನೋಡಿ

### ಬೆಂಬಲಿತ ಪೂರೈಕೆದಾರರು {#supported-providers}

* **OpenSRS** — ನೋಂದಣಿ, ನವೀಕರಣ, DNS ನಿರ್ವಹಣೆ ಮತ್ತು WHOIS ಗೌಪ್ಯತೆಯೊಂದಿಗೆ ಸಂಪೂರ್ಣ ಏಕೀಕರಣ
* **Namecheap** — ನೋಂದಣಿ, ನವೀಕರಣ, ಬ್ಯಾಚ್ ಲಭ್ಯತೆ ಪರಿಶೀಲನೆ ಮತ್ತು ಉಚಿತ WhoisGuard‌ನೊಂದಿಗೆ ಸಂಪೂರ್ಣ ಏಕೀಕರಣ
* **HostAfrica** — ನೋಂದಣಿ, ನವೀಕರಣ, ವರ್ಗಾವಣೆಗಳು, DNS ನಿರ್ವಹಣೆ, EPP ಕೋಡ್ ಪಡೆಯುವುದು, ರಿಜಿಸ್ಟ್ರಾರ್ ಲಾಕ್ ಮತ್ತು ID ರಕ್ಷಣೆ
* **Openprovider** — ನೋಂದಣಿ, ನವೀಕರಣ, ವರ್ಗಾವಣೆಗಳು, DNS ನಿರ್ವಹಣೆ, EPP ಕೋಡ್ ಪಡೆಯುವುದು, ರಿಜಿಸ್ಟ್ರಾರ್ ಲಾಕ್, WHOIS ಗೌಪ್ಯತೆ ಮತ್ತು ರಿಸೆಲ್ಲರ್ TLD ಸಿಂಕ್
* **Hostinger** — ನೋಂದಣಿ, ಡೊಮೇನ್ ಹುಡುಕಾಟ, ನೇಮ್‌ಸರ್ವರ್ ನವೀಕರಣಗಳು, ರಿಜಿಸ್ಟ್ರಾರ್ ಲಾಕ್ ಮತ್ತು WHOIS ಗೌಪ್ಯತೆ; ನವೀಕರಣಗಳನ್ನು Hostinger ಸ್ವಯಂ-ನವೀಕರಣ ನಿರ್ವಹಿಸುತ್ತದೆ
* **GoDaddy** — ನೋಂದಣಿ ಮತ್ತು ನವೀಕರಣ
* **ResellerClub** — ನೋಂದಣಿ, ನವೀಕರಣ ಮತ್ತು DNS ನಿರ್ವಹಣೆ
* **NameSilo** — ನೋಂದಣಿ ಮತ್ತು ನವೀಕರಣ
* **Enom** — ನೋಂದಣಿ, ನವೀಕರಣ ಮತ್ತು DNS ನಿರ್ವಹಣೆ

### ತಾಂತ್ರಿಕ ವಿವರಗಳು {#technical-details}

* **ಅಗತ್ಯವಿದೆ:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **ಆರ್ಕಿಟೆಕ್ಚರ್:** Domain_Selling_Capability ಇಂಟರ್‌ಫೇಸ್‌ನೊಂದಿಗೆ Integration Registry ಮಾದರಿ
* **ಆಟೋಲೋಡಿಂಗ್:** PSR-4 ಕ್ಲಾಸ್ ಮ್ಯಾಪಿಂಗ್‌ನೊಂದಿಗೆ Jetpack ಆಟೋಲೋಡರ್
* **ಮಾನದಂಡಗಳು:** WordPress ಕೋಡಿಂಗ್ ಮಾನದಂಡಗಳು, ಕಟ್ಟುನಿಟ್ಟಾದ ಟೈಪಿಂಗ್, ಸಿದ್ಧಪಡಿಸಿದ ಸ್ಟೇಟ್‌ಮೆಂಟ್‌ಗಳು

## ಸ್ಥಾಪನೆ {#installation}

1. **Multisite Ultimate v2.4.12+** ಸ್ಥಾಪಿಸಿ ನೆಟ್‌ವರ್ಕ್-ಸಕ್ರಿಯಗೊಳಿಸಲಾಗಿದೆ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ
2. ಪ್ಲಗಿನ್ ಅನ್ನು `/wp-content/plugins/ultimate-multisite-domain-seller/` ಗೆ ಅಪ್‌ಲೋಡ್ ಮಾಡಿ
3. WordPress Network Admin ನಲ್ಲಿ **ಪ್ಲಗಿನ್‌ಗಳು** ನಿಂದ ಪ್ಲಗಿನ್ ಅನ್ನು ನೆಟ್‌ವರ್ಕ್-ಸಕ್ರಿಯಗೊಳಿಸಿ
4. ಮೊದಲ ಸಕ್ರಿಯಗೊಳಿಸುವಿಕೆಯಲ್ಲಿ **ಸೆಟಪ್ ವಿಜಾರ್ಡ್** ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಆರಂಭವಾಗುತ್ತದೆ — ಈ ಹಂತಗಳನ್ನು ಅನುಸರಿಸಿ:
   * ನಿಮ್ಮ ಡೊಮೇನ್ ರಿಜಿಸ್ಟ್ರಾರ್ ಅನ್ನು ಸಂಪರ್ಕಿಸಿ
   * ಸಂಪರ್ಕವನ್ನು ಪರೀಕ್ಷಿಸಿ
   * ಲಭ್ಯವಿರುವ TLDs ಮತ್ತು ಸಗಟು ಬೆಲೆಗಳನ್ನು ಆಮದು ಮಾಡಿ
   * ಬೆಲೆ ಮಾರ್ಕಪ್‌ನೊಂದಿಗೆ ನಿಮ್ಮ ಮೊದಲ ಡೊಮೇನ್ ಉತ್ಪನ್ನವನ್ನು ರಚಿಸಿ
5. ನಿಮ್ಮ ಚೆಕ್‌ಔಟ್ ಫಾರ್ಮ್‌ಗೆ **ಡೊಮೇನ್ ಆಯ್ಕೆ** ಕ್ಷೇತ್ರವನ್ನು **Ultimate Multisite > ಚೆಕ್‌ಔಟ್ ಫಾರ್ಮ್‌ಗಳು** ನಲ್ಲಿ ಸೇರಿಸಿ

## ಪದೇಪದೇ ಕೇಳಲಾಗುವ ಪ್ರಶ್ನೆಗಳು {#frequently-asked-questions}

### Multisite Ultimate ನ ಯಾವ ಆವೃತ್ತಿ ಅಗತ್ಯ? {#what-version-of-multisite-ultimate-is-required}

ಆವೃತ್ತಿ 2.4.12 ಅಥವಾ ಹೆಚ್ಚಿನದು. ಆಡ್‌ಆನ್ WP Ultimo v1.x ಅಥವಾ 2.4.12 ಕ್ಕಿಂತ ಕೆಳಗಿನ Ultimate Multisite ಆವೃತ್ತಿಗಳೊಂದಿಗೆ ಹೊಂದಿಕೆಯಾಗುವುದಿಲ್ಲ.

### ಯಾವ ಡೊಮೇನ್ ಪೂರೈಕೆದಾರರು ಬೆಂಬಲಿತರಾಗಿದ್ದಾರೆ? {#which-domain-providers-are-supported}

ಒಂಬತ್ತು ರಿಜಿಸ್ಟ್ರಾರ್‌ಗಳು ಬೆಂಬಲಿತರಾಗಿವೆ: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo**, ಮತ್ತು **Enom**. ರಿಜಿಸ್ಟ್ರಾರ್ APIs ವಿಭಿನ್ನ ಕಾರ್ಯಾಚರಣೆಗಳನ್ನು ಒದಗಿಸುವುದರಿಂದ ಸಾಮರ್ಥ್ಯಗಳು ಪೂರೈಕೆದಾರರ ಪ್ರಕಾರ ಬದಲಾಗುತ್ತವೆ; ಪ್ರತಿ ಪೂರೈಕೆದಾರರ ಸಂಕ್ಷಿಪ್ತ ನೋಟಕ್ಕಾಗಿ ಸೆಟಪ್ ಮಾರ್ಗದರ್ಶಿಯನ್ನು ನೋಡಿ. ಆಡ್‌ಆನ್‌ನ Integration Registry ವಿನ್ಯಾಸವು ಕಸ್ಟಮ್ ಪೂರೈಕೆದಾರರನ್ನು ಸೇರಿಸುವುದನ್ನು ಸುಲಭಗೊಳಿಸುತ್ತದೆ — [ಡೆವಲಪರ್ ಮಾರ್ಗದರ್ಶಿ](/developer/integration-guide/custom-registrar) ನೋಡಿ.

### ನಾನು WHOIS ಗೌಪ್ಯತೆ ರಕ್ಷಣೆಯನ್ನು ನೀಡಬಹುದೇ? {#can-i-offer-whois-privacy-protection}

ಹೌದು. ಪ್ರತಿ ಡೊಮೇನ್ ಉತ್ಪನ್ನದಲ್ಲಿ ಮೂರು ಮೋಡ್‌ಗಳೊಂದಿಗೆ WHOIS ಗೌಪ್ಯತೆ ಸೆಟ್ಟಿಂಗ್ ಇರುತ್ತದೆ: ನಿಷ್ಕ್ರಿಯ, ಯಾವಾಗಲೂ ಸೇರಿಸಲಾಗಿದೆ, ಅಥವಾ ಗ್ರಾಹಕರ ಆಯ್ಕೆ. ಗ್ರಾಹಕರ ಆಯ್ಕೆ ಎಂದು ಸೆಟ್ ಮಾಡಿದಾಗ, ಚೆಕ್‌ಔಟ್ ಸಮಯದಲ್ಲಿ ಗ್ರಾಹಕರು ಬೆಲೆಯೊಂದಿಗೆ ಒಂದು ಚೆಕ್‌ಬಾಕ್ಸ್ ಅನ್ನು ನೋಡುತ್ತಾರೆ (ಅಥವಾ Namecheap ನ WhoisGuard ಗಾಗಿ "ಉಚಿತ"). ನೋಂದಣಿ ಸಮಯದಲ್ಲಿ ಗೌಪ್ಯತೆ ಫ್ಲ್ಯಾಗ್ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ರಿಜಿಸ್ಟ್ರಾರ್‌ಗೆ ಕಳುಹಿಸಲಾಗುತ್ತದೆ.

### ವಿಭಿನ್ನ TLDs ಗೆ ವಿಭಿನ್ನ ಬೆಲೆ ನಿಗದಿಪಡಿಸಬಹುದೇ? {#can-i-set-different-pricing-for-different-tlds}

ಹೌದು. ವಿಭಿನ್ನ TLD ಗುಂಪುಗಳಿಗಾಗಿ ಪ್ರತ್ಯೇಕ ಡೊಮೇನ್ ಉತ್ಪನ್ನಗಳನ್ನು ರಚಿಸಿ. ಪ್ರತಿ ಉತ್ಪನ್ನವು ತನ್ನದೇ ಆದ ಮಾರ್ಕಪ್ ಪ್ರಕಾರ (ಶೇಕಡಾವಾರು, ಸ್ಥಿರ ಮಾರ್ಕಪ್, ಅಥವಾ ಸ್ಥಿರ ಬೆಲೆ), ಬೆಂಬಲಿತ TLDs, ಮತ್ತು ಪರಿಚಯಾತ್ಮಕ ಬೆಲೆಯನ್ನು ಹೊಂದಿರಬಹುದು.

### ಗ್ರಾಹಕರು ಯಾವ ಚೆಕ್‌ಔಟ್ ಆಯ್ಕೆಗಳನ್ನು ನೋಡುತ್ತಾರೆ? {#what-checkout-options-do-customers-see}

ಡೊಮೇನ್ ಆಯ್ಕೆ ಕ್ಷೇತ್ರವು ಗ್ರಾಹಕರಿಗೆ ಮೂರು ಟ್ಯಾಬ್‌ಗಳನ್ನು ನೀಡುತ್ತದೆ: **ಉಪಡೊಮೇನ್** (ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿ ಉಚಿತ ಉಪಡೊಮೇನ್ ಬಳಸಿ), **ಹೊಸ ಡೊಮೇನ್ ನೋಂದಾಯಿಸಿ** (ಹೊಸ ಡೊಮೇನ್ ಹುಡುಕಿ ಮತ್ತು ನೋಂದಾಯಿಸಿ), ಮತ್ತು **ಇರುವ ಡೊಮೇನ್** (ಅವರು ಈಗಾಗಲೇ ಹೊಂದಿರುವ ಡೊಮೇನ್ ಅನ್ನು ಮ್ಯಾಪ್ ಮಾಡಿ). ನೀವು ಸಕ್ರಿಯಗೊಳಿಸಿದ ಮೋಡ್‌ಗಳು ಮಾತ್ರ ಕಾಣಿಸುತ್ತವೆ.

### ಇದು ಸ್ವಯಂಚಾಲಿತ ನವೀಕರಣಗಳನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆಯೇ? {#does-it-support-automatic-renewals}

ಹೌದು. ಡೊಮೇನ್‌ಗಳನ್ನು ಸ್ವಯಂ-ನವೀಕರಣಕ್ಕೆ ಸೆಟ್ ಮಾಡಬಹುದು ಮತ್ತು ಅವು ಗ್ರಾಹಕರ ಸದಸ್ಯತ್ವ ಸ್ಥಿತಿಗೆ ಜೋಡಿಸಲಾಗುತ್ತವೆ. ನವೀಕರಣ ಪಾವತಿಗಳನ್ನು ಗ್ರಾಹಕರ ಈಗಿರುವ ಪಾವತಿ ಗೇಟ್‌ವೇ ಮೂಲಕ ಪ್ರಕ್ರಿಯೆಗೊಳಿಸಲಾಗುತ್ತದೆ.

### ಗ್ರಾಹಕರು ತಮ್ಮ DNS ದಾಖಲೆಗಳನ್ನು ನಿರ್ವಹಿಸಬಹುದೇ? {#can-customers-manage-their-dns-records}

ಹೌದು. A, AAAA, CNAME, MX, ಮತ್ತು TXT ದಾಖಲೆಗಳನ್ನು ಸೇರಿಸಲು, ಸಂಪಾದಿಸಲು, ಮತ್ತು ಅಳಿಸಲು ಗ್ರಾಹಕರು ತಮ್ಮ ಡ್ಯಾಶ್‌ಬೋರ್ಡ್‌ನಲ್ಲಿ DNS ನಿರ್ವಹಣಾ ಇಂಟರ್ಫೇಸ್ ಪಡೆಯುತ್ತಾರೆ. OpenSRS, ResellerClub, Enom, HostAfrica, ಮತ್ತು Openprovider ಜೊತೆಗೆ DNS ನಿರ್ವಹಣೆ ಲಭ್ಯವಿದೆ. ಹೋಸ್ಟ್ ಮಾಡಿದ ಡೊಮೇನ್‌ಗಳಿಗೆ Hostinger DNS ಅನ್ನು ಮೂಲ Hostinger ಡೊಮೇನ್-ಮ್ಯಾಪಿಂಗ್ ಇಂಟಿಗ್ರೇಶನ್ ನಿರ್ವಹಿಸುತ್ತದೆ.

### ಸೆಟಪ್ ವಿಜಾರ್ಡ್ ಇದೆಯೇ? {#is-there-a-setup-wizard}

ಹೌದು. ಮೊದಲ ಸಕ್ರಿಯಗೊಳಿಸುವಿಕೆಯಲ್ಲಿ ಸೆಟಪ್ ವಿಜಾರ್ಡ್ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಚಲಿಸುತ್ತದೆ ಮತ್ತು ರಿಜಿಸ್ಟ್ರಾರ್ ಸಂಪರ್ಕಿಸುವುದು, TLDs ಆಮದು ಮಾಡುವುದು, ಮತ್ತು ನಿಮ್ಮ ಮೊದಲ ಡೊಮೇನ್ ಉತ್ಪನ್ನ ರಚಿಸುವ ಹಂತಗಳಲ್ಲಿ ನಿಮಗೆ ಮಾರ್ಗದರ್ಶನ ನೀಡುತ್ತದೆ. ಆಡ್‌ಆನ್ ಸೆಟ್ಟಿಂಗ್‌ಗಳಿಂದ ನೀವು ವಿಜಾರ್ಡ್ ಅನ್ನು ಮತ್ತೆ ಚಾಲನೆ ಮಾಡಬಹುದು.
