# Domain Seller

**Multisite Ultimate: Domain Seller** ನಿಮ್ಮ WordPress Multisite network ಅನ್ನು ಶಕ್ತಿಯುತವಾದ ಡೊಮೈನ್ ಮಾರಾಟ ವೇದಿಕೆಯಾಗಿ ಪರಿವರ್ತಿಸುತ್ತದೆ. ಗ್ರಾಹಕರು साइन-ಅಪ್ ಮಾಡುವಾಗ ಅಥವಾ ತಮ್ಮ dashboard ನಿಂದ ನೇರವಾಗಿ ಕಸ್ಟಮ್ ಡೊಮೇನ್‌ಗಳನ್ನು ನೋಂದಾಯಿಸಲು, ನೈಜ-ಸಮಯದ ಲಭ್ಯತೆ ಪರಿಶೀಲನೆ ಮತ್ತು ಪಾರದರ್ಶಕ ಬೆಲೆಗಳೊಂದಿಗೆ ಈ プラットフォーム ಸಹಾಯ ಮಾಡುತ್ತದೆ.

### Domain Seller ಏಕೆ?

ಯಾವುದೇ WaaS (Website as a Service) ವೇದಿಕೆಗಾಗಿ ಡೊಮೈನ್ ನೋಂದಣಿ ಅತ್ಯಂತ ಬೇಡಿಕೆಯಿರುವ ಅಪ್‌ಸೆಲ್ (upsell) ಆಗಿದೆ. ನಿಮ್ಮ ಗ್ರಾಹಕರನ್ನು ಮೂರನೇ ವ್ಯಕ್ತಿಯ ರಿಜಿಸ್ಟ್ರರ್‌ಗೆ ಕಳುಹಿಸುವ ಬದಲು, ನೀವು ನೇರವಾಗಿ ಡೊಮೇನ್‌ಗಳನ್ನು ನೀಡಬಹುದು — ಇದರಿಂದ ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿ ಹೆಚ್ಚಿನ ಆದಾಯ ಉಳಿಯುತ್ತದೆ ಮತ್ತು ಗ್ರಾಹಕರಿಗೆ ತಡೆರಹಿತ ಅನುಭವ ಸಿಗುತ್ತದೆ.

### ಪ್ರಮುಖ ವೈಶಿಷ್ಟ್ಯಗಳು (Key Features)

#### ಬಹು ಡೊಮೈನ್ ರಿಜಿಸ್ಟ್ರಾರ್ ಬೆಂಬಲ (Multiple Domain Registrar Support)
* **OpenSRS** — ಸಂಪೂರ್ಣ ನೋಂದಣಿ, ನವೀಕರಣ ಮತ್ತು ನಿರ್ವಹಣೆ
* **Namecheap** — WhoisGuard ಮತ್ತು sandbox ಪರೀಕ್ಷೆಯೊಂದಿಗೆ ಸಂಪೂರ್ಣ ಏಕೀಕರಣ (integration)
* **ವಿಸ್ತರಿಸಬಹುದಾದ Architecture** — Integration Registry pattern ಮೇಲೆ ನಿರ್ಮಿಸಲಾಗಿದೆ, ಇದು ಹೆಚ್ಚಿನ ಪроваೈಡರ್‌ಗಳನ್ನು ಸೇರಿಸಲು ಸುಲಭವಾಗಿಸುತ್ತದೆ
* **ಬिल्ಟ್-ಇನ್ Connection Testing** — ಲೈವ್ ಆಗುವ ಮೊದಲು ಪроваೈಡರ್ credentials ಮತ್ತು ಸಂಪರ್ಕವನ್ನು ಪರಿಶೀಲಿಸುತ್ತದೆ

#### ಮಾರ್ಗದರ್ಶಿ ಸೆಟಪ್ ವಿಜಾರ್ಡ್ (Guided Setup Wizard)
* **ಒಂದು-ಕ್ಲಿಕ್ ಆನ್‌ಬೋರ್ಡಿಂಗ್** — ಪроваೈಡರ್ ಕಾನ್ಫಿಗರೇಶನ್, TLD ಇಂಪೋರ್ಟ್ ಮತ್ತು ಡೊಮೈನ್ ಉತ್ಪನ್ನ ರಚನೆಯ ಮೂಲಕ ನಿಮಗೆ ಮಾರ್ಗದರ್ಶನ ನೀಡುತ್ತದೆ
* **ಸ್ವಯಂಚಾಲಿತ TLD ಸಿಂಕ್** — ನಿಮ್ಮ ಸಂಪರ್ಕಿತ ಪроваೈಡರ್‌ಗಳಿಂದ ಲಭ್ಯವಿರುವ ಎಲ್ಲಾ TLDಗಳು ಮತ್ತು wholesale ಬೆಲೆಗಳನ್ನು ಇಂಪೋರ್ಟ್ ಮಾಡುತ್ತದೆ
* **Default product creation** — ಸೂಕ್ತವಾದ ಮಾರ್ಕ್ಅಪ್ (markup) ডিಫಾಲ್ಟ್‌ಗಳೊಂದಿಗೆ ಬಳಸಲು ಸಿದ್ಧವಾದ ಡೊಮೈನ್ ಉತ್ಪನ್ನವನ್ನು ರಚಿಸುತ್ತದೆ

#### ಹೊಂದಿಕೊಳ್ಳುವ ಡೊಮೈನ್ ಬೆಲೆ ನಿಗದಿ (Flexible Domain Pricing)
* **ಮೂರು ಮಾರ್ಕ್ಅಪ್ ಮೋಡ್‌ಗಳು** — ಶೇಕಡಾವಾರು ಮಾರ್ಕ್ಅಪ್, ನಿಶ್ಚಿತ ಮಾರ್ಕ್ಅಪ್, ಅಥವಾ TLDಗೆ ನಿಶ್ಚಿತ ಬೆಲೆ
* **TLD-ನಿರ್ದಿಷ್ಟ ಉತ್ಪನ್ನಗಳು** — ವಿಭಿನ್ನ TLD ಗುಂಪುಗಳಿಗೆ (.com/.net, .io, country codes, ಇತ್ಯಾದಿ) ಪ್ರತ್ಯೇಕ ಡೊಮೈನ್ ಉತ್ಪನ್ನಗಳನ್ನು ರಚಿಸಿ
* **ಪರಿಚಯಾತ್ಮಕ ಬೆಲೆ ನಿಗದಿ** — ಹೊಸ ನೋಂದಣıları ಆಕರ್ಷಿಸಲು ಮೊದಲ ವರ್ಷದ ರಿಯಾಯಿತಿ ಬೆಲೆಯನ್ನು ನೀಡಿ
* **ನೈಜ-ಸಮಯದ ಬೆಲೆ ನಿಗದಿ** — ಬುದ್ಧಿವಂತ 캐ಶಿಂಗ್ (caching) ಜೊತೆಗೆ ಪроваೈಡರ್‌ಗಳಿಂದ ಲೈವ್ wholesale ಬೆಲೆ
* **Coupon support** — Multisite Ultimate discount codes ಜೊತೆಗೆ ಸಂಪೂರ್ಣ compatibility

#### WHOIS Privacy Protection
* **ಪ್ರತಿ-ಉತ್ಪನ್ನ ಕಾನ್ಫಿಗರೇಶನ್** — WHOIS privacy ಅನ್ನು Disabled, Always Included, ಅಥವಾ Customer Choice ಎಂದು ಹೊಂದಿಸಿ
* **Customer opt-in** — Customer Choice ಎಂದು ಹೊಂದಿಸಿದಾಗ, ಚೆಕ್‌ಔಟ್ ಸಮಯದಲ್ಲಿ ಗ್ರಾಹಕರು privacy protection ಅನ್ನು ಸೇರಿಸಲು ಒಂದು ಚೆಕ್‌ಬಾಕ್ಸ್ ಕಾಣಿಸುತ್ತದೆ
* **ಕಾನ್ಫಿಗರಬಲ್ ಬೆಲೆ ನಿಗದಿ** — privacy add-on ಗಾಗಿ ಪ್ರತಿ ವರ್ಷದ ಬೆಲೆಯನ್ನು ನಿಗದಿಪಡಿಸಿ, ಅಥವಾ ಅದನ್ನು ಉಚಿತವಾಗಿ ನೀಡಿ
* **Registrar integration** — ನೋಂದಣಾ ಸಮಯದಲ್ಲಿ WhoisGuard (Namecheap) ಅಥವಾ WHOIS privacy (OpenSRS) ಅನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸಕ್ರಿಯಗೊಳಿಸುತ್ತದೆ

#### ಏಕೀಕೃತ ಚೆಕ್‌ಔಟ್ ಅನುಭವ (Unified Checkout Experience)
* **ಮೂರು ಡೊಮೈನ್ ಮೋಡ್‌ಗಳು** — ಗ್ರಾಹಕರು ಉಪ-ಡೊಮೈನ್ (ಉಚಿತ), ಹೊಸ ಡೊಮೈನ್ ನೋಂದಾಯಿಸುವುದು, ಅಥವಾ ಅವರು ಈಗಾಗಲೇ ಹೊಂದಿರುವ ಡೊಮೈನ್ ಅನ್ನು ಮ್ಯಾಪ್ ಮಾಡುವುದು ಎಂಬುದರ ನಡುವೆ ಆಯ್ಕೆ ಮಾಡುತ್ತಾರೆ
* **ನೈಜ-ಸಮಯದ ಡೊಮೈನ್ ಹುಡುಕಾಟ** — ಯಾವುದೇ page reloads ಇಲ್ಲದ AJAX-ಚಾಲಿತ ಲಭ್ಯತೆ ಪರಿಶೀಲನೆ
* **Smart suggestions** — ಆದ್ಯತೆಯ ಡೊಮೈನ್ ಲಭ್ಯವಿಲ್ಲದಿದ್ದರೆ ಪರ್ಯಾಯ TLD ಸಲಹೆಗಳು
* **ಪಾರದರ್ಶಕ ಬೆಲೆ ನಿಗದಿ** — ನೋಂದಣಿ ಬೆಲೆ, ನವೀಕರಣ ಬೆಲೆ ಮತ್ತು ಐಚ್ಛಿಕ WHOIS privacy ಶುಲ್ಕದ ಸ್ಪಷ್ಟ ಪ್ರದರ್ಶನ
* **Registrant contact fields** — ಚೆಕ್‌ಔಟ್ ಸಮಯದಲ್ಲಿ ಅಗತ್ಯವಿರುವ registrant ಮಾಹಿತಿಯನ್ನು (ಹೆಸರು, ವಿಳಾಸ, ಫೋನ್) ಆನ್‌ಲೈನ್‌ನಲ್ಲಿ ಸಂಗ್ರಹಿಸುತ್ತದೆ
* **Auto-generated site URL** — ಆಯ್ಕೆ ಮಾಡಿದ ಡೊಮೈನ್‌ನಿಂದ site address ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಪಡೆಯಲಾಗುತ್ತದೆ

#### ಸ್ವಯಂಚಾಲಿತ ನವೀಕರಣ ವ್ಯವಸ್ಥೆ (Auto-Renewal System)
* **ಸભ્યಪದತ್ವ-ಸಂಪರ್ಕಿತ ನವೀಕರಣಗಳು** — ಗ್ರಾಹಕರ ಸದಸ್ಯತ್ವ ಸಕ್ರಿಯವಾಗವಿದ್ದರೆ ಡೊಮೇನ್‌ಗಳು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ನವೀಕರಿಸಲ್ಪಡುತ್ತವೆ
* **Smart scheduling** — ಡೊಮೈನ್ ಅವಧಿ ಮುಗಿಯುವ ದಿನಾಂಕಗಳ ಆಧಾರದ ಮೇಲೆ ನವೀಕರಣ ಕೆಲಸಗಳನ್ನು ಕ್ಯೂ (queue) ಮಾಡಲಾಗುತ್ತದೆ
* **Retry logic** — ವಿಫಲವಾದ ನವೀಕರಣ ಪ್ರಯತ್ನಗಳಿಗೆ ಬುದ್ಧಿವಂತ ಪುನರಾವರ್ತನೆಗಳು
* **Email notifications** — ಮುಂಬರುವ ನವೀಕರಣಗಳು, ಯಶಸ್ವಿ ನವೀಕರಣಗಳು, ವೈಫಲ್ಯಗಳು ಮತ್ತು ಅವಧಿ ಮುಗಿಯುವಿಕೆಗಳಿಗೆ ಕಾನ್ಫಿಗರಬಲ್ ಎಚ್ಚರಿಕೆಗಳು

#### ಗ್ರಾಹಕ DNS ನಿರ್ವಹಣೆ (Customer DNS Management)
* **Self-service DNS** — ಗ್ರಾಹಕರು ತಮ್ಮ dashboard ನಿಂದ DNS records (A, AAAA, CNAME, MX, TXT) ಸೇರಿಸಲು, ಸಂಪಾದಿಸಲು ಮತ್ತು ಅಳಿಸಲು ಸಾಧ್ಯವಾಗುತ್ತದೆ
* **Domain status** — ನೈಜ-ಸಮಯದ ನೋಂದಣಿ ಸ್ಥಿತಿ, ಅವಧಿ ಮುಗಿಯುವ ದಿನಾಂಕಗಳು ಮತ್ತು nameserver ಮಾಹಿತಿ
* **Automatic domain mapping** — ನೋಂದಾಯಿಸಲಾದ ಅಥವಾ ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಡೊಮೇನ್‌ಗಳನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಗ್ರಾಹಕರ site ಗೆ ಮ್ಯಾಪ್ ಮಾಡಲಾಗುತ್ತದೆ

#### ಆಡ್ಮಿನ್ ಟೂಲ್ಸ್ (Admin Tools)
* **Manual domain registration** — ಆಡ್ಮಿನ್ ಪ್ಯಾನೆಲ್‌ನಿಂದ ಗ್ರಾಹಕರ ಪರವಾಗಿ ಡೊಮೇನ್‌ಗಳನ್ನು ನೋಂದಾಯಿಸಿ
* **Domain monitoring** — ಎಲ್ಲಾ ಗ್ರಾಹಕರ ಡೊಮೇನ್‌ಗಳು, ಸ್ಥಿತಿಗಳು ಮತ್ತು ಅವಧಿ ಮುಗಿಯುವ ದಿನಾಂಕಗಳನ್ನು ಒಂದೇ ಪುಟದಿಂದ ವೀಕ್ಷಿಸಿ
* **Provider-specific logging** — ಪ್ರತಿ ರಿಜಿಸ್ಟ್ರಾರ್‌ಗಾಗಿ ಪ್ರತ್ಯೇಕ log ಚಾನೆಲ್‌ಗಳು (ಉದಾಹರಣೆಗೆ, domain-seller-namecheap.log)
* **Registration details** — ಕೋರ್ ಡೊಮೈನ್ ಎಡಿಟ್ ಪುಟದಲ್ಲಿ registrant ಮಾಹಿತಿ ಮತ್ತು ನೋಂದಣಿ ಡೇಟಾವನ್ನು ವೀಕ್ಷಿಸಿ

### ಬೆಂಬಲಿತ ಪроваೈಡರ್‌ಗಳು (Supported Providers)

* **OpenSRS** — ನೋಂದಣಿ, ನವೀಕರಣ, DNS ನಿರ್ವಹಣೆ ಮತ್ತು WHOIS privacy ಜೊತೆಗೆ ಸಂಪೂರ್ಣ ಏಕೀಕರಣ
* **Namecheap** — ನೋಂದಣಿ, ನವೀಕರಣ, ಬ್ಯಾಚ್ ಲಭ್ಯತೆ ಪರಿಶೀಲನೆ ಮತ್ತು ಉಚಿತ WhoisGuard ಜೊತೆಗೆ ಸಂಪೂರ್ಣ ಏಕೀಕರಣ
* **GoDaddy** — ನೋಂದಣಿ ಮತ್ತು ನವೀಕರಣ
* **ResellerClub** — ನೋಂದಣಿ, ನವೀಕರಣ ಮತ್ತು DNS ನಿರ್ವಹಣೆ
* **NameSilo** — ನೋಂದಣಿ ಮತ್ತು ನವೀಕರಣ
* **Enom** — ನೋಂದಣಿ, ನವೀಕರಣ ಮತ್ತು DNS ನಿರ್ವಹಣೆ

### ತಾಂತ್ರಿಕ ವಿವರಗಳು (Technical Details)

* **ಬೇಕಾದುದು:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **Architecture:** Integration Registry pattern with Domain_Selling_Capability interface
* **Autoloading:** Jetpack autoloader with PSR-4 class mapping
* **Standards:** WordPress coding standards, strict typing, prepared statements

## Installation

1. **Multisite Ultimate v2.4.12+** ಅನ್ನು ಸ್ಥಾಪಿಸಲಾಗಿದೆ ಮತ್ತು network-activated ಆಗಿದೆ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ
2. ಪ್ಲಗ್ಇನ್ ಅನ್ನು `/wp-content/plugins/ultimate-multisite-domain-seller/` ಗೆ ಅಪ್‌ಲೋಡ್ ಮಾಡಿ
3. WordPress Network Admin ನಲ್ಲಿ **Plugins** ನಿಂದ ಪ್ಲಗ್ಇನ್ ಅನ್ನು Network-activate ಮಾಡಿ
4. ಮೊದಲ activation ಸಮಯದಲ್ಲಿ **Setup Wizard** ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಪ್ರಾರಂಭವಾಗುತ್ತದೆ — ಈ ಹಂತಗಳನ್ನು ಅನುಸರಿಸಿ:
   * ನಿಮ್ಮ ಡೊಮೈನ್ ರಿಜಿಸ್ಟ್ರಾರ್ ಅನ್ನು ಸಂಪರ್ಕಿಸಿ (OpenSRS ಅಥವಾ Namecheap)
   * ಸಂಪರ್ಕವನ್ನು ಪರೀಕ್ಷಿಸಿ
   * ಲಭ್ಯವಿರುವ TLDಗಳು ಮತ್ತು wholesale ಬೆಲೆಗಳನ್ನು ಇಂಪೋರ್ಟ್ ಮಾಡಿ
   * ಬೆಲೆ ಮಾರ್ಕ್ಅಪ್‌ನೊಂದಿಗೆ ನಿಮ್ಮ ಮೊದಲ ಡೊಮೈನ್ ಉತ್ಪನ್ನವನ್ನು ರಚಿಸಿ
5. **Ultimate Multisite > Checkout Forms** ನಲ್ಲಿ ನಿಮ್ಮ ಚೆಕ್‌ಔಟ್ ಫಾರ್ಮ್‌ಗೆ **Domain Selection** field ಅನ್ನು ಸೇರಿಸಿ

## Frequently Asked Questions

### Multisite Ultimate ನ ಯಾವ version ಬೇಕು?

Version 2.4.12 ಅಥವಾ ಅದಕ್ಕಿಂತ ಹೆಚ್ಚಿನದು. ಈ add-on သည် WP Ultimo v1.x ಅಥವಾ 2.4.12 ಕ್ಕಿಂತ ಕಡಿಮೆ Multisite ಆವร์ಶನ್‌ಗಳೊಂದಿಗೆ ಹೊಂದಿಕೆಯಾಗುವುದಿಲ್ಲ.

### ಯಾವ ಡೊಮೈನ್ ಪроваೈಡರ್‌ಗಳನ್ನು ಬೆಂಬಲಿಸಲಾಗುತ್ತದೆ?

ಆರು ರಿಜಿಸ್ಟ್ರರ್‌ಗಳನ್ನು ಸಂಪೂರ್ಣವಾಗಿ ಬೆಂಬಲಿಸಲಾಗುತ್ತದೆ: **OpenSRS**, **Namecheap**, **GoDaddy**, **ResellerClub**, **NameSilo**, ಮತ್ತು **Enom**. ಈ add-on ನ Integration Registry architecture ಯಿಂದ ಕಸ್ಟಮ್ ಪроваೈಡರ್‌ಗಳನ್ನು ಸೇರಿಸುವುದು ಸರಳವಾಗಿದೆ — [developer guide](/developer/integration-guide/custom-registrar) ನೋಡಿ.

### ನಾನು WHOIS privacy protection ನೀಡಬಹುದೇ?

ಹೌದು. ಪ್ರತಿ ಡೊಮೈನ್ ಉತ್ಪನ್ನದಲ್ಲಿ ಮೂರು ಮೋಡ್‌ಗಳೊಂದಿಗೆ WHOIS Privacy ಸೆಟ್ಟಿಂಗ್ ಇದೆ: Disabled, Always Included, ಅಥವಾ Customer Choice. Customer Choice ಎಂದು ಹೊಂದಿಸಿದಾಗ, ಚೆಕ್‌ಔಟ್ ಸಮಯದಲ್ಲಿ ಗ್ರಾಹಕರಿಗೆ ಬೆಲೆಯೊಂದಿಗೆ (ಅಥವಾ Namecheap ನ WhoisGuard ಗಾಗಿ "Free") ಒಂದು ಚೆಕ್‌ಬಾಕ್ಸ್ ಕಾಣಿಸುತ್ತದೆ. privacy flag ಅನ್ನು ನೋಂದಣಿ ಸಮಯದಲ್ಲಿ ರಿಜಿಸ್ಟ್ರರ್‌ಗೆ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಕಳುಹಿಸಲಾಗುತ್ತದೆ.

### ನಾನು ವಿಭಿನ್ನ TLDಗಳಿಗೆ ವಿಭಿನ್ನ ಬೆಲೆ ನಿಗದಿಪಡಿಸಬಹುದೇ?

ಹೌದು. ವಿಭಿನ್ನ TLD ಗುಂಪುಗಳಿಗೆ ಪ್ರತ್ಯೇಕ ಡೊಮೈನ್ ಉತ್ಪನ್ನಗಳನ್ನು ರಚಿಸಿ. ಪ್ರತಿ ಉತ್ಪನ್ನಕ್ಕೆ ತನ್ನದೇ ಆದ ಮಾರ್ಕ್ಅಪ್ ಪ್ರಕಾರ (ಶೇಕಡಾವಾರು, ನಿಶ್ಚಿತ ಮಾರ್ಕ್ಅಪ್, ಅಥವಾ ನಿಶ್ಚಿತ ಬೆಲೆ), ಬೆಂಬಲಿತ TLDಗಳು ಮತ್ತು ಪರಿಚಯಾತ್ಮಕ ಬೆಲೆ ಇರಬಹುದು.

### ಗ್ರಾಹಕರು ಯಾವ ಚೆಕ್‌ಔಟ್ ಆಯ್ಕೆಗಳನ್ನು ನೋಡುತ್ತಾರೆ?

Domain Selection field ಗ್ರಾಹಕರಿಗೆ ಮೂರು ಟ್ಯಾಬ್‌ಗಳನ್ನು ನೀಡುತ್ತದೆ: **Subdomain** (ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿ ಉಚಿತ ಉಪ-ಡೊಮೈನ್ ಬಳಸಿ), **Register New Domain** (ಹೊಸ ಡೊಮೈನ್ ಹುಡುಕಿ ಮತ್ತು ನೋಂದಾಯಿಸಿ), ಮತ್ತು **Existing Domain** (ಅವರು ಈಗಾಗಲೇ ಹೊಂದಿರುವ ಡೊಮೈನ್ ಅನ್ನು ಮ್ಯಾಪ್ ಮಾಡಿ). ನೀವು ಸಕ್ರಿಯಗೊಳಿಸುವ ಮೋಡ್‌ಗಳು ಮಾತ್ರ ಕಾಣಿಸುತ್ತವೆ.

### ಇದು ಸ್ವಯಂಚಾಲಿತ ನವೀಕರಣಗಳನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆಯೇ?

ಹೌದು. ಡೊಮೇನ್‌ಗಳನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ನವೀಕರಿಸಲು ಹೊಂದಿಸಬಹುದು ಮತ್ತು ಅವು ಗ್ರಾಹಕರ ಸದಸ್ಯತ್ವದ ಸ್ಥಿತಿಗೆ ಜೋಡಿಸಲ್ಪಟ್ಟಿರುತ್ತವೆ. ನವೀಕರಣ ಪಾವತಿಗಳು ಗ್ರಾಹಕರ ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ payment gateway ಮೂಲಕ ಪ್ರಕ್ರಿಯೆಗೊಳ್ಳುತ್ತವೆ.

### ಗ್ರಾಹಕರು ತಮ್ಮ DNS records ಅನ್ನು ನಿರ್ವಹಿಸಬಹುದೇ?

ಹೌದು. ಗ್ರಾಹಕರು A, AAAA, CNAME, MX, ಮತ್ತು TXT records ಅನ್ನು ಸೇರಿಸಲು, ಸಂಪಾದಿಸಲು ಮತ್ತು ಅಳಿಸಲು ತಮ್ಮ dashboard ನಲ್ಲಿ DNS ನಿರ್ವಹಣಾ ಇಂಟರ್ಫೇಸ್ ಅನ್ನು ಪಡೆಯುತ್ತಾರೆ. DNS ನಿರ್ವಹಣೆಯನ್ನು OpenSRS, ResellerClub, ಮತ್ತು Enom ಜೊತೆಗೆ ಲಭ್ಯವಿದೆ.

### ಸೆಟಪ್ ವಿಜಾರ್ಡ್ ಇದೆಯೇ?

ಹೌದು. ಸೆಟಪ್ ವಿಜಾರ್ಡ್ ಮೊದಲ activation ಸಮಯದಲ್ಲಿ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ ಮತ್ತು ನಿಮಗೆ ರಿಜಿಸ್ಟ್ರಾರ್ ಅನ್ನು ಸಂಪರ್ಕすること, TLD ಗಳನ್ನು ಇಂಪೋರ್ಟ್ ಮಾಡುವುದು ಮತ್ತು ನಿಮ್ಮ ಮೊದಲ ಡೊಮೈನ್ ಉತ್ಪನ್ನವನ್ನು ರಚಿಸುವ ಮೂಲಕ ಮಾರ್ಗದರ್ಶನ ನೀಡುತ್ತದೆ. ನೀವು add-on ಸೆಟ್ಟಿಂಗ್‌ಗಳಿಂದ ವಿಜಾರ್ಡ್ ಅನ್ನು ಮರು-run ಮಾಡಬಹುದು.
