* **ಹೊಸದಾಗಿ:** ಡೊಮೇನ್ ಉತ್ಪನ್ನಗಳಿಗೆ ಪರಿಚಯಾತ್ಮಕ ಬೆಲೆ ಬೆಂಬಲ
* **ಹೊಸದಾಗಿ:** Cypress ನೊಂದಿಗೆ E2E test suite
* **ಹೊಸದಾಗಿ:** ಡೊಮೇನ್ life cycle notification ಗಾಗಿ ಇಮೇಲ್ ಟೆಂಪ್ಲೇಟ್‌ಗಳು
* **ಹೊಸದಾಗಿ:** ಆಡಮಿನ್ ಡೊಮೇನ್ ನೋಂದಣಿ ಮೋಡಲ್‌ನಲ್ಲಿ ರಿಜಿಸ್ಟ್ರೆಂಟ್ ವಿಳಾಸ ಫೀಲ್ಡ್‌ಗಳು, ಸೆಟ್ಟಿಂಗ್‌ಗಳಿಂದ ಪೂರ್ವ-ಭರ್ತಿ ಮಾಡಲಾಗಿದೆ
* **ಹೊಸದಾಗಿ:** ಸೇರಿಸಲು, ಸಂಪಾದಿಸಲು ಮತ್ತು ರೆಕಾರ್ಡ್ ಅನ್ನು ಅಳಿಸಲು ಬೆಂಬಲದೊಂದಿಗೆ ಗ್ರಾಹಕ DNS ನಿರ್ವಹಣಾ ಇಂಟರ್ಫೇಸ್
* **ಹೊಸದಾಗಿ:** ಸ್ವಯಂಚಾಲಿತ ಡೊಮೇನ್ ಮ್ಯಾಪಿಂಗ್‌ನೊಂದಿಗೆ "Bring your own domain" ಚೆಕ್‌ಔಟ್ ಆಯ್ಕೆ
* **ಹೊಸದಾಗಿ:** ಚೆಕ್‌ಔಟ್ ಸಮಯದಲ್ಲಿ ಡೊಮೇನ್‌ನಿಂದ site URL ಅನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ರಚಿಸುತ್ತದೆ
* **ಹೊಸದಾಗಿ:** ಸೆಟ್ಟಿಂಗ್‌ಗಳಲ್ಲಿ ಡಿಫಾಲ್ಟ್ nameserver ಮತ್ತು DNS ರೆಕಾರ್ಡ್ ಕಾನ್ಫಿಗರೇಶನ್
* **ಹೊಸದಾಗಿ:** ಕೋರ್ ಡೊಮೇನ್ ಎಡಿಟ್ ಪುಟದಲ್ಲಿ ಡೊಮೇನ್ ನೋಂದಣಿ ವಿವರಗಳು ಮತ್ತು DNS ನಿರ್ವಹಣೆ
* **ಹೊಸದಾಗಿ:** ಸೆಟಪ್ ವಿಝಾರ್ಡ್ ಸೂಕ್ತ ಡಿಫಾಲ್ಟ್‌ಗಳೊಂದಿಗೆ ಡೊಮೇನ್ ಉತ್ಪನ್ನವನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ರಚಿಸುತ್ತದೆ
* **ಹೊಸದಾಗಿ:** ಎಲ್ಲಾ ಕಾನ್ಫಿಗರ್ ಮಾಡಿದ ಪ್ರೊವೈಡರ್‌ಗಳಾದ್ಯಂತ cron ಮೂಲಕ ದೈನಂದಿನ ಸ್ವಯಂಚಾಲಿತ TLD ಸಿಂಕ್
* **ಹೊಸದಾಗಿ:** ಪ್ರತಿ-ಉತ್ಪನ್ನದ ಕಾನ್ಫಿಗರೇಶನ್‌ನೊಂದಿಗೆ WHOIS ಗೌಪ್ಯತೆ (always on, customer choice, ಅಥವಾ disabled)
* **ಹೊಸದಾಗಿ:** ಬೆಲೆ ಪ್ರದರ್ಶನ ಮತ್ತು ಡಾರ್ಕ್ ಮೋಡ್ ಬೆಂಬಲದೊಂದಿಗೆ WHOIS ಗೌಪ್ಯತೆ ಚೆಕ್‌ಔಟ್ ಚೆಕ್‌ಬಾಕ್ಸ್
* **ಹೊಸದಾಗಿ:** ಕೈಯಾರೆ ಡೊಮೇನ್ ನೋಂದಣಿಗಾಗಿ Register Domain ಆಡಮಿನ್ ಪುಟ
* **ಹೊಸದಾಗಿ:** Ultimate Multisite update server ಮೂಲಕ ಸ್ವಯಂಚಾಲಿತ plugin ಅಪ್‌ಡೇಟ್‌ಗಳು
* **ಹೊಸದಾಗಿ:** ಉತ್ಪನ್ನ ಪಟ್ಟಿ ಟೇಬಲ್‌ನಲ್ಲಿ ಡೊಮೇನ್ ಉತ್ಪನ್ನದ ಪ್ರಕಾರ ಫಿಲ್ಟರ್ ಟ್ಯಾಬ್, ಬೈಕೆ ಸ್ಟೈಲಿಂಗ್‌ನೊಂದಿಗೆ
* **ಹೊಸದಾಗಿ:** ಡೊಮೇನ್ ಚೆಕ್‌ಔಟ್ ಫಾರ್ಮ್‌ನಲ್ಲಿ ರಿಜಿಸ್ಟ್ರೆಂಟ್ ಸಂಪರ್ಕ ಫೀಲ್ಡ್‌ಗಳು (ಹೆಸರು, ವಿಳಾಸ, ನಗರ, ರಾಜ್ಯ, ಪಿನ್ ಕೋಡ್, ದೇಶ, ಫೋನ್)
* **ಹೊಸದಾಗಿ:** ರಿಜಿಸ್ಟ್ರಾರ್ API ಅನ್ನು ಕರೆಯುವ ಮೊದಲು ರಿಜಿಸ್ಟ್ರೆಂಟ್ ಫೀಲ್ಡ್ ವ್ಯಾಲಿಡೇಶನ್‌ನೊಂದಿಗೆ ಸ್ಪಷ್ಟದ ত্রুটি ಸಂದೇಶಗಳು
* **ಹೊಸದಾಗಿ:** ಡೊಮೇನ್ ನೋಂದಣಿ ಘಟನೆಗಳಿಗಾಗಿ ಪ್ರೊವೈಡರ್-ನಿರ್ದಿಷ್ಟ ಲಾಗ್ ಚಾನೆಲ್‌ಗಳು (ಉದಾಹರಣೆಗೆ domain-seller-namecheap.log)
* **ಹೊಸದಾಗಿ:** ಮುಖ್ಯ ನೋಂದಣಿ/sign-up ಚೆಕ್‌ಔಟ್ ಫಾರ್ಮ್‌ನಲ್ಲಿ ರಿಜಿಸ್ಟ್ರೆಂಟ್ ಸಂಪರ್ಕ ಫೀಲ್ಡ್‌ಗಳು (ಡೊಮೇನ್ ನೋಂದಾಯಿಸುವಾಗ ತೋರಿಸಲಾಗುತ್ತದೆ)
* **ಸುಧಾರಿಸಲಾಗಿದೆ:** ಡೊಮೇನ್ ಸರ್ಚ್ ಚೆಕ್‌ಔಟ್ ಫೀಲ್ಡ್ ಅನ್ನು ಸಬ್‌ಡೊಮೇನ್, ರಿಜಿಸ್ಟರ್, ಮತ್ತು ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಡೊಮೇನ್ ಟ್ಯಾಬ್‌ಗಳನ್ನು ಬೆಂಬಲಿಸುವ ಏಕೀಕೃತ ಡೊಮೇನ್ ಆಯ್ಕೆ ಫೀಲ್ಡ್‌ನೊಂದಿಗೆ ಬದಲಾಯಿಸಲಾಗಿದೆ
* **ಸುಧಾರಿಸಲಾಗಿದೆ:** ಡೊಮೇನ್ ಉತ್ಪನ್ನ ಸೆಟ್ಟಿಂಗ್‌ಗಳು ಕೋರ್ ವಿಜೆಟ್ ವ್ಯವಸ್ಥೆಯ ಮೂಲಕ ಉತ್ಪನ್ನ ಎಡಿಟ್ ಪುಟದಲ್ಲಿ ಇನ್‌ಲೈನ್‌ನಲ್ಲಿ ರೆಂಡರ್ ಆಗುತ್ತವೆ
* **ಸುಧಾರಿಸಲಾಗಿದೆ:** ಗ್ರಾಹಕ ಡೊಮೇನ್ ಮಾಹಿತಿ ಕೋರ್ ಡೊಮೇನ್ ಮ್ಯಾಪಿಂಗ್ ವಿಜೆಟ್‌ನಲ್ಲಿ ಇನ್‌ಪುಟ್ ಆಗಿದೆ, ಪ್ರತ್ಯೇಕ ಮೆಟಾಬಾಕ್ಸ್‌ನಲ್ಲಿಲ್ಲ
* **ಸುಧಾರಿಸಲಾಗಿದೆ:** TLD ಇಂಪೋರ್ಟ್ ವಿಝಾರ್ಡ್ ಅನ್ನು ಎಲ್ಲಾ ಪ್ರೊವೈಡರ್‌ಗಳಿಂದ ಒಂದು-ಕ್ಲಿಕ್ ಸಿಂಕ್ ಆಗಿ ಸರಳೀಕರಿಸಲಾಗಿದೆ
* **ಸುಧಾರಿಸಲಾಗಿದೆ:** Namecheap ಡೊಮೇನ್ ಲಭ್ಯತೆಯು ವೇಗವಾದ ಹುಡುಕಾಟಕ್ಕಾಗಿ ಬ್ಯಾಚ್ API ಕૉಲ್ ಅನ್ನು ಬಳಸುತ್ತದೆ
* **ಸುಧಾರಿಸಲಾಗಿದೆ:** Namecheap ಬೆಲೆ API ಸರಿಯಾದ ಪ್ಯಾರಾಮೀಟರ್‌ಗಳು ಮತ್ತು ಪ್ರತಿಕ್ರಿಯೆ ಪಾರ್ಸಿಂಗ್ ಅನ್ನು ಬಳಸುತ್ತದೆ
