# Domain Seller ಬದಲಾವಣೆ ದಾಖಲೆ

ಆವೃತ್ತಿ 1.3.0 - 2026-06-02 ರಂದು ಬಿಡುಗಡೆ
- ಹೊಸದು: HostAfrica reseller balance ತುಂಬಾ ಕಡಿಮೆಯಾದಾಗ network-admin ಎಚ್ಚರಿಕೆಯನ್ನು ಸೇರಿಸಲಾಗಿದೆ
- ಹೊಸದು: ಹೊಸದಾಗಿ ನೋಂದಾಯಿಸಿದ domains ಅನ್ನು network site‌ಗೆ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಮ್ಯಾಪ್ ಮಾಡುವುದನ್ನು ಸೇರಿಸಲಾಗಿದೆ
- ಸರಿಪಡಿಸಲಾಗಿದೆ: ಹೊಸ domain ನೋಂದಾಯಿಸುವಾಗ ಮಾತ್ರ registrant field requirements ಅನ್ವಯಿಸಲಾಗಿದೆ
- ಸರಿಪಡಿಸಲಾಗಿದೆ: monitor balance notices ಅನ್ನು ವಜಾಗೊಳಿಸಬಹುದಾಗಿಸಲಾಗಿದೆ
- ಸರಿಪಡಿಸಲಾಗಿದೆ: WooCommerce registrant billing details ಉಳಿಯುವಂತೆ ಖಚಿತಪಡಿಸಲಾಗಿದೆ
- ಸರಿಪಡಿಸಲಾಗಿದೆ: ನೋಂದಣಿಯ ವೇಳೆ registrant contact requirements ಜಾರಿಗೊಳಿಸಲಾಗಿದೆ
- ಸರಿಪಡಿಸಲಾಗಿದೆ: 0% markup ಜೊತೆಗೆ domain registration products ರಚನೆಯಾಗುವುದನ್ನು ತಡೆಯಲಾಗಿದೆ
- ಸರಿಪಡಿಸಲಾಗಿದೆ: checkout session flow ಮೂಲಕ domain selections ಮತ್ತು pricing ಉಳಿಸಲಾಗಿದೆ
- ಸರಿಪಡಿಸಲಾಗಿದೆ: HostAfrica domain pricing currency display ಸುಧಾರಿಸಲಾಗಿದೆ
- ಸರಿಪಡಿಸಲಾಗಿದೆ: WP-core query-var ಹೊಂದಾಣಿಕೆ ದೋಷಗಳನ್ನು ತಡೆಯಲು checkout form-action ವರ್ತನೆ ಸುಧಾರಿಸಲಾಗಿದೆ
- ಸುಧಾರಿಸಲಾಗಿದೆ: setup guidance ನಲ್ಲಿ HostAfrica reseller configuration documentation ಲಿಂಕ್ ಮಾಡಲಾಗಿದೆ

ಆವೃತ್ತಿ 1.0.1 - 2026-02-27 ರಂದು ಬಿಡುಗಡೆ

* **ಹೊಸದು:** ದೊಡ್ಡ ಪ್ರಮಾಣದ ಬೆಲೆ ನಿರ್ವಹಣೆಗೆ TLD ಆಮದು ಸಾಧನ
* **ಹೊಸದು:** ಡೊಮೇನ್ ಉತ್ಪನ್ನಗಳಿಗೆ ಪರಿಚಯಾತ್ಮಕ ಬೆಲೆ ಬೆಂಬಲ
* **ಹೊಸದು:** Cypress ಜೊತೆಗೆ E2E ಪರೀಕ್ಷಾ ಸಮೂಹ
* **ಹೊಸದು:** ಡೊಮೇನ್ ಜೀವನಚಕ್ರ ಅಧಿಸೂಚನೆಗಳಿಗೆ ಇಮೇಲ್ ಟೆಂಪ್ಲೇಟ್‌ಗಳು
* **ಹೊಸದು:** ನಿರ್ವಾಹಕ ಡೊಮೇನ್ ನೋಂದಣಿ ಮಾದರಿಯಲ್ಲಿ ನೋಂದಣಿದಾರರ ವಿಳಾಸ ಕ್ಷೇತ್ರಗಳು, ಸೆಟ್ಟಿಂಗ್‌ಗಳಿಂದ ಮುಂಚಿತವಾಗಿ ಭರ್ತಿ ಮಾಡಲಾಗಿದೆ
* **ಹೊಸದು:** ದಾಖಲೆ ಸೇರಿಸುವ, ಸಂಪಾದಿಸುವ ಮತ್ತು ಅಳಿಸುವ ಬೆಂಬಲದೊಂದಿಗೆ ಗ್ರಾಹಕರ DNS ನಿರ್ವಹಣಾ ಇಂಟರ್ಫೇಸ್
* **ಹೊಸದು:** ಸ್ವಯಂಚಾಲಿತ ಡೊಮೇನ್ ಮ್ಯಾಪಿಂಗ್‌ನೊಂದಿಗೆ "ನಿಮ್ಮದೇ ಡೊಮೇನ್ ತರಿರಿ" ಚೆಕ್‌ಔಟ್ ಆಯ್ಕೆ
* **ಹೊಸದು:** ಚೆಕ್‌ಔಟ್ ಸಮಯದಲ್ಲಿ ಡೊಮೇನ್ ಹೆಸರಿನಿಂದ ಸೈಟ್ URL ಸ್ವಯಂ ರಚನೆ
* **ಹೊಸದು:** ಸೆಟ್ಟಿಂಗ್‌ಗಳಲ್ಲಿ ಡೀಫಾಲ್ಟ್ ನೇಮ್‌ಸರ್ವರ್ ಮತ್ತು DNS ದಾಖಲೆ ಸಂರಚನೆ
* **ಹೊಸದು:** ಮುಖ್ಯ ಡೊಮೇನ್ ಸಂಪಾದನೆ ಪುಟದಲ್ಲಿ ಡೊಮೇನ್ ನೋಂದಣಿ ವಿವರಗಳು ಮತ್ತು DNS ನಿರ್ವಹಣೆ
* **ಹೊಸದು:** ಸೆಟಪ್ ವಿಜಾರ್ಡ್ ಸಮಂಜಸವಾದ ಡೀಫಾಲ್ಟ್‌ಗಳೊಂದಿಗೆ ಡೀಫಾಲ್ಟ್ ಡೊಮೇನ್ ಉತ್ಪನ್ನವನ್ನು ಸ್ವಯಂ ರಚಿಸುತ್ತದೆ
* **ಹೊಸದು:** ಕಾನ್ಫಿಗರ್ ಮಾಡಿದ ಎಲ್ಲಾ ಪೂರೈಕೆದಾರರಾದ್ಯಂತ cron ಮೂಲಕ ದೈನಂದಿನ ಸ್ವಯಂಚಾಲಿತ TLD ಸಿಂಕ್
* **ಹೊಸದು:** ಪ್ರತಿ ಉತ್ಪನ್ನದ ಸಂರಚನೆಯೊಂದಿಗೆ WHOIS ಗೌಪ್ಯತಾ ರಕ್ಷಣೆ (ಯಾವಾಗಲೂ ಆನ್, ಗ್ರಾಹಕರ ಆಯ್ಕೆ, ಅಥವಾ ನಿಷ್ಕ್ರಿಯ)
* **ಹೊಸದು:** ಬೆಲೆ ಪ್ರದರ್ಶನ ಮತ್ತು ಡಾರ್ಕ್ ಮೋಡ್ ಬೆಂಬಲದೊಂದಿಗೆ WHOIS ಗೌಪ್ಯತಾ ಚೆಕ್‌ಔಟ್ ಚೆಕ್‌ಬಾಕ್ಸ್
* **ಹೊಸದು:** ಕೈಯಾರೆ ಡೊಮೇನ್ ನೋಂದಣಿಗೆ Register Domain ನಿರ್ವಾಹಕ ಪುಟ
* **ಹೊಸದು:** Ultimate Multisite ಅಪ್‌ಡೇಟ್ ಸರ್ವರ್ ಮೂಲಕ ಸ್ವಯಂಚಾಲಿತ ಪ್ಲಗಿನ್ ಅಪ್‌ಡೇಟ್‌ಗಳು
* **ಹೊಸದು:** ಉತ್ಪನ್ನ ಪಟ್ಟಿ ಟೇಬಲ್‌ನಲ್ಲಿ ನೇರಳೆ ಬ್ಯಾಡ್ಜ್ ಶೈಲಿಯೊಂದಿಗೆ ಡೊಮೇನ್ ಉತ್ಪನ್ನ ಪ್ರಕಾರದ ಫಿಲ್ಟರ್ ಟ್ಯಾಬ್
* **ಹೊಸದು:** ಡೊಮೇನ್ ಚೆಕ್‌ಔಟ್ ಫಾರ್ಮ್‌ನಲ್ಲಿ ನೋಂದಣಿದಾರರ ಸಂಪರ್ಕ ಕ್ಷೇತ್ರಗಳು (ಹೆಸರು, ವಿಳಾಸ, ನಗರ, ರಾಜ್ಯ, ಅಂಚೆ ಕೋಡ್, ದೇಶ, ಫೋನ್)
* **ಹೊಸದು:** ಸ್ಪಷ್ಟ ದೋಷ ಸಂದೇಶಗಳೊಂದಿಗೆ ರಿಜಿಸ್ಟ್ರಾರ್ API ಕರೆಯುವ ಮೊದಲು ನೋಂದಣಿದಾರರ ಕ್ಷೇತ್ರ ಮಾನ್ಯೀಕರಣ
* **ಹೊಸದು:** ಡೊಮೇನ್ ನೋಂದಣಿ ಘಟನೆಗಳಿಗೆ ಪೂರೈಕೆದಾರ-ನಿರ್ದಿಷ್ಟ ಲಾಗ್ ಚಾನಲ್‌ಗಳು (ಉದಾ. domain-seller-namecheap.log)
* **ಹೊಸದು:** ಮುಖ್ಯ ನೋಂದಣಿ/ಸೈನ್‌ಅಪ್ ಚೆಕ್‌ಔಟ್ ಫಾರ್ಮ್‌ನಲ್ಲಿ ನೋಂದಣಿದಾರರ ಸಂಪರ್ಕ ಕ್ಷೇತ್ರಗಳು (ಡೊಮೇನ್ ನೋಂದಾಯಿಸುವಾಗ ತೋರಿಸಲಾಗುತ್ತದೆ)
* **ಸುಧಾರಿತ:** ಡೊಮೇನ್ ಹುಡುಕಾಟ ಚೆಕ್‌ಔಟ್ ಕ್ಷೇತ್ರವನ್ನು ಸಬ್‌ಡೊಮೇನ್, ನೋಂದಣಿ ಮತ್ತು ಇರುವ ಡೊಮೇನ್ ಟ್ಯಾಬ್‌ಗಳನ್ನು ಬೆಂಬಲಿಸುವ ಏಕೀಕೃತ ಡೊಮೇನ್ ಆಯ್ಕೆ ಕ್ಷೇತ್ರದಿಂದ ಬದಲಾಯಿಸಲಾಗಿದೆ
* **ಸುಧಾರಿತ:** ಮುಖ್ಯ ವಿಜೆಟ್ ವ್ಯವಸ್ಥೆಯ ಮೂಲಕ ಉತ್ಪನ್ನ ಸಂಪಾದನೆ ಪುಟದಲ್ಲಿ ಡೊಮೇನ್ ಉತ್ಪನ್ನ ಸೆಟ್ಟಿಂಗ್‌ಗಳು ಇನ್‌ಲೈನ್ ಆಗಿ ರೆಂಡರ್ ಆಗುತ್ತವೆ
* **ಸುಧಾರಿತ:** ಗ್ರಾಹಕರ ಡೊಮೇನ್ ಮಾಹಿತಿ ಸ್ವತಂತ್ರ ಮೆಟಾಬಾಕ್ಸ್ ಬದಲು ಮುಖ್ಯ ಡೊಮೇನ್ ಮ್ಯಾಪಿಂಗ್ ವಿಜೆಟ್‌ಗೆ ಹುಕ್ ಆಗುತ್ತದೆ
* **ಸುಧಾರಿತ:** TLD ಆಮದು ವಿಜಾರ್ಡ್ ಅನ್ನು ಎಲ್ಲಾ ಪೂರೈಕೆದಾರರಿಂದ ಒಂದು-ಕ್ಲಿಕ್ ಸಿಂಕ್‌ಗೆ ಸರಳಗೊಳಿಸಲಾಗಿದೆ
* **ಸುಧಾರಿತ:** Namecheap ಡೊಮೇನ್ ಲಭ್ಯತೆ ವೇಗವಾದ ಹುಡುಕಾಟಕ್ಕಾಗಿ ಬ್ಯಾಚ್ API ಕರೆಯನ್ನು ಬಳಸುತ್ತದೆ
* **ಸುಧಾರಿತ:** Namecheap ಬೆಲೆ API ಸರಿಯಾದ ಪ್ಯಾರಾಮೀಟರ್‌ಗಳು ಮತ್ತು ಪ್ರತಿಕ್ರಿಯೆ ಪಾರ್ಸಿಂಗ್ ಬಳಸುತ್ತದೆ
* **ಸುಧಾರಿತ:** ಒಂದೇ ನೆಟ್‌ವರ್ಕ್ ಆಯ್ಕೆಯಲ್ಲಿ ಕೇಂದ್ರೀಕೃತ TLD ಸಂಗ್ರಹಣೆ
