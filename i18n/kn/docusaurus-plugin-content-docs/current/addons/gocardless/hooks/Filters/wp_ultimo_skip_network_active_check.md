# ಫಿಲ್ಟರ್: wp_ultimo_skip_network_active_check {#filter-wpultimoskipnetworkactivecheck}

ಈ ಚೆಕ್‌ ಅನ್ನು ನೇರವಾಗಿ ದಾಟಲು (short-circuit) 개발ರ್‌ಗಳಿಗೆ ಅವಕಾಶ ನೀಡುತ್ತದೆ.

ನೀವು composer ಆಧಾರಿತ ಮತ್ತು ಇತರ ಕಸ್ಟಮ್ ಸೆಟಪ್‌ಗಳನ್ನು (ಉದಾಹರಣೆಗೆ Bedrock) ಬಳಸುತ್ತಿರುವಾಗ ಇದು ಉಪಯುಕ್ತವಾಗಿದೆ. ಏಕೆಂದರೆ ಆ ಸೆಟಪ್‌ಗಳಲ್ಲಿ mu-plugins ಅನ್ನು ಪ್ಲಗ್ಇನ್‌ಗಳಂತೆ ಬಳಸುವುದು ಸಾಮಾನ್ಯವಾಗಿದೆ.

### Since {#since}

- 2.0.0
### Source {#source}

[`inc/class-requirements.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-gocardless/blob/main/inc/class-requirements.php#L229) ನಲ್ಲಿ 229ನೇ ಸಾಲಿನಲ್ಲಿ ವ್ಯಾಖ್ಯಾನಿಸಲಾಗಿದೆ.

## Returns {#returns}
(ಈ ವಿಭಾಗದಲ್ಲಿ ಯಾವುದೇ ವಿವರಣೆ ಇಲ್ಲದ ಕಾರಣ, ಅದನ್ನು ಹಾಗೆಯೇ ಬಿಡಲಾಗುತ್ತದೆ.)
