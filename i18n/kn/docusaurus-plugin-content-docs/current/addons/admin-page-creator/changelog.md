# ಅಡ್ಮಿನ್ ಪುಟ ಕ್ರಿಯೇಟರ್ ಬದಲಾವಣೆಗಳ ದಾಖಲೆ

### Version 1.9.0 - Released on 2026-01-18

* ಸರಿಪಡಿಸಲಾಗಿದೆ: ಆಕ್ಟಿವೇಟ್ ಮಾಡುವಾಗ ಗಂಭೀರ ದೋಷಗಳು (Fatal errors) ಬರುವುದು.
* ಮರುಸಂರಚಿಸಲಾಗಿದೆ: ಕಂಟೆಂಟ್ ಸೋರ್ಸ್ ಕ್ಲಾಸ್‌ಗಳನ್ನು ಸಿಂಗಲ್ಟನ್ ಟ್ರೇಟ್ (Singleton trait) ಬಳಸಿಕೊಂಡು ಸರಿಯಾದ ಪೇರೆಂಟ್ ಕನ್‌ಸ್ಟ್ರಕ್ಟರ್ ಇನಿಷಿಯಲೈಸೇಶನ್‌ನೊಂದಿಗೆ ಬಳಸಲಾಗಿದೆ.
* ಸರಿಪಡಿಸಲಾಗಿದೆ: ಕಂಟೆಂಟ್ ಟೈಪ್ ಸೆಲೆಕ್ಟರ್ ರೆಂಡರ್ ಆಗದಂತೆ ತಡೆಯುತ್ತಿದ್ದ Vue.js ಟೆಂಪ್ಲೇಟ್ 컴파일ೇಷನ್ ದೋಷಗಳು.
* ಸರಿಪಡಿಸಲಾಗಿದೆ: ಮಿಸ್ಸಿಂಗ್ Vue ಡೇಟಾ ಪ್ರಾಪರ್ಟೀಸ್ (template_id, external_link_url_embedable, external_link_url_checking).
* ಸರಿಪಡಿಸಲಾಗಿದೆ: ಪೇಜ್ ಲೋಡ್ ಆದಾಗ TinyMCE ಎಡಿಟರ್ ಸರಿಯಾಗಿ ಇನಿಷಿಯಲೈಜ್ ಆಗದಿರುವುದು.
* ಸರಿಪಡಿಸಲಾಗಿದೆ: Composer ಮೂಲಕ Divi support ಫೈಲ್ ಆಟೋಲೋಡ್ ಆಗದಿರುವುದು.
* ಸರಿಪಡಿಸಲಾಗಿದೆ: Divi support ಕ್ಲಾಸ್‌ನಲ್ಲಿ ಹಾಳಾದ PHP ಸಿಂಟ್ಯಾಕ್ಸ್.
* ಸುಧಾರಿಸಲಾಗಿದೆ: JavaScript ಆಸ್ಸೆಟ್ ಆಯೋಜನೆ ಮತ್ತು ಮಿನಿಫಿಕೇಶನ್.
* ಸುಧಾರಿಸಲಾಗಿದೆ: ಲೇಟೆಸ್ಟ್ WordPress ಮತ್ತು page builder ಆವร์ಶನ್‌ಗಳೊಂದಿಗೆ ಹೊಂದಾಣಿಕೆ.
