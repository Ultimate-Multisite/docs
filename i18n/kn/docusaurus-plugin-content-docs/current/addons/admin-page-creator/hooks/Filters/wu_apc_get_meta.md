# Filter: wu_apc_get_meta

ಡೆವಲಪರ್‌ಗಳು ಅಂತಿಮ ಮೌಲ್ಯಗಳನ್ನು (final values) ಫಿಲ್ಟರ್ ಮಾಡಲು ಮತ್ತು ಹೊಸ ಪ್ರೊಸೆಸಿಂಗ್ ಕೇಸ್‌ಗಳನ್ನು (processing cases) ಸೇರಿಸಲು ಇದು ಅನುಮತಿಸುತ್ತದೆ.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $default_value | `string` | ಇದು ಡೀಫಾಲ್ಟ್ ಮೌಲ್ಯ. |
| $meta_type | `string` | ಪಡೆಯಬೇಕಾದ ಮೆಟಾ ಡೇಟಾದ ವಿಧ. |
| $meta_key | `string` | ಮೆಟಾ ಫೀಲ್드의 ಹೆಸರು. |

### Since

- 1.4.0
### Source

ಇದನ್ನು [`inc/wu-apc-functions.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-admin-page-creator/blob/main/inc/wu-apc-functions.php#L93) ನಲ್ಲಿ 93ನೇ ಸಾಲಿನಲ್ಲಿ ವ್ಯಾಖ್ಯಾನಿಸಲಾಗಿದೆ.


## Returns
ಹೊಸ ಡೀಫಾಲ್ಟ್ ಮೌಲ್ಯ.
