# Action: wp_ultimo_woocommerce_load

ಎಲ್ಲಾ add-on ಅವಲಂಬನೆಗಳು (dependencies) ಲೋಡ್ ಆದ ನಂತರ ಇದು ಕಾರ್ಯಗತಗೊಳ್ಳುತ್ತದೆ (triggers).

ಪ್ಲಗ್ಇನ್ ಡೆವಲಪರ್‌ಗಳು ಹೊಸ ಕಾರ್ಯಕ್ಷಮತೆಯನ್ನು (functionality) ಸೇರಿಸಲು ಇದನ್ನು ಬಳಸಬಹುದು.
