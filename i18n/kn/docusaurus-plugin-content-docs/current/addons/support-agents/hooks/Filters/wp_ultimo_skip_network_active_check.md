# Filter: wp_ultimo_skip_network_active_check {#filter-wpultimoskipnetworkactivecheck}

ಈ ಚೆಕ್‌ ಅನ್ನು ನೇರವಾಗಿ ದಾಟಲು (short-circuit) developer ಗಳಿಗೆ ಅವಕಾಶ ನೀಡುತ್ತದೆ.

ನೀವು composer ಆಧಾರಿತ ಮತ್ತು ಇತರ ಕಸ್ಟಮ್ ಸೆಟಪ್‌ಗಳನ್ನು (custom setups) ಬಳಸುತ್ತಿರುವಾಗ ಇದು ಉಪಯುಕ್ತವಾಗಿದೆ. ಉದಾಹರಣೆಗೆ, Bedrock ನಂತಹ ಸೆಟಪ್‌ಗಳಲ್ಲಿ, mu-plugins ಅನ್ನು ಪ್ಲಗ್ಇನ್‌ಗಳಾಗಿ ಬಳಸುವುದು ಸಾಮಾನ್ಯವಾಗಿದೆ.
