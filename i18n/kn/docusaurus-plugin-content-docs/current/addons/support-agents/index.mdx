# Support Agents {#support-agents}

ಸಂಪೂರ್ಣ ನೆಟ್‌ವರ್ಕ್ ಅನ್ನು ನೀವೇ ನಿರ್ವಹಿಸುವುದು ಸುಲಭದ ಕೆಲಸವಲ್ಲ. ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್ ಬೆಳೆಯಲು ಶುರುವಾದಂತೆ, ನಿಮಗೆ ಬೆಳೆಯುತ್ತಿರುವ サಪೋರ್ಟ್ (support) ಬೇಡಿಕೆ ಕಾಣುತ್ತದೆ. ನಿಮ್ಮ ಕೆಲಸದ ಹೊರೆ ಕಡಿಮೆ ಮಾಡಲು ಸಹಾಯ ಮಾಡುವ ಜನರನ್ನು ಸೇರಿಸಿಕೊಳ್ಳುವುದು ಉತ್ತಮ. ಆದರೆ, ಸಂಪೂರ್ಣ ಪ್ರವೇಶ/ಅನುಮತಿ (full access/permission) ನೀಡುವುದು ಅಪಾಯಕಾರಿ ಮತ್ತು ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ಗೆ ಹಾನಿಯಾಗಬಹುದು. ಇಲ್ಲಿಯೇ ಈ Support Agent add-on ಸಹಾಯ ಮಾಡುತ್ತದೆ.

ಈ add-on ನಿಮಗೆ 'Support Agent' ಎಂಬ ಹೊಸ ರೀತಿಯ WordPress ಬಳಕೆದಾರರನ್ನು (user) ರಚಿಸಲು ಅನುಮತಿಸುತ್ತದೆ, ಮತ್ತು ಆ ಬಳಕೆದಾರರು ಯಾವ ಅನುಮತಿಗಳನ್ನು (permissions) ಹೊಂದಬೇಕು ಎಂಬುದರ ಮೇಲೆ ನಿಮಗೆ ಸಂಪೂರ್ಣ ನಿಯಂತ್ರಣ ನೀಡುತ್ತದೆ. ಅನುಮತಿ ನೀಡುವುದು ಕೇವಲ ಟಾಗಲ್ ಬಟನ್ ಆನ್ (on) ಮಾಡಿದಷ್ಟು ಸುಲಭ, ಮತ್ತು ಅದನ್ನು ತೆಗೆದುಹಾಕಬೇಕಾದರೆ ಟಾಗಲ್ ಆಫ್ (off) ಮಾಡಿದಷ್ಟು ಸುಲಭ.

## Features {#features}

* WordPress ಮತ್ತು Ultimate Multisite ಅನುಮತಿಗಳ ವಿವರವಾದ ಪಟ್ಟಿ
* ಸುಲಭವಾಗಿ ಪ್ರವೇಶವನ್ನು (access) ಆನ್ ಮತ್ತು ಆಫ್ ಮಾಡಬಹುದು
* ಅಪರಿಮಿತ ಸಂಖ್ಯೆಯ Agent ಖಾತೆಗಳನ್ನು ರಚಿಸಬಹುದು

## Installation {#installation}

1. 'ultimate-multisite-support-agents' ಅನ್ನು '/wp-content/plugins/' ಡೈರೆಕ್ಟರಿಗೆ ಅಪ್‌ಲೋಡ್ ಮಾಡಿ
2. WordPress ನಲ್ಲಿರುವ 'Plugins' ಮೆನುವಿನ ಮೂಲಕ ಪ್ಲಗ್ಇನ್ ಅನ್ನು ಆಕ್ಟಿವೇಟ್ ಮಾಡಿ
3. ಪ್ಲಗ್ಇನ್ ಅನ್ನು ಸೆಟ್ ಅಪ್ ಮಾಡಲು ಹಂತ ಹಂತದ Wizard ಅನ್ನು ಅನುಸರಿಸಿ
