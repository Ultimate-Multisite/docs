# Content Sync ಬದಲಾವಣೆಗಳ ವಿವರ

### Version 1.1.0 - ಬಿಡುಗಡೆ ದಿನಾಂಕ: 2025-12-31 {#version-110---released-on-2025-12-31}
* ಹೊಸ: ಎಲ್ಲಾ ಪೋಸ್ಟ್ ಟೈಪ್‌ಗಳನ್ನು (posts, pages, custom post types) ಸಿಂಕ್ ಮಾಡಲು ಬೆಂಬಲ.
* ಹೊಸ: ಕಂಟೆಂಟ್ ಆಯ್ಕೆ ಹಂತದಲ್ಲಿ ಪೋಸ್ಟ್ ಟೈಪ್ ಫಿಲ್ಟರಿಂಗ್.
* ಹೊಸ: ಮೂಲ ಸೈಟ್ ಆಯ್ಕೆ ಹಂತ - ಮುಖ್ಯ ಸೈಟ್ ಮಾತ್ರವಲ್ಲದೆ ಯಾವುದೇ ಸೈಟ್‌ನಿಂದ ಸಿಂಕ್ ಮಾಡಬಹುದು.
* ಹೊಸ: ಟೆಂಪ್ಲೇಟ್ ಸೈಟ್‌ಗಳಿಗೆ ಚೈಲ್ಡ್ ಸೈಟ್ ಪೂರ್ವ-ಆಯ್ಕೆ.
* ಹೊಸ: ಎಲ್ಲಾ ಪೋಸ್ಟ್ ಟೈಪ್‌ಗಳಿಗಾಗಿ post_content ನಿಂದ മീഡിയ extraction.
* ಹೊಸ: ಟ್ಯಾಕ್ಸಾನಮಿ ಸಿಂಕ್ ಮಾಡುವುದು (categories, tags, custom taxonomies).
* ಬದಲಾವಣೆ: "Elementor Template Sync" ಅನ್ನು "Content Sync" ಎಂದು ಮರುನಾಮಕರಣ ಮಾಡಲಾಗಿದೆ.
* ಬದಲಾವಣೆ: "Select Templates" ಹಂತವನ್ನು "Select Data" ಎಂದು ಮರುನಾಮಕರಣ ಮಾಡಲಾಗಿದೆ.
* ಸುಧಾರಣೆ: Elementor-ನಿರ್ದಿಷ್ಟ ಕೋಡ್ ಈಗ elementor_library ಪೋಸ್ಟ್ ಟೈಪ್‌ಗೆ ಮಾತ್ರ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ.

### Version 1.0.0 - ಬಿಡುಗಡೆ ದಿನಾಂಕ: 2025-12-19 {#version-100---released-on-2025-12-19}
* ಆರಂಭಿಕ ಬಿಡುಗಡೆ.
* ಟೆಂಪ್ಲೇಟ್ ಮತ್ತು ಸೈಟ್ ಆಯ್ಕೆಗಾಗಿ multi-step wizard.
* ನೈಜ-ಸಮಯದ ಪ್ರಗತಿ ಪ್ರತಿಕ್ರಿಯೆಯೊಂದಿಗೆ ಬ್ಯಾಚ್ ಪ್ರೊಸೆಸಿಂಗ್.
* ಸ್ವಯಂಚಾಲಿತ ID ಮ್ಯಾಪಿಂಗ್‌ನೊಂದಿಗೆ ಸಂಪೂರ್ಣ മീഡിയ ಸಿಂಕ್.
* Ultimate Multisite ಜೊತೆಗೆ ಸೆಟ್ಟಿಂಗ್ಸ್ ಏಕೀಕರಣ.
* ನೆಟ್‌ವರ್ಕ್ ಸೆಟ್ಟಿಂಗ್‌ಗಳ ಮೂಲಕ ಸೈಟ್ ವಿನಾಯಿತಿ ಬೆಂಬಲ.
* ಸಿಂಕ್ ಮಾಡಿದ ನಂತರ Elementor cache ತೆರವುಗೊಳಿಸುವಿಕೆ.
* ಟೆಂಪ್ಲೇಟ್ ಟ್ಯಾಕ್ಸಾನಮಿ ಸಂರಕ್ಷಣೆ.
