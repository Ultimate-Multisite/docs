# Content Sync

Multisite Ultimate: Content Sync

Multisite Ultimate-ನ ಈ Addon, ನೆಟ್‌ವರ್ಕ್ ಆಡಳಿತಗಾರರು (network administrators) ಯಾವುದೇ ಸೈಟ್‌ನಿಂದ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿರುವ ಇತರ ಸೈಟ್‌ಗಳಿಗೆ ವಿಷಯವನ್ನು (content) ಸಮನ್ವಯಗೊಳಿಸಲು (synchronize) ಅನುವು ಮಾಡಿಕೊಡುತ್ತದೆ. ಇದು ಪೋಸ್ಟ್‌ಗಳು, ಪುಟಗಳು (pages), ಕಸ್ಟಮ್ ಪೋಸ್ಟ್ ಟೈಪ್‌ಗಳು ಮತ್ತು Elementor ಟೆಂಪ್ಲೇಟ್‌ಗಳು ಸೇರಿದಂತೆ ಎಲ್ಲಾ ಪೋಸ್ಟ್ ಪ್ರಕಾರಗಳನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ. ವಿಷಯವನ್ನು ಆಯ್ಕೆ ಮಾಡಲು ಮತ್ತು ಗುರಿ ಸೈಟ್‌ಗಳನ್ನು (target sites) ಆರಿಸಲು ಈ Addon ಒಂದು multi-step wizard ಇಂಟರ್‌ಫೇಸ್ ಅನ್ನು ಒಳಗೊಂಡಿದೆ, ಜೊತೆಗೆ ಬ್ಯಾಚ್ ಪ್ರೊಸೆಸಿಂಗ್ ಮತ್ತು ನೈಜ-ಸಮಯದ ಪ್ರಗತಿ ಪ್ರತಿಕ್ರಿಯೆ (real-time progress feedback)ಯನ್ನು ನೀಡುತ್ತದೆ.

## Features

* Content Selection: ಥಂಬ್‌ನೇಲ್‌ಗಳು ಮತ್ತು ಪೋಸ್ಟ್ ಟೈಪ್ ಫಿಲ್ಟರಿಂಗ್‌ನೊಂದಿಗೆ ವಿಶ್ual ಕಾರ್ಡ್ ಗ್ರಿಡ್.
* All Post Types: ಪೋಸ್ಟ್‌ಗಳು, ಪುಟಗಳು, ಉತ್ಪನ್ನಗಳು (products), Elementor ಟೆಂಪ್ಲೇಟ್‌ಗಳು ಮತ್ತು ಯಾವುದೇ ಕಸ್ಟಮ್ ಪೋಸ್ಟ್ ಟೈಪ್ ಅನ್ನು ಸಿಂಕ್ ಮಾಡಬಹುದು.
* Site Selection: ಯಾವ ಸೈಟ್‌ಗಳಿಗೆ ವಿಷಯವನ್ನು ಸಿಂಕ್ ಮಾಡಬೇಕೆಂದು ಆಯ್ಕೆಮಾಡಿ. ಇದರಲ್ಲಿ ಮೂಲ ಸೈಟ್ ಅನ್ನು ಹೊರಗಿಡಬಹುದು ಮತ್ತು ಚೈಲ್ಡ್ ಸೈಟ್‌ಗಳನ್ನು ಮೊದಲೇ ಆಯ್ಕೆ ಮಾಡಬಹುದು.
* Batch Processing: ಪ್ರಗತಿ ನವೀಕರಣಗಳೊಂದಿಗೆ ಒಂದೇ ಬಾರಿಗೆ 10 ಸೈಟ್‌ಗಳಿಗೆ ಸಿಂಕ್ ಮಾಡುತ್ತದೆ.
* Media Sync: ಪೋಸ್ಟ್ ವಿಷಯದಿಂದ (post content) ಚಿತ್ರಗಳು, ವೀಡಿಯೊಗಳಂತಹ 미디어 ಫೈಲ್‌ಗಳನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ನಕಲಿಸಿ ಮತ್ತು ಗುರಿ ಸೈಟ್‌ಗಳಿಗೆ ಮ್ಯಾಪ್ ಮಾಡುತ್ತದೆ.
* Taxonomy Sync: ವರ್ಗಗಳು (categories), ಟ್ಯಾಗ್‌ಗಳು (tags) ಮತ್ತು ಕಸ್ಟಮ್ ಟ್ಯಾಕ್ಸಾನಮಿಗಳನ್ನು ಉಳಿಸುತ್ತದೆ.
* Elementor Support: CSS ಮರು-ಜನನ (regeneration) ಮತ್ತು ಟೆಂಪ್ಲೇಟ್ ID ಮ್ಯಾಪಿಂಗ್‌ನೊಂದಿಗೆ Elementor ಟೆಂಪ್ಲೇಟ್‌ಗಳಿಗೆ ಸಂಪೂರ್ಣ ಬೆಂಬಲ.
* Conflict Resolution: ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ವಿಷಯವನ್ನು ಬದಲಾಯಿಸುವ ಅಥವಾ ಅದನ್ನು ಬಿಟ್ಟುಹೋಗುವ (skip) ಆಯ್ಕೆ.
* Cache Management: ಸಿಂಕ್ ಆದ ನಂತರ ಗುರಿ ಸೈಟ್‌ಗಳಲ್ಲಿ Elementor ಕ್ಯಾಶ್ ಅನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ತೆರವುಗೊಳಿಸುತ್ತದೆ.

## Requirements

* WordPress 5.3 ಅಥವಾ ಅದಕ್ಕಿಂತ ಹೆಚ್ಚಿನದು
* PHP 7.4 ಅಥವಾ ಅದಕ್ಕಿಂತ ಹೆಚ್ಚಿನದು
* Multisite Ultimate plugin (ಸಕ್ರಿಯವಾಗಿರಬೇಕು)
* Elementor plugin (ಐಚ್ಛಿಕ, Elementor ಟೆಂಪ್ಲೇಟ್ ಸಿಂಕ್‌ಗೆ ಅಗತ್ಯ)

## Installation

1. 'ultimate-multisite-content-sync' ಅನ್ನು '/wp-content/plugins/' ಡೈರೆಕ್ಟರಿಗೆ ಅಪ್‌ಲೋಡ್ ಮಾಡಿ.
2. ನಿಮ್ಮ WordPress Network Admin ನಲ್ಲಿರುವ 'Plugins' ಮೆನುವಿನ ಮೂಲಕ ಪ್ಲಗ್ಇನ್ ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ.
3. Ultimate Multisite → Settings → Content Sync ನಲ್ಲಿ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಿ.
4. ಸಿಂಕ್ ಮಾಡಲು Ultimate Multisite → Content Sync ಗೆ ಹೋಗಿ.
