# ಚಾಕ್ ನೊರಿಸ್ ಫ್ಯಾಕ್ಟ್ಸ್ ಚೇಂಜ್‌ಲಾಗ್

### 1.0.1 - 2025-11-03 {#101---2025-11-03}
* ಸೇರಿಸಲಾಗಿದೆ: Ultimate Multisite ಅಪ್ಡೇಟ್ ಸರ್ವರ್ ಮೂಲಕ ಸ್ವಯಂಚಾಲಿತ ಅಪ್ಡೇಟ್ ಕಾರ್ಯನಿರ್ವಹಣೆ.
* ನವೀಕರಿಸಲಾಗಿದೆ: WordPress 6.6 ವರೆಗೆ ಪರೀಕ್ಷಿಸಲಾಗಿದೆ.
* ನವೀಕರಿಸಲಾಗಿದೆ: composer autoloader ಜೊತೆಗೆ ಸುಧಾರಿಸಿದ plugin ರಚನೆ.
* ನವೀಕರಿಸಲಾಗಿದೆ: ಆಧುನಿಕ WordPress ಇನ್‌ಸ್ಟಾಲೇಷನ್‌ಗಳೊಂದಿಗೆ ಸುಧಾರಿಸಿದ ಹೊಂದಾಣಿಕೆ.

### 1.0.0 - 2024-10-21 {#100---2024-10-21}
* ಮೊದಲ ಬಿಡುಗಡೆ
* 110+ ಚಾಕ್ ನೊರಿಸ್ ಫ್ಯಾಕ್ಟ್ಸ್
* ಸಂಪೂರ್ಣ ಅನುವಾದ ಬೆಂಬಲ
* ಎಲ್ಲಾ ಪರದೆಯ ಗಾತ್ರಗಳಿಗೆ ಪ್ರತಿಕ್ರಿಯಾತ್ಮಕ ವಿನ್ಯಾಸ (Responsive design)
* RTL ಭಾಷಾ ಬೆಂಬಲ
* Block editor ಹೊಂದಾಣಿಕೆ
* screen reader ಬೆಂಬಲದೊಂದಿಗೆ ಪ್ರವೇಶಿಸುವಿಕೆ ವೈಶಿಷ್ಟ್ಯಗಳು
