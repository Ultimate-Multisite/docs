# Mailchimp

Multisite Ultimate: MailChimp ಸಂಯೋಜನೆ (Integration)

ನಿಮ್ಮ Multisite Ultimate ನೆಟ್‌ವರ್ಕ್‌ ಅನ್ನು ನಿಮ್ಮ Mailchimp ಜೊತೆಗೆ ಸಂಯೋಜಿಸಿ. ನೀವು ಕೆಲವು ಯೋಜನೆಗಳನ್ನು (plans) Mailchimp ಲಿಸ್ಟ್‌ಗಳಿಗೆ ಲಿಂಕ್ ಮಾಡಬಹುದು ಮತ್ತು ಬಳಕೆದಾರರು ನೋಂದಾಯಿಸುವಾಗ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಆ ಲಿಸ್ಟ್‌ಗಳಿಗೆ ಚಂದಾದarları ಸೇರಿಸಬಹುದು.

## Installation (ಸ್ಥಾಪನೆ)

1. 'ultimate-multisite-mailchimp' ಅನ್ನು '/wp-content/plugins/' ಡೈರೆಕ್ಟರಿಗೆ ಅಪ್‌ಲೋಡ್ ಮಾಡಿ
2. ನಿಮ್ಮ WordPress Network Admin ನಲ್ಲಿರುವ 'Plugins' ಮೆನುವಿನ ಮೂಲಕ ಪ್ಲಗ್ಇನ್ ಅನ್ನು ಆಕ್ಟಿವೇಟ್ ಮಾಡಿ

## Usage Guide (ಬಳಕೆಯ ಮಾರ್ಗದರ್ಶಿ)

ನಮ್ಮ Ultimate Multisite: MailChimp Integration Add-on ಬಳಸಿ ನಿಮ್ಮ ಚಂದಾದarları ಇಮೇಲ್‌ಗಳನ್ನು ನಿಮ್ಮ MailChimp ಲಿಸ್ಟ್‌ಗಳೊಂದಿಗೆ ಸಿಂಕ್ (Sync) ಮಾಡುವುದು ತುಂಬಾ ಸುಲಭ. ಈ ಟ್ಯುಟೋರಿಯಲ್ ಎಲ್ಲವನ್ನೂ ಸೆಟ್ ಅಪ್ ಮಾಡಲು ನಿಮಗೆ ಸಹಾಯ ಮಾಡುತ್ತದೆ!

## Getting an API key (API ಕೀ ಪಡೆಯುವುದು)

ನಾವು ವಿವಿಧ ಸೆಟ್ಟಿಂಗ್‌ಗಳ ಬಗ್ಗೆ ತಿಳಿದುಕೊಳ್ಳಲು ಪ್ರಾರಂಭಿಸುವ ಮೊದಲು, ನಮಗೆ ಒಂದು MailChimp API ಕೀ ಬೇಕು. ಬಳಕೆದಾರರು ನೋಂದಾಯಿಸಿದಾಗ ಲಿಸ್ಟ್‌ಗಳು, ಗ್ರೂಪ್‌ಗಳನ್ನು ಪಡೆಯಲು ಮತ್ತು ಆ ಲಿಸ್ಟ್‌ಗಳು ಹಾಗೂ ಗ್ರೂಪ್‌ಗಳಿಗೆ ಇಮೇಲ್ ವಿಳಾಸಗಳನ್ನು ಸೇರಿಸಲು Ultimate Multisite ಗೆ MailChimp API ಯೊಂದಿಗೆ ಮಾತನಾಡಲು ಇದು ಅನುಮತಿಸುತ್ತದೆ.

ನಿಮ್ಮ ಸ್ವಂತ MailChimp API ಕೀ ಪಡೆಯಲು ಈ ಟ್ಯುಟೋರಿಯಲ್ ಅನ್ನು ಅನುಸರಿಸಿ: [About API Keys | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

<!-- Screenshot unavailable: MailChimp API key settings in Ultimate Multisite Add-on Settings -->

_ನಿಮ್ಮ MailChimp API ಕೀ ಅನ್ನು_ _**Ultimate Multisite Settings - > Add-on Settings -> MailChimp**_ ನಲ್ಲಿ ಸೇರಿಸಿ

## Integration Mode (ಸಂಯೋಜನಾ ವಿಧಾನ)

MailChimp Integration add-on ಎರಡು "ಸಂಯೋಜನಾ ವಿಧಾನಗಳನ್ನು" (integration modes) ನೀಡುತ್ತದೆ, ಇದರರ್ಥ ಇಮೇಲ್ ವಿಳಾಸಗಳನ್ನು ಸಿಂಕ್ ಮಾಡಲು ನೀವು ಈ add-on ಅನ್ನು ಬಳಸಲು ಎರಡು ವಿಭಿನ್ನ ಮಾರ್ಗಗಳನ್ನು ಹೊಂದಿದ್ದೀರಿ.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_ಹೆಚ್ಚಿನ flexibility ಗಾಗಿ ಎರಡು ಸಂಯೋಜನಾ ವಿಧಾನಗಳು!_

### Mode 1: Multiple Lists (ಬಹು ಲಿಸ್ಟ್‌ಗಳು)

ಡೀಫಾಲ್ಟ್ ಮೋಡ್ **Multiple Lists** ಅನ್ನು ಬಳಸುತ್ತದೆ. ಈ ಮೋಡ್ ನಿಮಗೆ ಪ್ರತಿ ಯೋಜನೆಯ (plan) ಗಾಗಿ ಅನೇಕ MailChimp ಲಿಸ್ಟ್‌ಗಳನ್ನು ಆಯ್ಕೆ ಮಾಡಲು ಅನುಮತಿಸುತ್ತದೆ. ನೀವು ನಿಮ್ಮ ಬಳಕೆದಾರರನ್ನು ವಿಭಿನ್ನ ಲಿಸ್ಟ್‌ಗಳಲ್ಲಿ ವಿಭಾಗಿಸುತ್ತಿದ್ದರೆ (segment) ಇದು ಅತ್ಯುತ್ತಮ ಆಯ್ಕೆಯಾಗಿದೆ.

ಉದಾಹರಣೆಗೆ, ನಿಮ್ಮ MailChimp ಖಾತೆಯಲ್ಲಿ ನಿಮಗೆ 3 ವಿಭಿನ್ನ ಲಿಸ್ಟ್‌ಗಳು ಇದೆ ಎಂದು ಊಹಿಸಿ: **Newsletter**, **Plan A**, ಮತ್ತು **Plan B**. ನಿಮ್ಮ ಎಲ್ಲಾ ಬಳಕೆದಾರರು, ಅವರು ಯಾವ ಯೋಜನೆಯಲ್ಲಿದ್ದರೂ, ನ್ಯೂಸ್‌ಲೆಟರ್ ಲಿಸ್ಟ್‌ನಲ್ಲಿ ಸೇರಬೇಕು, ಇದರಿಂದ ಅವರು ನಿಮ್ಮ ಅಪ್‌ಡೇಟ್‌ಗಳನ್ನು ತಪ್ಪಿಸಿಕೊಳ್ಳುವುದಿಲ್ಲ.

ಇದನ್ನು ಸಾಧಿಸಲು, ನೀವು Multiple Lists ಮೋಡ್ ಅನ್ನು ಬಳಸಿಕೊಂಡು Plan A MailChimp ಸೆಟ್ಟಿಂಗ್‌ಗಳಿಗೆ **Newsletter** ಮತ್ತು **Plan A** ಲಿಸ್ಟ್‌ಗಳನ್ನು ಸೇರಿಸಬಹುದು, ಮತ್ತು Plan B MailChimp ಸೆಟ್ಟಿಂಗ್‌ಗಳಿಗೆ **Newsletter** ಮತ್ತು **Plan B** ಲಿಸ್ಟ್‌ಗಳನ್ನು ಸೇರಿಸಬಹುದು.

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_ಹೊಸ ಖಾತೆಗಳು ಮತ್ತು ರದ್ದುಗೊಳಿಸಿದ ಖಾತೆಗಳಿಗೆ ಡೀಫಾಲ್ಟ್ ಲಿಸ್ಟ್‌ಗಳನ್ನು ಆಯ್ಕೆಮಾಡಿ_

### Mode 2: Single List + Multiple Groups (ಒಂದೇ ಲಿಸ್ಟ್ + ಬಹು ಗ್ರೂಪ್‌ಗಳು)

MailChimp ನಿಮ್ಮ ಲಿಸ್ಟ್‌ ಅನ್ನು **Groups** ಎಂಬ ಚಿಕ್ಕ ಬಳಕೆದಾರರ ಗುಂಪುಗಳಾಗಿ ವಿಭಾಗಿಸಲು (segment) ಅನುಮತಿಸುತ್ತದೆ. ಇದು ನಿಮಗೆ ಹೆಚ್ಚು ನಿಯಂತ್ರಣವನ್ನು ನೀಡುತ್ತದೆ, ಏಕೆಂದರೆ ನೀವು MailChimp ನಲ್ಲಿ ಒಂದೇ ರೀತಿಯ ಕ್ಯಾಂಪೇನ್‌ನ ಅನೇಕ ಪ್ರತಿಗಳನ್ನು ರಚಿಸುವ ಅಗತ್ಯವಿಲ್ಲದೆ, ನಿಮ್ಮ ಚಂದಾದarlarıಗಳ ನಿರ್ದಿಷ್ಟ ಉಪ-ಸಮೂಹಕ್ಕೆ (subset) ಕ್ಯಾಂಪೇನ್ ಕಳುಹಿಸಬಹುದು (ಇದು ನಂತರ ಆ ಕ್ಯಾಂಪೇನ್‌ನ ಮೆಟ್ರಿಕ್‌ಗಳನ್ನು ವಿಶ್ಲೇಷಿಸಲು ಕಷ್ಟವಾಗುತ್ತದೆ).

ಉದಾಹರಣೆಗೆ, ನೀವು **My Network Subscribers** ಎಂಬ ಒಂದೇ ಲಿಸ್ಟ್ ಅನ್ನು ಹೊಂದಬಹುದು, ಮತ್ತು ಅದರ ಒಳಗೆ **Plan A** ಮತ್ತು **Plan B** ಗಾಗಿ ಗ್ರೂಪ್‌ಗಳನ್ನು ಹೊಂದಬಹುದು.

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_“single list + multiple groups” ಸಂಯೋಜನಾ ಮೋಡ್ ಬಳಸುತ್ತಿದ್ದರೆ, ಡೀಫಾಲ್ಟ್ ಲಿಸ್ಟ್‌ಗಳು ಮತ್ತು ಗ್ರೂಪ್‌ಗಳನ್ನು ಆಯ್ಕೆಮಾಡಿ_

## Default Lists and Groups (ಡೀಫಾಲ್ಟ್ ಲಿಸ್ಟ್‌ಗಳು ಮತ್ತು ಗ್ರೂಪ್‌ಗಳು)

ಯಾವ ಸಂಯೋಜನಾ ವಿಧಾನವು ನಿಮ್ಮ ನಿರ್ದಿಷ್ಟ ಬಳಕೆಯ ಸಂದರ್ಭಕ್ಕೆ ಹೆಚ್ಚು ಸೂಕ್ತ ಎಂದು ನೀವು ಆಯ್ಕೆ ಮಾಡಿದ ನಂತರ, ಡೀಫಾಲ್ಟ್ ಲಿಸ್ಟ್‌ಗಳನ್ನು (ಮತ್ತು ಮೋಡ್ ಅನ್ನು ಅವಲಂಬಿಸಿ ಗ್ರೂಪ್‌ಗಳನ್ನು) ಆಯ್ಕೆ ಮಾಡುವ ಸಮಯ ಬಂದಿದೆ. ಒಂದು ನಿರ್ದಿಷ್ಟ ಲಿಸ್ಟ್ ಅನ್ನು ಆಯ್ಕೆ ಮಾಡದಿದ್ದರೆ, ಡೀಫಾಲ್ಟ್ ಲಿಸ್ಟ್ ಅನ್ನು ಬಳಸಲಾಗುತ್ತದೆ.

ಖಾತೆ ರದ್ದುಗೊಳಿಸಿದ ಬಳಕೆದಾರರಿಗಾಗಿ ನೀವು ಡೀಫಾಲ್ಟ್ ಲಿಸ್ಟ್ ಅನ್ನು ಸಹ ಆಯ್ಕೆ ಮಾಡಬಹುದು. ಯಾವುದೇ ಬಳಕೆದಾರರು ತಮ್ಮ ಚಂದಾದಾರಿಕೆಯನ್ನು ಮುಕ್ತಾಯಗೊಳಿಸಿ ಮತ್ತು ತಮ್ಮ ಖಾತೆಯನ್ನು ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಿಂದ ತೆಗೆದುಹಾಕಿದರೆ, ಅವರ ಇಮೇಲ್ ವಿಳಾಸವನ್ನು ರದ್ದುಗೊಳಿಸಿದ ಖಾತೆಗಳಿಗೆ ನಿಗದಿಪಡಿಸಿದ ಲಿಸ್ಟ್‌ಗೆ ವರ್ಗಾಯಿಸಲಾಗುತ್ತದೆ.

## Enabling Double Subscription (ಡಬಲ್ ಸಬ್‌ಸ್ಕ್ರಿಪ್ಷನ್ ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸುವುದು)

ಅದೇ ಸೆಟ್ಟಿಂಗ್ ಪುಟದಲ್ಲಿ, ನೀವು **Double Subscription** ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಲು ಸಾಧ್ಯವಾಗುತ್ತದೆ. ಈ ಆಯ್ಕೆಯು ಬಳಕೆದಾರರನ್ನು ಡೀಫಾಲ್ಟ್ ಲಿಸ್ಟ್ ಅನ್ನು ಆಯ್ಕೆ ಮಾಡಿದ 곳 ಮತ್ತು ಆ ಬಳಕೆದಾರರ ಯೋಜನೆಗೆ (plan) ಆಯ್ಕೆ ಮಾಡಿದ ಲಿಸ್ಟ್ ಎರಡರಲ್ಲೂ ಸೇರಿಸುತ್ತದೆ. ಈ ಆಯ್ಕೆಯನ್ನು ಸಕ್ರಿಯಗೊಳಿಸದಿದ್ದರೆ, ಬಳಕೆದಾರರನ್ನು ಆ ಬಳಕೆದಾರರ ಯೋಜನೆಗೆ ಆಯ್ಕೆ ಮಾಡಿದ ಲಿಸ್ಟ್‌ನಲ್ಲಿ ಮಾತ್ರ ಸೇರಿಸಲಾಗುತ್ತದೆ. ಆ ನಿರ್ದಿಷ್ಟ ಯೋಜನೆಗೆ ಏನೂ ನಿಗದಿಪಡಿಸದಿದ್ದರೆ, ಬಳಕೆದಾರರನ್ನು ಡೀಫಾಲ್ಟ್ ಲಿಸ್ಟ್‌ನಲ್ಲಿ ಸೇರಿಸಲಾಗುತ್ತದೆ.

<!-- Screenshot unavailable: MailChimp double subscription toggle setting -->

_ನೋಂದಾಯಿಸುವಾಗ ನಿಮ್ಮ ಬಳಕೆದಾರರನ್ನು ಡೀಫಾಲ್ಟ್ ಲಿಸ್ಟ್ ಮತ್ತು ಯೋಜನೆಯ ಲಿಸ್ಟ್ ಎರಡರಲ್ಲೂ ಚಂದಾದಾರರನ್ನಾಗಿ ಮಾಡಬಹುದು_

## Selecting Lists and Groups for each of the Plans (ಪ್ರತಿ ಯೋಜನೆಗೆ ಲಿಸ್ಟ್‌ಗಳು ಮತ್ತು ಗ್ರೂಪ್‌ಗಳನ್ನು ಆಯ್ಕೆಮಾಡುವುದು)

ನಿಮ್ಮ **Plans** ಗೆ ಹೋಗಿ ಮತ್ತು ನೀವು ಸಂಪಾದಿಸಲು ಬಯಸುವ ಯೋಜನೆಯನ್ನು ಆಯ್ಕೆಮಾಡಿ. ಗಮನಿಸಿ, **Advanced Options** ಮೆಟಾ-ಬಾಕ್ಸ್‌ನಲ್ಲಿ, MailChimp ಎಂಬ ಹೊಸ ಟ್ಯಾಬ್ ಲಭ್ಯವಿರುತ್ತದೆ.

ಆ ಟ್ಯಾಬ್‌ನಲ್ಲಿ, ಆ ಯೋಜನೆಗೆ ನೀವು ಅನೇಕ ಲಿಸ್ಟ್‌ಗಳು ಅಥವಾ ಒಂದೇ ಲಿಸ್ಟ್ ಮತ್ತು ಅನೇಕ ಗ್ರೂಪ್‌ಗಳನ್ನು (ಸಂಯೋಜನಾ ಮೋಡ್ ಅನ್ನು ಅವಲಂಬಿಸಿ) ಆಯ್ಕೆ ಮಾಡಲು ಸಾಧ್ಯವಾಗುತ್ತದೆ.
