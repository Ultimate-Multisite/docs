# ಮೀಟರ್ಡ್ ಪ್ಲಾನ್‌ಗಳ ಬದಲಾವಣೆಗಳ ದಾಖಲೆ

Version 1.1.0 - ಬಿಡುಗಡೆ ದಿನಾಂಕ: 2026-05-05
- ಹೊಸದಾಗಿ: ಮಲ್ಟಿಸಿಟ್ ಸಬ್‌ಸೈಟ್‌ಗಳಿಗೆ AI ಟೋಕನ್ ಬಿಲ್ಲಿಂಗ್ — ಕಾನ್ಫಿಗರ್ ಮಾಡಬಹುದಾದ ಪ್ರತಿ ಟೋಕನ್ ದರಗಳೊಂದಿಗೆ ಗ್ರಾಹಕರ সাইಟ್‌ಗಳಲ್ಲಿ AI ಟೋಕನ್ ಬಳಕೆಯನ್ನು ಟ್ರ್ಯಾಕ್ ಮಾಡಿ ಮತ್ತು ಬಿಲ್ ಮಾಡಿ.
- ಹೊಸದಾಗಿ: ಕನೆಕ್ಟರ್ ಎನ್‌ಫೋರ್ಸ್‌ಮೆಂಟ್ ಅನ್ನು ಡೈನಾಮಿಕ್ ಲಿಮಿಟ್ ಡಿಸ್ಕವರಿ ಮತ್ತು ರೈಟ್-ಥ್ರೂ ಜೊತೆಗೆ ಮರುನಿರ್ಮಿಸಲಾಗಿದೆ, ಇದು ಎಲ್ಲಾ ಕನೆಕ್ಟರ್‌ಗಳಲ್ಲಿ ನೈಜ-ಸಮಯದ ನಿಖರತೆಯನ್ನು ಖಚಿತಪಡಿಸುತ್ತದೆ.
- ತಿದ್ದುಪಡಿ: ಡೇಟಾಬೇಸ್ ಟೇಬಲ್ ಅಪ್‌ಗ್ರೇಡ್‌ಗಳು ಈಗ ಬರ್ಲಿನ್‌ಡಿಬಿ (BerlinDB) ಸ್ಕೀಮಾ ವ್ಯಾಖ್ಯಾನಗಳಿಗೆ ಸರಿಯಾಗಿ ಹೊಂದಿಕೆಯಾಗಿವೆ, ಇದು ಹೊಸ ಇನ್‌ಸ್ಟಾಲ್‌ಗಳಲ್ಲಿ ಅಪ್‌ಗ್ರೇಡ್ ವೈಫಲ್ಯಗಳನ್ನು ತಡೆಯುತ್ತದೆ.
- ತಿದ್ದುಪಡಿ: ಡೇಟಾಬೇಸ್ ಅಪ್‌ಗ್ರೇಡ್ ಕಾಲ್‌ಬ್ಯಾಕ್‌ಗಳನ್ನು ಸರಿಯಾದ ಸ್ವರೂಪಕ್ಕೆ ಪರಿವರ್ತಿಸಲಾಗಿದೆ, ಇದು ಮೌನ ಅಪ್‌ಗ್ರೇಡ್ ವೈಫಲ್ಯಗಳನ್ನು ಪರಿಹರಿಸುತ್ತದೆ.
- ತಿದ್ದುಪಡಿ: AI Usage Overage Markup ಇನ್‌ಪುಟ್ ಫೀಲ್ಡ್‌ನಲ್ಲಿ ಭಿನ್ನ ಮೌಲ್ಯಗಳನ್ನು (Fractional values) ಈಗ ಸ್ವೀಕರಿಸಲಾಗುತ್ತದೆ.
- ತಿದ್ದುಪಡಿ: ಪ್ಲಗ್ಇನ್ ಲೋಡ್ ಆಗುವಾಗ ಫೇಟಲ್ ಎರೆರ್ಸ್ ಮತ್ತು ಡಬಲ್-ಇನಿಷಿಯಲೈಸೇಶನ್ ಸಮಸ್ಯೆಗಳನ್ನು ಪರಿಹರಿಸಲಾಗಿದೆ.
- ಸುಧಾರಣೆ: `check-env` npm ಸ್ಕ್ರಿಪ್ಟ್ ಅನ್ನು ಸೇರಿಸಲಾಗಿದೆ, ಇದರಿಂದಾಗಿ Developer environments ಮೊದಲ ಬಾರಿಗೆ ರನ್ ಆದಾಗ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಕಾನ್ಫಿಗರ್ ಆಗುತ್ತದೆ.
