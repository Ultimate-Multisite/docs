# Filter: wu_captcha_whitelisted

ಪ್ರಸ್ತುತ ಬಳಕೆದಾರರು ಅಥವಾ ರೀಕ್ವೆಸ್ಟ್ ಅನ್ನು ವೈಟ್‌ಲಿಸ್ಟ್ ಮಾಡಬೇಕೇ ಎಂಬುದನ್ನು ಫಿಲ್ಟರ್ ಮಾಡುತ್ತದೆ.

## ಪ್ಯಾರಾಮೀಟರ್‌ಗಳು {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $whitelisted | `bool` | ಕ್ಯಾಪ್ಚಾ ವ್ಯಾಲಿಡೇಶನ್ ಅನ್ನು ಸ್ಕಿಪ್ ಮಾಡಬೇಕೇ (ಬಿಟ್ಟುಬಿಡಬೇಕೇ). |

### ಯಾವಾಗ ಲಭ್ಯವಾಯಿತು {#since}

- 1.2.0
