# Support Tickets (ಸಹಾಯ ಟಿಕೆಟ್‌ಗಳು)

**Support Tickets** ಎಂಬುದು Ultimate Multisite ಗಾಗಿ ಒಂದು ಶಕ್ತಿಶಾಲಿ addon ಆಗಿದೆ. ಇದು ನಿಮ್ಮ WordPress network ಗೆ ಸಮಗ್ರ ಸಹಾಯ ಟಿಕೆಟ್ (support ticket) ವ್ಯವಸ್ಥೆಯನ್ನು ಸೇರಿಸುತ್ತದೆ. SaaS ವೇದಿಕೆಗಳು, WaaS (Website as a Service) ಪೂರೈಕೆದಾರರು ಮತ್ತು network ಮಟ್ಟದಲ್ಲೂ ಹಾಗೂ 개별 site ಮಟ್ಟದಲ್ಲೂ ಗ್ರಾಹಕ ಬೆಂಬಲವನ್ನು ನೀಡಬೇಕಾದ multisite networkಗಳಿಗೆ ಇದು ಅತ್ಯುತ್ತಮವಾಗಿದೆ.

## Support Tickets ಏಕೆ ಆರಿಸಬೇಕು?

* **ಬಹು-ಮಟ್ಟದ ಬೆಂಬಲ (Multi-Level Support)**: network ಆಡಳಿತಗಾರರು (Network admins) network-ವ್ಯಾಪಿ ಟಿಕೆಟ್‌ಗಳನ್ನು ನಿರ್ವಹಿಸುತ್ತಾರೆ, ಆದರೆ site ಮಾಲೀಕರು ತಮ್ಮದೇ ಗ್ರಾಹಕ ಬೆಂಬಲವನ್ನು ನಿರ್ವಹಿಸುತ್ತಾರೆ.
* **ವೃತ್ತಿಪರ ಕಾರ್ಯಪ್ರವಾಹ (Professional Workflow)**: ಆದ್ಯತೆಗಳು (priorities), ಸ್ಥಿತಿಗಳು (statuses) ಮತ್ತು ನಿಯೋಜನೆಗಳೊಂದಿಗೆ ಸಂಪೂರ್ಣ ಟಿಕೆಟ್ life cycle management.
* **ಗ್ರಾಹಕರ ಅನುಭವ (Customer Experience)**: ಗ್ರಾಹಕರಿಗೆ ಸಹಾಯ ಪಡೆಯುವುದು ಸುಲಭವಾಗುವ ಸುಂದರವಾದ, responsive interface.
* **ಸುರಕ್ಷಿತ ಮತ್ತು ವಿಶ್ವಾಸಾರ್ಹ (Secure & Reliable)**: WordPress security best practices ಬಳಸಿ ನಿರ್ಮಿಸಲಾಗಿದೆ ಮತ್ತು ಆಪ್ಟಿಮೈಸ್ ಮಾಡಿದ database performance ಹೊಂದಿದೆ.
* **ಸುರಸಮಿಶ್ರಣ (Seamless Integration)**: Ultimate Multisite ನ ಗ್ರಾಹಕ ಮತ್ತು site management ಜೊತೆಗೆ ಪರಿಪೂರ್ಣವಾಗಿ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ.

## ಪ್ರಮುಖ ವೈಶಿಷ್ಟ್ಯಗಳು (Key Features)

**ಸಂಪೂರ್ಣ ಟಿಕೆಟ್ ನಿರ್ವಹಣೆ (Complete Ticket Management)**

* ಸುಲಭ ಉಲ್ಲೇಖಕ್ಕಾಗಿ ವಿಶಿಷ್ಟ ಟಿಕೆಟ್ ಸಂಖ್ಯೆ ಪದ್ಧತಿ.
* ಕಾನ್ಫಿಗರ್ ಮಾಡಬಹುದಾದ ಆದ್ಯತೆ ಮಟ್ಟಗಳು (Low, Normal, High, Critical).
* ಸಮಗ್ರ ಸ್ಥಿತಿ ಕಾರ್ಯಪ್ರವಾಹ (Open, In Progress, Waiting, Resolved, Closed).
* ಟಿಕೆಟ್ ವರ್ಗೀಕರಣ (General, Technical, Billing, Feature Request, Bug Report).
* ಐಚ್ಛಿಕ ಸ್ವಯಂಚಾಲಿತ ನಿಯೋಜನೆಯೊಂದಿಗೆ (optional auto-assignment) ಟಿಕೆಟ್‌ಗಳನ್ನು ಬೆಂಬಲ ಏಜೆಂಟ್‌ಗಳಿಗೆ ನಿಯೋಜಿಸುವುದು.

**ಥ್ರೆಡಿಂಗ್ ಸಂಭಾಷಣೆಗಳು (Threaded Conversations)**

* ಟಿಕೆಟ್ ಉತ್ತರಗಳಿಗಾಗಿ ಸಂಪೂರ್ಣ ಸಂಭಾಷಣಾ ಥ್ರೆಡಿಂಗ್.
* ಸಿಬ್ಬಂದಿ (staff) ಮತ್ತು ಗ್ರಾಹಕರ ಪ್ರತಿಕ್ರಿಯೆಗಳಿಗೆ ವಿಭಿನ್ನ ಶೈಲಿ.
* ಗ್ರಾಹಕರಿಗೆ ಕಾಣಿಸದ ಖಾಸಗಿ ಸಿಬ್ಬಂದಿ ಟಿಪ್ಪಣಿಗಳು (Private staff notes).
* ನೈಜ-ಸಮಯದ AJAX-ಚಾಲಿತ interface.
* ಫಾರ್ಮ್ಯಾಟಿಂಗ್ ಆಯ್ಕೆಗಳೊಂದಿಗೆ ಆಧುನಿಕ text editor.

**ಫೈಲ್ ಲಗತ್ತುಗಳು (File Attachments)**

* ಡ್ರ್ಯಾಗ್-ಅಂಡ್-ಡ್ರಾಪ್ ಬೆಂಬಲದೊಂದಿಗೆ ಸುರಕ್ಷಿತ ಫೈಲ್ ಅಪ್‌ಲೋಡ್.
* ಕಾನ್ಫಿಗರ್ ಮಾಡಬಹುದಾದ ಫೈಲ್ ಪ್ರಕಾರ ಮತ್ತು ಗಾತ್ರದ ನಿರ್ಬಂಧಗಳು.
* ದುರುದ್ದೇಶಪೂರಿತ ಫೈಲ್ ಪತ್ತೆಹಚ್ಚುವಿಕೆಯೊಂದಿಗೆ ಸುಧಾರಿತ ಭದ್ರತೆ.
* ಪ್ರವೇಶ ನಿಯಂತ್ರಣಗಳೊಂದಿಗೆ ರಕ್ಷಿತ ಡೌನ್‌ಲೋಡ್‌ಗಳು.

**ಸ್ಮಾರ್ಟ್ ನೋಟಿಫಿಕೇಶನ್‌ಗಳು (Smart Notifications)**

* ಹೊಸ ಟಿಕೆಟ್‌ಗಳು, ಉತ್ತರಗಳು ಮತ್ತು ಸ್ಥಿತಿ ಬದಲಾವಣೆಗಳಿಗೆ ಸ್ವಯಂಚಾಲಿತ ಇಮೇಲ್ ಎಚ್ಚರಿಕೆಗಳು.
* ಕಸ್ಟಮೈಸ್ ಮಾಡಬಹುದಾದ ಇಮೇಲ್ ಟೆಂಪ್ಲೇಟ್‌ಗಳು.
* ಸಿಬ್ಬಂದಿ ಮತ್ತು ಗ್ರಾಹಕರಿಗಾಗಿ ಪಾತ್ರ ಆಧಾರಿತ ನೋಟಿಫಿಕೇಶನ್‌ಗಳು.
* ವೈಯಕ್ತೀಕರಿಸಿದ ಸಂದೇಶಗಳಿಗಾಗಿ ಡೈನಾಮಿಕ್ ಟೆಂಪ್ಲೇಟ್ ವೇರಿಯೇಬಲ್‌ಗಳು.

**ಶಕ್ತಿಶಾಲಿ ಆಡಳಿತಾತ್ಮಕ interface (Powerful Admin Interface)**

* ಸಾರ್ಟಿಂಗ್ ಮತ್ತು ಫಿಲ್ಟರಿಂಗ್‌ನೊಂದಿಗೆ ಸುಧಾರಿತ ಟಿಕೆಟ್ ಪಟ್ಟಿ.
* ಸ್ಥಿತಿ, ಆದ್ಯತೆ ಮತ್ತು ನಿಯೋಜನೆಗಳಿಗಾಗಿ ತ್ವರಿತ ಸಂಪಾದನೆ (Quick edit).
* ಸಮರ್ಥ ನಿರ್ವಹಣೆಗಾಗಿ ಬೃಹತ್ ಕ್ರಿಯೆಗಳು (Bulk actions).
* ಕಾರ್ಯಕ್ಷಮತೆ ಮೆಟ್ರಿಕ್‌ಗಳೊಂದಿಗೆ ಸ್ಟ್ಯಾಟಿಸ್ಟಿಕ್ಸ್ ಡ್ಯಾಶ್‌ಬೋರ್ಡ್.
* ಸಮಗ್ರ ಸೆಟ್ಟಿಂಗ್ಸ್ ಪ್ಯಾನೆಲ್.

**ಸುಂದರ ಗ್ರಾಹಕ ಪೋರ್ಟಲ್ (Beautiful Customer Portal)**

* ಎಲ್ಲಾ ಸಾಧನಗಳಲ್ಲಿ ಸ್ವಚ್ಛವಾದ, responsive interface.
* ಮೊಬೈಲ್-ಮೊದಲ ವಿನ್ಯಾಸ (Mobile-first design).
* ಸುಧಾರಿತ ಹುಡುಕಾಟ ಮತ್ತು ಫಿಲ್ಟರಿಂಗ್.
* ಫಾರ್ಮ್ಯಾಟಿಂಗ್ ಟೂಲ್‌ಬಾರ್‌ನೊಂದಿಗೆ ಆಧುನಿಕ ಎಡಿಟರ್.
* ಬಳಸಲು ಸುಲಭವಾದ ಟಿಕೆಟ್ ಸಲ್ಲಿಸುವ ಫಾರ್ಮ್‌ಗಳು.

**ಹೆಚ್ಚು ಹೊಂದಿಕೊಳ್ಳುವ ಸಂಯೋಜನೆ (Flexible Integration)**

* ಶಾರ್ಟ್‌ಕೋಡ್‌ಗಳು ಅಥವಾ Ultimate Multisite UI ಅಂಶಗಳನ್ನು ಬಳಸಿ.
* ನಿಮ್ಮ ಸೈಟ್‌ನ ಎಲ್ಲಿ ಬೇಕಾದರೂ ಟಿಕೆಟ್ ಫಾರ್ಮ್‌ಗಳು ಮತ್ತು ಪಟ್ಟಿಗಳನ್ನು ಪ್ರದರ್ಶಿಸಿ.
* multisite network structure ಜೊತೆಗೆ ಸುಗಮವಾಗಿ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ.
* hooks ಮತ್ತು filters ಮೂಲಕ ವಿಸ್ತರಿಸಬಹುದು.

## ಯಾರಿಗಾಗಿ ಸೂಕ್ತ?

* ಗ್ರಾಹಕ ಬೆಂಬಲದ ಅಗತ್ಯವಿರುವ SaaS ವೇದಿಕೆಗಳು.
* ಅನೇಕ client sites ಅನ್ನು ನಿರ್ವಹಿಸುವ WaaS ಪೂರೈಕೆದಾರರು.
* ಹಂತ ಹಂತದ ಬೆಂಬಲದೊಂದಿಗೆ ಸದಸ್ಯತ್ವ ಸೈಟ್‌ಗಳು.
* ಅನೇಕ ಬ್ರ್ಯಾಂಡ್‌ಗಳನ್ನು ನಿರ್ವಹಿಸುವ network ಆಡಳಿತಗಾರರು.
* ಗ್ರಾಹಕ ಸೇವೆಯನ್ನು ಒದಗಿಸುವ site ಮಾಲೀಕರು.

## ಅಳವಡಿಕೆ (Installation)

1. ನಿಮ್ಮ WordPress network ನಲ್ಲಿ Ultimate Multisite plugin ಅನ್ನು ಅಳವಡಿಸಿ ಮತ್ತು ಸಕ್ರಿಯಗೊಳಿಸಲಾಗಿದೆ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ.
2. addon ಫೈಲ್‌ಗಳನ್ನು ನಿಮ್ಮ `/wp-content/plugins/` ಡೈರೆಕ್ಟರಿಗೆ ಅಪ್‌ಲೋಡ್ ಮಾಡಿ.
3. WordPress ನಲ್ಲಿ 'Plugins' ಮೆನುವಿನ ಮೂಲಕ network ನಲ್ಲಿ plugin ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ.
4. ನಿಮ್ಮ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಲು Network Admin > Support Tickets ಗೆ ನ್ಯಾವಿಗೇಟ್ ಮಾಡಿ.

## ಆಗಾಗ್ಗೆ ಕೇಳಲಾಗುವ ಪ್ರಶ್ನೆಗಳು (Frequently Asked Questions)

## ಇದಕ್ಕಾಗಿ Ultimate Multisite ಅಗತ್ಯವಿದೆಯೇ?

ಹೌದು, ಈ addon ಅನ್ನು ಅಳವಡಿಸಲು Ultimate Multisite ಅನ್ನು ಅಳವಡಿಸಿ ಮತ್ತು ಸಕ್ರಿಯಗೊಳಿಸುವುದು ಅಗತ್ಯ. ಇದು Ultimate Multisite ನ ಗ್ರಾಹಕ ಮತ್ತು site management ವೈಶಿಷ್ಟ್ಯಗಳೊಂದಿಗೆ ಆಳವಾಗಿ ಸಂಯೋಜಿಸುತ್ತದೆ.

## site ಮಾಲೀಕರು ತಮ್ಮದೇ ಗ್ರಾಹಕ ಬೆಂಬಲವನ್ನು ನಿರ್ವಹಿಸಬಹುದೇ?

ಖಂಡಿತ! site ಮಾಲೀಕರು ತಮ್ಮದೇ ಗ್ರಾಹಕರಿಗೆ ಬೆಂಬಲ ನೀಡಲು ಟಿಕೆಟ್ ವ್ಯವಸ್ಥೆಯನ್ನು ಬಳಸಬಹುದು, ಆದರೆ network ಆಡಳಿತಗಾರರು network-ವ್ಯಾಪಿ support ಟಿಕೆಟ್‌ಗಳನ್ನು ನಿರ್ವಹಿಸಬಹುದು.

## ಗ್ರಾಹಕರು ಟಿಕೆಟ್‌ಗಳನ್ನು ಹೇಗೆ ಸಲ್ಲಿಸುತ್ತಾರೆ?

ಶಾರ್ಟ್‌ಕೋಡ್‌ಗಳು ಅಥವಾ Ultimate Multisite UI ಅಂಶಗಳನ್ನು ಬಳಸಿ ನಿಮ್ಮ ಸೈಟ್‌ನ ಎಲ್ಲಿ ಬೇಕಾದರೂ ಟಿಕೆಟ್ ಸಲ್ಲಿಸುವ ಫಾರ್ಮ್‌ಗಳನ್ನು ಸೇರಿಸಬಹುದು. ಗ್ರಾಹಕರು ಸರಳವಾಗಿ ಫಾರ್ಮ್ ಅನ್ನು ಭರ್ತಿ ಮಾಡಿ ತಮ್ಮ ವಿನಂತಿಯನ್ನು ಸಲ್ಲಿಸುತ್ತಾರೆ.

## ನಾನು ಟಿಕೆಟ್ ಸ್ಥಿತಿಗಳು ಮತ್ತು ಆದ್ಯತೆಗಳನ್ನು ಕಸ್ಟಮೈಸ್ ಮಾಡಬಹುದೇ?

ಹೌದು! ನಿಮ್ಮ ಬೆಂಬಲ ಕಾರ್ಯಪ್ರವಾಹಕ್ಕೆ ಹೊಂದಿಕೊಳ್ಳಲು ನೀವು ಆದ್ಯತೆ ಮಟ್ಟಗಳು, ಸ್ಥಿತಿಗಳು ಮತ್ತು ಟಿಕೆಟ್ ಪ್ರಕಾರಗಳನ್ನು network admin ಸೆಟ್ಟಿಂಗ್‌ಗಳಲ್ಲಿ ಸಂಪೂರ್ಣವಾಗಿ ಕಸ್ಟಮೈಸ್ ಮಾಡಬಹುದು.

## ಇದು ಫೈಲ್ ಲಗತ್ತುಗಳನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆಯೇ?

ಹೌದು, ಗ್ರಾಹಕರು ಮತ್ತು ಸಿಬ್ಬಂದಿ ಇಬ್ಬರೂ ಟಿಕೆಟ್‌ಗಳು ಮತ್ತು ಉತ್ತರಗಳಿಗೆ ಫೈಲ್‌ಗಳನ್ನು ಲಗತ್ತಿಸಬಹುದು. ಭದ್ರತೆಗಾಗಿ ನೀವು ಅನುಮತಿಸಲಾದ ಫೈಲ್ ಪ್ರಕಾರಗಳು ಮತ್ತು ಗಾತ್ರದ ಮಿತಿಗಳ ಮೇಲೆ ಸಂಪೂರ್ಣ ನಿಯಂತ್ರಣವನ್ನು ಹೊಂದಿದ್ದೀರಿ.

## ನೋಟಿಫಿಕೇಶನ್‌ಗಳನ್ನು ಹೇಗೆ ನಿರ್ವಹಿಸಲಾಗುತ್ತದೆ?

ಹೊಸ ಟಿಕೆಟ್‌ಗಳು, ಉತ್ತರಗಳು ಮತ್ತು ಸ್ಥಿತಿ ಬದಲಾವಣೆಗಳಿಗೆ ವ್ಯವಸ್ಥೆಯು ಸ್ವಯಂಚಾಲಿತ ಇಮೇಲ್ ನೋಟಿಫಿಕೇಶನ್‌ಗಳನ್ನು ಕಳುಹಿಸುತ್ತದೆ. ಎಲ್ಲಾ ಇಮೇಲ್ ಟೆಂಪ್ಲೇಟ್‌ಗಳನ್ನು ನೀವು ಸೆಟ್ಟಿಂಗ್‌ಗಳಲ್ಲಿ ಕಸ್ಟಮೈಸ್ ಮಾಡಬಹುದು.

## ನಾನು ಟಿಕೆಟ್‌ಗಳನ್ನು ನಿರ್ದಿಷ್ಟ support ಏಜೆಂಟ್‌ಗಳಿಗೆ ನಿಯೋಜಿಸಬಹುದೇ?

ಹೌದು, ನೀವು ಟಿಕೆಟ್‌ಗಳನ್ನು ಏಜೆಂಟ್‌ಗಳಿಗೆ ಕೈಯಾರೆ ನಿಯೋಜಿಸಬಹುದು ಅಥವಾ ಟಿಕೆಟ್‌ಗಳನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ವಿತರಿಸಲು auto-assignment ಅನ್ನು ಆನ್ ಮಾಡಬಹುದು.

## ಸಿಬ್ಬಂದಿಗಾಗಿ ಖಾಸಗಿ ಟಿಪ್ಪಣಿಗಳು ಇದೆಯೇ?

ಹೌದು, ಸಿಬ್ಬಂದಿ ಸದಸ್ಯರು ಖಾಸಗಿ ಉತ್ತರಗಳನ್ನು ಸೇರಿಸಬಹುದು, ಅದು ಇತರ ಸಿಬ್ಬಂದಿ ಸದಸ್ಯರಿಗೆ ಮಾತ್ರ ಗೋಚರಿಸುತ್ತದೆ, ಗ್ರಾಹಕರಿಗೆ ಅಲ್ಲ.

## ಇದು ಮೊಬೈಲ್-ಸ್ನೇಚಿಯಾಗಿವೆಯೇ?

ಖಂಡಿತ! ಸಂಪೂರ್ಣ interface ಅನ್ನು mobile-first ವಿಧಾನದೊಂದಿಗೆ ನಿರ್ಮಿಸಲಾಗಿದೆ ಮತ್ತು ಎಲ್ಲಾ ಸಾಧನಗಳಲ್ಲಿ ಸುಂದರವಾಗಿ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ.

## ನಾನು ಶಾರ್ಟ್‌ಕೋಡ್‌ಗಳನ್ನು ಬಳಸಿ ಟಿಕೆಟ್ ಪಟ್ಟಿಗಳು ಮತ್ತು ಫಾರ್ಮ್‌ಗಳನ್ನು ಪ್ರದರ್ಶಿಸಬಹುದೇ?

ಹೌದು! ಲಭ್ಯವಿರುವ ಶಾರ್ಟ್‌ಕೋಡ್‌ಗಳು:
* `[wu_support_tickets]` - ಗ್ರಾಹಕರ ಟಿಕೆಟ್ ಪಟ್ಟಿಯನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ
* `[wu_support_ticket_form]` - ಹೊಸ ಟಿಕೆಟ್ ಸಲ್ಲಿಸುವ ಫಾರ್ಮ್ ಅನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ
* `[wu_support_ticket_view ticket_id="123"]` - ನಿರ್ದಿಷ್ಟ ಟಿಕೆಟ್ ಅನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ

## ಯಾವ ಫೈಲ್ ಪ್ರಕಾರಗಳನ್ನು ಅಪ್‌ಲೋಡ್ ಮಾಡಬಹುದು?

ನೀವು ಸೆಟ್ಟಿಂಗ್‌ಗಳಲ್ಲಿ ಅನುಮತಿಸಲಾದ ಫೈಲ್ ಪ್ರಕಾರಗಳನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಬಹುದು. ದುರುದ್ದೇಶಪೂರಿತ ಫೈಲ್‌ಗಳಿಂದ ರಕ್ಷಿಸಲು ವ್ಯವಸ್ಥೆಯು ಸುಧಾರಿತ ಭದ್ರತಾ ಸ್ಕ್ಯಾನಿಂಗ್ ಅನ್ನು ಒಳಗೊಂಡಿದೆ.

## ನಾನು ಟಿಕೆಟ್ ಡೇಟಾವನ್ನು ರಫ್ತು (export) ಮಾಡಬಹುದೇ?

ಹೌದು, addon ನಲ್ಲಿ ಟಿಕೆಟ್ ಡೇಟಾ ಮತ್ತು ಅಂಕಿಅಂಶಗಳಿಗಾಗಿ export ಸಾಮರ್ಥ್ಯಗಳನ್ನು ಒಳಗೊಂಡಿದೆ.
