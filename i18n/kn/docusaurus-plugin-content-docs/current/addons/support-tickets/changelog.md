### 1.0.3 - 2026-05-01 {#103---2026-05-01}
* ತಿದ್ದುಪಡಿ: nullable ಟಿಕೆಟ್ ಮಾಡೆಲ್ ಸೆಟರ್‌ಗಳಲ್ಲಿ ನಲ್ (null) ಮೌಲ್ಯಗಳನ್ನು ಸ್ವೀಕರಿಸಲು.
* ತಿದ್ದುಪಡಿ: ಸರಿಯಾದ ಮಲ್ಟಿಸೈಟ್ ಬೆಂಬಲಕ್ಕಾಗಿ ಎಲ್ಲಾ ಟಿಕೆಟ್‌ಗಳನ್ನು ಗ್ಲೋಬಲ್ ನೆಟ್‌ವರ್ಕ್-ವೈಡ್ ಟೇಬಲ್‌ಗಳಲ್ಲಿ ಸಂಗ್ರಹಿಸಲಾಗಿದೆ.
* ತಿದ್ದುಪಡಿ: ಸ್ಟಾಫ್ ಫೀಲ್ಡ್‌ಗಳನ್ನು ಮರೆಮಾಚಲಾಗಿದೆ ಮತ್ತು ಗ್ರಾಹಕರ new-ticket ಫಾರ್ಮ್‌ನಲ್ಲಿ ಬ್ಲಾಂಕ್-పేಜ್ ರಿಡೈರೆಕ್ಟ್ ಅನ್ನು ಸರಿಪಡಿಸಲಾಗಿದೆ.
* ತಿದ್ದುಪಡಿ: ಮೆಟಾಡೇಟಾ ಸಂಗ್ರಹಣೆಗಾಗಿ undefined add_meta() ಕರೆಗಳನ್ನು ಸರಿಯಾದ update_meta() ಯೊಂದಿಗೆ ಬದಲಾಯಿಸಲಾಗಿದೆ.
* ತಿದ್ದುಪಡಿ: ಅಸಂಘಟಿತ capability ಚೆಕ್‌ ಅನ್ನು ಸರಿಯಾದ wu_view_all_support_tickets ಯೊಂದಿಗೆ ಬದಲಾಯಿಸಲಾಗಿದೆ.
* ತಿದ್ದುಪಡಿ: ಟಿಕೆಟ್ ಸ್ಥಿತಿ, ನಿಯೋಜನೆ ಮತ್ತು ತ್ವರಿತ-ಸಂಪಾದನೆ ಕ್ರಿಯೆಗಳಿಗಾಗಿ ಕಾಣೆಯಾದ AJAX ಹ್ಯಾಂಡಲರ್‌ಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ.
* ತಿದ್ದುಪಡಿ: ಟಿಕೆಟ್ ವೀಸ್‌‌ನಲ್ಲಿ ಸ್ಟಾಫ್ ಪ್ರತಿಕ್ರಿಯೆ ಪತ್ತೆಹಚ್ಚಲು ವಿಧಾನದ ಹೆಸರನ್ನು ಸರಿಪಡಿಸಲಾಗಿದೆ.
* ತಿದ್ದುಪಡಿ: ನಕಲಿ ಪ್ರತಿಕ್ರಿಯೆ ಹ್ಯಾಂಡಲರ್‌ಗಳನ್ನು ಸಂಘಟಿಸಿ ಮತ್ತು nonce ಕ್ರಿಯೆಗಳ ಹೆಸರುಗಳನ್ನು சீರಾಗಿಸಲಾಗಿದೆ.
* ತಿದ್ದುಪಡಿ: [wu_submit_ticket] ಶಾರ್ಟ್‌ಕೋಡ್‌ಗಾಗಿ ಕಾಣೆಯಾದ ಫ್ರ front-end ವೀವನ್ನು ಸೇರಿಸಲಾಗಿದೆ.
* ತಿದ್ದುಪಡಿ: ಕಾಣೆಯಾದ user_id ಕಾಲಮ್ ಅನ್ನು ಸೇರಿಸಲಾಗಿದೆ ಮತ್ತು Support_Ticket::get_user_id() ವಿಧಾನವನ್ನು ಸರಿಪಡಿಸಲಾಗಿದೆ.
* ತಿದ್ದುಪಡಿ: ಪ್ರಿオリಟಿ ಫಿಲ್ಟರ್ ಆಯ್ಕೆ ಟ್ಯಾಗ್‌ನಲ್ಲಿರುವ ಹೆಚ್ಚುವರಿ ಡಬಲ್-ಕೋಟ್ ಅನ್ನು ತೆಗೆದುಹಾಕಲಾಗಿದೆ.
* ತಿದ್ದುಪಡಿ: ಸೂಪರ್ ಅಡ್ಮին್‌ಗಳಿಗಾಗಿ ನೆಟ್‌ವರ್ಕ್ ಅಡ್ಮಿನ್ ಟಿಕೆಟ್ ನಿರ್ವಹಣಾ ಪ್ಯಾನೆಲ್ ಅನ್ನು ಸೇರಿಸಲಾಗಿದೆ.
* ಸುಧಾರಣೆ: ಅಡ್ಮಿನ್ CSS ಅನ್ನು ಒಂದೇ ಬಾಹ್ಯ ಶೈಲಿ ಹಾಳೆಯಲ್ಲಿ ಸಂಘಟಿಸಲಾಗಿದೆ.
* ಸುಧಾರಣೆ: ಸಬ್‌ಸೈಟ್ ಅಡ್ಮಿನ್ ಮೆನುವಿನಿಂದ ಸೆಟ್ಟಿಂಗ್ಸ್ ಸಬ್‌ಮೆನುವನ್ನು ತೆಗೆದುಹಾಕಲಾಗಿದೆ.
* ಸುಧಾರಣೆ: ಸಪೋರ್ಟ್ ಟಿಕೆಟ್ ಪುಟಗಳಲ್ಲಿ ಮಾತ್ರ ಫ್ರ front-end ಆಸ್ತಿಗಳನ್ನು (assets) koşitionally ಲೋಡ್ ಮಾಡಲಾಗಿದೆ.
* ಸುಧಾರಣೆ: Bedrock ರೂಟ್ ಆಟೋಲೋಡರ್ ಈಗಾಗಲೇ ಅವಲಂಬನೆಗಳನ್ನು ಲೋಡ್ ಮಾಡಿದ್ದರೆ, ಪ್ಲಗ್ಇನ್ ಆಟೋಲೋಡರ್ ಅನ್ನು ಬಿಟ್ಟುಬಿಡಲಾಗುತ್ತದೆ.

### 1.0.2 - 2025-12-11 {#102---2025-12-11}
* ಸೇರಿಸಲಾಗಿದೆ: ಆಡ್ಮಿನ್ ಮತ್ತು ಫ್ರ front-end ಗಾಗಿ ಸಂಪೂರ್ಣ ಟಿಕೆಟ್ ವೀಕ್ಷಣಾ ಕಾರ್ಯನಿರ್ವಹಣೆ.
* ಸೇರಿಸಲಾಗಿದೆ: ಟಿಕೆಟ್ ಪ್ರತಿಕ್ರಿಯೆ ಸಲ್ಲಿಸುವಿಕೆಗಾಗಿ AJAX ಹ್ಯಾಂಡಲರ್.
* ಸೇರಿಸಲಾಗಿದೆ: ಸರಿಯಾದ ಫಾರ್ಮ್ ನಿರ್ವಹಣೆಯೊಂದಿಗೆ ಟಿಕೆಟ್ ಪ್ರತಿಕ್ರಿಯೆ ಕಾರ್ಯನಿರ್ವಹಣೆಗೆ ಬೆಂಬಲ.
* ಸೇರಿಸಲಾಗಿದೆ: ಆಡ್ಮಿನ್‌ನಲ್ಲಿ ಟಿಕೆಟ್ ಸಲ್ಲಿಸುವಿಕೆಗಳು ಮತ್ತು ಪ್ರತಿಕ್ರಿಯೆಗಳಿಗೆ ಸರಿಯಾದ ನೋಟಿಸ್ ಪ್ರದರ್ಶನ.
* ಸೇರಿಸಲಾಗಿದೆ: ಗ್ರಾಹಕರ ಸ್ವಯಂ-ಸಲ್ಲಿಸುವಿಕೆಗಳಿಗೆ ಪ್ರಸ್ತುತ ಬಳಕೆದಾರರೊಂದಿಗೆ ಸ್ವಯಂಚಾಲಿತ ಟಿಕೆಟ್ ಅಸೋಸಿಯೇಷನ್.
* ಸೇರಿಸಲಾಗಿದೆ: ಗ್ರಾಹಕರು ಟಿಕೆಟ್ ಮಾಲೀಕತ್ವವನ್ನು ಬದಲಾಯಿಸುವುದನ್ನು ತಡೆಯಲು ಭದ್ರತಾ ಸುಧಾರಣೆ.
* ಸೇರಿಸಲಾಗಿದೆ: ಕಾಣೆಯಾದ ಸಹಾಯಕರ ಕಾರ್ಯಗಳು (wu_format_date, wu_user_can_view_ticket, ಇತ್ಯಾದಿ).
* ಸೇರಿಸಲಾಗಿದೆ: ಸರಿಯಾದ ಫೈಲ್ ಲಗತ್ತು ಡೌನ್‌ಲೋಡ್ ಮತ್ತು ನಿರ್ವಹಣೆ.
* ಸೇರಿಸಲಾಗಿದೆ: ಟಿಕೆಟ್ ಪ್ರತಿಕ್ರಿಯೆಗಳು ಮತ್ತು ಸ್ಥಿತಿ ಬದಲಾವಣೆಗಳಿಗೆ ಇಮೇಲ್ ಅಧಿಸೂಚನೆ ವ್ಯವಸ್ಥೆ.
* ತಿದ್ದುಪಡಿ: ಟಿಕೆಟ್ ಪ್ರತಿಕ್ರಿಯೆ ಫಾರ್ಮ್ ಈಗ ಅಗತ್ಯ ಟಿಕೆಟ್ ID ಅನ್ನು ಒಳಗೊಂಡಿದೆ.
* ತಿದ್ದುಪಡಿ: ಅಡ್ಮಿನ್ ಟಿಕೆಟ್ ವೀಸ್‌ಗಳಲ್ಲಿ ನೋಟಿಸ್ ಗೋಚರತೆ ಸಮಸ್ಯೆಗಳನ್ನು ಸರಿಪಡಿಸಲಾಗಿದೆ.
* ತಿದ್ದುಪಡಿ: ಸಪೋರ್ಟ್ ಟಿಕೆಟ್ ಕಾರ್ಯಗಳಲ್ಲಿ ಸಿಂಟ್ಯಾಕ್ಸ್ ದೋಷಗಳನ್ನು ಸರಿಪಡಿಸಲಾಗಿದೆ.
* ಸುಧಾರಣೆ: ಪ್ರತಿಕ್ರಿಯೆಗಳು ಮತ್ತು ಲಗತ್ತುಗಳಿಗಾಗಿ ಸರಿಯಾದ ಕ್ವೆರಿ ಮತ್ತು ಸ್ಕೀಮಾ ವರ್ಗಗಳೊಂದಿಗೆ ಡೇಟಾಬೇಸ್ ರಚನೆ.
* ಸುಧಾರಣೆ: ಟಿಕೆಟ್ ಮಾಡೆಲ್‌ಗಳಿಗಾಗಿ ಆಟ್ರಿಬ್ಯೂಟ್‌ಗಳ ಬದಲು ನಿಜವಾದ ಗುಣಲಕ್ಷಣಗಳಿಗೆ (real properties) ಮೈಗ್ರೇಷನ್.
