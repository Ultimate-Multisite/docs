# Ultimate Multisite ದಾಖಲೆಗಳು {#ultimate-multisite-documentation}

Ultimate Multisite ದಾಖಲೆಗಳಿಗೆ ಸ್ವಾಗತ. Ultimate Multisite ಅನ್ನು WordPress Multisite ಯನ್ನು ಸಂಪೂರ್ಣ Website as a Service (WaaS) ವೇದಿಕೆಯಾಗಿ ಪರಿವರ್ತಿಸುತ್ತದೆ.

## ದಾಖಲೆ ವಿಭಾಗಗಳು {#documentation-sections}

### [User Guide](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

ನಿಮ್ಮ WaaS ವೇದಿಕೆಯನ್ನು ಸೆಟಪ್ ಮಾಡಲು, ಕಾನ್ಫಿಗರ್ ಮಾಡಲು ಮತ್ತು ನಿರ್ವಹಿಸಲು ಬೇಕಾದ ಎಲ್ಲವನ್ನೂ ಇಲ್ಲಿ ಸಿಗುತ್ತದೆ. ಇದು installation, payment gateways, domain mapping, client management ಮತ್ತು ಹೆಚ್ಚಿನದರ ಬಗ್ಗೆ ಒಳಗೊಂಡಿದೆ.

### [Developer Documentation](/developer/hooks) {#developer-documentation}

Ultimate Multisite ಮೇಲೆ ಕೆಲಸ ಮಾಡುವ developers ಗಾಗಿ ಇದು ಒಂದು technical reference. ಇದರಲ್ಲಿ hooks reference, REST API documentation, ಮತ್ತು addon development guides ಸೇರಿವೆ.

### [Addons](/addons) {#addons}

ಇವುಗಳು paid addon plugins ಗಳ ದಾಖಲೆಗಳು. ಇವು Ultimate Multisite ಗೆ WooCommerce integration, analytics, domain selling ಮತ್ತು ಹೆಚ್ಚಿನ ವೈಶಿಷ್ಟ್ಯಗಳನ್ನು ಸೇರಿಸುತ್ತವೆ.

## ಸಹಾಯ ಪಡೆಯುವುದು ಹೇಗೆ {#getting-help}

- [GitHub Issues](https://github.com/Ultimate-Multisite/ultimate-multisite/issues) -- ಬಗ್‌ಗಳನ್ನು ವರದಿ ಮಾಡಲು
- [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) -- core plugin ಅನ್ನು ಡೌನ್‌ಲೋಡ್ ಮಾಡಲು
