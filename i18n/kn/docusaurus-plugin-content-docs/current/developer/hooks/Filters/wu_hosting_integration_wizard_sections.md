# Filter: wu_hosting_integration_wizard_sections {#filter-wuhostingintegrationwizardsections}

ಹೋಸ್ಟಿಂಗ್ ಇಂಟಿಗ್ರೇಷನ್ ಸೆಟಪ್‌ಗಾಗಿ ವೀಜಾರ್ಡ್ ವಿಭಾಗಗಳನ್ನು ಫಿಲ್ಟರ್ ಮಾಡುತ್ತದೆ.

Addons ನಮಗೆ ವೀಜಾರ್ಡ್ ವಿಭಾಗಗಳನ್ನು ಸೇರಿಸಲು, ತೆಗೆದುಹಾಕಲು ಅಥವಾ ಮಾರ್ಪಡಿಸಲು ಅನುಮತಿಸುತ್ತದೆ.

| Name | Type | Description |
|------|------|-------------|
| $sections | `array` | ವೀಜಾರ್ಡ್ ವಿಭಾಗಗಳು. |
| $integration | `\WP_Ultimo\Integrations\Host_Providers\Base_Host_Provider\|\WP_Ultimo\Integrations\Integration` | ಕಾನ್ಫಿಗರ್ (configure) ಮಾಡುತ್ತಿರುವ ಇಂಟಿಗ್ರೇಷನ್. |
