# ಫಿಲ್ಟರ್: wu_sso_enabled

ಕ್ರೋಸ್-ಡೊಮೈನ್ ಸಿಂಗಲ್-ಸೈನ್-ಆನ್ (single-sign-on) ಸಾಮರ್ಥ್ಯವನ್ನು ಆನ್ ಅಥವಾ ಆಫ್ ಮಾಡಲು ಇದನ್ನು ಬಳಸಲಾಗುತ್ತದೆ.

ಸಿಂಗಲ್-ಸೈನ್-ಆನ್ ಅನ್ನು ಸಂಪೂರ್ಣವಾಗಿ ಆಫ್ ಮಾಡಲು ಅಥವಾ ಅದನ್ನು ನಿರ್ದಿಷ್ಟ ಸಂದರ್ಭಗಳಲ್ಲಿ ಮಾತ್ರ ಆನ್ ಮಾಡಲು ಈ ಮೌಲ್ಯವನ್ನು ಫಿಲ್ಟರ್ ಮಾಡಬಹುದು.

## ಪ್ಯಾರಾಮೀಟರ್‌ಗಳು {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $enabled | `bool` | SSO ಅನ್ನು ಆನ್ ಮಾಡಬೇಕೇ? ಆನ್ ಮಾಡಲು `True`, ಆಫ್ ಮಾಡಲು `false` ಬಳಸಿ. |

### Since {#since}

- 2.0.11
### Source {#source}

[`inc/sso/class-sso.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/sso/class-sso.php#L110) ನಲ್ಲಿ ಲೈನ್ 110 ರಲ್ಲಿ ವ್ಯಾಖ್ಯಾನಿಸಲಾಗಿದೆ


## Returns {#returns}
SSO ಅನ್ನು ಆನ್ ಮಾಡಲಾಗಿದೆಯೇ ಅಥವಾ ಇಲ್ಲವೇ ಎಂಬುದನ್ನು ತಿಳಿಸುತ್ತದೆ.
