# Filter: wu_stripe_generate_idempotency_key {#filter-wustripegenerateidempotencykey}

Stripe ಚಾರ್ಜ್ ಆಯ್ಕೆಗಳೊಂದಿಗೆ ಕಳುಹಿಸಲಾದ `idempotency_key` ಮೌಲ್ಯವನ್ನು ಫಿಲ್ಟರ್ ಮಾಡುತ್ತದೆ.

## ಪ್ಯಾರಾಮೀಟರ್‌ಗಳು {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $idempotency_key | `string` | `idempotency key` ನ ಮೌಲ್ಯ. |
| $args | `array` | ಕೀ ಅನ್ನು ರಚಿಸಲು ಸಹಾಯ ಮಾಡುವ ಆರ್ಗ್ಯುಮೆಂಟ್‌ಗಳು. |
| $context | `string` | `idempotency key` ಅನ್ನು ರಚಿಸುವ ಸಂದರ್ಭ (Context). |

- 3.5.0
