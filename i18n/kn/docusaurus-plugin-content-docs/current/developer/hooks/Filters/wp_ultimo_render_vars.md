# Filter: wp_ultimo_render_vars {#filter-wpultimorendervars}

ಪ್ಲಗ್ಇನ್ developers ಗಳಿಗೆ ರೇಂಡರ್ ಸಂದರ್ಭಕ್ಕೆ (render context) ಜಾಗತಿಕವಾಗಿ ಹೆಚ್ಚುವರಿ ವೇರಿಯಬಲ್ ಅನ್ನು ಸೇರಿಸಲು ಇದು ಅನುಮತಿಸುತ್ತದೆ.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $args | `array` | ರೇಂಡರ್ ಕૉಲ್ ಮೂಲಕ ಕಳಿಸಿದ ವೇರಿಯಬಲ್‌ಗಳನ್ನು ಹೊಂದಿರುವ ಅರೇ. |
| $view | `string` | ರೇಂಡರ್ ಮಾಡಬೇಕಾದ ವೀව්‌ನ ಹೆಸರು. |
| $default_view | `string` | ಫಾಲ್ಬ್ಯಾಕ್ ವೀව්‌ನ ಹೆಸರು. |

### Since {#since}

- 2.0.0
