# Filter: wu_password_require_uppercase {#filter-wupasswordrequireuppercase}

ಪಾಸ್‌ವರ್ಡ್‌ಗಳಲ್ಲಿ ಅಪ್ಪರ್ಕೇಸ್ ಅಕ್ಷರಗಳನ್ನು (uppercase letters) ಕೇಳಬೇಕೇ ಇಲ್ಲವೇ ಎಂಬುದನ್ನು ಫಿಲ್ಟರ್ ಮಾಡುತ್ತದೆ.

## ಪ್ಯಾರಾಮೀಟರ್‌ಗಳು {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | ಅಪ್ಪರ್ಕೇಸ್ ಅಕ್ಷರಗಳು ಬೇಕೇ ಇಲ್ಲವೇ. ನಿಯಮಗಳನ್ನು ಜಾರಿಗೊಳಿಸಿದಾಗ ಡೀಫಾಲ್ಟ್ ಆಗಿ true ಇರುತ್ತದೆ. |
| $defender_active | `bool` | Defender Pro Strong Password ಸಕ್ರಿಯವಾಗಿದೆಯೇ ಇಲ್ಲವೇ. |
