# Filter: wu_password_require_special

ಪಾಸ್‌ವರ್ಡ್‌ಗಳಲ್ಲಿ ವಿಶೇಷ ಅಕ್ಷರಗಳನ್ನು (special characters) ಕಡ್ಡಾಯಗೊಳಿಸಬೇಕೇ ಇಲ್ಲವೇ ಎಂಬುದನ್ನು ಫಿಲ್ಟರ್ ಮಾಡುತ್ತದೆ.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | ವಿಶೇಷ ಅಕ್ಷರಗಳು ಬೇಕೇ ಇಲ್ಲವೇ. ನಿಯಮಗಳನ್ನು ಜಾರಿಗೊಳಿಸಿದಾಗ ಡೀಫಾಲ್ಟ್ ಆಗಿ true ಇರುತ್ತದೆ. |
| $defender_active | `bool` | Defender Pro Strong Password ಸಕ್ರಿಯವಾಗಿದೆಯೇ (active) ಎಂಬುದನ್ನು ಸೂಚಿಸುತ್ತದೆ. |
