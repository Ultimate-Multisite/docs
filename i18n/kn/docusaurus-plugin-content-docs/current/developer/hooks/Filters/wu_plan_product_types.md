# Filter: wu_plan_product_types {#filter-wuplanproducttypes}

ಯೋಜನೆಗಳೆಂದು ಪರಿಗಣಿಸಬೇಕಾದ ಉತ್ಪನ್ನದ ವಿಧಗಳನ್ನು (product types) ಫಿಲ್ಟರ್ ಮಾಡಲು ಇದು ಸಹಾಯ ಮಾಡುತ್ತದೆ.

ಈ ಫಿಲ್ಟರ್, validation ಮತ್ತು segregation ಪ್ರಕ್ರಿಯೆಗಳಲ್ಲಿ ಯೋಜನೆಗಳೆಂದು ಗುರುತಿಸಬೇಕಾದ ಹೆಚ್ಚುವರಿ ಉತ್ಪನ್ನದ ವಿಧಗಳನ್ನು ನೋಂದಾಯಿಸಲು addons-ಗಳಿಗೆ ಅವಕಾಶ ನೀಡುತ್ತದೆ.
