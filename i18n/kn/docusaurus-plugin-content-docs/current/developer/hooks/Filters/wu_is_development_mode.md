# ಫಿಲ್ಟರ್: wu_is_development_mode

ಪ್ಲಗ್ಇನ್ ಅಭಿವರ್ಧಕರು (plugin developers) ಅವರು ಡೆವಲಪ್‌ಮೆಂಟ್ ಮೋಡ್‌ಗಾಗಿ ಹೆಚ್ಚುವರಿ ಪರೀಕ್ಷೆಗಳನ್ನು (tests) ಸೇರಿಸಲು ಇದನ್ನು ಬಳಸಬಹುದು.

## ಪ್ಯಾರಾಮೀಟರ್‌ಗಳು (Parameters)

| ಹೆಸರು (Name) | ವಿಧ (Type) | ವಿವರಣೆ (Description) |
|------|------|-------------|
| $is_development_mode | `bool` | ಪ್ರಸ್ತುತ ಡೆವಲಪ್‌ಮೆಂಟ್ ಸ್ಥಿತಿ. |
| $site_url | `string` | ಸೈಟ್‌ನ URL. |

### ರಿಂದ (Since)

- 2.0.0
