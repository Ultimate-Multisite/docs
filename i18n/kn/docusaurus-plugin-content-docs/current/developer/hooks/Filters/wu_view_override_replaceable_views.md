# Filter: wu_view_override_replaceable_views

developers-ಗಳಿಗೆ ಬದಲಾಯಿಸಬಹುದಾದ (replaceable) ಪಟ್ಟಿಗೆ ಹೆಚ್ಚುವರಿ ಫೋಲ್ಡರ್‌ಗಳನ್ನು ಸೇರಿಸಲು ಇದು ಅನುಮತಿಸುತ್ತದೆ.

ಎಚ್ಚರ ವಹಿಸಿ, ಏಕೆಂದರೆ ಹೆಚ್ಚುವರಿ ಫೋಲ್ಡರ್‌ಗಳನ್ನು ಅನುಮತಿಸುವುದರಿಂದ, Ultimate Multisite ಆವೃತ್ತಿಗಳ ಬದಲಿಗೆ ಹಳೆಯ (out-of-date) ಪ್ರತಿಗಳು ಲೋಡ್ ಆಗುವ ಸಾಧ್ಯತೆ ಇದೆ.

## ಪ್ಯಾರಾಮೀಟರ್‌ಗಳು (Parameters) {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $replaceable_views | `array` | ಅನುಮತಿಸಲಾದ ಫೋಲ್ಡರ್‌ಗಳ ಪಟ್ಟಿ. |

### ಯಾವಾಗ ಲಭ್ಯ: {#since}

- 2.0.0
### ಮೂಲ (Source) {#source}

[inc/functions/template.php](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L50) ನಲ್ಲಿ 50ನೇ ಸಾಲಿನಲ್ಲಿ ವ್ಯಾಖ್ಯಾನಿಸಲಾಗಿದೆ.
