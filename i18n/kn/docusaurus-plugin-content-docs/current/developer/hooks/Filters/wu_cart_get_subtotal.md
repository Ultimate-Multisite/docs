# Filter: wu_cart_get_subtotal

'ಆರಂಭಿಕ ಮೊತ್ತ' (initial amount) ಒಟ್ಟು (total) ಅನ್ನು ಫಿಲ್ಟರ್ ಮಾಡಲು ಇದನ್ನು ಬಳಸಲಾಗುತ್ತದೆ.

## ಪ್ಯಾರಾಮೀಟರ್‌ಗಳು {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $subtotal | `float` | ಇಂದು ಪಾವತಿಸಬೇಕಾದ ಒಟ್ಟು ಮೊತ್ತ. |
| $this | `\WP_Ultimo\Checkout\Cart` | ಕಾರ್ಟ್ ಆಬ್ಜೆಕ್ಟ್ (Cart object). |

### ಮೂಲ {#source}

[`inc/checkout/class-cart.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-cart.php#L2112) ಫೈಲ್‌ನ 2112ನೇ ಸಾಲಿನಲ್ಲಿ ಇದನ್ನು ವ್ಯಾಖ್ಯಾನಿಸಲಾಗಿದೆ.
