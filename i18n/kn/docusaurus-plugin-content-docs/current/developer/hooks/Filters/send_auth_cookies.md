# Filter: send_auth_cookies

ಆಥ್ ಕುಕೀಗಳನ್ನು (auth cookies) ನಿಜವಾಗಿ ಕ್ಲೈಂಟ್‌ಗೆ ಕಳುಹಿಸುವುದನ್ನು ತಡೆಯಲು ಇದು ಅನುಮತಿಸುತ್ತದೆ.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $send | `bool` | ಕ್ಲೈಂಟ್‌ಗೆ ಆಥ್ ಕುಕೀಗಳನ್ನು ಕಳುಹಿಸಬೇಕೇ ಇಲ್ಲವೇ ಎಂಬುದನ್ನು ನಿರ್ಧರಿಸುತ್ತದೆ. |

### Since {#since}

- 4.7.4

- [`inc/sso/auth-functions.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/sso/auth-functions.php#L150) ನಲ್ಲಿ 150ನೇ ಸಾಲಿನಲ್ಲಿ ವ್ಯಾಖ್ಯಾನಿಸಲಾಗಿದೆ
