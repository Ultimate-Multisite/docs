# Filter: wu_documentation_get_link

ಪ್ಲಗ್ಇನ್ ಅಭಿವರ್ಧಕರು ಲಿಂಕ್‌ಗಳನ್ನು ಫಿಲ್ಟರ್ ಮಾಡಲು ಇದು ಅವಕಾಶ ನೀಡುತ್ತದೆ.

ಇದು ಎಷ್ಟು ಉಪಯುಕ್ತ ಎಂಬುದು ನನಗೆ ಖಚಿತವಿಲ್ಲ, ಆದರೆ ಇದನ್ನು ಸೇರಿಸುವುದು ಯಾವುದೇ ತೊಂದರೆಯಲ್ಲ.
