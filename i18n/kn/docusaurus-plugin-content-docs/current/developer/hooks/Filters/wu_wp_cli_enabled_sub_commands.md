# ಫಿಲ್ಟರ್: wu_wp_cli_enabled_sub_commands

ಈ ಘಟಕಕ್ಕೆ (entity) ಯಾವ ಉಪ-ಕಮಾಂಡ್‌ಗಳು (sub_commands) ಸಕ್ರಿಯಗೊಂಡಿವೆ ಎಂಬುದನ್ನು ಫಿಲ್ಟರ್ ಮಾಡುತ್ತದೆ.

## ಪ್ಯಾರಾಮೀಟರ್‌ಗಳು {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $sub_commands | `array` | ಡೀಫಾಲ್ಟ್ ಉಪ-ಕಮಾಂಡ್‌ಗಳು. |
| $command_base | `string` | ಮೂಲದ (root) ನಂತರದ ಕಮಾಂಡ್‌ಗಳಲ್ಲಿ ಬಳಸುವ ಬೇಸ್. |
| $this | `\Base_Manager` | ಆಬ್ಜೆಕ್ಟ್‌ನ ಉದಾಹರಣೆ (object instance). |

### ಯಾವಾಗ ಲಭ್ಯ {#since}

- 2.0.0
