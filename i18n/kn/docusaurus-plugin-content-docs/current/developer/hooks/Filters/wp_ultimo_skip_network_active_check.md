# ಫಿಲ್ಟರ್: wp_ultimo_skip_network_active_check

ಈ ಪರಿಶೀಲನೆಯನ್ನು ದಾಟಲು (short-circuit) 개발ರ್‌ಗಳಿಗೆ ಅವಕಾಶ ನೀಡುತ್ತದೆ.

ನೀವು composer ಆಧಾರಿತ ಮತ್ತು ಇತರ ಕಸ್ಟಮ್ ಸೆಟಪ್‌ಗಳನ್ನು (ಉದಾಹರಣೆಗೆ Bedrock) ಬಳಸುತ್ತಿರುವಾಗ ಇದು ಉಪಯುಕ್ತವಾಗಿದೆ. ಏಕೆಂದರೆ ಆ ರೀತಿಯ ಸೆಟಪ್‌ಗಳಲ್ಲಿ mu-plugins ಅನ್ನು ಪ್ಲಗ್ಇನ್‌ಗಳಂತೆ ಬಳಸುವುದು ಸಾಮಾನ್ಯವಾಗಿದೆ.

### Since

- 2.0.0
### Source

[`inc/class-requirements.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-requirements.php#L256) ನಲ್ಲಿ 256ನೇ ಸಾಲಿನಲ್ಲಿ ವ್ಯಾಖ್ಯಾನಿಸಲಾಗಿದೆ.
