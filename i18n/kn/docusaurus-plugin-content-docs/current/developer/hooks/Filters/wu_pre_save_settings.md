# Filter: wu_pre_save_settings

Ultimate Multisite ನಿಂದ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಉಳಿಸುವ ಮೊದಲು (before save) ಅವುಗಳನ್ನು ಫಿಲ್ಟರ್ ಮಾಡಲು developers-ಗಳಿಗೆ ಇದು ಅನುಮತಿಸುತ್ತದೆ.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $settings | `array` | ಉಳಿಸಬೇಕಾದ ಸೆಟ್ಟಿಂಗ್‌ಗಳು. |
| $settings_to_save | `array` | ಸೇರಿಸಬೇಕಾದ ಹೊಸ ಸೆಟ್ಟಿಂಗ್‌ಗಳು. |
| $saved_settings | `array` | ಪ್ರಸ್ತುತ ಉಳಿಸಿದ ಸೆಟ್ಟಿಂಗ್‌ಗಳು. |

### Since {#since}

- 2.0.18
### Source {#source}

[`inc/class-settings.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-settings.php#L296) ನಲ್ಲಿ 296ನೇ ಸಾಲಿನಲ್ಲಿ ವ್ಯಾಖ್ಯಾನಿಸಲಾಗಿದೆ.
