# Action: wu_dashboard_widgets {#action-wudashboardwidgets}

ಪ್ಲಗ್ಇನ್ developers ಗಳಿಗೆ ನೆಟ್‌ವರ್ಕ್ ಡ್ಯಾಶ್‌ಬೋರ್ಡ್ ಪ್ಯಾನೆಲ್‌ಗೆ 위젯ಗಳನ್ನು ಸೇರಿಸಲು ಇದು ಅವಕಾಶ ನೀಡುತ್ತದೆ.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $tab | `string` | ಪ್ರಸ್ತುತ ಟ್ಯಾಬ್‌ನ ಹೆಸರು. |
| $screen | `\WP_Screen` | ಸ್ಕ್ರೀನ್ ಆಬ್ಜೆಕ್ಟ್. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Ultimate Multisite ಆಡ್ಮಿನ್ ಪುಟದ ಉದಾಹರಣೆ (instance). |
