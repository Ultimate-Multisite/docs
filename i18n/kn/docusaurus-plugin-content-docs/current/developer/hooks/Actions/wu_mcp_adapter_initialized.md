# Action: wu_mcp_adapter_initialized

MCP adapter ಇನಿಷಿಯಲೈಸ್ (initialized) ಆದ ನಂತರ ಇದು ಕಾರ್ಯಗತವಾಗುತ್ತದೆ.

ಇತರ ಪ್ಲಗ್ಇನ್‌ಗಳು ಮತ್ತು ಥೀಮ್‌ಗಳು ತಮ್ಮದೇ ಆದ ಸಾಮರ್ಥ್ಯಗಳನ್ನು (abilities) ನೋಂದಾಯಿಸಲು ಇದು ಅವಕಾಶ ನೀಡುತ್ತದೆ.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | MCP adapter ಇನ್‌ಸ್ಟೆನ್ಸ್ (instance). |
