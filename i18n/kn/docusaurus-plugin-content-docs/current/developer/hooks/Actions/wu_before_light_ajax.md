# Action: wu_before_light_ajax

ಕೆಲವ ಸಂದರ್ಭಗಳಲ್ಲಿ, ಆಕ್ಷನ್‌ಗಳನ್ನು (actions) ನಿರ್ವಹಿಸಲು ನಮಗೆ ಹೆಚ್ಚುವರಿ ಅವಲಂಬನೆಗಳನ್ನು (dependencies) ಲೋಡ್ ಮಾಡಬೇಕಾಗುತ್ತದೆ.

ಈ ಆಕ್ಷನ್ ಅನ್ನು ಅಗತ್ಯವಿದ್ದಾಗ ಹೆಚ್ಚುವರಿ ಅವಲಂಬನೆಗಳನ್ನು ಲೋಡ್ ಮಾಡಲು ಬಳಸಬಹುದು.
