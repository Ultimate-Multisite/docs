# Action: wu_register_rest_routes {#action-wuregisterrestroutes}

ಹೆಚ್ಚುವರಿ routes ಅನ್ನು ನೋಂದಾಯಿಸಲು (register) ಇದು ಅನುಮತಿಸುತ್ತದೆ.

ಇದು ನಮ್ಮ /register ಎಂಡ್‌ಪಾಯಿಂಟ್ (endpoint) ಮೂಲಕ ಬಳಸಲ್ಪಡುತ್ತದೆ.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $api_instance | `self` | ಪ್ರಸ್ತುತ API ಇನ್‌ಸ್ಟೆನ್ಸ್ (instance). |
