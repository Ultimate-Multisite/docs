# Action: wu_domain_mapping_load

ನಮ್ಮ ಮುಖ್ಯ ಡೊಮೇನ್ ಮ್ಯಾಪಿಂಗ್ (Domain Mapping) ಲೋಡ್ ಆದ ನಂತರ ಇದು ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ.

ಯಾವುದೇ ಅಡ್-ಆನ್ ಕಾರ್ಯನಿರ್ವಹಣೆಯನ್ನು (add-on functionality) ನಿರ್ವಹಿಸಲು ನೀವು ಇದರಲ್ಲಿ ಹುಕ್ (hook) ಮಾಡಬಹುದು.

### ಮೂಲ (Source)

ಇದು [`inc/class-domain-mapping.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-domain-mapping.php#L175) ಫೈಲ್‌ನ 175ನೇ ಸಾಲಿನಲ್ಲಿ ವ್ಯಾಖ್ಯಾನಿಸಲಾಗಿದೆ.
