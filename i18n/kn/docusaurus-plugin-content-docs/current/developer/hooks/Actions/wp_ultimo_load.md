# Action: wp_ultimo_load

ಎಲ್ಲಾ ಅವಲಂಬನೆಗಳು (dependencies) ಲೋಡ್ ಆದಾಗ ಇದು ಕಾರ್ಯಗತಗೊಳ್ಳುತ್ತದೆ.

ಪ್ಲಗ್ಇನ್ ಡೆವಲಪರ್‌ಗಳಿಗೆ ಹೊಸ ಕಾರ್ಯಕ್ಷಮತೆಯನ್ನು (functionality) ಸೇರಿಸಲು ಇದು ಅನುಮತಿಸುತ್ತದೆ. ಉದಾಹರಣೆಗೆ, ಹೊಸ ಹೋಸ್ಟಿಂಗ್ ಪроваೈಡರ್‌ಗಳಿಗೆ ಬೆಂಬಲ ನೀಡುವುದು ಇತ್ಯಾದಿ.

- 2.0.0
