# Action: wu_register_integrations {#action-wuregisterintegrations}

ಇಂಟಿಗ್ರೇಷನ್‌ಗಳು ತಮ್ಮನ್ನು ತಾವು ನೋಂದಾಯಿಸಿಕೊಳ್ಳಲು (register) ಇದು ಕಾರ್ಯಗತವಾಗುತ್ತದೆ.

ಕೋರ್ (Core) ತನ್ನ ಇಂಟಿಗ್ರೇಷನ್‌ಗಳನ್ನು ಇಲ್ಲಿ ನೋಂದಾಯಿಸುತ್ತದೆ.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $registry | `\Integration_Registry` | ರಿಜಿಸ್ಟ್ರಿ ಇನ್‌ಸ್ಟೆನ್ಸ್ (The registry instance). |

- 2.5.0
