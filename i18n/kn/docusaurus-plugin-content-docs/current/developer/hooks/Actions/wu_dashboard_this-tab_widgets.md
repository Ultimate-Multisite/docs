# Action: wu_dashboard_\{$this->tab\}_widgets

ಪ್ಲಗ್ಇನ್ ಅಭಿವರ್ಧಕರು ನೆಟ್‌ವರ್ಕ್ ಡ್ಯಾಶ್‌ಬೋರ್ಡ್ ಪ್ಯಾನೆಲ್‌ಗೆ 위젯ಗಳನ್ನು ಸೇರಿಸಲು ಇದು ಅನುಮತಿಸುತ್ತದೆ.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $tab | `string` | ಪ್ರಸ್ತುತ ಟ್ಯಾಬ್ (The current tab). |
| $screen | `\WP_Screen` | ಸ್ಕ್ರೀನ್ ಆಬ್ಜೆಕ್ಟ್ (The screen object). |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Ultimate Multisite ಆಡ್ಮಿನ್ ಪುಟದ ಉದಾಹರಣೆ (Ultimate Multisite admin page instance). |
