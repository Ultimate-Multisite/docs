# ಇಮೇಲ್‌ಗಳು ಮತ್ತು ಪ್ರಸಾರಗಳನ್ನು ಕಳುಹಿಸುವುದು (v2)

_**ಮುಖ್ಯ ಸೂಚನೆ: ಈ ಲೇಖನವು Ultimate Multisite ಆವೃತ್ತಿ 2.x ಅನ್ನು ಉಲ್ಲೇಖಿಸುತ್ತದೆ.**_

Ultimate Multisite ನಲ್ಲಿ ಒಂದು ವೈಶಿಷ್ಟ್ಯವಿದೆ, ಇದು ನಿರ್ದಿಷ್ಟ ಬಳಕೆದಾರರಿಗೆ ಅಥವಾ ಬಳಕೆದಾರರ ಗುಂಪಿಗೆ ಇಮೇಲ್ ಕಳುಹಿಸುವ ಮೂಲಕ ನಿಮ್ಮ ಗ್ರಾಹಕರೊಂದಿಗೆ ಸಂವಹನ ನಡೆಸಲು ಅನುವು ಮಾಡಿಕೊಡುತ್ತದೆ. ಜೊತೆಗೆ, ಪ್ರಕಟಣೆಗಳನ್ನು ಪ್ರಸಾರ ಮಾಡಲು ಅವರ admin dashboard ನಲ್ಲಿ ಸೂಚನೆಗಳನ್ನು ಕಳುಹಿಸಬಹುದು.

## ಪ್ರಸಾರಗಳೊಂದಿಗೆ ನಿಮ್ಮ ಗ್ರಾಹಕರ dashboard ನಲ್ಲಿ admin ಸೂಚನೆಗಳನ್ನು ಸೇರಿಸಿ

Ultimate Multisite ಪ್ರಸಾರ ವೈಶಿಷ್ಟ್ಯವನ್ನು ಬಳಸಿ, ನಿಮ್ಮ ಬಳಕೆದಾರರ subsite admin dashboard ನಲ್ಲಿ **admin ಸೂಚನೆಗಳನ್ನು** ಸೇರಿಸಬಹುದು.

ಸಿಸ್ಟಮ್ ನಿರ್ವಹಣೆ ಬಗ್ಗೆ ಪ್ರಕಟಣೆ ಮಾಡಬೇಕಾದರೆ ಅಥವಾ ನಿಮ್ಮ ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಬಳಕೆದಾರರಿಗೆ ಹೊಸ ಉತ್ಪನ್ನಗಳು ಅಥವಾ ಸೇವೆಗಳನ್ನು ನೀಡಬೇಕಾದರೆ ಇದು ತುಂಬಾ ಉಪಯುಕ್ತವಾಗಿದೆ. ನಿಮ್ಮ ಬಳಕೆದಾರರ dashboard ನಲ್ಲಿ admin ಸೂಚನೆ ಹೀಗೆ ಕಾಣಿಸುತ್ತದೆ.

![ಗ್ರಾಹಕರ dashboard ನಲ್ಲಿ ತೋರಿಸಲಾದ Admin ಸೂಚನೆ ಪ್ರಸಾರ](/img/admin/broadcasts-list.png)

admin ಸೂಚನೆಯನ್ನು ಪ್ರಾರಂಭಿಸಲು, ನಿಮ್ಮ network admin dashboard ಗೆ ಹೋಗಿ ಮತ್ತು **Ultimate Multisite** ಮೆನು ಅಡಿಯಲ್ಲಿ **Broadcasts** ಆಯ್ಕೆಯನ್ನು ಕಾಣಬಹುದು.

![Ultimate Multisite admin ನಲ್ಲಿ Broadcasts ಮೆನು](/img/admin/broadcasts-list.png)

ಈ ಪುಟದಿಂದ, ಮೇಲ್ಭಾಗದಲ್ಲಿರುವ **Add Broadcast** ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.

ಇದು Add broadcast modal ವಿಂಡೋವನ್ನು ತೆರೆಯುತ್ತದೆ, ಅಲ್ಲಿ ನೀವು ಯಾವ ರೀತಿಯ ಪ್ರಸಾರವನ್ನು ಕಳುಹಿಸಲು ಬಯಸುತ್ತೀರಿ ಎಂದು ಆಯ್ಕೆ ಮಾಡಬಹುದು.

**Message** ಆಯ್ಕೆ ಮಾಡಿ ನಂತರ **Next Step** ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.

![Message ಪ್ರಕಾರ ಆಯ್ಕೆಯಾದ Add broadcast modal](/img/admin/broadcasts-list.png)

ಮುಂದಿನ ವಿಂಡೋ ನಿಮ್ಮನ್ನು **Target customer** ಅಥವಾ **Target product** ಬಗ್ಗೆ ಕೇಳುತ್ತದೆ. ನೀವು ಒಂದಕ್ಕಿಂತ ಹೆಚ್ಚು ಬಳಕೆದಾರರನ್ನು ಅಥವಾ ಒಂದಕ್ಕಿಂತ ಹೆಚ್ಚು ಉತ್ಪನ್ನಗಳನ್ನು ಆಯ್ಕೆ ಮಾಡಬಹುದು ಎಂಬುದನ್ನು ಗಮನಿಸಿ.

ಬಳಕೆದಾರ ಖಾತೆ ಅಥವಾ ಉತ್ಪನ್ನವನ್ನು ಹುಡುಕಲು, ಫೀಲ್ಡ್‌ನಲ್ಲಿ keyword ಟೈಪ್ ಮಾಡಲು ಪ್ರಾರಂಭಿಸಬೇಕು.

**Message type** ಫೀಲ್ಡ್ ಅಡಿಯಲ್ಲಿ, ಸೂಚನೆಯ ಬಣ್ಣವನ್ನು ಆಯ್ಕೆ ಮಾಡಬಹುದು. ಇದು ನಿಮ್ಮ ಸಂದೇಶದ ತುರ್ತುತನವನ್ನು ಒತ್ತಿಹೇಳುತ್ತದೆ.

ನಂತರ **Next Step** ಕ್ಲಿಕ್ ಮಾಡಬಹುದು.

![ಪ್ರಸಾರಕ್ಕಾಗಿ Target customer ಮತ್ತು product ಆಯ್ಕೆ](/img/admin/broadcasts-list.png)

ಮುಂದಿನ ವಿಂಡೋದಲ್ಲಿ ನೀವು ವಿಷಯ ಮತ್ತು ಬಳಕೆದಾರರಿಗೆ ಪ್ರಸಾರ ಮಾಡಲು ಬಯಸುವ ವಿಷಯ/ಸಂದೇಶವನ್ನು ನಮೂದಿಸುವ ಮೂಲಕ ನಿಮ್ಮ ಸಂದೇಶವನ್ನು ರಚಿಸಲು ಪ್ರಾರಂಭಿಸಬಹುದು.

![ಪ್ರಸಾರ ಸಂದೇಶದ ವಿಷಯ ಮತ್ತು ವಿಷಯ editor](/img/admin/broadcasts-list.png)

ನಿಮ್ಮ ಸಂದೇಶವನ್ನು ರಚಿಸಿದ ನಂತರ, **Send** ಬಟನ್ ಒತ್ತಬಹುದು.

ಅಷ್ಟೇ. admin ಸೂಚನೆಯು ತಕ್ಷಣವೇ ನಿಮ್ಮ ಬಳಕೆದಾರರ dashboard ನಲ್ಲಿ ಕಾಣಿಸುತ್ತದೆ.

## ನಿಮ್ಮ ಗ್ರಾಹಕರಿಗೆ ಇಮೇಲ್‌ಗಳನ್ನು ಕಳುಹಿಸಿ

Ultimate Multisite ಪ್ರಸಾರ ವೈಶಿಷ್ಟ್ಯವನ್ನು ಬಳಸಿ, ನಿಮ್ಮ ಬಳಕೆದಾರರಿಗೆ ಇಮೇಲ್ ಕಳುಹಿಸಬಹುದು. ಇಮೇಲ್ ಅನ್ನು ನಿರ್ದಿಷ್ಟ ಬಳಕೆದಾರರಿಗೆ ಮಾತ್ರ ಕಳುಹಿಸುವ ಅಥವಾ ಅವರು ಚಂದಾದಾರರಾಗಿರುವ ಉತ್ಪನ್ನ ಅಥವಾ ಯೋಜನೆಯ ಆಧಾರದ ಮೇಲೆ ನಿರ್ದಿಷ್ಟ ಬಳಕೆದಾರ ಗುಂಪನ್ನು ಗುರಿಯಾಗಿಸುವ ಆಯ್ಕೆ ನಿಮಗಿದೆ.

ಇಮೇಲ್ ಪ್ರಸಾರವನ್ನು ಪ್ರಾರಂಭಿಸಲು, ನಿಮ್ಮ network admin dashboard ಗೆ ಹೋಗಿ ಮತ್ತು Ultimate Multisite ಮೆನು ಅಡಿಯಲ್ಲಿ Broadcast ಆಯ್ಕೆಯನ್ನು ಕಾಣಬಹುದು.

![Ultimate Multisite admin ನಲ್ಲಿ Broadcasts ಪುಟ](/img/admin/broadcasts-list.png)

ಈ ಪುಟದಿಂದ, ಮೇಲ್ಭಾಗದಲ್ಲಿರುವ **Add broadcast** ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.

ಇದು Add broadcast modal ವಿಂಡೋವನ್ನು ತೆರೆಯುತ್ತದೆ, ಅಲ್ಲಿ ನೀವು ಯಾವ ರೀತಿಯ ಪ್ರಸಾರವನ್ನು ಕಳುಹಿಸಲು ಬಯಸುತ್ತೀರಿ ಎಂದು ಆಯ್ಕೆ ಮಾಡಬಹುದು. **Email** ಆಯ್ಕೆ ಮಾಡಿ ನಂತರ **Next Step** ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.

![Email ಪ್ರಕಾರ ಆಯ್ಕೆಯಾದ Add broadcast modal](/img/admin/broadcasts-list.png)

ಮುಂದಿನ ವಿಂಡೋ ನಿಮ್ಮನ್ನು **Target customer** ಅಥವಾ **Target product** ಬಗ್ಗೆ ಕೇಳುತ್ತದೆ. ನೀವು ಒಂದಕ್ಕಿಂತ ಹೆಚ್ಚು ಬಳಕೆದಾರರನ್ನು ಅಥವಾ ಒಂದಕ್ಕಿಂತ ಹೆಚ್ಚು ಉತ್ಪನ್ನಗಳನ್ನು ಆಯ್ಕೆ ಮಾಡಬಹುದು ಎಂಬುದನ್ನು ಗಮನಿಸಿ.

ಬಳಕೆದಾರ ಖಾತೆ ಅಥವಾ ಉತ್ಪನ್ನವನ್ನು ಹುಡುಕಲು, ಫೀಲ್ಡ್‌ನಲ್ಲಿ keyword ಟೈಪ್ ಮಾಡಲು ಪ್ರಾರಂಭಿಸಬೇಕು.

ನಿಮ್ಮ ಗುರಿ ಪ್ರೇಕ್ಷಕರನ್ನು ಆಯ್ಕೆ ಮಾಡಿದ ನಂತರ, **Next Step** ಕ್ಲಿಕ್ ಮಾಡಬಹುದು.

![ಇಮೇಲ್ ಪ್ರಸಾರಕ್ಕಾಗಿ Target customer ಮತ್ತು product ಆಯ್ಕೆ](/img/admin/broadcasts-list.png)

ಮುಂದಿನ ವಿಂಡೋದಲ್ಲಿ ನೀವು ವಿಷಯ ಮತ್ತು ಬಳಕೆದಾರರಿಗೆ ಕಳುಹಿಸಲು ಬಯಸುವ ವಿಷಯ/ಸಂದೇಶವನ್ನು ನಮೂದಿಸುವ ಮೂಲಕ ನಿಮ್ಮ ಇಮೇಲ್ ರಚಿಸಲು ಪ್ರಾರಂಭಿಸಬಹುದು.

![ಇಮೇಲ್ ಪ್ರಸಾರದ ವಿಷಯ ಮತ್ತು ವಿಷಯ editor](/img/admin/broadcasts-list.png)

ನಿಮ್ಮ ಸಂದೇಶವನ್ನು ರಚಿಸಿದ ನಂತರ, **Send** ಬಟನ್ ಒತ್ತಬಹುದು.

ಪ್ರಸಾರ ವೈಶಿಷ್ಟ್ಯವನ್ನು ಬಳಸಿ ನಿಮ್ಮ ಅಂತಿಮ ಬಳಕೆದಾರರಿಗೆ ಇಮೇಲ್ ಕಳುಹಿಸುವುದು ಎಷ್ಟು ಸುಲಭ ಎಂದು ನೋಡಿ.

## ಸಿಸ್ಟಮ್ ಇಮೇಲ್‌ಗಳು

Ultimate Multisite ನಲ್ಲಿ ಸಿಸ್ಟಮ್ ಇಮೇಲ್‌ಗಳೆಂದರೆ ನೋಂದಣಿ, ಪಾವತಿ, domain mapping ಮುಂತಾದ ನಿರ್ದಿಷ್ಟ ಕ್ರಿಯೆಗಳ ನಂತರ ಸಿಸ್ಟಮ್‌ನಿಂದ ಕಳುಹಿಸಲಾಗುವ **ಸ್ವಯಂಚಾಲಿತ ಅಧಿಸೂಚನೆಗಳು**. ಈ ಇಮೇಲ್‌ಗಳನ್ನು Ultimate Multisite ಸೆಟ್ಟಿಂಗ್‌ಗಳಿಂದ ಸಂಪಾದಿಸಬಹುದು ಅಥವಾ ಮಾರ್ಪಡಿಸಬಹುದು. ಇದು ಮತ್ತೊಂದು Ultimate Multisite ಸ್ಥಾಪನೆಯಿಂದ ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಮರುಹೊಂದಿಸಲು ಮತ್ತು ಆಮದು ಮಾಡಲು ಅನುವು ಮಾಡಿಕೊಡುವ ವೈಶಿಷ್ಟ್ಯದೊಂದಿಗೆ ಬರುತ್ತದೆ.

### ಮರುಹೊಂದಿಸುವಿಕೆ ಮತ್ತು ಆಮದು

ಹೊಸ Ultimate Multisite ಆವೃತ್ತಿಗಳು, ಹಾಗೆಯೇ add-on ಗಳು, ಕಾಲಕಾಲಕ್ಕೆ ಹೊಸ ಇಮೇಲ್‌ಗಳನ್ನು ನೋಂದಾಯಿಸಬಹುದು ಮತ್ತು ನೋಂದಾಯಿಸುತ್ತವೆ.

ಸಂಘರ್ಷಗಳು ಮತ್ತು ಇತರ ಸಮಸ್ಯೆಗಳನ್ನು ತಡೆಗಟ್ಟಲು, ನಿರ್ದಿಷ್ಟ ವೈಶಿಷ್ಟ್ಯದ ಸರಿಯಾದ ಕಾರ್ಯನಿರ್ವಹಣೆಗೆ ಅವು ನಿರ್ಣಾಯಕವಾಗಿದ್ದರೆ ಮಾತ್ರ, **ನಿಮ್ಮ ಸ್ಥಾಪನೆಯಲ್ಲಿ ಹೊಸ ಇಮೇಲ್ ಟೆಂಪ್ಲೇಟ್‌ಗಳನ್ನು ಸಿಸ್ಟಮ್ ಇಮೇಲ್‌ಗಳಾಗಿ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸೇರಿಸುವುದಿಲ್ಲ**.

ಆದಾಗ್ಯೂ, super admin ಗಳು ಮತ್ತು ಏಜೆಂಟ್‌ಗಳು importer ಟೂಲ್ ಮೂಲಕ ಈ ಹೊಸದಾಗಿ ನೋಂದಾಯಿತ ಇಮೇಲ್‌ಗಳನ್ನು ಆಮದು ಮಾಡಬಹುದು. ಆ ಪ್ರಕ್ರಿಯೆಯು ಹೊಸ ಇಮೇಲ್ ಟೆಂಪ್ಲೇಟ್‌ನ ವಿಷಯ ಮತ್ತು ಸಂರಚನೆಯೊಂದಿಗೆ ಹೊಸ ಸಿಸ್ಟಮ್ ಇಮೇಲ್ ಅನ್ನು ರಚಿಸುತ್ತದೆ, super admin ಗೆ ಅವರು ಬಯಸುವ ಯಾವುದೇ ಮಾರ್ಪಾಡುಗಳನ್ನು ಮಾಡಲು ಅಥವಾ ಅವುಗಳನ್ನು ಹಾಗೆಯೇ ಇಟ್ಟುಕೊಳ್ಳಲು ಅನುವು ಮಾಡಿಕೊಡುತ್ತದೆ.

#### ಸಿಸ್ಟಮ್ ಇಮೇಲ್‌ಗಳನ್ನು ಆಮದು ಮಾಡುವುದು ಹೇಗೆ

ನಿಮ್ಮ Ultimate Multisite Settings ಪುಟಕ್ಕೆ ಹೋಗಿ ಮತ್ತು **Emails** ಟ್ಯಾಬ್‌ಗೆ ಹೋಗಿ.

![Ultimate Multisite ಸೆಟ್ಟಿಂಗ್‌ಗಳಲ್ಲಿ Emails ಟ್ಯಾಬ್](/img/config/settings-emails.png)

ನಂತರ, sidebar ನಲ್ಲಿ **Customize System Emails** ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ.

![Sidebar ನಲ್ಲಿ Customize System Emails ಬಟನ್](/img/config/settings-emails.png)

System Emails ಪುಟದಲ್ಲಿ, ಮೇಲ್ಭಾಗದಲ್ಲಿ **Reset & Import** ಆ್ಯಕ್ಷನ್ ಬಟನ್ ಕಾಣಿಸುತ್ತದೆ. ಆ ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿದರೆ import ಮತ್ತು reset modal ವಿಂಡೋ ತೆರೆಯುತ್ತದೆ.

![System Emails ಪುಟದಲ್ಲಿ Reset and Import ಆ್ಯಕ್ಷನ್ ಬಟನ್](/img/config/settings-emails.png)

ನಂತರ, ಯಾವ ಸಿಸ್ಟಮ್ ಇಮೇಲ್‌ಗಳನ್ನು ಆಮದು ಮಾಡಬಹುದು ಎಂದು ನೋಡಲು Import Emails ಆಯ್ಕೆಗಳನ್ನು toggle ಮಾಡಬಹುದು.

![ಲಭ್ಯವಿರುವ ಸಿಸ್ಟಮ್ ಇಮೇಲ್‌ಗಳನ್ನು ತೋರಿಸುವ Import Emails ಆಯ್ಕೆಗಳು](/img/config/settings-emails.png)

#### ಸಿಸ್ಟಮ್ ಇಮೇಲ್‌ಗಳನ್ನು ಮರುಹೊಂದಿಸುವುದು

ಕೆಲವೊಮ್ಮೆ, ನಿರ್ದಿಷ್ಟ ಇಮೇಲ್ ಟೆಂಪ್ಲೇಟ್‌ಗೆ ಮಾಡಿದ ಬದಲಾವಣೆಗಳು ನಿಮಗೆ ಇನ್ನು ಮುಂದೆ ಸೂಕ್ತವಾಗುತ್ತಿಲ್ಲ ಎಂದು ಅರಿವಾಗುತ್ತದೆ ಮತ್ತು ಅದನ್ನು ಅವುಗಳ **ಡೀಫಾಲ್ಟ್ ಸ್ಥಿತಿಗೆ** ಮರುಹೊಂದಿಸಲು ಬಯಸುತ್ತೀರಿ.

ಅಂತಹ ಸಂದರ್ಭಗಳಲ್ಲಿ, ನಿಮಗೆ ಎರಡು ಆಯ್ಕೆಗಳಿವೆ: ನೀವು ಸಿಸ್ಟಮ್ ಇಮೇಲ್ ಅನ್ನು ಸರಳವಾಗಿ ಅಳಿಸಬಹುದು ಮತ್ತು ಅದನ್ನು ಮತ್ತೆ ಆಮದು ಮಾಡಬಹುದು (ಮೇಲಿನ ಸೂಚನೆಗಳನ್ನು ಬಳಸಿ) - ಇದು send metrics ಮತ್ತು ಇತರ ವಿಷಯಗಳನ್ನು ಅಳಿಸುತ್ತದೆ, ಇದು ಈ ವಿಧಾನವನ್ನು ಕಡಿಮೆ ಆದ್ಯತೆಯನ್ನಾಗಿ ಮಾಡುತ್ತದೆ.

ಅಥವಾ ಆ ಇಮೇಲ್ ಟೆಂಪ್ಲೇಟ್ ಅನ್ನು ಮರುಹೊಂದಿಸಲು **Reset & Import tool** ಅನ್ನು ಬಳಸಬಹುದು.

ಇಮೇಲ್ ಟೆಂಪ್ಲೇಟ್ ಅನ್ನು ಮರುಹೊಂದಿಸಲು, Reset & Import tool ತಲುಪುವವರೆಗೆ ಮೇಲಿನ ಹಂತಗಳನ್ನು ಅನುಸರಿಸಬಹುದು, ನಂತರ **Reset** ಆಯ್ಕೆಯನ್ನು toggle ಮಾಡಿ ಮತ್ತು ಅವುಗಳ ಡೀಫಾಲ್ಟ್ ವಿಷಯಕ್ಕೆ ಮರುಹೊಂದಿಸಲು ಬಯಸುವ ಇಮೇಲ್‌ಗಳನ್ನು ಆಯ್ಕೆ ಮಾಡಿ.
