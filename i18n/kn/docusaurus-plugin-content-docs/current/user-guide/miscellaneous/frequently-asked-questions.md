# ಪದೇ ಪದೇ ಕೇಳಲಾಗುವ ಪ್ರಶ್ನೆಗಳು

ಬೆಂಬಲ ವಿನಂತಿಗಳ ಸಮಯದಲ್ಲಿ ಸಾಮಾನ್ಯವಾಗಿ ಕೇಳಲಾಗುವ ಕೆಲವು **ಪದೇ ಪದೇ ಕೇಳಲಾಗುವ ಪ್ರಶ್ನೆಗಳು** ಇಲ್ಲಿವೆ.

  * **Ultimate Multisite ಎಂದರೇನು?**

Ultimate Multisite ಒಂದು WordPress plugin ಆಗಿದ್ದು, WordPress multisite ಇನ್‌ಸ್ಟಾಲ್‌ಗಳಿಗಾಗಿ ರಚಿಸಲಾಗಿದೆ. ಇದು ನಿಮ್ಮ WordPress ಇನ್‌ಸ್ಟಾಲ್ ಅನ್ನು [WordPress.com](https://WordPress.com) ನಂತಹ ಪ್ರೀಮಿಯಂ ಸೈಟ್‌ಗಳ ನೆಟ್‌ವರ್ಕ್ ಆಗಿ ಪರಿವರ್ತಿಸುತ್ತದೆ – ಗ್ರಾಹಕರು ಮಾಸಿಕ, ತ್ರೈಮಾಸಿಕ, ಅಥವಾ ವಾರ್ಷಿಕ ಶುಲ್ಕಗಳ ಮೂಲಕ ಸೈಟ್‌ಗಳನ್ನು ರಚಿಸಲು ಅನುಮತಿಸುತ್ತದೆ (ನೀವು ಉಚಿತ ಯೋಜನೆಗಳನ್ನೂ ರಚಿಸಬಹುದು).

  * **Multisite ಇಲ್ಲದೆ Ultimate Multisite ಕೆಲಸ ಮಾಡುತ್ತದೆಯೇ?**

ಇಲ್ಲ. Ultimate Multisite ಗೆ WordPress multisite, subdirectory, ಅಥವಾ subdomain ಅಗತ್ಯವಿದೆ.

  * **Ultimate Multisite ಗೆ ಏನು ಅಗತ್ಯವಿದೆ?**

Ultimate Multisite ಗೆ ಅಗತ್ಯವಿರುವುದು:

**WordPress ಆವೃತ್ತಿ** : v5.3+ (ಶಿಫಾರಸು: ಇತ್ತೀಚಿನ ಸ್ಥಿರ ಆವೃತ್ತಿ)  
**PHP ಆವೃತ್ತಿ** : 7.4.x (> 8.0 ಬೆಂಬಲ ಶೀಘ್ರದಲ್ಲೇ ಬರುತ್ತಿದೆ)  
**MySQL ಆವೃತ್ತಿ** : v5+ (ಶಿಫಾರಸು: ಆವೃತ್ತಿ 5.6, ನಿಮ್ಮ hosting provider ನಲ್ಲಿ 8.0 ಲಭ್ಯವಿಲ್ಲದಿದ್ದರೆ)

_Multisite (subdomain ಅಥವಾ subdirectory) ಸಹ ಸಕ್ರಿಯಗೊಳಿಸಬೇಕು._ [WordPress Multisite ಅನ್ನು ಹೇಗೆ ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡುವುದು](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **Subdirectories ಇರುವ Multisite ನೊಂದಿಗೆ Ultimate Multisite ಕೆಲಸ ಮಾಡುತ್ತದೆಯೇ?**

ಹೌದು. Ultimate Multisite subdirectory ಇನ್‌ಸ್ಟಾಲ್‌ಗಳು ಮತ್ತು subdirectory ಇನ್‌ಸ್ಟಾಲ್‌ಗಳೆರಡನ್ನೂ ಬೆಂಬಲಿಸುತ್ತದೆ.

  * **Subdomains ಇರುವ Multisite ನೊಂದಿಗೆ Ultimate Multisite ಕೆಲಸ ಮಾಡುತ್ತದೆಯೇ?**

ಹೌದು. Ultimate Multisite subdirectory ಇನ್‌ಸ್ಟಾಲ್‌ಗಳು ಮತ್ತು subdomain ಇನ್‌ಸ್ಟಾಲ್‌ಗಳೆರಡನ್ನೂ ಬೆಂಬಲಿಸುತ್ತದೆ.

  * **ಬಳಕೆದಾರರು ಯಾವ Plugins ಮತ್ತು Themes ಬಳಸಬಹುದು ಎಂಬುದನ್ನು ಸೀಮಿತಗೊಳಿಸಲು Ultimate Multisite ಬಳಸಬಹುದೇ?**

ಹೌದು. Ultimate Multisite ಮೂಲಕ ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿ ಪ್ರತಿ ಯೋಜನೆಗೆ ಯಾವ plugins ಮತ್ತು themes ಲಭ್ಯವಿರಬೇಕು ಎಂದು ಆಯ್ಕೆ ಮಾಡಬಹುದು.

  * **ಬಳಕೆದಾರರು ತಮ್ಮ ಸೈಟ್‌ನಲ್ಲಿ ರಚಿಸಬಹುದಾದ ಪೋಸ್ಟ್‌ಗಳ ಸಂಖ್ಯೆಯನ್ನು ಸೀಮಿತಗೊಳಿಸಲು Ultimate Multisite ಬಳಸಬಹುದೇ?**

ಹೌದು. Ultimate Multisite ಮೂಲಕ ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿ **ಲಭ್ಯವಿರುವ ಪ್ರತಿ post type** ಗೆ ಪೋಸ್ಟ್ ಕೋಟಾಗಳನ್ನು ಸ್ಥಾಪಿಸಬಹುದು. admin ಆಹ್ವಾನಿಸಬಹುದಾದ ಬಳಕೆದಾರರ ಸಂಖ್ಯೆ, **ಡಿಸ್ಕ್ ಸ್ಪೇಸ್ ಮೊತ್ತ,** ಮತ್ತು ಇನ್ನೂ ಹೆಚ್ಚಿನದನ್ನು ಸೀಮಿತಗೊಳಿಸಬಹುದು.

  * **Ultimate Multisite Site Templates ಅನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆಯೇ?**

ಹೌದು. Ultimate Multisite ಮೂಲಕ ನಿಮಗೆ ಬೇಕಾದಷ್ಟು Site Templates ರಚಿಸಬಹುದು. Site Templates ಎಂದರೆ themes ಅಲ್ಲ, ಅವು ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿರುವ ಸಾಮಾನ್ಯ ಸೈಟ್‌ಗಳು. ನೀವು plugins, themes ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಬಹುದು ಮತ್ತು ಆ Site Template ಅನ್ನು ನಿಮಗೆ ಬೇಕಾದ ರೀತಿಯಲ್ಲಿ ಕಸ್ಟಮೈಸ್ ಮಾಡಬಹುದು. ಬಳಕೆದಾರರು ಆ template ಅನ್ನು ಆಯ್ಕೆ ಮಾಡಿದರೆ, ಅವರ ಸೈಟ್ ಎಲ್ಲಾ ಕಸ್ಟಮೈಸೇಶನ್‌ಗಳೊಂದಿಗೆ ಆ template ನ ನಕಲಾಗಿ ರಚಿಸಲಾಗುತ್ತದೆ.

ನೀವು ಒಂದು ನಿರ್ದಿಷ್ಟ ಯೋಜನೆಗೆ ನಿರ್ದಿಷ್ಟ template ಅನ್ನು ಸಹ ಹೊಂದಿಸಬಹುದು. ಹಾಗೆ ಮಾಡುವುದರಿಂದ, ಆ ಯೋಜನೆಯಲ್ಲಿ ಸೈನ್ ಅಪ್ ಮಾಡುವ ಪ್ರತಿ ಬಳಕೆದಾರರ ಸೈಟ್ ಆ Site Template ನ ಡುಪ್ಲಿಕೇಟ್ ಆಗಿ ರಚಿಸಲಾಗುತ್ತದೆ.

  * **Ultimate Multisite WooCommerce ನೊಂದಿಗೆ ಸಂಯೋಜಿಸುತ್ತದೆಯೇ?**

ಹೌದು!

ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್ ಬಳಕೆದಾರರಿಗೆ WooCommerce ಲಭ್ಯವಾಗಿರಬೇಕೆಂದು ನೀವು ಬಯಸಿದರೆ, ಅವರು ಸ್ಟೋರ್‌ಗಳನ್ನು ಸ್ಥಾಪಿಸಲು ಅನುಮತಿಸುವುದು Ultimate Multisite ಬಳಸಿ ಸಾಧ್ಯ. ಪ್ರತಿ ಯೋಜನೆಯ ಹಂತದಲ್ಲಿ ನಿಮ್ಮ ಗ್ರಾಹಕರು ರಚಿಸಬಹುದಾದ ಉತ್ಪನ್ನಗಳ ಸಂಖ್ಯೆಯನ್ನು ಸಹ ಸೀಮಿತಗೊಳಿಸಬಹುದು.

Ultimate Multisite ಗಾಗಿ ಚಂದಾ ಪಾವತಿಗಳ ವಿಷಯದಲ್ಲಿ, ನಿಮ್ಮ ಮುಖ್ಯ ಸೈಟ್‌ನಲ್ಲಿ WooCommerce gateways ಬಳಸಿ ನಿಮ್ಮ Ultimate Multisite ಗ್ರಾಹಕರು ಪಾವತಿಸಲು ಅನುಮತಿಸಲು [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) ಬಳಸಬಹುದು. ಆ ರೀತಿಯಲ್ಲಿ, WooCommerce ಗಾಗಿ ಅಭಿವೃದ್ಧಿಪಡಿಸಲಾದ ಪಾವತಿ gateways ನ ದೊಡ್ಡ ಲೈಬ್ರರಿಯನ್ನು ನೀವು ಬಳಸಿಕೊಂಡು ನಿಮ್ಮ Ultimate Multisite ಗ್ರಾಹಕರಿಗೆ ಬಿಲ್ ಮಾಡಬಹುದು.

  * **WPMUDEV Pro Sites ನಿಂದ ಚಂದಾಗಳನ್ನು ಮೈಗ್ರೇಟ್ ಮಾಡಲು ಮಾರ್ಗವಿದೆಯೇ?**

ಹೌದು, Ultimate Multisite ಈಗ ನಿಮ್ಮ Pro Sites ನೆಟ್‌ವರ್ಕ್ ಅನ್ನು ಮೈಗ್ರೇಟ್ ಮಾಡಲು [ಸಹಾಯಕ ಟೂಲ್](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) ಒದಗಿಸುತ್ತದೆ.

  * **ನನ್ನ Plan Edit Page ನಲ್ಲಿ "Allowed Plugins" ಟ್ಯಾಬ್ ಖಾಲಿಯಾಗಿದೆ ಏಕೆ?**

Ultimate Multisite ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿ network-activated ಆಗಿಲ್ಲದ plugins ಗೆ ಮಾತ್ರ ಪ್ರವೇಶವನ್ನು ಅನುಮತಿಸಲು ಅಥವಾ ನಿರಾಕರಿಸಲು ಅವಕಾಶ ನೀಡುತ್ತದೆ. Plugin ಒಂದನ್ನು network-activated ಮಾಡಿದಾಗ, ಅದರ ಸ್ವಭಾವದಿಂದಲೇ, ಅದು ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನ ಪ್ರತಿ ಸೈಟ್‌ನಲ್ಲಿ ಡೀಫಾಲ್ಟ್ ಆಗಿ ಲೋಡ್ ಆಗುತ್ತದೆ.

"Allowed Plugins" ಟ್ಯಾಬ್‌ನಲ್ಲಿ ಯಾವುದೇ plugins ಪಟ್ಟಿ ಮಾಡಲಾಗಿಲ್ಲವೆಂದು ಕಂಡರೆ, ಬೇಕಾದ plugin network-active ಆಗಿದೆಯೇ ಇಲ್ಲವೇ ಎಂದು ಪರಿಶೀಲಿಸಿ.

  * **ನನ್ನ ಸ್ವಂತ Pricing Table ವಿನ್ಯಾಸವನ್ನು ಬಳಸಬಹುದೇ?**

ಹೌದು. ಆವೃತ್ತಿ 1.4.0 ರಿಂದ, Ultimate Multisite template overrides ಅನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ.

  * **Ultimate Multisite WP Engine ನೊಂದಿಗೆ ಕೆಲಸ ಮಾಡುತ್ತದೆಯೇ?**

ಹೌದು. Ultimate Multisite ಯಾವಾಗಲೂ WP Engine ನಿಂದ ಬೆಂಬಲಿತವಾಗಿತ್ತು, ಆದರೆ domain mapping ಕೆಲಸ ಮಾಡಲು admin ನಿಂದ ಹಸ್ತಚಾಲಿತ ಹೆಚ್ಚುವರಿ ಹಂತಗಳು ಅಗತ್ಯವಿದ್ದವು (WP Engine panel ಗೆ domain ಸೇರಿಸುವುದು). **ಆವೃತ್ತಿ 1.6.0 ರಿಂದ ಇದು ಇನ್ನು ಮುಂದೆ ಅಗತ್ಯವಿಲ್ಲ.** Ultimate Multisite ನೇರವಾಗಿ WP Engine API ಯೊಂದಿಗೆ ಸಂಯೋಜಿಸುತ್ತದೆ ಮತ್ತು mapping ನಂತರ domains ಸ್ವಯಂಚಾಲಿತವಾಗಿ panel ಗೆ ಸೇರಿಸಲಾಗುತ್ತದೆ.

  * **ಯಾವ hosting provider ಅನ್ನು ಶಿಫಾರಸು ಮಾಡುತ್ತೀರಿ?**

Ultimate Multisite ಯಾವುದೇ hosting provider ನೊಂದಿಗೆ ಕೆಲಸ ಮಾಡಬೇಕು. ಒಂದೇ ವ್ಯತ್ಯಾಸವೆಂದರೆ Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane ಮತ್ತು [Runcloud.io](http://Runcloud.io) ನಂತಹ ಕೆಲವು hosting ಕಂಪನಿಗಳು domain mapping ಸಾಮರ್ಥ್ಯಗಳಿಗಾಗಿ ಹೆಚ್ಚು ನಿಕಟ ಸಂಯೋಜನೆಯನ್ನು ಹೊಂದಿವೆ, ಆದ್ದರಿಂದ ಅವು ತಕ್ಷಣವೇ ಕೆಲಸ ಮಾಡುತ್ತವೆ. ಇತರ hostings ಗಾಗಿ, ನೀವು domain mapping ಅನ್ನು ಹಸ್ತಚಾಲಿತವಾಗಿ ಹೊಂದಿಸಬೇಕಾಗಬಹುದು.
