# Site Builder Mode ತೆಗೆದುಹಾಕುವ ಸೂಚನೆ {#site-builder-mode-removal-notice}

**Superdav AI Agent v1.12.0 ರಲ್ಲಿ Site Builder mode ಅನ್ನು ತೆಗೆದುಹಾಕಲಾಗಿದೆ.** ನೀವು Site Builder mode ಬಳಸುತ್ತಿದ್ದರೆ, ಥೀಮ್ ರಚನೆ ಮತ್ತು ಸೈಟ್ ಸೆಟಪ್‌ಗಾಗಿ **Setup Assistant agent** ಗೆ ಬದಲಾಯಿಸಿಕೊಳ್ಳಬೇಕು.

## ಏನಾಯಿತು? {#what-happened}

### Site Builder Mode (ಹಳೆಯ ವಿಧಾನ) {#site-builder-mode-legacy}

Site Builder mode ಎಂಬುದು ಈ ಕೆಳಗಿನ ಕಾರ್ಯಗಳಿಗಾಗಿ ವೀಜಾರ್ಡ್ ಆಧಾರಿತ ಇಂಟರ್ಫೇಸ್ ಆಗಿತ್ತು:

- ಟೆಂಪ್ಲೇಟ್‌ಗಳಿಂದ ಸೈಟ್‌ಗಳನ್ನು ರಚಿಸುವುದು
- ಮೂಲ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡುವುದು
- ಒಂದು ಥೀಮ್ ಅನ್ನು ಆಯ್ಕೆ ಮಾಡುವುದು
- ಆರಂಭಿಕ ವಿಷಯವನ್ನು ಸೆಟಪ್ ಮಾಡುವುದು

### ಇದರ ಬದಲಿಗೆ ಏನು ಬಂದಿದೆ? {#what-replaced-it}

**Setup Assistant agent** ಈಗ ಎಲ್ಲಾ Site Builder ಕಾರ್ಯಕ್ಷಮತೆಗಳನ್ನು ಈ ಕೆಳಗಿನ ಅಂಶಗಳೊಂದಿಗೆ ನಿರ್ವಹಿಸುತ್ತದೆ:

- ಹೆಚ್ಚು ಹೊಂದಿಕೊಳ್ಳುವ, ಏಜೆಂಟ್-ನಡೆಸುವ ಸೆಟಪ್
- ಉತ್ತಮ ಥೀಮ್ ಕಸ್ಟಮೈಸೇಶನ್ ಆಯ್ಕೆಗಳು
- ಥೀಮ್ ಬಿಲ್ಡರ್ ಆನ್‌ಬೋರ್ಡಿಂಗ್‌ನೊಂದಿಗೆ ಏಕೀಕರಣ
- ಭವಿಷ್ಯದ ಸೆಷನ್‌ಗಳಿಗಾಗಿ ನಿರಂತರ site_brief ಮೆಮೊರಿ

## ನೀವು Site Builder Mode ಬಳಸುತ್ತಿದ್ದರೆ {#if-you-were-using-site-builder-mode}

### ನಿಮ್ಮ ಸೈಟ್‌ಗಳು ಸುರಕ್ಷಿತವಾಗಿವೆ {#your-sites-are-safe}

- Site Builder mode ಬಳಸಿ ರಚಿಸಿದ ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಸೈಟ್‌ಗಳು ಕೆಲಸ ಮಾಡುವುದನ್ನು ಮುಂದುವರಿಸುತ್ತವೆ
- ಯಾವುದೇ ಡೇಟಾ ನಷ್ಟ ಅಥವಾ ಸೈಟ್ ಅಡಚಣೆ ಇಲ್ಲ
- ನೀವು ನಿಮ್ಮ ಸೈಟ್‌ಗಳನ್ನು normal ರೀತಿಯಲ್ಲಿ ನಿರ್ವಹಿಸಲು ಮುಂದುವರಿಯಬಹುದು

### Setup Assistant Agent ಗೆ ಬದಲಾಯಿಸಿಕೊಳ್ಳುವುದು {#migrate-to-setup-assistant-agent}

ಹೊಸ ಸೈಟ್ ಸೆಟಪ್ ಅಥವಾ ಥೀಮ್ ಬದಲಾವಣೆಗಳಿಗಾಗಿ, Setup Assistant agent ಅನ್ನು ಬಳಸಿ:

```
"Help me set up a new site"
```

ಅಥವಾ

```
"Start the Theme Builder onboarding"
```

Setup Assistant agent ಹೆಚ್ಚು ಹೊಂದಿಕೊಳ್ಳುವಿಕೆಯೊಂದಿಗೆ ಅದೇ ಕಾರ್ಯಕ್ಷಮತೆಯನ್ನು ಒದಗಿಸುತ್ತದೆ.

## ಹೋಲಿಕೆ: Site Builder vs. Setup Assistant {#comparison-site-builder-vs-setup-assistant}

| Feature | Site Builder (ತೆಗೆದುಹಾಕಲಾಗಿದೆ) | Setup Assistant (ಹೊಸದು) |
|---------|----------------------|----------------------|
| ಸೆಟಪ್ ವಿಧಾನ | ವೀಜಾರ್ಡ್ ಫಾರ್ಮ್ | ಏಜೆಂಟ್ ಸಂಭಾಷಣೆ |
| ಥೀಮ್ ಆಯ್ಕೆ | ಪೂರ್ವ-ನಿರ್ définie ಟೆಂಪ್ಲೇಟ್‌ಗಳು | ಕಸ್ಟಮ್ ರಚನೆ |
| ಕಸ್ಟಮೈಸೇಶನ್ | ಸೀಮಿತ ಆಯ್ಕೆಗಳು | ಸಂಪೂರ್ಣ ಡಿಸೈನ್ ಸಿಸ್ಟಮ್ |
| ಸೈಟ್ ಬ್ರೀಫ್ | ಸಂಗ್ರಹಿಸುವುದಿಲ್ಲ | ನಿರಂತರ ಮೆಮೊರಿ |
| ಭವಿಷ್ಯದ ಸೆಷನ್‌ಗಳು | ಸೆಟಪ್ ಪುನರಾವರ್ತನೆ | ಸಂಗ್ರಹಿಸಿದ site_brief ಬಳಸುವುದು |
| ಹೊಂದಿಕೊಳ್ಳುವಿಕೆ | ಸ್ಥಿರ ಕಾರ್ಯಪ್ರವಾಹ | ಹೊಂದಿಕೊಳ್ಳುವ ಸಂಭಾಷಣೆ |

## Setup Assistant Agent ಗೆ ಬದಲಾಯಿಸಿಕೊಳ್ಳುವುದು {#migrating-to-setup-assistant-agent}

### ಹೊಸ ಸೈಟ್‌ಗಳಿಗಾಗಿ {#for-new-sites}

Site Builder mode ಬಳಸುವ ಬದಲು:

1. ವಿನಂತಿಸಿ: "Help me set up a new site"
2. Setup Assistant agent ನಿಮಗೆ ಈ ಕೆಳಗಿನ ಮೂಲಕ ಮಾರ್ಗದರ್ಶನ ಮಾಡುತ್ತದೆ:
   - ಸೈಟ್ ಉದ್ದೇಶ ಮತ್ತು ಗುರಿಗಳು
   - ಗುರಿ ಪ್ರೇಕ್ಷಕರು (Target audience)
   - ಬ್ರ್ಯಾಂಡ್ ಗುರುತು (Brand identity)
   - ಥೀಮ್ ರಚನೆ
   - ಆರಂಭಿಕ ಕಾನ್ಫಿಗರೇಶನ್

### ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಸೈಟ್‌ಗಳಿಗಾಗಿ {#for-existing-sites}

Site Builder mode ನಿಂದ ನಿಮಗೆ ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಸೈಟ್ವಿದ್ದರೆ:

1. ನೀವು ಅದನ್ನು ಹಾಗೆಯೇ ಬಳಸುವುದನ್ನು ಮುಂದುವರಿಸಬಹುದು
2. ಥೀಮ್ ಅನ್ನು ಅಪ್ಡೇಟ್ ಮಾಡಲು, ವಿನಂತಿಸಿ: "Redesign my site"
3. Setup Assistant agent ನಿಮಗೆ ಹೊಸ ಥೀಮ್ ರಚಿಸಲು ಸಹಾಯ ಮಾಡುತ್ತದೆ
4. ನಿಮ್ಮ ಸೈಟ್ ಡೇಟಾ ಬದಲಾಗದೆ ಉಳಿಯುತ್ತದೆ

### ಥೀಮ್ ಬದಲಾವಣೆಗಳಿಗಾಗಿ {#for-theme-changes}

Site Builder mode ನ ಥೀಮ್ ಆಯ್ಕೆಯ ಬದಲಿಗೆ:

1. ವಿನಂತಿಸಿ: "Change my theme"
2. Setup Assistant agent ಈ ಕೆಳಗಿನವುಗಳನ್ನು ಮಾಡುತ್ತದೆ:
   - ನಿಮ್ಮ ಡಿಸೈನ್ ಆದ್ಯತೆಗಳ ಬಗ್ಗೆ ಕೇಳುತ್ತದೆ
   - ಕಸ್ಟಮ್ ಥೀಮ್ ಅನ್ನು ರಚಿಸುತ್ತದೆ
   - ಅದನ್ನು ನಿಮ್ಮ ಸೈಟ್‌ನಲ್ಲಿ ಸಕ್ರಿಯಗೊಳಿಸುತ್ತದೆ

## ಪ್ರಮುಖ ವ್ಯತ್ಯಾಸಗಳು {#key-differences}

### Site Builder Mode {#site-builder-mode}

```
1. ಟೆಂಪ್ಲೇಟ್ ಆರಿಸಿ
2. ಥೀಮ್ ಆಯ್ಕೆಮಾಡಿ
3. ಮೂಲ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಿ
4. ಮುಗಿದಿದೆ
```

### Setup Assistant Agent {#setup-assistant-agent}

```
1. ನಿಮ್ಮ ಸೈಟ್‌ನ ಉದ್ದೇಶವನ್ನು ವಿವರಿಸಿ
2. ನಿಮ್ಮ ಗುರಿ ಪ್ರೇಕ್ಷಕರನ್ನು ವ್ಯಾಖ್ಯಾನಿಸಿ
3. ಡಿಸೈನ್ ಆದ್ಯತೆಗಳನ್ನು ಆರಿಸಿ
4. ಏಜೆಂಟ್ ಕಸ್ಟಮ್ ಥೀಮ್ ರಚಿಸುತ್ತದೆ
5. ಏಜೆಂಟ್ ಥೀಮ್ ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸುತ್ತದೆ
6. ಭವಿಷ್ಯದ ಸೆಷನ್‌ಗಳಿಗಾಗಿ ಸೈಟ್ ಬ್ರೀಫ್ ಸಂಗ್ರಹ කෙරේ
```

## Setup Assistant Agent ನ ಪ್ರಯೋಜನಗಳು {#benefits-of-setup-assistant-agent}

### ಹೆಚ್ಚು ಹೊಂದಿಕೊಳ್ಳುವಿಕೆ {#more-flexible}

- ನಿಮ್ಮ ಸೈಟ್ ಅನ್ನು ನೈಸರ್ಗಿಕ ಭಾಷೆಯಲ್ಲಿ ವಿವರಿಸಿ
- ಕಸ್ಟಮ್ ಶಿಫಾರಸುಗಳನ್ನು ಪಡೆಯಿರಿ
- ನಿಮ್ಮ ನಿರ್ದಿಷ್ಟ ಅಗತ್ಯಗಳಿಗೆ ಹೊಂದಿಕೊಳ್ಳುತ್ತದೆ

### ಉತ್ತಮ ಕಸ್ಟಮೈಸೇಶನ್ {#better-customization}

- ಕಸ್ಟಮ್ ಥೀಮ್ ರಚನೆ
- ಡಿಸೈನ್ ಸಿಸ್ಟಮ್ ನಿರ್ಧಾರಗಳು
- ನಿರಂತರ ಡಿಸೈನ್ ಟೋಕನ್‌ಗಳು

### ನಿರಂತರ ಮೆಮೊರಿ {#persistent-memory}

- ನಿಮ್ಮ site_brief ಅನ್ನು ಸಂಗ್ರಹಿಸಲಾಗುತ್ತದೆ
- ಭವಿಷ್ಯದ ಏಜೆಂಟ್‌ಗಳು ನಿಮ್ಮ ಸೈಟ್ ಅನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳುತ್ತವೆ
- ಸೆಟಪ್ ಮಾಹಿತಿಯನ್ನು ಪುನರಾವರ್ತಿಸುವ ಅಗತ್ಯವಿಲ್ಲ

### ஒருங்கிணைந்த ಕಾರ್ಯಪ್ರವಾಹ {#integrated-workflow}

- ಥೀಮ್ ಬಿಲ್ಡರ್ ಆನ್‌ಬೋರ್ಡಿಂಗ್
- ಡಿಸೈನ್ ಸಿಸ್ಟಮ್ ಎಸ್ಟೆಟಿಕ್ಸ್ ಸ್ಕಿಲ್
- ಲಭ್ಯತೆ ಗೋಚರತೆ ನಿಯಂತ್ರಣಗಳು (Ability Visibility controls)
- ಎಲ್ಲವೂ ಒಟ್ಟಾಗಿ ಸುಗಮವಾಗಿ ಕೆಲಸ ಮಾಡುತ್ತವೆ

## ಸಮಸ್ಯೆ ಪರಿಹಾರ (Troubleshooting) {#troubleshooting}

### ನನಗೆ Site Builder mode ಸಿಗುತ್ತಿಲ್ಲ {#i-cant-find-site-builder-mode}

Site Builder mode ಅನ್ನು ತೆಗೆದುಹಾಕಲಾಗಿದೆ. ಬದಲಾಗಿ Setup Assistant agent ಅನ್ನು ಬಳಸಿ:

```
"Help me set up a new site"
```

### Site Builder ನಿಂದ ಸೈಟ್ ಅನ್ನು ಮರುಸೃಷ್ಟಿಸಲು ಬಯಸುತ್ತೇನೆ {#i-want-to-recreate-a-site-from-site-builder}

ನೀವು ಅದನ್ನು Setup Assistant agent ಬಳಸಿ ಮರುಸೃಷ್ಟಿಸಬಹುದು:

1. ವಿನಂತಿಸಿ: "Help me set up a new site"
2. ನಿಮ್ಮ ಮೂಲ ಸೈಟ್‌ನ ಉದ್ದೇಶ ಮತ್ತು ವಿನ್ಯಾಸವನ್ನು ವಿವರಿಸಿ
3. ಏಜೆಂಟ್ ಇದೇ ರೀತಿಯ ಥೀಮ್ ಅನ್ನು ರಚಿಸುತ್ತದೆ
4. ನಿಮ್ಮ site_brief ಭವಿಷ್ಯದ ಉಲ್ಲೇಖಕ್ಕಾಗಿ ಸಂಗ್ರಹಿಸಲಾಗುತ್ತದೆ

### ನನ್ನ ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ Site Builder ಸೈಟ್ ಕೆಲಸ ಮಾಡುತ್ತಿಲ್ಲ {#my-existing-site-builder-site-isnt-working}

Site Builder mode ಬಳಸಿ ರಚಿಸಿದ ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಸೈಟ್‌ಗಳು ಕೆಲಸ ಮಾಡುವುದನ್ನು ಮುಂದುವರಿಸುತ್ತವೆ. ನೀವು ಸಮಸ್ಯೆಗಳನ್ನು ಎದುರಿಸುತ್ತಿದ್ದರೆ:

1. ನಿಮ್ಮ ಥೀಮ್ ಇನ್ನೂ ಸಕ್ರಿಯವಾಗಿದೆ ಎಂದು ಪರಿಶೀಲಿಸಿ
2. ನಿಮ್ಮ plugins ಸಕ್ರಿಯಗೊಂಡಿವೆಯೇ ಎಂದು ಪರಿಶೀಲಿಸಿ
3. WordPressのエラー ಲಾಗ್‌ಗಳನ್ನು ಪರಿಶೀಲಿಸಿ
4. ಸಮಸ್ಯೆಗಳು ಮುಂದುವರಿದರೆ サಪೋರ್ಟ್ ಅನ್ನು ಸಂಪರ್ಕಿಸಿ

### ನಾನು ನನ್ನ ಹಳೆಯ Site Builder ಟೆಂಪ್ಲೇಟ್‌ಗಳನ್ನು ಬಳಸಬಹುದೇ? {#can-i-still-use-my-old-site-builder-templates}

Site Builder ಟೆಂಪ್ಲೇಟ್‌ಗಳು ಲಭ್ಯವಿಲ್ಲ. ಆದಾಗ್ಯೂ:

- ನಿಮ್ಮ ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಸೈಟ್‌ಗಳು ಕೆಲಸ ಮಾಡುವುದನ್ನು ಮುಂದುವರಿಸುತ್ತವೆ
- ನೀವು Setup Assistant agent ಬಳಸಿ ಇದೇ ರೀತಿಯ ಸೈಟ್‌ಗಳನ್ನು ಮರುಸೃಷ್ಟಿಸಬಹುದು
- Setup Assistant agent ಹೆಚ್ಚು ಕಸ್ಟಮೈಸೇಶನ್ ಆಯ್ಕೆಗಳನ್ನು ಒದಗಿಸುತ್ತದೆ

## ಮುಂದಿನ ಹಂತಗಳು {#next-steps}

1. **ಹೊಸ ಸೈಟ್‌ಗಳಿಗಾಗಿ**: Setup Assistant agent ಅನ್ನು ಬಳಸಿ
2. **ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಸೈಟ್‌ಗಳಿಗಾಗಿ**: ಹಾಗೆಯೇ ಬಳಸುವುದನ್ನು ಮುಂದುವರಿಸಿ
3. **ಥೀಮ್ ಬದಲಾವಣೆಗಳಿಗಾಗಿ**: Setup Assistant agent ನಿಂದ ಸಹಾಯ ಕೇಳಿ
4. **ವಿನ್ಯಾಸವನ್ನು ಪರಿಷ್ಕರಿಸಲು**: Design System Aesthetics skill ಅನ್ನು ಬಳಸಿ
