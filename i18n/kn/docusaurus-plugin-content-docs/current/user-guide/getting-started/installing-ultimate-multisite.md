# Ultimate Multisite ಅನ್ನು ಇನ್ಸ್ಟಾಲ್ ಮಾಡುವುದು

:::note
ಈ ಟ್ಯುಟೋರಿಯಲ್ ನೀವು ಈಗಾಗಲೇ WordPress Multisite ಅನ್ನು ಇನ್ಸ್ಟಾಲ್ ಮಾಡಿ ಕಾನ್ಫಿಗರ್ ಮಾಡಿದ್ದೀರಿ ಎಂದು ಊಹಿಸುತ್ತದೆ. ಹೇಗೆ ಮಾಡಬೇಕೆಂದು ತಿಳಿಯಲು, WP Beginner ನ [ಈ ಟ್ಯುಟೋರಿಯಲ್](https://www.wpbeginner.com/glossary/multisite/) ನೋಡಿ.
:::

## Plugin ಅನ್ನು ಇನ್ಸ್ಟಾಲ್ ಮಾಡುವುದು

Ultimate Multisite [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) ನಲ್ಲಿ ಉಚಿತವಾಗಿ ಲಭ್ಯವಿದೆ.

ನಿಮ್ಮ **Network Admin Dashboard** ನಿಂದ, **Plugins → Add New Plugin** ಗೆ ಹೋಗಿ.

![Network Admin Add New Plugin ಪುಟ](/img/installation/add-new-plugin.png)

**"Ultimate Multisite"** (ನಿಖರವಾದ ಫಲಿತಾಂಶಕ್ಕಾಗಿ ಉದ್ಧರಣ ಚಿಹ್ನೆಗಳೊಂದಿಗೆ) ಎಂದು ಹುಡುಕಿ, ಅದು ಮೊದಲ ಫಲಿತಾಂಶವಾಗಿ ಕಾಣಿಸುತ್ತದೆ. **Install Now** ಕ್ಲಿಕ್ ಮಾಡಿ.

![Ultimate Multisite ತೋರಿಸುವ ಹುಡುಕಾಟ ಫಲಿತಾಂಶಗಳು](/img/installation/search-ultimate-multisite.png)

ಇನ್ಸ್ಟಾಲ್ ಆದ ನಂತರ, ನಿಮ್ಮ ಸಂಪೂರ್ಣ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿ plugin ಅನ್ನು ಆಕ್ಟಿವೇಟ್ ಮಾಡಲು **Network Activate** ಕ್ಲಿಕ್ ಮಾಡಿ.

![Plugin ಇನ್ಸ್ಟಾಲ್ ಆಗಿದೆ Network Activate ಬಟನ್ ಜೊತೆಗೆ](/img/installation/plugin-installed.png)

ಆಕ್ಟಿವೇಟ್ ಆದ ನಂತರ, ನೀವು ಸ್ವಯಂಚಾಲಿತವಾಗಿ Setup Wizard ಗೆ ರೀಡೈರೆಕ್ಟ್ ಆಗುತ್ತೀರಿ.

![Plugin ಆಕ್ಟಿವೇಟ್ ಆಗಿ wizard ಗೆ ರೀಡೈರೆಕ್ಟ್ ಆಯಿತು](/img/installation/plugin-activated.png)

## Setup Wizard

Setup Wizard ಸುಮಾರು 10 ನಿಮಿಷಗಳಲ್ಲಿ Ultimate Multisite ಅನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಲು ನಿಮಗೆ ಮಾರ್ಗದರ್ಶನ ಮಾಡುತ್ತದೆ.

### ಸ್ವಾಗತ

ಪ್ರಾರಂಭಿಸಲು **Get Started** ಕ್ಲಿಕ್ ಮಾಡಿ.

![Setup Wizard ಸ್ವಾಗತ ಪರದೆ](/img/installation/wizard-welcome.png)

### ಪೂರ್ವ-ಇನ್ಸ್ಟಾಲ್ ಪರಿಶೀಲನೆಗಳು

ಈ ಹಂತವು ನಿಮ್ಮ ಸಿಸ್ಟಮ್ ಮಾಹಿತಿ ಮತ್ತು WordPress ಇನ್ಸ್ಟಾಲೇಶನ್ ಅನ್ನು ಪರಿಶೀಲಿಸಿ Ultimate Multisite ನ ಅವಶ್ಯಕತೆಗಳನ್ನು ಪೂರೈಸುತ್ತದೆಯೇ ಎಂದು ಖಚಿತಪಡಿಸುತ್ತದೆ. ಎಲ್ಲವೂ ಸರಿಯಾಗಿದ್ದರೆ, **Go to the Next Step** ಕ್ಲಿಕ್ ಮಾಡಿ.

![ಸಿಸ್ಟಮ್ ಅವಶ್ಯಕತೆಗಳನ್ನು ತೋರಿಸುವ ಪೂರ್ವ-ಇನ್ಸ್ಟಾಲ್ ಪರಿಶೀಲನೆಗಳು](/img/installation/wizard-pre-install-checks.png)

### ಇನ್ಸ್ಟಾಲೇಶನ್

ಇನ್ಸ್ಟಾಲರ್ ಅಗತ್ಯವಾದ database tables ಮತ್ತು Ultimate Multisite ಕಾರ್ಯನಿರ್ವಹಿಸಲು ಬೇಕಾದ `sunrise.php` ಫೈಲ್ ಅನ್ನು ರಚಿಸುತ್ತದೆ. ಮುಂದುವರಿಯಲು **Install** ಕ್ಲಿಕ್ ಮಾಡಿ.

![database tables ಮತ್ತು sunrise.php ತೋರಿಸುವ ಇನ್ಸ್ಟಾಲೇಶನ್ ಹಂತ](/img/installation/wizard-installation.png)

### ನಿಮ್ಮ ಕಂಪನಿ

ನಿಮ್ಮ ಕಂಪನಿ ಮಾಹಿತಿಯನ್ನು ಭರ್ತಿ ಮಾಡಿ ಮತ್ತು ನಿಮ್ಮ ಡೀಫಾಲ್ಟ್ ಕರೆನ್ಸಿಯನ್ನು ಸೆಟ್ ಮಾಡಿ. ಈ ಮಾಹಿತಿಯನ್ನು ನಿಮ್ಮ WaaS ಪ್ಲಾಟ್‌ಫಾರ್ಮ್‌ನಾದ್ಯಂತ ಬಳಸಲಾಗುತ್ತದೆ. ಮುಗಿದ ನಂತರ **Continue** ಕ್ಲಿಕ್ ಮಾಡಿ.

![ನಿಮ್ಮ ಕಂಪನಿ ಕಾನ್ಫಿಗರೇಶನ್ ಹಂತ](/img/installation/wizard-your-company.png)

### ಡೀಫಾಲ್ಟ್ ಕಂಟೆಂಟ್

ಈ ಹಂತವು ಪೂರ್ವನಿರ್ಧರಿತ templates, products, ಮತ್ತು ಇತರ ಸ್ಟಾರ್ಟರ್ ಕಂಟೆಂಟ್ ಅನ್ನು ಇನ್ಸ್ಟಾಲ್ ಮಾಡಲು ಅನುಮತಿಸುತ್ತದೆ. Ultimate Multisite ನ ವೈಶಿಷ್ಟ್ಯಗಳನ್ನು ಪರಿಚಯಿಸಿಕೊಳ್ಳಲು ಇದು ಉತ್ತಮ ಮಾರ್ಗ. ಡೀಫಾಲ್ಟ್ ಕಂಟೆಂಟ್ ಸೇರಿಸಲು **Install** ಕ್ಲಿಕ್ ಮಾಡಿ, ಅಥವಾ ಮೊದಲಿನಿಂದ ಪ್ರಾರಂಭಿಸಲು ಬಯಸಿದರೆ ಈ ಹಂತವನ್ನು ಸ್ಕಿಪ್ ಮಾಡಿ.

![ಡೀಫಾಲ್ಟ್ ಕಂಟೆಂಟ್ ಇನ್ಸ್ಟಾಲೇಶನ್ ಹಂತ](/img/installation/wizard-default-content.png)

### ಶಿಫಾರಸು ಮಾಡಿದ Plugins

ಐಚ್ಛಿಕವಾಗಿ ಶಿಫಾರಸು ಮಾಡಿದ ಸಹಾಯಕ plugins ಅನ್ನು ಇನ್ಸ್ಟಾಲ್ ಮಾಡಿ. ಅವುಗಳನ್ನು ಸೇರಿಸಲು **Install** ಕ್ಲಿಕ್ ಮಾಡಿ ಅಥವಾ ಮುಂದುವರಿಯಲು ಸ್ಕಿಪ್ ಮಾಡಿ.

![ಶಿಫಾರಸು ಮಾಡಿದ plugins ಹಂತ](/img/installation/wizard-recommended-plugins.png)

### ಸಿದ್ಧ!

ಅಷ್ಟೇ! ನಿಮ್ಮ Ultimate Multisite ಇನ್ಸ್ಟಾಲೇಶನ್ ಪೂರ್ಣವಾಗಿದೆ. ನೀವು ಈಗ **Network Admin Dashboard** ನಿಂದ ನಿಮ್ಮ Website as a Service ಪ್ಲಾಟ್‌ಫಾರ್ಮ್ ಅನ್ನು ನಿರ್ಮಿಸಲು ಪ್ರಾರಂಭಿಸಬಹುದು.

![ಸೆಟಪ್ ಪೂರ್ಣ - ಸಿದ್ಧ ಪರದೆ](/img/installation/wizard-ready.png)

![Ultimate Multisite ಸಕ್ರಿಯವಾಗಿರುವ Network Admin Dashboard](/img/installation/network-dashboard.png)

ಮುಂದುವರಿಯಿರಿ ಮತ್ತು ಸಂತೋಷದಿಂದ ಬಳಸಿ!
