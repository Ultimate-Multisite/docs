# ServerPilot ಸಂಯೋಜನೆ

## ಸಾರಾಂಶ
ServerPilot ಎಂಬುದು DigitalOcean, Amazon, Google, ಅಥವಾ ಯಾವುದೇ ಇತರ ಸರ್ವರ್ ಪೂರೈಕೆದಾರರಲ್ಲಿ WordPress ಮತ್ತು ಇತರ PHP ವೆಬ್‌ಸೈಟ್‌ಗಳನ್ನು ಹೋಸ್ಟ್ ಮಾಡಲು ಬಳಸುವ ಕ್ಲೌಡ್ ಸೇವೆಯಾಗಿದೆ. ಈ ಸಂಯೋಜನೆಯು Ultimate Multisite ಮತ್ತು ServerPilot ನಡುವೆ ಸ್ವಯಂಚಾಲಿತ ಡೊಮೇನ್ ಸಿಂಕ್ ಮತ್ತು SSL ಪ್ರಮಾಣಪತ್ರ ನಿರ್ವಹಣೆಯನ್ನು ಸಕ್ರಿಯಗೊಳಿಸುತ್ತದೆ.

## ವೈಶಿಷ್ಟ್ಯಗಳು
- ಸ್ವಯಂಚಾಲಿತ ಡೊಮೇನ್ ಸಿಂಕ್
- Let's Encrypt ಮೂಲಕ SSL ಪ್ರಮಾಣಪತ್ರ ನಿರ್ವಹಣೆ
- ಸ್ವಯಂಚಾಲಿತ SSL ನವೀಕರಣ

## ಅಗತ್ಯತೆಗಳು
ಈ ಕೆಳಗಿನ constant ಗಳನ್ನು ನಿಮ್ಮ `wp-config.php` ಫೈಲ್‌ನಲ್ಲಿ ವ್ಯಾಖ್ಯಾನಿಸಬೇಕು:

```php
define('WU_SERVER_PILOT_CLIENT_ID', 'your_client_id');
define('WU_SERVER_PILOT_API_KEY', 'your_api_key');
define('WU_SERVER_PILOT_APP_ID', 'your_app_id');
```

## ಸೆಟಪ್ ಸೂಚನೆಗಳು

### 1. ನಿಮ್ಮ ServerPilot API ರುಜುವಾತುಗಳನ್ನು ಪಡೆಯಿರಿ

1. ನಿಮ್ಮ ServerPilot dashboard ಗೆ ಲಾಗ್ ಇನ್ ಮಾಡಿ
2. "Account" > "API" ಗೆ ಹೋಗಿ
3. ನಿಮ್ಮಲ್ಲಿ ಈಗಾಗಲೇ API key ಇಲ್ಲದಿದ್ದರೆ ಹೊಸದನ್ನು ರಚಿಸಿ
4. ನಿಮ್ಮ Client ID ಮತ್ತು API Key ಅನ್ನು ಕಾಪಿ ಮಾಡಿ

### 2. ನಿಮ್ಮ App ID ಪಡೆಯಿರಿ

1. ನಿಮ್ಮ ServerPilot dashboard ನಲ್ಲಿ, "Apps" ಗೆ ಹೋಗಿ
2. ನಿಮ್ಮ WordPress multisite ಹೋಸ್ಟ್ ಆಗಿರುವ app ಅನ್ನು ಆಯ್ಕೆ ಮಾಡಿ
3. App ID ಯು URL ನಲ್ಲಿ ಕಾಣಿಸುತ್ತದೆ: `https://manage.serverpilot.io/apps/{APP_ID}`

### 3. wp-config.php ಗೆ Constant ಗಳನ್ನು ಸೇರಿಸಿ

ಈ ಕೆಳಗಿನ constant ಗಳನ್ನು ನಿಮ್ಮ `wp-config.php` ಫೈಲ್‌ಗೆ ಸೇರಿಸಿ:

```php
define('WU_SERVER_PILOT_CLIENT_ID', 'your_client_id');
define('WU_SERVER_PILOT_API_KEY', 'your_api_key');
define('WU_SERVER_PILOT_APP_ID', 'your_app_id');
```

### 4. ಸಂಯೋಜನೆಯನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ

1. ನಿಮ್ಮ WordPress admin ನಲ್ಲಿ, Ultimate Multisite > Settings ಗೆ ಹೋಗಿ
2. "Domain Mapping" ಟ್ಯಾಬ್‌ಗೆ ನ್ಯಾವಿಗೇಟ್ ಮಾಡಿ
3. "Host Integrations" ಗೆ ಸ್ಕ್ರೋಲ್ ಮಾಡಿ
4. ServerPilot ಸಂಯೋಜನೆಯನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ
5. "Save Changes" ಕ್ಲಿಕ್ ಮಾಡಿ

## ಇದು ಹೇಗೆ ಕೆಲಸ ಮಾಡುತ್ತದೆ

### ಡೊಮೇನ್ ಸಿಂಕಿಂಗ್

Ultimate Multisite ನಲ್ಲಿ ಡೊಮೇನ್ ಮ್ಯಾಪ್ ಮಾಡಿದಾಗ:

1. ಸಂಯೋಜನೆಯು ServerPilot ನಿಂದ ಪ್ರಸ್ತುತ ಡೊಮೇನ್‌ಗಳ ಪಟ್ಟಿಯನ್ನು ಪಡೆಯುತ್ತದೆ
2. ಇದು ಹೊಸ ಡೊಮೇನ್ ಅನ್ನು ಪಟ್ಟಿಗೆ ಸೇರಿಸುತ್ತದೆ (ಅನ್ವಯವಾದರೆ www ಆವೃತ್ತಿಯೊಂದಿಗೆ)
3. ಇದು ನವೀಕರಿಸಿದ ಪಟ್ಟಿಯನ್ನು API ಮೂಲಕ ServerPilot ಗೆ ಕಳುಹಿಸುತ್ತದೆ
4. ServerPilot ನಿಮ್ಮ ಅಪ್ಲಿಕೇಶನ್‌ಗಾಗಿ ಡೊಮೇನ್ ಪಟ್ಟಿಯನ್ನು ನವೀಕರಿಸುತ್ತದೆ

### SSL ಪ್ರಮಾಣಪತ್ರ ನಿರ್ವಹಣೆ

ಡೊಮೇನ್‌ಗಳು ಸಿಂಕ್ ಆದ ನಂತರ:

1. ಸಂಯೋಜನೆಯು ನಿಮ್ಮ ಅಪ್ಲಿಕೇಶನ್‌ಗಾಗಿ AutoSSL ಅನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸಕ್ರಿಯಗೊಳಿಸುತ್ತದೆ
2. ServerPilot Let's Encrypt ಬಳಸಿ SSL ಪ್ರಮಾಣಪತ್ರ ವಿತರಣೆ ಮತ್ತು ಅಳವಡಿಕೆಯನ್ನು ನಿರ್ವಹಿಸುತ್ತದೆ
3. ServerPilot SSL ಪ್ರಮಾಣಪತ್ರಗಳ ಸ್ವಯಂಚಾಲಿತ ನವೀಕರಣವನ್ನೂ ನಿರ್ವಹಿಸುತ್ತದೆ

## SSL ಪ್ರಮಾಣಪತ್ರ ಪರಿಶೀಲನೆ

ServerPilot SSL ಪ್ರಮಾಣಪತ್ರಗಳನ್ನು ನೀಡಿ ಅಳವಡಿಸಲು ಸ್ವಲ್ಪ ಸಮಯ ತೆಗೆದುಕೊಳ್ಳಬಹುದಾದ್ದರಿಂದ, ಸಂಯೋಜನೆಯು SSL ಪ್ರಮಾಣಪತ್ರ ಪರಿಶೀಲನೆ ಪ್ರಯತ್ನಗಳ ಸಂಖ್ಯೆಯನ್ನು ಹೆಚ್ಚಿಸಲು ಕಾನ್ಫಿಗರ್ ಮಾಡಲಾಗಿದೆ. ಡೀಫಾಲ್ಟ್ ಆಗಿ, ಇದು 5 ಬಾರಿ ಪ್ರಯತ್ನಿಸುತ್ತದೆ, ಆದರೆ filter ಗಳನ್ನು ಬಳಸಿ ಇದನ್ನು ಸರಿಹೊಂದಿಸಬಹುದು.

## ಸಮಸ್ಯೆ ಪರಿಹಾರ

### API ಸಂಪರ್ಕ ಸಮಸ್ಯೆಗಳು
- ನಿಮ್ಮ Client ID ಮತ್ತು API Key ಸರಿಯಾಗಿದೆಯೇ ಎಂದು ಪರಿಶೀಲಿಸಿ
- ನಿಮ್ಮ App ID ಸರಿಯಾಗಿದೆಯೇ ಎಂದು ಪರಿಶೀಲಿಸಿ
- ನಿಮ್ಮ ServerPilot ಖಾತೆಗೆ ಅಗತ್ಯ ಅನುಮತಿಗಳಿವೆಯೇ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ

### SSL ಪ್ರಮಾಣಪತ್ರ ಸಮಸ್ಯೆಗಳು
- SSL ಪ್ರಮಾಣಪತ್ರಗಳನ್ನು ನೀಡುವ ಮೊದಲು ಡೊಮೇನ್‌ಗಳು ನಿಮ್ಮ ಸರ್ವರ್‌ಗೆ ಸರಿಯಾದ DNS ದಾಖಲೆಗಳನ್ನು ಹೊಂದಿರಬೇಕು ಎಂದು ServerPilot ಅಗತ್ಯಪಡಿಸುತ್ತದೆ
- SSL ಪ್ರಮಾಣಪತ್ರಗಳು ನೀಡಲ್ಪಡದಿದ್ದರೆ, ನಿಮ್ಮ ಡೊಮೇನ್‌ಗಳು ನಿಮ್ಮ ಸರ್ವರ್‌ನ IP ವಿಳಾಸಕ್ಕೆ ಸರಿಯಾಗಿ ಪಾಯಿಂಟ್ ಆಗಿವೆಯೇ ಎಂದು ಪರಿಶೀಲಿಸಿ
- ServerPilot SSL ಪ್ರಮಾಣಪತ್ರಗಳನ್ನು ನೀಡಿ ಅಳವಡಿಸಲು ಸ್ವಲ್ಪ ಸಮಯ ತೆಗೆದುಕೊಳ್ಳಬಹುದು (ಸಾಮಾನ್ಯವಾಗಿ 5-15 ನಿಮಿಷಗಳು)

### ಡೊಮೇನ್ ಸೇರಿಸಲಾಗಿಲ್ಲ
- ಯಾವುದೇ ದೋಷ ಸಂದೇಶಗಳಿಗಾಗಿ Ultimate Multisite ಲಾಗ್‌ಗಳನ್ನು ಪರಿಶೀಲಿಸಿ
- ಡೊಮೇನ್ ಈಗಾಗಲೇ ServerPilot ಗೆ ಸೇರಿಸಲಾಗಿಲ್ಲವೇ ಎಂದು ಪರಿಶೀಲಿಸಿ
- ನೀವು ಸೇರಿಸುತ್ತಿರುವ ಡೊಮೇನ್‌ಗಳ ಸಂಖ್ಯೆಯನ್ನು ನಿಮ್ಮ ServerPilot ಯೋಜನೆ ಬೆಂಬಲಿಸುತ್ತದೆಯೇ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ

### ಡೊಮೇನ್ ತೆಗೆಯುವಿಕೆ
- ಪ್ರಸ್ತುತ, ServerPilot API ಪ್ರತ್ಯೇಕ ಡೊಮೇನ್‌ಗಳನ್ನು ತೆಗೆಯುವ ಮಾರ್ಗವನ್ನು ಒದಗಿಸುವುದಿಲ್ಲ
- Ultimate Multisite ನಲ್ಲಿ ಡೊಮೇನ್ ಮ್ಯಾಪಿಂಗ್ ತೆಗೆದಾಗ, ತೆಗೆದ ಡೊಮೇನ್ ಅನ್ನು ಹೊರತುಪಡಿಸಿ ServerPilot ನಲ್ಲಿನ ಡೊಮೇನ್ ಪಟ್ಟಿಯನ್ನು ಸಂಯೋಜನೆ ನವೀಕರಿಸುತ್ತದೆ
