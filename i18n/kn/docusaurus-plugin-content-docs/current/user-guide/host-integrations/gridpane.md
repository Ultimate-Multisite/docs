# GridPane ಸಂಯೋಜನೆ

## ಅವಲೋಕನ
GridPane ಎನ್ನುವುದು ಗಂಭೀರ WordPress ವೃತ್ತಿಪರರಿಗಾಗಿ ನಿರ್ಮಿಸಲಾದ ವಿಶೇಷ WordPress ಹೋಸ್ಟಿಂಗ್ ಕಂಟ್ರೋಲ್ ಪ್ಯಾನೆಲ್ ಆಗಿದೆ. ಈ ಸಂಯೋಜನೆಯು Ultimate Multisite ಮತ್ತು GridPane ನಡುವೆ ಸ್ವಯಂಚಾಲಿತ ಡೊಮೇನ್ ಸಿಂಕಿಂಗ್ ಮತ್ತು SSL ಪ್ರಮಾಣಪತ್ರ ನಿರ್ವಹಣೆಯನ್ನು ಸಾಧ್ಯಗೊಳಿಸುತ್ತದೆ.

## ವೈಶಿಷ್ಟ್ಯಗಳು
- ಸ್ವಯಂಚಾಲಿತ ಡೊಮೇನ್ ಸಿಂಕಿಂಗ್
- SSL ಪ್ರಮಾಣಪತ್ರ ನಿರ್ವಹಣೆ
- SUNRISE constant ನ ಸ್ವಯಂಚಾಲಿತ ಕಾನ್ಫಿಗರೇಶನ್

## ಅವಶ್ಯಕತೆಗಳು
ನಿಮ್ಮ `wp-config.php` ಫೈಲ್‌ನಲ್ಲಿ ಈ ಕೆಳಗಿನ constants ಅನ್ನು ವ್ಯಾಖ್ಯಾನಿಸಬೇಕು:

```php
define('WU_GRIDPANE', true);
define('WU_GRIDPANE_API_KEY', 'your_api_key');
define('WU_GRIDPANE_SERVER_ID', 'your_server_id');
define('WU_GRIDPANE_APP_ID', 'your_app_id');
```

## ಸೆಟಪ್ ಸೂಚನೆಗಳು

### 1. ನಿಮ್ಮ GridPane API ರುಜುವಾತುಗಳನ್ನು ಪಡೆಯಿರಿ

1. ನಿಮ್ಮ GridPane dashboard ಗೆ ಲಾಗ್ ಇನ್ ಆಗಿ
2. "Settings" > "API" ಗೆ ಹೋಗಿ
3. ನಿಮ್ಮ ಬಳಿ ಈಗಾಗಲೇ API key ಇಲ್ಲದಿದ್ದರೆ ಒಂದನ್ನು ಜನರೇಟ್ ಮಾಡಿ
4. ನಿಮ್ಮ API key ಅನ್ನು ಕಾಪಿ ಮಾಡಿ

### 2. ನಿಮ್ಮ Server ಮತ್ತು Site ID ಗಳನ್ನು ಪಡೆಯಿರಿ

1. ನಿಮ್ಮ GridPane dashboard ನಲ್ಲಿ, "Servers" ಗೆ ಹೋಗಿ
2. ನಿಮ್ಮ WordPress multisite ಹೋಸ್ಟ್ ಆಗಿರುವ ಸರ್ವರ್ ಅನ್ನು ಆಯ್ಕೆಮಾಡಿ
3. Server ID ಯನ್ನು ಗಮನಿಸಿ (URL ನಲ್ಲಿ ಅಥವಾ ಸರ್ವರ್ ವಿವರಗಳ ಪುಟದಲ್ಲಿ ಕಾಣಿಸುತ್ತದೆ)
4. "Sites" ಗೆ ಹೋಗಿ ಮತ್ತು ನಿಮ್ಮ WordPress ಸೈಟ್ ಅನ್ನು ಆಯ್ಕೆಮಾಡಿ
5. Site ID ಯನ್ನು ಗಮನಿಸಿ (URL ನಲ್ಲಿ ಅಥವಾ ಸೈಟ್ ವಿವರಗಳ ಪುಟದಲ್ಲಿ ಕಾಣಿಸುತ್ತದೆ)

### 3. wp-config.php ಗೆ Constants ಸೇರಿಸಿ

ನಿಮ್ಮ `wp-config.php` ಫೈಲ್‌ಗೆ ಈ ಕೆಳಗಿನ constants ಅನ್ನು ಸೇರಿಸಿ:

```php
define('WU_GRIDPANE', true);
define('WU_GRIDPANE_API_KEY', 'your_api_key');
define('WU_GRIDPANE_SERVER_ID', 'your_server_id');
define('WU_GRIDPANE_APP_ID', 'your_site_id');
```

### 4. ಸಂಯೋಜನೆಯನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ

1. ನಿಮ್ಮ WordPress admin ನಲ್ಲಿ, Ultimate Multisite > Settings ಗೆ ಹೋಗಿ
2. "Domain Mapping" ಟ್ಯಾಬ್‌ಗೆ ನ್ಯಾವಿಗೇಟ್ ಮಾಡಿ
3. "Host Integrations" ಗೆ ಕೆಳಗೆ ಸ್ಕ್ರಾಲ್ ಮಾಡಿ
4. GridPane ಸಂಯೋಜನೆಯನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ
5. "Save Changes" ಕ್ಲಿಕ್ ಮಾಡಿ

## ಇದು ಹೇಗೆ ಕೆಲಸ ಮಾಡುತ್ತದೆ

Ultimate Multisite ನಲ್ಲಿ ಡೊಮೇನ್ ಮ್ಯಾಪ್ ಮಾಡಿದಾಗ:

1. ಸಂಯೋಜನೆಯು ನಿಮ್ಮ ಸೈಟ್‌ಗೆ ಡೊಮೇನ್ ಸೇರಿಸಲು GridPane ನ API ಗೆ ವಿನಂತಿಯನ್ನು ಕಳುಹಿಸುತ್ತದೆ
2. GridPane ಸ್ವಯಂಚಾಲಿತವಾಗಿ SSL ಪ್ರಮಾಣಪತ್ರ ಒದಗಿಸುವಿಕೆಯನ್ನು ನಿರ್ವಹಿಸುತ್ತದೆ
3. ಡೊಮೇನ್ ಮ್ಯಾಪಿಂಗ್ ತೆಗೆದುಹಾಕಿದಾಗ, ಸಂಯೋಜನೆಯು GridPane ನಿಂದ ಡೊಮೇನ್ ಅನ್ನು ತೆಗೆದುಹಾಕುತ್ತದೆ

ಡೊಮೇನ್ ಮ್ಯಾಪಿಂಗ್ ಸರಿಯಾಗಿ ಕೆಲಸ ಮಾಡಲು ಅಗತ್ಯವಿರುವ ನಿಮ್ಮ wp-config.php ಫೈಲ್‌ನಲ್ಲಿರುವ SUNRISE constant ಅನ್ನು ಸಹ ಈ ಸಂಯೋಜನೆಯು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ನಿರ್ವಹಿಸುತ್ತದೆ.

## SUNRISE Constant ನಿರ್ವಹಣೆ

GridPane ಸಂಯೋಜನೆಯ ಒಂದು ವಿಶಿಷ್ಟ ವೈಶಿಷ್ಟ್ಯವೆಂದರೆ GridPane ನ ಸ್ವಂತ ಡೊಮೇನ್ ಮ್ಯಾಪಿಂಗ್ ವ್ಯವಸ್ಥೆಯೊಂದಿಗೆ ಸಂಘರ್ಷಗಳನ್ನು ತಪ್ಪಿಸಲು wp-config.php ನಲ್ಲಿ SUNRISE constant ಅನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಹಿಂತಿರುಗಿಸುತ್ತದೆ. ಇದು ಎರಡೂ ವ್ಯವಸ್ಥೆಗಳು ಯಾವುದೇ ಸಮಸ್ಯೆಗಳಿಲ್ಲದೆ ಒಟ್ಟಿಗೆ ಕೆಲಸ ಮಾಡಬಹುದು ಎಂದು ಖಚಿತಪಡಿಸುತ್ತದೆ.

## ಸಮಸ್ಯೆ ಪರಿಹಾರ

### API ಸಂಪರ್ಕ ಸಮಸ್ಯೆಗಳು
- ನಿಮ್ಮ API key ಸರಿಯಾಗಿದೆಯೇ ಎಂದು ಪರಿಶೀಲಿಸಿ
- ನಿಮ್ಮ server ಮತ್ತು site ID ಗಳು ಸರಿಯಾಗಿವೆಯೇ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ
- ನಿಮ್ಮ GridPane ಖಾತೆಯು ಅಗತ್ಯ ಅನುಮತಿಗಳನ್ನು ಹೊಂದಿದೆಯೇ ಎಂದು ಖಾತ್ರಿಪಡಿಸಿಕೊಳ್ಳಿ

### SSL ಪ್ರಮಾಣಪತ್ರ ಸಮಸ್ಯೆಗಳು
- GridPane SSL ಪ್ರಮಾಣಪತ್ರಗಳನ್ನು ನೀಡಲು ಸ್ವಲ್ಪ ಸಮಯ ತೆಗೆದುಕೊಳ್ಳಬಹುದು
- ನಿಮ್ಮ ಡೊಮೇನ್‌ಗಳು ನಿಮ್ಮ ಸರ್ವರ್‌ನ IP ವಿಳಾಸಕ್ಕೆ ಸರಿಯಾಗಿ ಪಾಯಿಂಟ್ ಆಗಿವೆಯೇ ಎಂದು ಪರಿಶೀಲಿಸಿ
- ನಿಮ್ಮ ಸೈಟ್‌ಗಾಗಿ GridPane SSL ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಪರಿಶೀಲಿಸಿ

### ಡೊಮೇನ್ ಸೇರಿಸಲಾಗಿಲ್ಲ
- ಯಾವುದೇ ದೋಷ ಸಂದೇಶಗಳಿಗಾಗಿ Ultimate Multisite ಲಾಗ್‌ಗಳನ್ನು ಪರಿಶೀಲಿಸಿ
- ಡೊಮೇನ್ ಈಗಾಗಲೇ GridPane ಗೆ ಸೇರಿಸಲಾಗಿಲ್ಲ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ
- ನಿಮ್ಮ ಡೊಮೇನ್‌ನ DNS ರೆಕಾರ್ಡ್‌ಗಳು ಸರಿಯಾಗಿ ಕಾನ್ಫಿಗರ್ ಆಗಿವೆಯೇ ಎಂದು ಖಾತ್ರಿಪಡಿಸಿಕೊಳ್ಳಿ
