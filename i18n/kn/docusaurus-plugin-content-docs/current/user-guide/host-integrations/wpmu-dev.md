# WPMU DEV ಇಂಟಿಗ್ರೇಶನ್

## ಅವಲೋಕನ
WPMU DEV ಒಂದು ಸಮಗ್ರ WordPress ಪ್ಲಾಟ್‌ಫಾರ್ಮ್ ಆಗಿದ್ದು, WordPress ಸೈಟ್‌ಗಳಿಗೆ ಹೋಸ್ಟಿಂಗ್, ಪ್ಲಗಿನ್‌ಗಳು ಮತ್ತು ಸೇವೆಗಳನ್ನು ನೀಡುತ್ತದೆ. ಈ ಇಂಟಿಗ್ರೇಶನ್ Ultimate Multisite ಮತ್ತು WPMU DEV ಹೋಸ್ಟಿಂಗ್ ನಡುವೆ ಸ್ವಯಂಚಾಲಿತ ಡೊಮೇನ್ ಸಿಂಕಿಂಗ್ ಮತ್ತು SSL ಸರ್ಟಿಫಿಕೇಟ್ ನಿರ್ವಹಣೆಯನ್ನು ಸಕ್ರಿಯಗೊಳಿಸುತ್ತದೆ.

## ವೈಶಿಷ್ಟ್ಯಗಳು
- ಸ್ವಯಂಚಾಲಿತ ಡೊಮೇನ್ ಸಿಂಕಿಂಗ್
- SSL ಸರ್ಟಿಫಿಕೇಟ್ ನಿರ್ವಹಣೆ
- ವಿಸ್ತೃತ SSL ಸರ್ಟಿಫಿಕೇಟ್ ಪರಿಶೀಲನಾ ಪ್ರಯತ್ನಗಳು

## ಅವಶ್ಯಕತೆಗಳು
ನೀವು WPMU DEV ನಲ್ಲಿ ಹೋಸ್ಟ್ ಮಾಡುತ್ತಿದ್ದರೆ ಇಂಟಿಗ್ರೇಶನ್ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಪತ್ತೆಹಚ್ಚುತ್ತದೆ ಮತ್ತು ಅಂತರ್ನಿರ್ಮಿತ API ಅನ್ನು ಬಳಸುತ್ತದೆ. ನೀವು WPMU DEV ನಲ್ಲಿ ಹೋಸ್ಟ್ ಮಾಡುತ್ತಿದ್ದರೆ ಯಾವುದೇ ಹೆಚ್ಚುವರಿ ಕಾನ್ಫಿಗರೇಶನ್ ಅಗತ್ಯವಿಲ್ಲ.

WPMU DEV ನಲ್ಲಿ ಹೋಸ್ಟ್ ಮಾಡುವಾಗ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ವ್ಯಾಖ್ಯಾನಿಸಲಾದ `WPMUDEV_HOSTING_SITE_ID` ಕಾನ್‌ಸ್ಟಂಟ್ ಇರುವಿಕೆಯನ್ನು ಇಂಟಿಗ್ರೇಶನ್ ಪರಿಶೀಲಿಸುತ್ತದೆ.

## ಸೆಟಪ್ ಸೂಚನೆಗಳು

### 1. WPMU DEV ಹೋಸ್ಟಿಂಗ್ ಪರಿಶೀಲಿಸಿ

ನೀವು WPMU DEV ನಲ್ಲಿ ಹೋಸ್ಟ್ ಮಾಡುತ್ತಿದ್ದರೆ, ಅಗತ್ಯ ಕಾನ್‌ಸ್ಟಂಟ್‌ಗಳು ಈಗಾಗಲೇ ವ್ಯಾಖ್ಯಾನಿಸಲ್ಪಟ್ಟಿರಬೇಕು. ಇವುಗಳನ್ನು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ:

1. `WPMUDEV_HOSTING_SITE_ID` ಕಾನ್‌ಸ್ಟಂಟ್ ನಿಮ್ಮ ಎನ್ವಿರಾನ್‌ಮೆಂಟ್‌ನಲ್ಲಿ ವ್ಯಾಖ್ಯಾನಿಸಲಾಗಿದೆ
2. API ಪ್ರವೇಶದೊಂದಿಗೆ ಸಕ್ರಿಯ WPMU DEV ಸದಸ್ಯತ್ವ ನಿಮ್ಮಲ್ಲಿದೆ

### 2. ಇಂಟಿಗ್ರೇಶನ್ ಸಕ್ರಿಯಗೊಳಿಸಿ

1. ನಿಮ್ಮ WordPress ಅಡ್ಮಿನ್‌ನಲ್ಲಿ, Ultimate Multisite > Settings ಗೆ ಹೋಗಿ
2. "Domain Mapping" ಟ್ಯಾಬ್‌ಗೆ ನ್ಯಾವಿಗೇಟ್ ಮಾಡಿ
3. "Host Integrations" ಗೆ ಸ್ಕ್ರಾಲ್ ಮಾಡಿ
4. WPMU DEV ಇಂಟಿಗ್ರೇಶನ್ ಸಕ್ರಿಯಗೊಳಿಸಿ
5. "Save Changes" ಕ್ಲಿಕ್ ಮಾಡಿ

## ಇದು ಹೇಗೆ ಕೆಲಸ ಮಾಡುತ್ತದೆ

### ಡೊಮೇನ್ ಸಿಂಕಿಂಗ್

Ultimate Multisite ನಲ್ಲಿ ಡೊಮೇನ್ ಮ್ಯಾಪ್ ಆದಾಗ:

1. ನಿಮ್ಮ ಹೋಸ್ಟಿಂಗ್ ಖಾತೆಗೆ ಡೊಮೇನ್ ಸೇರಿಸಲು ಇಂಟಿಗ್ರೇಶನ್ WPMU DEV API ಅನ್ನು ಬಳಸುತ್ತದೆ
2. ಇದು ಡೊಮೇನ್‌ನ www ಆವೃತ್ತಿಯನ್ನು ಸಹ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸೇರಿಸುತ್ತದೆ
3. WPMU DEV ಡೊಮೇನ್ ಕಾನ್ಫಿಗರೇಶನ್ ಮತ್ತು SSL ಸರ್ಟಿಫಿಕೇಟ್ ನೀಡಿಕೆಯನ್ನು ನಿರ್ವಹಿಸುತ್ತದೆ

### SSL ಸರ್ಟಿಫಿಕೇಟ್ ನಿರ್ವಹಣೆ

SSL ಸರ್ಟಿಫಿಕೇಟ್‌ಗಳನ್ನು ನೀಡಲು ಮತ್ತು ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡಲು ಸ್ವಲ್ಪ ಸಮಯ ತೆಗೆದುಕೊಳ್ಳಬಹುದಾದ್ದರಿಂದ, WPMU DEV ಹೋಸ್ಟಿಂಗ್‌ಗಾಗಿ SSL ಸರ್ಟಿಫಿಕೇಟ್ ಪರಿಶೀಲನಾ ಪ್ರಯತ್ನಗಳ ಸಂಖ್ಯೆಯನ್ನು ಹೆಚ್ಚಿಸಲು ಇಂಟಿಗ್ರೇಶನ್ ಕಾನ್ಫಿಗರ್ ಮಾಡಲಾಗಿದೆ. ಪೂರ್ವನಿಯೋಜಿತವಾಗಿ, ಸ್ಟ್ಯಾಂಡರ್ಡ್ 5 ಪ್ರಯತ್ನಗಳಿಗೆ ಹೋಲಿಸಿದರೆ, SSL ಸರ್ಟಿಫಿಕೇಟ್ ಪರಿಶೀಲನೆಗಾಗಿ 10 ಬಾರಿವರೆಗೆ ಪ್ರಯತ್ನಿಸುತ್ತದೆ.

## ಮುಖ್ಯ ಟಿಪ್ಪಣಿಗಳು

### ಡೊಮೇನ್ ತೆಗೆದುಹಾಕುವಿಕೆ

ಪ್ರಸ್ತುತ, WPMU DEV API ಡೊಮೇನ್‌ಗಳನ್ನು ತೆಗೆದುಹಾಕಲು ಮಾರ್ಗವನ್ನು ಒದಗಿಸುವುದಿಲ್ಲ. Ultimate Multisite ನಲ್ಲಿ ಡೊಮೇನ್ ಮ್ಯಾಪಿಂಗ್ ತೆಗೆದುಹಾಕಿದಾಗ, ಡೊಮೇನ್ ನಿಮ್ಮ WPMU DEV ಹೋಸ್ಟಿಂಗ್ ಖಾತೆಯಲ್ಲಿ ಉಳಿಯುತ್ತದೆ. ಅಗತ್ಯವಿದ್ದರೆ WPMU DEV ಹೋಸ್ಟಿಂಗ್ ಡ್ಯಾಶ್‌ಬೋರ್ಡ್‌ನಿಂದ ನೀವು ಅದನ್ನು ಹಸ್ತಚಾಲಿತವಾಗಿ ತೆಗೆದುಹಾಕಬೇಕಾಗುತ್ತದೆ.

### API ದೃಢೀಕರಣ

ನಿಮ್ಮ WordPress ಡೇಟಾಬೇಸ್‌ನಲ್ಲಿ `wpmudev_apikey` ಆಯ್ಕೆಯಾಗಿ ಸಂಗ್ರಹವಾಗಿರುವ WPMU DEV API ಕೀಯನ್ನು ಇಂಟಿಗ್ರೇಶನ್ ಬಳಸುತ್ತದೆ. ನಿಮ್ಮ ಸೈಟ್ ಅನ್ನು WPMU DEV ಗೆ ಸಂಪರ್ಕಿಸಿದಾಗ ಇದು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸೆಟಪ್ ಆಗುತ್ತದೆ.

## ಸಮಸ್ಯೆ ನಿವಾರಣೆ

### API ಸಂಪರ್ಕ ಸಮಸ್ಯೆಗಳು
- ನಿಮ್ಮ ಸೈಟ್ WPMU DEV ಗೆ ಸರಿಯಾಗಿ ಸಂಪರ್ಕಗೊಂಡಿದೆಯೇ ಎಂದು ಪರಿಶೀಲಿಸಿ
- ನಿಮ್ಮ WordPress ಡೇಟಾಬೇಸ್‌ನಲ್ಲಿ `wpmudev_apikey` ಆಯ್ಕೆ ಸೆಟ್ ಆಗಿದೆಯೇ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ
- ನಿಮ್ಮ WPMU DEV ಸದಸ್ಯತ್ವ ಸಕ್ರಿಯವಾಗಿದೆ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ

### SSL ಸರ್ಟಿಫಿಕೇಟ್ ಸಮಸ್ಯೆಗಳು
- WPMU DEV SSL ಸರ್ಟಿಫಿಕೇಟ್‌ಗಳನ್ನು ನೀಡಲು ಸ್ವಲ್ಪ ಸಮಯ ತೆಗೆದುಕೊಳ್ಳಬಹುದು (ಸಾಮಾನ್ಯವಾಗಿ 5-15 ನಿಮಿಷಗಳು)
- SSL ಸರ್ಟಿಫಿಕೇಟ್‌ಗಳಿಗಾಗಿ 10 ಬಾರಿವರೆಗೆ ಪರಿಶೀಲಿಸಲು ಇಂಟಿಗ್ರೇಶನ್ ಕಾನ್ಫಿಗರ್ ಮಾಡಲಾಗಿದೆ
- ಹಲವಾರು ಪ್ರಯತ್ನಗಳ ನಂತರವೂ SSL ಸರ್ಟಿಫಿಕೇಟ್‌ಗಳು ನೀಡಲ್ಪಡದಿದ್ದರೆ, WPMU DEV ಬೆಂಬಲವನ್ನು ಸಂಪರ್ಕಿಸಿ

### ಡೊಮೇನ್ ಸೇರಿಸಲಾಗಲಿಲ್ಲ
- ಯಾವುದೇ ದೋಷ ಸಂದೇಶಗಳಿಗಾಗಿ Ultimate Multisite ಲಾಗ್‌ಗಳನ್ನು ಪರಿಶೀಲಿಸಿ
- ಡೊಮೇನ್ ಈಗಾಗಲೇ WPMU DEV ಗೆ ಸೇರಿಸಲಾಗಿಲ್ಲ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ
- ನೀವು ಸೇರಿಸುತ್ತಿರುವ ಡೊಮೇನ್‌ಗಳ ಸಂಖ್ಯೆಯನ್ನು ನಿಮ್ಮ WPMU DEV ಹೋಸ್ಟಿಂಗ್ ಪ್ಲಾನ್ ಬೆಂಬಲಿಸುತ್ತದೆ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ
