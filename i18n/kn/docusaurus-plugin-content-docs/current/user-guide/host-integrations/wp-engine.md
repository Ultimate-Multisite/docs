# WP Engine ಏಕೀಕರಣ

## ಅವಲೋಕನ
WP Engine ಒಂದು ಪ್ರೀಮಿಯಂ ನಿರ್ವಹಿತ WordPress ಹೋಸ್ಟಿಂಗ್ ಪ್ಲಾಟ್‌ಫಾರ್ಮ್ ಆಗಿದ್ದು, WordPress ಸೈಟ್‌ಗಳಿಗೆ ಉತ್ತಮೀಕರಿಸಿದ ಕಾರ್ಯಕ್ಷಮತೆ, ಭದ್ರತೆ ಮತ್ತು ಸ್ಕೇಲೆಬಿಲಿಟಿಯನ್ನು ಒದಗಿಸುತ್ತದೆ. ಈ ಏಕೀಕರಣವು Ultimate Multisite ಮತ್ತು WP Engine ನಡುವೆ ಸ್ವಯಂಚಾಲಿತ ಡೊಮೇನ್ ಸಿಂಕಿಂಗ್ ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸುತ್ತದೆ.

## ವೈಶಿಷ್ಟ್ಯಗಳು
- ಸ್ವಯಂಚಾಲಿತ ಡೊಮೇನ್ ಸಿಂಕಿಂಗ್
- ಮಲ್ಟಿಸೈಟ್ ಇನ್‌ಸ್ಟಾಲೇಶನ್‌ಗಳಿಗೆ ಸಬ್‌ಡೊಮೇನ್ ಬೆಂಬಲ
- WP Engine ನ ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ವ್ಯವಸ್ಥೆಗಳೊಂದಿಗೆ ತಡೆರಹಿತ ಏಕೀಕರಣ

## ಅಗತ್ಯತೆಗಳು
ನೀವು WP Engine ನಲ್ಲಿ ಹೋಸ್ಟ್ ಮಾಡುತ್ತಿದ್ದರೆ ಏಕೀಕರಣವು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಪತ್ತೆ ಮಾಡಿ ಅಂತರ್ನಿರ್ಮಿತ WP Engine API ಅನ್ನು ಬಳಸುತ್ತದೆ. WP Engine plugin ಸಕ್ರಿಯವಾಗಿದ್ದು ಸರಿಯಾಗಿ ಕಾನ್ಫಿಗರ್ ಆಗಿದ್ದರೆ ಯಾವುದೇ ಹೆಚ್ಚುವರಿ ಕಾನ್ಫಿಗರೇಶನ್ ಅಗತ್ಯವಿಲ್ಲ.

ಆದಾಗ್ಯೂ, ನೀವು ಏಕೀಕರಣವನ್ನು ಹಸ್ತಚಾಲಿತವಾಗಿ ಕಾನ್ಫಿಗರ್ ಮಾಡಬೇಕಾದರೆ, ನಿಮ್ಮ `wp-config.php` ಫೈಲ್‌ನಲ್ಲಿ ಈ constants ಗಳಲ್ಲಿ ಒಂದನ್ನು ಡಿಫೈನ್ ಮಾಡಬಹುದು:

```php
define('WPE_APIKEY', 'your_api_key'); // ಆದ್ಯತೆಯ ವಿಧಾನ
// ಅಥವಾ
define('WPE_API', 'your_api_key'); // ಪರ್ಯಾಯ ವಿಧಾನ
```

## ಸೆಟಪ್ ಸೂಚನೆಗಳು

### 1. WP Engine Plugin ಅನ್ನು ಪರಿಶೀಲಿಸಿ

ನೀವು WP Engine ನಲ್ಲಿ ಹೋಸ್ಟ್ ಮಾಡುತ್ತಿದ್ದರೆ, WP Engine plugin ಈಗಾಗಲೇ ಇನ್‌ಸ್ಟಾಲ್ ಆಗಿ ಸಕ್ರಿಯಗೊಂಡಿರಬೇಕು. ಇವುಗಳನ್ನು ಪರಿಶೀಲಿಸಿ:

1. WP Engine plugin ಸಕ್ರಿಯವಾಗಿದೆ
2. `wp-content/mu-plugins/wpengine-common/class-wpeapi.php` ಫೈಲ್ ಇದೆ

### 2. ಏಕೀಕರಣವನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ

1. ನಿಮ್ಮ WordPress admin ನಲ್ಲಿ, Ultimate Multisite > Settings ಗೆ ಹೋಗಿ
2. "Domain Mapping" ಟ್ಯಾಬ್‌ಗೆ ನ್ಯಾವಿಗೇಟ್ ಮಾಡಿ
3. "Host Integrations" ಗೆ ಕೆಳಗೆ ಸ್ಕ್ರಾಲ್ ಮಾಡಿ
4. WP Engine ಏಕೀಕರಣವನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ
5. "Save Changes" ಕ್ಲಿಕ್ ಮಾಡಿ

## ಇದು ಹೇಗೆ ಕೆಲಸ ಮಾಡುತ್ತದೆ

### ಡೊಮೇನ್ ಸಿಂಕಿಂಗ್

Ultimate Multisite ನಲ್ಲಿ ಡೊಮೇನ್ ಮ್ಯಾಪ್ ಆದಾಗ:

1. ಏಕೀಕರಣವು ನಿಮ್ಮ WP Engine ಇನ್‌ಸ್ಟಾಲೇಶನ್‌ಗೆ ಡೊಮೇನ್ ಸೇರಿಸಲು WP Engine API ಬಳಸುತ್ತದೆ
2. WP Engine ಡೊಮೇನ್ ಕಾನ್ಫಿಗರೇಶನ್ ಮತ್ತು SSL certificate ನೀಡಿಕೆಯನ್ನು ನಿರ್ವಹಿಸುತ್ತದೆ
3. ಡೊಮೇನ್ ಮ್ಯಾಪಿಂಗ್ ತೆಗೆದುಹಾಕಿದಾಗ, ಏಕೀಕರಣವು WP Engine ನಿಂದ ಡೊಮೇನ್ ಅನ್ನು ತೆಗೆದುಹಾಕುತ್ತದೆ

### ಸಬ್‌ಡೊಮೇನ್ ಬೆಂಬಲ

ಸಬ್‌ಡೊಮೇನ್ ಮಲ್ಟಿಸೈಟ್ ಇನ್‌ಸ್ಟಾಲೇಶನ್‌ಗಳಿಗೆ:

1. ಹೊಸ ಸೈಟ್ ರಚಿಸಿದಾಗ ಏಕೀಕರಣವು ಪ್ರತಿ ಸಬ್‌ಡೊಮೇನ್ ಅನ್ನು WP Engine ಗೆ ಸೇರಿಸುತ್ತದೆ
2. WP Engine ಸಬ್‌ಡೊಮೇನ್ ಕಾನ್ಫಿಗರೇಶನ್ ಅನ್ನು ನಿರ್ವಹಿಸುತ್ತದೆ
3. ಸೈಟ್ ಅಳಿಸಿದಾಗ, ಏಕೀಕರಣವು WP Engine ನಿಂದ ಸಬ್‌ಡೊಮೇನ್ ಅನ್ನು ತೆಗೆದುಹಾಕುತ್ತದೆ

## ಪ್ರಮುಖ ಟಿಪ್ಪಣಿಗಳು

### ವೈಲ್ಡ್‌ಕಾರ್ಡ್ ಡೊಮೇನ್‌ಗಳು

ಸಬ್‌ಡೊಮೇನ್ ಮಲ್ಟಿಸೈಟ್ ಇನ್‌ಸ್ಟಾಲೇಶನ್‌ಗಳಿಗೆ, ವೈಲ್ಡ್‌ಕಾರ್ಡ್ ಡೊಮೇನ್ ಕಾನ್ಫಿಗರೇಶನ್ ಕೋರಲು WP Engine ಬೆಂಬಲವನ್ನು ಸಂಪರ್ಕಿಸುವುದು ಸೂಕ್ತ. ಇದು ಪ್ರತಿಯೊಂದನ್ನು ಪ್ರತ್ಯೇಕವಾಗಿ ಸೇರಿಸದೆ ಎಲ್ಲಾ ಸಬ್‌ಡೊಮೇನ್‌ಗಳು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಕೆಲಸ ಮಾಡಲು ಅನುಮತಿಸುತ್ತದೆ.

### SSL Certificates

ಈ ಏಕೀಕರಣದ ಮೂಲಕ ಸೇರಿಸಲಾದ ಎಲ್ಲಾ ಡೊಮೇನ್‌ಗಳಿಗೆ WP Engine ಸ್ವಯಂಚಾಲಿತವಾಗಿ SSL certificate ನೀಡಿಕೆ ಮತ್ತು ನವೀಕರಣವನ್ನು ನಿರ್ವಹಿಸುತ್ತದೆ. ಯಾವುದೇ ಹೆಚ್ಚುವರಿ ಕಾನ್ಫಿಗರೇಶನ್ ಅಗತ್ಯವಿಲ್ಲ.

## ಸಮಸ್ಯೆ ಪರಿಹಾರ

### API ಸಂಪರ್ಕ ಸಮಸ್ಯೆಗಳು
- WP Engine plugin ಸಕ್ರಿಯವಾಗಿದ್ದು ಸರಿಯಾಗಿ ಕಾನ್ಫಿಗರ್ ಆಗಿದೆಯೇ ಎಂದು ಪರಿಶೀಲಿಸಿ
- ನೀವು API key ಅನ್ನು ಹಸ್ತಚಾಲಿತವಾಗಿ ಡಿಫೈನ್ ಮಾಡಿದ್ದರೆ, ಅದು ಸರಿಯಾಗಿದೆಯೇ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ
- API ಯೊಂದಿಗೆ ಸಮಸ್ಯೆಗಳಿದ್ದರೆ WP Engine ಬೆಂಬಲವನ್ನು ಸಂಪರ್ಕಿಸಿ

### ಡೊಮೇನ್ ಸೇರಿಸಲಾಗಿಲ್ಲ
- ಯಾವುದೇ ದೋಷ ಸಂದೇಶಗಳಿಗಾಗಿ Ultimate Multisite logs ಅನ್ನು ಪರಿಶೀಲಿಸಿ
- ಡೊಮೇನ್ ಈಗಾಗಲೇ WP Engine ಗೆ ಸೇರಿಸಲಾಗಿಲ್ಲ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ
- ನಿಮ್ಮ WP Engine ಪ್ಲಾನ್ ನೀವು ಸೇರಿಸುತ್ತಿರುವ ಡೊಮೇನ್‌ಗಳ ಸಂಖ್ಯೆಯನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆಯೇ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ

### ಸಬ್‌ಡೊಮೇನ್ ಸಮಸ್ಯೆಗಳು
- ಸಬ್‌ಡೊಮೇನ್‌ಗಳು ಕೆಲಸ ಮಾಡದಿದ್ದರೆ, ವೈಲ್ಡ್‌ಕಾರ್ಡ್ ಡೊಮೇನ್ ಕಾನ್ಫಿಗರೇಶನ್ ಕೋರಲು WP Engine ಬೆಂಬಲವನ್ನು ಸಂಪರ್ಕಿಸಿ
- ಮುಖ್ಯ ಡೊಮೇನ್ ಮತ್ತು ಸಬ್‌ಡೊಮೇನ್‌ಗಳಿಗೆ ನಿಮ್ಮ DNS ಸೆಟ್ಟಿಂಗ್‌ಗಳು ಸರಿಯಾಗಿ ಕಾನ್ಫಿಗರ್ ಆಗಿವೆಯೇ ಎಂದು ಪರಿಶೀಲಿಸಿ
