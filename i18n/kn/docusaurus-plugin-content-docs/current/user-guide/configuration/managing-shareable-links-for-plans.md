# ಯೋಜನೆಗಳಿಗೆ ಹಂಚಿಕೊಳ್ಳಬಹುದಾದ ಲಿಂಕ್‌ಗಳನ್ನು ನಿರ್ವಹಿಸುವುದು

Ultimate Multisite ನಿಮ್ಮ ಗ್ರಾಹಕರೊಂದಿಗೆ ಯೋಜನೆ ಲಿಂಕ್‌ಗಳನ್ನು ನೇರವಾಗಿ ಹಂಚಿಕೊಳ್ಳುವ ಆಯ್ಕೆಯನ್ನು ನೀಡುತ್ತದೆ. ಇದನ್ನು ಮಾಡಲು, ನಿಮ್ಮ ಯಾವುದೇ ಯೋಜನೆಯ ಸಂಪಾದನೆ ಪುಟಕ್ಕೆ ಹೋಗಿ ಮತ್ತು _ಹಂಚಿಕೊಳ್ಳಬಹುದಾದ ಲಿಂಕ್ ನಕಲಿಸಲು ಕ್ಲಿಕ್ ಮಾಡಿ_ ಬಟನ್ ಒತ್ತಿ.

![ಹಂಚಿಕೊಳ್ಳಬಹುದಾದ ಲಿಂಕ್ ಬಟನ್ ಇರುವ ಉತ್ಪನ್ನ ಸಂಪಾದನೆ ಪುಟ](/img/config/product-edit-full.png)

_ಯೋಜನೆಯ ಸಂಪಾದನೆ ಪುಟದಲ್ಲಿ, ಹಂಚಿಕೊಳ್ಳಬಹುದಾದ ಲಿಂಕ್‌ಗಳನ್ನು ರಚಿಸಲು ಬಟನ್ ಕಾಣಸಿಗುತ್ತದೆ_

ಈ ವ್ಯವಸ್ಥೆಯು ಲಿಂಕ್ ರಚಿಸಲು ಡೀಫಾಲ್ಟ್ ಬಿಲ್ಲಿಂಗ್ ಆವರ್ತನವನ್ನು ಬಳಸುತ್ತದೆ ಎಂಬುದನ್ನು ಗಮನಿಸಿ. ಅಂದರೆ ನಿಮ್ಮ ಡೀಫಾಲ್ಟ್ ಬಿಲ್ಲಿಂಗ್ ಮಾಸಿಕವಾಗಿದ್ದರೆ, ಉದಾಹರಣೆಗೆ, ನಿಮಗೆ ಈ ರೀತಿಯ ಲಿಂಕ್ ಸಿಗುತ್ತದೆ: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

ಹಂಚಿಕೊಳ್ಳಬಹುದಾದ ಲಿಂಕ್ ತ್ರೈಮಾಸಿಕಕ್ಕೆ ಡೀಫಾಲ್ಟ್ ಆಗಬೇಕೆಂದಿದ್ದರೆ, ಕೊನೆಯಲ್ಲಿ 3 ಸೇರಿಸಬೇಕಾಗುತ್ತದೆ

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

ವಾರ್ಷಿಕ ಶೆಡ್ಯೂಲ್‌ನಲ್ಲಿ _premium_ ಯೋಜನೆ ಆಯ್ಕೆ ಮಾಡಲು, 1 ಮತ್ತು ನಂತರ ಪದವನ್ನು ಸೇರಿಸಿ:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

ವಿವಿಧ ಬಿಲ್ಲಿಂಗ್ ಆವರ್ತನಗಳಿಗೆ ಒಂದೇ ಲಿಂಕ್ ಇರಲು ಸಾಧ್ಯವಿಲ್ಲ ಎಂಬುದನ್ನು ನೆನಪಿಡಿ. ನಿಮ್ಮ ಪ್ರತಿಯೊಂದು ಬಿಲ್ಲಿಂಗ್ ಆವರ್ತನಕ್ಕೆ ಪ್ರತ್ಯೇಕ ಲಿಂಕ್ ಹಂಚಿಕೊಳ್ಳಬೇಕು.

ನಿಮ್ಮ ಗ್ರಾಹಕರೊಂದಿಗೆ ಲಿಂಕ್ ಹಂಚಿಕೊಂಡ ನಂತರ, ಅದು ಅವರನ್ನು ನಿಮ್ಮ ಸೈನ್‌ಅಪ್ ಪ್ರಕ್ರಿಯೆಯ ಮೊದಲ ಹಂತಕ್ಕೆ ಕರೆದೊಯ್ಯುತ್ತದೆ.
