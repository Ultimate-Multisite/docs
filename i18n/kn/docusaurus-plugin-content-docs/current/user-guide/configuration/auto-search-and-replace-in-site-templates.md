# ಸೈಟ್ ಟೆಂಪ್ಲೇಟ್‌ಗಳಲ್ಲಿ ಆಟೋ ಸರ್ಚ್ ಮತ್ತು ರಿಪ್ಲೇಸ್ (v2)

_**ಈ ಟ್ಯುಟೋರಿಯಲ್‌ಗೆ WP UItimo ಆವೃತ್ತಿ 2.x ಅಗತ್ಯವಿದೆ.**_

Ultimate Multisite ನ ಅತ್ಯಂತ ಶಕ್ತಿಶಾಲಿ ವೈಶಿಷ್ಟ್ಯಗಳಲ್ಲಿ ಒಂದು ಎಂದರೆ ನೋಂದಣಿ ಫಾರ್ಮ್‌ಗೆ ಬೇಕಾದಷ್ಟು ಟೆಕ್ಸ್ಟ್, ಬಣ್ಣ, ಮತ್ತು ಸೆಲೆಕ್ಟ್ ಫೀಲ್ಡ್‌ಗಳನ್ನು ಸೇರಿಸುವ ಸಾಮರ್ಥ್ಯ. ಆ ಡೇಟಾವನ್ನು ಸಂಗ್ರಹಿಸಿದ ನಂತರ, ಆಯ್ಕೆ ಮಾಡಿದ ಸೈಟ್ ಟೆಂಪ್ಲೇಟ್‌ನ ಕೆಲವು ಭಾಗಗಳಲ್ಲಿ ವಿಷಯವನ್ನು ಮುಂಚಿತವಾಗಿ ಭರ್ತಿ ಮಾಡಲು ಅದನ್ನು ಬಳಸಬಹುದು. ನಂತರ, ಹೊಸ ಸೈಟ್ ಪ್ರಕಟವಾದಾಗ, Ultimate Multisite ನೋಂದಣಿ ಸಮಯದಲ್ಲಿ ನಮೂದಿಸಿದ ನಿಜವಾದ ಮಾಹಿತಿಯೊಂದಿಗೆ ಪ್ಲೇಸ್‌ಹೋಲ್ಡರ್‌ಗಳನ್ನು ಬದಲಾಯಿಸುತ್ತದೆ.

ಉದಾಹರಣೆಗೆ, ನಿಮ್ಮ ಟೆಂಪ್ಲೇಟ್ ಸೈಟ್‌ಗಳನ್ನು ಪ್ಲೇಸ್‌ಹೋಲ್ಡರ್‌ಗಳೊಂದಿಗೆ ಮಾಡಬಹುದು. ಪ್ಲೇಸ್‌ಹೋಲ್ಡರ್‌ಗಳನ್ನು ಡಬಲ್ ಕರ್ಲಿ ಬ್ರೇಸ್‌ಗಳಲ್ಲಿ ಸುತ್ತುವರಿಸಿ ಸೇರಿಸಬೇಕು - {{placeholder_name}}.

ನಂತರ, ಆ ಡೇಟಾವನ್ನು ಸಂಗ್ರಹಿಸಲು ಹೊಂದಿಕೆಯಾಗುವ ನೋಂದಣಿ ಫೀಲ್ಡ್ ಅನ್ನು ಸರಳವಾಗಿ ಸೇರಿಸಬಹುದು

ನಿಮ್ಮ ಗ್ರಾಹಕರು ನೋಂದಣಿ ಸಮಯದಲ್ಲಿ ಆ ಫೀಲ್ಡ್ ಅನ್ನು ಭರ್ತಿ ಮಾಡಬಹುದು.

Ultimate Multisite ನಂತರ ಗ್ರಾಹಕರು ಒದಗಿಸಿದ ಡೇಟಾದೊಂದಿಗೆ ಪ್ಲೇಸ್‌ಹೋಲ್ಡರ್‌ಗಳನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಬದಲಾಯಿಸುತ್ತದೆ.

## **"ಪ್ಲೇಸ್‌ಹೋಲ್ಡರ್‌ಗಳಿಂದ ತುಂಬಿದ ಟೆಂಪ್ಲೇಟ್" ಸಮಸ್ಯೆಯನ್ನು ಪರಿಹರಿಸುವುದು**

ಇದೆಲ್ಲಾ ಅದ್ಭುತವಾಗಿದೆ, ಆದರೆ ನಾವು ಒಂದು ಕೆಟ್ಟ ಸಮಸ್ಯೆಯನ್ನು ಎದುರಿಸುತ್ತೇವೆ: ಈಗ ನಮ್ಮ ಸೈಟ್ ಟೆಂಪ್ಲೇಟ್‌ಗಳು - ನಮ್ಮ ಗ್ರಾಹಕರು ಭೇಟಿ ನೀಡಬಹುದಾದವು - ಹೆಚ್ಚು ಅರ್ಥ ತಿಳಿಸದ ಕೆಟ್ಟ ಪ್ಲೇಸ್‌ಹೋಲ್ಡರ್‌ಗಳಿಂದ ತುಂಬಿವೆ.

ಇದನ್ನು ಪರಿಹರಿಸಲು, ಪ್ಲೇಸ್‌ಹೋಲ್ಡರ್‌ಗಳಿಗೆ ನಕಲಿ ಮೌಲ್ಯಗಳನ್ನು ಹೊಂದಿಸುವ ಆಯ್ಕೆಯನ್ನು ನಾವು ನೀಡುತ್ತೇವೆ, ಮತ್ತು ನಿಮ್ಮ ಗ್ರಾಹಕರು ಭೇಟಿ ನೀಡುತ್ತಿರುವಾಗ ಟೆಂಪ್ಲೇಟ್ ಸೈಟ್‌ಗಳಲ್ಲಿ ಅವುಗಳ ವಿಷಯಗಳನ್ನು ಹುಡುಕಿ ಬದಲಾಯಿಸಲು ಆ ಮೌಲ್ಯಗಳನ್ನು ನಾವು ಬಳಸುತ್ತೇವೆ.

**Ultimate Multisite > Settings > Sites** ಗೆ ಹೋಗಿ, ನಂತರ ಸೈಡ್‌ಬಾರ್‌ನಲ್ಲಿ Edit Placeholders ಲಿಂಕ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡುವ ಮೂಲಕ ಟೆಂಪ್ಲೇಟ್ ಪ್ಲೇಸ್‌ಹೋಲ್ಡರ್‌ಗಳ ಎಡಿಟರ್ ಅನ್ನು ಪ್ರವೇಶಿಸಬಹುದು.

![Sites ಸೆಟ್ಟಿಂಗ್ಸ್ ಸೈಡ್‌ಬಾರ್‌ನಲ್ಲಿ Edit Placeholders ಲಿಂಕ್](/img/config/settings-sites.png)

ಅದು ನಿಮ್ಮನ್ನು ಪ್ಲೇಸ್‌ಹೋಲ್ಡರ್‌ಗಳ ವಿಷಯ ಎಡಿಟರ್‌ಗೆ ಕರೆದೊಯ್ಯುತ್ತದೆ, ಅಲ್ಲಿ ನೀವು ಪ್ಲೇಸ್‌ಹೋಲ್ಡರ್‌ಗಳು ಮತ್ತು ಅವುಗಳ ಸಂಬಂಧಿತ ವಿಷಯವನ್ನು ಸೇರಿಸಬಹುದು.

![ಪ್ಲೇಸ್‌ಹೋಲ್ಡರ್ ಹೆಸರುಗಳು ಮತ್ತು ಮೌಲ್ಯಗಳೊಂದಿಗೆ ಪ್ಲೇಸ್‌ಹೋಲ್ಡರ್‌ಗಳ ವಿಷಯ ಎಡಿಟರ್](/img/config/settings-sites.png)
