# ಥೀಮ್-ಅವೇರ್ ಸ್ಕಿಲ್ಸ್ (Theme-Aware Skills)

Superdav AI Agent v1.10.0 ನಾಲ್ಕು ಹೊಸ ಥೀಮ್-ಅವೇರ್ (Theme-aware) ಬिल्ಟ್-ಇನ್ ಸ್ಕಿಲ್ಸ್‌ಗಳನ್ನು ಒಳಗೊಂಡಿದೆ. ಇವುಗಳು ನಿಮ್ಮ ಸಕ್ರಿಯವಾದ WordPress ಥೀಮ್‌ಗೆ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಹೊಂದಿಕೊಳ್ಳುತ್ತವೆ. ಈ ಸ್ಕಿಲ್ಸ್‌ಗಳು ನಿಮ್ಮ ಥೀಮ್‌ನ ರಚನೆ ಮತ್ತು ವೈಶಿಷ್ಟ್ಯಗಳಿಗೆ ಅನುಗುಣವಾಗಿ ವಿಶೇಷ ಮಾರ್ಗದರ್ಶನ ಮತ್ತು ಸಾಮರ್ಥ್ಯಗಳನ್ನು ಒದಗಿಸುತ್ತವೆ.

## ಥೀಮ್-ಅವೇರ್ ಸ್ಕಿಲ್ಸ್ ಎಂದರೇನು?

ಥೀಮ್-ಅವೇರ್ ಸ್ಕಿಲ್ಸ್‌ಗಳು ಎಂದರೆ ಪೂರ್ವ-ಸಂಯೋಜಿತ ಜ್ಞಾನದ ಆಧಾರಗಳು (knowledge bases) ಮತ್ತು ಉಪಕರಣಗಳ ಸೆಟ್‌ಗಳು. ನಿಮ್ಮ ಸೈಟ್‌ನಲ್ಲಿ ಪ್ರಸ್ತುತ ಸಕ್ರಿಯವಾಗಿರುವ WordPress ಥೀಮ್ ಆಧಾರದ ಮೇಲೆ AI ಅಸಿಸ್ಟೆಂಟ್ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಇವುಗಳನ್ನು ಆಯ್ಕೆ ಮಾಡುತ್ತದೆ. ನೀವು ಥೀಮ್ ಬದಲಾಯಿಸಿದಾಗ, ಅಸಿಸ್ಟೆಂಟ್‌ನ ಲಭ್ಯವಿರುವ ಸ್ಕಿಲ್ಸ್‌ಗಳು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಅಪ್ಡೇಟ್ ಆಗುತ್ತವೆ — ಯಾವುದೇ ಕೈಯಾರೆ ಸೆಟಪ್ ಮಾಡುವ ಅಗತ್ಯವಿಲ್ಲ.

ಪ್ರತಿಯೊಂದು ಸ್ಕಿಲ್‌ನಲ್ಲಿ ಈ ಕೆಳಗಿನ ಅಂಶಗಳು ಸೇರಿವೆ:

- **ಥೀಮ್-ನಿರ್ದಿಷ್ಟ ಡಾಕ್ಯುಮೆಂಟೇಶನ್** — ಥೀಮ್ ಅನ್ನು ಬಳಸಲು ಮತ್ತು ಕಸ್ಟಮೈಸ್ ಮಾಡಲು ಮಾರ್ಗದರ್ಶನ
- **ಬ್ಲಾಕ್ ಮತ್ತು ಪ್ಯಾಟರ್ನ್ referencs** — ಲಭ್ಯವಿರುವ ಬ್ಲಾಕ್‌ಗಳು, ಪ್ಯಾಟರ್ನ್‌ಗಳು ಮತ್ತು ವಿನ್ಯಾಸ ಆಯ್ಕೆಗಳು
- **ಕಸ್ಟಮೈಸೇಶನ್ ಉದಾಹರಣೆಗಳು** — ಸಾಮಾನ್ಯ ಕಾರ್ಯಗಳಿಗಾಗಿ ಕೋಡ್ ಸ್ನಿಪ್ಪೆಟ್‌ಗಳು ಮತ್ತು ಕಾನ್ಫಿಗರೇಶನ್ ಪ್ಯಾಟರ್ನ್‌ಗಳು
- **ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು (Best practices)** — ಥೀಮ್‌ನ ರಚನೆ ಮತ್ತು ಕಾರ್ಯಪ್ರವಾಹಕ್ಕೆ (workflow) ಸಲಹೆಗಳು

## ಲಭ್ಯವಿರುವ ಥೀಮ್-ಅವೇರ್ ಸ್ಕಿಲ್ಸ್

### ಬ್ಲಾಕ್ ಥೀಮ್ಸ್ (Block Themes)

**ಯಾವುದಕ್ಕೆ ಅನ್ವಯಿಸುತ್ತದೆ:** WordPress ಬ್ಲಾಕ್-ಆಧಾರಿತ (Full Site Editing) ರಚನೆಯನ್ನು ಬಳಸುವ ಥೀಮ್‌ಗಳು.

ಬ್ಲಾಕ್ ಥೀಮ್ಸ್ ಸ್ಕಿಲ್ ಈ ಕೆಳಗಿನ ವಿಷಯಗಳ ಬಗ್ಗೆ ಮಾರ್ಗದರ್ಶನ ನೀಡುತ್ತದೆ:

- ಬ್ಲಾಕ್ ಎಡಿಟರ್ ಬಳಸಿ ಟೆಂಪ್ಲೇಟ್‌ಗಳನ್ನು ರಚಿಸುವ ಮತ್ತು ಸಂಪಾದಿಸುವ ವಿಧಾನ
- ಬ್ಲಾಕ್ ಪ್ಯಾಟರ್ನ್‌ಗಳು ಮತ್ತು ಮರುಬಳಕೆಯ ಬ್ಲಾಕ್‌ಗಳೊಂದಿಗೆ ಕೆಲಸ ಮಾಡುವುದು
- ಗ್ಲೋಬಲ್ ಸ್ಟೈಲ್‌ಗಳು ಮತ್ತು theme.json ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಕಸ್ಟಮೈಸ್ ಮಾಡುವುದು
- ಥೀಮ್ ಬ್ಲಾಕ್‌ಗಳು ಮತ್ತು variation ಗಳನ್ನು ಬಳಸುವುದು
- ನಿಮ್ಮ ಸೈಟ್‌ಗಾಗಿ ಕಸ್ಟಮ್ ಬ್ಲಾಕ್ ಪ್ಯಾಟರ್ನ್‌ಗಳನ್ನು ನಿರ್ಮಿಸುವುದು

**ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸಕ್ರಿಯವಾಗುತ್ತದೆ ಯಾವಾಗ:** ನಿಮ್ಮ ಸಕ್ರಿಯ ಥೀಮ್ ಒಂದು ಬ್ಲಾಕ್ ಥೀಮ್ ಆಗಿದ್ದರೆ (ಇದು `block-templates` ವೈಶಿಷ್ಟ್ಯವನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ).

### ಕ್ಲಾಸಿಕ್ ಥೀಮ್ಸ್ (Classic Themes)

**ಯಾವುದಕ್ಕೆ ಅನ್ವಯಿಸುತ್ತದೆ:** PHP ಟೆಂಪ್ಲೇಟ್‌ಗಳು ಮತ್ತು ಕ್ಲಾಸಿಕ್ ಎಡಿಟರ್ ಅನ್ನು ಬಳಸುವ ಸಾಂಪ್ರದಾಯಿಕ WordPress ಥೀಮ್‌ಗಳು.

ಕ್ಲಾಸಿಕ್ ಥೀಮ್ಸ್ ಸ್ಕಿಲ್ ಈ ಕೆಳಗಿನ ವಿಷಯಗಳ ಬಗ್ಗೆ ಮಾರ್ಗದರ್ಶನ ನೀಡುತ್ತದೆ:

- PHP ಟೆಂಪ್ಲೇಟ್ ಫೈಲ್‌ಗಳು ಮತ್ತು hooks ನೊಂದಿಗೆ ಕೆಲಸ ಮಾಡುವುದು
- ಕಸ್ಟಮೈಜರ್ ಮೂಲಕ ಥೀಮ್ appearances ಅನ್ನು ಕಸ್ಟಮೈಸ್ ಮಾಡುವುದು
- ವಿಜೆಟ್ ಏರಿಯಾಗ್‌ಗಳು ಮತ್ತು ಸೈಡ್‌ಬಾರ್‌ಗಳನ್ನು ಬಳಸುವುದು
- CSS ಅನ್ನು ಮಾರ್ಪಡಿಸುವುದು ಮತ್ತು child theme development
- ಥೀಮ್ ಹೈರಾರ್ಕಿ ಮತ್ತು ಟೆಂಪ್ಲೇಟ್ ಟ್ಯಾಗ್‌ಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವುದು

**ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸಕ್ರಿಯವಾಗುತ್ತದೆ ಯಾವಾಗ:** ನಿಮ್ಮ ಸಕ್ರಿಯ ಥೀಮ್ ಒಂದು ಕ್ಲಾಸಿಕ್ (non-block) ಥೀಮ್ ಆಗಿದ್ದರೆ.

### Kadence Blocks

**ಯಾವುದಕ್ಕೆ ಅನ್ವಯಿಸುತ್ತದೆ:** ಸುಧಾರಿತ ಬ್ಲಾಕ್-ಆಧಾರಿತ ವಿನ್ಯಾಸಕ್ಕಾಗಿ Kadence Blocks plugin ಅನ್ನು ಬಳಸುವ ಸೈಟ್‌ಗಳು.

Kadence Blocks ಸ್ಕಿಲ್ ಈ ಕೆಳಗಿನ ವಿಷಯಗಳ ಬಗ್ಗೆ ಮಾರ್ಗದರ್ಶನ ನೀಡುತ್ತದೆ:

- Kadence ನ ಸುಧಾರಿತ ಬ್ಲಾಕ್ ಲೈಬ್ರರಿ ಬಳಸುವುದು (Hero, Testimonials, Pricing, ಇತ್ಯಾದಿ)
- Kadence ಬ್ಲಾಕ್ ಸೆಟ್ಟಿಂಗ್‌ಗಳು ಮತ್ತು responsive ಆಯ್ಕೆಗಳನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡುವುದು
- Kadence ನ grid ಮತ್ತು container ಬ್ಲಾಕ್‌ಗಳೊಂದಿಗೆ ಕಸ್ಟಮ್ ಲೇಔಟ್‌ಗಳನ್ನು ನಿರ್ಮಿಸುವುದು
- ನಿಮ್ಮ ಥೀಮ್‌ನೊಂದಿಗೆ Kadence ಬ್ಲಾಕ್‌ಗಳನ್ನು ಸಂಯೋಜಿಸುವುದು (Integrating)
- Kadence ನ design system ಮತ್ತು presets ಅನ್ನು ಬಳಸಿಕೊಳ್ಳುವುದು

**ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸಕ್ರಿಯವಾಗುತ್ತದೆ ಯಾವಾಗ:** Kadence Blocks plugin ನಿಮ್ಮ ಸೈಟ್‌ನಲ್ಲಿ ಸಕ್ರಿಯವಾಗಿದ್ದರೆ.

### Kadence Theme

**ಯಾವುದಕ್ಕೆ ಅನ್ವಯಿಸುತ್ತದೆ:** ಬ್ಲಾಕ್-ಆಧಾರಿತ ವಿನ್ಯಾಸ ಮತ್ತು ಕಸ್ಟಮೈಸೇಶನ್‌ಗಾಗಿ Kadence ಥೀಮ್ ಅನ್ನು ಬಳಸುವ ಸೈಟ್‌ಗಳು.

Kadence Theme ಸ್ಕಿಲ್ ಈ ಕೆಳಗಿನ ವಿಷಯಗಳ ಬಗ್ಗೆ ಮಾರ್ಗದರ್ಶನ ನೀಡುತ್ತದೆ:

- ಗ್ಲೋಬಲ್ ಸ್ಟೈಲ್‌ಗಳು ಮತ್ತು theme.json ಮೂಲಕ Kadence ಥೀಮ್ ಅನ್ನು ಕಸ್ಟಮೈಸ್ ಮಾಡುವುದು
- Kadence ನ ಬिल्ಟ್-ಇನ್ ಬ್ಲಾಕ್ ಪ್ಯಾಟರ್ನ್‌ಗಳು ಮತ್ತು ಟೆಂಪ್ಲೇಟ್‌ಗಳನ್ನು ಬಳಸುವುದು
- Kadence ಥೀಮ್ ಸೆಟ್ಟಿಂಗ್‌ಗಳು ಮತ್ತು ಆಯ್ಕೆಗಳನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡುವುದು
- Kadence ನ design system ಬಳಸಿ ಕಸ್ಟಮ್ ವಿನ್ಯಾಸಗಳನ್ನು ನಿರ್ಮಿಸುವುದು
- ಜನಪ್ರಿಯ pluginಗಳು ಮತ್ತು ಉಪಕರಣಗಳೊಂದಿಗೆ Kadence ಅನ್ನು ಸಂಯೋಜಿಸುವುದು

**ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸಕ್ರಿಯವಾಗುತ್ತದೆ ಯಾವಾಗ:** Kadence ಥೀಮ್ ನಿಮ್ಮ ಸಕ್ರಿಯ ಥೀಮ್ ಆಗಿದ್ದರೆ.

## ಸ್ಕಿಲ್ಸ್ ಹೇಗೆ ಆಯ್ಕೆಯಾಗುತ್ತವೆ

ಅಸಿಸ್ಟೆಂಟ್ ಪ್ರತಿ ಸಂದೇಶದೊಂದಿಗೆ ನಿಮ್ಮ ಸಕ್ರಿಯ ಥೀಮ್ ಮತ್ತು ಸ್ಥಾಪಿಸಲಾದ plugin ಗಳನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಪತ್ತೆಹಚ್ಚುತ್ತದೆ. ಹೊಂದಿಕೆಯಾಗುವ ಥೀಮ್-ಅವೇರ್ ಸ್ಕಿಲ್ ಲಭ್ಯವಿದ್ದರೆ, ಅದನ್ನು ಅಸಿಸ್ಟೆಂಟ್‌ನ ಸಂದರ್ಭಕ್ಕೆ (context) ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಲೋಡ್ ಮಾಡಲಾಗುತ್ತದೆ. ನೀವು ಸ್ಕಿಲ್ಸ್‌ಗಳನ್ನು ಕೈಯಾರೆ ಆನ್ ಅಥವಾ ಸ್ವಿಚ್ ಮಾಡುವ ಅಗತ್ಯವಿಲ್ಲ.

### ಬಹು ಸ್ಕಿಲ್ಸ್ (Multiple Skills)

ನಿಮ್ಮ ಸೈಟ್‌ಗೆ ಅನೇಕ ಸ್ಕಿಲ್ಸ್‌ಗಳು ಅನ್ವಯಿಸಿದರೆ (ಉದಾಹರಣೆಗೆ, ನಿಮ್ಮ ಬಳಿ Kadence Blocks ಮತ್ತು Kadence Theme ಎರಡೂ ಸಕ್ರಿಯವಾಗಿದ್ದರೆ), ಅಸಿಸ್ಟೆಂಟ್‌ಗೆ ಎಲ್ಲಾ ಅನ್ವಯಿಸುವ ಸ್ಕಿಲ್ಸ್‌ಗಳ ಪ್ರವೇಶವಿರುತ್ತದೆ ಮತ್ತು ಪ್ರತಿಯೊಂದರಿಂದಲೂ ಮಾರ್ಗದರ್ಶನವನ್ನು ಉಲ್ಲೇಖಿಸಬಹುದು.

### ಥೀಮ್ ಬದಲಾಯಿಸುವಾಗ (Switching Themes)

ನೀವು ನಿಮ್ಮ ಸಕ್ರಿಯ ಥೀಮ್ ಬದಲಾಯಿಸಿದಾಗ, ಮುಂದಿನ ಸಂದೇಶದೊಂದಿಗೆ ಅಸಿಸ್ಟೆಂಟ್‌ನ ಲಭ್ಯವಿರುವ ಸ್ಕಿಲ್ಸ್‌ಗಳು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಅಪ್ಡೇಟ್ ಆಗುತ್ತವೆ. ಉದಾಹರಣೆಗೆ:

1. ನೀವು **Block Themes** ಸ್ಕಿಲ್ ಸಕ್ರಿಯವಾಗಿರುವ ಬ್ಲಾಕ್ ಥೀಮ್ ಅನ್ನು ಬಳಸುತ್ತಿದ್ದೀರಿ.
2. ನೀವು ಕ್ಲಾಸಿಕ್ ಥೀಮ್‌ಗೆ ಬದಲಾಯಿಸುತ್ತೀರಿ.
3. ನಿಮ್ಮ ಮುಂದಿನ ಸಂದೇಶದಲ್ಲಿ, **Classic Themes** ಸ್ಕಿಲ್ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಲೋಡ್ ಆಗುತ್ತದೆ ಮತ್ತು **Block Themes** ಸ್ಕಿಲ್ ಲಭ್ಯವಿರುವುದಿಲ್ಲ.

## ಥೀಮ್-ಅವೇರ್ ಸ್ಕಿಲ್ಸ್ ಬಳಸುವುದು

ಥೀಮ್-ಅವೇರ್ ಸ್ಕಿಲ್ ಅನ್ನು ಬಳಸಿಕೊಳ್ಳಲು, ನೀವು ಚಾಟ್ ಇಂಟರ್‌ಫೇಸ್‌ನಲ್ಲಿ ಏನು ಮಾಡಲು ಬಯಸುತ್ತೀರಿ ಎಂಬುದನ್ನು ವಿವರಿಸಿ. ಅಸಿಸ್ಟೆಂಟ್ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸೂಕ್ತ ಸ್ಕಿಲ್‌ನ ಮಾರ್ಗದರ್ಶನವನ್ನು ಉಲ್ಲೇಖಿಸುತ್ತದೆ.

### ಉದಾಹರಣೆ ಪ್ರಾಂಪ್ಟ್‌ಗಳು (Example Prompts)

**ಬ್ಲಾಕ್ ಥೀಮ್ಸ್‌ಗಾಗಿ:**
> "ಬ್ಲಾಕ್ ಪ್ಯಾಟರ್ನ್‌ಗಳನ್ನು ಬಳಸಿ ಹಿನ್ನೆಲೆ ಚಿತ್ರ ಮತ್ತು ಮಧ್ಯದಲ್ಲಿರುವ ಪಠ್ಯದೊಂದಿಗೆ ಒಂದು ಹೀರೋ ಸೆಕ್ಷನ್ ಅನ್ನು ರಚಿಸಿ."

**ಕ್ಲಾಸಿಕ್ ಥೀಮ್ಸ್‌ಗಾಗಿ:**
> "child theme ಬಳಸಿ ಸೈಡ್‌ಬಾರ್‌ನಲ್ಲಿ ಕಸ್ಟಮ್ ವಿಜೆಟ್ ಏರಿಯಾವನ್ನು ಸೇರಿಸಿ."

**Kadence Blocks ಗಾಗಿ:**
