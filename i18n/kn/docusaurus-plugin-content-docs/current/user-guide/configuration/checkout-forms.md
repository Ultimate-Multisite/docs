# Checkout Forms

Checkout Forms ಹೊಸ ಗ್ರಾಹಕರನ್ನು ಪರಿವರ್ತಿಸಲು ವಿವಿಧ ವಿಧಾನಗಳನ್ನು ಪ್ರಯೋಗಿಸಲು ಸುಲಭ ಮತ್ತು ಹೊಂದಿಕೊಳ್ಳುವ ಮಾರ್ಗವಾಗಿದೆ.

Ultimate Multisite 2.0 ನಲ್ಲಿ Checkout Form ಎಡಿಟರ್ ಇದೆ, ಇದು ನಿಮಗೆ ಬೇಕಾದಷ್ಟು ಫಾರ್ಮ್‌ಗಳನ್ನು ವಿವಿಧ ಫೀಲ್ಡ್‌ಗಳು, ಆಫರ್‌ನಲ್ಲಿರುವ ಉತ್ಪನ್ನಗಳು ಇತ್ಯಾದಿಗಳೊಂದಿಗೆ ರಚಿಸಲು ಅನುಮತಿಸುತ್ತದೆ.

ಈ ವೈಶಿಷ್ಟ್ಯವನ್ನು ಪ್ರವೇಶಿಸಲು, ಎಡ ಸೈಡ್-ಬಾರ್‌ನಲ್ಲಿರುವ Checkout Forms ಮೆನುಗೆ ಹೋಗಿ.

![Checkout Forms ಪಟ್ಟಿ](/img/config/checkout-forms-list.png)

ಈ ಪುಟದಲ್ಲಿ, ನಿಮ್ಮ ಬಳಿ ಇರುವ ಎಲ್ಲಾ checkout forms ಅನ್ನು ನೋಡಬಹುದು.

ಹೊಸದನ್ನು ರಚಿಸಲು ಬಯಸಿದರೆ, ಪುಟದ ಮೇಲ್ಭಾಗದಲ್ಲಿರುವ Add Checkout Form ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ.

ನಿಮ್ಮ ಆರಂಭಿಕ ಹಂತವಾಗಿ ಈ ಮೂರು ಆಯ್ಕೆಗಳಲ್ಲಿ ಒಂದನ್ನು ಆರಿಸಬಹುದು: single step, multi-step ಅಥವಾ blank. ನಂತರ, Go to the Editor ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ.

![Checkout Form ಎಡಿಟರ್](/img/config/checkout-form-editor.png)

ಪರ್ಯಾಯವಾಗಿ, ನಿಮ್ಮ ಬಳಿ ಈಗಾಗಲೇ ಇರುವ ಫಾರ್ಮ್‌ಗಳನ್ನು ಅದರ ಹೆಸರಿನ ಕೆಳಗಿರುವ ಆಯ್ಕೆಗಳ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡುವ ಮೂಲಕ ಎಡಿಟ್ ಅಥವಾ ಡುಪ್ಲಿಕೇಟ್ ಮಾಡಬಹುದು. ಅಲ್ಲಿ, ಫಾರ್ಮ್‌ನ shortcode ಕಾಪಿ ಮಾಡಲು ಅಥವಾ ಫಾರ್ಮ್ ಅನ್ನು ಡಿಲೀಟ್ ಮಾಡಲು ಆಯ್ಕೆಗಳನ್ನು ಕಾಣಬಹುದು.

![Checkout form ಹೋವರ್ ಆಕ್ಷನ್‌ಗಳು](/img/config/checkout-form-hover-actions.png)

### Checkout Form ಅನ್ನು ಎಡಿಟ್ ಮಾಡುವುದು

ವಿವಿಧ ಉದ್ದೇಶಗಳಿಗಾಗಿ checkout forms ರಚಿಸಬಹುದು. ಈ ಉದಾಹರಣೆಯಲ್ಲಿ ನಾವು ನೋಂದಣಿ ಫಾರ್ಮ್ ಮೇಲೆ ಕೆಲಸ ಮಾಡುತ್ತೇವೆ.

Checkout form ಎಡಿಟರ್‌ಗೆ ನ್ಯಾವಿಗೇಟ್ ಮಾಡಿದ ನಂತರ, ನಿಮ್ಮ ಫಾರ್ಮ್‌ಗೆ ಒಂದು ಹೆಸರು (ಇದು ಆಂತರಿಕ ಉಲ್ಲೇಖಕ್ಕಾಗಿ ಮಾತ್ರ ಬಳಸಲಾಗುತ್ತದೆ) ಮತ್ತು ಒಂದು slug (shortcodes ರಚಿಸಲು ಬಳಸಲಾಗುತ್ತದೆ, ಉದಾಹರಣೆಗೆ) ನೀಡಿ.

![Checkout Form ಎಡಿಟರ್](/img/config/checkout-form-editor.png)

ಫಾರ್ಮ್‌ಗಳು steps ಮತ್ತು fields ಗಳಿಂದ ಮಾಡಲ್ಪಟ್ಟಿವೆ. Add New Checkout Step ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡುವ ಮೂಲಕ ಹೊಸ step ಸೇರಿಸಬಹುದು.

![Add New Checkout Step](/img/config/checkout-form-add-step.png)

ಮೋಡಲ್ ವಿಂಡೋದ ಮೊದಲ ಟ್ಯಾಬ್‌ನಲ್ಲಿ, ನಿಮ್ಮ ಫಾರ್ಮ್‌ನ step ನ ವಿಷಯವನ್ನು ಭರ್ತಿ ಮಾಡಿ. ಅದಕ್ಕೆ ಒಂದು ID, ಹೆಸರು ಮತ್ತು ವಿವರಣೆ ನೀಡಿ. ಈ ಐಟಂಗಳು ಹೆಚ್ಚಾಗಿ ಆಂತರಿಕವಾಗಿ ಬಳಸಲ್ಪಡುತ್ತವೆ.

![Checkout form step](/img/config/checkout-form-step.png)

ಮುಂದೆ, step ನ ವಿಸಿಬಿಲಿಟಿಯನ್ನು ಹೊಂದಿಸಿ. "Always show", "Only show for logged in users" ಅಥವಾ "Only show for guests" ಮಧ್ಯೆ ಆಯ್ಕೆ ಮಾಡಬಹುದು.

![Checkout form step](/img/config/checkout-form-step.png)

ಕೊನೆಯದಾಗಿ, step ಸ್ಟೈಲ್ ಅನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಿ. ಇವು ಐಚ್ಛಿಕ ಫೀಲ್ಡ್‌ಗಳು.

![Checkout form step](/img/config/checkout-form-step.png)

ಈಗ, ನಮ್ಮ ಮೊದಲ step ಗೆ fields ಸೇರಿಸುವ ಸಮಯ. Add New Field ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ನಿಮಗೆ ಬೇಕಾದ section ನ ಪ್ರಕಾರವನ್ನು ಆಯ್ಕೆ ಮಾಡಿ.

![Fields ಇರುವ Checkout form step](/img/config/checkout-form-step.png)

ಪ್ರತಿ field ಭರ್ತಿ ಮಾಡಬೇಕಾದ ವಿವಿಧ ಪ್ಯಾರಾಮೀಟರ್‌ಗಳನ್ನು ಹೊಂದಿದೆ. ಈ ಮೊದಲ ಎಂಟ್ರಿಗೆ, ನಾವು "Username" field ಅನ್ನು ಆಯ್ಕೆ ಮಾಡುತ್ತೇವೆ.

![Checkout form step](/img/config/checkout-form-step.png)

![Checkout form step](/img/config/checkout-form-step.png)

![Checkout form step](/img/config/checkout-form-step.png)

ನಿಮಗೆ ಬೇಕಾದಷ್ಟು steps ಮತ್ತು fields ಸೇರಿಸಬಹುದು. ನಿಮ್ಮ ಗ್ರಾಹಕರು ಆಯ್ಕೆ ಮಾಡಲು ನಿಮ್ಮ ಉತ್ಪನ್ನಗಳನ್ನು ಪ್ರದರ್ಶಿಸಲು, Pricing Table field ಬಳಸಿ. ನಿಮ್ಮ ಕ್ಲೈಂಟ್‌ಗಳಿಗೆ ಒಂದು ಟೆಂಪ್ಲೇಟ್ ಆಯ್ಕೆ ಮಾಡಲು ಅವಕಾಶ ನೀಡಲು ಬಯಸಿದರೆ, Template Selection field ಸೇರಿಸಿ. ಹೀಗೆ ಮುಂದುವರಿಯಿರಿ.

_**ಗಮನಿಸಿ:** username, email, password, site title, site URL, order summary, payment, ಮತ್ತು submit button ಗಳು checkout form ರಚಿಸಲು ಕಡ್ಡಾಯ fields ಆಗಿವೆ._

ನಿಮ್ಮ checkout form ಮೇಲೆ ಕೆಲಸ ಮಾಡುವಾಗ, ನಿಮ್ಮ ಕ್ಲೈಂಟ್‌ಗಳಿಗೆ ಫಾರ್ಮ್ ಹೇಗೆ ಕಾಣಿಸುತ್ತದೆ ಎಂದು ನೋಡಲು ಯಾವಾಗಲೂ Preview ಬಟನ್ ಬಳಸಬಹುದು. ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಬಳಕೆದಾರ ಅಥವಾ ಸಂದರ್ಶಕನಾಗಿ ವೀಕ್ಷಣೆ ನಡುವೆ ಬದಲಾಯಿಸಬಹುದು.

![Checkout Form save](/img/config/checkout-form-save.png)

![Checkout Form ಎಡಿಟರ್](/img/config/checkout-form-editor.png)

ಕೊನೆಯದಾಗಿ, Advanced Options ನಲ್ಲಿ "Thank You" ಪುಟಕ್ಕೆ ಸಂದೇಶವನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಬಹುದು, conversions ಟ್ರ್ಯಾಕ್ ಮಾಡಲು snippets ಸೇರಿಸಬಹುದು, ನಿಮ್ಮ checkout form ಗೆ custom CSS ಸೇರಿಸಬಹುದು ಅಥವಾ ಕೆಲವು ದೇಶಗಳಿಗೆ ಮಾತ್ರ ನಿರ್ಬಂಧಿಸಬಹುದು.

![Advanced Options](/img/config/checkout-form-advanced.png)

ಬಲ ಕಾಲಮ್‌ನಲ್ಲಿರುವ ಈ ಆಯ್ಕೆಯನ್ನು ಟಾಗಲ್ ಮಾಡುವ ಮೂಲಕ ನಿಮ್ಮ checkout form ಅನ್ನು ಹಸ್ತಚಾಲಿತವಾಗಿ ಎನೇಬಲ್ ಅಥವಾ ಡಿಸೇಬಲ್ ಮಾಡಬಹುದು, ಅಥವಾ ಫಾರ್ಮ್ ಅನ್ನು ಶಾಶ್ವತವಾಗಿ ಡಿಲೀಟ್ ಮಾಡಬಹುದು.

![Active toggle](/img/config/checkout-form-active.png)

ನಿಮ್ಮ checkout form ಅನ್ನು save ಮಾಡಲು ಮರೆಯಬೇಡಿ!

![Save ಬಟನ್](/img/config/checkout-form-save.png)

ನಿಮ್ಮ ಫಾರ್ಮ್‌ನ shortcode ಪಡೆಯಲು Generate Shortcode ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ಮೋಡಲ್ ವಿಂಡೋದಲ್ಲಿ ತೋರಿಸಿದ ಫಲಿತಾಂಶವನ್ನು ಕಾಪಿ ಮಾಡಿ.

![Shortcode ಇರುವ Save ಬಟನ್](/img/config/checkout-form-save.png)
