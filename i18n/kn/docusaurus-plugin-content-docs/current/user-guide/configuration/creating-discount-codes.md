# ಡಿಸ್ಕೌಂಟ್ ಕೋಡ್‌ಗಳನ್ನು ರಚಿಸುವುದು (v2)

_**ಮುಖ್ಯ ಸೂಚನೆ: ಈ ಲೇಖನವು Ultimate Multisite ಆವೃತ್ತಿ 2.x ಗೆ ಸಂಬಂಧಿಸಿದೆ.**_

Ultimate Multisite ಬಳಸಿ ನಿಮ್ಮ ಗ್ರಾಹಕರಿಗೆ ಅವರ ಚಂದಾದಾರಿಕೆಗಳ ಮೇಲೆ ರಿಯಾಯಿತಿ ನೀಡಲು ಡಿಸ್ಕೌಂಟ್ ಕೋಡ್‌ಗಳನ್ನು ರಚಿಸಬಹುದು. ಇದನ್ನು ರಚಿಸುವುದು ತುಂಬಾ ಸುಲಭ!

## ಡಿಸ್ಕೌಂಟ್ ಕೋಡ್‌ಗಳನ್ನು ರಚಿಸುವುದು ಮತ್ತು ಎಡಿಟ್ ಮಾಡುವುದು

ಡಿಸ್ಕೌಂಟ್ ಕೋಡ್ ರಚಿಸಲು ಅಥವಾ ಎಡಿಟ್ ಮಾಡಲು, **Ultimate Multisite > Discount Codes** ಗೆ ಹೋಗಿ.

![](/img/config/discount-codes-list.png)

ಅಲ್ಲಿ ನೀವು ಈಗಾಗಲೇ ರಚಿಸಿರುವ ಡಿಸ್ಕೌಂಟ್ ಕೋಡ್‌ಗಳ ಪಟ್ಟಿಯನ್ನು ನೋಡಬಹುದು.

ಹೊಸ ಕೂಪನ್ ರಚಿಸಲು **Add Discount** **Code** ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಬಹುದು ಅಥವಾ ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಕೋಡ್‌ಗಳ ಮೇಲೆ ಹೋವರ್ ಮಾಡಿ **Edit** ಕ್ಲಿಕ್ ಮಾಡಿ ಎಡಿಟ್ ಮಾಡಬಹುದು.

![ಹೋವರ್ ಆಯ್ಕೆಗಳೊಂದಿಗೆ ಡಿಸ್ಕೌಂಟ್ ಕೋಡ್‌ಗಳ ಪಟ್ಟಿ](/img/config/discount-codes-list.png)

![Add Discount Code ಬಟನ್](/img/config/discount-codes-list.png)

ನಿಮ್ಮ ಕೂಪನ್ ಕೋಡ್ ರಚಿಸಲು ಅಥವಾ ಎಡಿಟ್ ಮಾಡಲು ಪುಟಕ್ಕೆ ರೀಡೈರೆಕ್ಟ್ ಆಗುತ್ತೀರಿ. ಈ ಉದಾಹರಣೆಯಲ್ಲಿ ನಾವು ಹೊಸದನ್ನು ರಚಿಸುತ್ತೇವೆ.

![ಡಿಸ್ಕೌಂಟ್ ಕೋಡ್ ಎಡಿಟ್ ಪುಟ](/img/config/discount-codes-list.png)

ಇಲ್ಲಿ ಲಭ್ಯವಿರುವ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ನೋಡೋಣ:

**Enter Discount Code:** ಇದು ನಿಮ್ಮ ಡಿಸ್ಕೌಂಟ್ ಕೋಡ್‌ನ ಹೆಸರು ಮಾತ್ರ. ನಿಮ್ಮ ಗ್ರಾಹಕರು ಚೆಕ್‌ಔಟ್ ಫಾರ್ಮ್‌ನಲ್ಲಿ ಬಳಸಬೇಕಾದ ಕೋಡ್ ಇದಲ್ಲ.

**Description:** ಇಲ್ಲಿ, ಈ ಕೂಪನ್ ಯಾವುದಕ್ಕಾಗಿ ಎಂದು ಸಂಕ್ಷಿಪ್ತವಾಗಿ ವಿವರಿಸಬಹುದು.

![ಡಿಸ್ಕೌಂಟ್ ಕೋಡ್ ಹೆಸರು ಮತ್ತು ವಿವರಣೆ ಫೀಲ್ಡ್‌ಗಳು](/img/config/discount-codes-list.png)

**Coupon code:** ನಿಮ್ಮ ಗ್ರಾಹಕರು ಚೆಕ್‌ಔಟ್ ಸಮಯದಲ್ಲಿ ನಮೂದಿಸಬೇಕಾದ ಕೋಡ್ ಅನ್ನು ಇಲ್ಲಿ ವ್ಯಾಖ್ಯಾನಿಸುತ್ತೀರಿ.

![ಕೂಪನ್ ಕೋಡ್ ಫೀಲ್ಡ್](/img/config/discount-codes-list.png)

**Discount:** ಇಲ್ಲಿ, ನಿಮ್ಮ ಡಿಸ್ಕೌಂಟ್ ಕೋಡ್‌ಗೆ **ಶೇಕಡಾವಾರು** ಅಥವಾ **ನಿಶ್ಚಿತ ಮೊತ್ತ** ಹೊಂದಿಸಬಹುದು.

![ಡಿಸ್ಕೌಂಟ್ ಶೇಕಡಾವಾರು ಅಥವಾ ನಿಶ್ಚಿತ ಮೊತ್ತ ಸೆಟ್ಟಿಂಗ್](/img/config/discount-codes-list.png)

**Apply to renewals:** ಈ ಆಯ್ಕೆಯನ್ನು ಆಫ್ ಮಾಡಿದರೆ, ಈ ಡಿಸ್ಕೌಂಟ್ ಕೋಡ್ **ಮೊದಲ ಪಾವತಿಗೆ** ಮಾತ್ರ ಅನ್ವಯವಾಗುತ್ತದೆ. ಇತರ ಎಲ್ಲಾ ಪಾವತಿಗಳಿಗೆ ಯಾವುದೇ ರಿಯಾಯಿತಿ ಇರುವುದಿಲ್ಲ. ಈ ಆಯ್ಕೆಯನ್ನು ಆನ್ ಮಾಡಿದರೆ, ಡಿಸ್ಕೌಂಟ್ ಕೋಡ್ ಭವಿಷ್ಯದ ಎಲ್ಲಾ ಪಾವತಿಗಳಿಗೆ ಮಾನ್ಯವಾಗಿರುತ್ತದೆ.

**Setup fee discount:** ಈ ಆಯ್ಕೆಯನ್ನು ಆಫ್ ಮಾಡಿದರೆ, ಕೂಪನ್ ಕೋಡ್ ಆರ್ಡರ್‌ನ **ಸೆಟಪ್ ಫೀಗೆ ಯಾವುದೇ ರಿಯಾಯಿತಿ ನೀಡುವುದಿಲ್ಲ**. ಈ ಆಯ್ಕೆಯನ್ನು ಆನ್ ಮಾಡಿದರೆ, ಈ ಕೂಪನ್ ಕೋಡ್ ನಿಮ್ಮ ಪ್ಲಾನ್‌ಗಳ ಸೆಟಪ್ ಫೀಗೆ ಅನ್ವಯಿಸುವ ರಿಯಾಯಿತಿಯನ್ನು (ಶೇಕಡಾವಾರು ಅಥವಾ ನಿಶ್ಚಿತ ಮೊತ್ತ) ಹೊಂದಿಸಬಹುದು.

![Apply to renewals ಮತ್ತು setup fee discount ಆಯ್ಕೆಗಳು](/img/config/discount-codes-list.png)

**Active:** ಈ ಕೂಪನ್ ಕೋಡ್ ಅನ್ನು ಹಸ್ತಚಾಲಿತವಾಗಿ ಸಕ್ರಿಯಗೊಳಿಸಿ ಅಥವಾ ನಿಷ್ಕ್ರಿಯಗೊಳಿಸಿ.

![ಡಿಸ್ಕೌಂಟ್ ಕೋಡ್‌ಗಾಗಿ Active ಟಾಗಲ್](/img/config/discount-codes-list.png)

**Advanced Optio** **ns** ಅಡಿಯಲ್ಲಿ, ಈ ಕೆಳಗಿನ ಸೆಟ್ಟಿಂಗ್‌ಗಳಿವೆ:

**Limit uses:**

  * **Uses:** ಡಿಸ್ಕೌಂಟ್ ಕೋಡ್ ಎಷ್ಟು ಬಾರಿ ಬಳಸಲಾಗಿದೆ ಎಂದು ಇಲ್ಲಿ ನೋಡಬಹುದು.

  * **Max uses:** ಬಳಕೆದಾರರು ಈ ಡಿಸ್ಕೌಂಟ್ ಕೋಡ್ ಅನ್ನು ಎಷ್ಟು ಬಾರಿ ಬಳಸಬಹುದು ಎಂಬುದನ್ನು ಮಿತಿಗೊಳಿಸುತ್ತದೆ. ಉದಾಹರಣೆಗೆ, ಇಲ್ಲಿ 10 ಹಾಕಿದರೆ, ಕೂಪನ್ ಅನ್ನು 10 ಬಾರಿ ಮಾತ್ರ ಬಳಸಬಹುದು. ಈ ಮಿತಿಯ ನಂತರ, ಕೂಪನ್ ಕೋಡ್ ಇನ್ನು ಮುಂದೆ ಬಳಸಲು ಸಾಧ್ಯವಿಲ್ಲ.

![Uses ಮತ್ತು max uses ಫೀಲ್ಡ್‌ಗಳೊಂದಿಗೆ Limit uses ಸೆಟ್ಟಿಂಗ್](/img/config/discount-codes-list.png)**Start & expiration dates:** ನಿಮ್ಮ ಕೂಪನ್‌ಗೆ ಆರಂಭ ದಿನಾಂಕ ಮತ್ತು/ಅಥವಾ ಅವಧಿ ಮುಕ್ತಾಯ ದಿನಾಂಕ ಸೇರಿಸುವ ಆಯ್ಕೆ ಇಲ್ಲಿ ಇದೆ.

![ಆರಂಭ ಮತ್ತು ಅವಧಿ ಮುಕ್ತಾಯ ದಿನಾಂಕ ಫೀಲ್ಡ್‌ಗಳು](/img/config/discount-codes-list.png)

**Limit products:** **Select products** ಅನ್ನು ಆನ್ ಮಾಡಿದರೆ, ನಿಮ್ಮ ಎಲ್ಲಾ ಉತ್ಪನ್ನಗಳು ತೋರಿಸಲ್ಪಡುತ್ತವೆ. ಯಾವ ಉತ್ಪನ್ನ ಈ ಕೂಪನ್ ಕೋಡ್ ಅನ್ನು ಸ್ವೀಕರಿಸಬಹುದು ಎಂಬುದನ್ನು ಹಸ್ತಚಾಲಿತವಾಗಿ ಆಯ್ಕೆ ಮಾಡಬಹುದು (ಆನ್ ಅಥವಾ ಆಫ್ ಮಾಡುವ ಮೂಲಕ). ಇಲ್ಲಿ ಆಫ್ ಮಾಡಲಾದ ಉತ್ಪನ್ನಗಳಿಗೆ ನಿಮ್ಮ ಗ್ರಾಹಕರು ಈ ಕೂಪನ್ ಕೋಡ್ ಬಳಸಲು ಪ್ರಯತ್ನಿಸಿದರೆ ಯಾವುದೇ ಬದಲಾವಣೆ ಕಾಣಿಸುವುದಿಲ್ಲ.

![Limit products ಆಯ್ಕೆ ಟಾಗಲ್‌ಗಳು](/img/config/discount-codes-list.png)

ಈ ಎಲ್ಲಾ ಆಯ್ಕೆಗಳನ್ನು ಹೊಂದಿಸಿದ ನಂತರ, ನಿಮ್ಮ ಕೂಪನ್ ಉಳಿಸಲು **Save Discount Code** ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ಮುಗಿಯಿತು!

![Save Discount Code ಬಟನ್](/img/config/discount-codes-list.png)

ಕೂಪನ್ ಈಗ ನಿಮ್ಮ ಪಟ್ಟಿಯಲ್ಲಿದೆ ಮತ್ತು ಅಲ್ಲಿಂದ, ಅದನ್ನು **ಎಡಿಟ್ ಅಥವಾ ಡಿಲೀಟ್** ಮಾಡಲು ಕ್ಲಿಕ್ ಮಾಡಬಹುದು.

![ಎಡಿಟ್ ಮತ್ತು ಡಿಲೀಟ್ ಆಯ್ಕೆಗಳೊಂದಿಗೆ ಪಟ್ಟಿಯಲ್ಲಿ ಡಿಸ್ಕೌಂಟ್ ಕೋಡ್](/img/config/discount-codes-list.png)

### URL ಪ್ಯಾರಾಮೀಟರ್‌ಗಳನ್ನು ಬಳಸುವುದು:

ನಿಮ್ಮ ಬೆಲೆ ಟೇಬಲ್‌ಗಳನ್ನು ಕಸ್ಟಮೈಸ್ ಮಾಡಲು ಅಥವಾ ನಿಮ್ಮ ವೆಬ್‌ಸೈಟ್‌ಗಾಗಿ ಉತ್ತಮ ಕೂಪನ್ ಕೋಡ್ ಪುಟವನ್ನು ನಿರ್ಮಿಸಲು ಮತ್ತು ನಿಮ್ಮ ಚೆಕ್‌ಔಟ್ ಫಾರ್ಮ್‌ಗೆ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಡಿಸ್ಕೌಂಟ್ ಕೋಡ್ ಅನ್ವಯಿಸಲು ಬಯಸಿದರೆ, URL ಪ್ಯಾರಾಮೀಟರ್‌ಗಳ ಮೂಲಕ ಇದನ್ನು ಮಾಡಬಹುದು.

ಮೊದಲು, ನಿಮ್ಮ ಪ್ಲಾನ್‌ಗಾಗಿ ಹಂಚಿಕೊಳ್ಳಬಹುದಾದ ಲಿಂಕ್ ಪಡೆಯಬೇಕು. ಇದನ್ನು ಮಾಡಲು, **Ultimate Multisite > Products** ಗೆ ಹೋಗಿ ಮತ್ತು ಪ್ಲಾನ್ ಆಯ್ಕೆ ಮಾಡಿ.

**Click to Copy Shareable Link** ಬಟನ್ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ. ಇದು ಈ ನಿರ್ದಿಷ್ಟ ಪ್ಲಾನ್‌ಗೆ ಹಂಚಿಕೊಳ್ಳಬಹುದಾದ ಲಿಂಕ್ ನೀಡುತ್ತದೆ. ನಮ್ಮ ಸಂದರ್ಭದಲ್ಲಿ, ನೀಡಲಾದ ಹಂಚಿಕೊಳ್ಳಬಹುದಾದ ಲಿಂಕ್ [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/)_._

![ಹಂಚಿಕೊಳ್ಳಬಹುದಾದ ಲಿಂಕ್ ಬಟನ್ ಹೊಂದಿರುವ ಉತ್ಪನ್ನ ಪುಟ](/img/config/products-list.png)

ಈ ನಿರ್ದಿಷ್ಟ ಪ್ಲಾನ್‌ಗೆ ನಿಮ್ಮ ಡಿಸ್ಕೌಂಟ್ ಕೋಡ್ ಅನ್ವಯಿಸಲು, URL ಗೆ **?discount_code=XXX** ಪ್ಯಾರಾಮೀಟರ್ ಸೇರಿಸಿ. ಇಲ್ಲಿ **XXX** ಎಂಬುದು ಕೂಪನ್ ಕೋಡ್.

ನಮ್ಮ ಈ ಉದಾಹರಣೆಯಲ್ಲಿ, ಈ ನಿರ್ದಿಷ್ಟ ಉತ್ಪನ್ನಕ್ಕೆ **50OFF** ಕೂಪನ್ ಕೋಡ್ ಅನ್ವಯಿಸುತ್ತೇವೆ.

ಈ ನಿರ್ದಿಷ್ಟ ಪ್ಲಾನ್ ಮತ್ತು 50OFF ಡಿಸ್ಕೌಂಟ್ ಕೋಡ್ ಅನ್ವಯಿಸಿದ URL ಹೀಗೆ ಕಾಣುತ್ತದೆ: [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/) _**?discount_code=50OFF**_.
