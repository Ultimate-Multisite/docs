# ಇಂಟರ್ನೆಟ್ ಸರ್ಚ್

Gratis AI Agent v1.5.0 ಒಂದು **ಇಂಟರ್ನೆಟ್ ಸರ್ಚ್** ಸಾಮರ್ಥ್ಯವನ್ನು ಸೇರಿಸಿದೆ. ಇದು AI ಅಸಿಸ್ಟೆಂಟ್‌ಗೆ ಸಂಭಾಷಣೆಯ ಸಮಯದಲ್ಲಿ ವೆಬ್‌ನಿಂದ ನವೀಕೃತ ಮಾಹಿತಿಯನ್ನು ಪಡೆಯಲು ಸಹಾಯ ಮಾಡುತ್ತದೆ. ಇದು [Brave Search API](https://brave.com/search/api/) ಅಥವಾ [Tavily API](https://tavily.com/) ಮೂಲಕ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ.

## ಇದು ಹೇಗೆ ಕೆಲಸ ಮಾಡುತ್ತದೆ

ಇಂಟರ್ನೆಟ್ ಸರ್ಚ್ ಅನ್ನು ಆನ್ ಮಾಡಿದಾಗ, ಪ್ರಶ್ನೆಯು ಪ್ರಸ್ತುತ ಅಥವಾ ಬಾಹ್ಯ ಮಾಹಿತಿಯನ್ನು ಬಯಸುತ್ತದೆ ಎಂದು ಅಸಿಸ್ಟೆಂಟ್ ನಿರ್ಧರಿಸಿದಾಗ, ಅದು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ನಿಮ್ಮ ಕಾನ್ಫಿಗರ್ ಮಾಡಿದ ಸರ್ಚ್ ಪ್ರೊವೈಡರ್‌ಗೆ ಪ್ರಶ್ನಿಸುತ್ತದೆ — ಉದಾಹರಣೆಗೆ, ಇತ್ತೀಚಿನ ಸುದ್ದಿ, ಲೈವ್ ಬೆಲೆಗಳು, ಅಥವಾ ಮಾದರಿಯ ತರಬೇತಿ ಕಡಿತದ ನಂತರ ಬದಲಾಗಿರುವ ದಾಖಲೆಗಳು.

ಫಲಿತಾಂಶಗಳನ್ನು ನೈಜ ಸಮಯದಲ್ಲಿ ಪಡೆದು, ಅದು ಪ್ರತಿಕ್ರಿಯೆಯನ್ನು ರಚಿಸುವ ಮೊದಲು ಅಸಿಸ್ಟೆಂಟ್‌ನ ಸಂದರ್ಭಕ್ಕೆ (context) ಸೇರಿಸಲಾಗುತ್ತದೆ. ಸರ್ಚ್ ಫಲಿತಾಂಶಗಳನ್ನು ಬಳಸಿಕೊಂಡು ಪ್ರಶ್ನೆಗೆ ಉತ್ತರಿಸಿದಾಗ ಅಸಿಸ್ಟೆಂಟ್ ಅದನ್ನು ಸೂಚಿಸುತ್ತದೆ.

## ಇಂಟರ್ನೆಟ್ ಸರ್ಚ್ ಅನ್ನು ಆನ್ ಮಾಡುವುದು

ಇಂಟರ್ನೆಟ್ ಸರ್ಚ್‌ಗೆ ನಿಮ್ಮ ಆಯ್ಕೆಯ ಸರ್ಚ್ ಪ್ರೊವೈಡರ್‌ನಿಂದ ಒಂದು API ಕೀ ಬೇಕಾಗುತ್ತದೆ. ಇದನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಲು:

1. **Gratis AI Agent → Settings → Advanced** ಗೆ ಹೋಗಿ.
2. **Internet Search Provider** ಡ್ರಾಪ್‌ಡೌನ್ ಆಯ್ಕೆಯನ್ನು ಹುಡುಕಿ ಮತ್ತು **Brave Search** ಅಥವಾ **Tavily** ಅನ್ನು ಆಯ್ಕೆಮಾಡಿ.
3. ಸಂಬಂಧಪಟ್ಟ ಫೀಲ್ಡ್‌ನಲ್ಲಿ ನಿಮ್ಮ API ಕೀ ಅನ್ನು ನಮೂದಿಸಿ. ಪ್ರತಿ ಫೀಲ್ಡ್ ಪಕ್ಕದಲ್ಲಿ ನೋಂದಣಿ (Sign-up) ಲಿಂಕ್‌ಗಳು ಕಾಣಿಸುತ್ತವೆ.
4. **Save Settings** ಕ್ಲಿಕ್ ಮಾಡಿ.

ಕೀ ಅನ್ನು ಉಳಿಸಿದ ನಂತರ, ಇಂಟರ್ನೆಟ್ ಸರ್ಚ್ ಸಾಮರ್ಥ್ಯವು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಅಸಿಸ್ಟೆಂಟ್‌ಗೆ ಲಭ್ಯವಾಗುತ್ತದೆ.

## Brave Search

### Brave Search API ಕೀ ಪಡೆಯುವುದು

1. [Brave Search API page](https://brave.com/search/api/) ಗೆ ಭೇಟಿ ನೀಡಿ.
2. ಒಂದು ಯೋಜನೆಯಲ್ಲಿ (plan) ನೋಂದಾಯಿಸಿ. ಮಾಸಿಕ ವಿನಂತಿ ಮಿತಿಯೊಂದಿಗೆ ಉಚಿತ ಟಿಯರ್ ಲಭ್ಯವಿದೆ.
3. Brave Search developer ಡ್ಯಾಶ್‌ಬೋರ್ಡ್‌ನಿಂದ ನಿಮ್ಮ API ಕೀ ಅನ್ನು ನಕಲಿಸಿ.
4. ಅದನ್ನು Gratis AI Agent ಸೆಟ್ಟಿಂಗ್‌ಗಳಲ್ಲಿರುವ **Brave Search API Key** ಫೀಲ್ಡ್‌ನಲ್ಲಿ ಅಂಟಿಸಿ.

### ಬಳಕೆಯ ಮಿತಿಗಳು

ಬಳಕೆಯು ಮಾಡಿದ ಪ್ರಶ್ನೆಗಳ ಸಂಖ್ಯೆಯ ಆಧಾರದ ಮೇಲೆ Brave Search ನಿಂದ ಬಿಲ್ ಮಾಡಲಾಗುತ್ತದೆ. ಸರ್ಚ್ ಅನ್ನು déclencher ಮಾಡುವ ಪ್ರತಿ AI ಪ್ರತಿಕ್ರಿಯೆಯು ಒಂದು ಪ್ರಶ್ನೆಯಂತೆ (query) ગಣಿಕೆಯಾಗುತ್ತದೆ. ಅನಿರೀಕ್ಷಿತ ಶುಲ್ಕಗಳನ್ನು ತಪ್ಪಿಸಲು [Brave Search developer dashboard](https://brave.com/search/api/) ನಲ್ಲಿ ನಿಮ್ಮ ಬಳಕೆಯನ್ನು ಮೇಲ್ವಿಚಾರಣೆ ಮಾಡಿ.

## Tavily

Superdav AI Agent v1.10.0 ಒಂದು ಪರ್ಯಾಯ ಇಂಟರ್ನೆಟ್ ಸರ್ಚ್ ಪ್ರೊವೈಡರ್ ಆಗಿ **Tavily** ಅನ್ನು ಸೇರಿಸಿದೆ, ಇದು ಹೆಚ್ಚು ಸಮೃದ್ಧ ಸರ್ಚ್ ಫಲಿತಾಂಶಗಳು ಮತ್ತು ಸುಧಾರಿತ ಸಂಶೋಧನಾ ಸಾಮರ್ಥ್ಯಗಳನ್ನು ನೀಡುತ್ತದೆ.

### Tavily API ಕೀ ಪಡೆಯುವುದು

1. [Tavily API page](https://tavily.com/) ಗೆ ಭೇಟಿ ನೀಡಿ.
2. ಒಂದು ಖಾತೆಯಲ್ಲಿ ನೋಂದಾಯಿಸಿ. ಮಾಸಿಕ ವಿನಂತಿ ಮಿತಿಯೊಂದಿಗೆ ಉಚಿತ ಟಿಯರ್ ಲಭ್ಯವಿದೆ.
3. Tavily ಡ್ಯಾಶ್‌ಬೋರ್ಡ್‌ನಿಂದ ನಿಮ್ಮ API ಕೀ ಅನ್ನು ನಕಲಿಸಿ.
4. ಅದನ್ನು Gratis AI Agent ಸೆಟ್ಟಿಂಗ್‌ಗಳಲ್ಲಿರುವ **Tavily API Key** ಫೀಲ್ಡ್‌ನಲ್ಲಿ ಅಂಟಿಸಿ.

### ಬಳಕೆಯ ಮಿತಿಗಳು

ಬಳಕೆಯು ಮಾಡಿದ API ಕರೆಗಳ ಸಂಖ್ಯೆಯ ಆಧಾರದ ಮೇಲೆ Tavily ನಿಂದ ಬಿಲ್ ಮಾಡಲಾಗುತ್ತದೆ. ಸರ್ಚ್ ಅನ್ನು déclencher ಮಾಡುವ ಪ್ರತಿ AI ಪ್ರತಿಕ್ರಿಯೆಯು ಒಂದು ಕರೆ (call) ಆಗಿ ગಣಿಕೆಯಾಗುತ್ತದೆ. ಅನಿರೀಕ್ಷಿತ ಶುಲ್ಕಗಳನ್ನು ತಪ್ಪಿಸಲು [Tavily dashboard](https://tavily.com/) ನಲ್ಲಿ ನಿಮ್ಮ ಬಳಕೆಯನ್ನು ಮೇಲ್ವಿಚಾರಣೆ ಮಾಡಿ.

## ಇಂಟರ್ನೆಟ್ ಸರ್ಚ್ ಅನ್ನು ನಿಷ್ಕ್ರಿಯಗೊಳಿಸುವುದು

ಕ್ರಿಯಾತ್ಮಕ ಸರ್ಚ್ ಪ್ರೊವೈಡರ್ ಫೀಲ್ಡ್‌ನಿಂದ API ಕೀ ಅನ್ನು ತೆಗೆದುಹಾಕಿ ಮತ್ತು ಉಳಿಸಿ. ಇಂಟರ್ನೆಟ್ ಸರ್ಚ್ ಸಾಮರ್ಥ್ಯವು ಅಸಿಸ್ಟೆಂಟ್‌ಗೆ ಇನ್ನು ಮುಂದೆ ನೀಡಲಾಗುವುದಿಲ್ಲ.

:::note
ಇಂಟರ್ನೆಟ್ ಸರ್ಚ್ ಪ್ರತಿಕ್ರಿಯೆಗಳಿಗೆ ವಿಳಂಬವನ್ನು (latency) ಸೇರಿಸುತ್ತದೆ ಏಕೆಂದರೆ ಉತ್ತರವನ್ನು ರಚಿಸುವ ಮೊದಲು ಅಸಿಸ್ಟೆಂಟ್ ಸರ್ಚ್ ಫಲಿತಾಂಶಗಳಿಗಾಗಿ ಕಾಯಬೇಕಾಗುತ್ತದೆ. ಸಮಯಕ್ಕೆ ನಿರ್ದಿಷ್ಟವಾದ ಬಳಕೆ ಸಂದರ್ಭಗಳಿಗಾಗಿ, ನೈಜ-ಸಮಯದ ಸರ್ಚ್ ಅಗತ್ಯವಿದೆಯೇ ಅಥವಾ ಅಸಿಸ್ಟೆಂಟ್‌ನ ಅಂತರ್ನಿರ್ಮಿತ ಜ್ಞಾನ ಸಾಕೇ ಅಥವಾ ಇಲ್ಲವೇ ಎಂದು ಪರಿಗಣಿಸಿ.
:::
