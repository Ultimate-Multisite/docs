# Theme Builder ಸಾಮರ್ಥ್ಯಗಳು: Scaffold ಮತ್ತು Activate Block Themes

Superdav AI Agent v1.12.0 ಎರಡು ಶಕ್ತಿಶಾಲಿ ಸಾಮರ್ಥ್ಯಗಳನ್ನು ಪರಿಚಯಿಸುತ್ತದೆ. ಇವುಗಳು ಚಾಟ್ ಇಂಟರ್ಫೇಸ್‌ನಿಂದ ನೇರವಾಗಿ ಕಸ್ಟಮ್ ಬ್ಲಾಕ್ ಥೀಮ್‌ಗಳನ್ನು ರಚಿಸಲು ಮತ್ತು ನಿಯೋಜಿಸಲು ನಿಮಗೆ ಅನುಮತಿಸುತ್ತವೆ.

## ಅವಲೋಕನ (Overview) {#overview}

**scaffold-block-theme** ಮತ್ತು **activate-theme** ಸಾಮರ್ಥ್ಯಗಳು ಏಜೆಂಟ್‌ಗಳಿಗೆ ಈ ಕೆಳಗಿನ ಕೆಲಸಗಳನ್ನು ಮಾಡಲು ಸಹಾಯ ಮಾಡುತ್ತವೆ:
- ನಿಮ್ಮ ನಿರ್ದಿಷ್ಟತೆಗಳ ಆಧಾರದ ಮೇಲೆ ಸಂಪೂರ್ಣ, ಪ್ರೊಡಕ್ಷನ್-ಸಿದ್ಧ ಬ್ಲಾಕ್ ಥೀಮ್‌ಗಳನ್ನು ರಚಿಸುವುದು
- ಕೈಯಾರೆ ಮಧ್ಯಪ್ರವೇಶವಿಲ್ಲದೆ ನಿಮ್ಮ ಸೈಟ್‌ನಲ್ಲಿ ಥೀಮ್‌ಗಳನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸಕ್ರಿಯಗೊಳಿಸುವುದು
- ಮಾರ್ಗದರ್ಶಿ ವಿನ್ಯಾಸ ನಿರ್ಧಾರಗಳ ಮೂಲಕ ಸುಸಂಬದ್ಧ ದೃಶ್ಯ ಗುರುತನ್ನು (cohesive visual identities) ಸೃಷ್ಟಿಸುವುದು

## ಬ್ಲಾಕ್ ಥೀಮ್ ಅನ್ನು Scaffold ಮಾಡುವುದು (Scaffold Block Theme) {#scaffold-block-theme}

**scaffold-block-theme** ಸಾಮರ್ಥ್ಯವು ಸಂಪೂರ್ಣ ಥೀಮ್ ರಚನೆಯೊಂದಿಗೆ ಹೊಸ WordPress ಬ್ಲಾಕ್ ಥೀಮ್ ಅನ್ನು ರಚಿಸುತ್ತದೆ. ಇದರಲ್ಲಿ ಈ ಕೆಳಗಿನವುಗಳು ಸೇರಿವೆ:

- ವಿನ್ಯಾಸ ಟೋಕನ್‌ಗಳೊಂದಿಗೆ `theme.json` ಕಾನ್ಫಿಗರೇಶನ್
- ಸಾಮಾನ್ಯ ಲೇಔಟ್‌ಗಳಿಗಾಗಿ ಬ್ಲಾಕ್ ಟೆಂಪ್ಲೇಟ್ ಫೈಲ್‌ಗಳು
- ಕಸ್ಟಮ್ ಬ್ಲಾಕ್ ಶೈಲಿಗಳು ಮತ್ತು ವ್ಯತ್ಯಾಸಗಳು (variations)
- ಥೀಮ್ ಮೆಟಾಡೇಟಾ ಮತ್ತು サಪೋರ್ಟ್ ಘೋಷಣೆಗಳು (support declarations)

### ಹೇಗೆ ಕರೆಯುವುದು (How to Invoke) {#how-to-invoke}

Superdav AI Agent ಜೊತೆಗಿನ ನಿಮ್ಮ ಚಾಟ್‌ನಲ್ಲಿ, ನೀವು ಥೀಮ್ ರಚನೆಯನ್ನು ವಿನಂತಿಸಬಹುದು:

```
"Create a block theme called 'Modern Agency' with a blue and white color scheme,
sans-serif typography, and a professional layout"
```

ಏಜೆಂಟ್ ಈ ಕೆಳಗಿನ ಕೆಲಸಗಳನ್ನು ಮಾಡುತ್ತದೆ:
1. ಸಂಭಾಷಣೆಯ ಮೂಲಕ ನಿಮ್ಮ ವಿನ್ಯಾಸ ಆದ್ಯತೆಗಳನ್ನು ಸಂಗ್ರಹಿಸುತ್ತದೆ
2. ಸಂಪೂರ್ಣ ಥೀಮ್ ರಚನೆಯನ್ನು ರಚಿಸುತ್ತದೆ
3. ಎಲ್ಲಾ ಅಗತ್ಯ ಥೀಮ್ ಫೈಲ್‌ಗಳನ್ನು ಸೃಷ್ಟಿಸುತ್ತದೆ
4. ಸಕ್ರಿಯಗೊಳಿಸಲು ಥೀಮ್ ಅನ್ನು ಸಿದ್ಧಪಡಿಸುತ್ತದೆ

### ನಿರೀಕ್ಷಿತ ಔಟ್‌ಪುಟ್ (Expected Output) {#expected-output}

ಸಾಮರ್ಥ್ಯವು ಯಶಸ್ವಿಯಾಗಿ ಕಾರ್ಯಗತಗೊಂಡಾಗ, ನೀವು ಈ ಕೆಳಗಿನವುಗಳನ್ನು ನೋಡುತ್ತೀರಿ:

- ಥೀಮ್ ಅನ್ನು scaffold ಮಾಡಲಾಗಿದೆ ಎಂಬ confirmation
- ಥೀಮ್‌ನ ಹೆಸರು ಮತ್ತು ಸ್ಥಳ
- ಅನ್ವಯಿಸಲಾದ ವಿನ್ಯಾಸ ಟೋಕನ್‌ಗಳ ಸಾರಾಂಶ (ಬಣ್ಣಗಳು, ಟೈಪೋಗ್ರಫಿ, ಸ್ಪೇಸಿಂಗ್)
- ಸಕ್ರಿಯಗೊಳಿಸಲು ಸಿದ್ಧವಾದ ಸ್ಥಿತಿ

ಉದಾಹರಣೆ ಔಟ್‌ಪುಟ್:
```
✓ Theme "Modern Agency" scaffolded successfully
  Location: /wp-content/themes/modern-agency/
  Colors: Primary #0066CC, Secondary #FFFFFF
  Typography: Inter (sans-serif)
  Status: Ready to activate
```

## ಥೀಮ್ ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸುವುದು (Activate Theme) {#activate-theme}

**activate-theme** ಸಾಮರ್ಥ್ಯವು ನಿಮ್ಮ ಸೈಟ್ ಅನ್ನು ಹೊಸದಾಗಿ scaffold ಮಾಡಿದ ಅಥವಾ ಈಗಾಗಲೇ ಇರುವ ಬ್ಲಾಕ್ ಥೀಮ್‌ಗೆ ಬದಲಾಯಿಸುತ್ತದೆ.

### ಹೇಗೆ ಕರೆಯುವುದು (How to Invoke) {#how-to-invoke-1}

ಥೀಮ್ ಅನ್ನು scaffold ಮಾಡಿದ ನಂತರ, ನೀವು ಅದನ್ನು ತಕ್ಷಣವೇ ಸಕ್ರಿಯಗೊಳಿಸಬಹುದು:

```
"Activate the Modern Agency theme"
```

ಅಥವಾ ಯಾವುದೇ ಇರುವ ಥೀಮ್ ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಬಹುದು:

```
"Switch to the Twentytwentyfour theme"
```

### ನಿರೀಕ್ಷಿತ ಔಟ್‌ಪುಟ್ (Expected Output) {#expected-output-1}

ಸಕ್ರಿಯಗೊಳಿಸುವಿಕೆಯು ಯಶಸ್ವಿಯಾದಾಗ:

- ಸಕ್ರಿಯ ಥೀಮ್‌ನ confirmation
- ಹಿಂದಿನ ಥೀಮ್‌ನ ಹೆಸರು (ಉಲ್ಲೇಖಕ್ಕಾಗಿ)
- ಥೀಮ್ ಈಗ ಲೈವ್ ಇರುವ ಸೈಟ್ URL
- ಯಾವುದೇ ಥೀಮ್-ನಿರ್ದಿಷ್ಟ ಸೆಟಪ್ ಟಿಪ್ಪಣಿಗಳು

ಉದಾಹರಣೆ ಔಟ್‌ಪುಟ್:
```
✓ Theme activated successfully
  Active theme: Modern Agency
  Previous theme: Twentytwentyfour
  Live at: https://yoursite.com
  Note: Check your homepage to verify the layout
```

## ಕಾರ್ಯಪ್ರವಾಹ: Scaffold ಮತ್ತು Activate (Workflow: Scaffold and Activate) {#workflow-scaffold-and-activate}

ಸಾಮಾನ್ಯ ಕಾರ್ಯಪ್ರವಾಹವು ಎರಡೂ ಸಾಮರ್ಥ್ಯಗಳನ್ನು ಸಂಯೋಜಿಸುತ್ತದೆ:

1. **ಥೀಮ್ ರಚನೆಯನ್ನು ವಿನಂತಿಸಿ**: "ನನ್ನ SaaS لینڈಿಂಗ್ ಪುಟಕ್ಕಾಗಿ ಬ್ಲಾಕ್ ಥೀಮ್ ರಚಿಸಿ"
2. **ಏಜೆಂಟ್ ಥೀಮ್ ಅನ್ನು scaffold ಮಾಡುತ್ತದೆ**: ಫೈಲ್‌ಗಳು ಮತ್ತು ವಿನ್ಯಾಸ ಟೋಕನ್‌ಗಳನ್ನು ರಚಿಸುತ್ತದೆ
3. **ಸಮರ್ಪಕ ಕರಾರು ಮತ್ತು ಪರಿಷ್ಕರಣೆ**: ಅಗತ್ಯವಿದ್ದರೆ ವಿನ್ಯಾಸ ಬದಲಾವಣೆಗಳ ಬಗ್ಗೆ ಚರ್ಚಿಸಿ
4. **ಸಕ್ರಿಯಗೊಳಿಸಿ**: "ಈಗ ಥೀಮ್ ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ"
5. **ಪರಿಶೀಲಿಸಿ**: ಹೊಸ ವಿನ್ಯಾಸ ಲೈವ್ ಆಗಿದೆಯೇ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಲು ನಿಮ್ಮ ಸೈಟ್‌ಗೆ ಭೇಟಿ ನೀಡಿ

## ವಿನ್ಯಾಸ ಟೋಕನ್‌ಗಳು ಮತ್ತು ಕಸ್ಟಮೈಸೇಶನ್ (Design Tokens and Customization) {#design-tokens-and-customization}

Scaffold ಮಾಡಿದ ಥೀಮ್‌ಗಳು WordPress ವಿನ್ಯಾಸ ಟೋಕನ್‌ಗಳನ್ನು (theme.json ಮೂಲಕ) ಈ ಕೆಳಗಿನವುಗಳಿಗಾಗಿ ಬಳಸುತ್ತವೆ:

- **ಬಣ್ಣಗಳು (Colors)**: ಪ್ರೈಮರಿ, ಸೆಕೆಂಡರಿ, ಆಕ್ಸೆಂಟ್, ನ್ಯೂಟ್ರಲ್ ಪ್ಯಾಲೆಟ್
- **ಟೈಪೋಗ್ರಫಿ (Typography)**: ಫಾಂಟ್ ಕುಟುಂಬಗಳು, ಗಾತ್ರಗಳು, ವಿಕೆಟ್‌ಗಳು, ಲೈನ್ ಹೈಟ್‌ಗಳು
- **ಸ್ಪೇಸಿಂಗ್ (Spacing)**: ಪ್ಯಾಡಿಂಗ್, ಮಾರ್ಜಿನ್, ಗ್ಯಾಪ್ ಸ್ಕೇಲ್‌ಗಳು
- **ಬಾರ್ಡರ್‌ಗಳು (Borders)**: ವಿಕೃತಿ (Radius) ಮತ್ತು ಅಗಲ ಟೋಕನ್‌ಗಳು
- **ಶ್ಯಾಡೋಗಳು (Shadows)**: ಎಲಿവേഷನ್ ಮಟ್ಟಗಳು

ಈ ಟೋಕನ್‌ಗಳು `theme.json` ನಲ್ಲಿ ಕೇಂದ್ರೀಕೃತವಾಗಿರುತ್ತವೆ, ಇದರಿಂದಾಗಿ ಒಂದೇ ಫೈಲ್‌ನಿಂದ ನಿಮ್ಮ ಇಡೀ ವಿನ್ಯಾಸ ವ್ಯವಸ್ಥೆಯನ್ನು (design system) ಹೊಂದಿಸಲು ಸುಲಭವಾಗುತ್ತದೆ.

## ಮಿತಿಗಳು ಮತ್ತು ಟಿಪ್ಪಣಿಗಳು (Limitations and Notes) {#limitations-and-notes}

- ಥೀಮ್‌ಗಳನ್ನು `/wp-content/themes/` ನಲ್ಲಿ scaffold ಮಾಡಲಾಗುತ್ತದೆ ಮತ್ತು ಅವು WordPress ನಾಮಕರಣ ಕನ್ವೆನ್ಷನ್‌ಗಳನ್ನು ಅನುಸರಿಸಬೇಕು
- ಸಕ್ರಿಯಗೊಳಿಸಲು ನಿಮ್ಮ WordPress ಸೈಟ್‌ನಲ್ಲಿ ಸೂಕ್ತ ಅನುಮತಿಗಳು ಬೇಕಾಗುತ್ತವೆ
- ಥೀಮ್‌ಗಳಲ್ಲಿ ಕಸ್ಟಮ್ PHP ಕೋಡ್ ಕಡಿಮೆ ಇರುತ್ತದೆ; ಸಂಕೀರ್ಣ ಕಾರ್ಯಕ್ಷಮತೆಗಾಗಿ ಪ್ಲಗ್ಇನ್‌ಗಳನ್ನು ಬಳಸಿ
- ಬ್ಲಾಕ್ ಥೀಮ್‌ಗಳು WordPress 5.9 ಮತ್ತು ನಂತರದೊಂದಿಗೆ ಉತ್ತಮವಾಗಿ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತವೆ

## ಸಮಸ್ಯೆ ಪರಿಹಾರ (Troubleshooting) {#troubleshooting}

**Scaffolding ನಂತರ ಥೀಮ್ ಕಾಣಿಸದಿದ್ದರೆ**
- ಥೀಮ್ ಡೈರೆಕ್ಟರಿ ಅಸ್ತಿತ್ವದಲ್ಲಿದೆ ಮತ್ತು ಸರಿಯಾದ ಅನುಮತಿಗಳನ್ನು ಹೊಂದಿದೆಯೇ ಎಂದು ಪರಿಶೀಲಿಸಿ
- `theme.json` ವ್ಯಾಲಿಡ್ JSON ಆಗಿದೆಯೇ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ
- ಥೀಮ್‌ನ ಹೆಸರು ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಥೀಮ್‌ಗಳೊಂದಿಗೆ ಸಂಘರ್ಷ ಹೊಂದಿಲ್ಲ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ

**ಸಕ್ರಿಯಗೊಳಿಸುವಿಕೆ ವಿಫಲವಾದರೆ**
- ನಿಮಗೆ ಆಡಮಿನಿಸ್ಟ್ರೇಟರ್ ಅನುಮತಿಗಳು ಇದೆಯೇ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ
- ಥೀಮ್ ಡೈರೆಕ್ಟರಿಯನ್ನು WordPress ಓದಲು ಸಾಧ್ಯವಾಗಿದೆಯೇ ಎಂದು ಪರಿಶೀಲಿಸಿ
- ವಿವರಗಳಿಗಾಗಿ WordPress error logs ಅನ್ನು ಪರಿಶೀಲಿಸಿ

**ವಿನ್ಯಾಸ ಟೋಕನ್‌ಗಳು ಅನ್ವಯಿಸದಿದ್ದರೆ**
- `theme.json` syntax ಸರಿಯಾಗಿದೆಯೇ ಎಂದು ಪರಿಶೀಲಿಸಿ
- ಯಾವುದೇ 캐ಶಿಂಗ್ ಪ್ಲಗ್ಇನ್‌ಗಳನ್ನು (caching plugins) ತೆರವುಗೊಳಿಸಿ
- ನೀವು ಬಳಸುತ್ತಿರುವ ಟೋಕನ್‌ಗಳನ್ನು ನಿಮ್ಮ WordPress ಆವೃತ್ತಿಯು ಬೆಂಬಲಿಸುತ್ತದೆಯೇ ಎಂದು ಪರಿಶೀಲಿಸಿ

## ಮುಂದಿನ ಹಂತಗಳು (Next Steps) {#next-steps}

ನಿಮ್ಮ ಥೀಮ್ ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿದ ನಂತರ, ನೀವು ಈ ಕೆಳಗಿನವುಗಳನ್ನು ಮಾಡಬಹುದು:
- ಟೈಪೋಗ್ರಫಿ, ಬಣ್ಣಗಳು ಮತ್ತು ಸ್ಪೇಸಿಂಗ್ ಅನ್ನು ಪರಿಷ್ಕರಿಸಲು **Design System Aesthetics skill** ಅನ್ನು ಬಳಸಿ
- WordPress ಬ್ಲಾಕ್ ಎಡಿಟರ್ ಮೂಲಕ 개별 ಬ್ಲಾಕ್ ಶೈಲಿಗಳನ್ನು ಕಸ್ಟಮೈಸ್ ಮಾಡಿ
- ಥೀಮ್‌ನ `style.css` ಫೈಲ್‌ನಲ್ಲಿ ಕಸ್ಟಮ್ CSS ಸೇರಿಸಿ
- ನಿರ್ದಿಷ್ಟ ಪುಟ ಪ್ರಕಾರಗಳಿಗಾಗಿ ಕಸ್ಟಮ್ ಬ್ಲಾಕ್ ಟೆಂಪ್ಲೇಟ್‌ಗಳನ್ನು ರಚಿಸಿ
