# Ultimate Multisite: Ad Injector ಬಳಸಿ ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿ ಜಾಹೀರಾತುಗಳನ್ನು ಹಾಕುವುದು

Ultimate Multisite: Ad Injector ಒಂದು **ಉಚಿತ add-on** ಆಗಿದ್ದು, ಇದು ನಿಮ್ಮ ಪ್ಲಾನ್‌ಗಳ ಎಡಿಟ್ ಪೇಜ್‌ನಿಂದಲೇ ಜಾಹೀರಾತುಗಳನ್ನು ಹಾಕಲು ಅಗತ್ಯವಿರುವ ಟೂಲ್‌ಗಳನ್ನು ಒದಗಿಸುತ್ತದೆ.

Ultimate Multisite: Ad Injector ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡಿದ ನಂತರ, ಅದನ್ನು ಆಕ್ಟಿವೇಟ್ ಮಾಡಲು ನಿಮ್ಮ _Plugins_ ಪೇಜ್‌ಗೆ ಹೋಗಿ.

add-on ಆಕ್ಟಿವ್ ಆದ ನಂತರ, ಪ್ಲಾನ್ ಎಡಿಟ್ ಸ್ಕ್ರೀನ್‌ನಲ್ಲಿರುವ _Advanced Options_ ಮೆಟಾ ಬಾಕ್ಸ್‌ನಲ್ಲಿ _Ad Settings_ ಎಂಬ ಹೊಸ ಆಯ್ಕೆ ಕಾಣಿಸುತ್ತದೆ.

![ಪ್ಲಾನ್ ಎಡಿಟ್ ಸ್ಕ್ರೀನ್‌ನಲ್ಲಿ Ad Settings](/img/config/product-edit-full.png)

_ಈಗ ನಿಮ್ಮ ಪ್ರತಿಯೊಂದು ಪ್ಲಾನ್‌ಗೆ ಜಾಹೀರಾತು ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಎಡಿಟ್ ಮಾಡಬಹುದು_

ಆ ಟ್ಯಾಬ್‌ನಲ್ಲಿ, ಫ್ರಂಟ್-ಎಂಡ್‌ನಲ್ಲಿ ಪೋಸ್ಟ್‌ಗಳು ಮತ್ತು ಪೇಜ್‌ಗಳಿಗೆ, ಕಂಟೆಂಟ್‌ನ ಮೊದಲು ಮತ್ತು ನಂತರ ಜಾಹೀರಾತು ಕೋಡ್‌ಗಳನ್ನು ಸೇರಿಸಬಹುದು.
