# ಇತರ ಪ್ಲಗಿನ್‌ಗಳೊಂದಿಗೆ ತಿಳಿದಿರುವ ಅಸಾಮರಸ್ಯಗಳು

Ultimate Multisite ಅನ್ನು ಸ್ಥಾಪಿಸಿ ಸಕ್ರಿಯಗೊಳಿಸಿದ ನಂತರ, ಇತರ ಪ್ಲಗಿನ್‌ಗಳೊಂದಿಗೆ ಇನ್ನೂ ಪರಿಹರಿಸದ ಅಸಾಮರಸ್ಯಗಳಿಂದಾಗಿ ನೀವು ಸಮಸ್ಯೆಗಳನ್ನು ಎದುರಿಸಬಹುದು.

ಪ್ರತಿಯೊಂದು ಹೊಸ ಬಿಡುಗಡೆಯೊಂದಿಗೆ ಈ ಪಟ್ಟಿಯನ್ನು ಕಡಿಮೆ ಮಾಡಲು ನಾವು ಸಕ್ರಿಯವಾಗಿ ಕೆಲಸ ಮಾಡುತ್ತಿದ್ದೇವೆ, ಆದರೆ ಒಂದು ಪ್ಲಗಿನ್ ಇಲ್ಲಿ ಪಟ್ಟಿಯಲ್ಲಿದ್ದರೆ, ಸಂಪೂರ್ಣವಾಗಿ ಕಾರ್ಯನಿರ್ವಹಿಸುವ Ultimate Multisite Network ಹೊಂದಲು ನೀವು ಅದನ್ನು ನಿಷ್ಕ್ರಿಯಗೊಳಿಸಬೇಕಾಗಬಹುದು.

ಅವುಗಳ ಪಟ್ಟಿ ಮತ್ತು ಈ ಸಂದರ್ಭಗಳಲ್ಲಿ ನೀವು ಏನು ಮಾಡಬೇಕು ಎಂಬುದು ಇಲ್ಲಿದೆ:
