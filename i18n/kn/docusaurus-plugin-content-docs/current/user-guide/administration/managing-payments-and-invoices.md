# ಪಾವತಿಗಳು ಮತ್ತು ಇನ್‌ವಾಯ್ಸ್‌ಗಳನ್ನು ನಿರ್ವಹಿಸುವುದು

## ಪಾವತಿ ಸೆಟ್ಟಿಂಗ್‌ಗಳು

ಪಾವತಿಗಳನ್ನು ಸ್ವೀಕರಿಸಲು ಪ್ರಾರಂಭಿಸುವ ಮೊದಲು, ಪಾವತಿ-ಸಂಬಂಧಿತ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಬೇಕಾಗುತ್ತದೆ. **Ultimate Multisite → Settings** ಗೆ ಹೋಗಿ ಮತ್ತು **Payment** ಟ್ಯಾಬ್ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ.

![ಪಾವತಿ ಸೆಟ್ಟಿಂಗ್‌ಗಳ ಟ್ಯಾಬ್](/img/admin/settings-payments-top.png)

### ಸಾಮಾನ್ಯ ಪಾವತಿ ಆಯ್ಕೆಗಳು

ಸಾಮಾನ್ಯ ಸೆಟ್ಟಿಂಗ್‌ಗಳಲ್ಲಿ ನೀವು ಈ ಕೆಳಗಿನವುಗಳನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಬಹುದು:

- **Currency** — ವಹಿವಾಟುಗಳಿಗೆ ಬಳಸುವ ಡೀಫಾಲ್ಟ್ ಕರೆನ್ಸಿ
- **Currency Position** — ಕರೆನ್ಸಿ ಚಿಹ್ನೆ ಎಲ್ಲಿ ಕಾಣಿಸುತ್ತದೆ (ಮೊತ್ತದ ಮೊದಲು/ನಂತರ)

![ಪಾವತಿ ಗೇಟ್‌ವೇ ಸೆಟ್ಟಿಂಗ್‌ಗಳು](/img/admin/settings-payments-options.png)

### ಪಾವತಿ ಗೇಟ್‌ವೇಗಳು

Ultimate Multisite ಹಲವಾರು ಪಾವತಿ ಗೇಟ್‌ವೇಗಳನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ. Payment ಸೆಟ್ಟಿಂಗ್‌ಗಳ ಟ್ಯಾಬ್‌ನಿಂದ ಪ್ರತಿಯೊಂದನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಬಹುದು ಮತ್ತು ಕಾನ್ಫಿಗರ್ ಮಾಡಬಹುದು.

![ಪಾವತಿ ಗೇಟ್‌ವೇ ಕಾನ್ಫಿಗರೇಶನ್](/img/admin/settings-payments-gateways.png)

ಲಭ್ಯವಿರುವ ಗೇಟ್‌ವೇಗಳು:

- **Stripe** — Stripe ಮೂಲಕ ಕ್ರೆಡಿಟ್ ಕಾರ್ಡ್ ಪಾವತಿಗಳು
- **PayPal** — PayPal ಪಾವತಿಗಳು
- **Manual** — ಆಫ್‌ಲೈನ್ ಅಥವಾ ಕಸ್ಟಮ್ ಪಾವತಿ ಪ್ರಕ್ರಿಯೆಗಾಗಿ

ಪ್ರತಿ ಗೇಟ್‌ವೇಗೆ ತನ್ನದೇ ಆದ ಕಾನ್ಫಿಗರೇಶನ್ ವಿಭಾಗವಿದೆ, ಅಲ್ಲಿ ನೀವು API ಕೀಗಳು ಮತ್ತು ಇತರ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ನಮೂದಿಸಬಹುದು.

![ಹೆಚ್ಚುವರಿ ಗೇಟ್‌ವೇ ಸೆಟ್ಟಿಂಗ್‌ಗಳು](/img/admin/settings-payments-gateways-2.png)

### Sandbox ಮೋಡ್

ಲೈವ್ ಆಗುವ ಮೊದಲು ನಿಮ್ಮ ಪಾವತಿ ಇಂಟಿಗ್ರೇಶನ್ ಅನ್ನು ಪರೀಕ್ಷಿಸಲು **Sandbox Mode** ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಬಹುದು. Sandbox ಮೋಡ್ ಸಕ್ರಿಯವಾಗಿರುವಾಗ, ನಿಜವಾದ ಶುಲ್ಕಗಳು ವಿಧಿಸಲ್ಪಡುವುದಿಲ್ಲ.

## ಪಾವತಿಗಳನ್ನು ವೀಕ್ಷಿಸುವುದು

ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿರುವ ಎಲ್ಲಾ ವಹಿವಾಟುಗಳನ್ನು ನೋಡಲು Ultimate Multisite ಅಡಿಯಲ್ಲಿ **Payments** ಪುಟಕ್ಕೆ ಹೋಗಿ.

![ಪಾವತಿಗಳ ಪಟ್ಟಿ](/img/admin/payments-list.png)

ಸ್ಥಿತಿಯ ಪ್ರಕಾರ (completed, pending, failed, refunded) ಪಾವತಿಗಳನ್ನು ಫಿಲ್ಟರ್ ಮಾಡಬಹುದು ಮತ್ತು ನಿರ್ದಿಷ್ಟ ವಹಿವಾಟುಗಳನ್ನು ಹುಡುಕಬಹುದು.

ಲೈನ್ ಐಟಮ್‌ಗಳು, ಸಂಬಂಧಿತ ಸದಸ್ಯತ್ವ, ಗ್ರಾಹಕರ ಮಾಹಿತಿ, ಮತ್ತು ಪಾವತಿ ಗೇಟ್‌ವೇ ಡೇಟಾ ಸೇರಿದಂತೆ ಸಂಪೂರ್ಣ ವಿವರಗಳನ್ನು ನೋಡಲು ಪಾವತಿಯ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ.

## ಇನ್‌ವಾಯ್ಸ್‌ಗಳು

Ultimate Multisite ಪಾವತಿಗಳಿಗೆ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಇನ್‌ವಾಯ್ಸ್‌ಗಳನ್ನು ರಚಿಸಬಹುದು. Payment ಸೆಟ್ಟಿಂಗ್‌ಗಳಿಂದ ಇನ್‌ವಾಯ್ಸ್ ಟೆಂಪ್ಲೇಟ್ ಮತ್ತು ಸಂಖ್ಯೆ ಫಾರ್ಮ್ಯಾಟ್ ಅನ್ನು ಕಸ್ಟಮೈಜ್ ಮಾಡಬಹುದು.

ಇನ್‌ವಾಯ್ಸ್ ಕಸ್ಟಮೈಜೇಶನ್ ಆಯ್ಕೆಗಳು:

- ಇನ್‌ವಾಯ್ಸ್‌ಗಳಲ್ಲಿ ತೋರಿಸುವ **ಕಂಪನಿಯ ಹೆಸರು ಮತ್ತು ವಿಳಾಸ**
- **ಇನ್‌ವಾಯ್ಸ್ ಸಂಖ್ಯೆ** ಫಾರ್ಮ್ಯಾಟ್ ಮತ್ತು ಅನುಕ್ರಮ
- ಇನ್‌ವಾಯ್ಸ್ ಹೆಡರ್‌ನಲ್ಲಿ ತೋರಿಸುವ **ಲೋಗೋ**
- ನಿಯಮಗಳು, ಟಿಪ್ಪಣಿಗಳು, ಅಥವಾ ಕಾನೂನು ಮಾಹಿತಿಗಾಗಿ **ಕಸ್ಟಮ್ ಫೂಟರ್ ಪಠ್ಯ**

ಇನ್‌ವಾಯ್ಸ್ ಟೆಂಪ್ಲೇಟ್ ಅನ್ನು ಕಸ್ಟಮೈಜ್ ಮಾಡಲು, **Ultimate Multisite → Settings → Payment** ಗೆ ಹೋಗಿ ಮತ್ತು ಇನ್‌ವಾಯ್ಸ್-ಸಂಬಂಧಿತ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಹುಡುಕಿ.
