# ಪಾವತಿಗಳು ಮತ್ತು ಇನ್‌ವಾಯ್ಸ್‌ಗಳನ್ನು ನಿರ್ವಹಿಸುವುದು

## ಪಾವತಿ ಸೆಟ್ಟಿಂಗ್‌ಗಳು

ನೀವು ಪಾವತಿಗಳನ್ನು ಸ್ವೀಕರಿಸಲು ಆರಂಭಿಸುವ ಮೊದಲು, ಪಾವತಿ-ಸಂಬಂಧಿತ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಸಂರಚಿಸಬೇಕು. **Ultimate Multisite → Settings** ಗೆ ಹೋಗಿ ಮತ್ತು **Payment** ಟ್ಯಾಬ್ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ.

![ಪಾವತಿ ಸೆಟ್ಟಿಂಗ್‌ಗಳ ಟ್ಯಾಬ್](/img/admin/settings-payments-top.png)

ಪಾವತಿ ಸೆಟ್ಟಿಂಗ್‌ಗಳ ಪುಟದ ಸಂಪೂರ್ಣ ನೋಟ ಇಲ್ಲಿದೆ:

![ಪಾವತಿ ಸೆಟ್ಟಿಂಗ್‌ಗಳ ಸಂಪೂರ್ಣ ಪುಟ](/img/admin/settings-payments-full.png)

### ಸಾಮಾನ್ಯ ಪಾವತಿ ಆಯ್ಕೆಗಳು

ಸಾಮಾನ್ಯ ಸೆಟ್ಟಿಂಗ್‌ಗಳಲ್ಲಿ ನೀವು ಇವುಗಳನ್ನು ಸಂರಚಿಸಬಹುದು:

- **ಕರೆನ್ಸಿ** — ವ್ಯವಹಾರಗಳಿಗೆ ಬಳಸುವ ಡೀಫಾಲ್ಟ್ ಕರೆನ್ಸಿ
- **ಕರೆನ್ಸಿ ಸ್ಥಾನ** — ಕರೆನ್ಸಿ ಚಿಹ್ನೆ ಎಲ್ಲಿ ಕಾಣಿಸಿಕೊಳ್ಳುತ್ತದೆ (ಮೊತ್ತದ ಮೊದಲು/ನಂತರ)

![ಪಾವತಿ gateway ಸೆಟ್ಟಿಂಗ್‌ಗಳು](/img/admin/settings-payments-options.png)

### ಪಾವತಿ gateways

Ultimate Multisite ಹಲವಾರು ಪಾವತಿ gateways ಅನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ. Payment ಸೆಟ್ಟಿಂಗ್‌ಗಳ ಟ್ಯಾಬ್‌ನಿಂದ ಪ್ರತಿಯೊಂದನ್ನೂ ಸಕ್ರಿಯಗೊಳಿಸಿ ಸಂರಚಿಸಬಹುದು.

![ಪಾವತಿ gateway ಸಂರಚನೆ](/img/admin/settings-payments-gateways.png)

ಲಭ್ಯವಿರುವ gateways ಇವುಗಳನ್ನು ಒಳಗೊಂಡಿವೆ:

- **Stripe** — Stripe ಮೂಲಕ ಕ್ರೆಡಿಟ್ ಕಾರ್ಡ್ ಪಾವತಿಗಳು
- **PayPal** — PayPal ಪಾವತಿಗಳು
- **Manual** — ಆಫ್‌ಲೈನ್ ಅಥವಾ ಕಸ್ಟಮ್ ಪಾವತಿ ಪ್ರಕ್ರಿಯೆಗಾಗಿ

ಪ್ರತಿ gateway ಗೆ API keys ಮತ್ತು ಇತರ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ನಮೂದಿಸುವ ತನ್ನದೇ ಸಂರಚನಾ ವಿಭಾಗವಿದೆ.

![ಹೆಚ್ಚುವರಿ gateway ಸೆಟ್ಟಿಂಗ್‌ಗಳು](/img/admin/settings-payments-gateways-2.png)

### Sandbox Mode

ಲೈವ್‌ಗೆ ಹೋಗುವ ಮೊದಲು ನಿಮ್ಮ ಪಾವತಿ ಏಕೀಕರಣವನ್ನು ಪರೀಕ್ಷಿಸಲು **Sandbox Mode** ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಬಹುದು. sandbox mode ಸಕ್ರಿಯವಾಗಿರುವಾಗ, ನಿಜವಾದ ಶುಲ್ಕಗಳನ್ನು ವಿಧಿಸಲಾಗುವುದಿಲ್ಲ.

## ಪಾವತಿಗಳನ್ನು ನೋಡುವುದು

ನಿಮ್ಮ ಜಾಲದಾದ್ಯಂತ ಇರುವ ಎಲ್ಲಾ ವ್ಯವಹಾರಗಳನ್ನು ನೋಡಲು Ultimate Multisite ಅಡಿಯಲ್ಲಿ ಇರುವ **Payments** ಪುಟಕ್ಕೆ ಹೋಗಿ.

![ಪಾವತಿಗಳ ಪಟ್ಟಿ](/img/admin/payments-list.png)

ಸ್ಥಿತಿ (ಪೂರ್ಣಗೊಂಡ, ಬಾಕಿ, ವಿಫಲ, ಮರುಪಾವತಿಸಿದ) ಆಧಾರದ ಮೇಲೆ ಪಾವತಿಗಳನ್ನು ಶೋಧಿಸಬಹುದು ಮತ್ತು ನಿರ್ದಿಷ್ಟ ವ್ಯವಹಾರಗಳನ್ನು ಹುಡುಕಬಹುದು.

ಪಾವತಿಯ ಪೂರ್ಣ ವಿವರಗಳನ್ನು ನೋಡಲು ಅದರ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ. ಇದರಲ್ಲಿ ಸಾಲಿನ ಐಟಂಗಳು, ಸಂಬಂಧಿತ membership, ಗ್ರಾಹಕ ಮಾಹಿತಿ ಮತ್ತು ಪಾವತಿ gateway ಡೇಟಾ ಒಳಗೊಂಡಿರುತ್ತದೆ.

## ಇನ್‌ವಾಯ್ಸ್‌ಗಳು

Ultimate Multisite ಪಾವತಿಗಳಿಗೆ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಇನ್‌ವಾಯ್ಸ್‌ಗಳನ್ನು ರಚಿಸಬಹುದು. Payment ಸೆಟ್ಟಿಂಗ್‌ಗಳಿಂದ ಇನ್‌ವಾಯ್ಸ್ ಟೆಂಪ್ಲೇಟ್ ಮತ್ತು ಸಂಖ್ಯಾಕ್ರಮ ಸ್ವರೂಪವನ್ನು ಕಸ್ಟಮೈಸ್ ಮಾಡಬಹುದು.

ಇನ್‌ವಾಯ್ಸ್ ಕಸ್ಟಮೈಸೇಶನ್ ಆಯ್ಕೆಗಳು ಇವುಗಳನ್ನು ಒಳಗೊಂಡಿವೆ:

- ಇನ್‌ವಾಯ್ಸ್‌ಗಳಲ್ಲಿ ಪ್ರದರ್ಶಿಸಲಾದ **ಕಂಪನಿ ಹೆಸರು ಮತ್ತು ವಿಳಾಸ**
- **ಇನ್‌ವಾಯ್ಸ್ ಸಂಖ್ಯಾಕ್ರಮ** ಸ್ವರೂಪ ಮತ್ತು ಕ್ರಮ
- ಇನ್‌ವಾಯ್ಸ್ ಹೆಡರ್‌ನಲ್ಲಿ ಪ್ರದರ್ಶಿಸಲಾದ **ಲೋಗೋ**
- ನಿಯಮಗಳು, ಟಿಪ್ಪಣಿಗಳು ಅಥವಾ ಕಾನೂನು ಮಾಹಿತಿಗಾಗಿ **ಕಸ್ಟಮ್ footer text**

ಇನ್‌ವಾಯ್ಸ್ ಟೆಂಪ್ಲೇಟ್ ಅನ್ನು ಕಸ್ಟಮೈಸ್ ಮಾಡಲು, **Ultimate Multisite → Settings → Payment** ಗೆ ಹೋಗಿ ಮತ್ತು ಇನ್‌ವಾಯ್ಸ್-ಸಂಬಂಧಿತ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಹುಡುಕಿ.
