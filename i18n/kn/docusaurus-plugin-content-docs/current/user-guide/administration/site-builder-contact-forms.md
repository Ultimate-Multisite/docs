# Site Builder ಸಂಪರ್ಕ ಫಾರ್ಮ್‌ಗಳು {#site-builder-contact-forms}

Superdav AI Agent v1.10.0 ಅನ್ನು ಬಳಸುವುದರಿಂದ, ನೀವು Site Builder agent ಮೂಲಕ ಚಾಟ್ ಇಂಟರ್‌ಫೇಸ್‌ನಿಂದ ನೇರವಾಗಿ ಸಂಪರ್ಕ ಫಾರ್ಮ್‌ಗಳನ್ನು ರಚಿಸುವ ಸಾಮರ್ಥ್ಯವನ್ನು ಪಡೆಯುತ್ತೀರಿ. ಇದು ಚಾಟ್‌ನಿಂದ ಹೊರಹೋಗದೆ ಯಾವುದೇ ಪುಟಕ್ಕೆ ಸಂಪೂರ್ಣವಾಗಿ ಕಾರ್ಯನಿರ್ವಹಿಸುವ ಸಂಪರ್ಕ ಫಾರ್ಮ್‌ಗಳನ್ನು ಸೇರಿಸಲು ನಿಮಗೆ ಅನುಮತಿಸುತ್ತದೆ.

## Overview {#overview}

Site Builder ಸಂಪರ್ಕ ಫಾರ್ಮ್ ಸಾಮರ್ಥ್ಯ (`create_contact_form`) ನಿಮ್ಮಲ್ಲಿติดตั้งಿರುವ ಫಾರ್ಮ್ plugin ಅನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಪತ್ತೆಹಚ್ಚುತ್ತದೆ ಮತ್ತು ಆ plugin ನ ಸ್ಥಳೀಯ ಸಾಮರ್ಥ್ಯಗಳನ್ನು ಬಳಸಿ ಸಂಪರ್ಕ ಫಾರ್ಮ್ ಅನ್ನು ರಚಿಸುತ್ತದೆ. ಬೆಂಬಲಿತ ಫಾರ್ಮ್ plugin ಗಳು ಇವು:

- Contact Form 7
- WPForms
- Fluent Forms
- Gravity Forms

ಈ ಅಸಿಸ್ಟೆಂಟ್ ನಿಮ್ಮ ಸೈಟ್‌ನಲ್ಲಿ ಲಭ್ಯವಿರುವ ಅತ್ಯುತ್ತಮ ಫಾರ್ಮ್ plugin ಅನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಆಯ್ಕೆ ಮಾಡುತ್ತದೆ ಮತ್ತು ಆ plugin ಗೆ ಅನುಗುಣವಾಗಿ ಸಂಪರ್ಕ ಫಾರ್ಮ್ ಅನ್ನು ರಚಿಸುತ್ತದೆ.

## ಸಂಪರ್ಕ ಫಾರ್ಮ್ ರಚಿಸುವುದು {#creating-a-contact-form}

Site Builder ಬಳಸಿ ಸಂಪರ್ಕ ಫಾರ್ಮ್ ಅನ್ನು ರಚಿಸಲು:

1. WordPress ಆಡ್ಮಿನ್‌ನಲ್ಲಿ **Gratis AI Agent** ಚಾಟ್ ಪ್ಯಾನೆಲ್ ಅನ್ನು ತೆರೆಯಿರಿ.
2. ಚಾಟ್ ಹೆಡರ್‌ನಲ್ಲಿರುವ agent ಐಕಾನ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡುವ ಮೂಲಕ **Site Builder** agent ಗೆ ಬದಲಾಯಿಸಿ.
3. ನೀವು ರಚಿಸಲು ಬಯಸುವ ಸಂಪರ್ಕ ಫಾರ್ಮ್ ಅನ್ನು ವಿವರಿಸಿ. ಉದಾಹರಣೆಗೆ:

   > "ಹೆಸರು, ಇಮೇಲ್, ಸಂದೇಶ ಮತ್ತು ಫೋನ್ ಸಂಖ್ಯೆ ಕ್ಷೇತ್ರಗಳೊಂದಿಗೆ ಸಂಪರ್ಕ ಪುಟಕ್ಕೆ ಒಂದು ಸಂಪರ್ಕ ಫಾರ್ಮ್ ಸೇರಿಸಿ."

   ಅಥವಾ ಸರಳವಾಗಿ:

   > "ಸಂಪರ್ಕ ಪುಟಕ್ಕಾಗಿ ಒಂದು ಸಂಪರ್ಕ ಫಾರ್ಮ್ ರಚಿಸಿ."

4. Site Builder ಸಂಪರ್ಕ ಫಾರ್ಮ್ ಅನ್ನು ರಚಿಸಿ, ಮತ್ತು ಅದನ್ನು ಎಂಬೆಡ್ ಮಾಡಲು ಸಿದ್ಧವಾದ shortcode ಅನ್ನು ನಿಮಗೆ ನೀಡುತ್ತದೆ.

## ರಚಿಸಲಾದ Shortcode ಅನ್ನು ಬಳಸುವುದು {#using-the-generated-shortcode}

Site Builder ಸಂಪರ್ಕ ಫಾರ್ಮ್ ಅನ್ನು ರಚಿಸಿದ ನಂತರ, ಅದು ಒಂದು shortcode ಅನ್ನು ನಿಮಗೆ ನೀಡುತ್ತದೆ (ಉದಾಹರಣೆಗೆ, `[contact-form-7 id="123"]`). ನೀವು ಇದನ್ನು ಹೀಗೆ ಬಳಸಬಹುದು:

1. **ಪುಟ ಅಥವಾ ಪೋಸ್ಟ್‌ನಲ್ಲಿ ಎಂಬೆಡ್ ಮಾಡುವುದು** — shortcode ಅನ್ನು ನಕಲಿಸಿ ಮತ್ತು ಬ್ಲಾಕ್ ಎಡಿಟರ್ ಅಥವಾ ಕ್ಲಾಸಿಕ್ ಎಡಿಟರ್ ಬಳಸಿ ಯಾವುದೇ ಪುಟ ಅಥವಾ ಪೋಸ್ಟ್‌ನಲ್ಲಿ ಅಂಟಿಸಿ.
2. **Site Builder ಮೂಲಕ ಸೇರಿಸುವುದು** — ಫಾರ್ಮ್ ಅನ್ನು ನಿರ್ದಿಷ್ಟ ಪುಟಕ್ಕೆ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸೇರಿಸಲು Site Builder ಗೆ ಕೇಳಿ:

   > "ಸಂಪರ್ಕ ಪುಟಕ್ಕೆ ಸಂಪರ್ಕ ಫಾರ್ಮ್ ಅನ್ನು ಸೇರಿಸಿ."

3. **template ನಲ್ಲಿ ಬಳಸುವುದು** — ನಿಮಗೆ PHP ಬಗ್ಗೆ ತಿಳಿದಿದ್ದರೆ, ನೀವು shortcode ಅನ್ನು ಥೀಮ್ template ಫೈಲ್‌ನಲ್ಲಿ ಸೇರಿಸಬಹುದು.

## ಸಂಪರ್ಕ ಫಾರ್ಮ್ ಅನ್ನು ಕಸ್ಟಮೈಸ್ ಮಾಡುವುದು {#customizing-the-contact-form}

Site Builder ಸಂಪರ್ಕ ಫಾರ್ಮ್ ಅನ್ನು ರಚಿಸಿದ ನಂತರ, ನೀವು ಅದನ್ನು ಮತ್ತಷ್ಟು ಕಸ್ಟಮೈಸ್ ಮಾಡಬಹುದು:

### ಚಾಟ್ ಇಂಟರ್‌ಫೇಸ್ ಮೂಲಕ {#via-the-chat-interface}

ಫಾರ್ಮ್ ಅನ್ನು ಮಾರ್ಪಡಿಸಲು Site Builder ಗೆ ಕೇಳಿ:

> "ವಿಷಯ (subject) ಕ್ಷೇತ್ರವನ್ನು ಸೇರಿಸಲು ಮತ್ತು ಸಂದೇಶ ಕ್ಷೇತ್ರವನ್ನು ಕಡ್ಡಾಯ (required) ಮಾಡಲು ಸಂಪರ್ಕ ಫಾರ್ಮ್ ಅನ್ನು ಅಪ್ಡೇಟ್ ಮಾಡಿ."

Site Builder ಫಾರ್ಮ್ ಅನ್ನು ಅಪ್ಡೇಟ್ ಮಾಡಿ ನವೀಕರಿಸಿದ shortcode ಅನ್ನು ನಿಮಗೆ ನೀಡುತ್ತದೆ.

### ಫಾರ್ಮ್ plugin ನ ಆಡ್ಮಿನ್ ಇಂಟರ್‌ಫೇಸ್ ಮೂಲಕ {#via-the-form-plugins-admin-interface}

ನೀವು ಫಾರ್ಮ್ ಅನ್ನು ನೇರವಾಗಿ ನಿಮ್ಮ ಫಾರ್ಮ್ plugin ನ ಸೆಟ್ಟಿಂಗ್‌ಗಳಲ್ಲಿಯೂ ಕಸ್ಟಮೈಸ್ ಮಾಡಬಹುದು:

1. WordPress ಆಡ್ಮಿನ್‌ನಲ್ಲಿ **Contact Form 7** (ಅಥವಾ ನಿಮ್ಮ ಅಳವಡಿಸಿದ ಫಾರ್ಮ್ plugin) ಗೆ ಹೋಗಿ.
2. Site Builder ರಚಿಸಿದ ಫಾರ್ಮ್ ಅನ್ನು ಹುಡುಕಿ.
3. ಅಗತ್ಯವಿರುವಂತೆ ಫಾರ್ಮ್ ಕ್ಷೇತ್ರಗಳು, ವ್ಯಾಲಿಡೇಶನ್ ನಿಯಮಗಳು ಮತ್ತು ಇಮೇಲ್ ನೋಟಿಫಿಕೇಶನ್‌ಗಳನ್ನು ಸಂಪಾದಿಸಿ.

## ಫಾರ್ಮ್ plugin ಗಳು ಮತ್ತು compatibility {#form-plugins-and-compatibility}

Site Builder ನಿಮ್ಮ ಸೈಟ್‌ನಲ್ಲಿ ಯಾವ ಫಾರ್ಮ್ plugin ಅಳವಡಿಸಲಾಗಿದೆ ಎಂಬುದನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಪತ್ತೆಹಚ್ಚುತ್ತದೆ ಮತ್ತು ಸಂಪರ್ಕ ಫಾರ್ಮ್ ಅನ್ನು ರಚಿಸಲು ಅದನ್ನು ಬಳಸುತ್ತದೆ. ಅನೇಕ ಫಾರ್ಮ್ plugin ಗಳು ಅಳವಡಿಸಲಾಗಿಲ್ಲದಿದ್ದರೆ, Site Builder ಈ ಕೆಳಗಿನ ಕ್ರಮದಲ್ಲಿ ಅವುಗಳಿಗೆ ಆದ್ಯತೆ ನೀಡುತ್ತದೆ:

1. Contact Form 7
2. WPForms
3. Fluent Forms
4. Gravity Forms

ಈ plugin ಗಳಲ್ಲಿ ಯಾವುದೂ ಅಳವಡಿಸದಿದ್ದರೆ, ಸಂಪರ್ಕ ಫಾರ್ಮ್ ರಚಿಸುವ ಮೊದಲು ಒಂದನ್ನು ಅಳವಡಿಸಲು Site Builder ಶಿಫಾರಸು ಮಾಡುತ್ತದೆ.

## Email Notifications {#email-notifications}

Site Builder ರಚಿಸಿದ ಸಂಪರ್ಕ ಫಾರ್ಮ್‌ಗಳು ಡೀಫಾಲ್ಟ್ ಆಗಿ ಸೈಟ್ ಆಡಮಿನ್ ಅನ್ನು ಇಮೇಲ್ ನೋಟಿಫಿಕೇಶನ್‌ಗಳನ್ನು ಕಳುಹಿಸಲು ಕಾನ್ಫಿಗರ್ ಮಾಡಲಾಗಿದೆ. ನೀವು ಸ್ವೀಕರಿಸುವ ಇಮೇಲ್ ವಿಳಾಸ ಮತ್ತು ನೋಟಿಫಿಕೇಶನ್ ಸಂದೇಶವನ್ನು ಕಸ್ಟಮೈಸ್ ಮಾಡಬಹುದು:

1. ನಿಮ್ಮ ಫಾರ್ಮ್ plugin ನ ಆಡ್ಮಿನ್ ಇಂಟರ್‌ಫೇಸ್‌ಗೆ ಹೋಗಿ.
2. Site Builder ರಚಿಸಿದ ಫಾರ್ಮ್ ಅನ್ನು ಹುಡುಕಿ.
3. ಇಮೇಲ್ ನೋಟಿಫಿಕೇಶನ್ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಸಂಪಾದಿಸಿ.

ವಿವರವಾದ ಸೂಚನೆಗಳಿಗಾಗಿ, ನಿಮ್ಮ ಫಾರ್ಮ್ plugin ನ documentation ಅನ್ನು ನೋಡಿ:

- [Contact Form 7 Email Notifications](https://contactform7.com/docs/)
- [WPForms Email Notifications](https://wpforms.com/docs/)
- [Fluent Forms Email Notifications](https://fluentforms.com/docs/)
- [Gravity Forms Email Notifications](https://docs.gravityforms.com/)

## ಉದಾಹರಣೆ ಬಳಕೆಯ ಸಂದರ್ಭಗಳು {#example-use-cases}

### ಸರಳ ಸಂಪರ್ಕ ಫಾರ್ಮ್ {#simple-contact-form}

> "ಹೆಸರು, ಇಮೇಲ್ ಮತ್ತು ಸಂದೇಶ ಕ್ಷೇತ್ರಗಳೊಂದಿಗೆ ಒಂದು ಸರಳ ಸಂಪರ್ಕ ಫಾರ್ಮ್ ರಚಿಸಿ."

### multi-Step ಫಾರ್ಮ್ {#multi-step-form}

> "ಸಂಪರ್ಕ ಮಾಹಿತಿಗಾಗಿ ಮೊದಲ ಹಂತ ಮತ್ತು ಸಂದೇಶ ಮತ್ತು ಆದ್ಯತೆಯ ಸಂಪರ್ಕ ವಿಧಾನಕ್ಕಾಗಿ ಎರಡನೇ ಹಂತದೊಂದಿಗೆ ಸಂಪರ್ಕ ಫಾರ್ಮ್ ರಚಿಸಿ."

### Conditional Logic ಇರುವ ಫಾರ್ಮ್ {#form-with-conditional-logic}

> "ಡ್ರಾಪ್‌ಡೌನ್‌ನಲ್ಲಿ ಬಳಕೆದಾರರ ಆಯ್ಕೆಯ ಆಧಾರದ ಮೇಲೆ ವಿಭಿನ್ನ ಕ್ಷೇತ್ರಗಳನ್ನು ತೋರಿಸುವ ಸಂಪರ್ಕ ಫಾರ್ಮ್ ರಚಿಸಿ."

### File Upload ಇರುವ ಫಾರ್ಮ್ {#form-with-file-upload}

> "ಬಳಕೆದಾರರು ಫೈಲ್ ಅಥವಾ ಲಗತ್ತನ್ನು ಅಪ್‌ಲೋಡ್ ಮಾಡಲು ಅನುಮತಿಸುವ ಸಂಪರ್ಕ ಫಾರ್ಮ್ ರಚಿಸಿ."

:::note

:::
