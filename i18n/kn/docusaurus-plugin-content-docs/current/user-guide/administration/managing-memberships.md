# ಸದಸ್ಯತ್ವಗಳನ್ನು ನಿರ್ವಹಿಸುವುದು

Ultimate Multisite ನಲ್ಲಿ ನೀವು ಸದಸ್ಯತ್ವಗಳನ್ನು ನಿರ್ವಹಿಸಬಹುದು ಮತ್ತು ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿರುವ ಎಲ್ಲಾ ಚಂದಾದಾರಿಕೆಗಳನ್ನು ಪ್ರವೇಶಿಸಬಹುದು. ಯೋಜನೆಗಳು ಮತ್ತು ಉತ್ಪನ್ನಗಳು, ಸೈಟ್‌ಗಳ ಸಂಖ್ಯೆ, ಬಿಲ್ಲಿಂಗ್ ಮೊತ್ತ ಮತ್ತು ಆವರ್ತನ, ಪಾವತಿ ಇತಿಹಾಸ, ಮತ್ತು ಸಮಯದ ಮುದ್ರೆಗಳಂತಹ ಚಂದಾದಾರಿಕೆ ವಿವರಗಳನ್ನು ಪರಿಶೀಲಿಸಿ ಮತ್ತು ಸಂಪಾದಿಸಿ.

ಈ ಪುಟವನ್ನು ಪ್ರವೇಶಿಸಲು, Ultimate Multisite ಮೆನುವಿನಲ್ಲಿ **Memberships** ಟ್ಯಾಬ್‌ಗೆ ಹೋಗಿ.

![Memberships list page](/img/admin/memberships-list.png)

Memberships ಪುಟದಲ್ಲಿ ನಿಮ್ಮ ಗ್ರಾಹಕರ ಪಟ್ಟಿ ಮತ್ತು ಅವರ ಖಾತೆಗಳಿಗೆ ಸಂಬಂಧಿಸಿದ ಉತ್ಪನ್ನಗಳನ್ನು ನೀವು ನೋಡಬಹುದು. ಸ್ಥಿತಿಯ ಆಧಾರದ ಮೇಲೆ ಸದಸ್ಯತ್ವಗಳನ್ನು ಫಿಲ್ಟರ್ ಮಾಡಬಹುದು: ಸಕ್ರಿಯ, ಬಾಕಿ, ಹೋಲ್ಡ್‌ನಲ್ಲಿ, ಅವಧಿ ಮುಗಿದಿದೆ, ಅಥವಾ ರದ್ದಾಗಿದೆ.

ಸದಸ್ಯತ್ವ ವಿವರಗಳನ್ನು ಸಂಪಾದಿಸಬಹುದಾದ ಪುಟವನ್ನು ಪ್ರವೇಶಿಸಲು ಸದಸ್ಯತ್ವದ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ — ಬಿಲ್ಲಿಂಗ್ ಮಾಹಿತಿ, ಸೈಟ್ ಅಥವಾ ಭೇಟಿ ಮಿತಿಗಳು, ಮತ್ತು ಇನ್ನಷ್ಟು.

ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿ ಈಗಾಗಲೇ ನೋಂದಾಯಿಸಲಾದ ಬಳಕೆದಾರರಿಗೆ ಹೊಸ ಸದಸ್ಯತ್ವಗಳನ್ನು ಸಹ ಸೇರಿಸಬಹುದು. **Add Membership** ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ವಿವರಗಳನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಿ.

![Add Membership modal](/img/admin/membership-add-modal.png)

ನಂತರ **Create Membership** ಕ್ಲಿಕ್ ಮಾಡಿ. ಹೊಸ ಸದಸ್ಯತ್ವವು ಸದಸ್ಯತ್ವಗಳ ಪಟ್ಟಿಯಲ್ಲಿ ಕಾಣಿಸುತ್ತದೆ.
