# ಸದಸ್ಯತ್ವಗಳನ್ನು ನಿರ್ವಹಿಸುವುದು

Ultimate Multisite ನಲ್ಲಿ ನೀವು ಸದಸ್ಯತ್ವಗಳನ್ನು ನಿರ್ವಹಿಸಬಹುದು ಮತ್ತು ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿರುವ ಎಲ್ಲಾ ಚಂದಾದಾರಿಕೆಗಳನ್ನು ಪ್ರವೇಶಿಸಬಹುದು. ಯೋಜನೆಗಳು ಮತ್ತು ಉತ್ಪನ್ನಗಳು, ಸೈಟ್‌ಗಳ ಸಂಖ್ಯೆ, ಬಿಲ್ಲಿಂಗ್ ಮೊತ್ತ ಮತ್ತು ಅವಧಿ, ಪಾವತಿ ಇತಿಹಾಸ, ಹಾಗೂ ಸಮಯಮುದ್ರೆಗಳಂತಹ ಚಂದಾದಾರಿಕೆ ವಿವರಗಳನ್ನು ಪರಿಶೀಲಿಸಿ ಮತ್ತು ಸಂಪಾದಿಸಿ.

ಈ ಪುಟವನ್ನು ಪ್ರವೇಶಿಸಲು, Ultimate Multisite ಮೆನುವಿನಲ್ಲಿರುವ **ಸದಸ್ಯತ್ವಗಳು** ಟ್ಯಾಬ್‌ಗೆ ಹೋಗಿ.

![ಸದಸ್ಯತ್ವಗಳ ಪಟ್ಟಿ ಪುಟ](/img/admin/memberships-list.png)

ಸದಸ್ಯತ್ವಗಳ ಪುಟದಲ್ಲಿ, ನಿಮ್ಮ ಗ್ರಾಹಕರ ಪಟ್ಟಿ ಮತ್ತು ಅವರ ಖಾತೆಗಳಿಗೆ ಸಂಬಂಧಿಸಿದ ಉತ್ಪನ್ನಗಳನ್ನು ನೀವು ನೋಡಬಹುದು. ಸದಸ್ಯತ್ವಗಳನ್ನು ಸ್ಥಿತಿಯ ಪ್ರಕಾರ ಫಿಲ್ಟರ್ ಮಾಡಬಹುದು: ಸಕ್ರಿಯ, ಬಾಕಿ, ತಡೆಹಿಡಿದ, ಅವಧಿ ಮುಗಿದ, ಅಥವಾ ರದ್ದುಗೊಂಡ.

ಉಚಿತ ಸದಸ್ಯತ್ವಗಳನ್ನು **ಜೀವಮಾನ** ಸದಸ್ಯತ್ವಗಳಾಗಿ ಪರಿಗಣಿಸಲಾಗುತ್ತದೆ. ಅವುಗಳಿಗೆ ಅವಧಿ ಮುಗಿಯುವ ದಿನಾಂಕ ಇರುವುದಿಲ್ಲ ಮತ್ತು ಅವು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಮುಗಿಯುವುದಿಲ್ಲ. ಪ್ರಯೋಗಾವಧಿ ಅಥವಾ ನಿಗದಿತ ಅವಧಿಯ ನಂತರ ಉಚಿತ ಯೋಜನೆ ಮುಗಿಯಬೇಕಾದರೆ, ಪ್ರಯೋಗಾವಧಿಯೊಂದಿಗೆ ಇರುವ ಪಾವತಿಸಿದ ಉತ್ಪನ್ನವನ್ನು ಬಳಸಿ, ಸದಸ್ಯತ್ವವನ್ನು ಕೈಯಾರೆ ರದ್ದುಮಾಡಿ, ಅಥವಾ ಅವರ ಪ್ರವೇಶ ಬದಲಾಗಬೇಕಾದಾಗ ಗ್ರಾಹಕರನ್ನು ಮತ್ತೊಂದು ಯೋಜನೆಗೆ ಸ್ಥಳಾಂತರಿಸಿ.

ಸದಸ್ಯತ್ವದ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿದರೆ, ಸದಸ್ಯತ್ವದ ವಿವರಗಳನ್ನು ಸಂಪಾದಿಸಬಹುದಾದ ಪುಟಕ್ಕೆ ಹೋಗಬಹುದು — ಬಿಲ್ಲಿಂಗ್ ಮಾಹಿತಿ, ಸೈಟ್ ಅಥವಾ ಭೇಟಿ ಮಿತಿಗಳು, ಮತ್ತು ಇನ್ನಷ್ಟು.

![ಸದಸ್ಯತ್ವ ಸಂಪಾದನೆ ಇಂಟರ್ಫೇಸ್](/img/admin/membership-edit.png)

ಸದಸ್ಯತ್ವ ಸಂಪಾದನೆ ಪುಟದ ಸಂಪೂರ್ಣ ನೋಟ ಇಲ್ಲಿದೆ:

![ಸದಸ್ಯತ್ವ ಸಂಪಾದನೆಯ ಸಂಪೂರ್ಣ ಇಂಟರ್ಫೇಸ್](/img/admin/membership-edit-full.png)

ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿ ಹಿಂದೆ ನೋಂದಾಯಿಸಲಾದ ಬಳಕೆದಾರರಿಗೆ ಸಂಬಂಧಿಸಿದ ಹೊಸ ಸದಸ್ಯತ್ವಗಳನ್ನು ಸಹ ನೀವು ಸೇರಿಸಬಹುದು. **ಸದಸ್ಯತ್ವ ಸೇರಿಸಿ** ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ವಿವರಗಳನ್ನು ಸಂರಚಿಸಿ.

![ಸದಸ್ಯತ್ವ ಸೇರಿಸುವ ಮಾದರಿ](/img/admin/membership-add-modal.png)

ನಂತರ **ಸದಸ್ಯತ್ವ ರಚಿಸಿ** ಕ್ಲಿಕ್ ಮಾಡಿ. ಹೊಸ ಸದಸ್ಯತ್ವವು ಸದಸ್ಯತ್ವಗಳ ಪಟ್ಟಿಯಲ್ಲಿ ಕಾಣಿಸುತ್ತದೆ.
