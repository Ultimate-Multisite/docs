# Ultimate Multisite Dashboard ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವುದು

Ultimate Multisite ನ ಹೆಚ್ಚು ಮುಂದುವರಿದ ಕಾನ್ಫಿಗರೇಶನ್ ಆಯ್ಕೆಗಳು ಮತ್ತು ಕಾರ್ಯಾಚರಣೆಯ ಅಂಶಗಳಿಗೆ ಹೋಗುವ ಮೊದಲು, dashboard ಅನ್ನು ಒಮ್ಮೆ ನೋಡೋಣ.

ಅನೇಕ WordPress plugins ಗಳಂತೆ, Ultimate Multisite ಕೂಡ **Ultimate Multisite** ಎಂಬ ಹೆಸರಿನ network-level ಮೆನು ಎಂಟ್ರಿಯನ್ನು ರಚಿಸುತ್ತದೆ, ಅದರ ಕೆಳಗೆ ಹಲವಾರು sub-menu ಎಂಟ್ರಿಗಳಿರುತ್ತವೆ.

## Network Dashboard Widgets

Ultimate Multisite network-level dashboard ನಲ್ಲಿ ಕೆಲವು ಉಪಯುಕ್ತ widgets ಗಳನ್ನು ಸೇರಿಸುತ್ತದೆ. ಈ dashboard ಅನ್ನು **My Sites → Network Admin → Dashboard** ಗೆ ಹೋಗುವ ಮೂಲಕ ಕಾಣಬಹುದು. ಈ widgets ಗಳನ್ನು network administrator ಗೆ ಮಾಹಿತಿ ಮತ್ತು ಸಾಮಾನ್ಯ ಕಾರ್ಯಾಚರಣೆಗಳಿಗೆ ಸುಲಭ ಪ್ರವೇಶ ನೀಡಲು ವಿನ್ಯಾಸಗೊಳಿಸಲಾಗಿದೆ.

![Ultimate Multisite widgets ಗಳೊಂದಿಗೆ Network Dashboard](/img/admin/network-dashboard-full.png)

### First Steps

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

ಈ widget Ultimate Multisite ನ ಹೊಸ ಇನ್‌ಸ್ಟಾಲೇಶನ್‌ನಲ್ಲಿ ಕಾಣಿಸಿಕೊಳ್ಳುತ್ತದೆ ಮತ್ತು payment method ಕಾನ್ಫಿಗರ್ ಮಾಡುವುದು ಮತ್ತು test account ರಚಿಸುವುದು ಮುಂತಾದ ಹೆಚ್ಚುವರಿ ಕಾರ್ಯಗಳನ್ನು ಪೂರ್ಣಗೊಳಿಸಲು network administrator ಗೆ ನೆನಪಿಸುತ್ತದೆ.

### Summary

![Summary widget](/img/admin/network-dashboard-summary.png)

Summary widget ದಿನದ signups ಮತ್ತು ಅವುಗಳ ಆದಾಯದ ಬಗ್ಗೆ ವರದಿ ಮಾಡುತ್ತದೆ. MRR (Monthly Recurring Revenue) ಆದಾಯ ಉತ್ಪಾದಿಸುವ memberships ಹೊಂದಿರುವ ಗ್ರಾಹಕರ ಅಂದಾಜು ಒಟ್ಟು ಮೊತ್ತವನ್ನು ತೋರಿಸುತ್ತದೆ.

### Activity Stream

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

ಈ widget Ultimate Multisite ನಾದ್ಯಂತ ನಡೆಯುವ ಘಟನೆಗಳ ಬಗ್ಗೆ ವರದಿ ಮಾಡುತ್ತದೆ. ಘಟನೆಗಳಲ್ಲಿ signups, cancellations, site creations, deletions ಮತ್ತು ಇತರ ಚಟುವಟಿಕೆಗಳು ಸೇರಿವೆ.

ಹೆಚ್ಚಿನ ವಿವರಣೆಗಾಗಿ ದಯವಿಟ್ಟು ಡಾಕ್ಯುಮೆಂಟೇಶನ್‌ನ Events ವಿಭಾಗವನ್ನು ನೋಡಿ.

### Right Now

![Right Now widget](/img/admin/network-dashboard-right-now.png)

ಈ utility widget network ನಲ್ಲಿನ users ಮತ್ತು sites ಗಳ ಸಂಕ್ಷಿಪ್ತ ಸಾರಾಂಶವನ್ನು ತೋರಿಸುತ್ತದೆ. ಮೇಲ್ಭಾಗದಲ್ಲಿರುವ ಸುಲಭ shortcuts ಗಳು ಹೊಸ sites ಅಥವಾ users ಗಳನ್ನು ಒಂದೇ ಕ್ಲಿಕ್‌ನಲ್ಲಿ ರಚಿಸಲು ಅನುವು ಮಾಡಿಕೊಡುತ್ತವೆ.

### News and Discussions

![News and Discussions widget](/img/admin/network-dashboard-news.png)

ಈ widget ಇತ್ತೀಚಿನ Ultimate Multisite ಸುದ್ದಿಗಳನ್ನು ಪಡೆದು ತೋರಿಸುತ್ತದೆ. ಅಪ್‌ಡೇಟ್‌ಗಳು, ಮುಖ್ಯ bugs ಮತ್ತು security fixes ಬಗ್ಗೆ ಮಾಹಿತಿಗಾಗಿ ಈ widget ಮೇಲೆ ಗಮನವಿಡಿ.

## Ultimate Multisite Dashboard

Network-level dashboard network ಬಗ್ಗೆ ಸಮಗ್ರ ಮಾಹಿತಿಯನ್ನು ತೋರಿಸಿದರೆ, Ultimate Multisite dashboard (Ultimate Multisite ಮೆನುವಿನ ಮೇಲ್ಮಟ್ಟದಲ್ಲಿದೆ) ಸೇವೆಯ ಬಗ್ಗೆ ಮಾಹಿತಿಯನ್ನು ತೋರಿಸುತ್ತದೆ.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

### Monthly Recurring Revenue Growth

MRR (Monthly Recurring Revenue) ಚಾರ್ಟ್ 12-ತಿಂಗಳ calendar ಅವಧಿಯಲ್ಲಿ ಆದಾಯದ ಮಾಸಿಕ ವಿವರಣೆಯನ್ನು ತೋರಿಸುತ್ತದೆ.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

ಹೊಸ ಆದಾಯವನ್ನು ನೀಲಿ ಬಣ್ಣದಲ್ಲಿ ಟ್ರ್ಯಾಕ್ ಮಾಡಲಾಗುತ್ತದೆ ಮತ್ತು cancellations ಅನ್ನು ಕೆಂಪು ಬಣ್ಣದಲ್ಲಿ ತೋರಿಸಲಾಗುತ್ತದೆ.

### Date-range Support

ಗ್ರಾಹಕರ ಸಂಖ್ಯೆ ಮತ್ತು ಚಟುವಟಿಕೆಯ ಪ್ರಮಾಣವನ್ನು ಅವಲಂಬಿಸಿ, ಅಂಕಿಅಂಶಗಳು ತುಂಬಾ ಹೆಚ್ಚಾಗಬಹುದು. Dashboard ನಲ್ಲಿ ಅಂತರ್ನಿರ್ಮಿತವಾಗಿರುವ date-range support network administrator ಗೆ ಅಗತ್ಯವಿರುವ ಮಾಹಿತಿ ಮತ್ತು ಸಮಯಾವಧಿಯ ಮೇಲೆ ಮಾತ್ರ ಗಮನ ಕೇಂದ್ರೀಕರಿಸಲು ಅನುವು ಮಾಡಿಕೊಡುತ್ತದೆ.

![Date range selector ಮತ್ತು stats](/img/admin/um-dashboard-date-range.png)

Date range ನೇರವಾಗಿ ಕೆಳಗಿರುವ widgets ಗಳು ತೋರಿಸುವ ಮಾಹಿತಿಯ ಅವಧಿಯನ್ನು ನಿಯಂತ್ರಿಸುತ್ತದೆ, ಇದರಲ್ಲಿ revenue, new memberships, signups by country, signups by form ಮತ್ತು most visited sites ಸೇರಿವೆ.

![Dashboard statistics widgets](/img/admin/um-dashboard-stats.png)

:::tip
**Most Visited Sites** widget ಕೆಲಸ ಮಾಡಲು, **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** ನಲ್ಲಿ visit counting ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ.
:::
