# ಡ್ಯಾಶ್‌ಬೋರ್ಡ್ ಅವಲೋಕನ

Ultimate Multisite ನಲ್ಲಿ Ultimate Multisite ಮೆನು ಅಡಿಯಲ್ಲಿ ಪ್ರತ್ಯೇಕ **Dashboard** ಇದೆ, ಮತ್ತು Network Dashboard ಗೆ ವಿಜೆಟ್‌ಗಳನ್ನು ಸಹ ಸೇರಿಸುತ್ತದೆ. ಈ ಲೇಖನದಲ್ಲಿ ಎರಡೂ ಸ್ಥಳಗಳಲ್ಲಿ ನೀವು ಏನನ್ನು ನೋಡುತ್ತೀರಿ ಎಂಬುದನ್ನು ವಿವರಿಸಲಾಗಿದೆ.

## Network Dashboard

Ultimate Multisite **My Sites → Network Admin → Dashboard** ನಲ್ಲಿರುವ ನೆಟ್‌ವರ್ಕ್-ಮಟ್ಟದ ಡ್ಯಾಶ್‌ಬೋರ್ಡ್‌ಗೆ ಉಪಯುಕ್ತ ವಿಜೆಟ್‌ಗಳನ್ನು ಸೇರಿಸುತ್ತದೆ.

![Ultimate Multisite ವಿಜೆಟ್‌ಗಳೊಂದಿಗೆ Network Dashboard](/img/admin/network-dashboard-full.png)

ಪ್ರತಿ ವಿಜೆಟ್ ಏನನ್ನು ಒದಗಿಸುತ್ತದೆ ಎಂಬುದು ಇಲ್ಲಿದೆ:

**First Steps:** ಹೊಸ ಇನ್‌ಸ್ಟಾಲೇಶನ್‌ನಲ್ಲಿ ಕಾಣಿಸಿಕೊಳ್ಳುತ್ತದೆ ಮತ್ತು ಪೇಮೆಂಟ್ ವಿಧಾನವನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡುವುದು ಮತ್ತು ಟೆಸ್ಟ್ ಅಕೌಂಟ್ ರಚಿಸುವಂತಹ ಸೆಟಪ್ ಕಾರ್ಯಗಳನ್ನು ಪೂರ್ಣಗೊಳಿಸಲು ನಿಮಗೆ ನೆನಪಿಸುತ್ತದೆ.

![First Steps ವಿಜೆಟ್](/img/admin/network-dashboard-first-steps.png)

**Summary:** ದಿನದ ಸೈನ್‌ಅಪ್‌ಗಳು ಮತ್ತು ಆದಾಯವನ್ನು ವರದಿ ಮಾಡುತ್ತದೆ. MRR (Monthly Recurring Revenue) ಆದಾಯ-ಉತ್ಪಾದಿಸುವ ಮೆಂಬರ್‌ಶಿಪ್‌ಗಳಿರುವ ಗ್ರಾಹಕರ ಅಂದಾಜು ಒಟ್ಟು ಮೊತ್ತವನ್ನು ಸೂಚಿಸುತ್ತದೆ.

![Summary ವಿಜೆಟ್](/img/admin/network-dashboard-summary.png)

**Activity Stream:** Ultimate Multisite ನಾದ್ಯಂತ ನಡೆಯುವ ಈವೆಂಟ್‌ಗಳನ್ನು ವರದಿ ಮಾಡುತ್ತದೆ — ಸೈನ್‌ಅಪ್‌ಗಳು, ರದ್ದತಿಗಳು, ಸೈಟ್ ರಚನೆಗಳು, ಅಳಿಸುವಿಕೆಗಳು, ಮತ್ತು ಇತರ ಚಟುವಟಿಕೆಗಳು.

![Activity Stream ವಿಜೆಟ್](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿರುವ ಬಳಕೆದಾರರು ಮತ್ತು ಸೈಟ್‌ಗಳ ಸಂಕ್ಷಿಪ್ತ ಸಾರಾಂಶವನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ ಮತ್ತು ಹೊಸ ಸೈಟ್‌ಗಳು ಅಥವಾ ಬಳಕೆದಾರರನ್ನು ರಚಿಸಲು ಶಾರ್ಟ್‌ಕಟ್‌ಗಳನ್ನು ಒದಗಿಸುತ್ತದೆ.

![Right Now ವಿಜೆಟ್](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** ಅಪ್‌ಡೇಟ್‌ಗಳು, ಗಂಭೀರ ಬಗ್‌ಗಳು, ಮತ್ತು ಭದ್ರತಾ ಸರಿಪಡಿಸುವಿಕೆಗಳನ್ನು ಒಳಗೊಂಡಂತೆ ಇತ್ತೀಚಿನ Ultimate Multisite ಸುದ್ದಿಗಳನ್ನು ತರುತ್ತದೆ ಮತ್ತು ಪ್ರದರ್ಶಿಸುತ್ತದೆ.

![News ವಿಜೆಟ್](/img/admin/network-dashboard-news.png)

## Ultimate Multisite Dashboard

Network dashboard ನೆಟ್‌ವರ್ಕ್ ಬಗ್ಗೆ ಸಮಗ್ರ ಮಾಹಿತಿಯನ್ನು ಪ್ರಸ್ತುತಪಡಿಸುತ್ತದೆ, ಆದರೆ Ultimate Multisite dashboard (Ultimate Multisite ಮೆನುವಿನ ಮೇಲ್ಮಟ್ಟದಲ್ಲಿ) ಸೇವೆಯ ಬಗ್ಗೆ ಮಾಹಿತಿಯನ್ನು ಪ್ರಸ್ತುತಪಡಿಸುತ್ತದೆ.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Monthly Recurring Revenue Growth:** 12-ತಿಂಗಳ ಕ್ಯಾಲೆಂಡರ್ ಅವಧಿಯಲ್ಲಿ ಆದಾಯದ ಮಾಸಿಕ ವಿಭಜನೆಯನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ. ಹೊಸ ಆದಾಯವನ್ನು ನೀಲಿ ಬಣ್ಣದಲ್ಲಿ ತೋರಿಸಲಾಗುತ್ತದೆ ಮತ್ತು ರದ್ದತಿಗಳು ಕೆಂಪು ಬಣ್ಣದಲ್ಲಿ ಕಾಣಿಸಿಕೊಳ್ಳುತ್ತವೆ.

![MRR Growth ಚಾರ್ಟ್](/img/admin/um-dashboard-mrr.png)

**Date-range Support:** ನಿಮಗೆ ಬೇಕಾದ ಮಾಹಿತಿಯ ಮೇಲೆ ಗಮನ ಕೇಂದ್ರೀಕರಿಸಲು ಸಮಯಾವಧಿಯ ಪ್ರಕಾರ ಡ್ಯಾಶ್‌ಬೋರ್ಡ್ ಡೇಟಾವನ್ನು ಫಿಲ್ಟರ್ ಮಾಡಿ. ದಿನಾಂಕ ಶ್ರೇಣಿಯು ಕೆಳಗಿನ ವಿಜೆಟ್‌ಗಳಲ್ಲಿ ಏನನ್ನು ಪ್ರದರ್ಶಿಸಲಾಗುತ್ತದೆ ಎಂಬುದನ್ನು ನಿಯಂತ್ರಿಸುತ್ತದೆ.

![ದಿನಾಂಕ ಶ್ರೇಣಿ ಮತ್ತು ಡ್ಯಾಶ್‌ಬೋರ್ಡ್ ಅಂಕಿಅಂಶಗಳು](/img/admin/um-dashboard-date-range.png)

ಡ್ಯಾಶ್‌ಬೋರ್ಡ್ ವಿಜೆಟ್‌ಗಳು ಇವುಗಳನ್ನು ಒಳಗೊಂಡಿವೆ:

- **Revenue** — ಅವಧಿಯ ಒಟ್ಟು ಆದಾಯ MRR ಮತ್ತು ಪ್ರತಿ-ಉತ್ಪನ್ನ ವಿಭಜನೆಗಳೊಂದಿಗೆ
- **New Memberships** — ಆಯ್ಕೆಮಾಡಿದ ಅವಧಿಯಲ್ಲಿ ಎಷ್ಟು ಹೊಸ ಮೆಂಬರ್‌ಶಿಪ್‌ಗಳನ್ನು ರಚಿಸಲಾಯಿತು
- **Signups by Countries** — ಸೈನ್‌ಅಪ್‌ಗಳ ಭೌಗೋಳಿಕ ವಿಭಜನೆ
- **Signups by Forms** — ಎಷ್ಟು ಗ್ರಾಹಕರು ಪ್ರತಿ ಚೆಕ್‌ಔಟ್ ಫಾರ್ಮ್ ಮೂಲಕ ಸೈನ್ ಅಪ್ ಆದರು ಮತ್ತು ಅಡ್ಮಿನ್‌ಗಳು ಎಷ್ಟು ಜನರನ್ನು ಸೇರಿಸಿದರು
- **Most Visited Sites** — ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿ ಅತ್ಯಂತ ಹೆಚ್ಚು ಭೇಟಿ ನೀಡಿದ ಸೈಟ್‌ಗಳು (ಇದಕ್ಕೆ **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** ನಲ್ಲಿ ಭೇಟಿ ಎಣಿಕೆಯನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಬೇಕು)

![ಡ್ಯಾಶ್‌ಬೋರ್ಡ್ ಅಂಕಿಅಂಶಗಳು](/img/admin/um-dashboard-stats.png)
