# ಗ್ರಾಹಕರನ್ನು ನಿರ್ವಹಿಸುವುದು

**Customers** ಪುಟವು ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿ ನೋಂದಾಯಿಸಲಾದ ಸದಸ್ಯರ ಪಟ್ಟಿಯನ್ನು ತೋರಿಸುತ್ತದೆ. ಈ ಪುಟವು ಹೆಸರು, ಇಮೇಲ್ ವಿಳಾಸ, ಕೊನೆಯ ಲಾಗಿನ್ ಸೇರಿದಂತೆ ಮೂಲಭೂತ ಬಳಕೆದಾರ ಮಾಹಿತಿಯನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ, ಮತ್ತು **switch to** ಲಿಂಕ್ ಮೂಲಕ ನೀವು ಬಳಕೆದಾರರ ಸಬ್‌ಸೈಟ್‌ಗೆ ಸೈಟ್ ನಿರ್ವಾಹಕರಾಗಿ ಲಾಗಿನ್ ಆಗಬಹುದು.

![Customers list page](/img/admin/customers-list.png)

ಆ ಗ್ರಾಹಕರ ಬಗ್ಗೆ ವಿವರವಾದ ಮಾಹಿತಿಯನ್ನು ನೋಡಲು ಗ್ರಾಹಕರ ಕಾರ್ಡ್‌ನ ಕೆಳಗೆ **Manage** ಕ್ಲಿಕ್ ಮಾಡಿ.

ಗ್ರಾಹಕರ ವಿವರ ಪುಟದಲ್ಲಿ, **At a Glance** ಬ್ಲಾಕ್ ಗ್ರಾಹಕ ID, ಕೊನೆಯ ಲಾಗಿನ್ ಸಮಯ, ಮತ್ತು ಈ ಗ್ರಾಹಕರು ಖರ್ಚು ಮಾಡಿದ ಒಟ್ಟು ಮೊತ್ತವನ್ನು ತೋರಿಸುತ್ತದೆ.

**Customer Options** ಬ್ಲಾಕ್ ಮೂಲಕ ನೀವು ಈ ಗ್ರಾಹಕರನ್ನು **VIP** ಆಗಿ ಹೊಂದಿಸಬಹುದು (ಅವರನ್ನು ಸುಲಭವಾಗಿ ಹುಡುಕಲು ಒಂದು ಟ್ಯಾಗ್). **Billing Info** ಟ್ಯಾಬ್ ಇಮೇಲ್, ವಿಳಾಸ, ಮತ್ತು ದೇಶದಂತಹ ಬಿಲ್ಲಿಂಗ್ ಮಾಹಿತಿಯನ್ನು ತೋರಿಸುತ್ತದೆ.

**Fields** ಬ್ಲಾಕ್ ಗ್ರಾಹಕರ ಬಳಕೆದಾರಹೆಸರು, ಇಮೇಲ್ ತೋರಿಸುತ್ತದೆ, ಮತ್ತು **Email Verification** ಸ್ಥಿತಿಯನ್ನು ಬದಲಾಯಿಸಲು ಅನುಮತಿಸುತ್ತದೆ.

Customer Options ಬ್ಲಾಕ್‌ನ ಕೆಳಗೆ ನೀವು ನಾಲ್ಕು ಹೆಚ್ಚುವರಿ ಬ್ಲಾಕ್‌ಗಳನ್ನು ಕಾಣಬಹುದು:

- **Memberships** — ಗ್ರಾಹಕರು ಚಂದಾದಾರರಾಗಿರುವ ಪ್ರಸ್ತುತ ಸಬ್‌ಸ್ಕ್ರಿಪ್ಷನ್‌ಗಳು
- **Payments** — ಗ್ರಾಹಕರ ಪಾವತಿ ಇತಿಹಾಸ
- **Sites** — ಈ ಗ್ರಾಹಕರ ಒಡೆತನದಲ್ಲಿರುವ ಸೈಟ್‌ಗಳು
- **Events** — ಈ ಗ್ರಾಹಕರಿಗೆ ಸಂಬಂಧಿಸಿದ ಘಟನೆಗಳು (ಇಮೇಲ್ ಪರಿಶೀಲನೆ, ಬಿಲ್ಲಿಂಗ್ ಬದಲಾವಣೆಗಳು, ಇತ್ಯಾದಿ)

ಯಾವುದೇ ಐಟಂನ ವಿವರ ಪುಟಕ್ಕೆ ಹೋಗಲು ಅದರ ಪಕ್ಕದಲ್ಲಿರುವ ಬಾಣದ ಐಕಾನ್ (→) ಕ್ಲಿಕ್ ಮಾಡಿ. ಉದಾಹರಣೆಗೆ, ಸೈಟ್‌ನ ಪಕ್ಕದಲ್ಲಿರುವ ಬಾಣವನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿದರೆ ಸೈಟ್ ಎಡಿಟ್ ಪುಟಕ್ಕೆ ಹೋಗುತ್ತದೆ, ಅಲ್ಲಿ ನೀವು ಮೆಂಬರ್‌ಶಿಪ್ ಸಂಯೋಜನೆ, ಸೈಟ್ ಪ್ರಕಾರ, ಮತ್ತು ಸಕ್ರಿಯ ಸ್ಥಿತಿಯನ್ನು ಮಾರ್ಪಡಿಸಬಹುದು.

ಮೆಂಬರ್‌ಶಿಪ್‌ಗಳು ಮತ್ತು ಪಾವತಿಗಳಿಗೂ ಇದೇ ಅನ್ವಯಿಸುತ್ತದೆ — ಪ್ರತಿಯೊಂದೂ ವಿವರ ಪುಟಗಳಿಗೆ ಲಿಂಕ್ ಆಗಿರುತ್ತದೆ, ಅಲ್ಲಿ ನೀವು ಮೆಂಬರ್‌ಶಿಪ್ ಸ್ಥಿತಿ, ಸಂಬಂಧಿತ ಗ್ರಾಹಕ, ಉತ್ಪನ್ನಗಳು, payment gateway ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಎಡಿಟ್ ಮಾಡಬಹುದು, ಮತ್ತು ಮರುಪಾವತಿಗಳನ್ನು ಸಹ ಪ್ರಕ್ರಿಯೆಗೊಳಿಸಬಹುದು.

ಗ್ರಾಹಕ ಪುಟದ ಬಲಭಾಗದಲ್ಲಿ, ನೀವು ಇವುಗಳನ್ನು ಕಾಣಬಹುದು:

- **Last Login & IPs** — ಕೊನೆಯ ಲಾಗಿನ್ ಸಮಯ, IP ವಿಳಾಸ, ಮತ್ತು ದೇಶ
- **Timestamps** — ಗ್ರಾಹಕ ಖಾತೆಯನ್ನು ರಚಿಸಿದ ಮತ್ತು ಕೊನೆಯದಾಗಿ ಮಾರ್ಪಡಿಸಿದ ಸಮಯ
- **Delete** — ಗ್ರಾಹಕರನ್ನು ಶಾಶ್ವತವಾಗಿ ಅಳಿಸುವ ಆಯ್ಕೆ

:::warning
ಗ್ರಾಹಕರನ್ನು ಅಳಿಸುವುದು ಶಾಶ್ವತವಾಗಿದೆ ಮತ್ತು ಅದನ್ನು ರದ್ದುಗೊಳಿಸಲು ಸಾಧ್ಯವಿಲ್ಲ.
:::
