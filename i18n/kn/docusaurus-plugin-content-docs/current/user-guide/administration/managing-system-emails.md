# ಸಿಸ್ಟಮ್ ಇಮೇಲ್‌ಗಳ ನಿರ್ವಹಣೆ

Ultimate Multisite ಸೈನ್‌ಅಪ್ ದೃಢೀಕರಣ, ಪಾವತಿ ರಸೀದಿ, ಮತ್ತು ಸದಸ್ಯತ್ವ ಬದಲಾವಣೆಗಳಂತಹ ವಿವಿಧ ಈವೆಂಟ್‌ಗಳಿಗೆ ಸಿಸ್ಟಮ್ ಇಮೇಲ್‌ಗಳನ್ನು ಕಳುಹಿಸುತ್ತದೆ. ನಿಮ್ಮ ಬ್ರ್ಯಾಂಡ್‌ಗೆ ಹೊಂದಿಕೆಯಾಗುವಂತೆ ಈ ಇಮೇಲ್ ಟೆಂಪ್ಲೇಟ್‌ಗಳನ್ನು ನೀವು ಕಸ್ಟಮೈಜ್ ಮಾಡಬಹುದು.

## ಸಿಸ್ಟಮ್ ಇಮೇಲ್‌ಗಳನ್ನು ವೀಕ್ಷಿಸುವುದು

ಇಮೇಲ್-ಸಂಬಂಧಿತ ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ನೋಡಲು **Ultimate Multisite → Settings** ಗೆ ಹೋಗಿ ಮತ್ತು **Emails** ಟ್ಯಾಬ್ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ.

![Settings - Emails tab](/img/admin/settings-emails.png)

ಅಲ್ಲಿಂದ, ಎಲ್ಲಾ ಇಮೇಲ್ ಟೆಂಪ್ಲೇಟ್‌ಗಳನ್ನು ನೋಡಲು **System Emails** ಪುಟಕ್ಕೆ ಹೋಗಬಹುದು.

![System Emails list](/img/admin/system-emails-list.png)

## ಹೊಸ ಇಮೇಲ್ ಟೆಂಪ್ಲೇಟ್‌ಗಳನ್ನು ಇಂಪೋರ್ಟ್ ಮಾಡುವುದು

Ultimate Multisite ನ ಹೊಸ ಆವೃತ್ತಿಗಳು ಮತ್ತು add-on ಗಳು ಹೊಸ ಇಮೇಲ್ ಟೆಂಪ್ಲೇಟ್‌ಗಳನ್ನು ನೋಂದಾಯಿಸಬಹುದು. ಸಮಸ್ಯೆಗಳನ್ನು ತಡೆಯಲು, ಹೊಸ ಟೆಂಪ್ಲೇಟ್‌ಗಳು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸೇರಿಸಲ್ಪಡುವುದಿಲ್ಲ — ನೀವು ಅವುಗಳನ್ನು ಹಸ್ತಚಾಲಿತವಾಗಿ ಇಂಪೋರ್ಟ್ ಮಾಡಬೇಕು.

ಹೊಸ ಇಮೇಲ್ ಟೆಂಪ್ಲೇಟ್‌ಗಳನ್ನು ಇಂಪೋರ್ಟ್ ಮಾಡಲು:

1. **System Emails** ಪುಟಕ್ಕೆ ಹೋಗಿ
2. ಪುಟದ ಮೇಲ್ಭಾಗದಲ್ಲಿರುವ **Reset & Import** ಬಟನ್ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ
3. ಲಭ್ಯವಿರುವ ಟೆಂಪ್ಲೇಟ್‌ಗಳನ್ನು ನೋಡಲು **Import Emails** ಆಯ್ಕೆಯನ್ನು ಟಾಗಲ್ ಮಾಡಿ
4. ನೀವು ಇಂಪೋರ್ಟ್ ಮಾಡಲು ಬಯಸುವ ಟೆಂಪ್ಲೇಟ್‌ಗಳನ್ನು ಆಯ್ಕೆ ಮಾಡಿ ಮತ್ತು ದೃಢೀಕರಿಸಿ

## ಇಮೇಲ್ ಟೆಂಪ್ಲೇಟ್‌ಗಳನ್ನು ರೀಸೆಟ್ ಮಾಡುವುದು

ನೀವು ಇಮೇಲ್ ಟೆಂಪ್ಲೇಟ್ ಅನ್ನು ಕಸ್ಟಮೈಜ್ ಮಾಡಿದ್ದರೆ ಮತ್ತು ಡೀಫಾಲ್ಟ್ ವಿಷಯಕ್ಕೆ ಹಿಂತಿರುಗಲು ಬಯಸಿದರೆ, ನಿಮಗೆ ಎರಡು ಆಯ್ಕೆಗಳಿವೆ:

1. **ಅಳಿಸಿ ಮತ್ತು ಮರು-ಇಂಪೋರ್ಟ್ ಮಾಡಿ** — ಸಿಸ್ಟಮ್ ಇಮೇಲ್ ಅನ್ನು ಅಳಿಸಿ ಮತ್ತು ಮತ್ತೆ ಇಂಪೋರ್ಟ್ ಮಾಡಿ (ಇದು ಕಳುಹಿಸಿದ ಮೆಟ್ರಿಕ್ಸ್ ಅನ್ನು ಅಳಿಸುತ್ತದೆ)
2. **ಟೂಲ್ ಮೂಲಕ ರೀಸೆಟ್ ಮಾಡಿ** — **Reset & Import** ಟೂಲ್ ಬಳಸಿ, **Reset** ಆಯ್ಕೆಯನ್ನು ಟಾಗಲ್ ಮಾಡಿ, ಮತ್ತು ನೀವು ರೀಸೆಟ್ ಮಾಡಲು ಬಯಸುವ ಇಮೇಲ್‌ಗಳನ್ನು ಆಯ್ಕೆ ಮಾಡಿ

Reset ಆಯ್ಕೆಯು ಉತ್ತಮವಾಗಿದೆ ಏಕೆಂದರೆ ಇದು ಡೀಫಾಲ್ಟ್ ವಿಷಯವನ್ನು ಮರುಸ್ಥಾಪಿಸುವಾಗ ಕಳುಹಿಸಿದ ಮೆಟ್ರಿಕ್ಸ್ ಮತ್ತು ಇತರ ಮೆಟಾಡೇಟಾವನ್ನು ಉಳಿಸಿಕೊಳ್ಳುತ್ತದೆ.
