# ಸೈಟ್‌ಗಳನ್ನು ನಿರ್ವಹಿಸುವುದು

ಸೈಟ್‌ಗಳು (ಅಥವಾ ಸಬ್‌ಸೈಟ್‌ಗಳು) ನಿಮ್ಮ WaaS ವ್ಯವಹಾರದ ಮೂಲ ಆಧಾರ. Ultimate Multisite ನಲ್ಲಿ 3 ವಿಧದ ಸೈಟ್‌ಗಳಿವೆ:

- **ಗ್ರಾಹಕರ ಒಡೆತನದ** — ನಿರ್ದಿಷ್ಟ ಗ್ರಾಹಕರಿಗೆ ಸೇರಿದ ಸೈಟ್‌ಗಳು
- **Site Templates** — ಗ್ರಾಹಕರು ಪ್ರಾರಂಭಿಕ ಹಂತವಾಗಿ ಆಯ್ಕೆ ಮಾಡಬಹುದಾದ ಪೂರ್ವ-ವ್ಯಾಖ್ಯಾನಿತ ಸೈಟ್‌ಗಳು
- **Main Site** — ನಿಮ್ಮ ಪ್ರಾಥಮಿಕ ನೆಟ್‌ವರ್ಕ್ ಸೈಟ್

## ಸೈಟ್‌ಗಳನ್ನು ವೀಕ್ಷಿಸುವುದು

ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿರುವ ಎಲ್ಲಾ ಸಬ್‌ಸೈಟ್‌ಗಳನ್ನು ನೋಡಲು **Ultimate Multisite → Sites** ಗೆ ಹೋಗಿ. ಪ್ರತಿ ಸೈಟ್ ಅನ್ನು ಅದರ ವಿಧದ ಪ್ರಕಾರ ಲೇಬಲ್ ಮಾಡಲಾಗಿದೆ ಮತ್ತು ನೀವು All Sites, Customer-Owned, Templates, ಅಥವಾ Pending ಮೂಲಕ ಫಿಲ್ಟರ್ ಮಾಡಬಹುದು.

![ಸೈಟ್‌ಗಳ ಪಟ್ಟಿ ಪುಟ](/img/admin/sites-list.png)

## ಹೊಸ ಸೈಟ್ ಸೇರಿಸುವುದು

ಹೊಸ ಸೈಟ್ ರಚಿಸಲು **Add Site** ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ. ನೀವು ಈ ಕೆಳಗಿನ ಮಾಹಿತಿ ನೀಡಬೇಕಾಗುತ್ತದೆ:

- **Site Title** — ಹೊಸ ಸೈಟ್‌ನ ಹೆಸರು
- **Site URL** — ಸೈಟ್‌ನ domain/path
- **Site Type** — ಇದು ಗ್ರಾಹಕ ಸೈಟ್, template, ಅಥವಾ ಸಾಮಾನ್ಯ ಸೈಟ್ ಎಂಬುದು

**Copy Site** ಆಯ್ಕೆಯು ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ site template ಆಧಾರದ ಮೇಲೆ ಹೊಸ ಸೈಟ್ ರಚಿಸಲು ಅನುಮತಿಸುತ್ತದೆ. ಇದನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿದಾಗ, ಯಾವ template ಅನ್ನು ಪ್ರಾರಂಭಿಕ ಹಂತವಾಗಿ ಬಳಸಬೇಕೆಂದು ಆಯ್ಕೆ ಮಾಡಬಹುದು. ಮೀಡಿಯಾ ಫೈಲ್‌ಗಳನ್ನು ಸೇರಿಸಲು **Copy Media on Duplication** ಅನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿರುವುದನ್ನು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ.

## ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ ಸೈಟ್ ನಿರ್ವಹಣೆ

**Edit Site** ಪುಟ ತೆರೆಯಲು ಯಾವುದೇ ಸೈಟ್‌ನ **Manage** ಕ್ಲಿಕ್ ಮಾಡಿ. ಇಲ್ಲಿ ನೀವು ಕಾಣುವುದು:

### ಮೂಲಭೂತ ಮಾಹಿತಿ

ಸೈಟ್ ಹೆಸರು, ವಿಧ, site ID, ಮತ್ತು ವಿವರಣೆ. ಮ್ಯಾಪ್ ಮಾಡಿದ domain, ಸಂಯೋಜಿತ membership, ಮತ್ತು ಸೈಟ್‌ನ ಮಾಲೀಕತ್ವ ಹೊಂದಿರುವ ಗ್ರಾಹಕ ಖಾತೆಯನ್ನೂ ನೀವು ನೋಡುತ್ತೀರಿ.

### ಸೈಟ್ ಆಯ್ಕೆಗಳು

ಸೈಟ್‌ನ ಸಾಮರ್ಥ್ಯಗಳು ಮತ್ತು ಮಿತಿಗಳನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಿ:

- **Visit limits** — ಸೈಟ್ ಭೇಟಿಗಳ ಗರಿಷ್ಠ ಸಂಖ್ಯೆ
- **User account limits** — ಪ್ರತಿ user role ಗೆ ಮಿತಿಗಳು
- **Disk space** — ಸೈಟ್‌ಗೆ ಸಂಗ್ರಹಣಾ ಕೋಟಾ
- **Custom domain** — ಈ ಸೈಟ್‌ಗೆ domain mapping ಸಕ್ರಿಯಗೊಳಿಸಿ
- **Plugin and theme visibility** — ಯಾವ plugins ಮತ್ತು themes ಗೋಚರ, ಮರೆಮಾಡಿದ, ಅಥವಾ ಪೂರ್ವ-ಸಕ್ರಿಯಗೊಳಿಸಲಾಗಿದೆ ಎಂಬುದನ್ನು ನಿಯಂತ್ರಿಸಿ

ಪೂರ್ವನಿಯೋಜಿತವಾಗಿ, ಸೈಟ್‌ಗಳು membership ಮಟ್ಟದಲ್ಲಿ ಹೊಂದಿಸಿದ ಮಿತಿಗಳನ್ನು ಅನುಸರಿಸುತ್ತವೆ. ಸೈಟ್ ಮಟ್ಟದಲ್ಲಿ ಮಿತಿಗಳನ್ನು ಹೊಂದಿಸುವುದು membership ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಅತಿಕ್ರಮಿಸುತ್ತದೆ.

### ಸಂಯೋಜನೆಗಳು

ಸೈಟ್ ಆಯ್ಕೆಗಳ ಕೆಳಗೆ ಈ ಮಾಹಿತಿ ಕಾಣಸಿಗುತ್ತದೆ:

- ಸೈಟ್‌ಗೆ ಸಂಯೋಜಿತ **Mapped domains**
- ಸೈಟ್ ಯಾವ **Membership** ಅಡಿಯಲ್ಲಿದೆ
- ಸೈಟ್‌ಗೆ ಲಿಂಕ್ ಮಾಡಿದ **Customer account**

### ಬಲ ಸೈಡ್‌ಬಾರ್

ಬಲ ಭಾಗದಲ್ಲಿ ನೀವು ಇವುಗಳನ್ನು ಮಾಡಬಹುದು:

- Toggle ಬಟನ್‌ನೊಂದಿಗೆ **ಸೈಟ್ ಸಕ್ರಿಯ/ನಿಷ್ಕ್ರಿಯಗೊಳಿಸಿ**
- **ಸೈಟ್ ವಿಧವನ್ನು ಬದಲಾಯಿಸಿ** ಅಥವಾ ಮಾಲೀಕತ್ವವನ್ನು ಪುನಃ ನಿಯೋಜಿಸಿ
- **ಸೈಟ್ ಚಿತ್ರ/ಥಂಬ್‌ನೇಲ್ ಹೊಂದಿಸಿ** (site templates ಗಾಗಿ frontend ನಲ್ಲಿ ಪ್ರದರ್ಶಿಸಲಾಗುತ್ತದೆ)
- **ಸೈಟ್ ಅನ್ನು ಶಾಶ್ವತವಾಗಿ ಅಳಿಸಿ**

:::warning
ಸೈಟ್ ಅಳಿಸುವುದು ಹಿಂತಿರುಗಿಸಲಾಗದ ಕ್ರಿಯೆ. ಸೈಟ್ ಮತ್ತು ಅದರ ಎಲ್ಲಾ ವಿಷಯಗಳನ್ನು ಶಾಶ್ವತವಾಗಿ ತೆಗೆದುಹಾಕಲಾಗುತ್ತದೆ.
:::
