# ನನ್ನ ಅಕೌಂಟ್ ಪೇಜ್ ಅನ್ನು ಮೊದಲ ಬಾರಿಗೆ ಹೇಗೆ ಪ್ರವೇಶಿಸಬಹುದು?

Ultimate Multisite ಗೆ ಸ್ವಾಗತ!

ನಮ್ಮ plugin ಖರೀದಿಸಿದ ನಂತರ ನಿಮ್ಮ ಇ-ಮೇಲ್‌ನಲ್ಲಿ ಬಂದ ಲಿಂಕ್ ಮೂಲಕ ನಿಮ್ಮ ಅಕೌಂಟ್ ಪೇಜ್ ಅನ್ನು ಪ್ರವೇಶಿಸಿ:

<!-- Screenshot unavailable: Account page email with access link -->

ಖರೀದಿ ಸಮಯದಲ್ಲಿ ನೀವು ನೀಡಿದ ಇಮೇಲ್ ಮತ್ತು ನಿಮ್ಮ ಇಮೇಲ್‌ನಲ್ಲಿ ಬಂದ ಪಾಸ್‌ವರ್ಡ್ ಬಳಸಿ.

<!-- Screenshot unavailable: Login page with email and password fields -->

ಈ ಅಕೌಂಟ್‌ಗೆ ನೀವು ಮೊದಲ ಬಾರಿಗೆ ಲಾಗಿನ್ ಆಗುತ್ತಿರುವುದರಿಂದ, ನಿಮ್ಮ ಪಾಸ್‌ವರ್ಡ್ ಬದಲಾಯಿಸಲು ಕೇಳಲಾಗುತ್ತದೆ. ಅದನ್ನು ಬದಲಾಯಿಸಿ ಮತ್ತು _Save_ ಬಟನ್ ಒತ್ತಿ.

<!-- Screenshot unavailable: Change password form with Save button -->

ನಿಮ್ಮ ಅಕೌಂಟ್ ಪೇಜ್‌ನಲ್ಲಿ, ಬಿಲ್ಲಿಂಗ್‌ಗಳು, ನವೀಕರಣಗಳು, ರದ್ದತಿಗಳು ಮತ್ತು ಇನ್ನೂ ಅನೇಕ ವಿಷಯಗಳಂತಹ ನಿಮ್ಮ ಎಲ್ಲಾ ಅಕೌಂಟ್ ವಿವರಗಳನ್ನು ನಿರ್ವಹಿಸಬಹುದು.
