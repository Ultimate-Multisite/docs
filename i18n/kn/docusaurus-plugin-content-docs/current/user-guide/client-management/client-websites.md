# ನಿಮ್ಮ ಗ್ರಾಹಕರ ವೆಬ್‌ಸೈಟ್‌ಗಳು (v2)

ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿ ನೋಂದಣಿಯಾದ ನಂತರ, ನಿಮ್ಮ ಗ್ರಾಹಕರು ತಮ್ಮ dashboard ಅನ್ನು ಪ್ರವೇಶಿಸಬಹುದು.

**Account** ಪುಟದ ಜೊತೆಗೆ, ಅಲ್ಲಿ ಅವರು ತಮ್ಮ ಪಾವತಿಗಳು, ಸದಸ್ಯತ್ವಗಳು, ಡೊಮೇನ್‌ಗಳು, ಪ್ಲಾನ್ ಮಿತಿಗಳು ಮತ್ತು ಇನ್ನಷ್ಟು ವಿಷಯಗಳ ಬಗ್ಗೆ ಮುಖ್ಯ ಮಾಹಿತಿಯನ್ನು ನೋಡಬಹುದು, ಅವರಿಗೆ **My Sites** ಪುಟಕ್ಕೂ ಪ್ರವೇಶವಿದೆ, ಅಲ್ಲಿ ಅವರು ತಮ್ಮ ಎಲ್ಲಾ ವೆಬ್‌ಸೈಟ್‌ಗಳನ್ನು ನೋಡಬಹುದು ಮತ್ತು ನಿರ್ವಹಿಸಬಹುದು. ಅವರು ತಮ್ಮ dashboard ನಲ್ಲಿ My Sites ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿದರೆ ಸಾಕು.

![ಗ್ರಾಹಕ dashboard ನಲ್ಲಿ My Sites ಲಿಂಕ್](/img/admin/customers-list.png)

ಅವರಿಗೆ ತಮ್ಮ ಎಲ್ಲಾ ವೆಬ್‌ಸೈಟ್‌ಗಳು ತೋರಿಸಲಾಗುತ್ತದೆ, ಪ್ರತಿಯೊಂದರ ಕೆಳಗೆ **Manage** ಬಟನ್ ಇರುತ್ತದೆ. ಅದರ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿದರೆ, ಅವರು **ಆ ನಿರ್ದಿಷ್ಟ ವೆಬ್‌ಸೈಟ್‌ನ dashboard** ಗೆ ಹೋಗುತ್ತಾರೆ.

ಒಂದಕ್ಕಿಂತ ಹೆಚ್ಚು ವೆಬ್‌ಸೈಟ್‌ಗಳನ್ನು ಹೊಂದಲು ಅನುಮತಿಸುವ ಪ್ಲಾನ್‌ಗಳಲ್ಲಿ, ತಮ್ಮ ಯಾವ ವೆಬ್‌ಸೈಟ್ **primary** ಆಗಿದೆ ಎಂಬುದನ್ನು ಸಹ ಅವರು ನೋಡಬಹುದು.

![Manage ಬಟನ್‌ಗಳೊಂದಿಗೆ ಗ್ರಾಹಕ ವೆಬ್‌ಸೈಟ್‌ಗಳನ್ನು ತೋರಿಸುವ My Sites ಪುಟ](/img/admin/customers-list.png)

ತಮ್ಮ primary ವೆಬ್‌ಸೈಟ್ ಅನ್ನು ಬದಲಾಯಿಸಲು ಬಯಸಿದರೆ, **Account** ಮೆನು ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ Account ಪುಟಕ್ಕೆ ಹೋಗಬೇಕು.

![ಗ್ರಾಹಕ dashboard ನಲ್ಲಿ Account ಮೆನು ಲಿಂಕ್](/img/admin/customers-list.png)

ನಂತರ, **Change Default Site** ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಬೇಕು.

![Account ಪುಟದಲ್ಲಿ Change Default Site ಬಟನ್](/img/admin/customers-list.png)

ಒಂದು ಪಾಪ್-ಅಪ್ ವಿಂಡೋ ಕಾಣಿಸಿಕೊಳ್ಳುತ್ತದೆ. ಡ್ರಾಪ್-ಡೌನ್ ಲಿಸ್ಟ್ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ಯಾವ ವೆಬ್‌ಸೈಟ್ ಅನ್ನು primary ವೆಬ್‌ಸೈಟ್ ಆಗಿ ಮಾಡಲು ಬಯಸುತ್ತೀರಿ ಎಂಬುದನ್ನು ಆಯ್ಕೆ ಮಾಡಿ.

![Change Default Site ಡ್ರಾಪ್‌ಡೌನ್ ಆಯ್ಕೆ](/img/admin/customers-list.png)

ನಂತರ, **Change Default Site** ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ ಮತ್ತು ಮುಗಿಯಿತು!

![Default site ಬದಲಾವಣೆಯನ್ನು ದೃಢೀಕರಿಸುವುದು](/img/admin/customers-list.png)
