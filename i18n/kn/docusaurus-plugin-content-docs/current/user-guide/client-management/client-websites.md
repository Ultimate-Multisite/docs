# ನಿಮ್ಮ ಕ್ಲೈಂಟ್‌ನ ವೆಬ್‌ಸೈಟ್‌ಗಳು (v2)

ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿ ನೋಂದಾಯಿಸಿದ ನಂತರ, ನಿಮ್ಮ ಗ್ರಾಹಕರು ತಮ್ಮ Dashboard ಅನ್ನು ಪ್ರವೇಶಿಸಬಹುದು.

**Account** ಪುಟದ ಹೊರತಾಗಿ, ಅಲ್ಲಿ ಅವರು ತಮ್ಮ ಪಾವತಿಗಳು, ಸದಸ್ಯತ್ವಗಳು, ಡೊಮೇನ್‌ಗಳು, ಯೋಜನೆಯ ಮಿತಿಗಳು ಮತ್ತು ಇನ್ನಷ್ಟು ಕುರಿತು ಪ್ರಮುಖ ಮಾಹಿತಿಯನ್ನು ನೋಡಬಹುದು. ಅವರಿಗೆ **ನನ್ನ ಸೈಟ್‌ಗಳು** ಪುಟಕ್ಕೂ ಪ್ರವೇಶವಿದೆ, ಅಲ್ಲಿ ಅವರು ತಮ್ಮ ಎಲ್ಲಾ ವೆಬ್‌ಸೈಟ್‌ಗಳನ್ನು ನೋಡಿ ನಿರ್ವಹಿಸಬಹುದು. ಅವರು ಮಾಡಬೇಕಾದದ್ದು ತಮ್ಮ Dashboard ನಲ್ಲಿ ನನ್ನ ಸೈಟ್‌ಗಳು ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡುವುದು ಮಾತ್ರ.

![ಗ್ರಾಹಕರ Dashboard ನಲ್ಲಿ ನನ್ನ ಸೈಟ್‌ಗಳು ಲಿಂಕ್](/img/account-page/my-sites-link.png)

ಅವರಿಗೆ ಅವರ ಎಲ್ಲಾ ವೆಬ್‌ಸೈಟ್‌ಗಳನ್ನು ತೋರಿಸಲಾಗುತ್ತದೆ, ಪ್ರತಿಯೊಂದರ ಕೆಳಗೆ **ನಿರ್ವಹಿಸಿ** ಬಟನ್ ಇರುತ್ತದೆ. ಅವರು ಅದರ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿದರೆ, ಅವರನ್ನು **ಈ ನಿರ್ದಿಷ್ಟ ವೆಬ್‌ಸೈಟ್‌ನ Dashboard** ಗೆ ಮರುನಿರ್ದೇಶಿಸಲಾಗುತ್ತದೆ.

ಒಂದಕ್ಕಿಂತ ಹೆಚ್ಚು ವೆಬ್‌ಸೈಟ್‌ಗಳನ್ನು ಹೊಂದಲು ಅವಕಾಶ ನೀಡುವ ಯೋಜನೆಗಳಲ್ಲಿ, ಅವರ ವೆಬ್‌ಸೈಟ್‌ಗಳಲ್ಲಿ ಯಾವುದು **ಪ್ರಾಥಮಿಕ** ಎಂದು ಕೂಡ ಅವರು ನೋಡಬಹುದು.

![ನಿರ್ವಹಿಸಿ ಬಟನ್‌ಗಳೊಂದಿಗೆ ಗ್ರಾಹಕರ ವೆಬ್‌ಸೈಟ್‌ಗಳನ್ನು ತೋರಿಸುವ ನನ್ನ ಸೈಟ್‌ಗಳು ಪುಟ](/img/account-page/my-sites-page.png)

ಸೈಟ್ ಸಂಪಾದನೆ ಇಂಟರ್ಫೇಸ್‌ನ ಸಂಪೂರ್ಣ ನೋಟ ಇಲ್ಲಿದೆ:

![ಸೈಟ್ ಸಂಪಾದನೆಯ ಸಂಪೂರ್ಣ ಇಂಟರ್ಫೇಸ್](/img/admin/site-edit-full.png)

ಅವರು ತಮ್ಮ ಪ್ರಾಥಮಿಕ ವೆಬ್‌ಸೈಟ್ ಅನ್ನು ಬದಲಾಯಿಸಲು ಬಯಸಿದರೆ, **Account** ಮೆನು ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡುವ ಮೂಲಕ Account ಪುಟಕ್ಕೆ ಹೋಗಬೇಕು.

![ಗ್ರಾಹಕರ Dashboard ನಲ್ಲಿ Account ಮೆನು ಲಿಂಕ್](/img/account-page/account-menu.png)

ನೀವು ಗ್ರಾಹಕರ ವಿವರಗಳನ್ನು ನೋಡಬಹುದು ಮತ್ತು ಸಂಪಾದಿಸಬಹುದು:

![ಗ್ರಾಹಕ ಸಂಪಾದನೆ ಇಂಟರ್ಫೇಸ್](/img/admin/customer-edit.png)

ನಂತರ, ಅವರು **ಡೀಫಾಲ್ಟ್ ಸೈಟ್ ಬದಲಾಯಿಸಿ** ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಬೇಕು.

![Account ಪುಟದಲ್ಲಿನ ಡೀಫಾಲ್ಟ್ ಸೈಟ್ ಬದಲಾಯಿಸಿ ಬಟನ್](/img/account-page/change-default-site.png)

ಒಂದು ಪಾಪ್-ಅಪ್ ವಿಂಡೋ ಕಾಣಿಸುತ್ತದೆ. ಡ್ರಾಪ್-ಡೌನ್ ಪಟ್ಟಿಯ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ, ಯಾವ ವೆಬ್‌ಸೈಟ್ ಅನ್ನು ಪ್ರಾಥಮಿಕ ವೆಬ್‌ಸೈಟ್ ಮಾಡಬೇಕೆಂದು ಆಯ್ಕೆಮಾಡಿ.

![ಡೀಫಾಲ್ಟ್ ಸೈಟ್ ಬದಲಾಯಿಸಿ ಡ್ರಾಪ್-ಡೌನ್ ಆಯ್ಕೆ](/img/account-page/change-default-site-dropdown.png)

ನಂತರ, **ಡೀಫಾಲ್ಟ್ ಸೈಟ್ ಬದಲಾಯಿಸಿ** ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿದರೆ ಸಾಕು, ಕೆಲಸ ಮುಗಿಯಿತು!

![ಡೀಫಾಲ್ಟ್ ಸೈಟ್ ಬದಲಾವಣೆಯನ್ನು ದೃಢೀಕರಿಸುವುದು](/img/account-page/change-default-site-confirm.png)
