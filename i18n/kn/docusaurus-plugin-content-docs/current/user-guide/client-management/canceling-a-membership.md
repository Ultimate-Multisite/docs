# ಸದಸ್ಯತ್ವವನ್ನು ರದ್ದುಪಡಿಸುವುದು (v2) {#canceling-a-membership-v2}

_**ಪ್ರಮುಖ ಸೂಚನೆ: ಈ ಲೇಖನವು Ultimate Multisite ಆವೃತ್ತಿ 2.x ಅನ್ನು ಸೂಚಿಸುತ್ತದೆ.**_

ನಿಮ್ಮ ಗ್ರಾಹಕರ ಬದಿಯಲ್ಲಿ ಸದಸ್ಯತ್ವವನ್ನು ರದ್ದುಪಡಿಸುವುದು ತುಂಬಾ ಸುಲಭ.

ನಿಮ್ಮ ಬಳಕೆದಾರರು ತಮ್ಮ ಸದಸ್ಯತ್ವವನ್ನು ರದ್ದುಪಡಿಸಲು ಬಯಸಿದರೆ, ಮೊದಲು ತಮ್ಮ ****Account**** ಪುಟಕ್ಕೆ ಹೋಗಬೇಕು.

ಅಲ್ಲಿ, ಅವರು **ಅಪಾಯ ವಲಯ** ಗೆ ಹೋಗಿ, **Account ಅಳಿಸಿ** ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಬೇಕು.

![Account ಪುಟದ ಅಪಾಯ ವಲಯ, Account ಅಳಿಸಿ ಬಟನ್ ಜೊತೆಗೆ](/img/account-page/danger-zone.png)

ಅವರ ದೃಢೀಕರಣವನ್ನು ಕೇಳುವ ಒಂದು ಘಟಕ ತೆರೆಯುತ್ತದೆ. ಅವರು **Account ಅಳಿಸುವಿಕೆಯನ್ನು ದೃಢೀಕರಿಸಿ** ಆಯ್ಕೆಯನ್ನು ಆನ್ ಮಾಡಿ, **Account ಅಳಿಸಿ** ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಬೇಕು.

![Account ಅಳಿಸುವಿಕೆಯನ್ನು ದೃಢೀಕರಿಸಿ ಸಂವಾದ](/img/account-page/delete-account-confirm.png)

ಇದರಿಂದ ಅವರನ್ನು ಜಾಲತಾಣದಿಂದ ಲಾಗ್ ಔಟ್ ಮಾಡಲಾಗುತ್ತದೆ ಮತ್ತು ನಿಮ್ಮ ಜಾಲದಲ್ಲಿ ಅವರ ಸದಸ್ಯತ್ವ ರದ್ದಾಗುತ್ತದೆ. ಜೊತೆಗೆ, ನಿಮ್ಮ ಜಾಲದಿಂದ ಅವರ ಜಾಲತಾಣವನ್ನೂ ಅಳಿಸಲಾಗುತ್ತದೆ.

![Account ಅಳಿಸಿದ ನಂತರ ಲಾಗ್ ಔಟ್ ಆಗಿರುವುದು](/img/account-page/logged-out-after-delete.png)

![ಸದಸ್ಯತ್ವ ರದ್ದಾಗಿದೆ ಎಂಬ ದೃಢೀಕರಣ](/img/account-page/membership-canceled.png)

**ಪ್ರಮುಖ:** ಈ ಕ್ರಿಯೆಯಿಂದ ಗ್ರಾಹಕರು ನಿಮ್ಮ ಜಾಲದಿಂದ ಹೊರಗೊಳ್ಳುವುದಿಲ್ಲ ಎಂಬುದನ್ನು ಗಮನಿಸಿ. ಏಕೆಂದರೆ ಒಬ್ಬ ಗ್ರಾಹಕರಿಗೆ ಒಂದಕ್ಕಿಂತ ಹೆಚ್ಚು ಸದಸ್ಯತ್ವಗಳು ಇರಬಹುದು ಅಥವಾ ಅವರಿಗೆ ಯಾವುದೇ ಸದಸ್ಯತ್ವ ಲಿಂಕ್ ಆಗಿರದೇ ಇರಬಹುದು.

![ರದ್ದುಗೊಳಿಸಿದ ನಂತರವೂ ಗ್ರಾಹಕರು ಜಾಲದಲ್ಲಿ ಇರುತ್ತಾರೆ](/img/admin/customer-after-cancel.png)

![ಸಕ್ರಿಯ ಸದಸ್ಯತ್ವಗಳಿಲ್ಲದ ಗ್ರಾಹಕರು](/img/admin/memberships-customer-empty.png)
