# ನನ್ನ ಚಂದಾದಾರಿಕೆಯನ್ನು ಹೇಗೆ ರದ್ದುಗೊಳಿಸಬಹುದು?

ನೀವು Ultimate Multisite ಲೈಸೆನ್ಸ್ ಹೊಂದಿದ್ದರೆ, ನೀವು ಬಯಸಿದಾಗ ಅದರ ನವೀಕರಣವನ್ನು ರದ್ದುಗೊಳಿಸಬಹುದು. ಕೆಳಗಿನ ಹಂತಗಳನ್ನು ಅನುಸರಿಸಿ:


<!-- Screenshot unavailable: Account page email with access link -->

ಖರೀದಿಯ ಸಮಯದಲ್ಲಿ ನೀವು ನೀಡಿದ ಇಮೇಲ್ ಮತ್ತು ನಿಮ್ಮ ಪಾಸ್‌ವರ್ಡ್ ಬಳಸಿ.

<!-- Screenshot unavailable: Login page with email and password fields -->

ಇದು ನಿಮ್ಮ Account Page ನ ಆರಂಭಿಕ ಪರದೆ:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

ನಿಮ್ಮ ಚಂದಾದಾರಿಕೆಯನ್ನು ರದ್ದುಗೊಳಿಸಲು, ಎಡಭಾಗದ ಮೆನುವಿನಲ್ಲಿ _Renewing & Billing_ ಗೆ ಹೋಗಿ.

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

ಪಕ್ಕದ ವಿಂಡೋ ತೆರೆಯಲು ಬಲಭಾಗದ ಬಾಣದ ಗುರುತಿನ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ. ನಂತರ, **auto-renew** ಅನ್ನು ರದ್ದುಗೊಳಿಸುವ ಆಯ್ಕೆಯನ್ನು ಆರಿಸಿ.

ಸಿಸ್ಟಮ್ ನಿಮಗೆ ದೃಢೀಕರಣ ಸಂದೇಶವನ್ನು ತೋರಿಸುತ್ತದೆ.

ನಿಮ್ಮ ಚಂದಾದಾರಿಕೆಯನ್ನು ರದ್ದುಗೊಳಿಸಲು ನೀವು ಖಚಿತವಾಗಿದ್ದರೆ, _**Cancel Renewals**_ ಬಟನ್ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ. ಈ ಕ್ರಿಯೆಯ ನಂತರ, ಒಂದು ಸಣ್ಣ ಸಮೀಕ್ಷೆಗೆ ಉತ್ತರಿಸಲು ನಿಮ್ಮನ್ನು ಕೇಳಲಾಗುತ್ತದೆ.

ಮುಗಿಯಿತು! ನಿಮ್ಮ ಚಂದಾದಾರಿಕೆ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ನವೀಕರಣಗೊಳ್ಳುವುದಿಲ್ಲ.

**ನಿಮ್ಮ ಚಂದಾದಾರಿಕೆ ಅವಧಿ ಮುಗಿಯುವವರೆಗೆ ನಿಮ್ಮ ಕೀ ಮಾನ್ಯವಾಗಿರುತ್ತದೆ**. ನಿಮ್ಮ ಚಂದಾದಾರಿಕೆಯನ್ನು ಮರುಸಕ್ರಿಯಗೊಳಿಸಲು ಬಯಸಿದರೆ, ಅದನ್ನು ಹಸ್ತಚಾಲಿತವಾಗಿ ಮಾಡಬೇಕಾಗುತ್ತದೆ.
