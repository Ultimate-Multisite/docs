# Ultimate Multisite Hooks ಮತ್ತು filters

Ultimate Multisite ಹಲವಾರು functions ಮತ್ತು database queries ಗಳನ್ನು ಒಳಗೊಂಡಿದೆ. ಇದು hooks ಮತ್ತು filters ಗಳನ್ನು ಹೊಂದಿದ್ದು, ಅವುಗಳನ್ನು ಬಳಸಿ ನೀವು ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ functions ಗಳನ್ನು ಮಾರ್ಪಡಿಸಬಹುದು ಅಥವಾ ಹೊಸ code ಸೇರಿಸಬಹುದು.

ಅನುಭವಿ ಬಳಕೆದಾರರು ತಮ್ಮ network ಅನ್ನು ಮತ್ತಷ್ಟು customize ಮಾಡಲು ಸಹಾಯವಾಗುವಂತೆ ನಾವು ಈ hooks ಮತ್ತು filters ಗಳ ಪಟ್ಟಿಯನ್ನು ಉಲ್ಲೇಖವಾಗಿ ಸಿದ್ಧಪಡಿಸಿದ್ದೇವೆ. ಕೆಳಗಿನ ಲಿಂಕ್ ಕ್ಲಿಕ್ ಮಾಡಿ Github ನಿಂದ ಈ ಪಟ್ಟಿಯನ್ನು ಪ್ರವೇಶಿಸಬಹುದು.
