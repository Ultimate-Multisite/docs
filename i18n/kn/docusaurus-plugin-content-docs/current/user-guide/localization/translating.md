# Ultimate Multisite ಅನುವಾದ

Ultimate Multisite ಇಂಗ್ಲಿಷ್‌ನಲ್ಲಿ ಅಭಿವೃದ್ಧಿಪಡಿಸಲಾಗಿದೆ, ಆದರೆ ಈ plugin ಸಂಪೂರ್ಣವಾಗಿ ಅನುವಾದಿಸಬಹುದಾದಂತಹದ್ದು ಮತ್ತು ನಾವು ಅನುವಾದ ಕೊಡುಗೆಗಳನ್ನು ಸಕ್ರಿಯವಾಗಿ ಸ್ವಾಗತಿಸುತ್ತೇವೆ.

## ಮುಖ್ಯ Plugin ಅನುವಾದಗಳು

ಮುಖ್ಯ Ultimate Multisite plugin ಅನ್ನು ಅಧಿಕೃತ **WordPress.org ಅನುವಾದ ವೇದಿಕೆಯ** ಮೂಲಕ ಅನುವಾದಿಸಲಾಗುತ್ತದೆ:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

ಇದು ಪ್ರಮಾಣಿತ WordPress ಸಮುದಾಯ ಅನುವಾದ ವ್ಯವಸ್ಥೆಯಾಗಿದೆ. ಅನುವಾದಗಳು ಸಾಕಷ್ಟು ಅನುಮೋದನೆ ಮಿತಿಯನ್ನು ತಲುಪಿದಾಗ, ಅವು WordPress plugin ನವೀಕರಣಗಳ ಮೂಲಕ ಎಲ್ಲಾ Ultimate Multisite ಬಳಕೆದಾರರಿಗೆ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ವಿತರಿಸಲ್ಪಡುತ್ತವೆ.

**ನಿಮ್ಮ ಸಹಾಯ ಬೇಕು!** ನೀವು ಇಂಗ್ಲಿಷ್ ಹೊರತುಪಡಿಸಿ ಬೇರೆ ಭಾಷೆ ಮಾತನಾಡುವವರಾಗಿದ್ದರೆ, ದಯವಿಟ್ಟು ಅನುವಾದಗಳಿಗೆ ಕೊಡುಗೆ ನೀಡಲು ಪರಿಗಣಿಸಿ. ಕೆಲವು ಪದಗಳನ್ನು ಅನುವಾದಿಸಿದರೂ ಸಹ ಸಹಾಯಕವಾಗುತ್ತದೆ. ಪ್ರಾರಂಭಿಸಲು:

1. [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/) ಗೆ ಭೇಟಿ ನೀಡಿ
2. ನಿಮ್ಮ ಭಾಷೆಯನ್ನು ಆಯ್ಕೆ ಮಾಡಿ
3. "Contribute Translation" ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ
4. ನಿಮಗೆ ಉಚಿತ WordPress.org ಖಾತೆ ಬೇಕಾಗುತ್ತದೆ

ನಿಮ್ಮ ಭಾಷೆಗೆ ಅನುವಾದ ಸಂಪಾದಕರಾಗಲು (ಅನುವಾದಗಳನ್ನು ಅನುಮೋದಿಸಲು ಸಾಧ್ಯವಾಗುವಂತೆ) ಬಯಸಿದರೆ, [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) ಮೂಲಕ ನಮ್ಮನ್ನು ಸಂಪರ್ಕಿಸಿ.

## Addon ಅನುವಾದಗಳು

Addon plugins ಗಳನ್ನು ಪ್ರತ್ಯೇಕವಾಗಿ ಇಲ್ಲಿ ಅನುವಾದಿಸಲಾಗುತ್ತದೆ:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Addon ಅನುವಾದಗಳು ಪ್ರಸ್ತುತ AI ಮೂಲಕ ಆರಂಭಿಕ ಹಂತವಾಗಿ ರಚಿಸಲ್ಪಟ್ಟಿವೆ. ಅವು ಎಲ್ಲಾ ಬೆಂಬಲಿತ ಭಾಷೆಗಳನ್ನು ಒಳಗೊಂಡಿವೆ ಆದರೆ ಅಸಹಜ ವಾಕ್ಯ ರಚನೆ ಅಥವಾ ತಪ್ಪುಗಳನ್ನು ಹೊಂದಿರಬಹುದು. ಅವುಗಳನ್ನು ಹೆಚ್ಚು ಸ್ವಾಭಾವಿಕ ಮತ್ತು ನಿಖರವಾಗಿಸಲು ಮಾನವ ಅನುವಾದಕರು ಪರಿಶೀಲಿಸಿ ಸುಧಾರಣೆಗಳನ್ನು ಸೂಚಿಸಲು ನಾವು ಸ್ವಾಗತಿಸುತ್ತೇವೆ.

## ಸ್ಥಳೀಯ ಅನುವಾದಗಳು

ಮೂಲಕ್ಕೆ ಕೊಡುಗೆ ನೀಡುವ ಬದಲು ನಿಮ್ಮ ಸ್ವಂತ ಯೋಜನೆಗಾಗಿ ಸ್ಥಳೀಯವಾಗಿ ಅನುವಾದಗಳನ್ನು ರಚಿಸಲು ಬಯಸಿದರೆ, ಎರಡು ಆಯ್ಕೆಗಳಿವೆ:

### Poedit ಬಳಸುವುದು

`.pot` ಫೈಲ್ ಅನ್ನು ಅನುವಾದಿಸಲು ನೀವು **[Poedit](https://poedit.net)** ಡೆಸ್ಕ್‌ಟಾಪ್ ಅಪ್ಲಿಕೇಶನ್ ಬಳಸಬಹುದು:

1. [poedit.net](https://poedit.net) ನಿಂದ Poedit ಡೌನ್‌ಲೋಡ್ ಮಾಡಿ
2. `wp-content/plugins/ultimate-multisite/lang/` ನಿಂದ `.pot` ಫೈಲ್ ತೆರೆಯಿರಿ
3. Poedit ಇಂಟರ್‌ಫೇಸ್‌ನಲ್ಲಿ ಪದಗಳನ್ನು ಅನುವಾದಿಸಿ
4. ಉಳಿಸಿ -- Poedit `.po` ಮತ್ತು `.mo` ಫೈಲ್‌ಗಳನ್ನು ರಚಿಸುತ್ತದೆ
5. ಔಟ್‌ಪುಟ್ ಫೈಲ್‌ಗಳನ್ನು `lang` ಫೋಲ್ಡರ್‌ನಲ್ಲಿ ಇರಿಸಿ

### Loco Translate ಬಳಸುವುದು

**[Loco Translate](https://wordpress.org/plugins/loco-translate/)** WordPress plugin ಯಾವುದೇ ಡೆಸ್ಕ್‌ಟಾಪ್ ಉಪಕರಣಗಳಿಲ್ಲದೆ ನೇರವಾಗಿ admin panel ನಿಂದ ಅನುವಾದಿಸಲು ಅನುಮತಿಸುತ್ತದೆ. ಇದು ನಿಮಗಾಗಿ ಅನುವಾದ ಫೈಲ್‌ಗಳನ್ನು ರಚಿಸುವುದು ಮತ್ತು ಕಂಪೈಲ್ ಮಾಡುವುದನ್ನು ನಿರ್ವಹಿಸುತ್ತದೆ.
