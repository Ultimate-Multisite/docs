# 레슨 4: 틈새 템플릿 구축하기

템플릿은 귀하의 틈새 가치 제안의 핵심입니다. 피트니스 스튜디오 사장님이 가입했을 때, 이미 클래스 일정, 트레이너 프로필, 적절한 이미지가 갖춰진 피트니스 웹사이트처럼 보이는 사이트를 본다면, 아무것도 없는 백지 상태의 사이트를 본 경우보다 훨씬 더 오래 머무를 가능성이 높습니다.

## 어디까지 진행했나요

Ultimate Multisite을 설치하고 설정하여 작동하는 FitSite 네트워크를 갖추었습니다. 이제 FitSite가 피트니스 비즈니스에 맞춤 제작된 것처럼 느껴지게 만드는 템플릿을 구축할 차례입니다.

## 틈새 템플릿이 중요한 이유

일반적인 템플릿은 고객에게 어려운 작업을 수행하도록 강요합니다. 즉, 어떤 페이지가 필요한지, 어떤 콘텐츠 구조가 효과적인지, 그리고 자신의 산업에 맞게 어떻게 보이게 할지 등을 스스로 알아내야 합니다. 틈새 템플릿은 이러한 마찰을 제거합니다.

FitSite에 가입하는 피트니스 스튜디오 사장님은 다음을 볼 수 있어야 합니다.

- 피트니스 스튜디오 웹사이트처럼 보이는 홈페이지
- 클래스, 일정, 트레이너, 가격 책정 페이지가 이미 생성되어 있음
- 자신의 산업에 맞는 이미지와 자리 표시자 콘텐츠
- 전문적이고 피트니스 브랜드에 맞는 디자인

그들은 자신의 세부 정보만 채우면 됩니다. 처음부터 모든 것을 만들 필요가 없습니다.

## 템플릿 계획하기

구축하기 전에 어떤 템플릿을 제공할지 결정해야 합니다. FitSite의 경우, 세 가지를 만들 것입니다.

### 템플릿 1: 스튜디오 필수 (Studio Essential)

소규모 스튜디오 및 개인 트레이너를 위한 템플릿입니다.

- 히어로 이미지, 클래스 하이라이트, 행동 유도 버튼이 있는 **홈페이지**
- 스튜디오 스토리와 미션이 담긴 **소개** 페이지
- 일정 레이아웃이 포함된 **클래스** 페이지
- 프로필 카드가 있는 **트레이너** 페이지
- 위치 지도와 문의 양식이 있는 **연락처** 페이지
- 피트니스에 적합한 색상 구성의 **깔끔하고 현대적인 디자인**

### 템플릿 2: 짐 프로 (Gym Pro)

더 많은 서비스를 제공하는 규모 있는 체육관을 위한 템플릿입니다.

- 스튜디오 필수의 모든 기능에 더하여:
- 가격표가 있는 **멤버십** 페이지
- 시설 사진을 위한 **갤러리** 페이지
- 피트니스 팁과 뉴스를 위한 **블로그** 섹션
- 홈페이지의 **후기** 섹션
- **더 눈에 띄는 브랜딩** 옵션

### 템플릿 3: 피트니스 체인 (Fitness Chain)

여러 지점을 운영하는 사업체를 위한 템플릿입니다.

- 짐 프로의 모든 기능에 더하여:
- 여러 시설 목록이 있는 **지점** 페이지
- **프랜차이즈/지점** 하위 페이지 템플릿
- 지점별 세부 정보가 포함된 **중앙 집중식 브랜딩**
- 지점 전반의 **직원 디렉토리**

## 템플릿 사이트 구축하기

Ultimate Multisite에서 템플릿이란, 새로운 고객 사이트가 어떻게 보이기를 원하는지 설정된 WordPress 사이트를 의미합니다. 만드는 방법은 다음과 같습니다.

### 1단계: 템플릿 사이트 생성

1. 네트워크 관리자에서 **Sites > Add New**로 이동합니다.
2. `template-studio-essential`라는 이름의 사이트를 생성합니다.
3. 이 사이트가 작업할 캔버스가 됩니다.

### 2단계: 테마 설치 및 설정

템플릿 사이트 대시보드로 전환하여 다음 작업을 수행합니다.

1. 피트니스 비즈니스에 적합한 테마를 활성화합니다.
2. 테마 설정, 색상, 타이포그래피를 구성합니다.
3. 피트니스에 적합한 내비게이션을 포함하여 헤더와 푸터를 설정합니다.

:::tip 테마 선택
피트니스에 잘 어울리면서도 고객이 관리하기 어려울 정도로 복잡하지 않은 테마를 선택하세요. Astra, GeneratePress, 또는 Kadence와 같은 테마는 가볍고, 사용자 정의가 용이하며, 지원이 잘 되기 때문에 좋습니다.
:::

### 3단계: 페이지 만들기

다음 요소들로 각 페이지를 구축합니다.

- 피트니스에 자연스럽게 읽히는 **자리 표시자 콘텐츠** ("Welcome to [Studio Name]" 같은 문구, "Lorem ipsum" 같은 문구는 안 됩니다)
- 피트니스 활동을 보여주는 무료 스톡 사진 사이트의 **자리 표시자 이미지**
- 페이지 빌더 또는 블록 에디터를 사용한 **기능적인 레이아웃**

자리 표시자 콘텐츠는 가능한 한 지침서 역할을 하도록 만드세요. 일반적인 채우기 텍스트 대신, "여기에 스튜디오에 대한 간략한 설명과 특별한 점을 작성하세요. 트레이닝 철학, 경력 연수, 또는 고객이 기대할 수 있는 점을 언급하세요."와 같은 내용을 작성하는 것이 좋습니다.

### 4단계: 플러그인 설정

피트니스 스튜디오가 필요로 하는 플러그인을 설치하고 활성화합니다.

- 예약 또는 스케줄링 플러그인 (플랜 티어에 따라 해당되는 경우)
- 문의 양식 플러그인
- SEO 플러그인 (피트니스 관련 기본 설정으로 미리 구성)

### 5단계: 템플릿으로 표시하기

1. **Ultimate Multisite > Sites**로 이동합니다.
2. 템플릿 사이트를 편집합니다.
3. **Site Template** 토글을 활성화합니다.

이 과정을 제공하려는 모든 템플릿에 대해 반복합니다.

## 템플릿 품질 체크리스트

템플릿을 사용 가능하게 만들기 전에 다음 사항을 확인하세요.

- [ ] 모든 페이지가 올바르게 로드되고 전문적으로 보이는지
- [ ] 자리 표시자 콘텐츠가 도움이 되고 틈새에 특화되었는지
- [ ] 이미지가 적절하고 라이선스가 제대로 되었는지
- [ ] 모든 페이지에서 모바일 반응형이 작동하는지
- [ ] 내비게이션이 논리적이고 완전한지
- [ ] 문의 양식이 작동하는지
- [ ] 깨진 링크나 누락된 에셋이 없는지
- [ ] 페이지 로드 속도가 허용 가능한지

## 현재 FitSite 네트워크 구성

```
FitSite Network
├── WordPress Multisite (subdomain mode)
├── Ultimate Multisite (configured)
├── Hosting with wildcard SSL
├── Domain mapping configured
├── Site Templates
│   ├── Studio Essential (small studios, trainers)
│   ├── Gym Pro (established gyms)
│   └── Fitness Chain (multi-location)
└── Ready for plan configuration (next lesson)
```

## 이번 레슨에서 구축한 것

- 다양한 피트니스 비즈니스 규모에 맞춰 설계된 **세 가지 틈새 템플릿**
- 플랫폼이 맞춤 제작된 것처럼 느껴지게 하는 **피트니스에 적합한 콘텐츠 및 이미지**
- 고객이 커스터마이징하는 방법을 안내하는 **지침서 역할을 하는 자리 표시자 콘텐츠**
- 템플릿이 운영 준비가 되었는지 확인하는 **품질 체크리스트**
