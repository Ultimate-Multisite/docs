# Fluent Forms

이 애드온은 Multisite Ultimate에서 제품, 멤버십 또는 사이트에 Fluent Forms에 대한 제한을 추가할 수 있게 해줍니다.  
Fluent Forms는 커스텀 포스트 타입 대신 커스텀 테이블을 사용하므로 일반적인 제한 옵션이 Fluent Forms에서 작동하지 않습니다.

## Features

- Fluent Forms 또는 Conversational Fluent Forms에 대한 제한을 추가합니다.  
- 고객이 제한에 도달하면 플랜 업그레이드를 권장하는 친절한 메시지를 받습니다.
