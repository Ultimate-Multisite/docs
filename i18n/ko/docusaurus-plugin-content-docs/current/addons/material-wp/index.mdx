# Material WP

WordPress 관리 인터페이스는 오랫동안 같은 모습을 유지해 왔습니다. 하지만 Material Design 버전의 관리 인터페이스가 어떻게 보일지 궁금해 본 적이 있나요?  

Material WP는 그와 같은 외관과 느낌을 제공하도록 개발된 관리 테마로, 고객에게 보다 맞춤화된 대시보드를 제공하는 데 도움을 줍니다.

## Features

- 맞춤 CSS 지원
- 간편한 스타일링 설정
- 메뉴 재정렬
- 간편한 가져오기/내보내기

## Documentation

**Material WP로 관리 패널 맞춤화**

## Installation

1. 'material-wp'를 '/wp-content/plugins/' 디렉터리에 업로드합니다
2. WordPress의 '플러그인' 메뉴를 통해 플러그인을 활성화합니다

## Usage Guide

WordPress 관리 인터페이스는 오랫동안 같은 모습을 유지해 왔습니다. 하지만 Material Design 버전의 관리 인터페이스가 어떻게 보일지 궁금해 본 적이 있나요?  

Material WP는 그와 같은 외관과 느낌을 제공하도록 개발된 관리 테마로, 고객에게 보다 맞춤화된 대시보드를 제공하는 데 도움을 줍니다. 또한 Material WP는 Ultimate Multisite의 프리미엄 플러그인 핵심에 포함되어 있습니다.

**설치한** 뒤, _플러그인_ 페이지로 이동해 네트워크에서 활성화 버튼을 클릭합니다.
