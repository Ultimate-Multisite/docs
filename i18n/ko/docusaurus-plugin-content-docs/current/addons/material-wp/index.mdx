# Material WP {#material-wp}

WordPress 관리 인터페이스는 오랫동안 그대로였습니다. 하지만 관리 인터페이스의 Material Design 버전이 어떤 모습일지 궁금했던 적이 있나요? 

Material WP는 그런 모양과 느낌을 제공하도록 개발된 관리 테마로, 고객에게 더 맞춤화된 Dashboard를 제공할 수 있도록 도와줍니다.

## 기능 {#features}

- 사용자 지정 CSS 지원
- 쉬운 스타일링 설정
- 메뉴 순서 변경
- 쉬운 가져오기/내보내기

## 설치 {#installation}

1. 'material-wp'를 '/wp-content/plugins/' 디렉터리에 업로드합니다
2. WordPress의 '플러그인' 메뉴를 통해 플러그인을 활성화합니다

## 사용 가이드 {#usage-guide}
