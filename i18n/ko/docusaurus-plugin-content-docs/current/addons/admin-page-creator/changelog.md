# Admin Page Creator 변경 로그

### 버전 1.9.0 - 릴리즈 날짜 2026-01-18

* 수정: 활성화 시 치명적인 오류.
* 리팩터링: 콘텐츠 소스 클래스를 Singleton 트레이트를 사용하고 적절한 부모 생성자 초기화를 적용하도록 변경.
* 수정: Vue.js 템플릿 컴파일 오류로 인해 콘텐츠 타입 선택기가 렌더링되지 않음.
* 수정: 누락된 Vue 데이터 속성 (template_id, external_link_url_embedable, external_link_url_checking).
* 수정: TinyMCE 에디터가 페이지 로드 시 제대로 초기화되지 않음.
* 수정: Divi 지원 파일이 Composer를 통해 자동 로드되지 않음.
* 수정: Divi 지원 클래스에서 손상된 PHP 문법.
* 개선: JavaScript 자산 조직 및 최소화.
* 개선: 최신 WordPress 및 페이지 빌더 버전과의 호환성.
