# 필터: wu_apc_should_display_admin_menu

관리자가 커스터마이징 과정을 마친 경우 관리 메뉴 페이지를 숨길 수 있도록 허용합니다.

## 매개변수

| 이름 | 유형 | 설명 |
|------|------|------|
| $should_display_admin_menu | `bool` | 페이지를 표시해야 하는지 여부 |
| $network_active | `bool` | 플러그인이 네트워크 관리자에서 활성화되어 있는지 확인합니다 |
