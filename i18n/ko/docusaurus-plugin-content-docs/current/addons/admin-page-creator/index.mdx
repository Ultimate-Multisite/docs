# Admin Page Creator

## 내용
Admin Page Creator는 WordPress 관리 대시보드 내에서 새로운 관리 페이지와 하위 페이지를 쉽게 추가할 수 있게 해줍니다. 새 사용자에게 랜딩 페이지 역할을 할 수 있는 환영 페이지나 튜토리얼 페이지를 만들 때 사용할 수 있습니다. Beaver Builder, Elementor, Brizy, Oxygen 같은 페이지 빌더를 사용해 번거롭게 페이지를 디자인할 수 있습니다.  
또한 커스텀 HTML, CSS, JavaScript 지원과 플랜 및 사용자 역할 기반 접근 제어를 제공합니다!

## 기능
* 커스텀 HTML 및 CSS 지원
* JavaScript 지원
* PHP 콘텐츠 모드
* 사용자 역할 기반 권한
* 페이지 빌더 지원

## 문서
[Multisite Ultimate를 사용한 커스텀 관리 페이지 및 하위 페이지 만들기: Admin Page Creator](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## 설치

1. **ultimate-multisite-admin-page-creator**를 **/wp-content/plugins/** 디렉터리에 업로드합니다.  
2. WordPress 네트워크 관리에서 **Plugins** 메뉴를 통해 플러그인을 활성화합니다.

## 사용 가이드

WordPress는 사이트의 프론트엔드에 콘텐츠를 추가하는 것을 매우 쉽게 만들어 주지만, 관리 측면에서 페이지와 하위 페이지를 만드는 데는 기본적인 코딩 기술이 필요합니다. Ultimate Multisite: Admin Page Creator는 프론트엔드와 같은 편리함을 네트워크 사이트의 백엔드에 제공하려고 합니다.

## Admin Page Creator를 사용하는 이유?

고객의 관리 패널에 튜토리얼이 포함된 커스텀 페이지를 제공하고 싶거나, FAQ와 연락처 위젯이 포함된 커스텀 페이지를 추가하고 싶을 수 있습니다. Admin Page Creator를 사용하면 가능성은 무한하며, 오늘 튜토리얼에서 이를 탐구할 것입니다.

## 첫 번째 커스텀 관리 페이지 만들기

Ultimate Multisite: Admin Page Creator를 설치하고 활성화한 후, 네트워크 관리 메뉴로 이동해 Admin Pages로 이동합니다.

![Custom Admin Pages list on Network Admin](/img/admin/admin-pages-list.png)

_네트워크 관리에서 커스텀 관리 페이지 목록_

지금까지 생성한 모든 관리 페이지 목록이 표시됩니다(첫 방문이므로 목록은 비어 있을 것입니다). **Add new Admin Page**를 클릭하면 Admin Page 편집기로 리디렉션됩니다.

## 커스텀 관리 페이지의 요소

Admin Page 편집기에서 페이지를 커스터마이즈할 수 있는 다양한 옵션이 제공됩니다.

### 페이지 제목

이것은 페이지 상단에 표시되는 제목이며, 관리 메뉴 라벨이 아닙니다.

_페이지 제목 입력_

### 콘텐츠 유형 선택기

Ultimate Multisite: Admin Page Creator는 페이지 콘텐츠에 대해 두 가지 옵션을 제공합니다: 기본 WordPress WYSIWYG 편집기를 사용하거나 HTML 편집기로 전환할 수 있습니다.  
HTML에 익숙하지 않다면 기본 WordPress 편집기를 사용하세요. 일반 포스트나 페이지와 마찬가지로 페이지 내용을 편집할 수 있습니다.

## 메뉴 옵션

측면에 있는 General Options 메타 박스에서 메뉴 유형, 라벨 등을 구성할 수 있습니다.

### 메뉴 유형

새 커스텀 관리 페이지는 두 가지 메뉴 유형을 가질 수 있습니다: 최상위 메뉴와 하위 메뉴.  
최상위 메뉴 페이지는 이름에서 알 수 있듯이 새로 만든 페이지를 기본 WordPress 페이지와 함께 관리 메뉴에 추가합니다.  
최상위 옵션을 선택하면 메뉴 순서와 메뉴 아이콘도 선택할 수 있습니다.  
이 페이지를 하위 메뉴 페이지로 만들고 싶다면, 이 페이지를 연결할 상위 페이지를 선택할 수 있습니다. 커스텀 페이지를 기본 WordPress 페이지나 최상위 커스텀 관리 페이지에 추가할 수 있습니다.

## 고급 옵션

커스텀 관리 페이지에 다양한 콘텐츠를 추가할 수 있습니다. CSS 규칙 및 파일, JavaScript, 외부 라이브러리 등이 포함됩니다. Advanced Options 탭을 사용해 이를 수행할 수 있습니다.

### 커스텀 CSS

CSS 탭에서 자체 커스텀 CSS 코드를 추가하거나 원격 사이트(예: CDN, Google Fonts 등)에서 CSS 파일을 가져올 수 있습니다.

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_커스텀 CSS도 지원됩니다. 외부 스타일 파일을 포함할 수도 있습니다!_

### 커스텀 JavaScript

JavaScript에도 동일하게 적용됩니다. 커스텀 JavaScript 코드에서 사용할 외부 라이브러리를 추가할 수 있습니다.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_자신의 JavaScript 코드를 추가하세요!_

## 권한

Advanced Options에서도 이 페이지에 접근할 서브 사이트를 선택할 수 있습니다. 예를 들어, 다른 플랜 티어에 따라 독점 콘텐츠를 제공하려는 경우 유용합니다.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_이 페이지가 표시될 위치를 완전히 제어할 수 있습니다_

**중요** : 페이지가 표시되려면 두 조건이 모두 충족되어야 합니다. 즉, 사용자가 특정 플랜의 클라이언트이지만 선택된 역할 중 하나를 가지고 있지 않으면(또는 그 반대의 경우) 페이지가 그들의 패널에 표시되지 않습니다.

## 활성화

커스텀 관리 페이지를 삭제하거나 권한을 조정하지 않고 비활성화하려면 Active 옵션을 선택 해제하세요.  
최상위 페이지를 비활성화하면 해당 상위 페이지에 연결된 모든 하위 페이지도 자동으로 비활성화됩니다.
