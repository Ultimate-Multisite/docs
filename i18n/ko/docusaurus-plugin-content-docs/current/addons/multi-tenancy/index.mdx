# Multi-Tenancy

서브사이트를 서버와 데이터베이스에 분산시켜 성능과 보안을 향상시킵니다

## Requirements

- WordPress 5.3 or higher
- PHP 7.4 or higher
- Multisite Ultimate plugin (active)

## Installation

1. Upload the addon files to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the addon settings in the network admin

## Development

### Setup

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Testing

이 애드온은 포괄적인 테스트 프레임워크를 포함합니다:

- **단위 테스트**: PHPUnit 기반 테스트와 WordPress 테스트 스위트
- **Base Test Class**: 일반 테스트 시나리오에 대한 헬퍼 메서드를 제공합니다
- **Test Coverage**: 코드 커버리지를 추적하도록 구성되었습니다
- **Multisite Testing**: 테스트가 멀티사이트 환경에서 실행됩니다

### Code Standards

- **WordPress Coding Standards**: PHPCS를 통해 적용됩니다
- **PHP 7.4+ Compatibility**: 최신 PHP 기능이 지원됩니다
- **Static Analysis**: 타입 검사를 위한 PHPStan 통합
- **Code Modernization**: PHP 업그레이드를 위한 Rector 규칙

### Build System

- **Asset Processing**: CSS/JS 파일 최소화
- **Translation**: POT 파일 생성
- **Packaging**: 배포용 아카이브 생성
- **Development Mode**: 디버깅용 미최적화 자산

## Configuration

이 애드온은 네트워크 관리자 인터페이스를 통해 액세스할 수 있는 다양한 구성 옵션을 지원합니다.

## Hooks and Filters

### Actions

- `ultimate-multisite-addon-template_init` - 애드온 초기화 후 실행됩니다
- `ultimate-multisite-addon-template_loaded` - 모든 애드온 파일이 로드된 후 실행됩니다

### Filters

- `ultimate-multisite-addon-template_settings` - 애드온 설정을 수정합니다
- `ultimate-multisite-addon-template_enabled` - 애드온 활성화/비활성화 상태를 재정의합니다
