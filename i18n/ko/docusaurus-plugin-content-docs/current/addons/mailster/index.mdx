# Mailster

Ultimate Multisite Mailster Integration addon을 사용하여 멀티사이트 네트워크를 강력한 이메일 리스트 구축 기계로 변환하세요. 고객이 가입하거나 구매할 때 자동으로 새로운 고객을 Mailster 이메일 리스트에 추가하여 멤버십 플랫폼과 이메일 마케팅 전략 사이에 원활한 연결을 만듭니다.

## Key Features

**Automatic Subscriber Sync**
* 가입 시 고객을 Mailster 리스트에 자동으로 추가합니다.
* 가입 시 즉시 동기화할지, 결제 확인을 기다릴지 선택합니다.
* 수동 개입 없이 이메일 리스트를 최신 상태로 유지합니다.

**Flexible List Management**
* 모든 신규 고객에 대한 전역 기본 리스트를 설정합니다.
* 타깃 캠페인을 위해 제품별 리스트로 기본값을 재정의합니다.
* Mailster의 친숙한 인터페이스로 리스트 할당을 쉽게 관리합니다.

**GDPR-Compliant Opt-In Options**
* 자동 구독 또는 체크박스 기반 동의 중 선택합니다.
* 확인된 구독을 위해 이중 옵트인(다중 동의)을 활성화합니다.
* 컴플라이언스 요구 사항에 따라 옵트인 체크박스를 표시/숨깁니다.

**Smart Data Mapping**
* 고객 이름 및 연락처 정보를 자동으로 동기화합니다.
* 청구 주소를 구독자 필드에 매핑합니다.
* 타깃 캠페인을 위해 가입 날짜를 추적합니다.

**Product-Level Control**
* 다른 제품에 서로 다른 리스트를 할당합니다.
* 고객이 구매한 항목을 기반으로 타깃 캠페인을 생성합니다.
* 필요에 따라 특정 제품에 대해 Mailster를 비활성화합니다.

**Professional Experience**
* 리스트 선택을 위한 네이티브 Mailster UI 통합.
* 문제 해결 및 검증을 위한 상세 로깅.
* 기존 구독자의 원활한 업데이트.

## Perfect For

* 이메일 리스트를 구축하는 멤버십 사이트 소유자
* 신규 고객을 육성하려는 SaaS 플랫폼
* 등록 기준으로 학생을 세분화하는 코스 제작자
* Ultimate Multisite와 Mailster를 운영하는 모든 사용자

## How It Works

1. Ultimate Multisite 네트워크에 애드온을 설치하고 활성화합니다.
2. Ultimate Multisite 설정에서 전역 기본 리스트를 구성합니다.
3. 필요에 따라 제품별 리스트로 재정의합니다.
4. 옵트인 모드와 구독 타이밍을 선택합니다.
5. Mailster 리스트가 자동으로 성장하는 것을 확인하세요!

## Installation

1. 플러그인 파일을 Ultimate Multisite 애드온 디렉터리에 업로드합니다.
2. 메인 사이트에 Mailster가 설치되고 활성화되어 있는지 확인합니다.
3. Ultimate Multisite 애드온 페이지에서 애드온을 활성화합니다.
4. 설정 > Mailster Integration으로 이동해 구성합니다.
5. 기본 리스트를 선택하고 옵트인 설정을 구성합니다.

## Frequently Asked Questions

## Does this addon work without Ultimate Multisite?

아니요, 이 애드온은 동작하려면 Ultimate Multisite가 필요합니다. Mailster를 Ultimate Multisite의 고객 및 제품 시스템과 통합하도록 특별히 설계되었습니다.

## Do I need Mailster installed on all sites?

아니요, Mailster는 메인 네트워크 사이트에만 설치 및 활성화하면 됩니다. 애드온은 모든 서브사이트에서 자동으로 연결됩니다.

## Can I assign different lists to different products?

네! 전역 기본 리스트를 설정한 뒤 제품별로 재정의할 수 있습니다. 이를 통해 고객이 구매한 항목을 기반으로 매우 타깃팅된 이메일 캠페인을 만들 수 있습니다.

## What happens if a customer already exists in Mailster?

기본적으로 애드온은 기존 구독자를 업데이트하고 새 리스트에 추가합니다. 기존 구독자를 건너뛰려면 설정에서 이 기능을 비활성화할 수 있습니다.

## Is this GDPR compliant?

네! 이 애드온은 고객이 이메일 수신에 명시적으로 동의하도록 요구하는 체크박스 기반 옵트인 모드를 포함합니다. 이는 적절한 개인정보 정책과 결합하면 GDPR 요구 사항을 충족합니다.

## Can I use double opt-in?

물론입니다! 이중 옵트인 설정을 활성화하면 Mailster가 구독을 활성화하기 전에 확인 이메일을 보냅니다.
