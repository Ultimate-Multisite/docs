# 지원 티켓 변경 로그

### 1.0.2 - 2025-12-11
* 추가: 관리자 및 프런트엔드용 전체 티켓 보기 기능
* 추가: 티켓 답변 제출용 AJAX 핸들러
* 추가: 적절한 양식 처리를 통한 티켓 답변 기능 지원
* 추가: 관리자에서 티켓 제출 및 답변에 대한 적절한 공지 표시
* 추가: 고객 자체 제출 시 현재 사용자와 자동으로 티켓 연결
* 추가: 고객이 티켓 소유권을 무시하지 못하도록 보안 강화
* 추가: 누락된 헬퍼 함수 (wu_format_date, wu_user_can_view_ticket 등)
* 추가: 파일 첨부 다운로드 및 처리 개선
* 추가: 티켓 답변 및 상태 변경에 대한 이메일 알림 시스템
* 수정: 티켓 답변 양식에 필요한 티켓 ID 포함
* 수정: 관리자 티켓 보기에서 공지 표시 문제
* 수정: 지원 티켓 함수의 문법 오류
* 개선: 응답 및 첨부 파일용 적절한 쿼리 및 스키마 클래스를 포함한 데이터베이스 구조
* 개선: 티켓 모델에 대한 속성 대신 실제 프로퍼티로 마이그레이션
