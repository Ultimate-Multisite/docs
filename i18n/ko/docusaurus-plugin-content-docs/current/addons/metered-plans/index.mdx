# 측정 플랜

Ultimate Multisite: Metered Plans 애드온은 유연한 측정 기반 청구 기능을 도입하여 고객에게 청구하는 방식을 혁신합니다. 초과 시 사용자 접근을 차단하는 엄격한 하드 리미트 대신, 초과 사용량에 대해 자동으로 청구되는 과금 기능을 활성화합니다.

## 핵심 기능

* **Metered Billing**: 실제 사용량에 따라 고객에게 청구합니다. 고정 단계가 아니라.
* **Overage Management**: 플랜 한도를 초과하는 사용량에 대해 자동으로 청구합니다.
* **Usage Tracking**: 고객의 리소스 소비를 실시간으로 모니터링합니다.
* **Flexible Pricing**: 다양한 리소스(스토리지, 대역폭, 사이트 등)에 대해 맞춤 요율을 설정합니다.
* **Automatic Invoicing**: 사용량 임계값을 초과하면 자동으로 인보이스를 생성합니다.
* **Usage Reports**: 고객 및 관리자를 위한 상세 사용량 분석 및 보고서를 제공합니다.

## 적합한 대상

* **Hosting Providers**: 고객의 요구에 따라 확장 가능한 플랜을 제공합니다.
* **SaaS Platforms**: 실제 리소스 소비량에 따라 청구합니다.
* **Agency Networks**: 클라이언트 프로젝트를 위한 유연한 가격 모델을 제공합니다.
* **Reseller Hosting**: 파트너가 사용량 기반 가격을 제공할 수 있도록 합니다.

## 작동 방식

1. 측정 리소스를 구성합니다 (디스크 공간, 대역폭, 사이트 수 등).
2. 기본 플랜 한도와 초과 요율을 설정합니다.
3. 시스템은 네트워크 전반에 걸쳐 사용량을 자동으로 추적합니다.
4. 고객이 한도를 초과하면 초과 요금이 계산됩니다.
5. 청구 주기에 따라 인보이스가 자동으로 생성됩니다.
6. 고객은 실시간으로 사용량과 향후 요금을 확인할 수 있습니다.

## 지원되는 리소스

* 디스크 저장소
* 대역폭/데이터 전송
* 사이트 수
* 사용자 수
* 사용자 정의 리소스 유형

## 필수 사항

* Multisite Ultimate 2.0.0 이상
* WordPress Multisite 설치
* PHP 7.4 이상

## 설치

1. 플러그인 파일을 업로드합니다 `/wp-content/plugins/ultimate-multisite-metered-plans`.
2. WordPress의 '플러그인' 화면에서 플러그인을 활성화합니다.
3. Multisite Ultimate → 설정 → 측정 플랜으로 이동합니다.
4. 측정 리소스와 가격을 구성합니다.
5. 제품 편집기에서 제품에 대한 측정 기반 청구를 활성화합니다.

## 자주 묻는 질문

### 고객이 한도를 초과하면 어떻게 됩니까?

접근을 차단하는 대신, 시스템은 초과 사용량을 추적하고 귀하가 설정한 요율에 따라 다음 인보이스에 추가합니다.

### 다른 제품마다 다른 초과 요율을 설정할 수 있나요?

네, 각 제품은 고유한 측정 리소스와 초과 요금을 가질 수 있습니다.

### 고객에게 사용량 알림은 어떻게 제공됩니까?

고객은 대시보드에서 현재 사용량을 확인하고 한계에 접근하거나 초과할 때 선택적 알림을 받을 수 있습니다.

### 여전히 하드 리미트를 사용할 수 있나요?

네, 측정 기반 청구는 제품별로 선택 사항입니다. 전통적인 하드 리미트 제품과 측정 기반 제품을 혼합하여 사용할 수 있습니다.
