# Site Exporter 변경 로그

버전: 1.0.6 - 2026-05-11에 릴리스됨
* 신규: 이제 사이트 내보내기는 자체 부팅 `index.php`를 함께 묶어, 별도의 plugin 설치 없이 ZIP을 새 호스트에 설치할 수 있습니다
* 신규: 네트워크 내보내기를 통해 관리자는 Site Export 관리자 페이지에서 모든 하위 사이트를 단일 아카이브로 내보낼 수 있습니다

버전: 1.0.5 - 2026-05-05에 릴리스됨
* 수정: 테스트 부트스트랩 실패를 방지하기 위해, `get_config` 같은 모킹되지 않은 메서드를 처리하도록 WP_CLI 폴리필 스텁에 __callStatic을 추가했습니다

버전: 1.0.4 - 2025-11-25에 릴리스됨

* 수정: Ultimate Multisite 2.4와의 호환성.

버전: 1.0.2 - 2025-09-28에 릴리스됨

* `ultimate-multisite`로 접두사 이름 변경; 텍스트 도메인 업데이트; 버전 상향.

### 버전 1.0.1 - 2023-08-09에 릴리스됨

* 개선: 이제 importer 파일의 올바른 파일 경로를 확보합니다.
* 수정: 가져오기 중 오류를 방지하기 위해 객체 캐시를 비활성화했습니다.
* 수정: importer plugin 다운로드 링크가 작동하지 않는 문제를 수정했습니다.

### 버전 1.0.0 - 2022-12-23에 릴리스됨

* 수정: 내보내기 양식 팝업이 로드되지 않음;
* 수정: 내보내기 페이지에서 Javascript 파일이 로드되지 않음;
* 내부: 새로운 빌드 프로세스;

### 버전 1.0.0-beta.4 - 2022-01-13에 릴리스됨

* 내부: 훅 및 필터 생성기 추가;
* 내부: 개발자 편의성을 위한 WP Ultimo 스텁 추가;
