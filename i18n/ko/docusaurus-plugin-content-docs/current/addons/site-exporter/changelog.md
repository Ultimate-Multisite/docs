# 사이트 익스포터 변경 로그

Version: 1.0.4 - 2025-11-25에 릴리즈

* Ultimate Multisite 2.4와의 호환성 수정.

Version: 1.0.2 - 2025-09-28에 릴리즈

* 접두사를 ultimate-multisite로 변경; 텍스트 도메인 업데이트; 버전 업그레이드.

### Version 1.0.1 - 2023-08-09에 릴리즈

* 개선: 이제 가져오기 파일의 올바른 파일 경로가 확보됩니다.
* 수정: 가져오기 중 오류를 방지하기 위해 객체 캐시를 비활성화했습니다.
* 수정: 가져오기 플러그인 다운로드 링크가 작동하지 않는 문제를 해결했습니다.

### Version 1.0.0 - 2022-12-23에 릴리즈

* 수정: 내보내기 양식 팝업이 로드되지 않음;
* 수정: 내보내기 페이지에서 자바스크립트 파일이 로드되지 않음;
* 내부: 새로운 빌드 프로세스;

### Version 1.0.0-beta.4 - 2022-01-13에 릴리즈

* 내부: 훅 및 필터 생성기 추가;
* 내부: 개발자 편의를 위한 Ultimate Multisite 스텁 추가;
