# Discovery Interview {#discovery-interview}

**Discovery Interview**는 Theme Builder 디자인 프로세스의 첫 단계입니다. 이 인터뷰는 귀하의 사이트, 비즈니스, 디자인 선호도에 대한 정보를 수집하여 맞춤형 테마를 만드는 데 사용됩니다.

## 개요 {#overview}

Discovery Interview는 일련의 질문을 통해 귀하의 브랜드, 타겟 오디언스, 디자인 목표를 이해하도록 안내합니다. 버전 1.16.0부터는 디자인에 대한 시각적 영감을 제공할 수 있는 **사진 업로드 단계**가 추가되었습니다.

## 인터뷰 단계 {#interview-steps}

### 1. 비즈니스 정보 {#1-business-information}

- 사이트 이름 및 태그라인
- 비즈니스 유형 또는 산업
- 주요 비즈니스 목표
- 타겟 오디언스 설명

### 2. 디자인 선호도 {#2-design-preferences}

- 선호하는 색상 조합
- 디자인 스타일 (모던, 클래식, 미니멀리즘, 경쾌함 등)
- 영감 웹사이트 또는 브랜드
- 필수 디자인 요소

### 3. 사진 업로드 (v1.16.0 신규) {#3-photo-upload-new-in-v1160}

디자인 방향에 영향을 줄 사진을 하나 이상 업로드하세요.

#### 허용 형식 {#accepted-formats}

| 형식 | 확장자 | 상세 내용 |
|--------|-----------|---------|
| **JPEG** | `.jpg`, `.jpeg` | 사진에 권장 |
| **PNG** | `.png` | 투명도 지원 |
| **WebP** | `.webp` | 최신 형식, 파일 크기가 작음 |
| **GIF** | `.gif` | 정지 이미지 전용 (애니메이션 불가) |

#### 파일 요구 사항 {#file-requirements}

- **최대 파일 크기:** 10 MB
- **최소 크기:** 400 × 300 픽셀
- **권장 크기:** 1200 × 800 픽셀 이상
- **가로세로 비율:** 모든 비율 허용

#### 사진이 디자인 방향에 영향을 주는 방식 {#how-photos-influence-design-direction}

업로드한 사진은 Theme Builder가 다음을 이해하는 데 도움을 줍니다.

- **색상 팔레트** — 지배적인 색상이 추출되어 테마에 사용됩니다.
- **시각적 스타일** — 모던, 소박함, 미니멀리즘, 생동감 등.
- **분위기 및 분위기** — 전문적, 창의적, 친근함, 고급스러움 등.
- **디자인 요소** — 패턴, 질감, 구성 스타일.
- **타이포그래피 영감** — 시각적 테마를 보완하는 글꼴 스타일.

### 4. 콘텐츠 구조 {#4-content-structure}

- 필요한 주요 페이지 (홈, 소개, 서비스, 문의 등)
- 블로그 또는 뉴스 섹션 필요 여부
- 이커머스 기능 필요 여부
- 특별 기능 또는 통합 필요 여부

### 5. 검토 및 확인 {#5-review-and-confirmation}

- 제공된 모든 정보 요약
- 디자인 생성이 시작되기 전 확인 절차

## 사진 업로드 모범 사례 {#photo-upload-best-practices}

### 업로드할 내용 {#what-to-upload}

- **브랜드 영감** — 로고, 색상 견본, 또는 브랜드 가이드라인
- **경쟁사 사이트** — 감명 깊게 본 사이트의 스크린샷
- **무드 보드** — 원하는 미학을 나타내는 이미지 모음
- **제품 사진** — 전시할 제품이 있는 경우
- **팀 또는 사무실 사진** — 개인 또는 전문 브랜딩용
- **라이프스타일 이미지** — 타겟 오디언스를 대표하는 이미지

### 피해야 할 내용 {#what-to-avoid}

- **저화질 이미지** — 흐릿하거나 픽셀이 깨진 사진
- **관련성 낮은 이미지** — 브랜드를 대표하지 않는 사진
- **텍스트가 많은 이미지** — 텍스트가 많은 스크린샷
- **저작권이 있는 자료** — 이미지 사용 권한을 확보했는지 확인하세요.
- **애니메이션 GIF** — 정지 이미지만 처리됩니다.

### 최고의 결과를 위한 팁 {#tips-for-best-results}

1. **2~3개의 이미지 업로드** — 여러 이미지가 더 나은 맥락을 제공합니다.
2. **고화질 사진 사용** — 선명하고 조명이 잘 된 이미지가 가장 좋습니다.
3. **다양성 포함** — 색상, 스타일, 주제를 섞어 사용하세요.
4. **맥락 제공** — 각 이미지를 선택한 이유를 간략하게 설명하세요.
5. **구체적으로 설명** — 특정 요소가 마음에 든다면, 그 부분을 짚어주세요.

## 디자인 방향 결과물 {#design-direction-output}

Discovery Interview를 완료하면 Theme Builder가 다음을 생성합니다.

- **색상 팔레트** — 기본색, 보조색, 강조색
- **타이포그래피** — 제목 및 본문 텍스트용 글꼴 패밀리
- **레이아웃 스타일** — 간격, 정렬, 구성 요소 배치
- **시각적 요소** — 아이콘, 패턴, 장식 요소
- **무드 보드** — 디자인 방향의 시각적 표현

## 다음 단계 {#next-steps}

Discovery Interview를 완료한 후:

1. 생성된 디자인 방향 검토
2. 피드백 제공 또는 조정 요청
3. [Design Direction](./design-direction.md) 단계로 진행
4. Theme Builder가 맞춤형 테마를 생성합니다.
