# 도구 및 설정

Theme Builder는 사용자의 WordPress 사이트를 분석하고, 디자인하며, 구축하는 데 필요한 일련의 도구들을 사용합니다. 버전 1.16.0부터 **sd-ai-agent/site-scrape가 Tier 1 도구로 지정**되어, Theme Builder에서 기본적으로 사용할 수 있게 되었습니다.

## 사용 가능한 도구

### Tier 1 도구 (항상 사용 가능)

Tier 1 도구는 추가 설정 없이 Theme Builder에서 기본적으로 사용할 수 있습니다.

#### sd-ai-agent/site-scrape

**용도:** 기존 웹사이트를 분석하여 디자인 영감, 콘텐츠 구조, 기능성을 추출합니다.

**기능:**

- **웹사이트 분석** — 경쟁사 또는 영감을 얻을 웹사이트를 스크래핑하고 분석합니다.
- **디자인 추출** — 색상, 글꼴, 레이아웃 패턴을 식별합니다.
- **콘텐츠 구조** — 페이지 구성과 계층 구조를 이해합니다.
- **기능 감지** — 플러그인 및 기능성을 식별합니다.
- **성능 분석** — 페이지 속도 및 최적화를 확인합니다.
- **SEO 분석** — 메타 태그 및 구조화된 데이터를 검토합니다.

**사용 예시:**

```
example.com의 디자인을 분석하고, 이를 제 사이트의 영감으로 활용해 주세요.
```

**추출 항목:**

- 색상 팔레트 및 색상 사용 방식
- 타이포그래피 (글꼴 및 크기)
- 레이아웃 구조 및 간격
- 내비게이션 패턴
- 콘텐츠 구성
- 이미지 및 미디어 사용
- 인터랙티브 요소
- 모바일 반응성

**제한 사항:**

- 비밀번호로 보호된 사이트는 스크래핑할 수 없습니다.
- robots.txt 및 사이트 정책을 준수합니다.
- 동적 콘텐츠를 포착하지 못할 수 있습니다.
- JavaScript가 많은 사이트는 분석이 제한적일 수 있습니다.
- 대규모 사이트는 분석하는 데 시간이 더 오래 걸릴 수 있습니다.

### Tier 2 도구 (선택 사항)

Tier 2 도구는 Theme Builder 설정에서 명시적으로 활성화해야 사용할 수 있습니다.

#### Advanced Analytics

상세한 성능 지표를 제공합니다:

- 페이지 로드 시간
- Core Web Vitals
- SEO 점수
- 접근성 점수
- 모범 사례 점수

#### Content Optimizer

다음 항목에 대한 분석 및 개선 사항을 제안합니다:

- 가독성
- SEO 최적화
- 키워드 사용
- 콘텐츠 구조
- 콜투액션(Call-to-action) 배치

### Tier 3 도구 (프리미엄)

Tier 3 도구는 추가 설정 또는 프리미엄 기능이 필요합니다.

#### AI Content Generator

다음 항목에 대한 콘텐츠를 생성합니다:

- 제품 설명
- 서비스 페이지
- 블로그 게시물
- 메타 설명
- 콜투액션 문구

#### Advanced Design Tools

고급 디자인 기능을 제공합니다:

- 사용자 지정 CSS 생성
- 애니메이션 제작
- 인터랙티브 요소 디자인
- 고급 색채 이론
- 타이포그래피 최적화

## 도구 설정

### 도구 활성화

Theme Builder에서 추가 도구를 활성화하려면:

1. **Gratis AI Agent → Settings**로 이동합니다.
2. **Theme Builder → Tools**로 이동합니다.
3. 필요한 대로 도구를 켜거나 끕니다.
4. 설정을 저장합니다.

### 도구 권한

일부 도구는 권한이 필요합니다:

- **사이트 스크래핑** — 인터넷 접속 권한이 필요합니다.
- **분석 도구** — Google Analytics 통합이 필요합니다.
- **콘텐츠 생성** — API 키가 필요합니다.
- **고급 기능** — 프리미엄 구독이 필요할 수 있습니다.

### API 키 및 자격 증명

API 키가 필요한 도구의 경우 설정을 구성합니다:

1. **Gratis AI Agent → Settings → API Keys**로 이동합니다.
2. 각 도구에 대한 자격 증명을 입력합니다.
3. 연결을 테스트합니다.
4. 안전하게 저장합니다.

## sd-ai-agent/site-scrape 사용하기

### 기본 사용법

Theme Builder에게 웹사이트 분석을 요청합니다:

```
경쟁사 사이트인 competitor.com의 디자인을 분석하고, 제 사이트를 위한 디자인 개선 사항을 제안해 주세요.
```

### 특정 분석 요청

특정 유형의 분석을 요청합니다:

```
example.com에서 색상 팔레트를 추출하여 영감으로 사용해 주세요.
```

```
example.com의 내비게이션 구조를 분석하고, 유사한 구성을 제 사이트에 적용해 주세요.
```

```
example.com의 모바일 반응성을 확인하고, 제 사이트도 동등하게 반응하도록 해주세요.
```

### 비교 분석

여러 사이트를 비교합니다:

```
site1.com과 site2.com의 디자인을 비교하여, 두 사이트의 가장 좋은 요소를 결합한 디자인을 만들어 주세요.
```

## 도구 제한 사항 및 고려 사항

### 속도 제한 (Rate Limiting)

- 서버 과부하를 방지하기 위해 스크래핑은 속도 제한이 적용됩니다.
- 동일한 사이트에 대한 여러 요청은 제한될 수 있습니다.
- 대규모 사이트는 분석하는 데 시간이 더 오래 걸릴 수 있습니다.

### 콘텐츠 정확성

- 동적 콘텐츠는 완전히 포착되지 않을 수 있습니다.
- JavaScript로 렌더링된 콘텐츠는 불완전할 수 있습니다.
- 실시간 데이터가 반영되지 않을 수 있습니다.
- 일부 콘텐츠는 유료 장벽 뒤에 있을 수 있습니다.

### 법적 및 윤리적 고려 사항

- robots.txt 및 사이트 정책을 준수합니다.
- 저작권이 있는 콘텐츠를 복제하기 위해 스크래핑하지 않습니다.
- 분석은 영감을 얻기 위한 것이지, 복사하기 위한 것이 아닙니다.
- 추출된 콘텐츠를 사용할 권리가 있는지 확인합니다.
- 분석된 사이트의 서비스 약관을 따릅니다.

### 성능 영향

- 대규모 사이트 스크래핑은 시간이 걸릴 수 있습니다.
- 여러 스크래핑을 동시에 수행하면 더 느릴 수 있습니다.
- 네트워크 연결 상태가 속도에 영향을 미칩니다.
- 대규모 데이터 세트는 더 많은 처리가 필요할 수 있습니다.

## 모범 사례

### 사이트 분석 활용하기

1. **여러 사이트 분석** — 여러 출처에서 영감을 얻습니다.
2. **구조에 집중** — 레이아웃 및 구성 패턴을 학습합니다.
3. **색상 추출** — 색상 팔레트를 시작점으로 사용합니다.
4. **타이포그래피 연구** — 마음에 드는 글꼴 조합을 식별합니다.
5. **내비게이션 검토** — 메뉴 구성을 이해합니다.

### 윤리적 스크래핑

1. **robots.txt 준수** — 사이트 가이드라인을 따릅니다.
2. **콘텐츠 복사 금지** — 분석은 영감만을 위해 사용합니다.
3. **권리 확인** — 추출된 콘텐츠를 사용할 수 있는지 확인합니다.
4. **출처 명시** — 영감의 출처를 밝힙니다.
5. **약관 준수** — 사이트 서비스 약관을 준수합니다.

### 도구 효과 극대화하기

1. **구체적으로 요청** — 특정 유형의 분석을 요청합니다.
2. **맥락 제공** — 사이트의 목적을 설명합니다.
3. **기대치 설정** — 디자인 목표를 설명합니다.
4. **반복 개선** — 결과에 따라 다듬어 나갑니다.
5. **도구 결합** — 포괄적인 분석을 위해 여러 도구를 사용합니다.

## 문제 해결

### 사이트 스크래핑 불가

- 사이트가 공개적으로 접근 가능한지 확인합니다.
- robots.txt가 스크래핑을 허용하는지 확인합니다.
- 다른 사이트를 시도해 봅니다.
- 인터넷 연결을 확인합니다.
- 문제가 지속되면 지원팀에 문의합니다.

### 분석이 불완전함

- 사이트에 동적 콘텐츠가 있을 수 있습니다.
- JavaScript가 완전히 렌더링되지 않았을 수 있습니다.
- 대규모 사이트는 시간 초과가 발생할 수 있습니다.
- 대신 특정 페이지를 분석해 봅니다.
- 특정 분석 유형을 요청합니다.

### 성능 저하

- 대규모 사이트는 분석하는 데 시간이 더 오래 걸립니다.
- 여러 스크래핑을 동시에 수행하면 더 느립니다.
- 네트워크 연결 상태가 속도에 영향을 미칩니다.
- 비피크 시간대에 시도해 봅니다.
- 먼저 작은 섹션부터 분석합니다.
