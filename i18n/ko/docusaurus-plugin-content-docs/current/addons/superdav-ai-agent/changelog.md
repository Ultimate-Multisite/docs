# 변경 로그

## 1.16.0 — 배포일 2026-05-20

### 새로 추가된 기능

- **로고 SVG 생성 기능** — Theme Builder에서 네임스페이스 안전한 검증을 거쳐 사용자 지정 로고 SVG를 생성하고 삽입할 수 있습니다.
- **디스커버리 인터뷰 사진 업로드** — Theme Builder의 디스커버리 인터뷰에 사진 업로드 단계가 추가되어 더 풍부한 디자인 컨텍스트를 확보할 수 있습니다.
- **팔레트 대비 검증 기능** — 테마에 적용하기 전에 색상 조합이 WCAG를 준수하는지 확인할 수 있습니다.
- **호스피탈리티 메뉴** — Theme Builder에서 숙박/요식업체에 맞는 구조화된 식음료 메뉴 페이지를 생성할 수 있습니다.
- **데스크톱 및 모바일 미리보기 렌더링** — 디자인 방향을 선택하는 동안 데스크톱 및 모바일 기기에서 디자인을 미리 볼 수 있습니다.
- **탐색 레이블 매개변수** — Create Menu 기능에서 페이지 제목과 별개인 고유한 `navigation_label`을 지원합니다.
- **Tier 1 도구 사용 가능** — sd-ai-agent/site-scrape가 이제 Theme Builder에서 기본적으로 사용할 수 있는 Tier 1 도구로 추가되었습니다.

### 수정된 기능

- **AI 클라이언트 캐시** — 이제 트랜지언트(transients)를 기반으로 하여 요청 간 지속성을 유지하므로, 장시간 실행되는 에이전트 작업 중 데이터 손실을 방지합니다.
- **플러그인 행 액션 링크** — 명확성을 위해 수정 및 이름이 변경되었습니다.

## 1.10.0 — 배포일 2026-05-05

### 새로 추가된 기능

- **Tavily 인터넷 검색** — Brave Search와 함께 더 풍부한 인터넷 검색 결과를 위해 검색 제공업체로 Tavily가 추가되었습니다.
- **테마 인식 내장 스킬** — Block Themes, Classic Themes, Kadence Blocks, Kadence Theme 스킬 가이드가 플러그인에 포함되어 제공됩니다.
- **사이트 빌더 문의 양식 기능** — 채팅 인터페이스에서 어떤 페이지에든 문의 양식을 직접 추가할 수 있습니다.

### 개선된 기능

- **WooCommerce 통합 기능 리팩토링** — 이제 네이티브 WooCommerce API를 사용하여 안정성과 호환성이 향상되었습니다.
- **제공업체 목록 자동 새로고침** — 플러그인을 활성화하거나 비활성화할 때 자동으로 목록이 새로고침됩니다.

### 수정된 기능

- **navigate-to 기능** — 일부 관리자 페이지에서 무한 새로고침 루프가 발생하는 문제가 수정되었습니다.
- **list-posts 기능** — 카테고리 및 태그 이름을 슬러그로 올바르게 해석합니다.
- **WP-CLI 명령어** — 이전 리팩토링으로 인해 누락되었던 네임스페이스 별칭이 복원되었습니다.
- **이벤트 자동화** — 자동화 테이블이 아직 생성되지 않은 사이트에서도 정상적으로 처리합니다.
- **memory-save 기능** — 시스템 명령어 빌더에서 올바른 네임스페이스 접두사를 사용합니다.
- **Scalar 도구 결과** — AI에 반환되기 전에 올바르게 래핑됩니다.
- **사용량 통계** — 이전 버전에서 업그레이드할 때 레거시 기능 키 형식을 올바르게 처리합니다.
