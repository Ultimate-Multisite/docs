# 변경 로그 {#changelog}

## 1.18.0 — 2026-06-29에 릴리스됨 {#1180--released-on-2026-06-29}

### 신규 {#new}

- **Google Calendar 도구** — 일정 인식 자동화를 위해 구성된 캘린더와 이벤트 읽기
- **연락처 매핑 및 참석자 도우미** — 캘린더 참가자를 사이트 사용자 및 연락처와 매칭
- **사용자 승인 게이트 및 알림 기록** — 검토를 위해 자동화를 일시 중지하고 중복 알림 방지
- **TextBee SMS 제공업체** — TextBee를 통해 구성된 문자 메시지 알림 전송
- **고급 동반 패키지** — 신뢰할 수 있는 개발자용 파일시스템, 데이터베이스, WP-CLI, REST 디스패처, plugin 빌더, git 스냅샷, 사용자 관리 및 벤치마크 도구를 추가하며, WordPress.org 빌드와 별도로 배포됨

### 개선됨 {#improved}

- **관리형 Superdav 서비스 설정** — 지원되는 사이트를 위한 호스팅 서비스 endpoint 및 자동 연결 프로비저닝 추가
- **릴리스 패키징** — 별도의 코어 및 고급 ZIP을 빌드하고, 둘 다 GitHub에 게시하며, 코어 패키지만 WordPress.org로 전송

### 수정됨 {#fixed}

- **AI 요청 신뢰성** — 모델 선택, 요청 시간 초과, 기본 설정 모델, 추론 텍스트 처리 및 잘못된 도구 호출에 대한 재시도 안내 개선
- **캘린더 및 알림 강화** — Google Calendar 토큰 및 알림 중복 제거 강화
- **온보딩 및 승인 재개** — 프런트엔드 온보딩 실행 및 확인된 기능 재개 수정
- **WordPress.org 패키징 검토 문제** — 코어 릴리스에 대한 패키징 검토 피드백 반영

## 1.16.0 — 2026-05-20에 릴리스됨 {#1160--released-on-2026-05-20}

### 신규 {#new-1}

- **Logo SVG 생성 기능** — Theme Builder가 이제 네임스페이스 안전한 정리를 통해 사용자 지정 로고 SVG를 생성하고 삽입할 수 있음
- **탐색 인터뷰의 사진 업로드** — Theme Builder 탐색 인터뷰에 이제 더 풍부한 디자인 맥락을 위한 사진 업로드 단계 포함
- **팔레트 대비 검증 기능** — theme에 적용하기 전에 WCAG 준수 여부를 위해 색상 쌍 확인
- **호스피탈리티 메뉴** — Theme Builder가 이제 호스피탈리티 비즈니스를 위한 구조화된 음식 및 음료 메뉴 페이지를 생성할 수 있음
- **데스크톱 및 모바일 미리보기 렌더링** — 디자인 방향 선택 중 데스크톱 및 모바일 기기에서 디자인 미리보기
- **탐색 레이블 매개변수** — 메뉴 생성 기능이 이제 페이지 제목과 별도의 고유한 `navigation_label`을 지원
- **Tier 1 도구 사용 가능** — sd-ai-agent/site-scrape가 이제 Theme Builder에서 기본적으로 사용할 수 있는 Tier 1 도구임

### 수정됨 {#fixed-1}

- **AI Client 캐시** — 이제 요청 간 지속성을 위해 transients로 뒷받침되어 장시간 실행되는 agent 작업에서 데이터 손실 방지
- **Plugin 행 작업 링크** — 명확성을 위해 수정 및 이름 변경

## 1.10.0 — 2026-05-05에 릴리스됨 {#1100--released-on-2026-05-05}

### 신규 {#new-2}

- **Tavily 인터넷 검색** — Brave Search와 함께 더 풍부한 인터넷 검색 결과를 위해 Tavily를 검색 제공업체로 추가
- **Theme 인식 내장 스킬** — Block Themes, Classic Themes, Kadence Blocks 및 Kadence Theme 스킬 가이드가 이제 plugin과 함께 제공됨
- **사이트 빌더 연락처 양식 기능** — 채팅 인터페이스에서 직접 어떤 페이지에나 연락처 양식 추가

### 개선됨 {#improved-1}

- **WooCommerce 통합 리팩터링** — 더 나은 신뢰성과 호환성을 위해 이제 기본 WooCommerce API 사용
- **제공업체 목록 자동 새로고침** — plugin이 활성화되거나 비활성화될 때마다

### 수정됨 {#fixed-2}

- **navigate-to 기능** — 일부 관리자 페이지에서 무한 새로고침 루프 수정
- **list-posts 기능** — 이제 카테고리 및 태그 이름을 slug로 올바르게 해석
- **WP-CLI 명령** — 이전 리팩터링 후 누락된 네임스페이스 별칭 복원
- **이벤트 자동화** — 자동화 테이블이 아직 생성되지 않은 사이트를 원활하게 처리
- **memory-save 기능** — 이제 시스템 지침 빌더에서 올바른 네임스페이스 접두사 사용
- **스칼라 도구 결과** — 이제 AI에 반환되기 전에 올바르게 래핑됨
- **사용 통계** — 이제 이전 버전에서 업그레이드할 때 레거시 기능 키 형식을 올바르게 처리
