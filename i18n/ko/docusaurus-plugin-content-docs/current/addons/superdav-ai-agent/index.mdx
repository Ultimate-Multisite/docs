# Superdav AI Agent

Superdav AI Agent는 Gratis AI Agent를 확장하여 테마 인식 기능, 향상된 인터넷 검색 기능, 개선된 사이트 빌더 기능을 추가한 고급 WordPress AI 비서입니다. 버전 1.16.0에서는 사용자 지정 로고 생성, 팔레트 대비 검증, 발견 인터뷰 시 사진 업로드, 구조화된 호스피탈리티 메뉴 생성, 그리고 테마 빌더에서 데스크톱/모바일 미리보기 렌더링 기능이 추가되었습니다.

## 주요 기능

- **테마 인식 기능** — 현재 사용 중인 WordPress 테마와 설치된 플러그인을 기반으로 안내 및 기능을 자동으로 조정합니다.
- **다중 인터넷 검색 제공업체** — 더 풍부한 검색 결과와 고급 리서치 기능을 위해 Brave Search와 Tavily 중 선택할 수 있습니다.
- **사이트 빌더 연락처 양식** — 설치된 양식 플러그인을 사용하여 채팅 인터페이스에서 바로 작동하는 연락처 양식을 만들 수 있습니다.
- **향상된 사이트 빌더** — 플러그인 발견 및 오류 복구 기능이 개선된 다단계 사이트 빌드 계획을 제공합니다.
- **내장 테마 기능** — Block Themes, Classic Themes, Kadence Blocks, Kadence Theme에 특화된 안내 기능을 제공합니다.
- **개선된 WooCommerce 통합** — 안정성과 호환성을 높이기 위해 네이티브 WooCommerce API를 사용하도록 재구성되었습니다.
- **자동 제공업체 감지** — 플러그인이나 테마가 활성화되거나 비활성화될 때 사용 가능한 플러그인과 테마를 자동으로 감지하고 새로 고칩니다.
- **사용자 지정 로고 생성** — 자동 네임스페이스 안전화(sanitisation)를 통해 사용자 지정 SVG 로고를 생성하고 삽입할 수 있습니다.
- **팔레트 대비 검증** — 테마에 적용하기 전에 색상 조합이 WCAG 접근성 표준을 준수하는지 확인할 수 있습니다.
- **사진 기반 발견** — 테마 빌더 발견 인터뷰 중 사진을 업로드하여 디자인 방향에 영향을 줄 수 있습니다.
- **호스피탈리티 메뉴** — 레스토랑, 카페, 호스피탈리티 비즈니스를 위한 구조화된 식음료 메뉴 페이지를 생성합니다.
- **데스크톱 및 모바일 미리보기** — 디자인 방향을 선택하는 동안 데스크톱 및 모바일 장치에서 디자인을 미리 볼 수 있습니다.
- **Tier 1 사이트 스크래핑** — sd-ai-agent/site-scrape가 기본적으로 제공되어 경쟁사 및 영감 웹사이트를 분석할 수 있습니다.

## 요구 사항

- WordPress 6.0 이상
- PHP 8.1 이상
- WordPress Multisite (네트워크 활성화) 또는 단일 사이트 설치
- 활성화된 AI 제공업체 커넥터 플러그인 (Anthropic, OpenAI 또는 호환되는 플러그인)
- Gratis AI Agent v1.9.0 이상 (Superdav AI Agent는 Gratis AI Agent를 확장합니다)

## 설치

1. 플러그인 파일을 `/wp-content/plugins/superdav-ai-agent`에 업로드합니다.
2. WordPress의 **플러그인** 화면에서 플러그인을 활성화합니다.
3. Gratis AI Agent도 활성화되어 있는지 확인합니다.
4. **Gratis AI Agent → 설정**으로 이동하여 인터넷 검색 제공업체를 구성합니다 (선택 사항).

## 빠른 시작

활성화하면 Superdav AI Agent가 테마 인식 기능과 추가 기능을 통해 Gratis AI Agent 설치를 자동으로 향상시킵니다. WordPress 관리자 사이드바에서 **Gratis AI Agent** 패널을 열고 새로운 기능을 사용해 보세요.

**예시 명령어:**

```
Add a contact form to the Contact page.
```

```
Search the web for the latest WordPress security best practices.
```

```
Create a hero section using the active theme's block patterns.
```

## 테마 인식 기능

Superdav AI Agent는 활성 테마와 설치된 플러그인을 자동으로 감지하여 적절한 테마 인식 기능을 비서의 컨텍스트에 로드합니다. 사용 가능한 기능은 다음과 같습니다.

| 기능 | 적용 대상 |
|---|---|
| **Block Themes** | WordPress 블록 기반(Full Site Editing) 아키텍처를 사용하는 테마 |
| **Classic Themes** | PHP 템플릿과 클래식 에디터를 사용하는 전통적인 WordPress 테마 |
| **Kadence Blocks** | Kadence Blocks 플러그인이 활성화된 사이트 |
| **Kadence Theme** | Kadence 테마가 활성 테마인 사이트 |

각 기능에 대한 자세한 안내는 [Theme-Aware Skills](../../user-guide/configuration/theme-aware-skills)를 참조하세요.

## 인터넷 검색 제공업체

Superdav AI Agent v1.10.0에서는 인터넷 검색 쿼리에 대한 대안으로 Brave Search 외에 **Tavily**를 추가했습니다. 두 제공업체 모두 실시간 검색 결과를 제공하며, **Gratis AI Agent → 설정 → 고급**에서 구성할 수 있습니다.

| 제공업체 | 기능 | 설정 |
|---|---|---|
| **Brave Search** | 빠르고 개인 정보 보호에 중점을 둔 검색 결과 | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | 더 풍부한 검색 결과, 고급 리서치 기능 | [Tavily API](https://tavily.com/) |

구성 지침은 [Internet Search](../../user-guide/configuration/internet-search)를 참조하세요.

## 사이트 빌더 연락처 양식

사이트 빌더 에이전트는 이제 채팅 인터페이스에서 직접 연락처 양식을 만들 수 있습니다. 사이트 빌더에게 연락처 양식 생성을 요청하기만 하면, 설치된 양식 플러그인(Contact Form 7, WPForms, Fluent Forms, 또는 Gravity Forms)을 자동으로 감지하여 바로 사용할 수 있는 양식을 생성합니다.

자세한 안내는 [Site Builder Contact Forms](../../user-guide/administration/site-builder-contact-forms)를 참조하세요.

## 테마 빌더 기능

Superdav AI Agent v1.16.0의 테마 빌더에는 강력한 새로운 기능들이 포함되어 있습니다.

### 사진 업로드를 통한 발견 인터뷰

테마 빌더 발견 인터뷰에는 이제 시각적 영감을 제공할 수 있는 사진 업로드 단계가 포함됩니다. 브랜드, 경쟁사 사이트 또는 디자인 영감을 주는 사진을 업로드하여 테마 빌더가 디자인 방향을 이해하도록 도와주세요.

자세한 안내는 [Discovery Interview](./theme-builder/discovery-interview.md)를 참조하세요.

### 데스크톱 및 모바일 미리보기를 통한 디자인 방향 설정

테마 빌더가 전체 테마를 생성하기 전에 데스크톱과 모바일 장치 모두에서 디자인을 검토할 수 있습니다. 여러 반응형 브레이크포인트에서 실시간 미리보기를 확인하세요.

자세한 안내는 [Design Direction](./theme-builder/design-direction.md)를 참조하세요.

### 호스피탈리티 메뉴

테마 빌더는 이제 레스토랑, 카페, 바 및 기타 호스피탈리티 비즈니스를 위한 구조화된 식음료 메뉴 페이지를 생성할 수 있습니다. 메뉴에는 가격, 설명, 식단 정보, 알레르기 경고 등이 포함됩니다.

자세한 안내는 [Hospitality Menus](./theme-builder/hospitality-menus.md)를 참조하세요.

### 새로운 기능

- **[Generate Logo SVG](./abilities/generate-logo-svg.md)** — 자동 안전화를 통해 사용자 지정 SVG 로고를 생성합니다.
- **[Validate Palette Contrast](./abilities/validate-palette-contrast.md)** — 색상 조합이 WCAG 표준을 준수하는지 확인합니다.
- **[Create Menu](./abilities/create-menu.md)** — 고유한 탐색 레이블을 가진 내비게이션 메뉴를 생성합니다.

사용 가능한 도구 및 Tier 1 사이트 스크래핑 도구에 대한 정보는 [Tools and Configuration](./theme-builder/tools.md)를 참조하세요.

## 설정

모든 설정은 WordPress 관리자에서 **Gratis AI Agent → 설정**에 있습니다. Superdav AI Agent는 다음 구성 옵션을 추가합니다.

| 설정 | 설명 |
|---|---|
| **Internet Search Provider** | Brave Search와 Tavily 중 선택합니다 (선택 사항) |
| **Tavily API Key** | Tavily 인터넷 검색용 API 키 (Tavily를 선택한 경우) |

## v1.16.0 개선 사항

- **AI 클라이언트 캐시** — 이제 트랜지언트를 백업하여 요청 간 지속성을 확보함으로써 장시간 실행되는 에이전트 작업 중 데이터 손실을 방지합니다.
- **Tier 1 사이트 스크래핑 도구** — sd-ai-agent/site-scrape가 테마 빌더에서 기본적으로 제공되어 경쟁사 및 영감 웹사이트를 분석할 수 있습니다.
- **플러그인 행 액션 링크** — 명확성을 위해 수정 및 이름이 변경되었습니다.

## v1.10.0 개선 사항

- **WooCommerce 통합 재구성** — 더 나은 안정성과 호환성을 위해 네이티브 WooCommerce API를 사용합니다.
- **제공업체 목록 자동 새로 고침** — 플러그인이 활성화되거나 비활성화될 때 자동으로 감지하여 사용 가능한 제공업체 목록을 업데이트합니다.
- **향상된 사이트 빌더** — 오류 복구 기능이 개선된 다단계 사이트 빌드 계획을 제공합니다.
- **테마 인식 기능** — 활성 테마에 자동으로 적응하는 네 가지 새로운 내장 기능을 추가했습니다.

## v1.10.0 버그 수정

- 일부 관리자 페이지에서 무한 새로고침 루프를 유발하던 `navigate-to` 기능 수정
- 카테고리 및 태그 이름을 슬러그로 올바르게 해석하지 못하던 `list-posts` 기능 수정
- 이전 리팩토링 후 누락되던 네임스페이스 별칭을 포함하도록 WP-CLI 명령어 수정
- 자동화 테이블이 아직 생성되지 않은 사이트를 우아하게 처리하도록 이벤트 자동화 기능 수정
- 시스템 명령어 빌더에서 올바른 네임스페이스 접두사를 사용하는 `memory-save` 기능 수정
- 스칼라 도구 결과가 AI에 반환되기 전에 올바르게 래핑되도록 수정
- 이전 버전에서 업그레이드할 때 사용 통계가 레거시 기능 키 형식을 올바르게 처리하도록 수정
