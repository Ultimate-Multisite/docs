# Superdav AI Agent {#superdav-ai-agent}

Superdav AI Agent는 Theme 인식 기술, 향상된 인터넷 검색 기능, 사이트 빌더 기능, 선택적 자동화 통합으로 Gratis AI Agent를 확장하는 고급 WordPress AI 도우미입니다. 버전 1.18.0은 Google Calendar 읽기 도구, 연락처 및 참석자 매핑, 사람 승인 게이트, 알림 중복 제거, TextBee SMS notifications, 관리형 Superdav 서비스 프로비저닝, 신뢰할 수 있는 개발자 워크플로를 위해 별도로 배포되는 Advanced 동반 패키지를 추가합니다.

## 주요 기능 {#key-features}

- **Theme 인식 기술** — 활성 WordPress theme과 설치된 plugins를 기반으로 안내와 기능을 자동으로 조정합니다
- **여러 인터넷 검색 제공자** — 더 풍부한 검색 결과와 고급 조사 기능을 위해 Brave Search와 Tavily 중에서 선택합니다
- **사이트 빌더 문의 양식** — 설치된 양식 plugin을 사용하여 채팅 인터페이스에서 바로 완전히 작동하는 문의 양식을 만듭니다
- **향상된 사이트 빌더** — 더 나은 plugin 탐색과 오류 복구를 갖춘 개선된 다단계 사이트 구축 계획입니다
- **내장 theme 기술** — Block Themes, Classic Themes, Kadence Blocks, Kadence Theme을 위한 전문 안내입니다
- **개선된 WooCommerce 통합** — 더 나은 안정성과 호환성을 위해 네이티브 WooCommerce API를 사용하도록 리팩터링되었습니다
- **자동 제공자 감지** — plugins와 themes가 활성화되거나 비활성화될 때 사용 가능한 항목을 자동으로 감지하고 새로 고칩니다
- **사용자 지정 로고 생성** — 자동 네임스페이스 안전 정리 기능으로 사용자 지정 SVG 로고를 생성하고 삽입합니다
- **팔레트 대비 검증** — theme에 적용하기 전에 색상 쌍의 WCAG 접근성 준수 여부를 확인합니다
- **사진 강화 탐색** — 디자인 방향에 영향을 주기 위해 Theme Builder 탐색 인터뷰 중 사진을 업로드합니다
- **접객업 메뉴** — 레스토랑, 카페, 접객업 비즈니스를 위한 구조화된 음식 및 음료 메뉴 페이지를 생성합니다
- **데스크톱 및 모바일 미리보기** — 디자인 방향 선택 중 데스크톱 및 모바일 기기에서 디자인을 미리 봅니다
- **1단계 사이트 스크래핑** — 경쟁사 및 영감이 되는 웹사이트 분석을 위해 sd-ai-agent/site-scrape를 이제 기본으로 사용할 수 있습니다
- **Google Calendar 읽기 도구** — 자동화가 예정된 일정 맥락을 판단할 수 있도록 구성된 캘린더와 이벤트를 읽습니다
- **연락처 및 참석자 매핑** — 후속 작업을 생성하기 전에 캘린더 참가자를 알려진 사이트 사용자 및 연락처와 일치시킵니다
- **사람 승인 게이트** — 메시지를 보내거나, 작업을 재개하거나, 변경하기 전에 검토를 위해 민감한 자동화를 일시 중지합니다
- **알림 기록 및 중복 제거** — 이미 전송된 알림을 추적하여 반복 자동화 실행이 같은 수신자에게 두 번 알리지 않도록 합니다
- **TextBee SMS notifications** — SMS 제공자가 활성화된 경우 TextBee를 통해 구성된 문자 메시지 알림을 보냅니다
- **관리형 Superdav 서비스 프로비저닝** — 수동 endpoint 구성 대신 자동 설정으로 지원되는 사이트를 호스팅된 Superdav endpoints에 연결합니다
- **Advanced 동반 패키지** — 파일 시스템, 데이터베이스, WP-CLI, REST 디스패처, plugin 빌더, git 스냅샷, 사용자 관리, 벤치마크 워크플로를 위한 별도 배포 신뢰 개발자 도구입니다

## 요구 사항 {#requirements}

- WordPress 6.0 이상
- PHP 8.1 이상
- WordPress Multisite(네트워크 활성화) 또는 단일 사이트 설치
- 활성 AI 제공자 커넥터 plugin(Anthropic, OpenAI 또는 호환 가능)
- Gratis AI Agent v1.9.0 이상(Superdav AI Agent는 Gratis AI Agent를 확장합니다)

## 설치 {#installation}

1. plugin 파일을 `/wp-content/plugins/superdav-ai-agent`에 업로드합니다
2. WordPress의 **Plugins** 화면에서 plugin을 활성화합니다
3. Gratis AI Agent도 활성화되어 있는지 확인합니다
4. **Gratis AI Agent → Settings**로 이동하여 필요에 따라 인터넷 검색 제공자, 관리형 Superdav 서비스 연결, Google Calendar 또는 TextBee 자격 증명을 구성합니다

WordPress.org 패키지에는 핵심 Superdav AI Agent 기능이 포함되어 있습니다. Advanced 동반 패키지는 신뢰할 수 있는 개발자 도구가 명시적으로 설치되고 승인된 환경을 위해 별도로 빌드 및 배포됩니다.

## 빠른 시작 {#quick-start}

활성화되면 Superdav AI Agent가 Theme 인식 기술과 추가 기능으로 Gratis AI Agent 설치를 자동으로 강화합니다. WordPress 관리자 사이드바에서 **Gratis AI Agent** 패널을 열고 새 기능을 사용하기 시작하세요.

**예시 지시 사항:**

```
문의 페이지에 문의 양식을 추가하세요.
```

```

```

```
활성 theme의 블록 패턴을 사용하여 hero 섹션을 만드세요.
```

```
내일 구성된 Google Calendar 이벤트를 검토하고 참석자를 위한 알림 초안을 작성하세요.
```

```
제가 메시지를 승인한 후 SMS 알림을 보내세요.
```

## Theme 인식 기술 {#theme-aware-skills}

Superdav AI Agent는 활성 theme과 설치된 plugins를 자동으로 감지하여 적절한 Theme 인식 기술을 도우미의 맥락에 로드합니다. 사용 가능한 기술은 다음과 같습니다.

| 기술 | 적용 대상 |
|---|---|
| **Block Themes** | WordPress 블록 기반(Full Site Editing) 아키텍처를 사용하는 themes |
| **Classic Themes** | PHP 템플릿과 클래식 편집기를 사용하는 전통적인 WordPress themes |
| **Kadence Blocks** | Kadence Blocks plugin이 활성화된 사이트 |
| **Kadence Theme** | Kadence theme이 활성 theme인 사이트 |

각 기술에 대한 자세한 안내는 [Theme 인식 기술](../../user-guide/configuration/theme-aware-skills)을 참조하세요.

## 인터넷 검색 제공자 {#internet-search-providers}

Superdav AI Agent v1.10.0은 인터넷 검색 쿼리를 위한 Brave Search의 대안으로 **Tavily**를 추가합니다. 두 제공자 모두 실시간 검색 결과를 제공하며 **Gratis AI Agent → Settings → Advanced**에서 구성할 수 있습니다.

| 제공자 | 기능 | 설정 |
|---|---|---|
| **Brave Search** | 빠르고 개인정보 보호에 중점을 둔 검색 결과 | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | 더 풍부한 검색 결과, 고급 조사 기능 | [Tavily API](https://tavily.com/) |

구성 지침은 [인터넷 검색](../../user-guide/configuration/internet-search)을 참조하세요.

## 캘린더, 승인 및 알림 워크플로 {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0은 자동화 워크플로를 콘텐츠 및 사이트 구축 작업을 넘어 확장합니다.

- **Google Calendar 도구**를 사용하면 캘린더 자격 증명을 사용할 수 있을 때 agent가 구성된 캘린더와 이벤트 세부 정보를 읽을 수 있습니다.
- **연락처 및 참석자 도우미**는 알림 또는 후속 작업을 준비하기 전에 이벤트 참가자를 WordPress 사용자 또는 알려진 연락처에 매핑합니다.
- **사람 승인 게이트**는 승인된 사용자가 제안된 작업을 검토할 때까지 자동화를 일시 중지할 수 있게 합니다.
- **알림 기록**은 자동화가 재시도되거나 반복적으로 예약될 때 중복 캘린더 또는 알림 리마인더를 방지합니다.
- **TextBee SMS 지원**은 TextBee 제공자가 구성된 경우 문자 메시지 알림을 사용할 수 있게 합니다.

민감한 메시지가 전송되기 전에 여전히 사람의 검토가 필요한 일정 인식 지원, 참석자 후속 조치 및 알림 워크플로에 이러한 도구를 사용하세요.

## 구성 {#configuration}

모든 설정은 WordPress 관리자에서 **Gratis AI Agent → Settings** 아래에 있습니다. Superdav AI Agent는 다음 구성 옵션을 추가합니다.

| 설정 | 설명 |
|---|---|
| **Internet Search Provider** | Brave Search와 Tavily 중에서 선택합니다(선택 사항) |
| **Tavily API Key** | Tavily 인터넷 검색용 API 키(Tavily가 선택된 경우) |
| **Managed Superdav Service** | 지원되는 사이트를 자동 endpoint 프로비저닝이 포함된 호스팅 Superdav 서비스에 연결합니다 |
| **Google Calendar Credentials** | agent가 캘린더와 이벤트를 읽을 수 있도록 캘린더 액세스를 구성합니다 |
| **TextBee SMS Provider** | 승인된 SMS 알림 워크플로를 위한 TextBee 자격 증명을 구성합니다 |

## 고급 컴패니언 패키지 {#advanced-companion-package}

Superdav AI Agent v1.18.0은 신뢰할 수 있는 개발자 워크플로를 위해 별도로 배포되는 고급 컴패니언 패키지를 추가합니다. 이러한 기능은 WordPress.org core 패키지에 번들로 포함되어 있지 않습니다. 관리자가 agent가 개발자 도구에 액세스하는 것을 명시적으로 신뢰하는 사이트에만 컴패니언 패키지를 설치하세요.

고급 도구에는 파일 시스템 작업, 데이터베이스 액세스, WP-CLI, REST 디스패처 워크플로, plugin builder 유틸리티, git 스냅샷, 사용자 관리 도우미 및 벤치마크 도구가 포함됩니다. 이러한 도구는 신뢰할 수 있는 관리자에게만 제한하고 프로덕션 사이트에서 활성화하기 전에 승인 게이트 구성을 검토하세요.

## v1.18.0의 개선 사항 {#improvements-in-v1180}

- **Google Calendar 지원** — 일정 인식 자동화를 위해 구성된 캘린더와 이벤트 읽기
- **연락처 및 참석자 매핑** — 캘린더 참가자를 사이트 사용자 및 연락처와 일치시키기
- **사람 승인 게이트** — 계속하기 전에 검토를 위해 민감한 자동화를 일시 중지
- **알림 중복 제거** — 재시도 또는 반복 실행 시 중복 알림을 방지하기 위해 리마인더 기록
- **TextBee SMS 제공자** — 필요한 승인 후 구성된 문자 메시지 알림 보내기
- **Managed Superdav 서비스 설정** — 지원되는 사이트를 위해 호스팅 서비스 endpoint를 자동으로 프로비저닝
- **고급 컴패니언 배포** — 신뢰할 수 있는 개발자 도구를 WordPress.org core 빌드와 별도로 게시
- **안정성 수정** — 모델 선택, 요청 시간 초과, 기본 설정 모델 처리, 추론 텍스트 처리, 재시도 안내, 캘린더 token 강화, 알림 강화, 온보딩 시작, 확인된 기능 재개 및 WordPress.org 패키징 검토 문제 개선

## v1.16.0의 개선 사항 {#improvements-in-v1160}

- **AI 클라이언트 캐시** — 이제 요청 간 지속성을 위해 transients로 지원되어, 장시간 실행되는 agent 작업에서 데이터 손실을 방지합니다
- **Tier 1 사이트 스크레이프 도구** — 경쟁사 및 영감용 웹사이트를 분석하기 위해 이제 Theme Builder에서 기본적으로 sd-ai-agent/site-scrape를 사용할 수 있습니다
- **플러그인 행 작업 링크** — 명확성을 위해 수정 및 이름이 변경되었습니다

## v1.10.0의 개선 사항 {#improvements-in-v1100}

- **WooCommerce 통합 리팩터링** — 이제 더 나은 안정성과 호환성을 위해 기본 WooCommerce APIs를 사용합니다
- **Provider 목록 자동 새로고침** — 플러그인이 활성화되거나 비활성화될 때 자동으로 감지하고 사용 가능한 Provider 목록을 업데이트합니다
- **향상된 사이트 빌더** — 더 나은 오류 복구를 갖춘 다단계 사이트 구축 계획으로 개선되었습니다
- **테마 인식 기술** — 활성 테마에 자동으로 맞춰지는 네 가지 새로운 내장 기술

## v1.10.0의 버그 수정 {#bug-fixes-in-v1100}

- 일부 관리자 페이지에서 무한 새로고침 루프를 일으키던 `navigate-to` 기능을 수정했습니다
- `list-posts` 기능이 카테고리 및 태그 이름을 슬러그로 올바르게 확인하지 못하던 문제를 수정했습니다
- 이전 리팩터링 이후 WP-CLI 명령에 네임스페이스 별칭이 누락되던 문제를 수정했습니다
- 자동화 테이블이 아직 생성되지 않은 사이트를 이벤트 자동화가 원활하게 처리하도록 수정했습니다
- 시스템 지침 빌더에서 `memory-save` 기능이 올바른 네임스페이스 접두사를 사용하도록 수정했습니다
- 스칼라 도구 결과가 AI에 반환되기 전에 올바르게 래핑되도록 수정했습니다
- 이전 버전에서 업그레이드할 때 사용 통계가 기존 기능 키 형식을 올바르게 처리하도록 수정했습니다
