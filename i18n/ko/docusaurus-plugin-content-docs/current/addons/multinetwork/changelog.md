# Multi-Network 변경 로그

### 1.3.0
* 신규: 네트워크 템플릿 미리보기 도구 — 구매하기 전에 라이브 패널로 네트워크 템플릿을 찾아보고 미리 볼 수 있습니다.
* 신규: 결제 시 네트워크 템플릿 선택 도구 — 고객이 가입 결제 흐름 중에 네트워크 템플릿을 선택할 수 있습니다.
* 신규: 네트워크 간 magic link SSO — 고객이 magic-link 인증을 통해 하위 네트워크에 원활하게 접근할 수 있습니다.
* 신규: 네트워크 미디어 복제 — 템플릿에서 네트워크를 복제할 때 사이트 미디어가 올바르게 복제됩니다.
* 수정: 이제 multi-tenancy addon 없이도 네트워크 복제가 작동합니다. 복제 후 super admin 권한이 다시 부여됩니다.
* 수정: 이제 네트워크 생성이 WooCommerce 및 모든 결제 게이트웨이와 호환됩니다.
* 수정: 이제 대체 도메인 생성에 포트 번호가 포함됩니다.
* 수정: 모든 theme에서 올바르게 렌더링되도록 템플릿 선택 도구 CSS를 대기열에 추가된 스타일시트로 이동했습니다.
* 수정: 사이트 쿼리 범위에서 정의되지 않은 `network_id` 키에 대한 보호 처리를 추가했습니다.
* 수정: 컨텍스트 전환 전 네트워크 권한 확인으로 `wpdb` 테이블 속성 오류를 방지합니다.
