# 멀티 네트워크

Ultimate Multisite: Multinetwork 애드온은 강력한 멀티네트워크 기능으로 Ultimate Multisite 플랫폼을 확장합니다. 이 애드온을 사용하면 고객이 개별 사이트뿐만 아니라 전체 WordPress 멀티사이트 네트워크를 구매하고 관리할 수 있습니다.

## 주요 기능

* **네트워크 프로비저닝**: 고객을 위해 완전한 WordPress 멀티사이트 네트워크를 자동으로 생성하고 구성합니다
* **네트워크 템플릿**: 고객이 가입 전에 미리 만들어진 네트워크 템플릿을 둘러보고, 미리 보고, 선택할 수 있게 합니다
* **네트워크 관리**: 멀티네트워크 환경을 위한 전체 관리 기능
* **격리된 네트워크**: 각 고객 네트워크는 자체 데이터베이스 테이블과 구성으로 독립적으로 작동합니다
* **교차 네트워크 SSO**: 매직 링크 인증으로 고객이 하위 네트워크에 원활하게 액세스할 수 있게 합니다
* **확장 가능한 아키텍처**: 최적화된 리소스 할당으로 여러 네트워크를 효율적으로 처리합니다
* **통합**: Ultimate Multisite의 구독 및 청구 시스템과 원활하게 통합됩니다

## 사용 사례

* **화이트 라벨 호스팅**: 고객에게 자체 브랜드가 적용된 멀티사이트 네트워크를 제공합니다
* **에이전시 솔루션**: 에이전시가 단일 플랫폼에서 여러 클라이언트 네트워크를 관리할 수 있게 합니다
* **엔터프라이즈 배포**: 복잡한 멀티 네트워크 요구사항이 있는 대규모 조직을 지원합니다
* **교육 플랫폼**: 여러 학교 또는 부서를 위한 별도 네트워크를 생성합니다

## 요구사항

* Ultimate Multisite 2.0.0 이상
* WordPress Multisite 설치
* PHP 7.4 이상
* MySQL 5.6 이상

## 설치

1. 플러그인 파일을 `/wp-content/plugins/ultimate-multisite-multinetwork` 디렉터리에 업로드합니다
2. WordPress의 '플러그인' 화면을 통해 플러그인을 활성화합니다
3. Ultimate Multisite → 설정에서 멀티네트워크 설정을 구성합니다

## 네트워크 템플릿

버전 1.3.0은 새 네트워크를 구매하는 고객을 위한 템플릿 검색 및 선택 흐름을 추가합니다. 각 새 고객 네트워크가 준비된 구조, 디자인, 콘텐츠 라이브러리 또는 업종별 설정에서 시작되도록 하려면 네트워크 템플릿을 사용하세요.

### 템플릿 미리보기 도구

네트워크 템플릿 미리보기 도구를 사용하면 고객이 결제 전에 사용 가능한 템플릿을 둘러볼 수 있습니다. 각 템플릿은 라이브 미리보기 패널에서 열 수 있으므로, 고객은 네트워크를 구매하기 전에 레이아웃과 콘텐츠를 비교할 수 있습니다.

미리보기 도구를 사용하여 고객이 다음을 할 수 있도록 도와주세요:

* 사용 가능한 각 네트워크 템플릿의 모양과 구조 검토
* 구매 흐름을 벗어나지 않고 여러 템플릿 비교
* 가입 전에 자신의 사용 사례에 가장 잘 맞는 템플릿 선택

템플릿 이름과 설명은 고객이 이해하기 쉽게 유지하세요. 미리보기 도구는 고객이 에이전시 네트워크, 학교 네트워크, 프랜차이즈 네트워크 또는 화이트 라벨 호스팅 번들처럼 의도된 사용 사례를 이해할 수 있을 때 가장 유용합니다.

### 결제 시 템플릿 선택기

템플릿 선택기는 가입 결제 흐름에 필드로 추가할 수 있습니다. 구성되면 고객은 결제 중 원하는 네트워크 템플릿을 선택하고, 선택한 템플릿은 새 네트워크가 프로비저닝될 때 사용됩니다.

결제 흐름에서 선택기를 사용하려면:

1. 템플릿으로 제공하려는 네트워크를 준비합니다.
2. 관련 가입 또는 결제 양식에서 Multinetwork 템플릿 선택 필드를 활성화합니다.
3. 해당 결제 흐름에서 사용할 수 있어야 하는 템플릿을 선택합니다.
4. 결제 흐름을 게시하고 가입을 테스트하여 선택한 템플릿이 새 네트워크에 적용되는지 확인합니다.

결제 흐름에서 하나의 네트워크 제품만 제공하더라도, 선택기는 안내형 온보딩 단계로 여전히 유용할 수 있습니다. 여러 플랜 퍼널의 경우 선택한 제품 또는 플랜과 호환되는 템플릿만 표시하세요.

## 교차 네트워크 매직 링크 SSO

교차 네트워크 매직 링크 SSO를 사용하면 고객이 다시 수동으로 로그인하지 않고도 기본 Account 영역과 하위 네트워크 사이를 이동할 수 있습니다. 고객이 교차 네트워크 액세스 링크를 따라가면 Ultimate Multisite가 시간 제한이 있는 매직 링크를 생성하고 대상 네트워크에 고객을 인증합니다.

이는 둘 이상의 네트워크를 관리하거나 구매 직후 새로 복제된 템플릿 네트워크에 즉시 액세스해야 하는 고객의 경험을 개선합니다.

관리자 체크리스트:

* Multinetwork 애드온이 WordPress 멀티사이트 설치 전체에서 활성화되어 있는지 확인합니다.
* 고객에게 교차 네트워크 링크가 제공되기 전에 올바른 네트워크 액세스 권한이 있는지 확인합니다.
* 매직 링크가 의도한 대상 네트워크로 확인되도록 네트워크 도메인과 매핑된 도메인을 올바르게 구성해 둡니다.
* 네트워크, 도메인 또는 결제 설정을 변경한 후 고객 dashboard와 결제 후 액세스 버튼을 테스트합니다.

매직 링크는 로그인 링크로 취급해야 합니다. 원활한 고객 액세스를 위해 설계되었으며, 의도된 세션 밖으로 전달하거나 재사용해서는 안 됩니다.

## 자주 묻는 질문

### 멀티네트워크란 무엇인가요?

멀티네트워크는 단일 WordPress 설치에서 여러 독립적인 멀티사이트 네트워크를 실행할 수 있게 해주는 WordPress 기능입니다.

### 일반 멀티사이트와 어떻게 다른가요?

일반 멀티사이트는 하나의 네트워크 안에 여러 사이트를 허용합니다. 멀티네트워크는 하나의 WordPress 설치 안에서 각각 자체 사이트를 가진 여러 네트워크를 허용합니다.

### 특별한 서버 구성이 필요한가요?

예, 멀티네트워크에는 특정한 서버 설정과 구성이 필요합니다. 자세한 설정 지침은 문서를 참조하세요.
