# 멀티네트워크

The Ultimate Multisite: Multinetwork addon extends your Multisite Ultimate platform with powerful multinetwork functionality. This addon enables your customers to purchase and manage entire WordPress multisite networks, not just individual sites.

## 주요 기능

* **네트워크 프로비저닝**: 고객을 위해 완전한 WordPress 멀티사이트 네트워크를 자동으로 생성하고 구성합니다
* **네트워크 관리**: 멀티네트워크 환경에 대한 완전한 관리 기능
* **격리된 네트워크**: 각 고객 네트워크는 자체 데이터베이스 테이블과 구성을 가지고 독립적으로 운영됩니다
* **확장 가능한 아키텍처**: 최적화된 리소스 할당으로 여러 네트워크를 효율적으로 처리합니다
* **통합**: Multisite Ultimate의 구독 및 청구 시스템과 원활하게 통합됩니다

## 사용 사례

* **화이트 라벨 호스팅**: 고객에게 자체 브랜드의 멀티사이트 네트워크를 제공합니다
* **에이전시 솔루션**: 에이전시가 단일 플랫폼에서 여러 클라이언트 네트워크를 관리할 수 있도록 합니다
* **엔터프라이즈 배포**: 복잡한 멀티네트워크 요구 사항을 가진 대규모 조직을 지원합니다
* **교육 플랫폼**: 서로 다른 학교나 부서에 대한 별도 네트워크를 만듭니다

## 요구 사항

* Multisite Ultimate 2.0.0 이상
* WordPress Multisite 설치
* PHP 7.4 이상
* MySQL 5.6 이상

## 설치

1. 플러그인 파일을 `/wp-content/plugins/ultimate-multisite-multinetwork` 디렉터리에 업로드합니다
2. WordPress의 '플러그인' 화면에서 플러그인을 활성화합니다
3. Multisite Ultimate → 설정에서 멀티네트워크 설정을 구성합니다

## 자주 묻는 질문

## 멀티네트워크란?

멀티네트워크는 단일 WordPress 설치에서 여러 개의 독립적인 멀티사이트 네트워크를 실행할 수 있게 해 주는 WordPress 기능입니다

## 일반 멀티사이트와의 차이점은 무엇인가요?

일반 멀티사이트는 하나의 네트워크 내에서 여러 사이트를 허용합니다. 멀티네트워크는 각 네트워크마다 자체 사이트를 갖는 여러 네트워크를 하나의 WordPress 설치 내에서 실행할 수 있게 해 줍니다

## 특별한 서버 구성이 필요한가요?

네, 멀티네트워크는 특정 서버 설정 및 구성이 필요합니다. 자세한 설정 지침은 문서를 참조하십시오
