# 액션: move_site

새 네트워크로 사이트가 이동된 후에 발생합니다.

## Parameters

| 이름 | 유형 | 설명 |
|------|------|-------------|
| $site_id | `int` | 이동된 사이트의 ID. |
| $old_network_id | `int` | 사이트의 원래 네트워크 ID. |
| $new_network_id | `int` | 사이트가 이동된 네트워크 ID. |
