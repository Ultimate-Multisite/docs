# 캡차

**봇을 차단하세요. 수익을 보호하세요. 설정 번거로움 없이.**

맞춤 도메인으로 멀티사이트 네트워크를 운영하고 계신가요? 고통을 아시죠: Google reCAPTCHA와 hCaptcha는 관리 콘솔에 *모든 도메인*을 등록해야 합니다. 새로운 고객 사이트를 추가하시나요? 캡차 구성을 업데이트합니다. 맞춤 도메인을 매핑하시나요? 다시 업데이트합니다. 끝이 없습니다.

**Ultimate Multisite: 캡차가 모든 것을 바꿉니다.**

## Cap Captcha: 바로 작동하는 자체 호스팅 보호

우리의 새로운 **Cap Captcha** 제공업체는 멀티사이트 네트워크에 게임 체인저입니다:

* **제로 설정** — 애드온을 활성화하면 보호됩니다. API 키, Google 계정, hCaptcha 대시보드, 도메인 화이트리스트가 필요 없습니다.
* **모든 도메인에서 자동으로 작동** — 서브도메인, 매핑된 도메인, 고객 도메인—모두 즉시 보호됩니다.
* **자체 호스팅 & 프라이빗** — 데이터가 서버를 떠나지 않습니다. 제3자 추적 없이 완전 GDPR 준수.
* **보이지 않는 작업 증명** — 사용자가 백그라운드에서 계산 퍼즐을 해결합니다. 물방울 클릭, 이미지 격자 같은 번거로운 작업이 없습니다.
* **경량** — 단 20KB. 결제 과정을 느리게 하지 않습니다.

## WooCommerce 카드 테스트 보호

카드 테스트 공격은 매년 수십억 달러를 비용으로 발생시킵니다. 사기꾼은 봇을 사용해 도난당한 신용카드를 귀하의 WooCommerce 결제에 검증합니다—귀하는 처리 수수료를 지불하고, 그들은 유효한 카드 번호를 얻습니다.

**우리는 가장 중요한 곳을 보호합니다:**

* **스토어 API 보호** — 전통적인 폼 캡차를 우회하는 `/wp-json/wc/store/v1/checkout`에서 봇 공격을 차단합니다.
* **PayPal 결제 호환** — WooCommerce PayPal Payments와 원활하게 작동합니다.
* **실시간 통계** — 차단 중인 공격 수를 정확히 확인합니다.

## 여전히 Google reCAPTCHA 또는 hCaptcha가 필요하신가요?

우리는 완전한 지원을 제공합니다. 다음을 지원합니다:

* Google reCAPTCHA v2 (Checkbox)
* Google reCAPTCHA v2 (Invisible)
* Google reCAPTCHA v3 (Score-based)
* hCaptcha (Standard)
* hCaptcha (Invisible)

네트워크 설정에서 언제든지 제공업체를 전환할 수 있습니다.

## 확장성을 위해 구축되었습니다

* 사이트별 재정의가 가능한 네트워크 전역 설정
* 보호를 모니터링하는 통계 대시보드
* 조정 가능한 보안 수준 (빠름, 중간, 최대)
* 확장 가능한 제공업체 아키텍처를 갖춘 개발자 친화적

## 설치

1. `ultimate-multisite-captcha`를 `/wp-content/plugins/` 디렉터리에 업로드합니다
2. 네트워크에서 플러그인을 활성화합니다
3. **그게 전부입니다. 보호됩니다.**

Cap Captcha는 자동으로 활성화됩니다—설정할 필요가 없고, API 키를 입력할 필요도 없습니다. 네트워크 전체에서 지원되는 모든 엔드포인트가 활성화 순간에 보호됩니다.

**옵션:** Google reCAPTCHA 또는 hCaptcha를 대신 사용하려면 **Ultimate Multisite → Settings → Captcha**로 이동하여 원하는 제공업체를 선택하세요.

## 자주 묻는 질문

### 설정이 필요하나요?

아니요! Cap Captcha는 애드온을 활성화하면 자동으로 활성화됩니다. 네트워크 전체가 즉시 보호됩니다—설정할 필요가 없고, API 키를 입력할 필요도 없습니다.

### Cap Captcha에 API 키가 필요하나요?

아니요. Cap Captcha는 완전히 자체 호스팅됩니다. 외부 계정, 도메인 등록, 설정 번거로움이 없습니다.

### 이로 인해 결제가 느려지나요?

Cap Captcha는 20KB에 불과하며 백그라운드에서 작업 증명 챌린지를 실행합니다. 고객은 지연을 느끼지 못합니다.

### 카드 테스트 공격을 방어하나요?

네! 전통적인 캡차가 HTML 폼만 보호하는 것과 달리, 우리의 WooCommerce 통합은 스토어 API에 직접 연결되어 API 수준에서 봇 공격을 차단합니다—대부분의 카드 테스트 공격이 발생하는 곳입니다.

### 다른 사이트에서 다른 캡차 제공업체를 사용할 수 있나요?

캡차 제공업체는 네트워크 전체에 설정되지만, 필요에 따라 난이도 설정을 조정할 수 있습니다.

### 이 서비스는 GDPR을 준수하나요?

Cap Captcha는 100% 자체 호스팅되며 외부 서비스에 데이터가 전송되지 않습니다. Google reCAPTCHA와 hCaptcha의 경우, 각 서비스의 개인정보 보호 정책을 확인해 주세요.
