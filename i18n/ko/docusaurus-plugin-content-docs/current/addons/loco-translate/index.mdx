# Loco Translate

The Ultimate Multisite: Loco Translate Integration addon은 귀하의 WordPress 멀티사이트 네트워크에 사이트별 번역 기능을 제공합니다. 네트워크의 서로 다른 사이트가 동일한 플러그인이나 테마의 서로 다른 번역을 필요로 할 때, 이 애드온이 완벽한 솔루션입니다.

**이 애드온이 수행하는 작업**

이 애드온은 Ultimate Multisite와 인기 있는 Loco Translate 플러그인을 원활하게 통합하여 네트워크의 각 서브사이트가 자체 맞춤 번역을 유지할 수 있도록 합니다. 다중 지역 플랫폼을 운영하거나, 다양한 언어 커뮤니티를 지원하거나, 단순히 네트워크 전반에 걸쳐 번역 유연성이 필요할 때, 이 애드온은 그 과정을 손쉽게 만들어 줍니다.

**적합한 대상**

* 다양한 현지화 요구를 가진 다중 지역 비즈니스
* 다양한 커뮤니티를 지원하는 교육 기관
* 다국어 지원을 제공하는 SaaS 플랫폼
* 여러 클라이언트 사이트를 관리하는 웹 에이전시
* 번역 유연성을 요구하는 모든 멀티사이트 네트워크

## 설치

1. Ultimate Multisite가 설치되고 네트워크에서 활성화되어 있는지 확인하세요
2. Loco Translate 플러그인을 설치하고 활성화하세요
3. 이 애드온을 플러그인 디렉터리에 업로드하세요
4. Loco Translate Integration 애드온을 네트워크에서 활성화하세요
5. Loco Translate를 사용해 서브사이트에서 사이트별 번역을 생성하기 시작하세요

그게 전부입니다! 추가 설정 없이 통합이 자동으로 작동합니다.

## 자주 묻는 질문

## 설치 후 설정이 필요할까요?

아니요! 통합은 자동입니다. 애드온과 Loco Translate가 모두 활성화되면 서브사이트는 즉시 자체 맞춤 번역을 생성할 수 있습니다.

## 서브사이트가 맞춤 번역을 만들지 않으면 어떻게 되나요?

사이트는 자동으로 네트워크 전체 번역을 대체로 사용합니다. 아무것도 깨지지 않으며 모든 것이 원활하게 작동합니다.

## 서브사이트가 네트워크 전체 번역에 접근할 수 있나요?

일반 사이트 관리자는 자신의 사이트 번역만 보고 편집할 수 있습니다. 슈퍼 관리자는 보고 있는 사이트에 따라 네트워크 전체 번역과 사이트별 번역 모두에 접근할 수 있습니다.

## 이 기능이 플러그인과 테마 모두에 적용되나요?

네! 사이트별 번역은 WordPress 플러그인과 테마 모두에 적용됩니다.

## 이 기능이 멀티사이트 네트워크를 느리게 하나요?

아니요. 애드온은 필요할 때만 사이트별 번역을 확인하는 효율적인 파일 로딩을 사용합니다. 눈에 띄는 성능 저하가 없습니다.

## 네트워크 번역을 특정 사이트에 복사해 시작점으로 삼을 수 있나요?

네! Loco Translate에서 번역을 편집할 때 네트워크 번역을 시작점으로 사용하고 각 사이트에 맞게 커스터마이즈할 수 있습니다.

## WooCommerce 및 기타 플러그인에서도 적용되나요?

확실히 그렇습니다! 표준 WordPress 번역 함수를 사용하는 모든 플러그인이나 테마는 사이트별 번역과 함께 작동합니다.
