# 지원 에이전트

전체 네트워크를 혼자 관리하는 것은 쉽지 않은 일입니다. 네트워크가 성장하기 시작하면 지원에 대한 수요가 증가하는 것을 보게 될 것입니다. 업무 부담을 덜어줄 사람을 영입하는 것이 좋은 아이디어가 될 것입니다. 하지만 완전한 접근 권한/권한을 부여하는 것은 위험하며 네트워크를 위험에 빠뜨릴 수 있습니다. 바로 이 Support Agent 애드온이 필요한 이유입니다.

## Features

- WordPress 및 Ultimate Multisite 권한 항목별 목록
- 간편하게 접근 권한을 활성화/비활성화
- 제한 없이 에이전트 계정을 생성

## Installation

1. 'ultimate-multisite-support-agents'를 '/wp-content/plugins/' 디렉터리에 업로드합니다.  
2. WordPress의 '플러그인' 메뉴를 통해 플러그인을 활성화합니다.  
3. 단계별 마법사를 따라 플러그인을 설정합니다.
