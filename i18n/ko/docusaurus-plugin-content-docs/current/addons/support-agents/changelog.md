# 지원 에이전트 변경 로그

버전: 1.0.13 - 2025-09-28에 릴리스

* 접두사를 ultimate-multisite로 변경; 텍스트 도메인 업데이트; 버전 증가.

버전 1.0.12 - 2025-09-17에 릴리스

* 수정: Ultimate Multisite 2.4.4 호환성

버전 1.0.11 - 2023-11-21에 릴리스

* 수정: Ultimate Multisite 2.3.0 호환성

버전 1.0.10 - 2023-08-30에 릴리스

* 수정: 대량 지원 에이전트 제거

버전 1.0.9 - 2023-08-09에 릴리스

* 추가: 체크아웃 양식을 읽을 수 있는 기능 추가.
* 수정: 지원 에이전트 권한 양식 레이아웃;
* 수정: 지원 에이전트 삭제 양식의 경고 메시지;
* 수정: 권한이 없는 사용자에게 네트워크 대시보드 위젯 숨김.
* 수정: 지원 에이전트 편집 및 삭제 기능 수정.
* 수정: 지원 에이전트 생성 프로세스 수정.
* 수정: 고객을 지원 에이전트로 추가 방지.
* 수정: 권한이 없는 사용자를 지원 에이전트 편집 관리자 페이지로 리디렉션.
* 수정: 지원 에이전트 권한 양식 레이아웃 개선.
* 내부: 새로운 빌드 프로세스 구현.

버전 1.0.8 - 2022-01-21에 릴리스

* 내부: 훅 및 필터 생성기 추가;
* 내부: 개발자 편의를 위한 Ultimate Multisite 스텁 추가;
* 추가: 체크아웃 양식 기능;
* 수정: Ultimate Multisite 양식과 이 애드온 옵션 간의 기능 이름 불일치;

버전 1.0.7 - 2021-12-02에 릴리스

* 수정: 지원 에이전트 삭제가 작동하지 않음;
* 수정: 대량 지원 에이전트 삭제가 작동하지 않음;

버전 1.0.6 - 2021-11-22에 릴리스

* 수정: 코어 성능 최적화 후 새 모달 추가가 작동하지 않음;
* 수정: 코어에 그리드 아이템 뷰가 존재하고 애드온에는 없음;

버전 1.0.5 - 2021-10-31에 릴리스

* 수정: PHP 8과의 작은 호환성 문제;
* 개선: wp_ultimo_skip_network_active_check가 이제 wu_is_must_use()를 사용합니다;

버전 1.0.4 - 2021-09-24

* 추가: mu-plugins 기반 설정용 wp_ultimo_skip_network_active_check 필터;

버전 1.0.3 - 2021-08-29에 릴리스

* 추가: REST API 지원 - 지원 에이전트를 REST를 통해 생성할 수 있음;

버전 1.0.2 - 2021-07-26에 릴리스

* 수정: 에이전트 권한 업데이트가 작동하지 않음;

버전 1.0.1 - 2021-05-28에 릴리스

* 추가: 사용자가 이미 슈퍼 관리자일 때 알림;
* 추가: 사용자 전환 허용 권한;
* 개선: 더 나은 양식 필드;

버전 1.0.0 - 초기 릴리스
