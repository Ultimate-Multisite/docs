# 콘텐츠 동기화

Multisite Ultimate: 콘텐츠 동기화

이 애드온은 Multisite Ultimate용으로, 네트워크 관리자가 네트워크 내의 다른 사이트에 콘텐츠를 동기화할 수 있도록 해줍니다. 게시물, 페이지, 사용자 정의 게시물 유형 및 Elementor 템플릿을 포함한 모든 게시물 유형을 지원합니다. 이 애드온은 콘텐츠와 대상 사이트를 선택하기 위한 다단계 마법사 인터페이스를 포함하며, 일괄 처리 및 실시간 진행 상황 피드백을 제공합니다.

## Features

* 콘텐츠 선택: 썸네일과 게시물 유형 필터링이 있는 시각적 카드 그리드
* 모든 게시물 유형: 게시물, 페이지, 제품, Elementor 템플릿 및 모든 사용자 정의 게시물 유형을 동기화
* 사이트 선택: 콘텐츠를 동기화할 사이트를 선택하고, 소스 사이트 제외 및 하위 사이트 사전 선택 기능 포함
* 일괄 처리: 한 번에 최대 10개 사이트에 동기화하며 진행 상황을 실시간으로 업데이트
* 미디어 동기화: 게시물 콘텐츠에서 이미지, 비디오 등 미디어 파일을 자동으로 복사하고 대상 사이트에 재매핑
* 분류 체계 동기화: 카테고리, 태그 및 사용자 정의 분류 체계를 보존
* Elementor 지원: CSS 재생성 및 템플릿 ID 재매핑이 포함된 Elementor 템플릿에 대한 완전한 지원
* 충돌 해결: 기존 콘텐츠를 교체하거나 건너뛸 수 있는 옵션
* 캐시 관리: 동기화 후 대상 사이트에서 Elementor 캐시를 자동으로 비우기

## Requirements

* WordPress 5.3 이상
* PHP 7.4 이상
* Multisite Ultimate 플러그인 (활성화됨)
* Elementor 플러그인 (선택 사항, Elementor 템플릿 동기화에 필요)

## Installation

1. ‘ultimate-multisite-content-sync’를 ‘/wp-content/plugins/’ 디렉터리에 업로드하세요
2. WordPress 네트워크 관리자의 ‘Plugins’ 메뉴에서 플러그인을 활성화하세요
3. Ultimate Multisite → Settings → Content Sync에서 설정을 구성하세요
4. Ultimate Multisite → Content Sync으로 이동하여 동기화를 시작하세요
