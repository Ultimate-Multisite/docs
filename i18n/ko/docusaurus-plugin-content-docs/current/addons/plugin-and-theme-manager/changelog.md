# Plugin & Theme Manager Changelog

버전: 2.0.4 - 2026-XX-XX에 출시되었습니다

* WP_Ultimo_Plugin_And_Theme_Manager 클래스에서 동적 속성 생성에 대한 PHP 8.2+ 비추천 경고를 수정했습니다.

버전: 2.0.3 - 2025-09-28에 출시되었습니다

* Ultimate Multisite v2.4와의 호환성을 수정했습니다.

버전: 2.0.1 - 2025-09-28에 출시되었습니다

* 접두사를 ultimate-multisite로 변경하고, 텍스트 도메인을 업데이트하며, 버전을 올렸습니다.

버전 2.0.0 - 2021-11-25에 출시되었습니다

* 재작성: 2.0 새로운 코드를 사용하도록 완전 재작성했습니다;

버전 1.2.5 - 11/03/2020

* 수정: 자식 테마인 경우 부모 테마 이름을 필터링했습니다;
* 수정: 카테고리 필터가 정확히 일치하는 플러그인만 반환하도록 했습니다;
* 수정: 테마 페이지에서 필터가 작동하지 않는 문제를 해결했습니다;
* 수정: 필요 시 삭제된 테마/플러그인 정보를 정리했습니다;

버전 1.2.4 - 29/03/2019

* 수정: 플러그인 세부 정보가 모달 창을 열지 않는 문제를 해결했습니다;

버전 1.2.3 - 15/03/2019

* 수정: Material WP와 함께 사용할 때 반응성 문제를 해결했습니다;

버전 1.2.2 - 27/02/2019

* 수정: 플러그인 목록이 20개 항목에서 끊기는 문제를 해결했습니다;
* 개선: 해당 플러그인 또는 테마가 삭제될 때 플러그인 및 테마 사용자 정의 데이터가 제거됩니다;

버전 1.2.1 - 29/01/2019

* 수정: 이전 릴리스의 작은 문제를 해결했습니다;
* 개선: 대형 플러그인 및 테마 테이블의 성능을 향상시켰습니다;

버전 1.2.0 - 21/01/2019

* 수정: 수많은 작은 버그를 해결했습니다!

버전 1.1.2 - 07/12/2018

* 수정: 해당 옵션이 비활성화되어 있음에도 불구하고 작성자가 표시되는 문제를 해결했습니다;
* 개선: 인라인 양식을 admin_notices 후크에서 in_admin_header로 이동했습니다;
* 개선: 하위 사이트의 내보내기 화면에서 Multisite Ultimate 커스텀 포스트 타입을 제거했습니다;

버전 1.1.1 - 07/12/2018

* 개선: 업데이트용 새 URL을 추가했습니다;

버전 1.1.0 - 15/08/2018

* 수정: 카테고리 저장 시 작은 문제를 해결했습니다;
* 수정: JS에서 발생한 오류를 해결했습니다;
* 추가: 플러그인 페이지용 새로운 표시 모델: 플러그인 카드!;
