# Plugin & Theme Manager

Multisite Ultimate: Plugin and Theme Manager

Edit or hide the meta information (제목, 작성자, 썸네일 및 설명) of installed plugins and themes. Create categories for plugins and themes to allow your users to filter them and add a beautiful custom Plugins page to your users' panel.

To use the plugin, you can just navigate to your Plugins or Themes pages on your network admin and click on "Edit Plugin/Theme info".

## Installation

1. 'ultimate-multisite-plugin-and-theme-manager'를 '/wp-content/plugins/' 디렉터리에 업로드합니다.
2. WordPress 네트워크 관리자의 'Plugins' 메뉴를 통해 플러그인을 활성화합니다.

## Usage Guide

Ultimate Multisite: Plugin and Theme Manager는 네트워크 사용자가 플러그인 및 테마 옵션을 보는 방식과 상호작용을 완전히 제어할 수 있도록 해 주는 강력한 **premium** Ultimate Multisite 애드온입니다.

이 애드온은 세 가지 목적을 수행합니다:

  1. 플러그인과 테마의 메타데이터를 완전히 변경할 수 있습니다. 이름, 설명, 작성자 이름, 썸네일까지 포함됩니다!

  2. 플러그인과 테마를 분류하여 사용자가 쉽게 정렬할 수 있도록 합니다.

  3. 구식이고 지루한 플러그인 페이지를 카테고리별 정렬이 가능한 새로운 테마 스타일 페이지로 교체할 수 있는 옵션을 제공합니다.

다운로드 및 설치 후, 새로운 탭 - _애드온 설정_ - 가 Ultimate Multisite Settings 페이지에 표시됩니다.

Plugin and Theme Manager는 두 가지 표시 유형을 제공합니다: 플러그인 스타일과 테마 스타일. 따라서 필요한 조정을 마친 후, 고객이 _Plugins_ 페이지를 보는 모습은 다음과 같습니다:
