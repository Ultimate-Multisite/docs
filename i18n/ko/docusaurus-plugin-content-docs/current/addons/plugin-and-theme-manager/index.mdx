# 플러그인 및 테마 관리자

Ultimate Multisite: 플러그인 및 테마 관리자

설치된 플러그인과 테마의 메타 정보(제목, 작성자, 썸네일 및 설명)를 편집하거나 숨기세요. 플러그인과 테마에 대한 카테고리를 만들어 사용자가 필터링할 수 있게 하고, 사용자 패널에 아름다운 사용자 지정 플러그인 페이지를 추가하세요.

플러그인을 사용하려면 네트워크 관리자의 플러그인 또는 테마 페이지로 이동한 다음 "플러그인/테마 정보 편집"을 클릭하면 됩니다.

## 설치

1. 'ultimate-multisite-plugin-and-theme-manager'를 '/wp-content/plugins/' 디렉터리에 업로드합니다
2. WordPress 네트워크 관리자에서 '플러그인' 메뉴를 통해 플러그인을 활성화합니다

## 사용 가이드

Ultimate Multisite: 플러그인 및 테마 관리자는 네트워크 사용자가 네트워크에서 플러그인 및 테마 옵션을 보고 상호작용하는 방식을 완전히 제어할 수 있게 해주는 강력한 **프리미엄** Ultimate Multisite 애드온입니다.

이 애드온은 세 가지 목적을 제공합니다:

  1. 플러그인과 테마 모두의 메타데이터를 완전히 변경할 수 있습니다. 여기에는 이름, 설명, 작성자 이름, 심지어 썸네일까지 포함됩니다!

  2. 플러그인과 테마를 카테고리화하여 사용자가 쉽게 정렬할 수 있게 합니다.

  3. 오래되고 지루한 플러그인 페이지를 카테고리별 정렬 기능이 있는 새로운 테마 같은 페이지로 대체할 수 있는 옵션을 제공합니다.

다운로드하고 설치한 후 Ultimate Multisite 설정 페이지에 새 탭 - _애드온 설정_ \- 이 표시됩니다.
