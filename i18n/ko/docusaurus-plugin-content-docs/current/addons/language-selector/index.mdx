# 언어 선택기 {#language-selector}

Multisite Ultimate: 가입 시 언어 선택기

가입 흐름에 언어 선택기 필드를 추가합니다.

## 설치 {#installation}

1. 'ultimate-multisite-language-selector'를 '/wp-content/plugins/' 디렉터리에 업로드합니다
2. WordPress 네트워크 관리자에서 '플러그인' 메뉴를 통해 plugin을 활성화합니다

## 사용 가이드 {#usage-guide}

이 무료 add-on을 사용하면 사용자가 가입 흐름과 로그인 페이지에서 보고 싶은 언어를 선택할 수 있습니다.

[라이선스를 활성화](../../user-guide/getting-started/installing-ultimate-multisite.md)한 후에는 라이선스로 접근할 수 있는 Ultimate Multisite add-on을 설치할 수 있습니다.

Ultimate Multisite 설정 페이지로 이동한 다음, 왼쪽 메뉴에서 _Add-ons_ 레이블을 클릭합니다.

![Add-ons 페이지](/img/admin/addons-page.png)

_Language Selector 설치를 시작하려면 Add-ons 페이지로 이동하세요_

거기에서 저희가 제공하는 모든 add-on을 찾을 수 있습니다.

무료 add-on은 가입한 구독 plan과 관계없이 모든 고객이 사용할 수 있습니다.

add-on의 썸네일 아래에 있는 _Install it Now!_ 버튼을 클릭합니다. 그런 다음 Plugins 페이지에서 활성화합니다.

![설치 버튼이 있는 Add-ons 페이지](/img/admin/addons-page.png)

_add-on을 설치하려면 버튼을 클릭하세요_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_Plugins 페이지에서 활성화하세요_

이제 Ultimate Multisite의 설정 메뉴로 이동하여 _Network Settings_ 탭으로 들어갑니다. 페이지 하단에서 _Languages_라는 새 섹션을 찾을 수 있습니다.

![설정 페이지](/img/config/settings-general.png)

_완전히 새로운 섹션이 나타납니다!_

모든 언어를 옵션으로 표시하려면 비워 두거나, 사용 가능하게 만들 언어를 선택할 수 있습니다. 언어는 WordPress 설정에 설치한 언어입니다.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_이 WordPress 설치에서 사용 가능한 언어 목록_

변경 사항을 저장한 후, 선택한 언어가 포함된 선택 메뉴가 로그인 페이지와 가입 흐름에 표시됩니다.

_로그인 페이지의 Language Selector_

![등록 페이지](/img/config/registration-page.png)

## URL을 통해 언어 설정하기 {#setting-the-language-via-the-url}

URL에 locale 매개변수를 전달하여 페이지의 언어를 변경할 수도 있습니다. 예를 들어 이탈리아어 등록 페이지로 리디렉션하려면 URL에 ?locale=it_IT를 추가하세요.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

이 기능이 작동하려면 해당 언어가 WordPress 네트워크에 설치되어 있어야 합니다.
