# 언어 선택기

멀티사이트 어드밴스: 가입 시 언어 선택기

가입 흐름에 언어 선택 필드를 추가합니다.

## 설치

1. ‘ultimate-multisite-language-selector’를 ‘/wp-content/plugins/’ 디렉터리에 업로드합니다.
2. WordPress 네트워크 관리자의 ‘플러그인’ 메뉴를 통해 플러그인을 활성화합니다.

## 사용 가이드

이 무료 애드온을 사용하면 사용자가 가입 흐름과 로그인 페이지에서 보고 싶은 언어를 선택하도록 할 수 있습니다.

라이센스를 [활성화한](../../user-guide/getting-started/installing-ultimate-multisite.md) 후, 라이센스가 제공하는 Ultimate Multisite 애드온을 설치할 수 있습니다.

Ultimate Multisite 설정 페이지로 이동한 뒤, 왼쪽 메뉴에서 _Add-ons_ 라벨을 클릭합니다.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-rzVqJCVpY1.png)

_언어 선택기를 설치하기 위해 Add-ons 페이지로 이동_

그곳에서 제공하는 모든 애드온을 확인할 수 있습니다.

무료 애드온은 가입한 구독 플랜에 관계없이 모든 고객에게 제공됩니다.

애드온 썸네일 아래에 있는 _Install it Now!_ 버튼을 클릭한 뒤, 플러그인 페이지에서 활성화합니다.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-bJgpO3QvLh.png)

_애드온을 설치하려면 버튼을 클릭하세요_

_플러그인 페이지에서 활성화하세요_

이제 Ultimate Multisite의 설정 메뉴로 이동해 _Network Settings_ 탭을 엽니다. 페이지 하단에서 _Languages_라는 새 섹션을 찾을 수 있습니다.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-CaH7BHDh50.png)

_새로운 섹션이 나타났습니다!_

빈 상태로 두면 모든 언어를 옵션으로 표시하거나, 사용 가능한 언어를 선택할 수 있습니다. 언어는 WordPress 설정에 설치된 언어입니다.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-PGljAd4C24.png)

_이 WP 설치에서 사용 가능한 언어 목록_

변경 사항을 저장하면 선택한 언어가 로그인 페이지와 가입 흐름에 표시되는 선택 메뉴가 나타납니다.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-qUnJL7JHxY.png)

_로그인 페이지의 언어 선택기_

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-HnKLDvxuWU.png)

_가입 흐름의 언어 선택기_

## URL을 통한 언어 설정

URL에 locale 매개변수를 우회하여 페이지 언어를 변경할 수도 있습니다. 예를 들어 이탈리아어로 등록 페이지로 리디렉션하려면 URL에 ?locale=it_IT를 추가합니다.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

이 기능이 작동하려면 언어가 WordPress 네트워크에 설치되어 있어야 합니다.
