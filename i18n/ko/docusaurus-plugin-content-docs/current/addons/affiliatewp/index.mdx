# AffiliateWP

Multisite Ultimate: AffiliateWP 통합

강력한 AffiliateWP를 사용하여 Ultimo Network의 고객 기반을 성장시키세요!

## 설치 {#installation}

1. 'ultimate-multisite-git '을 '/wp-content/plugins/' 디렉터리에 업로드합니다
2. WordPress의 '플러그인' 메뉴를 통해 플러그인을 활성화합니다
3. 단계별 마법사를 따라 플러그인을 설정합니다

## 사용 가이드 {#usage-guide}

[AffiliateWP](https://affiliatewp.com/)는 온라인 비즈니스를 위한 제휴 프로그램을 구축할 때 가장 많이 찾는 솔루션이 되었습니다. Ultimate Multisite: AffiliateWP는 프리미엄 네트워크에서도 동일한 것을 구축할 수 있게 해 주는 다리입니다.

## 시작하기 {#getting-started}

먼저, [**AffiliateWP**](https://affiliatewp.com/)가 **활성화되어 있어야 합니다**! **이와 통합되는 플러그인**이 없다면 저희 애드온을 사용할 수 없습니다. 고객이 자신의 사이트에서도 이를 사용하게 할 계획이 없다면, AffiliateWP는 메인 사이트에서만 활성화되어 있어야 합니다.

또한 **Ultimate Multisite**와 **Ultimate Multisite: AffiliateWP**가 모두 **네트워크 전체에서** **활성화**되어 있어야 합니다.

## AffiliateWP 통합으로 Ultimate Multisite 추가하기 {#adding-ultimate-multisite-as-an-affiliatewp-integration}

메인 사이트에서 제휴 설정으로 이동한 다음, 통합 탭에서 Ultimate Multisite를 선택하세요. 이렇게 하면 [AffiliateWP](https://affiliatewp.com/)가 Ultimate Multisite 가입 흐름에서 발생하는 방문과 전환을 추적하기 시작합니다.

<!-- 스크린샷 없음: Ultimate Multisite 체크박스가 있는 AffiliateWP 통합 설정 탭 -->

_하단의 Ultimate Multisite 상자를 선택하고 저장하세요_

필요한 작업은 사실상 이것이 전부입니다. 추천 링크에서 들어온 방문이 추적되며, 해당 사용자가 나중에 첫 결제를 하면 전환이 생성되고 그 전환을 담당한 제휴자에게 커미션이 따로 배정됩니다.

하지만 새로 생성된 사용자에게서 받은 첫 결제만 추적됩니다. 반복 추천 결제를 활성화하려면 [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)이 필요합니다

## 반복 추천 활성화하기 {#enabling-recurring-referrals}

[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)을 설치한 후, 제휴 설정의 반복 추천 탭으로 이동하여 반복 추천 사용을 활성화하세요.

<!-- 스크린샷 없음: 반복 추천 활성화 토글이 있는 AffiliateWP 반복 추천 설정 -->

_정말 간단합니다: 반복 추천을 활성화하면 모든 준비가 끝납니다!_

다시 말하지만, 필요한 작업은 이것이 전부입니다!

이제 추천된 Account에서 새 결제가 수신될 때마다, 담당 제휴자는 자동으로 자신의 몫을 받게 됩니다!
