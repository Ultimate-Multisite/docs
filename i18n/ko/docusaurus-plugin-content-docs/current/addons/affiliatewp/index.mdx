# AffiliateWP

Multisite Ultimate: AffiliateWP Integration

강력한 AffiliateWP를 사용하여 Ultimo Network의 고객 기반을 확장하세요!

## Installation

1. 'ultimate-multisite-git'을 '/wp-content/plugins/' 디렉터리에 업로드하세요.
2. WordPress의 'Plugins' 메뉴를 통해 플러그인을 활성화하세요.
3. 단계별 마법사를 따라 플러그인을 설정하세요.

## Usage Guide

[AffiliateWP](https://affiliatewp.com/)는 온라인 비즈니스를 위한 제휴 프로그램을 구축할 때 가장 많이 사용되는 솔루션이 되었습니다. Ultimate Multisite: AffiliateWP는 프리미엄 네트워크에서도 동일한 기능을 구축할 수 있게 해 주는 다리 역할을 합니다.

## Getting Started

우선, **AffiliateWP**가 **활성화**되어 있어야 합니다! 통합되는 플러그인이 없으면 애드온을 사용할 수 없습니다. 고객이 자신의 사이트에서도 사용하도록 계획하지 않는 경우, AffiliateWP는 메인 사이트에서만 활성화되어야 합니다.

또한, **Ultimate Multisite**와 **Ultimate Multisite: AffiliateWP**가 **네트워크 전체에서 활성화**되어 있어야 합니다.

## Adding Ultimate Multisite as an AffiliateWP Integration

메인 사이트에서 Affiliates Settings로 이동한 뒤, Integrations 탭에서 Ultimate Multisite를 선택하세요. 그러면 [AffiliateWP](https://affiliatewp.com/)가 Ultimate Multisite 가입 흐름에서 방문 및 전환을 추적하도록 설정됩니다.

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_아래쪽에 있는 Ultimate Multisite 박스를 체크하고 저장하세요_

그게 거의 전부입니다. 추천 링크를 통해 유입된 방문은 추적되며, 해당 사용자가 첫 결제를 할 때 전환이 생성되고 전환을 담당한 제휴사에게 커미션이 할당됩니다.

하지만 새로 생성된 사용자로부터 받은 첫 결제만 추적됩니다. 반복 추천 결제를 활성화하려면 [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)이 필요합니다.

## Enabling Recurring Referrals

[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)을 설치한 후, Affiliates Settings Recurring Referrals 탭으로 이동해 반복 추천 사용을 활성화하세요.

<!-- Screenshot unavailable: AffiliateWP Recurring Referrals settings with Enable Recurring Referrals toggle -->

_가장 간단합니다: 반복 추천을 활성화하면 완료됩니다!_

다시 말하지만, 이것이 전부입니다!

이제 추천 계정에 새 결제가 들어올 때마다 담당 제휴사는 자동으로 커미션을 받게 됩니다.
