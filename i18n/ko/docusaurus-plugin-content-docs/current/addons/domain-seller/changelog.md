# 도메인 판매자 변경 로그

버전: 2.0.1 - 2025-09-28에 출시

* 프리픽스를 ultimate-multisite로 바꾸고, 텍스트 도메인을 업데이트하며, 버전을 올립니다.

### 버전 2.0.0 - 2024-XX-XX

**Multisite Ultimate v2를 위한 주요 재작성**

* **New:** 최신 PHP 7.4+ 아키텍처를 사용한 완전한 재작성
* **New:** Multisite Ultimate v2 체크아웃 시스템과의 원활한 통합
* **New:** 유연한 가격 옵션을 제공하는 도메인 제품 관리
* **New:** 다중 도메인 제공업체 지원 아키텍처
* **New:** 자동 갱신 및 구독 통합
* **New:** 고객 도메인 관리 인터페이스
* **New:** 관리자 도메인 모니터링 및 로그
* **New:** 도메인 제품에 대한 쿠폰 지원
* **New:** 종합 설정 관리
* **New:** 개발자 친화적인 확장 가능한 코드베이스
* **Improved:** 전체 기능 지원이 포함된 업데이트된 OpenSRS 제공업체
* **Improved:** Multisite Ultimate v2와 일관된 현대적 UI
* **Fixed:** 모든 오래된 v1 코드를 v2 표준에 맞게 업데이트
* **Removed:** 레거시 v1 호환성 (중단 변경)
