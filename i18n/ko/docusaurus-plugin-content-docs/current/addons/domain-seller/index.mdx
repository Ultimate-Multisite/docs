# Domain Seller {#domain-seller}

**Multisite Ultimate: Domain Seller**는 WordPress Multisite 네트워크를 강력한 도메인 판매 플랫폼으로 전환합니다. 고객이 가입 중 또는 자신의 Dashboard에서 직접 사용자 지정 도메인을 등록할 수 있게 하며, 실시간 가용성 확인과 투명한 가격 책정을 제공합니다.

### 왜 Domain Seller인가요? {#why-domain-seller}

도메인 등록은 모든 WaaS(서비스형 웹사이트) 플랫폼에서 가장 많이 요청되는 상향 판매 항목 중 하나입니다. 고객을 서드파티 등록기관으로 보내는 대신, 도메인을 직접 제공할 수 있습니다. 이를 통해 네트워크 내에서 더 많은 수익을 유지하면서 고객에게 마찰 없는 경험을 제공할 수 있습니다.

### 주요 기능 {#key-features}

#### 여러 도메인 등록기관 지원 {#multiple-domain-registrar-support}
* **OpenSRS** — 전체 등록, 갱신 및 관리
* **Namecheap** — WhoisGuard 및 샌드박스 테스트와의 완전한 통합
* **HostAfrica** — Domains Reseller API를 통한 도메인 검색, 가격 책정, 등록, 이전, DNS 및 ID 보호
* **Openprovider** — 리셀러 가격 책정, 등록, 갱신, 이전, DNS, WHOIS 개인정보 보호 및 TLD 동기화
* **Hostinger** — 공유 Hostinger 통합 토큰을 사용한 도메인 검색 및 등록
* **확장 가능한 아키텍처** — Integration Registry 패턴을 기반으로 구축되어 더 많은 제공업체를 쉽게 추가할 수 있음
* **내장 연결 테스트** — 라이브로 전환하기 전에 제공업체 자격 증명과 연결 상태를 검증

#### 안내형 설정 마법사 {#guided-setup-wizard}
* **원클릭 온보딩** — 제공업체 구성, TLD 가져오기 및 도메인 상품 생성을 안내
* **자동 TLD 동기화** — 연결된 제공업체에서 사용 가능한 모든 TLD와 도매 가격을 가져옴
* **기본 상품 생성** — 합리적인 기본 마크업이 적용된 바로 사용할 수 있는 도메인 상품을 생성

#### 유연한 도메인 가격 책정 {#flexible-domain-pricing}
* **세 가지 마크업 모드** — 백분율 마크업, 고정 마크업 또는 TLD별 고정 가격
* **TLD별 상품** — 다양한 TLD 그룹(.com/.net, .io, 국가 코드 등)에 대해 별도의 도메인 상품 생성
* **도입 가격** — 신규 등록을 유도하기 위해 할인된 첫해 가격 제공
* **실시간 가격 책정** — 지능형 캐싱을 통한 제공업체의 실시간 도매 가격
* **쿠폰 지원** — Multisite Ultimate 할인 코드와 완전 호환

#### WHOIS 개인정보 보호 {#whois-privacy-protection}
* **상품별 구성** — WHOIS 개인정보 보호를 비활성화, 항상 포함 또는 고객 선택으로 설정
* **고객 옵트인** — 고객 선택으로 설정하면 결제 체크박스를 통해 고객이 개인정보 보호를 추가할 수 있음
* **구성 가능한 가격** — 개인정보 보호 애드온에 연간 가격을 설정하거나 무료로 제공
* **등록기관 통합** — 등록 시 WhoisGuard(Namecheap) 또는 WHOIS 개인정보 보호(OpenSRS)를 자동으로 활성화

#### 통합 결제 경험 {#unified-checkout-experience}
* **세 가지 도메인 모드** — 고객은 서브도메인(무료), 새 도메인 등록 또는 이미 소유한 기존 도메인 매핑 중에서 선택
* **실시간 도메인 검색** — 페이지 새로고침 없는 AJAX 기반 가용성 확인
* **스마트 제안** — 원하는 도메인이 사용 중일 때 대체 TLD 제안
* **투명한 가격 책정** — 등록 가격, 갱신 가격 및 선택적 WHOIS 개인정보 보호 요금을 명확하게 표시
* **등록자 연락처 필드** — 결제 중 필요한 등록자 정보(이름, 주소, 전화번호)를 인라인으로 수집
* **자동 생성 사이트 URL** — 선택한 도메인에서 사이트 주소를 자동으로 파생

#### 자동 갱신 시스템 {#auto-renewal-system}
* **멤버십 연결 갱신** — 고객의 멤버십이 활성 상태인 동안 도메인이 자동으로 갱신됨
* **스마트 예약** — 도메인 만료일을 기준으로 갱신 작업을 대기열에 추가
* **재시도 로직** — 실패한 갱신 시도에 대한 지능형 재시도
* **이메일 알림** — 예정된 갱신, 성공한 갱신, 실패 및 만료에 대한 구성 가능한 알림

#### 고객 DNS 관리 {#customer-dns-management}
* **셀프서비스 DNS** — 고객은 자신의 Dashboard에서 DNS 레코드(A, AAAA, CNAME, MX, TXT)를 추가, 편집 및 삭제할 수 있음
* **도메인 상태** — 실시간 등록 상태, 만료일 및 네임서버 정보
* **자동 도메인 매핑** — 등록된 도메인 또는 기존 도메인이 고객의 사이트에 자동으로 매핑됨

#### 관리자 도구 {#admin-tools}
* **수동 도메인 등록** — 관리자 패널에서 고객을 대신해 도메인 등록
* **도메인 모니터링** — 한 페이지에서 모든 고객 도메인, 상태 및 만료일 보기
* **제공업체별 로깅** — 각 등록기관에 대한 별도 로그 채널(예: domain-seller-namecheap.log)
* **등록 세부 정보** — 핵심 도메인 편집 페이지에서 등록자 정보 및 등록 데이터 보기

### 지원되는 제공업체 {#supported-providers}

* **OpenSRS** — 등록, 갱신, DNS 관리 및 WHOIS 개인정보 보호와의 전체 통합
* **Namecheap** — 등록, 갱신, 일괄 가용성 확인 및 무료 WhoisGuard와의 전체 통합
* **HostAfrica** — 등록, 갱신, 이전, DNS 관리, EPP 코드 검색, 등록기관 잠금 및 ID 보호
* **Openprovider** — 등록, 갱신, 이전, DNS 관리, EPP 코드 검색, 등록기관 잠금, WHOIS 개인정보 보호 및 리셀러 TLD 동기화
* **Hostinger** — 등록, 도메인 조회, 네임서버 업데이트, 등록기관 잠금 및 WHOIS 개인정보 보호; 갱신은 Hostinger 자동 갱신으로 처리됨
* **GoDaddy** — 등록 및 갱신
* **ResellerClub** — 등록, 갱신 및 DNS 관리
* **NameSilo** — 등록 및 갱신
* **Enom** — 등록, 갱신 및 DNS 관리

### 기술 세부 정보 {#technical-details}

* **필수:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **아키텍처:** Domain_Selling_Capability 인터페이스가 있는 Integration Registry 패턴
* **자동 로드:** PSR-4 클래스 매핑을 사용하는 Jetpack 자동 로더
* **표준:** WordPress 코딩 표준, 엄격한 타입 지정, 준비된 문

## 설치 {#installation}

1. **Multisite Ultimate v2.4.12+**가 설치되어 있고 네트워크 활성화되어 있는지 확인하세요
2. plugin을 `/wp-content/plugins/ultimate-multisite-domain-seller/`에 업로드하세요
3. WordPress 네트워크 관리자에서 **Plugins**를 통해 plugin을 네트워크 활성화하세요
4. **설정 마법사**는 첫 활성화 시 자동으로 실행됩니다 — 다음 단계를 따르세요:
   * 도메인 등록기관 연결
   * 연결 테스트
   * 사용 가능한 TLD와 도매 가격 가져오기
   * 가격 마크업이 있는 첫 번째 도메인 제품 만들기
5. **Ultimate Multisite > 체크아웃 양식**에서 체크아웃 양식에 **도메인 선택** 필드를 추가하세요

## 자주 묻는 질문 {#frequently-asked-questions}

### 어떤 버전의 Multisite Ultimate가 필요한가요? {#what-version-of-multisite-ultimate-is-required}

버전 2.4.12 이상입니다. 이 addon은 WP Ultimo v1.x 또는 2.4.12 미만의 Ultimate Multisite 버전과 호환되지 않습니다.

### 어떤 도메인 제공업체가 지원되나요? {#which-domain-providers-are-supported}

아홉 개의 등록기관이 지원됩니다: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo**, **Enom**. 등록기관 API가 서로 다른 작업을 노출하므로 기능은 제공업체에 따라 다릅니다. 제공업체별 개요는 설정 가이드를 참조하세요. addon의 통합 레지스트리 아키텍처를 사용하면 사용자 지정 제공업체를 쉽게 추가할 수 있습니다 — [개발자 가이드](/developer/integration-guide/custom-registrar)를 참조하세요.

### WHOIS 개인정보 보호를 제공할 수 있나요? {#can-i-offer-whois-privacy-protection}

예. 각 도메인 제품에는 비활성화, 항상 포함, 고객 선택의 세 가지 모드가 있는 WHOIS 개인정보 보호 설정이 있습니다. 고객 선택으로 설정하면 고객은 체크아웃 중 가격(또는 Namecheap의 WhoisGuard의 경우 "무료")이 표시된 체크박스를 볼 수 있습니다. 개인정보 보호 플래그는 등록 시 자동으로 등록기관에 전달됩니다.

### 서로 다른 TLD에 서로 다른 가격을 설정할 수 있나요? {#can-i-set-different-pricing-for-different-tlds}

예. 서로 다른 TLD 그룹에 대해 별도의 도메인 제품을 만드세요. 각 제품은 자체 마크업 유형(퍼센트, 고정 마크업 또는 고정 가격), 지원되는 TLD, 도입 가격을 가질 수 있습니다.

### 고객에게 어떤 체크아웃 옵션이 표시되나요? {#what-checkout-options-do-customers-see}

도메인 선택 필드는 고객에게 세 개의 탭을 제공합니다: **서브도메인**(네트워크에서 무료 서브도메인 사용), **새 도메인 등록**(새 도메인 검색 및 등록), **기존 도메인**(이미 소유한 도메인 매핑). 활성화한 모드만 표시됩니다.

### 자동 갱신을 지원하나요? {#does-it-support-automatic-renewals}

예. 도메인은 자동 갱신으로 설정할 수 있으며 고객의 멤버십 상태와 연결됩니다. 갱신 결제는 고객의 기존 결제 게이트웨이를 통해 처리됩니다.

### 고객이 DNS 레코드를 관리할 수 있나요? {#can-customers-manage-their-dns-records}

예. 고객은 자신의 Dashboard에서 A, AAAA, CNAME, MX, TXT 레코드를 추가, 편집, 삭제할 수 있는 DNS 관리 인터페이스를 이용할 수 있습니다. DNS 관리는 OpenSRS, ResellerClub, Enom, HostAfrica, Openprovider에서 사용할 수 있습니다. 호스팅된 도메인에 대한 Hostinger DNS는 핵심 Hostinger 도메인 매핑 통합에서 처리됩니다.

### 설정 마법사가 있나요? {#is-there-a-setup-wizard}

예. 설정 마법사는 첫 활성화 시 자동으로 실행되며 등록기관 연결, TLD 가져오기, 첫 번째 도메인 제품 만들기를 안내합니다. addon 설정에서 마법사를 다시 실행할 수도 있습니다.
