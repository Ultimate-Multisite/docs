# VAT

Multisite Ultimate: European VAT

Multisite Ultimate의 기본 세금 처리에 추가 기능과 도구를 제공하여 유럽 VAT를 보다 잘 지원합니다.  
자동으로 EU VAT율을 세금율 페이지에 추가합니다.

## Installation

1. ‘ultimate-multisite-vat’를 ‘/wp-content/plugins/’ 디렉터리에 업로드합니다.  
2. WordPress의 ‘Plugins’ 메뉴를 통해 플러그인을 활성화합니다.  
3. Ultimate Multisite > Settings > Taxes로 이동하여 EU Vat를 활성화합니다.  
4. Tax Rates를 클릭하고 하단에서 EU VAT Rates를 업데이트한 뒤 Tax Rates를 저장합니다.  
5. 완료!
