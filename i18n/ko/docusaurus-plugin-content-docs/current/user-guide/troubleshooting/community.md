# Ultimate Multisite 커뮤니티 가입 방법

Ultimate Multisite Community는 다른 Ultimate Multisite 사용자와 연락하고 팁, 경험 등을 공유할 수 있는 공간입니다. Ultimate Multisite Community에 접속하려면 https://community.ultimatemultisite.com/home 으로 이동하세요.

환영 페이지에서 “**여기에서 계정 만들기** ” 링크를 클릭하세요.

<!-- Screenshot unavailable: Community welcome page with creating your account link -->

등록 페이지로 리디렉션됩니다. 이제 사용자 이름, 이메일, 비밀번호를 입력해야 합니다. 캡차를 완료하고 **Register** 버튼을 클릭하세요.

<!-- Screenshot unavailable: Community register page with username, email, password fields and Captcha -->

등록이 완료되면 프로필을 만들 차례입니다. 이 페이지에서 사진을 업로드하고, 전체 이름을 입력하고, 간단한 소개글을 작성할 수 있습니다. 또한 소셜 미디어 링크를 추가하고 계정 권한을 설정하세요.

<!-- Screenshot unavailable: Community profile creation page with photo upload and bio fields -->

이제 “Introductions” 채널에 있습니다. 여기서 자신을 소개하고 다른 멤버들을 조금 알아볼 수 있습니다. **New Post** 버튼을 클릭해 게시물을 만들 수 있습니다.

<!-- Screenshot unavailable: Community Introductions channel with New Post button -->

Lounge 섹션과 “Dev Logs” 채널은 누구나 열려 있습니다. 하지만 커뮤니티의 다른 채널에 접근하려면 특별한 권한이 필요합니다.

<!-- Screenshot unavailable: Community channels showing Lounge section and Dev Logs channel -->

이 접근 권한은 활성 Ultimate Multisite 라이선스 키를 제공하면 부여됩니다. 그렇게 하려면 어떤 비공개 채널 링크를 클릭한 뒤 **Unlock** 버튼을 누르세요.

<!-- Screenshot unavailable: Community private channel with Unlock button -->

잠금 해제 접근 페이지로 리디렉션됩니다. Ultimate Multisite 라이선스 키를 입력하고 Unlock Access 버튼을 클릭하세요:

<!-- Screenshot unavailable: Community unlock access page with license key input -->

라이선스가 검증되면 비공개 채널에 접근할 수 있습니다. 이 과정은 최대 24시간이 걸릴 수 있으니, 링크를 클릭해 커뮤니티로 돌아가 다른 공간을 즐기면서 요청을 처리하도록 하세요.

<!-- Screenshot unavailable: Community license validation confirmation page -->
