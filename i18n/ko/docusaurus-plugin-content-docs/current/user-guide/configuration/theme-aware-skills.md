# 테마별 스킬 {#theme-aware-skills}

Superdav AI Agent v1.10.0에는 현재 사용 중인 WordPress 테마에 자동으로 맞춰지는 네 가지 새로운 테마별 내장 스킬이 탑재되었습니다. 이 스킬들은 테마의 구조와 기능에 맞춰진 전문적인 가이드와 기능을 제공합니다.

## 테마별 스킬이란 무엇인가요? {#what-are-theme-aware-skills}

테마별 스킬은 AI 어시스턴트가 사용자의 사이트에서 현재 활성화된 WordPress 테마를 기반으로 자동으로 선택하는 사전 구성된 지식 기반 및 도구 세트입니다. 테마를 변경하면 어시스턴트가 사용할 수 있는 스킬 목록이 자동으로 업데이트됩니다. 수동으로 설정할 필요가 없습니다.

각 스킬에는 다음 내용이 포함되어 있습니다.

- **테마별 문서** — 테마 사용 및 사용자 정의에 대한 가이드
- **블록 및 패턴 참조** — 사용 가능한 블록, 패턴 및 디자인 옵션
- **사용자 정의 예시** — 일반적인 작업을 위한 코드 스니펫 및 설정 패턴
- **모범 사례** — 테마의 아키텍처 및 워크플로우에 대한 권장 사항

## 사용 가능한 테마별 스킬 {#available-theme-aware-skills}

### 블록 테마 (Block Themes) {#block-themes}

**적용 대상:** WordPress의 블록 기반(Full Site Editing) 아키텍처를 사용하는 테마.

블록 테마 스킬은 다음 내용에 대한 가이드를 제공합니다.

- 블록 편집기를 사용한 템플릿 생성 및 편집
- 블록 패턴 및 재사용 가능한 블록 작업
- 글로벌 스타일 및 `theme.json` 설정 사용자 정의
- 테마 블록 및 변형 사용
- 사이트를 위한 사용자 정의 블록 패턴 구축

**자동 활성화 시점:** 활성 테마가 블록 테마( `block-templates` 기능을 지원)인 경우.

### 클래식 테마 (Classic Themes) {#classic-themes}

**적용 대상:** PHP 템플릿과 클래식 에디터를 사용하는 전통적인 WordPress 테마.

클래식 테마 스킬은 다음 내용에 대한 가이드를 제공합니다.

- PHP 템플릿 파일 및 훅(hooks) 작업
- 커스터마이저를 통한 테마 외관 사용자 정의
- 위젯 영역 및 사이드바 사용
- CSS 수정 및 자식 테마 개발
- 테마 계층 구조 및 템플릿 태그 이해

**자동 활성화 시점:** 활성 테마가 클래식(블록이 아닌) 테마인 경우.

### Kadence Blocks {#kadence-blocks}

**적용 대상:** 고급 블록 기반 디자인을 위해 Kadence Blocks 플러그인을 사용하는 사이트.

Kadence Blocks 스킬은 다음 내용에 대한 가이드를 제공합니다.

- Kadence의 고급 블록 라이브러리 사용 (Hero, Testimonials, Pricing 등)
- Kadence 블록 설정 및 반응형 옵션 구성
- Kadence의 그리드 및 컨테이너 블록을 사용한 사용자 정의 레이아웃 구축
- 테마와 Kadence 블록 통합
- Kadence의 디자인 시스템 및 프리셋 활용

**자동 활성화 시점:** 사이트에 Kadence Blocks 플러그인이 활성화된 경우.

### Kadence Theme {#kadence-theme}

**적용 대상:** 블록 기반 디자인 및 사용자 정의를 위해 Kadence 테마를 사용하는 사이트.

Kadence Theme 스킬은 다음 내용에 대한 가이드를 제공합니다.

- 글로벌 스타일 및 `theme.json`을 통한 Kadence 테마 사용자 정의
- Kadence의 내장 블록 패턴 및 템플릿 사용
- Kadence 테마 설정 및 옵션 구성
- Kadence의 디자인 시스템을 사용한 사용자 정의 디자인 구축
- 인기 플러그인 및 도구와 Kadence 통합

**자동 활성화 시점:** 활성 테마가 Kadence 테마인 경우.

## 스킬 선택 방법 {#how-skills-are-selected}

어시스턴트는 매 메시지마다 활성 테마와 설치된 플러그인을 자동으로 감지합니다. 일치하는 테마별 스킬이 있으면, 해당 스킬이 어시스턴트의 컨텍스트에 자동으로 로드됩니다. 사용자가 수동으로 스킬을 활성화하거나 전환할 필요가 없습니다.

### 여러 스킬 {#multiple-skills}

사이트에 여러 스킬이 적용되는 경우(예: Kadence Blocks와 Kadence Theme이 모두 활성화된 경우), 어시스턴트는 적용 가능한 모든 스킬에 접근할 수 있으며 각 스킬의 가이드를 참조할 수 있습니다.

### 테마 전환 시 {#switching-themes}

활성 테마를 변경하면, 어시스턴트가 사용할 수 있는 스킬 목록이 다음 메시지에서 자동으로 업데이트됩니다. 예를 들어:

1. **Block Themes** 스킬이 활성화된 블록 테마를 사용하고 있습니다.
2. 클래식 테마로 전환합니다.
3. 다음 메시지에서 **Classic Themes** 스킬이 자동으로 로드되고, **Block Themes** 스킬은 더 이상 사용할 수 없게 됩니다.

## 테마별 스킬 사용하기 {#using-theme-aware-skills}

테마별 스킬을 활용하려면, 채팅 인터페이스에 원하는 작업을 간단히 설명하기만 하면 됩니다. 어시스턴트가 자동으로 적절한 스킬의 가이드를 참조하여 답변합니다.

### 예시 프롬프트 {#example-prompts}

**블록 테마의 경우:**
> "블록 패턴을 사용하여 배경 이미지와 중앙 정렬된 텍스트가 있는 히어로 섹션을 만드세요."

**클래식 테마의 경우:**
> "자식 테마를 사용하여 사이드바에 사용자 정의 위젯 영역을 추가하세요."

**Kadence Blocks의 경우:**
> "Kadence Testimonials 블록을 사용하여 사용 후기 섹션을 만드세요."

**Kadence Theme의 경우:**
> "헤더 레이아웃과 내비게이션 메뉴 스타일을 사용자 정의하세요."

어시스턴트는 활성 테마와 플러그인에 맞춰진 테마별 가이드와 코드 예시를 제공할 것입니다.

:::note
테마별 스킬은 Superdav AI Agent v1.10.0 이상 버전부터 자동으로 사용 가능합니다. 추가적인 설정이나 구성은 필요하지 않습니다.
:::
