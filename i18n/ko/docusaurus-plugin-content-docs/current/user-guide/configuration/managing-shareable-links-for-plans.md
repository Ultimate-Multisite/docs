# 플랜 공유 가능한 링크 관리

Ultimate Multisite는 귀하가 고객에게 직접 플랜 링크를 공유할 수 있는 옵션을 제공합니다. 이를 위해서는 플랜 편집 페이지 중 하나로 이동하여 _Click to copy the Shareable Link_ 버튼을 클릭하십시오.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_플랜 편집 페이지에서 공유 가능한 링크를 생성하는 버튼을 찾을 수 있습니다_

이 메커니즘은 링크를 생성할 때 기본 청구 주기를 사용한다는 점에 유의하십시오. 예를 들어 기본 청구 주기가 월간인 경우, 최종적으로 다음과 같은 링크가 생성됩니다: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

공유 가능한 링크를 분기별로 기본 설정하려면 끝에 3을 추가해야 합니다

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

연간 일정으로 _premium_ 플랜을 선택하려면 1을 추가한 뒤 단어를 붙입니다:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

다른 청구 주기에 대해 하나의 링크만 사용할 수 없다는 점을 기억하십시오. 각 청구 주기마다 특정 링크를 공유해야 합니다.

링크를 고객과 공유하면 해당 링크가 귀하의 가입 흐름의 첫 번째 단계로 리디렉션됩니다
