# 요금제의 공유 가능한 링크 관리 {#managing-shareable-links-for-plans}

Ultimate Multisite는 고객과 요금제 링크를 직접 공유할 수 있는 옵션을 제공합니다. 이렇게 하려면 요금제의 편집 페이지 중 하나로 이동하여 _공유 가능한 링크를 복사하려면 클릭_ 버튼을 클릭하세요.

![공유 가능한 링크 버튼이 있는 상품 편집 페이지](/img/config/product-edit-full.png)

_요금제의 편집 페이지에서 공유 가능한 링크를 만드는 버튼을 찾을 수 있습니다_

이 메커니즘은 링크를 만들 때 기본 청구 주기를 사용한다는 점에 유의하세요. 즉, 예를 들어 기본 청구가 월간인 경우 최종적으로 다음과 같은 링크가 됩니다: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

공유 가능한 링크의 기본값을 분기별로 설정하려면 끝에 3을 추가해야 합니다.

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

연간 일정으로 _premium_ 요금제를 선택하려면 1과 해당 단어를 추가하면 됩니다:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

서로 다른 청구 주기에 대해 하나의 링크만 사용할 수는 없다는 점을 기억하세요. 각 청구 주기마다 특정 링크 하나를 공유해야 합니다.

고객과 링크를 공유하면, 해당 링크는 고객을 가입 흐름의 첫 단계로 리디렉션합니다.
