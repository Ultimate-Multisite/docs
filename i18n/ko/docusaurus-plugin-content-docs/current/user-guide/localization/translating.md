# Ultimate Multisite 번역

Ultimate Multisite는 영어로 개발되었지만, 플러그인은 완전히 번역 가능하며 번역 기여를 적극적으로 환영합니다.

## Core Plugin Translations

핵심 Ultimate Multisite 플러그인은 공식 **WordPress.org 번역 플랫폼**을 통해 번역됩니다:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

이것은 표준 WordPress 커뮤니티 번역 시스템입니다. 번역이 충분한 승인 임계값에 도달하면, 자동으로 모든 Ultimate Multisite 사용자에게 WordPress 플러그인 업데이트를 통해 배포됩니다.

**우리가 당신의 도움이 필요합니다!** 영어 이외의 언어를 구사한다면 번역 기여를 고려해 주세요. 몇 개의 문자열만 번역해도 도움이 됩니다. 시작하려면:

1. [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/) 방문
2. 언어 선택
3. '번역 기여' 클릭
4. 무료 WordPress.org 계정이 필요합니다

**언어별 번역 편집자(번역 승인 가능)를 원하신다면, [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions)를 통해 연락해 주세요.**

## Addon Translations

애드온 플러그인은 별도로 다음에서 번역됩니다:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

애드온 번역은 현재 AI가 생성한 초기 버전입니다. 모든 지원 언어를 포함하지만 어색한 표현이나 부정확함이 있을 수 있습니다. 인간 번역가가 검토하고 개선안을 제안해 주시면 보다 자연스럽고 정확한 번역을 만들 수 있도록 도와드립니다.

## Local Translations

업스트림에 기여하기보다 자신의 프로젝트를 위해 로컬에서 번역을 만들고 싶다면, 두 가지 옵션이 있습니다:

### Using Poedit

데스크톱 애플리케이션 **[Poedit](https://poedit.net)**를 사용하여 `.pot` 파일을 번역할 수 있습니다:

1. [poedit.net](https://poedit.net)에서 Poedit 다운로드
2. `wp-content/plugins/ultimate-multisite/lang/`에서 `.pot` 파일 열기
3. Poedit 인터페이스에서 문자열 번역
4. 저장 -- Poedit이 `.po`와 `.mo` 파일을 생성합니다
5. 출력 파일을 `lang` 폴더에 배치

### Using Loco Translate

**[Loco Translate](https://wordpress.org/plugins/loco-translate/)** WordPress 플러그인은 데스크톱 도구 없이 관리자 패널에서 직접 번역할 수 있게 해 줍니다. 번역 파일 생성 및 컴파일을 대신 처리해 줍니다.
