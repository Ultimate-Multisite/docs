# Codeable에서 맞춤 개발 도움을 받는 방법

때로는 네트워크 흐름이 우리가 원하는 방식 그대로 작동하도록 멋진 맞춤화가 꼭 필요할 때가 있습니다. 이것이 찾고 있는 것이라면, Ultimate Multisite 전문 지식을 갖춘 경험 많은 개발자를 찾을 수 있는 곳인 [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE)을 추천합니다.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE)은 고객과 검증된 WordPress 프리랜서 개발자를 연결해 주는 플랫폼으로, 어떤 규모의 프로젝트든 작업할 수 있습니다. 원하는 작업에 특별히 적합한 전문가와 매칭될 수 있도록 꼼꼼한 지원 절차를 갖추고 있습니다.

### Codeable에서 프로젝트를 만드는 방법 {#how-to-create-a-project-in-codeable}

먼저 [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE)로 이동하여 **프로젝트 시작** 버튼을 클릭하세요.

<!-- 스크린샷 사용 불가: 프로젝트 시작 버튼이 있는 Codeable 홈페이지 -->

이제 프로젝트를 제출하려면 왼쪽에서 **프로젝트 시작**을 클릭하세요.

<!-- 스크린샷 사용 불가: 프로젝트 시작 옵션이 있는 Codeable 프로젝트 제출 페이지 -->

"I need help with"에서 **Customizing**을 선택하세요:

<!-- 스크린샷 사용 불가: Customizing이 선택된 Codeable 프로젝트 양식 -->

"My"에서 **Plugin**을 선택하세요:

<!-- 스크린샷 사용 불가: Plugin이 선택된 Codeable 프로젝트 양식 -->

그런 다음 **Continue** 버튼을 클릭하세요.

<!-- 스크린샷 사용 불가: Codeable 프로젝트 양식 Continue 버튼 -->

이제 프로젝트와 필요 사항을 최대한 잘 설명할 차례입니다. 가능한 한 많은 정보를 제공해 보세요!

<!-- 스크린샷 사용 불가: Codeable 프로젝트 설명 양식 -->

가격 산정을 돕기 위해 다음 질문에 답하세요.

<!-- 스크린샷 사용 불가: Codeable 프로젝트 가격 산정 질문 -->

그 후 아직 [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) Account가 없다면 하나를 만들고 **내 프로젝트 게시** 버튼을 클릭하세요.

<!-- 스크린샷 사용 불가: Codeable Account 생성 및 내 프로젝트 게시 버튼 -->

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) 프로젝트를 게시하면, 플랫폼이 프로젝트에 필요한 기술 세트와 맞는 개발자와 연결해 줍니다.

이 과정은 WordPress와 Ultimate Multisite 경험이 있는, 해당 작업에 적합한 사람을 찾는 데 도움이 됩니다!

Codeable의 최저 요금은 시간당 미화 $70–$120입니다. 견적은 프로젝트의 작업 범위, 복잡성, 긴급도에 따라 달라집니다.

견적을 받으면, 필요에 가장 잘 맞는 개발자를 고용하고 함께 프로젝트를 시작할 수 있습니다!
