# 사이트 빌더 문의 양식 {#site-builder-contact-forms}

Superdav AI Agent v1.10.0을 사용하면 사이트 빌더 에이전트를 통해 채팅 인터페이스에서 직접 문의 양식을 만들 수 있습니다. 이 기능을 사용하면 채팅을 벗어나지 않고도 모든 페이지에 완벽하게 작동하는 문의 양식을 추가할 수 있습니다.

## 개요 {#overview}

사이트 빌더 문의 양식 기능(`create_contact_form`)은 설치된 양식 플러그인을 자동으로 감지하여 해당 플러그인의 기본 기능을 사용하여 문의 양식을 생성합니다. 지원되는 양식 플러그인은 다음과 같습니다:

- Contact Form 7
- WPForms
- Fluent Forms
- Gravity Forms

이 어시스턴트는 사이트에서 사용 가능한 최적의 양식 플러그인을 자동으로 선택하고 해당 플러그인에 맞춘 문의 양식을 생성합니다.

## 문의 양식 생성하기 {#creating-a-contact-form}

사이트 빌더를 사용하여 문의 양식을 만들려면 다음 단계를 따르세요:

1. WordPress 관리자에서 **Gratis AI Agent** 채팅 패널을 엽니다.
2. 채팅 헤더에 있는 에이전트 아이콘을 클릭하여 **Site Builder** 에이전트로 전환합니다.
3. 만들고 싶은 문의 양식을 설명합니다. 예를 들어:

   > "연락처 페이지에 이름, 이메일, 메시지, 전화번호 필드가 있는 문의 양식을 추가해 주세요."

   또는 간단하게:

   > "연락처 페이지에 문의 양식을 만들어 주세요."

4. 사이트 빌더가 문의 양식을 생성하고 바로 삽입할 수 있는 쇼트코드를 반환합니다.

## 생성된 쇼트코드 사용하기 {#using-the-generated-shortcode}

사이트 빌더가 문의 양식을 생성하면 쇼트코드(예: `[contact-form-7 id="123"]`)를 반환합니다. 다음 방법으로 사용할 수 있습니다:

1. **페이지나 게시물에 삽입** — 쇼트코드를 복사하여 블록 에디터 또는 클래식 에디터의 모든 페이지나 게시물에 붙여넣습니다.
2. **사이트 빌더를 통해 추가** — 사이트 빌더에게 특정 페이지에 양식을 자동으로 추가하도록 요청합니다:

   > "연락처 페이지에 문의 양식을 추가해 주세요."

3. **템플릿에서 사용** — PHP에 익숙하다면, 테마 템플릿 파일에 쇼트코드를 추가할 수 있습니다.

## 문의 양식 사용자 지정 {#customizing-the-contact-form}

사이트 빌더가 문의 양식을 생성한 후, 더 세부적으로 사용자 지정할 수 있습니다:

### 채팅 인터페이스를 통해 {#via-the-chat-interface}

사이트 빌더에게 양식 수정을 요청합니다:

> "문의 양식에 제목 필드를 추가하고 메시지 필드를 필수 항목으로 만들어 주세요."

사이트 빌더는 양식을 업데이트하고 수정된 쇼트코드를 반환합니다.

### 양식 플러그인 관리자 인터페이스를 통해 {#via-the-form-plugins-admin-interface}

양식 플러그인의 설정에서 직접 양식을 사용자 지정할 수도 있습니다:

1. WordPress 관리자에서 **Contact Form 7** (또는 설치된 양식 플러그인)으로 이동합니다.
2. 사이트 빌더가 생성한 양식을 찾습니다.
3. 필요한 대로 양식 필드, 유효성 검사 규칙, 이메일 알림을 편집합니다.

## 양식 플러그인 및 호환성 {#form-plugins-and-compatibility}

사이트 빌더는 사이트에 설치된 양식 플러그인을 자동으로 감지하여 이를 사용하여 문의 양식을 생성합니다. 여러 양식 플러그인이 설치된 경우, 사이트 빌더는 다음 순서로 우선순위를 지정합니다:

1. Contact Form 7
2. WPForms
3. Fluent Forms
4. Gravity Forms

이 플러그인 중 어느 것도 설치되어 있지 않은 경우, 사이트 빌더는 문의 양식을 생성하기 전에 하나를 설치할 것을 권장합니다.

## 이메일 알림 {#email-notifications}

사이트 빌더가 생성한 문의 양식은 기본적으로 사이트 관리자에게 이메일 알림을 보내도록 설정됩니다. 수신 이메일 주소와 알림 메시지를 사용자 지정할 수 있습니다:

1. 양식 플러그인의 관리자 인터페이스로 이동합니다.
2. 사이트 빌더가 생성한 양식을 찾습니다.
3. 이메일 알림 설정을 편집합니다.

자세한 지침은 사용 중인 양식 플러그인의 문서를 참조하세요:

- [Contact Form 7 Email Notifications](https://contactform7.com/docs/)
- [WPForms Email Notifications](https://wpforms.com/docs/)
- [Fluent Forms Email Notifications](https://fluentforms.com/docs/)
- [Gravity Forms Email Notifications](https://docs.gravityforms.com/)

## 사용 사례 예시 {#example-use-cases}

### 간단한 문의 양식 {#simple-contact-form}

> "이름, 이메일, 메시지 필드가 있는 간단한 문의 양식을 만들어 주세요."

### 다단계 양식 {#multi-step-form}

> "연락처 정보가 담긴 첫 번째 단계와 메시지 및 선호 연락 방법이 담긴 두 번째 단계로 구성된 문의 양식을 만들어 주세요."

### 조건부 로직이 적용된 양식 {#form-with-conditional-logic}

> "드롭다운 메뉴에서 사용자가 선택한 항목에 따라 다른 필드를 보여주는 문의 양식을 만들어 주세요."

### 파일 업로드 기능이 있는 양식 {#form-with-file-upload}

> "사용자가 파일이나 첨부 파일을 업로드할 수 있는 문의 양식을 만들어 주세요."

:::note
문의 양식 생성 기능은 Superdav AI Agent v1.10.0 이상에서 사용할 수 있습니다. 이 기능을 사용하려면 사이트 빌더 에이전트가 활성화되어 있어야 합니다.
:::
