# 사이트 내보내기 {#site-export}

**사이트 내보내기** 관리 페이지를 사용하면 네트워크 관리자가 하나의 사이트 또는 전체 네트워크를 다운로드 가능한 아카이브로 패키징하여 마이그레이션, 백업 또는 인계 작업을 할 수 있습니다.

## 사이트 하나 내보내기 {#exporting-one-site}

**Ultimate Multisite > 사이트 내보내기**로 이동하여 **새 사이트 내보내기 생성**을 선택합니다. 내보내고자 하는 서브사이트를 선택한 다음, 아카이브에 업로드, 플러그인, 테마를 포함할지 여부를 선택합니다.

내보내기가 완료되면 **기존 내보내기** 목록에서 ZIP 파일을 다운로드합니다. 내보내기 ZIP 파일에는 이제 자체 부팅 `index.php`와 `readme.txt`가 포함되어 있어, 별도의 가져오기 플러그인을 먼저 설치하지 않고도 새 호스팅 환경에 아카이브를 업로드하고 시작할 수 있습니다.

## 전체 네트워크 내보내기 {#exporting-the-whole-network}

네트워크 내의 모든 서브사이트를 포함하는 단일 아카이브가 필요할 때는 사이트 내보내기 페이지에서 **네트워크 내보내기**를 사용합니다. 이는 호스팅 환경을 마이그레이션하거나, 재해 복구 훈련, 또는 각 서브사이트가 함께 이동해야 하는 스테이징 환경 재구축 전에 유용합니다.

네트워크 내보내기는 단일 사이트 내보내기보다 훨씬 클 수 있으므로, 트래픽이 적은 시간대에 실행하고 업로드, 플러그인, 테마 및 생성된 ZIP 파일에 충분한 여유 공간이 있는지 대상 저장 공간을 확인하세요.

### 네트워크 가져오기 번들 {#network-import-bundles}

Ultimate Multisite 2.12.0부터 사이트 내보내기는 **네트워크 가져오기 번들**을 생성할 수 있습니다. 이는 네트워크 전체의 사이트 복원을 간소화하기 위해 설계된 전문 아카이브입니다. 네트워크 가져오기 번들에는 여러 사이트를 새로운 네트워크 설치로 복원하는 데 필요한 모든 파일과 메타데이터가 포함되어 있습니다.

#### 네트워크 가져오기 번들 생성하기 {#generating-a-network-import-bundle}

1. **Ultimate Multisite > 사이트 내보내기**로 이동합니다.
2. **새 네트워크 내보내기 생성**을 클릭합니다.
3. 내보내기 유형으로 **네트워크 가져오기 번들**을 선택합니다.
4. 업로드, 플러그인, 테마 포함 여부를 선택합니다.
5. **생성**을 클릭합니다.
6. **기존 내보내기** 목록에서 번들 ZIP 파일을 다운로드합니다.

#### 네트워크 가져오기 번들로 복원하기 {#restoring-from-a-network-import-bundle}

네트워크 가져오기 번들에서 사이트를 복원하려면 다음 단계를 따르세요.

1. 대상 호스트에 Ultimate Multisite를 설치합니다.
2. 멀티사이트 설정 마법사를 완료합니다.
3. 새 네트워크에서 **Ultimate Multisite > 사이트 내보내기**로 이동합니다.
4. **네트워크 번들 가져오기**를 클릭합니다.
5. 네트워크 가져오기 번들 ZIP 파일을 업로드합니다.
6. 화면에 표시되는 가져오기 지침을 따릅니다.
7. 가져오기 프로세스는 모든 사이트, 콘텐츠 및 구성을 복원합니다.

네트워크 가져오기 번들은 다음을 보존합니다.
- 사이트 콘텐츠 (게시물, 페이지, 사용자 정의 포스트 타입)
- 사이트 설정 및 옵션
- 사용자 역할 및 권한
- 플러그인 및 테마 (번들에 포함된 경우)
- 미디어 업로드 (번들에 포함된 경우)
- 사용자 정의 데이터베이스 테이블 및 데이터

## 자체 부팅 내보내기 ZIP 설치하기 {#installing-a-self-booting-export-zip}

자체 부팅 ZIP 파일을 새 호스트에 복원하려면 다음 단계를 따르세요.

1. 내보낸 ZIP 내용을 대상 웹 루트에 업로드합니다.
2. 브라우저에서 업로드된 `index.php`를 엽니다.
3. 번들 내보내기 패키지에서 제공하는 화면 설치 지침을 따릅니다.
4. 임시 파일을 제거하기 전에 번들로 제공된 `readme.txt`를 검토하여 내보내기 관련 참고 사항을 확인합니다.

추가 기능별 설치 및 가져오기 세부 정보는 [Site Exporter 추가 기능 문서](/addons/site-exporter)를 참조하세요.

Ultimate Multisite 2.9.0에서 추가된 단일 사이트 도구에 대한 내용은 [내보내기 및 가져오기](/user-guide/administration/export-import)를 참조하세요.
