# 대시보드 개요

Ultimate Multisite는 Ultimate Multisite 메뉴 아래에 전용 **Dashboard**를 제공하며, 네트워크 대시보드에도 위젯을 추가합니다. 이 문서는 두 곳에서 볼 수 있는 내용을 다룹니다.

## 네트워크 대시보드

Ultimate Multisite는 **My Sites → Network Admin → Dashboard**에서 찾을 수 있는 네트워크 수준 대시보드에 유용한 위젯을 삽입합니다.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

각 위젯이 제공하는 내용은 다음과 같습니다:

**First Steps:** 새 설치 시 나타나며 결제 수단 설정 및 테스트 계정 생성과 같은 설정 작업을 완료하도록 알립니다.

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Summary:** 하루 동안의 가입자 수와 수익을 보고합니다. MRR (Monthly Recurring Revenue)은 수익 창출 멤버십을 가진 고객 전체의 예상 총액을 나타냅니다.

![Summary widget](/img/admin/network-dashboard-summary.png)

**Activity Stream:** Ultimate Multisite 전반의 이벤트를 보고합니다 — 가입, 취소, 사이트 생성, 삭제 및 기타 활동.

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** 네트워크의 사용자와 사이트에 대한 간단한 요약을 표시하며, 새 사이트 또는 사용자를 만들 수 있는 바로 가기를 제공합니다.

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** 최신 Ultimate Multisite 뉴스를 가져와 표시합니다. 업데이트, 치명적인 버그 및 보안 수정 사항이 포함됩니다.

![News widget](/img/admin/network-dashboard-news.png)

## Ultimate Multisite 대시보드

네트워크 대시보드는 네트워크에 대한 전체 정보를 제공하는 반면, Ultimate Multisite 대시보드(Ultimate Multisite 메뉴의 최상위 수준)는 서비스에 대한 정보를 제공합니다.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Monthly Recurring Revenue Growth:** 12개월 달력 기간 동안의 월별 수익 분해를 표시합니다. 신규 수익은 파란색으로 표시되고, 취소는 빨간색으로 표시됩니다.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**Date-range Support:** 필요한 정보를 집중하기 위해 시간 범위별로 대시보드 데이터를 필터링합니다. 날짜 범위는 아래 위젯에 표시되는 내용을 제어합니다.

![Date range and dashboard stats](/img/admin/um-dashboard-date-range.png)

대시보드 위젯은 다음과 같습니다:

- **Revenue** — 기간 동안의 총 수익과 MRR 및 제품별 분해
- **New Memberships** — 선택한 기간에 생성된 신규 멤버십 수
- **Signups by Countries** — 가입자에 대한 지리적 분해
- **Signups by Forms** — 각 체크아웃 폼을 통해 가입한 고객 수와 관리자에 의해 추가된 수
- **Most Visited Sites** — 네트워크에서 가장 많이 방문한 사이트 (방문 수 계산을 **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**에서 활성화해야 함)

![Dashboard statistics](/img/admin/um-dashboard-stats.png)
