# 내보내기 및 가져오기 {#export--import}

Ultimate Multisite 2.9.0에서는 **도구 > 내보내기 및 가져오기** 메뉴에 단일 사이트 **내보내기 및 가져오기** 도구가 추가되었습니다. 이 도구는 하나의 WordPress 사이트를 ZIP 파일로 패키징하거나, 해당 ZIP 파일을 복원하거나, 호환되는 Ultimate Multisite 및 단일 사이트 WordPress 설치 간에 사이트를 이동해야 할 때 사용합니다.

## 필수 권한 {#required-permissions}

내보내기 또는 가져오기를 수행하는 사이트의 WordPress **도구** 메뉴에 접근할 수 있는 관리자 계정으로 로그인해야 합니다. 멀티사이트 네트워크의 경우, 네트워크 수준의 Ultimate Multisite 도구에서 서브사이트를 내보내거나 가져올 때는 네트워크 관리자 계정을 사용하세요.

내보내기 ZIP 다운로드는 인증된 다운로드 엔드포인트를 통해 제공되므로, 다운로드가 완료될 때까지 관리자 세션을 유지하고 생성된 다운로드 URL을 외부에 공유하지 마세요.

## 사이트를 ZIP으로 내보내기 {#exporting-a-site-to-a-zip}

1. 복사하려는 사이트의 WordPress 관리자 페이지로 이동하여 **도구 > 내보내기 및 가져오기**로 이동합니다.
2. 내보내기 영역을 열고 패키징하려는 사이트를 선택합니다.
3. 업로드, 플러그인, 테마 등 선택적으로 포함할 콘텐츠가 있다면 해당 옵션을 선택합니다.
4. 내보내기를 시작하고 프로세스가 완료될 때까지 기다립니다. 사이트가 클 경우, ZIP 파일이 준비되기 전에 백그라운드에서 완료되는 시간이 걸릴 수 있습니다.
5. 내보내기 목록에서 완료된 ZIP 파일을 다운로드합니다.

ZIP 파일은 안전한 위치에 보관하세요. 사이트 콘텐츠, 설정, 미디어 파일 및 선택된 코드 자산을 포함할 수 있습니다.

## 내보내기에 포함되는 내용 {#what-the-export-includes}

내보내기 ZIP 파일에는 다음 내용이 포함될 수 있습니다.

- 선택된 사이트의 데이터베이스 콘텐츠 및 구성 정보.
- 업로드가 포함된 경우 업로드된 미디어 파일.
- 해당 옵션이 선택된 경우 플러그인 및 테마.
- 대상 설치에서 사이트를 재구축하는 데 내보내기 및 가져오기 도구가 사용하는 가져오기 메타데이터.

정확한 ZIP 크기는 미디어 양, 선택된 플러그인 및 테마, 사이트 데이터베이스 테이블 크기에 따라 달라집니다.

## ZIP에서 사이트 가져오기 {#importing-a-site-from-a-zip}

1. 대상 WordPress 사이트에서 **도구 > 내보내기 및 가져오기**로 이동합니다.
2. 가져오기 영역을 열고 내보내기 및 가져오기 도구로 생성된 ZIP 파일을 업로드합니다.
3. 사이트가 새 주소를 사용해야 하는 경우 대체 URL을 입력하거나, 원래 URL을 유지하려면 해당 필드를 비워둡니다.
4. 가져오기가 완료된 후 업로드된 ZIP 파일을 삭제할지 여부를 선택합니다.
5. **가져오기 시작**을 클릭합니다.
6. 가져오기가 완료될 때까지 보류 중인 가져오기를 모니터링합니다. 프로세스를 완료하기 전에 중지해야 하는 경우에만 **가져오기 취소**를 사용하세요.
7. 정상적인 트래픽이나 고객 접근을 허용하기 전에 가져온 사이트를 검토합니다.

단일 사이트 WordPress 설치의 경우, ZIP 파일을 가져오면 현재 사이트가 가져온 사이트로 대체됩니다. 시작하기 전에 대상 사이트의 전체 백업을 생성하고, 같은 시간에 같은 사이트에 대해 여러 번의 가져오기를 시작하는 것은 피하세요.

## 제한 사항 및 호환성 참고 사항 {#limitations-and-compatibility-notes}

- 업로드 디렉터리나 미디어 라이브러리가 매우 큰 경우 ZIP 파일 크기가 커질 수 있습니다. 대규모 사이트를 내보내거나 가져오기 전에 PHP 업로드 제한, 실행 제한, 디스크 공간, 메모리 및 서버 타임아웃 설정을 확인하세요.
- 미디어 라이브러리가 매우 큰 경우, 트래픽이 적은 유지보수 시간 동안 이동해야 할 수 있습니다.
- 대상 WordPress 설치는 호환되는 WordPress, PHP, Ultimate Multisite, 플러그인 및 테마 버전을 실행해야 합니다.
- 단일 사이트 가져오기는 대상 사이트를 대체합니다. 병합 도구가 아닙니다.
- 멀티사이트-대-단일 사이트 및 단일 사이트-대-멀티사이트 이동은 대상 환경이 내보내진 사이트의 플러그인, 테마, URL 및 필요한 Ultimate Multisite 구성 요소를 실행할 수 있을 때만 지원됩니다.
- ZIP 파일은 비공개로 유지하세요. 내보내진 사이트의 데이터베이스 콘텐츠, 업로드 파일 및 구성 세부 정보를 포함할 수 있습니다.

이전 네트워크 수준 내보내기 워크플로우는 [Site Export](/user-guide/administration/site-export)를 참조하세요.
