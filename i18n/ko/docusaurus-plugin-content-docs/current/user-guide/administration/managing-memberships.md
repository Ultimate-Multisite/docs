# 멤버십 관리 {#managing-memberships}

Ultimate Multisite에서는 멤버십을 관리하고 네트워크의 모든 구독에 접근할 수 있습니다. 플랜과 제품, 사이트 수, 청구 금액과 주기, 결제 내역, 타임스탬프 같은 구독 세부 정보를 검토하고 편집하세요.

이 페이지에 접근하려면 Ultimate Multisite 메뉴의 **Memberships** 탭으로 이동하세요.

![멤버십 목록 페이지](/img/admin/memberships-list.png)

Memberships 페이지에서는 고객 목록과 해당 Account에 연결된 제품을 볼 수 있습니다. 멤버십은 상태별로 필터링할 수 있습니다: 활성, 대기 중, 보류 중, 만료됨, 취소됨.

무료 멤버십은 **평생** 멤버십으로 처리됩니다. 만료일이 지정되지 않으며 자동으로 만료되지 않습니다. 무료 플랜이 체험 기간 또는 고정 기간 후 종료되어야 한다면, 체험 기간이 있는 유료 제품을 사용하거나, 멤버십을 수동으로 취소하거나, 접근 권한이 변경되어야 할 때 고객을 다른 플랜으로 이동하세요.

멤버십을 클릭하면 청구 정보, 사이트 또는 방문 제한 등 멤버십 세부 정보를 편집할 수 있는 페이지에 접근할 수 있습니다.

![멤버십 편집 인터페이스](/img/admin/membership-edit.png)

멤버십 편집 페이지의 전체 모습은 다음과 같습니다:

![멤버십 편집 전체 인터페이스](/img/admin/membership-edit-full.png)

네트워크에 이전에 등록된 사용자와 연결된 새 멤버십도 추가할 수 있습니다. **Add Membership**을 클릭하고 세부 정보를 구성하세요.

![멤버십 추가 모달](/img/admin/membership-add-modal.png)

그런 다음 **Create Membership**을 클릭하세요. 새 멤버십이 멤버십 목록에 표시됩니다.
