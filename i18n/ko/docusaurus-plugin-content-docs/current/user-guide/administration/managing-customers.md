# 고객 관리

**Customers** 페이지는 네트워크에 등록된 회원 목록을 보여줍니다. 이 페이지는 이름, 이메일 주소, 마지막 로그인 및 사용자의 하위 사이트에 사이트 관리자 권한으로 로그인할 수 있는 **switch to** 링크를 포함한 기본 사용자 정보를 표시합니다.

![Customers list page](/img/admin/customers-list.png)

**Manage** 버튼을 고객 카드 아래에서 클릭하면 해당 고객에 대한 상세 정보를 확인할 수 있습니다.

고객 상세 페이지에서 **At a Glance** 블록은 고객 ID, 마지막 로그인 시간, 그리고 이 고객이 지출한 총 금액을 보여줍니다.

**Customer Options** 블록을 사용하면 이 고객을 **VIP**(더 쉽게 찾을 수 있도록 하는 태그)로 설정할 수 있습니다. **Billing Info** 탭은 이메일, 주소, 국가 등 청구 정보를 표시합니다.

**Fields** 블록은 고객 사용자 이름, 이메일을 표시하고 **Email Verification** 상태를 변경할 수 있게 해줍니다.

Customer Options 블록 아래에는 네 개의 추가 블록이 있습니다:

- **Memberships** — 현재 고객이 구독 중인 구독
- **Payments** — 고객의 결제 내역
- **Sites** — 고객이 소유한 사이트
- **Events** — 고객과 관련된 이벤트(이메일 인증, 청구 변경 등)

항목 옆에 있는 화살표 아이콘(→)을 클릭하면 해당 항목의 상세 페이지로 이동합니다. 예를 들어, 사이트 옆의 화살표를 클릭하면 사이트 편집 페이지로 이동하여 멤버십 연결, 사이트 유형, 활성 상태를 수정할 수 있습니다.

멤버십과 결제에도 동일하게 적용됩니다. 각 항목은 멤버십 상태, 연결된 고객, 제품, 결제 게이트웨이 설정을 편집하고 환불을 처리할 수 있는 상세 페이지로 연결됩니다.

고객 페이지 오른쪽에는 다음 항목이 있습니다:

- **Last Login & IPs** — 마지막 로그인 시간, IP 주소, 국가
- **Timestamps** — 고객 계정이 생성된 시점과 마지막 수정 시점
- **Delete** — 고객을 영구적으로 삭제하는 옵션

:::warning
고객을 삭제하면 영구적으로 삭제되며 되돌릴 수 없습니다.
:::
