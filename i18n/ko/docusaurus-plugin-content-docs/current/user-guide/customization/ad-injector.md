# Ultimate Multisite: Ad Injector를 사용한 네트워크에 광고 삽입

Ultimate Multisite: Ad Injector는 **무료 애드온**으로, 편집 페이지에서 바로 광고를 배치할 수 있는 도구를 제공합니다.

Ultimate Multisite: Ad Injector를 설치한 후, _플러그인_ 페이지로 이동해 활성화하세요.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-KgDegqpbeb.png)

_네트워크 활성화 버튼을 클릭하세요_

애드온이 활성화되면, 플랜 편집 화면의 _고급 옵션_ 메타 박스에 _광고 설정_이라는 새 옵션이 생깁니다.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-DA7Z84yQTu.png)

_이제 각 플랜에 대한 광고 설정을 편집할 수 있습니다_

해당 탭에서, 게시물과 페이지의 프론트엔드에 콘텐츠 앞뒤에 광고 코드를 추가할 수 있습니다.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-Rw9wObGUEN.png)

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-1TKkWh8qoF.png)

광고 코드를 삽입하면, 해당 플랜의 게시물과 페이지에 표시됩니다.

해당 플랜 사용자용 관리 페이지 헤더에도 광고를 포함할 수 있습니다.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-kdpwvfMgTp.png)

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-1CPGJZn3km.png)

사용자 백엔드에 광고를 표시하려면 관리 패널 광고 코드 옵션으로 이동하세요

### 광고를 넘어

Ultimate Multisite: Ad Injector를 사용해 고급 플랜에 지원을 제공할 수도 있습니다.

단순히 _관리 패널 광고_ 옵션으로 이동해 지원 도구용 HTML 코드를 추가하세요.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-hnOhLc8s8i.png)

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-09wylykj5l.png)

특정 플랜에 고객 지원을 추가하세요

이렇게 하면 사용자는 관리 패널에서 바로 연락할 수 있습니다!
