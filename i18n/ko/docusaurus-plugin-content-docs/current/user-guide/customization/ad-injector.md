# Ultimate Multisite로 네트워크에 광고 배치하기: Ad Injector

Ultimate Multisite: Ad Injector는 플랜 편집 페이지에서 바로 플랜에 광고를 배치할 수 있는 도구를 제공하는 **무료 add-on**입니다.

Ultimate Multisite: Ad Injector를 설치한 후 _Plugins_ 페이지로 이동해 활성화하세요.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Network Activate 버튼을 클릭하세요_

add-on이 활성화되면 플랜 편집 화면의 _Advanced Options_ 메타 박스에 _Ad Settings_라는 새 옵션이 표시됩니다.

![플랜 편집 화면의 Ad Settings](/img/config/product-edit-full.png)

_이제 각 플랜의 광고 설정을 편집할 수 있습니다_

해당 탭에서 프런트엔드의 글과 페이지에 콘텐츠 전후로 광고 코드를 추가할 수 있습니다.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_광고 코드를 삽입하면 해당 특정 플랜의 글과 페이지에 표시됩니다._

해당 플랜 사용자에게 Admin Pages의 헤더에 광고가 포함되도록 할 수도 있습니다.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_사용자의 백엔드에 광고를 표시하려면 Admin Panel Ad Code 옵션으로 이동하세요_

### 광고를 넘어서 {#beyond-advertising}

Ultimate Multisite: Ad Injector를 사용하여 상위 티어 플랜에 지원을 제공할 수도 있습니다.

_Admin Panel Ads_ 옵션으로 이동하여 지원 도구의 HTML 코드를 추가하기만 하면 됩니다.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_특정 플랜에 고객 지원을 추가하세요_

이렇게 하면 사람들이 Admin Panel에서 바로 당신에게 연락할 수 있습니다!
