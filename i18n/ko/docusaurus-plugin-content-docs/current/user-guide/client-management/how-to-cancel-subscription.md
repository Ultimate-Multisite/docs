# 내 subscription을 어떻게 취소할 수 있나요?

Ultimate Multisite license가 있는 경우, 원할 때 언제든지 갱신을 취소할 수 있습니다. 아래 단계를 따라 하세요:


<!-- Screenshot unavailable: Account page email with access link -->

구매 시 제공한 이메일과 비밀번호를 사용하세요.

<!-- Screenshot unavailable: Login page with email and password fields -->

이것은 Account Page 초기 화면입니다:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

subscription을 취소하려면 왼쪽 메뉴에서 _Renewing & Billing_으로 이동하세요.

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

오른쪽의 화살표를 클릭하여 사이드 창을 여세요. 그런 다음 **auto-renew**를 취소하는 옵션을 선택해야 합니다.

시스템에서 확인 메시지를 표시합니다.

subscription을 취소하려는 것이 확실하다면 _**Cancel Renewals**_ 버튼을 클릭하세요. 이 작업 후 간단한 설문에 답하라는 요청을 받게 됩니다.

완료되었습니다! subscription은 자동으로 갱신되지 않습니다.

**subscription이 만료될 때까지 유효한 키를 보유하게 됩니다**. subscription을 다시 활성화하려는 경우 수동으로 진행해야 합니다.
