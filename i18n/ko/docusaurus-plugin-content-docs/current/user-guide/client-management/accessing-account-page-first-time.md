# 처음으로 내 Account 페이지에 어떻게 접근할 수 있나요?

Ultimate Multisite에 오신 것을 환영합니다!

저희 플러그인을 구매한 후 이메일로 받은 링크를 통해 Account 페이지에 접근하세요.

<!-- 스크린샷을 사용할 수 없음: 접근 링크가 포함된 Account 페이지 이메일 -->

구매 시 제공한 이메일과 이메일로 받은 비밀번호를 사용하세요.

<!-- 스크린샷을 사용할 수 없음: 이메일 및 비밀번호 필드가 있는 로그인 페이지 -->

이 Account에 처음 로그인하는 것이므로 비밀번호를 변경하라는 요청을 받게 됩니다. 비밀번호를 변경한 후 _저장_ 버튼을 클릭하세요.

<!-- 스크린샷을 사용할 수 없음: 저장 버튼이 있는 비밀번호 변경 양식 -->

Account 페이지에서 청구, 갱신, 취소 및 기타 여러 항목과 같은 모든 Account 세부 정보를 관리할 수 있습니다.
