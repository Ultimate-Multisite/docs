# Your Client's Websites (v2)

네트워크에 등록한 후, 고객은 대시보드에 접근할 수 있습니다.

**Account** 페이지에서는 결제, 멤버십, 도메인, 플랜 제한 등 중요한 정보를 확인할 수 있으며, **My Sites** 페이지에서도 모든 웹사이트를 보고 관리할 수 있습니다. 대시보드에서 My Sites를 클릭하면 됩니다.

![My Sites link in the customer dashboard](/img/admin/customers-list.png)

모든 웹사이트가 표시되며, 각 웹사이트 아래에 **Manage** 버튼이 있습니다. 버튼을 클릭하면 해당 웹사이트의 **dashboard** 로 리디렉션됩니다.

![My Sites page showing customer websites with Manage buttons](/img/admin/customers-list.png)

플랜에 따라 여러 웹사이트를 보유할 수 있는 경우, 어떤 웹사이트가 **primary**인지 확인할 수 있습니다.

![Account menu link in customer dashboard](/img/admin/customers-list.png)

기본 웹사이트를 변경하려면 **Account** 메뉴를 클릭해 Account 페이지로 이동합니다.

그런 다음 **Change Default Site** 를 클릭합니다.

![Change Default Site button on account page](/img/admin/customers-list.png)

팝업 창이 나타납니다. 드롭다운 목록을 클릭하고 기본 웹사이트로 만들고 싶은 웹사이트를 선택합니다.

![Change Default Site dropdown selection](/img/admin/customers-list.png)

마지막으로 **Change Default Site** 를 클릭하면 완료됩니다!

![Confirming the default site change](/img/admin/customers-list.png)
