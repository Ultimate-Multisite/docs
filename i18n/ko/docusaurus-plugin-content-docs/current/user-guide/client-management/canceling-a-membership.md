# 멤버십 취소 (v2)

_**중요한 주의: 이 문서는 Ultimate Multisite 버전 2.x를 참조합니다.**_

고객 측에서 멤버십을 취소하는 것은 꽤 쉽습니다.

사용자가 멤버십을 취소하려면 먼저 ****계정**** 페이지에 접근해야 합니다.

그곳에서 **위험 영역** 으로 이동한 뒤 **계정 삭제** 를 클릭해야 합니다.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-7IO8WYhID5.png)

모듈이 열려 확인을 요청합니다. **계정 삭제 확인** 옵션을 켜고 **계정 삭제** 를 클릭해야 합니다.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-NavCHgYJYo.png)

이렇게 하면 사이트에서 로그아웃되고 네트워크에서 멤버십이 취소되며, 사이트가 네트워크에서 삭제됩니다.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-KFkpD0CRhT.png)

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-jWkEmpf9ij.png)

**중요:** 이 작업은 고객을 네트워크에서 제외하지 않으며, 고객이 하나 이상의 멤버십을 가질 수 있거나 멤버십이 없을 수도 있음을 유의하십시오.
