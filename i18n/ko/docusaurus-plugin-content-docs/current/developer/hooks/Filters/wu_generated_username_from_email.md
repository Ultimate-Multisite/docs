# Filter: wu_generated_username_from_email

생성된 고객 사용자 이름을 필터링합니다.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $username | `string` | 생성된 사용자 이름. |
| $email | `string` | 새 고객 이메일 주소. |
| $new_user_args | `array` | 새 사용자 인수 배열, 첫 번째 및 마지막 이름이 포함될 수 있음. |
| $suffix | `string` | 사용자 이름에 추가 문자열을 붙여 고유하게 만듭니다. |

- 3.7.0
