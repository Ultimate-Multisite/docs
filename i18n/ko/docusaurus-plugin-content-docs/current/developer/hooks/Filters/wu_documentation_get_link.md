# 필터: wu_documentation_get_link

플러그인 개발자가 링크를 필터링할 수 있도록 합니다.

어떻게 유용할지는 모르겠지만, 존재하는 것이 해가 되지는 않습니다.

## 매개변수

| 이름 | 타입 | 설명 |
|------|------|------|
| $link | `string` | 등록된 링크 |
| $slug | `string` | 링크를 검색하는 데 사용되는 슬러그 |
| $default_link | `string` | 기본 등록된 링크 |

### 버전

- 1.7.0

### 소스

정의된 위치: [`inc/class-documentation.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-documentation.php#L125) 125번째 줄.
