# 필터: wu_documentation_get_link

플러그인 개발자가 링크를 필터링할 수 있도록 허용합니다.  
그것이 얼마나 유용할지는 확실하지 않지만, 갖고 있는 것이 해가 되지는 않습니다.

## 매개변수

| 이름 | 타입 | 설명 |
|------|------|------|
| `$link` | 문자열 | 링크가 등록된 것 |
| `$slug` | 문자열 | 링크를 검색하는 데 사용되는 슬러그 |
| `$default_link` | 문자열 | 기본 링크가 등록된 것 |

- 1.7.0
