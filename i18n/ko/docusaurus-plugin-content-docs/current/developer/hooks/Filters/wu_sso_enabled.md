# 필터: wu_sso_enabled

교차 도메인 싱글 사인온 기능을 활성화/비활성화합니다.

필터 이 값을 사용하여 싱글 사인온을 완전히 끄거나 조건부로 활성화합니다.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $enabled | `bool` | SSO가 활성화되어야 합니까? 켜려면 true, 끄려면 거짓-같은. |

- 2.0.11
