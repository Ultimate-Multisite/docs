#### Algorithm
```
for every test case
        read the string s
        best = 0
        current = 0
        for every character c in s
                if c is a letter or a digit
                        current += 1
                        best = max(best , current)
                else
                        current = 0
        output best
```

`current` keeps the length of the current block of allowed characters,
`best` keeps the maximum length seen so far.



--------------------------------------------------------------------

#### Correctness Proof  

We prove that the algorithm outputs the length of the longest
contiguous substring of `s` that consists only of letters and digits.

---

##### Lemma 1  
During the scan of the string, after processing a character `c`,
`current` equals the length of the longest suffix of the already processed
prefix that contains only allowed characters.

**Proof.**

*If `c` is allowed*  
`current` is increased by one, therefore it is the length of the suffix
ending at `c` that contains only allowed characters.

*If `c` is not allowed*  
The suffix ending at `c` cannot contain only allowed characters,
so the longest such suffix has length `0`.  
The algorithm sets `current = 0`. âˆ



##### Lemma 2  
After each iteration `best` equals the maximum length of an allowed
substring seen so far.

**Proof.**

`best` is updated only when a new allowed character is read.
At that moment `current` is the length of the suffix ending at this
character that contains only allowed characters (Lemma&nbsp;1).
`best` is set to `max(best,current)`, therefore it is the maximum of
its previous value and the length of the current suffix â€“ i.e. the
maximum length seen up to now. âˆ



##### Theorem  
After the whole string has been processed, `best` equals the length of
the longest contiguous substring of `s` that contains only letters and
digits.

**Proof.**

By Lemma&nbsp;2, after the last character has been processed `best`
is the maximum length of an allowed substring that appears anywhere in
`s`.  
Consequently the algorithm outputs the required length. âˆ
