# 필터: wu_stripe_existing_product_id

필터는 확인할 제품의 ID를 지정합니다. 이 ID가 존재하면 새 구독이 이 제품을 사용합니다.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $product_id | `string` | Stripe 제품을 확인할 ID. |
| $name | `string` | 제품 이름. |
