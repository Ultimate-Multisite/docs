# 필터: wu_pre_save_settings

개발자가 Ultimate Multisite에 의해 저장되기 전에 설정을 필터링할 수 있도록 허용합니다.

## 매개변수

| 이름 | 유형 | 설명 |
|------|------|-------------|
| $settings | `array` | 저장될 설정입니다. |
| $settings_to_save | `array` | 추가할 새 설정입니다. |
| $saved_settings | `array` | 현재 저장된 설정입니다. |

### 버전

- 2.0.18
