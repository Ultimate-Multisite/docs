# 필터: wu_independent_billing_cycle_product_types

독립 청구 주기를 가진 제품 유형을 필터링합니다.

독립 청구 주기를 가진 제품은 장바구니에서 플랜의 청구 기간과 일치하도록 강제되지 않습니다. 이는 도메인 등록과 같이 플랜이 월간이든 연간이든 상관없이 항상 연간 청구되는 제품에 유용합니다.
