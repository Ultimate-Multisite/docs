# Action: wu_sso_loaded

필요한 경우 플러그인 개발자가 추가 후크를 추가할 수 있도록 허용합니다.

SSO는 일출 시 실행되는 것이므로 초기화 시까지 지연해야 합니다.

## 매개변수

| Name | Type | Description |
|------|------|-------------|
| $sso | `self` | SSO 클래스입니다. |
