# Ultimate Multisite 문서

Ultimate Multisite 문서에 오신 것을 환영합니다. Ultimate Multisite는 WordPress Multisite를 완전한 Website as a Service (WaaS) 플랫폼으로 변환합니다.

## 문서 섹션

### [사용자 가이드](/user-guide/getting-started/ultimate-multisite-101)

WaaS 플랫폼을 설정, 구성 및 관리하는 데 필요한 모든 것을 제공합니다. 설치, 결제 게이트웨이, 도메인 매핑, 클라이언트 관리 등을 포함합니다.

### [개발자 문서](/developer/hooks)

Ultimate Multisite를 기반으로 개발하는 개발자를 위한 기술 참조입니다. 훅 참조, REST API 문서, 애드온 개발 가이드를 포함합니다.

### [애드온](/addons)

WooCommerce 통합, 분석, 도메인 판매 등 추가 기능을 제공하는 유료 애드온 플러그인에 대한 문서입니다.
