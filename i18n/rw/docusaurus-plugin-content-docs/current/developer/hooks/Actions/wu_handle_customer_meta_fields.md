# Igikorwa: wu_handle_customer_meta_fields

Emerera abategura plugin kubika amakuru meta mu buryo butandukanye niba babikeneye.

## Ibipimo

| Izina | Ubwoko | Ibisobanuro |
|------|------|-------------|
| $meta_repository | `array` | Urutonde rw’imyanya meta, rufite imiterere ya key =&gt; value. |
| $customer | `\Customer` | Ikintu cy’umukiriya wa Ultimate Multisite. |
| $checkout | `\Checkout` | Icyiciro cya checkout. |
