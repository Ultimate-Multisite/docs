# Igikorwa: wu_page_added

Yemerera abategura plugin gukoresha ibindi bintu igihe paji zandikwa.

Bitandukanye na wu_page_load, ikora gusa igihe paji runaka irimo kurebwa, iyi hook ikora mu kwandika buri paji ya admin yongewemo hakoreshejwe code ya Ultimate Multisite.

## Ibipimo

| Izina | Ubwoko | Ibisobanuro |
|------|------|-------------|
| $page_id | `string` | ID y’iyi paji. |
| $page_hook | `string` | Izina rya hook ry’iyi paji. |
