# Filter: wu_bypass_unset_current_user

Emerera abatunganya porogaramu kurenga code ikuraho umukoresha uriho ubu.

Gusubiza ikindi kintu cyose kitari null bizatuma ikurwaho ry’umukoresha winjiye ubu rirengwaho. Ibi bishobora kuba ingirakamaro mu bihe bimwe, urugero, igihe ukorana na site zungirije zikoreshwa nka paneli z’ubuyobozi.

## Ibipimo {#parameters}

| Izina | Ubwoko | Ibisobanuro |
|------|------|-------------|
| $null_or_bypass | `mixed` | Null kugira ngo ukomeze, ikindi kintu cyose kugira ngo ubirengeho. |
| $current_user | `false\|\WP_User` | Ikintu cy’umukoresha uriho ubu. |
