# Akayunguruzo: wu_independent_billing_cycle_product_types {#filter-wuindependentbillingcycleproducttypes}

Yungurura ubwoko bw’ibicuruzwa bifite inziga z’ubwishyu zigenga.

Ibicuruzwa bifite inziga z’ubwishyu zigenga ntibihatirwa guhuza n’igihe cy’ubwishyu cya plan muri cart. Ibi bifasha ku bicuruzwa nko kwandikisha domain bihora byishyurwa buri mwaka, hatitawe niba plan ari iya buri kwezi cyangwa iya buri mwaka.

## Ibipimo {#parameters}

| Izina | Ubwoko | Ibisobanuro |
|------|------|-------------|
| $types | `array` | Array y’ubwoko bw’ibicuruzwa bifite inziga z’ubwishyu zigenga. |
