# Muyunguruzi: wu_get_site_domain_and_path

Yemerera abategura porogaramu guhindura amatsinda ya domain/path.

Ibi bishobora kuba ingirakamaro mu bintu byinshi, nko gushyira mu bikorwa ubwoko runaka bw’igisubizo cya staging, servers zitandukanye, n’ibindi.

## Ibipimo {#parameters}

| Izina | Ubwoko | Ibisobanuro |
|------|------|-------------|
| $d | `object` | Object iriho ubu irimo imfunguzo za domain na path. |
| $path_or_subdomain | `string` | Path/subdomain y’umwimerere yoherejwe muri function. |

### Kuva {#since}

- 2.0.0
### Inkomoko {#source}

Bisobanuwe muri [`inc/functions/site.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/site.php#L235) ku murongo wa 235


## Ibisubizwa {#returns}
Object irimo imfunguzo za domain na path.
