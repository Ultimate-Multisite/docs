# Akayunguruzo: wu_magic_link_enforce_ip

Akayunguruzo niba kugenzura aderesi ya IP bigomba gukurikizwa.

Bishyire kuri false kugira ngo tokens zishobore gukora ziturutse ku miyoboro itandukanye. Ibi bigabanya umutekano ariko bikongera uburyo bwo gukoresha (urugero, ku bakoresha telefone zigendanwa bahindura imiyoboro).

## Ibipimo

| Izina | Ubwoko | Ibisobanuro |
|------|------|-------------|
| $enforce | `bool` | Niba guhuza aderesi ya IP bigomba gukurikizwa. |
