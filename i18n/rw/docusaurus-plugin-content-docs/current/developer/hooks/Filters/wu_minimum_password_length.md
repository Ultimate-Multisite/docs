# Akayunguruzo: wu_minimum_password_length {#filter-wuminimumpasswordlength}

Yungurura uburebure buke bw'ijambo ry'ibanga.

Bishyirwa mu bikorwa gusa iyo wu_enforce_password_rules ari true.

## Ibipimo {#parameters}

| Izina | Ubwoko | Ibisobanuro |
|------|------|-------------|
| $min_length | `int` | Uburebure buke bw'ijambo ry'ibanga. Mburabuzi 12 (bihuye na Defender Pro). |
| $defender_active | `bool` | Niba Defender Pro Strong Password ikora. |
