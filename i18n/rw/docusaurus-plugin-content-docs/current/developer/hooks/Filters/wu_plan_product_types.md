# Akayunguruzo: wu_plan_product_types {#filter-wuplanproducttypes}

Yungurura ubwoko bw’ibicuruzwa bugomba gufatwa nk’imigambi.

Aka kayunguruzo gatuma inyongera zandikisha ubundi bwoko bw’ibicuruzwa bugomba kumenyekana nk’imigambi mu kugenzura no mu gutandukanya.

## Ibipimo {#parameters}

| Izina | Ubwoko | Ibisobanuro |
|------|------|-------------|
| $plan_types | `array` | Array y’ubwoko bw’ibicuruzwa bwo gufata nk’imigambi. |
