# Akayunguruzo: wu_should_create_domain_record_for_site

Kayungurura niba Ultimate Multisite igomba gukora inyandiko ya domain ku rubuga rushya rwakozwe.

Koresha aka kayunguruzo kugira ngo uhagarike cyangwa wigize inyuma gukora mu buryo bwikora inyandiko za domain ku mbuga zikoresha domain shingiro isangiwe y’ifishi ya checkout, host y’imbere, cyangwa domain indi integration izacunga ukwawo.

## Ibipimo

| Izina | Ubwoko | Ibisobanuro |
|------|------|-------------|
| $create | `bool` | Niba inyandiko ya domain igomba gukorwa. |
| $site | `WP_Site` | Ikintu cy’urubuga rushya rwakozwe. |

### Kuva

- 2.13.0

### Inkomoko

Bisobanuwe muri `inc/functions/domain.php`.


## Ibisubizo

Boolean igaragaza niba inyandiko ya domain igomba gukorwa.
