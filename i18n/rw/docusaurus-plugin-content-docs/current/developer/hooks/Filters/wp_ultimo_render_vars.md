# Akayunguruzo: wp_ultimo_render_vars

Yemerera abategura imigereka kongeramo impinduragaciro y’inyongera mu miterere yo kugaragaza ku rwego rusange.

## Ibipimo

| Izina | Ubwoko | Ibisobanuro |
|------|------|-------------|
| $args | `array` | Array irimo impinduragaciro zatanzwe n’ihamagara ryo kugaragaza. |
| $view | `string` | Izina ry’icyerekanwa kigomba kugaragazwa. |
| $default_view | `string` | Izina rya fallback_view |
