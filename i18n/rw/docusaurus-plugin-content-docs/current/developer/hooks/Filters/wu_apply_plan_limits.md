# Akayunguruzo: wu_apply_plan_limits

Emerera abategura plugin guhagarika imipaka hakiri kare.

Ushobora gukoresha aka kayunguruzo kugira ngo ukore code iyo ari yo yose mbere y’uko imipaka itangizwa. Niba akayunguruzo kawe gasubije agaciro kose kafatwa nk’ukuri, inzira izakomeza; niba gasubije agaciro kose gafatwa nk’ikinyoma, code izasubira inyuma kandi nta n’imwe muri hooks zikurikira izakora.
