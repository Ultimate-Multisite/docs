# Muyunguruzo: wu_get_checkout_variables {#filter-wugetcheckoutvariables}

Emerera abategura inyongera kuyungurura amaseti yateguwe mbere y'urupapuro rwo kwishyura.

Witonde, kubura kw'imfunguzo bishobora kwangiza burundu kwishyura ku ruhande rw'imbere rugaragarira abakoresha.

## Ibipimo {#parameters}

| Izina | Ubwoko | Ibisobanuro |
|------|------|-------------|
| $variables | `array` | Impinduragaciro zahinduwe iz'aho zikoreshwa. |
| $checkout | `\Checkout` | Icyiciro cya checkout. |

### Kuva {#since}

- 2.0.0
### Inkomoko {#source}

Byasobanuwe muri [`inc/checkout/class-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout.php#L1970) ku murongo wa 1970


## Ibyo isubiza {#returns}
Array nshya y'impinduragaciro.
