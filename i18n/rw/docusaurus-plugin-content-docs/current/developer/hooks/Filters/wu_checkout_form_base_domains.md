# Muyunguruzi: wu_checkout_form_base_domains

Iyungurura base domains zisangiwe za checkout-form zitagomba guhinduka records za mapped-domain.

Koresha uyu muyunguruzi igihe integration itanga base domains z’inyongera ku myanya ya **Site URL** ya checkout form. Domains zigarurwa n’uyu muyunguruzi zifatwa nka hosts zisangiwe zo kwiyandikisha aho kuba custom domains za buri site.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $domains | `array` | Base domains zisangiwe zakusanyijwe zivuye mu configuration ya checkout form. |

### Kuva

- 2.13.0

### Inkomoko

Byasobanuwe muri `inc/functions/domain.php`.


## Igarura

Array ya base domains za checkout-form zanormalized.
