# Ultimate Multisite Inyandiko

Murakaza neza mu nyandiko za Ultimate Multisite. Ultimate Multisite ihindura WordPress Multisite ikayigira urubuga rwuzuye rwa Website as a Service (WaaS).

## Ibice by’Inyandiko {#documentation-sections}

### [Ubuyobozi bw’Ukoreshwa](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

Ibintu byose ukeneye kugira ngo ushyireho, utunganye, kandi ucunge urubuga rwawe rwa WaaS. Bikubiyemo ishyiraho, amarembo yo kwishyura, guhuza domain, gucunga abakiriya, n’ibindi.

### [Inyandiko z’Abakora Porogaramu](/developer/hooks) {#developer-documentation}

Icyerekezo cya tekiniki ku bakora porogaramu bubakira kuri Ultimate Multisite. Kirimo icyerekezo cya hooks, inyandiko za REST API, n’amabwiriza yo guteza imbere addon.

### [Inyongera](/addons) {#addons}

Inyandiko za plugins za addon zishyurwa zagura Ultimate Multisite zikongeramo ibiranga by’inyongera nko kwinjiza WooCommerce, isesengura, kugurisha domain, n’ibindi.

## Kubona Ubufasha {#getting-help}

- [Ibiganiro bya GitHub](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) -- Baza ibibazo kandi usangize ibitekerezo
- [Ibibazo bya GitHub](https://github.com/Ultimate-Multisite/ultimate-multisite/issues) -- Menyesha amakosa
- [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) -- Kuramo plugin y’ibanze
