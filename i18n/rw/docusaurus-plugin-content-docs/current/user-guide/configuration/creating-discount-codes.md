# Gukoresha Discount Codes (v2) {#creating-discount-codes-v2}

_**Urugero Muhere: Iyiwe ari mu Ultimate Multisite version 2.x.**_

Na Ultimate Multisite, uyuwe mushobora gukora discount codes kugira ngo umuntu wako yisobanure ku mibonye yawe. kandi gukora ni gukora gusa!

## Gukora no Gutera Discount Codes {#creating-and-editing-discount-codes}

Kugira ngo ukore cyangwa utere discount code, gukora **Ultimate Multisite > Discount Codes**.

![Discount codes list — empty state before any codes exist](/img/config/discount-codes-empty.png)

Hano uza kubona list y'discount codes wako wariwe gukoresheje.

Ushobora gukora **Add Discount Code** kugira ngo utore coupon shya cyangwa utera uko uyuze uburyo bwo gutera ku byiza (edit) uyuze ukwibuka kandi uza kora.

![Discount codes list with hover actions showing Edit and Delete links](/img/config/discount-codes-list-hover.png)

![Add Discount Code button in the page header](/img/config/discount-codes-add-button.png)

Uza gukoreshwa ku myemezo y'page aho uza gukora cyangwa utera coupon code yawe. Mu gihe cyacu, twakora shya.

![Discount code edit page with all sections visible](/img/config/discount-code-edit.png)

Tugereye kugira ngo twibone ibyo byo gutera ku byiza (settings) byose:

**Enter Discount Code:** Iki ni gice cy'discount code yawe. Igihe cyo si code uza kubyara umuntu wako mu gihe cyo gutera ku byiza (checkout form).

**Description:** Hano, uza kuba ufite ubushobozi bwo kuvuga mu buryo gusa ibyo coupon cyangwa se.

![Discount code name and description fields at the top of the edit page](/img/config/discount-code-description.png)

Ushobora kandi kubona discount code nk'igice cy'active (yihariye) cyangwa inactive (yitera).

![Discount code active status](/img/config/discount-code-active.png)

**Coupon code:** Hano ni hejuru uza gukora code umuntu wako azakoresha mu gihe cyo gutera ku byiza (checkout).

![Champ ya ku gukoresha coupon code aho abantu bafite mu gihe cyo gutera umukono](https://example.com/img/config/discount-code-coupon-field.png)

**Discount (Uruko):** Hano, uba mushobora gukoresha **ikoranabuhinduzi (percentage)** cyangwa **umikoresho itandukanye (fixed amount)** mu gihe cyo gutera coupon code yawe.

![Gukoresha umikoresho itandukanye cyangwa ikoranabuhinduzi mu gihe cyo gutera discount](https://example.com/img/config/discount-code-amount.png)

**Gukoreshwa ku gihe cyo gukoresha (Apply to renewals):** Niba uburyo bwa iki cyo gukoreshwa bishobora guhindurwa, coupon code irakoze gukoreshwa mu gihe cyo gutera umukono w'ibanze. Icyo gusa cyo gukoreshwa ku gihe cyose cy'ibindi byiza bitagira discount. Niba uburyo bwa iki cyo gukoreshwa bishobora guhindurwa, coupon code irakoze gukoreshwa mu gihe cyose cyo gutera umukono w'ibanze.

**Uruko ku gihe cyo gutera umukono (Setup fee discount):** Niba uburyo bwa iki cyo gukoreshwa bishobora guhindurwa, coupon code irakoze gukoresha **umikoresho itandukanye mu gihe cyo gutera umukono w'ibanze**. Niba uburyo bwa iki cyo gukoreshwa bishobora guhindurwa, uba mushobora gukoresha uruko (ikoranabuhinduzi cyangwa umikoresho itandukanye) ku gihe cyo gutera umukono w'ibanze cy'ibindi byiza.

![Uburyo bwo gukoreshwa ku gihe cyo gukoresha ku gihe cyo gutera umukono (Apply to renewals and setup fee discount toggle options)](https://example.com/img/config/discount-code-renewals.png)

**Active (Icyo kigishamo):** Uba mushobora gukemura cyangwa guhagarira ku coupon code yawe mu buryo bwa mbere.

![Icyo kigishamo kugira ngo ughagaze cyangwa ughagarire ku coupon code](https://example.com/img/config/discount-code-active.png)

Mu **Advanced Options (Uburyo bwo gukoresha ubushobozi bwa mbere)**, twari na uburyo bwa mbere:

![Icyo kigishamo cy'uburyo bwa mbere bw'uko coupon code yashobora guhindurwa](https://example.com/img/config/discount-code-advanced.png)

**Limit uses (Gukemura umikoresho):**

  * **Uses (Uko gukoreshwa):** Hano, uba mushobora kumenya ko coupon code yashobaye gukoreshwa uko gishobye.

  * **Max uses (Uko gukoresha ku gihe cyose):** Iki cyo kizabagarira uko abantu bashobye gukoresha coupon code yawe. Urugero, niba ugiye kugaragaza 10 hano, coupon code yashobaye gukoreshwa igihe cyo 10. Nyuma y'uko urugero rwa mbere rugera, coupon code irakoze gukoreshwa.

![Gukemura umikoresho ku gihe cyose n'ubwo bwo gukoresha (Limit uses setting with current uses count and max uses field)](https://example.com/img/config/discount-code-limit-uses.png)

**Uruko n'ubwumva no kwizera:** Urugereraho uburyo bwo gushyira umwirondoro (start date) n'ubwumva (expiration date) ku coupon yawe.

![Start and expiration date fields for scheduling the discount code](/img/config/discount-code-dates.png)

**Gusobanura ibintu:** Niba ugiye gukoresha **Select products**, ibintu byose bwawe bizakwibonye. Uzi umuntu wumva gushobora gukoresha (bishingira cyangwa guhagarara) uburyo bwo gukoresha coupon code uyu. Ibintu byose byagagarara hano byitera n'ubwumva, nta ngaruka zizera niba abakoresha ku coupon code uyu.

![Limit products section with per-product toggle switches](/img/config/discount-code-limit-products.png)

Nyuma yose wakoze ibyo byose, gukora **Save Discount Code** kugira ngo uburyo bwawe bwo coupon yitoke kandi byarinduke!

![Save Discount Code button at the bottom of the edit page](/img/config/discount-code-save.png)

Coupon yawe yari mu giciro ryawe, kandi ku gihe cyo, uzi umuntu wumva gukora **edit cyangwa guhagarara** (delete) kuri coupon yawe.

![Discount code row in the list with Edit and Delete hover actions](/img/config/discount-codes-list-hover.png)

### Gukoresha URL Parameters: {#using-url-parameters}

Niba wifashije gukoresha ibyo bishobora kubyara (customize) ibyo byo mu rubuga rwa ubwumva (pricing tables) cyangwa gukora uburyo bwiza bwo coupon code ku rubuga rwawe, kandi wifashije gushyira coupon code ku checkout form yawe mu gihe cyo, uzi gukora ibyo bikorwa bishobora gukoreshwa (URL parameters).

Mbere, ugomba gukora link yashobora guhagarara (shareable link) kuri plan yawe. Uko gukora iki, gaha ku **Ultimate Multisite > Products** kandi uba umuntu wumva.

Gukora **Click to Copy Shareable Link**. Ibi bituma link yashobora guhagarara ku plan uyu. Mu gihe cyacu, link yashobora guhagarara yari [**mynetworkdomain.com/register/premium/**](http://mynetworkdomain.com/register/premium/)_.

![Product page with shareable link button](/img/config/products-list.png)

Kugira uburyo bwo gukoresha coupon code yawe ku gihe cy'ibizuba by'ibyo, kugira ngo ushije parameter **?discount_code=XXX** kuri URL. Uruwa XXX ni coupon code yawe.

Mu gihe cyacu, twagira gukoresha coupon code **50OFF** ku gihe cy'ibizuba by'ibyo.

URL y'igice cy'ibizuba by'ibyo kandi yakoresheje coupon code 50OFF yitandukanye na: [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/) _**?discount_code=50OFF**_.
