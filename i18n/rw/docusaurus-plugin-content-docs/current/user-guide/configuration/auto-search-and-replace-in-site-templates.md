# Gukoresha n'Urugero mu Site Templates (v2)

_**Icyo uburyo rwa hariyo rwo gukoresha bishobora rwa WP UItimo version 2.x.**_

Icyo ni cyo gukoresha cyiza cyane cy'Ultimate Multisite ni ubushobozi bwo gushyira amakuru, uburyo, n'ibyo mu buryo bwo gukoresha (select fields) ku form y'ubwujisoro. Icyiza cyangwa twagutse ibyo amakuru byose, twashobora gukoresha ibyo kugira ngo twanyage ibintu mu gihe cy'site template y'uko twyanyije. Nyuma, iyo site yshoborwe kumenya (published), Ultimate Multisite izahindura ibyo placeholder byo mu gihe cy'ubwujisoro n'amakuru yose yakoreshwa mu gihe cy'ubwujisoro.

Urugero, uza kuba ufite site templates y'uko twanyije n'ibyo placeholder. Ibyo placeholder bifite gukoresha bishobora byahindurwa - {{placeholder_name}}. Nyuma, uza kuba ufite gushyira form y'ubwujisoro isobanura ibyo amakuru.

Umuvugizi wako azakwibona ibyo placeholder mu gihe cy'ubwujisoro.

Nyuma, Ultimate Multisite izahindura ibyo placeholder n'amakuru yose yakoreshwa mu gihe cy'ubwujisoro mu buryo bw'umwihariko.

## **Gukemura ikibazo cy'site template bifite placeholders byinshi**

Icyo cyose ni cyiza, ariko twishobora gushobora ku ikibazo cy'umwihariko: site templates zacu - zikoreshwa na abakoresha batanze - zifite ibyo placeholder byinhiso byitera ubwoba kandi bitagira inama nyinshi.

Kugira ngo twemerere uburyo bwawe, twafite uburyo bwo gushyira amakuru y'ibyo placeholder mu gihe cy'ubwujisoro (fake values), kandi twashobora gukoresha ibyo amakuru byose kugira ngo twanyage n'shindure ibyo mu site templates mu gihe abakoresha batanze bashobora kubona.

Uza kuba ufite uburyo bwo gushyira placeholder mu gihe cy'site template uburyo bwawe: yandika ku **Ultimate Multisite > Settings > Sites**, ugira ngo utangire mu igice cy'Site Template Options, kenshiya kuri link ya **Edit Placeholders**.

![Site Template Options area in the Sites settings page](/img/config/settings-sites-templates-section.png)

Icyo kizakwibona ku gihe cy'site template cyo gukoresha amakuru, aho uza kuba ufite ubushobozi bwo gushyira placeholders n'amakuru yayo.
