# Gukoresha Link za Ubwumuryo (Shareable Links)

Ultimate Multisite yifashisha uburyo bwo kwishamo ubwumuryo bw'ibintu by'ubwumuryo (plan links) ku batatuza mu gihe. Kumenya uburyo bwawe, gukora ibi, ugenda ku cyo gukoresha gukoresha plan yose kandi ukoresha ubutumwa "Click to copy the Shareable Link".

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_Ku gihe cyo gukoresha plan, uza kumenya ubutumwa bwo gukora ubwumuryo bw'ibintu by'ubwumuryo (shareable links)_

Mugisha ko uzi ko uburyo bwawe bwo gukoresha ibi bishobora gukoresha uburyo bwo gukoresha ibi by'ubwumuryo (default billing frequency) kugira ngo uburengera. Ibi by meaningko ko, niba ubutumwa bwawe bwo gukoresha ibi ni mu gihe cy'umwuka (monthly), umuntu azabona link nk'iki: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Niba wifuza ko ubwumuryo bw'ibintu by'ubwumuryo bwo gukoresha ibi ni mu gihe cy'umwuka (quarterly), uza gutera 3 ku mwera.

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Kugira ngo wibone plan y'ubwumuryo "premium" mu gihe cy'umwuka (yearly schedule), uza gutera 1 kandi ukoresha igice cy'urugero:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Mugisha ko ushobora gukoresha link imwe gusa ku gihe cy'ubwumuryo bwo gukoresha ibi by'ubwumuryo (billing frequencies) bitandukanye. Uza gutera link imwe gusa kuri buri gihe cy'ubwumuryo bwawe.

Icyiza cyane, iyo ufungura link ku batatuza mu gihe cy'ubwumuryo bwo gukoresha ibi by'ubwumuryo (shareable link), izabona ko bakora ubushobozi bw'ibanze y'uko bashobora gutangira gukoresha.
