# Amakuru ya Ultimate Multisite

Icyo ugiye kugenda mbere y'uko wumva ko hari iki cyo gukoresha plugin yetwe Ultimate Multisite? Murakoze, hari ibyerekezo byiza by'ingenzi gukoresha Ultimate Multisite: **domain name**, umuyobozi mwiza (hosting provider) n'ikoreshwa ya WordPress y'Umuvugizi (Multisite). Icyo ni cyose ukwibona!

## Domain Name {#domain-name}

Gukora uburyo bwo gukoresha domain name, ugomba kwibona iki cyo wumva mu niche yawe kandi iki cyo uza gukora n'Ultimate Multisite: Ugiye gukora agency? Gutangira ibintu byo gutya (shop-like websites)? Cyangwa wumva ko uza gukoresha hosting y'WordPress? Icyose cyari cyihariye n'Ultimate Multisite.

Plugin yetwe itanga serivisi z'subscription, gukoresha amafaranga, checkout forms, discount vouchers no kumenya abakoresha (customer communications). Uruko rwa kazi rishobora kuba rishingiye ku gukoresha, gukorera no guhagarara WordPress Multisite ni yiza cyane n'Ultimate Multisite.

Umuvugizi wumva uwo niche wumva, uza kubyaraho n'plugin yetwe kugira ngo uza gutya websites nk'umuryango (as a service)! Ugingize domain name yizera ku njira yawe yawe.

## Umuyobozi mwiza (Hosting Provider) {#hosting-provider}

Uko wumva domain name yiza itanga niche yawe kandi ikoreshwa, ugomba kwibona umuyobozi mwiza wo gukoresha network yawe n'Ultimate Multisite. Hari uburyo bwo gukora ibyo mu mwaka. Uruko rwa hosting rishobora guhagarara neza n'Ultimate Multisite, kandi iki cyari cyo kumenya cyane cyane ku buryo uza gutya iki.

Ultimate Multisite igomba gukora mu gihe cyose y'ubwuga bw'ubwugwa bwo gutegura (hosting). Icyo gusa cy'ingenzi ni ko uburyo bwo gukoresha ibyemezo by'ubwoko bw'ubwugwa bwo gutegura (domain mapping) cyari mu gihe cyose. Ariko, hari impamvu zo mu gihe cyiza nk'uko _WPMU DEV Hosting_, _WP Engine_, _Closte_, _Cloudways_, _Gridpane_ n'ibindi, hari uburyo bwo gukoresha ibi byemezo by'ubwoko bw'ubwugwa bwo gutegura (domain mapping) mu gihe cyiza. Ku gihe cyose, ushobora kugira ngo ugire ibi byemezo by'ubwoko bw'ubwugwa bwo gutegura (custom domains) mu buryo bwo gukoresha mu gihe cyiza. Ni impamvu nziza kandi ko ugeraho hosting provider wumva ubushobozi bwo AutoSSL. _Closte_, _Runcloud_ n'ibindi, hari ibyo byumva ibi.

Mu gihe cyo gutegura ibyemezo by'ubwoko bw'ubwugwa (server specifications), nta uburyo bwo gukoresha mu gihe cyose. Ibi byaba bitewe n'ibintu runaka ushaka gukora no ku muryango wawe. Uburyo bwo gutegura ibyemezo by'ubwoko bw'ubwugwa (shared hosting) ni uburyo bwiza bwo gufata umusaruro, ariko nk'uko u muryango wako uzatangira gukomeza, ushobora kugira VPS (Virtual Private Server) ikomeye.

## WordPress Multisite {#wordpress-multisite}

Ubu gihe, twitangira kuvuga ku bijyanye n'installing WordPress Multisite.

WordPress Multisite yishobora kugufasha kugira uburyo bwo gutegura ibi byemezo by'ubwoko bw'ubwugwa (sites) mu gihe cyiza cyane. Iki ni ikintu cyari kigize mu gihe cyiza, ariko ntabwo kirimo gukora mu gihe cyose. Ukenya uburyo bwo gukoresha iki kuri WordPress no ku panel ya hosting yawe.

Kubera ko WordPress yishobora gukoresha Multisite **ni umuntu w'ubwoko bw'Ultimate Multisite**. Kuko igice cyiza cyane cy'Ultimate Multisite ni gufasha abantu gutegura no gukoresha ibyemezo by'ubwoko bw'ubwugwa (websites) nk'ibyo bishobora gukora mu gihe cyiza. Icyo gusa, ni iki kigomba gukorwa kuri WordPress yihereye ku Multisite.

Dufite **ingingo ikomeye** itanga uburyo bwo gukora ibyo byose kugira ngo ushage step-by-step uko ugahindura (enable) igice cy'Multisite kuri WordPress yawe.

Nyuma y'uko wamukoreshe Multisite kuri WordPress, ni igihe cyiza kumenya Ultimate Multisite. Kugira ngo ugeraho Ultimate Multisite neza, ugomba kugira:

  * WordPress: v5.3+ (Urugero: Uburyo bwiza bwo gukora)

  * PHP: 7.4.x (Ubukoranyi buza ku > 8.0 mu gihe cyiza)

  * MySQL: v5+ (Urugero: 5.6, mu kiba ushoze ubuherereye kuri hosting provider yawe)

Iyi ni omurimo w'ubwujobozi bwo software ya Ultimate Multisite.

Mugisha mu maso ko Ultimate Multisite yashobora gukora neza na PHP 8.0, ariko twakorera ko twishobora kugenzura n'iyo na PHP 7.4.x.

Gusa, twakorera ko twishobora kuyikoresha ku **main domains** (domaini zizewe), kandi si mu subdomains cyangwa subdirectories. Yashobora gukora neza ku subdomains/subdirectories, ariko hari uburyo bwo kuba hari ibibazo byiza.

Kugira inama nyamukuru ku buryo bwo gukoresha Ultimate Multisite, **gukoresha hano**.
