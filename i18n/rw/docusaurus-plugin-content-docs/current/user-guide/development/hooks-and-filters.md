# Hooks n'Filters ya Ultimate Multisite {#ultimate-multisite-hooks-and-filters}

Ultimate Multisite yari yitwa n'uburyo bwose bwo gukora ibyerekezo (functions) no guhindura ibyemezo by'urugero (database queries). Yaba hari hooks n'filters zituma kugufasha gushyira cyangwa gushyira uburyo bwo gukora ibindi ibintu ku byemezo bya za mbere.

Yaha twakoresheye ubushobozi bwo gukunda (list) y'izo hooks n'filters kugira ngo bashobore gutuma abantu bakoresha cyane babone uburyo bwo gukora ibindi ibintu ku network yabo. Mushobora kumenya iyo list mu Github uburyo bwo gushakira link yacu hejuru:

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
