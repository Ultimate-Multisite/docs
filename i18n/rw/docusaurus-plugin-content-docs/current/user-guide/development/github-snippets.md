# Uburyo u snippet's ya Ultimate Multisite ku repository ya GitHub yetu {#how-to-use-ultimate-multisite-snippets-on-our-github-repository}

Hari ko dushobozi zikoresha code snippets muri GitHub repository zishobora kubyitwa cyane na abantu bashobora gukoresha Ultimate Multisite, bafite ibyo babaye mu gihe bafite uburyo bwo gukoresha ibintu myiza nk'ubwumvikane nko gukoresha script ya Google Analytics ku mga muryango y'ibyo bishobora kubyitwa cyangwa gukoresha meta box mu dashboard ya admin.

Igihe cyacu cyo gukoresha u gihe cyangwa aho gukoresha ibi byo.

Muri link igihugu, wumva snippets.

https://github.com/next-press/wp-ultimo-snippets/

Hari uburyo runaka 2 bwo gukoresha code

  1. Ku file ya functions.php y'umwe mu theme yawe.

  2. Must-Use Plugins (mu-plugins)

# Uburyo bwo gukoresha snippet ku file ya functions.php ya theme yawe. {#how-to-add-the-snippet-on-your-themes-functionsphp-file}

  1. Gukoresha account ya admin ya WordPress Network yawe kandi ugenda kuri Themes > Theme Editor (Bona screenshot hasi).

  2. Ku rubuga rwa "Edit Themes", kugira ngo ube umenye ko wari wumva theme yawe y'ibyo bishobora kubyitwa mu dropdown field y'ubwoko bw'umunsi ku cyerekezo cy'umunsi wawe (nko kuri screenshot hasi #3).

  3. Gukoresha file ya functions.php muri "Theme Files" kugira ngo ube umenye ko wari wumva file. Gukoresha ibyo mu gihe cyo gukoresha snippet ya Ultimate Multisite yakwiriye ku GitHub repository.

# Uburyo bwo gukora Must-Use Plugins (mu-plugins) {#how-to-create-must-use-plugins-mu-plugins}

WordPress yari yashyira uburyo rwa gukoresha ibintu myiza cyangwa "Must-Use Plugins", cyangwa mu gihe cy'ubwumvikane "mu-plugins".

I mu-plugins izi zishobora gukoreshwa mbere yose plugins zawe, kandi ntizwi kubyitwa. Mu multisite network, code muri mu-plugins izi zishobora gukoreshwa ku muryango wose ugiye.

1. Gukoresha FTP cyangwa SSH kugira ngo ube umenye igice cy'filesystem ya WordPress yawe.

2. Muho w'ibereye ya WordPress yawe, yera directory gishaka mu-plugins.

<!-- Screenshot imwezi: File manager ikamukura wp-content directory n'impamwe folder mu-plugins -->

3. Yera file PHP rurimi rw'umwihariko ku rugero wu-snippet.php kuri computer yawe, uburyo bwo gukoresha Notepad cyangwa code editor runaka.

4. Gukoresha code snippet ya Ultimate Multisite wari ufite mu GitHub repository, yera mu file uyu kandi ugusa. Ufite uburyo bwo gushyira uyu code ku mwezi w'umwihariko kugira ngo umenye ko ari mu plugin yawe (mu-plugin).
