# Uburyo Ubufasha Bwo Gutera Ubwumelera Bwa Customization ku Codeable? {#how-to-get-custom-development-help-from-codeable}

Kugera ibintu runaka cyane, twifashishije uburyo bwo gutera ubwumelera bwacu kugira ngo bishobore mu buryo twifuza. Niba ari iki ushaka, twamukunda [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), umuryango uho utagira abanyamwuga bakoresha Ultimate Multisite, aho ushobora kugera abanyamwuga bafite uburyo bwiza bwo gukora ibyo.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) ni platformi ituma abakoresha WordPress bakoresha, kandi yihagarira n'abanyamwuga bafite uburyo bwiza bwo gukora ibyo, bashobora gukora ubushobozi bwa project mu gihe runaka. Bafite uburyo bwo gukoresha abantu bakoresha (application process) kugira ngo uhereye na abanyamwuga bafite uburyo bwiza bwo gukora ibyo ushaka gukora.

### Uburyo bwo gukora project mu Codeable? {#how-to-create-a-project-in-codeable}

Mbere, gukora Codeable kandi ugukoresha button **Start a Project**.

Urugero, kugira ngo ushije project, gukoresha uburyo bwo gukora left kugira ngo **Start a Project**.

Mu "I need help with", ugushyira **Customizing** :

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

Mu "My", ugushyira **Plugin** :

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Kandi ugukoresha button **Continue**.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Urugero, ugiye gukora project na guhagarika umuntu wumva. Gukora ibyo ushobora! Shaka kugira inama nyinshi!

<!-- Screenshot unavailable: Codeable project description form -->

Gusura ibi byo mu gutera ubwoko bw'ibyo ugiye gukora (pricing).

<!-- Screenshot unavailable: Codeable project pricing questions -->

Nyuma y'ibyo, ntabwo ufite aka ho [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) account, ugukora uyu, kandi ukubone **Publish My Project** button.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

Icyiza cyane iyo wamukoresheye gukoresha [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) project, uyu platformu uzakubona abanyamwuga (developers) babone uburyo bwo gukora ibyo ushaka kandi bakwiriye ku kazi kawe.

Icyo cyo gukora cyitwa kera kugufasha kugera ku muntu wumvikane kandi wari afite uburyo bwo gukoresha WordPress n'Ultimate Multisite!

Uruko rw'Codeable rurangiramo ni $70–$120 USD ku gihe runaka. Icyo cyitwa kera kigira ubwoba bishobora guhinduka mu buryo bwo gukora umuntu, uburyo bwo gukora ibyo ushaka, kandi ukuri rw'umugambi (urgency) w'umugambi.

Icyiza iyo uza umukono w'umugambi, uza kuba ufite ubushobozi bwo gukoresha abanyamwuga bakwiriye kandi uza gushyira umugambi wawe mu kazi!
