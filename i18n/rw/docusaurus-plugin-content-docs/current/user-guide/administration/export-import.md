# Export & Import {#export--import}

Ultimate Multisite 2.9.0 yifashije **Export & Import** (Kugera no Gutera) uburyo runaka mu **Tools > Export & Import**. Ureba uyu zindiwe n'ubwo ushaka kugabanya site ya WordPress mu ZIP file, gutera ubuzima bwa ZIP, cyangwa gukoresha site hagati y'Ultimate Multisite na single-site WordPress zitera.

## Ubwumvikane (Required permissions) {#required-permissions}

Ushobora gufata umwanya nk'umuntu w'ubwujaji (administrator) ukeneye kwerekana menu ya **Tools** kuri site yifashijwe cyangwa gukera. Mu network ya multisite, ushobora gukoresha aka ntambi ry'ubwujaji bw'ubwoko (network administrator account) mu gihe utera cyangwa gutera subsites zitera z'Ultimate Multisite.

Uburyo bwo gutera ZIP download bifashijwe ku uburyo bwo gufata umwanya (authenticated download endpoint), nko mu gihe ushaka ko download yisobanura, ugukomeza gukoresha umwanya w'ubujaji (admin session) kandi ushobora kwibanda URL z'download cyangwa ukwibanda ku bantu bose.

## Gutera site mu ZIP {#exporting-a-site-to-a-zip}

1. Mu admin ya WordPress ya site ushaka gufata, gukora **Tools > Export & Import**.
2. Gukoresha ubushobozi bwo gutera kandi ugabona site ushaka kugabanya.
3. Gukoresha ibyo ushaka kuvuga ko ukeneye kwerekana (optional content) nk'ubwoko bw'uploads, plugins n'themes, mu gihe byo byo byo byo bishobora gukoreshwa.
4. Gutangira gutera kandi ugukomeza kugera ku gihe cyose. Sites zikomeye zishobora gutera mu ngufi mbere y'uko ZIP igaragara.
5. Gukoresha ZIP yisobanura mu gice cyo gutera (exports list).

Icyo mu gihe cyo gukoresha uburyo bwo gukoresha (export) sita ryiza, hari ibintu byiza kwigira ngo umuntu yitandukanye yorera.

*   Inyandiko z'ubwumvikane cyane (Very large uploads directories) cyangwa imyitobo y'amashusho (media libraries) nshobora gutera ZIP files zikomeye cyane. Twifashisha uburyo bwo gukoresha PHP upload limits, execution limits, ubwoko bw'ubwoko bw'umunsi (disk space), memory, no muvugizi w'umunsi (server timeout settings) mbere y'uko twagira gukora export cyangwa import sita ryiza.
*   Imyitobo y'amashusho byiza cyane nshobora gutera ngo bishobore guhindurwa mu gihe cyo kwizera (maintenance window) aho umuntu si ari muri sita ryiza.
*   Icyemezo cy'umunsi (target WordPress installation) cyangwa sita ryiza rishobora gukoresha WordPress, PHP, Ultimate Multisite, plugin, n'theme zikwiye.
*   Import y'umunsi wujyanye (single-site import) igira uburyo bwo guhindura sita ryiza. Iyo si uburyo bwo gutandukanya (merge tool).
*   Uburyo bwo gukoresha Multisite-to-single-site no single-site-to-multisite moves byiza gusa iyo umunsi w'umunsi wujyanye (target environment) yashobora gukoresha plugin, theme, URLs, n'ibintu byo mu bwoko bwa Ultimate Multisite byose byo mu site yashoboye guhindurwa.
*   Gukoresha ZIP cyiza ko ari cyiza (private). Iyo yashobora gutera database content, ibintu byo kwitwa (uploaded files), n'ibintu byo gukoresha (configuration details) byo mu site yashoboye guhindurwa.

Kugira uburyo bwo gukoresha export cyiza ku gihe cyo gukoresha network, wumva [Site Export](/user-guide/administration/site-export).
