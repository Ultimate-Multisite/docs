# Gukoresha no Kwiboneka n'Inyandiko

## Icyemezo cyo Guhindura Ukozi n'Inyandiko

Mbere y'uko ubatangira gukora inyandiko, ugomba guhindura ibyerekeye uko uboneza amafaranga. Gukoresha **Ultimate Multisite → Settings** kandi ukoresha tab ya **Payment**.

![Payment settings tab](/img/admin/settings-payments-top.png)

Iki ni uburyo bwo kubona uburyo bwose bw'ibyo bikorwa mu cyemezo cyo guhindura ukozi n'inyandiko:

![Payment settings full page](/img/admin/settings-payments-full.png)

### Icyemezo cyo Guhindura Amafaranga (General Payment Options)

Mu gihe cy'ibyo bikorwa mu cyemezo cyo guhindura, uza kuba ufite ubushobozi bwo gukora ibi:

- **Currency** — Amafaranga y'ibanze itandukanye yose itandukanye n'ubwoko bw'ibyo ukora.
- **Currency Position** — Uko umbwereka cyo amafaranga (kugera mbere cyangwa hagati y'amafaranga).

![Payment gateway settings](/img/admin/settings-payments-options.png)

### Icyemezo cyo Guhindura Inyandiko z'Ukozi (Payment Gateways)

Ultimate Multisite yishobora gukoresha inyandiko z'ukozi nyinshi. Uza kuba ufite ubushobozi bwo guhindura kandi ukubwira uburyo bwose ku gihe cy'ibyo bikorwa mu tab ya Payment settings.

![Payment gateway configuration](/img/admin/settings-payments-gateways.png)

Inyandiko zishobora gukoreshwa ni:

- **Stripe** — Ukozi bwa credit card bishobora gukorwa binyuze mu Stripe.
- **PayPal** — Ukozi bwa PayPal.
- **Manual** — Iyo ukenya inyandiko cyangwa gukora ukozi mu gihe cy'umwihariko (offline).

Urugero rwa inyandiko runaka rishobora gukoreshwa aho wiboneza API keys n'ibindi byemezo.

![Additional gateway settings](/img/admin/settings-payments-gateways-2.png)

Gukoresha umenyezi y'umutekano kugira ngo umbwire uburyo bwose bwo gukora (line items), ubushobozi bwa umuryango, ibyemezo mu gihe cy'umutekano n'ubwoko bw'umutekano.

## Invoices (Inyemezo)

Ultimate Multisite igaragaza inyemezo z'umutekano mu gihe uburyo bwo gukora umutekano. Ufashishijwe kugira ngo ushageza umutekano w'inyemezo n'ubwoko bw'ibyo byerekana ku Payment settings.

Uburyo bwo guhindura inyemezo (Invoice customization) bishobora gukoreshwa mu gihe cy'umutekano:

- **Izina ry'urwego rwa kompani no nyumburage** byerekana mu myemezo
- **Uburyo bw'inyemezo n'ubwoko bw'ibyo byerekana** (Invoice numbering)
- **Logo** byerekana ku cyerekezo cya inyemezo
- **Urugero rwa footer w'umutekano** (Custom footer text) kugira ngo uburyo bwo gukoresha ibyo mu gihe cy'ubwako, ibindi byemezo, cyangwa amakuru y'amajyubaga.

Kugira ngo uhindure inyemezo, gukora **Ultimate Multisite → Settings → Payment** kandi ugushaka ibyo bikurikira ku bishobora gukoreshwa mu gihe cy'umutekano.
