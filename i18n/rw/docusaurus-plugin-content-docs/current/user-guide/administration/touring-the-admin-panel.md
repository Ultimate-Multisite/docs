# Gukoresha ku Admin Panel

Ultimate Multisite yashobora kuba yakuriye kandi yashobora kuvuga mu buryo ufite uburyo bwo gukora, ariko n'ubwoko uyu mwiza cyane, yashobora kugufasha gutanga inyandiko y'ubwujgwa. Turemeze ku myemezo y'admin kugira ngo umenye aho wumva.

## Dashboard {#dashboard}

**Dashboard** ya Ultimate Multisite isobanura inyandiko z'ibintu by'ingenzi n'ubwumvikane, mu gihe cyo kubona amafaranga, ibikorwa by'ubwoko, uburyo bw'ubwujgwa bwo abanyamwuga, uburyo bwo gukoresha umuntu (visitor counts), no data y'ubwoko bw'umwihariko.

![Ultimate Multisite Dashboard bottom section](/img/admin/um-dashboard-bottom.png)

![Network Dashboard full page](/img/admin/network-dashboard-full-page.png)

## Checkout Forms (Urugero rw'Urugendo) {#checkout-forms}

**Checkout Forms** zikufasha gukora amagambo ya registration mu buryo ushaka. Ufite uburyo bwo guhindura ibintu, kugira ngo wugire custom classes n'scripts, no gukomeza uburenganzira ku buryo umuntu yashobora kwumva cyangwa akunda.

![Checkout Forms list](/img/admin/checkout-forms-list.png)

## Products (Icyiciro Cy'ibintu) {#products}

Icyiciro cy'**Products** ni hejuru uburyo wumva ibintu by'ubwoko, packages n'ibyo mwiza mu network yawe. Ufite uburyo bwo gukora amafaranga, igihe cyo gukoresha (billing frequencies), no kugabanya ibindi by'ingenzi ku gihe cy'ibintu by'ubwoko.

![Products list](/img/admin/products-list.png)

## Memberships (Uburyo bwo Gukoresha Umuntu) {#memberships}

Icyiciro cy'**Memberships** kishobora kubona uburyo bwose bwo gukoresha umuntu mu network yawe. Ufite uburyo bwo gushyira umbwise no guhindura ibyo by'ubwujgwa, nk'ibintu n'ibyo mwiza, amafaranga y'ubwujgwa, igihe cyo gukoresha (frequencies), uburyo bw'umwihariko, no imyaka y'amafaranga.

![Memberships list](/img/admin/memberships-list.png)

## Payments (Amafaranga) {#payments}

Icyo **Payments** yawe uburyo bwiza bwo kumenya uburyo bw'ubwoko bw'ibyo wari yashyiramo mu gihe cyose cy'ubwoko bw'ubwoko bw'ubwoko, kandi hari ibyemezo by'ubwoko bw'ibyo byerekana ibintu by'ingenzi n'ibyo bishobora kuba ari ibintu by'ubwoko n'amashyiramo.

![Payments list](/img/admin/payments-list.png)

## Abakoresha (Customers) {#customers}

Icyo **Abakoresha** cyerekana abantu bose bashyiramo mu gihe cyose cy'ubwoko bw'ubwoko, kandi hari ibyemezo by'ubwoko bw'ibyo byerekana amakuru y'umuntu n'email, igihe cyo kwihagarara cyo cyo kwihagarara, no link "switch to" kugira ngo umukoresha yihagarare mu subsite yayo.

![Customers list](/img/admin/customers-list.png)

## Subsites (Sites) {#sites}

Icyo **Subsites** cyerekana subsites bose bashyiramo mu gihe cyose cy'ubwoko bw'ubwoko, kandi hari site templates n'abakoresha bafite subsites. Uzi gukora ibyo byiza nk'uko uzi gukora ibintu by'ingenzi n'ibyo bishobora kuba ari ibintu by'ubwoko n'amashyiramo, plugins n'themes, no site thumbnails.

![Sites list](/img/admin/sites-list.png)

## Domains {#domains}

Icyo **Domains** cyerekana domains zikoreshwa ku subsites. Nk'umukoresha w'ubwoko bw'ubwoko (super admin), uzi gushyira cyangwa gukoresha ibi domain names kandi uzi gushobora kubona DNS records n'amashyiramo.

![Domains list](/img/admin/domains-list.png)

## Discount Codes {#discount-codes}

**Discount Codes** bishobora kugufasha gukora coupon codes kugira ngo wumukoreshe ubwoko bw'ubwoko ku muntu wese. Uzi gukoresha igice cyo discount kandi uzi gushyira umuntu cyangwa amahamya mu gihe cyose.

![Discount Codes list](/img/admin/discount-codes-list.png)

## Broadcasts {#broadcasts}

**Broadcasts** ni gikorwa cyo gufasha kwerekana amakuru cyangwa gukoresha uburyo bwo gukoresha ibi message by'ubwoko ku abakoresha b'ubwoko bw'ubwoko. Uzi gushyira amahamya mu gihe cyose aho umuntu yashyiramo plan cyangwa product yashobora kuba ariyo.

![Broadcasts list](/img/admin/broadcasts-list.png)

## Icyemezo (Settings) {#settings}

Icyo **Icyemezo** niho uzi gukoresha Ultimate Multisite — ibyemezo by'ubwoko bw'ubwoko, ibyemezo by'ubwoko bw'ibyo byerekana amakuru y'umuntu n'email, API na webhooks, uburyo bwo gushyira domain mapping, no ibindi by'ubwoko.

Urugero mu gukoresha (AI connector settings) uburyo bwo guhindura OAuth provider pools zikoreshwa cyane: Anthropic Max, OpenAI ChatGPT/Codex, na Google AI Pro. Icyo cyo gukoresha kintu kigomba super admins kugira ngo bashobore gutanga accounts, gufasha accounts zishobora kwibonja (refresh), gufasha gukoresha emails byo mu gihe cy'umwihariko (remove accounts by email), no gukoresha OAuth fallback w'umwihariko iyo ingano y'umwihariko (sandboxed environment) igabanya redirect ya browser. Uburyo bwo gukoresha Cursor Pro bwa mu admin panel bwa yose yari yagizeye.

Accounts za ChatGPT/Codex zishobora gukoresha uburyo bwo gukoresha tools byo gutanga (connector-backed tool use) aho igikorwa cy'umuntu kigaragaza ko hari tools, kandi ibyo mu gihe cy'umwihariko (admin workflows) cyari yashobora gukoresha Codex tool behavior iyo account ya OpenAI yari yagizeye.

## Events {#events}

Uburyo bwa **Events** bishobora gukoresha gukoresha ibyo byose by'umwihariko (events) n'ibyo byose by'umwihariko mu rwego rwawe. Bishobora kwigize ibyerekosho nk'ibyo byo guhindura plan, gufasha abantu bafite aka kanya (signups), no ibindi bikoresho — ni ikindi cyiza kugira ngo umbwire ibyo byose mu multisite yawe.

![Events list](/img/admin/events-list.png)

## Webhooks {#webhooks}

**Webhooks** Bishobora kugufasha gukoresha data ku ngingo itandukanye. Ni ikindi cyiza kugira ngo ufungura data ku ntego zikoreshwa nk'Zapier.

![Webhooks list](/img/admin/webhooks-list.png)
