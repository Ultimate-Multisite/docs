# Gutangiro mu Ubwumuryango (Memberships) {#managing-memberships}

Mu Ultimate Multisite, uzi gukemura ubwumuryango n'ubwumuryango bwose bwa subscriptions ku rugendo rwawe. Uvugera kandi utera uburyo bwo gukoresha ibyerekezo by'ubwumuryango (subscription details) nk'ibyo bikurikira: amakuru y'ubwumuryango n'ibintu by'ubucuruzi (plans and products), nyumbura ya rugendo, igice cyo kwishyura no gihe cyo kwishyura, uburyo bwo gukoresha imi, no igihe cyo gushyura.

Kugira ngo ugerageze uyu ukuri, ugenda ku tab **Memberships** mu menu ya Ultimate Multisite.

![Memberships list page](/img/admin/memberships-list.png)

Ku myemezo y'Memberships, uza kumenya umwanzuro waho n'ibintu by'ubucuruzi byahinduye na akazi kawe. Uzi gukoresha ibi bikurikira kugira ngo utere ibyo ukeneye: status (active, pending, on hold, expired, cyangwa cancelled).

Ubwumuryango bwa mbere (Free memberships) bishobora guhindurwa mu bwumuryango **bwa igihe cyose** (lifetime memberships). Bishobanura ko bisanzwe bitagira igihe cyo gukemura kandi bitagahinduka mu gihe cyo kwihindura. Niba uburyo bwa mbere bwa rugendo rwawe bishobora guhagarara nyuma y'trial cyangwa igihe cyiza, usa koreshe uburyo bwo kwishyura (paid product) bifite trial, ugire umwanzuro waho, cyangwa ugerageze umuntu ku bwumuryango buvugwa mu gihe cyo gukora ibyo ukeneye.

Gukoresha ubwumuryango runaka (Click on a membership) kugira ngo utere ukuri w'ubwumuryango — amakuru y'ubwumuryango, igice cyo kwihindura ry'site cyangwa gukoresha ibyo bishobora kubyara.

![Membership edit interface](/img/admin/membership-edit.png)

Uzi gushaka uburyo bwose bwo gukemura ubwumuryango (full view of the membership edit page):

![Membership edit full interface](/img/admin/membership-edit-full.png)

Ushobora kandi kugira ibyo by'ubwumuryango bishya bikurikira abantu bahagize mu rugendo rwawe. Ugerageze **Add Membership** kandi uburyo bwo gukoresha ibyo bishobora kubyara.

![Add Membership modal](/img/admin/membership-add-modal.png)

Kandi ugenda gukora **Create Membership**. Ubwumuryango buhuye ku myemezo y'Memberships.
