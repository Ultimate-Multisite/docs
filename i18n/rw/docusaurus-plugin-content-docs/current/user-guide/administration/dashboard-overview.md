# Uburyo mu Dashboard

Ultimate Multisite yashobora uba na **Dashboard** ikomeye kandi ku menu ya Ultimate Multisite, kandi yashobora guhuza widgets ku Network Dashboard. Iyi ndokuriye itangira ibyo wumva muri ubwonko bwo mu gihe cyose.

## Network Dashboard

Ultimate Multisite yugira widgets zishobora gukoreshwa mu dashboard ya network (ubwoko bw'Umuco) ifite ku **My Sites → Network Admin → Dashboard**.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

Iyi ni ibyo byo widget cyose bishobora kugufasha:

**First Steps:** Iri gice rishobora kubona mu gihe cy'umukoresha wari yashyiraho, kandi kigaragaza ko ugomba gukemura ibikorwa by'ubwujgwa nk'uko uba umukoresha wamashobora gukorera (configure) uburyo bwo gukoresha (payment method) no gukora umukoresha w'igikorwa (test account).

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Summary:** Iri gice rishobora kumenya ko hari abagize uburyo bwo gukoresha no igera ry'umwuka ku gihe cyose. MRR (Monthly Recurring Revenue) yashobora guhindura uko umuntu ashobora gutanga igera mu gihe cyose n'abantu bashobora gutanga igera.

![Summary widget](/img/admin/network-dashboard-summary.png)

**Activity Stream:** Iri gice rishobora kumenya ibyo byagye mu Ultimate Multisite — abagize uburyo bwo gukoresha, abagize uburyo bwo gufata (cancellations), ugukoresha siti, ugusubiza (deletions), no ibindi bikorwa byose.

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** Iri gice rishobora kumenya uburyo bwo abagize uburyo bwo gukoresha n'abandi siti mu network kandi kigaragaza shortcuts kugira ngo ugukore umunsi w'isi (create new sites) cyangwa umuntu.

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** Iri gice rishobora gutanga no gushyira mu buryo bwo kubona iby'ingenzi by'Ultimate Multisite, bikurikira uburyo bw'ubwujgwa (updates), ibibazo by'ingenzi (critical bugs), n'ibikorwa by'ubwujgwa (security fixes).

![News widget](/img/admin/network-dashboard-news.png)

## Ultimate Multisite Dashboard

Ntabwo Network Dashboard yishobora gukoresha uburyo bwo kubona ibyo byose mu network, **Ultimate Multisite dashboard** (ikiye ku cyiciro cy'umutwe w'Ultimate Multisite menu) yishobora gukoresha iby'ingenzi by'ubwujgwa bw'ikoranabuhanga.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Urugero rw'Urugero rwaMuryango (Monthly Recurring Revenue Growth):** Icyo cyonyi uburyo bw'umuryango w'ubwuga mu gihe runaka 12. Urugero rwaGukoreshwa rishobora kugaragaza urugero rwaGukoreshwa mu gihe runaka 12. Icyo cyangwa umuryango w'ibyo gushyiramo cyangwa gukoreshwa cyashobora kugaragazwa mu rwego rwaMuryango (blue) kandi ibyo byashobora guhagarwa mu rwego rwaKugabanya (red).

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**Uburyo bwo gukoresha itariki (Date-range Support):** Urugero rwaMuryango rishobora guhindura uburyo bw'ubwuga bwa dashboard cyangwa gukoreshwa mu gihe runaka kenshi kugira ngo ushije ku byo uvuga. Icyo cyangwa uburyo bwo itariki bishobora gutera ibyo bishobora guhonwa mu widgets zifite ku mwanya.

![Date range and dashboard stats](/img/admin/um-dashboard-date-range.png)

Iwidgets za dashboard zishobora gukoreshwa:

- **Urugero (Revenue)** — Urugero rwose mu gihe runaka n'ubwuga bwa MRR kandi uburyo bw'ibyo byashyiramo ku gukoresha ibyo bishobora guhonwa mu mwanya.
- **Abajyanye Bya Gukoreshwa (New Memberships)** — Icyo cyangwa abajyanye bafite ubwoko bwo gukoreshwa mu gihe runaka kenshi.
- **Ubugingo Byaho mu Buri Kurwanda (Signups by Countries)** — Uburyo bw'ibyo bishobora guhonwa ku kurwanda mu gihe runaka kenshi.
- **Abajyanye ByaForms (Signups by Forms)** — Icyangwa abakoresha bafite uburyo bwo gukoreshwa mu gihe runaka kenshi cyangwa abashyiramo bafite mu gihe runaka kenshi.
- **Ibyo Bishobora Gukoreshwa Cyera (Most Visited Sites)** — Ibyo byishobora guhonwa ku kurwanda mu gikorwa cyiza mu rwego rwaMuryango (bifasha gukoresha uburyo bwo gukoresha ibyo bishobora guhonwa mu mwanya). (Bishobora gutera ubushobozi bwo gukoresha ibyo bishobora guhonwa ku kurwanda mu gikorwa cyiza mu rwego rwaMuryango kwa gukoresha uburyo bwo gukoresha ibyo bishobora guhonwa mu mwanya).

![Dashboard statistics](/img/admin/um-dashboard-stats.png)
