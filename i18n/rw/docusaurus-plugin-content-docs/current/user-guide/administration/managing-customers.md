# Kurera Umucozi (Managing Customers) {#managing-customers}

Icyo **Customers** page igaragaza umwihariko wamwe naba yashyirwe mu gukoresha uburyo bwawe. Icyo cyo kigaragaza amakuru y'umuntu ku rugero, nk'izina ry'umuntu, email, igihe cyo kwihagarara (last login), no link **gukora** (switch to) w'uko urashobora gukoresha uburyo bwawe mu subsite ya umuntu nk'umukoresha w'ubwoko bw'uburyo.

![Customers list page](/img/admin/customers-list.png)

Gukoresha **Manage** ku cyagaye cyo umuntu kugira ngo ushije amakuru y'ingenzi kuri uwo muntu.

Ku cyagaye cyo umuntu, igice cyo **At a Glance** kigaragaza ID ya umuntu, igihe cyo kwihagarara (last login time), no nyuma y'amafaranga yashyize uwo muntu.

Igice cyo **Customer Options** kugaragaza ko ushobora gukora uwo muntu ari **VIP** (tag ikoresho kugira ngo umenye mu buryo bwawe). Igice cyo **Billing Info** kigaragaza amakuru y'ubwujisoro nk'email, igikorwa, no guherero.

Igice cyo **Fields** kigaragaza izina ry'umuntu ku gukoresha uwo muntu, email, kandi kugaragaza ko ushobora gukora ubwoko bw'ubwujisoro (Email Verification status).

Ku gice cyo Customer Options ukwira uburyo bwawe, uza kubona igice runaka rwa nyuma:

- **Memberships** — Ubu bujya bwo umuntu yashyize
- **Payments** — Icyemezo cy'umuntu ku byiza (payment history)
- **Sites** — Ibyo subsite umuntu yashyize
- **Events** — Ibyo bikorwa kuri uwo muntu (ubwoko bw'ubujisoro, guherero mu bujya bwo umujisoro, n'ibindi)

Gukoresha igice cyo arrow (→) kera ku gihe cyo ibintu byose kugira ngo ushije kuri cyagaye cyo uwo muntu. Urugero, gukoresha igice cyo arrow ku gihe cyo subsite kugaragaza cyo kwandikiza ubujisoro bwa umubujisoro, uburyo bw'uburyo, no ko ari mu gihe cy'ubwoko bw'uburyo.

Icyo gishobora gukorwa ku Memberships n'Payments — byose bifite link kera kuri cyagaye cyo ibintu byose ushobora guhindura ubujisoro bwo umujisoro, uwo muntu wari mu bujya bwo umuntu, ibintu byiza (products), uburyo bw'ubwujisoro (payment gateway settings), kandi ntabwo gukora guherero.

Ku gice cyo umuntu, uza kubona:

- **Urugero rw'Urugero n'IPs** — Icyerekezo ry'uko uyu muganga yashyirwe, IP address, noho mu gihugu.
- **Timestamps** — Icyerekezo cyo kugaragaza igihe cyo gukoreshwa ry'umuganga n'igihe cyo guhindurwa cyo kwibonye.
- **Gusubiza (Delete)** — Uburyo bwo gusubiza umuganga mu buryo bw'ubunyangamugayo.

:::warning
Gusubiza umuganga ni gikorwa cy'ubunyangamugayo kandi ntashobora guhindurwa.
:::
