# Icyo muhozo w'Abakoresha bawe (v2) {#your-clients-websites-v2}

Nyuma y'uko wariye gushyira kuri network yawe, abakoresha bvawe bashobora kugera ku dashboard yabo.

Ntabwo ku **Account** page, aho bashobora kubona impamvu z'ingenzi za ubuzuzo bwawe (payments), uburyo bwo gukoresha (memberships), domain, uburyo bwo kwizera (plan limitations) n'ibindi byinshi, bashobora guhererwa ku **My Sites** page, aho bashobora kubona no gutera imbere yose zawe z'websites. Nyaho gukora ni gukoresha My Sites kuri dashboard yawe.

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

Bashobora kubona yose zawe websites, kandi hari **Manage** button kera ku mwe muzo. Niba bashobora gukoresha uyu button, bishobora guhererwa ku **dashboard ya website y'ibyo**.

Bashobora kandi kubona uburyo w'website yabo wari **primary** (umwe w'ingenzi) kuri plans zikoresha ibyo bashobora gukoresha websites nyinshi.

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

Iyi ni umwujiza wose w'uburyo bwo gutera website:

![Site edit full interface](/img/admin/site-edit-full.png)

Niba bashaka guhindura website yabo yari primary, bashobora gukora ibi bikurikira ku Account page uburyo bwo gukoresha **Account** menu.

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

Bashobora kandi kubona no gutera ubwoko bw'abakoresha:

![Customer edit interface](/img/admin/customer-edit.png)

Nyuma, bashobora gukora **Change Default Site**.

![Change Default Site button on account page](/img/account-page/change-default-site.png)

Umwujiza (pop-up window) uzaboneka. Gukoresha drop-down list kandi ugereye ku website wifuza gukora primary website.

![Change Default Site dropdown selection](/img/account-page/change-default-site-dropdown.png)

Nyuma, gukora **Change Default Site** kandi wariwe!

![Gukoresha uburenganzira umugambi w'umwebwe](/img/account-page/change-default-site-confirm.png)
