# Gukira Umuco (v2)

_**Urugero Muhere: Icyo uburyo rwo rugero rwo Ultimate Multisite version 2.x.**_

Gukira umuhere ku muryango wanyu ni gukora gukora mu buryo bwiza kandi buri gihe.

Nta ko umuntu yashaka gukora umuhere we, ugiye gukoresha **Account** page yawe mbere.

Mu gihe runzi rwo, yaba ari ngo yagire ku **Danger Zone**, kandi yagire mu gukora **Delete Account**.

![Account page Danger Zone with Delete Account button](/img/account-page/danger-zone.png)

Urugero (module) uzahindura uburyo bwo kumenya n'ubwujgwa. Yaba ari ngo yagire ku **Confirm Account Deletion**, kandi yagire mu gukora **Delete Account**.

![Confirm Account Deletion dialog](/img/account-page/delete-account-confirm.png)

Icyo kizabasha kwitera umuntu mu site n'ugukira umuhere we mu muryango wanyu, kandi cyangwa se guhagarara site yayo mu muryango wanyu.

![Logged out after account deletion](/img/account-page/logged-out-after-delete.png)

![Membership canceled confirmation](/img/account-page/membership-canceled.png)

**Urugero:** Urugero uyu, nta kintu kizabasha kwitera umukoresha mu muryango wanyu, cyane cyane ko umukoresha ashobora kuba afite umuhere uburyo runaka cyangwa umuhere ntabwo afite umuhere mu gukoresha.

![Customer still exists in the network after cancellation](/img/admin/customer-after-cancel.png)

![Customer with no active memberships](/img/admin/memberships-customer-empty.png)
