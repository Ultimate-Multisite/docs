# Uburyo nshobora kugera ku ipage yawe y'ubwoko mu gihe cyo gishobora kuba cyo gishobora (Ultimate Multisite)?

Murakoze ku Ultimate Multisite!

Gera uburyo bwo kwiga ipage yawe y'ubwoko mu gihe cyo gishobora kuba cyo gishobora (account page) binyuze mu link wari ukwibonye mu email nyuma yo kugura plugin yetu:

<!-- Screenshot unavailable: Account page email with access link -->

Gukoresha email wari ukwibonye mu gihe cyo gukura n'ubwoko (purchase) kandi uburyo bwo kwiga (password) wari ukwibonye mu email.

<!-- Screenshot unavailable: Login page with email and password fields -->

Kuko iki ni igihe cyo mbere ugiye kwiga ku gihe cyo gishobora kuba cyo gishobora (account), uza kubazwamo kugira ngo ubone uburyo bwo guhindura uburyo bwo kwiga (password). Hindura kandi ugukoresha buto _Save_ (Gukoresha buto) button.

<!-- Screenshot unavailable: Change password form with Save button -->

Ku ipage yawe y'ubwoko, uza kuba ufite ubushobozi bwo gukora ibindi byose ku gihe cyo gishobora kuba cyo gishobora (manage) ibyo bintu byose by'ubwoko n'ibindi, nk'uko byerekana ibyiza (billings), guhindura uburyo bwo kwiga (renewals), guhindura ku gihe cyo gukoresha (cancellations), kandi ibindi.
