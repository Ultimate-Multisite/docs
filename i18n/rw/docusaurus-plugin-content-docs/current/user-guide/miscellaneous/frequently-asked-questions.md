# Urugerwa ku Kinyarwanda {#frequently-asked-questions}

Iki ni uburyo bwo gukoresha Ultimate Multisite mu WordPress.

* **Ni iki kintu Ultimate Multisite?**

Ultimate Multisite ni plugin ya WordPress, yashyizwe ku mpamvu ya WordPress multisite, ituma ubushobozi bwa WordPress runa mu gice cy'umuryango w'ibyo (network) by'imana – nk'uko [WordPress.com](https://WordPress.com) – ituma abakoresha bashobora gukora siti binyuze mu gihe cy'umwuka, imyaka, cyangwa imyaka myiza (mwezi, imyaka, cyangwa imyaka). (Ushobora kandi gukora Free plans).

* **Yego Ultimate Multisite yorera nta Multisite?**

Oya. Ultimate Multisite yorera WordPress multisite, subdirectory, cyangwa subdomain.

* **Ni iki gikorwa mu gihe cy'umwihariko (requirements) ku Ultimate Multisite?**

Ultimate Multisite yorera ibi:

**WordPress Version** : v5.3+ (Urugerwa: Urugerwa rwiza rishobora kuba ari uwo wiza kandi wari wese).
**PHP Version** : 7.4.x (Ushobora kubyara ku > 8.0 mu gihe cy'umwihariko).
**MySQL Version** : v5+ (Urugerwa: Urugerwa rwiza rishobora kuba ari uwo wiza, mu gihe ushaka uburyo bwo gukoresha 8.0 ku hosting yawe).

_Multisite (subdomain cyangwa subdirectory) kandi yorera guhindurwa._ [Uburyo bwo gukoresha WordPress Multisite](/user-guide/getting-started/how-to-install-wordpress-multisite).

* **Yego Ultimate Multisite yorera n'ubushobozi bwa Multisite bishobora kuba byari mu subdirectory?**

Yego. Ultimate Multisite yorera ubushobozi bwa subdirectory kandi cyangwa subdomain.

* **Yego Ultimate Multisite yorera guhindurwa n'ibyo abantu bashobora gukoresha Plugins na Themes?**

Yego. N'Ultimate Multisite, uza kubona ibyo plugins na themes byari byari mu gihe cy'umwihariko (plan) ku muryango wanyu.

* **Yego Ultimate Multisite yorera guhindurwa n'ubushobozi bwo gukora posts abantu bashobora gukora kuri siti ryavuga?**

Yego. Na Ultimate Multisite uburyo mu gukoresha post quotas ku **ukoza kwa post type cyose** muri network yawe. Umoze kandi kugabanya nyumbura z'abantu (users) zikora admin yashobora guhagarira, **ubwo bw'umwihariko bw'ibanga by'disk space**, n'ibindi.

  * **Ultimate Multisite yishobora gukoresha Site Templates?**

Yego. Na Ultimate Multisite uburyo bwo gukora Site Templates cyose uhoze. Site Templates si themes, ariko ni sites zikoreshwa muri network yawe. Umoze gukoresha plugins, themes, kandi ugabanya (customize) Site Template iyo umuntu yashobora gukora uko yashobora. Iyo umuntu yashobora gukoresha iyo template, site yayo izakoreshwa nk'umukoresha w'iyo template, kandi imyitwarire yose itangwa mu gihe cyose.

Umoze kandi kugabanya template ikirimo uburyo bwo gukoresha plan ikirimo. Binyuze muri iki, umuntu wese uzashobora gukoresha iyo plan izakoreshwa nk'umukoresha w'iyo Site Template.

  * **Ultimate Multisite yishobora gutanga (integrate) na WooCommerce?**

Yego!

Iyo wumva ko abantu b'network yawe bashobora gukoresha WooCommerce, kugira ngo bashobore gukora stores, ibyo hari uburyo bwo gukora binyuze muri Ultimate Multisite. Umoze kandi kugabanya nyumbura z'ibintu (products) abagize umuntu azashobora gukora ku gihe cyose cy'plan ikirimo.

Mu gihe cyo kureba ibyiza by'ubwumvikane bwo Ultimate Multisite, uburyo bwo gukoresha [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) uburyo bwo guhagarira abantu b'network yawe bashobora gukoresha payment gateways zikoreshwa muri main site yawe. Iyo, uburyo bwo gukoresha ibyo byose by'ubwumvikane bwo WooCommerce cyangwa ibindi byiza kandi wamuryera umuntu abagize Ultimate Multisite.

  * **Hari uburyo bwo guhindura (migrate) subscriptions zikoreshwa muri WPMUDEV Pro Sites?**

Ultimate Multisite uyuza uwe [ikintu cyiza kumenya uko wumva gukora migration](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) mu Pro Sites network yawe.

  * **Kugira ngo "Allowed Plugins" tab yari itabara ku Page Edit ya Plan yanjye?**

Ultimate Multisite igaragaza gukora uburyo bwo kwishyura cyangwa guhagarara umwanya ku plugins zitagira akazi mu network yawe. Iyo plugin ikoreshwa mu network, ibyo bikorwa byiza kandi byiteraho mu gihe cyose kuri site yose mu network yawe.

Niba usahagurira plugins mu "Allowed Plugins" tab, ugomba gushyira umwujaji kumenya n'uko plugin yose ikoreshwa mu network cyangwa nta cyo.

  * **Inshuti yanjye ntabwo nshobora gukoresha uburyo bwanjye bwo gukoresha Pricing Table?**

Yego. Kujye ku version 1.4.0, Ultimate Multisite igaragaza gukoresha template overrides (gukoresha ubwoko bw'template).

  * **Ultimate Multisite yishobora gukora n'WP Engine?**

Yego. Ultimate Multisite yari ikoreshwa mu gihe cyose n'WP Engine, ariko kugira ngo uburyo bwo gukoresha domain mapping (gukoresha ibintu byo gukoresha domain) ryashobora gutuma umuntu atagira inshuti zikomeye ku admin (gukoresha domain mu panel ya WP Engine). **Icyo cyari cyiteraho mu gihe cyose n'version 1.6.0.** Ultimate Multisite yishobora gukoreshwa neza na API ya WP Engine kandi domain zitagira gutangwa mu panel cyangwa bishobora guhagarara mu gihe cyose iyo uyarashyiraho (mapping).

  * **Ushobora kumenya uburyo bwo gukoresha hosting w'ibyo?**

Ultimate Multisite yishobora gukora n'hosting yose. Icyo gusa ni ko hari impamvu zo mu gihe cyose nk'Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane kandi [Runcloud.io](http://Runcloud.io) zishobora gukoresha neza cyane ku domain mapping, bityo zikora ibyo mu gihe cyose (out of the box). Ku gihe cyose cy'hosting, ushobora kugira ngo uburyo bwo gukoresha domain mapping ugomba gushyira umwujaji kumenya.
