# Uburyo ku gukoresha Domain Mapping (v2)

_**Urugero WAMBWE: Iyiho kuri uyuwe yari yitangira Ultimate Multisite version 2.x.**_

Icyemezo cyiza cyane cy'umwihariko mu rugendo rwa premium ni ubushobozi bwo gukoresha umuntu wacu ko atangire domain top-level ku siti zacyo. Urugero, uburyo bishobora kuba bishobora kuba byiza: [_**joesbikeshop.yournetwork.com**_](http://joesbikeshop.yournetwork.com) cyangwa [_**joesbikeshop.com**_](http://joesbikeshop.com)? Iyo ni iki kigomba kubaho mu mwanya: Ultimate Multisite yifashishije uburyo bwa mbere, nta bufite uburyo bwo gukoresha plugins z'ibindi.

## Uburyo bwo gukora domain mapping ni iki?

Nko cyo itangira, domain mapping ni ubushobozi bishobora kubyara Ultimate Multisite kugira ngo yashobore gutanga umusanzu w'umuntu wacu ko atangire domain y'umwihariko kandi yifashije uwo siti mu network ku gihe cyo.

### Uburyo bwo gukora domain mapping kuri Ultimate Multisite Network yawe

Domain mapping ikoresha ibyemezo byiza ku mwezi wowe kugira ngo yitandire. Umwanya, Ultimate Multisite yifashishije uburyo bw'ubwujumbura (automation) kugira ngo umenye ibyo ugomba gukora mu buryo mwiza.

Mu gihe cyo ushyira Ultimate Multisite, igice cy'umwihariko (wizard) kigomba gukoresha **sunrise.php** kandi kuguherera ku folder itangwa. **Uwo mwezi wowe utagira ubushobozi bwo kwiga mu gihe cyo uza gukora ibyo.**

<!-- Screenshot unavailable: Ultimate Multisite installation wizard with sunrise.php step -->

Ibi byera ko iyo igice cy'umwihariko cy'Ultimate Multisite yisigye gukemura ibyemezo mu network yawe, uza kubona ubushobozi bwo gukora domain y'umwihariko mu gihe cyo.

Ushobora kwerekana ko domain mapping kuri Ultimate Multisite si ikigomba koreshwa. Ufite uburyo bwo gukoresha ifungurura (native) ya WordPress Multisite cyangwa uburyo bw'ibindi bwo gukora domain mapping.

Iyo ugukunda ukuthi kufanele ubuze ukuthi kufanele ubisuse (disable) ukungena isihloko (domain mapping) seUltimate Multisite ukuze uba ngaphansi kwizindlela zinaqondile zokugena isihloko ezine. Ungakwazi ukususa le feature ngaphansi kwe **Ultimate Multisite > Settings > Domain Mapping**.

![Domain Mapping settings page showing admin redirect, mapping message and DNS options](/img/config/domain-mapping-settings.png)

Le feature iyenza ukuthi ube ngaphansi kwezinto ezifanele noma chaqondile ukuthi abantu bakwazi ukufinyelela i-admin dashboard yabo kuwo onamodini wabo (custom domain) kanye ne-subdomain noma kuwo onamodini owodwa kuphela.

Uma ubhale **Force redirect to mapped domain**, abantu bakwazi ukufinyelela i-admin dashboard yabo kuwo onamodini wabo kuphela.

**Maso:** Ibi ni maso ariko uyu domain ariko. Iyo wero, iyo uza koza mu gihe, yaba ariko kuva **Checking DNS**. **Is Secure?:** Icyo ni cyo gukora, ntabwo Ultimate Multisite izakoresha kugira ngo ibone niba domain ariko uzi SSL cyangwa atari mbere y'ibyo kuko ikoreshwa, ushobora gukora umusaruro wowe kugira ngo ushije niba uza gukoresha domain n'ubwo uzi SSL cyangwa utari. ### Gukoresha domain y'ibanze nk'subsite user

Abakoresha bwa subsite bashobora kandi gukoresha domain z'ibanze mu dashboard ya subsite admin.

Cyane cyane, ugomba kwiga ko uza gukoresha uyu option ku gihe cyo gukora **Domain mapping**. Ubona ishyirwaho hano.

### Kuza ku bijyanye n'Ubwumvikane bwo Domain Syncing

Domain Syncing ni ikoranabuhinduzi aho Ultimate Multisite yashobora gushyira umwanya wa domain name ya custom mu hosting account yawe nk'ubwumvikane bw'ibindi domain (add-on domain) **kugira ngo uburyo bwo gukoresha domain mapping bishobore gukora**.

Domain syncing yitera ku buryo bwa mbere iyo umuhanga w'ubusite (hosting provider) yashobora gushyira uburenganzira n'igikorwa cyo domain mapping ya Ultimate Multisite. Iyo, ibi hosting providers ni _Runcloud, Closte, WP Engine, Gridpane, WPMU Dev, Cloudways,_ kandi _Cpanel._

Iyo uburyo bwo gushyira umuhanga w'ubusite (host-provider integration) rishobora kuba ari cyiza, Ultimate Multisite yashobora kandi guhindura igikorwa cyo DNS cyangwa subdomain ku gihe cyo gukoresha ibindi sites. Nta igikorwa cyo background (background job) cyitera iyo nta uburyo bwo gushyira umuhanga w'ubusite (integration) rishobora kubona. Iyo, igikorwa cyo DNS n'SSL ku domain names zikoreshwa bishobora kwizera mu gihe cyo gukora ibindi by'ibindi.

Ushobora kugabanya uburyo bwo gukoresha ubutumvikande (integration) kuri Ultimate Multisite mu gihe cy'Integration'.

![Integrations tab in Ultimate Multisite settings showing hosting providers](/img/config/integrations-tab.png)

<!-- Screenshot unavailable: Hosting provider Configuration links on the Integrations settings tab -->

_Ushobora gukunda ko uzi ko ntabwo mu gihe cy'Integration' ubutumvikande bwa hosting wako, **ugomba gukora ubwumvikane (sync) cyangwa kugabanya izina ry'domain** kuri account yawe y'hosting mu buryo bw'ibindi byo.
