# Gukoresha Icyerekezo cyo Kinyarwanda

Icyerekezo cy'uwo ni cyari gishobora umbwireye n'ubwoko bw'**Language Selector** (addon).

Kugira uburyo bwo gusoma inyandiko zikomeye, wumva ku [Language Selector addon documentation](/addons/language-selector).
