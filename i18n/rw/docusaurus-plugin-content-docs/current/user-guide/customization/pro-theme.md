# Gukoresha Panel ya Admin na PRO Theme {#customizing-your-admin-panel-with-pro-theme}

Muriye mu kazi n'abantu (clients) ugiye kugira ngo umubiri w'admin ya WordPress ube n'umwihariko. Iyo ushaka umubiri ufite umurimo kandi w'umwihariko, PRO Theme yashobora kukugushyira mu muryango wawe.

PRO Theme ni theme ya admin yashobora kugufasha mu gihe ugiye gukoresha (customize) dashboard ya abantu bawe. kandi ni igice cy'ibindi plugins za Ultimate Multisite zishobora kuba zikoreshwa niyo.

Nyuma yo kubyara, gukora ku peji ya _Plugins_ yanyu kandi ugushaka gukorera (activate) kuri Network yawe.
