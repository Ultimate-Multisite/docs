# Gukoresha Ads ku mwezi w'ubwoko n'Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector ni **add-on ufite** ufungura uburyo bwo gukoresha tools zikufasha gukoresha ads kuri njye mu gihe cyo gutegura (edit page) byawe.

Nyuma yo gukoresha Ultimate Multisite: Ad Injector, gukora ku _Plugins_ page yawe kugira ngo ufungure.

<!-- Screenshot is not available: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Gukora kuri Button y'Network Activate_

Nyuma yo gutera add-on, _Advanced Options_ meta box kuri edit screen ya mwezi uza kuba na uburyo bwo gukoresha umuntu w'ubwoko (option) runaka mu gihe cyo gutegura. Uza kuba ari _Ad Settings_.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_Ushobora gukoresha Ad settings ku mwezi w'ubwoko w'ibindi byawe_

Ku gihe cyo, uza kuba ufite ubushobozi bwo gukoresha ad codes kuri posts n'pages mu front-end, mbere kandi nyuma y'ikintu cyangwa ibyo.

<!-- Screenshot is not available: Ad code fields for posts and pages before content -->

<!-- Screenshot is not available: Example of ad displayed on a post page for a specific plan -->

_Nyuma yo gukoresha ad code yawe, izitwa kuri posts n'pages za mwezi w'ubwoko uyuza ku gihe cyo._

Ushobora kandi gutera ads mu header ya Admin Pages ku bantu bafite mwezi w'ubwoko uyu.

<!-- Screenshot is not available: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot is not available: Example of ad displayed on the user admin panel -->

_Gukoresha Admin Panel Ad Code option niba wumva wifuza gukoresha ads ku back-end yawe (user's back-end)_

Ushobora kandi gukoresha Ultimate Multisite: Ad Injector kugira ngo ufungura ubushobozi bwo gufasha mwezi w'ubwoko uyuhe.

Gukora ku _Admin Panel Ads_ option kandi ugufasha HTML code yawe y'support tool.
