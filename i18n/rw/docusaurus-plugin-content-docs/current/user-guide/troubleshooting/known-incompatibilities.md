# Uburyo n'ubwitonzi n'ibindiwe n'ibindiwe n'ibindiwe

Nyuma yemeza no gukora Ultimate Multisite, ushobora kugira ibintu byiza ku buryo bwo kubaho n'ibindiwe n'ibindiwe n'ibindiwe.

Turi twakora neza kugabanya ubutumwa bwa ibyo bishobora kuba byiza n'uko turera uburyo bwo gukoresha, ariko niba plugin yashobora kubaho hano, ushobora gutera ko ugomba kuyitera imbere kugira ngo Ultimate Multisite Network itege neza.

Iki ni ubutumwa mu gihe cyose kandi iki cyo ugomba gukora:

**Uburyo bwo WPMU DEV plugins n'ibindiwe** Icyo gishobora kuba byiza n'ibindiwe plugin zishobora kuba zashobora gutera ko umugambi w'Ultimate Multisite wumutse ku gihe cyo gukoresha, ariko nta za kumenya. Hano hari ubutumwa mu gihe cyose kandi iki cyo ugomba gukora:

  * Signup Language: _Ultimate Multisite yashobora kugira add-on (plugin ikoresho) igaragaza uburyo bwo gukoresha, ariko itwa Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Iki cyitwa mu gihe cyose kandi nta za kumenya ko ugomba gukora plugin ikindi._

  * New Blog Templates: _Iki cyitwa mu gihe cyose kandi nta za kumenya ko ugomba gukora plugin ikindi._

  * Set Password on Multisite Blog Creation: _Iki cyitwa mu gihe cyose kandi nta za kumenya ko ugomba gukora plugin ikindi._

  * Signup TOS: _Iki cyitwa mu gihe cyose kandi nta za kumenya ko ugomba gukora plugin ikindi._

  * Anti-Splog: Iyo ni itandukanye, iyo iki cyitwa mu gihe cyose.

**Ubutumwa** : Ingo mu byo bindiwe n'ibindiwe plugin z'WPMU Dev, ushobora gukoresha neza n'Ultimate Multisite. Urugero ni _Smush_ , _Forminator_ , _Defender_, n'ibindi.
