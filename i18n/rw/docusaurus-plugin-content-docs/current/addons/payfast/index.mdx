# PayFast

Multisite Ultimate: Payfast Gateway

Yongeramo Payfast - uburyo bwihuse kandi butekanye bwo kwishyura kuri interineti muri Afurika y'Epfo - nk'ihitamo ryo kwishyura rya Multisite Ultimate.

## Kwishyiriraho {#installation}

1. Ohereza 'ultimate-multisite-payfast' mu bubiko bwa '/wp-content/plugins/'
2. Koresha iyi plugin unyuze muri menu ya 'Plugins' muri WordPress
3. Kurikiza Wizard intambwe ku yindi kugira ngo ushyireho iyi plugin

## Ubuyobozi bwo gukoresha {#usage-guide}

PayFast ni umutunganyabwishyu wo kuri interineti w'imbere ku bucuruzi bwo muri Afurika y'Epfo. Yiyemeje gufasha abacuruzi bo kuri interineti gukura, ituma ibikorwa byo kwishyurana bigenda neza kandi bifite umutekano. Nk'uko umukoresha wa Ultimate Multisite yabisabye, twafashe umwanzuro wo gukora inyongera izatuma guhuza na PayFast bishoboka. Iyi nyandiko irerekana uko washyiraho PayFast nk'uburyo bwawe bwo kwakira ubwishyu muri network yawe ya Ultimate Multisite.

Intambwe ya mbere ni ugushyiraho iyo nyongera.

# Uko washyiraho inyongera ya Payfast:

Jya kuri paji y'inyongera maze uhitemo inyongera ya Ultimate Multisite: Payfast Gateway.

![Paji y'inyongera](/img/admin/addons-page.png)

Idirishya ry'inyongera rizafunguka. Kanda kuri Install Now.

![Paji y'inyongera ifite ihitamo ryo kwishyiriraho](/img/admin/addons-page.png)

Nyuma y'uko kwishyiriraho birangiye, uzoherezwa kuri paji ya plugins. Hano, kanda gusa kuri Network Activate maze inyongera ya _Payfast_ izakorerwa kuri network yawe.

# Gushyiraho PayFast

Nyuma yo gushyiraho inyongera, ubu ushobora kujya mu igenamiterere rya Ultimate Multisite ugatangira gutunganya PayFast.

Munsi ya paji ya **igenamiterere rya Ultimate Multisite**, ushobora kubona agace ka "Payments". Munsi y'aka gace, wagombye kubona "**Payfast** " hamwe na buto yo guhinduranya kugira ngo ukoreshe cyangwa uhagarike ubu buryo bwo kwishyura.

![Igenamiterere ry'uburyo bwo kwishyura](/img/config/settings-payment-gateways.png)Komeza ukande kuri buto yo guhinduranya kugira ngo uyikoreshe. Hanyuma izagura agasanduku igaragaze imirima itandukanye ukeneye kuzuza nka **Mechant ID, Merchant Key na PassPhrase.** Ushobora kubona aya makuru muri Account yawe ya **Payfast**

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Intambwe ikurikira ni kwinjira muri Account yawe ya PayFast kugira ngo ubone amakuru ukeneye.

Umaze kwinjira, ushobora guhita ubona ibyangombwa ukeneye kuri dashboard ya Account yawe ya **Payfast**.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->Ushobora gusa gukoporora no komeka ibyangombwa maze ukuzuza ibisobanuro mu igenamiterere rya Ultimate Multisite. Hanyuma ukande kuri buto ya **"Save settings** " kugira ngo ubike impinduka.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Ni byo, guhuza ubwishyu bwa PayFast birarangiye!
