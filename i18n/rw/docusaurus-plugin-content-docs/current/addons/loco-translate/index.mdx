# Loco Translate {#loco-translate}

Ultimate Multisite: Loco Translate Integration addon izana ubushobozi bw’ubusemuzi bwihariye kuri buri rubuga muri network yawe ya WordPress multisite. Niba warigeze gukenera ko imbuga zitandukanye muri network yawe zigira ubusemuzi butandukanye bwa plugin cyangwa theme imwe, iyi addon ni igisubizo kiboneye.

**Icyo Iyi Addon Ikora**

Iyi addon ihuza Ultimate Multisite na plugin izwi cyane ya Loco Translate mu buryo bworoshye, igatuma buri subsite muri network yawe ishobora kugira ubusemuzi bwayo bwihariye. Waba ukoresha platform ikorera mu turere twinshi, utanga serivisi ku baturage bavuga indimi zitandukanye, cyangwa ukeneye gusa guhinduka mu busemuzi muri network yawe, iyi addon ibikora mu buryo bworoshye.

**Ibereye Aba:**

* Ubucuruzi bukorera mu turere twinshi bufite ibikenewe bitandukanye mu guhuza n’aho bukorera
* Ibigo by’uburezi bitanga serivisi ku baturage batandukanye
* Platform za SaaS zitanga ubufasha mu ndimi nyinshi
* Ibigo bikora web bicunga imbuga nyinshi z’abakiriya
* Network iyo ari yo yose ya multisite ikeneye guhinduka mu busemuzi

## Kwishyiriraho {#installation}

1. Menya neza ko Ultimate Multisite yashyizweho kandi yakorewe network activation
2. Shyiraho kandi ukore activate plugin ya Loco Translate
3. Shyira iyi addon muri plugins directory yawe
4. Kora network activate kuri addon ya Loco Translate Integration
5. Tangira gukora ubusemuzi bwihariye kuri buri rubuga ukoresheje Loco Translate kuri subsite iyo ari yo yose

Birarangiye! Uko guhuzwa gukora mu buryo bwikora nta yindi configuration ikenewe.

## Ibibazo Bikunze Kubazwa {#frequently-asked-questions}

## Ese nkeneye kugira icyo nkora muri configuration nyuma yo kuyishyiraho? {#do-i-need-to-configure-anything-after-installation}

Oya! Uko guhuzwa ni automatic. Iyo addon na Loco Translate byombi bimaze kuba active, subsites zishobora guhita zitangira gukora ubusemuzi bwazo bwihariye.

## Bigenda bite iyo subsite idakoze ubusemuzi bwihariye? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

Urubuga ruzahita rukoresha ubusemuzi bwa network yose nka fallback. Nta kintu cyangirika, kandi byose bikora neza nta nkomyi.

## Ese subsites zishobora kugera ku buseemuzi bwa network yose? {#can-subsites-access-network-wide-translations}

Abayobozi b’urubuga basanzwe babona kandi bagahindura gusa ubusemuzi bw’urubuga rwabo. Super admins bashobora kugera ku buseemuzi bwa network yose no ku buseemuzi bwihariye bw’urubuga, bitewe n’urubuga barimo kureba.

## Ese ibi bikorana na plugins na themes byombi? {#does-this-work-with-both-plugins-and-themes}

Yego! Ubusemuzi bwihariye kuri buri rubuga bukorana na WordPress plugins na themes byombi.

## Ese ibi bizatindiza network yanjye ya multisite? {#will-this-slow-down-my-multisite-network}

Oya. Iyi addon ikoresha uburyo bunoze bwo gufungura files bugenzura ubusemuzi bwihariye kuri buri rubuga gusa igihe bukenewe. Nta ngaruka zigaragara ku mikorere.

## Ese nshobora gukoporora ubusemuzi bwa network nkabujyana ku rubuga runaka nk’aho mpera? {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

Yego! Mu gihe uhindura ubusemuzi muri Loco Translate, ushobora gutangirana byoroshye n’ubusemuzi bwa network hanyuma ukabuhindura ku buryo bubereye buri rubuga.

## Ese ibi bikorana na WooCommerce n’izindi plugins? {#does-this-work-with-woocommerce-and-other-plugins}

Rwose! Plugin cyangwa theme iyo ari yo yose ikoresha standard WordPress translation functions izakorana n’ubusemuzi bwihariye kuri buri rubuga.
