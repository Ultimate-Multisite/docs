# VAT {#vat}

Multisite Ultimate: VAT yo mu Burayi

Yongeraho ibiranga n'ibikoresho by'inyongera ku micungire isanzwe y'imisoro ya Multisite Ultimate kugira ngo ishyigikire neza VAT yo mu Burayi.
Yongeraho mu buryo bwikora ibipimo bya EU VAT ku rupapuro rw'Ibipimo by'Imisoro kuva

## Gushyiraho {#installation}

1. Ohereza 'ultimate-multisite-vat' mu bubiko bwa '/wp-content/plugins/'
2. Koresha plugin unyuze muri menu ya 'Plugins' muri WordPress
3. Jya kuri Ultimate Multisite > Igenamiterere > Imisoro maze ukore EU Vat.
4. Kanda ku Bipimo by'Imisoro maze uvugurure Ibipimo bya EU VAT hasi, hanyuma ubike Ibipimo by'Imisoro.
5. Birangiye!
