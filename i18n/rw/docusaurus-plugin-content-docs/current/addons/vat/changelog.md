# Amateka y’impinduka za VAT {#vat-changelog}

Verisiyo 1.0.7 - Yasohotse ku wa 2026-02-03

* Gukosora: Nimero za VAT zo mu Budage (DE) zananiwe kwemezwa kubera amakosa ya MS_MAX_CONCURRENT_REQ ya serivisi ya VIES SOAP. Byahinduwe biva kuri SOAP API ishaje bijya kuri EU VIES REST API yemewe kugira ngo kwemeza birusheho kwizerwa.
* Gukosora: Nimero za VAT ubu ziremerwa iyo serivisi ya VIES itaboneka by’agateganyo, aho kugira ngo zanzwe nabi. Inyandiko ya log ikorwa kugira ngo izasuzumwe nyuma.
* Gukosora: Umwanya wa VAT/Tax ID ubu ugaragara ku bihugu byose, si ku banyamuryango ba EU gusa. Tax IDs zitari iza EU (urugero, nimero za Swiss CHE) zibikwa kugira ngo zigaragare kuri fagitire hatabayeho kwemezwa na VIES.
* Gukosora: Gushakisha VAT prefix kuri Greece (GR), Monaco (MC), na Isle of Man (IM) byari byapfuye kubera uburyo butari bwo bwo gushakisha muri array.
* Kunonosora: Havanyweho dependency ya dev ya wp-cli-bundle itari ikenewe yashoboraga guteza amakosa akomeye iyo ikoreshwa hamwe na addon ya Site Exporter.
* Kunonosora: Hiyongereyeho suite yuzuye y’ibizamini bya unit (ibizamini 53).

Verisiyo 1.0.6 - Yasohotse ku wa 2026-01-25

* Gukosora: Ubucuruzi bwa B2B bwo mu gihugu kimwe ubu bwishyura VAT neza. Dukurikije amategeko ya VAT ya EU, reverse charge ikoreshwa gusa ku bucuruzi bwa B2B bwambukiranya imipaka, si igihe igihugu cy’umukiriya gihuye n’igihugu cy’isosiyete.

Verisiyo 1.0.5 - Yasohotse ku wa 2026-01-22

* Gukosora: Addon itarimo gupakira cyangwa gukora neza.
* Guhindura: Inkomoko y’amakuru y’ibipimo bya VAT yahinduwe iva kuri euvatrates.com ijya muri repository ya ibericode/vat-rates kugira ngo amakuru arusheho kwizerwa kandi abungabungwe neza.
* Gukosora: Hakosowe ikosa ry’inyuguti mu gaciro k’ihitamo rya super_reduced_rates.
* Kunonosora: Hiyongereyeho igenzura rya null mu micungire ya kode y’igihugu kugira ngo hirindwe amakosa.
* Guhindura: Havanyweho amadosiye y’ubusemuzi yari apfunyitse, hashingirwa ku mavugurura yikora binyuze muri Traduttore.
