# Amateka y'impinduka ya gahunda zipimwa {#metered-plans-changelog}

Verisiyo 1.1.0 - Yasohotse ku wa 2026-05-05
- Gishya: Kwishyuza AI token ku masite mato ya multisite — ukurikirane kandi wishyuze ikoreshwa rya AI token ku masite y'abakiriya ukoresheje ibiciro bishyirwaho kuri buri token
- Gishya: Gushyira mu bikorwa Connector byongeye kubakwa hifashishijwe kuvumbura imipaka mu buryo buhinduka no kwandika ako kanya, bikemeza ukuri kw'ako kanya kuri Connector zose
- Gikosowe: Kuzamura ameza ya database ubu bihujwe neza n'ibisobanuro bya BerlinDB schema, bikumira kunanirwa kuzamura ku mashyirwaho mashya
- Gikosowe: Database upgrade callbacks zahinduwe ku miterere ikwiye, bikemura kunanirwa kuzamura kutagaragara
- Gikosowe: Indangagaciro z'ibice ubu ziremewe mu mwanya wo kwinjizamo AI Usage Overage Markup
- Gikosowe: Hakemuwe amakosa akomeye n'ibibazo byo gutangiza inshuro ebyiri igihe plugin itangiye
- Byanonosowe: Hyongewemo check-env npm script kugira ngo ibidukikije by'abakora iterambere byishyireho ubwabyo ku ikorwa rya mbere
