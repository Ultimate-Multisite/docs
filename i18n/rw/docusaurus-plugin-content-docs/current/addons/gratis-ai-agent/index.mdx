# Gratis AI Agent {#gratis-ai-agent}

Gratis AI Agent ni umufasha wa WordPress AI wigenga ushobora gutegura, kubaka, no gucunga imbuga za WordPress zuzuye binyuze mu mabwiriza y'ururimi rusanzwe. Itanga urutonde ruteguye rwa **ubushobozi** — ibikorwa byihariye, bishobora kugeragezwa agent ishobora kwifashisha — bikubiyemo byose kuva ku micungire y'ibirimo kugeza ku guhindura sisitemu y'imiterere. Verisiyo 1.9.0 yongeramo ubushobozi bwo gukora ibirimo, isuzuma ry'amapaji mu buryo bugaragara hakoreshejwe amashusho afatwa ku ruhande rw'umukiriya, imyirondoro itanu ya agent yubatswemo, n'ibirango by'imikorere byo kugenzura uburenganzira no gushyiraho ikirango.

## Ibiranga by'ingenzi {#key-features}

- **Sisitemu y'ubushobozi** — kataloge y'ibikorwa igizwe n'ibice, ishobora kwagurwa, agent ishobora kwifashisha kuri installation yawe ya WordPress
- **Imicungire y'ibirimo** — gukora posts n'amapaji (harimo guha amapaji templates), gukora posts nyinshi icyarimwe mu ihamagara rimwe, gushyiraho amashusho agaragara, no kubaka contact forms
- **Isuzuma rigaragara** — gufata screenshots z'amapaji ari gukora, kugereranya uko yari ameze mbere/nyuma, no kubona isuzuma ry'imiterere ryakozwe na AI rikubiyemo layout, typography, ibara, n'ubugerwaho
- **Agents batanu bubatswemo** — Content Writer, Site Builder, Design Studio, Plugin Manager, na Support Assistant; bashobora guhinduranywa ukoresheje Agent Picker iri mu mutwe w'ikiganiro
- **Ibirango by'imikorere** — toggles zo kugenzura uburenganzira no gushyiraho ikirango muri Settings → Feature Flags ku mbogamizi z'inshingano, amazina ya white-label, no guhindura widget
- **Imicungire ya Custom Post Type** — kwandika muri sisitemu, kugaragaza urutonde, no gusiba custom post types bikagumana amakuru no nyuma yo kongera gutangira
- **Imicungire ya Custom Taxonomy** — kwandika muri sisitemu, kugaragaza urutonde, no gusiba custom taxonomies bikagumana amakuru
- **Ubushobozi bwa Design System** — gushyiramo custom CSS, gucunga block patterns zatoranyijwe, gushyiraho logo y'urubuga, no gukoresha presets za theme.json
- **Imicungire ya Global Styles** — gusoma no kwandika indangagaciro za theme.json z'amabara, typography, spacing, n'ibindi
- **Imicungire ya Navigation Menu** — gukora no kuvugurura navigation menus n'ibiyigize mu buryo bwa porogaramu
- **Imicungire y'amahitamo** — gusoma no kwandika amahitamo ya WordPress hamwe na blocklist y'umutekano yubatswemo yo kurinda igenamiterere rikomeye
- **Installable Abilities Registry** — kuvumbura no gukora ability packs z'inyongera udahinduye amadosiye y'ingenzi ya plugin
- **Site Builder Orchestration v2** — gahunda zo kubaka urubuga z'intambwe nyinshi hamwe no kuvumbura plugin, gukurikirana aho bigeze, no kwisubiza mu makosa
- **Itegeko rya WP-CLI benchmark** — gukoresha itsinda ry'ibipimo bya benchmark ry'ubushobozi bwa agent uhereye kuri command line kuri CI no gusuzuma model

## Ibisabwa {#requirements}

- WordPress 6.0 cyangwa irenzeho
- PHP 8.1 cyangwa irenzeho
- WordPress Multisite (network-activated) cyangwa installation y'urubuga rumwe
- Plugin connector y'utanga AI ikora (Anthropic, OpenAI, cyangwa ihuye na byo)

## Installation {#installation}

1. Ohereza amadosiye ya plugin kuri `/wp-content/plugins/gratis-ai-agent`
2. Kora plugin ukoresheje screen ya **Plugins** muri WordPress
3. Jya kuri **Gratis AI Agent → Settings** maze uhuze utanga AI wawe
4. (Si ngombwa) Reba muri **Abilities Registry** kugira ngo ushyiremo ability packs z'inyongera

## Gutangira vuba {#quick-start}

Iyo imaze gukora no guhuzwa n'utanga AI, fungura panel ya **Gratis AI Agent** muri sidebar ya WordPress admin. Andika amabwiriza mu rurimi rusanzwe, agent izategura kandi ishyire mu bikorwa intambwe zisabwa ikoresheje ubushobozi ifite.

**Ingero z'amabwiriza:**

```
Kora custom post type yitwa "Portfolio" ifite taxonomy yitwa "Project Category".
```

```
Shyira colour palette ya dark-mode muri theme maze ushyireho ibara ry'ibanze kuri #1a1a2e.
```

```
Baka website ya restaurant ifite menu yo kuri murandasi, booking form, na paji y'itumanaho.
```

## Incamake y'ubushobozi {#abilities-overview}

Ubushobozi ni ibikorwa by'ibanze agent ishobora gukora. Buri bushobozi ni class ya PHP yanditswe muri sisitemu itanga schema (parameters na return types) agent isoma igihe iri gukora. Ubushobozi bushyirwa mu byiciro by'imirimo:

| Igice | Ubushobozi |
|---|---|
| **Imicungire y'ibirimo** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **Isuzuma rigaragara** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **Custom Post Types** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **Custom Taxonomies** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **Design System** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **Global Styles** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **Navigation Menus** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **Imicungire y'amahitamo** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **Ubushobozi bushobora gushyirwamo** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **Site Builder** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

Reba [Ibisobanuro by'ubushobozi](./abilities) kugira ngo ubone inyandiko zuzuye za parameters n'ingero z'imikoreshereze.

## WP-CLI {#wp-cli}

Gratis AI Agent izana umuryango w'amategeko wa `wp gratis-ai-agent` wo gukoresha benchmarks no gucunga ubushobozi uhereye kuri command line. Reba [Ibisobanuro bya WP-CLI](./wp-cli) kugira ngo ubone urutonde rwose rw'amategeko.

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## Igenamiterere {#configuration}

Igenamiterere ryose riri munsi ya **Gratis AI Agent → Settings** muri WordPress admin:

| Igenamiterere | Ibisobanuro |
|---|---|
| **Utanga AI** | Hitamo kandi ugene plugin connector y'utanga AI ikora |
| **Model** | Hitamo language model izakoreshwa mu masesiyo ya agent |
| **Max Tokens** | Ingengo ya tokens kuri buri busabe (default 4096) |
| **Ability Timeout** | Amasegonda ntarengwa ubushobozi bumwe bushobora kumara bukora (default 30) |
| **Debug Logging** | Andika ability calls n'imitekerereze ya agent muri WordPress debug log |

## Blocklist y'umutekano y'imicungire y'amahitamo {#options-management-safety-blocklist}

Ubushobozi bwo Gucunga Amahitamo burimo blocklist yubatswemo ibuza agent gusoma cyangwa kwandika amahitamo ya WordPress arimo amakuru y'ibanga. Blocklist isanzwe ikubiyemo:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (amahitamo ya URL — koresha igikoresho cyihariye cyo kwimura kuri ibi)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (imfunguzo z'ibanga)
- `active_plugins`, `template` (gukora plugin/theme — koresha ubushobozi bwihariye)
- Ibyangombwa bya payment gateway na API keys byanditswe na plugins zashyizwemo

Ibyinjira by'inyongera bishobora kongerwamo hifashishijwe filter `gratis_ai_agent_options_blocklist`.

## Hooks na Filters {#hooks-and-filters}

### Actions {#actions}

- `gratis_ai_agent_ability_registered` — itangizwa nyuma y'uko ubushobozi bwanditswe, yakira izina rya class y'ubushobozi
- `gratis_ai_agent_plan_started` — itangizwa igihe plan yo kubaka site itangiye gushyirwa mu bikorwa
- `gratis_ai_agent_plan_completed` — itangizwa igihe plan irangiye neza
- `gratis_ai_agent_plan_failed` — itangizwa igihe plan ihuye n'ikosa ridashobora gukosorwa

### Filters {#filters}

- `gratis_ai_agent_abilities` — hindura urutonde rw'ubushobozi bwanditswe mbere y'uko agent irupakira
- `gratis_ai_agent_options_blocklist` — yagura cyangwa ugabanye blocklist y'umutekano w'amahitamo
- `gratis_ai_agent_benchmark_questions` — ongeramo ibibazo bya benchmark byihariye mu itsinda ry'isuzuma
- `gratis_ai_agent_plan_plugins` — hindura urutonde rwo gushakisha plugin mu gihe cyo gukora site plan
