# Muyunguruzi: signup_get_available_languages {#filter-signupgetavailablelanguages}

Iyungurura urutonde rw’indimi ziboneka zo kwiyandikisha ku rubuga rwa front-end.

Gutanga array irimo ubusa kuri iyi hook bizahagarika kugaragaza iryo genamiterere ku ifishi yo kwiyandikisha, kandi ururimi rusanzwe ruzakoreshwa igihe urubuga ruremwaga. Indimi zitarashyirwaho zizakurwamo.

## Ibigenderwaho {#parameters}

| Izina | Ubwoko | Ibisobanuro |
|------|------|-------------|
| $available_languages | `array` | Indimi ziboneka. |
