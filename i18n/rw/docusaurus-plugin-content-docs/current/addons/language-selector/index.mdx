# Muhitamo w'Ururimi {#language-selector}

Ultimate Multisite: Muhitamo w'Ururimi mu gihe cyo kwiyandikisha

Ongeraho umwanya wo guhitamo ururimi mu rugendo rwo kwiyandikisha.

## Kwinjiza {#installation}

1. Ohereza 'ultimate-multisite-language-selector' mu bubiko bwa '/wp-content/plugins/'
2. Koresha plugin unyuze muri menu ya 'Plugins' muri WordPress Network Admin yawe

## Ubuyobozi bw'Imikoreshereze {#usage-guide}

Ukoresheje iyi add-on y'ubuntu, ushobora kureka abakoresha bawe bagahitamo ururimi bifuza kubona mu rugendo rwo kwiyandikisha no ku rupapuro rwo kwinjira.

Nyuma yo [gukoresha uruhushya rwawe](../../user-guide/getting-started/installing-ultimate-multisite.md), uzashobora kwinjiza add-ons za Ultimate Multisite uruhushya rwawe ruguha uburenganzira bwo kugeraho.

Jya ku rupapuro rw'igenamiterere rwa Ultimate Multisite yawe, hanyuma, muri menu iri ibumoso, ukande ikirango cya _Add-ons_.

![Urupapuro rwa add-ons](/img/admin/addons-page.png)

_Jya ku rupapuro rwa Add-ons kugira ngo utangire kwinjiza Muhitamo w'Ururimi_

Aho, uzahasanga add-ons zose dutanga.

Add-ons z'ubuntu ziboneka ku bakiriya bacu bose, hatitawe kuri plan y'ubunyamuryango biyandikishijeho.

Kanda buto ya _Install it Now!_ iri munsi y'ishusho nto y'iyo add-on. Hanyuma uyikoreshe ku rupapuro rwa Plugins.

![Urupapuro rwa add-ons rufite buto yo kwinjiza](/img/admin/addons-page.png)

_Kanda buto kugira ngo winjize add-on_

<!-- Ifoto ya ecran itaboneka: urupapuro rwa WP Plugins rwerekana buto ya Network Activate ya add-on ya Muhitamo w'Ururimi -->

_Yikoreshe ku rupapuro rwa Plugins rwawe_

Noneho, jya muri menu y'Igenamiterere rya Ultimate Multisite maze winjire muri tab ya _Network Settings_. Aho, hasi ku rupapuro, uzahasanga igice gishya cyitwa _Languages_.

![Urupapuro rw'igenamiterere](/img/config/settings-general.png)

_Igice gishya rwose kiragaragara!_

Ushobora kubireka ubusa kugira ngo indimi zose zigaragare nk'amahitamo, cyangwa ugahitamo izo wifuza gutuma ziboneka. Indimi ni izo winjije mu mitegurire ya WordPress yawe.

<!-- Ifoto ya ecran itaboneka: menu imanuka yo guhitamo ururimi yerekana indimi ziboneka muri WP install -->

_Urutonde rw'indimi ziboneka muri iyi WP install_

Nyuma yo kubika impinduka wakoze, menu yo guhitamo ifite indimi wahisemo izagaragazwa ku rupapuro rwo Kwinjira ndetse no mu Rugendo rwo Kwiyandikisha.

![Igenamiterere ryo kwinjira no kwiyandikisha](/img/config/settings-login-registration.png)

_Muhitamo w'Ururimi ku Rupapuro rwo Kwinjira_

_Muhitamo w'Ururimi mu Rugendo rwo Kwiyandikisha_

## Gushyiraho Ururimi hakoreshejwe URL {#setting-the-language-via-the-url}

Ushobora kandi guhindura ururimi rw'urupapuro wohereza parameter ya locale muri URL. Niba ushaka kohereza umuntu ku rupapuro rwo kwiyandikisha mu Gitaliyani, urugero, ongera ?locale=it_IT kuri URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Ururimi rugomba kuba rwinjijwe kuri network ya WordPress yawe kugira ngo ibi bikore.
