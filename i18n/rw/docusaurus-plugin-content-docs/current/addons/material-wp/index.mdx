# Material WP {#material-wp}

Interineti y'ubuyobozi ya WordPress imaze igihe kirekire imeze kimwe. Ariko wigeze wibaza uko verisiyo ya Material Design y'interineti y'ubuyobozi yaba imeze? 

Material WP ni theme y'ubuyobozi yakozwe kugira ngo iguhe iyo sura n'imikorere, igufasha guha abakiriya bawe Dashboard yihariye kurushaho.

## Ibiranga {#features}

- Gushyigikira CSS yihariye
- Igenamiterere ryoroshye ryo gutunganya imiterere
- Kongera gutondeka menu
- Kuzana/kohereza hanze byoroshye

## Inyandiko {#documentation}

**Gutunganya Admin Panel yawe ukoresheje Material WP**

## Kwishyiriraho {#installation}

1. Shyira 'material-wp' mu bubiko bwa '/wp-content/plugins/'
2. Koresha plugin unyuze muri menu ya 'Plugins' muri WordPress

## Uko ikoreshwa {#usage-guide}

Interineti y'ubuyobozi ya WordPress imaze igihe kirekire imeze kimwe. Ariko wigeze wibaza uko verisiyo ya Material Design y'interineti y'ubuyobozi yaba imeze?

Material WP ni theme y'ubuyobozi yakozwe kugira ngo iguhe iyo sura n'uko wumva imeze, igufasha guha abakiriya bawe Dashboard yihariye kurushaho. Material WP kandi iri mu gice nyamukuru cya plugins zishyurwa za Ultimate Multisite.

Nyuma yo **kuyishyiraho**, jya kuri paji yawe ya _Plugins_ maze ukande kugira ngo uyikoreshe kuri Network yawe.
