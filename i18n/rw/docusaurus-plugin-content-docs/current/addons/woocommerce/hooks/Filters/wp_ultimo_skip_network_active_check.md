# Filter: wp_ultimo_skip_network_active_check {#filter-wpultimoskipnetworkactivecheck}

Byemerera abateza imbere guhita basimbuka igenzura ry’activation ya network.

Ibi bifite akamaro igihe ukoresha imiterere ishingiye kuri composer n’indi miterere yihariye, nka Bedrock, urugero, aho gukoresha plugins nka mu-plugins ari byo bisanzwe.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $skip_network_activation_check | `bool` | Niba dukwiye gusimbuka igenzura cyangwa tutarigusimbuka, bisanzwe ni false. |

### Kuva {#since}

- 2.0.0
### Inkomoko {#source}

Bisobanuwe muri [`inc/class-requirements.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-woocommerce/blob/main/inc/class-requirements.php#L272) ku murongo wa 272


## Ibisubizo {#returns}
true niba ushaka gusimbuka igenzura, false bitabaye ibyo.
