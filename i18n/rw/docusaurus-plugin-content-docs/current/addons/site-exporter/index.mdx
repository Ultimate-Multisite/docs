# Mwoherezi w'Urubuga

## Ibirimo {#content}

Twumva ko hakenewe ubushobozi bwo kwimura imbuga nto ziri muri network yawe zijya ku ishyirwaho ry'urubuga rumwe, cyangwa se ibinyuranye na byo, igihe umukiriya wawe afashe icyemezo cyo kwimura urubuga rwe. Addon ya mwoherezi n'umwinjizi w'urubuga yoroshya iki gikorwa. Iki gikoresho cyohereza kandi kikinjiza imbuga gikoresheje dosiye za zip, zishobora no gukora nk'ububiko bw'umutekano.

## Ibiranga {#features}

* Kohereza no Kwinjiza Imbuga

* Imigaragarire yoroshye gukoresha

* Gukora backup y'urubuga

## Inyandiko {#documentation}
**Uko Wohereza no Kwinjiza Urubuga**

## Gushyiraho {#installation}

1. Ohereza **ultimate-multisite-site-exporter** muri directory ya **/wp-content/plugins/**
2. Koresha plugin unyuze muri menu ya **Plugins** muri WordPress
3. Kurikiza Wizard intambwe ku yindi kugira ngo ushyireho plugin

## Uko Bikoreshwa {#usage-guide}

Ukoresheje Ultimate Multisite ushobora kohereza urubuga ruvuye muri network yawe cyangwa kwinjiza urubuga muri multisite yawe cyangwa mu ishyirwaho ry'urubuga rumwe.

Kugira ngo utangire iki gikorwa, jya kuri paji yacu ya add-ons maze ushyireho add-on ya Ultimate Multisite: Site Exporter & Importer.![Paji ya add-ons](/img/admin/addons-page.png)

![Paji ya add-ons ifite uburyo bwo gushyiraho](/img/admin/addons-page.png)

Jya kuri paji yawe ya Plugins maze ukande kuri **Network Activate** add-on umaze gushyiraho.<!-- Ifoto ya ecran ntiboneka: paji ya WP Plugins igaragaza buto ya Network Activate ya addon ya Site Exporter -->

Uburyo bwa **Export & Import** buboneka kuri menu ya Ultimate Multisite, ibumoso.

<!-- Ifoto ya ecran ntiboneka: menu ya Ultimate Multisite igaragaza uburyo bwa Export and Import -->

## Uko wohereza urubuga {#how-to-export-a-site}

Kohereza urubuga ruvuye muri network yawe, jya kuri **Export & Import** kuri menu ya Ultimate Multisite maze, kuri tab ya **Export Sites**, ukande kuri **Generate new Site Export**.

<!-- Ifoto ya ecran ntiboneka: tab ya Export Sites ifite buto ya Generate new Site Export -->

Shaka urubuga ushaka kohereza maze uhitemo ibintu ushaka gushyira muri dosiye yoherejwe. Ushobora gushyiramo themes, plugins, na uploads. Niba urubuga rwawe ari runini cyane, ushobora guhitamo gukoresha igikorwa cyo kohereza mu background.<!-- Ifoto ya ecran ntiboneka: amahitamo yo kohereza urubuga afite gushaka urubuga, udusanduku two gushyiramo themes/plugins/uploads -->

Kanda buto ya **Export Site**. Urubuga rwawe rwoherejwe ruzaboneka ku rutonde rwa **Existing Exports**. Kanda buto ya **Download ZIP** kugira ngo ukuremo dosiye. Export ZIPs zirimo `index.php` yitangiza ubwayo na `readme.txt`, bityo zishobora gushyirwa kuri host nshya utabanje gushyiraho plugin y'umwinjizi itandukanye.<!-- Ifoto ya ecran ntiboneka: urutonde rwa Existing Exports rufite buto ya Download ZIP -->

## Uko wohereza network yose {#how-to-export-the-whole-network}

Kohereza imbuga nto zose hamwe, jya kuri **Export & Import** kuri menu ya Ultimate Multisite maze ukoreshe uburyo bwa **Network Export** kuri ecran yo kohereza. Archive ikorwa iba irimo imbuga nto zose muri ZIP imwe, ikoreshwa mu kwimura host, kongera kubaka staging, cyangwa backups zo ku rwego rwa network.

Network exports zishobora kuba nini. Zikoreshe igihe traffic ari nke kandi wemeze ko server ifite umwanya uhagije wo kubika archive n'ibindi byose byashyizwemo nka uploads, plugins, na themes.

## Uko winjiza urubuga {#how-to-import-a-site}

Kwinjiza urubuga, jya kuri tab ya **Import a Site** maze wohereze dosiye ya .zip y'urubuga ushaka kwinjiza. Ushobora guhindura URL ukayigira nshya cyangwa ugasiga URL ya kera usize umwanya uri ubusa. Nanone, ushobora gushyira akamenyetso mu gasanduku kugira ngo usibe dosiye ya .zip mu ishyirwaho ryawe nyuma yo kwinjiza. Hanyuma ukande kuri **Begin Import**.

<!-- Ifoto ya ecran ntiboneka: tab ya Import a Site ifite field yo kohereza zip na buto ya Begin Import -->

Igikorwa gishobora gufata akanya gato. Uzabona import itegereje mu rutonde ruri munsi y'umwinjizi.

<!-- Ifoto ya ecran ntiboneka: import itegereje mu rutonde ruri munsi y'umwinjizi -->

Ushobora guhagarika igikorwa ukanda **Cancel Import**.

<!-- Ifoto ya ecran ntiboneka: buto ya Cancel Import yo guhagarika igikorwa cyo kwinjiza -->

Nyuma y'uko igikorwa kirangiye, urubuga rwawe ruba rusanzwe ruboneka kuri network yawe.

## Uko winjiza urubuga ku ishyirwaho rimwe {#how-to-import-a-site-on-a-single-install}

Kwinjiza urubuga ku ishyirwaho ryawe rimwe, banza ushyireho kandi ukoreshe umwinjizi ku rubuga rwawe. Ushobora kubona link yo kuwukuramo kuri tab y'umwoherezi.

<!-- Ifoto ya ecran ntiboneka: tab y'umwoherezi ifite link yo gukuramo umwinjizi w'urubuga rumwe -->

Ushobora kubona umwinjizi kuri menu ya **Tools**, ibumoso, ku kantu kayo ka **Export & Import**.

<!-- Ifoto ya ecran ntiboneka: menu ya Tools y'urubuga rumwe igaragaza submenu ya Export and Import -->

Ohereza dosiye ya .zip y'urubuga ushaka kwinjiza. Ushobora guhindura URL ukayigira nshya cyangwa ugasiga URL ya kera usize umwanya uri ubusa. Nanone, ushobora gushyira akamenyetso mu gasanduku kugira ngo usibe dosiye ya .zip mu ishyirwaho ryawe nyuma yo kwinjiza. Hanyuma ukande kuri **Begin Import**.

<!-- Ifoto ya ecran ntiboneka: form yo kwinjiza ku rubuga rumwe ifite kohereza zip na buto ya Begin Import -->

Igikorwa gishobora gufata akanya gato. Uzabona import itegereje mu rutonde ruri munsi y'umwinjizi.

<!-- Ifoto ya ecran ntiboneka: import itegereje y'urubuga rumwe mu rutonde -->

Ushobora guhagarika igikorwa ukanda **Cancel Import**.

<!-- Ifoto ya ecran ntiboneka: buto ya Cancel Import yo guhagarika igikorwa cyo kwinjiza -->

Nyuma y'uko igikorwa kirangiye, urubuga rwawe ruzasimburwa n'urubuga umaze kwinjiza.
