# Uwashyiraho Paji za Admin

## Ibirimo
Uwashyiraho paji za admin azagufasha kongeramo byoroshye paji nshya za admin na paji z’imbere muri WordPress admin dashboard. Ushobora kuwukoresha mu gukora paji yo kwakira cyangwa paji y’inyigisho ishobora kuba paji ibanza ku bakoresha bashya. Ushobora gukoresha page builder nka Beaver Builder, Elementor, Brizy, na Oxygen kugira ngo utegure paji nta ngorane.

Unatanga ubufasha bwa HTML, CSS na JavaScript byihariye, ndetse n’igenzura ry’uburenganzira hashingiwe kuri plan n’uruhare rw’umukoresha!

## Ibiranga
* Gushyigikira HTML na CSS byihariye
* Gushyigikira JavaScript
* Uburyo bw’Ibirimo bya PHP
* Uburenganzira bushingiye ku ruhare rw’Umukoresha
* Gushyigikira page builders

## Inyandiko
[Gukora Paji za Admin na Paji z’imbere zihariye ukoresheje Multisite Ultimate: Admin Page Creator](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## Kwinjiza

1. Ohereza **ultimate-multisite-admin-page-creator** muri directory ya **/wp-content/plugins/**
2. Koresha plugin unyuze muri menu ya **Amacomekwaho** muri Admin w’Urusobe rwa WordPress yawe

## Amabwiriza yo Gukoresha

Nubwo WordPress yoroshya cyane kongera ibirimo ku ruhande rw’imbere rw’imbuga, gukora paji na paji z’imbere ku ruhande rwa admin bisaba ubumenyi bw’ibanze bwo kwandika code. Ultimate Multisite: Admin Page Creator igamije kuzana ku ruhande rw’inyuma rw’imbuga z’urusobe rwawe ubwo buryo bworoshye bwo gukoresha busanzwe ku ruhande rw’imbere.

## Kuki wakoresha Admin Page Creator?

Wenda ushaka gutanga paji yihariye ku kanama ka admin k’abakiriya bawe irimo inyigisho, cyangwa wenda ushaka kongeramo paji yihariye irimo ibibazo bikunze kubazwa n’agakoresho ko kuvugana. Ibishoboka ntibigira iherezo na Admin Page Creator, kandi ni byo turi busuzume mu nyigisho y’uyu munsi.

## Gukora Paji yawe ya mbere ya Admin Yihariye

Nyuma yo kwinjiza no gukoresha Ultimate Multisite: Admin Page Creator, jya kuri menu ya admin y’urusobe rwawe maze ujye kuri Paji za Admin.

![Urutonde rwa Paji za Admin Zihariye kuri Admin w’Urusobe](/img/admin/admin-pages-list.png)

_Urutonde rwa Paji za Admin Zihariye kuri Admin w’Urusobe_

Uzabona urutonde rwa paji za admin zose umaze gukora kugeza ubu (kubera ko ari bwo bwa mbere uhageze, urutonde ruzaba rurimo ubusa). Komeza ukande **Ongeraho Paji nshya ya Admin**. Uzoherezwa kuri editor ya Paji ya Admin.

## Ibice bigize Paji ya Admin Yihariye

Kuri editor ya Paji ya Admin, uzaba ufite amahitamo atandukanye menshi yo guhindura paji yawe uko ushaka.

### Umutwe wa Paji

Uyu ni umutwe uzagaragara hejuru ya paji, si izina rigaragara muri menu ya admin.

<!-- Ishusho ntibonetse: Editor ya Paji ya Admin yerekana field y’Umutwe wa Paji -->

_Injiza Umutwe wa Paji_

### Uhitamo Ubwoko bw’Ibirimo

Ultimate Multisite: Admin Page Creator iguha amahitamo abiri ku birebana n’ibirimo bya paji: ushobora gukoresha editor ya WordPress WYSIWYG isanzwe cyangwa ugahindukira kuri editor ya HTML.

Niba utamenyereye HTML, komera kuri editor ya WordPress isanzwe. Uzashobora guhindura ibirimo bya paji nk’uko wabigenza kuri Post cyangwa Paji isanzwe.

<!-- Ishusho ntibonetse: Admin Page Creator ikoresha editor ya WordPress WYSIWYG isanzwe -->

_Gukoresha editor ya WordPress WYSIWYG isanzwe_

<!-- Ishusho ntibonetse: Admin Page Creator ikoresha editor ya HTML ifite kugaragaza syntax -->

_Gukoresha ihitamo rya editor ya HTML, rifite kugaragaza syntax n’imbuzi z’amakosa_

## Amahitamo ya Menu

Ku ruhande, muri meta-box ya Amahitamo Rusange, uzashobora gutunganya ubwoko bwa menu, izina, n’ibindi byinshi.

### Ubwoko bwa Menu

Paji yawe nshya ya admin yihariye ishobora kugira ubwoko bubiri bwa menu: menu yo ku rwego rwo hejuru na submenu.

Paji ya menu yo ku rwego rwo hejuru, nk’uko izina ribivuga, izongeramo paji yawe nshya wakoze muri menu ya admin hamwe na paji zisanzwe za WordPress.

Nuhitamo ihitamo ryo ku rwego rwo hejuru, uzashobora guhitamo uko menu ikurikirana n’agashushondanga ka menu na ko.

Niba ushaka kugira iyi paji paji ya sub-menu, uzashobora guhitamo paji y’umubyeyi yo kuyifatanyaho. Ushobora kongera paji yawe yihariye kuri paji zisanzwe za WordPress cyangwa kuri paji zawe za admin zihariye zo ku rwego rwo hejuru.

<!-- Ishusho ntibonetse: Amahitamo y’ubwoko bwa menu ya Admin Page Creator arimo izina, uko ikurikirana, n’igenamiterere ry’agashushondanga -->

_Hindura byuzuye ikintu cya menu kuri iyi Paji Yihariye_

<!-- Ishusho ntibonetse: Guhitamo paji y’umubyeyi ya sub-menu muri Admin Page Creator -->

_Ushobora kandi kongera iyi paji nka sub-menu y’izindi paji za menu, harimo n’izawe bwite zihariye zo ku rwego rwo hejuru._

## Amahitamo Ateye Imbere

Ugomba kuba ushobora kongera ubwoko bwose bw’ibirimo kuri paji zawe za admin zihariye. Ibyo birimo amategeko na dosiye za CSS, JavaScript, ndetse n’amasomero yo hanze. Ushobora kubikora ukoresheje Tab y’Amahitamo Ateye Imbere.

### CSS Yihariye

Kuri tab ya CSS, ushobora kongeramo code yawe ya CSS yihariye, ndetse no kwinjiza dosiye za CSS zivuye ku mbuga za kure (nka CDNs, Google Fonts, n’ibindi).

<!-- Ishusho ntibonetse: Editor ya CSS yihariye ya Admin Page Creator ifite kwinjiza stylesheet yo hanze -->

_CSS yihariye na yo irashyigikirwa. Ushobora kandi gushyiramo dosiye z’imisusire zo hanze!_

### JavaScript Yihariye

Ibyo na byo ni ko bimeze kuri JavaScript. Ushobora kongeramo amasomero yo hanze kugira ngo uyakoreshe muri code yawe ya JavaScript yihariye.

<!-- Ishusho ntibonetse: Editor ya JavaScript yihariye ya Admin Page Creator ifite kwinjiza isomero ryo hanze -->

_Ongeramo code yawe bwite ya JavaScript!_

## Uburenganzira

Nanone mu Mahitamo Ateye Imbere, uzashobora guhitamo sub-sites zizabona iyi paji. Ibi ni ingirakamaro niba uteganya gutanga ibirimo byihariye bitandukanye ku byiciro bitandukanye bya plan, urugero.

<!-- Ishusho ntibonetse: Igenamiterere ry’uburenganzira rya Admin Page Creator ku igenzura ry’uburenganzira rishingiye kuri plan n’uruhare -->

_Ufite igenzura ryuzuye ry’aho iyi paji izagaragara_

**Icy'ingenzi** : Ibi bisabwa byombi bigomba kuba byujujwe kugira ngo urupapuro rugaragazwe. Ibyo bivuze ko niba umukoresha ari umukiriya wa gahunda runaka, ariko akaba adafite imwe mu nshingano zatoranyijwe (cyangwa se ikinyuranyo cyabyo), urupapuro ntiruzagaragara kuri paneri ye.
