# Umuyobozi w’Imigereka n’Insanganyamatsiko {#plugin--theme-manager}

Ultimate Multisite: Umuyobozi w’Imigereka n’Insanganyamatsiko

Hindura cyangwa uhishe amakuru ya meta (umutwe, umwanditsi, thumbnail n’ibisobanuro) y’imigereka n’insanganyamatsiko byashyizweho. Kora ibyiciro by’imigereka n’insanganyamatsiko kugira ngo abakoresha bawe babashe kubiyungurura, kandi wongere ipaji nziza yihariye y’Imigereka mu rubuga rw’abakoresha bawe.

Kugira ngo ukoreshe uyu mugereka, ushobora gusa kujya ku mapaji yawe y’Imigereka cyangwa Insanganyamatsiko mu buyobozi bw’urusobe rwawe maze ugakanda kuri "Hindura amakuru y’Umugereka/Insanganyamatsiko".

## Gushyiraho {#installation}

1. Ohereza 'ultimate-multisite-plugin-and-theme-manager' muri direktoriyo ya '/wp-content/plugins/'
2. Koresha uyu mugereka unyuze muri menyu ya 'Imigereka' muri WordPress Network Admin yawe

## Uburyo bwo Gukoresha {#usage-guide}

Ultimate Multisite: Umuyobozi w’Imigereka n’Insanganyamatsiko ni Ultimate Multisite add-on ikomeye ya **premium** iguha kugenzura byuzuye uburyo abakoresha urusobe rwawe babona kandi bakorana n’amahitamo y’imigereka n’insanganyamatsiko ku rusobe rwawe.

Iyi add-on ifite intego eshatu:

  1. Igufasha guhindura burundu metadata y’imigereka n’insanganyamatsiko. Ibyo birimo izina, ibisobanuro, izina ry’umwanditsi, ndetse na thumbnails!

  2. Igufasha gushyira imigereka n’insanganyamatsiko mu byiciro, kugira ngo abakoresha bawe babashe kubitondeka mu buryo bworoshye.

  3. Iguha amahitamo yo gusimbuza ipaji ishaje kandi irambiranye y’imigereka n’ipaji nshya imeze nk’iy’insanganyamatsiko ifite gutondeka hakurikijwe ibyiciro.

Nyuma yo kuyikurura no kuyishyiraho, tab nshya - _Igenamiterere rya add-on_ \- izagaragara ku ipaji y’Igenamiterere rya Ultimate Multisite.
