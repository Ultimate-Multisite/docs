# Fluent Forms

Iyi nyongera ituma bishoboka kongeramo imipaka kuri Fluent Forms kuri Multisite Ultimate mu bicuruzwa, mu banyamuryango cyangwa ku mbuga.
Fluent Forms ikoresha imbonerahamwe zihariye aho gukoresha ubwoko bw’inyandiko bwihariye, bityo amahitamo asanzwe yo gushyiraho imipaka ntakorana na Fluent Forms.
