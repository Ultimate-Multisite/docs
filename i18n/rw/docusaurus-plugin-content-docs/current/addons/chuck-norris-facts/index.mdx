# Amakuru ya Chuck Norris {#chuck-norris-facts}

Ongeramo urwenya ruke mu mirimo yawe ya buri munsi yo kuyobora WordPress ukoresheje addon ya **Ultimate Multisite - Amakuru ya Chuck Norris**. Iyo ikoreshejwe, uzabona amakuru ya Chuck Norris atoranyijwe ku buryo butunguranye agaragara mu nguni yo hejuru iburyo kuri buri paji y’ubuyobozi.

Biratunganye mu gutuma ikipe yawe iseka, koroshya umwuka mu gihe cy’amasomo maremare y’ubuyobozi, cyangwa gusa kwizihiza ubuhangange bw’akataraboneka bwa Chuck Norris.

## Kuki Amakuru ya Chuck Norris? {#why-chuck-norris-facts}

* **Kongera Umwete w’Ikipe**: Ongeramo ikintu gishimishije mu buyobozi bwa network yawe
* **Gukuraho Ubukonje**: Ni intangiriro nziza y’ikiganiro ku banyamuryango bashya b’ikipe
* **Kongera Umusaruro**: Guseka gato byihuse bishobora kunoza kwibanda n’umusaruro
* **Urwenya Rutajya Rusaza**: Amakuru ya Chuck Norris ntajya asaza

## Ibiranga by’ingenzi {#key-features}

**Amakuru 110+ y’Umunyabigwi**
Icyegeranyo cyuzuye cy’urwenya n’amakuru ya kera ya Chuck Norris yizihiza uwo munyabigwi ubwe.

**Kugaragara Kudabangamira**
Amakuru agaragara muri admin bar mu buryo butabangamira imigendekere y’akazi kawe.

**Yiteguye Guhindurwa mu zindi ndimi**
Amakuru yose apfunyitswe muri translation functions, bigatuma byoroha kuyahindura mu rurimi rwawe cyangwa gukora amakuru yihariye y’ikipe yawe.

**Yubahiriza Neza Ingano z’Ekrani**
Amakuru agaragara neza kuri ekrani za desktop, tablet, na mobile - Chuck Norris ni igitangaza ku bikoresho byose.

**Inkunga ya RTL**
Inkunga yuzuye ku ndimi zandikwa uhereye iburyo ujya ibumoso kuko Chuck Norris arenga inzitizi z’indimi.

**Ijyana na Block Editor**
Amakuru ahishwa mu buryo bw’ubwenge muri block editor kugira ngo hirindwe amakimbirane y’imigaragarire.

**Yibanda ku Bworoherezwe bwo Gukoresha**
Irimo inyandiko ikwiriye ya screen reader kugira ngo yubahirize byuzuye ibisabwa by’ubworoherezwe bwo gukoresha.

## Bibereye {#perfect-for}

* Abayobozi ba network za multisite bashaka kongeramo umwihariko
* Amatsinda y’iterambere akunda urwenya rwiza
* Umuntu wese ukeneye igipimo cya buri munsi cy’ubuhangange bwa Chuck Norris
* Imbuga zishaka kurema ahantu ho gukorera hashimishije kandi heza

## Kwinjiza {#installation}

1. Menya neza ko plugin ya Ultimate Multisite yinjijwe kandi yakoreshejwe kuri network yawe ya WordPress
2. Shyira dosiye za addon muri directory yawe ya `/wp-content/plugins/`
3. Koresha plugin kuri network unyuze muri menu ya 'Plugins' muri WordPress
4. Ishimire amakuru ya Chuck Norris atoranyijwe ku buryo butunguranye kuri buri paji y’ubuyobozi!

Nta igenamiterere rikenewe - yikoreshe gusa maze useke!

## Ibibazo Bikunze Kubazwa {#frequently-asked-questions}

## Ese ibi bisaba Ultimate Multisite? {#does-this-require-ultimate-multisite}

Nubwo byakozwe nka addon ya Ultimate Multisite, plugin ikora yigenga kandi ishobora gukora kuri installation iyo ari yo yose ya WordPress.

## Nshobora kongeramo amakuru yanjye bwite? {#can-i-add-my-own-facts}

Yego! Amakuru yiteguye guhindurwa mu zindi ndimi. Ushobora gukora dosiye yawe bwite ya `.po` irimo amakuru yihariye maze ukayishyira muri directory ya `/lang`. Cyangwa ushobora kuyungurura facts array ukoresheje porogaramu.

## Ese ibi bidindiza admin yanjye? {#does-this-slow-down-my-admin}

Oya rwose! Plugin iroroshye cyane kandi nta ngaruka igira ku mikorere. Yerekana inyandiko gusa - nta guhamagara API zo hanze cyangwa gutunganya ibintu biremereye.

## Nshobora kubizimya kuri paji zimwe na zimwe? {#can-i-disable-it-on-certain-pages}

Amakuru ahishwa mu buryo bwikora kuri paji aho ashobora kubangamira (nko muri block editor). Ushobora kandi gukoresha WordPress hooks kugira ngo uhindure aho amakuru agaragara.

## Ese amakuru akwiriye abantu bose? {#are-the-facts-appropriate-for-all-audiences}

Yego! Amakuru yose ni urwenya rusukuye, rubereye umuryango, rwizihiza icyubahiro cy’akataraboneka cya Chuck Norris.

## Nshobora guhindura amakuru mu rundi rurimi? {#can-i-translate-the-facts-to-another-language}

Rwose! Amakuru yose akoresha WordPress translation functions. Kora dosiye y’ubuhinduzi y’ururimi rwawe maze uyishyire muri directory ya `/lang`. Text domain: `ultimate-multisite-chuck-norris-facts`

## Ese ikorana na multisite? {#does-it-work-with-multisite}

Yego! Yakozwe by’umwihariko ku ma-network ya multisite kandi ishobora gukorerwa kuri network.
