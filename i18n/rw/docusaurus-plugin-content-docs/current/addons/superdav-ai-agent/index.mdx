# Superdav AI Agent

Superdav AI Agent ni umufasha wa AI wa WordPress wateye imbere, wagura Gratis AI Agent akongeramo ubumenyi bujyanye na theme, ubushobozi bwongerewe bwo gushakisha kuri internet, ibiranga kubaka urubuga, n’imikoranire y’automation ishobora kongerwamo. Verisiyo 1.18.0 yongeramo ibikoresho byo gusoma Google Calendar, guhuza contact n’abitabira, amarembo yo kwemeza n’umuntu, gukuraho kwibutsa byisubiramo, imenyesha rya SMS rya TextBee, gutegura service ya Superdav icungwa, n’ipaki ya Advanced companion itangwa ukwabo igenewe imikorere y’abatekinisiye bizewe.

## Ibiranga by’ingenzi {#key-features}

- **Ubumenyi bujyanye na theme** — buhuza amabwiriza n’ubushobozi mu buryo bwikora hashingiwe kuri theme ya WordPress iri gukora n’ama plugins yashyizwemo
- **Abatanga serivisi benshi bo gushakisha kuri internet** — hitamo hagati ya Brave Search na Tavily kugira ngo ubone ibisubizo by’ishakisha bikungahaye n’ubushobozi bw’ubushakashatsi bwimbitse
- **Ama contact forms y’umwubatsi w’urubuga** — kora contact forms zikora neza mu buryo bwuzuye uhereye muri chat interface ukoresheje form plugin yashyizwemo
- **Umwubatsi w’urubuga wongerewe ubushobozi** — gahunda zinoze zo kubaka urubuga mu byiciro byinshi, zifite gutahura plugins neza kurushaho no kwisubiza mu gihe habaye amakosa
- **Ubumenyi bwa theme bwubatswemo** — amabwiriza yihariye kuri Block Themes, Classic Themes, Kadence Blocks, na Kadence Theme
- **Imikoranire ya WooCommerce yanozwe** — yavuguruwe kugira ngo ikoreshe WooCommerce APIs kavukire, itange kwizerwa no gukorana neza kurushaho
- **Gutahura provider mu buryo bwikora** — itahura kandi ivugurura mu buryo bwikora plugins na themes bihari igihe bikoreshejwe cyangwa bihagaritswe
- **Gukora logo yihariye** — kora kandi ushyiremo logo za SVG zihariye, hamwe no gusukura mu buryo bwikora butekanye kuri namespace
- **Kugenzura contrast ya palette** — genzura amabara abiri kugira ngo yubahirize WCAG accessibility mbere yo kuyashyira kuri theme yawe
- **Ivumbura ryongerewemo amafoto** — shyiraho amafoto mu kiganiro cy’ivumbura cya Theme Builder kugira ngo agire uruhare ku cyerekezo cy’igishushanyo
- **Menus z’ubwakiranyi** — kora amapaji ya menu y’ibiribwa n’ibinyobwa ateguye neza agenewe restaurants, cafes, n’ubucuruzi bw’ubwakiranyi
- **Previews za desktop na mobile** — reba uko igishushanyo cyawe kizagaragara kuri desktop no ku bikoresho bya mobile mu gihe cyo guhitamo icyerekezo cy’igishushanyo
- **Gusoma imbuga kwa Tier 1** — sd-ai-agent/site-scrape ubu iboneka ku buryo busanzwe bwo gusesengura imbuga z’abanywanyi n’izitanga inspiration
- **Ibikoresho byo gusoma Google Calendar** — soma calendars n’ibikorwa byateguwe kugira ngo automations zishobore gutekereza ku bijyanye na schedule iri imbere
- **Guhuza contact n’abitabira** — huza abitabira muri calendar n’abakoresha urubuga bazwi na contacts mbere yo gukora imirimo ikurikiraho
- **Amarembo yo kwemeza n’umuntu** — hagarika automations zoroshye kwangirika kugira ngo zisuzumwe mbere yo kohereza ubutumwa, gukomeza akazi, cyangwa gukora impinduka
- **Inyandiko zo kwibutsa no gukuraho ibyasubiwemo** — kurikirana reminders zamaze koherezwa kugira ngo imikorere ya automation isubiyemo itamenyesha uwakira umwe kabiri
- **Imenyesha rya SMS rya TextBee** — ohereza imenyesha rya text-message ryateguwe binyuze muri TextBee igihe SMS provider yashyizweho
- **Gutegura service ya Superdav icungwa** — huza imbuga zishyigikiwe na endpoints za Superdav zakiriwe, hakoreshejwe setup yikora aho gukora endpoint configuration intoki
- **Ipaki ya Advanced companion** — ibikoresho by’abatekinisiye bizewe bitangwa ukwabyo bigenewe imikorere ya filesystem, database, WP-CLI, REST dispatcher, plugin builder, git snapshot, user-management, na benchmark

## Ibisabwa {#requirements}

- WordPress 6.0 cyangwa irenzeho
- PHP 8.1 cyangwa irenzeho
- WordPress Multisite (network-activated) cyangwa installation y’urubuga rumwe
- AI provider connector plugin ikora (Anthropic, OpenAI, cyangwa ihuye na byo)
- Gratis AI Agent v1.9.0 cyangwa irenzeho (Superdav AI Agent yagura Gratis AI Agent)

## Gushyiraho {#installation}

1. Shyira dosiye za plugin muri `/wp-content/plugins/superdav-ai-agent`
2. Koresha plugin unyuze kuri ecran ya **Plugins** muri WordPress
3. Menya neza ko Gratis AI Agent na yo yakoreshejwe
4. Jya kuri **Gratis AI Agent → Settings** maze utegure internet search provider yawe, managed Superdav service connection, Google Calendar, cyangwa TextBee credentials uko bikenewe

Ipaki ya WordPress.org irimo ibiranga by’ibanze bya Superdav AI Agent. Ipaki ya Advanced companion yubakwa kandi igatangwa ukwabo ku bidukikije aho ibikoresho by’abatekinisiye bizewe bishyirwaho kandi bikemererwa mu buryo bweruye.

## Ubumenyi bujyanye na theme {#theme-aware-skills}

Superdav AI Agent itahura mu buryo bwikora theme yawe iri gukora n’ama plugins yashyizwemo, igashyira ubumenyi bukwiye bujyanye na theme muri context y’umufasha. Ubumenyi buhari burimo:

| Ubumenyi | Bukoreshwa kuri |
|---|---|
| **Block Themes** | Themes zikoresha imyubakire ya WordPress ishingiye kuri block (Full Site Editing) |
| **Classic Themes** | Themes gakondo za WordPress zikoresha PHP templates na classic editor |
| **Kadence Blocks** | Imbuga zifite plugin ya Kadence Blocks ikora |
| **Kadence Theme** | Imbuga zifite Kadence theme nka theme iri gukora |

Reba [Ubumenyi bujyanye na theme](../../user-guide/configuration/theme-aware-skills) kugira ngo ubone amabwiriza arambuye kuri buri bumenyi.

## Abatanga serivisi zo gushakisha kuri internet {#internet-search-providers}

Superdav AI Agent v1.10.0 yongeramo **Tavily** nk’ubundi buryo bwa Brave Search ku bibazo byo gushakisha kuri internet. Providers zombi zitanga ibisubizo by’ishakisha bya real-time kandi zishobora gutegurwa muri **Gratis AI Agent → Settings → Advanced**.

| Utanga serivisi | Ibiranga | Gutunganya |
|---|---|---|
| **Brave Search** | Ibisubizo by’ishakisha byihuta kandi byita ku bwite | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Ibisubizo by’ishakisha bikungahaye kurushaho, ubushobozi bw’ubushakashatsi buteye imbere | [Tavily API](https://tavily.com/) |

Reba [Ishakisha rya Internet](../../user-guide/configuration/internet-search) ku mabwiriza yo gutunganya.

## Imigendekere y’imirimo ya Calendar, Kwemeza, n’Itangazo {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 yagura imigendekere y’imirimo yikora ikarenga imirimo y’ibirimo no kubaka site:

- **Ibikoresho bya Google Calendar** bituma agent isoma calendars zatunganijwe n’ibisobanuro by’ibikorwa igihe ibyangombwa bya calendar bihari.
- **Abafasha b’abo kuvugana n’abitabiriye** bahuza abitabiriye igikorwa n’abakoresha WordPress cyangwa abo kuvugana bazwi mbere yo gutegura ibyibutsa cyangwa imirimo yo gukurikirana.
- **Amarembo yo kwemeza k’umuntu** yemerera ikora ryikora guhagarara kugeza umukoresha wemerewe asuzumye igikorwa giteganyijwe.
- **Inyandiko z’ibyibutsa** zirinda ibyibutsa bya calendar cyangwa amatangazo kwigana igihe ikora ryikora ryongeye kugeragezwa cyangwa ryateguwe kenshi.
- **Ubufasha bwa TextBee SMS** bushoboza amatangazo y’ubutumwa bugufi igihe umutanga serivisi wa TextBee yatunganijwe.

Koresha ibi bikoresho ku bufasha buzi gahunda, gukurikirana abitabiriye, n’imigendekere y’imirimo y’ibyibutsa bikomeza gusaba isuzuma ry’umuntu mbere y’uko ubutumwa bworoshye bwohererezwa.

## Iboneza {#configuration}

Igenamiterere ryose riri munsi ya **Gratis AI Agent → Settings** muri admin ya WordPress. Superdav AI Agent yongeramo amahitamo akurikira y’iboneza:

| Igenamiterere | Ibisobanuro |
|---|---|
| **Internet Search Provider** | Hitamo hagati ya Brave Search na Tavily (si ngombwa) |
| **Tavily API Key** | API key ya Tavily internet search (niba Tavily yatoranyijwe) |
| **Managed Superdav Service** | Huza sites zishyigikiwe na serivisi ya Superdav yakiriwe hamwe no gutanga endpoints mu buryo bwikora |
| **Google Calendar Credentials** | Tunganya uburyo bwo kugera kuri calendar kugira ngo agent ishobore gusoma calendars n’ibikorwa |
| **TextBee SMS Provider** | Tunganya ibyangombwa bya TextBee ku migendekere y’imirimo y’amatangazo ya SMS yemejwe |

## Advanced Companion Package {#advanced-companion-package}

Superdav AI Agent v1.18.0 yongeramo Advanced companion package itangwa ukwandi ku migendekere y’imirimo y’abateza imbere bizewe. Ubu bushobozi ntibupakirwa muri core package ya WordPress.org. Shyira companion package gusa kuri sites aho abayobozi bizera mu buryo bweruye ko agent igera ku bikoresho by’abateza imbere.

Ibikoresho bya Advanced birimo ibikorwa bya filesystem, kugera kuri database, WP-CLI, imigendekere y’imirimo ya REST dispatcher, utilities za plugin builder, git snapshots, abafasha bo gucunga abakoresha, n’ibikoresho bya benchmark. Gumana ibi bikoresho ku bayobozi bizewe gusa kandi usuzume iboneza rya approval-gate mbere yo kubikoresha kuri sites ziri mu musaruro.
