# Abakozi b'ubufasha

Gucunga urusobe rwose wenyine si umurimo woroshye. Uko urusobe rwawe rutangira gukura, uzatangira kubona ubwiyongere bw'ubusabe bw'ubufasha. Byaba ari igitekerezo cyiza kuzana abantu bazagufasha kugabanya umutwaro w'akazi ufite. Icyakora, gutanga uburenganzira/buruhushya busesuye birimo ibyago kandi bishobora gushyira urusobe rwawe mu kaga. Aho ni ho iyi add-on y'Umukozi w'ubufasha iza ikenewe.

Iyi add-on igufasha gukora ubwoko bushya bw'ukoresha WordPress bwitwa Umukozi w'ubufasha, kandi ikaguha igenzura ryuzuye ku burenganzira uwo mukoresha afite. Gutanga uruhushya biroroshye nko gufungura buto ya toggle, no kuyifunga igihe uhisemo kurukuraho.

## Ibiranga

- Urutonde rurambuye rw'uruhushya rwa WordPress & Ultimate Multisite
- Fungura & funga uburenganzira bwo kugera ku buryo bworoshye
- Kora konti z'Abakozi zitagira umubare ntarengwa

## Kwinjiza

1. Shyira 'ultimate-multisite-support-agents' muri diregiteri ya '/wp-content/plugins/'
2. Koresha plugin unyuze muri menyu ya 'Plugins' muri WordPress
3. Kurikiza Wizard intambwe ku yindi kugira ngo ushyire plugin ku murongo
