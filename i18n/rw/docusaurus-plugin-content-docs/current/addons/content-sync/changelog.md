# Amateka y'impinduka ya Content Sync

### Verisiyo 1.1.0 - Yasohotse ku wa 2025-12-31 {#version-110---released-on-2025-12-31}
* Gishya: Gushyigikira guhuza ubwoko bwose bw'inyandiko (inyandiko, paji, ubwoko bw'inyandiko bwihariye)
* Gishya: Kuyungurura ubwoko bw'inyandiko mu ntambwe yo guhitamo ibirimo
* Gishya: Intambwe yo guhitamo urubuga nk'isoko - huza uhereye ku rubuga urwo ari rwo rwose, atari urubuga nyamukuru gusa
* Gishya: Guhitamo mbere urubuga rw'umwana ku mbuga z'icyitegererezo
* Gishya: Gukuramo media muri post_content ku bwoko bwose bw'inyandiko
* Gishya: Guhuza taxonomy (ibyiciro, tags, custom taxonomies)
* Byahinduwe: Izina ryahinduwe riva kuri "Elementor Template Sync" riba "Content Sync"
* Byahinduwe: Intambwe ya "Select Templates" yahinduwe izina iba "Select Data"
* Byanonosowe: Code yihariye ya Elementor ubu ikora gusa ku bwoko bw'inyandiko bwa elementor_library

### Verisiyo 1.0.0 - Yasohotse ku wa 2025-12-19 {#version-100---released-on-2025-12-19}
* Gusohora kwa mbere
* Wizard y'intambwe nyinshi yo guhitamo icyitegererezo n'urubuga
* Gutunganya mu matsinda hamwe n'ibitekerezo by'iterambere mu gihe nyacyo
* Guhuza media byuzuye hamwe no kongera kugena ID mu buryo bwikora
* Guhuza igenamiterere na Ultimate Multisite
* Gushyigikira gukuramo urubuga hakoreshejwe igenamiterere rya network
* Gusiba cache ya Elementor nyuma yo guhuza
* Kubungabunga taxonomy y'icyitegererezo
