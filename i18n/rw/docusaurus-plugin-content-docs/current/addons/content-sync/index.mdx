# Guhuza Ibirimo

Multisite Ultimate: Guhuza Ibirimo

Iyi nyongera ya Multisite Ultimate yemerera abayobozi b’urusobe guhuza ibirimo kuva ku rubuga urwo ari rwo rwose bijya ku zindi mbuga ziri mu rusobe. Ishyigikira ubwoko bwose bw’inyandiko harimo inyandiko, amapaji, ubwoko bw’inyandiko bwihariye, n’inyandikorugero za Elementor. Iyi nyongera irimo interinetse ya wizard ifite intambwe nyinshi yo guhitamo ibirimo n’imbuga zigenewe, hamwe no gutunganya mu matsinda n’ibisubizo by’iterambere mu gihe nyacyo.

## Ibiranga

* Guhitamo Ibirimo: Urusobe rw’amakarita agaragara rufite amashusho mato n’iyungurura ry’ubwoko bw’inyandiko
* Ubwoko Bwose bw’Inyandiko: Huza inyandiko, amapaji, ibicuruzwa, inyandikorugero za Elementor, n’ubwoko bwose bw’inyandiko bwihariye
* Guhitamo Urubuga: Hitamo imbuga ushaka guhozaho ibirimo, hamwe no gukuramo urubuga rw’inkomoko no guhitamo mbere imbuga z’abana
* Gutunganya mu Matsinda: Bihuza ku mbuga 10 icyarimwe hamwe n’amakuru y’iterambere
* Guhuza Media: Bikopa kandi bikongera guhuza mu buryo bwikora amadosiye ya media (amashusho, videwo) kuva mu birimo by’inyandiko ajya ku mbuga zigenewe
* Guhuza Taxonomy: Bibika ibyiciro, tagi, na taxonomies zihariye
* Ubufasha bwa Elementor: Ubufasha bwuzuye ku nyandikorugero za Elementor hamwe no kongera kubyaza CSS no kongera guhuza ID z’inyandikorugero
* Gukemura Amakimbirane: Ihitamo ryo gusimbuza ibirimo bihari cyangwa kubisimbuka
* Gucunga Cache: Bisiba mu buryo bwikora cache ya Elementor ku mbuga zigenewe nyuma yo guhuza

## Ibisabwa

* WordPress 5.3 cyangwa irenga
* PHP 7.4 cyangwa irenga
* plugin ya Multisite Ultimate (ikora)
* plugin ya Elementor (ntabwo ari ngombwa, isabwa ku guhuza inyandikorugero za Elementor)

## Kwinjiza

1. Ohereza 'ultimate-multisite-content-sync' muri directory ya '/wp-content/plugins/'
2. Kora plugin ukoresheje menu ya 'Amacomeka' muri Ubuyobozi bw’Urusobe bwa WordPress bwawe
3. Hindura igenamiterere muri Ultimate Multisite → Igenamiterere → Guhuza Ibirimo
4. Jya kuri Ultimate Multisite → Guhuza Ibirimo kugira ngo utangire guhuza
