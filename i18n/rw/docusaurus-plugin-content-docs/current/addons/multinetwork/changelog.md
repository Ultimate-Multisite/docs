# Urutonde rw'impinduka rwa Multi-Network {#multi-network-changelog}

### 1.3.0 {#130}
* Gishya: Urebera inyandikorugero y'urusobe — reba kandi ugerageze kureba inyandikorugero z'urusobe ukoresheje panel ib live mbere yo kugura.
* Gishya: Uhitamo inyandikorugero y'urusobe kuri checkout — abakiriya bashobora guhitamo inyandikorugero y'urusobe mu gihe cy'inzira ya signup checkout.
* Gishya: Cross-network magic link SSO — abakiriya bashobora kugera kuri sub-networks mu buryo bworoshye binyuze muri magic-link authentication.
* Gishya: Gukoporora ibitangazamakuru by'urusobe — ibitangazamakuru bya site bikopororwa neza iyo ukoporora imirongo y'urusobe uhereye ku nyandikorugero.
* Ikosora: Gukoporora urusobe ubu bikora hatabayeho multi-tenancy addon; uburenganzira bwa super admin bwongera gutangwa nyuma yo gukoporora.
* Ikosora: Gukora urusobe ubu bihuzwa na WooCommerce n'ama payment gateways yose.
* Ikosora: Nimero ya port ubu ishyirwa mu gukora fallback domain.
* Ikosora: CSS y'uhitamo inyandikorugero yimuriwe muri stylesheet ishyirwa ku murongo kugira ngo igaragare neza kuri themes zose.
* Ikosora: Kurinda `network_id` key itasobanuwe muri scope y'ibibazo bya site.
* Ikosora: Kugenzura uburenganzira bw'urusobe mbere yo guhindura context birinda amakosa y'ibiranga table ya `wpdb`.
