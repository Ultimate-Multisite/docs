# Dokumentasyon ng Ultimate Multisite {#ultimate-multisite-documentation}

Maligayang pagdating sa dokumentasyon ng Ultimate Multisite. Ginagawang ganap na Website as a Service (WaaS) platform ng Ultimate Multisite ang WordPress Multisite.

## Mga Seksyon ng Dokumentasyon {#documentation-sections}

### [Gabay ng Gumagamit](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

Lahat ng kailangan mo para i-set up, i-configure, at pamahalaan ang iyong WaaS platform. Saklaw nito ang pag-install, mga payment gateway, domain mapping, pamamahala ng client, at higit pa.

### [Dokumentasyon para sa Developer](/developer/hooks) {#developer-documentation}

Teknikal na sanggunian para sa mga developer na bumubuo sa Ultimate Multisite. Kasama ang sanggunian ng hooks, dokumentasyon ng REST API, at mga gabay sa paggawa ng addon.

### [Mga Addon](/addons) {#addons}

Dokumentasyon para sa mga bayad na addon plugin na nagpapalawak sa Ultimate Multisite gamit ang mga karagdagang feature tulad ng integrasyon ng WooCommerce, analytics, pagbebenta ng domain, at higit pa.

## Pagkuha ng Tulong {#getting-help}

- [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) -- Magtanong at magbahagi ng mga ideya
- [GitHub Issues](https://github.com/Ultimate-Multisite/ultimate-multisite/issues) -- Mag-ulat ng mga bug
- [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) -- I-download ang core plugin
