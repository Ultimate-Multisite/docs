# VAT

Ultimate Multisite: VAT sa Europa

Nagdaragdag ng karagdagang mga tampok at kasangkapan sa default na paghawak ng buwis ng Ultimate Multisite upang mas masuportahan ang VAT sa Europa.
Awtomatikong nagdaragdag ng mga rate ng EU VAT sa pahina ng Mga Rate ng Buwis mula sa

## Pag-install

1. I-upload ang 'ultimate-multisite-vat' sa direktoryong '/wp-content/plugins/'
2. I-activate ang plugin sa pamamagitan ng menu na 'Mga Plugin' sa WordPress
3. Pumunta sa Ultimate Multisite > Mga Setting > Mga Buwis at Paganahin ang EU VAT.
4. Pindutin ang Mga Rate ng Buwis at I-update ang Mga Rate ng EU VAT sa ibaba at I-save ang Mga Rate ng Buwis.
5. Tapos na!
