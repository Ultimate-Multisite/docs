# Site Exporter {#site-exporter}

## Nilalaman {#content}

Nauunawaan namin ang pangangailangang mailipat ang mga subsite sa loob ng iyong network papunta sa isang single site installation at kabaliktaran kung magpasya ang iyong customer na ilipat ang kanilang website. Pinapadali ng Site exporter and importer addon ang gawaing ito. Ang tool na ito ay nag-e-export at nag-i-import ng mga site gamit ang mga zip file na maaari ring magsilbing backup.

## Mga Tampok {#features}

* Mag-export at Mag-import ng mga Site

* Madaling-gamitin na interface

* Gumawa ng backup ng site

## Dokumentasyon {#documentation}
**Paano Mag-export at Mag-import ng Site**

## Pag-install {#installation}

1. I-upload ang **ultimate-multisite-site-exporter** sa directory na **/wp-content/plugins/**
2. I-activate ang plugin sa pamamagitan ng menu na **Plugins** sa WordPress
3. Sundin ang step by step na Wizard upang i-set up ang plugin

## Paano mag-export ng site {#how-to-export-a-site}

Upang mag-export ng site mula sa iyong network, pumunta sa **Export & Import** sa menu ng Ultimate Multisite at, sa tab na **Export Sites**, i-click ang **Generate new Site Export**.

<!-- Hindi available ang screenshot: tab na Export Sites na may Generate new Site Export button -->

Hanapin ang site na gusto mong i-export at piliin ang mga item na gusto mong isama sa na-export na file. Maaari mong isama ang mga theme, plugin, at upload. Kung masyadong malaki ang iyong site, maaari mong piliing patakbuhin ang proseso ng pag-export sa background.<!-- Hindi available ang screenshot: mga opsyon sa pag-export ng site na may paghahanap ng site, mga checkbox para isama ang themes/plugins/uploads -->

I-click ang button na **Export Site**. Magiging available ang iyong na-export na site sa listahang **Existing Exports**. I-click ang button na **Download ZIP** upang i-download ang file. Kasama sa mga Export ZIP ang self-booting na `index.php` at `readme.txt`, kaya maaari silang i-install sa bagong host nang hindi muna nag-i-install ng hiwalay na importer plugin.<!-- Hindi available ang screenshot: listahang Existing Exports na may Download ZIP button -->

## Paano i-export ang buong network {#how-to-export-the-whole-network}

Upang i-export ang lahat ng subsite nang magkakasama, pumunta sa **Export & Import** sa menu ng Ultimate Multisite at gamitin ang opsyong **Network Export** sa screen ng pag-export. Ang nabuong archive ay naglalaman ng lahat ng subsite sa iisang ZIP para sa mga paglipat ng host, staging rebuild, o mga backup sa antas ng network.

Maaaring malaki ang mga network export. Patakbuhin ang mga ito kapag mababa ang traffic at kumpirmahing may sapat na storage ang server para sa archive at anumang kasamang upload, plugin, at theme.

## Paano mag-import ng site {#how-to-import-a-site}

Upang mag-import ng site, pumunta sa tab na **Import a Site** at i-upload ang .zip file ng site na gusto mong i-import. Maaari mong palitan ang URL ng bago o panatilihin ang lumang URL sa pamamagitan ng pag-iwan sa espasyo na blangko. Maaari mo ring lagyan ng check ang kahon upang tanggalin ang .zip file mula sa iyong installation pagkatapos ng import. Pagkatapos ay i-click ang **Begin Import**.

<!-- Hindi available ang screenshot: tab na Import a Site na may zip upload field at Begin Import button -->

Maaaring tumagal nang kaunti ang proseso. Makakakita ka ng pending na import sa listahan sa ibaba ng importer.

<!-- Hindi available ang screenshot: pending na import sa listahan sa ibaba ng importer -->

Maaari mong ihinto ang proseso sa pamamagitan ng pag-click sa **Cancel Import**.

<!-- Hindi available ang screenshot: Cancel Import button upang ihinto ang proseso ng import -->

Pagkatapos makumpleto ang proseso, available na ang iyong site sa iyong network.

## Paano mag-import ng site sa single install {#how-to-import-a-site-on-a-single-install}

Upang mag-import ng site sa iyong single install, i-install at i-activate muna ang importer sa iyong site. Makikita mo ang link para i-download ito sa tab ng exporter.

<!-- Hindi available ang screenshot: tab ng Exporter na may link para i-download ang single-site importer -->

Makikita mo ang importer sa menu na **Tools**, sa kaliwa, sa subitem na **Export & Import**.

<!-- Hindi available ang screenshot: menu ng Single-site Tools na nagpapakita ng submenu na Export and Import -->

I-upload ang .zip file ng site na gusto mong i-import. Maaari mong palitan ang URL ng bago o panatilihin ang lumang URL sa pamamagitan ng pag-iwan sa espasyo na blangko. Maaari mo ring lagyan ng check ang kahon upang tanggalin ang .zip file mula sa iyong installation pagkatapos ng import. Pagkatapos ay i-click ang **Begin Import**.

<!-- Hindi available ang screenshot: single-site import form na may zip upload at Begin Import button -->

Maaaring tumagal nang kaunti ang proseso. Makakakita ka ng pending na import sa listahan sa ibaba ng importer.

<!-- Hindi available ang screenshot: single-site pending na import sa listahan -->

Maaari mong ihinto ang proseso sa pamamagitan ng pag-click sa **Cancel Import**.

<!-- Hindi available ang screenshot: Cancel Import button upang ihinto ang proseso ng import -->

Pagkatapos matapos ang proseso, mapapalitan ang iyong site ng site na kaka-import mo lang.
