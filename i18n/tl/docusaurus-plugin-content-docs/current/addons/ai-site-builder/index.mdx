# AI Tagabuo ng Site

ultimate-multisite-ai-site-builder

## Mga Kinakailangan {#requirements}

- WordPress 5.3 o mas mataas
- PHP 7.4 o mas mataas
- Multisite Ultimate plugin (aktibo)

## Pag-install {#installation}

1. I-upload ang mga file ng addon sa iyong direktoryong `/wp-content/plugins/`
2. I-activate ang plugin sa pamamagitan ng menu na 'Plugins' sa WordPress
3. I-configure ang mga setting ng addon sa network admin

## Pag-develop {#development}

### Pag-setup {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Pagsusuri {#testing}

Kasama sa addon ang isang komprehensibong framework sa pagsusuri:

- **Mga Unit Test**: Mga test na nakabatay sa PHPUnit kasama ang WordPress test suite
- **Base Test Class**: Nagbibigay ng mga helper method para sa mga karaniwang scenario ng test
- **Saklaw ng Test**: Naka-configure upang subaybayan ang saklaw ng code
- **Multisite Testing**: Tumatakbo ang mga test sa multisite environment

### Mga Pamantayan sa Code {#code-standards}

- **WordPress Coding Standards**: Ipinapatupad sa pamamagitan ng PHPCS
- **Compatibility sa PHP 7.4+**: Sinusuportahan ang mga modernong feature ng PHP
- **Static Analysis**: Integrasyon ng PHPStan para sa pagsusuri ng type
- **Modernisasyon ng Code**: Mga panuntunan ng Rector para sa mga upgrade ng PHP

### Build System {#build-system}

- **Pagproseso ng Asset**: Minification para sa mga CSS/JS file
- **Pagsasalin**: Pagbuo ng POT file
- **Pag-package**: Paglikha ng archive para sa distribusyon
- **Development Mode**: Mga hindi na-minify na asset para sa debugging

## Configuration {#configuration}

Sinusuportahan ng addon ang iba't ibang opsyon sa configuration na maa-access sa pamamagitan ng network admin interface.

## Mga Hook at Filter {#hooks-and-filters}

### Mga Action {#actions}

- `ultimate-multisite-ai-site-builder_init` - Pinaandar pagkatapos ng pagsisimula ng addon
- `ultimate-multisite-ai-site-builder_loaded` - Pinaandar pagkatapos ma-load ang lahat ng file ng addon

### Mga Filter {#filters}

- `ultimate-multisite-ai-site-builder_settings` - Baguhin ang mga setting ng addon
- `ultimate-multisite-ai-site-builder_enabled` - I-override ang status ng pag-enable/pag-disable ng addon
