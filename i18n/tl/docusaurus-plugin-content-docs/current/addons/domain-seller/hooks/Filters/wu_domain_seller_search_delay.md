# Filter: wu_domain_seller_search_delay

Sinasala ang domain search debounce delay sa millisecond.

Dagdagan ang halagang ito upang mabawasan ang mga API call sa mababagal na koneksyon.

## Mga Parameter

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $delay | `int` | Debounce delay sa millisecond. Default 500. |

### Mula noong

- 2.1.0
