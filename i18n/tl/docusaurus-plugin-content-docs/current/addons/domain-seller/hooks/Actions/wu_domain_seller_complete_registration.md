# Aksyon: wu_domain_seller_complete_registration

Gumagana kapag ang isang bayad ay lumipat sa katayuang dapat mag-trigger ng pagpaparehistro ng domain.

## Mga Parameter {#parameters}

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $payment | `\WP_Ultimo\Models\Payment` | Ang object ng bayad. |
| $old_status | `string` | Ang dating katayuan ng bayad bago ang paglipat. |
