# Loco Translate

Ang karagdagang module na Ultimate Multisite: Loco Translate Integration ay nagdadala ng mga kakayahang pagsasalin na partikular sa sityo sa iyong WordPress multisite network. Kung kinailangan mo na ang magkakaibang sityo sa iyong network na magkaroon ng magkakaibang salin ng parehong karagdagang module o tema, ang karagdagang module na ito ang perpektong solusyon.

**Ano ang Ginagawa ng Karagdagang Module na Ito**

Walang putol na isinasama ng karagdagang module na ito ang Ultimate Multisite sa sikat na karagdagang module na Loco Translate, na nagbibigay-daan sa bawat sub-sityo sa iyong network na panatilihin ang sarili nitong pasadyang mga salin. Nagpapatakbo ka man ng platapormang para sa maraming rehiyon, naglilingkod sa iba't ibang komunidad ng wika, o kailangan lang ng kakayahang umangkop sa pagsasalin sa buong network mo, ginagawa itong madali ng karagdagang module na ito.

**Perpekto Para Sa:**

* Mga negosyong may maraming rehiyon na may magkakaibang pangangailangan sa lokalisasyon
* Mga institusyong pang-edukasyon na naglilingkod sa sari-saring komunidad
* Mga platapormang SaaS na nag-aalok ng suporta sa maraming wika
* Mga ahensiya sa web na namamahala ng maraming sityo ng kliyente
* Anumang multisite network na nangangailangan ng kakayahang umangkop sa pagsasalin

## Pag-install

1. Tiyaking naka-install at naka-activate sa network ang Ultimate Multisite
2. I-install at i-activate ang karagdagang module na Loco Translate
3. I-upload ang karagdagang module na ito sa iyong direktoryo ng mga karagdagang module
4. I-activate sa network ang karagdagang module na Loco Translate Integration
5. Simulang gumawa ng mga salin na partikular sa sityo sa pamamagitan ng Loco Translate sa anumang sub-sityo

Iyon lang! Awtomatikong gumagana ang integrasyon nang walang anumang karagdagang pagsasaayos.

## Mga Madalas Itanong

## Kailangan ko bang magsaayos ng anuman pagkatapos ng pag-install?

Hindi! Awtomatiko ang integrasyon. Kapag aktibo na ang karagdagang module at ang Loco Translate, agad nang makapagsisimula ang mga sub-sityo na gumawa ng sarili nilang mga pasadyang salin.

## Ano ang mangyayari kung ang isang sub-sityo ay hindi gumawa ng mga pasadyang salin?

Awtomatikong gagamit ang sityo ng mga salin sa buong network bilang fallback. Walang masisira, at tuloy-tuloy na gagana ang lahat.

## Maa-access ba ng mga sub-sityo ang mga salin sa buong network?

Nakikita at nae-edit lamang ng mga regular na admin ng sityo ang mga salin ng sarili nilang sityo. Maa-access ng mga super admin ang parehong mga salin sa buong network at partikular sa sityo depende sa kung aling sityo ang tinitingnan nila.

## Gumagana ba ito sa parehong mga karagdagang module at tema?

Oo! Gumagana ang mga salin na partikular sa sityo para sa parehong mga karagdagang module at tema ng WordPress.

## Pababagalin ba nito ang aking multisite network?

Hindi. Gumagamit ang karagdagang module ng mahusay na paglo-load ng file na sumusuri lamang ng mga salin na partikular sa sityo kapag kinakailangan. Walang kapansin-pansing epekto sa pagganap.

## Maaari ko bang kopyahin ang mga salin ng network sa isang partikular na sityo bilang panimulang punto?

Oo! Habang ine-edit ang mga salin sa Loco Translate, madali kang makapagsisimula gamit ang mga salin ng network at maiangkop ang mga ito para sa bawat sityo.

## Gumagana ba ito sa WooCommerce at iba pang mga karagdagang module?

Oo naman! Anumang karagdagang module o tema na gumagamit ng mga karaniwang function ng pagsasalin ng WordPress ay gagana sa mga salin na partikular sa sityo.
