# Discovery Interview

Ang **Discovery Interview** ay ang unang hakbang sa proseso ng disenyo ng Theme Builder. Nangangalap ito ng impormasyon tungkol sa iyong site, negosyo, at mga kagustuhan sa disenyo upang lumikha ng isinapersonal na theme.

## Pangkalahatang-ideya {#overview}

Ginagabayan ka ng Discovery Interview sa isang serye ng mga tanong upang maunawaan ang iyong brand, target na audience, at mga layunin sa disenyo. Sa bersyong 1.16.0, kasama na ngayon sa interview ang isang **hakbang sa pag-upload ng larawan** na nagpapahintulot sa iyong magbigay ng biswal na inspirasyon para sa iyong disenyo.

## Mga Hakbang ng Interview {#interview-steps}

### 1. Impormasyon ng Negosyo {#1-business-information}

- Pangalan ng site at tagline
- Uri ng negosyo o industriya
- Pangunahing mga layunin ng negosyo
- Paglalarawan ng target na audience

### 2. Mga Kagustuhan sa Disenyo {#2-design-preferences}

- Ninanais na color scheme
- Estilo ng disenyo (moderno, klasiko, minimalist, mapaglaro, atbp.)
- Mga website o brand na inspirasyon
- Mga kailangang-kailangang elemento ng disenyo

### 3. Pag-upload ng Larawan (Bago sa v1.16.0) {#3-photo-upload-new-in-v1160}

Mag-upload ng isa o higit pang mga larawan upang maimpluwensyahan ang direksyon ng disenyo:

#### Mga Tinatanggap na Format {#accepted-formats}

| Format | Extension | Mga Detalye |
|--------|-----------|---------|
| **JPEG** | `.jpg`, `.jpeg` | Inirerekomenda para sa mga litrato |
| **PNG** | `.png` | Sumusuporta sa transparency |
| **WebP** | `.webp` | Modernong format, mas maliit na laki ng file |
| **GIF** | `.gif` | Mga static na larawan lamang (walang animation) |

#### Mga Kinakailangan sa File {#file-requirements}

- **Pinakamalaking laki ng file:** 10 MB
- **Pinakamababang sukat:** 400 × 300 pixels
- **Inirerekomendang sukat:** 1200 × 800 pixels o mas malaki
- **Aspect ratio:** Katanggap-tanggap ang anumang ratio

#### Paano Naiimpluwensyahan ng Mga Larawan ang Direksyon ng Disenyo {#how-photos-influence-design-direction}

Tinutulungan ng mga larawang ina-upload mo ang Theme Builder na maunawaan ang:

- **Color palette** — kinukuha ang mga nangingibabaw na kulay at ginagamit sa theme
- **Biswal na estilo** — moderno, rustic, minimalist, makulay, atbp.
- **Mood at atmospera** — propesyonal, malikhain, magiliw, marangya, atbp.
- **Mga elemento ng disenyo** — mga pattern, texture, at estilo ng komposisyon
- **Inspirasyon sa typography** — mga estilo ng font na bumabagay sa biswal na theme

### 4. Istruktura ng Nilalaman {#4-content-structure}

- Mga pangunahing page na kailangan (Home, About, Services, Contact, atbp.)
- Kinakailangan ang seksyon ng blog o balita
- Kinakailangan ang e-commerce na functionality
- Mga espesyal na feature o integrasyon

### 5. Pagsusuri at Kumpirmasyon {#5-review-and-confirmation}

- Buod ng lahat ng ibinigay na impormasyon
- Kumpirmasyon bago magsimula ang pagbuo ng disenyo

## Pinakamahuhusay na Kasanayan sa Pag-upload ng Larawan {#photo-upload-best-practices}

### Ano ang I-upload {#what-to-upload}

- **Inspirasyon ng brand** — mga logo, color swatch, o alituntunin ng brand
- **Mga site ng kakumpitensya** — mga screenshot ng mga site na hinahangaan mo
- **Mga mood board** — mga koleksyon ng mga larawang kumakatawan sa ninanais mong estetika
- **Mga larawan ng produkto** — kung mayroon kang mga produktong ipapakita
- **Mga larawan ng team o opisina** — para sa personal o propesyonal na branding
- **Mga lifestyle image** — kumakatawan sa iyong target na audience

### Ano ang Iiwasan {#what-to-avoid}

- **Mababang kalidad na mga larawan** — malalabo o pixelated na larawan
- **Mga hindi nauugnay na larawan** — mga larawang hindi kumakatawan sa iyong brand
- **Mga larawang maraming teksto** — mga screenshot na maraming teksto
- **Materyal na may copyright** — tiyaking may karapatan kang gamitin ang mga larawan
- **Mga animated GIF** — mga static na larawan lamang ang pinoproseso

### Mga Tip para sa Pinakamagandang Resulta {#tips-for-best-results}

1. **Mag-upload ng 2-3 larawan** — nagbibigay ng mas magandang konteksto ang maraming larawan
2. **Gumamit ng de-kalidad na mga larawan** — pinakamainam ang malinaw at maayos ang liwanag na mga larawan
3. **Isama ang pagkakaiba-iba** — halo ng kulay, estilo, at paksa
4. **Magbigay ng konteksto** — ilarawan nang maikli kung bakit mo pinili ang bawat larawan
5. **Maging tiyak** — kung gusto mo ang isang partikular na elemento, ituro ito

## Output ng Direksyon ng Disenyo {#design-direction-output}

Pagkatapos ng Discovery Interview, bumubuo ang Theme Builder ng:

- **Color palette** — primary, secondary, at accent na mga kulay
- **Typography** — mga pamilya ng font para sa headings at body text
- **Estilo ng layout** — spacing, alignment, at ayos ng component
- **Mga biswal na elemento** — mga icon, pattern, at pandekorasyong elemento
- **Mood board** — biswal na representasyon ng direksyon ng disenyo

## Mga Susunod na Hakbang {#next-steps}

Pagkatapos makumpleto ang Discovery Interview:

1. Suriin ang nabuong direksyon ng disenyo
2. Magbigay ng feedback o humiling ng mga pag-aayos
3. Magpatuloy sa hakbang na [Direksyon ng Disenyo](./design-direction.md)
4. Lumilikha ang Theme Builder ng iyong custom na theme
