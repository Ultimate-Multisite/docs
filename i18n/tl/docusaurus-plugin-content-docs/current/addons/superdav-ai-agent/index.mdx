# Superdav AI Agent {#superdav-ai-agent}

Ang Superdav AI Agent ay isang advanced na WordPress AI assistant na nagpapalawak sa Gratis AI Agent gamit ang mga kakayahang nakaayon sa theme, pinahusay na mga kakayahan sa paghahanap sa internet, mga feature ng tagabuo ng site, at opsyonal na mga integration para sa automation. Nagdaragdag ang bersyon 1.18.0 ng mga tool sa pagbasa ng Google Calendar, pagmamapa ng contact at attendee, mga gate ng pag-apruba ng tao, deduplication ng reminder, mga notification sa TextBee SMS, pinamamahalaang provisioning ng serbisyo ng Superdav, at isang hiwalay na ipinamamahaging Advanced companion package para sa mga workflow ng pinagkakatiwalaang developer.

## Pangunahing Mga Feature {#key-features}

- **Mga kakayahang nakaayon sa theme** — awtomatikong inaangkop ang gabay at mga kakayahan batay sa iyong aktibong WordPress theme at mga naka-install na plugin
- **Maramihang provider ng paghahanap sa internet** — pumili sa pagitan ng Brave Search at Tavily para sa mas mayamang mga resulta ng paghahanap at advanced na mga kakayahan sa pananaliksik
- **Mga contact form ng tagabuo ng site** — gumawa ng ganap na gumaganang mga contact form nang direkta mula sa interface ng chat gamit ang iyong naka-install na form plugin
- **Pinahusay na tagabuo ng site** — pinahusay na mga multi-phase na plano sa paggawa ng site na may mas mahusay na pagtuklas ng plugin at pagbawi mula sa error
- **Mga built-in na kakayahan sa theme** — espesyal na gabay para sa Block Themes, Classic Themes, Kadence Blocks, at Kadence Theme
- **Pinahusay na integration sa WooCommerce** — nirefactor upang gamitin ang mga native na WooCommerce API para sa mas mahusay na pagiging maaasahan at compatibility
- **Awtomatikong pagtukoy ng provider** — awtomatikong tinutukoy at nire-refresh ang mga available na plugin at theme kapag na-activate o na-deactivate ang mga ito
- **Pagbuo ng custom na logo** — bumuo at mag-embed ng mga custom na SVG logo na may awtomatikong namespace-safe na sanitisation
- **Pag-validate ng contrast ng palette** — suriin ang mga pares ng kulay para sa pagsunod sa accessibility ng WCAG bago ilapat sa iyong theme
- **Pagtuklas na pinahusay ng larawan** — mag-upload ng mga larawan habang nasa discovery interview ng Theme Builder upang maimpluwensiyahan ang direksyon ng disenyo
- **Mga menu para sa hospitality** — bumuo ng mga structured na pahina ng menu ng pagkain at inumin para sa mga restaurant, cafe, at negosyong hospitality
- **Mga preview sa desktop at mobile** — i-preview ang iyong disenyo sa mga desktop at mobile device habang pumipili ng direksyon ng disenyo
- **Tier 1 na pag-scrape ng site** — available na ngayon ang sd-ai-agent/site-scrape bilang default para sa pagsusuri ng mga website ng kakumpitensya at inspirasyon
- **Mga tool sa pagbasa ng Google Calendar** — basahin ang mga naka-configure na calendar at event upang makapag-isip ang mga automation tungkol sa paparating na konteksto ng schedule
- **Pagmamapa ng contact at attendee** — itugma ang mga participant sa calendar sa mga kilalang user at contact ng site bago bumuo ng follow-up na gawain
- **Mga gate ng pag-apruba ng tao** — i-pause ang mga sensitibong automation para sa pagsusuri bago magpadala ng mga mensahe, ipagpatuloy ang gawain, o gumawa ng mga pagbabago
- **Mga tala ng reminder at deduplication** — subaybayan ang mga reminder na naipadala na upang hindi ma-notify ng paulit-ulit na automation runs ang parehong recipient nang dalawang beses
- **Mga notification sa TextBee SMS** — magpadala ng mga naka-configure na text-message notification sa pamamagitan ng TextBee kapag naka-enable ang isang SMS provider
- **Pinamamahalaang provisioning ng serbisyo ng Superdav** — ikonekta ang mga sinusuportahang site sa mga hosted Superdav endpoint gamit ang awtomatikong setup sa halip na manu-manong configuration ng endpoint
- **Advanced companion package** — hiwalay na ipinamamahaging mga tool para sa pinagkakatiwalaang developer para sa filesystem, database, WP-CLI, REST dispatcher, tagabuo ng plugin, git snapshot, user-management, at mga workflow ng benchmark

## Mga Kinakailangan {#requirements}

- WordPress 6.0 o mas mataas
- PHP 8.1 o mas mataas
- WordPress Multisite (network-activated) o single-site na installation
- Isang aktibong AI provider connector plugin (Anthropic, OpenAI, o compatible)
- Gratis AI Agent v1.9.0 o mas mataas (pinapalawak ng Superdav AI Agent ang Gratis AI Agent)

## Pag-install {#installation}

1. I-upload ang mga file ng plugin sa `/wp-content/plugins/superdav-ai-agent`
2. I-activate ang plugin sa pamamagitan ng screen na **Plugins** sa WordPress
3. Tiyaking naka-activate din ang Gratis AI Agent
4. Pumunta sa **Gratis AI Agent → Settings** at i-configure ang iyong provider ng paghahanap sa internet, koneksyon sa pinamamahalaang serbisyo ng Superdav, Google Calendar, o mga credential ng TextBee ayon sa kailangan

Naglalaman ang WordPress.org package ng mga pangunahing feature ng Superdav AI Agent. Ang Advanced companion package ay binubuo at ipinamamahagi nang hiwalay para sa mga environment kung saan ang mga pinagkakatiwalaang tool ng developer ay tahasang naka-install at pinahintulutan.

## Mga Provider ng Paghahanap sa Internet {#internet-search-providers}

Nagdaragdag ang Superdav AI Agent v1.10.0 ng **Tavily** bilang alternatibo sa Brave Search para sa mga query sa paghahanap sa internet. Parehong nag-aalok ang dalawang provider ng real-time na mga resulta ng paghahanap at maaaring i-configure sa **Gratis AI Agent → Settings → Advanced**.

| Provider | Mga Tampok | Setup |
|---|---|---|
| **Brave Search** | Mabilis, nakatuon sa privacy na mga resulta ng paghahanap | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Mas mayamang mga resulta ng paghahanap, mga advanced na kakayahan sa pananaliksik | [Tavily API](https://tavily.com/) |

Tingnan ang [Paghahanap sa Internet](../../user-guide/configuration/internet-search) para sa mga tagubilin sa configuration.

## Mga Workflow ng Kalendaryo, Pag-apruba, at Notification {#calendar-approval-and-notification-workflows}

Pinalalawak ng Superdav AI Agent v1.18.0 ang mga automation workflow lampas sa mga gawain sa nilalaman at pagbuo ng site:

- **Mga tool ng Google Calendar** ay nagbibigay-daan sa agent na basahin ang mga naka-configure na kalendaryo at detalye ng event kapag available ang mga credential ng kalendaryo.
- **Mga helper para sa contact at attendee** ay nagma-map ng mga kalahok sa event sa mga WordPress user o kilalang contact bago maghanda ng mga paalala o follow-up na gawain.
- **Mga human approval gate** ay nagpapahintulot sa automation na huminto hanggang masuri ng isang awtorisadong user ang iminungkahing aksyon.
- **Mga tala ng paalala** ay pumipigil sa mga duplicate na paalala sa kalendaryo o notification kapag muling sinubukan o paulit-ulit na naka-schedule ang automation.
- **Suporta sa TextBee SMS** ay nagpapagana ng mga notification sa text message kapag naka-configure ang TextBee provider.

Gamitin ang mga tool na ito para sa tulong na may kamalayan sa iskedyul, follow-up sa attendee, at mga workflow ng paalala na nangangailangan pa rin ng pagsusuri ng tao bago ipadala ang mga sensitibong mensahe.

## Configuration {#configuration}

Ang lahat ng setting ay nasa ilalim ng **Gratis AI Agent → Settings** sa admin ng WordPress. Nagdaragdag ang Superdav AI Agent ng mga sumusunod na opsyon sa configuration:

| Setting | Paglalarawan |
|---|---|
| **Internet Search Provider** | Pumili sa pagitan ng Brave Search at Tavily (opsyonal) |
| **Tavily API Key** | API key para sa paghahanap sa internet ng Tavily (kung pinili ang Tavily) |
| **Managed Superdav Service** | Ikonekta ang mga sinusuportahang site sa hosted na Superdav service na may awtomatikong provisioning ng endpoint |
| **Google Calendar Credentials** | I-configure ang access sa kalendaryo upang mabasa ng agent ang mga kalendaryo at event |
| **TextBee SMS Provider** | I-configure ang mga credential ng TextBee para sa mga naaprubahang SMS notification workflow |

## Advanced na Kasamang Package {#advanced-companion-package}

Nagdaragdag ang Superdav AI Agent v1.18.0 ng hiwalay na ipinamamahaging Advanced na kasamang package para sa mga pinagkakatiwalaang workflow ng developer. Hindi naka-bundle ang mga kakayahang ito sa core package ng WordPress.org. I-install ang kasamang package lamang sa mga site kung saan tahasang pinagkakatiwalaan ng mga administrator ang agent na ma-access ang developer tooling.

Kasama sa mga advanced na tool ang mga operasyon sa filesystem, access sa database, WP-CLI, mga workflow ng REST dispatcher, mga utility ng tagabuo ng plugin, mga git snapshot, mga helper sa pamamahala ng user, at benchmark tooling. Panatilihing limitado ang mga tool na ito sa mga pinagkakatiwalaang administrator at suriin ang configuration ng approval gate bago paganahin ang mga ito sa mga production site.
