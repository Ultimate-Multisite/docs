# PayFast {#payfast}

Multisite Ultimate: Payfast Gateway

Idinaragdag ang Payfast - isang mabilis at ligtas na online payments gateway sa South Africa - bilang opsyon sa pagbabayad ng Multisite Ultimate.

## Pag-install {#installation}

1. I-upload ang 'ultimate-multisite-payfast' sa directory na '/wp-content/plugins/'
2. I-activate ang plugin sa pamamagitan ng menu na 'Plugins' sa WordPress
3. Sundin ang sunod-sunod na gabay upang i-set up ang plugin

## Gabay sa Paggamit {#usage-guide}

Ang PayFast ay isang nangungunang online payment processor para sa mga negosyo sa South Africa. Nakatuon ito sa pagtulong sa mga online merchant na lumago sa pamamagitan ng paggawa ng mga transaksyon na tuloy-tuloy at ligtas. Ayon sa kahilingan ng user ng Ultimate Multisite, nagpasya kaming gumawa ng add-on na magpapahintulot sa integrasyon sa PayFast. Ipapakita ng artikulong ito kung paano i-set up ang PayFast bilang iyong payment gateway upang makatanggap ng bayad sa iyong Ultimate Multisite network.

Ang unang hakbang ay i-install ang add-on.

# Paano i-install ang Payfast add-on: {#how-to-install-the-payfast-add-on}

Pumunta sa pahina ng mga add-on at piliin ang add-on na Ultimate Multisite: Payfast Gateway.

![Pahina ng mga add-on](/img/admin/addons-page.png)

Lilitaw ang window ng add-on. I-click ang Install Now.

![Pahina ng mga add-on na may opsyon sa pag-install](/img/admin/addons-page.png)

Pagkatapos ng pag-install, ire-redirect ka sa pahina ng mga plugin. Dito, i-click lang ang Network Activate at maa-activate ang _Payfast_ add-on sa iyong network.

# Pag-set Up ng PayFast {#setting-up-payfast}

Pagkatapos i-install ang add-on, maaari ka nang pumunta sa mga setting ng Ultimate Multisite at simulang i-configure ang PayFast.

Sa ilalim ng pahina ng **Ultimate Multisite settings**, makikita mo ang tab na "Payments". Sa ilalim ng tab na ito, dapat mong makita ang "**Payfast** " kasama ang toggle button upang i-enable o i-disable ang payment gateway na ito.

![Mga setting ng payment gateway](/img/config/settings-payment-gateways.png)Sige at i-click ang toggle button upang i-enable ito. Pagkatapos ay ie-expand nito ang kahon upang ipakita ang iba't ibang field na kailangan mong punan tulad ng **Mechant ID, Merchant Key at PassPhrase.** Makukuha mo ang impormasyong ito mula sa iyong **Payfast** account

<!-- Hindi available ang screenshot: Pinalawak na mga setting ng PayFast na nagpapakita ng mga field na Merchant ID, Merchant Key, at PassPhrase -->

Ang susunod na hakbang ay mag-login sa iyong PayFast account upang makuha ang impormasyong kailangan mo.

Pagka-login, makikita mo agad ang mga credential na kailangan mo mula sa dashboard ng iyong **Payfast** account.

<!-- Hindi available ang screenshot: Dashboard ng PayFast account na nagpapakita ng mga credential na Merchant ID at Merchant Key -->Maaari mo lang kopyahin at i-paste ang credential at punan ang mga detalye sa iyong mga setting ng Ultimate Multisite. Pagkatapos ay i-click ang button na **"Save settings** " upang i-save ang mga pagbabago.

<!-- Hindi available ang screenshot: Mga setting ng Ultimate Multisite PayFast na may napunang mga credential at Save button -->

Iyon lang, kumpleto na ang iyong integrasyon ng PayFast payment!
