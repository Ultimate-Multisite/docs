# Fluent Forms

Ginagawang posible ng add-on na ito na magdagdag ng mga limitasyon sa Fluent Forms sa Multisite Ultimate sa mga produkto, membership o site.
Gumagamit ang Fluent Forms ng custom tables sa halip na custom post types kaya hindi gumagana ang karaniwang mga opsyon sa paglimita sa Fluent Forms.

## Mga Tampok {#features}

- Magdagdag ng mga limitasyon para sa Fluent Forms o Conversational Fluent Forms
- Makakakuha ang mga customer ng magiliw na mensahe na humihikayat sa kanila na i-upgrade ang kanilang plan kapag naabot nila ang limitasyon
