# Filter: wu_limits_is_fluent_form_above_limit {#filter-wulimitsisfluentformabovelimit}

Sinusuri kung pinapayagan ang ibinigay na uri ng form sa plan na ito. Pinapahintulutan ang mga developer ng plugin na i-filter ang ibinabalik na value

## Mga Parameter {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $value | `bool` | Kung ang uri ng form ay lampas sa limitasyon o hindi |
| $form_type | `string` | Ang uri ng form na sinusuri |
| $form_count | `int` | Kasalukuyang bilang ng form |
| $quota | `int` | Ang pinapayagang quota |
