# Action: switch_network

Nagti-trigger kapag inililipat ang kasalukuyang konteksto ng network.

## Mga Parameter {#parameters}

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $new_network_id | `int` | ID ng network na nililipatan. |
| $old_network_id | `int` | ID ng dating kasalukuyang network. |
