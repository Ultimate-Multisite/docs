# Filter: networks_user_is_network_admin

Sinasala ang mga network kung saan administrator ang isang user.

## Mga Parameter

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $ | `array\|bool` | Listahan ng mga network ID o false kung walang mga network para sa user. |
| $ | `int` | User ID kung saan dapat ibalik ang mga network. |
