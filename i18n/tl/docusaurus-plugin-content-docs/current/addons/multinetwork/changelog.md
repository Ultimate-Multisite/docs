# Changelog ng Multi-Network

### 1.3.0 {#130}
* Bago: Previewer ng template ng network — mag-browse at mag-preview ng mga template ng network gamit ang live panel bago bumili.
* Bago: Tagapili ng template ng network sa checkout — maaaring pumili ang mga customer ng template ng network habang nasa daloy ng signup checkout.
* Bago: Cross-network magic link SSO — walang abalang maa-access ng mga customer ang mga sub-network sa pamamagitan ng magic-link authentication.
* Bago: Pag-clone ng media ng network — tama na nadodoble ang media ng site kapag nagki-clone ng mga network mula sa template.
* Ayos: Gumagana na ngayon ang pag-clone ng network kahit walang multi-tenancy addon; muling ibinibigay ang mga pribilehiyo ng super admin pagkatapos ng clone.
* Ayos: Tugma na ngayon ang paggawa ng network sa WooCommerce at lahat ng payment gateway.
* Ayos: Kasama na ngayon ang port number sa paggawa ng fallback domain.
* Ayos: Inilipat ang CSS ng tagapili ng template sa isang naka-enqueue na stylesheet para sa tamang pag-render sa lahat ng theme.
* Ayos: Proteksiyon laban sa hindi natukoy na `network_id` key sa saklaw ng query ng site.
* Ayos: Pinipigilan ng pagsusuri ng pahintulot sa network bago ang context switching ang mga error sa property ng table ng `wpdb`.
