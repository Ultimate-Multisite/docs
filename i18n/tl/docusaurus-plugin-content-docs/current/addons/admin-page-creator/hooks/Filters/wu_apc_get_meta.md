## Mga Parameter

| Name | Type | Description |
|------|------|-------------|
| $default_value | `string` | Ang default na halaga. |
| $meta_type | `string` | Uri ng meta data na kukunin. |
| $meta_key | `string` | Pangalan ng meta field. |
