# Filter: wu_apc_should_display_admin_menu {#filter-wuapcshoulddisplayadminmenu}

Payagan ang mga admin na itago ang mga pahina ng admin menu, kung natapos na nila ang proseso ng pag-customize.

## Mga Parameter {#parameters}

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $should_display_admin_menu | `bool` | Kung kailangan nating ipakita ang mga pahina. |
| $network_active | `bool` | Suriin kung aktibo ang plugin sa network admin. |
