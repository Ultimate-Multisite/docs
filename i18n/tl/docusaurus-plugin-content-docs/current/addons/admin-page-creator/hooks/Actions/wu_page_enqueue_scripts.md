# Action: wu_page_enqueue_scripts

Nagdaragdag ng custom na action dito upang magkaroon tayo ng lugar para i-enqueue ang mga script at style kung kinakailangan\ at tiyaking na-load lang ang mga ito sa tamang mga page

## Mga Parameter {#parameters}

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $ | `\WU_Page` | Ang kasalukuyang admin page |
