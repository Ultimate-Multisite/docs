# Admin Page Creator {#admin-page-creator}

## Nilalaman {#content}
Papayagan ka ng Admin Page Creator na madaling magdagdag ng mga bagong admin page at sub-page sa loob ng WordPress admin dashboard. Maaari itong gamitin upang gumawa ng welcome page o tutorial page na maaaring magsilbing landing page para sa mga bagong gumagamit. Maaari kang gumamit ng page builder tulad ng Beaver Builder, Elementor, Brizy, at Oxygen upang idisenyo ang mga pahina nang walang abala.

Nag-aalok din ito ng suporta para sa custom HTML, CSS at JavaScript, pati na rin ng access control batay sa plano at tungkulin ng gumagamit!

## Mga Tampok {#features}
* Suporta sa Custom HTML at CSS
* Suporta sa JavaScript
* PHP Content Mode
* Pahintulot batay sa tungkulin ng gumagamit
* Suporta sa mga page builder

## Pag-install {#installation}

1. I-upload ang **ultimate-multisite-admin-page-creator** sa direktoryong **/wp-content/plugins/**
2. I-activate ang dagdag sa pamamagitan ng menu na **Plugins** sa iyong WordPress Network Admin

## Gabay sa Paggamit {#usage-guide}

Bagama’t ginagawang napakadali ng WordPress ang pagdagdag ng nilalaman sa front end ng mga websayt, ang paggawa ng mga pahina at sub-page sa panig ng admin ay nangangailangan ng ilang pangunahing kasanayan sa pag-code. Layunin ng Ultimate Multisite: Admin Page Creator na dalhin ang kaparehong kadalian ng paggamit ng front end sa back end ng iyong mga network websayt.

## Bakit gamitin ang Admin Page Creator? {#why-use-admin-page-creator}

Marahil gusto mong mag-alok ng custom page sa admin panel ng iyong mga kliyente na may mga tutorial, o marahil gusto mong magdagdag ng custom page na may mga FAQ at contact widget. Walang hanggan ang mga posibilidad sa Admin Page Creator, at iyon ang ating susuriin sa tutorial ngayon.

## Paggawa ng iyong unang Custom Admin Page {#creating-your-first-custom-admin-page}

Pagkatapos i-install at i-activate ang Ultimate Multisite: Admin Page Creator, pumunta sa iyong network admin menu at pumunta sa Admin Pages.

![Listahan ng mga Custom Admin Page sa Network Admin](/img/admin/admin-pages-list.png)

_Listahan ng mga Custom Admin Page sa Network Admin_

Makakakita ka ng listahan ng lahat ng admin page na nagawa mo na hanggang ngayon (dahil ito ang iyong unang pagbisita, walang laman ang listahan). Magpatuloy at i-click ang **Magdagdag ng bagong Admin Page**. Ire-redirect ka sa editor ng Admin Page.

## Mga Opsyon sa Menu {#menu-options}

Sa gilid, sa meta-box ng General Options, magagawa mong i-configure ang uri ng menu, label, at marami pang iba.

### Mga Uri ng Menu {#menu-types}

Maaaring magkaroon ang iyong bagong custom admin page ng dalawang magkaibang uri ng menu: top-level menu at submenu.

Ang top-level menu page, gaya ng ipinahihiwatig ng pangalan, ay magdaragdag ng iyong bagong gawang pahina sa admin menu kasabay ng mga default na pahina ng WordPress.

Kung pipiliin mo ang opsyong top-level, magagawa mo ring piliin ang pagkakasunod-sunod ng menu at isang icon ng menu.

Kung gusto mong gawing sub-menu page ang pahinang ito, magagawa mong pumili ng parent page na pagdudugtungan ng pahinang ito. Maaari mong idagdag ang iyong custom page sa mga default na pahina ng WordPress o sa iyong top-level custom admin pages.

## Mga Advanced na Opsyon {#advanced-options}

Dapat mong maidagdag ang lahat ng uri ng nilalaman sa iyong mga custom admin page. Kabilang dito ang mga panuntunan at file ng CSS, JavaScript, at mga external library rin. Magagawa mo iyon gamit ang Advanced Options Tab.

### Custom CSS {#custom-css}

Sa CSS tab, maaari mong idagdag ang sarili mong custom CSS code, pati na rin mag-import ng mga CSS file mula sa mga remote websayt (tulad ng mga CDN, Google Fonts, atbp).

<!-- Hindi available ang screenshot: Custom CSS editor ng Admin Page Creator na may import ng external stylesheet -->

_Sinusuportahan din ang Custom CSS. Maaari ka ring magsama ng mga external style file!_

### Custom JavaScript {#custom-javascript}

Pareho rin ito para sa JavaScript. Maaari kang magdagdag ng mga external library na gagamitin sa iyong custom JavaScript code.

<!-- Hindi available ang screenshot: Custom JavaScript editor ng Admin Page Creator na may import ng external library -->

_Idagdag ang sarili mong JavaScript code!_

## Mga Pahintulot {#permissions}

Gayundin sa Advanced Options, magagawa mong piliin kung aling mga sub-site ang magkakaroon ng access sa pahinang ito. Kapaki-pakinabang ito kung nagpaplano kang maghatid ng iba’t ibang eksklusibong nilalaman sa iba’t ibang antas ng plano, halimbawa.

<!-- Hindi available ang screenshot: Mga setting ng pahintulot ng Admin Page Creator para sa access control batay sa plano at tungkulin -->

_Mayroon kang ganap na kontrol kung saan lilitaw ang pahinang ito_

**Mahalaga** : Dapat matugunan ang parehong kondisyon para maipakita ang pahina. Ibig sabihin, kung ang user ay kliyente ng isang partikular na plano, ngunit wala sa isa sa mga napiling tungkulin (o kabaliktaran), hindi lilitaw ang pahina sa kanilang panel.

## Aktibo {#active}

Kung gusto mong i-disable ang isang custom na pahina ng admin nang hindi ito binubura o ginagalaw ang mga pahintulot, alisin lang ang tsek sa opsyong Aktibo.

Ang pag-deactivate ng isang top-level na pahina ay awtomatikong magde-deactivate ng lahat ng sub-page na nakakabit sa parent page na iyon.
