# Material WP

Matagal nang pareho ang admin interface ng WordPress. Pero naisip mo na ba kung ano ang magiging hitsura ng bersyong Material Design ng admin interface? 

Ang Material WP ay isang admin theme na binuo para ibigay sa iyo ang ganoong hitsura at pakiramdam, na tumutulong sa iyong magbigay sa iyong mga client ng mas naka-customize na Dashboard.

## Mga Tampok {#features}

- Suporta sa custom CSS
- Madaling mga setting sa pag-istilo
- Muling pagsasaayos ng menu
- Madaling import/export

## Dokumentasyon {#documentation}

**Pag-customize ng iyong Admin Panel gamit ang Material WP**

## Pag-install {#installation}

1. I-upload ang 'material-wp' sa direktoryong '/wp-content/plugins/'
2. I-activate ang plugin sa pamamagitan ng menu na 'Plugins' sa WordPress

## Gabay sa Paggamit {#usage-guide}

Matagal nang pareho ang admin interface ng WordPress. Pero naisip mo na ba kung ano ang magiging hitsura ng bersyong Material Design ng admin interface?

Pagkatapos **i-install ito**, pumunta sa iyong pahina ng _Plugins_ at i-click para i-activate ito sa iyong Network.
