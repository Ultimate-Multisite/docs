# Tagapili ng Wika {#language-selector}

Multisite Ultimate: Tagapili ng Wika sa Pag-sign-up

Magdagdag ng field na tagapili ng wika sa daloy ng pag-sign-up.

## Pag-install {#installation}

1. I-upload ang 'ultimate-multisite-language-selector' sa direktoryong '/wp-content/plugins/'
2. I-activate ang plugin sa pamamagitan ng menu na 'Mga Plugin' sa iyong WordPress Network Admin

## Gabay sa Paggamit {#usage-guide}

Sa libreng add-on na ito, maaari mong hayaan ang iyong mga user na pumili kung aling wika ang gusto nilang makita sa daloy ng pag-sign-up at sa pahina ng login.

Pagkatapos [i-activate ang iyong lisensya](../../user-guide/getting-started/installing-ultimate-multisite.md), magagawa mong i-install ang mga add-on ng Ultimate Multisite na binibigyan ka ng access ng iyong lisensya.

Pumunta sa pahina ng mga setting ng Ultimate Multisite at pagkatapos, sa menu sa kaliwa, i-click ang label na _Mga Add-on_.

![Pahina ng mga add-on](/img/admin/addons-page.png)

_Pumunta sa pahina ng Mga Add-on upang simulan ang pag-install ng Tagapili ng Wika_

Doon, makikita mo ang lahat ng add-on na iniaalok namin.

Available ang mga libreng add-on sa lahat ng aming customer, anuman ang subscription plan na kanilang pinili.

I-click ang button na _I-install ito Ngayon!_ sa ibaba ng thumbnail ng add-on. Pagkatapos ay i-activate ito sa pahina ng Mga Plugin.

![Pahina ng mga add-on na may button sa pag-install](/img/admin/addons-page.png)

_I-click ang button upang i-install ang add-on_

<!-- Hindi available ang screenshot: pahina ng WP Plugins na nagpapakita ng button na Network Activate para sa add-on na Tagapili ng Wika -->

_I-activate ito sa iyong pahina ng Mga Plugin_

Ngayon, pumunta sa menu ng Mga Setting ng Ultimate Multisite at buksan ang tab na _Mga Setting ng Network_. Doon, sa ibaba ng pahina, makakakita ka ng bagong seksyon na tinatawag na _Mga Wika_.

![Pahina ng mga setting](/img/config/settings-general.png)

_May lumitaw na bagong-bagong seksyon!_

Maaari mo itong iwang blangko upang ipakita ang lahat ng wika bilang mga opsyon o piliin kung alin ang gusto mong gawing available. Ang mga wika ay yaong mga na-install mo sa iyong WordPress setup.

<!-- Hindi available ang screenshot: dropdown ng pagpili ng wika na nagpapakita ng mga available na wika sa WP install -->

_Listahan ng mga wikang available sa WP install na ito_

Pagkatapos i-save ang mga pagbabagong ginawa mo, isang menu ng pagpili na may mga wikang pinili mo ang ipapakita sa pahina ng Login pati na rin sa Daloy ng Pag-sign-up.

![Mga setting ng login at pagpaparehistro](/img/config/settings-login-registration.png)

_Tagapili ng Wika sa Pahina ng Login_

![Pahina ng pagpaparehistro](/img/config/registration-page.png)

_Tagapili ng Wika sa Daloy ng Pag-sign-up_

## Pagtatakda ng Wika sa Pamamagitan ng URL {#setting-the-language-via-the-url}

Maaari mo ring baguhin ang wika ng isang pahina sa pamamagitan ng pagpasa ng parameter na locale sa URL. Kung gusto mong i-redirect sa isang pahina ng pagpaparehistro sa Italyano, halimbawa, idagdag ang ?locale=it_IT sa URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Dapat naka-install ang wika sa iyong WordPress network upang gumana ito.
