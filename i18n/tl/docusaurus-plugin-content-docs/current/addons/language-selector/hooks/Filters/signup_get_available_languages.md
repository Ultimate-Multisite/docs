# Filter: signup_get_available_languages {#filter-signupgetavailablelanguages}

Sinasala ang listahan ng mga available na wika para sa mga signup ng front-end site.

Ang pagpasa ng walang-lamang array sa hook na ito ay magdi-disable sa output ng setting sa signup form, at gagamitin ang default na wika kapag ginagawa ang site. Aalisin ang mga wikang hindi pa naka-install.

## Mga Parameter {#parameters}

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $available_languages | `array` | Mga available na wika. |

### Mula Noong {#since}

- 4.4.0
