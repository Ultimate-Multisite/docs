# Pag-sync ng Nilalaman

Multisite Ultimate: Pag-sync ng Nilalaman

Pinapayagan ng add-on na ito para sa Multisite Ultimate ang mga tagapangasiwa ng network na i-sync ang nilalaman mula sa anumang site papunta sa iba pang mga site sa network. Sinusuportahan nito ang lahat ng uri ng post kabilang ang mga post, pahina, custom na uri ng post, at mga template ng Elementor. Kasama sa add-on ang isang multi-step na wizard interface para sa pagpili ng nilalaman at mga target na site, na may batch processing at real-time na feedback sa progreso.

## Mga Tampok {#features}

* Pagpili ng Nilalaman: Visual na grid ng card na may mga thumbnail at pag-filter ayon sa uri ng post
* Lahat ng Uri ng Post: I-sync ang mga post, pahina, produkto, mga template ng Elementor, at anumang custom na uri ng post
* Pagpili ng Site: Piliin kung aling mga site ang pagsi-sync-an ng nilalaman, na may pagbubukod ng pinagmulang site at paunang pagpili ng child site
* Batch Processing: Nagsi-sync sa 10 site nang sabay-sabay na may mga update sa progreso
* Media Sync: Awtomatikong kinokopya at nire-remap ang mga media file (mga larawan, video) mula sa nilalaman ng post papunta sa mga target na site
* Taxonomy Sync: Pinapanatili ang mga kategorya, tag, at custom na taxonomy
* Suporta sa Elementor: Buong suporta para sa mga template ng Elementor na may muling pagbuo ng CSS at pag-remap ng template ID
* Pagresolba ng Conflict: Opsyon na palitan ang umiiral na nilalaman o laktawan ito
* Pamamahala ng Cache: Awtomatikong nililinis ang cache ng Elementor sa mga target na site pagkatapos ng sync

## Mga Kinakailangan {#requirements}

* WordPress 5.3 o mas mataas
* PHP 7.4 o mas mataas
* Multisite Ultimate plugin (aktibo)
* Elementor plugin (opsyonal, kinakailangan para sa pag-sync ng template ng Elementor)

## Pag-install {#installation}

1. I-upload ang 'ultimate-multisite-content-sync' sa direktoryong '/wp-content/plugins/'
2. I-activate ang plugin sa pamamagitan ng menu na 'Plugins' sa iyong WordPress Network Admin
3. I-configure ang mga setting sa Ultimate Multisite → Settings → Content Sync
4. Pumunta sa Ultimate Multisite → Content Sync upang simulan ang pag-sync
