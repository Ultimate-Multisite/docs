# Changelog ng Content Sync {#content-sync-changelog}

### Bersyon 1.1.0 - Inilabas noong 2025-12-31 {#version-110---released-on-2025-12-31}
* Bago: Suporta para sa pagsi-sync ng lahat ng uri ng post (mga post, mga page, custom post types)
* Bago: Pag-filter ng uri ng post sa hakbang ng pagpili ng nilalaman
* Bago: Hakbang ng pagpili ng source site - mag-sync mula sa anumang site, hindi lang sa pangunahing site
* Bago: Paunang pagpili ng child site para sa mga template site
* Bago: Pagkuha ng media mula sa post_content para sa lahat ng uri ng post
* Bago: Pagsi-sync ng taxonomy (mga kategorya, mga tag, custom taxonomies)
* Binago: Pinalitan ang pangalan mula "Elementor Template Sync" tungong "Content Sync"
* Binago: Ang hakbang na "Select Templates" ay pinalitan ng pangalang "Select Data"
* Pinahusay: Ang code na partikular sa Elementor ay tumatakbo na lang ngayon para sa elementor_library post type

### Bersyon 1.0.0 - Inilabas noong 2025-12-19 {#version-100---released-on-2025-12-19}
* Paunang release
* Multi-step na wizard para sa pagpili ng template at site
* Batch processing na may real-time na feedback sa progreso
* Buong pag-sync ng media na may awtomatikong ID remapping
* Integrasyon ng settings sa Ultimate Multisite
* Suporta sa pagbubukod ng site sa pamamagitan ng network settings
* Pag-clear ng Elementor cache pagkatapos ng sync
* Pagpapanatili ng taxonomy ng template
