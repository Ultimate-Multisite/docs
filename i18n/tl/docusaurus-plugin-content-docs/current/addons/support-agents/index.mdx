# Mga Ahente ng Suporta {#support-agents}

Ang pamamahala ng buong network nang mag-isa ay hindi madaling gawain. Habang nagsisimulang lumaki ang iyong network, magsisimula kang makakita ng tumataas na pangangailangan para sa suporta. Magandang ideya na kumuha ng mga taong tutulong sa iyong pagaanin ang iyong trabaho. Gayunpaman, mapanganib ang pagbibigay ng buong access/pahintulot at maaaring makompromiso ang iyong network. Dito pumapasok ang add-on na ito para sa Ahente ng Suporta.

Pinapayagan ka ng add-on na ito na lumikha ng bagong uri ng WordPress user na tinatawag na Ahente ng Suporta, at binibigyan ka ng ganap na kontrol sa kung anong mga pahintulot mayroon ang user na iyon. Ang pagbibigay ng pahintulot ay kasingdali ng pag-on ng toggle button at pag-off nito kung magpapasya kang alisin ito.

## Mga Tampok {#features}

- Detalyadong listahan ng pahintulot ng WordPress at Ultimate Multisite
- Madaling paganahin at huwag paganahin ang access
- Gumawa ng walang limitasyong bilang ng mga Account ng Ahente

## Pag-install {#installation}

1. I-upload ang 'ultimate-multisite-support-agents' sa direktoryong '/wp-content/plugins/'
2. I-activate ang plugin sa pamamagitan ng menu na 'Mga Plugin' sa WordPress
3. Sundin ang sunod-sunod na Wizard upang i-set up ang plugin
