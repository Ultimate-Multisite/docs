# Tagapamahala ng Plugin at Theme {#plugin--theme-manager}

Multisite Ultimate: Tagapamahala ng Plugin at Theme

I-edit o itago ang meta na impormasyon (pamagat, may-akda, thumbnail at paglalarawan) ng mga naka-install na plugin at theme. Gumawa ng mga kategorya para sa mga plugin at theme upang payagan ang iyong mga user na i-filter ang mga ito at magdagdag ng magandang custom na pahina ng mga Plugin sa panel ng iyong mga user.

Upang gamitin ang plugin, maaari ka lamang pumunta sa iyong mga pahina ng Plugin o Theme sa iyong network admin at i-click ang "I-edit ang impormasyon ng Plugin/Theme".

## Pag-install {#installation}

1. I-upload ang 'ultimate-multisite-plugin-and-theme-manager' sa direktoryong '/wp-content/plugins/'
2. I-activate ang plugin sa pamamagitan ng menu na 'Plugins' sa iyong WordPress Network Admin

## Gabay sa Paggamit {#usage-guide}

Ultimate Multisite: Plugin and Theme Manager ay isang makapangyarihang **premium** na add-on ng Ultimate Multisite na nagbibigay sa iyo ng ganap na kontrol sa paraan ng pagtingin at pakikipag-ugnayan ng iyong mga network user sa mga opsyon ng plugin at theme sa iyong network.

May tatlong layunin ang add-on na ito:

  1. Hinahayaan ka nitong ganap na baguhin ang metadata ng parehong mga plugin at theme. Kasama rito ang pangalan, paglalarawan, pangalan ng may-akda, at maging ang mga thumbnail!

  2. Hinahayaan ka nitong ikategorya ang mga plugin at theme, upang payagan ang iyong mga user na madaling ayusin ang mga ito.

  3. Binibigyan ka nito ng opsyong palitan ang luma at nakakainip na pahina ng mga plugin ng bagong pahinang parang themes na may pag-uuri ayon sa mga kategorya.

Pagkatapos itong i-download at i-install, may bagong tab - _Mga Setting ng Add-on_ \- na lalabas sa pahina ng Mga Setting ng Ultimate Multisite.
