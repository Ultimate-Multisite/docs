# Action: wu_mcp_adapter_initialized

Gumagana matapos ma-initialize ang MCP adapter.

Pinapahintulutan ang ibang mga plugin at theme na irehistro ang sarili nilang mga kakayahan.

## Mga Parameter {#parameters}

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | Ang instance ng MCP adapter. |

- 2.5.0
