# Action: wu_dashboard_widgets {#action-wudashboardwidgets}

|------|------|-------------|
| $tab | `string` | Ang kasalukuyang tab. |
| $screen | `\WP_Screen` | Ang screen object. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Instance ng admin page ng Ultimate Multisite. |
