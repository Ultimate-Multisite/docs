# Aksyon: wu_register_integrations

Tumatakbo upang payagan ang mga integration na irehistro ang kanilang sarili.

Inirerehistro ng core ang mga integration nito rito.

## Mga Parameter {#parameters}

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $registry | `\Integration_Registry` | Ang instance ng registry. |

### Mula noong {#since}

- 2.5.0
