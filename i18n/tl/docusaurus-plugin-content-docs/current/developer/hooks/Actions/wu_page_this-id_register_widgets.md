# Action: wu_page_\{$this->id\}_register_widgets

Nati-trigger pagkatapos mairehistro ang mga widget para sa pahinang ito.

The dynamic portion of the hook name, `$this-&gt;id`, refers to the page id.

## Mga Parameter {#parameters}

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $id | `string` | Ang page id. |
| $page_hook | `string` | Ang page hook. |
| $page | `object` | Ang page object. |
