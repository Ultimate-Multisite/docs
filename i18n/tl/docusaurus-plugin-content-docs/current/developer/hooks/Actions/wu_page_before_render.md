# Aksyon: wu_page_before_render

## Mga Parameter

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $page_id | `string` | Ang id ng pahinang ito. |
| $page | `object` | Ang object ng pahina. |
