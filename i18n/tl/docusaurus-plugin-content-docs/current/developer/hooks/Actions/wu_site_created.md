# Aksyon: wu_site_created

Hindi tumatakbo kung dinoble mula sa isang template.

## Mga Parameter

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $data | `array` | Ang datos ng object na itatabi. |
| $site | `\Site` | Ang instance ng object. |
