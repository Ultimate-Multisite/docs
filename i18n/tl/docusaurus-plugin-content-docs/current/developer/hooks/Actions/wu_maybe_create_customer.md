# Action: wu_maybe_create_customer

Payagan ang mga developer ng plugin na gumawa ng karagdagang mga bagay kapag naidagdag ang customer.

Narito kung saan natin idinaragdag ang mga hook para sa pagdaragdag ng customer-&gt;user sa pangunahing site din, halimbawa.

## Mga Parameter

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $customer | `\Customer` | Ang customer na posibleng nagawa. |
| $checkout | `\Checkout` | Ang kasalukuyang checkout class. |

### Mula Noong

- 2.0.0

Tinukoy sa [`inc/checkout/class-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout.php#L1156) sa linya 1156
