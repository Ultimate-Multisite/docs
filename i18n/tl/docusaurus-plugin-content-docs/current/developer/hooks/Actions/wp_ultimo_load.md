# Aksyon: wp_ultimo_load {#action-wpultimoload}

Nati-trigger kapag na-load na ang lahat ng mga dependency

Pinapayagan ang mga developer ng pandagdag na magdagdag ng bagong functionality. Halimbawa, suporta para sa mga bagong tagapagbigay ng pagho-host, atbp.

### Mula noong {#since}

- 2.0.0
