# Filter: wu_apply_plan_limits

Payagan ang mga tagabuo ng plugin na agarang lampasan ang mga limitasyon.

Maaari mong gamitin ang filter na ito upang magpatakbo ng anumang code bago simulan ang alinman sa mga limitasyon. Kung magbabalik ang iyong filter ng anumang truthy na halaga, magpapatuloy ang proseso; kung magbabalik ito ng anumang falsy na halaga, babalik ang code at walang tatakbo sa mga hook sa ibaba.

### Mula noong

- 1.7.0
