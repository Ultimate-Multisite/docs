# Filter: wu_pre_save_settings {#filter-wupresavesettings}

Pahintulutan ang mga developer na i-filter ang mga setting bago i-save ng Ultimate Multisite.

## Mga Parameter {#parameters}

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $settings | `array` | Ang mga setting na ise-save. |
| $settings_to_save | `array` | Ang mga bagong setting na idaragdag. |
| $saved_settings | `array` | Ang kasalukuyang mga setting na naka-save. |
