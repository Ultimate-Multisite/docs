# Filter: wu_plan_product_types {#filter-wuplanproducttypes}

I-filter ang mga uri ng product na dapat ituring bilang mga plan.

Pinapayagan ng filter na ito ang mga addon na magrehistro ng karagdagang mga uri ng product na dapat kilalanin bilang mga plan sa validation at segregation.

## Mga Parameter {#parameters}

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $plan_types | `array` | Array ng mga uri ng product na ituturing bilang mga plan. |

### Mula Noong {#since}

- 2.3.0
### Pinagmulan {#source}

Tinukoy sa [`inc/functions/product.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/product.php#L196) sa linya 196
