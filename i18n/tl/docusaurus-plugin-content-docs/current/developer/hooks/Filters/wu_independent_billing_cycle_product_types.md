# Salain: wu_independent_billing_cycle_product_types

Salain ang mga uri ng produkto na may mga nagsasariling siklo ng pagsingil.

Ang mga produktong may mga nagsasariling siklo ng pagsingil ay hindi pinipilit na tumugma sa panahon ng pagsingil ng plano sa cart. Kapaki-pakinabang ito para sa mga produkto tulad ng pagpaparehistro ng domain na palaging sinisingil taun-taon kahit buwanan o taunang plano.

## Mga Parameter {#parameters}

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $types | `array` | Array ng mga uri ng produkto na may mga nagsasariling siklo ng pagsingil. |

### Mula Noong {#since}

- 2.5.0
