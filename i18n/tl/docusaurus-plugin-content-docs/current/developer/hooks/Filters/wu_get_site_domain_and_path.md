# Filter: wu_get_site_domain_and_path

Payagan ang mga developer na manipulahin ang mga pares ng domain/path.

Maaari itong maging kapaki-pakinabang para sa maraming bagay, tulad ng pagpapatupad ng isang uri ng staging solution, iba't ibang server, atbp.

## Mga Parameter {#parameters}

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $d | `object` | Ang kasalukuyang object na naglalaman ng mga key na domain at path. |
| $path_or_subdomain | `string` | Ang orihinal na path/subdomain na ipinasa sa function. |

### Mula noong {#since}

- 2.0.0
### Pinagmulan {#source}

Tinukoy sa [`inc/functions/site.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/site.php#L235) sa linya 235


## Ibinabalik {#returns}
Isang object na naglalaman ng mga key na domain at path.
