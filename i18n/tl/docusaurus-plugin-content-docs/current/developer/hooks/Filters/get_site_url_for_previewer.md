# Filter: get_site_url_for_previewer

Payagan ang mga developer ng plugin na i-filter ang URL na ginagamit sa previewer

## Mga Parameter {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $domain | `string` | Default na domain na ginagamit ngayon, kapaki-pakinabang para sa mga manipulasyon |
| $domain_options | `array` | Listahan ng lahat ng domain option na inilagay sa Ultimate Multisite Settings -&gt; Network Settings -&gt; Domain Options |

### Mula Noong {#since}

- 1.7.2
### Pinagmulan {#source}

Tinukoy sa [`inc/checkout/class-legacy-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-legacy-checkout.php#L812) sa linya 812


## Ibinabalik {#returns}
Bagong domain na gagamitin
