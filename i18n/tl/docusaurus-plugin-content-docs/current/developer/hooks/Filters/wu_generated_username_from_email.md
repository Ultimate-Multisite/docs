# Filter: wu_generated_username_from_email

I-filter ang nabuong username ng customer.

## Mga Parameter

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $username | `string` | Nabuong username. |
| $email | `string` | Email address ng bagong customer. |
| $new_user_args | `array` | Array ng mga arg ng bagong user, maaaring kasama ang una at apelyido. |
