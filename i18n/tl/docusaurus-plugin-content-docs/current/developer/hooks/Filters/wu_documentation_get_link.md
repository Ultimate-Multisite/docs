# Filter: wu_documentation_get_link {#filter-wudocumentationgetlink}

Payagan ang mga developer ng pampalawak na i-filter ang mga link.

Hindi sigurado kung paano iyon magiging kapaki-pakinabang, pero walang masamang magkaroon nito

## Mga Parameter {#parameters}

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $link | `string` | Ang link na nakarehistro |
| $slug | `string` | Ang slug na ginamit upang kunin ang link |
| $default_link | `string` | Ang default na link na nakarehistro |
