# Filter: wp_ultimo_render_vars

Pinapayagan ang mga tagapagbuo ng karagdagang bahagi na magdagdag ng karagdagang variable sa konteksto ng pag-render sa buong sistema.

## Mga Parameter

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $args | `array` | Array na naglalaman ng mga variable na ipinasa ng render call. |
| $view | `string` | Pangalan ng view na ire-render. |
| $default_view | `string` | Pangalan ng fallback_view |

### Mula noong

- 2.0.0
### Pinagmulan

Tinukoy sa [`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L31) sa linya 31
