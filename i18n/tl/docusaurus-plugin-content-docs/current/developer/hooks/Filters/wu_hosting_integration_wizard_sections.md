# Filter: wu_hosting_integration_wizard_sections

Sinasala ang mga seksyon ng wizard para sa setup ng hosting integration.

Pinapayagan ang mga addon na magdagdag, mag-alis, o magbago ng mga seksyon ng wizard.

## Mga Parameter

| Name | Type | Description |
|------|------|-------------|
| $sections | `array` | Mga seksyon ng wizard. |
| $integration | `\WP_Ultimo\Integrations\Host_Providers\Base_Host_Provider\|\WP_Ultimo\Integrations\Integration` | Ang integration na kino-configure. |
| $page | `\Hosting_Integration_Wizard_Admin_Page` | Ang instance ng pahina ng wizard. |

### Mula noong

- 2.5.0
