# Filter: wu_get_checkout_variables

Payagan ang mga developer ng plugin na salain ang mga pre-set ng isang checkout page.

Mag-ingat, maaaring tuluyang masira ng mga nawawalang key ang checkout sa front-end.

## Mga Parameter

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $variables | `array` | Mga na-localize na variable. |
| $checkout | `\Checkout` | Ang checkout class. |

### Mula noong

- 2.0.0
### Pinagmulan

Tinukoy sa [`inc/checkout/class-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout.php#L1970) sa linya 1970


## Mga Ibinabalik
Ang bagong array ng mga variable.
