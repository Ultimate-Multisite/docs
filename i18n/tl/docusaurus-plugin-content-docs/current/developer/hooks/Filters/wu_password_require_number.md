Salain kung ire-require ang mga numero sa mga password.

## Mga Parameter

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $require | `bool` | Kung ire-require ang mga numero. Default na true kapag ipinatutupad ang mga tuntunin. |
| $defender_active | `bool` | Kung aktibo ang Defender Pro Strong Password. |
