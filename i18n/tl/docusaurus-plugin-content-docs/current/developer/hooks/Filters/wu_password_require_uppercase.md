# Filter: wu_password_require_uppercase

I-filter kung mangangailangan ng malalaking titik sa mga password.

## Mga Parameter

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $require | `bool` | Kung mangangailangan ng malalaking titik. Default na true kapag ipinapatupad ang mga panuntunan. |
| $defender_active | `bool` | Kung aktibo ang Defender Pro Strong Password. |

### Mula noong

- 2.4.0
