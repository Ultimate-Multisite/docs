# Filter: wu_view_override_replaceable_views

Pinapayagan ang mga developer na magdagdag ng karagdagang mga folder sa listahang mapapalitan.

Mag-ingat, dahil ang pagpapahintulot ng karagdagang mga folder ay maaaring magdulot na ma-load ang mga lumang kopya sa halip na mga bersyon ng Ultimate Multisite.

## Mga Parameter

| Pangalan | Uri | Paglalarawan |
|------|------|-------------|
| $replaceable_views | `array` | Listahan ng mga pinapayagang folder. |

### Mula noong

- 2.0.0
### Pinagmulan

Tinukoy sa [`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L50) sa linya 50
