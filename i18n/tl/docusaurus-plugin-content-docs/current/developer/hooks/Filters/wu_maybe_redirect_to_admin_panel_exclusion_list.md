# Filter: wu_maybe_redirect_to_admin_panel_exclusion_list {#filter-wumayberedirecttoadminpanelexclusionlist}

Gumawa ng listahan ng pagbubukod ng mga parameter na pumipigil sa awtomatikong pag-redirect.

Kailangan ito dahil kung hindi, hindi magagawang i-edit ng tagabuo ng pahina ang pahina ng pag-login kapag natukoy na ito.
