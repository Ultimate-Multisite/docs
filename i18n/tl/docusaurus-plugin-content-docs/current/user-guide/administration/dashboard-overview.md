# Pangkalahatang-tingin sa Dashboard

Ang Ultimate Multisite ay may nakalaang **Dashboard** sa ilalim ng Ultimate Multisite menu, at nagdaragdag din ito ng mga widget sa Network Dashboard. Tatalakayin sa artikulong ito kung ano ang makikita mo sa parehong lugar.

## Ang Network Dashboard

Inilalagay ng Ultimate Multisite ang mga kapaki-pakinabang na widget sa network-level dashboard na matatagpuan sa **My Sites → Network Admin → Dashboard**.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

Narito kung ano ang ibinibigay ng bawat widget:

**First Steps:** Lumilitaw ito sa isang bagong instalasyon at nagpapaalala sa iyo na tapusin ang mga gawain tulad ng pag-configure ng payment method at paggawa ng test account.

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Summary:** Nag-uulat ito tungkol sa bilang ng mga sign-up at kita para sa araw na iyon. Ang MRR (Monthly Recurring Revenue) ay nagpapakita ng inaasahang kabuuang halaga sa lahat ng customer na may mga membership na kumikita.

![Summary widget](/img/admin/network-dashboard-summary.png)

**Activity Stream:** Nag-uulat ito tungkol sa mga pangyayari sa buong Ultimate Multisite — sign-up, cancellation, paglikha ng site, pagtanggal, at iba pang aktibidad.

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** Nagpapakita ito ng maikling buod ng mga user at site sa network na may mga shortcut para gumawa ng bagong site o user.

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** Kinukuha nito at ipinapakita ang pinakabagong balita ng Ultimate Multisite kabilang ang mga update, kritikal na bug, at security fixes.

![News widget](/img/admin/network-dashboard-news.png)

## Ang Ultimate Multisite Dashboard

Habang ang network dashboard ay nagpapakita ng pangkalahatang impormasyon tungkol sa network, ang ultimate multisite dashboard (sa pinaka-taas na antas ng Ultimate Multisite menu) naman ay nagpapakita ng impormasyon tungkol sa serbisyo.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Paglago ng Monthly Recurring Revenue (MRR):** Nagpapakita ito ng buwanang paghihimay ng kita sa loob ng 12-buwang panahon ng kalendaryo. Ang bagong kita ay ipinapakita sa kulay asul habang ang mga pagkansela ay lumalabas sa pula.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**Suporta para sa Date-range:** Mag-filter ng data ng dashboard base sa panahon upang mag-focus sa impormasyong kailangan mo. Ang date range ang nagkokontrol kung ano ang ipinapakita sa mga widget sa ibaba.

![Date range and dashboard stats](/img/admin/um-dashboard-date-range.png)

Ang mga widget ng dashboard ay kinabibilangan ng:

- **Kita (Revenue)** — Kabuuang kita para sa panahong iyon kasama ang MRR at paghihimay bawat produkto
- **Bagong Miyembro** — Ilan pang bagong membership ang nilikha sa napiling panahon
- **Pagpaparehistro ayon sa Bansa (Signups by Countries)** — Geograpikal na paghihimay ng mga pagpaparehistro
- **Pagpaparehistro ayon sa Mga Form (Signups by Forms)** — Ilang customer ang nagparehistro sa bawat checkout form kumpara sa idinagdag ng mga admin
- **Pinakamaraming Binibisit na Sites (Most Visited Sites)** — Ang pinakamaraming binibisiting sites sa iyong network (nangangailangan ng pag-enable ng visit counting sa **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**)

![Dashboard statistics](/img/admin/um-dashboard-stats.png)
