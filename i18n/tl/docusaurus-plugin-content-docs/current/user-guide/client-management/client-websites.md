# Mga Website ng Iyong Kliyente (v2) {#your-clients-websites-v2}

Pagkatapos mag-register sa network mo, ang mga customer nila ay makakapasok sa kanilang dashboard.

Bukod sa pahinang **Account**, kung saan makikita nila ang mahahalagang impormasyon tungkol sa kanilang mga bayad, membership, domain, limitasyon ng plano at iba pa, mayroon din silang access sa pahinang **My Sites**, kung saan makikita at mapapamahalaan nila ang lahat ng kanilang mga website. Ang kailangan lang nilang gawin ay i-click ang My Sites sa kanilang dashboard.

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

Makikita doon ang lahat ng kanilang mga website, na may **Manage** button sa ibaba ng bawat isa. Kung i-click nila ito, dadalhin sila sa **dashboard ng partikular na website na iyon**.

Maa rin nilang makita kung alin sa kanilang mga website ang **primary** para sa mga planang nagpapahintulot sa kanila na magkaroon ng higit sa isang website.

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

Narito ang buong tingin sa interface ng pag-edit ng site:

![Site edit full interface](/img/admin/site-edit-full.png)

Kung gusto nilang baguhin ang kanilang primary website, kailangan nilang pumunta sa Account page sa pamamagitan ng pag-click sa menu na **Account**.

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

Maa rin nilang ma-view at ma-edit ang mga detalye ng customer:

![Customer edit interface](/img/admin/customer-edit.png)

Pagkatapos, dapat i-click nila ang **Change Default Site**.

![Change Default Site button on account page](/img/account-page/change-default-site.png)

May lalabas na pop-up window. I-click ang drop-down list at piliin kung aling website ang gusto mong gawing primary website.

![Change Default Site dropdown selection](/img/account-page/change-default-site-dropdown.png)

Pagkatapos, i-click lang ang **Change Default Site** at tapos na!

![Kinukumpirma ang pagbabago ng default site](/img/account-page/change-default-site-confirm.png)
