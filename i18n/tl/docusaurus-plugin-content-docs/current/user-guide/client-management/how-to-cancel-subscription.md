# Paano ko ma-cancel ang aking subscription? {#how-can-i-cancel-my-subscription}

Kung mayroon kang Ultimate Multisite license, maaari mong kanselahin ang renewal nito anumang oras na gusto mo. Sundin lang ang mga hakbang sa ibaba:

<!-- Screenshot unavailable: Account page email with access link -->

Gamitin ang email na ibinigay mo noong bumili ka at ang iyong password.

<!-- Screenshot unavailable: Login page with email and password fields -->

Ito ang unang screen ng iyong Account Page:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

Para kanselahin ang iyong subscription, sa menu sa kaliwa, pumunta sa _Renewing & Billing._

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

I-click ang arrow sa kanan para magbukas ng side window. Pagkatapos, piliin mo ang opsyon na magko-cancel ng **auto-renew**.

Magpapakita sa iyo ng sistema ng confirmation message.

Kung sigurado kang gusto mong kanselahin ang iyong subscription, i-click lang ang _**Cancel Renewals**_ button. Pagkatapos gawin ito, hihingin sa iyo ng isang mabilis na survey.

Tapos na! Hindi awtomatikong ma-renew ang iyong subscription.

**Magkakaroon ka ng valid key hanggang sa mag-expire ang iyong subscription**. Kung gusto mong i-reactivate ang iyong subscription, kailangan mo itong gawin nang mano-mano.
