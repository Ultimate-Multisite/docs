# Madalas Itanong na mga Tanong (Frequently Asked Questions) {#frequently-asked-questions}

Narito ang ilang mga **Madalas Itanong na Tanong** na madalas lumalabas sa mga request para sa suporta.

  * **Ano ang Ultimate Multisite?**

Ang Ultimate Multisite ay isang WordPress plugin, ginawa para sa WordPress multisite installs, na nagpapalit sa iyong WordPress install tungo sa isang premium network ng mga site – parang [WordPress.com](https://WordPress.com) – na nagpapahintulot sa mga kliyente na gumawa ng mga site gamit ang buwanan, quarterly, o taunang bayad (maaari ka ring gumawa ng Free plans).

  * **Gagana ba ang Ultimate Multisite nang walang Multisite?**

Hindi. Ang Ultimate Multisite ay nangangailangan ng WordPress multisite, subdirectory, o subdomain.

  * **Ano ang mga requirements para sa Ultimate Multisite?**

Ang Ultimate Multisite ay nangangailangan ng:

**WordPress Version** : v5.3+ (Inirerekomenda: Huling Stable na Bersyon)
**PHP Version** : 7.4.x (Suporta sa > 8.0 malapit na darating)
**MySQL Version** : v5+ (Inirerekomenda: Bersyon 5.6, kung wala kang 8.0 sa iyong hosting provider)

_Kailangan din i-activate ang Multisite (subdomain o subdirectory)._ [Paano Mag-install ng WordPress Multisite](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **Gagana ba ang Ultimate Multisite sa Multisite na may Subdirectories?**

Oo. Sinusuportahan ng Ultimate Multisite ang parehong subdirectory installs at subdirectory installs.

  * **Gagana ba ang Ultimate Multisite sa Multisite na may Subdomains?**

Oo. Sinusuportahan ng Ultimate Multisite ang parehong subdirectory installs at subdomain installs.

  * **Maaari ko bang gamitin ang Ultimate Multisite para limitahan kung aling Plugins at Themes ang magagamit ng mga user?**

Oo. Sa Ultimate Multisite, maaari mong piliin kung aling plugins at themes ang available sa bawat plan sa iyong network.

  * **Maaari ko bang gamitin ang Ultimate Multisite para limitahan ang bilang ng posts na makakagawa ang isang user sa kanyang site?**

Oo. Sa Ultimate Multisite, pwede kang magtatag ng post quotas para sa **bawat available na post type** sa iyong network. Pwede mo ring limitahan ang bilang ng mga user na pwedeng iimbitahan ng admin, ang **dami ng disk space**, at marami pang iba.

  * **Sumusuporta ba ang Ultimate Multisite ng Site Templates?**

Oo. Sa Ultimate Multisite, kaya mong gumawa ng kahit anong dami ng Site Templates. Ang Site Templates ay hindi themes; sila ay mga normal na site sa iyong network. Pwede kang mag-activate ng plugins, themes, at i-customize ang Site Template na iyon sa anumang paraan na gusto mo. Kung pipiliin ng isang user ang template na iyon, gagawin ang kanyang site bilang kopya ng template na iyon, kasama ang lahat ng mga customization.

Pwede mo ring magtakda ng partikular na template para sa isang plan. Sa paggawa nito, ang bawat user na magre-register sa plan na iyon ay magkakaroon ng site na ginawa bilang duplicate ng Site Template na iyon.

  * **Nakaka-integrate ba ang Ultimate Multisite sa WooCommerce?**

Oo!

Kung gusto mong magkaroon ng WooCommerce para sa mga user mo sa network, na papayagan silang mag-set up ng stores, posible ito gamit ang Ultimate Multisite. Kaya mo pa limitahan ang bilang ng mga produkto na gagawin ng iyong mga kliyente sa bawat tier ng plan.

Tungkol naman sa subscription payments para sa Ultimate Multisite, pwede mong gamitin ang [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) para payagan ang iyong mga Ultimate Multisite clients na magbayad gamit ang WooCommerce gateways sa iyong pangunahing site. Sa ganoong paraan, maaari mong gamitin ang malaking library ng payment gateways na binuo para sa WooCommerce at gamitin ito para singilin ang iyong mga Ultimate Multisite clients.

  * **May paraan ba para ilipat ang mga subscription mula sa WPMUDEV Pro Sites?**

Oo, ang Ultimate Multisite ay nag-aalok na ngayon ng [isang kapaki-pakinabang na tool para mag-migrate](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) sa iyong Pro Sites network.

  * **Bakit walang laman ang tab na “Allowed Plugins” sa aking Plan Edit Page?**

Pinapayagan lang ng Ultimate Multisite mo kung papayagan o hindi access ang mga plugin na hindi naka-activate sa network mo. Kapag ang isang plugin ay naka-activate sa network, natural itong maglo-load nang default sa bawat site ng iyong network.

Kung wala kang nakikitang anumang listahan ng mga plugin sa tab na “Allowed Plugins”, tingnan kung ang gustong plugin ay aktibo ba o hindi sa network mo.

  * **Pwede ko bang gamitin ang sarili kong disenyo ng Pricing Table?**

Oo. Simula sa bersyon 1.4.0, sinusuportahan ng Ultimate Multisite ang template overrides.

  * **Gumagana ba ang Ultimate Multisite kasama ang WP Engine?**

Oo. Ang Ultimate Multisite ay laging sinusuportahan ng WP Engine, pero kailangan ng manual na dagdag na hakbang mula sa admin (pagdaragdag ng domain sa WP Engine panel) para gumana ang domain mapping. **Hindi na ito nangyayari simula sa bersyon 1.6.0.** Ang Ultimate Multisite ay direktang nakikipag-ugnayan sa WP Engine API at awtomatikong idadagdag ang mga domain sa panel pagkatapos ng mapping.

  * **Anong hosting provider ang inirerekomenda mo?**

Ang Ultimate Multisite ay gumagana sa anumang hosting provider. Ang tanging pagkakaiba ay ang ilang kumpanya ng hosting tulad ng Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane at [Runcloud.io](http://Runcloud.io) ay may mas mahigpit na integrasyon para sa domain mapping kaya gumagana sila nang direkta (out of the box). Para sa ibang hosting, maaaring kailanganin mong i-set up ang domain mapping nang mano-mano.
