# Paano Humingi ng Tulong

Mayroon kang balidong License key, at kasama diyan ang access sa aming team para tulungan ka sa mga problema na maaaring dumating habang nagse-set up mo ang Ultimate Multisite.

Ang araling ito ay tumatalakay sa mga available na channel ng suporta para sa mga may license at kung anong impormasyon ang kailangan mong ihanda bago makipag-ugnayan sa aming support team.

## Help Site (Site ng Tulong)

Maaari kang mag-access ng iba't ibang articles at tutorial mula sa aming help site na matatagpuan sa link sa ibaba.

[)

Hinihikayat namin ang ating mga user na pamilyar sa mga pangunahing feature at function ng Ultimate Multisite. Mayroon tayong mga "Getting Started" articles sa aming help site at makikita rin doon ang mga sagot sa mga karaniwang tanong, tulad kung paano mag-install ng add-ons, kung paano i-integrate ang payment gateways, atbp.

## Contact Support (Makipag-ugnayan sa Suporta)

Kung ikaw ay nahihirapan at hindi makahanap ng sagot sa help site namin, maaari kang makipag-ugnayan sa aming support team sa pamamagitan ng pag-click sa speech bubble icon na matatagpuan sa kanang ibabang bahagi ng aming website O sa pamamagitan ng direktang pagpapadala ng email sa support@ultimatemultisite.com.

Kapag nakikipag-ugnayan sa suporta para sa mga technical issues, siguraduhing magbigay ka ng detalyadong paglalarawan ng iyong problema kasama ang screenshot, error messages, o anumang impormasyon na makakatulong sa aming mga tao para madali nilang maintindihan at ma-evaluate ang iyong isyu. Kailangan mo ring ibigay ang iyong license key. Ang mga aktibong may-ari ng lisensya lamang ang karapat-dapat sa suporta.

## Community Site (Site ng Komunidad)

Ang aming [community site](https://community.ultimatemultisite.com/) ay isang lugar para makipag-ugnayan ang mga Ultimate Multisite users sa isa't isa. Ito ay nagsisilbing sistema ng impormasyon kung saan maaaring mag-post, mag-comment sa mga diskusyon, magbigay ng payo, makipagtulungan, o maghanap ng tulong (crowdsource) ang mga miyembro. Kakailanganin mong ibigay ang iyong Ultimate Multisite license key para ma-unlock ang mga pribadong espasyo/page.
