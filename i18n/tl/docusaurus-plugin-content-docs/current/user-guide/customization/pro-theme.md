# Pag-customize ng iyong Admin Panel gamit ang PRO Theme

Madalas kapag gumagawa tayo ng trabaho para sa kliyente, kailangan nating magbigay sa kanila ng custom na karanasan sa admin dashboard ng WordPress. Kung naghahanap ka ng mas propesyonal na itsura at pakiramdam, makakatulong ang PRO Theme.

Ang PRO Theme ay isang admin theme na binuo para tulungan ka kapag nag-customize ka ng Dashboard ng iyong mga kliyente. At bahagi rin ito ng core ng premium plugins ng Ultimate Multisite.

Pagkatapos mong i-install ito, pumunta sa iyong _Plugins_ page at i-click para i-activate ito sa iyong Network.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_I-Activate ito sa buong network mo sa iyong Plugins page_

_**TANDAAN:**_ _Kung i-install mo lang ito, lalabas ito sa mga kliyente mo bilang isang opsyon na pwedeng markahan para idagdag ang theme na iyon sa user na iyon lamang._

Ngayon, mayroon ka nang ganap na rediseign ng iyong admin interface! Mayroon ka ring bagong menu sa kaliwang column na tinatawag na PRO Theme.
