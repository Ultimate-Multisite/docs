# Paglalagay ng Ads sa iyong network gamit ang Ultimate Multisite: Ad Injector {#placing-ads-on-your-network-with-ultimate-multisite-ad-injector}

Ang **Ultimate Multisite: Ad Injector** ay isang **libreng add-on** na nagbibigay sa iyo ng mga tool para maglagay ng ads sa iyong mga plano mismo mula sa kanilang edit page.

Pagkatapos mong i-install ang Ultimate Multisite: Ad Injector, pumunta sa iyong _Plugins_ page para i-activate ito.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_I-click ang Network Activate button_

Kapag aktibo na ang add-on, ang _Advanced Options_ meta box sa screen ng pag-edit ng plano ay magkakaroon ng bagong opsyon na tinatawag na _Ad Settings_.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_Ngayon maaari mo nang i-edit ang ad settings para sa bawat isa sa iyong mga plano_

Sa tab na iyon, makakapagdagdag ka ng ad codes para sa posts at pages sa front-end, bago at pagkatapos ng content.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Pagkatapos mong ilagay ang iyong ad code, ito ay lalabas sa mga posts at pages ng partikular na plano._

Maaari ka ring magkaroon ng ads na kasama sa header ng Admin Pages para sa mga user ng plan na iyon.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Pumunta sa Admin Panel Ad Code option kung gusto mong magpakita ng ads sa back-end ng iyong user_

### Higit pa sa Pag-aanunsyo (Advertising) {#beyond-advertising}

Maa rin mong gamitin ang Ultimate Multisite: Ad Injector para magbigay ng suporta sa mga high-tier plans.

Pumunta lang sa _Admin Panel Ads_ option at idagdag ang HTML code para sa iyong support tool.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Hindi screen capture unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Magdagdag ng customer support sa mga partikular na plano_

Sa ganitong paraan, maaari kang maabot nila mismo mula sa kanilang Admin Panel!
