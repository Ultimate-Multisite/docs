# Paano Kumuha ng Custom Development Help mula sa Codeable?

Minsan, talagang kailangan natin ng mga astig na customization para gawing eksaktong gusto natin ang daloy ng ating network flows. Kung ito ang hinahanap mo, inirerekomenda namin ang [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), isang lugar kung saan makakahanap ka ng mga beteranong developer na may Ultimate Multisite expertise.

Ang [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) ay isang platform na nag-uugnay sa mga customer at mga kwalipikadong WordPress freelance developers, na kayang magtrabaho sa mga proyekto anuman ang laki. Mayroon silang masusing proseso ng aplikasyon para masigurado na ikaw ay itutugma sa mga eksperto na partikular na kwalipikado para sa gawain na nasa isip mo.

### Paano gumawa ng project sa Codeable?

Una, pumunta ka sa [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) at i-click ang **Start a Project** button.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Ngayon, para magsumite ng project, i-click mo sa kaliwa para sa **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

Sa "I need help with", piliin ang **Customizing**:

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

Sa "My", piliin ang **Plugin**:

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Pagkatapos ay i-click ang **Continue** button.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Ngayon, oras na para ilarawan ang iyong project at pangangailangan mo sa pinakamahusay na paraan na kaya mo. Subukang magbigay ng maraming impormasyon hangga't maaari!

<!-- Screenshot unavailable: Codeable project description form -->

Sagutin ang mga sumusunod na tanong para makatulong sa pagpepresyo (pricing).

<!-- Screenshot unavailable: Codeable project pricing questions -->

Pagkatapos nito, kung wala ka pang account sa [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), gumawa ka ng isa at i-click ang **Publish My Project** button.

<!-- Hindi available na screenshot: Paglikha ng Codeable account at Publish My Project button -->

Kapag na-publish mo na ang iyong [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) project, ikokonekta ka ng platform sa mga developer na tugma sa kasanayan na kailangan para sa iyong proyekto.

Makakatulong ang prosesong ito upang mahanap mo ang tamang tao para sa trabaho mo, na may karanasan sa WordPress at Ultimate Multisite!

Ang minimum rate ng Codeable ay $70–$120 USD bawat oras. Ang mga estimate ay magbabago depende sa saklaw ng trabaho (scope of work), kumplikasyon, at pagkaapurahan (urgency) ng proyekto.

Kapag nakakuha ka na ng iyong estimate, makakapag-hire ka ng developer na pinakaangkop sa iyong pangangailangan at sisimulan natin ang iyong proyekto nang magkasama!
