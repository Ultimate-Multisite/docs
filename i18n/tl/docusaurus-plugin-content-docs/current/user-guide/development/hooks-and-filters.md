# Ultimate Multisite Hooks at Filters

Ang Ultimate Multisite ay binubuo ng maraming function at database queries. Mayroon itong mga hook at filter na magbibigay-daan sa iyo na baguhin o magdagdag ng bagong code sa mga kasalukuyang function.

Gumawa kami ng listahan ng mga hook at filter na ito bilang reference para matulungan ang mga advanced user na mas i-customize pa ang kanilang network. Maaari mong ma-access ang listahan mula sa Github sa pamamagitan ng pag-click sa link sa ibaba.
