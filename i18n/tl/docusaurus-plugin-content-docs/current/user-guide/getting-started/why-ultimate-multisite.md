# Bakit ang Ultimate Multisite

Kung nakarating ka sa artikulong ito, malamang narinig mo na ang plugin namin ang solusyon para maisakatuparan ang ideya ng iyong negosyo. Sana ay mayroon kang mga sagot na kailangan mo pagkatapos basahin itong bahagi.

## Ano ang Ultimate Multisite? {#what-is-ultimate-multisite}

Ang Ultimate Multisite ay isang WordPress Multisite plugin na nagbibigay-daan sa iyo na mag-alok ng WaaS (Website as a Service) sa iyong mga customer. Gamit ang aming plugin, maaari kang magkaroon ng iba't ibang subscription tiers at maaaring magbayad ang mga customer sa iyo ng paulit-ulit na bayad para magkaroon sila ng site na naka-host sa iyong Multisite network.

Pinapayagan ka ng Ultimate Multisite na tumanggap ng iyong mga bayad gamit ang iba't ibang gateway (tulad ng WooCommerce o Stripe), gumawa ng mga site template para mas madali ang buhay mo kapag may bagong user na nagrehistro sa iyong network, mag-mapa ng domain, at i-personalize ang iyong registration forms para makagawa ka ng customized experience para sa iyong mga customer.

At ito lang ang simula dahil maaari mong palawakin ang mga function nito gamit ang add-ons at webhooks.

## Iba pang mga feature {#other-features}

Nag-aalok din ang Ultimate Multisite ng:

  * Domain Mapping integration kasama ang Managed WordPress hosting providers: Awtomatikong idinadagdag ng Ultimate Multisite ang mga na-mapa na domain sa admin panels ng WP Engine, Cloudways, cPanel, RunCloud, Closte, at iba pa kapag may bagong domain na na-mapa, para hindi mo na kailangang gawin ito nang mano-mano. Sa ilang pagkakataon, magpapatakbo din tayo ng SSL certificate fetching para sa iyo.

  * Webhooks: Ang Ultimate Multisite ay may malakas na sistema ng Webhook na nagpapahintulot sa iyo na iugnay ang iyong network sa maraming iba't ibang serbisyo. Nag-aalok din kami ng direktang integrasyon sa Zapier, na nagbubukas ng iyong network sa mahigit 1000+ apps at serbisyo na available sa kanilang platform.

* Malalakas na Admin Pages para tulungan ang iyong mga customer gamit ang paborito mong page builder: Ang Ultimate Multisite ay mayroong magandang add-on na tinatawag na Admin Pages Creator, na sa madaling salita ay nagbibigay-daan sa iyo na magdagdag ng mga ganap na naka-customize na Admin Pages para sa dashboard ng iyong mga customer gamit ang Elementor, Beaver Builder, at Brizy templates.

* Dalawang magkaibang Admin Themes para i-customize ang karanasan sa dashboard ng iyong mga customer: Check Material WP at PRO.

At mas advanced pa rito:

* Suporta sa Template Overrides: Sa Ultimate Multisite, maaari mong palitan ang default HTML templates para sa signup flow gamit ang sarili mong bersyon nang hindi nag-aalala na babaguhin ang core plugin files.

* Search and Replace API: Maaari mo ring lagyan ng annotation ang site templates ng mga placeholder tulad ng {{site_name}} at gamitin ang Search and Replace API para awtomatikong punan ang mga placeholder na iyon ng impormasyong ibinibigay ng iyong customer pagkatapos mag-sign up.

## Mga Kaso ng Paggamit (Use cases) {#use-cases}

Ngayon ay oras na para makita ang ilang posibleng gamit ng Ultimate Multisite.

### Kaso 1: Isang Ahensya (An Agency) {#case-1-an-agency}

Karaniwan, ang pangunahing kasanayan ng isang ahensya ay nakasalalay sa disenyo ng mga website na may mga aspeto tulad ng kanilang hosting o marketing na nakalista bilang karagdagang serbisyo.

Para sa mga ahensya, nag-aalok ang Ultimate Multisite ng napakalaking benepisyo sa kakayahan nitong mag-host at mag-manage ng maraming website sa iisang platform. Mas malaki pa ito para sa mga ahensyang nagpapatatag ng kanilang disenyo sa partikular na mga theme tulad ng GeneratePress, Astra, OceanWP, o iba pa dahil magagamit nila ang kakayahan ng Ultimate Multisite upang awtomatikong i-activate ang mga temang ito para sa bawat bagong site.

Ang mahigpit na kontrol sa mga plugin at theme ay pinapanatili batay sa bawat produkto sa pamamagitan ng mga setting ng Ultimate Multisite na nagpapahintulot na maging available o itago ang mga plugin at theme pati na rin ang kanilang activation state kapag inilalapat (instantiated) para sa isang bagong site.

Makakahanap ng kapayapaan ang mga agency sa pamamagitan ng Ultimate Multisite na nagpapahintulot sa kanila na gawin ang kanilang pinakamagaling—ang magdisenyo ng mga pambihirang website.

### Kaso 2: Niche Provider {#case-2-niche-provider}

May lumang kasabihan na nagsasabi, “Gawin mo nang isa at gawin mo ito nang mahusay.” Para sa maraming espesyalista, ang ibig sabihin nito ay gumawa ng produkto o serbisyo batay sa iisang pangunahing ideya.

Kung nakatuon ka sa pagbuo ng isang niche network ng mga site, malamang na gusto mong magbigay ng mga serbisyo batay sa isang karaniwang framework at platform. Dito makakatulong ang mga site template!

Isa sa mga makabagong feature ng Ultimate Multisite, ang template site ay isa kung saan naka-install at aktibo na ang theme, naka-install at aktibo na ang mga kinakailangang plugins, at mayroon nang mga sample post o page. Kapag ang isang customer ay gumawa ng bagong site batay sa template, ang nilalaman at settings ng template ay kinokopya sa bagong ginawang site.

Para sa isang provider ng niche sites at serbisyo, nagbibigay ito ng walang katumbas na bentahe sa kakayahang agad na lumikha ng isang handa nang site na may mga custom plugins at disenyo. Ang customer ay kailangan lang magbigay ng minimal na input para matapos ang serbisyo.

### Kaso 3: WordPress Web Hosting {#case-3-wordpress-web-hosting}

Maraming paraan kung paano mag-host ng WordPress sites, ngunit bihira itong maging simple tulad ng pagbibigay ng web space sa isang customer na may pre-installed na bersyon ng WordPress. Ito ay dahil maraming desisyon at konsiderasyon ang kailangang pagsamahin upang magbigay ng makabuluhang serbisyo.

Ang Ultimate Multisite ay nagtatagumpay sa larangang ito sa pamamagitan ng pagbibigay ng isang kumpletong turnkey solution para sa hosting ng mga WordPress sites. Kasama sa solusyon ang mga pangunahing mekanismo upang magbigay ng subscription services, koleksyon ng bayad, checkout forms, discount vouchers, at komunikasyon sa customer.

Maraming mahahalagang trabaho para sa tamang pag-install, pag-configure, at pagpapanatili ng WordPress Multisite ang ginagawang madali ng Ultimate Multisite kaya't ang mga network administrator ay kailangan lang isaalang-alang ang mga aspeto na may kaugnayan sa kanilang serbisyo o niche, tulad ng product tiers, pricing, at mga alok ng serbisyo.

Para sa mga developer na gustong makipag-integrate sa Ultimate Multisite, nag-aalok din ang solusyon ng isang kumpletong RESTful API at Webhooks para sa event notification.

Hindi na kailangan umasa sa maraming external plugins at license, ang Ultimate Multisite ay nagbibigay ng feature-rich at katumbas na solusyon sa Wix, Squarespace, [WordPress.com](https://WordPress.com), at iba pa.
