# Pamamahala sa mga Shareable Link para sa mga Plano

Nagbibigay ang Ultimate Multisite sa iyo ng opsyon na i-share nang direkta ang mga link ng plano sa iyong mga kliyente. Para gawin ito, pumunta sa anumang edit page ng iyong mga plano at i-click ang _Click to copy the Shareable Link_ button.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_Sa Plan's Edit Page, makikita mo ang button para gumawa ng shareable links_

Napansin na ginagamit ng mekanismong ito ang default billing frequency para buuin ang link. Ibig sabihin, kung ang iyong default billing ay buwanan, halimbawa, sa huli, magkakaroon ka ng link na ganito: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Kung gusto mong maging quarterly ang shareable link, kailangan mong magdagdag ng 3 sa dulo

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Para piliin ang plano _premium_ sa taunang iskedyul, magdadagdag ka ng 1 at pagkatapos ay ang salitang:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Tandaan na hindi mo pwedeng magkaroon lang ng isang link para sa iba't ibang billing frequency. Dapat kang mag-share ng isang tiyak na link para sa bawat billing frequency mo.

Kapag naibahagi mo na ang link sa iyong mga customer, i-redirect nila sila sa unang hakbang ng iyong signup flow.
