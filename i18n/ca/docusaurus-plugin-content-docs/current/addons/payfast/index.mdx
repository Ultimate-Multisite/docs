# PayFast {#payfast}

Ultimate Multisite: Passarela Payfast

Afegeix Payfast — una passarela de pagaments en línia ràpida i segura a Sud-àfrica — com a opció de pagament per a Ultimate Multisite.

## Instal·lació {#installation}

1. Carrega 'ultimate-multisite-payfast' al directori '/wp-content/plugins/'
2. Activa el plugin des del menú 'Plugins' de WordPress
3. Segueix el Wizard pas a pas per configurar el plugin

## Guia d'ús {#usage-guide}

PayFast és un processador de pagaments en línia líder per a empreses de Sud-àfrica. Està dedicat a ajudar els mercaders en línia a créixer fent que les transaccions siguin fluides i segures. Segons la petició d'un usuari d'Ultimate Multisite, vam decidir crear un add-on que permetrà la integració amb PayFast. Aquest article mostra com configurar PayFast com a la teva passarela de pagaments per rebre pagaments a la teva xarxa Ultimate Multisite.

El primer pas és instal·lar l'add-on.

# Com instal·lar l'add-on de Payfast: {#how-to-install-the-payfast-add-on}

Navega a la pàgina d'add-ons i selecciona l'add-on Ultimate Multisite: Payfast Gateway.

![Add-ons page](/img/admin/addons-page.png)

La finestra de l'add-on apareixerà. Fes clic a Install Now.

![Add-ons page with install option](/img/admin/addons-page.png)

Després de completar l'instal·lació, seràs redirigit a la pàgina de plugins. Aquí, només has de fer clic a Network Activate i l'add-on _Payfast_ s'activarà a la teva xarxa.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# Configurar PayFast {#setting-up-payfast}

Després d'instal·lar l'add-on, ja pots accedir a la configuració d'Ultimate Multisite i començar a configurar PayFast.

A la pàgina de **Ultimate Multisite settings**, pots trobar la pestanya "Payments". Aquesta pestanya ha de contenir "**Payfast** " juntament amb un botó de toggle per activar o desactivar aquesta passarela de pagaments.

![Payment gateway settings](/img/config/settings-payment-gateways.png)Continua i fes clic al botó de toggle per activar-lo. Això farà que la caixa s'expansi per mostrar diferents camps que has de rellenar, com ara **Mechant ID, Merchant Key i PassPhrase.** Pots obtenir aquesta informació del teu compte de **Payfast**

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

El pas següent és iniciar sessió al teu compte de PayFast per obtenir la informació que necessites.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->Només has de copiar i enganxar les credencials i rellenar els detalls a la teva configuració d'Ultimate Multisite. Després, fes clic al botó **"Save settings"** per guardar els canvis.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

És tot, la teva integració de pagaments de PayFast està completada!
