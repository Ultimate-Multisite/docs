# Acció: switch_network

S'executa quan es canvia el context de la xarxa actual.

## Paràmetres

| Nom | Tipus | Descripció |
|------|------|-------------|
| $new_network_id | `int` | L'ID de la xarxa a la qual es canvia. |
| $old_network_id | `int` | L'ID de l'anterior xarxa actual. |
