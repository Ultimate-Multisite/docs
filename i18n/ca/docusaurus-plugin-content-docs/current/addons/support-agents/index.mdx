# Agents de Suport

Gestionar una xarxa completa per tu mateix no és una tasca fàcil. A mesura que la teva xarxa comenci a créixer, notaràs una demanda de suport creixent. Serà una bona idea incorporar persones que et puguin ajudar a allíviar la càrrega de treball. No obstant això, donar accés/permís complet és arriscat i pot comprometre la teva xarxa. Aquí és on entra aquest add-on d'Agent de Suport.

Aquest add-on et permet crear un nou tipus d'usuari de WordPress anomenat Agent de Suport, i et dona control total sobre quins permisos té aquest usuari. Conceder permisos és tan fàcil com activar un interruptor (*toggle*) i desactivarlo si decides eliminar-lo.

## Funcionalitats

- Llista detallada de permisos de WordPress i Ultimate Multisite
- Activar i desactivar l'accés amb facilitat
- Crear un nombre il·limitat de comptes d'Agents

## Instal·lació

1. Carrega 'ultimate-multisite-support-agents' al directori '/wp-content/plugins/'
2. Activa el plugin des del menú 'Plugins' de WordPress
3. Segueix el *Wizard* pas a pas per configurar el plugin
