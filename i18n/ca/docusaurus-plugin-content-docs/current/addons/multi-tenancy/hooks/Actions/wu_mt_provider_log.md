# Acció: wu_mt_provider_log {#action-wumtproviderlog}

S'executa quan un proveïdor registra un missatge.

## Paràmetres {#parameters}

| Nom | Tipus | Descripció |
|------|------|-------------|
| $message | `string` | El missatge de registre. |
| $level | `string` | El nivell de registre. |
| $context | `array` | Context addicional. |
