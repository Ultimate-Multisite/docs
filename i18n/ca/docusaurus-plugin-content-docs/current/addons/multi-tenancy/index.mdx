# Multitenència

Distribueix subllocs entre servidors i bases de dades per millorar el rendiment i la seguretat

## Aspectes destacats de la versió 1.2.0

Multi-Tenancy 1.2.0 afegeix la base d’arrendatari sobirà per als fluxos de treball d’aïllament en producció:

- Autoinici de sessió d’arrendatari sense estat amb visites SSO de curta durada i protegides contra repetició.
- Aïllament de base de dades i sistema de fitxers per sublloc per a arrendataris sobirans.
- Capacitat de multitenència de Hostinger per a l’aprovisionament d’arrendataris conscient de l’amfitrió.
- Portes de verificació de migració sobirana per a comprovacions de dades heretades i processament d’enviament asíncron.
- Enllaç configurable de l’amfitrió de base de dades a la mateixa màquina per a instal·lacions Bedrock, FrankenPHP i localhost.
- Millores en l’arrencada de l’arrendatari, l’escriptor de base de dades, l’arrencada d’esquema i l’aprovisionament d’usuaris.
- Desmuntatge sobirà més segur que neteja les credencials de l’arrendatari.

Consulta les guies d’usuari per al [flux de treball de migració sobirana](/user-guide/getting-started/sovereign-migration-workflow), l’[aïllament de multitenència](/user-guide/configuration/multi-tenancy-isolation) i les [operacions d’arrendatari sobirà](/user-guide/administration/sovereign-tenant-operations).

## Requisits

- WordPress 5.3 o superior
- PHP 7.4 o superior
- plugin Multisite Ultimate (actiu)

## Instal·lació

1. Puja els fitxers del complement al directori `/wp-content/plugins/`
2. Activa el plugin a través del menú 'Plugins' de WordPress
3. Configura els paràmetres del complement a l’administració de xarxa

### Proves

El complement inclou un marc de proves complet:

- **Proves unitàries**: Proves basades en PHPUnit amb el conjunt de proves de WordPress
- **Classe base de proves**: Proporciona mètodes d’ajuda per a escenaris de prova habituals
- **Cobertura de proves**: Configurada per fer el seguiment de la cobertura del codi
- **Proves multisite**: Les proves s’executen en un entorn multisite

### Estàndards de codi

- **Estàndards de codificació de WordPress**: Aplicats mitjançant PHPCS
- **Compatibilitat amb PHP 7.4+**: Funcions modernes de PHP compatibles
- **Anàlisi estàtica**: Integració de PHPStan per a la comprovació de tipus
- **Modernització del codi**: Regles de Rector per a actualitzacions de PHP

### Sistema de compilació

- **Processament d’actius**: Minificació per a fitxers CSS/JS
- **Traducció**: Generació de fitxer POT
- **Empaquetament**: Creació d’arxiu per a la distribució
- **Mode de desenvolupament**: Actius no minificats per a la depuració

## Configuració

El complement admet diverses opcions de configuració accessibles a través de la interfície d’administració de xarxa.

### Configuració d’arrendatari sobirà

Per als arrendataris sobirans, configura la base de dades de l’arrendatari, l’arrel del sistema de fitxers, l’enllaç de l’amfitrió de base de dades i la capacitat de l’amfitrió abans d’encaminar el trànsit cap a l’arrendatari. Executa la verificació de migració després de cada canvi que afecti les credencials de la base de dades, l’enllaç de l’amfitrió, l’encaminament o l’aprovisionament d’usuaris.

Utilitza `localhost` només quan el temps d’execució de l’arrendatari i les concessions de la base de dades esperin accés per sòcol local o a la mateixa màquina. Utilitza `127.0.0.1` o un nom d’amfitrió de servei quan aquest sigui l’amfitrió de concessió configurat pel teu proveïdor.

### Capacitat de Hostinger

La capacitat de multitenència de Hostinger funciona amb el token d’integració compartit de Hostinger. Confirma que el domini és present a hPanel, que el DNS apunta al compte de Hostinger i que els paràmetres de la base de dades de l’arrendatari coincideixen amb hPanel abans de la verificació.

## Hooks i filtres

### Accions

- `ultimate-multisite-addon-template_init` - S’activa després de la inicialització del complement
- `ultimate-multisite-addon-template_loaded` - S’activa després que tots els fitxers del complement s’hagin carregat

### Filtres

- `ultimate-multisite-addon-template_settings` - Modifica els paràmetres del complement
- `ultimate-multisite-addon-template_enabled` - Sobreescriu l’estat d’activació/desactivació del complement
