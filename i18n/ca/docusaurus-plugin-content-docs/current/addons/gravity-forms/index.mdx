# Gravity Forms

**Transforma l'experiència de registre dels teus usuaris amb la potència de Gravity Forms**

Integra de manera perfecta el constructor de forms més avançat del món amb Multisite Ultimate per crear forms de registre impressionants i amb una alta taxa de conversió que impulsen les inscripcions de membres i augmenten els ingressos.

### 🎯 **Augmenta les conversió fins un 40%** {#-boost-conversions-by-up-to-40}
Substitueix els forms de pagament predeterminats i tediosos per forms de Gravity Forms bonics i personalitzats que converteixen els visitants en clients pagadors.

Suporta tant el redireccionament de pagament directes com l'integració amb WooCommerce per a la màxima compatibilitat amb passarelles de pagament.

## 🚦 Com funciona {#-how-it-works}

### 1. **Diseña el teu form** {#1-design-your-form}
Utilitza el constructor *drag-and-drop* de Gravity Forms per crear l'experiència de registre perfecta.

### 2. **Mapeja els teus camps** {#2-map-your-fields}
Conecta els camps del form als punts de dades de Multisite Ultimate amb la nostra interfície de mapeig visual.

### 3. **Configura el pagament** {#3-configure-payment}
Escull entre redireccionament de pagament o integració amb WooCommerce per al processament de pagament.

### 4. **Publica** {#4-go-live}
Publica el teu form i comença a convertir visitants en clients pagadors!
