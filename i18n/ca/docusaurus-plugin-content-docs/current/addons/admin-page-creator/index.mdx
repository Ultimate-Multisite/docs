# Creador de pàgines d'administració {#admin-page-creator}

## Contingut {#content}
El creador de pàgines d'administració et permetrà afegir fàcilment noves pàgines i subpàgines dins del panell d'administració de WordPress. Es pot utilitzar per crear una pàgina de benvinguda o una pàgina tutorial que serveixi com a pàgina de destinació (landing page) per a nous usuaris. Pots utilitzar page builders com Beaver Builder, Elementor, Brizy i Oxygen per dissenyar les pàgines sense complicacions.

També ofereix suport per a HTML, CSS i JavaScript personalitzats, així com control d'accés basat en el pla i el rol de l'usuari!

## Funcionalitats {#features}
* Suport a HTML i CSS personalitzats
* Suport a JavaScript
* Mode de contingut PHP
* Permissió basada en el rol de l'usuari
* Suport a page builders

## Instal·lació {#installation}

1. Carrega **ultimate-multisite-admin-page-creator** al directori **/wp-content/plugins/**
2. Activa el plugin mitjançant el menú **Plugins** del Panell d'Administració de la teva rede (Network Admin)

## Guia d'ús {#usage-guide}

Mentre que WordPress facilita molt l'afecció de contingut al front end dels llocs web, crear pàgines i subpàgines a la banda d'administració requereix algunes habilitats bàsiques de codificació. Ultimate Multisite: Admin Page Creator té com a objectiu portar aquesta mateixa facilitat d'ús del front end al back end dels llocs web de la teva rede.

## Per què utilitzar Admin Page Creator? {#why-use-admin-page-creator}

Potser vols oferir una pàgina personalitzada al panell d'administració dels teus clients amb tutorials, o potser vols afegir una pàgina personalitzada amb FAQ i un widget de contacte. Les possibilitats són infinites amb Admin Page Creator, i això és el que explorarem en el tutorial d'avui.

## Creant la teva primera Pàgina d'Administració Personalitzada {#creating-your-first-custom-admin-page}

Després d'instal·lar i activar Ultimate Multisite: Admin Page Creator, dirigeix-te al menú d'administració de la teva rede i accedeix a Pàgines d'Administració (Admin Pages).

![Custom Admin Pages list on Network Admin](/img/admin/admin-pages-list.png)

_Llista de Pàgines d'Administració Personalitzades al Panell de la Rede_

Veràs una llista de totes les pàgines d'administració que has creat fins ara (ja que és la teva primera visita, la llista estarà buida). Continua i fes clic a **Add new Admin Page**. Seràs redirigit a l'editor de Pàgina d'Administració.

## Els elements d'una Pàgina d'Administració Personalitzada {#the-elements-of-a-custom-admin-page}

A l'editor de Pàgina d'Administració, tindràs un nombre de opcions diferents per personalitzar la teva pàgina.

### Títol de la Pàgina {#page-title}

Aquest és el títol que es mostrarà en la part superior de la pàgina, no l'etiqueta del menú d'administració.

Ultimate Multisite: Admin Page Creator et ofereix dues opcions quan es tracta del contingut de la pàgina: pots utilitzar l'editor WYSIWYG predeterminat de WordPress o passar a un editor HTML.

## Opcions de Menú {#menu-options}

A la banda, a la caixa meta Opcions Generals (General Options), podràs configurar el tipus de menú, l'etiqueta i molt més.

### Tipus de Menú {#menu-types}

La teva nova pàgina d'administració personalitzada pot tenir dos tipus de menú diferents: menú de nivell superior (top-level menu) i submenú.

Una pàgina de menú de nivell superior, com el que suggereix el nom, afegirà la pàgina que acabes de crear al menú d'administració juntament amb les pàgines predeterminades de WordPress.

Si seleccioneixes l'opció de nivell superior, també podreu seleccionar l'ordre del menú i un ícone de menú.

Si vols que aquesta pàgina sigui una pàgina de submenú, podràs seleccionar una pàgina mare a la qual adjuntar aquesta pàgina. Pots afegir la teva pàgina personalitzada a pàgines predeterminades de WordPress o a les teves pàgines d'administració personalitzades de nivell superior.

## Opcions Avançades {#advanced-options}

Hauries de poder afegir tot tipus de contingut a les teves pàgines d'administració personalitzades. Això inclou regles i fitxers CSS, JavaScript i biblioteques externes. Pots fer-ho utilitzant la pestanya Opcions Avançades (Advanced Options Tab).

### CSS Personalitzat {#custom-css}

A la pestanya CSS, pots afegir el teu propi codi CSS personalitzat, així com importar fitxers CSS de sites remots (com CDNs, Google Fonts, etc.).

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_També es suporta CSS personalitzat. Pots incloure fitxers de estil externs!_

### JavaScript Personalitzat {#custom-javascript}

El mateix és vàlid per a JavaScript. Pots afegir biblioteques externes per utilitzar-les en el teu codi JavaScript personalitzat.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Afegir el teu propi codi JavaScript!_

## Permissió {#permissions}

També a les Opcions Avançades, podràs seleccionar quins sub-sites tindran accés a aquesta pàgina. Això és útil si planegeixes servir contingut exclusiu diferent a diferents nivells de pla, per exemple.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Tens control total sobre on va mostrar-se aquesta pàgina_

**Important** : Les dues condicions han de ser complides perquè la pàgina es mostri. Això significa que si l'usuari és client d'un pla determinat, però no té un dels rols seleccionats (o viceversa), la pàgina no apareixerà al seu panell.

## Actiu {#active}

Si vols deshabilitar una pàgina d'administració personalitzada sense eliminar-la o modificar els permisos, només desmarca l'opció Actiu (Active).

Desactivar una pàgina de nivell superior desactivarà automàticament totes les subpàgines adjuntes a aquesta pàgina mare.
