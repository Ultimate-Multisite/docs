# Filtre: wu_apc_should_display_admin_menu

Aquest filtre permet als administradors ocultar les pàgines del menú d'administració si han acabat el procés de personalització.

## Paràmetres

| Nom | Tipus | Descripció |
|------|------|-------------|
| $should_display_admin_menu | `bool` | Si hem de mostrar les pàgines. |
| $network_active | `bool` | Comprovar si el plugin està actiu per l'administració de la xarxa. |
