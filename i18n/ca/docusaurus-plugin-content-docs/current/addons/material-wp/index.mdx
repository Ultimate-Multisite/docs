# Material WP {#material-wp}

L'interfície d'administració de WordPress no ha canviat des de fa molt de temps. Però mai t'has preguntat com es veuria una versió de l'administració amb Material Design?

Material WP és un tema d'administració desenvolupat per donar-te aquest aspecte i experiència, ajudant-te a proporcionar als teus clients un Dashboard més personalitzat.

## Funcionalitats {#features}

- Suport per a CSS personalitzat
- Configuració de estil fàcil
- Reordenació del menú
- Import/export fàcil

## Documentació {#documentation}

**Personalitza el teu Panell d'Administració amb Material WP**

## Instal·lació {#installation}

1. Carrega 'material-wp' al directori '/wp-content/plugins/'
2. Activa el plugin des del menú 'Plugins' de WordPress

## Guia d'ús {#usage-guide}

L'interfície d'administració de WordPress no ha canviat des de fa molt de temps. Però mai t'has preguntat com es veuria una versió de l'administració amb Material Design?
