# Loco Translate {#loco-translate}

L'addon Loco Translate Integration per a Ultimate Multisite porta capacitats de traducció específiques del lloc a la teva xarxa multisite de WordPress. Si alguna vegada has necessitat que diferents llocs de la teva xarxa tinguin diferents traduccions del mateix plugin o tema, aquest addon és la solució perfecta.

**Què fa aquest addon**

Aquest addon integra de manera perfecta Ultimate Multisite amb el popular plugin Loco Translate, permetent que cada subsite de la teva xarxa mantingui les seves pròpies traduccions personalitzades. Ja sigui que estiguis gestionant una plataforma multi-regional, serveint a diferents comunitats lingüístiques, o simplement necessitis flexibilitat de traducció a tota la teva xarxa, aquest addon ho fa fàcil.

**Ideal per:**

* Empreses multi-regionals amb necessitats de localització diferents
* Institucions educatives que serveixen comunitats diverses
* Plataformes SaaS que ofereixen suport multilingüe
* Agències web que gestionen múltiples llocs de clients
* Qualsevol xarxa multisite que requereixi flexibilitat de traducció

## Instal·lació {#installation}

1. Assegura't que Ultimate Multisite està instal·lat i que la xarxa està activada
2. Instala i activa el plugin Loco Translate
3. Carrega aquest addon al teu directori de plugins
4. Activa l'addon Loco Translate Integration a nivell de xarxa
5. Comença a crear traduccions específiques del lloc mitjançant Loco Translate en qualsevol subsite

És tot! La integració funciona automàticament sense cap configuració addicional.

## Preguntes Freqüents {#frequently-asked-questions}

## Necessito configurar alguna cosa després de l'instal·lació? {#do-i-need-to-configure-anything-after-installation}

No! La integració és automàtica. Tanmateix l'addon i Loco Translate estiguin actius, els subsites poden començar immediatament a crear les seves pròpies traduccions personalitzades.

## Què passa si un subsite no crea traduccions personalitzades? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

El lloc utilitzarà automàticament les traduccions a nivell de xarxa com a reserva. Res encara, i tot funciona de manera fluida.

## Els subsites poden accedir a les traduccions a nivell de xarxa? {#can-subsites-access-network-wide-translations}

Els admins regulars del lloc només veuen i editen les traduccions del seu propi lloc. Els superadmins poden accedir tant a les traduccions a nivell de xarxa com a les específiques del lloc, depenent del lloc que estiguin veient.

## Funciona amb plugins i temes? {#does-this-work-with-both-plugins-and-themes}

Sí! Les traduccions específiques del lloc funcionen tant per plugins com per temes de WordPress.

## Això ralentitzarà la meva xarxa multisite? {#will-this-slow-down-my-multisite-network}

No. L'addon utilitza un càrrega de fitxer eficient que només comprueba les traduccions específiques del lloc quan és necessari. No hi ha un impacte visible en el rendiment.

## Puc copiar les traduccions de la xarxa a un lloc específic com a punt de partida? {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

Sí! Mentre edites traduccions a Loco Translate, pots començar fàcilment amb les traduccions de la xarxa i personalitzar-les per a cada lloc.

## Funciona amb WooCommerce i altres plugins? {#does-this-work-with-woocommerce-and-other-plugins}

Absolutament! Qualsevol plugin o tema que utilitzi funcions de traducció estàndard de WordPress funcionarà amb les traduccions específiques del lloc.
