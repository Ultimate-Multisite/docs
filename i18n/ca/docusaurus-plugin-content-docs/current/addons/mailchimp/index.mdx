# MailChimp

Multisite Ultimate: Integració amb MailChimp

Vincula la teva xarxa Multisite Ultimate a MailChimp. Pots vincular plans específics a les Llistes de MailChimp i afegir automàticament els subscriptors a aquestes llistes quan es registren.

## Instal·lació

1. Carrega 'ultimate-multisite-mailchimp' al directori '/wp-content/plugins/'
2. Activa el plugin des del menú 'Plugins' de l'Administració de la Xarxa de WordPress

## Guia d'ús

Sincronitzar els correus electrònics dels subscriptors amb les teves llistes de MailChimp és molt fàcil amb el nostre Add-on d'Integració Ultimate Multisite: MailChimp. Aquest tutorial et ajudarà a configurar-ho tot!

## Obtenció d'una clau API

Abans de començar a veure les diferents configuracions disponibles, necessitem una clau API de MailChimp. Això permetrà a Ultimate Multisite comunicar-se amb l'API de MailChimp per recuperar llistes, grups i afegir adreces de correu electrònic a aquestes llistes i grups quan els usuaris es registren.

Segueix aquest tutorial per obtenir la teva pròpia clau API de MailChimp: [About API Keys | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

<!-- Screenshot unavailable: MailChimp API key settings in Ultimate Multisite Add-on Settings -->

_Afegir la teva clau API de MailChimp a_ _**Configuració de Ultimate Multisite - > Configuració de l'Add-on -> MailChimp**_

## Mode d'integració

L'add-on d'Integració de MailChimp ofereix dos "modes d'integració", cosa que significa que hi ha dues maneres diferents de potre utilitzar l'add-on per sincronitzar adreces de correu electrònic.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_Dos modes d'integració per a una flexibilitat més gran!_

### Mode 1: Múltiples Llistes

El mode per defecte utilitza **Múltiples Llistes**. Aquest mode et permet seleccionar múltiples llistes de MailChimp per a cada pla. És la millor opció si segmentes els teus usuaris en diferents llistes.

Per exemple, suposem que tens 3 llistes diferents a la teva compte de MailChimp: **Newsletter**, **Plan A** i **Plan B**. Vols que tots els usuaris, independentment del pla, també estiguin inclosos a la llista Newsletter, perquè no es perdin les teves actualitzacions.

Per aconseguir-ho, pots utilitzar el mode de Múltiples Llistes per afegir les llistes **Newsletter** i **Plan A** a la Configuració de MailChimp del Plan A, i **Newsletter** i **Plan B** a la Configuració de MailChimp del Plan B.

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_Selecciona les llistes per defecte per comptes nous i per comptes cancel·lats_

### Mode 2: Llista única + Múltiples Grups

MailChimp et permet segmentar la teva llista en grups més petits d'usuàri appellats **Grups**. Això et permet tenir més control, ja que pots enviar una campanya a un subconjunt específic dels teus subscriptors sense haver de crear múltiples còpies de la mateixa campanya a MailChimp (cosa que dificulta l'anàlisi de les mètriques d'aquesta campanya més tard).

Per exemple, pots tenir una llista única anomenada **Subscriptors de la meva Xarxa**, i dins hi tenir grups per **Plan A** i **Plan B**.

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_Selecciona les llistes i grups per defecte, si utilitzes el mode d'integració "llista única + múltiples grups"_

## Llistes i Grups per defecte

Després de seleccionar quin mode d'integració té més sentit per al teu cas d'ús concret, és hora de seleccionar les llistes (i grups, depenent del mode) per defecte. La llista per defecte es trigarà quan un pla concret no té una Llista específica seleccionada.

També pots seleccionar la llista per defecte per comptes cancel·lats. Quan un usuari termini la seva subscripció i elimini el seu compte de la teva xarxa, la seva adreça de correu electrònic es trencarà a la llista configurada per comptes cancel·lats.

## Activar la Subscripció Dupla

A la mateixa pàgina de configuració, podràs activar la **Subscripció Dupla**. Aquesta opció afegirà l'usuari tant a la llista per defecte seleccionada COM A a la llista seleccionada per al pla de l'usuari. Si aquesta opció no està activada, l'usuari només s'afegirà a la llista seleccionada per al pla. Si no hi ha cap llista configurada per aquest pla en particular, l'usuari s'afegirà a la llista per defecte seleccionada.

<!-- Screenshot unavailable: MailChimp double subscription toggle setting -->

_Pots tenir els teus usuaris subscriptos tant a la llista per defecte com a la llista del pla al registrar-se_

## Seleccionar Llistes i Grups per cada pla

Veu a **Plans** i selecciona el pla que vols editar. Nota que a la meta-caja **Opcions avançades** hi haurà una nova pestanya anomenada MailChimp.

En aquesta pestanya, podràs seleccionar múltiples llistes o una llista única i múltiples grups (depenent del mode d'integració) per a aquest pla.

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_Selecciona les Llistes i Grups que vols utilitzar per cada un dels teus plans_

Quan un nou usuari es registri per aquest pla, la seva adreça de correu electrònic s'afegirà a les llistes i grups seleccionats!
