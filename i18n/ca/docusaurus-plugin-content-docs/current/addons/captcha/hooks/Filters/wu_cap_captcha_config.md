# Filtre: wu_cap_captcha_config

Filtra la configuració de Cap Captcha.

Permet als desenvolupadors personalitzar les configuracions de Cap Captcha.

## Paràmetres {#parameters}

| Name | Type | Descripció |
|------|------|-------------|
| $config | `array` | L'array de configuració de Cap. |
| $level | `string` | El nivell de seguretat (ràpid, mitjà, màx). |
