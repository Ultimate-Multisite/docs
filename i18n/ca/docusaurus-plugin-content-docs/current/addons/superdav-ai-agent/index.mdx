# Superdav AI Agent

Superdav AI Agent és un assistent d'IA avançat per a WordPress que amplia Gratis AI Agent amb habilitats conscients del tema, capacitats millorades de cerca a internet, funcionalitats de constructor de llocs i integracions d'automatització opcionals. La versió 1.18.0 afegeix eines de lectura de Google Calendar, assignació de contactes i assistents, portes d'aprovació humana, deduplicació de recordatoris, notificacions SMS de TextBee, aprovisionament del servei Superdav gestionat i un paquet complementari Advanced distribuït per separat per a fluxos de treball de desenvolupadors de confiança.

## Funcions clau {#key-features}

- **Habilitats conscients del tema** — adapta automàticament l'orientació i les capacitats segons el teu tema WordPress actiu i les extensions instal·lades
- **Diversos proveïdors de cerca a internet** — tria entre Brave Search i Tavily per obtenir resultats de cerca més rics i capacitats de recerca avançades
- **Formularis de contacte del constructor de llocs** — crea formularis de contacte plenament funcionals directament des de la interfície de xat utilitzant l'extensió de formularis instal·lada
- **Constructor de llocs millorat** — plans millorats de construcció de llocs en múltiples fases amb millor descobriment d'extensions i recuperació d'errors
- **Habilitats de tema integrades** — orientació especialitzada per a temes de blocs, temes clàssics, Kadence Blocks i Kadence Theme
- **Integració millorada amb WooCommerce** — refactoritzada per utilitzar les API natives de WooCommerce per a una millor fiabilitat i compatibilitat
- **Detecció automàtica de proveïdors** — detecta i actualitza automàticament les extensions i els temes disponibles quan s'activen o es desactiven
- **Generació de logotips personalitzats** — genera i incrusta logotips SVG personalitzats amb sanejament automàtic segur per a espais de noms
- **Validació de contrast de paleta** — comprova parelles de colors per garantir el compliment d'accessibilitat WCAG abans d'aplicar-les al teu tema
- **Descobriment millorat amb fotos** — puja fotos durant l'entrevista de descobriment de Theme Builder per influir en la direcció del disseny
- **Menús d'hostaleria** — genera pàgines estructurades de menús de menjar i beguda per a restaurants, cafeteries i negocis d'hostaleria
- **Previsualitzacions d'escriptori i mòbil** — previsualitza el teu disseny en dispositius d'escriptori i mòbils durant la selecció de la direcció de disseny
- **Raspament de llocs de nivell 1** — sd-ai-agent/site-scrape ara està disponible per defecte per analitzar webs de competidors i d'inspiració
- **Eines de lectura de Google Calendar** — llegeix calendaris i esdeveniments configurats perquè les automatitzacions puguin raonar sobre el context de la planificació propera
- **Assignació de contactes i assistents** — relaciona els participants del calendari amb usuaris i contactes coneguts del lloc abans de generar tasques de seguiment
- **Portes d'aprovació humana** — pausa automatitzacions sensibles per revisar-les abans d'enviar missatges, reprendre la feina o fer canvis
- **Registres de recordatoris i deduplicació** — fes seguiment dels recordatoris que ja s'han enviat perquè les execucions repetides d'automatitzacions no notifiquin dues vegades el mateix destinatari
- **Notificacions SMS de TextBee** — envia notificacions de missatge de text configurades mitjançant TextBee quan hi ha un proveïdor d'SMS habilitat
- **Aprovisionament del servei Superdav gestionat** — connecta llocs compatibles a endpoints Superdav allotjats amb configuració automàtica en lloc de configuració manual d'endpoints
- **Paquet complementari Advanced** — eines per a desenvolupadors de confiança distribuïdes per separat per a fluxos de treball de sistema de fitxers, base de dades, WP-CLI, dispatcher REST, constructor d'extensions, instantània git, gestió d'usuaris i benchmarks

## Requisits {#requirements}

- WordPress 6.0 o superior
- PHP 8.1 o superior
- WordPress Multisite (activat en xarxa) o instal·lació d'un sol lloc
- Una extensió activa de connector de proveïdor d'IA (Anthropic, OpenAI o compatible)
- Gratis AI Agent v1.9.0 o superior (Superdav AI Agent amplia Gratis AI Agent)

## Instal·lació {#installation}

1. Puja els fitxers de l'extensió a `/wp-content/plugins/superdav-ai-agent`
2. Activa l'extensió mitjançant la pantalla **Extensions** de WordPress
3. Assegura't que Gratis AI Agent també estigui activat
4. Ves a **Gratis AI Agent → Settings** i configura el teu proveïdor de cerca a internet, la connexió del servei Superdav gestionat, Google Calendar o les credencials de TextBee segons calgui

El paquet de WordPress.org conté les funcions principals de Superdav AI Agent. El paquet complementari Advanced es compila i distribueix per separat per a entorns on les eines de desenvolupador de confiança s'instal·len i s'autoritzen explícitament.

## Inici ràpid {#quick-start}

Un cop activat, Superdav AI Agent millora automàticament la teva instal·lació de Gratis AI Agent amb habilitats conscients del tema i capacitats addicionals. Obre el panell **Gratis AI Agent** a la barra lateral d'administració de WordPress i comença a utilitzar les noves funcions.

**Instruccions d'exemple:**

```
Afegeix un formulari de contacte a la pàgina Contacte.
```

```
Cerca al web les últimes millors pràctiques de seguretat de WordPress.
```

```
Crea una secció hero utilitzant els patrons de blocs del tema actiu.
```

```
Revisa els esdeveniments configurats de Google Calendar de demà i redacta recordatoris per als assistents.
```

```
Envia un recordatori per SMS després que jo aprovi el missatge.
```

## Habilitats conscients del tema {#theme-aware-skills}

Superdav AI Agent detecta automàticament el teu tema actiu i les extensions instal·lades, carregant l'habilitat conscient del tema adequada al context de l'assistent. Les habilitats disponibles inclouen:

| Habilitat | S'aplica a |
|---|---|
| **Temes de blocs** | Temes que utilitzen l'arquitectura basada en blocs de WordPress (edició completa del lloc) |
| **Temes clàssics** | Temes tradicionals de WordPress que utilitzen plantilles PHP i l'editor clàssic |
| **Kadence Blocks** | Llocs amb l'extensió Kadence Blocks activa |
| **Kadence Theme** | Llocs amb el tema Kadence com a tema actiu |

Consulta [Habilitats conscients del tema](../../user-guide/configuration/theme-aware-skills) per obtenir orientació detallada sobre cada habilitat.

## Proveïdors de cerca a internet {#internet-search-providers}

Superdav AI Agent v1.10.0 afegeix **Tavily** com a alternativa a Brave Search per a consultes de cerca a internet. Tots dos proveïdors ofereixen resultats de cerca en temps real i es poden configurar a **Gratis AI Agent → Settings → Advanced**.

| Proveïdor | Funcionalitats | Configuració |
|---|---|---|
| **Brave Search** | Resultats de cerca ràpids i centrats en la privadesa | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Resultats de cerca més rics, capacitats de recerca avançades | [Tavily API](https://tavily.com/) |

Consulta [Cerca a Internet](../../user-guide/configuration/internet-search) per a les instruccions de configuració.

## Fluxos de treball de calendari, aprovació i notificació {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 amplia els fluxos de treball d’automatització més enllà de les tasques de contingut i de creació de llocs:

- **Les eines de Google Calendar** permeten que l’agent llegeixi calendaris configurats i detalls d’esdeveniments quan les credencials del calendari estan disponibles.
- **Els assistents de contactes i assistents a esdeveniments** associen els participants dels esdeveniments amb usuaris de WordPress o contactes coneguts abans de preparar recordatoris o tasques de seguiment.
- **Les portes d’aprovació humana** permeten que una automatització es posi en pausa fins que un usuari autoritzat revisi l’acció proposada.
- **Els registres de recordatoris** eviten recordatoris de calendari o notificació duplicats quan una automatització es reintenta o es programa repetidament.
- **El suport per a SMS de TextBee** permet notificacions per missatge de text quan el proveïdor TextBee està configurat.

Fes servir aquestes eines per a assistència conscient de l’horari, seguiment d’assistents i fluxos de treball de recordatoris que encara requereixen revisió humana abans d’enviar missatges sensibles.

## Formularis de contacte del constructor de llocs {#site-builder-contact-forms}

L’agent constructor de llocs ara pot crear formularis de contacte directament des de la interfície de xat. Només cal que demanis al constructor de llocs que creï un formulari de contacte, i detectarà automàticament l’extensió de formularis instal·lada (Contact Form 7, WPForms, Fluent Forms o Gravity Forms) i generarà un formulari llest per utilitzar.

Consulta [Formularis de contacte del constructor de llocs](../../user-guide/administration/site-builder-contact-forms) per obtenir orientació detallada.

## Configuració {#configuration}

Tots els paràmetres són a **Gratis AI Agent → Settings** a l’administració de WordPress. Superdav AI Agent afegeix les opcions de configuració següents:

| Paràmetre | Descripció |
|---|---|
| **Proveïdor de cerca a Internet** | Tria entre Brave Search i Tavily (opcional) |
| **Clau API de Tavily** | Clau API per a la cerca a Internet de Tavily (si Tavily està seleccionat) |
| **Servei Superdav gestionat** | Connecta llocs compatibles al servei Superdav allotjat amb aprovisionament automàtic de punts de connexió |
| **Credencials de Google Calendar** | Configura l’accés al calendari perquè l’agent pugui llegir calendaris i esdeveniments |
| **Proveïdor de SMS TextBee** | Configura les credencials de TextBee per a fluxos de treball de notificacions SMS aprovades |

## Paquet complementari avançat {#advanced-companion-package}

Superdav AI Agent v1.18.0 afegeix un paquet complementari avançat distribuït per separat per a fluxos de treball de desenvolupador de confiança. Aquestes capacitats no s’inclouen al paquet principal de WordPress.org. Instal·la el paquet complementari només en llocs on els administradors confiïn explícitament que l’agent accedeixi a eines de desenvolupador.

Les eines avançades inclouen operacions del sistema de fitxers, accés a bases de dades, WP-CLI, fluxos de treball de despatxador REST, utilitats de creació d’extensions, instantànies git, assistents de gestió d’usuaris i eines de proves de rendiment. Mantén aquestes eines limitades a administradors de confiança i revisa la configuració de portes d’aprovació abans d’activar-les en llocs de producció.

## Millores a v1.18.0 {#improvements-in-v1180}

- **Suport de Google Calendar** — llegeix calendaris i esdeveniments configurats per a automatitzacions conscients de l’horari

- **Portes d’aprovació humana** — posa en pausa automatitzacions sensibles per revisar-les abans de continuar
- **Desduplicació de recordatoris** — registra recordatoris per evitar notificacions duplicades en reintents o execucions repetides
- **Proveïdor de SMS TextBee** — envia notificacions configurades per missatge de text després de les aprovacions requerides
- **Configuració del servei Superdav gestionat** — aprovisiona automàticament punts de connexió del servei allotjat per a llocs compatibles
- **Distribució del complement avançat** — publica eines per a desenvolupadors de confiança per separat de la versió principal de WordPress.org
- **Correccions de fiabilitat** — millora la selecció de models, els temps d’espera de sol·licituds, la gestió del model de configuració per defecte, la gestió de text de raonament, l’orientació de reintents, l’enduriment de tokens de calendari, l’enduriment de recordatoris, els llançaments d’incorporació, la represa d’habilitats confirmades i els problemes de revisió d’empaquetatge de WordPress.org

- **Eina d’extracció de llocs de nivell 1** — sd-ai-agent/site-scrape ara està disponible per defecte a Theme Builder per analitzar llocs web de competidors i d’inspiració
- **Enllaços d’acció de fila de plugin** — corregits i reanomenats per a més claredat

## Millores a v1.10.0 {#improvements-in-v1100}

- **Integració de WooCommerce refactoritzada** — ara utilitza les API natives de WooCommerce per a una fiabilitat i compatibilitat millors
- **Actualització automàtica de la llista de proveïdors** — detecta automàticament quan s’activen o es desactiven plugins i actualitza la llista de proveïdors disponibles

## Correccions d’errors a v1.10.0 {#bug-fixes-in-v1100}

- S’ha corregit la capacitat `navigate-to` que provocava bucles de recàrrega infinits en algunes pàgines d’administració
- S’ha corregit que la capacitat `list-posts` no resolgués correctament els noms de categories i etiquetes a slugs
- S’han corregit les ordres WP-CLI a les quals faltaven àlies d’espai de noms després d’una refactorització anterior
- S’ha corregit la gestió adequada per part de l’automatització d’esdeveniments dels llocs on les taules d’automatització encara no s’han creat
- S’ha corregit que la capacitat `memory-save` utilitzés el prefix d’espai de noms correcte al constructor d’instruccions del sistema
- S’ha corregit que els resultats escalars d’eines s’embolcallessin correctament abans de retornar-se a l’AI
- S’ha corregit que les estadístiques d’ús gestionessin correctament el format de clau de capacitat heretat en actualitzar des de versions anteriors
