# Acció: wp_ultimo_vat_load {#action-wpultimovatload}

S'executa quan totes les dependències han estat carregades.

Permaneix als desenvolupadors de plugins afegir nova funcionalitat. Per exemple, suport per a nous proveïdors d'hosting, etc.
