# Filtre: wu_upgrade_to_unlock_url

Permet als desenvolupadors modificar l'URL d'actualització per desbloquejar.

## Paràmetres

| Nom | Tipus | Descripció |
|------|------|-------------|
| $url | `string` | L'URL d'actualització. |
| $args | `array` | El mòdul i el tipus de limitació que necessita ser actualitzat. |

### Font

Definit a [`inc/functions/limitations.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/limitations.php#L177) a la línia 177.
