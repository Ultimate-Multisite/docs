# Filtre: wu_pre_save_settings

Permet als desenvolupadors filtrar la configuració abans de desar-la per Ultimate Multisite.

## Paràmetres {#parameters}

| Nom | Tipus | Descripció |
|------|------|-------------|
| $settings | `array` | La configuració que es desarà. |
| $settings_to_save | `array` | La nova configuració que s’afegirà. |
| $saved_settings | `array` | La configuració actual desada. |

### Des de {#since}

- 2.0.18

Definit a [`inc/class-settings.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-settings.php#L296) a la línia 296
