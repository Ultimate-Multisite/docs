Filtra si cal requerir caràcters especials a les contrasenyes.

## Paràmetres {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | Si cal requerir caràcters especials. Per defecte és true quan s'apliquen les regles. |
| $defender_active | `bool` | Si Defender Pro Strong Password està actiu. |
