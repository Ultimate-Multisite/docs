# Filtre: wu_add_new_domain_modal_fields

Permet als addons (com Domain Seller) modificar o substituir els camps de registratió del domini.

## Paràmetres

| Nom | Tip | Descripció |
|------|------|-------------|
| $fields | `array` | Els camps del formulari. |
