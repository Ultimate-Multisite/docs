# Filtre: wu_documentation_get_link {#filter-wudocumentationgetlink}

Permet als desenvolupadors d’extensions filtrar els enllaços.

No és clar com podria ser útil, però no fa cap mal tenir-ho.

## Paràmetres {#parameters}

| Nom | Tipus | Descripció |
|------|------|-------------|
| $link | `string` | L’enllaç registrat |
| $slug | `string` | L’identificador curt utilitzat per recuperar l’enllaç |
| $default_link | `string` | L’enllaç predeterminat registrat |
