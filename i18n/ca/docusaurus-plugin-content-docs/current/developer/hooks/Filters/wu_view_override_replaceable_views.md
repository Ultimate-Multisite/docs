# Filter: wu_view_override_replaceable_views {#filter-wuviewoverridereplaceableviews}

Permet als desenvolupadors afegir carpetes addicionals a la llista de vistes substituibles.

Està guia, ja que permetre carpetes addicionals pot fer que es carreguin còpies desactualitzades en lloc de les versions d'Ultimate Multisite.

## Paràmetres {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $replaceable_views | `array` | Llista de carpetes autoritzades. |

### Des {#since}

- 2.0.0
### Font {#source}

Definit en [`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L50) a la línia 50
