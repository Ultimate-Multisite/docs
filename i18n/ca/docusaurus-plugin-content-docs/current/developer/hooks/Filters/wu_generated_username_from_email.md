# Filtre: wu_generated_username_from_email

Filtra el nom d'usuari generat del client.

## Paràmetres

| Nom | Tipus | Descripció |
|------|------|-------------|
| $username | `string` | Nom d'usuari generat. |
| $email | `string` | Adreça de correu electrònic del client nou. |
| $new_user_args | `array` | Array d'args de l'usuari nou, possiblement incloent-hi noms i cognoms. |
| $suffix | `string` | Cadena per afegir al nom d'usuari per fer-lo únic. |
