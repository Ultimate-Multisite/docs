# Aquesta acció: wu_mcp_adapter_initialized {#action-wumcpadapterinitialized}

Es activa després que s'inicialitza l'adaptador MCP.

Permet als altres plugins i temes registrar les seves pròpies habilitats.

## Paràmetres {#parameters}

| Nom | Tip | Descripció |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | L'instància de l'adaptador MCP. |

### Des de {#since}

- 2.5.0
