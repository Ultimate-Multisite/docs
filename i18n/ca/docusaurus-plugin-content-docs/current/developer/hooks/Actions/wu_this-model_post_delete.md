# Aquesta acció: wu_\{$this->model\}_post_delete

Es activada després que un object es guarda a la base de dades.

## Paràmetres

| Nom | Tip | Descripció |
|------|------|-------------|
| $result | `bool` | True si l'objecte va ser eliminat amb èxit. |
| $this | `\Base_Model` | L'instància de l'objecte. |
