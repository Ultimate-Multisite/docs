# Aquesta acció: wu_magic_link_invalid_token {#action-wumagiclinkinvalidtoken}

Es activa quan es troba un token de magic link no màrítim (inválid).

## Paràmetres {#parameters}

| Nom | Tip | Descripció |
|------|------|-------------|
| $reason | `string` | La raó per la qual el token és invàlid. |

- 2.0.0
