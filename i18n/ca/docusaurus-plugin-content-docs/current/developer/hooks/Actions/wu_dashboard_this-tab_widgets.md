# Aquesta acció: wu_dashboard_\{$this->tab\}_widgets {#action-wudashboardthis-tabwidgets}

Permet als desenvolupadors de plugins afegir widgets al Panel del Dashboard de la Xarxa (Network Dashboard).

## Paràmetres {#parameters}

| Nom | Tip | Descripció |
|------|------|-------------|
| $tab | `string` | La pelta actual. |
| $screen | `\WP_Screen` | L'objecte del pantalla. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Instància de la pàgina d'administració del Dashboard de Ultimate Multisite. |

### Des de {#since}

- 2.0.0
