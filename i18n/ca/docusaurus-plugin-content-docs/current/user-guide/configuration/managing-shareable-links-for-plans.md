# Gestió de enllaços compartibles per plans {#managing-shareable-links-for-plans}

Ultimate Multisite et ofereix la opció de compartir enllaços de plans directament amb els vostres clients. Per fer-ho, aneu a qualsevol pàgina d'edició dels vostres plans i cliqueu el botó _Click to copy the Shareable Link_.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_A la pàgina d'edició del Plan, podeu trobar el botó per crear enllaços compartibles_

Observeu que aquest mecanisme utilitza la freqüència de facturació per defecte per construir l'enllaç. Això significa que, si la vostra facturació per defecte és mensual, per exemple, finalment tindreu un enllaç com aquest: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Si voleu que l'enllaç compartible sigui per defecte trimestral, haureu de triar el número 3 al final.

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Per seleccionar el plan _premium_ amb un càrrec anual, hauríeu d'afegir 1 i després la paraula:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Recordeu que no podeu tenir un només enllaç per a diferents freqüències de facturació. Haureu de compartir un enllaç específic per a cada una de les vostres freqüències de facturació.

Una vegada que compartegeu l'enllaç amb els vostres clients, aquest els redirigirà al primer pas del vostre flux d'alta.
