# Col·locar anuncis a la teva xarxa amb Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector és un **complement gratuït** que et proporciona els mitjans per col·locar anuncis als teus plans, directament des de la seva pàgina d'edició.

Després d'instal·lar Ultimate Multisite: Ad Injector, veu a la pàgina de _Plugins_ per activar-lo.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Fes clic al botó Activar a la Xarxa_

Quan el complement estigui actiu, la caixa meta _Advanced Options_ a la pantalla d'edició del plan tindrà una nova opció anomenada _Ad Settings_.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_Ara pots editar la configuració d'anuncis per a cada un dels teus plans_

En aquesta pestanya, podràs afegir els codis d'anuncis per a articles i pàgines al front-end, abans i després del contingut.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Després d'inserir el codi del teu anunci, es mostrarà als articles i pàgines d'aquest plan específic._

També pots fer que hi hagin anuncis a la capçalera de les Pàgines d'administració per als usuaris d'aquest plan.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Veu a l'opció de Codici d'anunci al Panell d'administració si vols mostrar anuncis al back-end del teu usuari_

### Més enllà de la publicitat

També pots utilitzar Ultimate Multisite: Ad Injector per oferir suport als plans de nivell superior.

Només has de veure a l'opció _Admin Panel Ads_ i afegir el codi HTML per al teu eina de suport.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Afegir suport al client a plans específics_

Dequesta manera, la gent pot arribar-te directament des del seu Panell d'administració!
