# Personalitzant el teu Admin Panel amb PRO Theme {#customizing-your-admin-panel-with-pro-theme}

Sovíament, quan fem treball per a clients, ens apareix la necessitat de proporcionar-hi una experiència personalitzada al *dashboard* d'administració de WordPress. Si estàs buscant un aspecte i una usabilitat més professionals, PRO Theme et pot ajudar.

PRO Theme és un tema d'administració desenvolupat per ajudar-te quan personalitzes el *Dashboard* dels teus clients. I també és part del *core* de *plugins* premium d'Ultimate Multisite.

Després de l'instal·lar, veu a la pàgina de *Plugins* i cliqueu per activar-lo a la vostra *Network*.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Activa-ho a nivell de *Network* a la teva pàgina de *Plugins*_

_**NOTA:**_ _Si només l'instal·les, apareixerà als teus clients com una opció que es pot marcar per afegir el tema només a aquest usuari._

Ja, has de la interfície d'administració completament rediseñada! També tens un menú nou a la columna de l'esquerra anomenat PRO Theme.
