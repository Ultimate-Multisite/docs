# Hooks i filters d'Ultimate Multisite {#ultimate-multisite-hooks-and-filters}

Ultimate Multisite està compulat de moltes funcions i truyestos de base de dades. Té hooks i filters que et permetran modificar o afegir codi nou a funcions ja existents.

Hem creat una llista d'aquests hooks i filters com a referència per ajudar els usuaris avançats a personalitzar encara més la seva network. Pots accedir a la llista des de Github fent clic al enllaç de més avall.
