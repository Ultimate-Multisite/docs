# Com obtenir ajuda de desenvolupament personalitzada de Codeable?

A vegades, realment necessitem una personalització fantàstica per fer que la nostra xarxa flueixi exactament com volem. Si és el que busqueu, recomanem [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), un lloc on podeu trobar desenvolupadors amb experiència en Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) és una plataforma que connecta clients amb desenvolupadors *freelance* qualificats de WordPress, capaços de treballar en projectes de qualsevol mida. Tenen un procés de sol·licitat meticulós per assegurar que es col·liqueu amb experts específicament qualificats per a la tasca que teniu en ment.

### Com crear un projecte a Codeable? {#how-to-create-a-project-in-codeable}

Primer, aneu a [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) i cliqueu al botó **Start a Project**.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Ara, per enviar un projecte, cliqueu a la esquerra sobre **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

En "I need help with", trieu **Customizing** :

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

En "My", trieu **Plugin** :

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Després, cliqueu al botó **Continue**.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Ara, és hora de descriar el vostre projecte i necessitats de la millor manera possible. Intenteu donar la veritable informació possible!

<!-- Screenshot unavailable: Codeable project description form -->

Responeu a les preguntes següents per ajudar amb el preu.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Després de això, si encara no teniu compte de [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), creeu-ne un i cliqueu al botó **Publish My Project**.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

Una vegada que hagiu publicat el vostre projecte de [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), la plataforma us connectarà amb desenvolupadors que compleixin amb el nivell de competències requerit per al vostre projecte.

Aquest procés ajudarà a trobar la persona adequada per al vostre treball, que té experiència amb WordPress i Ultimate Multisite!

La taxa mínima de Codeable és de 70–120 USD per hora. Les estimacions canviaran en funció de l'àmbit de treball, la complexitat i l'urgència d'un projecte.

Quan tingueu l'estimació, podreu contractar el desenvolupador que millor s'adapti a les vostres necessitats i començar el vostre projecte junts!
