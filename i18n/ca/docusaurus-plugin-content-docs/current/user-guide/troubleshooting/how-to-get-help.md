# Com a ajuda

Teniu la teva clau de llicència i adjunta a ella l'accés al nostre equip per ajudar-te a solucionar qualsevol problema que puguis trobar durant el configuració d'Ultimate Multisite.

Aquesta lecció cobreix els canals de suport disponibles per als titulars de llicències i la informació que necessitarà tenir a mà per contactar el nostre equip de suport.

## Lloc Web d'Ajuda (Help Site) {#help-site}

Pots accedir a diferents articles i tutorial de la nostra lloc web d'ajuda, que trobes al link a continuació.

[)

Encorem els nostres usuaris a familiaritzar-se amb les funcions bàsiques d'Ultimate Multisite. Hi tens articles "Com començar" disponibles en la nostra lloc web d'ajuda i respostes a preguntes comunes, com per exemple com instal·lar add-ons o com integrar pasarel·les de pagament.

## Contacte de Suport (Contact Support) {#contact-support}

Si et quedes bloquejat i no trobes la resposta en la nostra lloc web d'ajuda, pots contactar el nostre equip de suport fent clic a l'icon de bolla de paraula situat a la part inferior dreta del nostre web O enviant un correu electrònic directament a support@ultimatemultisite.com.

Quan contactis el suport per problemes tècnics, assegura't de proporcionar una descripció detallada del teu problema juntament amb una captura d' pantalla, missatges d'error o qualsevol informació que sigui útil per al nostre personal de suport per entendre i evaluar fàcilment el teu problema. També cal proporcionar la teva clau de llicència. Només els titulars de llicències actives tenen dret a suport.

## Lloc Web Comunitari (Community Site) {#community-site}

El nostre [site comunitari](https://community.ultimatemultisite.com/) és un lloc per als usuaris d'Ultimate Multisite per interactuar entre ells. Serveix com a un sistema d'informació on els membres poden publicar, comentar discussions, donar consells, col·laborar o recollir contribuents (crowdsource). Per desbloquejar espais/pàgines privades, se us demanarà que proporcionis la teva clau de llicència d'Ultimate Multisite.
