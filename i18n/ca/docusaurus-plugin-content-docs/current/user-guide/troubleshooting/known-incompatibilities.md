# Incompatibilitats con altres Plugins Coneguts

Després d'instal·lar i activar Ultimate Multisite, pots trobar problemes per incompatibilitats no tractades amb altres plugins.

Estem treballant activament per reduir aquesta llista amb cada nova publicació, però si un plugin està aquí, podràs necessitar desactivar-lo per tenir una Xarxa Ultimate Multisite completment funcional.

Aquí teniu una llista d'aquests i què has de fer en aquests casos:

**Alguns plugins WPMU DEV** Alguns dels plugins que no són mantenits per WPMU DEV estan coneguts per causar que el flux d'enregistrament d'Ultimate Multisite s'detengui a l'últim pas (l'usuari no pot crear una compte). Aquí teniu una llista dels plugins que causen aquests problemes:

  * Signup Language: _Ultimate Multisite té un add-on que ofereix la mateixa funcionalitat, anomenat Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Aquest està integrat en Ultimate Multisite, així que no cal cap plugin extra._

  * New Blog Templates: _Aquest està integrat en Ultimate Multisite, així que no cal cap plugin extra._

  * Set Password on Multisite Blog Creation: _Aquest està integrat en Ultimate Multisite, així que no cal cap plugin extra._

  * Signup TOS: _Aquest està integrat en Ultimate Multisite, així que no cal cap plugin extra._

  * Anti-Splog: Tot i ser menys potempent, també està _integrat en Ultimate Multisite._

**Nota**: La majoria d'plugins de dev de WPMU poden utilitzar-se normalment juntament amb Ultimate Multisite. Exemplos inclou _Smush_, _Forminator_, _Defender_, etc.
