# Traducció de Ultimate Multisite

Ultimate Multisite està desenvolupat en anglès, però el plugin és completament traduïble i valorem molt les contribucions de traducció.

## Traduccions del Plugin Central {#core-plugin-translations}

El plugin central de Ultimate Multisite es tradueix a través de la plataforma oficial de traducció de **WordPress.org**:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

Aquesta és la manera estàndard en què funciona el sistema de traducció de la comunitat de WordPress. Quan les traduccions arriben a un número suficient d'aprovacions, es distribuïn automàticament a tots els usuaris d'Ultimate Multisite mitjanques les actualitzacions del plugin de WordPress.

**¡Necessitem la vostra ajuda!** Si parles una llengua que no sigui l'anglès, considera contribuir amb traduccions. Fins i tot traduir unes poques frases ajuda molt. Per començar:

1. Visita [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)
2. Selecciona la teva llengua
3. Clica en "Contribuir traducció" (Contribute Translation)
4. Necessitaràs una compte gratuïta de WordPress.org

Si vols convertir-te en editora de traduccions per a la teva llengua (capaz d'aprovar les traduccions), contacta-nos a través de [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions).

## Traduccions dels Addons {#addon-translations}

Els plugins add-on es tradueixen separat a:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Les traductions des addons sont actuellement générées par IA comme point de départ. Elles couvrent toutes les langues prises en charge, mais peuvent contenir des formulations maladroites ou des inexactitudes. Nous accueillons avec plaisir les traducteurs humains pour les examiner et suggérer des améliorations afin de rendre le texte plus naturel et précis.

## Traductions locales {#local-translations}

Si vous préférez créer vos propres traductions localement pour votre projet plutôt que de contribuer en amont, voici deux options :

### Utiliser Poedit {#using-poedit}

Vous pouvez utiliser l'application de bureau **[Poedit](https://poedit.net)** pour traduire le fichier `.pot` :

1. Téléchargez Poedit depuis [poedit.net](https://poedit.net)
2. Ouvrez le fichier `.pot` situé dans `wp-content/plugins/ultimate-multisite/lang/`
3. Traduisez les chaînes dans l'interface de Poedit
4. Enregistrez -- Poedit génère les fichiers `.po` et `.mo`
5. Placez les fichiers de sortie dans le dossier `lang`

### Utiliser Loco Translate {#using-loco-translate}

Le plugin WordPress **[Loco Translate](https://wordpress.org/plugins/loco-translate/)** vous permet de traduire directement depuis le panneau d'administration sans avoir besoin d'outils de bureau. Il s'occupe de créer et compiler les fichiers de traduction pour vous.
