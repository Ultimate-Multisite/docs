# Els llocs web del teu client (v2)

Després de registrar-se a la teva xarxa, els teus clients poden accedir al seu *dashboard*.

A més de la pàgina **Account**, on poden veure informació important sobre els seus pagaments, membresdies, domínis, limitacions del pla i més, també tenen accés a la pàgina **My Sites**, on poden veure i gestionar tots els seus llocs web. Només han de fer clic a My Sites al seu *dashboard*.

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

Els mostrarà els seus llocs web, amb un botó **Manage** just per sota cadascun. Si hi fem clic, seran rediriguts al **dashboard d'aquest lloc web específic**.

També poden veure quin dels seus llocs web és el **principal** per als plans que els permeten tenir més d'un lloc web.

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

Aquí teniu una vista completa de la interfície d'edició del lloc web:

![Site edit full interface](/img/admin/site-edit-full.png)

Si volen canviar el seu lloc web principal, han de tornar a la pàgina **Account** fent clic al menú **Account**.

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

També podeu veure i editar els detalls del client:

![Customer edit interface](/img/admin/customer-edit.png)

Després, han de fer clic a **Change Default Site**.

![Change Default Site button on account page](/img/account-page/change-default-site.png)

Apareixerà una finestra pop-up. Fageu clic a la llista desplegable i seleccioneu quin lloc web voleu fer el principal.

![Change Default Site dropdown selection](/img/account-page/change-default-site-dropdown.png)

Després, només feu clic a **Change Default Site** i ja és tot!

![Confirming the default site change](/img/account-page/change-default-site-confirm.png)
