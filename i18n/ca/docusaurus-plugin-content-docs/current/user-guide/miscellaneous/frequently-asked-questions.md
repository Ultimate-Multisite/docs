# Preguntes Frecquents {#frequently-asked-questions}

Aquí teniu algunes **Preguntes Frecquents** que sovint surten durant les sol·licitacions de suport.

  * **Què és Ultimate Multisite?**

Ultimate Multisite és un plugin per a WordPress, fa per instal·lacions multisite de WordPress, que transforma la vostra instal·lació de WordPress en una xarxa premium de webs – com [WordPress.com](https://WordPress.com) –, permetent als clients crear webs mitjançant tasques mensuals, trimestrals o anuals (també podreu crear plans Gratuits).

  * **S'anirà bé Ultimate Multisite sense Multisite?**

No. Ultimate Multisite requereix WordPress multisite, subdirectori o subdomain.

  * **Quins són els requisits per a Ultimate Multisite?**

Ultimate Multisite requereix:

**Versió de WordPress** : v5.3+ (Recomanda-se: Última Versió Estable)
**Versió de PHP** : 7.4.x (Soporte per a > 8.0 aprovat imminent)
**Versió de MySQL** : v5+ (Recomanda-se: Versió 5.6, en cas de no tenir disponible la 8.0 al vostre proveïdor d'hosting)

També cal activar el _Multisite (subdomini o subdirectori)_. [Com instal·lar WordPress Multisite](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **S'anirà bé Ultimate Multisite amb Multisite amb Subdirectoris?**

Sí. Ultimate Multisite suporta tant les instal·lacions de subdirectori com les instal·lacions de subdirectori.

  * **S'anirà bé Ultimate Multisite amb Multisite amb Subdomènis?**

Sí. Ultimate Multisite suporta tant les instal·lacions de subdirectori com les instal·lacions de subdomain.

* **Puc puc utilitzar Ultimate Multisite per limitar quins Plugins i Temes poden utilitzar els usuaris?**

Sí. Amb Ultimate Multisite pots seleccionar quins plugins i temes estan disponibles per a cada pla de la teva xarxa.

  * **Puc puc utilitzar Ultimate Multisite per limitar el nombre de posts que un usuari pot crear al seu site?**

Sí. Amb Ultimate Multisite pots estableix quotas de posts per a **cada tipus de post disponible** en la teva xarxa. També pots limitar el nombre d'usuaris que l'administrador pot invitar, la **quantitat d'espai de disco**, i moltes altres coses.

  * **Ultimate Multisite suporta les Plantilles de Site (Site Templates)?**

Sí. Amb Ultimate Multisite pots crear tant Plantilles de Site com vulguis. Les Plantilles de Site no són temes, són sites normals a la teva xarxa. Pots activar plugins, temes i personalitzar aquesta Plantilla de Site com vulguis. Si un usuari selecciona aquesta plantilla, el seu site es crearà com una còpia d'aquesta plantilla, amb totes les personalitzacions ja aplicades.

També pots assignar una plantilla determinada a un pla. Així, cada usuari que s'registri en aquest pla tindrà el seu site creat com una duplicat d'aquesta Plantilla de Site.

  * **Ultimate Multisite es integra amb WooCommerce?**

¡Sí!

Si vols que els usuaris de la teva xarxa tinguin WooCommerce disponible per a ells, permetent-los configurar botigues, això és possible utilitzant Ultimate Multisite. Fins i tot pots limitar el nombre de productes que els teus clients podran crear en cada nivell de pla.

En termes de paiements par abonnement pour Ultimate Multisite, pots d'en jeu l'add-on [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) per permetre que vos clients Ultimate Multisite paient en utilisant les passerelles WooCommerce sur votre site principal. De cette façon, vous pouvez accéder à la vaste bibliothèque de passerelles de paiement développées pour WooCommerce et les utiliser pour facturer vos clients Ultimate Multisite.

* **Y a-t-il un moyen de migrer les abonnements depuis WPMUDEV Pro Sites ?**

Oui, Ultimate Multisite vous propose maintenant [un outil pratique pour effectuer la migration](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) de votre réseau de Pro Sites.

* **Pourquoi l'onglet « Plugins autorisés » est-il vide sur ma page d'édition de plan ?**

Ultimate Multisite ne vous permet d'autoriser ou de refuser l'accès qu'aux plugins qui ne sont pas activés sur votre réseau. Lorsqu'un plugin est activé sur le réseau, il sera chargé par défaut sur chaque site de votre réseau, par nature.

Si vous ne voyez aucun plugin listé dans l'onglet « Plugins autorisés », vérifiez si le plugin souhaité est actif sur le réseau ou non.

* **Puis-je utiliser mon propre design de tableau de prix ?**

Oui. Depuis la version 1.4.0, Ultimate Multisite prend en charge les remplacements de modèles (template overrides).

* **Ultimate Multisite fonctionne-t-il avec WP Engine ?**

Sí. Ultimate Multisite siempre ha sido compatible con WP Engine, pero para que el mapeo de domini funcionara necesitabas pasos manuales adicionales por parte del administrador (añadir el dominio al panel de WP Engine). **Esto ya no es necesario desde la versión 1.6.0.** Ultimate Multisite se integra directamente con l'API de WP Engine y los domini se añaden automáticamente al panel después del mapeo.

* **¿Qué proveedor d'hébergement recomanes?**

Ultimate Multisite debería funcionar con cualquier proveedor d'hébergement. La única diferencia es que algunas empresas de hosting como Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane y [Runcloud.io](http://Runcloud.io) tienen una integración más ajustada para las capacidades de mapeo de domini, por lo que funcionan "out of the box". Para otros hosts, es posible que necesites configurar el mapeo de domini manualmente.
