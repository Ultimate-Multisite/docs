# ଫିଲ୍ଟର୍: wu_get_checkout_variables

plugin ବିକାଶକମାନଙ୍କୁ checkout ପୃଷ୍ଠାର ପୂର୍ବ-ନିର୍ଦ୍ଧାରିତ ସେଟ୍‌ଗୁଡ଼ିକୁ ଫିଲ୍ଟର୍ କରିବାକୁ ଦିଏ।

ସାବଧାନ ରୁହନ୍ତୁ, ଅନୁପସ୍ଥିତ କୀଗୁଡ଼ିକ front-end ରେ checkout କୁ ସମ୍ପୂର୍ଣ୍ଣ ଭାଙ୍ଗିଦେଇପାରେ।

## ପାରାମିଟର୍‌ଗୁଡ଼ିକ

| ନାମ | ପ୍ରକାର | ବର୍ଣ୍ଣନା |
|------|------|-------------|
| $variables | `array` | ସ୍ଥାନୀୟକୃତ ଭେରିଏବଲ୍‌ଗୁଡ଼ିକ। |
| $checkout | `\Checkout` | checkout ଶ୍ରେଣୀ। |

- 2.0.0

## ଫେରାଇଦିଏ
ନୂଆ ଭେରିଏବଲ୍‌ଗୁଡ଼ିକର array।
