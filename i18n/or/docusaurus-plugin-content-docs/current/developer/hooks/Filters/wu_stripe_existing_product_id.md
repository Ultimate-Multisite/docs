# ଫିଲ୍ଟର: wu_stripe_existing_product_id {#filter-wustripeexistingproductid}

ଯାଞ୍ଚ କରିବାକୁ ଥିବା ଉତ୍ପାଦର ପରିଚୟକକୁ ଫିଲ୍ଟର କରେ। ଏହା ଥିଲେ, ନୂଆ ସଦସ୍ୟତା ଏହି ଉତ୍ପାଦକୁ ବ୍ୟବହାର କରିବ।

## ପାରାମିଟର {#parameters}

| ନାମ | ପ୍ରକାର | ବର୍ଣ୍ଣନା |
|------|------|-------------|
| $product_id | `string` | ଯାଞ୍ଚ କରିବାକୁ ଥିବା Stripe ଉତ୍ପାଦର ପରିଚୟକ। |
| $name | `string` | ଉତ୍ପାଦର ନାମ। |
