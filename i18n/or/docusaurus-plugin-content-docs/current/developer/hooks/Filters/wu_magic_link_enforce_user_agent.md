# ଫିଲ୍ଟର: wu_magic_link_enforce_user_agent

ୟୁଜର ଏଜେଣ୍ଟ ଯାଞ୍ଚ ବାଧ୍ୟତାମୂଳକ କରାଯିବ କି ନାହିଁ, ତାହା ଫିଲ୍ଟର କରନ୍ତୁ।

ଟୋକେନଗୁଡ଼ିକୁ ଭିନ୍ନ ବ୍ରାଉଜର/ଡିଭାଇସରେ କାମ କରିବାକୁ ଦେବା ପାଇଁ false କରନ୍ତୁ। ଏହା ସୁରକ୍ଷା କମାଏ, କିନ୍ତୁ ବ୍ୟବହାର ସହଜ କରେ।

## ପାରାମିଟରଗୁଡ଼ିକ {#parameters}

| ନାମ | ପ୍ରକାର | ବର୍ଣ୍ଣନା |
|------|------|-------------|
| $enforce | `bool` | ୟୁଜର ଏଜେଣ୍ଟ ମେଳକ ବାଧ୍ୟତାମୂଳକ କରାଯିବ କି ନାହିଁ। |
