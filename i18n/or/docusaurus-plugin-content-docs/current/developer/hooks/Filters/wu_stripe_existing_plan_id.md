# ଫିଲ୍ଟର: wu_stripe_existing_plan_id

ଯାଞ୍ଚ କରିବାକୁ ଥିବା ଯୋଜନାର ID କୁ ଫିଲ୍ଟର କରେ। ଯଦି ଏହା ଥାଏ, ନୂଆ ସଦସ୍ୟତା ଏହି ଯୋଜନା ବ୍ୟବହାର କରିବ।

|------|------|-------------|

| $membership_level | `object` | ସଦସ୍ୟତା ସ୍ତର object। |
