# ଫିଲ୍ଟର: wu_independent_billing_cycle_product_types

ସ୍ୱାଧୀନ ବିଲିଂ ଚକ୍ର ଥିବା ଉତ୍ପାଦ ପ୍ରକାରଗୁଡ଼ିକୁ ଫିଲ୍ଟର କରନ୍ତୁ।

ସ୍ୱାଧୀନ ବିଲିଂ ଚକ୍ର ଥିବା ଉତ୍ପାଦଗୁଡ଼ିକୁ କାର୍ଟରେ ଥିବା ପ୍ଲାନର ବିଲିଂ ଅବଧି ସହ ମେଳ କରିବାକୁ ବାଧ୍ୟ କରାଯାଏ ନାହିଁ। ଏହା ଡୋମେନ ପଞ୍ଜୀକରଣ ପରି ଉତ୍ପାଦ ପାଇଁ ଉପଯୋଗୀ, ଯେଉଁଗୁଡ଼ିକ ପ୍ଲାନ ମାସିକ କିମ୍ବା ବାର୍ଷିକ ଯାହା ହେଉ ନାହିଁ, ସବୁବେଳେ ବାର୍ଷିକ ବିଲ୍ କରାଯାଏ।
