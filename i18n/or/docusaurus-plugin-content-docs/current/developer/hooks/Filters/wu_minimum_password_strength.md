# ଫିଲ୍ଟର: wu_minimum_password_strength {#filter-wuminimumpasswordstrength}

ଆବଶ୍ୟକ ନ୍ୟୁନତମ ପାସୱାର୍ଡ ଶକ୍ତିକୁ ଫିଲ୍ଟର କରନ୍ତୁ (zxcvbn ସ୍କୋର)।

ଶକ୍ତି ସ୍ତର: - 0, 1: ବହୁତ ଦୁର୍ବଳ - 2: ଦୁର୍ବଳ - 3: ମଧ୍ୟମ - 4: ଶକ୍ତିଶାଳୀ (ଡିଫଲ୍ଟ)

## ପାରାମିଟର {#parameters}

| ନାମ | ପ୍ରକାର | ବର୍ଣ୍ଣନା |
|------|------|-------------|
| $min_strength | `int` | ଆବଶ୍ୟକ ନ୍ୟୁନତମ ଶକ୍ତି ସ୍ତର। |
| $strength_setting | `string` | admin ସେଟିଂ ମୂଲ୍ୟ (medium, strong, super_strong)। |
