# ଫିଲ୍ଟର: wu_pre_save_settings {#filter-wupresavesettings}

Ultimate Multisite ଦ୍ୱାରା ସେଭ୍ କରିବା ପୂର୍ବରୁ ସେଟିଂସ୍‌କୁ ଫିଲ୍ଟର କରିବାକୁ ବିକାଶକାରୀଙ୍କୁ ଅନୁମତି ଦିଅନ୍ତୁ।

## ପାରାମିଟରଗୁଡ଼ିକ {#parameters}

| ନାମ | ପ୍ରକାର | ବର୍ଣ୍ଣନା |
|------|------|-------------|
| $settings | `array` | ସେଭ୍ କରାଯିବାକୁ ଥିବା ସେଟିଂସ୍। |
| $settings_to_save | `array` | ଯୋଡ଼ିବାକୁ ଥିବା ନୂଆ ସେଟିଂସ୍। |
| $saved_settings | `array` | ବର୍ତ୍ତମାନ ସେଭ୍ ହୋଇଥିବା ସେଟିଂସ୍। |

### ଯେହେତୁ {#since}

- 2.0.18
