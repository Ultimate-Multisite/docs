# ଫିଲ୍ଟର: wu_use_domain_mapping

କୌଣସି ମ୍ୟାପିଂ ବ୍ୟବହାର ହେବା ଉଚିତ କି ନୁହେଁ ନିର୍ଣ୍ଣୟ କରନ୍ତୁ

ସାଧାରଣତଃ, ଆପଣ କେବଳ ସକ୍ରିୟ ମ୍ୟାପିଂକୁ ବ୍ୟବହାର କରିବାକୁ ଅନୁମତି ଦେବାକୁ ଚାହିବେ। ତେବେ, ଯଦି ଆପଣ ଅଧିକ ଉନ୍ନତ ଲଜିକ୍ ବ୍ୟବହାର କରିବାକୁ ଚାହାନ୍ତି, କିମ୍ବା ନିଷ୍କ୍ରିୟ ଡୋମେନଗୁଡ଼ିକୁ ମଧ୍ୟ ମ୍ୟାପ୍ କରିବାକୁ ଦେବାକୁ ଚାହାନ୍ତି, ତେବେ ଏଠାରେ ଫିଲ୍ଟର କରନ୍ତୁ।
