# ଫିଲ୍ଟର: wu_hosting_integration_wizard_sections

ହୋଷ୍ଟିଂ ଇଣ୍ଟିଗ୍ରେସନ୍ ସେଟଅପ୍ ପାଇଁ ୱିଜାର୍ଡ ସେକ୍ସନ୍‌ଗୁଡ଼ିକୁ ଫିଲ୍ଟର କରେ।

ଆଡଅନ୍‌ଗୁଡ଼ିକୁ ୱିଜାର୍ଡ ସେକ୍ସନ୍ ଯୋଡ଼ିବା, କାଢ଼ିବା କିମ୍ବା ବଦଳାଇବାକୁ ଅନୁମତି ଦିଏ।

## ପାରାମିଟର {#parameters}

| ନାମ | ପ୍ରକାର | ବିବରଣୀ |
|------|------|-------------|
| $sections | `array` | ୱିଜାର୍ଡ ସେକ୍ସନ୍‌ଗୁଡ଼ିକ। |
| $integration | `\WP_Ultimo\Integrations\Host_Providers\Base_Host_Provider\|\WP_Ultimo\Integrations\Integration` | କନଫିଗର୍ କରାଯାଉଥିବା ଇଣ୍ଟିଗ୍ରେସନ୍। |
| $page | `\Hosting_Integration_Wizard_Admin_Page` | ୱିଜାର୍ଡ ପୃଷ୍ଠା ଇନ୍‌ଷ୍ଟାନ୍ସ। |
