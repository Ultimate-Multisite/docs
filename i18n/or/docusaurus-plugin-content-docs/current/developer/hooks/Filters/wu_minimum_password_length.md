# ଫିଲ୍ଟର୍: wu_minimum_password_length

ନ୍ୟୁନତମ ପାସୱାର୍ଡ ଲମ୍ବକୁ ଫିଲ୍ଟର୍ କରନ୍ତୁ।

କେବଳ wu_enforce_password_rules true ଥିଲେ ଲାଗୁ ହୁଏ।

## ପାରାମିଟର୍‌ଗୁଡ଼ିକ

| ନାମ | ପ୍ରକାର | ବର୍ଣ୍ଣନା |
|------|------|-------------|
| $min_length | `int` | ନ୍ୟୁନତମ ପାସୱାର୍ଡ ଲମ୍ବ। ପୂର୍ବନିର୍ଦ୍ଧାରିତ ୧୨ (Defender Pro ସହ ମେଳ ଖାଏ)। |
| $defender_active | `bool` | Defender Pro Strong Password ସକ୍ରିୟ ଅଛି କି ନାହିଁ। |
