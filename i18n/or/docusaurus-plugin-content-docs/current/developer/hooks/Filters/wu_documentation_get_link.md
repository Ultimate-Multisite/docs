# ଫିଲ୍ଟର: wu_documentation_get_link

plugin ବିକାଶକାରୀମାନଙ୍କୁ ଲିଙ୍କଗୁଡ଼ିକୁ ଫିଲ୍ଟର କରିବାକୁ ଦିଅନ୍ତୁ।

ଏହା କିପରି ଉପଯୋଗୀ ହୋଇପାରେ ନିଶ୍ଚିତ ନୁହେଁ, କିନ୍ତୁ ଏହା ରହିଲେ କ୍ଷତି ନାହିଁ

## ପାରାମିଟରଗୁଡ଼ିକ {#parameters}

| ନାମ | ପ୍ରକାର | ବର୍ଣ୍ଣନା |
|------|------|-------------|
| $link | `string` | ନିବନ୍ଧିତ ଲିଙ୍କ |
| $slug | `string` | ଲିଙ୍କ ଆଣିବା ପାଇଁ ବ୍ୟବହୃତ slug |
| $default_link | `string` | ନିବନ୍ଧିତ ଡିଫଲ୍ଟ ଲିଙ୍କ |
