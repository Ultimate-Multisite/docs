# ଫିଲ୍ଟର: wu_country_get_cities {#filter-wucountrygetcities}

ଏକ ଦେଶର ଏକ ରାଜ୍ୟ ପାଇଁ ସହରଗୁଡ଼ିକର ତାଲିକା ଫେରାଇଦିଏ।

## ପାରାମିଟରଗୁଡ଼ିକ {#parameters}

| ନାମ | ପ୍ରକାର | ବର୍ଣ୍ଣନା |
|------|------|-------------|
| $cities | `array` | ରାଜ୍ୟର ସହର ନାମଗୁଡ଼ିକର ତାଲିକା। କୌଣସି କୀ ନାହିଁ। |
| $country_code | `string` | ଦେଶ ପାଇଁ ଦୁଇ-ଅକ୍ଷର ISO କୋଡ୍। |
| $state_code | `string` | ରାଜ୍ୟ ପାଇଁ ଦୁଇ-ଅକ୍ଷର ISO କୋଡ୍। |
| $current_country | `\WP_Ultimo\Country\Country` | ବର୍ତ୍ତମାନ class ର ଇନ୍‌ଷ୍ଟାନ୍ସ। |

### ଏହି ସଂସ୍କରଣରୁ {#since}

- 2.0.11
### ସ୍ରୋତ {#source}

[`inc/country/class-country.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/country/class-country.php#L146) ରେ 146 ନମ୍ବର ଧାଡ଼ିରେ ପରିଭାଷିତ


## ଫେରାଇଦିଏ {#returns}
ରାଜ୍ୟଗୁଡ଼ିକର ଫିଲ୍ଟର୍ କରାଯାଇଥିବା ତାଲିକା।
