# ଫିଲ୍ଟର: wu_generated_username_from_email {#filter-wugeneratedusernamefromemail}

ସୃଷ୍ଟି ହୋଇଥିବା ଗ୍ରାହକ ବ୍ୟବହାରକାରୀନାମ ଫିଲ୍ଟର କରନ୍ତୁ।

## ପାରାମିଟର {#parameters}

| ନାମ | ପ୍ରକାର | ବର୍ଣ୍ଣନା |
|------|------|-------------|
| $username | `string` | ସୃଷ୍ଟି ହୋଇଥିବା ବ୍ୟବହାରକାରୀନାମ। |
| $email | `string` | ନୂଆ ଗ୍ରାହକ ଇମେଲ୍ ଠିକଣା। |
| $new_user_args | `array` | ନୂଆ ବ୍ୟବହାରକାରୀ ଆର୍ଗ୍ୟୁମେଣ୍ଟର ଆରେ, ପ୍ରଥମ ଓ ଶେଷ ନାମ ଥାଇପାରେ। |
| $suffix | `string` | ଏହାକୁ ଅନନ୍ୟ କରିବାକୁ ବ୍ୟବହାରକାରୀନାମରେ ଅକ୍ଷରଶୃଙ୍ଖଳା ଯୋଡ଼ନ୍ତୁ। |
