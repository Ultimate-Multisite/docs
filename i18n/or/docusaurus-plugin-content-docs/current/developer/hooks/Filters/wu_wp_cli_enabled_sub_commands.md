# ଫିଲ୍ଟର: wu_wp_cli_enabled_sub_commands {#filter-wuwpclienabledsubcommands}

ଏହି ଏଣ୍ଟିଟି ପାଇଁ କେଉଁ sub_commands ସକ୍ରିୟ ହେବ, ତାହା ଫିଲ୍ଟର କରେ।

## ପାରାମିଟର {#parameters}

| ନାମ | ପ୍ରକାର | ବର୍ଣ୍ଣନା |
|------|------|-------------|
| $sub_commands | `array` | ଡିଫଲ୍ଟ sub_commands। |
| $command_base | `string` | root ପରେ command ରେ ବ୍ୟବହୃତ ଆଧାର। |
| $this | `\Base_Manager` | ଅବଜେକ୍ଟ ଇନ୍‌ସ୍ଟାନ୍ସ। |
