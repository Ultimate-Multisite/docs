# ଫିଲ୍ଟର୍: wu_magic_link_enforce_ip

IP ଠିକଣା ଯାଞ୍ଚ ବାଧ୍ୟତାମୂଳକ କରିବେ କି ନାହିଁ, ତାହା ଫିଲ୍ଟର୍ କରନ୍ତୁ।

ଭିନ୍ନ ନେଟୱର୍କରୁ ଟୋକେନ୍‌ଗୁଡ଼ିକୁ କାମ କରିବାକୁ ଦେବା ପାଇଁ false କୁ ସେଟ୍ କରନ୍ତୁ। ଏହା ସୁରକ୍ଷା କମାଏ, କିନ୍ତୁ ବ୍ୟବହାର ସହଜ କରେ (ଯେପରିକି, ନେଟୱର୍କ ବଦଳାଉଥିବା ମୋବାଇଲ୍ ବ୍ୟବହାରକାରୀଙ୍କ ପାଇଁ)।

## ପାରାମିଟର୍‌ଗୁଡ଼ିକ {#parameters}

| ନାମ | ପ୍ରକାର | ବର୍ଣ୍ଣନା |
|------|------|-------------|
| $enforce | `bool` | IP ଠିକଣା ମେଳକୁ ବାଧ୍ୟତାମୂଳକ କରିବେ କି ନାହିଁ। |
