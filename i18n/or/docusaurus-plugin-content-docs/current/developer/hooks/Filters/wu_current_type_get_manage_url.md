# Filter: wu_current_\{$type\}_get_manage_url {#filter-wucurrenttypegetmanageurl}

ବିକାଶକାରୀମାନଙ୍କୁ ସାଇଟ ପରିଚାଳନା URL ପାରାମିଟରଗୁଡ଼ିକ ସଂଶୋଧନ କରିବାକୁ ଦିଅନ୍ତୁ।

## ପାରାମିଟର {#parameters}

| ନାମ | ପ୍ରକାର | ବର୍ଣ୍ଣନା |
|------|------|-------------|
| $manage_site_url | `string` | ସାଇଟ ପରିଚାଳନା URL। |
| $id | `int` | ସାଇଟ ID। |
| $site_hash | `string` | ସାଇଟ ହ୍ୟାଶ। |

### ଯେହେତୁ {#since}

- 2.0.9

## ଫେରାଇଥାଏ {#returns}
ସଂଶୋଧିତ ପରିଚାଳନା URL।
