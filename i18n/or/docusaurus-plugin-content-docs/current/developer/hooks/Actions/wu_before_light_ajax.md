# କାର୍ଯ୍ୟ: wu_before_light_ajax

କେତେକ କ୍ଷେତ୍ରରେ, କାର୍ଯ୍ୟଗୁଡ଼ିକ ସମ୍ଭାଳିବା ପାଇଁ ଆମକୁ ଅତିରିକ୍ତ ଶକ୍ତି ଲୋଡ୍ କରିବାକୁ ପଡ଼ିବ।

ଆବଶ୍ୟକ ହେଲେ ଅତିରିକ୍ତ ନିର୍ଭରତାଗୁଡ଼ିକ ଲୋଡ୍ କରିବା ପାଇଁ ଏହି କାର୍ଯ୍ୟ ବ୍ୟବହାର କରାଯାଇପାରେ।
