# Action: wu_page_\{$this->id\}_register_widgets {#action-wupagethis-idregisterwidgets}

ଏହି ପୃଷ୍ଠା ପାଇଁ widgets ପଞ୍ଜୀକୃତ ହେବା ପରେ ଚାଲୁ ହୁଏ।

The dynamic portion of the hook name, `$this-&gt;id`, refers to the page id.

## ପାରାମିଟର {#parameters}

| ନାମ | ପ୍ରକାର | ବର୍ଣ୍ଣନା |
|------|------|-------------|
| $id | `string` | ପୃଷ୍ଠା id। |
| $page_hook | `string` | ପୃଷ୍ଠା hook। |
| $page | `object` | ପୃଷ୍ଠା object। |
