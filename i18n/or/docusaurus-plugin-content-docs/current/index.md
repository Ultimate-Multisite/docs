# Ultimate Multisite ଡକ୍ୟୁମେଣ୍ଟେସନ

Ultimate Multisite ଡକ୍ୟୁମେଣ୍ଟେସନକୁ ସ୍ୱାଗତ। Ultimate Multisite, WordPress Multisite କୁ ଏକ ପୂର୍ଣ୍ଣ ସେବା ଭାବରେ ୱେବସାଇଟ୍ (WaaS) ମଞ୍ଚରେ ପରିଣତ କରେ।

## ଡକ୍ୟୁମେଣ୍ଟେସନ ବିଭାଗଗୁଡ଼ିକ {#documentation-sections}

### [ବ୍ୟବହାରକାରୀ ମାର୍ଗଦର୍ଶିକା](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

ଆପଣଙ୍କ WaaS ମଞ୍ଚ ସେଟ୍ ଅପ୍, କନଫିଗର୍ ଏବଂ ପରିଚାଳନା କରିବା ପାଇଁ ଆବଶ୍ୟକ ସବୁକିଛି। ଇନ୍‌ଷ୍ଟଲେସନ୍, ପେମେଣ୍ଟ ଗେଟୱେ, ଡୋମେନ୍ ମ୍ୟାପିଂ, କ୍ଲାଇଏଣ୍ଟ ପରିଚାଳନା ଏବଂ ଅଧିକ କିଛି କଭର୍ କରେ।

### [ବିକାଶକାରୀ ଡକ୍ୟୁମେଣ୍ଟେସନ](/developer/hooks) {#developer-documentation}

Ultimate Multisite ଉପରେ ନିର୍ମାଣ କରୁଥିବା ବିକାଶକାରୀଙ୍କ ପାଇଁ ପ୍ରାୟୋଗିକ ସନ୍ଦର୍ଭ। hooks ସନ୍ଦର୍ଭ, REST API ଡକ୍ୟୁମେଣ୍ଟେସନ ଏବଂ ଆଡଅନ୍ ବିକାଶ ମାର୍ଗଦର୍ଶିକା ସମ୍ମିଳିତ।

### [ଆଡଅନ୍‌ଗୁଡ଼ିକ](/addons) {#addons}

WooCommerce ଏକୀକରଣ, ବିଶ୍ଳେଷଣ, ଡୋମେନ୍ ବିକ୍ରୟ ଏବଂ ଅଧିକ ବୈଶିଷ୍ଟ୍ୟ ସହିତ Ultimate Multisite କୁ ବିସ୍ତାର କରୁଥିବା ପେଇଡ୍ ଆଡଅନ୍ ପ୍ଲଗଇନ୍‌ଗୁଡ଼ିକ ପାଇଁ ଡକ୍ୟୁମେଣ୍ଟେସନ।
