# Content Sync ପରିବର୍ତ୍ତନ ତାଲିକା

### ସଂସ୍କରଣ 1.1.0 - 2025-12-31 ରେ ମୁକ୍ତିପ୍ରାପ୍ତ {#version-110---released-on-2025-12-31}
* ନୂଆ: ସମସ୍ତ ପୋଷ୍ଟ ପ୍ରକାର ସମନ୍ୱୟ ପାଇଁ ସମର୍ଥନ (ପୋଷ୍ଟ, ପୃଷ୍ଠା, କଷ୍ଟମ୍ ପୋଷ୍ଟ ପ୍ରକାର)
* ନୂଆ: ବିଷୟବସ୍ତୁ ଚୟନ ପଦକ୍ଷେପରେ ପୋଷ୍ଟ ପ୍ରକାର ଫିଲ୍ଟରିଂ
* ନୂଆ: ଉତ୍ସ ସାଇଟ୍ ଚୟନ ପଦକ୍ଷେପ - କେବଳ ମୁଖ୍ୟ ସାଇଟ୍ ନୁହେଁ, ଯେକୌଣସି ସାଇଟ୍‌ରୁ ସମନ୍ୱୟ
* ନୂଆ: ଟେମ୍ପଲେଟ୍ ସାଇଟ୍‌ଗୁଡ଼ିକ ପାଇଁ ଚାଇଲ୍ଡ ସାଇଟ୍ ପୂର୍ବ-ଚୟନ
* ନୂଆ: ସମସ୍ତ ପୋଷ୍ଟ ପ୍ରକାର ପାଇଁ post_content ରୁ ମାଧ୍ୟମ ନିଷ୍କାଷଣ
* ନୂଆ: ଟ୍ୟାକ୍ସୋନୋମି ସମନ୍ୱୟ (ଶ୍ରେଣୀ, ଟ୍ୟାଗ୍, କଷ୍ଟମ୍ ଟ୍ୟାକ୍ସୋନୋମି)
* ପରିବର୍ତ୍ତିତ: "Elementor Template Sync" ରୁ "Content Sync" କୁ ନାମ ବଦଳାଗଲା
* ପରିବର୍ତ୍ତିତ: "Select Templates" ପଦକ୍ଷେପର ନାମ "Select Data" କୁ ବଦଳାଗଲା
* ଉନ୍ନତ: Elementor-ନିର୍ଦ୍ଦିଷ୍ଟ କୋଡ୍ ଏବେ କେବଳ elementor_library ପୋଷ୍ଟ ପ୍ରକାର ପାଇଁ ଚାଲେ

### ସଂସ୍କରଣ 1.0.0 - 2025-12-19 ରେ ମୁକ୍ତିପ୍ରାପ୍ତ {#version-100---released-on-2025-12-19}
* ପ୍ରାରମ୍ଭିକ ମୁକ୍ତି
* ଟେମ୍ପଲେଟ୍ ଏବଂ ସାଇଟ୍ ଚୟନ ପାଇଁ ବହୁ-ପଦକ୍ଷେପ ୱିଜାର୍ଡ
* ବାସ୍ତବ-ସମୟ ଅଗ୍ରଗତି ପ୍ରତିକ୍ରିୟା ସହିତ ବ୍ୟାଚ୍ ପ୍ରକ୍ରିୟାକରଣ
* ସ୍ୱୟଂଚାଳିତ ID ପୁନଃମ୍ୟାପିଂ ସହିତ ପୂର୍ଣ୍ଣ ମାଧ୍ୟମ ସମନ୍ୱୟ
* Ultimate Multisite ସହିତ ସେଟିଂସ୍ ଏକୀକରଣ
* ନେଟୱର୍କ ସେଟିଂସ୍ ମାଧ୍ୟମରେ ସାଇଟ୍ ବାଦ ସମର୍ଥନ
* ସମନ୍ୱୟ ପରେ Elementor କ୍ୟାଶ୍ ସଫାକରଣ
* ଟେମ୍ପଲେଟ୍ ଟ୍ୟାକ୍ସୋନୋମି ସଂରକ୍ଷଣ
