# ବିଷୟବସ୍ତୁ ସମନ୍ୱୟ

Multisite Ultimate: ବିଷୟବସ୍ତୁ ସମନ୍ୱୟ

Multisite Ultimate ପାଇଁ ଏହି ବିସ୍ତାରଣ network ପରିଚାଳକମାନଙ୍କୁ ଯେକୌଣସି ସାଇଟ୍‌ରୁ network ଭିତରର ଅନ୍ୟ ସାଇଟ୍‌ଗୁଡ଼ିକୁ ବିଷୟବସ୍ତୁ ସମନ୍ୱୟ କରିବାକୁ ଦେଇଥାଏ। ଏହା ପୋଷ୍ଟ, ପୃଷ୍ଠା, କଷ୍ଟମ୍ ପୋଷ୍ଟ ପ୍ରକାର ଏବଂ Elementor ଟେମ୍ପଲେଟ୍ ସହିତ ସମସ୍ତ ପୋଷ୍ଟ ପ୍ରକାରକୁ ସମର୍ଥନ କରେ। ବିଷୟବସ୍ତୁ ଓ ଲକ୍ଷ୍ୟ ସାଇଟ୍ ବାଛିବା ପାଇଁ ଏହି ବିସ୍ତାରଣରେ ବହୁ-ପଦକ୍ଷେପ wizard interface ରହିଛି, ସହିତ batch processing ଏବଂ ତତ୍କ୍ଷଣାତ୍ ପ୍ରଗତି feedback ମଧ୍ୟ ମିଳେ।

## ବୈଶିଷ୍ଟ୍ୟ {#features}

* ବିଷୟବସ୍ତୁ ବାଛନା: ଥମ୍ନେଲ୍ ଓ ପୋଷ୍ଟ ପ୍ରକାର ଫିଲ୍ଟରିଂ ସହିତ ଭିଜୁଆଲ୍ କାର୍ଡ୍ ଗ୍ରିଡ୍
* ସମସ୍ତ ପୋଷ୍ଟ ପ୍ରକାର: ପୋଷ୍ଟ, ପୃଷ୍ଠା, ପ୍ରୋଡକ୍ଟ, Elementor ଟେମ୍ପଲେଟ୍ ଏବଂ ଯେକୌଣସି କଷ୍ଟମ୍ ପୋଷ୍ଟ ପ୍ରକାର ସମନ୍ୱୟ କରନ୍ତୁ
* ସାଇଟ୍ ବାଛନା: କେଉଁ ସାଇଟ୍‌ଗୁଡ଼ିକୁ ବିଷୟବସ୍ତୁ ସମନ୍ୱୟ କରିବେ ବାଛନ୍ତୁ, ଉତ୍ସ ସାଇଟ୍ ବାଦ ଦେବା ଏବଂ child ସାଇଟ୍ ପୂର୍ବ-ବାଛନା ସହିତ
* Batch Processing: ପ୍ରଗତି ଅଦ୍ୟତନ ସହିତ ଏକ ସମୟରେ ୧୦ଟି ସାଇଟ୍‌କୁ ସମନ୍ୱୟ କରେ
* ମିଡିଆ ସମନ୍ୱୟ: ପୋଷ୍ଟ ବିଷୟବସ୍ତୁରୁ ମିଡିଆ ଫାଇଲ୍‌ଗୁଡ଼ିକୁ (ଛବି, ଭିଡିଓ) ଲକ୍ଷ୍ୟ ସାଇଟ୍‌କୁ ସ୍ୱୟଂଚାଳିତ ଭାବେ କପି ଓ ପୁନଃମ୍ୟାପ୍ କରେ
* Taxonomy ସମନ୍ୱୟ: ବିଭାଗ, ଟ୍ୟାଗ୍ ଏବଂ କଷ୍ଟମ୍ taxonomy ସୁରକ୍ଷିତ ରଖେ
* Elementor ସମର୍ଥନ: CSS ପୁନର୍ଜନନ ଏବଂ ଟେମ୍ପଲେଟ୍ ID ପୁନଃମ୍ୟାପିଂ ସହିତ Elementor ଟେମ୍ପଲେଟ୍ ପାଇଁ ପୂର୍ଣ୍ଣ ସମର୍ଥନ
* ବିରୋଧ ସମାଧାନ: ଥିବା ବିଷୟବସ୍ତୁକୁ ବଦଳାଇବା କିମ୍ବା ଛାଡ଼ିଦେବାର ବିକଳ୍ପ
* Cache ପରିଚାଳନା: ସମନ୍ୱୟ ପରେ ଲକ୍ଷ୍ୟ ସାଇଟ୍‌ଗୁଡ଼ିକରେ Elementor cache ସ୍ୱୟଂଚାଳିତ ଭାବେ ସଫା କରେ

## ଆବଶ୍ୟକତା {#requirements}

* WordPress ୫.୩ କିମ୍ବା ତାଠାରୁ ଉଚ୍ଚ
* PHP ୭.୪ କିମ୍ବା ତାଠାରୁ ଉଚ୍ଚ
* Multisite Ultimate ପ୍ଲଗଇନ୍ (ସକ୍ରିୟ)
* Elementor ପ୍ଲଗଇନ୍ (ଇଚ୍ଛାଧୀନ, Elementor ଟେମ୍ପଲେଟ୍ ସମନ୍ୱୟ ପାଇଁ ଆବଶ୍ୟକ)

## ସ୍ଥାପନ {#installation}

1. 'ultimate-multisite-content-sync' କୁ '/wp-content/plugins/' ଡିରେକ୍ଟୋରୀରେ ଅପଲୋଡ୍ କରନ୍ତୁ
2. ଆପଣଙ୍କ WordPress Network Admin ରେ 'ପ୍ଲଗଇନ୍‌ଗୁଡ଼ିକ' ମେନୁ ମାଧ୍ୟମରେ ପ୍ଲଗଇନ୍ ସକ୍ରିୟ କରନ୍ତୁ
3. Ultimate Multisite → Settings → ବିଷୟବସ୍ତୁ ସମନ୍ୱୟ ରେ ସେଟିଂସ୍ ବିନ୍ୟାସ କରନ୍ତୁ
4. ସମନ୍ୱୟ ଆରମ୍ଭ କରିବାକୁ Ultimate Multisite → ବିଷୟବସ୍ତୁ ସମନ୍ୱୟ କୁ ଯାଆନ୍ତୁ
