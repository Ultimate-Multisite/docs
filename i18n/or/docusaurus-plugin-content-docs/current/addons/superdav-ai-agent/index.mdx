# Superdav AI Agent

Superdav AI Agent ଏକ ଉନ୍ନତ WordPress AI ସହାୟକ, ଯାହା Gratis AI Agent କୁ ଥିମ-ସଚେତନ କୌଶଳ, ଉନ୍ନତ ଇଣ୍ଟରନେଟ ସନ୍ଧାନ କ୍ଷମତା, ସାଇଟ ନିର୍ମାଣ ସୁବିଧା, ଏବଂ ବୈକଳ୍ପିକ ସ୍ୱୟଂଚାଳନ ସମନ୍ୱୟ ସହିତ ବିସ୍ତାର କରେ। ସଂସ୍କରଣ ୧.୧୮.୦ Google Calendar ପଢ଼ିବା ଟୁଲ, ସମ୍ପର୍କ ଓ ଅଂଶଗ୍ରହୀ ମ୍ୟାପିଂ, ମାନବ ଅନୁମୋଦନ ଗେଟ, ରିମାଇଣ୍ଡର ଡିଡୁପ୍ଲିକେସନ, TextBee ଏସଏମଏସ ସୂଚନା, ପରିଚାଳିତ Superdav ସେବା ପ୍ରୋଭିଜନିଂ, ଏବଂ ବିଶ୍ୱସ୍ତ ବିକାଶକାରୀ କାର୍ଯ୍ୟପ୍ରବାହ ପାଇଁ ପୃଥକ ଭାବେ ବିତରିତ Advanced ସହାୟକ ପ୍ୟାକେଜ ଯୋଡ଼େ।

## ମୁଖ୍ୟ ସୁବିଧା

- **ଥିମ-ସଚେତନ କୌଶଳ** — ଆପଣଙ୍କ ସକ୍ରିୟ WordPress ଥିମ ଏବଂ ସ୍ଥାପିତ ପ୍ଲଗଇନ ଆଧାରରେ ନିର୍ଦ୍ଦେଶନା ଓ କ୍ଷମତାକୁ ସ୍ୱୟଂଚାଳିତ ଭାବେ ଅନୁକୂଳ କରେ
- **ଅନେକ ଇଣ୍ଟରନେଟ ସନ୍ଧାନ ପ୍ରଦାତା** — ସମୃଦ୍ଧ ସନ୍ଧାନ ଫଳାଫଳ ଓ ଉନ୍ନତ ଗବେଷଣା କ୍ଷମତା ପାଇଁ Brave Search ଏବଂ Tavily ମଧ୍ୟରୁ ବାଛନ୍ତୁ
- **ସାଇଟ ନିର୍ମାତା ସମ୍ପର୍କ ଫର୍ମ** — ଆପଣଙ୍କ ସ୍ଥାପିତ ଫର୍ମ ପ୍ଲଗଇନ ବ୍ୟବହାର କରି ଚ୍ୟାଟ ଇଣ୍ଟରଫେସରୁ ସିଧାସଳଖ ସମ୍ପୂର୍ଣ୍ଣ କାର୍ଯ୍ୟକ୍ଷମ ସମ୍ପର୍କ ଫର୍ମ ତିଆରି କରନ୍ତୁ
- **ଉନ୍ନତ ସାଇଟ ନିର୍ମାତା** — ଭଲ ପ୍ଲଗଇନ ଆବିଷ୍କାର ଓ ତ୍ରୁଟି ପୁନରୁଦ୍ଧାର ସହିତ ଉନ୍ନତ ବହୁ-ପର୍ଯ୍ୟାୟ ସାଇଟ ନିର୍ମାଣ ଯୋଜନା
- **ଅନ୍ତର୍ନିହିତ ଥିମ କୌଶଳ** — Block Themes, Classic Themes, Kadence Blocks, ଏବଂ Kadence Theme ପାଇଁ ବିଶେଷ ନିର୍ଦ୍ଦେଶନା
- **ଉନ୍ନତ WooCommerce ସମନ୍ୱୟ** — ଭଲ ଭରସାଯୋଗ୍ୟତା ଓ ସୁସଙ୍ଗତତା ପାଇଁ ମୂଳ WooCommerce API ବ୍ୟବହାର କରିବାକୁ ପୁନର୍ଗଠିତ
- **ସ୍ୱୟଂଚାଳିତ ପ୍ରଦାତା ଚିହ୍ନଟ** — ପ୍ଲଗଇନ ଓ ଥିମ ସକ୍ରିୟ କିମ୍ବା ନିଷ୍କ୍ରିୟ ହେଲେ ଉପଲବ୍ଧଗୁଡ଼ିକୁ ସ୍ୱୟଂଚାଳିତ ଭାବେ ଚିହ୍ନଟ ଓ ସତେଜ କରେ
- **କଷ୍ଟମ ଲୋଗୋ ସୃଷ୍ଟି** — ସ୍ୱୟଂଚାଳିତ ନେମସ୍ପେସ-ସୁରକ୍ଷିତ ପରିଷ୍କାର ସହିତ କଷ୍ଟମ SVG ଲୋଗୋ ସୃଷ୍ଟି ଓ ଏମ୍ବେଡ କରନ୍ତୁ
- **ପ୍ୟାଲେଟ କନ୍ଟ୍ରାଷ୍ଟ ବୈଧତା ଯାଞ୍ଚ** — ଆପଣଙ୍କ ଥିମରେ ପ୍ରୟୋଗ କରିବା ପୂର୍ବରୁ WCAG ପ୍ରବେଶଯୋଗ୍ୟତା ଅନୁସରଣ ପାଇଁ ରଙ୍ଗ ଯୁଗଳ ଯାଞ୍ଚ କରନ୍ତୁ
- **ଫଟୋ-ଉନ୍ନତ ଆବିଷ୍କାର** — ଡିଜାଇନ ଦିଗକୁ ପ୍ରଭାବିତ କରିବା ପାଇଁ Theme Builder ଆବିଷ୍କାର ସାକ୍ଷାତକାର ସମୟରେ ଫଟୋ ଅପଲୋଡ କରନ୍ତୁ
- **ଆତିଥ୍ୟ ମେନୁ** — ରେଷ୍ଟୁରାଣ୍ଟ, କାଫେ ଏବଂ ଆତିଥ୍ୟ ବ୍ୟବସାୟ ପାଇଁ ସଂରଚିତ ଖାଦ୍ୟ ଓ ପାନୀୟ ମେନୁ ପୃଷ୍ଠା ସୃଷ୍ଟି କରନ୍ତୁ
- **ଡେସ୍କଟପ ଏବଂ ମୋବାଇଲ ପୂର୍ବାବଲୋକନ** — ଡିଜାଇନ-ଦିଗ ଚୟନ ସମୟରେ ଡେସ୍କଟପ ଓ ମୋବାଇଲ ଉପକରଣରେ ଆପଣଙ୍କ ଡିଜାଇନ ପୂର୍ବାବଲୋକନ କରନ୍ତୁ
- **ଟିୟର ୧ ସାଇଟ ସ୍କ୍ରାପିଂ** — sd-ai-agent/site-scrape ବର୍ତ୍ତମାନ ପ୍ରତିଦ୍ୱନ୍ଦ୍ୱୀ ଓ ପ୍ରେରଣାଦାୟକ ୱେବସାଇଟ ବିଶ୍ଳେଷଣ ପାଇଁ ପୂର୍ବନିର୍ଦ୍ଧାରିତ ଭାବେ ଉପଲବ୍ଧ
- **Google Calendar ପଢ଼ିବା ଟୁଲ** — ବିନ୍ୟାସିତ କ୍ୟାଲେଣ୍ଡର ଓ ଘଟଣା ପଢ଼ନ୍ତୁ, ଯାହାଦ୍ୱାରା ସ୍ୱୟଂଚାଳନ ଆସନ୍ତା ସମୟସୂଚୀ ପରିପ୍ରେକ୍ଷ୍ୟ ବୁଝିପାରିବ
- **ସମ୍ପର୍କ ଏବଂ ଅଂଶଗ୍ରହୀ ମ୍ୟାପିଂ** — ପରବର୍ତ୍ତୀ କାମ ସୃଷ୍ଟି କରିବା ପୂର୍ବରୁ କ୍ୟାଲେଣ୍ଡର ଅଂଶଗ୍ରହୀମାନଙ୍କୁ ଜଣାଶୁଣା ସାଇଟ ବ୍ୟବହାରକାରୀ ଓ ସମ୍ପର୍କ ସହିତ ମେଳାନ୍ତୁ
- **ମାନବ ଅନୁମୋଦନ ଗେଟ** — ସନ୍ଦେଶ ପଠାଇବା, କାମ ପୁନରାରମ୍ଭ କରିବା, କିମ୍ବା ପରିବର୍ତ୍ତନ କରିବା ପୂର୍ବରୁ ସମୀକ୍ଷା ପାଇଁ ସମ୍ବେଦନଶୀଳ ସ୍ୱୟଂଚାଳନକୁ ବିରତ କରନ୍ତୁ
- **ରିମାଇଣ୍ଡର ରେକର୍ଡ ଏବଂ ଡିଡୁପ୍ଲିକେସନ** — ପୂର୍ବରୁ ପଠାଯାଇଥିବା ରିମାଇଣ୍ଡରଗୁଡ଼ିକୁ ଟ୍ରାକ କରନ୍ତୁ, ଯାହାଦ୍ୱାରା ପୁନରାବୃତ୍ତ ସ୍ୱୟଂଚାଳନ ଚଳାଣ ସେହି ଗ୍ରାହକଙ୍କୁ ଦୁଇଥର ସୂଚନା ନଦେବ
- **TextBee ଏସଏମଏସ ସୂଚନା** — ଏସଏମଏସ ପ୍ରଦାତା ସକ୍ରିୟ ଥିଲେ TextBee ମାଧ୍ୟମରେ ବିନ୍ୟାସିତ ଟେକ୍ସଟ-ସନ୍ଦେଶ ସୂଚନା ପଠାନ୍ତୁ
- **ପରିଚାଳିତ Superdav ସେବା ପ୍ରୋଭିଜନିଂ** — ହାତେ endpoint ବିନ୍ୟାସ ପରିବର୍ତ୍ତେ ସ୍ୱୟଂଚାଳିତ ସେଟଅପ ସହିତ ସମର୍ଥିତ ସାଇଟଗୁଡ଼ିକୁ ହୋଷ୍ଟେଡ Superdav endpoint ସହିତ ସଂଯୋଗ କରନ୍ତୁ
- **Advanced ସହାୟକ ପ୍ୟାକେଜ** — ଫାଇଲସିଷ୍ଟମ, ଡାଟାବେସ, WP-CLI, REST ଡିସ୍ପାଚର, ପ୍ଲଗଇନ ନିର୍ମାତା, git ସ୍ନାପଶଟ, ବ୍ୟବହାରକାରୀ-ପରିଚାଳନା, ଏବଂ ବେଞ୍ଚମାର୍କ କାର୍ଯ୍ୟପ୍ରବାହ ପାଇଁ ପୃଥକ ଭାବେ ବିତରିତ ବିଶ୍ୱସ୍ତ-ବିକାଶକାରୀ ଟୁଲ

## ଆବଶ୍ୟକତା

- WordPress ୬.୦ କିମ୍ବା ତାଠାରୁ ଅଧିକ
- PHP ୮.୧ କିମ୍ବା ତାଠାରୁ ଅଧିକ
- WordPress Multisite (ନେଟୱର୍କ-ସକ୍ରିୟ) କିମ୍ବା ଏକକ-ସାଇଟ ସ୍ଥାପନ
- ଏକ ସକ୍ରିୟ AI ପ୍ରଦାତା ସଂଯୋଜକ ପ୍ଲଗଇନ (Anthropic, OpenAI, କିମ୍ବା ସୁସଙ୍ଗତ)
- Gratis AI Agent v1.9.0 କିମ୍ବା ତାଠାରୁ ଅଧିକ (Superdav AI Agent, Gratis AI Agent କୁ ବିସ୍ତାର କରେ)

## ସ୍ଥାପନ

1. ପ୍ଲଗଇନ ଫାଇଲଗୁଡ଼ିକୁ `/wp-content/plugins/superdav-ai-agent` କୁ ଅପଲୋଡ କରନ୍ତୁ
2. WordPress ରେ **ପ୍ଲଗଇନ** ପରଦା ମାଧ୍ୟମରେ ପ୍ଲଗଇନ ସକ୍ରିୟ କରନ୍ତୁ
3. Gratis AI Agent ମଧ୍ୟ ସକ୍ରିୟ ଅଛି ବୋଲି ନିଶ୍ଚିତ କରନ୍ତୁ
4. **Gratis AI Agent → Settings** କୁ ଯାଇ ଆବଶ୍ୟକତା ଅନୁଯାୟୀ ଆପଣଙ୍କ ଇଣ୍ଟରନେଟ ସନ୍ଧାନ ପ୍ରଦାତା, ପରିଚାଳିତ Superdav ସେବା ସଂଯୋଗ, Google Calendar, କିମ୍ବା TextBee ପରିଚୟପତ୍ର ବିନ୍ୟାସ କରନ୍ତୁ

WordPress.org ପ୍ୟାକେଜରେ ମୂଳ Superdav AI Agent ସୁବିଧା ରହିଛି। ବିଶ୍ୱସ୍ତ ବିକାଶକାରୀ ଟୁଲ ସ୍ପଷ୍ଟ ଭାବେ ସ୍ଥାପିତ ଓ ଅନୁମୋଦିତ ଥିବା ପରିବେଶ ପାଇଁ Advanced ସହାୟକ ପ୍ୟାକେଜ ପୃଥକ ଭାବେ ନିର୍ମିତ ଓ ବିତରିତ ହୁଏ।

## ଥିମ-ସଚେତନ କୌଶଳ

Superdav AI Agent ଆପଣଙ୍କ ସକ୍ରିୟ ଥିମ ଏବଂ ସ୍ଥାପିତ ପ୍ଲଗଇନଗୁଡ଼ିକୁ ସ୍ୱୟଂଚାଳିତ ଭାବେ ଚିହ୍ନଟ କରେ, ଏବଂ ସହାୟକର ପରିପ୍ରେକ୍ଷ୍ୟରେ ଉପଯୁକ୍ତ ଥିମ-ସଚେତନ କୌଶଳ ଲୋଡ କରେ। ଉପଲବ୍ଧ କୌଶଳମାନଙ୍କ ମଧ୍ୟରେ ଅଛି:

| କୌଶଳ | କାହା ପାଇଁ ପ୍ରୟୋଜ୍ୟ |
|---|---|
| **Block Themes** | WordPress ବ୍ଲକ-ଆଧାରିତ (Full Site Editing) ସଂରଚନା ବ୍ୟବହାର କରୁଥିବା ଥିମ |
| **Classic Themes** | PHP ଟେମ୍ପଲେଟ ଏବଂ କ୍ଲାସିକ ସମ୍ପାଦକ ବ୍ୟବହାର କରୁଥିବା ପାରମ୍ପରିକ WordPress ଥିମ |
| **Kadence Blocks** | Kadence Blocks ପ୍ଲଗଇନ ସକ୍ରିୟ ଥିବା ସାଇଟ |
| **Kadence Theme** | ସକ୍ରିୟ ଥିମ ଭାବେ Kadence ଥିମ ଥିବା ସାଇଟ |

ପ୍ରତ୍ୟେକ କୌଶଳ ବିଷୟରେ ବିସ୍ତୃତ ନିର୍ଦ୍ଦେଶନା ପାଇଁ [ଥିମ-ସଚେତନ କୌଶଳ](../../user-guide/configuration/theme-aware-skills) ଦେଖନ୍ତୁ।

Superdav AI Agent v1.10.0 ଇଣ୍ଟରନେଟ ସନ୍ଧାନ ପ୍ରଶ୍ନ ପାଇଁ Brave Search ର ବିକଳ୍ପ ଭାବେ **Tavily** ଯୋଡ଼େ। ଉଭୟ ପ୍ରଦାତା ତତ୍କ୍ଷଣାତ୍ ସନ୍ଧାନ ଫଳାଫଳ ଦିଅନ୍ତି ଏବଂ **Gratis AI Agent → Settings → Advanced** ରେ ବିନ୍ୟାସ କରାଯାଇପାରେ।

| Provider | ବିଶେଷତା | ସେଟଅପ |
|---|---|---|
| **Brave Search** | ଦ୍ରୁତ, ଗୋପନୀୟତା-କେନ୍ଦ୍ରିତ ସନ୍ଧାନ ଫଳାଫଳ | [Brave Search API](https://brave.com/search/api/) |

## କ୍ୟାଲେଣ୍ଡର, ଅନୁମୋଦନ, ଏବଂ ସୂଚନା ୱର୍କଫ୍ଲୋ

Superdav AI Agent v1.18.0 ବିଷୟବସ୍ତୁ ଓ ସାଇଟ୍-ନିର୍ମାଣ କାର୍ଯ୍ୟରୁ ଆଗକୁ ଅଟୋମେସନ୍ ୱର୍କଫ୍ଲୋକୁ ବିସ୍ତାର କରେ:

- **Google Calendar ଟୁଲ୍‌ଗୁଡ଼ିକ** କ୍ୟାଲେଣ୍ଡର କ୍ରେଡେନ୍ସିଆଲ୍ ଉପଲବ୍ଧ ଥିଲେ ଏଜେଣ୍ଟକୁ କନଫିଗର୍ କରାଯାଇଥିବା କ୍ୟାଲେଣ୍ଡର ଓ ଇଭେଣ୍ଟ ବିବରଣୀ ପଢ଼ିବାକୁ ଦିଏ।
- **ଯୋଗାଯୋଗ ଓ ଅଂଶଗ୍ରାହୀ ସହାୟକଗୁଡ଼ିକ** ସ୍ମରଣିକା କିମ୍ବା ପରବର୍ତ୍ତୀ କାର୍ଯ୍ୟ ପ୍ରସ୍ତୁତ କରିବା ପୂର୍ବରୁ ଇଭେଣ୍ଟ ଅଂଶଗ୍ରାହୀମାନଙ୍କୁ WordPress ବ୍ୟବହାରକାରୀ କିମ୍ବା ପରିଚିତ ଯୋଗାଯୋଗ ସହିତ ମ୍ୟାପ୍ କରେ।
- **ମାନବ ଅନୁମୋଦନ ଗେଟ୍‌ଗୁଡ଼ିକ** ପ୍ରସ୍ତାବିତ କାର୍ଯ୍ୟକୁ ଏକ ଅନୁମୋଦିତ ବ୍ୟବହାରକାରୀ ସମୀକ୍ଷା କରିବା ପର୍ଯ୍ୟନ୍ତ ଅଟୋମେସନ୍‌କୁ ବିରତ କରିବାକୁ ଦିଏ।
- **ସ୍ମରଣିକା ରେକର୍ଡଗୁଡ଼ିକ** ଅଟୋମେସନ୍ ପୁନଃଚେଷ୍ଟା କିମ୍ବା ବାରମ୍ବାର ସୂଚୀବଦ୍ଧ ହେଲେ ନକଲି କ୍ୟାଲେଣ୍ଡର କିମ୍ବା ସୂଚନା ସ୍ମରଣିକାକୁ ରୋକେ।
- **TextBee SMS ସମର୍ଥନ** TextBee ପ୍ରଦାତା କନଫିଗର୍ ହୋଇଥିଲେ ଟେକ୍ସ୍ଟ-ମେସେଜ୍ ସୂଚନାକୁ ସକ୍ଷମ କରେ।

ସମୟସୂଚୀ-ସଚେତନ ସହାୟତା, ଅଂଶଗ୍ରାହୀ ପରବର୍ତ୍ତୀ କାର୍ଯ୍ୟ, ଏବଂ ସମ୍ବେଦନଶୀଳ ସନ୍ଦେଶ ପଠାଇବା ପୂର୍ବରୁ ଏପର୍ଯ୍ୟନ୍ତ ମାନବ ସମୀକ୍ଷା ଆବଶ୍ୟକ ଥିବା ସ୍ମରଣିକା ୱର୍କଫ୍ଲୋ ପାଇଁ ଏହି ଟୁଲ୍‌ଗୁଡ଼ିକ ବ୍ୟବହାର କରନ୍ତୁ।

Site Builder ଏଜେଣ୍ଟ ଏବେ ଚାଟ୍ ଇଣ୍ଟରଫେସ୍‌ରୁ ସିଧାସଳଖ ଯୋଗାଯୋଗ ଫର୍ମ ତିଆରି କରିପାରେ। କେବଳ Site Builderକୁ ଏକ ଯୋଗାଯୋଗ ଫର୍ମ ତିଆରି କରିବାକୁ କହନ୍ତୁ, ଏବଂ ଏହା ଆପଣଙ୍କର ସ୍ଥାପିତ ଫର୍ମ plugin (Contact Form 7, WPForms, Fluent Forms, କିମ୍ବା Gravity Forms) ସ୍ୱୟଂଚାଳିତ ଭାବେ ଚିହ୍ନଟ କରି ବ୍ୟବହାର-ପ୍ରସ୍ତୁତ ଫର୍ମ ସୃଷ୍ଟି କରିବ।

ବିସ୍ତୃତ ମାର୍ଗଦର୍ଶନ ପାଇଁ [ସାଇଟ୍ ବିଲ୍ଡର୍ ଯୋଗାଯୋଗ ଫର୍ମ](../../user-guide/administration/site-builder-contact-forms) ଦେଖନ୍ତୁ।

## Theme Builder ବିଶେଷତା

Superdav AI Agent v1.16.0 ରେ Theme Builder ଶକ୍ତିଶାଳୀ ନୂଆ କ୍ଷମତାଗୁଡ଼ିକ ସମ୍ମିଳିତ କରେ:

### ଫଟୋ ଅପଲୋଡ୍ ସହିତ ଡିସକଭରି ସାକ୍ଷାତ୍କାର

Theme Builder ଡିସକଭରି ସାକ୍ଷାତ୍କାରରେ ଏବେ ଏକ ଫଟୋ ଅପଲୋଡ୍ ପଦକ୍ଷେପ ଅଛି, ଯେଉଁଠାରେ ଆପଣ ଦୃଶ୍ୟ ପ୍ରେରଣା ଦେଇପାରିବେ। ଆପଣଙ୍କ ବ୍ରାଣ୍ଡ, ପ୍ରତିଦ୍ୱନ୍ଦ୍ୱୀ ସାଇଟ୍, କିମ୍ବା ଡିଜାଇନ୍ ପ୍ରେରଣାର ଫଟୋ ଅପଲୋଡ୍ କରନ୍ତୁ, ଯାହା Theme Builderକୁ ଆପଣଙ୍କ ଡିଜାଇନ୍ ଦିଗ ବୁଝିବାରେ ସହାୟତା କରିବ।

ବିସ୍ତୃତ ମାର୍ଗଦର୍ଶନ ପାଇଁ [ଡିସକଭରି ସାକ୍ଷାତ୍କାର](./theme-builder/discovery-interview.md) ଦେଖନ୍ତୁ।

### ଡେସ୍କଟପ୍ ଓ ମୋବାଇଲ୍ ପୂର୍ବାବଲୋକନ ସହିତ ଡିଜାଇନ୍ ଦିଗ

Theme Builder ଆପଣଙ୍କ ସମ୍ପୂର୍ଣ୍ଣ theme ସୃଷ୍ଟି କରିବା ପୂର୍ବରୁ ଡେସ୍କଟପ୍ ଓ ମୋବାଇଲ୍ ଡିଭାଇସ୍ ଉଭୟରେ ଆପଣଙ୍କ ଡିଜାଇନ୍ ସମୀକ୍ଷା କରନ୍ତୁ। ଅନେକ ରେସ୍ପନ୍ସିଭ୍ ବ୍ରେକ୍ପଏଣ୍ଟରେ ରିୟଲ୍-ଟାଇମ୍ ପୂର୍ବାବଲୋକନ ଦେଖନ୍ତୁ।

ବିସ୍ତୃତ ମାର୍ଗଦର୍ଶନ ପାଇଁ [ଡିଜାଇନ୍ ଦିଗ](./theme-builder/design-direction.md) ଦେଖନ୍ତୁ।

### ଆତିଥ୍ୟ ମେନୁ

Theme Builder ଏବେ ରେଷ୍ଟୁରାଣ୍ଟ, କାଫେ, ବାର୍, ଏବଂ ଅନ୍ୟ ଆତିଥ୍ୟ ବ୍ୟବସାୟ ପାଇଁ ସଂରଚିତ ଖାଦ୍ୟ ଓ ପାନୀୟ ମେନୁ ପୃଷ୍ଠା ସୃଷ୍ଟି କରିପାରେ। ମେନୁରେ ମୂଲ୍ୟ, ବର୍ଣ୍ଣନା, ଖାଦ୍ୟ-ସମ୍ବନ୍ଧୀୟ ସୂଚନା, ଏବଂ ଆଲର୍ଜେନ୍ ସତର୍କତା ଥାଏ।

ବିସ୍ତୃତ ମାର୍ଗଦର୍ଶନ ପାଇଁ [ଆତିଥ୍ୟ ମେନୁ](./theme-builder/hospitality-menus.md) ଦେଖନ୍ତୁ।

- **[ମେନୁ ତିଆରି କରନ୍ତୁ](./abilities/create-menu.md)** — ସ୍ୱତନ୍ତ୍ର ନାଭିଗେସନ୍ ଲେବଲ୍ ସହିତ ନାଭିଗେସନ୍ ମେନୁ ତିଆରି କରନ୍ତୁ
