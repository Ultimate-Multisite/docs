# ପରିବର୍ତ୍ତନ ତାଲିକା {#changelog}

## 1.18.0 — ୨୦୨୬-୦୬-୨୯ରେ ମୁକ୍ତିଲାଭ {#1180--released-on-2026-06-29}

### ନୂଆ {#new}

- **Google Calendar ଟୁଲ୍‌ଗୁଡ଼ିକ** — ସମୟସୂଚୀ-ସଚେତନ ସ୍ୱୟଂଚାଳନ ପାଇଁ ବିନ୍ୟାସିତ କ୍ୟାଲେଣ୍ଡର ଓ ଇଭେଣ୍ଟ ପଢ଼ନ୍ତୁ
- **ଯୋଗାଯୋଗ ମ୍ୟାପିଂ ଓ ଅଂଶଗ୍ରାହୀ ସହାୟକ** — କ୍ୟାଲେଣ୍ଡର ଅଂଶଗ୍ରାହୀଙ୍କୁ ସାଇଟ୍ ବ୍ୟବହାରକାରୀ ଓ ଯୋଗାଯୋଗ ସହିତ ମେଳ କରନ୍ତୁ
- **ମାନବ ଅନୁମୋଦନ ଗେଟ୍ ଓ ସ୍ମରଣ ରେକର୍ଡ** — ସମୀକ୍ଷା ପାଇଁ ସ୍ୱୟଂଚାଳନକୁ ବିରତ କରନ୍ତୁ ଏବଂ ପୁନରାବୃତ୍ତି ସୂଚନା ଏଡ଼ାନ୍ତୁ
- **TextBee SMS ପ୍ରଦାନକାରୀ** — TextBee ମାଧ୍ୟମରେ ବିନ୍ୟାସିତ ପାଠ୍ୟ-ବାର୍ତ୍ତା ସୂଚନା ପଠାନ୍ତୁ
- **ଉନ୍ନତ ସହଚର ପ୍ୟାକେଜ୍** — WordPress.org ନିର୍ମାଣରୁ ଅଲଗା ଭାବେ ବିତରିତ ବିଶ୍ୱସ୍ତ-ବିକାଶକାରୀ ଫାଇଲ୍‌ସିଷ୍ଟମ୍, ଡାଟାବେସ୍, WP-CLI, REST ଡିସ୍ପାଚର୍, plugin ନିର୍ମାତା, git ସ୍ନାପସଟ୍, ବ୍ୟବହାରକାରୀ-ପରିଚାଳନା ଓ ବେଞ୍ଚମାର୍କ ଟୁଲ୍ ଯୋଡ଼ନ୍ତୁ

### ଉନ୍ନତ {#improved}

- **ପରିଚାଳିତ Superdav ସେବା ସେଟଅପ୍** — ସମର୍ଥିତ ସାଇଟ୍‌ଗୁଡ଼ିକ ପାଇଁ ହୋଷ୍ଟ କରାଯାଇଥିବା ସେବା endpoint ଓ ସ୍ୱୟଂଚାଳିତ ସଂଯୋଗ ବ୍ୟବସ୍ଥା ଯୋଡ଼ନ୍ତୁ
- **ରିଲିଜ୍ ପ୍ୟାକେଜିଂ** — ଅଲଗା core ଓ Advanced ZIP ତିଆରି କରନ୍ତୁ, ଉଭୟକୁ GitHubରେ ପ୍ରକାଶ କରନ୍ତୁ, ଏବଂ କେବଳ core ପ୍ୟାକେଜ୍‌କୁ WordPress.orgକୁ ପଠାନ୍ତୁ

### ସୁଧାରିତ {#fixed}

- **AI ଅନୁରୋଧ ନିର୍ଭରଯୋଗ୍ୟତା** — ମଡେଲ୍ ଚୟନ, ଅନୁରୋଧ ସମୟସୀମା, ଡିଫଲ୍ଟ ସେଟଅପ୍ ମଡେଲ୍, ତର୍କ-ପାଠ୍ୟ ପରିଚାଳନା, ଏବଂ ଅବୈଧ ଟୁଲ୍ କଲ୍ ପାଇଁ ପୁନଃଚେଷ୍ଟା ମାର୍ଗଦର୍ଶନକୁ ଉନ୍ନତ କରନ୍ତୁ
- **କ୍ୟାଲେଣ୍ଡର ଓ ସ୍ମରଣ ସୁଦୃଢ଼କରଣ** — Google Calendar ଟୋକେନ୍ ଓ ସ୍ମରଣ ପୁନରାବୃତ୍ତି ହଟାଇବାକୁ ଅଧିକ ମଜବୁତ କରନ୍ତୁ
- **ଆରମ୍ଭିକ ପରିଚୟ ଓ ଅନୁମୋଦନ ପୁନରାରମ୍ଭ** — frontend ଆରମ୍ଭିକ ପରିଚୟ ଲଞ୍ଚ ଓ ନିଶ୍ଚିତ କ୍ଷମତା ପୁନରାରମ୍ଭକୁ ଠିକ୍ କରନ୍ତୁ
- **WordPress.org ପ୍ୟାକେଜିଂ ସମୀକ୍ଷା ସମସ୍ୟା** — core ରିଲିଜ୍ ପାଇଁ ପ୍ୟାକେଜିଂ ସମୀକ୍ଷା ମତାମତକୁ ସମାଧାନ କରନ୍ତୁ

## 1.16.0 — ୨୦୨୬-୦୫-୨୦ରେ ମୁକ୍ତିଲାଭ {#1160--released-on-2026-05-20}

### ନୂଆ {#new-1}

- **Logo SVG ସୃଷ୍ଟି କରିବାର କ୍ଷମତା** — Theme Builder ବର୍ତ୍ତମାନ namespace-ସୁରକ୍ଷିତ ପରିଷ୍କାରଣ ସହିତ କଷ୍ଟମ୍ ଲୋଗୋ SVG ସୃଷ୍ଟି ଓ ଏମ୍ବେଡ୍ କରିପାରେ
- **ଡିସକଭରି ସାକ୍ଷାତ୍କାରରେ ଫଟୋ ଅପଲୋଡ୍** — ଅଧିକ ସମୃଦ୍ଧ ଡିଜାଇନ୍ ପରିପ୍ରେକ୍ଷ୍ୟ ପାଇଁ Theme Builder ଡିସକଭରି ସାକ୍ଷାତ୍କାରରେ ବର୍ତ୍ତମାନ ଫଟୋ ଅପଲୋଡ୍ ପଦକ୍ଷେପ ଅଛି
- **Palette Contrast ବୈଧ କରିବାର କ୍ଷମତା** — themeରେ ପ୍ରୟୋଗ କରିବା ପୂର୍ବରୁ WCAG ଅନୁପାଳନ ପାଇଁ ରଙ୍ଗ ଯୁଗଳ ଯାଞ୍ଚ କରନ୍ତୁ
- **ଆତିଥ୍ୟ ମେନୁ** — ଆତିଥ୍ୟ ବ୍ୟବସାୟ ପାଇଁ Theme Builder ବର୍ତ୍ତମାନ ଗଠିତ ଖାଦ୍ୟ ଓ ପାନୀୟ ମେନୁ ପୃଷ୍ଠା ସୃଷ୍ଟି କରିପାରେ
- **ଡେସ୍କଟପ୍ ଓ ମୋବାଇଲ୍ ପୂର୍ବଦର୍ଶନ ରେଣ୍ଡରିଂ** — ଡିଜାଇନ୍-ଦିଗ ଚୟନ ସମୟରେ ଡେସ୍କଟପ୍ ଓ ମୋବାଇଲ୍ ଡିଭାଇସ୍‌ରେ ଆପଣଙ୍କ ଡିଜାଇନ୍ ପୂର୍ବଦର୍ଶନ କରନ୍ତୁ
- **ନେଭିଗେସନ୍ ଲେବେଲ୍ ପାରାମିଟର୍** — Create Menu କ୍ଷମତା ବର୍ତ୍ତମାନ ପୃଷ୍ଠା ଶିରୋନାମାରୁ ଅଲଗା ଏକ ସ୍ୱତନ୍ତ୍ର `navigation_label`କୁ ସମର୍ଥନ କରେ
- **Tier 1 ଟୁଲ୍ ଉପଲବ୍ଧତା** — `sd-ai-agent/site-scrape` ବର୍ତ୍ତମାନ Theme Builderରେ ଡିଫଲ୍ଟ ଭାବେ ଉପଲବ୍ଧ ଏକ Tier 1 ଟୁଲ୍

### ସୁଧାରିତ {#fixed-1}

- **AI Client କ୍ୟାଶ୍** — ବର୍ତ୍ତମାନ cross-request ସ୍ଥାୟୀତ୍ୱ ପାଇଁ transients ଦ୍ୱାରା ସମର୍ଥିତ, ଦୀର୍ଘ ସମୟ ଚାଲୁଥିବା agent କାର୍ଯ୍ୟରେ ତଥ୍ୟ ହାନିକୁ ରୋକେ
- **Plugin ଧାଡ଼ି କାର୍ଯ୍ୟ ଲିଙ୍କ** — ସ୍ପଷ୍ଟତା ପାଇଁ ସଂଶୋଧିତ ଓ ପୁନର୍ନାମିତ

## 1.10.0 — ୨୦୨୬-୦୫-୦୫ରେ ମୁକ୍ତିଲାଭ {#1100--released-on-2026-05-05}

### ନୂଆ {#new-2}

- **Tavily ଇଣ୍ଟରନେଟ୍ ସନ୍ଧାନ** — Brave Search ସହିତ ଅଧିକ ସମୃଦ୍ଧ ଇଣ୍ଟରନେଟ୍ ସନ୍ଧାନ ଫଳାଫଳ ପାଇଁ Tavilyକୁ ସନ୍ଧାନ ପ୍ରଦାନକାରୀ ଭାବେ ଯୋଡ଼ନ୍ତୁ
- **Theme-ସଚେତନ ନିର୍ମିତ କୌଶଳ** — Block Themes, Classic Themes, Kadence Blocks, ଏବଂ Kadence Theme କୌଶଳ ମାର୍ଗଦର୍ଶିକା ବର୍ତ୍ତମାନ plugin ସହିତ ଆସେ
- **ସାଇଟ୍ ନିର୍ମାତା ଯୋଗାଯୋଗ ଫର୍ମ କ୍ଷମତା** — ଚାଟ୍ ଇଣ୍ଟରଫେସ୍‌ରୁ ସିଧାସଳଖ ଯେକୌଣସି ପୃଷ୍ଠାରେ ଯୋଗାଯୋଗ ଫର୍ମ ଯୋଡ଼ନ୍ତୁ

### ଉନ୍ନତ {#improved-1}

- **WooCommerce ସମନ୍ୱୟ ପୁନର୍ଗଠିତ** — ଉତ୍ତମ ନିର୍ଭରଯୋଗ୍ୟତା ଓ ସୁସଙ୍ଗତତା ପାଇଁ ବର୍ତ୍ତମାନ ମୂଳ WooCommerce API ବ୍ୟବହାର କରେ
- **ପ୍ରଦାନକାରୀ ତାଲିକା ସ୍ୱୟଂଚାଳିତ ଭାବେ ତାଜା ହୁଏ** — ଯେତେବେଳେ କୌଣସି plugin ସକ୍ରିୟ କିମ୍ବା ନିଷ୍କ୍ରିୟ କରାଯାଏ

### ସୁଧାରିତ {#fixed-2}

- **navigate-to କ୍ଷମତା** — କିଛି admin ପୃଷ୍ଠାରେ ଅସୀମ ପୁନଃଲୋଡ୍ ଲୁପ୍ ଠିକ୍ କରାଗଲା
- **list-posts କ୍ଷମତା** — ବର୍ତ୍ତମାନ category ଓ tag ନାମକୁ ଠିକ୍ ଭାବେ slugରେ ପରିଣତ କରେ
- **WP-CLI କମାଣ୍ଡଗୁଡ଼ିକ** — ପୂର୍ବ ପୁନର୍ଗଠନ ପରେ ହରାଇଥିବା namespace aliasଗୁଡ଼ିକୁ ପୁନଃସ୍ଥାପିତ କରାଗଲା
- **ଇଭେଣ୍ଟ ସ୍ୱୟଂଚାଳନ** — ସ୍ୱୟଂଚାଳନ ଟେବୁଲ୍ ଏପର୍ଯ୍ୟନ୍ତ ସୃଷ୍ଟି ହୋଇନଥିବା ସାଇଟ୍‌ଗୁଡ଼ିକୁ ସୁଚାରୁ ଭାବେ ପରିଚାଳନା କରେ
- **memory-save କ୍ଷମତା** — ବର୍ତ୍ତମାନ ସିଷ୍ଟମ୍ ନିର୍ଦ୍ଦେଶ ନିର୍ମାତାରେ ସଠିକ୍ namespace prefix ବ୍ୟବହାର କରେ
- **Scalar ଟୁଲ୍ ଫଳାଫଳ** — AIକୁ ଫେରାଇବା ପୂର୍ବରୁ ବର୍ତ୍ତମାନ ଠିକ୍ ଭାବେ ମୁଡ଼ାଯାଏ
- **ବ୍ୟବହାର ପରିସଂଖ୍ୟାନ** — ପୁରୁଣା ସଂସ୍କରଣରୁ upgrade ସମୟରେ ବର୍ତ୍ତମାନ legacy କ୍ଷମତା key ଫର୍ମାଟ୍‌କୁ ଠିକ୍ ଭାବେ ପରିଚାଳନା କରେ
