# Logo SVG ସୃଷ୍ଟି କରନ୍ତୁ

**Logo SVG ସୃଷ୍ଟି କରନ୍ତୁ** କ୍ଷମତା Theme Builder କୁ ସ୍ୱୟଂଚାଳିତ namespace-ନିରାପଦ ସ୍ୱଚ୍ଛତାକରଣ ସହିତ ଆପଣଙ୍କ WordPress ସାଇଟରେ ସିଧାସଳଖ custom ଲୋଗୋ SVG ତିଆରି ଏବଂ ଏମ୍ବେଡ୍ କରିବାକୁ ଦିଏ।

## ସାରାଂଶ

ଏହି କ୍ଷମତା ଆପଣଙ୍କ ସାଇଟର ବ୍ରାଣ୍ଡିଂ ଦିଗ ଏବଂ ଡିଜାଇନ ପସନ୍ଦ ଆଧାରରେ scalable vector graphics (SVG) ଲୋଗୋ ସୃଷ୍ଟି କରେ। ସୃଷ୍ଟି ହୋଇଥିବା SVG ଗୁଡ଼ିକୁ WordPress ରେ ବ୍ୟବହାର ପାଇଁ ନିରାପଦ କରିବାକୁ ସ୍ୱୟଂଚାଳିତ ଭାବେ ସ୍ୱଚ୍ଛ କରାଯାଏ, ଏବଂ ଦୃଶ୍ୟ ଅଖଣ୍ଡତା ରହିଥାଏ।

## ପାରାମିଟର

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `site_name` | string | ହଁ | ଯେଉଁ ସାଇଟ ପାଇଁ ଲୋଗୋ ସୃଷ୍ଟି କରିବାକୁ ହେବ, ସେହି ସାଇଟର ନାମ |
| `style` | string | ହଁ | ଡିଜାଇନ ଶୈଳୀ (ଯେପରି, "modern", "classic", "minimalist", "playful") |
| `colors` | array | ନା | ଲୋଗୋରେ ବ୍ୟବହାର କରିବାକୁ hex ରଙ୍ଗ କୋଡ୍‌ର ଆରେ (ଯେପରି, `["#678233", "#ffffff"]`) |
| `width` | number | ନା | ପିକ୍ସେଲରେ SVG ପ୍ରସ୍ଥ (ଡିଫଲ୍ଟ: 200) |
| `height` | number | ନା | ପିକ୍ସେଲରେ SVG ଉଚ୍ଚତା (ଡିଫଲ୍ଟ: 200) |
| `include_text` | boolean | ନା | ଲୋଗୋରେ ସାଇଟ ନାମକୁ ଟେକ୍ସଟ୍ ଭାବେ ରଖିବା କି ନାହିଁ (ଡିଫଲ୍ଟ: true) |

## ଆଉଟପୁଟ ଫର୍ମାଟ୍

ଏହି କ୍ଷମତା ନିମ୍ନଲିଖିତ ଗଠନ ସହିତ ଏକ SVG ଷ୍ଟ୍ରିଙ୍ଗ୍ ଫେରାଏ:

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## SVG ସ୍ୱଚ୍ଛତାକରଣ ଆଚରଣ

ସୃଷ୍ଟି ହୋଇଥିବା SVG ଗୁଡ଼ିକ ସ୍ୱୟଂଚାଳିତ namespace-ନିରାପଦ ସ୍ୱଚ୍ଛତାକରଣ ମଧ୍ୟରୁ ଯାଆନ୍ତି, ଯାହାଦ୍ୱାରା:

- **ଅନିରାପଦ ଆଟ୍ରିବ୍ୟୁଟ୍ ହଟାଏ** — event handler, script, ଏବଂ ସମ୍ଭାବ୍ୟ ଜୋଖିମପୂର୍ଣ୍ଣ ଆଟ୍ରିବ୍ୟୁଟ୍ କାଢ଼ିଦିଏ
- **namespace ସୁରକ୍ଷିତ ରଖେ** — ଠିକ୍ ରେଣ୍ଡରିଂ ପାଇଁ SVG namespace (xmlns, xlink) ରଖେ
- **ଗଠନ ଯାଞ୍ଚ କରେ** — SVG W3C ମାନକ ଅନୁସରଣ କରୁଛି ବୋଲି ନିଶ୍ଚିତ କରେ
- **entity ଏନକୋଡ୍ କରେ** — ଟେକ୍ସଟ୍ ବିଷୟବସ୍ତୁରେ ବିଶେଷ ଅକ୍ଷରଗୁଡ଼ିକୁ ଠିକ୍ ଭାବେ escape କରେ
- **ବାହ୍ୟ reference ହଟାଏ** — ବାହ୍ୟ stylesheet ଏବଂ ଇମେଜ୍ reference କାଢ଼ିଦିଏ

ଏହା ନିଶ୍ଚିତ କରେ ଯେ ଅତିରିକ୍ତ ସ୍ୱଚ୍ଛତାକରଣ ଆବଶ୍ୟକ ନହୋଇ SVG କୁ ସିଧାସଳଖ WordPress ରେ ଏମ୍ବେଡ୍ କରିବା ନିରାପଦ।

## ବ୍ୟବହାର ଉଦାହରଣ

**ପ୍ରମ୍ପ୍ଟ:**
```
ମୋର "CloudSync" ନାମକ tech startup ପାଇଁ ନୀଳ ଏବଂ ଧଳା ରଙ୍ଗ ବ୍ୟବହାର କରି ଏକ modern ଲୋଗୋ ସୃଷ୍ଟି କରନ୍ତୁ।
```

**ଫଳାଫଳ:**
ଏହି କ୍ଷମତା ଏକ SVG ଲୋଗୋ ସୃଷ୍ଟି କରେ ଯାହା:
- "CloudSync" ସାଇଟ ନାମକୁ ସାମିଲ କରେ
- ନିର୍ଦ୍ଦିଷ୍ଟ ନୀଳ ଏବଂ ଧଳା ରଙ୍ଗ ଯୋଜନା ବ୍ୟବହାର କରେ
- modern ଡିଜାଇନ ସିଦ୍ଧାନ୍ତ ଅନୁସରଣ କରେ
- ସ୍ୱୟଂଚାଳିତ ଭାବେ ସ୍ୱଚ୍ଛ କରାଯାଇଛି ଏବଂ ବ୍ୟବହାର ପାଇଁ ପ୍ରସ୍ତୁତ

## Theme Builder ସହିତ ଏକୀକରଣ

Theme Builder ର design-direction ଚୟନ ବ୍ୟବହାର କରିବାବେଳେ, Generate Logo SVG କ୍ଷମତା:

1. ଆପଣଙ୍କ ଡିଜାଇନ ଦିଗ ଏବଂ ରଙ୍ଗ palette ବିଶ୍ଳେଷଣ କରେ
2. ଆପଣଙ୍କ ପସନ୍ଦ ସହିତ ମେଳ ଖାଉଥିବା custom SVG ଲୋଗୋ ସୃଷ୍ଟି କରେ
3. ଆପଣଙ୍କ ସାଇଟର header/branding ଅଞ୍ଚଳରେ ଲୋଗୋକୁ ସ୍ୱୟଂଚାଳିତ ଭାବେ ଏମ୍ବେଡ୍ କରେ
4. WordPress media ରେ SVG କୁ custom ଲୋଗୋ ଭାବେ ସଞ୍ଚୟ କରେ

## ଶ୍ରେଷ୍ଠ ପ୍ରଚଳନ

- **ସ୍ପଷ୍ଟ ଶୈଳୀ ପସନ୍ଦ ଦିଅନ୍ତୁ** — ଆପଣ ଚାହୁଁଥିବା ଡିଜାଇନ ଶୈଳୀ ବର୍ଣ୍ଣନା କରନ୍ତୁ (modern, classic, playful, ଇତ୍ୟାଦି)
- **ରଙ୍ଗ ନିର୍ଦ୍ଦିଷ୍ଟ କରନ୍ତୁ** — ସମନ୍ୱୟ ପାଇଁ ଆପଣଙ୍କ brand ରଙ୍ଗ ସାମିଲ କରନ୍ତୁ
- **ରେଣ୍ଡରିଂ ପରୀକ୍ଷା କରନ୍ତୁ** — ଭିନ୍ନ screen ଆକାରରେ ଲୋଗୋ ଠିକ୍ ଭାବେ ଦେଖାଯାଉଛି କି ଯାଞ୍ଚ କରନ୍ତୁ
- **ଆହୁରି customize କରନ୍ତୁ** — ଆକାର ଏବଂ ସ୍ଥାନ ବଦଳାଇବାକୁ WordPress ର ଲୋଗୋ customization ଟୁଲ୍ ବ୍ୟବହାର କରନ୍ତୁ

## ସୀମାବଦ୍ଧତା

- SVG ଲୋଗୋଗୁଡ଼ିକ static graphics ଭାବେ ସୃଷ୍ଟି ହୁଏ (animated ନୁହେଁ)
- ଅନେକ ଉପାଦାନ ଥିବା ଜଟିଳ ଲୋଗୋ ପାଇଁ manual refinement ଆବଶ୍ୟକ ହୋଇପାରେ
- custom font ସମର୍ଥିତ ନୁହେଁ; ଟେକ୍ସଟ୍ system font ବ୍ୟବହାର କରେ
- ବହୁତ ବଡ଼ କିମ୍ବା ବହୁତ ଛୋଟ dimensions ଗୁଣବତ୍ତାକୁ ପ୍ରଭାବିତ କରିପାରେ
