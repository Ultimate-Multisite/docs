# ମେନୁ ସୃଷ୍ଟି କରନ୍ତୁ

**ମେନୁ ସୃଷ୍ଟି କରନ୍ତୁ** କ୍ଷମତା WordPress ନେଭିଗେସନ ମେନୁ ସୃଷ୍ଟି କରେ, ଯେଉଁଥିରେ ପୃଷ୍ଠା ଶୀର୍ଷକରୁ ଅଲଗା ନେଭିଗେସନ ଲେବେଲ ପାଇଁ ସମର୍ଥନ ରହିଛି।

## ସାରାଂଶ

ଏହି କ୍ଷମତା ମାନକ ମେନୁ ସୃଷ୍ଟି କାର୍ଯ୍ୟକୁ `navigation_label` ପାରାମିଟର ଦେବାର ସୁବିଧା ସହିତ ବିସ୍ତାର କରେ। ଏଥିରେ ନେଭିଗେସନରେ ଦେଖାଯାଉଥିବା ଲେବେଲ ପୃଷ୍ଠା ଶୀର୍ଷକରୁ ଭିନ୍ନ ଥିବା ମେନୁ ସୃଷ୍ଟି କରିପାରିବେ, ଯାହା ସାଇଟ ଗଠନ ଓ ବ୍ୟବହାରକାରୀ ଅନୁଭବରେ ଅଧିକ ଲଚିଳାପଣ ଦେଇଥାଏ।

## ପାରାମିଟରଗୁଡ଼ିକ

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | string | ହଁ | ମେନୁ ନାମ, ଯେପରିକି `Primary Navigation` |
| `location` | string | ନା | ଏହି ମେନୁକୁ ନିର୍ଦ୍ଦିଷ୍ଟ କରିବା ପାଇଁ ଥିମ୍ ସ୍ଥାନ, ଯେପରିକି `primary` |
| `navigation_label` | string | ନା | ନେଭିଗେସନରେ ଦେଖାଇବାକୁ ଥିବା ଲେବେଲ (ପୃଷ୍ଠା ଶୀର୍ଷକରୁ ଭିନ୍ନ) |

## ଫେରତ ମୂଲ୍ୟ

```json
{
  "success": true,
  "menu_id": 7,
  "name": "Primary Navigation",
  "navigation_label": "Main Menu"
}
```

## ନେଭିଗେସନ ଲେବେଲ ବନାମ ପୃଷ୍ଠା ଶୀର୍ଷକ

`navigation_label` ପାରାମିଟର ଆପଣଙ୍କୁ ଆଭ୍ୟନ୍ତରୀଣ ମେନୁ ନାମକୁ ବ୍ୟବହାରକାରୀଙ୍କୁ ଦେଖାଯାଉଥିବା ଲେବେଲରୁ ଅଲଗା କରିବାକୁ ଦିଏ:

- **`name`** — WordPress ଦ୍ୱାରା ବ୍ୟବହୃତ ଆଭ୍ୟନ୍ତରୀଣ ମେନୁ ପରିଚୟକ (ଯେପରିକି, "ପ୍ରାଥମିକ ନେଭିଗେସନ")
- **`navigation_label`** — ସାଇଟ ଦର୍ଶକମାନଙ୍କୁ ନେଭିଗେସନରେ ଦେଖାଯାଉଥିବା ଲେବେଲ (ଯେପରିକି, "ମୁଖ୍ୟ ମେନୁ")

ଏହା ଉପଯୋଗୀ ଯେତେବେଳେ:
- ଆପଣଙ୍କ ଆଭ୍ୟନ୍ତରୀଣ ନାମକରଣ ଶୈଳୀ ବ୍ୟବହାରକାରୀଙ୍କୁ ଦେଖାଯାଉଥିବା ଲେବେଲରୁ ଭିନ୍ନ ଥାଏ
- ଆପଣ ପ୍ରଶାସନ ପ୍ୟାନେଲ ତୁଳନାରେ ନେଭିଗେସନରେ ଛୋଟ ଲେବେଲ ଚାହୁଁଛନ୍ତି
- ଭିନ୍ନ ଲେବେଲ ଲମ୍ବ ସହ ଅନେକ ଭାଷାକୁ ସମର୍ଥନ କରିବାକୁ ଆବଶ୍ୟକ
- ଆପଣ ନିର୍ଦ୍ଦିଷ୍ଟ ଅଞ୍ଚଳ କିମ୍ବା ବ୍ୟବହାରକାରୀ ଗୋଷ୍ଠୀ ପାଇଁ ମେନୁ ତିଆରି କରୁଛନ୍ତି

## ବ୍ୟବହାର ଉଦାହରଣ

### ଉଦାହରଣ ୧: ନେଭିଗେସନ ଲେବେଲ ସହ ସରଳ ମେନୁ

**ପ୍ରମ୍ପ୍ଟ:**
```
"ମୁଖ୍ୟ ନେଭିଗେସନ" ନାମରେ ଏକ ପ୍ରାଥମିକ ନେଭିଗେସନ ମେନୁ ସୃଷ୍ଟି କରନ୍ତୁ, ଯାହାର ନେଭିଗେସନ ଲେବେଲ "ମେନୁ" ହେବ।
```

**ଫଳାଫଳ:**
```json
{
  "success": true,
  "menu_id": 7,
  "name": "Main Navigation",
  "navigation_label": "Menu"
}
```

### ଉଦାହରଣ ୨: ନିର୍ଦ୍ଦିଷ୍ଟ ଥିମ୍ ସ୍ଥାନ ପାଇଁ ମେନୁ

**ପ୍ରମ୍ପ୍ଟ:**
```
"ଫୁଟର ଲିଙ୍କ" ନାମରେ ଏକ ଫୁଟର ମେନୁ ସୃଷ୍ଟି କରନ୍ତୁ, ଯାହାର ନେଭିଗେସନ ଲେବେଲ "ଦ୍ରୁତ ଲିଙ୍କ" ହେବ, ଏବଂ ଏହାକୁ ଫୁଟର ସ୍ଥାନକୁ ନିର୍ଦ୍ଦିଷ୍ଟ କରନ୍ତୁ।
```

**ଫଳାଫଳ:**
```json
{
  "success": true,
  "menu_id": 8,
  "name": "Footer Links",
  "navigation_label": "Quick Links",
  "location": "footer"
}
```

## Theme Builder ସହ ଏକୀକରଣ

Theme Builder ବ୍ୟବହାର କରିବାବେଳେ, ମେନୁ ସୃଷ୍ଟି କରିବା କ୍ଷମତା:

1. ଉପଲବ୍ଧ ଥିମ୍ ମେନୁ ସ୍ଥାନଗୁଡ଼ିକୁ ସ୍ୱୟଂଚାଳିତ ଭାବେ ଚିହ୍ନଟ କରେ
2. ଆପଣଙ୍କ ଡିଜାଇନ ପାଇଁ ଉଚିତ ନେଭିଗେସନ ଲେବେଲ ସହ ମେନୁ ସୃଷ୍ଟି କରେ
3. ମେନୁଗୁଡ଼ିକୁ ସଠିକ୍ ଥିମ୍ ସ୍ଥାନକୁ ନିର୍ଦ୍ଦିଷ୍ଟ କରେ
4. ସୃଷ୍ଟି ପରେ ମେନୁ ଆଇଟମ ଯୋଡ଼ିବାକୁ ସମର୍ଥନ କରେ

## ସମ୍ବନ୍ଧିତ କ୍ଷମତାଗୁଡ଼ିକ

- **`add_menu_item`** — ଥିବା ମେନୁରେ ଆଇଟମ ଯୋଡ଼ନ୍ତୁ
- **`update_menu`** — ମେନୁର ନାମ ବଦଳାନ୍ତୁ କିମ୍ବା ଏହାକୁ ଥିମ୍ ସ୍ଥାନକୁ ପୁନଃନିର୍ଦ୍ଦିଷ୍ଟ କରନ୍ତୁ
- **`delete_menu`** — ଆପଣଙ୍କ ସାଇଟରୁ ମେନୁ ହଟାନ୍ତୁ

## ଉତ୍ତମ ପ୍ରଚଳନଗୁଡ଼ିକ

- **ସ୍ପଷ୍ଟ ନେଭିଗେସନ ଲେବେଲ ବ୍ୟବହାର କରନ୍ତୁ** — ବ୍ୟବହାରକାରୀଙ୍କ ପାଇଁ ଲେବେଲଗୁଡ଼ିକୁ ସଂକ୍ଷିପ୍ତ ଓ ବର୍ଣ୍ଣନାତ୍ମକ ରଖନ୍ତୁ
- **ଥିମ୍ ସ୍ଥାନ ସହ ମେଳ କରନ୍ତୁ** — ଠିକ୍ ଭାବେ ଦେଖାଇବା ପାଇଁ ମେନୁଗୁଡ଼ିକୁ ସଠିକ୍ ଥିମ୍ ସ୍ଥାନକୁ ନିର୍ଦ୍ଦିଷ୍ଟ କରନ୍ତୁ
- **ମେନୁ ଗଠନ ଯୋଜନା କରନ୍ତୁ** — ଆଇଟମ ସୃଷ୍ଟି କରିବା ପୂର୍ବରୁ ଆପଣଙ୍କ ମେନୁ କ୍ରମବିନ୍ୟାସ ନିର୍ଦ୍ଧାରଣ କରନ୍ତୁ
- **ପ୍ରତିକ୍ରିୟାଶୀଳତା ପରୀକ୍ଷା କରନ୍ତୁ** — ମୋବାଇଲ ଉପକରଣରେ ମେନୁଗୁଡ଼ିକ ସଠିକ୍ ଭାବେ ଦେଖାଯାଉଛି କି ଯାଞ୍ଚ କରନ୍ତୁ
- **ଲେବେଲଗୁଡ଼ିକୁ ସ୍ଥାନୀୟକରଣ କରନ୍ତୁ** — ଭିନ୍ନ ଭାଷା ସଂସ୍କରଣ ପାଇଁ ଭିନ୍ନ ନେଭିଗେସନ ଲେବେଲ ବ୍ୟବହାର କରନ୍ତୁ

## ସୀମାବଦ୍ଧତା

- ନେଭିଗେସନ ଲେବେଲଗୁଡ଼ିକ କେବଳ ଦେଖାଇବା ପାଇଁ; WordPress ପରିଚୟ ପାଇଁ ଆଭ୍ୟନ୍ତରୀଣ `name` ଏପର୍ଯ୍ୟନ୍ତ ବ୍ୟବହୃତ ହୁଏ
- ଥିମ୍ ସମର୍ଥନ ଭିନ୍ନ ହୁଏ; ସମସ୍ତ ଥିମ୍ ସମସ୍ତ ମେନୁ ସ୍ଥାନକୁ ସମର୍ଥନ କରେନାହିଁ
- ମେନୁ ସୃଷ୍ଟି ପରେ ମେନୁ ଆଇଟମଗୁଡ଼ିକୁ ଅଲଗାଭାବେ ଯୋଡ଼ିବା ଆବଶ୍ୟକ
- ନେଭିଗେସନ ଲେବେଲ ବଦଳାଇବାକୁ ମେନୁ ଅଦ୍ୟତନ କରିବା ଆବଶ୍ୟକ
