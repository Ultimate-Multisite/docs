# ଆତିଥ୍ୟ ମେନୁ

**ଆତିଥ୍ୟ ମେନୁ** ବୈଶିଷ୍ଟ୍ୟ Theme Builder କୁ ଆପଣଙ୍କ WordPress ସାଇଟ୍‌ରେ ସଂରଚିତ ଖାଦ୍ୟ ଓ ପାନୀୟ ମେନୁ ପୃଷ୍ଠା ସିଧାସଳଖ ତିଆରି ଏବଂ ସଂଯୋଜନ କରିବାକୁ ଦେଇଥାଏ।

## ସାରାଂଶ {#overview}

Theme Builder ବର୍ତ୍ତମାନ ରେଷ୍ଟୁରାଣ୍ଟ, କାଫେ, ବାର୍ ଏବଂ କ୍ୟାଟରିଂ ସେବା ସହିତ ଆତିଥ୍ୟ ବ୍ୟବସାୟ ପାଇଁ ପେଶାଦାର, ସଂରଚିତ ମେନୁ ପୃଷ୍ଠା ତିଆରି କରିପାରେ। ଏହି ମେନୁଗୁଡ଼ିକ ଆପଣଙ୍କ ସାଇଟ୍‌ର ଡିଜାଇନ୍‌ରେ ପୂର୍ଣ୍ଣ ଭାବେ ଏକୀକୃତ ହୁଏ ଏବଂ ସହଜରେ ଅଦ୍ୟତନ ଓ ପରିଚାଳନା କରାଯାଇପାରେ।

## ସମର୍ଥିତ ଆତିଥ୍ୟ ପ୍ରକାର {#supported-hospitality-types}

- **ରେଷ୍ଟୁରାଣ୍ଟ** — ପୂର୍ଣ୍ଣ-ସେବା ଭୋଜନ ମେନୁ
- **କାଫେ** — କଫି ଓ ହାଲୁକା ଖାଦ୍ୟ ମେନୁ
- **ବାର୍ ଏବଂ ଲାଉଞ୍ଜ** — ପାନୀୟ ଓ ଆପେଟାଇଜର୍ ମେନୁ
- **ବେକାରୀ** — ପେଷ୍ଟ୍ରି ଓ ପାଉରୁଟି ମେନୁ
- **କ୍ୟାଟରିଂ ସେବା** — କାର୍ଯ୍ୟକ୍ରମ ମେନୁ ବିକଳ୍ପ
- **ଫୁଡ୍ ଟ୍ରକ୍** — ଚଳନଶୀଳ ଖାଦ୍ୟ ସେବା ମେନୁ
- **ବ୍ରୁଅରୀ ଏବଂ ୱାଇନରୀ** — ବିବରଣୀ ସହିତ ପାନୀୟ ମେନୁ

## ମେନୁ ସଂରଚନା {#menu-structure}

### ମେନୁ ବର୍ଗ {#menu-categories}

ମେନୁଗୁଡ଼ିକୁ ବର୍ଗ ଅନୁସାରେ ସଜାଯାଏ:

- **ଆପେଟାଇଜର୍** — ଆରମ୍ଭିକ ଖାଦ୍ୟ ଓ ଛୋଟ ପ୍ଲେଟ୍
- **ମୁଖ୍ୟ ବ୍ୟଞ୍ଜନ** — ପ୍ରଧାନ ଭୋଜନ
- **ସାଇଡ୍** — ସହାୟକ ଖାଦ୍ୟ ଓ ସବ୍ଜି
- **ମିଠା** — ମିଠା ଖାଦ୍ୟ
- **ପାନୀୟ** — ପାନୀୟ ଦ୍ରବ୍ୟ (ମଦ୍ୟଯୁକ୍ତ ଓ ମଦ୍ୟବିହୀନ)
- **ବିଶେଷ** — ଦୈନିକ କିମ୍ବା ଋତୁକାଳୀନ ବିଶେଷ ଖାଦ୍ୟ

### ମେନୁ ଆଇଟମ୍ ଫର୍ମାଟ୍ {#menu-item-format}

ପ୍ରତ୍ୟେକ ମେନୁ ଆଇଟମ୍‌ରେ ରହେ:

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### ଆଇଟମ୍ ଫିଲ୍ଡ {#item-fields}

| ଫିଲ୍ଡ | ପ୍ରକାର | ବିବରଣୀ |
|-------|------|-------------|
| `name` | string | ବ୍ୟଞ୍ଜନ କିମ୍ବା ପାନୀୟର ନାମ |
| `description` | string | ଆଇଟମ୍‌ର ବିସ୍ତୃତ ବିବରଣୀ |
| `price` | string | ମୂଲ୍ୟ (ମୁଦ୍ରା ସହିତ ଫର୍ମାଟ୍ କରାଯାଇଥିବା) |
| `dietary_info` | array | ଖାଦ୍ୟ ସମ୍ବନ୍ଧୀୟ ଗୁଣ (ଭେଗାନ୍, ଗ୍ଲୁଟେନ୍-ମୁକ୍ତ, ଇତ୍ୟାଦି) |
| `allergens` | array | ସାଧାରଣ ଆଲର୍ଜେନ୍ (ନଟ୍ସ, ଶେଲଫିଶ୍, ଇତ୍ୟାଦି) |
| `availability` | string | କେବେ ଉପଲବ୍ଧ (ଦୈନିକ, ଋତୁକାଳୀନ, ଇତ୍ୟାଦି) |

## ଆତିଥ୍ୟ ମେନୁ ତିଆରି କରିବା {#creating-hospitality-menus}

### ପଦକ୍ଷେପ ୧: ମେନୁ ସୂଚନା ଦିଅନ୍ତୁ {#step-1-provide-menu-information}

ଆପଣଙ୍କ ମେନୁ ବିଷୟରେ Theme Builder କୁ କୁହନ୍ତୁ:

```
ମୋ ଇଟାଲିଆନ୍ ରେଷ୍ଟୁରାଣ୍ଟ ପାଇଁ ଆପେଟାଇଜର୍,
ପାସ୍ତା ବ୍ୟଞ୍ଜନ, ମୁଖ୍ୟ ବ୍ୟଞ୍ଜନ ଏବଂ ମିଠା ସହିତ ଏକ ରେଷ୍ଟୁରାଣ୍ଟ ମେନୁ ତିଆରି କରନ୍ତୁ। ମୂଲ୍ୟ ଓ ବିବରଣୀ ଯୋଡ଼ନ୍ତୁ।
```

### ପଦକ୍ଷେପ ୨: Theme Builder ମେନୁ ତିଆରି କରେ {#step-2-theme-builder-generates-menu}

Theme Builder:

1. ସଂରଚିତ ମେନୁ ପୃଷ୍ଠା ତିଆରି କରେ
2. ଏହାକୁ ଆପଣଙ୍କ ସାଇଟ୍‌ର ଥିମ୍ ସହିତ ମେଳ ହେବା ପରି ଡିଜାଇନ୍ କରେ
3. ଆଇଟମ୍‌ଗୁଡ଼ିକୁ ବର୍ଗରେ ସଜାଏ
4. ମୂଲ୍ୟ ଓ ବିବରଣୀ ଫର୍ମାଟ୍ କରେ
5. ଖାଦ୍ୟ ସମ୍ବନ୍ଧୀୟ ଓ ଆଲର୍ଜେନ୍ ସୂଚନା ଯୋଡ଼େ

### ପଦକ୍ଷେପ ୩: ସମୀକ୍ଷା ଏବଂ ଅନୁକୂଳନ କରନ୍ତୁ {#step-3-review-and-customize}

ଆପଣ ପାରିବେ:

1. ମେନୁ ଆଇଟମ୍ ଓ ମୂଲ୍ୟ ସମ୍ପାଦନ କରିବା
2. ବର୍ଗ ଯୋଡ଼ିବା କିମ୍ବା ହଟାଇବା
3. ବର୍ଗ ଭିତରେ ଆଇଟମ୍‌ଗୁଡ଼ିକୁ ପୁନଃକ୍ରମିତ କରିବା
4. ବିବରଣୀ ଓ ଖାଦ୍ୟ ସୂଚନା ଅଦ୍ୟତନ କରିବା
5. ଶୈଳୀ ଓ ଲେଆଉଟ୍ ସମଯୋଜନ କରିବା

## ମେନୁ ପ୍ରଦର୍ଶନ ବିକଳ୍ପ {#menu-display-options}

### ପୂର୍ଣ୍ଣ ମେନୁ ପୃଷ୍ଠା {#full-menu-page}

ଆପଣଙ୍କ ସମ୍ପୂର୍ଣ୍ଣ ମେନୁ ଦେଖାଉଥିବା ଏକ ନିର୍ଦ୍ଦିଷ୍ଟ ପୃଷ୍ଠା:

- ବର୍ଗ ଅନୁସାରେ ସଜାଯାଇଥିବା
- ସନ୍ଧାନଯୋଗ୍ୟ ଏବଂ ଫିଲ୍ଟର୍ କରିପାରିବା
- ମୁଦ୍ରଣ-ଅନୁକୂଳ ଲେଆଉଟ୍
- ମୋବାଇଲ୍-ଅନୁକୂଳ ଡିଜାଇନ୍

### ମେନୁ ୱିଜେଟ୍ {#menu-widget}

ଅନ୍ୟ ପୃଷ୍ଠାରେ ମେନୁ ଅଂଶ ସଂଯୋଜନ କରନ୍ତୁ:

- ମୂଳପୃଷ୍ଠାରେ ବିଶେଷ ଆଇଟମ୍
- ସାଇଡ୍‌ବାର୍‌ରେ ଦୈନିକ ବିଶେଷ
- ବାର୍ ପୃଷ୍ଠାରେ ପାନୀୟ ମେନୁ
- ଫୁଟର୍‌ରେ ମିଠା ପ୍ରଦର୍ଶନ

### ମେନୁ PDF {#menu-pdf}

ଡାଉନଲୋଡ୍ କରିପାରିବା PDF ମେନୁ ତିଆରି କରନ୍ତୁ:

- ପେଶାଦାର ଫର୍ମାଟିଂ
- ମୁଦ୍ରଣ-ପ୍ରସ୍ତୁତ ଗୁଣବତ୍ତା
- ଛବି ଓ ବିବରଣୀ ସମେତ
- ସେୟାର୍ ଏବଂ ଇମେଲ୍ କରିବାକୁ ସହଜ

## ଖାଦ୍ୟ ସମ୍ବନ୍ଧୀୟ ଓ ଆଲର୍ଜେନ୍ ସୂଚନା {#dietary-and-allergen-information}

### ଖାଦ୍ୟ ସମ୍ବନ୍ଧୀୟ ଗୁଣ {#dietary-attributes}

ଖାଦ୍ୟ ସୂଚନା ସହିତ ଆଇଟମ୍‌ଗୁଡ଼ିକୁ ଚିହ୍ନିତ କରନ୍ତୁ:

- **ଭେଗାନ୍** — କୌଣସି ପଶୁଜାତ ଉତ୍ପାଦ ନାହିଁ
- **ଶାକାହାରୀ** — ମାଂସ ନାହିଁ
- **ଗ୍ଲୁଟେନ୍-ମୁକ୍ତ** — ସିଲିଆକ୍ ରୋଗ ପାଇଁ ସୁରକ୍ଷିତ
- **ଦୁଗ୍ଧ-ମୁକ୍ତ** — ଦୁଗ୍ଧ ଉତ୍ପାଦ ନାହିଁ
- **ନଟ୍-ମୁକ୍ତ** — ଗଛ ନଟ୍ କିମ୍ବା ବାଦାମ ନାହିଁ
- **କମ୍-କାର୍ବ** — କମାଯାଇଥିବା କାର୍ବୋହାଇଡ୍ରେଟ୍
- **ଅଧିକ-ପ୍ରୋଟିନ୍** — ପ୍ରୋଟିନ୍-ସମୃଦ୍ଧ

### ଆଲର୍ଜେନ୍ ସତର୍କତା {#allergen-warnings}

ସାଧାରଣ ଆଲର୍ଜେନ୍ ଯୋଡ଼ନ୍ତୁ:

- **ନଟ୍ସ** — ଗଛ ନଟ୍ ଏବଂ ବାଦାମ
- **ଶେଲଫିଶ୍** — କ୍ରସ୍ଟେସିଆନ୍ ଏବଂ ମଲସ୍କ
- **ମାଛ** — ସମସ୍ତ ମାଛ ପ୍ରଜାତି
- **ଦୁଗ୍ଧ** — ଦୁଗ୍ଧ ଏବଂ ଦୁଗ୍ଧ ଉତ୍ପାଦ
- **ଅଣ୍ଡା** — କୁକୁଡ଼ା ଅଣ୍ଡା
- **ସୋୟା** — ସୋୟା ଉତ୍ପାଦ
- **ଗ୍ଲୁଟେନ୍** — ଗହମ ଏବଂ ଗ୍ଲୁଟେନ୍-ଯୁକ୍ତ ଶସ୍ୟ
- **ତିଳ** — ତିଳ ମଞ୍ଜି ଏବଂ ତେଲ

## ମେନୁ ପରିଚାଳନା {#menu-management}

### ମୂଲ୍ୟ ଅଦ୍ୟତନ କରିବା {#updating-prices}

ସହଜରେ ମେନୁ ମୂଲ୍ୟ ଅଦ୍ୟତନ କରନ୍ତୁ:

1. ମେନୁ ପୃଷ୍ଠାକୁ ଯାଆନ୍ତୁ
2. "ମେନୁ ସମ୍ପାଦନ କରନ୍ତୁ" କ୍ଲିକ୍ କରନ୍ତୁ
3. ଆଇଟମ୍‌ଗୁଡ଼ିକର ମୂଲ୍ୟ ଅଦ୍ୟତନ କରନ୍ତୁ
4. ପରିବର୍ତ୍ତନ ସେଭ୍ କରନ୍ତୁ
5. ପରିବର୍ତ୍ତନ ଆପଣଙ୍କ ସାଇଟ୍‌ରେ ତୁରନ୍ତ ଦେଖାଯାଏ

### ଋତୁକାଳୀନ ଆଇଟମ୍ ଯୋଡ଼ିବା {#adding-seasonal-items}

ଋତୁକାଳୀନ ମେନୁ ଭିନ୍ନତା ତିଆରି କରନ୍ତୁ:

1. ନୂତନ ମେନୁ ସଂସ୍କରଣ ତିଆରି କରନ୍ତୁ
2. ଋତୁକାଳୀନ ଆଇଟମ୍ ଯୋଡ଼ନ୍ତୁ
3. ଆଇଟମ୍‌ଗୁଡ଼ିକୁ "ଋତୁକାଳୀନ" ଭାବେ ଚିହ୍ନିତ କରନ୍ତୁ
4. ଉପଲବ୍ଧତା ତାରିଖ ସୂଚୀଭୁକ୍ତ କରନ୍ତୁ
5. ଋତୁ ସମୟରେ ସ୍ୱୟଂଚାଳିତ ଭାବେ ପ୍ରଦର୍ଶନ କରନ୍ତୁ

### ବିଶେଷ ଆଇଟମ୍ ପରିଚାଳନା {#managing-specials}

ଦୈନିକ କିମ୍ବା ସାପ୍ତାହିକ ବିଶେଷ ଦେଖାନ୍ତୁ:

1. "ବିଶେଷ" ବର୍ଗ ତିଆରି କରନ୍ତୁ
2. ଉପଲବ୍ଧତା ତାରିଖ ସହିତ ଆଇଟମ୍ ଯୋଡ଼ନ୍ତୁ
3. ମୂଳପୃଷ୍ଠାରେ ବିଶେଷଗୁଡ଼ିକୁ ଉଲ୍ଲେଖନୀୟ କରନ୍ତୁ
4. ଦୈନିକ କିମ୍ବା ସାପ୍ତାହିକ ଅଦ୍ୟତନ କରନ୍ତୁ
5. ପୁରୁଣା ବିଶେଷଗୁଡ଼ିକୁ ଆର୍କାଇଭ୍ କରନ୍ତୁ

## Theme Builder ସହିତ ଏକୀକରଣ {#integration-with-theme-builder}

ଆତିଥ୍ୟ ସାଇଟ୍‌ ପାଇଁ Theme Builder ବ୍ୟବହାର କରିବାବେଳେ:

1. **ସ୍ୱୟଂଚାଳିତ ମେନୁ ଚିହ୍ନଟ** — ଆପଣ ଖାଦ୍ୟ/ପାନୀୟ ବ୍ୟବସାୟ କି ନୁହେଁ ଚିହ୍ନଟ କରେ
2. **ମେନୁ ପୃଷ୍ଠା ସୃଷ୍ଟି** — ପେଶାଦାର ମେନୁ ପୃଷ୍ଠା ତିଆରି କରେ
3. **ଡିଜାଇନ୍ ମେଳ** — ମେନୁଗୁଡ଼ିକ ଆପଣଙ୍କ ସାଇଟ୍‌ର ଥିମ୍ ସହିତ ମେଳ ହୁଏ
4. **ମୋବାଇଲ୍ ଅନୁକୂଳନ** — ଫୋନ୍‌ରେ ମେନୁ ସୁନ୍ଦର ଭାବେ ଦେଖାଯାଏ
5. **SEO ଅନୁକୂଳନ** — ମେନୁଗୁଡ଼ିକ ସର୍ଚ୍ଚ-ଇଞ୍ଜିନ୍ ଅନୁକୂଳ

## ଶ୍ରେଷ୍ଠ ପ୍ରଚଳନ {#best-practices}

### ମେନୁ ଡିଜାଇନ୍ {#menu-design}

- **ସ୍ପଷ୍ଟ ସଂଗଠନ** — ଯୁକ୍ତିସଙ୍ଗତ ବର୍ଗ ସଂରଚନା
- **ପଢ଼ିବାଯୋଗ୍ୟ ବିବରଣୀ** — ରୁଚିକର ଏବଂ ସୂଚନାପୂର୍ଣ୍ଣ
- **ସୁସଙ୍ଗତ ମୂଲ୍ୟ** — ସ୍ପଷ୍ଟ ମୁଦ୍ରା ଏବଂ ଫର୍ମାଟିଂ
- **ପେଶାଦାର ଫଟୋ** — ଉଚ୍ଚ-ଗୁଣବତ୍ତା ଖାଦ୍ୟ ଛବି
- **ଖାଲି ସ୍ଥାନ** — ପୃଷ୍ଠାକୁ ଅତ୍ୟଧିକ ଭିଡ଼ କରନ୍ତୁ ନାହିଁ

### ବିଷୟବସ୍ତୁ {#content}

- **ସଠିକ୍ ବିବରଣୀ** — ଆଇଟମ୍‌ଗୁଡ଼ିକୁ ସଠିକ୍ ଭାବେ ବର୍ଣ୍ଣନା କରନ୍ତୁ
- **ବିଶେଷଗୁଡ଼ିକୁ ଉଲ୍ଲେଖନୀୟ କରନ୍ତୁ** — ବିଶେଷଗୁଡ଼ିକୁ ଅଲଗା ଦେଖାନ୍ତୁ
- **ଆଲର୍ଜେନ୍ ଯୋଡ଼ନ୍ତୁ** — ସବୁବେଳେ ଆଲର୍ଜେନ୍ ତାଲିକାଭୁକ୍ତ କରନ୍ତୁ
- **ନିୟମିତ ଅଦ୍ୟତନ କରନ୍ତୁ** — ମୂଲ୍ୟ ଓ ଆଇଟମ୍‌ଗୁଡ଼ିକୁ ସାମ୍ପ୍ରତିକ ରଖନ୍ତୁ
- **ରୁଚିକର ଭାଷା ବ୍ୟବହାର କରନ୍ତୁ** — ଆଇଟମ୍‌ଗୁଡ଼ିକୁ ସ୍ୱାଦିଷ୍ଟ ଲାଗିବା ପରି ଲେଖନ୍ତୁ

### ପ୍ରବେଶଗମ୍ୟତା {#accessibility}

- **ପଢ଼ିବାଯୋଗ୍ୟ ଫଣ୍ଟ** — ସ୍ପଷ୍ଟ, ସହଜରେ ପଢ଼ିହେବା ଟାଇପ୍‌ଫେସ୍ ବ୍ୟବହାର କରନ୍ତୁ
- **ପର୍ଯ୍ୟାପ୍ତ ବିପରୀତତା** — ପାଠ୍ୟ ପଢ଼ିବାଯୋଗ୍ୟ ଥିବା ନିଶ୍ଚିତ କରନ୍ତୁ
- **ଖାଦ୍ୟ ଲେବେଲ୍** — ଖାଦ୍ୟ ବିକଳ୍ପଗୁଡ଼ିକୁ ସ୍ପଷ୍ଟ ଭାବେ ଚିହ୍ନିତ କରନ୍ତୁ
- **ଆଲର୍ଜେନ୍ ସତର୍କତା** — ଆଲର୍ଜେନ୍‌ଗୁଡ଼ିକୁ ପ୍ରମୁଖ ଭାବେ ଦେଖାନ୍ତୁ
- **ମୋବାଇଲ୍-ଅନୁକୂଳ** — ସମସ୍ତ ଡିଭାଇସ୍‌ରେ ପରୀକ୍ଷା କରନ୍ତୁ
