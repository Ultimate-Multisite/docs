# ସାଧନ ଏବଂ ବିନ୍ୟାସ

Theme Builder ଆପଣଙ୍କ WordPress ସାଇଟ୍‌କୁ ବିଶ୍ଳେଷଣ, ଡିଜାଇନ୍ ଏବଂ ନିର୍ମାଣ କରିବାକୁ କିଛି ସାଧନ ବ୍ୟବହାର କରେ। ସଂସ୍କରଣ ୧.୧୬.୦ ରେ, **sd-ai-agent/site-scrape ଏବେ ସ୍ତର ୧ ସାଧନ**, ତେଣୁ ଏହା Theme Builder ରେ ପୂର୍ବନିର୍ଦ୍ଧାରିତ ଭାବେ ଉପଲବ୍ଧ। Superdav AI Agent v1.18.0 ବିଶ୍ୱସନୀୟ ବିକାଶକାରୀ କାର୍ଯ୍ୟପ୍ରବାହ ପାଇଁ ଅଲଗା ଭାବେ ବଣ୍ଟିତ Advanced ସହଚର ପ୍ୟାକେଜ୍‌କୁ ମଧ୍ୟ ଆଣିଛି; ସେହି ସାଧନଗୁଡ଼ିକ WordPress.org ମୂଳ ପ୍ୟାକେଜ୍‌ର ଅଂଶ ନୁହେଁ ଏବଂ ସେଗୁଡ଼ିକୁ ଅଲଗା ଭାବେ ସଂସ୍ଥାପନ ଓ ଅନୁମୋଦନ କରିବା ଆବଶ୍ୟକ।

## ଉପଲବ୍ଧ ସାଧନ

### ସ୍ତର ୧ ସାଧନ (ସବୁବେଳେ ଉପଲବ୍ଧ)

ସ୍ତର ୧ ସାଧନଗୁଡ଼ିକ କୌଣସି ଅତିରିକ୍ତ ବିନ୍ୟାସ ବିନା Theme Builder ରେ ପୂର୍ବନିର୍ଦ୍ଧାରିତ ଭାବେ ଉପଲବ୍ଧ।

#### sd-ai-agent/site-scrape

**ଉଦ୍ଦେଶ୍ୟ:** ଡିଜାଇନ୍ ପ୍ରେରଣା, ବିଷୟବସ୍ତୁ ଗଠନ ଏବଂ କାର୍ଯ୍ୟକ୍ଷମତା ବାହାର କରିବା ପାଇଁ ରହିଥିବା ୱେବସାଇଟ୍‌ଗୁଡ଼ିକୁ ବିଶ୍ଳେଷଣ କରେ।

**କ୍ଷମତା:**

- **ୱେବସାଇଟ୍ ବିଶ୍ଳେଷଣ** — ପ୍ରତିଦ୍ୱନ୍ଦ୍ୱୀ କିମ୍ବା ପ୍ରେରଣାମୂଳକ ୱେବସାଇଟ୍‌ଗୁଡ଼ିକୁ ସ୍କ୍ରେପ୍ ଏବଂ ବିଶ୍ଳେଷଣ କରେ
- **ଡିଜାଇନ୍ ନିଷ୍କର୍ଷଣ** — ରଙ୍ଗ, ଫଣ୍ଟ ଏବଂ ଲେଆଉଟ୍ ଧାରା ଚିହ୍ନଟ କରେ
- **ବିଷୟବସ୍ତୁ ଗଠନ** — ପୃଷ୍ଠା ସଂଗଠନ ଏବଂ କ୍ରମବିନ୍ୟାସ ବୁଝେ
- **ବୈଶିଷ୍ଟ୍ୟ ଚିହ୍ନଟ** — plugin ଏବଂ କାର୍ଯ୍ୟକ୍ଷମତା ଚିହ୍ନଟ କରେ
- **କାର୍ଯ୍ୟଦକ୍ଷତା ବିଶ୍ଳେଷଣ** — ପୃଷ୍ଠା ଗତି ଏବଂ ଅନୁକୂଳନ ଯାଞ୍ଚ କରେ
- **SEO ବିଶ୍ଳେଷଣ** — ମେଟା ଟ୍ୟାଗ୍ ଏବଂ ସଂରଚିତ ତଥ୍ୟ ସମୀକ୍ଷା କରେ

**ବ୍ୟବହାର:**

```
example.com ର ଡିଜାଇନ୍ ବିଶ୍ଳେଷଣ କରନ୍ତୁ ଏବଂ ଏହାକୁ ମୋ ସାଇଟ୍ ପାଇଁ ପ୍ରେରଣା ଭାବେ ବ୍ୟବହାର କରନ୍ତୁ।
```

**ଏହା କଣ ବାହାର କରେ:**

- ରଙ୍ଗ ପ୍ୟାଲେଟ୍ ଏବଂ ରଙ୍ଗ ବ୍ୟବହାର
- ଟାଇପୋଗ୍ରାଫି (ଫଣ୍ଟ ଏବଂ ଆକାର)
- ଲେଆଉଟ୍ ଗଠନ ଏବଂ ଅନ୍ତରାଳ
- ନେଭିଗେସନ୍ ଧାରା
- ବିଷୟବସ୍ତୁ ସଂଗଠନ
- ଛବି ଏବଂ ମିଡିଆ ବ୍ୟବହାର
- ଇଣ୍ଟରାକ୍ଟିଭ୍ ଉପାଦାନ
- ମୋବାଇଲ୍ ଅନୁକୂଳତା

**ସୀମାବଧତା:**

- ପାସୱାର୍ଡ-ସୁରକ୍ଷିତ ସାଇଟ୍‌ଗୁଡ଼ିକୁ ସ୍କ୍ରେପ୍ କରିପାରିବ ନାହିଁ
- robots.txt ଏବଂ ସାଇଟ୍ ନୀତିଗୁଡ଼ିକୁ ମାନେ
- ଗତିଶୀଳ ବିଷୟବସ୍ତୁ ଧରିପାରିନପାରେ
- JavaScript-ଭାରୀ ସାଇଟ୍‌ଗୁଡ଼ିକର ବିଶ୍ଳେଷଣ ସୀମିତ ହୋଇପାରେ
- ବଡ଼ ସାଇଟ୍‌ଗୁଡ଼ିକୁ ବିଶ୍ଳେଷଣ କରିବାରେ ଅଧିକ ସମୟ ଲାଗିପାରେ

### ସ୍ତର ୨ ସାଧନ (ଇଚ୍ଛାଧୀନ)

ସ୍ତର ୨ ସାଧନଗୁଡ଼ିକ Theme Builder ସେଟିଂସ୍‌ରେ ସ୍ପଷ୍ଟ ଭାବେ ସକ୍ରିୟ କରାଗଲେ ଉପଲବ୍ଧ ହୁଅନ୍ତି।

#### ଉନ୍ନତ ବିଶ୍ଳେଷଣ

ବିସ୍ତୃତ କାର୍ଯ୍ୟଦକ୍ଷତା ମାପଦଣ୍ଡ ପ୍ରଦାନ କରେ:

- ପୃଷ୍ଠା ଲୋଡ୍ ସମୟ
- Core Web Vitals
- SEO ସ୍କୋର
- ପ୍ରବେଶଯୋଗ୍ୟତା ସ୍କୋର
- ସର୍ବୋତ୍ତମ ପ୍ରଥା ସ୍କୋର

#### ବିଷୟବସ୍ତୁ ଅନୁକୂଳକ

ଏହା ପାଇଁ ବିଶ୍ଳେଷଣ କରି ସୁଧାର ପ୍ରସ୍ତାବ କରେ:

- ପଠନଯୋଗ୍ୟତା
- SEO ଅନୁକୂଳନ
- କୀୱାର୍ଡ ବ୍ୟବହାର
- ବିଷୟବସ୍ତୁ ଗଠନ
- କାର୍ଯ୍ୟ ଆହ୍ୱାନ ସ୍ଥାନନିର୍ଦ୍ଧାରଣ

### ସ୍ତର ୩ ସାଧନ (ପ୍ରିମିୟମ୍)

ସ୍ତର ୩ ସାଧନ ପାଇଁ ଅତିରିକ୍ତ ବିନ୍ୟାସ କିମ୍ବା ପ୍ରିମିୟମ୍ ବୈଶିଷ୍ଟ୍ୟ ଆବଶ୍ୟକ।

#### AI ବିଷୟବସ୍ତୁ ଉତ୍ପାଦକ

ଏହା ପାଇଁ ବିଷୟବସ୍ତୁ ସୃଷ୍ଟି କରେ:

- product ବର୍ଣ୍ଣନା
- ସେବା ପୃଷ୍ଠା
- ବ୍ଲଗ୍ ପୋଷ୍ଟ
- ମେଟା ବର୍ଣ୍ଣନା
- କାର୍ଯ୍ୟ ଆହ୍ୱାନ କପି

#### ଉନ୍ନତ ଡିଜାଇନ୍ ସାଧନ

ଉନ୍ନତ ଡିଜାଇନ୍ କ୍ଷମତା ପ୍ରଦାନ କରେ:

- କଷ୍ଟମ୍ CSS ସୃଷ୍ଟି
- ଆନିମେସନ୍ ସୃଷ୍ଟି
- ଇଣ୍ଟରାକ୍ଟିଭ୍ ଉପାଦାନ ଡିଜାଇନ୍
- ଉନ୍ନତ ରଙ୍ଗ ସିଦ୍ଧାନ୍ତ
- ଟାଇପୋଗ୍ରାଫି ଅନୁକୂଳନ

### ଉନ୍ନତ ସହଚର ସାଧନ (ଅଲଗା ପ୍ୟାକେଜ୍)

Advanced ସହଚର ପ୍ୟାକେଜ୍‌ଟି ମୂଳ Superdav AI Agent ରିଲିଜ୍‌ରୁ ଅଲଗା ଭାବେ ବଣ୍ଟନ ହୁଏ। ଏହା ବିଶ୍ୱସନୀୟ ବିକାଶକାରୀ ଏବଂ ସାଇଟ୍-ମାଲିକ ପରିବେଶ ପାଇଁ ଉଦ୍ଦିଷ୍ଟ, ଯେଉଁଠାରେ ପ୍ରଶାସକମାନେ agent କୁ ଉଚ୍ଚ-ପ୍ରଭାବୀ ସାଧନ ବ୍ୟବହାର କରିବାକୁ ସ୍ପଷ୍ଟ ଭାବେ ଅନୁମତି ଦିଅନ୍ତି।

Advanced ସହଚର କ୍ଷମତାରେ ଏଗୁଡ଼ିକ ଥାଇପାରେ:

- **ଫାଇଲ୍ ପ୍ରଣାଳୀ ସାଧନ** — ବିଶ୍ୱସନୀୟ ପରିବେଶରେ ଅନୁମୋଦିତ ଫାଇଲ୍‌ଗୁଡ଼ିକୁ ଯାଞ୍ଚ ଏବଂ ସଂଶୋଧନ କରେ
- **ଡାଟାବେସ୍ ସାଧନ** — ସ୍ପଷ୍ଟ ଅନୁମୋଦନ ଥିଲେ ସାଇଟ୍ ତଥ୍ୟକୁ query କିମ୍ବା ଅଦ୍ୟତନ କରେ
- **WP-CLI ସାଧନ** — WordPress ରକ୍ଷାଣାବେକ୍ଷଣ ଏବଂ ଯାଞ୍ଚ କମାଣ୍ଡ ଚଲାଏ
- **REST ଡିସ୍ପାଚର୍ ସାଧନ** — ନିୟନ୍ତ୍ରିତ ଡିସ୍ପାଚ୍ କାର୍ଯ୍ୟପ୍ରବାହ ମାଧ୍ୟମରେ ପଞ୍ଜିକୃତ REST endpoint କୁ କଲ୍ କରେ
- **Plugin ନିର୍ମାତା ସାଧନ** — plugin code ର ଆଧାର ତିଆରି କରି ପୁନରାବୃତ୍ତିରେ ସୁଧାର କରେ
- **Git snapshot ସାଧନ** — ଝୁମ୍ପପୂର୍ଣ୍ଣ ବିକାଶକାରୀ କାର୍ଯ୍ୟ ପୂର୍ବରୁ snapshot ସୃଷ୍ଟି କରେ
- **ବ୍ୟବହାରକାରୀ-ପରିଚାଳନା ସାଧନ** — ଅନୁମତି ଥିଲେ ବ୍ୟବହାରକାରୀ ରେକର୍ଡ ସମୀକ୍ଷା କିମ୍ବା ସମନ୍ୱୟ କରିବାରେ ସହାୟତା କରେ
- **ବେଞ୍ଚମାର୍କ ସାଧନ** — ବିକାଶ ସମୀକ୍ଷା ପାଇଁ କାର୍ଯ୍ୟଦକ୍ଷତା କିମ୍ବା କ୍ଷମତା ବେଞ୍ଚମାର୍କ ଚଲାଏ

ଏଗୁଡ଼ିକୁ ପ୍ରତ୍ୟେକ ସଂସ୍ଥାପନ ପାଇଁ ସାଧାରଣତଃ ଉପଲବ୍ଧ Theme Builder ସାଧନ ଭାବେ ଲେଖିବେ ନାହିଁ। Advanced ସହଚର ପ୍ୟାକେଜ୍ ସଂସ୍ଥାପିତ, ସକ୍ରିୟ, ଏବଂ ବିଶ୍ୱସନୀୟ ପ୍ରଶାସକଙ୍କ ପାଇଁ ସୀମିତ ଥିଲେ ମାତ୍ର ସେଗୁଡ଼ିକ ଉପଲବ୍ଧ।

## ସାଧନ ବିନ୍ୟାସ

### ସାଧନ ସକ୍ରିୟ କରିବା

Theme Builder ରେ ଅତିରିକ୍ତ ସାଧନ ସକ୍ରିୟ କରିବାକୁ:

1. **Gratis AI Agent → Settings** କୁ ଯାଆନ୍ତୁ
2. **Theme Builder → Tools** କୁ ଯାଆନ୍ତୁ
3. ଆବଶ୍ୟକତା ଅନୁସାରେ ସାଧନ ଅନ୍/ଅଫ୍ କରନ୍ତୁ
4. ସେଟିଂସ୍ ସଞ୍ଚୟ କରନ୍ତୁ

### ସାଧନ ଅନୁମତି

କିଛି ସାଧନ ପାଇଁ ଅନୁମତି ଆବଶ୍ୟକ:

- **ସାଇଟ୍ ସ୍କ୍ରେପିଂ** — ଇଣ୍ଟରନେଟ୍ ପ୍ରବେଶ ଆବଶ୍ୟକ
- **ବିଶ୍ଳେଷଣ** — Google Analytics ସମନ୍ୱୟ ଆବଶ୍ୟକ
- **ବିଷୟବସ୍ତୁ ସୃଷ୍ଟି** — API keys ଆବଶ୍ୟକ
- **ଉନ୍ନତ ବୈଶିଷ୍ଟ୍ୟ** — ପ୍ରିମିୟମ୍ subscription ଆବଶ୍ୟକ ହୋଇପାରେ
- **Advanced ସହଚର ସାଧନ** — ଅଲଗା ଭାବେ ବଣ୍ଟିତ Advanced ପ୍ୟାକେଜ୍ ଏବଂ ସ୍ପଷ୍ଟ ପ୍ରଶାସକ ବିଶ୍ୱାସ ଆବଶ୍ୟକ

### API Keys ଏବଂ ପ୍ରମାଣପତ୍ର

ଯେଉଁ ସାଧନ ପାଇଁ API keys ଆବଶ୍ୟକ, ସେଗୁଡ଼ିକୁ ବିନ୍ୟାସ କରନ୍ତୁ:

1. **Gratis AI Agent → Settings → API Keys** କୁ ଯାଆନ୍ତୁ
2. ପ୍ରତ୍ୟେକ ସାଧନ ପାଇଁ ପ୍ରମାଣପତ୍ର ପ୍ରବେଶ କରନ୍ତୁ
3. ସଂଯୋଗ ପରୀକ୍ଷା କରନ୍ତୁ
4. ସୁରକ୍ଷିତ ଭାବେ ସଞ୍ଚୟ କରନ୍ତୁ

### Advanced ସହଚର ପ୍ୟାକେଜ୍ ସଂସ୍ଥାପନ

Advanced ସହଚର ପ୍ୟାକେଜ୍ WordPress.org ମୂଳ ZIP ରୁ ଅଲଗା ଭାବେ ରିଲିଜ୍ ହୁଏ। ରିଲିଜ୍ ପାଇଁ କେବଳ ଆଧିକାରିକ ପ୍ରକଳ୍ପ ବଣ୍ଟନ ଚ୍ୟାନେଲ୍‌ରୁ ଏହାକୁ ସଂସ୍ଥାପନ କରନ୍ତୁ, ତାପରେ ପ୍ରବେଶକୁ ବିଶ୍ୱସନୀୟ ପ୍ରଶାସକଙ୍କ ପାଇଁ ସୀମିତ କରନ୍ତୁ। ପ୍ରୋଡକ୍ସନ୍ ସାଇଟ୍‌ରେ ଫାଇଲ୍ ପ୍ରଣାଳୀ, ଡାଟାବେସ୍, WP-CLI, REST ଡିସ୍ପାଚର୍, plugin builder, ବ୍ୟବହାରକାରୀ-ପରିଚାଳନା, କିମ୍ବା ବେଞ୍ଚମାର୍କ ସାଧନ ସକ୍ରିୟ କରିବା ପୂର୍ବରୁ ମାନବ ଅନୁମୋଦନ ଗେଟ୍ ଏବଂ git snapshot ଆଚରଣ ସମୀକ୍ଷା କରନ୍ତୁ।

## sd-ai-agent/site-scrape ବ୍ୟବହାର

### ମୂଳ ବ୍ୟବହାର

```
ଏବଂ ମୋ ସାଇଟ୍ ପାଇଁ ଡିଜାଇନ୍ ସୁଧାର ପ୍ରସ୍ତାବ କରନ୍ତୁ।
```

### ନିର୍ଦ୍ଦିଷ୍ଟ ବିଶ୍ଳେଷଣ

ନିର୍ଦ୍ଦିଷ୍ଟ ପ୍ରକାରର ବିଶ୍ଳେଷଣ ଅନୁରୋଧ କରନ୍ତୁ:

```
example.com ରୁ ରଙ୍ଗ ପ୍ୟାଲେଟ୍ ବାହାର କରନ୍ତୁ ଏବଂ ଏହାକୁ ପ୍ରେରଣା ଭାବେ ବ୍ୟବହାର କରନ୍ତୁ।
```

```
example.com ର ନେଭିଗେସନ୍ ଗଠନ ବିଶ୍ଳେଷଣ କରନ୍ତୁ ଏବଂ ସେହିପରି
ସଂଗଠନକୁ ମୋ ସାଇଟ୍‌ରେ ପ୍ରୟୋଗ କରନ୍ତୁ।
```

```
example.com ର ମୋବାଇଲ୍ ଅନୁକୂଳତା ଯାଞ୍ଚ କରନ୍ତୁ ଏବଂ ମୋ ସାଇଟ୍
ସେହିପରି ଅନୁକୂଳ ଥିବାକୁ ସୁନିଶ୍ଚିତ କରନ୍ତୁ।
```

### ତୁଳନାତ୍ମକ ବିଶ୍ଳେଷଣ

ଅନେକ ସାଇଟ୍‌କୁ ତୁଳନା କରନ୍ତୁ:

```
site1.com ଏବଂ site2.com ର ଡିଜାଇନ୍‌ଗୁଡ଼ିକୁ ତୁଳନା କରନ୍ତୁ ଏବଂ ଦୁହିଁର
ସର୍ବୋତ୍ତମ ଉପାଦାନ ମିଶାଇ ଏକ ଡିଜାଇନ୍ ତିଆରି କରନ୍ତୁ।
```

## ସାଧନ ସୀମାବଧତା ଏବଂ ବିଚାରବିମର୍ଶ

### ହାର ସୀମାବଦ୍ଧତା

- ସର୍ଭରର ଅତିରିକ୍ତ ଭାର ରୋକିବା ପାଇଁ ସ୍କ୍ରାପିଂରେ ହାର ସୀମା ରହିଛି
- ସେହି ଏକେ ସାଇଟକୁ ଅନେକ ଅନୁରୋଧ ଧୀର କରାଯାଇପାରେ
- ବଡ଼ ସାଇଟଗୁଡ଼ିକୁ ବିଶ୍ଳେଷଣ କରିବାକୁ ଅଧିକ ସମୟ ଲାଗିପାରେ

### ବିଷୟବସ୍ତୁର ସଠିକତା

- ଗତିଶୀଳ ବିଷୟବସ୍ତୁ ସମ୍ପୂର୍ଣ୍ଣ ଧରାପଡ଼ିନପାରେ
- JavaScript ଦ୍ୱାରା ଦେଖାଯାଉଥିବା ବିଷୟବସ୍ତୁ ଅସମ୍ପୂର୍ଣ୍ଣ ହୋଇପାରେ
- ତତ୍କାଳୀନ ତଥ୍ୟ ପ୍ରତିଫଳିତ ନହୋଇପାରେ
- କିଛି ବିଷୟବସ୍ତୁ ପେୱାଲ୍ ପଛରେ ଥାଇପାରେ

### ଆଇନଗତ ଏବଂ ନୈତିକ ବିଚାର

- robots.txt ଏବଂ ସାଇଟ ନୀତିକୁ ସମ୍ମାନ କରନ୍ତୁ
- ପୁନଃପ୍ରକାଶ ପାଇଁ କପିରାଇଟ୍ ଥିବା ବିଷୟବସ୍ତୁ ସ୍କ୍ରାପ୍ କରନ୍ତୁ ନାହିଁ
- ବିଶ୍ଳେଷଣକୁ ପ୍ରେରଣା ପାଇଁ ବ୍ୟବହାର କରନ୍ତୁ, ନକଲ ପାଇଁ ନୁହେଁ
- ନିଷ୍କାଷିତ ବିଷୟବସ୍ତୁ ବ୍ୟବହାର କରିବାର ଅଧିକାର ଆପଣଙ୍କ ପାଖରେ ଅଛି କି ନାହିଁ ଯାଞ୍ଚ କରନ୍ତୁ
- ବିଶ୍ଳେଷିତ ସାଇଟଗୁଡ଼ିକର ସେବା ସର୍ତ୍ତ ଅନୁସରଣ କରନ୍ତୁ

### କାର୍ଯ୍ୟଦକ୍ଷତାର ପ୍ରଭାବ

- ବଡ଼ ସାଇଟ ସ୍କ୍ରାପ୍ କରିବାକୁ ସମୟ ଲାଗିପାରେ
- ଏକାସାଥିରେ ଅନେକ ସ୍କ୍ରାପ୍ ଧୀର ହୋଇପାରେ
- ନେଟୱର୍କ ସଂଯୋଗ ଗତିକୁ ପ୍ରଭାବିତ କରେ
- ବଡ଼ ତଥ୍ୟସମୂହ ପାଇଁ ଅଧିକ ପ୍ରକ୍ରିୟାକରଣ ଆବଶ୍ୟକ ହୋଇପାରେ

## ଶ୍ରେଷ୍ଠ ପ୍ରଚଳନ

### ସାଇଟ ବିଶ୍ଳେଷଣ ବ୍ୟବହାର

1. **ଅନେକ ସାଇଟ ବିଶ୍ଳେଷଣ କରନ୍ତୁ** — ବିଭିନ୍ନ ଉତ୍ସରୁ ପ୍ରେରଣା ସଂଗ୍ରହ କରନ୍ତୁ
2. **ଗଠନ ଉପରେ ଧ୍ୟାନ ଦିଅନ୍ତୁ** — ଲେଆଉଟ୍ ଏବଂ ସଂଗଠନ ଶୈଳୀ ଶିଖନ୍ତୁ
3. **ରଙ୍ଗ ନିଷ୍କାଷଣ କରନ୍ତୁ** — ରଙ୍ଗ ପାଲେଟ୍‌କୁ ଆରମ୍ଭିକ ବିନ୍ଦୁ ଭାବେ ବ୍ୟବହାର କରନ୍ତୁ
4. **ଟାଇପୋଗ୍ରାଫି ଅଧ୍ୟୟନ କରନ୍ତୁ** — ଆପଣଙ୍କୁ ଭଲ ଲାଗୁଥିବା ଫଣ୍ଟ ସଂଯୋଗ ଚିହ୍ନଟ କରନ୍ତୁ
5. **ନାଭିଗେସନ୍ ସମୀକ୍ଷା କରନ୍ତୁ** — ମେନୁ ସଂଗଠନ ବୁଝନ୍ତୁ

### ନୈତିକ ସ୍କ୍ରାପିଂ

1. **robots.txt କୁ ସମ୍ମାନ କରନ୍ତୁ** — ସାଇଟ ନିର୍ଦ୍ଦେଶାବଳୀ ଅନୁସରଣ କରନ୍ତୁ
2. **ବିଷୟବସ୍ତୁ ନକଲ କରନ୍ତୁ ନାହିଁ** — ବିଶ୍ଳେଷଣକୁ କେବଳ ପ୍ରେରଣା ପାଇଁ ବ୍ୟବହାର କରନ୍ତୁ
3. **ଅଧିକାର ଯାଞ୍ଚ କରନ୍ତୁ** — ନିଷ୍କାଷିତ ବିଷୟବସ୍ତୁ ବ୍ୟବହାର କରିପାରିବେ କି ନାହିଁ ନିଶ୍ଚିତ କରନ୍ତୁ
4. **ଉତ୍ସ ଉଲ୍ଲେଖ କରନ୍ତୁ** — ପ୍ରେରଣା ଉତ୍ସକୁ ଶ୍ରେୟ ଦିଅନ୍ତୁ
5. **ସର୍ତ୍ତ ଅନୁସରଣ କରନ୍ତୁ** — ସାଇଟର ସେବା ସର୍ତ୍ତ ପାଳନ କରନ୍ତୁ

### ଟୁଲ୍‌ର ପ୍ରଭାବଶୀଳତା ବଢ଼ାଇବା

1. **ନିର୍ଦ୍ଦିଷ୍ଟ ହୁଅନ୍ତୁ** — ନିର୍ଦ୍ଦିଷ୍ଟ ପ୍ରକାରର ବିଶ୍ଳେଷଣ ଅନୁରୋଧ କରନ୍ତୁ
2. **ପ୍ରସଙ୍ଗ ଦିଅନ୍ତୁ** — ଆପଣଙ୍କ ସାଇଟର ଉଦ୍ଦେଶ୍ୟ ବ୍ୟାଖ୍ୟା କରନ୍ତୁ
3. **ଆଶା ସ୍ଥିର କରନ୍ତୁ** — ଆପଣଙ୍କ ଡିଜାଇନ୍ ଲକ୍ଷ୍ୟ ବର୍ଣ୍ଣନା କରନ୍ତୁ
4. **ପୁନରାବୃତ୍ତି କରନ୍ତୁ** — ଫଳାଫଳ ଆଧାରରେ ସଂଶୋଧନ କରନ୍ତୁ
5. **ଟୁଲ୍‌ଗୁଡ଼ିକ ମିଶାନ୍ତୁ** — ସମ୍ପୂର୍ଣ୍ଣ ବିଶ୍ଳେଷଣ ପାଇଁ ଅନେକ ଟୁଲ୍ ବ୍ୟବହାର କରନ୍ତୁ

## ସମସ୍ୟା ସମାଧାନ

### ଅସମ୍ପୂର୍ଣ୍ଣ ବିଶ୍ଳେଷଣ

- ସାଇଟରେ ଗତିଶୀଳ ବିଷୟବସ୍ତୁ ଥାଇପାରେ
- JavaScript ସମ୍ପୂର୍ଣ୍ଣ ଭାବେ ରେଣ୍ଡର୍ ହୋଇନପାରେ
- ବଡ଼ ସାଇଟର ସମୟ ସମାପ୍ତ ହୋଇପାରେ
- ପରିବର୍ତ୍ତେ ନିର୍ଦ୍ଦିଷ୍ଟ ପୃଷ୍ଠା ବିଶ୍ଳେଷଣ କରିଦେଖନ୍ତୁ
- ନିର୍ଦ୍ଦିଷ୍ଟ ବିଶ୍ଳେଷଣ ପ୍ରକାର ଅନୁରୋଧ କରନ୍ତୁ

### ଧୀର କାର୍ଯ୍ୟଦକ୍ଷତା

- ବଡ଼ ସାଇଟ ବିଶ୍ଳେଷଣ କରିବାକୁ ଅଧିକ ସମୟ ଲାଗେ
- ଏକାସାଥିରେ ଅନେକ ସ୍କ୍ରାପ୍ ଧୀର ହୁଏ
- ନେଟୱର୍କ ସଂଯୋଗ ଗତିକୁ ପ୍ରଭାବିତ କରେ
