# ଡୋମେନ ବିକ୍ରେତା

**Multisite Ultimate: ଡୋମେନ ବିକ୍ରେତା** ଆପଣଙ୍କ WordPress Multisite ନେଟୱର୍କକୁ ଏକ ଶକ୍ତିଶାଳୀ ଡୋମେନ ବିକ୍ରୟ ମଞ୍ଚରେ ପରିଣତ କରେ। ଗ୍ରାହକମାନଙ୍କୁ ସାଇନ୍-ଅପ୍ ସମୟରେ କିମ୍ବା ସେମାନଙ୍କ Dashboard ରୁ ସିଧାସଳଖ କଷ୍ଟମ୍ ଡୋମେନ ପଞ୍ଜିକରଣ କରିବାକୁ ଦିଅନ୍ତୁ, ଯେଉଁଥିରେ ତୁରନ୍ତ ଉପଲବ୍ଧତା ଯାଞ୍ଚ ଏବଂ ସ୍ପଷ୍ଟ ମୂଲ୍ୟ ରହିଥାଏ।

### ଡୋମେନ ବିକ୍ରେତା କାହିଁକି? {#why-domain-seller}

ଡୋମେନ ପଞ୍ଜିକରଣ ଯେକୌଣସି WaaS (Website as a Service) ମଞ୍ଚ ପାଇଁ ସବୁଠାରୁ ଅଧିକ ଚାହିଦା ଥିବା ଅପସେଲ୍‌ମାନଙ୍କ ମଧ୍ୟରୁ ଗୋଟିଏ। ଗ୍ରାହକମାନଙ୍କୁ ତୃତୀୟ-ପକ୍ଷ ରେଜିଷ୍ଟ୍ରାରଙ୍କ ପାଖକୁ ପଠାଇବା ପରିବର୍ତ୍ତେ, ଆପଣ ସିଧାସଳଖ ଡୋମେନ ଦେଇପାରିବେ — ଏହା ଆପଣଙ୍କ ନେଟୱର୍କରେ ଅଧିକ ଆୟ ରଖେ ଏବଂ ଗ୍ରାହକଙ୍କୁ ସହଜ ଅନୁଭବ ଦିଏ।

### ମୁଖ୍ୟ ବୈଶିଷ୍ଟ୍ୟ {#key-features}

#### ଅନେକ ଡୋମେନ ରେଜିଷ୍ଟ୍ରାର ସମର୍ଥନ {#multiple-domain-registrar-support}
* **OpenSRS** — ସମ୍ପୂର୍ଣ୍ଣ ପଞ୍ଜିକରଣ, ନବୀକରଣ ଏବଂ ପରିଚାଳନା
* **Namecheap** — WhoisGuard ଏବଂ ସ୍ୟାଣ୍ଡବକ୍ସ ପରୀକ୍ଷା ସହ ସମ୍ପୂର୍ଣ୍ଣ ଏକୀକରଣ
* **HostAfrica** — Domains Reseller API ମାଧ୍ୟମରେ ଡୋମେନ ସନ୍ଧାନ, ମୂଲ୍ୟ, ପଞ୍ଜିକରଣ, ହସ୍ତାନ୍ତର, DNS ଏବଂ ID ସୁରକ୍ଷା
* **Openprovider** — ରିସେଲର୍ ମୂଲ୍ୟ, ପଞ୍ଜିକରଣ, ନବୀକରଣ, ହସ୍ତାନ୍ତର, DNS, WHOIS ଗୋପନୀୟତା ଏବଂ TLD ସିଙ୍କ୍
* **Hostinger** — ସେୟାର୍ କରାଯାଇଥିବା Hostinger ଏକୀକରଣ ଟୋକେନ୍ ବ୍ୟବହାର କରି ଡୋମେନ ସନ୍ଧାନ ଏବଂ ପଞ୍ଜିକରଣ
* **ବିସ୍ତାରଯୋଗ୍ୟ ଗଠନ** — Integration Registry ପ୍ୟାଟର୍ନ ଉପରେ ନିର୍ମିତ, ତେଣୁ ଅଧିକ ପ୍ରଦାତା ଯୋଡ଼ିବା ସହଜ
* **ବିଲ୍ଟ-ଇନ୍ ସଂଯୋଗ ପରୀକ୍ଷା** — ଲାଇଭ୍ ହେବା ପୂର୍ବରୁ ପ୍ରଦାତା ପ୍ରମାଣପତ୍ର ଏବଂ ସଂଯୋଗକୁ ଯାଞ୍ଚ କରନ୍ତୁ

#### ନିର୍ଦ୍ଦେଶିତ ସେଟଅପ୍ ୱିଜାର୍ଡ {#guided-setup-wizard}
* **ଗୋଟିଏ କ୍ଲିକ୍‌ରେ ଆରମ୍ଭ** — ପ୍ରଦାତା ବିନ୍ୟାସ, TLD ଆମଦାନି ଏବଂ ଡୋମେନ ପଣ୍ୟ ସୃଷ୍ଟି ମାଧ୍ୟମରେ ଆପଣଙ୍କୁ ନେଇଯାଏ
* **ସ୍ୱୟଂଚାଳିତ TLD ସିଙ୍କ୍** — ଆପଣଙ୍କ ସଂଯୁକ୍ତ ପ୍ରଦାତାମାନଙ୍କୁଠାରୁ ସମସ୍ତ ଉପଲବ୍ଧ TLD ଏବଂ ହୋଲସେଲ୍ ମୂଲ୍ୟ ଆମଦାନି କରେ
* **ଡିଫଲ୍ଟ ପଣ୍ୟ ସୃଷ୍ଟି** — ଉପଯୁକ୍ତ ଡିଫଲ୍ଟ ମାର୍କଅପ୍ ସହ ବ୍ୟବହାର-ପ୍ରସ୍ତୁତ ଡୋମେନ ପଣ୍ୟ ତିଆରି କରେ

#### ଲଚିଳା ଡୋମେନ ମୂଲ୍ୟ {#flexible-domain-pricing}
* **ତିନୋଟି ମାର୍କଅପ୍ ମୋଡ୍** — ପ୍ରତିଶତ ମାର୍କଅପ୍, ନିର୍ଦ୍ଦିଷ୍ଟ ମାର୍କଅପ୍, କିମ୍ବା ପ୍ରତି TLD ପାଇଁ ନିର୍ଦ୍ଦିଷ୍ଟ ମୂଲ୍ୟ
* **TLD-ନିର୍ଦ୍ଦିଷ୍ଟ ପଣ୍ୟ** — ଭିନ୍ନ TLD ଗୋଷ୍ଠୀ (.com/.net, .io, ଦେଶ କୋଡ୍ ଇତ୍ୟାଦି) ପାଇଁ ଅଲଗା ଡୋମେନ ପଣ୍ୟ ତିଆରି କରନ୍ତୁ
* **ପ୍ରାରମ୍ଭିକ ମୂଲ୍ୟ** — ନୂତନ ପଞ୍ଜିକରଣ ଆକର୍ଷିତ କରିବାକୁ ପ୍ରଥମ ବର୍ଷ ପାଇଁ ରିଆୟତି ମୂଲ୍ୟ ଦିଅନ୍ତୁ
* **ତୁରନ୍ତ ମୂଲ୍ୟ** — ବୁଦ୍ଧିମାନ କ୍ୟାସିଂ ସହ ପ୍ରଦାତାମାନଙ୍କୁଠାରୁ ଲାଇଭ୍ ହୋଲସେଲ୍ ମୂଲ୍ୟ
* **କୁପନ୍ ସମର୍ଥନ** — Multisite Ultimate ଡିସ୍କାଉଣ୍ଟ କୋଡ୍‌ଗୁଡ଼ିକ ସହ ପୂର୍ଣ୍ଣ ସୁସଙ୍ଗତତା

#### WHOIS ଗୋପନୀୟତା ସୁରକ୍ଷା {#whois-privacy-protection}
* **ପ୍ରତି-ପଣ୍ୟ ବିନ୍ୟାସ** — WHOIS ଗୋପନୀୟତାକୁ ଅକ୍ଷମ, ସଦା ସମ୍ମିଳିତ, କିମ୍ବା ଗ୍ରାହକ ପସନ୍ଦ ଭାବେ ସେଟ୍ କରନ୍ତୁ
* **ଗ୍ରାହକ ଅପ୍ଟ-ଇନ୍** — ଗ୍ରାହକ ପସନ୍ଦ ଭାବେ ସେଟ୍ କଲେ, checkout ରେ ଥିବା ଚେକବକ୍ସ ଗ୍ରାହକମାନଙ୍କୁ ଗୋପନୀୟତା ସୁରକ୍ଷା ଯୋଡ଼ିବାକୁ ଦିଏ
* **ବିନ୍ୟାସଯୋଗ୍ୟ ମୂଲ୍ୟ** — ଗୋପନୀୟତା ଆଡ଼-ଅନ୍ ପାଇଁ ପ୍ରତି ବର୍ଷ ମୂଲ୍ୟ ସେଟ୍ କରନ୍ତୁ, କିମ୍ବା ଏହାକୁ ମାଗଣା ଦିଅନ୍ତୁ
* **ରେଜିଷ୍ଟ୍ରାର ଏକୀକରଣ** — ପଞ୍ଜିକରଣ ସମୟରେ WhoisGuard (Namecheap) କିମ୍ବା WHOIS ଗୋପନୀୟତା (OpenSRS) ସ୍ୱୟଂଚାଳିତ ଭାବେ ସକ୍ରିୟ କରେ

#### ଏକୀକୃତ checkout ଅନୁଭବ {#unified-checkout-experience}
* **ତିନୋଟି ଡୋମେନ ମୋଡ୍** — ଗ୍ରାହକମାନେ ସବ୍‌ଡୋମେନ (ମାଗଣା), ନୂତନ ଡୋମେନ ପଞ୍ଜିକରଣ, କିମ୍ବା ସେମାନଙ୍କ ପାଖରେ ପୂର୍ବରୁ ଥିବା ଡୋମେନ ମ୍ୟାପ୍ କରିବା ମଧ୍ୟରୁ ବାଛନ୍ତି
* **ତୁରନ୍ତ ଡୋମେନ ସନ୍ଧାନ** — ପୃଷ୍ଠା ପୁଣି ଲୋଡ୍ ନକରି AJAX-ଚାଳିତ ଉପଲବ୍ଧତା ଯାଞ୍ଚ
* **ସ୍ମାର୍ଟ ପରାମର୍ଶ** — ପସନ୍ଦର ଡୋମେନ ନିଆଯାଇଥିଲେ ବିକଳ୍ପ TLD ପରାମର୍ଶ
* **ସ୍ପଷ୍ଟ ମୂଲ୍ୟ** — ପଞ୍ଜିକରଣ ମୂଲ୍ୟ, ନବୀକରଣ ମୂଲ୍ୟ ଏବଂ ଇଚ୍ଛାଧୀନ WHOIS ଗୋପନୀୟତା ଶୁଳ୍କର ସ୍ପଷ୍ଟ ପ୍ରଦର୍ଶନ
* **ପଞ୍ଜିକର୍ତ୍ତା ସମ୍ପର୍କ କ୍ଷେତ୍ର** — checkout ସମୟରେ ଆବଶ୍ୟକ ପଞ୍ଜିକର୍ତ୍ତା ସୂଚନା (ନାମ, ଠିକଣା, ଫୋନ୍) ଇନ୍ଲାଇନ୍ ସଂଗ୍ରହ କରେ
* **ସ୍ୱୟଂ-ସୃଷ୍ଟ ସାଇଟ୍ URL** — ବାଛାଯାଇଥିବା ଡୋମେନରୁ ସାଇଟ୍ ଠିକଣା ସ୍ୱୟଂଚାଳିତ ଭାବେ ନିଆଯାଏ

#### ସ୍ୱୟଂ-ନବୀକରଣ ପ୍ରଣାଳୀ {#auto-renewal-system}
* **membership-ସଂଲଗ୍ନ ନବୀକରଣ** — ଗ୍ରାହକଙ୍କ membership ସକ୍ରିୟ ଥିବାବେଳେ ଡୋମେନଗୁଡ଼ିକ ସ୍ୱୟଂଚାଳିତ ଭାବେ ନବୀକରଣ ହୁଏ
* **ସ୍ମାର୍ଟ ସମୟସୂଚୀ** — ଡୋମେନ ମିଆଦ ସମାପ୍ତି ତାରିଖ ଆଧାରରେ ନବୀକରଣ କାମଗୁଡ଼ିକୁ କ୍ୟୁ କରାଯାଏ
* **ପୁନଃଚେଷ୍ଟା ତର୍କ** — ବିଫଳ ନବୀକରଣ ଚେଷ୍ଟା ପାଇଁ ବୁଦ୍ଧିମାନ ପୁନଃଚେଷ୍ଟା
* **ଇମେଲ୍ ସୂଚନା** — ଆସନ୍ତା ନବୀକରଣ, ସଫଳ ନବୀକରଣ, ବିଫଳତା ଏବଂ ମିଆଦ ସମାପ୍ତି ପାଇଁ ବିନ୍ୟାସଯୋଗ୍ୟ ସତର୍କତା

#### ଗ୍ରାହକ DNS ପରିଚାଳନା {#customer-dns-management}
* **ସ୍ୱୟଂ-ସେବା DNS** — ଗ୍ରାହକମାନେ ସେମାନଙ୍କ Dashboard ରୁ DNS ରେକର୍ଡ (A, AAAA, CNAME, MX, TXT) ଯୋଡ଼ି, ସମ୍ପାଦନା ଏବଂ ମିଟାଇପାରିବେ
* **ଡୋମେନ ସ୍ଥିତି** — ତୁରନ୍ତ ପଞ୍ଜିକରଣ ସ୍ଥିତି, ମିଆଦ ସମାପ୍ତି ତାରିଖ ଏବଂ ନେମସର୍ଭର ସୂଚନା
* **ସ୍ୱୟଂଚାଳିତ ଡୋମେନ ମ୍ୟାପିଂ** — ପଞ୍ଜିକୃତ କିମ୍ବା ବର୍ତ୍ତମାନ ଥିବା ଡୋମେନଗୁଡ଼ିକ ସ୍ୱୟଂଚାଳିତ ଭାବେ ଗ୍ରାହକଙ୍କ ସାଇଟ୍‌କୁ ମ୍ୟାପ୍ ହୁଏ

#### ଆଡ୍ମିନ୍ ଟୁଲ୍ {#admin-tools}
* **ମାନୁଆଲ୍ ଡୋମେନ ପଞ୍ଜିକରଣ** — ଆଡ୍ମିନ୍ ପ୍ୟାନେଲ୍‌ରୁ ଗ୍ରାହକମାନଙ୍କ ପକ୍ଷରୁ ଡୋମେନ ପଞ୍ଜିକରଣ କରନ୍ତୁ
* **ଡୋମେନ ନିରୀକ୍ଷଣ** — ଗୋଟିଏ ପୃଷ୍ଠାରୁ ସମସ୍ତ ଗ୍ରାହକ ଡୋମେନ, ସ୍ଥିତି ଏବଂ ମିଆଦ ସମାପ୍ତି ତାରିଖ ଦେଖନ୍ତୁ
* **ପ୍ରଦାତା-ନିର୍ଦ୍ଦିଷ୍ଟ ଲଗିଂ** — ପ୍ରତ୍ୟେକ ରେଜିଷ୍ଟ୍ରାର ପାଇଁ ଅଲଗା ଲଗ୍ ଚ୍ୟାନେଲ୍ (ଯେପରିକି, domain-seller-namecheap.log)
* **ପଞ୍ଜିକରଣ ବିବରଣୀ** — ମୂଳ ଡୋମେନ ସମ୍ପାଦନା ପୃଷ୍ଠାରେ ପଞ୍ଜିକର୍ତ୍ତା ସୂଚନା ଏବଂ ପଞ୍ଜିକରଣ ତଥ୍ୟ ଦେଖନ୍ତୁ

### ସମର୍ଥିତ ପ୍ରଦାତା {#supported-providers}

* **OpenSRS** — ପଞ୍ଜିକରଣ, ନବୀକରଣ, DNS ପରିଚାଳନା ଏବଂ WHOIS ଗୋପନୀୟତା ସହ ସମ୍ପୂର୍ଣ୍ଣ ଏକୀକରଣ
* **Namecheap** — ପଞ୍ଜିକରଣ, ନବୀକରଣ, ବ୍ୟାଚ୍ ଉପଲବ୍ଧତା ଯାଞ୍ଚ ଏବଂ ମାଗଣା WhoisGuard ସହ ସମ୍ପୂର୍ଣ୍ଣ ଏକୀକରଣ
* **HostAfrica** — ପଞ୍ଜିକରଣ, ନବୀକରଣ, ହସ୍ତାନ୍ତର, DNS ପରିଚାଳନା, EPP କୋଡ୍ ପ୍ରାପ୍ତି, ରେଜିଷ୍ଟ୍ରାର ଲକ୍ ଏବଂ ID ସୁରକ୍ଷା
* **Openprovider** — ପଞ୍ଜିକରଣ, ନବୀକରଣ, ହସ୍ତାନ୍ତର, DNS ପରିଚାଳନା, EPP କୋଡ୍ ପ୍ରାପ୍ତି, ରେଜିଷ୍ଟ୍ରାର ଲକ୍, WHOIS ଗୋପନୀୟତା ଏବଂ ରିସେଲର୍ TLD ସିଙ୍କ୍
* **Hostinger** — ପଞ୍ଜିକରଣ, ଡୋମେନ ଖୋଜ, ନେମସର୍ଭର ଅପଡେଟ୍, ରେଜିଷ୍ଟ୍ରାର ଲକ୍ ଏବଂ WHOIS ଗୋପନୀୟତା; ନବୀକରଣଗୁଡ଼ିକ Hostinger ସ୍ୱୟଂ-ନବୀକରଣ ଦ୍ୱାରା ପରିଚାଳିତ
* **GoDaddy** — ପଞ୍ଜିକରଣ ଏବଂ ନବୀକରଣ
* **ResellerClub** — ପଞ୍ଜିକରଣ, ନବୀକରଣ ଏବଂ DNS ପରିଚାଳନା
* **NameSilo** — ପଞ୍ଜିକରଣ ଏବଂ ନବୀକରଣ
* **Enom** — ପଞ୍ଜିକରଣ, ନବୀକରଣ ଏବଂ DNS ପରିଚାଳନା

### ପ୍ରାଯୁକ୍ତିକ ବିବରଣୀ {#technical-details}

* **ଆବଶ୍ୟକ:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **ଗଠନ:** Domain_Selling_Capability interface ସହ Integration Registry ପ୍ୟାଟର୍ନ
* **ଅଟୋଲୋଡିଂ:** PSR-4 କ୍ଲାସ୍ ମ୍ୟାପିଂ ସହ Jetpack ଅଟୋଲୋଡର୍
* **ମାନକ:** WordPress କୋଡିଂ ମାନକ, କଠୋର ଟାଇପିଂ, ପ୍ରସ୍ତୁତ ଷ୍ଟେଟମେଣ୍ଟ

## ସଂସ୍ଥାପନ {#installation}

1. **Multisite Ultimate v2.4.12+** ସଂସ୍ଥାପିତ ଏବଂ ନେଟୱର୍କ-ସକ୍ରିୟ ଅଛି କି ନିଶ୍ଚିତ କରନ୍ତୁ
2. ପ୍ଲଗିନ୍‌କୁ `/wp-content/plugins/ultimate-multisite-domain-seller/` କୁ ଅପଲୋଡ୍ କରନ୍ତୁ
3. WordPress ନେଟୱର୍କ ଆଡମିନ୍‌ର **ପ୍ଲଗିନ୍‌ଗୁଡ଼ିକ** ରୁ ପ୍ଲଗିନ୍‌କୁ ନେଟୱର୍କ-ସକ୍ରିୟ କରନ୍ତୁ
4. ପ୍ରଥମ ସକ୍ରିୟକରଣରେ **ସେଟଅପ୍ ୱିଜାର୍ଡ** ସ୍ୱୟଂଚାଳିତ ଭାବେ ଆରମ୍ଭ ହୁଏ — ଏହି ପଦକ୍ଷେପଗୁଡ଼ିକ ଅନୁସରଣ କରନ୍ତୁ:
   * ଆପଣଙ୍କ ଡୋମେନ୍ ରେଜିଷ୍ଟ୍ରାର୍ ସଂଯୋଗ କରନ୍ତୁ
   * ସଂଯୋଗ ପରୀକ୍ଷା କରନ୍ତୁ
   * ଉପଲବ୍ଧ TLD ଏବଂ ହୋଲସେଲ୍ ମୂଲ୍ୟ ଆମଦାନୀ କରନ୍ତୁ
   * ମୂଲ୍ୟ ମାର୍କଅପ୍ ସହିତ ଆପଣଙ୍କ ପ୍ରଥମ ଡୋମେନ୍ ଉତ୍ପାଦ ତିଆରି କରନ୍ତୁ
5. **Ultimate Multisite > ଚେକଆଉଟ୍ ଫର୍ମଗୁଡ଼ିକ** ରେ ଆପଣଙ୍କ ଚେକଆଉଟ୍ ଫର୍ମକୁ **ଡୋମେନ୍ ଚୟନ** ଫିଲ୍ଡ ଯୋଡ଼ନ୍ତୁ

## ବାରମ୍ବାର ପଚରାଯାଉଥିବା ପ୍ରଶ୍ନ {#frequently-asked-questions}

### Multisite Ultimate ର କେଉଁ ସଂସ୍କରଣ ଆବଶ୍ୟକ? {#what-version-of-multisite-ultimate-is-required}

ସଂସ୍କରଣ 2.4.12 କିମ୍ବା ତାଠାରୁ ଉଚ୍ଚ। ଏହି ଆଡଅନ୍ WP Ultimo v1.x କିମ୍ବା 2.4.12 ଠାରୁ କମ୍ Ultimate Multisite ସଂସ୍କରଣ ସହିତ ସୁସଙ୍ଗତ ନୁହେଁ।

### କେଉଁ ଡୋମେନ୍ ପ୍ରଦାତାମାନେ ସମର୍ଥିତ? {#which-domain-providers-are-supported}

ନଅଟି ରେଜିଷ୍ଟ୍ରାର୍ ସମର୍ଥିତ: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo**, ଏବଂ **Enom**। ପ୍ରଦାତା ଅନୁସାରେ କ୍ଷମତା ଭିନ୍ନ ହୁଏ, କାରଣ ରେଜିଷ୍ଟ୍ରାର୍ API ଭିନ୍ନ ଭିନ୍ନ କାର୍ଯ୍ୟ ଦେଖାଇଥାଏ; ପ୍ରତ୍ୟେକ ପ୍ରଦାତାର ସାରାଂଶ ପାଇଁ ସେଟଅପ୍ ଗାଇଡ୍ ଦେଖନ୍ତୁ। ଆଡଅନ୍‌ର Integration Registry ଆର୍କିଟେକ୍ଚର୍ କଷ୍ଟମ୍ ପ୍ରଦାତା ଯୋଡ଼ିବାକୁ ସରଳ କରେ — [ବିକାଶକାରୀ ଗାଇଡ୍](/developer/integration-guide/custom-registrar) ଦେଖନ୍ତୁ।

### ମୁଁ WHOIS ଗୋପନୀୟତା ସୁରକ୍ଷା ଦେଇପାରିବି କି? {#can-i-offer-whois-privacy-protection}

ହଁ। ପ୍ରତ୍ୟେକ ଡୋମେନ୍ ଉତ୍ପାଦରେ ତିନୋଟି ମୋଡ୍ ସହିତ ଏକ WHOIS ଗୋପନୀୟତା ସେଟିଂ ଅଛି: ଅକ୍ଷମ, ସବୁବେଳେ ଅନ୍ତର୍ଭୁକ୍ତ, କିମ୍ବା ଗ୍ରାହକ ଚୟନ। ଗ୍ରାହକ ଚୟନ ଭାବେ ସେଟ୍ କଲେ, ଗ୍ରାହକମାନେ ଚେକଆଉଟ୍ ସମୟରେ ମୂଲ୍ୟ ସହିତ ଏକ ଚେକବକ୍ସ ଦେଖନ୍ତି (କିମ୍ବା Namecheap ର WhoisGuard ପାଇଁ "ମାଗଣା")। ରେଜିଷ୍ଟ୍ରେସନ୍ ସମୟରେ ଗୋପନୀୟତା ଫ୍ଲାଗ୍ ସ୍ୱୟଂଚାଳିତ ଭାବେ ରେଜିଷ୍ଟ୍ରାର୍‌କୁ ପଠାଯାଏ।

### ଭିନ୍ନ TLD ପାଇଁ ମୁଁ ଭିନ୍ନ ମୂଲ୍ୟ ଧାର୍ଯ୍ୟ କରିପାରିବି କି? {#can-i-set-different-pricing-for-different-tlds}

ହଁ। ଭିନ୍ନ TLD ଗୋଷ୍ଠୀ ପାଇଁ ଅଲଗା ଡୋମେନ୍ ଉତ୍ପାଦ ତିଆରି କରନ୍ତୁ। ପ୍ରତ୍ୟେକ ଉତ୍ପାଦର ନିଜସ୍ୱ ମାର୍କଅପ୍ ପ୍ରକାର (ପ୍ରତିଶତ, ନିର୍ଦ୍ଦିଷ୍ଟ ମାର୍କଅପ୍, କିମ୍ବା ନିର୍ଦ୍ଦିଷ୍ଟ ମୂଲ୍ୟ), ସମର୍ଥିତ TLD, ଏବଂ ପ୍ରାରମ୍ଭିକ ମୂଲ୍ୟ ରହିପାରେ।

### ଗ୍ରାହକମାନେ କେଉଁ ଚେକଆଉଟ୍ ବିକଳ୍ପ ଦେଖନ୍ତି? {#what-checkout-options-do-customers-see}

ଡୋମେନ୍ ଚୟନ ଫିଲ୍ଡ ଗ୍ରାହକମାନଙ୍କୁ ତିନୋଟି ଟ୍ୟାବ୍ ଦେଇଥାଏ: **ସବଡୋମେନ୍** (ଆପଣଙ୍କ ନେଟୱର୍କରେ ମାଗଣା ସବଡୋମେନ୍ ବ୍ୟବହାର କରନ୍ତୁ), **ନୂଆ ଡୋମେନ୍ ରେଜିଷ୍ଟର୍ କରନ୍ତୁ** (ନୂଆ ଡୋମେନ୍ ଖୋଜନ୍ତୁ ଏବଂ ରେଜିଷ୍ଟର୍ କରନ୍ତୁ), ଏବଂ **ଅବସ୍ଥିତ ଡୋମେନ୍** (ସେମାନେ ପୂର୍ବରୁ ମାଲିକ ଥିବା ଡୋମେନ୍ ମ୍ୟାପ୍ କରନ୍ତୁ)। ଆପଣ ସକ୍ରିୟ କରିଥିବା ମୋଡ୍‌ଗୁଡ଼ିକ ମାତ୍ର ଦେଖାଯିବ।

### ଏହା ସ୍ୱୟଂଚାଳିତ ନବୀକରଣ ସମର୍ଥନ କରେ କି? {#does-it-support-automatic-renewals}

ହଁ। ଡୋମେନ୍‌ଗୁଡ଼ିକୁ ସ୍ୱୟଂଚାଳିତ ନବୀକରଣ ପାଇଁ ସେଟ୍ କରାଯାଇପାରେ ଏବଂ ସେଗୁଡ଼ିକ ଗ୍ରାହକଙ୍କ ସଦସ୍ୟତା ସ୍ଥିତି ସହିତ ଯୋଡ଼ା ଥାଏ। ନବୀକରଣ ଦେୟ ଗ୍ରାହକଙ୍କ ଅବସ୍ଥିତ ପେମେଣ୍ଟ ଗେଟୱେ ମାଧ୍ୟମରେ ପ୍ରକ୍ରିୟାକରଣ ହୁଏ।

### ଗ୍ରାହକମାନେ ନିଜ DNS ରେକର୍ଡ ପରିଚାଳନା କରିପାରିବେ କି? {#can-customers-manage-their-dns-records}

ହଁ। A, AAAA, CNAME, MX, ଏବଂ TXT ରେକର୍ଡ ଯୋଡ଼ିବା, ସମ୍ପାଦନ କରିବା, ଏବଂ ମିଟାଇବା ପାଇଁ ଗ୍ରାହକମାନେ ନିଜ Dashboard ରେ ଏକ DNS ପରିଚାଳନା ଇଣ୍ଟରଫେସ୍ ପାଆନ୍ତି। OpenSRS, ResellerClub, Enom, HostAfrica, ଏବଂ Openprovider ସହିତ DNS ପରିଚାଳନା ଉପଲବ୍ଧ। ହୋଷ୍ଟ କରାଯାଇଥିବା ଡୋମେନ୍ ପାଇଁ Hostinger DNS କୋର୍ Hostinger ଡୋମେନ୍-ମ୍ୟାପିଂ ଇଣ୍ଟିଗ୍ରେସନ୍ ଦ୍ୱାରା ପରିଚାଳିତ ହୁଏ।

### ସେଟଅପ୍ ୱିଜାର୍ଡ ଅଛି କି? {#is-there-a-setup-wizard}

ହଁ। ସେଟଅପ୍ ୱିଜାର୍ଡ ପ୍ରଥମ ସକ୍ରିୟକରଣରେ ସ୍ୱୟଂଚାଳିତ ଭାବେ ଚାଲେ ଏବଂ ଏକ ରେଜିଷ୍ଟ୍ରାର୍ ସଂଯୋଗ କରିବା, TLD ଆମଦାନୀ କରିବା, ଏବଂ ଆପଣଙ୍କ ପ୍ରଥମ ଡୋମେନ୍ ଉତ୍ପାଦ ତିଆରି କରିବାରେ ଆପଣଙ୍କୁ ପଦକ୍ଷେପ ଦର ପଦକ୍ଷେପ ସାହାଯ୍ୟ କରେ। ଆପଣ ଆଡଅନ୍ ସେଟିଂରୁ ମଧ୍ୟ ୱିଜାର୍ଡକୁ ପୁଣି ଚାଲାଇପାରିବେ।
