# Material WP

WordPress ପ୍ରଶାସକ ଇଣ୍ଟରଫେସ୍ ବହୁ ଦିନ ଧରି ସମାନ ରହିଛି। କିନ୍ତୁ ପ୍ରଶାସକ ଇଣ୍ଟରଫେସ୍‌ର Material Design ସଂସ୍କରଣ କେମିତି ଦେଖାଯିବ ବୋଲି ଆପଣ କେବେ ଭାବିଛନ୍ତି କି? 

Material WP ହେଉଛି ଏକ ପ୍ରଶାସକ ଥିମ୍, ଯାହା ଆପଣଙ୍କୁ ସେହି ଦେଖା ଓ ଅନୁଭବ ଦେବା ପାଇଁ ବିକଶିତ। ଏହା ଆପଣଙ୍କ clients ମାନଙ୍କୁ ଅଧିକ ନିଜସ୍ୱୀକୃତ Dashboard ଦେବାରେ ସାହାଯ୍ୟ କରେ।

## ବୈଶିଷ୍ଟ୍ୟ

- Custom CSS ସମର୍ଥନ
- ସହଜ ଶୈଳୀ ସେଟିଙ୍ଗ୍
- ମେନୁ ପୁନଃକ୍ରମବଦ୍ଧ କରିବା
- ସହଜ ଆମଦାନି/ରପ୍ତାନି

## ଡକ୍ୟୁମେଣ୍ଟେସନ୍

**Material WP ସହିତ ଆପଣଙ୍କ ପ୍ରଶାସକ ପ୍ୟାନେଲ୍ ନିଜସ୍ୱୀକୃତ କରିବା**

## ସ୍ଥାପନ

1. 'material-wp' କୁ '/wp-content/plugins/' ଡିରେକ୍ଟରୀକୁ ଅପଲୋଡ୍ କରନ୍ତୁ
2. WordPress ର 'ପ୍ଲଗିନ୍' ମେନୁ ମାଧ୍ୟମରେ ପ୍ଲଗିନ୍‌ଟି ସକ୍ରିୟ କରନ୍ତୁ

## ବ୍ୟବହାର ମାର୍ଗଦର୍ଶିକା

WordPress ପ୍ରଶାସକ ଇଣ୍ଟରଫେସ୍ ବହୁ ଦିନ ଧରି ସମାନ ରହିଛି। କିନ୍ତୁ ପ୍ରଶାସକ ଇଣ୍ଟରଫେସ୍‌ର Material Design ସଂସ୍କରଣ କେମିତି ଦେଖାଯିବ ବୋଲି ଆପଣ କେବେ ଭାବିଛନ୍ତି କି?

Material WP ହେଉଛି ଏକ ପ୍ରଶାସକ ଥିମ୍, ଯାହା ଆପଣଙ୍କୁ ସେହି ଦେଖା ଓ ଅନୁଭବ ଦେବା ପାଇଁ ବିକଶିତ। ଏହା ଆପଣଙ୍କ clients ମାନଙ୍କୁ ଅଧିକ ନିଜସ୍ୱୀକୃତ Dashboard ଦେବାରେ ସାହାଯ୍ୟ କରେ। Material WP, Ultimate Multisite ର ପ୍ରିମିୟମ୍ ପ୍ଲଗିନ୍ କୋର୍‌ର ମଧ୍ୟ ଅଂଶ।

**ଏହାକୁ ସ୍ଥାପନ କରିବା** ପରେ, ଆପଣଙ୍କ _ପ୍ଲଗିନ୍_ ପୃଷ୍ଠାକୁ ଯାଆନ୍ତୁ ଏବଂ ଆପଣଙ୍କ Network ରେ ଏହାକୁ ସକ୍ରିୟ କରିବାକୁ କ୍ଲିକ୍ କରନ୍ତୁ।

_ଆପଣଙ୍କ ପ୍ଲଗିନ୍ ପୃଷ୍ଠାରେ ଏହାକୁ ସମଗ୍ର Network ରେ ସକ୍ରିୟ କରନ୍ତୁ_

_**ଟିପ୍ପଣୀ:**_ _ଯଦି ଆପଣ କେବଳ ଏହାକୁ ସକ୍ରିୟ କରନ୍ତି, ଏହା ଆପଣଙ୍କ client ଙ୍କ ପାଖରେ ଏକ ବିକଳ୍ପ ଭାବେ ଦେଖାଯିବ, ଯାହାକୁ କେବଳ ସେହି ଉପଯୋଗକର୍ତ୍ତାଙ୍କୁ ଥିମ୍ ଯୋଡ଼ିବା ପାଇଁ ଚିହ୍ନିତ କରାଯାଇପାରିବ।_

ଏବେ, ଆପଣଙ୍କ ପ୍ରଶାସକ ଇଣ୍ଟରଫେସ୍ ସମ୍ପୂର୍ଣ୍ଣ ଭାବେ ପୁନଃଡିଜାଇନ୍ ହୋଇଛି! ବାମ କଲମ୍‌ରେ Material WP ନାମର ଏକ ନୂତନ ମେନୁ ମଧ୍ୟ ରହିଛି
