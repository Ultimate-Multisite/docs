# Mailchimp

Ultimate Multisite: Mailchimp ସମନ୍ୱୟ

ଆପଣଙ୍କ Ultimate Multisite ନେଟୱର୍କକୁ ଆପଣଙ୍କ Mailchimp ସହ ସମନ୍ୱୟ କରନ୍ତୁ। ଆପଣ ନିର୍ଦ୍ଦିଷ୍ଟ ଯୋଜନାଗୁଡ଼ିକୁ Mailchimp ତାଲିକା ସହ ଯୋଡ଼ିପାରିବେ ଏବଂ ସାଇନଅପ୍ ସମୟରେ ସ୍ୱୟଂଚାଳିତ ଭାବେ ସଦସ୍ୟମାନଙ୍କୁ ସେହି ତାଲିକାଗୁଡ଼ିକରେ ଯୋଡ଼ିପାରିବେ।

## ସଂସ୍ଥାପନ

1. 'ultimate-multisite-mailchimp' କୁ '/wp-content/plugins/' ଡାଇରେକ୍ଟୋରୀକୁ ଅପଲୋଡ୍ କରନ୍ତୁ
2. ଆପଣଙ୍କ WordPress ନେଟୱର୍କ ଆଡମିନ୍‌ରେ 'ପ୍ଲଗିନ୍‌ଗୁଡ଼ିକ' ମେନୁ ମାଧ୍ୟମରେ plugin କୁ ସକ୍ରିୟ କରନ୍ତୁ

## ବ୍ୟବହାର ମାର୍ଗଦର୍ଶିକା

ଆମ Ultimate Multisite: MailChimp Integration Add-on ସହ ଆପଣଙ୍କ ସଦସ୍ୟମାନଙ୍କ ଇମେଲ୍‌କୁ ଆପଣଙ୍କ MailChimp ତାଲିକା ସହ ସିଙ୍କ୍ କରିବା ବହୁତ ସହଜ। ଏହି ଟ୍ୟୁଟୋରିଆଲ୍ ଆପଣଙ୍କୁ ସବୁକିଛି ସେଟ୍‌ଅପ୍ କରିବାରେ ସାହାଯ୍ୟ କରିବ!

## API କୀ ପାଇବା

ଉପଲବ୍ଧ ବିଭିନ୍ନ ସେଟିଂସ୍‌କୁ ଆରମ୍ଭ କରିବା ପୂର୍ବରୁ, ଆମକୁ ଏକ MailChimp API କୀ ଦରକାର। ଏହା Ultimate Multisite କୁ MailChimp API ସହ କଥାହେବାକୁ ଦେବ, ଯାହା ଦ୍ୱାରା ତାଲିକା ଓ ଗୋଷ୍ଠୀ ଆଣିବା, ଏବଂ ବ୍ୟବହାରକାରୀମାନେ ସାଇନଅପ୍ କଲେ ସେହି ତାଲିକା ଓ ଗୋଷ୍ଠୀରେ ଇମେଲ୍ ଠିକଣା ଯୋଡ଼ିବା ସମ୍ଭବ ହେବ।

ଆପଣଙ୍କ ନିଜ MailChimp API କୀ ପାଇବା ପାଇଁ ଏହି ଟ୍ୟୁଟୋରିଆଲ୍ ଅନୁସରଣ କରନ୍ତୁ: [API କୀ ବିଷୟରେ | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

## ସମନ୍ୱୟ ମୋଡ୍

MailChimp Integration add-on ଦୁଇଟି “ସମନ୍ୱୟ ମୋଡ୍” ଦେଇଥାଏ, ଅର୍ଥାତ୍ ଇମେଲ୍ ଠିକଣା ସିଙ୍କ୍ କରିବା ପାଇଁ add-on ବ୍ୟବହାର କରିବାର ଦୁଇଟି ଭିନ୍ନ ଉପାୟ ଅଛି।

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_ଅଧିକ ଲଚିଳାପଣ ପାଇଁ ଦୁଇଟି ସମନ୍ୱୟ ମୋଡ୍!_

### ମୋଡ୍ ୧: ଏକାଧିକ ତାଲିକା

ଡିଫଲ୍ଟ ମୋଡ୍ **ଏକାଧିକ ତାଲିକା** ବ୍ୟବହାର କରେ। ଏହି ମୋଡ୍ ଆପଣଙ୍କୁ ପ୍ରତ୍ୟେକ ଯୋଜନା ପାଇଁ ଏକାଧିକ MailChimp ତାଲିକା ଚୟନ କରିବାକୁ ଦେବ। ଆପଣ ଯଦି ବ୍ୟବହାରକାରୀମାନଙ୍କୁ ଭିନ୍ନ ତାଲିକାରେ ବିଭାଜନ କରନ୍ତି, ଏହା ସର୍ବୋତ୍ତମ ବିକଳ୍ପ।

ଉଦାହରଣ ସ୍ୱରୂପ, ଧରନ୍ତୁ ଆପଣଙ୍କ MailChimp Account ରେ ୩ଟି ଭିନ୍ନ ତାଲିକା ଅଛି: **Newsletter** , **Plan A** , ଏବଂ **Plan B**। ଆପଣ ଚାହୁଁଛନ୍ତି ଯେ ସମସ୍ତ ବ୍ୟବହାରକାରୀ, ସେମାନଙ୍କ ଯୋଜନା ଯେକୌଣସି ହେଉ, Newsletter ତାଲିକାରେ ମଧ୍ୟ ରୁହନ୍ତୁ, ଯାହା ଫଳରେ ସେମାନେ ଆପଣଙ୍କ ଅପଡେଟ୍ ଛାଡ଼ିବେ ନାହିଁ।

ଏହା କରିବା ପାଇଁ, ଆପଣ Multiple Lists ମୋଡ୍ ବ୍ୟବହାର କରି Plan A MailChimp Settings ରେ **Newsletter** ଏବଂ **Plan A** ଦୁଇଟି ତାଲିକା ଯୋଡ଼ିପାରିବେ, ଏବଂ Plan B MailChimp Settings ରେ **Newsletter** ଏବଂ **Plan B** ଯୋଡ଼ିପାରିବେ।

### ମୋଡ୍ ୨: ଏକକ ତାଲିକା + ଏକାଧିକ ଗୋଷ୍ଠୀ

MailChimp ଆପଣଙ୍କ ତାଲିକାକୁ **ଗୋଷ୍ଠୀ** କୁହାଯାଉଥିବା ବ୍ୟବହାରକାରୀମାନଙ୍କ ଛୋଟ ଗୋଷ୍ଠୀରେ ବିଭାଜନ କରିବାକୁ ଦେଇଥାଏ। ଏହା ଆପଣଙ୍କୁ ଅଧିକ ନିୟନ୍ତ୍ରଣ ଦେଇଥାଏ, କାରଣ ଏକେ କ୍ୟାମ୍ପେନ୍‌ର ଅନେକ କପି MailChimp ରେ ସୃଷ୍ଟି କରିବା ଛଡ଼ା, ଆପଣ ସଦସ୍ୟମାନଙ୍କ ଏକ ନିର୍ଦ୍ଦିଷ୍ଟ ଉପଗୋଷ୍ଠୀକୁ କ୍ୟାମ୍ପେନ୍ ପଠାଇପାରିବେ। ପରେ ସେହି କ୍ୟାମ୍ପେନ୍‌ର ମାପଦଣ୍ଡ ବିଶ୍ଳେଷଣ କରିବା ମଧ୍ୟ ସହଜ ହେବ।

ଉଦାହରଣ ସ୍ୱରୂପ, ଆପଣଙ୍କ ପାଖରେ **My Network Subscribers,** ନାମରେ ଏକ ତାଲିକା ଥାଇପାରେ, ଏବଂ ତାହାର ଭିତରେ **Plan A** ଏବଂ **Plan B** ପାଇଁ ଗୋଷ୍ଠୀ ଥାଇପାରେ।

_“ଏକକ ତାଲିକା + ଏକାଧିକ ଗୋଷ୍ଠୀ” ସମନ୍ୱୟ ମୋଡ୍ ବ୍ୟବହାର କରୁଥିଲେ, ଡିଫଲ୍ଟ ତାଲିକା ଏବଂ ଗୋଷ୍ଠୀଗୁଡ଼ିକୁ ଚୟନ କରନ୍ତୁ_

## ଡିଫଲ୍ଟ ତାଲିକା ଏବଂ ଗୋଷ୍ଠୀ

ଆପଣଙ୍କ ନିର୍ଦ୍ଦିଷ୍ଟ ବ୍ୟବହାର ପାଇଁ କେଉଁ ସମନ୍ୱୟ ମୋଡ୍ ଅଧିକ ଉପଯୁକ୍ତ ତାହା ଚୟନ କରିବା ପରେ, ଡିଫଲ୍ଟ ତାଲିକାଗୁଡ଼ିକୁ ଚୟନ କରିବାର ସମୟ ଆସେ (ମୋଡ୍ ଅନୁସାରେ ଗୋଷ୍ଠୀମାନେ ମଧ୍ୟ)। ଯେତେବେଳେ କୌଣସି ଦିଆଯାଇଥିବା ଯୋଜନାରେ ନିର୍ଦ୍ଦିଷ୍ଟ ତାଲିକା ଚୟନ ନଥାଏ, ସେତେବେଳେ ଡିଫଲ୍ଟ ତାଲିକା ବ୍ୟବହୃତ ହେବ।

ଆପଣ ବାତିଲ୍ Account ପାଇଁ ଡିଫଲ୍ଟ ତାଲିକା ମଧ୍ୟ ଚୟନ କରିପାରିବେ। ଯେତେବେଳେ କୌଣସି ବ୍ୟବହାରକାରୀ ନିଜ ସଦସ୍ୟତା ସମାପ୍ତ କରି ଆପଣଙ୍କ ନେଟୱର୍କରୁ ନିଜ Account କାଢ଼ିଦେବେ, ସେତେବେଳେ ତାଙ୍କ ଇମେଲ୍ ଠିକଣା ବାତିଲ୍ Account ପାଇଁ ସେଟ୍ ହୋଇଥିବା ତାଲିକାକୁ ସ୍ଥାନାନ୍ତରିତ ହେବ।

## ଦୁଇଥର ସଦସ୍ୟତା ସକ୍ରିୟ କରିବା

ସେହି ସେଟିଂସ୍ ପୃଷ୍ଠାରେ, ଆପଣ **ଦୁଇଥର ସଦସ୍ୟତା** ସକ୍ରିୟ କରିପାରିବେ। ଏହି ବିକଳ୍ପ ବ୍ୟବହାରକାରୀକୁ ଚୟନିତ ଡିଫଲ୍ଟ ତାଲିକା ଏବଂ ସେହି ବ୍ୟବହାରକାରୀଙ୍କ ଯୋଜନା ପାଇଁ ଚୟନିତ ତାଲିକା — ଦୁଇଟିରେ ଯୋଡ଼ିବ। ଯଦି ଏହି ବିକଳ୍ପ ସକ୍ରିୟ ନୁହେଁ, ବ୍ୟବହାରକାରୀ କେବଳ ତାଙ୍କ ଯୋଜନା ପାଇଁ ଚୟନିତ ତାଲିକାରେ ଯୋଡ଼ାଯିବେ। ସେହି ନିର୍ଦ୍ଦିଷ୍ଟ ଯୋଜନା ପାଇଁ କିଛି ସେଟ୍ ନଥାଇଲେ, ବ୍ୟବହାରକାରୀ ଚୟନିତ ଡିଫଲ୍ଟ ତାଲିକାରେ ଯୋଡ଼ାଯିବେ।

## ପ୍ରତ୍ୟେକ ଯୋଜନା ପାଇଁ ତାଲିକା ଏବଂ ଗୋଷ୍ଠୀ ଚୟନ

ଆପଣଙ୍କ **ଯୋଜନାଗୁଡ଼ିକ** କୁ ଯାଆନ୍ତୁ ଏବଂ ଆପଣ ସମ୍ପାଦନ କରିବାକୁ ଚାହୁଁଥିବା ଯୋଜନାକୁ ଚୟନ କରନ୍ତୁ। ଧ୍ୟାନ ଦିଅନ୍ତୁ, **ଉନ୍ନତ ବିକଳ୍ପ** ମେଟା-ବକ୍ସରେ MailChimp ନାମରେ ଏକ ନୂତନ ଟ୍ୟାବ୍ ଉପଲବ୍ଧ ହେବ।

ସେହି ଟ୍ୟାବ୍‌ରେ, ଆପଣ ସେହି ଯୋଜନା ପାଇଁ ଏକାଧିକ ତାଲିକା କିମ୍ବା ଏକକ ତାଲିକା ଏବଂ ଏକାଧିକ ଗୋଷ୍ଠୀ (ସମନ୍ୱୟ ମୋଡ୍ ଅନୁସାରେ) ଚୟନ କରିପାରିବେ।

ଯେତେବେଳେ କୌଣସି ନୂତନ ବ୍ୟବହାରକାରୀ ସେହି ଯୋଜନା ପାଇଁ ସାଇନଅପ୍ କରିବେ, ତାଙ୍କ ଇମେଲ୍ ଠିକଣା ଚୟନିତ ତାଲିକା ଏବଂ ଗୋଷ୍ଠୀଗୁଡ଼ିକରେ ଯୋଡ଼ାଯିବ!
