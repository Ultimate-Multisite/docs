# Gratis AI Agent

Gratis AI Agent ହେଉଛି ଏକ ସ୍ୱୟଂଚାଳିତ WordPress AI ସହାୟକ, ଯାହା ପ୍ରାକୃତିକ ଭାଷାର ନିର୍ଦ୍ଦେଶ ମାଧ୍ୟମରେ ସମ୍ପୂର୍ଣ୍ଣ WordPress ସାଇଟ୍‌ଗୁଡ଼ିକୁ ଯୋଜନା, ନିର୍ମାଣ ଏବଂ ପରିଚାଳନା କରିପାରେ। ଏହା **କ୍ଷମତା**ର ଏକ ଗଠିତ ସେଟ୍ ଦେଇଥାଏ — ଏମିତି ପୃଥକ, ପରୀକ୍ଷଣଯୋଗ୍ୟ କାର୍ଯ୍ୟଗୁଡ଼ିକ ଯାହାକୁ agent ଚାଲୁ କରିପାରେ — ଯାହା ବିଷୟବସ୍ତୁ ପରିଚାଳନାରୁ ଡିଜାଇନ୍ ସିଷ୍ଟମ୍ କଷ୍ଟମାଇଜେସନ୍ ପର୍ଯ୍ୟନ୍ତ ସବୁକିଛି ଆବରଣ କରେ। ସଂସ୍କରଣ ୧.୯.୦ ବିଷୟବସ୍ତୁ ସୃଷ୍ଟି କ୍ଷମତା, କ୍ଲାଇଏଣ୍ଟ-ପକ୍ଷ ସ୍କ୍ରିନସଟ୍‌ ମାଧ୍ୟମରେ ଭିଜୁଆଲ୍ ପୃଷ୍ଠା ସମୀକ୍ଷା, ପାଞ୍ଚଟି ଅନ୍ତର୍ନିର୍ମିତ agent ପ୍ରୋଫାଇଲ୍, ଏବଂ ପ୍ରବେଶ ନିୟନ୍ତ୍ରଣ ଓ ବ୍ରାଣ୍ଡିଙ୍ଗ ପାଇଁ ବୈଶିଷ୍ଟ୍ୟ ଫ୍ଲାଗ୍ ଯୋଡ଼େ।

## ମୁଖ୍ୟ ବୈଶିଷ୍ଟ୍ୟ {#key-features}

- **କ୍ଷମତା ସିଷ୍ଟମ୍** — ଆପଣଙ୍କ WordPress ଇନ୍‌ସ୍ଟଲେସନ୍‌ରେ agent ଚାଲୁ କରିପାରୁଥିବା କାର୍ଯ୍ୟଗୁଡ଼ିକର ଏକ ମଡ୍ୟୁଲାର୍, ବିସ୍ତାରଯୋଗ୍ୟ ତାଲିକା
- **ବିଷୟବସ୍ତୁ ପରିଚାଳନା** — ପୋଷ୍ଟ ଏବଂ ପୃଷ୍ଠା ସୃଷ୍ଟି କରନ୍ତୁ (ପୃଷ୍ଠା ଟେମ୍ପଲେଟ୍ ନିର୍ଦ୍ଧାରଣ ସହିତ), ଗୋଟିଏ କଲ୍‌ରେ ଏକାଧିକ ପୋଷ୍ଟ ବ୍ୟାଚ୍‌ରେ ସୃଷ୍ଟି କରନ୍ତୁ, ଫିଚର୍ ଛବି ସେଟ୍ କରନ୍ତୁ, ଏବଂ ସମ୍ପର୍କ ଫର୍ମ ତିଆରି କରନ୍ତୁ
- **ଭିଜୁଆଲ୍ ସମୀକ୍ଷା** — ଲାଇଭ୍ ପୃଷ୍ଠାର ସ୍କ୍ରିନସଟ୍ ନିଅନ୍ତୁ, ପୂର୍ବ/ପର ଅବସ୍ଥାର ତଫାତ୍ ଦେଖନ୍ତୁ, ଏବଂ ଲେଆଉଟ୍, ଟାଇପୋଗ୍ରାଫି, ରଙ୍ଗ ଓ ପ୍ରବେଶଯୋଗ୍ୟତାକୁ ଆବରଣ କରୁଥିବା AI-ସୃଷ୍ଟ ଡିଜାଇନ୍ ସମୀକ୍ଷା ପାଆନ୍ତୁ
- **ପାଞ୍ଚଟି ଅନ୍ତର୍ନିର୍ମିତ agent** — ବିଷୟବସ୍ତୁ ଲେଖକ, ସାଇଟ୍ ନିର୍ମାତା, ଡିଜାଇନ୍ ଷ୍ଟୁଡିଓ, ପ୍ଲଗିନ୍ ପରିଚାଳକ, ଏବଂ ସହାୟତା ସହାୟକ; ଚ୍ୟାଟ୍ ହେଡର୍‌ରେ ଥିବା agent ଚୟନକାରୀ ମାଧ୍ୟମରେ ବଦଳାଯୋଗ୍ୟ
- **ବୈଶିଷ୍ଟ୍ୟ ଫ୍ଲାଗ୍** — ଭୂମିକା ସୀମାବଦ୍ଧତା, ହ୍ୱାଇଟ୍-ଲେବେଲ୍ ନାମକରଣ, ଏବଂ ୱିଜେଟ୍ କଷ୍ଟମାଇଜେସନ୍ ପାଇଁ ସେଟିଂସ୍ → ବୈଶିଷ୍ଟ୍ୟ ଫ୍ଲାଗ୍‌ରେ ପ୍ରବେଶ-ନିୟନ୍ତ୍ରଣ ଓ ବ୍ରାଣ୍ଡିଙ୍ଗ ଟଗଲ୍
- **କଷ୍ଟମ୍ ପୋଷ୍ଟ ପ୍ରକାର ପରିଚାଳନା** — ରିଷ୍ଟାର୍ଟ ପରେ ମଧ୍ୟ ରହିଥିବା ସ୍ଥାୟୀତ୍ୱ ସହିତ କଷ୍ଟମ୍ ପୋଷ୍ଟ ପ୍ରକାର ପଞ୍ଜୀକରଣ, ତାଲିକାକରଣ ଏବଂ ଡିଲିଟ୍ କରନ୍ତୁ
- **କଷ୍ଟମ୍ ବର୍ଗୀକରଣ ପରିଚାଳନା** — ସ୍ଥାୟୀତ୍ୱ ସହିତ କଷ୍ଟମ୍ ବର୍ଗୀକରଣ ପଞ୍ଜୀକରଣ, ତାଲିକାକରଣ ଏବଂ ଡିଲିଟ୍ କରନ୍ତୁ
- **ଡିଜାଇନ୍ ସିଷ୍ଟମ୍ କ୍ଷମତା** — କଷ୍ଟମ୍ CSS ଇଞ୍ଜେକ୍ଟ କରନ୍ତୁ, ଚୟନିତ ବ୍ଲକ୍ ପ୍ୟାଟର୍ନ ପରିଚାଳନା କରନ୍ତୁ, ସାଇଟ୍ ଲୋଗୋ ସେଟ୍ କରନ୍ତୁ, ଏବଂ theme.json ପ୍ରିସେଟ୍ ପ୍ରୟୋଗ କରନ୍ତୁ
- **ଗ୍ଲୋବାଲ୍ ଶୈଳୀ ପରିଚାଳନା** — ରଙ୍ଗ ପ୍ୟାଲେଟ୍, ଟାଇପୋଗ୍ରାଫି, ସ୍ପେସିଂ ଏବଂ ଅଧିକ ପାଇଁ theme.json ମୂଲ୍ୟ ପଢ଼ନ୍ତୁ ଓ ଲେଖନ୍ତୁ
- **ନେଭିଗେସନ୍ ମେନୁ ପରିଚାଳନା** — ନେଭିଗେସନ୍ ମେନୁ ଏବଂ ସେଗୁଡ଼ିକର ଆଇଟମ୍‌ଗୁଡ଼ିକୁ ପ୍ରୋଗ୍ରାମ୍ ଭାବେ ସୃଷ୍ଟି ଏବଂ ଅପଡେଟ୍ କରନ୍ତୁ
- **ବିକଳ୍ପ ପରିଚାଳନା** — ଗୁରୁତ୍ୱପୂର୍ଣ୍ଣ ସେଟିଂସ୍‌କୁ ସୁରକ୍ଷିତ ରଖିବା ପାଇଁ ଅନ୍ତର୍ନିର୍ମିତ ସୁରକ୍ଷା ବ୍ଲକ୍ଲିଷ୍ଟ ସହିତ WordPress ବିକଳ୍ପ ପଢ଼ନ୍ତୁ ଏବଂ ଲେଖନ୍ତୁ
- **ସ୍ଥାପନଯୋଗ୍ୟ କ୍ଷମତା ରେଜିଷ୍ଟ୍ରି** — ମୂଳ ପ୍ଲଗିନ୍ ଫାଇଲ୍‌ଗୁଡ଼ିକୁ ବଦଳାଇବା ବିନା ଅତିରିକ୍ତ କ୍ଷମତା ପ୍ୟାକ୍ ଖୋଜନ୍ତୁ ଏବଂ ସକ୍ରିୟ କରନ୍ତୁ
- **ସାଇଟ୍ ବିଲ୍ଡର୍ ଅର୍କେଷ୍ଟ୍ରେସନ୍ v2** — ପ୍ଲଗିନ୍ ଖୋଜ, ପ୍ରଗତି ଟ୍ରାକିଂ ଏବଂ ତ୍ରୁଟି ପୁନରୁଦ୍ଧାର ସହିତ ବହୁ-ପଦକ୍ଷେପ ସାଇଟ୍ ନିର୍ମାଣ ଯୋଜନା
- **WP-CLI ବେଞ୍ଚମାର୍କ କମାଣ୍ଡ** — CI ଏବଂ ମଡେଲ୍ ମୂଲ୍ୟାୟନ ପାଇଁ କମାଣ୍ଡ ଲାଇନ୍‌ରୁ agent କ୍ଷମତା ବେଞ୍ଚମାର୍କ ସୁଇଟ୍ ଚଲାନ୍ତୁ

## ଆବଶ୍ୟକତା {#requirements}

- WordPress ୬.୦ କିମ୍ବା ତାଠାରୁ ଉଚ୍ଚ
- PHP ୮.୧ କିମ୍ବା ତାଠାରୁ ଉଚ୍ଚ
- WordPress Multisite (ନେଟୱର୍କ-ସକ୍ରିୟ) କିମ୍ବା ସିଙ୍ଗଲ୍-ସାଇଟ୍ ଇନ୍‌ସ୍ଟଲେସନ୍
- ଏକ ସକ୍ରିୟ AI ପ୍ରଦାନକାରୀ କନେକ୍ଟର୍ ପ୍ଲଗିନ୍ (Anthropic, OpenAI, କିମ୍ବା ସୁସଙ୍ଗତ)

## ସ୍ଥାପନ {#installation}

1. ପ୍ଲଗିନ୍ ଫାଇଲ୍‌ଗୁଡ଼ିକୁ `/wp-content/plugins/gratis-ai-agent` କୁ ଅପଲୋଡ୍ କରନ୍ତୁ
2. WordPress ର **ପ୍ଲଗିନ୍‌ଗୁଡ଼ିକ** ସ୍କ୍ରିନ୍ ମାଧ୍ୟମରେ ପ୍ଲଗିନ୍ ସକ୍ରିୟ କରନ୍ତୁ
3. **Gratis AI Agent → ସେଟିଂସ୍** କୁ ଯାଆନ୍ତୁ ଏବଂ ଆପଣଙ୍କ AI ପ୍ରଦାନକାରୀକୁ ସଂଯୋଗ କରନ୍ତୁ
4. (ଇଚ୍ଛାନୁସାରେ) ଅତିରିକ୍ତ କ୍ଷମତା ପ୍ୟାକ୍ ସ୍ଥାପନ କରିବାକୁ **କ୍ଷମତା ରେଜିଷ୍ଟ୍ରି** ବ୍ରାଉଜ୍ କରନ୍ତୁ

## ଶୀଘ୍ର ଆରମ୍ଭ {#quick-start}

ସକ୍ରିୟ ହୋଇ AI ପ୍ରଦାନକାରୀ ସହିତ ସଂଯୋଗ ହେବା ପରେ, WordPress ପ୍ରଶାସନ ସାଇଡବାର୍‌ରେ **Gratis AI Agent** ପ୍ୟାନେଲ୍ ଖୋଲନ୍ତୁ। ଏକ ପ୍ରାକୃତିକ ଭାଷାର ନିର୍ଦ୍ଦେଶ ଟାଇପ୍ କରନ୍ତୁ, ଏବଂ agent ତାହାର ଉପଲବ୍ଧ କ୍ଷମତା ବ୍ୟବହାର କରି ଆବଶ୍ୟକ ପଦକ୍ଷେପଗୁଡ଼ିକୁ ଯୋଜନା କରି କାର୍ଯ୍ୟକରିତ କରିବ।

ଅନଲାଇନ୍ ମେନୁ, ବୁକିଂ ଫର୍ମ ଏବଂ ସମ୍ପର୍କ ପୃଷ୍ଠା ସହିତ ଏକ ରେଷ୍ଟୁରାଣ୍ଟ ୱେବସାଇଟ୍ ତିଆରି କରନ୍ତୁ।

## କ୍ଷମତା ସାରାଂଶ {#abilities-overview}

କ୍ଷମତାଗୁଡ଼ିକ ହେଉଛି agent କରିପାରୁଥିବା ମୂଳ କାର୍ଯ୍ୟ। ପ୍ରତ୍ୟେକ କ୍ଷମତା ହେଉଛି ଏକ ପଞ୍ଜୀକୃତ PHP class, ଯାହା agent ରନ୍‌ଟାଇମ୍‌ରେ ପଢ଼ୁଥିବା ଏକ ସ୍କିମା (ପାରାମିଟର୍ ଏବଂ ଫେରତ ପ୍ରକାର) ଦେଇଥାଏ। କ୍ଷମତାଗୁଡ଼ିକୁ କାର୍ଯ୍ୟାତ୍ମକ କ୍ଷେତ୍ରରେ ଗୋଷ୍ଠୀଭୁକ୍ତ କରାଯାଇଛି:

| କ୍ଷେତ୍ର | କ୍ଷମତା |
|---|---|
| **ବିଷୟବସ୍ତୁ ପରିଚାଳନା** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **ଭିଜୁଆଲ୍ ସମୀକ୍ଷା** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **କଷ୍ଟମ୍ ପୋଷ୍ଟ ପ୍ରକାର** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **କଷ୍ଟମ୍ ବର୍ଗୀକରଣ** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **ଡିଜାଇନ୍ ସିଷ୍ଟମ୍** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **ଗ୍ଲୋବାଲ୍ ଶୈଳୀ** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **ନେଭିଗେସନ୍ ମେନୁ** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **ବିକଳ୍ପ ପରିଚାଳନା** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **ସ୍ଥାପନଯୋଗ୍ୟ କ୍ଷମତା** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **ସାଇଟ୍ ବିଲ୍ଡର୍** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

ସମ୍ପୂର୍ଣ୍ଣ ପାରାମିଟର୍ ଡକ୍ୟୁମେଣ୍ଟେସନ୍ ଏବଂ ବ୍ୟବହାର ଉଦାହରଣ ପାଇଁ [କ୍ଷମତା ସନ୍ଦର୍ଭ](./abilities) ଦେଖନ୍ତୁ।

## WP-CLI {#wp-cli}

Gratis AI Agent କମାଣ୍ଡ ଲାଇନ୍‌ରୁ ବେଞ୍ଚମାର୍କ ଚଲାଇବା ଏବଂ କ୍ଷମତା ପରିଚାଳନା କରିବା ପାଇଁ `wp gratis-ai-agent` କମାଣ୍ଡ ପରିବାର ସହିତ ଆସେ। ସମ୍ପୂର୍ଣ୍ଣ କମାଣ୍ଡ ତାଲିକା ପାଇଁ [WP-CLI ସନ୍ଦର୍ଭ](./wp-cli) ଦେଖନ୍ତୁ।

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## ବିନ୍ୟାସ {#configuration}

ସମସ୍ତ ସେଟିଂସ୍ WordPress ପ୍ରଶାସନରେ **Gratis AI Agent → ସେଟିଂସ୍** ଅଧୀନରେ ଅଛି:

| ସେଟିଂ | ବର୍ଣ୍ଣନା |
|---|---|
| **AI ପ୍ରଦାନକାରୀ** | ସକ୍ରିୟ AI ପ୍ରଦାନକାରୀ କନେକ୍ଟର୍ ଚୟନ ଏବଂ ବିନ୍ୟାସ କରନ୍ତୁ |
| **ମଡେଲ୍** | agent ସେସନ୍ ପାଇଁ ବ୍ୟବହାର କରିବାକୁ ଭାଷା ମଡେଲ୍ ବାଛନ୍ତୁ |
| **ସର୍ବାଧିକ ଟୋକନ୍** | ପ୍ରତି-ଅନୁରୋଧ ଟୋକନ୍ ବଜେଟ୍ (ଡିଫଲ୍ଟ ୪୦୯୬) |

| **ଡିବଗ୍ ଲଗିଂ** | କ୍ଷମତା କଲ୍ ଏବଂ agent ର ଯୁକ୍ତିକୁ WordPress ଡିବଗ୍ ଲଗ୍‌ରେ ଲଗ୍ କରନ୍ତୁ |

## ବିକଳ୍ପ ପରିଚାଳନା ସୁରକ୍ଷା ବ୍ଲକ୍ଲିଷ୍ଟ {#options-management-safety-blocklist}

Options Management କ୍ଷମତାରେ ଏକ ନିର୍ମିତ blocklist ରହିଛି, ଯାହା agent କୁ ସେନ୍ସିଟିଭ୍ WordPress options ପଢ଼ିବା କିମ୍ବା ଲେଖିବାରୁ ରୋକେ। ଡିଫଲ୍ଟ blocklist ରେ ଏଗୁଡ଼ିକ ଅଛି:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (URL options — ଏଗୁଡ଼ିକ ପାଇଁ ଏକ ନିର୍ଦ୍ଦିଷ୍ଟ ମାଇଗ୍ରେସନ୍ ଟୁଲ୍ ବ୍ୟବହାର କରନ୍ତୁ)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (ଗୁପ୍ତ କୀ)
- `active_plugins`, `template` (plugin/theme ସକ୍ରିୟକରଣ — ନିର୍ଦ୍ଦିଷ୍ଟ କ୍ଷମତା ବ୍ୟବହାର କରନ୍ତୁ)
- ସ୍ଥାପିତ plugin ଦ୍ୱାରା ପଞ୍ଜିକୃତ ପେମେଣ୍ଟ ଗେଟୱେ ପ୍ରମାଣପତ୍ର ଓ API କୀ

`gratis_ai_agent_options_blocklist` filter ମାଧ୍ୟମରେ ଅତିରିକ୍ତ ଏଣ୍ଟ୍ରି ଯୋଡ଼ାଯାଇପାରେ।

## Hooks ଓ Filters {#hooks-and-filters}

### Actions {#actions}

- `gratis_ai_agent_ability_registered` — କୌଣସି କ୍ଷମତା ପଞ୍ଜିକୃତ ହେବା ପରେ ଚାଲୁ ହୁଏ, କ୍ଷମତା class ନାମ ପାଏ
- `gratis_ai_agent_plan_started` — ଏକ ସାଇଟ୍ ବିଲ୍ଡର୍ ଯୋଜନା ଚାଲିବା ଆରମ୍ଭ ହେଲେ ଚାଲୁ ହୁଏ
- `gratis_ai_agent_plan_completed` — କୌଣସି ଯୋଜନା ସଫଳତାର ସହିତ ସମାପ୍ତ ହେଲେ ଚାଲୁ ହୁଏ
- `gratis_ai_agent_plan_failed` — କୌଣସି ଯୋଜନାରେ ଅପୁନରୁଦ୍ଧାରଯୋଗ୍ୟ ତ୍ରୁଟି ହେଲେ ଚାଲୁ ହୁଏ

### Filters {#filters}

- `gratis_ai_agent_abilities` — agent ଏହାକୁ ଲୋଡ୍ କରିବା ପୂର୍ବରୁ ପଞ୍ଜିକୃତ କ୍ଷମତା ତାଲିକା ସଂଶୋଧନ କରନ୍ତୁ
- `gratis_ai_agent_options_blocklist` — options ସୁରକ୍ଷା blocklist କୁ ବଢ଼ାନ୍ତୁ କିମ୍ବା କମାନ୍ତୁ
- `gratis_ai_agent_benchmark_questions` — ମୂଲ୍ୟାଙ୍କନ ସୁଇଟ୍‌ରେ ନିଜସ୍ୱ benchmark ପ୍ରଶ୍ନ ଯୋଡ଼ନ୍ତୁ
- `gratis_ai_agent_plan_plugins` — ସାଇଟ୍ ଯୋଜନା ତିଆରି ସମୟରେ plugin ଆବିଷ୍କାର ତାଲିକା ସଂଶୋଧନ କରନ୍ତୁ
