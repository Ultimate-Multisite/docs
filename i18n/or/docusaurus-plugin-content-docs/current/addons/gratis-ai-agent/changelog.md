# ପରିବର୍ତ୍ତନ ତାଲିକା {#changelog}

## 1.9.0 — 2026-04-28 ରେ ପ୍ରକାଶିତ {#190--released-on-2026-04-28}

### ନୂଆ {#new}

- **`create_contact_form` କ୍ଷମତା** — ସକ୍ରିୟ ଫର୍ମ ପ୍ଲଗଇନ୍‌ (Contact Form 7, WPForms, Fluent Forms, କିମ୍ବା Gravity Forms) ବ୍ୟବହାର କରି ଏକ ସଂପର୍କ ଫର୍ମ ତିଆରି କରେ ଏବଂ ଯେକୌଣସି ପୋଷ୍ଟ କିମ୍ବା ପୃଷ୍ଠାରେ ଏମ୍ବେଡ୍‌ କରିବା ପାଇଁ ପ୍ରସ୍ତୁତ ଏକ ଶର୍ଟକୋଡ୍‌ ଫେରାଇଦିଏ।
- **`set_featured_image` କ୍ଷମତା** — ଥିବା Media Library ଅଟାଚମେଣ୍ଟ ID କିମ୍ବା ଦୂରସ୍ଥ URL ରୁ ପୋଷ୍ଟ କିମ୍ବା ପୃଷ୍ଠାକୁ ଏକ ବୈଶିଷ୍ଟ୍ୟ ଛବି ନିର୍ଦ୍ଧାରଣ କରେ; URL ଦିଆଯାଇଲେ ଛବିଟିକୁ ସ୍ୱୟଂଚାଳିତ ଭାବେ ଆମଦାନି କରେ।
- **`batch_create_posts` କ୍ଷମତା** — ଗୋଟିଏ କ୍ଷମତା କଲ୍‌ରେ ଅନେକ ପୋଷ୍ଟ ତିଆରି କରେ। `create_post` ସହିତ ସମାନ ପାରାମିଟର୍‌ ସମର୍ଥନ କରେ, ପ୍ରତ୍ୟେକ ପୋଷ୍ଟର ସଫଳତା/ବିଫଳତା ଜଣାଏ, ଏବଂ ଏକ ବୈକଳ୍ପିକ `stop_on_error` ମୋଡ୍‌ ରହିଛି।
- **`page_template` ପାରାମିଟର୍‌** — `create_post` ଏବଂ `update_post` ରେ ଯୋଡ଼ାଗଲା। ସୃଷ୍ଟି କିମ୍ବା ଅଦ୍ୟତନ ସମୟରେ ଏକ PHP ପୃଷ୍ଠା ଟେମ୍ପ୍ଲେଟ୍‌ ଫାଇଲ୍‌ (ଉଦାହରଣ `page-full-width.php`) ନିର୍ଦ୍ଧାରଣ କରେ। ଥିମ୍‌ ଡିଫଲ୍ଟକୁ ଫେରିବା ପାଇଁ `update_post` କୁ ଏକ ଖାଲି ଷ୍ଟ୍ରିଙ୍ଗ୍‌ ପଠାନ୍ତୁ।
- **କ୍ଲାଇଏଣ୍ଟ-ପକ୍ଷ ସ୍କ୍ରିନଶଟ୍‌ କ୍ଷମତା** — `capture_screenshot`, `compare_screenshots`, ଏବଂ `review_page_design`। ସର୍ଭର୍‌-ପକ୍ଷ ହେଡଲେସ୍‌ ବ୍ରାଉଜର୍‌ ମାଧ୍ୟମରେ ଲାଇଭ୍‌ ପୃଷ୍ଠାର ପୂର୍ଣ୍ଣ କିମ୍ବା ଭ୍ୟୁପୋର୍ଟ ସ୍କ୍ରିନଶଟ୍‌ ଧରନ୍ତୁ, ଦୁଇଟି ସ୍କ୍ରିନଶଟ୍‌ ତୁଳନା କରନ୍ତୁ, ଏବଂ ଲେଆଉଟ୍‌, ଟାଇପୋଗ୍ରାଫି, ରଙ୍ଗ ଓ ପ୍ରବେଶଯୋଗ୍ୟତା ଆବରଣ କରୁଥିବା AI-ସୃଷ୍ଟ ଡିଜାଇନ୍‌ ସମୀକ୍ଷା ପାଆନ୍ତୁ।
- **ପାଞ୍ଚଟି ନିର୍ମିତ ଏଜେଣ୍ଟ** — ବିଷୟବସ୍ତୁ ଲେଖକ, ସାଇଟ୍‌ ନିର୍ମାତା, ଡିଜାଇନ୍‌ ଷ୍ଟୁଡିଓ, ପ୍ଲଗଇନ୍‌ ପରିଚାଳକ, ଏବଂ ସହାୟତା ସହାୟକ। ପ୍ରତ୍ୟେକ ଏଜେଣ୍ଟ ପାଖରେ ନିର୍ଦ୍ଦିଷ୍ଟ ଟୁଲ୍‌ ସେଟ୍‌, ଅନୁକୂଳିତ ସିଷ୍ଟମ୍‌ ପ୍ରମ୍ପ୍ଟ, ଏବଂ ଆରମ୍ଭିକ ପରାମର୍ଶ ରହିଛି। ଚାଟ୍‌ ହେଡର୍‌ରେ ଥିବା ନୂଆ **ଏଜେଣ୍ଟ ଚୟନକାରୀ** ଦ୍ୱାରା ବଦଳାଯାଇପାରେ। [ନିର୍ମିତ ଏଜେଣ୍ଟ](../../user-guide/configuration/built-in-agents) ଦେଖନ୍ତୁ।
- **ବୈଶିଷ୍ଟ୍ୟ ଫ୍ଲାଗ୍‌ଗୁଡ଼ିକ** — ପ୍ରବେଶ-ନିୟନ୍ତ୍ରଣ ଟଗଲ୍‌ (ପ୍ରଶାସକଙ୍କ ପାଇଁ ସୀମିତ, ନେଟୱର୍କ ପ୍ରଶାସକଙ୍କ ପାଇଁ ସୀମିତ, ସବସ୍କ୍ରାଇବର୍‌ ପ୍ରବେଶ, ଅସଦସ୍ୟଙ୍କ ପାଇଁ ଅକ୍ଷମ) ଏବଂ ବ୍ରାଣ୍ଡିଂ ବିକଳ୍ପ (ଫୁଟର୍‌ ଶ୍ରେୟ ଲୁଚାନ୍ତୁ, କଷ୍ଟମ୍‌ ଏଜେଣ୍ଟ ନାମ, ଏଜେଣ୍ଟ ଚୟନକାରୀ ଲୁଚାନ୍ତୁ, ଚାଟ୍‌ ଅଭାତାର୍‌ ଭାବେ ସାଇଟ୍‌ ଆଇକନ୍‌ ବ୍ୟବହାର କରନ୍ତୁ) ସହିତ ନୂଆ **ସେଟିଂସ୍‌ → ବୈଶିଷ୍ଟ୍ୟ ଫ୍ଲାଗ୍‌ଗୁଡ଼ିକ** ଟ୍ୟାବ୍‌। [Gratis AI Agent ସେଟିଂସ୍‌](../../user-guide/administration/gratis-ai-agent-settings) ଦେଖନ୍ତୁ।
- **ଶେଷ ସେସନ୍‌ ପୁନରୁଦ୍ଧାର** — ପୃଷ୍ଠା ଲୋଡ୍‌ ଓ ଉଇଜେଟ୍‌ ଖୋଲିବା ସମୟରେ ଚାଟ୍‌ ପ୍ୟାନେଲ୍‌ ବର୍ତ୍ତମାନ ସବୁଠାରୁ ସମ୍ପ୍ରତିକ ଆଲୋଚନାକୁ ସ୍ୱୟଂଚାଳିତ ଭାବେ ପୁଣି ଲୋଡ୍‌ କରେ, ତେଣୁ ପୃଷ୍ଠା ନାଭିଗେସନ୍‌ ମଧ୍ୟରେ ପ୍ରସଙ୍ଗ କେବେ ହରାଏ ନାହିଁ।
- **ପ୍ଲଗଇନ୍‌ କାର୍ଯ୍ୟ ଲିଙ୍କ୍‌ଗୁଡ଼ିକ** — ସେଟିଂସ୍‌ ଏବଂ କ୍ଷମତା ରେଜିଷ୍ଟ୍ରିକୁ ଶୀଘ୍ର ଲିଙ୍କ୍‌ ବର୍ତ୍ତମାନ WordPress **ପ୍ଲଗଇନ୍‌ଗୁଡ଼ିକ → ସଂସ୍ଥାପିତ ପ୍ଲଗଇନ୍‌ଗୁଡ଼ିକ** ସ୍କ୍ରିନ୍‌ରେ ପ୍ଲଗଇନ୍‌ ବିବରଣୀ ତଳେ ଦେଖାଯାଏ।

### ସୁଧାରିତ {#improved}

- **ଛବି ସ୍ରୋତ ପୁନଃଚେଷ୍ଟା** — ଡାଉନଲୋଡ୍‌ ବିଫଳ ହେଲେ AI-ସୃଷ୍ଟ ଛବିକୁ ଫେରିବା ପୂର୍ବରୁ ଏଜେଣ୍ଟ ବର୍ତ୍ତମାନ ସମସ୍ତ ବିନ୍ୟାସିତ ମାଗଣା ଛବି ସ୍ରୋତକୁ ପୁଣି ଚେଷ୍ଟା କରେ।
- **ମଡେଲ୍‌ ସୂଚନା ପ୍ୟାନେଲ୍‌** — ଚାଟ୍‌ ହେଡର୍‌ରେ ସବୁବେଳେ ଦୃଶ୍ୟମାନ; ପ୍ରଥମ ସନ୍ଦେଶ ପରେ ଆଉ ଲୁଚାଯାଏ ନାହିଁ।
- **ସ୍ୱୟଂ-ସ୍କ୍ରୋଲ୍‌ ଆଚରଣ** — ବ୍ୟବହାରକାରୀ ପଢ଼ିବା ପାଇଁ ଉପରକୁ ସ୍କ୍ରୋଲ୍‌ କଲେ ସ୍ୱୟଂ-ସ୍କ୍ରୋଲ୍‌ ବିରାମ ନିଏ; ଏକ ଭାସୁଥିବା **ତଳକୁ ସ୍କ୍ରୋଲ୍‌ କରନ୍ତୁ** ବଟନ୍‌ ଦେଖାଯାଏ ଏବଂ ବ୍ୟବହାରକାରୀ ନୂତନତମ ସନ୍ଦେଶକୁ ପହଞ୍ଚିଲେ ସ୍ୱୟଂଚାଳିତ ଭାବେ ଅଲୋପ ହୋଇଯାଏ।
- **ଏଜେଣ୍ଟ ଚୟନକାରୀ UI** — ପ୍ରତ୍ୟେକ ଏଜେଣ୍ଟ ପାଇଁ ଆଇକନ୍‌ ସହିତ ଫର୍ମ-ଟେବୁଲ୍‌ ଓଭରଲେ ଭାବେ ପୁନର୍ଡିଜାଇନ୍‌ କରାଗଲା, ଯାହା ଏକ ନଜରରେ ଏଜେଣ୍ଟ ଚିହ୍ନଟ ଓ ବଦଳାଇବାକୁ ସହଜ କରେ।
- **ଆଲସ୍ୟରେ ଲୋଡ୍‌ ହେଉଥିବା JS ଖଣ୍ଡଗୁଡ଼ିକ** — ଚାଟ୍‌ ଉଇଜେଟ୍‌ର ପ୍ରାରମ୍ଭିକ JavaScript ବଣ୍ଡଲ୍‌ ବର୍ତ୍ତମାନ ଆଲସ୍ୟରେ ଲୋଡ୍‌ ହେଉଥିବା ଖଣ୍ଡରେ ବିଭକ୍ତ, ଯାହା ପ୍ରାରମ୍ଭିକ ବଣ୍ଡଲ୍‌ ଆକାରକୁ 75–90% କମାଏ।
- **ଚାଟ୍‌ ଉଇଜେଟ୍‌ ପୁନର୍ଡିଜାଇନ୍‌** — ପୂର୍ବତନ କଷ୍ଟମ୍‌ ଅଭାତାର୍‌କୁ ଏକୀକୃତ AI ଆଇକନ୍‌ ପରିବର୍ତ୍ତନ କରିଛି; ନିର୍ମିତ ଏଜେଣ୍ଟ ସିଷ୍ଟମ୍‌ ସହିତ ସମନ୍ୱିତ।
- **URL ଲିଙ୍କୀକରଣ** — ସିଷ୍ଟମ୍‌ ସନ୍ଦେଶ ଏବଂ ତ୍ରୁଟି ସନ୍ଦେଶ ବୁବୁଲ୍‌ରେ ଦେଖାଯାଉଥିବା URL ଗୁଡ଼ିକ ବର୍ତ୍ତମାନ କ୍ଲିକ୍‌ଯୋଗ୍ୟ ଲିଙ୍କ୍‌ ଭାବେ ଦେଖାଯାଏ।

### ସଠିକ୍‌ କରାଗଲା {#fixed}

- **କ୍ଷମତା ଖୋଜିପାଇବା** — ବିବରଣୀ, ସିଷ୍ଟମ୍‌ ପ୍ରମ୍ପ୍ଟ ସନ୍ଦର୍ଭ, ଏବଂ namespace ସମନ୍ୱୟ ସଠିକ୍‌ କରାଗଲା, ଯାହାଦ୍ୱାରା ସମସ୍ତ କ୍ଷମତା ଏଜେଣ୍ଟର ଟୁଲ୍‌ ତାଲିକାରେ ଭରସାଯୋଗ୍ୟ ଭାବେ ଦେଖାଯାଏ।
- **ପ୍ରଦାନକାରୀ କ୍ୟାଶ୍‌** — ପ୍ରଦାନକାରୀମାନଙ୍କୁ ବର୍ତ୍ତମାନ ଏକ ସଂସ୍କରଣ କାଉଣ୍ଟର୍‌ ମାଧ୍ୟମରେ ସାଇଟ୍‌-ବ୍ୟାପୀ କ୍ୟାଶ୍‌ କରାଯାଏ, ଯାହା multisite ନେଟୱର୍କରେ ପୁରୁଣା-ପ୍ରଦାନକାରୀ ସମସ୍ୟାକୁ ରୋକେ।
- **`ability_invalid_output`** — 12ଟି କ୍ଷମତା ହ୍ୟାଣ୍ଡଲର୍‌ରେ ସମାଧାନ ହୋଇଛି; ସମସ୍ତେ ଠିକ୍‌ ଗଠିତ JSON ପ୍ରତିକ୍ରିୟା ଫେରାଇଦିଅନ୍ତି।
- **`pending_client_tool_calls` ପାଇପଲାଇନ୍‌** — ଆରମ୍ଭରୁ ଶେଷ ପର୍ଯ୍ୟନ୍ତ ସଂଯୋଜିତ, ଯାହାଦ୍ୱାରା କ୍ଲାଇଏଣ୍ଟ-ପକ୍ଷ ଟୁଲ୍‌ କଲ୍‌ଗୁଡ଼ିକ ଠିକ୍‌ଭାବେ ସମ୍ପୂର୍ଣ୍ଣ ହୋଇ ଫଳାଫଳକୁ ମଡେଲ୍‌କୁ ଫେରାଇଦିଅନ୍ତି।
- **ଇତିହାସ ଡ୍ରୟର୍‌** — ଫେରାଇପାରିବା ନଥିବା ପରିବର୍ତ୍ତନଗୁଡ଼ିକୁ ଫେରାଇବା ତାଲିକାରୁ ବାଦ ଦିଆଯାଇଛି; **ସମ୍ପୂର୍ଣ୍ଣ ଇତିହାସ ଦେଖନ୍ତୁ** ଲିଙ୍କ୍‌ ବର୍ତ୍ତମାନ ଠିକ୍‌ଭାବେ କାମ କରେ।
- **ପରିବର୍ତ୍ତନ/ଫେରାଇବା ସିଷ୍ଟମ୍‌** — ପାଞ୍ଚଟି ଭିନ୍ନ ବଗ୍‌ ସଠିକ୍‌ କରାଗଲା ଏବଂ ନୂଆ ପ୍ରଶାସନ ଇଣ୍ଟରଫେସ୍‌ ଅଧୀନରେ ଏକୀକୃତ କରାଗଲା।
- **ସେଟିଂସ୍‌ ସେଭ୍‌ ଟୋଷ୍ଟ** — **ସେଟିଂସ୍‌ ସେଭ୍‌ କରନ୍ତୁ** କ୍ଲିକ୍‌ କରିବା ପରେ ସ୍ନାକବାର୍‌ ସୂଚନା ବର୍ତ୍ତମାନ ଭରସାଯୋଗ୍ୟ ଭାବେ ଦେଖାଯାଏ।
- **ଟ୍ରାଶ୍‌ ପ୍ରସଙ୍ଗ ମେନୁ** — **ସ୍ଥାୟୀ ଭାବେ ଡିଲିଟ୍‌ କରନ୍ତୁ** ବିକଳ୍ପ ଯୋଡ଼ାଗଲା, ଯାହାଦ୍ୱାରା ଟ୍ରାଶ୍‌ ଭ୍ୟୁ ଛାଡ଼ିବା ବିନା ଆଇଟମ୍‌ଗୁଡ଼ିକୁ ହାର୍ଡ-ଡିଲିଟ୍‌ କରାଯାଇପାରେ।
- **ସମ୍ପାଦନା ଓ ପୁଣି ପଠାନ୍ତୁ** — ସମ୍ପାଦନା ଆଇକନ୍‌ କ୍ଲିକ୍‌ କଲେ ବର୍ତ୍ତମାନ ସମଗ୍ର ଥ୍ରେଡ୍‌ର ସମସ୍ତ ସନ୍ଦେଶ ନୁହେଁ, କେବଳ କ୍ଲିକ୍‌ କରାଯାଇଥିବା ସନ୍ଦେଶ ପାଇଁ ସମ୍ପାଦନା ମୋଡ୍‌ ଖୋଲେ।
- **ଚାଟ୍‌ ଲେଆଉଟ୍‌ ଉଚ୍ଚତା** — ପୃଷ୍ଠାର ଉପରେ ପ୍ଲଗଇନ୍‌-ଇଞ୍ଜେକ୍ଟ କରାଯାଇଥିବା ବିଷୟବସ୍ତୁ (ପ୍ରଶାସନ ସୂଚନା, ବ୍ୟାନର୍‌) ଦେଖାଦେଲେ ଚାଟ୍‌ ପ୍ୟାନେଲ୍‌ ନିଜ ଉଚ୍ଚତାକୁ ଅନୁକୂଳ କରେ, ଯାହା ଇନପୁଟ୍‌ ଅଞ୍ଚଳକୁ ସ୍କ୍ରିନ୍‌ ବାହାରକୁ ଠେଲାଯିବାରୁ ରୋକେ।

## 1.4.0 — 2026-04-09 ରେ ପ୍ରକାଶିତ {#140--released-on-2026-04-09}

### ନୂଆ {#new-1}

- **WP-CLI ବେଞ୍ଚମାର୍କ କମାଣ୍ଡ** (`wp gratis-ai-agent benchmark`) — CI ପାଇପଲାଇନ ଏବଂ ମଡେଲ ମୂଲ୍ୟାଙ୍କନ କାର୍ଯ୍ୟପ୍ରବାହ ପାଇଁ କମାଣ୍ଡ ଲାଇନରୁ Agent Capabilities v1 ବେଞ୍ଚମାର୍କ ସୁଇଟ ଚଳାଏ। ପ୍ରତି-ପ୍ରଶ୍ନ ଚଳାଣ, ପ୍ରଦାତା/ମଡେଲ ଓଭରରାଇଡ, ଏବଂ JSON/CSV ଆଉଟପୁଟ ସମର୍ଥନ କରେ।
- **Agent Capabilities v1 ବେଞ୍ଚମାର୍କ ସୁଇଟ** — ପୂର୍ଣ୍ଣ କ୍ଷମତା ପୃଷ୍ଠଭୂମିକୁ ପରୀକ୍ଷା କରୁଥିବା ଜଟିଳ, ବହୁ-ପଦକ୍ଷେପ ପ୍ରମ୍ପ୍ଟଗୁଡ଼ିକର ଏକ ସଂରଚିତ ସେଟ, ସ୍କୋରିଂ ଏବଂ ଟୋକେନ/ଅବଧି ରିପୋର୍ଟିଂ ସହିତ।
- **କଷ୍ଟମ ପୋଷ୍ଟ ଟାଇପ କ୍ଷମତା** — `register_post_type`, `list_post_types`, `delete_post_type`। ପଞ୍ଜୀକରଣଗୁଡ଼ିକ options ଟେବୁଲ ମାଧ୍ୟମରେ ପୁନଃଆରମ୍ଭ ପରେ ମଧ୍ୟ ରହିଥାଏ।
- **କଷ୍ଟମ Taxonomy କ୍ଷମତା** — `register_taxonomy`, `list_taxonomies`, `delete_taxonomy`। ବୈକଳ୍ପିକ rewrite slug ସହିତ ସ୍ତରୀୟ ଏବଂ ସମତଳ taxonomies ସମର୍ଥନ କରେ।
- **ଡିଜାଇନ ସିଷ୍ଟମ କ୍ଷମତା** — `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset`। ପାଞ୍ଚଟି ଚୟନିତ preset ସମ୍ମିଳିତ: `minimal-dark`, `warm-editorial`, `corporate-blue`, `vibrant-startup`, `classic-blog`।
- **Global Styles କ୍ଷମତା** — `get_global_styles`, `set_global_styles`, `reset_global_styles`। WordPress Global Styles API ମାଧ୍ୟମରେ theme.json ମୂଲ୍ୟଗୁଡ଼ିକ ପଢ଼େ ଏବଂ ଲେଖେ।
- **ନେଭିଗେସନ ମେନୁ ପରିଚାଳନା କ୍ଷମତା** — `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus`। nested menu ଏବଂ theme location assignment ସମର୍ଥନ କରେ।
- **Options ପରିଚାଳନା କ୍ଷମତା** — `get_option`, `set_option`, `delete_option`, `list_options`। ଗୁରୁତ୍ୱପୂର୍ଣ୍ଣ WordPress options କୁ ପରିବର୍ତ୍ତନରୁ ସୁରକ୍ଷା କରୁଥିବା ଏକ ନିର୍ମିତ safety blocklist ସମ୍ମିଳିତ।
- **ସଂସ୍ଥାପନଯୋଗ୍ୟ କ୍ଷମତା ରେଜିଷ୍ଟ୍ରି** — `list_available_abilities`, `install_ability`, `recommend_plugin`। WordPress plugins ଭାବେ ବଣ୍ଟିତ ability packs ଖୋଜନ୍ତୁ ଏବଂ ସକ୍ରିୟ କରନ୍ତୁ।
- **ସାଇଟ ବିଲ୍ଡର ଅର୍କେଷ୍ଟ୍ରେସନ v2** — `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`। plugin discovery, ପଦକ୍ଷେପ ଆଉଟପୁଟ ସନ୍ଦର୍ଭ, ପ୍ରଗତି ଟ୍ରାକିଂ, ଏବଂ ସ୍ୱୟଂଚାଳିତ ତ୍ରୁଟି ପୁନରୁଦ୍ଧାର ସହିତ ବହୁ-ପର୍ଯ୍ୟାୟ ସାଇଟ ନିର୍ମାଣ ପ୍ଲାନ।
- **ରେଷ୍ଟୁରାଣ୍ଟ ୱେବସାଇଟ ବେଞ୍ଚମାର୍କ ପ୍ରଶ୍ନ** (`q-restaurant-website`) — CPT ପଞ୍ଜୀକରଣ, ଡିଜାଇନ ସିଷ୍ଟମ, ନେଭିଗେସନ, ଏବଂ plugin discovery ଆବରଣ କରୁଥିବା ଆରମ୍ଭରୁ ଶେଷ ପର୍ଯ୍ୟନ୍ତ ପରୀକ୍ଷା।
