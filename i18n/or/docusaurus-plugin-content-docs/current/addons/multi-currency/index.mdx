# ବହୁ-ମୁଦ୍ରା

Ultimate Multisite: ବହୁ-ମୁଦ୍ରା ଆପଣଙ୍କ ନେଟୱର୍କକୁ ଏକାଧିକ ମୁଦ୍ରାରେ ପେମେଣ୍ଟ ଗ୍ରହଣ କରିବାକୁ ସକ୍ଷମ କରେ। ଏହା ସଜୀବ କିମ୍ବା ମାନୁଆଲ ବିନିମୟ ହାର ବ୍ୟବହାର କରି ମୂଲ୍ୟକୁ ସ୍ୱୟଂଚାଳିତ ଭାବେ ରୂପାନ୍ତର କରେ, ଯେକୌଣସି ମୁଦ୍ରା ପାଇଁ ପ୍ରତ୍ୟେକ ଉତ୍ପାଦର ନିର୍ଦ୍ଦିଷ୍ଟ ମୂଲ୍ୟ ସେଟ୍ କରିବାକୁ ଦିଏ, ଏବଂ ଭୌଗୋଳିକ ଅବସ୍ଥାନ ଦ୍ୱାରା ପ୍ରତ୍ୟେକ ଦର୍ଶକଙ୍କ ପସନ୍ଦର ମୁଦ୍ରା ଚିହ୍ନଟ କରେ — ସବୁକିଛି ଗୋଟିଏ ମାତ୍ର ଆଡମିନ Dashboard ରୁ ପରିଚାଳିତ ହୁଏ।

## ପ୍ରମୁଖ ବୈଶିଷ୍ଟ୍ୟ

- **ସ୍ୱୟଂଚାଳିତ ବିନିମୟ ହାର ରୂପାନ୍ତର** — ଆପଣ ବାଛିଥିବା ପ୍ରଦାତାଠାରୁ ଆଣାଯାଇଥିବା ହାର ବ୍ୟବହାର କରି, ଆପଣଙ୍କ ମୂଳ ମୁଦ୍ରାରୁ ମୂଲ୍ୟଗୁଡ଼ିକ ରୂପାନ୍ତର ହୁଏ
- **ପ୍ରତ୍ୟେକ ଉତ୍ପାଦ ପାଇଁ ମୂଲ୍ୟ ଓଭରରାଇଡ୍** — ସ୍ୱୟଂଚାଳିତ ରୂପାନ୍ତରକୁ ଏଡ଼ାଇବାକୁ ଯେକୌଣସି ଉତ୍ପାଦରେ ମୁଦ୍ରା-ନିର୍ଦ୍ଦିଷ୍ଟ ସଠିକ୍ ମୂଲ୍ୟ ସେଟ୍ କରନ୍ତୁ
- **ମୁଦ୍ରା ଚୟନକାରୀ ସାଇନଅପ୍ ଫିଲ୍ଡ** — ଚେକଆଉଟ୍/ସାଇନଅପ୍ ସମୟରେ ଗ୍ରାହକମାନଙ୍କୁ ନିଜ ପସନ୍ଦର ମୁଦ୍ରା ବାଛିବାକୁ ଦିଏ
- **ଏକାଧିକ ବିନିମୟ ହାର ପ୍ରଦାତା** — ECB, Frankfurter, Stripe, କିମ୍ବା ସମ୍ପୂର୍ଣ୍ଣ ମାନୁଆଲ ହାର
- **ଭୌଗୋଳିକ ଅବସ୍ଥାନ ଆଧାରିତ ମୁଦ୍ରା ଚିହ୍ନଟ** — ଦର୍ଶକଙ୍କ ଦେଶ ସହ ମେଳ ଖାଉଥିବା ମୁଦ୍ରାକୁ ସ୍ୱୟଂଚାଳିତ ଭାବେ ପୂର୍ବରୁ ବାଛେ
- **ଆଡମିନ Dashboard** — ସକ୍ରିୟ ମୁଦ୍ରା, ବିନିମୟ ହାର, ଏବଂ ପ୍ରଦାତା ସେଟିଂଗୁଡ଼ିକୁ ଗୋଟିଏ ସ୍ଥାନରୁ ପରିଚାଳନା କରନ୍ତୁ

## ଆବଶ୍ୟକତା

- Ultimate Multisite 2.0.0 କିମ୍ବା ତାଠାରୁ ଉଚ୍ଚ
- WordPress Multisite ଇନଷ୍ଟଲେସନ୍
- PHP 7.4 କିମ୍ବା ତାଠାରୁ ଉଚ୍ଚ

## ଇନଷ୍ଟଲେସନ୍

1. plugin ଫାଇଲଗୁଡ଼ିକୁ `/wp-content/plugins/ultimate-multisite-multi-currency` କୁ ଅପଲୋଡ୍ କରନ୍ତୁ
2. WordPress ର **Plugins** ସ୍କ୍ରିନ୍ ମାଧ୍ୟମରେ plugin କୁ ସକ୍ରିୟ କରନ୍ତୁ (ନେଟୱର୍କ ସକ୍ରିୟ)
3. ଆଡମିନ Dashboard ଖୋଲିବାକୁ **Ultimate Multisite → ବହୁ-ମୁଦ୍ରା** କୁ ଯାଆନ୍ତୁ
4. ଆପଣ ଗ୍ରହଣ କରିବାକୁ ଚାହୁଁଥିବା ମୁଦ୍ରାଗୁଡ଼ିକ ଯୋଡ଼ନ୍ତୁ ଏବଂ ଆପଣଙ୍କ ବିନିମୟ ହାର ପ୍ରଦାତାକୁ କନଫିଗର୍ କରନ୍ତୁ

## କନଫିଗରେସନ୍

### ଆଡମିନ Dashboard

ବହୁ-ମୁଦ୍ରା ଆଡମିନ Dashboard (**Ultimate Multisite → ବହୁ-ମୁଦ୍ରା**) ହେଉଛି ଏହା ପାଇଁ କେନ୍ଦ୍ରୀୟ ସ୍ଥାନ:

- ମୁଦ୍ରା ଯୋଡ଼ିବା, ସକ୍ରିୟ କରିବା, କିମ୍ବା ନିଷ୍କ୍ରିୟ କରିବା
- ବିନିମୟ ହାର ପ୍ରଦାତା ବାଛିବା ଏବଂ ଆବଶ୍ୟକ ସମୟରେ ହାର ପୁନଃସତେଜ କରିବା
- ସମସ୍ତ ସକ୍ରିୟ ମୁଦ୍ରା ପାଇଁ ବର୍ତ୍ତମାନ ହାର ସମୀକ୍ଷା କରିବା
- ସ୍ୱୟଂଚାଳିତ ପ୍ରଦାତା ଉପଯୁକ୍ତ ନ ହେଲେ ମାନୁଆଲ ହାର ସେଟ୍ କରିବା

### ବିନିମୟ ହାର ପ୍ରଦାତା

| ପ୍ରଦାତା | ବର୍ଣ୍ଣନା |
|---|---|
| **ECB** | ୟୁରୋପୀୟ କେନ୍ଦ୍ରୀୟ ବ୍ୟାଙ୍କ — EUR-ଆଧାରିତ ନେଟୱର୍କ ପାଇଁ ମାଗଣା ଦୈନିକ ହାର |
| **Frankfurter** | ଖୋଲା-ଉତ୍ସ ECB ମିରର୍ API — ମାଗଣା, API କୀ ଦରକାର ନାହିଁ |
| **Stripe** | Stripe ର ସଜୀବ FX ହାର ବ୍ୟବହାର କରେ — ସକ୍ରିୟ Stripe ଇଣ୍ଟିଗ୍ରେସନ୍ ଆବଶ୍ୟକ |
| **ମାନୁଆଲ** | ଆପଣ ସିଧାସଳଖ ହାର ଦାଖଲ କରନ୍ତି; ବାହ୍ୟ API କଲ୍ ନାହିଁ |

**Ultimate Multisite → ବହୁ-ମୁଦ୍ରା → ସେଟିଂସ୍ → ବିନିମୟ ହାର ପ୍ରଦାତା** ଅଧୀନରେ ଆପଣଙ୍କ ପ୍ରଦାତା ବାଛନ୍ତୁ। ହାରଗୁଡ଼ିକୁ ଯେକୌଣସି ସମୟରେ ହାତେ ପୁନଃସତେଜ କରାଯାଇପାରେ କିମ୍ବା ସ୍ୱୟଂଚାଳିତ ଭାବେ ଅଦ୍ୟତନ ପାଇଁ ସୂଚୀଭୁକ୍ତ କରାଯାଇପାରେ।

### ପ୍ରତ୍ୟେକ ଉତ୍ପାଦ ପାଇଁ ମୂଲ୍ୟ ଓଭରରାଇଡ୍

ସ୍ୱୟଂଚାଳିତ ରୂପାନ୍ତର ପର୍ଯ୍ୟାପ୍ତ ସଠିକ୍ ନ ହେଲେ, ଆପଣ ଯେକୌଣସି ଉତ୍ପାଦରେ ନିର୍ଦ୍ଦିଷ୍ଟ ମୁଦ୍ରା ପାଇଁ ସଠିକ୍ ମୂଲ୍ୟ ସେଟ୍ କରିପାରିବେ:

1. ଉତ୍ପାଦ ଏଡିଟର୍ ଖୋଲନ୍ତୁ (**Ultimate Multisite → ଉତ୍ପାଦ → ସମ୍ପାଦନା**)
2. **ମୁଦ୍ରା ମୂଲ୍ୟ ନିର୍ଦ୍ଧାରଣ** ବିଭାଗକୁ ସ୍କ୍ରଲ୍ କରନ୍ତୁ
3. ଆପଣ ଓଭରରାଇଡ୍ କରିବାକୁ ଚାହୁଁଥିବା ପ୍ରତ୍ୟେକ ମୁଦ୍ରା ପାଇଁ ସଠିକ୍ ମୂଲ୍ୟ ଦାଖଲ କରନ୍ତୁ
4. ଉତ୍ପାଦ ସେଭ୍ କରନ୍ତୁ

ଓଭରରାଇଡ୍ ନଥିବା ଉତ୍ପାଦଗୁଡ଼ିକ ଆପଣଙ୍କ ମୂଳ ମୁଦ୍ରାରୁ ରୂପାନ୍ତରିତ ମୂଲ୍ୟ ବ୍ୟବହାର କରେ।

### ମୁଦ୍ରା ଚୟନକାରୀ ସାଇନଅପ୍ ଫିଲ୍ଡ

ମୁଦ୍ରା ଚୟନକାରୀ ଫିଲ୍ଡକୁ ଆପଣଙ୍କ ସାଇନଅପ୍/ଚେକଆଉଟ୍ ପ୍ରବାହରେ ଯୋଡ଼ାଯାଇପାରେ, ଯାହାଦ୍ୱାରା ଗ୍ରାହକମାନେ ପଞ୍ଜୀକରଣ ସମୟରେ ନିଜ ପସନ୍ଦର ମୁଦ୍ରା ବାଛନ୍ତି:

1. **Ultimate Multisite → ବହୁ-ମୁଦ୍ରା → ସେଟିଂସ୍** କୁ ଯାଆନ୍ତୁ
2. **ସାଇନଅପ୍ରେ ମୁଦ୍ରା ଚୟନକାରୀ ଦେଖାନ୍ତୁ** ସକ୍ରିୟ କରନ୍ତୁ
3. ଇଚ୍ଛାକୃତ ଭାବେ ଗୋଟିଏ ଡିଫଲ୍ଟ ମୁଦ୍ରା ସେଟ୍ କରନ୍ତୁ କିମ୍ବା ଭୌଗୋଳିକ ଅବସ୍ଥାନକୁ ପୂର୍ବ-ଚୟିତ ମୂଲ୍ୟ ନିର୍ଣ୍ଣୟ କରିବାକୁ ଦିଅନ୍ତୁ

ବାଛାଯାଇଥିବା ମୁଦ୍ରା ସଦସ୍ୟତା ସହ ସଞ୍ଚୟ ହୁଏ ଏବଂ ପରବର୍ତ୍ତୀ ସମସ୍ତ ବିଲିଂ ପାଇଁ ବ୍ୟବହୃତ ହୁଏ।

### ଭୌଗୋଳିକ ଅବସ୍ଥାନ ଆଧାରିତ ମୁଦ୍ରା ଚିହ୍ନଟ

ଭୌଗୋଳିକ ଅବସ୍ଥାନ ଚିହ୍ନଟ ସକ୍ରିୟ ଥିଲେ, ଆଡଅନ୍ ଦର୍ଶକଙ୍କ ଦେଶକୁ ଗୋଟିଏ ମୁଦ୍ରା ସହ ମ୍ୟାପ୍ କରେ ଏବଂ ମୁଦ୍ରା ଚୟନକାରୀ ଫିଲ୍ଡରେ ତାହାକୁ ପୂର୍ବରୁ ବାଛେ। ସକ୍ରିୟ କରିବାକୁ:

1. **Ultimate Multisite → ବହୁ-ମୁଦ୍ରା → ସେଟିଂସ୍** କୁ ଯାଆନ୍ତୁ
2. **ଦର୍ଶକଙ୍କ ଅବସ୍ଥାନରୁ ମୁଦ୍ରା ଚିହ୍ନଟ କରନ୍ତୁ** ସକ୍ରିୟ କରନ୍ତୁ
3. **ଭୌଗୋଳିକ ଅବସ୍ଥାନ ମ୍ୟାପିଂ** ଟେବୁଲ୍‌ରେ ଦେଶଗୁଡ଼ିକୁ ମୁଦ୍ରା ସହ ମ୍ୟାପ୍ କରନ୍ତୁ

କୌଣସି ଦର୍ଶକଙ୍କ ଦେଶ ପାଇଁ ମ୍ୟାପିଂ ନଥିଲେ, ନେଟୱର୍କର ମୂଳ ମୁଦ୍ରାକୁ ଫଲବ୍ୟାକ୍ ଭାବେ ବ୍ୟବହାର କରାଯାଏ।

## ବାରମ୍ବାର ପଚରାଯାଉଥିବା ପ୍ରଶ୍ନ

### ବିନିମୟ ହାର ପ୍ରଦାତା ଉପଲବ୍ଧ ନଥିଲେ କଣ ହୁଏ?

ଆଡଅନ୍ ଶେଷଥର ସଫଳତାର ସହ ଆଣାଯାଇଥିବା ହାରକୁ ଫଲବ୍ୟାକ୍ କରେ। କେବେ ମଧ୍ୟ ହାର ଆଣାଯାଇନଥିଲେ, ଏହା ଆପଣ କନଫିଗର୍ କରିଥିବା ମାନୁଆଲ ହାରକୁ ଫଲବ୍ୟାକ୍ କରେ। ସୁରକ୍ଷା ବ୍ୟବସ୍ଥା ଭାବେ ମାନୁଆଲ ହାର କନଫିଗର୍ କରିବା ସୁପାରିଶ କରାଯାଏ।

### ଭିନ୍ନ ମୁଦ୍ରା ପାଇଁ ମୁଁ ଭିନ୍ନ ପ୍ରଦାତା ବ୍ୟବହାର କରିପାରିବି କି?

ନା — ସମସ୍ତ ମୁଦ୍ରା ପାଇଁ ଏକ ସମୟରେ ଗୋଟିଏ ପ୍ରଦାତା ସକ୍ରିୟ ଥାଏ। ବ୍ୟକ୍ତିଗତ ହାର ଉପରେ ପୂର୍ଣ୍ଣ ନିୟନ୍ତ୍ରଣ ଆବଶ୍ୟକ ହେଲେ **ମାନୁଆଲ** ବ୍ୟବହାର କରନ୍ତୁ।

### ଭୌଗୋଳିକ ଅବସ୍ଥାନ ପାଇଁ ଅଲଗା plugin ଦରକାର କି?

କୌଣସି ବାହ୍ୟ ଭୌଗୋଳିକ ଅବସ୍ଥାନ plugin ଆବଶ୍ୟକ ନାହିଁ। ଆଡଅନ୍ WordPress ର ନିଜସ୍ୱ IP-ରୁ-ଦେଶ ଚିହ୍ନଟ ବ୍ୟବହାର କରେ। ଅଧିକ ସଠିକତା ପାଇଁ, ଦିଆଯାଇଥିବା ଫିଲ୍ଟର୍ ହୁକ୍ ମାଧ୍ୟମରେ ଏକ ନିବିଦିତ ଭୌଗୋଳିକ ଅବସ୍ଥାନ ସେବାକୁ ଇଣ୍ଟିଗ୍ରେଟ୍ କରାଯାଇପାରେ।

### ପ୍ରତ୍ୟେକ ଉତ୍ପାଦ ଓଭରରାଇଡ୍ କର ପୂର୍ବରୁ ଲାଗୁ ହୁଏ କି ପରେ?

ପ୍ରତ୍ୟେକ ଉତ୍ପାଦ ଓଭରରାଇଡ୍ ଲକ୍ଷ୍ୟ ମୁଦ୍ରାରେ କର-ପୂର୍ବ ମୂଲ୍ୟ ସେଟ୍ କରେ। ଆପଣଙ୍କ ଅବସ୍ଥିତ କର କନଫିଗରେସନ୍ ବ୍ୟବହାର କରି ଓଭରରାଇଡ୍ ମୂଲ୍ୟ ଉପରେ କର ଗଣନା କରାଯାଏ।
