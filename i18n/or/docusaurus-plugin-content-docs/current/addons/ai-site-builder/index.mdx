# AI Site Builder {#ai-site-builder}

ultimate-multisite-ai-site-builder

## ଆବଶ୍ୟକତା {#requirements}

- WordPress 5.3 କିମ୍ବା ତାଠାରୁ ଉଚ୍ଚ
- PHP 7.4 କିମ୍ବା ତାଠାରୁ ଉଚ୍ଚ
- Multisite Ultimate plugin (ସକ୍ରିୟ)

## ସଂସ୍ଥାପନ {#installation}

1. addon ଫାଇଲଗୁଡ଼ିକୁ ଆପଣଙ୍କ `/wp-content/plugins/` directory ରେ upload କରନ୍ତୁ
2. WordPress ର 'Plugins' menu ମାଧ୍ୟମରେ plugin ସକ୍ରିୟ କରନ୍ତୁ
3. network admin ରେ addon settings ବିନ୍ୟାସ କରନ୍ତୁ

## ବିକାଶ {#development}

### ସେଟଅପ୍ {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### ପ୍ରକଳ୍ପ ଗଠନ {#project-structure}

```
ultimate-multisite-ai-site-builder/
├── inc/                          # PHP classes ଏବଂ includes
│   ├── class-multisite-ultimate-updater.php  # update handler
│   └── class-ai_site_builder-main.php             # ମୁଖ୍ୟ କାର୍ଯ୍ୟକ୍ଷମତା
├── tests/                        # unit tests
│   ├── bootstrap.php            # test bootstrap
│   ├── class-*-base.php         # ମୂଳ test class
│   └── class-*-test.php         # test cases
├── views/                       # template files
├── assets/                      # static assets
│   ├── css/                     # stylesheets
│   ├── js/                      # JavaScript files
```

### ପରୀକ୍ଷା {#testing}

addon ରେ ଏକ ସମ୍ପୂର୍ଣ୍ଣ testing framework ରହିଛି:

- **Unit Tests**: WordPress test suite ସହିତ PHPUnit-ଆଧାରିତ tests
- **ମୂଳ Test Class**: ସାଧାରଣ test ପରିସ୍ଥିତି ପାଇଁ helper methods ପ୍ରଦାନ କରେ
- **Test Coverage**: code coverage ଟ୍ରାକ୍ କରିବାକୁ ବିନ୍ୟାସିତ
- **Multisite Testing**: multisite ପରିବେଶରେ tests ଚାଲେ

### Code Standards {#code-standards}

- **WordPress Coding Standards**: PHPCS ମାଧ୍ୟମରେ ଲାଗୁ କରାଯାଇଛି
- **PHP 7.4+ ସୁସଙ୍ଗତତା**: ଆଧୁନିକ PHP ବୈଶିଷ୍ଟ୍ୟ ସମର୍ଥିତ
- **Static Analysis**: type checking ପାଇଁ PHPStan integration
- **Code Modernization**: PHP upgrades ପାଇଁ Rector rules

### Build System {#build-system}

- **Asset Processing**: CSS/JS files ପାଇଁ minification
- **ଅନୁବାଦ**: POT file ସୃଷ୍ଟି
- **Packaging**: distribution ପାଇଁ archive ସୃଷ୍ଟି
- **Development Mode**: debugging ପାଇଁ unminified assets

## ବିନ୍ୟାସ {#configuration}

addon ବିଭିନ୍ନ configuration options ସମର୍ଥନ କରେ, ଯାହା network admin interface ମାଧ୍ୟମରେ ପହଞ୍ଚିହେବ।

## Hooks ଏବଂ Filters {#hooks-and-filters}

### Actions {#actions}

- `ultimate-multisite-ai-site-builder_init` - addon initialization ପରେ fired ହୁଏ
- `ultimate-multisite-ai-site-builder_loaded` - ସମସ୍ତ addon files loaded ହେବା ପରେ fired ହୁଏ

### Filters {#filters}

- `ultimate-multisite-ai-site-builder_settings` - addon settings ପରିବର୍ତ୍ତନ କରନ୍ତୁ
- `ultimate-multisite-ai-site-builder_enabled` - addon ସକ୍ରିୟ/ନିଷ୍କ୍ରିୟ ସ୍ଥିତି override କରନ୍ତୁ
