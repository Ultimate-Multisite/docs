# Admin Page Creator

## ବିଷୟବସ୍ତୁ
Admin Page Creator ଆପଣଙ୍କୁ WordPress admin dashboard ଭିତରେ ସହଜରେ ନୂଆ admin ପୃଷ୍ଠା ଏବଂ ଉପ-ପୃଷ୍ଠା ଯୋଡ଼ିବାକୁ ଦେବ। ଏହାକୁ ନୂଆ ବ୍ୟବହାରକାରୀଙ୍କ ପାଇଁ ଲ୍ୟାଣ୍ଡିଂ ପୃଷ୍ଠା ଭାବେ କାମ କରିପାରୁଥିବା ସ୍ୱାଗତ ପୃଷ୍ଠା କିମ୍ବା ଟ୍ୟୁଟୋରିଆଲ୍ ପୃଷ୍ଠା ତିଆରି କରିବାରେ ବ୍ୟବହାର କରିପାରିବେ। ଅସୁବିଧା ବିନା ପୃଷ୍ଠାଗୁଡ଼ିକୁ ଡିଜାଇନ୍ କରିବାକୁ Beaver Builder, Elementor, Brizy, ଏବଂ Oxygen ପରି page builder ବ୍ୟବହାର କରିପାରିବେ।

ଏହା custom HTML, CSS ଏବଂ JavaScript ସମର୍ଥନ ସହିତ plan ଏବଂ user role ଆଧାରିତ access control ମଧ୍ୟ ଦେଇଥାଏ!

## ବୈଶିଷ୍ଟ୍ୟ
* Custom HTML ଏବଂ CSS ସମର୍ଥନ
* JavaScript ସମର୍ଥନ
* PHP Content Mode
* User role ଆଧାରିତ ଅନୁମତି
* page builder ଗୁଡ଼ିକୁ ସମର୍ଥନ

2. ଆପଣଙ୍କ WordPress Network Admin ର **Plugins** ମେନୁ ମାଧ୍ୟମରେ plugin କୁ ସକ୍ରିୟ କରନ୍ତୁ

## ବ୍ୟବହାର ଗାଇଡ୍

WordPress ସାଇଟ୍‌ଗୁଡ଼ିକର front end ରେ ବିଷୟବସ୍ତୁ ଯୋଡ଼ିବାକୁ ଖୁବ ସହଜ କରିଦେଇଥିବାବେଳେ, admin ପାର୍ଶ୍ୱରେ ପୃଷ୍ଠା ଏବଂ ଉପ-ପୃଷ୍ଠା ତିଆରି କରିବାକୁ କିଛି ମୂଳ coding ଦକ୍ଷତା ଦରକାର ହୁଏ। Ultimate Multisite: Admin Page Creator ଆପଣଙ୍କ network sites ର back end କୁ front end ଭଳି ସହଜ ବ୍ୟବହାର ଅନୁଭବ ଦେବାକୁ ଲକ୍ଷ୍ୟ କରେ।

## Admin Page Creator କାହିଁକି ବ୍ୟବହାର କରିବେ?

ହୋଇପାରେ ଆପଣ ଆପଣଙ୍କ clients ଙ୍କ admin panel ରେ ଟ୍ୟୁଟୋରିଆଲ୍ ସହିତ ଏକ custom ପୃଷ୍ଠା ଦେବାକୁ ଚାହୁଁଛନ୍ତି, କିମ୍ବା FAQs ଏବଂ contact widget ସହିତ ଏକ custom ପୃଷ୍ଠା ଯୋଡ଼ିବାକୁ ଚାହୁଁଛନ୍ତି। Admin Page Creator ସହିତ ସମ୍ଭାବନାଗୁଡ଼ିକ ଅସୀମ, ଏବଂ ଆଜିର ଟ୍ୟୁଟୋରିଆଲ୍‌ରେ ଆମେ ସେହିଥିକେ ଦେଖିବା।

## ଆପଣଙ୍କ ପ୍ରଥମ Custom Admin Page ତିଆରି କରିବା

Ultimate Multisite: Admin Page Creator ସଂସ୍ଥାପନ ଏବଂ ସକ୍ରିୟ କରିବା ପରେ, ଆପଣଙ୍କ network admin menu କୁ ଯାଇ Admin Pages କୁ ଯାଆନ୍ତୁ।

![Network Admin ରେ Custom Admin Pages ତାଲିକା](/img/admin/admin-pages-list.png)

_Network Admin ରେ Custom Admin Pages ତାଲିକା_

ଆପଣ ଏପର୍ଯ୍ୟନ୍ତ ତିଆରି କରିଥିବା ସମସ୍ତ admin ପୃଷ୍ଠାର ତାଲିକା ଦେଖିବେ (ଏହା ଆପଣଙ୍କ ପ୍ରଥମ ଭ୍ରମଣ ଥିବାରୁ, ତାଲିକା ଖାଲି ରହିବ)। ଆଗକୁ ବଢ଼ନ୍ତୁ ଏବଂ **Add new Admin Page** ଉପରେ କ୍ଲିକ୍ କରନ୍ତୁ। ଆପଣଙ୍କୁ Admin Page editor କୁ ପୁନର୍ନିର୍ଦ୍ଦେଶିତ କରାଯିବ।

## Custom Admin Page ର ଉପାଦାନଗୁଡ଼ିକ

Admin Page editor ରେ, ଆପଣଙ୍କ ପୃଷ୍ଠାକୁ customize କରିବା ପାଇଁ ଅନେକ ଭିନ୍ନ ବିକଳ୍ପ ରହିବ।

ଏହା ପୃଷ୍ଠାର ଉପରେ ଦେଖାଯିବା ଶୀର୍ଷକ, admin menu label ନୁହେଁ।

Ultimate Multisite: Admin Page Creator ପୃଷ୍ଠାର ବିଷୟବସ୍ତୁ ପାଇଁ ଆପଣଙ୍କୁ ଦୁଇଟି ବିକଳ୍ପ ଦେଇଥାଏ: ଆପଣ default WordPress WYSIWYG editor ବ୍ୟବହାର କରିପାରିବେ କିମ୍ବା HTML editor କୁ ବଦଳାଇପାରିବେ।

## ମେନୁ ବିକଳ୍ପ

ପାର୍ଶ୍ୱରେ, General Options meta-box ରେ, ଆପଣ menu type, label, ଏବଂ ଆଉ ବହୁତ କିଛି configure କରିପାରିବେ।

### ମେନୁ ପ୍ରକାର

ଆପଣଙ୍କ ନୂଆ custom admin ପୃଷ୍ଠାରେ ଦୁଇଟି ଭିନ୍ନ menu type ରହିପାରେ: top-level menu ଏବଂ submenu।

ନାମରୁ ବୁଝାଯାଉଥିବା ପରି, top-level menu page ଆପଣଙ୍କ ନୂଆ ତିଆରି ହୋଇଥିବା ପୃଷ୍ଠାକୁ default WordPress ପୃଷ୍ଠାଗୁଡ଼ିକ ସହିତ admin menu ରେ ଯୋଡ଼ିଦେବ।

ଆପଣ top-level ବିକଳ୍ପ ବାଛିଲେ, menu order ଏବଂ menu icon ମଧ୍ୟ ବାଛିପାରିବେ।

ଏହି ପୃଷ୍ଠାକୁ sub-menu page କରିବାକୁ ଚାହୁଁଥିଲେ, ଏହି ପୃଷ୍ଠାକୁ ଯୋଡ଼ିବା ପାଇଁ ଏକ parent page ବାଛିପାରିବେ। ଆପଣ default WordPress ପୃଷ୍ଠାଗୁଡ଼ିକୁ କିମ୍ବା ଆପଣଙ୍କ top-level custom admin ପୃଷ୍ଠାଗୁଡ଼ିକୁ ଆପଣଙ୍କ custom ପୃଷ୍ଠା ଯୋଡ଼ିପାରିବେ।

## ଉନ୍ନତ ବିକଳ୍ପ

ଆପଣ ଆପଣଙ୍କ custom admin ପୃଷ୍ଠାଗୁଡ଼ିକୁ ସମସ୍ତ ପ୍ରକାରର ବିଷୟବସ୍ତୁ ଯୋଡ଼ିପାରିବା ଉଚିତ। ଏଥିରେ CSS rules ଏବଂ files, JavaScript, ଏବଂ external libraries ମଧ୍ୟ ଅନ୍ତର୍ଭୁକ୍ତ। Advanced Options Tab ବ୍ୟବହାର କରି ଆପଣ ଏହା କରିପାରିବେ।

### Custom CSS

CSS tab ରେ, ଆପଣ ନିଜ custom CSS code ଯୋଡ଼ିପାରିବେ, ସେହିପରି remote sites ରୁ CSS files import କରିପାରିବେ (CDNs, Google Fonts, ଇତ୍ୟାଦି ପରି)।

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_Custom CSS ମଧ୍ୟ ସମର୍ଥିତ। ଆପଣ external style files ମଧ୍ୟ ସାମିଲ କରିପାରିବେ!_

### Custom JavaScript

JavaScript ପାଇଁ ମଧ୍ୟ ଏହା ଲାଗୁ ହୁଏ। ଆପଣ ଆପଣଙ୍କ custom JavaScript code ରେ ବ୍ୟବହାର କରିବାକୁ external libraries ଯୋଡ଼ିପାରିବେ।

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_ଆପଣଙ୍କ ନିଜ JavaScript code ଯୋଡ଼ନ୍ତୁ!_

## ଅନୁମତି

Advanced Options ରେ ମଧ୍ୟ, କେଉଁ sub-sites ଏହି ପୃଷ୍ଠାକୁ access ପାଇବେ ତାହା ଆପଣ ବାଛିପାରିବେ। ଉଦାହରଣ ସ୍ୱରୂପ, ଆପଣ ଭିନ୍ନ plan tiers ପାଇଁ ଭିନ୍ନ exclusive content ଦେବାକୁ ଯୋଜନା କରୁଥିଲେ ଏହା ଉପଯୋଗୀ।

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_ଏହି ପୃଷ୍ଠା କେଉଁଠି ଦେଖାଯିବ, ତାହାର ଆପଣଙ୍କ ପାଖରେ ସମ୍ପୂର୍ଣ୍ଣ ନିୟନ୍ତ୍ରଣ ଅଛି_

**ଗୁରୁତ୍ୱପୂର୍ଣ୍ଣ** : ପୃଷ୍ଠା ଦେଖାଯିବା ପାଇଁ ଉଭୟ ସର୍ତ୍ତ ପୂରଣ ହେବା ଆବଶ୍ୟକ। ଅର୍ଥାତ୍, ଯଦି ବ୍ୟବହାରକାରୀ କୌଣସି ନିର୍ଦ୍ଦିଷ୍ଟ ଯୋଜନାର client ହେଲେ ମଧ୍ୟ, ଚୟନିତ ଭୂମିକାମାନଙ୍କ ମଧ୍ୟରୁ ଗୋଟିଏ ନଥାଏ (କିମ୍ବା ତାହାର ଉଲ୍ଟା), ତେବେ ପୃଷ୍ଠା ତାଙ୍କ panel ରେ ଦେଖାଯିବ ନାହିଁ।

## ସକ୍ରିୟ

ଯଦି ଆପଣ କୌଣସି custom admin ପୃଷ୍ଠାକୁ ହଟାଇବା ବିନା କିମ୍ବା permissions ସହ ଅସୁବିଧା କରିବା ବିନା disable କରିବାକୁ ଚାହୁଁଛନ୍ତି, କେବଳ Active ବିକଳ୍ପକୁ ଅନଚେକ୍ କରନ୍ତୁ।

ଟପ୍-ଲେଭେଲ୍ ପୃଷ୍ଠାକୁ ନିଷ୍କ୍ରିୟ କଲେ, ସେହି ମୂଳ ପୃଷ୍ଠା ସହ ଯୋଡ଼ା ଥିବା ସମସ୍ତ ଉପ-ପୃଷ୍ଠା ସ୍ୱୟଂଚାଳିତ ଭାବେ ନିଷ୍କ୍ରିୟ ହେବ।
