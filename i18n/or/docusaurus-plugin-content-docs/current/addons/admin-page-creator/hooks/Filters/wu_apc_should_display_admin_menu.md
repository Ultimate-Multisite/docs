# Filter: wu_apc_should_display_admin_menu {#filter-wuapcshoulddisplayadminmenu}

ପ୍ରଶାସକମାନେ ଅନୁକୂଳନ ପ୍ରକ୍ରିୟା ସମାପ୍ତ କରିଥିଲେ, ସେମାନଙ୍କୁ ପ୍ରଶାସନ ମେନୁ ପୃଷ୍ଠାଗୁଡ଼ିକୁ ଲୁଚାଇବାକୁ ଅନୁମତି ଦିଅନ୍ତୁ।
