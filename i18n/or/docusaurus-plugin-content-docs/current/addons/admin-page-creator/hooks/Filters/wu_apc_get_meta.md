# ଫିଲ୍ଟର: wu_apc_get_meta

ଡେଭେଲପରମାନଙ୍କୁ ଶେଷ ମୂଲ୍ୟଗୁଡ଼ିକୁ ଫିଲ୍ଟର କରିବାକୁ ଏବଂ ନୂତନ ପ୍ରକ୍ରିୟାକରଣ ମାମଲା ଯୋଡ଼ିବାକୁ ଦିଅନ୍ତୁ।

## ପାରାମିଟର

| ନାମ | ପ୍ରକାର | ବର୍ଣ୍ଣନା |
|------|------|-------------|
| $default_value | `string` | ପୂର୍ବନିର୍ଦ୍ଧାରିତ ମୂଲ୍ୟ। |
| $meta_type | `string` | ଆଣିବାକୁ ଥିବା ମେଟା ତଥ୍ୟର ପ୍ରକାର। |
| $meta_key | `string` | ମେଟା ଫିଲ୍ଡର ନାମ। |
