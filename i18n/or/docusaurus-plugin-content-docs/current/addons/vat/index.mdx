# VAT {#vat}

Multisite Ultimate: ଇଉରୋପୀୟ VAT

ଇଉରୋପୀୟ VAT କୁ ଭଲଭାବେ ସମର୍ଥନ କରିବା ପାଇଁ Multisite Ultimate ର ଡିଫଲ୍ଟ କର ପରିଚାଳନାରେ ଅତିରିକ୍ତ ବୈଶିଷ୍ଟ୍ୟ ଓ ଉପକରଣ ଯୋଗ କରେ।
ଏଠାରୁ କର ହାର ପୃଷ୍ଠାରେ EU VAT ହାର ସ୍ୱୟଂଚାଳିତ ଭାବେ ଯୋଗ କରେ

## ସଂସ୍ଥାପନ {#installation}

1. 'ultimate-multisite-vat' କୁ '/wp-content/plugins/' ଡାଇରେକ୍ଟୋରୀକୁ ଅପଲୋଡ୍ କରନ୍ତୁ
2. WordPress ର 'ପ୍ଲଗଇନଗୁଡ଼ିକ' ମେନୁ ମାଧ୍ୟମରେ ପ୍ଲଗଇନକୁ ସକ୍ରିୟ କରନ୍ତୁ
3. Ultimate Multisite > ସେଟିଂସ୍ > କର କୁ ଯାଇ EU Vat ସକ୍ରିୟ କରନ୍ତୁ।
4. କର ହାର ଉପରେ କ୍ଲିକ୍ କରନ୍ତୁ, ତଳେ EU VAT ହାର ଅଦ୍ୟତନ କରନ୍ତୁ ଏବଂ କର ହାର ସଞ୍ଚୟ କରନ୍ତୁ।
5. ସମାପ୍ତ!
