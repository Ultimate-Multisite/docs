# ଫିଲ୍ଟର: networks_pre_user_is_network_admin {#filter-networkspreuserisnetworkadmin}

ଜଣେ ବ୍ୟବହାରକାରୀ ଯେଉଁ networks ର ପ୍ରଶାସକ, ପ୍ରକ୍ରିୟାକୁ ଆଗରୁ ବନ୍ଦ କରିବା ପାଇଁ ସେଗୁଡ଼ିକୁ ଫିଲ୍ଟର କରେ।

## ପାରାମିଟର {#parameters}

| ନାମ | ପ୍ରକାର | ବିବରଣୀ |
|------|------|-------------|
| $ | `array\|bool\|null` | network ID ଗୁଡ଼ିକର ତାଲିକା କିମ୍ବା false। null ଛାଡ଼ି ଅନ୍ୟ କିଛି ହେଲେ ପ୍ରକ୍ରିୟା ଆଗରୁ ବନ୍ଦ ହେବ। |
| $ | `int` | ଯେଉଁ ବ୍ୟବହାରକାରୀ ID ପାଇଁ networks ଫେରାଇବା ଉଚିତ। |
