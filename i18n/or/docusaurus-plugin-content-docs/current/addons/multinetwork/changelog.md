# ବହୁ-ନେଟୱାର୍କ ପରିବର୍ତ୍ତନ ତାଲିକା

### 1.3.0 {#130}
* ନୂଆ: ନେଟୱାର୍କ ଟେମ୍ପଲେଟ୍ ପ୍ରିଭ୍ୟୁଅର୍ — କିଣିବା ପୂର୍ବରୁ ଲାଇଭ୍ ପ୍ୟାନେଲ୍‌ରେ ନେଟୱାର୍କ ଟେମ୍ପଲେଟ୍‌ଗୁଡ଼ିକୁ ଖୋଜନ୍ତୁ ଏବଂ ପ୍ରିଭ୍ୟୁ କରନ୍ତୁ।
* ନୂଆ: ଚେକଆଉଟ୍‌ରେ ନେଟୱାର୍କ ଟେମ୍ପଲେଟ୍ ପିକର୍ — ସାଇନଅପ୍ ଚେକଆଉଟ୍ ପ୍ରବାହ ସମୟରେ ଗ୍ରାହକମାନେ ନେଟୱାର୍କ ଟେମ୍ପଲେଟ୍ ବାଛିପାରିବେ।
* ନୂଆ: କ୍ରସ୍-ନେଟୱାର୍କ magic link SSO — ଗ୍ରାହକମାନେ magic-link ପ୍ରମାଣୀକରଣ ମାଧ୍ୟମରେ ଉପ-ନେଟୱାର୍କଗୁଡ଼ିକୁ ସହଜରେ ପ୍ରବେଶ କରିପାରିବେ।
* ନୂଆ: ନେଟୱାର୍କ ମିଡିଆ କ୍ଲୋନିଂ — ଟେମ୍ପଲେଟ୍‌ରୁ ନେଟୱାର୍କ କ୍ଲୋନ୍ କରିବାବେଳେ ସାଇଟ୍ ମିଡିଆ ସଠିକ୍ ଭାବରେ ନକଲ୍ ହୁଏ।
* ସଂଶୋଧନ: ବହୁ-ଟେନାନ୍ସି ଆଡଅନ୍ ବିନା ମଧ୍ୟ ନେଟୱାର୍କ କ୍ଲୋନିଂ ଏବେ କାମ କରେ; କ୍ଲୋନ୍ ପରେ ସୁପର୍ ଆଡମିନ୍ ଅଧିକାର ପୁଣି ଦିଆଯାଏ।
* ସଂଶୋଧନ: ନେଟୱାର୍କ ସୃଷ୍ଟି ଏବେ WooCommerce ଏବଂ ସମସ୍ତ ପେମେଣ୍ଟ ଗେଟୱେ ସହ ସୁସଙ୍ଗତ।
* ସଂଶୋଧନ: ଫଲବ୍ୟାକ୍ ଡୋମେନ୍ ତିଆରିରେ ପୋର୍ଟ ନମ୍ବର ଏବେ ସାମିଲ୍ କରାଯାଏ।
* ସଂଶୋଧନ: ସମସ୍ତ ଥିମ୍‌ରେ ଠିକ୍ ରେଣ୍ଡରିଂ ପାଇଁ ଟେମ୍ପଲେଟ୍ ପିକର୍ CSS କୁ ଏନ୍‌କ୍ୟୁ ହୋଇଥିବା stylesheet କୁ ସରାଯାଇଛି।
* ସଂଶୋଧନ: ସାଇଟ୍ କ୍ୱେରି ସ୍କୋପ୍‌ରେ ଅନିର୍ଦ୍ଧାରିତ `network_id` କୀ ବିରୁଦ୍ଧରେ ସୁରକ୍ଷା।
* ସଂଶୋଧନ: ପ୍ରସଙ୍ଗ ବଦଳାଇବା ପୂର୍ବରୁ ନେଟୱାର୍କ ଅନୁମତି ଯାଞ୍ଚ କରିବା ଦ୍ୱାରା `wpdb` ଟେବୁଲ୍ ପ୍ରୋପର୍ଟି ତ୍ରୁଟି ରୋକାଯାଏ।
