# ପ୍ଲଗଇନ ଓ ଥିମ ପରିଚାଳକ

Multisite Ultimate: ପ୍ଲଗଇନ ଓ ଥିମ ପରିଚାଳକ

ଇନଷ୍ଟଲ୍ ହୋଇଥିବା ପ୍ଲଗଇନ ଓ ଥିମର ମେଟା ସୂଚନା (ଶୀର୍ଷକ, ଲେଖକ, ଥମ୍ବନେଲ୍ ଓ ବିବରଣୀ) ସମ୍ପାଦନ କରନ୍ତୁ କିମ୍ବା ଲୁଚାନ୍ତୁ। ଆପଣଙ୍କ ବ୍ୟବହାରକାରୀମାନେ ସେଗୁଡ଼ିକୁ ଫିଲ୍ଟର୍ କରିପାରିବେ ବୋଲି ପ୍ଲଗଇନ ଓ ଥିମ ପାଇଁ ବର୍ଗ ସୃଷ୍ଟି କରନ୍ତୁ, ଏବଂ ଆପଣଙ୍କ ବ୍ୟବହାରକାରୀଙ୍କ ପ୍ୟାନେଲ୍‌ରେ ଏକ ସୁନ୍ଦର କଷ୍ଟମ୍ ପ୍ଲଗଇନ ପୃଷ୍ଠା ଯୋଡ଼ନ୍ତୁ।

ପ୍ଲଗଇନ ବ୍ୟବହାର କରିବାକୁ, ଆପଣ ନେଟୱର୍କ ପ୍ରଶାସନରେ ଥିବା ଆପଣଙ୍କ ପ୍ଲଗଇନ କିମ୍ବା ଥିମ ପୃଷ୍ଠାକୁ ଯାଇ "Edit Plugin/Theme info" ଉପରେ କ୍ଲିକ୍ କରିପାରିବେ।

## ଇନଷ୍ଟଲେସନ୍ {#installation}

1. 'ultimate-multisite-plugin-and-theme-manager' କୁ '/wp-content/plugins/' ଡିରେକ୍ଟୋରିକୁ ଅପଲୋଡ୍ କରନ୍ତୁ
2. ଆପଣଙ୍କ WordPress ନେଟୱର୍କ ପ୍ରଶାସନରେ 'Plugins' ମେନୁ ମାଧ୍ୟମରେ ପ୍ଲଗଇନକୁ ସକ୍ରିୟ କରନ୍ତୁ

## ବ୍ୟବହାର ମାର୍ଗଦର୍ଶିକା {#usage-guide}

Ultimate Multisite: ପ୍ଲଗଇନ ଓ ଥିମ ପରିଚାଳକ ହେଉଛି ଏକ ଶକ୍ତିଶାଳୀ **ପ୍ରିମିୟମ୍** Ultimate Multisite ଆଡ୍-ଅନ୍, ଯାହା ଆପଣଙ୍କ ନେଟୱର୍କ ବ୍ୟବହାରକାରୀମାନେ ଆପଣଙ୍କ ନେଟୱର୍କରେ ପ୍ଲଗଇନ ଓ ଥିମ ବିକଳ୍ପଗୁଡ଼ିକୁ କିପରି ଦେଖନ୍ତି ଓ ସେମାନଙ୍କ ସହ କିପରି ଆନ୍ତର୍କ୍ରିୟା କରନ୍ତି, ତାହା ଉପରେ ଆପଣଙ୍କୁ ପୂର୍ଣ୍ଣ ନିୟନ୍ତ୍ରଣ ଦିଏ।

ଏହି ଆଡ୍-ଅନ୍ ତିନୋଟି ଉଦ୍ଦେଶ୍ୟ ପୂରଣ କରେ:

  1. ଏହା ଆପଣଙ୍କୁ ପ୍ଲଗଇନ ଓ ଥିମ ଉଭୟର ମେଟାଡାଟାକୁ ସମ୍ପୂର୍ଣ୍ଣ ଭାବେ ବଦଳାଇବାକୁ ଦିଏ। ଏଥିରେ ନାମ, ବିବରଣୀ, ଲେଖକଙ୍କ ନାମ, ଏବଂ ଥମ୍ବନେଲ୍ ପର୍ଯ୍ୟନ୍ତ ଅନ୍ତର୍ଭୁକ୍ତ!

  2. ଏହା ଆପଣଙ୍କୁ ପ୍ଲଗଇନ ଓ ଥିମଗୁଡ଼ିକୁ ବର୍ଗୀକୃତ କରିବାକୁ ଦିଏ, ଯାହାଦ୍ୱାରା ଆପଣଙ୍କ ବ୍ୟବହାରକାରୀମାନେ ସହଜରେ ସେଗୁଡ଼ିକୁ ସଜାଇପାରିବେ।

  3. ଏହା ଆପଣଙ୍କୁ ପୁରୁଣା, ନିରସ ପ୍ଲଗଇନ ପୃଷ୍ଠାକୁ ବର୍ଗ ଅନୁସାରେ ସଜାଯାଇଥିବା ଏକ ନୂଆ ଥିମ-ଭଳି ପୃଷ୍ଠା ସହିତ ପରିବର୍ତ୍ତନ କରିବାର ବିକଳ୍ପ ଦିଏ।

ଏହାକୁ ଡାଉନଲୋଡ୍ ଓ ଇନଷ୍ଟଲ୍ କରିବା ପରେ, Ultimate Multisite Settings ପୃଷ୍ଠାରେ ଏକ ନୂଆ ଟ୍ୟାବ୍ - _ଆଡ୍-ଅନ୍ ସେଟିଂସ୍_ \- ଦେଖାଯିବ।

_Ultimate Multisite: ପ୍ଲଗଇନ ଓ ଥିମ ପରିଚାଳକ ଦ୍ୱାରା ଯୋଡ଼ାଯାଇଥିବା ବିଶ୍ୱସ୍ତରୀୟ ବିକଳ୍ପଗୁଡ଼ିକ ଏଠାରେ ମିଳିବ_

ଆପଣଙ୍କ ନେଟୱର୍କ _ପ୍ଲଗଇନ_ ପୃଷ୍ଠାରେ ଏବେ ଆପଣ ସମସ୍ତ ପ୍ଲଗଇନର ମେଟା ସୂଚନା, ଯେପରିକି ନାମ, ଲେଖକ, ଏବଂ ପ୍ଲଗଇନର ଥମ୍ବନେଲ୍ ପର୍ଯ୍ୟନ୍ତ ସମ୍ପାଦନ କରିପାରିବେ।

_ନିର୍ଦ୍ଦିଷ୍ଟ ପ୍ଲଗଇନ ସୂଚନା ଦେଖାଇବେ କି ନାହିଁ, ତାହା ମଧ୍ୟ ଆପଣ ବାଛିପାରିବେ_

ପ୍ଲଗଇନ ଓ ଥିମ ପରିଚାଳକ ଦୁଇ ପ୍ରକାର ଦେଖାଣି ଦେଉଛି: ପ୍ଲଗଇନ ଶୈଳୀ ଓ ଥିମ ଶୈଳୀ। ତେଣୁ, ଆପଣଙ୍କ ଆବଶ୍ୟକ ସମନ୍ୱୟ କରିସାରିବା ପରେ, ଆପଣଙ୍କ ଗ୍ରାହକମାନେ ତାଙ୍କର _ପ୍ଲଗଇନ_ ପୃଷ୍ଠାକୁ ଏଭଳି ଦେଖିବେ:

_ପ୍ଲଗଇନ ଶୈଳୀ ଦେଖାଉଥିବା ଗ୍ରାହକଙ୍କ ପ୍ଲଗଇନ ପୃଷ୍ଠା_
