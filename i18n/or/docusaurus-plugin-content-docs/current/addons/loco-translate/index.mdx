# Loco Translate {#loco-translate}

Ultimate Multisite: Loco Translate Integration ଆଡଅନ ଆପଣଙ୍କ WordPress ମଲ୍ଟିସାଇଟ ନେଟୱର୍କକୁ ସାଇଟ-ନିର୍ଦ୍ଦିଷ୍ଟ ଅନୁବାଦ କ୍ଷମତା ଆଣେ। ଯଦି ଆପଣଙ୍କୁ କେବେ ନେଟୱର୍କର ଭିନ୍ନ ସାଇଟଗୁଡ଼ିକରେ ସମାନ ପ୍ଲଗଇନ କିମ୍ବା ଥିମର ଭିନ୍ନ ଅନୁବାଦ ଦରକାର ହୋଇଥାଏ, ଏହି ଆଡଅନ ସଠିକ୍ ସମାଧାନ।

**ଏହି ଆଡଅନ କଣ କରେ**

ଏହି ଆଡଅନ Ultimate Multisite କୁ ଲୋକପ୍ରିୟ Loco Translate ପ୍ଲଗଇନ ସହ ସହଜରେ ଏକତ୍ର କରେ, ଯାହା ଦ୍ୱାରା ଆପଣଙ୍କ ନେଟୱର୍କର ପ୍ରତ୍ୟେକ ସବସାଇଟ ନିଜର କଷ୍ଟମ ଅନୁବାଦ ରଖିପାରେ। ଆପଣ ବହୁ-ଅଞ୍ଚଳୀୟ ପ୍ଲାଟଫର୍ମ ଚାଲାଉଛନ୍ତି, ଭିନ୍ନ ଭାଷା ସମୁଦାୟକୁ ସେବା ଦେଉଛନ୍ତି, କିମ୍ବା କେବଳ ନେଟୱର୍କରେ ଅନୁବାଦ ଲଚିଳାପଣ ଦରକାର—ଏହି ଆଡଅନ ଏହାକୁ ସହଜ କରେ।

**ଏହା ପାଇଁ ଉପଯୁକ୍ତ:**

* ଭିନ୍ନ ସ୍ଥାନୀୟକରଣ ଆବଶ୍ୟକତା ଥିବା ବହୁ-ଅଞ୍ଚଳୀୟ ବ୍ୟବସାୟ
* ବିବିଧ ସମୁଦାୟକୁ ସେବା ଦେଉଥିବା ଶିକ୍ଷାନୁଷ୍ଠାନ
* ବହୁଭାଷୀ ସହାୟତା ଦେଉଥିବା SaaS ପ୍ଲାଟଫର୍ମ
* ଅନେକ କ୍ଲାଇଏଣ୍ଟ ସାଇଟ ପରିଚାଳନା କରୁଥିବା ୱେବ ଏଜେନ୍ସି
* ଅନୁବାଦ ଲଚିଳାପଣ ଦରକାର ଥିବା ଯେକୌଣସି ମଲ୍ଟିସାଇଟ ନେଟୱର୍କ

## ସ୍ଥାପନ {#installation}

1. Ultimate Multisite ସ୍ଥାପିତ ଏବଂ ନେଟୱର୍କରେ ସକ୍ରିୟ ଅଛି ବୋଲି ନିଶ୍ଚିତ କରନ୍ତୁ
2. Loco Translate ପ୍ଲଗଇନ ସ୍ଥାପନ କରି ସକ୍ରିୟ କରନ୍ତୁ
3. ଏହି ଆଡଅନକୁ ଆପଣଙ୍କ ପ୍ଲଗଇନ ଡିରେକ୍ଟରୀକୁ ଅପଲୋଡ କରନ୍ତୁ
4. Loco Translate Integration ଆଡଅନକୁ ନେଟୱର୍କରେ ସକ୍ରିୟ କରନ୍ତୁ
5. ଯେକୌଣସି ସବସାଇଟରେ Loco Translate ମାଧ୍ୟମରେ ସାଇଟ-ନିର୍ଦ୍ଦିଷ୍ଟ ଅନୁବାଦ ତିଆରି କରିବା ଆରମ୍ଭ କରନ୍ତୁ

ଏତିକି! ଏହି ଏକତ୍ରୀକରଣ କୌଣସି ଅତିରିକ୍ତ ବିନ୍ୟାସ ଛଡ଼ା ସ୍ୱୟଂଚାଳିତ ଭାବେ କାମ କରେ।

## ବାରମ୍ବାର ପଚରାଯାଉଥିବା ପ୍ରଶ୍ନ {#frequently-asked-questions}

## ସ୍ଥାପନ ପରେ ମୋତେ କିଛି ବିନ୍ୟାସ କରିବାକୁ ପଡ଼ିବ କି? {#do-i-need-to-configure-anything-after-installation}

ନା! ଏକତ୍ରୀକରଣ ସ୍ୱୟଂଚାଳିତ। ଆଡଅନ ଏବଂ Loco Translate ଉଭୟ ସକ୍ରିୟ ହେଲେ, ସବସାଇଟଗୁଡ଼ିକ ସତେଜ ଭାବେ ନିଜର କଷ୍ଟମ ଅନୁବାଦ ତିଆରି କରିପାରିବେ।

## କୌଣସି ସବସାଇଟ କଷ୍ଟମ ଅନୁବାଦ ତିଆରି ନ କଲେ କଣ ହେବ? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

ସାଇଟ ସ୍ୱୟଂଚାଳିତ ଭାବେ ନେଟୱର୍କ-ବ୍ୟାପୀ ଅନୁବାଦକୁ ପର୍ଯ୍ୟାୟ ବିକଳ୍ପ ଭାବେ ବ୍ୟବହାର କରିବ। କିଛି ଭାଙ୍ଗିବ ନାହିଁ, ଏବଂ ସବୁକିଛି ସହଜରେ କାମ କରିବ।

## ସବସାଇଟଗୁଡ଼ିକ ନେଟୱର୍କ-ବ୍ୟାପୀ ଅନୁବାଦକୁ ପହଞ୍ଚିପାରିବେ କି? {#can-subsites-access-network-wide-translations}

ସାଧାରଣ ସାଇଟ ପ୍ରଶାସକମାନେ କେବଳ ନିଜ ସାଇଟର ଅନୁବାଦ ଦେଖିବେ ଏବଂ ସମ୍ପାଦନା କରିବେ। ସୁପର ପ୍ରଶାସକମାନେ କେଉଁ ସାଇଟ ଦେଖୁଛନ୍ତି ତାହା ଅନୁସାରେ ନେଟୱର୍କ-ବ୍ୟାପୀ ଏବଂ ସାଇଟ-ନିର୍ଦ୍ଦିଷ୍ଟ ଅନୁବାଦ ଉଭୟକୁ ପହଞ୍ଚିପାରିବେ।

## ଏହା ପ୍ଲଗଇନ ଏବଂ ଥିମ ଉଭୟ ସହ କାମ କରେ କି? {#does-this-work-with-both-plugins-and-themes}

ହଁ! ସାଇଟ-ନିର୍ଦ୍ଦିଷ୍ଟ ଅନୁବାଦ WordPress ପ୍ଲଗଇନ ଏବଂ ଥିମ ଉଭୟ ପାଇଁ କାମ କରେ।

## ଏହା ମୋର ମଲ୍ଟିସାଇଟ ନେଟୱର୍କକୁ ଧୀର କରିଦେବ କି? {#will-this-slow-down-my-multisite-network}

ନା। ଆଡଅନ କାର୍ଯ୍ୟକ୍ଷମ ଫାଇଲ ଲୋଡିଂ ବ୍ୟବହାର କରେ, ଯାହା କେବଳ ଦରକାର ପଡ଼ିଲେ ସାଇଟ-ନିର୍ଦ୍ଦିଷ୍ଟ ଅନୁବାଦ ଯାଞ୍ଚ କରେ। କାର୍ଯ୍ୟଦକ୍ଷତାରେ ଧ୍ୟାନଯୋଗ୍ୟ ପ୍ରଭାବ ନାହିଁ।

## ଆରମ୍ଭିକ ବିନ୍ଦୁ ଭାବେ ମୁଁ ନେଟୱର୍କ ଅନୁବାଦକୁ ନିର୍ଦ୍ଦିଷ୍ଟ ସାଇଟକୁ କପି କରିପାରିବି କି? {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

ହଁ! Loco Translate ରେ ଅନୁବାଦ ସମ୍ପାଦନା କରୁଥିବାବେଳେ, ଆପଣ ସହଜରେ ନେଟୱର୍କ ଅନୁବାଦରୁ ଆରମ୍ଭ କରି ପ୍ରତ୍ୟେକ ସାଇଟ ପାଇଁ ସେଗୁଡ଼ିକୁ କଷ୍ଟମ କରିପାରିବେ।

## ଏହା WooCommerce ଏବଂ ଅନ୍ୟ ପ୍ଲଗଇନ ସହ କାମ କରେ କି? {#does-this-work-with-woocommerce-and-other-plugins}

ନିଶ୍ଚୟ! ସ୍ଟାଣ୍ଡାର୍ଡ WordPress ଅନୁବାଦ ଫଙ୍କସନ ବ୍ୟବହାର କରୁଥିବା ଯେକୌଣସି ପ୍ଲଗଇନ କିମ୍ବା ଥିମ ସାଇଟ-ନିର୍ଦ୍ଦିଷ୍ଟ ଅନୁବାଦ ସହ କାମ କରିବ।
