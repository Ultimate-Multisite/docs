# ମାପିତ ଯୋଜନାଗୁଡ଼ିକର ପରିବର୍ତ୍ତନ ତାଲିକା

ସଂସ୍କରଣ ୧.୧.୦ - ୨୦୨୬-୦୫-୦୫ ରେ ମୁକ୍ତିଲାଭ
- ନୂଆ: ବହୁ-ସାଇଟ ଉପସାଇଟଗୁଡ଼ିକ ପାଇଁ AI token billing — ସଂରଚନାଯୋଗ୍ୟ ପ୍ରତି-token ଦର ସହିତ ଗ୍ରାହକ ସାଇଟଗୁଡ଼ିକରେ AI token ବ୍ୟବହାରକୁ ଟ୍ରାକ୍ କରନ୍ତୁ ଏବଂ ବିଲ୍ କରନ୍ତୁ
- ନୂଆ: ସମସ୍ତ connector ରେ ପ୍ରକୃତ-ସମୟ ସଠିକତା ନିଶ୍ଚିତ କରିବାକୁ ଗତିଶୀଳ ସୀମା ଆବିଷ୍କାର ଏବଂ write-through ସହିତ connector enforcement ପୁନଃନିର୍ମାଣ କରାଗଲା
- ସୁଧାର: Database table ଉନ୍ନୟନଗୁଡ଼ିକ ଏବେ BerlinDB schema ପରିଭାଷାସହ ସଠିକ ଭାବେ ସମନ୍ୱିତ, ଯାହା ନୂଆ ସଂସ୍ଥାପନରେ ଉନ୍ନୟନ ବିଫଳତାକୁ ରୋକେ
- ସୁଧାର: Database upgrade callbacks ଠିକ୍ ଫର୍ମାଟକୁ ପରିବର୍ତ୍ତିତ, ନିରବ ଉନ୍ନୟନ ବିଫଳତାକୁ ସମାଧାନ କରେ
- ସୁଧାର: AI Usage Overage Markup input field ରେ ଏବେ ଭାଗାଂଶ ମୂଲ୍ୟ ସ୍ୱୀକୃତ
- ସୁଧାର: plugin load ସମୟରେ ଘାତକ ତ୍ରୁଟି ଏବଂ ଦୁଇଥର-ଆରମ୍ଭ ହେବା ସମସ୍ୟା ସମାଧାନ
- ଉନ୍ନତ: check-env npm script ଯୋଡ଼ାଗଲା, ଯେପରି developer environment ଗୁଡ଼ିକ ପ୍ରଥମ ଚାଳନାରେ ସ୍ୱୟଂ-ସଂରଚିତ ହୁଅନ୍ତି

### ୧.୦.୩
* Plugin Update Checker v5 କୁ ଅଦ୍ୟତନ
* ଆଧୁନିକ WordPress plugin headers ଯୋଡ଼ାଗଲା
* ସର୍ବଶେଷ WordPress ସଂସ୍କରଣସହ ସମନ୍ୱୟତା ଉନ୍ନତ
* ବ୍ୟବହାର ଟ୍ରାକିଂ କାର୍ଯ୍ୟଦକ୍ଷତା ଉନ୍ନତ

### ୧.୦.୨
* ତ୍ରୁଟି ସୁଧାର ଏବଂ କାର୍ଯ୍ୟଦକ୍ଷତା ଉନ୍ନତି
* ବ୍ୟବହାର ରିପୋର୍ଟିଂ ଉନ୍ନତ
