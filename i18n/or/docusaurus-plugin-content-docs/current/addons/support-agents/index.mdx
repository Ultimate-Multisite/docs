# ସହାୟତା ଏଜେଣ୍ଟ

ନିଜେ ଏକ ସମ୍ପୂର୍ଣ୍ଣ ନେଟୱର୍କ ପରିଚାଳନା କରିବା ସହଜ କାମ ନୁହେଁ। ଆପଣଙ୍କ ନେଟୱର୍କ ବଢ଼ିବା ଆରମ୍ଭ କଲେ, ସହାୟତା ପାଇଁ ଚାହିଦା ବଢ଼ୁଥିବା ଦେଖିବେ। ଆପଣଙ୍କ କାମର ଭାର କମେଇବାରେ ସହାଯ୍ୟ କରିବା ପାଇଁ ଲୋକଙ୍କୁ ଆଣିବା ଏକ ଭଲ ବିଚାର ହେବ। କିନ୍ତୁ ପୂର୍ଣ୍ଣ ଆକ୍ସେସ୍/ଅନୁମତି ଦେବା ଜୁଆଁପୂର୍ଣ୍ଣ ଏବଂ ଆପଣଙ୍କ ନେଟୱର୍କକୁ ବିପଦରେ ପକାଇପାରେ। ଏଠିଠାରେ ଏହି ସହାୟତା ଏଜେଣ୍ଟ ଆଡ୍-ଅନ୍ କାମକୁ ଆସେ।

ଏହି ଆଡ୍-ଅନ୍ ଆପଣଙ୍କୁ ସହାୟତା ଏଜେଣ୍ଟ ବୋଲି ଏକ ନୂଆ ପ୍ରକାରର WordPress ବ୍ୟବହାରକାରୀ ତିଆରି କରିବାକୁ ଦିଏ, ଏବଂ ସେହି ବ୍ୟବହାରକାରୀଙ୍କ ପାଖରେ କେଉଁ ଅନୁମତି ରହିବ ତାହାର ସମ୍ପୂର୍ଣ୍ଣ ନିୟନ୍ତ୍ରଣ ଦିଏ। ଅନୁମତି ଦେବା ଟଗଲ୍ ବଟନ୍ ଚାଲୁ କରିବା ପରି ସହଜ, ଏବଂ ଏହାକୁ ହଟାଇବାକୁ ଚାହିଁଲେ ବନ୍ଦ କରିଦିଅନ୍ତୁ।

## ବୈଶିଷ୍ଟ୍ୟ

- WordPress ଏବଂ Ultimate Multisite ଅନୁମତିର ବିଭାଗୀକୃତ ତାଲିକା
- ସହଜରେ ଆକ୍ସେସ୍ ଚାଲୁ ଏବଂ ବନ୍ଦ କରନ୍ତୁ
- ଅସୀମିତ ସଂଖ୍ୟାର ଏଜେଣ୍ଟ Account ତିଆରି କରନ୍ତୁ

## ସଂସ୍ଥାପନ

1. 'ultimate-multisite-support-agents' କୁ '/wp-content/plugins/' ଡାଇରେକ୍ଟରୀକୁ ଅପଲୋଡ୍ କରନ୍ତୁ
2. WordPress ରେ 'ପ୍ଲଗଇନ୍‌ଗୁଡ଼ିକ' ମେନୁ ମାଧ୍ୟମରେ ପ୍ଲଗଇନ୍ ସକ୍ରିୟ କରନ୍ତୁ
3. ପ୍ଲଗଇନ୍ ସେଟ୍ ଅପ୍ କରିବା ପାଇଁ ପଦକ୍ରମିକ ୱିଜାର୍ଡକୁ ଅନୁସରଣ କରନ୍ତୁ
