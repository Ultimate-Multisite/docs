# ବହୁ-ଭଡ଼ାଟିଆ ବ୍ୟବସ୍ଥା

କାର୍ଯ୍ୟଦକ୍ଷତା ଏବଂ ସୁରକ୍ଷା ପାଇଁ ଉପସ୍ଥଳୀଗୁଡ଼ିକୁ ସର୍ଭର ଏବଂ ଡାଟାବେସରେ ବଣ୍ଟନ କରନ୍ତୁ

## ସଂସ୍କରଣ ୧.୨.୦ ରିଲିଜ୍‌ର ମୁଖ୍ୟାଂଶ

Multi-Tenancy ୧.୨.୦ ଉତ୍ପାଦନ ଅଲଗାକରଣ କାର୍ଯ୍ୟପ୍ରବାହ ପାଇଁ ସାର୍ବଭୌମ ଭଡ଼ାଟିଆ ଭିତ୍ତି ଯୋଗ କରେ:

- ସ୍ୱଳ୍ପକାଳୀନ, ପୁନଃଚାଳନ-ସୁରକ୍ଷିତ SSO ଭ୍ରମଣ ସହିତ ଅବସ୍ଥାହୀନ ଭଡ଼ାଟିଆ ସ୍ୱୟଂଚାଳିତ ଲଗଇନ।
- ସାର୍ବଭୌମ ଭଡ଼ାଟିଆମାନଙ୍କ ପାଇଁ ପ୍ରତି-ଉପସ୍ଥଳୀ ଡାଟାବେସ ଏବଂ ଫାଇଲସିଷ୍ଟମ ଅଲଗାକରଣ।
- ହୋଷ୍ଟ-ସଚେତନ ଭଡ଼ାଟିଆ ପ୍ରସ୍ତୁତି ପାଇଁ Hostinger ବହୁ-ଭଡ଼ାଟିଆ କ୍ଷମତା।
- ପୁରୁଣା ତଥ୍ୟ ଯାଞ୍ଚ ଏବଂ ଅସମୟିକ ପୁଶ୍ ପ୍ରକ୍ରିୟାକରଣ ପାଇଁ ସାର୍ବଭୌମ ସ୍ଥାନାନ୍ତରଣ ଯାଞ୍ଚ ଦ୍ୱାର।
- Bedrock, FrankenPHP, ଏବଂ localhost ସ୍ଥାପନ ପାଇଁ ବିନ୍ୟାସଯୋଗ୍ୟ ସମାନ-ମେସିନ୍ ଡାଟାବେସ ହୋଷ୍ଟ ବାଇଣ୍ଡିଂ।
- ଭଡ଼ାଟିଆ ବୁଟସ୍ଟ୍ରାପ, ଡାଟାବେସ ଲେଖକ, ସ୍କିମା ବୁଟସ୍ଟ୍ରାପ, ଏବଂ ବ୍ୟବହାରକାରୀ ପ୍ରସ୍ତୁତିର ସୁଧାର।
- ଭଡ଼ାଟିଆ ପରିଚୟପତ୍ର ସଫା କରୁଥିବା ଅଧିକ ସୁରକ୍ଷିତ ସାର୍ବଭୌମ ଅପସାରଣ।

[ସାର୍ବଭୌମ ସ୍ଥାନାନ୍ତରଣ କାର୍ଯ୍ୟପ୍ରବାହ](/user-guide/getting-started/sovereign-migration-workflow), [ବହୁ-ଭଡ଼ାଟିଆ ଅଲଗାକରଣ](/user-guide/configuration/multi-tenancy-isolation), ଏବଂ [ସାର୍ବଭୌମ ଭଡ଼ାଟିଆ ପରିଚାଳନା](/user-guide/administration/sovereign-tenant-operations) ପାଇଁ ବ୍ୟବହାରକାରୀ ମାର୍ଗଦର୍ଶିକା ଦେଖନ୍ତୁ।

## ଆବଶ୍ୟକତା

- WordPress ୫.୩ କିମ୍ବା ତାଠାରୁ ଉଚ୍ଚ
- PHP ୭.୪ କିମ୍ବା ତାଠାରୁ ଉଚ୍ଚ
- Multisite Ultimate plugin (ସକ୍ରିୟ)

## ସ୍ଥାପନ

1. ଆଡଅନ୍ ଫାଇଲଗୁଡ଼ିକୁ ଆପଣଙ୍କ `/wp-content/plugins/` ଡିରେକ୍ଟରୀକୁ ଅପଲୋଡ୍ କରନ୍ତୁ
2. WordPress ର 'Plugins' ମେନୁ ମାଧ୍ୟମରେ ପ୍ଲଗଇନ ସକ୍ରିୟ କରନ୍ତୁ
3. ନେଟୱର୍କ ଆଡମିନରେ ଆଡଅନ୍ ସେଟିଂସ୍ ବିନ୍ୟାସ କରନ୍ତୁ

### ପରୀକ୍ଷା

ଆଡଅନ୍‌ରେ ଏକ ସମ୍ପୂର୍ଣ୍ଣ ପରୀକ୍ଷା ଫ୍ରେମୱର୍କ ସାମିଲ ଅଛି:

- **ୟୁନିଟ୍ ପରୀକ୍ଷା**: WordPress ପରୀକ୍ଷା ସୁଇଟ୍ ସହିତ PHPUnit-ଆଧାରିତ ପରୀକ୍ଷା
- **ମୂଳ ପରୀକ୍ଷା କ୍ଲାସ୍**: ସାଧାରଣ ପରୀକ୍ଷା ପରିସ୍ଥିତି ପାଇଁ ସହାୟକ ପ୍ରଣାଳୀ ଯୋଗାଇଥାଏ
- **ପରୀକ୍ଷା ଆବରଣ**: କୋଡ୍ ଆବରଣ ଟ୍ରାକ୍ କରିବାକୁ ବିନ୍ୟାସିତ
- **ବହୁସ୍ଥଳୀ ପରୀକ୍ଷା**: ପରୀକ୍ଷାଗୁଡ଼ିକ ବହୁସ୍ଥଳୀ ପରିବେଶରେ ଚାଲେ

### କୋଡ୍ ମାନଦଣ୍ଡ

- **WordPress କୋଡିଂ ମାନଦଣ୍ଡ**: PHPCS ମାଧ୍ୟମରେ ଲାଗୁ କରାଯାଏ
- **PHP ୭.୪+ ସୁସଙ୍ଗତତା**: ଆଧୁନିକ PHP ବୈଶିଷ୍ଟ୍ୟ ସମର୍ଥିତ
- **ସ୍ଥିର ବିଶ୍ଳେଷଣ**: ପ୍ରକାର ଯାଞ୍ଚ ପାଇଁ PHPStan ସମନ୍ୱୟ
- **କୋଡ୍ ଆଧୁନିକୀକରଣ**: PHP ଅପଗ୍ରେଡ୍ ପାଇଁ Rector ନିୟମ

### ନିର୍ମାଣ ପ୍ରଣାଳୀ

- **ଆସେଟ୍ ପ୍ରକ୍ରିୟାକରଣ**: CSS/JS ଫାଇଲ ପାଇଁ ସଂକୋଚନ
- **ଅନୁବାଦ**: POT ଫାଇଲ ତିଆରି
- **ପ୍ୟାକେଜିଂ**: ବଣ୍ଟନ ପାଇଁ ଆର୍କାଇଭ୍ ତିଆରି
- **ବିକାଶ ମୋଡ୍**: ଡିବଗିଂ ପାଇଁ ଅସଂକୋଚିତ ଆସେଟ୍

## ବିନ୍ୟାସ

ଆଡଅନ୍ ନେଟୱର୍କ ଆଡମିନ୍ ଇଣ୍ଟରଫେସ୍ ମାଧ୍ୟମରେ ଉପଲବ୍ଧ ବିଭିନ୍ନ ବିନ୍ୟାସ ବିକଳ୍ପକୁ ସମର୍ଥନ କରେ।

### ସାର୍ବଭୌମ ଭଡ଼ାଟିଆ ବିନ୍ୟାସ

ସାର୍ବଭୌମ ଭଡ଼ାଟିଆମାନଙ୍କ ପାଇଁ, ଭଡ଼ାଟିଆ ଡାଟାବେସ, ଫାଇଲସିଷ୍ଟମ ମୂଳ, ଡାଟାବେସ ହୋଷ୍ଟ ବାଇଣ୍ଡିଂ, ଏବଂ ହୋଷ୍ଟ କ୍ଷମତା ବିନ୍ୟାସ କରନ୍ତୁ; ତାପରେ ଟ୍ରାଫିକ୍‌କୁ ଭଡ଼ାଟିଆଙ୍କ ପାଖକୁ ରାଉଟ୍ କରନ୍ତୁ। ଡାଟାବେସ ପରିଚୟପତ୍ର, ହୋଷ୍ଟ ବାଇଣ୍ଡିଂ, ରାଉଟିଂ, କିମ୍ବା ବ୍ୟବହାରକାରୀ ପ୍ରସ୍ତୁତିକୁ ପ୍ରଭାବିତ କରୁଥିବା ପ୍ରତ୍ୟେକ ପରିବର୍ତ୍ତନ ପରେ ସ୍ଥାନାନ୍ତରଣ ଯାଞ୍ଚ ଚଳାନ୍ତୁ।

ଭଡ଼ାଟିଆ ରନ୍ଟାଇମ୍ ଏବଂ ଡାଟାବେସ ଅନୁମତି ସ୍ଥାନୀୟ ସକେଟ୍ କିମ୍ବା ସମାନ-ମେସିନ୍ ପ୍ରବେଶ ଆଶା କଲେ ମାତ୍ର `localhost` ବ୍ୟବହାର କରନ୍ତୁ। ଆପଣଙ୍କ ପ୍ରଦାନକାରୀ ଦ୍ୱାରା ସେହି ଅନୁମତି ହୋଷ୍ଟ ବିନ୍ୟାସିତ ଥିଲେ `127.0.0.1` କିମ୍ବା ଏକ ସେବା ହୋଷ୍ଟନାମ ବ୍ୟବହାର କରନ୍ତୁ।

### Hostinger କ୍ଷମତା

Hostinger ବହୁ-ଭଡ଼ାଟିଆ କ୍ଷମତା ସେୟାର୍ କରାଯାଇଥିବା Hostinger ସମନ୍ୱୟ ଟୋକେନ୍ ସହିତ କାମ କରେ। ଯାଞ୍ଚ ପୂର୍ବରୁ ଡୋମେନ୍ hPanel ରେ ଉପସ୍ଥିତ ଅଛି, DNS Hostinger Account କୁ ସୂଚାଉଛି, ଏବଂ ଭଡ଼ାଟିଆ ଡାଟାବେସ ସେଟିଂସ୍ hPanel ସହିତ ମେଳ ଖାଉଛି ବୋଲି ନିଶ୍ଚିତ କରନ୍ତୁ।

## ହୁକ୍ ଏବଂ ଫିଲ୍ଟର

### କାର୍ଯ୍ୟ

- `ultimate-multisite-addon-template_init` - ଆଡଅନ୍ ଆରମ୍ଭ ପରେ ଚାଲୁ ହୁଏ
- `ultimate-multisite-addon-template_loaded` - ସମସ୍ତ ଆଡଅନ୍ ଫାଇଲ ଲୋଡ୍ ହେବା ପରେ ଚାଲୁ ହୁଏ

### ଫିଲ୍ଟର

- `ultimate-multisite-addon-template_settings` - ଆଡଅନ୍ ସେଟିଂସ୍ ପରିବର୍ତ୍ତନ କରନ୍ତୁ
- `ultimate-multisite-addon-template_enabled` - ଆଡଅନ୍ ସକ୍ରିୟ/ନିଷ୍କ୍ରିୟ ସ୍ଥିତିକୁ ଓଭରରାଇଡ୍ କରନ୍ତୁ

## ଲାଇସେନ୍ସ

ଏହି ଆଡଅନ୍ GPL v3 କିମ୍ବା ପରବର୍ତ୍ତୀ ଲାଇସେନ୍ସ ଅଧୀନରେ ଅଛି।

## ସହାୟତା

ସହାୟତା ଏବଂ ଡକ୍ୟୁମେଣ୍ଟେସନ୍ ପାଇଁ, [MultisiteUltimate.com](https://multisiteultimate.com) ଦେଖନ୍ତୁ
## ପରିବର୍ତ୍ତନ ତାଲିକା

- ସଂସ୍କରଣ ୧.୨.୦ (୨୦୨୬-୦୬-୦୩): ଅବସ୍ଥାହୀନ ଭଡ଼ାଟିଆ ସ୍ୱୟଂଚାଳିତ ଲଗଇନ, Hostinger ବହୁ-ଭଡ଼ାଟିଆ କ୍ଷମତା, ପ୍ରତି-ଉପସ୍ଥଳୀ ଡାଟାବେସ ଏବଂ ଫାଇଲସିଷ୍ଟମ ଅଲଗାକରଣ, ସାର୍ବଭୌମ ସ୍ଥାନାନ୍ତରଣ ଯାଞ୍ଚ ଦ୍ୱାର, ଭଡ଼ାଟିଆ ବୁଟସ୍ଟ୍ରାପ/ଡାଟାବେସ ଲେଖକ ସୁଧାର, ବିନ୍ୟାସଯୋଗ୍ୟ MySQL ହୋଷ୍ଟ ବାଇଣ୍ଡିଂ, ଏବଂ ଅଧିକ ସୁରକ୍ଷିତ ସାର୍ବଭୌମ ଅପସାରଣ ଯୋଗ କରନ୍ତୁ।

- ସଂସ୍କରଣ ୧.୦.୧ (୨୦୨୫-୦୯-୨୮): ପ୍ରିଫିକ୍ସକୁ ultimate-multisite ଭାବେ ପୁନର୍ନାମକରଣ କରନ୍ତୁ; ଟେକ୍ସଟ୍ ଡୋମେନ୍ ଅଦ୍ୟତନ କରନ୍ତୁ; ସଂସ୍କରଣ ବୃଦ୍ଧି କରନ୍ତୁ।


- ସଂସ୍କରଣ  (୨୦୨୫-୦୯-୨୮): ପ୍ରିଫିକ୍ସକୁ ultimate-multisite ଭାବେ ପୁନର୍ନାମକରଣ କରନ୍ତୁ; ଟେକ୍ସଟ୍ ଡୋମେନ୍ ଅଦ୍ୟତନ କରନ୍ତୁ; ସଂସ୍କରଣ ବୃଦ୍ଧି କରନ୍ତୁ।
