# ସୁସଙ୍ଗତ ଏଣ୍ଡପଏଣ୍ଟ ପାଇଁ Ultimate AI Connector ପରିବର୍ତ୍ତନ ସୂଚୀ

## ସଂସ୍କରଣ 2.0.0 — 2026-04-24 ରେ ମୁକ୍ତିଲାଭ

* ନୂଆ: ବହୁ-ପ୍ରଦାତା ସମର୍ଥନ — ଏକାଧିକ AI ଏଣ୍ଡପଏଣ୍ଟ କନଫିଗର୍ କରନ୍ତୁ ଏବଂ ପ୍ରଦାତାମାନଙ୍କ ମଧ୍ୟରେ ସ୍ୱୟଂଚାଳିତ ଫଲ୍‌ବ୍ୟାକ୍ ସହ ଅନୁରୋଧଗୁଡ଼ିକୁ ରୁଟ୍ କରନ୍ତୁ।
* ସମାଧାନ: ପ୍ରତି ପ୍ରଦାତା ପାଇଁ ଠିକ୍ ପ୍ରଦାତା ID, ପଞ୍ଜୀକରଣ URL, ଏବଂ HTTP ଫିଲ୍ଟର ସ୍କୋପିଂ ସହ ବହୁ-ପ୍ରଦାତା SDK ଇଣ୍ଟିଗ୍ରେସନ୍।
* ସମାଧାନ: ନୂଆ ପ୍ରଦାତା କାର୍ଡ ଏବେ ଯୋଡ଼ାହେବା ସହିତ ସ୍ୱୟଂଚାଳିତ ଭାବେ ବିସ୍ତାରିତ ହୁଏ; plugin ଅପଡେଟ୍‌ରେ ସ୍କ୍ରିପ୍ଟ କ୍ୟାଶ୍ ବସ୍ଟିଂ।
* ସମାଧାନ: ନୂଆ ଯୋଡ଼ାଯାଇଥିବା ପ୍ରଦାତାମାନଙ୍କ ପାଇଁ eval() ଏବଂ ସ୍ୱୟଂ-ବିସ୍ତାର ବ୍ୟବହାର ପାଇଁ ଡାଇନାମିକ୍ ପ୍ରଦାତା କ୍ଲାସ୍ ନେମସ୍ପେସ୍।
* ସମାଧାନ: WordPress 6.9+ ସୁସଙ୍ଗତତା ପାଇଁ ସ୍ଥିର Card/CardBody/CardHeader/CardDivider କମ୍ପୋନେଣ୍ଟ ବ୍ୟବହାର କରେ (ଆଉ ପରୀକ୍ଷାମୂଳକ ନୁହେଁ)।
* ସମାଧାନ: ପ୍ରଦାତାକୁ ଟାଣି ପୁନଃକ୍ରମିତ କରିବା ପାଇଁ ଅନିର୍ଦ୍ଧାରିତ DragHandle କୁ ୟୁନିକୋଡ୍ ଗ୍ରିପ୍ ଆଇକନ୍‌ରେ ବଦଳାଗଲା।
* ସମାଧାନ: ପ୍ରତ୍ୟେକ ପୃଷ୍ଠା ଲୋଡ୍‌ରେ ଚାଲୁଥିବା ବ୍ଲକିଂ HTTP ଅନୁରୋଧକୁ ହଟାଗଲା।
* ଉନ୍ନତ: GitHub Actions ୱର୍କଫ୍ଲୋଗୁଡ଼ିକୁ Node.js 24 କୁ ଅପଗ୍ରେଡ୍ କରାଗଲା।
