# ଭାଷା ଚୟନକାରୀ {#language-selector}

Multisite Ultimate: ସାଇନ୍-ଅପ୍‌ରେ ଭାଷା ଚୟନକାରୀ

ସାଇନ୍-ଅପ୍ ପ୍ରବାହରେ ଏକ ଭାଷା ଚୟନକାରୀ କ୍ଷେତ୍ର ଯୋଡ଼ନ୍ତୁ।

## ସ୍ଥାପନ {#installation}

1. 'ultimate-multisite-language-selector' କୁ '/wp-content/plugins/' ଡିରେକ୍ଟରୀକୁ ଅପଲୋଡ୍ କରନ୍ତୁ
2. ଆପଣଙ୍କ WordPress ନେଟୱର୍କ ପ୍ରଶାସନରେ 'ପ୍ଲଗଇନ୍‌ଗୁଡ଼ିକ' ମେନୁ ମାଧ୍ୟମରେ ପ୍ଲଗଇନ୍‌ଟି ସକ୍ରିୟ କରନ୍ତୁ

## ବ୍ୟବହାର ମାର୍ଗଦର୍ଶିକା {#usage-guide}

ଏହି ମାଗଣା ଆଡ୍-ଅନ୍ ସହିତ, ଆପଣ ଆପଣଙ୍କ ବ୍ୟବହାରକାରୀଙ୍କୁ ସାଇନ୍-ଅପ୍ ପ୍ରବାହ ଏବଂ ଲଗଇନ୍ ପୃଷ୍ଠାରେ କେଉଁ ଭାଷା ଦେଖିବାକୁ ଚାହାନ୍ତି ତାହା ବାଛିବାକୁ ଦେଇପାରିବେ।

[ଆପଣଙ୍କ ଲାଇସେନ୍ସ ସକ୍ରିୟ କରିବା ପରେ](../../user-guide/getting-started/installing-ultimate-multisite.md), ଆପଣଙ୍କ ଲାଇସେନ୍ସ ଯେଉଁ Ultimate Multisite ଆଡ୍-ଅନ୍‌ଗୁଡ଼ିକୁ ପ୍ରବେଶ ଦିଏ, ସେଗୁଡ଼ିକୁ ଆପଣ ସ୍ଥାପନ କରିପାରିବେ।

ଆପଣଙ୍କ Ultimate Multisite ସେଟିଂସ୍ ପୃଷ୍ଠାକୁ ଯାଆନ୍ତୁ ଏବଂ ତାପରେ ବାମ ପାର୍ଶ୍ୱର ମେନୁରେ _ଆଡ୍-ଅନ୍‌ଗୁଡ଼ିକ_ ଲେବଲ୍‌ରେ କ୍ଲିକ୍ କରନ୍ତୁ।

![ଆଡ୍-ଅନ୍‌ଗୁଡ଼ିକ ପୃଷ୍ଠା](/img/admin/addons-page.png)

_ଭାଷା ଚୟନକାରୀ ସ୍ଥାପନ ଆରମ୍ଭ କରିବାକୁ ଆଡ୍-ଅନ୍‌ଗୁଡ଼ିକ ପୃଷ୍ଠାକୁ ଯାଆନ୍ତୁ_

ସେଠାରେ, ଆମେ ଦେଉଥିବା ସମସ୍ତ ଆଡ୍-ଅନ୍ ଆପଣ ପାଇବେ।

ମାଗଣା ଆଡ୍-ଅନ୍‌ଗୁଡ଼ିକ ଆମ ସମସ୍ତ ଗ୍ରାହକଙ୍କ ପାଇଁ ଉପଲବ୍ଧ, ସେମାନେ କେଉଁ ସବ୍ସକ୍ରିପ୍ସନ୍ ପ୍ଲାନ୍‌ରେ ସାଇନ୍ ଅପ୍ କରିଥିଲେ ତାହା ଯାହାହେଉ।

ଆଡ୍-ଅନ୍‌ର ଥମ୍ବନେଲ୍ ତଳେ ଥିବା _ଏବେ ସ୍ଥାପନ କରନ୍ତୁ!_ ବଟନ୍‌ରେ କ୍ଲିକ୍ କରନ୍ତୁ। ତାପରେ ପ୍ଲଗଇନ୍‌ଗୁଡ଼ିକ ପୃଷ୍ଠାରେ ଏହାକୁ ସକ୍ରିୟ କରନ୍ତୁ।

![ସ୍ଥାପନ ବଟନ୍ ସହିତ ଆଡ୍-ଅନ୍‌ଗୁଡ଼ିକ ପୃଷ୍ଠା](/img/admin/addons-page.png)

_ଆଡ୍-ଅନ୍ ସ୍ଥାପନ କରିବାକୁ ବଟନ୍‌ରେ କ୍ଲିକ୍ କରନ୍ତୁ_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_ଆପଣଙ୍କ ପ୍ଲଗଇନ୍‌ଗୁଡ଼ିକ ପୃଷ୍ଠାରେ ଏହାକୁ ସକ୍ରିୟ କରନ୍ତୁ_

ଏବେ, Ultimate Multisite ର ସେଟିଂସ୍ ମେନୁକୁ ଯାଆନ୍ତୁ ଏବଂ _ନେଟୱର୍କ ସେଟିଂସ୍_ ଟ୍ୟାବ୍‌କୁ ପ୍ରବେଶ କରନ୍ତୁ। ସେଠାରେ, ପୃଷ୍ଠାର ତଳ ଭାଗରେ _ଭାଷାଗୁଡ଼ିକ_ ନାମର ଏକ ନୂତନ ଖଣ୍ଡ ପାଇବେ।

![ସେଟିଂସ୍ ପୃଷ୍ଠା](/img/config/settings-general.png)

_ସମ୍ପୂର୍ଣ୍ଣ ନୂଆ ଖଣ୍ଡ ଦେଖାଯାଏ!_

ସମସ୍ତ ଭାଷାକୁ ବିକଳ୍ପ ଭାବେ ଦେଖାଇବା ପାଇଁ ଆପଣ ଏହାକୁ ଖାଲି ରଖିପାରିବେ, କିମ୍ବା କେଉଁଗୁଡ଼ିକୁ ଉପଲବ୍ଧ କରିବେ ତାହା ବାଛିପାରିବେ। ଏହି ଭାଷାଗୁଡ଼ିକ ହେଉଛି ଆପଣଙ୍କ WordPress ସେଟଅପ୍‌ରେ ସ୍ଥାପିତ ଥିବା ଭାଷାଗୁଡ଼ିକ।

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_ଏହି WP ସ୍ଥାପନରେ ଉପଲବ୍ଧ ଭାଷାଗୁଡ଼ିକର ତାଲିକା_

ଆପଣ କରିଥିବା ପରିବର୍ତ୍ତନଗୁଡ଼ିକ ସେଭ୍ କରିବା ପରେ, ଆପଣ ବାଛିଥିବା ଭାଷାଗୁଡ଼ିକ ସହିତ ଏକ ଚୟନ ମେନୁ ଲଗଇନ୍ ପୃଷ୍ଠା ଏବଂ ସାଇନ୍-ଅପ୍ ପ୍ରବାହରେ ଦେଖାଯିବ।

![ଲଗଇନ୍ ଏବଂ ପଞ୍ଜୀକରଣ ସେଟିଂସ୍](/img/config/settings-login-registration.png)

_ଲଗଇନ୍ ପୃଷ୍ଠାରେ ଭାଷା ଚୟନକାରୀ_

![ପଞ୍ଜୀକରଣ ପୃଷ୍ଠା](/img/config/registration-page.png)

_ସାଇନ୍-ଅପ୍ ପ୍ରବାହରେ ଭାଷା ଚୟନକାରୀ_

## URL ମାଧ୍ୟମରେ ଭାଷା ସେଟ୍ କରିବା {#setting-the-language-via-the-url}

ଆପଣ URL କୁ locale ପାରାମିଟର୍ ପଠାଇ ଏକ ପୃଷ୍ଠାର ଭାଷା ମଧ୍ୟ ପରିବର୍ତ୍ତନ କରିପାରିବେ। ଉଦାହରଣ ସ୍ୱରୂପ, ଯଦି ଆପଣ ଇଟାଲୀୟ ଭାଷାରେ ଏକ ପଞ୍ଜୀକରଣ ପୃଷ୍ଠାକୁ ପୁନଃନିର୍ଦ୍ଦେଶ କରିବାକୁ ଚାହାନ୍ତି, URL ରେ ?locale=it_IT ଯୋଡ଼ନ୍ତୁ।

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

ଏହା କାମ କରିବା ପାଇଁ ଭାଷାଟି ଆପଣଙ୍କ WordPress ନେଟୱର୍କରେ ସ୍ଥାପିତ ହେବା ଆବଶ୍ୟକ।
