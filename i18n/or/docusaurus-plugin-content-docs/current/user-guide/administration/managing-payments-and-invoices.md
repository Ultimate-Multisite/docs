# చెల్లింపులు మరియు ఇన్వాయిస్‌లను నిర్వహించడం

## చెల్లింపు సెట్టింగ్‌లు

చెల్లింపులు స్వీకరించడం ప్రారంభించే ముందు, మీరు చెల్లింపుకు సంబంధించిన సెట్టింగ్‌లను కాన్ఫిగర్ చేయాలి. **Ultimate Multisite → Settings** కి వెళ్లి, **Payment** ట్యాబ్‌పై క్లిక్ చేయండి.

![Payment settings tab](/img/admin/settings-payments-top.png)

చెల్లింపు సెట్టింగ్‌ల పూర్తి వీక్షణ ఇక్కడ ఉంది:

![Payment settings full page](/img/admin/settings-payments-full.png)

### సాధారణ చెల్లింపు ఎంపికలు

సాధారణ సెట్టింగ్‌లలో మీరు వీటిని కాన్ఫిగర్ చేయవచ్చు:

- **Currency** — లావాదేవీలకు ఉపయోగించే డిఫాల్ట్ కరెన్సీ
- **Currency Position** — కరెన్సీ గుర్తు ఎక్కడ కనిపిస్తుంది (మొదట/తర్వాత మొత్తం తర్వాత)

![Payment gateway settings](/img/admin/settings-payments-options.png)

### చెల్లింపు గేట్‌వేలు

Ultimate Multisite బహుళ చెల్లింపు గేట్‌వేలను సపోర్ట్ చేస్తుంది. మీరు Payment సెట్టింగ్‌ల ట్యాబ్ నుండి ప్రతి దానిని ఎనేబుల్ చేసి కాన్ఫిగర్ చేయవచ్చు.

![Payment gateway configuration](/img/admin/settings-payments-gateways.png)

ఉపయోగించడానికి అందుబాటులో ఉన్న గేట్‌వేలు:

- **Stripe** — Stripe ద్వారా క్రెడిట్ కార్డ్ చెల్లింపులు
- **PayPal** — PayPal చెల్లింపులు
- **Manual** — ఆఫ్‌లైన్ లేదా కస్టమ్ చెల్లింపు ప్రాసెసింగ్ కోసం

ప్రతి గేట్‌వేకు దాని స్వంత కాన్ఫిగరేషన్ సెక్షన్ ఉంటుంది, అక్కడ మీరు API keys మరియు ఇతర సెట్టింగ్‌లను నమోదు చేస్తారు.

![Additional gateway settings](/img/admin/settings-payments-gateways-2.png)

### స్యాండ్‌బాక్స్ మోడ్ (Sandbox Mode)

లైవ్ చేయడానికి ముందు మీ చెల్లింపు ఇంటిగ్రేషన్‌ను పరీక్షించడానికి మీరు **Sandbox Mode**ని ఎనేబుల్ చేయవచ్చు. స్యాండ్‌బాక్స్ మోడ్ యాక్టివ్‌గా ఉన్నప్పుడు, నిజమైన ఛార్జీలు తీసుకోబడవు.

## చెల్లింపులను వీక్షించడం

మీ నెట్‌వర్క్‌లోని అన్ని లావాదేవీలను చూడటానికి Ultimate Multisite కింద **Payments** పేజీకి వెళ్లండి.

![Payments list](/img/admin/payments-list.png)

మీరు స్థితి (completed, pending, failed, refunded) ద్వారా చెల్లింపులను ఫిల్టర్ చేయవచ్చు మరియు నిర్దిష్ట లావాదేవీల కోసం శోధించవచ్చు.

କୌଣସି ପେମେଣ୍ଟ ଉପରେ କ୍ଲିକ୍ କରି ଏହାର ସମ୍ପୂର୍ଣ୍ଣ ବିବରଣୀ ଦେଖନ୍ତୁ, ଯାହା ମଧ୍ୟରେ ଆଇଟମ, ଜଡ଼ିତ ମେମ୍ବେରିଶିପ୍, ଗ୍ରାହକ ସୂଚନା ଏବଂ ପେମେଣ୍ଟ ଗେଟୱେୟ ତଥ୍ୟ ଅନ୍ତର୍ଭୁକ୍ତ ହୋଇଥାଏ।

## ଇନଭଏସ (Invoices)

Ultimate Multisite ଆପଣଙ୍କ ପାଇଁ ପେମେଣ୍ଟ ପାଇଁ ଆକାରିକା (invoices) ଆଧାରିତ ଭାବରେ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ତିଆରି କରିପାରିବ। ଆପଣ ପେମେଣ୍ଟ ସେଟିଙ୍ଗସ (Payment settings) ମଧ୍ୟରୁ ଇନଭଏସ୍ ଟେମ୍ପଲେଟ୍ ଏବଂ ନମ୍ବରୀୟ ବ୍ୟବସ୍ଥାକୁ କଷ୍ଟମାଇଜ୍ କରିପାରିବେ।

ଇନଭଏସ୍ କଷ୍ଟମାଇଜେସନ୍ ବିକଳ୍ପଗୁଡ଼ିକ ମଧ୍ୟରେ ଅନ୍ତର୍ଭୁକ୍ତ:

- ଇନଭଏସ୍ ରେ **କମ୍ପାନୀ ନାମ ଏବଂ ଠିକଣା** ଦେଖିବା
- **ଇନଭଏସ୍ ନମ୍ବରୀୟ** ବ୍ୟବସ୍ଥା ଏବଂ ସ୍ରେତି (sequence)
- ଇନଭଏସ୍ ହେଡର୍ ରେ ପ୍ରଦର୍ଶିତ **ଲୋଗୋ**
- ନିୟମ, ନୋଟ୍ କିମ୍ବା ଆଇନଗତ ସୂଚନା ପାଇଁ **କଷ୍ଟମ ଫୁটার ଲେଖା**

ଇନଭଏସ୍ ଟେମ୍ପଲେଟ୍ କଷ୍ଟମାଇଜ୍ କରିବା ପାଇଁ, **Ultimate Multisite → Settings → Payment** ରେ ଯାଆନ୍ତୁ ଏବଂ ଇନଭଏସ୍ ସମ୍ବନ୍ଧୀୟ ସେଟିଙ୍ଗଗୁଡ଼ିକୁ ଦେଖନ୍ତୁ।
