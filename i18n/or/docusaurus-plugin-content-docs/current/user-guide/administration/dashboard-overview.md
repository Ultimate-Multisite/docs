# ଡ୍ୟାସବୋର୍ଡ୍ ଓଭରଭିୟ Overview {#dashboard-overview}

Ultimate Multisite ମେନୁରେ ଏକ ବି dedicated **Dashboard** ଅଛି, ଏବଂ ଏହା Network Dashboard ରେ widgets ମଧ୍ୟ ଯୋଗ କରେ। ଏହି ଆଇକାର୍ଡରେ ଆପଣ ଦୁଇଟି ସ୍ଥାନରେ କ'ଣ ଦେଖିବେ, ସେଥିପାଇଁ ଆମେ ବର୍ଣ୍ଣନା କରିବା।

## ନେଟୱର୍କ ଡ୍ୟାସବୋର୍ଡ୍ (The Network Dashboard) {#the-network-dashboard}

Ultimate Multisite 'My Sites → Network Admin → Dashboard' ରେ ଥିବା network-level dashboard ରେ ଉପଯୁକ୍ତ widgets ପ୍ରବେଶ କରେ।

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

ଏହି ପ୍ରତ୍ୟେକ widget କ'ଣ ଦେଇଥାଏ, ସେ ବିଷୟରେ ଜାଣନ୍ତୁ:

**ପ୍ରଥମ ପଦକ୍ଷେପ (First Steps):** ଏହା ନୂଆ ଇନଷ୍ଟଲେସନ୍ ଉପରେ ଦେଖାଯାଏ ଏବଂ ଆପଣଙ୍କୁ payment method ସିଟିଙ୍ଗ କରିବା ଏବଂ ଏକ test account ସୃଷ୍ଟି କରିବା ଭଳି setup tasks ପୂରଣ କରିବାକୁ ସ୍ମରଣ କରେ।

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**ସାରାଂଶ (Summary):** ଏହା ଦିନର signups ଏବଂ ଆୟ ବିଷୟରେ ରିପୋର୍ଟ କରେ। MRR (Monthly Recurring Revenue) ମାସିକ ଆୟ ସହିତ ଆୟ產生କ ସଦସ୍ୟମାନଙ୍କ ସମ୍ପୂର୍ଣ୍ଣ ଅନୁମାନକୁ ଦର୍ଶାଏ।

![Summary widget](/img/admin/network-dashboard-summary.png)

**କାର୍ଯ୍ୟ ପ୍ରବାହ (Activity Stream):** Ultimate Multisite ଉପରେ ଘଟଣାଗୁଡ଼ିକ—signups, cancellation, site creation, deletion ଏବଂ ଅନ୍ୟାନ୍ୟ କାର୍ଯ୍ୟ—ସମ୍ପର୍କରେ ରିପୋର୍ଟ କରେ।

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**ବର୍ତ୍ତମାନ (Right Now):** ଏହା network ମଧ୍ୟରେ users ଏବଂ sites ର ଏକ ସଂକ୍ଷିପ୍ତ ସାରାଂଶ ଦେଖାଏ, ଯେଉଁଥିରେ ନୂଆ site ବା user ସୃଷ୍ଟି କରିବା ପାଇଁ shortcuts ଥାଏ।

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**ଖବର ଓ ଆଲୋଚନା (News & Discussions):** ଏହା Ultimate Multisite ର ନୂଆ ଖବର, ଅପଡେଟ୍, ଗୁରୁତ୍ୱପୂର୍ଣ୍ଣ bugs ଏବଂ ସୁରକ୍ଷା ସିଟିଫିକ୍ସ ଦେଖାଇଥାଏ।

![News widget](/img/admin/network-dashboard-news.png)

## Ultimate Multisite ଡ୍ୟାସବୋର୍ଡ୍ (The Ultimate Multisite Dashboard) {#the-ultimate-multisite-dashboard}

ଯେଉଁଠାରେ network dashboard ମଧ୍ୟରେ ସମଗ୍ର network ବିଷୟରେ ସୂଚନା ଦିଆଯାଏ, ତାହା Ultimate Multisite ଡ୍ୟାସବୋର୍ଡ୍ (Ultimate Multisite ମେନୁର ଉପ-ସ୍ତରରେ) ସେବା ବିଷୟରେ ସୂଚନା ପ୍ରଦାନ କରେ।

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**ମାସିକ ପୁନରାବୃତ୍ତି ରାଶି ବୃଦ୍ଧି (Monthly Recurring Revenue Growth):** ଏହା ଦୁଇ ବର୍ଷର କ୍ୟାଲେଣ୍ଡର ଅବଧିରେ ମାସିକ ଭାଙ୍ଗଣ ବିଭାଗ ଦେଖାଏ। ନୂଆ ଆୟ ନୀଳ ରଙ୍ଗରେ ଦିଆଯାଇଥାଏ, ଯେଉଁଥିରେ ବାତିଲ ହୋଇଥିବା ଆୟ ଲାଲି ରଙ୍ଗରେ ଦର୍ଶାଯାଏ।

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**ତାରିଖ-ପ୍ରେକ୍ରିତି ସମର୍ଥନ (Date-range Support):** ଆପଣ ଯାହା ଜରୁରୀ, ତା' ଉପରେ ଧ୍ୟାନ ଦେବା ପାଇଁ ଡାଶବୋର୍ଡର ତଥ୍ୟକୁ କେବେ ବିଷୟରେ ଫିଲ୍ଟର କରି ସମୟ ଅବଧି ବଦଳାଇପାରିବେ। ଏହି ତାରିଖ-ପ୍ରେକ୍ରିତି ଆପଣଙ୍କୁ ନିମ୍ନରେ ଥିବା widgets ମଧ୍ୟରେ କ'ଣ ଦର୍ଶାଉଛି, ତାହା ନିର୍ଦ୍ଧାରଣ କରିଥାଏ।

![Date range and dashboard stats](/img/admin/um-dashboard-date-range.png)

ଡାଶବୋର୍ଡର widgets ମଧ୍ୟରେ ଅନ୍ତର୍ଭୁକ୍ତ:

- **ଆୟ (Revenue)** — ସେହି ଅବଧି ପାଇଁ ମୋଟ ଆୟ, ଯେଉଁଥିରେ MRR ଏବଂ ପ୍ରତି-ପ୍ରଦୋଷ (per-product) ବିଭାଗ ଦିଆଯାଇଥାଏ।
- **ନୂଆ ସଦସ୍ୟତା (New Memberships)** — ବାଛିଥିବା ଅବଧିରେ କେତେ ନୂଆ ସଦସ୍ୟତା ସୃଷ୍ଟି ହୋଇଛି।
- **ବିଶ୍ୱ ଦ୍ୱାରା ଆପଣଙ୍କର ସାଇଟ୍‌ରେ ଉପଯୋଗ (Signups by Countries)** — ଏକ ଗୁରୁତ୍ୱପୂର୍ଣ୍ଣ ବିଷୟ, ଯେଉଁଠାରେ signups କେଉଁ ଭୌଗୋଳିକ ଅଞ୍ଚଳରୁ ହୋଇଛି, ତାହାର ବିଭାଗ ଦିଆଯାଇଥାଏ।
- **ଫର୍ମ ମାଧ୍ୟମରେ signups (Signups by Forms)** — ପ୍ରତ୍ୟେକ checkout form ମାଧ୍ୟମରେ କେତେ ଗ୍ରାହକ sign up ହୋଇଛି ଏବଂ ଅଡମିନ୍ ଦ୍ୱାରା କେତେ sign up ହୋଇଛି, ତାହା ଜଣାପଡ଼ିବ।
- **ଅଧିକ ଦର୍ଶିବାକୁ ଆସୁଥିବା ସାଇଟ୍ (Most Visited Sites)** — ଆପଣଙ୍କ ନେଟୱର୍କରେ ସବୁଠାରୁ ଅଧିକ ଦର୍ଶିବାକୁ ଆସୁଥିବା ସାଇଟ୍‌ଗୁଡ଼ିକ (ଏଥି ପାଇଁ **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** ରେ ବିଜିଟ୍ ବစ်ିଷ୍ଟ କାଉଣ୍ଟିଂ ଏନେବଲ କରିବା ଆବଶ୍ୟକ)।
