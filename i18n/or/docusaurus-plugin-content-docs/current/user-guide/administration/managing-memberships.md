# ସଦସ୍ୟତା ପରିଚାଳନା (Managing Memberships) {#managing-memberships}

Ultimate Multisite ରେ ଆପଣ ଆପଣଙ୍କ ନେଟୱର୍କର ସମସ୍ତ ସబ్‌ସକ୍ରାଇବ୍ସ ଏବଂ ଅନୁପଲବନକୁ ପରିଚାଳିତ କରିପାରିବେ। ଆପଣ ପ୍ଲାନ୍ ଏବଂ ପ୍ରଡକ୍ଟଗୁଡ଼ିକ, ସାଇଟ୍ ଗୁଡ଼ିକର ସଂଖ୍ୟା, ବିଲିଂ ମୂଲ୍ୟ ଏବଂ ଆବୃତ୍ତି (frequency), ପଇସା ଇତିହାସ ଏବଂ ଟିମ୍ପଷ୍ଟାମ୍ପ୍‌କୁ ଯାଞ୍ଚ କରି ଏବଂ ସଂଶୋଧନ କରିପାରିବେ।

ଏହି ପୃଷ୍ଠାକୁ ପ୍ରବେଶ କରିବା ପାଇଁ, Ultimate Multisite ମେନୁରେ **Memberships** ଟ୍ୟାବ୍ (tab) ଉପରେ କ୍ଲିକ୍ କରନ୍ତୁ।

![Memberships list page](/img/admin/memberships-list.png)

Memberships ପୃଷ୍ଠାରେ ଆପଣ ଆପଣଙ୍କ ଗ୍ରାହକମାନଙ୍କ ଏବଂ ସେମାନଙ୍କ ଆକାଉଣ୍ଟଗୁଡ଼ିକ ସହିତ ଜଡିତ ପ୍ରଡକ୍ଟଗୁଡ଼ିକର ଏକ ତାଲିକା ଦେଖିପାରିବେ। ଆପଣ ଅସ୍ଥିତି (status) ଅନୁଯାୟୀ ମେମ୍ବେରିଶିପ୍‌କୁ ଫିଲ୍ଟର କରିପାରିବେ: active, pending, on hold, expired, or cancelled.

ମୁକ୍ତ ସଦସ୍ୟତାଗୁଡ଼ିକ **ଜୀବନବର୍ତ୍ତି** (lifetime) ମେମ୍ବେରିଶିପ୍‌ ଭାବରେ ବୁଝାଯାଏ। ଏଥିରେ କୌଣସି ଅବସର ନିର୍ଦ୍ଧାରଣ ତାରିଖ ଆସେ ନାହିଁ ଏବଂ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ବ୍ୟାପୀ ହୋଇଯାଏ ନାହିଁ। ଯଦି କୌଣସି ମୁକ୍ତ ପ୍ଲାନ୍ ଏକ ଅଭାବကာလ (trial) କିମ୍ବା ନିର୍ଦ୍ଧାରିତ ସମୟ ପରେ ଶେଷ ହେବା ଆବଶ୍ୟକ, ତେବେ ଏଥିପାଇଁ ଏକ ଅଭାବကာଳ ସହିତ ଦେଇଥିବା ଏକ ପେଡ୍ (paid) ପ୍ରଡକ୍ଟ ବ୍ୟବହାର କରନ୍ତୁ, ମାନସିକ ଭାବରେ ମେମ୍ବେରିଶିପ୍ ରୁ ଅଲଗା କରିଦିଅନ୍ତୁ, କିମ୍ବା ଯେତେବେଳେ ଆପଣଙ୍କର ଅନୁପଲବନ ବଦଳିବା ଆବଶ୍ୟକ ହେବ, ସେତେବେଳେ ଗ୍ରାହକଙ୍କୁ ଅନ୍ୟ ଏକ ପ୍ଲାନ୍‌ରେ ସ୍ଥାନାନ୍ତର କରନ୍ତୁ।

ମେମ୍ବେରିଶିପ୍ ଉପରେ କ୍ଲିକ୍ କରି ଆପଣ ସେହି ପୃଷ୍ଠାକୁ ଯାଇପାରିବେ, ଯେଉଁଠାରେ ଆପଣ ମେମ୍ବେରିଶିପ୍ ବିବରଣୀ—ବିଲିଂ ଇନ୍ଫୋ, ସାଇଟ୍ ବା ଭିଜିଟ୍ ସୀମା ଏବଂ ଅନ୍ୟାନ୍ୟ—ସଂଶୋଧନ କରିପାରିବେ।

![Membership edit interface](/img/admin/membership-edit.png)

ଏଠାରେ ମେମ୍ବେରିଶିପ୍ ଇଡିଟ ପୃଷ୍ଠାର ଏକ ସମ୍ପୂର୍ଣ୍ଣ ଦୃଶ୍ୟ ଦେଖାଯାଇଛି:

![Membership edit full interface](/img/admin/membership-edit-full.png)

ଆପଣ ଆପଣଙ୍କ ନେଟୱର୍କରେ ଯେଉଁ ଲୋକମାନେ ପୂର୍ବରୁ ରେଜିଷ୍ଟర్ ହୋଇଥିଲେ, ସେମାନଙ୍କ ସହିତ ଜଡିତ ନୂଆ ମେମ୍ବେରିଶିପ୍ ମଧ୍ୟ ଯୋଗ କରିପାରିବେ। **Add Membership** ଉପରେ କ୍ଲିକ୍ କରନ୍ତୁ ଏବଂ ବିବରଣୀ ସେଟିଂ କରନ୍ତୁ।

![Add Membership modal](/img/admin/membership-add-modal.png)

ତା'ପରେ **Create Membership** ଉପରେ କ୍ଲିକ୍ କରନ୍ତୁ। ନୂଆ ମେମ୍ବେରିଶିପ୍ଟି ମେମ୍ବେରିଶିପ୍‌ଗୁଡ଼ିକର ତାଲିକାରେ ଦେଖାଯିବ।
