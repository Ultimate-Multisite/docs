# Ultimate Multisite Dashboard ବୁଝିବା

Ultimate Multisite ର ଆଗ୍ରହୀ ସମସ୍ତ ଅଧିକ ଉନ୍ନତ କନଫିଗୁରେସନ୍ ଏବଂ କାର୍ଯ୍ୟକାଳୀକରଣ ପଦକ୍ଷେପକୁ ଭଲ ଭାବରେ ବୁଝିବା ପୂର୍ବ, ଆମେ ଏହି dashboard ବିଷୟରେ ଦେଖିବା।

WordPress ର ଅନେକ plugin ପରି, Ultimate Multisite **Ultimate Multisite** ନାମରେ ଏକ network-level menu entry ସୃଷ୍ଟି କରେ, ଯାହାର ତଳେ ଅନେକ sub-menu entry ଥାଏ।

## Network Dashboard Widgets {#network-dashboard-widgets}

Ultimate Multisite network-level dashboard ମଧ୍ୟରେ କିଛି ଉପଯୋଗୀ widgets ପ୍ରବେଶ କରେ। ଏହି dashboard ପାଇବା ପାଇଁ **My Sites → Network Admin → Dashboard** ବା କରିବାକୁ ହୁଏ। ଏହି widgets ସେତେବେଳେ network administratorଙ୍କ ପାଇଁ ସୂଚନା ଏବଂ ସାଧାରଣ କାର୍ଯ୍ୟଗୁଡ଼ିକରେ ସହଜ ପ୍ରବେଶ ଯୋଗାଇବା ପାଇଁ ଡିଜାଇନ୍ କରାଯାଇଛି।

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

### ପ୍ରଥମ ପଦକ୍ଷେପ (First Steps) {#first-steps}

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

ଏହି widget Ultimate Multisite ର ନୂଆ ଇନଷ୍ଟାଲେସନ୍ ଉପରେ ଦେଖାଯାଏ ଏବଂ network administratorଙ୍କୁ payment method କନଫିଗରେ କରିବା ଏବଂ ଏକ test account ସୃଷ୍ଟି କରିବା ଭଳି ଅତିରିକ୍ତ କାର୍ଯ୍ୟ ପୂରଣ କରିବାକୁ ସ୍ମରଣ କରେ।

### ସାରାଂଶ (Summary) {#summary}

![Summary widget](/img/admin/network-dashboard-summary.png)

Summary widget ଦିନିକିଆ signups ଏବଂ ସେମାନଙ୍କ ଆୟ ବିଷୟରେ ରିପୋର୍ଟ ପ୍ରଦାନ କରେ। MRR (Monthly Recurring Revenue) ମାସର ଅନୁକୂଳ ମୋଟ ପରିମାଣକୁ ଦର୍ଶାଏ, ଯାହା revenue-generating membership ଥିବା ଗ୍ରାହକମାନଙ୍କ ସମ୍ବନ୍ଧରେ ଆପଣାଯାଇଛି।

### କାର୍ଯ୍ୟ ପ୍ରବାହ (Activity Stream) {#activity-stream}

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

ଏହି widget Ultimate Multisite ମଧ୍ୟରେ ଘଟଣାଗୁଡ଼ିକ ବିଷୟରେ ରିପୋର୍ଟ ଦିଏ। ଘଟଣାଗୁଡ଼ିକ ମଧ୍ୟରେ signups, cancellation, site creation, deletion ଏବଂ ଅନ୍ୟାନ୍ୟ କାର୍ଯ୍ୟ ସାମିଲ ହୋଇଥାଏ।

ଅଧିକ ବିସ୍ତୃତ ବର୍ଣ୍ଣନା ପାଇଁ ଦୟାକରି documentation ର Events sectionକୁ ଦେଖନ୍ତୁ।

### ବର୍ତ୍ତମାନ (Right Now) {#right-now}

![Right Now widget](/img/admin/network-dashboard-right-now.png)

ଏହି ୟୁଟିଲିଟି ୱିଜେଟ୍ ନେଟୱାର୍କରେ ୟୁଜର୍ ଏବଂ ସାଇଟଗୁଡ଼ିକର ଏକ ଛୋଟ ସାରାସମୟ ଦେଖାଏ। ଉପରେ ଥିବା ହ୍ୟାଣ୍ଡି ଶର୍ଟକଟ୍‌ଗୁଡ଼ିକ ମାଧ୍ୟମରେ ନୂଆ ସାଇଟ କିମ୍ବା ୟୁଜର୍ ତିଆରି କରିବା ପାଇଁ ଏକ-କ୍ଲିକ୍ ପ୍ରବେଶ ମିଳିଥାଏ।

### ଖବର ଏବଂ ଆଲୋଚନା (News and Discussions) {#news-and-discussions}

![News and Discussions widget](/img/admin/network-dashboard-news.png)

ଏହି ୱିଜେଟ୍‌ଟି ଅଲ୍ଟିମେଟ ମల్ଟିସାଇଟ୍‌ର ନୂଆ ଖବରଗୁଡ଼ିକୁ ଆଣି ଏବଂ ପ୍ରଦର୍ଶନ କରେ। ଅପଡେଟ୍, ଗୁରୁତ୍ୱପୂର୍ଣ୍ଣ ବ୍ୟାଗ୍ ଏବଂ ସୁରକ୍ଷା ସଂଶୋଧନ ବିଷୟରେ ସୂଚନା ପାଇବା ପାଇଁ ଏହି ୱିଜେଟ୍‌କୁ ନଜର ରଖନ୍ତୁ।

## ଅଲ୍ଟିମେଟ ମల్ଟିସାଇଟ୍ ଡ୍ୟାଶବର୍ଡ (Ultimate Multisite Dashboard) {#ultimate-multisite-dashboard}

ଯେଉଁଠି ନେଟୱାର୍କ-ସ୍ତରୀୟ ଡ୍ୟାଶବର୍ଡ ନେଟୱାର୍କ ବିଷୟରେ ସମ୍ପୂର୍ଣ୍ଣ ସୂଚନା ଦେଖାଏ, ଅଲ୍ଟିମେଟ ମల్ଟିସାଇଟ୍ ଡ୍ୟାଶବର୍ଡ (ଯାହା ଅଲ୍ଟିମେଟ ମల్ଟିସାଇଟ୍ ମେନୁର ଉପରେ ଥିବ) ସେବା ବିଷୟରେ ସୂଚନା ପ୍ରଦାନ କରେ।

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

### ମାସିକ ପୁନରାବୃତ୍ତି ଆୟ ବୃଦ୍ଧି (Monthly Recurring Revenue Growth) {#monthly-recurring-revenue-growth}

MRR (ମାସିକ ପୁନରାବୃତ୍ତି ଆୟ) ଚାର୍ଟଟି ୧୨ ମାସର କ୍ୟାଲେଣ୍ଡର ଅବଧିରେ ଆୟର ମାସିକ ବିଭାଷଣ ଦେଖାଏ।

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

ନୂଆ ଆୟକୁ ନୀଳ ରଙ୍ଗରେ ଏବଂ ବ୍ୟାପାର ସୁଦ୍ଧା (cancellations) ଲାଲି ରଙ୍ଗରେ ପ୍ରତିଫଳିତ ହୋଇଥାଏ।

### ତାରିଖ-ପ୍ରେକ୍ଷୀ ସମର୍ଥନ (Date-range Support) {#date-range-support}

ଗ୍ରାହକ ଏବଂ କାର୍ଯ୍ୟକଳାପର ପରିମା ପର୍ଯ୍ୟନ୍ତ, ସ୍tatଟିଷ୍ଟିକ ଆବଶ୍ୟକତା ଅଧିକ ହୋଇପାରେ। ଡ୍ୟାଶବର୍ଡରେ ଥିବା ତାରିଖ-ପ୍ରେକ୍ଷୀ ସମର୍ଥନ ନେଟୱାର୍କ ପ୍ରଶାସକକୁ କେବଳ ଆବଶ୍ୟକ ବିବରଣୀ ଏବଂ ସମୟ ଅବଧି ଉପରେ ଧ୍ୟାନ ଦେବା ପାଇଁ ସୁଯୋଗ ଦିଏ।

![Date range selector and stats](/img/admin/um-dashboard-date-range.png)

ତାରିଖ-ପ୍ରେକ୍ଷୀ ବୈଶିଷ୍ଟ୍ୟଗୁଡ଼ିକ, ଯାହା ଆୟ, ନୂଆ ମେମ୍ବେରସ୍କିପ୍, ଦେଶ ଅନୁଯାୟୀ ସାଇନ୍ଅପ୍, ଫର୍ମ ଅନୁଯାୟୀ ସାଇନ୍ଅପ୍ ଏବଂ ସବୁଠାରୁ ଅଧିକ ଭେଟିଥିବା ସାଇଟଗୁଡ଼ିକ ବାଛିବା ପାଇଁ ଉପରେ ଥିବା ୱିଜେଟ୍‌ଗୁଡ଼ିକ ଦ୍ୱାରା ପ୍ରଦର୍ଶିତ ବିବରଣୀକୁ ନିୟନ୍ତ୍ରଣ କରେ।

![Dashboard statistics widgets](/img/admin/um-dashboard-stats.png)

:::tip
**Most Visited Sites** ୱିଜେଟ୍‌ଟି କାର୍ଯ୍ୟ କରିବା ପାଇଁ, **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** ରେ ବିଜିଟ୍ ଭିଶିଟ୍ ଗଣନା ଅନେକ (visit counting) ସୁବିଧା ଦିଅନ୍ତୁ।
:::
