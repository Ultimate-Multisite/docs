# সাইট বিল্ডার মোড সরানোর বিজ্ঞপ্তি {#site-builder-mode-removal-notice}

**Superdav AI Agent v1.12.0-এ Site Builder mode সরিয়ে ফেলা হয়েছে।** আপনি যদি Site Builder mode ব্যবহার করতেন, তবে থিম তৈরি এবং সাইট সেটআপের জন্য **Setup Assistant agent** এ স্থানান্তরিত হওয়া উচিত।

## কী ঘটেছে? {#what-happened}

### Site Builder Mode (পুরানো) {#site-builder-mode-legacy}

Site Builder mode ছিল নিচের কাজগুলোর জন্য একটি উইজার্ড-ভিত্তিক ইন্টারফেস:

- টেমপ্লেট থেকে সাইট তৈরি করা
- প্রাথমিক সেটিংস কনফিগার করা
- থিম নির্বাচন করা
- প্রাথমিক কন্টেন্ট সেটআপ করা

### এর পরিবর্তে কী এসেছে? {#what-replaced-it}

**Setup Assistant agent** এখন নিচের বিষয়গুলো দিয়ে সমস্ত Site Builder ফাংশন পরিচালনা করে:

- আরও নমনীয়, এজেন্ট-নির্দেশিত সেটআপ
- থিম কাস্টমাইজ করার উন্নত বিকল্প
- Theme Builder অনবোর্ডিংয়ের সাথে ইন্টিগ্রেশন
- ভবিষ্যতের সেশনের জন্য স্থায়ী site_brief মেমরি

## আপনি যদি Site Builder Mode ব্যবহার করতেন {#if-you-were-using-site-builder-mode}

### আপনার সাইটগুলো নিরাপদ {#your-sites-are-safe}

- Site Builder mode দিয়ে তৈরি বিদ্যমান সাইটগুলো কাজ করা চালিয়ে যাবে
- কোনো ডেটা ক্ষতি বা সাইটের কোনো বিঘ্ন ঘটবে না
- আপনি স্বাভাবিকভাবে আপনার সাইটগুলো পরিচালনা করতে পারবেন

### Setup Assistant Agent এ স্থানান্তরিত হন {#migrate-to-setup-assistant-agent}

নতুন সাইট সেটআপ বা থিম পরিবর্তনের জন্য, Setup Assistant agent ব্যবহার করুন:

```
"Help me set up a new site"
```

অথবা

```
"Start the Theme Builder onboarding"
```

Setup Assistant agent একই কার্যকারিতা আরও বেশি নমনীয়তার সাথে প্রদান করে।

## তুলনা: Site Builder বনাম Setup Assistant {#comparison-site-builder-vs-setup-assistant}

| বৈশিষ্ট্য | Site Builder (সরানো হয়েছে) | Setup Assistant (নতুন) |
|---------|----------------------|----------------------|
| সেটআপ পদ্ধতি | উইজার্ড ফর্ম | এজেন্ট কথোপকথন |
| থিম নির্বাচন | পূর্বনির্ধারিত টেমপ্লেট | কাস্টম তৈরি |
| কাস্টমাইজেশন | সীমিত বিকল্প | সম্পূর্ণ ডিজাইন সিস্টেম |
| সাইট ব্রিফ | সংরক্ষণ করা হয়নি | স্থায়ী মেমরি |
| ভবিষ্যতের সেশন | পুনরাবৃত্তি সেটআপ | সংরক্ষিত site_brief ব্যবহার করুন |
| নমনীয়তা | স্থির ওয়ার্কফ্লো | অভিযোজিত কথোপকথন |

## Setup Assistant Agent এ মাইগ্রেট করা {#migrating-to-setup-assistant-agent}

### নতুন সাইটের জন্য {#for-new-sites}

साइट বিল্ডার মোড ব্যবহার না করার জন্য:

১. অনুরোধ: "আমার নতুন সাইট সেটআপ করতে সাহায্য করুন"
২. সেটআপ অ্যাসিস্ট্যান্ট এজেন্ট আপনাকে নিম্নলিখিত বিষয়গুলিতে গাইড করবে:
   - সাইটের উদ্দেশ্য এবং লক্ষ্য
   - টার্গেট অডিয়েন্স (লক্ষ্য দর্শক)
   - ব্র্যান্ড আইডেন্টিটি (ব্র্যান্ড পরিচিতি)
   - থিম তৈরি
   - প্রাথমিক কনফিগারেশন

### বিদ্যমান সাইটগুলির জন্য {#for-existing-sites}

যদি আপনার Site Builder মোডের মাধ্যমে একটি বিদ্যমান সাইট থাকে:

১. আপনি এটিকে যেমন আছে তেমনই ব্যবহার করতে পারেন।
২. থিম আপডেট করার জন্য অনুরোধ করুন: "আমার সাইটটি নতুন করে ডিজাইন করুন"
৩. সেটআপ অ্যাসিস্ট্যান্ট এজেন্ট আপনাকে একটি নতুন থিম তৈরি করতে সাহায্য করবে।
৪. আপনার সাইটের ডেটা অপরিবর্তিত থাকবে।

### থিম পরিবর্তনের জন্য {#for-theme-changes}

Site Builder মোডের থিম নির্বাচনের পরিবর্তে:

১. অনুরোধ: "আমার থিম পরিবর্তন করুন"
২. সেটআপ অ্যাসিস্ট্যান্ট এজেন্ট যা করবে:
   - আপনার ডিজাইনের পছন্দ সম্পর্কে জিজ্ঞাসা করবে
   - একটি কাস্টম থিম তৈরি করবে
   - আপনার সাইটে এটি সক্রিয় করবে

## মূল পার্থক্যসমূহ {#key-differences}

### Site Builder Mode {#site-builder-mode}

```
১. টেমপ্লেট নির্বাচন করুন
২. থিম নির্বাচন করুন
৩. বেসিক সেটিংস কনফিগার করুন
৪. সম্পন্ন
```

### Setup Assistant Agent {#setup-assistant-agent}

```
১. আপনার সাইটের উদ্দেশ্য বর্ণনা করুন
২. আপনার টার্গেট অডিয়েন্স সংজ্ঞায়িত করুন
৩. ডিজাইনের পছন্দগুলি বেছে নিন
৪. এজেন্ট কাস্টম থিম তৈরি করবে
৫. এজেন্ট থিম সক্রিয় করবে
৬. ভবিষ্যতের সেশনের জন্য সাইট ব্রিফ সংরক্ষণ করা হবে
```

## সেটআপ অ্যাসিস্ট্যান্ট এজেন্টের সুবিধা {#benefits-of-setup-assistant-agent}

### আরও নমনীয় (More Flexible) {#more-flexible}

- স্বাভাবিক ভাষায় আপনার সাইট বর্ণনা করুন।
- কাস্টম সুপারিশ পান।
- আপনার নির্দিষ্ট প্রয়োজনের সাথে মানিয়ে নিন।

### উন্নত কাস্টমাইজেশন (Better Customization) {#better-customization}

- কাস্টম থিম তৈরি।
- ডিজাইন সিস্টেমের সিদ্ধান্ত গ্রহণ।
- স্থায়ী ডিজাইন টোকেন।

### স্থায়ী মেমরি (Persistent Memory) {#persistent-memory}

- আপনার `site_brief` সংরক্ষিত থাকে।
- ভবিষ্যতের এজেন্টগুলি আপনার সাইট সম্পর্কে বুঝতে পারে।
- সেটআপ তথ্য বারবার দেওয়ার প্রয়োজন নেই।

### সমন্বিত ওয়ার্কফ্লো (Integrated Workflow) {#integrated-workflow}

- থিম বিল্ডার অনবোর্ডিং।
- ডিজাইন সিস্টেম নান্দনিক দক্ষতা (Design System Aesthetics skill)।
- দৃশ্যমানতা নিয়ন্ত্রণ করার ক্ষমতা (Ability Visibility controls)।
- সবকিছু নির্বিঘ্নে একসাথে কাজ করে।

## সমস্যা সমাধান (Troubleshooting) {#troubleshooting}

### আমি Site Builder মোড খুঁজে পাচ্ছি না {#i-cant-find-site-builder-mode}

Site Builder মোড সরিয়ে ফেলা হয়েছে। এর পরিবর্তে সেটআপ অ্যাসিস্ট্যান্ট এজেন্ট ব্যবহার করুন:

"ମୋର ଏକ ନୂଆ ସାଇଟ୍ सेटअप କରିବାରେ ମୋଡ଼ି ଆଶୀର୍ବାଦ କରନ୍ତୁ"

### ମୁଁ Site Builder ରୁ ଏକ ସାଇଟ୍ ପୁଣି ତିଆରି କରିବାକୁ ଚାହୁଁଛି {#i-want-to-recreate-a-site-from-site-builder}

ଆପଣ Setup Assistant agent ବ୍ୟବହାର କରି ଏହାକୁ ପୁଣି ତିଆରି କରିପାରିବେ:

1. ଅନୁରୋଧ କରନ୍ତୁ: "ମୋର ଏକ ନୂଆ ସାଇଟ୍ सेटअप କରିବାରେ ମୋଡ଼ି ଆଶୀର୍ବାଦ କରନ୍ତୁ"
2. ଆପଣଙ୍କ ମୂଳ ସାଇଟ୍‌ର ଉଦ୍ଦେଶ୍ୟ ଓ ଡିଜାଇନ୍ ବିଷୟରେ ବର୍ଣ୍ଣନା କରନ୍ତୁ
3. ଏହି agent ଗୋଟିଏ ସମାନ থିମ୍ (theme) ତିଆରି କରିବ
4. ଆପଣଙ୍କର site_brief ଭବିଷ୍ୟତ ଅବହରଣ ପାଇଁ ସଞ୍ଚୟ ହେବ

### ମୋର ବିଲୁଡ଼ି Site Builder ସାଇଟ୍ କାମ କରୁନାହିଁ {#my-existing-site-builder-site-isnt-working}

Site Builder ମାଧ୍ୟମରେ ତିଆରି ହୋଇଥିବା ସାଇଟ୍ଗୁଡ଼ିକ ଆଜି ମଧ୍ୟ କାମ କରୁଛି। ଯଦି ଆପଣ ଏଠାରେ କୌଣସି ସମସ୍ୟା ଅନୁଭବ କରୁଛନ୍ତି:

1. ଯାଞ୍ଚ କରନ୍ତୁ ଯେ ଆପଣଙ୍କ থିମ୍ (theme) ଆଜି ମଧ୍ୟ ակଟିଭ (active) ଅଛି
2. ଯାଞ୍ଚ କରନ୍ତୁ ଆପଣଙ୍କ ପ୍ଲଗିନ୍ସ (plugins) ଏକଟିଏ ଚାଲିଛି କି ନାହିଁ
3. WordPress error logs ଯାଞ୍ଚ କରନ୍ତୁ
4. ସମସ୍ୟା ଲାଗିଲେ ସହାୟତା ପାଇଁ ସପୋର୍ଟକୁ ଯୋଗ ଦିଅନ୍ତୁ

### ମୁଁ ମୋର ପୁରୁଣା Site Builder template ବ୍ୟବହାର କରିପାରିବି କି? {#can-i-still-use-my-old-site-builder-templates}

Site Builder template ବର୍ତ୍ତମାନ ଉପଲବ୍ଧ ନୁହେଁ। କିନ୍ତୁ:

- ଆପଣଙ୍କର ବିଲୁଡ଼ି ସାଇଟ୍ଗୁଡ଼ିକ କାମ କରୁଛି
- Setup Assistant agent ମାଧ୍ୟମରେ ଏକ ସମାନ ସାଇଟ୍ ପୁଣି ତିଆରି କରିପାରିବେ
- Setup Assistant agent ଆପଣଙ୍କ ପାଇଁ ଅଧିକ କଷ୍ଟମାଇଜେସନ୍ (customization) ବିକଳ୍ପ ଦେଇଥାଏ

## ଆଗକୁ କ'ଣ କରିବା? {#next-steps}

1. **ନୂଆ ସାଇଟ୍‌ ପାଇଁ**: Setup Assistant agent ବ୍ୟବହାର କରନ୍ତୁ
2. **ବିଲୁଡ଼ି ସାଇଟ୍ ପାଇଁ**: ଏମାନଙ୍କୁ ଯେପରି ଅଛି, ସେଥିରେ ବ୍ୟବହାର କରିବା ଆରମ୍ଭ କରନ୍ତୁ
3. **ଥିମ୍ ବଦଳାଇବା ପାଇଁ**: Setup Assistant agent ଠାରୁ ସାହାଯ୍ୟ ମାଗନ୍ତୁ
4. **ଡିଜାଇନ୍ ସୁଧାର ପାଇଁ**: Design System Aesthetics skill ବ୍ୟବହାର କରନ୍ତୁ
