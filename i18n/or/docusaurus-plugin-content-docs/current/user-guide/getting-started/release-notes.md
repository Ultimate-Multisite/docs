# ରିଲିସ୍ ନୋଟ୍ସ (Release Notes) {#release-notes}

## ବର୍ଷ ୨.୧୩.୦ — ୨୦୨୬-୦୬-୦୫ ରେ ରିଲିସ୍ ହୋଇଛି {#version-2130--released-on-2026-06-05}

- ନୂଆ: କଷ୍ଟମର୍ ଖାତା, ଚେକ୍ଆଉଟ୍, ବିଲିଂ, ସାଇଟ୍, ଆଇନଭଏସ୍, ଟେମ୍ପଲେଟ ବଦଳାଇବା ଏବଂ ଡୋमेन ମ୍ୟପିଂ ପ୍ରଣାଳୀ ପାଇଁ ସୋଭରେନ୍-ଟେଣ୍ଟ (sovereign-tenant) ସମର୍ଥନ ଯୋଗ କରାଯାଇଛି, ଯାହା ଦ୍ୱାରା ଟେଣ୍ଟ ନେଟୱର୍କଗୁଡ଼ିକ ମ୍ୟାନ ସାଇଟ୍‌କୁ ପଛକୁ ନେଇ ମ୍ୟାନେଜ୍ କରାଯାଇଥିବା କାର୍ଯ୍ୟ ପାଇଁ ଗ୍ରାହକମାନଙ୍କୁ ନିର୍ଦ୍ଦେଶ ଦେଇପାରିବ।
- ନୂଆ: ପୁନରୁଆ ସଦସ୍ୟତା ପାଇଁ ନୂଆ ଅନୁମତି ପ୍ରମାଣିକତା (renewal-credential checks) ଯୋଗ କରାଯାଇଛି, ଯାହାଦ୍ୱାରା ଗେଟୱେୟାରେ ସଞ୍ଚୟ ହୋଇଥିବା ବିଲିଂ ଆଗ୍ରୀମତି, ସବସ୍କ୍ରିପସନ୍ ବା ଭାଉଣ୍ଟ ଟୋକନ୍ ଅଭାବରେ ଅଟୋ-ରିନୁଏଲ୍ ବଦଳାଇ ଦେବାକୁ ଅସମର୍ଥ କରାଯିବ।
- ନୂଆ: ପେଣ୍ଡିଂ ସାଇଟ୍ ସୃଷ୍ଟି ପାଇଁ HMAC-ବିଶିଷିତ ଲୁପବ୍ୟାହନ (HMAC-verified loopback publishing) ଯୋଗ କରାଯାଇଛି, ଯାହାଦ୍ୱାରା ବ୍ୟାଗ୍ରାଉଣ୍ଡ ଜବ୍‌ଗୁଡ଼ିକ ବିଳମ୍ବ ହେବା ସମୟରେ ଅସ୍ତେଶିତ ହୋଷ୍ଟଗୁଡ଼ିକରେ ଚେକ୍ଆଉଟ୍-କୁ-ସାଇଟ୍ ପ୍ରଭିତି (provisioning) ଅଧିକ ନିର୍ଭରଯୋଗ୍ୟ କରିବ।
- ନୂଆ: SSO URL, ଚେକ୍ଆଉଟ୍-ଫର୍ମ ବେସ୍ ଡୋମେନ୍ ଏବଂ ଆଟୋମାଟିକ୍ ଡୋमेन-ରେକର୍ଡ ସୃଷ୍ଟି ପାଇଁ ଡେଭଲପର ଏକ୍ସଟେଣ୍ଡ୍ ପଏଣ୍ଟ (developer extension points) ଯୋଗ କରାଯାଇଛି।
- ମରାମତି: ମ୍ୟାପ୍ କରାଯାଇଥିବା ଡୋमेन, ଆନୋନିମସ୍ ବ୍ରୋକର୍ ବିଜିତ, ଲଗଆଉଟ ଏବଂ କ୍ରସ-ପ୍ଲଗିନ୍ ଦେପେଣ୍ଡେନ୍ସ କ୍ଲାଷ୍ଟର ସଂଘର୍ଷରେ SSO ଅଧିକ ନିର୍ଭରଯୋଗ୍ୟ ହୋଇଛି।
- ମରାମତି: ପେଣ୍ଡିଂ ସାଇଟ୍ ସୃଷ୍ଟି ବର୍ତ୍ତମାନ ଅବୁରଣ (stale publish flags) ରୁ ସଂଶୋଧନ କରିପାରୁଛି ଏବଂ ଗ୍ରାହକମାନଙ୍କୁ ସାଇଟ୍-ସୃଷ୍ଟି ଫର୍ମରେ ଅବିଚ୍ଛେଦ କରୁଥିବା ପରି ବାନ୍ଧି ରଖିବାରୁ ବଞ୍ଚିବ।
- ମରାମତି: ସାଧାରଣ ଗ୍ରାହକ ପ୍ରବାହକୁ ବାଧା ଦେଉଥିବା ଚେକ୍ଆଉଟ୍, ବିଲିଂ ଆଡ୍ରେସ୍, ପାସୱାର୍ଡ ରିସେଟ, ଇମେଲ୍ ଭେରିଫିକେସନ୍, ଟେମ୍ପଲେଟ ବଦଳାଇବା, ଟୁର୍‌ରେ ଏବଂ କଷ୍ଟମର୍ ଡ୍ୟାଶବୋର୍ଡର ଅନ୍ତରାଳ ସମସ୍ୟାଗୁଡ଼ିକ ବର୍ତ୍ତମାନ ନିୟମିତ ଗ୍ରାହକ ପ୍ରବାହକୁ ବାଧା ଦେଉନାହିଁ।
- ମରାମତି: ବ୍ରାଡକାଷ୍ଟ୍ ଇମେଲ୍ ବର୍ତ୍ତମାନ ଗ୍ରହକମାନଙ୍କୁ ବ୍ୟକ୍ତିଗତ ରଖିବାରେ ସାହାଯ୍ୟ କରୁଛି, ଏବଂ ଗ୍ରହକତା ତାଲିକା ବା ମେଲ୍ ଟ୍ରାନ୍ସପୋର୍ଟ୍ ବିଫଳ ହେବା ସମୟରେ SMTP/ପ୍ଲଗିନ୍ ଭେତା (fatal errors) ରୁ ଦୂରେଇ ରହିଛି।
- ମରାମତି: ସଦସ୍ୟତାର ପୁନରୁଆ, ବର୍ତ୍ତମାନ ପ୍ରକାଶ କରିବା ଏବଂ ପେମେଣ୍ଟ ସଂଗ୍ରହର ଅନ୍ତରାଳ ସମସ୍ୟାଗୁଡ଼ିକ ବର୍ତ୍ତମାନ ତୁରନ୍ତ ବନ୍ଦ ହେବା, କ୍ରାଶ୍ (crashes) ବା ଆବଶ୍ୟକୀୟ ପେମେଣ୍ଟ ମିସ୍ କରିବାରୁ ଦୂରେଇ ରହିଛି।
- ଉନ୍ନତି: WordPress ସହିତ ଅନୁସଙ୍ଗତତା 7.0 ପର୍ଯ୍ୟନ୍ତ ପରୀକ୍ଷା କରାଯାଇଛି, ପ୍ରଡକ୍ସନ୍ Vue ଆସେଟ୍‌ଗୁଡ଼ିକ npm ଉତ୍ସର
