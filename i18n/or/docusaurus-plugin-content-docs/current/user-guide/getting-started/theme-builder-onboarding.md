# থিম বিল্ডার অনবোর্ডিং ফ্লো (Theme Builder Onboarding Flow)

Superdav AI Agent v1.12.0 আপনার প্রাথমিক সেটআপের সময় কাস্টম ব্লক থিম তৈরি করতে সাহায্য করার জন্য একটি গাইড করা **থিম বিল্ডার অনবোর্ডিং ফ্লো** চালু করেছে। এটি পুরনো সাইট বিল্ডার মোডটিকে আরও নমনীয়, এজেন্ট-সহায়ক পদ্ধতির সাথে প্রতিস্থাপন করে।

## থিম বিল্ডার অনবোর্ডিং ফ্লো কী?

থিম বিল্ডার অনবোর্ডিং ফ্লো হলো একটি ইন্টারেক্টিভ সেটআপ উইজার্ড যা:

- ডিজাইন সিদ্ধান্তগুলিতে আপনাকে গাইড করে (রঙ, টাইপোগ্রাফি, লেআউট)
- আপনার সাইটের ভিজ্যুয়াল আইডেন্টিটি পছন্দগুলি সংগ্রহ করে
- আপনার প্রয়োজন অনুসারে কাস্টম ব্লক থিম তৈরি করে
- সম্পন্ন হলে স্বয়ংক্রিয়ভাবে থিমটি সক্রিয় করে দেয়

এই ফ্লোটি **Setup Assistant agent** দ্বারা চালিত হয়, যা স্পষ্ট করার জন্য প্রশ্ন জিজ্ঞাসা করে এবং ধাপে ধাপে আপনার থিমটি তৈরি করে।

## থিম বিল্ডার অনবোর্ডিং শুরু করা

### প্রথমবার সেটআপ (First-Run Setup)

যখন আপনি নতুন ওয়ার্ডপ্রেস ইনস্টলেশনে প্রথমবারের মতো Superdav AI Agent চালু করবেন, তখন আপনি দেখতে পাবেন:

```
Superdav AI Agent-এ স্বাগতম!

আপনি কী করতে চান?
১. কাস্টম থিম তৈরি করুন (Theme Builder)
২. একটি বিদ্যমান থিম ব্যবহার করুন
৩. আপাতত সেটআপ এড়িয়ে চলুন
```

থিম বিল্ডার অনবোর্ডিং ফ্লোতে প্রবেশ করতে **"Build a custom theme"** নির্বাচন করুন।

### ম্যানুয়াল অ্যাক্টিভেশন (Manual Activation)

আপনি যেকোনো সময় নিম্নলিখিত কমান্ডগুলি অনুরোধ করে থিম বিল্ডার অনবোর্ডিং শুরু করতে পারেন:

```
"Start the Theme Builder onboarding"
```

অথবা

```
"Help me create a custom theme"
```

## অনবোর্ডিং ধাপসমূহ (The Onboarding Steps)

### ধাপ ১: মোড নির্বাচন (Mode Selection)

Setup Assistant agent আপনার পছন্দ সম্পর্কে জিজ্ঞাসা করবে:

```
আপনি কীভাবে আপনার থিম তৈরি করতে চান?
- গাইড করা (আমি প্রশ্ন করব এবং আপনার জন্য এটি তৈরি করে দেব)
- হাতে-কলমে (আমি আপনাকে বিকল্পগুলি দেখাব এবং আপনি সিদ্ধান্ত নেবেন)
```

বেশিরভাগ ব্যবহারকারীর জন্য **গাইড করা মোড** সুপারিশ করা হয়; এজেন্ট আপনার শিল্প এবং লক্ষ্যগুলির উপর ভিত্তি করে ডিজাইনের পরামর্শ দেবে।

### ধাপ ২: সাইট নির্দিষ্টকরণ (Site Specification)

আপনার সাইটের বিষয়ে আপনাকে জিজ্ঞাসা করা হবে:

- **साइटର উদ্দেশ্য**: ই-কমার্স, ব্লগ, পোর্টফোলিও, SaaS ইত্যাদি।
- **লক্ষ্য দর্শক**: আপনার ভিজিটর কারা?
- **ব্র্যান্ড রং**: প্রাথমিক এবং মাধ্যমিক রং (অথবা "আমার নিশ্চিত নই")
- **সুর**: পেশাদার, সৃজনশীল, মজাদার, ন্যূনতম ইত্যাদি।

এই তথ্যগুলো আপনার **site_brief** মেমোরিতে সংরক্ষণ করা থাকে, যে রেফারেন্স হিসেবে এজেন্টরা ভবিষ্যতের সেশনে এগুলো ব্যবহার করে।

### ধাপ ৩: ডিজাইন সিস্টেমের সিদ্ধান্তসমূহ

এজেন্ট আপনাকে ডিজাইন টোকেন নির্বাচনের মাধ্যমে পথ দেখাবে:

- **টাইপোগ্রাফি**: ফন্ট পরিবার (সেরিফ, স্যানস-সেরিফ, মনোপেস), এবং সাইজ স্কেল
- **রঙের প্যালেট**: প্রাথমিক, মাধ্যমিক, অ্যাকসেন্ট এবং নিউট্রাল রং
- **স্পেসিং**: কম্প্যাক্ট, নরমাল বা স্পেশাস লেআউট
- **মোশন**: অ্যানিমেশন এবং ট্রানজিশন (যদি চান)

### ধাপ ৪: থিম তৈরি

সেটআপ অ্যাসিস্ট্যান্ট এজেন্ট আপনার কাস্টম ব্লক থিমটি নিম্নলিখিত জিনিসগুলো দিয়ে কাঠামোবদ্ধ করবে:

- আপনার সমস্ত ডিজাইন টোকেন সহ `theme.json`
- সাধারণ লেআউটের জন্য ব্লক টেমপ্লেট (হোমপেজ, ব্লগ, কন্টাক্ট)
- আপনার ডিজাইন সিস্টেমের সাথে মিলে যাওয়া কাস্টম ব্লক স্টাইল
- থিম মেটাডেটা এবং ওয়ার্ডপ্রেস সাপোর্ট ঘোষণা

### ধাপ ৫: সক্রিয়করণ এবং যাচাইকরণ

থিমটি স্বয়ংক্রিয়ভাবে সক্রিয় হয়ে যাবে, এবং আপনি দেখতে পাবেন:

```
✓ আপনার কাস্টম থিম এখন লাইভ!
  থিমের নাম: [আপনার সাইটের নাম] থিম
  রং: [প্রাথমিক] / [মাধ্যমিক]
  টাইপোগ্রাফি: [ফন্ট পরিবার]

  নতুন ডিজাইনটি দেখতে আপনার সাইট ভিজিট করুন।
```

এরপর আপনি থিমটি সঠিকভাবে প্রদর্শিত হচ্ছে কিনা তা যাচাই করার জন্য আপনার সাইট ভিজিট করতে পারেন।

## সাইট স্পেসিফিকেশন এবং site_brief মেমরি

অনবোর্ডিংয়ের সময়, এজেন্ট আপনার সাইটের নির্দিষ্ট বিবরণ একটি **site_brief** মেমোরি ক্যাটাগরিতে সংগ্রহ করে। এর মধ্যে অন্তর্ভুক্ত থাকে:

- সাইটের উদ্দেশ্য এবং লক্ষ্য
- লক্ষ্য দর্শক
- ব্র্যান্ড রং এবং সুর (Tone)
- ডিজাইনের পছন্দসমূহ
- কন্টেন্ট কাঠামো

### site_brief কেন গুরুত্বপূর্ণ

ভবিষ্যতে সেশনগুলোতে, এজেন্টরা আপনার site_brief রেফারেন্স হিসেবে ব্যবহার করবে যাতে তারা...

### আপনার সাইটের সংক্ষিপ্ত বিবরণ দেখা (Viewing Your site_brief)

আপনি এজেন্টকে জিজ্ঞাসা করতে পারেন:

```
"Show me my site brief"
```

অথবা

```
"What do you know about my site?"
```

এজেন্ট আপনার সংরক্ষিত সাইট স্পেসিফিকেশনটি দেখাবে।

## অনবোর্ডিং-এর পরে কাস্টমাইজ করা (Customizing After Onboarding)

Theme Builder অনবোর্ডিং শেষ হওয়ার পর, আপনি যা করতে পারেন:

### ডিজাইন সিস্টেমের নান্দনিক দক্ষতা ব্যবহার করা (Use the Design System Aesthetics Skill)

ডিজাইনের পরিবর্তন করার অনুরোধ করতে পারেন:

```
"Refine the typography to be more modern"
```

অথবা

```
"Adjust the color palette to be warmer"
```

**Design System Aesthetics skill** আপনাকে নির্দিষ্ট ডিজাইনের আপডেটগুলো কীভাবে করতে হবে, তা ধাপে ধাপে বলে দেবে।

### theme.json সরাসরি সম্পাদনা করা (Edit theme.json Directly)

উন্নত ব্যবহারকারীদের জন্য, আপনি `/wp-content/themes/[theme-name]/theme.json` ফাইলটি এডিট করে নিচের বিষয়গুলো পরিবর্তন করতে পারেন:

- Color tokens
- Typography scales
- Spacing values
- Border এবং shadow এর সংজ্ঞা

### কাস্টম ব্লক টেমপ্লেট তৈরি করা (Create Custom Block Templates)

ওয়ার্ডপ্রেস ব্লক এডিটর ব্যবহার করে নিচের জিনিসগুলোর জন্য কাস্টম টেমপ্লেট তৈরি করুন:

- Homepage layouts
- Blog post pages
- Product pages
- Contact forms

## সমস্যা সমাধান (Troubleshooting)

**অনবোর্ডিং প্রক্রিয়াটি সম্পন্ন হয়নি (The onboarding flow didn't complete)**
- প্রক্রিয়াটি আবার শুরু করুন: "Start the Theme Builder onboarding"
- নিশ্চিত করুন যে আপনার WordPress ইনস্টলেশনটি আপ টু ডেট আছে।
- যাচাই করুন যে Setup Assistant এজেন্টটি চালু করা আছে কিনা।

**ମୋର site_brief ବ୍ୟବହାର କରାଯାଉନାହିଁ**
- ଯାଞ୍ଚ କରନ୍ତୁ ଯେ ଆପଣଙ୍କ ଇଜେଣ୍ଟ ପାଠ୍ୟ (memory) ରେ ପ୍ରବେଶ କରିପାରୁଛନ୍ତି।
- ଇଜେଣ୍ଟକୁ "ମୋର site brief ମନେ ରଖ" ବୋଲି କୁହନ୍ତୁ।
- ଯାଞ୍ଚ କରନ୍ତୁ ଆପଣଙ୍କ ସେଟିଙ୍ଗରେ ମନେ ରଖ (memory) ଅନ୍ୟାନ୍ୟ ବିଶେଷତା ଭାବରେ ଏକେଇଁ ହୋଇଛି କି ନାହିଁ।

**ଉତ୍ପନ୍ନ କରାଯାଇଥିବା ธีମ୍ ମୋର ପସନ୍ଦ ସହିତ ମେଳ ହେଉନାହିଁ**
- ଏହାକୁ ସୁଧାରିବା ପାଇଁ Design System Aesthetics କୌଶଳ ବ୍ୟବହାର କରନ୍ତୁ।
- ଇଜେଣ୍ଟକୁ "[ବିଶେଷ ପରିବର୍ତ୍ତନ] ସହିତ ธีମ୍ ନୂଆ କରିଦିଅ" ବୋଲି କୁହନ୍ତୁ।
- ସଠିକ୍ ନିୟନ୍ତ୍ରଣ ପାଇଁ theme.json ରେ ସିଧାସଳଖ ଏହାକୁ एडिट କରନ୍ତୁ।

## ଆଗକୁ କ'ଣ କରିବେ (Next Steps)

Theme Builder ଶପ୍ତକାରୀ (onboarding) ସମାପ୍ତ କରିବା ପରେ:

1. **ଆପଣଙ୍କ ସାଇଟ୍ ଯାଞ୍ଚ କରନ୍ତୁ**: ନୂଆ ธีମ୍ ଦେଖିବା ପାଇଁ ଆପଣଙ୍କ ସାଇଟ୍‌କୁ ବୁଲନ୍ତୁ।
2. **ଡିଜାଇନ୍ ସୁଧାରନ୍ତୁ**: ଅନୁସଙ୍ଗତ ପରିବର୍ତ୍ତନ କରିବା ପାଇଁ Design System Aesthetics କୌଶଳ ବ୍ୟବହାର କରନ୍ତୁ।
3. **ଅବସ୍ଥା ଯୋଗ କରନ୍ତୁ**: ଆପଣଙ୍କ ସାଇଟ୍‌ର ବିଷୟବସ୍ତୁ (content) ତିଆରି କରିବା ଆରମ୍ଭ କରନ୍ତୁ।
4. **କୌଶଳଗୁଡ଼ିକୁ ଅନୁଭବ କରନ୍ତୁ**: scaffold-block-theme ଏବଂ activate-theme ଭଳି ଅନ୍ୟାନ୍ୟ agent କୌଶଳ ବିଷୟରେ ଜାଣନ୍ତୁ।
