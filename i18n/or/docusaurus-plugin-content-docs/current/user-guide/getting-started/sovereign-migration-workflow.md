# ସୋଭରେନ୍ ମାଇଗ୍ରେସନ୍ ୱାର୍କଫ୍ {#sovereign-migration-workflow}

Ultimate Multisite: Multi-Tenancy 1.2.0 ଆପଣଙ୍କୁ ଏକ ସ୍ଥାପତ୍ୟିକ (sovereign) ଟେଣ୍ଟ୍ ଭିତରେ ସ୍ଥାପନ କରିବା ପାଇଁ ମାନକ ସବସାଇଟକୁ ମିଗ୍ରେଟ କରିବା ପୂର୍ବରୁ ଯାଞ୍ଚ କରାଇବା ପାଇଁ ମାଇଗ୍ରେସନ୍ ଭେରିଫିକେସନ୍ ଗେଟ୍ (verification gates) ଯୋଡ଼ିଛି।

## ଆରମ୍ଭ କରିବା ପୂର୍ବରୁ {#before-you-start}

ଆପଣ ନିଶ୍ଚିତ କରନ୍ତୁ ଯେ ଆପଣଙ୍କ ପାଖରେ ଅଛି:

- ସ୍ରୋସ साईटର ଏକ ବର୍ତ୍ତମାନର ବ୍ୟାକ୍ଅପ୍।
- Multi-Tenancy addon ୍ଚାର୍ଯ୍ୟ କରୁଥିବା ଅଛି।
- ଗନ୍ତବଳୀ (destination) ଟେଣ୍ଟ୍ ଡାଟାବେସ୍, ଫାଇଲସିଷ୍ଟମ୍ ରୁଟ ଏବଂ ଡୋमेन ପ୍ରସ୍ତୁତ।
- ଆପଣଙ୍କର ଡେଷ୍ଟିનેସନ୍ ପରିବେଶ ସହିତ ମେଳ ହେଉଥିବା ଡାଟାବେସ୍ ହୋଷ୍ଟ ବାଇଣ୍ଡିଂ (database host binding)।
- ନେଟୱାର୍କ ପାଇଁ WP-CLI କମାଣ୍ଡ୍ ଚଲାଇବାକୁ ଅନୁମତି।

## ଆବଶ୍ୟକୀୟ ୱାର୍କଫ୍ {#recommended-workflow}

1. ଗନ୍ତବଳୀ ଟେଣ୍ଟ୍ ଡାଟାବେସ୍ ଏବଂ ଫାଇଲସିଷ୍ଟମ୍ ପ୍ରସ୍ତୁତ କରନ୍ତୁ।
2. ଟେଣ୍ଟ isolation ସେଟିଙ୍ଗ୍‌କୁ ରେଜିଷ୍ଟର କରନ୍ତୁ ବା ଅପଡେଟ୍ କରନ୍ତୁ।
3. ଟେଣ୍ଟ ମାଇଗ୍ରେସନ୍ ଚଲାଇବାକୁ ଦିଅନ୍ତୁ।
4. ଟେଣ୍ଟ୍ ୟୁଜର୍‌ମାନଙ୍କୁ ପ୍ରସ୍ତୁତ କରନ୍ତୁ ବା ଯାଞ୍ଚ କରନ୍ତୁ।
5. ଅସଙ୍ଗତିକ ମାଇଗ୍ରେସନ୍ ଜବ୍‌କୁ (async migration jobs) ଡ୍ରେନ୍ (drain) କରନ୍ତୁ।
6. ସୋଭରେନ୍ ମାଇଗ୍ରେସନ୍ ଭେରିଫିକେସନ୍ ଚଲାଇବାକୁ ଦିଅନ୍ତୁ।
7. SSO ସହିତ ଟେଣ୍ଟ୍‌କୁ ବୁଝନ୍ତୁ (Visit with SSO)।
8. ଭେରିଫିକେସନ୍ ପାସ ହେବା ପରେ କେବଳ DNS ବଦଳାଇବା କିମ୍ବା ରୁଟିଂ (routing) କରିବା।

## ଭେରିଫିକେସନ୍ ଗେଟ୍‌ଗୁଡ଼ିକ {#verification-gates}

ଭେରିଫିକେସନ୍ ୱାର୍କଫ୍ ମାଇଗ୍ରେସନ୍‌କୁ ଅନେକ ଦିଗରୁ ଯାଞ୍ଚ କରେ:

- ଟେଣ୍ଟ isolation ସ୍କିମା (schema) ଅଛି ଏବଂ ଟେଣ୍ଟ ଡାଟାବେସ୍ ରାଇଟର୍ ଦ୍ୱାରା ଲେଖାଯାଇପାରିବ।
- ବିବେଷଣୀ କରାଯାଇଥିବା ଡାଟାବେସ୍ ହୋଷ୍ଟରେ ବୈଧ ଅନୁମତି (grants) ଅଛି।
- ଟେଣ୍ଟ୍ ୟୁଜର୍‌ମାନମାନେ ଅଛନ୍ତି ଏବଂ ଆଶା କରାଯାଉଥିବା ସୋଭରେନ୍ ଇନଷ୍ଟଲ୍ ୟୁଜର୍ ଗଣନା ସହିତ ମେଳ ହେଉଛି।
- ଅସଙ୍ଗତିକ ପୁష్ କ୍ୟୁ (async push queue) ସଫଳତାର ସହ ଡ୍ରେନ୍ ହୋଇପାରିବ।
- ସୋଭରେନ୍ ଟେଣ୍ଟ୍‌ ପାଇଁ ପୂର୍ବରୁ ନେଟୱାର୍କ-ପକ୍ଷର ଡାଟା ପଥ (data paths) ଆଉ ଆବଶ୍ୟକ ନୁହେଁ।

ଯଦି ଭେରିଫିକେସନ୍ ବିଫଳ ହୁଏ, ତାହାକୁ ପ୍ରୋଜେକ୍ଟ୍ ଆରମ୍ଭ କରିବା ପୂର୍ବରୁ ବାଧା ସୃଷ୍ଟି କରୁଥିବା କାରଣ ଭାବରେ ଗ୍ରହଣ କରନ୍ତୁ। ଡାଟାବେସ୍, ୟୁଜର୍, କ୍ୟୁ, କିମ୍ବା ରୁଟିଂ ସମସ୍ୟାକୁ ମରାମତି କରିବା ପରେ, ଟେଣ୍ଟ୍‌କୁ କଷ୍ଟମର୍ସଙ୍କ ପାଇଁ ପ୍ରଦର୍ଶନ କରିବା ପୂର୍ବରୁ ପୁଣିଥରେ ଭେରିଫିକେସନ୍ ଚଲାଇଲେ।

## ପ୍ରଥମ ପ୍ରଡକ୍ସନ୍ ବୁଝିବା (First production visit) {#first-production-visit}

ଭେରିଫିକେସନ୍ ପାସ ହେବା ପରେ, ପ୍ରଥମ ଇଡିନ ଆଡମିନ୍ ବିଜିଟ୍‌ಗಾಗಿ ସାଇଟ ମ୍ୟାନେଜମେଣ୍ଟ ସ୍କ୍ରିନ୍
