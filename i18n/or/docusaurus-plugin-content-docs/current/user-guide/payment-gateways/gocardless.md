# GoCardless ଗେଟୱେୟ (Gateway)

ଏହି ଅବଶ୍ୟକତାଟି **GoCardless** ଆଡଉନ୍ (addon) ଦ୍ୱାରା ପ୍ରଦାନ କରାଯାଇଛି।

ସମ୍ପୂର୍ଣ୍ଣ ବିବରଣୀ ପାଇଁ, [GoCardless ଆଡଉନ୍ ଡକ୍ୟୁମେଣ୍ଟेशन](https://addons.wordpress.org/category/woocommerce/gocardless/) ଦେଖନ୍ତୁ।
