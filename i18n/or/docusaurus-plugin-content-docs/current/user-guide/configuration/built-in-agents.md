# ଅନ୍ତର୍ନିହିତ ଏଜେଣ୍ଟଗୁଡ଼ିକ

Gratis AI Agent v1.9.0 ରେ ପାଞ୍ଚଟି ଅନ୍ତର୍ନିହିତ ଏଜେଣ୍ଟ ମିଳେ। ପ୍ରତ୍ୟେକଟି ନିର୍ଦ୍ଦିଷ୍ଟ କାମ ପାଇଁ ଉପକରଣ ସମୂହ, ଅନୁକୂଳିତ ସିଷ୍ଟମ୍ ନିର୍ଦ୍ଦେଶ ଏବଂ ସେହି କ୍ଷେତ୍ରର ସାଧାରଣ କାମ ସହ ମେଳ ଥିବା ଆରମ୍ଭିକ ପରାମର୍ଶ ସହ ପୂର୍ବରୁ ସଜାଯାଇଛି। ଏଜେଣ୍ଟ ବଦଳାଇଲେ ସହାୟକ କଣ କରିପାରିବ ଏବଂ କିପରି ଉତ୍ତର ଦେବ, ତାହା ବଦଳେ — ଆପଣଙ୍କ ପକ୍ଷରୁ କୌଣସି ସଜାଣା ଦରକାର ନାହିଁ। ସମ୍ବନ୍ଧିତ ସମନ୍ୱୟଗୁଡ଼ିକ ସଜାଇଦିଆଯାଇଥିଲେ Superdav AI Agent v1.18.0 ଏହି କାର୍ଯ୍ୟପ୍ରବାହଗୁଡ଼ିକରେ ସୂଚୀ-ସଚେତନ ଉପକରଣ, ସ୍ମରଣିକା ରେକର୍ଡ, ଅନୁମୋଦନ ଦ୍ୱାର ଏବଂ SMS ସୂଚନା ଯୋଡ଼ିପାରେ।

## ଏଜେଣ୍ଟ କଣ?

ପ୍ରତ୍ୟେକ ଏଜେଣ୍ଟ ହେଉଛି ଏକ ନାମିତ ସଜାଣା ପ୍ରୋଫାଇଲ୍, ଯାହା ଏହିଗୁଡ଼ିକୁ ଏକାଠି କରେ:

- **ଉପକରଣ** — ଏଜେଣ୍ଟ ବ୍ୟବହାର କରିପାରୁଥିବା କ୍ଷମତାଗୁଡ଼ିକ (ଉଦାହରଣ ସ୍ୱରୂପ, ଜଣେ ଲେଖା ଲେଖକଙ୍କ ପାଖରେ ପୋଷ୍ଟ-ତିଆରି କ୍ଷମତାର ପ୍ରବେଶ ଥାଏ; ଜଣେ ଡିଜାଇନ୍ ଷ୍ଟୁଡିଓଙ୍କ ପାଖରେ CSS ଏବଂ theme.json କ୍ଷମତାର ପ୍ରବେଶ ଥାଏ)
- **ସିଷ୍ଟମ୍ ନିର୍ଦ୍ଦେଶ** — ଏଜେଣ୍ଟର ଭାବ, ପ୍ରାଥମିକତା ଏବଂ ସୀମାବନ୍ଧନ ନିର୍ଦ୍ଧାରଣ କରୁଥିବା ନିର୍ଦ୍ଦେଶ
- **ପରାମର୍ଶ** — ଶୀଘ୍ର ଆରମ୍ଭ କରିବାରେ ସହାୟତା ପାଇଁ ଚାଟ୍ ଇଣ୍ଟରଫେସ୍‌ରେ ଦେଖାଯାଉଥିବା ପୂର୍ବଲିଖିତ ପ୍ରମ୍ପ୍ଟ

## ଏଜେଣ୍ଟ ଚୟନକାରୀକୁ ପ୍ରବେଶ

1. WordPress ପ୍ରଶାସନ ସାଇଡ୍‌ବାରରେ **Gratis AI Agent** ପ୍ୟାନେଲ୍ ଖୋଲନ୍ତୁ।
2. ଚାଟ୍ ହେଡର୍‌ର ଉପର-ବାମ ପାର୍ଶ୍ୱରେ ଥିବା **ଏଜେଣ୍ଟ ଆଇକନ୍** କ୍ଲିକ୍ କରନ୍ତୁ (ସକ୍ରିୟ ଏଜେଣ୍ଟ ଅନୁସାରେ ଆଇକନ୍ ବଦଳେ)।
3. **ଏଜେଣ୍ଟ ଚୟନକାରୀ** ଫର୍ମ-ଟେବୁଲ୍ ଓଭରଲେ ଭାବେ ଖୋଲେ। ପ୍ରତ୍ୟେକ ଏଜେଣ୍ଟ ତାହାର ଆଇକନ୍, ନାମ ଏବଂ ଗୋଟିଏ-ଧାଡ଼ି ବର୍ଣ୍ଣନା ସହ ତାଲିକାଭୁକ୍ତ ହୁଏ।
4. ଏଜେଣ୍ଟକୁ ସକ୍ରିୟ କରିବାକୁ ତାହାର ଧାଡ଼ି କ୍ଲିକ୍ କରନ୍ତୁ। ଚାଟ୍ ହେଡର୍ ସତେଜ ଭାବେ ଅଦ୍ୟତନ ହୁଏ।

ଆପଣ ଆଲୋଚନା ମଝିରେ ମଧ୍ୟ ଏଜେଣ୍ଟ ବଦଳାଇପାରିବେ — ନୂତନ ଏଜେଣ୍ଟର ସିଷ୍ଟମ୍ ନିର୍ଦ୍ଦେଶ ପରବର୍ତ୍ତୀ ସନ୍ଦେଶରୁ ପ୍ରଭାବୀ ହୁଏ।

## ପାଞ୍ଚଟି ଅନ୍ତର୍ନିହିତ ଏଜେଣ୍ଟ

### ଲେଖା ଲେଖକ

**କେନ୍ଦ୍ରବିନ୍ଦୁ:** ପୋଷ୍ଟ, ପୃଷ୍ଠା ଏବଂ ସମ୍ପର୍କ ଫର୍ମ ତିଆରି ଓ ସମ୍ପାଦନ।

**ଉପଲବ୍ଧ ଉପକରଣ:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`। Superdav AI Agent v1.18.0 ସମନ୍ୱୟଗୁଡ଼ିକ ସକ୍ରିୟ ଥିଲେ, ସଜାଇଦିଆଯାଇଥିବା କ୍ୟାଲେଣ୍ଡର୍ ପରିପ୍ରେକ୍ଷ୍ୟ, ଅନୁମୋଦନ ଦ୍ୱାର, ସ୍ମରଣିକା ଏବଂ SMS ସୂଚନା ଉପକରଣଗୁଡ଼ିକ ମଧ୍ୟ ଅନୁମୋଦିତ କାର୍ଯ୍ୟପ୍ରବାହ ପାଇଁ ଉପଲବ୍ଧ ହୋଇପାରେ।

**ଏହା ଯାହା ଭଲ କରେ:**
- ସଂକ୍ଷିପ୍ତ ବିବରଣୀ କିମ୍ବା ରୂପରେଖାରୁ ବ୍ଲଗ୍ ପୋଷ୍ଟ ଖସଡ଼ା କରି ପ୍ରକାଶ କରିବା
- ନୂତନ ସାଇଟ୍ ପାଇଁ ଲ୍ୟାଣ୍ଡିଂ ପୃଷ୍ଠାର ବ୍ୟାଚ୍ ତିଆରି କରିବା
- ସମ୍ପର୍କ ଏବଂ ପ୍ରଶ୍ନ ଫର୍ମ ତିଆରି କରିବା
- URL କିମ୍ବା ଖୋଜରୁ ପୋଷ୍ଟରେ ବିଶେଷ ଛବି ସେଟ୍ କରିବା
- ସଜାଇଦିଆଯାଇଥିବା Google Calendar ପରିପ୍ରେକ୍ଷ୍ୟରୁ କାର୍ଯ୍ୟକ୍ରମ ପରବର୍ତ୍ତୀ ସନ୍ଦେଶ ଖସଡ଼ା କରିବା, ତାପରେ ସୂଚନା ପଠାଇବା ପୂର୍ବରୁ ଅନୁମୋଦନ ପାଇଁ ରୁକିବା

**ଆରମ୍ଭିକ ପରାମର୍ଶ:**
- *WordPress multisite ର ଲାଭ ବିଷୟରେ ୫୦୦-ଶବ୍ଦର ଏକ ବ୍ଲଗ୍ ପୋଷ୍ଟ ଲେଖନ୍ତୁ।*
- *ଏକ ପରିଚୟ, ସେବା ଏବଂ ସମ୍ପର୍କ ପୃଷ୍ଠା ତିଆରି କରି ସେଗୁଡ଼ିକୁ ପ୍ରକାଶ କରନ୍ତୁ।*
- *ସମ୍ପର୍କ ପୃଷ୍ଠାରେ ଏକ ବୁକିଂ ପ୍ରଶ୍ନ ଫର୍ମ ଯୋଡ଼ନ୍ତୁ।*
- *ଆସନ୍ତାକାଲିର ସଜାଇଦିଆଯାଇଥିବା କ୍ୟାଲେଣ୍ଡର୍ କାର୍ଯ୍ୟକ୍ରମର ଅଂଶଗ୍ରାହୀମାନଙ୍କ ପାଇଁ ଏକ ସ୍ମରଣିକା ଖସଡ଼ା କରନ୍ତୁ ଏବଂ ପଠାଇବା ପୂର୍ବରୁ ଅନୁମୋଦନ ପାଇଁ ଅପେକ୍ଷା କରନ୍ତୁ।*

### ସାଇଟ୍ ନିର୍ମାତା

**କେନ୍ଦ୍ରବିନ୍ଦୁ:** ଗୋଟିଏ ପ୍ରମ୍ପ୍ଟରୁ ଆରମ୍ଭ କରି ସମ୍ପୂର୍ଣ୍ଣ ୱେବସାଇଟ୍ ତିଆରି।

**ଉପଲବ୍ଧ ଉପକରଣ:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`। Superdav AI Agent v1.18.0 ସହିତ, ପ୍ରଶାସକମାନେ ସକ୍ରିୟ କରିଥିବା ସ୍ଥାନରେ ସଜାଇଦିଆଯାଇଥିବା ପରିଚାଳିତ-ସେବା, ଅନୁମୋଦନ, ସ୍ମରଣିକା, କ୍ୟାଲେଣ୍ଡର୍ ଏବଂ SMS ଉପକରଣ ଉପଲବ୍ଧ ହୋଇପାରେ।

**ଏହା ଯାହା ଭଲ କରେ:**
- ବର୍ଣ୍ଣିତ ବ୍ୟବସାୟ ପ୍ରକାର ପାଇଁ ବହୁ-ପର୍ଯ୍ୟାୟ ସାଇଟ୍ ନିର୍ମାଣ ଯୋଜନା ସୃଷ୍ଟି କରିବା
- ପ୍ରତ୍ୟେକ ପର୍ଯ୍ୟାୟ ସ୍ୱୟଂଚାଳିତ ଭାବେ କାର୍ଯ୍ୟକରିତ କରିବା — ଗଠନ, ବିଷୟବସ୍ତୁ, ନାଭିଗେସନ୍, ଡିଜାଇନ୍
- ହସ୍ତଚାଳିତ ହସ୍ତକ୍ଷେପ ଦରକାର ନକରି ଯୋଜନା ମଝିରେ ତ୍ରୁଟିରୁ ପୁନରୁଦ୍ଧାର କରିବା
- ନିର୍ମାଣର ଅଂଶ ଭାବେ ପରାମର୍ଶିତ ପ୍ଲଗିନ୍ ସ୍ଥାପନ କରିବା
- ଚାଟ୍ ଇଣ୍ଟରଫେସ୍‌ରୁ ସିଧାସଳଖ ସମ୍ପର୍କ ଫର୍ମ ତିଆରି କରିବା (Superdav AI Agent v1.10.0+)
- ଅନୁମୋଦନ ଦ୍ୱାର ଏବଂ ସ୍ମରଣିକା ରେକର୍ଡ ସକ୍ରିୟ ଥିଲେ, ପୁନରାବୃତ୍ତ ସୂଚନା ବିନା ଲଞ୍ଚ ସ୍ମରଣିକା କିମ୍ବା ଅଂଶଗ୍ରାହୀ ପରବର୍ତ୍ତୀ ଯୋଗାଯୋଗ ସମନ୍ୱୟ କରିବା

**ଆରମ୍ଭିକ ପରାମର୍ଶ:**
- *ଗ୍ୟାଲେରୀ ପୋଷ୍ଟ ପ୍ରକାର, ଏକ ବୁକିଂ ପୃଷ୍ଠା ଏବଂ ଏକ ସମ୍ପର୍କ ଫର୍ମ ସହ ଏକ ଫଟୋଗ୍ରାଫି ପୋର୍ଟଫୋଲିଓ ସାଇଟ୍ ତିଆରି କରନ୍ତୁ।*
- *ଅନଲାଇନ୍ ମେନୁ, ଖୋଲା ସମୟ ଏବଂ ଏକ ଟେବୁଲ୍-ବୁକିଂ ପ୍ରଶ୍ନ ଫର୍ମ ସହ ଏକ ରେଷ୍ଟୁରାଣ୍ଟ ୱେବସାଇଟ୍ ତିଆରି କରନ୍ତୁ।*
- *ସେବା ପୃଷ୍ଠା, ପୋର୍ଟଫୋଲିଓ ବିଭାଗ ଏବଂ ଏକ ବ୍ଲଗ୍ ସହ ଏକ ଫ୍ରିଲାନ୍ସ ପରାମର୍ଶ ସାଇଟ୍ ସେଟ୍ ଅପ୍ କରନ୍ତୁ।*
- *ସାଇଟ୍ ନିର୍ମାତା ବ୍ୟବହାର କରି ସମ୍ପର୍କ ପୃଷ୍ଠାରେ ଏକ ସମ୍ପର୍କ ଫର୍ମ ଯୋଡ଼ନ୍ତୁ।*
- *ସାଇଟ୍ ଲଞ୍ଚ ଚେକଲିଷ୍ଟ ଅନୁମୋଦିତ ହେବା ପରେ, ସଜାଇଦିଆଯାଇଥିବା ହିତାଧିକାରୀ ସମ୍ପର୍କକୁ ଏକ SMS ସ୍ମରଣିକା ପଠାନ୍ତୁ।*

## ଏଜେଣ୍ଟମାନଙ୍କୁ କଷ୍ଟମାଇଜ୍ କରିବା

ପ୍ରତ୍ୟେକ ବିଲ୍ଟ-ଇନ୍ ଏଜେଣ୍ଟକୁ `gratis_ai_agent_agents` ଫିଲ୍ଟର୍ ମାଧ୍ୟମରେ ବିସ୍ତାର କିମ୍ବା ପରିବର୍ତ୍ତନ କରାଯାଇପାରେ।

### ଥିବା ଏଜେଣ୍ଟରେ କଷ୍ଟମ୍ ସିଷ୍ଟମ୍ ପ୍ରମ୍ପ୍ଟ ଯୋଡ଼ିବା

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### ନୂଆ ଏଜେଣ୍ଟ ନିବନ୍ଧନ କରିବା

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

ଫିଲ୍ଟର୍ ଚାଲିବା ସହିତ ସହିତ ନୂଆ ଏଜେଣ୍ଟ ଏଜେଣ୍ଟ ଚୟନକାରୀରେ ଦେଖାଯାଏ।

### ବିଲ୍ଟ-ଇନ୍ ଏଜେଣ୍ଟକୁ ହଟାଇବା

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
