# થીમ-જાગૃત કૌશલ્યો (Theme-Aware Skills)

Superdav AI Agent v1.10.0 માં ચાર નવા થીમ-જાગૃત બિલ્ટ-ઇન કૌશલ્યો આવ્યા છે જે તમારા સક્રિય વર્ડપ્રેસ થીમ સાથે આપોઆપ અનુકૂલિત થાય છે. આ કૌશલ્યો તમારા થીમની રચના અને સુવિધાઓને અનુરૂપ વિશેષ માર્ગદર્શન અને ક્ષમતાઓ પૂરી પાડે છે.

## થીમ-જાગૃત કૌશલ્યો શું છે?

થીમ-જાગૃત કૌશલ્યો એ પૂર્વ-રૂપક સેટ કરેલા જ્ઞાન આધાર (knowledge bases) અને ટૂલ સેટ્સ છે જે AI સહાયક તમારા સાઇટ પર હાલમાં સક્રિય વર્ડપ્રેસ થીમ પર આધારિત આપમેળે પસંદ કરે છે. જ્યારે તમે થીમ બદલો છો, ત્યારે સહાયકની ઉપલબ્ધ કૌશલ્યો આપોઆપ અપડેટ થઈ જાય છે — કોઈ મેન્યુઅલ રૂપરેખાંકન (configuration)ની જરૂર નથી.

દરેક કૌશલ્યમાં આ શામેલ હોય છે:

- **થીમ-વિશિષ્ટ દસ્તાવેજીકરણ** — થીમને વાપરવા અને કસ્ટમાઇઝ કરવા અંગે માર્ગદર્શન
- **બ્લોક અને પેટર્ન સંદર્ભો** — ઉપલબ્ધ બ્લોક્સ, પેટર્ન અને ડિઝાઇન વિકલ્પો
- **કસ્ટમાઇઝેશન ઉદાહરણો** — સામાન્ય કાર્યો માટે કોડ સ્નિપેટ્સ અને રૂપરેખાંકન પેટર્ન
- **શ્રેષ્ઠ પ્રથાઓ (Best practices)** — થીમની રચના અને વર્કફ્લો માટેની ભલામણો

## ઉપલબ્ધ થીમ-જાગૃત કૌશલ્યો

### બ્લોક થીમ્સ (Block Themes)

**લાગુ પડતું છે:** એવા થીમ્સ જે વર્ડપ્રેસ બ્લોક-આધારિત (Full Site Editing) આર્કિટેક્ચરનો ઉપયોગ કરે છે.

બ્લોક થીમ્સ કૌશલ્ય નીચેના બાબતો પર માર્ગદર્શન પૂરું પાડે છે:

- બ્લોક એડીટરનો ઉપયોગ કરીને ટેમ્પલેટ્સ બનાવવું અને સંપાદિત કરવા
- બ્લોક પેટર્ન અને પુનઃઉપયોગી બ્લોક્સ સાથે કામ કરવું
- વૈશ્વિક styles અને theme.json સેટિંગ્સને કસ્ટમાઇઝ કરવું
- થીમ બ્લોક્સ અને તેના વેરિએશન્સનો ઉપયોગ કરવો
- તમારી સાઇટ માટે કસ્ટમ બ્લોક પેટર્ન બનાવવું

**આ આપમેળે સક્રિય થાય છે જ્યારે:** તમારું સક્રિય થીમ એક બ્લોક થીમ હોય (જે `block-templates` સુવિધાને સપોર્ટ કરે છે).

### ક્લાસિક થીમ્સ (Classic Themes)

**લાગુ પડતું છે:** પરંપરાગત વર્ડપ્રેસ થીમ્સ જે PHP ટેમ્પલેટ્સ અને ક્લાસિક એડીટરનો ઉપયોગ કરે છે.

ક્લાસિક થીમ્સ કૌશલ્ય નીચેના બાબતો પર માર્ગદર્શન પૂરું પાડે છે:

PHP template files ଏବଂ hooks ସହିତ କାର୍ଯ୍ୟ କରିବା
କଷ୍ଟମାଇଜର ମାଧ୍ୟମରେ ଥିମ୍ ଆପେୟାରନ୍ସକୁ ବଦଳାଇବା
widget areas ଏବଂ sidebar ବ୍ୟବହାର କରିବା
CSS ପରିବର୍ତ୍ତିତ କରିବା ଏବଂ child theme ବିକାଶ

**ଯେତେବେଳେ ସ୍ୱୟଂଚିତ ଭାଙ୍ଗୁଛି:** ଆପଣଙ୍କର ଉଚ୍ଚମାନ (classic, non-block) ଥିମ୍ ବ୍ୟବହାର କରୁଛନ୍ତି।

### Kadence Blocks

**କେଉଁଠି ପ୍ରଯୋଜ୍ୟ:** ଆଧୁନିକ ବ୍ଲକ୍-ଆଧାରିତ ဒီဇိုင်း ପାଇଁ Kadence Blocks plugin ବ୍ୟବହାର କରୁଥିବା ସାଇଟ్‌ଗୁଡ଼ିକରେ।

Kadence Blocks ଦକ୍ଷତାଟି ନିମ୍ନଲିଖିତ ବିଷୟରେ ମାର୍ଗଦର୍ଶନ ପ୍ରଦାନ କରେ:

- Kadence ର ଆଧୁନିକ ବ୍ଲକ୍ ଲାଇବ୍ରେରୀ (Hero, Testimonials, Pricing, ଆଦି) ବ୍ୟବହାର କରିବା
- Kadence ବ୍ଲକ୍ ସେଟିଙ୍ଗ ଏବଂ responsive ବିକଳ୍ପଗୁଡ଼ିକୁ କନଫିଗ୍ୟୁରେଟ୍ କରିବା
- Kadence ର grid ଏବଂ container blocks ମାଧ୍ୟମରେ কଷ୍ଟମ Layouts ନିର୍ମିତ କରିବା
- ଆପଣଙ୍କ ଥିମ୍ ସହିତ Kadence blocksକୁ ବାନ୍ଧିବା (integrate)
- Kadence ର design system ଏବଂ presetsର ଉପଯୋଗ କରିବା

**କେଉଁଠି ସ୍ୱୟଂଚିତ ଭାଙ୍ଗୁଛି:** ଆପଣଙ୍କ ସାଇଟ୍‌ରେ Kadence Blocks plugin କାର୍ଯ୍ୟ କରୁଛି।

### Kadence Theme

**କେଉଁଠି ପ୍ରଯୋଜ୍ୟ:** ବ୍ଲକ୍-ଆଧାରିତ ဒီဇိုင်း ଏବଂ কଷ୍ଟମାଇଜେସନ୍ ପାଇଁ Kadence theme ବ୍ୟବହାର କରୁଥିବା ସାଇଟ్‌ଗୁଡ଼ିକରେ।

Kadence Theme ଦକ୍ଷତାଟି ନିମ୍ନଲିଖିତ ବିଷୟରେ ମାର୍ଗଦର୍ଶନ ପ୍ରଦାନ କରେ:

- global styles ଏବଂ theme.json ମାଧ୍ୟମରେ Kadence themeକୁ কଷ୍ଟମାଇଜ କରିବା
- Kadence ର built-in block patterns ଏବଂ templates ବ୍ୟବହାର କରିବା
- Kadence theme ସେଟିଙ୍ଗ ଏବଂ ଅପସନ୍ସ କନଫିଗ୍ୟୁରେଟ୍ କରିବା
- Kadence ର design system ମାଧ୍ୟମରେ কଷ୍ଟମ ဒီଜାଇନ୍ ନିର୍ମିତ କରିବା
- ପ୍ରସିଦ୍ଧ plugin ଏବଂ tools ସହିତ Kadenceକୁ ବାନ୍ଧିବା

**କେଉଁଠି ସ୍ୱୟଂଚିତ ଭାଙ୍ଗୁଛି:** Kadence theme ଆପଣଙ୍କର active theme।

## ଦକ୍ଷତାଗୁଡ଼ିକ କିପରି ଚୟନ କରାଯାଏ

ଆସିଥିବା ପ୍ରତିଟି ମେସେଜରେ, ଅସହାୟ (assistant) ଆପଣଙ୍କର active theme ଏବଂ install করা plugins များကို ସ୍ୱୟଂଚିତ ଭାବରେ ଚିହ୍ନଟ କରିଥାଏ। ଯଦି କୌଣସି ମେଳଥିବା theme-aware skill ಲହୀନ ଅଛି, ତେବେ ସେହି skillଟି ସ୍ୱୟଂଚିତ ଭାବରେ assistant ର context ମଧ୍ୟରେ ଲୋଡ ହୋଇଯାଏ। ଆପଣକୁ manual ଭାବରେ କୌଣସି skill অন କରିବା ବା ପରିବର୍ତ୍ତନ କରିବାର ଆବଶ୍ୟକତା ନାହିଁ।

### ଅନେକ ଦକ୍ଷତା (Multiple Skills)

ଯଦି ଆପଣଙ୍କ ସାଇଟ୍‌ରେ ଅନେକ କୌଶଳ (skill) ପ୍ରୟୋଗ ହୁଏ (ଉଦାହରଣ ସ୍ୱରୂପ, ଯଦି ଆପଣ Kadence Blocks ଏବଂ Kadence Theme ଉଭୟ ակଟିଭ୍ ରଖିଛନ୍ତି), ତେବେ ଅସିଷ୍ଟେଣ୍ଟଙ୍କ ପାଖରେ ସମସ୍ତ ପ୍ରଯୁଜ୍ୟ କୌଶଳଗୁଡ଼ିକର ପ୍ରବେଶ ଥାଏ ଏବଂ ସେ ପ୍ରତ୍ୟେକରଠାରୁ ମାର୍ଗଦର୍ଶନକୁ ଉଲ୍ଲେଖ କରିପାରିବ।

### ଥିମ୍ ବଦଳାଇବା (Switching Themes)

ଯେତେବେଳେ ଆପଣ ଆପଣଙ୍କର ակଟିଭ୍ ଥିମ୍ ବଦଳାନ୍ତି, ପରବର୍ତ୍ତୀ ମେସେଜରେ ଅସିଷ୍ଟେଣ୍ଟଙ୍କର లୋକ କୌଶଳଗୁଡ଼ିକ ଆଧାରରେ ଆଧାର କରି ସ୍ୱୟଂଚାଳିତ ଭାବରେ ଅପଡେଟ୍ ହୋଇଯାଏ। ଉଦାହରଣ:

1. ଆପଣ **Block Themes** କୌଶଳଟି ակଟିଭ୍ ରଖିଥିବା ଏକ ବ୍ଲକ୍ ଥିମ୍ ବ୍ୟବହାର କରୁଛନ୍ତି।
2. ଆପଣ ଏହାକୁ କ୍ଲାସିକ୍ ଥିମ୍‌ରେ ପରିବର୍ତ୍ତନ କରୁଛନ୍ତି।
3. ଆପଣଙ୍କ ପରବର୍ତ୍ତୀ ମେସେଜରେ, **Classic Themes** କୌଶଳଟି ସ୍ୱୟଂଚାଳିତ ଭାବରେ ଲୋଡ୍ ହୋଇଯିବ ଏବଂ **Block Themes** କୌଶଳଟି ଅପଲବିତ ହୋଇଯିବ।

## ଥିମ୍-ସଚେତକ କୌଶଳ ବ୍ୟବହାର କରିବା (Using Theme-Aware Skills)

ଏକ ଥିମ୍-ସଚେତକ କୌଶଳର ଲାଭ ନେବା ପାଇଁ, ଆପଣ ଚାଟ ଇଣ୍ଟରଫେସ୍‌ରେ ଆପଣ କ'ଣ କରିବାକୁ ଚାହୁଁଛନ୍ତି, ତାହା ବର୍ଣ୍ଣନା କଲେ ହେବ। ଅସିଷ୍ଟେଣ୍ଟ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ଉପଯୁକ୍ତ କୌଶଳର ମାର୍ଗଦର୍ଶନକୁ ଉଲ୍ଲେଖ କରିବ।

### ଉଦାହରଣ ପ୍ରମନ୍ତ (Example Prompts)

**Block Themes ପାଇଁ:**
> "ବ୍ଲକ୍ ପ୍ୟାଟର୍ନ୍ ବ୍ୟବହାର କରି ଏକ ହେରୋ ସେକ୍ସନ, ଯେଉଁଥିରେ ବ୍ୟାଗ୍ରାଉଣ୍ଡ ଇମେଜ୍ ଏବଂ ମଝିରେ ଲେଖା ରହିବ।"

**Classic Themes ପାଇଁ:**
> "ଚାଇଲ୍ ଥିମ୍ ବ୍ୟବହାର କରି साइडबारରେ ଏକ କଷ୍ଟମ୍ ୱିଜେଟ୍ ଆରିଆ ଅଡ଼ନ୍ତୁ।"

**Kadence Blocks ପାଇଁ:**
> "Kadence Testimonials ବ୍ଲକ୍ ବ୍ୟବହାର କରି ଏକ ଟେଷ୍ଟିମେଣ୍ଟସ ସେକ୍ସନ ବାନ୍ଧନ୍ତୁ।"

**Kadence Theme ପାଇଁ:**
> "ଆପଣଙ୍କର ଆକଟିଭ୍ ଥିମ୍ ଏବଂ ଉପଗ୍ରହନା (plugins) ଅନୁଯାୟୀ ହେଡର୍ ଲେଆଉଟ୍ ଏବଂ ନੈଭିଗେସନ୍ ମେନୁ ଷ୍ଟାଇଲିଂକୁ କଷ୍ଟମାଇଜ୍ କରନ୍ତୁ।"

ଅସିଷ୍ଟେଣ୍ଟ ଆପଣଙ୍କ ակଟିଭ୍ ଥିମ୍ ଏବଂ ପ୍ଲଗିନ୍‌ଗୁଡ଼ିକ ଅନୁଯାୟୀ ଥିମ୍-ବିଶେଷ ମାର୍ଗଦର୍ଶନ ଏବଂ କୋଡ୍ ଉଦାହରଣ ପ୍ରଦାନ କରିବ।
