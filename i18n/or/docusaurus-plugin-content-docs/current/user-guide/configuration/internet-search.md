# ఇంటర్నెట్ సెర్చ్ (Internet Search)

Gratis AI Agent v1.5.0 లో **ఇంటర్నెట్ సెర్చ్** సామర్థ్యం జోడించబడింది. దీనివల్ల, సంభాషణ సమయంలో AI అసిస్టెంట్ వెబ్‌సైట్ల నుండి తాజా సమాచారాన్ని పొందగలదు. ఇది [Brave Search API](https://brave.com/search/api/) లేదా [Tavily API](https://tavily.com/) ద్వారా పనిచేస్తుంది.

## ఇది ఎలా పనిచేస్తుంది (How It Works)

ఇంటర్నెట్ సెర్చ్ ఎనేబుల్ చేసినప్పుడు, ప్రశ్నలకు ప్రస్తుత లేదా బాహ్య సమాచారం అవసరమని AI గుర్తించినప్పుడు, అది మీ కాన్ఫిగర్ చేసిన సెర్చ్ ప్రొవైడర్‌ను ఆటోమేటిక్‌గా ప్రశ్నించగలదు. ఉదాహరణకు, తాజా వార్తలు, లైవ్ ధరలు లేదా మోడల్ శిక్షణ ముగిసినప్పటి నుండి మారిన డాక్యుమెంటేషన్ వంటివి.

ఫలితాలను నిజ సమయంలో (real time) తీసుకుని, AI సమాధానం రూపొందించే ముందు దాని సందర్భంలోకి (context) చేరుస్తుంది. ఒక ప్రశ్నలకు సమాధానం ఇవ్వడానికి అది సెర్చ్ ఫలితాలను ఉపయోగించిందని AI తెలియజేస్తుంది.

## ఇంటర్నెట్ సెర్చ్‌ను ఎలా ఎనేబుల్ చేయాలి (Enabling Internet Search)

ఇంటర్నెట్ సెర్చ్‌కు మీరు ఎంచుకున్న సెర్చ్ ప్రొవైడర్ నుండి ఒక API key అవసరం. దాన్ని కాన్ఫిగర్ చేయడానికి:

1. **Gratis AI Agent → Settings → Advanced** కి వెళ్లండి.
2. **Internet Search Provider** డ్రాప్‌డౌన్‌ను కనుగొని, **Brave Search** లేదా **Tavily** ని ఎంచుకోండి.
3. మీ API key ను దానికి సంబంధించిన ఫీల్డ్‌లో నమోదు చేయండి. ప్రతి ఫీల్డ్ పక్కన సైన్-అప్ లింక్‌లు కనిపిస్తాయి.
4. **Save Settings** పై క్లిక్ చేయండి.

కీ సేవ్ అయిన తర్వాత, ఇంటర్నెట్ సెర్చ్ సామర్థ్యం ఆటోమేటిక్‌గా అసిస్టెంట్‌కు అందుబాటులోకి వస్తుంది.

## Brave Search

### Brave Search API Key పొందడం (Obtaining a Brave Search API Key)

1. [Brave Search API page](https://brave.com/search/api/) ను సందర్శించండి.
2. ఒక ప్లాన్ కోసం సైన్ అప్ చేయండి. నెలవారీ రిక్వెస్ట్ పరిమితితో కూడిన ఉచిత స్థాయి (free tier) అందుబాటులో ఉంటుంది.
3. Brave Search డెవలపర్ డాష్‌బోర్డ్ నుండి మీ API key ను కాపీ చేయండి.
4. దానిని Gratis AI Agent సెట్టింగ్‌లలో ఉన్న **Brave Search API Key** ఫీల్డ్‌లో పేస్ట్ చేయండి.

### వినియోగ పరిమితులు (Usage Limits)

ব্যবহারের চার্জ ব্রেইভ সার্চ দ্বারা করা কোয়েরির সংখ্যার ওপর নির্ভর করে। প্রতিটি এআই রেসপন্স যা সার্চ ট্রিগার করে, তা একটি কোয়েরি হিসেবে গণ্য হয়। অপ্রত্যাশিত চার্জ এড়াতে [Brave Search developer dashboard](https://brave.com/search/api/) এ আপনার ব্যবহার পর্যবেক্ষণ করুন।

## Tavily

Superdav AI Agent v1.10.0 **Tavily** কে একটি বিকল্প ইন্টারনেট সার্চ প্রোভাইডার হিসাবে যোগ করেছে, যা আরও সমৃদ্ধ সার্চ রেজাল্ট এবং উন্নত গবেষণা ক্ষমতা প্রদান করে।

### Tavily API কী পাওয়া

১. [Tavily API page](https://tavily.com/) ভিজিট করুন।
২. একটি অ্যাকাউন্ট খুলুন। মাসিক অনুরোধের সীমা সহ একটি ফ্রি টিয়ার উপলব্ধ।
৩. Tavily ড্যাশবোর্ড থেকে আপনার API কী কপি করুন।
৪. Gratis AI Agent সেটিংসের **Tavily API Key** ফিল্ডে এটি পেস্ট করুন।

### ব্যবহারের সীমা

ব্যবহারের চার্জ Tavily দ্বারা করা API কলের সংখ্যার ওপর নির্ভর করে। প্রতিটি এআই রেসপন্স যা সার্চ ট্রিগার করে, তা একটি কল হিসেবে গণ্য হয়। অপ্রত্যাশিত চার্জ এড়াতে [Tavily dashboard](https://tavily.com/) এ আপনার ব্যবহার পর্যবেক্ষণ করুন।

## ইন্টারনেট সার্চ বন্ধ করা

অ্যাক্টিভ সার্চ প্রোভাইডার ফিল্ড থেকে API কী সরিয়ে দিন এবং সেভ করুন। এরপর অ্যাসিস্ট্যান্টের জন্য ইন্টারনেট সার্চ করার ক্ষমতা আর থাকবে না।

:::note
ইন্টারনেট সার্চ রেসপন্সে দেরি যোগ করে কারণ উত্তর তৈরি করার আগে অ্যাসিস্ট্যান্টকে সার্চ রেজাল্টের জন্য অপেক্ষা করতে হয়। সময়-সংবেদনশীল ব্যবহারের ক্ষেত্রে, রিয়েল-টাইম সার্চ কি প্রয়োজন তা বা অ্যাসিস্ট্যান্টের অন্তর্নির্মিত জ্ঞান যথেষ্ট কিনা তা বিবেচনা করুন।
:::
