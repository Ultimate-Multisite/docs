# ଭାଷା ସେଲକ୍ଟର (Language Selector) {#language-selector}

ଏହି ଅବଶ୍ୟକତାଟି **Language Selector** ଆଡଉନ୍ (addon) ଦ୍ୱାରା ପ୍ରଦାନ କରାଯାଇଛି।

ସମ୍ପୂର୍ଣ୍ଣ ଡକ୍ୟୁମେଣ୍ଟ ପାଇଁ, [Language Selector ଆଡଉନ୍ ଡକ୍ୟୁମେଣ୍ଟକୁ](https://addons.wordpress.org/en/addon/language-selector/) ଦେଖନ୍ତୁ।
