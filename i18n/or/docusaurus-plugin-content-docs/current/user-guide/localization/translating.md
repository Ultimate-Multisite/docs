# Ultimate Multisite ର ଅନୁବାଦ {#translating-ultimate-multisite}

Ultimate Multisite ଇଂରାଜରେ ବିକଶିତ ହୋଇଥିଲେ, କିନ୍ତୁ ଏହି ปลug-ଇନ୍ (plugin) ସମ୍ପୂର୍ଣ୍ଣ ଅନୁବାଦୀୟ ଏବଂ ଆମେ ସକ୍ରିୟ ଭାବରେ ଅନୁବାଦ ପ୍ରଦାନକୁ ସ୍ୱାଗତ କରୁଛୁ।

## ମୂଳ प्लug-ଇନ୍ ଅନୁବାଦ (Core Plugin Translations) {#core-plugin-translations}

ମୂଳ Ultimate Multisite pluginଟି  official **WordPress.org ଅନୁବାଦ ପ୍ଲାଟଫର୍ମ** ମାଧ୍ୟମରେ ଅନୁବାଦ କରାଯାଇଛି:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

ଏହା ହେଉଛି WordPress କମ୍ୟୁନିଟିର ମାନକ ଅନୁବାଦ ବ୍ୟବସ୍ଥା। ଯେତେବେଳେ ଅନୁବାଦଗୁଡ଼ିକ ଆବଶ୍ୟକ ସ୍ୱୀକୃତି ଧାଡ଼ି ପୂର୍ଣ୍ଣ କରିଦେଲେ, ସେଗୁଡ଼ିକ WordPress plugin update ମାଧ୍ୟମରେ Ultimate Multisite ର ସମସ୍ତ ବ୍ୟବହାରକାରୀଙ୍କ ପାଖରେ ଆତ୍ମଗତ ଭାବରେ ବିତରଣ କରାଯାଏ।

**ଆମକୁ ଆପଣଙ୍କ ସାହାଯ୍ୟ ଦରକାର!** ଯଦି ଆପଣ ଇଂରାଜ ବ୍ୟତୀତ ଅନ୍ୟ କୌଣସି ଭାଷାରେ କଥା କହନ୍ତି, ତେବେ ଅନୁବାଦ ପ୍ରଦାନ କରିବା ବିଚାର କରିପାରନ୍ତୁ। କିଛି ସ୍Tr(strings) ଅନୁବାଦ କରିବା ମଧ୍ୟ ସାହାଯ୍ୟ କରିବ। ଆରମ୍ଭ କରିବା ପାଇଁ:

1. [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/) ଉପରେ ଯାଆନ୍ତୁ
2. ଆପଣଙ୍କ ଭାଷାଟି ବାଛନ୍ତୁ
3. "Contribute Translation" କ୍ଲିକ୍ କରନ୍ତୁ
4. ଆପଣଙ୍କ ପାଇଁ ଏକ ମଜବୁତ WordPress.org account ଥିବା ଆବଶ୍ୟକ

ଯଦି ଆପଣ ଆପଣଙ୍କ ଭାଷା ପାଇଁ ଅନୁବାଦ ସମ୍ପାଦକ (translation editor) ହେବାକୁ ଚାହୁଁଛନ୍ତି (ଅନୁବାଦଗୁଡ଼ିକ ଅନୁମୋଦନ କରିବାର କ୍ଷମତା), ତେବେ [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) ମାଧ୍ୟମରେ ଆସନ୍ତୁ।

## Addon ଅନୁବାଦ (Addon Translations) {#addon-translations}

Addon plugins ଗୁଡ଼ିକ ବିଶେଷ ଭାବରେ ଏଠାରେ ଅନୁବାଦ କରାଯାଇଛି:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

ସର୍ବବେଳେ Addon ଅନୁବାଦଗୁଡ଼ିକ ଆରମ୍ଭ କରି AI ଦ୍ୱାରା ସୃଷ୍ଟି କରାଯାଇଛି। ଏହା ସମସ୍ତ ସମର୍ଥିତ ଭାଷାକୁ ବ୍ୟାପୀ କରେ, କିନ୍ତୁ ଏଥିରେ କେବେକ ଅସଙ୍ଗତ ଉପଯୋଗ (awkward phrasing) ବା ଅନୁଚ୍ଛେଦ ମଧ୍ୟ ରହିପାରେ। ଆମେ ମଣିଷୀୟ ଅନୁବାଦକମାନଙ୍କୁ ସେଗୁଡ଼ିକ ଯାଞ୍ଚ କରିବା ଏବଂ ଆହୁରି ସ୍ୱଭାବିକ ଓ ସଠିକ୍ କରିବା ପାଇଁ ପ୍ରସ୍ତୁତି କରୁଛୁ।

## ସ୍ଥାନୀୟ ଅନୁବାଦ (Local Translations) {#local-translations}

ଆପଣ ନିଜ ଉପରେ ପ୍ରୋଜେକ୍ଟ ପାଇଁ ସ୍ଥାନୀୟ ଅନୁବାଦ କରିବାକୁ ଚାହୁଁଛନ୍ତି, ଅନ୍ୟମାନଙ୍କ ପାଇଁ କାର୍ଯ୍ୟ କରିବା ବଦଳରେ, ଆପଣ ଦୁଇଟି ବିକଳ୍ପର ବ୍ୟବହାର କରିପାରିବେ:

### Poedit ବ୍ୟବହାର କରିବା {#using-poedit}

ଆପଣ `.pot` ଫାଇଲକୁ ଅନୁବାଦ କରିବା ପାଇଁ **[Poedit](https://poedit.net)** ଡେସ୍କଟପ୍ ଆପ୍လီକେସନ୍ ବ୍ୟବହାର କରିପାରିବେ:

1. [poedit.net](https://poedit.net) ରୁ Poedit ଡାଉନଲୋଡ କରନ୍ତୁ
2. `wp-content/plugins/ultimate-multisite/lang/` ଠାରୁ `.pot` ଫାଇଲଟି ଖୋଲନ୍ତୁ
3. Poedit ଇଣ୍ଟରଫେସ୍ ମଧ୍ୟରେ ସ୍Trିଂଗଗୁଡ଼ିକ ଅନୁବାଦ କରନ୍ତୁ
4. సేವ್ କରନ୍ତୁ -- Poedit ଆପଣଙ୍କ ପାଇଁ `.po` ଏବଂ `.mo` ଫାଇଲ୍ ଗୁଡ଼ିକ ତିଆରି କରିଦେବ
5. ଆଉଟପୁట్ ଫାଇଲଗୁଡ଼ିକୁ `lang` ఫోల్డర్‌లో ରଖନ୍ତୁ

### Loco Translate ବ୍ୟବହାର କରିବା {#using-loco-translate}

**[Loco Translate](https://wordpress.org/plugins/loco-translate/)** WordPress ପ୍ଲଗିନ୍ ଆପଣଙ୍କୁ ଡେସ୍କଟପ୍ ଟୁଲ୍ ବିନା ଅଡମିନ୍ ପ୍ୟାନେଲରୁ ସିଧାସଳଖ ଅନୁବାଦ କରିବାର ସୁବିଧା ଦିଏ। ଏହା ଆପଣଙ୍କ ପାଇଁ ଅନୁବାଦ ଫାଇଲ୍ ତିଆରି କରିବା ଏବଂ କମ୍ପାଇଲ୍ କରିବାର କାମ ମଧ୍ୟ ସମ୍ପୂର୍ଣ୍ଣ ଭାବରେ କରିଦେବ।
