# ମୁଁ ପ୍ରଥମ ଥର ପାଇଁ ମୋ ଆକାଉଣ୍ଟ ପେଜକୁ କିପରି ଲଗ୍ଇନ୍ କରିପାରିବି? {#how-can-i-access-my-account-page-for-the-first-time}

Ultimate Multisite-କୁ ସ୍ୱାଗତ!

ଆମର ଏହି pluginଟି କିଣିବା ପରେ ଆପଣଙ୍କୁ ଆପଣଙ୍କ e-mail ମଧ୍ୟରେ ଯେଉଁ linkଟି ପାଇବେ, ତାହା ମାଧ୍ୟମରେ ଆପଣ ଆପଣଙ୍କ ଆକାଉଣ୍ଟ ପେଜକୁ ପ୍ରବେଶ କରିପାରିବେ:

<!-- ସ୍କ୍ରିନ୍‌ସଟ୍ ଅପଲଏବଲ: ଏକ୍ସେସ୍ ଲିଙ୍କ ସହିତ ଆକାଉଣ୍ଟ ପେଜର e-mail -->

ଆପଣ କିଣିବା ସମୟରେ ଯେଉଁ e-mail ରେ ଆପଣ ଆପଣଙ୍କର email ଏବଂ password ଦେଇଥିଲେ, ତାହା ବ୍ୟବହାର କରନ୍ତୁ।

<!-- ସ୍କ୍ରିନ୍‌ସଟ୍ ଅପଲଏବଲ: email ଏବଂ password ଫିଲ୍ଡ୍ ସହିତ ଲଗ୍ଇନ୍ ପେଜ -->

କାରଣ ଏହା ଆପଣଙ୍କ ଆକାଉଣ୍ଟରେ ପ୍ରଥମ ଥର ଲଗ୍ଇନ୍ କରିବା, ତେଣୁ ଆପଣଙ୍କୁ ଆପଣଙ୍କ password ବଦଳାଇବାର କହିବ। ଏହାକୁ ବଦଳାଇ ଦିଅନ୍ତୁ ଏବଂ _Save_ ବଟନ ଉପରେ କ୍ଲିକ୍ କରନ୍ତୁ।

<!-- ସ୍କ୍ରିନ୍‌ସଟ୍ ଅପଲଏବଲ: Save ବଟନ ସହିତ password ବଦଳାଇବା form -->

ଆପଣଙ୍କ ଆକାଉଣ୍ଟ ପେଜରେ, ଆପଣ ନିମନ୍ତେ ବିଲିଂ (billings), renewed (renewal), cancelations ଏବଂ ଅନେକ ଅନ୍ୟ କିଛି ବି ଦେଖି ଓ ମ୍ୟାନେଜ କରିପାରିବେ।
