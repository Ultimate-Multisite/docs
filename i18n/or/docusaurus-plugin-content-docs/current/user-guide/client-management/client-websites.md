# ଆପଣଙ୍କ କ୍ଲାଇଣ୍ଟମାନଙ୍କ ୱେବସାଇଟ୍ (v2)

ଆପଣ ନିଜର ନେଟୱାର୍କରେ ଏକାଡମିକ ଭାବରେ ଯୋଗାଯୋଗ କରିବା ପରେ, ଆପଣଙ୍କ ଗ୍ରାହକମାନେ ସେମାନଙ୍କ ଡ୍ୟାଶବର୍ଡ୍‌କୁ ପ୍ରବେଶ କରିପାରିବେ।

**Account** ପୃଷ୍ଠା ବ୍ୟତୀତ, ଯେଉଁଠାରେ ସେମାନେ ନିଜର ପେମେଣ୍ଟ, ମେମ୍ବେରିସ୍କିପ୍, ଡୋमेन, ପ୍ଲାନ ସୀମା ଏବଂ ଅନ୍ୟାନ୍ୟ ଗୁରୁତ୍ୱପୂର୍ଣ୍ଣ ସୂଚନା ଦେଖିପାରନ୍ତି, ସେ ଆପଣଙ୍କ **My Sites** ପୃଷ୍ଠାକୁ ମଧ୍ୟ ପ୍ରବେଶ କରିପାରିବେ, ଯେଉଁଠାରେ ସେମାନେ ନିଜର ସମସ୍ତ ୱେବସାଇଟ୍‌କୁ ଦେଖି ଏବଂ ପରିଚାଳନା କରିପାରିବେ। ସେମାନଙ୍କର କାମ ହେଉଛି ସେମାନଙ୍କ ଡ୍ୟାଶବର୍ଡ୍‌ରେ My Sites ଉପରେ କ୍ଲିକ୍ କରିବା।

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

ସେମାନଙ୍କର ସମସ୍ତ ୱେବସାଇଟ୍ ଦେଖାଯିବ, ଏବଂ ପ୍ରତ୍ୟେକ ୱେବସାଇଟ୍‌ Ke niche **Manage** ବଟନ୍ ରହିବ। ଯଦି ସେ ଏହା ଉପରେ କ୍ଲିକ୍ କରନ୍ତି, ତେବେ ସେ ସେହି ନିର୍ଦ୍ଦିଷ୍ଟ ୱେବସାଇଟ୍‌ର **dashboard** ରେ ପରିମାଳିତ ହୋଇଯିବେ।

ସେ ଏହା ମଧ୍ୟ ଦେଖିପାରିବେ ଯେ କେଉଁଟି ସେମାନଙ୍କର **primary** ୱେବସାଇଟ୍, ଯାହା ଅନେକ ୱେବସାଇଟ୍ ରୁ ଅଧିକ ଥିବା ପ୍ଲାନଗୁଡ଼ିକ ପାଇଁ ସମ୍ଭବ।

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

ଏଠାରେ ୱେବସାଇଟ୍ ଏଡିଟ୍ ଆଇଣ୍ଟରଫେସ୍‌ର ଏକ ସମ୍ପୂର୍ଣ୍ଣ ଦୃଶ୍ୟ ଦିଆଗଲା:

![Site edit full interface](/img/admin/site-edit-full.png)

ଯଦି ସେ ନିଜର ପ୍ରାଥମିକ ୱେବସାଇଟ୍ ବଦଳାଇବାକୁ ଚାହାଁନ୍ତି, ତେବେ ସେ **Account** ମେନୁରେ କ୍ଲିକ୍ କରି Account ପୃଷ୍ଠାକୁ ଯିବା ଆବଶ୍ୟକ।

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

ସେ ନିଜର ଗ୍ରାହକ ବିବରଣୀ ମଧ୍ୟ ଦେଖି ଏଡିଟ୍ କରିପାରନ୍ତି:

![Customer edit interface](/img/admin/customer-edit.png)

ତା'ପରେ, ସେ **Change Default Site** ଉପରେ କ୍ଲିକ୍ କରିବା ଆବଶ୍ୟକ।

![Change Default Site button on account page](/img/account-page/change-default-site.png)

ଏକ ପ๊อปଅପ୍ ବିନ୍ଦୁ ଦେଖାଯିବ। డ్రాପଡାଉନ୍ ତାଲିକାରେ କ୍ଲିକ୍ କରି ଆପଣ କେଉଁ ୱେବସାଇଟ୍‌କୁ ପ୍ରାଥମିକ ୱେବସାଇଟ୍ କରିବାକୁ ଚାହାଁନ୍ତି, ସେଥିରେ ବାଛି ନିଅନ୍ତୁ।

![Change Default Site dropdown selection](/img/account-page/change-default-site-dropdown.png)

ତା'ପରେ, କେବଳ **Change Default Site** ଉପରେ କ୍ଲିକ୍ କରନ୍ତୁ ଏବଂ ଆପଣଙ୍କର କାମ ସମାପ୍ତ!

![ଡିଫାଉଟ୍ ସାଇଟ୍ ବଦଳାଇବାକୁ ନିଶ୍ଚିତ କରିବା](/img/account-page/change-default-site-confirm.png)
