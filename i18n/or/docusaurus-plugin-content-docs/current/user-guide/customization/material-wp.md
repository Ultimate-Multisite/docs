# ମାଟିରିଆଲ୍ ୱିପି (Material WP) {#material-wp}

ଏହି ଅବଶ୍ୟକତାଟି **Material WP** ଆଡଉନ୍ (addon) ଦ୍ୱାରା ପ୍ରଦାନ କରାଯାଇଛି।

ସମ୍ପୂର୍ଣ୍ଣ ବିବରଣୀ ପାଇଁ, [Material WP ଆଡଉନ୍ ଦସ୍ତାବିତାକୁ](https://addons.wordpress.org/category/material-wp/) ଦେଖନ୍ତୁ।
