# আপনার অ্যাডমিন প্যানেল কাস্টমাইজ করা PRO থিম দিয়ে

কাজের সময় প্রায়শই আমাদের ক্লায়েন্টদের ওয়ার্ডপ্রেস ড্যাশবোর্ডে একটি কাস্টম অভিজ্ঞতা দেওয়ার প্রয়োজন হয়। আপনি যদি আরও পেশাদার লুক এবং ফিল খুঁজতে থাকেন, তাহলে PRO Theme আপনাকে সাহায্য করতে পারে।

PRO Theme হলো এমন একটি অ্যাডমিন থিম যা আপনার ক্লায়েন্টদের ড্যাশবোর্ড কাস্টমাইজ করার সময় আপনাকে সাহায্য করে। আর এটি Ultimate Multisite-এর প্রিমিয়াম প্লাগইনগুলোর মূল অংশও বটে।

এটি ইনস্টল করার পর, আপনার _Plugins_ পেজে যান এবং আপনার নেটওয়ার্কে এটিকে অ্যাক্টিভেট করতে ক্লিক করুন।

<!-- স্ক্রিনশট পাওয়া যায়নি: WP Plugins পেজ যেখানে PRO Theme এর জন্য Network Activate বা নেটওয়ার্ক অ্যাক্টিভেট বোতাম দেখা যাচ্ছে -->

_আপনার প্লাগইন পেজে এটি নেটওয়ার্ক-ব্যাপী অ্যাক্টিভেট করুন_

_**দ্রষ্টব্য:**_ _আপনি যদি শুধু এটি ইনস্টল করেন, তবে আপনার ক্লায়েন্টদের কাছে এটি একটি অপশন হিসেবে দেখাবে যা শুধুমাত্র সেই ব্যবহারকারীর জন্য থিমটি যোগ করার জন্য চিহ্নিত করা যেতে পারে._

এখন, আপনার অ্যাডমিন ইন্টারফেস সম্পূর্ণ নতুনভাবে ডিজাইন করা হয়েছে! বাম দিকের কলামে 'PRO Theme' নামে একটি নতুন মেনুও পাবেন।
