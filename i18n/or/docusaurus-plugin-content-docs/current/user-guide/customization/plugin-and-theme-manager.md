# પ્લગઇન અને થીમ મેનેજર {#plugin-and-theme-manager}

ଏହି ଅବଶ୍ୟକତାଟି **ప్ଲଗିନ୍ ଏବଂ થીମ ମ୍ୟାନେଜର** ଆଡଉନ (addon) ଦ୍ୱାରା ପ୍ରଦାନ କରାଯାଇଛି।

ସମ୍ପୂର୍ଣ୍ଣ ବିବਰତ୍ତ ପାଇଁ, [પ્ଲଗିନ୍ ଏବଂ થીମ ମ୍ୟାନେଜର ଆଡଉନର ବିବਰତ୍ତ](https://addons.wordpress.org/en/addon/plugin-and-theme-manager/) ଦେଖନ୍ତୁ।
