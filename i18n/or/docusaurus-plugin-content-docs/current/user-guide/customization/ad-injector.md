# ನಿಮ್ಮ ನೆಟ್‌ವರ್ಕ್‌ನಲ್ಲಿ ಜಾಹೀರಾತುಗಳನ್ನು ಇಡುವುದಕ್ಕೆ Ultimate Multisite: Ad Injector {#placing-ads-on-your-network-with-ultimate-multisite-ad-injector}

Ultimate Multisite: Ad Injector ಒಂದು **ಉಚಿತ add-on** ಇದು ನಿಮಗೆ ನಿಮ್ಮ ಪ್ಲಾನ್‌ಗಳ ಮೇಲೆ ನೇರವಾಗಿ ಅವುಗಳ ಎಡಿಟ್ (edit) ಪುಟದಿಂದ ಜಾಹೀರಾತುಗಳನ್ನು ಹಾಕಲು ಬೇಕಾದ ಉಪಕರಣಗಳನ್ನು ನೀಡುತ್ತದೆ.

Ultimate Multisite: Ad Injector ಅನ್ನು ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡಿದ ನಂತರ, ಅದನ್ನು ಆಕ್ಟಿವೇಟ್ ಮಾಡಲು ನಿಮ್ಮ _Plugins_ ಪುಟಕ್ಕೆ ಹೋಗಿ.

<!-- ಸ್ಕ್ರೀನ್‌ಶಾಟ್ ಲಭ್ಯವಿಲ್ಲ: WP Plugins ಪುಟದಲ್ಲಿ Ad Injector add-on ಗಾಗಿ ನೆಟ್‌ವರ್ಕ್ ಆಕ್ಟಿವೇಟ್ ಬಟನ್ ತೋರಿಸುತ್ತದೆ -->

_Click the Network Activate button_ (ನೆಟ್‌ವರ್ಕ್ ಆಕ್ಟಿವೇಟ್ ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ)

add-on ಆಕ್ಟಿವೇಟ್ ಆದ ನಂತರ, ಪ್ಲಾನ್ ಎಡಿಟ್ ಸ್ಕ್ರೀನ್‌ನಲ್ಲಿರುವ _Advanced Options_ ಮೆಟಾ ಬಾಕ್ಸ್‌ನಲ್ಲಿ _Ad Settings_ ಎಂಬ ಹೊಸ ಆಯ್ಕೆ ಇರುತ್ತದೆ.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png) (ಪ್ಲಾನ್ ಎಡಿಟ್ ಸ್ಕ್ರೀನ್‌ನಲ್ಲಿ Ad Settings)

_ಈಗ ನೀವು ನಿಮ್ಮ ಪ್ರತಿಯೊಂದು ಪ್ಲಾನ್‌ಗಳಿಗಾಗಿ ಜಾಹೀರಾತು ಸೆಟ್ಟಿಂಗ್‌ಗಳನ್ನು ಸಂಪಾದಿಸಬಹುದು_

ಆ ಟ್ಯಾಬ್‌ನಲ್ಲಿ, ಫ್ರಂಟ್‌ಎಂಡ್‌ನಲ್ಲಿ, ವಿಷಯದ ಮೊದಲು ಮತ್ತು ನಂತರ ಪೋಸ್ಟ್‌ಗಳು ಮತ್ತು ಪೇಜ್‌ಗಳಿಗೆ ಜಾಹೀರಾತು ಕೋಡ್‌ಗಳನ್ನು ಸೇರಿಸಲು ನೀವು ಸಾಧ್ಯವಾಗುತ್ತದೆ.

<!-- ಸ್ಕ್ರೀನ್‌ಶಾಟ್ ಲಭ್ಯವಿಲ್ಲ: ವಿಷಯದ ಮೊದಲು ಮತ್ತು ನಂತರ ಪೋಸ್ಟ್‌ಗಳಿಗಾಗಿ Ad code ಫೀಲ್ಡ್‌ಗಳು -->

<!-- ಸ್ಕ್ರೀನ್‌ಶಾಟ್ ಲಭ್ಯವಿಲ್ಲ: ನಿರ್ದಿಷ್ಟ ಪ್ಲಾನ್‌ನ ಒಂದು ಪೋಸ್ಟ್ ಪುಟದಲ್ಲಿ ಪ್ರದರ್ಶಿಸಲಾದ ಜಾಹೀರಾತಿನ ಉದಾಹರಣೆ -->

_ನಿಮ್ಮ ಜಾಹೀರಾತು ಕೋಡ್ ಅನ್ನು ಸೇರಿಸಿದ ನಂತರ, ಅದು ಆ ನಿರ್ದಿಷ್ಟ ಪ್ಲಾನ್‌ನ ಪೋಸ್ಟ್‌ಗಳು ಮತ್ತು ಪೇಜ್‌ಗಳಲ್ಲಿ ಕಾಣಿಸುತ್ತದೆ._

ಆ ಪ್ಲಾನ್ ಬಳಕೆದಾರರಿಗೆ ಅಡ್ಮಿನ್ ಪೇಜ್‌ಗಳ ಹೆಡರ್‌ನಲ್ಲಿಯೂ ಜಾಹೀರಾತುಗಳನ್ನು ಸೇರಿಸಬಹುದು.

<!-- ಸ್ಕ್ರೀನ್‌ಶಾಟ್ ಲಭ್ಯವಿಲ್ಲ: ಪ್ಲಾನ್ ಸೆಟ್ಟಿಂಗ್‌ಗಳಲ್ಲಿ ಅಡ್ಮಿನ್ ಪ್ಯಾನೆಲ್ Ad Code ಆಯ್ಕೆ -->

<!-- ಸ್ಕ್ರೀನ್‌ಶಾಟ್ ಲಭ್ಯವಿಲ್ಲ: ಬಳಕೆದಾರರ ಅಡ್ಮಿನ್ ಪ್ಯಾನೆಲ್‌ನಲ್ಲಿ ಪ್ರದರ್ಶಿಸಲಾದ ಜಾಹೀರಾತಿನ ಉದಾಹರಣೆ -->

_ನಿಮ್ಮ ಬಳಕೆದಾರರ ಬ್ಯಾಕ್‌ಎಂಡ್‌ನಲ್ಲಿ ಜಾಹೀರಾತುಗಳನ್ನು ಪ್ರದರ್ಶಿಸಲು ನೀವು Admin Panel Ad Code ಆಯ್ಕೆಗೆ ಹೋಗಿ_

### ಜಾಹೀರಾತುಗಳಿಗಿಂತ ಮುಂದೆ {#beyond-advertising}

ನೀವು Ultimate Multisite: Ad Injector ಅನ್ನು ಹೈ-ಟಿಯರ್ ಪ್ಲಾನ್‌ಗಳಿಗೆ ಬೆಂಬಲ ನೀಡಲು ಬಳಸಬಹುದು.

_Admin Panel Ads_ ಆಯ್ಕೆಗೆ ಹೋಗಿ ಮತ್ತು ನಿಮ್ಮ ಸಪೋರ್ಟ್ ಟೂಲ್‌ಗಾಗಿ HTML ಕೋಡ್ ಅನ್ನು ಸೇರಿಸಿ.

_নির্দিষ্ট প্ল্যানগুলিতে কাস্টমার সাপোর্ট যোগ করুন_

এভাবে ব্যবহারকারীরা তাদের অ্যাডমিন প্যানেল থেকেই আপনার সাথে যোগাযোগ করতে পারবেন!
