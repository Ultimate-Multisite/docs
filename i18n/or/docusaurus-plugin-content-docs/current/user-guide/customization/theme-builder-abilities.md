# থিম বিল্ডার ক্ষমতা: স্ক্যাফোল্ড এবং অ্যাক্টিভেট ব্লক থিম {#theme-builder-abilities-scaffold-and-activate-block-themes}

Superdav AI Agent v1.12.0 দুটি শক্তিশালী ক্ষমতা নিয়ে এসেছে যা আপনাকে চ্যাট ইন্টারফেস থেকে কাস্টম ব্লক থিম তৈরি এবং স্থাপন করতে সাহায্য করে।

## ওভারভিউ {#overview}

**scaffold-block-theme** এবং **activate-theme** ক্ষমতাগুলি এজেন্টদের সক্ষম করে:
- আপনার নির্দিষ্ট বিবরণ অনুযায়ী সম্পূর্ণ, প্রোডাকশন-রেডি ব্লক থিম তৈরি করা
- ম্যানুয়াল হস্তক্ষেপ ছাড়াই আপনার সাইটে স্বয়ংক্রিয়ভাবে থিম অ্যাক্টিভেট করা
- নির্দেশিত ডিজাইনের সিদ্ধান্তের মাধ্যমে সুসংহত ভিজ্যুয়াল আইডেন্টিটি তৈরি করা

## স্ক্যাফোল্ড ব্লক থিম {#scaffold-block-theme}

**scaffold-block-theme** ক্ষমতাটি একটি নতুন ওয়ার্ডপ্রেস ব্লক থিম তৈরি করে যার মধ্যে সম্পূর্ণ থিম কাঠামো অন্তর্ভুক্ত থাকে, যার মধ্যে রয়েছে:

- ডিজাইন টোকেন সহ `theme.json` কনফিগারেশন
- সাধারণ লেআউটের জন্য ব্লক টেমপ্লেট ফাইল
- কাস্টম ব্লক স্টাইল এবং ভ্যারিয়েশন
- থিম মেটাডেটা এবং সাপোর্ট ঘোষণা

### কিভাবে ব্যবহার করবেন {#how-to-invoke}

Superdav AI Agent এর সাথে আপনার চ্যাটে, আপনি থিম তৈরির অনুরোধ করতে পারেন:

```
"একটি 'Modern Agency' নামে একটি ব্লক থিম তৈরি করো যার নীল ও সাদা রঙের স্কিম থাকবে,
sans-serif টাইপোগ্রাফি এবং একটি পেশাদার লেআউট থাকবে।"
```

এজেন্টটি যা করবে:
১. কথোপকথনের মাধ্যমে আপনার ডিজাইনের পছন্দগুলি সংগ্রহ করবে
২. সম্পূর্ণ থিম কাঠামো তৈরি করবে
৩. সমস্ত প্রয়োজনীয় থিম ফাইল তৈরি করবে
৪. অ্যাক্টিভেশনের জন্য থিম প্রস্তুত করবে

### প্রত্যাশিত আউটপুট {#expected-output}

ক্ষমতা সফলভাবে কার্যকর হলে, আপনি দেখতে পাবেন:

- থিমটি স্ক্যাফোল্ড করা হয়েছে তার নিশ্চিতকরণ
- থিমের নাম এবং অবস্থান
- প্রয়োগ করা ডিজাইন টোকেনগুলির সারাংশ (রঙ, টাইপোগ্রাফি, স্পেসিং)
- অ্যাক্টিভেট করার জন্য প্রস্তুত অবস্থা

উদাহরণ আউটপুট:
```
✓ থিম "Modern Agency" সফলভাবে স্ক্যাফোল্ড করা হয়েছে
  অবস্থান: /wp-content/themes/modern-agency/
  রঙ: প্রাথমিক #0066CC, মাধ্যমিক #FFFFFF
  টাইপোগ্রাফি: Inter (sans-serif)
  স্থিতি: অ্যাক্টিভেট করার জন্য প্রস্তুত
```

## থিম অ্যাক্টিভেট করুন {#activate-theme}

**activate-theme** କ୍ଷମତା ଆପଣଙ୍କ साईटକୁ ନୂଆ ସ୍କାଫଲଡ୍ କରାଯାଇଥିବା ବା ଅସ୍ତିତ୍ୱରେ ଥିବା ବ୍ଲକ୍ థీମ୍‌କୁ ପରିବର୍ତ୍ତନ କରିଦିଏ।

### କିପରି ବ୍ୟବହାର କରିବେ {#how-to-invoke-1}

థీମ୍ ସ୍କାଫଲଡ୍ (scaffolding) କରିବା ପରେ, ଆପଣ ଏହାକୁ ତୁରନ୍ତ ଅକ୍ଟିଭେଟ କରିପାରିବେ:

```
"Modern Agency theme" କୁ ଅକ୍ଟିଭେଟ କରନ୍ତୁ
```

କିମ୍ବା କୌଣସି ବିଲୁଡ୍ଥିବା థీମ୍ ଅକ୍ଟିଭେଟ କରିବା ପାଇଁ:

```
"Twentytwentyfour theme"-ରେ ସ୍ୱଇଚ୍ଛା କରନ୍ତୁ
```

### ଆଶାତ ମିଳିବାର ଫଳ (Expected Output) {#expected-output-1}

ଅକ୍ଟିଭେଟ ହେଲେ:

- ଅକ୍ଟିଭ୍ థీମ୍ ପ୍ରମାଣିକତା
- ପୂର୍ବରୁ ଥିବା థీମ୍ ନାମ (ପ୍ରତି ସୂଚନା ପାଇଁ)
- साईट ୟୁଆଇ (URL) ଯେଉଁଠି థీମ୍ ବର୍ତ୍ତମାନ ଲାଇଭ ଅଛି
- କୌଣସି థీମ୍-ବିଶେଷ ସେଟଅପ୍ ନୋଟ୍

ଉଦାହରଣୀୟ ଫଳ:
```
✓ థీମ୍ ସଫଳତାର ସହିତ ଅକ୍ଟିଭେଟ ହୋଇଛି
  ଅକ୍ଟିଭ୍ థీମ୍: Modern Agency
  ପୂର୍ବରୁଥିବା థీମ୍: Twentytwentyfour
  ଲାଇଭ ଅଛି: https://yoursite.com
  ନୋଟ୍: লেଆଉଟ୍ ଯାଞ୍ଚ କରିବା ପାଇଁ ଆପଣଙ୍କ ହୋମପେଜ୍ ଦେଖନ୍ତୁ
```

## କାର୍ଯ୍ୟ ପ୍ରଣାଳୀ: ସ୍କାଫଲଡ୍ ଏବଂ ଅକ୍ଟିଭେଟ {#workflow-scaffold-and-activate}

ଏକ ସାଧାରଣ କାର୍ଯ୍ୟ ପ୍ରଣାଳୀରେ, ଆପଣ ଦୁଇଟି କ୍ଷମତାକୁ ମିଶାଇ ବ୍ୟବହାର କରନ୍ତି:

1. **థీମ୍ ତିଆରି ଅନୁରୋଧ କରିବା**: "ମୋର SaaS ଲ୍ୟାଣ୍ଡିଂ ପେଜ୍ ପାଇଁ ଏକ ବ୍ଲକ୍ థీମ୍ ତିଆରି କରନ୍ତୁ"
2. **এଜେଣ୍ଟ ଦ୍ୱାରା థీମ୍ ସ୍କାଫଲଡ୍ ହେବା**: ଫାଇଲ୍ ଏବଂ ဒီဇိုင်း ଟୋକନ୍‌ଗୁଡ଼ିକ ତିଆରି କରିବା
3. **ପରୀକ୍ଷଣ ଏବଂ ସୁଧାର**: ଆବଶ୍ୟକ ହେଲେ ဒီଜାଇନ ପରିବର୍ତ୍ତନ ବିଷୟରେ ଆଲୋଚନା କରିବା
4. **ଅକ୍ଟିଭେଟ କରିବା**: "ଏବେ థీମ୍ ଅକ୍ଟିଭେଟ କରନ୍ତୁ"
5. **ପ୍ରମାଣିକତା**: ନୂଆ ဒီଜାଇନ୍ ଲାଇଭ ଅଛି ବୋଲି ସ୍ଥିର କରିବା ପାଇଁ ଆପଣଙ୍କ साईटକୁ ଯାଆନ୍ତୁ

## ဒီଜାଇନ ଟୋକନ୍ ଏବଂ କଷ୍ଟମାଇଜେସନ୍ (Design Tokens and Customization) {#design-tokens-and-customization}

ସ୍କାଫଲଡ୍ ହୋଇଥିବା థీମ୍ଗୁଡ଼ିକ WordPress ဒီଜାଇନ ଟୋକନ୍‌ଗୁଡ଼ିକୁ (`theme.json` ମାଧ୍ୟମରେ) ବ୍ୟବହାର କରନ୍ତି:

- **रଙ୍ଗ (Colors)**: ପ୍ରାଥମିକ, ଦ୍ୱିତୀୟ, ଆକସେଣ୍ଟ, ନିଉट्रଲ ପ୍ୟାଲେଟ୍
- **ଫଣ୍ଟ (Typography)**: ଫଣ୍ଟ ଫ୍ୟାମିଲି, ଆକାର, ବେଇସ, లైନ୍ ହାଇଟ୍
- **ଦୂରତା (Spacing)**: ପ୍ୟାଡିଂ, ମାର୍ଜିନ୍, ଗ୍ୟାପ ସ୍କେଲ୍‌ଗୁଡ଼ିକ
- **ବର୍ଡର (Borders)**: ବ୍ଲୁମି ଓ ୱିଡ୍ଥ ଟୋକନ୍
- **ଶାଡୋଉନ୍ (Shadows)**: ଉଚ୍ଚତା ମାତ୍ରାର ସ୍ତର

ଏହି ଟୋକନ୍ଗୁଡ଼ିକ `theme.json` ରେ କେନ୍ଦ୍ରୀଭୂତ, ଯାହା ଏକ ଫାଇଲ୍ ମଧ୍ୟରୁ ଆପଣଙ୍କ ପୂର୍ଣ୍ଣ ဒီଜାଇନ ସିଷ୍ଟମ୍‌କୁ ଅତି ସହଜରେ ଠିକ୍ କରିବାକୁ ସାମର୍ଥ୍ୟ ଦିଏ।

## ସୀମା ଏବଂ ନୋଟ୍ସ (Limitations and Notes) {#limitations-and-notes}

## ସମସ୍ୟାଗୁଡ଼ିକର ଉପରେ କାର୍ଯ୍ୟ କରିବା (Troubleshooting) {#troubleshooting}

**Theme scaffolding କରିବା ପରେ Theme ଦେଖାଯାଉନାହିଁ**
- Theme ଡିରେଟୋରୀଟି ଅଛି ଏବଂ ତା'ଠାରୁ ସଠିକ୍ ଅନୁମତି (permissions) ଅଛି କି ନା ତାହା ଯାଞ୍ଚ କରନ୍ତୁ।
- `theme.json` ଟି ବୈଧ JSON (valid JSON) ଅଛି କି ନା ତାହା ଯାଞ୍ଚ କରନ୍ତୁ।
- Theme ର ନାମ ପୂର୍ବରେ ଥିବା ଅନ୍ୟ Theme ସହିତ କୌଣସି ବି ମିଳିପାରେ କି ନା ତାହା ଦେଖନ୍ତୁ।

**Activation ବିଫଳ ହେଉଛି**
- ଆପଣଙ୍କର ॲଡମିନିଷ୍ଟ୍ରେଟର (administrator) ଅନୁମତି ଅଛି କି ନା ସେଥିପାଇଁ ନିଶ୍ଚିତ କରନ୍ତୁ।
- Theme ଡିରେଟୋରୀଟି WordPress ଦ୍ୱାରା ପଢ଼ିବାକୁ (readable) ଅଛି କି ନା ତାହା ଯାଞ୍ଚ କରନ୍ତୁ।
- ବିବର୍ତ୍ତିକାର (details) ଜାଣିବା ପାଇଁ WordPress error logs ମଧ୍ୟରେ ଯାଞ୍ଚ କରନ୍ତୁ।

**Design tokens ପ୍ରୟୋଗ ହେଉନାହିଁ**
- `theme.json` ର sintax ସଠିକ୍ ଅଛି କି ନା ତାହା ଯାଞ୍ଚ କରନ୍ତୁ।
- କ୍ୟାସିଙ୍ଗ (caching) ପ୍ଲଗିନ୍‌ଗୁଡ଼ିକ ସଫା କରନ୍ତୁ।
- ଆପଣ ବ୍ୟବହାର କରୁଥିବା tokens ଗୁଡ଼ିକୁ ଆପଣଙ୍କ WordPress version ସମର୍ଥନ କରେ କି ନା ତାହା ଯାଞ୍ଚ କରନ୍ତୁ।

## ପରବର୍ତ୍ତୀ ପଦକ୍ଷେପ (Next Steps) {#next-steps}

ଆପଣ ଆପଣଙ୍କ Theme ॲକ୍ଟିଭେଟ କରିବା ପରେ ଏହା କରିପାରିବେ:
- Typography, ରଙ୍ଗ ଏବଂ ସ୍ପେସିଂକୁ ସୁଧାରିବା ପାଇଁ **Design System Aesthetics skill** ବ୍ୟବହାର କରନ୍ତୁ।
- WordPress block editor ମାଧ୍ୟମରେ ବିଭିନ୍ନ block ଷ୍ଟାଇଲ୍‌କୁ କଷ୍ଟମାଇଜ୍ (customize) କରନ୍ତୁ।
- Theme ର `style.css` ଫାଇల్‌ରେ কଷ୍ଟମ CSS ଯୋଗ କରନ୍ତୁ।
- ନିର୍ଦ୍ଦିଷ୍ଟ page type ପାଇଁ কଷ୍ଟମ block templates ତିଆରି କରନ୍ତୁ।
