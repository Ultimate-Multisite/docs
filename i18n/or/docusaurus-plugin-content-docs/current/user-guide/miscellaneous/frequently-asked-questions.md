# ଅନୁଭବ କରାଯାଉଥିବା ପ୍ରଶ୍ନ (Frequently Asked Questions) {#frequently-asked-questions}

ଏଠାରେ ସେହି **ଅନୁଭବ କରାଯାଉଥିବା ପ୍ରଶ୍ନଗୁଡ଼ିକ ଦିଆଗଲା** ଯାହା ସମର୍ଥନ ଅନୁରୋଧ ମଧ୍ୟରେ ଅନେକ ଥରେ ଆସିଥାଏ।

  * **Ultimate Multisite କ'ଣ?**

Ultimate Multisite ହେଉଛି ଏକ WordPress plugin, ଯାହା WordPress multisite install ପାଇଁ ତିଆରି କରାଯାଇଛି। ଏହା ଆପଣଙ୍କର WordPress installକୁ [WordPress.com](https://WordPress.com) ଭଳି ପ୍ରମୁଖ ସାଇଟ୍ ନେଟୱାର୍କରେ ପରି ବଦଳାଇ ଦିଏ, ଯାହାद्वारे ଗ୍ରାହକମାନେ ମାସିକ, ତିବ୍ରିମାସ୍ରିକ କିମ୍ବା ବର୍ଷର ଦେୟ ଦେଇ ସାଇଟ୍ ତିଆରି କରିପାରନ୍ତି (ଆପଣ ମାସିକ ପ୍ଲାନ୍ ମଧ୍ୟ ତିଆରି କରିପାରିବେ)।

  * **Multisite ବିନା Ultimate Multisite କାମ କରିବ କି?**

ନା। Ultimate Multisite ପାଇଁ WordPress multisite, ସବୁଡିଡିରେଟ୍ରି (subdirectory), ବା ସବୁଡିନମେଡିନ୍ (subdomain) ଆବଶ୍ୟକ।

  * **Ultimate Multisite ର ଆବଶ୍ୟକତା କ'ଣ?**

Ultimate Multisite ପାଇଁ ନିମ୍ନଲିଖିତ ଜିନିଷଗୁଡ଼ିକ ଆବଶ୍ୟକ:

**WordPress Version** : v5.3+ (ପ୍ରସିଦ୍ଧ ସବୁଠାରୁ ଅଛି - Last Stable Version)
**PHP Version** : 7.4.x (8.0 ରୁ ଅଧିକ ପାଇଁ ଜରୁରୀ ଭାବରେ ଆଗକୁ ଆସିବ)
**MySQL Version** : v5+ (ପ୍ରସିଦ୍ଧ ସବୁଠାରୁ ଅଛି - Hosting provider ଉପରେ 8.0 ନଥିଲେ, 5.6 ବ୍ୟବହାର କରାଯାଇପାରେ)

_Multisite (subdomain ବା subdirectory) ମଧ୍ୟ ଏହାକୁ ଆକ୍ଟିଭେଟ କରିବାକୁ ପଡ଼ିବ._ [WordPress Multisite କିପରି ଇନଷ୍ଟଲ୍ କରିବ](/user-guide/getting-started/how-to-install-wordpress-multisite)।

  * **ସବୁଡିଡିରେଟ୍ରି ସହିତ Multisite ସହ Ultimate Multisite କାମ କରିବ କି?**

ହଁ। Ultimate Multisite ସବୁଡିଡିରେଟ୍ରି ଇନଷ୍ଟଲ୍ ଏବଂ ସବୁଡିଡିରେଟ୍ରି ଇନଷ୍ଟଲ୍—ଉଭୟକୁ ସମର୍ଥନ କରେ।

  * **ସବୁଡିନମେଡିନ୍ ସହିତ Multisite ସହ Ultimate Multisite କାମ କରିବ କି?**

ହଁ। Ultimate Multisite ସବୁଡିଡିରେଟ୍ରି ଇନଷ୍ଟଲ୍ ଏବଂ ସବୁଡିନମେଡିନ୍ ଇନଷ୍ଟଲ୍—ଉଭୟକୁ ସମର୍ଥନ କରେ।

  * **ବ୍ୟବହାରକାରୀମାନେ କ'ଣ Plugins ଏବଂ Themes ବାଛିବାକୁ ସୀମା ପକାଇବା ପାଇଁ Ultimate Multisite ବ୍ୟବହାର କରିପାରିବେ?**

ହଁ। Ultimate Multisite ସହିତ ଆପଣ ନିଜ ନେଟୱାର୍କର ପ୍ରତ୍ୟେକ ପ୍ଲାନ୍‌ରେ କେଉଁ Plugins ଏବଂ Themes ଉପଲବ୍ଧ ଅଛି, ତାହା ବାଛିପାରିବେ।

  * **ବ୍ୟବହାରକାରୀମାନେ ନିଜ ସାଇଟ୍‌ରେ କେତେ post ତିଆରି କରିପାରିବେ, ତାହା ସୀମା ପକାଇବା ପାଇଁ Ultimate Multisite ବ୍ୟବହାର କରିପାରିବେ କି?**

ହଁ। Ultimate Multisite ସହିତ ଆପଣ ଆପଣଙ୍କ ନେଟୱରରେ ଥିବା **ସବୁ ବିଲ୍ ପ୍ରକାର (post type)** ପାଇଁ ପୋଷ୍ଟ କୋଟା (post quotas) ସ୍ଥାପନ କରିପାରିବେ। ଆପଣ ଅ্যাডମିନ୍ କେଉଁ ବ୍ୟକ୍ତିଙ୍କୁ ଅମତିତ କରିବା, **ଡିସ୍କ ସ୍ପେସ୍**, ଏବଂ ଅନେକ ଅନେକ ଅନ୍ୟ ଜିନିଷ ମଧ୍ୟ ସୀମାଭୁକ୍ତ କରିପାରିବେ।

  * **Ultimate Multisite Site Templates တွေကို ସମର୍ଥନ କରେ କି?**

ହଁ। Ultimate Multisite ସହିତ ଆପଣ ଯେତେକ ବିଲ୍ ପ୍ରକାର (Site Templates) ତିଆରି କରିପାରିବେ। Site Templates ହେଉଛି থିମ୍ ନୁହେଁ, ଏହା ଆପଣଙ୍କ ନେଟୱରରେ ଅନ୍ୟ ସାଧାରଣ ସାଇଟ୍‌। ଆପଣ ଯେକୌଣସି ଭାବରେ ፕလଗିନ୍, থିମ୍ ଏବଂ ସେହି Site Templateକୁ କଷ୍ଟମାଇଜ (customize) କରିପାରିବେ। ଯଦି କୌଣସି ବ୍ୟକ୍ତି ସେହି templateଟି ବାଛିବେ, ତାଙ୍କର ସାଇଟ୍ଟି ସେହି templateର ଏକ ନକଲ ଭାବରେ ସୃଷ୍ଟି ହେବ, ସମସ୍ତ କଷ୍ଟମାଇଜେସ ସେଥିରେ ରହିଯିବ।

ଆପଣ ଏକ ନିର୍ଦ୍ଦିଷ୍ଟ plan ପାଇଁ ଏକ template set କରିପାରିବେ। ଏହା କରିବା ଦ୍ୱାରା, ଯେଉଁ ପ୍ରତି plan ଉପରେ କୌଣସି ବ୍ୟକ୍ତି ସାଇନ୍-ଅପ୍ (sign up) କରିବେ, ତାଙ୍କର ସାଇଟ୍ଟି ସେହି Site Templateର ଏକ ଦୁଇଆ (duplicate) ଭାବରେ ସୃଷ୍ଟି ହେବ।

  * **Ultimate Multisite WooCommerce ସହିତ ଅନ୍ତରାଳ କରେ କି?**

ହଁ!

ଯଦି ଆପଣ ଚାହାନ୍ତି ଯେ ଆପଣଙ୍କ ନେଟୱରର ବ୍ୟବହାରକାରୀମାନେ ତାଙ୍କ ପାଇଁ WooCommerce ବ୍ୟବହାର କରିପାରନ୍ତୁ, ସ୍ମୋର୍ (stores) ସ୍ଥାପନ କରିବାକୁ ଅନୁମତି ଦିଅନ୍ତି, ତେବେ Ultimate Multisite ବ୍ୟବହାର କରି ଏହା ସମ୍ଭବ। ଆପଣ ପ୍ରତ୍ୟେକ plan tierରେ ଆପଣଙ୍କ ଗ୍ରାହକମାନେ କେତେ product ସୃଷ୍ଟି କରିପାରିବେ, ତାହା ମଧ୍ୟ ସୀମାଭୁକ୍ତ କରିପାରିବେ।

Ultimate Multisite ପାଇଁ ସబ్‌ସ୍କ୍ରିପ୍ସନ୍ ପେମେଣ୍ଟ ପାଇଁ, ଆପଣ [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) ବ୍ୟବହାର କରିପାରିବେ ଯାହାଦ୍ୱାରା ଆପଣଙ୍କ Ultimate Multisite ଗ୍ରାହକମାନେ ଆପଣଙ୍କ ମୁଖ୍ୟ ସାଇଟ୍ ଉପରେ WooCommerce gateways ବ୍ୟବହାର କରି ପୈଠ କରିପାରିବେ। ଏହିପରି, ଆପଣ WooCommerce ପାଇଁ ବିକଶିତ ହୋଇଥିବା ବିରାଟ ପେମେଣ୍ଟ ଗେଟୱେୟରୀ (payment gateways) ଲାଇବ୍ରେରୀକୁ ବ୍ୟବହାର କରି ଆପଣଙ୍କ Ultimate Multisite ଗ୍ରାହକମାନଙ୍କୁ বিল କରିପାରିବେ।

  * **WPMUDEV Pro Sites ର ସబ్‌ସ୍କ୍ରିପ୍ସନ୍ ମIGRATION କରିବାର କୌଣସି ଉପାୟ ଅଛି କି?**

ହଁ, Ultimate Multisite ଆପଣଙ୍କୁ ଆମ Pro Sites ନେଟୱାର୍କକୁ ମିଗ୍ରେଟ କରିବାରେ [ଏକ ସାହାଯ୍ୟକାରୀ ଟୁଲ୍] ପ୍ରଦାନ କରେ।

  * **ମୋର Plan Edit Page ରେ “Allowed Plugins” ଟ୍ୟାବ୍ କାହିଁକି ଖାଲି ଅଛି?**

Ultimate Multisite ଆପଣଙ୍କ ନେଟୱାର୍କରେ ନେଟୱାର୍କ-ଆକ୍ଟିଭେଟ୍ (network-activated) ହୋଇନଥିବା ପ୍ଲଗିନ୍‌କୁ ଅନୁମତି ଦେବା କିମ୍ବା ପ୍ରତ୍ୟାହାର କରିବାରେ ହିଁ ଆପଣଙ୍କୁ ସମର୍ଥନ କରେ। ଯେତେବେଳେ ଏକ ପ୍ଲଗିନ୍ ନେଟୱାର୍କ-ଆକ୍ଟିଭେଟ୍ ହୋଇଥାଏ, ଏହା ପ୍ରକୃତରେ ଆପଣଙ୍କ ନେଟୱାର୍କର ପ୍ରତ୍ୟେକ ସାଇଟରେ ଡିଫାଉଟ୍ ଭାବରେ ଲୋଡ୍ ହୋଇଯିବ।

ଯଦି ଆପଣ “Allowed Plugins” ଟ୍ୟାବ୍ ରେ କୌଣସି ପ୍ଲଗିନ୍ ଦେଖୁନାହାନ୍ତି, ତେବେ ଯାଞ୍ଚ କରନ୍ତୁ ଯେ ଆପଣ ଚାହୁଁଥିବା ପ୍ଲଗିନ୍ ନେଟୱାର୍କ-ଆକ୍ଟିଭ୍ ଅଛି କି ନାହିଁ।

  * **ମୋର ନିଜର Pricing Table ଡିଜାଇନ୍ ବ୍ୟବହାର କରିପାରିବ କି?**

ହଁ। version 1.4.0 ରୁ, Ultimate Multisite template overrides ସମର୍ଥନ କରେ।

  * **Ultimate Multisite WP Engine ସହିତ କାମ କରେ କି?**

ହଁ। Ultimate Multisite ସବୁବେଳେ WP Engine ଦ୍ୱାରା ସମର୍ଥନ ପାଇଥିଲା, କିନ୍ତୁ ଡୋमेन ମ୍ୟପିଂ (domain mapping) କାମ କରିବା ପାଇଁ ଆଡମିନ୍‌ଙ୍କଠାରୁ ବାହାରର ଅତିକ୍ରମୀ ପଦକ୍ଷେପ ନେବାକୁ ପଡ଼ିଥିଲା (WP Engine ପାନେଲରେ ଡୋमेन ଯୋଗ କରିବା)। **version 1.6.0 ରୁ ଏହା ଆଉ ସେହିପରି ନୁହେଁ।** Ultimate Multisite ସିଧାସଳଖ WP Engine API ସହିତ ବ୍ୟାပ် ମିଳିଥାଏ ଏବଂ ମ୍ୟପିଂ ପରେ ଅଟୋମେଟିକ୍ ଭାବରେ ଡୋमेन ପାନେଲରେ ଯୋଗ ହୋଇଯାଏ।

  * **ଆପଣ କେଉଁ ହୋଷ୍ଟିଂ ପ୍ରଦୋତ୍ସରକୁ ସୁପାରିଶ କରନ୍ତି?**

Ultimate Multisite ଯେକୌଣସି ହୋଷ୍ଟିଂ ପ୍ରଦୋତ୍ସର ସହିତ କାମ କରିବା ଉଚିତ୍। ଏକମାତ୍ର ପାର୍ଥକ୍ୟ ହେଉଛି, Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane ଏବଂ [Runcloud.io](http://Runcloud.io) ଭଳି କିଛି ହୋଷ୍ଟିଂ କମ୍ପାନୀଙ୍କର ଡୋमेन ମ୍ୟପିଂ କ୍ଷମତା ପାଇଁ ଅଧିକ ସୁଦୂର ବାନ୍ଧି ରଖାଯାଇଛି, ଯାହାଫଳରେ ସେମାନେ ଏହାକୁ 'out of the box' କାମ କରିପାରନ୍ତି। ଅନ୍ୟ ହୋଷ୍ଟିଂ ପାଇଁ ଆପଣଙ୍କୁ ଡୋमेन ମ୍ୟପିଂ ମାନୁଷୀୟ ଭାବରେ ସେଟ୍ କରିବାକୁ ପଡ଼ିପାରେ।
