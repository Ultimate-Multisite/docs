# ଇମେଲ୍ ଏବଂ ବ୍ରାଡକାଷ୍ଟ୍ ପଠାଇବା (v2)

_**ଗୁରୁତ୍ୱପୂର୍ଣ୍ଣ ନୋଟ: ଏହି ଆଇକାଣିକ ବିଷୟଟି Ultimate Multisite version 2.x-କୁ ସମ୍ବନ୍ଧିତ।**_

Ultimate Multisiteରେ ଏକ ଅସବେକାର କାର୍ଯ୍ୟକଳାପ ରହିଛି ଯାହା ଆପଣଙ୍କୁ ନିର୍ଦ୍ଦିଷ୍ଟ ବ୍ୟକ୍ତି ବା ବ୍ୟବହାରକାରୀମାନଙ୍କ ଗୋଷ୍ଠୀକୁ ଇମେଲ୍ ପଠାଇବା ଏବଂ ସେମାନଙ୍କ ଅଡମିନ୍ ଡାଶବୋର୍ଡରେ ଘୋଷଣା ପ୍ରସାର କରିବା ପାଇଁ ସୁଯୋଗ ଦିଏ।

## ବ୍ରାଡକାଷ୍ଟ୍ ମାଧ୍ୟମରେ ଆପଣଙ୍କ ଗ୍ରାହକମାନଙ୍କ ଅଡମିନ୍ ଡାଶବୋର୍ଡରେ ଅଡମିନ୍ ନୋଟିସ (Admin Notices) ଯୋଗ କରନ୍ତୁ

Ultimate Multisite ବ୍ରାଡକାଷ୍ଟ୍ ବିଶେଷ କାର୍ଯ୍ୟକଳାପ ବ୍ୟବହାର କରି, ଆପଣ ଆପଣଙ୍କ ୟୁଜର୍ ସବସାଇଟ ଅଡମିନ୍ ଡାଶବୋର୍ଡରେ **ଅଡମିନ୍ ନୋଟିସ** ଯୋଗ କରିପାରିବେ।

ଯଦି ଆପଣ କୌଣସି ବିଷୟରେ ଘୋଷଣା କରିବାକୁ ଚାହୁଁଛନ୍ତି, ଯେପରିକି ସିଷ୍ଟମ୍ ମେନ୍ତ୍ରେନେନ୍ସ (system maintenance) ବା ନୂଆ ଉତ୍ପାଦ ବା ସେବା ଆପଣଙ୍କ ଅବସ୍ଥିତ ବ୍ୟବହାରକାରୀମାନଙ୍କୁ ଜଣାଇବା, ତେବେ ଏହା ଅତ୍ୟନ୍ତ ଉପଯୋଗୀ ହେବ। ଆପଣଙ୍କ ୟୁଜର୍‌ର ଡାଶବୋର୍ଡରେ ଅଡମିନ୍ ନୋଟିସ କିପରି ଦେଖାଯିବ, ତାହା ଏହିପରି ହେବ:

<!-- Screenshot unavailable: Admin notice broadcast shown on a customer's subsite dashboard -->

ଅଡମିନ୍ ନୋଟିସ ଆରମ୍ଭ କରିବା ପାଇଁ, ଆପଣଙ୍କ ନେଟୱାର୍କ ଅଡମିନ୍ ଡାଶବୋର୍ଡରେ ଯାଆନ୍ତୁ ଏବଂ **Ultimate Multisite** ମେନୁအောက်ରେ, ଆପଣ **Broadcasts** ବିକଳ୍ପଟି ଖୋଜିପାରିବେ।

![Broadcasts list page in Ultimate Multisite admin](/img/admin/broadcasts-list.png)

ଆପଣ ଅସ୍ଥିତ ବ୍ରାଡକାଷ୍ଟ୍‌ଗୁଡ଼ିକୁ ସଂଶୋଧନ କରିପାରିବେ:

![Broadcast edit interface](/img/admin/broadcast-edit.png)

ଏହି ପୃଷ୍ଠାରୁ, ଉପରେ **Add Broadcast** ବଟନ୍‌ଟି ଉପରେ କ୍ଲିକ୍ କରନ୍ତୁ।

ଏହା ଆପଣଙ୍କୁ Add broadcast ମୋଡାଲ ବିନ୍ଦୋ ଏକାଠି କରିବ, ଯେଉଁଠି ଆପଣ କେଉଁ ପ୍ରକାରର ବ୍ରାଡକାଷ୍ଟ୍ ପଠାଇବାକୁ ଚାହୁଁଛନ୍ତି, ତାହା ବାଛିପାରିବ।

ଆଗକୁ ଯାଇ **Message** ବାଛିଲେ ଏବଂ ତା'ପରେ **Next Step** ବଟନ୍‌ଟି କ୍ଲିକ୍ କରନ୍ତୁ।

![Add broadcast modal with the Message option selected](/img/admin/broadcast-add-message.png)

ପରବର୍ତ୍ତୀ ବିନ୍ଦୋ ଆପଣଙ୍କୁ କିମ୍ବା **Target customer** (ଲକ୍ଷ୍ୟ ଗ୍ରାହକ) ବା **Target product** (ଲକ୍ଷ୍ୟ ଉତ୍ପାଦ) ମଧ୍ୟରୁ ଗୋଟିଏ ଚୟନ କରିବାକୁ କହିବ। ଧ୍ୟାନ ଦିଅନ୍ତୁ, ଆପଣ ଏକထက် ଅଧିକ ୟୁଜର୍ ବା ଏକထက် ଅଧିକ ଉତ୍ପାଦ ବାଛିପାରିବେ।

କୌଣସି ୟୁଜର୍ ଆକାଉଣ୍ଟ କିମ୍ବା ପ୍ରଡକ୍ଟକୁ ଖୋଜିବା ପାଇଁ, ଆପଣ କ୍ୱାର୍ଡ୍ ବାଚ୍ (keyword)ଟି ଫିଲ୍ଡ ଭିତରେ ଟାଇପ୍ କରିବା ଆରମ୍ଭ କରିବାକୁ ପଡ଼ିବ।

**Message type** ଫିଲ୍ଡରେ, ଆପଣ ନୋଟିସିଂର ରଙ୍ଗ ବାଛିପାରିବେ। ଏହା ଆପଣଙ୍କ ମେସେଜର ଜରୁରତାକୁ ଦୃଷ୍ଟି ଆକର୍ଷଣ କରିବ।

ତା'ପରେ ଆପଣ **Next Step** ଉପରେ କ୍ଲିକ୍ କରିପାରିବେ।

![Message broadcast targets, target product and message type fields for a Message broadcast](/img/admin/broadcast-message-targets.png)

ଏହି ନିମ୍ନ ବିନ୍ଦୁଟି ସେଠାରେ ଆପଣ ଆପଣଙ୍କ ମେସେଜ ଲେଖିବା ଆରମ୍ଭ କରିପାରିବେ, ଯେଉଁଥିରେ ଆପଣ ସେୟାର୍ (subject) ଏବଂ ବ୍ୟବହାରକାରୀମାନଙ୍କୁ ପଠାଇବାକୁ ଚାହାଁନ୍ତି ତା' ବିଷୟବସ୍ତୁ/ମେସେଜଟି ପ୍ରବେଶ କରିପାରିବେ।

![Broadcast message subject and content editor on the compose step](/img/admin/broadcast-edit.png)

ଆପଣ ଆପଣଙ୍କ ମେସେଜ୍ ସୃଷ୍ଟି କରିବା ପରେ, ଆପଣ **Send** ବଟନ୍ ଉପରେ ଠିକ୍ କରନ୍ତୁ।

ଏତିକି। ପ୍ରଶାସନ (admin) ନୋଟିସିଂ ଆପଣଙ୍କ ୟୁଜର୍‌ଙ୍କ ଡ୍ୟାଶବର୍ଡ୍ ରେ ତୁରନ୍ତ ଦେଖାଯିବ।

## ଆପଣଙ୍କ ଗ୍ରାହକମାନଙ୍କୁ ଇମେଲ୍ ପଠାନ୍ତୁ

Ultimate Multisite ବ୍ରାଡକାଷ୍ଟ (broadcast) ବିଶେଷତା ବ୍ୟବହାର କରି, ଆପଣ ଆପଣଙ୍କ ୟୁଜର୍‌ମାନଙ୍କୁ ଏକ ଇମେଲ୍ ପଠାଇପାରିବେ। ଆପଣ କେବଳ ନିର୍ଦ୍ଦିଷ୍ଟ ୟୁଜର୍‌କୁ ଇମେଲ୍ ପଠାଇବା ବା ପ୍ରଡକ୍ଟ କିମ୍ବା ପ୍ଲାନ୍ ଅନୁଯାୟୀ ସେଉଁ ଗ୍ରୁପରେ ଅନ୍ତର୍ଭୁକ୍ତ ଥିବା ନିର୍ଦ୍ଦିଷ୍ଟ ୟୁଜର୍ ଗୋଷ୍ଠୀକୁ ଲକ୍ଷ୍ୟ କରିପାରିବେ।

ଇମେଲ୍ ବ୍ରାଡକାଷ୍ଟ ଆରମ୍ଭ କରିବା ପାଇଁ, ଆପଣ ଆପଣଙ୍କ ନେଟୱାର୍କ ଡାଶବର୍ଡ୍‌କୁ ଯାଆନ୍ତୁ ଏବଂ Ultimate Multisite ମେନୁରେ, ଆପଣ Broadcast ବିଷୟରେ ଅକ୍ସନ୍ ବାହାର କରିପାରିବେ।

![Broadcasts list page used as the starting point for an email broadcast](/img/admin/broadcasts-list.png)

ଏହି ପୃଷ୍ଠାରୁ, ଉପରେ **Add broadcast** ବଟନ୍ ଉପରେ କ୍ଲିକ୍ କରନ୍ତୁ।

ଏହା Add broadcast ମୋଡାଲ ବିନ୍ଦୁଟି ଆଣିବ ଯେଉଁଠାରେ ଆପଣ କେଉଁ ପ୍ରକାରର ବ୍ରାଡକାଷ୍ଟ ପଠାଇବାକୁ ଚାହୁଁଛନ୍ତି, ତାହା ବାଛିପାରିବେ। ଆଗକୁ ଯାଇ **Email** ବାଛିନ୍ତୁ ଏବଂ ତା'ପରେ **Next Step** ବଟନ୍ ଉପରେ କ୍ଲିକ୍ କରନ୍ତୁ।

![Add broadcast modal with the Email option selected](/img/admin/broadcast-add-email.png)

ପରବର୍ତ୍ତୀ ବିନ୍ଦୁଟି ଆପଣଙ୍କୁ କିମ୍ବା **Target customer** ବା **Target product** ମଧ୍ୟରୁ ଗୋଟିଏ ଦେଖାଇବ। ଲକ୍ଷ୍ୟ କରନ୍ତୁ ଯେ ଆପଣ ଏକထက် ଅଧିକ ୟୁଜର୍ ବା ଅଧିକ ପ୍ରଡକ୍ଟ ବାଛିପାରିବେ।

କୌଣସି ୟୁଜର୍ ଆକାଉଣ୍ଟ ବା ପ୍ରଡକ୍ଟକୁ ଖୋଜିବା ପାଇଁ ଆପଣ କ୍ୱାର୍ଡ୍ଟି ଫିଲ୍ଡ ଭିତରେ ଟାଇପ୍ କରିବା ଆରମ୍ଭ କରିବାକୁ ପଡ଼ିବ।

ଏହାପରେ, ଯେଉଁ ଲକ୍ଷ୍ୟ ଅନୁସନ୍ଧାନକାରୀ (target audience) ବାଛିଲେ, ଆପଣ **Next Step** କ୍ଲିକ୍ କରିପାରିବେ।

![Email broadcast target customers and product selection for an Email broadcast](/img/admin/broadcast-email-targets.png)

ଆଗକୁ ଆସୁଥିବା ବିନ୍ଦୁଟି ହେଉଛି ସେଠାରୁ ଆପଣ ଇମେଲ୍ ଲେଖିବା ଆରମ୍ଭ କରିପାରିବେ, ଯେଉଁଠି ଆପଣ ବିଷୟ (subject) ଏବଂ ୟୁଜର୍ମାନଙ୍କ ପାଇଁ ଯାହା ବାରଣ କରିବାକୁ ଚାହୁଁଛନ୍ତି ସେହି ଅନ୍ତହୀନ/संदेश (content/message) ପ୍ରବେଶ କରିପାରିବେ।

<!-- Screenshot unavailable: Email broadcast subject and content editor on the compose step -->

ଆପଣ ଆପଣଙ୍କର ବାର୍ତ୍ତା ସୃଷ୍ଟି କରିବା ପରେ, ଆପଣ **Send** ବଟନ୍ ଠିକ୍ କରିପାରିବେ।

ଏତିକି, ବ୍ରଡକାଷ୍ଟ (broadcast) ବିଶ୍ଳେଷଣ (feature) ବ୍ୟବହାର କରି ଆପଣଙ୍କ ଅନ୍ତର୍ଗତ ୟୁଜର୍ମାନଙ୍କ ପାଇଁ ଇମେଲ୍ ପଠାଇବା କେତେ ସହଜ।

## ସିଷ୍ଟମ୍ ଇମେଲ୍ (System emails)

Ultimate Multisite ମଧ୍ୟରେ ସିଷ୍ଟମ୍ ଇମେଲ୍ ହେଉଛି **ସ୍ୱୟଂଚାଳିତ ନୋଟିଫିକେସନ୍** ଯାହା କୌଣସି କାର୍ଯ୍ୟ ପରେ, ଯେପରିକି ରେଜିଷ୍ଟ୍ରେସନ୍, ପେମେଣ୍ଟ, ଡୋମେନ୍ ମ୍ୟପିଂ ଆଦି ସେହି କାର୍ଯ୍ୟ ପରେ ସିଷ୍ଟମ୍ ଦ୍ୱାରା ପଠାଯାଏ। ଏହି ଇମେଲ୍ଗୁଡ଼ିକ Ultimate Multisite ସେଟିଙ୍ଗରୁ ସଂଶୋଧନ କରାଯାଇପାରିବ ବା ପରିବର୍ତ୍ତନ କରାଯାଇପାରିବ। ଏଥିରେ ଏକ ବିଶେଷ ଅବସର ମଧ୍ୟ ରହିଛି ଯେ ଆପଣ ଅନ୍ୟ Ultimate Multisite ଇନଷ୍ଟଲରେ ଥିବା ଅସ୍ଥିତ ସେଟିଙ୍ଗକୁ ପୁଣିଥରେ ସେଟ୍ କରି (reset) ବା ଆମେରିକ୍ସ (import) କରିପାରିବେ।

### রিসেଟ୍ କରିବା ଏବଂ ଆମେରିକ୍ସ କରିବା (Resetting & Importing)

ନୂଆ Ultimate Multisite versionଗୁଡ଼ିକ, ଯେଉଁଥିରେ ଅଡ-ଅନ୍ସ (add-ons) ମଧ୍ୟ ଅନ୍ତର୍ଭୁକ୍ତ, କେବେକିେ କେବେ ନୂଆ ଇମେଲ୍ ସୃଷ୍ଟି କରିପାରନ୍ତି।

ରୋକିବା ପାଇଁ ଏବଂ ଅନ୍ୟାନ୍ୟ ସମସ୍ୟା ରୋକିବା ପାଇଁ, **ଯଦି ନୂଆ ଇମେଲ୍ ଟେମ୍ପଲେଟଗୁଡ଼ିକ କୌଣସି ବିଶେଷ କାର୍ଯ୍ୟର ସଠିକ୍ କାର୍ଯ୍ୟଚାଳନା ପାଇଁ ଅତ୍ୟନ୍ତ ଗୁରୁତ୍ୱପୂର୍ଣ୍ଣ ନୁହନ୍ତି, ତେବେ ଆମେ ଆପଣଙ୍କ ଇନଷ୍ଟଲରେ ସେଗୁଡ଼ିକ ନୂଆ ସିଷ୍ଟମ୍ ଇମେଲ୍ ଭାବରେ ଆଧାର କରିବାକୁ ସ୍ୱୟଂଚାଳିତ ଭାବରେ ଯୋଡ଼ିବୁ ନାହିଁ।**

ଅନ୍ୟପକ୍ଷରେ, ସୁପର ଅଡମିନ୍ (super admins) ଏବଂ ଏଜେଣ୍ଟମାନେ ଆମେରିକ୍ସ ଟୁଲ୍ ମାଧ୍ୟମରେ ଏହି ନୂଆରେଜିଷ୍ଟ୍ରେଡ୍ ଇମେଲଗୁଡ଼ିକକୁ ଆମେରିକ୍ସ କରିପାରନ୍ତି। ଏହି ପ୍ରକ୍ରିୟାଟି ନୂଆ ଇମେଲ୍ ଟେମ୍ପଲେଟର ବିଷୟବସ୍ତୁ ଏବଂ କନଫିଗୁରେସନ୍ ସହିତ ଏକ ନୂଆ ସିଷ୍ଟମ୍ ଇମେଲ୍ ସୃଷ୍ଟି କରିବ, ଯାହା ସୁପର ଅଡମିନ୍‌କୁ ଆପଣ ଚାହୁଁଥିବା କୌଣସି ବିଶେଷ ପରିବର୍ତ୍ତନ କରିବା ବା ଏମାନଙ୍କୁ ନିଜ ଭାବରେ ରଖିବା ସମ୍ଭାବନା ଦେବ।

#### ସିଷ୍ଟମ୍ ଇମେଲ୍‌କୁ ଆମେରିକ୍ସ କରିବାର ଉପାୟ (How to import system emails)

ଆପଣ ଆପଣଙ୍କ Ultimate Multisite Settings page ରେ ଯାଇ **Emails** ଟ୍ୟାବ୍‌କୁ ଯାଆନ୍ତୁ।

![Ultimate Multisite settings ରେ Emails tab ଦେଖାଉଥିବା ଚିତ୍ର](/img/config/settings-emails-tab.png)

ତା'ପରେ, sidebar ରେ, **Customize System Emails** ବଟନ୍ ଉପରେ କ୍ଲିକ୍ କରନ୍ତୁ।

#### System Emails रीसेट କରିବା (Reseting System Emails)

ଅନ୍ୟ କେବେ, ଆପଣ ଜାଣିପାରିବେ ଯେ ଏକ ନିର୍ଦ୍ଦିଷ୍ଟ email template ରେ ଆପଣ ଯେଉଁ ପରିବର୍ତ୍ତନ କରିଛନ୍ତି, ସେ ଆପଣଙ୍କ ପାଇଁ ବ୍ୟବହାର ହେଉନାହିଁ ଏବଂ ଆପଣ ଏହାକୁ ତା'ର **default state** ରେ ରଖିବାକୁ ଚାହୁଁଛନ୍ତି।

ଏହି କ୍ଷେତ୍ରରେ, ଆପଣ ଦୁଇଟି ବିକଳ୍ପ ପାଇବେ: ଆପଣ ସମ୍ପୂର୍ଣ୍ଣ ଭାବରେ system email ଟିକୁ delete କରି ଏହାକୁ ପୁଣି import କରିପାରିବେ (ଉପରୋକ୍ତ ନିର୍ଦ୍ଦେଶଗୁଡ଼ିକ ବ୍ୟବହାର କରି) - ଯାହା send metrics ଆଉ ଅନ୍ୟ କିଛି ଜିନିଷ ମଧ୍ୟ ମୂଳରେ ହଟାଇ ଦେଇଯିବ, ଯାହା ଏହି ପଦ୍ଧତିକୁ ସବୁଠାରୁ କମ୍ ପ୍ରାୟତଃ ପସନ୍ଦ କରାଯାଏ।

କିମ୍ବା ଆପଣ that email template ଟିକୁ रीसेट କରିବା ପାଇଁ **Reset & Import tool** ବ୍ୟବହାର କରିପାରିବେ।

email template ଟିକୁ रीसेट କରିବା ପାଇଁ, ଆପଣ ଉପରୋକ୍ତ ସ୍ଥିପରେ ଅନୁସରଣ କରି Reset & Import tool ପର୍ଯ୍ୟନ୍ତ ଯାଆନ୍ତୁ, ଏବଂ ତା'ପରେ **Reset** অপশনଟି ଟୋଗଲ କରନ୍ତୁ ଏବଂ ଆପଣ ଯେଉଁ emails ಗಳನ್ನು default content ରେ પાછા ରଖିବାକୁ ଚାହୁଁଛନ୍ତି, ସେଗୁଡ଼ିକ ବାଛନ୍ତୁ।
