# ଅନ୍ୟ ปลั๊กင်ଗୁଡ଼ିକ ସହିତ ଜଣାଶୁଣା ଅସଂଯୋଗ

Ultimate Multisite ଏକ ପରେ ଇନଷ୍ଟଲ୍ କରିବା ଏବଂ ଆକ୍ଟିଭେଟ୍ କରିବା ପରେ, ଅନ୍ୟ ପ୍ଲାଗିନ୍ଗଗୁଡ଼ିକ ସହିତ ଅସମାନତା ଯୋଗୁଁ ଆପଣଙ୍କୁ ସମସ୍ୟା ହୋଇପାରେ।

ଆମେ ପ୍ରତିଟି ନୂଆ ରିଲିଜ୍ ସହିତ ଏହି ସୂଚୀକୁ କମ୍ କରିବା ପାଇଁ ସକ୍ରିୟ ଭାବରେ କାମ କରୁଛୁ, କିନ୍ତୁ ଯଦି ଏଠାରେ କୌଣସି ପ୍ଲାଗିନ୍ ଉଲିଖନୀ ହୋଇଥାଏ, ତେବେ ଆପଣଙ୍କର ସମ୍ପୂର୍ଣ୍ଣ କାର୍ଯ୍ୟକାରୀ Ultimate Multisite ନେଟୱାର୍କ ପାଇଁ ଏହାକୁ ଅକ୍ଟିଭେଟ କରିବା ପୂର୍ବରୁ ଅକ୍ଟିଭେଟ କରିବାର ଆବଶ୍ୟକତା ରହିପାରେ।

ଏଠାରେ ସେଗୁଡ଼ିକର ଏକ ସୂଚୀ ଏବଂ ଏଥିରେ ଆପଣଙ୍କୁ କ'ଣ କରିବାକୁ ପଡିବ ତାହା ଦିଆଗଲା:

**କିଛି WPMU DEV ପ୍ଲାଗିନ୍** କିଛି ପ୍ଲାଗିନ୍ ଯାହା ବର୍ତ୍ତମାନ WPMU DEV ଦ୍ୱାରା ମଧ୍ୟ ରକ୍ଷା କରାଯାଉନାହିଁ, ସେଗୁଡ଼ିକ ଜଣାଶୁଣା ଭାବରେ Ultimate Multisite ରେ ନିଜର ଏକ ନିର୍ଦ୍ଦିଷ୍ଟ ପର୍ଯ୍ୟାୟରେ (ଯେପରିକି ବିଶେଷ ବ୍ୟକ୍ତିଙ୍କ ଆକାଉଣ୍ଟ ସୃଷ୍ଟି କରିବା) ପ୍ରଗତି ଅଟକାଇଥାନ୍ତି। ଏଠାରେ ସେହି ସମସ୍ୟା ସୃଷ୍ଟି କରୁଥିବା ପ୍ଲାଗିନ୍ଗୁଡ଼ିକର ଏକ ସୂଚୀ ଦିଆଗଲା:

  * Signup Language: _Ultimate Multisite ରେ ଏହାର ଏକ ଅଡ-ଅନ୍ (add-on) ରହିଛି ଯାହା ଏକ ସମାନ କାର୍ଯ୍ୟକତା ପ୍ରଦାନ କରେ, ଯାହାକୁ Ultimate Multisite: Language Switcher କୁହାଯାଏ।_

  * Set Blog Description on Blog Creation: _ଏହା Ultimate Multisite ରେ ବନ୍ଧି ରହିଛି, ତେଣୁ ଏଥିପାଇଁ ଅତିରିକ୍ତ ପ୍ଲାଗିନ୍ ଆବଶ୍ୟକ ନୁହେଁ।_

  * New Blog Templates: _ଏହା Ultimate Multisite ରେ ବନ୍ଧି ରହିଛି, ତେଣୁ ଏଥିପାଇଁ ଅତିରିକ୍ତ ପ୍ଲାଗିନ୍ ଆବଶ୍ୟକ ନୁହେଁ।_

  * Set Password on Multisite Blog Creation: _ଏହା Ultimate Multisite ରେ ବନ୍ଧି ରହିଛି, ତେଣୁ ଏଥିପାଇଁ ଅତିରିକ୍ତ ପ୍ଲାଗିନ୍ ଆବଶ୍ୟକ ନୁହେଁ।_

  * Signup TOS: _ଏହା Ultimate Multisite ରେ ବନ୍ଧି ରହିଛି, ତେଣୁ ଏଥିପାଇଁ ଅତିରିକ୍ତ ପ୍ଲାଗିନ୍ ଆବଶ୍ୟକ ନୁହେଁ।_
