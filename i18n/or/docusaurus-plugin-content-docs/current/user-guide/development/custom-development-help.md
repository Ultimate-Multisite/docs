# Codeable ମଧ୍ୟରୁ କଷ୍ଟମାଇଜେସନ୍ ସହାୟତା କିପରି ନ ಪಡೆಯବେ? {#how-to-get-custom-development-help-from-codeable}

କେବେ କେବେ ଆମକୁ ଆମର ୱାର୍କଫ୍‌ (network flows)କୁ ଆମ ଭାବରେ ଚାହିଁଥିବା ଅନୁଯାୟୀ କରିବା ପାଇଁ କିଛି ବହୁତ ଭଲ କଷ୍ଟମାଇଜେସନ୍ ଦରକାର ହୁଏ। ଯଦି ଆପଣ ଏହା ଖୋଜୁଛନ୍ତି, ତେବେ ଆମେ [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE)କୁ ପରାମର୍ଶ କରୁଛୁ, ଏକ ସେଠା ଯେଉଁଠି ଆପଣ Ultimate Multisite ବିଷମତ ଅଭିଜ୍ଞ ଡେଭଲପର୍ (developers)ଙ୍କ ସହିତ ମିଳିପାରିବେ।

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) ହେଉଛି ଏକ ପ୍ଲାଟଫର୍ମ ଯାହା ଗ୍ରାହକମାନଙ୍କୁ ଯୋଗ୍ୟ WordPress ଫ୍ର୍ରିଲାନ୍ସ ଡେଭଲପର୍ (freelance developers)ଙ୍କ ସହିତ ଯୋଡ଼ିଥାଏ, ଯେଉଁମାନେ କୌଣସି ବି ଆକାରର ପ୍ରୋଜେକ୍ଟରେ କାମ କରିପାରନ୍ତି। ଏଥିରେ ଏକ ବ୍ୟବୁଲ ମାତ୍ରାର ଆବେଦନ ପ୍ରକ୍ରିୟା ଅଛି ଯାହା ସୁନିଶ୍ଚିତ କରେ ଯେ ଆପଣଙ୍କୁ ଆପଣ ଚାହିଁଥିବା କାର୍ଯ୍ୟ ପାଇଁ ବିଶେଷ ଯୋଗ୍ୟ ବିଶେଷଜ୍ଞମାନଙ୍କ ସହିତ ମେଳ ହେବ।

### Codeable ରେ ପ୍ରୋଜେକ୍ଟ କିପରି ସୃଷ୍ଟି କରିବେ? {#how-to-create-a-project-in-codeable}

ପ୍ରଥମେ, [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE)କୁ ଯାଆନ୍ତୁ ଏବଂ **Start a Project** ବଟନ୍‌ଟି ଉପରେ କ୍ଲିକ୍ କରନ୍ତୁ।

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

ଏବେ, ପ୍ରୋଜେକ୍ଟ ସବମିତି ଦେବା ପାଇଁ, ଆଡ଼କୁ ଚାଲି **Start a Project** ଉପରେ କ୍ଲିକ୍ କରନ୍ତୁ।

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

"I need help with" (ମୋର ସହାୟତା ଦରକାର) ରେ, **Customizing** ବାଛନ୍ତୁ :

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

"My" (ମୋର) ରେ, **Plugin** ବାଛନ୍ତୁ :

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

ତା'ପରେ **Continue** ବଟନ୍‌ଟିକୁ କ୍ଲିକ୍ କରନ୍ତୁ।

<!-- Screenshot unavailable: Codeable project form Continue button -->

ଏବେ, ଆପଣଙ୍କ ପ୍ରୋଜେକ୍ଟ ଏବଂ ଆବଶ୍ୟକତାକୁ ସବୁଠାରୁ ଭଲ ଉପାୟରେ ବର୍ଣ୍ଣନା କରିବାର ସମୟ। ଯେତେ ବି ସମ୍ଭବ ତେରକାରୀ ସୂଚନା ଦେବାକୁ ଚେଷ୍ଟା କରନ୍ତୁ!

<!-- Screenshot unavailable: Codeable project description form -->

ବିଷୟବସ୍ତୁ ନିର୍ଦ୍ଧାରଣ କରିବାରେ ସାହାଯ୍ୟ ପାଇଁ ନିମ୍ନଲିଖିତ ପ୍ରଶ୍ନଗୁଡ଼ିକର ଉତ୍ତର ଦିଅନ୍ତୁ।

<!-- Screenshot unavailable: Codeable project pricing questions -->

ଏଥିରେ ଆପଣଙ୍କର [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) ଆକାଉଣ୍ଟ ନଥିଲେ, ଏହା ସୃଷ୍ଟି କରିବେ ଏବଂ **Publish My Project** ବଟନ୍ ଉପରେ କ୍ଲିକ୍ କରିବେ।

<!-- స్క్రీన్ ଅପଲଏବଲ: Codeable ଆକାଉଣ୍ଟ ସୃଷ୍ଟି ଏବଂ Publish My Project ବଟନ୍ -->

ଯଦି ଆପଣ ଆପଣଙ୍କର [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) ପ୍ରୋଜେକ୍ଟ ପ୍ଲାବ୍ଲିଶ୍ କରିଛନ୍ତି, ତେବେ ଏହି ପ୍ଲାଟଫର୍ମ ଆପଣଙ୍କୁ ଆପଣଙ୍କ ପ୍ରୋଜେକ୍ଟ ପାଇଁ ଆବଶ୍ୟକ କୌଶଳ ସହ ମେଳ ହେଉଥିବା ଡେଭଲପରମାନଙ୍କ ସହିତ ଯୋଡିବ।

ଏହି ପ୍ରକ୍ରିୟାଟି ଆପଣଙ୍କ ଚାକିରି ପାଇଁ ଠିକ୍ ବ୍ୟକ୍ତିକୁ ଖୋଜିବାରେ ସାହାଯ୍ୟ କରିବ, ଯାହାଙ୍କ ପାଖରେ WordPress ଏବଂ Ultimate Multisite ର ଅଭ୍ଳମ୍ବ ଅଛି!

Codeable ର ମୁନିମନ୍ତ୍ରକ ଦୈନିକ നിരക്ക് ହେଉଛି $70–$120 USD ପ୍ରତି ସମୟ। କାର୍ଯ୍ୟର পরিধি, ଜଟିଳତା ଏବଂ ପ୍ରୋଜେକ୍ଟର ଆବଶ୍ୟକତା ଅନୁସାରେ ଏହି ମୂଲ୍ୟ ବଦଳିପାରିବ।

ଏହା ପରେ ଯେତେବେଳେ ଆପଣଙ୍କର ଆକଳନ (estimate) ମିଳିବ, ସେତେବେଳେ ଆପଣ ଆପଣଙ୍କ ଆବଶ୍ୟକତାକୁ ସର୍ବୋତ୍ତର ଭାବରେ ଉପଯୁକ୍ତ ଡେଭଲପରକୁ ନିଯୁକ୍ତି କରି ଏହା ସହିତ ଆପଣଙ୍କ ପ୍ରୋଜେକ୍ଟ ଆରମ୍ଭ କରିପାରିବେ!
