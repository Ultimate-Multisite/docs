# WPMU DEV ইন্টিগ্রেশন

## ওভারভিউ {#overview}
WPMU DEV হলো একটি সম্পূর্ণ ওয়ার্ডপ্রেস প্ল্যাটফর্ম যা ওয়ার্ডপ্রেস সাইটগুলোর জন্য হোস্টিং, প্লাগইন এবং পরিষেবা প্রদান করে। এই ইন্টিগ্রেশনটি Ultimate Multisite এবং WPMU DEV হোস্টিংয়ের মধ্যে স্বয়ংক্রিয় ডোমেইন সিঙ্কিং এবং SSL সার্টিফিকেট পরিচালনার সুবিধা দেয়।

## বৈশিষ্ট্যসমূহ {#features}
- স্বয়ংক্রিয় ডোমেইন সিঙ্কিং
- SSL সার্টিফিকেট ব্যবস্থাপনা
- বর্ধিত SSL সার্টিফিকেট যাচাই করার চেষ্টা

## প্রয়োজনীয়তা {#requirements}
এই ইন্টিগ্রেশনটি স্বয়ংক্রিয়ভাবে সনাক্ত করে যে আপনি WPMU DEV-এ হোস্টিং করছেন এবং বিল্ট-ইন API ব্যবহার করে। যদি আপনি WPMU DEV-এ হোস্টিং করেন তবে অতিরিক্ত কোনো কনফিগারেশনের প্রয়োজন নেই।

ইন্টিগ্রেশনটি `WPMUDEV_HOSTING_SITE_ID` কনস্ট্যান্টটির উপস্থিতি পরীক্ষা করে, যা WPMU DEV-এ হোস্টিং করার সময় স্বয়ংক্রিয়ভাবে সংজ্ঞায়িত হয়।

## সেটআপ নির্দেশাবলী {#setup-instructions}

### ১. WPMU DEV হোস্টিং যাচাই করুন {#1-verify-wpmu-dev-hosting}

আপনি যদি WPMU DEV-এ হোস্টিং করেন, তবে প্রয়োজনীয় কনস্ট্যান্টগুলো ইতিমধ্যেই সংজ্ঞায়িত থাকা উচিত। নিচের বিষয়গুলো যাচাই করুন:

১. আপনার এনভায়রনমেন্টে `WPMUDEV_HOSTING_SITE_ID` কনস্ট্যান্টটি সংজ্ঞায়িত আছে কিনা
২. আপনার কাছে API অ্যাক্সেস সহ একটি সক্রিয় WPMU DEV সদস্যপদ আছে কিনা

### ২. ইন্টিগ্রেশন সক্ষম করুন {#2-enable-the-integration}

১. আপনার ওয়ার্ডপ্রেস অ্যাডমিন থেকে, Ultimate Multisite > Settings-এ যান।
২. "Domain Mapping" ট্যাবে যান।
৩. নিচে স্ক্রল করে "Host Integrations"-এ যান।
৪. WPMU DEV ইন্টিগ্রেশন চালু (Enable) করুন।
৫. "Save Changes" ক্লিক করুন।

## এটি কিভাবে কাজ করে {#how-it-works}

### ডোমেইন সিঙ্কিং {#domain-syncing}

যখন Ultimate Multisite-এ একটি ডোমেইন ম্যাপ করা হয়:

১. ইন্টিগ্রেশনটি আপনার হোস্টিং অ্যাকাউন্টে ডোমেইনটি যোগ করতে WPMU DEV API ব্যবহার করে।
২. এটি স্বয়ংক্রিয়ভাবে ডোমেইনের www সংস্করণটিও যোগ করে।
৩. ডোমেইন কনফিগারেশন এবং SSL সার্টিফিকেট ইস্যু করার কাজটি WPMU DEV পরিচালনা করে।

### SSL সার্টিফিকেট ব্যবস্থাপনা {#ssl-certificate-management}

ଏହି ଇଣ୍ଟିଗ୍ରେସନ୍‌କୁ SSL ସର୍ଟିଫିକେଟ୍ ଯାଞ୍ଚ କରିବାର ଚେଷ୍ଟା କରୁଥିବା ସଂଖ୍ୟା ବଢ଼ାଇବା ପାଇଁ ସେଟିଙ୍ଗ କରାଯାଇଛି, କାରଣ SSL ସର୍ଟିଫିକେଟ୍ ଜାରି ହେବା ଏବଂ ଇନଷ୍ଟଲ୍ ହେବା ପାଇଁ କିଛି ସମୟ ନେଇପାରେ। ডিଫାଉଟ୍ ଭାବରେ, ଏହା STD 5 ଚେଷ୍ଟା ଅପେକ୍ଷା SSL ସର୍ଟିଫିକେଟ୍ ଯାଞ୍ଚ କରିବାକୁ 10ကြိမ် ପର୍ଯ୍ୟନ୍ତ କରିବ, ଯାହା ମାନକ 5 ଚେଷ୍ଟାରୁ ବହୁତ ଅଧିକ।

## ଗୁରୁତ୍ୱପୂର୍ଣ୍ଣ ନୋଟ୍ସ {#important-notes}

### ଡୋମେନ୍ ହଟାଇବା (Domain Removal) {#domain-removal}

ବର୍ତ୍ତମାନ WPMU DEV API ଦ୍ୱାରା ଡୋମେନ୍ ବାହାର କରିବାର କୌଣସି ଉପାୟ ପ୍ରଦାନ କରାଯାଇନାହିଁ। Ultimate Multisite ରେ ଏକ ଡୋମେନ୍ ମ୍ୟପିଂ (domain mapping) ହଟାଇଲେ, ସେହି ଡୋମେନ୍ ଆପଣଙ୍କ WPMU DEV হোস্টিং ଆକାଉଣ୍ଟରେ ରହିବ। ଯଦି ଆବଶ୍ୟକ ହୁଏ, ଆପଣକୁ ଏହାକୁ WPMU DEV ହୋস্টিং ଡାଶବର୍ଡ୍ ମଧ୍ୟରୁ ମାନୁଆଲି ଭାବେ ਹଟାଇବାକୁ ପଡ଼ିବ।

### API ଅଥେନ୍ଟିଫିକେସନ୍ (API Authentication) {#api-authentication}

ଏହି ଇଣ୍ଟିଗ୍ରେସନ୍ ଆପଣଙ୍କ WordPress ଡାଟାବେସ୍‌ରେ `wpmudev_apikey` অপସନ ଭାବରେ ସଞ୍ଚୟ କରାଯାଇଥିବା WPMU DEV API key ବ୍ୟବହାର କରେ। ଆପଣ ଆପଣଙ୍କ ସାଇట్‌କୁ WPMU DEV ସହିତ ଯୋଡ଼ିଲେ ଏହା ଆଧିକାରରେ ସ୍ଥାପନ କରାଯାଏ।

## ଟ୍ରବୁଲଶୁଟିଂ (Troubleshooting) {#troubleshooting}

### API କନେକ୍ସନ୍ ସମସ୍ୟା {#api-connection-issues}
- ଯାଞ୍ଚ କରନ୍ତୁ ଯେ ଆପଣଙ୍କ ସାଇଟ୍ WPMU DEV ସହିତ ସଠିକ୍ ଭାବରେ ଜଡ଼ିତ ଅଛି।
- ଯାଞ୍ଚ କରନ୍ତୁ ଯେ `wpmudev_apikey` অপସନଟି ଆପଣଙ୍କ WordPress ଡାଟାବେସ୍‌ରେ ସେଟ ହୋଇଛି।
- ନିଶ୍ଚିତ କରନ୍ତୁ ଯେ ଆପଣଙ୍କ WPMU DEV ମେମ୍ବେରସିପ୍ (membership) ବ୍ୟାବହାରିକ ଅଛି।

### SSL ସର୍ଟିଫିକେଟ୍ ସମସ୍ୟା {#ssl-certificate-issues}
- WPMU DEV ଡୋମେନ୍ ପାଇଁ SSL ସର୍ଟିଫିକେଟ୍ ଜାରି କରିବାକୁ କିଛି ସମୟ ନେଇପାରେ (ଆଧୁର୍ଦ୍ଧରେ 5-15 ମିନିଟ୍)।
- ଏହି ଇଣ୍ଟିଗ୍ରେସନ୍ SSL ସର୍ଟିଫିକେଟ୍ ଯାଞ୍ଚ କରିବା ପାଇଁ 10ကြိမ် ପର୍ଯ୍ୟନ୍ତ କରାଯାଇଛି।
- ଯଦି ଅନେକ ଚେଷ୍ଟାର ପରେ ମଧ୍ୟ SSL ସର୍ଟିଫିକେଟ୍ ଜାରି ହେଉନାହିଁ, ତେବେ WPMU DEV ସପୋର୍ଟକୁ ଯୋଗାଯୋଗ କରନ୍ତୁ।

### ଡୋମେନ୍ ଯୋଡ଼ାଯାଇ ନଥିବା {#domain-not-added}
- କୌଣସି ଭୁଲ୍ ବାର୍ତ୍ତା ପାଇଁ Ultimate Multisite ଲଗ୍‌ରେ ଯାଞ୍ଚ କରନ୍ତୁ।
- ଯାଞ୍ଚ କରନ୍ତୁ ଯେ ଡୋମେନ୍ ଆପଣଙ୍କ WPMU DEV ରେ ପ୍ରବେଶିତ ହୋଇନାହିଁ।
- ନିଶ୍ଚିତ କରନ୍ତୁ ଯେ ଆପଣଙ୍କ WPMU DEV হোস্টিং ପ୍ଲାନଟି ଆପଣ ଯାହାକୁ ଯୋଡ଼ିବାକୁ ଚାହୁଁଛନ୍ତି ତାହା ସମର୍ଥନ କରେ।
