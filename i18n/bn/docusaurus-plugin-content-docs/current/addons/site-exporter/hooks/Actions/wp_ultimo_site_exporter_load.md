# Action: wp_ultimo_site_exporter_load

যখন সব নির্ভরতা (dependencies) লোড হয়ে যায় তখন এটি ট্রিগার হয়।

প্লাগইন ডেভেলপারদের নতুন কার্যকারিতা যোগ করতে এটি সুযোগ দেয়। উদাহরণস্বরূপ, নতুন হোস্টিং প্রোভাইডারদের সাপোর্ট যোগ করা ইত্যাদি।
