# Filter: wp_ultimo_site_exporter_url

ডেভেলপারদের প্লাগইন রুটের সাপেক্ষে রিল্যাটিভ পাথ (relative path) পরিচালনা করতে দেয়।

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $dir | `string` | প্লাগইন রুটের সাপেক্ষে যে পাথটি আপনি অ্যাক্সেস করতে চান। |

### Since

- 0.0.1
### Source

[`inc/class-helper.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-site-exporter/blob/main/inc/class-helper.php#L52) ফাইলে লাইন ৫২-এ সংজ্ঞায়িত।

## Returns
প্লাগইন রুটের সাপেক্ষে পাথ।
