# সাপোর্ট এজেন্টরা (Support Agents)

নিজে নিজে পুরো নেটওয়ার্ক পরিচালনা করা কোনো সহজ কাজ নয়। আপনার নেটওয়ার্ক যত বাড়তে শুরু করবে, সহায়তার চাহিদা তত বাড়তে থাকবে। আপনার কাজের চাপ কমাতে সাহায্য করার জন্য কিছু লোক নিয়োগ করা ভালো ধারণা। তবে, সম্পূর্ণ অ্যাক্সেস বা অনুমতি দেওয়া ঝুঁকিপূর্ণ এবং এটি আপনার নেটওয়ার্কের ক্ষতি করতে পারে। এখানেই এই Support Agent add-on টি কাজে আসে।

এই add-on টি আপনাকে Support Agent নামে এক নতুন ধরনের WordPress user তৈরি করতে দেয় এবং সেই ব্যবহারকারীটির কী কী অনুমতি থাকবে তার উপর আপনার সম্পূর্ণ নিয়ন্ত্রণ দেয়। অনুমতি দেওয়া একটি টগল বাটন অন করার মতোই সহজ, এবং যদি আপনি এটি সরাতে চান, তবে এটি অফ করে দিতে পারেন।

## বৈশিষ্ট্যসমূহ (Features)

*   WordPress এবং Ultimate Multisite-এর অনুমতিগুলির তালিকা।
*   সহজে অ্যাক্সেস চালু ও বন্ধ করা।
*   সীমাহীন সংখ্যক Agent অ্যাকাউন্ট তৈরি করা।

## ইনস্টলেশন (Installation)

1.  'ultimate-multisite-support-agents' ফাইলটি '/wp-content/plugins/' ডিরেক্টরিতে আপলোড করুন।
2.  WordPress-এর 'Plugins' মেনু থেকে প্লাগইনটি অ্যাক্টিভেট করুন।
3.  প্লাগইনটি সেটআপ করার জন্য ধাপে ধাপে উইজার্ডটি অনুসরণ করুন।
