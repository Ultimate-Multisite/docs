# Action: wp_ultimo_support_agents_load

যখন সমস্ত নির্ভরতা (dependencies) লোড হয়ে যায় তখন এটি ট্রিগার হয়।

প্লাগইন ডেভেলপারদের নতুন কার্যকারিতা যোগ করতে এটি সাহায্য করে। উদাহরণস্বরূপ, নতুন হোস্টিং প্রোভাইডারের জন্য সাপোর্ট যোগ করা ইত্যাদি।
