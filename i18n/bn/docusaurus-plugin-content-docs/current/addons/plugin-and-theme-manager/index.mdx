# Plugin ও Theme Manager {#plugin--theme-manager}

Multisite Ultimate: Plugin এবং Theme Manager

ইনস্টল করা প্লাগইন এবং থিমগুলোর মেটা তথ্য (title, author, thumbnail এবং description) এডিট বা লুকিয়ে রাখতে পারবেন। প্লাগইন এবং থিমগুলোর জন্য ক্যাটাগরি তৈরি করুন, যাতে আপনার ব্যবহারকারীরা সেগুলো ফিল্টার করতে পারে এবং আপনার ব্যবহারকারীদের প্যানেলে একটি সুন্দর কাস্টম Plugins পেজ যোগ করতে পারেন।

প্লাগইনটি ব্যবহার করতে, আপনি শুধু আপনার নেটওয়ার্ক অ্যাডমিনে আপনার Plugins বা Themes পেজে গিয়ে "Edit Plugin/Theme info" এ ক্লিক করতে পারেন।

## ইনস্টলেশন {#installation}

1. 'ultimate-multisite-plugin-and-theme-manager' ফাইলটি '/wp-content/plugins/' ডিরেক্টরিতে আপলোড করুন
2. আপনার WordPress Network Admin-এর 'Plugins' মেনু থেকে প্লাগইনটি অ্যাক্টিভেট করুন

## ব্যবহারের নির্দেশিকা {#usage-guide}

Ultimate Multisite: Plugin and Theme Manager হলো একটি শক্তিশালী **premium** Ultimate Multisite add-on, যা আপনার নেটওয়ার্কে আপনার ব্যবহারকারীরা কীভাবে প্লাগইন এবং থিমের অপশনগুলো দেখতে এবং ব্যবহার করতে পারে, তার উপর সম্পূর্ণ নিয়ন্ত্রণ দেয়।

এই add-on টি তিনটি উদ্দেশ্যে কাজ করে:

  1. এটি আপনাকে প্লাগইন এবং থলি—উভয়ের মেটাডেটা সম্পূর্ণভাবে পরিবর্তন করতে দেয়। এর মধ্যে নাম, বিবরণ (description), লেখকের নাম এবং এমনকি থাম্বনেইলও (thumbnails) পরিবর্তন করা যায়!

  2. এটি আপনাকে প্লাগইন এবং থিমগুলোকে ক্যাটাগরি করতে দেয়, যাতে আপনার ব্যবহারকারীরা সহজেই সেগুলোকে সাজিয়ে নিতে পারে।

  3. এটি আপনাকে পুরনো, একঘেয়ে plugins পেজটিকে ক্যাটাগরি অনুযায়ী সাজানো একটি নতুন themes-এর মতো পেজ দিয়ে প্রতিস্থাপন করার বিকল্প দেয়।

এটি ডাউনলোড এবং ইনস্টল করার পরে, Ultimate Multisite Settings পেজে একটি নতুন ট্যাব—_Add-on Settings_—দেখা যাবে।
