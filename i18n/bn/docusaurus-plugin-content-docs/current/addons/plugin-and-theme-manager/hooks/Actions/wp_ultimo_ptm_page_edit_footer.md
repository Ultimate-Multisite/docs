# Action: wp_ultimo_ptm_page_edit_footer

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $ | `object` | তথ্য ধারণকারী অবজেক্ট (Object holding the information) |
| $ | `\WU_Page` | মাল্টিসাইট আলটিমেট পেজ ইনস্ট্যান্স (Multisite Ultimate Page instance) |
