# মাল্টি-নেটওয়ার্ক চেঞ্জলগ {#multi-network-changelog}

### 1.3.0 {#130}
* নতুন: নেটওয়ার্ক টেমপ্লেট প্রিভিউয়ার — কেনার আগে লাইভ প্যানেলে নেটওয়ার্ক টেমপ্লেট ব্রাউজ এবং প্রিভিউ করুন।
* নতুন: চেকআউটে নেটওয়ার্ক টেমপ্লেট পিকার — গ্রাহকরা সাইনআপ চেকআউট ফ্লোর সময় একটি নেটওয়ার্ক টেমপ্লেট নির্বাচন করতে পারবেন।
* নতুন: ক্রস-নেটওয়ার্ক ম্যাজিক লিঙ্ক SSO — গ্রাহকরা ম্যাজিক-লিঙ্ক অথেন্টিকেশনের মাধ্যমে সাব-নেটওয়ার্কে সহজে অ্যাক্সেস করতে পারবেন।
* নতুন: নেটওয়ার্ক মিডিয়া ক্লোনিং — টেমপ্লেট থেকে নেটওয়ার্ক ক্লোন করার সময় সাইটের মিডিয়া সঠিকভাবে ডুপ্লিকেট হবে।
* ফিক্স: মাল্টি-টেন্যান্সি অ্যাডনন ছাড়া এখন নেটওয়ার্ক ক্লোনিং কাজ করবে; ক্লোন করার পরে সুপার অ্যাডমিন প্রিভিলেজ পুনরায় দেওয়া হবে।
* ফিক্স: নেটওয়ার্ক তৈরি এখন WooCommerce এবং সমস্ত পেমেন্ট গেটওয়ের সাথে সামঞ্জস্যপূর্ণ।
* ফিক্স: ফলব্যাক ডোমেইন জেনারেশনে এখন পোর্ট নম্বর অন্তর্ভুক্ত করা হয়েছে।
* ফিক্স: সমস্ত থিমের উপর সঠিক রেন্ডারিং নিশ্চিত করতে টেমপ্লেট পিকার CSS একটি এনকিউড স্টাইলশীটে সরানো হয়েছে।
* ফিক্স: সাইট কোয়েরি স্কোপে undefined `network_id` কী-এর বিরুদ্ধে সুরক্ষা যোগ করা হয়েছে।
* ফিক্স: কনটেক্সট সুইচিং করার আগে নেটওয়ার্ক পারমিশন চেক করার ফলে `wpdb` টেবিল প্রপার্টি ত্রুটি প্রতিরোধ করা গেছে।
