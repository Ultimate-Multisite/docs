# Action: add_network

যখন একটি নতুন নেটওয়ার্ক যোগ করা হয়, তখন এটি কার্যকর হয়।

## প্যারামিটারসমূহ {#parameters}

| নাম | ধরন | বিবরণ |
|------|------|-------------|
| $new_network_id | `int` | যোগ করা নেটওয়ার্কের আইডি। |
| $r | `array` | নেটওয়ার্ক আর্গুমেন্টগুলোর সম্পূর্ণ অ্যাসোসিয়েটিভ অ্যারে। |
