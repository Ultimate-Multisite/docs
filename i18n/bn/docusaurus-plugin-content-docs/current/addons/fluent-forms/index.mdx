# Fluent Forms {#fluent-forms}

এই অ্যাড-অনটি Ultimate Multisite-এ পণ্য, মেম্বারশিপ বা সাইটগুলির ক্ষেত্রে Fluent Forms-এর জন্য সীমা (limits) যোগ করার সুবিধা দেয়।
Fluent Forms কাস্টম পোস্ট টাইপের পরিবর্তে কাস্টম টেবিল ব্যবহার করে। এই কারণে, সাধারণ লিমিটিং অপশনগুলো Fluent Forms-এর সাথে কাজ করে না।

## বৈশিষ্ট্যসমূহ {#features}

- Fluent Forms বা Conversational Fluent Forms-এর জন্য সীমা যোগ করা
- যখন গ্রাহক সীমা অতিক্রম করবেন, তখন তারা প্ল্যান আপগ্রেড করার জন্য একটি বন্ধুত্বপূর্ণ বার্তা পাবেন।
