# ফিল্টার: wp_ultimo_skip_network_active_check {#filter-wpultimoskipnetworkactivecheck}

ডেভেলপারদের এই চেকটি বাইপাস করার সুযোগ দেয়।

এটি তখন খুব কাজে আসে যখন আপনি কম্পোজার-ভিত্তিক এবং অন্যান্য কাস্টম সেটআপ ব্যবহার করেন, যেমন Bedrock। এই ধরনের সেটআপে, mu-plugins হিসেবে প্লাগইন ব্যবহার করা সাধারণ ব্যাপার।
