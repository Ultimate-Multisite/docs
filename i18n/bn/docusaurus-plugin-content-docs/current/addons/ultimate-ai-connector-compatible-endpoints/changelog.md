# Ultimate AI Connector for Compatible Endpoints পরিবর্তন লগ

## Version 2.0.0 — প্রকাশিত হয়েছে 2026-04-24

* নতুন: মাল্টি-প্রোভাইডার সাপোর্ট — একাধিক AI এন্ডপয়েন্ট কনফিগার করুন এবং স্বয়ংক্রিয় ফলব্যাক (automatic fallback) সহ বিভিন্ন প্রোভাইডারের মধ্যে রিকোয়েস্ট রাউটিং করুন।
* ফিক্স: প্রতিটি প্রোভাইডারের জন্য সঠিক প্রোভাইডার আইডি, রেজিস্ট্রেশন ইউআরএল এবং HTTP ফিল্টার স্কোপিং সহ মাল্টি-প্রোভাইডার SDK ইন্টিগ্রেশন।
* ফিক্স: নতুন প্রোভাইডার কার্ড এখন যোগ করার সাথে সাথে স্বয়ংক্রিয়ভাবে প্রসারিত হয়; প্লাগইন আপডেট করার সময় স্ক্রিপ্ট ক্যাশে বাস্টিং (script cache busting) করা হয়েছে।
* ফিক্স: eval() এর জন্য ডাইনামিক প্রোভাইডার ক্লাস নেমস্পেস এবং নতুন যোগ করা প্রোভাইডারের জন্য স্বয়ংক্রিয় প্রসারণ আচরণ (auto-expand behaviour)।
* ফিক্স: WordPress 6.9+ সামঞ্জস্যতার জন্য স্থিতিশীল Card/CardBody/CardHeader/CardDivider কম্পোনেন্ট ব্যবহার করা হয়েছে (যা আর পরীক্ষামূলক নয়)।
* ফিক্স: প্রোভাইডার ড্র্যাগ-টু-রিয়র্ডার করার জন্য undefined DragHandle এর পরিবর্তে ইউনিকোড গ্রিপ আইকন ব্যবহার করা হয়েছে।
* ফিক্স: প্রতিটি পেজ লোডের সময় যে ব্লকিং HTTP রিকোয়েস্ট আসত, তা সরিয়ে ফেলা হয়েছে।
* উন্নত: GitHub Actions ওয়ার্কফ্লো Node.js 24 এ আপগ্রেড করা হয়েছে।
