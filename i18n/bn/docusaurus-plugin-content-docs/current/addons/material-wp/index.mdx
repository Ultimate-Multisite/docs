# Material WP {#material-wp}

ওয়ার্ডপ্রেস অ্যাডমিন ইন্টারফেস অনেক দিন ধরে একই রকম। কিন্তু আপনি কি কখনও ভেবে দেখেছেন যে অ্যাডমিন ইন্টারফেসের একটি Material Design সংস্করণ কেমন দেখতে হতে পারে?

Material WP হলো একটি অ্যাডমিন থিম যা আপনাকে সেই লুক এবং ফিল দিতে তৈরি করা হয়েছে, যা আপনাকে আপনার ক্লায়েন্টদেরকে আরও কাস্টমাইজড Dashboard দিতে সাহায্য করবে।

## বৈশিষ্ট্য (Features) {#features}

- কাস্টম CSS সাপোর্ট
- সহজে স্টাইলিং সেটিংস
- মেনু পুনর্বিন্যাস (Menu reordering)
- সহজ ইমপোর্ট/এক্সপোর্ট

## ডকুমেন্টেশন (Documentation) {#documentation}

**Material WP দিয়ে আপনার অ্যাডমিন প্যানেল কাস্টমাইজ করুন**

## ইনস্টলেশন (Installation) {#installation}

1. 'material-wp' ফাইলটি '/wp-content/plugins/' ডিরেক্টরিতে আপলোড করুন
2. ওয়ার্ডপ্রেসের 'Plugins' মেনু থেকে প্লাগইনটি অ্যাক্টিভেট করুন

## ব্যবহার নির্দেশিকা (Usage Guide) {#usage-guide}

ওয়ার্ডপ্রেস অ্যাডমিন ইন্টারফেস অনেক দিন ধরে একই রকম। কিন্তু আপনি কি কখনও ভেবে দেখেছেন যে অ্যাডমিন ইন্টারফেসের একটি Material Design সংস্করণ কেমন দেখতে হতে পারে?

Material WP হলো একটি অ্যাডমিন থিম যা আপনাকে সেই লুক এবং ফিল দিতে তৈরি করা হয়েছে, যা আপনাকে আপনার ক্লায়েন্টদেরকে আরও কাস্টমাইজড Dashboard দিতে সাহায্য করবে। Material WP হলো Ultimate Multisite-এর প্রিমিয়াম প্লাগইন কোর-এরও অংশ।

**এটি ইনস্টল করার** পরে, আপনার _Plugins_ পেজে যান এবং আপনার নেটওয়ার্কে এটি অ্যাক্টিভেট করুন।

এখন, আপনার অ্যাডমিন ইন্টারফেসটি সম্পূর্ণভাবে নতুন করে ডিজাইন করা! এছাড়াও, বাম কলামে আপনার Material WP নামে একটি নতুন মেনু পাবেন।
