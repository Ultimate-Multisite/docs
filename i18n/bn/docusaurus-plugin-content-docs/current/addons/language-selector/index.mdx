# ভাষা নির্বাচক (Language Selector)

Multisite Ultimate: সাইন-আপে ভাষা নির্বাচক

সাইন-আপ ফ্লোতে একটি ভাষা নির্বাচক ফিল্ড যোগ করুন।

## ইনস্টলেশন {#installation}

১. 'ultimate-multisite-language-selector' ফাইলটি '/wp-content/plugins/' ডিরেক্টরিতে আপলোড করুন।
২. আপনার WordPress Network Admin-এর 'Plugins' মেনু থেকে প্লাগইনটি অ্যাক্টিভেট করুন।

## ব্যবহার নির্দেশিকা (Usage Guide) {#usage-guide}

এই ফ্রি অ্যাড-অন ব্যবহার করে আপনি আপনার ব্যবহারকারীদের সাইন-আপ ফ্লো এবং লগইন পেজে কোন ভাষা দেখতে চান তা বেছে নিতে দিতে পারবেন।

[আপনার লাইসেন্স সক্রিয় করার পরে](../../user-guide/getting-started/installing-ultimate-multisite.md) আপনি সেই Ultimate Multisite অ্যাড-অনগুলি ইনস্টল করতে সক্ষম হবেন যা আপনার লাইসেন্স অ্যাক্সেস দেয়।

আপনার Ultimate Multisite সেটিংস পেজে যান এবং তারপর, বাম দিকের মেনুতে, _Add-ons_ লেবেলে ক্লিক করুন।

![Add-ons page](/img/admin/addons-page.png)

_Language Selector ইনস্টল করতে অ্যাড-অনস পেজে যান_

সেখানে আপনি আমাদের দেওয়া সমস্ত অ্যাড-অন খুঁজে পাবেন।

ফ্রি অ্যাড-অনগুলি আমাদের সকল গ্রাহকদের জন্য উপলব্ধ, তারা কোন সাবস্ক্রিপশন প্ল্যানে সাইন আপ করেছেন তা নির্বিশেষে।

অ্যাড-অনটির থাম্বনেইলের নিচে থাকা _Install it Now!_ বোতামে ক্লিক করুন। তারপর প্লাগইনস পেজে এটি অ্যাক্টিভেট করুন।

![Add-ons page with install button](/img/admin/addons-page.png)

_অ্যাড-অনটি ইনস্টল করতে বোতামে ক্লিক করুন_

_আপনার প্লাগইনস পেজে এটি অ্যাক্টিভেট করুন_

এখন, Ultimate Multisite-এর সেটিংস মেনুতে যান এবং _Network Settings_ ট্যাবে প্রবেশ করুন। সেখানে, পেজের একদম নিচে, আপনি _Languages_ নামে একটি নতুন সেকশন খুঁজে পাবেন।

![Settings page](/img/config/settings-general.png)

_একটি একদম নতুন সেকশন দেখা যাচ্ছে!_

আপনি চাইলে এটি ফাঁকা রাখতে পারেন যাতে সমস্ত ভাষা অপশন হিসেবে দেখানো হয়, অথবা আপনি কোন ভাষাগুলি উপলব্ধ করতে চান তা নির্বাচন করতে পারেন। ভাষাগুলি হলো সেইগুলি যা আপনি আপনার WordPress সেটআপে ইনস্টল করেছেন।

## URL এর মাধ্যমে ভাষা সেট করা {#setting-the-language-via-the-url}

আপনি URL-এর locale প্যারামিটার বাইপাস করেও একটি পেজের ভাষা পরিবর্তন করতে পারেন। উদাহরণস্বরূপ, আপনি যদি ইতালীয় ভাষায় একটি রেজিস্ট্রেশন পেজে রিডাইরেক্ট করতে চান, তবে URL-এ ?locale=it_IT যোগ করুন।

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

এটি কাজ করার জন্য ভাষাটি আপনার WordPress নেটওয়ার্কে ইনস্টল করা থাকতে হবে।
