# Filter: wu_apc_should_display_admin_menu

অ্যাডমিনদের কাস্টমাইজেশন প্রক্রিয়া শেষ করার পরে অ্যাডমিন মেনু পেজগুলো লুকিয়ে রাখার অনুমতি দেয়।

## প্যারামিটারগুলো {#parameters}

| Name | Type | বিবরণ |
|------|------|-------------|
| $should_display_admin_menu | `bool` | যদি আমাদের পেজগুলো দেখাতে হয়। |
| $network_active | `bool` | চেক করে যে প্লাগইনটি নেটওয়ার্ক অ্যাডমিন হিসেবে সক্রিয় কিনা। |
