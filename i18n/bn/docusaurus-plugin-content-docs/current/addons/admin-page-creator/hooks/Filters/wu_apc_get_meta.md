# Filter: wu_apc_get_meta {#filter-wuapcgetmeta}

ডেভেলপারদের চূড়ান্ত মানগুলো ফিল্টার করতে এবং নতুন প্রক্রিয়াকরণ ক্ষেত্র যোগ করতে দেয়।

## প্যারামিটারসমূহ {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $default_value | `string` | ডিফল্ট মান। |
| $meta_type | `string` | যে মেটা ডেটা পুনরুদ্ধার করতে হবে তার ধরন। |
| $meta_key | `string` | মেটা ফিল্ডের নাম। |

### শুরু {#since}

- 1.4.0
### উৎস {#source}

[inc/wu-apc-functions.php](https://github.com/Ultimate-Multisite/ultimate-multisite-admin-page-creator/blob/main/inc/wu-apc-functions.php#L93)-এ লাইন ৯৩-এ সংজ্ঞায়িত করা হয়েছে।


## ফেরত দেয় {#returns}
নতুন ডিফল্ট মান।
