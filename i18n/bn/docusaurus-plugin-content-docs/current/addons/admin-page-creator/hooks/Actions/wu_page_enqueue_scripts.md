# Action: wu_page_enqueue_scripts

এখানে একটি কাস্টম অ্যাকশন যোগ করা হচ্ছে, যাতে প্রয়োজন অনুযায়ী স্ক্রিপ্ট এবং স্টাইলগুলো এনকিউ (enqueue) করার একটি জায়গা থাকে এবং নিশ্চিত করা যায় যে সেগুলো শুধুমাত্র সঠিক পেজগুলোতে লোড হয়।

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $ | `\WU_Page` | বর্তমান অ্যাডমিন পেজটি |
