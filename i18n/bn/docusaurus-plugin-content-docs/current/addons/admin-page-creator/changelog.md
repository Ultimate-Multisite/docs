# Admin Page Creator Changelog

### Version 1.9.0 - Released on 2026-01-18

* Fixed: সক্রিয় করার সময় মারাত্মক ত্রুটি (Fatal errors when activating)।
* Refactored: কনটেন্ট সোর্স ক্লাসগুলোকে সঠিক প্যারেন্ট কনস্ট্রাক্টর ইনিশিয়ালাইজেশন সহ Singleton trait ব্যবহার করার জন্য পুনর্গঠিত করা হয়েছে।
* Fixed: Vue.js টেমপ্লেট কম্পাইলেশন ত্রুটি যা কনটেন্ট টাইপ সিলেক্টরকে রেন্ডার করতে বাধা দিচ্ছিল।
* Fixed: অনুপস্থিত Vue ডেটা প্রপার্টিগুলো (template_id, external_link_url_embedable, external_link_url_checking)।
* Fixed: পেজ লোডের সময় TinyMCE এডিটর সঠিকভাবে ইনিশিয়ালাইজ না হওয়া।
* Fixed: Composer এর মাধ্যমে Divi সাপোর্ট ফাইল স্বয়ংক্রিয়ভাবে লোড না হওয়া।
* Fixed: Divi সাপোর্ট ক্লাসে PHP সিনট্যাক্স দুর্নীতিগ্রস্ত হওয়া।
* Improved: JavaScript অ্যাসেট সংগঠন এবং মিনিফিকেশন উন্নত করা হয়েছে।
* Improved: সর্বশেষ WordPress এবং পেজ বিল্ডার সংস্করণগুলির সাথে সামঞ্জস্যতা উন্নত করা হয়েছে।
