# কনটেন্ট সিঙ্ক চেঞ্জলগ

### Version 1.1.0 - প্রকাশের তারিখ 2025-12-31 {#version-110---released-on-2025-12-31}
* নতুন: সব ধরনের পোস্ট টাইপ (পোস্ট, পেজ, কাস্টম পোস্ট টাইপ) সিঙ্ক করার সমর্থন।
* নতুন: কনটেন্ট নির্বাচনের ধাপে পোস্ট টাইপ ফিল্টারিং।
* নতুন: সোর্স সাইট নির্বাচন ধাপ - শুধুমাত্র মূল সাইট থেকেই নয়, যেকোনো সাইট থেকে সিঙ্ক করা যাবে।
* নতুন: টেমপ্লেট সাইটের জন্য চাইল্ড সাইট প্রি-সিলেকশন।
* নতুন: সব পোস্ট টাইপের জন্য post_content থেকে মিডিয়া এক্সট্রাকশন।
* নতুন: ট্যাক্সোনমি সিঙ্কিং (ক্যাটাগরি, ট্যাগ, কাস্টম ট্যাক্সোনমি)।
* পরিবর্তন: "Elementor Template Sync" এর নাম পরিবর্তন করে "Content Sync" করা হয়েছে।
* পরিবর্তন: "Select Templates" ধাপটির নাম পরিবর্তন করে "Select Data" করা হয়েছে।
* উন্নত: Elementor-নির্দিষ্ট কোড এখন শুধুমাত্র elementor_library পোস্ট টাইপের জন্য চলবে।

### Version 1.0.0 - প্রকাশের তারিখ 2025-12-19 {#version-100---released-on-2025-12-19}
* প্রাথমিক রিলিজ।
* টেমপ্লেট এবং সাইট নির্বাচনের জন্য বহু-ধাপের উইজার্ড।
* রিয়েল-টাইম অগ্রগতি ফিডব্যাক সহ ব্যাচ প্রসেসিং।
* স্বয়ংক্রিয় ID রিম্যাপিং সহ সম্পূর্ণ মিডিয়া সিঙ্ক।
* Ultimate Multisite এর সাথে সেটিংস ইন্টিগ্রেশন।
* নেটওয়ার্ক সেটিংসের মাধ্যমে সাইট বাদ দেওয়ার সমর্থন।
* সিঙ্ক করার পরে Elementor ক্যাশে ক্লিয়ারিং।
* টেমপ্লেট ট্যাক্সোনমি সংরক্ষণ।
