# Filter: ultimate_multisite_content_sync_url {#filter-ultimatemultisitecontentsyncurl}

ডেভেলপারদের প্লাগইন রুটের আপেক্ষিক পথটি (relative path) পরিচালনা করার সুযোগ দেয়।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $dir | `string` | প্লাগইন রুটের সাপেক্ষে যে পথটি আপনি অ্যাক্সেস করতে চান। |

### Since {#since}

- 1.0.0
### Source {#source}

[`inc/class-helper.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-content-sync/blob/main/inc/class-helper.php#L52)-এ লাইন ৫২-এ সংজ্ঞায়িত।

## Returns {#returns}
প্লাগইন রুটের সাপেক্ষে পথ।
