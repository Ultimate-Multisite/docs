# কন্টেন্ট সিঙ্ক {#content-sync}

Multisite Ultimate: কন্টেন্ট সিঙ্ক

Multisite Ultimate-এর জন্য এই অ্যাডনটি নেটওয়ার্ক অ্যাডমিনিস্ট্রেটরদের যেকোনো সাইট থেকে নেটওয়ার্কের অন্যান্য সাইটগুলিতে কন্টেন্ট সিঙ্ক্রোনাইজ করতে সাহায্য করে। এটি পোস্ট, পেজ, কাস্টম পোস্ট টাইপ এবং Elementor টেমপ্লেট সহ সমস্ত পোস্ট টাইপ সমর্থন করে। অ্যাডনটিতে কন্টেন্ট এবং টার্গেট সাইট নির্বাচন করার জন্য একটি মাল্টি-স্টেপ উইজার্ড ইন্টারফেস রয়েছে, যা ব্যাচ প্রসেসিং এবং রিয়েল-টাইম প্রোগ্রেস ফিডব্যাক প্রদান করে।

## বৈশিষ্ট্যসমূহ {#features}

* কন্টেন্ট নির্বাচন: থাম্বনেইল এবং পোস্ট টাইপ ফিল্টারিং সহ ভিজ্যুয়াল কার্ড গ্রিড
* সমস্ত পোস্ট টাইপ: পোস্ট, পেজ, প্রোডাক্ট, Elementor টেমপ্লেট এবং যেকোনো কাস্টম পোস্ট টাইপ সিঙ্ক করুন
* সাইট নির্বাচন: কোন সাইটগুলিতে কন্টেন্ট সিঙ্ক করতে হবে তা নির্বাচন করুন। এখানে সোর্স সাইট বাদ দেওয়া এবং চাইল্ড সাইট প্রি-সিলেকশনের সুবিধা রয়েছে।
* ব্যাচ প্রসেসিং: প্রোগ্রেস আপডেট সহ একবারে ১০টি সাইটে সিঙ্ক করে।
* মিডিয়া সিঙ্ক: পোস্ট কন্টেন্ট থেকে টার্গেট সাইটগুলিতে মিডিয়া ফাইলগুলি (ছবি, ভিডিও) স্বয়ংক্রিয়ভাবে কপি এবং রিম্যাপ করে।
* ট্যাক্সোনমি সিঙ্ক: ক্যাটাগরি, ট্যাগ এবং কাস্টম ট্যাক্সোনমি সংরক্ষণ করে।
* Elementor সাপোর্ট: CSS রিনোজেনেশন এবং টেমপ্লেট আইডি রিম্যাপিং সহ Elementor টেমপ্লেটগুলির জন্য সম্পূর্ণ সাপোর্ট।
* কনফ্লিক্ট রেজোলিউশন: বিদ্যমান কন্টেন্ট প্রতিস্থাপন করার বা এটি এড়িয়ে যাওয়ার বিকল্প।
* ক্যাশে ম্যানেজমেন্ট: সিঙ্ক করার পরে টার্গেট সাইটগুলিতে Elementor ক্যাশে স্বয়ংক্রিয়ভাবে পরিষ্কার করে।

## প্রয়োজনীয়তা {#requirements}

* WordPress 5.3 বা তার বেশি
* PHP 7.4 বা তার বেশি
* Multisite Ultimate প্লাগইন (সক্রিয়)
* Elementor প্লাগইন (ঐচ্ছিক, Elementor টেমপ্লেট সিঙ্কের জন্য প্রয়োজন)

## ইনস্টলেশন {#installation}

১. 'ultimate-multisite-content-sync' ফাইলটি '/wp-content/plugins/' ডিরেক্টরিতে আপলোড করুন।
২. আপনার WordPress Network Admin-এর 'Plugins' মেনু থেকে প্লাগইনটি অ্যাক্টিভেট করুন।
৩. Ultimate Multisite → Settings → Content Sync-এ সেটিংস কনফিগার করুন।
৪. সিঙ্কিং শুরু করতে Ultimate Multisite → Content Sync-এ নেভিগেট করুন।
