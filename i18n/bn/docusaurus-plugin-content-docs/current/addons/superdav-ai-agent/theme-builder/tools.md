# টুলস এবং কনফিগারেশন

থিম বিল্ডার আপনার ওয়ার্ডপ্রেস সাইট বিশ্লেষণ করতে, ডিজাইন করতে এবং তৈরি করতে কিছু টুলের সেট ব্যবহার করে। ভার্সন ১.১৬.০-এ, **sd-ai-agent/site-scrape এখন একটি টায়ার ১ টুল**, যা থিম বিল্ডারে ডিফল্টভাবে উপলব্ধ করে তুলেছে।

## উপলব্ধ টুলস

### টায়ার ১ টুলস (সর্বদা উপলব্ধ)

টায়ার ১ টুলস কোনো অতিরিক্ত কনফিগারেশন ছাড়াই থিম বিল্ডারে ডিফল্টভাবে উপলব্ধ।

#### sd-ai-agent/site-scrape

**উদ্দেশ্য:** ডিজাইন অনুপ্রেরণা, কনটেন্ট কাঠামো এবং কার্যকারিতা বের করার জন্য বিদ্যমান ওয়েবসাইটগুলি বিশ্লেষণ করা।

**কার্যকারিতা:**

- **ওয়েবসাইট বিশ্লেষণ** — প্রতিদ্বন্দ্বী বা অনুপ্রেরণামূলক ওয়েবসাইটগুলি স্ক্র্যাপ এবং বিশ্লেষণ করা
- **ডিজাইন এক্সট্রাকশন** — রং, ফন্ট এবং লেআউটের প্যাটার্নগুলি চিহ্নিত করা
- **কনটেন্ট কাঠামো** — পেজের সংগঠন এবং হায়ারার্কি বোঝা
- **ফিচার ডিটেকশন** — প্লাগইন এবং কার্যকারিতা চিহ্নিত করা
- **পারফরম্যান্স বিশ্লেষণ** — পেজের গতি এবং অপটিমাইজেশন পরীক্ষা করা
- **এসইও বিশ্লেষণ** — মেটা ট্যাগ এবং স্ট্রাকচার্ড ডেটা পর্যালোচনা করা

**ব্যবহার:**

```
Analyze the design of example.com and use it as inspiration for my site.
```

**যা এটি বের করে:**

- কালার প্যালেট এবং রঙের ব্যবহার
- টাইপোগ্রাফি (ফন্ট এবং আকার)
- লেআউট কাঠামো এবং স্পেসিং
- নেভিগেশন প্যাটার্ন
- কনটেন্ট সংগঠন
- ছবি এবং মিডিয়া ব্যবহার
- ইন্টারেক্টিভ উপাদান
- মোবাইল রেসপনসিভনেস

**সীমাবদ্ধতা:**

- পাসওয়ার্ড-সুরক্ষিত সাইট স্ক্র্যাপ করতে পারে না
- robots.txt এবং সাইট নীতিকে সম্মান করে
- ডাইনামিক কনটেন্ট নাও ধরতে পারে
- জাভাস্ক্রিপ্ট-ভারী সাইটগুলিতে বিশ্লেষণ সীমিত হতে পারে
- বড় সাইটগুলি বিশ্লেষণ করতে বেশি সময় নিতে পারে

### টায়ার ২ টুলস (ঐচ্ছিক)

টায়ার ২ টুলস থিম বিল্ডারের সেটিংসে স্পষ্টভাবে সক্ষম করার পরে উপলব্ধ হয়।

#### Advanced Analytics

বিস্তারিত পারফরম্যান্স মেট্রিক্স প্রদান করে:

- পেজ লোড টাইম
- কোর ওয়েব ভাইটালস (Core Web Vitals)
- এসইও স্কোর
- অ্যাক্সেসিবিলিটি স্কোর
- বেস্ট প্র্যাকটিসেস স্কোর

#### Content Optimizer

নিম্নলিখিত বিষয়গুলির জন্য বিশ্লেষণ করে এবং উন্নতির পরামর্শ দেয়:

- পঠনযোগ্যতা (Readability)
- এসইও অপটিমাইজেশন
- কীওয়ার্ড ব্যবহার
- কনটেন্ট কাঠামো
- কল-টু-অ্যাকশন প্লেসমেন্ট

### টায়ার ৩ টুলস (প্রিমিয়াম)

টায়ার ৩ টুলগুলির জন্য অতিরিক্ত কনফিগারেশন বা প্রিমিয়াম ফিচারের প্রয়োজন।

#### AI Content Generator

নিম্নলিখিত বিষয়গুলির জন্য কনটেন্ট তৈরি করে:

- প্রোডাক্ট ডেসক্রিপশন
- সার্ভিস পেজ
- ব্লগ পোস্ট
- মেটা ডেসক্রিপশন
- কল-টু-অ্যাকশন কপি

#### Advanced Design Tools

উন্নত ডিজাইন ক্ষমতা প্রদান করে:

- কাস্টম CSS জেনারেশন
- অ্যানিমেশন তৈরি
- ইন্টারেক্টিভ উপাদান ডিজাইন
- অ্যাডভান্সড কালার থিওরি
- টাইপোগ্রাফি অপটিমাইজেশন

## টুল কনফিগারেশন

### টুলস সক্ষম করা

থিম বিল্ডারে অতিরিক্ত টুলস সক্ষম করতে:

১. **Gratis AI Agent → Settings** এ যান
২. **Theme Builder → Tools** এ যান
৩. প্রয়োজন অনুযায়ী টুলস চালু/বন্ধ করুন
৪. সেটিংস সেভ করুন

### টুল পারমিশনস

কিছু টুলের জন্য পারমিশনের প্রয়োজন:

- **Site scraping** — ইন্টারনেট অ্যাক্সেসের প্রয়োজন
- **Analytics** — Google Analytics ইন্টিগ্রেশনের প্রয়োজন
- **Content generation** — এপিআই কী (API keys)-এর প্রয়োজন
- **Advanced features** — প্রিমিয়াম সাবস্ক্রিপশনের প্রয়োজন হতে পারে

### এপিআই কী এবং ক্রেডেনশিয়ালস

যেসব টুলের জন্য এপিআই কী প্রয়োজন, সেগুলোর কনফিগার করুন:

১. **Gratis AI Agent → Settings → API Keys** এ যান
২. প্রতিটি টুলের জন্য ক্রেডেনশিয়ালস দিন
৩. কানেকশন টেস্ট করুন
৪. নিরাপদে সেভ করুন

## sd-ai-agent/site-scrape ব্যবহার করা

### মৌলিক ব্যবহার

থিম বিল্ডারকে একটি ওয়েবসাইট বিশ্লেষণ করতে বলুন:

```
Analyze the design of my competitor's site at competitor.com
and suggest design improvements for my site.
```

### নির্দিষ্ট বিশ্লেষণ

নির্দিষ্ট ধরনের বিশ্লেষণের অনুরোধ করুন:

```
Extract the color palette from example.com and use it as inspiration.
```

```
Analyze the navigation structure of example.com and apply similar
organization to my site.
```

```
Check the mobile responsiveness of example.com and ensure my site
is equally responsive.
```

### তুলনা বিশ্লেষণ

একাধিক সাইট তুলনা করুন:

```
Compare the designs of site1.com and site2.com and create a design
that combines the best elements of both.
```

## টুল সীমাবদ্ধতা এবং বিবেচ্য বিষয়

### রেট লিমিটিং (Rate Limiting)

- সার্ভার ওভারলোড রোধ করতে স্ক্র্যাপিং রেট-লিমিটেড থাকে
- একই সাইটে একাধিক অনুরোধ থ্রটেল হতে পারে
- বড় সাইটগুলি বিশ্লেষণ করতে বেশি সময় নিতে পারে

### কনটেন্ট নির্ভুলতা (Content Accuracy)

- ডাইনামিক কনটেন্ট পুরোপুরি ধরা নাও পড়তে পারে
- জাভাস্ক্রিপ্ট দ্বারা রেন্ডার করা কনটেন্ট অসম্পূর্ণ হতে পারে
- রিয়েল-টাইম ডেটা প্রতিফলিত নাও হতে পারে
- কিছু কনটেন্ট পে-ওয়াল বা পেইড পেজের পিছনে থাকতে পারে

### আইনি এবং নৈতিক বিবেচনা

- robots.txt এবং সাইট নীতিকে সম্মান করুন
- পুনরুৎপাদনের জন্য কপিরাইটযুক্ত কনটেন্ট স্ক্র্যাপ করবেন না
- অনুপ্রেরণার জন্য বিশ্লেষণ ব্যবহার করুন, কপি করার জন্য নয়
- নিশ্চিত করুন যে আপনার কাছে নিষ্কাশিত কনটেন্ট ব্যবহারের অধিকার আছে
- বিশ্লেষণ করা সাইটগুলির পরিষেবার শর্তাবলী অনুসরণ করুন

### পারফরম্যান্স প্রভাব

- বড় সাইট স্ক্র্যাপ করতে সময় লাগতে পারে
- একাধিক সমসাময়িক স্ক্র্যাপ ধীর হতে পারে
- নেটওয়ার্ক কানেক্টিভিটি গতিকে প্রভাবিত করে
- বড় ডেটাসেটগুলির জন্য আরও বেশি প্রসেসিং প্রয়োজন হতে পারে

## সেরা অনুশীলন (Best Practices)

### সাইট বিশ্লেষণ ব্যবহার করা

১. **একাধিক সাইট বিশ্লেষণ করুন** — বিভিন্ন উৎস থেকে অনুপ্রেরণা সংগ্রহ করুন
২. **কাঠামোর উপর মনোযোগ দিন** — লেআউট এবং সংগঠনের প্যাটার্ন শিখুন
৩. **রং বের করুন** — কালার প্যালেটকে শুরুর বিন্দু হিসেবে ব্যবহার করুন
৪. **টাইপোগ্রাফি অধ্যয়ন করুন** — আপনার পছন্দের ফন্ট কম্বিনেশন চিহ্নিত করুন
৫. **নেভিগেশন পর্যালোচনা করুন** — মেনু সংগঠন বুঝুন

### নৈতিক স্ক্র্যাপিং

১. **robots.txt সম্মান করুন** — সাইট নির্দেশিকা অনুসরণ করুন
২. **কনটেন্ট কপি করবেন না** — শুধুমাত্র অনুপ্রেরণার জন্য বিশ্লেষণ ব্যবহার করুন
৩. **অধিকার যাচাই করুন** — নিশ্চিত করুন যে আপনি নিষ্কাশিত কনটেন্ট ব্যবহার করতে পারেন
৪. **উৎস উল্লেখ করুন** — অনুপ্রেরণার উৎসকে ক্রেডিট দিন
৫. **শর্তাবলী অনুসরণ করুন** — সাইট পরিষেবার শর্তাবলী মেনে চলুন

### টুল কার্যকারিতা সর্বাধিক করা

১. **নির্দিষ্ট হন** — নির্দিষ্ট ধরনের বিশ্লেষণের অনুরোধ করুন
২. **প্রসঙ্গ দিন** — আপনার সাইটের উদ্দেশ্য ব্যাখ্যা করুন
৩. **প্রত্যাশা সেট করুন** — আপনার ডিজাইন লক্ষ্যগুলি বর্ণনা করুন
৪. **পুনরাবৃত্তি করুন** — ফলাফলের ভিত্তিতে পরিমার্জন করুন
৫. **টুলস একত্রিত করুন** — ব্যাপক বিশ্লেষণের জন্য একাধিক টুল ব্যবহার করুন

## সমস্যা সমাধান (Troubleshooting)

### সাইট স্ক্র্যাপ হচ্ছে না

- পরীক্ষা করুন সাইটটি পাবলিকলি অ্যাক্সেসযোগ্য কিনা
- যাচাই করুন robots.txt স্ক্র্যাপিং অনুমতি দেয় কিনা
- অন্য একটি সাইট চেষ্টা করুন
- ইন্টারনেট সংযোগ পরীক্ষা করুন
- সমস্যাটি চলতে থাকলে সাপোর্টের সাথে যোগাযোগ করুন

### অসম্পূর্ণ বিশ্লেষণ

- সাইটে ডাইনামিক কনটেন্ট থাকতে পারে
- জাভাস্ক্রিপ্ট পুরোপুরি রেন্ডার নাও হতে পারে
- বড় সাইটগুলি টাইমআউট করতে পারে
- পরিবর্তে নির্দিষ্ট পেজ বিশ্লেষণ করার চেষ্টা করুন
- নির্দিষ্ট বিশ্লেষণ ধরনের অনুরোধ করুন

### ধীর পারফরম্যান্স

- বড় সাইটগুলি বিশ্লেষণ করতে বেশি সময় নেয়
- একাধিক সমসাময়িক স্ক্র্যাপ ধীর হয়
- নেটওয়ার্ক কানেক্টিভিটি গতির উপর প্রভাব ফেলে
- অফ-পিক আওয়ারে চেষ্টা করুন
- প্রথমে ছোট অংশগুলি বিশ্লেষণ করুন
