# Superdav AI Agent

Superdav AI Agent হলো একটি উন্নত WordPress এআই সহকারী যা Gratis AI Agent-কে থিম-সচেতন দক্ষতা (theme-aware skills), উন্নত ইন্টারনেট অনুসন্ধান ক্ষমতা এবং উন্নত সাইট বিল্ডার বৈশিষ্ট্য দিয়ে আরও শক্তিশালী করে তোলে। ভার্সন ১.১৬.০-এ কাস্টম লোগো তৈরি, প্যালেট কনট্রাস্ট ভ্যালিডেশন, ডিসকভারি ইন্টারভিউতে ফটো আপলোড, স্ট্রাকচার্ড হসপিটালিটি মেনু তৈরি এবং থিম বিল্ডারে ডেস্কটপ/মোবাইল প্রিভিউ রেন্ডারিং যোগ করা হয়েছে।

## মূল বৈশিষ্ট্য (Key Features)

- **থিম-সচেতন দক্ষতা (Theme-aware skills)** — আপনার সক্রিয় WordPress থিম এবং ইনস্টল করা প্লাগইনগুলির উপর ভিত্তি করে স্বয়ংক্রিয়ভাবে গাইডেন্স এবং ক্ষমতাকে মানিয়ে নেয়।
- **একাধিক ইন্টারনেট সার্চ প্রোভাইডার (Multiple internet search providers)** — আরও সমৃদ্ধ সার্চ ফলাফল এবং উন্নত গবেষণার ক্ষমতার জন্য Brave Search এবং Tavily এর মধ্যে বেছে নিতে পারেন।
- **সাইট বিল্ডার কন্টাক্ট ফর্ম (Site builder contact forms)** — আপনার ইনস্টল করা ফর্ম প্লাগইন ব্যবহার করে চ্যাট ইন্টারফেস থেকেই সম্পূর্ণ কার্যকরী কন্টাক্ট ফর্ম তৈরি করুন।
- **উন্নত সাইট বিল্ডার (Enhanced site builder)** — উন্নত প্লাগইন ডিসকভারি এবং ত্রুটি পুনরুদ্ধারের (error recovery) সাথে উন্নত মাল্টি-ফেজ সাইট বিল্ড প্ল্যান।
- **বিল্ট-ইন থিম দক্ষতা (Built-in theme skills)** — ব্লক থিম, ক্লাসিক থিম, Kadence Blocks এবং Kadence Theme-এর জন্য বিশেষ নির্দেশিকা।
- **উন্নত WooCommerce ইন্টিগ্রেশন (Improved WooCommerce integration)** — আরও ভালো নির্ভরযোগ্যতা এবং সামঞ্জস্যতার জন্য নেটিভ WooCommerce API ব্যবহার করার জন্য রিফ্যাক্টর করা হয়েছে।
- **স্বয়ংক্রিয় প্রোভাইডার সনাক্তকরণ (Automatic provider detection)** — প্লাগইন বা থিম সক্রিয় বা নিষ্ক্রিয় করা হলে স্বয়ংক্রিয়ভাবে উপলব্ধ প্লাগইন এবং থিমগুলি সনাক্ত করে এবং তা রিফ্রেশ করে।
- **কাস্টম লোগো তৈরি (Custom logo generation)** — স্বয়ংক্রিয় নেমস্পেস-নিরাপদ স্যানিটাইজেশন সহ কাস্টম SVG লোগো তৈরি এবং এমবেড করুন।
- **প্যালেট কনট্রাস্ট ভ্যালিডেশন (Palette contrast validation)** — আপনার থিমে প্রয়োগ করার আগে WCAG অ্যাক্সেসিবিলিটি সম্মতি যাচাই করতে রঙের জোড়া পরীক্ষা করুন।
- **ফটো-增强 ডিসকভারি (Photo-enhanced discovery)** — থিম বিল্ডার ডিসকভারি ইন্টারভিউ চলাকালীন ফটো আপলোড করুন যাতে ডিজাইন দিকনির্দেশনাকে প্রভাবিত করা যায়।
- **হসপিটালিটি মেনু (Hospitality menus)** — রেস্টুরেন্ট, ক্যাফে এবং হসপিটালিটি ব্যবসার জন্য স্ট্রাকচার্ড খাদ্য ও পানীয় মেনু পেজ তৈরি করুন।
- **ডেস্কটপ এবং মোবাইল প্রিভিউ (Desktop and mobile previews)** — ডিজাইন-ডিরেকশন নির্বাচনের সময় ডেস্কটপ এবং মোবাইল ডিভাইসে আপনার ডিজাইন প্রিভিউ করুন।
- **Tier 1 সাইট স্ক্র্যাপিং (Tier 1 site scraping)** — sd-ai-agent/site-scrape এখন ডিফল্টভাবে উপলব্ধ, যা প্রতিদ্বন্দ্বী এবং অনুপ্রেরণামূলক ওয়েবসাইটগুলি বিশ্লেষণ করতে ব্যবহৃত হয়।

## প্রয়োজনীয়তা (Requirements)

- WordPress 6.0 বা তার বেশি
- PHP 8.1 বা তার বেশি
- WordPress Multisite (network-activated) অথবা single-site ইনস্টলেশন
- একটি সক্রিয় এআই প্রোভাইডার কানেক্টর প্লাগইন (Anthropic, OpenAI, বা সামঞ্জস্যপূর্ণ)
- Gratis AI Agent v1.9.0 বা তার বেশি (Superdav AI Agent, Gratis AI Agent-কে প্রসারিত করে)

## ইনস্টলেশন (Installation)

1. প্লাগইন ফাইলগুলি `/wp-content/plugins/superdav-ai-agent`-এ আপলোড করুন।
2. WordPress-এর **Plugins** স্ক্রিনের মাধ্যমে প্লাগইনটি অ্যাক্টিভেট করুন।
3. নিশ্চিত করুন যে Gratis AI Agent-ও অ্যাক্টিভেট করা আছে।
4. **Gratis AI Agent → Settings** এ নেভিগেট করুন এবং আপনার ইন্টারনেট সার্চ প্রোভাইডার কনফিগার করুন (ঐচ্ছিক)।

## দ্রুত শুরু (Quick Start)

একবার অ্যাক্টিভেট হয়ে গেলে, Superdav AI Agent স্বয়ংক্রিয়ভাবে থিম-সচেতন দক্ষতা এবং অতিরিক্ত ক্ষমতা দিয়ে আপনার Gratis AI Agent ইনস্টলেশনকে উন্নত করে। WordPress অ্যাডমিন সাইডবারে **Gratis AI Agent** প্যানেলটি খুলুন এবং নতুন বৈশিষ্ট্যগুলি ব্যবহার করা শুরু করুন।

**উদাহরণ নির্দেশাবলী:**

```
Add a contact form to the Contact page.
```

```
Search the web for the latest WordPress security best practices.
```

```
Create a hero section using the active theme's block patterns.
```

## থিম-সচেতন দক্ষতা (Theme-Aware Skills)

Superdav AI Agent স্বয়ংক্রিয়ভাবে আপনার সক্রিয় থিম এবং ইনস্টল করা প্লাগইনগুলি সনাক্ত করে, উপযুক্ত থিম-সচেতন দক্ষতাটি অ্যাসিস্ট্যান্টের কনটেক্সটে লোড করে। উপলব্ধ দক্ষতাগুলির মধ্যে রয়েছে:

| Skill | Applies To |
|---|---|
| **Block Themes** | থিম যা WordPress ব্লক-ভিত্তিক (Full Site Editing) আর্কিটেকচার ব্যবহার করে |
| **Classic Themes** | PHP টেমপ্লেট এবং ক্লাসিক এডিটর ব্যবহার করে ঐতিহ্যবাহী WordPress থিম |
| **Kadence Blocks** | যে সাইটগুলিতে Kadence Blocks প্লাগইন সক্রিয় আছে |
| **Kadence Theme** | যে সাইটগুলিতে Kadence থিম সক্রিয় থিম হিসেবে আছে |

প্রতিটি দক্ষতার বিস্তারিত নির্দেশিকা পেতে [Theme-Aware Skills](../../user-guide/configuration/theme-aware-skills) দেখুন।

## ইন্টারনেট সার্চ প্রোভাইডার (Internet Search Providers)

Superdav AI Agent v1.10.0 ইন্টারনেট সার্চ কোয়েরির জন্য Brave Search-এর বিকল্প হিসেবে **Tavily** যোগ করেছে। উভয় প্রোভাইডারই রিয়েল-টাইম সার্চ ফলাফল অফার করে এবং এগুলি **Gratis AI Agent → Settings → Advanced** এ কনফিগার করা যেতে পারে।

| Provider | Features | Setup |
|---|---|---|
| **Brave Search** | দ্রুত, প্রাইভেসি-কেন্দ্রিক সার্চ ফলাফল | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | আরও সমৃদ্ধ সার্চ ফলাফল, উন্নত গবেষণার ক্ষমতা | [Tavily API](https://tavily.com/) |

কনফিগারেশন নির্দেশাবলীর জন্য [Internet Search](../../user-guide/configuration/internet-search) দেখুন।

## সাইট বিল্ডার কন্টাক্ট ফর্ম (Site Builder Contact Forms)

সাইট বিল্ডার এজেন্ট এখন চ্যাট ইন্টারফেস থেকে সরাসরি কন্টাক্ট ফর্ম তৈরি করতে পারে। শুধু সাইট বিল্ডারকে একটি কন্টাক্ট ফর্ম তৈরি করতে বলুন, এবং এটি স্বয়ংক্রিয়ভাবে আপনার ইনস্টল করা ফর্ম প্লাগইন (Contact Form 7, WPForms, Fluent Forms, বা Gravity Forms) সনাক্ত করে এবং ব্যবহার করার জন্য একটি ফর্ম তৈরি করে।

বিস্তারিত নির্দেশিকার জন্য [Site Builder Contact Forms](../../user-guide/administration/site-builder-contact-forms) দেখুন।

## থিম বিল্ডার বৈশিষ্ট্য (Theme Builder Features)

Superdav AI Agent v1.16.0-এ থিম বিল্ডারে শক্তিশালী নতুন ক্ষমতাগুলি অন্তর্ভুক্ত করা হয়েছে:

### ফটো আপলোড সহ ডিসকভারি ইন্টারভিউ (Discovery Interview with Photo Upload)

থিম বিল্ডার ডিসকভারি ইন্টারভিউতে এখন একটি ফটো আপলোড ধাপ রয়েছে যেখানে আপনি ভিজ্যুয়াল অনুপ্রেরণা দিতে পারেন। থিম বিল্ডারকে আপনার ডিজাইন দিকনির্দেশনা বুঝতে সাহায্য করার জন্য আপনার ব্র্যান্ড, প্রতিদ্বন্দ্বী সাইট বা ডিজাইন অনুপ্রেরণার ছবি আপলোড করুন।

বিস্তারিত নির্দেশিকার জন্য [Discovery Interview](./theme-builder/discovery-interview.md) দেখুন।

### ডেস্কটপ এবং মোবাইল প্রিভিউ সহ ডিজাইন দিকনির্দেশনা (Design Direction with Desktop and Mobile Previews)

থিম বিল্ডার আপনার সম্পূর্ণ থিম তৈরি করার আগে ডেস্কটপ এবং মোবাইল উভয় ডিভাইসে আপনার ডিজাইন পর্যালোচনা করুন। একাধিক রেসপনসিভ ব্রেকপয়েন্টে রিয়েল-টাইম প্রিভিউ দেখুন।

বিস্তারিত নির্দেশিকার জন্য [Design Direction](./theme-builder/design-direction.md) দেখুন।

### হসপিটালিটি মেনু (Hospitality Menus)

থিম বিল্ডার এখন রেস্টুরেন্ট, ক্যাফে, বার এবং অন্যান্য হসপিটালিটি ব্যবসার জন্য স্ট্রাকচার্ড খাদ্য ও পানীয় মেনু পেজ তৈরি করতে পারে। মেনুগুলিতে মূল্য, বিবরণ, খাদ্যের তথ্য এবং অ্যালার্জেন সতর্কতা অন্তর্ভুক্ত থাকে।

বিস্তারিত নির্দেশিকার জন্য [Hospitality Menus](./theme-builder/hospitality-menus.md) দেখুন।

### নতুন ক্ষমতা (New Abilities)

- **[Generate Logo SVG](./abilities/generate-logo-svg.md)** — স্বয়ংক্রিয় স্যানিটাইজেশন সহ কাস্টম SVG লোগো তৈরি করুন।
- **[Validate Palette Contrast](./abilities/validate-palette-contrast.md)** — WCAG সম্মতি যাচাই করার জন্য রঙের জোড়া পরীক্ষা করুন।
- **[Create Menu](./abilities/create-menu.md)** — স্বতন্ত্র নেভিগেশন লেবেল সহ নেভিগেশন মেনু তৈরি করুন।

উপলব্ধ টুল এবং Tier 1 সাইট-স্ক্র্যাপ টুল সম্পর্কে তথ্যের জন্য [Tools and Configuration](./theme-builder/tools.md) দেখুন।

## কনফিগারেশন (Configuration)

সমস্ত সেটিংস WordPress অ্যাডমিনে **Gratis AI Agent → Settings** এর অধীনে রয়েছে। Superdav AI Agent নিম্নলিখিত কনফিগারেশন বিকল্পগুলি যোগ করে:

| Setting | Description |
|---|---|
| **Internet Search Provider** | Brave Search এবং Tavily এর মধ্যে বেছে নিন (ঐচ্ছিক) |
| **Tavily API Key** | Tavily ইন্টারনেট সার্চের জন্য API কী (যদি Tavily নির্বাচন করা হয়) |

## v1.16.0 এ উন্নতি (Improvements in v1.16.0)

- **AI Client cache** — এখন ট্রানজিয়েন্ট (transients) দ্বারা সমর্থিত, যা ক্রস-রিকোয়েস্ট পারসিসটেন্স নিশ্চিত করে এবং দীর্ঘ চলমান এজেন্ট কাজগুলিতে ডেটা ক্ষতি রোধ করে।
- **Tier 1 site-scrape tool** — প্রতিদ্বন্দ্বী এবং অনুপ্রেরণামূলক ওয়েবসাইটগুলি বিশ্লেষণ করার জন্য থিম বিল্ডারে sd-ai-agent/site-scrape এখন ডিফল্টভাবে উপলব্ধ।
- **Plugin row action links** — স্পষ্টতার জন্য সংশোধন এবং নাম পরিবর্তন করা হয়েছে।

## v1.10.0 এ উন্নতি (Improvements in v1.10.0)

- **WooCommerce integration refactored** — এখন আরও ভালো নির্ভরযোগ্যতা এবং সামঞ্জস্যতার জন্য নেটিভ WooCommerce API ব্যবহার করে।
- **Provider list auto-refresh** — স্বয়ংক্রিয়ভাবে সনাক্ত করে যখন প্লাগইনগুলি সক্রিয় বা নিষ্ক্রিয় করা হয় এবং উপলব্ধ প্রোভাইডার তালিকা আপডেট করে।
- **Enhanced site builder** — উন্নত ত্রুটি পুনরুদ্ধারের সাথে উন্নত মাল্টি-ফেজ সাইট বিল্ড প্ল্যান।
- **Theme-aware skills** — চারটি নতুন বিল্ট-ইন দক্ষতা যা স্বয়ংক্রিয়ভাবে আপনার সক্রিয় থিমের সাথে মানিয়ে নেয়।
