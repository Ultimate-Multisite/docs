# Superdav AI Agent

Superdav AI Agent হলো একটি উন্নত WordPress AI সহকারী, যা Theme-সচেতন দক্ষতা, উন্নত ইন্টারনেট সার্চ ক্ষমতা, সাইট বিল্ডার বৈশিষ্ট্য এবং ঐচ্ছিক অটোমেশন ইন্টিগ্রেশন দিয়ে Gratis AI Agent-কে বিস্তৃত করে। সংস্করণ 1.18.0-এ Google Calendar পড়ার টুল, যোগাযোগ ও অংশগ্রহণকারী ম্যাপিং, মানব অনুমোদন গেট, রিমাইন্ডার ডিডুপ্লিকেশন, TextBee SMS নোটিফিকেশন, পরিচালিত Superdav সার্ভিস প্রভিশনিং এবং বিশ্বস্ত ডেভেলপার ওয়ার্কফ্লোর জন্য আলাদাভাবে বিতরণ করা Advanced companion প্যাকেজ যোগ করা হয়েছে।

## প্রধান বৈশিষ্ট্য {#key-features}

- **Theme-সচেতন দক্ষতা** — আপনার সক্রিয় WordPress theme এবং ইনস্টল করা plugin অনুযায়ী নির্দেশনা ও ক্ষমতা স্বয়ংক্রিয়ভাবে মানিয়ে নেয়
- **একাধিক ইন্টারনেট সার্চ provider** — আরও সমৃদ্ধ সার্চ ফলাফল এবং উন্নত গবেষণা ক্ষমতার জন্য Brave Search ও Tavily-এর মধ্যে বেছে নিন
- **সাইট বিল্ডার যোগাযোগ ফর্ম** — আপনার ইনস্টল করা ফর্ম plugin ব্যবহার করে chat interface থেকেই সম্পূর্ণ কার্যকর যোগাযোগ ফর্ম তৈরি করুন
- **উন্নত সাইট বিল্ডার** — আরও ভালো plugin আবিষ্কার ও ত্রুটি পুনরুদ্ধারসহ উন্নত বহু-পর্বের সাইট build plan
- **অন্তর্নির্মিত theme দক্ষতা** — Block Themes, Classic Themes, Kadence Blocks এবং Kadence Theme-এর জন্য বিশেষ নির্দেশনা
- **উন্নত WooCommerce ইন্টিগ্রেশন** — আরও ভালো নির্ভরযোগ্যতা ও সামঞ্জস্যের জন্য native WooCommerce API ব্যবহারে পুনর্গঠিত
- **স্বয়ংক্রিয় provider সনাক্তকরণ** — plugin ও theme সক্রিয় বা নিষ্ক্রিয় হলে উপলভ্য plugin ও theme স্বয়ংক্রিয়ভাবে শনাক্ত ও রিফ্রেশ করে
- **কাস্টম লোগো তৈরি** — স্বয়ংক্রিয় namespace-safe sanitisation সহ কাস্টম SVG লোগো তৈরি ও এমবেড করুন
- **প্যালেট কনট্রাস্ট যাচাই** — আপনার theme-এ প্রয়োগের আগে WCAG accessibility compliance-এর জন্য রঙের জোড়া পরীক্ষা করুন
- **ছবিসহ discovery** — ডিজাইনের দিকনির্দেশে প্রভাব ফেলতে Theme Builder discovery interview চলাকালে ছবি আপলোড করুন
- **Hospitality মেনু** — রেস্তোরাঁ, ক্যাফে এবং hospitality ব্যবসার জন্য কাঠামোবদ্ধ খাবার ও পানীয় মেনু পৃষ্ঠা তৈরি করুন
- **ডেস্কটপ ও মোবাইল preview** — design-direction selection-এর সময় desktop ও mobile device-এ আপনার ডিজাইনের preview দেখুন
- **Tier 1 site scraping** — প্রতিদ্বন্দ্বী ও অনুপ্রেরণার ওয়েবসাইট বিশ্লেষণের জন্য sd-ai-agent/site-scrape এখন default হিসেবে উপলভ্য
- **Google Calendar পড়ার টুল** — কনফিগার করা calendar ও event পড়ুন, যাতে automation আসন্ন schedule context বিবেচনা করতে পারে
- **যোগাযোগ ও অংশগ্রহণকারী mapping** — follow-up work তৈরির আগে calendar অংশগ্রহণকারীদের পরিচিত site user ও contact-এর সঙ্গে মিলিয়ে নিন
- **মানব অনুমোদন gate** — বার্তা পাঠানো, কাজ পুনরায় শুরু করা বা পরিবর্তন করার আগে review-এর জন্য সংবেদনশীল automation থামিয়ে রাখুন
- **রিমাইন্ডার record ও deduplication** — ইতিমধ্যে পাঠানো reminder track করুন, যাতে পুনরাবৃত্ত automation run একই প্রাপককে দুবার notify না করে
- **TextBee SMS নোটিফিকেশন** — SMS provider enabled থাকলে TextBee-এর মাধ্যমে কনফিগার করা text-message notification পাঠান
- **পরিচালিত Superdav service provisioning** — manual endpoint configuration-এর বদলে automatic setup সহ supported site-গুলোকে hosted Superdav endpoint-এর সঙ্গে connect করুন
- **Advanced companion package** — filesystem, database, WP-CLI, REST dispatcher, plugin builder, git snapshot, user-management এবং benchmark workflow-এর জন্য আলাদাভাবে বিতরণ করা trusted-developer tool

## প্রয়োজনীয়তা {#requirements}

- WordPress 6.0 বা উচ্চতর
- PHP 8.1 বা উচ্চতর
- WordPress Multisite (network-activated) অথবা single-site installation
- একটি active AI provider connector plugin (Anthropic, OpenAI, অথবা compatible)
- Gratis AI Agent v1.9.0 বা উচ্চতর (Superdav AI Agent, Gratis AI Agent-কে বিস্তৃত করে)

## ইনস্টলেশন {#installation}

1. plugin ফাইলগুলো `/wp-content/plugins/superdav-ai-agent`-এ আপলোড করুন
2. WordPress-এ **Plugins** screen থেকে plugin সক্রিয় করুন
3. Gratis AI Agent-ও সক্রিয় আছে তা নিশ্চিত করুন
4. **Gratis AI Agent → Settings**-এ যান এবং প্রয়োজন অনুযায়ী আপনার internet search provider, managed Superdav service connection, Google Calendar, অথবা TextBee credentials কনফিগার করুন

WordPress.org package-এ core Superdav AI Agent বৈশিষ্ট্যগুলো রয়েছে। Advanced companion package আলাদাভাবে build ও distribute করা হয় এমন environment-এর জন্য, যেখানে trusted developer tool স্পষ্টভাবে install ও authorise করা হয়।

## দ্রুত শুরু {#quick-start}

সক্রিয় হওয়ার পর, Superdav AI Agent theme-aware skill এবং অতিরিক্ত capability দিয়ে আপনার Gratis AI Agent installation স্বয়ংক্রিয়ভাবে উন্নত করে। WordPress admin sidebar-এ **Gratis AI Agent** panel খুলুন এবং নতুন বৈশিষ্ট্যগুলো ব্যবহার শুরু করুন।

**উদাহরণ নির্দেশনা:**

```
Contact পৃষ্ঠায় একটি contact form যোগ করুন।
```

```
সর্বশেষ WordPress নিরাপত্তা best practice-এর জন্য web-এ search করুন।
```

```
active theme-এর block pattern ব্যবহার করে একটি hero section তৈরি করুন।
```

```
আগামীকালের configured Google Calendar event review করুন এবং attendee-দের জন্য reminder draft করুন।
```

```
আমি message approve করার পর একটি SMS reminder পাঠান।
```

## Theme-সচেতন দক্ষতা {#theme-aware-skills}

Superdav AI Agent স্বয়ংক্রিয়ভাবে আপনার সক্রিয় theme এবং ইনস্টল করা plugin শনাক্ত করে, assistant-এর context-এ উপযুক্ত theme-aware skill লোড করে। উপলভ্য skill-গুলোর মধ্যে রয়েছে:

| দক্ষতা | প্রযোজ্য |
|---|---|
| **Block Themes** | WordPress block-based (Full Site Editing) architecture ব্যবহার করা theme |
| **Classic Themes** | PHP template এবং classic editor ব্যবহার করা প্রচলিত WordPress theme |
| **Kadence Blocks** | Kadence Blocks plugin সক্রিয় থাকা site |
| **Kadence Theme** | active theme হিসেবে Kadence theme থাকা site |

প্রতিটি skill সম্পর্কে বিস্তারিত নির্দেশনার জন্য [Theme-সচেতন দক্ষতা](../../user-guide/configuration/theme-aware-skills) দেখুন।

## ইন্টারনেট সার্চ provider {#internet-search-providers}

Superdav AI Agent v1.10.0 ইন্টারনেট সার্চ query-এর জন্য Brave Search-এর বিকল্প হিসেবে **Tavily** যোগ করেছে। উভয় provider real-time search result দেয় এবং **Gratis AI Agent → Settings → Advanced**-এ কনফিগার করা যায়।

| প্রদানকারী | বৈশিষ্ট্য | সেটআপ |
|---|---|---|
| **Brave Search** | দ্রুত, গোপনীয়তাকেন্দ্রিক অনুসন্ধান ফলাফল | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | আরও সমৃদ্ধ অনুসন্ধান ফলাফল, উন্নত গবেষণা সক্ষমতা | [Tavily API](https://tavily.com/) |

কনফিগারেশন নির্দেশনার জন্য [ইন্টারনেট অনুসন্ধান](../../user-guide/configuration/internet-search) দেখুন।

## ক্যালেন্ডার, অনুমোদন, এবং নোটিফিকেশন ওয়ার্কফ্লো {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 কনটেন্ট ও সাইট তৈরির কাজের বাইরে অটোমেশন ওয়ার্কফ্লো সম্প্রসারিত করে:

- **Google Calendar টুল** ক্যালেন্ডার credentials উপলভ্য থাকলে agent-কে কনফিগার করা ক্যালেন্ডার এবং ইভেন্টের বিবরণ পড়তে দেয়।
- **যোগাযোগ ও অংশগ্রহণকারী সহায়ক** রিমাইন্ডার বা ফলো-আপ কাজ প্রস্তুতের আগে ইভেন্ট অংশগ্রহণকারীদের WordPress ব্যবহারকারী বা পরিচিত যোগাযোগের সঙ্গে মিলিয়ে দেয়।
- **মানব অনুমোদন গেট** প্রস্তাবিত পদক্ষেপ কোনো অনুমোদিত ব্যবহারকারী পর্যালোচনা না করা পর্যন্ত অটোমেশন থামিয়ে রাখতে দেয়।
- **রিমাইন্ডার রেকর্ড** কোনো অটোমেশন আবার চেষ্টা করা হলে বা বারবার নির্ধারিত হলে ডুপ্লিকেট ক্যালেন্ডার বা নোটিফিকেশন রিমাইন্ডার ঠেকায়।
- **TextBee SMS সমর্থন** TextBee provider কনফিগার করা থাকলে টেক্সট-মেসেজ নোটিফিকেশন চালু করে।

সময়সূচি-সচেতন সহায়তা, অংশগ্রহণকারী ফলো-আপ, এবং এমন রিমাইন্ডার ওয়ার্কফ্লোর জন্য এই টুলগুলো ব্যবহার করুন, যেখানে সংবেদনশীল বার্তা পাঠানোর আগে এখনও মানব পর্যালোচনা প্রয়োজন।

## সাইট বিল্ডারের যোগাযোগ ফর্ম {#site-builder-contact-forms}

সাইট বিল্ডার agent এখন চ্যাট ইন্টারফেস থেকেই যোগাযোগ ফর্ম তৈরি করতে পারে। সাইট বিল্ডারকে শুধু একটি যোগাযোগ ফর্ম তৈরি করতে বলুন, এবং এটি স্বয়ংক্রিয়ভাবে আপনার ইনস্টল করা ফর্ম plugin (Contact Form 7, WPForms, Fluent Forms, বা Gravity Forms) শনাক্ত করে ব্যবহারের জন্য প্রস্তুত একটি ফর্ম তৈরি করবে।

বিস্তারিত নির্দেশনার জন্য [সাইট বিল্ডারের যোগাযোগ ফর্ম](../../user-guide/administration/site-builder-contact-forms) দেখুন।

## Theme Builder বৈশিষ্ট্য {#theme-builder-features}

Superdav AI Agent v1.16.0-এর Theme Builder-এ শক্তিশালী নতুন সক্ষমতা রয়েছে:

### ছবি আপলোডসহ ডিসকভারি সাক্ষাৎকার {#discovery-interview-with-photo-upload}

Theme Builder ডিসকভারি সাক্ষাৎকারে এখন একটি ছবি আপলোড ধাপ রয়েছে, যেখানে আপনি ভিজ্যুয়াল অনুপ্রেরণা দিতে পারেন। আপনার ব্র্যান্ড, প্রতিযোগী সাইট, বা ডিজাইন অনুপ্রেরণার ছবি আপলোড করুন, যাতে Theme Builder আপনার ডিজাইনের দিকনির্দেশনা বুঝতে পারে।

বিস্তারিত নির্দেশনার জন্য [ডিসকভারি সাক্ষাৎকার](./theme-builder/discovery-interview.md) দেখুন।

### ডেস্কটপ ও মোবাইল প্রিভিউসহ ডিজাইন দিকনির্দেশনা {#design-direction-with-desktop-and-mobile-previews}

Theme Builder আপনার সম্পূর্ণ theme তৈরি করার আগে ডেস্কটপ এবং মোবাইল—দুই ধরনের ডিভাইসেই আপনার ডিজাইন পর্যালোচনা করুন। একাধিক responsive breakpoint-এ রিয়েল-টাইম প্রিভিউ দেখুন।

বিস্তারিত নির্দেশনার জন্য [ডিজাইন দিকনির্দেশনা](./theme-builder/design-direction.md) দেখুন।

### আতিথেয়তা মেনু {#hospitality-menus}

Theme Builder এখন রেস্তোরাঁ, ক্যাফে, বার, এবং অন্যান্য আতিথেয়তা ব্যবসার জন্য কাঠামোবদ্ধ খাবার ও পানীয় মেনু পৃষ্ঠা তৈরি করতে পারে। মেনুতে মূল্য, বিবরণ, খাদ্যসংক্রান্ত তথ্য, এবং অ্যালার্জেন সতর্কতা থাকে।

বিস্তারিত নির্দেশনার জন্য [আতিথেয়তা মেনু](./theme-builder/hospitality-menus.md) দেখুন।

- **[Palette Contrast যাচাই করুন](./abilities/validate-palette-contrast.md)** — WCAG compliance-এর জন্য রঙের জোড়া পরীক্ষা করুন

## কনফিগারেশন {#configuration}

সব সেটিং WordPress admin-এ **Gratis AI Agent → Settings**-এর অধীনে রয়েছে। Superdav AI Agent নিচের কনফিগারেশন বিকল্প যোগ করে:

| সেটিং | বিবরণ |
|---|---|
| **Internet Search Provider** | Brave Search ও Tavily-এর মধ্যে বেছে নিন (ঐচ্ছিক) |
| **Tavily API Key** | Tavily ইন্টারনেট অনুসন্ধানের জন্য API key (Tavily নির্বাচন করা হলে) |
| **Managed Superdav Service** | স্বয়ংক্রিয় endpoint provisioning সহ সমর্থিত সাইটগুলোকে hosted Superdav service-এর সঙ্গে সংযুক্ত করুন |
| **Google Calendar Credentials** | ক্যালেন্ডার অ্যাক্সেস কনফিগার করুন, যাতে agent ক্যালেন্ডার ও ইভেন্ট পড়তে পারে |
| **TextBee SMS Provider** | অনুমোদিত SMS নোটিফিকেশন ওয়ার্কফ্লোর জন্য TextBee credentials কনফিগার করুন |

## উন্নত সহচর প্যাকেজ {#advanced-companion-package}

Superdav AI Agent v1.18.0 বিশ্বস্ত developer workflow-এর জন্য আলাদাভাবে বিতরণ করা একটি Advanced companion package যোগ করে। এই সক্ষমতাগুলো WordPress.org core package-এর সঙ্গে bundled নয়। শুধু সেই সাইটগুলোতে companion package ইনস্টল করুন, যেখানে administrators agent-কে developer tooling-এ access দিতে স্পষ্টভাবে বিশ্বাস করেন।

Advanced tools-এর মধ্যে filesystem operations, database access, WP-CLI, REST dispatcher workflows, plugin builder utilities, git snapshots, user-management helpers, এবং benchmark tooling রয়েছে। এই টুলগুলো বিশ্বস্ত administrators-এর মধ্যে সীমিত রাখুন এবং production site-এ চালু করার আগে approval-gate configuration পর্যালোচনা করুন।

- **Advanced companion distribution** — WordPress.org core build থেকে আলাদাভাবে trusted-developer tools প্রকাশ করুন
- **নির্ভরযোগ্যতা সংশোধন** — model selection, request timeout, default setup model handling, reasoning-text handling, retry guidance, calendar token hardening, reminder hardening, onboarding launches, confirmed ability resumes, এবং WordPress.org packaging review issues উন্নত করুন
