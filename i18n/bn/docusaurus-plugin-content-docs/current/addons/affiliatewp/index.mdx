# AffiliateWP {#affiliatewp}

Multisite Ultimate: AffiliateWP Integration

আপনার Ultimo Network-এর ক্লায়েন্ট বেস বাড়ানোর জন্য শক্তিশালী AffiliateWP ব্যবহার করুন!

## Installation {#installation}

১. 'ultimate-multisite-git ' ফাইলটি '/wp-content/plugins/' ডিরেক্টরিতে আপলোড করুন।
২. WordPress-এর 'Plugins' মেনু থেকে প্লাগইনটি অ্যাক্টিভেট করুন।
৩. প্লাগইনটি সেটআপ করার জন্য ধাপে ধাপে উইজার্ডটি অনুসরণ করুন।

## Usage Guide {#usage-guide}

আপনার অনলাইন ব্যবসার জন্য একটি অ্যাফিলিয়েট প্রোগ্রাম তৈরি করার সময় [AffiliateWP](https://affiliatewp.com/) একটি সেরা সমাধান হয়ে উঠেছে। Ultimate Multisite: AffiliateWP হলো সেই সেতু যা আপনাকে আপনার প্রিমিয়াম নেটওয়ার্কের জন্যও একই জিনিস তৈরি করতে সাহায্য করবে।

## Getting Started {#getting-started}

প্রথমত, আপনার **[AffiliateWP](https://affiliatewp.com/) সক্রিয় থাকা প্রয়োজন**! যদি আপনার কাছে **এই প্লাগইনটি না থাকে যা এর সাথে ইন্টিগ্রেট করে**, তবে আপনি আমাদের অ্যাড-অনটি ব্যবহার করতে পারবেন না। যদি আপনি আপনার ক্লায়েন্টদের তাদের সাইটগুলিতেও এটি ব্যবহার করতে না চান, তবে AffiliateWP শুধুমাত্র আপনার মূল সাইটে সক্রিয় থাকা উচিত।

এছাড়াও, আপনার **Ultimate Multisite** এবং **Ultimate Multisite: AffiliateWP** উভয়ই **network-wide সক্রিয়** থাকতে হবে।

## Adding Ultimate Multisite as an AffiliateWP Integration {#adding-ultimate-multisite-as-an-affiliatewp-integration}

আপনার মূল সাইটে, Affiliates Settings-এ যান এবং Integrations ট্যাবে Ultimate Multisite চেক করুন। এটি [AffiliateWP](https://affiliatewp.com/) কে Ultimate Multisite সাইন-আপ ফ্লো থেকে ভিজিট এবং কনভার্সন ট্র্যাক করা শুরু করতে বলবে।

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_নিচে Ultimate Multisite বক্সটি চেক করুন এবং সেভ করুন_

এটুকুই যথেষ্ট। রেফারেল লিঙ্ক থেকে আসা ভিজিট ট্র্যাক করা হবে এবং যখন সেই ব্যবহারকারী পরে তার প্রথম পেমেন্ট করবে, তখন কনভার্সন তৈরি হবে এবং কনভার্সনের জন্য দায়ী অ্যাফিলিয়েটকে কমিশন হিসেবে রাখা হবে।

তবে, শুধুমাত্র নতুন তৈরি হওয়া ব্যবহারকারীর কাছ থেকে প্রাপ্ত প্রথম পেমেন্টটিই ট্র্যাক করা হবে। রিকারিং রেফারেল পেমেন্ট সক্ষম করতে, আপনার [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) প্রয়োজন হবে।

## Enabling Recurring Referrals {#enabling-recurring-referrals}

[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) ইনস্টল করার পরে, Affiliates Settings-এর Recurring Referrals ট্যাবে যান এবং রিকারিং রেফারেল ব্যবহার সক্ষম করুন।
