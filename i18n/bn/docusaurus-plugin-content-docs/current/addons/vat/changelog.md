# VAT পরিবর্তন তালিকা {#vat-changelog}

Version 1.0.7 - প্রকাশের তারিখ 2026-02-03

* সংশোধন: জার্মান (DE) VAT নম্বরগুলো VIES SOAP সার্ভিস MS_MAX_CONCURRENT_REQ ত্রুটির কারণে যাচাইকরণে ব্যর্থ হচ্ছিল। আরও নির্ভরযোগ্য যাচাইকরণের জন্য লেগ্যাসি SOAP API থেকে অফিসিয়াল EU VIES REST API-তে পরিবর্তন করা হয়েছে।
* সংশোধন: এখন VIES সার্ভিস সাময়িকভাবে অনুপলব্ধ থাকলেও VAT নম্বর গ্রহণ করা হবে, ভুলভাবে প্রত্যাখ্যান করা হবে না। পরে পর্যালোচনার জন্য একটি লগ এন্ট্রি তৈরি করা হয়।
* সংশোধন: VAT/Tax ID ফিল্ডটি এখন সব দেশের জন্য দৃশ্যমান, শুধুমাত্র EU সদস্যদের জন্য নয়। নন-EU ট্যাক্স আইডি (যেমন সুইস CHE নম্বর) VIES যাচাইকরণ ছাড়াই ইনভয়েস প্রদর্শনের জন্য সংরক্ষণ করা হয়।
* সংশোধন: ভুল অ্যারে লুপআপ লজিকের কারণে গ্রীস (GR), মোনাকো (MC), এবং আইল অফ ম্যান (IM)-এর জন্য VAT প্রিফিক্স লুকআপ কাজ করছিল না।
* উন্নতি: অপ্রয়োজনীয় wp-cli-bundle dev নির্ভরতা সরিয়ে ফেলা হয়েছে, যা Site Exporter অ্যাডঅন-এর সাথে ব্যবহার করার সময় মারাত্মক ত্রুটি ঘটাতে পারত।
* উন্নতি: একটি ব্যাপক ইউনিট টেস্ট স্যুট (৫৩টি টেস্ট) যোগ করা হয়েছে।

Version 1.0.6 - প্রকাশের তারিখ 2026-01-25

* সংশোধন: এখন দেশীয় B2B লেনদেনে সঠিকভাবে VAT চার্জ করা হবে। EU VAT নিয়ম অনুযায়ী, রিভার্স চার্জ শুধুমাত্র আন্তঃসীমান্ত B2B লেনদেনের ক্ষেত্রে প্রযোজ্য, যখন গ্রাহকের দেশ কোম্পানির দেশের সাথে মেলে না।

Version 1.0.5 - প্রকাশের তারিখ 2026-01-22

* সংশোধন: অ্যাডঅন লোড হচ্ছে না বা সঠিকভাবে কাজ করছে না।
* পরিবর্তন: আরও নির্ভরযোগ্য এবং সক্রিয়ভাবে রক্ষণাবেক্ষণ করা ডেটার জন্য VAT রেট ডেটা সোর্স euvatrates.com থেকে ibericode/vat-rates রিপোজিটরিতে পরিবর্তন করা হয়েছে।
* সংশোধন: super_reduced_rates অপশন ভ্যালুর বানান ভুল সংশোধন করা হয়েছে।
* উন্নতি: ত্রুটি এড়াতে কান্ট্রি কোড হ্যান্ডলিং-এর জন্য নাল চেক যোগ করা হয়েছে।
* পরিবর্তন: Traduttore-এর মাধ্যমে স্বয়ংক্রিয় আপডেটের সুবিধার জন্য বান্ডেল করা অনুবাদ ফাইলগুলি সরিয়ে ফেলা হয়েছে।
