# Gratis AI Agent

Gratis AI Agent হলো একটি স্বয়ংক্রিয় (autonomous) WordPress AI সহকারী, যা স্বাভাবিক ভাষার নির্দেশাবলী ব্যবহার করে সম্পূর্ণ WordPress সাইট পরিকল্পনা করতে, তৈরি করতে এবং পরিচালনা করতে পারে। এটি **ক্ষমতা (abilities)**-এর একটি সুগঠিত সেট প্রকাশ করে — যা বিচ্ছিন্ন, পরীক্ষাযোগ্য অ্যাকশন যা এজেন্ট ব্যবহার করতে পারে — যা কন্টেন্ট ম্যানেজমেন্ট থেকে শুরু করে ডিজাইন সিস্টেম কাস্টমাইজেশন পর্যন্ত সবকিছু কভার করে। ভার্সন ১.৯.০-এ কন্টেন্ট তৈরির ক্ষমতা, ক্লায়েন্ট-সাইড স্ক্রিনশটগুলির মাধ্যমে ভিজ্যুয়াল পেজ রিভিউ, পাঁচটি বিল্ট-ইন এজেন্ট প্রোফাইল এবং অ্যাক্সেস কন্ট্রোল ও ব্র্যান্ডিংয়ের জন্য ফিচার ফ্ল্যাগ যুক্ত করা হয়েছে।

## মূল বৈশিষ্ট্য (Key Features) {#key-features}

- **Abilities system** — এটি হলো একটি মডুলার, সম্প্রসারণযোগ্য অ্যাকশন ক্যাটালগ যা আপনার WordPress ইনস্টলেশনে এজেন্ট ব্যবহার করতে পারে।
- **কন্টেন্ট ম্যানেজমেন্ট (Content Management)** — পোস্ট এবং পেজ তৈরি করা (পেজ টেমপ্লেট নির্ধারণ সহ), একবারে একাধিক পোস্ট তৈরি করা, ফিচার্ড ইমেজ সেট করা এবং কন্টাক্ট ফর্ম তৈরি করা।
- **ভিজ্যুয়াল রিভিউ (Visual Review)** — লাইভ পেজের স্ক্রিনশট ক্যাপচার করা, আগে/পরে (before/after) অবস্থার তুলনা করা এবং লেআউট, টাইপোগ্রাফি, রঙ এবং অ্যাক্সেসিবিলিটি কভার করে AI-জেনারেটেড ডিজাইন রিভিউ পাওয়া।
- **পাঁচটি বিল্ট-ইন এজেন্ট (Five built-in agents)** — Content Writer, Site Builder, Design Studio, Plugin Manager, এবং Support Assistant; চ্যাট হেডার-এ থাকা Agent Picker থেকে পরিবর্তন করা যায়।
- **ফিচার ফ্ল্যাগ (Feature flags)** — সেটিংস → Feature Flags-এ অ্যাক্সেস-কন্ট্রোল এবং ব্র্যান্ডিং টগল রয়েছে, যা রোল রেস্ট্রিকশন, হোয়াইট-লেবেল নাম এবং উইজেট কাস্টমাইজেশনের জন্য ব্যবহার করা যায়।
- **Custom Post Type ব্যবস্থাপনা** — কাস্টম পোস্ট টাইপ রেজিস্টার করা, তালিকাভুক্ত করা এবং মুছে ফেলা, যা রিস্টার্টের পরেও ডেটা ধরে রাখে (persistence)।
- **Custom Taxonomy ব্যবস্থাপনা** — কাস্টম ট্যাক্সোনমি রেজিস্টার করা, তালিকাভুক্ত করা এবং মুছে ফেলা, যা রিস্টার্টের পরেও ডেটা ধরে রাখে।
- **Design System abilities** — কাস্টম CSS ইনজেক্ট করা, কিউরেটেড ব্লক প্যাটার্ন পরিচালনা করা, সাইট লোগো সেট করা এবং theme.json প্রিসেট প্রয়োগ করা।
- **Global Styles ব্যবস্থাপনা** — কালার প্যালেট, টাইপোগ্রাফি, স্পেসিং ইত্যাদির জন্য theme.json মান পড়া এবং লেখা।
- **Navigation Menu ব্যবস্থাপনা** — নেভিগেশন মেনু এবং এর আইটেমগুলি প্রোগ্রাম্যাটিকভাবে তৈরি এবং আপডেট করা।
- **Options Management** — গুরুত্বপূর্ণ সেটিংস সুরক্ষিত রাখতে একটি বিল্ট-ইন সেফটি ব্লকলিস্ট সহ WordPress অপশন পড়া এবং লেখা।
- **Installable Abilities Registry** — কোর প্লাগইন ফাইল পরিবর্তন না করেই অতিরিক্ত ক্ষমতা প্যাক (ability packs) আবিষ্কার এবং সক্রিয় করা।
- **Site Builder Orchestration v2** — প্লাগইন আবিষ্কার, প্রোগ্রেস ট্র্যাকিং এবং এরর রিকভারি সহ বহু-ধাপের সাইট বিল্ড প্ল্যান।
- **WP-CLI benchmark command** — CI এবং মডেল মূল্যায়নের জন্য কমান্ড লাইন থেকে এজেন্ট ক্ষমতা বেঞ্চমার্ক স্যুট চালানো।

## প্রয়োজনীয়তা (Requirements) {#requirements}

- WordPress 6.0 বা তার বেশি
- PHP 8.1 বা তার বেশি
- WordPress Multisite (network-activated) অথবা single-site ইনস্টলেশন
- একটি সক্রিয় AI প্রোভাইডার কানেক্টর প্লাগইন (Anthropic, OpenAI, বা সামঞ্জস্যপূর্ণ)

## ইনস্টলেশন (Installation) {#installation}

1. প্লাগইন ফাইলগুলি `/wp-content/plugins/gratis-ai-agent`-এ আপলোড করুন।
2. WordPress-এর **Plugins** স্ক্রিনের মাধ্যমে প্লাগইনটি অ্যাক্টিভেট করুন।
3. **Gratis AI Agent → Settings** এ যান এবং আপনার AI প্রোভাইডার কানেক্ট করুন।
4. (ঐচ্ছিক) অতিরিক্ত ক্ষমতা প্যাক ইনস্টল করতে **Abilities Registry** ব্রাউজ করুন।

## দ্রুত শুরু (Quick Start) {#quick-start}

একবার অ্যাক্টিভেট এবং AI প্রোভাইডারের সাথে কানেক্ট করার পরে, WordPress অ্যাডমিন সাইডবারে **Gratis AI Agent** প্যানেলটি খুলুন। একটি স্বাভাবিক ভাষার নির্দেশাবলী টাইপ করুন এবং এজেন্টটি তার উপলব্ধ ক্ষমতাগুলি ব্যবহার করে প্রয়োজনীয় ধাপগুলি পরিকল্পনা ও কার্যকর করবে।

**উদাহরণ নির্দেশাবলী:**

```
Create a "Portfolio" custom post type with a "Project Category" taxonomy.
```

```
Inject a dark-mode colour palette into the theme and set the primary colour to #1a1a2e.
```

```
Build a restaurant website with an online menu, booking form, and contact page.
```

## ক্ষমতাগুলির ওভারভিউ (Abilities Overview) {#abilities-overview}

ক্ষমতাগুলি হলো সেই অ্যাটমিক অ্যাকশন যা এজেন্ট সম্পাদন করতে পারে। প্রতিটি ক্ষমতা একটি রেজিস্টার্ড PHP ক্লাস যা একটি স্কিমা (parameters এবং return types) প্রকাশ করে যা এজেন্ট রানটাইমে পড়ে। ক্ষমতাগুলিকে কার্যকরী ক্ষেত্রগুলিতে বিভক্ত করা হয়েছে:

| Area | Abilities |
|---|---|
| **কন্টেন্ট ম্যানেজমেন্ট** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **ভিজ্যুয়াল রিভিউ** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **Custom Post Types** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **Custom Taxonomies** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **Design System** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **Global Styles** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **Navigation Menus** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **Options Management** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **Installable Abilities** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **Site Builder** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

সম্পূর্ণ প্যারামিটার ডকুমেন্টেশন এবং ব্যবহারের উদাহরণগুলির জন্য [Abilities Reference](./abilities) দেখুন।

## WP-CLI {#wp-cli}

Gratis AI Agent কমান্ড লাইন থেকে বেঞ্চমার্ক চালানোর এবং ক্ষমতাগুলি পরিচালনা করার জন্য একটি `wp gratis-ai-agent` কমান্ড ফ্যামিলি নিয়ে আসে। সম্পূর্ণ কমান্ড তালিকা দেখতে [WP-CLI Reference](./wp-cli) দেখুন।

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## কনফিগারেশন (Configuration) {#configuration}

সমস্ত সেটিংস WordPress অ্যাডমিনে **Gratis AI Agent → Settings** এর অধীনে রয়েছে:

| Setting | Description |
|---|---|
| **AI Provider** | সক্রিয় AI প্রোভাইডার কানেক্টর নির্বাচন এবং কনফিগার করুন। |
| **Model** | এজেন্ট সেশনগুলির জন্য কোন ভাষা মডেল ব্যবহার করবেন তা বেছে নিন। |
| **Max Tokens** | প্রতি-অনুরোধ টোকেন বাজেট (ডিফল্ট ৪০৯৬)। |
| **Ability Timeout** | একটি একক ক্ষমতা সর্বাধিক কত সেকেন্ড চলতে পারে (ডিফল্ট ৩০)। |
| **Debug Logging** | ওয়ার্ডপ্রেস ডিবাগ লগে ক্ষমতা কল এবং এজেন্টের যুক্তি (reasoning) লগ করুন। |

## Options Management Safety Blocklist {#options-management-safety-blocklist}

Options Management ক্ষমতাটিতে একটি বিল্ট-ইন ব্লকলিস্ট রয়েছে যা এজেন্টকে সংবেদনশীল WordPress অপশন পড়তে বা লিখতে বাধা দেয়। ডিফল্ট ব্লকলিস্টটি কভার করে:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (URL অপশন — এগুলির জন্য একটি ডেডিকেটেড মাইগ্রেশন টুল ব্যবহার করুন)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (সিক্রেট কী)
- `active_plugins`, `template` (প্লাগইন/থিম অ্যাক্টিভেশন — ডেডিকেটেড ক্ষমতা ব্যবহার করুন)
- ইনস্টল করা প্লাগইন দ্বারা রেজিস্টার করা পেমেন্ট গেটওয়ে ক্রেডেনশিয়াল এবং API কী।

অতিরিক্ত এন্ট্রিগুলি `gratis_ai_agent_options_blocklist` ফিল্টারের মাধ্যমে যোগ করা যেতে পারে।

## Hooks and Filters {#hooks-and-filters}

### Actions {#actions}

- `gratis_ai_agent_ability_registered` — একটি ক্ষমতা রেজিস্টার হওয়ার পরে ট্রিগার হয়, ক্ষমতা ক্লাসের নাম গ্রহণ করে।
- `gratis_ai_agent_plan_started` — যখন একটি সাইট বিল্ডার প্ল্যান কার্যকর করা শুরু হয় তখন ট্রিগার হয়।
- `gratis_ai_agent_plan_completed` — যখন একটি প্ল্যান সফলভাবে শেষ হয় তখন ট্রিগার হয়।
- `gratis_ai_agent_plan_failed` — যখন একটি প্ল্যান পুনরুদ্ধার করা যায় না এমন ত্রুটির সম্মুখীন হয় তখন ট্রিগার হয়।

### Filters {#filters}

- `gratis_ai_agent_abilities` — এজেন্ট লোড করার আগে রেজিস্টার্ড ক্ষমতা তালিকা পরিবর্তন করুন।
- `gratis_ai_agent_options_blocklist` — অপশন সেফটি ব্লকলিস্ট বাড়ান বা কমান।
- `gratis_ai_agent_benchmark_questions` — মূল্যায়ন সেটে কাস্টম বেঞ্চমার্ক প্রশ্ন যোগ করুন।
- `gratis_ai_agent_plan_plugins` — সাইট প্ল্যান তৈরির সময় প্লাগইন আবিষ্কারের তালিকা পরিবর্তন করুন।
