# Mailchimp পরিবর্তন লগ

সংস্করণ: 1.1.1 - প্রকাশিত হয়েছে 2025-09-28

* প্রিফিক্স (prefix) ultimate-multisite এ পরিবর্তন করা হয়েছে; টেক্সট ডোমেইন আপডেট করা হয়েছে; এবং সংস্করণ আপডেট করা হয়েছে।

- যোগ করা হয়েছে: MailChimp গ্রুপগুলির মাধ্যমে একটি একক তালিকা (single list) এবং সেগমেন্ট ব্যবহার করার সুযোগ;
