# Mailchimp {#mailchimp}

Multisite Ultimate: Mailchimp ইন্টিগ্রেশন

আপনার Multisite Ultimate নেটওয়ার্ককে আপনার Mailchimp-এর সাথে সংযুক্ত করুন। আপনি নির্দিষ্ট প্ল্যানগুলিকে Mailchimp List-এর সাথে লিঙ্ক করতে পারেন এবং সাইনআপ করার সময় স্বয়ংক্রিয়ভাবে সাবস্ক্রাইবারদের সেই লিস্টগুলিতে যোগ করতে পারেন।

## ইনস্টলেশন {#installation}

১. 'ultimate-multisite-mailchimp' ফাইলটি '/wp-content/plugins/' ডিরেক্টরিতে আপলোড করুন।
২. আপনার WordPress Network Admin-এর 'Plugins' মেনু থেকে প্লাগইনটি অ্যাক্টিভেট করুন।

## ব্যবহারের নির্দেশিকা {#usage-guide}

আমাদের Ultimate Multisite: MailChimp Integration অ্যাড-অন ব্যবহার করে সাবস্ক্রাইবারদের ইমেলগুলিকে আপনার MailChimp লিস্টগুলির সাথে সিঙ্ক করা খুবই সহজ। এই টিউটোরিয়ালটি আপনাকে সবকিছু সেটআপ করতে সাহায্য করার জন্য তৈরি!

## একটি API key সংগ্রহ করা {#getting-an-api-key}

বিভিন্ন সেটিংস সম্পর্কে জানার আগে, আমাদের একটি MailChimp API key প্রয়োজন। এটি Ultimate Multisite-কে MailChimp API-এর সাথে কথা বলতে এবং ব্যবহারকারীরা সাইনআপ করার সময় লিস্ট, গ্রুপ পুনরুদ্ধার করতে এবং সেই লিস্ট ও গ্রুপগুলিতে ইমেল ঠিকানা যোগ করতে সাহায্য করবে।

আপনার নিজস্ব MailChimp API key পেতে এই টিউটোরিয়ালটি অনুসরণ করুন: [About API Keys | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

## ইন্টিগ্রেশন মোড {#integration-mode}

MailChimp Integration অ্যাড-অন দুটি "ইন্টিগ্রেশন মোড" অফার করে, যার মানে হলো ইমেল ঠিকানা সিঙ্ক করার জন্য আপনি অ্যাড-অনটি ব্যবহার করার দুটি ভিন্ন উপায় পাবেন।

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_বেশি নমনীয়তার জন্য দুটি ইন্টিগ্রেশন মোড!_

### মোড ১: Multiple Lists {#mode-1-multiple-lists}

ডিফল্ট মোডটি **Multiple Lists** ব্যবহার করে। এই মোডটি আপনাকে প্রতিটি প্ল্যানের জন্য একাধিক MailChimp লিস্ট নির্বাচন করার সুযোগ দেবে। আপনি যদি আপনার ব্যবহারকারীদের বিভিন্ন লিস্টে ভাগ (segment) করতে চান, তবে এটি সেরা বিকল্প।

উদাহরণস্বরূপ, ধরুন আপনার MailChimp অ্যাকাউন্টে ৩টি ভিন্ন লিস্ট আছে: **Newsletter**, **Plan A**, এবং **Plan B**। আপনি চান যে সমস্ত ব্যবহারকারী, তাদের প্ল্যান যাই হোক না কেন, তারা যেন Newsletter লিস্টেও অন্তর্ভুক্ত থাকে, যাতে তারা আপনার আপডেটগুলি মিস না করে।

এটি অর্জন করতে, আপনি Multiple Lists মোড ব্যবহার করে Plan A MailChimp Settings-এ **Newsletter** এবং **Plan A** উভয় লিস্ট যোগ করতে পারেন, এবং Plan B MailChimp Settings-এ **Newsletter** এবং **Plan B** যোগ করতে পারেন।

_নতুন অ্যাকাউন্ট এবং বাতিল হওয়া অ্যাকাউন্টের জন্য ডিফল্ট লিস্ট নির্বাচন করুন_

### মোড ২: Single List + Multiple Groups {#mode-2-single-list--multiple-groups}

MailChimp আপনাকে আপনার লিস্টটিকে **Groups** নামে পরিচিত ছোট ছোট ব্যবহারকারী গ্রুপে ভাগ করার সুযোগ দেয়। এটি আপনাকে আরও বেশি নিয়ন্ত্রণ দেয়, কারণ আপনি একাধিক ক্যাম্পেইনের একাধিক কপি তৈরি না করেও আপনার সাবস্ক্রাইবারদের একটি নির্দিষ্ট উপসেটকে ক্যাম্পেইন পাঠাতে পারেন (যা পরে সেই ক্যাম্পেইনের মেট্রিক্স বিশ্লেষণ করা কঠিন করে তোলে)।

উদাহরণস্বরূপ, আপনার একটি single list থাকতে পারে যার নাম **My Network Subscribers**, এবং এর ভেতরে **Plan A** এবং **Plan B** এর জন্য গ্রুপ থাকতে পারে।

## ডিফল্ট লিস্ট এবং গ্রুপ {#default-lists-and-groups}

আপনি কোন ইন্টিগ্রেশন মোডটি আপনার নির্দিষ্ট ব্যবহারের ক্ষেত্রে বেশি উপযুক্ত তা নির্বাচন করার পরে, ডিফল্ট লিস্ট (এবং মোড অনুযায়ী গ্রুপ) নির্বাচন করার সময়। কোনো নির্দিষ্ট প্ল্যানের জন্য কোনো লিস্ট নির্বাচন করা না থাকলে ডিফল্ট লিস্টটি ব্যবহার করা হবে।

আপনি বাতিল হওয়া অ্যাকাউন্টের জন্য ডিফল্ট লিস্টও নির্বাচন করতে পারেন। যখনই কোনো ব্যবহারকারী তার সাবস্ক্রিপশন বাতিল করে এবং আপনার নেটওয়ার্ক থেকে তার অ্যাকাউন্ট সরিয়ে নেয়, তখন তার ইমেল ঠিকানাটি বাতিল হওয়া অ্যাকাউন্টের জন্য সেট করা লিস্টে স্থানান্তরিত হবে।

## Double Subscription সক্ষম করা {#enabling-double-subscription}

একই সেটিংস পেজে, আপনি **Double Subscription** সক্ষম করতে পারবেন। এই অপশনটি ব্যবহারকারীকে ডিফল্ট নির্বাচিত লিস্ট এবং ব্যবহারকারীর প্ল্যানের জন্য নির্বাচিত উভয় লিস্টে যোগ করবে। যদি এই অপশনটি সক্ষম না করা থাকে, তবে ব্যবহারকারীকে শুধুমাত্র ব্যবহারকারীর প্ল্যানের জন্য নির্বাচিত লিস্টে যোগ করা হবে। যদি কোনো নির্দিষ্ট প্ল্যানের জন্য কিছুই সেট করা না থাকে, তবে ব্যবহারকারীকে ডিফল্ট নির্বাচিত লিস্টে যোগ করা হবে।

## প্রতিটি প্ল্যানের জন্য লিস্ট এবং গ্রুপ নির্বাচন করা {#selecting-lists-and-groups-for-each-of-the-plans}

আপনার **Plans** এ যান এবং যে প্ল্যানটি আপনি এডিট করতে চান তা নির্বাচন করুন। মনে রাখবেন যে **Advanced Options** মেটা-বক্সে, MailChimp নামে একটি নতুন ট্যাব উপলব্ধ থাকবে।

সেই ট্যাবে, আপনি সেই প্ল্যানের জন্য একাধিক লিস্ট বা একটি সিঙ্গেল লিস্ট এবং একাধিক গ্রুপ (ইন্টিগ্রেশন মোড অনুযায়ী) নির্বাচন করতে সক্ষম হবেন।

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_আপনার প্রতিটি প্ল্যানের জন্য আপনি যে লিস্ট এবং গ্রুপ ব্যবহার করতে চান তা নির্বাচন করুন_

যখন কোনো নতুন ব্যবহারকারী সেই প্ল্যানের জন্য সাইনআপ করবে, তখন তার ইমেল ঠিকানাটি নির্বাচিত লিস্ট এবং গ্রুপগুলিতে যোগ হয়ে যাবে!
