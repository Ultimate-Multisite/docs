# অ্যাকশন: wu_domain_seller_register_integrations {#action-wudomainsellerregisterintegrations}

|------|------|-------------|
| $registry | `\WP_Ultimo\Integrations\Integration_Registry` | ইন্টিগ্রেশন রেজিস্ট্রি। |
