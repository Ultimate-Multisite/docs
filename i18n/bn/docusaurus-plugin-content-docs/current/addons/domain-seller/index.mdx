# Domain Seller

**Multisite Ultimate: Domain Seller** আপনার WordPress Multisite নেটওয়ার্ককে শক্তিশালী ডোমেইন বিক্রির প্ল্যাটফর্মে রূপান্তর করে। গ্রাহকদের সাইন-আপের সময় বা তাদের dashboard থেকে সরাসরি কাস্টম ডোমেইন নিবন্ধন করতে দিন, সঙ্গে থাকছে রিয়েল-টাইম প্রাপ্যতা যাচাই এবং স্বচ্ছ মূল্য।

### কেন Domain Seller? {#why-domain-seller}

ডোমেইন নিবন্ধন যেকোনো WaaS (সেবা হিসেবে ওয়েবসাইট) প্ল্যাটফর্মের জন্য সবচেয়ে বেশি চাওয়া অতিরিক্ত বিক্রির সুযোগগুলোর একটি। আপনার গ্রাহকদের তৃতীয়-পক্ষের registrar-এ পাঠানোর বদলে আপনি সরাসরি ডোমেইন দিতে পারেন — এতে আপনার নেটওয়ার্কে বেশি আয় থাকে এবং গ্রাহকরা পান ঝামেলাহীন অভিজ্ঞতা।

### প্রধান বৈশিষ্ট্য {#key-features}

#### একাধিক ডোমেইন Registrar সমর্থন {#multiple-domain-registrar-support}
* **OpenSRS** — পূর্ণ নিবন্ধন, নবায়ন এবং ব্যবস্থাপনা
* **Namecheap** — WhoisGuard এবং sandbox পরীক্ষাসহ সম্পূর্ণ ইন্টিগ্রেশন
* **HostAfrica** — Domains Reseller API-এর মাধ্যমে ডোমেইন অনুসন্ধান, মূল্য, নিবন্ধন, স্থানান্তর, DNS এবং ID সুরক্ষা
* **Openprovider** — Reseller মূল্য, নিবন্ধন, নবায়ন, স্থানান্তর, DNS, WHOIS গোপনীয়তা এবং TLD sync
* **Hostinger** — শেয়ার করা Hostinger integration token ব্যবহার করে ডোমেইন অনুসন্ধান এবং নিবন্ধন
* **বর্ধনযোগ্য স্থাপত্য** — Integration Registry pattern-এর ওপর নির্মিত, তাই আরও provider যোগ করা সহজ
* **অন্তর্নির্মিত সংযোগ পরীক্ষা** — live করার আগে provider credentials এবং connectivity যাচাই করুন

#### নির্দেশিত Setup Wizard {#guided-setup-wizard}
* **এক-ক্লিক onboarding** — provider configuration, TLD import এবং ডোমেইন product তৈরি করার ধাপগুলোতে আপনাকে পথ দেখায়
* **স্বয়ংক্রিয় TLD sync** — আপনার সংযুক্ত provider-গুলো থেকে সব উপলভ্য TLD এবং wholesale pricing import করে
* **Default product তৈরি** — যুক্তিসংগত markup default-সহ ব্যবহারযোগ্য একটি ডোমেইন product তৈরি করে

#### নমনীয় ডোমেইন মূল্য {#flexible-domain-pricing}
* **তিনটি markup mode** — percentage markup, fixed markup, অথবা প্রতি TLD-তে fixed price
* **TLD-নির্দিষ্ট product** — বিভিন্ন TLD group-এর জন্য আলাদা ডোমেইন product তৈরি করুন (.com/.net, .io, country code ইত্যাদি)
* **প্রারম্ভিক মূল্য** — নতুন নিবন্ধন আকর্ষণ করতে প্রথম বছরের ছাড়যুক্ত মূল্য দিন
* **রিয়েল-টাইম মূল্য** — বুদ্ধিমান caching-সহ provider থেকে live wholesale pricing
* **Coupon সমর্থন** — Multisite Ultimate discount code-এর সঙ্গে পূর্ণ সামঞ্জস্য

#### WHOIS Privacy Protection {#whois-privacy-protection}
* **প্রতি-product configuration** — WHOIS privacy Disabled, Always Included, অথবা Customer Choice হিসেবে সেট করুন
* **গ্রাহকের opt-in** — Customer Choice সেট করা হলে checkout checkbox দিয়ে গ্রাহক privacy protection যোগ করতে পারেন
* **কনফিগারযোগ্য মূল্য** — privacy add-on-এর জন্য প্রতি বছরের মূল্য সেট করুন, অথবা বিনামূল্যে দিন
* **Registrar integration** — নিবন্ধনের সময় স্বয়ংক্রিয়ভাবে WhoisGuard (Namecheap) বা WHOIS privacy (OpenSRS) সক্রিয় করে

#### একীভূত Checkout অভিজ্ঞতা {#unified-checkout-experience}
* **তিনটি ডোমেইন mode** — গ্রাহকরা একটি subdomain (বিনামূল্যে), নতুন ডোমেইন নিবন্ধন, অথবা তাদের আগে থেকেই থাকা বিদ্যমান ডোমেইন mapping—এর মধ্যে বেছে নেন
* **রিয়েল-টাইম ডোমেইন অনুসন্ধান** — পেজ reload ছাড়াই AJAX-চালিত প্রাপ্যতা যাচাই
* **স্মার্ট পরামর্শ** — পছন্দের ডোমেইন নেওয়া থাকলে বিকল্প TLD পরামর্শ
* **স্বচ্ছ মূল্য** — নিবন্ধন মূল্য, নবায়ন মূল্য এবং ঐচ্ছিক WHOIS privacy fee পরিষ্কারভাবে দেখানো
* **Registrant contact field** — checkout-এর সময় inline-ভাবে প্রয়োজনীয় registrant তথ্য (নাম, ঠিকানা, ফোন) সংগ্রহ করে
* **স্বয়ংক্রিয়ভাবে তৈরি site URL** — নির্বাচিত ডোমেইন থেকে site address স্বয়ংক্রিয়ভাবে নির্ধারিত হয়

#### Auto-Renewal System {#auto-renewal-system}
* **Membership-সংযুক্ত নবায়ন** — গ্রাহকের membership সক্রিয় থাকা অবস্থায় ডোমেইন স্বয়ংক্রিয়ভাবে নবায়ন হয়
* **স্মার্ট scheduling** — ডোমেইনের মেয়াদোত্তীর্ণের তারিখ অনুযায়ী renewal job queue করা হয়
* **Retry logic** — ব্যর্থ renewal attempt-এর জন্য বুদ্ধিমান retry
* **Email notification** — আসন্ন নবায়ন, সফল নবায়ন, ব্যর্থতা এবং মেয়াদোত্তীর্ণের জন্য কনফিগারযোগ্য alert

#### গ্রাহকের DNS ব্যবস্থাপনা {#customer-dns-management}
* **Self-service DNS** — গ্রাহকরা তাদের dashboard থেকে DNS record (A, AAAA, CNAME, MX, TXT) যোগ, সম্পাদনা এবং মুছতে পারেন
* **ডোমেইন status** — রিয়েল-টাইম registration status, expiry date এবং nameserver তথ্য
* **স্বয়ংক্রিয় domain mapping** — নিবন্ধিত বা বিদ্যমান ডোমেইন গ্রাহকের site-এ স্বয়ংক্রিয়ভাবে map করা হয়

#### Admin Tools {#admin-tools}
* **ম্যানুয়াল ডোমেইন নিবন্ধন** — admin panel থেকে গ্রাহকের হয়ে ডোমেইন নিবন্ধন করুন
* **ডোমেইন monitoring** — এক পেজ থেকে সব গ্রাহক ডোমেইন, status এবং expiry date দেখুন
* **Provider-নির্দিষ্ট logging** — প্রতিটি registrar-এর জন্য আলাদা log channel (যেমন, domain-seller-namecheap.log)
* **নিবন্ধনের বিবরণ** — core domain edit page-এ registrant info এবং registration data দেখুন

### সমর্থিত Provider {#supported-providers}

* **OpenSRS** — নিবন্ধন, নবায়ন, DNS ব্যবস্থাপনা এবং WHOIS privacy-সহ পূর্ণ integration
* **Namecheap** — নিবন্ধন, নবায়ন, batch availability checking এবং বিনামূল্যের WhoisGuard-সহ পূর্ণ integration
* **HostAfrica** — নিবন্ধন, নবায়ন, স্থানান্তর, DNS ব্যবস্থাপনা, EPP code retrieval, registrar lock এবং ID protection
* **Openprovider** — নিবন্ধন, নবায়ন, স্থানান্তর, DNS ব্যবস্থাপনা, EPP code retrieval, registrar lock, WHOIS privacy এবং reseller TLD sync
* **Hostinger** — নিবন্ধন, domain lookup, nameserver update, registrar lock এবং WHOIS privacy; নবায়ন Hostinger auto-renew দ্বারা পরিচালিত হয়
* **GoDaddy** — নিবন্ধন এবং নবায়ন
* **ResellerClub** — নিবন্ধন, নবায়ন এবং DNS ব্যবস্থাপনা
* **NameSilo** — নিবন্ধন এবং নবায়ন
* **Enom** — নিবন্ধন, নবায়ন এবং DNS ব্যবস্থাপনা

### প্রযুক্তিগত বিবরণ {#technical-details}

* **প্রয়োজন:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **স্থাপত্য:** Domain_Selling_Capability interface-সহ Integration Registry pattern
* **Autoloading:** PSR-4 class mapping-সহ Jetpack autoloader
* **মান:** WordPress coding standards, strict typing, prepared statements

## ইনস্টলেশন {#installation}

1. নিশ্চিত করুন **Multisite Ultimate v2.4.12+** ইনস্টল করা এবং নেটওয়ার্ক-সক্রিয় করা আছে
2. plugin-টি `/wp-content/plugins/ultimate-multisite-domain-seller/`-এ আপলোড করুন
3. WordPress Network Admin-এর **Plugins** থেকে plugin-টি নেটওয়ার্ক-সক্রিয় করুন
4. প্রথম সক্রিয়করণে **সেটআপ উইজার্ড** স্বয়ংক্রিয়ভাবে চালু হয় — ধাপগুলো অনুসরণ করুন:
   * আপনার ডোমেইন রেজিস্ট্রার সংযুক্ত করুন
   * সংযোগ পরীক্ষা করুন
   * উপলভ্য TLD এবং পাইকারি মূল্য আমদানি করুন
   * মূল্য মার্কআপসহ আপনার প্রথম ডোমেইন product তৈরি করুন
5. **Ultimate Multisite > Checkout Forms**-এ আপনার checkout ফর্মে **ডোমেইন নির্বাচন** field যোগ করুন

## প্রায়শই জিজ্ঞাসিত প্রশ্ন {#frequently-asked-questions}

### Multisite Ultimate-এর কোন সংস্করণ প্রয়োজন? {#what-version-of-multisite-ultimate-is-required}

সংস্করণ 2.4.12 বা তার বেশি। addonটি WP Ultimo v1.x বা 2.4.12-এর নিচের Ultimate Multisite সংস্করণের সঙ্গে সামঞ্জস্যপূর্ণ নয়।

### কোন ডোমেইন provider সমর্থিত? {#which-domain-providers-are-supported}

নয়টি রেজিস্ট্রার সমর্থিত: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo**, এবং **Enom**। রেজিস্ট্রার API ভিন্ন ভিন্ন কাজ প্রকাশ করায় সক্ষমতা provider অনুযায়ী আলাদা; provider-ভিত্তিক সারসংক্ষেপের জন্য সেটআপ গাইড দেখুন। addon's Integration Registry স্থাপত্য custom provider যোগ করা সহজ করে — [developer guide](/developer/integration-guide/custom-registrar) দেখুন।

### আমি কি WHOIS গোপনীয়তা সুরক্ষা দিতে পারি? {#can-i-offer-whois-privacy-protection}

হ্যাঁ। প্রতিটি ডোমেইন product-এ তিনটি মোডসহ WHOIS Privacy সেটিং আছে: নিষ্ক্রিয়, সবসময় অন্তর্ভুক্ত, অথবা গ্রাহকের পছন্দ। Customer Choice সেট করা হলে, গ্রাহকেরা checkout-এর সময় দামের সঙ্গে একটি checkbox দেখেন (অথবা Namecheap-এর WhoisGuard-এর জন্য "Free")। registration সময় privacy flag স্বয়ংক্রিয়ভাবে রেজিস্ট্রারের কাছে পাঠানো হয়।

### আমি কি আলাদা TLD-এর জন্য আলাদা মূল্য নির্ধারণ করতে পারি? {#can-i-set-different-pricing-for-different-tlds}

হ্যাঁ। আলাদা TLD গ্রুপের জন্য পৃথক ডোমেইন product তৈরি করুন। প্রতিটি product-এর নিজস্ব markup type (শতাংশ, নির্দিষ্ট মার্কআপ, অথবা নির্দিষ্ট মূল্য), সমর্থিত TLD, এবং প্রারম্ভিক মূল্য থাকতে পারে।

### checkout-এ গ্রাহকেরা কী কী option দেখেন? {#what-checkout-options-do-customers-see}

Domain Selection field গ্রাহকদের তিনটি tab দেয়: **সাবডোমেইন** (আপনার নেটওয়ার্কে একটি বিনামূল্যের subdomain ব্যবহার করুন), **নতুন ডোমেইন নিবন্ধন** (নতুন ডোমেইন খুঁজে নিবন্ধন করুন), এবং **বিদ্যমান ডোমেইন** (তারা আগে থেকেই মালিক এমন একটি ডোমেইন ম্যাপ করুন)। আপনি যে মোডগুলো চালু করবেন, শুধু সেগুলোই দেখা যাবে।

### এটি কি স্বয়ংক্রিয় renewal সমর্থন করে? {#does-it-support-automatic-renewals}

হ্যাঁ। ডোমেইনগুলো auto-renew-এ সেট করা যায় এবং গ্রাহকের membership অবস্থার সঙ্গে যুক্ত থাকে। renewal পেমেন্ট গ্রাহকের বিদ্যমান payment gateway-এর মাধ্যমে প্রক্রিয়াকৃত হয়।

### গ্রাহকেরা কি তাদের DNS রেকর্ড পরিচালনা করতে পারেন? {#can-customers-manage-their-dns-records}

হ্যাঁ। গ্রাহকেরা তাদের dashboard-এ DNS পরিচালনার interface পান, যেখানে A, AAAA, CNAME, MX, এবং TXT রেকর্ড যোগ, সম্পাদনা, ও মুছে ফেলা যায়। DNS management OpenSRS, ResellerClub, Enom, HostAfrica, এবং Openprovider-এর সঙ্গে উপলভ্য। hosted ডোমেইনের জন্য Hostinger DNS core Hostinger domain-mapping integration দ্বারা পরিচালিত হয়।

### সেটআপ উইজার্ড আছে কি? {#is-there-a-setup-wizard}

হ্যাঁ। setup wizard প্রথম সক্রিয়করণে স্বয়ংক্রিয়ভাবে চলে এবং আপনাকে রেজিস্ট্রার সংযুক্ত করা, TLD আমদানি করা, এবং আপনার প্রথম ডোমেইন product তৈরির ধাপগুলো দেখায়। আপনি addon settings থেকেও wizard আবার চালাতে পারেন।
