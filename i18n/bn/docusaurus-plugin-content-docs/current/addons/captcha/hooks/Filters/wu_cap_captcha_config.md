# Filter: wu_cap_captcha_config {#filter-wucapcaptchaconfig}

ক্যাপ ক্যাপচা কনফিগারেশন ফিল্টার করুন।

এটি ডেভেলপারদের ক্যাপ ক্যাপচা সেটিংস কাস্টমাইজ করতে সাহায্য করে।

## প্যারামিটারসমূহ {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $config | `array` | ক্যাপ কনফিগারেশন অ্যারে। |
| $level | `string` | নিরাপত্তা স্তর (fast, medium, max)। |
