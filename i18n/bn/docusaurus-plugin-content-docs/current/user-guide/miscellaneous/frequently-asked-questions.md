# সচরাচর জিজ্ঞাসা

এখানে কিছু **সচরাচর জিজ্ঞাসা** দেওয়া হলো যেগুলো সাপোর্ট রিকোয়েস্টে প্রায়ই উঠে আসে।

  * **Ultimate Multisite কী?**

Ultimate Multisite একটি WordPress plugin, যা WordPress multisite install-এর জন্য তৈরি। এটি আপনার WordPress install-কে একটি প্রিমিয়াম সাইট নেটওয়ার্কে রূপান্তরিত করে – অনেকটা [WordPress.com](https://WordPress.com)-এর মতো – যেখানে ক্লায়েন্টরা মাসিক, ত্রৈমাসিক বা বার্ষিক ফি-এর বিনিময়ে সাইট তৈরি করতে পারেন (আপনি বিনামূল্যে প্ল্যানও তৈরি করতে পারেন)।

  * **Ultimate Multisite কি Multisite ছাড়া কাজ করবে?**

না। Ultimate Multisite-এর জন্য WordPress multisite, subdirectory বা subdomain প্রয়োজন।

  * **Ultimate Multisite-এর জন্য কী কী প্রয়োজন?**

Ultimate Multisite-এর জন্য প্রয়োজন:

**WordPress সংস্করণ**: v5.3+ (সুপারিশকৃত: সর্বশেষ স্থিতিশীল সংস্করণ)  
**PHP সংস্করণ**: 7.4.x (> 8.0 সাপোর্ট শীঘ্রই আসছে)  
**MySQL সংস্করণ**: v5+ (সুপারিশকৃত: সংস্করণ 5.6, যদি আপনার hosting provider-এ 8.0 না থাকে)

_Multisite (subdomain বা subdirectory) অবশ্যই সক্রিয় থাকতে হবে।_ [কীভাবে WordPress Multisite ইনস্টল করবেন](/user-guide/getting-started/how-to-install-wordpress-multisite)।

  * **Ultimate Multisite কি Subdirectory-সহ Multisite-এ কাজ করবে?**

হ্যাঁ। Ultimate Multisite subdirectory install এবং subdirectory install উভয়ই সাপোর্ট করে।

  * **Ultimate Multisite কি Subdomain-সহ Multisite-এ কাজ করবে?**

হ্যাঁ। Ultimate Multisite subdirectory install এবং subdomain install উভয়ই সাপোর্ট করে।

  * **Ultimate Multisite দিয়ে কি ব্যবহারকারীরা কোন Plugin ও Theme ব্যবহার করতে পারবে তা সীমাবদ্ধ করা যায়?**

হ্যাঁ। Ultimate Multisite দিয়ে আপনি আপনার নেটওয়ার্কের প্রতিটি প্ল্যানে কোন plugin ও theme পাওয়া যাবে তা নির্বাচন করতে পারেন।

  * **Ultimate Multisite দিয়ে কি কোনো ব্যবহারকারী তার সাইটে কতগুলো পোস্ট তৈরি করতে পারবে তা সীমাবদ্ধ করা যায়?**

হ্যাঁ। Ultimate Multisite দিয়ে আপনি আপনার নেটওয়ার্কের **প্রতিটি উপলব্ধ post type-এর** জন্য পোস্ট কোটা নির্ধারণ করতে পারেন। আপনি অ্যাডমিন কতজন ব্যবহারকারীকে আমন্ত্রণ জানাতে পারবে, **ডিস্ক স্পেসের পরিমাণ** এবং আরও অনেক কিছু সীমিত করতে পারেন।

  * **Ultimate Multisite কি Site Template সাপোর্ট করে?**

হ্যাঁ। Ultimate Multisite দিয়ে আপনি যত খুশি Site Template তৈরি করতে পারেন। Site Template কোনো theme নয়, এগুলো আপনার নেটওয়ার্কে সাধারণ সাইট। আপনি plugin, theme সক্রিয় করতে পারেন এবং সেই Site Template যেভাবে খুশি কাস্টমাইজ করতে পারেন। কোনো ব্যবহারকারী সেই template নির্বাচন করলে, তার সাইট সেই template-এর কপি হিসেবে তৈরি হবে, সমস্ত কাস্টমাইজেশনসহ।

আপনি কোনো নির্দিষ্ট প্ল্যানের জন্য একটি বিশেষ template সেট করতে পারেন। এটি করলে, সেই প্ল্যানে সাইন আপ করা প্রতিটি ব্যবহারকারীর সাইট সেই Site Template-এর ডুপ্লিকেট হিসেবে তৈরি হবে।

  * **Ultimate Multisite কি WooCommerce-এর সাথে ইন্টিগ্রেট করে?**

হ্যাঁ!

আপনি যদি চান আপনার নেটওয়ার্ক ব্যবহারকারীদের জন্য WooCommerce উপলব্ধ থাকুক, যাতে তারা স্টোর সেট আপ করতে পারে, তাহলে Ultimate Multisite ব্যবহার করে এটি সম্ভব। এমনকি আপনি প্রতিটি প্ল্যান টায়ারে আপনার ক্লায়েন্টরা কতগুলো পণ্য তৈরি করতে পারবে তাও সীমিত করতে পারেন।

Ultimate Multisite-এর সাবস্ক্রিপশন পেমেন্টের ক্ষেত্রে, আপনি [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) ব্যবহার করে আপনার Ultimate Multisite ক্লায়েন্টদের মূল সাইটে WooCommerce gateway ব্যবহার করে পেমেন্ট করার সুযোগ দিতে পারেন। এভাবে, WooCommerce-এর জন্য তৈরি বিশাল payment gateway লাইব্রেরি ব্যবহার করে আপনি আপনার Ultimate Multisite ক্লায়েন্টদের বিল করতে পারবেন।

  * **WPMUDEV Pro Sites থেকে সাবস্ক্রিপশন মাইগ্রেট করার কোনো উপায় আছে কি?**

হ্যাঁ, Ultimate Multisite এখন আপনাকে [মাইগ্রেট করতে সাহায্য করার জন্য একটি টুল](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) দিচ্ছে আপনার Pro Sites নেটওয়ার্ক মাইগ্রেট করতে।

  * **কেন আমার Plan Edit Page-এ "Allowed Plugins" ট্যাব খালি?**

Ultimate Multisite শুধুমাত্র সেই plugin-গুলোতে অ্যাক্সেস দেওয়া বা বন্ধ করতে দেয় যেগুলো আপনার নেটওয়ার্কে network-activated নয়। যখন কোনো plugin network-activated হয়, স্বাভাবিকভাবেই এটি আপনার নেটওয়ার্কের প্রতিটি সাইটে ডিফল্টভাবে লোড হবে।

"Allowed Plugins" ট্যাবে কোনো plugin না দেখা গেলে, দেখুন পছন্দের plugin টি network-active কিনা।

  * **আমি কি নিজের Pricing Table ডিজাইন ব্যবহার করতে পারি?**

হ্যাঁ। সংস্করণ 1.4.0 থেকে, Ultimate Multisite template override সাপোর্ট করে।

  * **Ultimate Multisite কি WP Engine-এ কাজ করে?**

হ্যাঁ। Ultimate Multisite সবসময় WP Engine-এ সাপোর্টেড ছিল, তবে domain mapping কাজ করাতে অ্যাডমিনের কাছ থেকে ম্যানুয়াল অতিরিক্ত পদক্ষেপ নিতে হতো (WP Engine প্যানেলে domain যোগ করা)। **সংস্করণ 1.6.0 থেকে এটি আর প্রয়োজন নেই।** Ultimate Multisite সরাসরি WP Engine API-এর সাথে ইন্টিগ্রেট করে এবং mapping-এর পরে domain স্বয়ংক্রিয়ভাবে প্যানেলে যোগ হয়ে যায়।

  * **আপনারা কোন hosting provider সুপারিশ করেন?**

Ultimate Multisite যেকোনো hosting provider-এ কাজ করা উচিত। শুধু পার্থক্য হলো, কিছু hosting company যেমন Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane এবং [Runcloud.io](http://Runcloud.io)-এ domain mapping ক্ষমতার জন্য আরও ঘনিষ্ঠ ইন্টিগ্রেশন আছে, তাই এগুলো সরাসরি কাজ করে। অন্যান্য hosting-এ আপনাকে ম্যানুয়ালি domain mapping সেট আপ করতে হতে পারে।
