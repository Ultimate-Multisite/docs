# PRO Theme দিয়ে আপনার অ্যাডমিন প্যানেল কাস্টমাইজ করা

ক্লায়েন্টের কাজ করার সময় প্রায়ই আমাদের WordPress অ্যাডমিন ড্যাশবোর্ডে তাদের জন্য একটি কাস্টম অভিজ্ঞতা তৈরি করার প্রয়োজন হয়। আপনি যদি আরও প্রফেশনাল চেহারা এবং অনুভূতি খুঁজছেন, তাহলে PRO Theme আপনার কাজে আসতে পারে।

PRO Theme হলো একটি অ্যাডমিন থিম যা আপনার ক্লায়েন্টদের Dashboard কাস্টমাইজ করতে সাহায্য করার জন্য তৈরি করা হয়েছে। এটি Ultimate Multisite-এর প্রিমিয়াম প্লাগইন কোরের একটি অংশও।

এটি ইনস্টল করার পর, আপনার _Plugins_ পেজে যান এবং আপনার Network-এ এটি সক্রিয় করতে ক্লিক করুন।

_**নোট:**_ _আপনি যদি শুধু এটি ইনস্টল করেন, তাহলে এটি আপনার ক্লায়েন্টদের কাছে একটি অপশন হিসেবে দেখাবে যেটি মার্ক করে শুধুমাত্র সেই ব্যবহারকারীর জন্য থিমটি যোগ করা যাবে।_

এখন, আপনার অ্যাডমিন ইন্টারফেস সম্পূর্ণ নতুনভাবে ডিজাইন করা হয়েছে! বাম কলামে PRO Theme নামে একটি নতুন মেনুও পাবেন।
