# হোস্টিংার মাল্টি-টেন্যান্সি {#hostinger-multi-tenancy}

Ultimate Multisite: Multi-Tenancy 1.2.0 হোস্টিংার মাল্টি-টেন্যান্সি সক্ষমতা যোগ করেছে, যার ফলে বিদ্যমান হোস্টিংার ডোমেইন-ম্যাপিং ইন্টিগ্রেশনের পাশাপাশি হোস্ট করা ডোমেইনগুলো সার্বভৌম টেন্যান্ট প্রোভিশনিং-এ অংশ নিতে পারে।

যখন টেন্যান্ট ডোমেইন এবং বিচ্ছিন্ন টেন্যান্ট অবকাঠামো হোস্টিংার hPanel এর মাধ্যমে পরিচালিত হয়, তখন এই সক্ষমতা ব্যবহার করুন।

## সেটআপ নোটস {#setup-notes}

১. **Ultimate Multisite > Settings > Host Integrations** এর অধীনে কোর হোস্টিংার ইন্টিগ্রেশন কনফিগার করুন।
২. নিশ্চিত করুন যে হোস্টিংার API টোকেন লক্ষ্য করা ডোমেইন বা সাবডোমেইন পরিচালনা করতে পারে।
৩. মাল্টি-টেন্যান্সি অ্যাড-অন সক্ষম করুন।
৪. টেন্যান্ট প্রকাশ করার আগে টেন্যান্ট আইসোলেশন কৌশল কনফিগার করুন।
৫. টেন্যান্টে প্রোডাকশন ট্র্যাফিক পাঠানোর আগে মাইগ্রেশন ভেরিফিকেশন ওয়ার্কফ্লো চালান।

হোস্টিংার সক্ষমতা হোস্ট-সাইড অপারেশনের জন্য শেয়ার করা হোস্টিংার সংযোগ ব্যবহার করে। ডিএনএস এখনও সঠিক হোস্টিংার অ্যাকাউন্টে নির্দেশিত হতে হবে, এবং hPanel অ্যাকাউন্টের সীমা প্রযোজ্য থাকবে।

## সক্ষমতা-নির্দিষ্ট পরিবর্তন {#capability-specific-changes}

- হোস্ট-সচেতন ডোমেইন অপারেশনের মাধ্যমে সার্বভৌম টেন্যান্ট তৈরি করা যেতে পারে।
- যাচাইকরণের আগে একই মেশিনের ডেটাবেস হোস্ট স্ট্রিংগুলো স্বাভাবিক (normalize) করা হয়।
- WordPress রানটাইম যদি স্থানীয় ওভাররাইড না চায়, তবে হোস্টিংার দ্বারা পরিচালিত টেন্যান্টকে hPanel এ দেখানো ডেটাবেস হোস্ট ভ্যালু ব্যবহার করতে হবে।
- SSO ভিজিট নির্ভর করে টেন্যান্ট ডোমেইনটি হোস্টিংার-হোস্টেড টেন্যান্টে রেজোলিউশন করার ওপর।

## হোস্টিংার টেন্যান্ট ট্রাবলশুটিং {#troubleshooting-hostinger-tenants}

- যদি কোনো টেন্যান্ট ইনস্টলেশন ব্যর্থ হয়, তবে যাচাই করুন যে ডোমেইনটি ইতিমধ্যেই হোস্টিংার অ্যাকাউন্টে সংযুক্ত আছে কিনা।
- যদি ডেটাবেস ভেরিফিকেশন ব্যর্থ হয়, তবে hPanel এর সাথে টেন্যান্টের DB ইউজারনেম, ডেটাবেস নাম এবং হোস্ট বাইন্ডিং তুলনা করুন।
- যদি **Visit (SSO)** ব্যর্থ হয়, তবে টেন্যান্ট ডোমেইনের জন্য ডিএনএস এবং SSL সক্রিয় আছে কিনা তা নিশ্চিত করুন।
- যদি টর-ডাউন করার পরে হোস্ট রিসোর্স পিছনে পড়ে থাকে, তবে ব্যাকআপ নিশ্চিত করার পরে hPanel থেকে যেকোনো অবশিষ্ট ডেটাবেস, ইউজার বা ফোল্ডার সরিয়ে ফেলুন।
