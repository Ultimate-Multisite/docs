# Ultimate Multisite: Ad Injector দিয়ে আপনার নেটওয়ার্কে বিজ্ঞাপন বসানো

Ultimate Multisite: Ad Injector হলো একটি **ফ্রি অ্যাড-অন** যা দিয়ে আপনি প্ল্যানের এডিট পেজ থেকেই সরাসরি বিজ্ঞাপন বসাতে পারবেন।

Ultimate Multisite: Ad Injector ইনস্টল করার পর, এটি অ্যাক্টিভ করতে আপনার _Plugins_ পেজে যান।

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Network Activate বাটনে ক্লিক করুন_

অ্যাড-অনটি অ্যাক্টিভ হয়ে গেলে, প্ল্যান এডিট স্ক্রিনের _Advanced Options_ মেটা বক্সে _Ad Settings_ নামে একটি নতুন অপশন দেখতে পাবেন।

![প্ল্যান এডিট স্ক্রিনে Ad Settings](/img/config/product-edit-full.png)

_এখন আপনি প্রতিটি প্ল্যানের জন্য আলাদাভাবে বিজ্ঞাপন সেটিংস এডিট করতে পারবেন_

এই ট্যাবে আপনি ফ্রন্ট-এন্ডে পোস্ট ও পেজের জন্য কনটেন্টের আগে এবং পরে বিজ্ঞাপনের কোড যোগ করতে পারবেন।
