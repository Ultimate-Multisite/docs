# সাইট টেমপ্লেটে Auto Search and Replace (v2)

_**এই টিউটোরিয়ালের জন্য WP UItimo সংস্করণ 2.x প্রয়োজন।**_

Ultimate Multisite-এর সবচেয়ে শক্তিশালী বৈশিষ্ট্যগুলোর একটি হলো নিবন্ধন ফর্মে ইচ্ছামতো টেক্সট, রং এবং নির্বাচনী ফিল্ড যোগ করার সুবিধা। ডেটা সংগ্রহ করার পর, নির্বাচিত সাইট টেমপ্লেটের নির্দিষ্ট অংশে কনটেন্ট আগে থেকেই পূরণ করতে আমরা সেটি ব্যবহার করতে পারি। এরপর নতুন সাইট প্রকাশিত হলে, Ultimate Multisite নিবন্ধনের সময় দেওয়া আসল তথ্য দিয়ে placeholder-গুলো বদলে দেবে।

উদাহরণস্বরূপ, আপনি placeholder দিয়ে আপনার টেমপ্লেট সাইট তৈরি করতে পারেন। Placeholder-গুলো ডাবল কার্লি ব্রেসের মধ্যে যোগ করতে হবে - {{placeholder_name}}।

এরপর সেই ডেটা সংগ্রহ করতে আপনি সহজেই একটি মিল থাকা নিবন্ধন ফিল্ড যোগ করতে পারেন

আপনার গ্রাহক তখন নিবন্ধনের সময় সেই ফিল্ড পূরণ করতে পারবেন।

এরপর Ultimate Multisite গ্রাহকের দেওয়া ডেটা দিয়ে placeholder-গুলো স্বয়ংক্রিয়ভাবে বদলে দেবে।

## **"placeholder-এ ভরা টেমপ্লেট" সমস্যার সমাধান** {#solving-the-template-full-of-placeholders-problem}

এসবই দারুণ, কিন্তু একটি অস্বস্তিকর সমস্যা দেখা দেয়: এখন আমাদের সাইট টেমপ্লেটগুলো - যেগুলো আমাদের গ্রাহকেরা দেখতে পারেন - এমন placeholder-এ ভরা থাকে, যেগুলো দেখতে খারাপ এবং খুব বেশি কিছু বোঝায় না।

এটি সমাধানের জন্য, আমরা placeholder-গুলোর জন্য ভুয়া মান সেট করার বিকল্প দিই। আপনার গ্রাহকেরা টেমপ্লেট সাইট ভিজিট করার সময় আমরা সেই মানগুলো ব্যবহার করে তাদের কনটেন্ট খুঁজে বদলে দিই।

Template placeholders editor-এ যেতে **Ultimate Multisite > Settings > Sites**-এ যান, Site Template Options এলাকায় স্ক্রল করুন, তারপর **Edit Placeholders** লিংকে ক্লিক করুন।

![Sites সেটিংস পেজে Site Template Options এলাকা](/img/config/settings-sites-templates-section.png)

এটি আপনাকে placeholder-গুলোর কনটেন্ট editor-এ নিয়ে যাবে, যেখানে আপনি placeholder এবং তাদের সংশ্লিষ্ট কনটেন্ট যোগ করতে পারবেন।

![Template placeholders editor-এর প্রবেশপথ](/img/config/settings-sites-templates-section.png)
