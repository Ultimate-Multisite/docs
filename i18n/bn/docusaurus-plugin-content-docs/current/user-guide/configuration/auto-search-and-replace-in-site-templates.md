# সাইট টেমপ্লেটে স্বয়ংক্রিয় সার্চ এবং রিপ্লেস (v2)

_**এই টিউটোরিয়ালের জন্য WP UItimo version 2.x প্রয়োজন।**_

Ultimate Multisite-এর সবচেয়ে শক্তিশালী ফিচারগুলোর মধ্যে একটি হলো রেজিস্ট্রেশন ফর্মে যেকোনো টেক্সট, কালার এবং সিলেক্ট ফিল্ড যোগ করার সুবিধা। এই ডেটা সংগ্রহ করার পর, আমরা সেটা দিয়ে নির্বাচিত সাইট টেমপ্লেটের নির্দিষ্ট অংশে কন্টেন্ট আগে থেকে বসিয়ে রাখতে পারি। তারপর, নতুন সাইট পাবলিশ হলে Ultimate Multisite স্বয়ংক্রিয়ভাবে placeholder-গুলো রেজিস্ট্রেশনের সময় দেওয়া আসল তথ্য দিয়ে বদলে দেবে।

উদাহরণস্বরূপ, আপনি আপনার টেমপ্লেট সাইটগুলো placeholder দিয়ে তৈরি করতে পারেন। Placeholder-গুলো ডাবল কার্লি ব্রেস দিয়ে ঘেরা থাকা উচিত - {{placeholder_name}}।

এরপর, সেই ডেটা সংগ্রহের জন্য একটি মিলানো রেজিস্ট্রেশন ফিল্ড যোগ করুন

আপনার গ্রাহক তখন রেজিস্ট্রেশনের সময় সেই ফিল্ড পূরণ করতে পারবেন।

Ultimate Multisite এরপর স্বয়ংক্রিয়ভাবে গ্রাহকের দেওয়া ডেটা দিয়ে placeholder-গুলো বদলে দেবে।

## **"placeholder ভরা টেমপ্লেট" সমস্যার সমাধান**

এই সব দারুণ, কিন্তু একটা বিরক্তিকর সমস্যা দেখা দেয়: এখন আমাদের সাইট টেমপ্লেটগুলো - যেগুলো গ্রাহকরা দেখতে পারেন - কুৎসিত placeholder দিয়ে ভর্তি যা তেমন কিছু বোঝায় না।

এই সমস্যা সমাধানের জন্য, আমরা placeholder-গুলোর জন্য নকল মান সেট করার অপশন দিই, এবং আপনার গ্রাহকরা টেমপ্লেট সাইট ভিজিট করার সময় সেই মানগুলো দিয়ে কন্টেন্ট search এবং replace করি।

টেমপ্লেট placeholder এডিটরে যেতে **Ultimate Multisite > Settings > Sites**-এ যান, তারপর সাইডবারে Edit Placeholders লিংকে ক্লিক করুন।

![Sites settings সাইডবারে Edit Placeholders লিংক](/img/config/settings-sites.png)

এটা আপনাকে placeholder কন্টেন্ট এডিটরে নিয়ে যাবে, যেখানে আপনি placeholder এবং তাদের সংশ্লিষ্ট কন্টেন্ট যোগ করতে পারবেন।

![Placeholder-এর নাম ও মান সহ Placeholder কন্টেন্ট এডিটর](/img/config/settings-sites.png)
