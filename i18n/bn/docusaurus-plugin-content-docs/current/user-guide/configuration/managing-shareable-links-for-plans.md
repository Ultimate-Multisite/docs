# প্ল্যানের জন্য শেয়ারযোগ্য লিংক পরিচালনা

Ultimate Multisite আপনাকে সরাসরি ক্লায়েন্টদের সাথে প্ল্যান লিংক শেয়ার করার সুবিধা দেয়। এটি করতে, আপনার যেকোনো প্ল্যানের এডিট পেজে যান এবং _Click to copy the Shareable Link_ বাটনে ক্লিক করুন।

![শেয়ারযোগ্য লিংক বাটন সহ প্রোডাক্ট এডিট পেজ](/img/config/product-edit-full.png)

_প্ল্যানের এডিট পেজে শেয়ারযোগ্য লিংক তৈরির বাটন পাবেন_

মনে রাখবেন, এই পদ্ধতি লিংক তৈরিতে ডিফল্ট বিলিং ফ্রিকোয়েন্সি ব্যবহার করে। অর্থাৎ, যদি আপনার ডিফল্ট বিলিং মাসিক হয়, তাহলে শেষ পর্যন্ত আপনি এরকম একটি লিংক পাবেন: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

আপনি যদি চান শেয়ারযোগ্য লিংকটি ডিফল্ট হিসেবে ত্রৈমাসিক দেখাক, তাহলে শেষে 3 যোগ করতে হবে

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

বার্ষিক সময়সূচিতে _premium_ প্ল্যান সিলেক্ট করতে, 1 এবং তারপর শব্দটি যোগ করুন:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

মনে রাখবেন, বিভিন্ন বিলিং ফ্রিকোয়েন্সির জন্য একটিমাত্র লিংক রাখা সম্ভব নয়। আপনাকে প্রতিটি বিলিং ফ্রিকোয়েন্সির জন্য আলাদা আলাদা লিংক শেয়ার করতে হবে।

আপনি যখন কাস্টমারদের সাথে লিংক শেয়ার করবেন, এটি তাদের সরাসরি সাইনআপ প্রক্রিয়ার প্রথম ধাপে নিয়ে যাবে।
