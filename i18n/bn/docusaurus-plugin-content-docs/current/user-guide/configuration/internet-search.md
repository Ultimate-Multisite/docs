# ইন্টারনেট সার্চ

Gratis AI Agent v1.5.0 একটি **Internet Search** ক্ষমতা যোগ করেছে, যা AI অ্যাসিস্ট্যান্টকে কথোপকথনের সময় ওয়েব থেকে আপ-টু-ডেট তথ্য পুনরুদ্ধার করতে সাহায্য করে। এটি [Brave Search API](https://brave.com/search/api/) বা [Tavily API](https://tavily.com/) দ্বারা চালিত।

## এটি কীভাবে কাজ করে

যখন ইন্টারনেট সার্চ চালু থাকে, তখন অ্যাসিস্ট্যান্ট স্বয়ংক্রিয়ভাবে আপনার কনফিগার করা সার্চ প্রোভাইডারকে প্রশ্নটি বর্তমান বা বাহ্যিক তথ্যের প্রয়োজন কিনা—যেমন, সাম্প্রতিক খবর, লাইভ মূল্য, বা এমন ডকুমেন্টেশন যা মডেলের প্রশিক্ষণের কাটঅফ তারিখের পরে পরিবর্তিত হতে পারে—তা নির্ধারণ করলে কোয়েরি করে।

ফলাফলগুলি রিয়েল-টাইমে পুনরুদ্ধার করা হয় এবং উত্তর তৈরি করার আগে অ্যাসিস্ট্যান্টের কনটেক্সটে যুক্ত করা হয়। অ্যাসিস্ট্যান্ট প্রশ্নটির উত্তর দিতে সার্চ ফলাফল ব্যবহার করেছে কিনা, তা নির্দেশ করে।

## ইন্টারনেট সার্চ সক্ষম করা

ইন্টারনেট সার্চের জন্য আপনার নির্বাচিত সার্চ প্রোভাইডার থেকে একটি API key প্রয়োজন। এটি কনফিগার করতে:

1. **Gratis AI Agent → Settings → Advanced** এ যান।
2. **Internet Search Provider** ড্রপডাউনটি খুঁজে বের করুন এবং হয় **Brave Search** অথবা **Tavily** নির্বাচন করুন।
3. সংশ্লিষ্ট ফিল্ডে আপনার API key লিখুন। প্রতিটি ফিল্ডের পাশে সাইন-আপ লিঙ্ক দেখানো হয়েছে।
4. **Save Settings** এ ক্লিক করুন।

key টি সেভ করার সাথে সাথেই ইন্টারনেট সার্চ ক্ষমতাটি স্বয়ংক্রিয়ভাবে অ্যাসিস্ট্যান্টের জন্য উপলব্ধ হয়ে যায়।

## Brave Search

### Brave Search API Key সংগ্রহ করা

1. [Brave Search API page](https://brave.com/search/api/) ভিজিট করুন।
2. একটি প্ল্যানের জন্য সাইন আপ করুন। মাসিক অনুরোধের সীমা সহ একটি ফ্রি টায়ার উপলব্ধ।
3. Brave Search developer dashboard থেকে আপনার API key কপি করুন।
4. Gratis AI Agent সেটিংসে **Brave Search API Key** ফিল্ডে এটি পেস্ট করুন।

### ব্যবহারের সীমা

ব্যবহারের জন্য Brave Search কোয়েরির সংখ্যার ভিত্তিতে বিল করে। প্রতিটি AI উত্তর যা সার্চ ট্রিগার করে, তা একটি কোয়েরি হিসাবে গণ্য হয়। অপ্রত্যাশিত চার্জ এড়াতে [Brave Search developer dashboard](https://brave.com/search/api/) এ আপনার ব্যবহার পর্যবেক্ষণ করুন।

## Tavily

Superdav AI Agent v1.10.0 একটি বিকল্প ইন্টারনেট সার্চ প্রোভাইডার হিসাবে **Tavily** যোগ করেছে, যা আরও সমৃদ্ধ সার্চ ফলাফল এবং উন্নত গবেষণা ক্ষমতা প্রদান করে।

### Tavily API Key সংগ্রহ করা

1. [Tavily API page](https://tavily.com/) ভিজিট করুন।
2. একটি অ্যাকাউন্টের জন্য সাইন আপ করুন। মাসিক অনুরোধের সীমা সহ একটি ফ্রি টায়ার উপলব্ধ।
3. Tavily dashboard থেকে আপনার API key কপি করুন।
4. Gratis AI Agent সেটিংসে **Tavily API Key** ফিল্ডে এটি পেস্ট করুন।

### ব্যবহারের সীমা

ব্যবহারের জন্য Tavily API কলের সংখ্যার ভিত্তিতে বিল করে। প্রতিটি AI উত্তর যা সার্চ ট্রিগার করে, তা একটি কল হিসাবে গণ্য হয়। অপ্রত্যাশিত চার্জ এড়াতে [Tavily dashboard](https://tavily.com/) এ আপনার ব্যবহার পর্যবেক্ষণ করুন।

## ইন্টারনেট সার্চ নিষ্ক্রিয় করা

সক্রিয় সার্চ প্রোভাইডারের ফিল্ড থেকে API key টি সরিয়ে দিন এবং সেভ করুন। ইন্টারনেট সার্চ ক্ষমতাটি আর অ্যাসিস্ট্যান্টকে অফার করা হবে না।

:::note
ইন্টারনেট সার্চ উত্তরগুলিতে ল্যাটেন্সি (latency) যোগ করে কারণ উত্তর তৈরি করার আগে অ্যাসিস্ট্যান্টকে সার্চ ফলাফলের জন্য অপেক্ষা করতে হয়। সময়-সংবেদনশীল ব্যবহারের ক্ষেত্রে, বিবেচনা করুন যে রিয়েল-টাইম সার্চ কি সত্যিই প্রয়োজন, নাকি অ্যাসিস্ট্যান্টের বিল্ট-ইন জ্ঞানই যথেষ্ট।
:::
