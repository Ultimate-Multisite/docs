# অন্যান্য প্লাগইনের সাথে পরিচিত অসামঞ্জস্যতা

Ultimate Multisite ইনস্টল ও সক্রিয় করার পর, অন্যান্য প্লাগইনের সাথে সমাধান না হওয়া অসামঞ্জস্যতার কারণে আপনি সমস্যার সম্মুখীন হতে পারেন।

আমরা প্রতিটি নতুন রিলিজে এই তালিকা কমিয়ে আনতে সক্রিয়ভাবে কাজ করছি, তবে কোনো প্লাগইন এখানে তালিকাভুক্ত থাকলে, সম্পূর্ণ কার্যকরী Ultimate Multisite Network পেতে আপনাকে সেটি নিষ্ক্রিয় করতে হতে পারে।

এখানে সেগুলোর একটি তালিকা এবং এসব ক্ষেত্রে আপনার কী করা উচিত তা দেওয়া হলো:

**কিছু WPMU DEV প্লাগইন** WPMU DEV দ্বারা _আর রক্ষণাবেক্ষণ করা হয় না_ এমন কিছু প্লাগইন Ultimate Multisite রেজিস্ট্রেশন প্রক্রিয়াকে শেষ ধাপে আটকে দেয় বলে জানা গেছে (ব্যবহারকারী অ্যাকাউন্ট তৈরি করতে পারেন না)। এই সমস্যা সৃষ্টিকারী প্লাগইনগুলোর একটি তালিকা এখানে দেওয়া হলো:

  * Signup Language: _Ultimate Multisite-এ একটি অ্যাড-অন রয়েছে যা একই কাজ করে, এটির নাম Ultimate Multisite: Language Switcher।_

  * Set Blog Description on Blog Creation: _এটি Ultimate Multisite-এ অন্তর্ভুক্ত, তাই আলাদা প্লাগইনের প্রয়োজন নেই।_

  * New Blog Templates: _এটি Ultimate Multisite-এ অন্তর্ভুক্ত, তাই আলাদা প্লাগইনের প্রয়োজন নেই।_

  * Set Password on Multisite Blog Creation: _এটি Ultimate Multisite-এ অন্তর্ভুক্ত, তাই আলাদা প্লাগইনের প্রয়োজন নেই।_

  * Signup TOS: _এটি Ultimate Multisite-এ অন্তর্ভুক্ত, তাই আলাদা প্লাগইনের প্রয়োজন নেই।_

  * Anti-Splog: কম শক্তিশালী হলেও, এটিও _Ultimate Multisite-এ অন্তর্ভুক্ত।_

**দ্রষ্টব্য**: অন্যান্য সব WPMU Dev প্লাগইন Ultimate Multisite-এর পাশাপাশি স্বাভাবিকভাবে ব্যবহার করা যায়। উদাহরণস্বরূপ _Smush_, _Forminator_, _Defender_ ইত্যাদি।
