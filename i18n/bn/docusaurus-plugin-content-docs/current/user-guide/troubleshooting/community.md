# Ultimate Multisite Community-তে কীভাবে যুক্ত হবেন

Ultimate Multisite Community হলো এমন একটি জায়গা যেখানে আপনি অন্যান্য Ultimate Multisite ব্যবহারকারীদের সাথে যোগাযোগ করতে পারবেন এবং টিপস, অভিজ্ঞতা ও আরও অনেক কিছু শেয়ার করতে পারবেন। Ultimate Multisite Community-তে যেতে https://community.ultimatemultisite.com/home এই লিংকে যান।

স্বাগত পেজে "**creating your account here**" লিংকে ক্লিক করুন।

<!-- Screenshot unavailable: Community welcome page with creating your account link -->

আপনাকে আমাদের রেজিস্ট্রেশন পেজে নিয়ে যাওয়া হবে। এখানে আপনাকে একটি username, আপনার ইমেইল এবং একটি password দিতে হবে। Captcha সম্পন্ন করুন এবং **Register** বাটনে ক্লিক করুন।

<!-- Screenshot unavailable: Community register page with username, email, password fields and Captcha -->

রেজিস্ট্রেশনের পর এবার আপনার প্রোফাইল তৈরি করার পালা। এই পেজে আপনি আপনার ছবি আপলোড করতে পারবেন, পুরো নাম লিখতে পারবেন, নিজের সম্পর্কে সংক্ষেপে কিছু লিখতে পারবেন ইত্যাদি। এছাড়াও, আপনার সোশ্যাল মিডিয়া লিংক যোগ করুন এবং আপনার অ্যাকাউন্টের জন্য অনুমতি সেট করুন।

<!-- Screenshot unavailable: Community profile creation page with photo upload and bio fields -->

এখন আপনি "Introductions" চ্যানেলে আছেন, যেখানে আপনি নিজের পরিচয় দিতে পারবেন এবং অন্যান্য সদস্যদের সম্পর্কে কিছুটা জানতে পারবেন। **New Post** বাটনে ক্লিক করে আপনি আপনার পোস্ট তৈরি করতে পারবেন।

<!-- Screenshot unavailable: Community Introductions channel with New Post button -->

Lounge সেকশনের চ্যানেলগুলো এবং "Dev Logs" চ্যানেল সবার জন্য উন্মুক্ত। তবে কমিউনিটির অন্যান্য চ্যানেলে প্রবেশ করতে আপনার বিশেষ অনুমতি প্রয়োজন হবে।

<!-- Screenshot unavailable: Community channels showing Lounge section and Dev Logs channel -->

আপনার সক্রিয় Ultimate Multisite license key দেওয়ার পর এই অ্যাক্সেস দেওয়া হবে। এটি করতে যেকোনো প্রাইভেট চ্যানেল লিংকে ক্লিক করুন এবং তারপর **Unlock** বাটনে ক্লিক করুন।

<!-- Screenshot unavailable: Community private channel with Unlock button -->

আপনাকে আমাদের unlock access পেজে নিয়ে যাওয়া হবে। আপনার Ultimate Multisite license key দিন এবং Unlock Access বাটনে ক্লিক করুন:

<!-- Screenshot unavailable: Community unlock access page with license key input -->

আপনার license যাচাই করা হবে এবং আপনি প্রাইভেট চ্যানেলগুলোতে প্রবেশ করতে পারবেন। এই প্রক্রিয়াটি সম্পন্ন হতে ২৪ ঘণ্টা পর্যন্ত সময় লাগতে পারে, তাই আমরা আপনার অনুরোধ প্রসেস করার সময় কমিউনিটিতে ফিরে যেতে লিংকে ক্লিক করুন এবং অন্যান্য স্পেসগুলো উপভোগ করুন।
