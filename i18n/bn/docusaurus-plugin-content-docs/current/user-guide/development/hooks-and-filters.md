# Ultimate Multisite Hooks এবং filters

Ultimate Multisite অনেকগুলো ফাংশন এবং ডেটাবেস কোয়েরি দিয়ে তৈরি। এতে hooks এবং filters আছে যেগুলো ব্যবহার করে আপনি বিদ্যমান ফাংশনে নতুন কোড যোগ করতে বা পরিবর্তন করতে পারবেন।

আমরা এই hooks এবং filters-এর একটি তালিকা তৈরি করেছি যাতে অভিজ্ঞ ব্যবহারকারীরা তাদের নেটওয়ার্ক আরও কাস্টমাইজ করতে পারেন। নিচের লিঙ্কে ক্লিক করে আপনি Github থেকে এই তালিকাটি দেখতে পারবেন।
