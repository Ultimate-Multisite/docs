# সাইট ব্যবস্থাপনা

সাইট (বা সাবসাইট) হলো আপনার WaaS ব্যবসার মূল ভিত্তি। Ultimate Multisite-এ ৩ ধরনের সাইট রয়েছে:

- **গ্রাহক-মালিকানাধীন** — নির্দিষ্ট গ্রাহকদের মালিকানাধীন সাইট
- **সাইট টেমপ্লেট** — পূর্ব-নির্ধারিত সাইট যা গ্রাহকরা শুরুর পয়েন্ট হিসেবে বেছে নিতে পারেন
- **মূল সাইট** — আপনার প্রাথমিক নেটওয়ার্ক সাইট

## সাইট দেখা

আপনার নেটওয়ার্কের সব সাবসাইট দেখতে **Ultimate Multisite → Sites**-এ যান। প্রতিটি সাইট টাইপ অনুযায়ী লেবেল করা থাকে এবং আপনি All Sites, Customer-Owned, Templates, অথবা Pending অনুযায়ী ফিল্টার করতে পারেন।

![Sites list page](/img/admin/sites-list.png)

## নতুন সাইট যোগ করা

নতুন সাইট তৈরি করতে **Add Site** বাটনে ক্লিক করুন। আপনাকে যা দিতে হবে:

- **Site Title** — নতুন সাইটের নাম
- **Site URL** — সাইটের ডোমেইন/পাথ
- **Site Type** — এটি গ্রাহক সাইট, টেমপ্লেট, নাকি সাধারণ সাইট

**Copy Site** অপশনটি ব্যবহার করে আপনি একটি বিদ্যমান সাইট টেমপ্লেটের উপর ভিত্তি করে নতুন সাইট তৈরি করতে পারেন। এটি চালু করলে, আপনি শুরুর পয়েন্ট হিসেবে কোন টেমপ্লেট ব্যবহার করবেন তা বেছে নিতে পারবেন। মিডিয়া ফাইল অন্তর্ভুক্ত করতে **Copy Media on Duplication** চালু আছে কিনা নিশ্চিত করুন।

## বিদ্যমান সাইট ব্যবস্থাপনা

**Edit Site** পেজ খুলতে যেকোনো সাইটে **Manage**-এ ক্লিক করুন। এখানে আপনি পাবেন:

### মৌলিক তথ্য

সাইটের নাম, টাইপ, সাইট ID, এবং বিবরণ। এছাড়াও আপনি দেখতে পাবেন ম্যাপ করা ডোমেইন, সংশ্লিষ্ট মেম্বারশিপ, এবং সাইটের মালিক গ্রাহক অ্যাকাউন্ট।

### সাইট অপশন

সাইটের সক্ষমতা এবং সীমা কনফিগার করুন:

- **ভিজিট লিমিট** — সাইট ভিজিটের সর্বোচ্চ সংখ্যা
- **ইউজার অ্যাকাউন্ট লিমিট** — প্রতি ইউজার রোল অনুযায়ী লিমিট
- **ডিস্ক স্পেস** — সাইটের স্টোরেজ কোটা
- **কাস্টম ডোমেইন** — এই সাইটের জন্য ডোমেইন ম্যাপিং চালু করুন
- **Plugin এবং theme দৃশ্যমানতা** — কোন plugin এবং theme দৃশ্যমান, লুকানো, বা পূর্ব-সক্রিয় থাকবে তা নিয়ন্ত্রণ করুন

ডিফল্টভাবে, সাইটগুলো মেম্বারশিপ লেভেলে সেট করা লিমিট অনুসরণ করে। সাইট লেভেলে লিমিট সেট করলে তা মেম্বারশিপ সেটিংস ওভাররাইড করবে।

### সংযুক্তি

সাইট অপশনের নিচে আপনি তথ্য পাবেন:

- সাইটের সাথে সংশ্লিষ্ট **ম্যাপ করা ডোমেইন**
- সাইটটি যে **মেম্বারশিপ**-এর অধীনে আছে
- সাইটের সাথে সংযুক্ত **গ্রাহক অ্যাকাউন্ট**

### ডান সাইডবার

ডান দিকে আপনি পারবেন:

- টগল বাটন দিয়ে **সাইট চালু/বন্ধ করতে**
- **সাইট টাইপ পরিবর্তন** বা মালিকানা পুনর্নির্ধারণ করতে
- **সাইট ছবি/থাম্বনেইল সেট করতে** (সাইট টেমপ্লেটের জন্য frontend-এ প্রদর্শিত হয়)
- **সাইট স্থায়ীভাবে মুছে ফেলতে**

:::warning
সাইট মুছে ফেলা অপরিবর্তনীয়। সাইট এবং এর সমস্ত কন্টেন্ট স্থায়ীভাবে মুছে যাবে।
:::
