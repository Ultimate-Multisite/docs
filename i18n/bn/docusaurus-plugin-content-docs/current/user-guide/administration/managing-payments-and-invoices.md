# পেমেন্ট এবং ইনভয়েস পরিচালনা

## পেমেন্ট সেটিংস

পেমেন্ট গ্রহণ শুরু করার আগে, আপনাকে পেমেন্ট সংক্রান্ত সেটিংস কনফিগার করতে হবে। **Ultimate Multisite → Settings**-এ যান এবং **Payment** ট্যাবে ক্লিক করুন।

![Payment settings tab](/img/admin/settings-payments-top.png)

### সাধারণ পেমেন্ট অপশন

সাধারণ সেটিংসে আপনি যা কনফিগার করতে পারবেন:

- **Currency** — লেনদেনের জন্য ডিফল্ট মুদ্রা
- **Currency Position** — মুদ্রার চিহ্ন কোথায় দেখাবে (পরিমাণের আগে/পরে)

![Payment gateway settings](/img/admin/settings-payments-options.png)

### পেমেন্ট গেটওয়ে

Ultimate Multisite একাধিক পেমেন্ট গেটওয়ে সাপোর্ট করে। Payment settings ট্যাব থেকে আপনি প্রতিটি গেটওয়ে চালু এবং কনফিগার করতে পারবেন।

![Payment gateway configuration](/img/admin/settings-payments-gateways.png)

যেসব গেটওয়ে পাওয়া যায়:

- **Stripe** — Stripe-এর মাধ্যমে ক্রেডিট কার্ড পেমেন্ট
- **PayPal** — PayPal পেমেন্ট
- **Manual** — অফলাইন বা কাস্টম পেমেন্ট প্রসেসিংয়ের জন্য

প্রতিটি গেটওয়ের নিজস্ব কনফিগারেশন সেকশন আছে যেখানে আপনি API key এবং অন্যান্য সেটিংস দিতে পারবেন।

![Additional gateway settings](/img/admin/settings-payments-gateways-2.png)

### Sandbox Mode

লাইভ হওয়ার আগে পেমেন্ট ইন্টিগ্রেশন পরীক্ষা করতে আপনি **Sandbox Mode** চালু করতে পারেন। Sandbox mode চালু থাকলে, কোনো আসল চার্জ কাটা হবে না।

## পেমেন্ট দেখা

আপনার নেটওয়ার্কের সব লেনদেন দেখতে Ultimate Multisite-এর অধীনে **Payments** পেজে যান।

![Payments list](/img/admin/payments-list.png)

আপনি স্ট্যাটাস অনুযায়ী (completed, pending, failed, refunded) পেমেন্ট ফিল্টার করতে পারবেন এবং নির্দিষ্ট লেনদেন সার্চ করতে পারবেন।

কোনো পেমেন্টে ক্লিক করলে এর পূর্ণ বিবরণ দেখা যাবে, যেমন line items, সংশ্লিষ্ট মেম্বারশিপ, গ্রাহকের তথ্য এবং পেমেন্ট গেটওয়ে ডেটা।

## ইনভয়েস

Ultimate Multisite স্বয়ংক্রিয়ভাবে পেমেন্টের জন্য ইনভয়েস তৈরি করতে পারে। Payment settings থেকে আপনি ইনভয়েস টেমপ্লেট এবং নম্বরিং ফরম্যাট কাস্টমাইজ করতে পারবেন।

ইনভয়েস কাস্টমাইজেশনের অপশনগুলো হলো:

- ইনভয়েসে প্রদর্শিত **কোম্পানির নাম ও ঠিকানা**
- **ইনভয়েস নম্বরিং** ফরম্যাট এবং ক্রম
- ইনভয়েস হেডারে প্রদর্শিত **লোগো**
- শর্তাবলী, নোট বা আইনি তথ্যের জন্য **কাস্টম ফুটার টেক্সট**

ইনভয়েস টেমপ্লেট কাস্টমাইজ করতে, **Ultimate Multisite → Settings → Payment**-এ যান এবং ইনভয়েস সংক্রান্ত সেটিংস খুঁজুন।
