# Closte ইন্টিগ্রেশন

## সংক্ষিপ্ত বিবরণ
Closte হলো Google Cloud ইনফ্রাস্ট্রাকচারের উপর নির্মিত একটি managed WordPress হোস্টিং প্ল্যাটফর্ম। এই ইন্টিগ্রেশনের মাধ্যমে Ultimate Multisite এবং Closte-এর মধ্যে স্বয়ংক্রিয় ডোমেইন সিঙ্কিং এবং SSL সার্টিফিকেট ব্যবস্থাপনা করা যায়।

## বৈশিষ্ট্যসমূহ
- স্বয়ংক্রিয় ডোমেইন সিঙ্কিং
- SSL সার্টিফিকেট ব্যবস্থাপনা
- Wildcard ডোমেইন সাপোর্ট
- Closte-এ চালালে আলাদা কনফিগারেশনের প্রয়োজন নেই

## প্রয়োজনীয়তা
আপনি যদি Closte ব্যবহার করেন, তাহলে আপনার `wp-config.php` ফাইলে নিচের constant টি সংজ্ঞায়িত করতে হবে:

```php
define('CLOSTE_CLIENT_API_KEY', 'your_api_key');
```

আপনি যদি Closte-এ হোস্ট করেন, তাহলে সাধারণত এই constant টি আগে থেকেই সংজ্ঞায়িত থাকে।

## সেটআপ নির্দেশনা

### ১. আপনার Closte API Key যাচাই করুন

আপনি যদি Closte-এ হোস্ট করেন, তাহলে `CLOSTE_CLIENT_API_KEY` constant টি আপনার `wp-config.php` ফাইলে আগে থেকেই সংজ্ঞায়িত থাকার কথা। আপনার `wp-config.php` ফাইল চেক করে এটি যাচাই করতে পারেন।

### ২. ইন্টিগ্রেশন সক্রিয় করুন

1. আপনার WordPress admin-এ যান Ultimate Multisite > Settings
2. "Domain Mapping" ট্যাবে যান
3. নিচে স্ক্রল করে "Host Integrations" এ যান
4. Closte ইন্টিগ্রেশন সক্রিয় করুন
5. "Save Changes" এ ক্লিক করুন

## এটি কীভাবে কাজ করে

Ultimate Multisite-এ কোনো ডোমেইন ম্যাপ করা হলে:

1. ইন্টিগ্রেশনটি Closte-এর API-তে একটি রিকোয়েস্ট পাঠায় আপনার অ্যাপ্লিকেশনে ডোমেইন যোগ করার জন্য
2. Closte স্বয়ংক্রিয়ভাবে SSL সার্টিফিকেট প্রভিশনিং সামলায়
3. কোনো ডোমেইন ম্যাপিং সরানো হলে, ইন্টিগ্রেশনটি Closte থেকে সেই ডোমেইন সরিয়ে দেবে

ইন্টিগ্রেশনটি Ultimate Multisite-এর DNS check interval সেটিংয়ের সাথেও কাজ করে, যার মাধ্যমে আপনি নির্ধারণ করতে পারেন সিস্টেম কত ঘন ঘন DNS propagation এবং SSL সার্টিফিকেট ইস্যু হওয়া চেক করবে।

## ডোমেইন রেকর্ড তৈরি

এই ইন্টিগ্রেশন নিশ্চিত করে যে কোনো সাইট তৈরি বা ডুপ্লিকেট করা হলে স্বয়ংক্রিয়ভাবে একটি ডোমেইন রেকর্ড তৈরি হয়। এটি Closte ইন্টিগ্রেশনের জন্য বিশেষভাবে গুরুত্বপূর্ণ, কারণ ডোমেইন রেকর্ড তৈরি হলেই Closte API ট্রিগার হয়ে ডোমেইন এবং SSL সার্টিফিকেট তৈরি করে।

## সমস্যা সমাধান

### API সংযোগ সমস্যা
- আপনার Closte API key সঠিক কিনা যাচাই করুন
- নিশ্চিত করুন যে আপনার Closte অ্যাকাউন্টে প্রয়োজনীয় অনুমতি আছে

### SSL সার্টিফিকেট সমস্যা
- Closte-এর SSL সার্টিফিকেট ইস্যু করতে কিছুটা সময় লাগতে পারে (সাধারণত ৫-১০ মিনিট)
- যাচাই করুন যে আপনার ডোমেইনগুলো সঠিকভাবে আপনার Closte সার্ভারের IP অ্যাড্রেসে পয়েন্ট করছে
- আপনার ডোমেইনের DNS রেকর্ড সঠিকভাবে কনফিগার করা আছে কিনা চেক করুন

### ডোমেইন যোগ হয়নি
- কোনো error message আছে কিনা Ultimate Multisite লগ চেক করুন
- যাচাই করুন যে ডোমেইনটি ইতিমধ্যে Closte-এ যোগ করা নেই
- নিশ্চিত করুন যে আপনার ডোমেইনের DNS রেকর্ড সঠিকভাবে কনফিগার করা আছে

### DNS Check Interval
- SSL সার্টিফিকেট ইস্যু হতে বেশি সময় লাগলে, Domain Mapping সেটিংসে DNS check interval সামঞ্জস্য করতে পারেন
- ডিফল্ট interval হলো ৩০০ সেকেন্ড (৫ মিনিট), তবে টেস্টিংয়ের সময় দ্রুত চেক করার জন্য এটি ১০ সেকেন্ড পর্যন্ত কমাতে পারেন
