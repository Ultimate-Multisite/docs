# WP Engine ইন্টিগ্রেশন

## সংক্ষিপ্ত বিবরণ
WP Engine হলো একটি প্রিমিয়াম ম্যানেজড WordPress হোস্টিং প্ল্যাটফর্ম যা WordPress সাইটগুলোর জন্য অপ্টিমাইজড পারফরম্যান্স, সিকিউরিটি এবং স্কেলেবিলিটি প্রদান করে। এই ইন্টিগ্রেশন Ultimate Multisite এবং WP Engine-এর মধ্যে স্বয়ংক্রিয় ডোমেইন সিঙ্কিং সক্ষম করে।

## বৈশিষ্ট্যসমূহ
- স্বয়ংক্রিয় ডোমেইন সিঙ্কিং
- মাল্টিসাইট ইনস্টলেশনের জন্য সাবডোমেইন সাপোর্ট
- WP Engine-এর বিদ্যমান সিস্টেমের সাথে নিরবচ্ছিন্ন ইন্টিগ্রেশন

## প্রয়োজনীয়তা
আপনি WP Engine-এ হোস্ট করছেন কিনা ইন্টিগ্রেশন স্বয়ংক্রিয়ভাবে সেটা শনাক্ত করে এবং বিল্ট-ইন WP Engine API ব্যবহার করে। WP Engine plugin সক্রিয় এবং সঠিকভাবে কনফিগার করা থাকলে আলাদা কোনো কনফিগারেশনের প্রয়োজন নেই।

তবে, আপনাকে যদি ম্যানুয়ালি ইন্টিগ্রেশন কনফিগার করতে হয়, তাহলে আপনার `wp-config.php` ফাইলে এই constant-গুলোর যেকোনো একটি ডিফাইন করতে পারেন:

```php
define('WPE_APIKEY', 'your_api_key'); // পছন্দনীয় পদ্ধতি
// অথবা
define('WPE_API', 'your_api_key'); // বিকল্প পদ্ধতি
```

## সেটআপ নির্দেশনা

### ১. WP Engine Plugin যাচাই করুন

আপনি WP Engine-এ হোস্ট করলে WP Engine plugin ইতোমধ্যে ইনস্টল ও সক্রিয় থাকা উচিত। নিশ্চিত করুন যে:

1. WP Engine plugin সক্রিয় আছে
2. `wp-content/mu-plugins/wpengine-common/class-wpeapi.php` ফাইলটি আছে

### ২. ইন্টিগ্রেশন সক্রিয় করুন

1. আপনার WordPress অ্যাডমিনে Ultimate Multisite > Settings-এ যান
2. "Domain Mapping" ট্যাবে নেভিগেট করুন
3. "Host Integrations"-এ স্ক্রল করুন
4. WP Engine ইন্টিগ্রেশন সক্রিয় করুন
5. "Save Changes" ক্লিক করুন

## এটি কীভাবে কাজ করে

### ডোমেইন সিঙ্কিং

Ultimate Multisite-এ কোনো ডোমেইন ম্যাপ করা হলে:

1. ইন্টিগ্রেশন WP Engine API ব্যবহার করে আপনার WP Engine ইনস্টলেশনে ডোমেইন যোগ করে
2. WP Engine ডোমেইন কনফিগারেশন এবং SSL সার্টিফিকেট ইস্যু করার কাজ সামলায়
3. ডোমেইন ম্যাপিং সরিয়ে দিলে ইন্টিগ্রেশন WP Engine থেকে ডোমেইনটি মুছে দেবে

### সাবডোমেইন সাপোর্ট

সাবডোমেইন মাল্টিসাইট ইনস্টলেশনের জন্য:

1. নতুন সাইট তৈরি হলে ইন্টিগ্রেশন প্রতিটি সাবডোমেইন WP Engine-এ যোগ করে
2. WP Engine সাবডোমেইন কনফিগারেশন সামলায়
3. কোনো সাইট মুছে ফেলা হলে ইন্টিগ্রেশন WP Engine থেকে সাবডোমেইনটি সরিয়ে দেবে

## গুরুত্বপূর্ণ তথ্য

### ওয়াইল্ডকার্ড ডোমেইন

সাবডোমেইন মাল্টিসাইট ইনস্টলেশনের জন্য WP Engine সাপোর্টের সাথে যোগাযোগ করে ওয়াইল্ডকার্ড ডোমেইন কনফিগারেশন রিকোয়েস্ট করার পরামর্শ দেওয়া হচ্ছে। এতে প্রতিটি সাবডোমেইন আলাদাভাবে যোগ না করেই সব সাবডোমেইন স্বয়ংক্রিয়ভাবে কাজ করবে।

### SSL সার্টিফিকেট

এই ইন্টিগ্রেশনের মাধ্যমে যোগ করা সব ডোমেইনের জন্য WP Engine স্বয়ংক্রিয়ভাবে SSL সার্টিফিকেট ইস্যু এবং রিনিউ করে। আলাদা কোনো কনফিগারেশনের প্রয়োজন নেই।

## সমস্যা সমাধান

### API কানেকশন সমস্যা
- WP Engine plugin সক্রিয় এবং সঠিকভাবে কনফিগার করা আছে কিনা যাচাই করুন
- আপনি ম্যানুয়ালি API key ডিফাইন করলে সেটি সঠিক কিনা পরীক্ষা করুন
- API নিয়ে সমস্যা হলে WP Engine সাপোর্টের সাথে যোগাযোগ করুন

### ডোমেইন যোগ হচ্ছে না
- কোনো error message আছে কিনা Ultimate Multisite logs দেখুন
- ডোমেইনটি ইতোমধ্যে WP Engine-এ যোগ করা আছে কিনা যাচাই করুন
- আপনার WP Engine প্ল্যান আপনি যত ডোমেইন যোগ করছেন তা সাপোর্ট করে কিনা নিশ্চিত করুন

### সাবডোমেইন সমস্যা
- সাবডোমেইন কাজ না করলে WP Engine সাপোর্টের সাথে যোগাযোগ করে ওয়াইল্ডকার্ড ডোমেইন কনফিগারেশন রিকোয়েস্ট করুন
- মূল ডোমেইন এবং সাবডোমেইনের জন্য আপনার DNS সেটিংস সঠিকভাবে কনফিগার করা আছে কিনা যাচাই করুন
