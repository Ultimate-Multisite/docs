# আমি কীভাবে প্রথমবার আমার অ্যাকাউন্ট পেজে প্রবেশ করব?

Ultimate Multisite-এ স্বাগতম!

আমাদের plugin কেনার পর আপনার ই-মেইলে যে লিঙ্ক পেয়েছেন, সেটি দিয়ে আপনার অ্যাকাউন্ট পেজে প্রবেশ করুন:

<!-- Screenshot unavailable: Account page email with access link -->

কেনার সময় যে ইমেইল দিয়েছিলেন এবং ইমেইলে পাওয়া পাসওয়ার্ড ব্যবহার করুন।

<!-- Screenshot unavailable: Login page with email and password fields -->

যেহেতু এই অ্যাকাউন্টে আপনি প্রথমবার লগইন করছেন, আপনাকে পাসওয়ার্ড পরিবর্তন করতে বলা হবে। পাসওয়ার্ড পরিবর্তন করে _Save_ বাটনে ক্লিক করুন।

<!-- Screenshot unavailable: Change password form with Save button -->

আপনার অ্যাকাউন্ট পেজ থেকে আপনি বিলিং, রিনিউয়াল, বাতিলকরণ এবং আরও অনেক কিছুসহ আপনার অ্যাকাউন্টের সব তথ্য পরিচালনা করতে পারবেন।
