# আপনার ক্লায়েন্টদের ওয়েবসাইট (v2)

আপনার নেটওয়ার্কে রেজিস্ট্রেশন করার পর, আপনার কাস্টমাররা তাদের ড্যাশবোর্ডে প্রবেশ করতে পারবেন।

**Account** পেজ ছাড়াও, যেখানে তারা তাদের পেমেন্ট, মেম্বারশিপ, ডোমেইন, প্ল্যানের সীমাবদ্ধতা এবং আরও অনেক গুরুত্বপূর্ণ তথ্য দেখতে পারবেন, তাদের কাছে **My Sites** পেজেও অ্যাক্সেস থাকে, যেখানে তারা তাদের সব ওয়েবসাইট দেখতে এবং পরিচালনা করতে পারবেন। তাদের শুধু ড্যাশবোর্ডে My Sites-এ ক্লিক করতে হবে।

![কাস্টমার ড্যাশবোর্ডে My Sites লিংক](/img/admin/customers-list.png)

তাদের সব ওয়েবসাইট দেখানো হবে, প্রতিটির নিচে একটি **Manage** বাটন থাকবে। এটিতে ক্লিক করলে তারা সেই **নির্দিষ্ট ওয়েবসাইটের ড্যাশবোর্ডে** চলে যাবেন।

যেসব প্ল্যানে একাধিক ওয়েবসাইট রাখার সুবিধা আছে, সেখানে তারা দেখতে পারবেন কোন ওয়েবসাইটটি তাদের **প্রাইমারি** ওয়েবসাইট।

![My Sites পেজে Manage বাটনসহ কাস্টমার ওয়েবসাইটগুলো](/img/admin/customers-list.png)

প্রাইমারি ওয়েবসাইট পরিবর্তন করতে চাইলে, তাদের **Account** মেনুতে ক্লিক করে Account পেজে যেতে হবে।

![কাস্টমার ড্যাশবোর্ডে Account মেনু লিংক](/img/admin/customers-list.png)

এরপর, তাদের **Change Default Site**-এ ক্লিক করতে হবে।

![অ্যাকাউন্ট পেজে Change Default Site বাটন](/img/admin/customers-list.png)

একটি পপ-আপ উইন্ডো আসবে। ড্রপ-ডাউন লিস্টে ক্লিক করুন এবং কোন ওয়েবসাইটটিকে প্রাইমারি ওয়েবসাইট করতে চান সেটি বেছে নিন।

![Change Default Site ড্রপডাউন সিলেকশন](/img/admin/customers-list.png)

তারপর, শুধু **Change Default Site**-এ ক্লিক করুন আর কাজ শেষ!

![ডিফল্ট সাইট পরিবর্তন নিশ্চিতকরণ](/img/admin/customers-list.png)
