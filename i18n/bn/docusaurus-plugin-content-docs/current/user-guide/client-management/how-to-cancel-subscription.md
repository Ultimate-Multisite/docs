# আমি কীভাবে আমার সাবস্ক্রিপশন বাতিল করতে পারি?

আপনার যদি Ultimate Multisite লাইসেন্স থাকে, আপনি যেকোনো সময় এর রিনিউয়াল বাতিল করতে পারেন। শুধু নিচের ধাপগুলো অনুসরণ করুন:


<!-- Screenshot unavailable: Account page email with access link -->

কেনাকাটার সময় যে ইমেইল দিয়েছিলেন সেটি এবং আপনার পাসওয়ার্ড ব্যবহার করুন।

<!-- Screenshot unavailable: Login page with email and password fields -->

এটি হলো আপনার Account Page-এর প্রাথমিক স্ক্রিন:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

আপনার সাবস্ক্রিপশন বাতিল করতে, বাম পাশের মেনু থেকে _Renewing & Billing_-এ যান।

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

একটি সাইড উইন্ডো খুলতে ডানদিকের তীর চিহ্নে ক্লিক করুন। তারপর, **auto-renew** বাতিল করার অপশনটি সিলেক্ট করুন।

সিস্টেম আপনাকে একটি নিশ্চিতকরণ বার্তা দেখাবে।

আপনি যদি নিশ্চিত হন যে আপনার সাবস্ক্রিপশন বাতিল করতে চান, তাহলে শুধু _**Cancel Renewals**_ বাটনে ক্লিক করুন। এই কাজটি করার পর, আপনাকে একটি ছোট সার্ভে-তে উত্তর দিতে বলা হবে।

হয়ে গেছে! আপনার সাবস্ক্রিপশন আর স্বয়ংক্রিয়ভাবে রিনিউ হবে না।

**আপনার সাবস্ক্রিপশনের মেয়াদ শেষ না হওয়া পর্যন্ত আপনার কাছে একটি বৈধ key থাকবে**। আপনি যদি আপনার সাবস্ক্রিপশন পুনরায় সক্রিয় করতে চান, তাহলে সেটি ম্যানুয়ালি করতে হবে।
