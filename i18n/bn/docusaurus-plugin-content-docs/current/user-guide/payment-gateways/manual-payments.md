# ম্যানুয়াল পেমেন্ট সেটআপ করা (v2)

_**গুরুত্বপূর্ণ নোট: এই আর্টিকেলটি Ultimate Multisite সংস্করণ 2.x-এর জন্য প্রযোজ্য।**_

ম্যানুয়াল পেমেন্ট হলো এমন একটি উপায় যার মাধ্যমে আপনি অন্যান্য পেমেন্ট পদ্ধতি অফার করতে পারেন যখন আপনার ব্যবহারকারীদের জন্য **Stripe** বা **PayPal** উপলব্ধ নেই। এটি ওয়্যার বা ব্যাংক ট্রান্সফার অথবা আপনার ব্যবহারকারীদের স্থানীয়ভাবে পাওয়া যেকোনো পেমেন্ট পদ্ধতি হতে পারে।

## কিভাবে ম্যানুয়াল পেমেন্ট চালু করবেন

ম্যানুয়াল পেমেন্ট সেটআপ করা খুবই সহজ। আপনাকে শুধু payment gateways-এর অধীনে এটি চালু করতে হবে এবং ব্যবহারকারী কিভাবে পেমেন্ট পাঠাবেন সে সম্পর্কে বিস্তারিত নির্দেশনা দিতে হবে।

প্রথমে, **Ultimate Multisite > Settings > Payments**-এ যান। **Payment Gateways**-এর নিচে, **Manual** টগল চালু করুন। আপনি দেখবেন যে একটি **Payment Instructions** বক্স আপনার সামনে দেখা যাবে।

এই বক্সে সেই তথ্য যোগ করুন যা আপনার কাস্টমারের পেমেন্ট করতে প্রয়োজন হবে। উদাহরণস্বরূপ, এটি আপনার ব্যাংক অ্যাকাউন্টের বিবরণ এবং আপনার ইমেইল হতে পারে যাতে কাস্টমার আপনাকে পেমেন্ট কনফার্মেশন পাঠাতে পারেন।

![Manual payment gateway টগল এবং payment instructions বক্স](/img/config/settings-payment-gateways.png)

সেটআপ করার পর, শুধু **Save Settings**-এ ক্লিক করুন এবং কাজ শেষ। যখন ব্যবহারকারীরা আপনার নেটওয়ার্কে রেজিস্টার করবেন, তারা একটি মেসেজ দেখতে পাবেন যে তারা ক্রয় সম্পন্ন করার জন্য আপনার নির্দেশনা পাবেন।

![রেজিস্ট্রেশনের সময় দেখানো ম্যানুয়াল পেমেন্ট মেসেজ](/img/config/settings-payment-gateways.png)

এবং তারা আপনার **Thank You** পেজেও আপনার পেমেন্ট নির্দেশনাসহ একটি মেসেজ পাবেন।

![পেমেন্ট নির্দেশনা দেখানো Thank You পেজ](/img/config/settings-payment-gateways.png)

## ম্যানুয়াল পেমেন্ট কনফার্ম করা

একটি ম্যানুয়াল পেমেন্ট কনফার্ম করতে, বাম পাশের বারে **Payments** মেনুতে যান। সেখানে আপনি আপনার নেটওয়ার্কের সমস্ত পেমেন্ট এবং তাদের বিবরণ দেখতে পাবেন, যার মধ্যে তাদের **status**-ও আছে। একটি ম্যানুয়াল পেমেন্ট সবসময় **Pending** স্ট্যাটাসে থাকবে যতক্ষণ না আপনি নিজে এটি পরিবর্তন করেন।

![Pending ম্যানুয়াল পেমেন্ট দেখানো পেমেন্ট তালিকা](/img/admin/payments-list.png)

**reference code**-এ ক্লিক করে পেমেন্ট পেজে প্রবেশ করুন। এই পেজে আপনি pending পেমেন্টের সমস্ত বিবরণ পাবেন, যেমন reference ID, products, timestamps এবং আরও অনেক কিছু।

![reference code এবং products সহ পেমেন্ট বিবরণ পেজ](/img/admin/payments-list.png)

ডান কলামে, আপনি পেমেন্টের স্ট্যাটাস পরিবর্তন করতে পারবেন। এটি **Completed**-এ পরিবর্তন করে এবং **Activate Membership** অপশন টগল করলে আপনার কাস্টমারের সাইট সক্রিয় হবে এবং তাদের membership চালু হয়ে যাবে।

![Activate Membership টগল সহ Completed-এ পেমেন্ট স্ট্যাটাস পরিবর্তন](/img/admin/payments-list.png)
