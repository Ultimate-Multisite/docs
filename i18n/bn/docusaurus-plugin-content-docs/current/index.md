# Ultimate Multisite ডকুমেন্টেশন {#ultimate-multisite-documentation}

Ultimate Multisite ডকুমেন্টেশনে আপনাকে স্বাগতম। Ultimate Multisite WordPress Multisite-কে একটি সম্পূর্ণ Website as a Service (WaaS) প্ল্যাটফর্মে রূপান্তরিত করে।

## ডকুমেন্টেশন বিভাগসমূহ {#documentation-sections}

### [User Guide](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

আপনার WaaS প্ল্যাটফর্ম সেটআপ, কনফিগার এবং পরিচালনা করার জন্য প্রয়োজনীয় সবকিছু এখানে পাবেন। এতে ইনস্টলেশন, পেমেন্ট গেটওয়ে, ডোমেইন ম্যাপিং, ক্লায়েন্ট ম্যানেজমেন্ট এবং আরও অনেক কিছু কভার করা হয়েছে।

### [Developer Documentation](/developer/hooks) {#developer-documentation}

Ultimate Multisite-এর উপর ভিত্তি করে কাজ করা ডেভেলপারদের জন্য এটি একটি টেকনিক্যাল রেফারেন্স। এতে hooks রেফারেন্স, REST API ডকুমেন্টেশন এবং অ্যাডন ডেভেলপমেন্ট গাইড অন্তর্ভুক্ত রয়েছে।

### [Addons](/addons) {#addons}

এটি পেইড অ্যাডন প্লাগইনগুলির ডকুমেন্টেশন, যা Ultimate Multisite-কে অতিরিক্ত বৈশিষ্ট্য যেমন WooCommerce ইন্টিগ্রেশন, অ্যানালিটিক্স, ডোমেইন বিক্রি এবং আরও অনেক কিছু দিয়ে প্রসারিত করে।
