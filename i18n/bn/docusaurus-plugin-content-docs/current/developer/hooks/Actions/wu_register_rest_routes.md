# অ্যাকশন: wu_register_rest_routes

অতিরিক্ত রুট (routes) নিবন্ধন করার অনুমতি দেয়।

এটি আমাদের `/register` এন্ডপয়েন্ট দ্বারা ব্যবহৃত হয়।

## প্যারামিটারসমূহ

| নাম | ধরন | বিবরণ |
|------|------|-------------|
| $api_instance | `self` | বর্তমান API ইনস্ট্যান্স। |
