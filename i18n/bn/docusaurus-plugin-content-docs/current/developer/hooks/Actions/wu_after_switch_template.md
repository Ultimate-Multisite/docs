# Action: wu_after_switch_template {#action-wuafterswitchtemplate}

যখন কোনো ব্যবহারকারী বা সুপার অ্যাডমিন সাইটের টেমপ্লেট পরিবর্তন করেন, তখন প্লাগইন ডেভেলপারদের ফাংশন হুক করার সুযোগ দেয়।
