# Action: wu_dashboard_{$this->tab}_widgets {#action-wudashboardthis-tabwidgets}

## প্যারামিটার {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $tab | `string` | বর্তমান ট্যাব। |
| $screen | `\WP_Screen` | স্ক্রিন অবজেক্ট। |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Ultimate Multisite অ্যাডমিন পেজের একটি উদাহরণ (instance)। |
