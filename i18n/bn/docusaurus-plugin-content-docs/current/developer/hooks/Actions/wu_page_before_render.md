# Action: wu_page_before_render {#action-wupagebeforerender}

প্লাগইন ডেভেলপারদের সুযোগ দেয় যে তারা পেজটি দেখানোর (print) আগে অতিরিক্ত কন্টেন্ট যোগ করতে পারে।

## প্যারামিটারসমূহ {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $page_id | `string` | এই পেজটির আইডি। |
| $page | `object` | পেজ অবজেক্টটি। |
