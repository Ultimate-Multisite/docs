# Action: wu_site_created {#action-wusitecreated}

যখন কোনো সাইট প্রথমবার তৈরি হয়, তখন এটি কার্যকর হয়।

যদি টেমপ্লেট থেকে ডুপ্লিকেট করা হয়, তবে এটি কার্যকর হবে না।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $data | `array` | যে ডেটা অবজেক্টটি সংরক্ষণ করা হবে। |
| $site | `\Site` | অবজেক্টের ইনস্ট্যান্স। |
