# Action: wu_mcp_abilities_registered

MCP ক্ষমতাগুলো একটি এনটিটির জন্য রেজিস্টার হওয়ার পরে এটি ট্রিগার হয়।

## প্যারামিটারসমূহ {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $ability_prefix | `string` | ক্ষমতা প্রিফিক্স। |
| $model_name | `string` | মডেলের নাম। |
| $this | `object` | ম্যানেজার ইনস্ট্যান্স। |
