# Action: wu_register_integrations {#action-wuregisterintegrations}

এটি চালু হয় যাতে ইন্টিগ্রেশনগুলো নিজেদের নিবন্ধন করতে পারে।

কোর (Core) এখানে তার ইন্টিগ্রেশনগুলো নিবন্ধন করে।

## প্যারামিটারগুলো {#parameters}

| Name | Type | বিবরণ |
|------|------|-------------|
| $registry | `\Integration_Registry` | রেজিস্ট্রি ইনস্ট্যান্সটি। |

### যেহেতু {#since}

- 2.5.0
