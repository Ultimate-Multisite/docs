# Action: wu_mcp_adapter_initialized

MCP অ্যাডাপ্টারটি ইনিশিয়ালাইজড হওয়ার পরে এটি ট্রিগার হয়।

এটি অন্যান্য প্লাগইন এবং থিমগুলিকে তাদের নিজস্ব ক্ষমতা (abilities) নিবন্ধন করতে সুযোগ দেয়।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | MCP অ্যাডাপ্টারটির ইনস্ট্যান্স। |

- 2.5.0
