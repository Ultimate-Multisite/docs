# Filter: wu_password_require_number

পাসওয়ার্ডে সংখ্যা ব্যবহার করা বাধ্যতামূলক হবে কি না, তা এই ফিল্টারটি নিয়ন্ত্রণ করে।

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | পাসওয়ার্ডে সংখ্যা ব্যবহার করা বাধ্যতামূলক হবে কি না। নিয়ম কার্যকর থাকলে ডিফল্ট মান 'true' থাকে। |
| $defender_active | `bool` | Defender Pro Strong Password সক্রিয় আছে কি না। |
