# Filter: wu_checkout_validation_aliases

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $validation_aliases | `array` | যে অ্যারেতে `id` => `alias` আকারে ডেটা থাকে। |
| $checkout | `\Checkout` | চেকআউট ক্লাসটি। |
