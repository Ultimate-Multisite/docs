# Filter: wu_pre_save_settings

Ultimate Multisite দ্বারা সেভ করার আগে ডেভেলপারদের সেটিংসে ফিল্টার করার সুযোগ দেয়।

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $settings | `array` | যে সেটিংগুলো সেভ করা হবে। |
| $settings_to_save | `array` | যোগ করার জন্য নতুন সেটিংগুলো। |
| $saved_settings | `array` | বর্তমানে সেভ করা সেটিংগুলো। |
