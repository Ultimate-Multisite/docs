# Filter: wu_add_new_domain_modal_fields {#filter-wuaddnewdomainmodalfields}

নতুন ডোমেইন যোগ করার মডালের জন্য ফিল্ডগুলিকে ফিল্টার করে।

এটি অ্যাডঅনগুলিকে (যেমন: Domain Seller) ডোমেইন রেজিস্ট্রেশনের ফিল্ডগুলি পরিবর্তন বা প্রতিস্থাপন করার সুযোগ দেয়।
