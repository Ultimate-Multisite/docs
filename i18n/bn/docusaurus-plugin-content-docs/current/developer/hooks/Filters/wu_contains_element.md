# Filter: wu_contains_element {#filter-wucontainselement}

ডেভেলপারদের প্রাথমিক সার্চের ফলাফল পরিবর্তন করার সুযোগ দেয়।

এটি থার্ড-পার্টি বিল্ডার বা এই ধরনের কাজের জন্য খুব উপযোগী।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $contains_elements | `bool` | যদি উপাদানটি কন্টেন্টের মধ্যে থাকে। |
| $content | `string` | যে কন্টেন্টটি পরীক্ষা করা হচ্ছে। |
| $element | `self` | বর্তমান উপাদানটি। |
| $post | `null\|\WP_Post` | যে পোস্টটি পরীক্ষা করতে হবে। |
