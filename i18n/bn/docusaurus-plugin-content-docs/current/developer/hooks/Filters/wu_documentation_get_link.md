# Filter: wu_documentation_get_link

প্লাগইন ডেভেলপারদের লিঙ্ক ফিল্টার করার সুযোগ দেয়।

এটা ঠিক বোঝা যাচ্ছে না যে এটা কীভাবে কাজে আসবে, তবে এটা না রাখলেও কোনো ক্ষতি নেই।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $link | `string` | যে লিঙ্কটি রেজিস্টার করা হয়েছে |
| $slug | `string` | লিঙ্কটি পুনরুদ্ধার করার জন্য ব্যবহৃত স্ল্যাগ |
| $default_link | `string` | রেজিস্টার করা ডিফল্ট লিঙ্ক |
