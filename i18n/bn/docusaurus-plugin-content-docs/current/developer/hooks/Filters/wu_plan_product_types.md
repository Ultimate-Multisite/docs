# Filter: wu_plan_product_types

যে প্রোডাক্ট টাইপগুলোকে প্ল্যান হিসেবে গণ্য করা হবে, তা ফিল্টার করে।

এই ফিল্টারটি অ্যাডঅনগুলোকে অতিরিক্ত প্রোডাক্ট টাইপ রেজিস্টার করার সুযোগ দেয়, যা ভ্যালিডেশন এবং সেগ্রিগেশন (segregation) করার সময় প্ল্যান হিসেবে চেনা যাবে।

## প্যারামিটারসমূহ {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | প্ল্যান হিসেবে গণ্য করার জন্য প্রোডাক্ট টাইপগুলোর একটি অ্যারে। |
