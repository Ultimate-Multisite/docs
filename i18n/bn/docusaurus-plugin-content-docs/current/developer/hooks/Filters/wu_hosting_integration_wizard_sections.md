# Filter: wu_hosting_integration_wizard_sections

হোস্টিং ইন্টিগ্রেশন সেটআপের জন্য উইজার্ড সেকশনগুলো ফিল্টার করে।

অ্যাডঅনগুলোকে উইজার্ড সেকশন যোগ, অপসারণ বা পরিবর্তন করার সুযোগ দেয়।

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $sections | `array` | উইজার্ড সেকশনগুলো। |
| $integration | `\WP_Ultimo\Integrations\Host_Providers\Base_Host_Provider\|\WP_Ultimo\Integrations\Integration` | কনফিগার করা ইন্টিগ্রেশনটি। |
| $page | `\Hosting_Integration_Wizard_Admin_Page` | উইজার্ড পেজ ইনস্ট্যান্স। |
