# Filter: wu_password_require_uppercase

পাসওয়ার্ডে বড় হাতের অক্ষর (uppercase letters) প্রয়োজন হবে কিনা তা ফিল্টার করে।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | বড় হাতের অক্ষর প্রয়োজন হবে কিনা। নিয়ম প্রয়োগ করা হলে ডিফল্টভাবে এটি `true` থাকে। |
| $defender_active | `bool` | Defender Pro Strong Password সক্রিয় আছে কিনা। |

- 2.4.0
