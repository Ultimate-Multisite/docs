# Filter: wu_view_override_replaceable_views

ডেভেলপারদের প্রতিস্থাপনযোগ্য (replaceable) তালিকায় অতিরিক্ত ফোল্ডার যোগ করার অনুমতি দেয়।

সাবধান থাকুন, কারণ অতিরিক্ত ফোল্ডার যোগ করার অনুমতি দিলে Ultimate Multisite-এর সংস্করণগুলোর পরিবর্তে পুরোনো কপি লোড হতে পারে।

## প্যারামিটারসমূহ {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $replaceable_views | `array` | অনুমোদিত ফোল্ডারগুলোর তালিকা। |

- 2.0.0
