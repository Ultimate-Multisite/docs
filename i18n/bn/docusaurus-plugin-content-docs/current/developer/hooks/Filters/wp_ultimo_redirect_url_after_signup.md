# Filter: wp_ultimo_redirect_url_after_signup

রিডাইরেক্ট URL সেট করুন।

এটি একটি লেগ্যাসি ফিল্টার। এখানে যে প্যারামিটারগুলো পাস করা হয়, তার মধ্যে কিছু প্যারামিটার উপলব্ধ নাও হতে পারে, যেমন $site_id।
