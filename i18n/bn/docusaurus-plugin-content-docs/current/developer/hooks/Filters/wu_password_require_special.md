# Filter: wu_password_require_special

পাসওয়ার্ডে বিশেষ অক্ষর (special characters) প্রয়োজন হবে কিনা তা ফিল্টার করে।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | বিশেষ অক্ষর প্রয়োজন হবে কিনা। নিয়ম কার্যকর থাকলে ডিফল্ট মান `true` থাকে। |
| $defender_active | `bool` | Defender Pro Strong Password সক্রিয় আছে কিনা। |
