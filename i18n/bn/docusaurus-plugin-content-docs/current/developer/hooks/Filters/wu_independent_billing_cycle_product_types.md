# Filter: wu_independent_billing_cycle_product_types

যে প্রোডাক্ট টাইপগুলোর স্বাধীন বিলিং সাইকেল আছে, সেগুলোকে ফিল্টার করে।

যে প্রোডাক্টগুলোর বিলিং সাইকেল স্বাধীন, সেগুলোকে কার্টে প্ল্যানের বিলিং পিরিয়ডের সাথে মেলাতে বাধ্য করা হয় না। এটি ডোমেইন রেজিস্ট্রেশনের মতো প্রোডাক্টগুলোর জন্য খুব দরকারি, যেগুলো প্ল্যানটি মাসিক হোক বা বার্ষিক, সবসময়ই বার্ষিক বিলিং করে।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | স্বাধীন বিলিং সাইকেল আছে এমন প্রোডাক্ট টাইপগুলোর অ্যারে। |
