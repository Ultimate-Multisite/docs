# Filter: wu_magic_link_enforce_user_agent

Filtrira, ali naj se uveljavi preverjanje uporabniškega agenta.

Nastavite na false, da žetonom omogočite delovanje v različnih brskalnikih/napravah. To zmanjša varnost, vendar poveča uporabnost.

## Parametri {#parameters}

| Ime | Vrsta | Opis |
|------|------|-------------|
| $enforce | `bool` | Ali naj se uveljavi ujemanje uporabniškega agenta. |
