# Filter: wu_username_from_email

Filtriraj uporabniško ime nove stranke.

## Parametri {#parameters}

| Ime | Vrsta | Opis |
|------|------|-------------|
| $username | `string` | Uporabniško ime stranke. |
| $email | `string` | E-poštni naslov nove stranke. |
| $new_user_args | `array` | Polje argumentov novega uporabnika, morda vključno z imenom in priimkom. |
| $suffix | `string` | Dodaj niz uporabniškemu imenu, da bo edinstveno. |
