# Filter: wu_plan_product_types

Filtrirajte vrste izdelkov, ki naj se obravnavajo kot načrti.

Ta filter omogoča dodatkom, da registrirajo dodatne vrste izdelkov, ki naj bodo prepoznane kot načrti pri preverjanju in ločevanju.

| $plan_types | `array` | Polje vrst izdelkov, ki se obravnavajo kot načrti. |
