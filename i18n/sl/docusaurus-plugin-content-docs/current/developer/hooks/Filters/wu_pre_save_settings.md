# Filter: wu_pre_save_settings {#filter-wupresavesettings}

Razvijalcem omogoči filtriranje nastavitev pred shranjevanjem z Ultimate Multisite.

## Parametri {#parameters}

| Ime | Vrsta | Opis |
|------|------|-------------|
| $settings | `array` | Nastavitve, ki bodo shranjene. |
| $settings_to_save | `array` | Nove nastavitve za dodajanje. |
| $saved_settings | `array` | Trenutno shranjene nastavitve. |
