# Dokumentacija Ultimate Multisite

Dobrodošli v dokumentaciji Ultimate Multisite. Ultimate Multisite pretvori WordPress Multisite v celovito platformo Website as a Service (WaaS).

## Razdelki dokumentacije

### [Uporabniški vodnik](/user-guide/getting-started/ultimate-multisite-101)

Vse, kar potrebujete za nastavitev, konfiguracijo in upravljanje svoje platforme WaaS. Zajema namestitev, plačilne prehode, preslikavo domen, upravljanje strank in več.

### [Dokumentacija za razvijalce](/developer/hooks)

Tehnična referenca za razvijalce, ki gradijo na Ultimate Multisite. Vključuje referenco hookov, dokumentacijo REST API in vodnike za razvoj dodatkov.

### [Dodatki](/addons)

Dokumentacija za plačljive dodatne plugine, ki razširjajo Ultimate Multisite z dodatnimi funkcijami, kot so integracija WooCommerce, analitika, prodaja domen in več.
