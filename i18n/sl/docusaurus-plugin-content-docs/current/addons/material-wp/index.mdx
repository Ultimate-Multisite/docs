# Material WP

WordPress skrbniški vmesnik je že dolgo enak. Toda ali ste se kdaj vprašali, kako bi bila videti različica skrbniškega vmesnika v slogu Material Design? 

Material WP je skrbniška tema, razvita zato, da vam zagotovi tak videz in občutek ter vam pomaga, da svojim strankam ponudite bolj prilagojen Dashboard.

## Funkcije {#features}

- Podpora za CSS po meri
- Preproste nastavitve oblikovanja
- Preurejanje menija
- Preprost uvoz/izvoz

## Dokumentacija {#documentation}

**Prilagajanje skrbniške plošče z Material WP**

## Namestitev {#installation}

1. Naložite 'material-wp' v imenik '/wp-content/plugins/'
2. Aktivirajte plugin prek menija 'Vtičniki' v WordPress

## Vodnik za uporabo {#usage-guide}

WordPress skrbniški vmesnik je že dolgo enak. Toda ali ste se kdaj vprašali, kako bi bila videti različica skrbniškega vmesnika v slogu Material Design?
