# Večnajemništvo

Porazdelite podmesta med strežnike in podatkovne zbirke za zmogljivost in varnost

## Poudarki izdaje različice 1.2.0 {#version-120-release-highlights}

Multi-Tenancy 1.2.0 dodaja temelj suverenega najemnika za poteke dela izolacije v produkciji:

- Brezstanje samodejno prijavljanje najemnika s kratkotrajnimi, pred ponovitvijo zaščitenimi obiski SSO.
- Izolacija podatkovne zbirke in datotečnega sistema po podmestu za suverene najemnike.
- Zmožnost večnajemništva Hostinger za zagotavljanje najemnikov z upoštevanjem gostitelja.
- Preveritvena vrata suverene migracije za preverjanja podedovanih podatkov in asinhrono obdelavo potiskanja.
- Nastavljiva vezava gostitelja podatkovne zbirke na istem računalniku za namestitve Bedrock, FrankenPHP in localhost.
- Izboljšave zagona najemnika, zapisovalnika podatkovne zbirke, zagona sheme in zagotavljanja uporabnikov.
- Varnejša odstranitev suverenega najemnika, ki počisti poverilnice najemnika.

Glejte uporabniške vodnike za [potek dela suverene migracije](/user-guide/getting-started/sovereign-migration-workflow), [izolacijo večnajemništva](/user-guide/configuration/multi-tenancy-isolation) in [operacije suverenega najemnika](/user-guide/administration/sovereign-tenant-operations).

## Zahteve {#requirements}

- WordPress 5.3 ali novejši
- PHP 7.4 ali novejši
- Vtičnik Multisite Ultimate (aktiven)

## Namestitev {#installation}

1. Naložite datoteke dodatka v imenik `/wp-content/plugins/`
2. Aktivirajte vtičnik prek menija »Plugins« v WordPress
3. Konfigurirajte nastavitve dodatka v skrbništvu omrežja

## Razvoj {#development}

### Nastavitev {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Testiranje {#testing}

Dodatek vključuje celovit testni okvir:

- **Enotni testi**: Testi na osnovi PHPUnit s testno zbirko WordPress
- **Osnovni testni razred**: Zagotavlja pomožne metode za pogoste testne scenarije
- **Pokritost testov**: Konfigurirano za spremljanje pokritosti kode
- **Testiranje Multisite**: Testi se izvajajo v okolju multisite

### Standardi kode {#code-standards}

- **WordPress Coding Standards**: Uveljavljeni prek PHPCS
- **Združljivost s PHP 7.4+**: Podprte so sodobne funkcije PHP
- **Statična analiza**: Integracija PHPStan za preverjanje tipov
- **Posodobitev kode**: Pravila Rector za nadgradnje PHP

### Sistem gradnje {#build-system}

- **Obdelava sredstev**: Minifikacija datotek CSS/JS
- **Prevajanje**: Ustvarjanje datoteke POT
- **Pakiranje**: Ustvarjanje arhiva za distribucijo
- **Razvojni način**: Neminificirana sredstva za odpravljanje napak

## Konfiguracija {#configuration}

Dodatek podpira različne možnosti konfiguracije, dostopne prek skrbniškega vmesnika omrežja.

### Konfiguracija suverenega najemnika {#sovereign-tenant-configuration}

Za suverene najemnike konfigurirajte podatkovno zbirko najemnika, koren datotečnega sistema, vezavo gostitelja podatkovne zbirke in zmožnost gostitelja, preden usmerite promet k najemniku. Po vsaki spremembi, ki vpliva na poverilnice podatkovne zbirke, vezavo gostitelja, usmerjanje ali zagotavljanje uporabnikov, zaženite preverjanje migracije.

Uporabite `localhost` samo, kadar izvajalno okolje najemnika in dovoljenja podatkovne zbirke pričakujejo lokalni vtičnik ali dostop z istega računalnika. Uporabite `127.0.0.1` ali ime gostitelja storitve, kadar je to gostitelj dovoljenj, ki ga je konfiguriral vaš ponudnik.

### Zmožnost Hostinger {#hostinger-capability}

Zmožnost večnajemništva Hostinger deluje s skupnim integracijskim žetonom Hostinger. Pred preverjanjem potrdite, da je domena prisotna v hPanel, da DNS kaže na Account Hostinger in da se nastavitve podatkovne zbirke najemnika ujemajo s hPanel.

## Hooki in filtri {#hooks-and-filters}

### Dejanja {#actions}

- `ultimate-multisite-addon-template_init` - Sproži se po inicializaciji dodatka
- `ultimate-multisite-addon-template_loaded` - Sproži se, ko so naložene vse datoteke dodatka

### Filtri {#filters}

- `ultimate-multisite-addon-template_settings` - Spremeni nastavitve dodatka
- `ultimate-multisite-addon-template_enabled` - Preglasi stanje omogočanja/onemogočanja dodatka
