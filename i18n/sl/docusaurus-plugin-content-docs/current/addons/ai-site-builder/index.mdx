# Graditelj AI spletnih mest {#ai-site-builder}

ultimate-multisite-ai-site-builder

## Zahteve {#requirements}

- WordPress 5.3 ali novejši
- PHP 7.4 ali novejši
- vtičnik Multisite Ultimate (aktiven)

## Namestitev {#installation}

1. Naložite datoteke dodatka v svoj imenik `/wp-content/plugins/`
2. Aktivirajte vtičnik prek menija 'Plugins' v WordPress
3. Konfigurirajte nastavitve dodatka v skrbništvu omrežja

## Razvoj {#development}

### Nastavitev {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Struktura projekta {#project-structure}

```
ultimate-multisite-ai-site-builder/
├── inc/                          # PHP razredi in vključitve
│   ├── class-multisite-ultimate-updater.php  # Upravljalnik posodobitev
│   └── class-ai_site_builder-main.php             # Glavna funkcionalnost
├── tests/                        # Enotni testi
│   ├── bootstrap.php            # Zagon testov
│   ├── class-*-base.php         # Osnovni testni razred
│   └── class-*-test.php         # Testni primeri
├── views/                       # Datoteke predlog
├── assets/                      # Statična sredstva
│   ├── css/                     # Slogovne predloge
│   ├── js/                      # Datoteke JavaScript
│   └── images/                  # Slike
├── lang/                        # Jezikovne datoteke
├── composer.json                # PHP odvisnosti
├── package.json                 # Node.js odvisnosti
├── phpunit.xml.dist             # Konfiguracija PHPUnit
├── .phpcs.xml.dist              # Konfiguracija standardov kode
└── rector.php                   # Pravila za posodobitev kode
```

### Testiranje {#testing}

Dodatek vključuje celovit okvir za testiranje:

- **Enotni testi**: testi na osnovi PHPUnit s testno zbirko WordPress
- **Osnovni testni razred**: zagotavlja pomožne metode za pogoste testne scenarije
- **Pokritost testov**: konfigurirano za sledenje pokritosti kode
- **Multisite testiranje**: testi se izvajajo v okolju multisite

### Standardi kode {#code-standards}

- **Standardi kodiranja WordPress**: uveljavljeni prek PHPCS
- **Združljivost s PHP 7.4+**: podprte so sodobne funkcije PHP
- **Statična analiza**: integracija PHPStan za preverjanje tipov
- **Posodobitev kode**: pravila Rector za nadgradnje PHP

### Sistem gradnje {#build-system}

- **Obdelava sredstev**: pomanjšanje datotek CSS/JS
- **Prevajanje**: ustvarjanje datoteke POT
- **Pakiranje**: ustvarjanje arhiva za distribucijo
- **Razvojni način**: nepomanjšana sredstva za odpravljanje napak

## Konfiguracija {#configuration}

Dodatek podpira različne konfiguracijske možnosti, dostopne prek skrbniškega vmesnika omrežja.

## Kljuke in filtri {#hooks-and-filters}

### Dejanja {#actions}

- `ultimate-multisite-ai-site-builder_init` - Sproži se po inicializaciji dodatka
- `ultimate-multisite-ai-site-builder_loaded` - Sproži se, ko so naložene vse datoteke dodatka

### Filtri {#filters}

- `ultimate-multisite-ai-site-builder_settings` - Spremenite nastavitve dodatka
- `ultimate-multisite-ai-site-builder_enabled` - Preglasite stanje omogočenosti/onemogočenosti dodatka
