# Agenti za podporo

Upravljanje celotnega omrežja samostojno ni lahka naloga. Ko se vaše omrežje začne širiti, boste začeli opažati vse večje povpraševanje po podpori. Dobro je vključiti ljudi, ki vam bodo pomagali zmanjšati delovno obremenitev. Vendar je podelitev polnega dostopa/dovoljenj tvegana in lahko ogrozi vaše omrežje. Tu nastopi ta dodatek za agente za podporo.

Ta dodatek vam omogoča, da ustvarite novo vrsto WordPress uporabnika, imenovanega agent za podporo, in vam daje popoln nadzor nad tem, katera dovoljenja ima ta uporabnik. Dodeljevanje dovoljenj je tako preprosto kot vklop preklopnega gumba, izklop pa prav tako, če se odločite dovoljenje odstraniti.

## Funkcije {#features}

- Razčlenjen seznam dovoljenj za WordPress in Ultimate Multisite
- Preprosto omogočanje in onemogočanje dostopa
- Ustvarite neomejeno število računov agentov

## Namestitev {#installation}

1. Naložite 'ultimate-multisite-support-agents' v imenik '/wp-content/plugins/'
2. Aktivirajte vtičnik prek menija 'Vtičniki' v WordPress
3. Sledite čarovniku korak za korakom, da nastavite vtičnik
