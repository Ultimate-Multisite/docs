# Dnevnik sprememb Multi-Network {#multi-network-changelog}

### 1.3.0 {#130}
* Novo: Predogledovalnik omrežnih predlog — brskajte po omrežnih predlogah in si jih oglejte v predogledu z živim panelom pred nakupom.
* Novo: Izbirnik omrežne predloge ob blagajni — stranke lahko med potekom prijavne blagajne izberejo omrežno predlogo.
* Novo: SSO z magic link med omrežji — stranke lahko nemoteno dostopajo do podomrežij prek preverjanja pristnosti z magic-link.
* Novo: Kloniranje omrežnih medijev — mediji spletnega mesta se pravilno podvojijo pri kloniranju omrežij iz predloge.
* Popravek: Kloniranje omrežij zdaj deluje brez dodatka za večnajemništvo; pravice super admin se po kloniranju ponovno dodelijo.
* Popravek: Ustvarjanje omrežja je zdaj združljivo z WooCommerce in vsemi plačilnimi prehodi.
* Popravek: Številka vrat je zdaj vključena v ustvarjanje nadomestne domene.
* Popravek: CSS izbirnika predlog je premaknjen v uvrščen slogovni list za pravilno prikazovanje v vseh temah.
* Popravek: Zaščita pred nedefiniranim ključem `network_id` v obsegu poizvedbe spletnega mesta.
* Popravek: Preverjanje dovoljenj omrežja pred preklopom konteksta preprečuje napake lastnosti tabele `wpdb`.
