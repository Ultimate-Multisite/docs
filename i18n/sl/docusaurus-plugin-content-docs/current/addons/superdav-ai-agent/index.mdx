# Superdav AI Agent

Superdav AI Agent je napreden WordPress AI pomočnik, ki razširi Gratis AI Agent s spretnostmi, prilagojenimi temi, izboljšanimi zmožnostmi internetnega iskanja, funkcijami gradnika spletnih mest in izbirnimi integracijami za avtomatizacijo. Različica 1.18.0 dodaja orodja za branje Google Calendar, preslikavo stikov in udeležencev, potrditvene korake s človeško odobritvijo, odstranjevanje podvojenih opomnikov, TextBee SMS obvestila, zagotavljanje upravljane storitve Superdav in ločeno distribuiran spremljevalni paket Advanced za zaupanja vredne razvijalske poteke dela.

## Ključne funkcije {#key-features}

- **Spretnosti, prilagojene temi** — samodejno prilagaja navodila in zmožnosti glede na vašo aktivno WordPress temo in nameščene vtičnike
- **Več ponudnikov internetnega iskanja** — izberite med Brave Search in Tavily za bogatejše rezultate iskanja in napredne raziskovalne zmožnosti
- **Kontaktni obrazci gradnika spletnih mest** — ustvarite popolnoma delujoče kontaktne obrazce neposredno iz vmesnika klepeta z uporabo nameščenega vtičnika za obrazce
- **Izboljšan gradnik spletnih mest** — izboljšani večfazni načrti gradnje spletnega mesta z boljšim odkrivanjem vtičnikov in obnovitvijo po napakah
- **Vgrajene spretnosti za teme** — specializirana navodila za Block Themes, Classic Themes, Kadence Blocks in Kadence Theme
- **Izboljšana integracija WooCommerce** — preoblikovana za uporabo izvornih WooCommerce API-jev za boljšo zanesljivost in združljivost
- **Samodejno zaznavanje ponudnikov** — samodejno zazna in osveži razpoložljive vtičnike in teme, ko so aktivirani ali deaktivirani
- **Ustvarjanje logotipa po meri** — ustvarite in vdelajte SVG logotipe po meri s samodejnim čiščenjem, varnim za imenski prostor
- **Preverjanje kontrasta palete** — preverite barvne pare glede skladnosti z dostopnostjo WCAG, preden jih uporabite v svoji temi
- **Odkrivanje, izboljšano s fotografijami** — naložite fotografije med odkrivalnim intervjujem v Theme Builder, da vplivate na oblikovno usmeritev
- **Gostinski meniji** — ustvarite strukturirane strani menijev s hrano in pijačo za restavracije, kavarne in gostinska podjetja
- **Predogledi za namizje in mobilne naprave** — predoglejte si svojo zasnovo na namiznih in mobilnih napravah med izbiro oblikovne usmeritve
- **Strganje spletnih mest Tier 1** — sd-ai-agent/site-scrape je zdaj privzeto na voljo za analizo konkurenčnih in navdihujočih spletnih mest
- **Orodja za branje Google Calendar** — berite konfigurirane koledarje in dogodke, da lahko avtomatizacije sklepajo o kontekstu prihajajočega urnika
- **Preslikava stikov in udeležencev** — pred ustvarjanjem nadaljnjega dela povežite udeležence koledarja z znanimi uporabniki in stiki spletnega mesta
- **Potrditveni koraki s človeško odobritvijo** — začasno zaustavite občutljive avtomatizacije za pregled pred pošiljanjem sporočil, nadaljevanjem dela ali izvajanjem sprememb
- **Zapisi opomnikov in odstranjevanje podvojitev** — sledite opomnikom, ki so že bili poslani, da ponovljeni zagoni avtomatizacije istega prejemnika ne obvestijo dvakrat
- **TextBee SMS obvestila** — pošiljajte konfigurirana besedilna obvestila prek TextBee, ko je omogočen ponudnik SMS
- **Zagotavljanje upravljane storitve Superdav** — povežite podprta spletna mesta z gostovanimi Superdav končnimi točkami s samodejno nastavitvijo namesto ročne konfiguracije končne točke
- **Spremljevalni paket Advanced** — ločeno distribuirana orodja za zaupanja vredne razvijalce za datotečni sistem, podatkovno zbirko, WP-CLI, REST dispečer, gradnik vtičnikov, git posnetek, upravljanje uporabnikov in primerjalne delovne poteke

## Zahteve {#requirements}

- WordPress 6.0 ali novejši
- PHP 8.1 ali novejši
- WordPress Multisite (aktiviran v omrežju) ali namestitev za eno spletno mesto
- Aktiven vtičnik za povezavo s ponudnikom AI (Anthropic, OpenAI ali združljiv)
- Gratis AI Agent v1.9.0 ali novejši (Superdav AI Agent razširi Gratis AI Agent)

## Namestitev {#installation}

1. Naložite datoteke vtičnika v `/wp-content/plugins/superdav-ai-agent`
2. Aktivirajte vtičnik prek zaslona **Vtičniki** v WordPress
3. Prepričajte se, da je aktiviran tudi Gratis AI Agent
4. Pojdite na **Gratis AI Agent → Nastavitve** in po potrebi konfigurirajte ponudnika internetnega iskanja, povezavo z upravljano storitvijo Superdav, Google Calendar ali poverilnice TextBee

Paket WordPress.org vsebuje osnovne funkcije Superdav AI Agent. Spremljevalni paket Advanced je zgrajen in distribuiran ločeno za okolja, kjer so zaupanja vredna razvijalska orodja izrecno nameščena in odobrena.


```
Preglej jutrišnje konfigurirane dogodke Google Calendar in pripravi osnutke opomnikov za udeležence.
```

```
Pošlji SMS opomnik, ko odobrim sporočilo.
```

## Spretnosti, prilagojene temi {#theme-aware-skills}

Superdav AI Agent samodejno zazna vašo aktivno temo in nameščene vtičnike ter naloži ustrezno spretnost, prilagojeno temi, v kontekst pomočnika. Razpoložljive spretnosti vključujejo:

| Spretnost | Velja za |
|---|---|
| **Block Themes** | Teme, ki uporabljajo WordPress blokovno arhitekturo (Full Site Editing) |
| **Classic Themes** | Tradicionalne WordPress teme, ki uporabljajo PHP predloge in klasični urejevalnik |
| **Kadence Blocks** | Spletna mesta z aktivnim vtičnikom Kadence Blocks |
| **Kadence Theme** | Spletna mesta s temo Kadence kot aktivno temo |

Za podrobna navodila za vsako spretnost glejte [Spretnosti, prilagojene temi](../../user-guide/configuration/theme-aware-skills).

## Ponudniki internetnega iskanja {#internet-search-providers}

Superdav AI Agent v1.10.0 dodaja **Tavily** kot alternativo Brave Search za poizvedbe internetnega iskanja. Oba ponudnika ponujata rezultate iskanja v realnem času in ju je mogoče konfigurirati v **Gratis AI Agent → Nastavitve → Napredno**.

| Ponudnik | Funkcije | Nastavitev |
|---|---|---|
| **Brave Search** | Hitri rezultati iskanja s poudarkom na zasebnosti | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Bogatejši rezultati iskanja, napredne raziskovalne zmožnosti | [Tavily API](https://tavily.com/) |

Za navodila za konfiguracijo glejte [Internetno iskanje](../../user-guide/configuration/internet-search).

## Poteki dela za koledar, odobritve in obvestila {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 razširja poteke avtomatizacije onkraj opravil vsebine in gradnje spletnega mesta:

- **Orodja Google Calendar** agentu omogočajo branje konfiguriranih koledarjev in podrobnosti dogodkov, ko so na voljo poverilnice za koledar.
- **Pomočniki za stike in udeležence** preslikajo udeležence dogodkov v uporabnike WordPress ali znane stike, preden pripravijo opomnike ali nadaljnja opravila.
- **Vrata za človeško odobritev** omogočajo, da se avtomatizacija zaustavi, dokler pooblaščeni uporabnik ne pregleda predlaganega dejanja.
- **Zapisi opomnikov** preprečujejo podvojene koledarske opomnike ali obvestila, ko se avtomatizacija ponovi ali večkrat načrtuje.
- **Podpora za SMS TextBee** omogoča obvestila prek besedilnih sporočil, ko je ponudnik TextBee konfiguriran.

Ta orodja uporabite za pomoč, ki upošteva urnik, nadaljnje ukrepanje z udeleženci in poteke dela z opomniki, ki še vedno zahtevajo človeški pregled, preden se pošljejo občutljiva sporočila.

## Konfiguracija {#configuration}

Vse nastavitve so v **Gratis AI Agent → Nastavitve** v skrbništvu WordPress. Superdav AI Agent doda naslednje možnosti konfiguracije:

| Nastavitev | Opis |
|---|---|
| **Ponudnik internetnega iskanja** | Izberite med Brave Search in Tavily (neobvezno) |
| **Tavily API Key** | API ključ za internetno iskanje Tavily (če je izbran Tavily) |
| **Upravljana storitev Superdav** | Povežite podprta spletna mesta z gostovano storitvijo Superdav s samodejnim zagotavljanjem endpointov |
| **Poverilnice Google Calendar** | Konfigurirajte dostop do koledarja, da lahko agent bere koledarje in dogodke |
| **Ponudnik SMS TextBee** | Konfigurirajte poverilnice TextBee za odobrene poteke dela z obvestili SMS |

## Napredni spremljevalni paket {#advanced-companion-package}

Superdav AI Agent v1.18.0 dodaja ločeno distribuiran napredni spremljevalni paket za zaupanja vredne poteke dela razvijalcev. Te zmožnosti niso vključene v osnovni paket WordPress.org. Spremljevalni paket namestite samo na spletna mesta, kjer skrbniki izrecno zaupajo agentu dostop do razvijalskih orodij.

Napredna orodja vključujejo operacije datotečnega sistema, dostop do podatkovne zbirke, WP-CLI, poteke dela REST dispatcher, pripomočke za izdelavo pluginov, posnetke git, pomočnike za upravljanje uporabnikov in orodja za primerjalno testiranje. Ta orodja omejite na zaupanja vredne skrbnike in pred omogočanjem na produkcijskih spletnih mestih preglejte konfiguracijo vrat za odobritev.

- Popravljeni WP-CLI ukazi, pri katerih so po prejšnji prenovi manjkali vzdevki imenskega prostora
- Popravljena avtomatizacija dogodkov, ki zdaj elegantno obravnava spletna mesta, kjer tabele za avtomatizacijo še niso ustvarjene
- Popravljena zmožnost `memory-save`, ki uporablja pravilno predpono imenskega prostora v graditelju sistemskih navodil
- Popravljeni skalarni rezultati orodij, ki so pravilno oviti, preden so vrnjeni AI-ju
- Popravljena statistika uporabe, ki ob nadgradnji iz starejših različic pravilno obravnava podedovano obliko ključa zmožnosti
