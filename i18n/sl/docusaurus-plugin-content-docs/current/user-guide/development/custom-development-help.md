# Kako dobiti prilagočeno razvojno pomoč od Codeable? {#how-to-get-custom-development-help-from-codeable}

Ponekad nam je potrebna prava prilagodba, da bi naši mreže tekčno te same kako želimo. Če ste to iščete, priporočamo [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), mesto kjer lahko najdete izkušene razvijalce s posebnim znanjištem o Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) je platforma ki povezuje stranke z kvalificiranimi WordPress freelancami, ki so sposobni delati projekte v vsakih velikostih. Imajo natančen proces prijav, da se osigurate, da ste pravilno povezani z eksperti, ki so specifično kvalificirani za zadatek, ki ga imate na umu.

### Kako ustvariti projekt v Codeable? {#how-to-create-a-project-in-codeable}

Prvi korak je obiščite [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) in kliknite gumb **Start a Project** (Začetek projekta).

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Zdaj, za predajanje projekta, kliknite na levi strani **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

V polju "I need help with" (Potrebujem pomoč z) izberite **Customizing** (Prilagočevanje):

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

V polju "My" (Moj) izberite **Plugin** (Dodatek):

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Potem kliknite na gumb **Continue** (Prehod).

<!-- Screenshot unavailable: Codeable project form Continue button -->

Zdaj je čas opisati svoj projekt in potrebe najbolj načinom, kako lahko. Poskusite dati čim več informacij!

<!-- Screenshot unavailable: Codeable project description form -->

Odgovorite na naslednja vprašanja za pomoč pri določanju cene.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Po tem, če še ni imate računa [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), ga ustvarite in kliknite na gumb **Publish My Project** (Objavi moj projekt).

<!-- Skenarno sliko ni na voljo: ustvarjanje računa Codeable in gumb Publish My Project -->

Ko boste objavili svoj [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) projekt, vas bo platforma povezala z razvijalci, ki imajo veščine, ki so potrebne za vaš projekt.

Ta proces vam bo pomagal najti pravo osebo za vašo delo, ki ima izkušanje s WordPressom in Ultimate Multisite!

Minimalna cena Codeable je 70–120 USD na uro. Ocenilna vrednost se bo spremenila v skladu z obsegom dela, kompleksno in nujnostjo projekta.

Ko boste dobili ocijen, boste lahko najeli razvijalca, ki najbolj ustreza vašim potrebam in začnete projekt skupaj!
