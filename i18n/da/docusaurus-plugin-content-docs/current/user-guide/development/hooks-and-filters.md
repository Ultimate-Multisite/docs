# Ultimate Multisite Hooks og filtre

Ultimate Multisite består af en hel masse funktioner og databaseforespørgsler. Det har hooks og filtre, som giver dig mulighed for at ændre eller tilføje ny kode til de eksisterende funktioner.

Vi har lavet en liste over disse hooks og filtre som en reference til at hjælpe avancerede brugere med yderligere at tilpasse deres netværk. Du kan få adgang til listen på Github ved at klikke på linket nedenfor.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
