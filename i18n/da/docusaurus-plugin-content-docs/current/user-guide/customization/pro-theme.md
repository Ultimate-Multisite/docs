# Tilpasning af dit Admin Panel med PRO Theme {#customizing-your-admin-panel-with-pro-theme}

Ofte når vi arbejder med kunder har vi brug for at give vores kunder en skræddersyet oplevelse i WordPress-administratoren. Hvis du leder efter et mere professionelt udseende og følelse, kan PRO Theme hjælpe dig.

PRO Theme er et admin tema udviklet til at hjælpe dig med at tilpasse dine kunders dashboards. Det er også en del af kernen i Ultimate Multisite’s premium plugins.

Efter installation skal du gå til din _Plugins_ side og klikke for at aktivere den på dit Network.

<!-- Skærmbillede utilgængeligt: WP Plugins side med Aktiver Network-knap for PRO Theme -->

_Aktiver det på hele netværket på din Plugins side_

_**OBS:**_ _Hvis du blot installerer det, vil det vise sig som en mulighed hos din kunde, der kan markeres til kun at tilføje temaet til den specifikke bruger._

Nu har du dit admin interface fuldstændig redesignet! Du får også en ny menu i venstre kolonne kaldet PRO Theme.

<!-- Skærmbillede utilgængeligt: Redesignet admin panel interface for PRO Theme -->

_Et helt nyt look til dit Admin Panel_

På denne menu vil du have adgang til fem faner:

<!-- Skærmbillede utilgængeligt: Styling Settings tab for PRO Theme -->

_Styling Settings_ (Stylingindstillinger)

<!-- Skærmbillede utilgængeligt: Functionality tab for PRO Theme -->

_Functionality_ (Funktionalitet)

<!-- Skærmbillede utilgængeligt: Custom CSS tab for PRO Theme -->

_Custom CSS_ (Individuel CSS)

<!-- Skærmbillede utilgængeligt: Import/Export tab for PRO Theme -->

_Import/Export_ (Importer/Eksport)

<!-- Skærmbillede utilgængeligt: Activate tab for PRO Theme -->

_Activate PRO Theme_ (Aktiver PRO Theme)

I hver af dem kan du aktivere og deaktivere stylingelementer, tillade eller blokere funktionaliteter og give dine kunder den tilpasning, de måske leder efter i dit netværk af sider.
