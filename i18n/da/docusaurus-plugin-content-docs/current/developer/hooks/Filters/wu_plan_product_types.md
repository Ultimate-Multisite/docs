# Filter: wu_plan_product_types {#filter-wuplanproducttypes}

Filtrer de produkttyper, der skal behandles som planer.

Dette filter gør det muligt for addons at registrere yderligere produkttyper, der skal genkendes som planer i validering og adskillelse.

## Parametre {#parameters}

| Navn | Type | Beskrivelse |
|------|------|-------------|
| $plan_types | `array` | Array af produkttyper, der skal behandles som planer. |

### Siden {#since}

- 2.3.0
