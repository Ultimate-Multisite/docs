# Filter: wu_independent_billing_cycle_product_types {#filter-wuindependentbillingcycleproducttypes}

Filtrer de produkttyper, der har uafhængige faktureringscyklusser.

Produkter med uafhængige faktureringscyklusser er ikke tvunget til at matche abonnementets faktureringsperiode i kurven. Dette er nyttigt for produkter som domæneregistreringer, der altid faktureres årligt, uanset om abonnementet er månedligt eller årligt.

## Parametre {#parameters}

| Navn | Type | Beskrivelse |
|------|------|-------------|
| $types | `array` | Array af produkttyper med uafhængige faktureringscyklusser. |
