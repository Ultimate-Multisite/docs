# Filter: secure_auth_cookie

Filtrerer, om auth cookie kun skal sendes over HTTPS.

## Parametre

| Navn | Type | Beskrivelse |
|------|------|-------------|
| $secure | `bool` | Om cookie kun skal sendes over HTTPS. |
| $user_id | `int` | Bruger-ID. |
