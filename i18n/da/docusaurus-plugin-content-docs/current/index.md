# Ultimate Multisite Dokumentation

Velkommen til dokumentationen for Ultimate Multisite. Ultimate Multisite transformerer WordPress Multisite til en fuld Website as a Service (WaaS) platform.

## Dokumentationssektioner {#documentation-sections}

### [Brugerguide](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

Alt, du har brug for at opsætte, konfigurere og administrere din WaaS platform. Dækker installation, betalingsgateways, domænemapping, klientadministration og mere.

### [Udviklerdokumentation](/developer/hooks) {#developer-documentation}

Teknisk reference til udviklere, der bygger oven på Ultimate Multisite. Indeholder reference til hooks, REST API-dokumentation og vejledninger til addon-udvikling.

### [Tilføjelser](/addons) {#addons}

Dokumentation for betalte addon plugins, der udvider Ultimate Multisite med yderligere funktioner som WooCommerce integration, analytics, domæne salg og mere.
