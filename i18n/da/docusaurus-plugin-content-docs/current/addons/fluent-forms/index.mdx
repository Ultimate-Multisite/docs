# Fluent Forms

Dette add-on gør det muligt at sætte grænser for Fluent Forms på Ultimate Multisite i produkter, medlemskaber eller sider.
Fluent Forms bruger specialskræddersyede tabeller i stedet for custom post types, så de normale begrænsningsmuligheder ikke virker med Fluent Forms.

## Funktioner

- Tilføj grænser for Fluent Forms eller Conversational Fluent Forms
- Giver kunder en venlig besked, der opfordrer dem til at opgradere deres plan, når de når grænsen
