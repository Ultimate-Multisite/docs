# Supporthenvendelser {#support-tickets}

**Supporthenvendelser** er en kraftfuld addon til Ultimate Multisite, der tilføjer et omfattende system til supporthenvendelser til dit WordPress-netværk. Perfekt til SaaS-platforme, WaaS-udbydere (Website as a Service) og multisite-netværk, der skal levere kundesupport både på netværksniveau og for individuelle websteder.

## Hvorfor vælge supporthenvendelser? {#why-choose-support-tickets}

* **Support på flere niveauer**: Netværksadministratorer håndterer henvendelser på tværs af netværket, mens webstedsejere administrerer deres egen kundesupport
* **Professionel arbejdsgang**: Komplet styring af henvendelsers livscyklus med prioriteter, statusser og tildelinger
* **Kundeoplevelse**: Smuk, responsiv grænseflade, der gør det nemt for kunder at få hjælp
* **Sikker og pålidelig**: Bygget med bedste praksis for WordPress-sikkerhed og optimeret databaseydelse
* **Problemfri integration**: Fungerer perfekt med Ultimate Multisites kunde- og webstedsstyring

## Nøglefunktioner {#key-features}

**Komplet håndtering af henvendelser**

* Unikt nummereringssystem til henvendelser for nem reference
* Konfigurerbare prioritetsniveauer (Lav, Normal, Høj, Kritisk)
* Omfattende statusarbejdsgang (Åben, I gang, Afventer, Løst, Lukket)
* Kategorisering af henvendelser (Generelt, Teknisk, Fakturering, Funktionsanmodning, Fejlrapport)
* Tildel henvendelser til supportagenter med valgfri automatisk tildeling

**Trådede samtaler**

* Fuld samtaletrådning for svar på henvendelser
* Tydelig styling for medarbejder- og kundesvar
* Private medarbejdernoter, som kunder ikke kan se
* Realtidsgrænseflade drevet af AJAX
* Moderne teksteditor med formateringsmuligheder

**Filvedhæftninger**

* Sikker filupload med træk-og-slip-understøttelse
* Konfigurerbare begrænsninger for filtype og størrelse
* Avanceret sikkerhed med registrering af skadelige filer
* Beskyttede downloads med adgangskontroller

**Smarte notifikationer**

* Automatiske e-mailadvarsler for nye henvendelser, svar og statusændringer
* Tilpasningsbare e-mailskabeloner
* Rollebaserede notifikationer til medarbejdere og kunder
* Dynamiske skabelonvariabler til personlige beskeder

**Kraftfuld admingrænseflade**

* Avanceret henvendelsesliste med sortering og filtrering
* Hurtig redigering af status, prioritet og tildelinger
* Massehandlinger til effektiv administration
* Statistikdashboard med ydelsesmålinger
* Omfattende indstillingspanel

**Smuk kundeportal**

* Ren, responsiv grænseflade på alle enheder
* Mobil-først-design
* Avanceret søgning og filtrering
* Moderne editor med formateringsværktøjslinje
* Brugervenlige formularer til indsendelse af henvendelser

**Fleksibel integration**

* Brug shortcodes eller Ultimate Multisite UI-elementer
* Vis henvendelsesformularer og lister hvor som helst på dit websted
* Fungerer problemfrit med multisite-netværksstrukturen
* Kan udvides med hooks og filtre

## Perfekt til {#perfect-for}

* SaaS-platforme, der har brug for kundesupport
* WaaS-udbydere, der administrerer flere kunders websteder
* Medlemswebsteder med niveaudelt support
* Netværksadministratorer, der administrerer flere brands
* Webstedsejere, der leverer kundeservice

## Installation {#installation}

1. Sørg for, at Ultimate Multisite plugin er installeret og aktiveret på dit WordPress-netværk
2. Upload addon-filerne til din `/wp-content/plugins/`-mappe
3. Netværksaktivér plugin gennem menuen 'Plugins' i WordPress
4. Gå til Network Admin > Supporthenvendelser for at konfigurere dine indstillinger

## Ofte stillede spørgsmål {#frequently-asked-questions}

## Kræver dette Ultimate Multisite? {#does-this-require-ultimate-multisite}

Ja, denne addon kræver, at Ultimate Multisite er installeret og aktiv. Den integrerer dybt med Ultimate Multisites funktioner til kunde- og webstedsstyring.

## Kan webstedsejere håndtere deres egen kundesupport? {#can-site-owners-handle-their-own-customer-support}

Absolut! Webstedsejere kan bruge henvendelsessystemet til at levere support til deres egne kunder, mens netværksadministratorer kan håndtere supporthenvendelser på tværs af netværket.

## Hvordan indsender kunder henvendelser? {#how-do-customers-submit-tickets}

Du kan tilføje formularer til indsendelse af henvendelser hvor som helst på dit websted ved hjælp af shortcodes eller Ultimate Multisite UI-elementer. Kunder udfylder blot formularen og indsender deres anmodning.

## Kan jeg tilpasse henvendelsesstatusser og prioriteter? {#can-i-customize-the-ticket-statuses-and-priorities}

Ja! Du kan tilpasse prioritetsniveauer, statusser og henvendelsestyper fuldt ud i netværkets administratorindstillinger, så de matcher din supportarbejdsgang.

## Understøtter det filvedhæftninger? {#does-it-support-file-attachments}

Ja, både kunder og medarbejdere kan vedhæfte filer til henvendelser og svar. Du har fuld kontrol over tilladte filtyper og størrelsesgrænser af hensyn til sikkerhed.

## Hvordan håndteres notifikationer? {#how-are-notifications-handled}

Systemet sender automatisk e-mailnotifikationer for nye henvendelser, svar og statusændringer. Du kan tilpasse alle e-mailskabeloner i indstillingerne.

## Kan jeg tildele henvendelser til specifikke supportagenter? {#can-i-assign-tickets-to-specific-support-agents}

Ja, du kan manuelt tildele henvendelser til agenter eller aktivere automatisk tildeling for at fordele henvendelser automatisk.

## Findes der private noter til medarbejdere? {#are-there-private-notes-for-staff}

Ja, medarbejdere kan tilføje private svar, der kun er synlige for andre medarbejdere, ikke for kunder.

## Er det mobilvenligt? {#is-it-mobile-friendly}

Absolut! Hele grænsefladen er bygget med en mobil-først-tilgang og fungerer smukt på alle enheder.

## Kan jeg vise henvendelseslister og formularer ved hjælp af shortcodes? {#can-i-display-ticket-lists-and-forms-using-shortcodes}

Ja! Tilgængelige shortcodes:
* `[wu_support_tickets]` - Vis kundens henvendelsesliste
* `[wu_support_ticket_form]` - Vis formular til indsendelse af ny henvendelse
* `[wu_support_ticket_view ticket_id="123"]` - Vis specifik henvendelse

## Hvilke filtyper kan uploades? {#what-file-types-can-be-uploaded}

Du kan konfigurere tilladte filtyper i indstillingerne. Systemet inkluderer avanceret sikkerhedsscanning for at beskytte mod skadelige filer.

## Kan jeg eksportere henvendelsesdata? {#can-i-export-ticket-data}

Ja, addon inkluderer eksportfunktioner til henvendelsesdata og statistik.
