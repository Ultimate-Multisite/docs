# Filter: wu_domain_seller_search_delay

Filtrerer debounce-forsinkelsen for domænesøgning i millisekunder.

Øg denne værdi for at reducere API-kald på langsomme forbindelser.

## Parametre

| Navn | Type | Beskrivelse |
|------|------|-------------|
| $delay | `int` | Debounce-forsinkelse i millisekunder. Standard 500. |
