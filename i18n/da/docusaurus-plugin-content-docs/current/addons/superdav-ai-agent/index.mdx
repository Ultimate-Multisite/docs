# Superdav AI Agent {#superdav-ai-agent}

Superdav AI Agent er en avanceret WordPress AI-assistent, der udvider Gratis AI Agent med temabevidste færdigheder, forbedrede internetsøgningsfunktioner, funktioner til webstedsbygning og valgfrie automatiseringsintegrationer. Version 1.18.0 tilføjer Google Calendar-læseværktøjer, kontakt- og deltagermapping, menneskelige godkendelsesporte, deduplikering af påmindelser, TextBee SMS-notifikationer, klargøring af administreret Superdav-tjeneste og en separat distribueret Advanced-ledsagerpakke til betroede udviklerarbejdsgange.

## Nøglefunktioner {#key-features}

- **Temabevidste færdigheder** — tilpasser automatisk vejledning og funktioner baseret på dit aktive WordPress-tema og installerede plugins
- **Flere internetsøgningsudbydere** — vælg mellem Brave Search og Tavily for rigere søgeresultater og avancerede researchfunktioner
- **Kontaktformularer til webstedsbygning** — opret fuldt funktionelle kontaktformularer direkte fra chatgrænsefladen ved hjælp af dit installerede formular-plugin
- **Forbedret webstedsbygger** — forbedrede flerfasede planer for webstedsopbygning med bedre plugin-opdagelse og fejlgendannelse
- **Indbyggede temafærdigheder** — specialiseret vejledning til Block Themes, Classic Themes, Kadence Blocks og Kadence Theme
- **Forbedret WooCommerce-integration** — refaktoreret til at bruge native WooCommerce-API'er for bedre driftssikkerhed og kompatibilitet
- **Automatisk udbyderregistrering** — registrerer og opdaterer automatisk tilgængelige plugins og temaer, når de aktiveres eller deaktiveres
- **Generering af brugerdefineret logo** — generér og indlejr brugerdefinerede SVG-logoer med automatisk namespace-sikker sanitisering
- **Validering af paletkontrast** — kontrollér farvepar for overholdelse af WCAG-tilgængelighed, før de anvendes på dit tema
- **Fotoforbedret opdagelse** — upload fotos under Theme Builder-opdagelsesinterviewet for at påvirke designretningen
- **Hospitality-menuer** — generér strukturerede sider med mad- og drikkemenuer til restauranter, caféer og hospitality-virksomheder
- **Forhåndsvisninger på desktop og mobil** — forhåndsvis dit design på desktop- og mobilenheder under valg af designretning
- **Tier 1-webstedsscraping** — sd-ai-agent/site-scrape er nu tilgængelig som standard til analyse af konkurrent- og inspirationswebsteder
- **Google Calendar-læseværktøjer** — læs konfigurerede kalendere og begivenheder, så automatiseringer kan ræsonnere om kommende planlægningskontekst
- **Kontakt- og deltagermapping** — match kalenderdeltagere med kendte webstedsbrugere og kontakter, før opfølgningsarbejde genereres
- **Menneskelige godkendelsesporte** — sæt følsomme automatiseringer på pause til gennemgang, før der sendes beskeder, arbejde genoptages, eller ændringer foretages
- **Påmindelsesposter og deduplikering** — spor påmindelser, der allerede er sendt, så gentagne automatiseringskørsler ikke underretter den samme modtager to gange
- **TextBee SMS-notifikationer** — send konfigurerede tekstbeskednotifikationer via TextBee, når en SMS-udbyder er aktiveret
- **Klargøring af administreret Superdav-tjeneste** — forbind understøttede websteder til hostede Superdav-endpoints med automatisk opsætning i stedet for manuel endpoint-konfiguration
- **Advanced-ledsagerpakke** — separat distribuerede betroede udviklerværktøjer til filsystem-, database-, WP-CLI-, REST-dispatcher-, plugin-bygger-, git-snapshot-, brugeradministrations- og benchmarkarbejdsgange

## Krav {#requirements}

- WordPress 6.0 eller nyere
- PHP 8.1 eller nyere
- WordPress Multisite (netværksaktiveret) eller single-site-installation
- Et aktivt AI-udbyderconnector-plugin (Anthropic, OpenAI eller kompatibelt)
- Gratis AI Agent v1.9.0 eller nyere (Superdav AI Agent udvider Gratis AI Agent)

## Installation {#installation}

1. Upload plugin-filerne til `/wp-content/plugins/superdav-ai-agent`
2. Aktivér plugin'et via skærmen **Plugins** i WordPress
3. Sørg for, at Gratis AI Agent også er aktiveret
4. Gå til **Gratis AI Agent → Settings**, og konfigurer din internetsøgningsudbyder, forbindelse til administreret Superdav-tjeneste, Google Calendar eller TextBee-legitimationsoplysninger efter behov

WordPress.org-pakken indeholder kernefunktionerne i Superdav AI Agent. Advanced-ledsagerpakken bygges og distribueres separat til miljøer, hvor betroede udviklerværktøjer eksplicit installeres og autoriseres.

## Temabevidste færdigheder {#theme-aware-skills}

Superdav AI Agent registrerer automatisk dit aktive tema og installerede plugins og indlæser den relevante temabevidste færdighed i assistentens kontekst. Tilgængelige færdigheder omfatter:

| Færdighed | Gælder for |
|---|---|
| **Block Themes** | Temaer, der bruger WordPress' blokbaserede (Full Site Editing) arkitektur |
| **Classic Themes** | Traditionelle WordPress-temaer, der bruger PHP-skabeloner og den klassiske editor |
| **Kadence Blocks** | Websteder med Kadence Blocks-plugin'et aktivt |
| **Kadence Theme** | Websteder med Kadence-temaet som det aktive tema |

Se [Temabevidste færdigheder](../../user-guide/configuration/theme-aware-skills) for detaljeret vejledning om hver færdighed.

## Internetsøgningsudbydere {#internet-search-providers}

Superdav AI Agent v1.10.0 tilføjer **Tavily** som et alternativ til Brave Search til internetsøgeforespørgsler. Begge udbydere tilbyder søgeresultater i realtid og kan konfigureres i **Gratis AI Agent → Settings → Advanced**.

| Udbyder | Funktioner | Opsætning |
|---|---|---|
| **Brave Search** | Hurtige, privatlivsfokuserede søgeresultater | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Rigere søgeresultater, avancerede researchfunktioner | [Tavily API](https://tavily.com/) |

Se [Internetsøgning](../../user-guide/configuration/internet-search) for konfigurationsinstruktioner.

## Kalender-, godkendelses- og notifikationsarbejdsgange {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 udvider automatiseringsarbejdsgange ud over indholds- og webstedsopbygningsopgaver:

- **Google Calendar-værktøjer** lader agenten læse konfigurerede kalendere og begivenhedsdetaljer, når kalenderlegitimationsoplysninger er tilgængelige.
- **Kontakt- og deltagerhjælpere** matcher begivenhedsdeltagere med WordPress-brugere eller kendte kontakter, før påmindelser eller opfølgningsopgaver forberedes.
- **Menneskelige godkendelsesporte** gør det muligt for en automatisering at sætte på pause, indtil en autoriseret bruger gennemgår den foreslåede handling.
- **Påmindelsesposter** forhindrer duplikerede kalender- eller notifikationspåmindelser, når en automatisering forsøges igen eller planlægges gentagne gange.
- **TextBee SMS-understøttelse** muliggør notifikationer via tekstbesked, når TextBee-udbyderen er konfigureret.

Brug disse værktøjer til tidsplanbevidst assistance, deltageropfølgning og påmindelsesarbejdsgange, der stadig kræver menneskelig gennemgang, før følsomme beskeder sendes.

## Konfiguration {#configuration}

Alle indstillinger findes under **Gratis AI Agent → Settings** i WordPress-administrationen. Superdav AI Agent tilføjer følgende konfigurationsmuligheder:

| Indstilling | Beskrivelse |
|---|---|
| **Internet Search Provider** | Vælg mellem Brave Search og Tavily (valgfrit) |
| **Tavily API Key** | API-nøgle til Tavily-internetsøgning (hvis Tavily er valgt) |
| **Managed Superdav Service** | Forbind understøttede websteder til den hostede Superdav-tjeneste med automatisk endpoint-klargøring |
| **Google Calendar Credentials** | Konfigurér kalenderadgang, så agenten kan læse kalendere og begivenheder |
| **TextBee SMS Provider** | Konfigurér TextBee-legitimationsoplysninger til godkendte SMS-notifikationsarbejdsgange |

## Avanceret companion-pakke {#advanced-companion-package}

Superdav AI Agent v1.18.0 tilføjer en separat distribueret avanceret companion-pakke til betroede udviklerarbejdsgange. Disse muligheder er ikke inkluderet i WordPress.org-kernepakken. Installér kun companion-pakken på websteder, hvor administratorer udtrykkeligt stoler på, at agenten får adgang til udviklerværktøjer.

Avancerede værktøjer inkluderer filsystemoperationer, databaseadgang, WP-CLI, REST-dispatcher-arbejdsgange, plugin-bygger-værktøjer, git-snapshots, brugeradministrationshjælpere og benchmark-værktøjer. Begræns disse værktøjer til betroede administratorer, og gennemgå konfigurationen af godkendelsesporte, før du aktiverer dem på produktionswebsteder.

## Fejlrettelser i v1.10.0 {#bug-fixes-in-v1100}

- Rettede `navigate-to`-evne, der forårsagede uendelige genindlæsningssløjfer på nogle admin-sider
- Rettede `list-posts`-evne, der ikke korrekt omsatte kategori- og tagnavne til slugs
- Rettede WP-CLI-kommandoer, der manglede namespace-aliaser efter en tidligere refaktorering
- Rettede hændelsesautomatisering, så den håndterer sites korrekt, hvor automatiseringstabeller endnu ikke er oprettet
- Rettede `memory-save`-evne, der bruger det korrekte namespace-præfiks i systeminstruktionsbyggeren
- Rettede skalære værktøjsresultater, så de bliver korrekt indpakket, før de returneres til AI'en
- Rettede brugsstatistik, så den korrekt håndterer det ældre evnenøgleformat ved opgradering fra ældre versioner
