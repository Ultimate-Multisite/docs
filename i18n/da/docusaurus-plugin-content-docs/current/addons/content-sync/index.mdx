# Indholdssynkronisering {#content-sync}

Multisite Ultimate: Indholdssynkronisering

Dette addon til Multisite Ultimate giver netværksadministratorer mulighed for at synkronisere indhold fra ét site til andre sites i netværket. Det understøtter alle posttyper, herunder posts, sider, custom post types og Elementor-templer. Addon'et inkluderer et flertrins wizard-interface til valg af indhold og mål-sites, med batchbehandling og realtidsfremskridtsfeedback.

## Funktioner {#features}

* Indholdsvalg: Visuel kortgitter med miniaturebilleder og posttypefiltrering
* Alle Posttyper: Synkroniser posts, sider, produkter, Elementor-templer og enhver custom post type
* Sitevalg: Vælg hvilke sites indhold skal synkroniseres til, med eksklusion af kilde-site og forudvalgt valg af child sites
* Batchbehandling: Synkroniserer til op til 10 sites ad gangen med fremskridtsopdateringer
* Mediesynkronisering: Kopierer og mapper automatisk mediefiler (billeder, videoer) fra postindhold til mål-sites
* Taksonomisyndkronisering: Bevarer kategorier, tags og custom taxonomier
* Elementor Support: Fuld understøttelse af Elementor-templer med CSS-genopbygning og remapping af template ID'er
* Konfliktløsning: Mulighed for at erstatte eksisterende indhold eller spring over det
* Cachehåndtering: Rydder automatisk Elementor cache på mål-sites efter synkronisering

## Krav {#requirements}

* WordPress 5.3 eller nyere
* PHP 7.4 eller nyere
* Multisite Ultimate plugin (aktiv)
* Elementor plugin (valgfrit, nødvendigt for synkronisering af Elementor-templer)

## Installation {#installation}

1. Upload 'ultimate-multisite-content-sync' til mappen '/wp-content/plugins/'
2. Aktiver plugin'et via menuen 'Plugins' i din WordPress Network Admin
3. Konfigurer indstillingerne i Ultimate Multisite → Settings → Content Sync
4. Gå til Ultimate Multisite → Content Sync for at starte synkroniseringen
