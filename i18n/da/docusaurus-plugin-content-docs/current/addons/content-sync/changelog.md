# Indholdssynkroniseringsændringer

### Version 1.1.0 - Udgivet den 2025-12-31 {#version-110---released-on-2025-12-31}
* **Nyt:** Understøtter synkronisering af alle posttyper (indlæg, sider, custom post types).
* **Nyt:** Posttypefiltrering i trinnet til valg af indhold.
* **Nyt:** Trin til valg af kilde-site – synkroniser fra ethvert site, ikke kun hovedsitet.
* **Nyt:** Forudvalg af child sites til template sites.
* **Nyt:** Medieudtrækning fra `post_content` for alle posttyper.
* **Nyt:** Synkronisering af taksonomier (kategorier, tags, custom taxonomies).
* **Ændret:** Navnet er ændret fra "Elementor Template Sync" til "Content Sync".
* **Ændret:** Trinnet "Select Templates" er omdøbt til "Select Data".
* **Forbedret:** Elementor-specifik kode kører nu kun for posttype `elementor_library`.

### Version 1.0.0 - Udgivet den 2025-12-19 {#version-100---released-on-2025-12-19}
* Indledende udgivelse.
* Flertrins-wizard til valg af template og site.
* Batchbehandling med realtidsfremskridtsfeedback.
* Fuldt medie-synkronisering med automatisk ID-ommapping.
* Integration med indstillinger for Ultimate Multisite.
* Understøttelse af site-udelukkelse via netværksindstillinger.
* Elementor cache-nulstilling efter synkronisering.
* Bevarelse af template taksonomier.
