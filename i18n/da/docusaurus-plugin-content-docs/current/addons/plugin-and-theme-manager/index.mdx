# Udvidelses- og temahåndtering {#plugin--theme-manager}

Ultimate Multisite: Udvidelses- og temahåndtering

Rediger eller skjul metaoplysningerne (titel, forfatter, miniaturebillede og beskrivelse) for installerede udvidelser og temaer. Opret kategorier til udvidelser og temaer, så dine brugere kan filtrere dem, og tilføj en flot brugerdefineret side for udvidelser til dine brugeres panel.

For at bruge udvidelsen kan du blot gå til dine sider for udvidelser eller temaer i din netværksadministration og klikke på "Rediger oplysninger om udvidelse/tema".

## Installation {#installation}

1. Upload 'ultimate-multisite-plugin-and-theme-manager' til mappen '/wp-content/plugins/'
2. Aktivér udvidelsen via menuen 'Udvidelser' i din WordPress-netværksadministration

## Brugsvejledning {#usage-guide}

Ultimate Multisite: Udvidelses- og temahåndtering er en stærk **premium** Ultimate Multisite-tilføjelse, der giver dig fuld kontrol over, hvordan dine netværksbrugere ser og interagerer med udvidelses- og temaindstillingerne på dit netværk.

Denne tilføjelse har tre formål:

  1. Den lader dig ændre metadataene fuldstændigt for både udvidelser og temaer. Det omfatter navn, beskrivelse, forfatternavn og endda miniaturebilleder!

  2. Den lader dig kategorisere udvidelser og temaer, så dine brugere nemt kan sortere dem.

  3. Den giver dig mulighed for at erstatte den gamle, kedelige side for udvidelser med en ny temalignende side med sortering efter kategorier.

Efter download og installation vises en ny fane - _Indstillinger for tilføjelse_ \- på siden Ultimate Multisite-indstillinger.
