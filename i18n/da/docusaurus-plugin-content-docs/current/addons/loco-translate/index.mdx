# Loco Translate

Ultimate Multisite: Loco Translate Integration-tilføjelsen giver webstedsspecifikke oversættelsesmuligheder til dit WordPress multisite-netværk. Hvis du nogensinde har haft brug for, at forskellige websteder i dit netværk har forskellige oversættelser af den samme plugin eller det samme tema, er denne tilføjelse den perfekte løsning.

**Hvad denne tilføjelse gør**

Denne tilføjelse integrerer problemfrit Ultimate Multisite med den populære Loco Translate-plugin, så hvert underwebsted i dit netværk kan vedligeholde sine egne tilpassede oversættelser. Uanset om du driver en multiregional platform, betjener forskellige sprogfællesskaber eller blot har brug for oversættelsesfleksibilitet på tværs af dit netværk, gør denne tilføjelse det ubesværet.

**Perfekt til:**

* Multiregionale virksomheder med forskellige lokaliseringsbehov
* Uddannelsesinstitutioner, der betjener forskellige fællesskaber
* SaaS-platforme, der tilbyder flersproget support
* Webbureauer, der administrerer flere klientwebsteder
* Ethvert multisite-netværk, der kræver oversættelsesfleksibilitet

## Installation

1. Sørg for, at Ultimate Multisite er installeret og netværksaktiveret
2. Installer og aktivér Loco Translate-pluginen
3. Upload denne tilføjelse til din plugin-mappe
4. Netværksaktivér Loco Translate Integration-tilføjelsen
5. Begynd at oprette webstedsspecifikke oversættelser via Loco Translate på ethvert underwebsted

Det var det! Integrationen fungerer automatisk uden yderligere konfiguration.

## Ofte stillede spørgsmål

## Skal jeg konfigurere noget efter installationen?

Nej! Integrationen er automatisk. Når både tilføjelsen og Loco Translate er aktive, kan underwebsteder straks begynde at oprette deres egne tilpassede oversættelser.

## Hvad sker der, hvis et underwebsted ikke opretter tilpassede oversættelser?

Webstedet bruger automatisk netværksdækkende oversættelser som fallback. Intet går i stykker, og alt fungerer problemfrit.

## Kan underwebsteder få adgang til netværksdækkende oversættelser?

Almindelige webstedsadministratorer ser og redigerer kun deres eget websteds oversættelser. Superadministratorer kan få adgang til både netværksdækkende og webstedsspecifikke oversættelser, afhængigt af hvilket websted de ser.

## Virker dette med både plugins og temaer?

Ja! Webstedsspecifikke oversættelser virker for både WordPress-plugins og temaer.

## Vil dette gøre mit multisite-netværk langsommere?

Nej. Tilføjelsen bruger effektiv filindlæsning, der kun tjekker for webstedsspecifikke oversættelser, når det er nødvendigt. Der er ingen mærkbar påvirkning af ydeevnen.

## Kan jeg kopiere netværksoversættelser til et specifikt websted som udgangspunkt?

Ja! Når du redigerer oversættelser i Loco Translate, kan du nemt starte med netværksoversættelser og tilpasse dem til hvert websted.

## Virker dette med WooCommerce og andre plugins?

Absolut! Enhver plugin eller ethvert tema, der bruger standard WordPress-oversættelsesfunktioner, vil virke med webstedsspecifikke oversættelser.
