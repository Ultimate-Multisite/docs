# Filter: networks_user_is_network_admin

Filtrerer de netværk, som en bruger er administrator for.

## Parametre {#parameters}

| Navn | Type | Beskrivelse |
|------|------|-------------|
| $ | `array\|bool` | Liste over netværks-ID'er eller false, hvis brugeren ikke har nogen netværk. |
| $ | `int` | Bruger-ID, som netværkene skal returneres for. |
