# Prilagoovanje Vašeg Admin Panela sa PRO Temom

Često kada radimo za klijente, dobijamo potrebu da im pružimo prilagođeno iskustvo u admin dashboard-u WordPressa. Ako tražite profesionalniji izgled i osećaj, PRO Theme vam može pomoći.

PRO Theme je admin tema razvijena kako bi vam pomogla pri prilagođavanju Dashboard-a vaših klijenata. Takođe je deo jezgra premium pluginova Ultimate Multisite.

Nakon instalacije, idite na stranicu _Plugins_ i kliknite da ga aktivirate na svojoj Mreži (Network).

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Aktivirajte ga u celoj mreži na stranici Plugins._

_**NAPOMENA:**_ _Ako ga samo instalirate, on će se pojaviti kao opcija za vašeg klijenta koju možete označiti da dodate temu samo tom korisniku._

Sada imate potpuno redizajnirani admin interfejs! Takođe imate novo meni u levom kolonu pod nazivom PRO Theme.

<!-- Screenshot unavailable: PRO Theme redesigned admin panel interface -->

_Potpuno novi izgled za vaš Admin Panel._

Na ovom meniju ćete imati pristup pet kartica (tabova):

<!-- Screenshot unavailable: PRO Theme Styling Settings tab -->

_Styling Settings_ (Podešavanja stila)

<!-- Screenshot unavailable: PRO Theme Functionality tab -->

_Functionality_ (Funkcionalnost)

<!-- Screenshot unavailable: PRO Theme Custom CSS tab -->

_Custom CSS_ (Prilagođeni CSS)

<!-- Screenshot unavailable: PRO Theme Import/Export tab -->

_Import/Export_ (Uvoz/Izvoz)

<!-- Screenshot unavailable: PRO Theme Activate tab -->

_Activate PRO Theme_ (Aktiviraj PRO Theme)

U svakoj od njih možete omogućiti i onemogućiti stilove, dozvoliti ili blokirati funkcionalnosti, te pružiti svojim klijentima prilagođavanje koje možda traže u vašoj mreži sa sitima.
