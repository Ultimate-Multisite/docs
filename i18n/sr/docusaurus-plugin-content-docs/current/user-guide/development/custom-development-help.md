# Kako dobiti prilagođenu razvojnu pomoć od Codeable-a?

Ponekad nam zaista treba neka sjajna prilagodba da bismo našim radnim tokovima na mreži bili tačno onak kakvi želimo. Ako tražite upravo to, preporučujemo [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), mesto gde možete pronaći iskusne developere sa iskustvom u Ultimate Multisite-u.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) je platforma koja povezuje korisnike sa kvalifikovanim freelanceri-developerima za WordPress, sposobnim da rade projekte bilo koje veličine. Imaju detaljan proces prijave kako biste bili upareni sa stručnjacima koji su specifično kvalifikovani za zadatak koji imate na umu.

### Kako kreirati projekat u Codeable-u? {#how-to-create-a-project-in-codeable}

Prvo, idite na [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) i kliknite dugme **Start a Project** (Počni projekat).

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Sada, da biste podneli projekat, kliknite na levo **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

U polju "I need help with" (Potrebno mi je pomoć sa), izaberite **Customizing** (Prilagođavanje):

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

U polju "My" (Moj), izaberite **Plugin** (Dodatak):

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Zatim kliknite dugme **Continue** (Nastavi).

<!-- Screenshot unavailable: Codeable project form Continue button -->

Sada je vreme da opišete svoj projekat i potrebe na najbolji mogući način. Pokušajte dati što više informacija!

<!-- Screenshot unavailable: Codeable project description form -->

Odgovorite na sledeća pitanja kako biste pomogli sa cenama.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Након то, ако још немате [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) акаунт, креирај је и кликните на дугме **Publish My Project** (Објави мој пројекат).

<!-- Скриншот недоступен: креирање акаунта Codeable-а и дугме Publish My Project -->

Када објавите свој [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) пројекат, платформа ће вас повезати са девелозерima који имају вештине које су потребне за ваш проект.

Ова проценат ће вам помоћи да нађете прави личност за вашу пословање, која има искуство са WordPress-ом и Ultimate Multisite-ом!

Минимална ставница Codeable-а је $70–$120 усарджу USD по сати. Процене ће се мењати у зависности од обмала сложности рада, сложености и брзине пројекта.

Када добијете процену, моћи ћете да започнете са нанима девелозера који најбоље одговара вашим потребима и заједно покренете свој проект!
