# VAT

Multisite Ultimate: европски ПДВ

Додаје додатне функције и алате подразумеваном руковању порезима у Multisite Ultimate-у ради боље подршке за европски ПДВ.
Аутоматски додаје стопе ЕУ ПДВ-а на страницу Пореске стопе из

## Инсталација {#installation}

1. Отпремите 'ultimate-multisite-vat' у директоријум '/wp-content/plugins/'
2. Активирајте додатак преко менија 'Додаци' у WordPress-у
3. Идите на Ultimate Multisite > Подешавања > Порези и омогућите ЕУ ПДВ.
4. Кликните на Пореске стопе и Ажурирај стопе ЕУ ПДВ-а при дну, а затим сачувајте пореске стопе.
5. Готово!
