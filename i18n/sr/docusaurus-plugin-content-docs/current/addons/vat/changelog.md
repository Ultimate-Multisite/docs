# Дневник измена за VAT

Верзија 1.0.7 - Објављено 2026-02-03

* Исправка: Немачки (DE) VAT бројеви нису пролазили проверу ваљаности због грешака MS_MAX_CONCURRENT_REQ у VIES SOAP сервису. Пребачено је са застарелог SOAP API-ја на званични ЕУ VIES REST API ради поузданије провере ваљаности.
* Исправка: VAT бројеви се сада прихватају када је VIES сервис привремено недоступан, уместо да буду погрешно одбијени. Унос у евиденцију се креира за каснији преглед.
* Исправка: Поље VAT/Tax ID је сада видљиво за све земље, не само за чланице ЕУ. Порески ID-јеви ван ЕУ (нпр. швајцарски CHE бројеви) чувају се за приказ на фактури без VIES провере ваљаности.
* Исправка: Проналажење VAT префикса за Грчку (GR), Монако (MC) и Острво Ман (IM) било је неисправно због нетачне логике претраге низа.
* Побољшање: Уклоњена је непотребна развојна зависност wp-cli-bundle која је могла да изазове фаталне грешке када се користи уз додатак Site Exporter.
* Побољшање: Додат је свеобухватан пакет јединичних тестова (53 теста).

Верзија 1.0.6 - Објављено 2026-01-25

* Исправка: Домаће B2B трансакције сада исправно наплаћују VAT. Према правилима ЕУ за VAT, обрнуто опорезивање примењује се само на прекограничне B2B трансакције, а не када се земља купца поклапа са земљом компаније.

Верзија 1.0.5 - Објављено 2026-01-22

* Исправка: Додатак се није учитавао или није исправно радио.
* Измена: Извор података о VAT стопама пребачен је са euvatrates.com на репозиторијум ibericode/vat-rates ради поузданијих и активно одржаваних података.
* Исправка: Исправљена је словна грешка у вредности опције super_reduced_rates.
* Побољшање: Додате су провере на null при руковању кодом земље ради спречавања грешака.
* Измена: Уклоњене су приложене датотеке превода у корист аутоматских ажурирања преко Traduttore.

Верзија: 1.0.3 - Објављено 2025-09-28

* Преименован префикс у ultimate-multisite; ажуриран text domain; повећање верзије.
