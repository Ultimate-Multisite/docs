# Креирање менија

Могућност **Креирање менија** креира WordPress навигационе меније уз подршку за засебне ознаке навигације које се разликују од наслова страница.

## Преглед {#overview}

Ова могућност проширује стандардну функционалност креирања менија могућношћу навођења параметра `navigation_label`. То вам омогућава да креирате меније у којима се ознака приказана у навигацији разликује од наслова странице, што пружа већу флексибилност у структури сајта и корисничком искуству.

## Параметри {#parameters}

| Параметар | Тип | Обавезно | Опис |
|-----------|------|----------|-------------|
| `name` | string | Да | Назив менија, нпр. `Primary Navigation` |
| `location` | string | Не | Локација теме којој треба доделити овај мени, нпр. `primary` |
| `navigation_label` | string | Не | Ознака за приказ у навигацији (различита од наслова странице) |

## Повратна вредност {#return-value}

```json
{
  "success": true,
  "menu_id": 7,
  "name": "Primary Navigation",
  "navigation_label": "Main Menu"
}
```

## Ознака навигације у односу на наслов странице {#navigation-label-vs-page-title}

Параметар `navigation_label` вам омогућава да раздвојите интерни назив менија од ознаке која се приказује корисницима:

- **`name`** — Интерни идентификатор менија који користи WordPress (нпр. „Primary Navigation“)
- **`navigation_label`** — Ознака која се приказује посетиоцима сајта у навигацији (нпр. „Главни мени“)

Ово је корисно када:
- Ваша интерна конвенција именовања разликује се од ознака које виде корисници
- Желите краће ознаке у навигацији него у администраторском панелу
- Потребно је да подржите више језика са различитим дужинама ознака
- Правите меније за одређене регионе или корисничке групе

## Примери употребе {#usage-examples}

### Пример 1: Једноставан мени са ознаком навигације {#example-1-simple-menu-with-navigation-label}

**Упит:**
```
Креирај примарни навигациони мени под називом „Главна навигација“ са ознаком навигације „Мени“.
```

**Резултат:**
```json
{
  "success": true,
  "menu_id": 7,
  "name": "Main Navigation",
  "navigation_label": "Menu"
}
```

### Пример 2: Мени за одређену локацију теме {#example-2-menu-for-specific-theme-location}

**Упит:**
```
Креирај мени у подножју под називом „Линкови у подножју“ са ознаком навигације „Брзи линкови“ и додели га локацији подножја.
```

**Резултат:**
```json
{
  "success": true,
  "menu_id": 8,
  "name": "Footer Links",
  "navigation_label": "Quick Links",
  "location": "footer"
}
```

## Интеграција са Theme Builder {#integration-with-theme-builder}

Када користите Theme Builder, могућност Креирање менија:

1. Аутоматски открива доступне локације менија теме
2. Креира меније са одговарајућим ознакама навигације за ваш дизајн
3. Додељује меније исправним локацијама теме
4. Подржава додавање ставки менија након креирања

## Повезане могућности {#related-abilities}

- **`add_menu_item`** — Додај ставке у постојећи мени
- **`update_menu`** — Преименуј мени или га поново додели локацији теме
- **`delete_menu`** — Уклони мени са вашег сајта

## Најбоље праксе {#best-practices}

- **Користите јасне ознаке навигације** — нека ознаке буду сажете и описне за кориснике
- **Ускладите локације теме** — доделите меније исправној локацији теме ради правилног приказа
- **Планирајте структуру менија** — одлучите о хијерархији менија пре креирања ставки
- **Тестирајте прилагодљивост** — проверите да ли се менији исправно приказују на мобилним уређајима
- **Локализујте ознаке** — користите различите ознаке навигације за различите језичке верзије

## Ограничења {#limitations}

- Ознаке навигације служе само за приказ; интерни `name` се и даље користи за WordPress идентификацију
- Подршка теме варира; не подржавају све теме све локације менија
- Ставке менија морају се додати засебно након креирања менија
- Промена ознаке навигације захтева ажурирање менија
