# Генерисање SVG логотипа

Могућност **Генерисање SVG логотипа** омогућава Theme Builder да креира и угради прилагођене SVG логотипе директно у ваш WordPress сајт уз аутоматску санитизацију безбедну за именске просторе.

## Преглед {#overview}

Ова могућност генерише скалабилне векторске графике (SVG) логотипе на основу правца брендирања вашег сајта и дизајнерских преференција. Генерисани SVG-ови се аутоматски санитизују како би били безбедни за употребу у WordPress-у, уз очување визуелног интегритета.

## Параметри {#parameters}

| Параметар | Тип | Обавезно | Опис |
|-----------|------|----------|-------------|
| `site_name` | string | Да | Назив сајта за који треба генерисати логотип |
| `style` | string | Да | Дизајнерски стил (нпр. „модеран“, „класичан“, „минималистички“, „разигран“) |
| `colors` | array | Не | Низ hex кодова боја за употребу у логотипу (нпр. `["#678233", "#ffffff"]`) |
| `width` | number | Не | Ширина SVG-а у пикселима (подразумевано: 200) |
| `height` | number | Не | Висина SVG-а у пикселима (подразумевано: 200) |
| `include_text` | boolean | Не | Да ли укључити назив сајта као текст у логотипу (подразумевано: true) |

## Формат излаза {#output-format}

Могућност враћа SVG ниску са следећом структуром:

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## Понашање SVG санитизације {#svg-sanitisation-behaviour}

Генерисани SVG-ови пролазе кроз аутоматску санитизацију безбедну за именске просторе ради:

- **Уклањања небезбедних атрибута** — уклања обрађиваче догађаја, скрипте и потенцијално опасне атрибуте
- **Очувања именских простора** — одржава SVG именске просторе (xmlns, xlink) ради правилног приказивања
- **Провере структуре** — обезбеђује да је SVG у складу са W3C стандардима
- **Кодирања ентитета** — правилно ескепује специјалне знакове у текстуалном садржају
- **Уклањања спољних референци** — уклања спољне листове стилова и референце на слике

Ово обезбеђује да је SVG безбедан за директно уграђивање у WordPress без потребе за додатном санитизацијом.

## Пример употребе {#usage-example}

**Упит:**
```
Генериши модеран логотип за мој технолошки стартап под називом „CloudSync“ користећи плаву и белу боју.
```

**Резултат:**
Могућност креира SVG логотип који:
- Укључује назив сајта „CloudSync“
- Користи наведену плаво-белу шему боја
- Прати модерне принципе дизајна
- Аутоматски је санитизован и спреман за употребу

## Интеграција са Theme Builder {#integration-with-theme-builder}

Када користите избор правца дизајна у Theme Builder-у, могућност Генерисање SVG логотипа:

1. Анализира ваш правац дизајна и палету боја
2. Генерише прилагођени SVG логотип који одговара вашим преференцијама
3. Аутоматски уграђује логотип у заглавље/област брендирања вашег сајта
4. Чува SVG као прилагођени логотип у WordPress медијима

## Најбоље праксе {#best-practices}

- **Наведите јасне стилске преференције** — опишите дизајнерски стил који желите (модеран, класичан, разигран итд.)
- **Наведите боје** — укључите боје свог бренда ради доследности
- **Тестирајте приказивање** — проверите да ли се логотип правилно приказује на различитим величинама екрана
- **Додатно прилагодите** — користите WordPress алате за прилагођавање логотипа ради подешавања величине и положаја

## Ограничења {#limitations}

- SVG логотипи се генеришу као статичке графике (нису анимирани)
- Сложени логотипи са много елемената могу захтевати ручно дотеривање
- Прилагођени фонтови нису подржани; текст користи системске фонтове
- Веома велике или веома мале димензије могу утицати на квалитет
