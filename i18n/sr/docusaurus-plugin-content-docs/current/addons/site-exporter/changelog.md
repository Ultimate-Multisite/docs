# Дневник измена Site Exporter {#site-exporter-changelog}

Верзија: 1.0.6 - Објављено 2026-05-11
* Ново: Извози сајта сада укључују самопокрећући index.php, тако да ZIP може да се инсталира на новом хосту без посебне инсталације plugin-а
* Ново: Извоз мреже омогућава администраторима да извезу све подсајтове у једној архиви са админ странице Site Export

Верзија: 1.0.5 - Објављено 2026-05-05
* Исправка: Додат је __callStatic у WP_CLI polyfill stub ради обраде немокованих метода као што је get_config, чиме се спречавају грешке при покретању тестова

Верзија: 1.0.4 - Објављено 2025-11-25

* Исправљена компатибилност са Ultimate Multisite 2.4.

Верзија: 1.0.2 - Објављено 2025-09-28

* Преименован префикс у ultimate-multisite; ажуриран text domain; повећана верзија.

### Верзија 1.0.1 - Објављено 2023-08-09 {#version-101---released-on-2023-08-09}

* Побољшано: Сада обезбеђује добијање исправне путање датотеке увозника.
* Исправљено: Онемогућен object cache ради спречавања грешака током увоза.
* Исправљено: Исправљен проблем са линком за преузимање importer plugin-а који није радио.

### Верзија 1.0.0 - Објављено 2022-12-23 {#version-100---released-on-2022-12-23}

* Исправљено: Искачући прозор обрасца за извоз се не учитава;
* Исправљено: Javascript датотеке се не учитавају на страници за извоз;
* Интерно: Нови процес изградње;

### Верзија 1.0.0-beta.4 - Објављено 2022-01-13 {#version-100-beta4---released-on-2022-01-13}
