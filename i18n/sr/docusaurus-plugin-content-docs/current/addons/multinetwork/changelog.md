# Дневник измена за Multi-Network

### 1.3.0 {#130}
* Ново: Прегледач шаблона мреже — прегледајте и приказујте шаблоне мреже у живом панелу пре куповине.
* Ново: Бирач шаблона мреже при checkout-у — купци могу да изаберу шаблон мреже током тока signup checkout-а.
* Ново: Cross-network magic link SSO — купци могу беспрекорно да приступају подмрежама путем magic-link аутентификације.
* Ново: Клонирање медија мреже — медији сајта се исправно дуплирају при клонирању мрежа из шаблона.
* Исправка: Клонирање мреже сада ради без додатка за више закупаца; привилегије super admin-а се поново додељују након клонирања.
* Исправка: Креирање мреже је сада компатибилно са WooCommerce и свим платним пролазима.
* Исправка: Број порта је сада укључен у генерисање резервног домена.
* Исправка: CSS бирача шаблона премештен је у enqueue-овани stylesheet ради исправног приказивања на свим темама.
* Исправка: Заштита од недефинисаног кључа `network_id` у опсегу упита сајта.
* Исправка: Провера дозвола мреже пре промене контекста спречава грешке својстава табеле `wpdb`.
