# Multi-Tenancy

Распоредите подсајтове преко сервера и база података ради перформанси и безбедности

## Најважније из издања верзије 1.2.0 {#version-120-release-highlights}

Multi-Tenancy 1.2.0 додаје основу за суверене закупце за радне токове изолације у продукцији:

- Аутоматско пријављивање закупаца без стања, са краткотрајним SSO посетама заштићеним од поновног слања.
- Изолација базе података и система датотека по подсајту за суверене закупце.
- Hostinger могућност више закупаца за обезбеђивање закупаца свесно хоста.
- Капије за проверу суверене миграције за провере наслеђених података и асинхрону обраду слања.
- Подесиво везивање хоста базе података на истој машини за Bedrock, FrankenPHP и localhost инсталације.
- Побољшања иницијализације закупца, писача базе података, иницијализације шеме и обезбеђивања корисника.
- Безбедније уклањање сувереног закупца које чисти акредитиве закупца.

Погледајте корисничке водиче за [радни ток суверене миграције](/user-guide/getting-started/sovereign-migration-workflow), [изолацију више закупаца](/user-guide/configuration/multi-tenancy-isolation) и [операције сувереног закупца](/user-guide/administration/sovereign-tenant-operations).

## Захтеви {#requirements}

- WordPress 5.3 или новији
- PHP 7.4 или новији
- Multisite Ultimate плагин (активан)

## Инсталација {#installation}

1. Отпремите датотеке додатка у свој `/wp-content/plugins/` директоријум
2. Активирајте плагин кроз мени „Плагини“ у WordPress
3. Подесите поставке додатка у администрацији мреже

## Развој {#development}

### Подешавање {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Тестирање {#testing}

Додатак укључује свеобухватан оквир за тестирање:

- **Јединични тестови**: Тестови засновани на PHPUnit са WordPress скупом тестова
- **Основна класа теста**: Пружа помоћне методе за уобичајене тест сценарије
- **Покривеност тестовима**: Подешено за праћење покривености кода
- **Multisite тестирање**: Тестови се извршавају у multisite окружењу

### Стандарди кода {#code-standards}

- **WordPress стандарди кодирања**: Спроводе се преко PHPCS
- **Компатибилност са PHP 7.4+**: Подржане су модерне PHP функције
- **Статичка анализа**: PHPStan интеграција за проверу типова
- **Модернизација кода**: Rector правила за PHP надоградње

### Систем изградње {#build-system}

- **Обрада ресурса**: Минификација CSS/JS датотека
- **Превод**: Генерисање POT датотеке
- **Паковање**: Креирање архиве за дистрибуцију
- **Развојни режим**: Неминификовани ресурси за отклањање грешака

## Конфигурација {#configuration}

Додатак подржава различите опције конфигурације доступне кроз интерфејс администрације мреже.

### Конфигурација сувереног закупца {#sovereign-tenant-configuration}

За суверене закупце, подесите базу података закупца, корен система датотека, везивање хоста базе података и могућност хоста пре усмеравања саобраћаја ка закупцу. Покрените проверу миграције после сваке промене која утиче на акредитиве базе података, везивање хоста, усмеравање или обезбеђивање корисника.

Користите `localhost` само када извршно окружење закупца и дозволе базе података очекују локални сокет или приступ на истој машини. Користите `127.0.0.1` или име хоста сервиса када је то хост дозволе који је подесио ваш провајдер.

### Hostinger могућност {#hostinger-capability}

Hostinger могућност више закупаца ради са дељеним Hostinger интеграционим токеном. Потврдите да је домен присутан у hPanel, да DNS показује на Hostinger Account и да се поставке базе података закупца поклапају са hPanel пре провере.

## Куке и филтери {#hooks-and-filters}

### Акције {#actions}

- `ultimate-multisite-addon-template_init` - Окида се после иницијализације додатка
- `ultimate-multisite-addon-template_loaded` - Окида се после учитавања свих датотека додатка

### Филтери {#filters}

- `ultimate-multisite-addon-template_settings` - Измените поставке додатка
- `ultimate-multisite-addon-template_enabled` - Премостите статус омогућавања/онемогућавања додатка
