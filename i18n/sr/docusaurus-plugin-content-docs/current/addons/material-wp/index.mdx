# Material WP

WordPress администраторски интерфејс је исти већ годинама. Али да ли сте се икада запитали како би изгледала Material Design верзија администраторског интерфејса? 

Material WP је администраторска тема развијена да вам пружи тај изглед и осећај, помажући вам да својим клијентима обезбедите прилагођенији Dashboard.

## Функције

- Подршка за прилагођени CSS
- Једноставна подешавања стилизовања
- Преуређивање менија
- Једноставан увоз/извоз

## Документација

**Прилагођавање вашег администраторског панела помоћу Material WP**

## Инсталација

1. Отпремите 'material-wp' у директоријум '/wp-content/plugins/'
2. Активирајте додатак преко менија 'Додаци' у WordPress
