# Admin Page Creator

## Садржај {#content}
Admin page creator ће вам омогућити да лако додате нове админ странице и подстранице у оквиру WordPress админ Dashboard-а. Може се користити за креирање странице добродошлице или странице са упутствима која може да служи као одредишна страница за нове кориснике. Можете користити градитељ страница као што су Beaver Builder, Elementor, Brizy и Oxygen да без муке дизајнирате странице.

Такође нуди подршку за прилагођени HTML, CSS и JavaScript, као и контролу приступа на основу плана и корисничке улоге!

## Функције {#features}
* Подршка за прилагођени HTML и CSS
* Подршка за JavaScript
* Режим PHP садржаја
* Дозволе на основу корисничке улоге
* Подршка за градитеље страница

## Инсталација {#installation}

1. Отпремите **ultimate-multisite-admin-page-creator** у директоријум **/wp-content/plugins/**
2. Активирајте додатак преко менија **Додаци** у вашем WordPress Network Admin-у

## Водич за коришћење {#usage-guide}

Иако WordPress заиста олакшава додавање садржаја на предњи део сајтова, креирање страница и подстраница на админ страни захтева неке основне вештине кодирања. Ultimate Multisite: Admin Page Creator има за циљ да донесе исту лакоћу коришћења са предњег дела на позадински део сајтова ваше мреже.

## Зашто користити Admin Page Creator? {#why-use-admin-page-creator}

## Креирање ваше прве прилагођене админ странице {#creating-your-first-custom-admin-page}

Након инсталирања и активирања Ultimate Multisite: Admin Page Creator-а, идите у мени мрежног администратора и отворите Admin Pages.

![Листа прилагођених админ страница у Network Admin-у](/img/admin/admin-pages-list.png)

_Листа прилагођених админ страница у Network Admin-у_

Видећете листу свих админ страница које сте до сада креирали (пошто је ово ваша прва посета, листа ће бити празна). Наставите и кликните на **Додај нову админ страницу**. Бићете преусмерени у уређивач админ странице.

Ако нисте упознати са HTML-ом, држите се подразумеваног WordPress уређивача. Моћи ћете да уређујете садржај странице баш као што бисте то радили са обичном објавом или страницом.

## Опције менија {#menu-options}

Са стране, у мета-кутији General Options, моћи ћете да конфигуришете тип менија, ознаку и још много тога.

### Типови менија {#menu-types}

Ваша нова прилагођена админ страница може имати два различита типа менија: мени највишег нивоа и подмени.

Страница менија највишег нивоа, као што име сугерише, додаће вашу новокреирану страницу у админ мени поред подразумеваних WordPress страница.

Ако изаберете опцију највишег нивоа, моћи ћете да изаберете редослед менија, као и икону менија.

Ако желите да ова страница буде страница подменија, моћи ћете да изаберете надређену страницу за коју ћете је повезати. Можете додати своју прилагођену страницу подразумеваним WordPress страницама или својим прилагођеним админ страницама највишег нивоа.

## Напредне опције {#advanced-options}

Требало би да можете да додате све врсте садржаја на своје прилагођене админ странице. То укључује CSS правила и датотеке, JavaScript, као и спољне библиотеке. То можете урадити помоћу картице Advanced Options.

### Прилагођени CSS {#custom-css}

На CSS картици можете додати сопствени прилагођени CSS код, као и увезти CSS датотеке са удаљених сајтова (као што су CDN-ови, Google Fonts итд).

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_Прилагођени CSS је такође подржан. Можете укључити и спољне датотеке стилова!_

### Прилагођени JavaScript {#custom-javascript}

Исто важи и за JavaScript. Можете додати спољне библиотеке које ћете користити у свом прилагођеном JavaScript коду.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Додајте сопствени JavaScript код!_

## Дозволе {#permissions}

Такође у Advanced Options, моћи ћете да изаберете који подсајтови ће имати приступ овој страници. Ово је корисно ако, на пример, планирате да различитим нивоима плана приказујете различит ексклузивни садржај.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Имате потпуну контролу над тим где ће се ова страница приказивати_

**Важно** : Оба услова морају бити испуњена да би страница била приказана. То значи да ако је корисник клијент датог плана, али нема једну од изабраних улога (или обрнуто), страница се неће појавити на његовом панелу.

## Активно {#active}

Ако желите да онемогућите прилагођену администраторску страницу без брисања или мењања дозвола, само уклоните ознаку са опције Активно.

Деактивирање странице највишег нивоа аутоматски ће деактивирати све подстранице повезане са том родитељском страницом.
