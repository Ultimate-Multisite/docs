# Филтер: wu_apc_get_meta

Омогућава програмерима да филтрирају коначне вредности и да додају нове случајеве обраде.

## Параметри

| Назив | Тип | Опис |
|------|------|-------------|
| $default_value | `string` | Подразумевана вредност. |
| $meta_type | `string` | Тип meta података који се преузимају. |
| $meta_key | `string` | Назив meta поља. |
