# Филтер: wu_apc_should_display_admin_menu

Омогућава администраторима да сакрију странице администраторског менија, ако су завршили процес прилагођавања.

## Параметри

| Назив | Тип | Опис |
|------|------|-------------|
| $should_display_admin_menu | `bool` | Да ли треба да прикажемо странице. |
| $network_active | `bool` | Проверава да ли је plugin мрежно активан у администраторском делу. |

### Од верзије

- 1.4.0
