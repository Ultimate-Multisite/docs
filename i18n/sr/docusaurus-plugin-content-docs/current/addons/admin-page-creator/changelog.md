# Дневник измена за Admin Page Creator

### Верзија 1.9.0 - Објављено 2026-01-18 {#version-190---released-on-2026-01-18}

* Исправљено: Фаталне грешке при активирању.
* Рефакторисано: Класе извора садржаја да користе Singleton trait уз правилну иницијализацију родитељског конструктора
* Исправљено: Грешке компилације Vue.js шаблона које су спречавале приказивање селектора типа садржаја
* Исправљено: Недостајућа Vue својства података (template_id, external_link_url_embedable, external_link_url_checking)
* Исправљено: TinyMCE уређивач се није правилно иницијализовао при учитавању странице
* Исправљено: Divi датотека за подршку није се аутоматски учитавала преко Composer-а
* Исправљено: Оштећена PHP синтакса у Divi класи за подршку
* Побољшано: Организација и минификација JavaScript ресурса
* Побољшано: Компатибилност са најновијим WordPress и верзијама градитеља страница
