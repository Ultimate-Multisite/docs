# Дневник измена Content Sync

### Верзија 1.1.0 - Објављено 2025-12-31 {#version-110---released-on-2025-12-31}
* Ново: Подршка за синхронизацију свих типова објава (објаве, странице, прилагођени типови објава)
* Ново: Филтрирање типа објаве у кораку избора садржаја
* Ново: Корак избора изворног сајта - синхронизација са било ког сајта, не само са главног сајта
* Ново: Унапред одабран дечји сајт за шаблонске сајтове
* Ново: Издвајање медија из post_content за све типове објава
* Ново: Синхронизација таксономија (категорије, ознаке, прилагођене таксономије)
* Измењено: Преименовано из "Elementor Template Sync" у "Content Sync"
* Измењено: Корак "Select Templates" преименован у "Select Data"
* Побољшано: Код специфичан за Elementor сада се покреће само за elementor_library тип објаве

### Верзија 1.0.0 - Објављено 2025-12-19 {#version-100---released-on-2025-12-19}
* Почетно издање
* Вишекорачни чаробњак за избор шаблона и сајта
* Серијска обрада са повратним информацијама о напретку у реалном времену
* Потпуна синхронизација медија са аутоматским поновним мапирањем ID-јева
* Интеграција подешавања са Ultimate Multisite
* Подршка за изузимање сајта преко мрежних подешавања
* Брисање Elementor кеша након синхронизације
* Очување таксономије шаблона
