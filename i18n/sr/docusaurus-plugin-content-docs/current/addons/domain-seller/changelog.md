# Дневник измена Domain Seller

Верзија 1.3.0 - Објављено 2026-06-02
- Ново: Додато упозорење у network-admin-у када стање HostAfrica препродавца постане прениско
- Ново: Додато аутоматско мапирање новорегистрованих домена на мрежни сајт
- Исправка: Захтеви за поља регистранта примењују се само при регистрацији новог домена
- Исправка: Обавештења о праћењу стања сада могу да се одбаце
- Исправка: Обезбеђено је очување WooCommerce података за обрачун регистранта
- Исправка: Примењени су захтеви за контакт регистранта током регистрације
- Исправка: Спречено је креирање производа за регистрацију домена са 0% марже
- Исправка: Очувани су избори домена и цене кроз ток сесије наплате
- Исправка: Побољшан је приказ валуте цена HostAfrica домена
- Исправка: Побољшано је понашање form-action-а при наплати како би се спречила неслагања са WP-core query-var вредностима
- Побољшано: Повезана је документација за конфигурацију HostAfrica препродавца у упутствима за подешавање

- Побољшано: Додати су docblock-ови за акције и филтере животног циклуса домена ради смерница за интеграцију за програмере
