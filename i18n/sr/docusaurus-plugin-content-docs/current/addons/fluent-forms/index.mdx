# Fluent Forms

Овај додатак омогућава додавање ограничења за Fluent Forms на Multisite Ultimate у производима, чланствима или сајтовима.
Fluent Forms користи прилагођене табеле уместо прилагођених типова објава, па уобичајене опције ограничења не раде са Fluent Forms.

## Функције

- Додајте ограничења за Fluent Forms или Conversational Fluent Forms
- Клијенти добијају пријатељску поруку која их подстиче да надограде свој план када достигну ограничење
