# Ultimate AI конектор за компатибилне крајње тачке — евиденција измена

## Верзија 2.0.0 — Објављено 2026-04-24

* Ново: Подршка за више провајдера — конфигуришите више AI крајњих тачака и усмеравајте захтеве уз аутоматско резервно пребацивање између провајдера.
* Исправка: Интеграција SDK-а за више провајдера са исправним ID-јевима провајдера, URL-овима за регистрацију и опсегом HTTP филтера по провајдеру.
* Исправка: Нова картица провајдера сада се аутоматски проширује при додавању; поништавање кеша скрипти при ажурирању додатка.
* Исправка: Динамички namespace класе провајдера за eval() и понашање аутоматског проширивања за новододате провајдере.
* Исправка: Користи стабилне Card/CardBody/CardHeader/CardDivider компоненте (више нису експерименталне) ради компатибилности са WordPress 6.9+.
* Исправка: Недефинисани DragHandle замењен је unicode иконом хватаљке за превлачење ради промене редоследа провајдера.
* Исправка: Уклоњен блокирајући HTTP захтев који се покретао при сваком учитавању странице.
* Побољшано: GitHub Actions радни токови надограђени су на Node.js 24.
