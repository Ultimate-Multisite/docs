# Gratis AI Agent

Gratis AI Agent је аутономни WordPress AI асистент који може да планира, изграђује и управља комплетним WordPress сајтовима путем упутстава на природном језику. Он излаже структуриран скуп **способности** — засебних, проверљивих радњи које агент може да позове — које покривају све, од управљања садржајем до прилагођавања дизајн система. Верзија 1.9.0 додаје способности креирања садржаја, визуелни преглед странице путем снимака екрана на страни клијента, пет уграђених профила агента и заставице функција за контролу приступа и брендирање.

## Кључне функције

- **Систем способности** — модуларни, прошириви каталог радњи које агент може да позове на вашој WordPress инсталацији
- **Управљање садржајем** — креирајте објаве и странице (укључујући додељивање шаблона страница), групно креирајте више објава у једном позиву, постављајте истакнуте слике и правите контакт форме
- **Визуелни преглед** — направите снимке екрана живих страница, упоредите стања пре/после и добијте AI-генерисан преглед дизајна који покрива распоред, типографију, боју и приступачност
- **Пет уграђених агената** — писац садржаја, градитељ сајта, дизајн студио, менаџер plugin-а и асистент за подршку; могу се мењати преко бирача агента у заглављу ћаскања
- **Заставице функција** — прекидачи за контролу приступа и брендирање у Подешавања → Заставице функција за ограничења улога, white-label именовање и прилагођавање виџета
- **Управљање прилагођеним типовима објава** — региструјте, листајте и бришите прилагођене типове објава уз трајност након поновних покретања
- **Управљање прилагођеним таксономијама** — региструјте, листајте и бришите прилагођене таксономије уз трајност
- **Способности дизајн система** — убризгајте прилагођени CSS, управљајте курираним block patterns, поставите лого сајта и примените theme.json пресете
- **Управљање глобалним стиловима** — читајте и уписујте theme.json вредности за палете боја, типографију, размаке и још много тога
- **Управљање навигационим менијима** — програмски креирајте и ажурирајте навигационе меније и њихове ставке
- **Управљање опцијама** — читајте и уписујте WordPress опције уз уграђену безбедносну листу блокирања ради заштите критичних подешавања
- **Регистар инсталираних способности** — откријте и активирајте додатне пакете способности без измене основних plugin датотека
- **Оркестрација градитеља сајта v2** — вишекорачни планови изградње сајта са откривањем plugin-а, праћењем напретка и опоравком од грешака
- **WP-CLI команда за benchmark** — покрените benchmark пакет способности агента из командне линије за CI и процену модела

## Захтеви

- WordPress 6.0 или новији
- PHP 8.1 или новији
- WordPress Multisite (активиран на мрежи) или инсталација једног сајта
- Активан plugin конектора AI провајдера (Anthropic, OpenAI или компатибилан)

## Инсталација

1. Отпремите plugin датотеке у `/wp-content/plugins/gratis-ai-agent`
2. Активирајте plugin преко екрана **Plugin-ови** у WordPress-у
3. Идите на **Gratis AI Agent → Подешавања** и повежите свог AI провајдера
4. (Опционално) Прегледајте **Регистар способности** да бисте инсталирали додатне пакете способности

## Брзи почетак

Када се активира и повеже са AI провајдером, отворите панел **Gratis AI Agent** у WordPress администраторској бочној траци. Унесите упутство на природном језику и агент ће испланирати и извршити потребне кораке користећи своје доступне способности.

**Примери упутстава:**

```
Креирај прилагођени тип објаве „Портфолио“ са таксономијом „Категорија пројекта“.
```

```
Убризгај палету боја за тамни режим у тему и постави примарну боју на #1a1a2e.
```

```
Изгради веб-сајт ресторана са онлајн менијем, формом за резервацију и контакт страницом.
```

## Преглед способности

Способности су атомске радње које агент може да изврши. Свака способност је регистрована PHP класа која излаже шему (параметре и типове повратних вредности) коју агент чита током извршавања. Способности су груписане у функционалне области:

| Област | Способности |
|---|---|
| **Управљање садржајем** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **Визуелни преглед** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **Прилагођени типови објава** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **Прилагођене таксономије** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **Дизајн систем** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **Глобални стилови** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **Навигациони менији** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **Управљање опцијама** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **Инсталиране способности** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **Градитељ сајта** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

Погледајте [Референцу способности](./abilities) за комплетну документацију параметара и примере употребе.

## WP-CLI

Gratis AI Agent испоручује породицу команди `wp gratis-ai-agent` за покретање benchmark-ова и управљање способностима из командне линије. Погледајте [WP-CLI референцу](./wp-cli) за комплетну листу команди.

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## Конфигурација

Сва подешавања су под **Gratis AI Agent → Подешавања** у WordPress администраторском делу:

| Подешавање | Опис |
|---|---|
| **AI провајдер** | Изаберите и конфигуришите активни конектор AI провајдера |
| **Модел** | Изаберите језички модел који ће се користити за сесије агента |
| **Максимум токена** | Буџет токена по захтеву (подразумевано 4096) |
| **Временско ограничење способности** | Максималан број секунди током којих појединачна способност може да се извршава (подразумевано 30) |
| **Евидентирање отклањања грешака** | Бележи позиве способности и резоновање агента у WordPress debug дневник |

## Безбедносна листа блокирања за управљање опцијама

Способност управљања опцијама укључује уграђену листу блокирања која спречава агента да чита или уписује осетљиве WordPress опције. Подразумевана листа блокирања обухвата:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (URL опције — користите наменски алат за миграцију за њих)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (тајни кључеви)
- `active_plugins`, `template` (активација plugin-а/theme-а — користите наменске способности)
- акредитиве платних пролаза и API кључеве које су регистровали инсталирани plugin-и

Додатни уноси могу се додати преко `gratis_ai_agent_options_blocklist` filter-а.

## Куке и филтери

### Радње

- `gratis_ai_agent_ability_registered` — покреће се након што је способност регистрована, прима назив класе способности
- `gratis_ai_agent_plan_started` — покреће се када план израде сајта започне извршавање
- `gratis_ai_agent_plan_completed` — покреће се када се план успешно заврши
- `gratis_ai_agent_plan_failed` — покреће се када план наиђе на непоправљиву грешку

### Филтери

- `gratis_ai_agent_abilities` — измените регистровану листу способности пре него што је агент учита
- `gratis_ai_agent_options_blocklist` — проширите или смањите безбедносну листу блокирања опција
- `gratis_ai_agent_benchmark_questions` — додајте прилагођена питања за референтно тестирање у пакет за евалуацију
- `gratis_ai_agent_plan_plugins` — измените листу за откривање plugin-а током генерисања плана сајта
