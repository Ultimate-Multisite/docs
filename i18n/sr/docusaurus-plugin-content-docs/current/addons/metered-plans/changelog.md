# Дневник измена за планове са мерењем

Верзија 1.1.0 - Објављена 2026-05-05
- Ново: Обрачун AI токена за multisite подсајтове — пратите и наплаћујте коришћење AI токена на сајтовима купаца уз подесиве стопе по токену
- Ново: Примена конектора је поново изграђена са динамичким откривањем лимита и write-through механизмом, што обезбеђује тачност у реалном времену на свим конекторима
- Исправка: Надоградње табела базе података сада су правилно усклађене са дефиницијама BerlinDB шеме, чиме се спречавају неуспеле надоградње при новим инсталацијама
- Исправка: Callback-ови за надоградњу базе података конвертовани су у исправан формат, чиме се решавају тихи неуспеси надоградње
- Исправка: Децималне вредности се сада прихватају у пољу за унос AI Usage Overage Markup
- Исправка: Решене су фаталне грешке и проблеми са двоструком иницијализацијом при учитавању plugin-а
- Побољшано: Додата је check-env npm скрипта како би се developer окружења сама подесила при првом покретању
