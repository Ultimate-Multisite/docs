# Градитељ сајтова са ВИ {#ai-site-builder}

ultimate-multisite-ai-site-builder

## Захтеви {#requirements}

- WordPress 5.3 или новији
- PHP 7.4 или новији
- Multisite Ultimate додатак (активан)

## Инсталација {#installation}

1. Отпремите датотеке додатка у ваш `/wp-content/plugins/` директоријум
2. Активирајте додатак преко менија 'Додаци' у WordPress
3. Подесите поставке додатка у администратору мреже

## Развој {#development}

### Подешавање {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Структура пројекта {#project-structure}

```
ultimate-multisite-ai-site-builder/
├── inc/                          # PHP класе и укључене датотеке
│   ├── class-multisite-ultimate-updater.php  # Руковалац ажурирањем
│   └── class-ai_site_builder-main.php             # Главна функционалност
├── tests/                        # Јединични тестови
│   ├── bootstrap.php            # Покретање тестова
│   ├── class-*-base.php         # Основна тест класа
│   └── class-*-test.php         # Тест случајеви
├── views/                       # Датотеке шаблона
├── assets/                      # Статичка средства
│   ├── css/                     # Листови стилова
│   ├── js/                      # JavaScript датотеке
│   └── images/                  # Слике
├── lang/                        # Језичке датотеке
├── composer.json                # PHP зависности
├── package.json                 # Node.js зависности
├── phpunit.xml.dist             # PHPUnit конфигурација
├── .phpcs.xml.dist              # Конфигурација стандарда кода
└── rector.php                   # Правила модернизације кода
```

### Тестирање {#testing}

Додатак укључује свеобухватан оквир за тестирање:

- **Јединични тестови**: Тестови засновани на PHPUnit са WordPress пакетом за тестирање
- **Основна тест класа**: Обезбеђује помоћне методе за уобичајене тест сценарије
- **Покривеност тестовима**: Подешено за праћење покривености кода
- **Multisite тестирање**: Тестови се извршавају у multisite окружењу

### Стандарди кода {#code-standards}

- **WordPress стандарди кодирања**: Спроводе се преко PHPCS
- **Компатибилност са PHP 7.4+**: Подржане су модерне PHP функције
- **Статичка анализа**: PHPStan интеграција за проверу типова
- **Модернизација кода**: Rector правила за PHP надоградње

### Систем изградње {#build-system}

- **Обрада средстава**: Минификација за CSS/JS датотеке
- **Превод**: Генерисање POT датотеке
- **Паковање**: Креирање архиве за дистрибуцију
- **Развојни режим**: Неминификована средства за отклањање грешака

## Конфигурација {#configuration}

Додатак подржава различите опције конфигурације доступне преко интерфејса администратора мреже.

## Куке и филтери {#hooks-and-filters}

### Радње {#actions}

- `ultimate-multisite-ai-site-builder_init` - Покреће се након иницијализације додатка
- `ultimate-multisite-ai-site-builder_loaded` - Покреће се након што су све датотеке додатка учитане

### Филтери {#filters}

- `ultimate-multisite-ai-site-builder_settings` - Измените поставке додатка
- `ultimate-multisite-ai-site-builder_enabled` - Прегазите статус омогућавања/онемогућавања додатка
