# Филтер: wu_password_require_special

Филтрира да ли треба захтевати специјалне знакове у лозинкама.

## Параметри {#parameters}

| Име | Тип | Опис |
|------|------|-------------|
| $require | `bool` | Да ли треба захтевати специјалне знакове. Подразумевано је true када се правила примењују. |
| $defender_active | `bool` | Да ли је Defender Pro Strong Password активан. |
