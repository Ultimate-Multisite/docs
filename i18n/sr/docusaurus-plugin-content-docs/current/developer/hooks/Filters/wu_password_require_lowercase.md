# Филтер: wu_password_require_lowercase

Филтрира да ли треба захтевати мала слова у лозинкама.

## Параметри

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | Да ли треба захтевати мала слова. Подразумевано true када се правила примењују. |
| $defender_active | `bool` | Да ли је Defender Pro Strong Password активан. |
