# Filter: wu_username_from_email

Филтрирајте корисничко име новог купца.

## Параметри

| Name | Type | Description |
|------|------|-------------|
| $username | `string` | Корисничко име купца. |
| $email | `string` | Адреса е-поште новог купца. |
| $new_user_args | `array` | Низ аргумената новог корисника, можда укључујући име и презиме. |
| $suffix | `string` | Додајте стринг корисничком имену да би било јединствено. |
