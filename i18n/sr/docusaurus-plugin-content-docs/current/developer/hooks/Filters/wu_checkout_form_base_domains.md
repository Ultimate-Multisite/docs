# Филтер: wu_checkout_form_base_domains

Филтрира дељене основне домене обрасца за checkout који не треба да постану записи мапираних домена.

Користите овај филтер када интеграција обезбеђује додатне основне домене за **Site URL** поља обрасца за checkout. Домени које овај филтер врати третирају се као дељени хостови за регистрацију уместо као прилагођени домени по сајту.

## Параметри

| Назив | Тип | Опис |
|------|------|-------------|
| $domains | `array` | Дељени основни домени прикупљени из конфигурације обрасца за checkout. |

### Од верзије

- 2.13.0

### Извор

Дефинисано у `inc/functions/domain.php`.


## Враћа

Низ нормализованих основних домена обрасца за checkout.
