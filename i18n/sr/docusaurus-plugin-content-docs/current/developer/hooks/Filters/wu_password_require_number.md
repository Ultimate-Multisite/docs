# Филтер: wu_password_require_number

Филтрира да ли треба захтевати бројеве у лозинкама.

## Параметри {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | Да ли треба захтевати бројеве. Подразумевано је true када се правила примењују. |
| $defender_active | `bool` | Да ли је Defender Pro Strong Password активан. |

- 2.4.0
