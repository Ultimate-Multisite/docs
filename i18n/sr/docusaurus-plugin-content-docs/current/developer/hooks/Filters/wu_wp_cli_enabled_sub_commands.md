# Филтер: wu_wp_cli_enabled_sub_commands

Филтрира који sub_commands су омогућени за овај ентитет.

## Параметри

| Име | Тип | Опис |
|------|------|-------------|
| $sub_commands | `array` | Подразумевани sub_commands. |
| $command_base | `string` | Основа која се користи у команди одмах после корена. |
| $this | `\Base_Manager` | Инстанца објекта. |

### Од верзије
