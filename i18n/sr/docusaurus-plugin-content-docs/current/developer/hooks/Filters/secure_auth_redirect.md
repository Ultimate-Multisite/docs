# Филтер: secure_auth_redirect

Филтрира да ли да се користи безбедно преусмеравање за аутентификацију.

## Параметри {#parameters}

| Назив | Тип | Опис |
|------|------|-------------|
| $secure | `bool` | Да ли да се користи безбедно преусмеравање за аутентификацију. Подразумевано је false. |

### Од верзије {#since}

- 3.1.0
