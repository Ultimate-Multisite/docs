# Филтер: wu_password_require_uppercase

Филтрира да ли треба захтевати велика слова у лозинкама.

## Параметри

| Назив | Тип | Опис |
|------|------|-------------|
| $require | `bool` | Да ли треба захтевати велика слова. Подразумевано је true када се правила примењују. |
| $defender_active | `bool` | Да ли је Defender Pro Strong Password активан. |

### Од верзије

- 2.4.0
### Извор

Дефинисано у [`inc/class-scripts.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-scripts.php#L553) у реду 553
