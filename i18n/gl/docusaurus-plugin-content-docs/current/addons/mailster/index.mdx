# Mailster

Transforma a túa rede multisite nunha potente máquina para crear listas de correo electrónico co complemento Ultimate Multisite Mailster Integration. Engade automaticamente novos clientes ás túas listas de correo de Mailster cando se rexistren ou fagan compras, creando unha conexión fluída entre a túa plataforma de membresía e a túa estratexia de márketing por correo electrónico.

## Funcionalidades principais {#key-features}

**Sincronización automática de subscritores**
* Engade automaticamente clientes ás listas de Mailster cando se rexistren
* Escolle sincronizar inmediatamente no rexistro ou agardar pola confirmación do pagamento
* Mantén as túas listas de correo actualizadas e ao día sen intervención manual

**Xestión flexible de listas**
* Define listas predeterminadas globais para todos os novos clientes
* Substitúe os valores predeterminados con listas específicas por produto para campañas segmentadas
* Xestiona facilmente as asignacións de listas coa interface familiar de Mailster

**Opcións de consentimento compatibles co RGPD**
* Escolle entre subscrición automática ou consentimento mediante caixa de verificación
* Activa o dobre consentimento para subscricións confirmadas
* Amosa/oculta a caixa de verificación de consentimento segundo os teus requisitos de cumprimento

**Mapeamento intelixente de datos**
* Sincroniza automaticamente os nomes dos clientes e a información de contacto
* Mapea os enderezos de facturación aos campos de subscritor
* Fai seguimento das datas de rexistro para campañas segmentadas

**Control a nivel de produto**
* Asigna listas diferentes a produtos diferentes
* Crea campañas segmentadas segundo o que merquen os clientes
* Desactiva Mailster para produtos específicos cando sexa necesario

**Experiencia profesional**
* Integración nativa da interface de Mailster para a selección de listas
* Rexistro detallado para resolución de problemas e verificación
* Actualizacións fluídas dos subscritores existentes

## Perfecto para {#perfect-for}

* Propietarios de sitios de membresía que crean listas de correo
* Plataformas SaaS que queren nutrir novos clientes
* Creadores de cursos que segmentan estudantes por inscrición
* Calquera persoa que execute Ultimate Multisite con Mailster

## Como funciona {#how-it-works}

1. Instala e activa o complemento na túa rede Ultimate Multisite
2. Configura as listas predeterminadas globais nos axustes de Ultimate Multisite
3. Opcionalmente, substitúeas con listas específicas por produto
4. Escolle o teu modo de consentimento e o momento da subscrición
5. Mira como as túas listas de Mailster medran automaticamente!

## Instalación {#installation}

1. Sobe os ficheiros do plugin ao directorio de complementos de Ultimate Multisite
2. Asegúrate de que Mailster estea instalado e activo no teu sitio principal
3. Activa o complemento a través da páxina de complementos de Ultimate Multisite
4. Navega ata Axustes > Integración con Mailster para configurar
5. Selecciona as túas listas predeterminadas e configura as preferencias de consentimento

## Preguntas frecuentes {#frequently-asked-questions}

## Funciona este complemento sen Ultimate Multisite? {#does-this-addon-work-without-ultimate-multisite}

Non, este complemento require Ultimate Multisite para funcionar. Está deseñado especificamente para integrar Mailster co sistema de clientes e produtos de Ultimate Multisite.

## Teño que ter Mailster instalado en todos os sitios? {#do-i-need-mailster-installed-on-all-sites}

Non, Mailster só precisa estar instalado e activado no teu sitio principal da rede. O complemento conectarase automaticamente a el desde todos os subsitios.

## Podo asignar listas diferentes a produtos diferentes? {#can-i-assign-different-lists-to-different-products}

Si! Podes definir listas predeterminadas globais e logo substituílas produto por produto. Isto permíteche crear campañas de correo electrónico moi segmentadas segundo o que merquen os clientes.

## Que acontece se un cliente xa existe en Mailster? {#what-happens-if-a-customer-already-exists-in-mailster}

Por defecto, o complemento actualizará os subscritores existentes e engadiraos a novas listas. Podes desactivar isto nos axustes se prefires omitir os subscritores existentes.

## É compatible co RGPD? {#is-this-gdpr-compliant}

Si! O complemento inclúe un modo de consentimento mediante caixa de verificación no que os clientes deben consentir explicitamente recibir correos electrónicos. Isto cumpre os requisitos do RGPD cando se combina con políticas de privacidade axeitadas.

## Podo usar dobre consentimento? {#can-i-use-double-opt-in}

Por suposto! Activa o axuste de dobre consentimento e Mailster enviará correos electrónicos de confirmación antes de activar as subscricións.
