# Acción: wp_ultimo_ptm_edit_page_after_title_input

Permite aos desenvolvedores de plugins engadir información adicional debaixo da entrada de texto

## Parámetros {#parameters}

| Nome | Tipo | Descrición |
|------|------|-------------|
| $ | `object` | Obxecto que contén a información |
| $ | `\WU_Page` | Instancia de páxina de Ultimate Multisite |
