# Filtro: wu_connector_enforcement_option_keys {#filter-wuconnectorenforcementoptionkeys}

Filtra a lista de claves de opción do provedor de AI aplicadas desde o sitio principal.

Engade claves de opción para plugins de provedores de AI personalizados ou de terceiros para que a súa configuración tamén se herde do sitio principal nos subsitios.

## Parámetros {#parameters}

| Nome | Tipo | Descrición |
|------|------|-------------|
| $keys | `string[]` | Nomes das claves de opción para aplicar (xa inclúe as claves de conector descubertas dinamicamente e EXTRA_PROVIDER_OPTIONS). |

- 1.2.0
