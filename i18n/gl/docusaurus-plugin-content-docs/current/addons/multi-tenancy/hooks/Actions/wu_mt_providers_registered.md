# Acción: wu_mt_providers_registered

Execútase despois de que os provedores predeterminados sexan rexistrados.

Usa este hook para rexistrar provedores personalizados.

## Parámetros

| Nome | Tipo | Descrición |
|------|------|-------------|
| $manager | `\Provider_Manager` | A instancia do xestor de provedores. |
