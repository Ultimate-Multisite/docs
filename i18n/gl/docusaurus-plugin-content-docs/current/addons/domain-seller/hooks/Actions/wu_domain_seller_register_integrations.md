# Acción: wu_domain_seller_register_integrations {#action-wudomainsellerregisterintegrations}

Permite que outros plugins rexistren integracións adicionais de vendedores de dominios.

## Parámetros {#parameters}

| Nome | Tipo | Descrición |
|------|------|-------------|
| $registry | `\WP_Ultimo\Integrations\Integration_Registry` | O rexistro de integracións. |
