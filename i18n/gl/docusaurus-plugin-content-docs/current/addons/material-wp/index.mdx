# Material WP

A interface de administración de WordPress leva sendo a mesma desde hai moito tempo. Pero algunha vez te preguntaches como sería unha versión Material Design da interface de administración? 

Material WP é un tema de administración desenvolvido para darche ese aspecto e experiencia, axudándote a ofrecer aos teus clientes un Dashboard máis personalizado.

## Funcionalidades

- Compatibilidade con CSS personalizado
- Axustes de estilo sinxelos
- Reordenación do menú
- Importación/exportación sinxela

## Documentación

**Personalizar o teu panel de administración con Material WP**

## Instalación

1. Sube 'material-wp' ao directorio '/wp-content/plugins/'
2. Activa o plugin a través do menú 'Plugins' en WordPress

## Guía de uso

A interface de administración de WordPress leva sendo a mesma desde hai moito tempo. Pero algunha vez te preguntaches como sería unha versión Material Design da interface de administración?

Despois de **instalalo**, vai á túa páxina de _Plugins_ e fai clic para activalo na túa Rede.
