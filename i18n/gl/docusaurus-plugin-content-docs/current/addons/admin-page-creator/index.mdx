# Admin Page Creator

## Contido {#content}
O creador de páxinas de administración permitirache engadir facilmente novas páxinas de administración e subpáxinas dentro do WordPress admin dashboard. Pode usarse para crear unha páxina de benvida ou unha páxina de titorial que sirva como páxina de destino para novos usuarios. Podes usar un construtor de páxinas como Beaver Builder, Elementor, Brizy e Oxygen para deseñar as páxinas sen complicacións.

Tamén ofrece compatibilidade con HTML, CSS e JavaScript personalizados, así como control de acceso baseado no plan e no rol de usuario!

## Funcionalidades {#features}
* Compatibilidade con HTML e CSS personalizados
* Compatibilidade con JavaScript
* Modo de contido PHP
* Permiso baseado no rol de usuario
* Compatibilidade con construtores de páxinas

## Documentación {#documentation}
[Crear páxinas de administración personalizadas e subpáxinas usando Multisite Ultimate: Admin Page Creator](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## Instalación {#installation}

1. Sube **ultimate-multisite-admin-page-creator** ao directorio **/wp-content/plugins/**
2. Activa o plugin a través do menú **Plugins** no teu WordPress Network Admin

## Guía de uso {#usage-guide}

Aínda que WordPress fai realmente doado engadir contido ao front end dos sitios, crear páxinas e subpáxinas no lado de administración require algunhas habilidades básicas de programación. Ultimate Multisite: Admin Page Creator pretende levar esa mesma facilidade de uso do front end ao back end dos sitios da túa rede.

## Por que usar Admin Page Creator? {#why-use-admin-page-creator}

Quizais queiras ofrecer unha páxina personalizada no panel de administración dos teus clientes con titoriais, quizais queiras engadir unha páxina personalizada con preguntas frecuentes e un widget de contacto. As posibilidades son infinitas con Admin Page Creator, e iso é o que exploraremos no titorial de hoxe.

## Crear a túa primeira páxina de administración personalizada {#creating-your-first-custom-admin-page}

Despois de instalar e activar Ultimate Multisite: Admin Page Creator, vai ao menú de administración da túa rede e entra en Páxinas de administración.

![Lista de páxinas de administración personalizadas no Network Admin](/img/admin/admin-pages-list.png)

_Lista de páxinas de administración personalizadas no Network Admin_

Verás unha lista de todas as páxinas de administración que creaches ata agora (como esta é a túa primeira visita, a lista estará baleira). Continúa e fai clic en **Engadir nova páxina de administración**. Serás redirixido ao editor da páxina de administración.

## Os elementos dunha páxina de administración personalizada {#the-elements-of-a-custom-admin-page}

No editor da páxina de administración, terás varias opcións diferentes para personalizar a túa páxina.

### Título da páxina {#page-title}

Este é o título que se mostrará na parte superior da páxina, non a etiqueta do menú de administración.

<!-- Captura de pantalla non dispoñible: editor da páxina de administración mostrando o campo Título da páxina -->

### Selector de tipo de contido {#content-type-selector}

Ultimate Multisite: Admin Page Creator dáche dúas opcións no que respecta ao contido da páxina: podes usar o editor WYSIWYG predeterminado de WordPress ou cambiar a un editor HTML.

Se non estás familiarizado con HTML, mantente co editor predeterminado de WordPress. Poderás editar o contido da páxina igual que farías cunha entrada ou páxina normal.

## Opcións de menú {#menu-options}

No lateral, na meta-caixa de opcións xerais, poderás configurar o tipo de menú, a etiqueta e moito máis.

### Tipos de menú {#menu-types}

A túa nova páxina de administración personalizada pode ter dous tipos de menú diferentes: menú de nivel superior e submenú.

Unha páxina de menú de nivel superior, como suxire o nome, engadirá a túa páxina recentemente creada ao menú de administración xunto ás páxinas predeterminadas de WordPress.

Se seleccionas a opción de nivel superior, tamén poderás seleccionar a orde do menú e unha icona de menú.

Se queres facer que esta páxina sexa unha páxina de submenú, poderás seleccionar unha páxina pai á que anexar esta páxina. Podes engadir a túa páxina personalizada ás páxinas predeterminadas de WordPress ou ás túas páxinas de administración personalizadas de nivel superior.

## Opcións avanzadas {#advanced-options}

Deberías poder engadir todo tipo de contido ás túas páxinas de administración personalizadas. Iso inclúe regras e ficheiros CSS, JavaScript e tamén bibliotecas externas. Podes facelo usando a pestana Opcións avanzadas.

### CSS personalizado {#custom-css}

Na pestana CSS, podes engadir o teu propio código CSS personalizado, así como importar ficheiros CSS desde sitios remotos (como CDNs, Google Fonts, etc.).

<!-- Captura de pantalla non dispoñible: editor de CSS personalizado de Admin Page Creator con importación de folla de estilo externa -->

_O CSS personalizado tamén é compatible. Tamén podes incluír ficheiros de estilo externos!_

### JavaScript personalizado {#custom-javascript}

O mesmo é válido para JavaScript. Podes engadir bibliotecas externas para usar no teu código JavaScript personalizado.

<!-- Captura de pantalla non dispoñible: editor de JavaScript personalizado de Admin Page Creator con importación de biblioteca externa -->

_Engade o teu propio código JavaScript!_

## Permisos {#permissions}

Tamén nas opcións avanzadas, poderás seleccionar que subsitios terán acceso a esta páxina. Isto é útil se estás a planear ofrecer contido exclusivo diferente a distintos niveis de plan, por exemplo.

<!-- Captura de pantalla non dispoñible: configuracións de permisos de Admin Page Creator para control de acceso baseado en plan e rol -->

_Tes control total sobre onde se vai mostrar esta páxina_

**Importante** : Ambas condicións deben cumprirse para que se mostre a páxina. Isto significa que, se o usuario é cliente dun plan determinado, pero non ten un dos roles seleccionados (ou viceversa), a páxina non aparecerá no seu panel.

## Activo {#active}

Se queres desactivar unha páxina de administración personalizada sen eliminala nin alterar os permisos, simplemente desmarca a opción Activo.

Desactivar unha páxina de nivel superior desactivará automaticamente todas as subpáxinas anexas a esa páxina pai.
