# Fluent Forms

Este complemento permite engadir límites a Fluent Forms en Multisite Ultimate en produtos, subscricións ou sitios.
Fluent Forms usa táboas personalizadas en vez de tipos de publicación personalizados, polo que as opcións normais de limitación non funcionan con Fluent Forms.

## Funcionalidades {#features}

- Engadir límites para Fluent Forms ou Conversational Fluent Forms
- Os clientes reciben unha mensaxe amable que lles solicita actualizar o seu plan cando alcanzan o límite
