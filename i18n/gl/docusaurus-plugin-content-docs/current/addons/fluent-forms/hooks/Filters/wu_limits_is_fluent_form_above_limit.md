# Filtro: wu_limits_is_fluent_form_above_limit

Comproba se un tipo de formulario dado está permitido neste plan. Permite aos desenvolvedores de plugin filtrar o valor de retorno

## Parámetros

| Nome | Tipo | Descrición |
|------|------|-------------|
| $value | `bool` | Se o tipo de formulario está por riba do límite ou non |
| $form_type | `string` | O tipo de formulario que se está comprobando |
| $form_count | `int` | Reconto actual de formularios |
| $quota | `int` | A cota permitida |
