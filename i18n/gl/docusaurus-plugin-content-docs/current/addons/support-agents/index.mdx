# Axentes de soporte

Xestionar unha rede enteira pola túa conta non é unha tarefa doada. A medida que a túa rede comece a medrar, empezarás a ver unha demanda crecente de soporte. Será unha boa idea incorporar persoas que che axuden a aliviar a túa carga de traballo. Porén, conceder acceso/permisos completos é arriscado e pode comprometer a túa rede. Aí é onde entra este add-on de axente de soporte.

Este add-on permíteche crear un novo tipo de usuario de WordPress chamado axente de soporte, e dáche control total sobre que permisos ten ese usuario. Conceder permisos é tan doado como activar un botón conmutador, e desactivalo se decides retiralos.

## Funcionalidades {#features}

- Lista detallada de permisos de WordPress e Ultimate Multisite
- Activa e desactiva o acceso con facilidade
- Crea un número ilimitado de contas de axente

## Instalación {#installation}

1. Carga 'ultimate-multisite-support-agents' no directorio '/wp-content/plugins/'
2. Activa o plugin a través do menú 'Plugins' en WordPress
3. Sigue o asistente paso a paso para configurar o plugin
