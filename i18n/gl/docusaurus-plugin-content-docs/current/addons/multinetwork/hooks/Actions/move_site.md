# Acción: move_site

Dispárase despois de que un sitio se movese a unha nova rede.

## Parámetros {#parameters}

| Nome | Tipo | Descrición |
|------|------|-------------|
| $site_id | `int` | ID do sitio que se moveu. |
| $old_network_id | `int` | ID da rede orixinal para o sitio. |
| $new_network_id | `int` | ID da rede á que se moveu o sitio. |

### Desde {#since}

- 1.3.0
