# Filtro: networks_user_is_network_admin

Filtra as redes das que un usuario é administrador.

## Parámetros {#parameters}

| Nome | Tipo | Descrición |
|------|------|-------------|
| $ | `array\|bool` | Lista de IDs de rede ou false se non hai redes para o usuario. |
| $ | `int` | ID de usuario para o que se deben devolver as redes. |
