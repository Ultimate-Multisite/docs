# Filtro: signup_get_available_languages

Filtra a lista de idiomas dispoñibles para os rexistros de sitios no front-end.

Pasar un array baleiro a este hook desactivará a saída da configuración no formulario de rexistro, e usarase o idioma predeterminado ao crear o sitio. Os idiomas que aínda non estean instalados eliminaranse.

## Parámetros {#parameters}

| Nome | Tipo | Descrición |
|------|------|-------------|
| $available_languages | `array` | Idiomas dispoñibles. |
