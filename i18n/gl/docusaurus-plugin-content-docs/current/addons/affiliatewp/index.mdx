# AffiliateWP {#affiliatewp}

Multisite Ultimate: integración con AffiliateWP

Usa o potente AffiliateWP para facer medrar a base de clientes da túa rede Ultimo!

## Instalación {#installation}

1. Sobe 'ultimate-multisite-git ' ao directorio '/wp-content/plugins/'
2. Activa o complemento a través do menú 'Complementos' en WordPress
3. Sigue o asistente paso a paso para configurar o complemento

## Guía de uso {#usage-guide}

[AffiliateWP](https://affiliatewp.com/) converteuse na solución de referencia ao crear un programa de afiliados para o teu negocio en liña. Ultimate Multisite: AffiliateWP é a ponte que che permitirá crear o mesmo para a túa rede premium.

## Primeiros pasos {#getting-started}

Antes de nada, **tes que ter** [**AffiliateWP**](https://affiliatewp.com/) **activo**! Non poderás usar o noso complemento se non tes **o complemento co que se integra**. No caso de que non teñas previsto permitir que os teus clientes o usen tamén nos seus sitios, AffiliateWP debería estar activo só no teu sitio principal.

Ademais, necesitarás tanto **Ultimate Multisite** como **Ultimate Multisite: AffiliateWP** **activados** **en toda a rede**.

## Engadir Ultimate Multisite como integración de AffiliateWP {#adding-ultimate-multisite-as-an-affiliatewp-integration}

No teu sitio principal, vai a Configuración de afiliados e, na pestana Integracións, marca Ultimate Multisite. Isto indicaralle a [AffiliateWP](https://affiliatewp.com/) que comece a rastrexar visitas e conversións desde o fluxo de rexistro de Ultimate Multisite.

<!-- Captura de pantalla non dispoñible: pestana de configuración de integracións de AffiliateWP coa caixa de verificación de Ultimate Multisite -->

_Marca a caixa de Ultimate Multisite na parte inferior e garda_

Iso é practicamente todo o que fai falta. As visitas procedentes de ligazóns de referencia serán rastrexadas e, cando ese usuario faga máis tarde o seu primeiro pagamento, crearase a conversión e reservarase unha comisión para o afiliado responsable da conversión.

Con todo, só se rastrexará o primeiro pagamento recibido do usuario recentemente creado. Para activar pagamentos de referencia recorrentes, necesitarás o [complemento AffiliateWP Recurring Referrals](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## Activar Recurring Referrals {#enabling-recurring-referrals}

Despois de instalar o [complemento AffiliateWP Recurring Referrals](https://affiliatewp.com/add-ons/pro/recurring-referrals/), vai á pestana Recurring Referrals da Configuración de afiliados e activa o uso de referencias recorrentes.

<!-- Captura de pantalla non dispoñible: configuración de AffiliateWP Recurring Referrals co interruptor Activar Recurring Referrals -->

_Tan sinxelo como pode ser: activa Recurring Referrals e xa está todo listo!_

De novo, iso é todo o que fai falta!

Agora, cada vez que se reciba un novo pagamento nunha conta referida, o afiliado responsable recibirá automaticamente a súa propia parte!
