# Como Obter Ajuda de Desenvolvimento Personalizado do Codeable?

Às vezes, precisamos mesmo de alguma personalização incrível para fazer os nossos fluxos de rede exatamente como queremos. Se é isso que procura, recomendamos o [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), um sítio onde pode encontrar programadores experientes com especialidade em Ultimate Multisite.

O [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) é uma plataforma que liga clientes a programadores freelancers qualificados de WordPress, capazes de trabalhar em projetos de qualquer tamanho. Eles têm um processo de candidatura muito cuidadoso para garantir que é associado a especialistas especificamente qualificados para a tarefa que tem em mente.

### Como criar um projeto no Codeable? {#how-to-create-a-project-in-codeable}

Primeiro, vá ao [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) e clique no botão **Start a Project** (Iniciar um Projeto).

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Agora, para enviar um projeto, clique à esquerda em **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

Em "I need help with" (Preciso de ajuda com), escolha **Customizing** (Personalização):

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

Em "My" (Meu), escolha **Plugin** (Plugin):

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Depois clique no botão **Continue** (Continuar).

<!-- Screenshot unavailable: Codeable project form Continue button -->

Agora, é hora de descrever o seu projeto e necessidades da melhor forma que puder. Tente dar a maior quantidade de informação possível!

<!-- Screenshot unavailable: Codeable project description form -->

Responda às seguintes perguntas para ajudar com os preços.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Despois disso, se ainda non tes conta de [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), créalo e pulsa no botón **Publish My Project** (Publicar o Meu Proyecto).

<!-- Screenshot unavailable: creación de conta Codeable e botão Publish My Project -->

Depois de publicar o teu proyecto en [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), a plataforma conectará ti con desenvolvedores que coincidan co conjunto de habilidades necesario para o teu proyecto.

Este proceso axuda a encontrar a persoa certa para o teu traballo, que ten experiencia con WordPress e Ultimate Multisite!

A taxa mínima do Codeable é de $70–$120 USD por hora. As estimativas cambiarán dependendo do escopo do traballo, da complexidade e da urgencia do proyecto.

Depois de obtener a estimativa, poderás contratar o desenvolvedor que mejor se adapte ás tus necesidades e iniciar o teu proyecto juntos!
