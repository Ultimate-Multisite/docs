# Hooks e filtros do Ultimate Multisite {#ultimate-multisite-hooks-and-filters}

O Ultimate Multisite é feito de unhas e funções de base de datos. Ele tem hooks e filtros que te permitem modificar ou adicionar novo código às funções existentes.

Criámos uma lista destes hooks e filtros como referência para ajudar os usuarios avanzados a personalizar ainda máis a súa rede. Podes acceder a esta lista no Github facendo clic no enlace de abau.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
