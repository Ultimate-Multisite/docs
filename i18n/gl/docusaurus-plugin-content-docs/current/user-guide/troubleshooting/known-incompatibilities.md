# Incompatibilidades conhecidas com outros Plugins

Depois de instalar e ativar o Ultimate Multisite, podes encontrar problemas devido a incompatibilidades não resolvidas com outros plugins.

Estamos a trabalhar ativamente para reduzir esta lista a cada nova versão, mas se um plugin estiver listado aqui, podes precisar de o desativar para ter uma rede Ultimate Multisite totalmente funcional.

Aqui tens uma lista deles e o que deves fazer nestas situações:

**Alguns plugins WPMU DEV** Alguns dos plugins _que não são mais mantidos pelo WPMU DEV_ são conhecidos por causar a paralisação do fluxo de registo do Ultimate Multisite na última etapa (o utilizador não consegue criar uma conta). Aqui está uma lista dos plugins que causam esses problemas:

  * Signup Language: _O Ultimate Multisite tem um add-on que oferece a mesma funcionalidade, chamado Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Isto já vem integrado no Ultimate Multisite, por isso não precisas de um plugin extra._

  * New Blog Templates: _Isto já vem integrado no Ultimate Multisite, por isso não precisas de um plugin extra._

  * Set Password on Multisite Blog Creation: _Isto já vem integrado no Ultimate Multisite, por isso não precisas de um plugin extra._

  * Signup TOS: _Isto já vem integrado no Ultimate Multisite, por isso não precisas de um plugin extra._

  * Anti-Splog: Embora menos poderoso, isto também é _integrado no Ultimate Multisite._

**Nota**: Todos os outros plugins WPMU Dev podem ser usados normalmente ao lado do Ultimate Multisite. Exemplos incluem _Smush_, _Forminator_, _Defender_, etc.
