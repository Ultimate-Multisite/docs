# Pergunto Frequentes {#frequently-asked-questions}

Aquí tes algunhas **Pergunto Frequentes** que suelen aparecer durante os pedidos de soporte.

  * **Que é o Ultimate Multisite?**

O Ultimate Multisite é un plugin de WordPress, feito para instalacións multisite de WordPress, que transforma a túa instalación de WordPress nunha rede premium de sitios – como [WordPress.com](https://WordPress.com) – permitindo aos clientes crear sitios mediante taxas mensais, trimestrais ou anuais (podes crear tamén planos Gratuitos).

  * **Funciona o Ultimate Multisite sen Multisite?**

Non. O Ultimate Multisite requiere WordPress multisite, subdirectorio ou subdomínio.

  * **Que son os requisitos para o Ultimate Multisite?**

O Ultimate Multisite requiere:

**Versión de WordPress** : v5.3+ (Recomendado: Última Versión Estable)
**Versión de PHP** : 7.4.x (Soporte para > 8.0 a unha próxima)
**Versión de MySQL** : v5+ (Recomendado: Versión 5.6, en caso de que non tives 8.0 dispoñible no teu provedor de hébéis)

_O Multisite (subdomínio ou subdirectorio) tamén debe estar activado._ [Como instalar WordPress Multisite](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **Funciona o Ultimate Multisite con Multisite con Subdirectorio?**

Sim. O Ultimate Multisite suporta tanto instalacións de subdirectorio como instalacións de subdirectorio.

  * **Funciona o Ultimate Multisite con Multisite con Subdomínios?**

Sim. O Ultimate Multisite suporta tanto instalacións de subdirectorio como instalacións de subdomínio.

  * **Podo usar o Ultimate Multisite para limitar que Plugins e Temas poden usar os usuarios?**

Sim. Con o Ultimate Multisite podes seleccionar que plugins e temas están dispoñibles para cada plano na túa rede.

  * **Podo usar o Ultimate Multisite para limitar o número de posts que un usuario pode crear no seu sitio?**

Sim. Com o Ultimate Multisite podes establecer cotas de posts para **todo o para cada tipo de post** na tua rede. Podes limitar tamén o número de usuarios que o administrador pode convidar, o **espaco de disco**, e moitos outros.

  * **O Ultimate Multisite suporta Site Templates?**

Sim. Con o Ultimate Multisite podes crear tantas Site Templates como desexas. As Site Templates non son temas, son sitios normais na tua rede. Podes activar plugins, temas e personalizar esa Site Template como que te convenga. Se un usuario selecciona esa template, o seu sitio será criado como unha copia daquela template, con todas as personalizacións en vigor.

Podes tamén establecer unha template específica para un plan. Ao facer isto, cada usuario que se inscriba en ese plan terá o seu sitio criado como duplicado daquela Site Template.

  * **O Ultimate Multisite integra co WooCommerce?**

¡Sim!

Se queres que os usuarios da tua rede t xen WooCommerce disponible para eles, permitindo-lles configurar lojas, eso é posible usando o Ultimate Multisite. Até podes limitar o número de produtos que os teus clientes poderán crear en cada nivel do plan.

En termos de pagamentos de suscripción para o Ultimate Multisite, podes usar o [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) para permitir que os teus clientes Ultimate Multisite paguen usando pasarelas (gateways) de WooCommerce no teu sitio principal. Así, podes acceder á enorme biblioteca de pasarelas de pagamento desenvolvidas para WooCommerce e usalo para facturar aos teus clientes Ultimate Multisite.

  * **Existe unha forma de migrar suscricións de WPMUDEV Pro Sites?**

Sim, o Ultimate Multisite agora ofrece [un útil para migrar] (https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) a toda a súa rede de Pro Sites.

  * **Por que está a aba "Plugins Permitidos" en branco na Páxina de Edición do Plan?**

Ultimate Multisite só permite permitir ou denegar o acceso a plugins que non estean activados na súa rede. Quando un plugin está activado na rede, por natureza, será cargado por defecto en cada sitio da súa rede.

Se non ve ningún plugin listado na aba "Plugins Permitidos", verifique se o plugin deseado está activo na rede ou non.

  * **Podo usar o meu propio diseño de Tabla de Precios?**

Si. Desde a versión 1.4.0, Ultimate Multisite suporta sobrescrituras de plantilla (template overrides).

  * **Funciona Ultimate Multisite con WP Engine?**

Si. Ultimate Multisite sempre foi soportado por WP Engine, pero para que o mapeamento de domínios funcionase era necesario dar pasos manuais extras do administrador (añadir o dominio ao panel de WP Engine). **Isto non é máis o caso desde a versión 1.6.0.** Ultimate Multisite integra directamente con a API de WP Engine e os domínios se añaden ao panel automáticamente después do mapeamento.

  * **Que provedor de hébito recomiñas?**

Ultimate Multisite debería funcionar con cualquier provedor de hébito. A única diferença é que algunha empresa de hébito como Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane e [Runcloud.io](http://Runcloud.io) ten unha integración máis estrita para as capacidades de mapeamento de domínios, así que funcionan "out of the box" (listos para usar). Para outros hébitos, podes ter que configurar o mapeamento de domínios manualmente.
