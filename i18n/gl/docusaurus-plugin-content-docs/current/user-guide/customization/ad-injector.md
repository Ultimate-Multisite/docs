# Colocación de Anuncios en a súa rede con Ultimate Multisite: Ad Injector {#placing-ads-on-your-network-with-ultimate-multisite-ad-injector}

Ultimate Multisite: Ad Injector é un **add-on gratuito** que te proporciona as ferramentas para colocar anuncios nos teus planes directamente desde a súa página de edición.

Tras instalar Ultimate Multisite: Ad Injector, ve á páxina de _Plugins_ para activalo.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Clic na botó "Activar rede"_

Una vez que o add-on este activo, a caixa de metadatos de _Advanced Options_ na pantalla de edición do plan terá unha nova opción chamada _Ad Settings_.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_Agora podes editar as configuracións dos anuncios para cada uno dos teus planes._

Nesta pestaña, poderás añadir os códigos dos anuncios para posts e páxinas no front-end, antes e despois do contido.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Tras introducir o código do teu anuncio, aparecerá nos posts e páxinas daquele plan específico._

TAMPO podes incluir anuncios no encabezado das Páxinas de Administración para os usuarios daquele plan.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Ve á opción "Ad Code" do Admin Panel se queres mostrar anuncios no back-end do teu usuario._

### Além da Publicidade {#beyond-advertising}

TAMPO podes usar Ultimate Multisite: Ad Injector para ofrecer soporte a planes de alto nivel.

Só ve á opción _Admin Panel Ads_ e añade o código HTML da túa ferramenta de soporte.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Captura de ecrã indisponível: Widget de soporte ao cliente exibido no painel de administração para um plano específico -->

_Adicionar suporte ao cliente a planos específicos_

Desta forma, as pessoas podem contactá-lo diretamente a partir do seu Painel de Administração!
