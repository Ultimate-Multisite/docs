# Personalizar o seu Painel de Administração com o Tema PRO

Muitas veces, ao fazer trabalho para clientes, precisamos dar-lhes uma experiência personalizada no painel de administração do WordPress. Se estás à procura de um aspeto e sensação mais profissional, o PRO Theme pode ajudar-te.

O PRO Theme é un tema de administração desenvolvido para te ajudar quando personalizas os Painéis dos teus clientes. E também faz parte do núcleo dos plugins premium do Ultimate Multisite.

Depois de o instalar, vai à tua página de _Plugins_ e clica para ativá-lo na tua Rede (Network).

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Ativa-o em toda a rede na tua página de Plugins_

_**NOTA:**_ _Se o instalas apenas, aparecerá no painel do teu cliente como uma opção que pode ser marcada para adicionar o tema apenas àquele utilizador._

Agora, tens a tua interface de administração completamente redesenhada! Também tens um novo menu na coluna da esquerda chamado PRO Theme.

<!-- Screenshot unavailable: PRO Theme redesigned admin panel interface -->

_Um aspeto totalmente novo para o teu Painel de Administração_

Neste menu terás acesso a cinco abas:

<!-- Screenshot unavailable: PRO Theme Styling Settings tab -->

_Configurações de Estilo (Styling Settings)_

<!-- Screenshot unavailable: PRO Theme Functionality tab -->

_Funcionalidade (Functionality)_

<!-- Screenshot unavailable: PRO Theme Custom CSS tab -->

_CSS Personalizado (Custom CSS)_

<!-- Screenshot unavailable: PRO Theme Import/Export tab -->

_Importação/Exportação (Import/Export)_

<!-- Screenshot unavailable: PRO Theme Activate tab -->

_Ativar o PRO Theme (Activate PRO Theme)_

Em cada uma delas, podes ativar e desativar itens de estilo, permitir ou bloquear funcionalidades, e dar aos teus clientes a personalização que podem estar à procura na tua rede de sites.
