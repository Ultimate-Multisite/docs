# Gestionar Clientes {#managing-customers}

A páxina **Customers** mostra a lista dos membros rexistrados na túa rede. Esta páxina mostra información básica do usuario incluí nome, dirección de email, última entrada e un enlace **switch to** que te permite iniciar sesión no subsite do usuario como administrador do sitio.

![Customers list page](/img/admin/customers-list.png)

Clica en **Manage** abaixo da carta dun cliente para ver información detallada sobre ese cliente.

Na páxina de detalle do cliente, o bloque **At a Glance** mostra o ID do cliente, a última hora de entrada e o total gasto feito por este cliente.

O bloque **Customer Options** permite establecer este cliente como un **VIP** (un tag para facilitar a súa localización). A pestaña **Billing Info** mostra información de facturación como email, dirección e país.

O bloque **Fields** mostra o nome de usuario do cliente, o email e permite cambiar o estado da **Email Verification**.

Debaixo do bloque Customer Options encontrarás cuatro bloques adicionais:

- **Memberships** — Subscritos actuais aos que está suscrito o cliente
- **Payments** — A historia de pagamentos do cliente
- **Sites** — Os sitios que este cliente pose
- **Events** — Eventos relacionados con este cliente (verificación de email, cambios de facturación, etc.)

Clica no ícono de flecha (→) ao lado de cada elemento para navegar á súa páxina de detalle. Por exemplo, clicar na flecha ao lado dun sitio leva-te á páxina de edición do sitio onde podes modificar a asociación da membresía, o tipo de sitio e o estado activo.

O mesmo se aplica ás membresías e aos pagamentos — cada uno enlaza coas páxinas de detalle onde podes editar o estado da membresía, o cliente asociado, os produtos, as configuración da pasarela de pagamento e incluso procesar reembolsos.

No lado direito da páxina do cliente, encontrarás:

- **Último Login & IPs** — Hora do último login, endereço IP e país
- **Timestamps** — Quando foi criada e modificada a conta do cliente
- **Deletar** — Opção para eliminar o cliente permanentemente

:::warning
Eliminar um cliente é permanente e não pode ser desfeito.
:::
