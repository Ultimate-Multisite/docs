# Gestionar Enlaces Compartibles para Planes {#managing-shareable-links-for-plans}

Ultimate Multisite ofrece a ti a opción de compartir enlaces de planes directamente con os clientes. Para hacer isto, ve a calquera das páxinas de edición dos teus planes e pulsa no botón _Click to copy the Shareable Link_ (Clic para copiar o Enlace Compartible).

![Páxina de edición do produto con botón de enlace compartible](/img/config/product-edit-full.png)

_Na páxina de edición do Plan, podes encontrar o botón para crear enlaces compartibles._

Nota que este mecanismo usa a frecuencia de facturación predeterminada para construir o enlace. Isto significa que se o teu facturación predeterminada é mensurial, por exemplo, ao final terás un enlace como este: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Se queres que o enlace compartible se estableza como trimestral, terás de añadir 3 no final:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Para seleccionar o plan _premium_ en unha escala anual, terás de añadir 1 e a palabra:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Ten en conta que non podes ter só un enlace para diferentes frecuencias de facturación. Debes compartir un enlace específico para cada unha das tus frecuencias de facturación.

Una vez que compartas o enlace con os teus clientes, eles serán redirexitados á primeira fase do teu flujo de inscripción (signup flow).
