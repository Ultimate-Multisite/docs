# Por que Ultimate Multisite

Se llegaste a este artigo, é probable que hayas oído que o nosso plugin pode ser a solução para fazer a tua ideia de negócio se tornar realidade. Esperamos que tenhas as respostas de que precisas ao final desta peça.

## O que é o Ultimate Multisite?

O Ultimate Multisite é un plugin WordPress Multisite que te permite oferecer aos clientes um WaaS (un Website as a Service, ou Website como Serviço). Com o nosso plugin, podes ter vários niveles de subscrición e fazer com que os clientes paguen-te uma taxa recorrente para ter un sitio alojado na tua rede Multisite.

O Ultimate Multisite permite-te receber os teus pagamentos através de diferentes gateways (como WooCommerce ou Stripe), criar modelos de sitios para facilitar a vida quando un novo utilizador se regista na tua rede, mapear domínios e personalizar os teus formulários de registo para crear unha experiencia personalizada para os teus clientes.

E isto é só o principio, pois podes extender as súas funcións con add-ons e webhooks.

## Outras características

O Ultimate Multisite también ofrece:

  * Integración de Mapeo de Dominios con proveedores de alojamiento WordPress gestionado: O Ultimate Multisite añade automáticamente os domínios mapeados aos paneles de administración de WP Engine, Cloudways, cPanel, RunCloud, Closte e outros cuando se mapea un novo dominio, para que non tenhas de facelo manualmente. En algun caso, incluso activamos a obtención de certificados SSL por ti.

  * Webhooks: O Ultimate Multisite tem un potente sistema de Webhook que te permite integrar a tua rede con unha serie de diferentes servizos. Incluso ofrecemos integración directa con Zapier, que abre a tua rede a uns 1000+ apps e servizos dispoñibles na súa plataforma.

Páxinas de Administrador potentes para axudarte con os teus clientes usando o teu constructor de páxinas favorito: Ultimate Multisite ten un add-on útil chamado Admin Pages Creator, que básicamente te permite añadir Páxinas de Administrador totalmente personalizadas ao dashboard dos teus clientes usando templates de Elementor, Beaver Builder e Brizy.

* Duas temáticas de Administrador diferentes para personalizar a experiencia do dashboard dos teus clientes: Check Material WP e PRO.

E moito máis avanzado:

* Soporte para Sobrescrituras de Templates (Template Overrides): Con Ultimate Multisite, podes substituir os templates HTML predeterminados para o flujo de inscripción con as túas propias versiones sen preocuparte de cambiar os ficheiros centrais do plugin.

* API de Pesquisa e Substitución (Search and Replace API): TAMPO podes anotar os templates do sitio con marcadores como {{site_name}} e usar a API de Pesquisa e Substitución para preencher esos marcadores automáticamente coa información que o teu cliente proporciona logo después da inscripción, automático.

## Casos de uso

Agora é a altura de ver algúns dos posibles casos de uso para Ultimate Multisite.

### Caso 1: Un Agência

Tipicamente as habilidades centrais dun agência residen no diseño de sitios web con aspectos como o seu albergue ou marketing listados como servizos adicionais.

Para as agencias, Ultimate Multisite presenta unha propuesta de valor increíble na súa capacidade de albergue e tamodar múltiples sitios web en unha única plataforma. Aún máis moitas agencias que estandardizan os seus diseños en temáticas específicas como GeneratePress, Astra, OceanWP ou outras poden aprovechar as capacidades de Ultimate Multisite para activar automáticamente estas temáticas para cada novo sitio.

O controlo estrito dos plugins e das temáticas se mantén por producto a través das configuración de Ultimate Multisite, que permiten que os plugins e as temáticas se poidan tornar dispoñibles ou ocultas así como o estado da súa activación ao instanciarse para un novo sitio.

As agencias encontrar paz de espírito con Ultimate Multisite, que les permite hacer lo que mejor saben hacer: diseñar sitios web excepcionales.

### Caso 2: Proveedor de Nicho

Hay un viejo dicho que dice: "Haz una cosa y hazla bien". Para muchos especialistas, esto significa crear un producto o servicio en torno a una idea central única.

Si te centras en desarrollar una red de sitios de nicho, probablemente querrás ofrecer servicios basados en un marco y plataforma comunes. ¡Ahí es donde las plantillas de sitio pueden ayudar!

Una de las características innovadoras de Ultimate Multisite, un sitio con plantilla (template site), es aquel donde el tema ha sido instalado y activado, los plugins necesarios instalados y activados, y se han creado publicaciones o páginas de muestra. Cuando un cliente crea un nuevo sitio basado en la plantilla, el contenido y la configuración de la plantilla se copian al sitio recién creado.

Para un proveedor de sitios y servicios de nicho, esto proporciona una ventaja inigualable en la capacidad de crear instantáneamente un sitio listo para usar con plugins y diseño personalizados. El cliente solo necesita proporcionar una entrada mínima para completar el servicio.

### Caso 3: Alojamiento Web de WordPress

Hay una miríada de formas de alojar sitios de WordPress, pero rara vez es tan simple como proporcionar espacio web a un cliente con una versión preinstalada de WordPress. Esto se debe a que muchas decisiones y consideraciones deben juntarse para ofrecer un servicio significativo.

Ultimate Multisite sobresale en esta área al proporcionar una solución integral llave en mano (turnkey solution) para el alojamiento de sitios de WordPress. La solución incluye los mecanismos básicos para ofrecer servicios de suscripción, cobro de pagos, formularios de pago (checkout forms), cupones de descuento y comunicaciones con el cliente.

Gran parte do traballo essencial para instalar, configurar e manter correctamente un WordPress Multisite é facilitado pelo Ultimate Multisite, porque os administradores de rede só precisam pensar nos aspetos que lhes dizem respeito ao seu serviço ou nicho, como os níveis de produto, preços e ofertas de serviço.

Para os developers que querem integrar com o Ultimate Multisite, a solução também oferece uma API RESTful completa e Webhooks para notificação de eventos.

Sem depender de uma miríade de plugins externos e licenças, o Ultimate Multisite fornece uma solução rica em funcionalidades e comparável à do Wix, Squarespace, [WordPress.com](https://WordPress.com) e outros.
