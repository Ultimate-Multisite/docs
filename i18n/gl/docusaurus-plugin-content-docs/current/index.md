# Documentación de Ultimate Multisite {#ultimate-multisite-documentation}

Benvido á documentación de Ultimate Multisite. Ultimate Multisite transforma WordPress Multisite nunha plataforma completa de sitio web como servizo (WaaS).

## Seccións da documentación {#documentation-sections}

### [Guía de usuario](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

Todo o que necesitas para configurar e xestionar a túa plataforma WaaS. Abrangue instalación, pasarelas de pagamento, asignación de dominios, xestión de clientes e máis.

### [Documentación para desenvolvedores](/developer/hooks) {#developer-documentation}

Referencia técnica para desenvolvedores que constrúen sobre Ultimate Multisite. Inclúe referencia de hooks, documentación da REST API e guías de desenvolvemento de complementos.

### [Complementos](/addons) {#addons}

Documentación para plugins de complemento de pago que amplían Ultimate Multisite con funcións adicionais como integración con WooCommerce, analíticas, venda de dominios e máis.
