# Filtro: wu_stripe_existing_product_id

Filtra o ID do produto que se vai comprobar. Se existe, a nova subscrición usará este produto.

## Parámetros {#parameters}

| Nome | Tipo | Descrición |
|------|------|-------------|
| $product_id | `string` | ID do produto de Stripe que se vai comprobar. |
| $name | `string` | Nome do produto. |
