# Filtro: wu_add_new_domain_modal_fields

Filtra os campos do modal para engadir un novo dominio.

Permite que os addons (p. ex. Domain Seller) modifiquen ou substitúan os campos de rexistro de dominios.

## Parámetros {#parameters}

| Nome | Tipo | Descrición |
|------|------|-------------|
| $fields | `array` | Os campos do formulario. |

### Desde {#since}

- 2.1.0
