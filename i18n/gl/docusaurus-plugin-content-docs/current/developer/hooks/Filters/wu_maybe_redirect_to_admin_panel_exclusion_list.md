# Filtro: wu_maybe_redirect_to_admin_panel_exclusion_list {#filter-wumayberedirecttoadminpanelexclusionlist}

Crea unha lista de exclusión de parámetros que impiden a redirección automática.

Isto é necesario porque, doutro xeito, o maquetador de páxinas non poderá editar a páxina de inicio de sesión unha vez que estea definida.
