# Filtro: wu_generated_username_from_email

Filtra o nome de usuario xerado do cliente.

## Parámetros

| Nome | Tipo | Descrición |
|------|------|-------------|
| $username | `string` | Nome de usuario xerado. |
| $email | `string` | Enderezo de correo electrónico do novo cliente. |
| $new_user_args | `array` | Array de argumentos do novo usuario, quizais incluíndo nomes e apelidos. |
| $suffix | `string` | Cadea que se engade ao nome de usuario para facelo único. |
