# Filter: wu_contains_element

Permite aos desenvolvedores cambiar os resultados da busca inicial.

Isto é útil para builders de terceiros e similares.

## Parámetros

| Nome | Tipo | Descrición |
|------|------|-------------|
| $contains_elements | `bool` | Se o elemento está contido no contido. |
| $content | `string` | O contido que se está examinando. |
| $element | `self` | O elemento actual. |
| $post | `null\|\WP_Post` | post que comprobar. |
