# Filtro: wu_plan_product_types

Filtra os tipos de produto que deben tratarse como plans.

Este filtro permite que os complementos rexistren tipos de produto adicionais que deben recoñecerse como plans na validación e na segregación.

## Parámetros {#parameters}

| Nome | Tipo | Descrición |
|------|------|-------------|
| $plan_types | `array` | Array de tipos de produto para tratar como plans. |

### Desde {#since}

- 2.3.0
