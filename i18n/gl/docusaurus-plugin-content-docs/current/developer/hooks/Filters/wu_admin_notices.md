# Filtro: wu_admin_notices

Permite aos desenvolvedores filtrar os avisos de administración engadidos por Ultimate Multisite.

## Parámetros {#parameters}

| Nome | Tipo | Descrición |
|------|------|-------------|
| $notices | `array` | Lista de avisos para ese panel en particular. |
| $all_notices | `array` | Lista de avisos engadidos, segregados por panel. |
| $panel | `string` | Panel do que recuperar os avisos. |
| $filter | `string` | Se os avisos descartables foron filtrados. |
| $dismissed_messages | `array` | Lista de claves de avisos descartados. |

- 2.0.0
