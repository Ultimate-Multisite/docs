# Filter: wu_independent_billing_cycle_product_types

Filtra os tipos de produto que teñen ciclos de facturación independentes.

Os produtos con ciclos de facturación independentes non están obrigados a coincidir co período de facturación do plan no carriño. Isto é útil para produtos como rexistros de dominios que sempre se facturan anualmente, independentemente de se o plan é mensual ou anual.

## Parámetros

| Nome | Tipo | Descrición |
|------|------|-------------|
| $types | `array` | Array de tipos de produto con ciclos de facturación independentes. |
