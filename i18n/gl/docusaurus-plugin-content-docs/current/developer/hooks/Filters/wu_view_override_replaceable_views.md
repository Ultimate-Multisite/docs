# Filtro: wu_view_override_replaceable_views

Permite aos desenvolvedores engadir cartafoles adicionais á lista substituíble.

Ten coidado, xa que permitir cartafoles adicionais podería facer que se carguen copias desactualizadas en lugar das versións de Ultimate Multisite.

## Parámetros {#parameters}

| Nome | Tipo | Descrición |
|------|------|-------------|
| $replaceable_views | `array` | Lista de cartafoles permitidos. |

### Desde {#since}

- 2.0.0
