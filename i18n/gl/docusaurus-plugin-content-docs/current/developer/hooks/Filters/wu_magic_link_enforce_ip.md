# Filtro: wu_magic_link_enforce_ip

Filtra se se debe aplicar a verificación do enderezo IP.

Establece en falso para permitir que os tokens funcionen desde redes diferentes. Isto reduce a seguridade pero aumenta a usabilidade (por exemplo, para usuarios móbiles que cambian de rede).

## Parámetros

| Nome | Tipo | Descrición |
|------|------|-------------|
| $enforce | `bool` | Se se debe aplicar a coincidencia do enderezo IP. |

- 2.0.0
