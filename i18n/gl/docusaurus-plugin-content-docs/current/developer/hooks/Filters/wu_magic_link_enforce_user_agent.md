# Filtro: wu_magic_link_enforce_user_agent

Filtra se se debe aplicar a verificación do axente de usuario.

Establece en false para permitir que os tokens funcionen en diferentes navegadores/dispositivos. Isto reduce a seguridade pero aumenta a usabilidade.

## Parámetros {#parameters}

| Nome | Tipo | Descrición |
|------|------|-------------|
| $enforce | `bool` | Se se debe aplicar a coincidencia do axente de usuario. |

### Desde {#since}

- 2.0.0
