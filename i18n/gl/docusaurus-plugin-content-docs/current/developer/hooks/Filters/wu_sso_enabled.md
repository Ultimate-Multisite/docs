# Filtro: wu_sso_enabled {#filter-wussoenabled}

Activa/desactiva a capacidade de inicio de sesión único entre dominios.

Filtra este valor para desactivar por completo o inicio de sesión único, ou para activalo condicionalmente no seu lugar.

## Parámetros {#parameters}

| Nome | Tipo | Descrición |
|------|------|-------------|
| $enabled | `bool` | Debe estar activado SSO? Verdadeiro para activado, equivalente a falso para desactivado. |

### Desde {#since}

- 2.0.11
### Fonte {#source}

Definido en [`inc/sso/class-sso.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/sso/class-sso.php#L110) na liña 110


## Devolve {#returns}
Se SSO está activado ou non.
