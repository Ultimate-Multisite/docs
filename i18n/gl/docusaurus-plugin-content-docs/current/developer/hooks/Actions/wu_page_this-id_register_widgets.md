# Action: wu_page_\{$this->id\}_register_widgets

Dispárase despois de que se rexistren os widgets para esta páxina.

The dynamic portion of the hook name, `$this-&gt;id`, refers to the page id.

## Parámetros

| Nome | Tipo | Descrición |
|------|------|-------------|
| $id | `string` | O id da páxina. |
| $page_hook | `string` | O hook da páxina. |
| $page | `object` | O obxecto da páxina. |
