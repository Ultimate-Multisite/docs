# Action: wu_\{$this->model\}_post_save {#action-wuthis-modelpostsave}

| Nome | Tipo | Descrición |
|------|------|-------------|
| $data | `array` | Os datos do obxecto que se gardarán. |
| $this | `\Base_Model` | A instancia do obxecto. |
| $new | `bool` | Verdadeiro se o obxecto é novo. |
