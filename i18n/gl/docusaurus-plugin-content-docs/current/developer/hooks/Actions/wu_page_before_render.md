# Action: wu_page_before_render

Permite aos desenvolvedores de plugin engadir contido adicional antes de que imprimamos a páxina.

## Parámetros

| Name | Type | Description |
|------|------|-------------|
| $page_id | `string` | O id desta páxina. |
| $page | `object` | O obxecto da páxina. |
