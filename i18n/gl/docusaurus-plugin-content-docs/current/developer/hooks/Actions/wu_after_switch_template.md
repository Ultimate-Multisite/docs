# Acción: wu_after_switch_template

Permite aos desenvolvedores de plugins enganchar funcións despois de que un usuario ou superadministrador cambie o modelo do sitio

## Parámetros

| Nome | Tipo | Descrición |
|------|------|-------------|
| $id | `int` | ID do sitio |
