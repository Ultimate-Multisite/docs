# Action: wu_dashboard_\{$this->tab\}_widgets

Permite aos desenvolvedores de complementos engadir widgets ao panel de Dashboard da rede.

## Parámetros

| Nome | Tipo | Descrición |
|------|------|-------------|
| $tab | `string` | A lapela actual. |
| $screen | `\WP_Screen` | O obxecto de pantalla. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Instancia da páxina de administración de Ultimate Multisite. |
