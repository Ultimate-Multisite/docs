# Acción: wu_mcp_adapter_initialized {#action-wumcpadapterinitialized}

Permite que outros complementos e temas rexistren as súas propias capacidades.

## Parámetros {#parameters}

| Nome | Tipo | Descrición |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | A instancia do adaptador MCP. |
