# פֿילטער: wu_apply_plan_limits {#filter-wuapplyplanlimits}

דערלויבן צוגעב־אַנטוויקלערס צו איבערשפּרינגען די באַגרענעצונגען.

איר קענט נוצן דעם פֿילטער צו לויפֿן וועלכן קאָד איר ווילט איידער סיי וועלכע פֿון די באַגרענעצונגען ווערן אָנגעהויבן. אויב אייער פֿילטער גיט צוריק אַ אמת־ווערטיקן ווערט, וועט דער פּראָצעס ווײַטערגיין; אויב ער גיט צוריק אַ פֿאַלש־ווערטיקן ווערט, וועט דער קאָד זיך אומקערן און קיינער פֿון די הוקס אונטן וועט נישט לויפֿן.
