# Gratis AI Agent {#gratis-ai-agent}

Gratis AI Agent איז אַן אויטאָנאָמער WordPress AI־אַסיסטענט וואָס קען פּלאַנירן, בויען, און פירן גאַנצע WordPress זייטלעך דורך אנווייזונגען אין נאַטירלעכער שפּראַך. ער שטעלט אַרויס אַ סטרוקטורירטן זאַמלונג פֿון **פֿעיִקייטן** — באַזונדערע, טעסטבאַרע אַקציעס וואָס דער אַגענט קען אַרויפֿרופן — וואָס דעקן אַלץ פֿון אינהאַלט־פֿאַרוואַלטונג ביז צופּאַסונג פֿון אַ דיזיין־סיסטעם. ווערסיע 1.9.0 לייגט צו פֿעיִקייטן פֿאַר שאַפֿן אינהאַלט, וויזועלע בלאַט־איבערקוק דורך קליענט־זייטיקע סקרינשאַטן, פֿינף אײַנגעבויטע אַגענט־פּראָפֿילן, און פֿונקציע־פֿאָן פֿאַר צוטריט־קאָנטראָל און בראַנדינג.

## הויפּט־פֿעיִקייטן {#key-features}

- **פֿעיִקייטן־סיסטעם** — אַ מאָדולאַרער, אויסברייטערבאַרער קאַטאַלאָג פֿון אַקציעס וואָס דער אַגענט קען אַרויפֿרופן אויף אײַער WordPress אינסטאַלאַציע
- **אינהאַלט־פֿאַרוואַלטונג** — שאַפֿן פּאָסטן און בלעטער (אַרײַנגערעכנט צוטיילן בלאַט־מוסטערן), שאַפֿן קייפּל פּאָסטן אין איין רוף, שטעלן אויסגעצייכנטע בילדער, און בויען קאָנטאַקט־פֿאָרמען
- **וויזועלע איבערקוק** — כאַפּן סקרינשאַטן פֿון לעבעדיקע בלעטער, פֿאַרגלײַכן פֿריִער/נאָך־צושטאַנדן, און באַקומען אַן AI־גענערירטע דיזיין־איבערקוק וואָס דעקקט אויסלייג, טיפאָגראַפֿיע, קאָליר, און צוטריטלעכקייט
- **פֿינף אײַנגעבויטע אַגענטן** — אינהאַלט־שרײַבער, זייטל־בויער, דיזיין־סטודיאָ, פּלוגין־פֿאַרוואַלטער, און שטיצע־אַסיסטענט; איבערשטעלבאַר דורך דעם אַגענט־אויסקלײַבער אין דעם שמועס־קעפּל
- **פֿונקציע־פֿאָן** — צוטריט־קאָנטראָל און בראַנדינג־איבערשטעלערס אין איינשטעלונגען → פֿונקציע־פֿאָן פֿאַר ראָלע־באַגרענעצונגען, ווײַס־לייבל נעמען, און ווידזשעט־צופּאַסונג
- **פֿאַרוואַלטונג פֿון אייגענע פּאָסט־טיפּן** — רעגיסטרירן, אויסליסטן, און אויסמעקן אייגענע פּאָסט־טיפּן מיט בלײַביקייט איבער ווידער־סטאַרטן
- **פֿאַרוואַלטונג פֿון אייגענע טאַקסאָנאָמיעס** — רעגיסטרירן, אויסליסטן, און אויסמעקן אייגענע טאַקסאָנאָמיעס מיט בלײַביקייט
- **דיזיין־סיסטעם פֿעיִקייטן** — אַרײַנשפּריצן אייגענע CSS, פֿאַרוואַלטן אויסגעקליבענע בלאָק־מוסטערן, שטעלן דעם זייטל־לאָגאָ, און אָנווענדן theme.json פּרעסעטן
- **גלאבאַלע סטילן־פֿאַרוואַלטונג** — לייענען און שרײַבן theme.json ווערטן פֿאַר קאָליר־פּאַלעטן, טיפאָגראַפֿיע, אָפּרוק, און מער
- **נאַוויגאַציע־מעניו־פֿאַרוואַלטונג** — שאַפֿן און דערהײַנטיקן נאַוויגאַציע־מעניוס און זייערע פּונקטן פּראָגראַמאַטיש
- **אָפּציעס־פֿאַרוואַלטונג** — לייענען און שרײַבן WordPress אָפּציעס מיט אַן אײַנגעבויטער זיכערהייט־בלאָקליסט כּדי צו באַשיצן קריטישע איינשטעלונגען
- **אינסטאַלירבאַרער פֿעיִקייטן־רעגיסטר** — אַנטדעקן און אַקטיווירן נאָך פֿעיִקייטן־פּעקלעך אָן ענדערן קערן־פּלוגין טעקעס
- **זייטל־בויער אָרקעסטעראַציע v2** — מער־טריטיקע זייטל־בוי־פּלענער מיט פּלוגין־אַנטדעקונג, פּראָגרעס־נאָכפֿאָלג, און טעות־אויפֿריכטונג
- **WP-CLI בענטשמאַרק־באַפֿעל** — לויפֿן די אַגענט־מעגלעכקייט בענטשמאַרק־סוויט פֿון דער באַפֿעל־שורה פֿאַר CI און מאָדעל־אָפּשאַצונג

## פֿאָדערונגען {#requirements}

- WordPress 6.0 אָדער העכער
- PHP 8.1 אָדער העכער
- WordPress Multisite (נעץ־אַקטיווירט) אָדער איין־זייטל אינסטאַלאַציע
- אַן אַקטיווער AI־פֿאַרזאָרגער פֿאַרבינדער־פּלוגין (Anthropic, OpenAI, אָדער קאַמפּאַטאַבעל)

## אינסטאַלאַציע {#installation}

1. אַרויפֿלאָדן די פּלוגין־טעקעס צו `/wp-content/plugins/gratis-ai-agent`
2. אַקטיווירן דעם פּלוגין דורך דעם **פּלוגינס** עקראַן אין WordPress
3. גיין צו **Gratis AI Agent → איינשטעלונגען** און פֿאַרבינדן אײַער AI־פֿאַרזאָרגער
4. (אָפּציאָנעל) דורכקוקן דעם **פֿעיִקייטן־רעגיסטר** כּדי צו אינסטאַלירן נאָך פֿעיִקייטן־פּעקלעך

## שנעלער אָנהייב {#quick-start}

ווען אַמאָל אַקטיווירט און פֿאַרבונדן מיט אַן AI־פֿאַרזאָרגער, עפֿנט דעם **Gratis AI Agent** פּאַנעל אין דער WordPress אַדמין־זײַטיקער לײַסטע. טיפּט אַן אנווייזונג אין נאַטירלעכער שפּראַך און דער אַגענט וועט פּלאַנירן און אויספֿירן די נייטיקע טריט ניצנדיק זײַנע בנימצא פֿעיִקייטן.

**בײַשפּיל־אנווייזונגען:**

```
שאַף אַ "Portfolio" אייגענעם פּאָסט־טיפּ מיט אַ "Project Category" טאַקסאָנאָמיע.
```

```
שפּריץ אַרײַן אַ טונקל־מאָדוס קאָליר־פּאַלעט אין דער טעמע און שטעל דעם הויפּט־קאָליר אויף #1a1a2e.
```

```
בוי אַ רעסטאָראַן־וועבזײַט מיט אַן אָנליין מעניו, רעזערווירונג־פֿאָרעם, און קאָנטאַקט־בלאַט.
```

## איבערבליק פֿון פֿעיִקייטן {#abilities-overview}

פֿעיִקייטן זענען די אַטאָמישע אַקציעס וואָס דער אַגענט קען אויספֿירן. יעדע פֿעיִקייט איז אַ רעגיסטרירטע PHP קלאַס וואָס שטעלט אַרויס אַ סכעמע (פּאַראַמעטערס און צוריקקער־טיפּן) וואָס דער אַגענט לייענט בײַ לויף־צײַט. פֿעיִקייטן זענען גרופּירט אין פֿונקציאָנעלע געביטן:

| געביט | פֿעיִקייטן |
|---|---|
| **אינהאַלט־פֿאַרוואַלטונג** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **וויזועלע איבערקוק** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **אייגענע פּאָסט־טיפּן** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **אייגענע טאַקסאָנאָמיעס** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **דיזיין־סיסטעם** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **גלאבאַלע סטילן** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **נאַוויגאַציע־מעניוס** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **אָפּציעס־פֿאַרוואַלטונג** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **אינסטאַלירבאַרע פֿעיִקייטן** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **זייטל־בויער** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

זעט די [פֿעיִקייטן־רעפֿערענץ](./abilities) פֿאַר פולע פּאַראַמעטער־דאָקומענטאַציע און באַניץ־בײַשפּילן.

## WP-CLI {#wp-cli}

Gratis AI Agent קומט מיט אַ `wp gratis-ai-agent` באַפֿעל־משפּחה פֿאַר לויפֿן בענטשמאַרקס און פֿאַרוואַלטן פֿעיִקייטן פֿון דער באַפֿעל־שורה. זעט די [WP-CLI רעפֿערענץ](./wp-cli) פֿאַר דער פולער באַפֿעל־ליסטע.

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## קאָנפֿיגוראַציע {#configuration}

אַלע איינשטעלונגען זענען אונטער **Gratis AI Agent → איינשטעלונגען** אין דער WordPress אַדמין:

| איינשטעלונג | באַשרײַבונג |
|---|---|
| **AI־פֿאַרזאָרגער** | אויסקלײַבן און קאָנפֿיגורירן דעם אַקטיוון AI־פֿאַרזאָרגער פֿאַרבינדער |
| **מאָדעל** | אויסקלײַבן דעם שפּראַך־מאָדעל צו ניצן פֿאַר אַגענט־סעסיעס |
| **מאַקסימום טאָקענס** | טאָקען־בודזשעט פּער בקשה (נאָרמאַל 4096) |
| **פֿעיִקייט־צײַטגרענעץ** | מאַקסימום סעקונדעס וואָס אַן אינדיווידועלע פֿעיִקייט קען לויפֿן (נאָרמאַל 30) |
| **דעבאַג־לאָגירן** | לאָגירן פֿעיִקייט־רופֿן און אַגענט־באַגרינדונג אין דעם WordPress דעבאַג־לאָג |

## אָפּציעס־פֿאַרוואַלטונג זיכערהייט־בלאָקליסט {#options-management-safety-blocklist}

די אָפּציעס־פֿאַרוואַלטונג פֿעיִקייט נעמט אַרײַן אַן אײַנגעבויטע בלאָקליסט וואָס פֿאַרהיט דעם אַגענט פֿון לייענען אָדער שרײַבן סענסיטיווע WordPress אָפּציעס. די נאָרמאַלע בלאָקליסט דעקקט:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (URL אָפּציעס — ניצט אַ באַזונדער געצײַג פֿאַר מיגראַציע פֿאַר די)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (געהיימע שליסלען)
- `active_plugins`, `template` (פּלוגין/טעמע אַקטיוואַציע — ניצט באַזונדערע פֿעיִקייטן)
- צאָלונג־גייטוויי קרעדענציאַלן און API שליסלען רעגיסטרירט דורך אינסטאַלירטע פּלוגינס

נאָך איינטראָגן קענען צוגעלייגט ווערן דורך דעם `gratis_ai_agent_options_blocklist` פֿילטער.

## Hooks און פֿילטערס {#hooks-and-filters}

### אַקציעס {#actions}

- `gratis_ai_agent_ability_registered` — ווערט אויסגעלאָזט נאָך אַ פֿעיִקייט ווערט רעגיסטרירט, באַקומען דעם פֿעיִקייט־קלאַס נאָמען
- `gratis_ai_agent_plan_started` — ווערט אויסגעלאָזט ווען אַ זייטל־בויער פּלאַן הייבט אָן אויספֿירונג
- `gratis_ai_agent_plan_completed` — ווערט אויסגעלאָזט ווען אַ פּלאַן ענדיקט זיך מיט הצלחה
- `gratis_ai_agent_plan_failed` — ווערט אויסגעלאָזט ווען אַ פּלאַן באַגעגנט אַן אומאויפֿריכטבאַרן טעות

### פֿילטערס {#filters}

- `gratis_ai_agent_abilities` — ענדערן די רעגיסטרירטע פֿעיִקייטן־ליסטע איידער דער אַגענט לאָדט זי
- `gratis_ai_agent_options_blocklist` — אויסברייטערן אָדער פֿאַרקלענערן די אָפּציעס־זיכערהייט בלאָקליסט
- `gratis_ai_agent_benchmark_questions` — צולייגן אייגענע בענטשמאַרק־פֿראַגעס צו דער אָפּשאַצונג־סוויט
- `gratis_ai_agent_plan_plugins` — ענדערן די פּלוגין־אַנטדעקונג ליסטע בעת זייטל־פּלאַן גענעראַציע
