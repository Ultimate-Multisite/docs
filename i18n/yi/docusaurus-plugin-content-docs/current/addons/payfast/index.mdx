# PayFast

Multisite Ultimate: Payfast Gateway

לייגט צו Payfast — אַ שנעלן און זיכערען אָנליין־צאָלונגען־טויער אין דרום־אַפֿריקע — ווי אַ Ultimate Multisite צאָלונג־אָפּציע.

## אינסטאַלאַציע {#installation}

1. לאָדט אַרויף 'ultimate-multisite-payfast' אין דעם '/wp-content/plugins/' Directory
2. אַקטיווירט דעם פּלוגין דורך דעם 'Plugins' מעניו אין WordPress
3. פֿאָלגט דעם שריט־בײַ־שריט Wizard כּדי אויפֿצושטעלן דעם פּלוגין

## באַניץ־ווײַזער {#usage-guide}

PayFast איז אַ פֿירנדיקער אָנליין־צאָלונג־פּראָצעסאָר פֿאַר ביזנעסער אין דרום־אַפֿריקע. עס איז געווידמעט צו העלפֿן אָנליין־סוחרים וואַקסן דורך מאַכן טראַנסאַקציעס גלאַט און זיכער. לויט דער בקשה פֿון Ultimate Multisite באַניצער, האָבן מיר באַשלאָסן צו שאַפֿן אַן add-on וואָס וועט מאַכן מעגלעך אינטעגראַציע מיט PayFast. דער אַרטיקל וועט ווײַזן ווי אַזוי אויפֿצושטעלן PayFast ווי אייער צאָלונג־טויער כּדי צו באַקומען צאָלונגען אין אייער Ultimate Multisite נעץ.

דער ערשטער שריט איז צו אינסטאַלירן דעם add-on.

# אויפֿשטעלן PayFast

נאָך אינסטאַלירן דעם add-on, קענט איר איצט גיין צו Ultimate Multisite settings און אָנהייבן קאָנפֿיגורירן PayFast.

אונטער דער **Ultimate Multisite settings** זײַט, קענט איר געפֿינען דעם "Payments" קוויטל. אונטער דעם קוויטל זאָלט איר זען "**Payfast** " צוזאַמען מיט אַ toggle קנעפּל צו אַקטיווירן אָדער דיסייבלען דעם צאָלונג־טויער.

![צאָלונג־טויער אײַנשטעלונגען](/img/config/settings-payment-gateways.png)גייט ווײַטער און דריקט דעם toggle קנעפּל צו אַקטיווירן עס. דערנאָך וועט דאָס אויסברייטערן דעם קעסטל צו ווײַזן פֿאַרשיידענע פֿעלדער וואָס איר דאַרפֿט אויספֿילן, אַזעלכע ווי **Mechant ID, Merchant Key and PassPhrase.** איר קענט באַקומען די אינפֿאָרמאַציע פֿון אייער **Payfast** Account
