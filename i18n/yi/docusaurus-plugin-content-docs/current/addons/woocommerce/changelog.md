ווערסיע 2.2.0 - אַרויסגעלאָזט דעם 2026-07-01
* נײַ: Ultimate Multisite שטייער־סומעס ווײַזן זיך איצט ווי באַזונדערע WooCommerce אָפּצאָל־שורות בײַ checkout, וואָס מאַכט שטייער־סך־הכּל קלאָרער פֿאַר צאָלונג.
* נײַ: צוגעגעבן אַן opt-in "אָנהאַלטן מיטגלידערשאַפֿטן גלײַך בײַ דורכגעפֿאַלענע באנײַונגען" אײַנשטעלונג און `wu_woo_suspend_on_payment_failure` filter פֿאַר וועבזײַטן וואָס ווילן דורכגעפֿאַלענע־באַנײַונג אָנהאַלטונג בעת דער WooCommerce Subscriptions retry window.
* פֿאַרריכטונג: אויסגעגליכן WooCommerce Subscription סטאַטוסן וואָס האָבן געקענט בלײַבן נישט סינכראָניזירט מיט Ultimate Multisite מיטגלידערשאַפֿטן נאָך דורכגעפֿאַלענע אָדער צוריקגעשטעלטע באַנײַונגען.
* פֿאַרריכטונג: צוגעגעבן די WooCommerce קראָם־וואַלוטע צו דער Ultimate Multisite וואַלוטע־רשימה ווען זי האָט געפֿעלט.
* פֿאַרריכטונג: אָפּגעהיט קונה־בילינג־דעטאַלן ווען מען ווײַזט אַבאָנענטן ווײַטער צו WooCommerce checkout.
* פֿאַרבעסערט: צוגעגעבן קאַמפּאַטיביליטעט מיט Jetpack Autoloader 5.
* פֿאַרבעסערט: אויסגערייניקט די שאַפֿונג פֿון אַרויסלאָז־פּעקל, כּדי GitHub און מאַרקפּלאַץ־zips זאָלן אויסמײַדן נעסטעד staging־טעקע־אָרדנערס און אַנטוויקלונג־טעקעס.
