# פילטער: wp_ultimo_woocommerce_checkout_fields_to_clean {#filter-wpultimowoocommercecheckoutfieldstoclean}

דערלויבט דעוועלאָפּערס צו פֿילטערן די פעליקייַט־WooCommerce רעכענונג־אַדרעס־פֿעלדער.

וויבאַלד Ultimate Multisite האָט רעכענונג־אַדרעס־פֿעלדער, קען עס האָבן זין צו באַזײַטיקן די פעליקייַט־פֿעלדער וואָס WooCommerce ווײַזט אויף דער checkout־פֿאָרם. לויט פעליקייַט ווערט קיין ווערט נישט איבערגעגעבן צו דער רשימה, אַזוי אַז אַלע פֿעלדער וואָס זענען קאָנפֿיגורירט אין WooCommerce צו ווערן געוויזן ווערן געהאַלטן.

### זינט {#since}

- 2.0.0

די רשימה פֿון פֿעלדער צו באַזײַטיקן. מעגלעכע ווערטן אַרײַננעמען billing_first_name, billing_last_name, billing_company, billing_country, billing_address_1, billing_address_2, billing_city, billing_state, billing_postcode, billing_email, און billing_phone.
