# VAT ענדערונגען־לאָג {#vat-changelog}

ווערסיע 1.0.7 - ארויסגעגעבן אויף 2026-02-03

* פֿאַרריכטונג: דײַטשישע (DE) VAT נומערן האָבן נישט דורכגעגאַנגען וואַלידאַציע צוליב VIES SOAP service MS_MAX_CONCURRENT_REQ פֿעלערן. איבערגעשטעלט פֿון דער אַלטער SOAP API צו דער אָפֿיציעלער EU VIES REST API פֿאַר מער פֿאַרלעסלעכע וואַלידאַציע.
* פֿאַרריכטונג: VAT נומערן ווערן איצט אָנגענומען ווען דער VIES דינסט איז צײַטווײַליק נישט בנימצא, אַנשטאָט צו ווערן אומריכטיק אָפּגעוואָרפֿן. אַ לאָג־אײַנטראָג ווערט געשאַפֿן פֿאַר שפּעטערדיקער איבערקוקונג.
* פֿאַרריכטונג: VAT/שטייער־אידענטיפיקאַציע פֿעלד איז איצט זעבאַר פֿאַר אַלע לענדער, נישט נאָר פֿאַר EU מיטגלידער. נישט־EU שטייער־אידענטיפיקאַציעס (למשל שווייצער CHE נומערן) ווערן אויפֿגעהיט פֿאַר רעכענונג־ווײַזונג אָן VIES וואַלידאַציע.
* פֿאַרריכטונג: VAT פּרעפֿיקס־זוכונג פֿאַר גריכנלאַנד (GR), מאָנאַקאָ (MC), און Isle of Man (IM) איז געווען צעבראָכן צוליב אומריכטיקער array זוכונג־לאָגיק.
* פֿאַרבעסערונג: אַוועקגענומען אומנייטיקע wp-cli-bundle dev dependency וואָס האָט געקענט גורם זײַן fatal errors ווען געניצט צוזאַמען מיטן Site Exporter צוגאב.
* פֿאַרבעסערונג: צוגעגעבן אַ ברייטע unit test סוויטע (53 טעסטן).
