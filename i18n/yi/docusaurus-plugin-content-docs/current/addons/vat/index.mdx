# VAT

Multisite Ultimate: אייראָפּעיִשער VAT

לייגט צו נאָך פֿעיִקייטן און מכשירים צו Multisite Ultimate'ס פעליקייַט־שטײַער־באַהאַנדלונג, כּדי בעסער צו שטיצן אייראָפּעיִשער VAT.
לייגט אויטאָמאַטיש צו EU VAT ראַטעס צו דער בלאַט פֿון שטײַער־ראַטעס פֿון

## אינסטאַלאַציע {#installation}

1. אַרויפֿלאָדן 'ultimate-multisite-vat' אין דעם '/wp-content/plugins/' וועגווײַזער
2. אַקטיווירן דעם plugin דורך דעם 'Plugins' מעניו אין WordPress
3. גיין צו Ultimate Multisite > באַשטעלונגען > שטײַערן און אַקטיווירן EU VAT.
4. גיי קליק אויף שטײַער־ראַטעס און דערהײַנטיקן EU VAT ראַטעס אונטן און אויפֿהיטן שטײַער־ראַטעס.
5. פֿאַרטיק!
