# מערערע וואלוטעס

Ultimate Multisite: מערערע וואלוטעס דערמעגלעכט אײַער נעץ אָנצונעמען צאָלונגען אין מערערע וואלוטעס. עס וואַנדלט אויטאָמאַטיש איבער פּרײַזן מיט לעבעדיקע אָדער מאַנועלע אויסטויש־קורסן, לאָזט אײַך באַשטימען גענויע פּרײַזן פּער פּראָדוקט פֿאַר וועלכער וואלוטע עס זאָל זײַן, און דערקענט יעדער באַזוכערס בילכער וואלוטע דורך געאָלאָקאַציע — אַלץ פֿאַרוואַלטעט פֿון איין איינציקן אַדמין Dashboard.

## הויפּט־אייגנשאַפֿטן

- **אויטאָמאַטישע אויסטויש־קורס־איבערוואַנדלונג** — פּרײַזן ווערן איבערגעוואַנדלט פֿון אײַער באַזע־וואלוטע מיט קורסן וואָס ווערן געבראַכט פֿון אײַער אויסגעקליבענעם צושטעלער
- **פּרײַז־איבערשרײַbungen פּער פּראָדוקט** — שטעלט איין גענויע וואלוטע־ספּעציפֿישע פּרײַזן אויף וועלכן פּראָדוקט עס זאָל זײַן כּדי איבערצוגיין די אויטאָמאַטישע איבערוואַנדלונג
- **וואלוטע־אויסקלײַבער־פֿעלד בײַ אײַנשרײַבונג** — לאָזט קונים אויסקלײַבן זייער בילכער וואלוטע בעת באַצאָלונג־אָפּשלוס/אײַנשרײַבונג
- **מערערע אויסטויש־קורס־צושטעלער** — ECB, Frankfurter, Stripe, אָדער גאָר מאַנועלע קורסן
- **וואלוטע־דערקענונג לויט געאָלאָקאַציע** — קלײַבט אויטאָמאַטיש פֿאָרויס אויס די וואלוטע וואָס פּאַסט צום באַזוכערס לאַנד
- **אַדמין Dashboard** — פֿאַרוואַלטעט אַקטיווע וואלוטעס, אויסטויש־קורסן, און צושטעלער־אײַנשטעלונגען פֿון איין אָרט

## באַדערפֿענישן

- Ultimate Multisite 2.0.0 אָדער העכער
- WordPress Multisite אינסטאַלאַציע
- PHP 7.4 אָדער העכער

## אינסטאַלאַציע

1. אַרויפֿלאָדט די צושטעק־טעקעס צו `/wp-content/plugins/ultimate-multisite-multi-currency`
2. אַקטיווירט דעם צושטעק דורך דעם **צושטעקן**־עקראַן אין WordPress (נעץ־אַקטיוויזירן)
3. גייט צו **Ultimate Multisite → מערערע וואלוטעס** צו עפֿענען דעם אַדמין Dashboard
4. לייגט צו די וואלוטעס וואָס איר ווילט אָננעמען און קאָנפֿיגורירט אײַער אויסטויש־קורס־צושטעלער

## קאָנפֿיגוראַציע

### אַדמין Dashboard

דער מערערע־וואלוטעס אַדמין Dashboard (**Ultimate Multisite → מערערע וואלוטעס**) איז דער צענטראַלער אָרט צו:

- צולייגן, אַקטיווירן, אָדער דיאַקטיווירן וואלוטעס
- אויסקלײַבן אַן אויסטויש־קורס־צושטעלער און דערפֿרישן קורסן לויט פֿאַרלאַנג
- איבערקוקן איצטיקע קורסן פֿאַר אַלע אַקטיווע וואלוטעס
- שטעלן מאַנועלע קורסן ווען אויטאָמאַטישע צושטעלער פּאַסן נישט

### אויסטויש־קורס־צושטעלער

| צושטעלער | באַשרײַבונג |
|---|---|
| **ECB** | אייראָפּעיִשע צענטראַל־באַנק — אומזיסטע טעגלעכע קורסן פֿאַר EUR־באַזירטע נעצן |
| **Frankfurter** | אָפֿן־קוואַל ECB שפּיגל־API — אומזיסט, קיין API שליסל נישט נייטיק |
| **Stripe** | ניצט Stripe'ס לעבעדיקע FX קורסן — פֿאָדערט אַן אַקטיווע Stripe אינטעגראַציע |
| **מאַנועל** | איר גיט אַרײַן קורסן דירעקט; קיין דרויסנדיקע API רופֿן |

קלײַבט אויס אײַער צושטעלער אונטער **Ultimate Multisite → מערערע וואלוטעס → אײַנשטעלונגען → אויסטויש־קורס־צושטעלער**. קורסן קען מען דערפֿרישן מאַנועל אין יעדער צײַט אָדער פּלאַנירן זיי זאָלן זיך דערהײַנטיקן אויטאָמאַטיש.

### פּרײַז־איבערשרײַbungen פּער פּראָדוקט

ווען אויטאָמאַטישע איבערוואַנדלונג איז נישט גענוי גענוג, קענט איר שטעלן אַן פּינקטלעכן פּרײַז פֿאַר אַ ספּעציפֿישער וואלוטע אויף וועלכן פּראָדוקט עס זאָל זײַן:

1. עפֿנט דעם פּראָדוקט־רעדאַקטאָר (**Ultimate Multisite → פּראָדוקטן → רעדאַקטירן**)
2. רוקט אַראָפּ צו דער **וואלוטע־פּרײַזן** אָפּטיילונג
3. גיט אַרײַן דעם גענויען פּרײַז פֿאַר יעדער וואלוטע וואָס איר ווילט איבערשרײַבן
4. היט אָפּ דעם פּראָדוקט

פּראָדוקטן אָן אַן איבערשרײַבונג ניצן דעם איבערגעוואַנדלטן פּרײַז פֿון אײַער באַזע־וואלוטע.

### וואלוטע־אויסקלײַבער־פֿעלד בײַ אײַנשרײַבונג

דאָס וואלוטע־אויסקלײַבער־פֿעלד קען צוגעלייגט ווערן צו אײַער אײַנשרײַבונג/באַצאָלונג־אָפּשלוס־פֿלוס, אַזוי אַז קונים קלײַבן אויס זייער בילכער וואלוטע בײַ רעגיסטראַציע:

1. גייט צו **Ultimate Multisite → מערערע וואלוטעס → אײַנשטעלונגען**
2. אַקטיווירט **ווײַזן וואלוטע־אויסקלײַבער בײַ אײַנשרײַבונג**
3. אויב געוואָלט, שטעלט אַ פעליקייט־וואלוטע אָדער לאָזט געאָלאָקאַציע באַשטימען דעם פֿאָרויס־אויסגעקליבענעם ווערט

די אויסגעקליבענע וואלוטע ווערט אָפּגעהיט מיטן אַבאָנעמענט און געניצט פֿאַר אַלע שפּעטערדיקע בילינג.

### וואלוטע־דערקענונג לויט געאָלאָקאַציע

ווען געאָלאָקאַציע־דערקענונג איז אַקטיווירט, מאַפּט דער צולייג דעם באַזוכערס לאַנד צו אַ וואלוטע און קלײַבט זי פֿאָרויס אויס אינעם וואלוטע־אויסקלײַבער־פֿעלד. צו אַקטיווירן:

1. גייט צו **Ultimate Multisite → מערערע וואלוטעס → אײַנשטעלונגען**
2. אַקטיווירט **דערקענען וואלוטע פֿון באַזוכער־אָרט**
3. מאַפּט לענדער צו וואלוטעס אין דער **געאָלאָקאַציע־מאַפּירונג** טאַבעלע

אויב קיין מאַפּירונג עקזיסטירט נישט פֿאַר אַ באַזוכערס לאַנד, ווערט געניצט די נעצווערק־באַזע־וואלוטע ווי דער צוריקפֿאַל.

## אָפֿט געשטעלטע פֿראַגעס

### וואָס געשעט אויב אַן אויסטויש־קורס־צושטעלער איז נישט בנימצא?

דער צולייג פֿאַלט צוריק צו די לעצטע הצלחהדיק געבראַכטע קורסן. אויב קורסן זענען קיינמאָל נישט געבראַכט געוואָרן, פֿאַלט עס צוריק צו די מאַנועלע קורסן וואָס איר האָט קאָנפֿיגורירט. עס איז רעקאָמענדירט צו קאָנפֿיגורירן מאַנועלע קורסן ווי אַ זיכערהייט־נעץ.

### קען איך ניצן אַנדערע צושטעלער פֿאַר אַנדערע וואלוטעס?

ניין — איין צושטעלער איז אַקטיוו אויף אַ מאָל פֿאַר אַלע וואלוטעס. ניצט **מאַנועל** אויב איר דאַרפֿט פֿולע קאָנטראָל איבער באַזונדערע קורסן.

### דאַרף געאָלאָקאַציע אַ באַזונדערן צושטעק?

קיין דרויסנדיקער געאָלאָקאַציע־צושטעק איז נישט נייטיק. דער צולייג ניצט WordPress'ס אײַנגעבויטע IP־צו־לאַנד דערקענונג. פֿאַר העכערע פּינקטלעכקייט קען מען אינטעגרירן אַ באַזונדערן געאָלאָקאַציע־דינסט דורך די צוגעשטעלטע פֿילטער־הוקס.

### ווערן איבערשרײַbungen פּער פּראָדוקט אָנגעלייגט פֿאַר אָדער נאָך שטײַער?

איבערשרײַbungen פּער פּראָדוקט שטעלן דעם פֿאַר־שטײַער פּרײַז אין דער ציל־וואלוטע. שטײַער ווערט אויסגערעכנט איבערן איבערשרײַבונג־פּרײַז מיט אײַער עקזיסטירנדיקער שטײַער־קאָנפֿיגוראַציע.
