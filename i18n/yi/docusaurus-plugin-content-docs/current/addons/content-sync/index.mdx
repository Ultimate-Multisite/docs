# אינהאַלט־סינכראָניזאַציע {#content-sync}

Multisite Ultimate: אינהאַלט־סינכראָניזאַציע

דער אַדאָן פֿאַר Multisite Ultimate דערמעגלעכט נעץ־אַדמיניסטראַטאָרן צו סינכראָניזירן אינהאַלט פֿון וועלכן פּלאַץ נאָר צו אַנדערע פּלאַץן אין דער נעץ. ער שטיצט אַלע פּאָסט־טיפּן, אַרײַנגערעכנט פּאָסטן, בלעטער, אייגענע פּאָסט־טיפּן, און Elementor מוסטערן. דער אַדאָן נעמט אַרײַן אַ מאַלטי־שריט וויזאַרד־צובינד צו אויסקלײַבן אינהאַלט און ציל־פּלאַץן, מיט פּעקל־פּראָצעסירונג און פּראָגרעס־צוריקמעלדונג אין אמתער צײַט.

## פֿעיִקייטן {#features}

* אינהאַלט־אויסקלײַב: וויזועלער קאַרטלעך־גריד מיט מיניאַטורן און פּאָסט־טיפּ פֿילטערונג
* אַלע פּאָסט־טיפּן: סינכראָניזירט פּאָסטן, בלעטער, פּראָדוקטן, Elementor מוסטערן, און יעדן אייגענעם פּאָסט־טיפּ
* פּלאַץ־אויסקלײַב: קלײַבט אויס צו וועלכע פּלאַץן צו סינכראָניזירן אינהאַלט, מיט אויסשליסונג פֿונעם מקור־פּלאַץ און פֿאָרויס־אויסקלײַב פֿון קינד־פּלאַץן
* פּעקל־פּראָצעסירונג: סינכראָניזירט צו 10 פּלאַץן אויף אַ מאָל מיט פּראָגרעס־דערהײַנטיקונגען
* מעדיע־סינכראָניזאַציע: קאָפּירט און איבערמאַפּט אויטאָמאַטיש מעדיע־טעקעס (בילדער, ווידעאָס) פֿון פּאָסט־אינהאַלט צו ציל־פּלאַץן
* טאַקסאָנאָמיע־סינכראָניזאַציע: היט אָפּ קאַטעגאָריעס, טאַגס, און אייגענע טאַקסאָנאָמיעס
* Elementor שטיצע: פולע שטיצע פֿאַר Elementor מוסטערן מיט CSS רידזשענעראַציע און איבערמאַפּן פֿון מוסטער־ID
* קאָנפֿליקט־לייזונג: אָפּציע צו פאַרבײַטן עקזיסטירנדיקן אינהאַלט אָדער אים איבערשפּרינגען
* Cache־פֿאַרוואַלטונג: רייניקט אויטאָמאַטיש דעם Elementor cache אויף ציל־פּלאַץן נאָך דער סינכראָניזאַציע

## פֿאָדערונגען {#requirements}

* WordPress 5.3 אָדער העכער
* PHP 7.4 אָדער העכער
* Multisite Ultimate פּלוגין (אַקטיוו)
* Elementor פּלוגין (אַפּציאָנעל, נייטיק פֿאַר Elementor מוסטער־סינכראָניזאַציע)

## אינסטאַלאַציע {#installation}

1. אַרויפֿלאָדן 'ultimate-multisite-content-sync' אין דעם '/wp-content/plugins/' Directory
2. אַקטיווירן דעם פּלוגין דורך דעם 'Plugins' מעניו אין אײַער WordPress Network Admin
3. קאָנפֿיגורירן סעטטינגס אין Ultimate Multisite → סעטטינגס → אינהאַלט־סינכראָניזאַציע
4. גיין צו Ultimate Multisite → אינהאַלט־סינכראָניזאַציע כּדי אָנהייבן סינכראָניזירן
