# Content Sync ענדערונגען־לאָג

### ווערסיע 1.1.0 - אַרויסגעלאָזט אויף 2025-12-31 {#version-110---released-on-2025-12-31}
* נײַ: שטיצע פֿאַר סינכראָניזירן אַלע פּאָסט־טיפּן (פּאָסטן, זײַטן, אייגענע פּאָסט־טיפּן)
* נײַ: פּאָסט־טיפּ־פֿילטערונג אין דעם אינהאַלט־אויסקלײַב־שריט
* נײַ: מקור־פּלאַץ־אויסקלײַב־שריט - סינכראָניזירן פֿון וועלכן פּלאַץ עס זאָל נישט זײַן, נישט נאָר פֿון דער הויפּט־פּלאַץ
* נײַ: קינד־פּלאַץ־פֿאָרויס־אויסקלײַב פֿאַר מוסטער־פּלאַץ
* נײַ: מעדיע־אויסציען פֿון post_content פֿאַר אַלע פּאָסט־טיפּן
* נײַ: טאַקסאָנאָמיע־סינכראָניזירן (קאַטעגאָריעס, טאַגס, אייגענע טאַקסאָנאָמיעס)
* געביטן: איבערגערופֿן פֿון "Elementor Template Sync" צו "Content Sync"
* געביטן: "קלײַב אויס מוסטערן" שריט איבערגערופֿן צו "קלײַב אויס דאַטן"
* פֿאַרבעסערט: Elementor־ספּעציפֿישער קאָד לויפֿט איצט נאָר פֿאַר elementor_library פּאָסט־טיפּ

### ווערסיע 1.0.0 - אַרויסגעלאָזט אויף 2025-12-19 {#version-100---released-on-2025-12-19}
* ערשטע אַרויסלאָזונג
* מער־שריטיקער וועגווײַזער פֿאַר מוסטער־און פּלאַץ־אויסקלײַב
* פּעקל־פּראָצעסירונג מיט רעאַל־צײַט פּראָגרעס־Feedback
* פולע מעדיע־סינכראָניזירונג מיט אויטאָמאַטישער ID־איבערמאַפּירונג
* סעטטינגס־אינטעגראַציע מיט Ultimate Multisite
* שטיצע פֿאַר פּלאַץ־אויסשליסונג דורך נעצווערק־סעטטינגס
* Elementor קאַש־רייניקונג נאָך סינכראָניזירונג
* אויפֿהיטן מוסטער־טאַקסאָנאָמיע
