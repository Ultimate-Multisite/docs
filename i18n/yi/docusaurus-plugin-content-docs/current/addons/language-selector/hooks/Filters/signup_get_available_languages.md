# פֿילטער: signup_get_available_languages {#filter-signupgetavailablelanguages}

פֿילטערט די רשימה פֿון פֿאַראַנענע שפּראַכן פֿאַר פֿראָנט־ענד וועבזײַטל־רעגיסטראַציעס.

איבערגעבן אַ ליידיקן array צו דעם hook וועט אויסשליסן דעם אַרויסגעבן פֿון דער באַשטעטיקונג אויפֿן רעגיסטראַציע־פֿאָרעם, און די פעליקייַט־שפּראַך וועט ווערן גענוצט בײַם שאַפֿן דאָס וועבזײַטל. שפּראַכן וואָס זענען נאָך נישט אינסטאַלירט וועלן ווערן אַוועקגענומען.

## פּאַראַמעטערס {#parameters}

| נאָמען | טיפּ | באַשרײַבונג |
|------|------|-------------|
| $available_languages | `array` | פֿאַראַנענע שפּראַכן. |
