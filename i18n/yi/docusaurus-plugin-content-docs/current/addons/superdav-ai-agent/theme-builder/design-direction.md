# פּלאַן־ריכטונג {#design-direction}

דער **פּלאַן־ריכטונג** שריט דערלויבט אײַך איבערצוקוקן, פֿאַרבעסערן, און ענדגילטיק באַשטעטיקן דעם וויזועלן פּלאַן פֿון אײַער טעמע איידער Theme Builder דזשענערירט די גאַנצע טעמע.

## איבערבליק {#overview}

נאָך פֿאַרענדיקן דעם אַנטדעקונג־אינטערוויו, ווײַזט Theme Builder אײַער פּלאַן־ריכטונג מיט **desktop און mobile preview rendering**. דאָס דערלויבט אײַך צו זען פּינקטלעך ווי אײַער פּלאַן וועט אויסזען אויף פֿאַרשידענע מיטלען איידער איר פֿאַרפֿליכטעט זיך צו דער טעמע.

## קאָמפּאָנענטן פֿון דער פּלאַן־ריכטונג {#design-direction-components}

### 1. קאָליר־פּאַלעטע {#1-color-palette}

די קאָליר־סכעמע פֿון אײַער טעמע, אַרײַנגערעכנט:

- **הויפּט־קאָליר** — דער הויפּט־בראַנד־קאָליר געניצט פֿאַר קנעפּלעך, פֿאַרבינדונגען, און אַקצענטן
- **צווייטיקער קאָליר** — אַ דערגאַנציקער קאָליר פֿאַר פֿאַרשיידנקייט
- **אַקצענט־קאָליר** — אַ הויכפּונקט־קאָליר פֿאַר וויכטיקע עלעמענטן
- **נייטראַלע קאָלירן** — גרויע און ווײַסע קאָלירן פֿאַר הינטערגרונטן און טעקסט
- **טעקסט־קאָליר** — הויפּט־טעקסט־קאָליר פֿאַר לייענבאַרקייט

### 2. טיפּאָגראַפֿיע {#2-typography}

שריפֿט־אויסקלייבן פֿאַר:

- **קעפּל־שריפֿט** — געניצט פֿאַר בלאַט־טיטלען און אָפּטיילונג־קעפּלעך
- **גוף־שריפֿט** — געניצט פֿאַר פּאַראַגראַף־טעקסט און אינהאַלט
- **שריפֿט־גרייסן** — רעספּאָנסיווע גרייסן פֿאַר פֿאַרשידענע עקראַן־גרייסן
- **שורה־הייך** — אָפּשטאַנד צווישן שורות פֿאַר לייענבאַרקייט
- **שריפֿט־וואָג** — פֿעטע, רעגולערע, און לײַכטע וואַריאַציעס

### 3. אויסלייג און אָפּשטאַנד {#3-layout-and-spacing}

- **קאָנטיינער־ברייט** — מאַקסימאַלע ברייט פֿאַר אינהאַלט
- **אינערלעכע און דרויסנדיקע אָפּשטאַנדן** — אָפּשטאַנד אַרום עלעמענטן
- **גריד־סיסטעם** — זיילן־אויסלייג פֿאַר רעספּאָנסיוון פּלאַן
- **קאָמפּאָנענט־אָפּשטאַנד** — דיסטאַנץ צווישן UI עלעמענטן

### 4. וויזועלע עלעמענטן {#4-visual-elements}

- **קנעפּל־סטילן** — הויפּט־, צווייטיקע, און דריטיקע קנעפּל־פּלענער
- **קאַרט־פּלענער** — אויסלייגן פֿאַר אינהאַלט־בלאָקן
- **אייקאָנען** — אייקאָן־סטיל און גרייס
- **ראַמען און שאָטנס** — סובטילע וויזועלע טיפֿקייט
- **Hover עפֿעקטן** — אינטעראַקטיווע צושטאַנד־סטיליזאַציע

## Preview Rendering {#preview-rendering}

### Desktop Preview {#desktop-preview}

דער desktop preview ווײַזט אײַער פּלאַן אין פולער ברייט (געוויינטלעך 1200px אָדער ברייטער):

- גאַנצע נאַוויגאַציע־מעניו
- פולשטענדיקער אויסלייג מיט אַלע זיילן
- גרויסע טיפּאָגראַפֿיע און אָפּשטאַנד
- אַלע וויזועלע עלעמענטן אין פולער גרייס

### Mobile Preview {#mobile-preview}

דער mobile preview ווײַזט אײַער פּלאַן אין mobile ברייט (געוויינטלעך 375px):

- רעספּאָנסיווע נאַוויגאַציע (האַמבורגער־מעניו)
- איין־זייליקער אויסלייג
- צוגעפּאַסטע טיפּאָגראַפֿיע און אָפּשטאַנד
- קנעפּל־גרייסן באַקוועם פֿאַר אָנרירן

### רעספּאָנסיווע ברעיקפּוינטן {#responsive-breakpoints}

Theme Builder רענדערט previews ביי די דאָזיקע ברעיקפּוינטן:

| מיטל | ברייט | Preview |
|--------|-------|---------|
| **Mobile** | 375px | פּאָרטרעט־טעלעפֿאָן |
| **טאַבלעט** | 768px | לאַנדשאַפֿט־טאַבלעט |
| **Desktop** | 1200px | פול־ברייט desktop |
| **גרויסער Desktop** | 1920px | אַלטראַ־ברייטע ווײַזונג |

## פֿאַרבעסערן אײַער פּלאַן {#refining-your-design}

### מאַכן צופּאַסונגען {#making-adjustments}

איר קענט פֿאַרבעסערן אײַער פּלאַן־ריכטונג דורך:

1. **צופּאַסן קאָלירן** — טוישן יעדן קאָליר אין דער פּאַלעטע
2. **טוישן שריפֿטן** — אויסקלייבן אַנדערע שריפֿט־סאָרטן
3. **מאָדיפֿיצירן אָפּשטאַנד** — צופּאַסן אינערלעכע און דרויסנדיקע אָפּשטאַנדן
4. **דערהײַנטיקן אויסלייג** — טוישן קאָנטיינער־ברייטן און גריד־זיילן
5. **צופּאַסן עלעמענטן** — מאָדיפֿיצירן קנעפּל־סטילן, קאַרט־פּלענער, אאַז״ו.

### Preview דערהײַנטיקונגען {#preview-updates}

ענדערונגען ווערן אָפּגעשפּיגלט אין עכטער צײַט:

- Desktop preview דערהײַנטיקט זיך גלײַך
- Mobile preview דערהײַנטיקט זיך גלײַך
- אַלע רעספּאָנסיווע ברעיקפּוינטן ווערן דערהײַנטיקט
- איר קענט איבערבײַטן צווישן previews כּדי צו באַשטעטיקן ענדערונגען

### פֿאַרגלײַך־אויסזען {#comparison-view}

פֿאַרגלײַכט אײַער פּלאַן־ריכטונג מיט:

- **אָריגינעלן פּלאַן** — זען וואָס האָט זיך געביטן
- **קאָנקורענטן־פּלענער** — פֿאַרגלײַכן מיט אינספּיראַציע־זײַטן
- **פֿריִערדיקע ווערסיעס** — זיך אומקערן צו פֿריִערדיקע פּלאַן־ריכטונגען

## באַשטעטיקונג פֿון דער פּלאַן־ריכטונג {#design-direction-approval}

ווען איר זענט צופֿרידן מיט אײַער פּלאַן:

1. **איבערקוקט ביידע previews** — באַשטעטיקט אַז desktop און mobile זעען גוט אויס
2. **קאָנטראָלירט אַלע קאָלירן** — זיכערט קאָנטראַסט און צוטריטלעכקייט
3. **פּרוּווט טיפּאָגראַפֿיע** — באַשטעטיקט לייענבאַרקייט אין אַלע גרייסן
4. **באַשטעטיקט אויסלייג** — קאָנטראָלירט אָפּשטאַנד און אויסגלײַכונג
5. **באַשטעטיקט פּלאַן** — גייט ווײַטער צו טעמע־דזשענעראַציע

## קומענדיקע שריט {#next-steps}

נאָך באַשטעטיקן אײַער פּלאַן־ריכטונג:

1. Theme Builder דזשענערירט אײַער גאַנצע טעמע
2. די טעמע ווערט אינסטאַלירט אויף אײַער WordPress זײַט
3. איר קענט ווײַטער צופּאַסן ניצנדיק WordPress customizer
4. גייט ווײַטער צו [האָספּיטאַליטי־מעניוס](./hospitality-menus.md) אָדער אַנדערע פֿעיִקייטן

## בעסטע פּראַקטיקן {#best-practices}

- **פּרוּווט אויף אמתע מיטלען** — ניצט אמתע טעלעפֿאָנען און טאַבלעטן אויב מעגלעך
- **קאָנטראָלירט לייענבאַרקייט** — זיכערט אַז טעקסט איז לייענבאַר אין אַלע גרייסן
- **באַשטעטיקט קאָנטראַסט** — ניצט [וואַלידירן פּאַלעטע־קאָנטראַסט](../abilities/validate-palette-contrast.md) פֿאַר צוטריטלעכקייט
- **באַטראַכט פאָרשטעלונג** — אָפּטימיזירט בילדער און שריפֿטן פֿאַר גיכקייט
- **פּלאַנירט פֿאַר אינהאַלט** — זיכערט אַז דער אויסלייג אַרבעט מיט אײַער אמתן אינהאַלט

## טראָבלשוטינג {#troubleshooting}

### Preview דערהײַנטיקט זיך נישט {#preview-not-updating}

- דערפֿרישט דעם בלאַט
- רייניקט דעם בלעטערער־קאַש
- פּרוּווט אַן אַנדער בלעטערער
- קאָנטראָלירט אינטערנעט־פֿאַרבינדונג

### קאָלירן זעען אויס אַנדערש {#colors-look-different}

- קאָנטראָלירט מאָניטאָר־קאָליר־אײַנשטעלונגען
- פּרוּווט אויף אַנדערע מיטלען
- באַשטעטיקט קאָליר־קאָנטראַסט מיט צוטריטלעכקייט־געצײַג
- באַטראַכט סימולאַטאָרן פֿאַר קאָליר־בלינדקייט

### טיפּאָגראַפֿיע־פּראָבלעמען {#typography-issues}

- באַשטעטיקט אַז שריפֿט־טעקעס ווערן געלאָדן
- קאָנטראָלירט שריפֿט־גרייס ביי פֿאַרשידענע ברעיקפּוינטן
- פּרוּווט מיט אמתן אינהאַלט
- באַטראַכט שורה־לענג פֿאַר לייענבאַרקייט
