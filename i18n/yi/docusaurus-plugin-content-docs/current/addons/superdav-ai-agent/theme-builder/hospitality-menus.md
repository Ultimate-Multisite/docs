# גאַסטגעבערײַ מעניוס {#hospitality-menus}

די **גאַסטגעבערײַ מעניוס** פֿעיִקייט דערמעגלעכט Theme Builder צו דזשענערירן און אַרײַנלייגן סטרוקטורירטע עסן־און־געטראַנק מעניו־בלעטער גלײַך אין אײַער WordPress וועבזײַט.

## איבערבליק {#overview}

Theme Builder קען איצט שאַפֿן פּראָפֿעסיאָנעלע, סטרוקטורירטע מעניו־בלעטער פֿאַר גאַסטגעבערײַ־געשעפֿטן אַרײַנגערעכנט רעסטאָראַנען, קאַפֿעען, באַרן, און קעיטערינג־דינסטן. די מעניוס זענען גאָר אינטעגרירט אין דער פּלאַן־פֿאָרעם פֿון אײַער וועבזײַט און קענען גרינג דערהײַנטיקט און פֿאַרוואַלטעט ווערן.

## געשטיצטע גאַסטגעבערײַ־טיפּן {#supported-hospitality-types}

- **רעסטאָראַנען** — פֿול־דינסט עסן־מעניוס
- **קאַפֿעען** — קאַווע און לייכטע שפּײַז מעניוס
- **באַרן און לאָונדזשעס** — געטראַנק און פֿאַרבײַסן מעניוס
- **בעקערײַען** — געבעקס און ברויט מעניוס
- **קעיטערינג־דינסטן** — געשעעניש־מעניו אָפּציעס
- **עסן־טראָקס** — באַוועגלעכע עסן־דינסט מעניוס
- **בירברויערײַען און ווײַנערײַען** — געטראַנק־מעניוס מיט באַשרײַבונגען

## מעניו־סטרוקטור {#menu-structure}

### מעניו־קאַטעגאָריעס {#menu-categories}

מעניוס ווערן אָרגאַניזירט אין קאַטעגאָריעס:

- **פֿאַרבײַסן** — אָנהייב־שפּײַזן און קליינע טעלער
- **הויפּט־געריכטן** — הויפּט קורסן
- **בײַלאַגעס** — צובײַסן און גרינסן
- **קינוחים** — זיסע נאַשערײַען
- **געטראַנקען** — געטראַנקען (אַלקאָהאָלישע און נישט־אַלקאָהאָלישע)
- **ספּעציעלעס** — טעגלעכע אָדער סעזאָנאַלע ספּעציעלעס

### מעניו־פּאָזיציע פֿאָרמאַט {#menu-item-format}

יעדע מעניו־פּאָזיציע נעמט אַרײַן:

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### פּאָזיציע־פֿעלדער {#item-fields}

| פֿעלד | טיפּ | באַשרײַבונג |
|-------|------|-------------|
| `name` | string | נאָמען פֿון געריכט אָדער געטראַנק |
| `description` | string | דעטאַלירטע באַשרײַבונג פֿון דער פּאָזיציע |
| `price` | string | פּרײַז (פֿאָרמאַטירט מיט וואַלוטע) |
| `dietary_info` | array | דיעטאַרישע אייגנשאַפֿטן (וועגאַניש, גלוטען־פֿרײַ, אאז״ו) |
| `allergens` | array | געוויינטלעכע אַלערגענן (ניס, שעלפֿיש, אאז״ו) |
| `availability` | string | ווען בנימצא (טעגלעך, סעזאָנאַל, אאז״ו) |

## שאַפֿן גאַסטגעבערײַ מעניוס {#creating-hospitality-menus}

### שריט 1: צושטעלן מעניו־אינפֿאָרמאַציע {#step-1-provide-menu-information}

דערציילט Theme Builder וועגן אײַער מעניו:

```
שאַף אַ רעסטאָראַן־מעניו פֿאַר מײַן איטאַליענישן רעסטאָראַן מיט פֿאַרבײַסן,
פּאַסטאַ־געריכטן, הויפּט־געריכטן, און קינוחים. נעם אַרײַן פּרײַזן און באַשרײַבונגען.
```

### שריט 2: Theme Builder דזשענערירט מעניו {#step-2-theme-builder-generates-menu}

Theme Builder:

1. שאַפֿט אַ סטרוקטורירט מעניו־בלאַט
2. פּלאַנירט עס כּדי עס זאָל פּאַסן צו דער טעמע פֿון אײַער וועבזײַט
3. אָרגאַניזירט פּאָזיציעס אין קאַטעגאָריעס
4. פֿאָרמאַטירט פּרײַזן און באַשרײַבונגען
5. לייגט צו דיעטאַרישע און אַלערגען־אינפֿאָרמאַציע

### שריט 3: איבערקוקן און צופּאַסן {#step-3-review-and-customize}

איר קענט:

1. רעדאַקטירן מעניו־פּאָזיציעס און פּרײַזן
2. צולייגן אָדער אַראָפּנעמען קאַטעגאָריעס
3. איבעראָרדענען פּאָזיציעס אין קאַטעגאָריעס
4. דערהײַנטיקן באַשרײַבונגען און דיעטאַרישע אינפֿאָרמאַציע
5. צופּאַסן סטילינג און אויסלייג

## מעניו־ווײַזונג אָפּציעס {#menu-display-options}

### גאַנץ מעניו־בלאַט {#full-menu-page}

אַ באַזונדערער בלאַט וואָס ווײַזט אײַער גאַנצן מעניו:

- אָרגאַניזירט לויט קאַטעגאָריע
- זוכבאַר און פֿילטערבאַר
- דרוק־פֿרײַנדלעכער אויסלייג
- מאָביל־רעאַגירנדיקער פּלאַן

### מעניו ווידזשעט {#menu-widget}

לייגט אַרײַן מעניו־אָפּטיילונגען אויף אַנדערע בלעטער:

- אויסגעקליבענע פּאָזיציעס אויף דער היים־בלאַט
- טעגלעכע ספּעציעלעס אויף דער זײַט־שפּאַלטע
- געטראַנק־מעניו אויף דער באַר־בלאַט
- קינוח־אויסשטעלונג אין פֿוסבלאַט

### מעניו PDF {#menu-pdf}

דזשענערירט אַ אַראָפּלאָדבאַרן PDF מעניו:

- פּראָפֿעסיאָנעלע פֿאָרמאַטירונג
- דרוק־גרייטע קוואַליטעט
- נעמט אַרײַן בילדער און באַשרײַבונגען
- גרינג צו טיילן און שיקן דורך בליצפּאָסט

## דיעטאַרישע און אַלערגען־אינפֿאָרמאַציע {#dietary-and-allergen-information}

### דיעטאַרישע אייגנשאַפֿטן {#dietary-attributes}

באַצייכנט פּאָזיציעס מיט דיעטאַרישער אינפֿאָרמאַציע:

- **וועגאַניש** — קיין חיות־פּראָדוקטן
- **וועגעטאַריש** — קיין פֿלייש
- **גלוטען־פֿרײַ** — זיכער פֿאַר צעליאַק־קרענק
- **מילכיקס־פֿרײַ** — קיין מילך־פּראָדוקטן
- **ניס־פֿרײַ** — קיין בוים־ניס אָדער ערדניס
- **נידעריק־קאַרבאָהידראַטן** — פֿאַררינגערטע קאַרבאָהידראַטן
- **הויך־פּראָטעיִן** — רײַך אין פּראָטעיִן

### אַלערגען־וואָרענונגען {#allergen-warnings}

נעמט אַרײַן געוויינטלעכע אַלערגענן:

- **ניס** — בוים־ניס און ערדניס
- **שעלפֿיש** — קראַסטאַציעס און מאָלוסקן
- **פֿיש** — אַלע פֿיש־מינים
- **מילכיקס** — מילך און מילך־פּראָדוקטן
- **אייער** — הינער־אייער
- **סוי** — סוי־פּראָדוקטן
- **גלוטען** — ווייץ און גלוטען־האַלטיקע תּבואה
- **סעסאַמי** — סעסאַמי־זוימען און בוימל

## מעניו־פֿאַרוואַלטונג {#menu-management}

### דערהײַנטיקן פּרײַזן {#updating-prices}

גרינג דערהײַנטיקט מעניו־פּרײַזן:

1. נאַוויגירט צו דעם מעניו־בלאַט
2. קליקט "רעדאַקטירן מעניו"
3. דערהײַנטיקט פּרײַזן פֿאַר פּאָזיציעס
4. היט ענדערונגען
5. ענדערונגען דערשײַנען גלײַך אויף אײַער וועבזײַט

### צולייגן סעזאָנאַלע פּאָזיציעס {#adding-seasonal-items}

שאַפֿט סעזאָנאַלע מעניו־וואַריאַציעס:

1. שאַפֿט אַ נײַע מעניו־ווערסיע
2. לייגט צו סעזאָנאַלע פּאָזיציעס
3. באַצייכנט פּאָזיציעס ווי "סעזאָנאַל"
4. פּלאַנירט בנימצא־דאַטעס
5. אויטאָמאַטיש ווײַזן במשך פֿון דער סעזאָן

### פֿאַרוואַלטן ספּעציעלעס {#managing-specials}

ווײַזט טעגלעכע אָדער וועכנטלעכע ספּעציעלעס:

1. שאַפֿט אַ "ספּעציעלעס" קאַטעגאָריע
2. לייגט צו פּאָזיציעס מיט בנימצא־דאַטעס
3. הייבט אַרויס ספּעציעלעס אויף דער היים־בלאַט
4. דערהײַנטיקט טעגלעך אָדער וועכנטלעך
5. אַרכיווירט אַלטע ספּעציעלעס

## אינטעגראַציע מיט Theme Builder {#integration-with-theme-builder}

ווען מען ניצט Theme Builder פֿאַר גאַסטגעבערײַ וועבזײַטן:

1. **אויטאָמאַטישע מעניו־דערקענונג** — אידענטיפֿיצירט צי איר זענט אַן עסן־/געטראַנק־געשעפֿט
2. **מעניו־בלאַט שאַפֿונג** — דזשענערירט פּראָפֿעסיאָנעלע מעניו־בלעטער
3. **פּלאַן־צופּאַסונג** — מעניוס פּאַסן צו דער טעמע פֿון אײַער וועבזײַט
4. **מאָביל־אָפּטימיזאַציע** — מעניוס ווײַזן זיך שיין אויף טעלעפֿאָנען
5. **SEO אָפּטימיזאַציע** — מעניוס זענען פֿרײַנדלעך פֿאַר זוך־מאָטאָרן

## בעסטע פּראַקטיקן {#best-practices}

### מעניו־פּלאַן {#menu-design}

- **קלאָרע אָרגאַניזאַציע** — לאָגישער קאַטעגאָריע־סטרוקטור
- **לייענבאַרע באַשרײַבונגען** — אַפּעטיטלעך און אינפֿאָרמאַטיוו
- **קאָנסיסטענטע פּרײַזן** — קלאָרע וואַלוטע און פֿאָרמאַטירונג
- **פּראָפֿעסיאָנעלע פֿאָטאָס** — הויך־קוואַליטעט עסן־בילדער
- **ליידיקער רוים** — נישט איבערפֿולן דעם בלאַט

### אינהאַלט {#content}

- **פּינקטלעכע באַשרײַבונגען** — באַשרײַבט פּאָזיציעס פּינקטלעך
- **הייבט אַרויס ספּעציעלעס** — מאַכט ספּעציעלעס זיך אָפּצייכענען
- **נעמט אַרײַן אַלערגענן** — ליסטעט שטענדיק אַלערגענן
- **דערהײַנטיקט רעגולער** — האַלט פּרײַזן און פּאָזיציעס איצטיק
- **ניצט אַפּעטיטלעכע שפּראַך** — מאַכט פּאָזיציעס קלינגען געשמאַק

### צוטריטלעכקייט {#accessibility}

- **לייענבאַרע שריפֿטן** — ניצט קלאָרע, לייענלעכע שריפֿט־מינים
- **גענוג קאָנטראַסט** — זיכערט אַז טעקסט איז לייענבאַר
- **דיעטאַרישע לייבלעך** — באַצייכנט קלאָר דיעטאַרישע אָפּציעס
- **אַלערגען־וואָרענונגען** — ווײַזט אַלערגענן באַמערקבאַר
- **מאָביל־פֿרײַנדלעך** — טעסט אויף אַלע מיטלען
