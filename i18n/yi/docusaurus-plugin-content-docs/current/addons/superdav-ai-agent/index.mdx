# Superdav AI Agent

Superdav AI Agent איז אַן אַוואַנסירטער WordPress AI אַסיסטענט וואָס פֿאַרברייטערט Gratis AI Agent מיט theme־באַוווּסטע פֿעיִקייטן, פֿאַרבעסערטע אינטערנעט־זוך מעגלעכקייטן, site־בויער פֿעיִקייטן, און אָפּציאָנעלע אויטאָמאַציע־אינטעגראַציעס. ווערסיע 1.18.0 לייגט צו Google Calendar לייענען־מכשירים, קאָנטאַקט־ און באַטייליקטע־מאַפּינג, מענטשלעכע באַשטעטיקונג־טויערן, דערמאָנונג־דעדופּליקאַציע, TextBee SMS מעלדונגען, פֿאַרוואַלטעטע Superdav דינסט־צושטעלונג, און אַ באַזונדער פֿאַרשפּרייטן Advanced באַגלייטער־פּעקל פֿאַר פֿאַרטרויטע אַנטוויקלער־אַרבעטספֿלוסן.

## הויפּט־פֿעיִקייטן {#key-features}

- **Theme־באַוווּסטע פֿעיִקייטן** — פּאַסט אויטאָמאַטיש צו אנווייזונגען און מעגלעכקייטן לויט דיין אַקטיוון WordPress theme און אינסטאַלירטע plugins
- **קייפֿל אינטערנעט־זוך צושטעלער** — קלייב צווישן Brave Search און Tavily פֿאַר רײַכערע זוך־רעזולטאַטן און אַוואַנסירטע פֿאָרשונג־מעגלעכקייטן
- **Site־בויער קאָנטאַקט־פֿאָרמען** — שאַף גאָר פֿונקציאָנעלע קאָנטאַקט־פֿאָרמען גלײַך פֿון דער שמועס־צובינד ניצנדיק דיין אינסטאַלירטן פֿאָרעם־plugin
- **פֿאַרבעסערטער site־בויער** — פֿאַרבעסערטע מולטי־פֿאַזע site־בוי־פּלענער מיט בעסערער plugin־אַנטדעקונג און פֿעלער־אויפֿהיילונג
- **אײַנגעבויטע theme־פֿעיִקייטן** — ספּעציאַליזירטע אנווייזונג פֿאַר Block Themes, Classic Themes, Kadence Blocks, און Kadence Theme
- **פֿאַרבעסערטע WooCommerce אינטעגראַציע** — איבערגעאַרבעט צו ניצן געבוירענע WooCommerce APIs פֿאַר בעסערער פֿאַרלאָזלעכקייט און קאָמפּאַטיביליטעט
- **אויטאָמאַטישע צושטעלער־דערקענונג** — דערקענט און דערפֿרישט אויטאָמאַטיש בנימצא plugins און themes ווען זיי ווערן אַקטיווירט אָדער דעאַקטיווירט
- **אייגענע לאָגאָ־שאַפֿונג** — גענעריר און לייג אַרײַן אייגענע SVG לאָגאָס מיט אויטאָמאַטישער namespace־זיכערער רייניקונג
- **פּאַלעט־קאָנטראַסט וואַלידאַציע** — קאָנטראָליר קאָליר־פּאָרן פֿאַר WCAG צוטריטלעכקייט־אײַנהאַלטונג איידער אָנווענדן אויף דיין theme
- **פֿאָטאָ־פֿאַרבעסערטע אַנטדעקונג** — לאָד אַרויף פֿאָטאָס בעת דעם Theme Builder אַנטדעקונג־אינטערוויו כּדי צו באַאײַנפֿלוסן די פּלאַן־ריכטונג
- **גאַסטגעבערײַ־מעניוס** — גענעריר סטרוקטורירטע עסן־ און געטראַנקען־מעניו בלעטער פֿאַר רעסטאָראַנען, קאַפֿעען, און גאַסטגעבערײַ־געשעפֿטן
- **דעסקטאָפּ־ און מאָביל־פֿאָרויסקוקן** — קוק פֿאָרויס דיין פּלאַן אויף דעסקטאָפּ און מאָביל־מכשירים בעת דער אויסקלײַב פֿון פּלאַן־ריכטונג
- **ראַנג 1 site־סקרייפּינג** — sd-ai-agent/site-scrape איז איצט בנימצא לויט פעליקייט פֿאַר אַנאַליזירן קאָנקורענט־ און אינספּיראַציע־וועבזײַטן
- **Google Calendar לייענען־מכשירים** — לייען קאָנפֿיגורירטע קאַלענדאַרן און געשעענישן אַזוי אַז אויטאָמאַציעס קענען באַטראַכטן קומענדיקן פּלאַן־קאָנטעקסט
- **קאָנטאַקט־ און באַטייליקטע־מאַפּינג** — פּאָר צו קאַלענדאַר־באַטייליקטע מיט באַקאַנטע site־ניצער און קאָנטאַקטן איידער מע גענערירט נאָכפֿאָלג־אַרבעט
- **מענטשלעכע באַשטעטיקונג־טויערן** — שטעל אָפּ סענסיטיווע אויטאָמאַציעס פֿאַר איבערבליק איידער שיקן אַרטיקלען, ווידעראָנהייבן אַרבעט, אָדער מאַכן ענדערונגען
- **דערמאָנונג־רעקאָרדן און דעדופּליקאַציע** — שפּור דערמאָנונגען וואָס זענען שוין געשיקט געוואָרן אַזוי אַז איבערגעחזרטע אויטאָמאַציע־לויפֿן זאָלן נישט מעלדן דעם זעלבן באַקומען צוויי מאָל
- **TextBee SMS מעלדונגען** — שיק קאָנפֿיגורירטע טעקסט־אָנזאָג מעלדונגען דורך TextBee ווען אַ SMS צושטעלער איז אַקטיווירט
- **פֿאַרוואַלטעטע Superdav דינסט־צושטעלונג** — פֿאַרבינד געשטיצטע sites צו האָסטירטע Superdav endpoints מיט אויטאָמאַטישער אײַנשטעלונג אַנשטאָט מאַנועלער endpoint קאָנפֿיגוראַציע
- **Advanced באַגלייטער־פּעקל** — באַזונדער פֿאַרשפּרייטע פֿאַרטרויטע־אַנטוויקלער מכשירים פֿאַר טעקע־סיסטעם, דאַטאַבאַזע, WP-CLI, REST dispatcher, plugin־בויער, git snapshot, ניצער־פֿאַרוואַלטונג, און benchmark אַרבעטספֿלוסן

## פֿאָדערונגען {#requirements}

- WordPress 6.0 אָדער העכער
- PHP 8.1 אָדער העכער
- WordPress Multisite (נעץ־אַקטיווירט) אָדער איינצל־site אינסטאַלאַציע
- אַן אַקטיווער AI צושטעלער־connector plugin (Anthropic, OpenAI, אָדער קאָמפּאַטיבל)
- Gratis AI Agent v1.9.0 אָדער העכער (Superdav AI Agent פֿאַרברייטערט Gratis AI Agent)

## אינסטאַלאַציע {#installation}

1. לאָד אַרויף די plugin טעקעס צו `/wp-content/plugins/superdav-ai-agent`
2. אַקטיוויר דעם plugin דורך דעם **Plugins** עקראַן אין WordPress
3. מאַך זיכער אַז Gratis AI Agent איז אויך אַקטיווירט
4. גיי צו **Gratis AI Agent → Settings** און קאָנפֿיגוריר דיין אינטערנעט־זוך צושטעלער, פֿאַרוואַלטעטע Superdav דינסט־פֿאַרבינדונג, Google Calendar, אָדער TextBee צוטריט־דאַטן לויט נויט

דער WordPress.org פּעקל אַנטהאַלט די קערן־פֿעיִקייטן פֿון Superdav AI Agent. דער Advanced באַגלייטער־פּעקל ווערט געבויט און פֿאַרשפּרייט באַזונדער פֿאַר סביבות וווּ פֿאַרטרויטע אַנטוויקלער־מכשירים ווערן בפֿירוש אינסטאַלירט און דערלויבט.

## שנעלער אָנהייב {#quick-start}

אַמאָל אַקטיווירט, פֿאַרבעסערט Superdav AI Agent אויטאָמאַטיש דיין Gratis AI Agent אינסטאַלאַציע מיט theme־באַוווּסטע פֿעיִקייטן און נאָך מעגלעכקייטן. עפֿן דעם **Gratis AI Agent** פּאַנעל אין דער WordPress אַדמין־זײַטן־לײַסטע און הייב אָן ניצן די נײַע פֿעיִקייטן.

**בײַשפּיל־אנווייזונגען:**

```
לייג צו אַ קאָנטאַקט־פֿאָרעם צו דער קאָנטאַקט־בלאַט.
```

```
זוך אין וועב נאָך די לעצטע בעסטע WordPress זיכערהייט־פּראַקטיקן.
```

```
שאַף אַ hero אָפּטיילונג ניצנדיק די block patterns פֿון דעם אַקטיוון theme.
```

```
איבערקוק די קאָנפֿיגורירטע Google Calendar געשעענישן פֿון מאָרגן און שרײַב אָן דערמאָנונגען פֿאַר באַטייליקטע.
```

```
שיק אַן SMS דערמאָנונג נאָך דעם וואָס איך באַשטעטיק דעם אָנזאָג.
```

## Theme־באַוווּסטע פֿעיִקייטן {#theme-aware-skills}

Superdav AI Agent דערקענט אויטאָמאַטיש דיין אַקטיוון theme און אינסטאַלירטע plugins, און לאָדט די פּאַסיקע theme־באַוווּסטע פֿעיִקייט אין דעם קאָנטעקסט פֿונעם אַסיסטענט. בנימצא פֿעיִקייטן אַרײַננעמען:

| פֿעיִקייט | פּאַסט צו |
|---|---|
| **Block Themes** | Themes וואָס ניצן די WordPress block־באַזירטע (Full Site Editing) אַרכיטעקטור |
| **Classic Themes** | טראַדיציאָנעלע WordPress themes וואָס ניצן PHP templates און דעם קלאַסישן רעדאַקטאָר |
| **Kadence Blocks** | Sites מיטן Kadence Blocks plugin אַקטיוו |
| **Kadence Theme** | Sites מיטן Kadence theme ווי דער אַקטיווער theme |

זע [Theme־באַוווּסטע פֿעיִקייטן](../../user-guide/configuration/theme-aware-skills) פֿאַר פּרטימדיקע אנווייזונג וועגן יעדער פֿעיִקייט.

## אינטערנעט־זוך צושטעלער {#internet-search-providers}

Superdav AI Agent v1.10.0 לייגט צו **Tavily** ווי אַן אַלטערנאַטיוו צו Brave Search פֿאַר אינטערנעט־זוך פֿראַגעס. ביידע צושטעלער פאָרשלאָגן רעאַל־צײַט זוך־רעזולטאַטן און קענען ווערן קאָנפֿיגורירט אין **Gratis AI Agent → Settings → Advanced**.

| צושטעלער | פֿעיִקייטן | אײַנשטעלונג |
|---|---|---|
| **Brave Search** | שנעלע, פּריוואַטקייט־פֿאָקוסירטע זוך־רעזולטאַטן | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | רײַכערע זוך־רעזולטאַטן, אַוואַנסירטע פֿאָרשונג־מעגלעכקייטן | [Tavily API](https://tavily.com/) |

זע [אינטערנעט־זוך](../../user-guide/configuration/internet-search) פֿאַר קאָנפֿיגוראַציע־אנווייזונגען.

## קאַלענדאַר, באַשטעטיקונג, און מעלדונג־אַרבעטספֿלוסן {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 פֿאַרברייטערט אויטאָמאַציע־אַרבעטספֿלוסן ווײַטער פֿון אינהאַלט־ און site־בוי־אויפֿגאַבעס:

- **Google Calendar מכשירים** לאָזן דעם agent לייענען קאָנפֿיגורירטע קאַלענדאַרן און געשעעניש־פּרטים ווען קאַלענדאַר־צוטריט־דאַטן זענען בנימצא.
- **קאָנטאַקט־ און באַטייליקטע־העלפֿערס** מאַפּן געשעעניש־באַטייליקטע צו WordPress ניצער אָדער באַקאַנטע קאָנטאַקטן איידער צוגרייטן דערמאָנונגען אָדער נאָכפֿאָלג־אויפֿגאַבעס.
- **מענטשלעכע באַשטעטיקונג־טויערן** לאָזן אַן אויטאָמאַציע זיך אָפּשטעלן ביז אַן אויטאָריזירטער ניצער איבערקוקט די פֿאָרגעלייגטע אַקציע.
- **דערמאָנונג־רעקאָרדן** פֿאַרמײַדן טאָפּל קאַלענדאַר־ אָדער מעלדונג־דערמאָנונגען ווען אַן אויטאָמאַציע ווערט ווידערפּרוּווט אָדער געפּלאַנט איבערגעחזרט.
- **TextBee SMS שטיצע** דערמעגלעכט טעקסט־אָנזאָג מעלדונגען ווען דער TextBee צושטעלער איז קאָנפֿיגורירט.

ניץ די מכשירים פֿאַר פּלאַן־באַוווּסטע הילף, באַטייליקטע־נאָכפֿאָלג, און דערמאָנונג־אַרבעטספֿלוסן וואָס פֿאָדערן נאָך אַלץ מענטשלעכן איבערבליק איידער סענסיטיווע אַרטיקלען ווערן געשיקט.

## Site־בויער קאָנטאַקט־פֿאָרמען {#site-builder-contact-forms}

דער Site Builder agent קען איצט שאַפֿן קאָנטאַקט־פֿאָרמען גלײַך פֿון דער שמועס־צובינד. פֿרעג פּשוט דעם Site Builder צו שאַפֿן אַ קאָנטאַקט־פֿאָרעם, און ער וועט אויטאָמאַטיש דערקענען דיין אינסטאַלירטן פֿאָרעם־plugin (Contact Form 7, WPForms, Fluent Forms, אָדער Gravity Forms) און גענערירן אַ גרייט־צו־נוצן פֿאָרעם.

זע [Site־בויער קאָנטאַקט־פֿאָרמען](../../user-guide/administration/site-builder-contact-forms) פֿאַר פּרטימדיקע אנווייזונג.

## Theme Builder פֿעיִקייטן {#theme-builder-features}

Theme Builder אין Superdav AI Agent v1.16.0 נעמט אַרײַן מאַכטיקע נײַע מעגלעכקייטן:

### אַנטדעקונג־אינטערוויו מיט פֿאָטאָ־אַרויפֿלאָד {#discovery-interview-with-photo-upload}

דער Theme Builder אַנטדעקונג־אינטערוויו נעמט איצט אַרײַן אַ פֿאָטאָ־אַרויפֿלאָד שריט וווּ דו קענסט צושטעלן וויזועלע אינספּיראַציע. לאָד אַרויף פֿאָטאָס פֿון דיין מאַרק, קאָנקורענט־sites, אָדער פּלאַן־אינספּיראַציע כּדי צו העלפֿן Theme Builder פֿאַרשטיין דיין פּלאַן־ריכטונג.

זע [אַנטדעקונג־אינטערוויו](./theme-builder/discovery-interview.md) פֿאַר פּרטימדיקע אנווייזונג.

### פּלאַן־ריכטונג מיט דעסקטאָפּ־ און מאָביל־פֿאָרויסקוקן {#design-direction-with-desktop-and-mobile-previews}

איבערקוק דיין פּלאַן אויף ביידע דעסקטאָפּ און מאָביל־מכשירים איידער Theme Builder גענערירט דיין גאַנצן theme. זע רעאַל־צײַט פֿאָרויסקוקן אויף קייפֿל רעספּאָנסיווע ברעיקפּוינטן.

זע [פּלאַן־ריכטונג](./theme-builder/design-direction.md) פֿאַר פּרטימדיקע אנווייזונג.

### גאַסטגעבערײַ־מעניוס {#hospitality-menus}

Theme Builder קען איצט גענערירן סטרוקטורירטע עסן־ און געטראַנקען־מעניו בלעטער פֿאַר רעסטאָראַנען, קאַפֿעען, באַרן, און אַנדערע גאַסטגעבערײַ־געשעפֿטן. מעניוס נעמען אַרײַן פּרײַזן, באַשרײַבונגען, דיעטע־אינפֿאָרמאַציע, און אַלערגען־וואָרענונגען.

זע [גאַסטגעבערײַ־מעניוס](./theme-builder/hospitality-menus.md) פֿאַר פּרטימדיקע אנווייזונג.

## קאָנפֿיגוראַציע {#configuration}

אַלע אײַנשטעלונגען זענען אונטער **Gratis AI Agent → Settings** אין WordPress אַדמין. Superdav AI Agent לייגט צו די ווײַטערדיקע קאָנפֿיגוראַציע־אָפּציעס:

| אײַנשטעלונג | באַשרײַבונג |
|---|---|
| **אינטערנעט־זוך צושטעלער** | קלייב צווישן Brave Search און Tavily (אָפּציאָנעל) |
| **Tavily API Key** | API שליסל פֿאַר Tavily אינטערנעט־זוך (אויב Tavily איז אויסגעקליבן) |
| **פֿאַרוואַלטעטע Superdav דינסט** | פֿאַרבינד געשטיצטע sites צו דער האָסטירטער Superdav דינסט מיט אויטאָמאַטישער endpoint צושטעלונג |
| **Google Calendar צוטריט־דאַטן** | קאָנפֿיגוריר קאַלענדאַר־צוטריט אַזוי אַז דער agent קען לייענען קאַלענדאַרן און געשעענישן |
| **TextBee SMS צושטעלער** | קאָנפֿיגוריר TextBee צוטריט־דאַטן פֿאַר באַשטעטיקטע SMS מעלדונג־אַרבעטספֿלוסן |

## Advanced באַגלייטער־פּעקל {#advanced-companion-package}

Superdav AI Agent v1.18.0 לייגט צו אַ באַזונדער פֿאַרשפּרייטן Advanced באַגלייטער־פּעקל פֿאַר פֿאַרטרויטע אַנטוויקלער־אַרבעטספֿלוסן. די מעגלעכקייטן זענען נישט אַרײַנגעבונדן אין דעם WordPress.org קערן־פּעקל. אינסטאַליר דעם באַגלייטער־פּעקל בלויז אויף sites וווּ אַדמיניסטראַטאָרן פֿאַרטרויען בפֿירוש דעם agent צו צוטריטן אַנטוויקלער־מכשירים.

Advanced מכשירים נעמען אַרײַן טעקע־סיסטעם אָפּעראַציעס, דאַטאַבאַזע־צוטריט, WP-CLI, REST dispatcher אַרבעטספֿלוסן, plugin־בויער נוצמכשירים, git snapshots, ניצער־פֿאַרוואַלטונג העלפֿערס, און benchmark מכשירים. האַלט די מכשירים באַגרענעצט צו פֿאַרטרויטע אַדמיניסטראַטאָרן און איבערקוק באַשטעטיקונג־טויער קאָנפֿיגוראַציע איידער זיי אַקטיווירן אויף פּראָדוקציע־sites.
