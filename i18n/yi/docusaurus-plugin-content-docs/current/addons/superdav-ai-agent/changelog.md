# ענדערונג־לאָג

## 1.18.0 — אַרויסגעלאָזט דעם 2026-06-29 {#1180--released-on-2026-06-29}

### נײַ {#new}

- **Google Calendar מכשירים** — לייענען קאָנפֿיגורירטע קאַלענדאַרן און געשעענישן פֿאַר פּלאַן־באַוווּסטע אויטאָמאַציעס
- **קאָנטאַקט־מאַפּינג און באַטייליקטע־העלפֿער** — צופּאַסן קאַלענדאַר־באַטייליקטע צו פּלאַץ־באַניצער און קאָנטאַקטן
- **מענטשלעכע באַשטעטיקונג־טויערן און דערמאָנונג־רעקאָרדס** — אָפּשטעלן אויטאָמאַציעס פֿאַר איבערבליק און אויסמײַדן טאָפּלטע מעלדונגען
- **TextBee SMS צושטעלער** — שיקן קאָנפֿיגורירטע טעקסט־מעלדונגען דורך TextBee
- **פֿאָרגעשריטענער באַגלייטער־פּעקל** — צולייגן געטרויען־אַנטוויקלער טעקע־סיסטעם, דאַטנבאַנק, WP-CLI, REST dispatcher, plugin builder, git snapshot, באַניצער־פֿאַרוואַלטונג, און benchmark מכשירים וואָס ווערן באַזונדער פֿאַרשפּרייט פֿון דער WordPress.org ווערסיע

### פֿאַרבעסערט {#improved}

- **פֿאַרוואַלטעטע Superdav דינסט־אײַנשטעלונג** — צולייגן געבאַלעבאָסטע דינסט־ענדפּונקטן און אויטאָמאַטישע פֿאַרבינדונג־צושטעלונג פֿאַר געשטיצטע פּלאַץן
- **אַרויסגאַבע־פּאַקירונג** — בויען באַזונדערע קערן און Advanced ZIPs, פֿאַרעפֿנטלעכן ביידע אויף GitHub, און שיקן בלויז דעם קערן־פּעקל צו WordPress.org

### פֿאַרראָכטן {#fixed}

- **AI בקשה־פֿאַרלעסלעכקייט** — פֿאַרבעסערן מאָדעל־אויסקלייב, בקשה־צײַטגרענעצן, סטאַנדאַרט אײַנשטעלונג־מאָדעל, באַהאַנדלונג פֿון reasoning־טעקסט, און ווידער־פּרוּוו־אנווײַזונגען פֿאַר אומגילטיקע געצײַג־רופֿן
- **קאַלענדאַר און דערמאָנונג־פֿעסטיקונג** — פֿאַרשטאַרקן Google Calendar tokens און דערמאָנונג־דעדופּליקאַציע
- **אָנבאָרדינג און באַשטעטיקונג־ווײַטערפֿירונגען** — פֿאַרריכטן frontend אָנבאָרדינג־לאָנטשן און באַשטעטיקטע פֿעיִקייט־ווײַטערפֿירונגען
- **WordPress.org פּאַקירונג־איבערבליק־ענינים** — באַהאַנדלען פּאַקירונג־איבערבליק־באַמערקונגען פֿאַר דער קערן־אַרויסגאַבע

## 1.16.0 — אַרויסגעלאָזט דעם 2026-05-20 {#1160--released-on-2026-05-20}

### נײַ {#new-1}

- **גענערירן Logo SVG פֿעיִקייט** — Theme Builder קען איצט גענערירן און אײַנבעטן אייגענע לאָגאָ־SVGs מיט נאָמענטייל־זיכערער רייניקונג
- **פֿאָטאָ־אַרויפֿלאָדן אין דער אַנטדעקונג־אינטערוויו** — Theme Builder אַנטדעקונג־אינטערוויו נעמט איצט אַרײַן אַ פֿאָטאָ־אַרויפֿלאָדן־שריט פֿאַר אַ רײַכערן פּלאַן־קאָנטעקסט
- **וואַלידירן פּאַלעט־קאָנטראַסט פֿעיִקייט** — קאָנטראָלירן קאָליר־פּאָרן פֿאַר WCAG צופּאַסיקייט איידער מע לייגט זיי אָן אויף אַ טעמע
- **גאַסטגעבערײַ־מעניוז** — Theme Builder קען איצט גענערירן סטרוקטורירטע עסן־און־געטראַנק מעניו־בלעטער פֿאַר גאַסטגעבערײַ־געשעפֿטן
- **דעסקטאָפּ און מאָבילע פֿאָרויסקוק־רענדערינג** — קוקן פֿאָרויס אויף אײַער פּלאַן אויף דעסקטאָפּ און מאָבילע מיטלען בעת דער אויסקלײַב פֿון פּלאַן־ריכטונג
- **נאַוויגאַציע־עטיקעט פּאַראַמעטער** — Create Menu פֿעיִקייט שטיצט איצט אַן אייגענעם `navigation_label`, באַזונדער פֿון דער בלאַט־קעפּל
- **Tier 1 געצײַג־פֿאַראַנקייט** — sd-ai-agent/site-scrape איז איצט אַ Tier 1 געצײַג, בנימצא לויטן סטאַנדאַרט אין Theme Builder

### פֿאַרראָכטן {#fixed-1}

- **AI Client cache** — איצט געשטיצט דורך transients פֿאַר פּערסיסטענץ צווישן בקשות, וואָס פֿאַרמײַדט דאַטן־אָנווער בײַ לאַנג־לויפֿנדיקע agent אויפֿגאַבן
- **Plugin ריי־אַקציע לינקען** — פֿאַרריכט און איבערגערופֿן פֿאַר קלאָרקייט

## 1.10.0 — אַרויסגעלאָזט דעם 2026-05-05 {#1100--released-on-2026-05-05}

### נײַ {#new-2}

- **Tavily אינטערנעץ־זוך** — צולייגן Tavily ווי אַ זוך־צושטעלער פֿאַר רײַכערע אינטערנעץ־זוך־רעזולטאַטן צוזאַמען מיט Brave Search
- **טעמע־באַוווּסטע אײַנגעבויטע פֿעיִקייטן** — Block Themes, Classic Themes, Kadence Blocks, און Kadence Theme פֿעיִקייט־וועגווייזערס קומען איצט מיטן plugin
- **פּלאַץ־בויער קאָנטאַקט־פֿאָרעם פֿעיִקייט** — צולייגן אַ קאָנטאַקט־פֿאָרעם צו יעדן בלאַט גלײַך פֿון דער שמועס־צובינד

### פֿאַרבעסערט {#improved-1}

- **WooCommerce אינטעגראַציע איבערגעאַרבעט** — ניצט איצט נאַטיווע WooCommerce APIs פֿאַר בעסערע פֿאַרלעסלעכקייט און קאַמפּאַטיביליטעט
- **צושטעלער־ליסטע דערפֿרישט זיך אויטאָמאַטיש** — ווען ס׳ווערט אַקטיווירט אָדער דעאַקטיווירט וועלכער plugin עס זאָל זײַן

### פֿאַרראָכטן {#fixed-2}

- **navigate-to פֿעיִקייט** — פֿאַרריכט אַן אָן־סוף ווידער־לאָדן שלייף אויף עטלעכע אַדמין־בלעטער
- **list-posts פֿעיִקייט** — לייזט איצט ריכטיק קאַטעגאָריע־ און צעטל־נעמען צו slugs
- **WP-CLI באַפֿעלן** — צוריקגעשטעלט פֿעלנדיקע namespace אַליאַסן נאָך אַ פֿריִערדיקער איבעראַרבעטונג
- **געשעעניש־אויטאָמאַציע** — באַהאַנדלט גלאַט פּלאַץן וווּ אויטאָמאַציע־טאַבעלעס זענען נאָך נישט באַשאַפֿן
- **memory-save פֿעיִקייט** — ניצט איצט דעם ריכטיקן namespace prefix אין דעם סיסטעם־אנווײַזונג־בויער
- **סקאַלאַרע געצײַג־רעזולטאַטן** — איצט ריכטיק אַרומגעוויקלט איידער זיי ווערן צוריקגעגעבן צו דער AI
- **ניץ־סטאַטיסטיק** — באַהאַנדלט איצט ריכטיק דעם אַלטן פֿעיִקייט־שליסל־פֿאָרמאַט בײַ אַ דערהײַנטיקונג פֿון עלטערע ווערסיעס
