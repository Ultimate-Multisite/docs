# שאַפֿן מעניו

די **שאַפֿן מעניו** פֿעיִקייט שאַפֿט WordPress נאַוויגאַציע־מעניוס מיט שטיצע פֿאַר באַזונדערע נאַוויגאַציע־לייבלס באַזונדער פֿון זײַט־טיטלען.

## איבערבליק

די פֿעיִקייט פֿאַרברייטערט די סטאַנדאַרטע מעניו־שאַפֿן פֿונקציאָנאַליטעט מיט דער מעגלעכקייט אָנצוגעבן אַ `navigation_label` פּאַראַמעטער. דאָס דערלויבט אײַך צו שאַפֿן מעניוס וווּ דער לייבל וואָס ווערט געוויזן אין נאַוויגאַציע איז אַנדערש פֿון דעם זײַט־טיטל, און גיט מער בייגיקייט אין וועבזײַט־סטרוקטור און באַניצער־דערפֿאַרונג.

## פּאַראַמעטערס

| פּאַראַמעטער | טיפּ | נייטיק | באַשרײַבונג |
|-----------|------|----------|-------------|
| `name` | string | יאָ | מעניו־נאָמען, למשל `Primary Navigation` |
| `location` | string | ניין | Theme־אָרט צו צוטיילן דעם מעניו צו, למשל `primary` |
| `navigation_label` | string | ניין | לייבל צו ווײַזן אין נאַוויגאַציע (באַזונדער פֿון זײַט־טיטל) |

## צוריקגעגעבענער ווערט

```json
{
  "success": true,
  "menu_id": 7,
  "name": "Primary Navigation",
  "navigation_label": "Main Menu"
}
```

## נאַוויגאַציע־לייבל קעגן זײַט־טיטל

דער `navigation_label` פּאַראַמעטער דערלויבט אײַך אָפּצוטיילן דעם אינערלעכן מעניו־נאָמען פֿון דעם לייבל וואָס ווערט געוויזן פֿאַר באַניצער:

- **`name`** — אינערלעכער מעניו־אידענטיפֿיקאַטאָר געניצט דורך WordPress (למשל, "Primary Navigation")
- **`navigation_label`** — דער לייבל וואָס ווערט געוויזן פֿאַר וועבזײַט־באַזוכער אין דער נאַוויגאַציע (למשל, "Main Menu")

דאָס איז נוצלעך ווען:
- אײַער אינערלעכע נאָמען־קאָנווענציע איז אַנדערש פֿון באַניצער־געזעענע לייבלס
- איר ווילט קירצערע לייבלס אין נאַוויגאַציע ווי אין דער אַדמין־פּאַנעל
- איר דאַרפֿט שטיצן מערערע שפּראַכן מיט אַנדערע לייבל־לענגען
- איר בויט מעניוס פֿאַר ספּעציפֿישע געגנטן אָדער באַניצער־גרופּעס

## באַניץ־בײַשפּילן

### בײַשפּיל 1: פּשוטער מעניו מיט נאַוויגאַציע־לייבל

**אָנפֿרעג:**
```
שאַף אַ הויפּט־נאַוויגאַציע־מעניו מיטן נאָמען "Main Navigation" מיטן נאַוויגאַציע־לייבל "Menu".
```

**רעזולטאַט:**
```json
{
  "success": true,
  "menu_id": 7,
  "name": "Main Navigation",
  "navigation_label": "Menu"
}
```

### בײַשפּיל 2: מעניו פֿאַר אַ ספּעציפֿישן Theme־אָרט

**אָנפֿרעג:**
```
שאַף אַ פֿוסצעטל־מעניו מיטן נאָמען "Footer Links" מיטן נאַוויגאַציע־לייבל "Quick Links" און טייל עס צו צום פֿוסצעטל־אָרט.
```

**רעזולטאַט:**
```json
{
  "success": true,
  "menu_id": 8,
  "name": "Footer Links",
  "navigation_label": "Quick Links",
  "location": "footer"
}
```

## אינטעגראַציע מיט Theme Builder

ווען מען ניצט Theme Builder, די שאַפֿן מעניו פֿעיִקייט:

1. דערקענט אויטאָמאַטיש בנימצא Theme־מעניו־ערטער
2. שאַפֿט מעניוס מיט פּאַסיקע נאַוויגאַציע־לייבלס פֿאַר אײַער פּלאַן
3. טיילט צו מעניוס צו די ריכטיקע Theme־ערטער
4. שטיצט צולייגן מעניו־אייטעמס נאָך שאַפֿן

## פֿאַרבונדענע פֿעיִקייטן

- **`add_menu_item`** — צולייגן אייטעמס צו אַן עקזיסטירנדיקן מעניו
- **`update_menu`** — איבערנעמען אַ מעניו אָדער אים ווידער צוטיילן צו אַ Theme־אָרט
- **`delete_menu`** — אַראָפּנעמען אַ מעניו פֿון אײַער וועבזײַט

## בעסטע פּראַקטיקן

- **ניצט קלאָרע נאַוויגאַציע־לייבלס** — האַלט לייבלס קורץ און באַשרײַבנדיק פֿאַר באַניצער
- **פּאַסט צו Theme־ערטער** — טיילט צו מעניוס צו דעם ריכטיקן Theme־אָרט פֿאַר געהעריקן ווײַזונג
- **פּלאַנירט מעניו־סטרוקטור** — באַשליסט אײַער מעניו־היעראַרכיע איידער איר שאַפֿט אייטעמס
- **פּרוּווט אָפּרופֿיקייט** — באַשטעטיקט אַז מעניוס ווערן ריכטיק געוויזן אויף מאָבילע מיטלען
- **לאָקאַליזירט לייבלס** — ניצט אַנדערע נאַוויגאַציע־לייבלס פֿאַר אַנדערע שפּראַך־ווערסיעס

## באַגרענעצונגען

- נאַוויגאַציע־לייבלס זענען בלויז פֿאַר ווײַזונג; דער אינערלעכער `name` ווערט נאָך אַלץ געניצט פֿאַר WordPress אידענטיפֿיקאַציע
- Theme־שטיצע בײַט זיך; נישט אַלע themes שטיצן אַלע מעניו־ערטער
- מעניו־אייטעמס מוז מען צולייגן באַזונדער נאָך מעניו־שאַפֿן
- טוישן אַ נאַוויגאַציע־לייבל דאַרף דערהײַנטיקן דעם מעניו
