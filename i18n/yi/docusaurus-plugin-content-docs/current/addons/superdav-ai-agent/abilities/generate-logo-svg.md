# שאַפֿן לאָגאָ SVG

די **שאַפֿן לאָגאָ SVG** פֿעיִקייט דערלויבט Theme Builder צו שאַפֿן און אַרײַנלייגן מנהג־לאָגאָ SVGs גלײַך אין אײַער WordPress וועבזײַטל מיט אויטאָמאַטישער namespace־זיכערער רייניקונג.

## איבערבליק

די פֿעיִקייט שאַפֿט פֿאַרגרעסערבאַרע וועקטאָר־גראַפֿיקן (SVG) לאָגאָס באַזירט אויף דער בראַנדינג־ריכטונג און דיזײַן־פּרעפֿערענצן פֿון אײַער וועבזײַטל. די געשאַפֿענע SVGs ווערן אויטאָמאַטיש גערייניקט כּדי צו פֿאַרזיכערן אַז זיי זענען זיכער צו נוצן אין WordPress, בשעת מען היט אָפּ די וויזועלע גאַנצקייט.

## פּאַראַמעטערס

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `site_name` | string | Yes | דער נאָמען פֿונעם וועבזײַטל פֿאַר וועלכן צו שאַפֿן דעם לאָגאָ |
| `style` | string | Yes | דער דיזײַן־סטיל (למשל, "מאָדערן", "קלאַסיש", "מינימאַליסטיש", "שפּילעוודיק") |
| `colors` | array | No | אַ ריי פֿון העקס־קאָליר־קאָדן צו נוצן אינעם לאָגאָ (למשל, `["#678233", "#ffffff"]`) |
| `width` | number | No | SVG ברייט אין פּיקסעלן (פעליקייַט: 200) |
| `height` | number | No | SVG הייך אין פּיקסעלן (פעליקייַט: 200) |
| `include_text` | boolean | No | צי אַרײַנצונעמען דעם וועבזײַטל־נאָמען ווי טעקסט אינעם לאָגאָ (פעליקייַט: true) |

## אַרויסגאַנג־פֿאָרמאַט

די פֿעיִקייט גיט צוריק אַ SVG שטריקל מיט דער ווײַטערדיקער סטרוקטור:

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## SVG רייניקונג־פֿירונג

די געשאַפֿענע SVGs גייען דורך אויטאָמאַטישע namespace־זיכערע רייניקונג כּדי צו:

- **אַראָפּנעמען אומזיכערע אַטריבוטן** — נעמט אַראָפּ געשעעניש־האַנדלערס, סקריפּטן, און פּאָטענציעל געפֿערלעכע אַטריבוטן
- **אָפּהיטן namespaces** — האַלט אָפּ SVG namespaces (xmlns, xlink) פֿאַר ריכטיקן ווײַז
- **וואַלידירן סטרוקטור** — פֿאַרזיכערט אַז דער SVG שטימט מיט W3C סטאַנדאַרטן
- **קאָדירן entities** — אַנטלויפֿט ריכטיק ספּעציעלע צייכנס אין טעקסט־אינהאַלט
- **אַראָפּנעמען אויסערלעכע רעפֿערענצן** — נעמט אַראָפּ אויסערלעכע סטילבלעטער און בילד־רעפֿערענצן

דאָס פֿאַרזיכערט אַז דער SVG איז זיכער אַרײַנצולייגן גלײַך אין WordPress אָן דאַרפֿן נאָך רייניקונג.

## באַנוץ־בײַשפּיל

**אָנפֿרעג:**
```
שאַף אַ מאָדערנעם לאָגאָ פֿאַר מײַן טעכנאָלאָגיע־סטאַרטאַפּ מיטן נאָמען "CloudSync" ניצנדיק בלויע און ווײַסע קאָלירן.
```

**רעזולטאַט:**
די פֿעיִקייט שאַפֿט אַ SVG לאָגאָ וואָס:
- נעמט אַרײַן דעם וועבזײַטל־נאָמען "CloudSync"
- ניצט די באַשטימטע בלויע און ווײַסע קאָליר־סכעמע
- פֿאָלגט מאָדערנע דיזײַן־פּרינציפּן
- איז אויטאָמאַטיש גערייניקט און גרייט צו נוצן

## אינטעגראַציע מיט Theme Builder

ווען מען ניצט Theme Builder'ס דיזײַן־ריכטונג־אויסקלײַב, די שאַפֿן לאָגאָ SVG פֿעיִקייט:

1. אַנאַליזירט אײַער דיזײַן־ריכטונג און קאָליר־פּאַלעטע
2. שאַפֿט אַ מנהג־SVG לאָגאָ וואָס פּאַסט צו אײַערע פּרעפֿערענצן
3. לייגט אויטאָמאַטיש אַרײַן דעם לאָגאָ אין אײַער וועבזײַטלס קעפּל/בראַנדינג־געגנט
4. שפּײַכערט דעם SVG ווי אַ מנהג־לאָגאָ אין WordPress מעדיע

## בעסטע פּראַקטיקן

- **גיט קלאָרע סטיל־פּרעפֿערענצן** — באַשרײַבט דעם דיזײַן־סטיל וואָס איר ווילט (מאָדערן, קלאַסיש, שפּילעוודיק, אאַז"ו)
- **באַשטימט קאָלירן** — נעמט אַרײַן אײַערע בראַנד־קאָלירן פֿאַר קאָנסיסטענץ
- **טעסט ווײַז** — באַשטעטיקט אַז דער לאָגאָ ווערט געוויזן ריכטיק איבער פֿאַרשידענע עקראַן־גרייסן
- **פּאַסט ווײַטער צו** — ניצט WordPress'ס לאָגאָ־קאַסטאַמיזאַציע־מכשירים צו צופּאַסן גרייס און פּלאַץ

## באַגרענעצונגען

- SVG לאָגאָס ווערן געשאַפֿן ווי סטאַטישע גראַפֿיקן (נישט אַנימירט)
- קאָמפּלעקסע לאָגאָס מיט פילע עלעמענטן קענען דאַרפֿן האַנטיקע פֿאַרפֿײַנערונג
- מנהג־שריפֿטן ווערן נישט געשטיצט; טעקסט ניצט סיסטעם־שריפֿטן
- זייער גרויסע אָדער זייער קליינע דימענסיעס קענען ווירקן אויף קוואַליטעט
