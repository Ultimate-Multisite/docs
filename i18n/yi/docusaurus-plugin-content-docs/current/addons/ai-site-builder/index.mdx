# AI פּלאַץ־בויער

ultimate-multisite-ai-site-builder

## פֿאָדערונגען

- WordPress 5.3 אָדער העכער
- PHP 7.4 אָדער העכער
- Multisite Ultimate פּלוגין (אַקטיוו)

## אינסטאַלאַציע

1. אַרויפֿלאָדן די צוגאָב־טעקעס אין אייער `/wp-content/plugins/` טעקע־רעדל
2. אַקטיווירן דעם פּלוגין דורך דעם 'פּלוגינס' מעניו אין WordPress
3. קאָנפֿיגורירן די צוגאָב־איינשטעלונגען אין דעם נעץ־אַדמין

## אַנטוויקלונג

### אויפֿשטעל

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### טעסטינג

די צוגאָב נעמט אַרײַן אַ ברייטע טעסטינג־ראַם:

- **איינס־טעסטן**: PHPUnit־באַזירטע טעסטן מיט WordPress טעסט־סוויט
- **באַזע־טעסט־קלאַס**: שטעלט צו הילף־מעטאָדן פֿאַר געוויינטלעכע טעסט־סצענאַרן
- **טעסט־דעקונג**: קאָנפֿיגורירט צו נאָכפֿאָלגן קאָד־דעקונג
- **Multisite טעסטינג**: טעסטן לויפֿן אין אַ multisite סבֿיבֿה

### קאָד־סטאַנדאַרטן

- **WordPress קאָדירונג־סטאַנדאַרטן**: דורכגעפֿירט דורך PHPCS
- **PHP 7.4+ קאָמפּאַטיביליטעט**: מאָדערנע PHP פֿעיִקייטן געשטיצט
- **סטאַטישע אַנאַליז**: PHPStan אינטעגראַציע פֿאַר טיפּ־קאָנטראָל
- **קאָד־מאָדערניזאַציע**: Rector כּללים פֿאַר PHP אַפּגריידס

### בוי־סיסטעם

- **אַסעט־פּראָצעסירונג**: מיניפֿיקאַציע פֿאַר CSS/JS טעקעס
- **איבערזעצונג**: POT טעקע־גענעראַציע
- **פּאַקעטירונג**: אַרכיוו־שאַפֿונג פֿאַר פֿאַרשפּרייטונג
- **אַנטוויקלונג־מאָדוס**: ניט־מיניפֿיצירטע אַסעטן פֿאַר דעבאַגינג

## קאָנפֿיגוראַציע

די צוגאָב שטיצט פֿאַרשיידענע קאָנפֿיגוראַציע־אָפּציעס וואָס זענען צוטריטלעך דורך דעם נעץ־אַדמין צובינד.

## Hooks און פֿילטערס

### אַקציעס

- `ultimate-multisite-ai-site-builder_init` - אויסגעלייזט נאָך דער איניציאַליזאַציע פֿון דער צוגאָב
- `ultimate-multisite-ai-site-builder_loaded` - אויסגעלייזט נאָך דעם ווי אַלע צוגאָב־טעקעס זענען געלאָדן

### פֿילטערס

- `ultimate-multisite-ai-site-builder_settings` - ענדערן צוגאָב־איינשטעלונגען
- `ultimate-multisite-ai-site-builder_enabled` - איבערשרײַבן דעם אַקטיווירן/דיאַקטיווירן סטאַטוס פֿון דער צוגאָב
