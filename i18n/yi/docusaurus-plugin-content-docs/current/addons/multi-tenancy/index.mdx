# Multi-Tenancy {#multi-tenancy}

צעשפּרייט אונטער־פּלעצער איבער סערווערס און דאַטאַבאַזעס פֿאַר פאָרשטעלונג און זיכערקייט

## Version 1.2.0 אַרויסלאָז־הויכפּונקטן {#version-120-release-highlights}

Multi-Tenancy 1.2.0 לייגט צו דעם זעלבשטענדיקן טענאַנט־יסוד פֿאַר פּראָדוקציע־איזאָלאַציע־אַרבעטספֿלוסן:

- סטאַטעלעס טענאַנט־אויטאָלאָגין מיט קורץ־גילטיקע, קעגן־ווידערשפּיל־באַשיצטע SSO באַזוכן.
- פּער־אונטער־פּלאַץ דאַטאַבאַזע־ און טעקע־סיסטעם־איזאָלאַציע פֿאַר זעלבשטענדיקע טענאַנטן.
- Hostinger מער־טענאַנטקייט־פֿעיִקייט פֿאַר האָסט־באַוווּסטע טענאַנט־פּראָוויזשאַנינג.
- זעלבשטענדיקע מיגראַציע־וועריפֿיקאַציע־טויערן פֿאַר לעגאַט־דאַטן־קאָנטראָלן און אַסינכראָנע פּוש־פּראַסעסירונג.
- קאָנפֿיגורירבאַרע זעלביקער־מאַשין דאַטאַבאַזע־האָסט־בינדונג פֿאַר Bedrock, FrankenPHP, און localhost אינסטאַלאַציעס.
- טענאַנט־bootstrap, דאַטאַבאַזע־שרײַבער, schema bootstrap, און באַניצער־פּראָוויזשאַנינג פֿאַרבעסערונגען.
- זיכערערע זעלבשטענדיקע אָפּבויִונג וואָס רייניקט טענאַנט־קרעדענשאַלן.

זעט די באַניצער־וועגווייזערס פֿאַר [זעלבשטענדיקן מיגראַציע־אַרבעטספֿלוס](/user-guide/getting-started/sovereign-migration-workflow), [מער־טענאַנטקייט־איזאָלאַציע](/user-guide/configuration/multi-tenancy-isolation), און [זעלבשטענדיקע טענאַנט־אָפּעראַציעס](/user-guide/administration/sovereign-tenant-operations).

## פֿאָדערונגען {#requirements}

- WordPress 5.3 אָדער העכער
- PHP 7.4 אָדער העכער
- Multisite Ultimate צופּלאָג (אַקטיוו)

## אינסטאַלאַציע {#installation}

1. אַרויפֿלאָדט די אַדאָן־טעקעס אין אײַער `/wp-content/plugins/` וועגווײַזער
2. אַקטיווירט דעם צופּלאָג דורך דעם 'צופּלאָגן' מעניו אין WordPress
3. קאָנפֿיגורירט די אַדאָן־אײַנשטעלונגען אין דער נעצווערק־אַדמיניסטראַציע

### טעסטינג {#testing}

דער אַדאָן נעמט אַרײַן אַ ברייטן טעסטינג־ראַם:

- **Unit Tests**: PHPUnit־באַזירטע טעסטן מיט WordPress טעסט־סוויט
- **באַזע־טעסט־קלאַס**: שטעלט צו הילף־מעטאָדן פֿאַר געוויינטלעכע טעסט־סצענאַרן
- **טעסט־דעקונג**: קאָנפֿיגורירט צו נאָכפֿאָלגן קאָד־דעקונג
- **Multisite טעסטינג**: טעסטן לויפֿן אין אַ multisite סבֿיבֿה

### קאָד־סטאַנדאַרטן {#code-standards}

- **WordPress קאָדירונג־סטאַנדאַרטן**: דורכגעפֿירט דורך PHPCS

## קאָנפֿיגוראַציע {#configuration}

דער אַדאָן שטיצט פֿאַרשיידענע קאָנפֿיגוראַציע־אָפּציעס וואָס זענען צוטריטלעך דורך דעם נעצווערק־אַדמין צובינד.

### זעלבשטענדיקע טענאַנט־קאָנפֿיגוראַציע {#sovereign-tenant-configuration}

פֿאַר זעלבשטענדיקע טענאַנטן, קאָנפֿיגורירט די טענאַנט־דאַטאַבאַזע, טעקע־סיסטעם־וואָרצל, דאַטאַבאַזע־האָסט־בינדונג, און האָסט־פֿעיִקייט איידער מ׳רוטירט פֿאַרקער צום טענאַנט. פֿירט אויס מיגראַציע־וועריפֿיקאַציע נאָך יעדער ענדערונג וואָס ווירקט אויף דאַטאַבאַזע־קרעדענשאַלן, האָסט־בינדונג, רוטירונג, אָדער באַניצער־פּראָוויזשאַנינג.

ניצט `localhost` נאָר ווען די טענאַנט־runtime און דאַטאַבאַזע־גראַנץ דערוואַרטן לאָקאַלן socket אָדער זעלביקער־מאַשין צוטריט. ניצט `127.0.0.1` אָדער אַ סערוויס־האָסטנאָמען ווען דאָס איז דער grant host וואָס אײַער provider האָט קאָנפֿיגורירט.

### Hostinger פֿעיִקייט {#hostinger-capability}

די Hostinger מער־טענאַנטקייט־פֿעיִקייט אַרבעט מיט דעם געטיילטן Hostinger אינטעגראַציע־טאָקען. באַשטעטיקט אַז דער דאָמיין איז פֿאַראַן אין hPanel, DNS ווײַזט צום Hostinger Account, און די טענאַנט־דאַטאַבאַזע־אײַנשטעלונגען שטימען מיט hPanel איידער וועריפֿיקאַציע.

## Hooks און Filters {#hooks-and-filters}

### אַקציעס {#actions}

- `ultimate-multisite-addon-template_init` - ווערט ausgelöst נאָך אַדאָן־איניציאַליזאַציע
- `ultimate-multisite-addon-template_loaded` - ווערט ausgelöst נאָך אַלע אַדאָן־טעקעס זענען געלאָדן

### Filters {#filters}

- `ultimate-multisite-addon-template_settings` - ענדערן אַדאָן־אײַנשטעלונגען
- `ultimate-multisite-addon-template_enabled` - איבערשרײַבן אַדאָן־אײַנ/אויס סטאַטוס
