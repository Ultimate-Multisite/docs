# AffiliateWP

Ultimate Multisite: AffiliateWP אינטעגראַציע

נוצט דעם שטאַרקן AffiliateWP כּדי צו פֿאַרגרעסערן די קליענטן־באַזע פֿון אייער Ultimo נעץ!

## אינסטאַלאַציע {#installation}

1. אַרויפֿלאָדן 'ultimate-multisite-git ' צום '/wp-content/plugins/' וועגווײַזער
2. אַקטיווירן דעם צאַפּל דורך דעם 'צאַפּלעך' מעניו אין WordPress
3. נאָכגיין דעם שריט־נאָך־שריט כּישוף־העלפֿער כּדי אײַנצושטעלן דעם צאַפּל

## באַניץ־וועגווייזער {#usage-guide}

[AffiliateWP](https://affiliatewp.com/) איז געוואָרן די הויפּט־לייזונג בײַם בויען אַן אַפֿיליאַט־פּראָגראַם פֿאַר אייער אָנליין געשעפֿט. Ultimate Multisite: AffiliateWP איז דער בריק וואָס וועט אייך דערמעגלעכן צו בויען דאָס זעלבע פֿאַר אייער פּרעמיום נעץ.

## אָנהייבן {#getting-started}

קודם־כּל דאַרפֿט איר **האָבן** [**AffiliateWP**](https://affiliatewp.com/) **אַקטיוו**! איר וועט נישט קענען נוצן אונדזער צוגאָב אויב איר האָט נישט **דעם צאַפּל וואָס אינטעגרירט זיך מיט אים**. אויב איר פּלאַנירט נישט צו לאָזן אייערע קליענטן עס אויך נוצן אויף זייערע זייטלעך, זאָל AffiliateWP זײַן אַקטיוו נאָר אויף אייער הויפּט־זייטל.

אויך דאַרפֿט איר האָבן ביידע **Ultimate Multisite** און **Ultimate Multisite: AffiliateWP** **אַקטיווירט** **איבער דער גאַנצער נעץ**.

## צולייגן Ultimate Multisite ווי אַן AffiliateWP אינטעגראַציע {#adding-ultimate-multisite-as-an-affiliatewp-integration}

אויף אייער הויפּט־זייטל, גייט צו אַפֿיליאַטן־איינשטעלונגען, און אויפֿן אינטעגראַציעס־קוויטל, צייכנט אָן Ultimate Multisite. דאָס וועט זאָגן [AffiliateWP](https://affiliatewp.com/) אָנהייבן נאָכפֿאָלגן באַזוכן און קאָנווערציעס פֿון דעם Ultimate Multisite רעגיסטראַציע־פֿלוס.
