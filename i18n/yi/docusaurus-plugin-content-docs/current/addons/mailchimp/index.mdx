# Mailchimp

Multisite Ultimate: Mailchimp אינטעגראַציע

אינטעגרירט אייער Multisite Ultimate נעצווערק מיט אייער Mailchimp. איר קענט פֿאַרבינדן געוויסע פּלענער צו Mailchimp ליסטעס און אויטאָמאַטיש צולייגן אַבאָנענטן צו יענע ליסטעס בײַם זיך־אויפשרײַבן.

## אינסטאַלאַציע

1. לאָודט אַרויף 'ultimate-multisite-mailchimp' צו דעם '/wp-content/plugins/' וועגווײַזער
2. אַקטיווירט דעם צאַפּל דורך דעם 'צאַפּלעך' מעניו אין אייער WordPress Network Admin

## באַניץ־וועגווייזער

סינכראָניזירן אַבאָנענטן׳ס בליצפּאָסט־אַדרעסן מיט אייערע MailChimp ליסטעס איז טאַקע גרינג מיט אונדזער Ultimate Multisite: MailChimp Integration Add-on. דער לערנאַרטיקל צילט אייך צו העלפֿן אַלץ אויפֿשטעלן!

## באַקומען אַן API key

איידער מיר אָנהייבן אַרײַנצוגיין אין די פֿאַרשיידענע בנימצאדיקע אײַנשטעלונגען, דאַרפֿן מיר אַ MailChimp API key. דאָס וועט לאָזן Ultimate Multisite רעדן מיט דעם MailChimp API כּדי אָפּצוהאָלן ליסטעס, גרופּעס, און צולייגן בליצפּאָסט־אַדרעסן צו יענע ליסטעס און גרופּעס ווען באַניצער שרײַבן זיך אײַן.

פֿאָלגט דעם לערנאַרטיקל כּדי צו באַקומען אייער אייגענעם MailChimp API key: [וועגן API Keys | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

<!-- בילדשירעם נישט בנימצא: MailChimp API key אײַנשטעלונגען אין Ultimate Multisite Add-on Settings -->

_לייגט צו אייער MailChimp API key אויף_ _**Ultimate Multisite אײַנשטעלונגען - > Add-on אײַנשטעלונגען -> MailChimp**_

## אינטעגראַציע־מאָדע

דער MailChimp Integration Add-on אָפפערט צוויי „אינטעגראַציע־מאָדעס“, דאָס הייסט אַז עס זענען דאָ צוויי פֿאַרשיידענע וועגן ווי איר קענט נוצן דעם Add-on צו סינכראָניזירן בליצפּאָסט־אַדרעסן.

<!-- בילדשירעם נישט בנימצא: MailChimp אינטעגראַציע־מאָדע אויסקלייב מיט אָפּציעס פֿאַר מערערע ליסטעס און איין ליסטע -->

_צוויי אינטעגראַציע־מאָדעס פֿאַר גרעסערע בויגזאַמקייט!_

### מאָדע 1: מערערע ליסטעס

די פעליקייט־מאָדע ניצט **מערערע ליסטעס**. די מאָדע וועט אייך לאָזן אויסקלייבן מערערע MailChimp ליסטעס פֿאַר יעדן פון די פּלענער. דאָס איז די בעסטע אָפּציע אויב איר צעטיילט אייערע באַניצער אין פֿאַרשיידענע ליסטעס.

למשל, לאָמיר זיך פֿאָרשטעלן אַז איר האָט 3 פֿאַרשיידענע ליסטעס אויף אייער MailChimp Account: **נײַעס־בריוו** , **פּלאַן A** , און **פּלאַן B**. איר ווילט אַז אַלע באַניצער, אומאָפּהענגיק פֿון זייער פּלאַן, זאָלן אויך זײַן אַרײַנגענומען אויף דער נײַעס־בריוו ליסטע, אַזוי זיי וועלן נישט פֿאַרפֿעלן אייערע דערהײַנטיקונגען.

כּדי דאָס צו דערגרייכן, קענט איר נוצן די מערערע ליסטעס מאָדע צו צולייגן ביידע ליסטעס **נײַעס־בריוו** און **פּלאַן A** צו די פּלאַן A MailChimp אײַנשטעלונגען, און **נײַעס־בריוו** און **פּלאַן B** צו די פּלאַן B MailChimp אײַנשטעלונגען.

<!-- בילדשירעם נישט בנימצא: MailChimp פעליקייט־ליסטעס אויסקלייב פֿאַר נײַע Accounts און אַנולירטע Accounts -->

_קלייבט אויס די פעליקייט־ליסטעס פֿאַר נײַע Accounts און פֿאַר אַנולירטע Accounts_

### מאָדע 2: איין ליסטע + מערערע גרופּעס

MailChimp לאָזט אייך צעטיילן אייער ליסטע אין קלענערע גרופּעס באַניצער, גערופֿן **גרופּעס**. דאָס לאָזט אייך האָבן מער קאָנטראָל, ווײַל איר קענט שיקן אַ קאַמפּאַניע צו אַ ספּעציפֿישן אונטערזאַץ פֿון אייערע אַבאָנענטן אָן צו דאַרפֿן שאַפֿן מערערע קאָפּיעס פֿון דער זעלבער קאַמפּאַניע אויף MailChimp (וואָס מאַכט עס שווער צו אַנאַליזירן די מעטריקן פֿון יענער קאַמפּאַניע שפּעטער).

למשל, איר קענט האָבן איין ליסטע גערופֿן **מײַנע נעצווערק־אַבאָנענטן,** און דערין האָבן גרופּעס פֿאַר **פּלאַן A** און **פּלאַן B**.

<!-- בילדשירעם נישט בנימצא: MailChimp איין ליסטע און גרופּעס אויסקלייב פֿאַר פעליקייט־ליסטעס -->

_קלייבט אויס די פעליקייט־ליסטעס און גרופּעס, אויב מען ניצט די „איין ליסטע + מערערע גרופּעס“ אינטעגראַציע־מאָדע_

## פעליקייט־ליסטעס און גרופּעס

נאָך דעם ווי איר קלייבט אויס וועלכע אינטעגראַציע־מאָדע מאַכט מער זין פֿאַר אייער באַזונדערן באַניץ־פֿאַל, איז צײַט אויסצוקלייבן די פעליקייט־ליסטעס (און גרופּעס, לויט דער מאָדע). די פעליקייט־ליסטע וועט געניצט ווערן ווען אַ געוויסער פּלאַן האָט נישט אויסגעקליבן קיין ספּעציפֿישע ליסטע.

איר קענט אויך אויסקלייבן די פעליקייט־ליסטע פֿאַר אַנולירטע Accounts. ווען אַ באַניצער פֿאַרענדיקט זײַן אָדער איר אַבאָנעמענט און נעמט אַראָפּ זײַן אָדער איר Account פֿון אייער נעצווערק, וועט זײַן אָדער איר בליצפּאָסט־אַדרעס אַריבערגעפֿירט ווערן צו דער ליסטע וואָס איז באַשטימט פֿאַר אַנולירטע Accounts.

## אַקטיווירן טאָפּל־אַבאָנעמענט

אויף דער זעלבער אײַנשטעלונגען־זײַט, וועט איר קענען אַקטיווירן **טאָפּל־אַבאָנעמענט**. די אָפּציע וועט צולייגן דעם באַניצער סײַ צו דער אויסגעקליבענער פעליקייט־ליסטע און סײַ צו דער ליסטע אויסגעקליבן פֿאַר יענעם באַניצער׳ס פּלאַן. אויב די אָפּציע איז נישט אַקטיווירט, וועט דער באַניצער בלויז צוגעלייגט ווערן צו דער ליסטע אויסגעקליבן פֿאַר דעם באַניצער׳ס פּלאַן. אויב גאָרנישט איז באַשטימט פֿאַר יענעם באַזונדערן פּלאַן, דאַן וועט דער באַניצער צוגעלייגט ווערן צו דער אויסגעקליבענער פעליקייט־ליסטע.

<!-- בילדשירעם נישט בנימצא: MailChimp טאָפּל־אַבאָנעמענט איבערשלייטער־אײַנשטעלונג -->

_איר קענט האָבן אייערע באַניצער אַבאָנירט סײַ צו דער פעליקייט־ליסטע און סײַ צו דער פּלאַן־ליסטע בײַם זיך־אויפשרײַבן_

## אויסקלייבן ליסטעס און גרופּעס פֿאַר יעדן פֿון די פּלענער

גייט צו אייערע **פּלענער** און קלייבט אויס דעם פּלאַן וואָס איר ווילט רעדאַגירן. באַמערקט אַז אויף דער **פֿאָרגעשריטענע אָפּציעס** מעטאַ־קעסטל, וועט זײַן בנימצא אַ נײַער קוויטל גערופֿן MailChimp.

אויף יענעם קוויטל, וועט איר קענען אויסקלייבן מערערע ליסטעס אָדער איין ליסטע און מערערע גרופּעס (לויט דער אינטעגראַציע־מאָדע) פֿאַר יענעם פּלאַן.

<!-- בילדשירעם נישט בנימצא: MailChimp קוויטל אויף דער פּלאַן־רעדאַגירן־זײַט וואָס ווײַזט אויסקלייב פֿון ליסטעס און גרופּעס -->

_קלייבט אויס די ליסטעס און גרופּעס וואָס איר ווילט נוצן פֿאַר יעדן פֿון אייערע פּלענער_

ווען אַ נײַער באַניצער שרײַבט זיך אײַן פֿאַר יענעם פּלאַן, וועט זײַן בליצפּאָסט־אַדרעס צוגעלייגט ווערן צו די אויסגעקליבענע ליסטעס און גרופּעס!
