# בויען אַ נישע־וועבזײַט־געשעפֿט מיט Ultimate Multisite

רובֿ מענטשן וואָס פּרוּוון פֿאַרקויפֿן וועבזײַטן קאָנקורירן אויף פּרײַז קעגן Wix, Squarespace, און טויזנטער פֿרילאַנסערס. זיי פֿאַרלירן. די וואָס געווינען קלײַבן אויס אַ ספּעציפֿישן עולם, בויען אַ לייזונג צוגעפּאַסט צו יענעם עולם, און נעמען געלט פֿאַר דעם ווערט וואָס זיי באַזאָרגן.

דער קורס לערנט אײַך ווי אַזוי צו טאָן פּונקט דאָס מיט Ultimate Multisite.

## וואָס איר וועט בויען {#what-you-will-build}

דורכאויס דעם קורס וועט איר בויען אַן אמתע, אַרבעטנדיקע נעץ מיטן נאָמען **FitSite** -- אַ וועבזײַט־פּלאַטפֿאָרמע געבויט ספּעציעל פֿאַר פֿיטנעס־סטודיאָס, פּערזענלעכע טרענירער, און דזשים־באַזיצער.

בײַ דער לעצטער לעקציע וועט FitSite זײַן אַ גאַנצע נישע־פּלאַטפֿאָרמע מיט:

- צילגעבויטע וועבזײַט־מוסטערן דיזײַנט פֿאַר פֿיטנעס־געשעפֿטן
- פּלאַן־שטאַפּלען וואָס פּאַסן צו ווי פֿיטנעס־געשעפֿטן פֿירן זיך טאַקע
- אַ בראַנדירטער checkout און onboarding־לויפֿ
- פּרײַזן צוגעפּאַסט צו וואָס דער מאַרק וועט באַצאָלן
- אַ סטראַטעגיע צו געפֿינען און האַלטן קונים אין דער פֿיטנעס־ווערטיקאַל

איר וועט מיטגיין שריט נאָך שריט, און צום סוף וועט איר האָבן סײַ אַן אַרבעטנדיקע פּלאַטפֿאָרמע סײַ דאָס וויסן צו איבערחזרן דעם פּראָצעס פֿאַר וועלכער נישע איר קלײַבט אויס.

## פֿאַרוואָס פֿיטנעס־סטודיאָס? {#why-fitness-studios}

מיר האָבן געדאַרפֿט אַ קאָנקרעטן בײַשפּיל וואָס יעדער קען נאָכפֿאָלגן. פֿיטנעס־סטודיאָס טויגן ווײַל:

- **יעדער פֿאַרשטייט דאָס געשעפֿט** -- קיין ספּעציאַליסטישע קענטעניש איז נישט נייטיק צו מיטגיין
- **זיי האָבן קלאָרע וועבזײַט־באַדערפֿענישן** -- קלאַס־פּלענער, טרענירער־פּראָפֿילן, בוקינג, אָרט־אינפֿאָרמאַציע
- **זיי זענען נישט גענוג באַדינט** -- אַלגעמיינע וועבזײַט־בויערס לייזן נישט זייערע ספּעציפֿישע פּראָבלעמען
- **נאַטירלעכע פּלאַן־שטאַפּלען עקזיסטירן** -- אַ איין פּערזענלעכער טרענירער האָט אַנדערע באַדערפֿענישן ווי אַ דזשים־קייט מיט עטלעכע לאָקאַציעס

אַלץ וואָס איר לערנט בײַם בויען FitSite געלט דירעקט פֿאַר וועלכער נישע איר קלײַבט אויס: רעסטאָראַנען, ריעל־עסטעיט אַגענטן, אַדוואָקאַטן־פֿירמעס, שילן, האַנדווערק־קאָנטראַקטאָרן, אָדער סײַ וועלכע אַנדערע ווערטיקאַל.

## פֿאָרויס־באַדערפֿענישן {#prerequisites}

- אַ WordPress Multisite אינסטאַלאַציע (אָדער וויליקייט אויפֿצושטעלן איינע)
- אַן אַקטיווע Ultimate Multisite ליצענץ
- גרונטלעכע באַקאַנטשאַפֿט מיטן WordPress אַדמין Dashboard

קיין קאָדירונג איז נישט נייטיק. אויב איר האָט נאָך קיינמאָל נישט גענוצט Ultimate Multisite פֿריִער, איז דאָס אין אָרדענונג -- לעקציע 3 הייבט אָן פֿון אַ פֿרישער אינסטאַלאַציע.
