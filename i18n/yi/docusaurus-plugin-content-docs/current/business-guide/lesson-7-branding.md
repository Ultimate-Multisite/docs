# לעקציע 7: מאַכן עס אײַערס

אײַערע קונים זאָלן קיינמאָל נישט פילן ווי זיי נוצן "עפּעס אַ WordPress פּלוגין." זיי זאָלן פילן ווי זיי נוצן FitSite -- אַ פּלאַטפֿאָרמע געבויט פֿאַר זייער אינדוסטריע. די לעקציע באַהאַנדלט בראַנדינג, ווײַס־לייבלינג, און מאַכן די פּלאַטפֿאָרמע זיך פֿילן ווי אַ פּראָדוקט.

## וווּ מיר האָבן זיך אָפּגעשטעלט

FitSite האָט אַ אַרבעטנדיקן קויפֿ־פּראָצעס וואָס פֿירט אייגנטימער פֿון פֿיטנעס־סטודיאָס פֿון פּלאַן־אויסקלייב ביז אַ לעבעדיקן וועבזײַטל. איצט מאַכן מיר די גאַנצע דערפֿאַרונג זיך פֿילן ווי אַ צוזאַמענהענגיקער, געבראַנדעטער פּראָדוקט.

## אײַער פּלאַטפֿאָרמע־דאָמיין

דער יסוד פֿון אײַער בראַנד איז אײַער דאָמיין. פֿאַר FitSite:

- **הויפּט־דאָמיין**: `fitsite.com` (אײַער מאַרקעטינג־וועבזײַטל און נעצווערק־וואָרצל)
- **קונה־וועבזײַטן**: `studioname.fitsite.com` (סובדאָמיינס)
- **אייגענע דאָמיינס**: קונים אויף Growth און Pro פּלענער קענען פֿאַרבינדן זייער אייגענעם דאָמיין

### אײַנשטעלן אײַער דאָמיין

1. רעגיסטרירט אײַער פּלאַטפֿאָרמע־דאָמיין
2. ווײַזט עס צו אײַער האָסטינג־פֿאַרזאָרגער
3. קאָנפֿיגורירט wildcard DNS (`*.fitsite.com`) פֿאַר קונה־סובדאָמיינס
4. פֿאַרזיכערט אַז wildcard SSL איז אַקטיוו

זעט [ווי אַזוי צו קאָנפֿיגורירן דאָמיין־מאַפּינג](/user-guide/domain-mapping/how-to-configure-domain-mapping) פֿאַר דעטאַלירטע אָנווײַזונגען.

## ווײַס־לייבלינג די Admin־דערפֿאַרונג

ווען אַ אייגנטימער פֿון אַ פֿיטנעס־סטודיאָ לאָגט זיך אַרײַן אין זײַן וועבזײַטל־Dashboard, זאָל ער זען אײַער בראַנד, נישט WordPress אָדער Ultimate Multisite בראַנדינג.

### אייגענע לאָגין־בלאַט

פּאַסט צו דעם WordPress לאָגין־בלאַט כּדי צו ווײַזן:

- אײַער FitSite לאָגאָ
- פֿיטנעס־פּאַסיקע הינטערגרונט־בילדער
- אײַערע בראַנד־פֿאַרבן

### Dashboard בראַנדינג

נוצט דעם [Admin־בלאַט־שאַפֿער](/addons/admin-page-creator) addon אָדער אייגענעם CSS צו:

- פֿאַרבײַטן דעם WordPress לאָגאָ מיט אײַער FitSite לאָגאָ
- צופּאַסן די admin־פֿאַרב־סכעמע זי זאָל שטימען מיט אײַער בראַנד
- צולייגן אַן אייגענעם Dashboard widget מיט פֿיטנעס־ספּעציפֿישע שנעלע לינקס און הילף־רעסורסן

### אייגענע Admin־בלעטער

באַטראַכט צו שאַפֿן אייגענע admin־בלעטער וואָס ברענגען אַרויס די מערסט באַטייליקע אַקציעס פֿאַר אייגנטימער פֿון פֿיטנעס־סטודיאָס:

- "רעדאַקטירט אײַער קלאַסן־פּלאַן"
- "דערהײַנטיקט טרענירער־פּראָפֿילן"
- "זען אײַער וועבזײַטל"

דאָס פֿאַרקלענערט די לערן־קורווע דורך שטעלן נישע־באַטייליקע אַקציעס אין צענטער, אַנשטאָט זיי צו באַהאַלטן אין דעם סטאַנדאַרטן WordPress מעניו.

## בראַנדינג פֿון אײַערע קאָמוניקאַציעס

יעדער אימעיל, חשבונית, און מעלדונג זאָל שטאַרקן אײַער בראַנד.

### סיסטעם־אימעילס

גייט צו **Ultimate Multisite > אײַנשטעלונגען > אימעילס** און פּאַסט צו אַלע סיסטעם־אימעילס:

- **פֿון נאָמען**: FitSite
- **פֿון אימעיל**: hello@fitsite.com
- **אימעיל־מוסטערן**: ניצט אײַערע בראַנד־פֿאַרבן און לאָגאָ
- **שפּראַך**: פֿיטנעס־ספּעציפֿיש דורכאויס

שליסל־אימעילס צו צופּאַסן:

| אימעיל | אַלגעמיינע ווערסיע | FitSite ווערסיע |
|-------|----------------|-----------------|
| באַגריסונג | "אײַער נײַער וועבזײַטל איז גרייט" | "אײַער פֿיטנעס־סטודיאָ וועבזײַטל איז לעבעדיק" |
| צאָלונג־קבלה | "צאָלונג באַקומען" | "FitSite אַבאָנעמענט־צאָלונג באַשטעטיקט" |
| פּראָבע־צײַט ענדיקט זיך | "אײַער פּראָבע־צײַט ענדיקט זיך באַלד" | "אײַער FitSite פּראָבע־צײַט ענדיקט זיך אין 3 טעג -- האַלט אײַער סטודיאָ־וועבזײַטל לעבעדיק" |

### חשבוניות

פּאַסט צו חשבונית־מוסטערן מיט:

- אײַער FitSite לאָגאָ און בראַנד־פֿאַרבן
- אײַערע געשעפֿט־דעטאַלן
- פֿיטנעס־ספּעציפֿישע פּראָדוקט־נעמען (נישט אַלגעמיינע פּלאַן־IDs)

## דער קונה־געווענדטער וועבזײַטל

אײַער הויפּט־דאָמיין (`fitsite.com`) דאַרף אַ מאַרקעטינג־וועבזײַטל וואָס פֿאַרקויפֿט די פּלאַטפֿאָרמע. דאָס איז באַזונדער פֿון דעם Ultimate Multisite נעצווערק־admin -- עס איז דאָס עפֿנטלעכע פּנים פֿון אײַער געשעפֿט.

שליסל־בלעטער:

- **היימבלאַט**: קלאָרע ווערט־פֿאָרלייג פֿאַר פֿיטנעס־געשעפֿטן
- **פֿעיִקייטן**: וואָס FitSite טוט, אין פֿיטנעס־טערמינען
- **פּרײַזן**: אײַערע דרײַ פּלענער מיט נישע־ספּעציפֿישע פֿעיִקייט־פֿאַרגלײַכן
- **בײַשפּילן**: ווײַזט וועבזײַטן געבויט אויף דער פּלאַטפֿאָרמע
- **אײַנשרײַבן זיך**: לינקס צו אײַער קויפֿ־פֿאָרעם

:::tip אײַער מאַרקעטינג־וועבזײַטל קען זײַן אַ נעצווערק־וועבזײַטל
שאַפֿט אײַער מאַרקעטינג־וועבזײַטל ווי אַ וועבזײַטל אין אײַער אייגענעם נעצווערק. דאָס לאָזט אײַך אים פֿאַרוואַלטן פֿון דעם זעלבן Dashboard און דעמאָנסטרירט די מעגלעכקייטן פֿון אײַער אייגענער פּלאַטפֿאָרמע.
:::

## אייגענער דאָמיין פֿאַר קונים

פֿאַר קונים אויף פּלענער וואָס אַרײַננעמען אייגענע דאָמיינס, דאָקומענטירט דעם פּראָצעס קלאָר:

1. דער קונה רעגיסטרירט אָדער טראַנספֿערט זײַן דאָמיין צו אַ רעגיסטראַטאָר
2. דער קונה דערהײַנטיקט DNS כּדי צו ווײַזן צו אײַער פּלאַטפֿאָרמע (גיט פּינקטלעכע רעקאָרדס)
3. Ultimate Multisite באַהאַנדלט דעם דאָמיין־מאַפּינג און SSL

שאַפֿט אַ הילף־אַרטיקל אָדער וויסן־באַזע־אײַנטראָג ספּעציפֿיש פֿאַר דעם פּראָצעס, געשריבן פֿאַר ניט־טעכנישע אייגנטימער פֿון פֿיטנעס־סטודיאָס.

## דער FitSite נעצווערק ביז איצט

```
FitSite Network
├── WordPress Multisite (subdomain mode)
├── Ultimate Multisite (configured + branded)
├── Platform Domain (fitsite.com + wildcard SSL)
├── Site Templates (Studio Essential, Gym Pro, Fitness Chain)
├── Products (Starter, Growth, Pro + Order Bumps)
├── Checkout Flow (niche-specific, tested)
├── Branding
│   ├── Custom login page
│   ├── Branded admin dashboard
│   ├── Niche-specific system emails
│   ├── Branded invoices
│   └── Marketing site on fitsite.com
└── Ready for onboarding flow (next lesson)
```

## וואָס מיר האָבן געבויט אין דער לעקציע

- **פּלאַטפֿאָרמע־דאָמיין און DNS** קאָנפֿיגורירט פֿאַר אַ געבראַנדעטער דערפֿאַרונג
- **ווײַס־לייבלד admin** מיט FitSite בראַנדינג דורכאויס
- **צוגעפּאַסטע קאָמוניקאַציעס** -- אימעילס, חשבוניות, און מעלדונגען אַלע לויטן בראַנד
- **אַ מאַרקעטינג־וועבזײַטל** וואָס פֿאַרקויפֿט FitSite צו אייגנטימער פֿון פֿיטנעס־סטודיאָס
- **אייגענער דאָמיין דאָקומענטאַציע** פֿאַר קונים וואָס ווילן זייער אייגענעם דאָמיין
