# מיגרירן פֿון V1 {#migrating-from-v1}

## Ultimate Multisite האָט איבערגעביטן פֿון זײַן אָריגינעלער 1.x משפּחה פֿון אַרויסגאַבעס צו דער 2.x משפּחה פֿון אַרויסגאַבעס. {#ultimate-multisite-has-switched-from-its-original-1x-family-of-releases-to-the-2x-family-of-releases}

Ultimate Multisite ווערסיע 2.0 און העכער איז אַ גאַנצע נײַשרײַבונג פֿון דער קאָד־באַזע, וואָס מיינט אַז עס איז זייער ווייניק געטיילט צווישן דער אַלטער ווערסיע און דער נײַער. צוליב דעם, ווען מען אַפּגריידט פֿון 1.x צו 2.x, דאַרפֿן אײַערע דאַטן ווערן מיגרירט צו אַ פֿאָרמאַט וואָס די נײַע ווערסיעס קענען פֿאַרשטיין.

אַ דאַנק, Ultimate Multisite 2.0+ **קומט מיט אַ מיגרירער** אַרײַנגעבויט אין דעם קערן, וואָס קען דערקענען דאַטן פֿון דער אַלטער ווערסיע און זיי קאָנווערטירן צום נײַעם פֿאָרמאַט. די מיגראַציע קומט פֿאָר בעת דעם **Setup Wizard** פֿון ווערסיע 2.0+.

די לעקציע באַהאַנדלט ווי דער מיגרירער אַרבעט, וואָס צו טאָן אין פֿאַלן פֿון דורכפֿאַל, און ווי צו טראָבלשוטן פּראָבלעמען וואָס קענען זיך אויפֿטאָן בעת דעם פּראָצעס.

_**וויכטיק: איידער איר הייבט אָן אַפּגריידן פֿון ווערסיע 1.x צו ווערסיע 2.0, מאַכט זיכער אַז איר שאַפֿט אַ באַקאַפּ פֿון אײַער פּלאַץ־דאַטאַבאַזע**_

## ערשטע שריט {#first-steps}

דער ערשטער שריט איז אַראָפּלאָדן די plugin .zip טעקע און אינסטאַלירן ווערסיע 2.0 אויף אײַער נעץ־אַדמין Dashboard.

נאָך דעם ווי איר [אינסטאַלירט און אַקטיווירט ווערסיע 2.0](../getting-started/installing-ultimate-multisite.md), וועט דער סיסטעם אויטאָמאַטיש דערקענען אַז אײַער Multisite לויפֿט אויף דער legacy ווערסיע, און איר וועט זען דעם אָנזאָג אויבן אויף דער plugin־זײַט.

_**באַמערקונג:** אויב איר האָט Ultimate Multisite 1.x אינסטאַלירט אויף אײַער Multisite, וועט איר האָבן די אָפּציע צו פאַרבײַטן דעם plugin מיט דער ווערסיע וואָס איר האָט פּונקט אַראָפּלאָדן. ביטע גייט ווײַטער און קליקט אויף **Replace current with uploaded**._

<!-- Screenshot unavailable: Plugin page showing option to replace v1 with uploaded v2 version -->

די קומענדיקע זײַט וועט אײַך לאָזן וויסן וואָס legacy add-ons איר האָט אינסטאַלירט צוזאַמען מיט ווערסיע 1.x. זי וועט האָבן אָנווייזונגען צי די ווערסיע וואָס איר ניצט איז קאָמפּאַטיבל מיט ווערסיע 2.0, אָדער צי איר דאַרפֿט אינסטאַלירן אַן אַפּגריידירטע ווערסיע פֿון דעם add-on נאָך דער מיגראַציע.

ווען איר זענט גרייט ווײַטערצוגיין, קענט איר קליקן דעם קנעפּל וואָס זאָגט **Visit the Installer to finish the upgrade**.

<!-- Screenshot unavailable: Button saying Visit the Installer to finish the upgrade -->

דאַן וועט עס אײַך ברענגען צו דער אינסטאַלאַציע־וויזערד־זײַט מיט עטלעכע באַגריסונג־אָנזאָגן. איר דאַרפֿט נאָר קליקן **Get Started** כּדי אַריבערצוגיין צו דער קומענדיקער זײַט.

<!-- Screenshot unavailable: Setup Wizard welcoming page with Get Started button -->

נאָך קליקן **Get Started** , וועט עס אײַך ווײַטערפֿירן צו די פֿאַר־אינסטאַלירונג־קאָנטראָלן_._ דאָס וועט אײַך ווײַזן אײַער סיסטעם־אינפֿאָרמאַציע און WordPress אינסטאַלאַציע, און אײַך זאָגן צי עס טרעפֿט **Ultimate Multisite's requirements**.

<!-- Screenshot unavailable: Pre-install Checks page showing system requirements confirmation -->

דער קומענדיקער שריט איז אַרײַנצולייגן אײַער Ultimate Multisite ליצענץ־שליסל און אַקטיווירן דעם plugin. דאָס וועט פֿאַרזיכערן אַז אַלע פֿעיִקייטן, אַרײַנגערעכנט add-ons, וועלן זײַן בנימצא אויף אײַער פּלאַץ.

<!-- Screenshot unavailable: License activation page with license key input and Agree and Activate button -->

נאָך אַרײַנלייגן אײַער שליסל, קליקט **Agree & Activate**.

נאָך דער ליצענץ־אַקטיוואַציע, קענט איר אָנהייבן די אמתע אינסטאַלאַציע דורך קליקן **Install** אויף דער קומענדיקער זײַט. דאָס וועט אויטאָמאַטיש שאַפֿן די נייטיקע טעקעס און דאַטאַבאַזע וואָס זענען נייטיק פֿאַר ווערסיע 2.0 צו פונקציאָנירן.

## איצט, די מיגראַציע {#now-the-migration}

דער מיגרירער האָט אַ אַרײַנגעבויטע זיכערהייט־פֿעיִקייט, וווּ ער וועט קאָנטראָלירן אײַער גאַנצן מולטיסייט כּדי זיכערצומאַכן אַז אַלע אײַערע Ultimate Multisite דאַטן קענען ווערן מיגרירט אָן קיין פּראָבלעמען. קליקט דעם **Run Check** קנעפּל צו אָנהייבן דעם פּראָצעס.

<!-- Screenshot unavailable: Migration page with Run Check button to verify data can be converted -->

נאָך דורכפֿירן דעם קאָנטראָל, האָט איר צוויי מעגלעכקייטן: דער רעזולטאַט קען זײַן אָדער **מיט** אַ טעות אָדער **אָן אַ טעות**.

### מיט טעות {#with-error}

אויב איר באַקומען אַ טעות־אָנזאָג, דאַרפֿט איר זיך ווענדן צו אונדזער שטיצע־מאַנשאַפֿט, כּדי זיי זאָלן אײַך קענען העלפֿן פֿאַרריכטן דעם טעות. מאַכט זיכער אַז איר **געבן דעם טעות־לאָג** ווען איר שאַפֿט אַ טיקעט. איר קענט אַראָפּלאָדן דעם לאָג, אָדער איר קענט קליקן דעם לינק וואָס זאָגט זיך צו קאָנטאַקטירן אונדזער שטיצע־מאַנשאַפֿט. עס וועט עפֿענען דעם הילף־ווידזשעט אויף דער רעכטער זײַט פֿון אײַער בלאַט מיט די פֿעלדער שוין פֿאַרפֿילט פֿאַר אײַך, אַרײַנגערעכנט די טעות־לאָגס אונטער דער באַשרײַבונג.

_**ווײַל דער סיסטעם האָט געפֿונען אַ טעות, וועט איר נישט קענען ווײַטערגיין צו מיגרירן צו ווערסיע 2.0. איר קענט דאַן צוריקדרייען צו ווערסיע 1.x כּדי ווידער אָנהייבן לויפֿן אײַער נעץ ביז דער טעות ווערט פֿאַרריכט.**_

### אָן טעות {#without-error}

אויב דער סיסטעם געפֿינט נישט קיין טעות, וועט איר זען אַ הצלחה־אָנזאָג און אַ **Migrate** קנעפּל אונטן, וואָס וועט אײַך דערלויבן ווײַטערצוגיין מיט דער מיגראַציע. אויף דער זײַט וועט מען אײַך דערמאָנען צו שאַפֿן אַ באַקאַפּ פֿון אײַער דאַטאַבאַזע איידער איר גייט ווײַטער, וואָס מיר שטאַרק רעקאָמענדירן. דריקט **Migrate** אויב איר האָט שוין אַ באַקאַפּ.

און דאָס איז אַלץ וואָס עס דאַרף!

איר קענט אָדער ווײַטער לויפֿן דעם Wizard־אויפֿשטעל כּדי דערהײַנטיקן אײַער לאָגאָ און אַנדערע זאַכן אויף אײַער נעץ, אָדער אָנהייבן נאַוויגירן אין אײַער Ultimate Multisite ווערסיע 2.0 מעניו און זײַן נײַעם צובינד. גייט ווײַטער און האָט אַ ביסל שפּאַס.
