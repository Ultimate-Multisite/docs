# WPMU DEV אינטעגראַציע

## איבערבליק {#overview}
WPMU DEV איז אַ פולשטענדיקע WordPress פּלאַטפאָרמע וואָס אָפפערט האָסטינג, plugins, און באַדינונגען פֿאַר WordPress וועבזײַטן. די אינטעגראַציע ערמעגלעכט אויטאָמאַטישע דאָמיין־סינכראָניזאַציע און SSL סערטיפיקאַט־פֿאַרוואַלטונג צווישן Ultimate Multisite און WPMU DEV האָסטינג.

## פֿעיִקייטן {#features}
- אויטאָמאַטישע דאָמיין־סינכראָניזאַציע
- SSL סערטיפיקאַט־פֿאַרוואַלטונג
- פֿאַרלענגערטע SSL סערטיפיקאַט־וועריפֿיקאַציע־פּרוּוון

## פֿאָדערונגען {#requirements}
די אינטעגראַציע דערקענט אויטאָמאַטיש צי איר האָסט אויף WPMU DEV און ניצט די אײַנגעבויטע API. קיין נאָך־קאָנפֿיגוראַציע איז ניט נייטיק אויב איר האָסט אויף WPMU DEV.

די אינטעגראַציע קאָנטראָלירט דעם אָנזײַן פֿון דער `WPMUDEV_HOSTING_SITE_ID` קאָנסטאַנטע, וואָס ווערט אויטאָמאַטיש דעפֿינירט ווען מען האָסט אויף WPMU DEV.

## אויפֿשטעל־אנווײַזונגען {#setup-instructions}

### 1. באַשטעטיקן WPMU DEV האָסטינג {#1-verify-wpmu-dev-hosting}

אויב איר האָסט אויף WPMU DEV, זאָלן די נייטיקע קאָנסטאַנטעס שוין זײַן דעפֿינירט. באַשטעטיקט אַז:

1. די `WPMUDEV_HOSTING_SITE_ID` קאָנסטאַנטע איז דעפֿינירט אין אײַער סבֿיבֿה
2. איר האָט אַן אַקטיווע WPMU DEV מיטגלידערשאַפֿט מיט API צוטריט

### 2. אַקטיווירן די אינטעגראַציע {#2-enable-the-integration}

1. אין אײַער WordPress אַדמין, גייט צו Ultimate Multisite > Settings
2. נאַוויגירט צום "Domain Mapping" קוויטל
3. ראָלט אַראָפּ צו "Host Integrations"
4. אַקטיווירט די WPMU DEV אינטעגראַציע
5. גיט אַ קוועטש אויף "Save Changes"

## ווי עס אַרבעט {#how-it-works}

### דאָמיין־סינכראָניזאַציע {#domain-syncing}

ווען אַ דאָמיין ווערט מאַפּט אין Ultimate Multisite:

1. די אינטעגראַציע ניצט די WPMU DEV API צו צולייגן דעם דאָמיין צו אײַער האָסטינג־Account
2. זי לייגט אויך אויטאָמאַטיש צו די www ווערסיע פֿון דעם דאָמיין
3. WPMU DEV באַהאַנדלט די דאָמיין־קאָנפֿיגוראַציע און SSL סערטיפיקאַט־אַרויסגעבן

### SSL סערטיפיקאַט־פֿאַרוואַלטונג {#ssl-certificate-management}

די אינטעגראַציע איז קאָנפֿיגורירט צו פֿאַרגרעסערן די צאָל SSL סערטיפיקאַט־וועריפֿיקאַציע־פּרוּוון פֿאַר WPMU DEV האָסטינג, ווײַל עס קען נעמען אַ ביסל צײַט ביז SSL סערטיפיקאַטן ווערן אַרויסגעגעבן און אינסטאַלירט. לויטן סטאַנדאַרט וועט זי פּרוּוון ביז 10 מאָל פֿאַר SSL סערטיפיקאַט־וועריפֿיקאַציע, פֿאַרגליכן מיט די סטאַנדאַרטע 5 פּרוּוון.

## וויכטיקע באַמערקונגען {#important-notes}

### דאָמיין־באַזײַטיקונג {#domain-removal}

דערווײַל שטעלט די WPMU DEV API נישט צו קיין וועג צו באַזײַטיקן דאָמיינס. ווען אַ דאָמיין־מאַפּינג ווערט באַזײַטיקט אין Ultimate Multisite, וועט דער דאָמיין בלײַבן אין אײַער WPMU DEV האָסטינג־Account. איר וועט דאַרפֿן אים מאַנועל אַראָפּנעמען פֿון דעם WPMU DEV האָסטינג־Dashboard אויב נייטיק.

### API אויטענטיפֿיקאַציע {#api-authentication}

די אינטעגראַציע ניצט דעם WPMU DEV API key וואָס איז סטאָרד אין אײַער WordPress דאַטאַבאַזע ווי די `wpmudev_apikey` אָפּציע. דאָס ווערט אויטאָמאַטיש אויפֿגעשטעלט ווען איר פֿאַרבינדט אײַער וועבזײַט מיט WPMU DEV.

## פּראָבלעמען־לייזונג {#troubleshooting}

### API פֿאַרבינדונג־פּראָבלעמען {#api-connection-issues}
- באַשטעטיקט אַז אײַער וועבזײַט איז ריכטיק פֿאַרבונדן מיט WPMU DEV
- קאָנטראָלירט אַז די `wpmudev_apikey` אָפּציע איז באַשטימט אין אײַער WordPress דאַטאַבאַזע
- פֿאַרזיכערט אַז אײַער WPMU DEV מיטגלידערשאַפֿט איז אַקטיוו

### SSL סערטיפיקאַט־פּראָבלעמען {#ssl-certificate-issues}
- WPMU DEV קען נעמען אַ ביסל צײַט אַרויסצוגעבן SSL סערטיפיקאַטן (געוויינטלעך 5–15 מינוט)
- די אינטעגראַציע איז קאָנפֿיגורירט צו קאָנטראָלירן ביז 10 מאָל פֿאַר SSL סערטיפיקאַטן
- אויב SSL סערטיפיקאַטן ווערן נאָך אַלץ נישט אַרויסגעגעבן נאָך מערערע פּרוּוון, קאָנטאַקטירט WPMU DEV שטיצע

### דאָמיין נישט צוגעלייגט {#domain-not-added}
- קאָנטראָלירט די Ultimate Multisite לאָגס פֿאַר וועלכע־ניט־איז טעות־מעלדונגען
- באַשטעטיקט אַז דער דאָמיין איז נישט שוין צוגעלייגט צו WPMU DEV
- פֿאַרזיכערט אַז אײַער WPMU DEV האָסטינג־פּלאַן שטיצט די צאָל דאָמיינס וואָס איר לייגט צו
