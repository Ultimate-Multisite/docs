# cPanel אינטעגראַציע

## איבערבליק
cPanel איז איינע פֿון די פּאָפּולערסטע קאָנטראָל־פּאַנעלן פֿאַר וועב־האָסטינג, וואָס ווערט געניצט דורך פילע געמיינזאַמע און דעדיקירטע האָסטינג־פּראָוויידערס. די אינטעגראַציע דערמעגלעכט אויטאָמאַטישע דאָמיין־סינכראָניזאַציע צווישן Ultimate Multisite און cPanel, און לאָזט אײַך אויטאָמאַטיש צולייגן דאָמיין־אַליאַסן און אונטערדאָמיינען צו אײַער cPanel Account.

## פֿעיִקייטן
- אויטאָמאַטישע שאַפֿונג פֿון צוגעלייגטע דאָמיינען אין cPanel
- אויטאָמאַטישע שאַפֿונג פֿון אונטערדאָמיינען אין cPanel (פֿאַר אונטערדאָמיין־מולטיסייט־אינסטאַלאַציעס)
- באַזייַטיקונג פֿון דאָמיינען ווען פֿאַרבינדונגען ווערן אויסגעמעקט

## פֿאָדערונגען
די ווײַטערדיקע קאָנסטאַנטן מוזן זײַן דעפֿינירט אין אײַער `wp-config.php` טעקע:

```php
define('WU_CPANEL_USERNAME', 'your_cpanel_username');
define('WU_CPANEL_PASSWORD', 'your_cpanel_password');
define('WU_CPANEL_HOST', 'your_cpanel_host');
```

אָפּציאָנעל קענט איר אויך דעפֿינירן:

```php
define('WU_CPANEL_PORT', 2083); // Default is 2083
define('WU_CPANEL_ROOT_DIR', '/public_html'); // Default is /public_html
```

## אויפֿשטעל־אינסטרוקציעס

### 1. באַקומען אײַערע cPanel קרעדענציאַלן

1. באַקומט אײַער cPanel באַניצערנאָמען און פּאַראָל פֿון אײַער האָסטינג־פּראָוויידער
2. באַשטימט אײַער cPanel האָסט (געוויינטלעך `cpanel.yourdomain.com` אָדער `yourdomain.com:2083`)

### 2. לייגט צו קאָנסטאַנטן צו wp-config.php

לייגט צו די ווײַטערדיקע קאָנסטאַנטן צו אײַער `wp-config.php` טעקע:

```php
define('WU_CPANEL_USERNAME', 'your_cpanel_username');
define('WU_CPANEL_PASSWORD', 'your_cpanel_password');
define('WU_CPANEL_HOST', 'your_cpanel_host');
```

אָפּציאָנעל קענט איר צופּאַסן דעם פּאָרט און דעם וואָרצל־טעקע־קאַטאַלאָג:

```php
define('WU_CPANEL_PORT', 2083); // Change if your cPanel uses a different port
define('WU_CPANEL_ROOT_DIR', '/public_html'); // Change if your document root is different
```

### 3. אַקטיווירט די אינטעגראַציע

1. אין אײַער WordPress אַדמין, גייט צו Ultimate Multisite > אײַנשטעלונגען
2. נאַוויגירט צום "דאָמיין־מאַפּינג" קוויטל
3. זעמלט אַראָפּ צו "האָסט־אינטעגראַציעס"
4. אַקטיווירט די cPanel אינטעגראַציע
5. דריקט "היט ענדערונגען"

## ווי עס אַרבעט

### צוגעלייגטע דאָמיינען

ווען אַ דאָמיין ווערט פֿאַרבונדן אין Ultimate Multisite:

1. די אינטעגראַציע שיקט אַן אָנפֿרעג צו cPanel'ס API כּדי צוצולייגן דעם דאָמיין ווי אַ צוגעלייגטן דאָמיין
2. דער דאָמיין ווערט קאָנפֿיגורירט צו ווײַזן אויף אײַער וואָרצל־טעקע־קאַטאַלאָג
3. ווען אַ דאָמיין־פֿאַרבינדונג ווערט באַזייַטיקט, וועט די אינטעגראַציע באַזייַטיקן דעם צוגעלייגטן דאָמיין פֿון cPanel

### אונטערדאָמיינען

פֿאַר אונטערדאָמיין־מולטיסייט־אינסטאַלאַציעס, ווען אַ נײַע וועבזײַט ווערט געשאַפֿן:

1. די אינטעגראַציע נעמט אַרויס דעם אונטערדאָמיין־טייל פֿונעם גאַנצן דאָמיין
2. זי שיקט אַן אָנפֿרעג צו cPanel'ס API כּדי צוצולייגן דעם אונטערדאָמיין
3. דער אונטערדאָמיין ווערט קאָנפֿיגורירט צו ווײַזן אויף אײַער וואָרצל־טעקע־קאַטאַלאָג

## וויכטיקע באַמערקונגען

- די אינטעגראַציע ניצט cPanel'ס API2 כּדי צו קאָמוניקירן מיט אײַער cPanel Account
- אײַער cPanel Account מוז האָבן דערלויבענישן צוצולייגן צוגעלייגטע דאָמיינען און אונטערדאָמיינען
- עטלעכע האָסטינג־פּראָוויידערס קענען באַגרענעצן די צאָל צוגעלייגטע דאָמיינען אָדער אונטערדאָמיינען וואָס איר קענט שאַפֿן
- די אינטעגראַציע באַהאַנדלט נישט קיין DNS קאָנפֿיגוראַציע; איר דאַרפֿט נאָך אַלץ ווײַזן אײַערע דאָמיינען צו דער IP אַדרעס פֿון אײַער סערווער

## טראָבלשוטינג

### API פֿאַרבינדונג־פּראָבלעמען
- באַשטעטיקט אַז אײַער cPanel באַניצערנאָמען און פּאַראָל זענען ריכטיק
- קאָנטראָלירט אַז אײַער cPanel האָסט איז ריכטיק און צוטריטלעך
- מאַכט זיכער אַז אײַער cPanel Account האָט די נייטיקע דערלויבענישן
- פּרוּווט ניצן דעם גאַנצן URL פֿאַרן האָסט (למשל, `https://cpanel.yourdomain.com`)

### דאָמיין נישט צוגעלייגט
- קאָנטראָלירט די Ultimate Multisite לאָגס פֿאַר קיין טעות־מעלדונגען
- באַשטעטיקט אַז דער דאָמיין איז נאָך נישט שוין צוגעלייגט צו cPanel
- מאַכט זיכער אַז אײַער cPanel Account האָט נישט דערגרייכט זײַן שיעור פֿאַר צוגעלייגטע דאָמיינען אָדער אונטערדאָמיינען

### SSL סערטיפֿיקאַט־פּראָבלעמען
- די אינטעגראַציע באַהאַנדלט נישט אַרויסגעבן SSL סערטיפֿיקאַטן
- איר וועט דאַרפֿן ניצן cPanel'ס SSL/TLS געצייג אָדער AutoSSL פֿעיִקייט כּדי אַרויסצוגעבן SSL סערטיפֿיקאַטן פֿאַר אײַערע דאָמיינען
- אַלטערנאַטיוו קענט איר ניצן אַ דינסט ווי Let's Encrypt מיט cPanel'ס AutoSSL
