# WP Engine אינטעגראַציע {#wp-engine-integration}

## איבערבליק {#overview}
WP Engine איז אַ פּרעמיום־פֿאַרוואַלטעטע WordPress האָסטינג־פּלאַטפֿאָרמע וואָס גיט אָפּטימיזירטע אויפֿפֿירונג, זיכערהייט, און סקאַלירבאַרקייט פֿאַר WordPress זייטלעך. די אינטעגראַציע דערמעגלעכט אויטאָמאַטישע דאָמען־סינכראָניזאַציע צווישן Ultimate Multisite און WP Engine.

## פֿעיִקייטן {#features}
- אויטאָמאַטישע דאָמען־סינכראָניזאַציע
- שטיצע פֿאַר סובדאָמען בײַ מולטיסייט־אינסטאַלאַציעס
- גלאַטע אינטעגראַציע מיט WP Engine'ס עקזיסטירנדיקע סיסטעמען

## פֿאָדערונגען {#requirements}
די אינטעגראַציע דערקענט אויטאָמאַטיש צי איר האָסט אויף WP Engine און ניצט דעם איינגעבויטן WP Engine API. קיין צוגעלייגטע קאָנפֿיגוראַציע איז נישט נייטיק אויב דער WP Engine plugin איז אַקטיוו און ריכטיק קאָנפֿיגורירט.

אָבער, אויב איר דאַרפֿט מאַנועל קאָנפֿיגורירן די אינטעגראַציע, קענט איר דעפֿינירן איינע פֿון די קאָנסטאַנטן אין אייער `wp-config.php` טעקע:

```php
define('WPE_APIKEY', 'your_api_key'); // Preferred method
// OR
define('WPE_API', 'your_api_key'); // Alternative method
```

## אויפֿשטעל־אינסטרוקציעס {#setup-instructions}

### 1. באַשטעטיקט דעם WP Engine Plugin {#1-verify-wp-engine-plugin}

אויב איר האָסט אויף WP Engine, זאָל דער WP Engine plugin שוין זײַן אינסטאַלירט און אַקטיווירט. באַשטעטיקט אַז:

1. דער WP Engine plugin איז אַקטיוו
2. די טעקע `wp-content/mu-plugins/wpengine-common/class-wpeapi.php` עקזיסטירט

### 2. אַקטיווירט די אינטעגראַציע {#2-enable-the-integration}

1. אין אייער WordPress אַדמין, גייט צו Ultimate Multisite > Settings
2. גייט צו דעם "Domain Mapping" קוויטל
3. ראָלט אַראָפּ צו "Host Integrations"
4. אַקטיווירט די WP Engine אינטעגראַציע
5. קליקט "Save Changes"

## ווי עס אַרבעט {#how-it-works}

### דאָמען־סינכראָניזאַציע {#domain-syncing}

ווען אַ דאָמען ווערט מאַפּט אין Ultimate Multisite:

1. די אינטעגראַציע ניצט דעם WP Engine API כּדי צוצולייגן דעם דאָמען צו אייער WP Engine אינסטאַלאַציע
2. WP Engine באַהאַנדלט די דאָמען־קאָנפֿיגוראַציע און SSL צערטיפֿיקאַט־אַרויסגעבן
3. ווען אַ דאָמען־מאַפּינג ווערט אַוועקגענומען, וועט די אינטעגראַציע אַוועקנעמען דעם דאָמען פֿון WP Engine

### סובדאָמען־שטיצע {#subdomain-support}

פֿאַר סובדאָמען מולטיסייט־אינסטאַלאַציעס:

1. די אינטעגראַציע לייגט צו יעדן סובדאָמען צו WP Engine ווען אַ נײַער זייטל ווערט באַשאַפֿן
2. WP Engine באַהאַנדלט די סובדאָמען־קאָנפֿיגוראַציע
3. ווען אַ זייטל ווערט אויסגעמעקט, וועט די אינטעגראַציע אַוועקנעמען דעם סובדאָמען פֿון WP Engine

## וויכטיקע באַמערקונגען {#important-notes}

### Wildcard דאָמענס {#wildcard-domains}

פֿאַר סובדאָמען מולטיסייט־אינסטאַלאַציעס, איז רעקאָמענדירט זיך צו פֿאַרבינדן מיט WP Engine שטיצע כּדי צו בעטן אַ wildcard דאָמען־קאָנפֿיגוראַציע. דאָס דערמעגלעכט אַלע סובדאָמען צו אַרבעטן אויטאָמאַטיש אָן דאַרפֿן צוצולייגן יעדן איינעם באַזונדער.

### SSL צערטיפֿיקאַטן {#ssl-certificates}

WP Engine באַהאַנדלט אויטאָמאַטיש SSL צערטיפֿיקאַט־אַרויסגעבן און באַנײַונג פֿאַר אַלע דאָמענס וואָס ווערן צוגעלייגט דורך דער אינטעגראַציע. קיין צוגעלייגטע קאָנפֿיגוראַציע איז נישט נייטיק.

## פּראָבלעם־לייזונג {#troubleshooting}

### API פֿאַרבינדונג־פּראָבלעמען {#api-connection-issues}
- באַשטעטיקט אַז דער WP Engine plugin איז אַקטיוו און ריכטיק קאָנפֿיגורירט
- אויב איר האָט מאַנועל דעפֿינירט דעם API שליסל, קאָנטראָלירט אַז ער איז ריכטיק
- פֿאַרבינדט זיך מיט WP Engine שטיצע אויב איר האָט שוועריקייטן מיטן API

### דאָמען נישט צוגעלייגט {#domain-not-added}
- קאָנטראָלירט די Ultimate Multisite לאָגס פֿאַר סיי וועלכע טעות־מעלדונגען
- באַשטעטיקט אַז דער דאָמען איז נישט שוין צוגעלייגט צו WP Engine
- מאַכט זיכער אַז אייער WP Engine פּלאַן שטיצט די צאָל דאָמענס וואָס איר לייגט צו

### סובדאָמען־פּראָבלעמען {#subdomain-issues}
- אויב סובדאָמען אַרבעטן נישט, פֿאַרבינדט זיך מיט WP Engine שטיצע כּדי צו בעטן אַ wildcard דאָמען־קאָנפֿיגוראַציע
- באַשטעטיקט אַז אייערע DNS אײַנשטעלונגען זענען ריכטיק קאָנפֿיגורירט פֿאַרן הויפּט־דאָמען און סובדאָמען
