# Ultimate Multisite פֿאָדערונגען {#ultimate-multisite-requirements}

די ערשטע זאַך וואָס איר קענט זיך פֿרעגן איידער איר קויפֿט Ultimate Multisite איז וואָס זענען די פֿאָדערונגען כּדי צו לויפֿן אונדזער plugin. נו, צום גליק, זענען דאָ בלויז אַ פּאָר גרונטלעכע פֿאָדערונגען פֿאַרן לויפֿן Ultimate Multisite: אַ **domain name** , אַ גוטער **hosting provider** און אַ WordPress **Multisite** אינסטאַלאַציע. דאָס איז אַלץ וואָס איר דאַרפֿט!

## Domain Name {#domain-name}

ערשטנס, לאָמיר רעדן וועגן דעם domain name.

ווען איר קלײַבט אַ domain name, האַלט אין זינען וואָס איז אײַער ספּעציפֿישע נישע און וואָס איר וועט שאַפֿן מיט Ultimate Multisite: ווילט איר שאַפֿן אַן אַגענטור? פֿאַרקויפֿן קראָם־ענלעכע וועבזײַטן? אָדער אפֿשר אַ WordPress וועב־hosting? אַלץ איז מעגלעך מיט Ultimate Multisite.

אונדזער plugin שטעלט צו אַבאָנעמענט־דינסטן, צאָלונג־אײַנזאַמלונג, checkout פֿאָרמען, אַראָפּרעכענונג־קופּאָנען און קונה־קאָמוניקאַציעס. אַ גרויסער טייל פֿון דער אַרבעט וואָס איז נייטיק צו אינסטאַלירן, קאָנפֿיגורירן און אונטערהאַלטן אַ WordPress Multisite ווערט פֿאַרגרינגערט דורך Ultimate Multisite.

יעדע נישע וואָס איר קלײַבט קען ווערן פֿאַרוואַלטעט ניצנדיק אונדזער plugin, אַזוי לאַנג ווי איר פֿאַרקויפֿט וועבזײַטן ווי אַ דינסט! מאַכט זיכער צו קלײַבן אַ domain name וואָס פּאַסט צו אײַער געשעפֿט־פּלאַן.

## Hosting Provider {#hosting-provider}

איצט אַז איר האָט אַ צוציענדיקן domain name וואָס רעפּרעזענטירט אײַער נישע און געשעפֿט, וועט איר דאַרפֿן אויסקלײַבן אַ גוטע וועב־hosting פֿירמע צו hostן אײַער נעץ מיט Ultimate Multisite. עס זענען דאָ עטלעכע גוטע אָפּציעס אויפֿן מאַרק. עטלעכע hosting providers אינטעגרירן זיך גוט מיט Ultimate Multisite, וואָס קען זײַן אַ באַשליסנדיקער פֿאַקטאָר, לויט דעם וואָס איר פּלאַנירט צו בויען.

Ultimate Multisite זאָל אַרבעטן בײַ יעדן hosting provider. דער איינציקער חילוק איז אַז עטלעכע hosting פֿירמעס ווי _WPMU DEV Hosting_ , _WP Engine_ , _Closte_ , _Cloudways_ , _Gridpane_ און _Runcloud_ האָבן אַ טייטערער אינטעגראַציע פֿאַר domain mapping. פֿאַר אַנדער hosting, קענט איר דאַרפֿן צו mapן די custom domains מאַנועל. עס איז אויך אַ גוטער געדאַנק צו באַקומען אַ hosting provider וואָס שטיצט AutoSSL. _Closte_ , _Runcloud_ און _cPanel_ שטיצן עס.

ווען עס קומט צו server ספּעציפֿיקאַציעס, איז נישטאָ קיין ספּעציפֿישע פֿאָרמולע, ווײַל עס וועט אָפענגען אויף די אייגנשאַפֿטן און פֿונקציעס וואָס איר וועט אָנבייטן און אויף דער גרייס פֿון אײַער נעץ. Shared hosting קען זײַן אַ גוטער אָנהייב, אָבער ווי נאָר אײַער נעץ הייבט אָן וואַקסן, וועט איר מסתּמא דאַרפֿן אַ dedicated VPS.

## WordPress Multisite {#wordpress-multisite}

איצט לאָמיר אַ ביסל רעדן וועגן אַ WordPress Multisite אינסטאַלאַציע.

WordPress Multisite דערלויבט אײַך צו האָבן אַ נעץ פֿון וועבזײַטן אויף איין איינציקער אינסטאַלאַציע. דאָס איז אַן אײַנגעבויטע אייגנשאַפֿט, אָבער זי איז נישט אַקטיוו לויטן סטאַנדאַרט. איר דאַרפֿט אַקטיווירן די אייגנשאַפֿט אינעווייניק אין אײַער WordPress אינסטאַלאַציע און וועב־hosting קאָנטראָל־פּאַנעל.

צו האָבן WordPress מיט Multisite אַקטיווירט **איז דער יסוד פֿאַר Ultimate Multisite**. ווײַל דער הויפּט־נוצן פֿון Ultimate Multisite איז צו העלפֿן באַניצער פֿאַרוואַלטן און פֿאַרקויפֿן וועבזײַטן ווי אַ דינסט, דאַרף דאָס געטאָן ווערן אויף WordPress מיט Multisite אַקטיווירט.

מיר האָבן **אַ טיף־גייענדיקן אַרטיקל** וואָס לערנט שריט נאָך שריט וואָס דאַרף געטאָן ווערן כּדי צו אַקטיווירן די Multisite אייגנשאַפֿט אויף אײַער WordPress אינסטאַלאַציע.

נאָך דעם וואָס איר האָט Multisite אַקטיווירט אויף WordPress, איז צייט צו אינסטאַלירן Ultimate Multisite. כּדי ריכטיק צו אינסטאַלירן Ultimate Multisite, מוזט איר האָבן:

  * WordPress: v5.3+ (רעקאָמענדירט: לעצטע סטאַבילע ווערסיע)

  * PHP: 7.4.x (שטיצע פֿאַר > 8.0 קומט באַלד)

  * MySQL: v5+ (רעקאָמענדירט: 5.6, אין פאַל איר האָט נישט 8.0 בנימצא בײַ אײַער hosting provider)

דאָס זענען די איינציקע software פֿאָדערונגען פֿאַר Ultimate Multisite.

האַלט אין זינען אַז Ultimate Multisite קען אַרבעטן מיט PHP 8.0, אָבער מיר רעקאָמענדירן צו לויפֿן עס מיט PHP 7.4.x.

אויך רעקאָמענדירן מיר עס צו אינסטאַלירן אויף **main domains** , נישט subdomains אָדער subdirectories. עס קען אַרבעטן אויף subdomains/subdirectories, אָבער עס קען ווײַזן עטלעכע טעותים.

פֿאַר אַ דעטאַלירטן וועגווײַזער ווי אַזוי צו אינסטאַלירן Ultimate Multisite, **דריקט דאָ**.
