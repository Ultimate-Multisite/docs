# Theme Builder אַרײַנפֿיר־פֿלוס

Superdav AI Agent v1.12.0 פֿירט אַרײַן אַ געפֿירטן **Theme Builder אַרײַנפֿיר־פֿלוס** וואָס העלפֿט אײַך שאַפֿן אַ קאַסטאַם בלאָק־טעמע בעת אײַער ערשט־אײַנשטעלונג. דאָס פֿאַרבײַט דעם אַלטן Site Builder מאָדוס מיט אַ מער בייגזאַמען, אַגענט־באַגלייטן צוגאַנג.

## וואָס איז דער Theme Builder אַרײַנפֿיר־פֿלוס? {#what-is-the-theme-builder-onboarding-flow}

דער Theme Builder אַרײַנפֿיר־פֿלוס איז אַן אינטעראַקטיווער אײַנשטעל־וויזאַרד וואָס:

- פֿירט אײַך דורך פּלאַן־באַשלוסן (קאָלירן, טיפּאָגראַפֿיע, אויסלייג)
- כאַפּט אײַערע פּרעפֿערענצן פֿאַר דער וויזועלער אידענטיטעט פֿון אײַער וועבזײַטל
- גענערירט אַ קאַסטאַם בלאָק־טעמע צוגעפּאַסט צו אײַערע באַדערפֿענישן
- אַקטיווירט די טעמע אויטאָמאַטיש ווען עס איז פֿאַרענדיקט

דער פֿלוס ווערט געטריבן דורך דעם **Setup Assistant agent**, וואָס פֿרעגט דערקלערנדיקע פֿראַגעס און בויט אײַער טעמע שטאַפּלווייז.

## אָנהייבן דעם Theme Builder אַרײַנפֿיר {#starting-the-theme-builder-onboarding}

### ערשט־לויף אײַנשטעלונג {#first-run-setup}

ווען איר לאָנטשט צום ערשטן מאָל Superdav AI Agent אויף אַ נײַער WordPress אינסטאַלאַציע, וועט איר זען:

```
Welcome to Superdav AI Agent!

What would you like to do?
1. Build a custom theme (Theme Builder)
2. Use an existing theme
3. Skip setup for now
```

קלײַבט אויס **"בויען אַ קאַסטאַם טעמע"** כּדי אַרײַנצוגיין אין דעם Theme Builder אַרײַנפֿיר־פֿלוס.

### מאַנועלע אַקטיוואַציע {#manual-activation}

איר קענט אויך אָנהייבן דעם Theme Builder אַרײַנפֿיר אין יעדן מאָמענט דורך בעטן:

```
"Start the Theme Builder onboarding"
```

אָדער

```
"Help me create a custom theme"
```

## די אַרײַנפֿיר־טריט {#the-onboarding-steps}

### טריט 1: מאָדוס־אויסקלייב {#step-1-mode-selection}

דער Setup Assistant agent פֿרעגט וועגן אײַער פּרעפֿערענץ:

```
How would you like to build your theme?
- Guided (I'll ask questions and build it for you)
- Hands-on (I'll show you options and you decide)
```

**געפֿירטער מאָדוס** איז רעקאָמענדירט פֿאַר רובֿ באַניצער; דער אַגענט מאַכט פּלאַן־רעקאָמענדאַציעס באַזירט אויף אײַער אינדוסטריע און צילן.

### טריט 2: וועבזײַטל־ספּעציפֿיקאַציע {#step-2-site-specification}

מען וועט אײַך פֿרעגן וועגן אײַער וועבזײַטל:

- **וועבזײַטל־ציל**: עלעקטראָנישער האַנדל, בלאָג, פּאָרטפֿאָליאָ, SaaS, אאַז״וו.
- **ציל־עולם**: ווער זענען אײַערע באַזוכער?
- **בראַנד־קאָלירן**: הויפּט־ און צווייטיקע קאָלירן (אָדער "איך בין נישט זיכער")
- **טאָן**: פּראָפֿעסיאָנעל, שעפֿעריש, שפּילעוודיק, מינימאַל, אאַז״וו.

די אינפֿאָרמאַציע ווערט אויפֿגעהיט אין אײַער **site_brief** זכּרון, וואָס אַגענטן רעפֿערירן צו אין צוקונפֿטיקע סעסיעס.

### טריט 3: באַשלוסן פֿאַרן פּלאַן־סיסטעם {#step-3-design-system-decisions}

דער אַגענט פֿירט אײַך דורך אויסקלייבן פּלאַן־טאָקענס:

- **טיפּאָגראַפֿיע**: שריפֿט־משפּחה (סעריף, אָן־סעריף, מאָנאָספּייס) און גרייס־סקאַלע
- **קאָליר־פּאַלעטע**: הויפּט־, צווייטיקע, אַקצענט־ און נייטראַלע קאָלירן
- **אָפּשטאַנד**: קאָמפּאַקטע, נאָרמאַלע, אָדער ברייטע אויסלייגן
- **באַוועגונג**: אַנימאַציעס און איבערגאַנגען (אויב געוואָלט)

### טריט 4: טעמע־גענערירונג {#step-4-theme-generation}

דער Setup Assistant agent סקאַפֿאָלדט אײַער קאַסטאַם בלאָק־טעמע מיט:

- `theme.json` וואָס אַנטהאַלט אַלע אײַערע פּלאַן־טאָקענס
- בלאָק־מוסטערן פֿאַר געוויינטלעכע אויסלייגן (היימזײַט, בלאָג, קאָנטאַקט)
- קאַסטאַם בלאָק־סטילן וואָס פּאַסן צו אײַער פּלאַן־סיסטעם
- טעמע־מעטאַדאַטן און WordPress שטיצע־דעקלאַראַציעס

### טריט 5: אַקטיוואַציע און באַשטעטיקונג {#step-5-activation-and-verification}

די טעמע ווערט אויטאָמאַטיש אַקטיווירט, און איר וועט זען:

```
✓ Your custom theme is now live!
  Theme name: [Your Site Name] Theme
  Colors: [Primary] / [Secondary]
  Typography: [Font Family]

  Visit your site to see the new design.
```

דערנאָך קענט איר באַזוכן אײַער וועבזײַטל כּדי צו באַשטעטיקן אַז די טעמע ווײַזט זיך ריכטיק.

## וועבזײַטל־ספּעציפֿיקאַציע און site_brief זכּרון {#site-specification-and-sitebrief-memory}

בעת דעם אַרײַנפֿיר כאַפּט דער אַגענט די ספּעציפֿיקאַציע פֿון אײַער וועבזײַטל אין אַ **site_brief** זכּרון־קאַטעגאָריע. דאָס נעמט אַרײַן:

- וועבזײַטל־ציל און צילן
- ציל־עולם
- בראַנד־קאָלירן און טאָן
- פּלאַן־פּרעפֿערענצן
- אינהאַלט־סטרוקטור

### פֿאַר וואָס site_brief איז וויכטיק {#why-sitebrief-matters}

אין צוקונפֿטיקע סעסיעס רעפֿערירן אַגענטן צו אײַער site_brief כּדי צו:

- אָנהאַלטן פּלאַן־קאָנסיסטענץ איבער ענדערונגען
- פֿאָרשלאָגן פֿעיִקייטן וואָס שטימען מיטן ציל פֿון אײַער וועבזײַטל
- צושטעלן קאָנטעקסט־באַוווּסטע רעקאָמענדאַציעס
- אויסמײַדן איבערחזרן אײַנשטעל־פֿראַגעס

### זען אײַער site_brief {#viewing-your-sitebrief}

איר קענט פֿרעגן דעם אַגענט:

```
"Show me my site brief"
```

אָדער

```
"What do you know about my site?"
```

דער אַגענט וועט ווײַזן אײַער אויפֿגעהיטענע וועבזײַטל־ספּעציפֿיקאַציע.

## קאַסטאַמיזירן נאָך דעם אַרײַנפֿיר {#customizing-after-onboarding}

נאָך דעם ווי דער Theme Builder אַרײַנפֿיר ווערט פֿאַרענדיקט, קענט איר:

### נוצן די Design System Aesthetics פֿעיִקייט {#use-the-design-system-aesthetics-skill}

בעט פּלאַן־פֿאַרפֿײַנערונגען:

```
"Refine the typography to be more modern"
```

אָדער

```
"Adjust the color palette to be warmer"
```

די **Design System Aesthetics פֿעיִקייט** פֿירט אײַך דורך געצילטע פּלאַן־דערהײַנטיקונגען.

### רעדאַקטירן theme.json דירעקט {#edit-themejson-directly}

פֿאַר אַוואַנסירטע באַניצער, רעדאַקטירט `/wp-content/themes/[theme-name]/theme.json` כּדי צוצופּאַסן:

- קאָליר־טאָקענס
- טיפּאָגראַפֿיע־סקאַלעס
- אָפּשטאַנד־ווערטן
- גרענעץ־ און שאָטן־דעפֿיניציעס

### שאַפֿן קאַסטאַם בלאָק־מוסטערן {#create-custom-block-templates}

נוצט דעם WordPress בלאָק־רעדאַקטאָר כּדי צו שאַפֿן קאַסטאַם מוסטערן פֿאַר:

- היימזײַט־אויסלייגן
- בלאָג־פּאָסט־זײַטן
- פּראָדוקט־זײַטן
- קאָנטאַקט־פֿאָרמען

## פֿאַרגלײַך: אַלטער קעגן נײַער אַרײַנפֿיר {#comparison-old-vs-new-onboarding}

| פֿעיִקייט | Site Builder (אַלט) | Theme Builder (נײַ) |
|---------|----------------------|-------------------|
| אײַנשטעל־מעטאָד | וויזאַרד־באַזירטע פֿאָרעם | אַגענט־געפֿירטע שמועס |
| טעמע־גענערירונג | באַגרענעצטע מוסטערן | קאַסטאַם סקאַפֿאָלדינג |
| פּלאַן־טאָקענס | מאַנועלע אַרײַנפֿיר | געפֿירטע באַשלוסן |
| בייגזאַמקייט | פֿיקסירטע אָפּציעס | קאַסטאַמיזירבאַר |
| צוקונפֿטיקע דערהײַנטיקונגען | נישט רעפֿערירט | אויפֿגעהיטן אין site_brief |

## פּראָבלעם־לייזונג {#troubleshooting}

**דער אַרײַנפֿיר־פֿלוס האָט זיך נישט פֿאַרענדיקט**
- אָנהייבט דעם פֿלוס ווידער: "Start the Theme Builder onboarding"
- קאָנטראָלירט אַז אײַער WordPress אינסטאַלאַציע איז דערהײַנטיקט
- באַשטעטיקט אַז דער Setup Assistant agent איז אַקטיווירט

**מײַן site_brief ווערט נישט גענוצט**
- באַשטעטיקט אַז דער אַגענט האָט צוטריט צו זכּרון
- בעט דעם אַגענט צו "recall my site brief"
- קאָנטראָלירט אַז זכּרון איז אַקטיווירט אין אײַערע אײַנשטעלונגען

**די גענערירטע טעמע פּאַסט נישט צו מײַנע פּרעפֿערענצן**
- ניצט די Design System Aesthetics פֿעיִקייט כּדי זי צו פֿאַרפֿײַנערן
- בעט דעם אַגענט צו "regenerate the theme with [specific changes]"
- רעדאַקטירט theme.json דירעקט פֿאַר פּינקטלעכער קאָנטראָל

## קומענדיקע טריט {#next-steps}

נאָכן פֿאַרענדיקן Theme Builder אַרײַנפֿיר:

1. **באַשטעטיקט אײַער וועבזײַטל**: באַזוכט אײַער וועבזײַטל כּדי צו זען די נײַע טעמע
2. **פֿאַרפֿײַנערט דעם פּלאַן**: ניצט Design System Aesthetics פֿעיִקייט פֿאַר צופּאַסונגען
3. **לייגט צו אינהאַלט**: הייבט אָן בויען דעם אינהאַלט פֿון אײַער וועבזײַטל
4. **דערפֿאָרשט פֿעיִקייטן**: לערנט זיך וועגן אַנדערע אַגענט־פֿעיִקייטן ווי scaffold-block-theme און activate-theme
