# מעלדונג וועגן באַזײַטיקן דעם וועבזײַטל־בויער־מאָדוס

**דער וועבזײַטל־בויער־מאָדוס איז אַוועקגענומען געוואָרן אין Superdav AI Agent v1.12.0.** אויב איר האָט געניצט דעם וועבזײַטל־בויער־מאָדוס, זאָלט איר מיגרירן צו דעם **Setup Assistant agent** פֿאַר טעמע־שאַפונג און וועבזײַטל־אויפֿשטעל.

## וואָס איז געשען? {#what-happened}

### וועבזײַטל־בויער־מאָדוס (אַלט) {#site-builder-mode-legacy}

דער וועבזײַטל־בויער־מאָדוס איז געווען אַ מכשף־באַזירטער צובינד פֿאַר:

- שאַפֿן וועבזײַטלעך פֿון מוסטערן
- קאָנפֿיגורירן יסודדיקע אײַנשטעלונגען
- אויסקלײַבן אַ טעמע
- אויפֿשטעלן אָנהייב־אינהאַלט

### וואָס האָט עס פֿאַרביטן? {#what-replaced-it}

דער **Setup Assistant agent** באַהאַנדלט איצט אַלע וועבזײַטל־בויער־פֿונקציאָנאַליטעט מיט:

- מער פֿלעקסיבלער, אַגענט־געפֿירטער אויפֿשטעל
- בעסערע אָפּציעס פֿאַר טעמע־צופּאַסונג
- אינטעגראַציע מיט Theme Builder אָנבאָרדינג
- שטענדיקער site_brief זכּרון פֿאַר צוקונפֿטיקע סעסיעס

## אויב איר האָט געניצט דעם וועבזײַטל־בויער־מאָדוס {#if-you-were-using-site-builder-mode}

### אײַערע וועבזײַטלעך זענען זיכער {#your-sites-are-safe}

- עקזיסטירנדיקע וועבזײַטלעך געשאַפֿן מיטן וועבזײַטל־בויער־מאָדוס וועלן ווײַטער אַרבעטן
- קיין דאַטן־פֿאַרלוסט אָדער וועבזײַטל־שטערונג
- איר קענט ווײַטער פֿאַרוואַלטן אײַערע וועבזײַטלעך נאָרמאַל

### מיגרירט צום Setup Assistant Agent {#migrate-to-setup-assistant-agent}

פֿאַר נײַעם וועבזײַטל־אויפֿשטעל אָדער טעמע־ענדערונגען, ניצט דעם Setup Assistant agent:

```
"העלף מיר אויפֿשטעלן אַ נײַ וועבזײַטל"
```

אָדער

```
"הייב אָן דעם Theme Builder אָנבאָרדינג"
```

דער Setup Assistant agent גיט די זעלבע פֿונקציאָנאַליטעט מיט מער פֿלעקסיביליטעט.

## פֿאַרגלײַך: וועבזײַטל־בויער קעגן Setup Assistant {#comparison-site-builder-vs-setup-assistant}

| פֿעיִקייט | וועבזײַטל־בויער (אַוועקגענומען) | Setup Assistant (נײַ) |
|---------|----------------------|----------------------|
| אויפֿשטעל־אופֿן | מכשף־פֿאָרעם | אַגענט־שמועס |
| טעמע־אויסקלײַב | פֿריִער באַשטימטע מוסטערן | אייגענע גענערירונג |
| צופּאַסונג | באַגרענעצטע אָפּציעס | פֿולער פּלאַן־סיסטעם |
| וועבזײַטל־קיצור | נישט אויפֿגעהיטן | שטענדיקער זכּרון |
| צוקונפֿטיקע סעסיעס | איבערחזרן אויפֿשטעל | ניצן אויפֿגעהיטענעם site_brief |
| פֿלעקסיביליטעט | פֿיקסירטער אַרבעטס־גאַנג | צופּאַסנדיקער שמועס |

## מיגרירן צום Setup Assistant Agent {#migrating-to-setup-assistant-agent}

### פֿאַר נײַע וועבזײַטלעך {#for-new-sites}

אַנשטאָט צו נוצן דעם וועבזײַטל־בויער־מאָדוס:

1. פֿרעגט: "העלף מיר אויפֿשטעלן אַ נײַ וועבזײַטל"
2. דער Setup Assistant agent וועט אײַך פֿירן דורך:
   - ציל און צוועקן פֿונעם וועבזײַטל
   - ציל־עולם
   - בראַנד־אידענטיטעט
   - טעמע־גענערירונג
   - אָנהייב־קאָנפֿיגוראַציע

### פֿאַר עקזיסטירנדיקע וועבזײַטלעך {#for-existing-sites}

אויב איר האָט אַן עקזיסטירנדיק וועבזײַטל פֿונעם וועבזײַטל־בויער־מאָדוס:

1. איר קענט עס ווײַטער נוצן ווי עס איז
2. כּדי צו דערהײַנטיקן די טעמע, פֿרעגט: "פּלאַן איבער מײַן וועבזײַטל"
3. דער Setup Assistant agent וועט אײַך העלפֿן שאַפֿן אַ נײַע טעמע
4. אײַערע וועבזײַטל־דאַטן בלײַבן אומגעענדערט

### פֿאַר טעמע־ענדערונגען {#for-theme-changes}

אַנשטאָט דער טעמע־אויסקלײַב פֿונעם וועבזײַטל־בויער־מאָדוס:

1. פֿרעגט: "ענדער מײַן טעמע"
2. דער Setup Assistant agent וועט:
   - פֿרעגן וועגן אײַערע פּלאַן־פֿאַרליבטן
   - גענערירן אַן אייגענע טעמע
   - אַקטיווירן זי אויף אײַער וועבזײַטל

## הויפּט־אונטערשיידן {#key-differences}

### וועבזײַטל־בויער־מאָדוס {#site-builder-mode}

```
1. קלײַבט אויס אַ מוסטער
2. קלײַבט אויס אַ טעמע
3. קאָנפֿיגורירט יסודדיקע אײַנשטעלונגען
4. פֿאַרטיק
```

### Setup Assistant Agent {#setup-assistant-agent}

```
1. באַשרײַבט דעם ציל פֿון אײַער וועבזײַטל
2. דעפֿינירט אײַער ציל־עולם
3. קלײַבט אויס פּלאַן־פֿאַרליבטן
4. דער אַגענט גענערירט אַן אייגענע טעמע
5. דער אַגענט אַקטיווירט די טעמע
6. דער וועבזײַטל־קיצור ווערט אויפֿגעהיטן פֿאַר צוקונפֿטיקע סעסיעס
```

## מעלות פֿונעם Setup Assistant Agent {#benefits-of-setup-assistant-agent}

### מער פֿלעקסיבל {#more-flexible}

- באַשרײַבט אײַער וועבזײַטל אין נאַטירלעכער שפּראַך
- באַקומט אייגענע רעקאָמענדאַציעס
- פּאַסט זיך צו צו אײַערע באַזונדערע באַדערפֿענישן

### בעסערע צופּאַסונג {#better-customization}

- אייגענע טעמע־גענערירונג
- באַשלוסן פֿון פּלאַן־סיסטעם
- שטענדיקע פּלאַן־טאָקענס

### שטענדיקער זכּרון {#persistent-memory}

- אײַער site_brief ווערט אויפֿגעהיטן
- צוקונפֿטיקע אַגענטן פֿאַרשטייען אײַער וועבזײַטל
- נישט נייטיק איבערצוחזרן אויפֿשטעל־אינפֿאָרמאַציע

### אינטעגרירטער אַרבעטס־גאַנג {#integrated-workflow}

- Theme Builder אָנבאָרדינג
- פּלאַן־סיסטעם־עסטעטיק־קענטעניש
- קאָנטראָלן פֿאַר פֿעיִקייט־זעבאַרקייט
- אַלץ אַרבעט צוזאַמען אָן שטערונגען

## פּראָבלעמען־לייזונג {#troubleshooting}

### איך קען נישט געפֿינען דעם וועבזײַטל־בויער־מאָדוס {#i-cant-find-site-builder-mode}

דער וועבזײַטל־בויער־מאָדוס איז אַוועקגענומען געוואָרן. ניצט אַנשטאָט דעם Setup Assistant agent:

```
"העלף מיר אויפֿשטעלן אַ נײַ וועבזײַטל"
```

### איך וויל ווידערשאַפֿן אַ וועבזײַטל פֿונעם וועבזײַטל־בויער {#i-want-to-recreate-a-site-from-site-builder}

איר קענט עס ווידערשאַפֿן מיטן Setup Assistant agent:

1. פֿרעגט: "העלף מיר אויפֿשטעלן אַ נײַ וועבזײַטל"
2. באַשרײַבט דעם ציל און פּלאַן פֿון אײַער אָריגינעלן וועבזײַטל
3. דער אַגענט וועט גענערירן אַן ענלעכע טעמע
4. אײַער site_brief וועט ווערן אויפֿגעהיטן פֿאַר צוקונפֿטיקן אָפּקלאַנג

### מײַן עקזיסטירנדיק וועבזײַטל פֿונעם וועבזײַטל־בויער אַרבעט נישט {#my-existing-site-builder-site-isnt-working}

עקזיסטירנדיקע וועבזײַטלעך געשאַפֿן מיטן וועבזײַטל־בויער־מאָדוס וועלן ווײַטער אַרבעטן. אויב איר דערלעבט פּראָבלעמען:

1. קאָנטראָלירט אַז אײַער טעמע איז נאָך אַקטיוו
2. באַשטעטיקט אַז אײַערע פּלוגינס זענען אַקטיווירט
3. קאָנטראָלירט WordPress גרײַז־לאָגס
4. פֿאַרבינדט זיך מיט שטיצע אויב די פּראָבלעמען בלײַבן

### קען איך נאָך נוצן מײַנע אַלטע וועבזײַטל־בויער־מוסטערן? {#can-i-still-use-my-old-site-builder-templates}

וועבזײַטל־בויער־מוסטערן זענען מער נישט פֿאַראַן. אָבער:

- אײַערע עקזיסטירנדיקע וועבזײַטלעך וועלן ווײַטער אַרבעטן
- איר קענט ווידערשאַפֿן ענלעכע וועבזײַטלעך מיטן Setup Assistant agent
- דער Setup Assistant agent גיט מער צופּאַסונג־אָפּציעס
