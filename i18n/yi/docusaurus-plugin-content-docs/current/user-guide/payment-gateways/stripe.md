# אויפֿשטעלן דעם Stripe צאָלונג־טויער (v2)

_**וויכטיקע באַמערקונג: דער אַרטיקל באַציט זיך צו Ultimate Multisite ווערסיע 2.x.**_

איר קענט אַקטיווירן ביז פֿיר צאָלונג־מעטאָדן אויף אונדזער צאָלונג־אײַנשטעלונגען־זײַט: Stripe, Stripe Checkout, PayPal און מאַנועל. אין דעם אַרטיקל וועלן מיר זען ווי אַזוי צו אינטעגרירן מיט **Stripe**.

## אַקטיווירן Stripe

כּדי צו אַקטיווירן Stripe ווי אַ פֿאַראַנענעם צאָלונג־טויער אויף אײַער נעצווערק, גייט צו **Ultimate Multisite > אײַנשטעלונגען > צאָלונגען** און צייכנט אָן דעם איבערשלאַלטער לעבן **Stripe** אָדער **Stripe Checkout** אין דער אָפּטיילונג פֿון אַקטיווע צאָלונג־טויערן.

![אַקטיווירן Stripe אין אַקטיווע צאָלונג־טויערן](/img/config/settings-payment-gateways.png)

### Stripe קעגן Stripe Checkout:

**Stripe:** דער מעטאָד וועט ווײַזן אַן אָרט אַרײַנצולייגן די קרעדיט־קאַרט־נומער בעת דער קויפֿ־אָפּשליסונג.

![Stripe אינליין קרעדיט־קאַרט־פֿעלד בעת דער קויפֿ־אָפּשליסונג](/img/config/settings-payment-gateways.png)

**Stripe Checkout:** דער מעטאָד וועט איבערפֿירן דעם קונה צו אַ Stripe Checkout־זײַט בעת דער קויפֿ־אָפּשליסונג.

![Stripe Checkout איבערפֿיר־זײַט בעת דער קויפֿ־אָפּשליסונג](/img/config/settings-payment-gateways.png)

באַקומען אײַערע Stripe API שליסלען

ווען Stripe איז אַקטיווירט ווי אַ צאָלונג־טויער, וועט איר דאַרפֿן אויספֿילן די פֿעלדער פֿאַר **Stripe פּובליקירבאַרער שליסל** און **Stripe געהיימער שליסל** . איר קענט דאָס באַקומען דורך זיך אַרײַנלאָגירן אין אײַער Stripe Account.

_**באַמערקונג:** איר קענט אַקטיווירן **זאַמדקאַסטן־מאָדוס** צו פּרוּוון צי דער צאָלונג־מעטאָד אַרבעט._

![Stripe API שליסל־פֿעלדער און זאַמדקאַסטן־מאָדוס איבערשלאַלטער](/img/config/settings-payment-gateways.png)

אויף אײַער Stripe Dashboard, קליקט **אַנטוויקלער** אין דער אויבערשטער־רעכטער ווינקל, און דערנאָך **API שליסלען** אין דער לינקער מעניו.

![Stripe Dashboard אַנטוויקלער־אָפּטיילונג מיט API שליסלען](/img/config/settings-payment-gateways.png)

איר קענט אָדער נוצן **פּרוּוו־דאַטן** (כּדי צו פּרוּוון צי די אינטעגראַציע אַרבעט אויף אײַער פּראָדוקציע־זײַט) אָדער נישט. צו טוישן דאָס, באַוועגט דעם **ווײַזן פּרוּוו־דאַטן** איבערשלאַלטער.

![Stripe ווײַזן פּרוּוו־דאַטן איבערשלאַלטער](/img/config/settings-payment-gateways.png)

קאָפּירט דעם ווערט פֿון דעם **פּובליקירבאַרן שליסל** און **געהיימען שליסל** , פֿון דער **Token** זייַַל און קלעפּט עס אַרײַן אין די Ultimate Multisite Stripe צאָלונג־טויער־פֿעלדער. דערנאָך קליקט אויף **אויפֿהיטן ענדערונגען**.

![Stripe פּובליקירבאַרע און געהיימע שליסל־ווערטן](/img/config/settings-payment-gateways.png)

![אַרײַנקלעפּן Stripe שליסלען אין Ultimate Multisite אײַנשטעלונגען](/img/config/settings-payment-gateways.png)

## אויפֿשטעלן Stripe Webhook

Stripe שיקט Webhook־געשעענישן וואָס מעלדן Ultimate Multisite יעדעס מאָל ווען אַ געשעעניש פּאַסירט אויף **אײַער Stripe Account**.

קליקט **אַנטוויקלער** און דערנאָך קלײַבט אויס דעם **Webhooks** פּונקט אין דער לינקער מעניו. דערנאָך, אויף דער רעכטער זײַט, קליקט **צולייגן ענדפּונקט** *.*

![Stripe Webhooks־זײַט מיט צולייגן ענדפּונקט קנעפּל](/img/config/settings-payment-gateways.png)

איר וועט דאַרפֿן אַן **ענדפּונקט URL** *.* Ultimate Multisite גענערירט אויטאָמאַטיש דעם ענדפּונקט URL, וואָס איר קענט געפֿינען גלײַך אונטער דעם **Webhook צוהערער URL** פֿעלד אין דער **Ultimate Multisite Stripe צאָלונג־טויער** אָפּטיילונג_._

![Webhook צוהערער URL פֿעלד אין Stripe צאָלונג־טויער־אײַנשטעלונגען](/img/config/settings-payment-gateways.png)

**קאָפּירט** דעם ענדפּונקט URL און **קלעפּט** עס אַרײַן אין דעם Stripe **ענדפּונקט URL** פֿעלד.

![אַרײַנקלעפּן ענדפּונקט URL אין Stripe Webhook אויפֿשטעלונג](/img/config/settings-payment-gateways.png)

ווײַטער דאַרף מען אויסקלײַבן אַ **געשעעניש** *.* אונטער דער אָפּציע דאַרפֿט איר פּשוט אָנצייכענען דעם **אויסקלײַבן אַלע געשעענישן** קעסטל און קליקן אויף **צולייגן געשעענישן**. נאָך דעם קליקט **צולייגן ענדפּונקט** כּדי אויפֿצוהיטן די ענדערונגען.

![אויסקלײַבן אַלע געשעענישן און צולייגן דעם Stripe ענדפּונקט](/img/config/settings-payment-gateways.png)

דאָס איז אַלץ, אײַער Stripe צאָלונג־אינטעגראַציע איז פֿאַרענדיקט!
