# ווי קען איך צוקומען צו מײַן Account-זײַט צום ערשטן מאָל?

ברוכים־הבאים צו Ultimate Multisite!

באַקומט צוטריט צו אײַער Account-זײַט דורך דעם לינק וואָס איר האָט באַקומען אין אײַער אימעיל נאָך דעם קויפֿן אונדזער פּלוגין:

<!-- סקרינשאָט נישט בנימצא: Account-זײַט־אימעיל מיט צוטריט־לינק -->

ניצט דעם אימעיל וואָס איר האָט אָנגעגעבן בעת דעם קויף און דעם פּאַראָל וואָס איר האָט באַקומען אין אײַער אימעיל.

<!-- סקרינשאָט נישט בנימצא: אַרײַנלאָג־זײַט מיט אימעיל־ און פּאַראָל־פֿעלדער -->

ווײַל דאָס איז אײַער ערשטער מאָל זיך אַרײַנצולאָגן אין דעם Account, וועט מען אײַך בעטן צו טוישן אײַער פּאַראָל. טוישט עס און גיט אַ קוועטש אויף דעם _אויפֿהיטן_־קנעפּל.

<!-- סקרינשאָט נישט בנימצא: פּאַראָל־טוישן־פֿאָרעם מיט אויפֿהיטן־קנעפּל -->

אויף אײַער Account-זײַט קענט איר פֿאַרוואַלטן אַלע אײַערע Account-פּרטים, אַזאַ ווי רעכענונגען, באנײַונגען, אָפּזאָגונגען און אַ סך אַנדערע זאַכן.
