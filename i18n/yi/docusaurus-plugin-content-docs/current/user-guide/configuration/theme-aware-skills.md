# טעמע־באַוווּסטע פֿעיִקייטן

Superdav AI Agent v1.10.0 קומט מיט פֿיר נײַע טעמע־באַוווּסטע אײַנגעבויטע פֿעיִקייטן וואָס פּאַסן זיך אויטאָמאַטיש צו צו אײַער אַקטיווער WordPress טעמע. די פֿעיִקייטן צושטעלן ספּעציאַליזירטע אָנפֿירונג און מעגלעכקייטן צוגעפּאַסט צו דער אַרכיטעקטור און פֿונקציעס פֿון אײַער טעמע.

## וואָס זענען טעמע־באַוווּסטע פֿעיִקייטן?

טעמע־באַוווּסטע פֿעיִקייטן זענען פֿאָרויס־קאָנפֿיגורירטע וויסן־באַזעס און געצײַג־זאַמלונגען וואָס דער AI אַסיסטענט קלײַבט אויטאָמאַטיש אויס לויט דער WordPress טעמע וואָס איז איצט אַקטיוו אויף אײַער פּלאַץ. ווען איר בײַט טעמעס, ווערן די בנימצא פֿעיִקייטן פֿונעם אַסיסטענט אויטאָמאַטיש דערהײַנטיקט — קיין האַנטיקע קאָנפֿיגוראַציע איז נישט נייטיק.

יעדע פֿעיִקייט נעמט אַרײַן:

- **טעמע־ספּעציפֿישע דאָקומענטאַציע** — אָנפֿירונג וועגן ניצן און צופּאַסן די טעמע
- **בלאָק־ און מוסטער־רעפֿערענצן** — בנימצא בלאָקס, מוסטערן, און פּלאַן־אָפּציעס
- **צופּאַסונג־בײַשפּילן** — קאָד־שטיקלעך און קאָנפֿיגוראַציע־מוסטערן פֿאַר געוויינטלעכע אויפֿגאַבן
- **בעסטע פּראַקטיקעס** — רעקאָמענדאַציעס פֿאַר דער אַרכיטעקטור און אַרבעטספֿלוס פֿון דער טעמע

## בנימצא טעמע־באַוווּסטע פֿעיִקייטן

### בלאָק־טעמעס

**גילט פֿאַר:** טעמעס וואָס ניצן די WordPress בלאָק־באַזירטע (גאַנצע פּלאַץ־רעדאַקטירונג) אַרכיטעקטור.

די בלאָק־טעמעס פֿעיִקייט גיט אָנפֿירונג וועגן:

- שאַפֿן און רעדאַקטירן מוסטערבלעטער ניצנדיק דעם בלאָק־רעדאַקטאָר
- אַרבעטן מיט בלאָק־מוסטערן און ווידער־ניצלעכע בלאָקס
- צופּאַסן גלאבאַלע סטילן און theme.json אײַנשטעלונגען
- ניצן טעמע־בלאָקס און וואַריאַציעס
- בויען אייגענע בלאָק־מוסטערן פֿאַר אײַער פּלאַץ

**ווערט אויטאָמאַטיש אַקטיווירט ווען:** אײַער אַקטיווע טעמע איז אַ בלאָק־טעמע (שטיצט די `block-templates` פֿונקציע).

### קלאַסישע טעמעס

**גילט פֿאַר:** טראַדיציאָנעלע WordPress טעמעס וואָס ניצן PHP מוסטערבלעטער און דעם קלאַסישן רעדאַקטאָר.

די קלאַסישע טעמעס פֿעיִקייט גיט אָנפֿירונג וועגן:

- אַרבעטן מיט PHP מוסטערבלאַט־טעקעס און hooks
- צופּאַסן דעם אויסזען פֿון דער טעמע דורך דעם Customizer
- ניצן ווידזשעט־געביטן און זײַט־לעסטן
- ענדערן CSS און אַנטוויקלען אַ קינד־טעמע
- פֿאַרשטיין טעמע־היעראַרכיע און מוסטערבלאַט־צייכנס

**ווערט אויטאָמאַטיש אַקטיווירט ווען:** אײַער אַקטיווע טעמע איז אַ קלאַסישע (ניט־בלאָק) טעמע.

### Kadence Blocks

**גילט פֿאַר:** פּלאַץ וואָס ניצן דעם Kadence Blocks plugin פֿאַר אַוואַנסירטן בלאָק־באַזירטן פּלאַן.

די Kadence Blocks פֿעיִקייט גיט אָנפֿירונג וועגן:

- ניצן Kadence'ס אַוואַנסירטע בלאָק־ביבליאָטעק (Hero, Testimonials, Pricing, אאַז״ו)
- קאָנפֿיגורירן Kadence בלאָק־אײַנשטעלונגען און רעספּאָנסיווע אָפּציעס
- בויען אייגענע אויסלייגן מיט Kadence'ס גריד־ און קאַנטיינער־בלאָקס
- אינטעגרירן Kadence בלאָקס מיט אײַער טעמע
- אויסניצן Kadence'ס פּלאַן־סיסטעם און פֿאָרויס־אײַנשטעלונגען

**ווערט אויטאָמאַטיש אַקטיווירט ווען:** דער Kadence Blocks plugin איז אַקטיוו אויף אײַער פּלאַץ.

### Kadence Theme

**גילט פֿאַר:** פּלאַץ וואָס ניצן די Kadence טעמע פֿאַר בלאָק־באַזירטן פּלאַן און צופּאַסונג.

די Kadence Theme פֿעיִקייט גיט אָנפֿירונג וועגן:

- צופּאַסן די Kadence טעמע דורך גלאבאַלע סטילן און theme.json
- ניצן Kadence'ס אײַנגעבויטע בלאָק־מוסטערן און מוסטערבלעטער
- קאָנפֿיגורירן Kadence טעמע־אײַנשטעלונגען און אָפּציעס
- בויען אייגענע פּלאַנען מיט Kadence'ס פּלאַן־סיסטעם
- אינטעגרירן Kadence מיט פּאָפּולערע plugins און געצײַגן

**ווערט אויטאָמאַטיש אַקטיווירט ווען:** די Kadence טעמע איז אײַער אַקטיווע טעמע.

## ווי פֿעיִקייטן ווערן אויסגעקליבן

דער אַסיסטענט דערקענט אויטאָמאַטיש אײַער אַקטיווע טעמע און אינסטאַלירטע plugins בײַ יעדן אָנזאָג. אויב אַ פּאַסיקע טעמע־באַוווּסטע פֿעיִקייט איז בנימצא, ווערט זי אויטאָמאַטיש אַרײַנגעלאָדן אין דעם קאָנטעקסט פֿונעם אַסיסטענט. איר דאַרפֿט נישט האַנטיק אַקטיווירן אָדער בײַטן פֿעיִקייטן.

### מערערע פֿעיִקייטן

אויב מערערע פֿעיִקייטן גילטן פֿאַר אײַער פּלאַץ (למשל, אויב איר האָט אַקטיוו סײַ Kadence Blocks סײַ Kadence Theme), האָט דער אַסיסטענט צוטריט צו אַלע גילטיקע פֿעיִקייטן און קען זיך באַציען צו אָנפֿירונג פֿון יעדערער.

### בײַטן טעמעס

ווען איר בײַט אײַער אַקטיווע טעמע, ווערן די בנימצא פֿעיִקייטן פֿונעם אַסיסטענט אויטאָמאַטיש דערהײַנטיקט בײַם קומעדיקן אָנזאָג. למשל:

1. איר ניצט אַ בלאָק־טעמע מיט דער **בלאָק־טעמעס** פֿעיִקייט אַקטיוו.
2. איר בײַט צו אַ קלאַסישער טעמע.
3. בײַ אײַער קומעדיקן אָנזאָג ווערט די **קלאַסישע טעמעס** פֿעיִקייט אויטאָמאַטיש אַרײַנגעלאָדן, און די **בלאָק־טעמעס** פֿעיִקייט איז מער נישט בנימצא.
