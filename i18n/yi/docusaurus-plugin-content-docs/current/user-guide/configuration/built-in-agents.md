# אײַנגעבויטע Agents

Gratis AI Agent v1.9.0 קומט מיט פֿינף אײַנגעבויטע agents, יעדער איינער פֿאָרויס־קאָנפֿיגורירט מיט אַ פֿאָקוסירטן זאַמלונג פֿון געצײַגן, אַן צוגעפּאַסטן סיסטעם־פּראָמפּט, און אָנהייב־פֿאָרשלאָגן וואָס פּאַסן צו געוויינטלעכע אויפֿגאַבן אין יענעם געביט. איבערבײַטן צווישן agents ענדערט וואָס דער אַסיסטענט קען טאָן און ווי ער ענטפֿערט — אָן קיין קאָנפֿיגוראַציע פֿון אײַער זײַט. Superdav AI Agent v1.18.0 קען צולייגן פּלאַן־באַוווּסטע געצײַגן, דערמאָנונג־רעקאָרדן, באַשטעטיקונג־טויערן, און SMS נאָוטיפֿיקאַציעס צו די אַרבעט־פֿליסן ווען די פֿאַרבונדענע אינטעגראַציעס זענען קאָנפֿיגורירט.

## וואָס איז אַן Agent?

יעדער agent איז אַ באַנאַמטער קאָנפֿיגוראַציע־פּראָפֿיל וואָס קאָמבינירט:

- **געצײַגן** — די פֿעיִקייטן וואָס דער agent מעג אַרויסרופֿן (למשל, אַ אינהאַלט־שרײַבער האָט צוטריט צו פּאָסט־שאַפֿונג־פֿעיִקייטן; אַ דיזײַן־סטודיע האָט צוטריט צו CSS און theme.json פֿעיִקייטן)
- **סיסטעם־פּראָמפּט** — אינסטרוקציעס וואָס שטעלן דעם טאָן, פּריאָריטעטן, און באַגרענעצונגען פֿונעם agent
- **פֿאָרשלאָגן** — פֿאָרויס־געשריבענע פּראָמפּטן וואָס ווערן געוויזן אין דער שמועס־צובינד כּדי אײַך העלפֿן אָנהייבן שנעל

## צוטריט צום Agent Picker

1. עפֿנט דעם **Gratis AI Agent** פּאַנעל אין דער WordPress אַדמין־זײַטנבאַר.
2. קליקט דעם **agent בילדל** אויבן־לינקס אין דער שמועס־קעפּל (דאָס בילדל ענדערט זיך כּדי אָפּצושפּיגלען דעם אַקטיוון agent).
3. דער **Agent Picker** עפֿנט זיך ווי אַ פֿאָרעם־טאַבעלע איבערלייג. יעדער agent ווערט אויסגערעכנט מיט זײַן בילדל, נאָמען, און אַ איין־שורהדיקער באַשרײַבונג.
4. קליקט אַן agent־ריי כּדי אים אַקטיווירן. דער שמועס־קעפּל דערהײַנטיקט זיך גלײַך.

איר קענט אויך איבערבײַטן agents אין מיטן שמועס — דער סיסטעם־פּראָמפּט פֿונעם נײַעם agent טרעט אין קראַפֿט פֿון דער קומענדיקער מעלדונג.

## די פֿינף אײַנגעבויטע Agents

### אינהאַלט־שרײַבער

**פֿאָקוס:** שאַפֿן און רעדאַגירן פּאָסטן, זײַטן, און קאָנטאַקט־פֿאָרמען.

**פֿאַראַנענע געצײַגן:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`. מיט Superdav AI Agent v1.18.0 אינטעגראַציעס אַקטיווירט, קאָנפֿיגורירטער קאַלענדאַר־קאָנטעקסט, באַשטעטיקונג־טויערן, דערמאָנונגען, און SMS נאָוטיפֿיקאַציע־געצײַגן קענען אויך זײַן פֿאַראַן פֿאַר באשטעטיקטע אַרבעט־פֿליסן.

**וואָס עס טוט גוט:**
- אויסאַרבעטן און פּובליקירן בלאָג־פּאָסטן פֿון אַ קורצער באַשרײַבונג אָדער אויסריס
- שאַפֿן גרופּעס פֿון לאַנדונג־זײַטן פֿאַר אַ נײַעם וועבזײַטל
- בויען קאָנטאַקט־ און אָנפֿרעג־פֿאָרמען
- שטעלן אויסגעצייכנטע בילדער אויף פּאָסטן פֿון אַ URL אָדער זוך
- אויסאַרבעטן נאָכפֿאָלג־מעלדונגען פֿאַר געשעענישן פֿון קאָנפֿיגורירטן Google Calendar קאָנטעקסט, דערנאָך אָפּשטעלן פֿאַר באַשטעטיקונג איידער שיקן נאָוטיפֿיקאַציעס

**אָנהייב־פֿאָרשלאָגן:**
- *שרײַב אַ בלאָג־פּאָסט פֿון 500 ווערטער וועגן די מעלות פֿון WordPress מולטיסײַט.*
- *שאַף אַן וועגן־אונדז, באַדינונגען, און קאָנטאַקט־זײַט און פּובליקיר זיי.*
- *לייג צו אַ בוקינג־אָנפֿרעג־פֿאָרעם צו דער קאָנטאַקט־זײַט.*
- *מאַך אַ פּלאַן פֿון אַ דערמאָנונג פֿאַר באַטייליקטע בײַ דער מאָרגנדיקער קאָנפֿיגורירטער קאַלענדאַר־געשעעניש און וואַרט אויף באַשטעטיקונג איידער שיקן עס.*

### וועבזײַטל־בויער

**פֿאָקוס:** אָנהייב־ביז־סוף וועבזײַטל־שאַפֿונג פֿון איין פּראָמפּט.

**פֿאַראַנענע געצײַגן:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`. מיט Superdav AI Agent v1.18.0, קאָנפֿיגורירטע געפֿירטע־סערוויס, באַשטעטיקונג, דערמאָנונג, קאַלענדאַר, און SMS געצײַגן קענען זײַן פֿאַראַן וווּ אַדמיניסטראַטאָרן אַקטיווירן זיי.

**וואָס עס טוט גוט:**
- גענערירן אַ מער־פֿאַזיקן וועבזײַטל־בוי־פּלאַן פֿאַר אַ באַשריבן געשעפֿט־טיפּ
- דורכפֿירן יעדע פֿאַזע אויטאָנאָם — סטרוקטור, אינהאַלט, נאַוויגאַציע, דיזײַן
- זיך ערהוילן פֿון טעותן אין מיטן פּלאַן אָן דאַרפֿן מאַנועלע אַרײַנמישונג
- אינסטאַלירן רעקאָמענדירטע plugins ווי טייל פֿונעם בוי
- שאַפֿן קאָנטאַקט־פֿאָרמען גלײַך פֿון דער שמועס־צובינד (Superdav AI Agent v1.10.0+)
- קאָאָרדינירן לאָנטש־דערמאָנונגען אָדער נאָכפֿאָלג פֿאַר באַטייליקטע אָן טאָפּלטע נאָוטיפֿיקאַציעס ווען באַשטעטיקונג־טויערן און דערמאָנונג־רעקאָרדן זענען אַקטיווירט

**אָנהייב־פֿאָרשלאָגן:**
- *בוי אַ פֿאָטאָגראַפֿיע־פּאָרטפֿאָליאָ וועבזײַטל מיט אַ גאַלעריע־פּאָסט־טיפּ, אַ בוקינג־זײַט, און אַ קאָנטאַקט־פֿאָרעם.*
- *שאַף אַ רעסטאָראַן־וועבזײַטל מיט אַן אָנליין מעניו, עפֿענונג־שעות, און אַ טיש־בוקינג אָנפֿרעג־פֿאָרעם.*
- *שטעל אויף אַ פֿרילאַנס־קאָנסאַלטינג וועבזײַטל מיט באַדינונג־זײַטן, אַ פּאָרטפֿאָליאָ־אָפּטייל, און אַ בלאָג.*
- *לייג צו אַ קאָנטאַקט־פֿאָרעם צו דער קאָנטאַקט־זײַט ניצנדיק דעם וועבזײַטל־בויער.*
- *נאָך דעם וואָס די וועבזײַטל־לאָנטש קאָנטראָל־רשימה איז באשטעטיקט, שיק אַן SMS דערמאָנונג צום קאָנפֿיגורירטן באַטייליקטן קאָנטאַקט.*

### דיזײַן־סטודיע

**פֿאָקוס:** וויזועלע צופּאַסונג — פֿאַרבן, טיפֿאָגראַפֿיע, CSS, און בלאָק־מוסטערן.

**פֿאַראַנענע געצײַגן:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**וואָס עס טוט גוט:**
- אָנווענדן באַנאַמטע theme פּרעסעטן (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- פֿײַנשטעלן גלאבאַלע טיפֿאָגראַפֿיע און פֿאַרב־פּאַלעטן דורך theme.json
- אַרײַנשפּריצן קאַסטאַם CSS פֿאַר סאָרט־ספּעציפֿישע איבעררײַדונגען
- נעמען אַ סקרינשאָט פֿון אַ זײַט און איבערקוקן זי פֿאַר דיזײַן־פּראָבלעמען

**אָנהייב־פֿאָרשלאָגן:**
- *ווענד אָן דעם warm-editorial פּרעסעט און דערנאָך שטעל דעם הויפּט־קאָליר אויף #2d6a4f.*
- *נעם אַ סקרינשאָט פֿון דער הויפּטזײַט און זאָג מיר וואָס דו וואָלטסט פֿאַרבעסערט.*
- *שאַף אַ ווידערניצלעכן העלד־בלאָק־מוסטער מיט אַ פול־ברייט הינטערגרונט־בילד און אַ צענטרירטן קעפּל.*

### Plugin Manager

**פֿאָקוס:** אַנטדעקן, אינסטאַלירן, און פירן WordPress plugins.

**פֿאַראַנענע געצײַגן:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**וואָס עס טוט גוט:**
- רעקאָמענדירן דעם בעסטן plugin פֿאַר אַ באַשריבן ניצ־פֿאַל
- אינסטאַלירן פֿעיִקייט־פּאַקס פֿון דער רעגיסטרי
- דורכזוכן דעם פֿאַראַנענעם פֿעיִקייט־קאַטאַלאָג לויט קאַטעגאָריע

**אָנהייב־פֿאָרשלאָגן:**
- *וואָס איז דער בעסטער plugin פֿאַר אַ membership וועגווײַזער?*
- *אינסטאַליר דעם WooCommerce פֿעיִקייטן־פּאַק.*
- *ווײַז מיר אַלע פֿאַראַנענע עקאָמערס פֿעיִקייט־פּאַקס.*

### שטיץ־אַסיסטענט

**פֿאָקוס:** ענטפֿערן פֿראַגעס וועגן וועבזײַטל־אינהאַלט, סעטטינגס, און WordPress קאָנפֿיגוראַציע.

**פֿאַראַנענע געצײַגן:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**וואָס עס טוט גוט:**
- אויפֿזוכן איצטיקע וועבזײַטל־איינשטעלונגען און אָפּציעס
- דערקלערן וועלכע פּאָסט־טיפּן, טאַקסאָנאָמיעס, און מעניוען זענען קאָנפֿיגורירט אויפֿן וועבזײַטל
- ענטפֿערן אויף פֿראַגעס "וואָס טוט די דאָזיקע איינשטעלונג?" דורך לייענען לעבעדיקע ווערטן
- דינען ווי אַ נאָר־לייענען דיאַגנאָסטישע שיכט איידער מען מאַכט ענדערונגען

**אָנהייב־פֿאָרשלאָגן:**
- *וועלכע פּלוגינס און איינשטעלונגען זענען איצט אַקטיוו אויף דעם וועבזײַטל?*
- *ליסט אַלע מנהג־פּאָסט־טיפּן וואָס זענען רעגיסטרירט אויף דעם וועבזײַטל.*
- *וועלכע נאַוויגאַציע־מעניוען עקזיסטירן און וווּ זענען זיי צוגעטיילט?*

## Superdav אויטאָמאַציע־אינטעגראַציעס

ווען Superdav AI Agent v1.18.0 אינטעגראַציעס זענען קאָנפֿיגורירט, קענען אײַנגעבויטע אַגענטן אָנטייל נעמען אין זיכערערע פּלאַן־באַוווּסטע אויטאָמאַציע־אַרבעטסלויפֿן:

- **Google Calendar לייענען־געצײַגן** לאָזן אַגענטן איבערקוקן קאָנפֿיגורירטע קאַלענדאַרן און געשעענישן איידער זיי מאַכן אַ פּלאַן פֿון נאָכפֿאָלג־אַרבעט.

## קאַסטאַמיזירן אַגענטן
