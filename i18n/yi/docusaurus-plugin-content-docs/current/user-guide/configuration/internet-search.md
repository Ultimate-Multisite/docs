# אינטערנעץ־זוך {#internet-search}

Gratis AI Agent v1.5.0 לייגט צו אַן **אינטערנעץ־זוך** מעגלעכקייט וואָס לאָזט דעם AI אַסיסטענט אַראָפּנעמען דערהײַנטיקטע אינפֿאָרמאַציע פֿון דער וועב בעת אַ שמועס. דאָס ווערט געטריבן דורך דער [Brave Search API](https://brave.com/search/api/) אָדער דער [Tavily API](https://tavily.com/).

## ווי עס אַרבעט {#how-it-works}

ווען אינטערנעץ־זוך איז אַקטיווירט, קען דער אַסיסטענט אויטאָמאַטיש אָנפֿרעגן אײַער קאָנפֿיגורירטן זוך־צושטעלער ווען ער באַשליסט אַז אַ פֿראַגע דאַרף איצטיקע אָדער אויסערלעכע אינפֿאָרמאַציע — למשל, לעצטע נײַעס, לעבעדיקע פּרײַזן, אָדער דאָקומענטאַציע וואָס קען זיך האָבן געביטן זינט דעם טרענירונג־אָפּשניט פֿון דעם מאָדעל.

רעזולטאַטן ווערן אַראָפּגענומען אין אמתער צײַט און אַרײַנגעשטעלט אין דעם קאָנטעקסט פֿונעם אַסיסטענט איידער ער שאַפֿט אַן ענטפֿער. דער אַסיסטענט ווײַזט אָן ווען ער האָט גענוצט זוך־רעזולטאַטן כּדי צו ענטפֿערן אַ פֿראַגע.

## אַקטיווירן אינטערנעץ־זוך {#enabling-internet-search}

אינטערנעץ־זוך דאַרף אַן API־שליסל פֿון אײַער אויסגעקליבענעם זוך־צושטעלער. כּדי עס צו קאָנפֿיגורירן:

1. גייט צו **Gratis AI Agent → אײַנשטעלונגען → פֿאָרגעשריטן**.
2. געפֿינט דעם **אינטערנעץ־זוך־צושטעלער** אַראָפּפֿאַל־מעניו און קלײַבט אויס אָדער **Brave Search** אָדער **Tavily**.
3. גיט אַרײַן אײַער API־שליסל אין דעם פּאַסיקן פֿעלד. רעגיסטראַציע־לינקען ווערן געוויזן לעבן יעדן פֿעלד.
4. קליקט **אויפֿהיטן אײַנשטעלונגען**.

אַמאָל דער שליסל איז אויפֿגעהיטן, איז די אינטערנעץ־זוך מעגלעכקייט אויטאָמאַטיש בנימצא פֿאַרן אַסיסטענט.

## Brave Search {#brave-search}

### באַקומען אַ Brave Search API Key {#obtaining-a-brave-search-api-key}

1. באַזוכט די [Brave Search API זײַט](https://brave.com/search/api/).
2. רעגיסטרירט זיך פֿאַר אַ פּלאַן. אַ פֿרײַער ראַנג איז בנימצא מיט אַ חודשלעכן בעטן־גרענעץ.
3. קאָפּירט אײַער API־שליסל פֿון דעם Brave Search אַנטוויקלער־Dashboard.
4. קלעפּט עס אַרײַן אין דעם **Brave Search API Key** פֿעלד אין Gratis AI Agent אײַנשטעלונגען.

### באַניץ־גרענעצן {#usage-limits}

באַניץ ווערט אָפּגערעכנט דורך Brave Search לויט דער צאָל אָנפֿרעגן וואָס ווערן געמאַכט. יעדער AI ענטפֿער וואָס טריגערט אַ זוך ציילט זיך ווי איין אָנפֿרעג. קאָנטראָלירט אײַער באַניץ אין דעם [Brave Search אַנטוויקלער־Dashboard](https://brave.com/search/api/) כּדי אויסצומײַדן אומדערוואַרטע אָפּצאָלן.

## Tavily {#tavily}

Superdav AI Agent v1.10.0 לייגט צו **Tavily** ווי אַן אַלטערנאַטיווער אינטערנעץ־זוך־צושטעלער, וואָס אָפערט רײַכערע זוך־רעזולטאַטן און פֿאָרגעשריטענע פֿאָרשונג־מעגלעכקייטן.

### באַקומען אַ Tavily API־שליסל {#obtaining-a-tavily-api-key}

1. באַזוכט די [Tavily API זײַט](https://tavily.com/).
2. רעגיסטרירט זיך פֿאַר אַן Account. אַ פֿרײַער ראַנג איז בנימצא מיט אַ חודשלעכן בעטן־גרענעץ.
3. קאָפּירט אײַער API־שליסל פֿון דעם Tavily Dashboard.
4. קלעפּט עס אַרײַן אין דעם **Tavily API־שליסל** פֿעלד אין Gratis AI Agent אײַנשטעלונגען.

### באַניץ־גרענעצן {#usage-limits-1}

באַניץ ווערט אָפּגערעכנט דורך Tavily לויט דער צאָל API־רופֿן וואָס ווערן געמאַכט. יעדער AI ענטפֿער וואָס טריגערט אַ זוך ציילט זיך ווי איין רוף. קאָנטראָלירט אײַער באַניץ אין דעם [Tavily Dashboard](https://tavily.com/) כּדי אויסצומײַדן אומדערוואַרטע אָפּצאָלן.

## דיסאַקטיווירן אינטערנעץ־זוך {#disabling-internet-search}

נעמט אַרויס דעם API־שליסל פֿון דעם אַקטיוון זוך־צושטעלער־פֿעלד און היט אויף. די אינטערנעץ־זוך מעגלעכקייט וועט מער נישט ווערן געפֿינט פֿאַרן אַסיסטענט.

:::note
אינטערנעץ־זוך לייגט צו לאַטענץ צו ענטפֿערס, ווײַל דער אַסיסטענט מוז וואַרטן אויף זוך־רעזולטאַטן איידער ער שאַפֿט אַן ענטפֿער. פֿאַר צײַט־סענסיטיווע באַניץ־פֿאַלן, באַטראַכט צי אמת־צײַט־זוך איז נייטיק אָדער צי דעם אַסיסטענטס אײַנגעבויטע וויסן איז גענוג.
:::
