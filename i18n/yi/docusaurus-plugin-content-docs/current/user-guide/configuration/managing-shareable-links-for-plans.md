# אָנפירן צעטיילבאַרע לינקען פֿאַר פּלענער {#managing-shareable-links-for-plans}

Ultimate Multisite אָפערט אײַך די אָפּציע צו טיילן פּלאַן־לינקען דירעקט מיט אײַערע קליענטן. כּדי דאָס צו טאָן, גייט צו איינע פֿון די רעדאַגיר־בלעטער פֿון אײַערע פּלענער און קוועטשט דעם _קוועטשט צו קאָפּירן דעם צעטיילבאַרן לינק_ קנעפּל.

![פּראָדוקט־רעדאַגיר־בלאַט מיט צעטיילבאַרן לינק־קנעפּל](/img/config/product-edit-full.png)

_אויף דעם רעדאַגיר־בלאַט פֿונעם פּלאַן קענט איר געפֿינען דעם קנעפּל צו שאַפֿן צעטיילבאַרע לינקען_

באַמערקט אַז דער דאָזיקער מעכאַניזם ניצט די פעליקייט־בילינג־אָפֿטקייַט צו בויען דעם לינק. דאָס מיינט אַז אויב אײַער פעליקייט־בילינג איז חודשלעך, צום בײַשפּיל, צום סוף וועט איר האָבן אַ לינק ווי דאָס: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

אויב איר ווילט אַז דער צעטיילבאַרער לינק זאָל דורך פעליקייט זיין קוואַרטאַללעך, וועט איר מוזן צולייגן 3 צום סוף

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

צו אויסקלײַבן דעם פּלאַן _premium_ אויף אַ יאָרלעכן פּלאַן, וואָלט איר צוגעלייגט 1 און דערנאָך דאָס וואָרט:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

האַלט אין זינען אַז איר קענט נישט האָבן בלויז איין לינק פֿאַר פאַרשידענע בילינג־אָפֿטקייטן. איר זאָלט טיילן איין ספּעציפֿישן לינק פֿאַר יעדער פֿון אײַערע בילינג־אָפֿטקייטן.

אַמאָל איר טיילט דעם לינק מיט אײַערע קונים, וועט ער זיי ווײַטערפֿירן צום ערשטן שריט פֿון אײַער רעגיסטראַציע־פֿלוס.
