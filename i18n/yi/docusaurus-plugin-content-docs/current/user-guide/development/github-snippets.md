# ווי צו צולייגן דעם קאָד־אויסשניט אין דער functions.php טעקע פֿון אײַער טעמע. {#how-to-add-the-snippet-on-your-themes-functionsphp-file}

  1. לאָגט זיך אַרײַן אין אײַער WordPress נעץ־אַדמין Dashboard און גייט צו טעמעס >טעמע־רעדאַקטאָר (זעט דעם סקרינשאָט אונטן).

  2. אויף דער "רעדאַקטירן טעמעס" זײַט, מאַכט זיכער אַז אײַער אַקטיווע טעמע איז אויסגעקליבן אין דעם דראָפּדאַון־פֿעלד וואָס געפֿינט זיך אויבן רעכטס אויף אײַער עקראַן (#3 אויפֿן סקרינשאָט אונטן).

  3. קליקט אויף דער functions.php טעקע אונטער דער "טעמע־טעקעס" אָפּטיילונג כּדי צו לאָדן די טעקע. ראָלט אַראָפּ ביז אונטן און קלעפּט אַרײַן דעם Ultimate Multisite קאָד־אויסשניט וואָס איר האָט באַקומען פֿון דער GitHub רעפּאָזיטאָריע.

# ווי צו שאַפֿן מוז־נוצן־צושטעקן (mu-plugins) {#how-to-create-must-use-plugins-mu-plugins}

WordPress האָט אַ פֿעיִקייט וואָס דערלויבט אײַך צו לאָדן אייגענע פֿונקציאָנאַליטעט גערופֿן "מוז־נוצן־צושטעקן", אָדער "mu-plugins" בקיצור.

די ספּעציעלע mu-plugins ווערן געלאָדן פֿאַר אַלע אַנדערע רעגולערע צושטעקן, און מע קען זיי נישט דעאַקטיווירן. אין אַ multisite נעץ, וועט דער קאָד אין די mu-plugins ווערן געלאָדן אויף אַלע זײַטן אין אײַער אינסטאַלאַציע.

1\. ניצט FTP אָדער SSH כּדי צוצוקומען צום טעקע־סיסטעם פֿון אײַער WordPress אינסטאַלירונג.

2\. אינעווייניק אין דער wp-content דירעקטאָריע פֿון אײַער WordPress אינסטאַלירונג, שאַפֿט אַ נײַע דירעקטאָריע מיטן נאָמען: mu-plugins.

<!-- סקרינשאָט נישט בנימצא: טעקע־פֿאַרוואַלטער ווײַזט די wp-content דירעקטאָריע מיטן mu-plugins טעקע־קאַסטן -->

3\. שאַפֿט אַ נײַע PHP טעקע אויף אײַער קאָמפּיוטער מיטן נאָמען wu-snippet.php ניצנדיק Notepad אָדער וועלכן קאָד־רעדאַקטאָר.

4\. לייגט אַרײַן אין דער טעקע דעם Ultimate Multisite קאָד־אויסשניט וואָס איר האָט באַקומען פֿון דער GitHub רעפּאָזיטאָריע און היט עס. איר קענט אויך צולייגן דעם קאָד אויבן פֿונעם קאָד־אויסשניט כּדי צו באַצייכענען אײַער MU plugin.
