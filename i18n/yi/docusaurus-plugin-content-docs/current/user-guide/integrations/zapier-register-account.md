# געשעעניש: רעגיסטרירן אַ חשבון דורך Zapier {#event-register-an-account-via-zapier}

אין דעם אַרטיקל [אינטעגרירן Ultimate Multisite מיט Zapier](zapier.md), האָבן מיר באַהאַנדלט ווי צו נוצן Zapier כּדי אויסצופירן פֿאַרשידענע אַקציעס אין Ultimate Multisite לויט טריגערס און געשעענישן. אין דעם אַרטיקל וועלן מיר ווײַזן ווי איר קענט אינטעגרירן אַפּליקאַציעס פֿון דריטע פּאַרטייען. מיר וועלן נוצן Google Sheets ווי דער מקור פֿון דאַטן און שיקן די אינפֿאָרמאַציע צו Ultimate Multisite כּדי צו רעגיסטרירן אַ חשבון.

איידער איר פֿאַרבינדט Zapier, גייט צו **Ultimate Multisite > Settings > API & Webhooks** און באַשטעטיקט אַז דער API איז אַקטיווירט. קאָפּירט דעם API Key און API Secret פֿון דעם עקראַן ווען Zapier בעט פֿאַר די Ultimate Multisite Account קרעדענציאַלן.

![API און Webhooks אײַנשטעלונגען מיט API Key, API Secret, און Enable API אָפּציעס](/img/admin/settings-api-webhooks.png)

ערשטנס, דאַרפֿט איר שאַפֿן אַ **Google Sheet** אונטער אײַער Google Drive. מאַכט זיכער אַז איר דעפֿינירט ריכטיק יעדע זייל, כּדי איר זאָלט שפּעטער קענען גרינג מאַפּן די דאַטן.

נאָך שאַפֿן אַ Google sheet, קענט איר זיך אַרײַנלאָגירן אין אײַער Zapier חשבון און אָנהייבן שאַפֿן אַ zap.

אונטער דעם זוך־פֿעלד פֿאַר **"App event"** קלײַבט אויס **"Google Sheets"**


דערנאָך, פֿאַר דעם "**Event** " פֿעלד, קלײַבט אויס "**New spreadsheet row** " און דריקט "**Continue** "

דער קומענדיקער שריט וועט אײַך בעטן אויסצוקלײַבן אַ **Google Account** וווּ דער **Google Sheet** איז אָפּגעהיט. מאַכט נאָר זיכער אַז דער ריכטיקער Google Account איז באַשטימט.


אונטער **"Set up trigger** ", וועט איר דאַרפֿן אויסקלײַבן און ספּעציפֿיצירן דעם Google spreadsheet און worksheet וואָס איר וועט נוצן, פֿון וווּ די דאַטן וועלן קומען. גייט ווײַטער און פֿילט דאָס אויס, און דריקט "**Continue** "

דערנאָך דאַרפֿט איר "**test your trigger** " כּדי זיכערצושטעלן אַז אײַער Google sheet איז ריכטיק פֿאַרבונדן.

אויב אײַער טעסט איז מצליח, זאָלט איר זען דעם רעזולטאַט וואָס ווײַזט עטלעכע ווערטן פֿון אײַערע spreadsheets. דריקט "**Continue** " כּדי ווײַטערצוגיין.

דער קומענדיקער שריט איז אויפֿצושטעלן די צווייטע אַקציע וואָס וועט שאַפֿן אָדער רעגיסטרירן אַ חשבון אין Ultimate Multisite. אינעם זוך־פֿעלד קלײַבט אויס "**Ultimate Multisite(2.0.2)** "


אונטער דעם "**Event** " פֿעלד, קלײַבט אויס "**Register an Account in Ultimate Multisite** " און דערנאָך דריקט דעם "**Continue** " קנעפּל.

אונטער "**Set up an action** ", וועט איר זען פֿאַרשידענע פֿעלדער וואָס זענען בנימצא פֿאַר קונה־דאַטן, מיטגלידערשאַפֿטן, פּראָדוקטן, אאַז״ו. איר קענט מאַפּן די ווערטן אונטער אײַער Google sheet און זיי צוטיילן צום פּאַסיקן פֿעלד וווּ זיי זאָלן ווערן אָנגעפֿילט, ווי געוויזן אין דער סקרינשאָט אונטן.


נאָך מאַפּן די ווערטן, קענט איר טעסטן די אַקציע.
