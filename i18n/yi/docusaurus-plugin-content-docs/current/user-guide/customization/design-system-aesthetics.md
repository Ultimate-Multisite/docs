## וואָס איז פּלאַן־סיסטעם עסטעטיק?

פּלאַן־סיסטעם עסטעטיק איז אַ סטרוקטורירטע פֿעיִקייט וואָס נעמט אַרײַן:

- **טיפּאָגראַפֿיע**: שריפֿט־משפּחות, גרייסן, וואָג, און שורה־הייכן

- **אָפּשטאַנד**: אינעווייניקער אָפּשטופּ, ראַנד, און ריס־סקאַלעס

- שאַפֿט נישט קיין איינמאָליקע קאָלירן אָדער גרייסן

- ✓ `primary`, `secondary`, `accent`

### סקאַלירבאַרקייט
