# Theme Builder פֿעיִקייטן: אויסשטעלן און אַקטיווירן בלאָק־טעמעס

Superdav AI Agent v1.12.0 ברענגט אַרײַן צוויי שטאַרקע פֿעיִקייטן וואָס לאָזן אײַך דזשענערירן און אַרויסשטעלן אייגענע בלאָק־טעמעס גלײַך פֿון דער שמועס־צובינד.

## איבערבליק

די **scaffold-block-theme** און **activate-theme** פֿעיִקייטן לאָזן אַגענטן:
- דזשענערירן גאַנצע, פּראָדוקציע־גרייטע בלאָק־טעמעס לויט אײַערע ספּעציפֿיקאַציעס
- אויטאָמאַטיש אַקטיווירן טעמעס אויף אײַער וועבזײַטל אָן מאַנועלער אַרײַנמישונג
- שאַפֿן צוזאַמענהענגיקע וויזועלע אידענטיטעטן דורך געפֿירטע פּלאַן־באַשלוסן

## אויסשטעלן אַ בלאָק־טעמע

די **scaffold-block-theme** פֿעיִקייט דזשענערירט אַ נײַע WordPress בלאָק־טעמע מיט אַ גאַנצער טעמע־סטרוקטור, אַרײַנגערעכנט:

- `theme.json` קאָנפֿיגוראַציע מיט פּלאַן־טאָקענס
- בלאָק־מוסטער טעקעס פֿאַר געוויינטלעכע אויסלייגס
- אייגענע בלאָק־סטילן און וואַריאַציעס
- טעמע־מעטאַדאַטן און שטיצע־דעקלאַראַציעס

### ווי אַזוי אַרויסצורופֿן

אין אײַער שמועס מיט Superdav AI Agent קענט איר בעטן טעמע־דזשענעראַציע:

```
"Create a block theme called 'Modern Agency' with a blue and white color scheme,
sans-serif typography, and a professional layout"
```

דער אַגענט וועט:
1. זאַמלען אײַערע פּלאַן־פּרעפֿערענצן דורך שמועס
2. דזשענערירן די גאַנצע טעמע־סטרוקטור
3. שאַפֿן אַלע נייטיקע טעמע־טעקעס
4. צוגרייטן די טעמע פֿאַר אַקטיוואַציע

### ערוואַרטע רעזולטאַט

ווען די פֿעיִקייט ווערט אויסגעפֿירט מיט הצלחה, וועט איר זען:

- באַשטעטיקונג אַז די טעמע איז אויסגעשטעלט געוואָרן
- דעם טעמע־נאָמען און אָרט
- אַ קיצור פֿון די אָנגענוצטע פּלאַן־טאָקענס (קאָלירן, טיפּאָגראַפֿיע, ספּייסינג)
- סטאַטוס גרייט צו אַקטיווירן

בײַשפּיל־רעזולטאַט:
```
✓ Theme "Modern Agency" scaffolded successfully
  Location: /wp-content/themes/modern-agency/
  Colors: Primary #0066CC, Secondary #FFFFFF
  Typography: Inter (sans-serif)
  Status: Ready to activate
```

## אַקטיווירן טעמע

די **activate-theme** פֿעיִקייט בײַט אײַער וועבזײַטל צו אַ נײַ־אויסגעשטעלטער אָדער עקזיסטירנדיקער בלאָק־טעמע.

### ווי אַזוי אַרויסצורופֿן

נאָכן אויסשטעלן אַ טעמע, קענט איר זי גלײַך אַקטיווירן:

```
"Activate the Modern Agency theme"
```

אָדער אַקטיווירן סײַ וועלכע עקזיסטירנדיקע טעמע:

```
"Switch to the Twentytwentyfour theme"
```

### ערוואַרטע רעזולטאַט

ווען אַקטיוואַציע געלינגט:

- באַשטעטיקונג פֿון דער אַקטיווער טעמע
- פֿריִערדיקער טעמע־נאָמען (פֿאַר רעפֿערענץ)
- וועבזײַטל־URL וווּ די טעמע איז איצט לעבן
- סײַ וועלכע טעמע־ספּעציפֿישע אויפֿשטעל־באַמערקונגען

בײַשפּיל־רעזולטאַט:
```
✓ Theme activated successfully
  Active theme: Modern Agency
  Previous theme: Twentytwentyfour
  Live at: https://yoursite.com
  Note: Check your homepage to verify the layout
```

## אַרבעטס־פֿלוס: אויסשטעלן און אַקטיווירן

אַ טיפּישער אַרבעטס־פֿלוס קאָמבינירט ביידע פֿעיִקייטן:

1. **בעטן טעמע־דזשענעראַציע**: "שאַף אַ בלאָק־טעמע פֿאַר מײַן SaaS לאַנדינג־זײַט"
2. **דער אַגענט שטעלט אויס די טעמע**: דזשענערירט טעקעס און פּלאַן־טאָקענס
3. **איבערקוקן און פֿאַרפֿײַנערן**: באַהאַנדלט פּלאַן־ענדערונגען אויב נייטיק
4. **אַקטיווירן**: "אַקטיוויר די טעמע איצט"
5. **באַשטעטיקן**: באַזוכט אײַער וועבזײַטל צו באַשטעטיקן אַז דער נײַער פּלאַן איז לעבן

## פּלאַן־טאָקענס און צופּאַסונג

אויסגעשטעלטע טעמעס ניצן WordPress פּלאַן־טאָקענס (דורך `theme.json`) פֿאַר:

- **קאָלירן**: פּרימערע, סעקונדערע, אַקצענט, נייטראַלע פּאַלעט
- **טיפּאָגראַפֿיע**: שריפֿט־משפּחות, גרייסן, וואָג, שורה־הייכן
- **ספּייסינג**: פּאַדינג, מאַרדזשין, שפּאַלט־סקאַלעס
- **ראַנדן**: ראַדיוס און ברייט־טאָקענס
- **שאָטנס**: הייבונג־שטאַפּלען

די טאָקענס זענען צענטראַליזירט אין `theme.json`, וואָס מאַכט עס גרינג צוצופּאַסן אײַער גאַנצע פּלאַן־סיסטעם פֿון איין טעקע.

## באַגרענעצונגען און באַמערקונגען

- טעמעס ווערן אויסגעשטעלט אין `/wp-content/themes/` און מוזן נאָכפֿאָלגן WordPress נאָמען־קאָנווענצן
- אַקטיוואַציע דאַרף פּאַסיקע דערלויבענישן אויף אײַער WordPress וועבזײַטל
- אייגענער PHP קאָד אין טעמעס איז מינימאַל; ניצט פּלוגינס פֿאַר קאָמפּלעקסע פֿונקציאָנאַליטעט
- בלאָק־טעמעס אַרבעטן בעסטער מיט WordPress 5.9 און שפּעטער

## טראָבלשוטינג

**די טעמע דערשײַנט נישט נאָך אויסשטעלן**
- באַשטעטיקט אַז דער טעמע־טעקע־קעסטל עקזיסטירט און האָט געהעריקע דערלויבענישן
- קאָנטראָלירט אַז `theme.json` איז גילטיקער JSON
- מאַכט זיכער אַז דער טעמע־נאָמען קומט נישט אין קאָנפֿליקט מיט עקזיסטירנדיקע טעמעס

**אַקטיוואַציע פֿאַלט דורך**
- באַשטעטיקט אַז איר האָט אַדמיניסטראַטאָר־דערלויבענישן
- קאָנטראָלירט אַז דער טעמע־טעקע־קעסטל איז לייענבאַר דורך WordPress
- איבערקוקט WordPress טעות־לאָגס פֿאַר פּרטים

**פּלאַן־טאָקענס ווערן נישט אָנגענומען**
- באַשטעטיקט אַז `theme.json` סינטאַקס איז ריכטיק
- רייניקט סײַ וועלכע קאַשינג־פּלוגינס
- קאָנטראָלירט אַז אײַער WordPress ווערסיע שטיצט די טאָקענס וואָס איר ניצט

## קומענדיקע שריט

נאָך אַקטיווירן אײַער טעמע, קענט איר:
- ניצן די **פּלאַן־סיסטעם עסטעטיק־בקיאות** צו פֿאַרפֿײַנערן טיפּאָגראַפֿיע, קאָלירן, און ספּייסינג
- צופּאַסן יחידישע בלאָק־סטילן דורך דעם WordPress בלאָק־רעדאַקטאָר
- צולייגן אייגענעם CSS אין דער טעמעס `style.css` טעקע
- שאַפֿן אייגענע בלאָק־מוסטערס פֿאַר ספּעציפֿישע זײַט־טיפּן
