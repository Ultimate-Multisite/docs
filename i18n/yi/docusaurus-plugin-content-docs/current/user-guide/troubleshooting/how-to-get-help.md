# ווי צו באַקומען הילף {#how-to-get-help}

איר האָט באַקומען אייער גילטיקן License key, און צוזאַמען דערמיט האָט איר צוטריט צו אונדזער קאָלעקטיוו כּדי אייך צו העלפֿן לייזן פּראָבלעמען וואָס איר קענט טרעפֿן בעת איר שטעלט אויף Ultimate Multisite.

די לעקציע באַדעקט די בנימצא שטיצע־קאַנאַלן פֿאַר License־האַלטערס און וואָס פֿאַר אינפֿאָרמאַציע איר דאַרפֿט האָבן בײַ דער האַנט כּדי צו קאָנטאַקטירן אונדזער שטיצע־קאָלעקטיוו.

## הילף־זײַטל {#help-site}

איר קענט צוטריט באַקומען צו פֿאַרשיידענע אַרטיקלען און טוטאָריאַלס פֿון אונדזער הילף־זײַטל, וואָס איר קענט געפֿינען אין דעם לינק אונטן.

מיר מוטיקן אונדזערע באַניצער זיך באַקאַנט צו מאַכן מיט די יסודותדיקע פֿעיִקייטן און פֿונקציעס פֿון Ultimate Multisite. עס זענען פֿאַראַן „Getting Started” אַרטיקלען אויף אונדזער הילף־זײַטל, און ענטפֿערס אויף געוויינטלעכע פֿראַגעס קען מען אויך דאָרטן געפֿינען, ווי למשל ווי צו אינסטאַלירן צוגעבן, ווי צו אינטעגרירן צאָלונג־גייטווייז, אאַז״ו.

## קאָנטאַקטירן שטיצע {#contact-support}

אויב איר בלײַבט שטעקן און קענט נישט געפֿינען דעם ענטפֿער אויף אונדזער הילף־זײַטל, קענט איר זיך ווענדן צו אונדזער שטיצע־קאָלעקטיוו אָדער דורך קליקן אויף דעם שמועס־בלאָז־אייקאָן וואָס געפינט זיך בײַם נידעריקן רעכטן זײַט פֿון אונדזער וועבזײַטל, אָדער דורך שיקן אַ בליצפּאָסט גלײַך צו support@ultimatemultisite.com.

ווען איר קאָנטאַקטירט שטיצע וועגן טעכנישע פּראָבלעמען, ביטע מאַכט זיכער אַז איר גיט אַ פּרטימדיקע באַשרײַבונג פֿון אייער פּראָבלעם צוזאַמען מיט אַ סקרינשאָט, טעות־מעלדונגען, אָדער יעדע אינפֿאָרמאַציע וואָס וועט זײַן נוצלעך פֿאַר אונדזערע שטיצע־מענטשן כּדי גרינג צו פֿאַרשטיין און אָפּשאַצן אייער פּראָבלעם. איר דאַרפֿט אויך צושטעלן אייער License key. נאָר אַקטיווע License־האַלטערס האָבן רעכט אויף שטיצע.
