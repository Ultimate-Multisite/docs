# באַקאַנטע אומקאָמפּאַטיביליטעטן מיט אַנדערע Plugins {#known-incompatibilities-with-other-plugins}

נאָך אינסטאַלירן און אַקטיווירן Ultimate Multisite, קענט איר זיך אָנשטויסן אויף פּראָבלעמען צוליב נישט־באַהאַנדלטע אומקאָמפּאַטיביליטעטן מיט אַנדערע plugins.

מיר אַרבעטן אַקטיוו צו פֿאַרקלענערן די רשימה מיט יעדן נײַעם אַרויסלאָז, אָבער אויב אַ plugin איז דאָ אויסגערעכנט, וועט איר מעגלעך דאַרפֿן עס דעאַקטיווירן כּדי צו האָבן אַ פול פֿונקציאָנירנדיקע Ultimate Multisite Network.

דאָ איז אַ רשימה פֿון זיי און וואָס איר זאָלט טאָן אין די דאָזיקע פֿאַלן:

**עטלעכע WPMU DEV plugins** עטלעכע פֿון די plugins וואָס ווערן _נישט מער_ אונטערגעהאַלטן דורך WPMU DEV זענען באַקאַנט צו מאַכן אַז דער Ultimate Multisite רעגיסטראַציע־לויפֿ זאָל זיך אָפּשטעלן בײַם לעצטן שריט (דער באַניצער קען נישט שאַפֿן אַן Account). דאָ איז אַ רשימה פֿון די plugins וואָס פֿאַרשאַפֿן יענע פּראָבלעמען:

  * Signup Language: _Ultimate Multisite האָט אַן add-on וואָס אָפפערט די זעלבע פֿונקציאָנאַליטעט, גערופֿן Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _דאָס איז אַרײַנגעבויט אין Ultimate Multisite, אַזוי אַז עס איז נישטאָ קיין נויט פֿאַר אַן עקסטרע plugin._

  * New Blog Templates: _דאָס איז אַרײַנגעבויט אין Ultimate Multisite, אַזוי אַז עס איז נישטאָ קיין נויט פֿאַר אַן עקסטרע plugin._

  * Set Password on Multisite Blog Creation: _דאָס איז אַרײַנגעבויט אין Ultimate Multisite, אַזוי אַז עס איז נישטאָ קיין נויט פֿאַר אַן עקסטרע plugin._

  * Signup TOS: _דאָס איז אַרײַנגעבויט אין Ultimate Multisite, אַזוי אַז עס איז נישטאָ קיין נויט פֿאַר אַן עקסטרע plugin._

  * Anti-Splog: כאָטש ווייניקער שטאַרק, איז דאָס אויך _אַרײַנגעבויט אין Ultimate Multisite._

**באַמערקונג** : אַלע אַנדערע WPMU Dev plugins קענען געניצט ווערן נאָרמאַל צוזאַמען מיט Ultimate Multisite. בײַשפּילן אַרײַננעמען _Smush_ , _Forminator_ , _Defender,_ אאַז״ו.
