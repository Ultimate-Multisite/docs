# פֿירן צאָלונגען און רעכענונגען {#managing-payments-and-invoices}

## צאָלונג־איינשטעלונגען {#payment-settings}

איידער איר הייבט אָן צו באַקומען צאָלונגען, דאַרפֿט איר קאָנפֿיגורירן די צאָלונג־פֿאַרבונדענע איינשטעלונגען. גייט צו **Ultimate Multisite → איינשטעלונגען** און קליקט אויף דעם **צאָלונג**־טאַב.

![צאָלונג־איינשטעלונגען טאַב](/img/admin/settings-payments-top.png)

דאָ איז אַ פולער בליק אויף דער צאָלונג־איינשטעלונגען־זײַט:

![צאָלונג־איינשטעלונגען פולע זײַט](/img/admin/settings-payments-full.png)

### אַלגעמיינע צאָלונג־אָפּציעס {#general-payment-options}

אין די אַלגעמיינע איינשטעלונגען קענט איר קאָנפֿיגורירן:

- **וואַלוטע** — די סטאַנדאַרט־וואַלוטע געניצט פֿאַר טראַנזאַקציעס
- **וואַלוטע־פּאָזיציע** — וווּ דער וואַלוטע־סימבאָל דערשיינט (פֿאַר/נאָך דער סומע)

![צאָלונג־גייטוויי איינשטעלונגען](/img/admin/settings-payments-options.png)

### צאָלונג־גייטווייען {#payment-gateways}

Ultimate Multisite שטיצט עטלעכע צאָלונג־גייטווייען. איר קענט אַקטיוויזירן און קאָנפֿיגורירן יעדן איינעם פֿון דעם צאָלונג־איינשטעלונגען־טאַב.

![צאָלונג־גייטוויי קאָנפֿיגוראַציע](/img/admin/settings-payments-gateways.png)

פֿאַראַנענע גייטווייען אַרײַננעמען:

- **Stripe** — קרעדיט־קאַרטל־צאָלונגען דורך Stripe
- **PayPal** — PayPal צאָלונגען
- **מאַנואַל** — פֿאַר אָפֿלײַן אָדער קאַסטאַם צאָלונג־פּראָסעסירונג

יעדער גייטוויי האָט זײַן אייגענע קאָנפֿיגוראַציע־אָפּטיילונג, וווּ איר גיט אַרײַן API שליסלען און אַנדערע איינשטעלונגען.

![נאָך גייטוויי איינשטעלונגען](/img/admin/settings-payments-gateways-2.png)

### Sandbox Mode {#sandbox-mode}

איר קענט אַקטיוויזירן **Sandbox Mode** כּדי צו פּרובירן אײַער צאָלונג־אינטעגראַציע איידער איר גייט לעבן. ווען sandbox mode איז אַקטיוו, וועלן נישט ווערן געמאַכט קיין אמתע אָפּצאָלונגען.

## באַטראַכטן צאָלונגען {#viewing-payments}

גייט צו דער **צאָלונגען**־זײַט אונטער Ultimate Multisite כּדי צו זען אַלע טראַנזאַקציעס איבער אײַער נעץ.

![צאָלונגען רשימה](/img/admin/payments-list.png)

איר קענט פֿילטערן צאָלונגען לויט סטאַטוס (פֿאַרענדיקט, פּענדינג, דורכגעפֿאַלן, צוריקגעצאָלט) און זוכן ספּעציפֿישע טראַנזאַקציעס.

קליקט אויף אַ צאָלונג כּדי צו זען אירע פולע פּרטים, אַרײַנגערעכנט די שורה־אייטעמס, פֿאַרבונדענע מיטגלידערשאַפֿט, קונה־אינפֿאָרמאַציע, און צאָלונג־גייטוויי דאַטן.

## רעכענונגען {#invoices}

Ultimate Multisite קען אויטאָמאַטיש דזשענערירן רעכענונגען פֿאַר צאָלונגען. איר קענט צופּאַסן דעם רעכענונג־מוסטער און נומערירונג־פֿאָרמאַט פֿון די צאָלונג־איינשטעלונגען.

רעכענונג־צופּאַסונג־אָפּציעס אַרײַננעמען:

- **פֿירמע נאָמען און אַדרעס** געוויזן אויף רעכענונגען
- **רעכענונג־נומערירונג** פֿאָרמאַט און סעקווענץ
- **לאָגאָ** געוויזן אויפֿן רעכענונג־קעפּל
- **אייגענער פֿוס־טעקסט** פֿאַר טערמינען, באַמערקונגען, אָדער לעגאַלע אינפֿאָרמאַציע

כּדי צו צופּאַסן דעם רעכענונג־מוסטער, גייט צו **Ultimate Multisite → איינשטעלונגען → צאָלונג** און זוכט די רעכענונג־פֿאַרבונדענע איינשטעלונגען.
