# Gratis AI Agent איינשטעלונגען

דער **איינשטעלונגען → פארגעשריטן** עקראַן אין Gratis AI Agent גיט אַדמיניסטראַטאָר־מדרגה קאָנפֿיגוראַציע פֿאַר באַקענד־אינטעגראַציעס. די זײַט דאָקומענטירט ווײַטערשיקן פֿון באַמערקונגען, זוך־סאַפּלייער־שליסלען, אויפֿשטעלונג פֿון אַ געראטן Superdav דינסט, Google Calendar קאָנטראָלן, TextBee SMS איינשטעלונגען, און נעץ־ברייטע פֿעיִקייט־פֿלעגס.

## צוטריט צו איינשטעלונגען {#accessing-settings}

1. אין דעם WordPress אַדמין, גייט צו **Gratis AI Agent → איינשטעלונגען**.
2. קוועטשט אויף דעם **פארגעשריטן** קוויטל.

## קאָנפֿיגוראַציע פֿון באַמערקונג־ענדפּונקט {#feedback-endpoint-configuration}

דער באַמערקונג־ענדפּונקט נעמט אָן POST בקשות פֿון דעם AI אַגענט יעדעס מאָל ווען אַ באַניצער שיקט אַרײַן באַמערקונג דורך דעם גראָבער־פֿינגער־אַראָפּ קנעפּל, דעם אויטאָ־פּראָמפּט באַנער, אָדער דעם `/report-issue` באַפֿעל.

| פֿעלד | באַשרײַבונג |
|---|---|
| **באַמערקונג־ענדפּונקט URL** | דער URL וואָס נעמט אָן באַמערקונג־אײַנשיקונגען ווי HTTP POST בקשות מיט אַ JSON גוף. |
| **Feedback API Key** | אַ bearer token געשיקט אין דעם `Authorization` header פֿון יעדער באַמערקונג־בקשה. לאָזט ליידיק אויב אײַער ענדפּונקט דאַרף נישט קיין אָטענטיפֿיקאַציע. |

### דערוואַרטע JSON ניץ־לאַסט {#expected-json-payload}

אײַער באַמערקונג־ענדפּונקט מוז אָננעמען אַ JSON גוף מיט לפּחות די ווײַטערדיקע פֿעלדער:

```json
{
  "message_id": "msg_abc123",
  "conversation_id": "conv_xyz789",
  "feedback_text": "The answer was incorrect about pricing.",
  "triage_category": "factual_error"
}
```

נאָך פֿעלדער קענען זײַן פֿאַראַן אין דער ניץ־לאַסט, אָפענגיק פֿון דעם שמועס־קאָנטעקסט.

### `triage_category` ווערטן {#triagecategory-values}

די AI טריאַזש־שיכט טיילט צו איינעם פֿון די ווײַטערדיקע ווערטן צו `triage_category` איידער זי שיקט ווײַטער די ניץ־לאַסט:

| ווערט | טײַטש |
|---|---|
| `factual_error` | דער אַסיסטענט האָט צוגעשטעלט אומריכטיקע פֿאַקטישע אינפֿאָרמאַציע. |
| `unhelpful_answer` | דער ענטפֿער איז געווען טעכניש ריכטיק, אָבער נישט נוצלעך. |
| `inappropriate_content` | דער ענטפֿער האָט אַנטהאַלטן אינהאַלט וואָס זאָל נישט געוויזן ווערן באַניצער. |
| `other` | די באַמערקונג האָט נישט געפּאַסט צו אַ באַקאַנטער קאַטעגאָריע. |

### אָטענטיפֿיקאַציע {#authentication}

אויב אײַער ענדפּונקט דאַרף אָטענטיפֿיקאַציע, שטעלט דאָס **Feedback API Key** פֿעלד אויף אײַער bearer token. דער אַגענט שיקט:

```
Authorization: Bearer <your-api-key>
```

אויב דאָס **Feedback API Key** פֿעלד איז ליידיק, ווערט קיין `Authorization` header נישט געשיקט.

### אויסשליסן זאַמלונג פֿון באַמערקונגען {#disabling-feedback-collection}

לאָזט ביידע פֿעלדער **באַמערקונג־ענדפּונקט URL** און **Feedback API Key** ליידיק. דער גראָבער־פֿינגער־אַראָפּ קנעפּל און באַמערקונג־UI בלײַבן קענטיק פֿאַר באַניצער, אָבער אײַנשיקונגען ווערן נישט ווײַטערגעשיקט צו קיין אויסערלעכער דינסט.

## Brave Search API Key {#brave-search-api-key}

אויך אויף דעם **פארגעשריטן** קוויטל, דאָס **Brave Search API Key** פֿעלד אַקטיווירט די [אינטערנעץ־זוך](../configuration/internet-search) פֿעיִקייט.

| פֿעלד | באַשרײַבונג |
|---|---|
| **Brave Search API Key** | אײַער API שליסל פֿון דעם Brave Search אַנטוויקלער־Dashboard. נייטיק צו אַקטיווירן אינטערנעץ־זוך אין דעם AI אַסיסטענט. |

דער פֿעלד־עטיקעט נעמט אַרײַן אַ קוועטשבאַרן לינק צו דער Brave Search API רעגיסטראַציע־זײַט. לאָזט ליידיק כּדי אויסצושליסן אינטערנעץ־זוך.

זעט [אינטערנעץ־זוך](../configuration/internet-search) פֿאַר ענד־באַניצער דאָקומענטאַציע וועגן דער פֿעיִקייט.

## געראַטענע Superdav דינסט {#managed-superdav-service}

Superdav AI Agent v1.18.0 לייגט צו געראַטענע Superdav דינסט־ענדפּונקטן און אויטאָמאַטישע פֿאַרבינדונג־צושטעלונג פֿאַר געשטיצטע זייטלעך. ניצט די קאָנטראָלן ווען אײַער זייטל זאָל זיך פֿאַרבינדן מיטן געבאַלעבאַטעטן סאַפּלייער אַנשטאָט אַ מאַנועל קאָנפֿיגורירטן דינסט־ענדפּונקט.

| פֿעלד | באַשרײַבונג |
|---|---|
| **געראַטענע Superdav דינסט** | אַקטיווירט די געבאַלעבאַטעטע Superdav דינסט־פֿאַרבינדונג פֿאַר געשטיצטע זייטלעך. |
| **צושטעלן פֿאַרבינדונג** | הייבט אָן אויטאָמאַטישע ענדפּונקט־און קרעדענציעל־צושטעלונג. ניצט דאָס נאָך באַשטעטיקן אַז דאָס זייטל זאָל נוצן דעם געראַטענעם סאַפּלייער. |
| **דינסט־ענדפּונקט / פֿאַרבינדונג־סטאַטוס** | ווײַזט דעם איצטיקן ענדפּונקט אָדער פֿאַרבינדונג־צושטאַנד נאָך צושטעלונג. |

נאָך צושטעלונג, היט איינשטעלונגען און באַשטעטיקט דעם פֿאַרבינדונג־סטאַטוס איידער איר פֿאַרלאָזט זיך אויף געראַטענע־דינסט אַרבעטספֿלוסן. אויב צושטעלונג פֿעלט אויס, איבערקוקט די געוויזענע ווידער־פּרוּוון אָנפֿירונג און באַשטעטיקט אַז דאָס זייטל האָט דערלויבעניש צו נוצן דעם געבאַלעבאַטעטן סאַפּלייער.

## Google Calendar קאָנפֿיגוראַציע {#google-calendar-configuration}

ווען Superdav AI Agent v1.18.0 קאַלענדאַר־פֿעיִקייטן זענען אַקטיווירט, קען דער אַגענט לייענען קאָנפֿיגורירטע קאַלענדאַרן און געשעעניש־דעטאַלן. קאַלענדאַר־מכשירים זענען לייענען־אָריענטירט און זענען נוצלעך פֿאַר פּלאַן־באַוווּסטע דערמאָנונגען, באַטייליקטע־נאָכפֿאָלג, און קאָנטאַקט־צופּאַסן.

| פֿעלד | באַשרײַבונג |
|---|---|
| **Google Calendar קרעדענציעלן** | היט די קרעדענציעלן אָדער token פֿאַרבינדונג וואָס איז נייטיק צו לייענען קאַלענדאַר־דאַטן. |
| **קאַלענדאַר־אויסקלייב** | באַגרענעצט וועלכע קאָנפֿיגורירטע קאַלענדאַרן דער אַגענט מעג דורכקוקן. |
| **קאַלענדאַר־פֿאַרבינדונג־סטאַטוס** | באַשטעטיקט צי די איצטיקע קרעדענציעלן קענען לייענען קאַלענדאַרן און געשעענישן. |

האַלט קאַלענדאַר־קרעדענציעלן באַגרענעצט צו די קאַלענדאַרן וואָס דער אַגענט דאַרף. פֿאַרבינדט ווידער אָדער דרייט איבער קרעדענציעלן אויב דער סטאַטוס ווײַזט אַן אויסגעלאָפֿענעם token.

## TextBee SMS מעלדונגען {#textbee-sms-notifications}

Superdav AI Agent v1.18.0 לייגט צו TextBee ווי אַן SMS סאַפּלייער פֿאַר קאָנפֿיגורירטע מעלדונג־אַרבעטספֿלוסן. SMS מעלדונגען זאָלן זײַן פֿאַרפּאַרט מיט מענטשלעכע באַשטעטיקונג־טויערן פֿאַר שפּירעוודיקע אָדער באַניצער־געווידמעטע אָנזאָגן.

| פֿעלד | באַשרײַבונג |
|---|---|
| **TextBee API שליסל** | אָטענטיפֿיצירט בקשות צום TextBee SMS סאַפּלייער. |
| **TextBee מיטל / שיקער** | קלייבט אויס דעם TextBee שיקער אָדער מיטל געניצט פֿאַר אַרויסגייענדיקע אָנזאָגן, ווען דער סאַפּלייער דאַרף עס. |
| **SMS מעלדונגען אַקטיווירט** | דערלויבט באַשטעטיקטע אַרבעטספֿלוסן צו שיקן טעקסט־אָנזאָג מעלדונגען. לאָזט אויסגעשלאָסן כּדי צו פֿאַרמיידן SMS שיקונגען. |

שיקט אַ פּראָבע־אָנזאָג נאָר צו אַ נומער וואָס געהערט אַן אַדמיניסטראַטאָר, דערנאָך באַשטעטיקט דעם אויפֿפֿיר פֿון באַשטעטיקונג־טויער איידער איר אַקטיווירט פּלאַנירטע אָדער באַטייליקטע־געווידמעטע דערמאָנונגען.

## פֿעיִקייט־פֿלעגס {#feature-flags}

אויך אַרײַנגעפֿירט אין v1.9.0, דער **איינשטעלונגען → פֿעיִקייט־פֿלעגס** קוויטל גיט איבערקיפּ־שאַלטערס פֿאַר אָפּציאָנעלע פֿונקציאָנאַליטעט. יעדער פֿלעג איז אָדער אַקטיווירט אָדער אויסגעשלאָסן נעץ־ברייט; דערווײַל איז נישטאָ קיין איבעררײַס לויט זייטל.

### צוטריט צו פֿעיִקייט־פֿלעגס {#accessing-feature-flags}

1. אין דעם WordPress אַדמין, גייט צו **Gratis AI Agent → איינשטעלונגען**.
2. קוועטשט אויף דעם **פֿעיִקייט־פֿלעגס** קוויטל.

### צוטריט־קאָנטראָל פֿלעגס {#access-control-flags}

| פֿאָן | פֿעליקייט | באַשרײַבונג |
|---|---|---|
| **באַגרענעצן צו אַדמיניסטראַטאָרן** | אויס | ווען אַקטיווירט, קענען בלויז באַניצער מיט דער `administrator` ראָלע עפֿענען דעם AI Agent שמועס־פּאַנעל. אַלע אַנדערע ראָלעס זעען אַנשטאָט אַ מעלדונג "פֿאַרבינד זיך מיט דײַן אַדמיניסטראַטאָר". |
| **באַגרענעצן צו Network Admins** | אויס | ווען אַקטיווירט אויף אַ מולטיסײַט־נעץ, קענען בלויז Super Admins נוצן דעם agent. יחידישע סײַט־אַדמיניסטראַטאָרן ווערן געבלאָקט. דאָס האָט פֿאָרראַנג איבער "באַגרענעצן צו אַדמיניסטראַטאָרן" אויב ביידע זענען אַקטיווירט. |
| **דערלויבן Subscriber צוטריט** | אויס | ווען אַקטיווירט, קענען באַניצער מיט דער `subscriber` ראָלע נוצן דעם שמועס־צובינד, אָבער זענען באַגרענעצט צו לייען־בלויז מעגלעכקייטן (קיין שאַפֿן פּאָסטן אָדער ענדערונגען אין סעטטינגס). |
| **דיאַקטיווירן פֿאַר נישט־מיטגלידער** | אויס | אינטעגרירט זיך מיט Ultimate Multisite מיטגלידערשאַפֿט־סטאַטוס. ווען אַקטיווירט, ווערט שמועס באַהאַלטן פֿאַר סײַטן וואָס האָבן נישט קיין אַקטיווע מיטגלידערשאַפֿט. |

### בראַנדינג־פֿאָנען {#branding-flags}

| פֿאָן | פֿעליקייט | באַשרײַבונג |
|---|---|---|
