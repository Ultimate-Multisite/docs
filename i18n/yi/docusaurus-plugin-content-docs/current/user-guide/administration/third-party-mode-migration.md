# מיגראַציע פֿון דריט־פּאַרטיי־מאָדוס {#third-party-mode-migration}

Superdav AI Agent v1.12.0 ענדערט ווי דריט־פּאַרטיי־פֿעיִקייטן ווערן באַהאַנדלט. **דריט־פּאַרטיי־מאָדוס איז איצט לויטן פֿעליקייט אויטאָ**, וואָס אַקטיווירט נייטיווע WordPress Abilities API אינטעגראַציע אויף WordPress 7.0+ אָן מאַנועלע קאָנפֿיגוראַציע.

## וואָס האָט זיך געביטן? {#what-changed}

### פֿאַר v1.12.0 {#before-v1120}

דריט־פּאַרטיי־פֿעיִקייטן האָבן געפֿאָדערט מאַנועלע קאָנפֿיגוראַציע:

- איר האָט געדאַרפֿט בפֿירוש אַקטיווירן "דריט־פּאַרטיי־מאָדוס"
- פֿעיִקייטן זענען געלאָדן געוואָרן פֿון אַ אייגענעם רעגיסטרי
- אינטעגראַציע מיט WordPress Abilities API איז געווען אָפּציאָנעל
- Legacy־מאָדוס איז געווען דער פֿעליקייט

### נאָך v1.12.0 {#after-v1120}

דריט־פּאַרטיי־פֿעיִקייטן אַרבעטן אויטאָמאַטיש:

- דריט־פּאַרטיי־מאָדוס איז לויטן פֿעליקייט "אויטאָ"
- פֿעיִקייטן אינטעגרירן זיך נייטיוו מיט WordPress Abilities API
- קיין מאַנועלע קאָנפֿיגוראַציע איז נישט נויטיק אויף WordPress 7.0+
- Legacy־מאָדוס איז נאָך אַלץ בנימצא פֿאַר עלטערע WordPress ווערסיעס

## ווער ווערט אַפעקטירט? {#who-is-affected}

### נײַע אינסטאַלאַציעס (WordPress 7.0+) {#new-installations-wordpress-70}

**קיין אַקציע איז נישט נויטיק.** דריט־פּאַרטיי־מאָדוס ווערט אויטאָמאַטיש געשטעלט אויף "אויטאָ", און פֿעיִקייטן אַרבעטן גלײַך נאָך אינסטאַלאַציע.

### עקזיסטירנדיקע אינסטאַלאַציעס {#existing-installations}

**אײַערע סעטטינגס ווערן אָפּגעהיט.** אויב איר האָט גענוצט:

- **Legacy־מאָדוס**: איר בלײַבט אין Legacy־מאָדוס (קיין ענדערונג)
- **מאַנועלער דריט־פּאַרטיי־מאָדוס**: איר בלײַבט אין מאַנועלן מאָדוס (קיין ענדערונג)
- **אויטאָ־מאָדוס**: איר גייט ווײַטער מיט אויטאָ־מאָדוס (קיין ענדערונג)

### WordPress ווערסיעס פֿאַר 7.0 {#wordpress-versions-before-70}

**Legacy־מאָדוס איז נאָך אַלץ בנימצא.** אויב איר זענט אויף WordPress 6.x אָדער פֿריִער:

- דריט־פּאַרטיי־מאָדוס איז לויטן פֿעליקייט "legacy"
- איר קענט מאַנועל אַקטיווירן דריט־פּאַרטיי־מאָדוס אויב איר ווילט
- אַפּגריידט צו WordPress 7.0+ צו נוצן דעם נייטיוון Abilities API

## פֿאַרשטיין די מאָדוסן {#understanding-the-modes}

### אויטאָ־מאָדוס (נײַער פֿעליקייט) {#auto-mode-new-default}

**אויטאָ־מאָדוס** ניצט נייטיווע WordPress Abilities API אינטעגראַציע:

- פֿעיִקייטן ווערן רעגיסטרירט דורך WordPress hooks
- פולע קאָמפּאַטיביליטעט מיט WordPress 7.0+ Abilities API
- אויטאָמאַטישע אַנטדעקונג פֿון דריט־פּאַרטיי־פֿעיִקייטן
- קיין מאַנועלע קאָנפֿיגוראַציע איז נישט נויטיק

**ווען צו נוצן**: WordPress 7.0+ מיט דריט־פּאַרטיי־פֿעיִקייטן

### מאַנועלער מאָדוס {#manual-mode}

**מאַנועלער מאָדוס** פֿאָדערט בפֿירושע קאָנפֿיגוראַציע:

- איר ספּעציפֿיצירט וועלכע דריט־פּאַרטיי־פֿעיִקייטן צו לאָדן
- נוצלעך פֿאַר טעסטן אָדער סעלעקטיוו לאָדן פֿעיִקייטן
- פֿאָדערט רעדאַגירן קאָנפֿיגוראַציע־טעקעס
- מער קאָנטראָל, אָבער מער אויפֿשטעלונג

**ווען צו נוצן**: טעסטן, סעלעקטיוו לאָדן פֿעיִקייטן, אָדער אייגענע קאָנפֿיגוראַציעס

### Legacy־מאָדוס {#legacy-mode}

**Legacy־מאָדוס** ניצט דעם אַלטן דריט־פּאַרטיי־פֿעיִקייט־סיסטעם:

- אייגענער פֿעיִקייט־רעגיסטרי (נישט WordPress Abilities API)
- צוריק־קאָמפּאַטיבל מיט עלטערע WordPress ווערסיעס
- קיין נייטיווע WordPress אינטעגראַציע
- פֿאַרעלטערט, אָבער נאָך אַלץ געשטיצט

**ווען צו נוצן**: WordPress 6.x אָדער פֿריִער, אָדער ווען איר דאַרפֿט Legacy־קאָמפּאַטיביליטעט

## קאָנטראָלירן אײַער איצטיקן מאָדוס {#checking-your-current-mode}

### דורך דעם אַדמין־פּאַנעל {#via-admin-panel}

1. גייט צו **WordPress Admin** → **Superdav AI Agent** → **סעטטינגס**
2. זוכט די **דריט־פּאַרטיי־מאָדוס** סעטטינג
3. איר וועט זען אײַער איצטיקן מאָדוס און אָפּציעס אים צו טוישן

### דורך קאָד {#via-code}

```php
$mode = get_option( 'superdav_third_party_mode' );
echo $mode; // 'auto', 'manual', or 'legacy'
```

## טוישן אײַער מאָדוס {#changing-your-mode}

### איבערגיין צו אויטאָ־מאָדוס {#switch-to-auto-mode}

אויב איר זענט אויף WordPress 7.0+ און ווילט נוצן אויטאָ־מאָדוס:

1. גייט צו **Superdav AI Agent** → **סעטטינגס**
2. געפֿינט **דריט־פּאַרטיי־מאָדוס**
3. קלײַבט אויס **אויטאָ (WordPress Abilities API)**
4. קליקט **אויפֿהיטן**

Superdav AI Agent וועט אויטאָמאַטיש אַנטדעקן און רעגיסטרירן דריט־פּאַרטיי־פֿעיִקייטן.

### איבערגיין צו מאַנועלן מאָדוס {#switch-to-manual-mode}

אויב איר ווילט מאַנועל קאָנטראָלירן וועלכע פֿעיִקייטן ווערן געלאָדן:

1. גייט צו **Superdav AI Agent** → **סעטטינגס**
2. געפֿינט **דריט־פּאַרטיי־מאָדוס**
3. קלײַבט אויס **מאַנועל**
4. קליקט **אויפֿהיטן**
5. רעדאַגירט אײַער קאָנפֿיגוראַציע־טעקע צו ספּעציפֿיצירן וועלכע פֿעיִקייטן צו לאָדן

### איבערגיין צו Legacy־מאָדוס {#switch-to-legacy-mode}

אויב איר דאַרפֿט Legacy־קאָמפּאַטיביליטעט:

1. גייט צו **Superdav AI Agent** → **סעטטינגס**
2. געפֿינט **דריט־פּאַרטיי־מאָדוס**
3. קלײַבט אויס **Legacy**
4. קליקט **אויפֿהיטן**

## מעלות פֿון אויטאָ־מאָדוס {#benefits-of-auto-mode}

### אויטאָמאַטישע אַנטדעקונג {#automatic-discovery}

פֿעיִקייטן ווערן אויטאָמאַטיש אַנטדעקט פֿון:

- אינסטאַלירטע plugins
- אַקטיווע theme
- מוז־ניצן plugins
- Drop-in plugins

קיין מאַנועלע רעגיסטראַציע איז נישט נויטיק.

### נייטיווע אינטעגראַציע {#native-integration}

פֿעיִקייטן אינטעגרירן זיך מיט WordPress Abilities API:

- קאָנסיסטענט מיט WordPress קערן
- אַרבעט מיט WordPress אַדמין
- קאָמפּאַטיבל מיט אַנדערע plugins וואָס נוצן Abilities API
- גרייט פֿאַר דער צוקונפֿט ווי WordPress אַנטוויקלט זיך

### פֿאַרפּשוטערטע פֿאַרוואַלטונג {#simplified-management}

- קיין קאָנפֿיגוראַציע־טעקעס צו רעדאַגירן
- קיין מאַנועלע פֿעיִקייט־רעגיסטראַציע
- פֿעיִקייט־זעבאַרקייט־קאָנטראָלן אַרבעטן אויטאָמאַטיש
- אַדמין־מעלדונגען וואָרענען אײַך וועגן נישט־קלאַסיפֿיצירטע פֿעיִקייטן

### בעסערע פאָרשטעלונג {#better-performance}

- פֿעיִקייטן ווערן געקאַשט
- Lazy-loaded אויף פֿאַרלאַנג
- אָפּטימיזירט פֿאַר WordPress 7.0+

## מיגראַציע־וועג {#migration-path}

### אויב איר זענט אויף WordPress 6.x {#if-youre-on-wordpress-6x}

1. **אַפּגריידט צו WordPress 7.0+** (ווען איר זענט גרייט)
2. **דערהײַנטיקט Superdav AI Agent** צו v1.12.0+
3. **טוישט דריט־פּאַרטיי־מאָדוס צו אויטאָ** (אָפּציאָנעל; Legacy־מאָדוס אַרבעט נאָך אַלץ)
4. **איבערקוקט פֿעיִקייט־זעבאַרקייט** צו פֿאַרזיכערן ריכטיקע צוטריט־קאָנטראָלן

### אויב איר זענט אויף WordPress 7.0+ {#if-youre-on-wordpress-70}

1. **דערהײַנטיקט Superdav AI Agent** צו v1.12.0+
2. **באַשטעטיקט אַז דריט־פּאַרטיי־מאָדוס איז געשטעלט אויף אויטאָ** (אַזוי זאָל עס זײַן לויטן פֿעליקייט)
3. **איבערקוקט פֿעיִקייט־זעבאַרקייט** צו פֿאַרזיכערן ריכטיקע צוטריט־קאָנטראָלן
4. **טעסט דריט־פּאַרטיי־פֿעיִקייטן** צו באַשטעטיקן אַז זיי אַרבעטן

## פּראָבלעם־לייזונג {#troubleshooting}

### פֿעיִקייטן ווערן נישט געלאָדן אין אויטאָ־מאָדוס {#abilities-arent-loading-in-auto-mode}

- באַשטעטיקט אַז איר זענט אויף WordPress 7.0+
- קאָנטראָלירט אַז דריט־פּאַרטיי־מאָדוס איז געשטעלט אויף "אויטאָ"
- באַשטעטיקט אַז דער plugin וואָס גיט די פֿעיִקייט איז אַקטיוו
- קאָנטראָלירט WordPress טעות־לאָגס פֿאַר רעגיסטראַציע־טעותים

### איך וויל האַלטן Legacy־מאָדוס {#i-want-to-keep-legacy-mode}

- גייט צו **סעטטינגס** → **דריט־פּאַרטיי־מאָדוס**
- קלײַבט אויס **Legacy**
- קליקט **אויפֿהיטן**
- Legacy־מאָדוס וועט ווײַטער אַרבעטן

### מײַנע אייגענע פֿעיִקייטן ווײַזן זיך נישט {#my-custom-abilities-arent-showing}

- באַשטעטיקט אַז זיי זענען רעגיסטרירט דורך WordPress hooks
- קאָנטראָלירט אַז זיי אימפּלאַמענטירן דעם Abilities API ריכטיק
- איבערקוקט WordPress טעות־לאָגס
- ניצט די **פֿעיִקייט־זעבאַרקייט** אַדמין־בלאַט צו זען אַלע רעגיסטרירטע פֿעיִקייטן

### איך באַקומען "נישט־קלאַסיפֿיצירטע פֿעיִקייט" מעלדונגען {#im-getting-unclassified-ability-notices}

- דאָס איז נאָרמאַל פֿאַר נײַע דריט־פּאַרטיי־פֿעיִקייטן
- איבערקוקט און קלאַסיפֿיצירט זיי אין דער אַדמין־מעלדונג
- זעט **פֿעיִקייט־זעבאַרקייט** פֿאַר פּרטים וועגן קלאַסיפֿיקאַציע

## צוריק־קאָמפּאַטיביליטעט {#backward-compatibility}

### עקזיסטירנדיקע קאָנפֿיגוראַציעס {#existing-configurations}

אויב איר האָט עקזיסטירנדיקע דריט־פּאַרטיי־פֿעיִקייט־קאָנפֿיגוראַציעס:

- **אַלטער מאָדוס**: אייער קאַנפֿיגוראַציע אַרבעט ווייטער
- **מאַנועלער מאָדוס**: אייער קאַנפֿיגוראַציע אַרבעט ווייטער
- **אויטאָ־מאָדוס**: אייער קאַנפֿיגוראַציע ווערט איגנאָרירט (אויטאָ־מאָדוס נעמט איבער)

כּדי צו האַלטן אייער אייגענע קאַנפֿיגוראַציע, בלײַבט אין מאַנועלער אָדער אַלטער מאָדוס.

### צײַטליניע פֿון אַרויסנעמען פֿון באַניץ {#deprecation-timeline}

- **v1.12.0**: אַלטער און מאַנועלער מאָדוסן ווערן נאָך גאָר געשטיצט
- **v1.13.0+**: אַלטער מאָדוס קען ווײַזן מודעות וועגן אַרויסנעמען פֿון באַניץ
- **v2.0.0**: אַלטער מאָדוס קען ווערן אַוועקגענומען (נאָך נישט באַשטימט)

## בעסטע פּראַקטיקן {#best-practices}

### פֿאַר נײַע אינסטאַלאַציעס {#for-new-installations}

- ניצט אויטאָ־מאָדוס (ער איז דער פעליקייט־ווערט)
- לאָזט Superdav AI Agent אַנטדעקן פֿעיִקייטן אויטאָמאַטיש
- ניצט פֿעיִקייט־זעבאַרקייט צו קאָנטראָלירן צוטריט

### פֿאַר עקזיסטירנדיקע אינסטאַלאַציעס {#for-existing-installations}

- אַפּגריידט צו WordPress 7.0+ ווען מעגלעך
- באַשטימט זיך איבער צו אויטאָ־מאָדוס פֿאַר פֿאַרפּשוטערטע פֿאַרוואַלטונג
- איבערקוקט און קלאַסיפֿיצירט פֿעיִקייטן מיט פֿעיִקייט־זעבאַרקייט

### פֿאַר אייגענע פֿעיִקייטן {#for-custom-abilities}

- רעגיסטרירט פֿעיִקייטן דורך WordPress hooks (Abilities API)
- פֿאַרמײַדט אייגענע פֿעיִקייט־רעגיסטרן
- טעסט אויף WordPress 7.0+ מיט אויטאָ־מאָדוס

## קומענדיקע שריט {#next-steps}

1. **קוקט nach אייער WordPress ווערסיע**: באַשטעטיקט אַז איר זענט אויף 7.0+ פֿאַר אויטאָ־מאָדוס
2. **איבערקוקט אייער דריט־פּאַרטיי־מאָדוס**: גייט צו Settings און קאָנטראָלירט אייער איצטיקן מאָדוס
3. **דערהײַנטיקט אויב נייטיק**: באַשטימט זיך איבער צו אויטאָ־מאָדוס אויב איר זענט אויף WordPress 7.0+
4. **קלאַסיפֿיצירט פֿעיִקייטן**: איבערקוקט און קלאַסיפֿיצירט יעדע נישט־קלאַסיפֿיצירטע פֿעיִקייט
5. **טעסט**: באַשטעטיקט אַז אייערע דריט־פּאַרטיי־פֿעיִקייטן אַרבעטן ריכטיק
