# Dashboard איבערבליק {#dashboard-overview}

Ultimate Multisite האָט אַ באַזונדערן **Dashboard** אונטער דעם Ultimate Multisite מעניו, און לייגט אויך צו ווידזשעטן צום נעץ־Dashboard. דער אַרטיקל באַהאַנדלט וואָס איר וועט זען אין ביידע ערטער.

## דער נעץ־Dashboard {#the-network-dashboard}

Ultimate Multisite לייגט אַרײַן נוצלעכע ווידזשעטן אין דעם נעץ־ניוואָ Dashboard וואָס מען געפֿינט בײַ **מײַנע זייטלעך → נעץ־אַדמיניסטראַציע → Dashboard**.

![נעץ־Dashboard מיט Ultimate Multisite ווידזשעטן](/img/admin/network-dashboard-full.png)

דאָ איז וואָס יעדער ווידזשעט גיט:

**ערשטע שריט:** דערשיינט אויף אַ פרישער אינסטאַלאַציע און דערמאָנט אײַך צו פֿאַרענדיקן אויפֿשטעל־אויפֿגאַבעס ווי קאָנפֿיגורירן אַ צאָלונג־מעטאָד און שאַפֿן אַ פּראָבע־Account.

![ערשטע שריט ווידזשעט](/img/admin/network-dashboard-first-steps.png)

**קיצער:** מעלדט די צאָל רעגיסטראַציעס און הכנסה פֿאַרן טאָג. דער MRR (חודשלעכע איבערקערנדיקע הכנסה) ווײַזט אָן אַ פּראָיעקטירטן סך־הכּל איבער קאַסטאַמערס מיט הכנסה־שאַפֿנדיקע מיטגלידערשאַפֿטן.

![קיצער ווידזשעט](/img/admin/network-dashboard-summary.png)

**אַקטיוויטעט־שטראָם:** מעלדט וועגן געשעענישן איבער Ultimate Multisite — רעגיסטראַציעס, אַנולירונגען, שאַפֿן זייטלעך, אויסמעקן, און אַנדערע אַקטיוויטעטן.

![אַקטיוויטעט־שטראָם ווידזשעט](/img/admin/network-dashboard-activity-stream.png)

**גלײַך איצט:** ווײַזט אַ קורצן קיצער פֿון באַניצער און זייטלעך אין דער נעץ מיט דורכוועגן צו שאַפֿן נײַע זייטלעך אָדער באַניצער.

![גלײַך איצט ווידזשעט](/img/admin/network-dashboard-right-now.png)

**נײַעס און דיסקוסיעס:** ברענגט און ווײַזט די לעצטע Ultimate Multisite נײַעס אַרײַנגערעכנט דערהײַנטיקונגען, קריטישע פֿעלערן, און זיכערהייט־פֿאַרריכטונגען.

![נײַעס ווידזשעט](/img/admin/network-dashboard-news.png)

## דער Ultimate Multisite Dashboard {#the-ultimate-multisite-dashboard}

בשעת דער נעץ־Dashboard שטעלט פֿאָר גאַנצהייטלעכע אינפֿאָרמאַציע וועגן דער נעץ, שטעלט דער Ultimate Multisite Dashboard (אויפֿן העכסטן ניוואָ פונעם Ultimate Multisite מעניו) פֿאָר אינפֿאָרמאַציע וועגן דעם דינסט.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**וואַקסטום פֿון חודשלעכער איבערקערנדיקער הכנסה:** ווײַזט אַ חודשלעכן צעשייד פֿון הכנסה איבער אַ 12־חדשים קאַלענדאַר־פּעריאָד. נײַע הכנסה ווערט געוויזן אין בלוי, בעת אַנולירונגען דערשײַנען אין רויט.

![MRR וואַקסטום דיאַגראַם](/img/admin/um-dashboard-mrr.png)

**שטיצע פֿאַר דאַטום־ראַנג:** פֿילטרירט Dashboard דאַטן לויט צײַט־פּעריאָד כּדי זיך צו קאָנצענטרירן אויף דער אינפֿאָרמאַציע וואָס איר דאַרפֿט. דער דאַטום־ראַנג קאָנטראָלירט וואָס ווערט געוויזן אין די ווידזשעטן אונטן.

![דאַטום־ראַנג און Dashboard סטאַטיסטיק](/img/admin/um-dashboard-date-range.png)

די Dashboard ווידזשעטן אַרײַננעמען:

- **הכנסה** — גאַנץ הכנסה פֿאַרן פּעריאָד מיט MRR און צעשיידונגען לויט פּראָדוקט
- **נײַע מיטגלידערשאַפֿטן** — וויפֿל נײַע מיטגלידערשאַפֿטן זענען געשאַפֿן געוואָרן אין דעם אויסגעקליבענעם פּעריאָד
- **רעגיסטראַציעס לויט לענדער** — געאָגראַפֿישער צעשייד פֿון רעגיסטראַציעס
- **רעגיסטראַציעס לויט פֿאָרמען** — וויפֿל קאַסטאַמערס האָבן זיך רעגיסטרירט דורך יעדער checkout פֿאָרעם קעגן צוגעלייגט דורך אַדמיניסטראַטאָרן
- **מערסט באַזוכטע זייטלעך** — די מערסט באַזוכטע זייטלעך אויף אײַער נעץ (פֿאָדערט אײַנשאַלטן ציילן באַזוכן בײַ **Ultimate Multisite → אײַנשטעלונגען → זייטלעך → אײַנשאַלטן באַגרענעצונג און ציילונג פֿון באַזוכן**)

![Dashboard סטאַטיסטיק](/img/admin/um-dashboard-stats.png)
