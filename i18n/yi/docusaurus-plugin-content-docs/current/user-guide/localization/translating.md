# איבערזעצן Ultimate Multisite

Ultimate Multisite ווערט אנטוויקלט אויף ענגליש, אָבער דער plugin איז פולשטענדיק איבערזעצלעך, און מיר באַגריסן אַקטיוו בײַשטײַערונגען צו איבערזעצונגען.

## איבערזעצונגען פֿונעם האַרץ־plugin

דער האַרץ־plugin פֿון Ultimate Multisite ווערט איבערגעזעצט דורך דער אָפֿיציעלער **WordPress.org איבערזעצונג־פּלאַטפֿאָרמע**:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

דאָס איז דער סטאַנדאַרטער WordPress קהילה־איבערזעצונג־סיסטעם. ווען איבערזעצונגען דערגרייכן אַ גענוגן באַשטעטיקונג־שוועל, ווערן זיי אויטאָמאַטיש פֿאַרשפּרייט צו אַלע Ultimate Multisite באַניצער דורך WordPress plugin דערהײַנטיקונגען.

**מיר דאַרפֿן אײַער הילף!** אויב איר רעדט אַ שפּראַך אַנדערש ווי ענגליש, ביטע באַטראַכט בײַצושטײַערן איבערזעצונגען. אַפֿילו איבערזעצן עטלעכע שטריקלעך העלפֿט. כּדי אָנהייבן:

1. באַזוכט [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)
2. קלײַבט אויס אײַער שפּראַך
3. קליקט "בײַשטײַערן איבערזעצונג"
4. איר וועט דאַרפֿן אַ פֿרײַען WordPress.org Account

אויב איר ווילט ווערן אַן איבערזעצונג־רעדאַקטאָר פֿאַר אײַער שפּראַך (קענען באַשטעטיקן איבערזעצונגען), meldet זיך דורך [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions).

די Addon איבערזעצונגען זענען דערווײַל דזשענערירט דורך AI ווי אַן אָנהייב־פּונקט. זיי דעקן אַלע געשטיצטע שפּראַכן, אָבער קענען אַנטהאַלטן אומבאַקוועמע פֿראַזירונג אָדער אומפּינקטלעכקייטן. מיר באַגריסן מענטשלעכע איבערזעצער איבערצוקוקן און פֿאָרשלאָגן פֿאַרבעסערונגען, כּדי מאַכן זיי מער נאַטירלעך און פּינקטלעך.

## לאָקאַלע איבערזעצונגען

אויב איר פּרעפֿערירט צו שאַפֿן איבערזעצונגען לאָקאַל פֿאַר אײַער אייגענעם פּראָיעקט אַנשטאָט בײַצושטײַערן אַרויף־סטראָם, זענען דאָ צוויי אָפּציעס:

### ניצן Poedit

איר קענט ניצן די דעסקטאָפּ־אַפּליקאַציע **[Poedit](https://poedit.net)** צו איבערזעצן די `.pot` טעקע:

1. אַראָפּלאָדן Poedit פֿון [poedit.net](https://poedit.net)
2. עפֿנט די `.pot` טעקע פֿון `wp-content/plugins/ultimate-multisite/lang/`
3. איבערזעצט די שטריקלעך אין דעם Poedit צובינד
4. היט אָפּ -- Poedit דזשענערירט `.po` און `.mo` טעקעס
5. לייגט די רעזולטאַט־טעקעס אין דעם `lang` טעקע־אָרדנער

### ניצן Loco Translate

דער **[Loco Translate](https://wordpress.org/plugins/loco-translate/)** WordPress plugin לאָזט אײַך איבערזעצן גלײַך פֿון דער אַדמין־פּאַנעל אָן קיין דעסקטאָפּ־געצייג. ער באַהאַנדלט שאַפֿן און קאָמפּילירן איבערזעצונג־טעקעס פֿאַר אײַך.
