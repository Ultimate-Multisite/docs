# תרגום Ultimate Multisite

Ultimate Multisite פותח באנגלית, אך התוסף ניתן לתרגום מלא ואנחנו מקבלים בשמחה תרומות תרגום.

## תרגומי התוסף הראשי

התוסף הראשי של Ultimate Multisite מתורגם דרך **פלטפורמת התרגום הרשמית של WordPress.org**:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

זוהי מערכת התרגום הקהילתית הסטנדרטית של WordPress. כאשר תרגומים מגיעים לסף אישור מספק, הם מופצים אוטומטית לכל משתמשי Ultimate Multisite דרך עדכוני התוסף של WordPress.

**אנחנו צריכים את עזרתכם!** אם אתם דוברים שפה נוספת מלבד אנגלית, אנא שקלו לתרום תרגומים. גם תרגום של כמה מחרוזות עוזר. כדי להתחיל:

1. בקרו ב-[translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)
2. בחרו את השפה שלכם
3. לחצו על "Contribute Translation"
4. תצטרכו חשבון WordPress.org חינמי

אם תרצו להפוך לעורכי תרגום לשפה שלכם (עם יכולת לאשר תרגומים), פנו אלינו דרך [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions).

## תרגומי Addon

תוספי Addon מתורגמים בנפרד ב:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

תרגומי ה-Addon נוצרו כרגע באמצעות בינה מלאכותית כנקודת התחלה. הם מכסים את כל השפות הנתמכות אך עשויים להכיל ניסוחים מגושמים או אי-דיוקים. אנו מזמינים מתרגמים אנושיים לסקור ולהציע שיפורים כדי להפוך אותם לטבעיים ומדויקים יותר.

## תרגומים מקומיים

אם אתם מעדיפים ליצור תרגומים מקומיים לפרויקט שלכם במקום לתרום לפרויקט הראשי, יש שתי אפשרויות:

### שימוש ב-Poedit

ניתן להשתמש באפליקציית שולחן העבודה **[Poedit](https://poedit.net)** לתרגום קובץ ה-`.pot`:

1. הורידו את Poedit מ-[poedit.net](https://poedit.net)
2. פתחו את קובץ ה-`.pot` מ-`wp-content/plugins/ultimate-multisite/lang/`
3. תרגמו את המחרוזות בממשק של Poedit
4. שמרו -- Poedit יוצר קבצי `.po` ו-`.mo`
5. מקמו את קבצי הפלט בתיקיית `lang`

### שימוש ב-Loco Translate

תוסף ה-WordPress **[Loco Translate](https://wordpress.org/plugins/loco-translate/)** מאפשר לכם לתרגם ישירות מלוח הבקרה ללא צורך בכלי שולחן עבודה. הוא מטפל עבורכם ביצירה ובקומפילציה של קבצי התרגום.
