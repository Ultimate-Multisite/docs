# כיצד ניתן לבטל את המנוי שלי?

אם יש לכם רישיון Ultimate Multisite, תוכלו לבטל את החידוש שלו בכל עת. פשוט עקבו אחר השלבים הבאים:


<!-- Screenshot unavailable: Account page email with access link -->

השתמשו בכתובת האימייל שסיפקתם בעת הרכישה ובסיסמה שלכם.

<!-- Screenshot unavailable: Login page with email and password fields -->

זהו המסך הראשוני של עמוד החשבון שלכם:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

כדי לבטל את המנוי, בתפריט שבצד שמאל, עברו אל _Renewing & Billing._

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

לחצו על החץ בצד ימין כדי לפתוח חלון צדדי. לאחר מכן, בחרו באפשרות שתבטל את **החידוש האוטומטי**.

המערכת תציג לכם הודעת אישור.

אם אתם בטוחים שברצונכם לבטל את המנוי, פשוט לחצו על הכפתור _**Cancel Renewals**_. לאחר פעולה זו, תתבקשו לענות על סקר קצר.

זהו! המנוי שלכם לא יחודש באופן אוטומטי.

**יהיה לכם מפתח תקף עד לפקיעת המנוי**. במידה ותרצו להפעיל מחדש את המנוי, תצטרכו לעשות זאת באופן ידני.
