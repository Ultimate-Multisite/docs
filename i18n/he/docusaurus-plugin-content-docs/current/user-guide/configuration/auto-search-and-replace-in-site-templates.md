# חיפוש והחלפה אוטומטיים בתבניות אתר (v2)

_**מדריך זה מתאים לגרסה 2.x של WP Ultimo.**_

אחת התכונות החזקות ביותר של Ultimate Multisite היא היכולת להוסיף שדות טקסט, צבע ובחירה לטופס ההרשמה. לאחר שאספנו את הנתונים האלה, נוכל להשתמש בהם כדי למלא מראש תוכן בחלקים מסוימים של תבנית האתר שנבחרה. כשהאתר החדש מתפרסם, Ultimate Multisite יחליף את הממלאי מקום (placeholders) במידע האמיתי שהוזן בזמן ההרשמה.

לדוגמה, אפשר ליצור את אתרי התבנית שלכם עם ממלאי מקום. יש להוסיף את ממלאי המקום בתוך סוגריים מסולסלים כפולים - {{placeholder_name}}.

לאחר מכן, פשוט הוסיפו שדה הרשמה תואם כדי לאסוף את המידע הזה

הלקוחות שלכם יוכלו למלא את השדה הזה במהלך ההרשמה.

Ultimate Multisite יחליף אוטומטית את ממלאי המקום בנתונים שהלקוח סיפק.

## **פתרון לבעיית "התבנית מלאה בממלאי מקום"**

כל זה נהדר, אבל נתקלים בבעיה מעצבנת: עכשיו תבניות האתר שלנו - שהלקוחות יכולים לבקר בהן - מלאות בממלאי מקום מכוערים שלא אומרים הרבה.

כדי לפתור את זה, אנחנו מציעים אפשרות להגדיר ערכים זמניים לממלאי המקום, ומשתמשים בערכים האלה כדי לחפש ולהחליף את התוכן באתרי התבנית בזמן שהלקוחות מבקרים בהם.

כדי לגשת לעורך ממלאי המקום של התבנית, היכנסו אל **Ultimate Multisite > Settings > Sites**, ואז בסרגל הצד לחצו על הקישור Edit Placeholders.

![קישור Edit Placeholders בסרגל הצד של הגדרות Sites](/img/config/settings-sites.png)

זה יעביר אתכם לעורך התוכן של ממלאי המקום, שם תוכלו להוסיף ממלאי מקום ואת הערכים המתאימים להם.

![עורך תוכן ממלאי מקום עם שמות וערכים](/img/config/settings-sites.png)
