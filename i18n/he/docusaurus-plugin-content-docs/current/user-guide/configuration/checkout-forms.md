# טופסי תשלום

טופסי תשלום הם דרך קלה וגמישה להתנסות בגישות שונות כשמנסים להמיר גולשים ללקוחות משלמים.

Ultimate Multisite 2.0 מציע עורך טופסי תשלום שמאפשר לכם ליצור כמה טפסים שתרצו, עם שדות שונים, מוצרים מוצעים ועוד.

כדי לגשת לתכונה הזו, עברו לתפריט Checkout Forms בסרגל הצד השמאלי.

![רשימת טופסי תשלום](/img/config/checkout-forms-list.png)

בעמוד זה תוכלו לראות את כל טופסי התשלום שיש לכם.

אם אתם רוצים ליצור טופס חדש, פשוט לחצו על Add Checkout Form בראש העמוד.

תוכלו לבחור באחת משלוש האפשרויות כנקודת התחלה: צעד בודד, מרובה צעדים או ריק. לאחר מכן, לחצו כדי לעבור לעורך.

![עורך טופס תשלום](/img/config/checkout-form-editor.png)

לחלופין, תוכלו לערוך או לשכפל טפסים קיימים על ידי לחיצה על האפשרויות מתחת לשם הטופס. שם תמצאו גם אפשרויות להעתקת ה-shortcode של הטופס או למחיקתו.

![פעולות בריחוף מעל טופס תשלום](/img/config/checkout-form-hover-actions.png)

### עריכת טופס תשלום

תוכלו ליצור טופסי תשלום למטרות שונות. בדוגמה זו נעבוד על טופס הרשמה.

לאחר שנכנסתם לעורך טופס התשלום, תנו לטופס שם (שישמש לשימוש פנימי בלבד) ו-slug (שמשמש ליצירת shortcodes, למשל).

![עורך טופס תשלום](/img/config/checkout-form-editor.png)

טפסים מורכבים משלבים ושדות. תוכלו להוסיף שלב חדש על ידי לחיצה על Add New Checkout Step.

![הוספת שלב תשלום חדש](/img/config/checkout-form-add-step.png)

בלשונית הראשונה של החלון, מלאו את תוכן השלב בטופס. תנו לו מזהה (ID), שם ותיאור. פריטים אלה משמשים בעיקר לשימוש פנימי.

![שלב בטופס תשלום](/img/config/checkout-form-step.png)

לאחר מכן, הגדירו את הנראות של השלב. תוכלו לבחור בין "הצג תמיד", "הצג רק למשתמשים מחוברים" או "הצג רק לאורחים".

![שלב בטופס תשלום](/img/config/checkout-form-step.png)

לבסוף, הגדירו את עיצוב השלב. אלה שדות אופציונליים.

![שלב בטופס תשלום](/img/config/checkout-form-step.png)

עכשיו, הגיע הזמן להוסיף שדות לשלב הראשון שלנו. פשוט לחצו על Add New Field ובחרו את סוג החלק שאתם רוצים.

![שלב בטופס תשלום עם שדות](/img/config/checkout-form-step.png)

לכל שדה יש פרמטרים שונים למילוי. לשדה הראשון, נבחר בשדה "Username".

![שלב בטופס תשלום](/img/config/checkout-form-step.png)

![שלב בטופס תשלום](/img/config/checkout-form-step.png)

![שלב בטופס תשלום](/img/config/checkout-form-step.png)

תוכלו להוסיף כמה שלבים ושדות שתצטרכו. כדי להציג את המוצרים שלכם ללקוחות לבחירה, השתמשו בשדה Pricing Table. אם אתם רוצים לאפשר ללקוחות לבחור תבנית, הוסיפו את שדה Template Selection. וכן הלאה.

_**שימו לב:** שם משתמש, אימייל, סיסמה, כותרת האתר, כתובת האתר, סיכום הזמנה, תשלום וכפתור שליחה הם שדות חובה ליצירת טופס תשלום._

בזמן שאתם עובדים על טופס התשלום, תוכלו תמיד להשתמש בכפתור Preview כדי לראות איך הלקוחות יראו את הטופס. תוכלו גם לעבור בין תצוגה כמשתמש קיים או כמבקר.

![שמירת טופס תשלום](/img/config/checkout-form-save.png)

![עורך טופס תשלום](/img/config/checkout-form-editor.png)

לבסוף, ב-Advanced Options תוכלו להגדיר את ההודעה לעמוד "תודה", להוסיף קטעי קוד למעקב אחר המרות, להוסיף CSS מותאם אישית לטופס התשלום או להגביל אותו למדינות מסוימות.

![אפשרויות מתקדמות](/img/config/checkout-form-advanced.png)

תוכלו גם להפעיל או לבטל ידנית את טופס התשלום על ידי שינוי המתג בעמודה הימנית, או למחוק את הטופס לצמיתות.

![מתג הפעלה](/img/config/checkout-form-active.png)

אל תשכחו לשמור את טופס התשלום!

![כפתור שמירה](/img/config/checkout-form-save.png)

כדי לקבל את ה-shortcode של הטופס, לחצו על Generate Shortcode והעתיקו את התוצאה שמוצגת בחלון.

![כפתור שמירה עם shortcode](/img/config/checkout-form-save.png)
