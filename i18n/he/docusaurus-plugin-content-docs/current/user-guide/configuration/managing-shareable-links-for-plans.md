# ניהול קישורים לשיתוף עבור תוכניות

Ultimate Multisite מאפשר לכם לשתף קישורים לתוכניות ישירות עם הלקוחות שלכם. כדי לעשות זאת, גשו לעמוד העריכה של אחת מהתוכניות שלכם ולחצו על הכפתור _לחצו להעתקת הקישור לשיתוף_.

![עמוד עריכת המוצר עם כפתור הקישור לשיתוף](/img/config/product-edit-full.png)

_בעמוד העריכה של התוכנית תוכלו למצוא את הכפתור ליצירת קישורים לשיתוף_

שימו לב שמנגנון זה משתמש בתדירות החיוב ברירת המחדל לבניית הקישור. המשמעות היא שאם ברירת המחדל שלכם היא חיוב חודשי, למשל, בסופו של דבר יהיה לכם קישור כזה: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

אם תרצו שהקישור לשיתוף יפנה כברירת מחדל לתוכנית רבעונית, תצטרכו להוסיף 3 בסוף

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

כדי לבחור את התוכנית _premium_ במחזור שנתי, הוסיפו 1 ולאחר מכן את המילה:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

זכרו שלא ניתן לקבל קישור אחד לתדירויות חיוב שונות. עליכם לשתף קישור ספציפי לכל אחת מתדירויות החיוב שלכם.

ברגע שתשתפו את הקישור עם הלקוחות שלכם, הוא יפנה אותם לשלב הראשון בתהליך ההרשמה שלכם.
