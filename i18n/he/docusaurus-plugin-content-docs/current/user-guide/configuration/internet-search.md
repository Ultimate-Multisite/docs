# חיפוש אינטרנט

תוסף Gratis AI Agent v1.5.0 מוסיף יכולת **חיפוש אינטרנט** המאפשרת לסייע ה-AI לשלוף מידע עדכני מהאינטרנט במהלך שיחה. יכולת זו מופעלת על ידי [Brave Search API](https://brave.com/search/api/) או [Tavily API](https://tavily.com/).

## איך זה עובד

כאשר חיפוש האינטרנט מופעל, הסייע יכול לבצע באופן אוטומטי שאילתה אצל ספק החיפוש שהוגדר עבורך, כאשר הוא מזהה ששאלה דורשת מידע עדכני או חיצוני – לדוגמה, חדשות אחרובות, תמחור בזמן אמת, או תיעוד שייתכן ששונה מאז תאריך הקיטוע של המודל.

התוצאות נשלפות בזמן אמת ומוזרקות להקשר של הסייע לפני שהוא מייצר תגובה. הסייע יציין מתי השתמש בתוצאות החיפוש כדי לענות על שאלה.

## הפעלת חיפוש אינטרנט

חיפוש אינטרנט דורש מפתח API מספק החיפוש שבחרת. כדי להגדיר אותו:

1. עבור אל **Gratis AI Agent → Settings → Advanced**.
2. אתר את תפריט הנפתח **Internet Search Provider** ובחר בין **Brave Search** או **Tavily**.
3. הזן את מפתח ה-API שלך בשדה המתאים. קישורי ההרשמה מוצגים ליד כל שדה.
4. לחץ על **Save Settings**.

לאחר שמירת המפתח, יכולת חיפוש האינטרנט זמינה אוטומטית לסייע.

## Brave Search

### השגת מפתח Brave Search API

1. בקר בדף [Brave Search API](https://brave.com/search/api/).
2. הירשם לתכנית. זמין תוכנית חינמית עם מגבלת בקשות חודשית.
3. העתק את מפתח ה-API שלך מדשבורד המפתחים של Brave Search.
4. הדבק אותו בשדה **Brave Search API Key** בהגדרות Gratis AI Agent.

### מגבלות שימוש

השימוש מחויב על ידי Brave Search בהתבסס על מספר השאילתות שבוצעו. כל תגובת AI המפעילה חיפוש נספרת כשאילתה אחת. עקוב אחר השימוש שלך ב-[Brave Search developer dashboard](https://brave.com/search/api/) כדי למנוע חיובים בלתי צפויים.

## Tavily

Superdav AI Agent v1.10.0 מוסיף את **Tavily** כאפשרות אלטרנטיבית לחיפוש אינטרנט, ומציע תוצאות חיפוש עשירות יותר ויכולות מחקר מתקדמות.

### השגת מפתח Tavily API

1. בקר בדף [Tavily API](https://tavily.com/).
2. הירשם לחשבון. זמין תוכנית חינמית עם מגבלת בקשות חודשית.
3. העתק את מפתח ה-API שלך מדשבורד Tavily.
4. הדבק אותו בשדה **Tavily API Key** בהגדרות Gratis AI Agent.

### מגבלות שימוש

השימוש מחויב על ידי Tavily בהתבסס על מספר קריאות ה-API שבוצעו. כל תגובת AI המפעילה חיפוש נספרת כקריאה אחת. עקוב אחר השימוש שלך ב-[Tavily dashboard](https://tavily.com/) כדי למנוע חיובים בלתי צפויים.

## השבתת חיפוש אינטרנט

הסר את מפתח ה-API משדה ספק החיפוש הפעיל ושמור. יכולת חיפוש האינטרנט לא תהיה זמינה עוד לסייע.

:::note
חיפוש אינטרנט מוסיף השהייה לתגובות מכיוון שהסייע חייב להמתין לתוצאות החיפוש לפני שהוא מייצר תשובה. עבור מקרי שימוש רגישים לזמן, שקול אם חיפוש בזמן אמת הכרחי או שמידע הליבה של הסייע מספיק.
:::
