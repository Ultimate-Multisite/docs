# מיומנויות מותאמות לנושא

Superdav AI Agent v1.10.0 מגיע עם ארבע מיומנויות מובנות חדשות, המותאמות לנושא (Theme-aware) ומתאימות אוטומטית לנושא וורדפרס הפעיל אצלכם. מיומנויות אלו מספקות הנחיות ויכולות מיוחדות המותאמות לארכיטקטורה ולתכונות של הנושא שלכם.

## מהן מיומנויות מותאמות לנושא?

מיומנויות מותאמות לנושא הן בסיסי ידע ומערכות כלים מוגדרות מראש, שהעוזרת ה-AI בוחרת באופן אוטומטי בהתבסס על נושא וורדפרס הפעיל באתר שלכם. כשאתם מחליפים נושאים, המיומנויות הזמינות של העוזרת מתעדכנות אוטומטית – אין צורך בהגדרה ידנית.

כל מיומנות כוללת:

- **תיעוד ספציפי לנושא** — הנחיות על שימוש בהתאמה אישית של הנושא.
- **הפניות לבלוקים ודפוסים** — בלוקים, דפוסים ואפשרויות עיצוב זמינות.
- **דוגמאות התאמה אישית** — קטעי קוד ודפוסי הגדרה למשימות נפוצות.
- **שיטות עבודה מומלצות** — המלצות לגבי ארכיטקטורת הנושא וזרימת העבודה שלו.

## מיומנויות מותאמות לנושא זמינות

### Block Themes (נושאי בלוקים)

**מתאים ל:** נושאים המשתמשים בארכיטקטורה מבוססת בלוקים של וורדפרס (Full Site Editing).

מיומנות Block Themes מספקת הדרכה בנושא:

- יצירה ועריכה של תבניות באמצעות עורך הבלוקים.
- עבודה עם דפוסי בלוקים ובלוקים הניתנים לשימוש חוזר.
- התאמה אישית של סגנונות גלובליים והגדרות theme.json.
- שימוש בבלוקים ווריאציות של נושא.
- בניית דפוסי בלוקים מותאמים אישית לאתר שלכם.

**מופעל אוטומטית כאשר:** הנושא הפעיל שלכם הוא נושא בלוקים (תומך בתכונה `block-templates`).

### Classic Themes (נושאי קלאסיים)

**מתאים ל:** נושאי וורדפרס מסורתיים המשתמשים בתבניות PHP ובעורך הקלאסי.

מיומנות Classic Themes מספקת הדרכה בנושא:

- עבודה עם קבצי תבניות PHP ו-hooks.
- התאמה אישית של מראה הנושא דרך ה-Customizer.
- שימוש באזורי ווידג'טים וסיידברים.
- שינוי CSS ופיתוח באמצעות נושא בן (child theme).
- הבנת היררכיית הנושא ותגיות התבנית.

**מופעל אוטומטית כאשר:** הנושא הפעיל שלכם הוא נושא קלאסי (לא מבוסס בלוקים).

### Kadence Blocks

**מתאים ל:** אתרים המשתמשים בתוסף Kadence Blocks לעיצוב מתקדם מבוסס בלוקים.

מיומנות Kadence Blocks מספקת הדרכה בנושא:

- שימוש בספריית הבלוקים המתקדמת של Kadence (Hero, Testimonials, Pricing וכו').
- הגדרת הגדרות בלוקים ואפשרויות רספונסיביות של Kadence.
- בניית פריסות מותאמות אישית באמצעות בלוקי הגריד והקונטיינר של Kadence.
- שילוב בלוקים של Kadence עם הנושא שלכם.
- מינוף מערכת העיצוב והפסיקטים של Kadence.

**מופעל אוטומטית כאשר:** תוסף Kadence Blocks פעיל באתר שלכם.

### Kadence Theme

**מתאים ל:** אתרים המשתמשים בנושא Kadence לעיצוב והתאמה אישית מבוססת בלוקים.

מיומנות Kadence Theme מספקת הדרכה בנושא:

- התאמה אישית של נושא Kadence באמצעות סגנונות גלובליים ו-theme.json.
- שימוש בדפוסים ובמבנים המובנים של Kadence.
- הגדרת הגדרות ואפשרויות של נושא Kadence.
- בניית עיצובים מותאמים אישית באמצעות מערכת העיצוב של Kadence.
- שילוב Kadence עם תוספים וכלים פופולריים.

**מופעל אוטומטית כאשר:** נושא Kadence הוא הנושא הפעיל שלכם.

## כיצד נבחרות המיומנויות

העוזרת מזהה אוטומטית את הנושא הפעיל והתוספים המותקנים בכל הודעה. אם קיימת מיומנות מותאמת לנושא תואמת, היא נטענת אוטומטית להקשר של העוזרת. אינכם צריכים להפעיל או להחליף מיומנויות באופן ידני.

### מספר מיומנויות

אם מספר מיומנויות מתאימות לאתר שלכם (לדוגמה, אם יש לכם גם Kadence Blocks וגם Kadence Theme פעילים), לעוזרת יש גישה לכל המיומנויות הרלוונטיות ויכולה להפנות להנחיות מכל אחת מהן.

### החלפת נושאים

כאשר אתם משנים את הנושא הפעיל שלכם, המיומנויות הזמינות של העוזרת מתעדכנות אוטומטית בהודעה הבאה. לדוגמה:

1. אתם משתמשים בנושא בלוקים עם מיומנות **Block Themes** פעילה.
2. אתם מחליפים לנושא קלאסי.
3. בהודעה הבאה שלכם, מיומנות **Classic Themes** נטענת אוטומטית, ומיומנות **Block Themes** אינה זמינה עוד.

## שימוש במיומנויות מותאמות לנושא

כדי למנף מיומנות מותאמת לנושא, פשוט תארו מה אתם רוצים לעשות בממשק הצ'אט. העוזרת תפנה אוטומטית להנחיות של המיומנות המתאימה.

### דוגמאות הנחיות (Prompts)

**עבור Block Themes:**
> "צור סקשן הגיבור (hero section) עם תמונת רקע וטקסט ממורכז באמצעות דפוסי בלוקים."

**עבור Classic Themes:**
> "הוסף אזור ווידג'טים מותאם אישית לסיידבר באמצעות נושא בן."

**עבור Kadence Blocks:**
> "בנה סקשן המלצות (testimonials) באמצעות בלוק ה-Testimonials של Kadence."

**עבור Kadence Theme:**
> "התאם אישית את פריסת הכותרת (header) ואת עיצוב תפריט הניווט."

העוזרת תספק הדרכה ספציפית לנושא ודוגמאות קוד המותאמות לנושא ולתוספים הפעילים שלכם.

:::note
מיומנויות מותאמות לנושא זמינות אוטומטית ב-Superdav AI Agent v1.10.0 ומעלה. אין צורך בהגדרה או תצורה נוספת.
:::
