# סוכנים מובנים {#built-in-agents}

Gratis AI Agent v1.9.0 מגיע עם חמישה סוכנים מובנים, שכל אחד מהם מוגדר מראש עם סט כלים ממוקד, הנחיית מערכת מותאמת, והצעות התחלה המתאימות למשימות נפוצות בתחום זה. מעבר בין סוכנים משנה מה העוזר יכול לעשות וכיצד הוא מגיב — ללא כל הגדרה מצדך. Superdav AI Agent v1.18.0 יכול להוסיף כלים מודעי לוח זמנים, רשומות תזכורת, שערי אישור, והתראות SMS לתהליכי עבודה אלה כאשר האינטגרציות הקשורות מוגדרות.

## מהו סוכן? {#what-is-an-agent}

כל סוכן הוא פרופיל תצורה בעל שם שמשלב:

- **כלים** — היכולות שהסוכן מורשה להפעיל (לדוגמה, לכותב תוכן יש גישה ליכולות יצירת פוסטים; לסטודיו עיצוב יש גישה ליכולות CSS ו-theme.json)
- **הנחיית מערכת** — הוראות שקובעות את הטון, סדרי העדיפויות והמגבלות של הסוכן
- **הצעות** — הנחיות כתובות מראש שמוצגות בממשק הצ'אט כדי לעזור לך להתחיל במהירות

## גישה לבורר הסוכנים {#accessing-the-agent-picker}

1. פתח את החלונית **Gratis AI Agent** בסרגל הצד של ניהול WordPress.
2. לחץ על **סמל הסוכן** בפינה השמאלית העליונה של כותרת הצ'אט (הסמל משתנה כדי לשקף את הסוכן הפעיל).
3. **בורר הסוכנים** נפתח כשכבת טבלת-טופס. כל סוכן מופיע עם הסמל, השם ותיאור בן שורה אחת שלו.
4. לחץ על שורת סוכן כדי להפעיל אותו. כותרת הצ'אט מתעדכנת מיד.

אפשר גם להחליף סוכנים באמצע שיחה — הנחיית המערכת של הסוכן החדש נכנסת לתוקף מההודעה הבאה.

## חמשת הסוכנים המובנים {#the-five-built-in-agents}

### כותב תוכן {#content-writer}

**מיקוד:** יצירה ועריכה של פוסטים, עמודים וטופסי יצירת קשר.

**כלים זמינים:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`. כאשר אינטגרציות Superdav AI Agent v1.18.0 מופעלות, הקשר לוח שנה מוגדר, שערי אישור, תזכורות וכלי התראות SMS עשויים להיות זמינים גם לתהליכי עבודה מאושרים.

**מה הוא עושה היטב:**
- ניסוח ופרסום פוסטים בבלוג מתוך תקציר או מתווה
- יצירת קבוצות של עמודי נחיתה עבור אתר חדש
- בניית טופסי יצירת קשר ופנייה
- הגדרת תמונות ראשיות בפוסטים מתוך URL או חיפוש
- ניסוח הודעות מעקב לאחר אירוע מתוך הקשר מוגדר של Google Calendar, ואז השהיה לאישור לפני שליחת התראות

**הצעות התחלה:**
- *כתוב פוסט בלוג בן 500 מילים על היתרונות של WordPress multisite.*
- *צור עמוד אודות, שירותים ויצירת קשר ופרסם אותם.*
- *הוסף טופס פנייה להזמנה לעמוד יצירת הקשר.*
- *נסח תזכורת למשתתפים באירוע לוח השנה המוגדר של מחר והמתן לאישור לפני שליחתה.*

---

### בונה אתרים {#site-builder}

**מיקוד:** יצירת אתר מקצה לקצה מתוך הנחיה אחת.

**כלים זמינים:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`. עם Superdav AI Agent v1.18.0, כלים מוגדרים לשירות מנוהל, אישור, תזכורת, לוח שנה ו-SMS עשויים להיות זמינים כאשר מנהלי מערכת מפעילים אותם.

**מה הוא עושה היטב:**
- יצירת תוכנית בניית אתר מרובת שלבים עבור סוג עסק מתואר
- ביצוע כל שלב באופן עצמאי — מבנה, תוכן, ניווט, עיצוב
- התאוששות משגיאות באמצע תוכנית ללא צורך בהתערבות ידנית
- התקנת תוספים מומלצים כחלק מהבנייה
- יצירת טופסי יצירת קשר ישירות מממשק הצ'אט (Superdav AI Agent v1.10.0+)
- תיאום תזכורות השקה או מעקב אחר משתתפים ללא התראות כפולות כאשר שערי אישור ורשומות תזכורת מופעלים

**הצעות התחלה:**
- *בנה אתר תיק עבודות לצילום עם סוג פוסט גלריה, עמוד הזמנה וטופס יצירת קשר.*
- *צור אתר למסעדה עם תפריט מקוון, שעות פתיחה וטופס פנייה להזמנת שולחן.*
- *הגדר אתר לייעוץ עצמאי עם עמודי שירותים, אזור תיק עבודות ובלוג.*
- *הוסף טופס יצירת קשר לעמוד יצירת הקשר באמצעות בונה האתרים.*
- *לאחר אישור רשימת הבדיקה להשקת האתר, שלח תזכורת SMS לאיש הקשר המוגדר של בעל העניין.*

### סטודיו עיצוב {#design-studio}

**מיקוד:** התאמה אישית חזותית — צבעים, טיפוגרפיה, CSS ותבניות בלוקים.

**כלים זמינים:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**מה הוא עושה היטב:**
- החלת קביעות מוגדרות מראש של ערכת עיצוב בעלות שם (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- כוונון עדין של טיפוגרפיה גלובלית ופלטות צבעים דרך theme.json
- הזרקת CSS מותאם אישית לעקיפות ספציפיות למותג
- צילום מסך של עמוד ובדיקתו לאיתור בעיות עיצוב

**הצעות התחלה:**
- *החל את הקביעה המוגדרת מראש warm-editorial ואז הגדר את הצבע הראשי ל-#2d6a4f.*
- *צלם מסך של דף הבית ואמור לי מה היית משפר.*
- *צור תבנית בלוק הירו לשימוש חוזר עם תמונת רקע ברוחב מלא וכותרת ממורכזת.*

### מנהל תוספים {#plugin-manager}

**מיקוד:** גילוי, התקנה וניהול של תוספי WordPress.

**כלים זמינים:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**מה הוא עושה היטב:**
- המלצה על התוסף הטוב ביותר עבור מקרה שימוש מתואר
- התקנת חבילות יכולות מהרישום
- עיון בקטלוג היכולות הזמין לפי קטגוריה

**הצעות התחלה:**
- *מה התוסף הטוב ביותר עבור ספריית חברים?*
- *התקן את חבילת היכולות של WooCommerce.*
- *הצג לי את כל חבילות יכולות המסחר האלקטרוני הזמינות.*

---

### עוזר תמיכה {#support-assistant}

**מיקוד:** מענה לשאלות על תוכן האתר, הגדרות ותצורת WordPress.

**כלים זמינים:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**מה הוא עושה היטב:**
- חיפוש הגדרות ואפשרויות האתר הנוכחיות
- הסבר אילו סוגי פוסטים, טקסונומיות ותפריטים מוגדרים באתר
- מענה על שאלות "מה ההגדרה הזו עושה?" באמצעות קריאת ערכים חיים
- משמש כשכבת אבחון לקריאה בלבד לפני ביצוע שינויים

**הצעות התחלה:**
- *אילו plugins והגדרות פעילים כרגע באתר הזה?*
- *הצג את כל סוגי הפוסטים המותאמים אישית הרשומים באתר הזה.*
- *אילו תפריטי ניווט קיימים והיכן הם משויכים?*

## אינטגרציות אוטומציה של Superdav {#superdav-automation-integrations}

כאשר האינטגרציות של Superdav AI Agent v1.18.0 מוגדרות, סוכנים מובנים יכולים להשתתף בתהליכי אוטומציה בטוחים יותר שמודעים ללוחות זמנים:

- **כלי קריאה של Google Calendar** מאפשרים לסוכנים לבדוק לוחות שנה ואירועים מוגדרים לפני ניסוח עבודת המשך.
- **מיפוי אנשי קשר ומשתתפים** עוזר להתאים משתתפי אירועים למשתמשי WordPress או לאנשי קשר מוכרים.
- **שערי אישור אנושי** משהים פעולות רגישות עד שמשתמש מורשה בודק ומאשר אותן.
- **רשומות תזכורת** מונעות התראות כפולות כאשר משימות מתוזמנות מנסות שוב או חוזרות.
- **התראות SMS של TextBee** שולחות הודעות טקסט מוגדרות רק כאשר פרטי הגישה ל-SMS והרשאות תהליך העבודה מופעלים.

תהליך עבודה מומלץ: בקשו מהסוכן להכין את ההודעה או הפעולה, בדקו את בקשת האישור, ואז אפשרו לפעולה המאושרת להמשיך. עבור תזכורות חוזרות, השאירו הפחתת כפילויות של תזכורות מופעלת כדי שאותו אירוע או איש קשר לא יקבלו התראה שוב ושוב.

## התאמה אישית של סוכנים {#customising-agents}

ניתן להרחיב או להחליף כל סוכן מובנה דרך המסנן `gratis_ai_agent_agents`.

### הוספת הנחיית מערכת מותאמת אישית לסוכן קיים {#adding-a-custom-system-prompt-to-an-existing-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### רישום סוכן חדש {#registering-a-new-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

הסוכן החדש מופיע בבוחר הסוכנים מיד לאחר שהמסנן רץ.

### הסרת סוכן מובנה {#removing-a-built-in-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
