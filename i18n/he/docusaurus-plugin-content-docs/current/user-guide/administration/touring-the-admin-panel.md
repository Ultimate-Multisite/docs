# סיור בלוח הניהול

Ultimate Multisite שואף להיות פשוט וברור ככל האפשר, אבל עם כלי עוצמתי כל כך, סיור מודרך יכול לעזור. נעבור על דפי הניהול כדי לעזור לכם להתמצא.

## Dashboard

ה-**Dashboard** של Ultimate Multisite מציג דוחות וניתוחים בסיסיים, כולל הכנסות, פעילויות אתרים, צמיחת חברויות, ספירות מבקרים ונתונים גאוגרפיים.

![Ultimate Multisite Dashboard חלק תחתון](/img/admin/um-dashboard-bottom.png)

הנה גם תצוגה מלאה של Dashboard הרשת:

![Network Dashboard דף מלא](/img/admin/network-dashboard-full-page.png)

## טופסי תשלום

**טופסי תשלום** מעניקים לכם את הגמישות ליצור דפי הרשמה מותאמים אישית. ניתן להתאים שדות, להכניס מחלקות וסקריפטים מותאמים אישית, ולהגביל גישה לפי מדינת המבקר או מיקומו.

![רשימת טופסי תשלום](/img/admin/checkout-forms-list.png)

## מוצרים

אזור **המוצרים** הוא המקום שבו יוצרים תוכניות, חבילות ושירותים שונים עבור הרשת שלכם. הגדירו מחירים, תדירויות חיוב, ומגבלות ומכסות ברמת המוצר.

![רשימת מוצרים](/img/admin/products-list.png)

## חברויות

דף **החברויות** מציג את כל המינויים ברשת שלכם. סקרו וערכו פרטי מינוי, כולל תוכניות, מוצרים, סכומי חיוב ותדירויות חיוב, היסטוריית תשלומים וחותמות זמן.

![רשימת חברויות](/img/admin/memberships-list.png)

## תשלומים

דף **התשלומים** מספק תצוגה מהירה של היסטוריית התשלומים בכל הרשת שלכם, עם מידע מפורט על עסקאות, כולל מוצרים וסכומים ספציפיים.

![רשימת תשלומים](/img/admin/payments-list.png)

## לקוחות

דף **הלקוחות** מציג את כל החברים הרשומים ברשת שלכם עם מידע משתמש בסיסי, כולל שם, כתובת דוא"ל, התחברות אחרונה וקישור "מעבר אל" לצורך התחברות לתת-האתר שלהם.

![רשימת לקוחות](/img/admin/customers-list.png)

## אתרים

דף **האתרים** מפרט את כל תתי-האתרים ברשת שלכם, כולל תבניות אתרים ואתרים בבעלות לקוחות. נהלו בקלות דומיינים ממופים, מגבלות ומכסות, תוספים וערכות עיצוב, ותמונות ממוזערות של אתרים.

![רשימת אתרים](/img/admin/sites-list.png)

## דומיינים

דף **הדומיינים** מוקדש לדומיינים מותאמים אישית הממופים לתת-אתרים. כמנהל-על, ניתן להוסיף או למפות ידנית שמות דומיין מותאמים אישית ולצפות ברשומות DNS ובלוגים.

![רשימת דומיינים](/img/admin/domains-list.png)

## קודי הנחה

**קודי הנחה** מאפשרים לכם ליצור קודי קופון כדי להציע הנחות למשתמשי הקצה שלכם. ציינו את ערך ההנחה וכוונו למשתמשים או קבוצות ספציפיים.

![רשימת קודי הנחה](/img/admin/discount-codes-list.png)

## שידורים

**שידורים** הם כלי לפריסת הודעות או לשליחת מסרים פרטיים למשתמשי תתי-האתרים שלכם. כוונו לקבוצות משתמשים ספציפיות לפי התוכנית או המוצר שאליהם הם מנויים.

![רשימת שידורים](/img/admin/broadcasts-list.png)

## הגדרות

דף **ההגדרות** הוא המקום שבו מגדירים את Ultimate Multisite — הגדרות הרשמה, תשלומים, API ו-webhooks, מיפוי דומיינים ואינטגרציות אחרות.

הגדרות מחבר ה-AI מציגות את מאגרי ספקי ה-OAuth הנתמכים כעת: Anthropic Max, OpenAI ChatGPT/Codex, ו-Google AI Pro. כל כרטיס ספק מאפשר למנהלי-על לחבר חשבונות, לרענן חשבונות שמורים, להסיר חשבונות לפי דוא"ל, ולהשתמש בחלופת OAuth ידנית כאשר סביבת sandbox חוסמת את הפניית הדפדפן. אפשרויות ההגדרה של Cursor Pro הוסרו מלוח הניהול.

חשבונות ChatGPT/Codex תומכים בשימוש בכלים המגובה במחבר כאשר הפעולה מאפשרת כלים, כך שתהליכי ניהול התלויים בפעולות הנתמכות על ידי מחבר יכולים להשתמש בהתנהגות כלי Codex לאחר שחיבור חשבון OpenAI הושלם.

## אירועים

דף **האירועים** שומר תיעוד של כל האירועים והלוגים ברשת שלכם. הוא עוקב אחר פעילויות כמו שינויי תוכנית, הרשמות ותנועות אחרות — שימושי לניטור המולטיסייט שלכם.

![רשימת אירועים](/img/admin/events-list.png)

## Webhooks

**Webhooks** מאפשרים לכם להעביר נתונים ליישום אחר. שימושי לשליחת נתונים מ-Ultimate Multisite לפלטפורמות כמו Zapier.

![רשימת Webhooks](/img/admin/webhooks-list.png)
