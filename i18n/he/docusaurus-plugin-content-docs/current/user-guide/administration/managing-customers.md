# ניהול לקוחות

דף **הלקוחות** מציג את רשימת המשתמשים הרשומים ברשת שלך. הדף מציג פרטי משתמש בסיסיים כולל שם, כתובת אימייל, כניסה אחרונה, וקישור **החלף אל** שמאפשר לך להתחבר לאתר המשנה של המשתמש כמנהל אתר.

![דף רשימת הלקוחות](/img/admin/customers-list.png)

לחצו על **נהל** מתחת לכרטיס הלקוח כדי לראות מידע מפורט על אותו לקוח.

בדף פרטי הלקוח, הבלוק **במבט חטוף** מציג את מזהה הלקוח, זמן הכניסה האחרונה, והסכום הכולל שהלקוח הוציא.

הבלוק **אפשרויות לקוח** מאפשר לכם להגדיר את הלקוח כ-**VIP** (תגית שמקלה על מציאתו). הלשונית **פרטי חיוב** מציגה מידע על החיוב כגון אימייל, כתובת ומדינה.

הבלוק **שדות** מציג את שם המשתמש של הלקוח, אימייל, ומאפשר לשנות את סטטוס **אימות האימייל**.

מתחת לבלוק אפשרויות לקוח תמצאו ארבעה בלוקים נוספים:

- **מנויים** — המנויים הנוכחיים שהלקוח רשום אליהם
- **תשלומים** — היסטוריית התשלומים של הלקוח
- **אתרים** — אתרים שהלקוח מחזיק בבעלותם
- **אירועים** — אירועים הקשורים ללקוח זה (אימות אימייל, שינויי חיוב וכו׳)

לחצו על סמל החץ (→) ליד כל פריט כדי לעבור לדף הפרטים שלו. לדוגמה, לחיצה על החץ ליד אתר תעביר אתכם לדף עריכת האתר שבו תוכלו לשנות את שיוך המנוי, סוג האתר והסטטוס הפעיל.

אותו הדבר חל על מנויים ותשלומים — כל אחד מקושר לדפי פרטים שבהם תוכלו לערוך את סטטוס המנוי, הלקוח המשויך, מוצרים, הגדרות שער תשלום, ואפילו לעבד החזרים.

בצד ימין של דף הלקוח תמצאו:

- **כניסה אחרונה וכתובות IP** — זמן כניסה אחרונה, כתובת IP ומדינה
- **חותמות זמן** — מתי חשבון הלקוח נוצר ועודכן לאחרונה
- **מחיקה** — אפשרות למחוק את הלקוח לצמיתות

:::warning
מחיקת לקוח היא פעולה בלתי הפיכה ולא ניתן לבטלה.
:::
