# ניהול מנויים

ב-Ultimate Multisite תוכלו לנהל מנויים ולגשת לכל המנויים ברשת שלכם. צפו וערכו פרטי מנוי כמו תוכניות ומוצרים, מספר האתרים, סכומי חיוב ותדירותם, היסטוריית תשלומים וחותמות זמן.

כדי לגשת לעמוד זה, היכנסו ללשונית **Memberships** בתפריט של Ultimate Multisite.

![עמוד רשימת המנויים](/img/admin/memberships-list.png)

בעמוד המנויים תוכלו לראות רשימה של הלקוחות שלכם והמוצרים המשויכים לחשבונותיהם. ניתן לסנן מנויים לפי סטטוס: פעיל, ממתין, מושהה, פג תוקף או מבוטל.

לחצו על מנוי כדי לגשת לעמוד שבו תוכלו לערוך את פרטי המנוי — פרטי חיוב, מגבלות אתרים או ביקורים ועוד.

תוכלו גם להוסיף מנויים חדשים המשויכים למשתמשים שנרשמו בעבר ברשת שלכם. לחצו על **Add Membership** והגדירו את הפרטים.

![חלון הוספת מנוי](/img/admin/membership-add-modal.png)

לאחר מכן לחצו על **Create Membership**. המנוי החדש יופיע ברשימת המנויים.
