# ניהול הודעות דוא"ל מערכת

Ultimate Multisite שולח הודעות דוא"ל מערכת שונות עבור אירועים כמו אישורי הרשמה, קבלות תשלום ושינויים במנויים. ניתן להתאים אישית את תבניות הדוא"ל האלה כך שיתאימו למותג שלך.

## צפייה בהודעות דוא"ל מערכת

נווט אל **Ultimate Multisite → הגדרות** ולחץ על לשונית **דוא"ל** כדי לראות את ההגדרות הקשורות לדוא"ל.

![הגדרות - לשונית דוא"ל](/img/admin/settings-emails.png)

משם, תוכל לגשת לדף **הודעות דוא"ל מערכת** כדי לראות את כל תבניות הדוא"ל.

![רשימת הודעות דוא"ל מערכת](/img/admin/system-emails-list.png)

## ייבוא תבניות דוא"ל חדשות

גרסאות חדשות של Ultimate Multisite ותוספים עשויות לרשום תבניות דוא"ל חדשות. כדי למנוע התנגשויות, תבניות חדשות אינן מתווספות באופן אוטומטי — עליך לייבא אותן ידנית.

לייבוא תבניות דוא"ל חדשות:

1. עבור לדף **הודעות דוא"ל מערכת**
2. לחץ על כפתור **איפוס וייבוא** בראש הדף
3. הפעל את אפשרות **ייבוא הודעות דוא"ל** כדי לראות את התבניות הזמינות
4. בחר את התבניות שברצונך לייבא ואשר

## איפוס תבניות דוא"ל

אם התאמת אישית תבנית דוא"ל וברצונך לחזור לתוכן ברירת המחדל, יש לך שתי אפשרויות:

1. **מחיקה וייבוא מחדש** — מחק את הודעת הדוא"ל המערכתית וייבא אותה שוב (פעולה זו מוחקת את מדדי השליחה)
2. **איפוס באמצעות הכלי** — השתמש בכלי **איפוס וייבוא**, הפעל את אפשרות **איפוס**, ובחר את הודעות הדוא"ל שברצונך לאפס

אפשרות האיפוס עדיפה מכיוון שהיא משמרת את מדדי השליחה ומטא-דאטה אחר תוך שחזור תוכן ברירת המחדל.
