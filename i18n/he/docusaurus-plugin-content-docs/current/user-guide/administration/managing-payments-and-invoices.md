# ניהול תשלומים וחשבוניות {#managing-payments-and-invoices}

## הגדרות תשלום {#payment-settings}

לפני שתתחילו לקבל תשלומים, תצטרכו להגדיר את ההגדרות הקשורות לתשלום. נווטו אל **Ultimate Multisite → Settings** ולחצו על הלשונית **Payment**.

![לשונית הגדרות תשלום](/img/admin/settings-payments-top.png)

הנה תצוגה מלאה של דף הגדרות התשלום:

![דף הגדרות תשלום מלא](/img/admin/settings-payments-full.png)

### אפשרויות תשלום כלליות {#general-payment-options}

בהגדרות הכלליות תוכלו להגדיר:

- **מטבע** — מטבע ברירת המחדל המשמש לעסקאות
- **מיקום המטבע** — המקום שבו מופיע סמל המטבע (לפני/אחרי הסכום)

![הגדרות שער תשלום](/img/admin/settings-payments-options.png)

### שערי תשלום {#payment-gateways}

Ultimate Multisite תומך במספר שערי תשלום. תוכלו להפעיל ולהגדיר כל אחד מהם מלשונית הגדרות התשלום.

![תצורת שער תשלום](/img/admin/settings-payments-gateways.png)

השערים הזמינים כוללים:

- **Stripe** — תשלומי כרטיס אשראי דרך Stripe
- **PayPal** — תשלומי PayPal
- **ידני** — לעיבוד תשלומים לא מקוון או מותאם אישית

לכל שער יש מקטע הגדרה משלו שבו מזינים מפתחות API והגדרות אחרות.

![הגדרות שער נוספות](/img/admin/settings-payments-gateways-2.png)

### מצב Sandbox {#sandbox-mode}

אפשר להפעיל את **מצב Sandbox** כדי לבדוק את אינטגרציית התשלומים לפני העלייה לאוויר. כאשר מצב Sandbox פעיל, לא יבוצעו חיובים אמיתיים.

## צפייה בתשלומים {#viewing-payments}

נווטו אל דף **התשלומים** תחת Ultimate Multisite כדי לראות את כל העסקאות ברחבי הרשת שלכם.

![רשימת תשלומים](/img/admin/payments-list.png)

אפשר לסנן תשלומים לפי סטטוס (הושלם, ממתין, נכשל, הוחזר) ולחפש עסקאות מסוימות.

לחצו על תשלום כדי לראות את פרטיו המלאים, כולל פריטי השורה, המינוי המשויך, פרטי הלקוח ונתוני שער התשלום.

## חשבוניות {#invoices}

Ultimate Multisite יכול ליצור חשבוניות עבור תשלומים באופן אוטומטי. אפשר להתאים אישית את תבנית החשבונית ואת פורמט המספור מתוך הגדרות התשלום.

אפשרויות התאמה אישית של חשבוניות כוללות:

- **שם וכתובת החברה** המוצגים בחשבוניות
- **מספור חשבוניות** פורמט ורצף
- **לוגו** המוצג בכותרת החשבונית
- **טקסט כותרת תחתונה מותאם אישית** לתנאים, הערות או מידע משפטי

כדי להתאים אישית את תבנית החשבונית, עברו אל **Ultimate Multisite → Settings → Payment** וחפשו את ההגדרות הקשורות לחשבוניות.
