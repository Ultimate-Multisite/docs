# ניהול תשלומים וחשבוניות

## הגדרות תשלום

לפני שתתחילו לקבל תשלומים, תצטרכו להגדיר את ההגדרות הקשורות לתשלומים. נווטו אל **Ultimate Multisite → Settings** ולחצו על לשונית **Payment**.

![לשונית הגדרות תשלום](/img/admin/settings-payments-top.png)

### אפשרויות תשלום כלליות

בהגדרות הכלליות תוכלו להגדיר:

- **מטבע** — המטבע שישמש כברירת מחדל לעסקאות
- **מיקום המטבע** — היכן יופיע סמל המטבע (לפני/אחרי הסכום)

![הגדרות שער תשלום](/img/admin/settings-payments-options.png)

### שערי תשלום

Ultimate Multisite תומך במספר שערי תשלום. תוכלו להפעיל ולהגדיר כל אחד מהם מלשונית הגדרות התשלום.

![הגדרת שער תשלום](/img/admin/settings-payments-gateways.png)

השערים הזמינים כוללים:

- **Stripe** — תשלומי כרטיס אשראי דרך Stripe
- **PayPal** — תשלומי PayPal
- **ידני** — לעיבוד תשלומים לא מקוון או מותאם אישית

לכל שער יש אזור הגדרות משלו שבו מזינים מפתחות API והגדרות נוספות.

![הגדרות שער נוספות](/img/admin/settings-payments-gateways-2.png)

### מצב בדיקות (Sandbox)

תוכלו להפעיל **מצב Sandbox** כדי לבדוק את אינטגרציית התשלומים לפני העלייה לאוויר. כאשר מצב Sandbox פעיל, לא יבוצעו חיובים אמיתיים.

## צפייה בתשלומים

נווטו לדף **Payments** תחת Ultimate Multisite כדי לראות את כל העסקאות ברשת שלכם.

![רשימת תשלומים](/img/admin/payments-list.png)

תוכלו לסנן תשלומים לפי סטטוס (הושלם, ממתין, נכשל, הוחזר) ולחפש עסקאות ספציפיות.

לחצו על תשלום כדי לראות את כל הפרטים שלו, כולל פריטי השורה, המנוי המשויך, פרטי הלקוח ונתוני שער התשלום.

## חשבוניות

Ultimate Multisite יכול ליצור חשבוניות באופן אוטומטי עבור תשלומים. תוכלו להתאים אישית את תבנית החשבונית ופורמט המספור מהגדרות התשלום.

אפשרויות ההתאמה האישית של החשבונית כוללות:

- **שם החברה וכתובתה** המוצגים בחשבונית
- **פורמט מספור החשבוניות** והרצף
- **לוגו** המוצג בכותרת החשבונית
- **טקסט תחתית מותאם אישית** לתנאים, הערות או מידע משפטי

כדי להתאים אישית את תבנית החשבונית, עברו אל **Ultimate Multisite → Settings → Payment** וחפשו את ההגדרות הקשורות לחשבוניות.
