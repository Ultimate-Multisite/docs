# ייצוא אתר

דף המנהל **Site Export** מאפשר למנהלי רשת לארוז אתר אחד, או את כל הרשת, לארכיון הניתן להורדה לצורך מיגרציה, גיבוי או העברה לידי גורם אחר.

## ייצוא אתר בודד

עבור אל **Ultimate Multisite > Site Export** ובחר **Generate new Site Export**. בחר את תת-האתר שברצונך לייצא, ולאחר מכן בחר אם הארכיון צריך לכלול העלאות (uploads), תוספים (plugins) וערכות נושא (themes).

לאחר שהייצוא יסתיים, הוריד את קובץ ה-ZIP מרשימת **Existing Exports**. קובצי ZIP המיוצאים כוללים כעת `index.php` שניתן להפעלה עצמית וקובץ `readme.txt`, כך שניתן להעלות את הארכיון לאחסון חדש ולהפעיל אותו מבלי להתקין קודם תוסף ייבוא נפרד.

## ייצוא כל הרשת

השתמש ב**Network Export** בדף Site Export כאשר אתה זקוק לארכיון יחיד המכיל את כל תת-האתרים ברשת. זה שימושי לפני מיגרציות אחסון, תרגילים להשבת נזק, או בנייה מחדש של סביבת ה-staging, במקרים שבהם כל תת-אתר צריך לנוע יחד.

מכיוון שייצוא רשת יכול להיות גדול בהרבה מ-ייצוא אתר בודד, הפעל אותו בתקופת תעבורה נמוכה וודא שהאחסון המיועד כולל מספיק מקום פנוי להעלאות, תוספים, ערכות נושא וקובצי ZIP שנוצרו.

### חבילות ייבוא רשת (Network Import Bundles)

החל מ-Ultimate Multisite 2.12.0, Site Exporter יכול ליצור **network import bundles** — ארכיבים מיוחדים שנועדו לשחזור אתרים ברמת הרשת בצורה יעילה. חבילת ייבוא רשת מכילה את כל הקבצים והמטא-דאטה הנדרשים כדי לשחזר מספר אתרים בהתקנה חדשה של רשת.

#### יצירת חבילת ייבוא רשת

1. עבור אל **Ultimate Multisite > Site Export**
2. לחץ על **Generate new Network Export**
3. בחר **Network Import Bundle** כסוג הייצוא
4. בחר אם לכלול העלאות, תוספים וערכות נושא
5. לחץ על **Generate**
6. הוריד את קובץ ה-ZIP של החבילה מרשימת **Existing Exports**

#### שחזור מחבילת ייבוא רשת

כדי לשחזר אתרים מחבילת ייבוא רשת:

1. התקן את Ultimate Multisite על האחסון המיועד
2. השלם את מַכשיר הגדרת הריבוי-אתרים (multisite setup wizard)
3. עבור אל **Ultimate Multisite > Site Export** ברשת החדשה
4. לחץ על **Import Network Bundle**
5. העלה את קובץ ה-ZIP של חבילת ייבוא הרשת
6. עקוב אחר הוראות הייבוא שמוצגות על המסך
7. תהליך הייבוא ישחזר את כל האתרים, התוכן וההגדרות שלהם

חבילות ייבוא רשת שומרות על:
- תוכן האתר (פוסטים, עמודים, סוגי פוסט מותאמים אישית)
- הגדרות ואפשרויות האתר
- תפקידי משתמש והרשאות
- תוספים וערכות נושא (אם נכללו בחבילה)
- העלאות מדיה (אם נכללו בחבילה)
- טבלאות נתונים ונתונים מותאמים אישית למסד הנתונים

## התקנת קובץ ZIP ייצוא שניתן להפעלה עצמית

כדי לשחזר קובץ ZIP שניתן להפעלה עצמית על אחסון חדש:

1. העלה את תוכן קובץ ה-ZIP המיוצא לשורש האתר (web root) המיועד.
2. פתח את קובץ `index.php` שהועלה בדפדפן.
3. עקוב אחר הוראות ההתקנה שמוצגות על המסך מחבילת הייצוא.
4. עיין בקובץ `readme.txt` המצורף כדי לקבל הערות ספציפיות לייצוא לפני שאתה מסיר קבצים זמניים.

לפרטים על התקנה וייבוא ספציפיים לתוספים, עיין ב-[Site Exporter addon documentation](/addons/site-exporter).

עבור הכלי של אתר בודד שהוסף ב-Ultimate Multisite 2.9.0, עיין ב-[Export & Import](/user-guide/administration/export-import).
