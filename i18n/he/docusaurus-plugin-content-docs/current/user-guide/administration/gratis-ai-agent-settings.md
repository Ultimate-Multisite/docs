# הגדרות Gratis AI Agent

מסך **הגדרות → מתקדם** (Settings → Advanced) ב-Gratis AI Agent מספק תצורה ברמת מנהל מערכת עבור אינטגרציות צד-שרת (backend integrations) שהוצגו בגרסה v1.5.0. דף זה מתעד את השדות של **נקודת קצה משוב** (Feedback Endpoint) ואת הפורמט הצפוי שלהם.

## גישה להגדרות

1. במנהל ה-WordPress, עבור אל **Gratis AI Agent → Settings**.
2. לחץ על לשונית **מתקדם**.

## תצורה של נקודת קצה המשוב

נקודת קצה המשוב מקבלת בקשות POST מה-AI agent בכל פעם שמשתמש מגיש משוב באמצעות כפתור האגמי (thumbs-down), באנר ההנחיה האוטומטי, או הפקודה `/report-issue`.

| שדה | תיאור |
|---|---|
| **Feedback Endpoint URL** | כתובת ה-URL שמקבלת הגשות משוב כבקשות HTTP POST עם גוף JSON. |
| **Feedback API Key** | אסימון נשא (bearer token) שנשלח בכותרת `Authorization` של כל בקשת משוב. השאר ריק אם נקודת הקצה שלך אינה דורשת אימות. |

### גוף JSON צפוי

נקודת קצה המשוב שלך חייבת לקבל גוף JSON עם לפחות השדות הבאים:

```json
{
  "message_id": "msg_abc123",
  "conversation_id": "conv_xyz789",
  "feedback_text": "התשובה הייתה שגויה לגבי תמחור.",
  "triage_category": "factual_error"
}
```

ייתכנו שדות נוספים בגוף ההעברה בהתאם להקשר השיחה.

### ערכי `triage_category`

שכבת המיון (triage layer) של ה-AI מקצה אחד מהערכים הבאים ל־`triage_category` לפני העברת המידע:

| ערך | משמעות |
|---|---|
| `factual_error` | המساعد סיפק מידע עובדתי שגוי. |
| `unhelpful_answer` | התשובה הייתה נכונה מבחינה טכנית אך לא שימושית. |
| `inappropriate_content` | התשובה הכילה תוכן שלא אמור להוצג למשתמשים. |
| `other` | המשוב לא תאם קטגוריה ידועה. |

### אימות (Authentication)

אם נקודת הקצה שלך דורשת אימות, הגדר את השדה **Feedback API Key** עם אסימון הנשא שלך. המערכת שולחת:

```
Authorization: Bearer <your-api-key>
```

אם השדה **Feedback API Key** ריק, לא נשלחת כותרת `Authorization`.

### השבתת איסוף משוב

השאר את שני השדות **Feedback Endpoint URL** ו-**Feedback API Key** ריקים. כפתור האגמי וממשק המשתמש למשוב נשארים גלויים למשתמשים, אך ההגשות אינן מועברות לשום שירות חיצוני.

## מפתח API של Brave Search

גם בלשונית **מתקדם**, השדה **Brave Search API Key** מאפשר את יכולת [חיפוש אינטרנט](../configuration/internet-search).

| שדה | תיאור |
|---|---|
| **Brave Search API Key** | מפתח ה-API שלך מדשבורד המפתחים של Brave Search. נדרש כדי להפעיל חיפוש אינטרנט במساعد ה-AI. |

תווית השדה כוללת קישור לחיצה לדף ההרשמה ל-Brave Search API. השאר ריק כדי לבטל את חיפוש האינטרנט.

ראה [חיפוש אינטרנט](../configuration/internet-search) למסמכי המשתמש הקצה בנוגע לתכונה זו.

## דגלי תכונה (Feature Flags)

הוסף גם בגרסה v1.9.0, לשונית **הגדרות → דגלי תכונה** מספקת מתגי הפעלה (toggle switches) לפונקציונליות אופציונלית. כל דגל מופעל או מושבת ברמת הרשת; אין כרגע אפשרות לדרוס הגדרות ברמת האתר.

### גישה לדגלי תכונה

1. במנהל ה-WordPress, עבור אל **Gratis AI Agent → Settings**.
2. לחץ על לשונית **Feature Flags**.

### דגלי בקרת גישה

| דגל | ברירת מחדל | תיאור |
|---|---|---|
| **Restrict to Administrators** | Off | כאשר מופעל, רק משתמשים עם התפקיד `administrator` יכולים לפתוח את פאנל הצ'אט של ה-AI Agent. כל התפקידים האחרים רואים הודעה של "צור קשר עם מנהל המערכת". |
| **Restrict to Network Admins** | Off | כאשר מופעל ברשת מרובת אתרים (multisite network), רק מנהלי מערכת (Super Admins) יכולים להשתמש במערכת. מנהלי אתרים בודדים נחסמים. גובר על "Restrict to Administrators" אם שניהם מופעלים. |
| **Allow Subscriber Access** | Off | כאשר מופעל, משתמשים עם התפקיד `subscriber` יכולים להשתמש בממשק הצ'אט אך מוגבלים ליכולות קריאה בלבד (אין יצירת פוסטים או שינוי הגדרות). |
| **Disable for Non-Members** | Off | משולב עם סטטוס המנוי של Ultimate Multisite. כאשר מופעל, הצ'אט מוסתר עבור אתרים שאין להם מנוי פעיל. |

### דגלי מיתוג (Branding Flags)

| דגל | ברירת מחדל | תיאור |
|---|---|---|
| **Hide "Powered by Gratis AI Agent" Footer** | Off | מסיר את שורת המיוחסת המופיעה בתחתית ווידג'ט הצ'אט. מומלץ לפריסות מותג לבן (white-label). |
| **Custom Agent Name** | *(blank)* | מחליף את התווית ברירת המחדל "Gratis AI Agent" בכותרת הצ'אט ובמערכת המנהל בשם המוצר שלך. השאר ריק כדי להשתמש במצב ברירת המחדל. |
| **Hide Agent Picker** | Off | כאשר מופעל, משתמשים אינם יכולים להחליף בין חמשת הסוכנים המובנים. הסוכן הנוכחי מקובע למה שנקבע כברירת מחדל בהגדרות → כללי. |
| **Use Site Icon as Chat Avatar** | Off | מחליף את אייקון ה-AI ברירת המחדל בכותרת ווידג'ט הצ'אט באייקון האתר של WordPress (שנקבע תחת מראה → התאמה אישית → זהות האתר). |

### יישום שינויים

לחץ על **שמירת הגדרות** לאחר הפעלת כל דגל. השינויים נכנסים לתוקף באופן מיידי — אין צורך לנקות מטמון או להפעיל מחדש תוסף.
