# הגדרות Gratis AI Agent

מסך **Settings → Advanced** ב-Gratis AI Agent מספק תצורה ברמת מנהל עבור אינטגרציות צד שרת. עמוד זה מתעד העברת משוב, מפתחות ספקי חיפוש, הגדרת שירות Superdav מנוהל, בקרות Google Calendar, הגדרות SMS של TextBee ודגלי תכונות ברמת הרשת.

## גישה להגדרות

1. בממשק הניהול של WordPress, עבור אל **Gratis AI Agent → Settings**.
2. לחץ על הלשונית **Advanced**.

## תצורת נקודת קצה למשוב

נקודת הקצה למשוב מקבלת בקשות POST מה-AI agent בכל פעם שמשתמש שולח משוב באמצעות כפתור האגודל כלפי מטה, באנר ההנחיה האוטומטית, או הפקודה `/report-issue`.

| שדה | תיאור |
|---|---|
| **Feedback Endpoint URL** | ה-URL שמקבל שליחות משוב כבקשות HTTP POST עם גוף JSON. |
| **Feedback API Key** | bearer token שנשלח ב-`Authorization` header של כל בקשת משוב. השאר ריק אם נקודת הקצה שלך אינה דורשת אימות. |

### מטען JSON צפוי

נקודת הקצה למשוב שלך חייבת לקבל גוף JSON עם לפחות השדות הבאים:

```json
{
  "message_id": "msg_abc123",
  "conversation_id": "conv_xyz789",
  "feedback_text": "The answer was incorrect about pricing.",
  "triage_category": "factual_error"
}
```

שדות נוספים עשויים להופיע במטען בהתאם להקשר השיחה.

### ערכי `triage_category`

שכבת המיון של ה-AI מקצה אחד מהערכים הבאים ל-`triage_category` לפני העברת המטען:

| ערך | משמעות |
|---|---|
| `factual_error` | העוזר סיפק מידע עובדתי שגוי. |
| `unhelpful_answer` | התגובה הייתה נכונה טכנית אך לא שימושית. |
| `inappropriate_content` | התגובה כללה תוכן שאסור להציג למשתמשים. |
| `other` | המשוב לא תאם קטגוריה מוכרת. |

### אימות

אם נקודת הקצה שלך דורשת אימות, הגדר את השדה **Feedback API Key** ל-bearer token שלך. ה-agent שולח:

```
Authorization: Bearer <your-api-key>
```

אם השדה **Feedback API Key** ריק, לא נשלח `Authorization` header.

### השבתת איסוף משוב

השאר את שני השדות **Feedback Endpoint URL** ו-**Feedback API Key** ריקים. כפתור האגודל כלפי מטה וממשק המשתמש למשוב נשארים גלויים למשתמשים, אך שליחות אינן מועברות לשום שירות חיצוני.

## Brave Search API Key

גם בלשונית **Advanced**, השדה **Brave Search API Key** מפעיל את יכולת [חיפוש באינטרנט](../configuration/internet-search).

| שדה | תיאור |
|---|---|
| **Brave Search API Key** | מפתח ה-API שלך מלוח הבקרה למפתחים של Brave Search. נדרש כדי להפעיל חיפוש באינטרנט בעוזר ה-AI. |

תווית השדה כוללת קישור לחיץ לעמוד ההרשמה ל-Brave Search API. השאר ריק כדי להשבית חיפוש באינטרנט.

ראה [חיפוש באינטרנט](../configuration/internet-search) לתיעוד למשתמשי קצה על תכונה זו.

## שירות Superdav מנוהל

Superdav AI Agent v1.18.0 מוסיף נקודות קצה לשירות Superdav מנוהל והקצאת חיבור אוטומטית לאתרים נתמכים. השתמש בבקרות אלה כאשר האתר שלך צריך להתחבר לספק המתארח במקום לנקודת קצה של שירות שהוגדרה ידנית.

| שדה | תיאור |
|---|---|
| **Managed Superdav Service** | מפעיל את חיבור שירות Superdav המתארח עבור אתרים נתמכים. |
| **Provision Connection** | מתחיל הקצאה אוטומטית של נקודת קצה ואישורי גישה. השתמש בזה לאחר אישור שהאתר צריך להשתמש בספק המנוהל. |
| **Service Endpoint / Connection Status** | מציג את נקודת הקצה הנוכחית או את מצב החיבור לאחר ההקצאה. |

לאחר ההקצאה, שמור את ההגדרות ואמת את מצב החיבור לפני הסתמכות על תהליכי עבודה של שירות מנוהל. אם ההקצאה נכשלת, בדוק את הנחיות הניסיון החוזר המוצגות ואשר שלאתר יש הרשאה להשתמש בספק המתארח.

## תצורת Google Calendar

כאשר תכונות היומן של Superdav AI Agent v1.18.0 מופעלות, ה-agent יכול לקרוא יומנים מוגדרים ופרטי אירועים. כלי היומן מכוונים לקריאה ושימושיים לתזכורות מודעות-ללוח-זמנים, מעקב אחר משתתפים והתאמת אנשי קשר.

| שדה | תיאור |
|---|---|
| **Google Calendar Credentials** | שומר את אישורי הגישה או חיבור ה-token הנדרשים לקריאת נתוני יומן. |
| **Calendar Selection** | מגביל אילו יומנים מוגדרים ה-agent רשאי לבדוק. |
| **Calendar Connection Status** | מאשר האם אישורי הגישה הנוכחיים יכולים לקרוא יומנים ואירועים. |

שמור את אישורי היומן מוגבלים ליומנים שה-agent צריך. התחבר מחדש או החלף אישורי גישה אם המצב מציין token שפג תוקפו.

## התראות SMS של TextBee

Superdav AI Agent v1.18.0 מוסיף את TextBee כספק SMS עבור תהליכי עבודה מוגדרים של התראות. יש לשלב התראות SMS עם שערי אישור אנושיים עבור הודעות רגישות או כאלה המוצגות למשתמשים.

| שדה | תיאור |
|---|---|
| **TextBee API Key** | מאמת בקשות לספק ה-SMS של TextBee. |
| **TextBee Device / Sender** | בוחר את שולח TextBee או המכשיר המשמש להודעות יוצאות, כאשר הספק דורש זאת. |
| **SMS Notifications Enabled** | מאפשר לתהליכי עבודה מאושרים לשלוח התראות בהודעות טקסט. השאר מושבת כדי למנוע שליחות SMS. |

שלח הודעת בדיקה רק למספר שבבעלות מנהל, ולאחר מכן אשר את התנהגות שער האישור לפני הפעלת תזכורות מתוזמנות או תזכורות המיועדות למשתתפים.

## דגלי תכונות

גם הוצגה ב-v1.9.0, הלשונית **Settings → Feature Flags** מספקת מתגי הפעלה/כיבוי לפונקציונליות אופציונלית. כל דגל מופעל או מושבת ברמת הרשת; אין כרגע עקיפה לכל אתר בנפרד.

### גישה לדגלי תכונות

1. בממשק הניהול של WordPress, עבור אל **Gratis AI Agent → Settings**.
2. לחץ על הלשונית **Feature Flags**.

### דגלי בקרת גישה

| דגל | ברירת מחדל | תיאור |
|---|---|---|
| **הגבל למנהלים** | כבוי | כאשר מופעל, רק משתמשים עם התפקיד `administrator` יכולים לפתוח את חלונית הצ'אט של AI Agent. כל שאר התפקידים רואים במקום זאת הודעת "פנה למנהל שלך". |
| **הגבל למנהלי רשת** | כבוי | כאשר מופעל ברשת מרובת אתרים, רק Super Admins יכולים להשתמש בסוכן. מנהלי אתרים בודדים נחסמים. מקבל עדיפות על פני "הגבל למנהלים" אם שניהם מופעלים. |
| **אפשר גישה למנויים** | כבוי | כאשר מופעל, משתמשים עם התפקיד `subscriber` יכולים להשתמש בממשק הצ'אט אך מוגבלים ליכולות קריאה בלבד (ללא יצירת פוסטים או שינויי הגדרות). |
| **השבת עבור מי שאינם חברים** | כבוי | משתלב עם סטטוס החברות של Ultimate Multisite. כאשר מופעל, הצ'אט מוסתר עבור אתרים שאין להם חברות פעילה. |

### דגלי מיתוג

| דגל | ברירת מחדל | תיאור |
|---|---|---|
| **הסתר את שורת הכותרת התחתונה "Powered by Gratis AI Agent"** | כבוי | מסיר את שורת ייחוס המיתוג המוצגת בתחתית יישומון הצ'אט. מומלץ לפריסות במיתוג לבן. |
| **שם סוכן מותאם אישית** | *(ריק)* | מחליף את התווית ברירת המחדל "Gratis AI Agent" בכותרת הצ'אט ובתפריט הניהול בשם המוצר שלך. השאר ריק כדי להשתמש בברירת המחדל. |
| **הסתר בורר סוכן** | כבוי | כאשר מופעל, משתמשים אינם יכולים לעבור בין חמשת הסוכנים המובנים. הסוכן הנוכחי מקובע למה שמוגדר כברירת המחדל בהגדרות ← כללי. |
| **השתמש בסמל האתר כאווטאר הצ'אט** | כבוי | מחליף את סמל ה-AI ברירת המחדל בכותרת יישומון הצ'אט בסמל אתר WordPress (מוגדר תחת עיצוב ← התאמה אישית ← זהות האתר). |

### דגלי בטיחות אוטומציה

Superdav AI Agent v1.18.0 מציג שערי אישור אנושיים ורשומות תזכורת להפעלות אוטומציה בטוחות יותר. הבקרות האלה עשויות להופיע בדגלי התכונות או בהגדרות האוטומציה המתקדמות, בהתאם לחבילה המותקנת.

| דגל | ברירת מחדל | תיאור |
|---|---|---|
| **דרוש אישור אנושי** | מומלץ מופעל | משהה אוטומציות רגישות עד שמשתמש מורשה בודק ומאשר את הפעולה המוצעת. |
| **מניעת כפילויות בתזכורות** | מופעל | מתעד תזכורות שנשלחו כדי שניסיונות חוזרים או הפעלות מתוזמנות לא ישלחו התראות כפולות. |
| **הפעל כלי לוח שנה** | כבוי עד להגדרה | מאפשר לסוכן לקרוא לוחות שנה ואירועים מוגדרים של Google. |
| **הפעל התראות SMS** | כבוי עד להגדרה | מאפשר לתהליכי עבודה מאושרים לשלוח התראות SMS של TextBee לאחר שמירת פרטי הגישה. |

### החלת שינויים

לחץ על **שמור הגדרות** לאחר שינוי כל דגל. השינויים נכנסים לתוקף מיד — אין צורך בניקוי מטמון או בהפעלה מחדש של ה-plugin.
