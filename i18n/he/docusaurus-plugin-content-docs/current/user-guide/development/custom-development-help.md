# איך לקבל עזרה בפיתוח מותאם אישית מ-Codeable?

לפעמים, אנחנו באמת צריכים התאמה אישית מעולה כדי שתהליכי הרשת שלנו יעבדו בדיוק כמו שאנחנו רוצים. אם זה מה שאתם מחפשים, אנחנו ממליצים על [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), מקום שבו תוכלו למצוא מפתחים מנוסים עם מומחיות ב-Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) היא פלטפורמה שמחברת לקוחות עם מפתחי WordPress פרילנסרים מוסמכים, שיכולים לעבוד על פרויקטים בכל גודל. יש להם תהליך מיון קפדני כדי להבטיח שתחוברו למומחים שמתאימים במיוחד למשימה שיש לכם בראש.

### איך יוצרים פרויקט ב-Codeable?

ראשית, היכנסו ל-[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) ולחצו על הכפתור **Start a Project**.

ב-"I need help with", בחרו **Customizing**:

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

ב-"My", בחרו **Plugin**:

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

לאחר מכן לחצו על הכפתור **Continue**.

עכשיו הגיע הזמן לתאר את הפרויקט והצרכים שלכם בצורה הטובה ביותר שאתם יכולים. נסו לתת כמה שיותר מידע!

<!-- Screenshot unavailable: Codeable project description form -->

ענו על השאלות הבאות כדי לעזור בתמחור.

<!-- Screenshot unavailable: Codeable project pricing questions -->

לאחר מכן, אם עדיין אין לכם חשבון ב-[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), צרו אחד ולחצו על הכפתור **Publish My Project**.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

ברגע שפרסמתם את הפרויקט שלכם ב-[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), הפלטפורמה תחבר אתכם עם מפתחים שמתאימים לסט הכישורים הנדרש לפרויקט שלכם.

תהליך זה יעזור למצוא את האדם הנכון לעבודה שלכם, עם ניסיון ב-WordPress וב-Ultimate Multisite!

התעריף המינימלי של Codeable הוא 70–120 דולר לשעה. ההערכות ישתנו בהתאם להיקף העבודה, המורכבות והדחיפות של הפרויקט.

ברגע שתקבלו את ההערכה שלכם, תוכלו לשכור את המפתח שהכי מתאים לצרכים שלכם ולהתחיל לעבוד על הפרויקט יחד!
