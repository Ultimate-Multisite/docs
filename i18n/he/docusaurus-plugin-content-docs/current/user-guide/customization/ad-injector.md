# הצגת פרסומות ברשת שלך עם Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector הוא **תוסף חינמי** שנותן לך את הכלים להציב פרסומות בתוכניות שלך ישירות מעמוד העריכה שלהן.

לאחר התקנת Ultimate Multisite: Ad Injector, עבור לעמוד ה-_Plugins_ שלך כדי להפעיל אותו.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_לחץ על כפתור Network Activate_

ברגע שהתוסף פעיל, תיבת המטא _Advanced Options_ במסך עריכת התוכנית תכלול אפשרות חדשה בשם _Ad Settings_.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_עכשיו אתה יכול לערוך את הגדרות הפרסום לכל אחת מהתוכניות שלך_

בלשונית הזו, תוכל להוסיף את קודי הפרסום לפוסטים ולעמודים בצד הלקוח, לפני ואחרי התוכן.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_לאחר הכנסת קוד הפרסומת, היא תופיע בפוסטים ובעמודים של התוכנית הספציפית הזו._

אתה יכול גם להציג פרסומות בכותרת של עמודי הניהול למשתמשים של אותה תוכנית.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_עבור לאפשרות Admin Panel Ad Code אם אתה רוצה להציג פרסומות בממשק הניהול של המשתמשים שלך_

### מעבר לפרסום

אתה יכול גם להשתמש ב-Ultimate Multisite: Ad Injector כדי להציע תמיכה לתוכניות הפרימיום.

פשוט עבור לאפשרות _Admin Panel Ads_ והוסף את קוד ה-HTML של כלי התמיכה שלך.

כך אנשים יכולים לפנות אליך ישירות מלוח הבקרה שלהם!
