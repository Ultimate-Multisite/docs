# התאמה אישית של לוח הניהול עם PRO Theme

לעיתים קרובות, כשאנחנו עובדים עם לקוחות, עולה הצורך לספק להם חוויה מותאמת אישית בלוח הניהול של WordPress. אם אתם מחפשים מראה ותחושה מקצועיים יותר, PRO Theme יכול לעזור לכם.

PRO Theme הוא ערכת עיצוב לממשק הניהול שפותחה כדי לסייע לכם בהתאמה אישית של לוח הבקרה ללקוחות שלכם. והוא גם חלק מליבת התוספים הפרימיום של Ultimate Multisite.

לאחר ההתקנה, עברו לעמוד _Plugins_ ולחצו כדי להפעיל אותו ברשת שלכם.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_הפעילו אותו ברמת הרשת בעמוד התוספים שלכם_

_**הערה:**_ _אם רק תתקינו אותו, הוא יופיע אצל הלקוחות שלכם כאפשרות שניתן לסמן כדי להוסיף את ערכת העיצוב רק למשתמש הספציפי הזה._

עכשיו, ממשק הניהול שלכם עוצב מחדש לגמרי! יש לכם גם תפריט חדש בעמודה השמאלית בשם PRO Theme.
