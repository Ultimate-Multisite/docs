# הגדרת תשלומים ידניים (v2) {#setting-up-manual-payments-v2}

_**הערה חשובה: מאמר זה מתייחס לגרסת Ultimate Multisite 2.x.**_

תשלומים ידניים הם דרך להציע שיטות תשלום אחרות אם **Stripe** או **PayPal** אינם זמינים למשתמשים שלכם. זה יכול להיות העברה בנקאית או כל שיטת תשלום אחרת הזמינה למשתמשים שלכם באופן מקומי.

## איך להפעיל תשלומים ידניים {#how-to-enable-manual-payments}

הגדרת תשלום ידני היא פשוטה מאוד. צריך רק להפעיל אותו תחת שערי התשלום ולהוסיף הוראות מפורטות לגבי האופן שבו המשתמש צריך לשלוח את התשלום.

ראשית, עברו אל **Ultimate Multisite > הגדרות > תשלומים**. מתחת ל-**שערי תשלום** , הפעילו את **ידני**. תראו שתיבה בשם **הוראות תשלום** תופיע עבורכם.

הוסיפו לתיבה זו את המידע שהלקוח שלכם יצטרך כדי לבצע את התשלום. זה יכול להיות פרטי חשבון הבנק שלכם והאימייל שלכם, כדי שהלקוח יוכל לשלוח לכם את אישור התשלום, למשל.

![מתג שער תשלום ידני עם אזור טקסט של הוראות תשלום](/img/config/manual-gateway-expanded.png)

הנה ממשק הגדרות השער הידני:

![הגדרות שער ידני](/img/config/manual-gateway-settings.png)

לאחר ההגדרה, פשוט לחצו על **שמירת הגדרות** וזהו. כאשר משתמשים נרשמים לרשת שלכם, הם יראו הודעה שמציינת שהם יקבלו את ההוראות שלכם להשלמת הרכישה.

![הודעת אישור הרשמה שמציינת למשתמש שהוא יקבל הוראות תשלום](/img/frontend/registration-manual-notice.png)

והם יקבלו גם הודעה בעמוד ה-**תודה** שלכם עם הוראות התשלום שלכם.

## אישור תשלומים ידניים {#confirming-manual-payments}

כדי לאשר תשלום ידני, עברו לתפריט **תשלומים** בסרגל השמאלי. שם תוכלו לראות את כל התשלומים ברשת שלכם ואת הפרטים שלהם, כולל ה**סטטוס** שלהם. לתשלום ידני תמיד יהיה סטטוס **בהמתנה** עד שתשנו אותו ידנית.

![רשימת תשלומים שמציגה תשלום ידני בהמתנה](/img/admin/payments-list.png)

היכנסו לעמוד התשלום על ידי לחיצה על **קוד ההפניה**. בעמוד זה יש לכם את כל פרטי התשלום הממתין, כגון מזהה הפניה, מוצרים, חותמות זמן ועוד.

![עמוד פרטי תשלום שמציג קוד הפניה, מוצרים וסכומים כוללים](/img/admin/payment-edit.png)

בעמודה הימנית, תוכלו לשנות את סטטוס התשלום. שינוי שלו ל-**הושלם** ו**הפעלת האפשרות הפעל חברות** יפעילו את האתר של הלקוח שלכם והחברות שלו תהיה פעילה.

![עמוד עריכת תשלום עם סטטוס מוגדר כהושלם ומתג הפעל חברות](/img/admin/payment-activate-membership.png)
