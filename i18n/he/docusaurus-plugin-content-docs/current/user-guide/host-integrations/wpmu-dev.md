# אינטגרציה עם WPMU DEV

## סקירה כללית
WPMU DEV היא פלטפורמת WordPress מקיפה שמציעה אחסון, תוספים ושירותים לאתרי WordPress. האינטגרציה הזו מאפשרת סנכרון אוטומטי של דומיינים וניהול תעודות SSL בין Ultimate Multisite לאחסון WPMU DEV.

## תכונות
- סנכרון דומיינים אוטומטי
- ניהול תעודות SSL
- ניסיונות אימות מורחבים לתעודות SSL

## דרישות
האינטגרציה מזהה אוטומטית אם האתר שלכם מתארח ב-WPMU DEV ומשתמשת ב-API המובנה. אם אתם מתארחים ב-WPMU DEV, לא נדרשת הגדרה נוספת.

האינטגרציה בודקת את נוכחות הקבוע `WPMUDEV_HOSTING_SITE_ID`, שמוגדר אוטומטית כשמתארחים ב-WPMU DEV.

## הוראות התקנה

### 1. אימות אחסון WPMU DEV

אם אתם מתארחים ב-WPMU DEV, הקבועים הנדרשים כבר אמורים להיות מוגדרים. ודאו ש:

1. הקבוע `WPMUDEV_HOSTING_SITE_ID` מוגדר בסביבה שלכם
2. יש לכם מנוי WPMU DEV פעיל עם גישה ל-API

### 2. הפעלת האינטגרציה

1. באזור הניהול של WordPress, עברו אל Ultimate Multisite > Settings
2. נווטו ללשונית "Domain Mapping"
3. גללו למטה אל "Host Integrations"
4. הפעילו את אינטגרציית WPMU DEV
5. לחצו על "Save Changes"

## איך זה עובד

### סנכרון דומיינים

כשדומיין ממופה ב-Ultimate Multisite:

1. האינטגרציה משתמשת ב-API של WPMU DEV כדי להוסיף את הדומיין לחשבון האחסון שלכם
2. היא גם מוסיפה אוטומטית את גרסת ה-www של הדומיין
3. WPMU DEV מטפל בהגדרת הדומיין והנפקת תעודת SSL

### ניהול תעודות SSL

האינטגרציה מוגדרת להגדיל את מספר ניסיונות אימות תעודות SSL עבור אחסון WPMU DEV, מכיוון שהנפקה והתקנה של תעודות SSL עשויות לקחת זמן. כברירת מחדל, היא תנסה עד 10 פעמים לאמת את תעודת ה-SSL, בהשוואה ל-5 ניסיונות סטנדרטיים.

## הערות חשובות

### הסרת דומיין

כרגע, ה-API של WPMU DEV לא מספק דרך להסיר דומיינים. כשמיפוי דומיין מוסר ב-Ultimate Multisite, הדומיין יישאר בחשבון האחסון שלכם ב-WPMU DEV. תצטרכו להסיר אותו ידנית מלוח הבקרה של אחסון WPMU DEV אם יש צורך.

### אימות API

האינטגרציה משתמשת במפתח ה-API של WPMU DEV שמאוחסן במסד הנתונים של WordPress כאפשרות `wpmudev_apikey`. זה מוגדר אוטומטית כשאתם מחברים את האתר שלכם ל-WPMU DEV.

## פתרון בעיות

### בעיות חיבור API
- ודאו שהאתר שלכם מחובר כראוי ל-WPMU DEV
- בדקו שהאפשרות `wpmudev_apikey` מוגדרת במסד הנתונים של WordPress
- ודאו שהמנוי שלכם ב-WPMU DEV פעיל

### בעיות בתעודות SSL
- ל-WPMU DEV עשוי לקחת זמן להנפיק תעודות SSL (בדרך כלל 5-15 דקות)
- האינטגרציה מוגדרת לבדוק עד 10 פעמים את תעודות ה-SSL
- אם תעודות SSL עדיין לא מונפקות לאחר ניסיונות מרובים, פנו לתמיכה של WPMU DEV

### הדומיין לא נוסף
- בדקו את יומני Ultimate Multisite לאיתור הודעות שגיאה
- ודאו שהדומיין לא כבר נוסף ל-WPMU DEV
- ודאו שתוכנית האחסון שלכם ב-WPMU DEV תומכת במספר הדומיינים שאתם מוסיפים
