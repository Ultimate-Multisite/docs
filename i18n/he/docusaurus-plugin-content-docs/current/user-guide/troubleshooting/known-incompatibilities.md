# אי-תאימויות ידועות עם תוספים אחרים

לאחר התקנה והפעלה של Ultimate Multisite, ייתכן שתיתקלו בבעיות עקב אי-תאימויות שטרם טופלו עם תוספים אחרים.

אנחנו עובדים באופן פעיל לצמצם את הרשימה הזו בכל גרסה חדשה, אך אם תוסף מופיע כאן, ייתכן שתצטרכו להשבית אותו כדי שרשת Ultimate Multisite תפעל באופן מלא.

להלן רשימת התוספים הבעייתיים ומה כדאי לעשות בכל מקרה:

**חלק מתוספי WPMU DEV** חלק מהתוספים שכבר _אינם_ מתוחזקים על ידי WPMU DEV ידועים כגורמים לעצירת תהליך ההרשמה של Ultimate Multisite בשלב האחרון (המשתמש לא מצליח ליצור חשבון). הנה רשימת התוספים שגורמים לבעיות אלו:

  * Signup Language: _ל-Ultimate Multisite יש הרחבה שמציעה את אותה פונקציונליות, בשם Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _זה כבר מובנה ב-Ultimate Multisite, כך שאין צורך בתוסף נוסף._

  * New Blog Templates: _זה כבר מובנה ב-Ultimate Multisite, כך שאין צורך בתוסף נוסף._

  * Set Password on Multisite Blog Creation: _זה כבר מובנה ב-Ultimate Multisite, כך שאין צורך בתוסף נוסף._

  * Signup TOS: _זה כבר מובנה ב-Ultimate Multisite, כך שאין צורך בתוסף נוסף._

  * Anti-Splog: למרות שהוא פחות חזק, גם זה _מובנה ב-Ultimate Multisite._

**הערה**: ניתן להשתמש בכל שאר תוספי WPMU Dev באופן רגיל לצד Ultimate Multisite. לדוגמה: _Smush_, _Forminator_, _Defender_ ועוד.
