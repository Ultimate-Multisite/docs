# תיעוד Ultimate Multisite {#ultimate-multisite-documentation}

ברוכים הבאים לתיעוד Ultimate Multisite. Ultimate Multisite הופך את WordPress Multisite לפלטפורמת Website as a Service (WaaS) מלאה.

## מקטעי התיעוד {#documentation-sections}

### [User Guide](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

כל מה שאתם צריכים כדי להקים, להגדיר ולנהל את פלטפורמת ה-WaaS שלכם. מכסה התקנה, שערים לתשלום, מיפוי דומיינים, ניהול לקוחות ועוד.

### [Developer Documentation](/developer/hooks) {#developer-documentation}

מקור מידע טכני למפתחים שבונים על Ultimate Multisite. כולל הפניות ל-hooks, תיעוד REST API ומדריכי פיתוח Addons.

### [Addons](/addons) {#addons}

תיעוד עבור תוספי Addon בתשלום המרחיבים את Ultimate Multisite עם תכונות נוספות כמו אינטגרציה עם WooCommerce, אנליטיקה, מכירת דומיינים ועוד.
