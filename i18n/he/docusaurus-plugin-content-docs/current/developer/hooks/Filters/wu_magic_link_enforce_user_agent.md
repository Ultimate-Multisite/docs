# Filter: wu_magic_link_enforce_user_agent

מסנן הקובע אם יש לאכוף אימות User Agent.

הגדר אותו ל-false כדי לאפשר ל-tokens לעבוד על פני דפדפנים/מכשירים שונים. זה מפחית את רמת האבטחה אבל מגדיל את השימושיות.

## פרמטרים {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $enforce | `bool` | האם לאכוף התאמה של User Agent. |

- 2.0.0
