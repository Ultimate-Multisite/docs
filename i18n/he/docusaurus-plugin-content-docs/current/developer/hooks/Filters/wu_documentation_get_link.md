# מסנן: wu_documentation_get_link

מאפשר למפתחי תוספים לסנן את הקישורים.

לא בטוח איך זה יכול להיות שימושי, אבל זה לא מזיק להכניס אותו.

## פרמטרים {#parameters}

| שם | סוג | תיאור |
|------|------|-------------|
| $link | `string` | הקישור הרשום |
| $slug | `string` | ה-slug המשמש לשליפת הקישור |
| $default_link | `string` | הקישור ברירת המחדל הרשום |
