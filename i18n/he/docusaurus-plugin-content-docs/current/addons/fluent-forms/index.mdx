# Fluent Forms {#fluent-forms}

תוסף זה מאפשר להוסיף הגבלות ל-Fluent Forms ב-Ultimate Multisite, עבור מוצרים, מנויים או אתרים.
מכיוון ש-Fluent Forms משתמש בטבלאות מותאמות אישית במקום בסוגי פוסט מותאמים אישית, אפשרויות ההגבלה הרגילות אינן עובדות עם Fluent Forms.

## תכונות {#features}

*   הוספת הגבלות עבור Fluent Forms או Conversational Fluent Forms
*   הלקוחות יקבלו הודעה ידידותית המעודדת אותם לשדרג את המנוי שלהם כאשר הם מגיעים למגבלה
