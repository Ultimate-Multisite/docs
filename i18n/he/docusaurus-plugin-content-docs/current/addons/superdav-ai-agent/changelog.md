# יומן שינויים

## 1.16.0 — שוחרר בתאריך 2026-05-20

### חדש

- **יכולת יצירת לוגו SVG** — כעת ניתן ליצור ולשלב לוגואים מותאמים אישית בפורמט SVG באמצעות Theme Builder, עם חיטוי בטוח המותאם למרחב השמות.
- **העלאת תמונה בראיון גילוי** — ראיון הגילוי של Theme Builder כולל כעת שלב להעלאת תמונה, כדי לספק הקשר עיצובי עשיר יותר.
- **יכולת אימות ניגודיות פלטת צבעים** — בדוק זוגות צבעים לתאימות WCAG לפני ההקפדה עליהם על תבנית.
- **תפריטי אירוח** — Theme Builder יכול כעת ליצור דפי תפריטי אוכל ושתייה מובנים עבור עסקי אירוח.
- **רינדור תצוגה למחשב ולנייד** — צפה בעיצוב שלך במכשי מחשב ונייד במהלך בחירת כיוון העיצוב.
- **פרמטר תווית ניווט** — יכולת Create Menu תומכת כעת ב־`navigation_label` ייעודי, נפרד משם הדף.
- **זמינות כלי Tier 1** — sd-ai-agent/site-scrape הוא כעת כלי Tier 1 הזמין כברירת מחדל ב-Theme Builder.

### תוקן

- **מטמון לקוח AI** — כעת מופעל על ידי טרנזיינטים (transients) לצורך שמירה בין בקשות, מה שמונע אובדן נתונים במשימות סוכנים ארוכות טווח.
- **קישורי פעולה שורת תוסף** — תוקן ושונה לשם ברור יותר.

## 1.10.0 — שוחרר בתאריך 2026-05-05

### חדש

- **חיפוש אינטרנט באמצעות Tavily** — הוספת Tavily כמספק חיפוש לתוצאות חיפוש אינטרנט עשירות יותר, לצד Brave Search.
- **מיומנויות מובנות מודעות לתבניות** — Block Themes, Classic Themes, Kadence Blocks ו-Kadence Theme skill guides מגיעים כעת עם התוסף.
- **יכולת טופס יצירת קשר בבונה אתרים** — הוסף טופס יצירת קשר לכל דף ישירות מממשק הצ'אט.

### שופר

- **שילוב WooCommerce שופץ** — משתמש כעת ב-WooCommerce APIs מובנים ליציבות ותאימות טובות יותר.
- **רשימת ספקים מתעדכנת אוטומטית** — כאשר תוסף כלשהו מופעל או מבוטל.

### תוקן

- **יכולת navigate-to** — תוקן לולאת ההרענון האינסופית בדפי ניהול מסוימים.
- **יכולת list-posts** — כעת פותר נכון שמות קטגוריות ותגיות ל־slugs.
- **פקודות WP-CLI** — שוחזרו כינויי ה־namespace החסרים לאחר refactor קודם.
- **אוטומציה של אירועים** — מטפל בצורה חלקה באתרים שבהם טבלאות האוטומציה עדיין לא נוצרו.
- **יכולת memory-save** — משתמש כעת בתחילתו הנכונה של ה־namespace בבונה הנחיות המערכת.
- **תוצאות כלי Scalar** — עטופות כעת כראוי לפני שהן מוחזרות ל־AI.
- **סטטיסטיקות שימוש** — מטפלות כעת כראוי בפורמט מפתח היכולת הישן (legacy) בעדכון מגרסאות ישנות יותר.
