# יומן שינויים {#changelog}

## 1.18.0 — שוחרר ב-2026-06-29 {#1180--released-on-2026-06-29}

### חדש {#new}

- **כלי Google Calendar** — קריאת לוחות שנה ואירועים מוגדרים עבור אוטומציות מודעות-לוח זמנים
- **מיפוי אנשי קשר ועוזרי משתתפים** — התאמת משתתפי לוח שנה למשתמשי האתר ולאנשי קשר
- **שערי אישור אנושי ורשומות תזכורת** — השהיית אוטומציות לבדיקה ומניעת התראות כפולות
- **ספק SMS של TextBee** — שליחת התראות הודעות טקסט מוגדרות דרך TextBee
- **חבילת ליווי מתקדמת** — הוספת כלי מערכת קבצים למפתחים מהימנים, מסד נתונים, WP-CLI, משגר REST, בונה plugin, צילום מצב של git, ניהול משתמשים ומדדי ביצועים המופצים בנפרד מגרסת WordPress.org

### שופר {#improved}

- **הגדרת שירות Superdav מנוהל** — הוספת נקודות קצה של שירות מתארח והקצאת חיבור אוטומטית לאתרים נתמכים
- **אריזת גרסה** — בניית קובצי ZIP נפרדים לליבה ול-Advanced, פרסום שניהם ב-GitHub, ושליחת חבילת הליבה בלבד ל-WordPress.org

### תוקן {#fixed}

- **אמינות בקשות AI** — שיפור בחירת מודל, זמני קצוב לבקשות, מודל ברירת מחדל להגדרה, טיפול בטקסט הנמקה והנחיות לניסיון חוזר עבור קריאות כלי לא תקפות
- **הקשחת לוח שנה ותזכורות** — חיזוק אסימוני Google Calendar ומניעת כפילויות בתזכורות
- **חידוש קליטה ואישורים** — תיקון הפעלות קליטה בחזית וחידוש יכולות מאושרות
- **בעיות סקירת אריזה של WordPress.org** — טיפול במשוב סקירת אריזה עבור גרסת הליבה

## 1.16.0 — שוחרר ב-2026-05-20 {#1160--released-on-2026-05-20}

### חדש {#new-1}

- **יכולת יצירת Logo SVG** — Theme Builder יכול כעת ליצור ולהטמיע קובצי SVG של לוגו מותאם אישית עם ניקוי בטוח למרחבי שמות
- **העלאת תמונה בריאיון גילוי** — ריאיון הגילוי של Theme Builder כולל כעת שלב העלאת תמונה להקשר עיצובי עשיר יותר
- **יכולת אימות ניגודיות פלטה** — בדיקת זוגות צבעים לעמידה ב-WCAG לפני החלה על theme
- **תפריטי אירוח** — Theme Builder יכול כעת ליצור עמודי תפריט אוכל ומשקאות מובנים לעסקי אירוח
- **רינדור תצוגה מקדימה למחשב ולנייד** — תצוגה מקדימה של העיצוב שלך במכשירי מחשב ונייד במהלך בחירת כיוון העיצוב
- **פרמטר תווית ניווט** — יכולת יצירת תפריט תומכת כעת ב-`navigation_label` נפרד מכותרת העמוד
- **זמינות כלי Tier 1** — sd-ai-agent/site-scrape הוא כעת כלי Tier 1 הזמין כברירת מחדל ב-Theme Builder

### תוקן {#fixed-1}

- **מטמון AI Client** — מגובה כעת ב-transients להתמדה בין בקשות, ומונע אובדן נתונים במשימות agent ארוכות
- **קישורי פעולת שורת plugin** — תוקנו ושמם שונה לשם בהירות

## 1.10.0 — שוחרר ב-2026-05-05 {#1100--released-on-2026-05-05}

### חדש {#new-2}

- **חיפוש אינטרנט של Tavily** — הוספת Tavily כספק חיפוש לתוצאות חיפוש אינטרנט עשירות יותר לצד Brave Search
- **מיומנויות מובנות מודעות-theme** — מדריכי מיומנות עבור Block Themes, Classic Themes, Kadence Blocks ו-Kadence Theme נכללים כעת עם ה-plugin
- **יכולת טופס יצירת קשר לבונה אתרים** — הוספת טופס יצירת קשר לכל עמוד ישירות מממשק הצ'אט

### שופר {#improved-1}

- **אינטגרציית WooCommerce עברה ארגון מחדש** — משתמשת כעת בממשקי API מקוריים של WooCommerce לאמינות ותאימות טובות יותר
- **רשימת ספקים מתרעננת אוטומטית** — כאשר plugin כלשהו מופעל או מושבת

### תוקן {#fixed-2}

- **יכולת navigate-to** — תוקנה לולאת טעינה מחדש אינסופית בחלק מעמודי הניהול
- **יכולת list-posts** — כעת פותרת נכון שמות קטגוריות ותגיות ל-slugs
- **פקודות WP-CLI** — שוחזרו כינויי מרחב שמות חסרים לאחר ארגון מחדש קודם
- **אוטומציית אירועים** — מטפלת בחינניות באתרים שבהם טבלאות אוטומציה עדיין לא נוצרו
- **יכולת memory-save** — משתמשת כעת בקידומת מרחב השמות הנכונה בבונה הוראות המערכת
- **תוצאות כלי סקלריות** — נעטפות כעת נכון לפני החזרתן ל-AI
- **סטטיסטיקות שימוש** — מטפלות כעת נכון בפורמט מפתח היכולת הישן בעת שדרוג מגרסאות ישנות יותר
