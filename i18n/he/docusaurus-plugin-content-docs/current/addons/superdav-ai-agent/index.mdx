# Superdav AI Agent

Superdav AI Agent הוא עוזר AI מתקדם ל-WordPress שמרחיב את Gratis AI Agent עם מיומנויות מודעות לתבנית, יכולות חיפוש משופרות באינטרנט, תכונות לבניית אתרים ואינטגרציות אוטומציה אופציונליות. גרסה 1.18.0 מוסיפה כלי קריאה של Google Calendar, מיפוי אנשי קשר ומשתתפים, שערי אישור אנושיים, מניעת כפילות בתזכורות, התראות SMS של TextBee, הקצאת שירות Superdav מנוהלת, וחבילת Advanced נלווית שמופצת בנפרד עבור תהליכי עבודה אמינים למפתחים.

## תכונות עיקריות {#key-features}

- **מיומנויות מודעות לתבנית** — מתאימה אוטומטית הנחיות ויכולות על בסיס תבנית ה-WordPress הפעילה והתוספים המותקנים שלך
- **ספקי חיפוש אינטרנט מרובים** — בחר בין Brave Search ל-Tavily לקבלת תוצאות חיפוש עשירות יותר ויכולות מחקר מתקדמות
- **טופסי יצירת קשר לבונה אתרים** — צור טופסי יצירת קשר מתפקדים במלואם ישירות מממשק הצ'אט באמצעות תוסף הטפסים המותקן שלך
- **בונה אתרים משופר** — תוכניות משופרות לבניית אתר רב-שלבית עם גילוי תוספים טוב יותר והתאוששות משגיאות
- **מיומנויות תבנית מובנות** — הנחיות ייעודיות עבור תבניות בלוקים, תבניות קלאסיות, Kadence Blocks ו-Kadence Theme
- **אינטגרציית WooCommerce משופרת** — עברה ארגון מחדש לשימוש ב-APIs מקוריים של WooCommerce לשיפור אמינות ותאימות
- **זיהוי ספקים אוטומטי** — מזהה ומרענן אוטומטית תוספים ותבניות זמינים כאשר הם מופעלים או מושבתים
- **יצירת לוגו מותאם אישית** — צור והטמע לוגואים מותאמים אישית מסוג SVG עם ניקוי אוטומטי בטוח למרחבי שמות
- **אימות ניגודיות לוח צבעים** — בדוק זוגות צבעים לעמידה בנגישות WCAG לפני החלה על התבנית שלך
- **גילוי משופר בתמונות** — העלה תמונות במהלך ריאיון הגילוי של Theme Builder כדי להשפיע על כיוון העיצוב
- **תפריטי אירוח** — צור עמודי תפריטי אוכל ומשקאות מובנים למסעדות, בתי קפה ועסקי אירוח
- **תצוגות מקדימות למחשב שולחני ולנייד** — הצג תצוגה מקדימה של העיצוב שלך במכשירי מחשב שולחני ונייד במהלך בחירת כיוון העיצוב
- **גירוד אתרים ברמה 1** — sd-ai-agent/site-scrape זמין כעת כברירת מחדל לניתוח אתרי מתחרים ואתרי השראה
- **כלי קריאה של Google Calendar** — קרא לוחות שנה ואירועים מוגדרים כדי שאוטומציות יוכלו להסיק מסקנות מהקשר לוח הזמנים הקרוב
- **מיפוי אנשי קשר ומשתתפים** — התאם משתתפי לוח שנה למשתמשי אתר ואנשי קשר מוכרים לפני יצירת עבודת המשך
- **שערי אישור אנושיים** — השהה אוטומציות רגישות לבדיקה לפני שליחת הודעות, חידוש עבודה או ביצוע שינויים
- **רשומות תזכורת ומניעת כפילות** — עקוב אחר תזכורות שכבר נשלחו כדי שהרצות אוטומציה חוזרות לא יודיעו לאותו נמען פעמיים
- **התראות SMS של TextBee** — שלח התראות הודעת טקסט מוגדרות דרך TextBee כאשר ספק SMS מופעל
- **הקצאת שירות Superdav מנוהלת** — חבר אתרים נתמכים לנקודות קצה מתארחות של Superdav עם הגדרה אוטומטית במקום הגדרת נקודת קצה ידנית
- **חבילת Advanced נלווית** — כלי מפתחים אמינים המופצים בנפרד עבור תהליכי עבודה של מערכת קבצים, מסד נתונים, WP-CLI, משגר REST, בונה תוספים, תמונת מצב של git, ניהול משתמשים ומדדי ביצועים

## דרישות {#requirements}

- WordPress 6.0 ומעלה
- PHP 8.1 ומעלה
- WordPress Multisite (מופעל ברשת) או התקנת אתר יחיד
- תוסף מחבר ספק AI פעיל (Anthropic, OpenAI, או תואם)
- Gratis AI Agent v1.9.0 ומעלה (Superdav AI Agent מרחיב את Gratis AI Agent)

## התקנה {#installation}

1. העלה את קובצי התוסף אל `/wp-content/plugins/superdav-ai-agent`
2. הפעל את התוסף דרך מסך **תוספים** ב-WordPress
3. ודא שגם Gratis AI Agent מופעל
4. נווט אל **Gratis AI Agent → הגדרות** והגדר את ספק חיפוש האינטרנט שלך, חיבור שירות Superdav מנוהל, Google Calendar או פרטי גישה של TextBee לפי הצורך

החבילה של WordPress.org מכילה את תכונות הליבה של Superdav AI Agent. חבילת Advanced הנלווית נבנית ומופצת בנפרד עבור סביבות שבהן כלי מפתחים אמינים מותקנים ומורשים במפורש.

## התחלה מהירה {#quick-start}

לאחר ההפעלה, Superdav AI Agent משפר אוטומטית את התקנת Gratis AI Agent שלך עם מיומנויות מודעות לתבנית ויכולות נוספות. פתח את חלונית **Gratis AI Agent** בסרגל הצד של ניהול WordPress והתחל להשתמש בתכונות החדשות.

**הוראות לדוגמה:**

```
הוסף טופס יצירת קשר לעמוד יצירת הקשר.
```

```
חפש באינטרנט את שיטות העבודה המומלצות העדכניות ביותר לאבטחת WordPress.
```

```
צור מקטע גיבור באמצעות תבניות הבלוקים של התבנית הפעילה.
```

```
בדוק את אירועי Google Calendar המוגדרים למחר ונסח תזכורות למשתתפים.
```

```
שלח תזכורת SMS לאחר שאאשר את ההודעה.
```

## מיומנויות מודעות לתבנית {#theme-aware-skills}

Superdav AI Agent מזהה אוטומטית את התבנית הפעילה והתוספים המותקנים שלך, וטוען את המיומנות המתאימה המודעת לתבנית אל הקשר העוזר. המיומנויות הזמינות כוללות:

| מיומנות | חל על |
|---|---|
| **תבניות בלוקים** | תבניות המשתמשות בארכיטקטורה מבוססת הבלוקים של WordPress (עריכת אתר מלאה) |
| **תבניות קלאסיות** | תבניות WordPress מסורתיות המשתמשות בתבניות PHP ובעורך הקלאסי |
| **Kadence Blocks** | אתרים שבהם תוסף Kadence Blocks פעיל |
| **Kadence Theme** | אתרים שבהם תבנית Kadence היא התבנית הפעילה |

ראה [מיומנויות מודעות לתבנית](../../user-guide/configuration/theme-aware-skills) לקבלת הנחיות מפורטות על כל מיומנות.

## ספקי חיפוש אינטרנט {#internet-search-providers}

Superdav AI Agent v1.10.0 מוסיף את **Tavily** כחלופה ל-Brave Search עבור שאילתות חיפוש באינטרנט. שני הספקים מציעים תוצאות חיפוש בזמן אמת וניתן להגדיר אותם ב-**Gratis AI Agent → הגדרות → מתקדם**.

| ספק | יכולות | הגדרה |
|---|---|---|
| **Brave Search** | תוצאות חיפוש מהירות וממוקדות פרטיות | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | תוצאות חיפוש עשירות יותר, יכולות מחקר מתקדמות | [Tavily API](https://tavily.com/) |

ראו [חיפוש באינטרנט](../../user-guide/configuration/internet-search) לקבלת הוראות הגדרה.

## לוחות שנה, אישורים וזרימות עבודה של התראות {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 מרחיב זרימות עבודה של אוטומציה מעבר למשימות תוכן ובניית אתרים:

- **כלי Google Calendar** מאפשרים לסוכן לקרוא לוחות שנה מוגדרים ופרטי אירועים כאשר פרטי הגישה ללוח השנה זמינים.
- **מסייעי אנשי קשר ומשתתפים** ממפים משתתפי אירועים למשתמשי WordPress או לאנשי קשר מוכרים לפני הכנת תזכורות או משימות מעקב.
- **שערי אישור אנושי** מאפשרים לאוטומציה להשהות עד שמשתמש מורשה יסקור את הפעולה המוצעת.
- **רשומות תזכורת** מונעות תזכורות כפולות של לוח שנה או התראות כאשר אוטומציה מופעלת מחדש או מתוזמנת שוב ושוב.
- **תמיכת SMS של TextBee** מאפשרת התראות בהודעות טקסט כאשר ספק TextBee מוגדר.

השתמשו בכלים אלה לסיוע מודע-לזמנים, מעקב אחר משתתפים וזרימות עבודה של תזכורות שעדיין דורשות סקירה אנושית לפני שליחת הודעות רגישות.

## הגדרה {#configuration}

כל ההגדרות נמצאות תחת **Gratis AI Agent → Settings** בממשק הניהול של WordPress. Superdav AI Agent מוסיף את אפשרויות ההגדרה הבאות:

| הגדרה | תיאור |
|---|---|
| **ספק חיפוש באינטרנט** | בחרו בין Brave Search ל-Tavily (אופציונלי) |
| **Tavily API Key** | מפתח API לחיפוש באינטרנט של Tavily (אם Tavily נבחר) |
| **שירות Superdav מנוהל** | חברו אתרים נתמכים לשירות Superdav המתארח עם הקצאת endpoint אוטומטית |
| **פרטי גישה ל-Google Calendar** | הגדירו גישה ללוח שנה כדי שהסוכן יוכל לקרוא לוחות שנה ואירועים |
| **ספק SMS של TextBee** | הגדירו פרטי גישה ל-TextBee עבור זרימות עבודה מאושרות של התראות SMS |

## חבילת עזר מתקדמת {#advanced-companion-package}

Superdav AI Agent v1.18.0 מוסיף חבילת עזר מתקדמת המופצת בנפרד עבור זרימות עבודה של מפתחים מהימנים. יכולות אלה אינן כלולות בחבילת הליבה של WordPress.org. התקינו את חבילת העזר רק באתרים שבהם מנהלי מערכת נותנים אמון מפורש בסוכן לגשת לכלי פיתוח.

כלים מתקדמים כוללים פעולות מערכת קבצים, גישה למסד נתונים, WP-CLI, זרימות עבודה של מנתב REST, כלי עזר לבניית תוספים, צילומי מצב של git, מסייעי ניהול משתמשים וכלי מדידה. הגבילו כלים אלה למנהלי מערכת מהימנים וסקרו את הגדרת שערי האישור לפני הפעלתם באתרי ייצור.

## תיקוני באגים ב-v1.10.0 {#bug-fixes-in-v1100}

- תוקנה יכולת `navigate-to` שגרמה ללולאות טעינה מחדש אינסופיות בחלק מעמודי הניהול
- תוקנה יכולת `list-posts` שלא פתרה נכון שמות קטגוריות ותגיות ל-slugs
- תוקנו פקודות WP-CLI שחסרו להן כינויי namespace לאחר ארגון מחדש קודם
- תוקן טיפול חינני של אוטומציית אירועים באתרים שבהם טבלאות האוטומציה עדיין לא נוצרו
- תוקנה יכולת `memory-save` כך שתשתמש בקידומת ה-namespace הנכונה בבונה הוראות המערכת
- תוקן עטיפה נכונה של תוצאות כלי סקלריות לפני החזרתן ל-AI
- תוקן טיפול נכון של סטטיסטיקות שימוש בפורמט מפתח היכולת הישן בעת שדרוג מגרסאות ישנות יותר
