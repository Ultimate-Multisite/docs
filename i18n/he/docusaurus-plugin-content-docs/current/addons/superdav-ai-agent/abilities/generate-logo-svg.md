# יצירת לוגו SVG

היכולת **Generate Logo SVG** מאפשרת ל-Theme Builder ליצור ולהטמיע לוגואים מותאמים אישית בפורמט SVG ישירות באתר ה-WordPress שלכם, עם ניקוי אוטומטי ובטוח (namespace-safe sanitisation).

## סקירה כללית {#overview}

יכולת זו יוצרת לוגואים בפורמט גרפי וקטורי מותאם (SVG), בהתבסס על כיוון המותג וההעדפות העיצוב של האתר שלכם. קבצי ה-SVG שנוצרים עוברים ניקוי אוטומטי כדי להבטיח שהם בטוחים לשימוש ב-WordPress, תוך שמירה על המראה המקורי והשלמות החזותית.

## פרמטרים {#parameters}

| פרמטר | סוג | נדרש | תיאור |
|-----------|------|----------|-------------|
| `site_name` | string | כן | שם האתר עבורו יש ליצור את הלוגו |
| `style` | string | כן | סגנון העיצוב (לדוגמה: "מודרני", "קלאסי", "מינימליסטי", "משחקי") |
| `colors` | array | לא | מערך של קודי צבע הקס' (hex) לשימוש בלוגו (לדוגמה: `["#678233", "#ffffff"]`) |
| `width` | number | לא | רוחב ה-SVG בפיקסלים (ברירת מחדל: 200) |
| `height` | number | לא | גובה ה-SVG בפיקסלים (ברירת מחדל: 200) |
| `include_text` | boolean | לא | האם לכלול את שם האתר כטקסט בלוגו (ברירת מחדל: true) |

## פורמט הפלט {#output-format}

היכולת מחזירה מחרוזת SVG עם המבנה הבא:

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## התנהגות ניקוי ה-SVG {#svg-sanitisation-behaviour}

קבצי ה-SVG שנוצרו עוברים ניקוי אוטומטי ובטוח (namespace-safe sanitisation) כדי:

- **להסיר מאפיינים לא בטוחים** — מסיר מנגנוני אירועים, סקריפטים ומאפיינים פוטנציאלית מסוכנים.
- **לשמור על מרחבי שמות** — שומר על מרחבי השמות של ה-SVG (xmlns, xlink) לצורך תצוגה תקינה.
- **לאמת את המבנה** — מוודא שה-SVG תואם לתקני W3C.
- **לקודד ישויות** — מבצע בריחה נכונה של תווים מיוחדים בתוכן הטקסט.
- **להסיר הפניות חיצוניות** — מסיר גיליונות סגנון והפניות תמונות חיצוניות.

זה מבטיח שה-SVG בטוח להטמעה ישירות ב-WordPress מבלי לדרוש ניקוי נוסף.

## דוגמה לשימוש {#usage-example}

**Prompt:**
```
Generate a modern logo for my tech startup called "CloudSync" using blue and white colors.
```

**Result:**
היכולת יוצרת לוגו SVG שמ:
- משלב את שם האתר "CloudSync".
- משתמש במערכת הצבעים הכחול והלבן שצוינה.
- עוקב אחר עקרונות עיצוב מודרניים.
- מנוקה אוטומטית ומוכן לשימוש.

## אינטגרציה עם Theme Builder {#integration-with-theme-builder}

כאשר משתמשים בבחירת כיוון העיצוב של Theme Builder, יכולת Generate Logo SVG:

1. מנתחת את כיוון העיצוב והפלטת הצבעים שלכם.
2. יוצרת לוגו SVG מותאם אישית התואם להעדפות שלכם.
3. משבצת אוטומטית את הלוגו באזור הכותרת/המיתוג של האתר שלכם.
4. שומרת את ה-SVG כ לוגו מותאם אישית במדיה של WordPress.

## שיטות עבודה מומלצות {#best-practices}

- **ספקו העדפות סגנון ברורות** — תארו את סגנון העיצוב שאתם רוצים (מודרני, קלאסי, משחקי וכו').
- **ציינו צבעים** — כללו את צבעי המותג שלכם לקבלת עקביות.
- **בדקו תצוגה** — ודאו שהלוגו מוצג כראוי על גבי גדלים שונים של מסך.
- **התאמה אישית נוספת** — השתמשו בכלים להתאמת לוגו של WordPress כדי להתאים את הגודל והמיקום.

## מגבלות {#limitations}

- לוגואים SVG נוצרים כגרפיקה סטטית (ולא מונפשת).
- לוגואים מורכבים עם הרבה אלמנטים עשויים לדרוש ליטוש ידני.
- גופנים מותאמים אישית אינם נתמכים; הטקסט משתמש בגופני המערכת.
- מימדים גדולים או קטנים מאוד עלולים להשפיע על האיכות.
