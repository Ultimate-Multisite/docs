# מע"מ

Multisite Ultimate: מע"מ אירופי

מוסיף תכונות וכלים נוספים לטיפול המיסים המוגדר במקור ב-Multisite Ultimate, כדי לתמוך טוב יותר במע"מ האירופי. מוסיף אוטומטית שיעורי מע"מ של האיחוד האירופי לדף שיעורי המיסים (Tax Rates) מ...

## התקנה {#installation}

1. העלה את 'ultimate-multisite-vat' לתיקייה '/wp-content/plugins/'
2. הפעל את ה-plugin דרך תפריט ה-'Plugins' ב-WordPress
3. עבור אל Ultimate Multisite > Settings > Taxes והפעל את EU Vat.
4. לחץ על Tax Rates, עדכן את שיעורי מע"מ האירופיים בתחתית, ושמור את שיעורי המיסים.
5. סיימת!
