# אינטגרציה עם WooCommerce

האינטגרציה עם WooCommerce עובדת בצורה דומה מאוד לאופן שבו עובד ה-Manual Gateway שלנו. היא מציעה גמישות עצומה למשתמשים שלך, ומאפשרת להם לבחור מבין כל צורות התשלום הזמינות בהתקנה של WooCommerce שלך. וזה פשוט: ברגע שתקופת החיוב של המשתמש מסתיימת, המנוי שלו מושעה. במקביל, יווצר הזמנה בהתקנת WooCommerce של האתר הראשי של הרשת. המשתמש יקבל לאחר מכן חשבונית במייל שלו, עם קישור לבצע את התשלום. לאחר קבלת התשלום, המנוי מתחדש.

## תכונות

- אינטגרציה קלה בין Ultimate Multisite ל-WooCommerce
- מספר שיטות תשלום זמינות
- תהליך אוטומטי לחלוטין עם WooCommerce Subscriptions

## תיעוד

**הגדרת האינטגרציה עם WooCommerce**

**הרחבת אפשרויות התשלום שלך עם Ultimate Multisite: אינטגרציה עם WooCommerce**

## התקנה

1. העלה את 'ultimate-multisite-woocommerce' לתיקייה '/wp-content/plugins/'
2. הפעל את התוסף דרך תפריט 'Plugins' ב-WordPress
3. עקוב אחר המדריך המפורט (Wizard) להגדרת התוסף

## מדריך שימוש

_**הערה חשובה: מאמר זה מתייחס לגרסה 2.x של Ultimate Multisite.**_

_**שים לב:** Ultimate Multisite: WooCommerce Integration דורש ש-WooCommerce יהיה מופעל לפחות באתר הראשי שלך._

אנחנו יודעים ש-Stripe ו-PayPal אינם זמינים במדינות מסוימות, מה שמגביל או מעכב את המשתמשים של Ultimate Multisite מלהשתמש ביעילות בתוסף שלנו. לכן יצרנו **תוסף כדי לשלב את WooCommerce**.

_WooCommerce_ הוא תוסף eCommerce פופולרי מאוד. מפתחים ברחבי העולם יצרו תוספים כדי לשלב אליו שערים שונים לתשלום. ניצלנו זאת כדי להרחיב את שערים התשלום שתוכל להשתמש בהם במערכת החיוב של Ultimate Multisite.

## הגדרת התוסף

ראשית, עליך להתקין את תוסף ה-WooCommerce Integration, והוא **חינמי למשתמשים מורשים**. כדי להתקין אותו, לחץ על Ultimate Multisite ב-Dashboard של ה-Super admin ובחר **Add-ons**.

![Add-ons page](/img/admin/addons-page.png)

כאן תוכל למצוא את כל התוספים של Ultimate Multisite. לחץ על תוסף **Ultimate Multisite: WooCommerce Integration**.

![Add-ons page showing WooCommerce Integration](/img/admin/addons-page.png)

חלון קופץ עם פרטי התוסף. פשוט לחץ על **Install Now** *.*

<!-- Screenshot unavailable: WooCommerce Integration addon install popup with Install Now button -->

לאחר ההתקנה, תופנה לדף התוספים. כאן, פשוט לחץ על **Network Activate** והתוסף _WooCommerce_ יופעל ברשת שלך.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for WooCommerce Integration -->

לאחר ההפעלה, אם עדיין לא התקנת והפעלת את תוסף WooCommerce באתר שלך, **תקבל תזכורת**.

<!-- Screenshot unavailable: Admin notice reminding to install and activate WooCommerce plugin -->

לאחר ההתקנה, תראה אפשרות נוספת בהגדרות התשלום של Ultimate Multisite. משם תוכל **להפעיל** את האפשרות של _WooCommerce_.

![Payment gateway settings](/img/config/settings-payment-gateways.png)

תוכל להגדיר כמה הגדרות בסיסיות, כמו שם התצוגה שיופיע בחזית האתר והוראות שינחו את המשתמשים שלך בתהליך ה-checkout. לאחר מכן תוכל לעבור ל**הגדרות WooCommerce של האתר הראשי** כדי להפעיל ולהגדיר את שער התשלום המועדף עליך.

## כיצד לנהל את התשלום

האינטגרציה עם WooCommerce עובדת בצורה דומה מאוד לאופן שבו עובד ה-**Manual Gateway** שלנו. החיסרון הוא שכל תשלום חייב להתבצע ידנית בסוף תקופת החיוב. כך עובד התהליך:

  * ברגע שסבב החיוב של המשתמש מסתיים, המנוי מושעה. במקביל, יווצר הזמנה בהתקנת _WooCommerce_ באתר הראשי של הרשת שלך.

  * המשתמשים יקבלו לאחר מכן חשבונית במייל שלהם, עם קישור לבצע את התשלום (הקישור מופיע גם בהיסטוריית החיוב שלהם מה-dashboard של תת-האתר). קישור זה מפנה את המשתמש ל-**checkout** של _**WooCommerce**_, שם יוכל לבחור מבין **שערים זמינים של WooCommerce** כדי לעבד את התשלום שלו.

  * לאחר קבלת התשלום, המנוי מתחדש, התשלום נרשם ב-Ultimate Multisite וההזמנה מוגדרת כהושלמה.

### כמה הערות חשובות:

  * הפונקציונליות המוגדרת כברירת מחדל של Ultimate Multisite עדיין עובדת עם תבנית האינטגרציה הזו. לדוגמה, **הנפקת החזרים** ממסך **ניהול המנויים** תתקשר עם WooCommerce כדי להנפיק את ההחזר באמצעות שער התשלום שנבחר.

  * Ultimate Multisite: WooCommerce Integration מסתמך על WP-Cron כדי לעבוד. זה עובד מצוין ברוב המקרים, אבל אם לאתר יש תנועה נמוכה, ה-cron לא מקבל הנחיה לבדוק אירועים מתוזמנים והאירוע מוחמץ, מה שיכול לגרום לעיכובים בהרצה.

## תוסף WooCommerce Subscriptions

אתחול תשלום ידני בכל חודש אינו אידיאלי עבור משתמשים סופיים, מכיוון שמנוי צפוי להתחדש אוטומטית בכל מחזור חיוב. לכן תכצבנו את תוסף ה-WooCommerce Integration כדי **לתמוך בתוסף WooCommerce Subscriptions**. זה יאוטמט את התשלום ויקל על המעבר עבור המשתמשים הסופיים שלך.

כל מה שאתה צריך לעשות הוא **להתקין** את [Woocommerce Subscriptions plugin](https://woocommerce.com/products/woocommerce-subscriptions/) ולוודא שהוא **מופעל באתר הראשי שלך**. לאחר מכן חזור להגדרות התשלום של Ultimate Multisite ו**הפעל את האינטגרציה**.

וזהו. המשתמשים הסופיים שלך כבר לא יצטרכו לבצע תשלום ידני בסוף תקופת החיוב שלהם.

מלבד יצירת הזמנה באתר הראשי שלך תחת _WooCommerce_, הוא ייצור גם מנוי עבור כל חשבון.
