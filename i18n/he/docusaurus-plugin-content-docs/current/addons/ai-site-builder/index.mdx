# בונה אתרים מבוסס AI

ultimate-multisite-ai-site-builder

## דרישות קדם

- WordPress 5.3 ומעלה
- PHP 7.4 ומעלה
- תוסף Multisite Ultimate (פעיל)

## התקנה

1. העלה את קבצי התוסף לתיקייה `/wp-content/plugins/`
2. הפעל את התוסף דרך תפריט 'Plugins' ב-WordPress
3. הגדר את הגדרות התוסף במנהל הרשת (network admin)

## פיתוח

### הגדרה

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

```

│   └── class-*-test.php         # Test cases
```

### בדיקות

התוסף כולל מסגרת בדיקות מקיפה:

- **בדיקות יחידה (Unit Tests)**: בדיקות מבוססות PHPUnit עם חבילת בדיקות של WordPress
- **מחלקת בדיקה בסיסית (Base Test Class)**: מספקת מתודות עזר לתרחישי בדיקה נפוצים
- **כיסוי בדיקות (Test Coverage)**: מוגדר למעקב אחר כיסוי הקוד
- **בדיקות רשת מרובת אתרים (Multisite Testing)**: הבדיקות רצות בסביבת Multisite

### תקני קוד

- **WordPress Coding Standards**: אוכפים באמצעות PHPCS
- **תאימות ל-PHP 7.4+**: נתמכות תכונות PHP מודרניות
- **ניתוח סטטי (Static Analysis)**: שילוב PHPStan לבדיקת סוגי נתונים
- **מודרניזציה של קוד (Code Modernization)**: כללי Rector לשדרוגי PHP

### מערכת בנייה (Build System)

- **עיבוד נכסים (Asset Processing)**: מיניפיקציה לקבצי CSS/JS
- **תרגום (Translation)**: יצירת קובצי POT
- **אריזה (Packaging)**: יצירת ארכיון להפצה
- **מצב פיתוח (Development Mode)**: נכסים לא ממוזערים (unminified) לצורך דיבוג

## תצורה

התוסף תומך באפשרויות תצורה שונות הנגישות דרך ממשק מנהל הרשת.

## Hooks ופילטרים

### פעולות (Actions)

- `ultimate-multisite-ai-site-builder_init` - מופעל לאחר אתחול התוסף
- `ultimate-multisite-ai-site-builder_loaded` - מופעל לאחר טעינת כל קבצי התוסף

### פילטרים (Filters)

- `ultimate-multisite-ai-site-builder_settings` - שינוי הגדרות התוסף
- `ultimate-multisite-ai-site-builder_enabled` - דריסה של סטטוס ההפעלה/כיבוי של התוסף
