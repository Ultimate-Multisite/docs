# יומן שינויים של Site Exporter

Version: 1.0.6 - שוחרר ב-2026-05-11
* חדש: ייצוא אתר כולל כעת קובץ index.php שמפעיל את עצמו, כך שניתן להתקין את קובץ ה-ZIP על שרת חדש מבלי להתקין plugin נפרד.
* חדש: ייצוא רשת מאפשר למנהלים לייצא את כל אתרי המשנה בארכיון אחד, ישירות מדף המנהל של Site Export.

Version: 1.0.5 - שוחרר ב-2026-05-05
* תיקון: הוספנו את __callStatic ל-polyfill stub של WP_CLI כדי לטפל בפונקציות שלא נאמדו (unmocked) כמו get_config, ובכך למנוע כשלים בהפעלה הראשונית של המבחנים.

Version: 1.0.4 - שוחרר ב-2025-11-25

* תיקון: תיקון התאימות עם Ultimate Multisite 2.4.

Version: 1.0.2 - שוחרר ב-2025-09-28

* שינוי: שינוי הקידומת ל-ultimate-multisite; עדכון ה-text domain; והעלאת גרסה.

### Version 1.0.1 - שוחרר ב-2023-08-09 {#version-101---released-on-2023-08-09}

* שיפור: מוודא כעת שמתקבל נתיב הקובץ הנכון של קובץ הייבוא.
* תיקון: השבתת המטמון של האובייקטים (object cache) כדי למנוע שגיאות במהלך הייבוא.
* תיקון: תיקון הבעיה שבה קישור ההורדה של plugin הייבוא לא עבד.
