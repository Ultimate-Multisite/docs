# יומן שינויים של תוכניות ממוצעות {#metered-plans-changelog}

Version 1.1.0 - שוחרר ב-2026-05-05
- חדש: חיוב AI tokens עבור תת-אתרי multisite — מעקב וחיוב על שימוש ב-AI tokens באתרים של לקוחות עם תעריפים הניתנים להגדרה לכל token
- חדש: אכיפת מחברים (Connector enforcement) נבנתה מחדש עם גילוי מגבלות דינמי ו-write-through, מה שמבטיח דיוק בזמן אמת בכל המחברים
- תיקון: שדרוגי טבלאות ה-Database מיושרים כעת כראוי עם הגדרות הסכמה (schema) של BerlinDB, ומונעים כשלים במהלך התקנות חדשות
- תיקון: קריאות משוב (callbacks) של שדרוג ה-Database הומרות לפורמט הנכון, מה שמפתור כשלים שקטים במהלך השדרוג
- תיקון: כעת מקבלים ערכים שברתיים (Fractional values) בשדה הקלט AI Usage Overage Markup
- תיקון: נפתרו שגיאות קריטיות ובעיות של אתחול כפול (double-initialization) בעת טעינת ה-plugin
- שיפור: הוספנו את ה-npm script check-env, כך שמסביבות מפתחים יוגדרו אוטומטית בפעם הראשונה
