# יומן שינויים של Mailster

### 1.0.0
* השקה ראשונית
* סנכרון אוטומטי של מנויים בהרשמה או בתשלום
* הקצאת רשימות ספציפיות למוצר
* אפשרות Opt-in המבוססת על תיבת סימון לצורך עמידה בתקנות ה-GDPR
* תמיכה ב-Double opt-in
* מיפוי שדות מקיף
* לוגים מפורטים
