# Material WP {#material-wp}

ממשק המנהל של WordPress נראה אותו דבר כבר שנים. אבל תנחשתם אי פעם איך ייראה ממשק מנהל בגרסת Material Design?

Material WP הוא תבנית מנהל (admin theme) שפותחה כדי לתת לכם את המראה והתחושה הזו, ועוזרת לכם לספק ללקוחות שלכם Dashboard מותאם אישית יותר.

## Features {#features}

- תמיכה ב-CSS מותאם אישית
- הגדרות עיצוב קלות
- סידור מחדש של התפריט
- ייבוא/ייצוא קל

## Documentation {#documentation}

**התאמה אישית של לוח המחוונים (Admin Panel) שלכם עם Material WP**

## Installation {#installation}

1. העלו את 'material-wp' לתיקייה '/wp-content/plugins/'
2. הפעילו את ה-plugin דרך תפריט 'Plugins' ב-WordPress

## Usage Guide {#usage-guide}

ממשק המנהל של WordPress נראה אותו דבר כבר שנים. אבל תנחשתם אי פעם איך ייראה ממשק מנהל בגרסת Material Design?

Material WP הוא תבנית מנהל שפותחה כדי לתת לכם את המראה והתחושה הזו, ועוזרת לכם לספק ללקוחות שלכם Dashboard מותאם אישית יותר. Material WP הוא גם חלק מליבת ה-plugins הפרימיום של Ultimate Multisite.

לאחר **התקנתו**, עבור לדף _Plugins_ ולכו על הפעלה ברשת (Network) שלכם.
