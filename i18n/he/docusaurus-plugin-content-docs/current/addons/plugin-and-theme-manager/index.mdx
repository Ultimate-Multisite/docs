# Plugin & Theme Manager

Multisite Ultimate: Plugin and Theme Manager

ערוך או הסתר את המידע המטא (כותרת, מחבר, תמונה ממוזערת ותיאור) של תוספים ואתרי תבניות מותקנים. צור קטגוריות לתוספים ואתרי תבניות כדי לאפשר למשתמשים שלך לסנן אותם ולהוסיף דף תוספים מותאם אישית ויפה מאוד לפאנל המשתמשים.

כדי להשתמש בתוסף, אתה יכול פשוט לנווט לדפי ה-Plugins או ה-Themes במנהל הניווט של הרשת (Network Admin) וללחוץ על "Edit Plugin/Theme info".

## התקנה

1. העלה את 'ultimate-multisite-plugin-and-theme-manager' לתיקייה '/wp-content/plugins/'
2. הפעל את התוסף באמצעות תפריט ה-'Plugins' במנהל הניווט של WordPress Network Admin שלך

## מדריך שימוש

Ultimate Multisite: Plugin and Theme Manager הוא תוסף **פרימיום** עוצמתי של Ultimate Multisite שמעניק לך שליטה מלאה באופן שבו משתמשי הרשת שלך רואים ומתקשרים עם אפשרויות התוספים והתבניות ברשת שלך.

תוסף זה משרת שלושה מטרות:

  1. הוא מאפשר לך לשנות לחלוטין את המטא-דאטה של תוספים ואתרי תבניות כאחד. זה כולל שם, תיאור, שם מחבר, ואפילו תמונות ממוזערות!

  2. הוא מאפשר לך לקטלג תוספים ואתרי תבניות, כדי לאפשר למשתמשים שלך למיין אותם בקלות.

  3. הוא נותן לך את האפשרות להחליף את דף התוספים הישן והמשעמם בדף חדש בסגנון תבניות, עם מיון לפי קטגוריות.

לאחר ההורדה וההתקנה, יופיע לשונית חדשה – _Add-on Settings_ – בדף הגדרות Ultimate Multisite.
