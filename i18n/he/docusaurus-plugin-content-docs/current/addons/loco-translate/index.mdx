# Loco Translate {#loco-translate}

תוסף ה-Ultimate Multisite: Loco Translate Integration מביא יכולות תרגום ספציפיות לאתר לרשת ה-WordPress Multisite שלכם. אם אי פעם נדרש לכם שאתרים שונים ברשת יקבלו תרגומים שונים לאותו תוסף או תבנית, התוסף הזה הוא הפתרון המושלם.

**מה התוסף הזה עושה**

התוסף הזה משלב בצורה חלקה את Ultimate Multisite עם תוסף Loco Translate הפופולרי, ומאפשר לכל תת-אתר ברשת לשמור על התרגומים המותאמים אישית שלו. בין אם אתם מפעילים פלטפורמה רב-אזורית, משרתים קהילות שפה שונות, או פשוט זקוקים לגמישות תרגום ברחבי הרשת – התוסף הזה הופך את זה לקל ופשוט.

**מושלם עבור:**

* עסקים רב-אזוריים עם צרכי לוקליזציה שונים
* מוסדות חינוך המשרתים קהילות מגוונות
* פלטפורמות SaaS המציעות תמיכה רב-לשונית
* סוכנויות ווב המנהלות מספר אתרי לקוחות
* כל רשת מולטיסייט הדורשת גמישות תרגום

## התקנה {#installation}

1. ודאו ש-Ultimate Multisite מותקן והרשת מופעלת
2. התקינו והפעילו את תוסף Loco Translate
3. העלו את התוסף הזה לתיקיית ה-plugins שלכם
4. הפעילו את תוסף Loco Translate Integration ברמת הרשת
5. התחילו ליצור תרגומים ספציפיים לאתר דרך Loco Translate בכל תת-אתר

זהו! האינטגרציה עובדת אוטומטית בלי צורך בהגדרות נוספות.

## שאלות נפוצות {#frequently-asked-questions}

## האם אני צריך להגדיר משהו לאחר ההתקנה? {#do-i-need-to-configure-anything-after-installation}

לא! האינטגרציה אוטומטית. ברגע שהתוסף וה-Loco Translate פעילים, תת-אתרים יכולים להתחיל מיד ליצור תרגומים מותאמים אישית משלהם.

## מה קורה אם תת-אתר לא יוצר תרגומים מותאמים אישית? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

האתר ישתמש אוטומטית בתרגומים ברמת הרשת כברירת מחדל. שום דבר לא נשבר, והכל עובד בצורה חלקה.

## האם תת-אתרים יכולים לגשת לתרגומים ברמת הרשת? {#can-subsites-access-network-wide-translations}

מנהלי אתר רגילים רואים ועורכים רק את התרגומים של האתר שלהם. מנהלי מערכת סופר (Super admins) יכולים לגשת גם לתרגומים ברמת הרשת וגם לתרגומים ספציפיים לאתר, תלוי באיזה אתר הם צופים.

## האם זה עובד גם עם תוספים וגם עם תבניות? {#does-this-work-with-both-plugins-and-themes}

כן! תרגומים ספציפיים לאתר עובדים גם עבור תוספי WordPress וגם עבור תבניות.

## האם זה יאט את רשת המולטיסייט שלי? {#will-this-slow-down-my-multisite-network}

לא. התוסף משתמש בטעינת קבצים יעילה שבודקת תרגומים ספציפיים לאתר רק כשצריך. אין השפעה נראית לעין על הביצועים.

## האם אני יכול להעתיק תרגומים ברמת הרשת לאתר ספציפי כנקודת התחלה? {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

כן! בזמן עריכת תרגומים ב-Loco Translate, תוכלו בקלות להתחיל עם תרגומי הרשת ולהתאים אותם לכל אתר.

## האם זה עובד עם WooCommerce ותוספים אחרים? {#does-this-work-with-woocommerce-and-other-plugins}

בהחלט! כל תוסף או תבנית שמשתמש בפונקציות התרגום הסטנדרטיות של WordPress יעבוד עם תרגומים ספציפיים לאתר.
