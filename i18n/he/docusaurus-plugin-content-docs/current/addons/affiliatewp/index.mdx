# AffiliateWP {#affiliatewp}

Ultimate Multisite: אינטגרציה עם AffiliateWP

השתמשו ב-AffiliateWP המאוד עוצמתי כדי להגדיל את בסיס הלקוחות של רשת ה-Ultimo שלכם!

## התקנה {#installation}

1. העלו את 'ultimate-multisite-git ' לתיקייה '/wp-content/plugins/'
2. הפעילו את ה-plugin דרך תפריט 'Plugins' ב-WordPress
3. עקבו אחר המדריך (Wizard) שלב אחר שלב כדי להגדיר את ה-plugin

## מדריך שימוש {#usage-guide}

[AffiliateWP](https://affiliatewp.com/) הפך לפתרון המועדף כשמבנים תוכנית שותפים (affiliate program) עבור העסק המקוון שלכם. Ultimate Multisite: AffiliateWP הוא הגשר שיאפשר לכם לבנות את אותו הדבר עבור הרשת הפרימיום שלכם.

## התחלה {#getting-started}

ראשית, אתם **צריכים לוודא** ש-[**AffiliateWP**](https://affiliatewp.com/) **מופעל**! לא תוכלו להשתמש ב-add-on שלנו אם אין לכם **את ה-plugin שמבצע את האינטגרציה**. אם אתם לא מתכננים לאפשר ללקוחות שלכם להשתמש בו באתרים שלהם גם כן, כדאי ש-AffiliateWP יהיה פעיל רק באתר הראשי שלכם.

בנוסף, תצטרכו לוודא שגם **Ultimate Multisite** וגם **Ultimate Multisite: AffiliateWP** **מופעלים** **בכל רשת ה-network**.

## הוספת Ultimate Multisite כאינטגרציה של AffiliateWP {#adding-ultimate-multisite-as-an-affiliatewp-integration}

באתר הראשי שלכם, היכנסו להגדרות השותפים (Affiliates Settings), ובכרטיסיית ה-Integrations, סמנו את Ultimate Multisite. זה יורה ל-[AffiliateWP](https://affiliatewp.com/) להתחיל לעקוב אחר ביקורים והמרות (conversions) מהתהליך של הרשמה ב-Ultimate Multisite.

<!-- תמונת מסך אינה זמינה: כרטיסיית הגדרות האינטגרציות של AffiliateWP עם תיבת הסימון Ultimate Multisite -->

_סמנו את תיבת Ultimate Multisite בתחתית ושמרו_

זה בערך כל מה שצריך. ביקורים שמגיעים מקישורי הפניה (referral links) יתועדו, וכאשר המשתמש הזה מבצע בהמשך את התשלום הראשון שלו, ההמרה (conversion) תיווצר וניכונו עמלה (commission) לשותף האחראי על ההמרה.

עם זאת, רק התשלום הראשון שמגיע מהמשתמש שנוצר זה עתה יתועד. כדי לאפשר תשלומים חוזרים מהפניות, תצטרכו את ה-[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## הפעלת תשלומים חוזרים מהפניות {#enabling-recurring-referrals}

לאחר התקנת ה-[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/), היכנסו להגדרות השותפים (Affiliates Settings) ובכרטיסיית Recurring Referrals, והפעילו את השימוש בהפניות חוזרות.

<!-- תמונת מסך אינה זמינה: הגדרות Recurring Referrals של AffiliateWP עם מתג Enable Recurring Referrals -->

_פשוט כמו שצריך: הפעילו את Recurring Referrals ואתם מוכנים!_

שוב, זה כל מה שצריך!

עכשיו, בכל פעם שמתקבל תשלום חדש בחשבון שהופנה, השותף האחראי יקבל את החלק שלו אוטומטית!
