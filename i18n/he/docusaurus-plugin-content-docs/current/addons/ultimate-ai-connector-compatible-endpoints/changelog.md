# יומן שינויים: Ultimate AI Connector עבור נקודות קצה תואמות

## גרסה 2.0.0 — שוחררה ב-2026-04-24

* **חדש:** תמיכה במספר ספקי שירות (Multi-provider support) — ניתן להגדיר מספר נקודות קצה של AI ולנתב בקשות עם גיבוי אוטומטי בין הספקים.
* **תיקון:** אינטגרציה של ה-SDK למספר ספקים עם מזהי ספקים נכונים, כתובות רישום (registration URLs) והגדרת היקף (scoping) של מסנני HTTP לכל ספק.
* **תיקון:** כרטיס ספק חדש מתרחב אוטומטית בעת הוספתו; מניעת שמירת מטמון (cache busting) בעדכון ה-plugin.
* **תיקון:** מרחב שמות (namespace) דינמי עבור מחלקת הספק לשימוש ב-eval() והתנהגות הרחבה אוטומטית עבור ספקים שנוספו לאחרונה.
* **תיקון:** משתמש ברכיבים יציבים (Card/CardBody/CardHeader/CardDivider) (שאינם ניסיוניים עוד) לתאימות עם WordPress 6.9 ומעלה.
* **תיקון:** החלפת ידית גרירה (DragHandle) לא מוגדרת באייקון אחיזה (grip icon) ביוניקוד לצורך סידור מחדש של הספקים באמצעות גרירה.
* **תיקון:** הוסרה בקשת ה-HTTP החוסמת שהופעלה בכל טעינת עמוד.
* **שיפור:** תהליכי העבודה (workflows) של GitHub Actions עודכנו ל-Node.js 24.
