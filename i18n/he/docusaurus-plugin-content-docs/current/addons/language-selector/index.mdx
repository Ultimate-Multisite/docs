# בורר שפה {#language-selector}

Ultimate Multisite: בורר שפה בתהליך ההרשמה

הוסף שדה בורר שפה לתהליך ההרשמה.

## התקנה {#installation}

1. העלה את 'ultimate-multisite-language-selector' לתיקייה '/wp-content/plugins/'
2. הפעל את ה-plugin דרך תפריט 'Plugins' במנהל הרשת של WordPress שלך

## מדריך שימוש {#usage-guide}

עם ה-add-on החינמי הזה, תוכל לאפשר למשתמשים שלך לבחור איזו שפה הם רוצים לראות בתהליך ההרשמה וגם בדף ההתחברות.

לאחר [הפעלת הרישיון שלך](../../user-guide/getting-started/installing-ultimate-multisite.md), תוכל להתקין את ה-add-ons של Ultimate Multisite שהרישיון שלך מאפשר לך גישה אליהם.

נווט לדף ההגדרות של Ultimate Multisite, ואז, בתפריט בצד שמאל, לחץ על התווית _Add-ons_.

![Add-ons page](/img/admin/addons-page.png)

_נווט לדף Add-ons כדי להתחיל להתקין את בורר השפה_

שם, תמצא את כל ה-add-ons שאנו מציעים.

ה-add-ons החינמיים זמינים לכל הלקוחות שלנו, ללא קשר למתווה המנוי שהם נרשמו אליו.

לחץ על כפתור _Install it Now!_ מתחת לתמונת ה-add-on. לאחר מכן, הפעל אותו בדף Plugins.

![Add-ons page with install button](/img/admin/addons-page.png)

_לחץ על הכפתור כדי להתקין את ה-add-on_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_הפעל אותו בדף Plugins שלך_

עכשיו, עבור לתפריט ההגדרות של Ultimate Multisite והיכנס ללשונית _Network Settings_. שם, בתחתית הדף, תמצא קטע חדש בשם _Languages_.

![Settings page](/img/config/settings-general.png)

_הופיע קטע חדש לגמרי!_

אתה יכול להשאיר אותו ריק כדי להציג את כל השפות כאפשרויות, או לבחור אילו שפות תרצה להפעיל. השפות הן אלו שהותקנו בהגדרות ה-WordPress שלך.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_רשימת השפות הזמינות בהתקנה זו של WP_

לאחר שמירת השינויים שביצעת, תופיע תפריט בחירה עם השפות שבחרת הן בדף ההתחברות והן בתהליך ההרשמה.

![Login and Registration settings](/img/config/settings-login-registration.png)

_בורר שפה בדף ההתחברות_

![Registration page](/img/config/registration-page.png)

_בורר שפה בתהליך ההרשמה_

## הגדרת השפה באמצעות ה-URL {#setting-the-language-via-the-url}

ניתן גם לשנות את שפת העמוד תוך עקיפת פרמטר ה-locale ב-URL. אם תרצה להפנות לדף הרשמה באיטלקית, לדוגמה, הוסף ?locale=it_IT ל-URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

השפה חייבת להיות מותקנת ברשת ה-WordPress שלך כדי שזה יעבוד.
