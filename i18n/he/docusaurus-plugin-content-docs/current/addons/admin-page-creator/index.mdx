# יוצר דפי ניהול (Admin Page Creator) {#admin-page-creator}

## תוכן {#content}
יוצר דפי ניהול מאפשר לך להוסיף בקלות דפי ניהול ודפי משנה חדשים בתוך לוח המחוונים של WordPress. ניתן להשתמש בו כדי ליצור דף היכרות או דף הדרכה שיכול לשמש כדף נחיתה למשתמשים חדשים. תוכל להשתמש ב-page builders כמו Beaver Builder, Elementor, Brizy ו-Oxygen כדי לעצב את הדפים ללא קושי.

התוסף מציע גם תמיכה ב-HTML, CSS ו-JavaScript מותאמים אישית, וכן שליטה בגישה המבוססת על תוכנית (plan) ותפקיד משתמש (user role)!

## תכונות {#features}
* תמיכה ב-HTML ו-CSS מותאמים אישית
* תמיכה ב-JavaScript
* מצב תוכן PHP
* הרשאות מבוססות תפקיד משתמש
* תמיכה ב-page builders

## תיעוד {#documentation}
[יצירת דפי ניהול ודפי משנה מותאמים אישית באמצעות Ultimate Multisite: Admin Page Creator](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## התקנה {#installation}

1. העלה את **ultimate-multisite-admin-page-creator** לתיקייה **/wp-content/plugins/**
2. הפעל את התוסף דרך תפריט **Plugins** במנהל הרשת של WordPress שלך

## מדריך שימוש {#usage-guide}

בעוד ש-WordPress הופך את הוספת תוכן לחזית האתר (front end) לקלה מאוד, יצירת דפים ודפי משנה בצד הניהול (admin side) דורשת מעט ידע בקוד. Ultimate Multisite: Admin Page Creator שואף להביא את אותה קלות שימוש של חזית האתר לצד הניהול של אתרי הרשת שלך.

## למה להשתמש ב-Admin Page Creator? {#why-use-admin-page-creator}

אולי תרצה להציע דף מותאם אישית בפאנל הניהול של הלקוחות שלך עם הדרכות, אולי תרצה להוסיף דף מותאם אישית עם שאלות נפוצות ווידג'ט יצירת קשר. האפשרויות אינסופיות עם Admin Page Creator, וזה מה שנחקור ב튜토리אל של היום.

## יצירת דף הניהול המותאם האישי הראשון שלך {#creating-your-first-custom-admin-page}

לאחר התקנת והפעלת Ultimate Multisite: Admin Page Creator, עבור לתפריט מנהל הרשת שלך (network admin) ולחץ על Admin Pages.

![רשימת דפי ניהול מותאמים אישית במנהל הרשת](/img/admin/admin-pages-list.png)

_רשימת דפי ניהול מותאמים אישית במנהל הרשת_

תראה רשימה של כל דפי הניהול שיצרת עד כה (מכיוון שזו ההגעה הראשונה שלך, הרשימה תהיה ריקה). עבור ולחץ על **Add new Admin Page**. תופנה לדף עורך דף הניהול.

## המרכיבים של דף ניהול מותאם אישית {#the-elements-of-a-custom-admin-page}

בעורך דף הניהול, תהיה לך מספר אפשרויות שונות כדי להתאים אישית את הדף שלך.

### כותרת הדף (Page Title) {#page-title}

זהו הכותרת שתוצג בחלק העליון של הדף, ולא התווית בתפריט המנהל.

<!-- Screenshot unavailable: Admin Page editor showing Page Title field -->

_הזן כותרת דף_

### בחירת סוג התוכן (Content Type Selector) {#content-type-selector}

Ultimate Multisite: Admin Page Creator נותן לך שתי אפשרויות לגבי תוכן הדף: אתה יכול להשתמש בעורך ה-WYSIWYG המוגדר כברירת מחדל של WordPress או לעבור לעורך HTML.

אם אינך מכיר HTML, היצמד לעורך ה-WYSIWYG המוגדר כברירת מחדל של WordPress. תוכל לערוך את תוכן הדף בדיוק כפי שהיית עושה עם פוסט או דף רגיל.

## אפשרויות תפריט (Menu Options) {#menu-options}

בצד, בתוך ה-meta-box של General Options, תוכל להגדיר את סוג התפריט, התווית ועוד הרבה יותר.

### סוגי תפריט (Menu Types) {#menu-types}

לדף הניהול המותאם אישית החדש שלך יכולים להיות שני סוגי תפריט שונים: תפריט ברמה העליונה ותפריט משנה.

דף תפריט ברמה העליונה, כפי שמציע השם, יוסיף את הדף שיצרת לאזור התפריטים של המנהל, לצד דפי WordPress המוגדרים כברירת מחדל.

אם תבחר באפשרות ברמה העליונה, תוכל לבחור גם את סדר התפריט ואייקון תפריט.

אם ברצונך להפוך את הדף הזה לדף תפריט משנה, תוכל לבחור דף הורה אליו תצרף את הדף הזה. תוכל להוסיף את הדף המותאם אישית שלך לדפי WordPress המוגדרים כברירת מחדל או לדפי הניהול המותאמים אישית ברמה העליונה שלך.

## אפשרויות מתקדמות (Advanced Options) {#advanced-options}

אתה אמור להיות מסוגל להוסיף כל מיני תוכן לדפי הניהול המותאמים אישית שלך. זה כולל כללי CSS וקבצים, JavaScript, וספריות חיצוניות. אתה יכול לעשות זאת באמצעות לשונית Advanced Options.

### CSS מותאם אישית (Custom CSS) {#custom-css}

בלשונית CSS, תוכל להוסיף קוד CSS מותאם אישית משלך, כמו גם לייבא קבצי CSS מאתרים חיצוניים (כמו CDNs, Google Fonts וכו').

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_תמיכה גם ב-CSS מותאם אישית. ניתן לכלול גם קבצי סגנון חיצוניים!_

### JavaScript מותאם אישית (Custom JavaScript) {#custom-javascript}

זה נכון גם עבור JavaScript. תוכל להוסיף ספריות חיצוניות לשימוש בקוד JavaScript המותאם אישית שלך.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_הוסף את קוד ה-JavaScript שלך!_

## הרשאות (Permissions) {#permissions}

גם ב-Advanced Options, תוכל לבחור אילו אתרי משנה יקבלו גישה לדף הזה. זה שימושי אם אתה מתכנן להציג תוכן בלעדי שונה לרמות תוכניות שונות, לדוגמה.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_יש לך שליטה מוחלטת היכן דף זה יופיע_

**חשוב**: שני התנאים חייבים להתקיים כדי שהדף יוצג. כלומר, אם המשתמש הוא לקוח של תוכנית נתונה, אך אין לו אחד מהתפקידים שנבחרו (או להפך), הדף לא יופיע בפאנל שלו.

## פעיל (Active) {#active}

אם ברצונך להשבית דף ניהול מותאם אישית מבלי למחוק אותו או לגעת בהרשאות, פשוט בטל את הסימון באפשרות Active.

ביטול הפעילות של דף ברמה העליונה יבטל אוטומטית את כל דפי המשנה המצורפים לדף ההורה הזה.
