# יומן שינויים של רשתות מרובות

### 1.3.0
* הוספה: צופה מקדימה לתבניות רשתות — עכשיו תוכלו לעיין ולצפות בתבניות רשתות בפאנל חי לפני הרכישה.
* הוספה: בחירת תבנית רשתות בקופה — לקוחות יכולים לבחור תבנית רשתות במהלך תהליך ההרשמה בקופה.
* הוספה: כניסה SSO חוצת רשתות באמצעות קישור קסם — לקוחות יכולים לגשת לתת-רשתות בצורה חלקה באמצעות אימות קישור קסם.
* הוספה: שכפול מדיה ברשתות — מדיה של האתר משוכפלת כראוי בעת שכפול רשתות מתבנית.
* תיקון: שכפול רשתות עובד כעת גם ללא תוסף הריבונות המרובת (multi-tenancy); הרשאות מנהל המערכת העל (super admin) מוענקות מחדש לאחר השכפול.
* תיקון: יצירת רשתות עובדת כעת גם עם WooCommerce ועם כל שער התשלום.
* תיקון: מספר הפורט כלול כעת ביצירת דומיין ברירת המחדל.
* תיקון: ה-CSS של בחירת התבנית הועבר לקובץ סגנון המוכנס בתור תור (enqueued stylesheet) לצורך רינדור נכון בכל המערכות.
* תיקון: מגן מפני מפתח `network_id` לא מוגדר בהקש ה-site query.
* תיקון: בדיקת הרשאות רשתות לפני מעבר הקשר מונעת שגיאות תכונות `wpdb` של טבלאות.
