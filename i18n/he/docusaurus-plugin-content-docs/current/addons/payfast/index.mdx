# PayFast

Multisite Ultimate: שער התשלומים של PayFast

מוסיף את Payfast – שער תשלומים מקוון מהיר ומאובטח בדרום אפריקה – כאפשרות תשלום ב-Multisite Ultimate.

## התקנה

1. העלה את 'ultimate-multisite-payfast' לתיקייה '/wp-content/plugins/'
2. הפעל את ה-plugin דרך תפריט ה-'Plugins' ב-WordPress
3. עקוב אחר המדריך (Wizard) שלב אחר שלב כדי להגדיר את ה-plugin

## מדריך שימוש

PayFast הוא מעבד תשלומים מקוון מוביל עבור עסקים בדרום אפריקה. הוא מקדיש את עצמו לסייע לסוחרים מקוונים לצמוח על ידי הפיכת העסקאות לחלקות ומאובטחות. בהתאם לבקשת המשתמשים של Ultimate Multisite, החלטנו ליצור תוסף שיאפשר אינטגרציה עם PayFast. מאמר זה יראה כיצד להגדיר את PayFast כשער התשלומים שלכם כדי לקבל תשלומים ברשת Ultimate Multisite שלכם.

הצעד הראשון הוא להתקין את התוסף.

# כיצד להתקין את תוסף Payfast:

עבור לדף התוספים (add-ons) ובחר את תוסף Ultimate Multisite: Payfast Gateway.

![Add-ons page](/img/admin/addons-page.png)

חלון התוסף יופיע. לחץ על Install Now.

![Add-ons page with install option](/img/admin/addons-page.png)

לאחר ההתקנה, תופנו לדף ה-plugins. כאן, פשוט לחץ על Network Activate והתוסף _Payfast_ יופעל ברשת שלכם.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# הגדרת PayFast

לאחר התקנת התוסף, תוכלו כעת לעבור להגדרות של Ultimate Multisite ולהתחיל להגדיר את PayFast.

בדף **Ultimate Multisite settings** תוכלו למצוא את לשונית "Payments" (תשלומים). תחת לשונית זו, תראו את "**Payfast**" יחד עם כפתור מתג (toggle button) להפעלה או כיבוי של שער התשלומים הזה.

![Payment gateway settings](/img/config/settings-payment-gateways.png)עכשיו לחץ על כפתור המתג כדי להפעיל אותו. זה ירחיב את התיבה ויציג שדות שונים שאתם צריכים למלא, כגון **Mechant ID, Merchant Key ו-PassPhrase**. את המידע הזה תוכלו להשיג מהחשבון שלכם ב-**Payfast**

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

הצעד הבא הוא להתחבר לחשבון ה-PayFast שלכם כדי לקבל את המידע שאתם צריכים.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->אתם יכולים פשוט להעתיק ולהדביק את המידע (credential) ולמלא את הפרטים בהגדרות ה-Ultimate Multisite שלכם. לאחר מכן, לחצו על כפתור **"Save settings"** כדי לשמור את השינויים.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

זהו, אינטגרציית התשלומים של PayFast שלכם הושלמה!
