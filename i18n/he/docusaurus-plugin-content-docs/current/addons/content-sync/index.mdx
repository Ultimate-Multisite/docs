# סנכרון תוכן (Content Sync) {#content-sync}

Multisite Ultimate: סנכרון תוכן

התוסף הזה עבור Multisite Ultimate מאפשר למנהלי רשת לסנכרן תוכן מכל אתר לאתרים אחרים ברשת. הוא תומך בכל סוגי הפוסטים, כולל פוסטים, עמודים, סוגי פוסטים מותאמים אישית ותבניות Elementor. התוסף כולל ממשק מודרך רב-שלבי לבחירת התוכן והאתרים היעד, עם עיבוד אצווה ומשוב התקדמות בזמן אמת.

## תכונות {#features}

* **בחירת תוכן:** רשת כרטיסים ויזואלית עם תמונות ממוזערות וסינון לפי סוג פוסט.
* **כל סוגי הפוסטים:** סנכרון פוסטים, עמודים, מוצרים, תבניות Elementor וכל סוג פוסט מותאם אישית.
* **בחירת אתרים:** בחירה אילו אתרים לקשר אליהם את התוכן, עם אפשרות להוציא את אתר המקור ולבצע בחירה מראש של אתרי ילדים.
* **עיבוד אצווה:** מסנכרן עד 10 אתרים בכל פעם עם עדכוני התקדמות.
* **סנכרון מדיה:** מעתיק וממפה אוטומטית קבצי מדיה (תמונות, וידאו) מתוכן הפוסט לאתרי היעד.
* **סנכרון טקסונומיה:** שומר על קטגוריות, תגיות וטקסונומיות מותאמות אישית.
* **תמיכת Elementor:** תמיכה מלאה בתבניות Elementor עם יצירת CSS מחדש ומאפוס מפתחות מזהה תבנית.
* **פתרון קונפליקטים:** אפשרות להחליף תוכן קיים או לדלג עליו.
* **ניהול מטמון (Cache):** מנקה אוטומטית את המטמון של Elementor באתרים היעד לאחר הסנכרון.

## דרישות {#requirements}

* WordPress 5.3 ומעלה
* PHP 7.4 ומעלה
* תוסף Multisite Ultimate (פעיל)
* תוסף Elementor (אופציונלי, נדרש לסנכרון תבניות Elementor)

## התקנה {#installation}

1. העלה את 'ultimate-multisite-content-sync' לתיקייה '/wp-content/plugins/'
2. הפעל את התוסף דרך תפריט 'Plugins' במנהל האתר של רשת ה-WordPress שלך.
3. הגדר את ההגדרות בנתיב Ultimate Multisite → Settings → Content Sync.
4. עבור אל Ultimate Multisite → Content Sync כדי להתחיל לסנכרן.
