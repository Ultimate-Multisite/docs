# Gerenciando Links Compartilháveis para Planos {#managing-shareable-links-for-plans}

O Ultimate Multisite oferece a você a opção de compartilhar links de planos diretamente com seus clientes. Para fazer isso, vá para qualquer uma das páginas de edição dos seus planos e clique no botão _Clique para copiar o Link Compartilhável_.

![Página de edição do produto com botão de link compartilhável](/img/config/product-edit-full.png)

_Na Página de Edição do Plano, você pode encontrar o botão para criar links compartilháveis._

Observe que este mecanismo usa a frequência de faturamento padrão para construir o link. Isso significa que se seu faturamento padrão for mensal, por exemplo, no final, você terá um link como este: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Se desejar que o link compartilhável seja padrão trimestral, você terá que adicionar 3 no final:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Para selecionar o plano _premium_ em um cronograma anual, você adicionaria 1 seguido da palavra:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Lembre-se de que você não pode ter apenas um link para diferentes frequências de faturamento. Você deve compartilhar um link específico para cada uma das suas frequências de faturamento.

Depois de compartilhar o link com seus clientes, ele os redirecionará para a primeira etapa do seu fluxo de inscrição.
