# Hooks e filtros do Ultimate Multisite {#ultimate-multisite-hooks-and-filters}

O Ultimate Multisite é composto por um monte de funções e consultas ao banco de dados. Ele possui hooks (ganchos) e filtros que permitem modificar ou adicionar novos códigos às funções existentes.

Criamos uma lista desses hooks e filtros como referência para ajudar usuários avançados a personalizar ainda mais sua rede. Você pode acessar a lista no Github clicando no link abaixo.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
