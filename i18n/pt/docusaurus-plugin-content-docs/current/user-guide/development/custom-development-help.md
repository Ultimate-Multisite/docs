# Como Obter Ajuda de Desenvolvimento Personalizada do Codeable? {#how-to-get-custom-development-help-from-codeable}

Às vezes, realmente precisamos de uma personalização incrível para fazer nossos fluxos de rede exatamente como queremos. Se é isso que você está procurando, recomendamos o [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), um lugar onde você pode encontrar desenvolvedores experientes com expertise em Ultimate Multisite.

O [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) é uma plataforma que conecta clientes a desenvolvedores freelancers qualificados do WordPress, capazes de trabalhar em projetos de qualquer tamanho. Eles têm um processo de candidatura meticuloso para garantir que você seja pareado com especialistas especificamente qualificados para a tarefa que tem em mente.

### Como criar um projeto no Codeable? {#how-to-create-a-project-in-codeable}

Primeiro, acesse o [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) e clique no botão **Start a Project** (Iniciar um Projeto).

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Agora, para enviar um projeto, clique à esquerda em **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

Em "I need help with" (Preciso de ajuda com), escolha **Customizing** (Personalização):

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

Em "My" (Meu), escolha **Plugin**:

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Depois clique no botão **Continue** (Continuar).

<!-- Screenshot unavailable: Codeable project form Continue button -->

Agora, é hora de descrever seu projeto e suas necessidades da melhor maneira possível. Tente fornecer o máximo de informações que puder!

<!-- Screenshot unavailable: Codeable project description form -->

Responda às seguintes perguntas para ajudar com a precificação.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Depois disso, se você ainda não tem uma conta [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), crie uma e clique no botão **Publish My Project** (Publicar Meu Projeto).

<!-- Screenshot unavailable: criação de conta Codeable e botão Publish My Project -->

Assim que você publicar seu projeto no [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), a plataforma fará a conexão com desenvolvedores que combinam com o conjunto de habilidades necessário para o seu projeto.

Este processo vai ajudar a encontrar a pessoa certa para o seu trabalho, que tenha experiência com WordPress e Ultimate Multisite!

A taxa mínima do Codeable é de $70–$120 USD por hora. As estimativas mudarão de acordo com o escopo do trabalho, a complexidade e a urgência do projeto.

Assim que você receber sua estimativa, poderá contratar o desenvolvedor que melhor se adapta às suas necessidades e começar seu projeto juntos!
