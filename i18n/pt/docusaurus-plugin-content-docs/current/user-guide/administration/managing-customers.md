# Gerenciando Clientes

A página **Clientes** mostra a lista dos membros registrados na sua rede. Esta página exibe informações básicas do usuário, incluindo nome, endereço de e-mail, último login e um link **para mudar para** que permite fazer login no subsistema do usuário como administrador do site.

![Customers list page](/img/admin/customers-list.png)

Clique em **Gerenciar** abaixo do cartão de um cliente para ver informações detalhadas sobre esse cliente.

Na página de detalhes do cliente, o bloco **Visão Geral** mostra o ID do cliente, a última hora de login e o valor total gasto por este cliente.

O bloco **Opções do Cliente** permite definir este cliente como **VIP** (uma tag para torná-lo mais fácil de encontrar). A aba **Informações de Faturamento** exibe informações de faturamento, como e-mail, endereço e país.

O bloco **Campos** mostra o nome de usuário do cliente, e-mail e permite alterar o status de **Verificação de E-mail**.

Abaixo do bloco Opções do Cliente você encontrará quatro blocos adicionais:

- **Assinaturas (Memberships)** — Assinaturas atuais às quais o cliente está inscrito
- **Pagamentos (Payments)** — Histórico de pagamentos do cliente
- **Sites** — Sites que este cliente possui
- **Eventos (Events)** — Eventos relacionados a este cliente (verificação de e-mail, alterações de faturamento, etc.)

Clique no ícone de seta (→) ao lado de qualquer item para navegar até sua página de detalhes. Por exemplo, clicar na seta ao lado de um site o levará para a página de edição do site, onde você poderá modificar a associação da assinatura, o tipo de site e o status ativo.

O mesmo se aplica às assinaturas e pagamentos — cada um linka para páginas de detalhes onde você pode editar o status da assinatura, o cliente associado, os produtos, as configurações do gateway de pagamento e até processar reembolsos.

No lado direito da página do cliente, você encontrará:

- **Último Login e IPs** — Hora do último login, endereço IP e país
- **Carimbos de Tempo (Timestamps)** — Quando a conta do cliente foi criada e modificada pela última vez
- **Excluir** — Opção para excluir permanentemente o cliente

:::warning
Excluir um cliente é permanente e não pode ser desfeito.
:::
