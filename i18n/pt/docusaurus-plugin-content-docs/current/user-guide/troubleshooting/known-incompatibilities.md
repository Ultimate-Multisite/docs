# Incompatibilidades conhecidas com outros Plugins {#known-incompatibilities-with-other-plugins}

Após instalar e ativar o Ultimate Multisite, você pode encontrar problemas devido a incompatibilidades não resolvidas com outros plugins.

Estamos trabalhando ativamente para reduzir esta lista a cada nova versão, mas se um plugin estiver listado aqui, talvez precise desativá-lo para ter uma Rede Ultimate Multisite totalmente funcional.

Aqui está uma lista deles e o que você deve fazer nesses casos:

**Alguns plugins WPMU DEV** Alguns dos plugins _não são mais mantidos_ pela WPMU DEV são conhecidos por causar a interrupção do fluxo de registro do Ultimate Multisite na última etapa (o usuário não consegue criar uma conta). Aqui está uma lista dos plugins que causam esses problemas:

  * Signup Language: _O Ultimate Multisite tem um add-on que oferece a mesma funcionalidade, chamado Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Isso já vem embutido no Ultimate Multisite, então não há necessidade de um plugin extra._

  * New Blog Templates: _Isso já vem embutido no Ultimate Multisite, então não há necessidade de um plugin extra._

  * Set Password on Multisite Blog Creation: _Isso já vem embutido no Ultimate Multisite, então não há necessidade de um plugin extra._

  * Signup TOS: _Isso já vem embutido no Ultimate Multisite, então não há necessidade de um plugin extra._

  * Anti-Splog: Embora menos poderoso, também é _embutido no Ultimate Multisite._

**Nota**: Todos os outros plugins da WPMU Dev podem ser usados normalmente junto com o Ultimate Multisite. Exemplos incluem _Smush_, _Forminator_, _Defender_, etc.
