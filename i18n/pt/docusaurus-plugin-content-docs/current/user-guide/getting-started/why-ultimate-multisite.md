# Por que o Ultimate Multisite {#why-ultimate-multisite}

Se você chegou a este artigo, provavelmente ouviu dizer que nosso plugin pode ser a solução para realizar sua ideia de negócio. Esperamos que você tenha as respostas de que precisa ao final desta matéria.

## O que é o Ultimate Multisite? {#what-is-ultimate-multisite}

O Ultimate Multisite é um plugin WordPress Multisite que permite oferecer aos seus clientes um WaaS (Website as a Service, ou Website como Serviço). Com nosso plugin, você pode ter vários níveis de assinatura e fazer com que os clientes paguem uma taxa recorrente para ter um site hospedado na sua rede Multisite.

O Ultimate Multisite permite que você receba seus pagamentos com diferentes gateways (como WooCommerce ou Stripe), crie modelos de sites para facilitar muito a sua vida quando um novo usuário se registra na sua rede, mapeie domínios e personalize seus formulários de registro para criar uma experiência personalizada para seus clientes.

E isso é apenas o começo, já que você pode estender suas funcionalidades com add-ons e webhooks.

## Outras funcionalidades {#other-features}

O Ultimate Multisite também oferece:

  * Integração de Mapeamento de Domínio com provedores de hospedagem WordPress gerenciados: O Ultimate Multisite adiciona automaticamente domínios mapeados aos painéis de administração do WP Engine, Cloudways, cPanel, RunCloud, Closte e outros quando um novo domínio é mapeado, para que você não precise fazer isso manualmente. Em alguns casos, nós até acionamos a busca por certificado SSL para você.

  * Webhooks: O Ultimate Multisite possui um sistema de Webhook poderoso que permite integrar sua rede com vários serviços diferentes. Nós até oferecemos integração direta com o Zapier, que abre sua rede para mais de 1000 aplicativos e serviços disponíveis na plataforma deles.

Páginas de Administração Poderosas para ajudar seus clientes usando seu construtor de páginas favorito: o Ultimate Multisite tem um add-on útil chamado Admin Pages Creator, que basicamente permite adicionar Páginas de Administração totalmente personalizadas ao painel dos seus clientes usando templates do Elementor, Beaver Builder e Brizy.

* Dois temas de Administração diferentes para personalizar a experiência do painel dos seus clientes: Check Material WP e PRO.

E muito mais avançado:

* Suporte a Sobrescrita de Templates (Template Overrides): Com o Ultimate Multisite, você pode substituir os templates HTML padrão para o fluxo de inscrição pelas suas próprias versões sem se preocupar em alterar os arquivos principais do plugin.

* API de Busca e Substituição (Search and Replace API): Você também pode anotar os templates do site com placeholders como `{{site_name}}` e usar a Search and Replace API para preencher automaticamente esses placeholders com as informações fornecidas pelo seu cliente logo após o cadastro.

## Casos de Uso {#use-cases}

Agora é hora de ver alguns dos possíveis casos de uso do Ultimate Multisite.

### Caso 1: Uma Agência {#case-1-an-agency}

Geralmente, as habilidades centrais de uma agência residem no design de sites com aspectos como hospedagem ou marketing listados como serviços adicionais.

Para agências, o Ultimate Multisite apresenta uma proposta de valor incrível em sua capacidade de hospedar e gerenciar múltiplos websites em uma única plataforma. Ainda mais, agências que padronizam seus designs em temas específicos, como GeneratePress, Astra, OceanWP ou outros, podem aproveitar as capacidades do Ultimate Multisite para ativar automaticamente esses temas para cada novo site.

O controle rigoroso sobre plugins e temas é mantido por produto através das configurações do Ultimate Multisite, permitindo que os plugins e temas sejam disponibilizados ou ocultados, bem como o estado de ativação quando instanciados para um novo site.

Agências terão tranquilidade com o Ultimate Multisite, permitindo que façam o que fazem de melhor: projetar websites excepcionais.

### Caso 2: Provedor de Nicho {#case-2-niche-provider}

Existe um ditado antigo que diz: "Faça uma coisa e faça bem". Para muitos especialistas, isso significa criar um produto ou serviço em torno de uma ideia central única.

Se você está focado em desenvolver uma rede de sites de nicho, provavelmente gostaria de oferecer serviços baseados em uma estrutura e plataforma comuns. É aí que os templates de site podem ajudar!

Uma das funcionalidades inovadoras do Ultimate Multisite é o template de site: é um site onde o tema já foi instalado e ativado, os plugins necessários foram instalados e ativados, e posts ou páginas de exemplo foram criados. Quando um cliente cria um novo site baseado no template, o conteúdo e as configurações do template são copiados para o site recém-criado.

Para um provedor de sites e serviços de nicho, isso oferece uma vantagem incomparável na capacidade de criar instantaneamente um site pronto para uso com plugins e design personalizados. O cliente só precisa fornecer uma entrada mínima para concluir o serviço.

### Caso 3: Hospedagem WordPress {#case-3-wordpress-web-hosting}

Existem inúmeras maneiras de hospedar sites WordPress, mas raramente é tão simples quanto fornecer espaço web a um cliente com uma versão pré-instalada do WordPress. Isso ocorre porque várias decisões e considerações precisam se juntar para oferecer um serviço significativo.

O Ultimate Multisite se destaca nesta área ao fornecer uma solução completa "turnkey" (pronta para uso) para a hospedagem de sites WordPress. A solução inclui os mecanismos básicos para fornecer serviços de assinatura, coleta de pagamentos, formulários de checkout, cupons de desconto e comunicações com o cliente.

Grande parte do trabalho essencial necessário para instalar, configurar e manter corretamente um WordPress Multisite é facilitada pelo Ultimate Multisite, pois os administradores de rede só precisam se preocupar com aspectos relacionados ao seu serviço ou nicho, como níveis de produto, preços e ofertas de serviço.

Para desenvolvedores que desejam integrar com o Ultimate Multisite, a solução também oferece uma API REST completa e Webhooks para notificação de eventos.

Sem depender de uma miríade de plugins e licenças externas, o Ultimate Multisite fornece uma solução rica em recursos e comparável à do Wix, Squarespace, [WordPress.com](https://WordPress.com) e outros.
