# Personalizando seu Painel de Administração com o PRO Theme {#customizing-your-admin-panel-with-pro-theme}

Muitas vezes, ao fazer trabalhos para clientes, precisamos oferecer uma experiência personalizada no painel administrativo do WordPress deles. Se você está procurando um visual e sensação mais profissionais, o PRO Theme pode ajudar.

O PRO Theme é um tema de administração desenvolvido para te auxiliar na personalização do Dashboard dos seus clientes. E ele também faz parte do core dos plugins premium do Ultimate Multisite.

Após instalar, vá para a sua página de _Plugins_ e clique em ativar no seu Network (Rede).

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Ative-o em toda a rede na sua página de Plugins_

_**NOTA:**_ _Se você apenas instalá, ele aparecerá para o seu cliente como uma opção que pode ser marcada para adicionar o tema apenas àquele usuário._

Agora, você tem sua interface de administração completamente redesenhada! Você também terá um novo menu na coluna da esquerda chamado PRO Theme.

<!-- Screenshot unavailable: PRO Theme redesigned admin panel interface -->

_Um visual totalmente novo para o seu Painel de Administração_

Neste menu, você terá acesso a cinco abas:

<!-- Screenshot unavailable: PRO Theme Styling Settings tab -->

_Configurações de Estilo (Styling Settings)_

<!-- Screenshot unavailable: PRO Theme Functionality tab -->

_Funcionalidades (Functionality)_

<!-- Screenshot unavailable: PRO Theme Custom CSS tab -->

_CSS Personalizado (Custom CSS)_

<!-- Screenshot unavailable: PRO Theme Import/Export tab -->

_Importar/Exportar (Import/Export)_

<!-- Screenshot unavailable: PRO Theme Activate tab -->

_Ativar o PRO Theme (Activate PRO Theme)_

Em cada uma delas, você pode ativar e desativar itens de estilo, permitir ou bloquear funcionalidades, e oferecer aos seus clientes a personalização que eles podem estar procurando na sua rede de sites.
