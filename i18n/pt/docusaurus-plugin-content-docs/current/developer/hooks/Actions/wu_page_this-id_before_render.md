# Action: wu_page_\{$this->id\}_before_render {#action-wupagethis-idbeforerender}

Permite que programadores de plugins adicionem conteúdo adicional antes de imprimirmos a página.

## Parâmetros {#parameters}

| Nome | Tipo | Descrição |
|------|------|-------------|
| $page_id | `string` | O ID desta página. |
| $page | `object` | O objeto da página. |
