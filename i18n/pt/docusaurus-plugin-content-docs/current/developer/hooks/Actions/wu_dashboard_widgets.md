# Action: wu_dashboard_widgets

Permite que desenvolvedores de plugins adicionem widgets ao painel do Dashboard de rede.

## Parâmetros {#parameters}

| Nome | Tipo | Descrição |
|------|------|-------------|
| $tab | `string` | A aba atual. |
| $screen | `\WP_Screen` | O objeto de tela. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Instância da página de administração do Ultimate Multisite. |

- 2.0.0
