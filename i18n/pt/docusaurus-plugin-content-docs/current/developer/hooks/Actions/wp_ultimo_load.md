# Ação: wp_ultimo_load

Aciona quando todas as dependências foram carregadas

Permite que programadores de plugins adicionem novas funcionalidades. Por exemplo, suporte para novos fornecedores de alojamento, etc.

### Desde {#since}

- 2.0.0
