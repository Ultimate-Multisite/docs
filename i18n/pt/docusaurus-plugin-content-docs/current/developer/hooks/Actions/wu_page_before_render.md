# Ação: wu_page_before_render

Permite que programadores de plugin adicionem conteúdo adicional antes de imprimirmos a página.

## Parâmetros {#parameters}

| Nome | Tipo | Descrição |
|------|------|-------------|
| $page_id | `string` | O id desta página. |
| $page | `object` | O objeto da página. |

- 1.8.2
