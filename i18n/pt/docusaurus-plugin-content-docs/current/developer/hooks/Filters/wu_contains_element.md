# Filtro: wu_contains_element {#filter-wucontainselement}

Permite que os programadores alterem os resultados da pesquisa inicial.

Isto é útil para builders de terceiros e afins.

## Parâmetros {#parameters}

| Nome | Tipo | Descrição |
|------|------|-------------|
| $contains_elements | `bool` | Se o elemento está contido no conteúdo. |
| $content | `string` | O conteúdo a ser examinado. |
| $element | `self` | O elemento atual. |
| $post | `null\|\WP_Post` | post a verificar. |

### Desde {#since}

- 2.0.0
