# Filtro: wu_view_override_replaceable_views {#filter-wuviewoverridereplaceableviews}

Permite que programadores adicionem pastas adicionais à lista substituível.

Tenha cuidado, pois permitir pastas adicionais pode fazer com que cópias desatualizadas sejam carregadas em vez das versões do Ultimate Multisite.

## Parâmetros {#parameters}

| Nome | Tipo | Descrição |
|------|------|-------------|
| $replaceable_views | `array` | Lista de pastas permitidas. |

### Desde {#since}

- 2.0.0
