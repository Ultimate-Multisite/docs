# Filtro: wu_plan_product_types {#filter-wuplanproducttypes}

Filtra os tipos de produto que devem ser tratados como planos.

Este filtro permite que addons registem tipos de produto adicionais que devem ser reconhecidos como planos na validação e segregação.

## Parâmetros {#parameters}

| Nome | Tipo | Descrição |
|------|------|-------------|
| $plan_types | `array` | Array de tipos de produto a tratar como planos. |

### Desde {#since}

- 2.3.0
