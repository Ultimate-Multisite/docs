# Filtro: secure_auth_cookie {#filter-secureauthcookie}

Filtra se o cookie de autenticação deve ser enviado apenas por HTTPS.

## Parâmetros {#parameters}

| Nome | Tipo | Descrição |
|------|------|-------------|
| $secure | `bool` | Se o cookie deve ser enviado apenas por HTTPS. |
| $user_id | `int` | ID do utilizador. |
