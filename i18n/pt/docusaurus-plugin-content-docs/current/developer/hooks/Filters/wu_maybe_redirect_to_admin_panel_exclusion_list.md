# Filter: wu_maybe_redirect_to_admin_panel_exclusion_list {#filter-wumayberedirecttoadminpanelexclusionlist}

Cria uma lista de exclusão de parâmetros que impedem o redirecionamento automático.

Isto é necessário porque, caso contrário, o construtor de páginas não conseguirá editar a página de login depois de ela estar definida.
