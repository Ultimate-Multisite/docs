# Filtro: wu_add_new_domain_modal_fields

Filtra os campos do modal de adicionar novo domínio.

Permite que addons (por exemplo, Domain Seller) modifiquem ou substituam os campos de registo de domínio.

## Parâmetros {#parameters}

| Nome | Tipo | Descrição |
|------|------|-------------|
| $fields | `array` | Os campos do formulário. |
