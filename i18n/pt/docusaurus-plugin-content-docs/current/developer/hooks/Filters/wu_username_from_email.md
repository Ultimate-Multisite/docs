# Filtro: wu_username_from_email

Filtra o nome de utilizador de novo cliente.

## Parâmetros

| Nome | Tipo | Descrição |
|------|------|-------------|
| $username | `string` | Nome de utilizador do cliente. |
| $email | `string` | Endereço de email do novo cliente. |
| $new_user_args | `array` | Array de argumentos do novo utilizador, talvez incluindo nomes próprios e apelidos. |
| $suffix | `string` | Acrescenta uma string ao nome de utilizador para o tornar único. |
