# Filtro: wu_password_require_uppercase

Filtra se deve exigir letras maiúsculas nas palavras-passe.

## Parâmetros {#parameters}

| Nome | Tipo | Descrição |
|------|------|-------------|
| $require | `bool` | Se deve exigir maiúsculas. Predefinição true quando as regras são aplicadas. |
| $defender_active | `bool` | Se Defender Pro Strong Password está ativo. |
