# Filtro: wu_independent_billing_cycle_product_types

Filtra os tipos de produto que têm ciclos de faturação independentes.

Produtos com ciclos de faturação independentes não são forçados a corresponder ao período de faturação do plano no carrinho. Isto é útil para produtos como registos de domínio que são sempre faturados anualmente, independentemente de o plano ser mensal ou anual.

## Parâmetros {#parameters}

| Nome | Tipo | Descrição |
|------|------|-------------|
| $types | `array` | Array de tipos de produto com ciclos de faturação independentes. |
