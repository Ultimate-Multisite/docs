# Ação: wp_ultimo_ptm_edit_page_after_title_input

Permite que os programadores de plugin adicionem informações adicionais abaixo da entrada de texto

## Parâmetros {#parameters}

| Nome | Tipo | Descrição |
|------|------|-------------|
| $ | `object` | Objeto que contém as informações |
| $ | `\WU_Page` | Instância de página Ultimate Multisite |
