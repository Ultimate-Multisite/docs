# Fluent Forms

Este complemento permite adicionar limites ao Fluent Forms no Multisite Ultimate em produtos, adesões ou sites.
O Fluent Forms usa tabelas personalizadas em vez de tipos de conteúdo personalizados, por isso as opções normais de limitação não funcionam com o Fluent Forms.

## Funcionalidades

- Adicione limites para Fluent Forms ou Conversational Fluent Forms
- Os clientes recebem uma mensagem amigável a incentivá-los a atualizar o seu plano quando atingem o limite
