# Seletor de Idioma

Ultimate Multisite: Seletor de Idioma no Registo

Adicione um campo de seletor de idioma ao fluxo de registo.

## Instalação

1. Carregue 'ultimate-multisite-language-selector' para o diretório '/wp-content/plugins/'
2. Ative o plugin através do menu 'Plugins' na Administração de Rede do WordPress

## Guia de Utilização

Com este add-on gratuito, pode permitir que os seus utilizadores escolham que idioma gostariam de ver no fluxo de registo e na página de início de sessão.

Depois de [ativar a sua licença](../../user-guide/getting-started/installing-ultimate-multisite.md), poderá instalar os add-ons Ultimate Multisite aos quais a sua licença lhe dá acesso.

Vá à página de definições do Ultimate Multisite e, depois, no menu à esquerda, clique na etiqueta _Add-ons_.

![Página de add-ons](/img/admin/addons-page.png)

_Vá à página de Add-ons para começar a instalar o Seletor de Idioma_

Aí, encontrará todos os add-ons que oferecemos.

Os add-ons gratuitos estão disponíveis para todos os nossos clientes, independentemente do plano de subscrição que escolheram.

Clique no botão _Install it Now!_ abaixo da miniatura do add-on. Depois, ative-o na página de Plugins.

![Página de add-ons com botão de instalação](/img/admin/addons-page.png)

_Clique no botão para instalar o add-on_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_Ative-o na sua página de Plugins_

Agora, vá ao menu Definições do Ultimate Multisite e entre no separador _Network Settings_. Aí, no fundo da página, encontrará uma nova secção chamada _Languages_.

![Página de definições](/img/config/settings-general.png)

_Aparece uma secção totalmente nova!_

Pode deixá-la em branco para apresentar todos os idiomas como opções ou selecionar quais gostaria de disponibilizar. Os idiomas são os que tem instalados na sua configuração do WordPress.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_Lista de idiomas disponíveis nesta instalação do WP_

Depois de guardar as alterações que fez, será apresentado um menu de seleção com os idiomas que escolheu na página de Início de Sessão, bem como no Fluxo de Registo.

![Definições de início de sessão e registo](/img/config/settings-login-registration.png)

_Seletor de Idioma na Página de Início de Sessão_

![Página de registo](/img/config/registration-page.png)

_Seletor de Idioma no Fluxo de Registo_

## Definir o Idioma através do URL

Também pode alterar o idioma de uma página passando o parâmetro locale para o URL. Se quiser redirecionar para uma página de registo em italiano, por exemplo, adicione ?locale=it_IT ao URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

O idioma tem de estar instalado na sua rede WordPress para que isto funcione.
