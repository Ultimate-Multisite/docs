# Agentes de Suporte {#support-agents}

Gerir uma rede inteira sozinho não é uma tarefa fácil. À medida que a sua rede começa a crescer, começará a ver uma procura crescente por suporte. Será uma boa ideia trazer pessoas que o ajudem a aliviar a sua carga de trabalho. No entanto, conceder acesso/permissão total é arriscado e pode comprometer a sua rede. É aqui que entra este complemento de Agentes de Suporte.

Este complemento permite-lhe criar um novo tipo de utilizador WordPress chamado Agente de Suporte e dá-lhe controlo total sobre as permissões que esse utilizador tem. Conceder permissão é tão fácil como ativar um botão de alternância, e desativá-la caso decida removê-la.

## Funcionalidades {#features}

- Lista detalhada de permissões WordPress e Ultimate Multisite
- Ative e desative o acesso com facilidade
- Crie um número ilimitado de contas de Agente

## Instalação {#installation}

1. Carregue 'ultimate-multisite-support-agents' para o diretório '/wp-content/plugins/'
2. Ative o plugin através do menu 'Plugins' no WordPress
3. Siga o assistente passo a passo para configurar o plugin
