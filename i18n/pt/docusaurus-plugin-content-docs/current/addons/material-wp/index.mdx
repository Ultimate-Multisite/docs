# Material WP {#material-wp}

A interface de administração do WordPress tem sido a mesma há anos. Mas já se perguntou como seria uma versão Material Design da interface de administração? 

Material WP é um tema de administração desenvolvido para lhe dar essa aparência e sensação, ajudando-o a oferecer aos seus clientes um Dashboard mais personalizado.

## Funcionalidades {#features}

- Suporte a CSS personalizado
- Definições de estilo fáceis
- Reordenação do menu
- Importação/exportação fácil

## Documentação {#documentation}

**Personalizar o seu painel de administração com Material WP**

## Instalação {#installation}

1. Carregue 'material-wp' para o diretório '/wp-content/plugins/'
2. Ative o plugin através do menu 'Plugins' no WordPress

## Guia de utilização {#usage-guide}

A interface de administração do WordPress tem sido a mesma há anos. Mas já se perguntou como seria uma versão Material Design da interface de administração?

Material WP é um tema de administração desenvolvido para lhe dar essa aparência e sensação, ajudando-o a oferecer aos seus clientes um Dashboard mais personalizado. Material WP também faz parte do núcleo de plugins premium do Ultimate Multisite.

Depois de **o instalar**, vá à sua página de _Plugins_ e clique para o ativar na sua rede.
