# Sincronização de Conteúdo

Multisite Ultimate: Sincronização de Conteúdo

Este complemento para Multisite Ultimate permite que administradores de rede sincronizem conteúdo de qualquer site para outros sites na rede. Suporta todos os tipos de publicação, incluindo publicações, páginas, tipos de publicação personalizados e modelos do Elementor. O complemento inclui uma interface de assistente em várias etapas para selecionar conteúdo e sites de destino, com processamento em lotes e feedback de progresso em tempo real.

## Funcionalidades {#features}

* Seleção de Conteúdo: grelha visual de cartões com miniaturas e filtragem por tipo de publicação
* Todos os Tipos de Publicação: sincronize publicações, páginas, produtos, modelos do Elementor e qualquer tipo de publicação personalizado
* Seleção de Site: selecione para quais sites sincronizar o conteúdo, com exclusão do site de origem e pré-seleção de sites filhos
* Processamento em Lotes: sincroniza para 10 sites de cada vez com atualizações de progresso
* Sincronização de Media: copia e remapeia automaticamente ficheiros de media (imagens, vídeos) do conteúdo da publicação para os sites de destino
* Sincronização de Taxonomia: preserva categorias, etiquetas e taxonomias personalizadas
* Suporte ao Elementor: suporte completo para modelos do Elementor com regeneração de CSS e remapeamento de IDs de modelo
* Resolução de Conflitos: opção para substituir conteúdo existente ou ignorá-lo
* Gestão de Cache: limpa automaticamente a cache do Elementor nos sites de destino após a sincronização

## Requisitos {#requirements}

* WordPress 5.3 ou superior
* PHP 7.4 ou superior
* Plugin Multisite Ultimate (ativo)
* Plugin Elementor (opcional, necessário para sincronização de modelos do Elementor)

## Instalação {#installation}

1. Carregue 'ultimate-multisite-content-sync' para o diretório '/wp-content/plugins/'
2. Ative o plugin através do menu 'Plugins' no seu Administrador de Rede WordPress
3. Configure as definições em Ultimate Multisite → Definições → Sincronização de Conteúdo
4. Navegue para Ultimate Multisite → Sincronização de Conteúdo para começar a sincronizar
