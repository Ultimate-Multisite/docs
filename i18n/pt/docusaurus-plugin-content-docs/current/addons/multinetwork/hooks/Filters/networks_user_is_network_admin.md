# Filtro: networks_user_is_network_admin

Filtra as redes das quais um utilizador é o administrador.

## Parâmetros {#parameters}

| Nome | Tipo | Descrição |
|------|------|-------------|
| $ | `array\|bool` | Lista de IDs de rede ou false se não houver redes para o utilizador. |
| $ | `int` | ID do utilizador para o qual as redes devem ser devolvidas. |
