# Loco Translate {#loco-translate}

O addon Ultimate Multisite: Loco Translate Integration traz capacidades de tradução específicas por site para a sua rede multissite WordPress. Se alguma vez precisou que diferentes sites na sua rede tivessem traduções diferentes do mesmo plugin ou tema, este addon é a solução perfeita.

**O que este addon faz**

Este addon integra perfeitamente o Ultimate Multisite com o popular plugin Loco Translate, permitindo que cada subsite na sua rede mantenha as suas próprias traduções personalizadas. Quer esteja a gerir uma plataforma multirregional, a servir diferentes comunidades linguísticas, ou simplesmente precise de flexibilidade de tradução em toda a sua rede, este addon torna tudo simples.

**Perfeito para:**

* Empresas multirregionais com diferentes necessidades de localização
* Instituições de ensino que servem comunidades diversas
* Plataformas SaaS que oferecem suporte multilingue
* Agências web que gerem vários sites de clientes
* Qualquer rede multissite que exija flexibilidade de tradução

## Instalação {#installation}

1. Certifique-se de que o Ultimate Multisite está instalado e ativado na rede
2. Instale e ative o plugin Loco Translate
3. Carregue este addon para o seu diretório de plugins
4. Ative na rede o addon Loco Translate Integration
5. Comece a criar traduções específicas por site através do Loco Translate em qualquer subsite

É tudo! A integração funciona automaticamente sem qualquer configuração adicional.

## Perguntas frequentes {#frequently-asked-questions}

## Preciso de configurar alguma coisa após a instalação? {#do-i-need-to-configure-anything-after-installation}

Não! A integração é automática. Assim que tanto o addon como o Loco Translate estiverem ativos, os subsites podem começar imediatamente a criar as suas próprias traduções personalizadas.

## O que acontece se um subsite não criar traduções personalizadas? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

O site usará automaticamente as traduções de toda a rede como alternativa. Nada deixa de funcionar, e tudo funciona sem problemas.

## Os subsites podem aceder às traduções de toda a rede? {#can-subsites-access-network-wide-translations}

Administradores de site comuns apenas veem e editam as traduções do seu próprio site. Superadministradores podem aceder tanto às traduções de toda a rede como às específicas por site, dependendo do site que estiverem a visualizar.

## Isto funciona tanto com plugins como com temas? {#does-this-work-with-both-plugins-and-themes}

Sim! As traduções específicas por site funcionam tanto para plugins como para temas WordPress.

## Isto vai tornar a minha rede multissite mais lenta? {#will-this-slow-down-my-multisite-network}

Não. O addon usa carregamento de ficheiros eficiente que só verifica traduções específicas por site quando necessário. Não há impacto percetível no desempenho.

## Posso copiar traduções da rede para um site específico como ponto de partida? {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

Sim! Ao editar traduções no Loco Translate, pode facilmente começar com traduções da rede e personalizá-las para cada site.

## Isto funciona com WooCommerce e outros plugins? {#does-this-work-with-woocommerce-and-other-plugins}

Sem dúvida! Qualquer plugin ou tema que use as funções de tradução padrão do WordPress funcionará com traduções específicas por site.
