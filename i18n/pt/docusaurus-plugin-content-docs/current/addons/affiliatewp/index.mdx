# AffiliateWP

Multisite Ultimate: Integração AffiliateWP

Use o poderoso AffiliateWP para aumentar a base de clientes da sua rede Ultimo!

## Instalação

1. Carregue 'ultimate-multisite-git ' para o diretório '/wp-content/plugins/'
2. Ative o plugin através do menu 'Plugins' no WordPress
3. Siga o assistente passo a passo para configurar o plugin

## Guia de utilização

[AffiliateWP](https://affiliatewp.com/) tornou-se a solução de referência ao criar um programa de afiliados para o seu negócio online. Ultimate Multisite: AffiliateWP é a ponte que lhe permitirá criar o mesmo para a sua rede premium.

## Primeiros passos

Antes de mais, você **precisa ter** [**AffiliateWP**](https://affiliatewp.com/) **ativo**! Não poderá usar o nosso add-on se não tiver **o plugin que se integra com ele**. Caso não planeie permitir que os seus clientes também o usem nos seus sites, o AffiliateWP deve estar ativo apenas no seu site principal.

Além disso, precisará de ter tanto o **Ultimate Multisite** como o **Ultimate Multisite: AffiliateWP** **ativados** **em toda a rede**.

## Adicionar o Ultimate Multisite como uma integração AffiliateWP

No seu site principal, vá a Definições de Afiliados e, no separador Integrações, marque Ultimate Multisite. Isto indicará ao [AffiliateWP](https://affiliatewp.com/) que deve começar a rastrear visitas e conversões a partir do fluxo de inscrição do Ultimate Multisite.

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_Marque a caixa Ultimate Multisite na parte inferior e guarde_

É praticamente tudo o que é necessário. As visitas provenientes de links de referência serão rastreadas e, quando esse utilizador fizer mais tarde o seu primeiro pagamento, a conversão será criada e uma comissão será reservada para o afiliado responsável pela conversão.

No entanto, apenas o primeiro pagamento recebido do utilizador recém-criado será rastreado. Para ativar pagamentos recorrentes de referências, precisará do [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## Ativar referências recorrentes

Depois de instalar o [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/), vá a Definições de Afiliados, separador Referências Recorrentes, e ative o uso de referências recorrentes.

<!-- Screenshot unavailable: AffiliateWP Recurring Referrals settings with Enable Recurring Referrals toggle -->

_Mais simples impossível: ative Enable Recurring Referrals e está tudo pronto!_

Novamente, é tudo o que é necessário!

Agora, sempre que um novo pagamento for recebido numa conta referenciada, o afiliado responsável receberá automaticamente a sua parte!
