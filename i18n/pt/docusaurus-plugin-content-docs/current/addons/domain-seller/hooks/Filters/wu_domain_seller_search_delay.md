# Filtro: wu_domain_seller_search_delay

Filtra o atraso de debounce da pesquisa de domínio em milissegundos.

Aumente este valor para reduzir as chamadas de API em ligações lentas.

## Parâmetros

| Nome | Tipo | Descrição |
|------|------|-------------|
| $delay | `int` | Atraso de debounce em milissegundos. Predefinição 500. |
