# Ama-hook nezihlungi ze-Ultimate Multisite

Ultimate Multisite yakhiwe inqwaba yemisebenzi nemibuzo yedathabheyisi. Inama-hook nezihlungi ezizokuvumela ukuthi uguqule noma wengeze ikhodi entsha emisebenzini ekhona.

Sidale uhlu lwala ma-hook nezihlungi njengereferensi yokusiza abasebenzisi abathuthukile benze ngokwezifiso inethiwekhi yabo ngokwengeziwe. Ungafinyelela uhlu ku-Github ngokuchofoza isixhumanisi esingezansi.
