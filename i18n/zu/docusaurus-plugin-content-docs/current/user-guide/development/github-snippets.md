# Indlela yokusebenzisa izingcezu zekhodi ze-Ultimate Multisite ku-GitHub repository yethu

Kunezingcezu zekhodi ezitholakala ku-GitHub repository ezivame ukucelwa abasebenzisi be-Ultimate Multisite abafuna ukwengeza ukusebenza okuncane njengokwengeza i-Google Analytics script emakhasini okubhalisa noma ukufihla i-meta box ku-admin dashboard.

Lesi sihloko sizokukhombisa ukuthi uyisebenzisa kanjani, noma ngokucacile ukuthi uyibeka kuphi le khodi.

Ungathola izingcezu zekhodi kusixhumanisi esingezansi.

https://github.com/next-press/wp-ultimo-snippets/

Kunezindlela ezi-2 zokwengeza ikhodi

  1. Efayeleni le-functions.php le-theme yakho.

  2. Must-Use Plugins (mu-plugins)

# Indlela yokwengeza ucezu lwekhodi efayeleni le-functions.php le-theme yakho.

  1. Ngena ku-WordPress Network admin dashboard yakho bese uya ku-Themes >Theme Editor (Bona isithombe-skrini esingezansi).

  2. Ekhasini elithi "Edit Themes", qiniseka ukuthi ukhethe i-theme yakho esebenzayo kunkambu yokwehlisa etholakala ohlangothini olungaphezulu kwesokudla kwesikrini sakho (#3 esithombeni-skrini esingezansi).

  3. Chofoza ifayela le-functions.php ngaphansi kwesigaba esithi "Theme Files" ukuze ulayishe ifayela. Skrola uye phansi bese unamathisela ucezu lwekhodi lwe-Ultimate Multisite oluthole ku-GitHub repository.

<!-- Isithombe-skrini asitholakali: WordPress Theme Editor ebonisa ukuhlelwa kwefayela le-functions.php -->

# Indlela yokudala ama-Must-Use Plugins (mu-plugins)

I-WordPress inesici esikuvumela ukuthi ulayishe ukusebenza okwenziwe ngokwezifiso okubizwa ngokuthi "Must-Use Plugins", noma "mu-plugins" ngamafuphi.

Lawa ma-mu-plugins akhethekile alayishwa ngaphambi kwawo wonke amanye ama-plugins avamile, futhi awakwazi ukukhutshazwa. Ku-multisite network, ikhodi kulawa ma-mu-plugins izolayishwa kuwo wonke ama-site ekufakweni kwakho.

1\. Sebenzisa i-FTP noma i-SSH ukufinyelela i-filesystem yokufakwa kwakho kwe-WordPress.

2\. Ngaphakathi kwe-directory ye-wp-content yokufakwa kwakho kwe-WordPress, dala i-directory entsha ebizwa ngokuthi: mu-plugins.

<!-- Isithombe-skrini asitholakali: File manager ebonisa i-directory ye-wp-content enefolda ye-mu-plugins -->

3\. Dala ifayela elisha le-PHP kukhompyutha yakho elibizwa ngokuthi wu-snippet.php usebenzisa i-Notepad noma noma iyiphi i-code editor.

4\. Faka ucezu lwekhodi lwe-Ultimate Multisite oluthole ku-GitHub repository efayeleni bese ulugcina. Ungaphinde wengeze le khodi phezulu kocezu lwekhodi ukuze ulebule i-mu plugin.
