# Umcimbi: Bhalisa i-akhawunti nge-Zapier

Esihlokweni [Ukuhlanganisa Ultimate Multisite ne-Zapier](zapier.md), sixoxe ngokuthi ungayisebenzisa kanjani i-Zapier ukwenza izenzo ezahlukene ngaphakathi kwe-Ultimate Multisite ngokusekelwe kuma-triggers nemicimbi. Kulesi sihloko, sizobonisa ukuthi ungazihlanganisa kanjani izinhlelo zokusebenza zezinkampani zangaphandle. Sizosebenzisa i-Google Sheets njengomthombo wedatha bese sithumela ulwazi ku-Ultimate Multisite ukuze kubhaliswe i-akhawunti.

Ngaphambi kokuxhuma i-Zapier, iya ku-**Ultimate Multisite > Settings > API & Webhooks** bese uqinisekisa ukuthi i-API ivuliwe. Kopisha i-API Key ne-API Secret kulesi sikrini lapho i-Zapier icela imininingwane ye-akhawunti ye-Ultimate Multisite.

![Izilungiselelo ze-API nama-Webhooks ezine-API Key, API Secret, nezinketho ze-Enable API](/img/admin/settings-api-webhooks.png)

Okokuqala, udinga ukudala i-**Google Sheet** ngaphansi kwe-Google Drive yakho. Qinisekisa ukuthi uchaza kahle ikholomu ngayinye ukuze ukwazi ukumatanisa idatha kalula kamuva.

Ngemva kokudala i-Google sheet, ungangena ku-akhawunti yakho ye-Zapier bese uqala ukudala i-zap.

Ngaphansi kwenkambu yokusesha ye-**"App event"** khetha **"Google Sheets"**


Bese kunkambu ethi "**Event** " khetha "**New spreadsheet row** " bese ucindezela "**Continue** "

Isinyathelo esilandelayo sizokucela ukuthi ukhethe i-**Google Account** lapho i-**Google Sheet** igcinwe khona. Ngakho qinisekisa nje ukuthi i-akhawunti ye-google efanele icacisiwe.


Ngaphansi kokuthi **"Set up trigger** ", uzodinga ukukhetha nokucacisa i-google spreadsheet ne-worksheet ozoyisebenzisa lapho idatha izobe ivela khona. Qhubeka ugcwalise lezo zinkambu bese ucindezela "**Continue** "

Okulandelayo ukuthi "**test your trigger** " ukuze uqinisekise ukuthi i-google sheet yakho ixhunywe kahle.

Uma ukuhlola kwakho kuphumelele, kufanele ubone umphumela okhombisa amanani athile avela kuma-spreadsheet akho. Chofoza "**Continue** " ukuze uqhubeke.

Isinyathelo esilandelayo ukusetha isenzo sesibili esizodala noma sibhalise i-akhawunti ku-Ultimate Multisite. Enkambwini yokusesha khetha "**Ultimate Multisite(2.0.2)** "


Ngaphansi kwenkambu ethi "**Event** ", khetha "**Register an Account in Ultimate Multisite** " bese uchofoza inkinobho ethi "**Continue** ".

Ngaphansi kokuthi "**Set up an action** ", uzobona izinkambu ezahlukene ezitholakalayo zedatha yekhasimende, ubulungu, imikhiqizo, njll. Ungamatanisa amanani angaphansi kwe-google sheet yakho bese uwabela enkambwini efanele lapho kufanele agcwaliswe khona njengoba kubonisiwe kusithombe-skrini esingezansi.


Ngemva kokumatanisa amanani, ungahlola isenzo.
