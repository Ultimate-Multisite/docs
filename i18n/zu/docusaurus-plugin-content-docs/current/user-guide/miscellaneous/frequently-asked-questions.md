# Imibuzo Evame Ukubuzwa {#frequently-asked-questions}

Nansi eminye **Imibuzo Evame Ukubuzwa** evame ukuvela ngesikhathi sezicelo zokwesekwa.

  * **Yini i-Ultimate Multisite?**

Ultimate Multisite iyi-WordPress plugin, eyenzelwe ukufakwa kwe-WordPress Multisite, eguqula ukufakwa kwakho kwe-WordPress kube inethiwekhi esezingeni eliphezulu yamasayithi – njenge-[WordPress.com](https://WordPress.com) – ivumele amaklayenti ukuthi adale amasayithi ngezinkokhelo zanyanga zonke, zekota, noma zonyaka (ungakwazi futhi ukudala izinhlelo Zamahhala).

  * **Ingabe i-Ultimate Multisite izosebenza ngaphandle kwe-Multisite?**

Cha. Ultimate Multisite idinga i-WordPress Multisite, i-subdirectory, noma i-subdomain.

  * **Yiziphi izidingo ze-Ultimate Multisite?**

Ultimate Multisite idinga:

**Inguqulo ye-WordPress** : v5.3+ (Kunconywa: Inguqulo Yokugcina Ezinzile)
**Inguqulo ye-PHP** : 7.4.x (Ukusekelwa ku-> 8.0 kuyeza maduze)
**Inguqulo ye-MySQL** : v5+ (Kunconywa: Inguqulo 5.6, uma ungenayo i-8.0 etholakalayo kumhlinzeki wakho wokusingatha)

_I-Multisite (i-subdomain noma i-subdirectory) nayo idinga ukuvulwa._ [Indlela Yokufaka i-WordPress Multisite](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **Ingabe i-Ultimate Multisite izosebenza ne-Multisite enama-Subdirectories?**

Yebo. Ultimate Multisite isekela kokubili ukufakwa kwe-subdirectory nokufakwa kwe-subdirectory.

  * **Ingabe i-Ultimate Multisite izosebenza ne-Multisite enama-Subdomains?**

Yebo. Ultimate Multisite isekela kokubili ukufakwa kwe-subdirectory nokufakwa kwe-subdomain.

  * **Ngingayisebenzisa i-Ultimate Multisite ukukhawulela ukuthi yiziphi Izandiso Namatimu abasebenzisi abangawasebenzisa?**

Yebo. Nge-Ultimate Multisite ungakhetha ukuthi yiziphi izandiso namatimu atholakalayo kuhlelo ngalunye kunethiwekhi yakho.

  * **Ngingayisebenzisa i-Ultimate Multisite ukukhawulela inani lokuthunyelwe umsebenzisi angakudala kusayithi lakhe?**

Yebo. Nge-Ultimate Multisite ungabeka izilinganiso zokuthunyelwe **zazo zonke izinhlobo zokuthunyelwe ezitholakalayo** kunethiwekhi yakho. Ungakhawulela futhi inani labasebenzisi umphathi angabamema, **inani lesikhala sediski,** nokunye okuningi.

  * **Ingabe i-Ultimate Multisite isekela Ama-Template Esayithi?**

Yebo. Nge-Ultimate Multisite ungakha Ama-Template Esayithi amaningi ngendlela ofuna ngayo. Ama-Template Esayithi awawona amatimu, angamasayithi ajwayelekile kunethiwekhi yakho. Ungavula izandiso, amatimu, futhi wenze ngokwezifiso lelo Template Yesayithi nganoma iyiphi indlela oyifunayo. Uma umsebenzisi ekhetha leyo template, isayithi lakhe lizokwakhiwa njengekhophi yaleyo template, nazo zonke izinguquko ezenziwe ngokwezifiso zikhona.

Ungasetha futhi i-template ethile yohlelo. Ngokwenza lokho, wonke umsebenzisi obhalisela lolo hlelo uzoba nesayithi lakhe elidalwe njengokuphindwa kwalelo Template Yesayithi.

  * **Ingabe i-Ultimate Multisite ihlanganiswa ne-WooCommerce?**

Yebo!

Uma ufuna abasebenzisi benethiwekhi yakho babe ne-WooCommerce etholakalayo kubo, ibavumele ukuthi basethe izitolo, lokho kungenzeka usebenzisa i-Ultimate Multisite. Ungakhawulela ngisho nenani lemikhiqizo amaklayenti akho azokwazi ukuyidala ezingeni ngalinye lohlelo.

Mayelana nezinkokhelo zokubhalisa ze-Ultimate Multisite, ungasebenzisa i-[Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) ukuze uvumele amaklayenti akho e-Ultimate Multisite akhokhe esebenzisa amasango e-WooCommerce kusayithi lakho eliyinhloko. Ngaleyo ndlela, ungasebenzisa umtapo omkhulu wamasango okukhokha athuthukiselwe i-WooCommerce futhi uwasebenzisele ukukhokhisa amaklayenti akho e-Ultimate Multisite.

  * **Ingabe ikhona indlela yokuthutha okubhaliselwe kusuka ku-WPMUDEV Pro Sites?**

Yebo, i-Ultimate Multisite manje ikunikeza [ithuluzi eliwusizo lokukusiza uthuthe](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) inethiwekhi yakho ye-Pro Sites.

  * **Kungani ithebhu ethi “Izandiso Ezivunyelwe” ingenalutho ekhasini lami Lokuhlela Uhlelo?**

Ultimate Multisite ikuvumela kuphela ukuthi uvumele noma unqabele ukufinyelela kuzandiso ezingavulwanga ezingeni lenethiwekhi kunethiwekhi yakho. Uma isandiso sivulwe ezingeni lenethiwekhi, ngokwemvelo yaso, sizolayishwa ngokuzenzakalelayo kusayithi ngalinye lenethiwekhi yakho.

Uma ungaboni noma yiziphi izandiso ezibhalwe kuthebhu ethi “Izandiso Ezivunyelwe”, hlola ukuthi isandiso osifunayo sivulwe ezingeni lenethiwekhi noma cha.

  * **Ngingasebenzisa idizayini yami yeThebula Lamanani?**

Yebo. Kusukela kunguqulo 1.4.0, Ultimate Multisite isekela ukweqa ama-template.

  * **Ingabe i-Ultimate Multisite isebenza ne-WP Engine?**

Yebo. Ultimate Multisite yayihlale isekelwa yi-WP Engine, kodwa ukwenza i-domain mapping isebenze kwakudinga izinyathelo ezengeziwe ezenziwa ngesandla ngumlawuli (ukwengeza i-domain kuphaneli ye-WP Engine). **Lokhu akusenjalo kusukela kunguqulo 1.6.0.** Ultimate Multisite ihlanganiswa ngokuqondile ne-WP Engine API futhi ama-domain engezwa kuphaneli ngokuzenzakalelayo ngemva kokumapingwa.

  * **Yimuphi umhlinzeki wokusingatha omncomayo?**

Ultimate Multisite kufanele isebenze nanoma yimuphi umhlinzeki wokusingatha. Umehluko kuphela ukuthi ezinye izinkampani zokusingatha ezifana ne-Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane kanye ne-[Runcloud.io](http://Runcloud.io) zinokuhlanganiswa okuqinile kwamakhono e-domain mapping ngakho zisebenza zisuka nje. Kokunye ukusingatha, kungase kudingeke usethe i-domain mapping ngesandla.
