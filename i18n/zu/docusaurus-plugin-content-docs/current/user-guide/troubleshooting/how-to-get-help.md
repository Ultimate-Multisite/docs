# Ungaluthola kanjani usizo

Usuthole ukhiye wakho we-License osebenzayo, futhi okuhambisana nawo ukufinyelela ethimbeni lethu ukuze likusize uxazulule izinkinga ongase uhlangabezane nazo ngesikhathi sakho sokusetha i-Ultimate Multisite.

Lesi sifundo sihlanganisa iziteshi zokwesekwa ezitholakalayo kubanikazi be-license nokuthi yiluphi uhlobo lolwazi okudingeka ube nalo ukuze uxhumane nethimba lethu lokwesekwa.

## Isiza Sosizo {#help-site}

Ungafinyelela izihloko nezifundo ezahlukene esizeni sethu sosizo ongasithola kusixhumanisi esingezansi.

[)

Sikhuthaza abasebenzisi bethu ukuthi bazijwayeze izici eziyisisekelo nemisebenzi ye-Ultimate Multisite. Kunezihloko ezithi "Ukuqalisa" ezitholakala esizeni sethu sosizo, futhi nezimpendulo zemibuzo evamile zingatholakala lapho, njengokuthi uzifaka kanjani izengezo, ukuhlanganisa kanjani izindlela zokukhokha, njll.

## Xhumana Nokwesekwa {#contact-support}

Uma ubambekile futhi ungakwazi ukuthola impendulo esizeni sethu sosizo, ungaxhumana nethimba lethu lokwesekwa ngokuchofoza isithonjana sebhamuza lenkulumo esitholakala ohlangothini olungezansi kwesokudla lwewebhusayithi yethu NOMA ngokuthumela i-email ngqo ku-support@ultimatemultisite.com.

Lapho uxhumana nokwesekwa ngezinkinga zobuchwepheshe, sicela uqinisekise ukuthi unikeza incazelo enemininingwane yenkinga yakho kanye nesithombe-skrini, imilayezo yamaphutha, noma noma yiluphi ulwazi oluzosiza abantu bethu bokwesekwa ukuthi baqonde futhi bahlole kalula inkinga yakho. Kudingeka futhi unikeze ukhiye wakho we-license. Abanikazi be-license abasebenzayo kuphela abafanele ukuthola ukwesekwa.

## Isiza Somphakathi {#community-site}

[Isiza sethu somphakathi](https://community.ultimatemultisite.com/) siyindawo yabasebenzisi be-Ultimate Multisite yokuxhumana bodwa. Sisebenza njengohlelo lolwazi lapho amalungu angathumela khona, aphawule ezingxoxweni, anikeze izeluleko, asebenzisane, noma aqoqe imibono emphakathini. Kuzodingeka unikeze ukhiye wakho we-license ye-Ultimate Multisite ukuze uvule izindawo/amakhasi ayimfihlo.
