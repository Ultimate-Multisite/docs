# Iphutha Ekufakeni Ifayela le-Sunrise

Ifayela le-sunrise.php liyifayela elikhethekile WordPress elilibhekayo ngesikhathi iziqalisa. Ukuze WordPress ikwazi ukuthola ifayela le-sunrise.php, kudingeka libe ngaphakathi kwe-**ifolda ye-wp-content**.

Uma uvula Ultimate Multisite bese udlula kuwizadi yokusetha efana nale onayo esithombeni-skrini, Ultimate Multisite izama ukukopisha ifayela lethu le-sunrise.php kufolda ye-wp-content.

<!-- Isithombe-skrini asitholakali: Ikhasi lewizadi yokusetha elibonisa isinyathelo sokufaka i-sunrise.php -->

Isikhathi esiningi, siyakwazi ukukopisha ifayela ngempumelelo futhi konke kusebenze. Nokho, uma okuthile kungasethwanga kahle (izimvume zefolda, ngokwesibonelo), ungase uhlangabezane nesimo lapho Ultimate Multisite ingakwazi ukukopisha ifayela.

Uma ufunda umlayezo wephutha u-Ultimo akunika wona, uzobona ukuthi yilokho kanye okwenzekile lapha: **Ukukopishwa kwe-Sunrise kwehlulekile**.

<!-- Isithombe-skrini asitholakali: Umlayezo wephutha obonisa ukuthi Ukukopishwa kwe-Sunrise kwehlulekile -->

Ukuze ukulungise lokho, ungamane ukopishe ifayela le-sunrise.php elingaphakathi kwefolda ye-plugin ye-wp-ultimo bese ulinamathisela kufolda yakho ye-wp-content. Ngemva kokwenza lokho, layisha kabusha ikhasi lewizadi futhi ukuhlolwa kufanele kudlule.

<!-- Isithombe-skrini asitholakali: Umphathi wamafayela obonisa i-sunrise.php ngaphakathi kwefolda ye-plugin ye-wp-ultimo --> Kunoma yikuphi, lokhu kungase kudinge ukuhlolwa okuvamile kwezimvume zamafolda akho ukuze ugweme ukuba nezinkinga esikhathini esizayo (hhayi kuphela nge-Ultimate Multisite kodwa nangamanye ama-plugin nama-theme futhi).

**Ithuluzi le-Health Check** eliyingxenye ye-WordPress (ungalifinyelela nge-site yakho enkulu **iphaneli ye-admin > Amathuluzi > Health Check**) liyakwazi ukukwazisa uma unezimvume zamafolda ezisethwe kumanani angase abangele izinkinga nge-WordPress.
