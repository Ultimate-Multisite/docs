# Ungajoyina kanjani Umphakathi we-Ultimate Multisite

Umphakathi we-Ultimate Multisite uyindawo lapho ungaxhumana khona nabanye abasebenzisi be-Ultimate Multisite futhi wabelane ngamathiphu, okuhlangenwe nakho nokunye. Ukuze ufinyelele Umphakathi we-Ultimate Multisite iya ku-https://community.ultimatemultisite.com/home.

Ekhasini lokwamukela iya kusixhumanisi esithi “**ukudala i-Account yakho lapha** ”.

<!-- Isithombe-skrini asitholakali: Ikhasi lokwamukela lomphakathi elinesixhumanisi sokudala i-Account yakho -->

Uzoqondiswa kabusha ekhasini lethu lokubhalisa. Manje udinga ukufaka igama lomsebenzisi, i-imeyili yakho nephasiwedi. Yenza i-Captcha bese uchofoza inkinobho ethi **Bhalisa**.

<!-- Isithombe-skrini asitholakali: Ikhasi lokubhalisa lomphakathi elinezinkambu zegama lomsebenzisi, i-imeyili, iphasiwedi kanye ne-Captcha -->

Ngemva kokubhalisa sekuyisikhathi sokudala iphrofayela yakho. Kuleli khasi ungakwazi ukulayisha isithombe sakho, ufake igama lakho eligcwele, ubhale umlando omfishane ngawe nokunye. Futhi, engeza izixhumanisi zakho zemithombo yezokuxhumana bese usetha izimvume ze-Account yakho.

<!-- Isithombe-skrini asitholakali: Ikhasi lokudala iphrofayela yomphakathi elinokulayishwa kwesithombe nezinkambu zomlando omfishane -->

Manje usekushaneli ethi “Izethulo”, lapho ungazethula khona futhi wazi kancane ngamanye amalungu. Ungadala okuthunyelwe kwakho ngokuchofoza inkinobho ethi **Okuthunyelwe Okusha**.

<!-- Isithombe-skrini asitholakali: Ishaneli yezethulo zomphakathi enenkinobho ethi Okuthunyelwe Okusha -->

Amashaneli asesigabeni se-Lounge kanye neshaneli ethi “Dev Logs” avulekele wonke umuntu. Nokho uzodinga imvume ekhethekile ukuze ufinyelele amanye amashaneli emphakathini.

<!-- Isithombe-skrini asitholakali: Amashaneli omphakathi abonisa isigaba se-Lounge kanye neshaneli ethi Dev Logs -->

Lokhu kufinyelela kuzonikezwa uma ususinikeze ukhiye wakho welayisensi we-Ultimate Multisite osebenzayo. Ukuze ukwenze lokho chofoza kunoma yisiphi isixhumanisi seshaneli yangasese bese uchofoza inkinobho ethi **Vula**.

<!-- Isithombe-skrini asitholakali: Ishaneli yangasese yomphakathi enenkinobho ethi Vula -->

Uzoqondiswa kabusha ekhasini lethu lokuvula ukufinyelela. Faka ukhiye wakho welayisensi we-Ultimate Multisite bese uchofoza inkinobho ethi Vula Ukufinyelela:

<!-- Isithombe-skrini asitholakali: Ikhasi lomphakathi lokuvula ukufinyelela elinokufaka ukhiye welayisensi -->

Ilayisensi yakho izoqinisekiswa futhi uzokwazi ukufinyelela amashaneli angasese. Le nqubo ingathatha amahora afinyelela kwangu-24, ngakho chofoza isixhumanisi ukuze ubuyele emphakathini futhi ujabulele ezinye izikhala ngenkathi sicubungula isicelo sakho.
