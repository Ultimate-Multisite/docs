# Ukubeka izikhangiso kunethiwekhi yakho nge-Ultimate Multisite: Ad Injector {#placing-ads-on-your-network-with-ultimate-multisite-ad-injector}

Ultimate Multisite: Ad Injector iyi-**add-on yamahhala** ekunika amathuluzi okubeka izikhangiso ezinhlelweni zakho ngqo ekhasini lazo lokuhlela.

Ngemva kokufaka Ultimate Multisite: Ad Injector, iya ekhasini lakho le-_Plugins_ ukuze uyivule.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Chofoza inkinobho ethi Network Activate_

Uma i-add-on isisebenza, ibhokisi le-meta le-_Advanced Options_ esikrinini sokuhlela uhlelo lizoba nenketho entsha ebizwa ngokuthi _Ad Settings_.

![Ad Settings esikrinini sokuhlela uhlelo](/img/config/product-edit-full.png)

_Manje ungahlela izilungiselelo zezikhangiso zohlelo ngalunye lwezinhlelo zakho_

Kuleyo thebhu, uzokwazi ukwengeza amakhodi ezikhangiso okuthunyelwe namakhasi ku-front-end, ngaphambi nangemva kokuqukethwe.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Ngemva kokufaka ikhodi yesikhangiso sakho, izovela kokuthunyelwe nasemakhasini alolo hlelo oluthile._

Ungakwazi futhi ukufaka izikhangiso ku-header yamakhasi e-Admin Pages kubasebenzisi balolo hlelo.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Iya kunketho ethi Admin Panel Ad Code uma ufuna ukubonisa izikhangiso ku-back-end yomsebenzisi wakho_

### Ngale Kokukhangisa {#beyond-advertising}

Ungasebenzisa futhi i-Ultimate Multisite: Ad Injector ukunikeza ukwesekwa ezinhlelweni ezisezingeni eliphezulu.

Vele uye kunketho ethi _Admin Panel Ads_ bese wengeza ikhodi ye-HTML yethuluzi lakho lokwesekwa.
