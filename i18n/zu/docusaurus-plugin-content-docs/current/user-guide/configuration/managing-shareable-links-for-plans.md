# Ukuphatha Izixhumanisi Ezabelanekayo Zama-plan

Ultimate Multisite ikunikeza inketho yokwabelana ngezixhumanisi zama-plan ngokuqondile namakhasimende akho. Ukuze wenze lokhu, iya kunoma yiliphi ikhasi lokuhlela lama-plan akho bese uchofoza inkinobho ethi _Chofoza ukuze ukopishe i-Shareable Link_.

![Ikhasi lokuhlela lomkhiqizo elinenkinobho yesixhumanisi esabelanekayo](/img/config/product-edit-full.png)

_Ekhasini Lokuhlela le-Plan, ungathola inkinobho yokwakha izixhumanisi ezabelanekayo_

Qaphela ukuthi le ndlela isebenzisa imvamisa yokukhokhisa ezenzakalelayo ukuze yakhe isixhumanisi. Lokhu kusho ukuthi uma ukukhokhisa kwakho okuzenzakalelayo kungokwenyanga, isibonelo, ekugcineni uzoba nesixhumanisi esinjengalesi: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Uma ufuna ukuthi isixhumanisi esabelanekayo sibe ngokuzenzakalelayo sekota, kuzodingeka ungeze 3 ekugcineni

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Ukuze ukhethe i-plan _premium_ ohlelweni lonyaka, ungeza 1 bese kuba igama:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Khumbula ukuthi awukwazi ukuba nesixhumanisi esisodwa kuphela sezimvamisa ezahlukene zokukhokhisa. Kufanele wabelane ngesixhumanisi esisodwa esithile ngemvamisa ngayinye yokukhokhisa kwakho.

Uma usuwabelane ngesixhumanisi namakhasimende akho, sizobaqondisa esinyathelweni sokuqala senqubo yakho yokubhalisa.
