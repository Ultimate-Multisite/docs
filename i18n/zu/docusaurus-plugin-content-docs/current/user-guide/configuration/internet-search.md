# Ukusesha Ku-inthanethi

Gratis AI Agent v1.5.0 yengeza ikhono le-**Ukusesha Ku-inthanethi** elivumela umsizi we-AI ukuthi athole ulwazi lwakamuva oluvela kuwebhu phakathi nengxoxo. Lokhu kunikwa amandla yi-[Brave Search API](https://brave.com/search/api/) noma i-[Tavily API](https://tavily.com/).

## Indlela Okusebenza Ngayo

Lapho ukusesha ku-inthanethi kunikwe amandla, umsizi angabuza ngokuzenzakalelayo umhlinzeki wokusesha ommisile lapho enquma ukuthi umbuzo udinga ulwazi lwamanje noma lwangaphandle — isibonelo, izindaba zakamuva, amanani abukhoma, noma amadokhumenti okungenzeka ashintshile kusukela emkhawulweni wokuqeqeshwa kwemodeli.

Imiphumela itholwa ngesikhathi sangempela futhi ifakwa kumongo womsizi ngaphambi kokuba akhiqize impendulo. Umsizi uyakhombisa lapho esebenzise imiphumela yokusesha ukuphendula umbuzo.

## Ukunika Amandla Ukusesha Ku-inthanethi

Ukusesha ku-inthanethi kudinga i-API key evela kumhlinzeki wokusesha omkhethile. Ukuze ukumise:

1. Iya ku-**Gratis AI Agent → Izilungiselelo → Okuthuthukile**.
2. Thola uhlu lokukhetha olwehlayo lwe-**Umhlinzeki Wokusesha Ku-inthanethi** bese ukhetha noma i-**Brave Search** noma i-**Tavily**.
3. Faka i-API key yakho enkambini ehambisanayo. Izixhumanisi zokubhalisa ziboniswa eduze kwenkambu ngayinye.
4. Chofoza **Londoloza Izilungiselelo**.

Uma ukhiye usulondoloziwe, ikhono le-Ukusesha Ku-inthanethi litholakala ngokuzenzakalelayo kumsizi.

## Brave Search

### Ukuthola i-Brave Search API Key

1. Vakashela [ikhasi le-Brave Search API](https://brave.com/search/api/).
2. Bhalisela uhlelo. Kukhona izinga lamahhala elinomkhawulo wezicelo wanyanga zonke.
3. Kopisha i-API key yakho ku-Brave Search developer dashboard.
4. Yinamathelise enkambini ye-**Brave Search API Key** kuzilungiselelo ze-Gratis AI Agent.

### Imikhawulo Yokusetshenziswa

Ukusetshenziswa kukhokhiswa yi-Brave Search ngokusekelwe enanini lemibuzo eyenziwe. Impendulo ngayinye ye-AI evusa ukusesha ibalwa njengombuzo owodwa. Gada ukusetshenziswa kwakho ku-[Brave Search developer dashboard](https://brave.com/search/api/) ukuze ugweme izindleko ezingalindelekile.

## Tavily

Superdav AI Agent v1.10.0 yengeza i-**Tavily** njengomunye umhlinzeki wokusesha ku-inthanethi, inikeza imiphumela yokusesha ecebile namakhono ocwaningo athuthukile.

### Ukuthola i-Tavily API Key

1. Vakashela [ikhasi le-Tavily API](https://tavily.com/).
2. Bhalisela i-Account. Kukhona izinga lamahhala elinomkhawulo wezicelo wanyanga zonke.
3. Kopisha i-API key yakho ku-Tavily dashboard.
4. Yinamathelise enkambini ye-**Tavily API Key** kuzilungiselelo ze-Gratis AI Agent.

### Imikhawulo Yokusetshenziswa

Ukusetshenziswa kukhokhiswa yi-Tavily ngokusekelwe enanini lama-API calls enziwe. Impendulo ngayinye ye-AI evusa ukusesha ibalwa njenge-call eyodwa. Gada ukusetshenziswa kwakho ku-[Tavily dashboard](https://tavily.com/) ukuze ugweme izindleko ezingalindelekile.

## Ukukhubaza Ukusesha Ku-inthanethi

Susa i-API key enkambini yomuhlinzeki wokusesha osebenzayo bese ulondoloza. Ikhono le-Ukusesha Ku-inthanethi ngeke lisatholakala kumsizi.

:::note
Ukusesha ku-inthanethi kwengeza ukubambezeleka ezimpendulweni ngoba umsizi kufanele alinde imiphumela yokusesha ngaphambi kokukhiqiza impendulo. Ezimweni zokusetshenziswa ezidinga isikhathi esisheshayo, cabanga ukuthi ukusesha ngesikhathi sangempela kuyadingeka yini noma ukuthi ulwazi olwakhelwe ngaphakathi kumsizi lwanele yini.
:::
