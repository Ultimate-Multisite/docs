# Ukusesha Nokushintsha Okuzenzakalelayo Kuma-template Esayithi (v2)

_**Lesi sifundo sidinga i-WP UItimo inguqulo 2.x.**_

Esinye sezici ezinamandla kakhulu ze-Ultimate Multisite ikhono lokwengeza umbhalo ongahleliwe, umbala, nezinkambu zokukhetha efomini lokubhalisa. Uma sesiyitholile leyo datha, singayisebenzisa ukugcwalisa kusengaphambili okuqukethwe ezingxenyeni ezithile ze-template yesayithi ekhethiwe. Bese, lapho isayithi elisha lishicilelwa, i-Ultimate Multisite izoshintsha izibambi-ndawo ngolwazi lwangempela olufakwe ngesikhathi sokubhalisa.

Isibonelo, ungakha amasayithi akho e-template anezibambi-ndawo. Izibambi-ndawo kufanele zengezwe zizungezwe ama-curly braces aphindwe kabili - {{placeholder_name}}.

Bese, ungavele wengeze inkambu yokubhalisa ehambisanayo ukuze uthole leyo datha

Ikhasimende lakho lizokwazi ukugcwalisa leyo nkambu ngesikhathi sokubhalisa.

I-Ultimate Multisite izobe isishintsha izibambi-ndawo ngedatha enikezwe ikhasimende ngokuzenzakalelayo.

## **Ukuxazulula inkinga ye-"template egcwele izibambi-ndawo"**

Konke lokho kuhle, kodwa siyahlangabezana nenkinga embi: manje ama-template ethu amasayithi - angavakashelwa amakhasimende ethu - agcwele izibambi-ndawo ezimbi ezingasho lutho oluningi.

Ukuze sikuxazulule lokho, sinikeza inketho yokusetha amanani mbumbulu ezibambi-ndawo, bese sisebenzisa lawo manani ukusesha nokushintsha okuqukethwe kwazo kumasayithi e-template ngesikhathi amakhasimende akho evakasha.

Ungafinyelela kumhleli wezibambi-ndawo ze-template ngokuya ku-**Ultimate Multisite > Izilungiselelo > Amasayithi**, uskrole uye endaweni ethi Izinketho ze-Template Yesayithi, bese uchofoza isixhumanisi esithi **Hlela Izibambi-ndawo**.

![Indawo Yezinketho ze-Template Yesayithi ekhasini lezilungiselelo Zamasayithi](/img/config/settings-sites-templates-section.png)

Lokho kuzokuyisa kumhleli wokuqukethwe kwezibambi-ndawo, lapho ungangeza khona izibambi-ndawo nokuqukethwe kwazo okuhambisanayo.

![Indawo yokungena yomhleli wezibambi-ndawo ze-template](/img/config/settings-sites-templates-section.png)
