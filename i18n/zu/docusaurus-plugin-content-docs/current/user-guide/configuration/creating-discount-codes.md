# Ukudala Amakhodi Esaphulelo (v2)

_**INOTHI ELIBALULEKILE: Lesi sihloko sibhekisela ku-Ultimate Multisite version 2.x.**_

Nge-Ultimate Multisite ungadala amakhodi esaphulelo ukuze unike amakhasimende akho izaphulelo kuma-subscriptions awo. Futhi ukuwadala kulula!

## Ukudala Nokuhlela Amakhodi Esaphulelo

Ukuze udale noma uhlele ikhodi yesaphulelo, iya ku-**Ultimate Multisite > Discount Codes**.

![Uhlu lwamakhodi esaphulelo — isimo esingenalutho ngaphambi kokuthi kube khona amakhodi](/img/config/discount-codes-empty.png)

Lapho uzoba nohlu lwamakhodi esaphulelo osuvele uwadalile.

Ungachofoza ku-**Add Discount** **Code** ukuze udale ikhuphoni entsha noma ungahlela lawo onawo ngokuhambisa igundane phezu kwawo bese uchofoza **Edit**.

![Uhlu lwamakhodi esaphulelo olunezenzo zokuhambisa igundane ezibonisa izixhumanisi ze-Edit ne-Delete](/img/config/discount-codes-list-hover.png)

![Inkinobho ethi Add Discount Code kunhlokweni yekhasi](/img/config/discount-codes-add-button.png)

Uzoqondiswa kabusha ekhasini lapho uzodala noma uhlele ikhodi yakho yekhuphoni. Kulesi sibonelo sizodala entsha.

![Ikhasi lokuhlela ikhodi yesaphulelo elinazo zonke izigaba ezibonakalayo](/img/config/discount-code-edit.png)

Ake sibheke izilungiselelo ezitholakala lapha:

**Enter Discount Code:** Leli yigama nje lekhodi yakho yesaphulelo. Lena akuyona ikhodi amakhasimende akho azodinga ukuyisebenzisa kufomu le-checkout.

**Description:** Lapha, ungachaza kafushane ukuthi le khuphoni ingeyani.

![Izinkambu zegama lekhodi yesaphulelo nencazelo phezulu ekhasini lokuhlela](/img/config/discount-code-description.png)

Ungaphinda ubuke ikhodi yesaphulelo njengeyasebenzayo noma engasebenzi:

![Isimo sokusebenza kwekhodi yesaphulelo](/img/config/discount-code-active.png)

**Coupon code:** Lapha yilapho uchaza khona ikhodi amakhasimende akho azodinga ukuyifaka ngesikhathi se-checkout.

![Inkambu yekhodi yekhuphoni lapho amakhasimende afaka khona ikhodi ku-checkout](/img/config/discount-code-coupon-field.png)

**Discount:** Lapha, ungasetha **iphesenti** noma **inani elinqunyiwe** lemali lekhodi yakho yesaphulelo.

![Isilungiselelo senani lesaphulelo esine-dropdown yephesenti noma yenani elinqunyiwe](/img/config/discount-code-amount.png)

**Apply to renewals:** Uma le nketho icishiwe, le khodi yesaphulelo izosetshenziswa kuphela **enkokhelweni yokuqala**. Zonke ezinye izinkokhelo ngeke zibe nesaphulelo. Uma le nketho ivuliwe, ikhodi yesaphulelo izosebenza kuzo zonke izinkokhelo zesikhathi esizayo.

**Setup fee discount:** Uma le nketho icishiwe, ikhodi yekhuphoni **ngeke inikeze noma yisiphi isaphulelo enkokhelweni yokusetha** ye-oda. Uma le nketho ivuliwe, ungasetha isaphulelo (iphesenti noma inani elinqunyiwe) le khodi yekhuphoni ezosisebenzisa enkokhelweni yokusetha yezinhlelo zakho.

![Izinketho zokuvula/ukuvala ze-Apply to renewals ne-setup fee discount](/img/config/discount-code-renewals.png)

**Active:** Sebenzisa mathupha noma uyekise ukusebenza le khodi yekhuphoni.

![Iswishi ye-Active yokunika amandla noma ukukhubaza mathupha ikhodi yesaphulelo](/img/config/discount-code-active.png)

Ngaphansi kwe-**Advanced Options**, sinezilungiselelo ezilandelayo:

![Izinketho ezithuthukile zekhodi yesaphulelo](/img/config/discount-code-advanced.png)

**Limit uses:**

  * **Uses:** Lapha, ungabona ukuthi ikhodi yesaphulelo isetshenziswe izikhathi ezingaki.

  * **Max uses:** Lokhu kuzokhawulela inani lezikhathi abasebenzisi abangasebenzisa ngazo le khodi yesaphulelo. Isibonelo, uma ufaka u-10 lapha, ikhuphoni ingasetshenziswa izikhathi eziyi-10 kuphela. Ngemva kwalo mkhawulo, ikhodi yekhuphoni ngeke isasetshenziswa.

![Isilungiselelo sokukhawulela ukusetshenziswa esinenani lamanje lokusetshenziswa nenkambu ye-max uses](/img/config/discount-code-limit-uses.png)

**Start & expiration dates:** Lapha uzoba nenketho yokwengeza usuku lokuqala kanye/noma usuku lokuphelelwa yisikhathi kukhuphoni yakho.

![Izinkambu zosuku lokuqala nolokuphelelwa yisikhathi zokuhlela ikhodi yesaphulelo](/img/config/discount-code-dates.png)

**Limit products:** Uma uvula **Select products**, yonke imikhiqizo yakho izoboniswa kuwe. Uzoba nenketho yokukhetha mathupha (ngokuvula noma ukuvala) ukuthi yimuphi umkhiqizo ongayamukela le khodi yekhuphoni. Imikhiqizo evaliwe lapha ngeke ibonise noma yiluphi ushintsho uma amakhasimende akho ezama ukusebenzisa le khodi yekhuphoni kuyo.

![Isigaba sokukhawulela imikhiqizo esinama-toggle switches omkhiqizo ngamunye](/img/config/discount-code-limit-products.png)

Ngemva kokusetha zonke lezi zinketho, chofoza ku-**Save Discount Code** ukuze ulondoloze ikhuphoni yakho futhi sekuphelile!

![Inkinobho ethi Save Discount Code ezansi ekhasini lokuhlela](/img/config/discount-code-save.png)

Ikhuphoni manje isesohlwini lwakho futhi, kusuka lapho, ungachofoza ukuze **uyihlele noma uyisuse**.

![Umugqa wekhodi yesaphulelo ohlwini onezenzo ze-Edit ne-Delete lapho uhambisa igundane](/img/config/discount-codes-list-hover.png)

### Ukusebenzisa Amapharamitha e-URL:

Uma ufuna ukwenza ngendlela oyifisayo amathebula akho entengo noma ukwakha ikhasi elihle lekhodi yekhuphoni le-website yakho futhi ufuna ukufaka ikhodi yesaphulelo kufomu lakho le-checkout ngokuzenzakalelayo, ungakwenza lokhu ngamapharamitha e-URL.

Okokuqala, udinga ukuthola isixhumanisi esabelwanekayo secebo lakho. Ukuze wenze lokhu, iya ku-**Ultimate Multisite > Products** bese ukhetha icebo.

Chofoza inkinobho ethi **Click to Copy Shareable Link**. Lokhu kuzokunika isixhumanisi esabelwanekayo saleli cebo elithile. Kithina, isixhumanisi esabelwanekayo esinikeziwe bekuyi-[_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/)_._

![Ikhasi lomkhiqizo elinenkinobho yesixhumanisi esabelwanekayo](/img/config/products-list.png)

Ukuze usebenzise ikhodi yakho yesaphulelo kuleli cebo elithile, vele wengeze ipharamitha ethi **?discount_code=XXX** ku-URL. Lapho **XXX** kuyikhodi yekhuphoni.

Esibonelweni sethu lapha, sizosebenzisa ikhodi yekhuphoni ethi **50OFF** kulo mkhiqizo othile.

I-URL yaleli cebo elithile futhi enekhodi yesaphulelo ethi 50OFF esetshenzisiwe izobukeka kanje: [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/) _**?discount_code=50OFF**_.
