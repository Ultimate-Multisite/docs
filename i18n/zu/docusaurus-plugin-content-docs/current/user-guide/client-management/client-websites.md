# Amawebhusayithi Ekhasimende Lakho (v2) {#your-clients-websites-v2}

Ngemva kokubhalisa kunethiwekhi yakho, amakhasimende akho angakwazi ukufinyelela ku-Dashboard yawo.

Ngaphandle kwekhasi le-**Account**, lapho engabona khona imininingwane ebalulekile mayelana nezinkokhelo zawo, ubulungu, ama-domain, imikhawulo ye-plan nokunye, futhi anokufinyelela ekhasini elithi **Ama-Site Ami**, lapho engabona futhi aphathe wonke amawebhusayithi awo. Okumele akwenze nje ukuchofoza ku-Ama-Site Ami ku-Dashboard yawo.

![Isixhumanisi se-Ama-Site Ami ku-Dashboard yekhasimende](/img/account-page/my-sites-link.png)

Azoboniswa wonke amawebhusayithi awo, ngenkinobho ethi **Phatha** ngezansi ngokuqondile kwalelo nalelo. Uma echofoza kuyo, azoqondiswa kabusha ku-**Dashboard yale webhusayithi ethile**.

Angabona futhi ukuthi iyiphi kumawebhusayithi awo eyi-**primary** kuma-plan awavumela ukuba abe newebhusayithi engaphezu kweyodwa.

![Ikhasi le-Ama-Site Ami elibonisa amawebhusayithi ekhasimende anezinkinobho zokuthi Phatha](/img/account-page/my-sites-page.png)

Nansi ukubuka okuphelele kwe-interface yokuhlela i-site:

![I-interface ephelele yokuhlela i-site](/img/admin/site-edit-full.png)

Uma efuna ukushintsha iwebhusayithi yawo eyinhloko, kudingeka aye ekhasini le-Account ngokuchofoza imenyu ethi **Account**.

![Isixhumanisi semenyu ye-Account ku-Dashboard yekhasimende](/img/account-page/account-menu.png)

Ungabuka futhi uhlele imininingwane yekhasimende:

![I-interface yokuhlela ikhasimende](/img/admin/customer-edit.png)

Bese, kufanele achofoze ku-**Shintsha i-Site Ezenzakalelayo**.

![Inkinobho ethi Shintsha i-Site Ezenzakalelayo ekhasini le-Account](/img/account-page/change-default-site.png)

Iwindi le-pop-up lizovela. Chofoza ohlwini olwehlayo bese ukhetha ukuthi iyiphi iwebhusayithi ofuna ukuyenza iwebhusayithi eyinhloko.

![Ukukhetha kuhlu olwehlayo lwe-Shintsha i-Site Ezenzakalelayo](/img/account-page/change-default-site-dropdown.png)

Bese, uchofoza nje ku-**Shintsha i-Site Ezenzakalelayo** futhi usuqedile!

![Ukuqinisekisa ukushintshwa kwe-site ezenzakalelayo](/img/account-page/change-default-site-confirm.png)
