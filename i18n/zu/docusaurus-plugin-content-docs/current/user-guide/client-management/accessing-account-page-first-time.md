# Ngingalifinyelela kanjani ikhasi lami le-Account okokuqala? {#how-can-i-access-my-account-page-for-the-first-time}

Siyakwamukela ku-Ultimate Multisite!

Finyelela ikhasi lakho le-Account ngesixhumanisi osithole ku-e-mail yakho ngemva kokuthenga i-plugin yethu:

<!-- Screenshot unavailable: Account page email with access link -->

Sebenzisa i-email oyinikezile ngesikhathi sokuthenga kanye ne-password oyithole ku-email yakho.

<!-- Screenshot unavailable: Login page with email and password fields -->

Njengoba kungokokuqala ungena kule Account, uzocelwa ukuthi ushintshe i-password yakho. Yishintshe bese uchofoza inkinobho ethi _Save_.

<!-- Screenshot unavailable: Change password form with Save button -->

Ekhasini lakho le-Account, ungaphatha yonke imininingwane ye-Account yakho njengokukhokhiswa, ukuvuselelwa, ukukhanselwa, nezinye izinto eziningi.
