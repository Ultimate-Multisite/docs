# Ukukhansela Ubulungu (v2) {#canceling-a-membership-v2}

_**INOTHI ELIBALULEKILE: Le athikili ibhekisela ku-Ultimate Multisite inguqulo 2.x.**_

Ukukhansela ubulungu ohlangothini lwamakhasimende akho kulula kakhulu.

Uma umsebenzisi wakho efuna ukukhansela ubulungu bakhe, kufanele aqale afinyelele ekhasini lakhe le-****Account****.

Lapho, kufanele achofoze aye ku-**Indawo Eyingozi**, bese achofoza ku-**Susa Account**.

![Ikhasi le-Account Indawo Eyingozi enenkinobho ethi Susa Account](/img/account-page/danger-zone.png)

Imojuli izovuleka icela ukuqinisekiswa kwabo. Kufanele bavule inketho ethi **Qinisekisa Ukususwa kwe-Account** bese bechofoza ku-**Susa Account**.

![Ingxoxo ethi Qinisekisa Ukususwa kwe-Account](/img/account-page/delete-account-confirm.png)

Lokhu kuzobakhipha kusayithi futhi kukhansele ubulungu babo kunethiwekhi yakho, kanye nokususa isayithi labo kunethiwekhi yakho.

![Ukhishiwe ngemva kokususwa kwe-account](/img/account-page/logged-out-after-delete.png)

![Ukuqinisekiswa kobulungu obukhanseliwe](/img/account-page/membership-canceled.png)

**Kubalulekile:** Qaphela ukuthi lesi senzo ngeke sikhiphe ikhasimende kunethiwekhi yakho, ngoba ikhasimende lingaba nobulungu obungaphezu kobunye noma lingabi nabulungu obuxhunywe kulo.

![Ikhasimende lisekhona kunethiwekhi ngemva kokukhansela](/img/admin/customer-after-cancel.png)

![Ikhasimende elingenabo ubulungu obusebenzayo](/img/admin/memberships-customer-empty.png)
