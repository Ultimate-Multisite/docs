# Ukwehlisela phansi uhlelo (v2)

_**INOTHI ELIBALULEKILE: Lesi sihloko sibhekisela ku-Ultimate Multisite version 2.x.**_

Ukwehlisela phansi uhlelo noma okubhaliselwe kuyisenzo esivamile amakhasimende akho angasenza uma enesabelomali esilinganiselwe noma enqume ukuthi ngeke adinge izinsiza eziningi ukuze asebenzise isayithi lawo elingaphansi.

## Indlela yokwehlisela phansi uhlelo

Amakhasimende akho angakwazi ukwehlisela phansi uhlelo lwawo nganoma yisiphi isikhathi ngokungena ku-Dashboard yokuphatha yesayithi lawo elingaphansi bese achofoza **Shintsha** ngaphansi kwekhasi lawo le-Account.

![Ikhasi le-Account lekhasimende elinekhadi elithi Ubulungu Bakho nenkinobho ethi Shintsha](/img/account-page/membership-change-button.png)

Lapho echofoza inkinobho ethi **Shintsha**, umsebenzisi/ikhasimende uzoqondiswa ekhasini lokukhokha lapho angakhetha khona uhlelo afuna ukushintshela kulo okubhaliselwe kwakhe.

![Ikhasi lezinketho zokwehlisela phansi uhlelo ohlangothini lwekhasimende](/img/account-page/downgrade-picker.png)

Kulesi sibonelo, sehlisela phansi uhlelo kusuka ku-**Premium** kuya ku-**Free**.

Ukuze aqhubeke, umsebenzisi udinga nje ukuchofoza inkinobho ethi **Qedela Ukukhokha**. Lokhu kuzombuyisela ekhasini le-Account elibonisa umlayezo mayelana noshintsho olulindile lobulungu. Izinguquko zizoqala ukusebenza **emjikelezweni olandelayo wokukhokhisa** wekhasimende.

![Ikhasi le-Account elibonisa ibhena yoshintsho lobulungu olulindile](/img/account-page/pending-change.png)

### Kwenzekani lapho umsebenzisi ehlisela phansi uhlelo lwakhe

Kubalulekile ukuqaphela ukuthi ukwehlisela phansi uhlelo akuguquli ukulungiselelwa okukhona kusayithi elingaphansi lomsebenzisi.

Akushintshi ngokuzenzakalelayo isifanekiso sesayithi ngoba ukushintsha isifanekiso sesayithi kuzosula ngokuphelele futhi kusethe kabusha isayithi elingaphansi. Lokhu kwenzelwa ukugwema ukulahleka kwedatha okungadingekile. Ngakho isikhala sediski, amatimu, ama-plugin njll. kuzohlala kunjengoba kunjalo ngaphandle kokuthunyelwe.

Siyaqonda ukuthi ukukhathazeka kwakho okuyinhloko kungaba imikhawulo namakota owamisa ngaphansi kohlelo ngalunye, kodwa kufanele sicabangele umonakalo obungadaleka kusayithi elingaphansi lomsebenzisi uma sisusa noma sishintsha noma yikuphi kokulungiselelwa kwalo.

Mayelana nokuthunyelwe okweqa umkhawulo obekwe ohlelweni, unezinketho ezi-3 ezahlukene: **Gcina okuthunyelwe njengoba kunjalo** *,* **Hambisa okuthunyelwe kudoti** *,* noma **Hambisa okuthunyelwe kokusalungiswa** *.* Ungakulungiselela lokhu ngaphansi kwezilungiselelo ze-Ultimate Multisite.

![Ikhasi le-Network Admin Settings Sites elibonisa izinketho zokuziphatha lapho umkhawulo wokuthunyelwe weqiwe](/img/account-page/settings-sites-post-limit.png)

### Kwenzekani enkokhelweni

Ku-version 2.0, akusadingi noma yiziphi izilungiso enkokhelweni mayelana nokubalwa ngokwengxenye yesikhathi.

Lokhu kungenxa yokuthi uhlelo luzolinda ubulungu obukhona ukuthi **buqede umjikelezo wabo wokukhokhisa ngaphambi kokuba** uhlelo/ubulungu obusha buqale ukusebenza. Inani elisha lokukhokhisa lobulungu obusha lizosetshenziswa ngokuzenzakalelayo futhi likhokhiswe emjikelezweni olandelayo wokukhokhisa.
