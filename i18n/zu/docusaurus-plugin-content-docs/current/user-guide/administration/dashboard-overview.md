# Ukubuka konke kwe-Dashboard {#dashboard-overview}

Ultimate Multisite ine-**Dashboard** ezinikele ngaphansi kwemenyu ye-Ultimate Multisite, futhi ingeza namawijethi ku-Dashboard Yenethiwekhi. Lesi sihloko sichaza lokho ozokubona kuzo zombili izindawo.

## I-Dashboard Yenethiwekhi {#the-network-dashboard}

Ultimate Multisite ifaka amawijethi awusizo ku-dashboard yezinga lenethiwekhi etholakala ku-**Amasayithi Ami → Ukuphatha Inethiwekhi → Dashboard**.

![I-Dashboard Yenethiwekhi enamawijethi e-Ultimate Multisite](/img/admin/network-dashboard-full.png)

Nakhu okuhlinzekwa iwije thi ngayinye:

**Izinyathelo Zokuqala:** Ivela ekufakweni okusha futhi ikukhumbuza ukuthi uqedele imisebenzi yokusetha njengokumisa indlela yokukhokha nokudala i-akhawunti yokuhlola.

![Iwije thi Yezinyathelo Zokuqala](/img/admin/network-dashboard-first-steps.png)

**Isifinyezo:** Ibika inani lokubhalisa nemali engenayo yosuku. I-MRR (Imali Engenayo Ephindaphindayo Yanyanga zonke) ikhombisa isamba esilindelekile kuwo wonke amakhasimende anobulungu obuletha imali engenayo.

![Iwije thi Yesifinyezo](/img/admin/network-dashboard-summary.png)

**Umsinga Womsebenzi:** Ibika ngezehlakalo ku-Ultimate Multisite yonke — ukubhalisa, ukukhanselwa, ukudalwa kwamasayithi, ukususwa, neminye imisebenzi.

![Iwije thi Yomsinga Womsebenzi](/img/admin/network-dashboard-activity-stream.png)

**Okwamanje:** Ibonisa isifinyezo esifushane sabasebenzisi namasayithi kunethiwekhi, nezinqamuleli zokudala amasayithi noma abasebenzisi abasha.

![Iwije thi Yokwamanje](/img/admin/network-dashboard-right-now.png)

**Izindaba Nezingxoxo:** Ilanda futhi ibonise izindaba zakamuva ze-Ultimate Multisite okuhlanganisa izibuyekezo, amaphutha abucayi, nezilungiso zokuphepha.

![Iwije thi Yezindaba](/img/admin/network-dashboard-news.png)

## I-Dashboard ye-Ultimate Multisite {#the-ultimate-multisite-dashboard}

Nakuba i-dashboard yenethiwekhi iveza ulwazi oluphelele mayelana nenethiwekhi, i-dashboard ye-Ultimate Multisite (ezingeni eliphezulu lemenyu ye-Ultimate Multisite) iveza ulwazi mayelana nesevisi.

![I-Dashboard ye-Ultimate Multisite](/img/admin/um-dashboard-top.png)

**Ukukhula Kwemali Engenayo Ephindaphindayo Yanyanga zonke:** Ibonisa ukuhlukaniswa kwemali engenayo kwanyanga zonke esikhathini sekhalenda sezinyanga eziyi-12. Imali engenayo entsha iboniswa ngombala oluhlaza okwesibhakabhaka kanti ukukhanselwa kubonakala ngombala obomvu.

![Ishadi Lokukhula kwe-MRR](/img/admin/um-dashboard-mrr.png)

**Ukusekelwa Kwebanga Lezinsuku:** Hlunga idatha ye-dashboard ngesikhathi ukuze ugxile kulwazi oludingayo. Ibanga lezinsuku lilawula okuboniswa kumawijethi angezansi.

![Ibanga lezinsuku nezibalo ze-dashboard](/img/admin/um-dashboard-date-range.png)

Amawijethi e-dashboard afaka:

- **Imali Engenayo** — Isamba semali engenayo saleso sikhathi, ne-MRR nokuhlukaniswa ngomkhiqizo ngamunye
- **Ubulungu Obusha** — Bangaki ubulungu obusha obudalwe esikhathini esikhethiwe
- **Ukubhalisa Ngamazwe** — Ukuhlukaniswa kokubhalisa ngokwendawo
- **Ukubhalisa Ngamafomu** — Bangaki amakhasimende abhalise ngefomu ngalinye le-checkout uma kuqhathaniswa nalabo abengezwe ngabaphathi
- **Amasayithi Avakashelwe Kakhulu** — Amasayithi avakashelwe kakhulu kunethiwekhi yakho (kudinga ukuvula ukubalwa kokuvakasha ku-**Ultimate Multisite → Izilungiselelo → Amasayithi → Vula Umkhawulo Nokubalwa Kokuvakasha**)

![Izibalo ze-Dashboard](/img/admin/um-dashboard-stats.png)
