# Ukuphatha Ubulungu {#managing-memberships}

Ku-Ultimate Multisite ungaphatha ubulungu futhi ufinyelele kukho konke okubhaliselwe kunethiwekhi yakho. Buyekeza futhi uhlele imininingwane yokubhaliselwe efana namapulani nemikhiqizo, inani lamasayithi, inani lokukhokha nobuningi bokukhokha, umlando wezinkokhelo, nezitembu zesikhathi.

Ukuze ufinyelele leli khasi, iya kuthebhu ethi **Ubulungu** kumenyu ye-Ultimate Multisite.

![Ikhasi lohlu lobulungu](/img/admin/memberships-list.png)

Ekhasini Lobulungu ungabona uhlu lwamakhasimende akho nemikhiqizo ehlotshaniswa nama-akhawunti awo. Ungahlunga ubulungu ngesimo: kuyasebenza, kulindile, kumisiwe okwesikhashana, kuphelelwe yisikhathi, noma kukhanseliwe.

Ubulungu bamahhala buthathwa njengobulungu **bempilo yonke**. Abutholi usuku lokuphelelwa yisikhathi futhi ngeke buphelelwe yisikhathi ngokuzenzakalelayo. Uma ipulani yamahhala kufanele iphele ngemva kwesikhathi sokuzama noma isikhathi esimisiwe, sebenzisa umkhiqizo okhokhelwayo onesikhathi sokuzama, khansela ubulungu mathupha, noma uhambise ikhasimende kwelinye ipulani lapho ukufinyelela kwalo kufanele kushintshe.

Chofoza kubulungu ukuze ufinyelele ekhasini lapho ungahlela khona imininingwane yobulungu — ulwazi lokukhokha, imikhawulo yesayithi noma yokuvakasha, nokunye.

![Isixhumi esibonakalayo sokuhlela ubulungu](/img/admin/membership-edit.png)

Nansi ukubuka okuphelele kwekhasi lokuhlela ubulungu:

![Isixhumi esibonakalayo esiphelele sokuhlela ubulungu](/img/admin/membership-edit-full.png)

Ungakwazi futhi ukwengeza ubulungu obusha obuhlotshaniswa nabasebenzisi ababhaliswe ngaphambilini kunethiwekhi yakho. Chofoza **Engeza Ubulungu** bese ulungisa imininingwane.

![Iwindi le-modal lokwengeza Ubulungu](/img/admin/membership-add-modal.png)

Bese uchofoza **Dala Ubulungu**. Ubulungu obusha buzovela ohlwini lobulungu.
