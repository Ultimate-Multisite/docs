# Ukuvakasha ku-Admin Panel

Ultimate Multisite ihlose ukuba lula futhi izichaze ngokwayo ngangokunokwenzeka, kodwa ngethuluzi elinamandla kangaka kungasiza ukuba nokuvakasha okuqondisiwe. Ake sihambe emakhasini okuphatha ukuze sikusize ujwayele indawo.

## Dashboard

I-**Dashboard** ye-Ultimate Multisite ibonisa imibiko eyisisekelo nokuhlaziywa kwedatha okuhlanganisa imali engenayo, imisebenzi yamasayithi, ukukhula kobulungu, izibalo zezivakashi, nedatha yezindawo.

## Amafomu Okukhokha

**Amafomu Okukhokha** akunikeza ukuguquguquka kokudala amakhasi okubhalisa angokwezifiso. Ungenza ngokwezifiso izinkambu, ufake amakilasi nemibhalo yangokwezifiso, futhi ukhawulele ukufinyelela ngokusekelwe ezweni noma endaweni yesivakashi.

![Uhlu lwamaFomu Okukhokha](/img/admin/checkout-forms-list.png)

## Imikhiqizo

Isigaba se-**Mikhiqizo** yilapho udala khona amapulani, amaphakheji, nezinsizakalo ezahlukene zenethiwekhi yakho. Chaza amanani, izikhathi zokukhokhisa, nemikhawulo nama-quota ezingeni lomkhiqizo.

![Uhlu lwemikhiqizo](/img/admin/products-list.png)

## Ubulungu

Ikhasi le-**Ubulungu** libonisa konke okubhaliselwe kunethiwekhi yakho. Buyekeza futhi uhlele imininingwane yokubhaliselwe okuhlanganisa amapulani, imikhiqizo, amanani okukhokhisa nezikhathi, umlando wokukhokha, nezitembu zesikhathi.

![Uhlu lobulungu](/img/admin/memberships-list.png)

## Izinkokhelo

Ikhasi le-**Zinkokhelo** linikeza ukubuka okusheshayo komlando wezinkokhelo kuyo yonke inethiwekhi yakho, ngolwazi olunemininingwane lokuthengiselana oluhlanganisa imikhiqizo ethile namanani.

![Uhlu lwezinkokhelo](/img/admin/payments-list.png)

## Amakhasimende

Ikhasi le-**Makhasimende** libonisa wonke amalungu abhalisiwe kunethiwekhi yakho ngolwazi oluyisisekelo lomsebenzisi oluhlanganisa igama, ikheli le-imeyili, ukungena kokugcina, nesixhumanisi esithi "shintshela ku" sokungena kusayithi labo elingaphansi.

![Uhlu lwamakhasimende](/img/admin/customers-list.png)

## Amasayithi

Ikhasi le-**Masayithi** libala wonke amasayithi angaphansi kunethiwekhi yakho okuhlanganisa izifanekiso zamasayithi namasayithi aphethwe amakhasimende. Phatha kalula ama-domain amatanisiwe, imikhawulo nama-quota, izandiso namatimu, nezithonjana zamasayithi.

![Uhlu lwamasayithi](/img/admin/sites-list.png)

## Ama-Domain

Ikhasi le-**Domain** linikelwe kuma-domain angokwezifiso amataniswe namasayithi angaphansi. Njenge-super admin, ungangeza noma umatanise mathupha amagama ama-domain angokwezifiso futhi ubuke amarekhodi e-DNS namalogi.

![Uhlu lwama-domain](/img/admin/domains-list.png)

## Amakhodi Esephulelo

**Amakhodi Esephulelo** akuvumela ukuthi udale amakhodi ekhuphoni ukuze unikeze izaphulelo kubasebenzisi bakho bokugcina. Cacisa inani lesaphulelo futhi uqondise abasebenzisi noma amaqembu athile.

![Uhlu lwamaKhodi Esephulelo](/img/admin/discount-codes-list.png)

## Ukusakaza

**Ukusakaza** kuyithuluzi lokukhulula izimemezelo noma ukuthumela imiyalezo yangasese kubasebenzisi bamasayithi akho angaphansi. Qondisa amaqembu athile abasebenzisi ngokusekelwe kuplani noma umkhiqizo ababhalisele kuwo.

![Uhlu lokusakaza](/img/admin/broadcasts-list.png)

## Izilungiselelo

Ikhasi le-**Zilungiselelo** yilapho ulungisa khona i-Ultimate Multisite — izilungiselelo zokubhalisa, izinkokhelo, i-API nama-webhook, ukumataniswa kwama-domain, nokunye ukuhlanganiswa.

Izilungiselelo zesixhumi se-AI zibala amaqoqo abahlinzeki be-OAuth asekelwayo njengamanje: Anthropic Max, OpenAI ChatGPT/Codex, ne-Google AI Pro. Ikhadi ngalinye lomhlinzeki livumela ama-super admin ukuthi axhume ama-akhawunti, avuselele ama-akhawunti alondoloziwe, asuse ama-akhawunti nge-imeyili, futhi asebenzise i-OAuth fallback yesandla lapho indawo evinjelwe ivimba ukuqondisa kabusha kwesiphequluli. Izinketho zokusetha ze-Cursor Pro zisusiwe ku-admin panel.

Ama-akhawunti e-ChatGPT/Codex asekela ukusetshenziswa kwamathuluzi okuxhaswe yisixhumi lapho ukusebenza kuvumela amathuluzi, ngakho ukugeleza komsebenzi wokuphatha okuncike ekusebenzeni okusekelwa yisixhumi kungasebenzisa ukuziphatha kwamathuluzi e-Codex ngemva kokuthi i-akhawunti ye-OpenAI ixhunyiwe.

## Izehlakalo

Ikhasi le-**Zehlakalo** ligcina irekhodi lazo zonke izehlakalo namalogi kunethiwekhi yakho. Lilandelela imisebenzi efana nezinguquko zeplani, ukubhalisa, nokunye ukunyakaza — kuwusizo ekuqapheni i-multisite yakho.

![Uhlu lwezehlakalo](/img/admin/events-list.png)

## Webhooks

**Webhooks** ikuvumela ukuthi uhambise idatha kolunye uhlelo lokusebenza. Kuwusizo ekuthumeleni idatha kusuka ku-Ultimate Multisite kuya kumapulatifomu afana ne-Zapier.

![Uhlu lwama-webhook](/img/admin/webhooks-list.png)
