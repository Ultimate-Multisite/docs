# Izinhlobo Zemali Eziningi {#multi-currency}

Ultimate Multisite: Izinhlobo Zemali Eziningi yenza inethiwekhi yakho ikwazi ukwamukela izinkokhelo ngezinhlobo zemali eziningi. Iguqula amanani ngokuzenzakalelayo isebenzisa amazinga okushintshanisa abukhoma noma afakwe ngesandla, ikuvumela ukusetha amanani aqondile ngomkhiqizo ngamunye kunoma yiluphi uhlobo lwemali, futhi ithola uhlobo lwemali oluthandwa isivakashi ngasinye nge-geolocation — konke kuphathwa kusuka ku-Dashboard yomlawuli eyodwa.

## Izici Eziyinhloko {#key-features}

- **Ukuguqulwa kwamazinga okushintshanisa okuzenzakalelayo** — amanani aguqulwa kusuka ohlotsheni lwemali oluyisisekelo usebenzisa amazinga alandwe kumhlinzeki omkhethile
- **Ukweqa amanani ngomkhiqizo ngamunye** — setha amanani aqondile aqondene nohlobo lwemali kunoma yimuphi umkhiqizo ukuze ugweme ukuguqulwa okuzenzakalelayo
- **Inkambu yokukhetha uhlobo lwemali ekubhaliseni** — ivumela amakhasimende akhethe uhlobo lwemali aluthandayo ngesikhathi se-checkout/sokubhalisa
- **Abahlinzeki abaningi bamazinga okushintshanisa** — ECB, Frankfurter, Stripe, noma amazinga afakwa ngesandla ngokuphelele
- **Ukutholwa kohlobo lwemali okususelwe ku-geolocation** — ikhetha ngokuzenzakalelayo uhlobo lwemali oluhambisana nezwe lesivakashi
- **Dashboard yomlawuli** — phatha izinhlobo zemali ezisebenzayo, amazinga okushintshanisa, nezilungiselelo zabahlinzeki endaweni eyodwa

## Izimfuneko {#requirements}

- Ultimate Multisite 2.0.0 noma ngaphezulu
- Ukufakwa kwe-WordPress Multisite
- PHP 7.4 noma ngaphezulu

## Ukufaka {#installation}

1. Layisha amafayela e-plugin ku-`/wp-content/plugins/ultimate-multisite-multi-currency`
2. Vula i-plugin ngesikrini se-**Ama-Plugin** ku-WordPress (Vula Kunethiwekhi)
3. Zulazulela ku-**Ultimate Multisite → Izinhlobo Zemali Eziningi** ukuze uvule i-Dashboard yomlawuli
4. Engeza izinhlobo zemali ofuna ukuzamukela bese ulungisa umhlinzeki wakho wamazinga okushintshanisa

## Ukumisa {#configuration}

### Dashboard Yomlawuli {#admin-dashboard}

I-Dashboard yomlawuli ye-Izinhlobo Zemali Eziningi (**Ultimate Multisite → Izinhlobo Zemali Eziningi**) iyindawo emaphakathi yokuthi:

- Wengeze, uvule, noma uvale izinhlobo zemali
- Ukhethe umhlinzeki wamazinga okushintshanisa futhi uvuselele amazinga uma kudingeka
- Ubuyekeze amazinga amanje azo zonke izinhlobo zemali ezisebenzayo
- Usethe amazinga ngesandla lapho abahlinzeki abazenzakalelayo bengafaneleki

### Abahlinzeki Bamazinga Okushintshanisa {#exchange-rate-providers}

| Umhlinzeki | Incazelo |
|---|---|
| **ECB** | European Central Bank — amazinga amahhala ansuku zonke amanethiwekhi asuselwe ku-EUR |
| **Frankfurter** | I-API ye-ECB mirror enomthombo ovulekile — mahhala, ayidingi ukhiye we-API |
| **Stripe** | Isebenzisa amazinga e-FX abukhoma e-Stripe — idinga ukuhlanganiswa kwe-Stripe okusebenzayo |
| **Ngesandla** | Ufaka amazinga ngqo; akukho zingcingo ze-API zangaphandle |

Khetha umhlinzeki wakho ngaphansi kokuthi **Ultimate Multisite → Izinhlobo Zemali Eziningi → Izilungiselelo → Umhlinzeki Wamazinga Okushintshanisa**. Amazinga angavuselelwa ngesandla noma nini noma ahlelelwe ukubuyekezwa ngokuzenzakalelayo.

### Ukweqa Amanani Ngomkhiqizo Ngamunye {#per-product-pricing-overrides}

Lapho ukuguqulwa okuzenzakalelayo kunganembile ngokwanele, ungasetha inani eliqondile lohlobo lwemali oluthile kunoma yimuphi umkhiqizo:

1. Vula umhleli womkhiqizo (**Ultimate Multisite → Imikhiqizo → Hlela**)
2. Skrolela esigabeni esithi **Amanani Ohlobo Lwemali**
3. Faka inani eliqondile lohlobo lwemali ngalunye ofuna ukulweqa
4. Londoloza umkhiqizo

Imikhiqizo engenakho ukweqa isebenzisa inani eliguquliwe elisuka ohlotsheni lwemali oluyisisekelo.

### Inkambu Yokukhetha Uhlobo Lwemali Ekubhaliseni {#currency-selector-signup-field}

Inkambu yokukhetha uhlobo lwemali ingangezwa ekugelezeni kwakho kokubhalisa/kwe-checkout ukuze amakhasimende akhethe uhlobo lwemali aluthandayo ngesikhathi sokubhalisa:

1. Iya ku-**Ultimate Multisite → Izinhlobo Zemali Eziningi → Izilungiselelo**
2. Vula okuthi **Bonisa isikhethi sohlobo lwemali ekubhaliseni**
3. Ngokuzikhethela setha uhlobo lwemali oluzenzakalelayo noma uvumele i-geolocation inqume inani elikhethwe kusengaphambili

Uhlobo lwemali olukhethiwe lugcinwa nokubhaliselwe futhi lusetshenziselwa zonke izinkokhiso ezilandelayo.

### Ukutholwa Kohlobo Lwemali Okususelwe Ku-Geolocation {#geolocation-based-currency-detection}

Lapho ukutholwa kwe-geolocation kuvuliwe, i-addon imepha izwe lesivakashi nohlobo lwemali bese ilukhetha kusengaphambili enkambini yokukhetha uhlobo lwemali. Ukuze ukuvule:

1. Iya ku-**Ultimate Multisite → Izinhlobo Zemali Eziningi → Izilungiselelo**
2. Vula okuthi **Thola uhlobo lwemali endaweni yesivakashi**
3. Mepha amazwe nezinhlobo zemali kuthebula le-**Geolocation Mapping**

Uma kungekho mapping yezwe lesivakashi, kusetshenziswa uhlobo lwemali oluyisisekelo lwenethiwekhi njengokubuyela emuva.

## Imibuzo Evame Ukubuzwa {#frequently-asked-questions}

### Kwenzekani uma umhlinzeki wamazinga okushintshanisa engatholakali? {#what-happens-if-an-exchange-rate-provider-is-unavailable}

I-addon ibuyela kumazinga agcine ukulandwa ngempumelelo. Uma kungekho mazinga ake alandwa, ibuyela kumazinga afakwe ngesandla owamisile. Kunconywa ukumisa amazinga ngesandla njengokuvikela.

### Ngingasebenzisa abahlinzeki abahlukene ezinhlotsheni zemali ezahlukene? {#can-i-use-different-providers-for-different-currencies}

Cha — umhlinzeki oyedwa usebenza ngesikhathi esisodwa kuzo zonke izinhlobo zemali. Sebenzisa okuthi **Ngesandla** uma udinga ukulawula okuphelele kwamazinga ngamanye.

### Ingabe i-geolocation idinga i-plugin ehlukile? {#does-geolocation-require-a-separate-plugin}

Ayidingeki i-plugin yangaphandle ye-geolocation. I-addon isebenzisa ukutholwa kwe-IP-to-country okwakhelwe ngaphakathi kwe-WordPress. Ukuze kutholakale ukunemba okuphezulu, isevisi ye-geolocation ezinikele ingahlanganiswa ngama-filter hooks anikeziwe.

### Ingabe ukweqa ngomkhiqizo ngamunye kusetshenziswa ngaphambi noma ngemva kwentela? {#are-per-product-overrides-applied-before-or-after-tax}

Ukweqa ngomkhiqizo ngamunye kusetha inani langaphambi kwentela ohlotsheni lwemali okuhlosiwe. Intela ibalwa ngaphezu kwenani lokweqa kusetshenziswa ukumiswa kwakho kwentela okukhona.
