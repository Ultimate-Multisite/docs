# Umthumeli Wesayithi

## Okuqukethwe {#content}

Siyasiqonda isidingo sokukwazi ukuhambisa amasayithi angaphansi ngaphakathi kwenethiwekhi yakho aye ekufakweni kwesayithi elilodwa, noma ngokuphambene nalokho, uma ikhasimende lakho linquma ukuthutha iwebhusayithi yalo. Isengezo somthumeli nomngenisi wesayithi senza lo msebenzi ube lula. Leli thuluzi lithumela futhi lingenisa amasayithi lisebenzisa amafayela e-zip, angasebenza futhi njengesipele.

## Izici {#features}

* Thumela futhi ungenise amasayithi

* Isixhumi esibonakalayo okulula ukusisebenzisa

* Dala isipele sesayithi

## Imibhalo {#documentation}
**Indlela yokuthumela nokungenisa isayithi**

## Ukufaka {#installation}

1. Layisha **ultimate-multisite-site-exporter** enkombeni ye-**/wp-content/plugins/**
2. Sebenzisa isandiso ngemenyu ye-**Izandiso** ku-WordPress
3. Landela i-Wizard yesinyathelo ngesinyathelo ukuze usethe isandiso

## Umhlahlandlela Wokusebenzisa {#usage-guide}

Nge-Ultimate Multisite ungathumela isayithi lisuke kunethiwekhi yakho noma ungenise isayithi ekufakweni kwakho kwe-multisite noma kwesayithi elilodwa.

Ukuqala inqubo, iya ekhasini lethu lezingezo bese ufaka isengezo se-Ultimate Multisite: Umthumeli Nomngenisi Wesayithi.![Ikhasi lezingezo](/img/admin/addons-page.png)

![Ikhasi lezingezo elinenketho yokufaka](/img/admin/addons-page.png)

Iya ekhasini lakho Lezandiso bese uchofoza ukuze **Usebenzise Kunethiwekhi** isengezo osanda kusifaka.<!-- Isithombe-skrini asitholakali: Ikhasi le-WP Plugins elibonisa inkinobho ye-Network Activate yesengezo se-Site Exporter -->

Inketho ethi **Thumela & Ngenisa** ingatholakala kumenyu ye-Ultimate Multisite, kwesokunxele.

## Indlela yokuthumela isayithi {#how-to-export-a-site}

Ukuze uthumele isayithi lisuke kunethiwekhi yakho, iya ku-**Thumela & Ngenisa** kumenyu ye-Ultimate Multisite bese, kuthebhu ethi **Thumela Amasayithi**, uchofoza ukuze **Ukhiqize Ukuthunyelwa Kwesayithi Okusha**.

<!-- Isithombe-skrini asitholakali: Ithebhu ethi Export Sites enenkinobho ethi Generate new Site Export -->

Sesha isayithi ofuna ukulithumela bese ukhetha izinto ofuna ukuzifaka kufayela elithunyelwayo. Ungafaka amatimu, izandiso, nokulayishiwe. Uma isayithi lakho likhulu kakhulu, ungakhetha ukuqhuba inqubo yokuthumela ngemuva.<!-- Isithombe-skrini asitholakali: Izinketho zokuthumela isayithi ezinokusesha isayithi namabhokisi okuhlola okufaka amatimu/izandiso/okulayishiwe -->

Chofoza inkinobho ethi **Thumela Isayithi**. Isayithi lakho elithunyelwe lizotholakala ohlwini lwe-**Okuthunyelwe Okukhona**. Chofoza inkinobho ethi **Landa i-ZIP** ukuze ulande ifayela. Ama-ZIP athunyelwe afaka i-`index.php` eziziqalisayo kanye ne-`readme.txt`, ngakho angafakwa ku-host entsha ngaphandle kokufaka kuqala isandiso somngenisi esihlukile.<!-- Isithombe-skrini asitholakali: Uhlu lwe-Existing Exports olunenkinobho ethi Download ZIP -->

## Indlela yokuthumela yonke inethiwekhi {#how-to-export-the-whole-network}

Ukuze uthumele wonke amasayithi angaphansi ndawonye, iya ku-**Thumela & Ngenisa** kumenyu ye-Ultimate Multisite bese usebenzisa inketho ethi **Ukuthumela Inethiwekhi** esikrinini sokuthumela. I-archive ekhiqiziwe iqukethe wonke amasayithi angaphansi ku-ZIP eyodwa yokuthutha ama-host, ukwakha kabusha izindawo zokuhlola, noma izipele ezisezingeni lenethiwekhi.

Ukuthunyelwa kwenethiwekhi kungaba kukhulu. Kuqhube lapho ithrafikhi iphansi futhi uqinisekise ukuthi iseva inesitoreji esanele se-archive nanoma yikuphi okulayishiwe, izandiso, namatimu afakiwe.

## Indlela yokungenisa isayithi {#how-to-import-a-site}

Ukuze ungenise isayithi, iya kuthebhu ethi **Ngenisa Isayithi** bese ulayisha ifayela le-.zip lesayithi ofuna ukulingenisa. Ungashintsha i-URL iye kwentsha noma ugcine i-URL endala ngokushiya isikhala singenalutho. Futhi, ungamaka ibhokisi ukuze ususe ifayela le-.zip ekufakweni kwakho ngemva kokungenisa. Bese uchofoza ukuze **Uqale Ukungenisa**.

<!-- Isithombe-skrini asitholakali: Ithebhu ethi Import a Site enenkambu yokulayisha i-zip nenkinobho ethi Begin Import -->

Inqubo ingathatha isikhashana. Uzobona ukungenisa okulindile ohlwini olungezansi komngenisi.

<!-- Isithombe-skrini asitholakali: Ukungenisa okulindile ohlwini olungezansi komngenisi -->

Ungamisa inqubo ngokuchofoza **Khansela Ukungenisa**.

<!-- Isithombe-skrini asitholakali: Inkinobho ethi Cancel Import yokumisa inqubo yokungenisa -->

Ngemva kokuthi inqubo isiqediwe, isayithi lakho selivele litholakala kunethiwekhi yakho.

## Indlela yokungenisa isayithi ekufakweni okukodwa {#how-to-import-a-site-on-a-single-install}

<!-- Isithombe-skrini asitholakali: Ithebhu yomthumeli enesixhumanisi sokulanda umngenisi wesayithi elilodwa -->
