# Isenzo: wu_page_added

Vumela abathuthukisi be-plugin ukuthi baqhube izinto ezengeziwe lapho amakhasi ebhaliswa.

Ngokungafani ne-wu_page_load, esebenza kuphela lapho ikhasi elithile libukwa, le hook isebenza ngesikhathi sokubhalisa kuwo wonke amakhasi okuphatha anezelwa kusetshenziswa ikhodi ye-WP Ultimo.

## Amapharamitha

| Igama | Uhlobo | Incazelo |
|------|------|-------------|
| $page_id | `string\|\WP_Ultimo_Site_Exporter\Admin_Pages\Base_Admin_Page` | I-ID yaleli khasi. |
| $page_hook | `string\|\WP_Ultimo_Site_Exporter\Admin_Pages\Base_Admin_Page` | I-hook ekhiqizwe kuleli khasi. |
