# I-Changelog Yokuvumelanisa Okuqukethwe

### Inguqulo 1.1.0 - Ikhishwe ngo-2025-12-31 {#version-110---released-on-2025-12-31}
* Okusha: Ukusekelwa kokuvumelanisa zonke izinhlobo zokuthunyelwe (okuthunyelwe, amakhasi, izinhlobo zokuthunyelwe ezenziwe ngokwezifiso)
* Okusha: Ukuhlunga uhlobo lokuthunyelwe esinyathelweni sokukhetha okuqukethwe
* Okusha: Isinyathelo sokukhetha isayithi eliwumthombo - vumelanisa kunoma yiliphi isayithi, hhayi kuphela isayithi eliyinhloko
* Okusha: Ukukhethwa kusengaphambili kwamasayithi ezingane kumasayithi ezifanekiso
* Okusha: Ukukhishwa kwemidiya ku-post_content kuzo zonke izinhlobo zokuthunyelwe
* Okusha: Ukuvumelanisa ama-taxonomy (izigaba, amathegi, ama-taxonomy enziwe ngokwezifiso)
* Kushintshiwe: Kuqanjwe kabusha kusuka ku-"Elementor Template Sync" kuya ku-"Ukuvumelanisa Okuqukethwe"
* Kushintshiwe: Isinyathelo se-"Khetha Izifanekiso" siqanjwe kabusha saba "Khetha Idatha"
* Kuthuthukisiwe: Ikhodi eqondene ne-Elementor manje isebenza kuphela kuhlobo lokuthunyelwe lwe-elementor_library
