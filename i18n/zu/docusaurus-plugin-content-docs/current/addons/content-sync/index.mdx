# Ukuvumelanisa Okuqukethwe

Multisite Ultimate: Ukuvumelanisa Okuqukethwe

Lesi sengezo se-Multisite Ultimate sivumela abaphathi benethiwekhi ukuvumelanisa okuqukethwe kusuka kunoma yiliphi isayithi kuya kwamanye amasayithi enethiwekhi. Sisekela zonke izinhlobo zokuthunyelwe okuhlanganisa okuthunyelwe, amakhasi, izinhlobo zokuthunyelwe ezenziwe ngokwezifiso, namathempulethi e-Elementor. Lesi sengezo sihlanganisa isixhumi esibonakalayo sewizadi enezinyathelo eziningi sokukhetha okuqukethwe namasayithi okuqondiswe kuwo, ngokucubungula ngamaqoqo nempendulo yenqubekelaphambili ngesikhathi sangempela.

## Izici

* Ukukhethwa Kokuqukethwe: Igridi yamakhadi ebonakalayo enezithonjana nokuhlunga ngohlobo lokuthunyelwe
* Zonke Izinhlobo Zokuthunyelwe: Vumelanisa okuthunyelwe, amakhasi, imikhiqizo, amathempulethi e-Elementor, nanoma yiluphi uhlobo lokuthunyelwe olwenziwe ngokwezifiso
* Ukukhethwa Kwesayithi: Khetha ukuthi yimaphi amasayithi okumele okuqukethwe kuvunyelaniswe kuwo, ngokukhishwa kwesayithi lomthombo nokukhethwa kusengaphambili kwesayithi eliyingane
* Ukucubungula Ngamaqoqo: Kuvumelanisa amasayithi ayi-10 ngesikhathi ngezibuyekezo zenqubekelaphambili
* Ukuvumelanisa Imidiya: Kukopisha ngokuzenzakalelayo futhi kumephu kabusha amafayela emidiya (izithombe, amavidiyo) kusuka kokuqukethwe kokuthunyelwe kuya kumasayithi okuqondiswe kuwo
* Ukuvumelanisa I-taxonomy: Kugcina izigaba, amathegi, nama-taxonomy enziwe ngokwezifiso
* Ukusekelwa kwe-Elementor: Ukusekelwa okugcwele kwamathempulethi e-Elementor ngokukhiqizwa kabusha kwe-CSS nokumethwa kabusha kwe-ID yethempulethi
* Ukuxazulula Ukungqubuzana: Inketho yokufaka okuqukethwe okukhona esikhundleni noma ukukweqa
* Ukuphathwa Kwe-cache: Kusula ngokuzenzakalelayo i-cache ye-Elementor kumasayithi okuqondiswe kuwo ngemva kokuvumelanisa

## Izimfuneko

* WordPress 5.3 noma ngaphezulu
* PHP 7.4 noma ngaphezulu
* Isandiso se-Multisite Ultimate (siyasebenza)
* Isandiso se-Elementor (uyazikhethela, siyadingeka ekuvumelaniseni ithempulethi ye-Elementor)

## Ukufaka

1. Layisha 'ultimate-multisite-content-sync' kuhla lwemibhalo lwe-'/wp-content/plugins/'
2. Sebenzisa isandiso ngemenyu ethi 'Izandiso' ku-WordPress Network Admin yakho
3. Lungiselela izilungiselelo ku-Ultimate Multisite → Izilungiselelo → Ukuvumelanisa Okuqukethwe
4. Zulazula uye ku-Ultimate Multisite → Ukuvumelanisa Okuqukethwe ukuze uqale ukuvumelanisa
