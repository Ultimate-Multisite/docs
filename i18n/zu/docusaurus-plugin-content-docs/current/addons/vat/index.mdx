# VAT

Multisite Ultimate: I-VAT yaseYurophu

Yengeza izici namathuluzi engeziwe ekuphathweni kwezintela okuzenzakalelayo kwe-Multisite Ultimate ukuze kusekelwe kangcono i-VAT yaseYurophu.
Yengeza ngokuzenzakalelayo amazinga e-EU VAT ekhasini lamaZinga Entela kusuka

## Ukufakwa

1. Layisha 'ultimate-multisite-vat' kuhla lwemibhalo lwe-'/wp-content/plugins/'
2. Yenza isandiso sisebenze ngemenyu ethi 'Izandiso' ku-WordPress
3. Iya ku-Ultimate Multisite > Izilungiselelo > Izintela bese unika amandla i-EU Vat.
4. Iya uchofoze kuZinga Entela bese ubuyekeza amazinga e-EU VAT ngezansi bese ulondoloza amazinga entela.
5. Kuphelile!
