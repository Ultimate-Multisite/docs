# Irekhodi lezinguquko ze-VAT

Inguqulo 1.0.7 - Ikhishwe ngo-2026-02-03

* Ukulungisa: Izinombolo ze-VAT zaseJalimane (DE) bezihluleka ukuqinisekiswa ngenxa yamaphutha e-VIES SOAP service MS_MAX_CONCURRENT_REQ. Kushintshiwe kusuka ku-SOAP API endala kuya ku-EU VIES REST API esemthethweni ukuze ukuqinisekiswa kuthembeke kakhudlwana.
* Ukulungisa: Izinombolo ze-VAT manje ziyamukelwa lapho i-VIES service ingatholakali okwesikhashana, esikhundleni sokwenqatshwa ngokungafanele. Kufakwa i-log entry ukuze kubuyekezwe kamuva.
* Ukulungisa: Inkambu ye-VAT/Tax ID manje iyabonakala kuwo wonke amazwe, hhayi kumalungu e-EU kuphela. Ama-tax ID angewona awe-EU (isb. izinombolo ze-Swiss CHE) agcinwa ukuze aboniswe ku-invoice ngaphandle kokuqinisekiswa kwe-VIES.
* Ukulungisa: Ukubheka isiqalo se-VAT se-Greece (GR), Monaco (MC), ne-Isle of Man (IM) bekuphukile ngenxa yomqondo wokubheka i-array ongalungile.
* Ukuthuthukisa: Kususiwe i-dev dependency ye-wp-cli-bundle ebingadingekile engabangela amaphutha abulalayo uma isetshenziswa kanye ne-addon ye-Site Exporter.
* Ukuthuthukisa: Kungezwe isethi ebanzi yokuhlolwa kwama-unit (izivivinyo ezingu-53).

Inguqulo 1.0.6 - Ikhishwe ngo-2026-01-25

* Ukulungisa: Ukuthengiselana kwasekhaya kwe-B2B manje kukhokhisa i-VAT ngendlela efanele. Ngokwemithetho ye-EU VAT, i-reverse charge isebenza kuphela ekuthengiselaneni kwe-B2B okweqa imingcele, hhayi lapho izwe lekhasimende lifana nezwe lenkampani.

Inguqulo 1.0.5 - Ikhishwe ngo-2026-01-22

* Ukulungisa: I-addon ibingalayishi noma ingasebenzi kahle.
* Ushintsho: Kushintshwe umthombo wedatha yamazinga e-VAT kusuka ku-euvatrates.com kuya ku-repository ye-ibericode/vat-rates ukuze kutholakale idatha ethembeke kakhudlwana futhi enakekelwa ngenkuthalo.
* Ukulungisa: Kulungiswe iphutha lokuthayipha lenani le-option elithi super_reduced_rates.
* Ukuthuthukisa: Kungezwe ukuhlola kwe-null ekuphatheni ikhodi yezwe ukuze kugwenywe amaphutha.
* Ushintsho: Kususiwe amafayela okuhumusha afakiwe ukuze kusetshenziswe izibuyekezo ezizenzakalelayo nge-Traduttore.
