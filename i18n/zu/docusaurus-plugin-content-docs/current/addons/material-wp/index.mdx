# Material WP

Isixhumi esibonakalayo sokuphatha se-WordPress besilokhu sifana isikhathi eside. Kodwa wake wazibuza ukuthi inguqulo ye-Material Design yesixhumi esibonakalayo sokuphatha ingabukeka kanjani? 

Material WP iyitimu yokuphatha eyakhiwe ukuze ikunike lokho kubukeka nomuzwa, ikusiza ukuthi unikeze amakhasimende akho i-Dashboard eyenziwe ngezifiso kakhulu.

## Izici

- Ukusekelwa kwe-CSS yangokwezifiso
- Izilungiselelo zesitayela ezilula
- Ukuhlela kabusha imenyu
- Ukungenisa/ukuthekelisa okulula

## Imibhalo

**Ukwenza Ngokwezifiso Iphaneli Yakho Yokuphatha nge-Material WP**

## Ukufaka

1. Layisha i-'material-wp' kuhla lwemibhalo oluthi '/wp-content/plugins/'
2. Yenza isandiso sisebenze ngemenyu ethi 'Izandiso' ku-WordPress
