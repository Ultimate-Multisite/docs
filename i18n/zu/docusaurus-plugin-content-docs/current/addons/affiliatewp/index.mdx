# AffiliateWP

Multisite Ultimate: Ukuhlanganiswa kwe-AffiliateWP

Sebenzisa i-AffiliateWP enamandla ukuze ukhulise isisekelo samakhasimende se-Ultimo Network yakho!

## Ukufaka

1. Layisha 'ultimate-multisite-git ' kuhla lwemibhalo '/wp-content/plugins/'
2. Vula i-plugin ngemenyu ethi 'Ama-plugin' ku-WordPress
3. Landela i-Wizard isinyathelo ngesinyathelo ukuze ulungiselele i-plugin

## Umhlahlandlela Wokusetshenziswa

[I-AffiliateWP](https://affiliatewp.com/) isiphenduke isixazululo esithandwayo lapho kwakhiwa uhlelo lwabaxhumanisi lwebhizinisi lakho eliku-inthanethi. Ultimate Multisite: AffiliateWP iyibhuloho elizokwenza ukwazi ukwakha okufanayo kunethiwekhi yakho ekhokhelwayo.

## Ukuqalisa

Okokuqala, **kudingeka ube** ne-[**AffiliateWP**](https://affiliatewp.com/) **esebenzayo**! Ngeke ukwazi ukusebenzisa isengezo sethu uma ungenayo **i-plugin ehlanganiswa nayo**. Uma ungahleleli ukuvumela amakhasimende akho ayisebenzise kumasayithi awo futhi, i-AffiliateWP kufanele isebenze kuphela kusayithi lakho eliyinhloko.

Futhi, uzodinga kokubili **Ultimate Multisite** kanye ne-**Ultimate Multisite: AffiliateWP** **kuvuliwe** **kunethiwekhi yonke**.

## Ukwengeza i-Ultimate Multisite Njengokuhlanganiswa kwe-AffiliateWP

Kusayithi lakho eliyinhloko, iya ku-Affiliates Settings, bese kuthebhu ethi Integrations, maka i-Ultimate Multisite. Lokhu kuzotshela [i-AffiliateWP](https://affiliatewp.com/) ukuthi iqale ukulandelela ukuvakasha nokuguqulwa okuvela enqubweni yokubhalisa ye-Ultimate Multisite.

<!-- Isithombe-skrini asitholakali: ithebhu yezilungiselelo ze-AffiliateWP Integrations enebhokisi lokumaka le-Ultimate Multisite -->

_Maka ibhokisi le-Ultimate Multisite ngezansi bese ulondoloza_

Yilokho nje okudingekayo. Ukuvakasha okuvela kuzixhumanisi zokudlulisela kuzolandelwa, futhi lapho lowo msebenzisi kamuva enza inkokhelo yakhe yokuqala, ukuguqulwa kuzodalwa bese ikhomishini ibekelwa eceleni umxhumanisi obhekele lokho kuguqulwa.

Kodwa kuzolandelwa kuphela inkokhelo yokuqala etholwe kumsebenzisi osanda kudalwa. Ukuze unike amandla izinkokhelo zokudlulisela eziphindaphindekayo, uzodinga i-[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## Ukunika Amandla Ukudluliselwa Okuphindaphindekayo

Ngemva kokufaka i-[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/), iya kuthebhu ethi Affiliates Settings Recurring Referrals, bese unika amandla ukusetshenziswa kokudluliselwa okuphindaphindekayo.

<!-- Isithombe-skrini asitholakali: izilungiselelo ze-AffiliateWP Recurring Referrals ezinokushintsha okuthi Enable Recurring Referrals -->

_Kulula kakhulu: Nika amandla i-Recurring Referrals futhi konke sekumi ngomumo!_

Futhi, yilokho nje okudingekayo!

Manje, njalo uma inkokhelo entsha itholwa ku-account edluliselwe, umxhumanisi obhekele lokho uzothola isabelo sakhe ngokuzenzakalelayo!
