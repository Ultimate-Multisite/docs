# Fluent Forms {#fluent-forms}

Lesi sengezo senza kube nokwenzeka ukwengeza imikhawulo ku-Fluent Forms ku-Multisite Ultimate emikhiqizweni, ebulungwini noma kumasayithi.
I-Fluent Forms isebenzisa amathebula enziwe ngokwezifiso esikhundleni sezinhlobo zokuthunyelwe ezenziwe ngokwezifiso, ngakho izinketho ezijwayelekile zokubeka imikhawulo azisebenzi ne-Fluent Forms.

## Izici {#features}

- Engeza imikhawulo ye-Fluent Forms noma i-Conversational Fluent Forms
- Amakhasimende athola umlayezo onobungane owawakhuthaza ukuthi athuthukise uhlelo lwawo lapho efinyelela umkhawulo
