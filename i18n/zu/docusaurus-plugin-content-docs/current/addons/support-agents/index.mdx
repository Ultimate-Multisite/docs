# Ama-ejenti Okusekela {#support-agents}

Ukuphatha inethiwekhi yonke uwedwa akuwona umsebenzi olula. Njengoba inethiwekhi yakho iqala ukukhula, uzoqala ukubona isidingo esikhulayo sokusekelwa. Kuyoba umqondo omuhle ukuletha abantu abazokusiza unciphise umthwalo wakho womsebenzi. Nokho, ukunikeza ukufinyelela/imvume egcwele kuyingozi futhi kungase kubeke inethiwekhi yakho engcupheni. Yilapho le add-on ye-ejenti yokusekela ingena khona.

Le add-on ikuvumela ukuthi udale uhlobo olusha lomsebenzisi we-WordPress olubizwa ngokuthi i-ejenti yokusekela, futhi ikunikeza ukulawula okuphelele phezu kwezimvume lowo msebenzisi anazo. Ukunikeza imvume kulula njengokuvula inkinobho yokushintsha, nokuyivala uma unquma ukuyisusa.

## Izici {#features}

- Uhlu olunezinto ngabanye lwezimvume ze-WordPress ne-Ultimate Multisite
- Nika amandla futhi ukhubaze ukufinyelela kalula
- Dala inani elingenamkhawulo lama-akhawunti e-ejenti

## Ukufaka {#installation}

1. Layisha 'ultimate-multisite-support-agents' kuhla lwemibhalo lwe-'/wp-content/plugins/'
2. Yenza i-plugin isebenze ngemenyu ethi 'Ama-plugin' ku-WordPress
3. Landela i-Wizard isinyathelo ngesinyathelo ukuze usethe i-plugin
