# Plugin & Theme Manager {#plugin--theme-manager}

Multisite Ultimate: Plugin and Theme Manager

Hlela noma ufihle ulwazi lwe-meta (isihloko, umbhali, isithonjana nencazelo) lwama-plugin namatimu afakiwe. Dala izigaba zama-plugin namatimu ukuze uvumele abasebenzisi bakho bawahlunge futhi wengeze ikhasi elihle lama-Plugin elenziwe ngokwezifiso kuphaneli yabasebenzisi bakho.

Ukuze usebenzise i-plugin, ungavele uye emakhasini akho ama-Plugin noma ama-Theme ku-network admin yakho bese uchofoza ku-"Hlela ulwazi lwe-Plugin/Theme".

## Ukufaka {#installation}

1. Layisha 'ultimate-multisite-plugin-and-theme-manager' enkombandleleni ethi '/wp-content/plugins/'
2. Yenza i-plugin isebenze ngemenyu ethi 'Ama-plugin' ku-WordPress Network Admin yakho

## Umhlahlandlela Wokusebenzisa {#usage-guide}

Ultimate Multisite: Plugin and Theme Manager iyi-add-on ye-Ultimate Multisite **premium** enamandla ekunikeza ukulawula okuphelele ngendlela abasebenzisi benethiwekhi yakho ababona futhi basebenzisane ngayo nezinketho zama-plugin namatimu kunethiwekhi yakho.

Le add-on inezinjongo ezintathu:

  1. Ikuvumela ukuthi ushintshe ngokuphelele i-metadata yawo womabili ama-plugin namatimu. Lokho kufaka igama, incazelo, igama lombhali, ngisho nezithonjana!

  2. Ikuvumela ukuthi uhlukanise ama-plugin namatimu ngezigaba, ukuze uvumele abasebenzisi bakho bawahlele kalula.

  3. Ikunikeza inketho yokushintsha ikhasi elidala, eliyisicefe lama-plugin ufake ikhasi elisha elifana nelamatimu elinokuhlela ngezigaba.

Ngemva kokuyilanda nokuyifaka, ithebhu entsha - _Izilungiselelo ze-add-on_ \- izovela ekhasini le-Ultimate Multisite Settings.
