# Umlando Wezinguquko Wezinhlelo Ezikalwayo

Inguqulo 1.1.0 - Ikhishwe ngo-2026-05-05
- Okusha: Ukukhokhisa ama-AI token kuma-subsite e-multisite — landela futhi ukhokhise ukusetshenziswa kwama-AI token kuwo wonke amasayithi amakhasimende ngamareyithi alungisekayo ngetoken ngayinye
- Okusha: Ukuphoqelelwa kwe-connector kwakhiwe kabusha ngokutholwa kwemikhawulo okuguquguqukayo kanye nokubhala okudluliswayo, kuqinisekisa ukunemba kwesikhathi sangempela kuwo wonke ama-connector
- Ukulungisa: Ukuthuthukiswa kwamathebula e-database manje kuqondaniswe kahle nezincazelo ze-schema ze-BerlinDB, kuvimbela ukwehluleka kokuthuthukisa ekufakweni okusha
- Ukulungisa: Ama-callback okuthuthukisa i-database aguqulelwe kufomethi efanele, kuxazulula ukwehluleka kokuthuthukisa okungabonakali
- Ukulungisa: Amanani angamaqhezu manje ayamukelwa enkambini yokufaka ye-AI Usage Overage Markup
- Ukulungisa: Kuxazululwe amaphutha abulalayo nezinkinga zokuqaliswa kabili lapho i-plugin ilayishwa
- Kuthuthukisiwe: Kwengezwe i-check-env npm script ukuze izindawo zonjiniyela zizilungiselele ngokwazo ekusebenzeni kokuqala
