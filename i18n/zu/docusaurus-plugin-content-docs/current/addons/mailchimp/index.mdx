# Mailchimp

Multisite Ultimate: Ukuhlanganiswa kwe-Mailchimp

Hlanganisa inethiwekhi yakho ye-Multisite Ultimate ne-Mailchimp yakho. Ungaxhumanisa amapulani athile nezinhlu ze-Mailchimp futhi wengeze ngokuzenzakalelayo ababhalisile kulezo zinhlu lapho bebhalisa.

## Ukufaka {#installation}

1. Layisha 'ultimate-multisite-mailchimp' enkombandleleni ethi '/wp-content/plugins/'
2. Vula i-plugin ngemenyu ethi 'Ama-plugin' ku-WordPress Network Admin yakho

## Umhlahlandlela Wokusebenzisa {#usage-guide}

Ukuvumelanisa ama-imeyili ababhalisile nezinhlu zakho ze-MailChimp kulula ngempela nge-Ultimate Multisite: Isengezo Sokuhlanganisa i-MailChimp. Lesi sifundo sihlose ukukusiza ukuthi usethe konke!

## Ukuthola ukhiye we-API {#getting-an-api-key}

Ngaphambi kokuthi siqale ukungena kuzilungiselelo ezahlukene ezitholakalayo, sidinga ukhiye we-MailChimp API. Lokhu kuzovumela i-Ultimate Multisite ukuthi ixhumane ne-MailChimp API ukuze ithole izinhlu, amaqembu, futhi yengeze amakheli e-imeyili kulezo zinhlu namaqembu lapho abasebenzisi bebhalisa.

Landela lesi sifundo ukuze uthole ukhiye wakho we-MailChimp API: [Mayelana Nokhiye be-API | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

<!-- Screenshot unavailable: MailChimp API key settings in Ultimate Multisite Add-on Settings -->

_Engeza ukhiye wakho we-MailChimp API ku-_ _**Izilungiselelo ze-Ultimate Multisite - > Izilungiselelo Zezengezo -> MailChimp**_

## Imodi Yokuhlanganisa {#integration-mode}

Isengezo Sokuhlanganisa i-MailChimp sinikeza “amodi okuhlanganisa” amabili, okusho ukuthi kunezindlela ezimbili ezahlukene ongasebenzisa ngazo isengezo ukuvumelanisa amakheli e-imeyili.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_Amamodi amabili okuhlanganisa ukuze kube nokuguquguquka okukhulu!_

### Imodi 1: Izinhlu Eziningi {#mode-1-multiple-lists}

Imodi ezenzakalelayo isebenzisa **Izinhlu Eziningi**. Le modi izokuvumela ukuthi ukhethe izinhlu eziningi ze-MailChimp kupulani ngalinye. Lena inketho engcono kakhulu uma uhlukanisa abasebenzisi bakho babe yizinhlu ezahlukene.

Isibonelo, ake sithi unohlu oluhlukene olu-3 ku-akhawunti yakho ye-MailChimp: **Incwajana yezindaba** , **Pulani A** , kanye no-**Pulani B**. Ufuna bonke abasebenzisi, kungakhathaliseki ipulani labo, nabo bafakwe ohlwini lweNcwajana yezindaba, ukuze bangaphuthelwa izibuyekezo zakho.

Ukuze ufinyelele lokho, ungasebenzisa imodi Yezinhlu Eziningi ukwengeza zombili izinhlu **Incwajana yezindaba** kanye no-**Pulani A** Ezilungiselelweni ze-MailChimp zePulani A, kanye no-**Incwajana yezindaba** no-**Pulani B** Ezilungiselelweni ze-MailChimp zePulani B.

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_Khetha izinhlu ezizenzakalelayo zama-akhawunti amasha nezama-akhawunti akhanseliwe_

### Imodi 2: Uhlu Olulodwa + Amaqembu Amaningi {#mode-2-single-list--multiple-groups}

I-MailChimp ikuvumela ukuthi uhlukanise uhlu lwakho lube amaqembu amancane abasebenzisi abizwa ngokuthi **Amaqembu**. Lokhu kukuvumela ukuthi ube nokulawula okwengeziwe njengoba ungathumela umkhankaso kusethi ethile yababhalisile bakho ngaphandle kokuthi udale amakhophi amaningi omkhankaso ofanayo ku-MailChimp (okwenza kube nzima ukuhlaziya izibalo zalowo mkhankaso kamuva).

Isibonelo, ungaba nohlu olulodwa olubizwa ngokuthi **Ababhalisile Benethiwekhi Yami,** futhi ngaphakathi kwalo kube namaqembu e-**Pulani A** kanye ne-**Pulani B**.

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_Khetha izinhlu namaqembu okuzenzakalelayo, uma usebenzisa imodi yokuhlanganisa ethi “uhlu olulodwa + amaqembu amaningi”_

## Izinhlu Namaqembu Okuzenzakalelayo {#default-lists-and-groups}

Ngemva kokukhetha ukuthi iyiphi imodi yokuhlanganisa enengqondo kakhulu esimweni sakho sokusetshenziswa esithile, sekuyisikhathi sokukhetha izinhlu ezizenzakalelayo (namaqembu, kuye ngemodi). Uhlu oluzenzakalelayo luzosetshenziswa lapho ipulani elithile lingenalo Uhlu oluthile olukhethiwe.

Ungakwazi futhi ukukhetha uhlu oluzenzakalelayo lwama-akhawunti akhanseliwe. Noma nini lapho umsebenzisi eqeda ukubhalisa kwakhe futhi esusa i-akhawunti yakhe kunethiwekhi yakho, ikheli lakhe le-imeyili lizothuthelwa ohlwini olusethelwe ama-akhawunti akhanseliwe.

## Ukuvumela Ukubhalisa Okuphindwe Kabili {#enabling-double-subscription}

Ekhasini elifanayo lezilungiselelo, uzokwazi ukuvumela **Ukubhalisa Okuphindwe Kabili**. Le nketho izokwengeza umsebenzisi kukho kokubili uhlu oluzenzakalelayo olukhethiwe KANYE nohlu olukhethwe kupulani lalowo msebenzisi. Uma le nketho ingavunyelwe, umsebenzisi uzokwengezwa kuphela ohlwini olukhethwe kupulani lomsebenzisi. Uma kungekho okusethiwe kulelo pulani elithile, umsebenzisi uzokwengezwa ohlwini oluzenzakalelayo olukhethiwe.

<!-- Screenshot unavailable: MailChimp double subscription toggle setting -->

_Ungenza abasebenzisi bakho babhalisele kokubili uhlu oluzenzakalelayo nohlu lwepulani lapho bebhalisa_

## Ukukhetha Izinhlu Namaqembu epulani ngalinye {#selecting-lists-and-groups-for-each-of-the-plans}

Yiya **Kumapulani** akho bese ukhetha ipulani ofuna ukulihlela. Qaphela ukuthi ku-meta-box ethi **Izinketho Ezithuthukile**, ithebhu entsha ebizwa ngokuthi MailChimp izotholakala.

Kuleyo thebhu, uzokwazi ukukhetha izinhlu eziningi noma uhlu olulodwa namaqembu amaningi (kuye ngemodi yokuhlanganisa) kwalelo pulani.

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_Khetha Izinhlu Namaqembu ofuna ukuwasebenzisa kupulani ngalinye lakho_

Lapho umsebenzisi omusha ebhalisela lelo pulani, ikheli lakhe le-imeyili lizokwengezwa ezinhlelweni nasemaqenjini akhethiwe!
