# Umlando wezinguquko

## 1.18.0 — Ikhishwe ngo-2026-06-29

### Okusha

- **Amathuluzi e-Google Calendar** — funda amakhalenda nemicimbi elungiselelwe ukuze kube nama-automation aqaphela uhlelo
- **Ukumatanisa othintwayo nabasizi babahambele** — fanisa ababambiqhaza bekhalenda nabasebenzisi besayithi kanye nothintwayo
- **Amasango okuvunywa ngumuntu namarekhodi ezikhumbuzi** — misa ama-automation ukuze abuyekezwe futhi ugweme izaziso eziphindaphindiwe
- **Umhlinzeki we-TextBee SMS** — thumela izaziso zemiyalezo yombhalo ezilungiselelwe nge-TextBee
- **Iphakheji elihambisanayo elithuthukile** — engeza amathuluzi esistimu yamafayela yonjiniyela abathembekile, isizindalwazi, WP-CLI, umthumeli we-REST, umakhi we-plugin, isithombe sesimo se-git, ukuphathwa kwabasebenzisi, namathuluzi okuhlola ukusebenza asatshalaliswa ngokwehlukana nokwakhiwa kwe-WordPress.org

### Kuthuthukisiwe

- **Ukusethwa kwesevisi ye-Managed Superdav** — engeza ama-endpoint esevisi ebanjwe nokuhlinzekwa kokuxhumeka okuzenzakalelayo kumasayithi asekelwayo
- **Ukupakishwa kokukhishwa** — yakha ama-ZIP ahlukene e-core ne-Advanced, kushicilele kokubili ku-GitHub, bese uthumela kuphela iphakheji ye-core ku-WordPress.org

### Kulungisiwe

- **Ukuthembeka kwezicelo ze-AI** — thuthukisa ukukhethwa kwemodeli, izikhathi zokuphelelwa kwezicelo, imodeli yokusetha ezenzakalelayo, ukuphathwa kombhalo wokucabangisisa, nesiqondiso sokuzama futhi kumakholi wamathuluzi angavumelekile
- **Ukuqiniswa kwekhalenda nezikhumbuzi** — qinisa amathokheni e-Google Calendar nokususwa kwezikhumbuzi eziphindaphindiwe
- **Ukuqalisa nokubuyiselwa kokuvunywa** — lungisa ukwethulwa kokuqalisa kwe-frontend nokubuyiselwa kwamakhono aqinisekisiwe
- **Izinkinga zokubuyekezwa kokupakishwa kwe-WordPress.org** — phendula impendulo yokubuyekezwa kokupakishwa yokukhishwa kwe-core

## 1.16.0 — Ikhishwe ngo-2026-05-20

### Okusha

- **Ikhono lokukhiqiza i-Logo SVG** — Theme Builder manje ingakhiqiza futhi ifake ama-SVG elogo angokwezifiso ngokuhlanzwa okuphephile ku-namespace
- **Ukulayishwa kwesithombe engxoxweni yokuthola ulwazi** — ingxoxo yokuthola ulwazi ye-Theme Builder manje ifaka isinyathelo sokulayisha isithombe ukuze kube nomongo womklamo ocebile
- **Ikhono lokuqinisekisa ukuqhathanisa kwephalethi** — hlola amapheya emibala ukuze ahambisane ne-WCAG ngaphambi kokuwasebenzisa ku-theme
- **Amamenyu ezokwamukela izivakashi** — Theme Builder manje ingakhiqiza amakhasi emenyu yokudla neziphuzo ahlelwe kahle amabhizinisi ezokwamukela izivakashi
- **Ukunikezwa kokubuka kuqala kudeskithophu nakumaselula** — buka kuqala umklamo wakho kumadivayisi edeskithophu naweselula ngesikhathi sokukhetha indlela yomklamo
- **Ipharamitha yelebula yokuzulazula** — ikhono le-Create Menu manje lisekela i-`navigation_label` ehlukile ehlukene nesihloko sekhasi
- **Ukutholakala kwamathuluzi e-Tier 1** — i-sd-ai-agent/site-scrape manje iyithuluzi le-Tier 1 elitholakala ngokuzenzakalelayo ku-Theme Builder

### Kulungisiwe

- **Inqolobane ye-AI Client** — manje isekelwa ama-transient ukuze kuqhubeke phakathi kwezicelo, ivimbele ukulahleka kwedatha emisebenzini ye-agent ethatha isikhathi eside
- **Izixhumanisi zezenzo zomugqa we-plugin** — zilungisiwe futhi zaqanjwa kabusha ukuze kucace

## 1.10.0 — Ikhishwe ngo-2026-05-05

### Okusha

- **Usesho lwe-inthanethi lwe-Tavily** — engeza i-Tavily njengomhlinzeki wosesho ukuze kube nemiphumela yosesho lwe-inthanethi ecebile kanye ne-Brave Search
- **Amakhono akhelwe ngaphakathi aqaphela i-theme** — imihlahlandlela yamakhono ye-Block Themes, Classic Themes, Kadence Blocks, ne-Kadence Theme manje iza ne-plugin
- **Ikhono lefomu lokuxhumana lomakhi wesayithi** — engeza ifomu lokuxhumana kunoma yiliphi ikhasi ngokuqondile kusixhumi sengxoxo

### Kuthuthukisiwe

- **Ukuhlanganiswa kwe-WooCommerce kuhlelwe kabusha** — manje kusebenzisa ama-API emvelo e-WooCommerce ukuze kube nokuthembeka nokuhambisana okungcono
- **Uhlu lwabahlinzeki luzivuselela ngokuzenzakalelayo** — lapho noma iyiphi i-plugin yenziwa isebenze noma icishwa

### Kulungisiwe

- **Ikhono le-navigate-to** — kulungiswe umjikelezo wokulayisha kabusha ongapheli kwamanye amakhasi e-admin
- **Ikhono le-list-posts** — manje lixazulula kahle amagama ezigaba nawamathegi abe ama-slug
- **Imiyalo ye-WP-CLI** — kubuyiselwe ama-alias e-namespace abengekho ngemva kokuhlelwa kabusha kwangaphambilini
- **I-automation yomcimbi** — iphatha kahle amasayithi lapho amathebula e-automation engakakhiwa
- **Ikhono le-memory-save** — manje lisebenzisa isiqalo se-namespace esilungile kumakhi wemiyalelo yesistimu
- **Imiphumela yamathuluzi e-scalar** — manje isongwa kahle ngaphambi kokubuyiselwa ku-AI
- **Izibalo zokusetshenziswa** — manje ziphatha kahle ifomethi endala yokhiye wekhono lapho kuthuthukiswa kusuka ezinguqulweni ezindala
