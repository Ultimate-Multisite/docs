# Superdav AI Agent {#superdav-ai-agent}

Superdav AI Agent ingumsizi we-AI we-WordPress othuthukile owandisa i-Gratis AI Agent ngamakhono aqonda itimu, amakhono athuthukisiwe okusesha ku-inthanethi, izici zokwakha isayithi, nokuhlanganiswa okuzenzakalelayo ongakukhetha. Inguqulo 1.18.0 yengeza amathuluzi okufunda e-Google Calendar, ukufanisa oxhumana nabo nabahambele umcimbi, amasango okugunyazwa umuntu, ukuvimbela ukuphindaphindwa kwezikhumbuzi, izaziso ze-SMS ze-TextBee, ukuhlinzekwa kwesevisi ye-Superdav ephethwe, kanye nephakheji elihambisana nayo le-Advanced elisatshalaliswa ngokwehlukana lemisebenzi yabathuthukisi abathembekile.

## Izici Eziyinhloko {#key-features}

- **Amakhono aqonda itimu** — alungisa ngokuzenzakalelayo isiqondiso namakhono ngokusekelwe kutimu yakho ye-WordPress esebenzayo nama-plugin afakiwe
- **Abahlinzeki abaningi bokusesha ku-inthanethi** — khetha phakathi kwe-Brave Search ne-Tavily ukuze uthole imiphumela yokusesha ecebile namakhono ocwaningo athuthukile
- **Amafomu okuxhumana omakhi wesayithi** — dala amafomu okuxhumana asebenza ngokugcwele ngokuqondile kusixhumi sengxoxo usebenzisa i-plugin yefomu oyifakile
- **Umakhi wesayithi othuthukisiwe** — izinhlelo zokwakha isayithi zezigaba eziningi ezithuthukisiwe ezinokutholwa okungcono kwama-plugin nokubuyiselwa emaphutheni
- **Amakhono etimu akhelwe ngaphakathi** — isiqondiso esikhethekile sama-Block Themes, Classic Themes, Kadence Blocks, ne-Kadence Theme
- **Ukuhlanganiswa kwe-WooCommerce okuthuthukisiwe** — kuhlelwe kabusha ukuze kusetshenziswe ama-API emvelo e-WooCommerce ukuze kube nokuthembeka nokuhambisana okungcono
- **Ukutholwa komhlinzeki okuzenzakalelayo** — kuthola ngokuzenzakalelayo futhi kuvuselele ama-plugin namatimu atholakalayo lapho kwenziwa kusebenze noma kuvalwa
- **Ukwenziwa kwelogo yangokwezifiso** — khiqiza futhi ushumeke amalogo e-SVG angokwezifiso ngokuhlanzwa okuzenzakalelayo okuphephile kwe-namespace
- **Ukuqinisekiswa kokugqama kwephalethi** — hlola imibhangqwana yemibala ukuze ihambisane nokufinyeleleka kwe-WCAG ngaphambi kokuyisebenzisa kutimu yakho
- **Ukutholwa okuthuthukiswe ngezithombe** — layisha izithombe ngesikhathi sengxoxo yokutholwa ye-Theme Builder ukuze kuthonye indlela yomklamo
- **Amamenyu ezokwamukela izivakashi** — khiqiza amakhasi emenyu yokudla neziphuzo ahlelekile ezindawo zokudlela, amakhefi, namabhizinisi ezokwamukela izivakashi
- **Ukubuka kuqala kwedeskithophu nokweselula** — buka kuqala umklamo wakho kudeskithophu nakumadivayisi eselula ngesikhathi sokukhetha indlela yomklamo
- **Ukuklwebha isayithi kwe-Tier 1** — sd-ai-agent/site-scrape manje kuyatholakala ngokuzenzakalelayo ukuze kuhlaziywe amawebhusayithi ancintisana nawe nawokukhuthaza
- **Amathuluzi okufunda e-Google Calendar** — funda amakhalenda nemicimbi elungiselelwe ukuze ukuzenzakalela kukwazi ukucabanga ngomongo weshejuli ozayo
- **Ukufanisa oxhumana nabo nabahambele umcimbi** — fanisa ababambiqhaza bekhalenda nabasebenzisi besayithi abaziwayo noxhumana nabo ngaphambi kokukhiqiza umsebenzi wokulandela
- **Amasango okugunyazwa umuntu** — misa kancane ukuzenzakalela okubucayi ukuze kubuyekezwe ngaphambi kokuthumela imiyalezo, ukuqhubeka nomsebenzi, noma ukwenza izinguquko
- **Amarekhodi ezikhumbuzi nokuvimbela ukuphindaphindwa** — landelela izikhumbuzi esezithunyelwe ukuze ukuqaliswa kokuzenzakalela okuphindaphindiwe kungamazisi umamukeli ofanayo kabili
- **Izaziso ze-SMS ze-TextBee** — thumela izaziso zemiyalezo yombhalo ezilungiselelwe nge-TextBee lapho umhlinzeki we-SMS enikwe amandla
- **Ukuhlinzekwa kwesevisi ye-Superdav ephethwe** — xhuma amasayithi asekelwayo kuma-endpoint e-Superdav asingathiwe ngokusetha okuzenzakalelayo esikhundleni sokulungiselela i-endpoint mathupha
- **Iphakheji elihambisana nayo le-Advanced** — amathuluzi abathuthukisi abathembekile asatshalaliswa ngokwehlukana emisebenzi yesistimu yamafayela, isizindalwazi, WP-CLI, umthumeli we-REST, umakhi we-plugin, isifinyezo se-git, ukuphathwa kwabasebenzisi, kanye ne-benchmark

## Izimfuneko {#requirements}

- WordPress 6.0 noma ngaphezulu
- PHP 8.1 noma ngaphezulu
- WordPress Multisite (yenziwe isebenze kunethiwekhi) noma ukufakwa kwesayithi elilodwa
- I-plugin yesixhumi somhlinzeki we-AI esebenzayo (Anthropic, OpenAI, noma ehambisanayo)
- Gratis AI Agent v1.9.0 noma ngaphezulu (Superdav AI Agent yandisa i-Gratis AI Agent)

## Ukufaka {#installation}

1. Layisha amafayela e-plugin ku-`/wp-content/plugins/superdav-ai-agent`
2. Yenza i-plugin isebenze ngesikrini se-**Plugins** ku-WordPress
3. Qinisekisa ukuthi i-Gratis AI Agent nayo yenziwe yasebenza
4. Zulazulela ku-**Gratis AI Agent → Settings** bese ulungisa umhlinzeki wakho wokusesha ku-inthanethi, uxhumano lwesevisi ye-Superdav ephethwe, Google Calendar, noma imininingwane ye-TextBee njengoba kudingeka

Iphakheji ye-WordPress.org iqukethe izici eziyinhloko ze-Superdav AI Agent. Iphakheji elihambisana nayo le-Advanced lakhiwa futhi lisatshalaliswa ngokwehlukana ezindaweni lapho amathuluzi abathuthukisi abathembekile efakwe futhi egunyazwe ngokusobala.

| **Kadence Blocks** | Amasayithi ane-plugin ye-Kadence Blocks esebenzayo |

## Abahlinzeki Bokusesha Ku-inthanethi {#internet-search-providers}

Superdav AI Agent v1.10.0 yengeza **Tavily** njengenye indlela ye-Brave Search yemibuzo yokusesha ku-inthanethi. Bobabili abahlinzeki banikeza imiphumela yokusesha yesikhathi sangempela futhi bangalungiselelwa ku-**Gratis AI Agent → Settings → Advanced**.

| Umhlinzeki | Izici | Ukusetha |
|---|---|---|
| **Brave Search** | Imiphumela yokusesha esheshayo, egxile kubumfihlo | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Imiphumela yokusesha ecebile, amakhono ocwaningo athuthukile | [Tavily API](https://tavily.com/) |

Bona [Ukusesha Ku-inthanethi](../../user-guide/configuration/internet-search) ukuze uthole imiyalelo yokulungiselela.

## Imisebenzi Yekhalenda, Yokugunyaza, Nezaziso {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 yandisa imisebenzi yokuzenzakalela ngale kwemisebenzi yokuqukethwe nokwakha isayithi:

- **Amathuluzi e-Google Calendar** avumela i-agent ukuthi ifunde amakhalenda alungiselelwe nemininingwane yemicimbi lapho imininingwane yekhalenda itholakala.
- **Abasizi boxhumana nabo nabahambele umcimbi** bafanisa ababambiqhaza bemicimbi nabasebenzisi be-WordPress noma oxhumana nabo abaziwayo ngaphambi kokulungiselela izikhumbuzi noma imisebenzi yokulandela.
- **Amasango okugunyazwa umuntu** avumela ukuzenzakalela ukuthi kume kancane kuze kube yilapho umsebenzisi ogunyaziwe ebuyekeza isenzo esiphakanyisiwe.
- **Amarekhodi ezikhumbuzi** avimbela izikhumbuzi zekhalenda noma zezaziso eziphindaphindiwe lapho ukuzenzakalela kuzanywa futhi noma kuhlelwa ngokuphindaphindiwe.
- **Ukusekelwa kwe-SMS kwe-TextBee** kwenza izaziso zemiyalezo yombhalo zisebenze lapho umhlinzeki we-TextBee elungiselelwe.

Sebenzisa la mathuluzi kosizo oluqonda ishejuli, ukulandela abahambele umcimbi, nemisebenzi yezikhumbuzi esadinga ukubuyekezwa ngumuntu ngaphambi kokuba kuthunyelwe imiyalezo ebucayi.

## Ukulungiselela {#configuration}

Zonke izilungiselelo zingaphansi kwe-**Gratis AI Agent → Settings** kumphathi we-WordPress. Superdav AI Agent yengeza izinketho ezilandelayo zokulungiselela:

| Isilungiselelo | Incazelo |
|---|---|
| **Umhlinzeki Wokusesha Ku-inthanethi** | Khetha phakathi kwe-Brave Search ne-Tavily (uyazikhethela) |
| **Tavily API Key** | Ukhiye we-API wokusesha ku-inthanethi kwe-Tavily (uma i-Tavily ikhethiwe) |
| **Isevisi ye-Superdav Ephethwe** | Xhuma amasayithi asekelwayo kusevisi ye-Superdav esingathiwe ngokuhlinzekwa kwe-endpoint okuzenzakalelayo |
| **Imininingwane ye-Google Calendar** | Lungiselela ukufinyelela kwekhalenda ukuze i-agent ikwazi ukufunda amakhalenda nemicimbi |
| **Umhlinzeki we-SMS we-TextBee** | Lungiselela imininingwane ye-TextBee yemisebenzi yezaziso ze-SMS egunyaziwe |

## Iphakheji Elihambisana Nayo le-Advanced {#advanced-companion-package}

Superdav AI Agent v1.18.0 yengeza iphakheji elihambisana nayo le-Advanced elisatshalaliswa ngokwehlukana lemisebenzi yabathuthukisi abathembekile. Lawa makhono awahlanganisiwe kuphakheji eyinhloko ye-WordPress.org. Faka iphakheji elihambisana nayo kuphela kumasayithi lapho abaphathi beyithemba ngokusobala i-agent ukuthi ifinyelele amathuluzi onjiniyela.

Amathuluzi e-Advanced afaka imisebenzi yesistimu yamafayela, ukufinyelela kusizindalwazi, WP-CLI, imisebenzi yomthumeli we-REST, izinsiza zomakhi we-plugin, izifinyezo ze-git, abasizi bokuphathwa kwabasebenzisi, namathuluzi e-benchmark. Gcina la mathuluzi elinganiselwe kubaphathi abathembekile futhi ubuyekeze ukulungiselela kwesango lokugunyaza ngaphambi kokuwanika amandla kumasayithi okukhiqiza.
