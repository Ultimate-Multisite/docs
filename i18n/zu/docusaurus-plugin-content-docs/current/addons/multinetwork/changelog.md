# Umlando Wezinguquko Wamanethiwekhi Amaningi

### 1.3.0 {#130}
* Okusha: Isibonisi sokuqala sethempulethi yenethiwekhi — phequlula futhi ubuke kusengaphambili amathempulethi enethiwekhi ngephaneli ebukhoma ngaphambi kokuthenga.
* Okusha: Isikhethi sethempulethi yenethiwekhi ngesikhathi sokukhokha — amakhasimende angakhetha ithempulethi yenethiwekhi ngesikhathi sokugeleza kokubhalisa nokukhokha.
* Okusha: I-SSO yesixhumanisi somlingo phakathi kwamanethiwekhi — amakhasimende angafinyelela amanethiwekhi angaphansi kalula ngokugunyazwa kwesixhumanisi somlingo.
* Okusha: Ukukopishwa kwemidiya yenethiwekhi — imidiya yesayithi ikopishwa kahle lapho kukopishwa amanethiwekhi kusuka kuthempulethi.
* Ukulungisa: Ukukopishwa kwenethiwekhi manje kusebenza ngaphandle kwesengezo se-multi-tenancy; amalungelo e-super admin aphinde anikezwe ngemva kokukopisha.
* Ukulungisa: Ukwakhiwa kwenethiwekhi manje kuyahambisana ne-WooCommerce nawo wonke amasango okukhokha.
* Ukulungisa: Inombolo ye-port manje ifakiwe ekwakheni i-domain yokubuyela emuva.
* Ukulungisa: I-CSS yesikhethi sethempulethi ithuthelwe ku-stylesheet efakwe ku-queue ukuze ibonakale kahle kuzo zonke izingqikithi.
* Ukulungisa: Ukuvikela ngokumelene nokhiye we-`network_id` ongachazwanga kusikophu sombuzo wesayithi.
* Ukulungisa: Ukuhlolwa kwemvume yenethiwekhi ngaphambi kokushintsha umongo kuvimbela amaphutha ezakhiwo zetafula le-`wpdb`.
