# Multi-Network

Isengezo se-Ultimate Multisite: Multinetwork sandisa inkundla yakho ye-Ultimate Multisite ngokusebenza okunamandla kwamanethiwekhi amaningi. Lesi sengezo sivumela amakhasimende akho ukuthi athenge futhi aphathe amanethiwekhi aphelele e-WordPress multisite, hhayi amasayithi ngamanye kuphela.

## Izici Eziyinhloko {#key-features}

* **Ukuhlinzekwa Kwenethiwekhi**: Dala futhi ulungiselele ngokuzenzakalelayo amanethiwekhi aphelele e-WordPress multisite amakhasimende akho
* **Izifanekiso Zenethiwekhi**: Vumela amakhasimende aphequlule, abuke kuqala, futhi akhethe isifanekiso senethiwekhi esakhiwe kusengaphambili ngaphambi kokubhalisa
* **Ukuphathwa Kwenethiwekhi**: Amandla aphelele okuphatha ezindaweni zamanethiwekhi amaningi
* **Amanethiwekhi Ahlukanisiwe**: Inethiwekhi yekhasimende ngayinye isebenza ngokuzimela inamathebula ayo edathabheyisi nokucushwa kwayo
* **I-SSO Enqamula Amanethiwekhi**: Nikeza amakhasimende ukufinyelela okushelelayo kuma-sub-network awo ngokuqinisekiswa kwe-magic-link
* **Isakhiwo Esikhulayo**: Phatha amanethiwekhi amaningi kahle ngokwabiwa kwezinsiza okulungiselelwe
* **Ukuhlanganiswa**: Kuhlanganiswa ngokushelelayo nezinhlelo zokubhaliselwe nezokukhokhisa ze-Ultimate Multisite

## Izimo Zokusetshenziswa {#use-cases}

* **Ukusingathwa kwe-White Label**: Nikeza amakhasimende amanethiwekhi awo e-multisite anophawu lwawo
* **Izixazululo Ze-ejensi**: Vumela ama-ejensi ukuthi aphathe amanethiwekhi amaningi amaklayenti esuka enkundleni eyodwa
* **Ukusetshenziswa Kwamabhizinisi Amakhulu**: Sekela izinhlangano ezinkulu ezinezidingo eziyinkimbinkimbi zamanethiwekhi amaningi
* **Izinkundla Zemfundo**: Dala amanethiwekhi ahlukene ezikole noma iminyango ehlukene

## Izidingo {#requirements}

* Ultimate Multisite 2.0.0 noma ngaphezulu
* Ukufakwa kwe-WordPress Multisite
* PHP 7.4 noma ngaphezulu
* MySQL 5.6 noma ngaphezulu

## Ukufakwa {#installation}

1. Layisha amafayela esandiso enkombeni ethi `/wp-content/plugins/ultimate-multisite-multinetwork`
2. Sebenzisa isandiso ngesikrini esithi 'Izandiso' ku-WordPress
3. Lungiselela izilungiselelo zamanethiwekhi amaningi ngo-Ultimate Multisite → Izilungiselelo

## Izifanekiso Zenethiwekhi {#network-templates}

Inguqulo 1.3.0 yengeza ukugeleza kokuthola nokukhetha izifanekiso kumakhasimende athenga amanethiwekhi amasha. Sebenzisa izifanekiso zenethiwekhi uma ufuna ukuthi inethiwekhi entsha yekhasimende ngayinye iqale ngesakhiwo esilungisiwe, idizayini, ilabhulali yokuqukethwe, noma ukusethwa okuqondene nomkhakha othile.

### Isibuki sokuqala sesifanekiso {#template-previewer}

Isibuki sokuqala sesifanekiso senethiwekhi sivumela amakhasimende ukuthi aphequlule izifanekiso ezitholakalayo ngaphambi kokukhokha. Isifanekiso ngasinye singavulwa kuphaneli yokubuka kuqala ebukhoma, ukuze amakhasimende aqhathanise izakhiwo nokuqukethwe ngaphambi kokuthi athenge inethiwekhi.

Sebenzisa isibuki sokuqala ukuze usize amakhasimende:

* Abuyekeze ukubukeka nesakhiwo sesifanekiso senethiwekhi ngasinye esitholakalayo
* Aqathanise izifanekiso eziningi ngaphandle kokushiya ukugeleza kokuthenga
* Akhethe isifanekiso esihambisana kangcono nesimo sawo sokusetshenziswa ngaphambi kokubhalisa

Gcina amagama nezincazelo zezifanekiso zilungele amakhasimende. Isibuki sokuqala siwusizo kakhulu lapho amakhasimende engakwazi ukuqonda isimo sokusetshenziswa esihlosiwe, njengamanethiwekhi ama-ejensi, amanethiwekhi ezikole, amanethiwekhi ama-franchise, noma amaphakheji okusingathwa kwe-white-label.

### Isikhethi sesifanekiso ekukhokheni {#template-picker-at-checkout}

Isikhethi sesifanekiso singangezwa ekugelezeni kokukhokha kokubhalisa njengenkambu. Uma silungisiwe, amakhasimende akhetha isifanekiso senethiwekhi asifunayo ngesikhathi sokukhokha, futhi isifanekiso esikhethiwe sisetshenziswa lapho inethiwekhi yawo entsha ihlinzekwa.

Ukuze usebenzise isikhethi ekugelezeni kokukhokha:

1. Lungiselela amanethiwekhi ofuna ukuwanikeza njengezifanekiso.
2. Nika amandla inkambu yokukhetha isifanekiso se-Multinetwork efomini elifanele lokubhalisa noma lokukhokha.
3. Khetha ukuthi yiziphi izifanekiso okufanele zitholakale kulokho kugeleza kokukhokha.
4. Shicilela ukugeleza kokukhokha bese uhlola ukubhalisa ukuze uqinisekise ukuthi isifanekiso esikhethiwe sisetshenziswa kunethiwekhi entsha.

Uma ukugeleza kokukhokha kunikeza umkhiqizo wenethiwekhi owodwa kuphela, isikhethi sisengaba wusizo njengesinyathelo sokuqalisa esiqondisiwe. Kuma-funnel anezinhlelo eziningi, bonisa kuphela izifanekiso ezihambisana nomkhiqizo noma uhlelo olukhethiwe.

## I-Magic Link SSO Enqamula Amanethiwekhi {#cross-network-magic-link-sso}

I-magic link SSO enqamula amanethiwekhi ivumela amakhasimende ukuthi ahambe phakathi kwendawo ye-akhawunti yawo eyinhloko nama-sub-network ngaphandle kokungena ngemvume mathupha futhi. Lapho ikhasimende lilandela isixhumanisi sokufinyelela esinqamula amanethiwekhi, i-Ultimate Multisite ikhiqiza i-magic link enomkhawulo wesikhathi futhi iqinisekisa ikhasimende kunethiwekhi okuyiwa kuyo.

Lokhu kuthuthukisa okuhlangenwe nakho kwamakhasimende aphatha inethiwekhi engaphezu kweyodwa noma adinga ukufinyelela amanethiwekhi ezifanekiso asanda kuklonwa ngokushesha ngemva kokuthenga.

Uhlu lokuhlola lomlawuli:

* Qinisekisa ukuthi isengezo se-Multinetwork siyasebenza kulo lonke ukufakwa kwe-WordPress multisite.
* Qinisekisa ukuthi amakhasimende anokufinyelela okufanele kwenethiwekhi ngaphambi kokunikezwa izixhumanisi ezinqamula amanethiwekhi.
* Gcina izizinda zenethiwekhi nezizinda ezimephuziwe zilungiselelwe kahle ukuze ama-magic link axazululeke kunethiwekhi okuhloswe ngayo.
* Hlola ideshibhodi yekhasimende nanoma yiziphi izinkinobho zokufinyelela zangemva kokukhokha ngemva kokushintsha izilungiselelo zenethiwekhi, zesizinda, noma zokukhokha.

Ama-magic link kufanele aphathwe njengezixhumanisi zokungena ngemvume. Akhelwe ukufinyelela kwekhasimende okushelelayo futhi akufanele adluliselwe phambili noma aphinde asetshenziswe ngaphandle kweseshini ehlosiwe.

## Imibuzo Evame Ukubuzwa {#frequently-asked-questions}

### Yini i-multinetwork? {#what-is-multinetwork}

I-Multinetwork iyisici se-WordPress esikuvumela ukuthi usebenzise amanethiwekhi amaningi azimele e-multisite ekufakweni okukodwa kwe-WordPress.

### Lokhu kuhluke kanjani ku-multisite evamile? {#how-is-this-different-from-regular-multisite}

I-multisite evamile ivumela amasayithi amaningi ngaphakathi kwenethiwekhi eyodwa. I-Multinetwork ivumela amanethiwekhi amaningi, ngalinye linamasayithi alo, wonke ngaphakathi kokufakwa okukodwa kwe-WordPress.

### Ingabe lokhu kudinga ukucushwa okukhethekile kweseva? {#does-this-require-special-server-configuration}

Yebo, i-multinetwork idinga ukusethwa nokucushwa kweseva okuqondile. Sicela ubheke amadokhumenti ukuze uthole imiyalelo enemininingwane yokusetha.
