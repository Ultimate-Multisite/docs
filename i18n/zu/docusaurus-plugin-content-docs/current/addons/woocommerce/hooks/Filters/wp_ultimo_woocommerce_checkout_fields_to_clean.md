# Isihlungi: wp_ultimo_woocommerce_checkout_fields_to_clean {#filter-wpultimowoocommercecheckoutfieldstoclean}

Sivumela abathuthukisi ukuthi bahlunge izinkambu zekheli lokukhokhisa ezizenzakalelayo ze-WooCommerce.

Njengoba i-Ultimate Multisite inezinkambu zekheli lokukhokhisa, kungase kube nengqondo ukususa izinkambu ezizenzakalelayo eziboniswa yi-WooCommerce efomini le-checkout. Ngokuzenzakalelayo, alikho inani elidluliselwa kulolu hlu, ngakho zonke izinkambu ezilungiselelwe ku-WooCommerce ukuthi ziboniswe ziyagcinwa.

### Kusukela {#since}

- 2.0.0
### Umthombo {#source}

Kuchazwe ku-[`inc/gateways/class-woocommerce-gateway.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-woocommerce/blob/main/inc/gateways/class-woocommerce-gateway.php#L277) emugqeni 277


## Okubuyiswayo {#returns}
Uhlu lwezinkambu okufanele zisuswe. Amanani atholakalayo afaka billing_first_name, billing_last_name, billing_company, billing_country, billing_address_1, billing_address_2, billing_city, billing_state, billing_postcode, billing_email, kanye ne-billing_phone.
