# Isihlungi: wp_ultimo_skip_network_active_check

Ivumela abathuthukisi ukuthi banqamule ngokushesha ukuhlola kokwenza kusebenze kwenethiwekhi.

Lokhu kuyasiza lapho kusetshenziswa izinhlelo ezisekelwe ku-composer nezinye izinhlelo ezenziwe ngokwezifiso, njenge-Bedrock, ngokwesibonelo, lapho ukusebenzisa ama-plugin njengama-mu-plugins kuyinto evamile.

## Amapharamitha

| Igama | Uhlobo | Incazelo |
|------|------|-------------|
| $skip_network_activation_check | `bool` | Uma kufanele seqe ukuhlola noma cha, okuzenzakalelayo kungu-false. |

### Kusukela

- 2.0.0
### Umthombo

Kuchazwe ku-[`inc/class-requirements.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-woocommerce/blob/main/inc/class-requirements.php#L272) emgqeni 272


## Ibuyisa
true uma ufisa ukweqa ukuhlola, false uma kungenjalo.
