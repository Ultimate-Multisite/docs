# Isihlungi: wp_ultimo_woocommerce_should_use_subscriptions

Vumela abathuthukisi ukuba baphoqe ukusetshenziswa kwe-WooCommerce Subscriptions.

Ngokuzenzakalelayo i-add-on ihlola uma (1) i-WooCommerce Subscriptions ikhona (kungaba i-plugin ephelele noma i-core ehlanganisiwe), bese (2) ihlola isilungiselelo esivumela ukusetshenziswa kwe-WooCommerce Subscriptions.

## Amapharamitha

| Igama | Uhlobo | Incazelo |
|------|------|-------------|
| $enable_woocommerce_subscription_integration | `bool` | Uma njengamanje sidinga ukusebenzisa i-Woo Subs. |
| $woo_gateway | `\WP_Ultimo_WooCommerce\Gateways\WooCommerce_Gateway` | Into yamanje. |

### Umthombo

Kuchazwe ku-[`inc/gateways/class-woocommerce-gateway.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-woocommerce/blob/main/inc/gateways/class-woocommerce-gateway.php#L361) emgqeni 361


## Okubuyiswayo
true ukuze kusetshenziswe i-WooCommerce Subs, false ukuze ingasetshenziswa.
