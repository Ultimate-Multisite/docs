# Irekhodi Lezinguquko le-WooCommerce Integration

Inguqulo 2.2.0 - Ikhishwe ngo-2026-07-01
* Okusha: Amanani entela e-Ultimate Multisite manje avela njengemigqa yezimali ehlukene ye-WooCommerce ngesikhathi sokukhokha, okwenza izamba zentela zicace kakhudlwana ngaphambi kokukhokha.
* Okusha: Kungezwe isilungiselelo sokuzikhethela esithi "Misa Ubulungu Ngokushesha Lapho Ukuvuselelwa Kwehlulekile" kanye nefiltha ethi `wu_woo_suspend_on_payment_failure` yamasayithi afuna ukumiswa ngenxa yokwehluleka kokuvuselela ngesikhathi sewindi lokuzama futhi le-WooCommerce Subscriptions.
* Lungisa: Kuhlanganiswe izimo ze-WooCommerce Subscription ebezingahlala zingavumelani nobulungu be-Ultimate Multisite ngemva kokuvuselelwa okuhlulekile noma okubuyiselwe.
* Lungisa: Kungezwe imali yesitolo se-WooCommerce ohlwini lwezimali lwe-Ultimate Multisite lapho ibingekho.
* Lungisa: Kugcinwe imininingwane yokukhokhisa yekhasimende lapho kuqondiswa ababhalisile ekukhokheni kwe-WooCommerce.
* Kuthuthukisiwe: Kungezwe ukuhambisana ne-Jetpack Autoloader 5.
* Kuthuthukisiwe: Kuhlanziwe ukukhiqizwa kwephakheji yokukhishwa ukuze ama-zip e-GitHub nawemakethe agweme izinkomba zesiteji ezifakwe ngaphakathi namafayela okuthuthukisa.
