# PayFast {#payfast}

Multisite Ultimate: Isango le-Payfast

Yengeza i-Payfast - isango lokukhokha ku-inthanethi elisheshayo nelivikelekile eNingizimu Afrika - njengenketho yokukhokha ye-Ultimate Multisite.

## Ukufaka {#installation}

1. Layisha 'ultimate-multisite-payfast' kunkomba ye-'/wp-content/plugins/'
2. Yenza kusebenze i-plugin ngemenyu ethi 'Ama-plugin' ku-WordPress
3. Landela i-Wizard yesinyathelo ngesinyathelo ukuze usethe i-plugin

## Umhlahlandlela Wokusebenzisa {#usage-guide}

I-PayFast ingumcubunguli ohamba phambili wokukhokha ku-inthanethi wamabhizinisi aseNingizimu Afrika. Izinikele ekusizeni abathengisi baku-inthanethi bakhule ngokwenza ukuthengiselana kube bushelelezi futhi kuvikeleke. Ngokwesicelo somsebenzisi we-Ultimate Multisite, sinqume ukudala isengezo esizokwenza ukuhlanganiswa ne-PayFast kwenzeke. Lesi sihloko sizobonisa ukuthi ungasetha kanjani i-PayFast njengesango lakho lokukhokha ukuze uthole inkokhelo kunethiwekhi yakho ye-Ultimate Multisite.

Isinyathelo sokuqala ukufaka isengezo.

# Indlela yokufaka isengezo se-Payfast: {#how-to-install-the-payfast-add-on}

Yiya ekhasini lezingezo bese ukhetha isengezo se-Ultimate Multisite: Isango le-Payfast.

![Ikhasi lezingezo](/img/admin/addons-page.png)

Iwindi lesengezo lizovela. Chofoza okuthi Faka Manje.

![Ikhasi lezingezo elinenketho yokufaka](/img/admin/addons-page.png)

Ngemva kokuthi ukufaka sekuphelile, uzoqondiswa kabusha ekhasini lama-plugin. Lapha, vele uchofoze okuthi Vula Kunethiwekhi futhi isengezo se-_Payfast_ sizovulwa kunethiwekhi yakho.

# Ukusetha i-PayFast {#setting-up-payfast}

Ngemva kokufaka isengezo, manje ungaya kuzilungiselelo ze-Ultimate Multisite futhi uqale ukumisa i-PayFast.

Ngaphansi kwekhasi **lezilungiselelo ze-Ultimate Multisite**, ungathola ithebhu ethi "Izinkokhelo". Ngaphansi kwale thebhu, kufanele ubone okuthi "**Payfast** " kanye nenkinobho yokuvula noma yokuvala leli sango lokukhokha.

![Izilungiselelo zesango lokukhokha](/img/config/settings-payment-gateways.png)Qhubeka uchofoze inkinobho yokuvula ukuze uyinike amandla. Izobe isandisa ibhokisi ukuze ibonise izinkambu ezahlukene okufanele uzigcwalise ezifana ne-**Mechant ID, Merchant Key ne-PassPhrase.** Ungathola lolu lwazi ku-akhawunti yakho ye-**Payfast**

<!-- Isithombe-skrini asitholakali: izilungiselelo ze-PayFast ezandisiwe ezibonisa izinkambu ze-Merchant ID, Merchant Key, ne-PassPhrase -->

Isinyathelo esilandelayo ukungena ngemvume ku-akhawunti yakho ye-PayFast ukuze uthole ulwazi oludingayo.

Lapho usungenile, ungabona ngokushesha imininingwane yokuqinisekisa oyidingayo ku-**Payfast** Account Dashboard yakho.

<!-- Isithombe-skrini asitholakali: i-PayFast Account Dashboard ebonisa imininingwane yokuqinisekisa ye-Merchant ID ne-Merchant Key -->Ungavele ukopishe futhi unamathisele imininingwane yokuqinisekisa bese ugcwalisa imininingwane kuzilungiselelo zakho ze-Ultimate Multisite. Bese uchofoza inkinobho ethi **"Londoloza izilungiselelo** " ukuze ulondoloze izinguquko.

<!-- Isithombe-skrini asitholakali: izilungiselelo ze-Ultimate Multisite PayFast ezinemininingwane yokuqinisekisa egcwalisiwe nenkinobho ethi Save -->

Sekuphelile, ukuhlanganiswa kwakho kokukhokha kwe-PayFast sekuphelele!
