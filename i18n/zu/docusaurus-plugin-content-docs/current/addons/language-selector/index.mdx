# Isikhethi Solimi

Multisite Ultimate: Isikhethi Solimi ekubhaliseni

Engeza inkambu yesikhethi solimi enqubweni yokubhalisa.

## Ukufaka {#installation}

1. Layisha 'ultimate-multisite-language-selector' enkombeni ethi '/wp-content/plugins/'
2. Vula i-plugin ngemenyu ethi 'Plugins' ku-WordPress Network Admin yakho

## Umhlahlandlela Wokusebenzisa {#usage-guide}

Ngalesi sengezo samahhala ungavumela abasebenzisi bakho bakhethe ukuthi yiluphi ulimi abangathanda ukulubona enqubweni yokubhalisa nasekhasini lokungena.

Ngemva [kokuvula ilayisensi yakho](../../user-guide/getting-started/installing-ultimate-multisite.md) uzokwazi ukufaka izengezo ze-Ultimate Multisite ilayisensi yakho ekunika ukufinyelela kuzo.

Yiya ekhasini lezilungiselelo ze-Ultimate Multisite yakho bese, kumenyu engakwesokunxele, uchofoze ilebula ethi _Izengezo_.

![Ikhasi lezengezo](/img/admin/addons-page.png)

_Yiya ekhasini Lezengezo ukuze uqale ukufaka Isikhethi Solimi_

Lapho, uzothola zonke izengezo esizinikezayo.

Izengezo zamahhala zitholakala kuwo wonke amakhasimende ethu, kungakhathaliseki uhlelo lokubhalisa ababhalisele kulo.

Chofoza inkinobho ethi _Yifake Manje!_ ngaphansi kwesithonjana sesengezo. Bese uyivula ekhasini le-Plugins.

![Ikhasi lezengezo elinenkinobho yokufaka](/img/admin/addons-page.png)

_Chofoza inkinobho ukuze ufake isengezo_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_Yivule ekhasini lakho le-Plugins_

Manje, yiya kumenyu Yezilungiselelo ze-Ultimate Multisite bese ufaka ithebhu ethi _Izilungiselelo Zenethiwekhi_. Lapho, ngezansi kwekhasi, uzothola isigaba esisha esibizwa ngokuthi _Izilimi_.

![Ikhasi lezilungiselelo](/img/config/settings-general.png)

_Isigaba esisha sha siyavela!_

Ungakushiya kungenalutho ukuze ubonise zonke izilimi njengezinketho noma ukhethe ukuthi yiziphi ofuna ukuzenza zitholakale. Izilimi yilezo ozifakile ekuhlelweni kwakho kwe-WordPress.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_Uhlu lwezilimi ezitholakala kulokhu kufakwa kwe-WP_

Ngemva kokulondoloza izinguquko ozenzile, imenyu yokukhetha enezilimi ozikhethile izoboniswa ekhasini Lokungena kanye naseNqubweni Yokubhalisa.

![Izilungiselelo zokungena nokubhalisa](/img/config/settings-login-registration.png)

_Isikhethi Solimi ekhasini Lokungena_

![Ikhasi lokubhalisa](/img/config/registration-page.png)

_Isikhethi Solimi eNqubweni Yokubhalisa_

## Ukusetha Ulimi nge-URL {#setting-the-language-via-the-url}

Ungashintsha futhi ulimi lwekhasi ngokudlulisa ipharamitha ye-locale ku-URL. Uma ufuna ukuqondisa kabusha ekhasini lokubhalisa ngesiNtaliyane, isibonelo, engeza ?locale=it_IT ku-URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Ulimi kufanele lufakwe kunethiwekhi yakho ye-WordPress ukuze lokhu kusebenze.
