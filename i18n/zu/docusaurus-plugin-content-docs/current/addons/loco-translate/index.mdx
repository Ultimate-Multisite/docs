# Loco Translate {#loco-translate}

Isengezo sokuhlanganisa i-Ultimate Multisite: Loco Translate siletha amandla okuhumusha aqondene nesayithi kunethiwekhi yakho ye-WordPress multisite. Uma uke wadinga ukuthi amasayithi ahlukene kunethiwekhi yakho abe nezinguqulo ezihlukene ze-plugin noma theme efanayo, lesi sengezo siyisixazululo esiphelele.

**Okwenziwa Yilesi Sengezo**

Lesi sengezo sihlanganisa i-Ultimate Multisite ngaphandle kwezihibe ne-plugin edumile ye-Loco Translate, sivumele isayithi ngalinye elingaphansi kunethiwekhi yakho ukuthi ligcine izinguqulo zalo ezenziwe ngokwezifiso. Kungakhathaliseki ukuthi usebenzisa inkundla yezifunda eziningi, uhlinzeka imiphakathi yezilimi ezahlukene, noma udinga nje ukuguquguquka kokuhumusha kunethiwekhi yakho, lesi sengezo sikwenza kube lula.

**Silungele:**

* Amabhizinisi asezifundeni eziningi anezidingo ezihlukene zokwenza okuqukethwe kuhambisane nendawo
* Izikhungo zemfundo ezihlinzeka imiphakathi ehlukahlukene
* Izinkundla ze-SaaS ezinikeza ukwesekwa kwezilimi eziningi
* Ama-ejensi ewebhu aphatha amasayithi amaklayenti amaningi
* Noma iyiphi inethiwekhi ye-multisite edinga ukuguquguquka kokuhumusha

## Ukufaka {#installation}

1. Qinisekisa ukuthi i-Ultimate Multisite ifakiwe futhi ivuliwe kunethiwekhi
2. Faka futhi uvule i-plugin ye-Loco Translate
3. Layisha lesi sengezo kufolda yakho yama-plugin
4. Vula kunethiwekhi isengezo sokuhlanganisa i-Loco Translate
5. Qala ukudala izinguqulo eziqondene nesayithi nge-Loco Translate kunoma yiliphi isayithi elingaphansi

Yilokho kuphela! Ukuhlanganisa kusebenza ngokuzenzakalela ngaphandle kwanoma yikuphi ukulungiselela okwengeziwe.

## Imibuzo Evame Ukubuzwa {#frequently-asked-questions}

## Ingabe kudingeka ngilungise noma yini ngemva kokufaka? {#do-i-need-to-configure-anything-after-installation}

Cha! Ukuhlanganisa kuyazenzakalela. Uma isengezo ne-Loco Translate sekuvuliwe, amasayithi angaphansi angaqala ngokushesha ukudala izinguqulo zawo ezenziwe ngokwezifiso.

## Kwenzekani uma isayithi elingaphansi lingadali izinguqulo ezenziwe ngokwezifiso? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

Isayithi lizosebenzisa ngokuzenzakalela izinguqulo ezisebenza kunethiwekhi yonke njengendlela yokubuyela emuva. Akukho okuphukayo, futhi konke kusebenza ngaphandle kwezihibe.

## Ingabe amasayithi angaphansi angafinyelela izinguqulo ezisebenza kunethiwekhi yonke? {#can-subsites-access-network-wide-translations}

Abaphathi abavamile besayithi babona futhi bahlele kuphela izinguqulo zesayithi labo. Abaphathi abakhulu bangafinyelela kokubili izinguqulo ezisebenza kunethiwekhi yonke nezinguqulo eziqondene nesayithi kuye ngokuthi yiliphi isayithi abalibukayo.

## Ingabe lokhu kusebenza kuwo womabili ama-plugin nama-theme? {#does-this-work-with-both-plugins-and-themes}

Yebo! Izinguqulo eziqondene nesayithi zisebenza kuwo womabili ama-plugin nama-theme e-WordPress.

## Ingabe lokhu kuzoyinciphisa inethiwekhi yami ye-multisite? {#will-this-slow-down-my-multisite-network}

Cha. Isengezo sisebenzisa ukulayisha amafayela okusebenzayo okuhlola kuphela izinguqulo eziqondene nesayithi lapho kudingeka. Awukho umthelela wokusebenza obonakalayo.

## Ngingakopisha izinguqulo zenethiwekhi kusayithi elithile njengendawo yokuqala? {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

Yebo! Ngesikhathi uhlela izinguqulo ku-Loco Translate, ungaqala kalula ngezinguqulo zenethiwekhi bese uzenza ngokwezifiso zesayithi ngalinye.

## Ingabe lokhu kusebenza ne-WooCommerce namanye ama-plugin? {#does-this-work-with-woocommerce-and-other-plugins}

Nakanjani! Noma iyiphi i-plugin noma theme esebenzisa imisebenzi evamile yokuhumusha ye-WordPress izosebenza nezinguqulo eziqondene nesayithi.
