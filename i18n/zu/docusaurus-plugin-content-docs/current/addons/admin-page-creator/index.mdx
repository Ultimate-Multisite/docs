# Umdali Wamakhasi Omphathi

## Okuqukethwe
Umdali wamakhasi omphathi uzokuvumela ukuthi wengeze kalula amakhasi amasha omphathi namakhasi angaphansi ngaphakathi kwe-dashboard yomphathi ye-WordPress. Ungasetshenziswa ukudala ikhasi lokwamukela noma ikhasi lokufundisa elingasebenza njengekhasi lokufikela labasebenzisi abasha. Ungasebenzisa umakhi wamakhasi ofana ne-Beaver Builder, Elementor, Brizy, ne-Oxygen ukuze uklame amakhasi ngaphandle kobunzima.

Uphinde unikeze ukusekelwa kwe-HTML, CSS ne-JavaScript yangokwezifiso, kanye nokulawula ukufinyelela okusekelwe ku-plan nendima yomsebenzisi!

## Izici
* Ukusekela i-HTML ne-CSS Yangokwezifiso
* Ukusekelwa kwe-JavaScript
* Imodi Yokuqukethwe ye-PHP
* Imvume esekelwe endimeni yoMsebenzisi
* Ukusekelwa kubakhi bamakhasi

## Umhlahlandlela Wokusebenzisa

Nakuba i-WordPress yenza kube lula ngempela ukwengeza okuqukethwe engxenyeni engaphambili yamasayithi, ukudala amakhasi namakhasi angaphansi ohlangothini lomphathi kudinga amakhono ayisisekelo okukhoda. I-Ultimate Multisite: Umdali Wamakhasi Omphathi uhlose ukuletha lokho lula okufanayo kokusebenzisa ingxenye engaphambili engxenyeni engemuva yamasayithi enethiwekhi yakho.

## Kungani usebenzise Umdali Wamakhasi Omphathi?

## Ukudala Ikhasi lakho Lokuqala Lomphathi Langokwezifiso

Ngemva kokufaka nokwenza kusebenze i-Ultimate Multisite: Umdali Wamakhasi Omphathi, iya kumenyu yomphathi wenethiwekhi yakho bese uya kuMakhasi Omphathi.

![Uhlu Lwamakhasi Omphathi Angokwezifiso ku-Network Admin](/img/admin/admin-pages-list.png)

_Uhlu Lwamakhasi Omphathi Angokwezifiso ku-Network Admin_

Uzobona uhlu lwawo wonke amakhasi omphathi osuwadalile kuze kube manje (njengoba lokhu kungukuvakasha kwakho kokuqala, uhlu luzoba lungenalutho). Qhubeka bese uchofoza **Engeza Ikhasi Lomphathi elisha**. Uzoqondiswa kabusha kusihleli seKhasi Lomphathi.

### Isikhethi Sohlobo Lokuqukethwe

I-Ultimate Multisite: Umdali Wamakhasi Omphathi ikunikeza izinketho ezimbili uma kukhulunywa ngokuqukethwe kwekhasi: ungasebenzisa isihleli esizenzakalelayo se-WordPress WYSIWYG noma ushintshele kusihleli se-HTML.

Uma ungajwayelene ne-HTML, namathela kusihleli esizenzakalelayo se-WordPress. Uzokwazi ukuhlela okuqukethwe kwekhasi njengoba nje ubungenza nge-Post noma iKhasi elivamile.

## Izinketho Zemenyu

Ngasohlangothini, ku-meta-box Yezinketho Ezivamile, uzokwazi ukumisa uhlobo lwemenyu, ilebula, nokuningi.

### Izinhlobo Zemenyu

Ikhasi lakho elisha lomphathi langokwezifiso lingaba nezinhlobo ezimbili ezahlukene zemenyu: imenyu yezinga eliphezulu nemenyu engaphansi.

Ikhasi lemenyu yezinga eliphezulu, njengoba igama lisikisela, lizokwengeza ikhasi lakho elisanda kudalwa kumenyu yomphathi eceleni kwamakhasi ezenzakalelayo e-WordPress.

Uma ukhetha inketho yezinga eliphezulu, uzokwazi ukukhetha ukuhleleka kwemenyu nesithonjana semenyu futhi.

Uma ufuna ukwenza leli khasi libe ikhasi lemenyu engaphansi, uzokwazi ukukhetha ikhasi lomzali ozolixhumanisa nalo leli khasi. Ungangeza ikhasi lakho langokwezifiso emakhasini ezenzakalelayo e-WordPress noma emakhasini akho omphathi angokwezifiso ezinga eliphezulu.

<!-- Screenshot unavailable: Admin Page Creator menu type options with label, order, and icon settings -->

_Yenza ngokwezifiso ngokuphelele into yemenyu yaleli Khasi Langokwezifiso_

<!-- Screenshot unavailable: Admin Page Creator sub-menu parent page selection -->

_Ungakwazi futhi ukwengeza leli khasi njengemenyu engaphansi kwamanye amakhasi emenyu, kuhlanganise nalawo akho angokwezifiso ezinga eliphezulu._

## Izinketho Ezithuthukile

Kufanele ukwazi ukwengeza zonke izinhlobo zokuqukethwe emakhasini akho omphathi angokwezifiso. Lokho kuhlanganisa imithetho namafayela e-CSS, i-JavaScript, kanye nemitapo yangaphandle futhi. Ungakwenza lokho usebenzisa iThebhu Yezinketho Ezithuthukile.

### I-CSS Yangokwezifiso

Kuthebhu ye-CSS, ungangeza ikhodi yakho ye-CSS yangokwezifiso, kanye nokungenisa amafayela e-CSS avela kumasayithi akude (njenge-CDNs, Google Fonts, njll).

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_I-CSS yangokwezifiso nayo iyasekelwa. Ungakwazi futhi ukufaka amafayela esitayela angaphandle!_

### I-JavaScript Yangokwezifiso

Okufanayo kuyasebenza ku-JavaScript. Ungangeza imitapo yangaphandle ongayisebenzisa kukhodi yakho ye-JavaScript yangokwezifiso.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Engeza ikhodi yakho ye-JavaScript!_

## Izimvume

Futhi Ezinkethweni Ezithuthukile, uzokwazi ukukhetha ukuthi yimaphi amasayithi angaphansi azoba nokufinyelela kuleli khasi. Lokhu kuyasiza uma uhlela ukunikeza okuqukethwe okukhethekile okuhlukile kumazinga ahlukene e-plan, ngokwesibonelo.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Unokulawula okuphelele kokuthi leli khasi lizovela kuphi_

**Okubalulekile** : Zombili izimo kufanele zihlangatshezwe ukuze ikhasi liboniswe. Lokho kusho ukuthi uma umsebenzisi eyikhasimende le-plan ethile, kodwa engenayo enye yezindima ezikhethiwe (noma ngokuphambene nalokho) ikhasi ngeke livele kuphaneli yakhe.

## Iyasebenza

Uma ufuna ukukhubaza ikhasi lomphathi langokwezifiso ngaphandle kokulisusa noma ukuphazamisa izimvume, vele ususe ukukhethwa kwenketho ethi Iyasebenza.

Ukukhubaza ikhasi lezinga eliphezulu kuzokhubaza ngokuzenzakalelayo wonke amakhasi angaphansi axhunywe kulelo khasi lomzali.

<!-- Screenshot unavailable: Final result of custom admin page displayed on user panel -->

_Umphumela wokugcina kuphaneli yomsebenzisi!_
