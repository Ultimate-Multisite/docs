# Imibhalo ye-Ultimate Multisite {#ultimate-multisite-documentation}

Siyakwamukela emibhalweni ye-Ultimate Multisite. Ultimate Multisite iguqula i-WordPress Multisite ibe inkundla ephelele yeWebhusayithi Njengesevisi (WaaS).

## Izigaba Zemibhalo {#documentation-sections}

### [Umhlahlandlela Womsebenzisi](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

Konke okudingayo ukuze usethe, ulungiselele, futhi uphathe inkundla yakho ye-WaaS. Kuhlanganisa ukufaka, amasango okukhokha, ukuxhumanisa ama-domain, ukuphathwa kwamaklayenti, nokunye okuningi.

### [Imibhalo Yonjiniyela](/developer/hooks) {#developer-documentation}

Ireferensi yezobuchwepheshe yonjiniyela abakha phezu kwe-Ultimate Multisite. Kuhlanganisa ireferensi yama-hooks, imibhalo ye-REST API, nemihlahlandlela yokuthuthukisa izengezo.

### [Izengezo](/addons) {#addons}

Imibhalo yama-plugin ezingezo akhokhelwayo anweba i-Ultimate Multisite ngezici ezengeziwe ezifana nokuhlanganiswa kwe-WooCommerce, izibalo, ukuthengisa ama-domain, nokunye okuningi.

## Ukuthola Usizo {#getting-help}

- [Izinkinga ze-GitHub](https://github.com/Ultimate-Multisite/ultimate-multisite/issues) -- Bika amaphutha
- [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) -- Landa i-plugin eyinhloko
