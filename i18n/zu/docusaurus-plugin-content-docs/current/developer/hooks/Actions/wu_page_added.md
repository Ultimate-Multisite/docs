# Isenzo: wu_page_added

Vumela abathuthukisi be-plugin ukuthi basebenzise izinto ezengeziwe lapho amakhasi ebhaliswa.

Ngokungafani ne-wu_page_load, esebenza kuphela lapho ikhasi elithile libonwa, le hook isebenza ekubhalisweni kwawo wonke amakhasi e-admin angezwayo kusetshenziswa ikhodi ye-Ultimate Multisite.

## Amapharamitha

| Igama | Uhlobo | Incazelo |
|------|------|-------------|
| $page_id | `string` | I-ID yaleli khasi. |
| $page_hook | `string` | Igama le-hook laleli khasi. |

- 2.0.0
