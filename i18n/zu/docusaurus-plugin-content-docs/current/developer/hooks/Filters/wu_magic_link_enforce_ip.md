# Isihlungi: wu_magic_link_enforce_ip {#filter-wumagiclinkenforceip}

Hlunga ukuthi kuqiniswe yini ukuqinisekiswa kwekheli le-IP.

Setha ku-false ukuze uvumele ama-token ukuthi asebenze kusuka kumanethiwekhi ahlukene. Lokhu kwehlisa ukuvikeleka kodwa kwandisa ukusebenziseka (isb., kubasebenzisi beselula abashintsha amanethiwekhi).

## Amapharamitha {#parameters}

| Igama | Uhlobo | Incazelo |
|------|------|-------------|
| $enforce | `bool` | Ukuthi kuqiniswe yini ukufaniswa kwekheli le-IP. |
