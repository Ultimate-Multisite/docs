| $payment | `\WP_Ultimo\Models\Payment` | i-instance yenkokhelo ye-Ultimate Multisite. |
| $cart | `\WP_Ultimo\Checkout\Cart` | i-oda lamanje lenqola ye-Ultimate Multisite. |
