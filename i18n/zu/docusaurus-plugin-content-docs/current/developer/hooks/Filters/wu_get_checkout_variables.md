# Isihlungi: wu_get_checkout_variables {#filter-wugetcheckoutvariables}

Vumela abathuthukisi be-plugin ukuthi bahlunge izilungiselelo zangaphambili zekhasi lokukhokha.

Qaphela, okhiye abashodayo bangaphula ngokuphelele ukukhokha ku-front-end.

## Amapharamitha {#parameters}

| Igama | Uhlobo | Incazelo |
|------|------|-------------|
| $variables | `array` | Okuguquguqukayo okwenziwe kwasendaweni. |
| $checkout | `\Checkout` | Ikilasi lokukhokha. |

### Kusukela {#since}

- 2.0.0
### Umthombo {#source}

Kuchazwe ku-[`inc/checkout/class-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout.php#L1970) emgqeni 1970


## Okubuyiswayo {#returns}
I-array entsha yokuguquguqukayo.
