# Isihlungi: wp_ultimo_render_vars {#filter-wpultimorendervars}

Vumela abathuthukisi be-plugin ukuthi bengeze i-variable eyengeziwe kumongo we-render emhlabeni wonke.

## Amapharamitha {#parameters}

| Igama | Uhlobo | Incazelo |
|------|------|-------------|
| $args | `array` | I-array equkethe ama-variable adluliswa ubizo lwe-render. |
| $view | `string` | Igama le-view ezokwenziwa i-render. |
| $default_view | `string` | Igama le-fallback_view |

### Kusukela {#since}

- 2.0.0
