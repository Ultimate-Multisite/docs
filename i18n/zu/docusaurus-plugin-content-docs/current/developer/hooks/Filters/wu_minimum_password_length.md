# Isihlungi: wu_minimum_password_length

Hlunga ubude obuncane bephasiwedi.

Kuphoqelelwa kuphela lapho i-`wu_enforce_password_rules` iyi-true.

## Amapharamitha {#parameters}

| Igama | Uhlobo | Incazelo |
|------|------|-------------|
| $min_length | `int` | Ubude obuncane bephasiwedi. Okuzenzakalelayo 12 (kufana ne-Defender Pro). |
| $defender_active | `bool` | Ukuthi i-Defender Pro Strong Password iyasebenza yini. |

### Kusukela {#since}

- 2.4.0
