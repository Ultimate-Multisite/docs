# Isihlungi: wu_apply_plan_limits

Vumela abathuthukisi bezandiso ukuba badlule imikhawulo.

Ungasebenzisa lesi sihlungi ukuze usebenzise ikhodi oyithandayo ngaphambi kokuba noma yimiphi imikhawulo iqaliswe. Uma isihlungi sakho sibuyisa noma yiliphi inani eliyiqiniso, inqubo izoqhubeka; uma sibuyisa noma yiliphi inani elingelona iqiniso, ikhodi izobuya futhi akukho neyodwa kuma-hook angezansi ezosebenza.
