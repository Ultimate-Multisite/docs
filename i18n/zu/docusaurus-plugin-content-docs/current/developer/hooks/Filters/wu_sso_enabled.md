# Isihlungi: wu_sso_enabled

Vumela/khubaza ikhono le-single-sign-on phakathi kwama-domain.

Hlunga leli nani ukuze uvale ngokuphelele i-single-sign-on, noma uyivumele ngokwemibandela esikhundleni salokho.

## Amapharamitha

| Igama | Uhlobo | Incazelo |
|------|------|-------------|
| $enabled | `bool` | Ingabe i-SSO kufanele ivunyelwe? True uma ivuliwe, false-ish uma ivaliwe. |

### Kusukela

- 2.0.11
### Umthombo

Ichazwe ku-[`inc/sso/class-sso.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/sso/class-sso.php#L110) emgqeni 110


## Okubuyiswayo
Ukuthi i-SSO ivunyelwe noma cha.
