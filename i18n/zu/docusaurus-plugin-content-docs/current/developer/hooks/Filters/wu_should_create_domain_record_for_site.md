# Isihlungi: wu_should_create_domain_record_for_site

Sihlunga ukuthi i-Ultimate Multisite kufanele yini idale irekhodi lesizinda lesayithi elisanda kudalwa.

Sebenzisa lesi sihlungi ukuvimbela noma ukuhlehlisa ukudalwa okuzenzakalelayo kwerekhodi lesizinda kumasayithi asebenzisa isizinda esiyisisekelo sefomu le-checkout elabiwe, i-host yangaphakathi, noma isizinda esizophathwa ngokwehlukile okunye ukuhlanganiswa.

## Amapharamitha

| Name | Type | Description |
|------|------|-------------|
| $create | `bool` | Ukuthi irekhodi lesizinda kufanele lidalwe yini. |
| $site | `WP_Site` | I-objekthi yesayithi esanda kudalwa. |

### Kusukela

- 2.13.0

### Umthombo

Kuchazwe ku-`inc/functions/domain.php`.


## Okubuyiswayo

I-Boolean ebonisa ukuthi kudalwe yini irekhodi lesizinda.
