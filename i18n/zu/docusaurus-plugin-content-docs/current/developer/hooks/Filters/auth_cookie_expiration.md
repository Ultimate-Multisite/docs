# Isihlungi: auth_cookie_expiration

Sihlunga ubude besikhathi sokuphelelwa yisikhathi se-cookie yokuqinisekisa.

## Amapharamitha

| Igama | Uhlobo | Incazelo |
|------|------|-------------|
| $length | `int` | Ubude besikhathi sokuphelelwa yisikhathi ngemizuzwana. |
| $user_id | `int` | User ID. |
| $remember | `bool` | Ukuthi kufanele kukhunjulwe ukungena komsebenzisi. Okuzenzakalelayo kungu-false. |

- 2.8.0
