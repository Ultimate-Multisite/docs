# Isihlungi: wu_checkout_form_base_domains {#filter-wucheckoutformbasedomains}

Sihluza izizinda eziyisisekelo zefomu le-checkout ezabelwanayo ezingafanele zibe amarekhodi ezizinda ezimephiwe.

Sebenzisa lesi sihlungi lapho ukuhlanganiswa kunikeza izizinda ezengeziwe eziyisisekelo zezinkambu ze-**URL Yesayithi** zefomu le-checkout. Izizinda ezibuyiswa yilesi sihlungi ziphathwa njengama-host okubhalisa abelwanayo esikhundleni sezizinda ezenziwe ngokwezifiso zesayithi ngalinye.

## Amapharamitha {#parameters}

| Igama | Uhlobo | Incazelo |
|------|------|-------------|
| $domains | `array` | Izizinda eziyisisekelo ezabelwanayo eziqoqwe ekucushweni kwefomu le-checkout. |

### Kusukela {#since}

- 2.13.0

### Umthombo {#source}

Kuchazwe ku-`inc/functions/domain.php`.


## Okubuyiswayo {#returns}

Array yezizinda eziyisisekelo zefomu le-checkout ezijwayelekile.
