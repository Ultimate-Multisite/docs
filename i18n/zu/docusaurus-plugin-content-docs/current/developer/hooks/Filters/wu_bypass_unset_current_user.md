# Isihlungi: wu_bypass_unset_current_user

Vumela abathuthukisi ukuthi badlule ikhodi yokususa ukusethwa komsebenzisi wamanje.

Ukubuyisa noma yini ngaphandle kuka-null kuzodlula ukususa ukusethwa komsebenzisi wamanje ongene ngemvume. Lokhu kungaba wusizo kwezinye izimo, ngokwesibonelo, lapho kusebenzelwana nama-sub-sites asetshenziswa njengamaphaneli okuphatha.

## Amapharamitha

| Igama | Uhlobo | Incazelo |
|------|------|-------------|
| $null_or_bypass | `mixed` | Null ukuze uqhubeke, noma yini enye ukuze uyidlule. |
| $current_user | `false\|\WP_User` | Into yomsebenzisi wamanje. |
