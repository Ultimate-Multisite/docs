# Isihlungi: wu_minimum_password_strength

Hlunga amandla amancane ephasiwedi adingekayo (isilinganiso se-zxcvbn).

Amazinga wamandla: - 0, 1: Buthakathaka kakhulu - 2: Buthakathaka - 3: Maphakathi - 4: Buqinile (okuzenzakalelayo)

## Amapharamitha {#parameters}

| Igama | Uhlobo | Incazelo |
|------|------|-------------|
| $min_strength | `int` | Izinga lamandla elincane elidingekayo. |
| $strength_setting | `string` | Inani lesethingi ye-admin (medium, strong, super_strong). |
