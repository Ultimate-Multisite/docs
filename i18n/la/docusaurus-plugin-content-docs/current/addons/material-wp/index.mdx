# Material WP

Administratoria interfacies WordPress per saecula eadem mansit. Sed umquamne miratus es qualis videretur versio Material Design administratoriae interfaciei? 

Material WP est thema administratorium elaboratum ut tibi illum aspectum sensumque det, te adiuvans ut clientibus tuis Dashboard magis ad usum accommodatum praebeas.

## Proprietates {#features}

- Sustentatio CSS consuetudinarii
- Faciles optiones ornatus
- Ordinis tabulae mutatio
- Facilis importatio/exportatio

## Documentatio {#documentation}

**Tuam Tabulam Administratoriam cum Material WP accommodare**

## Installatio {#installation}

1. Upload 'material-wp' ad directorium '/wp-content/plugins/'
2. Activa plugin per tabulam 'Plugins' in WordPress
