# Creator Paginae Administratoriae

## Materia
Creator paginae administratoriae sinet te facile novas paginas administratorias et subpaginas intra WordPress tabulam administratoriam addere. Adhiberi potest ad creandam paginam salutatoriam vel paginam institutionis, quae pro pagina adventus novis usoribus servire potest. Potes aedificatorem paginarum sicut Beaver Builder, Elementor, Brizy, et Oxygen adhibere ut paginas sine difficultate designes.

Etiam praebet auxilium HTML, CSS et JavaScript proprium, necnon moderationem accessus secundum consilium et munus usoris!

## Proprietates
* Auxilium HTML et CSS proprii
* Auxilium JavaScript
* Modus Materiae PHP
* Permissio secundum munus Usoris
* Auxilium aedificatoribus paginarum

## Documentatio
[Paginae administratoriae propriae et subpaginae creandae utens Multisite Ultimate: Creatore Paginae Administratoriae](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## Installatio

1. Insere **ultimate-multisite-admin-page-creator** in directorium **/wp-content/plugins/**
2. Activa plugin per menu **Pluginorum** in tua Administratione Retis WordPress

## Dux Usus

Dum WordPress facillimum facit materiam ad partem anteriorem sitorum addere, creare paginas et subpaginas in parte administratoria nonnullas artes fundamentales codicis requirit. Ultimate Multisite: Creator Paginae Administratoriae idem usus facilitatis partis anterioris ad partem posteriorem sitorum retis tui afferre intendit.

## Cur Creatore Paginae Administratoriae uti?

Fortasse vis paginam propriam in tabula administratoria clientium tuorum cum institutionibus offerre; fortasse vis paginam propriam cum quaestionibus frequentibus et instrumento contactus addere. Possibilitates sine fine sunt cum Creatore Paginae Administratoriae, et id est quod in institutione hodierna explorabimus.

## Primae Paginae Administratoriae Propriae creandae

Postquam Ultimate Multisite: Creatorem Paginae Administratoriae installaveris et activaveris, pete ad menu administrationis retis tui et vade ad Paginas Administratorias.

## Optiones Menu

In latere, in arca meta Optionum Generalium, poteris configurare genus menu, inscriptio, et multo plura.

### Genera Menu

Nova tua pagina administratoria propria duo genera menu habere potest: menu summi gradus et submenu.

Pagina menu summi gradus, ut nomen indicat, paginam tuam nuper creatam ad menu administratorium iuxta paginas WordPress praedefinitas addet.

Si optionem summi gradus eligis, etiam ordinem menu et iconem menu eligere poteris.

Si vis hanc paginam paginam sub-menu facere, paginam parentem eligere poteris cui haec pagina adiungatur. Paginam tuam propriam addere potes ad paginas WordPress praedefinitas aut ad paginas administratorias proprias summi gradus tuas.

<!-- Screenshot unavailable: Admin Page Creator menu type options with label, order, and icon settings -->

_Plena ratione apta elementum menu pro hac Pagina Propria_

<!-- Screenshot unavailable: Admin Page Creator sub-menu parent page selection -->

_Hanc paginam etiam addere potes ut sub-menu aliarum paginarum menu, inclusis tuis propriis summi gradus._

## Optiones Provectae

Debes posse omnia genera materiae ad paginas administratorias proprias addere. Id includit regulas et fasciculos CSS, JavaScript, atque bibliothecas externas quoque. Id facere potes utens Tabula Optionum Provectarum.

### CSS Proprium

In tabula CSS, potes codicem CSS proprium tuum addere, necnon fasciculos CSS e sitis remotis importare (sicut CDNs, Google Fonts, etc).

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_CSS proprium quoque sustinetur. Potes etiam fasciculos stili externos includere!_

### JavaScript Proprium

Idem valet pro JavaScript. Potes bibliothecas externas addere adhibendas in codice JavaScript proprio tuo.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Addere codicem JavaScript proprium tuum!_

## Permissiones

Etiam in Optionibus Provectis, eligere poteris quae subsita ad hanc paginam accessum habitura sint. Hoc utile est si cogitas materiam exclusivam diversam diversis gradibus consilii ministrare, exempli gratia.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Plena potestas tibi est ubi haec pagina apparitura sit_

**Grave** : Utraque condicio implenda est ut pagina ostendatur. Hoc significat, si usor cliens cuiusdam consilii est, sed unum ex muneribus selectis non habet (aut e contrario), pagina in eius tabula non apparebit.

## Activa

Si vis paginam administratoriam custom delere sine ea delenda aut permissiones perturbando, modo optionem Active deselege.

Inactivatio paginae summi gradus sponte omnes subpaginas illi paginae parenti adnexas inactivabit.
