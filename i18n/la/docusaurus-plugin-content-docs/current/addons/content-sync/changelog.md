# Diarium Mutationum Synchronizationis Contenti {#content-sync-changelog}

### Versio 1.1.0 - Emissa die 2025-12-31 {#version-110---released-on-2025-12-31}
* Novum: Sustentatio ad synchronizanda omnia genera publicationum (publicationes, paginae, genera publicationum propria)
* Novum: Filtratio generis publicationis in gradu selectionis contenti
* Novum: Gradus selectionis situs fontis - synchroniza ex quovis situ, non solum ex situ principali
* Novum: Praeselectio situs filii pro sitibus exemplaribus
* Novum: Extractio mediorum ex post_content pro omnibus generibus publicationum
* Novum: Synchronizatio taxonomiarum (categoriae, pittacia, taxonomiae propriae)
* Mutatum: Nomen mutatum est ex "Synchronizatio Exemplarium Elementor" in "Synchronizatio Contenti"
* Mutatum: Gradus "Selige Exemplaria" renominatus est "Selige Data"
* Emendatum: Codex Elementor-specificus nunc tantum currit pro genere publicationis elementor_library

### Versio 1.0.0 - Emissa die 2025-12-19 {#version-100---released-on-2025-12-19}
* Prima emissio
* Magus plurium graduum ad selectionem exemplaris et situs
* Processus per fasciculos cum responsione de progressu tempore reali
* Plena synchronizatio mediorum cum automatica remappatione ID
* Integratio configurationum cum Ultimate Multisite
* Sustentatio exclusionis situs per configurationes retis
* Purgatio memoriae temporariae Elementor post synchronizationem
* Conservatio taxonomiae exemplarium
