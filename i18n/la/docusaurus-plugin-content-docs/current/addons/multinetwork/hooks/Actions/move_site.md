# Actio: move_site {#action-movesite}

Excitatur postquam situs ad novum rete motus est.

## Parametra {#parameters}

| Nomen | Typus | Descriptio |
|------|------|-------------|
| $site_id | `int` | ID situs qui motus est. |
| $old_network_id | `int` | ID retis originalis situs. |
| $new_network_id | `int` | ID retis ad quod situs motus est. |

### Inde a {#since}

- 1.3.0
