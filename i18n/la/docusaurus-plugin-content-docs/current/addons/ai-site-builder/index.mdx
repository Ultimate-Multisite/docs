# Aedificator Situs AI

ultimate-multisite-ai-site-builder

## Requisita {#requirements}

- WordPress 5.3 aut recentior
- PHP 7.4 aut recentior
- plugin Multisite Ultimate (activum)

## Institutio {#installation}

1. Fasciculos additamenti in directorium tuum `/wp-content/plugins/` incarga
2. plugin per tabulam 'Plugins' in WordPress activa
3. Optiones additamenti in administratione retis configura

## Evolutio {#development}

### Apparatus {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Structura Projecti {#project-structure}

```
ultimate-multisite-ai-site-builder/
├── inc/                          # Classes PHP et inclusa
│   ├── class-multisite-ultimate-updater.php  # Tractator renovationis
│   └── class-ai_site_builder-main.php             # Functionalitas principalis
├── tests/                        # Probationes unitatis
│   ├── bootstrap.php            # Initium probationis
│   ├── class-*-base.php         # Classis probationis basis
│   └── class-*-test.php         # Casus probationis
├── views/                       # Fasciculi exemplarium
├── assets/                      # Res staticae
│   ├── css/                     # Folia stilorum
│   ├── js/                      # Fasciculi JavaScript
│   └── images/                  # Imagines
├── lang/                        # Fasciculi linguae
├── composer.json                # Dependentiae PHP
├── package.json                 # Dependentiae Node.js
├── phpunit.xml.dist             # Configuratio PHPUnit
├── .phpcs.xml.dist              # Configuratio normarum codicis
└── rector.php                   # Regulae modernizationis codicis
```

### Probatio {#testing}

Additamentum compagem probationis comprehensivam includit:

- **Probationes Unitatis**: Probationes PHPUnit fundatae cum serie probationis WordPress
- **Classis Probationis Basis**: Methodos auxiliares praebet pro communibus condicionibus probationis
- **Tegmen Probationis**: Configuratum ad tegmen codicis investigandum
- **Probatio Multisite**: Probationes in ambitu multisitus currunt

### Normae Codicis {#code-standards}

- **Normae Scribendi Codicis WordPress**: Per PHPCS coguntur
- **Compatibilitas PHP 7.4+**: Proprietates modernae PHP sustentantur
- **Analysis Statica**: Integratio PHPStan ad verificationem typorum
- **Modernizatio Codicis**: Regulae Rector pro promotionibus PHP

### Systema Aedificationis {#build-system}

- **Processio Rerum**: Minificatio pro fasciculis CSS/JS
- **Translatio**: Generatio fasciculi POT
- **Confectio Fasciculi**: Creatio archivi ad distributionem
- **Modus Evolutionis**: Res non minificatae ad depurationem

## Configuratio {#configuration}

Additamentum varias optiones configurationis sustinet, per interfaciem administrationis retis accessibiles.

## Hooks et Filtra {#hooks-and-filters}

### Actiones {#actions}

- `ultimate-multisite-ai-site-builder_init` - Emittitur post initializationem additamenti
- `ultimate-multisite-ai-site-builder_loaded` - Emittitur postquam omnes fasciculi additamenti onerati sunt

### Filtra {#filters}

- `ultimate-multisite-ai-site-builder_settings` - Optiones additamenti modifica
- `ultimate-multisite-ai-site-builder_enabled` - Statum activationis/deactivationis additamenti supera
