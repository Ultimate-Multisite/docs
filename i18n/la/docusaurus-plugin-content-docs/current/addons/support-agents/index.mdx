# Support Agents {#support-agents}

Totum rete per te ipsum administrare non est munus facile. Cum rete tuum crescere incipit, crescentem postulationem auxilii videre incipies. Bonum consilium erit homines adducere qui te onere laboris levare adiuvabunt. Tamen, plenam accessum/permissionem dare periculosum est et rete tuum in discrimen adducere potest. Hic est ubi hoc additamentum Support Agent intervenit.

Hoc additamentum sinit te novum genus usoris WordPress creare, quod Support Agent appellatur, atque tibi plenam potestatem dat de quibus permissionibus ille usor habeat. Permissionem concedere tam facile est quam globulum commutationis accendere, atque eum exstinguere si eam removere statueris.

## Proprietates {#features}

- Index per res digestus permissionum WordPress et Ultimate Multisite
- Accessum facile permittere et inhibere
- Numerum infinitum Account agentium creare

## Institutio {#installation}

1. 'ultimate-multisite-support-agents' in directorium '/wp-content/plugins/' immitte
2. Plugin per indicem 'Plugins' in WordPress activa
3. Wizard gradatim sequere ut plugin constituas
