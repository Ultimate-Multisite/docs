## Limitatio Frequentiae

Ultimate Multisite: Captcha v1.5.0 inducit limitatorem frequentiae cum sistendo absoluto, qui rete tuum ab impetibus brute-force et probationis chartarum protegit.

### Quomodo Operatur

Limitator frequentiae omnem petitionem GET et POST in superficiebus captcha-protectis numerat:

* **Puncta terminalia initii sessionis WordPress** — wp-login, register, lost-password, comments
* **Checkout WooCommerce** — my-account, checkout, pay-for-order
* **Ultimate Multisite** — checkout, inline-login

Cum visitator limitem frequentiae excedit, plugin respondet cum:

* Codice status **HTTP 429 (Nimis Multae Petitiones)**
* Header **Retry-After** indicante quando iterum temptandum sit
* **Somno tarpit fortuito** (1–5 secundis per defaltam, limite duro ad 15 secunda) ut oppugnatores retardentur

### Configuratio

#### Tempus Tarpit

Rege moram fortuitam petitionibus limite frequentiae affectis applicatam:

* **`cap_rate_limit_tarpit_min`** — Duratio minima somni in secundis (defaltum: 1)
* **`cap_rate_limit_tarpit_max`** — Duratio maxima somni in secundis (defaltum: 5, limite duro ad 15)

Hae optiones praesto sunt in **Ultimate Multisite → Optiones → Captcha**.

### Extensibilitas

#### Filtrum Indicis Albi IP

Exime ambitus IP fideles a limitatione frequentiae:

```php
add_filter( 'wu_cap_rate_limit_whitelist_ip', function( $is_whitelisted, $ip ) {
    // Whitelist your monitoring service
    if ( $ip === '203.0.113.42' ) {
        return true;
    }
    return $is_whitelisted;
}, 10, 2 );
```

#### Actio Ante Obstructionem

Exsequere logicam propriam statim antequam obstructio limitis frequentiae mittatur:

```php
add_action( 'wu_cap_rate_limit_will_block', function( $ip, $surface ) {
    // Log the block, notify admins, or update custom metrics
    error_log( "Rate limit triggered for $ip on $surface" );
}, 10, 2 );
```

## Detectio IP Clientis et Procuratores Fideles

Ultimate Multisite: Captcha v1.5.0 inducit detectionem IP clientis simulationi resistantem. Methodus `Captcha_Core::get_client_ip()` nunc est unica fons veritatis attributionis IP visitatoris per:

* Claves sitularum limitis frequentiae
* Provisor Captcha `remoteip` (reCAPTCHA, hCaptcha `siteverify`)
* Excerpta IP statistica

### Modus Fiduciae

Additamentum strictam hierarchiam fiduciae IP exigit:

1. **REMOTE_ADDR** — Semper creditum (IP paris immediati)
2. **CF-Connecting-IP** — Observatur tantum cum:
   - Par immediatus intra hodiernum intervallum IP Cloudflare est
   - `cap_trust_cloudflare_headers` activatum est (praedefinitum: OFF)
3. **X-Forwarded-For** — Observatur tantum cum:
   - Par immediatus in indice procuratorum creditorum a administratore tuo configurato est
   - Caput a dextra ad sinistram enodatur, saltibus creditis/Cloudflare praetermissis

### Configuratio

#### Fiducia Capitis Cloudflare

Activa fiduciam capitis `CF-Connecting-IP` Cloudflare:

* **`cap_trust_cloudflare_headers`** — Constitue ad ON ut fiduciam `CF-Connecting-IP` eligas
  - Praedefinitum: OFF (deactivatum)
  - Additamentum cum fasciculo incluso instantaneae CIDR Cloudflare traditur
  - Intervalla CIDR singulis hebdomadibus per wp-cron renovantur
  - Subsidium inclusum adhibetur si renovatio deficit

#### Configuratio Procuratorum Creditorum

Defini procuratores et libratores oneris primae lineae tuos:

* **`cap_trusted_proxies`** — Area textus CIDR aut IP nudorum (unum per lineam)
  - Commentarii permittuntur (lineae quae a `#` incipiunt)
  - Sine hac configuratione, `X-Forwarded-For` neglegitur etiam cum limitator frequentiae activatus est
  - Exemplum:
    ```
    # Our load balancer
    192.0.2.0/24

    # Backup proxy
    198.51.100.50
    ```

#### Auto-Detectio Prima Activatio

Cum limitatorem frequentiae primum activas, additamentum habitum tuum probabilem Cloudflare et procuratoris deprehendit atque nuntium administratoris unius-click "Adhibe configurationes detectas" ostendit.

* Additamentum valores servatos tuos **numquam superscribit**
* Si commeatus subsequens indicat configurationem tuam iam cum re non congruere (exempli gratia, Cloudflare intervalla CIDR mutavit), nuntius discrepantiae non dimittendus renovationem commendatam ostendit

### Cur Hoc Interest

Strictus modus fiduciae IP vectorem criticum falsificationis claudit: petitiones directa ad ministratorem originis quae caput `CF-Connecting-IP` fictum ferebant antea secundum IP fictum potius quam verum parem in situla positae essent. Hoc praesertim magni momenti est pro:

* **Limitatio frequentiae** — Impugnatores limites fingendis IPs vitare non possunt
* **Statistica** — Mensurae impugnationum tuae IPs visitatorum veros referunt, non fictos
* **Verificatio Captcha** — Provisores IP visitatoris rectum accipiunt ad aestimationem periculi
