# Ultimate AI Connector pro WebLLM Index Mutationum {#ultimate-ai-connector-for-webllm-changelog}

## Versio 1.1.0 — Edita die 2026-04-09 {#version-110--released-on-2026-04-09}

* Novum: Instrumentum sermonis fluctuans cum indice status admin-bar — quivis usor iam ingressus potest LLM in parte navigatorii directe ex fronte interrogare.
* Novum: Tempus exsecutionis SharedWorker — machina MLC nunc in SharedWorker currit, ut plures tabulae unam sessionem GPU communicent pro eo ut inter se de ea certent.
* Novum: Interceptor medius apiFetch — petitiones WordPress REST quae exemplari AI Client SDK respondent, perspicue ad localem WebLLM mediatorem diriguntur, nullo itinere HTTP loopback redeunte requisito.
* Novum: UI optionum instrumenti in tabula Connector ad instrumentum sermonis commutandum et mores auto-prompt configurandos.
* Emendatio: Coge backend horrei IndexedDB ut demissiones ponderum exemplaris supersint redirectionibus HuggingFace xet CDN quae viam praedefinitam Cache API frangunt.
* Emendatio: Praeteri substitutionem context_window KV-cache pro exemplaribus embedding (decodificatore carent et substitutio errorem temporis exsecutionis movebat).
* Emendatio: Exemplar candidatum initii frigidi in /webllm/v1/models annuntia antequam tabula operarii onerata sit, ut consumptores SDK exemplar statim videant.
* Emendatio: WebLLM in filtrum wpai_preferred_text_models inseri, ut proprietas AI Experiments ad machinam navigatorii dirigatur cum exemplar configuratum est.
* Melioratum: Renovatio cache-busting, normalizatio contenti, et emendationes relationum ferramentorum per probationem a principio ad finem patefactae.
