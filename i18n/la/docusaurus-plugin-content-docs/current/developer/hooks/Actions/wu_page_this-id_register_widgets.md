# Action: wu_page_\{$this->id\}_register_widgets {#action-wupagethis-idregisterwidgets}

Excitatur postquam widgets pro hac pagina registrata sunt.

The dynamic portion of the hook name, `$this-&gt;id`, refers to the page id.

## Parametra {#parameters}

| Nomen | Typus | Descriptio |
|------|------|-------------|
| $id | `string` | Id paginae. |
| $page_hook | `string` | Hook paginae. |
| $page | `object` | Obiectum paginae. |
