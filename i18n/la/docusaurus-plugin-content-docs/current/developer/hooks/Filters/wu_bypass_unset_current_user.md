# Filtrum: wu_bypass_unset_current_user {#filter-wubypassunsetcurrentuser}

Permitte evolutoribus codicem qui usorem praesentem removet praeterire.

Si quid aliud quam null redditur, remotio usoris praesentis qui in sessione est praeteribitur. Hoc in nonnullis condicionibus utile esse potest, exempli gratia, cum agitur de sub-sitis quae ut tabulae administratorum adhibentur.

## Parametri {#parameters}

| Nomen | Typus | Descriptio |
|------|------|-------------|
| $null_or_bypass | `mixed` | Null ad progrediendum, quidlibet aliud ad hoc praetereundum. |
| $current_user | `false\|\WP_User` | Obiectum usoris praesentis. |
