# Incompatibilitates noti omnis Pluginum aliorum {#known-incompatibilities-with-other-plugins}

Post installation et activation Ultimate Multisite, potes incidere in quaestionibus causa incompatibilitatum non adhibita cum aliis pluginibus.

Acta diligentia laboramus ut hoc listum minuamur cum omnibus novis versionibus, sed si plugin in hoc listu est, possuris esse necessarius deactivus eius ut network Ultimate Multisite plenius functionalis sit.

Hic est listum et quid debas facere in tales casibus:

**Alia WPMU DEV plugins** Alii pluginus qui iam non curant WPMU DEV sunt noti quod fluxus registrationis Ultimate Multisite ad finem (utilis non potest computare) obstant. Hic est listum pluginium quae illas quaestiones causant:

  * Signup Language: _Ultimate Multisite habet addendum quod eam ipsam faciliam, appellatum Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Hoc in Ultimate Multisite est construum, ita non est necessarius plugin extra._

  * New Blog Templates: _Hoc in Ultimate Multisite est construum, ita non est necessarius plugin extra._

  * Set Password on Multisite Blog Creation: _Hoc in Ultimate Multisite est construum, ita non est necessarius plugin extra._

  * Signup TOS: _Hoc in Ultimate Multisite est construum, ita non est necessarius plugin extra._

  * Anti-Splog: Quamquam minus potentior, hoc etiam _in Ultimate Multisite est construum._

**Nota**: Omnes alii pluginus WPMU Dev normaliter cum Ultimate Multisite adhiberi possunt. Exempla includunt _Smush_, _Forminator_, _Defender_, etc.
