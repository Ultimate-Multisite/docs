# Webbi Clientes Tuorum (v2)

Postquam in network tuam registri, clienti tuorum ad dashboard access possunt.

Alia quam in pagina **Account**, ubi omnia informa tationes importantes de paymentibus, membrschaftibus, dominis, limitationibus planum et aliis videri possunt, ad **My Sites** paginam etiam access habent, ubi omnia webbia tua videri et gerere possunt. Omnia quod debent facere est cliquare in My Sites in dashboard suum.

![Link My Sites in dashboard clientis](/img/account-page/my-sites-link.png)

Omnes webbia tuas eis erunt ostendae, cum **Manage** boton justo sub omnibus eorum. Si hoc clicent, ad **dashboard huius specifici website** redirecuntur.

Possunt etiam videre quam quale ex omnibus webbiis tuae est **primarius** pro planibus quae permittunt eos habere plure webbia.

![My Sites pagina ostendens webbias clientium cum boton Manage](/img/account-page/my-sites-page.png)

Hic est vultus completus interface editio siti:

![Interface editio siti completa](/img/admin/site-edit-full.png)

Si vis primum tuum website mutare, debet ad paginam Account ire per cliquem in menu **Account**.

![Link menu Account in dashboard clientis](/img/account-page/account-menu.png)

Possunt etiam details clientium videri et editum:

![Interface editio clientis](/img/admin/customer-edit.png)

Deinde, debent cliquare in **Change Default Site**.

![Boton Change Default Site in pagina account](/img/account-page/change-default-site.png)

Fenestra pop-up apparient. In lista drop-down cliqua et eligite quam website primum website facere vis.

![Selecitio dropdown Change Default Site](/img/account-page/change-default-site-dropdown.png)

Deinde, solum cliqua in **Change Default Site** et finis!

![Confirmans mutamenti situi default](/img/account-page/change-default-site-confirm.png)
