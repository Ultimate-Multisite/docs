# Quommodo uti snippets Ultimate Multisite in repository GitHub nostro {#how-to-use-ultimate-multisite-snippets-on-our-github-repository}

In repository GitHub sunt snippet codicis quae saepe solicitantur ab utilisateur Ultimate Multisite qui parvum operationes addere desiderant, sicut additionis scripti Google Analytics in paginis sign-up vel occulsi meta box a dashboard admin.

Hoc opus tibi monstrabit quomodo uti vel plus specificiter ubi quos codices collocare debas.

Snippets invenies in link inferiori.

https://github.com/next-press/wp-ultimo-snippets/

Duo modi ad codicem addere sunt:

  1. In filem functions.php temi tuum.

  2. Must-Use Plugins (mu-plugins)

# Quommodo snippet in filem functions.php temi tuo addere debet. {#how-to-add-the-snippet-on-your-themes-functionsphp-file}

  1. Accede ad dashboard admin Network WordPress tuum et ad Theme > Theme Editor (Vide screenshot inferior).

  2. In pagina "Edit Themes", assura te quod theme activum tuum selectum est in campo dropdown situato in angulo dextro superior screen tui (#3 in screenshot inferior).

  3. Fac ut cliccas in filem functions.php sub sectione "Theme Files" ut filium careres. Descendi ad inferem et incolla snippet Ultimate Multisite quum a repository GitHub habes.

# Quommodo Must-Use Plugins (mu-plugins) creare debet. {#how-to-create-must-use-plugins-mu-plugins}

WordPress habet modum qui tibi permittit carere operationes custom, quae vocant "Must-Use Plugins", vel "mu-plugins" breviter.

Hae speciales mu-plugins ante omnes plugin regulares carantur, et non deactivari possunt. In network multisite, codex in hais mu-plugins in omnibus locis installation tui caretur.

1. Utiliza FTP aut SSH ad accessum systemae filem installation WordPress tui.

2. Ine directorium wp-content de installation WordPress tuas, crea un novo directorium nominatus: mu-plugins.

<!-- Screenshot unavailable: File manager showing wp-content directory with mu-plugins folder -->

3. Crea un novum file PHP in computat tuo nominatum wu-snippet.php uti Notepad vel quolibet editor codicis.

4. Pon snippet codicis Ultimate Multisite quem accipi de GitHub repository in file et salva. Etiam hoc codicem super codicem addere potes ut tuum mu plugin labellare.
