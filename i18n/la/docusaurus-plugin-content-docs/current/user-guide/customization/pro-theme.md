# Customizatio panelis Tuum Cum PRO Theme {#customizing-your-admin-panel-with-pro-theme}

Frequentemente cum operibus pro clientibus necessitas habemus ut clientibus nostris experientiam in dashboard WordPress personalem daremus. Si quaeritis aspectum et sensum melius professionale, PRO Theme tibi adiuvat.

PRO Theme est thema admin sviluppatum ad te auxilium dare in customizatione Dashboard clientium tuorum. Etiam est pars core (core) pluginium premium Ultimate Multisite.

Post installationem, ad paginam _Plugins_ tua adveniens et clica ad activandam in Network tuo.

<!-- Captura non est disponibilis: WP Plugins page ostendens button Activare pro PRO Theme in Network -->

_Activare in omni network tuo in pagina Plugins_

_**NOTA:**_ _Si id solum installas, in clientibus tuis ut optionem ostenditur quam possis marcare ad themam unum ut hoc usum esse solo illius utente._

Nunc, interfaced admin tuam completus redesignatus est! Etiam habes novum menum in columna sinistra appellatum PRO Theme.

<!-- Captura non est disponibilis: Interfacies admin PRO Theme redesignata -->

_Nova forma pro panel tuo admin_

In hoc menu accessus ad quinque tabs habes:

<!-- Captura non est disponibilis: Tabula Configuratio Stylorum PRO Theme -->

_Styling Settings_ (Configuratio Stylorum)

<!-- Captura non est disponibilis: Tabula Functionalis PRO Theme -->

_Functionality_ (Functionalis)

<!-- Captura non est disponibilis: Tabula CSS Customis PRO Theme -->

_Custom CSS_ (CSS Customis)

<!-- Captura non est disponibilis: Tabula Import/Export PRO Theme -->

_Import/Export_ (Importare/Exportare)

<!-- Captura non est disponibilis: Tabula Activare PRO Theme -->

_Activate PRO Theme_ (Activare PRO Theme)

In omnibus eis, possis item stylorum activare et deactivare, functiones permittere vel bloccare, et clientibus tuis customizationem dare quod quaerunt in network sitiorum tuorum.
