# Ultimate Multisite Traducere

Ultimate Multisite est développé en anglais, mais le plugin est entièrement traduisible et nous accueillons activement les contributions de traduction.

## Traductions du Plugin Principal {#core-plugin-translations}

Le plugin principal Ultimate Multisite est traduit via la plateforme officielle **WordPress.org translation platform** :

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

C'est le système de traduction communautaire standard de WordPress. Lorsque les traductions atteignent un seuil d'approbation suffisant, elles sont automatiquement distribuées à tous les utilisateurs d'Ultimate Multisite via les mises à jour des plugins WordPress.

**Nous avons besoin de votre aide !** Si vous parlez une langue autre que l'anglais, veuillez envisager de contribuer avec des traductions. Même traduire quelques chaînes aide. Pour commencer :

1. Visitez [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)
2. Sélectionnez votre langue
3. Cliquez sur "Contribute Translation" (Contribuer à la traduction)
4. Vous aurez besoin d'un compte gratuit WordPress.org

Si vous souhaitez devenir éditeur de traduction pour votre langue (capable d'approuver les traductions), contactez-nous via [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions).

## Traductions des Addons {#addon-translations}

Les plugins add-on sont traduits séparément sur :

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Les traductions des addons sont actuellement générées par IA comme point de départ. Elles couvrent toutes les langues prises en charge mais peuvent contenir des formulations maladroites ou des inexactitudes. Nous accueillons les traducteurs humains pour examiner et suggérer des améliorations afin de les rendre plus naturelles et précises.

## Traductions Locales {#local-translations}

Si preferes crear traducciones localmente para tu propio proyecto en lugar de contribuir a la fuente principal, tienes dos opciones:

### Usando Poedit {#using-poedit}

Puedes usar la aplicación de escritorio **[Poedit](https://poedit.net)** para traducir el archivo `.pot`:

1. Descarga Poedit desde [poedit.net](https://poedit.net)
2. Abre el archivo `.pot` desde `wp-content/plugins/ultimate-multisite/lang/`
3. Traduce las cadenas en la interfaz de Poedit
4. Guarda -- Poedit genera los archivos `.po` y `.mo`
5. Coloca los archivos resultantes en la carpeta `lang`

### Usando Loco Translate {#using-loco-translate}

El plugin de WordPress **[Loco Translate](https://wordpress.org/plugins/loco-translate/)** te permite traducir directamente desde el panel de administración sin necesidad de herramientas de escritorio. Se encarga de crear y compilar los archivos de traducción por ti.
