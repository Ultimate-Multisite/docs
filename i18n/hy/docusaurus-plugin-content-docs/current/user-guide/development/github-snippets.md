# Մեծ կողմերով ցանցի (Ultimate Multisite) սնուպետների օգտագործումը մեր GitHub-ում

GitHub-ում հասանելի է կոդային հատվածներ, որոնք հաճախ խնդրվում են Ultimate Multisite օգտվողների կողմից՝ փոքր ֆունկցիոնալություններ ավելացնելու համար, ինչպիսիք են Google Analytics սկրիպտի ավելացում գրանցման էջերում կամ մետաբոքսը հեռացնելու ადմինի պաստառից։

Այս հոդվածը կցույց տա, թե ինչպես օգտագործել այդ կոդերը կամ ավելի կոնկրետ՝ որտեղ դրանք տեղադրել։

Սնուպետները կարող եք գտնել ստորև բերված հղումով.

https://github.com/next-press/wp-ultimo-snippets/

Կոդը ավելացնելու համար կան 2 տարբերակ.

  1. Ձեր թեմայի `functions.php` ֆայլում։

  2. Must-Use Plugins (mu-plugins)

# Ինչպես ավելացնել սնուպետը ձեր թեմայի functions.php ֆայլում։

  1. Մուտք գործեք ձեր WordPress Network admin dashboard և գնացեք Themes > Theme Editor (տես ստորև բերված պատկերը)։

  2. "Edit Themes" էջում համոզվեք, որ ընտրված թեման ճիշտ է այն դրոպ-դOWN դաշտում, որը գտնվում է ձեր էկրանի աջ վերև անկյունում (#3-ը ստորև բերված պատկերում)։

  3. Կլիկեք "Theme Files" բաժնի ներքևում գտնվող `functions.php` ֆայլի վրա՝ ֆայլը բեռնելու համար։ Սկսել ներքևից և տեղադրեք Ultimate Multisite սնուպետը, որը ստացել եք GitHub-ից։

# Ինչպես ստեղծել Must-Use Plugins (mu-plugins)

WordPress-ում կա մի հատկություն, որը թույլ է տալիս բեռնել հատուկ ֆունկցիոնալություն՝ "Must-Use Plugins" կամ կարճ անվանումով "mu-plugins"։

Այս հատուկ mu-plugins-ները բեռնվում են մյուս բոլոր սովորական պլագիններից առաջ, և դրանք չեն կարող անջատվել (deactivate)։ Մուլտիսայթ ցանցում այս mu-plugins-ների կոդը կբեռնվի ձեր տեղադրման բոլոր սайտերի վրա։

1. Օգտագործեք FTP կամ SSH՝ մուտք գործելով ձեր WordPress տեղադրման էֆիսիալ համակարգին (filesystem)։

2. WordPress-ի տեղադրման `wp-content` թղթապանակում ստեղծեք նոր թղթապանակ՝ անվան «mu-plugins».

<!-- Կարևոր է. Նկարը հասանելի չէ. Ֆայլի մանրամասն տեսքով `wp-content` թղթապանակը, որտեղ կա `mu-plugins` թղթապանակը -->

3. Ստեղծեք ձեր համակարգչում նոր PHP ֆայլ՝ անվան «wu-snippet.php»՝ Notepad կամ ցանկացած կոդի խմբագրիչով։

4. Տեղադրեք Ultimate Multisite-ի կոդի հատվածը, որը ստացել եք GitHub-ից, այս ֆայլում և պահպանեք այն։ Դուք կարող եք նաև այս կոդը տեղադրել կոդի հատվածի վերևում՝ ձեր mu plugin-ը նշելու համար։
