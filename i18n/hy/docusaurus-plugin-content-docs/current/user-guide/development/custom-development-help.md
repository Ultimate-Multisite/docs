# Codeable-ից ստանալ անհատական ծրագրային ապահովման օգնություն

Երբեմն, մեզ շատ հիանալի անհատականացում է պետք՝ մեր ցանցային հոսքերը հենց այնպես ստեղծելու համար, ինչպես մենք ուզում ենք։ Եթե դա ձեզ փնտրվում է, մենք խորհուրդ ենք տալիս [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE)-ը՝ այն վայրը, որտեղ կարող եք գտնել փորձառու ծրագրավորողներ Ultimate Multisite-ի փորձով։

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE)-ը հարթակ է՝ կապող հաճախորդներին որակավորված WordPress ֆրիլանս ծրագրավորողների հետ, ովքեր կարող են աշխատել ցանկացած չափի նախագծերի վրա։ Նրանք ունեն մանրամասն դիմումային գործընթաց՝ ապահովելու համար, որ ձեզ համապատասխանի փորձառու մասնագետներին, որոնք հատուկ որակավորված են այն խնդրի համար, որը մտածում եք։

### Ինչպես ստեղծել նախագիծ Codeable-ում?

Նախ, գնացեք [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE)-କୁ և սեղմեք **Start a Project** (Սկսել նախագիծ) կոճակը։

<!-- Կտեսարան չկա: Codeable-ի հիմնական էջ՝ Start a Project կոճակով -->

Այժմ, նախագիծ ներկայացնելու համար սեղմեք ձախում **Start a Project** (Սկսել նախագիծ)։

<!-- Կտեսարան չկա: Codeable-ի նախագծի ներկայացման էջ՝ Start a Project տարբերակով -->

«I need help with» (Ես օգնություն եմ խնդրում) դաշտում ընտրեք **Customizing** (Անհատականացումը)։

<!-- Կտեսարան չկա: Codeable-ի նախագծի ձևը՝ Customizing-ով ընտրված -->

«My» (Իմ) դաշտում ընտրեք **Plugin** (Լրացուցիչ/Պլագին)։

<!-- Կտեսարան չկա: Codeable-ի նախագծի ձևը՝ Plugin-ով ընտրված -->

Այնուհետև սեղմեք **Continue** (Շարունակել) կոճակը։

<!-- Կտեսարան չկա: Codeable-ի նախագծի ձևը Continue կոճակով -->

Այժմ ժամանակն է ներկայացնել ձեր նախագծի և կարիքները լավագույն եղանակով։ Փորձեք տալ հնարավորինս շատ տեղեկություն։

<!-- Կտեսարան չկա: Codeable-ի նախագծի նկարագրության ձևը -->

Հետևողական լինելու համար պատասխանեք հետևյալ հարցերին՝ գնահատման (pricing) օգնելու համար։

<!-- Կտեսարան չկա: Codeable-ի նախագծի գնահատման հարցերը -->

Հետո, եթե ունեք [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) հաշիվ չունեք, ստեղծեք մեկը և սեղմեք **Publish My Project** (Հրապարակել իմ նախագիծ) կոճակը։

<!-- Կարևոր է. Էկրանին պատկեր չկա՝ Codeable հաշվի ստեղծման և Publish My Project կոճակի -->

Մեկ անգամ երբ հրապարակեք ձեր [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) նախագիծը, պլատֆորմը կկապի ձեզ հետ մշակողների, ովքեր համապատասխանում են ձեր նախագծի համար անհրաժեշտ հմտություններին։

Այս գործընթացը կօգնի գտնել ճիշտ մարդուն՝ ձեր աշխատանքի համար, ով փորձ ունի WordPress-ի և Ultimate Multisite-ի հետ։

Codeable-ի նվազագույն գինը $70–$120 ԱՄՆ դոլար է ժամ başına։ Գնահատումները կփոխվեն աշխատանքի շրջանակով, բարդությամբ և նախագծի շտապությամբ։

Երբ ստանաք ձեր գնահատումը, կարող եք վարձել այն մշակողին, ով լավագույնս համապատասխանում է ձեր կարիքներին և միասին սկսել ձեր նախագիծը։
