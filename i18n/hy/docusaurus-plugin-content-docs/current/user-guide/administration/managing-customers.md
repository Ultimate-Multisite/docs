# Հաճախորդների կառավարում

**Customers** էջը ցուցադրում է ձեր ցանցում գրանցված անդամների ցուցակը։ Այս էջը ցուցադրում է օգտվողի հիմնական տեղեկություն՝ անուն, էլ․ հասցե, վերջին մուտքագրման ժամանակ և **switch to** (անցնել) հղում, որը թույլ է տալիս ձեզ մուտք գործել օգտվողի ենթա-сайтը՝ որպես սայթի ադմինիստրատոր։

![Customers list page](/img/admin/customers-list.png)

Յուրաքանչյուր հաճախորդի քարտի ներքևում կլուծեք **Manage** (Կառավարել), որպեսզի տեսնեք այդ հաճախորդի մասին մանրամասն տեղեկություն։

Հաճախորդի մանրամասների էջում **At a Glance** (Մեկնուրբ) բլոկը ցուցադրում է հաճախորդի ID-ն, վերջին մուտքագրման ժամանակը և այս հաճախորդի ընդհանուր կատարած գումարը։

**Customer Options** (Հաճախորդի տարբերակներ) բլոկը թույլ է տալիս ձեզ սահմանել այս հաճախորդին որպես **VIP** (թագավորություն), որը նշան է, որը դարձնում է նրան ավելի հեշտ գտնել։ **Billing Info** (Հաշվարկային տեղեկություններ) ներդիրը ցուցադրում է հաշվարկային տեղեկություն՝ էլ․ հասցե, երկիր և այլն։

**Fields** (Դաշտեր) բլոկը ցուցադրում է հաճախորդի օգտատիրոջ անունը, էլ․ հասցեն և թույլ է տալիս փոխել **Email Verification** (Էլ․ փոխհրավերացման) կարգավիճակը։

Customer Options բլոկի ներքևում կտեսնեք չորս լրացուցիչ բլոկներ.

- **Memberships** (Անդամակցություններ) — Ընթացիկ բաժանորդագրությունները, որոնցով հաճախորդը ներառված է
- **Payments** (Վճարումներ) — Հաճախորդի վճարումների պատմությունը
- **Sites** (Սայթեր) — Սայթերը, որոնք սեփականատիր են այս հաճախորդը
- **Events** (Իրադարձություններ) — Այս հաճախորդի հետ կապված իրադարձությունները (էլ․ փոխհրավերացում, հաշվարկային փոփոխություններ և այլն)

Ցանկացած տարրի աջ կողմում գտնվող սահմանագծի պատկերակը (→) սեղմեք՝ անցնելու դեթիլ էջին։ Օրինակ, սայթի աջ կողմում սահմանագծի վրա սեղմելը տանում է ձեզ սայթի խմբագրման էջ՝ որտեղ կարող եք փոփոխել անդամակցության կապը, սայթի տեսակը և ակտիվությունը։

Նույնը վերաբերում է անդամակցություններին և վճարումներին. յուրաքանչյուրը հղվում է դեթիլ էջերին, որտեղ կարող եք խմբագրել անդամակցության կարգավիճակը, կապված հաճախորդը, արտադրանքները, վճարման գեյթվեյի կարգավորումները և նույնիսկ մաքսիմալ հետ վերադարձ (refund) իրականացնել։

Հաճախորդի էջի աջ կողմում կտեսնեք.

* **Վերջին ինտերակցությունը և IP-ները** — Վերջին մուտքի ժամանակը, IP հասցեն և երկիրը։
* **Ժամանակագրումներ (Timestamps)** — Երբ ստեղծվել է հաճախորդի հաշիվը և վերջին թարմացման ժամանակը։
* **Ջնջել** — Հաճախորդին մշտականորեն ջնջելու տարբերակ։

:::warning
Հաճախորդի ջնջումը անվերապահ է և չի կարելի հետ վերադարձնել։
:::
