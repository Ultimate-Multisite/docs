# Ապրանքների կառավարումներ (Managing Memberships)

Ultimate Multisite-ում կարող եք կառավարել անդամակցությունները և մուտքը ձեր ցանցի բոլոր սոբսկրիաների վրա։ Ստուգեք և խմբագրեք սոբսկրիայի մանրամասները, ինչպիսիք են պլաները և արտադրանքները, կայքերի քանակը, հաշվարկման գումարը և հաճախականությունը, վճարումների պատմությունը և ժամանակագրությունները։

Այս էջան մուտք գործելու համար գնացեք Ultimate Multisite-ի մենյուի **Memberships** (Անդամակցություններ) ներդիրը։

![Memberships list page](/img/admin/memberships-list.png)

Memberships էջում կարող եք տեսնել ձեր հաճախորդների և դրանց հաշվի հետ կապված արտադրանքների ցուցակը։ Դուք կարող եք զտել անդամակցությունները վիճակի՝ ըստ հետևյալի. ակտիվ, սպասվում է, պահում են, ժամկետը տրված է (expired) կամ չեղարկված։

Ազատ անդամակցությունները համարվում են **կյանքի ընթացքում** (lifetime) անդամակցություններ։ Դրանք չեն ստանում վերջնաժամկետ և ինքնաբերաբար չեն մնում գործող։ Եթե ազատ պլանը պետք է ավարվի փորձի կամ որոշակի ժամկետից հետո, օգտագործեք վճարովի արտադրանք՝ փորձի հետ, ձեռքով չեղարկեք անդամակցությունը կամ այն մեկ այլ պլանով տեղափոխեք հաճախորդին, երբ դրա մուտքը պետք է փոխվի։

Սեղմեք անդամակցության վրա՝ մուտք գործելու համար այն էջը, որտեղ կարող եք խմբագրել անդամակցության մանրամասները. հաշվարկման տեղեկություն, կայքի կամ այցելության սահմանափակումներ և այլ բաներ։

![Membership edit interface](/img/admin/membership-edit.png)

Ահա անդամակցության խմբագրման էջի ամբողջական տեսարանը.

![Membership edit full interface](/img/admin/membership-edit-full.png)

Դուք նաև կարող եք ավելացնել նոր անդամակցություններ, որոնք կապված են ձեր ցանցում արդեն գրանցված օգտվողների հետ։ Սեղմեք **Add Membership** (Ավելացնել անդամակցություն) և կարգավորեք մանրամասները։

![Add Membership modal](/img/admin/membership-add-modal.png)

Այնուհետև սեղմեք **Create Membership** (Ստեղծել անդամակցություն)։ Նոր անդամակցությունը կհայտնվի անդամակցության ցուցակում։
