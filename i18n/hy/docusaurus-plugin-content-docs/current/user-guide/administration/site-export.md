# Սайտի արտահանման (Site Export) {#site-export}

**Site Export** ադմին էջը թույլ է տալիս ցանցային ադմինիստրատորներին մեկ սայթ կամ ամբողջ ցանցը փաթempել ներբեռնելի արխիվի մեջ՝ մուտքագրման, անվտանգության (backup) կամ փոխանցման (hand-off) աշխատանքների համար։

## Մեկ սայթի արտահանում {#exporting-one-site}

Գնացեք **Ultimate Multisite > Site Export** և ընտրեք **Generate new Site Export** (Ստեղծել նոր սայթի արտահանում)։ Ընտրեք այն ենթասայթը, որը ցանկանում եք արտահանել, ապա ընտրեք, թե արդյոք արխիվը պետք է ներառի վերբեռնումներ (uploads), plugin-ներ և themes։

Երբ արտահանման գործընթացը ավարտվի, ներբեռնեք ZIP-ը **Existing Exports** (Գոյություն ունեցող արտահանումներ) ցուցակից։ Այս արտահանված ZIP-երը ներառում են ինքնաբերաբար սկսվող `index.php` և `readme.txt`, որպեսզի արխիվը կարելի է վերբեռնել նոր հոստինի վրա և սկսել առանց նախապես առանձին ներմուծման plugin-ի տեղադրման։

## Ամբողջ ցանցի արտահանում {#exporting-the-whole-network}

Երբ ձեզ անհրաժեշտ է մեկ արխիվ, որը պարունակում է ցանցի բոլոր ենթասայթերը, օգտագործեք **Network Export** (Ցանցային արտահանում) Site Export էջում։ Սա հարմար է հոստինների փոխանցումների, աղետից հետո վերականգնման փորձարկումների կամ ստեյջին վերակառուցումների համար, երբ յուրաքանչյուր ենթասայթը պետք է միաժամանակ տեղափոխվի։

Քանի որ ցանցային արտահանումը կարող է շատ մեծ լինել՝ համեմատած մեկ սայթի արտահանման հետ, այն իրականացրեք ցածր ծանրաբեռնվածության ժամանակ և հաստատեք, որ թիրախային պահեստը բավարար է վերբեռնումների, plugin-ների, themes-ի և ստեղծված ZIP ֆայլերի համար։

### Ցանցային ներմուծման փաթեթներ (Network Import Bundles) {#network-import-bundles}

Ultimate Multisite 2.12.0-ից սկսած Site Exporter-ը կարող է ստեղծել **network import bundles** (ցանցային ներմուծման փաթեթներ). Սրանք հատուկ արխիվներ են, որոնք նախատեսված են ցանցային ամբողջական վերականգնման համար։ Մեկ network import bundle-ը պարունակում է բոլոր անհրաժեշտ ֆայլերը և 메տատվերը՝ մի քանի սայթներ նոր ցանցային տեղադրության մեջ վերականգնելու համար։

#### Ցանցային ներմուծման փաթեթի ստեղծում {#generating-a-network-import-bundle}

1. **Ultimate Multisite > Site Export**-ին գնացեք:
2. Սեղմեք **Generate new Network Export** (Ստեղծել նոր ցանցային արտահանում):
3. Արտահանման տիպի որպես **Network Import Bundle** (Ցանցային ներմուծման փաթեթ) ընտրեք:
4. Ընտրեք, թե արդյոք ներառել վերբեռնումները (uploads), plugin-ները և themes-ը։
5. Սեղմեք **Generate** (Ստեղծել):
6. Ներբեռնեք ZIP փաթեթը **Existing Exports** (Գոյություն ունեցող արտահանման) ցուցակից:

#### Ցանցային ներմուծման փաթեթից վերականգնում {#restoring-from-a-network-import-bundle}

Սайտերը վերականգնելու համար ցանցային ներմուծման փաթեթի (network import bundle) օգտագործում.

1. Ultimate Multisite-ը տեղադրեք ձեր թիրախ հոස්թինգի վրա:
2. Ավարտեք multisite սահմանման wizard-ը (հաղորդագրությունների ցուցակ):
3. Նոր ցանցում գտնվող Ultimate Multisite > Site Export-ին գնացեք:
4. Սեղմեք **Import Network Bundle** (Ներմուծել ցանցային փաթեթը):
5. Վերբեռնեք network import bundle ZIP ֆայլը:
6. Հետևեք էկրանի ցուցումներին ներմուծման համար:
7. Ներմուծման գործընթացը կվերականգնի բոլոր սайտերը, դրանց բովանդակությունը և կոնֆիգուրացիաները:

Network import bundles-ը պահպանում է.
- Սайտի բովանդակությունը (posts, pages, custom post types):
- Սайտի կարգավորումները և տարբերակները (settings and options):
- Օգտատերերի դերերը և թույլտվությունները (user roles and permissions):
- Plugin-ները և themes-ը (եթե ներառված են փաթեթում):
- Մեդիայի վերբեռնումները (եթե ներառված են փաթեթում):
- Հատուկ բազային աղյուսակներ և տվյալները:

## Ինքնաբուհական boot-վող export ZIP տեղադրելը {#installing-a-self-booting-export-zip}

Նոր հոස්թի վրա tự-boot-վող (self-booting) ZIP-ը վերականգնելու համար.

1. Արտահանված ZIP-ի բովանդակությունը տեղադրեք թիրախային web root-ում:
2. Բացեք վերբեռնված `index.php` ֆայլը բրաուզերով:
3. Հետևեք bundled export package-ի էկրանի ցուցումներին տեղադրման համար:
4. Ավելի հեռու մաքրելուց առաջ ստուգեք bundled `readme.txt`-ը՝ արտահանման մասին նշումները ստանալու համար:

Addon-ների կոնկրետ տեղադրման և ներմուծման մանրամասների համար බලන්න [Site Exporter addon documentation](/addons/site-exporter) հղումը։

Ultimate Multisite 2.9.0-ի մեջ ավելացված single-site tool-ի (մեկ սайտի գործիքի) մասին տեղեկությունների համար බලන්න [Export & Import](/user-guide/administration/export-import) հղումը։
