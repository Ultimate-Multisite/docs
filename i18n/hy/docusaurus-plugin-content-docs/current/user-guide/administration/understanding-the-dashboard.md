# Ultimate Multisite Dashboard-ի Ընդհանրացումը

Ultimate Multisite-ի ပိုမိုရှုပ်ထွေးတဲ့ စနစ်ပိုင်းဆိုင်ရာ တည်ဆောက်မှုရွေးချယ်စရာတွေနဲ့ လုပ်ငန်းလည်ပတ်မှု အချက်အလက်တွေကို မစတင်ခင်မှာ ဒီ dashboard ကို တစ်ချက်ကြည့်လိုက်ရအောင်။

WordPress plugin အများစုလိုပဲ၊ Ultimate Multisite က **Ultimate Multisite** လို့ အမည်ပေးထားတဲ့ network-level menu entry တစ်ခုကို ဖန်တီးပေးပြီး သူ့အောက်မှာ sub-menu entries အမျိုးမျိုး ရှိပါတယ်။

## Network Dashboard Widgets

Ultimate Multisite က network-level dashboard ထဲကို အသုံးဝင်တဲ့ widgets အချို့ကို ထည့်သွင်းပေးပါတယ်။ ဒီdashboard ကို **My Sites → Network Admin → Dashboard** ကို သွားရောက်ပြီး ရှာတွေ့နိုင်ပါတယ်။ ဒီwidgets တွေဟာ network administrator အတွက် အချက်အလက်နဲ့ ပုံမှန်လုပ်ဆောင်မှုတွေကို လွယ်ကူစွာ ရယူနိုင်ဖို့ ဒီဇိုင်းထုတ်ထားတာ ဖြစ်ပါတယ်။

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

### ပထမခြေလှမ်းများ (First Steps)

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

ဒီwidget ဟာ Ultimate Multisite ကို အသစ်တပ်ဆင်တဲ့အခါ ပေါ်လာပြီး network administrator ကို ငွေပေးချေမှု နည်းလမ်းတစ်ခု စနစ်ထည့်တာနဲ့ စမ်းသပ်အကောင့်တစ်ခု ဖန်တီးတာလိုမျိုး နောက်ထပ် လုပ်ဆောင်ရမယ့် အလုပ်တွေကို ပြန်သတိပေးပါတယ်။

### အကျဉ်းချုပ် (Summary)

![Summary widget](/img/admin/network-dashboard-summary.png)

Summary widget က တစ်နေ့အတွက် စာရင်းသွင်းမှု အရေအတွက်နဲ့ ဝင်ငွေအကြောင်း အစီရင်ခံတယ်။ MRR (Monthly Recurring Revenue) ကတော့ ဝင်ငွေရရှိတဲ့ အဖွဲ့ဝင်တွေရဲ့ စုစုပေါင်း ခန့်မှန်းခြေ ပမာဏကို ပြသပါတယ်။

### လှုပ်ရှားမှုစီးကြောင်း (Activity Stream)

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

ဒီwidget က Ultimate Multisite တစ်လျှောက် ဖြစ်ပျက်နေတဲ့ အဖြစ်အပျက်တွေကို အစီရင်ခံတယ်။ အဖြစ်အပျက်တွေမှာ စာရင်းသွင်းမှုများ၊ ပယ်ဖျက်မှုများ၊ site များ ဖန်တီးခြင်းနဲ့ ဖျက်သိမ်းခြင်းတို့ ပါဝင်ပါတယ်။

ပိုမိုအသေးစိတ် သိရှိလိုပါက documentation ထဲက Events section ကို ကြည့်ရှုနိုင်ပါတယ်။

### အခုချိန် (Right Now)

![Right Now widget](/img/admin/network-dashboard-right-now.png)

Այս utility widget-ը ցուցադրում է ցանցի օգտատերերի և կայքերի կարճ ամփոփում։ Վերևի հեշտ մուտքային կապերը թույլ են տալիս նոր կայքեր կամ օգտատերներ ստեղծելու համար՝ մեկ կտորով անցնելը։

### Տեղեկություններ և քննարկումներ (News and Discussions)

![News and Discussions widget](/img/admin/network-dashboard-news.png)

Այս widget-ը հավաքում է և ցուցադրում է Ultimate Multisite-ի ամենաթարմ տեղեկատվությունը։ Մշտական հետևեք այս widget-ին՝ թարմացումների, կարևոր սխալների և անվտանգության շտկումների մասին տեղեկություն ստանալու համար։

## Ultimate Multisite Dashboard (Վերջնական ցանցային վիճակագրություն)

Որտեղ ցանցային մակարդակի dashboard-ը ներկայացնում է ամբողջական տեղեկատվությունը ցանցի մասին, Ultimate Multisite dashboard-ը (որը գտնվում է Ultimate Multisite menuyի վերին մակարդակում) ներկայացնում է ծառայության մասին տեղեկությունները։

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

### ամսական կրկնվող եկամուտի աճ (Monthly Recurring Revenue Growth)

MRR (Monthly Recurring Revenue) գրաֆիկը ցույց է տալիս եկամուտների ամսական բաշխումը 12 ամսան տևողության օրվա ընթացքում։

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

Նոր եկամուտները հետևվում են կապույտ գույնով, իսկ չեղարկումները արտացոլվում են կարմիր գույնով։

### Ամսաթվի միջակայքի աջակցություն (Date-range Support)

Հաճախորդների և գործունեության քանակի կախված, վիճակագրությունը կարող է շատ բարդ դառնալ։ Dashboard-ում ներկառուցված ամսաթվի միջակայքի աջակցությունը թույլ է տալիս ցանցային ադմինիստրատորին կենտրոնանալ միայն այն տեղեկությունների և ժամանակաշրջանի վրա, որոնք անհրաժեշտ են։

![Date range selector and stats](/img/admin/um-dashboard-date-range.png)

Ամսաթվի միջակայքը կառավարում է այն ժամանակաշրջանը, որի տեղեկությունները ցուցադրվում են ներքևում գտնվող widget-ների կողմից՝ ներառյալ եկամուտները, նոր անդամակցությունները, երկրով հավաքված գրանցումները, ձևաթղթով գրանցումները և ամենաշատ այցելվող կայքերը։

![Dashboard statistics widgets](/img/admin/um-dashboard-stats.png)

:::tip
**Most Visited Sites** widget-ը աշխատեցնելու համար, թույլ տվեք այցերի հաշվումը **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**-ում։
:::
