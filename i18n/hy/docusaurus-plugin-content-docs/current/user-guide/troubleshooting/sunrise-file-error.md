# Sunrise Ֆայլը Տեղադրելիս Սխալ {#error-installing-the-sunrise-file}

`sunrise.php` ֆայլը հատուկ ֆայլ է, որը WordPress-ը փորձում է գտնել՝ իր սեփականությունը սկսելիս։ WordPress-ի համար `sunrise.php` ֆայլը հայտնաբերելու համար այն պետք է գտնվի **wp-content** թղթապանակի ներսում։

Երբ դուք ակտիվացնում եք Ultimate Multisite-ը և անցնում սեփական սկզբունքներին, ինչպես այն կա ձեր պատկերով տեսանելի սկզբունքների մեջ, Ultimate Multisite-ը փորձում է մեր `sunrise.php` ֆայլը պատճենել wp-content թղթապանակի մեջ։

<!-- Կարևոր նկար չկա. Սկզբնական սկզբունքների էջ՝ sunrise.php տեղադրման քայլը ցուցադրող -->

Մեծամասամբ մենք հաջողությամբ կարող ենք պատճենել ֆայլը և ամեն ինչ աշխատում է։ Սակայն, եթե որևէ բան ճիշտ չի սահմանված (օրինակ՝ թղթապանակների թույլտվություններ), կարող եք հանդիպել այն իրավիճակի, երբ Ultimate Multisite-ը չի կարողանում ֆայլը պատճենել։

Եթե կարդաք Ultimo-ի տվյալները, որը ձեզ տալիս է սխալի հաղորդագրությունը, դուք կտեսնեք, որ այստեղ հենց այդպես է եղել. **Sunrise պատճենումը ձախողվեց**։

<!-- Կարևոր նկար չկա. Սխալի հաղորդագրություն՝ Sunrise պատճենման ձախողումը ցուցադրող -->

Դրան ուղղելու համար կարելի է պարզապես `wp-ultimo` պլագինի թղթապանակի ներսում տեղադրված `sunrise.php` ֆայլը պատճենել և այն տեղադրել ձեր wp-content թղթապանակի մեջ։ Այդ գործողությունից հետո վերանայեք սկզբունքների էջը, և ստուգումները պետք է անցնեն։

<!-- Կարևոր նկար չկա. Ֆայլի կառավարման մատյան՝ sunrise.php-ը wp-ultimo պլագինի թղթապանակի ներսում --> Այնուամենայնիվ, սա կարող է պահանջել ձեր թղթապանակների թույլտվությունները ընդհանրացված ստուգելը՝ ապագայում խնդիրներ չունենալու համար (ոչ միայն Ultimate Multisite-ի հետ, այլ նաև այլ պլագինների և թեմաների հետ)։

WordPress-ի մաս է կազմող **Health Check tool**-ը (Դուք կարող եք դրան հասանելի լինել ձեր հիմնական սայլի **admin panel > Tools > Health Check**-ով), կարող է ցույց տալ, թե արդյոք ձեր թղթապանակների թույլտվությունները սահմանված են այն արժեքներով, որոնք կարող են խնդիրներ առաջացնել WordPress-ի հետ։
