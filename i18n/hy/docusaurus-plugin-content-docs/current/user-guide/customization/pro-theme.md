# Ձեր Ադмін պանելը կոնֆիգուրացնել PRO Theme-ով {#customizing-your-admin-panel-with-pro-theme}

Հաճախ, երբ մենք աշխատում ենք հաճախորդների հետ, մեզ անհրաժեշտ է տրամադրել նրանց WordPress-ի ադмін դաշտի համար սահմանված հատուկ փորձառություն։ Եթե փնտրում եք ավելի պրոֆեսիոն տեսք և զգացողություն, PRO Theme-ը կարող է ձեզ օգնել։

PRO Theme-ը մշակված է որպես ադмін թեմա՝ օգնելու ձեզ հաճախորդների դաշտերը կոնֆիգուրացնելիս։ Այն նաև Ultimate Multisite-ի պրեմիում պլագինների հիմքի մաս է։

Տեղադրելուց հետո գնացեք ձեր _Plugins_ էջը և սեղմեք այն՝ ակտիվացնելու համար ձեր ցանցում (Network)։

<!-- Կարող է բացակայել պատկերը: WP Plugins էջը, որը ցույց է տալիս PRO Theme-ի Network Activate կոճակը -->

_Ակտիվացրեք այն ձեր Plugins էջում ամբողջ ցանցում_

_**Նշում:**_ _Եթե դուք միայն տեղադրում եք, այն հաճախորդների համար կհայտնվի որպես տարբերակ, որը կարելի է նշել՝ թեմայի ավելացման համար միայն այդ օգտվողի համար._

Այժմ, ունեք ձեր ադмін ինտերֆեյսը ամբողջությամբ նոր դիզայնով։ Ձ ձախ սյունում կլինի նաև PRO Theme անվան MENU-ն։

<!-- Կարող է բացակայել պատկերը: PRO Theme-ի վերածված ադмін պանելի ինտերֆեյսը -->

_Ձեր Ադмін Պանելի համար ամբողջ նոր տեսք_

Այս մենյուով դու կունենաք հասանելիություն հինգแท็บի՝

<!-- Կարող է բացակայել պատկերը: PRO Theme-ի Styling Settings ներդրումը -->

_Styling Settings_ (Ստեղծման կարգավորումներ)

<!-- Կարող է բացակայել պատկերը: PRO Theme-ի Functionality ներդրումը -->

_Functionality_ (Ֆունկցիոնալություն)

<!-- Կարող է բացակայել պատկերը: PRO Theme-ի Custom CSS ներդրումը -->

_Custom CSS_ (Անհատական CSS)

<!-- Կարող է բացակայել պատկերը: PRO Theme-ի Import/Export ներդրումը -->

_Import/Export_ (ներդնել/բացահոնք)

<!-- Կարող է բացակայել պատկերը: PRO Theme-ի Activate ներդրումը -->

_Ակտիվացրեք PRO Theme-ը_

Յուրաքանչյուրի մեջ կարող եք ապահովել և անջատել ստիլինգ տարրերը, թույլ տալ կամ արգելել ֆունկցիոնալությունները և ձեր հաճախորդներին տրամադրել այն կոնֆիգուրացումը, որը կարող են փնտրել ձեր սայթերի ցանցում։

Ահա մի օրինակ, թե ինչ կարող եք ստանալ մի քանի կարգավորումներից հետո.

_Ձեր հաճախորդի Dashboard-ը_
