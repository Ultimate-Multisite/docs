# Ձեր ցանցում հայտարարություններ տեղադրել Ultimate Multisite-ի միջոցով՝ Ad Injector-ով

Ultimate Multisite: Ad Injector-ը **անվճար add-on** է, որը ձեզ տալիս է գովազդներ տեղադրելու համար գործիքները՝ դրանք հենց դրանց խմբագրման էջից։

Ultimate Multisite: Ad Injector-ը տեղադրելուց հետո, ակտիվացնելու համար գնացեք ձեր _Plugins_ էջը։

<!-- Էկրան샷 չկա. WP Plugins էջը՝ Ad Injector add-on-ի Network Activate կոճակով -->

_Սեղմեք Network Activate կոճակը_

Երբ add-on-ը ակտիվ լինի, plan-ի խմբագրման էկրանի _Advanced Options_ մետաբքսում կլինի նոր տարբերակ՝ _Ad Settings_։

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_Այժմ կարող եք խմբագրել ձեր յուրաքանչյուր plan-ի համար գովազդների կարգավորումները_

Այդ տաբում, դուք կկարողանաք ավելացնել post-երի և page-երի ad code-երը՝ content-ի առաջ, վերև և ներքև։

<!-- Էկրան샷 չկա. Post-երի և page-երի ad code-ի դաշտեր content-ից առաջ -->

<!-- Էկրան샷 չկա. Օրինակ՝ Ad-ի ցուցադրում post-ի էջում տվյալ plan-ի համար -->

_Ձեր ad-ի կոդը ඇතුළත්ելուց հետո այն կհայտնվի այդ Specific plan-ի posts-ի և page-երի վրա։_

Դուք նաև կարող եք 포함ել գովազդներ ադմին էջերի header-ում այդ plan-ի օգտատերերի համար։

<!-- Էկրան샷 չկա. Plan-ի կարգավորումներում Admin Panel Ad Code տարբերակը -->

<!-- Էկրան샷 չկա. Օրինակ՝ User admin panel-ում ցուցադրված ad -->

_Եթե ցանկանում եք ձեր օգտատերի back-end-ում գովազդներ ցուցադրել, գնացեք Admin Panel Ad Code տարբերակ_

### Գովազդներից դուրս

Դուք կարող եք նաև Ultimate Multisite: Ad Injector-ը օգտագործել՝ բարձր մակարդակի plan-երի համար աջակցություն ապահովելու համար։

Պարզապես գնացեք _Admin Panel Ads_ տարբերակ և ավելացրեք ձեր support tool-ի HTML կոդը։

<!-- Էկրան샷 չկա. Support tool-ի HTML կոդով Admin Panel Ads տարբերակ -->

<!-- Կարող է չլինել պատկեր՝ հաճախորդի աջակցության վի젯ը տեսանելի է ադմին պանելում որոշակի պլանների համար -->

_Հաճախորդի աջակցություն ավելացնել որոշակի պլանների համար_

Այսպես մարդիկ կարող են ձեզ հասնել ուղղակի իրենց Admin Panel-ից։
