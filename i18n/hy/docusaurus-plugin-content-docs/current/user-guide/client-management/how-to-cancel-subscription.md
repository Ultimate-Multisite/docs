# Ինչպե՞ս կարող եմ չեղարկելու մեր բաժանորդագրությունը։ {#how-can-i-cancel-my-subscription}

Եթե ունեք Ultimate Multisite լիցենզիա, կարող եք չեղարկել նրա վեր নবացումը ցանկացած պահի։ Պարզապես հետևեք ստորև ներկայացված քայլերին.

<!-- Screenshot unavailable: Account page email with access link -->

Օգտագործեք այն էмейլը, որը տրամադրել եք գնման ժամանակ և ձեր գաղտնաբառը։

<!-- Screenshot unavailable: Login page with email and password fields -->

Սա ձեր Հաշվի էջի սկզբնական էկրանն է.

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

Ձեր բաժանորդագրությունը չեղարկելու համար, ձախ կողմում գտնվող մենյուից գնացեք «Renewing & Billing» (Վեր নবացում և հաշվարկ)։

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

Բացեք կողային պատուհանը՝ աջ կողմում գտնվող սլայդերի վրա սեղմելով թեքիչը (arrow-ը)։ Այնուհետև ընտրեք այն տարբերակը, որը կտեղի ունենա **auto-renew**-ի չեղարկումը։

Система ձեզ կցուցադրի հաստատման հաղորդագրություն։

Եթե համոզված եք, որ ուզում եք չեղարկել բաժանորդագրությունը, պարզապես սեղմեք «**Cancel Renewals**» (Չեղարկել վեր নবացումները) կոճակը։ Այս գործողությունից հետո ձեզ կհանձնվի համառոտ հարցում պատասխանելու համար։

Ավարտված է։ Ձեր բաժանորդագրությունը ավտոմատ չի վեր নবացվի։

**Դուք ունեք գործող մինչև ձեր բաժանորդագրության ժամկետը։** Եթե ուզում եք վերականգնել ձեր բաժանորդագրությունը, դա անելու համար պետք է ստեղծեք այն ձեռքով։
