# Ձեր հաճախորդների կայքերը (v2)

Ձեր ցանցում գրանցվելուց հետո ձեր հաճախորդները կարող են մուտք գործել իրենց դաշտբերին։

**Account** էջի բացի, որտեղ նրանք կարող են տեսնել կարևոր տեղեկություն իրենց վճարումների, անդամակցության, ադմինային տիրույթների (domains), պլանի սահմանափակումների և այլ բաների մասին, նրանք հասանելի են **My Sites** էջին։ Այնտեղ նրանք կարող են տեսնել և կառավարել իրենց բոլոր կայքերը։ Բոլորը պետք է անեն միայն My Sites-ի վրա սեղմել իրենց դաշտբերում։

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

Նրանց կցուցադրվեն բոլոր կայքերը՝ յուրաքանչյուրի ներքևում **Manage** կոճակիով։ Եթե նրանք սեղմեն, նրան ուղղելու են այդ կոնկրետ կայքի դաշտբերին։

Նրանք կարող են նաև տեսնել, թե որ կայքն է **հիմնական** այնպիսի պլանների համար, որոնք թույլ են տալիս ունենալ մի քանի կայք։

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

Ահա կայքի խմբագրման ամբողջական տեսարանը.

![Site edit full interface](/img/admin/site-edit-full.png)

Եթե ցանկանում են փոխել իրենց հիմնական կայքը, պետք է գնան Account էջը՝ սեղմելով **Account** մենյուի վրա։

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

Նրանք կարող են նաև տեսնել և խմբագրել հաճախորդի մանրամասները.

![Customer edit interface](/img/admin/customer-edit.png)

Այնուհետև, պետք է սեղմեն **Change Default Site**-ի վրա։

![Change Default Site button on account page](/img/account-page/change-default-site.png)

Կհայտնվի pop-up պատուհան։ Սեղմեք դրոպ-դաուն ցուցակում և ընտրեք, թե որ կայքն եք ուզում դարձնել հիմնական կայքը։

![Change Default Site dropdown selection](/img/account-page/change-default-site-dropdown.png)

Այնուհետև, պարզապես սեղմեք **Change Default Site**-ի վրա և ավարտվեց։

![Default site-ի փոփոխության հաստատումը](/img/account-page/change-default-site-confirm.png)
