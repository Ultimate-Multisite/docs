# Կայքի արտահանող (Site Exporter) {#site-exporter}

Այս հնարավորությունը տրամադրվում է **Site Exporter** ավելացման միջոցով։

Լրելի համապատասխան փաստաթուղթի համար ստուգեք [Site Exporter ավելացման փաստաթուղթը](/addons/site-exporter)։
