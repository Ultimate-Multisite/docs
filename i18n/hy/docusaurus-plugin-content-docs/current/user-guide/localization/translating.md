# Ultimate Multisite-ի թարգմանություն

Ultimate Multisite-ը ստեղծված է անգլերենով, բայց պլագինը լիովին թարգմանելի է և մենք ակտիվորեն հաճախորդների կողմից ներդրումներ են ընդունում։

## Հիմնական ප්լագինների թարգմանություններ {#core-plugin-translations}

Ultimate Multisite-ի հիմնական ප්լագինը թարգմանվում է պաշտոնական **WordPress.org թարգմանչական հարթակով**.

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

Սա WordPress-ի ստանդարտ համայնքային թարգմանական համակարգն է։ Երբ թարգմանությունները հասնում են բավարար հաստատման շեմին, դրանք ավտոմատ կերպով բաշխվում են Ultimate Multisite-ի բոլոր օգտվողներին WordPress ප්լագինների թարմացումների միջոցով։

**Մենք ձեզ հարկավորություն ունենք!** Եթե դուք անգլերենից այլ լեզու խոսում եք, խնդրում ենք ներդրումներ կատարել թարգմանություններով։ Նույնիսկ մի քանի տող թարգմանելը օգնում է։ Սկսելու համար.

1. Պատվիրեք [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)
2. Ընտրեք ձեր լեզուն
3. Սեղմեք «Contribute Translation» (Թարգմանության ներդրում)
4. Ձեզ անհրաժեշտ կլինի WordPress.org հաշիվ

Եթե ցանկանում եք դառնալ ձեր լեզվի թարգմանական խմբագրիչ (որը կարող է հաստատել թարգմանությունները), դիմեք [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions)-ով։

## Աջակցության ප්լագինների թարգմանություններ {#addon-translations}

Աջակցության (Addon) ප්լագինները թարգմանվում են առանձին՝ հետևյալ հասցեում.

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Աջակցության թարգմանությունները ներկայումս AI-ի կողմից ստեղծված են որպես սկզբնական տարբերակ։ Դրանք ընդգրկում են բոլորաջակցվող լեզուները, բայց կարող են պարունակել անհարմար արտահայտություններ կամ անճշտություններ։ Մենք հաճախորդների կողմից մարդկային թարգմանիչներին ողջույն են հղում, որպեսզի դրանք ավելի բնական և ճշգրիտ դարձնեն։

## Տեղական թարգմանություններ {#local-translations}

Եթե ցանկանում եք թարգմանություններ ստեղծել ձեր սեփական նախագծի համար՝ առանց ուղղակի ներդրումների, ապա կան երկու տարբերակ.

2. Բացեք `.pot` ֆայլը `wp-content/plugins/ultimate-multisite/lang/` դիրքից:
3. Թարգմանեք տողերը Poedit-ի միջոցով:

5. Տեղադրեք արդյունքային ֆայլերը `lang` թղթապանակում:

### Loco Translate-ի օգտագործումը {#using-loco-translate}

**[Loco Translate](https://wordpress.org/plugins/loco-translate/)** WordPress-ի պլագինը թույլ է տալիս ուղղակիորեն ծառայության վահանից (admin panel) թարգմանել՝ առանց դեսքթոփ գործիքների։ Այն ինքներդ կստեղծի և կկոմպիլի թարգմանական ֆայլերը։
