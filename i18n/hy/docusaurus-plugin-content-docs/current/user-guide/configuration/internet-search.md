# Ինտերնետ որոնում {#internet-search}

Gratis AI Agent v1.5.0-ը հավելում է **Internet Search** (Ինտերնետային որոնում) հնարավորություն, որը թույլ է տալիս AI օգնականին զրուցման ընթացքում վեբից թարմացված տեղեկություններ ստանալ։ Սա աշխատում է [Brave Search API](https://brave.com/search/api/) կամ [Tavily API](https://tavily.com/) միջոցով։

## Ինչպես է այն աշխատում {#how-it-works}

Երբ ինտերնետային որոնումը թողարկված է, օգնականը կարող է ավտոմատ կերպով հարցում կատարել ձեր կազմաձևված որոնման մատակարարի (search provider)՝ որոշելու համար, որ հարցը պահանջում է ընթացիկ կամ արտաքին տեղեկություն. օրինակ՝ վերջին նորություններ, ապրիական գներ կամ փոփոխված մոդելի մարզման ժամանակից հետո փոփոխվելու կարողությամբ փաստաթղթեր։

Արդյունքները ստացվում են իրական ժամանակում և ներարկվում են օգնականի համատեքստում (context)՝ պատասխան տալուց առաջ։ Օգնականը ցույց է տալիս, թե երբ է օգտագործել որոնման արդյունքները հարցին պատասխանելու համար։

## Internet Search-ի ակտիվացում {#enabling-internet-search}

Ինտերնետային որոնման համար անհրաժեշտ է API key՝ ձեր ընտրած որոնման մատակարարից։ Կազմաձևելու համար հետևեք այս քայլերին.

1. Գնացեք **Gratis AI Agent → Settings → Advanced**-ի։
2. Գտեք **Internet Search Provider** (Ինտերնետային որոնման մատակարար) ընտրության դրոպ-դաուն և ընտրեք կամ **Brave Search**, կամ **Tavily**։
3. Մուտքագրեք ձեր API key-ը համապատասխան դաշտում։ Յուրաքանչյուր դաշտի աջ կողմում տեղադրված է հաշվի առնելով հղումներ՝ գրանցվելու համար։
4. Սեղմեք **Save Settings** (Կազմաձևումները պահել)։

Key-ը ստոխվածից հետո Internet Search-ի հնարավորությունը ավտոմատ կերպով հասանելի է օգնականի համար։

## Brave Search {#brave-search}

### Brave Search API Key-ի ստացում {#obtaining-a-brave-search-api-key}

1. Դուք այցելեք [Brave Search API page](https://brave.com/search/api/)։
2. Գրանցվեք պլանով։ Առկա է անվճոյական մակարդակ՝ ամսական հարցումների սահմանափակությամբ։
3. Պատճենեք ձեր API key-ը Brave Search developer dashboard-ից։
4. Տեղադրեք այն Gratis AI Agent-ի կարգավորումներում գտնվող **Brave Search API Key** դաշտում։

### Օգտագործման սահմանափակումներ {#usage-limits}

Օգտագործումը հաշվարկվում է Brave Search-ի կողմից կատարված հարցումների քանակով։ Յուրաքանչյուր AI պատասխան, որը գործարկում է որոնումը, հաշվվում է որպես մեկ հարցում։ Ինչ-որ անսպասելի վճարումներից խուսափելու համար հետևեք ձեր օգտագործմանը [Brave Search developer dashboard](https://brave.com/search/api/)։

## Tavily {#tavily}

Superdav AI Agent v1.10.0-ը ավելացնում է **Tavily** որպես այլընտրանքային ինտերնետի որոնման մատակարար, առաջարկելով հարստ hơn որոնման արդյունքներ և առաջադեմ հետազոտական հնարավորություններ։

### Tavily API Key-ի ստացում {#obtaining-a-tavily-api-key}

1. Բաժանորդագրվեք [Tavily API page](https://tavily.com/) կայքում։
2. Ստեղծեք հաշիվ։ Առկա է անվճար մակարդակ՝ ամսական հարցումների սահմանափակությամբ։
3. Պատճենեք ձեր API key-ը Tavily dashboard-ից։
4. Տեղադրեք այն Gratis AI Agent-ի կարգավորումներում գտնվող **Tavily API Key** դաշտում։

### Օգտագործման սահմանափակումներ {#usage-limits-1}

Օգտագործումը հաշվարկվում է Tavily-ի կողմից կատարված API կանչերի քանակով։ Յուրաքանչյուր AI պատասխան, որը գործարկում է որոնումը, հաշվվում է որպես մեկ կանչ։ Անսպասելի վճարումներից խուսափելու համար հետևեք ձեր օգտագործմանը [Tavily dashboard](https://tavily.com/)։

## Ինտերնետի որոնման անջատում {#disabling-internet-search}

Հեռացրեք API key-ը ակտիվ որոնման մատակարարի դաշտից և պահպանեք փոփոխությունը։ Այս գործողությամբ օգնականին այլևս չի առաջարկվի ինտերնետային որոնման հնարավորություն։

:::note
Ինտերնետային որոնումը պատասխաններին ավելացնում է ուշացում (latency) պատասխանների համար, քանի որ օգնականը պետք է սպասի որոնման արդյունքների՝ պատասխան գեներացնելուց առաջ։ Ժամանակին կախված դեպքերի համար հաշվի առեք, թե արդյոք անհրաժեշտ է իրական ժամանակի որոնումը, թե արդյոք օգնականի ներկառուցված գիտելիքները բավարար են։
:::
