# Ներկառուցված գործակալներ

Gratis AI Agent v1.9.0-ը տրամադրվում է հինգ ներկառուցված գործակալներով, որոնցից յուրաքանչյուրը նախապես կարգավորված է գործիքների կենտրոնացված հավաքածուով, հարմարեցված համակարգային հուշումով և տվյալ ոլորտի տարածված առաջադրանքներին համապատասխան մեկնարկային առաջարկներով։ Գործակալների միջև անցումը փոխում է, թե ինչ կարող է անել օգնականը և ինչպես է արձագանքում՝ առանց ձեր կողմից որևէ կարգավորման։ Superdav AI Agent v1.18.0-ը կարող է այս աշխատանքային հոսքերին ավելացնել ժամանակացույցին իրազեկ գործիքներ, հիշեցումների գրառումներ, հաստատման դարպասներ և SMS ծանուցումներ, երբ համապատասխան ինտեգրացիաները կարգավորված են։

## Ի՞նչ է գործակալը

Յուրաքանչյուր գործակալ անվանված կարգավորման պրոֆիլ է, որը միավորում է՝

- **Գործիքներ** — այն կարողությունները, որոնք գործակալին թույլատրված է կանչել (օրինակ՝ Բովանդակության հեղինակը հասանելիություն ունի գրառում ստեղծելու կարողություններին, իսկ Դիզայն ստուդիան՝ CSS և theme.json կարողություններին)
- **Համակարգային հուշում** — հրահանգներ, որոնք սահմանում են գործակալի տոնը, առաջնահերթությունները և սահմանափակումները
- **Առաջարկներ** — նախապես գրված հուշումներ, որոնք ցուցադրվում են զրույցի միջերեսում՝ արագ սկսելու համար

## Գործակալների ընտրիչին հասանելիություն

1. Բացեք **Gratis AI Agent** վահանակը WordPress ադմինիստրատորի կողային գոտում։
2. Սեղմեք **գործակալի պատկերակը** զրույցի վերնագրի վերևի ձախ մասում (պատկերակը փոխվում է՝ արտացոլելով ակտիվ գործակալը)։
3. **Agent Picker**-ը բացվում է որպես ձև-աղյուսակի վերադիր շերտ։ Յուրաքանչյուր գործակալ ցուցադրվում է իր պատկերակով, անունով և մեկ տողով նկարագրությամբ։
4. Սեղմեք գործակալի տողի վրա՝ այն ակտիվացնելու համար։ Զրույցի վերնագիրը անմիջապես թարմացվում է։

Կարող եք նաև փոխել գործակալներին զրույցի ընթացքում. նոր գործակալի համակարգային հուշումը ուժի մեջ է մտնում հաջորդ հաղորդագրությունից։

## Հինգ ներկառուցված գործակալները

### Բովանդակության հեղինակ

**Կիզակետ․** Գրառումների, էջերի և կոնտակտային ձևերի ստեղծում և խմբագրում։

**Հասանելի գործիքներ․** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`։ Superdav AI Agent v1.18.0 ինտեգրացիաները միացված լինելու դեպքում հաստատված աշխատանքային հոսքերին կարող են նաև հասանելի լինել կարգավորված օրացույցային համատեքստ, հաստատման դարպասներ, հիշեցումներ և SMS ծանուցման գործիքներ։

**Ինչում է լավ աշխատում․**
- Բլոգային գրառումների սևագրում և հրապարակում՝ համառոտ նկարագրությունից կամ ուրվագծից
- Նոր կայքի համար թիրախային էջերի խմբաքանակների ստեղծում
- Կոնտակտային և հարցման ձևերի կառուցում
- Գրառումների համար գլխավոր պատկերների սահմանում URL-ից կամ որոնումից
- Միջոցառման հետագա հաղորդագրությունների սևագրում՝ կարգավորված Google Calendar համատեքստից, այնուհետև դադար հաստատման համար՝ նախքան ծանուցումներ ուղարկելը

**Մեկնարկային առաջարկներ․**
- *Գրիր 500 բառանոց բլոգային գրառում WordPress multisite-ի առավելությունների մասին։*
- *Ստեղծիր «Մեր մասին», «Ծառայություններ» և «Կապ» էջերը և հրապարակիր դրանք։*
- *Ավելացրու ամրագրման հարցման ձև Կապ էջում։*
- *Սևագրիր հիշեցում վաղվա կարգավորված օրացույցային միջոցառման մասնակիցների համար և սպասիր հաստատման՝ նախքան այն ուղարկելը։*

### Կայքի կառուցող

**Կիզակետ․** Կայքի ամբողջական ստեղծում մեկ հուշումից։

**Հասանելի գործիքներ․** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`։ Superdav AI Agent v1.18.0-ի դեպքում կարգավորված կառավարվող ծառայության, հաստատման, հիշեցման, օրացույցի և SMS գործիքները կարող են հասանելի լինել այնտեղ, որտեղ ադմինիստրատորները միացնում են դրանք։

**Ինչում է լավ աշխատում․**
- Նկարագրված բիզնեսի տեսակի համար բազմափուլ կայքի կառուցման պլանի ստեղծում
- Յուրաքանչյուր փուլի ինքնավար կատարում՝ կառուցվածք, բովանդակություն, նավարկում, դիզայն
- Պլանի ընթացքում սխալներից վերականգնում՝ առանց ձեռքով միջամտություն պահանջելու
- Առաջարկվող plugin-ների տեղադրում որպես կառուցման մաս
- Կոնտակտային ձևերի ստեղծում անմիջապես զրույցի միջերեսից (Superdav AI Agent v1.10.0+)
- Գործարկման հիշեցումների կամ մասնակիցների հետագա կապի համակարգում՝ առանց կրկնվող ծանուցումների, երբ հաստատման դարպասները և հիշեցումների գրառումները միացված են

**Մեկնարկային առաջարկներ․**
- *Կառուցիր լուսանկարչական պորտֆոլիոյի կայք՝ պատկերասրահի գրառման տեսակով, ամրագրման էջով և կոնտակտային ձևով։*
- *Ստեղծիր ռեստորանի կայք՝ առցանց մենյուով, աշխատանքային ժամերով և սեղանի ամրագրման հարցման ձևով։*
- *Կարգավորիր անկախ խորհրդատուի կայք՝ ծառայությունների էջերով, պորտֆոլիոյի բաժնով և բլոգով։*
- *Ավելացրու կոնտակտային ձև Կապ էջում՝ օգտագործելով կայքի կառուցողը։*
- *Կայքի գործարկման ստուգացանկի հաստատումից հետո ուղարկիր SMS հիշեցում կարգավորված շահագրգիռ կողմի կոնտակտին։*

### Դիզայն ստուդիա

**Կիզակետ․** Տեսողական անհատականացում՝ գույներ, տպագրություն, CSS և բլոկային նմուշներ։

**Հասանելի գործիքներ․** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Ինչում է լավ աշխատում․**
- Անվանված theme նախադրվածքների կիրառում (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- Գլոբալ տպագրության և գունապնակների ճշգրտում theme.json-ի միջոցով
- Անհատական CSS-ի ներարկում բրենդին հատուկ վերագրումների համար
- Էջի սքրինշոթի վերցնում և դրա վերանայում դիզայնի խնդիրների համար

**Մեկնարկային առաջարկներ․**
- *Կիրառիր warm-editorial նախադրվածքը, ապա հիմնական գույնը սահմանիր #2d6a4f։*
- *Վերցրու գլխավոր էջի սքրինշոթը և ասա, թե ինչ կբարելավեիր։*
- *Ստեղծիր բազմակի օգտագործման hero բլոկային նմուշ՝ ամբողջ լայնությամբ ֆոնային պատկերով և կենտրոնացված վերնագրով։*

### Plugin-ի կառավարիչ

**Կիզակետ․** WordPress plugin-ների հայտնաբերում, տեղադրում և կառավարում։

**Հասանելի գործիքներ․** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Ինչում է լավ աշխատում․**
- Նկարագրված օգտագործման դեպքի համար լավագույն plugin-ի առաջարկում
- Ռեեստրից կարողությունների փաթեթների տեղադրում
- Հասանելի կարողությունների կատալոգի դիտում ըստ կատեգորիայի

**Մեկնարկային առաջարկներ․**
- *Ո՞րն է լավագույն plugin-ը membership տեղեկատուի համար։*
- *Տեղադրիր WooCommerce կարողությունների փաթեթը։*
- *Ցույց տուր բոլոր հասանելի ecommerce կարողությունների փաթեթները։*

### Աջակցության օգնական

**Կիզակետ․** Կայքի բովանդակության, կարգավորումների և WordPress կազմաձևման մասին հարցերին պատասխանելը։

**Հասանելի գործիքներ․** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Ինչն է լավ անում․**
- Որոնում է ընթացիկ կայքի կարգավորումները և ընտրանքները
- Բացատրում է, թե գրառման ինչ տեսակներ, դասակարգումներ և ընտրացանկեր են կազմաձևված կայքում
- Պատասխանում է «ի՞նչ է անում այս կարգավորումը» հարցերին՝ կարդալով կենդանի արժեքները
- Ծառայում է որպես միայն կարդալու ախտորոշիչ շերտ՝ նախքան փոփոխություններ կատարելը

**Սկսելու առաջարկներ․**
- *Ի՞նչ plugin-ներ և կարգավորումներ են այժմ ակտիվ այս կայքում։*

## Superdav ավտոմատացման ինտեգրումներ

Երբ Superdav AI Agent v1.18.0 ինտեգրումները կազմաձևված են, ներկառուցված գործակալները կարող են մասնակցել ավելի անվտանգ, ժամանակացույցը հաշվի առնող ավտոմատացման աշխատանքային հոսքերին․

- **Google Calendar կարդալու գործիքները** թույլ են տալիս գործակալներին ստուգել կազմաձևված օրացույցներն ու իրադարձությունները՝ նախքան հետագա աշխատանքի սևագրումը։
- **Կոնտակտների և մասնակիցների համադրումը** օգնում է իրադարձության մասնակիցներին համապատասխանեցնել WordPress օգտատերերի կամ հայտնի կոնտակտների հետ։
- **Մարդու հաստատման դարպասները** դադարեցնում են զգայուն գործողությունները, մինչև լիազորված օգտատերը վերանայի և հաստատի դրանք։
- **Հիշեցումների գրառումները** կանխում են կրկնվող ծանուցումները, երբ ժամանակացույցով առաջադրանքները կրկին փորձարկվում կամ կրկնվում են։
- **TextBee SMS ծանուցումները** ուղարկում են կազմաձևված տեքստային հաղորդագրություններ միայն այն դեպքում, երբ SMS հավատարմագրերը և աշխատանքային հոսքի թույլտվությունները միացված են։

Խորհուրդ տրվող աշխատանքային հոսք․ խնդրեք գործակալին պատրաստել հաղորդագրությունը կամ գործողությունը, վերանայեք հաստատման հուշումը, ապա թույլ տվեք, որ հաստատված գործողությունը վերսկսվի։ Կրկնվող հիշեցումների համար պահեք հիշեցումների կրկնօրինակման կանխումը միացված, որպեսզի նույն իրադարձությունը կամ կոնտակտը կրկնակի չծանուցվի։

## Գործակալների հարմարեցում

Յուրաքանչյուր ներկառուցված գործակալ կարող է ընդլայնվել կամ փոխարինվել `gratis_ai_agent_agents` ֆիլտրի միջոցով։

### Գոյություն ունեցող գործակալին հարմարեցված համակարգային հուշում ավելացնելը

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### Նոր գործակալ գրանցելը

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

Նոր գործակալը հայտնվում է գործակալների ընտրիչում անմիջապես ֆիլտրի գործարկումից հետո։

### Ներկառուցված գործակալ հեռացնելը

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
