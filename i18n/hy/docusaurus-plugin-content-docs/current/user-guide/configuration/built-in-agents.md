# Ներ내ված Գործակիցներ (Built-in Agents)

Gratis AI Agent v1.9.0-ը տրամադրում է հինգ ներկառուցված գործակից, որոնցից յուրաքանչյուրը նախապես կազմաձևված է կենտրոնացած գործիքների բազմությամբ՝ հատուկ համակարգային պրոմպտով և սկզբնական առաջարկներով, որոնք համապատասխանում են այդ ոլորտի ընդհանուր խնդիրներին։ Գործակիցների միջև անցնելը փոխում է այն, թե ինչ կարող է անել օգնականը և ինչպես է պատասխանում՝ առանց որևէ կոնֆիգուրացիայի ձեր կողմից։

## Ի՞նչ է Agent-ը (Գործակից)

Յուրաքանչյուր գործակից ստեղծված է անունով կոնֆիգուրացիոն պրոֆիլ, որը համատեղում է.

- **Tools** — այն հնարավորությունները, որոնց թույլատրվում է օգտագործել գործակիցը (օրինակ՝ Content Writer-ը ունի մուտք գործել գրառումներ ստեղծելու հնարավորություններ; Design Studio-ն ունի մուտք CSS և theme.json հնարավորություններ)
- **System prompt** — ցուցումներ, որոնք սահմանում են գործակցի տոնը, առաջնահերթությունները և սահմանափակումները
- **Suggestions** — նախապես գրված պրոմպտներ՝ քեխատիչի ինտերֆեյսում ցուցադրվող, որոնք օգնում են արագ սկսել

## Agent Picker-ին Մուտք գործում

1. Բացեք **Gratis AI Agent** պանելը WordPress-ի ադմինի կողմից։
2. Սեղմեք քեխատիչի վերևซ้ายում գտնվող **agent icon**-ին (հատուկ պատկերը փոխվում է ակտիվ գործակցի արտացոլման համար)։
3. **Agent Picker**-ը բացվում է որպես ձևաթուղթային աղյուսակի վերևի շերտ (overlay)։ Յուրաքանչյուր գործակից ներկայացված է իր պատկերով, անունով և մեկ տողով նկարագրությամբ։
4. Սեղմեք գործակցի տողի վրա՝ այն ակտիվացնելու համար։ Քեխատիչի վերևի հատվածը անմիջապես թարմանում է։

Դուք կարող եք նաև միջհարցազրույթի ընթացքում գործակիցները փոխել՝ նոր գործակցի համակարգային պրոմպտը կիրառվելով հաջորդ հաղորդագրությունից սկսած։

## Հինգ Ներ내ված Գործակիցներ (The Five Built-in Agents)

### Content Writer (Բովանդակության Գրող)

**Կենտրոնացում:** Գրառումների, էջերի և կոնտակտային ձևերի ստեղծումը և խմբագրումը։

**Հասանելի գործիքներ:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`

**Ինչ է լավ անում.**
- Բլոգային գրառումներ կազմել և հրապարակել կարճ տեքստի կամ սխեմանով։
- Նոր կայքի համար նախատեսված էջերի խմբեր ստեղծել։
- Հետևողականության և հարցումների ձևաչափեր (forms) կառուցել։
- URL-ից կամ որոնման միջոցով գտնված գրառումների վրա հատուկ պատկերներ տեղադրել։

**Սկզբնական առաջարկներ.**
- *Գրեք 500 բառանոց բլոգային գրառում WordPress multisite-ի առավելությունների մասին:*
- *Ստեղծեք «Մեր մասին», «Ծառայություններ» և «Կապ» էջերը և հրապարակեք դրանք:*
- *Հետևողականության էջին ավելացրեք գրանցման/հարցումների ձևաչափ։*

---

### Site Builder (Կայքի կառուցիչ)

**Կենտրոնացում.** Մեկ հրահանգով ամբողջական կայք ստեղծելը։

**Մասնակցող գործիքներ:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`

**Ինչ է լավ անում.**
- Նկարագրված բիզնես տեսակի համար մի քանի փուլով կայքի կառուցման պլան ստեղծել։
- Յուրաքանչյուր փուլին ինքնուրույն իրականացնել՝ կառուցվածքը, բովանդակությունը, նավարկությունը և դիզայնը։
- Պլանի ընթացքում սխալներից վերականգնվել առանց ձեռքով միջամտության։
- Կայքի կառուցման մաս որպես ներառված հարմարեցված պլագիններ տեղադրել։
- Հարցումների ձևաչափեր ուղղակի չատային ինտերֆեյսից ստեղծել (Superdav AI Agent v1.10.0+):

**Սկզբնական առաջարկներ.**
- *Նկարագրության գրառումով պատկերական պորտֆոլիո կայք ստեղծեք՝ գալերի գրառման տեսակով, գրանցման էջով և կապի ձևաչափով:*
- *Ռեստորանային կայք ստեղծեք՝ առցանց մենյուով, բացուցշարժական ժամերով և սեղանների ամրագրման հարցումների ձևաչափով:*
- *Ֆրիլանսերների խորհրդատվական կայք ստեղծեք՝ ծառայությունների էջերով, պորտֆոլիո բաժնով և բլոգով։*
- *Site Builder-ը օգտագործելով Contact էջում կապի ձևաչափ ավելացրեք:*

---

### Design Studio (Դիզայնի ստուդիա)

**Կենտրոնացում.** Տեսողական անհատականացում՝ գույներ, տիպոգրաֆիա, CSS և բլոկային օրինակներ։

**Մասնագիտացված գործիքներ**

**Կիրառելի գործիքներ:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Ինչ է լավ:**
- Նշված թեմա պրեսետների (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog) կիրառում։
- Ընդհանուր տիպոգրաֆիայի և գույների паլիտրերը կարգավորում theme.json-ով։
- Բրենդին հատուկ գերագրումների համար ստեղծվող custom CSS ներարկում։
- Էջի սքրինշոթ վերցնելը և դրա դիզայնի խնդիրները վերանայելը։

**Սկզբնական առաջարկներ:**
- *Կիրառեք warm-editorial պրեսետը, այնուհետև հիմնական գույնը սահմանեք #2d6a4f-ի:*
- *Վերցրեք տանփայլի (homepage) սքրինշոթ և ասեք, թե ինչը կարող եք բարելավել:*
- *Ստեղծեք կրկնվող հերոսային բլոկի պաтерն՝ լիարժեք լայնությամբ պատկերով և կենտրոնացված վերնագրով:*

---

### Plugin Manager (Պլագինների կառավարիչ)

**Կենտրոնացում:** WordPress-ի պլագինները գտնելու, տեղադրելու և կառավարելու համար։

**Կիրառելի գործիքներ:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Ինչ է լավ:**
- Նկարագրված օգտագործման դեպքում լավագույն պլագինը առաջարկել։
- ռեգիստրից ability packs տեղադրել։
- հասանելի ability կատալոգը բաժիններով անցնել։

**Սկզբնական առաջարկներ:**
- *Ի՞նչ է լավագույն պլագինի ընտրությունը անդամակցության դիրքերի համար:*
- *Տեղադրեք WooCommerce ability packs-ը:*
- *Ցուցադրեք բոլոր հասանելի e-commerce ability packs-ները:*

---

### Support Assistant (Աջակար ասիստենտ)

**Կենտրոնացում:** Սайտի բովանդակության, կարգավորումների և WordPress-ի կոնֆիգուրացիայի վերաբերյալ հարցերին պատասխանելը։

**Կիրառելի գործիքներ:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Ինչ է լավ գործում:**
- Ընթացիկ կայքի կարգավորումները և տարբերակները որոնելու համար։
- Բացատրել, թե ինչպես են կարգավորված post types (պոստի տեսակներ), taxonomies (տաքսոնոմին) և menus (փոխանցման մենյուներ) կայքում։
- «Ի՞նչ է անում այս կարգավորումը» հարցերին պատասխանելով ապրող արժեքները կարդալով։
- Փոփոխություններ կատարելուց առաջ ընթերցման միայն համար (read-only) վիճակ լինելով։

**Սկզբնական առաջարկներ:**
- *Ի՞նչ plugin-ներ և կարգավորումներ են ներկայումս ակտիվ այս կայքում:*
- *Ցուցադրել այս կայքում գրանցված բոլոր custom post types-ը։*
- *Որո՞նք են գոռոյական մենյուները և դրանք որտեղ են տեղադրված։*

## Ագենտների Կարգավորումներ (Customising Agents)

Յուրաքանչյուր ներկառուցված agent-ը կարելի է ընդլայնել կամ փոխարինել `gratis_ai_agent_agents` filter-ի միջոցով։

### Գոյություն ունեցող agent-ին սեփական system prompt (հիմնական հրահանգ) ավելացնելը

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### Նոր agent գրանցելը

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

Նոր գործակցում հայտնվում է Agent Picker-ի մեջ, որը անմիջապես фільտրի հետևից։

### Նկատելի ինտեգրված գործակցի հեռացում

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
