# Плановի համար կիսվող հղումների կառավարում {#managing-shareable-links-for-plans}

Ultimate Multisite-ը թույլ է տալիս ձեր պլանային հղումները ուղղակիորեն կիսել ունեցած գործընկերների հետ։ Սա անելու համար գնացեք ձեր ցանկացած պլանի խմբագրման էջ և սեղմեք «_Click to copy the Shareable Link_» (Կպչում եք կիսվող հղումը) ღիթը։

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_Պլանի խմբագրման էջում կարող եք գտնել կիսվող հղումներ ստեղծելու ღիթը_

Նկատի ունեցեք, որ այս մեխանիզմը օգտագործում է բաժանարար ամենօրյա հաճախականությունը՝ հղումը կառուցելու համար։ Սա նշանակում է, որ եթե ձեր սովորական հաշվարկը ամսական է, օրինակ, վերջում ունեք այսպիսի հղում. [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Եթե ցանկանում եք, որ կիսվող հղումը սովորական դառնա եռամսյակային, ապա պետք է վերջում ավելացնեք 3-ը.

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Եթե ցանկանում եք ընտրել _premium_ պլանը ամենամյա ժամանակացույցով, ապա կավելացնեք 1-ը և հետո բառը՝

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Հիշեք, որ կարող եք միայն մեկ հղում չունենալ տարբեր հաշվարկային հաճախականությունների համար։ Դուք պետք է կիսվեք մեկ հատուկ հղումով յուրաքանչյուր ձեր հաշվարկային հաճախականության համար։

Մի անգամ երբ կիսվում եք հղումը ձեր հաճախորդների հետ, այն նրանց ուղղակի կմուտեցնի ձեր գրանցման ընթացքի առաջին քայլին։
