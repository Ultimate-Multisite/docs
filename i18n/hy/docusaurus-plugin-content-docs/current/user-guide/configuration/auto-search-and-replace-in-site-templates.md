# Կ自動 որոնում և փոխարինում վեբ տիպերում (v2) {#auto-search-and-replace-in-site-templates-v2}

_**Այս ուղեցույցը պահանջում է Ultimate Multisite-ի 2.x տարբերակը։**_

Ultimate Multisite-ի ամենաուժեղ հնարավորություններից մեկը կարող եք ավելացնել ցանկացած տեքստ, գույն և ընտրվող դաշտեր գրանցման ձևի մեջ։ Երբ մենք այս տվյալները կհավաքենք, կարող ենք օգտագործել դրանք՝ նախապես լրացնելու վեբ տիպերի որոշ հատվածներում ընտրված բովանդակությունը։ Այնուհետև, երբ նոր սայթը կհրապարակվի, Ultimate Multisite-ը փոխարինում է այս դիրքերը իրական տեղեկություններով, որոնք մուտքագրվել են գրանցման ժամանակ։

Օրինակ, կարող եք ստեղծել ձեր վեբ տիպերը՝ օգտագործելով placeholders (դիրքավորումներ)։ Placeholders-ը պետք է ավելացվեն երկու կրկնակի կրկնակի փակագծերով՝ - {{placeholder_name}}։

Այնուհետև, կարող եք պարզապես ավելացնել համապատասխան գրանցման դաշտ՝ այդ տվյալները հավաքելու համար։

Ձեր հաճախորդը հետո կկարողանա այն դաշտը լրացնել գրանցման ընթացքում։

Ultimate Multisite-ը հետո ավտոմատ կերպով փոխարինում է placeholders-ները հաճախորդի կողմից տրամադրված տվյալներով։

## **«վեբ տիպերը դատարկ placeholder-ներով» խնդիրը լուծել** {#solving-the-template-full-of-placeholders-problem}

Բոլորը հիանալի է, բայց մենք հանդիպում ենք անհարմար խնդրի հետ. այժմ մեր վեբ տիպերը՝ որոնք կարող են այցելել մեր հաճախորդները, լի են անգեղ placeholder-ներով, որոնք շատ բան չեն ասում։

Դրան հաղթահարելու համար մենք առաջարկում ենք fake values (կեղծ արժեքներ) սահմանել placeholder-ների համար և այդ արժեքները օգտագործում ենք դրանց բովանդակությունը որոնելու և փոխարինելու համար վեբ տիպերի վրա, երբ ձեր հաճախորդները այցելում են։

Դուք կարող եք մուտք գործել template placeholders-ների խմբագրիչը՝ գնալով **Ultimate Multisite > Settings > Sites**-ին, սահեցնել Site Template Options (Վեբ տիպերի տարբերակներ) հատվածը և այնուհետև սեղմել **Edit Placeholders** հղումը։

![Site Template Options area in the Sites settings page](/img/config/settings-sites-templates-section.png)

Սա կտանի ձեզ placeholders-ների բովանդակության խմբագրիչին, որտեղ կարող եք ավելացնել placeholders և դրանց համապատասխան բովանդակությունը։

![Տողերի տեղադրման խմբագրի մուտքը](/img/config/settings-sites-templates-section.png)
