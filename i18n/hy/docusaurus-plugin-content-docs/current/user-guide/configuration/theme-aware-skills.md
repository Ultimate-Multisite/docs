# Թեմաներին համապատասխանու հմտություններ {#theme-aware-skills}

Superdav AI Agent v1.10.0-ը թողարկվել է չոր նոր, թեմաներին համապատասխանու ներկառուցված հմտություններով, որոնք ավտոմատ կերպով հարմարվում են ձեր WordPress-ի ընթացիկ թեմային։ Այս հմտությունները մատուցում են մասնագիտ화된 ուղեցույցներ և կարողություններ՝ հարմարեցված ձեր թեմայի կառուցվածքին և առանձնահատկություններին։

## Ի՞նչ են թեմաներին համապատասխանու հմտությունները (Theme-Aware Skills) {#what-are-theme-aware-skills}

Թեմաներին համապատասխանող հմտությունները նախապես կազմաձևված գիտելիքների բազաներ և գործիքների բ conjuntoներ են, որոնք AI օգնականը ավտոմատ կերպով ընտրում է ձեր սайտի վրա ներկայացած WordPress թեմայի հիման վրա։ Երբ դուք փոխում եք թեմաները, օգնականի հասանելի հմտությունները ավտոմատ կերպով թարմացվում են՝ առանց ձեռքով ցանկացած կարգավորման։

Յուրաքանչյուր հմտության մեջ ներառված է.

- **Թեմաների հատուկ փաստաթղթեր** — թեմայի օգտագործման և անհատականացման վերաբերյալ ուղեցույցներ
- **Բլոկների և ձևանմուշների (patterns) հղումներ** — հասանելի բլոկները, ձևանմուշները և դիզայնի տարբերակները
- **Անհատականացման օրինակներ** — ընդհանուր առաջադրանքների համար կոդային հատվածներ և կոնֆիգուրացիայի ձևանմուշներ
- **Լավագույն պրակտիկաներ (Best practices)** — թեմայի կառուցվածքի և աշխատանքի ընթացքի համար առաջարկություններ

## Հասանելի թեմաներին համապատասխանող հմտություններ {#available-theme-aware-skills}

### Бլոկային թեմաներ (Block Themes) {#block-themes}

**Կիրառվում է.** WordPress-ի բլոկային (Full Site Editing) կառուցվածքով թեմաների համար։

Бլոկային թեմայի հմտությունը մատուցում է ուղեցույցներ.

- Տողերի և տեխնոլոգիական տարրերի օգտագործման համար բլոկային խմբագրիչով
- Бլոկային ձևանմուշների (block patterns) և վերաօգտագործելի բլոկների հետ աշխատելու համար
- Ընդհանուր ոճերի և theme.json կարգավորումների անհատականացման համար
- Theme blocks-ի և variations-ի օգտագործման համար
- Ձեր սайտի համար հատուկ բլոկային ձևանմուշներ կառուցելու համար

**Ավտոմատ ակտիվացվում է, երբ.** Ձեր ընթացիկ թեմաները բլոկային են (աջակցում է `block-templates` հնարավորությանը)։

### Դասական թեմաներ (Classic Themes) {#classic-themes}

**Կիրառվում է.** Թվական WordPress թեմաների համար, որոնք օգտագործում են PHP տեխնոլոգիաներ և դասական խմբագրիչը։

Դասական թեմաների հմտությունը մատուցում է ուղեցույցներ.

PHP տեխնիկական տողերով և hooks-ով աշխատելը
Թեմայի տեսքը Customizer-ի միջոցով անհատականացում
Widget areas և sidebars օգտագործելը
CSS փոփոխություն և child theme մշակում
Theme hierarchy (թեմայի հերարխիա) և template tags-երի ըմբռնում

**Ավտոմատ կլուցվում է, երբ:** Ձեր ակտիվ թեման classic (non-block) թեմա է։

### Kadence Blocks {#kadence-blocks}
**Կիրառվում է.** Տեսակներով բլոկային դիզայնի համար Kadence Blocks plugin օգտագործող կայքերի համար։

Kadence Blocks հմտությունը տրամադրում է ուղեցույց՝ հետևելու համար.
- Kadence-ի առաջադեմ բլոկային գրադարանների (Hero, Testimonials, Pricing և այլն) օգտագործմանը։
- Kadence block-երի կարգավորմանը և responsive տարբերակներին։
- Կառուցել custom layout-ներ Kadence-ի grid և container blocks-ով։
- Kadence blocks-ը ձեր թեմայի հետ ինտեգրելու համար։
- Օգտագործել Kadence-ի design system-ը և presets-ները։

**Ավտոմատ կլուցվում է, երբ:** Kadence Blocks plugin-ը ակտիվ է ձեր կայքում։

### Kadence Theme {#kadence-theme}
**Կիրառվում է.** Block-based դիզայնի և անհատականացման համար Kadence theme օգտագործող կայքերի համար։

Kadence Theme հմտությունը տրամադրում է ուղեցույց՝ հետևելու համար.
- Կարգավորել Kadence թեման global styles և theme.json-ով։
- Օգտագործել Kadence-ի ներկառուցված block patterns-ը և templates-ը։
- Կարգավորել Kadence theme-ի կարգավորումները և տարբերակները։
- Custom design-ներ կառուցել Kadence-ի design system-ով։
- Kadencen ինտեգրել հայտնի plugin-ների և tools-երի հետ։

**Ավտոմատ կլուցվում է, երբ:** Kadence theme-ը ձեր ակտիվ թեման է։

## Ինչպես են ընտրվում Skills-ները (Հմտությունները) {#how-skills-are-selected}

Assistant-ը յուրաքանչյուր հաղորդագրության ժամանակ ավտոմատ կերպով ստուգում է ձեր ակտիվ թեման և տեղադրված plugin-ները։ Եթե համապատասխան, թեմային զգալի skill (theme-aware skill) հասանելի է, այն ավտոմատ կերպով լիցքավորվում է assistant-ի context-ում։ Ձեզ անհրաժեշտ չէ ձեռքով ակտիվացնել կամ փոխել skills-ները։

### Մի քանի Skills {#multiple-skills}

Եթե ձեր կայքում մի քանի հմտություններ են գործում (օրինակ, եթե ունեք ակտիվ և՛ Kadence Blocks, և՛ Kadence Theme), օգնականը մուտք ունի բոլոր համապատասխան հմտություններին և կարող է հղումներ կատարել յուրաքանչյուրից։

### Թեման փոխելը (Switching Themes) {#switching-themes}

Երբ դուք փոխում եք ակտիվ թեման, օգնականի հասանելի հմտությունները ավտոմատ կերպով թարմացվում են հաջորդ հաղորդագրության ժամանակ։ Օրինակ.

1. Դուք օգտագործում եք բլոկային թեմա՝ **Block Themes** հմտությունը ակտիվ է։
2. Դուք անցնում եք դեպի կլասիկ թեմա (classic theme)։
3. Հաջորդ հաղորդագրության ժամանակ, **Classic Themes** հմտությունը ավտոմատ լիցքավորվում է, և **Block Themes** հմտությունը այլևս հասանելի չի լինի։

## Թեմա-հասկանալի հմտությունների օգտագործում (Using Theme-Aware Skills) {#using-theme-aware-skills}

Թեմա-հասկանալի հմտությունները օգտվելու համար պարզապես նկարագրեք, թե ինչ ուզում եք անել քեզ հետ զրույցի ինտերֆեյսում։ Օգնականը ավտոմատ կերպով հղում կկատարի համապատասխան հմտության ուղեցույցներին։

### Օրինակներ (Example Prompts) {#example-prompts}

**Block Themes-ի համար:**
> «Ստեղծել հերոսային հատված՝ հետևի պատկերով և կենտրոնացված տեքստով բլոկային օրինաչափությունների միջոցով»։

**Classic Themes-ի համար:**
> «Օգտագործել զուգակից թեմայի (child theme) միջոցով սեղմի բազայում կողմնորոշման վահանակի համար հատուկ վի젯ային տարածք»։

**Kadence Blocks-ի համար:**
> «Կառուցել հաճախորդների գնահատականների (testimonials) հատված՝ օգտագործելով Kadence Testimonials բլոկը»։

**Kadence Theme-ի համար:**
> «Կարգավորել վերևի մասի դասավորությունը և նավիգացիոն մենյուի ոճը»։

Օգնականը կտա թեմային հատուկ ուղեցույցներ և կոդի օրինակներ, որոնք հարմարված են ձեր ակտիվ թեմաների և պլագինների համար։

:::note
Թեմա-հասկանալի հմտությունները ավտոմատ կերպով հասանելի են Superdav AI Agent v1.10.0 և ավելի նոր տարբերակներում։ Անհրաժեշտ է լրացուցիչ կարգավորում կամ կոնֆիգուրացիա չունենաք։
:::
