# ဘာကြောင့် Ultimate Multisite

ဒီဆောင်းပါးကို ရောက်လာတယ်ဆိုရင် ကျွန်တော်တို့ရဲ့ plugin က သင့်စီးပွားရေး အိုင်ဒီယာကို အကောင်အထည်ဖော်ဖို့ ဖြေရှင်းနည်း ဖြစ်နိုင်တယ်လို့ ကြားမိလိမ့်မယ်။ ဒီစာပိုဒ်ရဲ့ အဆုံးမှာ လိုအပ်တဲ့ အဖြေတွေ သင်ရရှိမယ်လို့ မျှော်လင့်ပါတယ်။

## Ultimate Multisite ဆိုတာ ဘာလဲ?

Ultimate Multisite ဟာ WordPress Multisite plugin တစ်ခုဖြစ်ပြီး သင့်ကောက်သည်တွေကို WaaS (Website as a Service) ပေးနိုင်ဖို့ စွမ်းဆောင်ရည်ပေးပါတယ်။ ကျွန်တော်တို့ရဲ့ plugin နဲ့ဆိုရင် အမျိုးမျိုးသော subscription tiers တွေ ရှိနိုင်ပြီး ဝန်ဆောင်မှုယူတဲ့သူတွေဟာ သင့် Multisite network မှာ site တစ်ခုကို host လုပ်ဖို့အတွက် သင့်ကို ပုံမှန်ကြေးတစ်ခု ပေးဆောင်နိုင်မှာ ဖြစ်ပါတယ်။

Ultimate Multisite က သင့်ငွေပေးချေမှုတွေကို မတူညီတဲ့ gateways တွေ (ဥပမာ- WooCommerce ဒါမှမဟုတ် Stripe) နဲ့ လက်ခံနိုင်စေတယ်၊ ဝန်ထမ်းအသစ်တွေ network မှာ မှတ်ပုံတင်တဲ့အခါ သင့်ဘဝကို ပိုလွယ်ကူစေမယ့် site templates တွေကို ဖန်တီးနိုင်တယ်၊ domain တွေကို map လုပ်နိုင်တယ်၊ နောက်ပြီး သင့် customer တွေအတွက် စိတ်ကြိုက် အတွေ့အကြုံတစ်ခု ဖန်တီးဖို့ registration forms တွေကို ကိုယ်ပိုင်စနစ်နဲ့ ပြုပြင်နိုင်ပါတယ်။

ဒါက အစပဲ ရှိသေးပါတယ်၊ ဘာလို့လဲဆိုတော့ add-ons နဲ့ webhooks တွေနဲ့ သူ့ရဲ့ လုပ်ဆောင်ချက်တွေကို တိုးချဲ့နိုင်လို့ပါပဲ။

## အခြားလုပ်ဆောင်ချက်များ

Ultimate Multisite က အောက်ပါအရာတွေကိုလည်း ပေးဆောင်ပေးပါတယ်-

  * Managed WordPress hosting providers တွေနဲ့ Domain Mapping integration: Ultimate Multisite ဟာ domain တစ်ခုကို map လုပ်လိုက်တဲ့အခါ WP Engine၊ Cloudways၊ cPanel၊ RunCloud၊ Closte စတဲ့ admin panels တွေထဲမှာ map လုပ်ထားတဲ့ domain တွေကို အလိုအလျောက် ထည့်ပေးပါတယ်။ ဒါကြောင့် သင်ကိုယ်တိုင် မလုပ်စရာ မလိုတော့ပါဘူး။ တချို့ကိစ္စတွေမှာ SSL certificate ကို fetch လုပ်ဖို့တောင် trigger လုပ်ပေးနိုင်ပါတယ်။

  * Webhooks: Ultimate Multisite မှာ သင့် network ကို ဝန်ဆောင်မှု အမျိုးမျိုးနဲ့ ချိတ်ဆက်နိုင်တဲ့ စွမ်းအားကောင်းတဲ့ Webhook system တစ်ခု ပါဝင်ပါတယ်။ Zapier နဲ့ တိုက်ရိုက်ပေါင်းစပ်မှုကိုပါ ကျွန်တော်တို့ ပေးထားပါတယ်၊ ဒါက သင့် network ကို သူတို့ရဲ့ platform ပေါ်မှာ ရရှိနိုင်တဲ့ app နဲ့ service ၁၀၀၀ ကျော်ကို ဖွင့်ပေးလိုက်သလို ဖြစ်စေပါတယ်။

* Ձեր հաճախորդներին օգնելու համար ձեր սիրելի էջի կառուցողով (page builder)՝ Ultimate Multisite-ը ունի հարմար add-on՝ Admin Pages Creator, որը հիմնականում թույլ է տալիս Elementor, Beaver Builder և Brizy template-ներ օգտագործելով ձեր հաճախորդների dashboard-ի համար ամբողջությամբ կատրիվացված Admin Pages ավելացնել։

* Dashboard-ի փորձառությունը ձեր հաճախորդների համար անհատականացնելու համար երկու տարբեր Admin Theme՝ Check Material WP և PRO:

Եվ ավելի առաջադեմ բաներ.

* Template Overrides-ի (Template Overrides) աջակցություն. Ultimate Multisite-ով կարող եք փոխարինել գրանցման հոսքի ստանդարտ HTML template-ները ձեր սեփական տարբերակներով՝ առանց անհանգստանալու, որ պետք է փոխեք core plugin-ի ֆայլերը։

* Search and Replace API. Դուք կարող եք նաև կայքի template-ները Annotation-ներով (օրինակ՝ `{{site_name}}` տեսակի) լրացնել և օգտագործել Search and Replace API-ը՝ այդ placeholder-ները ավտոմատ կերպով ձեր հաճախորդի կողմից գրանցվելուց անմիջապես տրամադրված տեղեկություններով լրացնել։

## Օգտագործման դեպքեր (Use cases)

Այժմ ժամանակն է Ultimate Multisite-ի հնարավոր օգտագործման մի քան հատկությունները տեսնելու համար։

### Դեպք 1. Գործակալություն (An Agency)

Տվյալով, գործակալության հիմնական հմտությունները կայանում են կայքերի դիզայնում այնպիսի կողմերով, ինչպես օրինակ՝ տեղադրման կամ մարքեթինգի ներառումը որպես լրացուցիչ ծառայություններ։

Գործակալությունների համար Ultimate Multisite-ը ներկայացնում է անհավանական արժեքային առաջարկ՝ բազմաթիվ կայքեր հյուպտոսելով և մեկ պլատֆորմի վրա կառավարելու ունակությամբ։ Ավելին, այն գործակալությունների համար, որոնք ստանդարտացնում են իրենց դիզայնը որոշակի theme-ներով (օրինակ՝ GeneratePress, Astra, OceanWP կամ այլ) Ultimate Multisite-ի ունակությունները կարող են օգտագործել այդ theme-երը ավտոմատ կերպով ակտիվացնել յուրաքանչյուր նոր կայքի համար։

Ultimate Multisite-ը թույլ է տալիս plugin-ներն ու theme-ները վերահսկել՝ յուրաքանչյուր product-ի հիմնավորման դեպքում, որը թույլ է տալիս plugin-ներին և theme-ներին հասանելի կամ թաքցնելու, ինչպես նաև դրանց ակտիվացման վիճակը՝ նոր կայքի համար ստեղծելիս։

Գործակիցները կստանան հանգիստ՝ Ultimate Multisite-ի միջոցով, որը թույլ է տալիս իրենց անել այն, ինչ ամենագարն են՝ եզակի වෙබ්անայք դիզայնել։

### Դեպք 2. Խորհրդային մատակարար (Niche Provider)

Գոյություն ունի հին արվածք, որը ասում է՝ «Մեկ բան անիր և այն լավ անիր»։ Շատ մասնագետների համար դա նշանակում է ստեղծել արտադրանք կամ ծառայություն մեկ հիմնական գաղափարի շուրջ։

Եթե դուք կենտրոնացած եք մի հատուկ կայքերի ցանցի զարգացման վրա, ամենայն հավանականությամբ ցանկանում եք առաջարկել ծառայություններ ընդհանուր շրջանակի և պլատֆորմի հիման վրա։ Այս դեպքում կայքերի տեխնոլոգիական ձևանմուշները (site templates) կարող են օգնել։

Ultimate Multisite-ի իսկական նորարար հատկություններից մեկը՝ տեխնոլոգիական ձևանմուշով կայքը այն դեպքն է, երբ թեմայի (theme) տեղադրված և ակտիվացված է, անհրաժեշտ պլագինները նույնպես տեղադրված և ակտիվացված են, ինչպես նաև օրինակային գրառումներ կամ էջեր ստեղծված են։ Երբ հաճախորհը ստեղծում է նոր կայք՝ հիմնվելով այդ ձևանմուշի վրա, ձևանմուշի բովանդակությունը և կարգավորումները պատճենվում են նոր ստեղծված կայքում։

Խորհրդային կայքերի և ծառայությունների մատակարարի համար դա աննախադեպ առավելություն է՝ հնարավորության տեսանկյունից, որով կարելի է անմիջապես ստեղծել պատրաստ կայք՝ հատուկ պլագիններով և դիզայնով։ Հաճախորհը պետք է մուտքագրի միայն նվազագույն տվյալներ՝ ծառայությունը շարունակելու համար։

### Դեպք 3. WordPress Տեղակայում (Web Hosting)

WordPress կայքեր հոස්ելու բազմաթիվ միջոցներ կան, բայց երբեք այն այնքան պարզ չէ, որ ուղղակի տրամադրել վեբ տարածքը հաճախորհին՝ WordPress-ի նախապես տեղադրված տարբերակով։ Սա այն պատճառով է, որ իմաստալից ծառայություն մատուցելու համար անհրաժեշտ է մի շարք որոշումներ և հաշվի առումներ համատեղել։

Ultimate Multisite-ը գերազանցում է այս ոլորտում՝ WordPress կայքերի հոස්տինգի համար ամբողջական պատրաստի լուծում (turnkey solution) տրամադրելով։ Լուծման մեջ ներառված են բաժանարար ծառայությունների, վճարումների մշակման, գնումների ձևաչափերի (checkout forms), զեղչային կոդերի և հաճախորհի հետ շփման համար անհրաժեշտ բոլոր հիմնական մեխանիզմները։

WordPress Multisite-ը ճիշտ տեղադրելու, կազմաձևելու և պահպանելու համար անհրաժեշտ մեծ մասնակի աշխատանքը հեշտացվում է Ultimate Multisite-ի միջոցով։ Սա այնքան հարմար է ցանցային ադմինիստրատորների համար, որ նրանք պետք է հաշվի առնեն միայն այն կողմերը, որոնք վերաբերում են իրենց ծառայությանը կամ հատուկ ոլորտին՝ օրինակ՝ արտադրանքի տiers-ները, գնանշումները և ծառայությունների առաջարկները։

Ultimate Multisite-ի հետ ինտեգրվելուց ուզած դեվելոփերների համար լուծումը նաև առաջարկում է համապարփակ RESTful API և Webhooks՝ իրադարձությունների կանչերի (event notification) համար։

Այնուամենայնիվ, ցանկացած արտաքին plugin-ների և լիցենզիաների վրա հենվելու անհրաժեշտության առանց Ultimate Multisite-ի՝ այն առաջարկում է հարուստ բնութագրերով և համեմատելի լուծում Wix-, Squarespace-ի, [WordPress.com](https://WordPress.com) և այլ платформների հետ։
