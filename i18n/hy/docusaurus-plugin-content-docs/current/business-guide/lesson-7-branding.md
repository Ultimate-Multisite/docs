# Դաս 7․ Դարձնել այն ձերը {#lesson-7-making-it-yours}

Ձեր հաճախորդները երբեք չպետք է զգան, թե օգտագործում են «ինչ-որ WordPress plugin»։ Նրանք պետք է զգան, որ օգտագործում են FitSite -- իրենց ոլորտի համար ստեղծված հարթակ։ Այս դասը ներկայացնում է բրենդավորումը, white-labeling-ը և հարթակն ապրանքի նման զգացնել տալը։

## Որտեղ կանգ առանք {#where-we-left-off}

FitSite-ն ունի աշխատող checkout հոսք, որը ֆիթնես ստուդիաների սեփականատերերին plan-ի ընտրությունից հասցնում է գործող կայք։ Այժմ ամբողջ փորձառությունը դարձնում ենք համահունչ, բրենդավորված ապրանքի նման։

## Ձեր հարթակի դոմենը {#your-platform-domain}

Ձեր բրենդի հիմքը ձեր դոմենն է։ FitSite-ի համար՝

- **Հիմնական դոմեն**: `fitsite.com` (ձեր մարքեթինգային կայքը և ցանցի արմատը)
- **Հաճախորդների կայքեր**: `studioname.fitsite.com` (ենթադոմեններ)
- **Սեփական դոմեններ**: Growth և Pro plan-երի հաճախորդները կարող են կապել իրենց սեփական դոմենը

### Ձեր դոմենի կարգավորումը {#setting-up-your-domain}

1. Գրանցեք ձեր հարթակի դոմենը
2. Ուղղեք այն ձեր հոսթինգ մատակարարին
3. Կարգավորեք wildcard DNS (`*.fitsite.com`) հաճախորդների ենթադոմենների համար
4. Համոզվեք, որ wildcard SSL-ը ակտիվ է

Մանրամասն հրահանգների համար տեսեք [Ինչպես կարգավորել դոմենի կապումը](/user-guide/domain-mapping/how-to-configure-domain-mapping)։

## Admin փորձառության White-Labeling {#white-labeling-the-admin-experience}

Երբ ֆիթնես ստուդիայի սեփականատերը մուտք է գործում իր կայքի dashboard, նա պետք է տեսնի ձեր բրենդը, ոչ թե WordPress-ի կամ Ultimate Multisite-ի բրենդավորումը։

### Անհատական մուտքի էջ {#custom-login-page}

Հարմարեցրեք WordPress-ի մուտքի էջը, որպեսզի ցուցադրվի՝

- Ձեր FitSite լոգոն
- Ֆիթնեսին համապատասխան ֆոնային պատկերներ
- Ձեր բրենդային գույները

### Dashboard-ի բրենդավորում {#dashboard-branding}

Օգտագործեք [Admin Page Creator](/addons/admin-page-creator) addon-ը կամ անհատական CSS՝

- WordPress-ի լոգոն ձեր FitSite լոգոյով փոխարինելու համար
- Admin գունային սխեման ձեր բրենդին համապատասխանեցնելու համար
- Ավելացնելու անհատական dashboard widget՝ ֆիթնեսին հատուկ արագ հղումներով և օգնության ռեսուրսներով

### Անհատական Admin էջեր {#custom-admin-pages}

Մտածեք անհատական admin էջեր ստեղծելու մասին, որոնք ֆիթնես ստուդիաների սեփականատերերի համար ամենաարդիական գործողություններն են ցուցադրում՝

- «Խմբագրել ձեր դասերի ժամանակացույցը»
- «Թարմացնել մարզիչների պրոֆիլները»
- «Դիտել ձեր կայքը»

Սա նվազեցնում է սովորելու բարդությունը՝ ոլորտին համապատասխան գործողությունները բերելով առաջ և կենտրոն՝ դրանք ստանդարտ WordPress menu-ում թաքցնելու փոխարեն։

## Ձեր հաղորդակցությունների բրենդավորումը {#branding-your-communications}

Յուրաքանչյուր էլ․ նամակ, հաշիվ-ապրանքագիր և ծանուցում պետք է ամրապնդի ձեր բրենդը։

### Համակարգային էլ․ նամակներ {#system-emails}

Գնացեք **Ultimate Multisite > Settings > Emails** և հարմարեցրեք բոլոր համակարգային էլ․ նամակները՝

- **Ուղարկողի անուն**: FitSite
- **Ուղարկողի էլ․ հասցե**: hello@fitsite.com
- **Էլ․ նամակների ձևանմուշներ**: Օգտագործեք ձեր բրենդային գույներն ու լոգոն
- **Լեզու**: Ամբողջությամբ ֆիթնեսին հատուկ

Հարմարեցնելու հիմնական էլ․ նամակները՝

| Էլ․ նամակ | Ընդհանուր տարբերակ | FitSite տարբերակ |
|-------|----------------|-----------------|
| Բարի գալուստ | «Ձեր նոր կայքը պատրաստ է» | «Ձեր ֆիթնես ստուդիայի կայքը գործում է» |
| Վճարման ստացական | «Վճարումը ստացվել է» | «FitSite subscription-ի վճարումը հաստատված է» |
| Փորձաշրջանի ավարտ | «Ձեր փորձաշրջանը շուտով ավարտվում է» | «Ձեր FitSite փորձաշրջանն ավարտվում է 3 օրից -- պահեք ձեր ստուդիայի կայքը գործող» |

### Հաշիվ-ապրանքագրեր {#invoices}

Հարմարեցրեք հաշիվ-ապրանքագրերի ձևանմուշները՝ ներառելով՝

- Ձեր FitSite լոգոն և բրենդային գույները
- Ձեր բիզնեսի տվյալները
- Ֆիթնեսին հատուկ product անուններ (ոչ ընդհանուր plan ID-ներ)

## Հաճախորդին տեսանելի կայքը {#the-customer-facing-site}

Ձեր հիմնական դոմենը (`fitsite.com`) պետք է ունենա մարքեթինգային կայք, որը վաճառում է հարթակը։ Սա առանձին է Ultimate Multisite network admin-ից -- այն ձեր բիզնեսի հանրային դեմքն է։

Հիմնական էջերը՝

- **Գլխավոր էջ**: Հստակ արժեքային առաջարկ ֆիթնես բիզնեսների համար
- **Հնարավորություններ**: Ինչ է անում FitSite-ը՝ ֆիթնեսի լեզվով
- **Գներ**: Ձեր երեք plan-երը՝ ոլորտին հատուկ հնարավորությունների համեմատություններով
- **Օրինակներ**: Ցուցադրեք հարթակի վրա կառուցված կայքերը
- **Գրանցվել**: Հղումներ դեպի ձեր checkout ձևը

:::tip Ձեր մարքեթինգային կայքը կարող է լինել ցանցի կայք
Ստեղծեք ձեր մարքեթինգային կայքը որպես կայք ձեր սեփական ցանցի ներսում։ Սա թույլ է տալիս կառավարել այն նույն dashboard-ից և ցուցադրում է ձեր սեփական հարթակի հնարավորությունները։
:::

## Սեփական դոմեն հաճախորդների համար {#custom-domain-for-customers}

Այն plan-երի հաճախորդների համար, որոնք ներառում են սեփական դոմեններ, գործընթացը հստակ փաստաթղթավորեք՝

1. Հաճախորդը գրանցում կամ տեղափոխում է իր դոմենը ռեգիստրատորի մոտ
2. Հաճախորդը թարմացնում է DNS-ը՝ այն ձեր հարթակին ուղղելու համար (տրամադրեք ճշգրիտ գրառումները)
3. Ultimate Multisite-ը մշակում է դոմենի կապումն ու SSL-ը

Ստեղծեք օգնության հոդված կամ գիտելիքների բազայի գրառում հատուկ այս գործընթացի համար՝ գրված ոչ տեխնիկական ֆիթնես ստուդիաների սեփականատերերի համար։

## FitSite ցանցը մինչ այս պահը {#the-fitsite-network-so-far}

```
FitSite Network
├── WordPress Multisite (subdomain mode)
├── Ultimate Multisite (configured + branded)
├── Platform Domain (fitsite.com + wildcard SSL)
├── Site Templates (Studio Essential, Gym Pro, Fitness Chain)
├── Products (Starter, Growth, Pro + Order Bumps)
├── Checkout Flow (niche-specific, tested)
├── Branding
│   ├── Custom login page
│   ├── Branded admin dashboard
│   ├── Niche-specific system emails
│   ├── Branded invoices
│   └── Marketing site on fitsite.com
└── Ready for onboarding flow (next lesson)
```

## Ինչ կառուցեցինք այս դասում {#what-we-built-this-lesson}

- **Հարթակի դոմեն և DNS**, կարգավորված բրենդավորված փորձառության համար
- **White-labeled admin**՝ FitSite բրենդավորմամբ ամբողջ ընթացքում
- **Հարմարեցված հաղորդակցություններ** -- էլ․ նամակներ, հաշիվ-ապրանքագրեր և ծանուցումներ՝ բոլորը բրենդին համապատասխան
- **Մարքեթինգային կայք**, որը FitSite-ը վաճառում է ֆիթնես ստուդիաների սեփականատերերին
- **Սեփական դոմենի փաստաթղթավորում** այն հաճախորդների համար, որոնք ուզում են իրենց սեփական դոմենը
