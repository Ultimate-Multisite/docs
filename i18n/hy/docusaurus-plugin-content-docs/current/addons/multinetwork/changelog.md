# Multi-Network փոփոխությունների մատյան

### 1.3.0 {#130}
* Նոր․ ցանցային ձևանմուշների նախադիտում — զննեք և նախադիտեք ցանցային ձևանմուշները կենդանի վահանակով՝ նախքան գնումը։
* Նոր․ ցանցային ձևանմուշի ընտրիչ վճարման փուլում — հաճախորդները կարող են ընտրել ցանցային ձևանմուշ գրանցման վճարման հոսքի ընթացքում։
* Նոր․ միջցանցային magic link SSO — հաճախորդները կարող են անխափան մուտք գործել ենթացանցեր magic-link նույնականացման միջոցով։
* Նոր․ ցանցային մեդիայի կլոնավորում — կայքի մեդիան ճիշտ կրկնօրինակվում է ձևանմուշից ցանցեր կլոնավորելիս։
* Ուղղում․ ցանցի կլոնավորումն այժմ աշխատում է առանց multi-tenancy հավելման․ super admin արտոնությունները կրկին տրամադրվում են կլոնավորումից հետո։
* Ուղղում․ ցանցի ստեղծումն այժմ համատեղելի է WooCommerce-ի և բոլոր վճարային դարպասների հետ։
* Ուղղում․ պորտի համարը հիմա ներառված է պահուստային դոմենի ստեղծման մեջ։
* Ուղղում․ ձևանմուշի ընտրիչի CSS-ը տեղափոխվել է հերթագրված ոճաթերթ՝ բոլոր թեմաներում ճիշտ արտապատկերման համար։
* Ուղղում․ պաշտպանություն չսահմանված `network_id` բանալուց կայքի հարցման շրջանակում։
* Ուղղում․ ցանցի թույլտվության ստուգումը նախքան համատեքստի փոխարկումը կանխում է `wpdb` աղյուսակի հատկության սխալները։
