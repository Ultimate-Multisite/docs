# Գործողություն: switch_network {#action-switchnetwork}

## Պարամետրեր {#parameters}

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $new_network_id | `int` | Ցանցի ID-ն, որին կատարվում է անցումը։ |
| $old_network_id | `int` | Նախկին ընթացիկ ցանցի ID-ն։ |
