### Տարբերակ 1.9.0 - Թողարկվել է 2026-01-18-ին {#version-190---released-on-2026-01-18}

* Ուղղվել է․ ակտիվացման ժամանակ առաջացող ճակատագրական սխալները։
* Վերակազմավորվել է․ բովանդակության աղբյուրի դասերը՝ Singleton trait օգտագործելու համար՝ ծնող constructor-ի ճիշտ սկզբնավորմամբ
* Ուղղվել է․ Vue.js ձևանմուշի կոմպիլյացիայի սխալները, որոնք խանգարում էին բովանդակության տեսակի ընտրիչի ցուցադրմանը
* Ուղղվել է․ բացակայող Vue տվյալների հատկությունները (template_id, external_link_url_embedable, external_link_url_checking)
* Ուղղվել է․ TinyMCE խմբագրիչը էջի բեռնման ժամանակ ճիշտ չէր սկզբնավորվում
* Ուղղվել է․ Divi աջակցման ֆայլը Composer-ի միջոցով ինքնաբեռնվող չէր
* Ուղղվել է․ վնասված PHP շարահյուսությունը Divi աջակցման դասում
* Բարելավվել է․ JavaScript ռեսուրսների կազմակերպումը և մինիմիզացիան
* Բարելավվել է․ համատեղելիությունը վերջին WordPress-ի և էջ կառուցողների տարբերակների հետ
