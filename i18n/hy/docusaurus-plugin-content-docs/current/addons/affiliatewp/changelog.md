# AffiliateWP փոփոխությունների մատյան

Տարբերակ՝ 2.0.1 - Թողարկվել է 2025-09-28-ին

* Նախածանցը վերանվանվել է ultimate-multisite-ի; թարմացվել է տեքստային տիրույթը; տարբերակի բարձրացում։



Տարբերակ՝ 2.0.0 - Թողարկվել է 2022-08-05-ին

* Plugin-ը վերակառուցվել է՝ ultimo վճարումների հիման վրա աշխատելու համար

Տարբերակ՝ 2.0.0-beta.2 - Թողարկվել է 2022-01-21-ին

* Ներքին՝ ավելացվել է hooks և filters գեներատոր;
* Ներքին՝ ավելացվել են Multisite Ultimate stubs՝ մշակողի աշխատանքի հարմարության համար;
* Ուղղվել է՝ Fatal error, երբ տվյալ զամբյուղում զեղչի կոդ չկա;
* Ուղղվել է՝ այցելությունները պատշաճ կերպով չէին փոխարկվում;

Տարբերակ 2.0.0-beta.1 - Թողարկվել է 2021-12-29-ին

* 2.0-ի վրա հիմնված տարբերակի սկզբնական թողարկում;
