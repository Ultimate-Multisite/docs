# Mailchimp

Multisite Ultimate: Mailchimp ինտեգրում

Ինտեգրեք ձեր Multisite Ultimate ցանցը ձեր Mailchimp-ի հետ։ Կարող եք որոշակի պլաններ կապել Mailchimp ցանկերին և գրանցման ժամանակ ավտոմատ կերպով բաժանորդներ ավելացնել այդ ցանկերում։

## Տեղադրում

1. Վերբեռնեք 'ultimate-multisite-mailchimp'-ը '/wp-content/plugins/' գրացուցակ
2. Ակտիվացրեք plugin-ը ձեր WordPress Network Admin-ի 'Plugins' ընտրացանկի միջոցով

## Օգտագործման ուղեցույց

Բաժանորդների էլ․ փոստերը ձեր MailChimp ցանկերի հետ համաժամեցնելն իսկապես հեշտ է մեր Ultimate Multisite: MailChimp Integration Add-on-ի միջոցով։ Այս ուղեցույցի նպատակն է օգնել ձեզ ամեն ինչ կարգավորել։

## API բանալի ստանալը

Նախքան հասանելի տարբեր կարգավորումները դիտարկելը, մեզ անհրաժեշտ է MailChimp API բանալի։ Սա թույլ կտա Ultimate Multisite-ին հաղորդակցվել MailChimp API-ի հետ՝ ցանկերը, խմբերը ստանալու և օգտատերերի գրանցման ժամանակ էլ․ փոստի հասցեներ այդ ցանկերում և խմբերում ավելացնելու համար։

Հետևեք այս ուղեցույցին՝ ձեր սեփական MailChimp API բանալին ստանալու համար․ [API բանալիների մասին | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

<!-- Screenshot unavailable: MailChimp API key settings in Ultimate Multisite Add-on Settings -->

## Ինտեգրման ռեժիմ

MailChimp Integration հավելումը առաջարկում է երկու «ինտեգրման ռեժիմ», ինչը նշանակում է, որ կա երկու տարբեր եղանակ, որով կարող եք օգտագործել հավելումը՝ էլ․ փոստի հասցեները համաժամեցնելու համար։

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_Երկու ինտեգրման ռեժիմ՝ ավելի մեծ ճկունության համար։_

### Ռեժիմ 1․ Բազմաթիվ ցանկեր

Լռելյայն ռեժիմն օգտագործում է **Բազմաթիվ ցանկեր**։ Այս ռեժիմը թույլ կտա յուրաքանչյուր պլանի համար ընտրել մի քանի MailChimp ցանկեր։ Սա լավագույն տարբերակն է, եթե ձեր օգտատերերին բաժանում եք տարբեր ցանկերի։

Օրինակ՝ ենթադրենք ձեր MailChimp Account-ում ունեք 3 տարբեր ցանկ՝ **Տեղեկագիր** , **Պլան A** և **Պլան B**։ Ցանկանում եք, որ բոլոր օգտատերերը, անկախ իրենց պլանից, նույնպես ներառվեն Տեղեկագիր ցանկում, որպեսզի բաց չթողնեն ձեր թարմացումները։

Դրան հասնելու համար կարող եք օգտագործել Բազմաթիվ ցանկեր ռեժիմը՝ **Տեղեկագիր** և **Պլան A** ցանկերը Պլան A-ի MailChimp կարգավորումներին ավելացնելու համար, իսկ **Տեղեկագիր** և **Պլան B** ցանկերը՝ Պլան B-ի MailChimp կարգավորումներին։

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_Ընտրեք լռելյայն ցանկերը նոր Account-ների և չեղարկված Account-ների համար_

### Ռեժիմ 2․ Մեկ ցանկ + բազմաթիվ խմբեր

MailChimp-ը թույլ է տալիս ձեր ցանկը բաժանել օգտատերերի ավելի փոքր խմբերի, որոնք կոչվում են **Խմբեր**։ Սա ձեզ ավելի մեծ վերահսկողություն է տալիս, քանի որ կարող եք արշավ ուղարկել ձեր բաժանորդների որոշակի ենթաբազմությանը՝ առանց MailChimp-ում նույն արշավի բազմաթիվ պատճեններ ստեղծելու անհրաժեշտության (ինչը հետագայում դժվարացնում է այդ արշավի ցուցանիշների վերլուծությունը)։

Օրինակ՝ կարող եք ունենալ մեկ ցանկ՝ **Իմ ցանցի բաժանորդները,** և դրա ներսում ունենալ խմբեր **Պլան A**-ի և **Պլան B**-ի համար։

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_Ընտրեք լռելյայն ցանկերն ու խմբերը, եթե օգտագործում եք «մեկ ցանկ + բազմաթիվ խմբեր» ինտեգրման ռեժիմը_

## Լռելյայն ցանկեր և խմբեր

Այն բանից հետո, երբ ընտրեք, թե որ ինտեգրման ռեժիմն է ավելի համապատասխան ձեր կոնկրետ օգտագործման դեպքին, ժամանակն է ընտրել լռելյայն ցանկերը (և խմբերը՝ կախված ռեժիմից)։ Լռելյայն ցանկը կօգտագործվի, երբ տվյալ պլանի համար հատուկ ցանկ ընտրված չէ։

Կարող եք նաև ընտրել լռելյայն ցանկը չեղարկված Account-ների համար։ Ամեն անգամ, երբ օգտատերը դադարեցնում է իր բաժանորդագրությունը և հեռացնում է իր Account-ը ձեր ցանցից, նրա էլ․ փոստի հասցեն կտեղափոխվի չեղարկված Account-ների համար սահմանված ցանկ։

## Կրկնակի բաժանորդագրության միացում

Նույն կարգավորումների էջում կկարողանաք միացնել **Կրկնակի բաժանորդագրությունը**։ Այս տարբերակը օգտատիրոջը կավելացնի և՛ ընտրված լռելյայն ցանկում, և՛ այդ օգտատիրոջ պլանի համար ընտրված ցանկում։ Եթե այս տարբերակը միացված չէ, օգտատերը կավելացվի միայն օգտատիրոջ պլանի համար ընտրված ցանկում։ Եթե տվյալ կոնկրետ պլանի համար ոչինչ սահմանված չէ, ապա օգտատերը կավելացվի ընտրված լռելյայն ցանկում։

_Կարող եք այնպես անել, որ ձեր օգտատերերը գրանցման ժամանակ բաժանորդագրվեն և՛ լռելյայն ցանկին, և՛ պլանի ցանկին_

## Յուրաքանչյուր պլանի համար ցանկերի և խմբերի ընտրություն

Գնացեք ձեր **Պլաններ** և ընտրեք այն պլանը, որը ցանկանում եք խմբագրել։ Նկատի ունեցեք, որ **Ընդլայնված ընտրանքներ** meta-box-ում հասանելի կլինի MailChimp կոչվող նոր ներդիր։

Այդ ներդիրում կկարողանաք ընտրել մի քանի ցանկ կամ մեկ ցանկ և մի քանի խումբ (կախված ինտեգրման ռեժիմից) տվյալ պլանի համար։

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_Ընտրեք այն ցանկերն ու խմբերը, որոնք ցանկանում եք օգտագործել ձեր յուրաքանչյուր պլանի համար_

Երբ նոր օգտատերը գրանցվի այդ պլանի համար, նրա էլ․ փոստի հասցեն կավելացվի ընտրված ցանկերում և խմբերում։
