# Mailster փոփոխությունների մատյան

### 1.0.0
* Սկզբնական թողարկում
* Բաժանորդների ավտոմատ համաժամացում գրանցման կամ վճարման ժամանակ
* Ապրանքին հատուկ ցուցակների նշանակումներ
* Checkbox-ի վրա հիմնված opt-in՝ GDPR համապատասխանության համար
* Double opt-in աջակցություն
* Դաշտերի համապարփակ քարտեզագրում
* Մանրամասն գրանցամատյան
