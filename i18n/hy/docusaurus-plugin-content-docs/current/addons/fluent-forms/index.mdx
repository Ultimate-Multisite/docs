# Fluent Forms {#fluent-forms}

Այս հավելումը հնարավորություն է տալիս Multisite Ultimate-ում սահմանափակումներ ավելացնել Fluent Forms-ի համար՝ ապրանքներում, անդամակցություններում կամ կայքերում։
Fluent Forms-ը օգտագործում է հատուկ աղյուսակներ՝ հատուկ գրառման տեսակների փոխարեն, ուստի սովորական սահմանափակման տարբերակները չեն աշխատում Fluent Forms-ի հետ։

## Հնարավորություններ {#features}

- Ավելացրեք սահմանափակումներ Fluent Forms-ի կամ Conversational Fluent Forms-ի համար
- Հաճախորդները ստանում են բարեկամական հաղորդագրություն, որը հուշում է նրանց բարելավել իրենց պլանը, երբ հասնում են սահմանաչափին
