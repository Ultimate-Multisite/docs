# Ultimate AI Connector-ի փոփոխությունների մատյան Compatible Endpoints-ի համար {#ultimate-ai-connector-for-compatible-endpoints-changelog}

## Տարբերակ 2.0.0 — Թողարկվել է 2026-04-24-ին {#version-200--released-on-2026-04-24}

* Նոր․ բազմամատակարարի աջակցություն — կարգավորեք մի քանի AI endpoint և ուղղորդեք հարցումները մատակարարների միջև ավտոմատ պահուստային անցումով։
* Ուղղում․ բազմամատակարարի SDK ինտեգրում՝ ճիշտ մատակարարի ID-ներով, գրանցման URL-ներով և HTTP ֆիլտրի սահմանափակմամբ ըստ մատակարարի։
* Ուղղում․ նոր մատակարարի քարտն այժմ ավելացնելուց հետո ինքնաբերաբար բացվում է. plugin-ի թարմացման ժամանակ script-ի քեշի շրջանցում։
* Ուղղում․ դինամիկ մատակարարի դասի namespace՝ eval()-ի համար, և ինքնաբացվող վարքագիծ նոր ավելացված մատակարարների համար։
* Ուղղում․ օգտագործում է կայուն Card/CardBody/CardHeader/CardDivider բաղադրիչներ (այլևս փորձարարական չեն)՝ WordPress 6.9+-ի համատեղելիության համար։
* Ուղղում․ չսահմանված DragHandle-ը փոխարինվել է unicode բռնման պատկերակով՝ մատակարարների քաշելով վերադասավորման համար։
* Ուղղում․ վերացվել է արգելափակող HTTP հարցումը, որը գործարկվում էր յուրաքանչյուր էջի բեռնման ժամանակ։
* Բարելավված․ GitHub Actions աշխատանքային հոսքերը թարմացվել են մինչև Node.js 24։
