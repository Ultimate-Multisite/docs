# Content Sync փոփոխությունների մատյան

### Տարբերակ 1.1.0 - Թողարկվել է 2025-12-31-ին
* Նոր․ բոլոր գրառման տեսակների համաժամացման աջակցություն (գրառումներ, էջեր, անհատական գրառման տեսակներ)
* Նոր․ գրառման տեսակի զտում բովանդակության ընտրության քայլում
* Նոր․ աղբյուր կայքի ընտրության քայլ՝ համաժամացրեք ցանկացած կայքից, ոչ միայն հիմնական կայքից
* Նոր․ ենթակայքի նախընտրություն ձևանմուշային կայքերի համար
* Նոր․ մեդիայի դուրսբերում post_content-ից բոլոր գրառման տեսակների համար
* Նոր․ դասակարգման համաժամացում (կատեգորիաներ, պիտակներ, անհատական դասակարգումներ)
* Փոխված․ «Elementor Template Sync»-ից վերանվանվել է «Content Sync»
* Փոխված․ «Select Templates» քայլը վերանվանվել է «Select Data»
* Բարելավված․ Elementor-ին հատուկ կոդը այժմ աշխատում է միայն elementor_library գրառման տեսակի համար

### Տարբերակ 1.0.0 - Թողարկվել է 2025-12-19-ին
* Սկզբնական թողարկում
* Բազմաքայլ վարպետ՝ ձևանմուշի և կայքի ընտրության համար
* Փաթեթային մշակում՝ իրական ժամանակում առաջընթացի արձագանքով
* Լիարժեք մեդիայի համաժամացում՝ ID-ների ավտոմատ վերահամապատասխանեցմամբ
* Կարգավորումների ինտեգրում Ultimate Multisite-ի հետ
* Կայքի բացառման աջակցություն ցանցային կարգավորումների միջոցով
* Elementor քեշի մաքրում համաժամացումից հետո
* Ձևանմուշի դասակարգման պահպանում
