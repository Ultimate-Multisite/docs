# Բովանդակության համաժամեցում

Multisite Ultimate՝ Բովանդակության համաժամեցում

Multisite Ultimate-ի այս հավելումը թույլ է տալիս ցանցի ադմինիստրատորներին համաժամեցնել բովանդակությունը ցանկացած կայքից դեպի ցանցի այլ կայքեր։ Այն աջակցում է բոլոր գրառման տեսակները, ներառյալ գրառումները, էջերը, հարմարեցված գրառման տեսակները և Elementor ձևանմուշները։ Հավելումը ներառում է բազմաքայլ վիզարդի միջերես՝ բովանդակություն և նպատակային կայքեր ընտրելու համար՝ խմբաքանակային մշակմամբ և իրական ժամանակում առաջընթացի հետադարձ կապով։

## Հնարավորություններ {#features}

* Բովանդակության ընտրություն․ տեսողական քարտերի ցանց՝ մանրապատկերներով և գրառման տեսակի զտմամբ
* Բոլոր գրառման տեսակները․ համաժամեցնում է գրառումներ, էջեր, ապրանքներ, Elementor ձևանմուշներ և ցանկացած հարմարեցված գրառման տեսակ
* Կայքերի ընտրություն․ ընտրեք, թե որ կայքերի հետ համաժամեցնել բովանդակությունը՝ սկզբնաղբյուր կայքի բացառմամբ և ենթակայքերի նախնական ընտրությամբ
* Խմբաքանակային մշակում․ միաժամանակ համաժամեցնում է մինչև 10 կայքի հետ՝ առաջընթացի թարմացումներով
* Մեդիայի համաժամեցում․ ավտոմատ պատճենում և վերաքարտեզագրում է մեդիա ֆայլերը (պատկերներ, տեսանյութեր) գրառման բովանդակությունից դեպի նպատակային կայքեր
* Տաքսոնոմիաների համաժամեցում․ պահպանում է կատեգորիաները, թեգերը և հարմարեցված տաքսոնոմիաները
* Elementor աջակցություն․ լիարժեք աջակցություն Elementor ձևանմուշների համար՝ CSS-ի վերագեներացմամբ և ձևանմուշների ID-ների վերաքարտեզագրմամբ
* Հակասությունների լուծում․ առկա բովանդակությունը փոխարինելու կամ այն բաց թողնելու տարբերակ
* Քեշի կառավարում․ համաժամեցումից հետո ավտոմատ մաքրում է Elementor քեշը նպատակային կայքերում

## Պահանջներ {#requirements}

* WordPress 5.3 կամ ավելի բարձր
* PHP 7.4 կամ ավելի բարձր
* Ultimate Multisite plugin (ակտիվ)
* Elementor plugin (ըստ ցանկության, պահանջվում է Elementor ձևանմուշների համաժամեցման համար)

## Տեղադրում {#installation}

1. Վերբեռնեք 'ultimate-multisite-content-sync'-ը '/wp-content/plugins/' պանակ
2. Ակտիվացրեք plugin-ը ձեր WordPress Network Admin-ի 'Plugins' ընտրացանկի միջոցով
3. Կարգավորեք կարգավորումները Ultimate Multisite → Settings → Content Sync բաժնում
4. Անցեք Ultimate Multisite → Content Sync՝ համաժամեցումը սկսելու համար
