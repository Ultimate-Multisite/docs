# Փլագինների և թեմաների կառավարչի փոփոխությունների մատյան

Տարբերակ՝ 2.0.8 - Թողարկվել է 2026-05-05-ին
* Բարելավված՝ փլագինների ցանկն այժմ օգտագործում է կայքի քեշը՝ տվյալների բազայի հարցումները նվազեցնելու և արտադրողականությունը բարելավելու համար
* Բարելավված՝ փլագինի autoloader-ը բաց է թողնվում, երբ Bedrock-ի արմատային autoloader-ն արդեն բեռնել է կախվածությունները

Տարբերակ՝ 2.0.7 - Թողարկվել է 2026-01-01-ին
