# Tutor Multisite Compatibility

WordPress plugin, որը ավելացնում է համատեղելիության շտկումներ Tutor LMS-ի համար multisite միջավայրերում։

## Նկարագրություն

Այս plugin-ը տրամադրում է անհրաժեշտ համատեղելիության շտկումներ՝ Tutor LMS-ը WordPress multisite ցանցերում գործարկելու համար։ Այն մշակում է օգտատերերի գրանցումը դասընթացներում, հրահանգիչների կառավարումը, դասընթացների սահմանափակումները և multisite-ին հատուկ այլ գործառույթներ։

## Հնարավորություններ

- **Տվյալների բազայի աղյուսակների կառավարում**: Ավտոմատ հայտնաբերում և ստեղծում է բացակայող Tutor LMS աղյուսակները, որոնք չեն ստեղծվել multisite-ի արտաքին բանալիների սահմանափակումների պատճառով
- **Multisite օգտատերերի գրանցում**: Հարմարեցված AJAX մշակիչներ՝ ցանցի կայքերում օգտատերերի գրանցումը կառավարելու համար
- **Հրահանգչի դերի կառավարում**: Հրահանգչի դերի ավտոմատ նշանակումն ու կառավարումը multisite միջավայրերում
- **Դասընթացների սահմանափակումներ**: Ինտեգրում Ultimate Multisite-ի հետ՝ կայքի պլանների հիման վրա դասընթացների և funnel-ների ստեղծման սահմանաչափերը կառավարելու համար
- **Հարմարեցված ադմինիստրատորի դիտումներ**: Multisite-ին հարմարեցված հրահանգիչների ցանկ և կառավարման միջերես
- **Ադմինիստրատորի գործիքներ**: Տվյալների բազայի կառավարման միջերես ցանցի և կայքի ադմինիստրատորների համար

## Պահանջներ

- WordPress 5.0+
- PHP 7.4+
- Tutor LMS plugin (ավտոմատ պարտադրվում է `Requires Plugins` header-ի միջոցով)
- WordPress Multisite տեղադրում
- Ultimate Multisite plugin (դասընթացների սահմանափակումների գործառույթի համար)

## Տեղադրում

1. Վերբեռնեք plugin-ի ֆայլերը `/wp-content/plugins/tutor-multisite-compatibility/`
2. Համոզվեք, որ Tutor LMS-ը տեղադրված և ակտիվացված է
3. Ակտիվացրեք plugin-ը WordPress-ի «Plugins» մենյուի միջոցով
4. Plugin-ը ավտոմատ կինտեգրվի Tutor LMS-ի հետ

## MU plugin-ից տեղափոխված գործառույթներ

Այս plugin-ը պարունակում է գործառույթներ, որոնք նախկինում գտնվում էին `wp-content/mu-plugins/daves.php`-ում՝

- Դասընթացի կարգավիճակի փոփոխման սահմանափակումներ
- Դասընթացի ստեղծման սահմանափակումներ
- CartFlows ինտեգրում և սահմանաչափեր
- Tutor հրահանգչի դերի կառավարում
- Multisite օգտատերերի գրանցման մշակում
- Դասընթացի product ID-ի ցուցադրում

## Տվյալների բազայի կառավարում

### Խնդիրը

WordPress multisite միջավայրերում Tutor LMS աղյուսակների ստեղծումը ձախողվում է այն աղյուսակների համար, որոնք ունեն արտաքին բանալիների սահմանափակումներ դեպի օգտատերերի աղյուսակ։ Սա տեղի է ունենում, քանի որ՝

- Կայքին հատուկ աղյուսակներն օգտագործում են այնպիսի նախածանցներ, ինչպիսիք են `wp_2_tutor_carts`
- Արտաքին բանալիները հղվում են `wp_2_users`-ին (որը գոյություն չունի)
- Իրական օգտատերերի աղյուսակը `wp_users` է (ընդհանուր ամբողջ ցանցի համար)

### Լուծումը

Այս plugin-ը ավտոմատ՝

1. **Հայտնաբերում է բացակայող աղյուսակները**: Ստուգում է աղյուսակները, որոնք չեն ստեղծվել արտաքին բանալիների սահմանափակումների պատճառով
2. **Ստեղծում է աղյուսակներ ճիշտ հղումներով**: Օգտագործում է օգտատերերի աղյուսակի ճիշտ անունը (`wp_users` multisite-ում)
3. **Մշակում է բազմաթիվ սցենարներ**:
   - Plugin-ի ակտիվացում/վերաակտիվացում
   - Ցանցում նոր կայքի ստեղծում
   - Աղյուսակների ձեռքով ստուգում ադմինիստրատորի միջերեսի միջոցով

### Ազդեցության ենթակա աղյուսակներ

- `tutor_coupon_usages` - Պարունակում է արտաքին բանալի դեպի օգտատերերի աղյուսակ
- `tutor_carts` - Պարունակում է արտաքին բանալի դեպի օգտատերերի աղյուսակ
- `tutor_notification_preferences` - TutorPro աղյուսակ՝ արտաքին բանալիով դեպի օգտատերերի աղյուսակ

### Ադմինիստրատորի միջերես

Մուտք գործեք տվյալների բազայի կառավարման գործիքներին այստեղ՝

- **Network Admin**: Settings → Tutor Multisite DB
- **Site Admin**: Tools → Tutor Multisite DB

Միջերեսը թույլ է տալիս՝
- Դիտել բացակայող աղյուսակները բոլոր կայքերում
- Ձեռքով գործարկել աղյուսակների ստեղծումը
- Ստուգել տվյալների բազայի կազմաձևումը
- Դիտել մանրամասն կարգավիճակի տեղեկատվություն

## Աջակցություն

Այս plugin-ը հատուկ մշակված է KursoPro հարթակի համար։ Տեխնիկական աջակցության համար կապվեք մշակման թիմի հետ։
