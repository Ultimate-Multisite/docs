# Ստեղծել լոգոյի SVG {#generate-logo-svg}

**Ստեղծել լոգոյի SVG** կարողությունը թույլ է տալիս Theme Builder-ին ստեղծել և ներդնել անհատական լոգոյի SVG-ներ անմիջապես ձեր WordPress կայքում՝ ավտոմատ namespace-safe մաքրմամբ։

## Ընդհանուր ակնարկ {#overview}

Այս կարողությունը ստեղծում է մասշտաբավորվող վեկտորային գրաֆիկայի (SVG) լոգոներ՝ հիմնվելով ձեր կայքի բրենդավորման ուղղության և դիզայնի նախընտրությունների վրա։ Ստեղծված SVG-ները ավտոմատ մաքրվում են՝ ապահովելու համար, որ դրանք անվտանգ են WordPress-ում օգտագործելու համար՝ պահպանելով տեսողական ամբողջականությունը։

## Պարամետրեր {#parameters}

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `site_name` | string | Yes | Կայքի անունը, որի համար պետք է ստեղծել լոգոն |
| `style` | string | Yes | Դիզայնի ոճը (օր․՝ "modern", "classic", "minimalist", "playful") |
| `colors` | array | No | Լոգոյում օգտագործվող hex գունային կոդերի զանգված (օր․՝ `["#678233", "#ffffff"]`) |
| `width` | number | No | SVG-ի լայնությունը պիքսելներով (լռելյայն՝ 200) |
| `height` | number | No | SVG-ի բարձրությունը պիքսելներով (լռելյայն՝ 200) |
| `include_text` | boolean | No | Արդյոք լոգոյում ներառել կայքի անունը որպես տեքստ (լռելյայն՝ true) |

## Ելքի ձևաչափ {#output-format}

Կարողությունը վերադարձնում է SVG տող՝ հետևյալ կառուցվածքով․

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## SVG-ի մաքրման վարքագիծ {#svg-sanitisation-behaviour}

Ստեղծված SVG-ները ենթարկվում են ավտոմատ namespace-safe մաքրման՝ հետևյալի համար․

- **Հեռացնել ոչ անվտանգ հատկանիշները** — հեռացնում է իրադարձությունների մշակիչները, սքրիպտները և հնարավոր վտանգավոր հատկանիշները
- **Պահպանել namespace-ները** — պահպանում է SVG namespace-ները (xmlns, xlink)՝ ճիշտ ցուցադրման համար
- **Ստուգել կառուցվածքը** — ապահովում է, որ SVG-ն համապատասխանի W3C ստանդարտներին
- **Կոդավորել entity-ները** — ճիշտ կերպով escape է անում տեքստային բովանդակության հատուկ նիշերը
- **Հեռացնել արտաքին հղումները** — հեռացնում է արտաքին ոճաթերթերը և պատկերի հղումները

Սա ապահովում է, որ SVG-ն անվտանգ լինի անմիջապես WordPress-ում ներդնելու համար՝ առանց լրացուցիչ մաքրում պահանջելու։

## Օգտագործման օրինակ {#usage-example}

**Հուշում․**
```
Ստեղծիր ժամանակակից լոգո իմ տեխնոլոգիական ստարտափի համար, որը կոչվում է "CloudSync"՝ օգտագործելով կապույտ և սպիտակ գույներ։
```

**Արդյունք․**
Կարողությունը ստեղծում է SVG լոգո, որը՝
- Ներառում է կայքի անունը՝ "CloudSync"
- Օգտագործում է նշված կապույտ և սպիտակ գունային սխեման
- Հետևում է ժամանակակից դիզայնի սկզբունքներին
- Ավտոմատ մաքրված է և պատրաստ է օգտագործման

## Ինտեգրում Theme Builder-ի հետ {#integration-with-theme-builder}

Theme Builder-ի դիզայնի ուղղության ընտրությունն օգտագործելիս Ստեղծել լոգոյի SVG կարողությունը՝

1. Վերլուծում է ձեր դիզայնի ուղղությունը և գունային պալիտրան
2. Ստեղծում է ձեր նախընտրություններին համապատասխան անհատական SVG լոգո
3. Ավտոմատ ներդնում է լոգոն ձեր կայքի վերնագրի/բրենդավորման հատվածում
4. Պահպանում է SVG-ն որպես անհատական լոգո WordPress մեդիայում

## Լավագույն գործելակերպեր {#best-practices}

- **Տրամադրեք հստակ ոճային նախընտրություններ** — նկարագրեք ձեր ուզած դիզայնի ոճը (ժամանակակից, դասական, խաղային և այլն)
- **Նշեք գույները** — ներառեք ձեր բրենդի գույները՝ հետևողականության համար
- **Փորձարկեք ցուցադրումը** — ստուգեք, որ լոգոն ճիշտ է ցուցադրվում տարբեր էկրանի չափերի վրա
- **Անհատականացրեք ավելին** — օգտագործեք WordPress-ի լոգոյի անհատականացման գործիքները՝ չափը և տեղադրությունը կարգավորելու համար

## Սահմանափակումներ {#limitations}

- SVG լոգոները ստեղծվում են որպես ստատիկ գրաֆիկա (ոչ անիմացված)
- Բազմաթիվ տարրերով բարդ լոգոները կարող են պահանջել ձեռքով կատարելագործում
- Անհատական տառատեսակները չեն աջակցվում․ տեքստն օգտագործում է համակարգային տառատեսակներ
- Շատ մեծ կամ շատ փոքր չափերը կարող են ազդել որակի վրա
