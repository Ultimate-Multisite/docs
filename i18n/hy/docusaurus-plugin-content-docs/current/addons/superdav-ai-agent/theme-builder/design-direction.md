# Դիզայնի ուղղություն

**Դիզայնի ուղղություն** քայլը թույլ է տալիս վերանայել, ճշգրտել և վերջնականացնել ձեր theme-ի տեսողական դիզայնը՝ նախքան Theme Builder-ը կստեղծի ամբողջական theme-ը։

## Ընդհանուր նկարագրություն {#overview}

Discovery Interview-ն ավարտելուց հետո Theme Builder-ը ներկայացնում է ձեր դիզայնի ուղղությունը՝ **desktop և mobile preview rendering**-ով։ Սա թույլ է տալիս տեսնել, թե ձեր դիզայնը տարբեր սարքերում ինչպես կերևա՝ նախքան theme-ին վերջնական համաձայնություն տալը։

## Դիզայնի ուղղության բաղադրիչներ {#design-direction-components}

### 1. Գունապնակ {#1-color-palette}

Ձեր theme-ի գունային սխեման, ներառյալ՝

- **Հիմնական գույն** — հիմնական բրենդային գույնը, որն օգտագործվում է կոճակների, հղումների և շեշտադրումների համար
- **Երկրորդական գույն** — լրացնող գույն՝ բազմազանության համար
- **Շեշտադրման գույն** — կարևոր տարրերի ընդգծման գույն
- **Չեզոք գույներ** — մոխրագույններ և սպիտակներ՝ ֆոնների և տեքստի համար
- **Տեքստի գույն** — հիմնական տեքստի գույն՝ ընթեռնելիության համար

### 2. Տառատեսակաբանություն {#2-typography}

Տառատեսակների ընտրություն՝

- **Վերնագրերի տառատեսակ** — օգտագործվում է էջերի վերնագրերի և բաժինների վերնագրերի համար
- **Հիմնական տեքստի տառատեսակ** — օգտագործվում է պարբերությունների տեքստի և բովանդակության համար
- **Տառաչափեր** — արձագանքող չափավորում տարբեր էկրանի չափերի համար
- **Տողերի բարձրություն** — տողերի միջև հեռավորություն՝ ընթեռնելիության համար
- **Տառատեսակի հաստություններ** — թավ, սովորական և բարակ տարբերակներ

### 3. Դասավորություն և հեռավորություններ {#3-layout-and-spacing}

- **Container-ի լայնություն** — բովանդակության առավելագույն լայնություն
- **Ներքին և արտաքին լուսանցքներ** — տարրերի շուրջ հեռավորություններ
- **Ցանցային համակարգ** — սյունակային դասավորություն արձագանքող դիզայնի համար
- **Բաղադրիչների միջև հեռավորություն** — միջերեսի տարրերի միջև հեռավորություն

### 4. Տեսողական տարրեր {#4-visual-elements}

- **Կոճակների ոճեր** — հիմնական, երկրորդական և երրորդական կոճակների դիզայններ
- **Քարտերի դիզայններ** — բովանդակության բլոկների դասավորություններ
- **Պատկերակներ** — պատկերակների ոճ և չափավորում
- **Եզրագծեր և ստվերներ** — նուրբ տեսողական խորություն
- **Hover էֆեկտներ** — փոխազդեցության վիճակի ոճավորում

## Preview Rendering {#preview-rendering}

### Desktop Preview {#desktop-preview}

Desktop preview-ը ցույց է տալիս ձեր դիզայնը ամբողջ լայնությամբ (սովորաբար 1200px կամ ավելի լայն)՝

- Ամբողջական նավիգացիոն մենյու
- Ամբողջական դասավորություն՝ բոլոր սյունակներով
- Մեծ տառատեսակաբանություն և հեռավորություններ
- Բոլոր տեսողական տարրերը ամբողջական չափով

### Mobile Preview {#mobile-preview}

Mobile preview-ը ցույց է տալիս ձեր դիզայնը mobile լայնությամբ (սովորաբար 375px)՝

- Արձագանքող նավիգացիա (hamburger մենյու)
- Մեկ սյունակով դասավորություն
- Կարգավորված տառատեսակաբանություն և հեռավորություններ
- Հպման համար հարմար կոճակների չափեր

### Արձագանքող բեկման կետեր {#responsive-breakpoints}

Theme Builder-ը preview-ները ցուցադրում է այս բեկման կետերում՝

| Սարք | Լայնություն | Preview |
|--------|-------|---------|
| **Mobile** | 375px | Ուղղահայաց հեռախոս |
| **Tablet** | 768px | Հորիզոնական tablet |
| **Desktop** | 1200px | Ամբողջ լայնությամբ desktop |
| **Large Desktop** | 1920px | Գերլայն էկրան |

## Ձեր դիզայնի ճշգրտում {#refining-your-design}

### Կարգավորումներ կատարելը {#making-adjustments}

Դուք կարող եք ճշգրտել ձեր դիզայնի ուղղությունը՝

1. **Գույների կարգավորում** — փոխեք գունապնակի ցանկացած գույն
2. **Տառատեսակների փոփոխություն** — ընտրեք տարբեր տառատեսակներ
3. **Հեռավորությունների փոփոխում** — կարգավորեք ներքին և արտաքին լուսանցքները
4. **Դասավորության թարմացում** — փոխեք container-ների լայնությունները և ցանցի սյունակները
5. **Տարրերի անհատականացում** — փոփոխեք կոճակների ոճերը, քարտերի դիզայնները և այլն

### Preview-ի թարմացումներ {#preview-updates}

Փոփոխությունները արտացոլվում են իրական ժամանակում՝

- Desktop preview-ն անմիջապես թարմացվում է
- Mobile preview-ն անմիջապես թարմացվում է
- Բոլոր արձագանքող բեկման կետերը թարմացվում են
- Փոփոխությունները ստուգելու համար կարող եք անցնել preview-ների միջև

### Համեմատության տեսք {#comparison-view}

Համեմատեք ձեր դիզայնի ուղղությունը՝

- **Սկզբնական դիզայն** — տեսեք, թե ինչ է փոխվել
- **Մրցակիցների դիզայններ** — համեմատեք ոգեշնչման կայքերի հետ
- **Նախորդ տարբերակներ** — վերադարձեք ավելի վաղ դիզայնի ուղղություններին

## Դիզայնի ուղղության հաստատում {#design-direction-approval}

Երբ գոհ եք ձեր դիզայնից՝

1. **Վերանայեք երկու preview-ներն էլ** — համոզվեք, որ desktop և mobile տեսքերը լավ են
2. **Ստուգեք բոլոր գույները** — ապահովեք հակադրություն և մատչելիություն
3. **Փորձարկեք տառատեսակաբանությունը** — ստուգեք ընթեռնելիությունը բոլոր չափերում
4. **Հաստատեք դասավորությունը** — ստուգեք հեռավորություններն ու հավասարեցումը
5. **Հաստատեք դիզայնը** — անցեք theme-ի ստեղծմանը

## Հաջորդ քայլեր {#next-steps}

Ձեր դիզայնի ուղղությունը հաստատելուց հետո՝

1. Theme Builder-ը ստեղծում է ձեր ամբողջական theme-ը
2. Theme-ը տեղադրվում է ձեր WordPress կայքում
3. Դուք կարող եք հետագայում անհատականացնել՝ օգտագործելով WordPress customizer-ը
4. Անցեք [Հյուրընկալության մենյուներ](./hospitality-menus.md)-ին կամ այլ հնարավորությունների

## Լավագույն փորձառություններ {#best-practices}

- **Փորձարկեք իրական սարքերում** — հնարավորության դեպքում օգտագործեք իրական հեռախոսներ և tablet-ներ
- **Ստուգեք ընթեռնելիությունը** — համոզվեք, որ տեքստը ընթեռնելի է բոլոր չափերում
- **Ստուգեք հակադրությունը** — մատչելիության համար օգտագործեք [Ստուգել գունապնակի հակադրությունը](../abilities/validate-palette-contrast.md)
- **Հաշվի առեք արտադրողականությունը** — օպտիմալացրեք պատկերներն ու տառատեսակները արագության համար
- **Պլանավորեք բովանդակության համար** — համոզվեք, որ դասավորությունը աշխատում է ձեր իրական բովանդակության հետ

## Խնդիրների լուծում {#troubleshooting}

### Preview-ն չի թարմացվում {#preview-not-updating}

- Թարմացրեք էջը
- Մաքրեք դիտարկիչի cache-ը
- Փորձեք այլ դիտարկիչ
- Ստուգեք ինտերնետ կապը

### Գույները տարբեր են երևում {#colors-look-different}

- Ստուգեք մոնիտորի գունային կարգավորումները
- Փորձեք տարբեր սարքերում
- Ստուգեք գունային հակադրությունը մատչելիության գործիքներով
- Հաշվի առեք գունային կուրության սիմուլյատորները

### Տառատեսակաբանության խնդիրներ {#typography-issues}

- Ստուգեք, որ տառատեսակների ֆայլերը բեռնվում են
- Ստուգեք տառաչափը տարբեր բեկման կետերում
- Փորձարկեք իրական բովանդակությամբ
- Ընթեռնելիության համար հաշվի առեք տողի երկարությունը
