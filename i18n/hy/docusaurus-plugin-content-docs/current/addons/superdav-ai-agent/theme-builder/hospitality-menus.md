# Հյուրընկալության մենյուներ

**Հյուրընկալության մենյուներ** հնարավորությունը թույլ է տալիս Theme Builder-ին ստեղծել և ներդնել կառուցվածքային սննդի և խմիչքների մենյուի էջեր անմիջապես ձեր WordPress կայքում։

## Ընդհանուր ակնարկ {#overview}

Theme Builder-ը այժմ կարող է ստեղծել պրոֆեսիոնալ, կառուցվածքային մենյուի էջեր հյուրընկալության բիզնեսների համար՝ ներառյալ ռեստորաններ, սրճարաններ, բարեր և քեյթրինգ ծառայություններ։ Այս մենյուները լիովին ինտեգրված են ձեր կայքի դիզայնին և կարող են հեշտությամբ թարմացվել ու կառավարվել։

## Աջակցվող հյուրընկալության տեսակներ {#supported-hospitality-types}

- **Ռեստորաններ** — լիարժեք սպասարկմամբ ճաշացանկեր
- **Սրճարաններ** — սուրճի և թեթև ուտեստների մենյուներ
- **Բարեր և լաունջներ** — խմիչքների և նախուտեստների մենյուներ
- **Հացաբուլկեղենի խանութներ** — խմորեղենի և հացի մենյուներ
- **Քեյթրինգ ծառայություններ** — միջոցառումների մենյուի տարբերակներ
- **Սննդի բեռնատարներ** — շարժական սննդի սպասարկման մենյուներ
- **Գարեջրատներ և գինեգործարաններ** — նկարագրություններով խմիչքների մենյուներ

## Մենյուի կառուցվածք {#menu-structure}

### Մենյուի կատեգորիաներ {#menu-categories}

Մենյուները կազմակերպված են կատեգորիաներով՝

- **Նախուտեստներ** — սկզբնական ուտեստներ և փոքր չափաբաժիններ
- **Հիմնական ուտեստներ** — գլխավոր ճաշատեսակներ
- **Կողմնակի ուտեստներ** — հավելումներ և բանջարեղեն
- **Աղանդերներ** — քաղցր ուտեստներ
- **Խմիչքներ** — ըմպելիքներ (ալկոհոլային և ոչ ալկոհոլային)
- **Հատուկ առաջարկներ** — օրվա կամ սեզոնային հատուկ առաջարկներ

### Մենյուի տարրի ձևաչափ {#menu-item-format}

Յուրաքանչյուր մենյուի տարր ներառում է՝

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### Տարրի դաշտեր {#item-fields}

| Դաշտ | Տեսակ | Նկարագրություն |
|-------|------|-------------|
| `name` | string | Ուտեստի կամ խմիչքի անուն |
| `description` | string | Տարրի մանրամասն նկարագրություն |
| `price` | string | Գին (ձևաչափված արժույթով) |
| `dietary_info` | array | Սննդակարգային հատկանիշներ (Vegan, Gluten-Free և այլն) |
| `allergens` | array | Ընդհանուր ալերգեններ (ընկույզներ, խեցեմորթներ և այլն) |
| `availability` | string | Երբ է հասանելի (ամեն օր, սեզոնային և այլն) |

## Հյուրընկալության մենյուների ստեղծում {#creating-hospitality-menus}

### Քայլ 1․ Տրամադրեք մենյուի տեղեկություններ {#step-1-provide-menu-information}

Պատմեք Theme Builder-ին ձեր մենյուի մասին՝

```
Ստեղծիր ռեստորանի մենյու իմ իտալական ռեստորանի համար՝ նախուտեստներով,
մակարոնեղենի ուտեստներով, հիմնական ուտեստներով և աղանդերներով։ Ներառիր գներ և նկարագրություններ։
```

### Քայլ 2․ Theme Builder-ը ստեղծում է մենյուն {#step-2-theme-builder-generates-menu}

Theme Builder-ը՝

1. Ստեղծում է կառուցվածքային մենյուի էջ
2. Նախագծում է այն՝ ձեր կայքի theme-ին համապատասխան
3. Կազմակերպում է տարրերը կատեգորիաներով
4. Ձևաչափում է գներն ու նկարագրությունները
5. Ավելացնում է սննդակարգային և ալերգենների տեղեկություններ

### Քայլ 3․ Վերանայեք և հարմարեցրեք {#step-3-review-and-customize}

Դուք կարող եք՝

1. Խմբագրել մենյուի տարրերն ու գները
2. Ավելացնել կամ հեռացնել կատեգորիաներ
3. Վերադասավորել տարրերը կատեգորիաների ներսում
4. Թարմացնել նկարագրություններն ու սննդակարգային տեղեկությունները
5. Կարգավորել ոճավորումն ու դասավորությունը

## Մենյուի ցուցադրման տարբերակներ {#menu-display-options}

### Ամբողջական մենյուի էջ {#full-menu-page}

Առանձին էջ, որը ցուցադրում է ձեր ամբողջական մենյուն՝

- Կազմակերպված ըստ կատեգորիայի
- Որոնելի և զտվող
- Տպագրության համար հարմար դասավորություն
- Բջջային սարքերին հարմարվող դիզայն

### Մենյուի վիջեթ {#menu-widget}

Ներդրեք մենյուի բաժինները այլ էջերում՝

- Առաջարկվող տարրեր գլխավոր էջում
- Օրվա հատուկ առաջարկներ կողագոտում
- Խմիչքների մենյու բարի էջում
- Աղանդերի ցուցադրություն ստորին հատվածում

### Մենյուի PDF {#menu-pdf}

Ստեղծեք ներբեռնվող PDF մենյու՝

- Պրոֆեսիոնալ ձևաչափում
- Տպագրության համար պատրաստ որակ
- Ներառում է պատկերներ և նկարագրություններ
- Հեշտ է տարածել և ուղարկել էլ․ փոստով

## Սննդակարգային և ալերգենների տեղեկություններ {#dietary-and-allergen-information}

### Սննդակարգային հատկանիշներ {#dietary-attributes}

Նշեք տարրերը սննդակարգային տեղեկություններով՝

- **Վեգան** — առանց կենդանական ծագման մթերքների
- **Բուսակերական** — առանց մսի
- **Առանց գլյուտենի** — անվտանգ ցելիակիայի դեպքում
- **Առանց կաթնամթերքի** — առանց կաթնամթերքի
- **Առանց ընկույզների** — առանց ծառի ընկույզների կամ գետնանուշի
- **Ցածր ածխաջրերով** — նվազեցված ածխաջրեր
- **Բարձր սպիտակուցային** — սպիտակուցով հարուստ

### Ալերգենների նախազգուշացումներ {#allergen-warnings}

Ներառեք ընդհանուր ալերգենները՝

- **Ընկույզներ** — ծառի ընկույզներ և գետնանուշ
- **Խեցեմորթներ** — խեցգետնակերպեր և փափկամարմիններ
- **Ձուկ** — բոլոր ձկնատեսակները
- **Կաթնամթերք** — կաթ և կաթնամթերք
- **Ձու** — հավի ձու
- **Սոյա** — սոյայի արտադրանք
- **Գլյուտեն** — ցորեն և գլյուտեն պարունակող հացահատիկներ
- **Քունջութ** — քունջութի սերմեր և յուղ

## Մենյուի կառավարում {#menu-management}

### Գների թարմացում {#updating-prices}

Հեշտությամբ թարմացրեք մենյուի գները՝

1. Անցեք մենյուի էջ
2. Սեղմեք «Խմբագրել մենյուն»
3. Թարմացրեք տարրերի գները
4. Պահպանեք փոփոխությունները
5. Փոփոխությունները անմիջապես կհայտնվեն ձեր կայքում

### Սեզոնային տարրերի ավելացում {#adding-seasonal-items}

Ստեղծեք սեզոնային մենյուի տարբերակներ՝

1. Ստեղծեք մենյուի նոր տարբերակ
2. Ավելացրեք սեզոնային տարրեր
3. Նշեք տարրերը որպես «Սեզոնային»
4. Պլանավորեք հասանելիության ամսաթվերը
5. Ավտոմատ ցուցադրեք սեզոնի ընթացքում

### Հատուկ առաջարկների կառավարում {#managing-specials}

Ցուցադրեք օրվա կամ շաբաթական հատուկ առաջարկներ՝

1. Ստեղծեք «Հատուկ առաջարկներ» կատեգորիա
2. Ավելացրեք տարրեր հասանելիության ամսաթվերով
3. Ընդգծեք հատուկ առաջարկները գլխավոր էջում
4. Թարմացրեք ամեն օր կամ ամեն շաբաթ
5. Արխիվացրեք հին հատուկ առաջարկները

## Ինտեգրում Theme Builder-ի հետ {#integration-with-theme-builder}

Theme Builder-ը հյուրընկալության կայքերի համար օգտագործելիս՝

1. **Մենյուի ավտոմատ հայտնաբերում** — որոշում է՝ արդյոք դուք սննդի/խմիչքի բիզնես եք
2. **Մենյուի էջի ստեղծում** — ստեղծում է պրոֆեսիոնալ մենյուի էջեր
3. **Դիզայնի համապատասխանեցում** — մենյուները համապատասխանում են ձեր կայքի theme-ին
4. **Բջջային օպտիմալացում** — մենյուները գեղեցիկ են ցուցադրվում հեռախոսներում
5. **SEO օպտիմալացում** — մենյուները հարմար են որոնողական համակարգերի համար

## Լավագույն գործելակերպեր {#best-practices}

### Մենյուի դիզայն {#menu-design}

- **Հստակ կազմակերպում** — տրամաբանական կատեգորիաների կառուցվածք
- **Ընթեռնելի նկարագրություններ** — ախորժելի և տեղեկատվական
- **Հետևողական գնագոյացում** — հստակ արժույթ և ձևաչափում
- **Պրոֆեսիոնալ լուսանկարներ** — բարձրորակ սննդի պատկերներ
- **Դատարկ տարածություն** — մի գերբեռնեք էջը

### Բովանդակություն {#content}

- **Ճշգրիտ նկարագրություններ** — ճշգրիտ նկարագրեք տարրերը
- **Ընդգծեք հատուկ առաջարկները** — հատուկ առաջարկները դարձրեք նկատելի
- **Ներառեք ալերգենները** — միշտ թվարկեք ալերգենները
- **Պարբերաբար թարմացրեք** — պահեք գներն ու տարրերը արդիական
- **Օգտագործեք ախորժելի լեզու** — այնպես արեք, որ տարրերը համեղ հնչեն

### Մատչելիություն {#accessibility}

- **Ընթեռնելի տառատեսակներ** — օգտագործեք հստակ, ընթեռնելի տառատեսակներ
- **Բավարար հակադրություն** — ապահովեք, որ տեքստը ընթեռնելի լինի
- **Սննդակարգային պիտակներ** — հստակ նշեք սննդակարգային տարբերակները
- **Ալերգենների նախազգուշացումներ** — ակնառու ցուցադրեք ալերգենները
- **Բջջայինին հարմար** — փորձարկեք բոլոր սարքերում
