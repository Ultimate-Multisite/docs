# Superdav AI Agent

Superdav AI Agent-ը առաջադեմ WordPress AI օգնական է, որը ընդլայնում է Gratis AI Agent-ը՝ թեմային հարմարեցված հմտություններով, բարելավված ինտերնետային որոնման հնարավորություններով, կայք կառուցելու գործառույթներով և կամընտիր ավտոմատացման ինտեգրացիաներով։ 1.18.0 տարբերակն ավելացնում է Google Calendar-ի ընթերցման գործիքներ, կոնտակտների և մասնակիցների համապատասխանեցում, մարդկային հաստատման դարպասներ, հիշեցումների կրկնօրինակման կանխում, TextBee SMS ծանուցումներ, կառավարվող Superdav ծառայության տրամադրում և առանձին տարածվող Advanced ուղեկից փաթեթ՝ վստահելի մշակողների աշխատանքային հոսքերի համար։

## Հիմնական հնարավորություններ {#key-features}

- **Թեմային հարմարեցված հմտություններ** — ավտոմատ հարմարեցնում է ուղղորդումն ու հնարավորությունները՝ ըստ ձեր ակտիվ WordPress թեմայի և տեղադրված plugin-ների
- **Ինտերնետային որոնման մի քանի մատակարարներ** — ընտրեք Brave Search-ի և Tavily-ի միջև՝ ավելի հարուստ որոնման արդյունքների և առաջադեմ հետազոտական հնարավորությունների համար
- **Կայք կառուցողի կոնտակտային ձևեր** — ստեղծեք լիարժեք գործող կոնտակտային ձևեր անմիջապես զրույցի միջերեսից՝ օգտագործելով ձեր տեղադրված ձևերի plugin-ը
- **Բարելավված կայք կառուցող** — բարելավված բազմափուլ կայքի կառուցման պլաններ՝ plugin-ների ավելի լավ հայտնաբերմամբ և սխալների վերականգնմամբ
- **Ներկառուցված թեմայի հմտություններ** — մասնագիտացված ուղղորդում Block Themes-ի, Classic Themes-ի, Kadence Blocks-ի և Kadence Theme-ի համար
- **Բարելավված WooCommerce ինտեգրում** — վերամշակված է՝ օգտագործելու WooCommerce-ի բնիկ API-ները՝ ավելի լավ հուսալիության և համատեղելիության համար
- **Մատակարարների ավտոմատ հայտնաբերում** — ավտոմատ հայտնաբերում և թարմացնում է հասանելի plugin-ներն ու թեմաները, երբ դրանք ակտիվացվում կամ ապաակտիվացվում են
- **Պատվերով լոգոյի ստեղծում** — ստեղծեք և ներդրեք պատվերով SVG լոգոներ՝ namespace-ի համար անվտանգ ավտոմատ մաքրմամբ
- **Պալիտրայի հակադրության ստուգում** — ստուգեք գունային զույգերը WCAG հասանելիության համապատասխանության համար՝ նախքան ձեր թեմայում կիրառելը
- **Լուսանկարներով ընդլայնված հայտնաբերում** — վերբեռնեք լուսանկարներ Theme Builder-ի հայտնաբերման հարցազրույցի ընթացքում՝ դիզայնի ուղղության վրա ազդելու համար
- **Հյուրընկալության մենյուներ** — ստեղծեք կառուցվածքային սննդի և խմիչքների մենյուի էջեր ռեստորանների, սրճարանների և հյուրընկալության բիզնեսների համար
- **Աշխատասեղանի և բջջային նախադիտումներ** — նախադիտեք ձեր դիզայնը աշխատասեղանի և բջջային սարքերում՝ դիզայնի ուղղության ընտրության ընթացքում
- **Tier 1 կայքի քերում** — sd-ai-agent/site-scrape-ն այժմ հասանելի է լռելյայն՝ մրցակիցների և ոգեշնչման կայքերը վերլուծելու համար
- **Google Calendar-ի ընթերցման գործիքներ** — կարդացեք կազմաձևված օրացույցներն ու իրադարձությունները, որպեսզի ավտոմատացումները կարողանան դատողություն անել առաջիկա ժամանակացույցի համատեքստի մասին
- **Կոնտակտների և մասնակիցների համապատասխանեցում** — համապատասխանեցրեք օրացույցի մասնակիցներին հայտնի կայքի օգտատերերի և կոնտակտների հետ՝ նախքան հետագա աշխատանքի ստեղծումը
- **Մարդկային հաստատման դարպասներ** — դադարեցրեք զգայուն ավտոմատացումները վերանայման համար՝ նախքան հաղորդագրություններ ուղարկելը, աշխատանքը վերսկսելը կամ փոփոխություններ կատարելը
- **Հիշեցման գրառումներ և կրկնօրինակման կանխում** — հետևեք արդեն ուղարկված հիշեցումներին, որպեսզի կրկնվող ավտոմատացման գործարկումները նույն ստացողին երկու անգամ չծանուցեն
- **TextBee SMS ծանուցումներ** — ուղարկեք կազմաձևված տեքստային հաղորդագրության ծանուցումներ TextBee-ի միջոցով, երբ SMS մատակարարը միացված է
- **Կառավարվող Superdav ծառայության տրամադրում** — միացրեք աջակցվող կայքերը հոսթինգով Superdav endpoint-ներին՝ ձեռքով endpoint կազմաձևելու փոխարեն ավտոմատ կարգավորմամբ
- **Advanced ուղեկից փաթեթ** — առանձին տարածվող վստահելի մշակողի գործիքներ ֆայլային համակարգի, տվյալների բազայի, WP-CLI-ի, REST dispatcher-ի, plugin կառուցողի, git snapshot-ի, օգտատերերի կառավարման և չափորոշիչների աշխատանքային հոսքերի համար

## Պահանջներ {#requirements}

- WordPress 6.0 կամ ավելի բարձր
- PHP 8.1 կամ ավելի բարձր
- WordPress Multisite (ցանցային ակտիվացումով) կամ մեկ կայքի տեղադրում
- Ակտիվ AI մատակարարի connector plugin (Anthropic, OpenAI կամ համատեղելի)
- Gratis AI Agent v1.9.0 կամ ավելի բարձր (Superdav AI Agent-ը ընդլայնում է Gratis AI Agent-ը)

## Տեղադրում {#installation}

1. Վերբեռնեք plugin-ի ֆայլերը `/wp-content/plugins/superdav-ai-agent` պանակ
2. Ակտիվացրեք plugin-ը WordPress-ի **Plugins** էկրանի միջոցով
3. Համոզվեք, որ Gratis AI Agent-ը նույնպես ակտիվացված է
4. Անցեք **Gratis AI Agent → Settings** և ըստ անհրաժեշտության կազմաձևեք ձեր ինտերնետային որոնման մատակարարին, կառավարվող Superdav ծառայության կապը, Google Calendar-ը կամ TextBee հավատարմագրերը

WordPress.org փաթեթը պարունակում է Superdav AI Agent-ի հիմնական հնարավորությունները։ Advanced ուղեկից փաթեթը կառուցվում և տարածվում է առանձին այն միջավայրերի համար, որտեղ վստահելի մշակողի գործիքները բացահայտ տեղադրված և լիազորված են։

## Արագ մեկնարկ {#quick-start}

Ակտիվացումից հետո Superdav AI Agent-ը ավտոմատ բարելավում է ձեր Gratis AI Agent տեղադրումը՝ թեմային հարմարեցված հմտություններով և լրացուցիչ հնարավորություններով։ Բացեք **Gratis AI Agent** վահանակը WordPress-ի ադմինիստրատորի կողային գոտում և սկսեք օգտագործել նոր հնարավորությունները։

**Օրինակ հրահանգներ՝**

```
Ավելացրու կոնտակտային ձև Contact էջին։
```

```
Որոնիր համացանցում WordPress անվտանգության վերջին լավագույն գործելակերպերը։
```

```
Ստեղծիր hero բաժին՝ օգտագործելով ակտիվ թեմայի block pattern-ները։
```

```
Վերանայիր վաղվա կազմաձևված Google Calendar իրադարձությունները և կազմիր հիշեցումների սևագրեր մասնակիցների համար։
```

```
Ուղարկիր SMS հիշեցում, երբ ես հաստատեմ հաղորդագրությունը։
```

## Թեմային հարմարեցված հմտություններ {#theme-aware-skills}

Superdav AI Agent-ը ավտոմատ հայտնաբերում է ձեր ակտիվ թեման և տեղադրված plugin-ները՝ բեռնելով համապատասխան թեմային հարմարեցված հմտությունը օգնականի համատեքստի մեջ։ Հասանելի հմտությունները ներառում են՝

| Հմտություն | Կիրառվում է |
|---|---|
| **Block Themes** | Թեմաներ, որոնք օգտագործում են WordPress-ի block-ի վրա հիմնված (Full Site Editing) ճարտարապետությունը |
| **Classic Themes** | Ավանդական WordPress թեմաներ, որոնք օգտագործում են PHP ձևանմուշներ և classic editor-ը |
| **Kadence Blocks** | Կայքեր, որտեղ Kadence Blocks plugin-ն ակտիվ է |
| **Kadence Theme** | Կայքեր, որտեղ Kadence թեման ակտիվ թեման է |

Տեսեք [Թեմային հարմարեցված հմտություններ](../../user-guide/configuration/theme-aware-skills)՝ յուրաքանչյուր հմտության վերաբերյալ մանրամասն ուղղորդման համար։

## Ինտերնետային որոնման մատակարարներ {#internet-search-providers}

Superdav AI Agent v1.10.0-ն ավելացնում է **Tavily**-ն որպես Brave Search-ի այլընտրանք ինտերնետային որոնման հարցումների համար։ Երկու մատակարարներն էլ առաջարկում են իրական ժամանակի որոնման արդյունքներ և կարող են կազմաձևվել **Gratis AI Agent → Settings → Advanced** բաժնում։

| Մատակարար | Հնարավորություններ | Կարգավորում |
|---|---|---|
| **Brave Search** | Արագ, գաղտնիության վրա կենտրոնացած որոնման արդյունքներ | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Ավելի հարուստ որոնման արդյունքներ, առաջադեմ հետազոտական հնարավորություններ | [Tavily API](https://tavily.com/) |

Կարգավորման հրահանգների համար տես [Ինտերնետային որոնում](../../user-guide/configuration/internet-search)։

## Օրացույցի, հաստատման և ծանուցումների աշխատանքային հոսքեր {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0-ն ընդլայնում է ավտոմատացման աշխատանքային հոսքերը՝ բովանդակության և կայքի կառուցման առաջադրանքներից այն կողմ.

- **Google Calendar գործիքները** թույլ են տալիս գործակալին կարդալ կարգավորված օրացույցները և միջոցառման մանրամասները, երբ օրացույցի հավատարմագրերը հասանելի են։
- **Կոնտակտների և մասնակիցների օգնականները** միջոցառման մասնակիցներին կապում են WordPress օգտատերերի կամ հայտնի կոնտակտների հետ՝ նախքան հիշեցումներ կամ հաջորդող առաջադրանքներ պատրաստելը։
- **Մարդկային հաստատման դարպասները** թույլ են տալիս ավտոմատացումը դադարեցնել, մինչև լիազորված օգտատերը վերանայի առաջարկվող գործողությունը։
- **Հիշեցման գրառումները** կանխում են կրկնվող օրացույցային կամ ծանուցման հիշեցումները, երբ ավտոմատացումը կրկին փորձարկվում է կամ պարբերաբար ժամանակացուցվում։
- **TextBee SMS աջակցությունը** միացնում է տեքստային հաղորդագրությունների ծանուցումները, երբ TextBee մատակարարը կարգավորված է։

Օգտագործեք այս գործիքները ժամանակացույցին տեղյակ աջակցության, մասնակիցների հետագա կապի և հիշեցումների աշխատանքային հոսքերի համար, որոնք դեռ պահանջում են մարդկային վերանայում՝ նախքան զգայուն հաղորդագրություններ ուղարկելը։

## Կայքի կառուցման կոնտակտային ձևեր {#site-builder-contact-forms}

Կայքի կառուցման գործակալն այժմ կարող է ստեղծել կոնտակտային ձևեր անմիջապես զրույցի միջերեսից։ Պարզապես խնդրեք կայքի կառուցման գործակալին ստեղծել կոնտակտային ձև, և այն ինքնաբար կհայտնաբերի ձեր տեղադրված ձևերի plugin-ը (Contact Form 7, WPForms, Fluent Forms կամ Gravity Forms) և կստեղծի օգտագործման պատրաստ ձև։

Մանրամասն ուղեցույցի համար տես [Կայքի կառուցման կոնտակտային ձևեր](../../user-guide/administration/site-builder-contact-forms)։

## Theme Builder-ի հնարավորություններ {#theme-builder-features}

Superdav AI Agent v1.16.0-ի Theme Builder-ը ներառում է հզոր նոր հնարավորություններ.

### Բացահայտման հարցազրույց՝ լուսանկար վերբեռնելու հնարավորությամբ {#discovery-interview-with-photo-upload}

Theme Builder-ի բացահայտման հարցազրույցն այժմ ներառում է լուսանկար վերբեռնելու քայլ, որտեղ կարող եք տրամադրել տեսողական ոգեշնչում։ Վերբեռնեք ձեր բրենդի, մրցակիցների կայքերի կամ դիզայնի ոգեշնչման լուսանկարներ, որպեսզի օգնեք Theme Builder-ին հասկանալ ձեր դիզայնի ուղղությունը։

Մանրամասն ուղեցույցի համար տես [Բացահայտման հարցազրույց](./theme-builder/discovery-interview.md)։

### Դիզայնի ուղղություն՝ աշխատասեղանի և բջջային նախադիտումներով {#design-direction-with-desktop-and-mobile-previews}

Վերանայեք ձեր դիզայնը թե՛ աշխատասեղանի, թե՛ բջջային սարքերում՝ նախքան Theme Builder-ը կստեղծի ձեր ամբողջական theme-ը։ Դիտեք իրական ժամանակի նախադիտումներ մի քանի արձագանքող կոտրման կետերում։

Մանրամասն ուղեցույցի համար տես [Դիզայնի ուղղություն](./theme-builder/design-direction.md)։

### Հյուրընկալության մենյուներ {#hospitality-menus}

Theme Builder-ն այժմ կարող է ստեղծել կառուցվածքային սննդի և խմիչքների մենյու էջեր ռեստորանների, սրճարանների, բարերի և հյուրընկալության այլ բիզնեսների համար։ Մենյուները ներառում են գներ, նկարագրություններ, սննդակարգային տեղեկատվություն և ալերգենների նախազգուշացումներ։

Մանրամասն ուղեցույցի համար տես [Հյուրընկալության մենյուներ](./theme-builder/hospitality-menus.md)։

## Ընդլայնված ուղեկից փաթեթ {#advanced-companion-package}

Superdav AI Agent v1.18.0-ն ավելացնում է առանձին տարածվող Ընդլայնված ուղեկից փաթեթ՝ վստահելի մշակողների աշխատանքային հոսքերի համար։ Այս հնարավորությունները ներառված չեն WordPress.org-ի հիմնական փաթեթում։ Տեղադրեք ուղեկից փաթեթը միայն այն կայքերում, որտեղ ադմինիստրատորները բացահայտորեն վստահում են գործակալին՝ մշակողների գործիքակազմին հասանելիություն ունենալու համար։

Ընդլայնված գործիքները ներառում են ֆայլային համակարգի գործողություններ, տվյալների բազայի հասանելիություն, WP-CLI, REST դիսպետչերի աշխատանքային հոսքեր, plugin կառուցման օգտակար գործիքներ, git լուսապատկերներ, օգտատերերի կառավարման օգնականներ և կատարողականի չափման գործիքակազմ։ Այս գործիքները սահմանափակեք վստահելի ադմինիստրատորներով և վերանայեք հաստատման դարպասների կարգավորումը՝ նախքան դրանք արտադրական կայքերում միացնելը։

- **1-ին մակարդակի կայքի քերծման գործիք** — sd-ai-agent/site-scrape-ը այժմ լռելյայն հասանելի է Theme Builder-ում՝ մրցակից և ոգեշնչման կայքերը վերլուծելու համար
- **Plugin-ի տողի գործողության հղումներ** — շտկվել և վերանվանվել են՝ հստակության համար

## Բարելավումներ v1.10.0-ում {#improvements-in-v1100}

- **WooCommerce ինտեգրումը վերակառուցվել է** — այժմ օգտագործում է բնիկ WooCommerce API-ներ՝ ավելի լավ հուսալիության և համատեղելիության համար
- **Մատակարարների ցանկի ինքնաթարմացում** — ավտոմատ հայտնաբերում է, երբ plugin-ները ակտիվացվում կամ ապաակտիվացվում են, և թարմացնում է հասանելի մատակարարների ցանկը
- **Ընդլայնված կայք կառուցող** — բարելավված բազմափուլ կայքի կառուցման պլաններ՝ սխալներից ավելի լավ վերականգնմամբ
- **Theme-ին հարմարեցված հմտություններ** — չորս նոր ներկառուցված հմտություններ, որոնք ավտոմատ հարմարվում են ձեր ակտիվ theme-ին

## Սխալների շտկումներ v1.10.0-ում {#bug-fixes-in-v1100}

- Շտկվել է `navigate-to` կարողությունը, որը որոշ admin էջերում առաջացնում էր անվերջ վերաբեռնման ցիկլեր
- Շտկվել է `list-posts` կարողությունը, որը ճիշտ չէր վերածում category-ի և tag-ի անունները slugs-ի
- Շտկվել են WP-CLI հրամանները, որոնց մոտ նախորդ վերակառուցումից հետո բացակայում էին namespace alias-ները
- Շտկվել է event automation-ը, որպեսզի սահուն մշակի այն կայքերը, որտեղ automation աղյուսակները դեռ ստեղծված չեն
- Շտկվել է `memory-save` կարողությունը, որը օգտագործում էր ճիշտ namespace նախածանցը system instruction builder-ում
- Շտկվել է scalar գործիքի արդյունքների ճիշտ փաթեթավորումը՝ նախքան AI-ին վերադարձվելը
- Շտկվել է usage statistics-ի կողմից legacy ability key ձևաչափի ճիշտ մշակումը՝ ավելի հին տարբերակներից թարմացման ժամանակ
