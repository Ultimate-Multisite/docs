# Փոփոխությունների մատյան {#changelog}

## 1.18.0 — Թողարկվել է 2026-06-29 {#1180--released-on-2026-06-29}

### Նոր {#new}

- **Google Calendar գործիքներ** — կարդում են կարգավորված օրացույցներն ու իրադարձությունները՝ ժամանակացույցին իրազեկ ավտոմատացումների համար
- **Կոնտակտների քարտեզագրում և մասնակիցների օգնականներ** — համադրում են օրացույցի մասնակիցներին կայքի օգտատերերի և կոնտակտների հետ
- **Մարդկային հաստատման դարպասներ և հիշեցումների գրառումներ** — դադարեցնում են ավտոմատացումները վերանայման համար և կանխում կրկնվող ծանուցումները
- **TextBee SMS մատակարար** — ուղարկում է կարգավորված տեքստային հաղորդագրությունների ծանուցումներ TextBee-ի միջոցով
- **Ընդլայնված ուղեկից փաթեթ** — ավելացնում է վստահելի մշակողի ֆայլային համակարգի, տվյալների բազայի, WP-CLI-ի, REST դիսպետչերի, plugin-ի կառուցման, git snapshot-ի, օգտատերերի կառավարման և benchmark գործիքներ, որոնք բաշխվում են WordPress.org կառուցվածքից առանձին

### Բարելավված {#improved}

- **Կառավարվող Superdav ծառայության կարգավորում** — ավելացվում են հյուրընկալված ծառայության endpoint-ներ և ավտոմատ կապի տրամադրում աջակցվող կայքերի համար
- **Թողարկման փաթեթավորում** — կառուցվում են առանձին core և Advanced ZIP-եր, երկուսն էլ հրապարակվում են GitHub-ում, և միայն core փաթեթն է ուղարկվում WordPress.org

### Ուղղված {#fixed}

- **AI հարցումների հուսալիություն** — բարելավվել են մոդելի ընտրությունը, հարցման ժամանակավրեպումները, լռելյայն կարգավորման մոդելը, դատողության տեքստի մշակումը և կրկնափորձի ուղեցույցները անվավեր գործիքային կանչերի համար
- **Օրացույցի և հիշեցումների ամրացում** — ուժեղացվել են Google Calendar token-ները և հիշեցումների կրկնօրինակման կանխումը
- **Նախնական կարգավորման և հաստատման վերսկսումներ** — ուղղվել են frontend նախնական կարգավորման գործարկումները և հաստատված կարողությունների վերսկսումները
- **WordPress.org փաթեթավորման վերանայման խնդիրներ** — շտկվել է core թողարկման փաթեթավորման վերանայման արձագանքը

## 1.16.0 — Թողարկվել է 2026-05-20 {#1160--released-on-2026-05-20}

### Նոր {#new-1}

- **Generate Logo SVG կարողություն** — Theme Builder-ն այժմ կարող է ստեղծել և ներդնել անհատական logo SVG-ներ՝ namespace-ի համար անվտանգ մաքրմամբ
- **Լուսանկարի վերբեռնում հայտնաբերման հարցազրույցում** — Theme Builder-ի հայտնաբերման հարցազրույցն այժմ ներառում է լուսանկարի վերբեռնման քայլ՝ ավելի հարուստ դիզայնային համատեքստի համար
- **Validate Palette Contrast կարողություն** — ստուգում է գունային զույգերը WCAG համապատասխանության համար՝ նախքան theme-ի վրա կիրառելը
- **Հյուրընկալության մենյուներ** — Theme Builder-ն այժմ կարող է ստեղծել կառուցվածքային սննդի և ըմպելիքների մենյուի էջեր հյուրընկալության բիզնեսների համար
- **Desktop և mobile նախադիտման արտապատկերում** — նախադիտեք ձեր դիզայնը desktop և mobile սարքերում՝ դիզայնի ուղղության ընտրության ընթացքում
- **Նավիգացիոն պիտակի պարամետր** — Create Menu կարողությունն այժմ աջակցում է առանձին `navigation_label`, որը տարբերվում է էջի վերնագրից
- **Tier 1 գործիքի հասանելիություն** — sd-ai-agent/site-scrape-ն այժմ Tier 1 գործիք է, որը լռելյայն հասանելի է Theme Builder-ում

### Ուղղված {#fixed-1}

- **AI Client cache** — այժմ ապահովվում է transients-ներով՝ հարցումների միջև կայունության համար, կանխելով տվյալների կորուստը երկարատև agent առաջադրանքների ժամանակ
- **Plugin-ի տողի գործողությունների հղումներ** — ուղղվել և վերանվանվել են հստակության համար

## 1.10.0 — Թողարկվել է 2026-05-05 {#1100--released-on-2026-05-05}

### Նոր {#new-2}

- **Tavily ինտերնետային որոնում** — ավելացվել է Tavily-ն որպես որոնման մատակարար՝ Brave Search-ի կողքին ավելի հարուստ ինտերնետային որոնման արդյունքների համար
- **Theme-ին իրազեկ ներկառուցված հմտություններ** — Block Themes, Classic Themes, Kadence Blocks և Kadence Theme հմտությունների ուղեցույցներն այժմ տրամադրվում են plugin-ի հետ
- **Կայքի կառուցողի կոնտակտային ձևի կարողություն** — ավելացրեք կոնտակտային ձև ցանկացած էջի վրա անմիջապես chat միջերեսից

### Բարելավված {#improved-1}

- **WooCommerce ինտեգրումը վերակառուցվել է** — այժմ օգտագործում է բնիկ WooCommerce API-ներ՝ ավելի լավ հուսալիության և համատեղելիության համար
- **Մատակարարների ցանկը թարմացվում է ավտոմատ** — երբ որևէ plugin ակտիվացվում կամ ապակտիվացվում է

### Ուղղված {#fixed-2}

- **navigate-to կարողություն** — ուղղվել է որոշ admin էջերում անվերջ վերաբեռնման ցիկլը
- **list-posts կարողություն** — այժմ ճիշտ է լուծում category և tag անունները որպես slug-եր
- **WP-CLI հրամաններ** — վերականգնվել են բացակայող namespace կեղծանունները նախորդ վերակառուցումից հետո
- **Իրադարձությունների ավտոմատացում** — նրբորեն մշակում է կայքերը, որտեղ ավտոմատացման աղյուսակները դեռ ստեղծված չեն
- **memory-save կարողություն** — այժմ օգտագործում է ճիշտ namespace նախածանցը համակարգային հրահանգների կառուցիչում
- **Սկալյար գործիքների արդյունքներ** — այժմ ճիշտ փաթեթավորվում են նախքան AI-ին վերադարձվելը
- **Օգտագործման վիճակագրություն** — այժմ ճիշտ է մշակում հին կարողության բանալու ձևաչափը հին տարբերակներից թարմացման ժամանակ
