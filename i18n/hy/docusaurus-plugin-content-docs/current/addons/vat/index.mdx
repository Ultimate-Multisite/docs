# VAT {#vat}

Multisite Ultimate՝ եվրոպական VAT

Ավելացնում է լրացուցիչ գործառույթներ և գործիքներ Multisite Ultimate-ի լռելյայն հարկերի մշակմանը՝ եվրոպական VAT-ն ավելի լավ աջակցելու համար։
Ավտոմատ կերպով ավելացնում է EU VAT-ի դրույքաչափերը Հարկային դրույքաչափերի էջում՝

## Տեղադրում {#installation}

1. Վերբեռնեք 'ultimate-multisite-vat'-ը '/wp-content/plugins/' գրացուցակ
2. Ակտիվացրեք plugin-ը WordPress-ի 'Plugins' ընտրացանկի միջոցով
3. Գնացեք Ultimate Multisite > Կարգավորումներ > Հարկեր և միացրեք EU Vat-ը։
4. Սեղմեք Հարկային դրույքաչափերի վրա, ներքևում թարմացրեք EU VAT-ի դրույքաչափերը և պահպանեք հարկային դրույքաչափերը։
5. Պատրաստ է։
