# VAT փոփոխությունների մատյան {#vat-changelog}

Տարբերակ 1.0.7 - Թողարկվել է 2026-02-03-ին

* Ուղղում․ Գերմանական (DE) VAT համարների վավերացումը ձախողվում էր VIES SOAP ծառայության MS_MAX_CONCURRENT_REQ սխալների պատճառով։ Հին SOAP API-ից անցում է կատարվել պաշտոնական EU VIES REST API-ին՝ ավելի հուսալի վավերացման համար։
* Ուղղում․ VAT համարներն այժմ ընդունվում են, երբ VIES ծառայությունը ժամանակավորապես անհասանելի է, սխալմամբ մերժվելու փոխարեն։ Ստեղծվում է մատյանի գրառում՝ հետագա վերանայման համար։
* Ուղղում․ VAT/Tax ID դաշտն այժմ տեսանելի է բոլոր երկրների համար, ոչ միայն EU անդամների։ Ոչ EU հարկային ID-ները (օրինակ՝ շվեյցարական CHE համարները) պահվում են հաշիվ-ապրանքագրում ցուցադրելու համար՝ առանց VIES վավերացման։
* Ուղղում․ Հունաստանի (GR), Մոնակոյի (MC) և Մեն կղզու (IM) համար VAT նախածանցի որոնումը խափանված էր զանգվածում որոնման սխալ տրամաբանության պատճառով։
* Բարելավում․ Հեռացվել է ավելորդ wp-cli-bundle մշակման կախվածությունը, որը կարող էր ճակատագրական սխալներ առաջացնել կայքի արտահանիչ հավելման հետ միասին օգտագործելիս։
* Բարելավում․ Ավելացվել է համապարփակ միավորային թեստերի փաթեթ (53 թեստ)։

Տարբերակ 1.0.6 - Թողարկվել է 2026-01-25-ին

* Ուղղում․ Ներքին B2B գործարքներն այժմ ճիշտ են գանձում VAT։ EU VAT կանոնների համաձայն՝ հակադարձ գանձումը կիրառվում է միայն անդրսահմանային B2B գործարքների համար, ոչ թե այն դեպքում, երբ հաճախորդի երկիրը համընկնում է ընկերության երկրի հետ։

Տարբերակ 1.0.5 - Թողարկվել է 2026-01-22-ին

* Ուղղում․ Հավելումը չէր բեռնվում կամ ճիշտ չէր աշխատում։
* Փոփոխություն․ VAT դրույքաչափերի տվյալների աղբյուրը euvatrates.com-ից փոխվել է ibericode/vat-rates պահոցի՝ ավելի հուսալի և ակտիվորեն պահպանվող տվյալների համար։
* Ուղղում․ Ուղղվել է super_reduced_rates տարբերակի արժեքի տառասխալը։
* Բարելավում․ Ավելացվել են null ստուգումներ երկրի կոդի մշակման համար՝ սխալները կանխելու նպատակով։
* Փոփոխություն․ Հեռացվել են փաթեթավորված թարգմանության ֆայլերը՝ հօգուտ Traduttore-ի միջոցով ավտոմատ թարմացումների։

Տարբերակ․ 1.0.3 - Թողարկվել է 2025-09-28-ին

* Վերանվանել նախածանցը ultimate-multisite-ի․ թարմացնել տեքստային տիրույթը․ տարբերակի բարձրացում։
