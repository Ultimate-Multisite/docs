# Զտիչ՝ wu_cap_captcha_config {#filter-wucapcaptchaconfig}

Զտում է Cap Captcha-ի կազմաձևումը։

Թույլ է տալիս մշակողներին հարմարեցնել Cap Captcha-ի կարգավորումները։

## Պարամետրեր {#parameters}

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $config | `array` | Cap-ի կազմաձևման զանգվածը։ |
| $level | `string` | Անվտանգության մակարդակը (fast, medium, max)։ |
