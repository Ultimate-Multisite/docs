# Chuck Norris փաստերի փոփոխությունների մատյան {#chuck-norris-facts-changelog}

### 1.0.1 - 2025-11-03 {#101---2025-11-03}
* Ավելացվել է․ ավտոմատ թարմացման գործառույթ Ultimate Multisite թարմացման սերվերի միջոցով
* Թարմացվել է․ փորձարկված է մինչև WordPress 6.6
* Թարմացվել է․ բարելավված plugin-ի կառուցվածք՝ composer autoloader-ով
* Թարմացվել է․ ընդլայնված համատեղելիություն ժամանակակից WordPress տեղադրումների հետ

### 1.0.0 - 2024-10-21 {#100---2024-10-21}
* Նախնական թողարկում
* 110+ Chuck Norris փաստեր
* Լիարժեք թարգմանության աջակցություն
* Արձագանքող դիզայն՝ էկրանի բոլոր չափերի համար

* Block editor-ի համատեղելիություն
* Մատչելիության գործառույթներ՝ screen reader-ի աջակցությամբ
