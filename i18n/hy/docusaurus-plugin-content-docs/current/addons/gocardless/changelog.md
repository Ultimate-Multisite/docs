# GoCardless փոփոխությունների մատյան {#gocardless-changelog}

Տարբերակ՝ 1.0.3 - Թողարկվել է 2026-05-05-ին
* Ուղղում՝ մեկուսացած սպասման վիճակում գտնվող կայքերն այժմ մաքրվում են, երբ GoCardless անդամակցությունը չեղարկվում է սպասման կամ պահման վիճակում
* Բարելավված՝ plugin-ի autoloader-ը շրջանցվում է, երբ Bedrock root autoloader-ն արդեն բեռնել է կախվածությունները

Տարբերակ՝ 1.0.1 - Թողարկվել է 2025-09-28-ին

* Վերանվանել prefix-ը ultimate-multisite-ի; թարմացնել text domain-ը; տարբերակի բարձրացում։



2021-09-24 - Տարբերակ 1.0.0-beta.4

* Ավելացվել է՝ filter wp_ultimo_skip_network_active_check mu-plugins-ի վրա հիմնված կարգավորումների համար;

2021-06-16 - Տարբերակ 1.0.0-beta.3

* Ուղղում՝ Webhook handlers-ը չի կարողանում հաստատել ստորագրությունները;

2021-06-14 - Տարբերակ 1.0.0-beta.2

* Բարելավված՝ հրապարակել կայքը GoCardless mandate ստեղծվելուց հետո;

Տարբերակ 1.0.0-beta.1 - Սկզբնական տարբերակ
