# Աջակցության գործակալներ {#support-agents}

Ամբողջ ցանցը միայնակ կառավարելը հեշտ գործ չէ։ Երբ ձեր ցանցը սկսի աճել, կսկսեք տեսնել աջակցության պահանջարկի ավելացում։ Լավ միտք կլինի ներգրավել մարդկանց, որոնք կօգնեն ձեզ թեթևացնել ձեր աշխատանքային ծանրաբեռնվածությունը։ Սակայն լիարժեք հասանելիություն/թույլտվություն տրամադրելը ռիսկային է և կարող է վտանգել ձեր ցանցը։ Ահա այստեղ է, որ այս Աջակցության գործակալի հավելումը օգտակար է դառնում։

Այս հավելումը թույլ է տալիս ստեղծել WordPress օգտատիրոջ նոր տեսակ՝ Աջակցության գործակալ, և ձեզ տալիս է լիակատար վերահսկողություն այն թույլտվությունների նկատմամբ, որոնք ունի այդ օգտատերը։ Թույլտվություն տրամադրելն այնքան հեշտ է, որքան փոխարկիչ կոճակը միացնելը, իսկ այն հեռացնել որոշելու դեպքում՝ անջատելը։

## Առանձնահատկություններ {#features}

- WordPress և Ultimate Multisite թույլտվությունների մանրամասն ցուցակ
- Հեշտությամբ միացնել և անջատել հասանելիությունը
- Ստեղծել անսահմանափակ քանակությամբ Գործակալի հաշիվներ

## Տեղադրում {#installation}

1. Վերբեռնեք 'ultimate-multisite-support-agents'-ը '/wp-content/plugins/' պանակ
2. Ակտիվացրեք փլագինը WordPress-ում 'Փլագիններ' ընտրացանկի միջոցով
3. Հետևեք քայլ առ քայլ Wizard-ին՝ փլագինը կարգավորելու համար
