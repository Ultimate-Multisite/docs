# Ֆիլտր՝ wp_ultimo_skip_network_active_check

Թույլ է տալիս մշակողներին շրջանցել այս ստուգումը։

Սա օգտակար է composer-ի վրա հիմնված և այլ անհատական կարգավորումներ օգտագործելիս, օրինակ՝ Bedrock-ի դեպքում, որտեղ plugin-ները որպես mu-plugins օգտագործելը սովորական է։

- 2.0.0
