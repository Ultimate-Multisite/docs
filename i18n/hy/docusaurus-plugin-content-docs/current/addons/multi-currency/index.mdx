# Բազմարժույթ

Ultimate Multisite: Բազմարժույթը թույլ է տալիս ձեր ցանցին ընդունել վճարումներ մի քանի արժույթներով։ Այն ավտոմատ կերպով փոխարկում է գները՝ օգտագործելով կենդանի կամ ձեռքով սահմանված փոխարժեքներ, թույլ է տալիս սահմանել ճշգրիտ գներ յուրաքանչյուր ապրանքի համար ցանկացած արժույթով և հայտնաբերում է յուրաքանչյուր այցելուի նախընտրած արժույթը գեոլոկացիայի միջոցով՝ ամբողջը կառավարվելով մեկ admin dashboard-ից։

## Հիմնական հնարավորություններ

- **Փոխարժեքի ավտոմատ փոխարկում** — գները փոխարկվում են ձեր բազային արժույթից՝ օգտագործելով ձեր ընտրած մատակարարից ստացված փոխարժեքները
- **Յուրաքանչյուր ապրանքի գների վերագրումներ** — սահմանեք ճշգրիտ արժույթային գներ ցանկացած ապրանքի համար՝ շրջանցելու ավտոմատ փոխարկումը
- **Գրանցման արժույթի ընտրիչ դաշտ** — թույլ է տալիս հաճախորդներին ընտրել իրենց նախընտրած արժույթը checkout/գրանցման ժամանակ
- **Փոխարժեքի մի քանի մատակարարներ** — ECB, Frankfurter, Stripe կամ ամբողջությամբ ձեռքով սահմանված փոխարժեքներ
- **Գեոլոկացիայի վրա հիմնված արժույթի հայտնաբերում** — ավտոմատ կերպով նախապես ընտրում է այցելուի երկրին համապատասխանող արժույթը
- **Admin dashboard** — կառավարեք ակտիվ արժույթները, փոխարժեքները և մատակարարի կարգավորումները մեկ վայրից

## Պահանջներ

- Ultimate Multisite 2.0.0 կամ ավելի բարձր
- WordPress Multisite տեղադրում
- PHP 7.4 կամ ավելի բարձր

## Տեղադրում

1. Վերբեռնեք plugin-ի ֆայլերը `/wp-content/plugins/ultimate-multisite-multi-currency`
2. Ակտիվացրեք plugin-ը WordPress-ի **Plugin-ներ** էկրանից (Ցանցային ակտիվացում)
3. Գնացեք **Ultimate Multisite → Բազմարժույթ**՝ admin dashboard-ը բացելու համար
4. Ավելացրեք այն արժույթները, որոնք ցանկանում եք ընդունել, և կարգավորեք ձեր փոխարժեքի մատակարարին

## Կարգավորում

### Admin Dashboard

Բազմարժույթի admin dashboard-ը (**Ultimate Multisite → Բազմարժույթ**) կենտրոնական վայրն է՝

- Ավելացնելու, միացնելու կամ անջատելու արժույթներ
- Ընտրելու փոխարժեքի մատակարար և ըստ պահանջի թարմացնելու փոխարժեքները
- Դիտելու ընթացիկ փոխարժեքները բոլոր ակտիվ արժույթների համար
- Սահմանելու ձեռքով փոխարժեքներ, երբ ավտոմատ մատակարարները հարմար չեն

### Փոխարժեքի մատակարարներ

| Մատակարար | Նկարագրություն |
|---|---|
| **ECB** | Եվրոպական կենտրոնական բանկ — անվճար օրական փոխարժեքներ EUR-ի վրա հիմնված ցանցերի համար |
| **Frankfurter** | Բաց կոդով ECB հայելային API — անվճար, API բանալի չի պահանջվում |
| **Stripe** | Օգտագործում է Stripe-ի կենդանի FX փոխարժեքները — պահանջում է ակտիվ Stripe ինտեգրում |
| **Ձեռքով** | Դուք ուղղակիորեն մուտքագրում եք փոխարժեքները․ արտաքին API կանչեր չկան |

Ընտրեք ձեր մատակարարին **Ultimate Multisite → Բազմարժույթ → Կարգավորումներ → Փոխարժեքի մատակարար** բաժնում։ Փոխարժեքները կարող են ցանկացած պահի թարմացվել ձեռքով կամ պլանավորվել ավտոմատ թարմացման համար։

### Յուրաքանչյուր ապրանքի գների վերագրումներ

Երբ ավտոմատ փոխարկումը բավարար ճշգրիտ չէ, կարող եք սահմանել ճշգրիտ գին կոնկրետ արժույթի համար ցանկացած ապրանքի վրա՝

1. Բացեք ապրանքի խմբագրիչը (**Ultimate Multisite → Ապրանքներ → Խմբագրել**)
2. Ոլորեք մինչև **Արժույթային գնագոյացում** բաժինը
3. Մուտքագրեք ճշգրիտ գինը յուրաքանչյուր արժույթի համար, որը ցանկանում եք վերագրել
4. Պահպանեք ապրանքը

Վերագրում չունեցող ապրանքներն օգտագործում են ձեր բազային արժույթից փոխարկված գինը։

### Գրանցման արժույթի ընտրիչ դաշտ

Արժույթի ընտրիչ դաշտը կարող է ավելացվել ձեր գրանցման/checkout հոսքին, որպեսզի հաճախորդները գրանցման ժամանակ ընտրեն իրենց նախընտրած արժույթը՝

1. Գնացեք **Ultimate Multisite → Բազմարժույթ → Կարգավորումներ**
2. Միացրեք **Ցույց տալ արժույթի ընտրիչը գրանցման ժամանակ**
3. Ըստ ցանկության սահմանեք լռելյայն արժույթ կամ թողեք, որ գեոլոկացիան որոշի նախապես ընտրված արժեքը

Ընտրված արժույթը պահվում է բաժանորդագրության հետ և օգտագործվում է հետագա բոլոր հաշվարկների համար։

### Գեոլոկացիայի վրա հիմնված արժույթի հայտնաբերում

Երբ գեոլոկացիայի հայտնաբերումը միացված է, addon-ը կապում է այցելուի երկիրը արժույթի հետ և նախապես ընտրում է այն արժույթի ընտրիչ դաշտում։ Միացնելու համար՝

1. Գնացեք **Ultimate Multisite → Բազմարժույթ → Կարգավորումներ**
2. Միացրեք **Հայտնաբերել արժույթը այցելուի գտնվելու վայրից**
3. Կապեք երկրները արժույթներին **Գեոլոկացիայի համապատասխանեցում** աղյուսակում

Եթե այցելուի երկրի համար համապատասխանեցում չկա, որպես պահուստային տարբերակ օգտագործվում է ցանցի բազային արժույթը։

## Հաճախ տրվող հարցեր

### Ի՞նչ է տեղի ունենում, եթե փոխարժեքի մատակարարը հասանելի չէ։

Addon-ը վերադառնում է վերջին հաջողությամբ ստացված փոխարժեքներին։ Եթե փոխարժեքներ երբևէ չեն ստացվել, այն վերադառնում է ձեր կարգավորած ձեռքով փոխարժեքներին։ Խորհուրդ է տրվում կարգավորել ձեռքով փոխարժեքներ՝ որպես ապահովագրական միջոց։

### Կարո՞ղ եմ տարբեր արժույթների համար օգտագործել տարբեր մատակարարներ։

Ոչ — միաժամանակ բոլոր արժույթների համար ակտիվ է մեկ մատակարար։ Օգտագործեք **Ձեռքով**, եթե ձեզ անհրաժեշտ է ամբողջական վերահսկողություն առանձին փոխարժեքների նկատմամբ։

### Գեոլոկացիան պահանջո՞ւմ է առանձին plugin։

Արտաքին գեոլոկացիայի plugin չի պահանջվում։ Addon-ը օգտագործում է WordPress-ի ներկառուցված IP-ից երկիր հայտնաբերման հնարավորությունը։ Ավելի բարձր ճշգրտության համար հատուկ գեոլոկացիայի ծառայություն կարող է ինտեգրվել տրամադրված filter hook-երի միջոցով։

### Յուրաքանչյուր ապրանքի վերագրումները կիրառվո՞ւմ են հարկից առաջ, թե՞ հետո։

Յուրաքանչյուր ապրանքի վերագրումները սահմանում են նախահարկային գինը թիրախային արժույթով։ Հարկը հաշվարկվում է վերագրված գնի վրա՝ օգտագործելով ձեր առկա հարկային կարգավորումը։
