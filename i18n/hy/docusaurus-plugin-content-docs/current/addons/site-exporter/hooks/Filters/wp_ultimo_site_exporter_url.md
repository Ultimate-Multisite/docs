# Զտիչ՝ wp_ultimo_site_exporter_url

Թույլ է տալիս մշակողներին կառավարել պլագինի արմատային պանակի հարաբերական ուղին։

## Պարամետրեր {#parameters}

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $dir | `string` | Ուղի՝ հարաբերական պլագինի արմատային պանակին, որին ցանկանում եք մուտք գործել։ |

### Սկսած {#since}

- 0.0.1
### Աղբյուր {#source}

Սահմանված է [`inc/class-helper.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-site-exporter/blob/main/inc/class-helper.php#L52)-ում՝ 52-րդ տողում


## Վերադարձնում է {#returns}
Ուղի՝ հարաբերական պլագինի արմատային պանակին։
