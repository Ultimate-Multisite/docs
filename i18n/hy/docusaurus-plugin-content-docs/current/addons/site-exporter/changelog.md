# Կայքի արտահանիչի փոփոխությունների մատյան

Տարբերակ՝ 1.0.6 - Թողարկվել է 2026-05-11-ին
* Նոր՝ Կայքի արտահանումներն այժմ ներառում են ինքնագործարկվող index.php, որպեսզի ZIP-ը հնարավոր լինի տեղադրել նոր հոսթում՝ առանց առանձին plugin-ի տեղադրման
* Նոր՝ Ցանցի արտահանումը թույլ է տալիս ադմինիստրատորներին արտահանել բոլոր ենթակայքերը մեկ արխիվում՝ Կայքի արտահանման ադմինիստրատորի էջից

Տարբերակ՝ 1.0.5 - Թողարկվել է 2026-05-05-ին
* Ուղղում՝ Ավելացվել է __callStatic WP_CLI polyfill stub-ին՝ չմոկավորված մեթոդները, օրինակ՝ get_config, մշակելու համար՝ կանխելով թեստի սկզբնաբեռնման ձախողումները

Տարբերակ՝ 1.0.4 - Թողարկվել է 2025-11-25-ին

* Ուղղում՝ համատեղելիություն Ultimate Multisite 2.4-ի հետ։

Տարբերակ՝ 1.0.2 - Թողարկվել է 2025-09-28-ին

* Վերանվանվել է նախածանցը ultimate-multisite-ի; թարմացվել է տեքստային տիրույթը; տարբերակի բարձրացում։

### Տարբերակ 1.0.1 - Թողարկվել է 2023-08-09-ին

* Բարելավված՝ Այժմ ապահովում է, որ ստացվի ներմուծողի ֆայլի ճիշտ ֆայլային ուղին։
* Ուղղված՝ Անջատվել է օբյեկտների քեշը՝ ներմուծման ընթացքում սխալները կանխելու համար։
* Ուղղված՝ Ուղղվել է ներմուծող plugin-ի ներբեռնման հղման չաշխատելու խնդիրը։

### Տարբերակ 1.0.0 - Թողարկվել է 2022-12-23-ին

* Ուղղված՝ Արտահանման ձևի popup-ը չէր բեռնվում;
* Ուղղված՝ Javascript ֆայլերը չէին բեռնվում արտահանման էջում;
* Ներքին՝ Նոր կառուցման գործընթաց;

### Տարբերակ 1.0.0-beta.4 - Թողարկվել է 2022-01-13-ին

* Ներքին՝ Ավելացվել է hooks և filters գեներատոր;
* Ներքին՝ Ավելացվել են WP Ultimo stubs՝ մշակողների աշխատանքը հարմարեցնելու համար;
