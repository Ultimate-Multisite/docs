# Չափված պլանների փոփոխությունների մատյան {#metered-plans-changelog}

Տարբերակ 1.1.0 - Թողարկվել է 2026-05-05-ին
- Նոր՝ AI token-ի հաշվարկ multisite ենթակայքերի համար — հետևեք և հաշվարկեք AI token-ի օգտագործումը հաճախորդների կայքերում՝ կարգավորվող մեկ token-ի դրույքաչափերով
- Նոր՝ Connector-ի պարտադրումը վերակառուցվել է դինամիկ սահմանաչափերի հայտնաբերմամբ և անմիջական գրանցմամբ՝ ապահովելով իրական ժամանակի ճշգրտություն բոլոր connector-ներում
- Ուղղում՝ տվյալների բազայի աղյուսակների թարմացումներն այժմ ճիշտ են համապատասխանեցված BerlinDB սխեմայի սահմանումներին՝ կանխելով թարմացման ձախողումները նոր տեղադրումներում
- Ուղղում՝ տվյալների բազայի թարմացման callback-ները փոխարկվել են ճիշտ ձևաչափի՝ լուծելով լուռ թարմացման ձախողումները
- Ուղղում՝ կոտորակային արժեքներն այժմ ընդունվում են AI Usage Overage Markup մուտքագրման դաշտում
- Ուղղում՝ plugin-ի բեռնման ժամանակ լուծվել են ճակատագրական սխալներն ու կրկնակի սկզբնավորման խնդիրները
- Բարելավված՝ ավելացվել է check-env npm script, որպեսզի մշակողի միջավայրերը առաջին գործարկման ժամանակ ինքնակարգավորվեն

### 1.0.3 {#103}
* Թարմացվել է Plugin Update Checker v5-ի
* Ավելացվել են ժամանակակից WordPress plugin-ի վերնագրեր
* Բարելավվել է համատեղելիությունը WordPress-ի վերջին տարբերակների հետ
* Բարելավվել է օգտագործման հետևման կատարողականը

### 1.0.2 {#102}
* Սխալների ուղղումներ և կատարողականի բարելավումներ
* Բարելավված օգտագործման հաշվետվություն

### 1.0.0 {#100}
* Սկզբնական թողարկում
* Չափված հաշվարկման հիմնական գործառույթներ
* Օգտագործման հետևում և գերօգտագործման հաշվարկ
* Ինքնաշխատ հաշիվ-ապրանքագրերի ստեղծում
