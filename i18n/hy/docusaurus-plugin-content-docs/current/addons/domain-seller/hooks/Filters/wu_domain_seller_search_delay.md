# Զտիչ՝ wu_domain_seller_search_delay

Զտում է դոմենի որոնման debounce հապաղումը միլիվայրկյաններով։

Բարձրացրեք այս արժեքը՝ դանդաղ կապերի դեպքում API կանչերը նվազեցնելու համար։

## Պարամետրեր {#parameters}

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $delay | `int` | Debounce հապաղում միլիվայրկյաններով։ Լռելյայն՝ 500։ |
