# Domain Seller-ի փոփոխությունների մատյան

Տարբերակ 1.3.0 - Թողարկվել է 2026-06-02-ին
- Նոր․ Ավելացվել է network-admin նախազգուշացում, երբ HostAfrica վերավաճառողի մնացորդը չափազանց ցածր է դառնում
- Նոր․ Ավելացվել է նոր գրանցված դոմենների ավտոմատ կապակցում ցանցի կայքին
- Ուղղում․ Գրանցողի դաշտերի պահանջները կիրառվել են միայն նոր դոմեն գրանցելիս
- Ուղղում․ Մոնիտորի մնացորդի ծանուցումները դարձվել են փակվող
- Ուղղում․ Ապահովվել է, որ WooCommerce գրանցողի վճարային տվյալները պահպանվեն
- Ուղղում․ Գրանցման ընթացքում պարտադրվել են գրանցողի կոնտակտային պահանջները
- Ուղղում․ Կանխվել է դոմենի գրանցման ապրանքների ստեղծումը 0% հավելագնով
- Ուղղում․ Պահպանվել են դոմենի ընտրությունները և գնագոյացումը checkout-ի սեսիայի հոսքի ընթացքում
- Ուղղում․ Բարելավվել է HostAfrica դոմենների գների արժույթի ցուցադրումը
- Ուղղում․ Բարելավվել է checkout ձևի action-ի վարքագիծը՝ WP-core query-var անհամապատասխանությունները կանխելու համար
- Բարելավում․ Կարգավորման ուղեցույցում հղում է ավելացվել HostAfrica վերավաճառողի կարգավորման փաստաթղթերին

Տարբերակ 1.2.0 - Թողարկվել է 2026-05-25-ին
- Նոր․ HostAfrica-ն ավելացվել է որպես դոմենների վաճառքի ինտեգրում՝ checkout-ի, կարգավորման վիզարդի, որոնման, TLD/գնագոյացման, գրանցման, երկարաձգման, փոխանցման, անունների սերվերի, DNS-ի, EPP կոդի, ռեգիստրատորի կողպման և ID պաշտպանության աջակցությամբ
- Նոր․ Openprovider-ն ավելացվել է որպես դոմենների վաճառքի ինտեգրում՝ վերավաճառողի գնագոյացման, գրանցման, երկարաձգման, փոխանցման, անունների սերվերի, DNS-ի, EPP կոդի, ռեգիստրատորի կողպման, WHOIS գաղտնիության և TLD համաժամացման աջակցությամբ
- Նոր․ Hostinger-ն ավելացվել է որպես դոմենների վաճառքի ինտեգրում՝ օգտագործելով հիմնական ինտեգրումից համօգտագործվող Hostinger API token-ը հասանելիության ստուգումների, գրանցման, անունների սերվերի թարմացումների, ռեգիստրատորի կողպման և WHOIS գաղտնիության համար
- Բարելավում․ Ավելացվել են դոմենի կենսացիկլի գործողության և զտիչի docblock-ներ՝ մշակողների ինտեգրման ուղեցույցի համար
- Բարելավում․ Addon-ի readme-ում plugin-ի համատեղելիության մետատվյալները թարմացվել են մինչև WordPress 7.0
- Բարելավում․ Թարմացվել են պլանավորման ձևանմուշները, որոնք օգտագործվում են առաջիկա թողարկման համակարգման համար

Տարբերակ 1.0.1 - Թողարկվել է 2026-02-27-ին

* **Նոր:** TLD ներմուծման գործիք՝ զանգվածային գների կառավարման համար
* **Նոր:** Ներածական գների աջակցություն դոմենային արտադրանքների համար
* **Նոր:** E2E թեստերի հավաքածու՝ Cypress-ով
* **Նոր:** Էլ․ փոստի ձևանմուշներ դոմենի կենսացիկլի ծանուցումների համար
* **Նոր:** Գրանցողի հասցեի դաշտեր admin դոմենի գրանցման մոդալում՝ նախապես լրացված կարգավորումներից
* **Նոր:** Հաճախորդի DNS կառավարման միջերես՝ գրառում ավելացնելու, խմբագրելու և ջնջելու աջակցությամբ
* **Նոր:** «Բերեք ձեր սեփական դոմենը» checkout տարբերակ՝ ավտոմատ դոմենի կապակցմամբ
* **Նոր:** Checkout-ի ընթացքում դոմենի անունից կայքի URL-ի ավտոմատ ստեղծում
* **Նոր:** Լռելյայն nameserver-ի և DNS գրառումների կազմաձևում կարգավորումներում
* **Նոր:** Դոմենի գրանցման մանրամասներ և DNS կառավարում հիմնական դոմենի խմբագրման էջում
* **Նոր:** Setup wizard-ը ավտոմատ ստեղծում է լռելյայն դոմենային արտադրանք՝ ողջամիտ լռելյայններով
* **Նոր:** Ամենօրյա ավտոմատ TLD համաժամացում cron-ի միջոցով բոլոր կազմաձևված մատակարարների միջով
* **Նոր:** WHOIS գաղտնիության պաշտպանություն՝ ըստ արտադրանքի կազմաձևմամբ (միշտ միացված, հաճախորդի ընտրություն կամ անջատված)
* **Նոր:** WHOIS գաղտնիության checkout նշավանդակ՝ գների ցուցադրմամբ և dark mode աջակցությամբ
* **Նոր:** Register Domain admin էջ՝ ձեռքով դոմենի գրանցման համար
* **Նոր:** Ավտոմատ plugin թարմացումներ Ultimate Multisite թարմացման սերվերի միջոցով
* **Նոր:** Դոմենային արտադրանքի տեսակի զտիչ ներդիր արտադրանքների ցանկի աղյուսակում՝ մանուշակագույն կրծքանշանի ոճավորմամբ
* **Նոր:** Գրանցողի կոնտակտային դաշտեր (անուն, հասցե, քաղաք, նահանգ, փոստային կոդ, երկիր, հեռախոս) դոմենի checkout ձևում
* **Նոր:** Գրանցողի դաշտերի վավերացում մինչև registrar API կանչելը՝ պարզ սխալի հաղորդագրություններով
* **Նոր:** Մատակարարին հատուկ մատյանների ալիքներ դոմենի գրանցման իրադարձությունների համար (օր.՝ domain-seller-namecheap.log)
* **Նոր:** Գրանցողի կոնտակտային դաշտեր հիմնական գրանցման/signup checkout ձևում (ցուցադրվում է դոմեն գրանցելիս)
