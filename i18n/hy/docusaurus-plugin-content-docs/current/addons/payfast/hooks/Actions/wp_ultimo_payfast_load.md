# Գործողություն՝ wp_ultimo_payfast_load {#action-wpultimopayfastload}

Գործարկվում է, երբ բոլոր կախվածությունները բեռնված են

Թույլ է տալիս plugin մշակողներին ավելացնել նոր գործառույթներ։ Օրինակ՝ աջակցություն նոր հոսթինգ մատակարարների համար և այլն։
