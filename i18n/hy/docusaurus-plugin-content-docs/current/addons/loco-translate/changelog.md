# Loco Translate փոփոխությունների մատյան

### 1.0.1 {#101}
* Բարելավված համատեղելիություն Loco Translate-ի վերջին տարբերակի հետ
* Բարելավված մուտքի վերահսկում՝ ավելի լավ անվտանգության համար
* Արտադրողականության օպտիմալացումներ

### 1.0.0 {#100}
* Սկզբնական թողարկում
* Կայքին հատուկ թարգմանությունների պանակներ
* Թարգմանությունների առաջնահերթության համակարգ
* Մուտքի վերահսկման իրականացում
* Անխափան UI ինտեգրում Loco Translate-ի հետ
