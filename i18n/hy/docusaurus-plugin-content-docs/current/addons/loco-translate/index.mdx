# Loco Translate {#loco-translate}

Ultimate Multisite: Loco Translate Integration հավելումը ձեր WordPress multisite ցանցին բերում է կայքին հատուկ թարգմանության հնարավորություններ։ Եթե երբևէ ձեզ անհրաժեշտ է եղել, որ ձեր ցանցի տարբեր կայքերը ունենան նույն plugin-ի կամ theme-ի տարբեր թարգմանություններ, այս հավելումը կատարյալ լուծում է։

**Ինչ է անում այս հավելումը**

Այս հավելումը անխափան ինտեգրում է Ultimate Multisite-ը հայտնի Loco Translate plugin-ի հետ՝ հնարավորություն տալով ձեր ցանցի յուրաքանչյուր ենթակայքին պահպանել իր սեփական անհատականացված թարգմանությունները։ Անկախ նրանից՝ դուք աշխատեցնում եք բազմատարածաշրջանային հարթակ, սպասարկում եք տարբեր լեզվական համայնքներ, թե պարզապես ձեր ցանցում թարգմանության ճկունության կարիք ունեք, այս հավելումը դա դարձնում է շատ հեշտ։

**Կատարյալ է՝**

* Բազմատարածաշրջանային բիզնեսների համար՝ տարբեր տեղայնացման կարիքներով
* Կրթական հաստատությունների համար, որոնք սպասարկում են բազմազան համայնքներ
* SaaS հարթակների համար, որոնք առաջարկում են բազմալեզու աջակցություն
* Վեբ գործակալությունների համար, որոնք կառավարում են բազմաթիվ հաճախորդների կայքեր
* Ցանկացած multisite ցանցի համար, որը պահանջում է թարգմանության ճկունություն

## Տեղադրում {#installation}

1. Համոզվեք, որ Ultimate Multisite-ը տեղադրված է և ակտիվացված է ցանցի մակարդակով
2. Տեղադրեք և ակտիվացրեք Loco Translate plugin-ը
3. Վերբեռնեք այս հավելումը ձեր plugin-ների պանակ
4. Ցանցի մակարդակով ակտիվացրեք Loco Translate Integration հավելումը
5. Սկսեք ստեղծել կայքին հատուկ թարգմանություններ Loco Translate-ի միջոցով ցանկացած ենթակայքում

Ահա և վերջ։ Ինտեգրումը աշխատում է ավտոմատ՝ առանց որևէ լրացուցիչ կարգավորման։

## Հաճախ տրվող հարցեր {#frequently-asked-questions}

## Տեղադրումից հետո պետք է որևէ բան կարգավորե՞մ {#do-i-need-to-configure-anything-after-installation}

Ոչ։ Ինտեգրումը ավտոմատ է։ Հենց որ և՛ հավելումը, և՛ Loco Translate-ը ակտիվ լինեն, ենթակայքերը կարող են անմիջապես սկսել ստեղծել իրենց սեփական անհատականացված թարգմանությունները։

## Ի՞նչ է տեղի ունենում, եթե ենթակայքը չի ստեղծում անհատականացված թարգմանություններ {#what-happens-if-a-subsite-doesnt-create-custom-translations}

Կայքը ավտոմատ կօգտագործի ցանցի մակարդակի թարգմանությունները որպես պահուստային տարբերակ։ Ոչինչ չի խափանվում, և ամեն ինչ աշխատում է անխափան։

## Կարո՞ղ են ենթակայքերը մուտք ունենալ ցանցի մակարդակի թարգմանություններին {#can-subsites-access-network-wide-translations}

Սովորական կայքի ադմինները տեսնում և խմբագրում են միայն իրենց սեփական կայքի թարգմանությունները։ Super admin-ները կարող են մուտք ունենալ և՛ ցանցի մակարդակի, և՛ կայքին հատուկ թարգմանություններին՝ կախված նրանից, թե որ կայքն են դիտում։

## Սա աշխատո՞ւմ է և՛ plugin-ների, և՛ theme-երի հետ {#does-this-work-with-both-plugins-and-themes}

Այո։ Կայքին հատուկ թարգմանությունները աշխատում են ինչպես WordPress plugin-ների, այնպես էլ theme-երի համար։

## Սա կդանդաղեցնի՞ իմ multisite ցանցը {#will-this-slow-down-my-multisite-network}

Ոչ։ Հավելումն օգտագործում է արդյունավետ ֆայլերի բեռնում, որը ստուգում է կայքին հատուկ թարգմանությունները միայն անհրաժեշտության դեպքում։ Կատարողականության նկատելի ազդեցություն չկա։

## Կարո՞ղ եմ ցանցային թարգմանությունները պատճենել որոշակի կայք՝ որպես մեկնարկային կետ {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

Այո։ Loco Translate-ում թարգմանություններ խմբագրելիս կարող եք հեշտությամբ սկսել ցանցային թարգմանություններից և հարմարեցնել դրանք յուրաքանչյուր կայքի համար։

## Սա աշխատո՞ւմ է WooCommerce-ի և այլ plugin-ների հետ {#does-this-work-with-woocommerce-and-other-plugins}

Իհարկե։ Ցանկացած plugin կամ theme, որն օգտագործում է ստանդարտ WordPress թարգմանության ֆունկցիաները, կաշխատի կայքին հատուկ թարգմանությունների հետ։
