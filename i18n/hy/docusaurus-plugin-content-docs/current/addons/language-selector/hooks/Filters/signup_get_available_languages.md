# Filter: signup_get_available_languages {#filter-signupgetavailablelanguages}

Զտում է ճակատային մասի կայքի գրանցումների համար հասանելի լեզուների ցանկը։

Այս hook-ին դատարկ array փոխանցելը կանջատի կարգավորման արտածումը գրանցման ձևում, և կայքը ստեղծելիս կօգտագործվի կանխադրված լեզուն։ Արդեն չտեղադրված լեզուները կհեռացվեն։

## Պարամետրեր {#parameters}

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $available_languages | `array` | Հասանելի լեզուներ։ |
