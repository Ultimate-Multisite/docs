# Լեզվի ընտրիչ {#language-selector}

Multisite Ultimate. լեզվի ընտրիչ գրանցման ընթացքում

Ավելացրեք լեզվի ընտրության դաշտ գրանցման ընթացքին։

## Տեղադրում {#installation}

1. Վերբեռնեք 'ultimate-multisite-language-selector'-ը '/wp-content/plugins/' գրացուցակ
2. Ակտիվացրեք plugin-ը ձեր WordPress Network Admin-ի 'Plugins' մենյուի միջոցով

## Օգտագործման ուղեցույց {#usage-guide}

Այս անվճար հավելումով կարող եք թույլ տալ ձեր օգտատերերին ընտրել, թե որ լեզուն են ցանկանում տեսնել գրանցման ընթացքում և մուտքի էջում։

Ձեր [լիցենզիան ակտիվացնելուց](../../user-guide/getting-started/installing-ultimate-multisite.md) հետո կկարողանաք տեղադրել Ultimate Multisite հավելումներ, որոնց հասանելիություն է տալիս ձեր լիցենզիան։

Գնացեք ձեր Ultimate Multisite կարգավորումների էջ և ապա ձախ կողմի մենյուում սեղմեք _Հավելումներ_ պիտակի վրա։

![Հավելումների էջ](/img/admin/addons-page.png)

_Գնացեք Հավելումների էջ՝ Language Selector-ի տեղադրումը սկսելու համար_

Այնտեղ կգտնեք մեր առաջարկած բոլոր հավելումները։

Անվճար հավելումները հասանելի են մեր բոլոր հաճախորդներին՝ անկախ բաժանորդագրության պլանից, որին նրանք գրանցվել են։

Սեղմեք հավելման մանրապատկերի տակ գտնվող _Տեղադրել հիմա՛_ կոճակը։ Այնուհետև ակտիվացրեք այն Plugins էջում։

![Հավելումների էջ՝ տեղադրման կոճակով](/img/admin/addons-page.png)

_Սեղմեք կոճակը՝ հավելումը տեղադրելու համար_

<!-- Սքրինշոթը հասանելի չէ. WP Plugins էջ, որը ցույց է տալիս Network Activate կոճակը Language Selector հավելման համար -->

_Ակտիվացրեք այն ձեր Plugins էջում_

Այժմ գնացեք Ultimate Multisite-ի Կարգավորումների մենյու և բացեք _Ցանցի կարգավորումներ_ ներդիրը։ Այնտեղ՝ էջի ներքևում, կգտնեք նոր բաժին՝ _Լեզուներ_ անունով։

![Կարգավորումների էջ](/img/config/settings-general.png)

_Լիովին նոր բաժին է հայտնվում՛_

Կարող եք այն դատարկ թողնել՝ բոլոր լեզուները որպես տարբերակներ ցուցադրելու համար, կամ ընտրել, թե որոնք եք ցանկանում հասանելի դարձնել։ Լեզուներն այն լեզուներն են, որոնք տեղադրել եք ձեր WordPress կարգավորման մեջ։

<!-- Սքրինշոթը հասանելի չէ. լեզվի ընտրության բացվող ցանկ, որը ցույց է տալիս WP տեղադրման վրա հասանելի լեզուները -->

_Այս WP տեղադրման վրա հասանելի լեզուների ցանկ_

Ձեր կատարած փոփոխությունները պահպանելուց հետո ձեր ընտրած լեզուներով ընտրության մենյու կցուցադրվի Մուտքի էջում, ինչպես նաև Գրանցման ընթացքի ժամանակ։

![Մուտքի և գրանցման կարգավորումներ](/img/config/settings-login-registration.png)

_Language Selector մուտքի էջում_

![Գրանցման էջ](/img/config/registration-page.png)

_Language Selector գրանցման ընթացքի ժամանակ_

## Լեզվի սահմանումը URL-ի միջոցով {#setting-the-language-via-the-url}

Կարող եք նաև փոխել էջի լեզուն՝ locale պարամետրը URL-ին փոխանցելով։ Օրինակ՝ եթե ցանկանում եք վերահղել դեպի գրանցման էջ իտալերենով, URL-ին ավելացրեք ?locale=it_IT։

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Որպեսզի սա աշխատի, լեզուն պետք է տեղադրված լինի ձեր WordPress ցանցում։
