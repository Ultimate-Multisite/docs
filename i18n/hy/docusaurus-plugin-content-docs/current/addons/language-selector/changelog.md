# Լեզվի ընտրիչի փոփոխությունների մատյան

Տարբերակ՝ 1.1.3 - Թողարկվել է 2026-05-05-ին
* Ուղղում․ մուտքային փոփոխականներն այժմ մաքրվում են, իսկ անհատական hook անուններին ավելացվում է նախածանց՝ երրորդ կողմի հավելումների հետ բախումները կանխելու համար

Տարբերակ՝ 1.1.0 - Թողարկվել է 2025-09-28-ին
