# Աջակցության հայտերի փոփոխությունների մատյան {#support-tickets-changelog}

### 1.0.4 - 2026-05-05 {#104---2026-05-05}
* Բարելավված է․ Հեռացվել է vendor/ պանակը Git հետևումից (արդեն ընդգրկված է .gitignore-ով)՝ նվազեցնելով պահոցի չափը
* Բարելավված է․ Փորձարկվել է մինչև WordPress 7.0

### 1.0.3 - 2026-05-01 {#103---2026-05-01}
* Ուղղում․ Ընդունել null արժեքներ nullable հայտի մոդելի setter-ներում
* Ուղղում․ Պահել բոլոր հայտերը գլոբալ՝ ամբողջ ցանցի աղյուսակներում՝ ճիշտ multisite աջակցության համար
* Ուղղում․ Թաքցնել անձնակազմի դաշտերը և ուղղել դատարկ էջի վերահղումը հաճախորդի նոր հայտի ձևում
* Ուղղում․ Փոխարինել չսահմանված add_meta() կանչերը ճիշտ update_meta()-ով՝ մետատվյալների պահպանման համար
* Ուղղում․ Փոխարինել չգրանցված հնարավորության ստուգումը ճիշտ wu_view_all_support_tickets-ով
* Ուղղում․ Ավելացնել բացակայող AJAX մշակիչներ հայտի կարգավիճակի, նշանակման և արագ խմբագրման գործողությունների համար
* Ուղղում․ Ուղղել մեթոդի անունը հայտերի դիտումներում անձնակազմի պատասխանի հայտնաբերման համար
* Ուղղում․ Միավորել կրկնվող պատասխանի մշակիչները և համապատասխանեցնել nonce գործողությունների անունները
* Ուղղում․ Ավելացնել բացակայող frontend դիտում [wu_submit_ticket] shortcode-ի համար
* Ուղղում․ Ավելացնել բացակայող user_id սյունակը և ուղղել Support_Ticket::get_user_id() մեթոդը
* Ուղղում․ Հեռացնել ավելորդ կրկնակի չակերտը առաջնահերթության ֆիլտրի option թեգում
* Ուղղում․ Ավելացնել ցանցի ադմինիստրատորի հայտերի կառավարման վահանակ super admin-ների համար
* Բարելավված է․ Միավորել ադմինիստրատորի CSS-ը մեկ արտաքին stylesheet-ի մեջ
* Բարելավված է․ Հեռացնել Settings ենթամենյուն ենթակայքի ադմինիստրատորի մենյուից
* Բարելավված է․ Պայմանականորեն բեռնել frontend ռեսուրսները միայն աջակցության հայտերի էջերում
* Բարելավված է․ Բաց թողնել plugin autoloader-ը, երբ Bedrock արմատային autoloader-ն արդեն բեռնել է կախվածությունները

### 1.0.2 - 2025-12-11 {#102---2025-12-11}
* Ավելացված է․ Հայտերի ամբողջական դիտման գործառույթ ադմինիստրատորի և frontend-ի համար
* Ավելացված է․ AJAX մշակիչ հայտի պատասխանների ուղարկման համար
* Ավելացված է․ Աջակցություն հայտի պատասխանման գործառույթի համար՝ ճիշտ ձևի մշակմամբ
* Ավելացված է․ Ճիշտ ծանուցումների ցուցադրում հայտերի ուղարկումների և պատասխանների համար ադմինիստրատորի բաժնում
* Ավելացված է․ Հայտի ավտոմատ կապակցում ընթացիկ օգտատիրոջ հետ հաճախորդի ինքնուրույն ուղարկումների համար
* Ավելացված է․ Անվտանգության բարելավում՝ հաճախորդներին հայտի սեփականությունը վերագրանցելուց կանխելու համար
* Ավելացված է․ Բացակայող օգնական ֆունկցիաներ (wu_format_date, wu_user_can_view_ticket և այլն)
* Ավելացված է․ Ֆայլերի կցորդների ճիշտ ներբեռնում և մշակում
* Ավելացված է․ Էլ․ փոստի ծանուցումների համակարգ հայտերի պատասխանների և կարգավիճակի փոփոխությունների համար
* Ուղղված է․ Հայտի պատասխանի ձևը այժմ ներառում է անհրաժեշտ հայտի ID-ն
* Ուղղված է․ Ծանուցումների տեսանելիության խնդիրներ ադմինիստրատորի հայտերի դիտումներում
* Ուղղված է․ Շարահյուսական սխալներ աջակցության հայտերի ֆունկցիաներում
* Բարելավված է․ Տվյալների բազայի կառուցվածք՝ պատասխանների և կցորդների համար ճիշտ հարցման և սխեմայի դասերով
* Բարելավված է․ Միգրացիա դեպի իրական հատկություններ՝ հայտերի մոդելների ատրիբուտների փոխարեն

### 1.0.1 - 2025-09-28 {#101---2025-09-28}
* Թարմացված է․ Նախածանցը վերանվանվել է ultimate-multisite՝ հետևողականության համար
* Թարմացված է․ Տեքստային տիրույթի ստանդարտացում
* Ուղղված է․ Փոքր սխալների ուղղումներ և բարելավումներ

### 1.0.0 - 2025-09-01 {#100---2025-09-01}
* Սկզբնական թողարկում
* Հայտերի կառավարման ամբողջական համակարգ
* Բազմամակարդակ հասանելիության վերահսկում
* Շղթայացված զրույցի համակարգ
* Ֆայլերի կցորդների աջակցություն
* Էլ․ փոստի ծանուցումների համակարգ
* Ադմինիստրատորի և հաճախորդի միջերեսներ
* Վիճակագրություն և հաշվետվություններ
