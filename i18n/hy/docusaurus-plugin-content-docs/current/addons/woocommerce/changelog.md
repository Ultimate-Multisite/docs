# WooCommerce ինտեգրման փոփոխությունների մատյան {#woocommerce-integration-changelog}

Տարբերակ 2.2.0 - Թողարկվել է 2026-07-01-ին
* Նոր․ Ultimate Multisite-ի հարկային գումարներն այժմ WooCommerce-ի վճարային առանձին տողերով են ցուցադրվում վճարման ձևակերպման ժամանակ՝ հարկերի ընդհանուր գումարներն ավելի պարզ դարձնելով մինչև վճարումը։
* Նոր․ Ավելացվել է համաձայնությամբ միացվող «Անմիջապես կասեցնել անդամակցությունները ձախողված երկարաձգումների դեպքում» կարգավորումը և `wu_woo_suspend_on_payment_failure` ֆիլտրը այն կայքերի համար, որոնք ցանկանում են ձախողված երկարաձգման դեպքում կասեցում կատարել WooCommerce Subscriptions-ի կրկնափորձերի պատուհանի ընթացքում։
* Ուղղում․ Համապատասխանեցվել են WooCommerce Subscription-ի կարգավիճակները, որոնք կարող էին չհամաժամեցված մնալ Ultimate Multisite-ի անդամակցությունների հետ ձախողված կամ վերականգնված երկարաձգումներից հետո։
* Ուղղում․ WooCommerce խանութի արժույթը ավելացվել է Ultimate Multisite-ի արժույթների ցանկին, երբ այն բացակայում էր։
* Ուղղում․ Պահպանվել են հաճախորդի վճարային տվյալները բաժանորդներին WooCommerce-ի վճարման ձևակերպում վերահղելիս։
* Բարելավում․ Ավելացվել է համատեղելիություն Jetpack Autoloader 5-ի հետ։
* Բարելավում․ Մաքրվել է թողարկման փաթեթի ստեղծումը, որպեսզի GitHub-ի և marketplace-ի zip-երը խուսափեն ներդրված staging պանակներից և մշակման ֆայլերից։
