# AI կայքի կառուցիչ

ultimate-multisite-ai-site-builder

## Պահանջներ

- WordPress 5.3 կամ ավելի բարձր
- PHP 7.4 կամ ավելի բարձր
- Multisite Ultimate plugin (ակտիվ)

## Տեղադրում

1. Վերբեռնեք հավելման ֆայլերը ձեր `/wp-content/plugins/` պանակ
2. Ակտիվացրեք plugin-ը WordPress-ի 'Plugins' ընտրացանկի միջոցով
3. Կարգավորեք հավելման կարգավորումները ցանցի ադմինիստրատորի բաժնում

## Մշակում

### Կարգաբերում

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Նախագծի կառուցվածք

```
ultimate-multisite-ai-site-builder/
├── inc/                          # PHP դասեր և ներառումներ
│   ├── class-multisite-ultimate-updater.php  # Թարմացման մշակիչ
│   └── class-ai_site_builder-main.php             # Հիմնական գործառույթներ
├── tests/                        # Միավորային թեստեր
│   ├── bootstrap.php            # Թեստերի սկզբնաբեռնում
│   ├── class-*-base.php         # Հիմնական թեստային դաս
│   └── class-*-test.php         # Թեստային դեպքեր
├── views/                       # Կաղապարի ֆայլեր
├── assets/                      # Ստատիկ ռեսուրսներ
│   ├── css/                     # Ոճաթերթեր
│   ├── js/                      # JavaScript ֆայլեր
│   └── images/                  # Պատկերներ
├── lang/                        # Լեզվային ֆայլեր
├── composer.json                # PHP կախվածություններ
├── package.json                 # Node.js կախվածություններ
├── phpunit.xml.dist             # PHPUnit կազմաձևում
├── .phpcs.xml.dist              # Կոդի ստանդարտների կազմաձևում
└── rector.php                   # Կոդի արդիականացման կանոններ
```

### Թեստավորում

Հավելումը ներառում է համապարփակ թեստավորման շրջանակ՝

- **Միավորային թեստեր**: PHPUnit-ի վրա հիմնված թեստեր՝ WordPress թեստային փաթեթով
- **Հիմնական թեստային դաս**: Տրամադրում է օգնական մեթոդներ սովորական թեստային սցենարների համար
- **Թեստերի ծածկույթ**: Կարգավորված է կոդի ծածկույթին հետևելու համար
- **Multisite թեստավորում**: Թեստերը գործարկվում են multisite միջավայրում

### Կոդի ստանդարտներ

- **WordPress կոդավորման ստանդարտներ**: Պարտադրվում են PHPCS-ի միջոցով
- **PHP 7.4+ համատեղելիություն**: Աջակցվում են ժամանակակից PHP հնարավորությունները
- **Ստատիկ վերլուծություն**: PHPStan ինտեգրում տիպերի ստուգման համար
- **Կոդի արդիականացում**: Rector կանոններ PHP թարմացումների համար

### Կառուցման համակարգ

- **Ռեսուրսների մշակում**: CSS/JS ֆայլերի փոքրացում
- **Թարգմանություն**: POT ֆայլի ստեղծում
- **Փաթեթավորում**: Արխիվի ստեղծում տարածման համար
- **Մշակման ռեժիմ**: Չփոքրացված ռեսուրսներ վրիպազերծման համար

## Կազմաձևում

Հավելումը աջակցում է կազմաձևման տարբեր ընտրանքների, որոնք հասանելի են ցանցի ադմինիստրատորի միջերեսի միջոցով։

## Hook-եր և զտիչներ

### Գործողություններ

- `ultimate-multisite-ai-site-builder_init` - Գործարկվում է հավելման սկզբնավորումից հետո
- `ultimate-multisite-ai-site-builder_loaded` - Գործարկվում է բոլոր հավելման ֆայլերի բեռնումից հետո

### Զտիչներ

- `ultimate-multisite-ai-site-builder_settings` - Փոփոխել հավելման կարգավորումները
- `ultimate-multisite-ai-site-builder_enabled` - Վերասահմանել հավելման միացված/անջատված կարգավիճակը
