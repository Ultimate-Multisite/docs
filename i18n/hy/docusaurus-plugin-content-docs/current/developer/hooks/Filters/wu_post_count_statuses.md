# Զտիչ: wu_post_count_statuses

Թույլ է տալիս plugin մշակողներին փոխել, թե որ գրառման կարգավիճակը պետք է հաշվվի։ Լռելյայն հաշվվում են հրապարակված և մասնավոր գրառումները

## Պարամետրեր

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $post_status | `array` | Գրառման կարգավիճակների ցանկը |
| $post_type | `string` | Գրառման տեսակի slug-ը |

### Սկսած

- 1.9.1
### Աղբյուր

Սահմանված է [`inc/limitations/class-limit-post-types.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/limitations/class-limit-post-types.php#L119)-ում՝ 119-րդ տողում


## Վերադարձնում է
Գրառման կարգավիճակների նոր զանգված
