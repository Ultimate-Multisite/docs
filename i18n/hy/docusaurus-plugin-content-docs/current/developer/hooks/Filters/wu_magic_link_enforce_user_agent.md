# Ֆիլտր՝ wu_magic_link_enforce_user_agent

Ֆիլտրում է՝ արդյոք պարտադրել օգտատիրոջ գործակալի ստուգումը։

Սահմանեք false՝ թույլ տալու, որ token-ները աշխատեն տարբեր դիտարկիչներում/սարքերում։ Սա նվազեցնում է անվտանգությունը, բայց բարձրացնում է օգտագործման հարմարությունը։

## Պարամետրեր {#parameters}

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $enforce | `bool` | Արդյոք պարտադրել օգտատիրոջ գործակալի համապատասխանությունը։ |
