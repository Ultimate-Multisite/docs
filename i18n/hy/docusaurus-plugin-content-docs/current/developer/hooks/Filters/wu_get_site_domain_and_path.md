# Ֆիլտր՝ wu_get_site_domain_and_path

Թույլ է տալիս մշակողներին փոփոխել տիրույթ/ուղի զույգերը։

Սա կարող է օգտակար լինել մի շարք բաների համար, օրինակ՝ որևէ տեսակի staging լուծում, տարբեր սերվերներ և այլն իրականացնելու համար։

## Պարամետրեր {#parameters}

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $d | `object` | Ընթացիկ օբյեկտը, որը պարունակում է տիրույթի և ուղու բանալիներ։ |
| $path_or_subdomain | `string` | Բնօրինակ ուղին/ենթատիրույթը, որը փոխանցվել է ֆունկցիային։ |

### Սկսած {#since}

- 2.0.0
### Աղբյուր {#source}

Սահմանված է [`inc/functions/site.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/site.php#L235)-ում՝ 235-րդ տողում


## Վերադարձնում է {#returns}
Օբյեկտ, որը պարունակում է տիրույթի և ուղու բանալիներ։
