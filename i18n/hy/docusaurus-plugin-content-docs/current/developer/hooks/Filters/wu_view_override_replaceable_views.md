# Զտիչ՝ wu_view_override_replaceable_views

Թույլ է տալիս մշակողներին ավելացնել լրացուցիչ պանակներ փոխարինելի ցանկին։

Զգույշ եղեք, քանի որ լրացուցիչ պանակների թույլատրումը կարող է հանգեցնել հնացած պատճենների բեռնմանը Ultimate Multisite տարբերակների փոխարեն։

## Պարամետրեր {#parameters}

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $replaceable_views | `array` | Թույլատրված պանակների ցանկ։ |

### Սկսած {#since}

- 2.0.0
