# Filter: wu_get_checkout_variables

Թույլ է տալիս plugin մշակողներին զտել վճարման էջի նախնական կարգավորումները։

Զգույշ եղեք․ բացակայող բանալիները կարող են ամբողջությամբ խափանել վճարումը front-end-ում։

## Պարամետրեր {#parameters}

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $variables | `array` | Տեղայնացված փոփոխականներ։ |
| $checkout | `\Checkout` | Վճարման դասը։ |

### Սկսած {#since}

- 2.0.0
