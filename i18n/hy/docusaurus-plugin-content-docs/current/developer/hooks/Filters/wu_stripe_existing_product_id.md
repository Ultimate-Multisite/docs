# Filter: wu_stripe_existing_product_id

Զտում է ստուգման ենթակա արտադրանքի ID-ն։ Եթե սա գոյություն ունի, նոր բաժանորդագրությունը կօգտագործի այս արտադրանքը։

## Պարամետրեր {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $product_id | `string` | Ստուգման ենթակա Stripe արտադրանքի ID-ն։ |
| $name | `string` | Արտադրանքի անուն։ |
