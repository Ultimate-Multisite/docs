# Զտիչ՝ wu_maybe_redirect_to_admin_panel_exclusion_list {#filter-wumayberedirecttoadminpanelexclusionlist}

Ստեղծում է պարամետրերի բացառման ցանկ, որոնք կանխում են ավտոմատ վերահղումը։

Սա անհրաժեշտ է, քանի որ հակառակ դեպքում էջի կառուցիչը չի կարողանա խմբագրել մուտքի էջը, երբ այն սահմանված է։

- 2.0.4
