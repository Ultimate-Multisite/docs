# Զտիչ: wu_admin_notices {#filter-wuadminnotices}

Թույլ է տալիս մշակողներին զտել Ultimate Multisite-ի կողմից ավելացված ադմինիստրատորի ծանուցումները։

## Պարամետրեր {#parameters}

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $notices | `array` | Տվյալ վահանակի ծանուցումների ցանկը։ |
| $all_notices | `array` | Ավելացված ծանուցումների ցանկը՝ բաժանված ըստ վահանակների։ |
| $panel | `string` | Վահանակը, որի ծանուցումները պետք է ստանալ։ |
| $filter | `string` | Արդյոք փակվող ծանուցումները զտվել են։ |
| $dismissed_messages | `array` | Փակված ծանուցումների բանալիների ցանկը։ |
