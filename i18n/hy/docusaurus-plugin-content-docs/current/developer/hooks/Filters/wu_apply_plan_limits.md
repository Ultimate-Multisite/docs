# Ֆիլտր․ wu_apply_plan_limits

Թույլ է տալիս plugin-ի մշակողներին շրջանցել սահմանափակումները։

Դուք կարող եք օգտագործել այս ֆիլտրը՝ կամայական code գործարկելու համար՝ նախքան որևէ սահմանափակումների մեկնարկը։ Եթե ձեր ֆիլտրը վերադարձնում է որևէ ճշմարտանման արժեք, գործընթացը կշարունակվի, իսկ եթե այն վերադարձնում է որևէ կեղծանման արժեք, code-ը կվերադառնա, և ստորև նշված hook-երից ոչ մեկը չի գործարկվի։

### Սկսած

- 1.7.0
