# Ֆիլտր՝ wu_wp_cli_enabled_sub_commands

Զտում է, թե որ sub_commands-ներն են միացված այս էության համար։

## Պարամետրեր

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $sub_commands | `array` | Լռելյայն sub_commands-ներ։ |
| $command_base | `string` | Հիմքը, որն օգտագործվում է հրամանում՝ անմիջապես արմատից հետո։ |
| $this | `\Base_Manager` | Օբյեկտի նմուշը։ |

### Սկսած

- 2.0.0
### Աղբյուր

Սահմանված է [`inc/apis/trait-wp-cli.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/apis/trait-wp-cli.php#L198)-ում՝ 198-րդ տողում
