# Ֆիլտր՝ wp_ultimo_skip_network_active_check {#filter-wpultimoskipnetworkactivecheck}

Թույլ է տալիս մշակողներին կարճ միացումով շրջանցել այս ստուգումը։

Սա օգտակար է composer-ի վրա հիմնված և այլ հարմարեցված կարգավորումներ օգտագործելիս, օրինակ՝ Bedrock, որտեղ plugins-ը որպես mu-plugins օգտագործելը սովորական է։
