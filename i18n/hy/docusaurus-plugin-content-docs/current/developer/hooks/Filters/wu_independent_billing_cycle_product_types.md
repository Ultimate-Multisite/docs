# Filter: wu_independent_billing_cycle_product_types {#filter-wuindependentbillingcycleproducttypes}

Զտում է այն ապրանքի տեսակները, որոնք ունեն անկախ վճարային ցիկլեր։

Անկախ վճարային ցիկլերով ապրանքները պարտադրված չեն զամբյուղում համապատասխանել պլանի վճարային ժամանակահատվածին։ Սա օգտակար է այնպիսի ապրանքների համար, ինչպիսիք են դոմենի գրանցումները, որոնք միշտ տարեկան են վճարվում՝ անկախ նրանից՝ պլանը ամսական է, թե տարեկան։

## Պարամետրեր {#parameters}

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $types | `array` | Անկախ վճարային ցիկլերով ապրանքի տեսակների զանգված։ |
