# Զտիչ: wu_generated_username_from_email

Զտել ստեղծված հաճախորդի օգտանունը։

## Պարամետրեր

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $username | `string` | Ստեղծված օգտանուն։ |
| $email | `string` | Նոր հաճախորդի էլ․ փոստի հասցե։ |
| $new_user_args | `array` | Նոր օգտատիրոջ args-ի զանգված, հնարավոր է՝ ներառելով անունն ու ազգանունը։ |
| $suffix | `string` | Օգտանվանը ավելացվող տող՝ այն եզակի դարձնելու համար։ |
