# Filter: wu_sso_url

Զտում է գեներացված SSO URL-ները՝ նախքան դրանք վերադարձվեն միջդոմենային հաճախորդի գործողությունների համար։

Օգտագործեք այս զտիչը, երբ ինտեգրմանը անհրաժեշտ է վստահելի համատեքստ ավելացնել ինքնիշխան վարձակալի SSO հղմանը կամ փոխարինել բրոքերի URL-ը՝ պահպանելով Ultimate Multisite-ի թոքենի վավերացումը։

## Պարամետրեր

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $sso_url | `string` | Գեներացված SSO URL։ |
| $user | `WP_User` | Օգտատեր, որը կվավերացվի SSO այցելությամբ։ |
| $site_id | `int` | Այցելության նպատակային կայքի ID։ |
| $redirect_to | `string` | Նպատակակետ URL՝ հաջող SSO վավերացումից հետո։ |

### Սկսած

- 2.13.0

### Աղբյուր

Սահմանված է `inc/sso/class-sso.php`-ում։
