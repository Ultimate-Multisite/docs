# Զտիչ՝ wu_pre_save_settings

Թույլ է տալիս մշակողներին զտել կարգավորումները նախքան Ultimate Multisite-ի կողմից պահպանումը։

## Պարամետրեր

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $settings | `array` | Պահպանվելիք կարգավորումները։ |
| $settings_to_save | `array` | Ավելացվելիք նոր կարգավորումները։ |
| $saved_settings | `array` | Ներկայում պահպանված կարգավորումները։ |

### Սկսած

- 2.0.18
