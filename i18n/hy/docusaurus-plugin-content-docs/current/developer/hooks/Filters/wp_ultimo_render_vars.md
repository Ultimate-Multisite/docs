# Ֆիլտր: wp_ultimo_render_vars {#filter-wpultimorendervars}

Թույլ է տալիս plugin մշակողներին գլոբալ կերպով ավելացնել լրացուցիչ փոփոխական render համատեքստում։

## Պարամետրեր {#parameters}

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $args | `array` | Զանգի render-ի կողմից փոխանցված փոփոխականներ պարունակող զանգված։ |
| $view | `string` | Render արվելիք view-ի անունը։ |
| $default_view | `string` | fallback_view-ի անունը |

- 2.0.0
