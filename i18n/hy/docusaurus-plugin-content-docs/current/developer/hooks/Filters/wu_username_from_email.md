# Զտիչ: wu_username_from_email {#filter-wuusernamefromemail}

Զտել նոր հաճախորդի օգտանունը։

## Պարամետրեր {#parameters}

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $username | `string` | Հաճախորդի օգտանունը։ |
| $email | `string` | Նոր հաճախորդի էլ․ փոստի հասցեն։ |
| $new_user_args | `array` | Նոր օգտատիրոջ արգումենտների զանգված, հնարավոր է՝ ներառյալ անունը և ազգանունը։ |
| $suffix | `string` | Օգտանվանը ավելացվող տող՝ այն եզակի դարձնելու համար։ |
