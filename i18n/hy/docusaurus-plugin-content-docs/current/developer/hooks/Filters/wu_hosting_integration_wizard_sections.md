# Filter: wu_hosting_integration_wizard_sections {#filter-wuhostingintegrationwizardsections}

Զտում է հոսթինգի ինտեգրման կարգավորման վիզարդի բաժինները։

Թույլ է տալիս հավելումներին ավելացնել, հեռացնել կամ փոփոխել վիզարդի բաժինները։

## Պարամետրեր {#parameters}

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $sections | `array` | Վիզարդի բաժիններ։ |
| $integration | `\WP_Ultimo\Integrations\Host_Providers\Base_Host_Provider\|\WP_Ultimo\Integrations\Integration` | Կարգավորվող ինտեգրումը։ |
| $page | `\Hosting_Integration_Wizard_Admin_Page` | Վիզարդի էջի օրինակ։ |
