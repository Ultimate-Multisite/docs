# Ֆիլտր՝ secure_auth_redirect {#filter-secureauthredirect}

Ֆիլտրում է՝ արդյոք օգտագործել անվտանգ նույնականացման վերահղում։

## Պարամետրեր {#parameters}

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $secure | `bool` | Արդյոք օգտագործել անվտանգ նույնականացման վերահղում։ Լռելյայն՝ false։ |
