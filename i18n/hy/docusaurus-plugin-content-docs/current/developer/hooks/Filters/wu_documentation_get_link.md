# Ֆիլտր՝ wu_documentation_get_link {#filter-wudocumentationgetlink}

Թույլ է տալիս plugin մշակողներին ֆիլտրել հղումները։

Հստակ չէ, թե դա ինչպես կարող է օգտակար լինել, բայց ունենալը չի խանգարում

## Պարամետրեր {#parameters}

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $link | `string` | Գրանցված հղումը |
| $slug | `string` | Հղումը ստանալու համար օգտագործված slug-ը |
| $default_link | `string` | Գրանցված լռելյայն հղումը |

- 1.7.0
