# Action: wu_dashboard_\{$this->tab\}_widgets {#action-wudashboardthis-tabwidgets}

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $tab | `string` | Ընթացիկ ներդիրը։ |
| $screen | `\WP_Screen` | Էկրանի օբյեկտը։ |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Ultimate Multisite-ի ադմինիստրատիվ էջի օրինակ։ |
