# Գործողություն՝ wu_dashboard_widgets

Թույլ է տալիս ընդլայնումների մշակողներին վիջեթներ ավելացնել ցանցային Dashboard վահանակում։

## Պարամետրեր

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $tab | `string` | Ընթացիկ ներդիրը։ |
| $screen | `\WP_Screen` | Էկրանի օբյեկտը։ |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Ultimate Multisite-ի կառավարման էջի նմուշը։ |

### Սկսած

- 2.0.0
