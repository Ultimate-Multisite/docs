# Action: wu_before_light_ajax

Որոշ դեպքերում մեզ անհրաժեշտ կլինի բեռնել լրացուցիչ ռեսուրսներ՝ գործողությունները մշակելու համար։

Այս գործողությունը կարող է օգտագործվել անհրաժեշտության դեպքում լրացուցիչ կախվածություններ բեռնելու համար։
