# Գործողություն: wu_sso_loaded

Թույլ է տալիս հավելումների մշակողներին անհրաժեշտության դեպքում ավելացնել լրացուցիչ hook-եր։

Սա պետք է հետաձգվի մինչև init, քանի որ SSO-ն աշխատում է sunrise-ում։
