# Գործողություն՝ wu_register_capabilities {#action-wuregistercapabilities}

Գործարկվում է, որպեսզի հավելումները կարողանան հնարավորությունների մոդուլներ կցել ինտեգրումներին։

## Պարամետրեր {#parameters}

| Անուն | Տիպ | Նկարագրություն |
|------|------|-------------|
| $registry | `\Integration_Registry` | Ռեեստրի օրինակը։ |
