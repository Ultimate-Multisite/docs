# Գործողություն՝ wu_mcp_adapter_initialized

Գործարկվում է MCP ադապտերի նախնականացումից հետո։

Թույլ է տալիս այլ plugin-ներին և theme-երին գրանցել իրենց սեփական հնարավորությունները։

## Պարամետրեր

| Անուն | Տեսակ | Նկարագրություն |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | MCP ադապտերի նմուշը։ |
