# Ultimate Multisite Documentation

Ultimate Multisite documentation में आपका स्वागत है। Ultimate Multisite, WordPress Multisite को एक पूर्ण Website as a Service (WaaS) प्लेटफॉर्म में बदल देता है।

## Documentation Sections

### [User Guide](/user-guide/getting-started/ultimate-multisite-101)

यह वह सब कुछ है जिसकी आपको अपना WaaS प्लेटफॉर्म सेट अप करने, कॉन्फ़िगर करने और मैनेज करने के लिए ज़रूरत है। इसमें इंस्टॉलेशन, पेमेंट गेटवे, डोमेन मैपिंग, क्लाइंट मैनेजमेंट और बहुत कुछ शामिल है।

### [Developer Documentation](/developer/hooks)

Ultimate Multisite पर काम करने वाले डेवलपर्स के लिए तकनीकी संदर्भ। इसमें हुक्स रेफरेंस, REST API डॉक्यूमेंटेशन, और ऐडऑन डेवलपमेंट गाइड शामिल हैं।

### [Addons](/addons)

भुगतान वाले ऐडऑन प्लगइन्स का डॉक्यूमेंटेशन जो Ultimate Multisite को अतिरिक्त सुविधाओं जैसे WooCommerce इंटीग्रेशन, एनालिटिक्स, डोमेन सेलिंग, और बहुत कुछ से बढ़ाते हैं।
