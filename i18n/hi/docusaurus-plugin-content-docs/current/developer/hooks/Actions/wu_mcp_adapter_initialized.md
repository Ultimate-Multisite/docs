# Action: wu_mcp_adapter_initialized

यह तब ट्रिगर होता है जब MCP अडैप्टर को आरंभ (initialize) किया जाता है।

यह अन्य प्लगइन्स और थीम्स को अपनी क्षमताएं (abilities) रजिस्टर करने की अनुमति देता है।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | MCP अडैप्टर का इंस्टेंस। |

- 2.5.0
