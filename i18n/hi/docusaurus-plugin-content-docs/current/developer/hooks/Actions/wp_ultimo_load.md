# Action: wp_ultimo_load {#action-wpultimoload}

जब सभी निर्भरताएँ (dependencies) लोड हो जाती हैं, तब यह ट्रिगर होता है।

यह प्लगइन डेवलपर्स को नई कार्यक्षमता (functionality) जोड़ने की अनुमति देता है। उदाहरण के लिए, नए होस्टिंग प्रोवाइडर्स (Hosting providers) के लिए सपोर्ट जोड़ना, आदि।
