# Action: wu_register_integrations

यह फंक्शन इंटीग्रेशन्स को खुद को रजिस्टर (register) करने की अनुमति देने के लिए फायर होता है।

कोर (Core) यहाँ अपने इंटीग्रेशन्स को रजिस्टर करता है।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $registry | `\Integration_Registry` | रजिस्ट्री का इंस्टेंस (instance)। |

### Since {#since}

- 2.5.0

[`inc/integrations/class-integration-registry.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/integrations/class-integration-registry.php#L113) में लाइन 113 पर परिभाषित है।
