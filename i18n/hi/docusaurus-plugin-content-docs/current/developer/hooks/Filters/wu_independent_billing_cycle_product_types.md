# Filter: wu_independent_billing_cycle_product_types

उन उत्पाद प्रकारों को फ़िल्टर करता है जिनका बिलिंग चक्र स्वतंत्र होता है।

जिन उत्पादों का बिलिंग चक्र स्वतंत्र होता है, उन्हें कार्ट में प्लान की बिलिंग अवधि से मेल खाने के लिए मजबूर नहीं किया जाता है। यह डोमेन पंजीकरण जैसे उत्पादों के लिए उपयोगी है, जिनका बिल हमेशा सालाना होता है, भले ही प्लान मासिक हो या वार्षिक।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | स्वतंत्र बिलिंग चक्र वाले उत्पाद प्रकारों का ऐरे (Array)। |
