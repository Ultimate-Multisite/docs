# फ़िल्टर: auth_redirect_scheme {#filter-authredirectscheme}

## पैरामीटर्स {#parameters}

| नाम | प्रकार | विवरण |
|------|------|-------------|
| $scheme | `string` | प्रमाणीकरण रीडायरेक्ट स्कीम। डिफ़ॉल्ट रूप से खाली। |
