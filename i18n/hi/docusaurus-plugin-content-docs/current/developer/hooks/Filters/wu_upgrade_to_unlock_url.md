# फ़िल्टर: wu_upgrade_to_unlock_url

डेवलपर्स को अपग्रेड करने के लिए अनलॉक URL बदलने की अनुमति देता है।

## पैरामीटर्स {#parameters}

| नाम | प्रकार | विवरण |
|------|------|-------------|
| $url | `string` | अपग्रेड URL। |
| $args | `array` | वह मॉड्यूल और सीमा (limit) का प्रकार जिसे अपग्रेड करने की आवश्यकता है। |

### स्रोत {#source}

[`inc/functions/limitations.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/limitations.php#L177) में लाइन 177 पर परिभाषित किया गया है।
