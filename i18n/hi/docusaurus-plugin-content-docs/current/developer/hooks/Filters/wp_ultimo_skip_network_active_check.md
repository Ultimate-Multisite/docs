# फ़िल्टर: wp_ultimo_skip_network_active_check {#filter-wpultimoskipnetworkactivecheck}

यह डेवलपर्स को इस जाँच को बायपास (short-circuit) करने की अनुमति देता है।

यह तब उपयोगी होता है जब आप composer-आधारित और अन्य कस्टम सेटअप का उपयोग कर रहे होते हैं, जैसे कि Bedrock, उदाहरण के लिए, जहाँ mu-plugins के रूप में प्लगइन्स का उपयोग करना सामान्य होता है।

### कब से {#since}

- 2.0.0
### स्रोत {#source}

[`inc/class-requirements.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-requirements.php#L256) में लाइन 256 पर परिभाषित किया गया है।
