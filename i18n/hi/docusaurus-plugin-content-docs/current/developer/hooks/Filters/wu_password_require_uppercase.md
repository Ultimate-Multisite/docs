# फ़िल्टर: wu_password_require_uppercase {#filter-wupasswordrequireuppercase}

यह फ़िल्टर निर्धारित करता है कि पासवर्ड में अपरकेस अक्षरों की आवश्यकता है या नहीं।

## पैरामीटर्स {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | क्या अपरकेस अक्षरों की आवश्यकता है। जब नियम लागू होते हैं तो डिफ़ॉल्ट रूप से सत्य (true) होता है। |
| $defender_active | `bool` | क्या Defender Pro Strong Password सक्रिय है। |
