# फ़िल्टर: wu_documentation_get_link

प्लगइन डेवलपर्स को लिंक को फ़िल्टर करने की अनुमति देता है।

यह स्पष्ट नहीं है कि यह कितना उपयोगी हो सकता है, लेकिन इसे रखना कोई नुकसान नहीं है।

## पैरामीटर्स

| नाम | प्रकार | विवरण |
|------|------|-------------|
| $link | `string` | पंजीकृत लिंक (The link registered) |
| $slug | `string` | लिंक प्राप्त करने के लिए उपयोग किया जाने वाला स्लग (The slug used to retrieve the link) |
| $default_link | `string` | पंजीकृत डिफ़ॉल्ट लिंक (The default link registered) |
