# Filter: wu_plan_product_types

उन उत्पाद प्रकारों को फ़िल्टर करें जिन्हें प्लान (plans) के रूप में माना जाना चाहिए।

यह फ़िल्टर ऐडऑन को अतिरिक्त उत्पाद प्रकार पंजीकृत करने की अनुमति देता है जिन्हें सत्यापन (validation) और अलगाव (segregation) में प्लान के रूप में पहचाना जाना चाहिए।

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | उन उत्पाद प्रकारों का ऐरे जिन्हें प्लान के रूप में माना जाना है। |
