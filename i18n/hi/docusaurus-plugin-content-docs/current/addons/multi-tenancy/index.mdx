# मल्टी-टेनेंसी (Multi-Tenancy)

बेहतर प्रदर्शन और सुरक्षा के लिए सबसाइट्स को अलग-अलग सर्वर और डेटाबेस पर व्यवस्थित करना।

## आवश्यकताएँ (Requirements)

- WordPress 5.3 या इससे उच्च संस्करण
- PHP 7.4 या इससे उच्च संस्करण
- मल्टीसाइट अल्टीमेट प्लगइन (सक्रिय)

## इंस्टॉलेशन (Installation)

1. एडऑन फ़ाइलों को अपने `/wp-content/plugins/` डायरेक्टरी में अपलोड करें।
2. वर्डप्रेस में 'Plugins' मेनू के माध्यम से प्लगइन को सक्रिय करें।
3. नेटवर्क एडमिन में एडऑन सेटिंग्स को कॉन्फ़िगर करें।

## डेवलपमेंट (Development)

### सेटअप (Setup)

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### टेस्टिंग (Testing)

इस एडऑन में एक व्यापक टेस्टिंग फ्रेमवर्क शामिल है:

- **यूनिट टेस्ट (Unit Tests)**: वर्डप्रेस टेस्ट सूट के साथ PHPUnit-आधारित टेस्ट।
- **बेस टेस्ट क्लास (Base Test Class)**: सामान्य टेस्ट परिदृश्यों के लिए सहायक तरीके (helper methods) प्रदान करता है।
- **टेस्ट कवरेज (Test Coverage)**: कोड कवरेज को ट्रैक करने के लिए कॉन्फ़िगर किया गया है।
- **मल्टीसाइट टेस्टिंग (Multisite Testing)**: टेस्ट मल्टीसाइट वातावरण में चलते हैं।

### कोड मानक (Code Standards)

- **वर्डप्रेस कोडिंग स्टैंडर्ड्स (WordPress Coding Standards)**: PHPCS के माध्यम से लागू।
- **PHP 7.4+ संगतता (PHP 7.4+ Compatibility)**: आधुनिक PHP सुविधाओं का समर्थन।
- **स्टैटिक एनालिसिस (Static Analysis)**: टाइप चेकिंग के लिए PHPStan इंटीग्रेशन।
- **कोड आधुनिकीकरण (Code Modernization)**: PHP अपग्रेड के लिए Rector नियम।

### बिल्ड सिस्टम (Build System)

- **एसेट प्रोसेसिंग (Asset Processing)**: CSS/JS फ़ाइलों के लिए मिनिफिकेशन।
- **ट्रांसलेशन (Translation)**: POT फ़ाइल जनरेशन।
- **पैकेजिंग (Packaging)**: वितरण के लिए आर्काइव बनाना।
- **डेवलपमेंट मोड (Development Mode)**: डीबगिंग के लिए अनमिनिफाइड एसेट्स।

## कॉन्फ़िगरेशन (Configuration)

यह एडऑन नेटवर्क एडमिन इंटरफ़ेस के माध्यम से एक्सेस किए जा सकने वाले विभिन्न कॉन्फ़िगरेशन विकल्प प्रदान करता है।

## हुक्स और फिल्टर्स (Hooks and Filters)

### एक्शन (Actions)

- `ultimate-multisite-addon-template_init` - एडऑन के इनिशियलाइज़ होने के बाद ट्रिगर होता है।
- `ultimate-multisite-addon-template_loaded` - सभी एडऑन फ़ाइलें लोड होने के बाद ट्रिगर होता है।

### फिल्टर्स (Filters)

- `ultimate-multisite-addon-template_settings` - एडऑन सेटिंग्स को संशोधित करता है।
- `ultimate-multisite-addon-template_enabled` - एडऑन के सक्षम/अक्षम स्थिति को ओवरराइड करता है।
