# फ़िल्टर: wu_captcha_whitelisted

यह फ़िल्टर यह तय करता है कि वर्तमान उपयोगकर्ता/अनुरोध को श्वेतसूची (whitelisted) में शामिल किया जाना चाहिए या नहीं।

## पैरामीटर्स {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $whitelisted | `bool` | क्या कैप्चा सत्यापन को छोड़ना है। |
