# Filter: wu_cap_captcha_config

कैप कैप्चा कॉन्फ़िगरेशन को फ़िल्टर करें।

यह डेवलपर्स को कैप कैप्चा सेटिंग्स को कस्टमाइज़ करने की अनुमति देता है।

## पैरामीटर्स

| Name | Type | Description |
|------|------|-------------|
| $config | `array` | कैप कॉन्फ़िगरेशन ऐरे। |
| $level | `string` | सुरक्षा स्तर (fast, medium, max)। |
