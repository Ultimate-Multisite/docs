# हॉस्पिटैलिटी मेनू

**हॉस्पिटैलिटी मेनू** फीचर Theme Builder को आपकी WordPress साइट में सीधे संरचित (structured) खाद्य और पेय मेनू पेज बनाने और एम्बेड करने की अनुमति देता है।

## अवलोकन (Overview)

Theme Builder अब हॉस्पिटैलिटी व्यवसायों, जिनमें रेस्टोरेंट, कैफे, बार और केटरिंग सेवाएं शामिल हैं, के लिए पेशेवर, संरचित मेनू पेज बना सकता है। ये मेनू आपकी साइट के डिज़ाइन में पूरी तरह से एकीकृत (integrated) होते हैं और इन्हें आसानी से अपडेट और मैनेज किया जा सकता है।

## समर्थित हॉस्पिटैलिटी प्रकार (Supported Hospitality Types)

- **रेस्टोरेंट** — फुल-सर्विस डाइनिंग मेनू
- **कैफे** — कॉफ़ी और हल्के भोजन के मेनू
- **बार और लाउंज** — पेय और ऐपेटाइज़र मेनू
- **बेकरी** — पेस्ट्री और ब्रेड मेनू
- **केटरिंग सेवाएं** — इवेंट मेनू विकल्प
- **फूड ट्रक** — मोबाइल फूड सर्विस मेनू
- **ब्रूअरी और वाइनरी** — विवरण के साथ पेय मेनू

## मेनू संरचना (Menu Structure)

### मेनू श्रेणियां (Menu Categories)

मेनू को श्रेणियों में व्यवस्थित किया जाता है:

- **ऐपेटाइज़र** — स्टार्टर्स और छोटे प्लेट्स
- **एंट्रीज़** — मुख्य व्यंजन
- **साइड्स** — साथ में परोसे जाने वाले और सब्जियां
- **डेज़र्ट** — मीठे व्यंजन
- **पेय (Beverages)** — ड्रिंक्स (शराब वाले और गैर-शराब वाले)
- **स्पेशल्स** — दैनिक या मौसमी विशेष

### मेनू आइटम फॉर्मेट (Menu Item Format)

हर मेनू आइटम में शामिल होता है:

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### आइटम फ़ील्ड (Item Fields)

| Field | Type | Description |
|-------|------|-------------|
| `name` | string | व्यंजन या पेय का नाम |
| `description` | string | आइटम का विस्तृत विवरण |
| `price` | string | कीमत (मुद्रा के साथ फॉर्मेट की गई) |
| `dietary_info` | array | आहार संबंधी विशेषताएँ (Vegan, Gluten-Free, आदि) |
| `allergens` | array | सामान्य एलर्जी कारक (Nuts, Shellfish, आदि) |
| `availability` | string | कब उपलब्ध है (Daily, Seasonal, आदि) |

## हॉस्पिटैलिटी मेनू बनाना (Creating Hospitality Menus)

### स्टेप 1: मेनू जानकारी प्रदान करें (Provide Menu Information)

Theme Builder को अपने मेनू के बारे में बताएं:

```
Create a restaurant menu for my Italian restaurant with appetizers,
pasta dishes, main courses, and desserts. Include prices and descriptions.
```

### स्टेप 2: Theme Builder मेनू बनाता है (Theme Builder Generates Menu)

Theme Builder:

1. एक संरचित मेनू पेज बनाता है
2. इसे आपकी साइट के थीम से मेल खाने के लिए डिज़ाइन करता है
3. आइटम को श्रेणियों में व्यवस्थित करता है
4. कीमतों और विवरणों को फॉर्मेट करता है
5. आहार और एलर्जी की जानकारी जोड़ता है

### स्टेप 3: समीक्षा करें और कस्टमाइज़ करें (Review and Customize)

आप ये कर सकते हैं:

1. मेनू आइटम और कीमतें एडिट कर सकते हैं
2. श्रेणियां जोड़ या हटा सकते हैं
3. श्रेणियों के भीतर आइटम का क्रम बदल सकते हैं
4. विवरण और आहार संबंधी जानकारी अपडेट कर सकते हैं
5. स्टाइलिंग और लेआउट को एडजस्ट कर सकते हैं

## मेनू डिस्प्ले विकल्प (Menu Display Options)

### फुल मेनू पेज (Full Menu Page)

एक समर्पित पेज जो आपका पूरा मेनू प्रदर्शित करता है:

- श्रेणी के अनुसार व्यवस्थित
- खोजा और फ़िल्टर किया जा सकने वाला
- प्रिंट-अनुकूल लेआउट
- मोबाइल-रिस्पॉन्सिव डिज़ाइन

### मेनू विजेट (Menu Widget)

अन्य पेजों पर मेनू सेक्शन एम्बेड करें:

- होमपेज पर मुख्य आइटम
- साइडबार पर दैनिक विशेष
- बार पेज पर पेय मेनू
- फुटर पर डेज़र्ट शोकेस

### मेनू PDF (Menu PDF)

डाउनलोड करने योग्य PDF मेनू जनरेट करें:

- पेशेवर फॉर्मेटिंग
- प्रिंट-तैयार गुणवत्ता
- छवियों और विवरणों को शामिल करता है
- साझा करना और ईमेल करना आसान

## आहार और एलर्जी की जानकारी (Dietary and Allergen Information)

### आहार विशेषताएँ (Dietary Attributes)

आइटम को आहार संबंधी जानकारी से चिह्नित करें:

- **Vegan** — कोई पशु उत्पाद नहीं
- **Vegetarian** — कोई मांस नहीं
- **Gluten-Free** — सीलिएक रोग के लिए सुरक्षित
- **Dairy-Free** — कोई दूध उत्पाद नहीं
- **Nut-Free** — कोई पेड़ के मेवे या मूंगफली नहीं
- **Low-Carb** — कम कार्बोहाइड्रेट
- **High-Protein** — प्रोटीन से भरपूर

### एलर्जी चेतावनी (Allergen Warnings)

सामान्य एलर्जी कारकों को शामिल करें:

- **Nuts** — पेड़ के मेवे और मूंगफली
- **Shellfish** — क्रस्टेशियंस और मोलस्क
- **Fish** — सभी मछली प्रजातियाँ
- **Dairy** — दूध और दूध उत्पाद
- **Eggs** — चिकन अंडे
- **Soy** — सोया उत्पाद
- **Gluten** — गेहूं और ग्लूटेन युक्त अनाज
- **Sesame** — तिल के बीज और तेल

## मेनू प्रबंधन (Menu Management)

### कीमतें अपडेट करना (Updating Prices)

मेनू की कीमतों को आसानी से अपडेट करें:

1. मेनू पेज पर जाएं
2. "Edit Menu" पर क्लिक करें
3. आइटमों के लिए कीमतें अपडेट करें
4. बदलाव सेव करें
5. बदलाव तुरंत आपकी साइट पर दिखाई देंगे

### मौसमी आइटम जोड़ना (Adding Seasonal Items)

मौसमी मेनू विविधताएँ बनाएं:

1. एक नया मेनू वर्जन बनाएं
2. मौसमी आइटम जोड़ें
3. आइटम को "Seasonal" के रूप में चिह्नित करें
4. उपलब्धता की तारीखें निर्धारित करें
5. सीजन के दौरान स्वचालित रूप से प्रदर्शित करें

### स्पेशल्स मैनेज करना (Managing Specials)

दैनिक या साप्ताहिक विशेष प्रदर्शित करें:

1. एक "Specials" श्रेणी बनाएं
2. उपलब्धता की तारीखों के साथ आइटम जोड़ें
3. होमपेज पर स्पेशल को हाइलाइट करें
4. दैनिक या साप्ताहिक अपडेट करें
5. पुराने स्पेशल को आर्काइव करें

## Theme Builder के साथ इंटीग्रेशन (Integration with Theme Builder)

जब आप हॉस्पिटैलिटी साइट्स के लिए Theme Builder का उपयोग करते हैं:

1. **Automatic menu detection** — यह पहचानता है कि आप खाद्य/पेय व्यवसाय हैं
2. **Menu page creation** — पेशेवर मेनू पेज जनरेट करता है
3. **Design matching** — मेनू आपकी साइट के थीम से मेल खाते हैं
4. **Mobile optimization** — मेनू फोन पर खूबसूरती से प्रदर्शित होते हैं
5. **SEO optimization** — मेनू सर्च इंजन के अनुकूल होते हैं

## सर्वोत्तम अभ्यास (Best Practices)

### मेनू डिज़ाइन (Menu Design)

- **स्पष्ट संगठन** — तार्किक श्रेणी संरचना
- **पठनीय विवरण** — भूख बढ़ाने वाले और जानकारीपूर्ण
- **संगत मूल्य निर्धारण** — स्पष्ट मुद्रा और फॉर्मेटिंग
- **पेशेवर तस्वीरें** — उच्च गुणवत्ता वाली खाद्य छवियां
- **व्हाइटस्पेस** — पेज को बहुत अधिक न भरें

### सामग्री (Content)

- **सटीक विवरण** — आइटम का सटीक वर्णन करें
- **स्पेशल्स को हाइलाइट करें** — स्पेशल को अलग दिखाएं
- **एलर्जी कारक शामिल करें** — हमेशा एलर्जी कारकों को सूचीबद्ध करें
- **नियमित रूप से अपडेट करें** — कीमतों और आइटम को वर्तमान रखें
- **भूख बढ़ाने वाली भाषा का उपयोग करें** — आइटम को स्वादिष्ट बनाएं

### एक्सेसिबिलिटी (Accessibility)

- **पठनीय फ़ॉन्ट** — स्पष्ट, पढ़ने योग्य टाइपफेस का उपयोग करें
- **पर्याप्त कंट्रास्ट** — सुनिश्चित करें कि टेक्स्ट पढ़ा जा सके
- **आहार लेबल** — आहार विकल्पों को स्पष्ट रूप से चिह्नित करें
- **एलर्जी चेतावनी** — एलर्जी कारकों को प्रमुखता से प्रदर्शित करें
- **मोबाइल-फ्रेंडली** — सभी डिवाइस पर टेस्ट करें
