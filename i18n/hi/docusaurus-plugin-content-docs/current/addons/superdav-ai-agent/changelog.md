# परिवर्तन सूची

## 1.18.0 — 2026-06-29 को जारी {#1180--released-on-2026-06-29}

### नया {#new}

- **Google Calendar उपकरण** — समय-सारिणी के अनुरूप स्वचालन के लिए कॉन्फ़िगर किए गए कैलेंडर और कार्यक्रम पढ़ें
- **संपर्क मैपिंग और सहभागी सहायक** — कैलेंडर प्रतिभागियों का मिलान साइट उपयोगकर्ताओं और संपर्कों से करें
- **मानवीय अनुमोदन द्वार और अनुस्मारक रिकॉर्ड** — समीक्षा के लिए स्वचालन रोकें और दोहराई गई सूचनाओं से बचें
- **TextBee SMS प्रदाता** — TextBee के माध्यम से कॉन्फ़िगर की गई टेक्स्ट-संदेश सूचनाएँ भेजें
- **उन्नत सहायक पैकेज** — विश्वसनीय-डेवलपर फ़ाइल सिस्टम, डेटाबेस, WP-CLI, REST डिस्पैचर, प्लगइन बिल्डर, git स्नैपशॉट, उपयोगकर्ता-प्रबंधन, और बेंचमार्क उपकरण जोड़ें, जो WordPress.org बिल्ड से अलग वितरित किए जाते हैं

### बेहतर किया गया {#improved}

- **प्रबंधित Superdav सेवा सेटअप** — समर्थित साइटों के लिए होस्ट किए गए सेवा endpoint और स्वचालित कनेक्शन प्रावधान जोड़ें
- **रिलीज़ पैकेजिंग** — अलग-अलग कोर और उन्नत ZIP बनाएँ, दोनों को GitHub पर प्रकाशित करें, और WordPress.org को केवल कोर पैकेज भेजें

### ठीक किया गया {#fixed}

- **AI अनुरोध विश्वसनीयता** — मॉडल चयन, अनुरोध टाइमआउट, डिफ़ॉल्ट सेटअप मॉडल, तर्क-पाठ हैंडलिंग, और अमान्य टूल कॉल के लिए पुनः प्रयास मार्गदर्शन में सुधार
- **कैलेंडर और अनुस्मारक सुदृढ़ीकरण** — Google Calendar टोकन और अनुस्मारक डीडुप्लिकेशन को मजबूत करें
- **ऑनबोर्डिंग और अनुमोदन फिर से शुरू करना** — फ्रंटएंड ऑनबोर्डिंग लॉन्च और पुष्टि की गई क्षमता फिर से शुरू करने की समस्या ठीक करें
- **WordPress.org पैकेजिंग समीक्षा समस्याएँ** — कोर रिलीज़ के लिए पैकेजिंग समीक्षा प्रतिक्रिया का समाधान करें

## 1.16.0 — 2026-05-20 को जारी {#1160--released-on-2026-05-20}

### नया {#new-1}

- **Logo SVG जनरेट करने की क्षमता** — Theme Builder अब namespace-सुरक्षित सैनिटाइजेशन के साथ कस्टम logo SVG बना और एम्बेड कर सकता है
- **डिस्कवरी इंटरव्यू में फ़ोटो अपलोड** — Theme Builder डिस्कवरी इंटरव्यू में अब अधिक समृद्ध डिज़ाइन संदर्भ के लिए फ़ोटो अपलोड चरण शामिल है
- **पैलेट कंट्रास्ट सत्यापित करने की क्षमता** — किसी theme पर लागू करने से पहले WCAG अनुपालन के लिए रंग जोड़ों की जाँच करें
- **हॉस्पिटैलिटी मेनू** — Theme Builder अब हॉस्पिटैलिटी व्यवसायों के लिए संरचित भोजन और पेय मेनू पृष्ठ बना सकता है
- **डेस्कटॉप और मोबाइल पूर्वावलोकन रेंडरिंग** — डिज़ाइन-दिशा चयन के दौरान अपने डिज़ाइन का डेस्कटॉप और मोबाइल उपकरणों पर पूर्वावलोकन करें
- **नेविगेशन लेबल पैरामीटर** — Create Menu क्षमता अब पृष्ठ शीर्षक से अलग एक विशिष्ट `navigation_label` का समर्थन करती है
- **Tier 1 टूल उपलब्धता** — sd-ai-agent/site-scrape अब Theme Builder में डिफ़ॉल्ट रूप से उपलब्ध Tier 1 टूल है

### ठीक किया गया {#fixed-1}

- **AI Client कैश** — अब क्रॉस-अनुरोध स्थायित्व के लिए transients द्वारा समर्थित है, जिससे लंबे समय तक चलने वाले agent कार्यों में डेटा हानि रुकती है
- **प्लगइन पंक्ति कार्रवाई लिंक** — स्पष्टता के लिए सुधारे और फिर से नामित किए गए

## 1.10.0 — 2026-05-05 को जारी {#1100--released-on-2026-05-05}

### नया {#new-2}

- **Tavily इंटरनेट खोज** — Brave Search के साथ अधिक समृद्ध इंटरनेट खोज परिणामों के लिए Tavily को खोज प्रदाता के रूप में जोड़ें
- **Theme-जागरूक अंतर्निहित कौशल** — Block Themes, Classic Themes, Kadence Blocks, और Kadence Theme कौशल मार्गदर्शिकाएँ अब प्लगइन के साथ आती हैं
- **साइट बिल्डर संपर्क फ़ॉर्म क्षमता** — चैट इंटरफ़ेस से सीधे किसी भी पृष्ठ पर संपर्क फ़ॉर्म जोड़ें

### बेहतर किया गया {#improved-1}

- **WooCommerce एकीकरण पुनर्गठित** — अब बेहतर विश्वसनीयता और संगतता के लिए मूल WooCommerce API का उपयोग करता है
- **प्रदाता सूची अपने-आप ताज़ा होती है** — जब कोई भी प्लगइन सक्रिय या निष्क्रिय किया जाता है

### ठीक किया गया {#fixed-2}

- **navigate-to क्षमता** — कुछ व्यवस्थापक पृष्ठों पर अनंत रीलोड लूप ठीक किया गया
- **list-posts क्षमता** — अब श्रेणी और टैग नामों को सही ढंग से slugs में बदलती है
- **WP-CLI कमांड** — पिछले पुनर्गठन के बाद गायब namespace aliases बहाल किए गए
- **कार्यक्रम स्वचालन** — उन साइटों को सहजता से संभालता है जहाँ स्वचालन तालिकाएँ अभी तक नहीं बनी हैं
- **memory-save क्षमता** — अब सिस्टम निर्देश बिल्डर में सही namespace prefix का उपयोग करती है
- **Scalar टूल परिणाम** — अब AI को लौटाए जाने से पहले सही ढंग से रैप किए जाते हैं
- **उपयोग आँकड़े** — अब पुराने संस्करणों से अपग्रेड पर legacy क्षमता key प्रारूप को सही ढंग से संभालते हैं
