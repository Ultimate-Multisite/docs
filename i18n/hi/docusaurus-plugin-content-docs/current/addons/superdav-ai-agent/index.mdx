# Superdav AI Agent {#superdav-ai-agent}

Superdav AI Agent एक उन्नत WordPress AI सहायक है, जो Gratis AI Agent को theme-सचेत कौशल, बेहतर इंटरनेट खोज क्षमताओं, साइट बिल्डर सुविधाओं और वैकल्पिक ऑटोमेशन इंटीग्रेशन के साथ विस्तारित करता है। संस्करण 1.18.0 में Google Calendar पढ़ने के टूल, संपर्क और सहभागी मैपिंग, मानवीय अनुमोदन गेट, रिमाइंडर डीडुप्लिकेशन, TextBee SMS सूचनाएँ, प्रबंधित Superdav सेवा प्रोविजनिंग, और भरोसेमंद डेवलपर वर्कफ़्लो के लिए अलग से वितरित Advanced companion पैकेज जोड़ा गया है।

## मुख्य सुविधाएँ {#key-features}

- **Theme-सचेत कौशल** — आपकी सक्रिय WordPress theme और इंस्टॉल किए गए प्लगइन के आधार पर मार्गदर्शन और क्षमताओं को अपने-आप अनुकूलित करता है
- **कई इंटरनेट खोज प्रदाता** — अधिक समृद्ध खोज परिणामों और उन्नत शोध क्षमताओं के लिए Brave Search और Tavily में से चुनें
- **साइट बिल्डर संपर्क फ़ॉर्म** — अपने इंस्टॉल किए गए फ़ॉर्म प्लगइन का उपयोग करके चैट इंटरफ़ेस से सीधे पूरी तरह काम करने वाले संपर्क फ़ॉर्म बनाएँ
- **बेहतर साइट बिल्डर** — बेहतर प्लगइन खोज और त्रुटि पुनर्प्राप्ति के साथ सुधरी हुई बहु-चरण साइट निर्माण योजनाएँ
- **अंतर्निहित theme कौशल** — Block Themes, Classic Themes, Kadence Blocks, और Kadence Theme के लिए विशेष मार्गदर्शन
- **बेहतर WooCommerce इंटीग्रेशन** — अधिक विश्वसनीयता और अनुकूलता के लिए मूल WooCommerce APIs का उपयोग करने हेतु पुनर्गठित
- **स्वचालित प्रदाता पहचान** — प्लगइन और theme सक्रिय या निष्क्रिय होने पर उपलब्ध प्लगइन और theme का अपने-आप पता लगाता और उन्हें रीफ़्रेश करता है
- **कस्टम लोगो निर्माण** — स्वचालित namespace-सुरक्षित सैनिटाइज़ेशन के साथ कस्टम SVG लोगो बनाएँ और एम्बेड करें
- **पैलेट कंट्रास्ट सत्यापन** — अपनी theme पर लागू करने से पहले WCAG पहुँच-योग्यता अनुपालन के लिए रंग जोड़ों की जाँच करें
- **फ़ोटो-संवर्धित डिस्कवरी** — डिज़ाइन दिशा को प्रभावित करने के लिए Theme Builder डिस्कवरी इंटरव्यू के दौरान फ़ोटो अपलोड करें
- **हॉस्पिटैलिटी मेनू** — रेस्तरां, कैफ़े और हॉस्पिटैलिटी व्यवसायों के लिए संरचित भोजन और पेय मेनू पेज बनाएँ
- **डेस्कटॉप और मोबाइल पूर्वावलोकन** — डिज़ाइन-दिशा चयन के दौरान अपने डिज़ाइन का डेस्कटॉप और मोबाइल उपकरणों पर पूर्वावलोकन करें
- **टियर 1 साइट स्क्रैपिंग** — प्रतिस्पर्धी और प्रेरणा वेबसाइटों का विश्लेषण करने के लिए sd-ai-agent/site-scrape अब डिफ़ॉल्ट रूप से उपलब्ध है
- **Google Calendar पढ़ने के टूल** — कॉन्फ़िगर किए गए कैलेंडर और इवेंट पढ़ें, ताकि ऑटोमेशन आने वाले शेड्यूल संदर्भ को समझ सकें
- **संपर्क और सहभागी मैपिंग** — फ़ॉलो-अप कार्य बनाने से पहले कैलेंडर प्रतिभागियों को ज्ञात साइट उपयोगकर्ताओं और संपर्कों से मिलाएँ
- **मानवीय अनुमोदन गेट** — संदेश भेजने, काम फिर से शुरू करने या बदलाव करने से पहले समीक्षा के लिए संवेदनशील ऑटोमेशन रोकें
- **रिमाइंडर रिकॉर्ड और डीडुप्लिकेशन** — पहले से भेजे गए रिमाइंडर ट्रैक करें, ताकि दोहराए गए ऑटोमेशन रन उसी प्राप्तकर्ता को दोबारा सूचना न भेजें
- **TextBee SMS सूचनाएँ** — SMS प्रदाता सक्षम होने पर TextBee के माध्यम से कॉन्फ़िगर की गई टेक्स्ट-संदेश सूचनाएँ भेजें
- **प्रबंधित Superdav सेवा प्रोविजनिंग** — मैन्युअल endpoint कॉन्फ़िगरेशन के बजाय स्वचालित सेटअप के साथ समर्थित साइटों को होस्ट किए गए Superdav endpoints से जोड़ें
- **Advanced companion पैकेज** — फ़ाइलसिस्टम, डेटाबेस, WP-CLI, REST डिस्पैचर, प्लगइन बिल्डर, git स्नैपशॉट, उपयोगकर्ता-प्रबंधन, और बेंचमार्क वर्कफ़्लो के लिए अलग से वितरित भरोसेमंद-डेवलपर टूल

## आवश्यकताएँ {#requirements}

- WordPress 6.0 या उच्चतर
- PHP 8.1 या उच्चतर
- WordPress Multisite (नेटवर्क-सक्रिय) या एकल-साइट इंस्टॉलेशन
- एक सक्रिय AI प्रदाता कनेक्टर प्लगइन (Anthropic, OpenAI, या संगत)
- Gratis AI Agent v1.9.0 या उच्चतर (Superdav AI Agent, Gratis AI Agent को विस्तारित करता है)

## इंस्टॉलेशन {#installation}

1. प्लगइन फ़ाइलों को `/wp-content/plugins/superdav-ai-agent` पर अपलोड करें
2. WordPress में **Plugins** स्क्रीन के माध्यम से प्लगइन सक्रिय करें
3. सुनिश्चित करें कि Gratis AI Agent भी सक्रिय है
4. **Gratis AI Agent → Settings** पर जाएँ और आवश्यकता अनुसार अपना इंटरनेट खोज प्रदाता, प्रबंधित Superdav सेवा कनेक्शन, Google Calendar, या TextBee क्रेडेंशियल कॉन्फ़िगर करें

WordPress.org पैकेज में मुख्य Superdav AI Agent सुविधाएँ शामिल हैं। Advanced companion पैकेज उन परिवेशों के लिए अलग से बनाया और वितरित किया जाता है जहाँ भरोसेमंद डेवलपर टूल स्पष्ट रूप से इंस्टॉल और अधिकृत किए जाते हैं।

## त्वरित शुरुआत {#quick-start}

सक्रिय होने के बाद, Superdav AI Agent आपके Gratis AI Agent इंस्टॉलेशन को theme-सचेत कौशल और अतिरिक्त क्षमताओं के साथ अपने-आप बेहतर बनाता है। WordPress एडमिन साइडबार में **Gratis AI Agent** पैनल खोलें और नई सुविधाओं का उपयोग शुरू करें।

**उदाहरण निर्देश:**

```
संपर्क पेज में एक संपर्क फ़ॉर्म जोड़ें।
```

```
नवीनतम WordPress सुरक्षा सर्वोत्तम तरीकों के लिए वेब पर खोजें।
```

```
सक्रिय theme के ब्लॉक पैटर्न का उपयोग करके एक हीरो सेक्शन बनाएँ।
```

```
कल के कॉन्फ़िगर किए गए Google Calendar इवेंट की समीक्षा करें और सहभागियों के लिए रिमाइंडर ड्राफ़्ट करें।
```

```
मेरे संदेश को अनुमोदित करने के बाद एक SMS रिमाइंडर भेजें।
```

## Theme-सचेत कौशल {#theme-aware-skills}

Superdav AI Agent आपकी सक्रिय theme और इंस्टॉल किए गए प्लगइन का अपने-आप पता लगाता है, और सहायक के संदर्भ में उपयुक्त theme-सचेत कौशल लोड करता है। उपलब्ध कौशलों में शामिल हैं:

| कौशल | इन पर लागू |
|---|---|
| **Block Themes** | WordPress ब्लॉक-आधारित (Full Site Editing) आर्किटेक्चर का उपयोग करने वाली theme |
| **Classic Themes** | PHP टेम्पलेट और क्लासिक संपादक का उपयोग करने वाली पारंपरिक WordPress theme |
| **Kadence Blocks** | Kadence Blocks प्लगइन सक्रिय वाली साइटें |
| **Kadence Theme** | सक्रिय theme के रूप में Kadence theme वाली साइटें |

प्रत्येक कौशल पर विस्तृत मार्गदर्शन के लिए [Theme-सचेत कौशल](../../user-guide/configuration/theme-aware-skills) देखें।

## इंटरनेट खोज प्रदाता {#internet-search-providers}

Superdav AI Agent v1.10.0 इंटरनेट खोज क्वेरी के लिए Brave Search के विकल्प के रूप में **Tavily** जोड़ता है। दोनों प्रदाता वास्तविक-समय खोज परिणाम देते हैं और उन्हें **Gratis AI Agent → Settings → Advanced** में कॉन्फ़िगर किया जा सकता है।

| प्रदाता | विशेषताएँ | सेटअप |
|---|---|---|
| **Brave Search** | तेज़, गोपनीयता-केंद्रित खोज परिणाम | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | अधिक समृद्ध खोज परिणाम, उन्नत शोध क्षमताएँ | [Tavily API](https://tavily.com/) |

कॉन्फ़िगरेशन निर्देशों के लिए [इंटरनेट खोज](../../user-guide/configuration/internet-search) देखें।

## कैलेंडर, स्वीकृति, और सूचना कार्यप्रवाह {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 सामग्री और वेबसाइट-निर्माण कार्यों से आगे स्वचालन कार्यप्रवाहों का विस्तार करता है:

- **Google Calendar उपकरण** एजेंट को कॉन्फ़िगर किए गए कैलेंडर और इवेंट विवरण पढ़ने देते हैं, जब कैलेंडर क्रेडेंशियल उपलब्ध हों।
- **संपर्क और सहभागी सहायक** रिमाइंडर या अनुवर्ती कार्य तैयार करने से पहले इवेंट सहभागियों को WordPress उपयोगकर्ताओं या ज्ञात संपर्कों से मिलाते हैं।
- **मानवीय स्वीकृति गेट** किसी अधिकृत उपयोगकर्ता द्वारा प्रस्तावित कार्रवाई की समीक्षा होने तक स्वचालन को रोकने देते हैं।
- **रिमाइंडर रिकॉर्ड** स्वचालन दोबारा चलाए जाने या बार-बार शेड्यूल होने पर डुप्लिकेट कैलेंडर या सूचना रिमाइंडर रोकते हैं।
- **TextBee SMS समर्थन** TextBee प्रदाता कॉन्फ़िगर होने पर टेक्स्ट-संदेश सूचनाएँ सक्षम करता है।

इन उपकरणों का उपयोग शेड्यूल-सचेत सहायता, सहभागी अनुवर्ती कार्रवाई, और ऐसे रिमाइंडर कार्यप्रवाहों के लिए करें जिनमें संवेदनशील संदेश भेजने से पहले अभी भी मानवीय समीक्षा आवश्यक हो।

## वेबसाइट निर्माता संपर्क फ़ॉर्म {#site-builder-contact-forms}

वेबसाइट निर्माता एजेंट अब चैट इंटरफ़ेस से सीधे संपर्क फ़ॉर्म बना सकता है। बस वेबसाइट निर्माता से संपर्क फ़ॉर्म बनाने को कहें, और यह आपके इंस्टॉल किए गए फ़ॉर्म plugin (Contact Form 7, WPForms, Fluent Forms, या Gravity Forms) का स्वतः पता लगाएगा और उपयोग के लिए तैयार फ़ॉर्म बनाएगा।

विस्तृत मार्गदर्शन के लिए [वेबसाइट निर्माता संपर्क फ़ॉर्म](../../user-guide/administration/site-builder-contact-forms) देखें।

## Theme Builder विशेषताएँ {#theme-builder-features}

Superdav AI Agent v1.16.0 में Theme Builder शक्तिशाली नई क्षमताएँ शामिल करता है:

### फ़ोटो अपलोड के साथ खोज साक्षात्कार {#discovery-interview-with-photo-upload}

Theme Builder खोज साक्षात्कार में अब एक फ़ोटो अपलोड चरण शामिल है, जहाँ आप दृश्य प्रेरणा दे सकते हैं। अपने ब्रांड, प्रतिस्पर्धी वेबसाइटों, या डिज़ाइन प्रेरणा की फ़ोटो अपलोड करें ताकि Theme Builder आपकी डिज़ाइन दिशा समझ सके।

विस्तृत मार्गदर्शन के लिए [खोज साक्षात्कार](./theme-builder/discovery-interview.md) देखें।

### डेस्कटॉप और मोबाइल पूर्वावलोकन के साथ डिज़ाइन दिशा {#design-direction-with-desktop-and-mobile-previews}

Theme Builder द्वारा आपकी पूरी theme बनाने से पहले डेस्कटॉप और मोबाइल, दोनों उपकरणों पर अपने डिज़ाइन की समीक्षा करें। कई प्रतिक्रियाशील ब्रेकपॉइंट पर रीयल-टाइम पूर्वावलोकन देखें।

विस्तृत मार्गदर्शन के लिए [डिज़ाइन दिशा](./theme-builder/design-direction.md) देखें।

### आतिथ्य मेनू {#hospitality-menus}

Theme Builder अब रेस्तरां, कैफ़े, बार, और अन्य आतिथ्य व्यवसायों के लिए संरचित भोजन और पेय मेनू पृष्ठ बना सकता है। मेनू में मूल्य, विवरण, आहार संबंधी जानकारी, और एलर्जन चेतावनियाँ शामिल होती हैं।

विस्तृत मार्गदर्शन के लिए [आतिथ्य मेनू](./theme-builder/hospitality-menus.md) देखें।

### नई क्षमताएँ {#new-abilities}

- **[Logo SVG बनाएँ](./abilities/generate-logo-svg.md)** — स्वचालित सैनिटाइज़ेशन के साथ कस्टम SVG लोगो बनाएँ
- **[Palette Contrast सत्यापित करें](./abilities/validate-palette-contrast.md)** — WCAG अनुपालन के लिए रंग जोड़ों की जाँच करें
- **[मेनू बनाएँ](./abilities/create-menu.md)** — अलग-अलग नेविगेशन लेबल के साथ नेविगेशन मेनू बनाएँ

उपलब्ध उपकरणों और Tier 1 वेबसाइट-स्क्रैप उपकरण की जानकारी के लिए [उपकरण और कॉन्फ़िगरेशन](./theme-builder/tools.md) देखें।

## कॉन्फ़िगरेशन {#configuration}

सभी सेटिंग्स WordPress admin में **Gratis AI Agent → Settings** के अंतर्गत हैं। Superdav AI Agent निम्नलिखित कॉन्फ़िगरेशन विकल्प जोड़ता है:

| सेटिंग | विवरण |
|---|---|
| **इंटरनेट खोज प्रदाता** | Brave Search और Tavily में से चुनें (वैकल्पिक) |
| **Tavily API Key** | Tavily इंटरनेट खोज के लिए API key (यदि Tavily चुना गया है) |
| **Managed Superdav Service** | समर्थित वेबसाइटों को स्वचालित endpoint प्रावधान के साथ होस्ट की गई Superdav सेवा से जोड़ें |
| **Google Calendar क्रेडेंशियल** | कैलेंडर पहुँच कॉन्फ़िगर करें ताकि एजेंट कैलेंडर और इवेंट पढ़ सके |
| **TextBee SMS प्रदाता** | स्वीकृत SMS सूचना कार्यप्रवाहों के लिए TextBee क्रेडेंशियल कॉन्फ़िगर करें |

## उन्नत सहायक पैकेज {#advanced-companion-package}

Superdav AI Agent v1.18.0 विश्वसनीय डेवलपर कार्यप्रवाहों के लिए अलग से वितरित उन्नत सहायक पैकेज जोड़ता है। ये क्षमताएँ WordPress.org core package में शामिल नहीं हैं। सहायक पैकेज केवल उन वेबसाइटों पर इंस्टॉल करें जहाँ प्रशासक एजेंट को डेवलपर उपकरणों तक पहुँच देने के लिए स्पष्ट रूप से भरोसा करते हैं।

उन्नत उपकरणों में फ़ाइल सिस्टम संचालन, डेटाबेस पहुँच, WP-CLI, REST डिस्पैचर कार्यप्रवाह, plugin निर्माता उपयोगिताएँ, git स्नैपशॉट, उपयोगकर्ता-प्रबंधन सहायक, और बेंचमार्क उपकरण शामिल हैं। इन उपकरणों को विश्वसनीय प्रशासकों तक सीमित रखें और उत्पादन वेबसाइटों पर इन्हें सक्षम करने से पहले स्वीकृति-गेट कॉन्फ़िगरेशन की समीक्षा करें।
