# Superdav AI Agent

Superdav AI Agent एक उन्नत WordPress AI असिस्टेंट है जो Gratis AI Agent को थीम-अवेयर स्किल्स, बेहतर इंटरनेट सर्च क्षमताओं और बेहतर साइट बिल्डर फीचर्स के साथ बढ़ाता है। Version 1.16.0 में कस्टम लोगो जनरेशन, पैलेट कंट्रास्ट वैलिडेशन, डिस्कवरी इंटरव्यू में फोटो अपलोड, स्ट्रक्चर्ड हॉस्पिटैलिटी मेनू जनरेशन, और Theme Builder में डेस्कटॉप/मोबाइल प्रीव्यू रेंडरिंग जैसी नई सुविधाएँ जोड़ी गई हैं।

## मुख्य विशेषताएँ (Key Features)

- **थीम-अवेयर स्किल्स (Theme-aware skills)** — यह आपके सक्रिय WordPress थीम और इंस्टॉल किए गए प्लगइन्स के आधार पर मार्गदर्शन और क्षमताओं को स्वचालित रूप से अनुकूलित करता है।
- **मल्टीपल इंटरनेट सर्च प्रोवाइडर्स (Multiple internet search providers)** — समृद्ध सर्च परिणाम और उन्नत रिसर्च क्षमताओं के लिए Brave Search और Tavily में से चुन सकते हैं।
- **साइट बिल्डर कॉन्टैक्ट फॉर्म्स (Site builder contact forms)** — अपने इंस्टॉल किए गए फॉर्म प्लगइन का उपयोग करके चैट इंटरफ़ेस से सीधे पूरी तरह कार्यात्मक कॉन्टैक्ट फॉर्म बनाएँ।
- **बेहतर साइट बिल्डर (Enhanced site builder)** — बेहतर प्लगइन डिस्कवरी और एरर रिकवरी के साथ मल्टी-फेज साइट बिल्ड प्लान में सुधार।
- **बिल्ट-इन थीम स्किल्स (Built-in theme skills)** — Block Themes, Classic Themes, Kadence Blocks, और Kadence Theme के लिए विशेष मार्गदर्शन।
- **सुधारित WooCommerce इंटीग्रेशन (Improved WooCommerce integration)** — बेहतर विश्वसनीयता और संगतता के लिए native WooCommerce APIs का उपयोग करने हेतु रीडिज़ाइन किया गया।
- **स्वचालित प्रोवाइडर डिटेक्शन (Automatic provider detection)** — जब प्लगइन्स सक्रिय या निष्क्रिय किए जाते हैं तो उपलब्ध प्लगइन्स और थीम का स्वचालित रूप से पता लगाता और रिफ्रेश करता है।
- **कस्टम लोगो जनरेशन (Custom logo generation)** — स्वचालित नेमस्पेस-सेफ सैनिटाइजेशन के साथ कस्टम SVG लोगो जनरेट और एम्बेड करें।
- **पैलेट कंट्रास्ट वैलिडेशन (Palette contrast validation)** — अपने थीम पर लागू करने से पहले रंग जोड़े की WCAG एक्सेसिबिलिटी अनुपालन की जाँच करें।
- **फोटो-एन्हांस्ड डिस्कवरी (Photo-enhanced discovery)** — डिज़ाइन दिशा को प्रभावित करने के लिए Theme Builder डिस्कवरी इंटरव्यू के दौरान फोटो अपलोड करें।
- **हॉस्पिटैलिटी मेनू (Hospitality menus)** — रेस्टोरेंट, कैफे और हॉस्पिटैलिटी व्यवसायों के लिए संरचित खाद्य और पेय मेनू पेज जनरेट करें।
- **डेस्कटॉप और मोबाइल प्रीव्यू (Desktop and mobile previews)** — डिज़ाइन दिशा चुनने के दौरान डेस्कटॉप और मोबाइल उपकरणों पर अपने डिज़ाइन का पूर्वावलोकन करें।
- **टियर 1 साइट स्क्रैपिंग (Tier 1 site scraping)** — sd-ai-agent/site-scrape अब डिफ़ॉल्ट रूप से प्रतिस्पर्धी और प्रेरणादायक वेबसाइटों का विश्लेषण करने के लिए उपलब्ध है।

## आवश्यकताएँ (Requirements)

- WordPress 6.0 या इससे उच्च
- PHP 8.1 या इससे उच्च
- WordPress Multisite (network-activated) या सिंगल-साइट इंस्टॉलेशन
- एक सक्रिय AI प्रोवाइडर कनेक्टर प्लगइन (Anthropic, OpenAI, या संगत)
- Gratis AI Agent v1.9.0 या इससे उच्च (Superdav AI Agent, Gratis AI Agent का विस्तार है)

## इंस्टॉलेशन (Installation)

1. प्लगइन फ़ाइलों को `/wp-content/plugins/superdav-ai-agent` पर अपलोड करें।
2. WordPress में **Plugins** स्क्रीन के माध्यम से प्लगइन को सक्रिय करें।
3. सुनिश्चित करें कि Gratis AI Agent भी सक्रिय है।
4. **Gratis AI Agent → Settings** पर जाएँ और अपने इंटरनेट सर्च प्रोवाइडर को कॉन्फ़िगर करें (वैकल्पिक)।

## त्वरित शुरुआत (Quick Start)

सक्रिय होने के बाद, Superdav AI Agent स्वचालित रूप से थीम-अवेयर स्किल्स और अतिरिक्त क्षमताओं के साथ आपके Gratis AI Agent इंस्टॉलेशन को बेहतर बनाता है। WordPress एडमिन साइडबार में **Gratis AI Agent** पैनल खोलें और नई सुविधाओं का उपयोग करना शुरू करें।

**उदाहरण निर्देश:**

```
Add a contact form to the Contact page.
```

```
Search the web for the latest WordPress security best practices.
```

```
Create a hero section using the active theme's block patterns.
```

## थीम-अवेयर स्किल्स (Theme-Aware Skills)

Superdav AI Agent स्वचालित रूप से आपके सक्रिय थीम और इंस्टॉल किए गए प्लगइन्स का पता लगाता है, और असिस्टेंट के संदर्भ में उपयुक्त थीम-अवेयर स्किल लोड करता है। उपलब्ध स्किल्स में शामिल हैं:

| Skill | Applies To |
|---|---|
| **Block Themes** | वे थीम जो WordPress ब्लॉक-आधारित (Full Site Editing) आर्किटेक्चर का उपयोग करती हैं |
| **Classic Themes** | पारंपरिक WordPress थीम जो PHP टेम्पलेट और क्लासिक एडिटर का उपयोग करती हैं |
| **Kadence Blocks** | जिन साइटों पर Kadence Blocks प्लगइन सक्रिय है |
| **Kadence Theme** | जिन साइटों पर Kadence थीम सक्रिय थीम है |

प्रत्येक स्किल पर विस्तृत मार्गदर्शन के लिए [Theme-Aware Skills](../../user-guide/configuration/theme-aware-skills) देखें।

## इंटरनेट सर्च प्रोवाइडर्स (Internet Search Providers)

Superdav AI Agent v1.10.0 ने इंटरनेट सर्च क्वेरी के लिए Brave Search के विकल्प के रूप में **Tavily** जोड़ा है। दोनों प्रोवाइडर्स रियल-टाइम सर्च परिणाम प्रदान करते हैं और इन्हें **Gratis AI Agent → Settings → Advanced** में कॉन्फ़िगर किया जा सकता है।

| Provider | Features | Setup |
|---|---|---|
| **Brave Search** | तेज़, गोपनीयता-केंद्रित सर्च परिणाम | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | समृद्ध सर्च परिणाम, उन्नत रिसर्च क्षमताएँ | [Tavily API](https://tavily.com/) |

कॉन्फ़िगरेशन निर्देशों के लिए [Internet Search](../../user-guide/configuration/internet-search) देखें।

## साइट बिल्डर कॉन्टैक्ट फॉर्म्स (Site Builder Contact Forms)

साइट बिल्डर एजेंट अब चैट इंटरफ़ेस से सीधे कॉन्टैक्ट फॉर्म बना सकता है। बस साइट बिल्डर से कॉन्टैक्ट फॉर्म बनाने के लिए कहें, और यह स्वचालित रूप से आपके इंस्टॉल किए गए फॉर्म प्लगइन (Contact Form 7, WPForms, Fluent Forms, या Gravity Forms) का पता लगाकर उपयोग के लिए तैयार फॉर्म जनरेट कर देगा।

विस्तृत मार्गदर्शन के लिए [Site Builder Contact Forms](../../user-guide/administration/site-builder-contact-forms) देखें।

## थीम बिल्डर फीचर्स (Theme Builder Features)

Superdav AI Agent v1.16.0 में Theme Builder में शक्तिशाली नई क्षमताएँ शामिल हैं:

### फोटो अपलोड के साथ डिस्कवरी इंटरव्यू (Discovery Interview with Photo Upload)

Theme Builder डिस्कवरी इंटरव्यू में अब फोटो अपलोड का चरण शामिल है जहाँ आप विज़ुअल प्रेरणा प्रदान कर सकते हैं। Theme Builder को आपकी डिज़ाइन दिशा समझने में मदद करने के लिए अपने ब्रांड, प्रतिस्पर्धी साइटों, या डिज़ाइन प्रेरणा की तस्वीरें अपलोड करें।

विस्तृत मार्गदर्शन के लिए [Discovery Interview](./theme-builder/discovery-interview.md) देखें।

### डेस्कटॉप और मोबाइल प्रीव्यू के साथ डिज़ाइन दिशा (Design Direction with Desktop and Mobile Previews)

Theme Builder द्वारा आपका पूरा थीम जनरेट करने से पहले डेस्कटॉप और मोबाइल दोनों उपकरणों पर अपने डिज़ाइन की समीक्षा करें। कई रिस्पॉन्सिव ब्रेकपॉइंट्स पर रियल-टाइम प्रीव्यू देखें।

विस्तृत मार्गदर्शन के लिए [Design Direction](./theme-builder/design-direction.md) देखें।

### हॉस्पिटैलिटी मेनू (Hospitality Menus)

Theme Builder अब रेस्टोरेंट, कैफे, बार और अन्य हॉस्पिटैलिटी व्यवसायों के लिए संरचित खाद्य और पेय मेनू पेज जनरेट कर सकता है। मेनू में मूल्य निर्धारण, विवरण, आहार संबंधी जानकारी और एलर्जी चेतावनी शामिल होती है।

विस्तृत मार्गदर्शन के लिए [Hospitality Menus](./theme-builder/hospitality-menus.md) देखें।

### नई क्षमताएँ (New Abilities)

- **[Generate Logo SVG](./abilities/generate-logo-svg.md)** — स्वचालित सैनिटाइजेशन के साथ कस्टम SVG लोगो बनाएँ।
- **[Validate Palette Contrast](./abilities/validate-palette-contrast.md)** — WCAG अनुपालन के लिए रंग जोड़ों की जाँच करें।
- **[Create Menu](./abilities/create-menu.md)** — विशिष्ट नेविगेशन लेबल के साथ नेविगेशन मेनू बनाएँ।

उपलब्ध टूल्स और टियर 1 साइट-स्क्रैप टूल के बारे में जानकारी के लिए [Tools and Configuration](./theme-builder/tools.md) देखें।

## कॉन्फ़िगरेशन (Configuration)

सभी सेटिंग्स WordPress एडमिन में **Gratis AI Agent → Settings** के तहत हैं। Superdav AI Agent निम्नलिखित कॉन्फ़िगरेशन विकल्प जोड़ता है:

| Setting | Description |
|---|---|
| **Internet Search Provider** | Brave Search और Tavily में से चुनें (वैकल्पिक) |
| **Tavily API Key** | Tavily इंटरनेट सर्च के लिए API कुंजी (यदि Tavily चुना गया है) |

## v1.16.0 में सुधार (Improvements in v1.16.0)

- **AI Client cache** — अब क्रॉस-रिक्वेस्ट पर्सिस्टेंस के लिए ट्रांजिएंट्स (transients) द्वारा समर्थित है, जिससे लंबे समय तक चलने वाले एजेंट कार्यों पर डेटा हानि नहीं होती।
- **टियर 1 साइट-स्क्रैप टूल** — sd-ai-agent/site-scrape अब Theme Builder में डिफ़ॉल्ट रूप से उपलब्ध है, जिसका उपयोग प्रतिस्पर्धी और प्रेरणादायक वेबसाइटों का विश्लेषण करने के लिए किया जाता है।
- **प्लगइन रो एक्शन लिंक्स** — स्पष्टता के लिए ठीक किया गया और नाम बदला गया।

## v1.10.0 में सुधार (Improvements in v1.10.0)

- **WooCommerce इंटीग्रेशन रीडिज़ाइन किया गया** — अब बेहतर विश्वसनीयता और संगतता के लिए native WooCommerce APIs का उपयोग करता है।
- **प्रोवाइडर लिस्ट ऑटो-रिफ्रेश** — स्वचालित रूप से पता लगाता है कि प्लगइन्स कब सक्रिय या निष्क्रिय किए गए हैं और उपलब्ध प्रोवाइडर सूची को अपडेट करता है।
- **बेहतर साइट बिल्डर** — बेहतर एरर रिकवरी के साथ मल्टी-फेज साइट बिल्ड प्लान में सुधार।
- **थीम-अवेयर स्किल्स** — चार नई बिल्ट-इन स्किल्स जो स्वचालित रूप से आपके सक्रिय थीम के अनुकूल होती हैं।

## v1.10.0 में बग फिक्स (Bug Fixes in v1.10.0)

- कुछ एडमिन पेजों पर अनंत रीलोड लूप का कारण बनने वाली `navigate-to` क्षमता को ठीक किया गया।
- `list-posts` क्षमता को श्रेणी और टैग नामों को सही ढंग से स्लग्ज़ में हल नहीं कर पा रहा था, इसे ठीक किया गया।
- पिछले रीफ़ैक्टर के बाद WP-CLI कमांड्स में नेमस्पेस एलियास गायब होने की समस्या को ठीक किया गया।
- इवेंट ऑटोमेशन को उन साइटों पर शालीनता से संभालने के लिए ठीक किया गया जहाँ ऑटोमेशन टेबल अभी तक नहीं बनाई गई हैं।
- सिस्टम इंस्ट्रक्शन बिल्डर में सही नेमस्पेस प्रीफ़िक्स का उपयोग करने वाली `memory-save` क्षमता को ठीक किया गया।
- AI को वापस भेजे जाने से पहले स्केलर टूल परिणामों को सही ढंग से रैप करने के लिए ठीक किया गया।
- पुराने संस्करणों से अपग्रेड करते समय उपयोग किए गए आँकड़ों को सही ढंग से संभालने के लिए उपयोग किए गए आँकड़ों को ठीक किया गया।
