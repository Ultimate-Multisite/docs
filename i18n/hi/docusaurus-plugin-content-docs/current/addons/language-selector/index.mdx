# Language Selector

Multisite Ultimate: Sign-up पर Language Selector

Sign-up flow में एक language selector field जोड़ें।

## Installation

1. 'ultimate-multisite-language-selector' को '/wp-content/plugins/' डायरेक्टरी में अपलोड करें
2. अपने WordPress Network Admin में 'Plugins' मेनू के माध्यम से प्लगइन को एक्टिवेट करें

## Usage Guide

इस मुफ़्त add-on के साथ, आप अपने उपयोगकर्ताओं को यह चुनने दे सकते हैं कि वे sign-up flow और login page पर कौन सी भाषा देखना चाहते हैं।

[अपनी license को activate करने के बाद](../../user-guide/getting-started/installing-ultimate-multisite.md), आप Ultimate Multisite add-ons को install कर पाएंगे, जो आपकी license द्वारा उपलब्ध कराए जाते हैं।

अपनी Ultimate Multisite settings page पर जाएं और फिर, बाईं ओर दिए गए मेनू पर, _Add-ons_ लेबल पर क्लिक करें।

![Add-ons page](/img/admin/addons-page.png)

_Language Selector इंस्टॉल करना शुरू करने के लिए Add-ons पेज पर जाएं_

वहां, आपको वे सभी add-ons मिलेंगे जो हम प्रदान करते हैं।

मुफ़्त add-ons हमारे सभी ग्राहकों के लिए उपलब्ध हैं, भले ही उन्होंने किस subscription plan पर साइन अप किया हो।

add-on के थंबनेल के नीचे दिए गए _Install it Now!_ बटन पर क्लिक करें। फिर इसे Plugins पेज पर एक्टिवेट करें।

![Add-ons page with install button](/img/admin/addons-page.png)

_add-on इंस्टॉल करने के लिए बटन पर क्लिक करें_

## URL के माध्यम से भाषा सेट करना

आप locale parameter को URL से बायपास करके भी पेज की भाषा बदल सकते हैं। उदाहरण के लिए, यदि आप इटालियन में रजिस्टर पेज पर रीडायरेक्ट करना चाहते हैं, तो URL में ?locale=it_IT जोड़ दें।

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

इसके काम करने के लिए भाषा को आपके WordPress network पर इंस्टॉल होना ज़रूरी है।
