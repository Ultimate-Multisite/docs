# Fluent Forms {#fluent-forms}

यह ऐड-ऑन Multisite Ultimate पर उत्पादों, मेंबरशिप या साइट्स में Fluent Forms पर लिमिट (सीमा) जोड़ने की सुविधा देता है।
Fluent Forms कस्टम पोस्ट टाइप (custom post types) के बजाय कस्टम टेबल्स (custom tables) का उपयोग करता है, इसलिए सामान्य लिमिटिंग विकल्प Fluent Forms के साथ काम नहीं करते हैं।

## विशेषताएँ {#features}

- Fluent Forms या Conversational Fluent Forms के लिए लिमिट जोड़ें।
- जब ग्राहक लिमिट तक पहुँच जाते हैं, तो उन्हें एक दोस्ताना संदेश मिलता है जो उन्हें अपना प्लान अपग्रेड करने के लिए प्रेरित करता है।
