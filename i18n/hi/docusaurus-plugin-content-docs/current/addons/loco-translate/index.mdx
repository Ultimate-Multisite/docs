# Loco Translate

Ultimate Multisite: Loco Translate Integration addon आपके WordPress multisite network में site-specific translation क्षमताओं को लाता है। यदि कभी आपको यह ज़रूरत पड़ी है कि आपके नेटवर्क के अलग-अलग साइट्स को एक ही plugin या theme के अलग-अलग अनुवाद चाहिए, तो यह addon एकदम सही समाधान है।

**यह Addon क्या करता है**

यह addon Ultimate Multisite को लोकप्रिय Loco Translate plugin के साथ सहजता से (seamlessly) एकीकृत करता है, जिससे आपके नेटवर्क की प्रत्येक सबसाइट अपनी कस्टम अनुवाद (custom translations) बनाए रख सकती है। चाहे आप एक मल्टी-रीजनल प्लेटफॉर्म चला रहे हों, विभिन्न भाषा समुदायों को सेवा दे रहे हों, या बस अपने नेटवर्क में अनुवाद लचीलेपन (translation flexibility) की ज़रूरत हो, यह addon इसे आसान बना देता है।

**किसके लिए एकदम सही:**

* विभिन्न स्थानीयकरण (localization) ज़रूरतों वाले मल्टी-रीजनल व्यवसाय
* विविध समुदायों को सेवा देने वाले शैक्षणिक संस्थान
* बहुभाषी समर्थन (multilingual support) देने वाले SaaS प्लेटफॉर्म
* कई क्लाइंट साइट्स को मैनेज करने वाली वेब एजेंसियां
* अनुवाद लचीलेपन की आवश्यकता वाला कोई भी multisite network

## इंस्टॉलेशन {#installation}

1. सुनिश्चित करें कि Ultimate Multisite इंस्टॉल और network activated है।
2. Loco Translate plugin इंस्टॉल और activate करें।
3. इस addon को अपने plugins directory में अपलोड करें।
4. Loco Translate Integration addon को network activate करें।
5. किसी भी सबसाइट पर Loco Translate के माध्यम से site-specific अनुवाद बनाना शुरू करें।

बस! यह integration बिना किसी अतिरिक्त कॉन्फ़िगरेशन के अपने आप काम करता है।

## अक्सर पूछे जाने वाले प्रश्न {#frequently-asked-questions}

## क्या इंस्टॉलेशन के बाद मुझे कुछ कॉन्फ़िगर करना होगा? {#do-i-need-to-configure-anything-after-installation}

नहीं! यह integration स्वचालित (automatic) है। एक बार जब addon और Loco Translate दोनों सक्रिय हो जाते हैं, तो सबसाइट तुरंत अपने कस्टम अनुवाद बनाना शुरू कर सकती हैं।

## अगर कोई सबसाइट कस्टम अनुवाद नहीं बनाती है तो क्या होगा? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

साइट स्वचालित रूप से fallback के रूप में network-wide translations का उपयोग करेगी। कुछ भी टूटता नहीं है, और सब कुछ सहजता से काम करता है।

## क्या सबसाइट्स network-wide translations तक पहुँच सकती हैं? {#can-subsites-access-network-wide-translations}

नियमित site admins केवल अपनी साइट के अनुवाद देख और एडिट कर सकते हैं। Super admins, इस बात पर निर्भर करते हुए कि वे किस साइट को देख रहे हैं, network-wide और site-specific दोनों अनुवाद तक पहुँच सकते हैं।

## क्या यह plugins और themes दोनों के साथ काम करता है? {#does-this-work-with-both-plugins-and-themes}

हाँ! site-specific translations WordPress plugins और themes दोनों के लिए काम करते हैं।

## क्या इससे मेरा multisite network धीमा हो जाएगा? {#will-this-slow-down-my-multisite-network}

नहीं। addon कुशल फ़ाइल लोडिंग (efficient file loading) का उपयोग करता है जो केवल ज़रूरत पड़ने पर site-specific translations की जाँच करता है। प्रदर्शन पर कोई ध्यान देने योग्य प्रभाव नहीं पड़ता है।

## क्या मैं network translations को शुरुआती बिंदु के रूप में किसी विशिष्ट साइट पर कॉपी कर सकता हूँ? {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

हाँ! Loco Translate में अनुवाद एडिट करते समय, आप आसानी से network translations से शुरुआत कर सकते हैं और उन्हें प्रत्येक साइट के लिए कस्टमाइज़ कर सकते हैं।

## क्या यह WooCommerce और अन्य plugins के साथ काम करता है? {#does-this-work-with-woocommerce-and-other-plugins}

बिल्कुल! कोई भी plugin या theme जो मानक WordPress translation functions का उपयोग करता है, वह site-specific translations के साथ काम करेगा।
