# GoCardless चेंजलॉग

Version: 1.0.3 - जारी किया गया 2026-05-05
* सुधार: अब जब GoCardless सदस्यता को लंबित (pending) या रोक (on-hold) स्थिति में रद्द किया जाता है, तो अनाथ (orphaned) लंबित साइटों को साफ़ कर दिया जाता है।
* बेहतर किया गया: जब Bedrock रूट ऑटलोडर ने पहले ही निर्भरताएँ (dependencies) लोड कर ली होती हैं, तो प्लगइन ऑटलोडर को छोड़ दिया जाता है।

Version: 1.0.1 - जारी किया गया 2025-09-28

* प्रीफ़िक्स का नाम ultimate-multisite में बदला गया; टेक्स्ट डोमेन अपडेट किया गया; संस्करण बढ़ाया गया।


2021-09-24 - Version 1.0.0-beta.4

* जोड़ा गया: mu-plugins पर आधारित सेटअप के लिए फ़िल्टर wp_ultimo_skip_network_active_check;

2021-06-16 - Version 1.0.0-beta.3

* सुधार: वेबहुक हैंडलर हस्ताक्षर की पुष्टि करने में विफल हो रहे थे;

2021-06-14 - Version 1.0.0-beta.2
