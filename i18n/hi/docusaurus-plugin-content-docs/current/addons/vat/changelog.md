# VAT Changelog {#vat-changelog}

Version 1.0.7 - Released on 2026-02-03

* सुधार: VIES SOAP सेवा में MS_MAX_CONCURRENT_REQ त्रुटियों के कारण जर्मन (DE) VAT नंबर सत्यापन (validation) में विफल हो रहे थे। अधिक विश्वसनीय सत्यापन के लिए पुराने SOAP API से आधिकारिक EU VIES REST API पर स्विच किया गया है।
* सुधार: अब VAT नंबर स्वीकार किए जाते हैं, भले ही VIES सेवा अस्थायी रूप से अनुपलब्ध हो, न कि गलत तरीके से अस्वीकार किए जाते हैं। बाद में समीक्षा के लिए एक लॉग एंट्री बनाई जाती है।
* सुधार: VAT/Tax ID फ़ील्ड अब केवल EU सदस्यों के लिए नहीं, बल्कि सभी देशों के लिए दिखाई देता है। गैर-EU टैक्स ID (जैसे स्विस CHE नंबर) को VIES सत्यापन के बिना इनवॉइस डिस्प्ले के लिए स्टोर किया जाता है।
* सुधार: ग्रीस (GR), मोनाको (MC), और आइल ऑफ मैन (IM) के लिए VAT प्रीफ़िक्स लुकअप गलत ऐरे लुकअप लॉजिक के कारण टूट गया था।
* सुधार: अनावश्यक wp-cli-bundle dev dependency को हटा दिया गया है, जो Site Exporter addon के साथ उपयोग किए जाने पर घातक त्रुटियाँ (fatal errors) पैदा कर सकता था।
* सुधार: एक व्यापक यूनिट टेस्ट सूट (53 टेस्ट) जोड़ा गया है।

Version 1.0.6 - Released on 2026-01-25

* सुधार: अब घरेलू B2B लेन-देन पर सही ढंग से VAT चार्ज होता है। EU VAT नियमों के अनुसार, रिवर्स चार्ज केवल सीमा-पार B2B लेन-देन पर लागू होता है, न कि तब जब ग्राहक का देश कंपनी के देश से मेल खाता हो।

Version 1.0.5 - Released on 2026-01-22

* सुधार: Addon लोड नहीं हो रहा था या ठीक से काम नहीं कर रहा था।
* बदलाव: अधिक विश्वसनीय और सक्रिय रूप से बनाए गए डेटा के लिए VAT दर डेटा स्रोत को euvatrates.com से ibericode/vat-rates रिपॉजिटरी में बदल दिया गया है।
* सुधार: super_reduced_rates ऑप्शन वैल्यू की टाइपो (typo) को ठीक किया गया है।
* सुधार: त्रुटियों को रोकने के लिए कंट्री कोड हैंडलिंग के लिए नल चेक (null checks) जोड़े गए हैं।
* बदलाव: Traduttore के माध्यम से स्वचालित अपडेट के पक्ष में बंडल किए गए अनुवाद फ़ाइलों को हटा दिया गया है।
