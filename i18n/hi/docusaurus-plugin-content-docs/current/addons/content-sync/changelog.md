# कंटेंट सिंक परिवर्तनों का विवरण

### Version 1.1.0 - Released on 2025-12-31 {#version-110---released-on-2025-12-31}
* नया: सभी पोस्ट प्रकारों (पोस्ट, पेज, कस्टम पोस्ट प्रकार) को सिंक करने का समर्थन।
* नया: कंटेंट चयन चरण में पोस्ट प्रकार फ़िल्टरिंग।
* नया: स्रोत साइट चयन चरण - अब केवल मुख्य साइट से नहीं, बल्कि किसी भी साइट से सिंक करें।
* नया: टेम्पलेट साइटों के लिए चाइल्ड साइट प्री-सिलेक्शन।
* नया: सभी पोस्ट प्रकारों के लिए post_content से मीडिया निकालना (Media extraction)।
* नया: टैक्सोनॉमी सिंकिंग (श्रेणियाँ, टैग, कस्टम टैक्सोनॉमी)।
* बदला गया: "Elementor Template Sync" का नाम बदलकर "Content Sync" कर दिया गया।
* बदला गया: "Select Templates" चरण का नाम बदलकर "Select Data" कर दिया गया।
* बेहतर किया गया: Elementor-विशिष्ट कोड अब केवल elementor_library पोस्ट प्रकार के लिए चलेगा।

### Version 1.0.0 - Released on 2025-12-19 {#version-100---released-on-2025-12-19}
* प्रारंभिक रिलीज़।
* टेम्पलेट और साइट चयन के लिए बहु-चरणीय विज़ार्ड।
* वास्तविक समय की प्रगति प्रतिक्रिया के साथ बैच प्रोसेसिंग।
* स्वचालित ID रीमैपिंग के साथ पूर्ण मीडिया सिंक।
* Ultimate Multisite के साथ सेटिंग्स का एकीकरण।
* नेटवर्क सेटिंग्स के माध्यम से साइट को बाहर करने का समर्थन।
* सिंक के बाद Elementor कैश क्लियरिंग।
* टेम्पलेट टैक्सोनॉमी का संरक्षण।
