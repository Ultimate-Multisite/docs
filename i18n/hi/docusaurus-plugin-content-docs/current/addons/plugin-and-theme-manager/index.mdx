# Plugin और Theme Manager

Multisite Ultimate: Plugin और Theme Manager

इंस्टॉल किए गए प्लगइन्स और थीम्स की मेटा जानकारी (शीर्षक, लेखक, थंबनेल और विवरण) को एडिट या छिपाएं। प्लगइन्स और थीम्स के लिए श्रेणियां (categories) बनाएं ताकि आपके यूज़र्स उन्हें फ़िल्टर कर सकें और आपके यूज़र्स के पैनल में एक सुंदर कस्टम प्लगइन्स पेज जोड़ सकें।

प्लगइन का उपयोग करने के लिए, आप बस अपने नेटवर्क एडमिन पर प्लगइन्स या थीम्स पेज पर नेविगेट कर सकते हैं और "Edit Plugin/Theme info" पर क्लिक कर सकते हैं।

## इंस्टॉलेशन {#installation}

1. 'ultimate-multisite-plugin-and-theme-manager' को '/wp-content/plugins/' डायरेक्टरी में अपलोड करें
2. अपने WordPress Network Admin में 'Plugins' मेनू के माध्यम से प्लगइन को एक्टिवेट करें

## उपयोग गाइड {#usage-guide}

Ultimate Multisite: Plugin और Theme Manager एक शक्तिशाली **प्रीमियम** Ultimate Multisite ऐड-ऑन है जो आपको यह पूरा नियंत्रण देता है कि आपके नेटवर्क के यूज़र्स आपके नेटवर्क पर प्लगइन और थीम के विकल्पों को कैसे देखते हैं और उनके साथ इंटरैक्ट करते हैं।

यह ऐड-ऑन तीन उद्देश्यों की पूर्ति करता है:

  1. यह आपको प्लगइन्स और थीम्स दोनों की मेटाडेटा को पूरी तरह से बदलने देता है। इसमें नाम, विवरण, लेखक का नाम और यहाँ तक कि थंबनेल भी शामिल हैं!

  2. यह आपको प्लगइन्स और थीम्स को श्रेणियां (categorize) करने देता है, जिससे आपके यूज़र्स उन्हें आसानी से सॉर्ट कर सकें।

  3. यह आपको पुराने, उबाऊ प्लगइन्स पेज को श्रेणियों (categories) द्वारा सॉर्ट किए गए एक नए थीम्स जैसे पेज से बदलने का विकल्प देता है।

इसे डाउनलोड और इंस्टॉल करने के बाद, Ultimate Multisite Settings पेज पर एक नया टैब - _Add-on Settings_ - दिखाई देगा।
