# फ़िल्टर: wp_ultimo_skip_network_active_check

डेवलपर्स को इस जाँच को बायपास (short-circuit) करने की अनुमति देता है।

यह तब उपयोगी होता है जब आप composer-आधारित और अन्य कस्टम सेटअप का उपयोग कर रहे हों, जैसे कि Bedrock, उदाहरण के लिए, जहाँ mu-plugins के रूप में प्लगइन्स का उपयोग करना सामान्य होता है।
