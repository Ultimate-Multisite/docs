# Filter: wu_apc_get_meta

डेवलपर्स को अंतिम मानों (final values) को फ़िल्टर करने और नए प्रोसेसिंग केस जोड़ने की अनुमति देता है।

## पैरामीटर्स {#parameters}

| Name | Type | विवरण |
|------|------|-------------|
| $default_value | `string` | डिफ़ॉल्ट मान। |
| $meta_type | `string` | प्राप्त किए जाने वाले मेटा डेटा का प्रकार। |
| $meta_key | `string` | मेटा फ़ील्ड का नाम। |

### कब से {#since}

- 1.4.0

नया डिफ़ॉल्ट मान।
