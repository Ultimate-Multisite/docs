# Ultimate Multisite इंस्टॉल करना

:::note
यह ट्यूटोरियल मानता है कि आपने पहले से WordPress Multisite इंस्टॉल और कॉन्फ़िगर कर लिया है। इसे कैसे करें, यह जानने के लिए WP Beginner का [यह ट्यूटोरियल](https://www.wpbeginner.com/glossary/multisite/) देखें।
:::

## Plugin इंस्टॉल करना

Ultimate Multisite [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) पर मुफ्त उपलब्ध है।

अपने **Network Admin Dashboard** में जाएं, फिर **Plugins → Add New Plugin** पर क्लिक करें।

![Network Admin Add New Plugin पेज](/img/installation/add-new-plugin.png)

**"Ultimate Multisite"** खोजें (सटीक मिलान के लिए कोट्स के साथ) और यह पहले रिजल्ट में दिखाई देगा। **Install Now** पर क्लिक करें।

![Ultimate Multisite दिखाते सर्च रिजल्ट्स](/img/installation/search-ultimate-multisite.png)

इंस्टॉल होने के बाद, पूरे नेटवर्क में plugin को एक्टिवेट करने के लिए **Network Activate** पर क्लिक करें।

![Network Activate बटन के साथ इंस्टॉल हुआ Plugin](/img/installation/plugin-installed.png)

एक्टिवेशन के बाद, आप अपने आप Setup Wizard पर रीडायरेक्ट हो जाएंगे।

![Plugin एक्टिवेट हुआ और wizard पर रीडायरेक्ट](/img/installation/plugin-activated.png)

## Setup Wizard

Setup Wizard लगभग 10 मिनट में Ultimate Multisite को कॉन्फ़िगर करने में आपकी मदद करेगा।

### Welcome

शुरू करने के लिए **Get Started** पर क्लिक करें।

![Setup Wizard welcome स्क्रीन](/img/installation/wizard-welcome.png)

### Pre-install Checks

यह स्टेप आपकी सिस्टम जानकारी और WordPress इंस्टॉलेशन की जांच करता है ताकि यह सुनिश्चित हो सके कि यह Ultimate Multisite की आवश्यकताओं को पूरा करता है। अगर सब कुछ ठीक है, तो **Go to the Next Step** पर क्लिक करें।

![सिस्टम आवश्यकताएं दिखाती Pre-install checks](/img/installation/wizard-pre-install-checks.png)

### Installation

इंस्टॉलर जरूरी database tables बनाएगा और `sunrise.php` फाइल इंस्टॉल करेगा जो Ultimate Multisite को काम करने के लिए चाहिए। आगे बढ़ने के लिए **Install** पर क्लिक करें।

![Database tables और sunrise.php दिखाता Installation स्टेप](/img/installation/wizard-installation.png)

### Your Company

अपनी कंपनी की जानकारी भरें और अपनी डिफ़ॉल्ट करेंसी सेट करें। यह जानकारी आपके पूरे WaaS प्लेटफॉर्म में इस्तेमाल होगी। पूरा होने पर **Continue** पर क्लिक करें।

![Your Company कॉन्फ़िगरेशन स्टेप](/img/installation/wizard-your-company.png)

### Default Content

यह स्टेप आपको पहले से बने templates, products और अन्य स्टार्टर कंटेंट इंस्टॉल करने देता है। Ultimate Multisite की फीचर्स से परिचित होने का यह एक बढ़िया तरीका है। डिफ़ॉल्ट कंटेंट जोड़ने के लिए **Install** पर क्लिक करें, या अगर आप शुरू से शुरू करना चाहते हैं तो इस स्टेप को स्किप करें।

![Default content installation स्टेप](/img/installation/wizard-default-content.png)

### Recommended Plugins

वैकल्पिक रूप से recommended companion plugins इंस्टॉल करें। इन्हें जोड़ने के लिए **Install** पर क्लिक करें या आगे बढ़ने के लिए स्किप करें।

![Recommended plugins स्टेप](/img/installation/wizard-recommended-plugins.png)

### Ready!

बस हो गया! आपका Ultimate Multisite इंस्टॉलेशन पूरा हो गया है। अब आप **Network Admin Dashboard** से अपना Website as a Service प्लेटफॉर्म बनाना शुरू कर सकते हैं।

![Setup पूरा - Ready स्क्रीन](/img/installation/wizard-ready.png)

![Ultimate Multisite एक्टिव के साथ Network Admin Dashboard](/img/installation/network-dashboard.png)

आगे बढ़ें और मज़े करें!
