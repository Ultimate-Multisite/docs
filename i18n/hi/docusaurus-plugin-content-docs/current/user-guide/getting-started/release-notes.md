# रिलीज़ नोट्स

## Version 2.13.0 — 2026-06-05 को जारी

- नया: ग्राहक खाते, चेकआउट, बिलिंग, साइट, इनवॉइस, टेम्पलेट बदलने और डोमेन मैपिंग प्रवाहों के लिए sovereign-tenant समर्थन जोड़ा गया, ताकि टेनेंट नेटवर्क प्रबंधित कार्रवाइयों के लिए ग्राहकों को मुख्य साइट पर वापस भेज सकें।
- नया: आवर्ती सदस्यताओं के लिए नवीनीकरण-प्रमाण जांचें जोड़ी गईं, ताकि सहेजा गया बिलिंग अनुबंध, सदस्यता या vault token न होने पर गेटवे ऑटो-नवीनीकरण बंद कर सकें।
- नया: लंबित साइट निर्माण के लिए HMAC-सत्यापित loopback प्रकाशन जोड़ा गया, ताकि उन होस्ट पर चेकआउट-से-साइट प्रावधान अधिक भरोसेमंद हो जहाँ पृष्ठभूमि कार्यों में देरी होती है।
- नया: SSO URL, चेकआउट-फ़ॉर्म आधार डोमेन और स्वचालित डोमेन-रिकॉर्ड निर्माण के लिए डेवलपर विस्तार बिंदु जोड़े गए।
- सुधार: मैप किए गए डोमेन, गुमनाम broker विज़िट, लॉगआउट और क्रॉस-प्लगइन निर्भरता टकरावों में SSO अब अधिक भरोसेमंद है।
- सुधार: लंबित साइट निर्माण अब पुराने publish flags से उबरता है और ग्राहकों को साइट-निर्माण स्क्रीन पर अटका छोड़ने से बचाता है।
- सुधार: साझा चेकआउट-फ़ॉर्म आधार डोमेन के लिए डोमेन रिकॉर्ड अब नहीं बनाए जाते, और कोई सक्रिय एकीकरण न होने पर अप्रयुक्त host-provider पृष्ठभूमि कार्य छोड़ दिए जाते हैं।
- सुधार: चेकआउट, बिलिंग पता, पासवर्ड रीसेट, ईमेल सत्यापन, टेम्पलेट बदलना, टूर और ग्राहक Dashboard के edge cases अब सामान्य ग्राहक प्रवाहों को नहीं रोकते।
- सुधार: प्रसारण ईमेल अब प्राप्तकर्ताओं को निजी रखते हैं और प्राप्तकर्ता सूचियाँ या मेल transports विफल होने पर SMTP/प्लगइन fatal errors से बचते हैं।
- सुधार: सदस्यता नवीनीकरण, समाप्ति प्रदर्शन और भुगतान संग्रह के edge cases अब तत्काल समाप्ति, क्रैश या आवश्यक भुगतान छूटने से बचते हैं।
- बेहतर: WordPress संगतता 7.0 तक परखी गई है, प्रोडक्शन Vue assets npm स्रोतों से फिर से बनाए गए हैं, और Cypress end-to-end कवरेज अब अधिक चेकआउट, सेटअप, SSO और गेटवे प्रवाहों को जांचता है।
