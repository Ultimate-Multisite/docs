# Site Builder Mode हटाने की सूचना

**Superdav AI Agent v1.12.0 में Site Builder mode हटा दिया गया है।** यदि आप Site Builder mode का उपयोग कर रहे थे, तो थीम बनाने और साइट सेटअप के लिए आपको **Setup Assistant agent** पर माइग्रेट करना चाहिए।

## क्या हुआ?

### Site Builder Mode (पुराना)

Site Builder mode एक विज़ार्ड-आधारित इंटरफ़ेस था जिसका उपयोग निम्नलिखित के लिए किया जाता था:

- टेम्पलेट्स से साइट बनाना
- बुनियादी सेटिंग्स कॉन्फ़िगर करना
- एक थीम चुनना
- शुरुआती कंटेंट सेट करना

### इसकी जगह क्या आया?

**Setup Assistant agent** अब सभी Site Builder कार्यक्षमता को संभालता है, जिसमें ये शामिल हैं:

- अधिक लचीला, agent-guided सेटअप
- बेहतर थीम कस्टमाइज़ेशन विकल्प
- Theme Builder ऑनबोर्डिंग के साथ इंटीग्रेशन
- भविष्य के सेशन के लिए स्थायी site_brief मेमोरी

## यदि आप Site Builder Mode का उपयोग कर रहे थे

### आपकी साइटें सुरक्षित हैं

- Site Builder mode से बनाई गई मौजूदा साइटें काम करना जारी रखेंगी
- कोई डेटा हानि या साइट में रुकावट नहीं होगी
- आप सामान्य रूप से अपनी साइटों का प्रबंधन करना जारी रख सकते हैं

### Setup Assistant Agent पर माइग्रेट करें

नई साइट सेटअप या थीम बदलाव के लिए, Setup Assistant agent का उपयोग करें:

```
"Help me set up a new site"
```

या

```
"Start the Theme Builder onboarding"
```

Setup Assistant agent अधिक लचीलेपन के साथ समान कार्यक्षमता प्रदान करता है।

## तुलना: Site Builder बनाम Setup Assistant

| Feature | Site Builder (हटा दिया गया) | Setup Assistant (नया) |
|---------|----------------------|----------------------|
| Setup method | विज़ार्ड फॉर्म | agent बातचीत |
| Theme selection | पूर्वनिर्धारित टेम्पलेट्स | कस्टम जनरेशन |
| Customization | सीमित विकल्प | पूर्ण डिज़ाइन सिस्टम |
| Site brief | स्टोर नहीं किया जाता | स्थायी मेमोरी |
| Future sessions | सेटअप दोहराना | संग्रहीत site_brief का उपयोग करना |
| Flexibility | निश्चित वर्कफ़्लो | अनुकूलनीय बातचीत |

## Setup Assistant Agent पर माइग्रेट करना

### नई साइटों के लिए

Site Builder mode का उपयोग करने के बजाय:

1. अनुरोध करें: "Help me set up a new site"
2. Setup Assistant agent आपको निम्नलिखित के माध्यम से मार्गदर्शन करेगा:
   - साइट का उद्देश्य और लक्ष्य
   - लक्षित दर्शक (Target audience)
   - ब्रांड पहचान (Brand identity)
   - थीम जनरेशन
   - प्रारंभिक कॉन्फ़िगरेशन

### मौजूदा साइटों के लिए

यदि आपके पास Site Builder mode से कोई मौजूदा साइट है:

1. आप इसका उपयोग वैसे ही करना जारी रख सकते हैं
2. थीम अपडेट करने के लिए, अनुरोध करें: "Redesign my site"
3. Setup Assistant agent को एक नई थीम बनाने में आपकी मदद मिलेगी
4. आपकी साइट का डेटा अपरिवर्तित रहेगा

### थीम बदलाव के लिए

Site Builder mode के थीम चयन के बजाय:

1. अनुरोध करें: "Change my theme"
2. Setup Assistant agent:
   - आपकी डिज़ाइन प्राथमिकताओं के बारे में पूछेगा
   - एक कस्टम थीम जनरेट करेगा
   - इसे आपकी साइट पर सक्रिय करेगा

## मुख्य अंतर

### Site Builder Mode

```
1. एक टेम्पलेट चुनें
2. एक थीम चुनें
3. बुनियादी सेटिंग्स कॉन्फ़िगर करें
4. हो गया
```

### Setup Assistant Agent

```
1. अपनी साइट का उद्देश्य बताएं
2. अपने लक्षित दर्शकों को परिभाषित करें
3. डिज़ाइन प्राथमिकताएं चुनें
4. agent कस्टम थीम जनरेट करता है
5. agent थीम सक्रिय करता है
6. भविष्य के सेशन के लिए site_brief स्टोर होता है
```

## Setup Assistant Agent के लाभ

### अधिक लचीला

- अपनी साइट का वर्णन प्राकृतिक भाषा में करें
- कस्टम सिफारिशें प्राप्त करें
- आपकी विशिष्ट ज़रूरतों के अनुकूल हो

### बेहतर कस्टमाइज़ेशन

- कस्टम थीम जनरेशन
- डिज़ाइन सिस्टम निर्णय
- स्थायी डिज़ाइन टोकन

### स्थायी मेमोरी

- आपका site_brief स्टोर होता है
- भविष्य के agents आपकी साइट को समझते हैं
- सेटअप जानकारी दोहराने की ज़रूरत नहीं

### एकीकृत वर्कफ़्लो

- Theme Builder ऑनबोर्डिंग
- Design System Aesthetics skill
- Ability Visibility controls
- सब कुछ सहजता से एक साथ काम करता है

## समस्या निवारण (Troubleshooting)

### मुझे Site Builder mode नहीं मिल रहा है

Site Builder mode हटा दिया गया है। इसके बजाय Setup Assistant agent का उपयोग करें:

```
"Help me set up a new site"
```

### मैं Site Builder से साइट को फिर से बनाना चाहता हूँ

आप इसे Setup Assistant agent के साथ फिर से बना सकते हैं:

1. अनुरोध करें: "Help me set up a new site"
2. अपनी मूल साइट के उद्देश्य और डिज़ाइन का वर्णन करें
3. agent एक समान थीम जनरेट करेगा
4. आपका site_brief भविष्य के संदर्भ के लिए स्टोर हो जाएगा

### मेरी मौजूदा Site Builder साइट काम नहीं कर रही है

Site Builder mode से बनाई गई मौजूदा साइटें काम करना जारी रखेंगी। यदि आपको कोई समस्या आ रही है:

1. जांच लें कि आपकी थीम अभी भी सक्रिय है
2. सत्यापित करें कि आपके plugins सक्षम हैं
3. WordPress error logs की जांच करें
4. यदि समस्या बनी रहती है तो सपोर्ट से संपर्क करें

### क्या मैं अभी भी अपने पुराने Site Builder टेम्पलेट्स का उपयोग कर सकता हूँ?

Site Builder टेम्पलेट अब उपलब्ध नहीं हैं। हालांकि:

- आपकी मौजूदा साइटें काम करना जारी रखेंगी
- आप Setup Assistant agent के साथ समान साइटें फिर से बना सकते हैं
- Setup Assistant agent अधिक कस्टमाइज़ेशन विकल्प प्रदान करता है

## अगले कदम

1. **नई साइटों के लिए**: Setup Assistant agent का उपयोग करें
2. **मौजूदा साइटों के लिए**: उन्हें वैसे ही उपयोग करना जारी रखें
3. **थीम बदलाव के लिए**: Setup Assistant agent से मदद मांगें
4. **डिज़ाइन सुधार के लिए**: Design System Aesthetics skill का उपयोग करें

## संबंधित विषय

- **Theme Builder Onboarding**: कस्टम थीम के लिए निर्देशित सेटअप
- **Setup Assistant Agent**: agent-guided साइट सेटअप
- **Site Specification Skill**: अपनी साइट के लक्ष्य और दर्शकों को परिभाषित करें
- **Design System Aesthetics Skill**: अपनी साइट की दृश्य पहचान को परिष्कृत करें
