# इनबिल्ट एजेंट्स

Gratis AI Agent v1.9.0 पाँच इनबिल्ट एजेंट्स के साथ आता है। हर एजेंट को एक विशिष्ट टूल सेट, एक अनुकूलित सिस्टम प्रॉम्प्ट और उस क्षेत्र के सामान्य कार्यों से मेल खाते शुरुआती सुझावों के साथ पहले से कॉन्फ़िगर किया गया है। एजेंट्स के बीच स्विच करने से असिस्टेंट क्या कर सकता है और वह कैसे जवाब देगा, यह बदल जाता है—और इसके लिए आपको कोई कॉन्फ़िगरेशन करने की ज़रूरत नहीं है।

## एजेंट क्या है?

हर एजेंट एक नाम वाली कॉन्फ़िगरेशन प्रोफ़ाइल होती है जो इन चीज़ों को मिलाती है:

- **टूल्स (Tools)** — वे क्षमताएं जिनका उपयोग एजेंट कर सकता है (उदाहरण के लिए, एक कंटेंट राइटर के पास पोस्ट बनाने की क्षमता तक पहुंच होती है; एक डिज़ाइन स्टूडियो के पास CSS और theme.json की क्षमता तक पहुंच होती है)
- **सिस्टम प्रॉम्प्ट (System prompt)** — निर्देश जो एजेंट के लहजे, प्राथमिकताओं और सीमाओं को सेट करते हैं
- **सुझाव (Suggestions)** — चैट इंटरफ़ेस में दिखाए गए पहले से लिखे गए प्रॉम्प्ट, जो आपको जल्दी शुरुआत करने में मदद करते हैं

## एजेंट पिकर तक पहुँचना

1. WordPress एडमिन साइडबार में **Gratis AI Agent** पैनल खोलें।
2. चैट हेडर के ऊपरी-बाएँ कोने में **एजेंट आइकन** पर क्लिक करें (यह आइकन सक्रिय एजेंट को दर्शाने के लिए बदल जाता है)।
3. **एजेंट पिकर (Agent Picker)** एक फॉर्म-टेबल ओवरले के रूप में खुलता है। प्रत्येक एजेंट को उसके आइकन, नाम और एक लाइन के विवरण के साथ सूचीबद्ध किया जाता है।
4. इसे सक्रिय करने के लिए किसी एजेंट की पंक्ति पर क्लिक करें। चैट हेडर तुरंत अपडेट हो जाता है।

आप बातचीत के बीच में भी एजेंट्स बदल सकते हैं—नया एजेंट का सिस्टम प्रॉम्प्ट अगले संदेश से प्रभावी हो जाता है।

## पाँच इनबिल्ट एजेंट्स

### कंटेंट राइटर (Content Writer)

**फोकस:** पोस्ट, पेज और संपर्क फ़ॉर्म बनाना और एडिट करना।

**उपलब्ध टूल्स:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`

**यह क्या अच्छा करता है:**
- एक संक्षिप्त विवरण या रूपरेखा से ब्लॉग पोस्ट का ड्राफ्ट बनाना और प्रकाशित करना
- एक नई साइट के लिए लैंडिंग पेजों के बैच बनाना
- संपर्क और पूछताछ फ़ॉर्म बनाना
- URL या खोज से पोस्ट पर फ़ीचर्ड इमेज सेट करना

**शुरुआती सुझाव:**
- *WordPress multisite के लाभों पर 500 शब्दों का ब्लॉग पोस्ट लिखें।*
- *एक 'About', 'Services', और 'Contact' पेज बनाएं और उन्हें प्रकाशित करें।*
- *Contact पेज पर एक बुकिंग पूछताछ फ़ॉर्म जोड़ें।*

---

### साइट बिल्डर (Site Builder)

**फोकस:** एक ही प्रॉम्प्ट से एंड-टू-एंड वेबसाइट बनाना।

**उपलब्ध टूल्स:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`

**यह क्या अच्छा करता है:**
- वर्णित व्यवसाय प्रकार के लिए एक मल्टी-फेज साइट बिल्ड प्लान बनाना
- प्रत्येक चरण को स्वायत्त रूप से निष्पादित करना — संरचना, सामग्री, नेविगेशन, डिज़ाइन
- मैन्युअल हस्तक्षेप की आवश्यकता के बिना योजना के बीच में त्रुटियों से उबरना
- बिल्ड के हिस्से के रूप में अनुशंसित प्लगइन्स इंस्टॉल करना
- चैट इंटरफ़ेस से सीधे संपर्क फ़ॉर्म बनाना (Superdav AI Agent v1.10.0+)

**शुरुआती सुझाव:**
- *एक गैलरी पोस्ट टाइप, एक बुकिंग पेज और एक संपर्क फ़ॉर्म के साथ एक फ़ोटोग्राफ़ी पोर्टफोलियो साइट बनाएं।*
- *ऑनलाइन मेनू, खुलने के समय और एक टेबल-बुकिंग पूछताछ फ़ॉर्म के साथ एक रेस्टोरेंट वेबसाइट बनाएं।*
- *सेवा पेजों, एक पोर्टफोलियो सेक्शन और एक ब्लॉग के साथ एक फ्रीलांस कंसल्टिंग साइट सेट करें।*
- *साइट बिल्डर का उपयोग करके Contact पेज पर एक संपर्क फ़ॉर्म जोड़ें।*

---

### डिज़ाइन स्टूडियो (Design Studio)

**फोकस:** विज़ुअल कस्टमाइज़ेशन — रंग, टाइपोग्राफी, CSS और ब्लॉक पैटर्न।

**उपलब्ध टूल्स:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**यह क्या अच्छा करता है:**
- नामित थीम प्रीसेट लागू करना (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- theme.json के माध्यम से ग्लोबल टाइपोग्राफी और कलर पैलेट को फ़ाइन-ट्यून करना
- ब्रांड-विशिष्ट ओवरराइड के लिए कस्टम CSS इंजेक्ट करना
- एक पेज का स्क्रीनशॉट लेना और डिज़ाइन समस्याओं के लिए उसकी समीक्षा करना

**शुरुआती सुझाव:**
- *warm-editorial प्रीसेट लागू करें और फिर प्राथमिक रंग को #2d6a4f पर सेट करें।*
- *होमपेज का स्क्रीनशॉट लें और मुझे बताएं कि आप क्या सुधार करेंगे।*
- *एक फुल-विड्थ बैकग्राउंड इमेज और सेंटर्ड हेडिंग के साथ एक पुन: प्रयोज्य हीरो ब्लॉक पैटर्न बनाएं।*

---

### प्लगइन मैनेजर (Plugin Manager)

**फोकस:** WordPress प्लगइन्स को खोजना, इंस्टॉल करना और प्रबंधित करना।

**उपलब्ध टूल्स:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**यह क्या अच्छा करता है:**
- वर्णित उपयोग के मामले के लिए सबसे अच्छा प्लगइन सुझाना
- रजिस्ट्री से एबिलिटी पैक इंस्टॉल करना
- श्रेणी के अनुसार उपलब्ध एबिलिटी कैटलॉग ब्राउज़ करना

**शुरुआती सुझाव:**
- *सदस्यता निर्देशिका (membership directory) के लिए सबसे अच्छा प्लगइन क्या है?*
- *WooCommerce एबिलिटी पैक इंस्टॉल करें।*
- *सभी उपलब्ध ई-कॉमर्स एबिलिटी पैक दिखाएं।*

---

### सपोर्ट असिस्टेंट (Support Assistant)

**फोकस:** साइट की सामग्री, सेटिंग्स और WordPress कॉन्फ़िगरेशन के बारे में सवालों के जवाब देना।

**उपलब्ध टूल्स:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**यह क्या अच्छा करता है:**
- वर्तमान साइट सेटिंग्स और विकल्पों को देखना
- यह समझाना कि साइट पर कौन से पोस्ट टाइप, टैक्सोनॉमी और मेनू कॉन्फ़िगर किए गए हैं
- लाइव मान पढ़कर "यह सेटिंग क्या करती है?" जैसे सवालों का जवाब देना
- बदलाव करने से पहले एक रीड-ओनली डायग्नोस्टिक लेयर के रूप में कार्य करना

**शुरुआती सुझाव:**
- *इस साइट पर वर्तमान में कौन से प्लगइन्स और सेटिंग्स सक्रिय हैं?*
- *इस साइट पर पंजीकृत सभी कस्टम पोस्ट टाइप सूचीबद्ध करें।*
- *कौन से नेविगेशन मेनू मौजूद हैं और उन्हें कहाँ असाइन किया गया है?*

## एजेंट्स को कस्टमाइज़ करना

प्रत्येक इनबिल्ट एजेंट को `gratis_ai_agent_agents` फ़िल्टर के माध्यम से बढ़ाया या बदला जा सकता है।

### किसी मौजूदा एजेंट में कस्टम सिस्टम प्रॉम्प्ट जोड़ना

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### एक नया एजेंट रजिस्टर करना

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

नया एजेंट फ़िल्टर चलने के तुरंत बाद एजेंट पिकर में दिखाई देता है।

### एक इनबिल्ट एजेंट को हटाना

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
