# इंटरनेट सर्च

Gratis AI Agent v1.5.0 में **इंटरनेट सर्च** की सुविधा जोड़ी गई है, जो AI असिस्टेंट को बातचीत के दौरान वेब से ताज़ा जानकारी प्राप्त करने देती है। यह सुविधा [Brave Search API](https://brave.com/search/api/) या [Tavily API](https://tavily.com/) द्वारा संचालित होती है।

## यह कैसे काम करता है

जब इंटरनेट सर्च सक्षम होता है, तो असिस्टेंट स्वचालित रूप से आपके कॉन्फ़िगर किए गए सर्च प्रोवाइडर से क्वेरी करता है, जब उसे लगता है कि किसी सवाल के लिए वर्तमान या बाहरी जानकारी की ज़रूरत है — उदाहरण के लिए, हाल की खबरें, लाइव मूल्य निर्धारण, या कोई दस्तावेज़ीकरण जो मॉडल के ट्रेनिंग कटऑफ के बाद बदल गया हो।

परिणाम वास्तविक समय में प्राप्त किए जाते हैं और जवाब उत्पन्न करने से पहले असिस्टेंट के कॉन्टेक्स्ट में डाल दिए जाते हैं। असिस्टेंट यह बताता है कि उसने सवाल का जवाब देने के लिए सर्च परिणामों का उपयोग किया है।

## इंटरनेट सर्च को सक्षम करना

इंटरनेट सर्च के लिए आपके चुने हुए सर्च प्रोवाइडर से एक API key की ज़रूरत होती है। इसे कॉन्फ़िगर करने के लिए:

1. **Gratis AI Agent → Settings → Advanced** पर जाएँ।
2. **Internet Search Provider** ड्रॉपडाउन का पता लगाएँ और या तो **Brave Search** या **Tavily** चुनें।
3. संबंधित फ़ील्ड में अपना API key दर्ज करें। प्रत्येक फ़ील्ड के बगल में साइन-अप लिंक प्रदर्शित होते हैं।
4. **Save Settings** पर क्लिक करें।

एक बार key सेव हो जाने पर, इंटरनेट सर्च की सुविधा स्वचालित रूप से असिस्टेंट के लिए उपलब्ध हो जाती है।

## Brave Search

### Brave Search API Key प्राप्त करना

1. [Brave Search API page](https://brave.com/search/api/) पर जाएँ।
2. एक प्लान के लिए साइन अप करें। मासिक अनुरोध सीमा के साथ एक फ्री टियर उपलब्ध है।
3. Brave Search developer dashboard से अपना API key कॉपी करें।
4. इसे Gratis AI Agent सेटिंग्स में **Brave Search API Key** फ़ील्ड में पेस्ट करें।

### उपयोग सीमाएँ

उपयोग Brave Search द्वारा किए गए क्वेरी की संख्या के आधार पर बिल किया जाता है। प्रत्येक AI जवाब जो सर्च को ट्रिगर करता है, उसे एक क्वेरी गिना जाता है। अप्रत्याशित शुल्क से बचने के लिए [Brave Search developer dashboard](https://brave.com/search/api/) में अपने उपयोग की निगरानी करें।

## Tavily

Superdav AI Agent v1.10.0 में **Tavily** को एक वैकल्पिक इंटरनेट सर्च प्रोवाइडर के रूप में जोड़ा गया है, जो बेहतर सर्च परिणाम और उन्नत रिसर्च क्षमताएं प्रदान करता है।

### Tavily API Key प्राप्त करना

1. [Tavily API page](https://tavily.com/) पर जाएँ।
2. एक अकाउंट के लिए साइन अप करें। मासिक अनुरोध सीमा के साथ एक फ्री टियर उपलब्ध है।
3. Tavily dashboard से अपना API key कॉपी करें।
4. इसे Gratis AI Agent सेटिंग्स में **Tavily API Key** फ़ील्ड में पेस्ट करें।

### उपयोग सीमाएँ

उपयोग Tavily द्वारा किए गए API कॉल की संख्या के आधार पर बिल किया जाता है। प्रत्येक AI जवाब जो सर्च को ट्रिगर करता है, उसे एक कॉल गिना जाता है। अप्रत्याशित शुल्क से बचने के लिए [Tavily dashboard](https://tavily.com/) में अपने उपयोग की निगरानी करें।

## इंटरनेट सर्च को अक्षम करना

सक्रिय सर्च प्रोवाइडर फ़ील्ड से API key हटा दें और सेव करें। इंटरनेट सर्च की सुविधा असिस्टेंट को और नहीं मिलेगी।

:::note
इंटरनेट सर्च जवाबों में देरी (latency) जोड़ता है क्योंकि जवाब देने से पहले असिस्टेंट को सर्च परिणामों का इंतज़ार करना पड़ता है। समय-संवेदनशील उपयोग के मामलों के लिए, विचार करें कि क्या वास्तविक समय सर्च ज़रूरी है या असिस्टेंट का इन-बिल्ट ज्ञान पर्याप्त है।
:::
