# WP Engine इंटीग्रेशन

## अवलोकन
WP Engine एक प्रीमियम मैनेज्ड WordPress होस्टिंग प्लेटफॉर्म है जो WordPress साइट्स के लिए बेहतर परफॉर्मेंस, सुरक्षा और स्केलेबिलिटी प्रदान करता है। यह इंटीग्रेशन Ultimate Multisite और WP Engine के बीच ऑटोमैटिक डोमेन सिंकिंग को सक्षम करता है।

## विशेषताएं
- ऑटोमैटिक डोमेन सिंकिंग
- मल्टीसाइट इंस्टॉलेशन के लिए सबडोमेन सपोर्ट
- WP Engine के मौजूदा सिस्टम के साथ सहज इंटीग्रेशन

## आवश्यकताएं
यह इंटीग्रेशन ऑटोमैटिक रूप से पता लगाता है कि आप WP Engine पर होस्ट कर रहे हैं या नहीं और बिल्ट-इन WP Engine API का उपयोग करता है। अगर WP Engine plugin एक्टिव और सही तरीके से कॉन्फ़िगर है, तो किसी अतिरिक्त कॉन्फ़िगरेशन की जरूरत नहीं है।

हालांकि, अगर आपको इंटीग्रेशन को मैन्युअली कॉन्फ़िगर करना है, तो आप अपनी `wp-config.php` फाइल में इनमें से एक constant डिफाइन कर सकते हैं:

```php
define('WPE_APIKEY', 'your_api_key'); // पसंदीदा तरीका
// या
define('WPE_API', 'your_api_key'); // वैकल्पिक तरीका
```

## सेटअप निर्देश

### 1. WP Engine Plugin की जांच करें

अगर आप WP Engine पर होस्ट कर रहे हैं, तो WP Engine plugin पहले से इंस्टॉल और एक्टिवेट होना चाहिए। सुनिश्चित करें कि:

1. WP Engine plugin एक्टिव है
2. फाइल `wp-content/mu-plugins/wpengine-common/class-wpeapi.php` मौजूद है

### 2. इंटीग्रेशन को इनेबल करें

1. अपने WordPress admin में, Ultimate Multisite > Settings पर जाएं
2. "Domain Mapping" टैब पर जाएं
3. "Host Integrations" तक स्क्रॉल करें
4. WP Engine इंटीग्रेशन को इनेबल करें
5. "Save Changes" पर क्लिक करें

## यह कैसे काम करता है

### डोमेन सिंकिंग

जब Ultimate Multisite में कोई डोमेन मैप किया जाता है:

1. इंटीग्रेशन WP Engine API का उपयोग करके डोमेन को आपके WP Engine इंस्टॉलेशन में जोड़ता है
2. WP Engine डोमेन कॉन्फ़िगरेशन और SSL सर्टिफिकेट जारी करने का काम संभालता है
3. जब कोई डोमेन मैपिंग हटाई जाती है, तो इंटीग्रेशन उस डोमेन को WP Engine से हटा देगा

### सबडोमेन सपोर्ट

सबडोमेन मल्टीसाइट इंस्टॉलेशन के लिए:

1. जब कोई नई साइट बनाई जाती है, तो इंटीग्रेशन प्रत्येक सबडोमेन को WP Engine में जोड़ता है
2. WP Engine सबडोमेन कॉन्फ़िगरेशन संभालता है
3. जब कोई साइट डिलीट की जाती है, तो इंटीग्रेशन उस सबडोमेन को WP Engine से हटा देगा

## महत्वपूर्ण नोट्स

### Wildcard डोमेन

सबडोमेन मल्टीसाइट इंस्टॉलेशन के लिए, WP Engine सपोर्ट से संपर्क करके wildcard डोमेन कॉन्फ़िगरेशन का अनुरोध करने की सलाह दी जाती है। इससे सभी सबडोमेन ऑटोमैटिक रूप से काम करेंगे और आपको प्रत्येक को अलग से जोड़ने की जरूरत नहीं होगी।

### SSL सर्टिफिकेट

WP Engine इस इंटीग्रेशन के माध्यम से जोड़े गए सभी डोमेन के लिए SSL सर्टिफिकेट जारी करने और रिन्यू करने का काम ऑटोमैटिक रूप से संभालता है। किसी अतिरिक्त कॉन्फ़िगरेशन की जरूरत नहीं है।

## समस्या निवारण

### API कनेक्शन समस्याएं
- सुनिश्चित करें कि WP Engine plugin एक्टिव और सही तरीके से कॉन्फ़िगर है
- अगर आपने API key मैन्युअली डिफाइन की है, तो जांचें कि वह सही है
- अगर API में कोई समस्या आ रही है, तो WP Engine सपोर्ट से संपर्क करें

### डोमेन नहीं जुड़ा
- किसी भी error message के लिए Ultimate Multisite logs देखें
- सुनिश्चित करें कि डोमेन पहले से WP Engine में नहीं जोड़ा गया है
- सुनिश्चित करें कि आपका WP Engine प्लान उतने डोमेन को सपोर्ट करता है जितने आप जोड़ रहे हैं

### सबडोमेन समस्याएं
- अगर सबडोमेन काम नहीं कर रहे हैं, तो wildcard डोमेन कॉन्फ़िगरेशन के लिए WP Engine सपोर्ट से संपर्क करें
- सुनिश्चित करें कि मुख्य डोमेन और सबडोमेन के लिए आपकी DNS सेटिंग्स सही तरीके से कॉन्फ़िगर हैं
