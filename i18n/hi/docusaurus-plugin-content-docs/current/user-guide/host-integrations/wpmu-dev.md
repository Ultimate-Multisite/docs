# WPMU DEV इंटीग्रेशन

## अवलोकन
WPMU DEV एक व्यापक WordPress प्लेटफॉर्म है जो WordPress साइट्स के लिए होस्टिंग, plugins और सेवाएं प्रदान करता है। यह इंटीग्रेशन Ultimate Multisite और WPMU DEV होस्टिंग के बीच स्वचालित डोमेन सिंकिंग और SSL सर्टिफिकेट प्रबंधन को सक्षम बनाता है।

## विशेषताएं
- स्वचालित डोमेन सिंकिंग
- SSL सर्टिफिकेट प्रबंधन
- SSL सर्टिफिकेट वेरिफिकेशन के लिए अधिक प्रयास

## आवश्यकताएं
यह इंटीग्रेशन स्वचालित रूप से पता लगाता है कि आप WPMU DEV पर होस्ट कर रहे हैं या नहीं और बिल्ट-इन API का उपयोग करता है। अगर आप WPMU DEV पर होस्ट कर रहे हैं तो किसी अतिरिक्त कॉन्फ़िगरेशन की आवश्यकता नहीं है।

यह इंटीग्रेशन `WPMUDEV_HOSTING_SITE_ID` constant की उपस्थिति की जांच करता है, जो WPMU DEV पर होस्ट करते समय स्वचालित रूप से परिभाषित हो जाता है।

## सेटअप निर्देश

### 1. WPMU DEV होस्टिंग की पुष्टि करें

अगर आप WPMU DEV पर होस्ट कर रहे हैं, तो आवश्यक constants पहले से परिभाषित होने चाहिए। सुनिश्चित करें कि:

1. `WPMUDEV_HOSTING_SITE_ID` constant आपके environment में परिभाषित है
2. आपके पास API एक्सेस के साथ एक सक्रिय WPMU DEV सदस्यता है

### 2. इंटीग्रेशन सक्षम करें

1. अपने WordPress admin में, Ultimate Multisite > Settings पर जाएं
2. "Domain Mapping" टैब पर जाएं
3. "Host Integrations" तक स्क्रॉल करें
4. WPMU DEV इंटीग्रेशन को सक्षम करें
5. "Save Changes" पर क्लिक करें

## यह कैसे काम करता है

### डोमेन सिंकिंग

जब Ultimate Multisite में कोई डोमेन मैप किया जाता है:

1. इंटीग्रेशन WPMU DEV API का उपयोग करके डोमेन को आपके होस्टिंग अकाउंट में जोड़ता है
2. यह डोमेन का www वर्जन भी स्वचालित रूप से जोड़ता है
3. WPMU DEV डोमेन कॉन्फ़िगरेशन और SSL सर्टिफिकेट जारी करने का काम संभालता है

### SSL सर्टिफिकेट प्रबंधन

यह इंटीग्रेशन WPMU DEV होस्टिंग के लिए SSL सर्टिफिकेट वेरिफिकेशन प्रयासों की संख्या बढ़ाने के लिए कॉन्फ़िगर किया गया है, क्योंकि SSL सर्टिफिकेट जारी और इंस्टॉल होने में कुछ समय लग सकता है। डिफ़ॉल्ट रूप से, यह SSL सर्टिफिकेट वेरिफिकेशन के लिए 10 बार तक प्रयास करेगा, जबकि मानक 5 प्रयास होते हैं।

## महत्वपूर्ण नोट्स

### डोमेन हटाना

वर्तमान में, WPMU DEV API डोमेन हटाने का तरीका प्रदान नहीं करता है। जब Ultimate Multisite में डोमेन मैपिंग हटा दी जाती है, तो डोमेन आपके WPMU DEV होस्टिंग अकाउंट में बना रहेगा। यदि आवश्यक हो तो आपको इसे WPMU DEV होस्टिंग dashboard से मैन्युअल रूप से हटाना होगा।

### API प्रमाणीकरण

यह इंटीग्रेशन WPMU DEV API key का उपयोग करता है जो आपके WordPress database में `wpmudev_apikey` option के रूप में संग्रहीत है। जब आप अपनी साइट को WPMU DEV से कनेक्ट करते हैं तो यह स्वचालित रूप से सेट हो जाता है।

## समस्या निवारण

### API कनेक्शन समस्याएं
- सुनिश्चित करें कि आपकी साइट WPMU DEV से ठीक से कनेक्ट है
- जांचें कि `wpmudev_apikey` option आपके WordPress database में सेट है
- सुनिश्चित करें कि आपकी WPMU DEV सदस्यता सक्रिय है

### SSL सर्टिफिकेट समस्याएं
- WPMU DEV को SSL सर्टिफिकेट जारी करने में कुछ समय लग सकता है (आमतौर पर 5-15 मिनट)
- यह इंटीग्रेशन SSL सर्टिफिकेट के लिए 10 बार तक जांच करने के लिए कॉन्फ़िगर किया गया है
- अगर कई प्रयासों के बाद भी SSL सर्टिफिकेट जारी नहीं हो रहे हैं, तो WPMU DEV सपोर्ट से संपर्क करें

### डोमेन नहीं जोड़ा गया
- किसी भी error messages के लिए Ultimate Multisite logs की जांच करें
- सुनिश्चित करें कि डोमेन पहले से WPMU DEV में नहीं जोड़ा गया है
- सुनिश्चित करें कि आपका WPMU DEV होस्टिंग प्लान आपके द्वारा जोड़े जा रहे डोमेन की संख्या को सपोर्ट करता है
