# सहायता कैसे प्राप्त करें

आपके पास वैध License key है, और इसके साथ आपको हमारी टीम से सहायता प्राप्त करने का अधिकार भी मिलता है। Ultimate Multisite सेटअप करते समय जो भी समस्याएँ आएँ, उन्हें हल करने में हम आपकी मदद करेंगे।

इस लेसन में हम बताएँगे कि license धारकों के लिए कौन-कौन से support चैनल उपलब्ध हैं और हमारी support टीम से संपर्क करने के लिए आपको कौन सी जानकारी तैयार रखनी होगी।

## Help Site

हमारी help site पर आपको विभिन्न लेख और tutorials मिलेंगे। नीचे दिए गए लिंक से आप इसे एक्सेस कर सकते हैं।

[)

हम अपने users को सलाह देते हैं कि वे Ultimate Multisite की बुनियादी सुविधाओं और कार्यों से परिचित हो जाएँ। हमारी help site पर "Getting Started" लेख उपलब्ध हैं और आम सवालों के जवाब भी वहाँ मिल जाएँगे — जैसे add-ons कैसे install करें, payment gateways को कैसे integrate करें, आदि।

## Support से संपर्क करें

अगर आप कहीं अटक जाएँ और help site पर जवाब न मिले, तो आप हमारी support टीम से संपर्क कर सकते हैं। इसके लिए हमारी website के दाईं ओर नीचे दिखने वाले speech bubble आइकन पर क्लिक करें या सीधे support@ultimatemultisite.com पर email भेजें।

तकनीकी समस्याओं के लिए support से संपर्क करते समय, कृपया अपनी समस्या का विस्तृत विवरण दें — साथ में screenshot, error messages, या कोई भी ऐसी जानकारी जो हमारी support टीम को आपकी समस्या समझने और उसका मूल्यांकन करने में मदद करे। आपको अपनी license key भी देनी होगी। केवल सक्रिय license धारक ही support प्राप्त करने के हकदार हैं।

## Community Site

हमारी [community site](https://community.ultimatemultisite.com/) Ultimate Multisite users के लिए एक-दूसरे से जुड़ने का मंच है। यह एक ऐसा प्लेटफॉर्म है जहाँ सदस्य पोस्ट कर सकते हैं, चर्चाओं पर टिप्पणी कर सकते हैं, सलाह दे सकते हैं, सहयोग कर सकते हैं, या मिलकर समाधान खोज सकते हैं। Private spaces/pages को अनलॉक करने के लिए आपको अपनी Ultimate Multisite license key देनी होगी।
