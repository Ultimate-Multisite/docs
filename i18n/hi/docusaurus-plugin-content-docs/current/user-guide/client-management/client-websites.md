# आपके ग्राहक की वेबसाइटें (v2) {#your-clients-websites-v2}

आपके नेटवर्क पर पंजीकरण करने के बाद, आपके ग्राहक अपने Dashboard तक पहुँच सकते हैं।

**Account** पृष्ठ के अलावा, जहाँ वे अपने भुगतान, सदस्यता, डोमेन, प्लान सीमाएँ और अन्य महत्वपूर्ण जानकारी देख सकते हैं, उन्हें **मेरी साइटें** पृष्ठ तक भी पहुँच मिलती है, जहाँ वे अपनी सभी वेबसाइटें देख और प्रबंधित कर सकते हैं। उन्हें बस अपने Dashboard पर मेरी साइटें पर क्लिक करना होता है।

![ग्राहक Dashboard में मेरी साइटें लिंक](/img/account-page/my-sites-link.png)

उन्हें उनकी सभी वेबसाइटें दिखाई जाएँगी, जिनमें हर वेबसाइट के ठीक नीचे **प्रबंधित करें** बटन होगा। अगर वे उस पर क्लिक करते हैं, तो उन्हें **इस विशेष वेबसाइट के Dashboard** पर भेज दिया जाएगा।

वे यह भी देख सकते हैं कि जिन प्लान में एक से अधिक वेबसाइट रखने की अनुमति है, उनमें उनकी कौन-सी वेबसाइट **प्राथमिक** है।

![प्रबंधित करें बटन के साथ ग्राहक वेबसाइटें दिखाता मेरी साइटें पृष्ठ](/img/account-page/my-sites-page.png)

यहाँ साइट संपादन इंटरफ़ेस का पूरा दृश्य है:

![साइट संपादन का पूरा इंटरफ़ेस](/img/admin/site-edit-full.png)

अगर वे अपनी प्राथमिक वेबसाइट बदलना चाहते हैं, तो उन्हें **Account** मेनू पर क्लिक करके Account पृष्ठ पर जाना होगा।

![ग्राहक Dashboard में Account मेनू लिंक](/img/account-page/account-menu.png)

आप ग्राहक विवरण भी देख और संपादित कर सकते हैं:

![ग्राहक संपादन इंटरफ़ेस](/img/admin/customer-edit.png)

फिर, उन्हें **डिफ़ॉल्ट साइट बदलें** पर क्लिक करना चाहिए।

![Account पृष्ठ पर डिफ़ॉल्ट साइट बदलें बटन](/img/account-page/change-default-site.png)

एक पॉप-अप विंडो दिखाई देगी। ड्रॉप-डाउन सूची पर क्लिक करें और चुनें कि आप किस वेबसाइट को प्राथमिक वेबसाइट बनाना चाहते हैं।

![डिफ़ॉल्ट साइट बदलें ड्रॉप-डाउन चयन](/img/account-page/change-default-site-dropdown.png)

फिर, बस **डिफ़ॉल्ट साइट बदलें** पर क्लिक करें और आपका काम हो गया!

![डिफ़ॉल्ट साइट बदलाव की पुष्टि](/img/account-page/change-default-site-confirm.png)
