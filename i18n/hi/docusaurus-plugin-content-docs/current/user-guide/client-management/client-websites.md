# आपके ग्राहकों की वेबसाइटें (v2)

आपके नेटवर्क पर रजिस्टर करने के बाद, आपके ग्राहक अपने dashboard तक पहुँच सकते हैं।

**Account** पेज के अलावा, जहाँ वे अपने भुगतान, सदस्यता, domains, प्लान की सीमाओं और अन्य महत्वपूर्ण जानकारी देख सकते हैं, उनके पास **My Sites** पेज तक भी पहुँच होती है, जहाँ वे अपनी सभी वेबसाइटें देख और प्रबंधित कर सकते हैं। उन्हें बस अपने dashboard में My Sites पर क्लिक करना होगा।

![ग्राहक dashboard में My Sites लिंक](/img/admin/customers-list.png)

उन्हें अपनी सभी वेबसाइटें दिखाई जाएंगी, और हर एक के नीचे एक **Manage** बटन होगा। अगर वे इस पर क्लिक करते हैं, तो उन्हें **उस विशेष वेबसाइट के dashboard** पर भेज दिया जाएगा।

वे यह भी देख सकते हैं कि उनकी कौन सी वेबसाइट **primary** है — उन प्लान्स के लिए जो उन्हें एक से अधिक वेबसाइट रखने की अनुमति देते हैं।

![My Sites पेज जिसमें ग्राहक की वेबसाइटें Manage बटन के साथ दिख रही हैं](/img/admin/customers-list.png)

अगर वे अपनी primary वेबसाइट बदलना चाहते हैं, तो उन्हें **Account** मेन्यू पर क्लिक करके Account पेज पर जाना होगा।

![ग्राहक dashboard में Account मेन्यू लिंक](/img/admin/customers-list.png)

फिर, उन्हें **Change Default Site** पर क्लिक करना चाहिए।

![Account पेज पर Change Default Site बटन](/img/admin/customers-list.png)

एक pop-up विंडो दिखाई देगी। drop-down लिस्ट पर क्लिक करें और चुनें कि आप कौन सी वेबसाइट को primary वेबसाइट बनाना चाहते हैं।

![Change Default Site dropdown चयन](/img/admin/customers-list.png)

फिर, बस **Change Default Site** पर क्लिक करें और हो गया!

![Default site परिवर्तन की पुष्टि](/img/admin/customers-list.png)
