# पहली बार अपने अकाउंट पेज को कैसे एक्सेस करें?

Ultimate Multisite में आपका स्वागत है!

हमारा plugin खरीदने के बाद आपको ईमेल में जो लिंक मिला है, उसके ज़रिए अपने अकाउंट पेज पर जाएं:

<!-- Screenshot unavailable: Account page email with access link -->

खरीदारी के समय दिए गए ईमेल और ईमेल में मिले पासवर्ड का इस्तेमाल करें।

<!-- Screenshot unavailable: Login page with email and password fields -->

चूंकि आप पहली बार इस अकाउंट में लॉग इन कर रहे हैं, आपसे पासवर्ड बदलने के लिए कहा जाएगा। पासवर्ड बदलें और _Save_ बटन पर क्लिक करें।

<!-- Screenshot unavailable: Change password form with Save button -->

अपने अकाउंट पेज पर आप बिलिंग, रिन्यूअल, कैंसिलेशन और कई अन्य चीज़ों जैसी सभी अकाउंट डिटेल्स मैनेज कर सकते हैं।
