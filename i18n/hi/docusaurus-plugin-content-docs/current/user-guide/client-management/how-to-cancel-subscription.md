# मैं अपनी सदस्यता कैसे रद्द कर सकता हूँ?

अगर आपके पास Ultimate Multisite लाइसेंस है, तो आप जब चाहें इसके नवीनीकरण को रद्द कर सकते हैं। बस नीचे दिए गए चरणों का पालन करें:


<!-- Screenshot unavailable: Account page email with access link -->

खरीदारी के समय दिए गए ईमेल और अपने पासवर्ड का उपयोग करें।

<!-- Screenshot unavailable: Login page with email and password fields -->

यह आपके Account Page की शुरुआती स्क्रीन है:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

अपनी सदस्यता रद्द करने के लिए, बाईं ओर के मेन्यू में _Renewing & Billing_ पर जाएँ।

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

एक साइड विंडो खोलने के लिए दाईं ओर के तीर पर क्लिक करें। फिर, आपको वह विकल्प चुनना होगा जो **auto-renew** को रद्द करेगा।

सिस्टम आपको एक पुष्टि संदेश दिखाएगा।

अगर आप सुनिश्चित हैं कि आप अपनी सदस्यता रद्द करना चाहते हैं, तो बस _**Cancel Renewals**_ बटन पर क्लिक करें। इस कार्रवाई के बाद, आपसे एक छोटा सर्वेक्षण भरने के लिए कहा जाएगा।

हो गया! आपकी सदस्यता अब स्वचालित रूप से नवीनीकृत नहीं होगी।

**आपकी सदस्यता समाप्त होने तक आपके पास एक वैध key रहेगी**। अगर आप अपनी सदस्यता फिर से शुरू करना चाहते हैं, तो आपको इसे मैन्युअल रूप से करना होगा।
