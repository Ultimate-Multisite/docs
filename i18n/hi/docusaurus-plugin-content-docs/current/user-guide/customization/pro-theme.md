# PRO Theme से अपने Admin Panel को कस्टमाइज़ करना

अक्सर जब हम क्लाइंट्स के लिए काम करते हैं, तो हमें WordPress के admin dashboard में उन्हें एक कस्टम अनुभव देने की ज़रूरत होती है। अगर आप एक ज़्यादा प्रोफेशनल लुक और फील की तलाश में हैं, तो PRO Theme आपकी मदद कर सकता है।

PRO Theme एक admin theme है जो आपके क्लाइंट्स के Dashboard को कस्टमाइज़ करने में आपकी मदद के लिए बनाया गया है। और यह Ultimate Multisite के premium plugins core का भी हिस्सा है।

इसे install करने के बाद, अपने _Plugins_ पेज पर जाएं और अपने Network पर इसे activate करने के लिए क्लिक करें।

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_अपने Plugins पेज पर इसे network-wide activate करें_

_**नोट:**_ _अगर आप इसे सिर्फ install करते हैं, तो यह आपके क्लाइंट्स को एक विकल्प के रूप में दिखाई देगा जिसे वे सिर्फ अपने user के लिए theme जोड़ने के लिए चुन सकते हैं।_

अब, आपका admin interface पूरी तरह से नए डिज़ाइन में है! आपके पास बाईं ओर के column में PRO Theme नाम का एक नया menu भी है।
