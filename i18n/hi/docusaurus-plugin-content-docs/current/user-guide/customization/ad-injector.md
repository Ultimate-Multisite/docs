# Ultimate Multisite: Ad Injector के साथ अपने नेटवर्क पर विज्ञापन लगाना

Ultimate Multisite: Ad Injector एक **मुफ्त add-on** है जो आपको सीधे plan के edit page से विज्ञापन लगाने की सुविधा देता है।

Ultimate Multisite: Ad Injector इंस्टॉल करने के बाद, इसे activate करने के लिए अपने _Plugins_ पेज पर जाएं।

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Network Activate बटन पर क्लिक करें_

Add-on active होने के बाद, plan edit स्क्रीन पर _Advanced Options_ meta box में एक नया option दिखेगा जिसका नाम _Ad Settings_ है।

![Plan edit स्क्रीन पर Ad Settings](/img/config/product-edit-full.png)

_अब आप अपने हर plan के लिए ad settings को edit कर सकते हैं_

इस tab में, आप front-end पर posts और pages के लिए content से पहले और बाद में ad codes जोड़ सकते हैं।

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_अपने विज्ञापन का code डालने के बाद, यह उस खास plan के posts और pages पर दिखाई देगा।_

आप उस plan के users के लिए Admin Pages के header में भी विज्ञापन शामिल कर सकते हैं।

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_अगर आप अपने user के back-end पर विज्ञापन दिखाना चाहते हैं तो Admin Panel Ad Code option पर जाएं_

### विज्ञापन से परे

आप Ultimate Multisite: Ad Injector का उपयोग high-tier plans के लिए सपोर्ट देने के लिए भी कर सकते हैं।

बस _Admin Panel Ads_ option पर जाएं और अपने support tool का HTML code जोड़ें।
