# ईमेल और ब्रॉडकास्ट भेजना (v2)

_**महत्वपूर्ण सूचना: यह लेख Ultimate Multisite संस्करण 2.x के लिए है।**_

Ultimate Multisite में एक ऐसी सुविधा है जो आपको अपने ग्राहकों से संपर्क करने की अनुमति देती है - आप किसी विशेष उपयोगकर्ता या उपयोगकर्ताओं के समूह को ईमेल भेज सकते हैं, साथ ही उनके admin dashboard पर घोषणाएं प्रसारित करने के लिए नोटिस भी भेज सकते हैं।

## Broadcasts के ज़रिए अपने ग्राहकों के dashboard पर admin notices जोड़ें

Ultimate Multisite की broadcast सुविधा का उपयोग करके, आप अपने उपयोगकर्ता की subsite admin dashboard पर **admin notices** जोड़ सकते हैं।

यह बेहद उपयोगी है जब आपको सिस्टम रखरखाव जैसी घोषणा करनी हो या अपने मौजूदा उपयोगकर्ताओं को नए उत्पाद या सेवाएं देनी हों। आपके उपयोगकर्ता के dashboard पर admin notice इस तरह दिखाई देगी।

![ग्राहक के dashboard पर दिखाई देने वाला Admin notice broadcast](/img/admin/broadcasts-list.png)

Admin notice शुरू करने के लिए, अपने network admin dashboard पर जाएं और **Ultimate Multisite** मेन्यू के अंतर्गत, आपको **Broadcasts** विकल्प मिलेगा।

![Ultimate Multisite admin में Broadcasts मेन्यू](/img/admin/broadcasts-list.png)

इस पेज पर, ऊपर **Add Broadcast** बटन पर क्लिक करें।

इससे Add broadcast मॉडल विंडो खुलेगी जहां आप चुन सकते हैं कि आप किस प्रकार का broadcast भेजना चाहते हैं।

आगे बढ़ें और **Message** चुनें फिर **Next Step** बटन पर क्लिक करें।

![Message प्रकार चयनित के साथ Add broadcast मॉडल](/img/admin/broadcasts-list.png)

अगली विंडो आपसे **Target customer** या **Target product** के बारे में पूछेगी। ध्यान दें कि आप एक से अधिक उपयोगकर्ता या एक से अधिक उत्पाद चुन सकते हैं।

किसी उपयोगकर्ता खाते या उत्पाद को खोजने के लिए, फ़ील्ड के अंदर कीवर्ड टाइप करना शुरू करें।

**Message type** फ़ील्ड के अंतर्गत, आप नोटिस का रंग चुन सकते हैं। इससे आपके संदेश की तात्कालिकता पर ज़ोर दिया जाएगा।

फिर आप **Next Step** पर क्लिक कर सकते हैं।

![Broadcast के लिए Target customer और product चयन](/img/admin/broadcasts-list.png)

अगली विंडो वह है जहां आप विषय और वह सामग्री/संदेश दर्ज करके अपना संदेश लिखना शुरू कर सकते हैं जो आप उपयोगकर्ताओं को प्रसारित करना चाहते हैं।

![Broadcast संदेश विषय और सामग्री एडिटर](/img/admin/broadcasts-list.png)

अपना संदेश बनाने के बाद, आप **Send** बटन दबा सकते हैं।

बस इतना ही। Admin notice तुरंत आपके उपयोगकर्ता के dashboard पर दिखाई देनी चाहिए।

## अपने ग्राहकों को ईमेल भेजें

Ultimate Multisite की broadcast सुविधा का उपयोग करके, आप अपने उपयोगकर्ताओं को ईमेल भेज सकते हैं। आपके पास केवल विशिष्ट उपयोगकर्ताओं को ईमेल भेजने या उनके द्वारा सब्सक्राइब किए गए उत्पाद या योजना के आधार पर किसी विशेष उपयोगकर्ता समूह को लक्षित करने का विकल्प है।

ईमेल broadcast शुरू करने के लिए, अपने network admin dashboard पर जाएं और Ultimate Multisite मेन्यू के अंतर्गत, आपको Broadcast विकल्प मिलेगा।

![Ultimate Multisite admin में Broadcasts पेज](/img/admin/broadcasts-list.png)

इस पेज पर, ऊपर **Add broadcast** बटन पर क्लिक करें।

इससे Add broadcast मॉडल विंडो खुलेगी जहां आप चुन सकते हैं कि आप किस प्रकार का broadcast भेजना चाहते हैं। आगे बढ़ें और **Email** चुनें फिर **Next Step** बटन पर क्लिक करें।

![Email प्रकार चयनित के साथ Add broadcast मॉडल](/img/admin/broadcasts-list.png)

अगली विंडो आपसे **Target customer** या **Target product** के बारे में पूछेगी। ध्यान दें कि आप एक से अधिक उपयोगकर्ता या एक से अधिक उत्पाद चुन सकते हैं।

किसी उपयोगकर्ता खाते या उत्पाद को खोजने के लिए, फ़ील्ड के अंदर कीवर्ड टाइप करना शुरू करें।

एक बार आपके लक्षित दर्शक चुने जाने के बाद, आप **Next Step** पर क्लिक कर सकते हैं।

![Email broadcast के लिए Target customer और product चयन](/img/admin/broadcasts-list.png)

अगली विंडो वह है जहां आप विषय और वह सामग्री/संदेश दर्ज करके अपना ईमेल लिखना शुरू कर सकते हैं जो आप उपयोगकर्ताओं को भेजना चाहते हैं।

![Email broadcast विषय और सामग्री एडिटर](/img/admin/broadcasts-list.png)

अपना संदेश बनाने के बाद, आप **Send** बटन दबा सकते हैं।

और broadcast सुविधा का उपयोग करके अपने अंतिम उपयोगकर्ताओं को ईमेल भेजना इतना आसान है।

## System emails

Ultimate Multisite में System emails वे **स्वचालित सूचनाएं** हैं जो पंजीकरण, भुगतान, domain mapping आदि जैसी कुछ कार्रवाइयों के बाद सिस्टम द्वारा भेजी जाती हैं। इन ईमेल को Ultimate Multisite सेटिंग्स से संपादित या संशोधित किया जा सकता है। इसमें एक ऐसी सुविधा भी है जो आपको किसी अन्य Ultimate Multisite इंस्टॉलेशन से मौजूदा सेटिंग्स को रीसेट और इम्पोर्ट करने की अनुमति देती है।

### रीसेट और इम्पोर्ट करना

नए Ultimate Multisite संस्करण, साथ ही add-ons, समय-समय पर नए ईमेल रजिस्टर कर सकते हैं और करेंगे।

विरोधों और अन्य समस्याओं को रोकने के लिए, **हम नए ईमेल टेम्पलेट्स को आपके इंस्टॉल पर स्वचालित रूप से System Emails के रूप में नहीं जोड़ेंगे**, जब तक कि वे किसी दी गई सुविधा के सही कामकाज के लिए महत्वपूर्ण न हों।

हालांकि, super admins और agents इम्पोर्टर टूल के माध्यम से इन नए रजिस्टर किए गए ईमेल को इम्पोर्ट कर सकते हैं। यह प्रक्रिया नए ईमेल टेम्पलेट की सामग्री और कॉन्फ़िगरेशन के साथ एक नया system email बनाएगी, जिससे super admin को कोई भी संशोधन करने या उन्हें वैसे ही रखने की अनुमति मिलेगी।

#### System emails कैसे इम्पोर्ट करें

अपने Ultimate Multisite Settings पेज पर जाएं और **Emails** टैब पर जाएं।

![Ultimate Multisite सेटिंग्स में Emails टैब](/img/config/settings-emails.png)

फिर, साइडबार पर, **Customize System Emails** बटन पर क्लिक करें।

![साइडबार पर Customize System Emails बटन](/img/config/settings-emails.png)

System Emails पेज पर, आपको ऊपर **Reset & Import** एक्शन बटन दिखाई देगा। उस बटन पर क्लिक करने से import और reset मॉडल विंडो खुलनी चाहिए।

![System Emails पेज पर Reset और Import एक्शन बटन](/img/config/settings-emails.png)

फिर, आप Import Emails विकल्पों को टॉगल करके देख सकते हैं कि कौन से system emails इम्पोर्ट के लिए उपलब्ध हैं।

![उपलब्ध system emails दिखाते Import Emails विकल्प](/img/config/settings-emails.png)

#### System Emails को रीसेट करना

कभी-कभी, आपको पता चलेगा कि किसी दिए गए ईमेल टेम्पलेट में आपने जो बदलाव किए हैं वे अब आपके लिए काम नहीं कर रहे हैं और आप इसे उनकी **डिफ़ॉल्ट स्थिति** में रीसेट करना चाहेंगे।

ऐसे मामलों में, आपके पास दो विकल्प हैं: आप बस system email को हटा सकते हैं और इसे वापस इम्पोर्ट कर सकते हैं (ऊपर दिए गए निर्देशों का उपयोग करके) - जो भेजने के मेट्रिक्स और अन्य चीजों को मिटा देगा, जिससे यह तरीका सबसे कम पसंदीदा है।

या आप उस ईमेल टेम्पलेट को रीसेट करने के लिए **Reset & Import टूल** का उपयोग कर सकते हैं।

किसी ईमेल टेम्पलेट को रीसेट करने के लिए, आप ऊपर दिए गए चरणों का पालन कर सकते हैं जब तक आप Reset & Import टूल तक नहीं पहुंच जाते, और फिर, **Reset** विकल्प को टॉगल करें और उन ईमेल को चुनें जिन्हें आप उनकी डिफ़ॉल्ट सामग्री पर वापस रीसेट करना चाहते हैं।

![ईमेल टेम्पलेट्स को डिफ़ॉल्ट पर पुनर्स्थापित करने के लिए Reset विकल्प](/img/config/settings-emails.png)
