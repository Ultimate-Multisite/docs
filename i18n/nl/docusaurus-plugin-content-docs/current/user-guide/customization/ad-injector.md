# Advertenties plaatsen op je netwerk met Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector is een **gratis add‑on** die je de tools geeft om advertenties op je plannen te plaatsen vanaf hun bewerkingspagina.

Na het installeren van Ultimate Multisite: Ad Injector, ga je naar je _Plugins_-pagina om het te activeren.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-KgDegqpbeb.png)

_Klik op de Network Activate-knop_

Zodra de add‑on actief is, heeft de meta box _Geavanceerde Opties_ op het bewerkingsscherm van het plan een nieuwe optie genaamd _Advertentie Instellingen_.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-DA7Z84yQTu.png)

_Nu kun je advertentie‑instellingen voor elk van je plannen bewerken_

Op die tab kun je de advertentiecodes voor berichten en pagina's in de front‑end toevoegen, vóór en na de inhoud.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-Rw9wObGUEN.png)

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-1TKkWh8qoF.png)

_Na het invoegen van je advertentiecode, verschijnt deze op de berichten en pagina's van dat specifieke plan._

Je kunt ook advertenties opnemen in de header van Admin Pages voor gebruikers van dat plan.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-kdpwvfMgTp.png)

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-1CPGJZn3km.png)

_Ga naar de optie Admin Panel Ad Code als je advertenties op de backend van je gebruikers wilt weergeven_

### Buiten Reclame

Je kunt ook Ultimate Multisite: Ad Injector gebruiken om ondersteuning te bieden aan high‑tier plannen.

Ga gewoon naar de optie _Admin Panel Advertenties_ en voeg de HTML‑code voor je ondersteuningshulpmiddel toe.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-hnOhLc8s8i.png)

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-09wylykj5l.png)

_Voeg klantenondersteuning toe aan specifieke plannen_

Op deze manier kunnen mensen je rechtstreeks bereiken via hun Admin Panel!
