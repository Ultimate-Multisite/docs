# Advertenties plaatsen op je netwerk met Ultimate Multisite: Ad Injector {#placing-ads-on-your-network-with-ultimate-multisite-ad-injector}

Ultimate Multisite: Ad Injector is een **gratis add-on** die je de tools geeft om advertenties op je abonnementen te plaatsen, rechtstreeks vanaf hun bewerkingspagina.

Ga na het installeren van Ultimate Multisite: Ad Injector naar je _Plugins_-pagina om het te activeren.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Klik op de knop Network Activate_

Zodra de add-on actief is, heeft de metabox _Advanced Options_ op het scherm voor het bewerken van abonnementen een nieuwe optie met de naam _Ad Settings_.

![Ad Settings op het scherm voor het bewerken van abonnementen](/img/config/product-edit-full.png)

_Nu kun je advertentie-instellingen bewerken voor elk van je abonnementen_

Op dat tabblad kun je de advertentiecodes toevoegen voor berichten en pagina’s in de front-end, vóór en na de inhoud.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Nadat je de code van je advertentie hebt ingevoegd, wordt deze weergegeven op de berichten en pagina’s van dat specifieke abonnement._

Je kunt ook advertenties laten opnemen in de header van Admin Pages voor gebruikers van dat abonnement.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Ga naar de optie Admin Panel Ad Code als je advertenties wilt weergeven in de back-end van je gebruiker_

### Verder dan adverteren {#beyond-advertising}

Je kunt Ultimate Multisite: Ad Injector ook gebruiken om ondersteuning aan te bieden aan abonnementen van een hoger niveau.

Ga gewoon naar de optie _Admin Panel Ads_ en voeg de HTML-code voor je ondersteuningstool toe.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Voeg klantenondersteuning toe aan specifieke abonnementen_

Op deze manier kunnen mensen je rechtstreeks vanuit hun Admin Panel bereiken!
