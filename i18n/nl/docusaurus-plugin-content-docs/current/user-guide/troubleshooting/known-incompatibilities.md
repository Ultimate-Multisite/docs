# Bekende incompatibiliteiten met andere plugins

Na het installeren en activeren van Ultimate Multisite, kunt u problemen tegenkomen als gevolg van niet aangepaste incompatibiliteiten met andere plugins.

We werken actief aan het verkleinen van deze lijst bij elke nieuwe release, maar als een plugin hier vermeld staat, moet u deze mogelijk deactiveren om een volledig functioneel Ultimate Multisite Network te hebben.

Hier is een lijst van hen en wat u in deze gevallen moet doen:

**Sommige WPMU DEV plugins** Sommige van de plugins die _niet meer_ worden onderhouden door WPMU DEV staan bekend om de registratieflow van Ultimate Multisite te laten stoppen op de laatste stap (de gebruiker kan geen account aanmaken). Hier is een lijst van de plugins die die problemen veroorzaken:

* Signup Language: _Ultimate Multisite heeft een add-on die dezelfde functionaliteit biedt, genaamd Ultimate Multisite: Language Switcher._
* Set Blog Description on Blog Creation: _Dit is ingebouwd in Ultimate Multisite, dus er is geen extra plugin nodig._
* New Blog Templates: _Dit is ingebouwd in Ultimate Multisite, dus er is geen extra plugin nodig._
* Set Password on Multisite Blog Creation: _Dit is ingebouwd in Ultimate Multisite, dus er is geen extra plugin nodig._
* Signup TOS: _Dit is ingebouwd in Ultimate Multisite, dus er is geen extra plugin nodig._
* Anti-Splog: Hoewel minder krachtig, is dit ook _ingebouwd in Ultimate Multisite._

**Opmerking** : Alle andere WPMU Dev plugins kunnen normaal worden gebruikt naast Ultimate Multisite. Voorbeelden zijn _Smush_, _Forminator_, _Defender_, etc.
