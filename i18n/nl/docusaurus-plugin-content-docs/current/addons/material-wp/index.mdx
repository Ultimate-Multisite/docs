# Material WP

De WordPress-beheerinterface is al jaren hetzelfde. Maar heb je je ooit afgevraagd hoe een Material Design-versie van de beheerinterface eruit zou zien? 

Material WP is een beheerthema dat is ontwikkeld om je die look-and-feel te geven, zodat je je klanten een meer aangepaste Dashboard kunt bieden.

## Functies

- Ondersteuning voor aangepaste CSS
- Eenvoudige stylinginstellingen
- Menuvolgorde aanpassen
- Eenvoudige import/export

## Documentatie

**Je beheerpaneel aanpassen met Material WP**

## Installatie

1. Upload 'material-wp' naar de map '/wp-content/plugins/'
2. Activeer de plugin via het menu 'Plugins' in WordPress

## Gebruikshandleiding

De WordPress-beheerinterface is al jaren hetzelfde. Maar heb je je ooit afgevraagd hoe een Material Design-versie van de beheerinterface eruit zou zien?

Ga na **het installeren ervan** naar je _Plugins_-pagina en klik om het op je Network te activeren.
