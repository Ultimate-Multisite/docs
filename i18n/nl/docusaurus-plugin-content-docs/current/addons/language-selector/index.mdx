# Taalselectie

Multisite Ultimate: Taalselectie op Aanmelden

Voeg een taalselectieveld toe aan de aanmeldstroom.

## Installatie

1. Upload 'ultimate-multisite-language-selector' naar de map '/wp-content/plugins/'
2. Activeer de plugin via het menu 'Plugins' in je WordPress Network Admin

## Gebruiksgids

Met deze gratis add-on kunnen gebruikers kiezen welke taal ze willen zien in de aanmeldstroom en op de inlogpagina.

Na het [activeren van je licentie](../../user-guide/getting-started/installing-ultimate-multisite.md) kun je Ultimate Multisite add-ons installeren waar je licentie toegang toe geeft.

Ga naar je Ultimate Multisite instellingenpagina en klik vervolgens op het menu aan de linkerkant op het label _Add-ons_.

![Add-ons page](/img/admin/addons-page.png)

_Ga naar de Add-ons pagina om Taalselectie te installeren_

Daar vind je alle add-ons die we aanbieden.

Gratis add-ons zijn beschikbaar voor al onze klanten, ongeacht het abonnement dat ze hebben gekozen.

Klik op de knop _Install it Now!_ onder de thumbnail van de add-on. Activeer het vervolgens op de Plugins pagina.

![Add-ons page with install button](/img/admin/addons-page.png)

_Klik op de knop om de add-on te installeren_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_Activeer het op je Plugins pagina_

Ga nu naar het Settings menu van Ultimate Multisite en open het tabblad _Network Settings_. Daaronder, onderaan de pagina, vind je een nieuwe sectie genaamd _Languages_.

![Settings page](/img/config/settings-general.png)

_Een geheel nieuwe sectie verschijnt!_

Je kunt het leeg laten om alle talen als opties weer te geven of selecteer welke je beschikbaar wilt maken. De talen zijn degene die je hebt geïnstalleerd in je WordPress installatie.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_Lijst van talen beschikbaar op deze WP-installatie_

Na het opslaan van de gemaakte wijzigingen wordt een selectie menu met de gekozen talen weergegeven op de Login pagina en op de Sign-up Flow.

![Login and Registration settings](/img/config/settings-login-registration.png)

_Taalselectie op de Login pagina_

![Registration page](/img/config/registration-page.png)

_Taalselectie op de Sign-up Flow_

## De taal instellen via de URL

Je kunt ook de taal van een pagina wijzigen door de locale parameter in de URL te overschrijven. Als je bijvoorbeeld naar een registratiepagina in het Italiaans wilt omleiden, voeg dan ?locale=it_IT toe aan de URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

De taal moet geïnstalleerd zijn op je WordPress netwerk om dit te laten werken.
