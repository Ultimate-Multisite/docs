# Captcha

**Stop Bots. Bescherm Inkomsten. Geen Configuratie Moeilijkheden.**

Een multisite netwerk met aangepaste domeinen draaien? Je kent het ongemak: Google reCAPTCHA en hCaptcha vereisen dat je *elk enkel domein* registreert in hun admin console. Een nieuwe klantsite toevoegen? Update de captcha-configuratie. Een aangepast domein koppelen? Opnieuw updaten. Het eindigt nooit.

**Ultimate Multisite: Captcha verandert alles.**

## Cap Captcha: Zelfgehoste Bescherming Die Werkt

Onze nieuwe **Cap Captcha** provider is een game-changer voor multisite netwerken:

* **Zero Configuration** — Activeer de addon en je bent beschermd. Geen API-sleutels, geen Google-account, geen hCaptcha-dashboard, geen domein-whitelisting.
* **Works on Every Domain Automatically** — Subdomeinen, gekoppelde domeinen, klantdomeinen—ze zijn allemaal meteen beschermd.
* **Self-Hosted & Private** — Je gegevens verlaten nooit je server. Volledig GDPR compliant zonder tracking van derden.
* **Invisible Proof-of-Work** — Gebruikers lossen een rekenpuzzel op in de achtergrond. Geen klikken op brandblussers, geen frustrerende afbeeldingsraster.
* **Lightweight** — Slechts 20KB. Zal je checkout niet vertragen.

## WooCommerce Kaart Test Bescherming

Kaart testaanvallen kosten handelaren jaarlijks miljarden. Fraudeurs gebruiken bots om gestolen creditcards te valideren tegen je WooCommerce checkout—jij betaalt de verwerkingskosten, zij krijgen geldige kaartnummers.

**We beschermen waar het het meest telt:**

* **Store API Protection** — Blokkeert botaanvallen op `/wp-json/wc/store/v1/checkout` die traditionele formulier captchas omzeilen
* **PayPal Payments Compatible** — Werkt naadloos met WooCommerce PayPal Payments
* **Real-Time Statistics** — Zie precies hoeveel aanvallen je blokkeert

## Installatie

1. Upload `ultimate-multisite-captcha` naar je `/wp-content/plugins/` directory
2. Netwerk Activeer de plugin
3. **Dat is het. Je bent beschermd.**

Cap Captcha activeert automatisch—geen instellingen om te configureren, geen API-sleutels om in te voeren. Elke ondersteunde eindpunt in je hele netwerk is beschermd zodra je activeert.

**Optioneel:** Om Google reCAPTCHA of hCaptcha te gebruiken, navigeer naar **Ultimate Multisite → Settings → Captcha** en selecteer je gewenste provider.

## Veelgestelde Vragen

## Moet ik iets configureren?

Nee! Cap Captcha is automatisch ingeschakeld wanneer je de addon activeert. Je hele netwerk is onmiddellijk beschermd—geen instellingen om te configureren, geen API-sleutels om in te voeren.

## Heb ik API-sleutels nodig voor Cap Captcha?

Nee. Cap Captcha is volledig zelfgehost. Geen externe accounts, geen domeinregistratie, geen configuratieproblemen.

## Zal dit mijn checkout vertragen?

Cap Captcha is slechts 20KB en voert proof-of-work uitdagingen in de achtergrond uit. Je klanten zullen geen vertraging merken.

## Beschermt dit tegen kaart testaanvallen?

Ja! In tegenstelling tot traditionele captchas die alleen HTML-forms beschermen, koppelt onze WooCommerce integratie rechtstreeks aan de Store API om botaanvallen op API-niveau te blokkeren—waar de meeste kaart testaanvallen plaatsvinden.

## Kan ik verschillende captcha providers op verschillende sites gebruiken?

De captcha provider is netwerkbreed ingesteld, maar je kunt de moeilijkheidsinstellingen aanpassen op basis van je behoeften.

## Is dit GDPR compliant?

Cap Captcha is 100% zelfgehost zonder gegevens naar externe services te verzenden. Voor Google reCAPTCHA en hCaptcha, bekijk alstublieft hun respectievelijke privacybeleid.
