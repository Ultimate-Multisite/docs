# Filter: networks_user_is_network_admin

Filtert de netwerken waarvan een gebruiker de beheerder is.

## Parameters {#parameters}

| Naam | Type | Beschrijving |
|------|------|-------------|
| $ | `array\|bool` | Lijst van netwerk-ID's of false als er geen netwerken voor de gebruiker zijn. |
| $ | `int` | Gebruikers-ID waarvoor de netwerken moeten worden geretourneerd. |
