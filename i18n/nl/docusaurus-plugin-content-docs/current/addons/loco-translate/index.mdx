# Loco Translate

De Ultimate Multisite: Loco Translate Integration addon brengt site-specifieke vertaalmogelijkheden naar je WordPress multisite netwerk. Als je ooit hebt gehad dat verschillende sites in je netwerk verschillende vertalingen van dezelfde plugin of thema nodig hadden, is deze addon de perfecte oplossing.

**Wat doet deze addon**

Deze addon integreert naadloos Ultimate Multisite met de populaire Loco Translate plugin, waardoor elke subsite in je netwerk zijn eigen aangepaste vertalingen kan onderhouden. Of je nu een multi-regio platform runt, verschillende taalgemeenschappen bedient, of simpelweg behoefte hebt aan vertaalflexibiliteit over je netwerk, deze addon maakt het moeiteloos.

**Perfect voor:**

* Multi-regio bedrijven met verschillende lokalisatiebehoeften
* Onderwijsinstellingen die diverse gemeenschappen bedienen
* SaaS-platforms die meertalige ondersteuning bieden
* Web agencies die meerdere klantsites beheren
* Elk multisite netwerk dat vertaalflexibiliteit vereist

## Installatie

1. Zorg dat Ultimate Multisite is geïnstalleerd en netwerk-geactiveerd
2. Installeer en activeer de Loco Translate plugin
3. Upload deze addon naar je plugins directory
4. Network activeer de Loco Translate Integration addon
5. Begin met het maken van site-specifieke vertalingen via Loco Translate op elke subsite

Dat is het! De integratie werkt automatisch zonder extra configuratie.

## Veelgestelde vragen

## Moet ik iets configureren na installatie?

Nee! De integratie is automatisch. Zodra zowel de addon als Loco Translate actief zijn, kunnen subsites meteen beginnen met het maken van hun eigen aangepaste vertalingen.

## Wat gebeurt er als een subsite geen aangepaste vertalingen maakt?

De site zal automatisch netwerkbreed vertalingen gebruiken als fallback. Er gebeurt niets mis, en alles werkt naadloos.

## Kunnen subsites netwerkbreed vertalingen benaderen?

Reguliere site-beheerders zien en bewerken alleen de vertalingen van hun eigen site. Superadmins kunnen zowel netwerkbreed als site-specifieke vertalingen bekijken, afhankelijk van welke site ze bekijken.

## Werkt dit met zowel plugins als themes?

Ja! Site-specifieke vertalingen werken voor zowel WordPress plugins als themes.

## Zal dit mijn multisite netwerk vertragen?

Nee. De addon gebruikt efficiënte bestandslaadmethoden die alleen site-specifieke vertalingen controleren wanneer nodig. Er is geen merkbare prestatie-impact.

## Kan ik netwerkvertalingen kopiëren naar een specifieke site als startpunt?

Ja! Tijdens het bewerken van vertalingen in Loco Translate kun je eenvoudig beginnen met netwerkvertalingen en deze aanpassen voor elke site.

## Werkt dit met WooCommerce en andere plugins?

Absoluut! Elke plugin of theme die standaard WordPress vertaalfuncties gebruikt, werkt met site-specifieke vertalingen.
