# Plugin & Theme Manager

Multisite Ultimate: Plugin and Theme Manager

Bewerk of verberg de meta-informatie (titel, auteur, thumbnail en beschrijving) van geïnstalleerde plugins en thema's. Maak categorieën voor plugins en thema's zodat uw gebruikers ze kunnen filteren en voeg een mooie aangepaste Plugins-pagina toe aan het paneel van uw gebruikers.

Om de plugin te gebruiken, kunt u gewoon naar uw Plugins- of Themes-pagina's op uw netwerkbeheer gaan en klikken op "Edit Plugin/Theme info".

## Installation

1. Upload 'ultimate-multisite-plugin-and-theme-manager' naar de map '/wp-content/plugins/'
2. Activeer de plugin via het 'Plugins'-menu in uw WordPress Network Admin

## Usage Guide

Ultimate Multisite: Plugin and Theme Manager is een krachtige **premium** Ultimate Multisite add-on die u volledige controle geeft over hoe uw netwerkgebruikers de plugin- en themavoorstellen zien en ermee omgaan op uw netwerk.

Deze add-on heeft drie doelen:

  1. Het laat u de metadata van zowel plugins als thema's volledig wijzigen. Dat omvat naam, beschrijving, auteurnaam en zelfs miniaturen!
  2. Het laat u plugins en thema's categoriseren, zodat uw gebruikers ze gemakkelijk kunnen sorteren.
  3. Het geeft u de mogelijkheid om de oude, saaie plugins-pagina te vervangen door een nieuwe themalike pagina met sortering op categorieën.

Na het downloaden en installeren verschijnt er een nieuwe tab - _Add-on Settings_ \- op de Ultimate Multisite Settings pagina.
