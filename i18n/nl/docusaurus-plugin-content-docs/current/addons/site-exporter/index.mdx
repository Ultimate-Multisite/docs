# Site Exporter

## Inhoud

We understand the need of being able to move subsites within your network going to a single site installation and vice versa should your customer decide to migrate their website. The Site exporter and importer addon makes this task easier. This tool export and import sites using zip files which can also serve as a backup.

## Functies

* Export en Import Sites

* Gebruiksvriendelijke interface

* Maak een site back-up

## Documentatie
**Hoe een site exporteren en importeren**

## Installatie

1. Upload **ultimate-multisite-site-exporter** naar de **/wp-content/plugins/** directory
2. Activeer de plugin via het **Plugins** menu in WordPress
3. Volg de stap-voor-stap Wizard om de plugin in te stellen

## Hoe een site exporteren

Om een site vanuit je netwerk te exporteren, ga naar **Export & Import** in het Ultimate Multisite menu en, op het tabblad **Export Sites**, klik op **Generate new Site Export**.![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-_bBztobh1xIFoj0Zo3zo7bx842u3XFMgpc2b8kiHmGmBU6RjuxlSOSY4ZNzaWTPOJrb0TIW_Q0Dv9WvT0yD-WwOkZ7bR36Ij5aJa_l19cDTfuj9DADQqI5dGy3eIeUmoQA=s0)

Zoek de site die je wilt exporteren en selecteer de items die je wilt opnemen in het geëxporteerde bestand. Je kunt thema's, plugins en uploads opnemen. Als je site te groot is, kun je kiezen om het exportproces op de achtergrond uit te voeren.![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-fk-i4RP0uZFj8kgfcQeRfAqWtCBelOjpOQARFfeBisVkeRSc1Qywq6Dpy66MnB6oBbnvdfJM359LdRSgqCLU1tNozo1u1OFH5svxtld8TZAXHY-CPGh92gWz9HQ8nlUcjg=s0)

Klik op de **Export Site** knop. Je geëxporteerde site staat beschikbaar in de lijst **Existing Exports**. Klik op de **Download ZIP** knop om het bestand te downloaden.![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-mouH16ZCUuNaUg0PwF2JmMhhsKPJOztau21QIeB7DXqAcX4VI5FL_itxvnKmFo439xuGUWsntM3doG6yBkjS2gi6bNTjg-kLHC19m19br5xsgBl0XH8hy5LvuPN1uNtx9g=s0)

## Hoe een site importeren

Om een site te importeren, ga naar het tabblad **Import a Site** en upload het .zip bestand van de site die je wilt importeren. Je kunt de URL wijzigen naar een nieuwe of de oude URL behouden door het veld leeg te laten. Je kunt ook het vakje aanvinken om het .zip bestand na de import te verwijderen. Klik vervolgens op **Begin Import**.![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-Uq6wNJ4TPTuAtSJqKunxSJEn5cli7vbwYGgL1G3fcHwdvU7gSwW_J4xlBAtZ1fyVbDU7Pupi8uNFhw1Lzm4S2fjVTkbQoz7fiZPIc1qiy8qj3iGt-MNVlyLWmIZXFxPLYA=s0)

Het proces kan even duren. Je ziet een lopende import in de lijst onder de importer.![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-kMkwSY3bYrE5vjfJtLviYImhNBn_PwmuamaRDlQKS9rGwIy3a_jXV0glMNTU8fBmV6IKkNN7obXNQFqNaAJtAAbWDftzX76-qdCDKCrqdru_9taciFCayE9FNszZJTODvA=s0)

Je kunt het proces stoppen door op **Cancel Import** te klikken.![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-oGmtCq7br6Y9GAfw_Ch53TP8nkJ0hw77sQJpSCQ7fEj5qk2lJo3WuMl7NYqB9c8SmORis53bWHILwGLxUTbpv4trWhxibu-yu5DNu6yNWXfkRo0bcnAZJ5F2K1hYIMH6PQ=s0)

Na het voltooien van het proces is je site al beschikbaar op je netwerk.
