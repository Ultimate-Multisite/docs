# Admin Page Creator

## Inhoud
Admin Page Creator maakt het mogelijk om eenvoudig nieuwe admin pagina's en subpagina's toe te voegen binnen het WordPress admin dashboard. Het kan worden gebruikt om een welkomstpagina of tutorialpagina te maken die kan dienen als landingspagina voor nieuwe gebruikers. Je kunt een page builder zoals Beaver Builder, Elementor, Brizy en Oxygen gebruiken om de pagina's zonder moeite te ontwerpen.

Het biedt ook ondersteuning voor custom HTML, CSS en JavaScript, evenals toegangscontrole op basis van het plan en de gebruikersrol!

## Functies
* Ondersteuning voor Custom HTML en CSS
* Ondersteuning voor JavaScript
* PHP Content Mode
* Toestemming op basis van gebruikersrol
* Ondersteuning voor page builders

## Installatie

1. Upload **ultimate-multisite-admin-page-creator** naar de **/wp-content/plugins/** directory
2. Activeer de plugin via het **Plugins** menu in je WordPress Network Admin

## Gebruiksgids

Terwijl WordPress het echt gemakkelijk maakt om content toe te voegen aan de front-end van sites, vereist het aanmaken van pagina's en subpagina's aan de adminzijde enige basisprogrammeervaardigheden. Ultimate Multisite: Admin Page Creator heeft als doel diezelfde gebruiksvriendelijkheid van de front-end naar de back-end van je netwerk sites te brengen.

## Je eerste aangepaste Admin Page maken

Na het installeren en activeren van Ultimate Multisite: Admin Page Creator, ga naar je netwerk admin menu en navigeer naar Admin Pages.

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212627fb34b55df443e4a5-Aml1wdfXt-69C790DA-7E3F-4DDD-A381-7F6633D91B2A.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212627fb34b55df443e4a5-Aml1wdfXt-69C790DA-7E3F-4DDD-A381-7F6633D91B2A.png)

_Custom Admin Pages lijst op het Network Admin_

Je zult een lijst zien van alle admin pagina's die je tot nu toe hebt gemaakt (omdat dit je eerste bezoek is, zal de lijst leeg zijn). Ga verder en klik op **Add new Admin Page**. Je wordt doorgestuurd naar de Admin Page editor.

## De elementen van een Custom Admin Page

Op de Admin Page editor heb je een aantal verschillende opties om je pagina aan te passen.

### Pagina Titel

Dit is de titel die bovenaan de pagina wordt weergegeven, niet het label in het admin menu.

_Voer een Pagina Titel in_

### Inhoudstype Selector

Ultimate Multisite: Admin Page Creator biedt je twee opties als het gaat om de inhoud van de pagina: je kunt de standaard WordPress WYSIWYG editor gebruiken of overschakelen naar een HTML editor.

Als je niet vertrouwd bent met HTML, blijf dan bij de standaard WordPress editor. Je kunt de inhoud van de pagina bewerken net zoals je dat zou doen met een normale Post of Page.

_Uit gebruik van de standaard WordPress WYSIWYG editor_

_Uit gebruik van de HTML editor optie, met syntax highlighting en foutmeldingen_

## Menu Opties

Aan de zijkant, in de General Options meta-box, kun je het menu type, label en nog veel meer configureren.

### Menu Types

Je nieuwe aangepaste admin pagina kan twee verschillende menu types hebben: top-level menu en submenu.

Een top-level menu pagina, zoals de naam al aangeeft, voegt je nieuw aangemaakte pagina toe aan het admin menu naast de standaard WordPress pagina's.

Als je de top-level optie selecteert, kun je ook de menu volgorde en een menu icoon selecteren.

Als je deze pagina een sub-menu pagina wilt maken, kun je een bovenliggende pagina selecteren om deze pagina aan te koppelen. Je kunt je aangepaste pagina toevoegen aan de standaard WordPress pagina's of aan je top-level custom admin pagina's.

_Compleet aanpassen van het menu item voor deze Custom Page_

_Je kunt deze pagina ook toevoegen als sub-menu van andere menu pagina's, inclusief je eigen custom top-level pagina's._

## Geavanceerde Opties

Je zou in staat moeten zijn om allerlei soorten content toe te voegen aan je custom admin pagina's. Dat omvat CSS regels en bestanden, JavaScript en externe bibliotheken. Je kunt dit doen via de Advanced Options Tab.

### Custom CSS

Op de CSS tab kun je je eigen custom CSS code toevoegen, evenals CSS bestanden importeren van externe sites (zoals CDNs, Google Fonts, etc).

_Custom CSS wordt ook ondersteund. Je kunt ook externe style bestanden opnemen!_

### Custom JavaScript

Hetzelfde geldt voor JavaScript. Je kunt externe bibliotheken toevoegen om te gebruiken in je custom JavaScript code.

_Voeg je eigen JavaScript code toe!_

## Toestemmingen

Ook in de Advanced Options kun je selecteren welke sub-sites toegang hebben tot deze pagina. Dit is handig als je van plan bent om verschillende exclusieve content te leveren aan verschillende plan tiers, bijvoorbeeld.

_Je hebt volledige controle over waar deze pagina zal verschijnen_

**Belangrijk** : Beide voorwaarden moeten worden voldaan voor de pagina om te worden weergegeven. Dat betekent dat als de gebruiker een klant is van een bepaald plan, maar niet een van de geselecteerde rollen heeft (of omgekeerd), de pagina niet op hun paneel verschijnt.

## Actief

Als je een custom admin pagina wilt uitschakelen zonder deze te verwijderen of de permissies te verstoren, vink dan gewoon de Active optie uit. Het deactiveren van een top-level pagina schakelt automatisch alle subpagina's die aan die bovenliggende pagina zijn gekoppeld uit.

_Het eindresultaat op het gebruikerspaneel!_
