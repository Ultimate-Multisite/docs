# Ondersteuningsagenten

Het beheren van een volledig netwerk alleen is geen gemakkelijke taak. Naarmate je netwerk groeit, zul je een toenemende vraag naar ondersteuning zien. Het is een goed idee om mensen in te schakelen die je helpen je werklast te verlichten. Het geven van volledige toegang/permises is echter riskant en kan je netwerk in gevaar brengen. Dat is waar dit Support Agent add-on in beeld komt.

## Functies

- Gedetailleerde lijst van WordPress & Ultimate Multisite permissies
- Toegang gemakkelijk inschakelen en uitschakelen
- Maak een onbeperkt aantal Agent-accounts

## Installatie

1. Upload 'ultimate-multisite-support-agents' naar de map '/wp-content/plugins/'
2. Activeer de plugin via het menu 'Plugins' in WordPress
3. Volg de stapsgewijze Wizard om de plugin in te stellen
