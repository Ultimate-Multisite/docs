# Fluent Forms

Deze add-on maakt het mogelijk om limieten toe te voegen aan Fluent Forms op Multisite Ultimate in producten, lidmaatschappen of sites.
Fluent Forms gebruikt aangepaste tabellen in plaats van aangepaste berichttypen, waardoor de normale limietopties niet werken met Fluent Forms.

## Functies

- Limieten toevoegen voor Fluent Forms of Conversational Fluent Forms
- Klanten krijgen een vriendelijke boodschap die hen aanspoort hun plan te upgraden wanneer ze de limiet bereiken
