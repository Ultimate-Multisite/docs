# Filter: wu_get_site_domain_and_path

Laat ontwikkelaars de domein/padparen manipuleren.

Dit kan nuttig zijn voor verschillende doeleinden, zoals het implementeren van een soort staging-oplossing, verschillende servers, enz.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $d | `object` | Het huidige object dat een domein- en pad-sleutels bevat. |
| $path_or_subdomain | `string` | De oorspronkelijke pad/subdomein die aan de functie is doorgegeven. |

- 2.0.0

Een object dat een domein- en pad-sleutels bevat.
