# Filter: wu_independent_billing_cycle_product_types

Filter de producttypes die onafhankelijke factureringscycli hebben.

Producten met onafhankelijke factureringscycli worden niet verplicht om het factureringsperioden van het plan in het winkelwagentje te matchen. Dit is handig voor producten zoals domain registrations die altijd jaarlijks factureren, ongeacht of het plan maandelijks of jaarlijks is.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | Array van producttypes met onafhankelijke factureringscycli. |
