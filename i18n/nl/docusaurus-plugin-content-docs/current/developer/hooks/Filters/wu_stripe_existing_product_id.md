# Filter: wu_stripe_existing_product_id

Filter het ID van het product om te controleren. Als dit bestaat, zal het nieuwe abonnement dit product gebruiken.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $product_id | `string` | ID van het Stripe-product om te controleren. |
| $name | `string` | Productnaam. |
