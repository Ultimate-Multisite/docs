# Filter: wu_view_override_replaceable_views {#filter-wuviewoverridereplaceableviews}

Stelt ontwikkelaars in staat extra mappen toe te voegen aan de vervangbare lijst.

Wees voorzichtig, want het toestaan van extra mappen kan leiden tot het laden van verouderde kopieën in plaats van de Ultimate Multisite-versies.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $replaceable_views | `array` | Lijst van toegestane mappen. |
