# Filter: wp_ultimo_render_vars

Sta pluginontwikkelaars toe om extra variabelen toe te voegen aan de rendercontext globaal.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $args | `array` | Array met variabelen die door de renderaanroep zijn doorgegeven. |
| $view | `string` | Naam van de weergave die gerenderd moet worden. |
| $default_view | `string` | Naam van de fallback_view |

### Since

- 2.0.0
