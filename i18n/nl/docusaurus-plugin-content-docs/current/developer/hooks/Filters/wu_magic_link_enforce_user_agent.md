# Filter: wu_magic_link_enforce_user_agent

Filter of het afdwingen van user agent-verificatie.

Stel in op false om tokens te laten werken op verschillende browsers/apparaten. Dit vermindert de beveiliging maar verhoogt de bruikbaarheid.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $enforce | `bool` | Of het afdwingen van user agent-vergelijking. |
