# Filter: wu_hosting_integration_wizard_sections

Filtert de wizard-secties voor het opzetten van hostingintegratie.

Staat addons toe om wizard-secties toe te voegen, te verwijderen of te wijzigen.

## Parameters

| Naam | Type | Beschrijving |
|------|------|-------------|
| $sections | `array` | Wizard-secties. |
| $integration | `\WP_Ultimo\Integrations\Host_Providers\Base_Host_Provider\|\WP_Ultimo\Integrations\Integration` | De integratie die wordt geconfigureerd. |
| $page | `\Hosting_Integration_Wizard_Admin_Page` | De wizard-pagina instantie. |

### Since

- 2.5.0
