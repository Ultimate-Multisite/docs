# Filter: wu_documentation_get_link {#filter-wudocumentationgetlink}

Laat pluginontwikkelaars de links filteren.

Niet zeker hoe dat nuttig zou kunnen zijn, maar het schaadt niet om het te hebben.

## Parameters {#parameters}

| Naam | Type | Beschrijving |
|------|------|-------------|
| $link | `string` | De geregistreerde link |
| $slug | `string` | De slug gebruikt om de link op te halen |
| $default_link | `string` | De standaardlink geregistreerd |

### Sinds {#since}

- 1.7.0
