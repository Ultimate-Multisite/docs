# Actie: wu_site_created

Wordt uitgevoerd nadat een site voor het eerst is aangemaakt.

Wordt niet uitgevoerd als het van een sjabloon is gedupliceerd.

## Parameters

| Naam | Type | Beschrijving |
|------|------|-------------|
| $data | `array` | De objectgegevens die worden opgeslagen. |
| $site | `\Site` | De objectinstantie. |

### Sinds

- 2.0.0
