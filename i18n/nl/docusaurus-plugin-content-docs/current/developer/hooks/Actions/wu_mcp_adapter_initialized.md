# Action: wu_mcp_adapter_initialized {#action-wumcpadapterinitialized}

Wordt geactiveerd nadat de MCP adapter is geïnitialiseerd.

Maakt het mogelijk voor andere plugins en thema's om hun eigen mogelijkheden te registreren.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | De MCP adapter instantie. |
