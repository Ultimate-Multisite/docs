## Parameters

| Naam | Type | Beschrijving |
|------|------|-------------|
| $data | `array` | De objectgegevens die opgeslagen zullen worden. |
| $this | `\Base_Model` | De objectinstantie. |
| $new | `bool` | Waar als het object nieuw is. |
