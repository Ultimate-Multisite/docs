# Документація Ultimate Multisite {#ultimate-multisite-documentation}

Ласкаво просимо до документації Ultimate Multisite. Ultimate Multisite перетворює WordPress Multisite на повноцінну платформу Website as a Service (WaaS).

## Розділи документації {#documentation-sections}

### [Посібник користувача](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

Усе, що вам потрібно, щоб налаштувати, конфігурувати та керувати вашою платформою WaaS. Охоплює встановлення, платіжні шлюзи, мапування доменів, керування клієнтами та багато іншого.

### [Документація для розробників](/developer/hooks) {#developer-documentation}

Технічний посібник для розробників, які створюють на базі Ultimate Multisite. Включає довідник хуків (hooks reference), документацію REST API та посібники з розробки аддонів.

### [Аддони](/addons) {#addons}

Документація для платних плагінів-аддонів, які розширюють Ultimate Multisite додатковими функціями, такими як інтеграція з WooCommerce, аналітика, продаж доменів та багато іншого.
