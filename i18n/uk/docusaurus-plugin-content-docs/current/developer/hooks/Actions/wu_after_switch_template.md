# Дія: wu_after_switch_template

Дозволяє розробникам плагінів підключати функції (гачки) після того, як користувач або супер-адміністратор змінить шаблон сайту.

## Параметри

| Ім'я | Тип | Опис |
|------|------|-------------|
| $id | `int` | ID сайту |

### З версії

- 1.9.8
