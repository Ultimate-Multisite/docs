# Дія: wu_page_before_render {#action-wupagebeforerender}

Дозволяє розробникам плагінів додавати додатковий контент перед тим, як ми виведемо сторінку.

## Параметри {#parameters}

| Ім'я | Тип | Опис |
|------|------|-------------|
| $page_id | `string` | Ідентифікатор цієї сторінки. |
| $page | `object` | Об'єкт сторінки. |

### З версії {#since}

- 1.8.2
