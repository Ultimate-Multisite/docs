# Action: wu_site_created

Виконується після першого створення сайту.

Не виконується, якщо сайт дублюється з шаблону.

## Параметри

| Name | Type | Description |
|------|------|-------------|
| $data | `array` | Об'єкт даних, який буде збережено. |
| $site | `\Site` | Екземпляр об'єкта. |
