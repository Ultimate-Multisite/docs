# Action: wu_{$this->id}_scripts {#action-wuthis-idscripts}

Активує хук для додавання скриптів (enqueue scripts hook).

Цей хук використовується елементом для підключення (хукінгу) свого методу `register_scripts`.

### З версії {#since}

- 2.0.0
