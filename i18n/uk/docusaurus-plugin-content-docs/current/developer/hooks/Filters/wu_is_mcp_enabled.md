# Фільтр: wu_is_mcp_enabled

Дозволяє розробникам плагінів примусово встановити заданий стан для адаптера MCP.

## Параметри

| Ім'я | Тип | Опис |
|------|------|-------------|
| $enabled | `bool` | Чи увімкнено адаптер MCP. |

### Зверніть увагу

- 2.5.0

Визначено в [`inc/class-mcp-adapter.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-mcp-adapter.php#L250) на рядку 250
