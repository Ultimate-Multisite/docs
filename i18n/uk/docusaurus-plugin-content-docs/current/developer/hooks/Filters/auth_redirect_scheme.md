# Фільтр: auth_redirect_scheme

Фільтрує схему перенаправлення аутентифікації.

## Параметри

| Name | Type | Опис |
|------|------|-------------|
| $scheme | `string` | Схема перенаправлення аутентифікації. За замовчуванням порожній. |
