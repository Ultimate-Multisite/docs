# Фільтр: wu_independent_billing_cycle_product_types {#filter-wuindependentbillingcycleproducttypes}

Фільтрує типи продуктів, які мають незалежні цикли білінгу.

Продукти з незалежними циклами білінгу не змушені відповідати періоду білінгу плану в кошику. Це корисно для таких продуктів, як реєстрації доменів, які завжди виставляються за рахунком за рік, незалежно від того, чи план є щомісячним, чи річним.

## Параметри {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | Масив типів продуктів з незалежними циклами білінгу. |
