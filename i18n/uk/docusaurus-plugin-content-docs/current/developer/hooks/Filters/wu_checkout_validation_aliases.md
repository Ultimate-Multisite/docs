# Фільтр: wu_checkout_validation_aliases

Дозволяє розробникам плагінів додавати власні псевдоніми (аліаси) у валідатор форм.

## Параметри

| Ім'я | Тип | Опис |
|------|------|-------------|
| $validation_aliases | `array` | Масив у форматі ID => псевдонім. |
| $checkout | `\Checkout` | Клас оформлення замовлення. |
