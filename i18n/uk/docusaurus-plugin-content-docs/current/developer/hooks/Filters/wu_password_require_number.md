# Фільтр: wu_password_require_number {#filter-wupasswordrequirenumber}

Фільтрує, чи потрібно вимагати цифри у паролях.

## Параметри {#parameters}

| Ім'я | Тип | Опис |
|------|------|-------------|
| $require | `bool` | Чи потрібно вимагати цифри. За замовчуванням `true`, якщо правила активні. |
| $defender_active | `bool` | Чи активний Defender Pro Strong Password. |
