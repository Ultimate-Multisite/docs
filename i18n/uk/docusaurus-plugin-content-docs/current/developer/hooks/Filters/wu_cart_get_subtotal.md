# Фільтр: wu_cart_get_subtotal

Фільтрує загальну суму "початкової вартості".

## Параметри

| Ім'я | Тип | Опис |
|------|------|-------------|
| $subtotal | `float` | Загальна сума, яку потрібно сплатити сьогодні. |
| $this | `\WP_Ultimo\Checkout\Cart` | Об'єкт кошика. |

### Джерело

Визначено у [`inc/checkout/class-cart.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-cart.php#L2112) на рядку 2112.
