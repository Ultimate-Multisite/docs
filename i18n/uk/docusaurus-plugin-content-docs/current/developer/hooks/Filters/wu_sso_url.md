# Фільтр: wu_sso_url {#filter-wussourl}

Фільтрує згенеровані SSO URL перед їх поверненням для міждоменних дій клієнта.

Використовуйте цей фільтр, коли інтеграції потрібно додати довірений контекст до SSO-посилання суверенного орендаря або замінити URL брокера, зберігаючи перевірку токена Ultimate Multisite.

## Параметри {#parameters}

| Назва | Тип | Опис |
|------|------|-------------|
| $sso_url | `string` | Згенерований SSO URL. |
| $user | `WP_User` | Користувач, якого буде автентифіковано під час SSO-візиту. |
| $site_id | `int` | ID цільового сайту для візиту. |
| $redirect_to | `string` | URL призначення після успішної SSO-перевірки. |

### Починаючи з {#since}

- 2.13.0

### Джерело {#source}

Визначено в `inc/sso/class-sso.php`.


## Повертає {#returns}

Відфільтрований SSO URL.
