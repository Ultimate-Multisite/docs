# Фільтр: wu_should_create_domain_record_for_site

Фільтрує, чи Ultimate Multisite має створювати запис домену для новоствореного сайту.

Використовуйте цей фільтр, щоб пригнічувати або відкладати автоматичне створення запису домену для сайтів, які використовують спільний базовий домен форми оформлення замовлення, внутрішній хост або домен, яким окремо керуватиме інша інтеграція.

## Параметри

| Назва | Тип | Опис |
|------|------|-------------|
| $create | `bool` | Чи має бути створений запис домену. |
| $site | `WP_Site` | Об’єкт новоствореного сайту. |

### Починаючи з

- 2.13.0

### Джерело

Визначено в `inc/functions/domain.php`.


## Повертає

Логічне значення, що вказує, чи створювати запис домену.
