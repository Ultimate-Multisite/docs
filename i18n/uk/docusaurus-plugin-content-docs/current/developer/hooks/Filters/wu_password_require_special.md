# Фільтр: wu_password_require_special

Визначає, чи потрібно вимагати спеціальні символи у паролях.

## Параметри {#parameters}

| Ім'я | Тип | Опис |
|------|------|-------------|
| $require | `bool` | Чи вимагати спеціальні символи. За замовчуванням `true`, якщо правила активні. |
| $defender_active | `bool` | Чи активний Defender Pro Strong Password. |
