# Filter: wu_current_{$type}_get_manage_url {#filter-wucurrenttypegetmanageurl}

## Параметри {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $manage_site_url | `string` | URL-адреса керування сайтом. |
| $id | `int` | ID сайту. |
| $site_hash | `string` | Хеш сайту. |

### Since {#since}

- 2.0.9
