# Фільтр: wu_billing_address_fields

Дозволяє розробникам плагінів фільтрувати поля адреси для виставлення рахунків.

## Параметри {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $fields | `array` | Масив полів адреси для виставлення рахунків. |
| $zip_only | `bool` | Чи потрібні нам лише поштовий індекс та країна. |
