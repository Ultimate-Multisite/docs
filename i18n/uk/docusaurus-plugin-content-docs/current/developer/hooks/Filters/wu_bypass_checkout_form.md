# Фільтр: wu_bypass_checkout_form {#filter-wubypasscheckoutform}

Дозволяє розробникам обійти виведення форми та встановити власну.

## Параметри {#parameters}

| Ім'я | Тип | Опис |
|------|------|-------------|
| $bypass | `string\|bool` | Чи потрібно обійти форму оформлення замовлення, або рядок, який потрібно повернути замість форми. |
| $atts | `array` | Параметри блоку/короткого коду оформлення замовлення. |
