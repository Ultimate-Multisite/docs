# Фільтр: send_auth_cookies

Дозволяє запобігти фактичній відправці cookie автентифікації клієнту.

## Параметри {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $send | `bool` | Чи потрібно відправляти cookie автентифікації клієнту. |

### Since {#since}

- 4.7.4
