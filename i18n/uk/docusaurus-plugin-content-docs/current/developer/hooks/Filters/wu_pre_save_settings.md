# Filter: wu_pre_save_settings

Дозволяє розробникам фільтрувати налаштування перед збереженням за допомогою Ultimate Multisite.

## Параметри {#parameters}

| Ім'я | Тип | Опис |
|------|------|-------------|
| $settings | `array` | Налаштування, які потрібно зберегти. |
| $settings_to_save | `array` | Нові налаштування, які потрібно додати. |
| $saved_settings | `array` | Поточні збережені налаштування. |

### Починаючи з {#since}

- 2.0.18
