# Filter: wu_view_override_replaceable_views

Дозволяє розробникам додавати додаткові папки до списку папок, які можна замінити.

Будьте обережні: дозвіл на додавання додаткових папок може призвести до завантаження застарілих копій замість версій Ultimate Multisite.

## Параметри {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $replaceable_views | `array` | Список дозволених папок. |

### Since {#since}

- 2.0.0
### Source {#source}

Визначено в [`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L50) на рядку 50
