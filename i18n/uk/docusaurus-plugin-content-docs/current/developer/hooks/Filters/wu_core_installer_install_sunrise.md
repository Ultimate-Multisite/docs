# Фільтр: wu_core_installer_install_sunrise {#filter-wucoreinstallerinstallsunrise}

Дозволяє провайдерам хостингу встановлювати константу по-іншому.

Повернення `true` запобігає спробі Ultimate Multisite записувати дані у файл `wp-config`.
