# Фільтр: wu_current_set_site

Дозволяє розробникам змінити стандартну поведінку та встановити поточний сайт вручну.

## Параметри

| Ім'я | Тип | Опис |
|------|------|-------------|
| $site | `\WP_Ultimo\Models\Site` | Поточний сайт, який потрібно встановити. |
| $ | `self` | Поточний екземпляр класу. |

### Зверніть увагу

- 2.0.9

Визначено у [`inc/class-current.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-current.php#L331) на рядку 331
