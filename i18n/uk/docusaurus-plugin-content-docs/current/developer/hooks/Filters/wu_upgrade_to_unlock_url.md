# Filter: wu_upgrade_to_unlock_url

Дозволяє розробникам змінити URL-адресу для розблокування після оновлення.

## Параметри {#parameters}

| Ім'я | Тип | Опис |
|------|------|-------------|
| $url | `string` | URL-адреса для оновлення. |
| $args | `array` | Модуль та тип обмеження, яке потребує оновлення. |

### Джерело {#source}

Визначено у [`inc/functions/limitations.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/limitations.php#L177) на рядку 177.
