# Фільтр: wu_plan_product_types

Фільтрує типи продуктів, які мають вважатися планами.

Цей фільтр дозволяє аддонам реєструвати додаткові типи продуктів, які мають розпізнаватися як плани під час валідації та сегментації.

## Параметри

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | Масив типів продуктів, які потрібно вважати планами. |

- 2.3.0
