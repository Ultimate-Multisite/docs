# Фільтр: wu_maybe_redirect_to_admin_panel_exclusion_list

Створіть список виключень параметрів, які запобігають автоматичному перенаправленню.

Це необхідно, оскільки інакше конструктор сторінок не зможе редагувати сторінку входу після її визначення.
