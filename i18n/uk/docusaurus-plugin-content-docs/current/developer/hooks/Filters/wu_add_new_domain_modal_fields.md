# Фільтр: wu_add_new_domain_modal_fields {#filter-wuaddnewdomainmodalfields}

Фільтрує поля для модального вікна додавання нового домену.

Дозволяє додаткам (наприклад, Domain Seller) змінювати або замінювати поля реєстрації домену.

## Параметри {#parameters}

| Ім'я | Тип | Опис |
|------|------|-------------|
| $fields | `array` | Поля форми. |
