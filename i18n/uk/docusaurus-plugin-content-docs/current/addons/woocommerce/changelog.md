# Журнал змін інтеграції з WooCommerce

Version 2.0.6 - Випущено 2026-01-16
* Покращення: Включено основні підписки в аддон. Тепер не потрібен розширення Woocommerce Subscriptions.

Version 2.0.5 - Випущено 2026-01-09
* Покращення: Завантаження перекладів з API glotpress.
* Виправлення: Критична помилка в деяких page builders.
* Виправлення: Нескінченне перенаправлення, коли клієнт є учасником основного сайту.

Version 2.0.4 - Випущено 2025-11-14
* Додано: Переклади для набагато більше мов.
* Змінено: Перейменовано на Ultimate Multisite: Woocommerce Integration.
* Додано: Сумісність з Woocommerce 10.2.1.
* Додано: Сумісність з Woocommerce Subscriptions 7.7.0.
* Виправлення: Сумісність з PHP 8.4
* Виправлення: Перенаправлення, коли сторінка облікового запису WC не існує.

Version 2.0.3 - Випущено 2025-08-13
* Змінено: Увімкнено автоматичні оновлення з новим marketplace.

Version 2.0.2 - Випущено 2025-07-05
* Змінено: Перейменовано на Multisite Ultimate: Woocommerce Integration.
* Додано: Сумісність з Woocommerce 9.8.1.
* Додано: Сумісність з Woocommerce Subscriptions 7.3.0.
* Виправлення: Скасування підписки клієнтом.
* Виправлення: Критична помилка при використанні блоку оформлення замовлення (checkout block).
* Покращення: Тепер сумісний з кастомними таблицями замовлень високої продуктивності WooCommerce.
* Виправлення: Скасування на касі WooCommerce все ще може підвищити статус членства.

Version 2.0.1 - Випущено 2023-08-09

* Додано: Сумісність з Woocommerce 7.9.0.
* Додано: Сумісність з Woocommerce Subscriptions 5.3.0.
* Додано: Підтримка оновлення членства.
* Додано: Повідомлення про пробні періоди та збори за налаштування в Woocommerce.
* Додано: Ідентифікація продуктів Ultimate Multisite Woocommerce за мета-значенням.
* Додано: Вставлено одноразове виправлення для позначення всіх продуктів Woocommerce, пов'язаних з Ultimate Multisite.
* Додано: Видалено продукти, створені Ultimate Multisite, зі списку Woocommerce.
* Покращення: Створено неперіодичний дисконт WooCommerce для застосування до кошика.
* Покращення: Відновлено періодичний дисконт для продукту Woocommerce.
* Покращення: Додано мітку періодичного дисконту до продукту Woocommerce.
* Покращення: Забезпечено тип продукту на касі.
* Виправлення: Збережено статус членства під час процесу пониження рівня.
* Виправлення: Перевірено, чи існує підписка, щоб уникнути помилок під час скасування.
* Виправлення: Додано дату початку підписки для використання в підписках Woocommerce.
* Внутрішнє: Впроваджено новий процес збірки PHP 8.1.

Version 2.0.0 - Повна переробка.

* Додано: Метод обробки скасування для видалення підписки woo при зміні шлюзу або скасуванні членства;
* Додано: Обробник для пониження та підвищення рівня членства;
* Покращення: Завантаження залежностей WooCommerce на форму оновлення клієнта в subsites для дозволу оновлення облікового запису;
* Покращення: Коректне завантаження кошика WooCommerce, якщо він не існує;
* Покращення: Забезпечення роботи з таблицями основного сайту при обробці каси;
* Покращення: Робить замовлення оновлення Ultimo на основі значення замовлення підписки WooCommerce, а не останнього платежу;
* Виправлення: Перехід за посиланням кнопки WU Membership;
* Виправлення: Встановлює замовлення Ultimo як оплачене, коли оновлення підписки WooCommerce оплачено;
* Збірка: Додано MPB як builder;

Version 2.0.0-beta-5 - Випущено 2022-01-21

* Внутрішнє: Додано генератор хуків та фільтрів;
* Внутрішнє: Додано заглушки Ultimate Multisite для покращення якості життя розробника;
* Виправлено: Запобігання створенню кількох продуктів, коли це не потрібно;

Version 2.0.0-beta.4 - 2021-09-23

* Виправлення: Вимагає, щоб WooCommerce був активний на рівні мережі, а не лише на основному сайті;
* Покращення: Додано фільтр, який дозволяє використовувати аддон як mu-plugin;

Version 2.0.0-beta.3 - 2021-05-28

* Виправлення: Контроль доступу до dashboard був занадто агресивним;
* Покращення: Додано посилання на допомогу WooCommerce до верхнього меню Ultimate Multisite;

Version 2.0.0-beta.2 - 2021-05-04

* Покращення: Створює відмінені платежі в Ultimo при створенні замовлення оновлення WCS;
* Покращення: Автозаповнює поля виставлення рахунків даними клієнта Ultimate Multisite;
* Покращення: Додає поля виставлення рахунків для шлюзів;

Version 2.0.0-beta.1 - 2021-05-04

* Початковий бета-реліз

* Виправлено: Інтеграція WooCommerce працює навіть тоді, коли WooCommerce не активний на рівні мережі та активований лише на основному сайті;
