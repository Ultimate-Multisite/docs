# Журнал змін інтеграції WooCommerce

Версія 2.2.0 - Випущено 2026-07-01
* Нове: суми податків Ultimate Multisite тепер відображаються як окремі рядки зборів WooCommerce під час оформлення замовлення, що робить підсумки податків зрозумілішими перед оплатою.
* Нове: додано налаштування з добровільним увімкненням "Suspend Memberships Immediately on Failed Renewals" і фільтр `wu_woo_suspend_on_payment_failure` для сайтів, які хочуть призупинення після невдалого поновлення протягом вікна повторних спроб WooCommerce Subscriptions.
* Виправлення: узгоджено статуси WooCommerce Subscription, які могли залишатися несинхронізованими з членствами Ultimate Multisite після невдалих або відновлених поновлень.
* Виправлення: додано валюту магазину WooCommerce до списку валют Ultimate Multisite, коли її бракувало.
* Виправлення: збережено платіжні дані клієнта під час перенаправлення підписників до оформлення замовлення WooCommerce.
* Покращено: додано сумісність із Jetpack Autoloader 5.
* Покращено: очищено створення релізного пакета, щоб архіви GitHub і маркетплейсу уникали вкладених проміжних каталогів і файлів розробки.

Версія 2.0.6 - Випущено 2026-01-16
* Покращення: включено основні підписки в додаток. Більше не потребує розширення Woocommerce Subscriptinos.

Версія 2.0.5 - Випущено 2026-01-09
* Покращення: завантаження перекладів із glotpress API.
* Виправлення: фатальна помилка в деяких конструкторах сторінок.
* виправлення: нескінченне перенаправлення, коли клієнт є учасником головного сайту.

Версія 2.0.4 - Випущено 2025-11-14
* Додано: переклади для значно більшої кількості мов.
* Змінено: перейменовано на Ultimate Multisite: Woocommerce Integration.
* Додано: сумісність із Woocommerce 10.2.1.
* Додано: сумісність із Woocommerce Subscriptions 7.7.0.
* Виправлення: сумісність із PHP 8.4
* Виправлення: перенаправлення, але коли сторінки Account WC не існує.

Версія 2.0.3 - Випущено 2025-08-13
* Змінено: увімкнено автоматичні оновлення з новим маркетплейсом.

Версія 2.0.2 - Випущено 2025-07-05
* Змінено: перейменовано на Multisite Ultimate: Woocommerce Integration.
* Додано: сумісність із Woocommerce 9.8.1.
* Додано: сумісність із Woocommerce Subscriptions 7.3.0.
* Виправлення: скасування підписки клієнтом.
* Виправлення: фатальна помилка під час використання блоку оформлення замовлення.
* Покращення: тепер сумісно з високопродуктивними користувацькими таблицями замовлень Woocommerce.
* Виправлення: скасування під час оформлення замовлення WooCommerce усе ще може оновити членство.

Версія 2.0.1 - Випущено 2023-08-09

* Додано: сумісність із Woocommerce 7.9.0.
* Додано: сумісність із Woocommerce Subscriptions 5.3.0.
* Додано: підтримку оновлень членства.
* Додано: сповіщення про пробні періоди та плати за налаштування у Woocommerce.
* Додано: ідентифікацію продуктів Ultimate Multisite Woocommerce за допомогою метазначення.
* Додано: одноразове виправлення, щоб позначити всі пов’язані з Ultimate Multisite продукти Woocommerce.
* Додано: видалено створені Ultimate Multisite продукти зі списку Woocommerce.
* Покращення: створено неповторювану знижку Woocommerce для застосування до кошика.
* Покращення: відновлено повторювану знижку для продукту Woocommerce.
* Покращення: додано мітку повторюваної знижки до продукту Woocommerce.
* Покращення: забезпечено тип продукту під час оформлення замовлення.
* Виправлення: збережено статус членства під час процесу переходу на нижчий план.
* Виправлення: перевірено, чи існує підписка, щоб уникнути помилок під час процесу скасування.
* Виправлення: додано дату початку підписки для використання в підписках Woocommerce.
* Внутрішнє: впроваджено новий процес збірки PHP 8.1.

Версія 2.0.0 - Повне переписування.

* Додано: метод обробки скасування для видалення woo-підписки під час зміни шлюзу або скасування членства;
* Додано: обробник для переходу членств на нижчий і вищий план;
* Покращення: завантаження залежностей woocommerce у формі оновлення клієнта на підсайтах, щоб дозволити оновлення Account;
* Покращення: коректне завантаження кошика Woocommerce, якщо він не існує;
* Покращення: забезпечення роботи з таблицями головного сайту під час обробки оформлення замовлення;
* Покращення: зроблено замовлення поновлення Ultimo на основі значення замовлення підписки Woocommerce, а не останнього платежу;
* Виправлення: посилання кнопки «Перейти до членства WU»;
* Виправлення: встановлення замовлення Ultimo як оплаченого, коли поновлення підписок Woocommerce оплачено;
* Збірка: додано MPB як конструктор;
