# Інтеграція з WooCommerce

WooCommerce Integration працює дуже схожим чином на те, як працює наш Manual Gateway. Він пропонує величезну гнучкість для ваших користувачів, дозволяючи їм обрати будь-яку форму оплати, доступну у вашій установці WooCommerce. І це просто: коли закінчується період оплати користувача, його підписка призупиняється. Водночас на основному сайті мережі буде створено замовлення в установці WooCommerce. Після цього користувач отримує рахунок-фактуру на свою електронну пошту з посиланням для здійснення платежу. Як тільки платіж отримано, підписка поновлюється.

## Функції

- Проста інтеграція між Ultimate Multisite та WooCommerce
- Доступно кілька способів оплати
- Повністю автоматизований процес за допомогою WooCommerce Subscriptions

## Документація

**Налаштування інтеграції з WooCommerce**

**Розширення ваших варіантів оплати за допомогою Ultimate Multisite: Інтеграція з WooCommerce**

## Встановлення

1. Завантажте 'ultimate-multisite-woocommerce' до каталогу '/wp-content/plugins/'
2. Активуйте плагін через меню 'Plugins' у WordPress
3. Дотримуйтесь покрокового майстра для налаштування плагіна

## Посібник із використання

_**ВАЖНА ПРИМІТКА: Ця стаття стосується версії Ultimate Multisite 2.x.**_

_**УВАГА:** Для використання Ultimate Multisite: WooCommerce Integration необхідно, щоб WooCommerce було активовано принаймні на вашому основному сайті._

Ми розуміємо, що _Stripe_ та _PayPal_ недоступні в деяких країнах, що обмежує або ускладнює ефективне використання нашого плагіна користувачами Ultimate Multisite. Тому ми створили **додаток для інтеграції _WooCommerce_**.

_WooCommerce_ — дуже популярний плагін для електронної комерції. Розробники в усьому світі створили доповнення для інтеграції різних платіжних шлюзів. Ми використали це, щоб розширити платіжні шлюзи, які ви можете використовувати в системі білінгу Ultimate Multisite.

## Налаштування

Спочатку вам потрібно встановити додаток WooCommerce Integration, який **безкоштовний для ліцензованих користувачів**. Щоб встановити його, натисніть на Ultimate Multisite у вашій панелі super admin та виберіть **Add-ons**.

![Add-ons page](/img/admin/addons-page.png)

Тут ви знайдете всі додатки Ultimate Multisite. Натисніть на додаток **Ultimate Multisite: WooCommerce Integration**.

![Add-ons page showing WooCommerce Integration](/img/admin/addons-page.png)

Виникне вікно з деталями додатка. Просто натисніть **Install Now** *.*

<!-- Screenshot unavailable: WooCommerce Integration addon install popup with Install Now button -->

Після завершення встановлення ви будете перенаправлені на сторінку плагінів. Тут просто натисніть **Network Activate**, і додаток _WooCommerce_ буде активовано у вашій мережі.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for WooCommerce Integration -->

Після активації, якщо ви все ще не встановили та не активували плагін WooCommerce на своєму сайті, **ви отримаєте нагадування**.

<!-- Screenshot unavailable: Admin notice reminding to install and activate WooCommerce plugin -->

Після встановлення ви побачите додаткову опцію в налаштуваннях платежів Ultimate Multisite. Звідти ви можете **увімкнути** опцію _WooCommerce_.

![Payment gateway settings](/img/config/settings-payment-gateways.png)

Ви можете налаштувати деякі базові параметри, такі як відображувальне ім'я, яке буде показано на фронтенді, та інструкції для керування вашими користувачами під час оформлення замовлення. Потім ви можете перейти до **налаштувань _WooCommerce_ вашого основного сайту**, щоб увімкнути та налаштувати бажаний платіжний шлюз.

## Як керувати платежами

WooCommerce Integration працює дуже схожим чином на те, як працює наш **Manual Gateway**. Недолік полягає в тому, що кожен платіж має бути здійснений вручну наприкінці періоду оплати. Ось як працює процес:

  * Після закінчення білінгового циклу користувача членство **призупиняється**. Водночас на основному сайті мережі буде створено замовлення в установці _WooCommerce_.

  * Користувачі потім отримують рахунок-фактуру на свою електронну пошту з посиланням для здійснення платежу (це посилання також відображається в історії оплати на їхній підсайтовій панелі). Це посилання перенаправляє користувача на **чек-аут _WooCommerce_**, де користувачі зможуть вибрати один із **доступних платіжних шлюзів _WooCommerce_**, щоб здійснити платіж.

  * Як тільки платіж отримано, членство **поновлюється**, платіж реєструється в Ultimate Multisite, а замовлення встановлюється як виконане.

### Деякі важливі примітки:

  * Стандартна функціональність Ultimate Multisite все ще працює з цією інтеграцією. Наприклад, **повернення коштів** зі екрана **Subscription Management** зв'яжеться з WooCommerce для здійснення повернення коштів за допомогою обраного платіжного шлюзу.

  * Ultimate Multisite: WooCommerce Integration покладається на WP-Cron для роботи. Це працює дуже добре більшість часу, але якщо на сайті низький трафік, cron не отримує команду перевірити заплановані події, і подія пропускається, що може спричинити затримки виконання.

## Плагін WooCommerce Subscriptions

Вручну ініціювати платіж щомісячно не є ідеальним для кінцевих користувачів, оскільки підписка очікується автоматично поновлюватися кожен білінговий цикл. Тому ми розробили додаток WooCommerce Integration, щоб **підтримувати плагін WooCommerce Subscriptions**. Це автоматизує платіж, полегшуючи транзакцію для ваших кінцевих користувачів.

Все, що вам потрібно зробити, це **встановити** [Woocommerce Subscriptions plugin](https://woocommerce.com/products/woocommerce-subscriptions/) та **активувати його на вашому основному сайті**. Потім поверніться до налаштувань платежів Ultimate Multisite та **увімкніть інтеграцію**.

І все. Вашим кінцевим користувачам більше не потрібно вручну здійснювати платіж наприкінці свого білінгового циклу.

Окрім створення замовлення на вашому основному сайті в _WooCommerce_, буде також створено підписку для кожного облікового запису.
