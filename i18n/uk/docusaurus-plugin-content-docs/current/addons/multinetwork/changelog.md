# Журнал змін для мультимережі {#multi-network-changelog}

### 1.3.0 {#130}
* Нове: Попередній переглядач шаблонів мережі — переглядайте та тестуйте шаблони мережі на живій панелі перед покупкою.
* Нове: Вибір шаблону мережі при оформленні замовлення — клієнти можуть вибрати шаблон мережі під час процесу реєстрації.
* Нове: SSO за магічною посиланням між мережами — клієнти можуть отримувати доступ до підмереж бездоганно, використовуючи автентифікацію за магічним посиланням.
* Нове: Клонування медіа мережі — медіасайту коректно дублюється при клонуванні мереж із шаблону.
* Виправлення: Клонування мереж тепер працює без додатку для мультиоренданості; привілеї супер-адміністратора відновлюються після клонування.
* Виправлення: Створення мереж тепер сумісне з WooCommerce та всіма платіжними шлюзами.
* Виправлення: Номер порту тепер включається у генерацію резервних доменів.
* Виправлення: CSS для вибору шаблону переміщено до чергової таблиці стилів для коректного відображення на всіх темах.
* Виправлення: Додано захист від відсутнього ключа `network_id` у області видимості запиту сайту.
* Виправлення: Перевірка дозволів мережі перед перемиканням контексту запобігає помилкам властивостей таблиці `wpdb`.
