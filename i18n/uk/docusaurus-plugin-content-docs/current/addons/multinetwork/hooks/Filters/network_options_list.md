# Фільтр: network_options_list

Фільтрує список мережевих опцій, які залежать від домену та шляху мережі.

## Параметри

| Name | Type | Description |
|------|------|-------------|
| $network_options | `array` | Список імен мережевих опцій. |

### З версії

- 1.3.0
### Джерело

Визначено в [`inc/functions/network.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-multinetwork/blob/main/inc/functions/network.php#L1615) на рядку 1615
