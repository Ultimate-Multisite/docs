# Журнал змін Domain Seller

Версія 1.3.0 - Випущено 2026-06-02
- Нове: Додано попередження для адміністратора мережі, коли баланс реселера HostAfrica стає надто низьким
- Нове: Додано автоматичне зіставлення новозареєстрованих доменів із сайтом мережі
- Виправлення: Вимоги до полів реєстранта застосовуються лише під час реєстрації нового домену
- Виправлення: Сповіщення моніторингу балансу зроблено такими, що їх можна відхилити
- Виправлення: Забезпечено збереження платіжних даних реєстранта WooCommerce
- Виправлення: Забезпечено виконання вимог до контактних даних реєстранта під час реєстрації
- Виправлення: Запобігання створенню продуктів реєстрації доменів із націнкою 0%
- Виправлення: Збережено вибір домену та ціни протягом процесу сесії оформлення замовлення
- Виправлення: Покращено відображення валюти цін на домени HostAfrica
- Виправлення: Покращено поведінку атрибута action форми оформлення замовлення, щоб запобігти невідповідностям query-var ядра WP
- Покращено: Додано посилання на документацію з налаштування реселера HostAfrica у підказках із налаштування

Версія 1.2.0 - Випущено 2026-05-25
- Нове: Додано HostAfrica як інтеграцію для продажу доменів із підтримкою оформлення замовлення, майстра налаштування, пошуку, TLD/ціноутворення, реєстрації, поновлення, перенесення, nameserver, DNS, EPP code, registrar lock та ID protection
- Нове: Додано Openprovider як інтеграцію для продажу доменів із підтримкою реселерських цін, реєстрації, поновлення, перенесення, nameserver, DNS, EPP code, registrar lock, WHOIS privacy та синхронізації TLD
- Нове: Додано Hostinger як інтеграцію для продажу доменів із використанням спільного Hostinger API token з основної інтеграції для перевірок доступності, реєстрації, оновлень nameserver, registrar lock та WHOIS privacy
- Покращено: Додано docblock-и для дій і фільтрів життєвого циклу домену для настанов щодо інтеграції для розробників
- Покращено: Оновлено метадані сумісності plugin до WordPress 7.0 у readme додатка
- Покращено: Оновлено шаблони планування, що використовуються для координації майбутніх випусків
