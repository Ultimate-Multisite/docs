# Селектор мови

Ultimate Multisite: Селектор мови при реєстрації

Додайте поле для вибору мови до процесу реєстрації.

## Встановлення {#installation}

1. Завантажте 'ultimate-multisite-language-selector' до каталогу '/wp-content/plugins/'
2. Активуйте плагін через меню 'Plugins' у вашому адміністративному інтерфейсі WordPress Network Admin

## Посібник із використання {#usage-guide}

За допомогою цього безкоштовного аддону ви можете дозволити користувачам вибрати мову, яку вони хочуть бачити на сторінці реєстрації та на сторінці входу.

Після [активації вашої ліцензії](../../user-guide/getting-started/installing-ultimate-multisite.md) ви зможете встановлювати аддони Ultimate Multisite, до яких дає доступ ваша ліцензія.

Перейдіть на сторінку налаштувань Ultimate Multisite, а потім у меню зліва натисніть мітку _Add-ons_.

![Add-ons page](/img/admin/addons-page.png)

_Перейдіть на сторінку Add-ons, щоб розпочати встановлення Language Selector_

Там ви знайдете всі аддони, які ми пропонуємо.

Безкоштовні аддони доступні всім нашим клієнтам, незалежно від тарифного плану, на який вони підписалися.

Натисніть кнопку _Install it Now!_ під мініатюрою аддону. Потім активуйте його на сторінці Plugins.

![Add-ons page with install button](/img/admin/addons-page.png)

_Натисніть кнопку для встановлення аддону_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_Активуйте його на сторінці Plugins_

Тепер перейдіть до меню Налаштування Ultimate Multisite та відкрийте вкладку _Network Settings_. Там, унизу сторінки, ви знайдете новий розділ під назвою _Languages_.

![Settings page](/img/config/settings-general.png)

_З'явився абсолютно новий розділ!_

Ви можете залишити його порожнім, щоб відобразити всі мови як опції, або вибрати ті, які ви хочете зробити доступними. Мови — це ті, які ви встановили у вашій конфігурації WordPress.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_Список мов, доступних у цій установці WP_

Після збереження внесених змін меню вибору мови з обраними вами мовами буде відображено як на сторінці входу, так і на сторінці реєстрації.

_Селектор мови на сторінці входу_

## Встановлення мови через URL {#setting-the-language-via-the-url}

Ви також можете змінити мову сторінки, обійшовши параметр локалі (locale) в URL. Наприклад, якщо ви хочете перенаправити на сторінку реєстрації італійською, додайте ?locale=it_IT до URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Для того, щоб це працювало, мова повинна бути встановлена у вашій мережі WordPress.
