# Журнал змін Site Exporter {#site-exporter-changelog}

Версія: 1.0.6 - Випущено 2026-05-11
* Нове: Експорт сайтів тепер пакує самозавантажуваний файл index.php, тому ZIP-архів можна встановити на свіжому хостингу без окремої установки плагіна.
* Нове: Експорт мережі дозволяє адміністраторам експортувати всі сабсайти в одному архіві зі сторінки адміністратора Site Export.

Версія: 1.0.5 - Випущено 2026-05-05
* Виправлено: Додано `__callStatic` до заглушки WP_CLI polyfill для обробки немокованих методів, таких як `get_config`, що запобігає збоям запуску тестів.

Версія: 1.0.4 - Випущено 2025-11-25

* Виправлено: Виправлено сумісність з Ultimate Multisite 2.4.

Версія: 1.0.2 - Випущено 2025-09-28

* Змінено префікс на ultimate-multisite; оновлено текстову область; збільшено номер версії.

### Версія 1.0.1 - Випущено 2023-08-09 {#version-101---released-on-2023-08-09}

* Покращено: Тепер гарантується отримання правильного шляху до файлу імпорту.
* Виправлено: Вимкнено кеш об'єктів, щоб запобігти помилкам під час імпорту.
* Виправлено: Виправлено проблему з непрацюючим посиланням на завантаження плагіна імпортера.

### Версія 1.0.0 - Випущено 2022-12-23 {#version-100---released-on-2022-12-23}

* Виправлено: Не завантажувалося спливаюче вікно форми експорту;
* Виправлено: Не завантажувалися файли JavaScript на сторінці експорту;
* Внутрішнє: Новий процес збірки;

### Версія 1.0.0-beta.4 - Випущено 2022-01-13 {#version-100-beta4---released-on-2022-01-13}

* Внутрішнє: Додано генератор хуків та фільтрів;
* Внутрішнє: Додано заглушки WP Ultimo для підвищення якості життя розробників;
