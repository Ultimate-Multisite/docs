# Фільтр: wu_site_exporter_plugin_exclusion_list {#filter-wusiteexporterpluginexclusionlist}

Дозволяє розробникам керувати списком плагінів, які можуть бути виключені зі створеного zip-архіву.

## Параметри {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $plugin_list | `array` | Плагіни, які будуть виключені. |
