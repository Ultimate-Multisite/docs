# Material WP

Адмін-інтерфейс WordPress не змінювався віками. Але чи задумувалися ви, як би виглядав адмін-інтерфейс у версії Material Design?

Material WP — це адмін-тема, розроблена, щоб надати вам такий вигляд та відчуття, допомагаючи вам надати клієнтам більш кастомізовану Панель керування (Dashboard).

## Features

- Підтримка користувацького CSS
- Прості налаштування стилів
- Переупорядкування меню
- Просте імпортування/експортування

## Documentation

**Кастомізація вашої адмін-панелі за допомогою Material WP**

## Installation

1. Завантажте 'material-wp' до каталогу '/wp-content/plugins/'
2. Активуйте плагін через меню 'Плагіни' у WordPress

## Usage Guide

Адмін-інтерфейс WordPress не змінювався віками. Але чи задумувалися ви, як би виглядав адмін-інтерфейс у версії Material Design?

Material WP — це адмін-тема, розроблена, щоб надати вам такий вигляд та відчуття, допомагаючи вам надати клієнтам більш кастомізовану Панель керування (Dashboard). Material WP також є частиною преміум-плагінів core від Ultimate Multisite.

Після **встановлення** перейдіть на сторінку _Плагіни_ та натисніть, щоб активувати його у вашій Мережі (Network).
