# AI Site Builder

ultimate-multisite-ai-site-builder

## Вимоги {#requirements}

- WordPress 5.3 або вище
- PHP 7.4 або вище
- Плагін Ultimate Multisite (активний)

## Встановлення {#installation}

1. Завантажте файли аддону до каталогу `/wp-content/plugins/`
2. Активуйте плагін через меню «Плагіни» у WordPress
3. Налаштуйте аддон у мережевому адміністраторі

## Розробка {#development}

### Налаштування {#setup}

```bash
# Встановлення залежностей
composer install
npm install

# Запуск тестів
npm run test

# Перевірка відповідності стандартам коду
vendor/bin/phpcs
vendor/bin/phpstan

# Виправлення проблем зі стилем коду
vendor/bin/phpcbf

# Збірка для продакшену
npm run build
```

```
ultimate-multisite-ai-site-builder/
```

### Тестування {#testing}

Аддон включає комплексний фреймворк для тестування:

- **Модульні тести (Unit Tests)**: Тести на основі PHPUnit із тестовим набором WordPress
- **Базовий клас тестів (Base Test Class)**: Надає допоміжні методи для типових сценаріїв тестування
- **Покриття тестами (Test Coverage)**: Налаштовано для відстеження покриття кодом
- **Тестування Multisite**: Тести виконуються в середовищі Multisite

### Стандарти коду {#code-standards}

- **Стандарти кодування WordPress**: Забезпечується за допомогою PHPCS
- **Сумісність з PHP 7.4+**: Підтримуються сучасні функції PHP
- **Статичний аналіз**: Інтеграція PHPStan для перевірки типів
- **Модернізація коду**: Правила Rector для оновлення PHP

### Система збірки {#build-system}

- **Обробка ресурсів (Asset Processing)**: Мініфікація файлів CSS/JS
- **Локалізація (Translation)**: Генерація файлів POT
- **Пакування (Packaging)**: Створення архіву для розповсюдження
- **Режим розробки (Development Mode)**: Немініфіковані ресурси для налагодження

## Конфігурація {#configuration}

Аддон підтримує різні параметри конфігурації, доступні через інтерфейс мережевого адміністратора.

## Хуки та фільтри {#hooks-and-filters}

### Дії (Actions) {#actions}

- `ultimate-multisite-ai-site-builder_init` - Викликається після ініціалізації аддону
- `ultimate-multisite-ai-site-builder_loaded` - Викликається після завантаження всіх файлів аддону

### Фільтри (Filters) {#filters}

- `ultimate-multisite-ai-site-builder_settings` - Зміна налаштувань аддону
- `ultimate-multisite-ai-site-builder_enabled` - Перевизначення статусу увімкнення/вимкнення аддону
