# Фільтр: wu_captcha_whitelisted {#filter-wucaptchawhitelisted}

Визначає, чи має поточний користувач/запит бути доданим до білого списку (whitelisted).

## Параметри {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $whitelisted | `bool` | Чи потрібно пропускати валідацію captcha. |
