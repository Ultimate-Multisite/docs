# Фільтр: wu_cap_captcha_config

Фільтрує конфігурацію Cap Captcha.

Дозволяє розробникам налаштувати параметри Cap Captcha.

## Параметри

| Name | Type | Description |
|------|------|-------------|
| $config | `array` | Масив конфігурації Cap. |
| $level | `string` | Рівень безпеки (fast, medium, max). |
