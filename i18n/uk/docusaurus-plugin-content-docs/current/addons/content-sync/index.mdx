# Синхронізація контенту {#content-sync}

Ultimate Multisite: Синхронізація контенту

Цей аддон для Ultimate Multisite дозволяє адміністраторам мережі синхронізувати контент з будь-якого сайту на інші сайти в мережі. Він підтримує всі типи записів, включаючи записи, сторінки, користувацькі типи записів та шаблони Elementor. Аддон містить багатоетапний інтерфейс-візар, що дозволяє вибрати контент та цільові сайти, а також виконує пакетну обробку з відгукним відображенням прогресу в реальному часі.

## Функції {#features}

* Вибір контенту: Візуальна сітка карток з мініатюрами та фільтрацією за типами записів
* Усі типи записів: Синхронізує записи, сторінки, товари, шаблони Elementor та будь-який користувацький тип записів
* Вибір сайтів: Дозволяє вибрати сайти, на які потрібно синхронізувати контент, з виключенням джерела та попереднім вибором дочірніх сайтів
* Пакетна обробка: Синхронізує до 10 сайтів одночасно з оновленням прогресу
* Синхронізація медіа: Автоматично копіює та перемапує медіафайли (зображення, відео) з контенту записів на цільові сайти
* Синхронізація таксономій: Зберігає категорії, теги та користувацькі таксономії
* Підтримка Elementor: Повна підтримка шаблонів Elementor з регенерацією CSS та перемапуванням ID шаблонів
* Вирішення конфліктів: Можливість замінити існуючий контент або пропустити його
* Управління кешем: Автоматично очищає кеш Elementor на цільових сайтах після синхронізації

## Вимоги {#requirements}

* WordPress 5.3 або вище
* PHP 7.4 або вище
* Плагін Ultimate Multisite (активний)
* Плагін Elementor (необов'язково, але потрібен для синхронізації шаблонів Elementor)

## Встановлення {#installation}

1. Завантажте 'ultimate-multisite-content-sync' до каталогу '/wp-content/plugins/'
2. Активуйте плагін через меню 'Плагіни' у вашому Адмініструванні мережі WordPress
3. Налаштуйте параметри в Ultimate Multisite → Settings → Content Sync
4. Перейдіть до Ultimate Multisite → Content Sync, щоб розпочати синхронізацію
