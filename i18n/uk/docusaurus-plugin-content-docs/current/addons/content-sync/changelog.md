# Журнал змін Content Sync

### Версія 1.1.0 - Випущено 2025-12-31
* Додано: Підтримка синхронізації всіх типів записів (записи, сторінки, кастомні типи записів).
* Додано: Фільтрація типів записів на кроці вибору контенту.
* Додано: Крок вибору джерела сайту — синхронізація з будь-якого сайту, а не лише з основного.
* Додано: Попередній вибір дочірнього сайту для сайтів-шаблонів.
* Додано: Вилучення медіа з post_content для всіх типів записів.
* Додано: Синхронізація таксономій (категорії, теги, кастомні таксономії).
* Змінено: Перейменовано з "Elementor Template Sync" на "Content Sync".
* Змінено: Крок "Select Templates" перейменовано на "Select Data".
* Покращено: Код, специфічний для Elementor, тепер виконується лише для типу запису elementor_library.

### Версія 1.0.0 - Випущено 2025-12-19
* Початковий реліз.
* Багакроковий майстер для вибору шаблонів та сайтів.
* Пакетна обробка зі зворотним зв'язком про прогрес у реальному часі.
* Повна синхронізація медіа з автоматичним перемапуванням ID.
* Інтеграція з налаштуваннями Ultimate Multisite.
* Підтримка виключення сайтів через мережеві налаштування.
* Очищення кешу Elementor після синхронізації.
* Збереження таксономій шаблонів.
