# Фільтр: ultimate_multisite_content_sync_url

Дозволяє розробникам керувати відносним шляхом до кореня плагіна.

## Параметри {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $dir | `string` | Шлях, відносний до кореня плагіна, до якого ви хочете отримати доступ. |

### Починаючи з {#since}

- 1.0.0
