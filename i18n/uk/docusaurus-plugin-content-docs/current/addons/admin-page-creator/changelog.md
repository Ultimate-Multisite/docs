# Changelog Admin Page Creator

### Версія 1.9.0 - Випущено 2026-01-18 {#version-190---released-on-2026-01-18}

* Виправлено: Критичні помилки при активації.
* Рефакторинг: Класи джерел контенту тепер використовують трейт Singleton із належною ініціалізацією батьківського конструктора.
* Виправлено: Помилки компіляції шаблонів Vue.js, які заважали відображенню селектора типів контенту.
* Виправлено: Відсутні властивості даних Vue (template_id, external_link_url_embedable, external_link_url_checking).
* Виправлено: Редактор TinyMCE не ініціалізувався належним чином при завантаженні сторінки.
* Виправлено: Файл підтримки Divi не завантажувався автоматично через Composer.
* Виправлено: Пошкоджений синтаксис PHP у класі підтримки Divi.
* Покращено: Організація та мініфікація JavaScript-ресурсів.
* Покращено: Сумісність з останніми версіями WordPress та page builder.
