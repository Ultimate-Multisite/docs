# Filter: wu_apc_get_meta

Дозволяє розробникам фільтрувати кінцеві значення та додавати нові сценарії обробки.

## Параметри

| Ім'я | Тип | Опис |
|------|------|-------------|
| $default_value | `string` | Значення за замовчуванням. |
| $meta_type | `string` | Тип метаданих, які потрібно отримати. |
| $meta_key | `string` | Назва метаполя. |

### Починаючи з

- 1.4.0

Визначено в [`inc/wu-apc-functions.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-admin-page-creator/blob/main/inc/wu-apc-functions.php#L93) на рядку 93
