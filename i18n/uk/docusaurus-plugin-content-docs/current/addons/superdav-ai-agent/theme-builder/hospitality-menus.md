# Меню для закладів гостинності {#hospitality-menus}

Функція **Меню для закладів гостинності** дозволяє Theme Builder створювати та вбудовувати структуровані меню страв та напоїв безпосередньо на ваш сайт WordPress.

## Огляд {#overview}

Тепер Theme Builder може створювати професійні, структуровані сторінки меню для закладів гостинності, таких як ресторани, кафе, бари та кейтерингові служби. Ці меню повністю інтегровані у дизайн вашого сайту, а їх оновлення та управління надзвичайно прості.

## Підтримувані типи закладів гостинності {#supported-hospitality-types}

- **Ресторани** — меню повного циклу обслуговування
- **Кафе** — меню кави та легких закусок
- **Бари та лаунжі** — меню напоїв та закусок
- **Пекарні** — меню випічки та хліба
- **Кейтерингові служби** — варіанти меню для заходів
- **Фудтраки** — меню мобільного харчування
- **Пивоварні та виноробні** — меню напоїв з описами

## Структура меню {#menu-structure}

### Категорії меню {#menu-categories}

Меню організовані за категоріями:

- **Закуски** — стартери та невеликі страви
- **Основні страви** — основні страви
- **Гарніри** — супутні страви та овочі
- **Десерти** — солодощі
- **Напої** — напої (алкогольні та безалкогольні)
- **Спеціальні пропозиції** — щоденні або сезонні акції

### Формат позиції меню {#menu-item-format}

Кожна позиція меню включає:

```json
{
  "name": "Смажений лосось",
  "description": "Свіжий атлантичний лосось із соусом з лимоном та маслом",
  "price": "$24.95",
  "dietary_info": ["Без глютену", "Високий вміст білка"],
  "allergens": ["Риба"],
  "availability": "Щодня"
}
```

#### Поля позиції {#item-fields}

| Поле | Тип | Опис |
|-------|------|-------------|
| `name` | string | Назва страви чи напою |
| `description` | string | Детальний опис позиції |
| `price` | string | Ціна (форматована з валютою) |
| `dietary_info` | array | Дієтичні атрибути (Веганське, Без глютену тощо) |
| `allergens` | array | Поширені алергени (Горіхи, Молюски тощо) |
| `availability` | string | Коли доступно (Щодня, Сезонно тощо) |

## Створення меню для закладів гостинності {#creating-hospitality-menus}

### Крок 1: Надання інформації про меню {#step-1-provide-menu-information}

Розкажіть Theme Builder про ваше меню:

```
Створи меню італійського ресторану із закусками,
пастою, основними стравами та десертами. Включи ціни та описи.
```

### Крок 2: Theme Builder генерує меню {#step-2-theme-builder-generates-menu}

Theme Builder:

1. Створює структуровану сторінку меню
2. Оформлює її відповідно до теми вашого сайту
3. Організовує позиції за категоріями
4. Форматує ціни та описи
5. Додає інформацію про дієтичні обмеження та алергени

### Крок 3: Перегляд та налаштування {#step-3-review-and-customize}

Ви можете:

1. Редагувати позиції та ціни в меню
2. Додавати або видаляти категорії
3. Змінювати порядок позицій у категоріях
4. Оновлювати описи та дієтичну інформацію
5. Коригувати стиль та макет

## Варіанти відображення меню {#menu-display-options}

### Повна сторінка меню {#full-menu-page}

Спеціалізована сторінка, що відображає ваше повне меню:

- Організовано за категоріями
- Можливий пошук та фільтрація
- Макет, зручний для друку
- Адаптивний дизайн для мобільних пристроїв

### Віджет меню {#menu-widget}

Вбудовуйте розділи меню на інших сторінках:

- Основні позиції на головній сторінці
- Щоденні акції в бічній панелі
- Меню напоїв на сторінці бару
- Вітрина десертів у футері

### PDF-меню {#menu-pdf}

Генеруйте для завантаження PDF-меню:

- Професійне оформлення
- Якість, готова до друку
- Включає зображення та описи
- Легко ділитися та надсилати електронною поштою

## Дієтична та алергенна інформація {#dietary-and-allergen-information}

### Дієтичні атрибути {#dietary-attributes}

Позначайте позиції дієтичною інформацією:

- **Веганське** — без продуктів тваринного походження
- **Вегетаріанське** — без м'яса
- **Без глютену** — безпечно для людей з целіакією
- **Без молочних продуктів** — без продуктів молока
- **Без горіхів** — без деревних горіхів чи арахісу
- **Низьковуглеводне** — з низьким вмістом вуглеводів
- **Високий вміст білка** — багате на білок

### Попередження про алергени {#allergen-warnings}

Включайте поширені алергени:

- **Горіхи** — деревні горіхи та арахіс
- **Молюски** — ракоподібні та молюски
- **Риба** — всі види риби
- **Молочні продукти** — молоко та молочні продукти
- **Яйця** — курячі яйця
- **Соя** — соєві продукти
- **Глютен** — пшениця та зерна, що містять глютен
- **Кунжут** — кунжутна насіння та олія

## Управління меню {#menu-management}

### Оновлення цін {#updating-prices}

Легко оновлюйте ціни в меню:

1. Перейдіть на сторінку меню
2. Натисніть «Редагувати меню»
3. Оновіть ціни для позицій
4. Збережіть зміни
5. Зміни з'являться на вашому сайті негайно

### Додавання сезонних позицій {#adding-seasonal-items}

Створюйте сезонні варіації меню:

1. Створіть нову версію меню
2. Додайте сезонні позиції
3. Позначте позиції як «Сезонні»
4. Встановіть дати доступності
5. Автоматично відображати протягом сезону

### Управління акціями {#managing-specials}

Відображайте щоденні або щотижневі акції:

1. Створіть категорію «Спеціальні пропозиції»
2. Додайте позиції з датами доступності
3. Виділіть акції на головній сторінці
4. Оновлюйте щодня або щотижня
5. Архівуйте старі акції

## Інтеграція з Theme Builder {#integration-with-theme-builder}

При використанні Theme Builder для закладів гостинності:

1. **Автоматичне виявлення меню** — визначає, чи є ваш бізнес харчовим/напоївним закладом
2. **Створення сторінки меню** — генерує професійні сторінки меню
3. **Відповідність дизайну** — меню відповідають темі вашого сайту
4. **Оптимізація для мобільних пристроїв** — меню красиво відображаються на телефонах
5. **SEO-оптимізація** — меню дружні до пошукових систем

## Найкращі практики {#best-practices}

### Дизайн меню {#menu-design}

- **Чітка організація** — логічна структура категорій
- **Легкозчитувані описи** — апетитні та інформативні
- **Послідовне ціноутворення** — чітка валюта та форматування
- **Професійні фотографії** — високоякісні зображення страв
- **Простір** — не перевантажуйте сторінку

### Контент {#content}

- **Точні описи** — описуйте позиції точно
- **Виділення акцій** — робіть акції помітними
- **Включення алергенів** — завжди перераховуйте алергени
- **Регулярне оновлення** — тримайте ціни та позиції актуальними
- **Використання апетитних формулювань** — робіть позиції смачними

### Доступність {#accessibility}

- **Легкочитабельні шрифти** — використовуйте чіткі, розбірливі шрифти
- **Достатній контраст** — забезпечте читабельність тексту
- **Дієтичні мітки** — чітко позначайте дієтичні варіанти
- **Попередження про алергени** — помітно відображайте алергени
- **Дружній до мобільних пристроїв** — тестуйте на всіх пристроях
