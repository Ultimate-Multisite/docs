# Інструменти та конфігурація

Theme Builder використовує набір інструментів для аналізу, дизайну та створення вашого сайту на WordPress. У версії 1.16.0 **sd-ai-agent/site-scrape став інструментом рівня 1 (Tier 1)**, що робить його доступним за замовчуванням у Theme Builder.

## Доступні інструменти

### Інструменти рівня 1 (Завжди доступні)

Інструменти рівня 1 доступні за замовчуванням у Theme Builder без додаткової конфігурації.

#### sd-ai-agent/site-scrape

**Призначення:** Аналізує існуючі вебсайти для вилучення дизайнерських ідей, структури контенту та функціоналу.

**Можливості:**

- **Аналіз вебсайтів** — сканування та аналіз сайтів конкурентів або джерел натхнення
- **Вилучення дизайну** — ідентифікація кольорів, шрифтів та патернів макету
- **Структура контенту** — розуміння організації та ієрархії сторінок
- **Виявлення функцій** — ідентифікація плагінів та функціоналу
- **Аналіз продуктивності** — перевірка швидкості завантаження та оптимізації сторінки
- **SEO-аналіз** — перегляд метатегів та структурованих даних

**Використання:**

```
Analyze the design of example.com and use it as inspiration for my site.
```

**Що він вилучає:**

- Палітра кольорів та використання кольорів
- Типографіка (шрифти та розміри)
- Структура макета та відступи
- Патерни навігації
- Організація контенту
- Використання зображень та медіа
- Інтерактивні елементи
- Адаптивність для мобільних пристроїв

**Обмеження:**

- Не може сканувати сайти з паролем
- Поважає robots.txt та політику сайту
- Може не захопити динамічний контент
- Сайти з великою кількістю JavaScript можуть мати обмежений аналіз
- Великі сайти можуть вимагати більше часу для аналізу

### Інструменти рівня 2 (Опціонально)

Інструменти рівня 2 доступні, коли їх явно увімкнено в налаштуваннях Theme Builder.

#### Advanced Analytics

Надає детальні показники продуктивності:

- Час завантаження сторінки
- Core Web Vitals
- SEO-оцінка
- Оцінка доступності
- Оцінка найкращих практик

#### Content Optimizer

Аналізує та пропонує покращення для:

- Читабельності
- SEO-оптимізації
- Використання ключових слів
- Структури контенту
- Розміщення закликів до дії (Call-to-action)

### Інструменти рівня 3 (Преміум)

Інструменти рівня 3 вимагають додаткової конфігурації або преміум-функцій.

#### AI Content Generator

Генерує контент для:

- Описів товарів
- Сторінок послуг
- Постів у блозі
- Мета-описів
- Тексту закликів до дії

#### Advanced Design Tools

Надає розширені можливості дизайну:

- Генерація кастомного CSS
- Створення анімації
- Дизайн інтерактивних елементів
- Розширена теорія кольору
- Оптимізація типографіки

## Конфігурація інструментів

### Увімкнення інструментів

Щоб увімкнути додаткові інструменти в Theme Builder:

1. Перейдіть до **Gratis AI Agent → Settings**
2. Перейдіть до **Theme Builder → Tools**
3. Увімкніть/вимкніть інструменти за потреби
4. Збережіть налаштування

### Дозволи інструментів

Деякі інструменти вимагають дозволів:

- **Site scraping** — вимагає доступу до інтернету
- **Analytics** — вимагає інтеграції Google Analytics
- **Content generation** — вимагає API-ключів
- **Advanced features** — може вимагати преміум-підписки

### API-ключі та облікові дані

Налаштуйте API-ключі для інструментів, які їх вимагають:

1. Перейдіть до **Gratis AI Agent → Settings → API Keys**
2. Введіть облікові дані для кожного інструменту
3. Перевірте з'єднання
4. Збережіть безпечно

## Використання sd-ai-agent/site-scrape

### Базове використання

Попросіть Theme Builder проаналізувати вебсайт:

```
Analyze the design of my competitor's site at competitor.com
and suggest design improvements for my site.
```

### Запит конкретного аналізу

Запит конкретних типів аналізу:

```
Extract the color palette from example.com and use it as inspiration.
```

```
Analyze the navigation structure of example.com and apply similar
organization to my site.
```

```
Check the mobile responsiveness of example.com and ensure my site
is equally responsive.
```

### Порівняльний аналіз

Порівняння кількох сайтів:

```
Compare the designs of site1.com and site2.com and create a design
that combines the best elements of both.
```

## Обмеження та міркування щодо інструментів

### Обмеження швидкості

- Сканування обмежується для запобігання перевантаженню сервера
- Кілька запитів до одного сайту можуть бути обмежені (throttled)
- Великі сайти можуть вимагати більше часу для аналізу

### Точність контенту

- Динамічний контент може бути не повністю захоплений
- Контент, згенерований JavaScript, може бути неповним
- Дані в реальному часі можуть не відображатися
- Деякі контент може бути захищений стіною оплати

### Юридичні та етичні міркування

- Поважайте robots.txt та політику сайту
- Не скануйте контент, захищений авторським правом, для відтворення
- Використовуйте аналіз для натхнення, а не для копіювання
- Переконайтеся, що ви маєте права на використання вилученого контенту
- Дотримуйтесь умов обслуговування аналізованих сайтів

### Вплив на продуктивність

- Сканування великих сайтів може зайняти час
- Кілька одночасних сканувань можуть бути повільнішими
- Швидкість залежить від мережевого підключення
- Великі масиви даних можуть вимагати більше обробки

## Найкращі практики

### Використання аналізу сайтів

1. **Аналізуйте кілька сайтів** — збирайте натхнення з кількох джерел
2. **Зосередьтеся на структурі** — вивчайте патерни макету та організації
3. **Вилучайте кольори** — використовуйте палітри кольорів як відправні точки
4. **Вивчайте типографіку** — ідентифікуйте шрифтові комбінації, які вам подобаються
5. **Переглядайте навігацію** — розумійте організацію меню

### Етичне сканування

1. **Поважайте robots.txt** — дотримуйтесь правил сайту
2. **Не копіюйте контент** — використовуйте аналіз лише для натхнення
3. **Перевіряйте права** — переконайтеся, що ви можете використовувати вилучений контент
4. **Вказуйте джерела** — цитуйте джерела натхнення
5. **Дотримуйтесь умов** — дотримуйтесь умов обслуговування сайту

### Максимізація ефективності інструментів

1. **Будьте конкретними** — запитуйте конкретні типи аналізу
2. **Надайте контекст** — поясніть мету вашого сайту
3. **Встановіть очікування** — опишіть свої дизайнерські цілі
4. **Ітеруйте** — вдосконалюйтеся на основі результатів
5. **Комбінуйте інструменти** — використовуйте кілька інструментів для комплексного аналізу

## Усунення несправностей

### Сайт не сканується

- Перевірте, чи сайт публічно доступний
- Переконайтеся, що robots.txt дозволяє сканування
- Спробуйте інший сайт
- Перевірте підключення до інтернету
- Зверніться до служби підтримки, якщо проблема зберігається

### Неповний аналіз

- Сайт може містити динамічний контент
- JavaScript може бути не повністю відрендерений
- Великі сайти можуть перевищити час виконання (timeout)
- Спробуйте проаналізувати конкретні сторінки замість усього сайту
- Запитуйте конкретні типи аналізу

### Низька продуктивність

- Аналіз великих сайтів займає більше часу
- Кілька одночасних сканувань повільніші
- Швидкість залежить від мережевого підключення
- Спробуйте під час непікових годин
- Спочатку проаналізуйте менші розділи
