# Інтерв'ю для визначення концепції

**Інтерв'ю для визначення концепції** — це перший етап процесу Theme Builder. Воно збирає інформацію про ваш сайт, бізнес та дизайнерські уподобання, щоб створити персоналізовану тему.

## Огляд

Інтерв'ю для визначення концепції проведе вас через серію запитань, щоб зрозуміти ваш бренд, цільову аудиторію та дизайнерські цілі. У версії 1.16.0 інтерв'ю тепер включає **етап завантаження фотографій**, який дозволяє надати візуальне натхнення для дизайну.

## Етапи інтерв'ю

### 1. Інформація про бізнес

- Назва та слоган сайту
- Тип бізнесу або галузь
- Основні бізнес-цілі
- Опис цільової аудиторії

### 2. Дизайнерські уподобання

- Бажана колірна схема
- Стиль дизайну (сучасний, класичний, мінімалістичний, грайливий тощо)
- Натхненні вебсайти чи бренди
- Обов'язкові дизайнерські елементи

### 3. Завантаження фотографій (Нове у v1.16.0)

Завантажте одну або кілька фотографій, щоб вплинути на напрямок дизайну:

#### Дозволені формати

| Формат | Розширення | Деталі |
|--------|-----------|---------|
| **JPEG** | `.jpg`, `.jpeg` | Рекомендовано для фотографій |
| **PNG** | `.png` | Підтримує прозорість |
| **WebP** | `.webp` | Сучасний формат, менший розмір файлу |
| **GIF** | `.gif` | Тільки статичні зображення (без анімації) |

#### Вимоги до файлів

- **Максимальний розмір файлу:** 10 МБ
- **Мінімальні розміри:** 400 × 300 пікселів
- **Рекомендовані розміри:** 1200 × 800 пікселів або більше
- **Співвідношення сторін:** Приймається будь-яке

#### Як фотографії впливають на напрямок дизайну

Завантажені фотографії допомагають Theme Builder зрозуміти:

- **Кольорову палітру** — виділяються домінантні кольори та використовуються в темі
- **Візуальний стиль** — сучасний, рустикальний, мінімалістичний, яскравий тощо.
- **Настрій та атмосферу** — професійний, креативний, дружній, розкішний тощо.
- **Дизайнерські елементи** — візерунки, текстури та композиційні стилі
- **Натхнення типографікою** — стилі шрифтів, які доповнюють візуальну тему

### 4. Структура контенту

- Основні необхідні сторінки (Головна, Про нас, Послуги, Контакти тощо)
- Чи потрібен розділ блогу чи новин
- Чи потрібна функціональність електронної комерції
- Спеціальні функції чи інтеграції

### 5. Огляд та підтвердження

- Підсумок усієї наданої інформації
- Підтвердження перед початком генерації дизайну

## Найкращі практики завантаження фотографій

### Що завантажувати

- **Натхнення бренду** — логотипи, колірні зразки або брендбук
- **Сайти конкурентів** — скріншоти сайтів, які вам подобаються
- **Мудборди** — колекції зображень, що відображають бажану естетику
- **Фотографії продуктів** — якщо ви маєте продукти для демонстрації
- **Фотографії команди чи офісу** — для особистого чи професійного брендингу
- **Образні зображення** — що представляють вашу цільову аудиторію

### Чого уникати

- **Зображення низької якості** — розмиті або пікселізовані фотографії
- **Непов'язані зображення** — фотографії, які не відображають ваш бренд
- **Зображення з великою кількістю тексту** — скріншоти з великою кількістю тексту
- **Матеріали з авторським правом** — переконайтеся, що у вас є права на використання зображень
- **Анімовані GIF** — обробляються лише статичні зображення

### Поради для найкращих результатів

1. **Завантажте 2–3 зображення** — кілька зображень дають кращий контекст
2. **Використовуйте фотографії високої якості** — найкраще працюють чіткі, добре освітлені зображення
3. **Включіть різноманіття** — змішуйте кольори, стилі та об'єкти
4. **Надайте контекст** — коротко опишіть, чому ви обрали кожне зображення
5. **Будьте конкретними** — якщо вам подобається певний елемент, вкажіть на нього

## Результат напрямку дизайну

Після інтерв'ю для визначення концепції Theme Builder генерує:

- **Кольорову палітру** — основні, вторинні та акцентні кольори
- **Типографіку** — сімейства шрифтів для заголовків та основного тексту
- **Стиль макету** — відступи, вирівнювання та розташування компонентів
- **Візуальні елементи** — іконки, візерунки та декоративні елементи
- **Мудборд** — візуальне представлення напрямку дизайну

## Наступні кроки

Після завершення інтерв'ю для визначення концепції:

1. Огляньте згенерований напрямок дизайну
2. Надайте відгук або запросіть корективи
3. Перейдіть до етапу [Напрямок дизайну](./design-direction.md)
4. Theme Builder створює вашу кастомну тему
