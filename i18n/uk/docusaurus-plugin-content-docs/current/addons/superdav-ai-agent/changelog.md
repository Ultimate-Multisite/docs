# Changelog

## 1.16.0 — Released on 2026-05-20

### Новинки

- **Генерація SVG логотипа** — Тема-конструктор тепер може генерувати та вбудовувати кастомні SVG-логотипи з безпечною санітизацією за простором імен
- **Завантаження фотографії в інтерв'ю для виявлення потреб** — Інтерв'ю для виявлення потреб у Тема-конструкторі тепер включає етап завантаження фотографії для більш детального контексту дизайну
- **Перевірка контрасту палітри** — можливість перевіряти пари кольорів на відповідність стандартам WCAG перед їх застосуванням до теми
- **Готельно-ресторанні меню** — Тема-конструктор тепер може генерувати структуровані сторінки меню їжі та напоїв для готельно-ресторанного бізнесу
- **Відображення прев'ю для десктопа та мобільного пристроїв** — переглядайте свій дизайн на десктопних та мобільних пристроях під час вибору напрямку дизайну
- **Параметр мітки навігації** — Функція створення меню тепер підтримує окрему `navigation_label`, відмінну від заголовка сторінки
- **Доступність інструменту Tier 1** — sd-ai-agent/site-scrape тепер є інструментом Tier 1, доступним за замовчуванням у Тема-конструкторі

### Виправлено

- **Кеш клієнта AI** — тепер підтримується за допомогою транзієнтів для збереження даних між запитами, запобігаючи втраті даних під час тривалих завдань агента
- **Посилання дій у рядку плагіна** — виправлено та перейменовано для більшої ясності

## 1.10.0 — Released on 2026-05-05

### Новинки

- **Пошук в інтернеті Tavily** — додано Tavily як провайдера пошуку для отримання більш насичених результатів пошуку в інтернеті поряд із Brave Search
- **Вбудовані навички, адаптовані до теми** — Блок-теми, Класичні теми, Kadence Blocks та посібники з навичок Kadence Theme тепер постачаються з плагіном
- **Функціонал форми контакту для сайт-білдера** — додайте контактну форму на будь-яку сторінку безпосередньо з інтерфейсу чату

### Покращено

- **Рефакторинг інтеграції WooCommerce** — тепер використовує нативні API WooCommerce для кращої надійності та сумісності
- **Автоматичне оновлення списку провайдерів** — відбувається при активації або деактивації будь-якого плагіна

### Виправлено

- **Функціонал navigate-to** — виправлено нескінченний цикл перезавантаження на деяких сторінках адміністратора
- **Функціонал list-posts** — тепер коректно вирішує назви категорій та тегів до слагів
- **Команди WP-CLI** — відновлено відсутні псевдоніми просторів імен після попереднього рефакторингу
- **Автоматизація подій** — коректно обробляє сайти, для яких таблиці автоматизації ще не створено
- **Функціонал memory-save** — тепер використовує правильний префікс простору імен у конструкторі системних інструкцій
- **Результати інструменту Scalar** — тепер коректно обгортаються перед поверненням до AI
- **Статистика використання** — тепер коректно обробляє старий формат ключа функціоналу при оновленні зі старих версій
