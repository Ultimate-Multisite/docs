# Журнал змін

## 1.18.0 — Випущено 2026-06-29

### Нове

- **Інструменти Google Calendar** — читання налаштованих календарів і подій для автоматизацій з урахуванням розкладу
- **Зіставлення контактів і помічники для учасників** — зіставлення учасників календаря з користувачами та контактами сайту
- **Шлюзи людського схвалення та записи нагадувань** — призупинення автоматизацій для перевірки та уникнення дублювання сповіщень
- **Провайдер SMS TextBee** — надсилання налаштованих текстових сповіщень через TextBee
- **Розширений супровідний пакет** — додано інструменти довіреного розробника для файлової системи, бази даних, WP-CLI, REST-диспетчера, конструктора плагінів, git-знімків, керування користувачами та бенчмаркінгу, що поширюються окремо від збірки WordPress.org

### Покращено

- **Налаштування керованого сервісу Superdav** — додано endpoints розміщеного сервісу та автоматичне підготовлення підключення для підтримуваних сайтів
- **Пакування випуску** — збирання окремих ZIP-архівів ядра та Advanced, публікація обох на GitHub і надсилання лише основного пакета до WordPress.org

### Виправлено

- **Надійність AI-запитів** — покращено вибір моделі, тайм-аути запитів, типову модель налаштування, обробку тексту міркувань і рекомендації щодо повторних спроб для недійсних викликів інструментів
- **Посилення календаря та нагадувань** — посилено токени Google Calendar і дедуплікацію нагадувань
- **Вступне налаштування та відновлення схвалень** — виправлено запуски вступного налаштування на frontend і відновлення підтверджених можливостей
- **Проблеми перевірки пакування WordPress.org** — враховано відгуки перевірки пакування для основного випуску

## 1.16.0 — Випущено 2026-05-20

### Нове

- **Можливість Generate Logo SVG** — Theme Builder тепер може генерувати та вбудовувати власні SVG-логотипи з безпечною для просторів імен санітизацією
- **Завантаження фото в інтерв’ю для виявлення потреб** — інтерв’ю Theme Builder для виявлення потреб тепер містить крок завантаження фото для багатшого контексту дизайну
- **Можливість Validate Palette Contrast** — перевірка пар кольорів на відповідність WCAG перед застосуванням до теми
- **Меню для сфери гостинності** — Theme Builder тепер може генерувати структуровані сторінки меню їжі та напоїв для бізнесів сфери гостинності
- **Рендеринг попереднього перегляду для desktop і mobile** — попередньо переглядайте свій дизайн на desktop і mobile-пристроях під час вибору напряму дизайну
- **Параметр навігаційної мітки** — можливість Create Menu тепер підтримує окремий `navigation_label`, відмінний від заголовка сторінки
- **Доступність інструмента Tier 1** — sd-ai-agent/site-scrape тепер є інструментом Tier 1, доступним за замовчуванням у Theme Builder

### Виправлено

- **Кеш AI Client** — тепер базується на transients для збереження між запитами, запобігаючи втраті даних під час тривалих завдань агента
- **Посилання дій у рядку плагіна** — виправлено та перейменовано для ясності

## 1.10.0 — Випущено 2026-05-05

### Нове

- **Інтернет-пошук Tavily** — додано Tavily як провайдера пошуку для багатших результатів інтернет-пошуку поряд із Brave Search
- **Вбудовані навички з урахуванням теми** — посібники з навичок Block Themes, Classic Themes, Kadence Blocks і Kadence Theme тепер постачаються з плагіном
- **Можливість контактної форми конструктора сайту** — додано контактну форму на будь-яку сторінку безпосередньо з інтерфейсу чату

### Покращено

- **Рефакторинг інтеграції WooCommerce** — тепер використовує нативні API WooCommerce для кращої надійності та сумісності
- **Список провайдерів оновлюється автоматично** — коли будь-який плагін активовано або деактивовано

### Виправлено

- **Можливість navigate-to** — виправлено нескінченний цикл перезавантаження на деяких сторінках адміністрування
- **Можливість list-posts** — тепер правильно перетворює назви категорій і тегів на slugs
- **Команди WP-CLI** — відновлено відсутні псевдоніми просторів імен після попереднього рефакторингу
- **Автоматизація подій** — коректно обробляє сайти, де таблиці автоматизації ще не створені
- **Можливість memory-save** — тепер використовує правильний префікс простору імен у конструкторі системних інструкцій
- **Скалярні результати інструментів** — тепер правильно обгортаються перед поверненням до AI
- **Статистика використання** — тепер правильно обробляє застарілий формат ключа можливості під час оновлення зі старіших версій
