# Superdav AI Agent {#superdav-ai-agent}

Superdav AI Agent — це розширений AI-асистент для WordPress, який доповнює Gratis AI Agent навичками з урахуванням теми, розширеними можливостями інтернет-пошуку, функціями конструктора сайтів і додатковими інтеграціями автоматизації. Версія 1.18.0 додає інструменти читання Google Calendar, зіставлення контактів і учасників, етапи схвалення людиною, дедуплікацію нагадувань, SMS-сповіщення TextBee, підготовку керованого сервісу Superdav і окремо поширюваний супровідний пакет Advanced для довірених робочих процесів розробників.

## Ключові можливості {#key-features}

- **Навички з урахуванням теми** — автоматично адаптує рекомендації та можливості на основі вашої активної теми WordPress і встановлених плагінів
- **Кілька провайдерів інтернет-пошуку** — вибирайте між Brave Search і Tavily для багатших результатів пошуку та розширених дослідницьких можливостей
- **Контактні форми конструктора сайтів** — створюйте повністю функціональні контактні форми безпосередньо з інтерфейсу чату, використовуючи встановлений плагін форм
- **Покращений конструктор сайтів** — удосконалені багатофазні плани створення сайтів із кращим виявленням плагінів і відновленням після помилок
- **Вбудовані навички тем** — спеціалізовані рекомендації для Block Themes, Classic Themes, Kadence Blocks і Kadence Theme
- **Покращена інтеграція WooCommerce** — перероблено для використання нативних API WooCommerce для кращої надійності та сумісності
- **Автоматичне виявлення провайдерів** — автоматично виявляє та оновлює доступні плагіни й теми, коли їх активують або деактивують
- **Генерація власного логотипа** — генеруйте та вбудовуйте власні SVG-логотипи з автоматичною безпечною для просторів імен санітизацією
- **Перевірка контрастності палітри** — перевіряйте пари кольорів на відповідність вимогам доступності WCAG перед застосуванням до вашої теми
- **Виявлення з використанням фотографій** — завантажуйте фотографії під час інтерв’ю виявлення Theme Builder, щоб вплинути на напрям дизайну
- **Меню для індустрії гостинності** — генеруйте структуровані сторінки меню страв і напоїв для ресторанів, кафе та бізнесів індустрії гостинності
- **Попередній перегляд на настільних і мобільних пристроях** — переглядайте свій дизайн на настільних і мобільних пристроях під час вибору напряму дизайну
- **Сканування сайтів рівня 1** — sd-ai-agent/site-scrape тепер доступний за замовчуванням для аналізу сайтів конкурентів і джерел натхнення
- **Інструменти читання Google Calendar** — читайте налаштовані календарі та події, щоб автоматизації могли враховувати контекст майбутнього розкладу
- **Зіставлення контактів і учасників** — зіставляйте учасників календаря з відомими користувачами сайту та контактами перед створенням подальших завдань
- **Етапи схвалення людиною** — призупиняйте чутливі автоматизації для перегляду перед надсиланням повідомлень, відновленням роботи або внесенням змін
- **Записи нагадувань і дедуплікація** — відстежуйте нагадування, які вже було надіслано, щоб повторні запуски автоматизації не сповіщали того самого отримувача двічі
- **SMS-сповіщення TextBee** — надсилайте налаштовані текстові сповіщення через TextBee, коли ввімкнено SMS-провайдера
- **Підготовка керованого сервісу Superdav** — підключайте підтримувані сайти до розміщених кінцевих точок Superdav з автоматичним налаштуванням замість ручної конфігурації кінцевої точки
- **Супровідний пакет Advanced** — окремо поширювані інструменти для довірених розробників для робочих процесів файлової системи, бази даних, WP-CLI, REST-диспетчера, конструктора плагінів, git-знімка, керування користувачами та бенчмарків

## Вимоги {#requirements}

- WordPress 6.0 або вище
- PHP 8.1 або вище
- WordPress Multisite (активований у мережі) або встановлення на одному сайті
- Активний плагін-конектор AI-провайдера (Anthropic, OpenAI або сумісний)
- Gratis AI Agent v1.9.0 або вище (Superdav AI Agent розширює Gratis AI Agent)

## Встановлення {#installation}

1. Завантажте файли плагіна до `/wp-content/plugins/superdav-ai-agent`
2. Активуйте плагін через екран **Плагіни** у WordPress
3. Переконайтеся, що Gratis AI Agent також активовано
4. Перейдіть до **Gratis AI Agent → Налаштування** і за потреби налаштуйте свого провайдера інтернет-пошуку, підключення до керованого сервісу Superdav, Google Calendar або облікові дані TextBee

Пакет WordPress.org містить основні можливості Superdav AI Agent. Супровідний пакет Advanced збирається та поширюється окремо для середовищ, де інструменти довірених розробників явно встановлено й авторизовано.

## Навички з урахуванням теми {#theme-aware-skills}

Superdav AI Agent автоматично виявляє вашу активну тему та встановлені плагіни, завантажуючи відповідну навичку з урахуванням теми в контекст асистента. Доступні навички включають:

| Навичка | Застосовується до |
|---|---|
| **Block Themes** | Теми, що використовують блокову архітектуру WordPress (Full Site Editing) |
| **Classic Themes** | Традиційні теми WordPress, що використовують PHP-шаблони та класичний редактор |
| **Kadence Blocks** | Сайти з активним плагіном Kadence Blocks |
| **Kadence Theme** | Сайти з темою Kadence як активною темою |

Див. [Навички з урахуванням теми](../../user-guide/configuration/theme-aware-skills), щоб отримати докладні рекомендації щодо кожної навички.

## Провайдери інтернет-пошуку {#internet-search-providers}

Superdav AI Agent v1.10.0 додає **Tavily** як альтернативу Brave Search для інтернет-пошукових запитів. Обидва провайдери пропонують результати пошуку в реальному часі та можуть бути налаштовані в **Gratis AI Agent → Налаштування → Розширені**.

| Постачальник | Можливості | Налаштування |
|---|---|---|
| **Brave Search** | Швидкі результати пошуку з акцентом на приватність | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Багатші результати пошуку, розширені можливості дослідження | [Tavily API](https://tavily.com/) |

Див. [Інтернет-пошук](../../user-guide/configuration/internet-search) для інструкцій із налаштування.

## Робочі процеси календаря, схвалення та сповіщень {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 розширює робочі процеси автоматизації за межі завдань зі створення контенту та сайтів:

- **Інструменти Google Calendar** дають агенту змогу читати налаштовані календарі та деталі подій, коли доступні облікові дані календаря.
- **Помічники для контактів і учасників** зіставляють учасників подій із користувачами WordPress або відомими контактами перед підготовкою нагадувань чи подальших завдань.
- **Шлюзи схвалення людиною** дозволяють автоматизації призупинятися, доки авторизований користувач не перегляне запропоновану дію.
- **Записи нагадувань** запобігають дублюванню календарних нагадувань або сповіщень, коли автоматизацію повторюють або планують неодноразово.
- **Підтримка TextBee SMS** вмикає сповіщення текстовими повідомленнями, коли постачальника TextBee налаштовано.

Використовуйте ці інструменти для допомоги з урахуванням розкладу, подальших дій щодо учасників і робочих процесів нагадувань, які все ще потребують перевірки людиною перед надсиланням чутливих повідомлень.

## Налаштування {#configuration}

Усі налаштування розташовані в **Gratis AI Agent → Settings** в адмінці WordPress. Superdav AI Agent додає такі параметри налаштування:

| Налаштування | Опис |
|---|---|
| **Internet Search Provider** | Виберіть між Brave Search і Tavily (необов’язково) |
| **Tavily API Key** | API-ключ для інтернет-пошуку Tavily (якщо вибрано Tavily) |
| **Managed Superdav Service** | Підключайте підтримувані сайти до розміщеного сервісу Superdav з автоматичним наданням endpoint |
| **Google Calendar Credentials** | Налаштуйте доступ до календаря, щоб агент міг читати календарі та події |
| **TextBee SMS Provider** | Налаштуйте облікові дані TextBee для затверджених робочих процесів SMS-сповіщень |

## Розширений супровідний пакет {#advanced-companion-package}

Superdav AI Agent v1.18.0 додає окремо розповсюджуваний розширений супровідний пакет для довірених робочих процесів розробників. Ці можливості не входять до основного пакета WordPress.org. Установлюйте супровідний пакет лише на сайтах, де адміністратори явно довіряють агенту доступ до інструментів розробника.

Розширені інструменти включають операції з файловою системою, доступ до бази даних, WP-CLI, робочі процеси диспетчера REST, утиліти конструктора plugin, знімки git, помічники керування користувачами та інструменти бенчмаркінгу. Обмежуйте ці інструменти довіреними адміністраторами та переглядайте налаштування шлюзу схвалення перед увімкненням їх на production-сайтах.
