# Superdav AI Agent

Superdav AI Agent — це розширений AI-помічник для WordPress, який доповнює Gratis AI Agent функціоналом, адаптованим до теми, покращеним пошуком в інтернеті та вдосконаленими можливостями конструктора сайтів. Версія 1.16.0 додає генерацію кастомних логотипів, валідацію контрасту палітри, завантаження фотографій під час інтерв'ю для виявлення потреб, генерацію структурованих меню для закладів гостинності та попередній перегляд на десктопі/мобільному пристроях у Theme Builder.

## Основні функції

- **Функціонал, адаптований до теми** — автоматично адаптує рекомендації та можливості на основі активної теми WordPress та встановлених плагінів.
- **Кілька провайдерів пошуку в інтернеті** — обирайте між Brave Search та Tavily для отримання більш детальних результатів пошуку та розширених дослідницьких можливостей.
- **Форми контактів у конструкторі сайтів** — створюйте повністю функціональні форми контактів безпосередньо з інтерфейсу чату, використовуючи ваш встановлений плагін для форм.
- **Покращений конструктор сайтів** — вдосконалені багатоетапні плани створення сайтів із кращим виявленням плагінів та відновленням після помилок.
- **Вбудовані функції, специфічні для тем** — спеціалізовані рекомендації для Block Themes, Classic Themes, Kadence Blocks та Kadence Theme.
- **Покращена інтеграція з WooCommerce** — рефакторинг для використання нативних API WooCommerce, що забезпечує кращу надійність та сумісність.
- **Автоматичне виявлення провайдера** — автоматично виявляє та оновлює доступні плагіни та теми при їх активації чи деактивації.
- **Генерація кастомних логотипів** — генеруйте та вбудовуйте кастомні SVG-логотипи з автоматичною санітизацією, безпечною для простору імен.
- **Валідація контрасту палітри** — перевіряйте комбінації кольорів на відповідність стандартам WCAG, перш ніж застосовувати їх до теми.
- **Покращене виявлення за допомогою фото** — завантажуйте фотографії під час інтерв'ю для виявлення потреб у Theme Builder, щоб вплинути на напрямок дизайну.
- **Меню для закладів гостинності** — генеруйте структуровані сторінки меню страв та напоїв для ресторанів, кафе та закладів гостинності.
- **Попередній перегляд на десктопі та мобільному** — переглядайте свій дизайн на десктопних та мобільних пристроях під час вибору напрямку дизайну.
- **Збір даних з сайтів рівня 1** — sd-ai-agent/site-scrape тепер доступний за замовчуванням для аналізу сайтів конкурентів та джерел натхнення.

## Вимоги

- WordPress 6.0 або вище
- PHP 8.1 або вище
- WordPress Multisite (активовано на рівні мережі) або односайтова інсталяція
- Активний плагін-коннектор AI-провайдера (Anthropic, OpenAI або сумісний)
- Gratis AI Agent v1.9.0 або вище (Superdav AI Agent розширює Gratis AI Agent)

## Встановлення

1. Завантажте файли плагіна до `/wp-content/plugins/superdav-ai-agent`
2. Активуйте плагін на екрані **Плагіни** у WordPress
3. Переконайтеся, що Gratis AI Agent також активований
4. Перейдіть до **Gratis AI Agent → Налаштування** та налаштуйте свого провайдера пошуку в інтернеті (необов'язково)

## Швидкий старт

Після активації Superdav AI Agent автоматично покращує вашу інсталяцію Gratis AI Agent, додаючи функціонал, адаптований до теми, та додаткові можливості. Відкрийте панель **Gratis AI Agent** у бічній панелі адміністратора WordPress та почніть користуватися новими функціями.

**Приклади інструкцій:**

```
Додайте форму контактів на сторінку "Контакти".
```

```
Знайдіть в інтернеті найкращі практики безпеки WordPress.
```

```
Створіть секцію-герой, використовуючи блокові шаблони активної теми.
```

## Функціонал, адаптований до теми

Superdav AI Agent автоматично виявляє вашу активну тему та встановлені плагіни, завантажуючи відповідний функціонал, адаптований до теми, у контекст помічника. Доступні функції включають:

| Функція | Застосовується до |
|---|---|
| **Block Themes** | Теми, що використовують блокову (Full Site Editing) архітектуру WordPress |
| **Classic Themes** | Традиційні теми WordPress, що використовують PHP-шаблони та класичний редактор |
| **Kadence Blocks** | Сайти з активним плагіном Kadence Blocks |
| **Kadence Theme** | Сайти з активною темою Kadence |

Дивіться [Функціонал, адаптований до теми](../../user-guide/configuration/theme-aware-skills) для детальних рекомендацій щодо кожної функції.

## Провайдери пошуку в інтернеті

Superdav AI Agent v1.10.0 додає **Tavily** як альтернативу Brave Search для пошукових запитів в інтернеті. Обидва провайдери пропонують результати пошуку в реальному часі та можуть бути налаштовані в **Gratis AI Agent → Налаштування → Advanced**.

| Провайдер | Функції | Налаштування |
|---|---|---|
| **Brave Search** | Швидкі результати пошуку, орієнтовані на конфіденційність | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Більш детальні результати пошуку, розширені дослідницькі можливості | [Tavily API](https://tavily.com/) |

Дивіться [Пошук в інтернеті](../../user-guide/configuration/internet-search) для інструкцій із налаштування.

## Форми контактів у конструкторі сайтів

Агент Site Builder тепер може створювати форми контактів безпосередньо з інтерфейсу чату. Просто попросіть Site Builder створити форму контактів, і він автоматично виявить ваш встановлений плагін для форм (Contact Form 7, WPForms, Fluent Forms або Gravity Forms) та згенерує готову до використання форму.

Дивіться [Форми контактів у конструкторі сайтів](../../user-guide/administration/site-builder-contact-forms) для детальних рекомендацій.

## Функції Theme Builder

Theme Builder у Superdav AI Agent v1.16.0 включає потужні нові можливості:

### Інтерв'ю для виявлення потреб із завантаженням фото

Інтерв'ю для виявлення потреб у Theme Builder тепер включає етап завантаження фотографій, де ви можете надати візуальне натхнення. Завантажте фотографії вашого бренду, сайтів конкурентів або джерел натхнення, щоб допомогти Theme Builder зрозуміти ваш напрямок дизайну.

Дивіться [Інтерв'ю для виявлення потреб](./theme-builder/discovery-interview.md) для детальних рекомендацій.

### Напрямок дизайну з попереднім переглядом на десктопі та мобільному

Перегляньте свій дизайн як на десктопних, так і на мобільних пристроях, перш ніж Theme Builder згенерує вашу повну тему. Переглядайте попередні перегляди в реальному часі на кількох адаптивних точках розриву.

Дивіться [Напрямок дизайну](./theme-builder/design-direction.md) для детальних рекомендацій.

### Меню для закладів гостинності

Theme Builder тепер може генерувати структуровані сторінки меню страв та напоїв для ресторанів, кафе, барів та інших закладів гостинності. Меню включають ціни, описи, інформацію про дієтичні обмеження та попередження про алергени.

Дивіться [Меню для закладів гостинності](./theme-builder/hospitality-menus.md) для детальних рекомендацій.

### Нові можливості

- **[Generate Logo SVG](./abilities/generate-logo-svg.md)** — створюйте кастомні SVG-логотипи з автоматичною санітизацією
- **[Validate Palette Contrast](./abilities/validate-palette-contrast.md)** — перевіряйте комбінації кольорів на відповідність WCAG
- **[Create Menu](./abilities/create-menu.md)** — створюйте навігаційні меню з окремими мітками навігації

Дивіться [Інструменти та конфігурація](./theme-builder/tools.md) для інформації про доступні інструменти та інструмент збору даних з сайтів рівня 1.

## Конфігурація

Усі налаштування знаходяться у розділі **Gratis AI Agent → Налаштування** в адмін-панелі WordPress. Superdav AI Agent додає такі параметри конфігурації:

| Налаштування | Опис |
|---|---|
| **Internet Search Provider** | Оберіть між Brave Search та Tavily (необов'язково) |
| **Tavily API Key** | API ключ для пошуку в інтернеті Tavily (якщо обрано Tavily) |

## Покращення у v1.16.0

- **Кеш клієнта AI** — тепер підтримується за допомогою transients для збереження даних між запитами, запобігаючи втраті даних під час тривалих завдань агента.
- **Інструмент збору даних з сайтів рівня 1** — sd-ai-agent/site-scrape тепер доступний за замовчуванням у Theme Builder для аналізу сайтів конкурентів та джерел натхнення.
- **Посилання дій у рядку плагінів** — виправлено та перейменовано для більшої ясності.

## Покращення у v1.10.0

- **Рефакторинг інтеграції з WooCommerce** — тепер використовує нативні API WooCommerce, що забезпечує кращу надійність та сумісність.
- **Автоматичне оновлення списку провайдерів** — автоматично виявляє, коли плагіни активуються чи деактивуються, та оновлює доступний список провайдерів.
- **Покращений конструктор сайтів** — вдосконалені багатоетапні плани створення сайтів із кращим відновленням після помилок.
- **Функціонал, адаптований до теми** — чотири нові вбудовані функції, які автоматично адаптуються до вашої активної теми.

## Виправлення помилок у v1.10.0

- Виправлено функціонал `navigate-to`, який спричиняв нескінченні цикли перезавантаження на деяких сторінках адмін-панелі.
- Виправлено функціонал `list-posts`, який некоректно вирішував імена категорій та тегів до слагів.
- Виправлено команди WP-CLI, яким бракувало псевдонімів просторів імен після попереднього рефакторингу.
- Виправлено автоматизацію подій, яка коректно обробляє сайти, де таблиці автоматизації ще не створені.
- Виправлено функціонал `memory-save`, який використовує правильний префікс простору імен у генераторі системних інструкцій.
- Виправлено, що результати скалярних інструментів коректно обгортаються перед поверненням до AI.
- Виправлено, що статистика використання коректно обробляє старий формат ключа функціоналу при оновленні зі старих версій.
