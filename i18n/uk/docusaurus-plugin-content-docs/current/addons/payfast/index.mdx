# PayFast {#payfast}

Ultimate Multisite: Шлюз Payfast

Додає Payfast — швидкий та безпечний онлайн-платіжний шлюз у Південній Африці — як платіжну опцію для Ultimate Multisite.

## Installation {#installation}

1. Завантажте 'ultimate-multisite-payfast' до каталогу '/wp-content/plugins/'
2. Активуйте плагін через меню 'Plugins' у WordPress
3. Дотримуйтесь покрокового майстра (Wizard) для налаштування плагіна

## Usage Guide {#usage-guide}

PayFast — провідний онлайн-платіжний процесор для бізнесу в Південній Африці. Він присвячений допомозі онлайн-торговцям у зростанні, забезпечуючи безперебійні та безпечні транзакції. За запитом користувачів Ultimate Multisite ми вирішили створити аддон, який уможливить інтеграцію з PayFast. У цій статті показано, як налаштувати PayFast як ваш платіжний шлюз для отримання платежів у вашій мережі Ultimate Multisite.

Перший крок — встановити аддон.

# Як встановити аддон Payfast: {#how-to-install-the-payfast-add-on}

Перейдіть на сторінку аддонів та виберіть аддон Ultimate Multisite: Payfast Gateway.

![Add-ons page](/img/admin/addons-page.png)

З'явиться вікно аддону. Натисніть на Install Now.

![Add-ons page with install option](/img/admin/addons-page.png)

Після завершення встановлення ви будете перенаправлені на сторінку плагінів. Тут просто натисніть Network Activate, і аддон _Payfast_ буде активовано у вашій мережі.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# Налаштування PayFast {#setting-up-payfast}

Після встановлення аддону ви можете перейти до налаштувань Ultimate Multisite та розпочати конфігурацію PayFast.

На сторінці **Ultimate Multisite settings** ви знайдете вкладку "Payments". На цій вкладці ви маєте побачити "**Payfast** " разом із перемикачем (toggle button) для активації або деактивації цього платіжного шлюзу.

![Payment gateway settings](/img/config/settings-payment-gateways.png)Просто натисніть на перемикач, щоб активувати його. Тоді він розгорне поле, щоб показати різні поля, які вам потрібно заповнити, такі як **Mechant ID, Merchant Key та PassPhrase.** Цю інформацію ви можете отримати зі свого облікового запису **Payfast**

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Наступний крок — увійти до свого облікового запису PayFast, щоб отримати необхідну інформацію.

Після входу ви одразу побачите облікові дані, які вам потрібні, на панелі керування (dashboard) вашого облікового запису **Payfast**.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->Ви можете просто скопіювати та вставити облікові дані та заповнити деталі у налаштуваннях Ultimate Multisite. Потім натисніть кнопку **"Save settings** ", щоб зберегти зміни.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Ось і все, ваша інтеграція платіжів PayFast завершена!
