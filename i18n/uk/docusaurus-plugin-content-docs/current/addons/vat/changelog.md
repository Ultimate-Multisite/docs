# Журнал змін (Changelog) щодо ПДВ

Version 1.0.7 - Випущено 2026-02-03

* Виправлення: Номери ПДВ Німеччини (DE) не проходили валідацію через помилки VIES SOAP service MS_MAX_CONCURRENT_REQ. Ми перейшли від застарілого SOAP API до офіційного EU VIES REST API для більш надійного валідаційного процесу.
* Виправлення: Тепер номери ПДВ приймаються, навіть якщо сервіс VIES тимчасово недоступний, а не відхиляються помилково. Для подальшого огляду створюється запис у журналі.
* Виправлення: Поле ПДВ/Ідентифікатор податків тепер видно для всіх країн, а не лише для членів ЄС. Неєвропейські податкові ідентифікатори (наприклад, швейцарські номери CHE) зберігаються для відображення у рахунках-фактурах без валідації VIES.
* Виправлення: Перевірка префікса ПДВ для Греції (GR), Маноккі (MC) та острова Мэн (IM) була порушена через неправильну логіку пошуку масивів.
* Покращення: Видалено непотрібну залежність розробника wp-cli-bundle, яка могла спричинити фатальні помилки при використанні разом із аддоном Site Exporter.
* Покращення: Додано комплексний набір юніт-тестів (53 тести).

Version 1.0.6 - Випущено 2026-01-25

* Виправлення: Внутрішньодержавні B2B-транзакції тепер коректно стягують ПДВ. Згідно з правилами ПДВ ЄС, зворотне оподаткування (reverse charge) застосовується лише до трансграничних B2B-транзакцій, а не тоді, коли країна клієнта збігається з країною компанії.

Version 1.0.5 - Випущено 2026-01-22

* Виправлення: Аддон не завантажувався або працював некоректно.
* Зміна: Джерело даних ставок ПДВ змінено з euvatrates.com на репозиторій ibericode/vat-rates для отримання більш надійних та активно підтримуваних даних.
* Виправлення: Виправлено друкарську помилку в значенні опції super_reduced_rates.
* Покращення: Додано перевірки на нульові значення для обробки кодів країн, щоб запобігти помилкам.
* Зміна: Видалено вбудовані файли перекладу на користь автоматичних оновлень через Traduttore.
