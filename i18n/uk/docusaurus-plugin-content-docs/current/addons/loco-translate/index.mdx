# Loco Translate

Додаток Loco Translate Integration для Ultimate Multisite надає можливість перекладу, специфічного для сайту, у вашу мережу WordPress Multisite. Якщо вам коли-небудь потрібно було, щоб різні сайти у вашій мережі мали різні переклади одного й того ж плагіна чи теми, цей додаток — ідеальне рішення.

**Що робить цей додаток**

Цей додаток плавно інтегрує Ultimate Multisite з популярним плагіном Loco Translate, дозволяючи кожному сабсайту у вашій мережі зберігати власні користувацькі переклади. Незалежно від того, чи керуєте ви багаторегіональною платформою, обслуговуєте різні мовні спільноти, чи просто потребуєте гнучкості перекладу по всій мережі, цей додаток робить це без зусиль.

**Ідеально підходить для:**

* Бізнесу, що працює в кількох регіонах і має різні потреби у локалізації
* Освітніх закладів, які обслуговують різноманітні спільноти
* SaaS-платформ, що пропонують багатомовну підтримку
* Веб-агенцій, які керують кількома сайтами клієнтів
* Будь-якої мережі Multisite, яка вимагає гнучкості перекладу

## Встановлення {#installation}

1. Переконайтеся, що Ultimate Multisite встановлено та активовано на рівні мережі
2. Встановіть та активуйте плагін Loco Translate
3. Завантажте цей додаток у каталог плагінів
4. Активуйте Loco Translate Integration addon на рівні мережі
5. Почніть створювати переклади, специфічні для сайту, через Loco Translate на будь-якому сабсайті

Ось і все! Інтеграція працює автоматично без будь-якої додаткової конфігурації.

## Часто задавані питання {#frequently-asked-questions}

## Чи потрібно щось налаштовувати після встановлення? {#do-i-need-to-configure-anything-after-installation}

Ні! Інтеграція відбувається автоматично. Як тільки додаток та Loco Translate активні, сабсайти можуть негайно почати створювати власні користувацькі переклади.

## Що станеться, якщо сабсайт не створить користувацькі переклади? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

Сайт автоматично використає мережеві переклади як запасний варіант. Нічого не зламається, і все працюватиме бездоганно.

## Чи можуть сабсайти отримувати доступ до мережевих перекладів? {#can-subsites-access-network-wide-translations}

Звичайні адміністратори сайтів бачать і редагують лише переклади свого сайту. Super admins можуть отримувати доступ як до мережевих, так і до перекладів, специфічних для сайту, залежно від того, який сайт вони переглядають.

## Чи працює це з плагінами та темами? {#does-this-work-with-both-plugins-and-themes}

Так! Переклади, специфічні для сайту, працюють як для плагінів WordPress, так і для тем.

## Чи сповільнить це мою мережу Multisite? {#will-this-slow-down-my-multisite-network}

Ні. Додаток використовує ефективне завантаження файлів, яке перевіряє наявність перекладів, специфічних для сайту, лише за потреби. Помітного впливу на продуктивність не буде.

## Чи можу я скопіювати мережеві переклади на конкретний сайт як відправну точку? {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

Так! Під час редагування перекладів у Loco Translate ви можете легко почати з мережевих перекладів та адаптувати їх для кожного сайту.

## Чи працює це з WooCommerce та іншими плагінами? {#does-this-work-with-woocommerce-and-other-plugins}

Абсолютно! Будь-який плагін чи тема, який використовує стандартні функції перекладу WordPress, працюватиме з перекладами, специфічними для сайту.
