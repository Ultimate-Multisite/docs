# Шаблон аддону

{{ADDON_DESCRIPTION}}

## Вимоги

- WordPress 5.3 або новіше
- PHP 7.4 або новіше
- Плагін Ultimate Multisite (активний)

## Встановлення

1. Завантажте файли аддону до каталогу `/wp-content/plugins/`
2. Активуйте плагін через меню «Плагіни» у WordPress
3. Налаштуйте аддон у мережевому адмініструванні

## Розробка

### Налаштування

```bash
# Встановлення залежностей
composer install
npm install

# Запуск тестів
npm run test

# Перевірка відповідності стандартам коду
vendor/bin/phpcs
vendor/bin/phpstan

# Виправлення проблем зі стилем коду
vendor/bin/phpcbf

# Збірка для продакшену
npm run build
```

### Тестування

Аддон включає комплексний фреймворк тестування:

- **Модульні тести (Unit Tests)**: Тести на основі PHPUnit із набором тестів WordPress
- **Базовий клас для тестів (Base Test Class)**: Надає допоміжні методи для типових сценаріїв тестування
- **Покриття кодом (Test Coverage)**: Налаштовано для відстеження покриття кодом
- **Тестування Multisite**: Тести виконуються в середовищі Multisite

### Стандарти коду

- **Стандарти кодування WordPress**: Забезпечуються за допомогою PHPCS
- **Сумісність з PHP 7.4+**: Підтримуються сучасні функції PHP
- **Статичний аналіз**: Інтеграція PHPStan для перевірки типів
- **Модернізація коду**: Правила Rector для оновлення PHP

### Система збірки

- **Обробка ресурсів (Asset Processing)**: Мініфікація файлів CSS/JS
- **Переклад (Translation)**: Генерація файлів POT
- **Пакування (Packaging)**: Створення архіву для розповсюдження
- **Режим розробки (Development Mode)**: Немініфіковані ресурси для налагодження

## Конфігурація

Аддон підтримує різні параметри конфігурації, доступні через інтерфейс мережевого адміністрування.

## Хуки та фільтри

### Дії (Actions)

- `ultimate-multisite-addon-template_init` - Виконується після ініціалізації аддону
- `ultimate-multisite-addon-template_loaded` - Виконується після завантаження всіх файлів аддону

### Фільтри (Filters)

- `ultimate-multisite-addon-template_settings` - Зміна налаштувань аддону
- `ultimate-multisite-addon-template_enabled` - Перевизначення статусу увімкнення/вимкнення аддону
