# Fluent Forms {#fluent-forms}

Цей додаток дозволяє встановлювати ліміти для Fluent Forms на Multisite Ultimate у продуктах, членствах чи сайти.
Fluent Forms використовує власні таблиці замість кастомних типів записів, тому звичайні опції обмеження не працюють із Fluent Forms.

## Функції {#features}

- Додає ліміти для Fluent Forms або Conversational Fluent Forms
- Користувачі отримують дружнє повідомлення, яке спонукає їх оновити план, коли вони досягають ліміту
