# Додатки для Ultimate Multisite

Додатки розширюють функціонал Ultimate Multisite, додаючи нові можливості та інтеграції. Більшість додатків є платними та їх можна придбати на [ultimatemultisite.com](https://ultimatemultisite.com).

## Встановлення

Додатки можна встановити двома способами:

1. **Через сторінку додатків** — Перейдіть на сторінку керування додатками вашого сайту в адмін-панелі WordPress.
2. **Вручну (завантаження)** — Завантажте їх зі [свого облікового запису](https://ultimatemultisite.com/my-account/downloads/) та завантажте через Плагіни > Додати новий.
