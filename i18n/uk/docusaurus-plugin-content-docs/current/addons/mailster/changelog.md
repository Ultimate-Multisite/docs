# Журнал змін Mailster {#mailster-changelog}

* Початкове випущення
* Автоматична синхронізація підписників при реєстрації або оплаті
* Призначення списків, специфічних для продукту
* Підписка за допомогою чекбоксів для відповідності GDPR
* Підтримка подвійного підтвердження підписки (Double opt-in)
* Комплексне зіставлення полів
* Детальне ведення журналів (логінг)
