# Mailster

Перетворіть вашу мережу Ultimate Multisite на потужну машину для збору email-адрес за допомогою аддону Ultimate Multisite Mailster Integration. Автоматично додавайте нових клієнтів до списків Mailster, коли вони реєструються або здійснюють покупки, створюючи безшовний зв'язок між вашою платформою членства та стратегією email-маркетингу.

## Основні функції {#key-features}

**Автоматична синхронізація підписників**
* Автоматично додає клієнтів до списків Mailster при реєстрації
* Можливість синхронізувати негайно при реєстрації або чекати підтвердження оплати
* Підтримуйте ваші списки email свіжими та актуальними без ручного втручання

**Гнучке керування списками**
* Встановіть глобальні список за замовчуванням для всіх нових клієнтів
* Перевизначте значення за замовчуванням, використовуючи списки, специфічні для продукту, для цільових кампаній
* Легко керуйте призначенням списків за допомогою знайомого інтерфейсу Mailster

**Варіанти збору згоди, відповідні GDPR**
* Виберіть між автоматичною підпискою та згодою через чекбокс
* Увімкніть подвійну підписку для підтверджених підписок
* Показуйте/приховуйте чекбокс збору згоди залежно від ваших вимог до відповідності нормам

**Розумне відображення даних (Data Mapping)**
* Автоматично синхронізує імена клієнтів та контактну інформацію
* Призначає адреси виставлення рахунків до полів підписників
* Відстежуйте дати реєстрації для цільових кампаній

**Керування на рівні продукту**
* Призначайте різні списки для різних продуктів
* Створюйте цільові кампанії на основі того, що купують клієнти
* Можливість вимкнути Mailster для певних продуктів за потреби

**Професійний досвід**
* Вбудована інтеграція UI Mailster для вибору списків
* Детальний журнал для усунення несправностей та верифікації
* Безшовне оновлення існуючих підписників

## Ідеально підходить для {#perfect-for}

* Власників сайтів-членств, які створюють email-листи
* SaaS-платформ, які прагнуть розвивати нових клієнтів
* Творців курсів, які сегментують студентів за реєстрацією
* Кожен, хто використовує Ultimate Multisite з Mailster

## Як це працює {#how-it-works}

1. Встановіть та активуйте аддон у вашій мережі Ultimate Multisite
2. Налаштуйте глобальні списки за замовчуванням у налаштуваннях Ultimate Multisite
3. За бажанням перевизначте їх, використовуючи списки, специфічні для продукту
4. Виберіть режим збору згоди та таймінг підписки
5. Спостерігайте, як ваші списки Mailster зростають автоматично!

## Встановлення {#installation}

1. Завантажте файли плагіна до каталогу аддонів Ultimate Multisite
2. Переконайтеся, що Mailster встановлено та активуровано на вашому основному сайті
3. Активуйте аддон на сторінці аддонів Ultimate Multisite
4. Перейдіть до Налаштування > Mailster Integration для конфігурації
5. Виберіть ваші списки за замовчуванням та налаштуйте переваги збору згоди

## Часто задавані питання {#frequently-asked-questions}

## Чи працює цей аддон без Ultimate Multisite? {#does-this-addon-work-without-ultimate-multisite}

Ні, цей аддон вимагає Ultimate Multisite для функціонування. Він спеціально розроблений для інтеграції Mailster з системою клієнтів та продуктів Ultimate Multisite.

## Чи потрібно мені встановлювати Mailster на всіх сайтах? {#do-i-need-mailster-installed-on-all-sites}

Ні, Mailster потрібно лише встановити та активувати на вашому основному сайті мережі. Аддон автоматично підключиться до нього з усіх дочірніх сайтів.

## Чи можу я призначати різні списки для різних продуктів? {#can-i-assign-different-lists-to-different-products}

Так! Ви можете встановити глобальні списки за замовчуванням, а потім перевизначити їх для кожного продукту. Це дозволяє створювати високоцільові email-кампанії на основі того, що купують клієнти.

## Що станеться, якщо клієнт вже існує в Mailster? {#what-happens-if-a-customer-already-exists-in-mailster}

За замовчуванням аддон оновить існуючих підписників та додасть їх до нових списків. Ви можете вимкнути цю функцію в налаштуваннях, якщо бажаєте пропустити існуючих підписників.

## Чи відповідає це GDPR? {#is-this-gdpr-compliant}

Так! Аддон включає режим збору згоди через чекбокс, де клієнти повинні явно погодитися на отримання email. Це відповідає вимогам GDPR у поєднанні з належною політикою конфіденційності.

## Чи можу я використовувати подвійну підписку? {#can-i-use-double-opt-in}

Абсолютно! Увімкніть налаштування подвійної підписки, і Mailster надішле підтверджувальні email, перш ніж активувати підписки.
