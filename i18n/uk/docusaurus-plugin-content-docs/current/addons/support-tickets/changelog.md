# Журнал змін плагіна Support Tickets

### 1.0.4 - 2026-05-05
* Покращено: Видалено каталог `vendor/` з відстеження Git (це вже покривається `.gitignore`), що зменшує розмір репозиторію.
* Покращено: Тестування до WordPress 7.0.

### 1.0.3 - 2026-05-01
* Виправлено: Дозволено значення null у сеттерах моделі тікетів, які можуть бути null.
* Виправлено: Усі тікети тепер зберігаються у глобальних таблицях для коректної підтримки мультисайту.
* Виправлено: Приховано поля персоналу та виправлено перенаправлення на порожню сторінку у формі створення нового тікета клієнтом.
* Виправлено: Замінено виклики `add_meta()` з undefined на правильні `update_meta()` для зберігання метаданих.
* Виправлено: Замінено перевірку не зареєстрованої можливості (capability) на правильний `wu_view_all_support_tickets`.
* Виправлено: Додано відсутні обробники AJAX для дій зі статусом тікета, призначенням та швидким редагуванням.
* Виправлено: Виправлено назву методу для виявлення відповіді персоналу у перегляді тікетів.
* Виправлено: Об'єднано дубльовані обробники відповідей та вирівняно назви дій nonce.
* Виправлено: Додано відсутній фронтенд-перегляд для шорткоду `[wu_submit_ticket]`.
* Виправлено: Додано відсутній стовпець `user_id` та виправлено метод `Support_Ticket::get_user_id()`.
* Виправлено: Видалено зайві подвійні лапки в тегу опції фільтрації пріоритету.
* Виправлено: Додано панель керування тікетами для адміністратора мережі для супер-адміністраторів.
* Покращено: Об'єднано CSS адміністрування в єдиний зовнішній файл стилів.
* Покращено: Видалено підменю Налаштування з адмін-меню підсайту.
* Покращено: Умовне завантаження фронтенд-ресурсів лише на сторінках підтримки.
* Покращено: Пропуск автозавантажувача плагіна, якщо кореневий автозавантажувач Bedrock вже завантажив залежності.

### 1.0.2 - 2025-12-11
* Додано: Повна функціональність перегляду тікетів для адмін-панелі та фронтенду.
* Додано: Обробник AJAX для надсилання відповідей на тікети.
* Додано: Підтримка функціональності відповіді на тікети з коректною обробкою форм.
* Додано: Правильне відображення повідомлень про подання тікетів та відповідей в адмін-панелі.
* Додано: Автоматичне прив'язування тікета до поточного користувача при самостійній подачі заяв клієнтом.
* Додано: Покращення безпеки для запобігання можливість клієнтам перевизначати власника тікета.
* Додано: Відсутні допоміжні функції (`wu_format_date`, `wu_user_can_view_ticket` тощо).
* Додано: Правильне завантаження та обробка файлів-вкладень.
* Додано: Систему сповіщень електронною поштою про відповіді на тікети та зміну статусу.
* Виправлено: Форма відповіді на тікет тепер включає необхідний ID тікета.
* Виправлено: Проблеми з видимістю повідомлень у перегляді тікетів в адмін-панелі.
* Виправлено: Синтаксичні помилки у функціях підтримки тікетів.
* Покращено: Структура бази даних із відповідними класами запитів та схем для відповідей та вкладень.
* Покращено: Міграція на реальні властивості замість атрибутів для моделей тікетів.

### 1.0.1 - 2025-09-28
* Оновлено: Змінено префікс на `ultimate-multisite` для узгодженості.
* Оновлено: Стандартизація текстової області.
* Виправлено: Незначні виправлення помилок та покращення.

### 1.0.0 - 2025-09-01
* Початкове випущення
* Повна система керування тікетами
* Багарівне керування доступом
* Система обговорення в гілках
* Підтримка файлів-вкладень
* Система сповіщень електронною поштою
* Інтерфейси для адміністраторів та клієнтів
* Статистика та звітність
