# Багатоореадність (Multi-Tenancy)

Розподіліть сабсайти між серверами та базами даних для підвищення продуктивності та безпеки

## Вимоги

- WordPress 5.3 або новіше
- PHP 7.4 або новіше
- Плагін Ultimate Multisite (активний)

## Встановлення

1. Завантажте файли аддону до каталогу `/wp-content/plugins/`
2. Активуйте плагін через меню «Плагіни» у WordPress
3. Налаштуйте параметри аддону в адмін-панелі мережі

## Розробка

### Налаштування

```bash
# Встановлення залежностей
composer install
npm install

# Запуск тестів
npm run test

# Перевірка стандартів кодування
vendor/bin/phpcs
vendor/bin/phpstan

# Виправлення проблем зі стилем коду
vendor/bin/phpcbf

# Збірка для продакшену
npm run build
```

### Тестування

Аддон включає комплексний фреймворк тестування:

- **Unit Tests**: Тести на основі PHPUnit із набором тестів WordPress
- **Base Test Class**: Надає допоміжні методи для типових сценаріїв тестування
- **Test Coverage**: Налаштовано для відстеження покриття кодом
- **Multisite Testing**: Тести виконуються в середовищі multisite

### Стандарти кодування

- **WordPress Coding Standards**: Забезпечується за допомогою PHPCS
- **Сумісність з PHP 7.4+**: Підтримуються сучасні функції PHP
- **Static Analysis**: Інтеграція PHPStan для перевірки типів
- **Code Modernization**: Правила Rector для оновлення PHP

### Система збірки

- **Asset Processing**: Мініфікація файлів CSS/JS
- **Translation**: Генерація файлів POT
- **Packaging**: Створення архіву для розповсюдження
- **Development Mode**: Немініфіковані ресурси для налагодження

## Конфігурація

Аддон підтримує різні параметри конфігурації, доступні через інтерфейс адмін-панелі мережі.

## Хуки та фільтри (Hooks and Filters)

### Actions (Дії)

- `ultimate-multisite-addon-template_init` - Викликається після ініціалізації аддону
- `ultimate-multisite-addon-template_loaded` - Викликається після завантаження всіх файлів аддону

### Filters (Фільтри)

- `ultimate-multisite-addon-template_settings` - Зміна налаштувань аддону
- `ultimate-multisite-addon-template_enabled` - Перевизначення статусу увімкнення/вимкнення аддону
