# Журнал змін Ultimate AI Connector для сумісних кінцевих точок {#ultimate-ai-connector-for-compatible-endpoints-changelog}

## Версія 2.0.0 — Випущено 2026-04-24 {#version-200--released-on-2026-04-24}

* Нове: Підтримка кількох провайдерів — налаштуйте кілька кінцевих точок ШІ та маршрутизуйте запити з автоматичним резервуванням між провайдерами.
* Виправлено: Інтеграція SDK для кількох провайдерів із коректними ID провайдерів, URL реєстрації та обмежуванням HTTP-фільтра для кожного провайдера.
* Виправлено: Нова картка провайдера тепер автоматично розкривається при додаванні; механізм усунення кешу скриптів при оновленні плагіна.
* Виправлено: Динамічний простір імен класу провайдера для eval() та автоматичне розкриття для нещодавно доданих провайдерів.
* Виправлено: Використовує стабільні компоненти Card/CardBody/CardHeader/CardDivider (більше не експериментальні) для сумісності з WordPress 6.9+.
* Виправлено: Замінено не визначений DragHandle на унікодну іконку захоплення для перетягування провайдерів та їх переупорядкування.
* Виправлено: Усунено блокуючий HTTP-запит, який спрацьовував при кожному завантаженні сторінки.
* Покращено: Робочі процеси GitHub Actions оновлено до Node.js 24.
