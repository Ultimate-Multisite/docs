# Журнал змін планів з оплатою за використання {#metered-plans-changelog}

Version 1.1.0 - Випущено 2026-05-05
- Нове: Облік токенів AI для підсайтів Multisite — відстежуйте та виставляйте рахунки за використання токенів AI на різних сайтах клієнтів із можливістю налаштування тарифів за кожен токен.
- Нове: Механізм застосування коннектора перероблено з динамічним виявленням лімітів та записом через (write-through), що забезпечує точність у реальному часі для всіх коннекторів.
- Виправлено: Оновлення таблиць бази даних тепер коректно узгоджені з визначеннями схеми BerlinDB, що запобігає збоям оновлення при новій установці.
- Виправлено: Зворотні виклики (callbacks) оновлення бази даних перетворено на правильний формат, що вирішує проблему тихих збоїв оновлення.
- Виправлено: Тепер приймаються дробові значення у полі введення надлишкового маркування використання AI.
- Виправлено: Вирішено фатальні помилки та проблеми подвійного ініціалізування під час завантаження плагіна.
- Покращено: Додано npm скрипт check-env, завдяки якому середовища розробників самоконфігуруються при першому запуску.
