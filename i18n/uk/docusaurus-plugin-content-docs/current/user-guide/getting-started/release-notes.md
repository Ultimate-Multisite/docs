# Примітки до випуску

## Версія 2.13.0 — випущено 2026-06-05 {#version-2130--released-on-2026-06-05}

- Нове: Додано підтримку sovereign-tenant для потоків облікового запису клієнта, оформлення замовлення, білінгу, сайту, рахунка-фактури, перемикання шаблонів і прив’язки доменів, щоб мережі орендарів могли спрямовувати клієнтів назад на головний сайт для керованих дій.
- Нове: Додано перевірки облікових даних поновлення для повторюваних членств, щоб шлюзи могли вимикати автоматичне поновлення, коли відсутня збережена білінгова угода, підписка або vault token.
- Нове: Додано HMAC-перевірену loopback-публікацію для очікуваного створення сайту, щоб зробити надання сайту після оформлення замовлення надійнішим на хостах, де фонові завдання затримуються.
- Нове: Додано точки розширення для розробників для SSO URL-адрес, базових доменів форми оформлення замовлення та автоматичного створення записів домену.
- Виправлено: SSO працює надійніше на прив’язаних доменах, під час анонімних відвідувань брокера, виходу з облікового запису та конфліктів залежностей між плагінами.
- Виправлено: Очікуване створення сайту тепер відновлюється після застарілих прапорців публікації та не залишає клієнтів застряглими на екрані створення сайту.
- Виправлено: Записи доменів більше не створюються для спільних базових доменів форми оформлення замовлення, а невикористані фонові завдання провайдера хостингу пропускаються, коли жодна інтеграція не активна.
- Виправлено: Граничні випадки оформлення замовлення, білінгової адреси, скидання пароля, підтвердження електронної пошти, перемикання шаблонів, турів і клієнтської панелі більше не блокують звичайні потоки клієнтів.
- Виправлено: Розсилки електронною поштою тепер зберігають приватність одержувачів, водночас уникаючи фатальних помилок SMTP/плагіна, коли списки одержувачів або поштові транспорти дають збій.
- Виправлено: Граничні випадки поновлення членства, відображення завершення терміну дії та збору платежів тепер уникають негайного завершення терміну дії, збоїв або пропущених обов’язкових платежів.
- Покращено: Сумісність із WordPress протестовано до 7.0, виробничі Vue-ресурси перебудовано з npm-джерел, а наскрізне покриття Cypress тепер перевіряє більше потоків оформлення замовлення, налаштування, SSO та шлюзів.

## Версія 2.12.0 — випущено 2026-05-15 {#version-2120--released-on-2026-05-15}

- Нове: Додано Hostinger (hPanel) як підтримуваного провайдера хостингу з інтеграцією прив’язки доменів
- Нове: Site Exporter тепер обробляє пакети імпорту мережі для спрощеного відновлення сайтів у межах усієї мережі
- Виправлено: BCC-розсилки електронною поштою тепер використовують заголовок undisclosed-recipients, щоб запобігти розкриттю адрес одержувачів
- Виправлено: Дата завершення терміну членства більше не пошкоджується під час збереження зі значенням, що не є датою
- Виправлено: Оновлення членства Stripe тепер коректно очищають знижки без виклику застарілого deleteDiscount API
- Виправлено: SSO-перенаправлення на сайтах із прив’язаними доменами тепер обмежені, щоб запобігти нескінченним циклам перенаправлення
- Виправлено: Вибір зображення в майстрі налаштування тепер коректно оновлює базову модель даних
- Виправлено: Site Exporter CLI тепер зберігає правильний вибір сайту мережі за замовчуванням
- Покращено: Видалено вбудований wp-cli з пакета плагіна, що зменшило розмір плагіна

## Версія 2.11.0 — випущено 2026-05-11 {#version-2110--released-on-2026-05-11}

- Нове: Експорти сайтів тепер містять самозавантажувальний `index.php`, щоб ZIP можна було встановити на новий хост без окремого встановлення плагіна.
- Нове: Експорт мережі дає адміністраторам змогу експортувати всі підсайти в один архів зі сторінки адміністрування Site Export.
- Нове: Перемикач плану Allow Site Templates тепер застосовується через fallback-ланцюг, коректно обмежуючи доступність шаблонів відповідно до лімітів плану.
- Нове: Редактор форми оформлення замовлення попереджає, коли продукт додається без налаштованого обов’язкового поля.
- Нове: Вкладка налаштувань Import/Export тепер чітко описує свою область дії та напряму посилається на інструмент Site Export.

## Версія 2.10.0 — випущено 2026-05-05 {#version-2100--released-on-2026-05-05}

- Нове: Покроковий майстер налаштування PayPal для ручного введення облікових даних із прапорцем OAuth для безшовного налаштування шлюзу.
- Нове: Панель клієнта для перемикання шаблонів перероблено з карткою поточного шаблону, сталою сіткою та кнопкою **Скинути поточний шаблон**.
- Виправлено: Перемикання шаблонів більше не зависає в інтерфейсі користувача в разі збою AJAX.
- Виправлено: Стани дозволів для перемикання шаблонів захищено від несанкціонованого доступу.
- Виправлено: Поля перевизначення сайту перевіряються перед збереженням.
- Виправлено: Запит білінгової адреси тепер показується, коли адреса порожня.
- Виправлено: Усунено повідомлення про застарілість перетворення null-to-string у PHP 8.1.
- Виправлено: Currents ліниво завантажуються перед init hook, щоб запобігти проблемам із таймінгом.
- Виправлено: Відфільтрований шлях SSO враховується в усіх потоках входу.
- Виправлено: Порожні параметри ідентичності сайту зберігаються під час збереження.

## Версія 2.9.0 — випущено 2026-04-30 {#version-290--released-on-2026-04-30}

- Нове: Експорт і імпорт одного сайту додано в **Інструменти > Експорт та імпорт**.
- Виправлено: ZIP-файли експорту тепер віддаються через автентифікований endpoint завантаження.
- Виправлено: Ризик SQL-ін’єкції та проблеми із запитами в запитах очікуваного експорту/імпорту усунено.
- Виправлено: Очікуваний сайт не публікувався, коли адміністратор вручну підтверджував електронну пошту клієнта.
- Виправлено: Осиротілі записи pending_site очищаються, коли членство відсутнє.
- Виправлено: Відступи навігації налаштувань і навігацію за пошуковими якорями виправлено.
- Виправлено: Очікувані сайти тепер показуються першими у вигляді All Sites.
- Виправлено: Додано заголовок User-Agent для провайдера знімків екрана (mShots), щоб запобігти помилкам 403.
- Виправлено: Усунено циклічну залежність розкладу cron для імпорту.
- Виправлено: ID турів нормалізовано до підкреслень у ключах налаштувань користувача.
