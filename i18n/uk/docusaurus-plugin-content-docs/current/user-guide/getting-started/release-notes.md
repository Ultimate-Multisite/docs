# Примітки до випуску

## Версія 2.12.0 — Випущено 2026-05-15

- Нове: Додано Hostinger (hPanel) як підтримуваного провайдера хостингу з інтеграцією мапування доменів
- Нове: Site Exporter тепер обробляє пакети імпорту мережі для оптимізованого відновлення сайтів у масштабі всієї мережі
- Виправлено: Броадкасти електронних листів BCC тепер використовують заголовок undisclosed-recipients, щоб запобігти розкриттю адрес отримувачів
- Виправлено: Дата закінчення членства більше не пошкоджується при збереженні з недіатовим значенням
- Виправлено: Оновлення членства Stripe тепер коректно знімають знижки без виклику застарілого API deleteDiscount
- Виправлено: Перенаправлення SSO на сайтах з мапуванням доменів тепер обмежено, щоб запобігти нескінченним циклам перенаправлення
- Виправлено: Вибір зображень у майстрі налаштування тепер коректно оновлює базову модель даних
- Виправлено: CLI Site Exporter тепер зберігає правильний вибір сайту за замовчуванням для мережі
- Покращено: Видалено вбудований wp-cli з пакета плагіна, зменшуючи розмір плагіна

## Версія 2.11.0 — Випущено 2026-05-11

- Нове: Експорт сайтів тепер пакує самозавантажуваний `index.php`, щоб ZIP можна було встановити на свіжому хості без окремої установки плагіна.
- Нове: Експорт мережі дозволяє адміністраторам експортувати всі сабсайти в одному архіві зі сторінки адміністратора Site Export.
- Нове: Перемикач плану Site Templates тепер примусово реалізується через ланцюжок резервних варіантів, коректно обмежуючи доступність шаблонів відповідно до лімітів плану.
- Нове: Редактор форми оформлення замовлення попереджає, коли продукт додано без налаштованого обов'язкового поля.
- Нове: Вкладка налаштувань імпорту/експорту тепер чітко описує свою сферу дії та безпосередньо посилається на інструмент Site Export.

## Версія 2.10.0 — Випущено 2026-05-05

- Нове: Покроковий майстер налаштування PayPal для ручного введення облікових даних з гейтом OAuth для безперебійного налаштування шлюзу.
- Нове: Панель клієнта для перемикання шаблонів перероблена з карткою поточного шаблону, постійною сіткою та кнопкою **Скинути поточний шаблон**.
- Виправлено: Перемикання шаблонів більше не "зависає" інтерфейс при збої AJAX.
- Виправлено: Стани дозволів перемикання шаблонів захищено від несанкціонованого доступу.
- Виправлено: Введення для перевизначення сайту валідується перед збереженням.
- Виправлено: Підказка про білінгову адресу тепер відображається, коли адреса порожня.
- Виправлено: Вирішено попередження про застарівання PHP 8.1 null-to-string.
- Виправлено: Currents завантажується через хук before init, щоб запобігти проблемам із часом.
- Виправлено: Фільтрований шлях SSO поважається у всіх потоках входу.
- Виправлено: Опції ідентифікації для порожнього сайту зберігаються при збереженні.

## Версія 2.9.0 — Випущено 2026-04-30

- Нове: Додано експорт та імпорт одного сайту у розділі **Інструменти > Експорт та імпорт**.
- Виправлено: ZIP-файли експорту тепер надаються через аутентизовану кінцеву точку завантаження.
- Виправлено: Виправлено ризик SQL-ін'єкції та проблеми з запитами в запитах очікуваного експорту/імпорту.
- Виправлено: Очікуваний сайт не публікується, коли адміністратор вручну підтверджує електронну пошту клієнта.
- Виправлено: Сиротині записи pending_site очищаються, коли відсутнє членство.
- Виправлено: Виправлено відступи навігації налаштувань та навігацію за якорями пошуку.
- Виправлено: Очікувані сайти тепер відображаються першими у перегляді Всі сайти.
- Виправлено: Додано заголовок User-Agent для провайдера скріншотів (mShots), щоб запобігти помилкам 403.
- Виправлено: Вирішено циклічну залежність планування cron імпорту.
- Виправлено: Ідентифікатори турів нормалізовано до підкреслень у ключах налаштувань користувача.
- Покращено: Замість Alchemy/Zippy тепер використовується ZipArchive для кращої сумісності.
