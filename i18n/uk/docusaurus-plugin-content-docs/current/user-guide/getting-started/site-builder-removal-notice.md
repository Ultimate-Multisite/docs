# Видалення режиму конструктора сайтів

**Режим конструктора сайтів (Site Builder mode) було вилучено у Superdav AI Agent v1.12.0.** Якщо ви користувалися режимом конструктора сайтів, вам слід перейти до використання **агента Setup Assistant** для створення тем та налаштування сайтів.

## Що сталося?

### Режим конструктора сайтів (Site Builder Mode) (Застарілий)

Режим конструктора сайтів був покроковим інтерфейсом для:

- Створення сайтів за шаблонами
- Налаштування базових параметрів
- Вибору теми
- Встановлення початкового контенту

### Що його замінило?

Тепер **агент Setup Assistant** обробляє всю функціональність Site Builder, пропонуючи:

- Більш гнучке налаштування з допомогою агента
- Кращі опції кастомізації теми
- Інтеграцію з онбордингом Theme Builder
- Зберігання пам'яті site_brief для майбутніх сесій

## Якщо ви використовували режим конструктора сайтів

### Ваші сайти в безпеці

- Існуючі сайти, створені в режимі конструктора сайтів, продовжують працювати
- Немає втрати даних чи збою роботи сайту
- Ви можете продовжувати керувати своїми сайтами у звичайному режимі

### Перехід на агент Setup Assistant

Для налаштування нового сайту або зміни теми використовуйте агент Setup Assistant:

```
"Help me set up a new site"
```

або

```
"Start the Theme Builder onboarding"
```

Агент Setup Assistant надає ту саму функціональність, але з більшою гнучкістю.

## Порівняння: Site Builder проти Setup Assistant

| Функція | Site Builder (Видалено) | Setup Assistant (Новий) |
|---------|------------------------|------------------------|
| Метод налаштування | Покроковий майстер | Розмова з агентом |
| Вибір теми | Заздалегідь визначені шаблони | Кастомна генерація |
| Кастомізація | Обмежені опції | Повний дизайн-система |
| Site brief | Не зберігається | Зберігання пам'яті |
| Майбутні сесії | Повторне налаштування | Використання збереженого site_brief |
| Гнучкість | Фіклізований робочий процес | Адаптивна розмова |

## Перехід на агент Setup Assistant

### Для нових сайтів

Замість використання режиму конструктора сайтів:

1. Запит: "Help me set up a new site"
2. Агент Setup Assistant проведе вас через:
   - Призначення та цілі сайту
   - Цільову аудиторію
   - Брендову ідентичність
   - Генерацію теми
   - Початкове налаштування

### Для існуючих сайтів

Якщо у вас є існуючий сайт, створений у режимі конструктора сайтів:

1. Ви можете продовжувати користуватися ним як є
2. Щоб оновити тему, зробіть запит: "Redesign my site"
3. Агент Setup Assistant допоможе вам створити нову тему
4. Дані вашого сайту залишаються незмінними

### Для зміни теми

Замість вибору теми в режимі конструктора сайтів:

1. Запит: "Change my theme"
2. Агент Setup Assistant:
   - Запитає про ваші дизайнерські вподобання
   - Згенерує кастомну тему
   - Активує її на вашому сайті

## Ключові відмінності

### Site Builder Mode

```
1. Choose a template
2. Select a theme
3. Configure basic settings
4. Done
```

### Setup Assistant Agent

```
1. Describe your site's purpose
2. Define your target audience
3. Choose design preferences
4. Agent generates custom theme
5. Agent activates theme
6. Site brief stored for future sessions
```

## Переваги агента Setup Assistant

### Більш гнучкий

- Опишіть свій сайт природною мовою
- Отримайте кастомні рекомендації
- Адаптується до ваших конкретних потреб

### Краща кастомізація

- Генерація кастомних тем
- Рішення на основі дизайн-системи
- Зберігання дизайн-токенів

### Зберігання пам'яті

- Ваш site_brief зберігається
- Майбутні агенти розуміють ваш сайт
- Не потрібно повторювати інформацію про налаштування

### Інтегрований робочий процес

- Онбординг Theme Builder
- Навичка Design System Aesthetics
- Керування видимістю можливостей
- Усе працює разом бездоганно

## Усунення несправностей

### Я не можу знайти режим конструктора сайтів

Режим конструктора сайтів було вилучено. Використовуйте агент Setup Assistant:

```
"Help me set up a new site"
```

### Я хочу відтворити сайт з Site Builder

Ви можете відтворити його за допомогою агента Setup Assistant:

1. Запит: "Help me set up a new site"
2. Опишіть призначення та дизайн вашого оригінального сайту
3. Агент згенерує схожу тему
4. Ваш site_brief буде збережено для подальшого використання

### Мій існуючий сайт Site Builder не працює

Існуючі сайти, створені в режимі конструктора сайтів, продовжують працювати. Якщо ви зіткнулися з проблемами:

1. Перевірте, чи ваша тема все ще активна
2. Переконайтеся, що ваші плагіни увімкнені
3. Перевірте журнали помилок WordPress
4. Зверніться до служби підтримки, якщо проблеми зберігаються

### Чи можу я все ще використовувати свої старі шаблони Site Builder?

Шаблони Site Builder більше недоступні. Однак:

- Ваші існуючі сайти продовжують працювати
- Ви можете відтворити схожі сайти за допомогою агента Setup Assistant
- Агент Setup Assistant надає більше опцій кастомізації

## Наступні кроки

1. **Для нових сайтів**: Використовуйте агента Setup Assistant
2. **Для існуючих сайтів**: Продовжуйте користуватися ними як є
3. **Для зміни теми**: Зверніться за допомогою до агента Setup Assistant
4. **Для доопрацювання дизайну**: Використовуйте навичку Design System Aesthetics

## Пов'язані теми

- **Theme Builder Onboarding**: Покрокове налаштування кастомних тем
- **Setup Assistant Agent**: Налаштування сайту з допомогою агента
- **Site Specification Skill**: Визначення цілей та аудиторії вашого сайту
- **Design System Aesthetics Skill**: Доопрацювання візуальної ідентичності вашого сайту
