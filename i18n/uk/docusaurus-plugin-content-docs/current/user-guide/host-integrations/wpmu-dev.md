# Інтеграція з WPMU DEV

## Огляд
WPMU DEV — це комплексна платформа для WordPress, яка пропонує хостинг, плагіни та сервіси для сайтів на WordPress. Ця інтеграція забезпечує автоматичну синхронізацію доменів та керування SSL-сертифікатами між Ultimate Multisite і хостингом WPMU DEV.

## Можливості
- Автоматична синхронізація доменів
- Керування SSL-сертифікатами
- Розширена кількість спроб перевірки SSL-сертифікатів

## Вимоги
Інтеграція автоматично визначає, чи ви використовуєте хостинг WPMU DEV, і застосовує вбудований API. Якщо ви на хостингу WPMU DEV, додаткове налаштування не потрібне.

Інтеграція перевіряє наявність константи `WPMUDEV_HOSTING_SITE_ID`, яка автоматично визначається при використанні хостингу WPMU DEV.

## Інструкції з налаштування

### 1. Перевірте хостинг WPMU DEV

Якщо ви використовуєте хостинг WPMU DEV, необхідні константи вже мають бути визначені. Переконайтеся, що:

1. Константа `WPMUDEV_HOSTING_SITE_ID` визначена у вашому середовищі
2. У вас є активна підписка WPMU DEV з доступом до API

### 2. Увімкніть інтеграцію

1. В адмінпанелі WordPress перейдіть до Ultimate Multisite > Settings
2. Відкрийте вкладку "Domain Mapping"
3. Прокрутіть вниз до "Host Integrations"
4. Увімкніть інтеграцію з WPMU DEV
5. Натисніть "Save Changes"

## Як це працює

### Синхронізація доменів

Коли домен прив'язується в Ultimate Multisite:

1. Інтеграція використовує API WPMU DEV, щоб додати домен до вашого хостинг-акаунту
2. Вона також автоматично додає www-версію домену
3. WPMU DEV бере на себе налаштування домену та випуск SSL-сертифіката

### Керування SSL-сертифікатами

Інтеграція налаштована на збільшену кількість спроб перевірки SSL-сертифікатів для хостингу WPMU DEV, оскільки випуск та встановлення сертифікатів може зайняти деякий час. За замовчуванням система робить до 10 спроб перевірки SSL-сертифіката, на відміну від стандартних 5 спроб.

## Важливі зауваження

### Видалення домену

Наразі API WPMU DEV не підтримує видалення доменів. Коли прив'язку домену видалено в Ultimate Multisite, домен залишиться у вашому хостинг-акаунті WPMU DEV. За потреби вам доведеться вручну видалити його з панелі керування хостингом WPMU DEV.

### Автентифікація API

Інтеграція використовує API-ключ WPMU DEV, який зберігається у базі даних WordPress як опція `wpmudev_apikey`. Він налаштовується автоматично, коли ви під'єднуєте свій сайт до WPMU DEV.

## Вирішення проблем

### Проблеми з підключенням до API
- Переконайтеся, що ваш сайт правильно під'єднаний до WPMU DEV
- Перевірте, чи встановлена опція `wpmudev_apikey` у базі даних WordPress
- Переконайтеся, що ваша підписка WPMU DEV активна

### Проблеми з SSL-сертифікатами
- Випуск SSL-сертифікатів WPMU DEV може зайняти деякий час (зазвичай 5–15 хвилин)
- Інтеграція налаштована на перевірку SSL-сертифікатів до 10 разів
- Якщо SSL-сертифікати все ще не випускаються після кількох спроб, зверніться до підтримки WPMU DEV

### Домен не додається
- Перевірте логи Ultimate Multisite на наявність повідомлень про помилки
- Переконайтеся, що домен ще не додано до WPMU DEV
- Перевірте, чи ваш тарифний план хостингу WPMU DEV підтримує потрібну кількість доменів
