# Інтеграція з GridPane

## Огляд
GridPane — це спеціалізована панель керування хостингом WordPress, створена для професійних користувачів WordPress. Ця інтеграція забезпечує автоматичну синхронізацію доменів та керування SSL-сертифікатами між Ultimate Multisite і GridPane.

## Можливості
- Автоматична синхронізація доменів
- Керування SSL-сертифікатами
- Автоматичне налаштування константи SUNRISE

## Вимоги
У вашому файлі `wp-config.php` потрібно визначити такі константи:

```php
define('WU_GRIDPANE', true);
define('WU_GRIDPANE_API_KEY', 'your_api_key');
define('WU_GRIDPANE_SERVER_ID', 'your_server_id');
define('WU_GRIDPANE_APP_ID', 'your_app_id');
```

## Інструкція з налаштування

### 1. Отримайте API-ключі GridPane

1. Увійдіть у панель керування GridPane
2. Перейдіть до "Settings" > "API"
3. Згенеруйте API-ключ, якщо у вас його ще немає
4. Скопіюйте ваш API-ключ

### 2. Отримайте ідентифікатори сервера та сайту

1. У панелі керування GridPane перейдіть до "Servers"
2. Виберіть сервер, на якому розміщено ваш WordPress multisite
3. Запишіть Server ID (він відображається в URL або на сторінці деталей сервера)
4. Перейдіть до "Sites" і виберіть ваш сайт WordPress
5. Запишіть Site ID (він відображається в URL або на сторінці деталей сайту)

### 3. Додайте константи до wp-config.php

Додайте такі константи до вашого файлу `wp-config.php`:

```php
define('WU_GRIDPANE', true);
define('WU_GRIDPANE_API_KEY', 'your_api_key');
define('WU_GRIDPANE_SERVER_ID', 'your_server_id');
define('WU_GRIDPANE_APP_ID', 'your_site_id');
```

### 4. Увімкніть інтеграцію

1. В адмінпанелі WordPress перейдіть до Ultimate Multisite > Settings
2. Відкрийте вкладку "Domain Mapping"
3. Прокрутіть вниз до "Host Integrations"
4. Увімкніть інтеграцію з GridPane
5. Натисніть "Save Changes"

## Як це працює

Коли домен прив'язується в Ultimate Multisite:

1. Інтеграція надсилає запит до API GridPane для додавання домену до вашого сайту
2. GridPane автоматично забезпечує отримання SSL-сертифіката
3. Коли прив'язку домену видалено, інтеграція видалить домен із GridPane

Інтеграція також автоматично керує константою SUNRISE у вашому файлі wp-config.php, яка необхідна для коректної роботи прив'язки доменів.

## Керування константою SUNRISE

Унікальна особливість інтеграції з GridPane полягає в тому, що вона автоматично скидає константу SUNRISE у wp-config.php, щоб уникнути конфліктів із власною системою прив'язки доменів GridPane. Це гарантує безперебійну роботу обох систем.

## Вирішення проблем

### Проблеми з підключенням до API
- Переконайтеся, що ваш API-ключ правильний
- Перевірте правильність ідентифікаторів сервера та сайту
- Упевніться, що ваш обліковий запис GridPane має необхідні дозволи

### Проблеми з SSL-сертифікатами
- GridPane може потребувати деякого часу для видачі SSL-сертифікатів
- Перевірте, що ваші домени правильно спрямовані на IP-адресу вашого сервера
- Перевірте налаштування SSL у GridPane для вашого сайту

### Домен не додано
- Перевірте логи Ultimate Multisite на наявність повідомлень про помилки
- Переконайтеся, що домен ще не додано до GridPane
- Упевніться, що DNS-записи вашого домену налаштовані правильно
