# Інтеграція з WP Engine

## Огляд
WP Engine — це преміальна платформа керованого хостингу для WordPress, яка забезпечує оптимізовану продуктивність, безпеку та масштабованість для сайтів WordPress. Ця інтеграція дозволяє автоматично синхронізувати домени між Ultimate Multisite та WP Engine.

## Можливості
- Автоматична синхронізація доменів
- Підтримка субдоменів для мультисайтових інсталяцій
- Безшовна інтеграція з існуючими системами WP Engine

## Вимоги
Інтеграція автоматично визначає, чи ваш сайт розміщено на WP Engine, і використовує вбудований WP Engine API. Додаткове налаштування не потрібне, якщо плагін WP Engine активний і правильно налаштований.

Однак, якщо вам потрібно налаштувати інтеграцію вручну, ви можете визначити одну з цих констант у файлі `wp-config.php`:

```php
define('WPE_APIKEY', 'your_api_key'); // Рекомендований метод
// АБО
define('WPE_API', 'your_api_key'); // Альтернативний метод
```

## Інструкція з налаштування

### 1. Перевірте плагін WP Engine

Якщо ваш сайт розміщено на WP Engine, плагін WP Engine має бути вже встановлений та активований. Переконайтеся, що:

1. Плагін WP Engine активний
2. Файл `wp-content/mu-plugins/wpengine-common/class-wpeapi.php` існує

### 2. Увімкніть інтеграцію

1. В адмінпанелі WordPress перейдіть до Ultimate Multisite > Settings
2. Перейдіть на вкладку "Domain Mapping"
3. Прокрутіть вниз до "Host Integrations"
4. Увімкніть інтеграцію з WP Engine
5. Натисніть "Save Changes"

## Як це працює

### Синхронізація доменів

Коли домен прив'язується в Ultimate Multisite:

1. Інтеграція використовує WP Engine API, щоб додати домен до вашої інсталяції WP Engine
2. WP Engine виконує налаштування домену та видачу SSL-сертифіката
3. Коли прив'язку домену видалено, інтеграція видалить домен з WP Engine

### Підтримка субдоменів

Для мультисайтових інсталяцій із субдоменами:

1. Інтеграція додає кожен субдомен до WP Engine при створенні нового сайту
2. WP Engine виконує налаштування субдомену
3. Коли сайт видалено, інтеграція видалить субдомен з WP Engine

## Важливі примітки

### Wildcard-домени

Для мультисайтових інсталяцій із субдоменами рекомендуємо звернутися до служби підтримки WP Engine із запитом на налаштування wildcard-домену. Це дозволить усім субдоменам працювати автоматично без необхідності додавати кожен окремо.

### SSL-сертифікати

WP Engine автоматично видає та оновлює SSL-сертифікати для всіх доменів, доданих через цю інтеграцію. Додаткове налаштування не потрібне.

## Усунення несправностей

### Проблеми з підключенням до API
- Переконайтеся, що плагін WP Engine активний і правильно налаштований
- Якщо ви вручну вказали ключ API, перевірте його правильність
- Зверніться до служби підтримки WP Engine, якщо у вас виникли проблеми з API

### Домен не додано
- Перевірте журнали Ultimate Multisite на наявність повідомлень про помилки
- Переконайтеся, що домен ще не додано до WP Engine
- Перевірте, чи ваш тарифний план WP Engine підтримує потрібну кількість доменів

### Проблеми з субдоменами
- Якщо субдомени не працюють, зверніться до служби підтримки WP Engine із запитом на налаштування wildcard-домену
- Переконайтеся, що ваші DNS-налаштування правильно сконфігуровані для основного домену та субдоменів
