# Контактні форми за допомогою Site Builder

Superdav AI Agent v1.10.0 додає можливість створювати контактні форми безпосередньо з інтерфейсу чату за допомогою агента Site Builder. Це дозволяє додавати повністю функціональні контактні форми на будь-яку сторінку, не виходячи з чату.

## Огляд

Функція контактних форм Site Builder (`create_contact_form`) автоматично виявляє встановлений вами plugin для форм і створює контактну форму, використовуючи його вбудовані можливості. Підтримувані plugin для форм включають:

- Contact Form 7
- WPForms
- Fluent Forms
- Gravity Forms

Асистент автоматично вибирає найкращий доступний plugin для форм на вашому сайті та генерує контактну форму, адаптовану до цього plugin.

## Створення контактної форми

Щоб створити контактну форму за допомогою Site Builder:

1. Відкрийте чат-панель **Gratis AI Agent** в адмін-панелі WordPress.
2. Перейдіть до агента **Site Builder**, натиснувши на іконку агента у заголовку чату.
3. Опишіть контактну форму, яку ви хочете створити. Наприклад:

   > "Додай контактну форму на сторінку «Контакти» з полями для імені, електронної пошти, повідомлення та номера телефону."

   Або просто:

   > "Створи контактну форму для сторінки «Контакти»."

4. Site Builder згенерує контактну форму та поверне shortcode, готовий до вбудовування.

## Використання згенерованого shortcode

Після того, як Site Builder створить контактну форму, він поверне shortcode (наприклад, `[contact-form-7 id="123"]`). Ви можете:

1. **Вбудувати його на сторінці чи в дописі** — Скопіюйте shortcode та вставте його на будь-яку сторінку чи в допис за допомогою block editor або classic editor.
2. **Додати його через Site Builder** — Попросіть Site Builder автоматично додати форму на певну сторінку:

   > "Додай контактну форму на сторінку «Контакти»."

3. **Використовувати його у шаблоні** — Якщо ви знайомі з PHP, ви можете додати shortcode до файлу шаблону теми.

## Налаштування контактної форми

Після створення Site Builder може допомогти вам налаштувати форму додатково:

### Через інтерфейс чату

Попросіть Site Builder змінити форму:

> "Онови контактну форму, додавши поле «Тема» та зробивши поле «Повідомлення» обов'язковим."

Site Builder оновить форму та поверне оновлений shortcode.

### Через адмін-інтерфейс plugin для форм

Ви також можете налаштувати форму безпосередньо в налаштуваннях вашого plugin для форм:

1. Перейдіть до **Contact Form 7** (або вашого встановленого plugin для форм) в адмін-панелі WordPress.
2. Знайдіть форму, створену Site Builder.
3. Відредагуйте поля форми, правила валідації та сповіщення електронною поштою за потреби.

## Plugin для форм та сумісність

Site Builder автоматично виявляє, який plugin для форм встановлено на вашому сайті, і використовує його для створення контактної форми. Якщо встановлено кілька plugin для форм, Site Builder надає їм пріоритет у такому порядку:

1. Contact Form 7
2. WPForms
3. Fluent Forms
4. Gravity Forms

Якщо жоден із цих plugin не встановлено, Site Builder рекомендуватиме встановити один перед створенням контактної форми.

## Сповіщення електронною поштою

Контактні форми, створені Site Builder, за замовчуванням налаштовані на надсилання сповіщень електронною поштою адміністратору сайту. Ви можете налаштувати адресу електронної пошти отримувача та повідомлення про сповіщення:

1. Перейдіть до адмін-інтерфейсу вашого plugin для форм.
2. Знайдіть форму, створену Site Builder.
3. Відредагуйте налаштування сповіщень електронною поштою.

Для детальних інструкцій зверніться до документації вашого plugin для форм:

- [Contact Form 7 Email Notifications](https://contactform7.com/docs/)
- [WPForms Email Notifications](https://wpforms.com/docs/)
- [Fluent Forms Email Notifications](https://fluentforms.com/docs/)
- [Gravity Forms Email Notifications](https://docs.gravityforms.com/)

## Приклади використання

### Проста контактна форма

> "Створи просту контактну форму з полями для імені, електронної пошти та повідомлення."

### Форма з кількома кроками

> "Створи контактну форму з першим кроком для контактної інформації та другим кроком для повідомлення та бажаного способу зв'язку."

### Форма з умовною логікою

> "Створи контактну форму, яка показує різні поля залежно від вибору користувача у випадаючому списку."

### Форма з завантаженням файлів

> "Створи контактну форму, яка дозволяє користувачам завантажити файл або вкладення."

:::note
Створення контактних форм доступне у Superdav AI Agent v1.10.0 та новіших версіях. Для використання цієї функції агент Site Builder має бути активним.
:::
