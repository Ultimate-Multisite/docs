# Керування сайтами

Сайти (або підсайти) — це основа вашого WaaS-бізнесу. Ultimate Multisite підтримує 3 типи сайтів:

- **Клієнтські** — Сайти, що належать конкретним клієнтам
- **Шаблони сайтів** — Заздалегідь налаштовані сайти, які клієнти можуть обрати як відправну точку
- **Головний сайт** — Основний сайт вашої мережі

## Перегляд сайтів

Перейдіть до **Ultimate Multisite → Sites**, щоб побачити всі підсайти вашої мережі. Кожен сайт позначено за типом, і ви можете фільтрувати за категоріями: All Sites, Customer-Owned, Templates або Pending.

![Сторінка списку сайтів](/img/admin/sites-list.png)

## Додавання нового сайту

Натисніть кнопку **Add Site**, щоб створити новий сайт. Вам потрібно буде вказати:

- **Site Title** — Назва нового сайту
- **Site URL** — Домен або шлях до сайту
- **Site Type** — Тип сайту: клієнтський, шаблон чи звичайний

Опція **Copy Site** дозволяє створити новий сайт на основі існуючого шаблону. Коли її увімкнено, ви можете обрати, який шаблон використати як основу. Переконайтеся, що опція **Copy Media on Duplication** увімкнена, щоб скопіювати також медіафайли.

## Керування існуючим сайтом

Натисніть **Manage** на будь-якому сайті, щоб відкрити сторінку **Edit Site**. Тут ви знайдете:

### Основна інформація

Назва сайту, тип, ідентифікатор сайту та опис. Також тут відображається прив'язаний домен, пов'язана підписка та обліковий запис клієнта, якому належить сайт.

### Налаштування сайту

Налаштуйте можливості та обмеження сайту:

- **Ліміти відвідувань** — Максимальна кількість відвідувань сайту
- **Ліміти облікових записів** — Обмеження за ролями користувачів
- **Дисковий простір** — Квота сховища для сайту
- **Власний домен** — Увімкнути прив'язку домену для цього сайту
- **Видимість плагінів і тем** — Керуйте тим, які плагіни та теми видимі, приховані або попередньо активовані

За замовчуванням сайти використовують обмеження, встановлені на рівні підписки. Якщо встановити обмеження на рівні сайту, вони матимуть пріоритет над налаштуваннями підписки.

### Зв'язки

Під налаштуваннями сайту ви знайдете інформацію про:

- **Прив'язані домени**, пов'язані з сайтом
- **Підписку**, до якої належить сайт
- **Обліковий запис клієнта**, прив'язаний до сайту

### Права бічна панель

У правій частині сторінки ви можете:

- **Увімкнути або вимкнути сайт** за допомогою перемикача
- **Змінити тип сайту** або переназначити власника
- **Встановити зображення/мініатюру сайту** (відображається на фронтенді для шаблонів сайтів)
- **Видалити сайт** назавжди

:::warning
Видалення сайту незворотне. Сайт і весь його вміст буде остаточно видалено.
:::
