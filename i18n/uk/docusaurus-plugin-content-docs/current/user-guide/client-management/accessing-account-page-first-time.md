# Як уперше потрапити на сторінку свого акаунта?

Вітаємо в Ultimate Multisite!

Щоб потрапити на сторінку свого акаунта, скористайтеся посиланням, яке ви отримали на електронну пошту після придбання нашого плагіна:

<!-- Screenshot unavailable: Account page email with access link -->

Введіть електронну адресу, яку ви вказали під час покупки, та пароль, надісланий вам на пошту.

<!-- Screenshot unavailable: Login page with email and password fields -->

Оскільки ви входите в цей акаунт уперше, вам запропонують змінити пароль. Змініть його та натисніть кнопку _Зберегти_.

<!-- Screenshot unavailable: Change password form with Save button -->

На сторінці акаунта ви можете керувати всіма його налаштуваннями: платіжними даними, продовженням підписки, скасуванням та багато чим іншим.
