# Здібності конструктора тем: Створення та активація блок-тем

Superdav AI Agent v1.12.0 додає дві потужні функції, які дозволяють генерувати та розгортати кастомні блок-теми безпосередньо через інтерфейс чату.

## Огляд {#overview}

Функції **scaffold-block-theme** та **activate-theme** дозволяють агентам:
- Генерувати повні, готові до використання блок-теми на основі ваших вимог
- Автоматично активувати теми на вашому сайті без ручного втручання
- Створювати цілісну візуальну ідентичність завдяки керованим рішенням щодо дизайну

## Створення базової структури блок-теми (Scaffold Block Theme) {#scaffold-block-theme}

Функція **scaffold-block-theme** генерує нову блок-тему WordPress з повною структурою теми, включаючи:

- Конфігурацію `theme.json` з дизайн-токенами
- Файли шаблонів блоків для поширених макетів
- Кастомні стилі та варіації блоків
- Метадані теми та декларації підтримки

### Як викликати {#how-to-invoke}

У чаті з Superdav AI Agent ви можете запитати генерацію теми:

```
"Create a block theme called 'Modern Agency' with a blue and white color scheme,
sans-serif typography, and a professional layout"
```

Агент виконає наступне:
1. Збережує ваші дизайнерські вподобання під час розмови
2. Згенерує повну структуру теми
3. Створить усі необхідні файли теми
4. Підготує тему до активації

### Очікуваний результат {#expected-output}

Після успішного виконання функції ви побачите:

- Підтвердження того, що структура теми була створена
- Назва та розташування теми
- Підсумок застосованих дизайн-токенів (кольори, типографіка, відступи)
- Статус "Готово до активації"

Приклад результату:
```
✓ Theme "Modern Agency" scaffolded successfully
  Location: /wp-content/themes/modern-agency/
  Colors: Primary #0066CC, Secondary #FFFFFF
  Typography: Inter (sans-serif)
  Status: Ready to activate
```

## Активація теми (Activate Theme) {#activate-theme}

Функція **activate-theme** перемикає ваш сайт на щойно створену або існуючу блок-тему.

### Як викликати {#how-to-invoke-1}

Після створення базової структури теми ви можете активувати її негайно:

```
"Activate the Modern Agency theme"
```

Або активувати будь-яку існуючу тему:

```
"Switch to the Twentytwentyfour theme"
```

### Очікуваний результат {#expected-output-1}

При успішній активації:

- Підтвердження активної теми
- Назва попередньої теми (для довідки)
- URL-адреса сайту, де тема тепер активна
- Будь-які примітки щодо налаштування, специфічні для теми

Приклад результату:
```
✓ Theme activated successfully
  Active theme: Modern Agency
  Previous theme: Twentytwentyfour
  Live at: https://yoursite.com
  Note: Check your homepage to verify the layout
```

## Робочий процес: Створення та активація {#workflow-scaffold-and-activate}

Типовий робочий процес поєднує обидві функції:

1. **Запит на генерацію теми**: "Create a block theme for my SaaS landing page"
2. **Агент створює базову структуру теми**: Генерує файли та дизайн-токени
3. **Перегляд та доопрацювання**: Обговорення змін у дизайні, якщо це необхідно
4. **Активація**: "Activate the theme now"
5. **Перевірка**: Відвідайте свій сайт, щоб підтвердити, що новий дизайн активний

## Дизайн-токени та кастомізація {#design-tokens-and-customization}

Теми, створені за допомогою функції scaffold, використовують дизайн-токени WordPress (через `theme.json`) для:

- **Кольори**: Основний, вторинний, акцентний, нейтральна палітра
- **Типографіка**: Сімейства шрифтів, розміри, ваги, висота рядка
- **Відступи**: Масштаби відступів (padding, margin, gap)
- **Рамки**: Токени радіуса та ширини
- **Тіні**: Рівні підняття (elevation)

Ці токени централізовані у `theme.json`, що дозволяє легко налаштувати всю вашу систему дизайну з одного файлу.

## Обмеження та примітки {#limitations-and-notes}

- Теми створюються у `/wp-content/themes/` і повинні відповідати іменах WordPress
- Активація вимагає відповідних дозволів на вашому сайті WordPress
- Кастомний PHP-код у темах мінімальний; для складної функціональності використовуйте плагіни
- Блок-теми найкраще працюють з WordPress 5.9 і новішими

## Усунення несправностей {#troubleshooting}

**Тема не з'являється після створення базової структури**
- Перевірте, чи існує каталог теми і чи має він відповідні дозволи
- Переконайтеся, що `theme.json` є коректним JSON
- Переконайтеся, що назва теми не конфліктує з існуючими темами

**Активація не вдалася**
- Підтвердіть, що у вас є права адміністратора
- Перевірте, чи каталог теми доступний для читання WordPress
- Перегляньте журнали помилок WordPress, щоб знайти деталі

**Дизайн-токени не застосовуються**
- Перевірте, чи синтаксис `theme.json` правильний
- Очистіть будь-які кешувальні плагіни
- Переконайтеся, що ваша версія WordPress підтримує використані токени

## Наступні кроки {#next-steps}

Після активації теми ви можете:
- Використовувати навичку **Design System Aesthetics** для доопрацювання типографіки, кольорів та відступів
- Кастомізувати стилі окремих блоків через редактор блоків WordPress
- Додавати кастомний CSS у файл `style.css` теми
- Створювати кастомні шаблони блоків для конкретних типів сторінок
