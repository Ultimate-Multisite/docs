# Налаштування панелі адміністратора за допомогою PRO Theme

Часто під час роботи з клієнтами виникає потреба створити для них індивідуальний інтерфейс в адмін-панелі WordPress. Якщо ви шукаєте більш професійний вигляд і зручність — PRO Theme може вам допомогти.

PRO Theme — це тема для адмін-панелі, розроблена для налаштування Dashboard ваших клієнтів. Вона також входить до набору преміум-плагінів Ultimate Multisite.

Після встановлення перейдіть на сторінку _Плагіни_ та натисніть кнопку активації для вашої мережі.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Активуйте плагін для всієї мережі на сторінці Плагіни_

_**ПРИМІТКА:**_ _Якщо ви просто встановите плагін без мережевої активації, він з'явиться у ваших клієнтів як опція, яку можна позначити, щоб додати тему лише для конкретного користувача._

Тепер ваш адміністративний інтерфейс повністю оновлено! У лівій колонці також з'явилося нове меню під назвою PRO Theme.
