# Розміщення реклами у вашій мережі за допомогою Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector — це **безкоштовний додаток**, який дає вам інструменти для розміщення реклами на ваших планах прямо зі сторінки їх редагування.

Після встановлення Ultimate Multisite: Ad Injector перейдіть на сторінку _Плагіни_, щоб активувати його.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Натисніть кнопку «Активувати для мережі»_

Після активації додатка в мета-блоці _Розширені налаштування_ на сторінці редагування плану з'явиться нова опція _Налаштування реклами_.

![Налаштування реклами на сторінці редагування плану](/img/config/product-edit-full.png)

_Тепер ви можете редагувати налаштування реклами для кожного з ваших планів_

На цій вкладці ви зможете додати рекламні коди для записів і сторінок у публічній частині сайту — до та після контенту.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Після вставки рекламного коду він відображатиметься на записах і сторінках відповідного плану._

Ви також можете додавати рекламу в заголовок сторінок адмінпанелі для користувачів певного плану.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Перейдіть до опції «Рекламний код адмінпанелі», якщо хочете показувати рекламу в адміністративній частині сайту ваших користувачів_

### Не лише реклама

Ви також можете використовувати Ultimate Multisite: Ad Injector, щоб запропонувати підтримку для преміум-планів.

Просто перейдіть до опції _Реклама в адмінпанелі_ та додайте HTML-код вашого інструменту підтримки.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Додайте підтримку клієнтів для певних планів_

Таким чином люди зможуть зв'язатися з вами прямо зі своєї адмінпанелі!
