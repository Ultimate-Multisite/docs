# Керування посиланнями для поширення планів

Ultimate Multisite дає вам можливість ділитися посиланнями на плани безпосередньо з клієнтами. Для цього перейдіть на сторінку редагування будь-якого плану і натисніть кнопку _Натисніть, щоб скопіювати посилання для поширення_.

![Сторінка редагування продукту з кнопкою посилання для поширення](/img/config/product-edit-full.png)

_На сторінці редагування плану ви знайдете кнопку для створення посилань для поширення_

Зверніть увагу, що цей механізм використовує стандартну частоту оплати для формування посилання. Це означає, що якщо ваша стандартна оплата, наприклад, щомісячна, ви отримаєте посилання на кшталт: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Якщо ви хочете, щоб посилання для поширення за замовчуванням вело на квартальну оплату, вам потрібно додати 3 в кінці

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Щоб обрати план _premium_ з річним розкладом оплати, додайте 1, а потім слово:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Майте на увазі, що ви не можете мати одне посилання для різних частот оплати. Вам потрібно ділитися окремим посиланням для кожної частоти оплати.

Коли ви поділитеся посиланням із клієнтами, воно перенаправить їх на перший крок процесу реєстрації.
