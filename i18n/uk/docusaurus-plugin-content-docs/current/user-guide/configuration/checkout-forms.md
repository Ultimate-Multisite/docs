# Форми оформлення замовлення

Форми оформлення замовлення — це простий і гнучкий спосіб експериментувати з різними підходами для залучення нових клієнтів.

Ultimate Multisite 2.0 пропонує редактор форм оформлення замовлення, який дозволяє створювати скільки завгодно форм з різними полями, товарами тощо.

Щоб отримати доступ до цієї функції, перейдіть до меню Checkout Forms на бічній панелі зліва.

![Список форм оформлення замовлення](/img/config/checkout-forms-list.png)

На цій сторінці ви бачите всі свої форми оформлення замовлення.

Якщо хочете створити нову форму, просто натисніть Add Checkout Form у верхній частині сторінки.

Ви можете обрати один із трьох варіантів для початку: односторінкова, багатосторінкова або порожня форма. Потім натисніть Go to the Editor.

![Редактор форми оформлення замовлення](/img/config/checkout-form-editor.png)

Також ви можете редагувати або дублювати наявні форми, натиснувши на опції під її назвою. Там же ви знайдете можливість скопіювати шорткод форми або видалити її.

![Дії при наведенні на форму оформлення замовлення](/img/config/checkout-form-hover-actions.png)

### Редагування форми оформлення замовлення

Ви можете створювати форми оформлення замовлення для різних цілей. У цьому прикладі ми працюватимемо з формою реєстрації.

Перейшовши до редактора форми оформлення замовлення, задайте їй назву (використовується лише для внутрішніх потреб) та slug (використовується, наприклад, для створення шорткодів).

![Редактор форми оформлення замовлення](/img/config/checkout-form-editor.png)

Форми складаються з кроків і полів. Ви можете додати новий крок, натиснувши Add New Checkout Step.

![Додати новий крок оформлення замовлення](/img/config/checkout-form-add-step.png)

На першій вкладці модального вікна заповніть вміст кроку форми. Вкажіть ID, назву та опис. Ці елементи переважно використовуються для внутрішніх потреб.

![Крок форми оформлення замовлення](/img/config/checkout-form-step.png)

Далі налаштуйте видимість кроку. Ви можете обрати "Always show", "Only show for logged in users" або "Only show for guests".

![Крок форми оформлення замовлення](/img/config/checkout-form-step.png)

Нарешті, налаштуйте стиль кроку. Ці поля необов'язкові.

![Крок форми оформлення замовлення](/img/config/checkout-form-step.png)

Тепер час додати поля до нашого першого кроку. Просто натисніть Add New Field і виберіть потрібний тип секції.

![Крок форми оформлення замовлення з полями](/img/config/checkout-form-step.png)

Кожне поле має різні параметри для заповнення. Для першого поля ми оберемо "Username".

![Крок форми оформлення замовлення](/img/config/checkout-form-step.png)

![Крок форми оформлення замовлення](/img/config/checkout-form-step.png)

![Крок форми оформлення замовлення](/img/config/checkout-form-step.png)

Ви можете додавати скільки завгодно кроків і полів. Щоб показати клієнтам ваші продукти на вибір, використовуйте поле Pricing Table. Якщо хочете дати клієнтам можливість обрати шаблон, додайте поле Template Selection. І так далі.

_**Примітка:** поля username, email, password, site title, site URL, order summary, payment та submit button є обов'язковими для створення форми оформлення замовлення._

Під час роботи над формою оформлення замовлення ви завжди можете скористатися кнопкою Preview, щоб побачити, як форма виглядатиме для клієнтів. Також ви можете перемикатися між переглядом від імені зареєстрованого користувача або відвідувача.

![Збереження форми оформлення замовлення](/img/config/checkout-form-save.png)

![Редактор форми оформлення замовлення](/img/config/checkout-form-editor.png)

Нарешті, у розділі Advanced Options ви можете налаштувати повідомлення для сторінки подяки, додати сніпети для відстеження конверсій, додати власний CSS до форми оформлення замовлення або обмежити її для певних країн.

![Розширені налаштування](/img/config/checkout-form-advanced.png)

Ви також можете вручну увімкнути або вимкнути форму оформлення замовлення за допомогою перемикача в правій колонці, або назавжди видалити форму.

![Перемикач активації](/img/config/checkout-form-active.png)

Не забудьте зберегти форму оформлення замовлення!

![Кнопка збереження](/img/config/checkout-form-save.png)

Щоб отримати шорткод форми, натисніть Generate Shortcode і скопіюйте результат, показаний у модальному вікні.

![Кнопка збереження з шорткодом](/img/config/checkout-form-save.png)
