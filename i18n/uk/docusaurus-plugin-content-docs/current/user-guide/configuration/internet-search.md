# Пошук в інтернеті

Gratis AI Agent v1.5.0 додає функцію **Пошук в інтернеті** (Internet Search), яка дозволяє AI-асистенту отримувати актуальну інформацію з вебсторінок під час розмови. Ця функція працює за допомогою [Brave Search API](https://brave.com/search/api/) або [Tavily API](https://tavily.com/).

## Як це працює

Коли пошук в інтернеті увімкнено, асистент може автоматично робити запити до вашого налаштованого пошукового провайдера, якщо визначить, що запитання вимагає актуальної або зовнішньої інформації — наприклад, останніх новин, поточних цін або документації, яка могла змінитися з моменту навчання моделі.

Результати отримуються в режимі реального часу та передаються в контекст асистента перед генерацією відповіді. Асистент повідомляє, що він використав результати пошуку для відповіді на запитання.

## Увімкнення пошуку в інтернеті

Для пошуку в інтернеті потрібен API-ключ від обраного пошукового провайдера. Щоб налаштувати його:

1. Перейдіть до **Gratis AI Agent → Settings → Advanced**.
2. Знайдіть випадаючий список **Internet Search Provider** та виберіть або **Brave Search**, або **Tavily**.
3. Введіть свій API-ключ у відповідне поле. Посилання для реєстрації відображаються поруч із кожним полем.
4. Натисніть **Save Settings**.

Після збереження ключа функція Пошук в інтернеті автоматично стає доступною для асистента.

## Brave Search

### Отримання API-ключа Brave Search

1. Відвідайте [Brave Search API page](https://brave.com/search/api/).
2. Зареєструйтеся на план. Доступний безкоштовний рівень з обмеженням кількості запитів на місяць.
3. Скопіюйте свій API-ключ із панелі розробника Brave Search.
4. Вставте його у поле **Brave Search API Key** в налаштуваннях Gratis AI Agent.

### Ліміти використання

Використання виставляється рахунок Brave Search на основі кількості виконаних запитів. Кожна відповідь AI, яка ініціює пошук, рахується як один запит. Відстежуйте ваше використання на [Brave Search developer dashboard](https://brave.com/search/api/), щоб уникнути несподіваних витрат.

## Tavily

Superdav AI Agent v1.10.0 додає **Tavily** як альтернативного провайдера пошуку в інтернеті, пропонуючи більш багаті результати пошуку та розширені можливості досліджень.

### Отримання API-ключа Tavily

1. Відвідайте [Tavily API page](https://tavily.com/).
2. Зареєструйтеся на обліковий запис. Доступний безкоштовний рівень з обмеженням кількості запитів на місяць.
3. Скопіюйте свій API-ключ із панелі Tavily.
4. Вставте його у поле **Tavily API Key** в налаштуваннях Gratis AI Agent.

### Ліміти використання

Використання виставляється рахунок Tavily на основі кількості викликів API. Кожна відповідь AI, яка ініціює пошук, рахується як один виклик. Відстежуйте ваше використання на [Tavily dashboard](https://tavily.com/), щоб уникнути несподіваних витрат.

## Вимкнення пошуку в інтернеті

Видаліть API-ключ із поля активного пошукового провайдера та збережіть зміни. Функція Пошук в інтернеті більше не буде доступна для асистента.

:::note
Пошук в інтернеті додає затримку до відповідей, оскільки асистенту потрібно чекати на результати пошуку, перш ніж генерувати відповідь. Для випадків, де час є критичним, подумайте, чи необхідний вам пошук у реальному часі, чи достатньо вбудованих знань асистента.
:::
