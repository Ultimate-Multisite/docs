# Автоматичний пошук і заміна в шаблонах сайтів (v2)

_**Цей посібник призначений для WP Ultimo версії 2.x.**_

Однією з найпотужніших функцій Ultimate Multisite є можливість додавати до форми реєстрації довільні текстові поля, поля вибору кольору та списки. Після того як ми отримали ці дані, їх можна використовувати для попереднього заповнення контенту в певних частинах обраного шаблону сайту. Тоді, коли новий сайт буде опубліковано, Ultimate Multisite замінить плейсхолдери на реальну інформацію, введену під час реєстрації.

Наприклад, ви можете створювати шаблони сайтів із плейсхолдерами. Плейсхолдери слід додавати в подвійних фігурних дужках — {{placeholder_name}}.

Потім ви просто додаєте відповідне поле реєстрації, щоб отримати ці дані.

Ваш клієнт зможе заповнити це поле під час реєстрації.

Ultimate Multisite автоматично замінить плейсхолдери на дані, які надав клієнт.

## **Вирішення проблеми "шаблон, заповнений плейсхолдерами"**

Все це чудово, але ми стикаємося з неприємною проблемою: тепер наші шаблони сайтів, які можуть переглядати клієнти, заповнені негарними плейсхолдерами, які мало про що говорять.

Щоб вирішити цю проблему, ми пропонуємо можливість встановлювати тимчасові значення для плейсхолдерів — вони використовуються для пошуку та заміни контенту в шаблонах сайтів, коли їх переглядають ваші клієнти.

Щоб відкрити редактор плейсхолдерів шаблону, перейдіть до **Ultimate Multisite > Settings > Sites**, а потім натисніть посилання Edit Placeholders на бічній панелі.

![Посилання Edit Placeholders на бічній панелі налаштувань Sites](/img/config/settings-sites.png)

Це відкриє редактор контенту плейсхолдерів, де ви можете додавати плейсхолдери та їхні відповідні значення.

![Редактор контенту плейсхолдерів із назвами та значеннями плейсхолдерів](/img/config/settings-sites.png)
