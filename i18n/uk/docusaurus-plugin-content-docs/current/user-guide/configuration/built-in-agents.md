# Вбудовані агенти {#built-in-agents}

Gratis AI Agent v1.9.0 постачається з п’ятьма вбудованими агентами, кожен із яких попередньо налаштований із цільовим набором інструментів, адаптованим системним промптом і початковими пропозиціями, що відповідають поширеним завданням у цій сфері. Перемикання між агентами змінює те, що асистент може робити і як він відповідає — без жодного налаштування з вашого боку. Superdav AI Agent v1.18.0 може додавати до цих робочих процесів інструменти з урахуванням розкладу, записи нагадувань, шлюзи затвердження та SMS-сповіщення, коли відповідні інтеграції налаштовано.

## Що таке агент? {#what-is-an-agent}

Кожен агент — це іменований профіль конфігурації, який поєднує:

- **Інструменти** — можливості, які агенту дозволено викликати (наприклад, Content Writer має доступ до можливостей створення записів; Design Studio має доступ до можливостей CSS і theme.json)
- **Системний промпт** — інструкції, які задають тон, пріоритети та обмеження агента
- **Пропозиції** — заздалегідь написані промпти, що відображаються в інтерфейсі чату, щоб допомогти вам швидко почати

## Доступ до вибору агента {#accessing-the-agent-picker}

1. Відкрийте панель **Gratis AI Agent** у бічній панелі адміністратора WordPress.
2. Натисніть **іконку агента** у верхньому лівому куті заголовка чату (іконка змінюється відповідно до активного агента).
3. **Agent Picker** відкриється як накладення у форматі таблиці форми. Кожен агент показаний зі своєю іконкою, назвою та однорядковим описом.
4. Натисніть рядок агента, щоб активувати його. Заголовок чату оновиться негайно.

Ви також можете перемикати агентів посеред розмови — системний промпт нового агента набуде чинності з наступного повідомлення.

## П’ять вбудованих агентів {#the-five-built-in-agents}

### Content Writer {#content-writer}

**Фокус:** Створення та редагування записів, сторінок і контактних форм.

**Доступні інструменти:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`. Якщо інтеграції Superdav AI Agent v1.18.0 увімкнено, налаштований контекст календаря, шлюзи затвердження, нагадування та інструменти SMS-сповіщень також можуть бути доступні для затверджених робочих процесів.

**Що він робить добре:**
- Створення чернеток і публікація дописів блогу на основі брифу або плану
- Створення пакетів лендінг-сторінок для нового сайту
- Побудова контактних форм і форм запитів
- Установлення головних зображень для записів із URL або пошуку
- Підготовка повідомлень для подальшої комунікації після події на основі налаштованого контексту Google Calendar, а потім пауза для затвердження перед надсиланням сповіщень

**Початкові пропозиції:**
- *Напиши допис у блог на 500 слів про переваги WordPress multisite.*
- *Створи сторінки «Про нас», «Послуги» та «Контакти» і опублікуй їх.*
- *Додай форму запиту на бронювання на сторінку «Контакти».*
- *Підготуй нагадування для учасників завтрашньої налаштованої події в календарі та зачекай на затвердження перед його надсиланням.*

---

### Site Builder {#site-builder}

**Фокус:** Наскрізне створення вебсайту з одного промпта.

**Доступні інструменти:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`. З Superdav AI Agent v1.18.0 налаштовані інструменти керованих послуг, затвердження, нагадувань, календаря та SMS можуть бути доступні там, де адміністратори їх увімкнуть.

**Що він робить добре:**
- Генерування багатоетапного плану створення сайту для описаного типу бізнесу
- Автономне виконання кожного етапу — структура, контент, навігація, дизайн
- Відновлення після помилок у середині плану без потреби в ручному втручанні
- Встановлення рекомендованих plugins як частини створення
- Створення контактних форм безпосередньо з інтерфейсу чату (Superdav AI Agent v1.10.0+)
- Координація нагадувань про запуск або подальшої комунікації з учасниками без дублювання сповіщень, коли ввімкнено шлюзи затвердження та записи нагадувань

**Початкові пропозиції:**
- *Створи сайт портфоліо фотографа з типом запису галереї, сторінкою бронювання та контактною формою.*
- *Створи вебсайт ресторану з онлайн-меню, годинами роботи та формою запиту на бронювання столика.*
- *Налаштуй сайт для фриланс-консультанта зі сторінками послуг, розділом портфоліо та блогом.*
- *Додай контактну форму на сторінку «Контакти» за допомогою site builder.*
- *Після затвердження контрольного списку запуску сайту надішли SMS-нагадування налаштованому контакту зацікавленої сторони.*

### Design Studio {#design-studio}

**Фокус:** Візуальне налаштування — кольори, типографіка, CSS і блокові шаблони.

**Доступні інструменти:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Що він робить добре:**
- Застосування іменованих пресетів теми (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- Точне налаштування глобальної типографіки та колірних палітр через theme.json
- Вставлення власного CSS для бренд-специфічних перевизначень
- Створення знімка екрана сторінки та її перевірка на проблеми дизайну

**Початкові пропозиції:**
- *Застосуй пресет warm-editorial, а потім встанови основний колір на #2d6a4f.*
- *Зроби знімок екрана головної сторінки та скажи, що б ти покращив.*
- *Створи багаторазовий hero-блоковий шаблон із фоновим зображенням на всю ширину та центрованим заголовком.*

### Plugin Manager {#plugin-manager}

**Фокус:** Пошук, встановлення та керування WordPress plugins.

**Доступні інструменти:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Що він робить добре:**
- Рекомендування найкращого plugin для описаного випадку використання
- Встановлення пакетів можливостей із реєстру
- Перегляд доступного каталогу можливостей за категоріями

**Початкові пропозиції:**
- *Який найкращий plugin для каталогу учасників?*
- *Встанови пакет можливостей WooCommerce.*
- *Покажи мені всі доступні пакети можливостей для електронної комерції.*

---

### Support Assistant {#support-assistant}

**Фокус:** Відповіді на запитання про контент сайту, налаштування та конфігурацію WordPress.

**Доступні інструменти:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Що він добре робить:**
- Шукає поточні налаштування та опції сайту
- Пояснює, які типи дописів, таксономії та меню налаштовано на сайті
- Відповідає на запитання «що робить це налаштування?», читаючи актуальні значення
- Слугує діагностичним шаром лише для читання перед внесенням змін

**Початкові пропозиції:**
- *Які плагіни та налаштування зараз активні на цьому сайті?*
- *Перелічіть усі користувацькі типи дописів, зареєстровані на цьому сайті.*
- *Які навігаційні меню існують і де вони призначені?*

## Інтеграції автоматизації Superdav {#superdav-automation-integrations}

Коли інтеграції Superdav AI Agent v1.18.0 налаштовано, вбудовані агенти можуть брати участь у безпечніших робочих процесах автоматизації з урахуванням розкладу:

- **Інструменти читання Google Calendar** дозволяють агентам перевіряти налаштовані календарі та події перед підготовкою подальшої роботи.
- **Зіставлення контактів і учасників** допомагає зіставляти учасників подій із користувачами WordPress або відомими контактами.
- **Шлюзи схвалення людиною** призупиняють чутливі дії, доки авторизований користувач не перегляне й не підтвердить їх.
- **Записи нагадувань** запобігають дублюванню сповіщень, коли заплановані завдання повторюються або виконуються повторно після невдачі.
- **TextBee SMS-сповіщення** надсилають налаштовані текстові повідомлення лише тоді, коли облікові дані SMS і дозволи робочого процесу ввімкнено.

Рекомендований робочий процес: попросіть агента підготувати повідомлення або дію, перегляньте запит на схвалення, а потім дозвольте схваленій дії продовжитися. Для повторюваних нагадувань залиште дедуплікацію нагадувань увімкненою, щоб ту саму подію або контакт не було сповіщено повторно.

## Налаштування агентів {#customising-agents}

Кожного вбудованого агента можна розширити або замінити через фільтр `gratis_ai_agent_agents`.

### Додавання користувацької системної підказки до наявного агента {#adding-a-custom-system-prompt-to-an-existing-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### Реєстрація нового агента {#registering-a-new-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

Новий агент з’являється в Agent Picker одразу після виконання фільтра.

### Видалення вбудованого агента {#removing-a-built-in-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
