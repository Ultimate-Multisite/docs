# Вбудовані Агенти

Gratis AI Agent v1.9.0 постачається з п'ятьма вбудованими агентами. Кожен з них попередньо налаштований із фокусованим набором інструментів, спеціалізованим системним промптом та стартовими підказками, які відповідають типовим завданням у певній сфері. Перемикання між агентами змінює можливості та спосіб відповіді помічника — і все це без жодної конфігурації з вашого боку.

## Що таке Агент?

Кожен агент — це іменований профіль конфігурації, який поєднує:

- **Інструменти (Tools)** — можливості, які агент може викликати (наприклад, Контент-райтер має доступ до функцій створення постів; Дизайн-студія має доступ до функцій CSS та theme.json).
- **Системний промпт (System prompt)** — інструкції, які задають тон, пріоритети та обмеження агента.
- **Підказки (Suggestions)** — попередньо написані промпти, які відображаються в інтерфейсі чату, щоб допомогти вам швидко розпочати роботу.

## Як отримати доступ до вибору Агента

1. Відкрийте панель **Gratis AI Agent** у бічній панелі адміністратора WordPress.
2. Натисніть на **іконку агента** у верхньому лівому куті заголовка чату (іконка змінюється відповідно до активного агента).
3. Відкривається **Agent Picker** (Вибір Агента) у вигляді оверлей-таблиці. Кожен агент перелічено з його іконкою, назвою та однорядковим описом.
4. Натисніть на рядок агента, щоб активувати його. Заголовок чату оновлюється негайно.

Ви також можете перемикати агентів під час розмови — системний промпт нового агента набуває чинності з наступного повідомлення.

## П'ять вбудованих Агентів

### Content Writer (Контент-райтер)

**Фокус:** Створення та редагування постів, сторінок та контактних форм.

**Доступні інструменти:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`

**Що робить добре:**
- Створення та публікація статей у блозі на основі короткого опису або плану.
- Створення пакетів цільових сторінок для нового сайту.
- Створення контактних та запитних форм.
- Встановлення зображень у виді (featured images) для постів за URL або пошуком.

**Стартові підказки:**
- *Напиши 500-слівну статтю для блогу про переваги WordPress multisite.*
- *Створи сторінки «Про нас», «Послуги» та «Контакти» і опублікуй їх.*
- *Додай форму запиту бронювання на сторінку «Контакти».*

---

### Site Builder (Сайт-білдер)

**Фокус:** Створення вебсайту «під ключ» за однією підказкою.

**Доступні інструменти:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`

**Що робить добре:**
- Генерація багатоетапного плану створення сайту для описаного типу бізнесу.
- Автономне виконання кожного етапу — структура, контент, навігація, дизайн.
- Відновлення після помилок під час плану без необхідності ручного втручання.
- Встановлення рекомендованих плагінів як частина збірки.
- Створення контактних форм безпосередньо з інтерфейсу чату (Superdav AI Agent v1.10.0+).

**Стартові підказки:**
- *Створи сайт-портфоліо для фотографа з типом постів «Галерея», сторінкою бронювання та контактною формою.*
- *Створи вебсайт ресторану з онлайн-меню, годинами роботи та формою запиту столика.*
- *Налаштуй сайт для фриланс-консультанта з сторінками послуг, розділом портфоліо та блогом.*
- *Додай контактну форму на сторінку «Контакти» за допомогою сайт-білдера.*

---

### Design Studio (Дизайн-студія)

**Фокус:** Візуальна кастомізація — кольори, типографіка, CSS та шаблони блоків.

**Доступні інструменти:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Що робить добре:**
- Застосування іменованих пресетів тем (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog).
- Тонке налаштування глобальної типографіки та палітри кольорів через theme.json.
- Впровадження кастомного CSS для перевизначення стилів під бренд.
- Створення скріншота сторінки та перегляд його на предмет проблем із дизайном.

**Стартові підказки:**
- *Застосуй пресет warm-editorial, а потім встанови основний колір на #2d6a4f.*
- *Зроби скріншот головної сторінки та скажи мені, що б ти покращив.*
- *Створи багаторазовий шаблон секції «Герой» з фоновим зображенням по повній ширині та відцентрованим заголовком.*

---

### Plugin Manager (Менеджер плагінів)

**Фокус:** Пошук, встановлення та управління плагінами WordPress.

**Доступні інструменти:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Що робить добре:**
- Рекомендація найкращого плагіна для описаного сценарію використання.
- Встановлення пакетів функціоналу з реєстру.
- Перегляд доступного каталогу функціоналу за категоріями.

**Стартові підказки:**
- *Який найкращий плагін для каталогу членства?*
- *Встанови пакет функціоналу WooCommerce.*
- *Покажи мені всі доступні пакети функціоналу для електронної комерції.*

---

### Support Assistant (Помічник підтримки)

**Фокус:** Відповіді на запитання про контент сайту, налаштування та конфігурацію WordPress.

**Доступні інструменти:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Що робить добре:**
- Пошук поточних налаштувань та опцій сайту.
- Пояснення, які типи постів, таксономії та меню налаштовані на сайті.
- Відповіді на запитання типу «що робить це налаштування?» шляхом зчитування активних значень.
- Виконання функції лише для читання (read-only) діагностичного шару перед внесенням змін.

**Стартові підказки:**
- *Які плагіни та налаштування зараз активні на цьому сайті?*
- *Перерахуй усі кастомні типи постів, зареєстровані на цьому сайті.*
- *Які навігаційні меню існують і куди вони призначені?*

## Кастомізація Агентів

Кожен вбудований агент можна розширити або замінити за допомогою фільтра `gratis_ai_agent_agents`.

### Додавання кастомного системного промпта до існуючого агента

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### Реєстрація нового агента

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

Новий агент з'явиться у виборі Агентів одразу після запуску фільтра.

### Видалення вбудованого агента

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
