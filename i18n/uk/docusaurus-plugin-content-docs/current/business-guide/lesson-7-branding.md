# Урок 7: Персоналізація {#lesson-7-making-it-yours}

Ваші клієнти не повинні відчувати, що вони користуються "якимсь плагіном WordPress". Вони мають відчувати, що використовують FitSite — платформу, створену спеціально для їхньої галузі. У цьому уроці ми розглянемо брендинг, white-labeling та те, як змусити платформу виглядати як готовий продукт.

## З чого ми почали {#where-we-left-off}

FitSite має робочий процес оформлення замовлення, який веде власників фітнес-студій від вибору тарифного плану до запуску сайту. Тепер ми змусимо весь досвід виглядати як цілісний, брендований продукт.

## Домен вашої платформи {#your-platform-domain}

Основою вашого бренду є ваш домен. Для FitSite:

- **Основний домен**: `fitsite.com` (ваш маркетинговий сайт та коренева мережа)
- **Сайт клієнта**: `studioname.fitsite.com` (піддомени)
- **Кастомні домени**: Клієнти на планах Growth та Pro можуть підключити власний домен

### Налаштування домену {#setting-up-your-domain}

1. Зареєструйте домен вашої платформи
2. Направте його на вашого хостинг-провайдера
3. Налаштуйте wildcard DNS (`*.fitsite.com`) для піддоменів клієнтів
4. Переконайтеся, що активний wildcard SSL

Детальні інструкції дивіться у [How to Configure Domain Mapping](/user-guide/domain-mapping/how-to-configure-domain-mapping).

## White-labeling адмін-досвіду {#white-labeling-the-admin-experience}

Коли власник фітнес-студії входить у панель керування свого сайту, він має бачити ваш бренд, а не брендинг WordPress чи Ultimate Multisite.

### Кастомна сторінка входу {#custom-login-page}

Налаштуйте сторінку входу WordPress, щоб вона відображала:

- Ваш логотип FitSite
- Фон, відповідний фітнесу
- Кольори вашого бренду

### Брендинг панелі керування {#dashboard-branding}

Використовуйте аддон [Admin Page Creator](/addons/admin-page-creator) або кастомний CSS, щоб:

- Замінити логотип WordPress на логотип FitSite
- Налаштувати схему кольорів адмін-панелі відповідно до вашого бренду
- Додати кастомний віджет на панель керування з швидкими посиланнями та ресурсами допомоги, специфічними для фітнесу

### Кастомні адмін-сторінки {#custom-admin-pages}

Розгляньте створення кастомних адмін-сторінок, які виділяють найбільш релевантні дії для власників фітнес-студій:

- "Редагувати розклад занять"
- "Оновлювати профілі тренерів"
- "Переглянути свій сайт"

Це зменшує криву навчання, виносячи дії, релевантні для ніші, на передній план, а не ховаючи їх у стандартному меню WordPress.

## Брендинг комунікацій {#branding-your-communications}

Кожен email, рахунок-фактура та сповіщення мають підсилювати ваш бренд.

### Системні email {#system-emails}

Перейдіть до **Ultimate Multisite > Settings > Emails** та налаштуйте всі системні email:

- **From name**: FitSite
- **From email**: hello@fitsite.com
- **Email templates**: Використовуйте кольори та логотип вашого бренду
- **Language**: Фітнес-специфічний у всьому тексті

Ключові email для налаштування:

| Email | Generic Version | FitSite Version |
|-------|----------------|-----------------|
| Welcome | "Your new site is ready" | "Ваш сайт фітнес-студії запущено" |
| Payment receipt | "Payment received" | "Підтверджено оплату підписки FitSite" |
| Trial ending | "Your trial is ending soon" | "Ваша пробна підписка FitSite закінчується через 3 дні — тримайте свій сайт фітнес-студії активним" |

### Рахунки-фактури {#invoices}

Налаштуйте шаблони рахунків-фактур, додавши:

- Ваш логотип FitSite та кольори бренду
- Ваші бізнес-відомості
- Назви продуктів, специфічні для фітнесу (а не загальні ID планів)

## Сайт для клієнтів {#the-customer-facing-site}

Ваш основний домен (`fitsite.com`) потребує маркетингового сайту, який продає цю платформу. Він відокремлений від адмін-мережі Ultimate Multisite — це публічне обличчя вашого бізнесу.

Ключові сторінки:

- **Головна**: Чітка цінність для фітнес-бізнесу
- **Features**: Що робить FitSite, але у фітнес-термінах
- **Pricing**: Ваші три плани з порівнянням функцій, специфічних для ніші
- **Examples**: Демонстрація сайтів, створених на платформі
- **Sign Up**: Посилання на вашу форму оформлення замовлення

:::tip Ваш маркетинговий сайт може бути сайтом мережі
Створіть свій маркетинговий сайт як сайт у власній мережі. Це дозволить вам керувати ним з однієї панелі керування та продемонструє можливості вашої власної платформи.
:::

## Кастомний домен для клієнтів {#custom-domain-for-customers}

Для клієнтів, які підключили домени, включені в їхні плани, чітко документуйте процес:

1. Клієнт реєструє або передає свій домен до реєстратора
2. Клієнт оновлює DNS, щоб він вказував на вашу платформу (надайте точні записи)
3. Ultimate Multisite займається мапінгом домену та SSL

Створіть навчальну статтю або запис у базі знань, спеціально для цього процесу, написаний для нетехнічних власників фітнес-студій.

## Мережа FitSite наразі {#the-fitsite-network-so-far}

```
FitSite Network
├── WordPress Multisite (subdomain mode)
├── Ultimate Multisite (configured + branded)
├── Platform Domain (fitsite.com + wildcard SSL)
├── Site Templates (Studio Essential, Gym Pro, Fitness Chain)
├── Products (Starter, Growth, Pro + Order Bumps)
├── Checkout Flow (niche-specific, tested)
├── Branding
│   ├── Custom login page
│   ├── Branded admin dashboard
│   ├── Niche-specific system emails
│   ├── Branded invoices
│   └── Marketing site on fitsite.com
└── Ready for onboarding flow (next lesson)
```

## Що ми створили в цьому уроці {#what-we-built-this-lesson}

- **Домен платформи та DNS** налаштовані для брендованого досвіду
- **White-labeled адмін-панель** з брендингом FitSite
- **Кастомізовані комунікації** — email, рахунки-фактури та сповіщення повністю відповідають бренду
- **Маркетинговий сайт**, який продає FitSite власникам фітнес-студій
- **Документація кастомних доменів** для клієнтів, які хочуть власний домен
