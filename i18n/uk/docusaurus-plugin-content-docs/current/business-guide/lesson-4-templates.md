# Урок 4: Створення нішевих шаблонів

Шаблони є основою вашої нішевої цінності. Власник фітнес-студії, який реєструється і бачить сайт, що вже виглядає як фітнес-вебсайт — з розкладом занять, профілями тренерів та відповідною графікою — з більшою ймовірністю залишиться, ніж той, хто бачить чистий аркуш.

## З чого ми зупинилися

Ми маємо робочу мережу FitSite з встановленим та налаштованим Ultimate Multisite. Тепер ми створюємо шаблони, які змусять FitSite виглядати так, ніби він створений спеціально для фітнес-бізнесу.

## Чому важливі нішеві шаблони

Загальні шаблони змушують ваших клієнтів виконувати важку роботу: визначати, які сторінки їм потрібні, яка структура контенту працює і як зробити, щоб це виглядало відповідно до їхньої галузі. Нішеві шаблони усувають ці труднощі.

Власник фітнес-студії, який реєструється на FitSite, повинен побачити:

- Головну сторінку, що виглядає як сайт фітнес-студії
- Сторінки для занять, розкладу, тренерів та цін, які вже створені
- Графіку та заповнювальний контент, що відповідає їхній галузі
- Дизайн, який виглядає професійно та відповідає бренду фітнесу

Вони просто заповнюють свої деталі. Їм не потрібно починати з нуля.

## Планування шаблонів

Перш ніж створювати, вирішіть, які шаблони ви хочете запропонувати. Для FitSite ми створимо три:

### Шаблон 1: Studio Essential

Для невеликих студій та персональних тренерів.

- **Головна сторінка** з героїчною графікою, виділенням занять та закликом до дії
- **Сторінка «Про нас»** зі історією та місією студії
- **Сторінка «Заняття»** з макетом розкладу
- **Сторінка «Тренери»** з картками профілів
- **Сторінка «Контакти»** з картою розташування та формою
- **Чистий, сучасний дизайн** з колірною гамою, відповідною фітнесу

### Шаблон 2: Gym Pro

Для великих, вже налагоджених залів з більшою кількістю послуг.

- Усе, що є у Studio Essential, плюс:
- **Сторінка «Членство»** з таблицею цін
- **Сторінка «Галерея»** для фотографій об'єктів
- **Блог** для порад із фітнесу та новин
- **Блок відгуків** на головній сторінці
- **Більше виразних** опцій брендування

### Шаблон 3: Fitness Chain

Для мереж з кількома локаціями.

- Усе, що є у Gym Pro, плюс:
- **Сторінка «Локації»** з кількома переліками об'єктів
- **Шаблон підсторінки «Франшиза/локація»**
- **Централізоване брендування** з деталями, специфічними для локації
- **Словник персоналу** для всіх локацій

## Створення сайту-шаблону

В Ultimate Multisite шаблон — це просто сайт WordPress, налаштований так, як ви хочете, щоб виглядали сайти нових клієнтів. Ось як його створити:

### Крок 1: Створення сайту-шаблону

1. Перейдіть до **Sites > Add New** у вашому мережевому адміні.
2. Створіть сайт під назвою `template-studio-essential`
3. Цей сайт стане вашим робочим полотном

### Крок 2: Встановлення та налаштування теми

Перейдіть до дашборду сайту-шаблону та:

1. Активуйте тему, придатну для фітнес-бізнесу
2. Налаштуйте параметри теми, кольори та типографіку
3. Налаштуйте шапку та підвал із навігацією, відповідною фітнесу

:::tip Theme Selection
Оберіть тему, яка достатньо гнучка, щоб добре виглядати для фітнесу, але не надто складна, щоб ваші клієнти не могли нею керувати. Теми, такі як Astra, GeneratePress або Kadence, добре працюють, оскільки вони легкі, налаштовувані та мають хорошу підтримку.
:::

### Крок 3: Створення сторінок

Створіть кожну сторінку, додавши:

- **Заповнювальний контент**, який читається природно для фітнесу («Ласкаво просимо до [Назва студії]», а не «Lorem ipsum»)
- **Заповнювальні зображення** з безкоштовних стокових фотосайтів, що демонструють фітнес-активності
- **Функціональні макети** за допомогою page builder або block editor

Намагайтеся зробити заповнювальний контент інструктивним, де це можливо. Замість загального філера тексту напишіть щось на кшталт: «Замініть це коротким описом вашої студії та того, що робить її особливою. Згадайте свою філософію тренувань, роки досвіду або що можуть очікувати клієнти».

### Крок 4: Налаштування плагінів

Встановіть та активуйте плагіни, які потрібні фітнес-студіям:

- Плагін для бронювання або розкладу (якщо це застосовно до вашого тарифного плану)
- Плагін для контактної форми
- SEO-плагін (попередньо налаштований з параметрами за замовчуванням, релевантними для фітнесу)

### Крок 5: Позначення як шаблону

1. Перейдіть до **Ultimate Multisite > Sites**
2. Відредагуйте сайт-шаблон
3. Увімкніть перемикач **Site Template**

Повторіть цей процес для кожного шаблону, який ви хочете запропонувати.

## Контрольний список якості шаблону

Перш ніж зробити шаблон доступним, перевірте:

- [ ] Усі сторінки завантажуються коректно та виглядають професійно
- [ ] Заповнювальний контент корисний та специфічний для ніші
- [ ] Зображення відповідні та належним чином ліцензовані
- [ ] Адаптивність для мобільних пристроїв працює на всіх сторінках
- [ ] Навігація логічна та повна
- [ ] Роботують контактні форми
- [ ] Немає зламаних посилань або відсутніх ресурсів
- [ ] Швидкість завантаження сторінки є прийнятною

## Мережа FitSite наразі

```
FitSite Network
├── WordPress Multisite (subdomain mode)
├── Ultimate Multisite (configured)
├── Hosting with wildcard SSL
├── Domain mapping configured
├── Site Templates
│   ├── Studio Essential (small studios, trainers)
│   ├── Gym Pro (established gyms)
│   └── Fitness Chain (multi-location)
└── Ready for plan configuration (next lesson)
```

- **Три нішево-специфічні шаблони**, розроблені для різних розмірів фітнес-бізнесу
- **Контент та зображення, відповідні фітнесу**, що створює відчуття, ніби платформа створена спеціально
- **Інструктивний заповнювальний контент**, який спрямовує клієнтів у процесі кастомізації
- **Контрольний список якості**, щоб гарантувати готовність шаблонів до використання
