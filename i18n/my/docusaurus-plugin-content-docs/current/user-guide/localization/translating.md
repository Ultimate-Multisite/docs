# Ultimate Multisite ကို ဘာသာပြန်ခြင်း

Ultimate Multisite ကို အင်္ဂလိပ်ဘာသာဖြင့် ဖန်တီးထားသော်လည်း plugin သည် ဘာသာပြန်နိုင်အောင် အပြည့်အဝ ပြင်ဆင်ထားပြီး ဘာသာပြန်ပံ့ပိုးကူညီမှုများကို ကျွန်ုပ်တို့ လှိုက်လှဲစွာ ကြိုဆိုပါသည်။

## Core Plugin ဘာသာပြန်ဆိုချက်များ

Core Ultimate Multisite plugin ကို **WordPress.org တရားဝင် ဘာသာပြန်ဆိုရေး platform** မှတစ်ဆင့် ဘာသာပြန်ဆိုထားပါသည်-

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

ဤသည်မှာ WordPress အသိုက်အဝန်း၏ စံသတ်မှတ် ဘာသာပြန်ဆိုရေးစနစ် ဖြစ်ပါသည်။ ဘာသာပြန်ဆိုချက်များသည် အတည်ပြုချက် အဆင့်သတ်မှတ်ချက်သို့ ရောက်ရှိသောအခါ WordPress plugin အပ်ဒိတ်များမှတစ်ဆင့် Ultimate Multisite အသုံးပြုသူအားလုံးထံ အလိုအလျောက် ဖြန့်ဝေပေးပါသည်။

**သင့်အကူအညီ လိုအပ်ပါသည်!** သင်သည် အင်္ဂလိပ်ဘာသာမဟုတ်သော အခြားဘာသာစကား ပြောဆိုနိုင်ပါက ဘာသာပြန်ဆိုချက်များ ပံ့ပိုးကူညီရန် စဉ်းစားပေးပါ။ စာကြောင်းအနည်းငယ် ဘာသာပြန်ရုံဖြင့်ပင် အထောက်အကူ ဖြစ်ပါသည်။ စတင်ရန်-

1. [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/) သို့ သွားပါ
2. သင့်ဘာသာစကားကို ရွေးချယ်ပါ
3. "Contribute Translation" ကို နှိပ်ပါ
4. အခမဲ့ WordPress.org အကောင့် တစ်ခု လိုအပ်ပါသည်

သင့်ဘာသာစကားအတွက် ဘာသာပြန်ဆိုရေး editor (ဘာသာပြန်ဆိုချက်များကို အတည်ပြုနိုင်သူ) ဖြစ်လိုပါက [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) မှတစ်ဆင့် ဆက်သွယ်ပါ။

## Addon ဘာသာပြန်ဆိုချက်များ

Addon plugin များကို အောက်ပါလိပ်စာတွင် သီးခြား ဘာသာပြန်ဆိုထားပါသည်-

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Addon ဘာသာပြန်ဆိုချက်များသည် လောလောဆယ် AI ဖြင့် ထုတ်လုပ်ထားသော အစပြုချက်အဆင့် ဖြစ်ပါသည်။ ပံ့ပိုးထားသော ဘာသာစကားအားလုံးကို လွှမ်းခြုံထားသော်လည်း မသင့်တော်သော စကားစုများ သို့မဟုတ် မမှန်ကန်မှုများ ပါဝင်နိုင်ပါသည်။ ပိုမိုသဘာဝကျပြီး တိကျမှန်ကန်အောင် ပြန်လည်သုံးသပ်ပြီး အကြံပြုချက်များ ပေးပို့မည့် လူသား ဘာသာပြန်ဆိုသူများကို ကြိုဆိုပါသည်။

## Local ဘာသာပြန်ဆိုချက်များ

Upstream သို့ ပံ့ပိုးကူညီမည့်အစား သင့်ကိုယ်ပိုင် ပရောဂျက်အတွက် ဘာသာပြန်ဆိုချက်များကို locally ဖန်တီးလိုပါက နည်းလမ်း နှစ်ခု ရှိပါသည်-

### Poedit အသုံးပြုခြင်း

`.pot` ဖိုင်ကို ဘာသာပြန်ဆိုရန် desktop application **[Poedit](https://poedit.net)** ကို အသုံးပြုနိုင်ပါသည်-

1. [poedit.net](https://poedit.net) မှ Poedit ကို ဒေါင်းလုဒ်လုပ်ပါ
2. `wp-content/plugins/ultimate-multisite/lang/` မှ `.pot` ဖိုင်ကို ဖွင့်ပါ
3. Poedit interface တွင် စာကြောင်းများကို ဘာသာပြန်ပါ
4. သိမ်းဆည်းပါ -- Poedit သည် `.po` နှင့် `.mo` ဖိုင်များကို ထုတ်ပေးပါသည်
5. Output ဖိုင်များကို `lang` folder တွင် ထားပါ

### Loco Translate အသုံးပြုခြင်း

**[Loco Translate](https://wordpress.org/plugins/loco-translate/)** WordPress plugin သည် desktop tool များ မလိုအပ်ဘဲ admin panel မှ တိုက်ရိုက် ဘာသာပြန်ဆိုနိုင်စေပါသည်။ ဘာသာပြန်ဆိုချက် ဖိုင်များကို ဖန်တီးခြင်းနှင့် compile လုပ်ခြင်းကို သင့်အတွက် ကိုင်တွယ်ဆောင်ရွက်ပေးပါသည်။
