# အကူအညီ ရယူနည်း

သင့်မှာ တရားဝင် License key ရှိပြီး ထို key နဲ့အတူ Ultimate Multisite စတင်အသုံးပြုစဉ် ကြုံတွေ့နိုင်တဲ့ ပြဿနာများကို ဖြေရှင်းပေးနိုင်ဖို့ ကျွန်ုပ်တို့အဖွဲ့ရဲ့ အကူအညီကို ရယူခွင့်ရှိပါတယ်။

ဒီသင်ခန်းစာမှာ license ပိုင်ရှင်များအတွက် ရရှိနိုင်တဲ့ support channel များနဲ့ ကျွန်ုပ်တို့ support အဖွဲ့ကို ဆက်သွယ်ရာမှာ လိုအပ်တဲ့ အချက်အလက်များအကြောင်း ရှင်းပြထားပါတယ်။

## Support ဆက်သွယ်ရန်

Help site မှာ အဖြေမတွေ့ဘဲ အခက်အခဲတွေ့နေရင် ကျွန်ုပ်တို့ website ရဲ့ ညာဘက်အောက်ခြေမှာရှိတဲ့ စကားပြောပူဖောင်း icon ကို နှိပ်ပြီး သို့မဟုတ် support@ultimatemultisite.com ကို တိုက်ရိုက် email ပို့ပြီး support အဖွဲ့ကို ဆက်သွယ်နိုင်ပါတယ်။

နည်းပညာဆိုင်ရာ ပြဿနာများအတွက် support ဆက်သွယ်တဲ့အခါ ကျေးဇူးပြု၍ သင့်ပြဿနာကို အသေးစိတ် ဖော်ပြပေးပါ။ screenshot များ၊ error message များ သို့မဟုတ် support အဖွဲ့က သင့်ပြဿနာကို လွယ်ကူစွာ နားလည်ပြီး အကဲဖြတ်နိုင်မယ့် အချက်အလက်များကိုလည်း ထည့်သွင်းပေးပါ။ သင့် license key ကိုလည်း ပေးရပါမယ်။ အသက်ဝင်နေတဲ့ license ပိုင်ရှင်များသာ support ခံစားခွင့်ရှိပါတယ်။
