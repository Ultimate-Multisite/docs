# Ultimate Multisite Community သို့ ဝင်ရောက်ပါ

Ultimate Multisite Community သည် အခြား Ultimate Multisite အသုံးပြုသူများနှင့် ဆက်သွယ်နိုင်ပြီး အကြံဉာဏ်များ၊ အတွေ့အကြုံများနှင့် အခြားအရာများကို မျှဝေနိုင်သော နေရာတစ်ခုဖြစ်ပါသည်။ Ultimate Multisite Community ကို ဝင်ရောက်ရန် https://community.ultimatemultisite.com/home သို့ သွားပါ။

ကြိုဆိုရေးစာမျက်နှာတွင် "**creating your account here**" လင့်ခ်ကို နှိပ်ပါ။

<!-- Screenshot unavailable: Community welcome page with creating your account link -->

ကျွန်ုပ်တို့၏ မှတ်ပုံတင်စာမျက်နှာသို့ ရောက်ရှိပါလိမ့်မည်။ ယခု username၊ သင့် email နှင့် password တစ်ခု ထည့်သွင်းရပါမည်။ Captcha ကို ဖြေပြီး **Register** ခလုတ်ကို နှိပ်ပါ။

<!-- Screenshot unavailable: Community register page with username, email, password fields and Captcha -->

မှတ်ပုံတင်ပြီးနောက် သင့်ပရိုဖိုင်ကို ဖန်တီးရပါမည်။ ဤစာမျက်နှာတွင် သင့်ဓာတ်ပုံကို တင်နိုင်သည်၊ သင့်အမည်အပြည့်အစုံ ထည့်နိုင်သည်၊ မိတ်ဆက်စာတိုလေးရေးနိုင်သည် စသည်ဖြင့် ပြုလုပ်နိုင်ပါသည်။ ထို့အပြင် သင့် social media လင့်ခ်များကို ထည့်သွင်းပြီး သင့်အကောင့်အတွက် ခွင့်ပြုချက်များကို သတ်မှတ်ပါ။

<!-- Screenshot unavailable: Community profile creation page with photo upload and bio fields -->

ယခု သင်သည် "Introductions" channel တွင် ရောက်ရှိနေပြီဖြစ်ပါသည်။ ဤနေရာတွင် သင့်ကိုယ်သင် မိတ်ဆက်နိုင်ပြီး အခြားအဖွဲ့ဝင်များအကြောင်း အနည်းငယ် သိရှိနိုင်ပါသည်။ **New Post** ခလုတ်ကို နှိပ်ပြီး သင့်ပို့စ်ကို ဖန်တီးနိုင်ပါသည်။

<!-- Screenshot unavailable: Community Introductions channel with New Post button -->

Lounge အပိုင်းရှိ channel များနှင့် "Dev Logs" channel တို့ကို လူတိုင်း ဝင်ကြည့်နိုင်ပါသည်။ သို့သော် community ရှိ အခြား channel များကို ဝင်ရောက်ရန် အထူးခွင့်ပြုချက် လိုအပ်ပါသည်။

<!-- Screenshot unavailable: Community channels showing Lounge section and Dev Logs channel -->

သင့်တွင် အသုံးပြုဆဲ Ultimate Multisite license key ရှိကြောင်း ပေးပို့ပြီးသည်နှင့် ဤဝင်ရောက်ခွင့်ကို ရရှိပါလိမ့်မည်။ ထိုသို့ပြုလုပ်ရန် မည်သည့် private channel လင့်ခ်ကိုမဆို နှိပ်ပြီး **Unlock** ခလုတ်ကို နှိပ်ပါ။

<!-- Screenshot unavailable: Community private channel with Unlock button -->

ကျွန်ုပ်တို့၏ unlock access စာမျက်နှာသို့ ရောက်ရှိပါလိမ့်မည်။ သင့် Ultimate Multisite license key ကို ထည့်သွင်းပြီး Unlock Access ခလုတ်ကို နှိပ်ပါ။

<!-- Screenshot unavailable: Community unlock access page with license key input -->

သင့် license ကို အတည်ပြုပြီး private channel များကို ဝင်ရောက်နိုင်ပါလိမ့်မည်။ ဤလုပ်ငန်းစဉ်သည် ၂၄ နာရီအထိ ကြာနိုင်ပါသည်။ ထို့ကြောင့် ကျွန်ုပ်တို့ သင့်တောင်းဆိုချက်ကို လုပ်ဆောင်နေစဉ် community သို့ ပြန်သွားရန် လင့်ခ်ကို နှိပ်ပြီး အခြားနေရာများကို စောင့်ဆိုင်းနေစဉ် အသုံးပြုနိုင်ပါသည်။
