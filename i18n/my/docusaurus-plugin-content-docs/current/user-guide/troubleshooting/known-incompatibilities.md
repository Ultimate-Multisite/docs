# အခြား Plugins များနှင့် လိုက်ဖက်မှုမရှိသော ပြဿနာများ

Ultimate Multisite ကို install လုပ်ပြီး activate လုပ်ပြီးနောက်၊ အခြား plugins များနှင့် လိုက်ဖက်မှုမရှိသော ပြဿနာများကြောင့် အခက်အခဲများ ကြုံတွေ့ရနိုင်ပါသည်။

ထုတ်ဝေမှုအသစ်တိုင်းတွင် ဤစာရင်းကို လျှော့ချနိုင်ရန် ကျွန်ုပ်တို့ တက်ကြွစွာ ကြိုးပမ်းလုပ်ဆောင်နေပါသည်။ သို့သော် plugin တစ်ခုသည် ဤစာရင်းတွင် ပါဝင်နေပါက၊ Ultimate Multisite Network အပြည့်အဝ အလုပ်လုပ်နိုင်ရန် ၎င်းကို deactivate လုပ်ရန် လိုအပ်နိုင်ပါသည်။

ဤပြဿနာများနှင့် ဖြေရှင်းနည်းများစာရင်းမှာ အောက်ပါအတိုင်းဖြစ်ပါသည်:

**WPMU DEV plugins အချို့** WPMU DEV မှ _ယခုအခါ ထိန်းသိမ်းမှုမပြုလုပ်တော့သော_ plugins အချို့သည် Ultimate Multisite registration လုပ်ငန်းစဉ်ကို နောက်ဆုံးအဆင့်တွင် ရပ်တန့်သွားစေတတ်ပါသည် (အသုံးပြုသူသည် account ဖန်တီးနိုင်ခြင်းမရှိတော့ပါ)။ ထိုပြဿနာများဖြစ်စေသော plugins များစာရင်းမှာ:

  * Signup Language: _Ultimate Multisite တွင် အလားတူလုပ်ဆောင်ချက်ပေးသော Ultimate Multisite: Language Switcher ဟုခေါ်သည့် add-on ရှိပါသည်။_

  * Set Blog Description on Blog Creation: _ဤလုပ်ဆောင်ချက်သည် Ultimate Multisite တွင် ပါဝင်ပြီးသားဖြစ်သောကြောင့် plugin အပို မလိုအပ်ပါ။_

  * New Blog Templates: _ဤလုပ်ဆောင်ချက်သည် Ultimate Multisite တွင် ပါဝင်ပြီးသားဖြစ်သောကြောင့် plugin အပို မလိုအပ်ပါ။_

  * Set Password on Multisite Blog Creation: _ဤလုပ်ဆောင်ချက်သည် Ultimate Multisite တွင် ပါဝင်ပြီးသားဖြစ်သောကြောင့် plugin အပို မလိုအပ်ပါ။_

  * Signup TOS: _ဤလုပ်ဆောင်ချက်သည် Ultimate Multisite တွင် ပါဝင်ပြီးသားဖြစ်သောကြောင့် plugin အပို မလိုအပ်ပါ။_

  * Anti-Splog: အင်အားနည်းသော်လည်း၊ ဤလုပ်ဆောင်ချက်သည်လည်း _Ultimate Multisite တွင် ပါဝင်ပြီးသားဖြစ်ပါသည်။_

**မှတ်ချက်** : အခြား WPMU Dev plugins များအားလုံးကို Ultimate Multisite နှင့်အတူ ပုံမှန်အတိုင်း အသုံးပြုနိုင်ပါသည်။ ဥပမာများအနေဖြင့် _Smush_ ၊ _Forminator_ ၊ _Defender_ စသည်တို့ ပါဝင်ပါသည်။
