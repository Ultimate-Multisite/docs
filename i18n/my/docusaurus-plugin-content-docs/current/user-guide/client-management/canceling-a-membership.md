# Membership ဖျက်သိမ်းခြင်း (v2)

_**အရေးကြီးသော မှတ်ချက်- ဤဆောင်းပါးသည် Ultimate Multisite version 2.x အတွက် ဖြစ်ပါသည်။**_

သင့်ဖောက်သည်များဘက်မှ membership ဖျက်သိမ်းခြင်းသည် အလွန်လွယ်ကူပါသည်။

သင့်အသုံးပြုသူက membership ဖျက်သိမ်းလိုပါက ၎င်းတို့၏ **Account** စာမျက်နှာကို ဦးစွာသွားရောက်ရပါမည်။

ထိုနေရာတွင် **Danger Zone** သို့သွားပြီး **Delete Account** ကို နှိပ်ရပါမည်။

![Delete Account ခလုတ်ပါသော Account စာမျက်နှာ Danger Zone](/img/admin/memberships-list.png)

အတည်ပြုချက်တောင်းသည့် dialog တစ်ခု ပွင့်လာပါမည်။ ၎င်းတို့သည် **Confirm Account Deletion** ရွေးချယ်မှုကို ဖွင့်ပြီး **Delete Account** ကို နှိပ်ရပါမည်။

![Account ဖျက်ခြင်း အတည်ပြုချက် dialog](/img/admin/memberships-list.png)

ဤလုပ်ဆောင်ချက်သည် ၎င်းတို့ကို ဆိုက်မှ logout လုပ်ပြီး သင့် network ပေါ်ရှိ membership ကို ဖျက်သိမ်းကာ ၎င်းတို့၏ site ကိုလည်း သင့် network မှ ဖျက်ပစ်ပါမည်။

![Account ဖျက်ပြီးနောက် logout ဖြစ်သွားခြင်း](/img/admin/memberships-list.png)

![Membership ဖျက်သိမ်းပြီး အတည်ပြုချက်](/img/admin/memberships-list.png)

**အရေးကြီးသည်-** ဤလုပ်ဆောင်ချက်သည် ဖောက်သည်ကို သင့် network မှ ဖယ်ရှားမည်မဟုတ်ကြောင်း သတိပြုပါ။ ဖောက်သည်တစ်ဦးတွင် membership တစ်ခုထက်ပိုပြီး ရှိနိုင်သလို membership လုံးဝမရှိဘဲလည်း ရှိနိုင်သောကြောင့် ဖြစ်ပါသည်။

![ဖျက်သိမ်းပြီးနောက် ဖောက်သည် network တွင် ရှိနေဆဲ](/img/admin/memberships-list.png) | ![အသက်ဝင်သော membership မရှိသည့် ဖောက်သည်](/img/admin/memberships-list.png)
