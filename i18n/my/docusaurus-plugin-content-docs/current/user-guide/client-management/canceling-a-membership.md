# အသင်းဝင်မှု ဖျက်သိမ်းခြင်း (v2) {#canceling-a-membership-v2}

_**အရေးကြီး မှတ်ချက်: ဤဆောင်းပါးသည် Ultimate Multisite ဗားရှင်း 2.x အတွက် ဖြစ်သည်။**_

သင့်ဖောက်သည်ဘက်မှ အသင်းဝင်မှု ဖျက်သိမ်းခြင်းမှာ အလွန်လွယ်ကူပါသည်။

သင့်အသုံးပြုသူသည် သူတို့၏ အသင်းဝင်မှုကို ဖျက်သိမ်းလိုပါက ပထမဆုံး သူတို့၏ ****Account**** စာမျက်နှာသို့ ဝင်ရပါမည်။

ထိုနေရာတွင် **အန္တရာယ်ဇုန်** သို့ သွားပြီး **Account ဖျက်ရန်** ကို နှိပ်ရပါမည်။

![Account စာမျက်နှာရှိ Account ဖျက်ရန် ခလုတ်ပါသော အန္တရာယ်ဇုန်](/img/account-page/danger-zone.png)

သူတို့၏ အတည်ပြုချက်ကို တောင်းသည့် မော်ဂျူးတစ်ခု ပွင့်လာပါမည်။ သူတို့သည် **Account ဖျက်မှု အတည်ပြုရန်** ရွေးချယ်မှုကို ဖွင့်ပြီး **Account ဖျက်ရန်** ကို နှိပ်ရပါမည်။

![Account ဖျက်မှု အတည်ပြုရန် ဒိုင်ယာလော့ဂ်](/img/account-page/delete-account-confirm.png)

ဤလုပ်ဆောင်မှုက သူတို့ကို site မှ ထွက်စေပြီး သင့်ကွန်ရက်ပေါ်ရှိ သူတို့၏ အသင်းဝင်မှုကို ဖျက်သိမ်းမည်ဖြစ်သည့်အပြင် သင့်ကွန်ရက်မှ သူတို့၏ site ကိုလည်း ဖျက်ပစ်ပါမည်။

![Account ဖျက်ပြီးနောက် ထွက်သွားပြီးဖြစ်သည်](/img/account-page/logged-out-after-delete.png)

![အသင်းဝင်မှု ဖျက်သိမ်းပြီးကြောင်း အတည်ပြုချက်](/img/account-page/membership-canceled.png)

**အရေးကြီး:** ဤလုပ်ဆောင်မှုသည် ဖောက်သည်ကို သင့်ကွန်ရက်မှ ထုတ်ပယ်မည်မဟုတ်ကြောင်း သတိပြုပါ။ အကြောင်းမှာ ဖောက်သည်တစ်ဦးတွင် သူတို့နှင့် ချိတ်ဆက်ထားသော အသင်းဝင်မှု တစ်ခုထက်ပို၍ ရှိနိုင်သလို အသင်းဝင်မှု မရှိနိုင်လည်း ဖြစ်သောကြောင့် ဖြစ်သည်။

![ဖျက်သိမ်းပြီးနောက် ဖောက်သည်သည် ကွန်ရက်တွင် ဆက်ရှိနေသည်](/img/admin/customer-after-cancel.png)

![လက်ရှိ အသက်ဝင်သော အသင်းဝင်မှု မရှိသည့် ဖောက်သည်](/img/admin/memberships-customer-empty.png)
