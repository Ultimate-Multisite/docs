# ကျွန်ုပ်၏ subscription ကို ဘယ်လို ပယ်ဖျက်နိုင်မလဲ?

Ultimate Multisite license ရှိပါက သင်အလိုရှိသည့်အချိန်တိုင်း renewal ကို ပယ်ဖျက်နိုင်ပါသည်။ အောက်ပါအဆင့်များကို လိုက်နာပါ:


<!-- Screenshot unavailable: Account page email with access link -->

ဝယ်ယူစဉ်က ပေးခဲ့သော email နှင့် password ကို အသုံးပြုပါ။

<!-- Screenshot unavailable: Login page with email and password fields -->

ဤသည်မှာ သင်၏ Account Page ပထမဆုံးမျက်နှာပြင်ဖြစ်ပါသည်:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

Subscription ပယ်ဖျက်ရန် ဘယ်ဘက်မီနူးတွင် _Renewing & Billing_ သို့ သွားပါ။

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

ဘေးဘက်ဝင်းဒိုးဖွင့်ရန် ညာဘက်ရှိ မြှားကို နှိပ်ပါ။ ထို့နောက် **auto-renew** ပယ်ဖျက်မည့် ရွေးချယ်မှုကို ရွေးပါ။

စနစ်မှ အတည်ပြုချက် မက်ဆေ့ချ် ပြသပါလိမ့်မည်။

Subscription ပယ်ဖျက်လိုကြောင်း သေချာပါက _**Cancel Renewals**_ ခလုတ်ကို နှိပ်ပါ။ ဤလုပ်ဆောင်ချက်ပြီးနောက် စစ်တမ်းတိုတစ်ခု ဖြေဆိုရန် တောင်းဆိုပါလိမ့်မည်။

ပြီးပါပြီ! သင်၏ subscription အလိုအလျောက် သက်တမ်းတိုးမည် မဟုတ်တော့ပါ။

**Subscription သက်တမ်းကုန်ဆုံးသည်အထိ သင့်တွင် အသုံးပြုနိုင်သော key ရှိနေပါမည်**။ Subscription ပြန်လည်အသက်သွင်းလိုပါက ကိုယ်တိုင်လုပ်ဆောင်ရပါမည်။
