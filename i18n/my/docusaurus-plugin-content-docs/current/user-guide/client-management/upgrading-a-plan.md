# အစီအစဉ် အဆင့်မြှင့်တင်ခြင်း (v2)

_**အရေးကြီးသော မှတ်ချက်: ဤဆောင်းပါးသည် Ultimate Multisite version 2.x ကို ရည်ညွှန်းပါသည်။**_

သင့်ဖောက်သည်များသည် ၎င်းတို့၏ အစီအစဉ်ကို အချိန်မရွေး အဆင့်မြှင့်တင်နိုင်ပါသည်။ အခြားအစီအစဉ်တစ်ခုသို့ အဆင့်မြှင့်တင်နိုင်သလို သင့် network တွင် ပေးအပ်ထားသော အပိုဝန်ဆောင်မှုများ သို့မဟုတ် package များကိုလည်း ဝယ်ယူနိုင်ပါသည်။

ဤ tutorial တွင် ၎င်းတို့၏ အစီအစဉ်ကို မည်သို့အဆင့်မြှင့်တင်နိုင်သည်နှင့် အဆင့်မြှင့်တင်ပြီးနောက် ဘာများဖြစ်ပျက်သည်ကို ရှင်းပြပေးပါမည်။

အစီအစဉ်ကို အဆင့်မြှင့်တင်ရန်၊ သင့်ဖောက်သည်များသည် ၎င်းတို့၏ dashboard သို့ ဝင်ရောက်ပြီး **Account** စာမျက်နှာသို့ သွားရပါမည်။

![Customer dashboard with Account page link](/img/admin/memberships-list.png)

Account စာမျက်နှာတွင် ၎င်းတို့၏ လက်ရှိအသင်းဝင်မှုနှင့် ၎င်းနှင့်ဆက်စပ်နေသော အစီအစဉ်ကို ပြသပေးပါမည်။ အခြားအစီအစဉ်သို့ အဆင့်မြှင့်တင်ရန်၊ **Your Membership** အပိုင်း၏ ညာဘက်အပေါ်ထောင့်ရှိ **Change** ကို နှိပ်ရပါမည်။

![Your Membership section with Change button](/img/admin/memberships-list.png)

ရရှိနိုင်သော အစီအစဉ်အားလုံးကို ပြသထားသည့် checkout form သို့ ပြန်လည်ညွှန်းပေးပါမည်။

အစီအစဉ်ကို အဆင့်မြှင့်တင်မည်မဟုတ်ဘဲ သီးခြား ဝန်ဆောင်မှု သို့မဟုတ် package တစ်ခုခု (ဥပမာ - ဤနေရာတွင် ပြထားသည့်အတိုင်း unlimited visits သို့မဟုတ် disk space) ကိုသာ ဝယ်ယူလိုပါက ၎င်းတို့၏ **လက်ရှိအစီအစဉ်အတွက် ရရှိနိုင်သော ဝန်ဆောင်မှုများနှင့် package များ** ကိုလည်း မြင်တွေ့နိုင်ပါသည်။

![Checkout form showing available plans and packages](/img/admin/memberships-list.png)

ဝယ်ယူလိုသော ထုတ်ကုန်ကို ရွေးချယ်ပြီးနောက်၊ ရှိပြီးသား credit ကို ဖယ်ထုတ်ပြီး ယခုအချိန်တွင် မည်မျှပေးဆောင်ရမည်နှင့် နောက်ငွေတောင်းခံရက်တွင် မည်မျှ ကောက်ခံမည်ကို မြင်တွေ့ရပါမည်။

ပုံမှန်အားဖြင့်၊ ထုတ်ကုန်သည် အခြားအစီအစဉ်ဖြစ်ပြီး အသင်းဝင်ကြေး ကောက်ခံမှုကြားကာလတွင် ငွေပေးချေမှုပြုလုပ်ပါက၊ ပထမအစီအစဉ်အတွက် ပေးဆောင်ခဲ့သော ပမာဏအတွက် credit ပြန်ရရှိပါမည်။

![Upgrade payment summary with credit and next billing amount](/img/admin/memberships-list.png)

လက်ရှိ subscription မှ ဘာမှမပြောင်းလဲမည့် အစီအစဉ် သို့မဟုတ် package ကို ရွေးချယ်ပါက၊ ထိုအကြောင်းကို ရှင်းပြသည့် မက်ဆေ့ချ်ကို မြင်တွေ့ရပါမည်။

![Message when selected plan does not change the subscription](/img/admin/memberships-list.png)

Checkout ပြီးဆုံးသွားသောအခါ၊ ထုတ်ကုန်အသစ်(များ)ကို သင့်ဖောက်သည်များ၏ အကောင့်တွင် ထည့်သွင်းပေးမည်ဖြစ်ပြီး ထုတ်ကုန်အသစ်(များ)၏ ကန့်သတ်ချက်များ သို့မဟုတ် အင်္ဂါရပ်အားလုံးကို ချက်ချင်းထည့်သွင်းပေးပါမည်: visits, disk space, posts စသည်တို့...

## အဆင့်မြှင့်တင်ခြင်းနှင့် အဆင့်လျှော့ချခြင်း လမ်းကြောင်းများ

သင့်ထုတ်ကုန်တစ်ခုစီတွင် **Up & Downgrades** tab တစ်ခုရှိပါမည်။ ထို tab ရှိ ပထမဆုံးရွေးချယ်စရာမှာ **Plan Group** ဟုခေါ်သော field ဖြစ်ပါသည်။

**Plan groups** သည် အချို့သောအစီအစဉ်များသည် တူညီသော "မိသားစု" တွင် ပါဝင်ကြောင်း Ultimate Multisite အား သိစေနိုင်ပြီး၊ ထို့ကြောင့် အဆင့်မြှင့်တင်ခြင်း/အဆင့်လျှော့ချခြင်း လမ်းကြောင်းရွေးချယ်စရာများကို တည်ဆောက်ရာတွင် အသုံးပြုသင့်ပါသည်။

![Up and Downgrades tab with Plan Group field](/img/config/product-upgrades.png)

ဥပမာ၊ **Free plan**၊ **Basic Plan** နှင့် **Premium Plan** ရရှိနိုင်ပါသည်။ **Free Plan** အောက်တွင် စာရင်းသွင်းထားသော အသုံးပြုသူများသည် **Premium Plan** သို့သာ အဆင့်မြှင့်တင်နိုင်စေလိုပြီး "Basic Plan" ကို အဆင့်မြှင့်တင်ရန် ရွေးချယ်စရာအဖြစ် မမြင်စေလိုပါ။ အောက်ပါ screenshot များတွင် ပြထားသည့်အတိုင်း Free နှင့် Premium အစီအစဉ်နှစ်ခုစလုံးအတွက် တူညီသော plan group အမည်ကို သတ်မှတ်ပေးရုံပါပဲ။

![Free Plan with High End plan group assigned](/img/config/product-upgrades.png)

![Premium Plan with High End plan group assigned](/img/config/product-upgrades.png)

ဤသို့လုပ်ဆောင်ခြင်းဖြင့် Ultimate Multisite အား network တွင် **High End** ဟုခေါ်သော အစီအစဉ် "မိသားစု" တစ်ခုရှိကြောင်း အသိပေးပါသည်။ အဆင့်မြှင့်တင်ခြင်း သို့မဟုတ် အဆင့်လျှော့ချခြင်းများ ပေးအပ်ရာတွင်၊ တူညီသောမိသားစုမှ အစီအစဉ်များကိုသာ အသုံးပြုသူအတွက် ရွေးချယ်စရာအဖြစ် တင်ပြပေးပါမည်။
