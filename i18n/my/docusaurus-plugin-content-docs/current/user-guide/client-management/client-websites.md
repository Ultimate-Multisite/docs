# သင့်ဖောက်သည်၏ ဝဘ်ဆိုက်များ (v2) {#your-clients-websites-v2}

သင့် network တွင် စာရင်းသွင်းပြီးနောက် သင့်ဖောက်သည်များသည် သူတို့၏ Dashboard ကို ဝင်ရောက်အသုံးပြုနိုင်သည်။

သူတို့၏ ငွေပေးချေမှုများ၊ membership များ၊ domain များ၊ plan ကန့်သတ်ချက်များနှင့် အခြားအရေးကြီးသော အချက်အလက်များကို ကြည့်နိုင်သည့် **Account** စာမျက်နှာအပြင်၊ သူတို့၏ ဝဘ်ဆိုက်အားလုံးကို ကြည့်ရှုစီမံနိုင်သည့် **ကျွန်ုပ်၏ ဝဘ်ဆိုက်များ** စာမျက်နှာကိုလည်း ဝင်ရောက်အသုံးပြုနိုင်သည်။ လုပ်ရမည့်အရာမှာ သူတို့၏ Dashboard ပေါ်ရှိ ကျွန်ုပ်၏ ဝဘ်ဆိုက်များကို နှိပ်ရုံသာဖြစ်သည်။

![ဖောက်သည် Dashboard ရှိ ကျွန်ုပ်၏ ဝဘ်ဆိုက်များ link](/img/account-page/my-sites-link.png)

သူတို့၏ ဝဘ်ဆိုက်အားလုံးကို ပြသမည်ဖြစ်ပြီး တစ်ခုချင်းစီ၏ အောက်တွင် **စီမံရန်** button တစ်ခု ပါရှိမည်။ ၎င်းကို နှိပ်ပါက **ဤသတ်မှတ်ထားသော ဝဘ်ဆိုက်၏ Dashboard** သို့ ပြန်ညွှန်ပြပေးမည်ဖြစ်သည်။

ဝဘ်ဆိုက်တစ်ခုထက်ပိုပြီး ရှိခွင့်ပေးသော plan များတွင် မည်သည့် ဝဘ်ဆိုက်သည် **ပင်မ** ဝဘ်ဆိုက်ဖြစ်သည်ကိုလည်း သူတို့မြင်နိုင်သည်။

![စီမံရန် button များပါသော ဖောက်သည် ဝဘ်ဆိုက်များကို ပြသသည့် ကျွန်ုပ်၏ ဝဘ်ဆိုက်များ စာမျက်နှာ](/img/account-page/my-sites-page.png)

ဤသည်မှာ ဝဘ်ဆိုက် ပြင်ဆင်ရေး interface အပြည့်အစုံ၏ မြင်ကွင်းဖြစ်သည်။

![ဝဘ်ဆိုက် ပြင်ဆင်ရေး interface အပြည့်အစုံ](/img/admin/site-edit-full.png)

သူတို့၏ ပင်မ ဝဘ်ဆိုက်ကို ပြောင်းလိုပါက **Account** menu ကို နှိပ်ပြီး Account စာမျက်နှာသို့ သွားရမည်။

![ဖောက်သည် Dashboard ရှိ Account menu link](/img/account-page/account-menu.png)

ဖောက်သည်အသေးစိတ်ကိုလည်း ကြည့်ရှုပြင်ဆင်နိုင်သည်။

![ဖောက်သည် ပြင်ဆင်ရေး interface](/img/admin/customer-edit.png)

ထို့နောက် **မူလ ဝဘ်ဆိုက် ပြောင်းရန်** ကို နှိပ်ရမည်။

![Account စာမျက်နှာရှိ မူလ ဝဘ်ဆိုက် ပြောင်းရန် button](/img/account-page/change-default-site.png)

pop-up window တစ်ခုပေါ်လာမည်။ drop-down list ကို နှိပ်ပြီး ပင်မ ဝဘ်ဆိုက်အဖြစ် သတ်မှတ်လိုသော ဝဘ်ဆိုက်ကို ရွေးချယ်ပါ။

![မူလ ဝဘ်ဆိုက် ပြောင်းရန် dropdown ရွေးချယ်မှု](/img/account-page/change-default-site-dropdown.png)

ထို့နောက် **မူလ ဝဘ်ဆိုက် ပြောင်းရန်** ကို နှိပ်လိုက်ရုံဖြင့် ပြီးပါပြီ။

![မူလ ဝဘ်ဆိုက် ပြောင်းလဲမှုကို အတည်ပြုခြင်း](/img/account-page/change-default-site-confirm.png)
