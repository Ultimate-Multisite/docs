# သင့်ဖောက်သည်များ၏ ဝဘ်ဆိုက်များ (v2)

သင့် network တွင် စာရင်းသွင်းပြီးနောက်၊ သင့်ဖောက်သည်များသည် သူတို့၏ dashboard ကို ဝင်ရောက်ကြည့်ရှုနိုင်ပါသည်။

သူတို့၏ ငွေပေးချေမှုများ၊ အသင်းဝင်မှုများ၊ domains များ၊ plan ကန့်သတ်ချက်များနှင့် အခြားအရေးကြီးသော အချက်အလက်များကို ကြည့်ရှုနိုင်သည့် **Account** စာမျက်နှာအပြင်၊ သူတို့၏ ဝဘ်ဆိုက်အားလုံးကို ကြည့်ရှုစီမံခန့်ခွဲနိုင်သည့် **My Sites** စာမျက်နှာကိုလည်း ဝင်ရောက်နိုင်ပါသည်။ သူတို့ dashboard ပေါ်ရှိ My Sites ကို နှိပ်ရုံသာ လိုပါသည်။

![ဖောက်သည် dashboard ရှိ My Sites လင့်ခ်](/img/admin/customers-list.png)

သူတို့၏ ဝဘ်ဆိုက်အားလုံးကို ပြသမည်ဖြစ်ပြီး၊ တစ်ခုချင်းစီ၏ အောက်တွင် **Manage** ခလုတ် ပါရှိပါသည်။ ၎င်းကို နှိပ်လိုက်ပါက၊ **ထိုဝဘ်ဆိုက်၏ dashboard** သို့ ရောက်သွားပါမည်။

ဝဘ်ဆိုက်တစ်ခုထက်ပိုပိုင်ဆိုင်ခွင့်ရှိသော plans များအတွက် မည်သည့်ဝဘ်ဆိုက်က **primary** ဖြစ်သည်ကိုလည်း သူတို့ မြင်နိုင်ပါသည်။

![ဖောက်သည်ဝဘ်ဆိုက်များနှင့် Manage ခလုတ်များပြသသော My Sites စာမျက်နှာ](/img/admin/customers-list.png)

Primary ဝဘ်ဆိုက်ကို ပြောင်းလဲလိုပါက၊ **Account** menu ကို နှိပ်၍ Account စာမျက်နှာသို့ သွားရပါမည်။

![ဖောက်သည် dashboard ရှိ Account menu လင့်ခ်](/img/admin/customers-list.png)

ထို့နောက်၊ **Change Default Site** ကို နှိပ်ပါ။

![Account စာမျက်နှာရှိ Change Default Site ခလုတ်](/img/admin/customers-list.png)

Pop-up ဝင်းဒိုး တစ်ခု ပေါ်လာပါမည်။ Drop-down list ကို နှိပ်ပြီး primary ဝဘ်ဆိုက်အဖြစ် သတ်မှတ်လိုသော ဝဘ်ဆိုက်ကို ရွေးချယ်ပါ။

![Change Default Site dropdown ရွေးချယ်မှု](/img/admin/customers-list.png)

ထို့နောက်၊ **Change Default Site** ကို နှိပ်ရုံဖြင့် ပြီးဆုံးပါပြီ။

![Default site ပြောင်းလဲမှုကို အတည်ပြုခြင်း](/img/admin/customers-list.png)
