# ကျွန်ုပ်၏ အကောင့်စာမျက်နှာကို ပထမဆုံးအကြိမ် ဘယ်လို ဝင်ရောက်နိုင်မလဲ။

Ultimate Multisite မှ ကြိုဆိုပါတယ်!

ကျွန်ုပ်တို့ plugin ကို ဝယ်ယူပြီးနောက် သင့်အီးမေးလ်သို့ ရရှိခဲ့သော လင့်ခ်မှတစ်ဆင့် သင့်အကောင့်စာမျက်နှာကို ဝင်ရောက်နိုင်ပါတယ်။

<!-- Screenshot unavailable: Account page email with access link -->

ဝယ်ယူစဉ်က သင်ပေးခဲ့သော အီးမေးလ်နှင့် သင့်အီးမေးလ်တွင် ရရှိခဲ့သော စကားဝှက်ကို အသုံးပြုပါ။

<!-- Screenshot unavailable: Login page with email and password fields -->

ဒီအကောင့်ကို ပထမဆုံးအကြိမ် ဝင်ရောက်ခြင်းဖြစ်သောကြောင့်၊ သင့်စကားဝှက်ကို ပြောင်းလဲရန် တောင်းဆိုခံရပါလိမ့်မယ်။ စကားဝှက်ကို ပြောင်းလဲပြီး _Save_ ခလုတ်ကို နှိပ်ပါ။

<!-- Screenshot unavailable: Change password form with Save button -->

သင့်အကောင့်စာမျက်နှာတွင် ငွေတောင်းခံခြင်း၊ သက်တမ်းတိုးခြင်း၊ ပယ်ဖျက်ခြင်းနှင့် အခြားအရာများစွာကဲ့သို့သော သင့်အကောင့်အသေးစိတ်အားလုံးကို စီမံခန့်ခွဲနိုင်ပါတယ်။
