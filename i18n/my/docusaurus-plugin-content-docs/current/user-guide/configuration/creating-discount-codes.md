# လျှော့စျေး ကုဒ်များ ဖန်တီးခြင်း (v2)

_**အရေးကြီးသော မှတ်ချက်: ဤဆောင်းပါးသည် Ultimate Multisite version 2.x အတွက် ဖြစ်ပါသည်။**_

Ultimate Multisite ဖြင့် သင့်ဖောက်သည်များအတွက် subscription များတွင် လျှော့စျေးပေးနိုင်ရန် လျှော့စျေး ကုဒ်များ ဖန်တီးနိုင်ပါသည်။ ဖန်တီးရန်လည်း လွယ်ကူပါသည်!

## လျှော့စျေး ကုဒ်များ ဖန်တီးခြင်းနှင့် ပြင်ဆင်ခြင်း

လျှော့စျေး ကုဒ် ဖန်တီးရန် သို့မဟုတ် ပြင်ဆင်ရန်၊ **Ultimate Multisite > Discount Codes** သို့ သွားပါ။

![](/img/config/discount-codes-list.png)

ထိုနေရာတွင် သင်ဖန်တီးပြီးသား လျှော့စျေး ကုဒ်များ စာရင်းကို တွေ့ရမည်ဖြစ်သည်။

coupon အသစ်တစ်ခု ဖန်တီးရန် **Add Discount** **Code** ကို နှိပ်နိုင်ပြီး၊ ရှိပြီးသားများကို ပြင်ဆင်ရန် ၎င်းတို့ပေါ်တွင် မောက်စ်တင်ပြီး **Edit** ကို နှိပ်ပါ။

![Hover actions ပါသော လျှော့စျေး ကုဒ်များ စာရင်း](/img/config/discount-codes-list.png)

![Add Discount Code ခလုတ်](/img/config/discount-codes-list.png)

coupon ကုဒ် ဖန်တီးရန် သို့မဟုတ် ပြင်ဆင်ရန် စာမျက်နှာသို့ ရောက်သွားမည်ဖြစ်သည်။ ဤဥပမာတွင် အသစ်တစ်ခု ဖန်တီးပါမည်။

![လျှော့စျေး ကုဒ် ပြင်ဆင်ရန် စာမျက်နှာ](/img/config/discount-codes-list.png)

ဤနေရာတွင် ရရှိနိုင်သော setting များကို ကြည့်ကြပါစို့:

**Enter Discount Code:** ဤသည်မှာ သင့်လျှော့စျေး ကုဒ်၏ အမည်သာ ဖြစ်ပါသည်။ ဤသည်မှာ သင့်ဖောက်သည်များ checkout form တွင် အသုံးပြုရမည့် ကုဒ် မဟုတ်ပါ။

**Description:** ဤနေရာတွင် ဤ coupon သည် မည်သည့်အတွက် ဖြစ်ကြောင်း အကျဉ်းချုပ် ဖော်ပြနိုင်ပါသည်။

![လျှော့စျေး ကုဒ် အမည်နှင့် ဖော်ပြချက် field များ](/img/config/discount-codes-list.png)

**Coupon code:** ဤနေရာတွင် checkout လုပ်စဉ် သင့်ဖောက်သည်များ ထည့်သွင်းရမည့် ကုဒ်ကို သတ်မှတ်ပါသည်။

![Coupon code field](/img/config/discount-codes-list.png)

**Discount:** ဤနေရာတွင် သင့်လျှော့စျေး ကုဒ်အတွက် **ရာခိုင်နှုန်း** သို့မဟုတ် **ပုံသေ ငွေပမာဏ** သတ်မှတ်နိုင်ပါသည်။

![လျှော့စျေး ရာခိုင်နှုန်း သို့မဟုတ် ပုံသေပမာဏ setting](/img/config/discount-codes-list.png)

**Apply to renewals:** ဤ option ကို ပိတ်ထားပါက၊ ဤလျှော့စျေး ကုဒ်သည် **ပထမဆုံး ငွေပေးချေမှု** အတွက်သာ အသုံးဝင်မည်ဖြစ်သည်။ အခြား ငွေပေးချေမှုများတွင် လျှော့စျေး ရမည်မဟုတ်ပါ။ ဤ option ကို ဖွင့်ထားပါက၊ လျှော့စျေး ကုဒ်သည် အနာဂတ် ငွေပေးချေမှုအားလုံးအတွက် အကျုံးဝင်မည်ဖြစ်သည်။

**Setup fee discount:** ဤ option ကို ပိတ်ထားပါက၊ coupon ကုဒ်သည် order ၏ **setup fee အတွက် မည်သည့် လျှော့စျေးမှ ပေးမည်မဟုတ်ပါ**။ ဤ option ကို ဖွင့်ထားပါက၊ ဤ coupon ကုဒ်သည် သင့် plan များ၏ setup fee အတွက် သက်ရောက်မည့် လျှော့စျေး (ရာခိုင်နှုန်း သို့မဟုတ် ပုံသေပမာဏ) သတ်မှတ်နိုင်ပါသည်။

![Apply to renewals နှင့် setup fee discount options](/img/config/discount-codes-list.png)

**Active:** ဤ coupon ကုဒ်ကို ကိုယ်တိုင် activate သို့မဟုတ် deactivate လုပ်ပါ။

![လျှော့စျေး ကုဒ်အတွက် Active toggle](/img/config/discount-codes-list.png)

**Advanced Optio** **ns** အောက်တွင် အောက်ပါ setting များ ရှိပါသည်:

**Limit uses:**

  * **Uses:** ဤနေရာတွင် လျှော့စျေး ကုဒ်ကို အကြိမ်မည်မျှ အသုံးပြုခဲ့သည်ကို မြင်ရနိုင်ပါသည်။

  * **Max uses:** အသုံးပြုသူများ ဤလျှော့စျေး ကုဒ်ကို အသုံးပြုနိုင်သည့် အကြိမ်ရေကို ကန့်သတ်ပါသည်။ ဥပမာ၊ ဤနေရာတွင် 10 ထည့်ပါက၊ coupon ကို 10 ကြိမ်သာ အသုံးပြုနိုင်မည်ဖြစ်သည်။ ဤကန့်သတ်ချက်ပြည့်ပြီးနောက်၊ coupon ကုဒ်ကို ထပ်မံ အသုံးပြု၍ မရတော့ပါ။

![uses နှင့် max uses fields ပါသော Limit uses setting](/img/config/discount-codes-list.png)**Start & expiration dates:** ဤနေရာတွင် သင့် coupon အတွက် စတင်ရက်စွဲ နှင့်/သို့မဟုတ် သက်တမ်းကုန်ရက်စွဲ ထည့်သွင်းရန် ရွေးချယ်ခွင့် ရှိပါသည်။

![စတင်ရက်စွဲနှင့် သက်တမ်းကုန်ရက်စွဲ fields](/img/config/discount-codes-list.png)

**Limit products:** **Select products** ကို ဖွင့်ထားပါက၊ သင့် product များအားလုံး ပြသမည်ဖြစ်သည်။ မည်သည့် product က ဤ coupon ကုဒ်ကို လက်ခံနိုင်သည်ကို (ဖွင့်/ပိတ် ဖြင့်) ကိုယ်တိုင် ရွေးချယ်နိုင်ပါသည်။ ဤနေရာတွင် ပိတ်ထားသော product များတွင် သင့်ဖောက်သည်များ ဤ coupon ကုဒ်ကို အသုံးပြုရန် ကြိုးစားပါက မည်သည့် ပြောင်းလဲမှုမှ ပြသမည်မဟုတ်ပါ။

![Limit products ရွေးချယ်ရန် toggles](/img/config/discount-codes-list.png)

ဤ option များအားလုံး သတ်မှတ်ပြီးနောက်၊ သင့် coupon ကို သိမ်းဆည်းရန် **Save Discount Code** ကို နှိပ်ပါ၊ ပြီးပါပြီ!

![Save Discount Code ခလုတ်](/img/config/discount-codes-list.png)

coupon သည် ယခု သင့်စာရင်းထဲတွင် ရှိပြီး၊ ထိုနေရာမှ **edit သို့မဟုတ် delete** လုပ်ရန် နှိပ်နိုင်ပါသည်။

![edit နှင့် delete options ပါသော စာရင်းထဲရှိ လျှော့စျေး ကုဒ်](/img/config/discount-codes-list.png)

### URL Parameters အသုံးပြုခြင်း:

သင့် pricing table များကို စိတ်ကြိုက်ပြင်ဆင်လိုပါက သို့မဟုတ် သင့်ဝဘ်ဆိုက်အတွက် coupon ကုဒ် စာမျက်နှာလှလှလေးတစ်ခု တည်ဆောက်လိုပြီး သင့် checkout form တွင် လျှော့စျေး ကုဒ်ကို အလိုအလျောက် အသုံးချလိုပါက၊ URL parameters ဖြင့် ပြုလုပ်နိုင်ပါသည်။

ပထမဦးစွာ၊ သင့် plan အတွက် မျှဝေနိုင်သော link ကို ရယူရန် လိုအပ်ပါသည်။ ဤသို့ ပြုလုပ်ရန်၊ **Ultimate Multisite > Products** သို့ သွားပြီး plan တစ်ခု ရွေးချယ်ပါ။

**Click to Copy Shareable Link** ခလုတ်ကို နှိပ်ပါ။ ဤသည်မှာ ဤ plan အတိအကျအတွက် မျှဝေနိုင်သော link ကို ပေးမည်ဖြစ်သည်။ ကျွန်ုပ်တို့၏ ကိစ္စတွင်၊ ရရှိသော မျှဝေနိုင်သော link မှာ [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/)_ ဖြစ်ပါသည်။_

![မျှဝေနိုင်သော link ခလုတ်ပါသော Product စာမျက်နှာ](/img/config/products-list.png)

ဤ plan အတိအကျအတွက် သင့်လျှော့စျေး ကုဒ်ကို အသုံးချရန်၊ URL တွင် **?discount_code=XXX** parameter ကို ထည့်ပါ။ **XXX** သည် coupon ကုဒ် ဖြစ်ပါသည်။

ဤဥပမာတွင်၊ ကျွန်ုပ်တို့သည် ဤ product အတိအကျအတွက် **50OFF** coupon ကုဒ်ကို အသုံးချပါမည်။

ဤ plan အတိအကျအတွက် URL နှင့် 50OFF လျှော့စျေး ကုဒ် အသုံးချပြီးသည်မှာ ဤကဲ့သို့ ဖြစ်မည်: [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/) _**?discount_code=50OFF**_။
