# Plan များအတွက် မျှဝေနိုင်သော Link များကို စီမံခန့်ခွဲခြင်း

Ultimate Multisite က သင့် client များထံ plan link များကို တိုက်ရိုက်မျှဝေနိုင်တဲ့ ရွေးချယ်မှုကို ပေးထားပါတယ်။ ဒီလိုလုပ်ဖို့အတွက် သင့် plan တစ်ခုခုရဲ့ edit page ကိုသွားပြီး _Click to copy the Shareable Link_ ခလုတ်ကို နှိပ်ပါ။

![မျှဝေနိုင်သော link ခလုတ်ပါဝင်သည့် Product edit page](/img/config/product-edit-full.png)

_Plan ရဲ့ Edit Page မှာ မျှဝေနိုင်သော link များဖန်တီးရန် ခလုတ်ကို တွေ့နိုင်ပါတယ်_

ဒီနည်းလမ်းက link ကိုတည်ဆောက်ဖို့ default billing frequency ကို အသုံးပြုတယ်ဆိုတာ သတိပြုပါ။ ဆိုလိုတာက သင့် default billing က လစဉ်ဖြစ်ရင် ဥပမာအားဖြင့် နောက်ဆုံးမှာ ဒီလို link တစ်ခုရရှိမှာ ဖြစ်ပါတယ်: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

မျှဝေနိုင်သော link ကို သုံးလတစ်ကြိမ် (quarterly) အဖြစ် default သတ်မှတ်ချင်ရင် အဆုံးမှာ 3 ထည့်ပေးရပါမယ်

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

_premium_ plan ကို နှစ်စဉ် (yearly) အစီအစဉ်နဲ့ ရွေးချယ်ချင်ရင် 1 ထည့်ပြီး နောက်မှာ year ဆိုတဲ့ စာလုံးထည့်ပေးရပါမယ်:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

မတူညီတဲ့ billing frequency များအတွက် link တစ်ခုတည်းနဲ့ မရနိုင်ဘူးဆိုတာ သတိထားပါ။ သင့် billing frequency တစ်ခုချင်းစီအတွက် သီးခြား link တစ်ခုစီ မျှဝေသင့်ပါတယ်။

သင့် customer များကို link မျှဝေလိုက်တဲ့အခါ သူတို့ကို သင့် signup flow ရဲ့ ပထမအဆင့်ကို redirect လုပ်ပေးသွားမှာ ဖြစ်ပါတယ်။
