# PRO Theme ဖြင့် သင့် Admin Panel ကို စိတ်ကြိုက်ပြင်ဆင်ခြင်း

Client အတွက် အလုပ်လုပ်သောအခါ WordPress ၏ admin dashboard တွင် စိတ်ကြိုက် အတွေ့အကြုံတစ်ခုကို ပေးဖို့ လိုအပ်လေ့ရှိပါတယ်။ ပိုမိုပရော်ဖက်ရှင်နယ်ကျတဲ့ ပုံစံနဲ့ ခံစားမှုကို ရှာဖွေနေတယ်ဆိုရင် PRO Theme က သင့်ကို အထောက်အကူပြုနိုင်ပါတယ်။

PRO Theme ဟာ သင့် client များရဲ့ Dashboard ကို စိတ်ကြိုက်ပြင်ဆင်တဲ့အခါ ကူညီဖို့ ဖန်တီးထားတဲ့ admin theme တစ်ခုဖြစ်ပါတယ်။ ပြီးတော့ Ultimate Multisite ရဲ့ premium plugins core ရဲ့ အစိတ်အပိုင်းတစ်ခုလည်း ဖြစ်ပါတယ်။

ထည့်သွင်းပြီးနောက်၊ သင့် _Plugins_ စာမျက်နှာကို သွားပြီး သင့် Network ပေါ်တွင် activate လုပ်ဖို့ နှိပ်ပါ။

_သင့် Plugins စာမျက်နှာတွင် network-wide activate လုပ်ပါ_

_**မှတ်ချက်:**_ _ထည့်သွင်းရုံသာ ဆိုရင်၊ သင့် client များဆီမှာ ထို user အတွက်သာ theme ကို ထည့်ရန် ရွေးချယ်နိုင်တဲ့ option အဖြစ် ပေါ်လာပါလိမ့်မယ်။_

ယခု သင့် admin interface ကို လုံးဝပြန်လည် ဒီဇိုင်းဆွဲထားပါပြီ! ဘယ်ဘက် column မှာ PRO Theme ဆိုတဲ့ menu အသစ်တစ်ခုလည်း ရှိပါတယ်။
