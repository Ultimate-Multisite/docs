# Ultimate Multisite: Ad Injector ဖြင့် သင့် network တွင် ကြော်ငြာများထည့်သွင်းခြင်း

Ultimate Multisite: Ad Injector သည် **အခမဲ့ add-on** တစ်ခုဖြစ်ပြီး သင့် plan များ၏ edit page မှ တိုက်ရိုက် ကြော်ငြာများထည့်သွင်းနိုင်သော tools များကို ပေးစွမ်းပါသည်။

Ultimate Multisite: Ad Injector ကို install လုပ်ပြီးနောက်၊ သင့် _Plugins_ page သို့သွားပြီး activate လုပ်ပါ။

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Network Activate ခလုတ်ကို နှိပ်ပါ_

Add-on ကို activate လုပ်ပြီးသည်နှင့်၊ plan edit screen ရှိ _Advanced Options_ meta box တွင် _Ad Settings_ ဟုခေါ်သော option အသစ်တစ်ခု ပေါ်လာပါမည်။

![Plan edit screen ရှိ Ad Settings](/img/config/product-edit-full.png)

_ယခု သင့် plan တစ်ခုချင်းစီအတွက် ကြော်ငြာ setting များကို ပြင်ဆင်နိုင်ပါပြီ_

ထို tab တွင်၊ front-end ရှိ posts နှင့် pages များအတွက် content ရဲ့ အရှေ့နှင့် အနောက်တွင် ကြော်ငြာ code များကို ထည့်သွင်းနိုင်ပါသည်။

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_သင့်ကြော်ငြာ၏ code ကို ထည့်သွင်းပြီးနောက်၊ ထို plan အတွက် posts နှင့် pages များတွင် ပေါ်လာပါမည်။_

ထို plan ကိုအသုံးပြုသူများအတွက် Admin Pages ၏ header တွင်လည်း ကြော်ငြာများ ထည့်သွင်းနိုင်ပါသည်။

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_သင့်အသုံးပြုသူ၏ back-end တွင် ကြော်ငြာများပြသလိုပါက Admin Panel Ad Code option သို့ သွားပါ_

### ကြော်ငြာထက်ကျော်လွန်၍

အဆင့်မြင့် plan များအတွက် support ပေးရန်လည်း Ultimate Multisite: Ad Injector ကို အသုံးပြုနိုင်ပါသည်။

_Admin Panel Ads_ option သို့သွားပြီး သင့် support tool အတွက် HTML code ကို ထည့်ပါ။

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_သီးခြား plan များအတွက် customer support ထည့်သွင်းပါ_

ဤနည်းဖြင့် လူများသည် ၎င်းတို့၏ Admin Panel မှ တိုက်ရိုက် သင့်ထံ ဆက်သွယ်နိုင်ပါသည်!
