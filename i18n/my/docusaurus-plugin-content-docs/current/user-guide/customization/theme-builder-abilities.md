# Theme Builder စွမ်းဆောင်ရည်များ- ဘလော့ခ် ธีมများ အခြေခံတည်ဆောက်ခြင်းနှင့် အသက်သွင်းခြင်း

Superdav AI Agent v1.12.0 သည် စကားပြောဆိုမှု (chat interface) မှတစ်ဆင့် စိတ်ကြိုက် ဘလော့ခ် ธีมများကို ဖန်တီးပြီး အသုံးပြုနိုင်စေသည့် စွမ်းဆောင်ရည် (abilities) နှစ်ခုကို မိတ်ဆက်ပေးလိုက်ပါသည်။

## အကျဉ်းချုပ် (Overview)

**scaffold-block-theme** နှင့် **activate-theme** စွမ်းဆောင်ရည်များသည် agent များအား အောက်ပါအရာများကို လုပ်ဆောင်နိုင်စေသည်-
- သင့်လိုအပ်ချက်များအပေါ် အခြေခံ၍ အပြည့်အစုံ၊ အသုံးပြုရန် အသင့်ဖြစ်သော ဘလော့ခ် ธีมများကို ဖန်တီးပေးခြင်း
- လက်ဖြင့် အကူအညီပေးရန်မလိုဘဲ သင့်ဝဘ်ဆိုက်ပေါ်တွင် ธีมများကို အလိုအလျောက် အသက်သွင်းပေးခြင်း
- လမ်းညွှန်မှုပေးသည့် ဒီဇိုင်းဆုံးဖြတ်ချက်များမှတစ်ဆင့် စုစည်းညီညွတ်သော ရုပ်ပုံအသွင်အပြင်များကို ဖန်တီးပေးခြင်း

## ဘလော့ခ် ธีม အခြေခံတည်ဆောက်ခြင်း (Scaffold Block Theme)

**scaffold-block-theme** စွမ်းဆောင်ရည်သည် အပြည့်အစုံသော ธีมဖွဲ့စည်းပုံ (theme structure) အပါအဝင် WordPress ဘလော့ခ် ธีมအသစ်တစ်ခုကို ဖန်တီးပေးသည်၊ ၎င်းတွင် အောက်ပါတို့ ပါဝင်သည်-

- ဒီဇိုင်း တိုကင်များပါဝင်သော `theme.json` ဖွဲ့စည်းမှု
- အသုံးများသော layout များအတွက် ဘလော့ခ် template ဖိုင်များ
- စိတ်ကြိုက် ဘလော့ခ် စတိုင်များနှင့် ကွဲပြားမှုများ
- ธีม၏ metadata နှင့် ပံ့ပိုးမှု ကြေညာချက်များ

### မည်သို့ခေါ်ယူရမည်နည်း (How to Invoke)

Superdav AI Agent နှင့် သင်၏ စကားပြောဆိုမှုတွင် ธีมဖန်တီးခိုင်းရန် အောက်ပါအတိုင်း တောင်းဆိုနိုင်သည်-

```
"Create a block theme called 'Modern Agency' with a blue and white color scheme,
sans-serif typography, and a professional layout"
```

agent သည် အောက်ပါအဆင့်များကို လုပ်ဆောင်မည်-
1. စကားပြောဆိုမှုမှတစ်ဆင့် သင့်ဒီဇိုင်းနှစ်သက်မှုများကို စုစည်းရယူမည်
2. အပြည့်အစုံသော ธีมဖွဲ့စည်းပုံကို ဖန်တီးပေးမည်
3. လိုအပ်သော ธีมဖိုင်အားလုံးကို ဖန်တီးပေးမည်
4. အသက်သွင်းရန်အတွက် ธีมကို ပြင်ဆင်ပေးမည်

### မျှော်လင့်ထားသော ရလဒ် (Expected Output)

စွမ်းဆောင်ရည်သည် အောင်မြင်စွာ လုပ်ဆောင်ပြီးပါက၊ သင်သည် အောက်ပါအရာများကို မြင်တွေ့ရမည်-

- ธีมကို အခြေခံတည်ဆောက်ပြီးကြောင်း အတည်ပြုချက်
- ธีมအမည်နှင့် တည်နေရာ
- အသုံးပြုထားသော ဒီဇိုင်း တိုကင်များ၏ အကျဉ်းချုပ် (အရောင်များ၊ စာလုံးပုံစံ၊ နေရာခြားမှု)
- အသက်သွင်းရန် အသင့်ဖြစ်ကြောင်း အသိပေးချက်

ဥပမာရလဒ်-
```
✓ Theme "Modern Agency" scaffolded successfully
  Location: /wp-content/themes/modern-agency/
  Colors: Primary #0066CC, Secondary #FFFFFF
  Typography: Inter (sans-serif)
  Status: Ready to activate
```

## ธีม အသက်သွင်းခြင်း (Activate Theme)

**activate-theme** စွမ်းဆောင်ရည်သည် သင့်ဝဘ်ဆိုက်ကို အသစ်အခြေခံတည်ဆောက်ထားသော သို့မဟုတ် ရှိပြီးသား ဘလော့ခ် ธีมတစ်ခုသို့ ပြောင်းလဲပေးသည်။

### မည်သို့ခေါ်ယူရမည်နည်း (How to Invoke)

ธีมကို အခြေခံတည်ဆောက်ပြီးနောက်၊ ချက်ချင်း အသက်သွင်းနိုင်သည်-

```
"Activate the Modern Agency theme"
```

သို့မဟုတ် ရှိပြီးသား ธีมတစ်ခုခုကို အသက်သွင်းနိုင်သည်-

```
"Switch to the Twentytwentyfour theme"
```

### မျှော်လင့်ထားသော ရလဒ် (Expected Output)

အသက်သွင်းခြင်း အောင်မြင်ပါက-

- အသက်ဝင်နေသော ธีมကို အတည်ပြုချက်
- ယခင် ธีมအမည် (မှတ်သားရန်)
- ธีมသည် ယခုဘယ်နေရာတွင် အသက်ဝင်နေကြောင်း ဝဘ်ဆိုက် URL
- ธีมနှင့် သက်ဆိုင်သော စနစ်ထည့်သွင်းမှု မှတ်စုများ

ဥပမာရလဒ်-
```
✓ Theme activated successfully
  Active theme: Modern Agency
  Previous theme: Twentytwentyfour
  Live at: https://yoursite.com
  Note: Check your homepage to verify the layout
```

## လုပ်ငန်းစဉ် (Workflow): အခြေခံတည်ဆောက်ခြင်းနှင့် အသက်သွင်းခြင်း

ပုံမှန်လုပ်ငန်းစဉ်တစ်ခုသည် စွမ်းဆောင်ရည်နှစ်ခုလုံးကို ပေါင်းစပ်အသုံးပြုသည်-

1. **ธีมဖန်တီးခိုင်းရန် တောင်းဆိုခြင်း**: "ကျွန်ုပ်၏ SaaS landing page အတွက် ဘလော့ခ် ธีมတစ်ခု ဖန်တီးပေးပါ"
2. **Agent မှ ธีมကို အခြေခံတည်ဆောက်ပေးခြင်း**: ဖိုင်များနှင့် ဒီဇိုင်း တိုကင်များကို ဖန်တီးပေးသည်
3. **ပြန်လည်သုံးသပ်ခြင်းနှင့် ပြင်ဆင်ခြင်း**: လိုအပ်ပါက ဒီဇိုင်းပြောင်းလဲမှုများကို ဆွေးနွေးခြင်း
4. **အသက်သွင်းခြင်း**: "ธีมကို ယခု အသက်သွင်းပေးပါ"
5. **စစ်ဆေးခြင်း**: ဒီဇိုင်းအသစ်သည် အသက်ဝင်နေကြောင်း အတည်ပြုရန် သင့်ဝဘ်ဆိုက်သို့ ဝင်ကြည့်ပါ

## ဒီဇိုင်း တိုကင်များနှင့် စိတ်ကြိုက်ပြင်ဆင်မှု (Design Tokens and Customization)

အခြေခံတည်ဆောက်ထားသော ธีมများသည် WordPress ဒီဇိုင်း တိုကင်များ (theme.json မှတစ်ဆင့်) ကို အသုံးပြုသည်-

- **အရောင်များ (Colors)**: Primary, secondary, accent, neutral palette
- **စာလုံးပုံစံ (Typography)**: Font families, sizes, weights, line heights
- **နေရာခြားမှု (Spacing)**: Padding, margin, gap scales
- **ဘောင်များ (Borders)**: Radius နှင့် width တိုကင်များ
- **အရိပ်များ (Shadows)**: Elevation အဆင့်များ

ဤတိုကင်များသည် `theme.json` တွင် ဗဟိုချုပ်ကိုင်ထားသောကြောင့်၊ သင့်ဒီဇိုင်းစနစ်တစ်ခုလုံးကို ဖိုင်တစ်ခုတည်းမှ အလွယ်တကူ ချိန်ညှိနိုင်ပါသည်။

## ကန့်သတ်ချက်များနှင့် မှတ်သားစရာများ (Limitations and Notes)

- ธีมများကို `/wp-content/themes/` တွင် အခြေခံတည်ဆောက်ပြီး WordPress အမည်သတ်မှတ်ခြင်း စည်းမျဉ်းများကို လိုက်နာရမည်
- အသက်သွင်းရန်အတွက် သင့် WordPress ဝဘ်ဆိုက်တွင် သင့်လျော်သော ခွင့်ပြုချက်များ လိုအပ်သည်
- ธีมများရှိ စိတ်ကြိုက် PHP code များသည် အနည်းငယ်သာရှိသည်၊ ရှုပ်ထွေးသော လုပ်ဆောင်ချက်များအတွက် plugins ကို အသုံးပြုပါ
- ဘလော့ခ် ธีมများသည် WordPress 5.9 နှင့် နောက်ပိုင်းများတွင် အကောင်းဆုံး အလုပ်လုပ်သည်

## ပြဿနာဖြေရှင်းခြင်း (Troubleshooting)

**အခြေခံတည်ဆောက်ပြီးနောက် ธีม မပေါ်ခြင်း**
- ธีม directory ရှိမရှိနှင့် ခွင့်ပြုချက်များ မှန်ကန်မှုရှိမရှိ စစ်ဆေးပါ
- `theme.json` သည် မှန်ကန်သော JSON ဖြစ်ကြောင်း စစ်ဆေးပါ
- ธีมအမည်သည် ရှိပြီးသား ธีมများနှင့် ပဋိပက္ခမဖြစ်ကြောင်း သေချာပါစေ

**အသက်သွင်းခြင်း ပျက်ကွက်ခြင်း**
- administrator ခွင့်ပြုချက်များ ရှိမရှိ အတည်ပြုပါ
- ธีม directory ကို WordPress မှ ဖတ်နိုင်မှုရှိမရှိ စစ်ဆေးပါ
- အသေးစိတ်အချက်အလက်များအတွက် WordPress error logs များကို ပြန်လည်သုံးသပ်ပါ

**ဒီဇိုင်း တိုကင်များ အသုံးမဝင်ခြင်း**
- `theme.json` syntax မှန်ကန်ကြောင်း အတည်ပြုပါ
- caching plugins များကို ရှင်းလင်းပေးပါ
- သင့် WordPress version သည် သင်အသုံးပြုနေသော တိုကင်များကို ပံ့ပိုးမှုပေးခြင်းရှိမရှိ စစ်ဆေးပါ

## နောက်အဆင့်များ (Next Steps)

သင့်ธีมကို အသက်သွင်းပြီးနောက်၊ သင်သည် အောက်ပါအရာများကို လုပ်ဆောင်နိုင်သည်-
- စာလုံးပုံစံ၊ အရောင်များနှင့် နေရာခြားမှုများကို ပိုမိုကောင်းမွန်စေရန် **Design System Aesthetics skill** ကို အသုံးပြုခြင်း
- WordPress block editor မှတစ်ဆင့် တစ်ဦးချင်း ဘလော့ခ် စတိုင်များကို စိတ်ကြိုက်ပြင်ဆင်ခြင်း
- ธีม၏ `style.css` ဖိုင်တွင် စိတ်ကြိုက် CSS ထည့်သွင်းခြင်း
- သီးသန့် page အမျိုးအစားများအတွက် စိတ်ကြိုက် ဘလော့ခ် template များ ဖန်တီးခြင်း
