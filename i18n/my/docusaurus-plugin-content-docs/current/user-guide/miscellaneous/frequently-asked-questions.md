# မကြာခဏမေးလေ့ရှိသော မေးခွန်းများ

ဤနေရာတွင် အထောက်အပံ့တောင်းဆိုမှုများအတွင်း မကြာခဏပေါ်လာလေ့ရှိသော **မေးခွန်းများ** ကို စုစည်းဖော်ပြထားပါသည်။

  * **Ultimate Multisite ဆိုတာ ဘာလဲ?**

Ultimate Multisite သည် WordPress multisite install များအတွက် ဖန်တီးထားသော WordPress plugin တစ်ခုဖြစ်ပြီး၊ သင်၏ WordPress install ကို [WordPress.com](https://WordPress.com) ကဲ့သို့ premium site များ၏ network တစ်ခုအဖြစ် ပြောင်းလဲပေးပါသည် — သုံးစွဲသူများအား လစဉ်၊ သုံးလတစ်ကြိမ် သို့မဟုတ် နှစ်စဉ်ကြေးများဖြင့် site များဖန်တီးနိုင်စေပါသည် (အခမဲ့ plan များလည်း ဖန်တီးနိုင်ပါသည်)။

  * **Ultimate Multisite သည် Multisite မပါဘဲ အလုပ်လုပ်မလား?**

မလုပ်ပါ။ Ultimate Multisite အတွက် WordPress multisite၊ subdirectory သို့မဟုတ် subdomain လိုအပ်ပါသည်။

  * **Ultimate Multisite အတွက် ဘာတွေလိုအပ်သလဲ?**

Ultimate Multisite အတွက် လိုအပ်ချက်များ:

**WordPress ဗားရှင်း** : v5.3+ (အကြံပြုထားသည်: နောက်ဆုံးတည်ငြိမ်သော ဗားရှင်း)  
**PHP ဗားရှင်း** : 7.4.x (> 8.0 အထောက်အပံ့ မကြာမီလာမည်)  
**MySQL ဗားရှင်း** : v5+ (အကြံပြုထားသည်: ဗားရှင်း 5.6၊ သင်၏ hosting provider တွင် 8.0 မရရှိနိုင်ပါက)

_Multisite (subdomain သို့မဟုတ် subdirectory) ကိုလည်း အသက်သွင်းထားရန်လိုပါသည်။_ [WordPress Multisite ထည့်သွင်းနည်း](/user-guide/getting-started/how-to-install-wordpress-multisite)။

  * **Ultimate Multisite သည် Subdirectory ပါသော Multisite နှင့် အလုပ်လုပ်မလား?**

လုပ်ပါသည်။ Ultimate Multisite သည် subdirectory install များနှင့် subdirectory install များ နှစ်မျိုးလုံးကို ပံ့ပိုးပါသည်။

  * **Ultimate Multisite သည် Subdomain ပါသော Multisite နှင့် အလုပ်လုပ်မလား?**

လုပ်ပါသည်။ Ultimate Multisite သည် subdirectory install များနှင့် subdomain install များ နှစ်မျိုးလုံးကို ပံ့ပိုးပါသည်။

  * **Ultimate Multisite ကို သုံးပြီး သုံးစွဲသူများ အသုံးပြုနိုင်သော Plugin များနှင့် Theme များကို ကန့်သတ်နိုင်မလား?**

နိုင်ပါသည်။ Ultimate Multisite ဖြင့် သင်၏ network ရှိ plan တစ်ခုချင်းစီအတွက် မည်သည့် plugin များနှင့် theme များ ရရှိနိုင်မည်ကို ရွေးချယ်နိုင်ပါသည်။

  * **Ultimate Multisite ကို သုံးပြီး သုံးစွဲသူတစ်ဦး၏ site တွင် ဖန်တီးနိုင်သော post အရေအတွက်ကို ကန့်သတ်နိုင်မလား?**

နိုင်ပါသည်။ Ultimate Multisite ဖြင့် သင်၏ network ရှိ **ရရှိနိုင်သော post type တိုင်း** အတွက် post quota များ သတ်မှတ်နိုင်ပါသည်။ admin က ဖိတ်ခေါ်နိုင်သော user အရေအတွက်၊ **disk space ပမာဏ** နှင့် အခြားအရာများကိုလည်း ကန့်သတ်နိုင်ပါသည်။

  * **Ultimate Multisite သည် Site Template များကို ပံ့ပိုးပါသလား?**

ပံ့ပိုးပါသည်။ Ultimate Multisite ဖြင့် Site Template များကို သင်လိုချင်သလောက် ဖန်တီးနိုင်ပါသည်။ Site Template များသည် theme များမဟုတ်ပါ၊ ၎င်းတို့သည် သင်၏ network ရှိ ပုံမှန် site များဖြစ်ပါသည်။ plugin များ၊ theme များကို activate လုပ်နိုင်ပြီး ထို Site Template ကို သင်လိုချင်သလို စိတ်ကြိုက်ပြင်ဆင်နိုင်ပါသည်။ သုံးစွဲသူတစ်ဦးက ထို template ကို ရွေးချယ်ပါက၊ သူ့ site သည် customization များအားလုံးပါဝင်သော ထို template ၏ မိတ္တူအဖြစ် ဖန်တီးခံရမည်ဖြစ်ပါသည်။

plan တစ်ခုအတွက် template တစ်ခုကိုလည်း သတ်မှတ်နိုင်ပါသည်။ ထိုသို့လုပ်ခြင်းဖြင့် ထို plan တွင် စာရင်းသွင်းသော သုံးစွဲသူတိုင်း၏ site သည် ထို Site Template ၏ ပုံတူအဖြစ် ဖန်တီးခံရမည်ဖြစ်ပါသည်။

  * **Ultimate Multisite သည် WooCommerce နှင့် ပေါင်းစပ်အလုပ်လုပ်ပါသလား?**

လုပ်ပါသည်!

သင်၏ network user များအတွက် WooCommerce ကို ရရှိစေလိုပြီး store များ တည်ဆောက်ခွင့်ပေးလိုပါက Ultimate Multisite ဖြင့် ဖြစ်နိုင်ပါသည်။ plan tier တစ်ခုချင်းစီတွင် သင့်ဖောက်သည်များ ဖန်တီးနိုင်သော product အရေအတွက်ကိုပင် ကန့်သတ်နိုင်ပါသည်။

Ultimate Multisite အတွက် subscription payment များနှင့်ပတ်သက်၍၊ [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) ကို အသုံးပြုပြီး သင်၏ Ultimate Multisite ဖောက်သည်များအား သင်၏ main site ရှိ WooCommerce gateway များဖြင့် ငွေပေးချေခွင့်ပေးနိုင်ပါသည်။ ထိုနည်းဖြင့် WooCommerce အတွက် ဖန်တီးထားသော payment gateway library ကြီးကို အသုံးချပြီး သင်၏ Ultimate Multisite ဖောက်သည်များကို ငွေတောင်းခံနိုင်ပါသည်။

  * **WPMUDEV Pro Sites မှ subscription များကို ပြောင်းရွှေ့ရန် နည်းလမ်းရှိပါသလား?**

ရှိပါသည်။ Ultimate Multisite တွင် သင်၏ Pro Sites network ကို ပြောင်းရွှေ့ရန် [အထောက်အကူဖြစ်စေမည့် tool တစ်ခု](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) ယခုရရှိနိုင်ပါပြီ။

  * **ကျွန်ုပ်၏ Plan Edit Page တွင် "Allowed Plugins" tab သည် အဘယ်ကြောင့် ဗလာဖြစ်နေသနည်း?**

Ultimate Multisite သည် သင်၏ network တွင် network-activate မလုပ်ထားသော plugin များကိုသာ ခွင့်ပြုရန် သို့မဟုတ် ငြင်းပယ်ရန် ခွင့်ပြုပါသည်။ plugin တစ်ခုကို network-activate လုပ်ထားသောအခါ၊ ၎င်း၏ သဘာဝအရ သင်၏ network ရှိ site တိုင်းတွင် default အနေဖြင့် load ဖြစ်မည်ဖြစ်ပါသည်။

"Allowed Plugins" tab တွင် plugin များ မမြင်ရပါက၊ လိုချင်သော plugin သည် network-active ဖြစ်မဖြစ် စစ်ဆေးပါ။

  * **ကျွန်ုပ်ကိုယ်ပိုင် Pricing Table ဒီဇိုင်းကို သုံးနိုင်ပါသလား?**

သုံးနိုင်ပါသည်။ ဗားရှင်း 1.4.0 မှစ၍ Ultimate Multisite သည် template override များကို ပံ့ပိုးပါသည်။

  * **Ultimate Multisite သည် WP Engine နှင့် အလုပ်လုပ်ပါသလား?**

လုပ်ပါသည်။ Ultimate Multisite ကို WP Engine က အမြဲပံ့ပိုးခဲ့ပါသည်။ သို့သော် domain mapping အလုပ်လုပ်စေရန် admin မှ လက်ဖြင့် အပိုအဆင့်များ (domain ကို WP Engine panel သို့ ထည့်ခြင်း) လုပ်ဆောင်ရန် လိုအပ်ခဲ့ပါသည်။ **ဗားရှင်း 1.6.0 မှစ၍ ဤသို့ မဟုတ်တော့ပါ။** Ultimate Multisite သည် WP Engine API နှင့် တိုက်ရိုက်ပေါင်းစပ်ထားပြီး domain များကို mapping ပြုလုပ်ပြီးနောက် panel သို့ အလိုအလျောက် ထည့်သွင်းပေးပါသည်။

  * **မည်သည့် hosting provider ကို အကြံပြုပါသလဲ?**

Ultimate Multisite သည် မည်သည့် hosting provider နှင့်မဆို အလုပ်လုပ်သင့်ပါသည်။ ခြားနားချက်မှာ Closte၊ Serverpilot၊ WPMUDEV၊ WP Engine၊ Cloudways၊ Gridpane နှင့် [Runcloud.io](http://Runcloud.io) ကဲ့သို့ hosting company အချို့သည် domain mapping စွမ်းရည်များအတွက် ပိုမိုကျစ်လစ်သော integration ရှိသဖြင့် ထည့်သွင်းပြီးသည်နှင့် ချက်ချင်းအလုပ်လုပ်ပါသည်။ အခြား hosting များအတွက် domain mapping ကို လက်ဖြင့် သတ်မှတ်ရန် လိုအပ်နိုင်ပါသည်။
