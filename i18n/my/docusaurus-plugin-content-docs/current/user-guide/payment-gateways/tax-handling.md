# အခွန်ကိုင်တွယ်ခြင်း

Ultimate Multisite တွင် ကျွန်ုပ်တို့၏ core plugin ထဲမှာ အခွန်ကောက်ခံမှု module တစ်ခု ပါဝင်ပြီးသားဖြစ်သောကြောင့်၊ သင့် plans များ၊ packages များနှင့် services များတွင် အရောင်းခွန်ကောက်ခံရန် လိုအပ်ပါက မည်သည့် add-ons မျှ ထည့်သွင်းစရာမလိုဘဲ လွယ်ကူစွာ ဆောင်ရွက်နိုင်ပါသည်။

ဥရောပတွင် တည်ရှိသော ကုမ္ပဏီများအတွက်၊ **VAT** လိုက်နာမှုကို ပိုမိုကောင်းမွန်စွာ **ပံ့ပိုးရန်** tools များနှင့် features များ ထပ်ထည့်ပေးသော **add-on** တစ်ခု ပေးထားပါသည်။

Ultimate Multisite သည် သင့်ကိုယ်စား အခွန်များကို အစိုးရထံ တင်သွင်းခြင်း သို့မဟုတ် ပေးသွင်းခြင်း မပြုပါ။ ကျွန်ုပ်တို့သည် ငွေပေးငွေယူပြုလုပ်ချိန်တွင် သင့်လျော်သော အခွန်များကို ကောက်ခံရန်သာ ကူညီပေးပါသည်။ **အခွန်များကို သင်ကိုယ်တိုင် ပေးသွင်းရန် လိုအပ်ပါသေးသည်။**

## အခွန်ကောက်ခံမှုကို ဖွင့်ခြင်း

အခွန်ကောက်ခံမှုသည် ပုံသေအားဖြင့် ဖွင့်ထားခြင်း မရှိပါ။ ဖွင့်ရန်အတွက်၊ **Ultimate Multisite > Settings > Taxes** သို့ သွားပြီး Enable Taxes setting ကို ဖွင့်ရန် toggle လုပ်ပါ။

![အခွန် settings ရှိ Enable Taxes toggle](/img/config/settings-taxes.png)

### အခွန်မပါ vs. အခွန်ပါပြီး

ပုံသေအားဖြင့်၊ သင့် product စျေးနှုန်းအားလုံးသည် အခွန်မပါ ဖြစ်သည်၊ ဆိုလိုသည်မှာ product ၏ စျေးနှုန်းတွင် အခွန်များ **မပါဝင်ပါ**။ ဝယ်ယူသူတစ်ဦးသည် ဝယ်ယူမှုတစ်ခုတွင် အခွန်ပေးဆောင်ရမည်ဟု ကျွန်ုပ်တို့ ဆုံးဖြတ်ပါက၊ စုစုပေါင်းပေါ်တွင် အခွန်များကို **ထပ်ထည့်** ပါမည်။

သင့် product ၏ စျေးနှုန်းတွင် အခွန်များ ပါဝင်စေလိုပါက၊ **Inclusive Tax** setting ကို ဖွင့်ခြင်းဖြင့် ဆောင်ရွက်နိုင်ပါသည်။

![Inclusive Tax setting toggle](/img/config/settings-taxes.png)

ပြုလုပ်ခဲ့သော ပြောင်းလဲမှုများကို **save** လုပ်ရန် မမေ့ပါနှင့်။

### 

## အခွန်နှုန်းများ ဖန်တီးခြင်း

အခွန်ကောက်ခံမှုကို ဖွင့်ပြီးနောက်၊ ကျွန်ုပ်တို့၏ အခွန်နှုန်း editor ကို အသုံးပြုပြီး တည်နေရာအလိုက် အခွန်နှုန်းများ ဖန်တီးရန် လိုအပ်ပါသည်။

Tax settings page ၏ sidebar ပေါ်ရှိ **Manage Tax Rates** button ကို နှိပ်ခြင်းဖြင့် editor ကို ဝင်ရောက်နိုင်ပါသည်။

![sidebar ပေါ်ရှိ Manage Tax Rates button](/img/config/settings-taxes.png)

အခွန်နှုန်း editor page တွင်၊ **Add new Row** button ကို နှိပ်ခြင်းဖြင့် အခွန်နှုန်းအသစ်များ ထည့်နိုင်ပါသည်။

![Add new Row button ပါသော အခွန်နှုန်း editor](/img/config/settings-taxes.png)

အခွန်နှုန်းတစ်ခုစီကို **title** (ပြေစာများတွင် အသုံးပြုသည်) ပေးရန် လိုအပ်ပါသည်။ ထို့နောက် ဤအခွန်ကို ကောက်ခံမည့် **country** (မဖြစ်မနေလိုအပ်သည်)၊ **state** နှင့် **city** (နှစ်ခုစလုံး ရွေးချယ်စရာ) တို့ကို ရွေးချယ်နိုင်ပါသည်။ နောက်ဆုံးတွင်၊ **ရာခိုင်နှုန်းဖြင့် အခွန်နှုန်း** ကို ထည့်ပါ။

### အခွန်အမျိုးအစားများ

မတူညီသော products အမျိုးအစားများအတွက် မတူညီသော အခွန်နှုန်းများ ထည့်ရန် Tax Categories အများအပြား ဖန်တီးနိုင်ပါသည်။

**Add new Tax Category** ကို နှိပ်ပါ၊ ထို့နောက် သင့် category ၏ အမည်ကို ရေးပြီး **Create** ကို နှိပ်ပါ။

![Add new Tax Category button](/img/config/settings-taxes.png)

![အခွန်အမျိုးအစားအသစ် ဖန်တီးခြင်း](/img/config/settings-taxes.png)

categories များကြားတွင် ကြည့်ရှုရန်၊ **Switch** ကို နှိပ်ပြီး အခွန်အသစ်များ ထည့်လိုသော category ကို ရွေးချယ်ပါ။

![အခွန်အမျိုးအစားများကြား ပြောင်းခြင်း](/img/config/settings-taxes.png)

![ကြည့်ရှုရန် အခွန်အမျိုးအစား ရွေးချယ်ခြင်း](/img/config/settings-taxes.png)

**Product edit page** သို့သွားပြီး Taxes tab သို့သွားခြင်းဖြင့် product တစ်ခုချင်းစီအတွက် အခွန်အမျိုးအစား သတ်မှတ်နိုင်ပါသည်။

![အခွန်အမျိုးအစားနှင့် taxable toggle ပါသော Product taxes tab](/img/config/product-taxes.png)

ထိုစခရင်တွင်ပင်၊ ထို product တွင် အခွန်မကောက်ခံသင့်ကြောင်း Ultimate Multisite ကို အသိပေးရန် **Is Taxable?** toggle ကို ပိတ်ထားနိုင်ပါသည်။

## ဥရောပ VAT ပံ့ပိုးမှု

အစောပိုင်းတွင် ဖော်ပြခဲ့သည့်အတိုင်း၊ ဥရောပ VAT စည်းမျဉ်းများကြောင့် အပိုလိုအပ်ချက်များရှိသော EU ရှိ ဝယ်ယူသူများအတွက် add-on တစ်ခု ရနိုင်ပါသည်။

ကျွန်ုပ်တို့၏ VAT tools များသည် အရေးကြီးသော အချက်အချို့ကို ကူညီပေးပါသည်:

  * EU VAT နှုန်းများကို လွယ်ကူစွာ ထည့်သွင်းခြင်း;

  * VAT Number ကောက်ခံခြင်းနှင့် အတည်ပြုခြင်း - နှင့် VAT ကင်းလွတ်ခွင့်ရှိသော အဖွဲ့အစည်းများအတွက် (တရားဝင် VAT numbers ရှိသော ကုမ္ပဏီများကဲ့သို့) reverse charging;

ထို add-on ကို ထည့်သွင်းရန်၊ **Ultimate Multisite > Settings** သို့သွားပြီး **Check our Add-ons** sidebar link ကို နှိပ်ပါ။

![add-ons sidebar link ပါသော Settings page](/img/config/settings-taxes.png)

ကျွန်ုပ်တို့၏ add-ons page သို့ ပြန်လည်ညွှန်းပေးပါမည်။ ထိုနေရာတွင်၊ **Ultimate Multisite VAT add-on** ကို ရှာဖွေပြီး ထည့်သွင်းနိုင်ပါသည်။

![add-ons page ပေါ်ရှိ VAT add-on](/img/config/settings-taxes.png)

![VAT add-on install dialog](/img/config/settings-taxes.png)

ထို့နောက်၊ **Network Admin > Plugins** သို့သွားပြီး ထို add-on ကို network-wide activate လုပ်ပါ။

![VAT add-on ကို Network Activate လုပ်ခြင်း](/img/config/settings-taxes.png)

**Tax Settings tab** သို့ ပြန်သွားပါက၊ ရွေးချယ်စရာအသစ်များ ရရှိနိုင်သည်ကို တွေ့ရပါမည်။ VAT tools အသစ်များကို ဖွင့်ရန် **Enable VAT Support** option ကို toggle လုပ်ပါ။ သင့် settings ကို **save** လုပ်ရန် မမေ့ပါနှင့်!

![အခွန် settings ရှိ Enable VAT Support toggle](/img/config/settings-taxes.png)

### VAT အခွန်နှုန်းများ ဆွဲယူခြင်း

ကျွန်ုပ်တို့၏ integration ထည့်သွင်းပေးသော tools များထဲမှ တစ်ခုမှာ EU အဖွဲ့ဝင်နိုင်ငံများအတွက် အခွန်နှုန်းများကို ထည့်သွင်းနိုင်သော စွမ်းရည်ဖြစ်သည်။ EU VAT support ကို ဖွင့်ပြီးနောက် အခွန်နှုန်း editor page သို့ သွားရောက်ခြင်းဖြင့် ဆောင်ရွက်နိုင်ပါသည်။

page ၏ အောက်ခြေတွင်၊ VAT ဆွဲယူခြင်း ရွေးချယ်စရာများကို တွေ့ရပါမည်။ နှုန်းအမျိုးအစားတစ်ခု ရွေးချယ်ပြီး **Update EU VAT Rates** button ကို နှိပ်ခြင်းဖြင့် EU အဖွဲ့ဝင်နိုင်ငံတစ်ခုစီအတွက် အခွန်နှုန်းများဖြင့် ဇယားကို အလိုအလျောက် ဖြည့်သွင်းပေးပါမည်။ ထို့နောက်၊ save လုပ်ရုံသာ လိုအပ်ပါသည်။

### ![Update EU VAT Rates button ပါသော VAT နှုန်း ဆွဲယူခြင်း ရွေးချယ်စရာများ](/img/config/settings-taxes.png)

ဆွဲယူပြီးနောက် တန်ဖိုးများကို တည်းဖြတ်နိုင်ပါသည်။ ထိုသို့ပြုလုပ်ရန်၊ လိုအပ်သော ဇယားလိုင်းကို တည်းဖြတ်ပြီး တန်ဖိုးအသစ်များကို save လုပ်ရန် နှိပ်ပါ။

### VAT အတည်ပြုခြင်း

VAT support ကို ဖွင့်ထားသောအခါ၊ Ultimate Multisite သည် checkout form တွင် billing address field အောက်၌ field အပိုတစ်ခု ထည့်ပေးပါမည်။ ထို field သည် EU တွင် တည်ရှိသော ဝယ်ယူသူများအတွက်သာ ပေါ်လာပါမည်။

![checkout form ပေါ်ရှိ VAT number field](/img/config/settings-taxes.png)

Ultimate Multisite သည် VAT Number ကို အတည်ပြုပြီး တရားဝင်သည်ဟု ပြန်လာပါက၊ reverse charge mechanism ကို အသုံးပြုပြီး ထို order တွင် အခွန်နှုန်းကို 0% သတ်မှတ်ပါသည်။
