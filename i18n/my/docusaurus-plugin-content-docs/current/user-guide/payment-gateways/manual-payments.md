# လက်ဖြင့် ငွေပေးချေမှုများ စနစ်တကျပြင်ဆင်ခြင်း (v2)

_**အရေးကြီး မှတ်ချက် - ဤဆောင်းပါးသည် Ultimate Multisite ဗားရှင်း 2.x အတွက် ဖြစ်သည်။**_

လက်ဖြင့် ငွေပေးချေမှုများသည် သင့်အသုံးပြုသူများအတွက် **Stripe** သို့မဟုတ် **PayPal** မရနိုင်သည့်အခါ အခြားငွေပေးချေမှုနည်းလမ်းများကို ပေးနိုင်ရန် အသုံးပြုသည့်နည်းလမ်းဖြစ်သည်။ ၎င်းသည် wire သို့မဟုတ် ဘဏ်လွှဲငွေ ဖြစ်နိုင်ပြီး သင့်အသုံးပြုသူများ၏ ဒေသတွင် ရနိုင်သော အခြားငွေပေးချေမှုနည်းလမ်းလည်း ဖြစ်နိုင်သည်။

## လက်ဖြင့် ငွေပေးချေမှုများကို ဖွင့်နည်း

လက်ဖြင့် ငွေပေးချေမှုကို စနစ်တကျပြင်ဆင်ခြင်းသည် အလွန်လွယ်ကူသည်။ ငွေပေးချေမှု gateway များအောက်တွင် ၎င်းကို ဖွင့်ပြီး အသုံးပြုသူက ငွေကို မည်သို့ပေးပို့ရမည်ဆိုသည့် အသေးစိတ်ညွှန်ကြားချက်များ ထည့်သွင်းရုံသာ လိုသည်။

ပထမဆုံး **Ultimate Multisite > ဆက်တင်များ > ငွေပေးချေမှုများ** သို့ သွားပါ။ **ငွေပေးချေမှု Gateway များ** အောက်တွင် **လက်ဖြင့်** ကို ဖွင့်ပါ။ ထို့နောက် သင့်အတွက် **ငွေပေးချေမှု ညွှန်ကြားချက်များ** အကွက်တစ်ခု ပေါ်လာပါမည်။

ဤအကွက်ထဲတွင် သင့်ဖောက်သည်က ငွေပေးချေမှုပြုလုပ်ရန် လိုအပ်မည့် အချက်အလက်များကို ထည့်ပါ။ ဥပမာအားဖြင့် သင့်ဘဏ်အကောင့်အသေးစိတ်များနှင့် ဖောက်သည်က ငွေပေးချေမှုအတည်ပြုချက် ပေးပို့နိုင်ရန် သင့်အီးမေးလ်ကို ထည့်နိုင်သည်။

![ငွေပေးချေမှု ညွှန်ကြားချက်များ စာသားဧရိယာပါသော လက်ဖြင့် ငွေပေးချေမှု gateway ဖွင့်ပိတ်ခလုတ်](/img/config/manual-gateway-expanded.png)

လက်ဖြင့် gateway ဆက်တင်များ အင်တာဖေ့စ်မှာ ဤအတိုင်းဖြစ်သည်။

![လက်ဖြင့် gateway ဆက်တင်များ](/img/config/manual-gateway-settings.png)

စနစ်တကျပြင်ဆင်ပြီးနောက် **ဆက်တင်များ သိမ်းဆည်းရန်** ကို နှိပ်ရုံဖြင့် ပြီးပါပြီ။ အသုံးပြုသူများ သင့်ကွန်ရက်တွင် စာရင်းသွင်းသည့်အခါ ဝယ်ယူမှုကို ပြီးမြောက်ရန် သင့်ညွှန်ကြားချက်များကို ရရှိမည်ဖြစ်ကြောင်း ပြောပြသည့် မက်ဆေ့ချ်ကို မြင်ရပါမည်။

![အသုံးပြုသူသည် ငွေပေးချေမှုညွှန်ကြားချက်များ ရရှိမည်ဖြစ်ကြောင်း ပြောပြသည့် စာရင်းသွင်းမှု အတည်ပြု မက်ဆေ့ချ်](/img/frontend/registration-manual-notice.png)

ထို့ပြင် သင့် **ကျေးဇူးတင်ပါသည်** စာမျက်နှာတွင်လည်း သင့်ငွေပေးချေမှုညွှန်ကြားချက်များပါသော မက်ဆေ့ချ်ကို ရရှိပါမည်။

## လက်ဖြင့် ငွေပေးချေမှုများကို အတည်ပြုခြင်း

လက်ဖြင့် ငွေပေးချေမှုကို အတည်ပြုရန် ဘယ်ဘက်ဘားရှိ **ငွေပေးချေမှုများ** မီနူးသို့ သွားပါ။ ထိုနေရာတွင် သင့်ကွန်ရက်ရှိ ငွေပေးချေမှုအားလုံးနှင့် ၎င်းတို့၏ အသေးစိတ်များကို **အခြေအနေ** အပါအဝင် မြင်နိုင်သည်။ လက်ဖြင့် ငွေပေးချေမှုတစ်ခုသည် သင်ကိုယ်တိုင် မပြောင်းလဲမချင်း **ဆိုင်းငံ့ထားသည်** အခြေအနေ အမြဲရှိနေမည်။

![ဆိုင်းငံ့ထားသော လက်ဖြင့် ငွေပေးချေမှုကို ပြသသော ငွေပေးချေမှုစာရင်း](/img/admin/payments-list.png)

**ကိုးကားကုဒ်** ကို နှိပ်၍ ငွေပေးချေမှုစာမျက်နှာသို့ ဝင်ပါ။ ဤစာမျက်နှာတွင် reference ID၊ ထုတ်ကုန်များ၊ အချိန်မှတ်တမ်းများနှင့် အခြားအချက်များကဲ့သို့ ဆိုင်းငံ့ထားသော ငွေပေးချေမှု၏ အသေးစိတ်အားလုံး ရှိသည်။

![ကိုးကားကုဒ်၊ ထုတ်ကုန်များနှင့် စုစုပေါင်းများကို ပြသသော ငွေပေးချေမှုအသေးစိတ် စာမျက်နှာ](/img/admin/payment-edit.png)

ညာဘက်ကော်လံတွင် ငွေပေးချေမှု၏ အခြေအနေကို ပြောင်းလဲနိုင်သည်။ ၎င်းကို **ပြီးမြောက်ပြီး** သို့ ပြောင်းပြီး **အဖွဲ့ဝင်မှု အသက်သွင်းရန်** ရွေးချယ်မှုကို **ဖွင့်** ပါက သင့်ဖောက်သည်၏ site ကို ဖွင့်ပေးမည်ဖြစ်ပြီး ၎င်းတို့၏ အဖွဲ့ဝင်မှုလည်း အသက်ဝင်လာမည်ဖြစ်သည်။

![အခြေအနေကို ပြီးမြောက်ပြီး သတ်မှတ်ထားပြီး အဖွဲ့ဝင်မှု အသက်သွင်းရန် ခလုတ်ပါသော ငွေပေးချေမှု တည်းဖြတ်စာမျက်နှာ](/img/admin/payment-activate-membership.png)
