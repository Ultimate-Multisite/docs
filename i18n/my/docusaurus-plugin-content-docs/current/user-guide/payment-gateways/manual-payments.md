# လက်ဖြင့် ငွေပေးချေမှုများ သတ်မှတ်ခြင်း (v2)

_**အရေးကြီးသော မှတ်ချက်: ဤဆောင်းပါးသည် Ultimate Multisite version 2.x အတွက် ရည်ညွှန်းပါသည်။**_

လက်ဖြင့် ငွေပေးချေမှုများသည် သင့်အသုံးပြုသူများအတွက် **Stripe** သို့မဟုတ် **PayPal** မရရှိနိုင်သောအခါ အခြားငွေပေးချေမှုနည်းလမ်းများကို ကမ်းလှမ်းရန် နည်းလမ်းတစ်ခုဖြစ်သည်။ ၎င်းသည် ဘဏ်ငွေလွှဲခြင်း သို့မဟုတ် သင့်အသုံးပြုသူများ ဒေသတွင်း ရရှိနိုင်သော အခြားငွေပေးချေမှုနည်းလမ်း တစ်ခုခုဖြစ်နိုင်သည်။

## လက်ဖြင့် ငွေပေးချေမှုများကို ဘယ်လို ဖွင့်မလဲ

လက်ဖြင့် ငွေပေးချေမှုကို သတ်မှတ်ရန် အလွန်လွယ်ကူပါသည်။ Payment gateways အောက်တွင် ၎င်းကို ဖွင့်ပြီး အသုံးပြုသူက ငွေပေးချေမှုကို မည်သို့ ပို့ရမည်ဆိုသည့် အသေးစိတ် ညွှန်ကြားချက်များကို ထည့်ရုံသာဖြစ်သည်။

ပထမဦးစွာ **Ultimate Multisite > Settings > Payments** သို့ သွားပါ။ **Payment Gateways** အောက်တွင် **Manual** ကို ဖွင့်ပါ။ **Payment Instructions** အကွက်တစ်ခု ပေါ်လာသည်ကို တွေ့ရပါမည်။

ဤအကွက်တွင် သင့်ဖောက်သည်က ငွေပေးချေမှု ပြုလုပ်ရန် လိုအပ်သော အချက်အလက်များကို ထည့်ပါ။ ဥပမာအားဖြင့် သင့်ဘဏ်အကောင့် အသေးစိတ်များနှင့် ဖောက်သည်က ငွေပေးချေမှု အတည်ပြုချက် ပို့နိုင်ရန် သင့်အီးမေးလ်ကို ထည့်နိုင်သည်။

![Manual payment gateway toggle နှင့် payment instructions အကွက်](/img/config/settings-payment-gateways.png)

သတ်မှတ်ပြီးနောက် **Save Settings** ကို နှိပ်ရုံဖြင့် ပြီးပါပြီ။ အသုံးပြုသူများ သင့် network တွင် စာရင်းသွင်းသောအခါ ဝယ်ယူမှု ပြီးမြောက်ရန် သင့်ညွှန်ကြားချက်များကို လက်ခံရရှိမည်ဟူသော မက်ဆေ့ချ်ကို မြင်ရပါမည်။

![စာရင်းသွင်းစဉ် ပြသသော လက်ဖြင့် ငွေပေးချေမှု မက်ဆေ့ချ်](/img/config/settings-payment-gateways.png)

ထို့အပြင် သင့် **Thank You** စာမျက်နှာတွင်လည်း သင့်ငွေပေးချေမှု ညွှန်ကြားချက်များပါသော မက်ဆေ့ချ်ကို လက်ခံရရှိပါမည်။

![ငွေပေးချေမှု ညွှန်ကြားချက်များ ပြသသော Thank You စာမျက်နှာ](/img/config/settings-payment-gateways.png)

## လက်ဖြင့် ငွေပေးချေမှုများကို အတည်ပြုခြင်း

လက်ဖြင့် ငွေပေးချေမှုကို အတည်ပြုရန် ဘယ်ဘက် ဘားတန်းရှိ **Payments** မီနူးသို့ သွားပါ။ ထိုတွင် သင့် network ပေါ်ရှိ ငွေပေးချေမှုအားလုံးနှင့် ၎င်းတို့၏ **status** အပါအဝင် အသေးစိတ်များကို မြင်နိုင်သည်။ လက်ဖြင့် ငွေပေးချေမှုသည် သင် ကိုယ်တိုင် မပြောင်းမချင်း **Pending** status ဖြင့် အမြဲရှိနေပါမည်။

![Pending လက်ဖြင့် ငွေပေးချေမှု ပြသသော ငွေပေးချေမှုများ စာရင်း](/img/admin/payments-list.png)

**reference code** ကို နှိပ်၍ ငွေပေးချေမှု စာမျက်နှာသို့ ဝင်ပါ။ ဤစာမျက်နှာတွင် reference ID၊ ပစ္စည်းများ၊ timestamps နှင့် အခြားအရာများကဲ့သို့ pending ငွေပေးချေမှု၏ အသေးစိတ်အားလုံးကို မြင်ရပါသည်။

![Reference code နှင့် ပစ္စည်းများပါသော ငွေပေးချေမှု အသေးစိတ် စာမျက်နှာ](/img/admin/payments-list.png)

ညာဘက် ကော်လံတွင် ငွေပေးချေမှု၏ status ကို ပြောင်းလဲနိုင်သည်။ ၎င်းကို **Completed** သို့ပြောင်းပြီး **Activate Membership** option ကို ဖွင့်ခြင်းဖြင့် သင့်ဖောက်သည်၏ site ကို အသက်ဝင်စေပြီး ၎င်းတို့၏ membership အသက်ဝင်ပါမည်။

![Activate Membership toggle နှင့်အတူ Completed သို့ ငွေပေးချေမှု status ပြောင်းလဲခြင်း](/img/admin/payments-list.png)
