# Customer များ စီမံခန့်ခွဲခြင်း

**Customers** စာမျက်နှာတွင် သင့် network တွင် မှတ်ပုံတင်ထားသော member များစာရင်းကို ပြသထားပါသည်။ ဤစာမျက်နှာတွင် အသုံးပြုသူ၏ အခြေခံအချက်အလက်များဖြစ်သည့် အမည်၊ အီးမေးလ်လိပ်စာ၊ နောက်ဆုံးဝင်ရောက်ချိန်နှင့် အသုံးပြုသူ၏ subsite သို့ site administrator အနေဖြင့် ဝင်ရောက်နိုင်သည့် **switch to** လင့်ခ်တို့ကို ပြသထားပါသည်။

![Customers list page](/img/admin/customers-list.png)

Customer တစ်ဦး၏ အသေးစိတ်အချက်အလက်များကို ကြည့်ရှုရန် ၎င်း၏ card အောက်ရှိ **Manage** ကို နှိပ်ပါ။

Customer အသေးစိတ်စာမျက်နှာတွင် **At a Glance** block သည် customer ID၊ နောက်ဆုံးဝင်ရောက်ချိန်နှင့် ဤ customer သုံးစွဲခဲ့သော စုစုပေါင်းငွေပမာဏတို့ကို ပြသပါသည်။

**Customer Options** block တွင် ဤ customer ကို **VIP** အဖြစ် သတ်မှတ်နိုင်ပါသည် (ရှာဖွေရလွယ်ကူစေရန် tag တစ်ခုအဖြစ်)။ **Billing Info** tab တွင် အီးမေးလ်၊ လိပ်စာနှင့် နိုင်ငံစသည့် ငွေပေးချေမှုအချက်အလက်များကို ပြသပါသည်။

**Fields** block တွင် customer username၊ အီးမေးလ်တို့ကို ပြသပြီး **Email Verification** အခြေအနေကို ပြောင်းလဲနိုင်ပါသည်။

Customer Options block အောက်တွင် ထပ်မံ block လေးခု ရှိပါသည်−

- **Memberships** — Customer စာရင်းသွင်းထားသော လက်ရှိ subscription များ
- **Payments** — Customer ၏ ငွေပေးချေမှုမှတ်တမ်း
- **Sites** — ဤ customer ပိုင်ဆိုင်သော site များ
- **Events** — ဤ customer နှင့်ဆိုင်သော event များ (အီးမေးလ်အတည်ပြုခြင်း၊ ငွေပေးချေမှုအချက်အလက်ပြောင်းလဲခြင်းစသည်)

မည်သည့်အရာ၏ အသေးစိတ်စာမျက်နှာသို့မဆို သွားရောက်ရန် ၎င်းဘေးရှိ မြှားသင်္ကေတ (→) ကို နှိပ်ပါ။ ဥပမာ site တစ်ခုဘေးရှိ မြှားကိုနှိပ်လျှင် membership ချိတ်ဆက်မှု၊ site အမျိုးအစားနှင့် active အခြေအနေတို့ကို ပြင်ဆင်နိုင်သည့် site edit စာမျက်နှာသို့ ရောက်ရှိပါသည်။

Membership များနှင့် payment များအတွက်လည်း အလားတူပင် — တစ်ခုချင်းစီသည် membership အခြေအနေ၊ ချိတ်ဆက်ထားသော customer၊ product များ၊ payment gateway setting များကို ပြင်ဆင်နိုင်ပြီး refund များပင် လုပ်ဆောင်နိုင်သည့် အသေးစိတ်စာမျက်နှာများသို့ ချိတ်ဆက်ထားပါသည်။

Customer စာမျက်နှာ၏ ညာဘက်တွင် အောက်ပါတို့ကို တွေ့ရပါမည်−

- **Last Login & IPs** — နောက်ဆုံးဝင်ရောက်ချိန်၊ IP လိပ်စာနှင့် နိုင်ငံ
- **Timestamps** — Customer အကောင့် ဖန်တီးချိန်နှင့် နောက်ဆုံးပြင်ဆင်ချိန်
- **Delete** — Customer ကို အပြီးတိုင်ဖျက်ရန် ရွေးချယ်စရာ

:::warning
Customer ကို ဖျက်ပစ်ခြင်းသည် အပြီးတိုင်ဖြစ်ပြီး ပြန်လည်ရယူ၍မရပါ။
:::
