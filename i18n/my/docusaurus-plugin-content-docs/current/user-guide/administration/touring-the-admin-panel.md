# Admin Panel ကို လှည့်လည်ကြည့်ရှုခြင်း {#touring-the-admin-panel}

Ultimate Multisite သည် ဖြစ်နိုင်သမျှ ရိုးရှင်းပြီး ကိုယ်တိုင်နားလည်လွယ်စေရန် ရည်ရွယ်ထားပါသည်။ သို့သော် ဤမျှ အားကောင်းသော ကိရိယာတစ်ခုအတွက် လမ်းညွှန်ကြည့်ရှုမှုရှိပါက ပိုမိုအထောက်အကူဖြစ်နိုင်သည်။ သင့်အနေဖြင့် အနေအထားသိရှိနိုင်ရန် admin စာမျက်နှာများကို အတူလျှောက်ကြည့်ကြပါစို့။

## Dashboard {#dashboard}

Ultimate Multisite **Dashboard** သည် ဝင်ငွေများ၊ site လှုပ်ရှားမှုများ၊ membership တိုးတက်မှု၊ လာရောက်သူအရေအတွက်များနှင့် ပထဝီဆိုင်ရာဒေတာများ အပါအဝင် အခြေခံအစီရင်ခံစာများနှင့် ခွဲခြမ်းစိတ်ဖြာမှုများကို ပြသသည်။

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

ဤသည်မှာ dashboard ၏ အပြည့်အစုံမြင်ကွင်းဖြစ်သည်။

![Ultimate Multisite Dashboard စာမျက်နှာအပြည့်](/img/admin/um-dashboard-full.png)

ထို့အပြင် dashboard ၏ အောက်ပိုင်းအပိုင်းမှာ—

![Ultimate Multisite Dashboard အောက်ပိုင်းအပိုင်း](/img/admin/um-dashboard-bottom.png)

network dashboard ၏ အပြည့်အစုံမြင်ကွင်းလည်း ဤတွင်ရှိသည်။

![Network Dashboard စာမျက်နှာအပြည့်](/img/admin/network-dashboard-full-page.png)

## Checkout Forms {#checkout-forms}

**Checkout Forms** သည် စိတ်ကြိုက် မှတ်ပုံတင်စာမျက်နှာများ ဖန်တီးနိုင်ရန် လိုအပ်သော လွတ်လပ်ပြောင်းလဲနိုင်မှုကို ပေးသည်။ field များကို စိတ်ကြိုက်ပြင်ဆင်နိုင်ပြီး custom class များနှင့် script များ ထည့်သွင်းနိုင်သည်။ ထို့အပြင် လာရောက်သူ၏ နိုင်ငံ သို့မဟုတ် တည်နေရာအပေါ် မူတည်၍ ဝင်ရောက်ခွင့်ကို ကန့်သတ်နိုင်သည်။

![Checkout Forms စာရင်း](/img/admin/checkout-forms-list.png)

## Products {#products}

**Products** အပိုင်းသည် သင့် network အတွက် plan၊ package နှင့် ဝန်ဆောင်မှုအမျိုးမျိုး ဖန်တီးရာနေရာဖြစ်သည်။ product အဆင့်တွင် ဈေးနှုန်းများ၊ billing အကြိမ်နှုန်းများနှင့် ကန့်သတ်ချက်များ၊ quota များကို သတ်မှတ်ပါ။

![Products စာရင်း](/img/admin/products-list.png)

## Memberships {#memberships}

**Memberships** စာမျက်နှာသည် သင့် network အတွင်းရှိ subscription အားလုံးကို ပြသသည်။ plan များ၊ product များ၊ billing ပမာဏများနှင့် အကြိမ်နှုန်းများ၊ payment မှတ်တမ်းနှင့် timestamp များ အပါအဝင် subscription အသေးစိတ်များကို စစ်ဆေးပြီး ပြင်ဆင်နိုင်သည်။

![Memberships စာရင်း](/img/admin/memberships-list.png)

## Payments {#payments}

**Payments** စာမျက်နှာသည် သင့် network တစ်ခုလုံးအတွင်းရှိ payment မှတ်တမ်းကို အမြန်ကြည့်ရှုနိုင်စေပြီး သတ်မှတ်ထားသော product များနှင့် ပမာဏများ အပါအဝင် transaction အသေးစိတ်အချက်အလက်များကို ပေးသည်။

![Payments စာရင်း](/img/admin/payments-list.png)

## Customers {#customers}

**Customers** စာမျက်နှာသည် သင့် network တွင် မှတ်ပုံတင်ထားသော member အားလုံးကို ပြသပြီး အမည်၊ email လိပ်စာ၊ နောက်ဆုံး login နှင့် ၎င်းတို့၏ subsite သို့ login ဝင်ရန် "switch to" link အပါအဝင် အခြေခံ user အချက်အလက်များကို ပြသည်။

![Customers စာရင်း](/img/admin/customers-list.png)

## Sites {#sites}

**Sites** စာမျက်နှာသည် site template များနှင့် customer ပိုင်ဆိုင်သော site များ အပါအဝင် သင့် network အတွင်းရှိ subsite အားလုံးကို စာရင်းပြုစုသည်။ mapped domain များ၊ ကန့်သတ်ချက်များနှင့် quota များ၊ plugin များနှင့် theme များ၊ site thumbnail များကို လွယ်ကူစွာ စီမံနိုင်သည်။

![Sites စာရင်း](/img/admin/sites-list.png)

## Domains {#domains}

**Domains** စာမျက်နှာသည် subsite များသို့ map လုပ်ထားသော custom domain များအတွက် သီးသန့်ဖြစ်သည်။ super admin အနေဖြင့် custom domain အမည်များကို ကိုယ်တိုင်ထည့်သွင်း သို့မဟုတ် map လုပ်နိုင်ပြီး DNS record များနှင့် log များကို ကြည့်ရှုနိုင်သည်။

![Domains စာရင်း](/img/admin/domains-list.png)

## Discount Codes {#discount-codes}

**Discount Codes** သည် သင့် end-user များကို discount ပေးရန် coupon code များ ဖန်တီးနိုင်စေသည်။ discount တန်ဖိုးကို သတ်မှတ်ပြီး သီးခြား user များ သို့မဟုတ် group များကို ရည်ရွယ်သတ်မှတ်နိုင်သည်။

![Discount Codes စာရင်း](/img/admin/discount-codes-list.png)

## Broadcasts {#broadcasts}

**Broadcasts** သည် သင့် subsite user များထံ ကြေညာချက်များ ဖြန့်ချိရန် သို့မဟုတ် ကိုယ်ပိုင်မက်ဆေ့ခ်ျများ ပို့ရန် ကိရိယာတစ်ခုဖြစ်သည်။ ၎င်းတို့ subscribe လုပ်ထားသော plan သို့မဟုတ် product အပေါ် မူတည်၍ သီးခြား user group များကို ရည်ရွယ်သတ်မှတ်နိုင်သည်။

![Broadcasts စာရင်း](/img/admin/broadcasts-list.png)

## Settings {#settings}

**Settings** စာမျက်နှာသည် Ultimate Multisite ကို configure လုပ်ရာနေရာဖြစ်ပြီး မှတ်ပုံတင်ခြင်း setting များ၊ payment များ၊ API နှင့် webhook များ၊ domain mapping နှင့် အခြား integration များ ပါဝင်သည်။

AI connector setting များတွင် လက်ရှိထောက်ပံ့ထားသော OAuth provider pool များဖြစ်သည့် Anthropic Max၊ OpenAI ChatGPT/Codex နှင့် Google AI Pro တို့ကို စာရင်းပြထားသည်။ provider card တစ်ခုစီသည် super admin များအား account များ ချိတ်ဆက်ခြင်း၊ သိမ်းဆည်းထားသော account များ refresh လုပ်ခြင်း၊ email ဖြင့် account များ ဖယ်ရှားခြင်းနှင့် sandboxed environment က browser redirect ကို ပိတ်ဆို့သည့်အခါ manual OAuth fallback အသုံးပြုခြင်းတို့ကို လုပ်နိုင်စေသည်။ Cursor Pro setup option များကို admin panel မှ ဖယ်ရှားပြီးဖြစ်သည်။

ChatGPT/Codex account များသည် လုပ်ဆောင်ချက်က tool များကို ခွင့်ပြုသည့်နေရာတွင် connector-backed tool အသုံးပြုမှုကို ထောက်ပံ့သည်။ ထို့ကြောင့် connector-supported operation များအပေါ် မူတည်သော admin workflow များသည် OpenAI account ချိတ်ဆက်ပြီးနောက် Codex tool အပြုအမူကို အသုံးပြုနိုင်သည်။

![Settings စာမျက်နှာ](/img/admin/settings-general.png)

ဤသည်မှာ general settings စာမျက်နှာ၏ အပြည့်အစုံမြင်ကွင်းဖြစ်သည်။

![Settings general စာမျက်နှာအပြည့်](/img/admin/settings-general-full.png)

![Settings emails စာမျက်နှာအပြည့်](/img/admin/settings-emails-full.png)

ထို့အပြင် payments settings စာမျက်နှာမှာ—

![Settings payments စာမျက်နှာအပြည့်](/img/admin/settings-payments-full.png)

ထို့အပြင် payments settings ၏ အောက်ပိုင်းအပိုင်းမှာ—

![Settings payments အောက်ပိုင်းအပိုင်း](/img/admin/settings-payments-bottom.png)

![Settings sites စာမျက်နှာအပြည့်](/img/admin/settings-sites-full.png)

## Events {#events}

**Events** စာမျက်နှာသည် သင့် network အတွင်းရှိ event နှင့် log အားလုံး၏ မှတ်တမ်းကို ထိန်းသိမ်းထားသည်။ plan ပြောင်းလဲမှုများ၊ signup များနှင့် အခြားလှုပ်ရှားမှုများကဲ့သို့ လုပ်ဆောင်ချက်များကို ခြေရာခံပြီး သင့် multisite ကို စောင့်ကြည့်ရန် အသုံးဝင်သည်။

![Events စာရင်း](/img/admin/events-list.png)

## Webhooks {#webhooks}

**Webhooks** သည် data ကို အခြား application တစ်ခုထံ ပေးပို့နိုင်စေသည်။ Ultimate Multisite မှ Zapier ကဲ့သို့ platform များသို့ data ပို့ရာတွင် အသုံးဝင်သည်။

![Webhooks စာရင်း](/img/admin/webhooks-list.png)
