# Admin Panel လေ့လာခြင်း

Ultimate Multisite က တတ်နိုင်သမျှ ရိုးရှင်းပြီး နားလည်လွယ်အောင် ဖန်တီးထားပေမယ့်၊ ဒီလောက် အစွမ်းထက်တဲ့ tool တစ်ခုအတွက် လမ်းညွှန်ပြသခြင်းက အထောက်အကူဖြစ်နိုင်ပါတယ်။ သင် အကျွမ်းတဝင်ဖြစ်အောင် admin စာမျက်နှာတွေကို အတူတူ လေ့လာကြည့်ရအောင်။

## Dashboard

Ultimate Multisite **Dashboard** မှာ ဝင်ငွေများ၊ site လှုပ်ရှားမှုများ၊ membership တိုးတက်မှု၊ ဧည့်သည်အရေအတွက်၊ နှင့် ပထဝီဝင်အချက်အလက်များ အပါအဝင် အခြေခံ report များနှင့် analytics များကို ပြသပါတယ်။

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

## Checkout Forms

**Checkout Forms** က စိတ်ကြိုက် မှတ်ပုံတင်စာမျက်နှာများ ဖန်တီးနိုင်တဲ့ လိုက်လျောညီထွေမှုကို ပေးပါတယ်။ field များကို စိတ်ကြိုက်ပြင်ဆင်ခြင်း၊ custom class များနှင့် script များ ထည့်သွင်းခြင်း၊ ဧည့်သည်ရဲ့ နိုင်ငံ သို့မဟုတ် တည်နေရာအပေါ် အခြေခံပြီး ဝင်ရောက်မှုကို ကန့်သတ်ခြင်းများ ပြုလုပ်နိုင်ပါတယ်။

![Checkout Forms list](/img/admin/checkout-forms-list.png)

## Products

**Products** အပိုင်းမှာ သင့် network အတွက် မတူညီတဲ့ plan များ၊ package များ၊ နှင့် ဝန်ဆောင်မှုများကို ဖန်တီးပါတယ်။ ဈေးနှုန်းများ၊ ငွေတောင်းခံမှုအကြိမ်ရေများ၊ နှင့် ကန့်သတ်ချက်များနှင့် quota များကို product အဆင့်မှာ သတ်မှတ်ပါ။

![Products list](/img/admin/products-list.png)

## Memberships

**Memberships** စာမျက်နှာမှာ သင့် network ရှိ subscription အားလုံးကို ပြသပါတယ်။ plan များ၊ product များ၊ ငွေတောင်းခံပမာဏနှင့် အကြိမ်ရေများ၊ ငွေပေးချေမှုမှတ်တမ်း၊ နှင့် အချိန်တံဆိပ်များ အပါအဝင် subscription အသေးစိတ်များကို ပြန်လည်စစ်ဆေးပြီး တည်းဖြတ်ပါ။

![Memberships list](/img/admin/memberships-list.png)

## Payments

**Payments** စာမျက်နှာက သင့် network တစ်ခုလုံးရဲ့ ငွေပေးချေမှုမှတ်တမ်းကို သီးခြား product များနှင့် ပမာဏများ အပါအဝင် အသေးစိတ် ငွေလွှဲအချက်အလက်များနှင့်အတူ အမြန်ကြည့်ရှုနိုင်စေပါတယ်။

![Payments list](/img/admin/payments-list.png)

## Customers

**Customers** စာမျက်နှာမှာ အမည်၊ email လိပ်စာ၊ နောက်ဆုံးဝင်ရောက်ချိန်၊ နှင့် သူတို့ရဲ့ subsite သို့ ဝင်ရောက်ရန် "switch to" link အပါအဝင် အခြေခံအသုံးပြုသူအချက်အလက်များနှင့်အတူ သင့် network မှာ မှတ်ပုံတင်ထားတဲ့ အဖွဲ့ဝင်အားလုံးကို ပြသပါတယ်။

![Customers list](/img/admin/customers-list.png)

## Sites

**Sites** စာမျက်နှာမှာ site template များနှင့် customer ပိုင်ဆိုင်တဲ့ site များ အပါအဝင် သင့် network ရှိ subsite အားလုံးကို စာရင်းပြုစုထားပါတယ်။ map လုပ်ထားတဲ့ domain များ၊ ကန့်သတ်ချက်များနှင့် quota များ၊ plugin များနှင့် theme များ၊ နှင့် site thumbnail များကို လွယ်ကူစွာ စီမံခန့်ခွဲပါ။

![Sites list](/img/admin/sites-list.png)

## Domains

**Domains** စာမျက်နှာက subsite များနှင့် map လုပ်ထားတဲ့ custom domain များအတွက် သီးသန့်ဖြစ်ပါတယ်။ super admin အနေနဲ့ custom domain name များကို ကိုယ်တိုင်ထည့်သွင်းခြင်း သို့မဟုတ် map လုပ်ခြင်း၊ DNS record များနှင့် log များကို ကြည့်ရှုခြင်းများ ပြုလုပ်နိုင်ပါတယ်။

![Domains list](/img/admin/domains-list.png)

## Discount Codes

**Discount Codes** က သင့်အသုံးပြုသူများကို လျှော့စျေးများ ပေးနိုင်ရန် coupon code များ ဖန်တီးနိုင်စေပါတယ်။ လျှော့စျေးတန်ဖိုးကို သတ်မှတ်ပြီး သီးခြားအသုံးပြုသူများ သို့မဟုတ် အုပ်စုများကို ပစ်မှတ်ထားပါ။

![Discount Codes list](/img/admin/discount-codes-list.png)

## Broadcasts

**Broadcasts** က သင့် subsite အသုံးပြုသူများထံ ကြေညာချက်များ ထုတ်ပြန်ခြင်း သို့မဟုတ် သီးသန့်မက်ဆေ့ချ်များ ပို့ခြင်းအတွက် tool တစ်ခုဖြစ်ပါတယ်။ သူတို့ subscribe လုပ်ထားတဲ့ plan သို့မဟုတ် product အပေါ် အခြေခံပြီး သီးခြားအသုံးပြုသူအုပ်စုများကို ပစ်မှတ်ထားပါ။

![Broadcasts list](/img/admin/broadcasts-list.png)

## Settings

**Settings** စာမျက်နှာမှာ Ultimate Multisite ကို ပြင်ဆင်သတ်မှတ်ပါတယ် — မှတ်ပုံတင်ခြင်း setting များ၊ ငွေပေးချေမှုများ၊ API နှင့် webhook များ၊ domain mapping၊ နှင့် အခြား integration များ။

![Settings page](/img/admin/settings-general.png)

## Events

**Events** စာမျက်နှာမှာ သင့် network ရှိ event များနှင့် log များအားလုံးကို မှတ်တမ်းတင်ထားပါတယ်။ plan ပြောင်းလဲမှုများ၊ အကောင့်ဖွင့်မှုများ၊ နှင့် အခြားလှုပ်ရှားမှုများကဲ့သို့သော လုပ်ဆောင်ချက်များကို ခြေရာခံပါတယ် — သင့် multisite ကို စောင့်ကြည့်ရန် အသုံးဝင်ပါတယ်။

![Events list](/img/admin/events-list.png)

## Webhooks

**Webhooks** က အခြား application တစ်ခုသို့ data ပို့ဆောင်နိုင်စေပါတယ်။ Ultimate Multisite မှ Zapier ကဲ့သို့သော platform များသို့ data ပို့ရန် အသုံးဝင်ပါတယ်။

![Webhooks list](/img/admin/webhooks-list.png)
