# စနစ် အီးမေးလ်များ စီမံခန့်ခွဲခြင်း

Ultimate Multisite သည် အကောင့်ဖွင့်ခြင်း အတည်ပြုချက်များ၊ ငွေပေးချေမှု ပြေစာများ၊ အသင်းဝင်မှု ပြောင်းလဲမှုများ စသည့် ဖြစ်ရပ်များအတွက် စနစ် အီးမေးလ်အမျိုးမျိုးကို ပေးပို့ပါသည်။ သင့်အမှတ်တံဆိပ်နှင့် ကိုက်ညီစေရန် ဤအီးမေးလ် နမူနာပုံစံများကို စိတ်ကြိုက်ပြင်ဆင်နိုင်ပါသည်။

## စနစ် အီးမေးလ်များ ကြည့်ရှုခြင်း

**Ultimate Multisite → Settings** သို့သွား၍ အီးမေးလ်ဆိုင်ရာ ဆက်တင်များကို ကြည့်ရှုရန် **Emails** tab ကို နှိပ်ပါ။

![Settings - Emails tab](/img/admin/settings-emails.png)

ထိုနေရာမှ အီးမေးလ် နမူနာပုံစံအားလုံးကို ကြည့်ရှုရန် **System Emails** စာမျက်နှာသို့ ဝင်ရောက်နိုင်ပါသည်။

![System Emails list](/img/admin/system-emails-list.png)

## အီးမေးလ် နမူနာပုံစံအသစ်များ ထည့်သွင်းခြင်း

Ultimate Multisite ဗားရှင်းအသစ်များနှင့် add-on များသည် အီးမေးလ် နမူနာပုံစံအသစ်များကို မှတ်ပုံတင်နိုင်ပါသည်။ ပဋိပက္ခများ မဖြစ်စေရန်အတွက် နမူနာပုံစံအသစ်များကို အလိုအလျောက် ထည့်သွင်းမပေးပါ — သင်ကိုယ်တိုင် ထည့်သွင်းရန် လိုအပ်ပါသည်။

အီးမေးလ် နမူနာပုံစံအသစ်များ ထည့်သွင်းရန်:

1. **System Emails** စာမျက်နှာသို့ သွားပါ
2. စာမျက်နှာ အပေါ်ဆုံးရှိ **Reset & Import** ခလုတ်ကို နှိပ်ပါ
3. ရရှိနိုင်သော နမူနာပုံစံများကို ကြည့်ရှုရန် **Import Emails** ရွေးချယ်ခွင့်ကို ဖွင့်ပါ
4. ထည့်သွင်းလိုသော နမူနာပုံစံများကို ရွေးချယ်ပြီး အတည်ပြုပါ

## အီးမေးလ် နမူနာပုံစံများ ပြန်လည်သတ်မှတ်ခြင်း

အီးမေးလ် နမူနာပုံစံတစ်ခုကို စိတ်ကြိုက်ပြင်ဆင်ထားပြီး မူလအကြောင်းအရာသို့ ပြန်လည်ပြောင်းလဲလိုပါက ရွေးချယ်စရာ နှစ်ခုရှိပါသည်:

1. **ဖျက်ပြီး ပြန်ထည့်သွင်းခြင်း** — စနစ် အီးမေးလ်ကို ဖျက်ပြီး ပြန်လည်ထည့်သွင်းပါ (ပေးပို့မှု စာရင်းအင်းများ ပျက်သွားပါမည်)
2. **ကိရိယာမှတစ်ဆင့် ပြန်လည်သတ်မှတ်ခြင်း** — **Reset & Import** ကိရိယာကို အသုံးပြု၍ **Reset** ရွေးချယ်ခွင့်ကို ဖွင့်ပြီး ပြန်လည်သတ်မှတ်လိုသော အီးမေးလ်များကို ရွေးချယ်ပါ

Reset ရွေးချယ်ခွင့်သည် ပေးပို့မှု စာရင်းအင်းများနှင့် အခြား metadata များကို ထိန်းသိမ်းထားပြီး မူလအကြောင်းအရာကို ပြန်လည်ရယူပေးသောကြောင့် ပိုမိုသင့်လျော်ပါသည်။
