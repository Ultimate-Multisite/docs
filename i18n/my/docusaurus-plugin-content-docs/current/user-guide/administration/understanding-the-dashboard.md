# Ultimate Multisite Dashboard ကို နားလည်ခြင်း

Ultimate Multisite ရဲ့ အဆင့်မြင့် ပြင်ဆင်မှု ရွေးချယ်စရာတွေနဲ့ လည်ပတ်မှုဆိုင်ရာ အချက်အလက်တွေကို စေ့စေ့လေ့လာမယ့်အရင် dashboard ကို အရင်ကြည့်ရအောင်။

WordPress plugin အများစုလိုပဲ Ultimate Multisite က **Ultimate Multisite** ဆိုတဲ့ network-level menu တစ်ခုကို ဖန်တီးပြီး အောက်မှာ sub-menu အများကြီး ထည့်ပေးထားပါတယ်။

## Network Dashboard Widgets များ

Ultimate Multisite က အသုံးဝင်တဲ့ widget တွေကို network-level dashboard ထဲမှာ ထည့်သွင်းပေးထားပါတယ်။ ဒီ dashboard ကို **My Sites → Network Admin → Dashboard** ကနေ သွားရောက်ကြည့်ရှုနိုင်ပါတယ်။ ဒီ widget တွေက network administrator အတွက် အချက်အလက်တွေနဲ့ လုပ်ဆောင်မှုတွေကို လွယ်ကူစွာ ရယူသုံးစွဲနိုင်အောင် ဒီဇိုင်းလုပ်ထားတာပါ။

![Ultimate Multisite widgets များပါဝင်သော Network Dashboard](/img/admin/network-dashboard-full.png)

### ပထမအဆင့်များ

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

ဒီ widget က Ultimate Multisite အသစ်တင်ပြီးတဲ့အခါ ပေါ်လာပြီး network administrator ကို ငွေပေးချေမှုနည်းလမ်း သတ်မှတ်ခြင်းနဲ့ test account ဖန်တီးခြင်းစတဲ့ အပိုလုပ်ဆောင်ရမယ့် အလုပ်တွေကို သတိပေးပါတယ်။

### အကျဉ်းချုပ်

![Summary widget](/img/admin/network-dashboard-summary.png)

Summary widget က တစ်နေ့တာ signup အရေအတွက်နဲ့ ဝင်ငွေကို ပြသပေးပါတယ်။ MRR (Monthly Recurring Revenue - လစဉ်ပုံမှန်ဝင်ငွေ) က ဝင်ငွေရှိတဲ့ membership တွေရဲ့ ခန့်မှန်းစုစုပေါင်းကို ပြသပါတယ်။

### လုပ်ဆောင်ချက်များ မှတ်တမ်း

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

ဒီ widget က Ultimate Multisite တစ်ခုလုံးမှာ ဖြစ်ပျက်နေတဲ့ အဖြစ်အပျက်တွေကို ပြသပါတယ်။ အဖြစ်အပျက်တွေထဲမှာ signup များ၊ ဖျက်သိမ်းမှုများ၊ site ဖန်တီးမှုများ၊ ဖျက်ပစ်မှုများနဲ့ အခြားလုပ်ဆောင်မှုများ ပါဝင်ပါတယ်။

ပိုပြီးအသေးစိတ်သိချင်ရင် documentation ရဲ့ Events အပိုင်းကို ဖတ်ရှုပါ။

### လက်ရှိအခြေအနေ

![Right Now widget](/img/admin/network-dashboard-right-now.png)

ဒီ utility widget က network ထဲက user တွေနဲ့ site တွေရဲ့ အကျဉ်းချုပ်ကို ပြသပါတယ်။ အပေါ်မှာရှိတဲ့ shortcut တွေက site အသစ် သို့မဟုတ် user အသစ်တွေကို တစ်ချက်နှိပ်ရုံနဲ့ ဖန်တီးနိုင်စေပါတယ်။

### သတင်းများနှင့် ဆွေးနွေးချက်များ

![News and Discussions widget](/img/admin/network-dashboard-news.png)

ဒီ widget က Ultimate Multisite ရဲ့ နောက်ဆုံးသတင်းတွေကို ဆွဲယူပြသပေးပါတယ်။ update များ၊ အရေးကြီး bug များနဲ့ လုံခြုံရေးပြင်ဆင်မှုတွေ အကြောင်း သိရှိဖို့ ဒီ widget ကို အမြဲစောင့်ကြည့်ပါ။

## Ultimate Multisite Dashboard

Network-level dashboard က network တစ်ခုလုံးရဲ့ အချက်အလက်တွေကို ပြသပေးသလို Ultimate Multisite dashboard (Ultimate Multisite menu ရဲ့ အပေါ်ဆုံးမှာ ရှိတဲ့) က service နဲ့ပတ်သက်တဲ့ အချက်အလက်တွေကို ပြသပေးပါတယ်။

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

### လစဉ်ပုံမှန်ဝင်ငွေ တိုးတက်မှု

MRR (Monthly Recurring Revenue - လစဉ်ပုံမှန်ဝင်ငွေ) chart က ၁၂ လ ပြက္ခဒိန်ကာလအတွင်း ဝင်ငွေကို လစဉ်ခွဲပြီး ပြသပေးပါတယ်။

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

ဝင်ငွေအသစ်တွေကို အပြာရောင်နဲ့ ပြသပြီး ဖျက်သိမ်းမှုတွေကို အနီရောင်နဲ့ ပြသပါတယ်။

### ရက်စွဲအပိုင်းအခြား ရွေးချယ်မှု

Customer အရေအတွက်နဲ့ လုပ်ဆောင်မှုပေါ်မူတည်ပြီး စာရင်းအင်းတွေက အများကြီးဖြစ်သွားနိုင်ပါတယ်။ Dashboard မှာပါတဲ့ ရက်စွဲအပိုင်းအခြား ရွေးချယ်မှုက network administrator ကို လိုချင်တဲ့ အချက်အလက်နဲ့ ကာလကိုပဲ အာရုံစိုက်ကြည့်နိုင်စေပါတယ်။

![Date range selector and stats](/img/admin/um-dashboard-date-range.png)

ရက်စွဲအပိုင်းအခြားက အောက်မှာရှိတဲ့ widget တွေမှာ ပြသမယ့် အချက်အလက်တွေရဲ့ ကာလကို ထိန်းချုပ်ပါတယ်။ ဝင်ငွေ၊ membership အသစ်များ၊ နိုင်ငံအလိုက် signup များ၊ form အလိုက် signup များနဲ့ အများဆုံးဝင်ရောက်ကြည့်ရှုတဲ့ site များ အပါအဝင်ပါ။

![Dashboard statistics widgets](/img/admin/um-dashboard-stats.png)

:::tip
**Most Visited Sites** widget အလုပ်လုပ်စေဖို့ **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** မှာ visit counting ကို ဖွင့်ထားပါ။
:::
