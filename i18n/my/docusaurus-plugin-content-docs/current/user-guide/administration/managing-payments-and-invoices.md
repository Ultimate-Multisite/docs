# ငွေပေးချေမှုများနှင့် Invoice များကို စီမံခန့်ခွဲခြင်း {#managing-payments-and-invoices}

## ငွေပေးချေမှု ဆက်တင်များ {#payment-settings}

ငွေပေးချေမှုများ စတင်လက်ခံမီ ငွေပေးချေမှုဆိုင်ရာ ဆက်တင်များကို သတ်မှတ်ထားရန် လိုအပ်ပါသည်။ **Ultimate Multisite → ဆက်တင်များ** သို့ သွားပြီး **ငွေပေးချေမှု** တက်ဘ်ကို နှိပ်ပါ။

![ငွေပေးချေမှု ဆက်တင်များ တက်ဘ်](/img/admin/settings-payments-top.png)

ဤသည်မှာ ငွေပေးချေမှု ဆက်တင်များ စာမျက်နှာ၏ အပြည့်အစုံမြင်ကွင်း ဖြစ်ပါသည်။

![ငွေပေးချေမှု ဆက်တင်များ စာမျက်နှာအပြည့်](/img/admin/settings-payments-full.png)

### အထွေထွေ ငွေပေးချေမှု ရွေးချယ်စရာများ {#general-payment-options}

အထွေထွေ ဆက်တင်များတွင် အောက်ပါတို့ကို သတ်မှတ်နိုင်သည်။

- **ငွေကြေး** — ငွေလွှဲပြောင်းမှုများအတွက် အသုံးပြုမည့် မူလငွေကြေး
- **ငွေကြေးနေရာချထားမှု** — ငွေကြေးသင်္ကေတ ပေါ်လာမည့်နေရာ (ပမာဏ၏ ရှေ့/နောက်)

![ငွေပေးချေမှု gateway ဆက်တင်များ](/img/admin/settings-payments-options.png)

### ငွေပေးချေမှု Gateway များ {#payment-gateways}

Ultimate Multisite သည် ငွေပေးချေမှု gateway အများအပြားကို ပံ့ပိုးပါသည်။ Payment ဆက်တင်များ တက်ဘ်မှ တစ်ခုချင်းစီကို ဖွင့်ပြီး သတ်မှတ်နိုင်သည်။

![ငွေပေးချေမှု gateway သတ်မှတ်ခြင်း](/img/admin/settings-payments-gateways.png)

ရရှိနိုင်သော gateway များတွင် အောက်ပါတို့ ပါဝင်သည်။

- **Stripe** — Stripe မှတစ်ဆင့် credit card ငွေပေးချေမှုများ
- **PayPal** — PayPal ငွေပေးချေမှုများ
- **Manual** — offline သို့မဟုတ် ကိုယ်ပိုင် ငွေပေးချေမှု လုပ်ဆောင်ခြင်းအတွက်

Gateway တစ်ခုချင်းစီတွင် API key များနှင့် အခြားဆက်တင်များ ထည့်သွင်းရန် ကိုယ်ပိုင်သတ်မှတ်မှု အပိုင်းရှိသည်။

![နောက်ထပ် gateway ဆက်တင်များ](/img/admin/settings-payments-gateways-2.png)

### Sandbox Mode {#sandbox-mode}

တိုက်ရိုက်အသုံးမပြုမီ သင်၏ ငွေပေးချေမှု ပေါင်းစည်းမှုကို စမ်းသပ်ရန် **Sandbox Mode** ကို ဖွင့်နိုင်သည်။ Sandbox mode အသက်ဝင်နေချိန်တွင် အမှန်တကယ် ငွေဖြတ်တောက်မှု မပြုလုပ်ပါ။

## ငွေပေးချေမှုများကို ကြည့်ရှုခြင်း {#viewing-payments}

သင့် network တစ်လျှောက်ရှိ ငွေလွှဲပြောင်းမှုအားလုံးကို ကြည့်ရန် Ultimate Multisite အောက်ရှိ **ငွေပေးချေမှုများ** စာမျက်နှာသို့ သွားပါ။

![ငွေပေးချေမှု စာရင်း](/img/admin/payments-list.png)

ငွေပေးချေမှုများကို အခြေအနေ (ပြီးစီးပြီး၊ စောင့်ဆိုင်းနေ၊ မအောင်မြင်၊ ပြန်အမ်းပြီး) ဖြင့် စစ်ထုတ်နိုင်ပြီး သီးခြား ငွေလွှဲပြောင်းမှုများကို ရှာဖွေနိုင်သည်။

စာကြောင်းအကြောင်းအရာများ၊ ဆက်စပ် membership၊ ဝယ်ယူသူ အချက်အလက်နှင့် ငွေပေးချေမှု gateway ဒေတာများ အပါအဝင် အသေးစိတ်အပြည့်အစုံကို ကြည့်ရန် ငွေပေးချေမှုတစ်ခုကို နှိပ်ပါ။

## Invoice များ {#invoices}

Ultimate Multisite သည် ငွေပေးချေမှုများအတွက် Invoice များကို အလိုအလျောက် ထုတ်ပေးနိုင်သည်။ Payment ဆက်တင်များမှ invoice template နှင့် နံပါတ်ပုံစံကို စိတ်ကြိုက်ပြင်ဆင်နိုင်သည်။

Invoice စိတ်ကြိုက်ပြင်ဆင်မှု ရွေးချယ်စရာများတွင် အောက်ပါတို့ ပါဝင်သည်။

- Invoice များပေါ်တွင် ပြသမည့် **ကုမ္ပဏီအမည်နှင့် လိပ်စာ**
- **Invoice နံပါတ်ပေးခြင်း** ပုံစံနှင့် အစဉ်လိုက်
- Invoice ခေါင်းစီးတွင် ပြသမည့် **Logo**
- စည်းကမ်းချက်များ၊ မှတ်ချက်များ သို့မဟုတ် ဥပဒေဆိုင်ရာ အချက်အလက်များအတွက် **စိတ်ကြိုက် footer စာသား**

Invoice template ကို စိတ်ကြိုက်ပြင်ဆင်ရန် **Ultimate Multisite → ဆက်တင်များ → Payment** သို့ သွားပြီး invoice ဆိုင်ရာ ဆက်တင်များကို ရှာပါ။
