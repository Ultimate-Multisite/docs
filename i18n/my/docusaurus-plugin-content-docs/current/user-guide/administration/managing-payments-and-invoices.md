# ငွေပေးချေမှုနှင့် ငွေတောင်းခံလွှာများ စီမံခန့်ခွဲခြင်း

## ငွေပေးချေမှု ဆက်တင်များ

ငွေပေးချေမှုများ စတင်လက်ခံရန် မလုပ်ဆောင်မီ ငွေပေးချေမှုဆိုင်ရာ ဆက်တင်များကို ပြင်ဆင်သတ်မှတ်ရန် လိုအပ်ပါသည်။ **Ultimate Multisite → Settings** သို့ သွားပြီး **Payment** tab ကို နှိပ်ပါ။

![Payment settings tab](/img/admin/settings-payments-top.png)

### အထွေထွေ ငွေပေးချေမှု ရွေးချယ်စရာများ

အထွေထွေ ဆက်တင်များတွင် အောက်ပါတို့ကို ပြင်ဆင်သတ်မှတ်နိုင်ပါသည်:

- **Currency** — ငွေလွှဲမှုများအတွက် အသုံးပြုမည့် မူလငွေကြေးအမျိုးအစား
- **Currency Position** — ငွေကြေးသင်္ကေတ ပေါ်လာမည့်နေရာ (ပမာဏရှေ့/နောက်)

![Payment gateway settings](/img/admin/settings-payments-options.png)

### ငွေပေးချေမှု Gateway များ

Ultimate Multisite သည် ငွေပေးချေမှု gateway အမျိုးမျိုးကို ပံ့ပိုးပေးပါသည်။ Payment settings tab မှ တစ်ခုချင်းစီကို ဖွင့်ပြီး ပြင်ဆင်သတ်မှတ်နိုင်ပါသည်။

![Payment gateway configuration](/img/admin/settings-payments-gateways.png)

ရရှိနိုင်သော gateway များမှာ:

- **Stripe** — Stripe မှတစ်ဆင့် ခရက်ဒစ်ကတ် ငွေပေးချေမှုများ
- **PayPal** — PayPal ငွေပေးချေမှုများ
- **Manual** — အော့ဖ်လိုင်း သို့မဟုတ် စိတ်ကြိုက် ငွေပေးချေမှု လုပ်ဆောင်ရန်

Gateway တစ်ခုချင်းစီတွင် API keys နှင့် အခြားဆက်တင်များ ထည့်သွင်းရန် ကိုယ်ပိုင် ပြင်ဆင်သတ်မှတ်မှု အပိုင်းရှိပါသည်။

![Additional gateway settings](/img/admin/settings-payments-gateways-2.png)

### Sandbox Mode

တကယ် အသုံးမပြုမီ ငွေပေးချေမှု ချိတ်ဆက်မှုကို စမ်းသပ်ရန် **Sandbox Mode** ကို ဖွင့်နိုင်ပါသည်။ Sandbox mode ဖွင့်ထားချိန်တွင် အမှန်တကယ် ငွေကောက်ခံမှုများ ပြုလုပ်မည်မဟုတ်ပါ။

## ငွေပေးချေမှုများ ကြည့်ရှုခြင်း

သင့် network တစ်ခုလုံးရှိ ငွေလွှဲမှုအားလုံးကို ကြည့်ရှုရန် Ultimate Multisite အောက်ရှိ **Payments** စာမျက်နှာသို့ သွားပါ။

![Payments list](/img/admin/payments-list.png)

ငွေပေးချေမှုများကို အခြေအနေအလိုက် (completed၊ pending၊ failed၊ refunded) စစ်ထုတ်နိုင်ပြီး သီးခြား ငွေလွှဲမှုများကို ရှာဖွေနိုင်ပါသည်။

ငွေပေးချေမှုတစ်ခုကို နှိပ်၍ line items များ၊ ဆက်စပ် membership၊ ဖောက်သည် အချက်အလက်များနှင့် ငွေပေးချေမှု gateway ဒေတာများ အပါအဝင် အသေးစိတ် အချက်အလက်အပြည့်အစုံကို ကြည့်ရှုနိုင်ပါသည်။

## ငွေတောင်းခံလွှာများ

Ultimate Multisite သည် ငွေပေးချေမှုများအတွက် ငွေတောင်းခံလွှာများကို အလိုအလျောက် ထုတ်ပေးနိုင်ပါသည်။ Payment settings မှ ငွေတောင်းခံလွှာ ပုံစံနှင့် နံပါတ်စဉ် ဖော်မတ်ကို စိတ်ကြိုက် ပြင်ဆင်နိုင်ပါသည်။

ငွေတောင်းခံလွှာ စိတ်ကြိုက်ပြင်ဆင်မှု ရွေးချယ်စရာများမှာ:

- ငွေတောင်းခံလွှာများတွင် ပြသမည့် **ကုမ္ပဏီအမည်နှင့် လိပ်စာ**
- **ငွေတောင်းခံလွှာ နံပါတ်စဉ်** ဖော်မတ်နှင့် အစဉ်လိုက်
- ငွေတောင်းခံလွှာ ခေါင်းစီးတွင် ပြသမည့် **Logo**
- စည်းကမ်းချက်များ၊ မှတ်ချက်များ သို့မဟုတ် ဥပဒေဆိုင်ရာ အချက်အလက်များအတွက် **စိတ်ကြိုက် footer စာသား**

ငွေတောင်းခံလွှာ ပုံစံကို စိတ်ကြိုက်ပြင်ဆင်ရန် **Ultimate Multisite → Settings → Payment** သို့ သွားပြီး ငွေတောင်းခံလွှာဆိုင်ရာ ဆက်တင်များကို ရှာပါ။
