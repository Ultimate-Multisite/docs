# Membership များ စီမံခန့်ခွဲခြင်း

Ultimate Multisite မှာ membership များကို စီမံခန့်ခွဲနိုင်ပြီး သင့် network ပေါ်ရှိ subscription အားလုံးကို ကြည့်ရှုနိုင်ပါတယ်။ Plan များနဲ့ product များ၊ site အရေအတွက်၊ ငွေတောင်းခံမှုပမာဏနဲ့ ကြိမ်နှုန်း၊ ငွေပေးချေမှုမှတ်တမ်း၊ အချိန်မှတ်တမ်းများ စတဲ့ subscription အသေးစိတ်အချက်အလက်များကို ပြန်လည်ကြည့်ရှုပြီး တည်းဖြတ်နိုင်ပါတယ်။

ဒီစာမျက်နှာကို ဝင်ရောက်ဖို့ Ultimate Multisite menu ထဲက **Memberships** tab ကို သွားပါ။

![Memberships list page](/img/admin/memberships-list.png)

Memberships စာမျက်နှာမှာ သင့်ဖောက်သည်များနဲ့ သူတို့ account များနဲ့ ဆက်စပ်နေတဲ့ product များစာရင်းကို မြင်ရပါမယ်။ Membership များကို status အလိုက် စစ်ထုတ်နိုင်ပါတယ် — active၊ pending၊ on hold၊ expired သို့မဟုတ် cancelled။

Membership တစ်ခုကို နှိပ်လိုက်ရင် membership အသေးစိတ်အချက်အလက်များ ဖြစ်တဲ့ ငွေတောင်းခံမှုအချက်အလက်၊ site သို့မဟုတ် visit ကန့်သတ်ချက်များ စတာတွေကို တည်းဖြတ်နိုင်တဲ့ စာမျက်နှာကို ဝင်ရောက်နိုင်ပါတယ်။

သင့် network မှာ အရင်က စာရင်းသွင်းထားတဲ့ user များနဲ့ ဆက်စပ်တဲ့ membership အသစ်များကိုလည်း ထည့်သွင်းနိုင်ပါတယ်။ **Add Membership** ကို နှိပ်ပြီး အသေးစိတ်အချက်အလက်များကို သတ်မှတ်ပါ။

![Add Membership modal](/img/admin/membership-add-modal.png)

ပြီးရင် **Create Membership** ကို နှိပ်ပါ။ Membership အသစ်က memberships စာရင်းမှာ ပေါ်လာပါလိမ့်မယ်။
