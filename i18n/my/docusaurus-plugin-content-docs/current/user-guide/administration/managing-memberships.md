# အသင်းဝင်မှုများကို စီမံခြင်း {#managing-memberships}

Ultimate Multisite တွင် အသင်းဝင်မှုများကို စီမံနိုင်ပြီး သင့် network ပေါ်ရှိ စာရင်းသွင်းမှုအားလုံးကို ဝင်ရောက်ကြည့်ရှုနိုင်သည်။ အစီအစဉ်များနှင့် ထုတ်ကုန်များ၊ ဆိုက်အရေအတွက်၊ ငွေတောင်းခံပမာဏနှင့် ကြိမ်နှုန်း၊ ငွေပေးချေမှုမှတ်တမ်း၊ အချိန်မှတ်တမ်းများ စသည့် စာရင်းသွင်းမှုအသေးစိတ်များကို ပြန်လည်စစ်ဆေးပြီး ပြင်ဆင်နိုင်သည်။

ဤစာမျက်နှာကို ဝင်ရောက်ရန် Ultimate Multisite မီနူးရှိ **အသင်းဝင်မှုများ** တက်ဘ်သို့ သွားပါ။

![အသင်းဝင်မှုများ စာရင်းစာမျက်နှာ](/img/admin/memberships-list.png)

အသင်းဝင်မှုများ စာမျက်နှာတွင် သင့်ဖောက်သည်များနှင့် ၎င်းတို့၏ အကောင့်များနှင့် ဆက်စပ်နေသော ထုတ်ကုန်များစာရင်းကို မြင်နိုင်သည်။ အသင်းဝင်မှုများကို အခြေအနေအလိုက် စစ်ထုတ်နိုင်သည်- လက်ရှိအသုံးပြုနေသည်၊ ဆိုင်းငံ့ထားသည်၊ ခေတ္တရပ်ထားသည်၊ သက်တမ်းကုန်သည်၊ သို့မဟုတ် ပယ်ဖျက်ထားသည်။

အခမဲ့အသင်းဝင်မှုများကို **တစ်သက်တာ** အသင်းဝင်မှုများအဖြစ် သတ်မှတ်သည်။ ၎င်းတို့တွင် သက်တမ်းကုန်ဆုံးရက် မရှိသလို အလိုအလျောက် သက်တမ်းကုန်မည်မဟုတ်ပါ။ အခမဲ့အစီအစဉ်တစ်ခုသည် စမ်းသုံးကာလ သို့မဟုတ် သတ်မှတ်ကာလပြီးနောက် ပြီးဆုံးသင့်ပါက စမ်းသုံးကာလပါသော ငွေပေးချေထုတ်ကုန်ကို အသုံးပြုပါ၊ အသင်းဝင်မှုကို ကိုယ်တိုင်ပယ်ဖျက်ပါ၊ သို့မဟုတ် ဝင်ရောက်ခွင့် ပြောင်းလဲသင့်သည့်အချိန်တွင် ဖောက်သည်ကို အခြားအစီအစဉ်သို့ ရွှေ့ပါ။

အသင်းဝင်မှုတစ်ခုကို နှိပ်ပါက အသင်းဝင်မှုအသေးစိတ်များ — ငွေတောင်းခံအချက်အလက်၊ ဆိုက် သို့မဟုတ် လည်ပတ်မှုကန့်သတ်ချက်များနှင့် အခြားအရာများ — ကို ပြင်ဆင်နိုင်သည့် စာမျက်နှာသို့ ဝင်ရောက်နိုင်သည်။

![အသင်းဝင်မှု ပြင်ဆင်ရေး အင်တာဖေ့စ်](/img/admin/membership-edit.png)

ဤသည်မှာ အသင်းဝင်မှု ပြင်ဆင်ရေးစာမျက်နှာ၏ အပြည့်အစုံ မြင်ကွင်းဖြစ်သည်-

![အသင်းဝင်မှု ပြင်ဆင်ရေး အပြည့်အစုံ အင်တာဖေ့စ်](/img/admin/membership-edit-full.png)

သင့် network တွင် ယခင်က မှတ်ပုံတင်ထားသော အသုံးပြုသူများနှင့် ဆက်စပ်သော အသင်းဝင်မှုအသစ်များကိုလည်း ထည့်နိုင်သည်။ **အသင်းဝင်မှု ထည့်ရန်** ကို နှိပ်ပြီး အသေးစိတ်များကို သတ်မှတ်ပါ။

![အသင်းဝင်မှု ထည့်ရန် modal](/img/admin/membership-add-modal.png)

ထို့နောက် **အသင်းဝင်မှု ဖန်တီးရန်** ကို နှိပ်ပါ။ အသင်းဝင်မှုအသစ်သည် အသင်းဝင်မှုများစာရင်းတွင် ပေါ်လာပါမည်။
