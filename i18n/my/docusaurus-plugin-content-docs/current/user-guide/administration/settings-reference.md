# Settings Reference

ဤစာမျက်နှာသည် Ultimate Multisite ၏ နေ့စဉ် စီမံခန့်ခွဲမှုနှင့် မကြာသေးမီက ပြောင်းလဲခဲ့သော အပြုအမူဆိုင်ရာ အချက်အလက်များကို မှတ်တမ်းတင်ထားပါသည်။

## Other Options

**Other Options** နေရာသည် **Ultimate Multisite > Settings > Login & Registration** အောက်တွင် ပေါ်လာပါသည်။

| Setting | Description |
|---|---|
| **Enable Jumper** | Admin နေရာတွင် Jumper အမြန်လမ်းညွှန် ကိရိယာကို ပြသပေးပါသည်။ ၎င်းကို အသုံးပြု၍ Ultimate Multisite မျက်နှာပြင်များ၊ network object များ၊ နှင့် ပံ့ပိုးထားသော admin ဦးတည်ရာနေရာများသို့ တိုက်ရိုက် ခုန်ဝင်နိုင်ပါသည်။ ဤ shortcut ကို မမြင်ချင်ပါက ပိတ်ထားနိုင်ပါသည်။ |

## Error reporting and telemetry

ယခင် error-reporting opt-in setting ကို settings စာမျက်နှာမှ ဖယ်ရှားလိုက်ပါပြီ။ Anonymous telemetry ကို ပိတ်ထားပြီး ၎င်းကို ဖွင့်ရန် UI toggle မရှိတော့ပါ။

settings စာမျက်နှာအတွက် internal runbooks သို့မဟုတ် screenshots များ ထိန်းသိမ်းထားပါက၊ စီမံခန့်ခွဲသူများ မရှိတော့သည့် setting တစ်ခုကို ရှာဖွေနေခြင်းမရှိစေရန် ယခင် error-reporting opt-in field ကို ဖယ်ရှားပေးပါ။

## Import/Export settings

**Import/Export** settings tab သည် မည်သည့် setting များကို ထိန်းချုပ်ထားကြောင်း ဖော်ပြပြီး site နှင့် network archive များအတွက် **Ultimate Multisite > Site Export** သို့ တိုက်ရိုက်ချိတ်ဆက်ထားပါသည်။ import/export configuration အတွက် settings tab ကို အသုံးပြုပါ၊ single-site export/import workflow အတွက် **Tools > Export & Import** ကို အသုံးပြုပါ၊ နှင့် အပြည့်အစုံ Network Export archive လိုအပ်သည့်အခါ Site Export tool ကို အသုံးပြုပါ။
