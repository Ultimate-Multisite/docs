# ဆိုဒ်များ စီမံခန့်ခွဲခြင်း

ဆိုဒ်များ (သို့မဟုတ် subsites) သည် သင့် WaaS လုပ်ငန်း၏ အဓိက အစိတ်အပိုင်းဖြစ်ပါသည်။ Ultimate Multisite တွင် ဆိုဒ် အမျိုးအစား ၃ မျိုးရှိပါသည်:

- **Customer-owned** — သတ်မှတ်ထားသော ဝယ်ယူသူများပိုင်ဆိုင်သော ဆိုဒ်များ
- **Site Templates** — ဝယ်ယူသူများ အစပြုရန် ရွေးချယ်နိုင်သော ကြိုတင်သတ်မှတ်ထားသည့် ဆိုဒ်များ
- **Main Site** — သင့် network ၏ အဓိကဆိုဒ်

## ဆိုဒ်များ ကြည့်ရှုခြင်း

သင့် network ရှိ subsites အားလုံးကို ကြည့်ရှုရန် **Ultimate Multisite → Sites** သို့ သွားပါ။ ဆိုဒ်တစ်ခုစီကို အမျိုးအစားအလိုက် အမှတ်အသားပြထားပြီး All Sites၊ Customer-Owned၊ Templates သို့မဟုတ် Pending အလိုက် စစ်ထုတ်နိုင်ပါသည်။

![Sites list page](/img/admin/sites-list.png)

## ဆိုဒ်အသစ် ထည့်သွင်းခြင်း

ဆိုဒ်အသစ်တစ်ခု ဖန်တီးရန် **Add Site** ခလုတ်ကို နှိပ်ပါ။ အောက်ပါအချက်များကို ဖြည့်သွင်းရမည်ဖြစ်ပါသည်:

- **Site Title** — ဆိုဒ်အသစ်၏ အမည်
- **Site URL** — ဆိုဒ်အတွက် domain/path
- **Site Type** — customer site၊ template သို့မဟုတ် သာမန်ဆိုဒ် ဖြစ်မဖြစ်

**Copy Site** ရွေးချယ်မှုဖြင့် ရှိနေပြီးသား site template ကို အခြေခံ၍ ဆိုဒ်အသစ်တစ်ခု ဖန်တီးနိုင်ပါသည်။ ဖွင့်ထားပါက အစပြုရန် မည်သည့် template ကို အသုံးပြုမည်ကို ရွေးချယ်နိုင်ပါသည်။ မီဒီယာဖိုင်များပါ ပါဝင်စေရန် **Copy Media on Duplication** ကို ဖွင့်ထားကြောင်း သေချာပါစေ။

## ရှိနေပြီးသား ဆိုဒ်တစ်ခုကို စီမံခန့်ခွဲခြင်း

**Edit Site** စာမျက်နှာကို ဖွင့်ရန် မည်သည့်ဆိုဒ်တွင်မဆို **Manage** ကို နှိပ်ပါ။ ဤနေရာတွင် အောက်ပါတို့ကို တွေ့ရမည်ဖြစ်ပါသည်:

### အခြေခံအချက်အလက်များ

ဆိုဒ်အမည်၊ အမျိုးအစား၊ site ID နှင့် ဖော်ပြချက်တို့ ပါဝင်ပါသည်။ ထို့အပြင် mapped domain၊ ဆက်စပ်နေသော membership နှင့် ဆိုဒ်ပိုင်ဆိုင်သည့် customer account တို့ကိုလည်း တွေ့ရမည်ဖြစ်ပါသည်။

### ဆိုဒ် ရွေးချယ်စရာများ

ဆိုဒ်၏ စွမ်းဆောင်ရည်နှင့် ကန့်သတ်ချက်များကို ပြင်ဆင်သတ်မှတ်ပါ:

- **Visit limits** — ဆိုဒ်လည်ပတ်မှု အများဆုံး အရေအတွက်
- **User account limits** — user role တစ်ခုစီအတွက် ကန့်သတ်ချက်များ
- **Disk space** — ဆိုဒ်အတွက် သိုလှောင်မှု ပမာဏ
- **Custom domain** — ဤဆိုဒ်အတွက် domain mapping ဖွင့်ခြင်း
- **Plugin and theme visibility** — မည်သည့် plugins နှင့် themes များကို မြင်နိုင်၊ ဝှက်ထား သို့မဟုတ် ကြိုတင် activate လုပ်ထားမည်ကို ထိန်းချုပ်ခြင်း

မူရင်းအတိုင်းဆိုပါက ဆိုဒ်များသည် membership အဆင့်တွင် သတ်မှတ်ထားသော ကန့်သတ်ချက်များကို လိုက်နာပါသည်။ ဆိုဒ်အဆင့်တွင် ကန့်သတ်ချက်များ သတ်မှတ်ပါက membership ဆက်တင်များကို အစားထိုးမည်ဖြစ်ပါသည်။

### ချိတ်ဆက်မှုများ

ဆိုဒ် ရွေးချယ်စရာများ အောက်တွင် အောက်ပါတို့နှင့် ပတ်သက်သော အချက်အလက်များ တွေ့ရမည်ဖြစ်ပါသည်:

- ဆိုဒ်နှင့် ချိတ်ဆက်ထားသော **Mapped domains**
- ဆိုဒ် ပါဝင်နေသော **Membership**
- ဆိုဒ်နှင့် ချိတ်ဆက်ထားသော **Customer account**

### ညာဘက် Sidebar

ညာဘက်ခြမ်းတွင် အောက်ပါတို့ကို လုပ်ဆောင်နိုင်ပါသည်:

- toggle ခလုတ်ဖြင့် **ဆိုဒ်ကို ဖွင့်/ပိတ် ပြုလုပ်ခြင်း**
- **ဆိုဒ် အမျိုးအစား ပြောင်းလဲခြင်း** သို့မဟုတ် ပိုင်ဆိုင်မှု ပြန်လည်သတ်မှတ်ခြင်း
- **ဆိုဒ် ပုံ/thumbnail သတ်မှတ်ခြင်း** (site templates အတွက် frontend တွင် ပြသပါသည်)
- **ဆိုဒ်ကို အပြီးအပိုင် ဖျက်ပစ်ခြင်း**

:::warning
ဆိုဒ်ဖျက်ခြင်းသည် ပြန်လည်ရယူ၍ မရနိုင်ပါ။ ဆိုဒ်နှင့် ၎င်း၏ အကြောင်းအရာအားလုံးကို အပြီးအပိုင် ဖယ်ရှားလိုက်မည်ဖြစ်ပါသည်။
:::
