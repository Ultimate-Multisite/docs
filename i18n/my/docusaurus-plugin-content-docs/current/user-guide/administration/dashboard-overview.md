# Dashboard ခြုံငုံသုံးသပ်ချက်

Ultimate Multisite တွင် Ultimate Multisite မီနူးအောက်၌ သီးသန့် **Dashboard** တစ်ခုရှိပြီး Network Dashboard သို့လည်း widget များထပ်ထည့်ပေးပါသည်။ ဤဆောင်းပါးတွင် နေရာနှစ်ခုစလုံးတွင် သင်မြင်ရမည့်အရာများကို ရှင်းပြထားပါသည်။

## Network Dashboard

Ultimate Multisite သည် **My Sites → Network Admin → Dashboard** တွင်ရှိသော network-level dashboard ထဲသို့ အသုံးဝင်သော widget များထည့်သွင်းပေးပါသည်။

![Ultimate Multisite widget များပါဝင်သော Network Dashboard](/img/admin/network-dashboard-full.png)

widget တစ်ခုချင်းစီက ဘာတွေပေးလဲဆိုတာ ဒီမှာကြည့်ပါ -

**First Steps:** အသစ်ထည့်သွင်းထားသောစနစ်တွင် ပေါ်လာပြီး ငွေပေးချေမှုနည်းလမ်းသတ်မှတ်ခြင်းနှင့် စမ်းသပ်အကောင့်ဖန်တီးခြင်းကဲ့သို့ setup လုပ်ငန်းများပြီးစီးရန် သတိပေးပါသည်။

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Summary:** ထိုနေ့အတွက် စာရင်းသွင်းမှုအရေအတွက်နှင့် ဝင်ငွေကိုပြသည်။ MRR (Monthly Recurring Revenue) သည် ဝင်ငွေရှိသော membership များရှိ ဝယ်ယူသူများအားလုံး၏ ခန့်မှန်းစုစုပေါင်းကိုညွှန်ပြသည်။

![Summary widget](/img/admin/network-dashboard-summary.png)

**Activity Stream:** Ultimate Multisite တစ်လျှောက်ဖြစ်ပွားသော အဖြစ်အပျက်များ — စာရင်းသွင်းမှုများ၊ ပယ်ဖျက်မှုများ၊ site ဖန်တီးမှုများ၊ ဖျက်သိမ်းမှုများနှင့် အခြားလှုပ်ရှားမှုများကို သတင်းပို့ပါသည်။

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** network အတွင်းရှိ အသုံးပြုသူများနှင့် site များ၏ အကျဉ်းချုပ်ကိုပြသပြီး site အသစ်များ သို့မဟုတ် အသုံးပြုသူအသစ်များဖန်တီးရန် ဖြတ်လမ်းများပါဝင်သည်။

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** အပ်ဒိတ်များ၊ အရေးကြီး bug များနှင့် လုံခြုံရေးပြင်ဆင်မှုများအပါအဝင် နောက်ဆုံးရ Ultimate Multisite သတင်းများကို ရယူပြသပေးသည်။

![News widget](/img/admin/network-dashboard-news.png)

## Ultimate Multisite Dashboard

Network dashboard သည် network တစ်ခုလုံးအကြောင်း ခြုံငုံအချက်အလက်များပြသနေချိန်တွင်၊ Ultimate Multisite dashboard (Ultimate Multisite မီနူး၏ အထက်ဆုံးအဆင့်တွင်) သည် ဝန်ဆောင်မှုအကြောင်း အချက်အလက်များပြသပါသည်။

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Monthly Recurring Revenue Growth:** လ ၁၂ လ ပြက္ခဒိန်ကာလတစ်လျှောက် ဝင်ငွေ၏ လစဉ်ခွဲခြမ်းစိတ်ဖြာချက်ကိုပြသည်။ ဝင်ငွေအသစ်ကို အပြာရောင်ဖြင့်ပြသပြီး ပယ်ဖျက်မှုများကို အနီရောင်ဖြင့်ပြသည်။

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**Date-range Support:** သင်လိုအပ်သော အချက်အလက်များကိုအာရုံစိုက်ရန် dashboard ဒေတာကို အချိန်ကာလအလိုက် စစ်ထုတ်ပါ။ ရက်စွဲအပိုင်းအခြားသည် အောက်ဖော်ပြပါ widget များတွင်ပြသသည့်အရာကို ထိန်းချုပ်ပါသည်။

![Date range နှင့် dashboard stats](/img/admin/um-dashboard-date-range.png)

Dashboard widget များတွင် ပါဝင်သည်များ -

- **Revenue** — ကာလအတွင်း စုစုပေါင်းဝင်ငွေ၊ MRR နှင့် ထုတ်ကုန်အလိုက် ခွဲခြမ်းစိတ်ဖြာချက်များ
- **New Memberships** — ရွေးချယ်ထားသောကာလအတွင်း membership အသစ်မည်မျှဖန်တီးခဲ့သည်
- **Signups by Countries** — စာရင်းသွင်းမှုများ၏ ပထဝီဝင်ခွဲခြမ်းစိတ်ဖြာချက်
- **Signups by Forms** — checkout form တစ်ခုချင်းစီမှတစ်ဆင့် ဝယ်ယူသူမည်မျှ စာရင်းသွင်းခဲ့သလဲနှင့် admin များက ထည့်သွင်းခဲ့သည့်အရေအတွက်
- **Most Visited Sites** — သင့် network ပေါ်ရှိ အလည်အပတ်အများဆုံး site များ (**Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** တွင် visit counting ဖွင့်ထားရန်လိုအပ်သည်)

![Dashboard statistics](/img/admin/um-dashboard-stats.png)
