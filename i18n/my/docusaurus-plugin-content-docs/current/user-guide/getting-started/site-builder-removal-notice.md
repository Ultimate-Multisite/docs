# Site Builder Mode ကို ဖယ်ရှားခြင်းဆိုင်ရာ အသိပေးချက်

**Superdav AI Agent v1.12.0 တွင် Site Builder mode ကို ဖယ်ရှားလိုက်ပါပြီ။** Site Builder mode ကို အသုံးပြုနေခဲ့သူများအနေဖြင့် theme ဖန်တီးခြင်းနှင့် site စနစ်တကျ တည်ဆောက်ခြင်းများအတွက် **Setup Assistant agent** သို့ ပြောင်းရွှေ့အသုံးပြုသင့်ပါသည်။

## ဘာတွေ ဖြစ်ခဲ့ပါသလဲ။

### Site Builder Mode (ယခင်ပုံစံ)

Site Builder mode သည် အောက်ပါအရာများအတွက် အဆင့်ဆင့် လမ်းညွှန်မှု (wizard-based) ပုံစံတစ်ခု ဖြစ်ခဲ့သည်-

- template များမှ site များ ဖန်တီးခြင်း
- အခြေခံ setting များကို စနစ်တကျ ချိန်ညှိခြင်း
- theme တစ်ခု ရွေးချယ်ခြင်း
- အစပိုင်း အကြောင်းအရာများကို စီစဉ်ပေးခြင်း

### ဘာက အစားထိုးခဲ့ပါသလဲ။

**Setup Assistant agent** သည် ယခုအခါ Site Builder လုပ်ဆောင်ချက်အားလုံးကို အောက်ပါအရာများဖြင့် ကိုင်တွယ်ပေးနေပါပြီ-

- ပိုမိုပြောင်းလွယ်ပြင်လွယ်ရှိသော၊ agent မှ လမ်းညွှန်ပေးသည့် setup များ
- ပိုမိုကောင်းမွန်သည့် theme စိတ်ကြယ်မှု (customization) ရွေးချယ်မှုများ
- Theme Builder onboarding နှင့် ပေါင်းစပ်လုပ်ဆောင်မှု
- အနာဂတ် session များအတွက် site_brief ကို အမြဲတမ်း မှတ်ထားနိုင်ခြင်း

## Site Builder Mode ကို အသုံးပြုခဲ့သူများအတွက်

### သင့် site များ ဘေးကင်းပါတယ်

- Site Builder mode ဖြင့် ဖန်တီးထားသော လက်ရှိ site များသည် ဆက်လက် အလုပ်လုပ်နေပါမည်။
- အချက်အလက်များ ပျောက်ဆုံးခြင်း သို့မဟုတ် site အနှောင့်အယံများ မရှိပါ။
- သင်သည် သင့် site များကို ပုံမှန်အတိုင်း စီမံခန့်ခွဲနိုင်ပါသည်။

### Setup Assistant Agent သို့ ပြောင်းရွှေ့ပါ

site အသစ် တည်ဆောက်ရန် သို့မဟုတ် theme ပြောင်းလဲရန်အတွက် Setup Assistant agent ကို အသုံးပြုပါ-

```
"Help me set up a new site"
```

သို့မဟုတ်

```
"Start the Theme Builder onboarding"
```

Setup Assistant agent သည် ပိုမိုပြောင်းလွယ်ပြင်လွယ်ရှိမှုဖြင့် ယခင်လုပ်ဆောင်ချက်များကို ပေးစွမ်းနိုင်ပါသည်။

## နှိုင်းယှဉ်ချက်- Site Builder နှင့် Setup Assistant

| Feature | Site Builder (ဖယ်ရှားပြီး) | Setup Assistant (အသစ်) |
|---------|----------------------|----------------------|
| Setup method | အဆင့်ဆင့် လမ်းညွှန်ပုံစံ (Wizard form) | Agent နှင့် စကားပြောဆိုမှု (Agent conversation) |
| Theme selection | ကြိုသတ်မှတ်ထားသော template များ | စိတ်ကြယ်စွာ ဖန်တီးပေးခြင်း (Custom generation) |
| Customization | ကန့်သတ်ထားသော ရွေးချယ်မှုများ | အပြည့်အဝ ဒီဇိုင်းစနစ် (Full design system) |
| Site brief | သိမ်းဆည်းမထားပါ | အမြဲတမ်း မှတ်ဉာဏ် (Persistent memory) |
| Future sessions | setup ကို ပြန်လုပ်ရမည် | သိမ်းထားသော site_brief ကို အသုံးပြုနိုင်သည် |
| Flexibility | အမြဲတမ်း လုပ်ဆောင်ပုံ (Fixed workflow) | လိုက်လျောညီထွေရှိသော စကားပြောဆိုမှု (Adaptive conversation) |

## Setup Assistant Agent သို့ ပြောင်းရွှေ့ခြင်း

### site အသစ်များအတွက်

Site Builder mode ကို အသုံးပြုမည့်အစား-

၁။ တောင်းဆိုပါ- "Help me set up a new site"
၂။ Setup Assistant agent သည် အောက်ပါအရာများမှတစ်ဆင့် လမ်းညွှန်ပေးပါမည်-
   - Site ရည်ရွယ်ချက်နှင့် ပန်းတိုင်များ
   - ပစ်မှတ်ထားသော ပရိသတ်
   - Brand ရုပ်ပုံလက္ခဏာ
   - Theme ဖန်တီးပေးခြင်း
   - အစပိုင်း စနစ်တကျ ချိန်ညှိမှုများ

### လက်ရှိ site များအတွက်

Site Builder mode မှ ရရှိလာသော လက်ရှိ site တစ်ခုရှိပါက-

၁။ အတိုင်းအတာအတိုင်း ဆက်လက်အသုံးပြုနိုင်ပါသည်။
၂။ theme ကို update လုပ်လိုပါက- "Redesign my site" ဟု တောင်းဆိုပါ။
၃။ Setup Assistant agent သည် theme အသစ်တစ်ခု ဖန်တီးပေးရန် ကူညီပေးပါမည်။
၄။ သင့် site data များသည် မပြောင်းလဲဘဲ ရှိနေပါမည်။

### Theme ပြောင်းလဲမှုများအတွက်

Site Builder mode ၏ theme ရွေးချယ်မှုအစား-

၁။ တောင်းဆိုပါ- "Change my theme"
၂။ Setup Assistant agent သည်-
   - သင့်ဒီဇိုင်းနှစ်သက်မှုများကို မေးမြန်းပါမည်။
   - စိတ်ကြယ်သော theme အသစ်တစ်ခု ဖန်တီးပေးပါမည်။
   - သင့် site ပေါ်တွင် အသက်သွင်းပေးပါမည်။

## အဓိက ကွာခြားချက်များ

### Site Builder Mode

```
၁။ template တစ်ခု ရွေးချယ်ပါ
၂။ theme တစ်ခု ရွေးချယ်ပါ
၃။ အခြေခံ setting များကို ချိန်ညှိပါ
၄။ ပြီးပါပြီ
```

### Setup Assistant Agent

```
၁။ သင့် site ရည်ရွယ်ချက်ကို ဖော်ပြပါ
၂။ သင့်ပစ်မှတ်ပရိသတ်ကို သတ်မှတ်ပါ
၃။ ဒီဇိုင်းနှစ်သက်မှုများကို ရွေးချယ်ပါ
၄။ Agent က စိတ်ကြယ်သော theme ကို ဖန်တီးပေးပါမည်
၅။ Agent က theme ကို အသက်သွင်းပေးပါမည်
၆။ အနာဂတ် session များအတွက် site brief ကို သိမ်းဆည်းပေးမည်
```

## Setup Assistant Agent ၏ အကျိုးကျေးဇူးများ

### ပိုမိုပြောင်းလွယ်ပြင်လွယ်ရှိခြင်း

- သင့် site ကို သဘာဝဘာသာစကားဖြင့် ဖော်ပြနိုင်ခြင်း
- စိတ်ကြယ်သော အကြံပြုချက်များ ရရှိနိုင်ခြင်း
- သင့်လိုအပ်ချက်များနှင့် လိုက်လျောညီထွေဖြစ်ခြင်း

### ပိုမိုကောင်းမွန်သော စိတ်ကြယ်မှု

- စိတ်ကြယ်သော theme ဖန်တီးပေးခြင်း
- ဒီဇိုင်းစနစ် ဆုံးဖြတ်ချက်များ
- အမြဲတမ်း ဒီဇိုင်း token များ

### အမြဲတမ်း မှတ်ဉာဏ်

- သင့် site_brief ကို သိမ်းဆည်းထားပေးမည်
- အနာဂတ် agent များက သင့် site ကို နားလည်ပေးမည်
- setup အချက်အလက်များကို ထပ်ခါထပ်ခါ ပြောပြစရာ မလိုတော့ပါ

### ပေါင်းစပ်လုပ်ဆောင်မှု (Integrated Workflow)

- Theme Builder onboarding
- Design System Aesthetics skill
- Ability Visibility controls
- အားလုံးသည် ချောမွေ့စွာ အတူတကွ လုပ်ဆောင်နိုင်ခြင်း

## ပြဿနာဖြေရှင်းခြင်း (Troubleshooting)

### Site Builder mode ကို ရှာမတွေ့ပါဘူး

Site Builder mode ကို ဖယ်ရှားလိုက်ပါပြီ။ အစား Setup Assistant agent ကို အသုံးပြုပါ။

```
"Help me set up a new site"
```

### Site Builder မှ site တစ်ခုကို ပြန်လည်ဖန်တီးလိုပါသည်

Setup Assistant agent ဖြင့် ပြန်လည်ဖန်တီးနိုင်ပါသည်။

၁။ တောင်းဆိုပါ- "Help me set up a new site"
၂။ မူလ site ၏ ရည်ရွယ်ချက်နှင့် ဒီဇိုင်းကို ဖော်ပြပါ။
၃။ agent က အလားတူ theme တစ်ခု ဖန်တီးပေးပါမည်။
၄။ သင့် site_brief ကို အနာဂတ်ကို ရည်ညွှန်းရန် သိမ်းဆည်းပေးပါမည်။

### ကျွန်တော့်ရဲ့ Site Builder site က အလုပ်မလုပ်တော့ပါဘူး

Site Builder mode ဖြင့် ဖန်တီးထားသော လက်ရှိ site များသည် ဆက်လက် အလုပ်လုပ်နေပါသည်။ ပြဿနာများ ကြုံတွေ့ရပါက-

၁။ သင့် theme က အသက်ဝင်နေသေးခြင်း ရှိ၊ မရှိ စစ်ဆေးပါ။
၂။ သင့် plugins များ အသက်ဝင်မှု ရှိ၊ မရှိ စစ်ဆေးပါ။
၃။ WordPress error logs များကို စစ်ဆေးပါ။
၄။ ပြဿနာများ ဆက်လက်ရှိနေပါက support ကို ဆက်သွယ်ပါ။

### ကျွန်တော့်ရဲ့ Site Builder template အဟောင်းတွေကို အသုံးပြုလို့ရသေးလား။

Site Builder template များသည် မရရှိတော့ပါ။ သို့သော်-

- သင့်လက်ရှိ site များသည် ဆက်လက် အလုပ်လုပ်နေပါသည်။
- Setup Assistant agent ဖြင့် အလားတူ site များကို ပြန်လည်ဖန်တီးနိုင်ပါသည်။
- Setup Assistant agent သည် ပိုမိုများပြားသော customization ရွေးချယ်မှုများကို ပေးစွမ်းပါသည်။

## နောက်အဆင့်များ

၁။ **site အသစ်များအတွက်**: Setup Assistant agent ကို အသုံးပြုပါ။
၂။ **လက်ရှိ site များအတွက်**: အတိုင်းအတာအတိုင်း ဆက်လက်အသုံးပြုပါ။
၃။ **theme ပြောင်းလဲမှုများအတွက်**: Setup Assistant agent ထံမှ အကူအညီတောင်းပါ။
၄။ **ဒီဇိုင်းကို ပိုမိုကောင်းမွန်စေရန်**: Design System Aesthetics skill ကို အသုံးပြုပါ။

## ဆက်စပ် ခေါင်းစဉ်များ

- **Theme Builder Onboarding**: စိတ်ကြယ်သော theme များအတွက် လမ်းညွှန် setup
- **Setup Assistant Agent**: Agent မှ လမ်းညွှန်ပေးသည့် site setup
- **Site Specification Skill**: သင့် site ၏ ပန်းတိုင်များနှင့် ပရိသတ်ကို သတ်မှတ်ခြင်း
- **Design System Aesthetics Skill**: သင့် site ၏ မြင်ရသော ရုပ်ပုံလက္ခဏာကို ပိုမိုကောင်းမွန်စေခြင်း
