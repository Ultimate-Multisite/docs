# Filter: wu_independent_billing_cycle_product_types {#filter-wuindependentbillingcycleproducttypes}

ကိုယ်ပိုင် ဘေလ်ကာလ (independent billing cycles) ရှိတဲ့ ထုတ်ကုန်အမျိုးအစားတွေကို စစ်ထုတ်ပေးပါတယ်။

ကိုယ်ပိုင် ဘေလ်ကာလရှိတဲ့ ထုတ်ကုန်တွေဟာ ကားတွရက် (cart) ထဲမှာ အစီအစဉ်ရဲ့ ဘေလ်ကာလနဲ့ ကိုက်ညီဖို့ အတင်းအကျပ် မလိုပါဘူး။ ဒါဟာ အစီအစဉ်က လစဉ်ပဲဖြစ်စေ၊ နှစ်စဉ်ပဲဖြစ်စေ၊ ဒိုမိန်းမှတ်ပုံတင်ခြင်းလိုမျိုး အမြဲတမ်း နှစ်အလိုက် ဘေလ်ကျရတဲ့ ထုတ်ကုန်တွေအတွက် အသုံးဝင်ပါတယ်။

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | ကိုယ်ပိုင် ဘေလ်ကာလရှိတဲ့ ထုတ်ကုန်အမျိုးအစားတွေရဲ့ အကွက်စာရင်း။ |
