# စစ်ထုတ်ကိရိယာ: wu_checkout_form_base_domains

မျှဝေသုံးသော ငွေပေးချေမှုဖောင် အခြေခံဒိုမိန်းများကို mapped-domain မှတ်တမ်းများအဖြစ် မဖြစ်စေရန် စစ်ထုတ်သည်။

Integration တစ်ခုက ငွေပေးချေမှုဖောင် **ဝက်ဘ်ဆိုက် URL** အကွက်များအတွက် အပိုအခြေခံဒိုမိန်းများ ပေးသည့်အခါ ဤစစ်ထုတ်ကိရိယာကို သုံးပါ။ ဤစစ်ထုတ်ကိရိယာမှ ပြန်ပေးသော ဒိုမိန်းများကို ဝက်ဘ်ဆိုက်တစ်ခုချင်းစီအတွက် စိတ်ကြိုက်ဒိုမိန်းများအစား မျှဝေသုံးသော မှတ်ပုံတင် host များအဖြစ် သတ်မှတ်သည်။

## ပါရာမီတာများ {#parameters}

| အမည် | အမျိုးအစား | ဖော်ပြချက် |
|------|------|-------------|
| $domains | `array` | ငွေပေးချေမှုဖောင် configuration မှ စုဆောင်းထားသော မျှဝေသုံးသည့် အခြေခံဒိုမိန်းများ။ |

### စတင်ပါဝင်သည့်ဗားရှင်း {#since}

- 2.13.0

### အရင်းအမြစ် {#source}

`inc/functions/domain.php` တွင် သတ်မှတ်ထားသည်။


## ပြန်ပေးတန်ဖိုး {#returns}

ပုံမှန်စံထားပြီးသော ငွေပေးချေမှုဖောင် အခြေခံဒိုမိန်းများ၏ array။
