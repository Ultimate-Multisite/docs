# Filter: wu_password_require_lowercase

စကားဝှက်များတွင် အက္ခရာအသေး (lowercase) ကို လိုအပ်စေရန် စစ်ထုတ်ခြင်း (Filter) ဖြစ်သည်။

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | အက္ခရာအသေးကို လိုအပ်စေမည်လား။ စည်းမျဉ်းများ အကောင်အထည်ဖော်ထားပါက မူလတန်ဖိုးမှာ true ဖြစ်သည်။ |
| $defender_active | `bool` | Defender Pro Strong Password သည် အသက်ဝင် (active) နေခြင်း ရှိ၊ မရှိ။ |
