# Filter: wu_current_\{$type\}_get_manage_url {#filter-wucurrenttypegetmanageurl}

Developer တွေအနေနဲ့ စီမံခန့်ခွဲမှု ဝက်ဘ်လိပ်စာ (manage site URL) ရဲ့ parameters တွေကို ပြင်ဆင်နိုင်ဖို့ ခွင့်ပြုထားတဲ့ filter တစ်ခု ဖြစ်ပါတယ်။

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $manage_site_url | `string` | စီမံခန့်ခွဲမှု ဝက်ဘ်လိပ်စာ (manage site URL) ကို ရည်ညွှန်းပါတယ်။ |
| $id | `int` | ဝက်ဘ်ဆိုဒ်ရဲ့ ID နံပါတ်။ |
| $site_hash | `string` | ဝက်ဘ်ဆိုဒ်ရဲ့ hash နံပါတ်။ |

### Since {#since}

- 2.0.9
### Source {#source}

[`inc/class-current.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-current.php#L214) ရဲ့ ၂၁၄ လိုင်းမှာ သတ်မှတ်ထားပါတယ်။

## Returns {#returns}
ပြင်ဆင်ပြီးသား စီမံခန့်ခွဲမှု URL ကို ပြန်ပေးပါတယ်။
