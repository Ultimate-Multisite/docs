# Filter: wu_magic_link_enforce_ip {#filter-wumagiclinkenforceip}

IP လိပ်စာ စစ်ဆေးအတည်ပြုခြင်းကို အတင်းအကျပ် လုပ်ဆောင်မလုပ်ဆောင်ရန် စစ်ထုတ်ခြင်း။

၎င်းကို false အဖြစ် သတ်မှတ်ထားပါက တိုကင်များသည် ကွန်ရက်အမျိုးမျိုးမှ အလုပ်လုပ်နိုင်စေမည်ဖြစ်သည်။ ၎င်းသည် လုံခြုံရေးကို အနည်းငယ် လျော့ကျစေသော်လည်း အသုံးပြုရလွယ်ကူမှုကို (ဥပမာ- ကွန်ရက်ပြောင်းနေသော မိုဘိုင်းအသုံးပြုသူများအတွက်) တိုးမြှင့်ပေးပါသည်။

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $enforce | `bool` | IP လိပ်စာကို ကိုက်ညီမှုရှိမရှိ အတင်းအကျပ် လုပ်ဆောင်မလုပ်ဆောင်ရန်။ |
