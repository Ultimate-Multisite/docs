# Filter: wu_password_require_number

စကားဝှက်များတွင် ဂဏန်းများ ပါရန် လိုအပ်ခြင်း ရှိ၊ မရှိ စစ်ထုတ်ရန် အသုံးပြုသည့် filter ဖြစ်ပါသည်။

## Parameters (သတ်မှတ်ချက်များ)

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | ဂဏန်းများ ပါရန် လိုအပ်ခြင်း ရှိ၊ မရှိ။ စည်းမျဉ်းများ အကောင်အထည်ဖော်ထားပါက ပုံသေအားဖြင့် true ဖြစ်သည်။ |
| $defender_active | `bool` | Defender Pro Strong Password ကို အသုံးပြုနေခြင်း ရှိ၊ မရှိ။ |
