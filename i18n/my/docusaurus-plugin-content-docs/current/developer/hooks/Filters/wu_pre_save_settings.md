# Filter: wu_pre_save_settings

Ultimate Multisite မှ သိမ်းဆည်းခြင်းမပြုမီ စနစ်ချိန်ညှိမှုများကို စစ်ထုတ်နိုင်ရန် developer များအား ခွင့်ပြုပါသည်။

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $settings | `array` | သိမ်းဆည်းရမည့် စနစ်ချိန်ညှိမှုများ။ |
| $settings_to_save | `array` | ထပ်မံထည့်သွင်းမည့် စနစ်ချိန်ညှိမှုအသစ်များ။ |
| $saved_settings | `array` | လက်ရှိသိမ်းဆည်းထားပြီးသော စနစ်ချိန်ညှိမှုများ။ |

### Since {#since}

- 2.0.18
