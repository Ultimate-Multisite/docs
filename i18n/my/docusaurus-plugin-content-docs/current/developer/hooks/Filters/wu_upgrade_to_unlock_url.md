# Filter: wu_upgrade_to_unlock_url {#filter-wuupgradetounlockurl}

Developer တွေအနေနဲ့ အဆင့်မြှင့်တင်မှု (upgrade) ကို ဖွင့်ပေးတဲ့ URL ကို ပြောင်းလဲနိုင်အောင် လုပ်ဆောင်ပေးပါတယ်။

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $url | `string` | အဆင့်မြှင့်တင်မှု URL ကို ရည်ညွှန်းပါတယ်။ |
| $args | `array` | အဆင့်မြှင့်ဖို့ လိုအပ်တဲ့ module နဲ့ limit အမျိုးအစားကို ရည်ညွှန်းပါတယ်။ |
