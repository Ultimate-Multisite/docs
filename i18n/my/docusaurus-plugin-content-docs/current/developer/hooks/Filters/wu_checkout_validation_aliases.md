# Filter: wu_checkout_validation_aliases

ပလပ်အင် တည်ဆောက်သူများအနေဖြင့် form validator အတွင်း၌ စိတ်ကြိုက် အမည်ပြောင်း (aliases) များကို ထည့်သွင်းနိုင်စေရန် ခွင့်ပြုပါသည်။

## ပါရာမီတာများ {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $validation_aliases | `array` | id => alias ပါဝင်သည့် array။ |
| $checkout | `\Checkout` | checkout class။ |

- 2.1
