# Filter: wu_add_new_domain_modal_fields {#filter-wuaddnewdomainmodalfields}

ဒိုမိန်းအသစ်ထည့်ရန် ပေါ်လာသည့် (modal) ကွက်လပ်များကို စစ်ထုတ်ပေးသည့် (filter) လုပ်ဆောင်ချက် ဖြစ်ပါသည်။

ဤ filter ကို အသုံးပြုခြင်းအားဖြင့် Addons များ (ဥပမာ- Domain Seller) သည် ဒိုမိန်း မှတ်ပုံတင်ကွက်လပ်များကို ပြင်ဆင်ခြင်း သို့မဟုတ် အစားထိုးခြင်းများ ပြုလုပ်နိုင်စေပါသည်။

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $fields | `array` | ဖောင်တွင် ပါဝင်သည့် ကွက်လပ်များ (form fields)။ |
