# Filter: wu_magic_link_url

ဖန်တီးပေးလိုက်တဲ့ magic link URL ကို စစ်ထုတ်နိုင်ပါတယ်။

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $magic_link | `string` | magic link URL ကို။ |
| $user_id | `int` | အသုံးပြုသူ ID ကို။ |
| $site_id | `int` | ဝက်ဘ်ဆိုဒ် ID ကို။ |
| $redirect_to | `string` | ပြန်လည်ညွှန်းပေးမည့် URL ကို။ |
