# Filter: wu_view_override_replaceable_views

Developer များအနေဖြင့် အစားထိုးနိုင်သည့် ဖိုဒါစာရင်း (replaceable list) တွင် အခြားဖိုဒါများကို ထပ်မံထည့်သွင်းနိုင်စေရန် ခွင့်ပြုပါသည်။

သတိထားရပါမည်။ အခြားဖိုဒါများကို ထပ်မံခွင့်ပြုမိပါက Ultimate Multisite ၏ ဗားရှင်းများအစား ခေတ်မမီတော့သည့် ကော်ပီများကို တင်ယူမိနိုင်ပါသည်။

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $replaceable_views | `array` | ခွင့်ပြုထားသည့် ဖိုဒါစာရင်း။ |

- 2.0.0
