# Filter: secure_auth_redirect

လုံခြုံတဲ့ အထောက်အထား စစ်ဆေးမှု ပြန်လည်ညွှန်းပေးခြင်းကို အသုံးပြုမလားဆိုတာကို စစ်ထုတ်ပေးတဲ့ Filter တစ်ခု ဖြစ်ပါတယ်။
