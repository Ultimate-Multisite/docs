# Filter: wu_admin_notices {#filter-wuadminnotices}

Ultimate Multisite မှ ထည့်သွင်းထားသော အုပ်ချုပ်မှုဆိုင်ရာ အသိပေးချက်များကို developer များ စစ်ထုတ်နိုင်စေရန် ခွင့်ပြုပါသည်။

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $notices | `array` | ထို panel သီးသန့်အတွက် အသိပေးချက်များ စာရင်း။ |
| $all_notices | `array` | panel အလိုက် ခွဲခြားထားသော အသိပေးချက်များ စာရင်း။ |
| $panel | `string` | အသိပေးချက်များကို ပြန်ရယူမည့် panel။ |
| $filter | `string` | ဖျက်သိမ်းနိုင်သော အသိပေးချက်များကို စစ်ထုတ်ပြီးပါက။ |
| $dismissed_messages | `array` | ဖျက်သိမ်းပြီးသော အသိပေးချက် key များ စာရင်း။ |
