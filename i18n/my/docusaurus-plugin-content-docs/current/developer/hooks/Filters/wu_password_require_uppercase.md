# Filter: wu_password_require_uppercase

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | အကြီးအက္ခရာများ ပါရန် လိုအပ်မလို။ စည်းမျဉ်းများ အကောင်အထည်ဖော်ထားပါက မူလတန်ဖိုးမှာ true ဖြစ်သည်။ |
| $defender_active | `bool` | Defender Pro Strong Password ကို အသုံးပြုနေခြင်း ရှိ၊ မရှိ။ |
