# Filter: wu_plan_product_types

အစီအစဉ်များ (plans) အဖြစ် သတ်မှတ်ပေးမည့် ကုန်ပစ္စည်းအမျိုးအစားများကို စစ်ထုတ်ပေးခြင်း။

ဤ filter ကို အသုံးပြုခြင်းဖြင့် add-on များသည် အတည်ပြုခြင်း (validation) နှင့် ခွဲခြားသတ်မှတ်ခြင်း (segregation) တို့တွင် အစီအစဉ်များအဖြစ် အသိအမှတ်ပြုသင့်သည့် ကုန်ပစ္စည်းအမျိုးအစားများကို ထပ်မံစာရင်းသွင်းနိုင်ပါသည်။

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | အစီအစဉ်များအဖြစ် သတ်မှတ်လိုသည့် ကုန်ပစ္စည်းအမျိုးအစားများ၏ အကောင်အထည်ဖော်မှု (Array)။ |
