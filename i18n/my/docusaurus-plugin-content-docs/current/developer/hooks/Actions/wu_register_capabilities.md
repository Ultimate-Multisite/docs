# Action: wu_register_capabilities

Add-ons များအား ၎င်းတို့၏ စွမ်းဆောင်ရည် မော်ဂျူး (capability modules) များကို ပေါင်းစပ်မှုများ (integrations) တွင် ချိတ်ဆက်ခွင့်ပေးရန်အတွက် အသုံးပြုသည့် လုပ်ဆောင်ချက်တစ်ခု ဖြစ်ပါသည်။

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $registry | `\Integration_Registry` | မှတ်ပုံတင်မှု အရာဝတ္ထု (registry instance) ဖြစ်ပါသည်။ |

### Since

- 2.5.0
