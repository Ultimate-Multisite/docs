# Action: wu_register_integrations

ပေါင်းစပ်မှုများ (integrations) ကို ကိုယ်တိုင် မှတ်ပုံတင်နိုင်စေရန် ဤ action ကို အသုံးပြုသည်။

အဓိကစနစ် (Core) က ၎င်း၏ ပေါင်းစပ်မှုများကို ဤနေရာတွင် မှတ်ပုံတင်သည်။

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $registry | `\Integration_Registry` | မှတ်ပုံတင်မှု အရာဝတ္ထု (registry instance)။ |

### Since {#since}

- 2.5.0
