# Action: wu_site_created

ဝက်ဘ်ဆိုဒ်တစ်ခုကို ပထမဆုံးအကြိမ် ဖန်တီးပြီးသည့်အခါ အလုပ်လုပ်သည့် Action တစ်ခု ဖြစ်ပါသည်။
