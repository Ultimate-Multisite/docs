# Action: wu_mcp_abilities_registered

MCP စွမ်းရည်များ (abilities) သည် entity တစ်ခုအတွက် မှတ်ပုံတင်ပြီးနောက် အလုပ်လုပ်သည့် Action တစ်ခု ဖြစ်ပါသည်။

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $ability_prefix | `string` | စွမ်းရည်၏ အစ (ability prefix) ဖြစ်ပါသည်။ |
| $model_name | `string` | မော်ဒယ်၏ အမည် (model name) ဖြစ်ပါသည်။ |
| $this | `object` | စီမံခန့်ခွဲမှု အရာဝတ္ထု (manager instance) ဖြစ်ပါသည်။ |
