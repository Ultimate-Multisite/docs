# Action: wu_mcp_adapter_initialized

MCP adapter ကို စတင်ဖွဲ့စည်းပြီးတဲ့အခါမှာ ဒီ action က အလုပ်လုပ်ပါတယ်။

ဒီ action ကို အသုံးပြုပြီး အခြား plugin တွေနဲ့ theme တွေက သူတို့ရဲ့ စွမ်းရည်တွေကို မှတ်ပုံတင်နိုင်ပါတယ်။

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | MCP adapter ရဲ့ အကောင်အထည်ဖော်မှု (instance) ဖြစ်ပါတယ်။ |

### Since {#since}

- 2.5.0
### Source {#source}

[`inc/class-mcp-adapter.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-mcp-adapter.php#L98) ရဲ့ ၉၈ လိုင်းမှာ သတ်မှတ်ထားပါတယ်။
