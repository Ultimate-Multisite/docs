# Action: wu_magic_link_login

Magic link မှတစ်ဆင့် အသုံးပြုသူတစ်ဦး ဝင်ရောက်ပြီးနောက် အလုပ်လုပ်သည့် Action တစ်ခု ဖြစ်ပါသည်။

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $user_id | `int` | အသုံးပြုသူ ID ကို ရည်ညွှန်းပါသည်။ |
| $site_id | `int` | ဝက်ဘ်ဆိုဒ် ID ကို ရည်ညွှန်းပါသည်။ |
