# Action: wu_register_rest_routes {#action-wuregisterrestroutes}

အပိုဆောင်း route များကို မှတ်ပုံတင်နိုင်စေရန် ခွင့်ပြုပါသည်။

၎င်းကို ကျွန်ုပ်တို့၏ /register endpoint မှ အသုံးပြုပါသည်။

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $api_instance | `self` | လက်ရှိ API instance ကို ရည်ညွှန်းပါသည်။ |

### Since {#since}

- 2.0.0

[`inc/class-api.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-api.php#L470) ၏ ၄၇၀ လိုင်းတွင် သတ်မှတ်ထားပါသည်။
