# Action: auth_redirect

အကောင့်ဝင်ခြင်း ပြန်လည်ညွှန်းပေးခြင်း မလုပ်ခင်မှာ အလုပ်လုပ်တဲ့ Action တစ်ခု ဖြစ်ပါတယ်။

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $user_id | `int` | အသုံးပြုသူ ID (User ID) ကို ရည်ညွှန်းပါတယ်။ |

- 2.8.0
