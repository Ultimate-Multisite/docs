# Action: wu_dashboard_widgets {#action-wudashboardwidgets}

ပလပ်အင် (plugin) ရေးသူများအနေဖြင့် Network Dashboard Panel တွင် widgets များထည့်သွင်းနိုင်စေရန် ခွင့်ပြုပါသည်။

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $tab | `string` | လက်ရှိ tab ကို ဖော်ပြသည်။ |
| $screen | `\WP_Screen` | Screen object ကို ကိုယ်စားပြုသည်။ |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Ultimate Multisite admin page ၏ instance တစ်ခု။ |
