# Action: wu_before_light_ajax

အချို့အခြေအနေများတွင် လုပ်ဆောင်ချက်များကို ကိုင်တွယ်ရန်အတွက် အပိုဆောင်း လုပ်ဆောင်ချက်များ ထည့်သွင်းရန် လိုအပ်နိုင်ပါသည်။
