# GoCardless ပြောင်းလဲမှုမှတ်တမ်း

Version: 1.0.3 - 2026-05-05 တွင် ထုတ်ဝေခဲ့သည်
* Fix: GoCardless အဖွဲ့ဝင်မှုကို pending သို့မဟုတ် on-hold အခြေအနေမှာ ပယ်ဖျက်လိုက်တဲ့အခါ ပစ်ကျနေတဲ့ (orphaned) pending site တွေကို အလိုအလျောက် ရှင်းလင်းပေးပါပြီ။
* Improved: Bedrock root autoloader က dependencies တွေကို ကြိုတင် load လုပ်ပြီးသားဆိုရင် Plugin autoloader ကို ခုန်ကျော် (skip) လုပ်သွားပါလိမ့်မယ်။

Version: 1.0.1 - 2025-09-28 တွင် ထုတ်ဝေခဲ့သည်

* prefix ကို ultimate-multisite အဖြစ် ပြောင်းလဲခဲ့သည်၊ text domain ကို update လုပ်ခဲ့သည်၊ နှင့် version ကို တိုးမြှင့်ခဲ့သည်။

2021-09-24 - Version 1.0.0-beta.4

* Added: mu-plugins အခြေခံ setup များအတွက် filter wp_ultimo_skip_network_active_check ကို ထည့်သွင်းခဲ့သည်။

2021-06-16 - Version 1.0.0-beta.3

* Fix: Webhook handlers များက signature တွေကို အတည်ပြုရာမှာ ကျရှုံးခဲ့တဲ့ ပြဿနာကို ပြင်ဆင်ခဲ့သည်။

2021-06-14 - Version 1.0.0-beta.2

* Improved: GoCardless mandate တစ်ခု ဖန်တီးပြီးတဲ့အခါ site ကို ထုတ်ဝေနိုင်အောင် ပိုမိုကောင်းမွန်စေခဲ့သည်။

Version 1.0.0-beta.1 - ပထမဆုံးဗားရှင်း
