# PayFast ပြောင်းလဲမှု မှတ်တမ်း

Version: 1.0.2 - 2025-09-28 တွင် ထုတ်ဝေခဲ့သည်

* အစပိုဒ်အမည်ကို ultimate-multisite အဖြစ် ပြောင်းလဲခြင်း၊ စာသားနယ်ပယ်ကို အဆင့်မြှင့်တင်ခြင်းနှင့် ဗားရှင်းနံပါတ် တိုးမြှင့်ခြင်း။

Version 1.0.0-beta.3 - 30/07/2021

* တိုးတက်မှု- အမှားရှာဖွေရန် ရည်ရွယ်သည့် ပိုမိုကောင်းမွန်သော အမှားသတင်းများကို ထည့်သွင်းထားခြင်း။

Version 1.0.0-beta.2 - 26/04/2021

* ပြင်ဆင်ထားသည်- Gateway base class အသစ်နှင့် တွဲဖက်အသုံးပြုနိုင်မှု။

Version 1.0.0-beta.1 - ပထမဆုံး ထုတ်ဝေမှု
