# Loco Translate ပြောင်းလဲမှု မှတ်တမ်း {#loco-translate-changelog}

### 1.0.1 {#101}
* နောက်ဆုံးပေါ် Loco Translate ဗားရှင်းနှင့် တွဲဖက်အသုံးပြုနိုင်မှု ပိုမိုကောင်းမွန်လာခြင်း
* ပိုမိုကောင်းမွန်သော လုံခြုံရေးအတွက် ဝင်ရောက်ခွင့် ထိန်းချုပ်မှု (Access control) ကို မြှင့်တင်ထားခြင်း
* စွမ်းဆောင်ရည် မြှင့်တင်မှုများ ပြုလုပ်ထားခြင်း

### 1.0.0 {#100}
* ပထမဆုံး ထုတ်ဝေမှု
* ဝက်ဘ်ဆိုဒ်အလိုက် ဘာသာပြန်မှု ဖိုင်တွဲများ (Site-specific translation directories)
* ဘာသာပြန်မှု ဦးစားပေးစနစ် (Translation priority system)
* ဝင်ရောက်ခွင့် ထိန်းချုပ်မှု (Access control) ကို အကောင်အထည်ဖော်ခြင်း
* Loco Translate နှင့် ချောမွေ့စွာ UI ပေါင်းစပ်မှု (Seamless UI integration)
