# Loco Translate {#loco-translate}

Ultimate Multisite: Loco Translate Integration addon သည် သင့် WordPress multisite network အတွက် ဝက်ဘ်ဆိုဒ်တစ်ခုချင်းစီအလိုက် ဘာသာပြန်နိုင်သည့် စွမ်းရည်ကို ပေါင်းစပ်ပေးထားပါသည်။ သင့် network အတွင်းရှိ ဝက်ဘ်ဆိုဒ်အချို့သည် ပလပ်အင် သို့မဟုတ် theme တစ်ခုတည်းကို မတူညီသော ဘာသာပြန်များ လိုအပ်သည့် အချိန်မျိုးတွင် ဤ addon သည် အကောင်းဆုံး ဖြေရှင်းချက် ဖြစ်ပါသည်။

**ဤ Addon က ဘာတွေ လုပ်ပေးသလဲ**

ဤ addon သည် Ultimate Multisite ကို လူကြိုက်များသည့် Loco Translate plugin နှင့် ချောမွေ့စွာ ပေါင်းစပ်ပေးထားပြီး၊ သင့် network ရှိ အခွဲဝက်ဘ်ဆိုဒ်တစ်ခုချင်းစီသည် ၎င်း၏ ကိုယ်ပိုင် ဘာသာပြန်များကို ထိန်းသိမ်းထားနိုင်စေပါသည်။ သင်သည် နိုင်ငံအများအပြားကို ဝန်ဆောင်မှုပေးသည့် ပလက်ဖောင်းတစ်ခု၊ ဘာသာစကားအသိုင်းအဝိုင်းအမျိုးမျိုးကို ဝန်ဆောင်မှုပေးသည်ဖြစ်စေ၊ သို့မဟုတ် သင့် network တစ်ခုလုံးတွင် ဘာသာပြန်မှု ပိုမိုပြောင်းလွယ်ပြင်လွယ်ရှိရန် လိုအပ်သည်ဖြစ်စေ၊ ဤ addon သည် အလွန်လွယ်ကူစေပါသည်။

**မည်သူများအတွက် အသင့်တော်ဆုံးလဲ:**

*   ဒေသအလိုက် ဘာသာစကားလိုအပ်ချက်များ မတူညီသော လုပ်ငန်းများ
*   ကွဲပြားသော အသိုင်းအဝိုင်းများကို ဝန်ဆောင်မှုပေးသည့် ပညာရေးအဖွဲ့အစည်းများ
*   ဘာသာစကားမျိုးစုံကို ပံ့ပိုးပေးသည့် SaaS ပလက်ဖောင်းများ
*   ဖောက်သည် ဝက်ဘ်ဆိုဒ်များစွာကို စီမံခန့်ခွဲသည့် Web agency များ
*   ဘာသာပြန်မှု ပိုမိုပြောင်းလွယ်ပြင်လွယ်ရှိရန် လိုအပ်သည့် မည်သည့် multisite network မဆို

## တပ်ဆင်ခြင်း (Installation) {#installation}

၁။ Ultimate Multisite ကို တပ်ဆင်ပြီး network ကို အသက်သွင်းထားကြောင်း သေချာပါစေ။
၂။ Loco Translate plugin ကို တပ်ဆင်ပြီး အသက်သွင်းပါ။
၃။ ဤ addon ကို သင့် plugins directory ထဲသို့ တင်ပါ။
၄။ Loco Translate Integration addon ကို network အဆင့်တွင် အသက်သွင်းပါ။
၅။ အခွဲဝက်ဘ်ဆိုဒ် (subsite) တစ်ခုခုမှတစ်ဆင့် Loco Translate ကို အသုံးပြု၍ ဝက်ဘ်ဆိုဒ်တစ်ခုချင်းစီအတွက် ဘာသာပြန်မှုများကို စတင်ဖန်တီးပါ။

ပြီးပါပြီ! အပိုဆောင်း စီစဉ်မှုများ မလိုဘဲ ပေါင်းစပ်မှုသည် အလိုအလျောက် အလုပ်လုပ်သွားပါလိမ့်မည်။

## မကြာခဏ မေးလေ့ရှိသော မေးခွန်းများ (Frequently Asked Questions) {#frequently-asked-questions}

## တပ်ဆင်ပြီးနောက် ဘာများ စီစဉ်ရဦးမလဲ။ {#do-i-need-to-configure-anything-after-installation}

မလိုပါဘူး! ပေါင်းစပ်မှုသည် အလိုအလျောက် ဖြစ်ပါတယ်။ addon နှင့် Loco Translate နှစ်ခုလုံး အသက်ဝင်သည်နှင့်၊ အခွဲဝက်ဆိုဒ်များသည် ၎င်းတို့၏ ကိုယ်ပိုင် ဘာသာပြန်များကို ချက်ချင်း စတင်ဖန်တီးနိုင်ပါသည်။

## အခွဲဝက်ဆိုဒ်တစ်ခုက ကိုယ်ပိုင် ဘာသာပြန်များ မဖန်တီးနိုင်ရင် ဘာဖြစ်မလဲ။ {#what-happens-if-a-subsite-doesnt-create-custom-translations}

ထိုဝက်ဘ်ဆိုဒ်သည် အလိုအလျောက် network တစ်ခုလုံးအတွက် ဘာသာပြန်များကို အရန်အဖြစ် အသုံးပြုပါလိမ့်မည်။ ဘာတစ်ခုမှ ပျက်စီးသွားမှာ မဟုတ်ဘဲ အရာအားလုံး ချောမွေ့စွာ အလုပ်လုပ်နေပါလိမ့်မည်။

## အခွဲဝက်ဆိုဒ်တွေက network တစ်ခုလုံးရဲ့ ဘာသာပြန်တွေကို ဝင်ကြည့်လို့ရလား။ {#can-subsites-access-network-wide-translations}

ပုံမှန် site admin များသည် ၎င်းတို့၏ ဝက်ဘ်ဆိုဒ်ကိုယ်ပိုင် ဘာသာပြန်များကိုသာ မြင်ရပြီး တည်းဖြတ်နိုင်ပါသည်။ Super admins များသည် မည်သည့် ဝက်ဘ်ဆိုဒ်ကို ကြည့်ရှုနေသည်ပေါ် မူတည်၍ network တစ်ခုလုံးနှင့် ဝက်ဘ်ဆိုဒ်တစ်ခုချင်းစီ၏ ဘာသာပြန်များကို နှစ်ခုစလုံး ဝင်ကြည့်နိုင်ပါသည်။

## plugin တွေနဲ့ theme တွေ နှစ်ခုလုံးမှာ အလုပ်လုပ်လား။ {#does-this-work-with-both-plugins-and-themes}

ဟုတ်ပါတယ်! ဝက်ဘ်ဆိုဒ်တစ်ခုချင်းစီအတွက် ဘာသာပြန်မှုများသည် WordPress plugin နှင့် theme နှစ်ခုစလုံးအတွက် အလုပ်လုပ်ပါသည်။

## ဒါက ကျွန်တော့် multisite network ကို နှေးကွေးစေမှာလား။ {#will-this-slow-down-my-multisite-network}

မနှေးစေပါဘူး။ ဤ addon သည် ထိရောက်သော file loading ကို အသုံးပြုပြီး၊ လိုအပ်သည့်အခါမှသာ ဝက်ဘ်ဆိုဒ်တစ်ခုချင်းစီ၏ ဘာသာပြန်များကို စစ်ဆေးပါသည်။ သိသာထင်ရှားသော စွမ်းဆောင်ရည် ကျဆင်းမှု မရှိပါဘူး။

## network ဘာသာပြန်တွေကို စတင်အသုံးပြုဖို့အတွက် ဝက်ဘ်ဆိုဒ်တစ်ခုမှာ ကူးယူလို့ရလား။ {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

ရပါတယ်! Loco Translate မှာ ဘာသာပြန်တွေကို တည်းဖြတ်နေစဉ်၊ network ဘာသာပြန်များဖြင့် အလွယ်တကူ စတင်ပြီး ဝက်ဘ်ဆိုဒ်တစ်ခုချင်းစီအတွက် စိတ်ကြိုက် ပြုပြင်နိုင်ပါတယ်။

## WooCommerce နဲ့ တခြား plugin တွေမှာလည်း အလုပ်လုပ်လား။ {#does-this-work-with-woocommerce-and-other-plugins}

လုံးဝ အလုပ်လုပ်ပါတယ်! စံနှုန်းသတ်မှတ်ထားသော WordPress ဘာသာပြန် လုပ်ဆောင်ချက်များကို အသုံးပြုသည့် မည်သည့် plugin သို့မဟုတ် theme မဆို ဝက်ဘ်ဆိုဒ်တစ်ခုချင်းစီ ဘာသာပြန်မှုများနှင့် အလုပ်လုပ်နိုင်ပါသည်။
