# Plugin နှင့် Theme Manager ၏ ပြောင်းလဲမှုမှတ်တမ်း {#plugin--theme-manager-changelog}

Version: 2.0.8 - ထုတ်ဝေသည့်ရက် ၂၀၂၆-၀၅-၀၅
* ပိုမိုကောင်းမွန်စေသည်- Plugin စာရင်းသည် database query များကို လျှော့ချရန်နှင့် စွမ်းဆောင်ရည် မြှင့်တင်ရန်အတွက် site cache ကို အသုံးပြုလာပြီ။
* ပိုမိုကောင်းမွန်စေသည်- Bedrock root autoloader က မှီခိုမှုများကို ကြိုတင် load လုပ်ပြီးသားဖြစ်တဲ့အခါ Plugin autoloader ကို ကျော်လွန်သွားအောင် လုပ်ထားပါတယ်။

Version: 2.0.7 - ထုတ်ဝေသည့်ရက် ၂၀၂၆-၀၁-၀၁
* ပိုမိုကောင်းမွန်စေသည်- တွဲဖက်အသုံးပြုနိုင်မှုဆိုင်ရာ အဆင့်မြှင့်တင်မှုများ။
