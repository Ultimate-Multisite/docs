# Plugin & Theme Manager {#plugin--theme-manager}

Multisite Ultimate: Plugin and Theme Manager

ထည့်သွင်းထားတဲ့ plugins တွေနဲ့ themes တွေရဲ့ metadata အချက်အလက် အကျဉ်းချုပ် (ခေါင်းစဉ်၊ စာရေးသူ၊ thumbnail နဲ့ ဖော်ပြချက်) တွေကို ပြင်ဆင်နိုင်သလို၊ ဖျောက်ထားနိုင်ပါတယ်။ plugins တွေနဲ့ themes တွေအတွက် အမျိုးအစားခွဲပြီး၊ သုံးစွဲသူတွေ filter လုပ်နိုင်ဖို့နဲ့ သုံးစွဲသူတွေရဲ့ panel မှာ လှပတဲ့ custom Plugins page တစ်ခုကို ထည့်သွင်းနိုင်ပါတယ်။

ဒီ plugin ကို အသုံးပြုဖို့အတွက်၊ သင့်ရဲ့ network admin မှာရှိတဲ့ Plugins ဒါမှမဟုတ် Themes စာမျက်နှာကို သွားပြီး "Edit Plugin/Theme info" ကို နှိပ်ရုံပါပဲ။

## Installation {#installation}

1. 'ultimate-multisite-plugin-and-theme-manager' ကို '/wp-content/plugins/' directory ထဲကို upload လုပ်ပါ။
2. သင့်ရဲ့ WordPress Network Admin က 'Plugins' menu ကနေတစ်ဆင့် plugin ကို activate လုပ်ပါ။

## Usage Guide {#usage-guide}

Ultimate Multisite: Plugin and Theme Manager ဆိုတာကတော့ သင့်ရဲ့ network ပေါ်မှာ plugins နဲ့ themes ရဲ့ option တွေကို network သုံးစွဲသူတွေ ဘယ်လိုမြင်ရမလဲ၊ ဘယ်လိုအပြန်အလှန်ဆက်ဆံရမလဲဆိုတာကို အပြည့်အဝ ထိန်းချုပ်နိုင်တဲ့ **premium** Ultimate Multisite add-on တစ်ခု ဖြစ်ပါတယ်။

ဒီ add-on က ရည်ရွယ်ချက် ၃ ခုကို လုပ်ဆောင်ပေးပါတယ်-

  1. plugins တွေနဲ့ themes တွေရဲ့ metadata အချက်အလက် အကျဉ်းချုပ်ကို လုံးဝပြောင်းလဲနိုင်စေပါတယ်။ ဒါမှာ နာမည်၊ ဖော်ပြချက်၊ စာရေးသူနာမည်၊  bahkan thumbnail တွေပါ ပါဝင်ပါတယ်။

  2. plugins တွေနဲ့ themes တွေကို အမျိုးအစားခွဲနိုင်စေပြီး၊ သုံးစွဲသူတွေအတွက် အလွယ်တကူ ခွဲခြားနိုင်အောင် ကူညီပေးပါတယ်။

  3. ဟောင်းပြီး ငြီးငွေ့စရာကောင်းတဲ့ plugins page ကို အမျိုးအစားအလိုက် စီစစ်နိုင်တဲ့ themes နဲ့တူတဲ့ page အသစ်တစ်ခုနဲ့ အစားထိုးခွင့် ပေးပါတယ်။

ဒေါင်းလုဒ်လုပ်ပြီး install လုပ်ပြီးတဲ့အခါမှာ Ultimate Multisite Settings page ပေါ်မှာ _Add-on Settings_ ဆိုတဲ့ tab အသစ်တစ်ခု ပေါ်လာပါလိမ့်မယ်။
