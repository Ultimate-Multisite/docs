# Action: delete_network {#action-deletenetwork}

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $network | `\WP_Network` | ဖျက်လိုက်တဲ့ network object ကို ဆိုလိုပါတယ်။ |
