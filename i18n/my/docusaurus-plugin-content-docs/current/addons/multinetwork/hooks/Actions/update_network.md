# Action: update_network

ရှိပြီးသား network တစ်ခုကို အဆင့်မြှင့်ပြီးတဲ့အခါမှာ အလုပ်လုပ်တဲ့ လုပ်ဆောင်ချက်တစ်ခု ဖြစ်ပါတယ်။

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $network_id | `int` | ထည့်သွင်းထားတဲ့ network ရဲ့ ID။ |
| $args | `array` | network အတွက် အသုံးပြုနိုင်တဲ့ key-value ပုံစံ array တစ်ခု။ |
