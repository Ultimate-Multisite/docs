# Action: add_network {#action-addnetwork}

ကွန်ရက်အသစ်တစ်ခု ထပ်မံထည့်သွင်းပြီးနောက် အလိုအလျောက် ခေါ်ဆောင်သည့် လုပ်ဆောင်ချက် (Action) ဖြစ်ပါသည်။

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $new_network_id | `int` | ထည့်သွင်းထားသော ကွန်ရက်၏ ID ကို ဖော်ပြသည်။ |
| $r | `array` | ကွန်ရက်အချက်အလက်များအားလုံးပါဝင်သော အစီအစဉ်ကျသည့် array (associative array) တစ်ခုလုံး။ |
