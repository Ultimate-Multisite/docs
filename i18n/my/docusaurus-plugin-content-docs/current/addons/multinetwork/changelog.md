# Multi-Network ပြောင်းလဲမှု မှတ်တမ်း

### 1.3.0
* အသစ်: ကွန်ရက် ပုံစံ ကြိုတင်ကြည့်ရှုစက် (Network template previewer) — ဝယ်ယူခြင်းမပြုမီ live panel ဖြင့် ကွန်ရက်ပုံစံများကို ကြည့်ရှုနိုင်ပြီ။
* အသစ်: ငွေရှင်းချိန်တွင် ကွန်ရက်ပုံစံ ရွေးချယ်စက် (Network template picker at checkout) — ဝယ်ယူသူများသည် စာရင်းသွင်းခြင်း (signup) ငွေရှင်းမှုလုပ်ငန်းစဉ်အတွင်း ကွန်ရက်ပုံစံကို ရွေးချယ်နိုင်ပြီ။
* အသစ်: ကွန်ရက်အချင်းချင်း မက်ဂျစ်လင့်ခ် SSO (Cross-network magic link SSO) — ဝယ်ယူသူများသည် magic-link authentication မှတစ်ဆင့် ကွန်ရက်ခွဲများသို့ ချောမွေ့စွာ ဝင်ရောက်နိုင်ပြီ။
* အသစ်: ကွန်ရက်မီဒီယာ ကူးချခြင်း (Network media cloning) — ပုံစံတစ်ခုမှ ကွန်ရက်များကို ကူးချသည့်အခါ site ၏ မီဒီယာများကို မှန်ကန်စွာ ပွားယူပေးသည်။
* ပြင်ဆင်မှု: ကွန်ရက်ကူးချခြင်းသည် multi-tenancy addon မပါဘဲ အလုပ်လုပ်နိုင်ပြီ။ ကူးချပြီးနောက် super admin အခွင့်အရေးများကို ပြန်လည်ပေးအပ်ထားသည်။
* ပြင်ဆင်မှု: ကွန်ရက်တည်ဆောက်ခြင်းသည် WooCommerce နှင့် ငွေပေးချေမှု ဂိတ်ဝများအားလုံးနှင့် တွဲဖက်အသုံးပြုနိုင်ပြီ။
* ပြင်ဆင်မှု: fallback domain ထုတ်ပေးရာတွင် port နံပါတ်ကို ထည့်သွင်းပေးထားသည်။
* ပြင်ဆင်မှု: ပုံစံရွေးချယ်စက် (Template picker) CSS ကို အကြောင်းအရာအားလုံးတွင် မှန်ကန်စွာ ပြသနိုင်ရန် enqueued stylesheet သို့ ရွှေ့ပြောင်းထားသည်။
* ပြင်ဆင်မှု: site query scope တွင် undefined `network_id` key ကို ကာကွယ်ပေးထားသည်။
* ပြင်ဆင်မှု: context switching မပြုမီ ကွန်ရက်ခွင့်ပြုချက် စစ်ဆေးခြင်းသည် `wpdb` table property အမှားများကို ကာကွယ်ပေးသည်။
