# Site Exporter ပြောင်းလဲမှုမှတ်တမ်း

Version: 1.0.6 - 2026-05-11 တွင် ထုတ်ဝေသည်
* အသစ်- ဝက်ဘ်ဆိုဒ် အချက်အလက်များ တင်ပို့သည့်အခါ ယခုအခါ self-booting index.php ကို စုစည်းပေးထားသောကြောင့် ZIP ဖိုင်ကို plugin အသစ်ထည့်သွင်းစရာမလိုဘဲ အသစ်စတင်ထားသော hosting တွင် တင်နိုင်ပြီ။
* အသစ်- Network export ကို အသုံးပြု၍ Site Export admin page မှ အခွဲဝက်ဘ်ဆိုဒ်အားလုံးကို အစုအဝေးတစ်ခုတည်းအဖြစ် တင်ပို့နိုင်ပြီ။

Version: 1.0.5 - 2026-05-05 တွင် ထုတ်ဝေသည်
* ပြင်ဆင်မှု- get_config ကဲ့သို့ mock မလုပ်ရသေးသော methods များကို ကိုင်တွယ်နိုင်ရန် WP_CLI polyfill stub တွင် __callStatic ကို ထည့်သွင်းခဲ့ခြင်းဖြင့် စမ်းသပ်စတင်မှု အမှားများ မဖြစ်အောင် ကာကွယ်ပေးနိုင်ပြီ။

Version: 1.0.4 - 2025-11-25 တွင် ထုတ်ဝေသည်

* Ultimate Multisite 2.4 နှင့် တွဲဖက်အသုံးပြုနိုင်မှု ပြင်ဆင်မှု။

Version: 1.0.2 - 2025-09-28 တွင် ထုတ်ဝေသည်

* prefix ကို ultimate-multisite အဖြစ် ပြောင်းလဲခြင်း၊ text domain ကို အဆင့်မြှင့်တင်ခြင်းနှင့် version ကို အဆင့်မြှင့်ခြင်း။

### Version 1.0.1 - 2023-08-09 တွင် ထုတ်ဝေသည်

* ပိုမိုကောင်းမွန်စေသည်- ယခုအခါ importer ဖိုင်၏ မှန်ကန်သော file path ကို ရရှိကြောင်း အာမခံပေးသည်။
* ပြင်ဆင်မှု- import လုပ်နေစဉ် အမှားများ မဖြစ်စေရန် object cache ကို ပိတ်ထားသည်။
* ပြင်ဆင်မှု- importer plugin download link မရရှိသည့် ပြဿနာကို ဖြေရှင်းပေးခဲ့သည်။

### Version 1.0.0 - 2022-12-23 တွင် ထုတ်ဝေသည်

* ပြင်ဆင်မှု- Export form popup မပေါ်ခြင်း။
* ပြင်ဆင်မှု- Export page တွင် Javascript ဖိုင်များ မပေါ်ခြင်း။
* အတွင်းပိုင်း- build process အသစ်။

### Version 1.0.0-beta.4 - 2022-01-13 တွင် ထုတ်ဝေသည်

* အတွင်းပိုင်း- hooks နှင့် filters generator ကို ထည့်သွင်းခဲ့သည်။
* အတွင်းပိုင်း- developer များ အသုံးပြုရလွယ်ကူစေရန် WP Ultimo stubs များကို ထည့်သွင်းခဲ့သည်။
