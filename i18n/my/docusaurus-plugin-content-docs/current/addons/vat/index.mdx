# VAT

Multisite Ultimate: European VAT

Multisite Ultimate ၏ မူလ အခွန် စီမံခန့်ခွဲမှု (tax handling) ကို ပိုမိုကောင်းမွန်စွာ ပံ့ပိုးနိုင်ရန်အတွက် ဥရောပ VAT ကို အထောက်အကူပြုသည့် အပိုဆောင်း အင်္ဂါရပ်များနှင့် ကိရိယာများကို ထပ်ပေါင်းပေးပါသည်။ EU VAT နှုန်းထားများကို Tax Rates စာမျက်နှာတွင် အလိုအလျောက် ထည့်သွင်းပေးပါသည်။

## Installation

1. 'ultimate-multisite-vat' ကို '/wp-content/plugins/' directory ထဲသို့ upload လုပ်ပါ။
2. WordPress ၏ 'Plugins' menu မှတစ်ဆင့် plugin ကို activate လုပ်ပါ။
3. Ultimate Multisite > Settings > Taxes သို့ သွားပြီး EU Vat ကို ဖွင့်ပါ။
4. Tax Rates ကို နှိပ်ပြီး အောက်ဘက်ရှိ EU VAT နှုန်းထားများကို update လုပ်ကာ Tax Rates ကို save လုပ်ပါ။
5. ပြီးပါပြီ!
