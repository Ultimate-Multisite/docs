# Content Sync

Multisite Ultimate: Content Sync

Multisite Ultimate အတွက် ဒီ add-on ကို သုံးပြီး ကွန်ရက် စီမံခန့်ခွဲသူတွေဟာ ဘယ် site က အကြောင်းအရာကိုမဆို ကွန်ရက်ထဲက တခြား site တွေဆီကို ချိန်ညှိပေးနိုင်ပါတယ်။ ဒါက posts, pages, custom post types တွေအပြင် Elementor templates တွေအပါအဝင် post type အားလုံးကို ပံ့ပိုးပေးပါတယ်။ ဒီ add-on မှာ အကြောင်းအရာရွေးချယ်ဖို့နဲ့ ပစ်မှတ် site တွေရွေးချယ်ဖို့အတွက် အဆင့်ဆင့် လမ်းညွှန်မှု (multi-step wizard) အင်တာဖေ့စ် ပါဝင်ပြီး၊ batch processing နဲ့ real-time progress feedback တွေလည်း ရရှိမှာပါ။

## အင်္ဂါရပ်များ (Features) {#features}

* Content Selection: Thumbnail တွေနဲ့ post type filter တွေပါတဲ့ visual card grid ပုံစံ။
* All Post Types: posts, pages, products, Elementor templates နဲ့ custom post type တွေ အားလုံးကို ချိန်ညှိပေးနိုင်ခြင်း။
* Site Selection: အကြောင်းအရာ ဘယ် site တွေဆီ ချိန်ညှိမလဲ ရွေးချယ်နိုင်ပြီး၊ source site ကို ချန်လှပ်ခြင်းနဲ့ child site တွေကို ကြိုတင်ရွေးချယ်နိုင်ခြင်း။
* Batch Processing: အချိန်တစ်ခုမှာ site ၁၀ ခုအထိ ချိန်ညှိပေးနိုင်ပြီး progress update တွေနဲ့ အမြဲသိရှိရမှာ။
* Media Sync: ပို့စ်အကြောင်းအရာထဲက media files (ပုံများ၊ ဗီဒီယိုများ) တွေကို ပစ်မှတ် site တွေဆီ အလိုအလျောက် ကူးယူပြီး ပြန်လည်ချိတ်ဆက်ပေးခြင်း။
* Taxonomy Sync: categories, tags နဲ့ custom taxonomies တွေကို ထိန်းသိမ်းပေးခြင်း။
* Elementor Support: CSS ပြန်လည်ထုတ်လုပ်ခြင်း (regeneration) နဲ့ template ID ပြန်လည်ချိတ်ဆက်ခြင်း (remapping) တို့နဲ့ Elementor templates တွေကို အပြည့်အဝ ပံ့ပိုးပေးခြင်း။
* Conflict Resolution: ရှိပြီးသား အကြောင်းအရာတွေကို အစားထိုးမလား၊ ဒါမှမဟုတ် ကျော်လွန်သွားမလား ရွေးချယ်ခွင့်ပေးခြင်း။
* Cache Management: sync လုပ်ပြီးတဲ့အခါ ပစ်မှတ် site တွေပေါ်က Elementor cache ကို အလိုအလျောက် ရှင်းပေးခြင်း။

## လိုအပ်ချက်များ (Requirements) {#requirements}

* WordPress 5.3 or higher
* PHP 7.4 or higher
* Multisite Ultimate plugin (active)
* Elementor plugin (optional, required for Elementor template sync)

## တပ်ဆင်ခြင်း (Installation) {#installation}

၁။ 'ultimate-multisite-content-sync' ကို '/wp-content/plugins/' directory ထဲကို upload လုပ်ပါ။
၂။ သင့် WordPress Network Admin ထဲက 'Plugins' menu ကနေ plugin ကို activate လုပ်ပါ။
၃။ Ultimate Multisite → Settings → Content Sync မှာ settings တွေကို configure လုပ်ပါ။
၄။ Sync လုပ်တာ စတင်ဖို့ Ultimate Multisite → Content Sync ကို သွားပါ။
