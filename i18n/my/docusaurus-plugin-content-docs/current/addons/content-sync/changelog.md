# Content Sync ပြောင်းလဲမှု မှတ်တမ်း

### Version 1.1.0 - 2025-12-31 တွင် ထုတ်ဝေသည်
* အသစ်: Post အမျိုးအစားအားလုံး (posts, pages, custom post types) ကို Sync လုပ်နိုင်မှု ပံ့ပိုးပေးသည်
* အသစ်: Content ရွေးချယ်သည့် အဆင့်တွင် Post အမျိုးအစား စစ်ထုတ်ခြင်း (Post type filtering) ကို ထည့်သွင်းသည်
* အသစ်: Source site ရွေးချယ်သည့် အဆင့် - အဓိက site တစ်ခုတည်းမှ မဟုတ်ဘဲ မည်သည့် site မှမဆို Sync လုပ်နိုင်သည်
* အသစ်: Template site များအတွက် Child site ကို ကြိုတင်ရွေးချယ်နိုင်မှု
* အသစ်: Post အမျိုးအစားအားလုံးမှ post_content မှ Media များကို ထုတ်ယူနိုင်မှု
* အသစ်: အမျိုးအစားခွဲခြားမှု Sync လုပ်နိုင်မှု (categories, tags, custom taxonomies)
* ပြောင်းလဲမှု: "Elementor Template Sync" မှ "Content Sync" ဟု အမည်ပြောင်းလဲသည်
* ပြောင်းလဲမှု: "Select Templates" အဆင့်ကို "Select Data" ဟု အမည်ပြောင်းလဲသည်
* ပိုမိုကောင်းမွန်စေသည်: Elementor သီးသန့် code များသည် elementor_library post type အတွက်သာ အလုပ်လုပ်စေရန် ကန့်သတ်ထားသည်

### Version 1.0.0 - 2025-12-19 တွင် ထုတ်ဝေသည်
* ပထမဆုံး ထုတ်ဝေမှု
* Template နှင့် site ရွေးချယ်မှုအတွက် အဆင့်များစွာပါသော လမ်းညွှန် (Multi-step wizard)
* အချိန်နှင့်တပြေးညီ တိုးတက်မှု အကြောင်းပြန်အသိပေးမှုဖြင့် အစုလိုက် လုပ်ဆောင်ခြင်း (Batch processing)
* ID ကို အလိုအလျောက် ပြန်လည်သတ်မှတ်ပေးသည့် (automatic ID remapping) Media အပြည့်အဝ Sync လုပ်နိုင်မှု
* Ultimate Multisite နှင့် Settings ပေါင်းစပ်မှု
* Network settings မှတစ်ဆင့် site ကို ချန်လှပ်ထားနိုင်မှု (Site exclusion support)
* Sync လုပ်ပြီးနောက် Elementor cache ကို ရှင်းလင်းပေးခြင်း
* Template taxonomy များကို ထိန်းသိမ်းထားနိုင်မှု
