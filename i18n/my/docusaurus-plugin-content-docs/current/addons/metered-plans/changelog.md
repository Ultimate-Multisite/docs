# Metered Plans ပြောင်းလဲမှုမှတ်တမ်း {#metered-plans-changelog}

Version 1.1.0 - 2026-05-05 တွင် ထုတ်ဝေခဲ့သည်
- အသစ်: multisite subsite များအတွက် AI token ငွေတောင်းခံခြင်း — စီမံကိန်းရနိုင်သော token တစ်ခုချင်းစီအတွက် နှုန်းထားများဖြင့် ဖောက်သည်ဆိုင်များတစ်လျှောက် AI token အသုံးပြုမှုကို ခြေရာခံပြီး ငွေတောင်းခံနိုင်ပြီ။
- အသစ်: Connector ကို အားဖြည့်မှု ပြန်လည်တည်ဆောက်ခဲ့ပြီး၊ dynamic limit discovery နှင့် write-through တို့ဖြင့် အကောင်အထည်ဖော်ကာ၊ connector အားလုံးတွင် အချိန်နှင့်တပြေးညီ တိကျမှုရှိစေသည်။
- ပြင်ဆင်မှု: Database စားပွဲ အဆင့်မြှင့်တင်မှုများသည် ယခုအခါ BerlinDB schema အဓိပ္ပာယ်ဖွင့်ဆိုချက်များနှင့် ကိုက်ညီစွာ အဆင့်မြှင့်တင်နိုင်ပြီဖြစ်၍၊ စနစ်အသစ် တပ်ဆင်ချိန်တွင် အဆင့်မြှင့်မှု ပျက်ကွက်ခြင်းကို ကာကွယ်ပေးသည်။
- ပြင်ဆင်မှု: Database အဆင့်မြှင့်တင်မှု ပြန်ခေါ်မှု (callbacks) များကို မှန်ကန်သော ပုံစံသို့ ပြောင်းလဲခဲ့ရာ၊ တိတ်ဆိတ်စွာ အဆင့်မြှင့်မှု ပျက်ကွက်ခြင်း ပြဿနာကို ဖြေရှင်းပေးခဲ့သည်။
- ပြင်ဆင်မှု: AI Usage Overage Markup input field တွင် အပိုင်းအစ တန်ဖိုးများကို လက်ခံနိုင်ပြီ။
- ပြင်ဆင်မှု: plugin ကို တင်ဆောင်ချိန်တွင် ဖြစ်ပေါ်သည့် အလွန်ပြင်းထန်သော အမှားများနှင့် နှစ်ခါစတင်မှု ပြဿနာများကို ဖြေရှင်းပေးခဲ့သည်။
- ပိုမိုကောင်းမွန်စေသည်: check-env npm script ကို ထည့်သွင်းခဲ့ရာ၊ developer environment များသည် ပထမဆုံးအကြိမ် အသုံးပြုချိန်တွင် ကိုယ်တိုင်ဖွဲ့စည်းပေးနိုင်ပြီ။
