# WooCommerce Integration

WooCommerce Integration သည် ကျွန်ုပ်တို့၏ Manual Gateway လုပ်ဆောင်ပုံနှင့် အလွန်နီးစပ်စွာ အလုပ်လုပ်ပါသည်။ ၎င်းသည် သင့်အသုံးပြုသူများအား အလွန်ပြ Flexibility များ ပေးစွမ်းနိုင်ပြီး၊ ၎င်းတို့၏ WooCommerce ထည့်သွင်းထားသည့် နေရာရှိ ရရှိနိုင်သော ငွေပေးချေမှု ပုံစံများထဲမှ မည်သည့်ပုံစံကိုမဆို ရွေးချယ်ခွင့် ပေးပါသည်။ ၎င်းသည် ရိုးရှင်းပါသည်။ အသုံးပြုသူ၏ ငွေပေးချေမှု ကာလ ကုန်ဆုံးသွားသည်နှင့်၊ အသုံးပြုသူ၏ စာရင်းပေးသွင်းမှုသည် ခေတ္တရပ်ဆိုင်းသွားမည်ဖြစ်သည်။ တစ်ချိန်တည်းမှာပင်၊ network ၏ အဓိက site တွင် WooCommerce ထည့်သွင်းထားသည့် နေရာ၌ အော်ဒါတစ်ခု ဖန်တီးပေးမည်ဖြစ်သည်။ ထို့နောက် အသုံးပြုသူသည် ငွေပေးချေမှု ပြုလုပ်ရန် လင့်ခ်ပါသည့် ငွေတောင်းခံလွှာ (invoice) ကို ၎င်းတို့၏ အီးမေးလ်မှ ရရှိမည်ဖြစ်သည်။ ငွေပေးချေမှု ရရှိသည်နှင့် စာရင်းပေးသွင်းမှုသည် သက်တမ်းတိုးသွားမည်ဖြစ်သည်။

## Features

- Ultimate Multisite နှင့် WooCommerce တို့အကြား အလွယ်တကူ ပေါင်းစပ်နိုင်ခြင်း
- ငွေပေးချေမှု နည်းလမ်းများစွာ ရရှိနိုင်ခြင်း
- WooCommerce Subscriptions ဖြင့် လုံးဝအလိုအလျောက် လုပ်ဆောင်နိုင်ခြင်း

## Installation

1. 'ultimate-multisite-woocommerce' ကို '/wp-content/plugins/' directory သို့ တင်ပါ။
2. WordPress ၏ 'Plugins' menu မှတစ်ဆင့် plugin ကို အသက်သွင်းပါ။
3. plugin ကို စနစ်တကျ တည်ဆောက်ရန် အဆင့်ဆင့် လမ်းညွှန် (Wizard) ကို လိုက်နာပါ။

## Usage Guide

_**အရေးကြီးမှတ်ချက်- ဤဆောင်းပါးသည် Ultimate Multisite version 2.x ကို ရည်ညွှန်းပါသည်။**_

_**သတိပြုရန်- Ultimate Multisite: WooCommerce Integration ကို အသုံးပြုရန်အတွက် WooCommerce ကို အနည်းဆုံး သင့်၏ အဓိက site တွင် အသက်သွင်းထားရန် လိုအပ်ပါသည်။**_

_Stripe_ နှင့် _PayPal_ တို့သည် နိုင်ငံအချို့တွင် မရရှိနိုင်ခြင်းကြောင့် Ultimate Multisite အသုံးပြုသူများအနေဖြင့် ကျွန်ုပ်တို့၏ plugin ကို ထိရောက်စွာ အသုံးပြုရာတွင် ကန့်သတ်မှုများ သို့မဟုတ် အခက်အခဲများ ရှိနိုင်သည်ကို ကျွန်ုပ်တို့ နားလည်ပါသည်။ ထို့ကြောင့် _WooCommerce_ ကို ပေါင်းစပ်ရန်အတွက် **add-on တစ်ခုကို ဖန်တီးထားခြင်း** ဖြစ်ပါသည်။

_WooCommerce_ သည် အလွန်ရေပန်းစားသော eCommerce plugin တစ်ခုဖြစ်သည်။ ကမ္ဘာတစ်ဝန်းရှိ developer များက ၎င်းနှင့် မတူညီသော ငွေပေးချေမှု ဂိတ်ဝေးများ ပေါင်းစပ်ရန် add-ons များကို ဖန်တီးခဲ့ကြသည်။ ကျွန်ုပ်တို့သည် ဤအချက်ကို အခွင့်ကောင်းယူကာ Ultimate Multisite billing system တွင် အသုံးပြုနိုင်သည့် ငွေပေးချေမှု ဂိတ်ဝေးများကို ချဲ့ထွင်နိုင်ခဲ့ပါသည်။

## Setting It Up

ပထမဦးစွာ၊ **လိုင်စင်ရရှိသူများအတွက် အခမဲ့** ဖြစ်သည့် WooCommerce Integration add-on ကို ထည့်သွင်းရန် လိုအပ်ပါသည်။ ၎င်းကို ထည့်သွင်းရန်၊ သင့် super admin dashboard တွင် Ultimate Multisite ကို နှိပ်ပြီး **Add-ons** ကို ရွေးချယ်ပါ။

![Add-ons page](/img/admin/addons-page.png)

ဤနေရာတွင် Ultimate Multisite add-ons အားလုံးကို တွေ့ရှိနိုင်ပါသည်။ **Ultimate Multisite: WooCommerce Integration** add-on ကို နှိပ်ပါ။

![Add-ons page showing WooCommerce Integration](/img/admin/addons-page.png)

add-on အသေးစိတ်အချက်အလက်များပါသည့် window တစ်ခု ပေါ်လာပါမည်။ **Install Now** ကို နှိပ်ရုံသာ ဖြစ်ပါသည်။

ထည့်သွင်းမှု ပြီးစီးသွားပါက၊ plugin များ စာမျက်နှာသို့ ပြန်လည်ရောက်ရှိသွားပါမည်။ ဤနေရာတွင် **Network Activate** ကို နှိပ်ရုံဖြင့် _WooCommerce_ add-on သည် သင့် network တစ်ခုလုံးတွင် အသက်သွင်းသွားမည်ဖြစ်သည်။

ထည့်သွင်းပြီးပါက၊ Ultimate Multisite payments settings အောက်တွင် အပိုရွေးချယ်မှုတစ်ခုကို မြင်ရမည်ဖြစ်သည်။ ထို့မှတစ်ဆင့် _WooCommerce_ option ကို **ဖွင့်ပေး** နိုင်ပါသည်။

![Payment gateway settings](/img/config/settings-payment-gateways.png)

Front end တွင် ပြသမည့် display name ကဲ့သို့သော အခြေခံအချက်အလက်အချို့နှင့် ၎င်းတို့၏ checkout flow အတွက် လမ်းညွှန်မှုပေးမည့် ညွှန်ကြားချက်များကို စနစ်တကျ ပြုလုပ်နိုင်ပါသည်။ ထို့နောက် သင်နှစ်သက်သည့် ငွေပေးချေမှု ဂိတ်ဝေးကို ဖွင့်ပြီး စနစ်တကျ ပြုလုပ်ရန်အတွက် သင့် **Main site ၏ _WooCommerce_ settings** သို့ သွားရောက်နိုင်ပါသည်။

## How to manage the payment

WooCommerce Integration သည် ကျွန်ုပ်တို့၏ **Manual Gateway** လုပ်ဆောင်ပုံနှင့် အလွန်နီးစပ်စွာ အလုပ်လုပ်ပါသည်။ အားနည်းချက်မှာ ငွေပေးချေမှုတိုင်းကို ငွေပေးချေမှု ကာလကုန်ဆုံးချိန်တွင် လက်ဖြင့် ပေးချေရမည်ဖြစ်သည်။ လုပ်ဆောင်ပုံ အဆင့်ဆင့်မှာ အောက်ပါအတိုင်း ဖြစ်သည်-

  * အသုံးပြုသူ၏ ငွေပေးချေမှု cycle ကုန်ဆုံးသည်နှင့်၊ အဖွဲ့ဝင်အဖြစ် စာရင်းသွင်းထားမှုကို **ခေတ္တရပ်ဆိုင်းထားမည်**။ တစ်ချိန်တည်းမှာပင်၊ network ၏ main site တွင် _WooCommerce_ ထည့်သွင်းထားသည့် နေရာ၌ အော်ဒါတစ်ခု ဖန်တီးပေးမည်။

  * အသုံးပြုသူများသည် ငွေပေးချေမှု ပြုလုပ်ရန် လင့်ခ်ပါသည့် ငွေတောင်းခံလွှာ (invoice) ကို ၎င်းတို့၏ အီးမေးလ်မှ ရရှိမည် (ထိုလင့်ခ်သည် ၎င်းတို့၏ subsite dashboard မှ ငွေပေးချေမှု မှတ်တမ်းတွင်လည်း ပေါ်နေပါမည်)။ ထိုလင့်ခ်သည် အသုံးပြုသူကို _**WooCommerce**_ **checkout** သို့ ပြန်လည်ညွှန်ပြပေးမည်ဖြစ်ပြီး၊ ၎င်းတို့သည် ၎င်းတို့၏ ငွေပေးချေမှု ပြုလုပ်ရန်အတွက် **ရရှိနိုင်သော _WooCommerce_ ဂိတ်ဝေးများ**ထဲမှ တစ်ခုကို ရွေးချယ်နိုင်မည်ဖြစ်သည်။

  * ငွေပေးချေမှု ရရှိသည်နှင့်၊ အဖွဲ့ဝင်အဖြစ် စာရင်းသွင်းမှုသည် **သက်တမ်းတိုးသွားမည်**၊ ငွေပေးချေမှုကို Ultimate Multisite တွင် မှတ်တမ်းတင်မည်ဖြစ်ပြီး အော်ဒါကို ပြီးစီးသည်အဖြစ် သတ်မှတ်ပေးမည်။

### Some Important Notes:

  * Ultimate Multisite ၏ default လုပ်ဆောင်ချက်များသည် ဤပေါင်းစပ်မှု ပုံစံနှင့် ဆက်လက်အလုပ်လုပ်နေပါသည်။ ဥပမာအားဖြင့်၊ **Subscription Management** screen မှ **ငွေပြန်အမ်းခြင်း** ပြုလုပ်ခြင်းသည် ရွေးချယ်ထားသော ငွေပေးချေမှု ပုံစံကို အသုံးပြု၍ ငွေပြန်အမ်းရန် WooCommerce နှင့် ဆက်သွယ်ပေးမည်ဖြစ်သည်။

  * Ultimate Multisite: WooCommerce Integration သည် အလုပ်လုပ်ရန် WP-Cron ကို အားထားရပါသည်။ ၎င်းသည် အချိန်အများစုတွင် အလွန်ကောင်းမွန်စွာ အလုပ်လုပ်သော်လည်း၊ site တွင် traffic နည်းပါက cron သည် စီစဉ်ထားသော အဖြစ်အပျက်များကို စစ်ဆေးရန် အကြောင်းကြားခြင်းမရှိဘဲ ထိုအဖြစ်အပျက်ကို လွတ်သွားစေနိုင်ပြီး၊ ၎င်းသည် လုပ်ဆောင်မှု နှောင့်နှေးခြင်းကို ဖြစ်စေနိုင်သည်။

## Woocommerce Subscriptions Plugin

လစဉ် ငွေပေးချေမှုများကို လက်ဖြင့် စတင်ပေးရခြင်းသည် အသုံးပြုသူများအတွက် အကောင်းဆုံး မဟုတ်ပေ၊ အကြောင်းမှာ စာရင်းပေးသွင်းမှုများသည် ငွေပေးချေမှု cycle အသစ်တိုင်းတွင် အလိုအလျောက် ပြန်လည်ဖြစ်ပေါ်ရန် မျှော်လင့်ထားသောကြောင့် ဖြစ်သည်။ ထို့ကြောင့် ကျွန်ုပ်တို့သည် WooCommerce Integration add-on ကို **WooCommerce Subscriptions plugin ကို ပံ့ပိုးပေးနိုင်ရန်** ဒီဇိုင်းထုတ်ထားပါသည်။ ၎င်းသည် ငွေပေးချေမှုများကို အလိုအလျောက်လုပ်ဆောင်ပေးခြင်းဖြင့် သင့်အသုံးပြုသူများအတွက် အရောင်းအဝယ်ကို ပိုမိုလွယ်ကူစေမည်ဖြစ်သည်။

သင်လုပ်ရမှာက [Woocommerce Subscriptions plugin](https://woocommerce.com/products/woocommerce-subscriptions/) ကို **ထည့်သွင်း**ပြီး ၎င်းကို **သင့်၏ main site တွင် အသက်သွင်းထားရန်** လိုအပ်ရုံသာ ဖြစ်သည်။ ထို့နောက် Ultimate Multisite payments settings သို့ ပြန်သွားပြီး **integration ကို ဖွင့်ပေး**ပါ။

ပြီးပါပြီ။ သင့်အသုံးပြုသူများသည် ၎င်းတို့၏ ငွေပေးချေမှု cycle ကုန်ဆုံးချိန်တွင် လက်ဖြင့် ငွေပေးချေရန် မလိုအပ်တော့ပေ။
