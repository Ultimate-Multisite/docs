# WooCommerce Integration ပြောင်းလဲမှုမှတ်တမ်း

Version 2.0.6 - 2026-01-16 တွင် ထုတ်ဝေခဲ့သည်
* တိုးတက်မှု- အဓိက Subscription များကို add-on ထဲမှာ ထည့်သွင်းထားပါတယ်။ Woocommerce Subscriptions extension ကိုတော့ မလိုတော့ပါဘူး။

Version 2.0.5 - 2026-01-09 တွင် ထုတ်ဝေခဲ့သည်
* တိုးတက်မှု- glotpress API မှ ဘာသာပြန်မှုများကို တင်ယူနိုင်ပြီ။
* ပြင်ဆင်မှု- အချို့သော page builder များတွင် ဖြစ်ပေါ်ခဲ့သည့် အလွန်ပြင်းထန်သော အမှား (Fatal error) ကို ပြင်ဆင်ခဲ့သည်။
* ပြင်ဆင်မှု- ဖောက်သည်သည် main site ၏ အဖွဲ့ဝင်ဖြစ်ပါက အဆုံးမရှိ ပြန်လည်လွှဲခြင်း (Infinite redirect) ကို ပြင်ဆင်ခဲ့သည်။

Version 2.0.4 - 2025-11-14 တွင် ထုတ်ဝေခဲ့သည်
* ထည့်သွင်းခဲ့သည်- ဘာသာစကား အများအပြားအတွက် ဘာသာပြန်မှုများ။
* ပြောင်းလဲခဲ့သည်- အမည်ကို Ultimate Multisite: Woocommerce Integration ဟု ပြောင်းလဲခဲ့သည်။
* ထည့်သွင်းခဲ့သည်- Woocommerce 10.2.1 နှင့် တွဲဖက်အသုံးပြုနိုင်မှု။
* ထည့်သွင်းခဲ့သည်- Woocommerce Subscriptions 7.7.0 နှင့် တွဲဖက်အသုံးပြုနိုင်မှု။
* ပြင်ဆင်မှု- PHP 8.4 နှင့် တွဲဖက်အသုံးပြုနိုင်မှု။
* ပြင်ဆင်မှု- WC account page မရှိသည့်အခါ ပြန်လည်လွှဲခြင်း (Redirect) ပြဿနာကို ပြင်ဆင်ခဲ့သည်။

Version 2.0.3 - 2025-08-13 တွင် ထုတ်ဝေခဲ့သည်
* ပြောင်းလဲခဲ့သည်- စျေးကွက်သစ် (new marketplace) ဖြင့် အလိုအလျောက် update လုပ်နိုင်အောင် လုပ်ဆောင်ပေးခဲ့သည်။

Version 2.0.2 - 2025-07-05 တွင် ထုတ်ဝေခဲ့သည်
* ပြောင်းလဲခဲ့သည်- အမည်ကို Multisite Ultimate: Woocommerce Integration ဟု ပြောင်းလဲခဲ့သည်။
* ထည့်သွင်းခဲ့သည်- Woocommerce 9.8.1 နှင့် တွဲဖက်အသုံးပြုနိုင်မှု။
* ထည့်သွင်းခဲ့သည်- Woocommerce Subscriptions 7.3.0 နှင့် တွဲဖက်အသုံးပြုနိုင်မှု။
* ပြင်ဆင်မှု- ဖောက်သည်မှ Subscription ကို ပယ်ဖျက်ခြင်း။
* ပြင်ဆင်မှု- checkout block ကို အသုံးပြုသည့်အခါ ဖြစ်ပေါ်ခဲ့သည့် အလွန်ပြင်းထန်သော အမှား (Fatal error)။
* တိုးတက်မှု- Woocommerce high performance custom order tables နှင့် တွဲဖက်အသုံးပြုနိုင်ပြီ။
* ပြင်ဆင်မှု- WooCommerce checkout တွင် ပယ်ဖျက်ခြင်း လုပ်ဆောင်သည့်တိုင် အဖွဲ့ဝင်အဆင့်ကို အဆင့်မြှင့်နိုင်ခြင်း။

Version 2.0.1 - 2023-08-09 တွင် ထုတ်ဝေခဲ့သည်

* ထည့်သွင်းခဲ့သည်- Woocommerce 7.9.0 နှင့် တွဲဖက်အသုံးပြုနိုင်မှု။
* ထည့်သွင်းခဲ့သည်- Woocommerce Subscriptions 5.3.0 နှင့် တွဲဖက်အသုံးပြုနိုင်မှု။
* ထည့်သွင်းခဲ့သည်- အဖွဲ့ဝင်အဆင့် (membership) အဆင့်မြှင့်မှုများကို ပံ့ပိုးပေးနိုင်မှု။
* ထည့်သွင်းခဲ့သည်- Woocommerce တွင် စမ်းသပ်ကာလ (trials) နှင့် စတင်တည်ဆောက်ခ (setup fees) များအကြောင်း အသိပေးချက်များ။
* ထည့်သွင်းခဲ့သည်- meta value ဖြင့် Ultimate Multisite Woocommerce ထုတ်ကုန်များကို ခွဲခြားသတ်မှတ်နိုင်မှု။
* ထည့်သွင်းခဲ့သည်- Ultimate Multisite နှင့် သက်ဆိုင်သော Woocommerce ထုတ်ကုန်အားလုံးကို mark လုပ်ရန်အတွက် တစ်ကြိမ်တည်း ပြင်ဆင်မှု (one-time fix) ကို ထည့်သွင်းခဲ့သည်။
* ထည့်သွင်းခဲ့သည်- Woocommerce စာရင်းမှ Ultimate Multisite ဖန်တီးထားသော ထုတ်ကုန်များကို ဖယ်ရှားခဲ့သည်။
* တိုးတက်မှု- cart တွင် အသုံးပြုရန်အတွက် non-recurring Woocommerce discount တစ်ခုကို ဖန်တီးပေးနိုင်ခဲ့သည်။
* တိုးတက်မှု- Woocommerce ထုတ်ကုန်သို့ recurring discount ကို ပြန်လည်ထည့်သွင်းပေးခဲ့သည်။
* တိုးတက်မှု- Woocommerce ထုတ်ကုန်သို့ recurring discount label ကို ထည့်သွင်းပေးခဲ့သည်။
* တိုးတက်မှု- checkout တွင် product type ကို သေချာစေခဲ့သည်။
* ပြင်ဆင်မှု- အဆင့်လျှော့ချသည့် လုပ်ငန်းစဉ်အတွင်း အဖွဲ့ဝင်အဆင့်ကို ထိန်းသိမ်းထားနိုင်ခဲ့သည်။
* ပြင်ဆင်မှု- ပယ်ဖျက်ခြင်း လုပ်ငန်းစဉ်အတွင်း အမှားများမဖြစ်စေရန် Subscription ရှိ၊ မရှိ စစ်ဆေးခဲ့သည်။
* ပြင်ဆင်မှု- Woocommerce subscriptions တွင် အသုံးပြုရန်အတွက် စတင် Subscription ရက်စွဲကို ထည့်သွင်းခဲ့သည်။
* အတွင်းပိုင်း- PHP 8.1 build process အသစ်ကို အကောင်အထည်ဖော်ခဲ့သည်။

Version 2.0.0 - အလုံးစုံ ပြန်လည်ရေးသားခဲ့သည်။

* ထည့်သွင်းခဲ့သည်: gateway ကို ပြောင်းလဲခြင်း သို့မဟုတ် အဖွဲ့ဝင်အဆင့်ကို ပယ်ဖျက်ခြင်းတို့ လုပ်သည့်အခါ woo subscription ကို ဖယ်ရှားသည့် process cancellation method ကို ထည့်သွင်းခဲ့သည်;
* ထည့်သွင်းခဲ့သည်: အဖွဲ့ဝင်အဆင့်ကို အဆင့်လျှော့ချခြင်းနှင့် အဆင့်မြှင့်ခြင်းများအတွက် handler ကို ထည့်သွင်းခဲ့သည်;
* တိုးတက်မှု: account update လုပ်နိုင်ရန် subsites ရှိ customer update form တွင် woocommerce dependencies များကို load လုပ်ပေးနိုင်ခဲ့သည်;
* တိုးတက်မှု: Woocommerce cart မရှိပါက မှန်ကန်စွာ load လုပ်ပေးနိုင်ခဲ့သည်;
* တိုးတက်မှု: checkout လုပ်ငန်းစဉ်ကို လုပ်ဆောင်သည့်အခါ main site tables တွင် ရှိနေကြောင်း သေချာစေခဲ့သည်;
* တိုးတက်မှု: Ultimo renewal order ကို နောက်ဆုံးပေးချေမှုမှ မဟုတ်ဘဲ Woocommerce subscription order value အပေါ် အခြေခံအောင် လုပ်ဆောင်ပေးခဲ့သည်;
* ပြင်ဆင်မှု: WU Membership button link သို့ သွားရောက်မှု။
* ပြင်ဆင်မှု: Woocommerce subscriptions renewal သည် ငွေပေးချေမှု အောင်မြင်ပါက Ultimo order ကို paid အဖြစ် သတ်မှတ်ပေးခဲ့သည်။
* build: MPB ကို builder အဖြစ် ထည့်သွင်းခဲ့သည်။
