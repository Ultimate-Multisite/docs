# WooCommerce ပေါင်းစည်းမှု ပြောင်းလဲမှုမှတ်တမ်း {#woocommerce-integration-changelog}

ဗားရှင်း 2.2.0 - 2026-07-01 တွင် ထုတ်ပြန်ခဲ့သည်
* အသစ်: Ultimate Multisite အခွန်ပမာဏများကို checkout တွင် သီးခြား WooCommerce အခကြေးငွေလိုင်းများအဖြစ် ယခု ပြသပြီး ငွေပေးချေမှုမတိုင်မီ အခွန်စုစုပေါင်းကို ပိုမိုရှင်းလင်းစေသည်။
* အသစ်: WooCommerce Subscriptions ပြန်လည်ကြိုးစားသည့်ကာလအတွင်း သက်တမ်းတိုးမှုမအောင်မြင်လျှင် ရပ်ဆိုင်းလိုသော site များအတွက် "Suspend Memberships Immediately on Failed Renewals" ဆက်တင်နှင့် `wu_woo_suspend_on_payment_failure` filter ကို ရွေးချယ်အသုံးပြုနိုင်ရန် ထည့်သွင်းခဲ့သည်။
* ပြင်ဆင်ချက်: သက်တမ်းတိုးမှု မအောင်မြင်ခြင်း သို့မဟုတ် ပြန်လည်အောင်မြင်ခြင်းနောက် Ultimate Multisite membership များနှင့် မညီမညာ ဖြစ်နေနိုင်သော WooCommerce Subscription အခြေအနေများကို ပြန်လည်ညှိနှိုင်းခဲ့သည်။
* ပြင်ဆင်ချက်: မရှိနေသည့်အခါ WooCommerce စတိုးငွေကြေးကို Ultimate Multisite ငွေကြေးစာရင်းသို့ ထည့်သွင်းခဲ့သည်။
* ပြင်ဆင်ချက်: subscriber များကို WooCommerce checkout သို့ ပြန်ညွှန်ချိန်တွင် customer billing အသေးစိတ်များကို ထိန်းသိမ်းထားခဲ့သည်။
* မြှင့်တင်ထားသည်: Jetpack Autoloader 5 နှင့် ကိုက်ညီမှု ထည့်သွင်းခဲ့သည်။
* မြှင့်တင်ထားသည်: GitHub နှင့် marketplace zip များတွင် အတွင်းထပ် staging လမ်းညွှန်များနှင့် development ဖိုင်များ မပါဝင်စေရန် release package ထုတ်လုပ်မှုကို သန့်ရှင်းပြင်ဆင်ခဲ့သည်။
