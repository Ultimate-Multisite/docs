# ပြောင်းလဲမှုမှတ်တမ်း (Changelog)

## 1.16.0 — 2026-05-20 တွင် ထုတ်ဝေခဲ့သည်

### အသစ်များ (New)

- **Logo SVG ဖန်တီးနိုင်မှု** — Theme Builder သည် ယခုအခါ namespace-safe sanitisation ဖြင့် စိတ်ကြိုက် logo SVG များကို ဖန်တီးပြီး ထည့်သွင်းနိုင်ပြီဖြစ်သည်။
- **တွေ့ရှိချက် အင်တာဗျူးတွင် ဓာတ်ပုံတင်နိုင်မှု** — Theme Builder discovery interview တွင် ပိုမိုပြည့်စုံသော ဒီဇိုင်းအကြောင်းအရာများအတွက် ဓာတ်ပုံတင်သည့် အဆင့်ကို ထည့်သွင်းထားသည်။
- **Palette Contrast စစ်ဆေးနိုင်မှု** — theme တစ်ခုတွင် အသုံးပြုမည့် အရောင်အတွဲများကို WCAG လိုက်နာမှု ရှိ၊ မရှိ စစ်ဆေးနိုင်ပြီ။
- **ဧည့်ခံကျွေးမွေးမှု မီနူးများ** — Theme Builder သည် ဧည့်ခံကျွေးမွေးမှု လုပ်ငန်းများအတွက် စနစ်တကျဖွဲ့စည်းထားသော အစားအသောက်နှင့် အဖျော်ယမကာ မီနူးစာမျက်နှာများကို ဖန်တီးပေးနိုင်ပြီ။
- **Desktop နှင့် mobile ကြိုတင်ကြည့်ရှုခြင်း** — ဒီဇိုင်းလမ်းညွှန်ရွေးချယ်မှုများ လုပ်နေစဉ်အတွင်း desktop နှင့် mobile စက်ပစ္စည်းများပေါ်တွင် မိမိ၏ ဒီဇိုင်းကို ကြိုတင်ကြည့်ရှုနိုင်ပြီ။
- **Navigation label parameter** — Create Menu စွမ်းဆောင်ရည်သည် ယခုအခါ စာမျက်နှာခေါင်းစဉ်နှင့် သီးခြားဖြစ်သော `navigation_label` ကို ပံ့ပိုးပေးနိုင်ပြီ။
- **Tier 1 tool ရရှိနိုင်မှု** — sd-ai-agent/site-scrape သည် Theme Builder တွင် default အနေဖြင့် ရရှိနိုင်သော Tier 1 tool တစ်ခု ဖြစ်လာပြီ။

### ပြင်ဆင်ခဲ့သည်များ (Fixed)

- **AI Client cache** — ယခုအခါ cross-request persistence အတွက် transients ဖြင့် ပံ့ပိုးထားသောကြောင့် အချိန်ကြာမြင့်စွာ လုပ်ဆောင်ရသော agent လုပ်ငန်းများတွင် အချက်အလက်များ ပျောက်ဆုံးခြင်းကို ကာကွယ်ပေးနိုင်ပြီ။
- **Plugin အတန်းအဆင့် လုပ်ဆောင်ချက် link များ** — ရှင်းလင်းစေရန် ပြင်ဆင်ပြီး အမည်ပြောင်းလဲပေးခဲ့သည်။

## 1.10.0 — 2026-05-05 တွင် ထုတ်ဝေခဲ့သည်

### အသစ်များ (New)

- **Tavily အင်တာနက် ရှာဖွေမှု** — Brave Search နှင့်အတူ ပိုမိုပြည့်စုံသော အင်တာနက် ရှာဖွေမှုရလဒ်များအတွက် Tavily ကို ရှာဖွေမှုပေးစပ်တစ်ခုအဖြစ် ထည့်သွင်းပေးခဲ့သည်။
- **Theme အသိရှိသော built-in စွမ်းရည်များ** — Block Themes, Classic Themes, Kadence Blocks နှင့် Kadence Theme စွမ်းရည်လမ်းညွှန်များကို plugin နှင့်အတူ ပါဝင်လာပြီ။
- **Site builder ဖြင့် ဆက်သွယ်ရန် ဖောင် တင်နိုင်မှု** — chat interface မှတစ်ဆင့် မည်သည့် စာမျက်နှာမဆို ဆက်သွယ်ရန် ဖောင်ကို တိုက်ရိုက်ထည့်သွင်းနိုင်ပြီ။

### ပိုမိုကောင်းမွန်စေသည်များ (Improved)

- **WooCommerce ပေါင်းစပ်မှုကို ပြန်လည်ဖွဲ့စည်းခြင်း** — ယခုအခါ ယုံကြည်စိတ်ချရမှုနှင့် တွဲဖက်အသုံးပြုနိုင်မှု ပိုမိုကောင်းမွန်စေရန် native WooCommerce APIs များကို အသုံးပြုထားသည်။
- **ပေးစပ်စာရင်းကို အလိုအလျောက် Refresh လုပ်ခြင်း** — plugin တစ်ခုခုကို activate လုပ်သည်ဖြစ်စေ၊ deactivate လုပ်သည်ဖြစ်စေ ပေးစပ်စာရင်းကို အလိုအလျောက် Refresh လုပ်ပေးသည်။

### ပြင်ဆင်ခဲ့သည်များ (Fixed)

- **navigate-to စွမ်းရည်** — အချို့ admin စာမျက်နှာများတွင် အဆုံးမရှိ reload loop ဖြစ်ခြင်းကို ပြင်ဆင်ခဲ့သည်။
- **list-posts စွမ်းရည်** — category နှင့် tag အမည်များကို slugs များအဖြစ် မှန်ကန်စွာ ဖြေရှင်းပေးနိုင်ပြီ။
- **WP-CLI command များ** — ယခင် ပြန်လည်ဖွဲ့စည်းမှုတစ်ခုအပြီး ပျောက်ဆုံးသွားသော namespace aliases များကို ပြန်လည်ရရှိစေခဲ့သည်။
- **Event automation** — automation tables များ မဖန်တီးရသေးသော site များကိုလည်း ချောမွေ့စွာ ကိုင်တွယ်နိုင်ပြီ။
- **memory-save စွမ်းရည်** — system instruction builder တွင် မှန်ကန်သော namespace prefix ကို အသုံးပြုထားပြီ။
- **Scalar tool ရလဒ်များ** — AI သို့ ပြန်ပေးမည့် အချိန်တွင် မှန်ကန်စွာ wrap လုပ်ပေးထားပြီ။
- **အသုံးပြုမှု စာရင်းအင်း** — ယခင် version များမှ upgrade လုပ်လာပါက legacy ability key format များကို မှန်ကန်စွာ ကိုင်တွယ်နိုင်ပြီ။
