# ပြောင်းလဲမှုမှတ်တမ်း

## 1.18.0 — 2026-06-29 တွင် ထုတ်ဝေခဲ့သည် {#1180--released-on-2026-06-29}

### အသစ် {#new}

- **Google Calendar ကိရိယာများ** — အချိန်ဇယားသိရှိသော အလိုအလျောက်လုပ်ဆောင်မှုများအတွက် သတ်မှတ်ထားသော ပြက္ခဒိန်များနှင့် အစီအစဉ်များကို ဖတ်ပါ
- **ဆက်သွယ်ရန်အချက်အလက် ချိတ်ဆက်ခြင်းနှင့် တက်ရောက်သူ ကူညီကိရိယာများ** — ပြက္ခဒိန်ပါဝင်သူများကို ဆိုက်အသုံးပြုသူများနှင့် ဆက်သွယ်ရန်အချက်အလက်များနှင့် ကိုက်ညီအောင်ချိတ်ဆက်ပါ
- **လူမှ အတည်ပြုမှု တံခါးများနှင့် သတိပေးချက် မှတ်တမ်းများ** — ပြန်လည်သုံးသပ်ရန် အလိုအလျောက်လုပ်ဆောင်မှုများကို ရပ်နားပြီး ထပ်နေသော အသိပေးချက်များကို ရှောင်ပါ
- **TextBee SMS ပံ့ပိုးသူ** — သတ်မှတ်ထားသော စာတိုအသိပေးချက်များကို TextBee မှတစ်ဆင့် ပို့ပါ
- **အဆင့်မြင့် တွဲဖက်ပက်ကေ့ဂျ်** — ယုံကြည်ရသော developer ဖိုင်စနစ်၊ ဒေတာဘေ့စ်၊ WP-CLI၊ REST dispatcher၊ plugin builder၊ git snapshot၊ အသုံးပြုသူစီမံခန့်ခွဲမှုနှင့် benchmark ကိရိယာများကို WordPress.org build မှ သီးခြားဖြန့်ချိထားပြီး ထည့်ပါ

### တိုးတက်မှုများ {#improved}

- **စီမံထားသော Superdav ဝန်ဆောင်မှု တပ်ဆင်မှု** — ပံ့ပိုးထားသော ဆိုက်များအတွက် hosted service endpoint များနှင့် အလိုအလျောက် ချိတ်ဆက်မှုပြင်ဆင်ခြင်းကို ထည့်ပါ
- **ထုတ်ဝေမှု ပက်ကေ့ဂျ်ပြုလုပ်ခြင်း** — core နှင့် Advanced ZIP များကို သီးခြားတည်ဆောက်ပြီး GitHub ပေါ်တွင် နှစ်ခုစလုံး ထုတ်ဝေကာ core package ကိုသာ WordPress.org သို့ ပို့ပါ

### ပြင်ဆင်ပြီး {#fixed}

- **AI တောင်းဆိုမှု ယုံကြည်စိတ်ချရမှု** — model ရွေးချယ်မှု၊ တောင်းဆိုမှု timeout များ၊ မူလ setup model၊ reasoning text ကိုင်တွယ်မှုနှင့် မမှန်ကန်သော tool call များအတွက် ထပ်မံကြိုးစားရန် လမ်းညွှန်မှုကို တိုးတက်စေသည်
- **ပြက္ခဒိန်နှင့် သတိပေးချက် ခိုင်မာစေခြင်း** — Google Calendar token များနှင့် သတိပေးချက် ထပ်တူဖယ်ရှားမှုကို ခိုင်မာစေသည်
- **စတင်အသုံးပြုခြင်းနှင့် အတည်ပြုမှု ပြန်လည်စတင်ခြင်းများ** — frontend onboarding စတင်မှုများနှင့် အတည်ပြုထားသော ability ပြန်လည်စတင်မှုများကို ပြင်ဆင်သည်
- **WordPress.org ပက်ကေ့ဂျ်ပြန်လည်သုံးသပ်မှု ပြဿနာများ** — core ထုတ်ဝေမှုအတွက် ပက်ကေ့ဂျ်ပြန်လည်သုံးသပ်မှု တုံ့ပြန်ချက်များကို ဖြေရှင်းသည်

## 1.16.0 — 2026-05-20 တွင် ထုတ်ဝေခဲ့သည် {#1160--released-on-2026-05-20}

### အသစ် {#new-1}

- **Logo SVG ထုတ်လုပ်နိုင်စွမ်း** — Theme Builder သည် namespace-safe sanitisation ဖြင့် စိတ်ကြိုက် logo SVG များကို ယခု ထုတ်လုပ်ပြီး ထည့်သွင်းနိုင်ပြီ
- **ရှာဖွေမေးမြန်းမှုတွင် ဓာတ်ပုံ အပ်လုဒ်** — Theme Builder ရှာဖွေမေးမြန်းမှုတွင် ဒီဇိုင်းအကြောင်းအရာ ပိုမိုကြွယ်ဝစေရန် ဓာတ်ပုံအပ်လုဒ် အဆင့်ကို ယခု ထည့်သွင်းထားသည်
- **Palette Contrast စစ်ဆေးနိုင်စွမ်း** — theme တစ်ခုတွင် အသုံးမပြုမီ အရောင်တွဲများကို WCAG လိုက်နာမှုအတွက် စစ်ဆေးပါ
- **ဧည့်ဝတ်ပြုလုပ်ငန်း မီနူးများ** — Theme Builder သည် ဧည့်ဝတ်ပြုလုပ်ငန်းများအတွက် ဖွဲ့စည်းထားသော အစားအစာနှင့် အဖျော်ယမကာ မီနူးစာမျက်နှာများကို ယခု ထုတ်လုပ်နိုင်ပြီ
- **Desktop နှင့် မိုဘိုင်း ကြိုတင်ကြည့်ရှုမှု rendering** — ဒီဇိုင်းဦးတည်ချက် ရွေးချယ်နေစဉ် desktop နှင့် မိုဘိုင်းစက်များပေါ်တွင် သင့်ဒီဇိုင်းကို ကြိုတင်ကြည့်ပါ
- **Navigation label parameter** — Create Menu ability သည် စာမျက်နှာခေါင်းစဉ်မှ သီးခြားဖြစ်သော `navigation_label` ကို ယခု ပံ့ပိုးထားသည်
- **Tier 1 tool ရရှိနိုင်မှု** — sd-ai-agent/site-scrape သည် Theme Builder တွင် မူလအတိုင်း ရရှိနိုင်သော Tier 1 tool ဖြစ်လာသည်

### ပြင်ဆင်ပြီး {#fixed-1}

- **AI Client cache** — request များအကြား ဆက်လက်တည်ရှိစေရန် transients ဖြင့် ယခု ပံ့ပိုးထားပြီး ကြာမြင့်သော agent task များတွင် ဒေတာဆုံးရှုံးမှုကို ကာကွယ်သည်
- **Plugin row action link များ** — ရှင်းလင်းမှုအတွက် ပြင်ဆင်ပြီး အမည်ပြောင်းထားသည်

## 1.10.0 — 2026-05-05 တွင် ထုတ်ဝေခဲ့သည် {#1100--released-on-2026-05-05}

### အသစ် {#new-2}

- **Tavily အင်တာနက်ရှာဖွေမှု** — Brave Search နှင့်အတူ ပိုမိုကြွယ်ဝသော အင်တာနက်ရှာဖွေမှု ရလဒ်များအတွက် Tavily ကို search provider အဖြစ် ထည့်ပါ
- **Theme-aware built-in skills** — Block Themes၊ Classic Themes၊ Kadence Blocks နှင့် Kadence Theme skill လမ်းညွှန်များကို ယခု plugin နှင့်အတူ ထည့်သွင်းပေးထားသည်
- **ဆိုက်တည်ဆောက်သူ ဆက်သွယ်ရန်ဖောင် ability** — chat interface မှ တိုက်ရိုက် မည်သည့်စာမျက်နှာတွင်မဆို ဆက်သွယ်ရန်ဖောင် ထည့်ပါ

### တိုးတက်မှုများ {#improved-1}

- **WooCommerce ပေါင်းစည်းမှု ပြန်လည်ဖွဲ့စည်းခြင်း** — ယုံကြည်စိတ်ချရမှုနှင့် ကိုက်ညီမှု ပိုကောင်းစေရန် ယခု native WooCommerce API များကို အသုံးပြုသည်
- **Provider စာရင်း အလိုအလျောက် ပြန်လည်အသစ်လုပ်ခြင်း** — မည်သည့် plugin ကိုမဆို activate သို့မဟုတ် deactivate လုပ်သည့်အခါ

### ပြင်ဆင်ပြီး {#fixed-2}

- **navigate-to ability** — admin စာမျက်နှာအချို့တွင် အဆုံးမရှိ reload loop ဖြစ်ခြင်းကို ပြင်ဆင်ခဲ့သည်
- **list-posts ability** — အမျိုးအစားနှင့် tag အမည်များကို slug များအဖြစ် ယခု မှန်ကန်စွာ ဖြေရှင်းသည်
- **WP-CLI command များ** — ယခင် refactor ပြီးနောက် ပျောက်ဆုံးနေသော namespace alias များကို ပြန်လည်ထည့်သွင်းခဲ့သည်
- **Event automation** — automation table များ မဖန်တီးရသေးသော ဆိုက်များကို အဆင်ပြေစွာ ကိုင်တွယ်သည်
- **memory-save ability** — system instruction builder တွင် မှန်ကန်သော namespace prefix ကို ယခု အသုံးပြုသည်
- **Scalar tool result များ** — AI သို့ ပြန်မပို့မီ ယခု မှန်ကန်စွာ wrap လုပ်ထားသည်
- **အသုံးပြုမှု စာရင်းအင်းများ** — ဗားရှင်းဟောင်းများမှ upgrade လုပ်ရာတွင် legacy ability key format ကို ယခု မှန်ကန်စွာ ကိုင်တွယ်သည်
