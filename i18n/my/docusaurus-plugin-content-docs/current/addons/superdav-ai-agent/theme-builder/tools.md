# ကိရိယာများနှင့် စနစ်ဖွဲ့စည်းမှု (Tools and Configuration)

Theme Builder သည် သင့် WordPress ဝက်ဘ်ဆိုဒ်ကို ခွဲခြမ်းစိတ်ဖြာခြင်း၊ ဒီဇိုင်းရေးဆွဲခြင်းနှင့် တည်ဆောက်ခြင်းတို့အတွက် ကိရိယာအစုံအလင်ကို အသုံးပြုထားပါသည်။ Version 1.16.0 တွင် **sd-ai-agent/site-scrape သည် Tier 1 ကိရိယာအဖြစ် အဆင့်မြှင့်တင်ခဲ့ပြီး** Theme Builder တွင် မူလအစကတည်းက အသုံးပြုနိုင်ပြီ ဖြစ်ပါသည်။

## ရရှိနိုင်သော ကိရိယာများ (Available Tools)

### Tier 1 ကိရိယာများ (အမြဲရရှိနိုင်သည်)

Tier 1 ကိရိယာများသည် အခြားဖွဲ့စည်းမှုများ ထပ်မံလုပ်ရန်မလိုဘဲ Theme Builder တွင် မူလအစကတည်းက အသုံးပြုနိုင်ပါသည်။

#### sd-ai-agent/site-scrape

**ရည်ရွယ်ချက်:** ရှိပြီးသား ဝက်ဘ်ဆိုဒ်များမှ ဒီဇိုင်းအလှအပ၊ အကြောင်းအရာဖွဲ့စည်းပုံနှင့် လုပ်ဆောင်နိုင်စွမ်းများကို ခွဲခြမ်းစိတ်ဖြာရန်။

**စွမ်းဆောင်ရည်များ:**

- **ဝက်ဘ်ဆိုဒ် ခွဲခြမ်းစိတ်ဖြာခြင်း (Website analysis)** — ပြိုင်ဘက် သို့မဟုတ် စိတ်ကူးရယူလိုသည့် ဝက်ဘ်ဆိုဒ်များကို စုဆောင်းပြီး ခွဲခြမ်းစိတ်ဖြာခြင်း
- **ဒီဇိုင်းထုတ်ယူခြင်း (Design extraction)** — အရောင်များ၊ စာလုံးပုံစံများနှင့် အပြင်အဆင်ပုံစံများကို ဖော်ထုတ်ခြင်း
- **အကြောင်းအရာဖွဲ့စည်းပုံ (Content structure)** — စာမျက်နှာဖွဲ့စည်းပုံနှင့် အဆင့်ဆင့်အစီအစဉ်ကို နားလည်ခြင်း
- **အင်္ဂါရပ်ရှာဖွေခြင်း (Feature detection)** — plugin များနှင့် လုပ်ဆောင်နိုင်စွမ်းများကို ဖော်ထုတ်ခြင်း
- **စွမ်းဆောင်ရည်ခွဲခြမ်းစိတ်ဖြာခြင်း (Performance analysis)** — စာမျက်နှာမြန်နှုန်းနှင့် အကောင်းဆုံးဖြစ်အောင် လုပ်ဆောင်မှုများကို စစ်ဆေးခြင်း
- **SEO ခွဲခြမ်းစိတ်ဖြာခြင်း (SEO analysis)** — meta tags များနှင့် structured data များကို ပြန်လည်သုံးသပ်ခြင်း

**အသုံးပြုပုံ:**

```
example.com ရဲ့ ဒီဇိုင်းကို ခွဲခြမ်းစိတ်ဖြာပြီး ကျွန်တော့် ဝက်ဘ်ဆိုဒ်အတွက် စိတ်ကူးရယူဖို့ အသုံးပြုပါ။
```

**ဘာတွေ ထုတ်ယူနိုင်သလဲ:**

- အရောင်အသွေး စုံလင်မှုနှင့် အရောင်အသုံးပြုမှု
- စာလုံးပုံစံ (fonts နှင့် အရွယ်အစားများ)
- အပြင်အဆင်ဖွဲ့စည်းပုံနှင့် နေရာလွတ်များ (Layout structure and spacing)
- လမ်းညွှန်မှု ပုံစံများ (Navigation patterns)
- အကြောင်းအရာ စီစဉ်ဖွဲ့စည်းမှု
- ပုံများနှင့် မီဒီယာ အသုံးပြုမှု
- အပြန်အလှန်တုံ့ပြန်သည့် အစိတ်အပိုင်းများ (Interactive elements)
- မိုဘိုင်းလ်တွင် အသုံးပြုနိုင်မှု (Mobile responsiveness)

**ကန့်သတ်ချက်များ:**

- စကားဝှက်နဲ့ကာထားတဲ့ ဝက်ဘ်ဆိုဒ်တွေကို ခွဲခြမ်းစိတ်ဖြာလို့ မရပါ
- robots.txt နှင့် ဝက်ဘ်ဆိုဒ် မူဝါဒများကို လေးစားပါတယ်
- အမြဲတမ်းပြောင်းလဲနေတဲ့ အကြောင်းအရာ (dynamic content) တွေကို မရနိုင်ပါ
- JavaScript အများအပြားပါတဲ့ ဝက်ဘ်ဆိုဒ်တွေမှာ ခွဲခြမ်းစိတ်ဖြာမှုက ကန့်သတ်ခံရနိုင်ပါတယ်
- ကြီးမားတဲ့ ဝက်ဘ်ဆိုဒ်တွေက ခွဲခြမ်းစိတ်ဖြာဖို့ အချိန်ပိုယူနိုင်ပါတယ်

### Tier 2 ကိရိယာများ (ရွေးချယ်နိုင်သည်)

Tier 2 ကိရိယာများသည် Theme Builder ၏ Settings တွင် ရှင်းရှင်းလင်းလင်း ဖွင့်ပေးမှသာ အသုံးပြုနိုင်ပါသည်။

#### Advanced Analytics

အသေးစိတ် စွမ်းဆောင်ရည် အတိုင်းအတာများကို ပေးစွမ်းသည်-

- စာမျက်နှာ တင်မြှောက်ချိန်များ (Page load times)
- Core Web Vitals
- SEO ရမှတ် (SEO score)
- အသုံးပြုရလွယ်ကူမှု ရမှတ် (Accessibility score)
- အကောင်းဆုံး အလေ့အကျင့် ရမှတ် (Best practices score)

#### Content Optimizer

အောက်ပါအရာများအတွက် ခွဲခြမ်းစိတ်ဖြာပြီး တိုးတက်မှုများ အကြံပြုသည်-

- ဖတ်ရလွယ်ကူမှု (Readability)
- SEO အကောင်းဆုံးဖြစ်အောင် လုပ်ဆောင်မှု
- Keyword အသုံးပြုမှု
- အကြောင်းအရာဖွဲ့စည်းပုံ
- လုပ်ဆောင်ရန် တိုက်တွန်းချက်များ (Call-to-action placement)

### Tier 3 ကိရိယာများ (Premium)

Tier 3 ကိရိယာများသည် အပိုဆောင်း ဖွဲ့စည်းမှု သို့မဟုတ် Premium အင်္ဂါရပ်များ လိုအပ်ပါသည်။

#### AI Content Generator

အောက်ပါအရာများအတွက် အကြောင်းအရာများ ဖန်တီးပေးသည်-

- ကုန်ပစ္စည်း ဖော်ပြချက်များ (Product descriptions)
- ဝန်ဆောင်မှု စာမျက်နှာများ
- ဘလော့ဂ် ပို့စ်များ
- Meta descriptions
- လုပ်ဆောင်ရန် တိုက်တွန်းချက် စာသားများ (Call-to-action copy)

#### Advanced Design Tools

အဆင့်မြင့် ဒီဇိုင်းစွမ်းရည်များကို ပေးစွမ်းသည်-

- Custom CSS ထုတ်ပေးခြင်း
- Animation ဖန်တီးခြင်း
- အပြန်အလှန်တုံ့ပြန်သည့် အစိတ်အပိုင်း ဒီဇိုင်း
- အဆင့်မြင့် အရောင်သီအိုရီ (Advanced color theory)
- စာလုံးပုံစံ အကောင်းဆုံးဖြစ်အောင် လုပ်ဆောင်ခြင်း (Typography optimization)

## ကိရိယာ ဖွဲ့စည်းမှု (Tool Configuration)

### ကိရိယာများ ဖွင့်ခြင်း (Enabling Tools)

Theme Builder တွင် အပိုဆောင်း ကိရိယာများ ဖွင့်ရန်-

1. **Gratis AI Agent → Settings** သို့ သွားပါ
2. **Theme Builder → Tools** သို့ သွားပါ
3. လိုအပ်သလို ကိရိယာများကို ဖွင့်/ပိတ် လုပ်ပါ
4. Settings များကို သိမ်းဆည်းပါ

### ကိရိယာ ခွင့်ပြုချက်များ (Tool Permissions)

အချို့ကိရိယာများသည် ခွင့်ပြုချက်များ လိုအပ်သည်-

- **Site scraping** — အင်တာနက် ဝင်ရောက်ခွင့် လိုအပ်သည်
- **Analytics** — Google Analytics ပေါင်းစပ်မှု လိုအပ်သည်
- **Content generation** — API keys များ လိုအပ်သည်
- **Advanced features** — Premium subscription လိုအပ်နိုင်သည်

### API Keys နှင့် Credentials များ

လိုအပ်သော ကိရိယာများအတွက် API keys များကို ဖွဲ့စည်းပါ-

1. **Gratis AI Agent → Settings → API Keys** သို့ သွားပါ
2. ကိရိယာတစ်ခုချင်းစီအတွက် credentials များကို ထည့်သွင်းပါ
3. ချိတ်ဆက်မှု စမ်းသပ်ပါ
4. လုံခြုံစွာ သိမ်းဆည်းပါ

## sd-ai-agent/site-scrape ကို အသုံးပြုခြင်း

### အခြေခံ အသုံးပြုပုံ

Theme Builder ကို ဝက်ဘ်ဆိုဒ်တစ်ခု ခွဲခြမ်းစိတ်ဖြာခိုင်းပါ-

```
Analyze the design of my competitor's site at competitor.com
and suggest design improvements for my site.
```

### သီးသန့် ခွဲခြမ်းစိတ်ဖြာမှု

သီးသန့် ခွဲခြမ်းစိတ်ဖြာမှု အမျိုးအစားများကို တောင်းဆိုပါ-

```
Extract the color palette from example.com and use it as inspiration.
```

```
Analyze the navigation structure of example.com and apply similar
organization to my site.
```

```
Check the mobile responsiveness of example.com and ensure my site
is equally responsive.
```

### နှိုင်းယှဉ် ခွဲခြမ်းစိတ်ဖြာမှု

ဝက်ဘ်ဆိုဒ်များစွာကို နှိုင်းယှဉ်ပါ-

```
Compare the designs of site1.com and site2.com and create a design
that combines the best elements of both.
```

## ကိရိယာ ကန့်သတ်ချက်များနှင့် ထည့်သွင်းစဉ်းစားရမည့်အချက်များ (Tool Limitations and Considerations)

### Rate Limiting (နှုန်းကန့်သတ်မှု)

- ဆာဗာဝန်ပိမှု မဖြစ်စေရန် scraping ကို နှုန်းကန့်သတ်ထားသည်
- ဝက်ဘ်ဆိုဒ်တစ်ခုကို အကြိမ်ကြိမ် တောင်းဆိုမှုများ ပြုလုပ်ပါက နှေးကွေးစေနိုင်သည်
- ကြီးမားတဲ့ ဝက်ဘ်ဆိုဒ်တွေက ခွဲခြမ်းစိတ်ဖြာဖို့ အချိန်ပိုယူနိုင်ပါတယ်

### အကြောင်းအရာ တိကျမှု (Content Accuracy)

- အမြဲတမ်းပြောင်းလဲနေတဲ့ အကြောင်းအရာတွေကို အပြည့်အဝ မရနိုင်ပါ
- JavaScript နဲ့ ဖန်တီးထားတဲ့ အကြောင်းအရာတွေက မပြည့်စုံနိုင်ပါ
- အချိန်နဲ့တပြေးညီ အချက်အလက်တွေ မပေါ်နိုင်ပါ
- အကြောင်းအရာအချို့က ငွေပေးရတဲ့ နံရံတွေနောက်ကွယ်မှာ ရှိနေနိုင်ပါတယ်

### ဥပဒေနှင့် ကျင့်ဝတ်ဆိုင်ရာ ထည့်သွင်းစဉ်းစားမှုများ (Legal and Ethical Considerations)

- robots.txt နှင့် ဝက်ဘ်ဆိုဒ် မူဝါဒများကို လေးစားပါ
- မူပိုင်ခွင့်ရှိ အကြောင်းအရာများကို ပြန်လည်ထုတ်ဝေရန် scraping မလုပ်ပါနှင့်
- ခွဲခြမ်းစိတ်ဖြာမှုကို စိတ်ကူးရယူရန်သာ အသုံးပြုပါ၊ ကူးချရန် မဟုတ်ပါ
- ထုတ်ယူထားတဲ့ အကြောင်းအရာတွေကို အသုံးပြုခွင့်ရှိကြောင်း အတည်ပြုပါ
- ခွဲခြမ်းစိတ်ဖြာထားတဲ့ ဝက်ဘ်ဆိုဒ်တွေရဲ့ ဝန်ဆောင်မှုစည်းကမ်းချက်တွေကို လိုက်နာပါ

### စွမ်းဆောင်ရည်အပေါ် သက်ရောက်မှု (Performance Impact)

- ကြီးမားတဲ့ ဝက်ဘ်ဆိုဒ်တွေကို scraping လုပ်ရင် အချိန်ယူရပါတယ်
- တစ်ချိန်တည်းမှာ scraping အများအပြား လုပ်ရင် ပိုနှေးပါတယ်
- ကွန်ရက်ချိတ်ဆက်မှုက အမြန်နှုန်းကို သက်ရောက်မှုရှိပါတယ်
- အချက်အလက် အစုအဝေးကြီးတွေက ပိုပြီး လုပ်ဆောင်မှုများနိုင်ပါတယ်

## အကောင်းဆုံး အလေ့အကျင့်များ (Best Practices)

### ဝက်ဘ်ဆိုဒ် ခွဲခြမ်းစိတ်ဖြာခြင်းကို အသုံးပြုခြင်း

1. **ဝက်ဘ်ဆိုဒ်များစွာ ခွဲခြမ်းစိတ်ဖြာပါ** — အရင်းအမြစ်များစွာမှ စိတ်ကူးရယူပါ။
2. **ဖွဲ့စည်းပုံကို အာရုံစိုက်ပါ** — အပြင်အဆင်နှင့် စီစဉ်ဖွဲ့စည်းပုံ ပုံစံများကို လေ့လာပါ။
3. **အရောင်များကို ထုတ်ယူပါ** — အရောင်အသွေး စုံလင်မှုများကို အစမှတ်အဖြစ် အသုံးပြုပါ။
4. **စာလုံးပုံစံကို လေ့လာပါ** — သင်ကြိုက်နှစ်သက်တဲ့ စာလုံးတွဲများကို ဖော်ထုတ်ပါ။
5. **လမ်းညွှန်မှုကို ပြန်လည်သုံးသပ်ပါ** — မီနူး စီစဉ်ဖွဲ့စည်းပုံကို နားလည်ပါ။

### ကျင့်ဝတ်သိက္ခာရှိစွာ Scraping လုပ်ခြင်း

1. **robots.txt ကို လေးစားပါ** — ဝက်ဘ်ဆိုဒ် လမ်းညွှန်ချက်များကို လိုက်နာပါ။
2. **အကြောင်းအရာကို ကူးမချပါနှင့်** — ခွဲခြမ်းစိတ်ဖြာမှုကို စိတ်ကူးရယူရန်သာ အသုံးပြုပါ။
3. **အခွင့်အရေးကို အတည်ပြုပါ** — ထုတ်ယူထားတဲ့ အကြောင်းအရာတွေကို အသုံးပြုခွင့်ရှိကြောင်း သေချာပါစေ။
4. **ရင်းမြစ်များကို ဖော်ပြပါ** — စိတ်ကူးရယူမှု ရင်းမြစ်များကို ဂုဏ်ပြုပါ။
5. **စည်းကမ်းချက်များကို လိုက်နာပါ** — ဝက်ဘ်ဆိုဒ် ဝန်ဆောင်မှုစည်းကမ်းချက်များကို လိုက်နာပါ။

### ကိရိယာ စွမ်းဆောင်ရည်ကို အမြင့်ဆုံးမြှင့်တင်ခြင်း

1. **တိကျစွာ မေးမြန်းပါ** — ခွဲခြမ်းစိတ်ဖြာမှု အမျိုးအစားများကို သီးသန့်တောင်းဆိုပါ။
2. **အကြောင်းအရာ ပေးပါ** — သင့်ဝက်ဘ်ဆိုဒ်ရဲ့ ရည်ရွယ်ချက်ကို ရှင်းပြပါ။
3. **မျှော်လင့်ချက်များ သတ်မှတ်ပါ** — သင့်ဒီဇိုင်း ရည်ရွယ်ချက်တွေကို ဖော်ပြပါ။
4. **ထပ်ခါထပ်ခါ ပြုပြင်ပါ** — ရလဒ်များအပေါ် အခြေခံပြီး ပြုပြင်ပါ (Refine)။
5. **ကိရိယာများကို ပေါင်းစပ်အသုံးပြုပါ** — အပြည့်အဝ ခွဲခြမ်းစိတ်ဖြာမှုအတွက် ကိရိယာများစွာကို အသုံးပြုပါ။

## ပြဿနာဖြေရှင်းခြင်း (Troubleshooting)

### ဝက်ဘ်ဆိုဒ်ကို Scrap မလုပ်နိုင်ခြင်း

- ဝက်ဘ်ဆိုဒ်က အများပြည်သူအတွက် ဝင်ရောက်နိုင်ခြင်း ရှိ၊ မရှိ စစ်ဆေးပါ
- robots.txt က scraping လုပ်ခွင့်ပေးထားခြင်း ရှိ၊ မရှိ အတည်ပြုပါ
- ဝက်ဘ်ဆိုဒ် အခြားတစ်ခုကို စမ်းကြည့်ပါ
- အင်တာနက် ချိတ်ဆက်မှုကို စစ်ဆေးပါ
- ပြဿနာဆက်ရှိနေပါက Support ကို ဆက်သွယ်ပါ

### ခွဲခြမ်းစိတ်ဖြာမှု မပြည့်စုံခြင်း

- ဝက်ဘ်ဆိုဒ်မှာ အမြဲတမ်းပြောင်းလဲနေတဲ့ အကြောင်းအရာတွေ ရှိနိုင်ပါတယ်
- JavaScript က အပြည့်အဝ မဖန်တီးနိုင်တာမျိုး ရှိနိုင်ပါတယ်
- ကြီးမားတဲ့ ဝက်ဘ်ဆိုဒ်တွေက အချိန်ကုန်နိုင်ပါတယ်
- အစား၊ သီးသန့် စာမျက်နှာတွေကို ခွဲခြမ်းစိတ်ဖြာဖို့ ကြိုးစားပါ
- သီးသန့် ခွဲခြမ်းစိတ်ဖြာမှု အမျိုးအစားများကို တောင်းဆိုပါ

### စွမ်းဆောင်ရည် နှေးကွေးခြင်း

- ကြီးမားတဲ့ ဝက်ဘ်ဆိုဒ်တွေက ခွဲခြမ်းစိတ်ဖြာဖို့ အချိန်ပိုယူပါတယ်
- တစ်ချိန်တည်းမှာ scraping အများအပြား လုပ်တာက ပိုနှေးပါတယ်
- ကွန်ရက်ချိတ်ဆက်မှုက အမြန်နှုန်းကို သက်ရောက်မှုရှိပါတယ်
- အချိန်အလယ်ပိုင်း အချိန်တွေမှာ ကြိုးစားကြည့်ပါ
- အပိုင်းငယ်လေးတွေကို အရင်ခွဲခြမ်းစိတ်ဖြာကြည့်ပါ
