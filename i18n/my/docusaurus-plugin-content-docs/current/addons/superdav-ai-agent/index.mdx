# Superdav AI Agent

Superdav AI Agent သည် Gratis AI Agent ကို theme သိရှိသော ကျွမ်းကျင်မှုများ၊ ပိုမိုကောင်းမွန်သော အင်တာနက်ရှာဖွေရေး စွမ်းရည်များ၊ ဝဘ်ဆိုက် တည်ဆောက်ရေး လုပ်ဆောင်ချက်များနှင့် ရွေးချယ်နိုင်သော အလိုအလျောက်လုပ်ဆောင်မှု ပေါင်းစည်းမှုများဖြင့် တိုးချဲ့ထားသော အဆင့်မြင့် WordPress AI အကူအညီဖြစ်သည်။ ဗားရှင်း 1.18.0 တွင် Google Calendar ဖတ်ရှုရေးကိရိယာများ၊ ဆက်သွယ်ရန်နှင့် တက်ရောက်သူ ချိတ်ဆက်သတ်မှတ်မှု၊ လူသားအတည်ပြုမှု ဂိတ်များ၊ သတိပေးချက် ထပ်နေမှုဖယ်ရှားခြင်း၊ TextBee SMS အကြောင်းကြားချက်များ၊ စီမံထားသော Superdav ဝန်ဆောင်မှု provision ပြုလုပ်ခြင်းနှင့် ယုံကြည်စိတ်ချရသော developer workflow များအတွက် သီးခြားဖြန့်ချိထားသော Advanced တွဲဖက် package တို့ ထည့်သွင်းထားသည်။

## အဓိက လုပ်ဆောင်ချက်များ

- **Theme သိရှိသော ကျွမ်းကျင်မှုများ** — သင့် active WordPress theme နှင့် ထည့်သွင်းထားသော plugin များအပေါ် မူတည်၍ လမ်းညွှန်မှုနှင့် စွမ်းရည်များကို အလိုအလျောက် လိုက်လျောညီထွေ ပြောင်းလဲပေးသည်
- **အင်တာနက်ရှာဖွေရေး provider များစွာ** — ပိုမိုပြည့်စုံသော ရှာဖွေမှုရလဒ်များနှင့် အဆင့်မြင့် သုတေသနစွမ်းရည်များအတွက် Brave Search နှင့် Tavily အကြား ရွေးချယ်နိုင်သည်
- **ဝဘ်ဆိုက်တည်ဆောက်ရေး ဆက်သွယ်ရန် form များ** — သင့်တွင် ထည့်သွင်းထားသော form plugin ကို အသုံးပြု၍ chat interface မှတစ်ဆင့် အပြည့်အဝ အလုပ်လုပ်သော ဆက်သွယ်ရန် form များကို တိုက်ရိုက် ဖန်တီးနိုင်သည်
- **ပိုမိုကောင်းမွန်သော ဝဘ်ဆိုက်တည်ဆောက်ရေး** — plugin ရှာဖွေတွေ့ရှိမှုနှင့် error ပြန်လည်ကောင်းမွန်ရေး ပိုကောင်းသော အဆင့်များစွာပါ ဝဘ်ဆိုက်တည်ဆောက်မှု plan များကို မြှင့်တင်ထားသည်
- **ပါဝင်ပြီးသား theme ကျွမ်းကျင်မှုများ** — Block Themes၊ Classic Themes၊ Kadence Blocks နှင့် Kadence Theme တို့အတွက် အထူးပြု လမ်းညွှန်မှုများ
- **ပိုမိုကောင်းမွန်သော WooCommerce ပေါင်းစည်းမှု** — ယုံကြည်စိတ်ချရမှုနှင့် သဟဇာတဖြစ်မှု ပိုကောင်းစေရန် native WooCommerce API များကို အသုံးပြုရန် ပြန်လည်ဖွဲ့စည်းထားသည်
- **Provider အလိုအလျောက် ရှာဖွေသိရှိခြင်း** — plugin များနှင့် theme များကို activate သို့မဟုတ် deactivate လုပ်သည့်အခါ ရနိုင်သော plugin များနှင့် theme များကို အလိုအလျောက် ရှာဖွေပြီး refresh လုပ်သည်
- **စိတ်ကြိုက် logo ဖန်တီးခြင်း** — namespace လုံခြုံသော sanitisation အလိုအလျောက်ပြုလုပ်မှုဖြင့် စိတ်ကြိုက် SVG logo များကို ဖန်တီးပြီး ထည့်သွင်းနိုင်သည်
- **Palette contrast စစ်ဆေးအတည်ပြုခြင်း** — သင့် theme တွင် မသုံးမီ WCAG accessibility လိုက်နာမှုအတွက် အရောင်အတွဲများကို စစ်ဆေးသည်
- **ဓာတ်ပုံဖြင့် မြှင့်တင်ထားသော ရှာဖွေဖော်ထုတ်မှု** — ဒီဇိုင်းဦးတည်ချက်ကို သက်ရောက်စေရန် Theme Builder ရှာဖွေဖော်ထုတ်ရေး မေးမြန်းမှုအတွင်း ဓာတ်ပုံများ upload လုပ်နိုင်သည်
- **ဧည့်ဝတ်ပြုလုပ်ငန်း menu များ** — စားသောက်ဆိုင်များ၊ ကဖေးများနှင့် ဧည့်ဝတ်ပြုလုပ်ငန်းများအတွက် ဖွဲ့စည်းထားသော အစားအသောက်နှင့် အဖျော်ယမကာ menu စာမျက်နှာများကို ဖန်တီးသည်
- **Desktop နှင့် mobile preview များ** — ဒီဇိုင်းဦးတည်ချက် ရွေးချယ်နေစဉ် desktop နှင့် mobile device များပေါ်တွင် သင့်ဒီဇိုင်းကို preview ကြည့်နိုင်သည်
- **Tier 1 ဝဘ်ဆိုက် scraping** — ပြိုင်ဘက်နှင့် အားကျစရာ ဝဘ်ဆိုက်များကို ဆန်းစစ်ရန် sd-ai-agent/site-scrape ကို ယခု မူလအတိုင်း ရနိုင်သည်
- **Google Calendar ဖတ်ရှုရေးကိရိယာများ** — အလိုအလျောက်လုပ်ဆောင်မှုများက လာမည့် အချိန်ဇယား context ကို စဉ်းစားနိုင်ရန် configure လုပ်ထားသော calendar များနှင့် event များကို ဖတ်နိုင်သည်
- **ဆက်သွယ်ရန်နှင့် တက်ရောက်သူ ချိတ်ဆက်သတ်မှတ်မှု** — follow-up အလုပ်များ မဖန်တီးမီ calendar ပါဝင်သူများကို သိရှိထားသော ဝဘ်ဆိုက်အသုံးပြုသူများနှင့် contact များနှင့် ကိုက်ညီအောင် ချိတ်ဆက်သည်
- **လူသားအတည်ပြုမှု ဂိတ်များ** — message များပို့ခြင်း၊ အလုပ်ပြန်စခြင်း သို့မဟုတ် ပြောင်းလဲမှုများ မပြုလုပ်မီ ထိခိုက်လွယ်သော အလိုအလျောက်လုပ်ဆောင်မှုများကို review အတွက် ခဏရပ်စေသည်
- **သတိပေးချက် record များနှင့် ထပ်နေမှုဖယ်ရှားခြင်း** — ထပ်မံလည်ပတ်သော အလိုအလျောက်လုပ်ဆောင်မှုများက လက်ခံသူတစ်ဦးတည်းကို နှစ်ကြိမ် မအကြောင်းကြားရန် ပို့ပြီးသား သတိပေးချက်များကို ခြေရာခံသည်
- **TextBee SMS အကြောင်းကြားချက်များ** — SMS provider ကို enable လုပ်ထားသည့်အခါ TextBee မှတစ်ဆင့် configure လုပ်ထားသော text-message အကြောင်းကြားချက်များကို ပို့သည်
- **စီမံထားသော Superdav ဝန်ဆောင်မှု provision ပြုလုပ်ခြင်း** — endpoint ကို လက်ဖြင့် configure လုပ်ရမည့်အစား အလိုအလျောက် setup ဖြင့် supported ဝဘ်ဆိုက်များကို hosted Superdav endpoint များနှင့် ချိတ်ဆက်သည်
- **Advanced တွဲဖက် package** — filesystem၊ database၊ WP-CLI၊ REST dispatcher၊ plugin builder၊ git snapshot၊ အသုံးပြုသူစီမံခန့်ခွဲမှုနှင့် benchmark workflow များအတွက် သီးခြားဖြန့်ချိထားသော ယုံကြည်စိတ်ချရသော developer ကိရိယာများ

## လိုအပ်ချက်များ

- WordPress 6.0 သို့မဟုတ် အထက်
- PHP 8.1 သို့မဟုတ် အထက်
- WordPress Multisite (network-activated) သို့မဟုတ် single-site installation
- Active ဖြစ်နေသော AI provider connector plugin (Anthropic၊ OpenAI သို့မဟုတ် compatible)
- Gratis AI Agent v1.9.0 သို့မဟုတ် အထက် (Superdav AI Agent သည် Gratis AI Agent ကို တိုးချဲ့ထားသည်)

## ထည့်သွင်းခြင်း

1. plugin file များကို `/wp-content/plugins/superdav-ai-agent` သို့ upload လုပ်ပါ
2. WordPress ရှိ **Plugins** screen မှတစ်ဆင့် plugin ကို activate လုပ်ပါ
3. Gratis AI Agent ကိုလည်း activate လုပ်ထားကြောင်း သေချာစေပါ
4. **Gratis AI Agent → Settings** သို့ သွားပြီး လိုအပ်သလို သင့် အင်တာနက်ရှာဖွေရေး provider၊ စီမံထားသော Superdav ဝန်ဆောင်မှု connection၊ Google Calendar သို့မဟုတ် TextBee credential များကို configure လုပ်ပါ

WordPress.org package တွင် အဓိက Superdav AI Agent လုပ်ဆောင်ချက်များ ပါဝင်သည်။ Advanced တွဲဖက် package ကို ယုံကြည်စိတ်ချရသော developer ကိရိယာများကို ရှင်းလင်းစွာ ထည့်သွင်းပြီး ခွင့်ပြုထားသော environment များအတွက် သီးခြား build လုပ်ပြီး ဖြန့်ချိသည်။

## အမြန်စတင်ရန်

Activate လုပ်ပြီးသည်နှင့် Superdav AI Agent သည် သင့် Gratis AI Agent installation ကို theme သိရှိသော ကျွမ်းကျင်မှုများနှင့် ထပ်ဆောင်းစွမ်းရည်များဖြင့် အလိုအလျောက် မြှင့်တင်ပေးသည်။ WordPress admin sidebar ရှိ **Gratis AI Agent** panel ကို ဖွင့်ပြီး လုပ်ဆောင်ချက်အသစ်များကို စတင်အသုံးပြုပါ။

**ဥပမာ ညွှန်ကြားချက်များ:**

```
ဆက်သွယ်ရန် စာမျက်နှာတွင် ဆက်သွယ်ရန် form တစ်ခု ထည့်ပါ။
```

```
နောက်ဆုံးပေါ် WordPress လုံခြုံရေး အကောင်းဆုံး လက်တွေ့နည်းလမ်းများကို ဝဘ်တွင် ရှာပါ။
```

```
Active theme ၏ block pattern များကို အသုံးပြု၍ hero section တစ်ခု ဖန်တီးပါ။
```

```
မနက်ဖြန်၏ configure လုပ်ထားသော Google Calendar event များကို review လုပ်ပြီး တက်ရောက်သူများအတွက် သတိပေးချက်များ မူကြမ်းရေးပါ။
```

```
message ကို ကျွန်ုပ်အတည်ပြုပြီးနောက် SMS သတိပေးချက်တစ်ခု ပို့ပါ။
```

## Theme သိရှိသော ကျွမ်းကျင်မှုများ

Superdav AI Agent သည် သင့် active theme နှင့် ထည့်သွင်းထားသော plugin များကို အလိုအလျောက် ရှာဖွေသိရှိပြီး သင့်လျော်သော theme သိရှိသော ကျွမ်းကျင်မှုကို assistant ၏ context ထဲသို့ load လုပ်သည်။ ရနိုင်သော ကျွမ်းကျင်မှုများမှာ:

| ကျွမ်းကျင်မှု | အသုံးပြုသည့်နေရာ |
|---|---|
| **Block Themes** | WordPress block-based (Full Site Editing) architecture ကို အသုံးပြုသော theme များ |
| **Classic Themes** | PHP template များနှင့် classic editor ကို အသုံးပြုသော ရိုးရာ WordPress theme များ |
| **Kadence Blocks** | Kadence Blocks plugin active ဖြစ်နေသော ဝဘ်ဆိုက်များ |
| **Kadence Theme** | Kadence theme ကို active theme အဖြစ် အသုံးပြုထားသော ဝဘ်ဆိုက်များ |

ကျွမ်းကျင်မှုတစ်ခုချင်းစီအတွက် အသေးစိတ်လမ်းညွှန်ချက်များကို [Theme သိရှိသော ကျွမ်းကျင်မှုများ](../../user-guide/configuration/theme-aware-skills) တွင် ကြည့်ပါ။

## အင်တာနက်ရှာဖွေရေး Provider များ

Superdav AI Agent v1.10.0 သည် အင်တာနက်ရှာဖွေရေး query များအတွက် Brave Search ၏ အခြားရွေးချယ်စရာအဖြစ် **Tavily** ကို ထည့်သွင်းထားသည်။ Provider နှစ်ခုလုံးသည် real-time ရှာဖွေမှုရလဒ်များကို ပေးပြီး **Gratis AI Agent → Settings → Advanced** တွင် configure လုပ်နိုင်သည်။

| ပံ့ပိုးသူ | လုပ်ဆောင်ချက်များ | တပ်ဆင်သတ်မှတ်ခြင်း |
|---|---|---|
| **Brave Search** | မြန်ဆန်ပြီး ကိုယ်ရေးအချက်အလက်ကာကွယ်ရေးကို ဦးစားပေးသော ရှာဖွေမှုရလဒ်များ | [Brave Search API](https://brave.com/search/api/) |

## ပြက္ခဒိန်၊ အတည်ပြုမှုနှင့် အကြောင်းကြားစာ လုပ်ငန်းစဉ်များ

Superdav AI Agent v1.18.0 သည် အကြောင်းအရာနှင့် ဆိုက်တည်ဆောက်ရေး လုပ်ငန်းများထက် ကျော်လွန်၍ အလိုအလျောက်လုပ်ဆောင်မှု လုပ်ငန်းစဉ်များကို တိုးချဲ့ထားသည်။

- **Google Calendar ကိရိယာများ** သည် ပြက္ခဒိန် အထောက်အထားများ ရရှိနိုင်ပါက agent ကို သတ်မှတ်ထားသော ပြက္ခဒိန်များနှင့် အစီအစဉ်အသေးစိတ်များ ဖတ်နိုင်စေသည်။
- **ဆက်သွယ်ရန်နှင့် တက်ရောက်သူ ကူညီကိရိယာများ** သည် သတိပေးချက်များ သို့မဟုတ် နောက်ဆက်တွဲလုပ်ငန်းများ ပြင်ဆင်မီ အစီအစဉ်ပါဝင်သူများကို WordPress အသုံးပြုသူများ သို့မဟုတ် သိရှိထားသော ဆက်သွယ်ရန်များနှင့် ချိတ်ဆက်ဖော်ပြသည်။
- **လူမှ အတည်ပြုရေး ဂိတ်များ** သည် ခွင့်ပြုထားသော အသုံးပြုသူတစ်ဦးက အဆိုပြုလုပ်ဆောင်ချက်ကို ပြန်လည်စစ်ဆေးပြီးသည်အထိ အလိုအလျောက်လုပ်ဆောင်မှုကို ရပ်ထားနိုင်စေသည်။
- **သတိပေးချက် မှတ်တမ်းများ** သည် အလိုအလျောက်လုပ်ဆောင်မှုကို ထပ်မံကြိုးစားသည့်အခါ သို့မဟုတ် ထပ်ခါတလဲလဲ အချိန်ဇယားဆွဲသည့်အခါ ပြက္ခဒိန် သို့မဟုတ် အကြောင်းကြားစာ သတိပေးချက်များ မထပ်စေရန် ကာကွယ်သည်။
- **TextBee SMS ပံ့ပိုးမှု** သည် TextBee ပံ့ပိုးသူကို သတ်မှတ်ထားပါက စာတိုအကြောင်းကြားချက်များ ပေးပို့နိုင်စေသည်။

အချိန်ဇယားကို သိရှိ၍ ကူညီခြင်း၊ တက်ရောက်သူများကို နောက်ဆက်တွဲဆက်သွယ်ခြင်းနှင့် အရေးကြီးသော မက်ဆေ့ချ်များ မပို့မီ လူမှ ပြန်လည်စစ်ဆေးရန် လိုအပ်နေသေးသော သတိပေးချက် လုပ်ငန်းစဉ်များအတွက် ဤကိရိယာများကို အသုံးပြုပါ။

## ဆိုက်တည်ဆောက်ရေး ဆက်သွယ်ရန် ဖောင်များ

ဆိုက်တည်ဆောက်ရေး agent သည် ယခု chat interface မှတစ်ဆင့် ဆက်သွယ်ရန် ဖောင်များကို တိုက်ရိုက်ဖန်တီးနိုင်သည်။ ဆိုက်တည်ဆောက်ရေး agent ကို ဆက်သွယ်ရန် ဖောင်တစ်ခု ဖန်တီးရန် မေးရုံသာဖြစ်ပြီး ၎င်းသည် သင်ထည့်သွင်းထားသော form plugin (Contact Form 7, WPForms, Fluent Forms, သို့မဟုတ် Gravity Forms) ကို အလိုအလျောက် သိရှိကာ အသုံးပြုရန် အသင့်ဖြစ်သော ဖောင်ကို ဖန်တီးပေးမည်။

အသေးစိတ်လမ်းညွှန်ချက်အတွက် [ဆိုက်တည်ဆောက်ရေး ဆက်သွယ်ရန် ဖောင်များ](../../user-guide/administration/site-builder-contact-forms) ကို ကြည့်ပါ။

## Theme Builder လုပ်ဆောင်ချက်များ

Superdav AI Agent v1.16.0 ရှိ Theme Builder တွင် အစွမ်းထက်သော လုပ်နိုင်စွမ်းအသစ်များ ပါဝင်သည်။

### ဓာတ်ပုံ အပ်လုဒ်နှင့် ရှာဖွေလေ့လာရေး မေးမြန်းမှု

Theme Builder ရှာဖွေလေ့လာရေး မေးမြန်းမှုတွင် ယခု သင် မြင်သာသော အင်စပရေးရှင်း ပေးနိုင်သည့် ဓာတ်ပုံ အပ်လုဒ် အဆင့် ပါဝင်သည်။ သင့် အမှတ်တံဆိပ်၊ ပြိုင်ဘက်ဆိုက်များ သို့မဟုတ် ဒီဇိုင်း အင်စပရေးရှင်း ဓာတ်ပုံများကို အပ်လုဒ်လုပ်ပြီး Theme Builder က သင့်ဒီဇိုင်းဦးတည်ချက်ကို နားလည်စေရန် ကူညီပါ။

အသေးစိတ်လမ်းညွှန်ချက်အတွက် [ရှာဖွေလေ့လာရေး မေးမြန်းမှု](./theme-builder/discovery-interview.md) ကို ကြည့်ပါ။

### ဒက်စ်တော့နှင့် မိုဘိုင်း ကြိုတင်ကြည့်ရှုမှုများပါသော ဒီဇိုင်းဦးတည်ချက်

Theme Builder က သင့် theme အပြည့်အစုံကို မဖန်တီးမီ သင့်ဒီဇိုင်းကို ဒက်စ်တော့နှင့် မိုဘိုင်း စက်နှစ်မျိုးလုံးတွင် ပြန်လည်စစ်ဆေးပါ။ တုံ့ပြန်ညှိနှိုင်းနိုင်သော breakpoint အများအပြားတွင် အချိန်နှင့်တစ်ပြေးညီ ကြိုတင်ကြည့်ရှုမှုများကို ကြည့်ပါ။

အသေးစိတ်လမ်းညွှန်ချက်အတွက် [ဒီဇိုင်းဦးတည်ချက်](./theme-builder/design-direction.md) ကို ကြည့်ပါ။

### ဧည့်ဝတ်ပြုရေး မီနူးများ

Theme Builder သည် ယခု စားသောက်ဆိုင်များ၊ ကဖေးများ၊ ဘားများနှင့် အခြားဧည့်ဝတ်ပြုရေး လုပ်ငန်းများအတွက် ဖွဲ့စည်းထားသော အစားအစာနှင့် အဖျော်ယမကာ မီနူး စာမျက်နှာများကို ဖန်တီးနိုင်သည်။ မီနူးများတွင် ဈေးနှုန်း၊ ဖော်ပြချက်များ၊ အစားအသောက်ဆိုင်ရာ အချက်အလက်များနှင့် ဓာတ်မတည့်စေသော အရာများဆိုင်ရာ သတိပေးချက်များ ပါဝင်သည်။

အသေးစိတ်လမ်းညွှန်ချက်အတွက် [ဧည့်ဝတ်ပြုရေး မီနူးများ](./theme-builder/hospitality-menus.md) ကို ကြည့်ပါ။

### လုပ်နိုင်စွမ်းအသစ်များ

- **[Logo SVG ဖန်တီးရန်](./abilities/generate-logo-svg.md)** — အလိုအလျောက် သန့်စင်မှုပါသော စိတ်ကြိုက် SVG logo များ ဖန်တီးပါ
- **[Palette Contrast စစ်ဆေးအတည်ပြုရန်](./abilities/validate-palette-contrast.md)** — WCAG ကိုက်ညီမှုအတွက် အရောင်အတွဲများကို စစ်ဆေးပါ
- **[မီနူး ဖန်တီးရန်](./abilities/create-menu.md)** — ကွဲပြားသော လမ်းညွှန် label များပါသော လမ်းညွှန်မီနူးများ ဖန်တီးပါ

ရရှိနိုင်သော ကိရိယာများနှင့် Tier 1 ဆိုက်-scrape ကိရိယာအကြောင်း အချက်အလက်များအတွက် [ကိရိယာများနှင့် တပ်ဆင်သတ်မှတ်မှု](./theme-builder/tools.md) ကို ကြည့်ပါ။

## အဆင့်မြင့် တွဲဖက်ပက်ကေ့ချ်

Superdav AI Agent v1.18.0 သည် ယုံကြည်စိတ်ချရသော developer လုပ်ငန်းစဉ်များအတွက် သီးခြားဖြန့်ဝေသော အဆင့်မြင့် တွဲဖက်ပက်ကေ့ချ်ကို ထည့်သွင်းထားသည်။ ဤလုပ်နိုင်စွမ်းများကို WordPress.org core package ထဲတွင် တွဲထည့်ထားခြင်း မရှိပါ။ စီမံခန့်ခွဲသူများက agent ကို developer ကိရိယာများ ဝင်ရောက်အသုံးပြုရန် ထင်ရှားစွာ ယုံကြည်သော ဆိုက်များတွင်သာ တွဲဖက်ပက်ကေ့ချ်ကို ထည့်သွင်းပါ။

အဆင့်မြင့် ကိရိယာများတွင် ဖိုင်စနစ် လုပ်ဆောင်ချက်များ၊ ဒေတာဘေ့စ် ဝင်ရောက်ခွင့်၊ WP-CLI၊ REST dispatcher လုပ်ငန်းစဉ်များ၊ plugin တည်ဆောက်ရေး utility များ၊ git snapshot များ၊ အသုံးပြုသူ စီမံခန့်ခွဲရေး ကူညီကိရိယာများနှင့် benchmark ကိရိယာများ ပါဝင်သည်။ ဤကိရိယာများကို ယုံကြည်ရသော စီမံခန့်ခွဲသူများအတွက်သာ ကန့်သတ်ထားပြီး production ဆိုက်များတွင် မဖွင့်မီ approval-gate တပ်ဆင်သတ်မှတ်မှုကို ပြန်လည်စစ်ဆေးပါ။

- **သတိပေးချက် ထပ်နေမှု ဖယ်ရှားခြင်း** — ပြန်လည်ကြိုးစားမှုများ သို့မဟုတ် ထပ်ခါတလဲလဲ လုပ်ဆောင်မှုများတွင် အကြောင်းကြားချက်များ မထပ်စေရန် သတိပေးချက်များကို မှတ်တမ်းတင်ပါ
- **TextBee SMS ပံ့ပိုးသူ** — လိုအပ်သော အတည်ပြုချက်များပြီးနောက် သတ်မှတ်ထားသော စာတိုအကြောင်းကြားချက်များကို ပို့ပါ
- **Managed Superdav service တပ်ဆင်မှု** — ပံ့ပိုးထားသော ဆိုက်များအတွက် hosted service endpoint များကို အလိုအလျောက် စီစဉ်ပေးပါ
- **အဆင့်မြင့် တွဲဖက် ဖြန့်ဝေမှု** — ယုံကြည်ရသော developer ကိရိယာများကို WordPress.org core build မှ သီးခြား ထုတ်ဝေပါ
- **ယုံကြည်စိတ်ချရမှု ပြင်ဆင်ချက်များ** — model ရွေးချယ်မှု၊ request timeout များ၊ မူလ setup model ကိုင်တွယ်မှု၊ reasoning-text ကိုင်တွယ်မှု၊ retry လမ်းညွှန်ချက်၊ calendar token ခိုင်မာစေမှု၊ reminder ခိုင်မာစေမှု၊ onboarding စတင်မှုများ၊ အတည်ပြုထားသော ability ပြန်စမှုများနှင့် WordPress.org packaging review ပြဿနာများကို တိုးတက်စေပါ

## v1.16.0 ရှိ တိုးတက်မှုများ
