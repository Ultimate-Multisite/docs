# Action: wu_captcha_integrations_init {#action-wucaptchaintegrationsinit}

အင်တီဂရေးရှင်းအားလုံး စတင်ဖွဲ့စည်းပြီးနောက်တွင် အလုပ်လုပ်ပါသည်။
