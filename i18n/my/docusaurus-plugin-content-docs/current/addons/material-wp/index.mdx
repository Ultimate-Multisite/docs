# Material WP

WordPress ရဲ့ admin interface ဟာ အချိန်ကြာကြီး အတူတူပဲ ရှိနေခဲ့တာပါ။ ဒါပေမဲ့ Material Design နဲ့ ဒီဇိုင်းထုတ်ထားတဲ့ admin interface တစ်ခုက ဘယ်လိုပုံစံမျိုး ဖြစ်မလဲဆိုတာ စဉ်းစားဖူးပါသလား။

Material WP ဆိုတာ သင့်ကို အဲဒီပုံစံနဲ့ ခံစားမှုမျိုး ပေးနိုင်ဖို့ ဖန်တီးထားတဲ့ admin theme တစ်ခုဖြစ်ပြီး၊ သင့် client တွေဆီကို ပိုပြီး စိတ်ကြိုက်ပြင်ဆင်ထားတဲ့ Dashboard တစ်ခု ပေးနိုင်ဖို့ ကူညီပေးမှာပါ။

## အင်္ဂါရပ်များ {#features}

- Custom CSS ကို အသုံးပြုနိုင်ခြင်း
- အလွယ်တကူ စတိုင်လ်ကျယ်မှုများ သတ်မှတ်နိုင်ခြင်း
- Menu ကို ပြန်လည်စီစဉ်နိုင်ခြင်း
- အလွယ်တကူ import/export လုပ်နိုင်ခြင်း

## စာရွက်စာတမ်းများ {#documentation}

**Material WP ဖြင့် သင့် Admin Panel ကို စိတ်ကြိုက်ပြင်ဆင်ခြင်း**

## တပ်ဆင်ခြင်း {#installation}

1. 'material-wp' ကို '/wp-content/plugins/' directory ထဲသို့ တင်ပါ။
2. WordPress ရဲ့ 'Plugins' menu မှတစ်ဆင့် plugin ကို activate လုပ်ပါ။

## အသုံးပြုနည်းလမ်းညွှန် {#usage-guide}

WordPress ရဲ့ admin interface ဟာ အချိန်ကြာကြီး အတူတူပဲ ရှိနေခဲ့တာပါ။ ဒါပေမဲ့ Material Design နဲ့ ဒီဇိုင်းထုတ်ထားတဲ့ admin interface တစ်ခုက ဘယ်လိုပုံစံမျိုး ဖြစ်မလဲဆိုတာ စဉ်းစားဖူးပါသလား။

Material WP ဆိုတာ သင့်ကို အဲဒီပုံစံနဲ့ ခံစားမှုမျိုး ပေးနိုင်ဖို့ ဖန်တီးထားတဲ့ admin theme တစ်ခုဖြစ်ပြီး၊ သင့် client တွေဆီကို ပိုပြီး စိတ်ကြိုက်ပြင်ဆင်ထားတဲ့ Dashboard တစ်ခု ပေးနိုင်ဖို့ ကူညီပေးမှာပါ။ Material WP ဟာ Ultimate Multisite ရဲ့ premium plugins core ရဲ့ အစိတ်အပိုင်းလည်း ဖြစ်ပါတယ်။

**တပ်ဆင်ပြီးနောက်**၊ သင့်ရဲ့ _Plugins_ စာမျက်နှာကို သွားပြီး သင့် Network မှာ activate လုပ်ရန် နှိပ်ပါ။

အခုဆိုရင် သင့်ရဲ့ admin interface ကို အပြည့်အဝ ပြန်လည်ဒီဇိုင်းဆွဲပြီးသွားပါပြီ။ ဒါ့အပြင် ဘယ်ဘက်ဘက်မှာ Material WP လို့ခေါ်တဲ့ menu အသစ်တစ်ခုလည်း ရရှိမှာပါ။
