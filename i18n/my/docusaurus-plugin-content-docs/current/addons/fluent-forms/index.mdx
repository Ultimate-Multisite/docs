# Fluent Forms {#fluent-forms}

ဒီ add-on ကို သုံးခြင်းအားဖြင့် Multisite Ultimate မှာရှိတဲ့ products, memberships ဒါမှမဟုတ် sites တွေမှာ Fluent Forms ကို ကန့်သတ်မှုတွေ ထည့်သွင်းနိုင်ပါတယ်။
Fluent Forms က custom post types တွေအစား custom tables တွေကို အသုံးပြုထားတာကြောင့် ပုံမှန်ကန့်သတ်မှု (limiting) ရွေးချယ်စရာတွေက Fluent Forms နဲ့ အလုပ်မလုပ်နိုင်ပါဘူး။

## အင်္ဂါရပ်များ {#features}

- Fluent Forms ဒါမှမဟုတ် Conversational Fluent Forms အတွက် ကန့်သတ်မှုများ ထည့်သွင်းနိုင်ခြင်း။
- သုံးစွဲသူတွေက ကန့်သတ်မှုအကန့်အသတ်ကို ရောက်ရှိတဲ့အခါ သူတို့ရဲ့ plan ကို အဆင့်မြှင့်ဖို့ တိုက်တွန်းတဲ့ ဖော်ရွေတဲ့ မက်ဆေ့ခ်ျကို ရရှိမှာ ဖြစ်ပါတယ်။
