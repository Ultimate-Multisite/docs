# Domain Seller

**Multisite Ultimate: Domain Seller** သည် သင့် WordPress Multisite network ကို စွမ်းအားမြင့် ဒိုမိန်းရောင်းချရေး ပလက်ဖောင်းတစ်ခုအဖြစ် ပြောင်းလဲပေးပါသည်။ စာရင်းသွင်းချိန်အတွင်း သို့မဟုတ် ၎င်းတို့၏ Dashboard မှတဆင့် ဖောက်သည်များအား စိတ်ကြိုက်ဒိုမိန်းများ မှတ်ပုံတင်နိုင်စေရန်၊ အချိန်နှင့်တပြေးညီ ရရှိနိုင်မှု စစ်ဆေးခြင်းနှင့် ပွင့်လင်းမြင်သာသော ဈေးနှုန်းသတ်မှတ်ခြင်းများကို လုပ်ဆောင်ပေးပါသည်။

### Domain Seller ကို ဘာကြောင့် အသုံးပြုသင့်သလဲ။

Domain မှတ်ပုံတင်ခြင်းသည် WaaS (Website as a Service) ပလက်ဖောင်းတိုင်းအတွက် အလိုအပ်ဆုံး အပိုဝန်ဆောင်မှုများထဲမှ တစ်ခုဖြစ်သည်။ သင့်ဖောက်သည်များကို ပြင်ပ Registrar တစ်ခုခုဆီ ပို့ဆောင်မနေဘဲ၊ ဒိုမိန်းများကို တိုက်ရိုက်ရောင်းချနိုင်ခြင်းဖြင့်၊ သင့် network အတွင်းမှာ ဝင်ငွေပိုမိုထိန်းသိမ်းနိုင်ပြီး ဖောက်သည်များအတွက်လည်း အခက်အခဲမရှိသော အတွေ့အကြုံကို ပေးစွမ်းနိုင်ပါသည်။

### အဓိက လုပ်ဆောင်ချက်များ

#### မတူညီသော Domain Registrar များကို ပံ့ပိုးမှု
* **OpenSRS** — မှတ်ပုံတင်ခြင်း၊ သက်တမ်းတိုးခြင်းနှင့် စီမံခန့်ခွဲခြင်း အပြည့်အဝ
* **Namecheap** — WhoisGuard နှင့် sandbox စမ်းသပ်မှုများအထိ အပြည့်အဝ ပေါင်းစပ်ထားခြင်း
* **ချဲ့ထွင်နိုင်သော Architecture** — Integration Registry pattern ပေါ်တွင် တည်ဆောက်ထားသဖြင့် ပံ့ပိုးပေးသူအသစ်များ ထပ်ထည့်ရန် လွယ်ကူစေသည်
* **Built-in Connection Testing** — အသုံးပြုမည့် မတိုင်မီ ပံ့ပိုးပေးသူ၏ အထောက်အထားများနှင့် ချိတ်ဆက်မှုများကို စစ်ဆေးပေးသည်

#### လမ်းညွှန်မှု Setup Wizard
* **တစ်ချက်နှိပ်ရုံဖြင့် စတင်နိုင်ခြင်း** — ပံ့ပိုးပေးသူဖွဲ့စည်းမှု၊ TLD တင်သွင်းခြင်းနှင့် ဒိုမိန်းထုတ်ကုန်များ ဖန်တီးခြင်းတို့ကို လမ်းညွှန်ပေးသည်
* **အလိုအလျောက် TLD Sync** — ချိတ်ဆက်ထားသော ပံ့ပိုးပေးသူများထံမှ ရရှိနိုင်သော TLD အားလုံးနှင့် လက်ကားဈေးနှုန်းများကို တင်သွင်းပေးသည်
* **Default product ဖန်တီးပေးခြင်း** — ကျိုးကြောင်းဆီလျော်သော အမြတ်အစွန်း (markup) များဖြင့် အသုံးပြုရန် အသင့်ဖြစ်နေသော ဒိုမိန်းထုတ်ကုန်တစ်ခုကို ဖန်တီးပေးသည်

#### ပြောင်းလွယ်ပြင်လွယ်ရှိသော Domain ဈေးနှုန်းသတ်မှတ်ခြင်း
* **အမြတ်အစွန်း (markup) နည်းလမ်း ၃ မျိုး** — ရာခိုင်နှုန်းအမြတ်၊ သတ်မှတ်ထားသော အမြတ်၊ သို့မဟုတ် TLD တစ်ခုချင်းစီအတွက် သတ်မှတ်ဈေးနှုန်း
* **TLD အလိုက် ထုတ်ကုန်များ** — မတူညီသော TLD အုပ်စုများ (.com/.net, .io, နိုင်ငံကုဒ်များ စသည်) အတွက် သီးခြားဒိုမိန်းထုတ်ကုန်များ ဖန်တီးနိုင်သည်
* **အစမ်းဈေးနှုန်းများ** — စာရင်းသွင်းသူအသစ်များကို ဆွဲဆောင်ရန် ပထမနှစ်အတွက် လျှော့စျေးပေးနိုင်သည်
* **အချိန်နှင့်တပြေးညီ ဈေးနှုန်းများ** — ပံ့ပိုးပေးသူများထံမှ အသိဉာဏ်ရှိသော caching ဖြင့် တိုက်ရိုက်လက်ကားဈေးနှုန်းများ
* **Coupon ပံ့ပိုးမှု** — Multisite Ultimate discount codes များနှင့် အပြည့်အဝ တွဲဖက်အသုံးပြုနိုင်သည်

#### WHOIS ကိုယ်ရေးကိုယ်တာ အကာအကွယ် (WHOIS Privacy Protection)
* **ထုတ်ကုန်အလိုက် ဖွဲ့စည်းမှု** — WHOIS ကိုယ်ရေးကိုယ်တာကို Disabled၊ အမြဲပါဝင်စေရန် (Always Included) သို့မဟုတ် ဖောက်သည်ရွေးချယ်မှု (Customer Choice) အဖြစ် သတ်မှတ်နိုင်သည်
* **ဖောက်သည်မှ ရွေးချယ်မှု** — Customer Choice အဖြစ် သတ်မှတ်ထားပါက၊ checkout လုပ်စဉ်အတွင်း ဖောက်သည်များအား ကိုယ်ရေးကိုယ်တာ အကာအကွယ်ကို ထပ်ထည့်ရန် checkbox ပေါ်လာမည်။
* **ဖွဲ့စည်းနိုင်သော ဈေးနှုန်းများ** — ကိုယ်ရေးကိုယ်တာ အပိုဝန်ဆောင်မှုအတွက် တစ်နှစ်တာ ဈေးနှုန်းသတ်မှတ်နိုင်သည်၊ သို့မဟုတ် အခမဲ့ပေးနိုင်သည်
* **Registrar ပေါင်းစပ်မှု** — စာရင်းသွင်းချိန်တွင် WhoisGuard (Namecheap) သို့မဟုတ် WHOIS ကိုယ်ရေးကိုယ်တာ (OpenSRS) ကို အလိုအလျောက် ဖွင့်ပေးသည်

#### စုစည်းထားသော Checkout အတွေ့အကြုံ
* **Domain နည်းလမ်း ၃ မျိုး** — ဖောက်သည်များသည် Subdomain (အခမဲ့)၊ ဒိုမိန်းအသစ် မှတ်ပုံတင်ခြင်း၊ သို့မဟုတ် ၎င်းတို့ပိုင်ဆိုင်ပြီးသား ဒိုမိန်းတစ်ခုကို ချိတ်ဆက်ခြင်းတို့မှ ရွေးချယ်နိုင်သည်။
* **အချိန်နှင့်တပြေးညီ Domain ရှာဖွေခြင်း** — စာမျက်နှာကို ပြန်လည်မတင်ဘဲ AJAX ဖြင့် အားပေးထားသော ရရှိနိုင်မှု စစ်ဆေးခြင်း
* **အသိဉာဏ်ရှိသော အကြံပြုချက်များ** — နှစ်သက်သော ဒိုမိန်းကို အသုံးပြုပြီးသားဖြစ်ပါက အခြား TLD အကြံပြုချက်များ ပေးသည်
* **ပွင့်လင်းမြင်သာသော ဈေးနှုန်းများ** — မှတ်ပုံတင်ဈေးနှုန်း၊ သက်တမ်းတိုးဈေးနှုန်းနှင့် ရွေးချယ်နိုင်သော WHOIS ကိုယ်ရေးကိုယ်တာ အခကြေးငွေတို့ကို ရှင်းရှင်းလင်းလင်း ပြသသည်
* **Registrant ဆက်သွယ်ရန် နေရာများ** — Checkout လုပ်စဉ်အတွင်း လိုအပ်သော မှတ်ပုံတင်သူ အချက်အလက်များ (အမည်၊ လိပ်စာ၊ ဖုန်း) ကို စုဆောင်းသည်
* **အလိုအလျောက် Site URL** — ရွေးချယ်ထားသော ဒိုမိန်းမှ Site လိပ်စာကို အလိုအလျောက် ဆင်းသက်ပေးသည်

#### အလိုအလျောက် သက်တမ်းတိုးစနစ်
* **အဖွဲ့ဝင်အခြေခံ သက်တမ်းတိုးခြင်း** — ဖောက်သည်၏ အဖွဲ့ဝင်အဆင့်အတန်း (membership) အသက်ဝင်နေသရွေ့ ဒိုမိန်းများသည် အလိုအလျောက် သက်တမ်းတိုးသည်။
* **အသိဉာဏ်ရှိသော အချိန်ဇယားဆွဲခြင်း** — သက်တမ်းကုန်ဆုံးမည့်ရက်များအပေါ် အခြေခံ၍ သက်တမ်းတိုးခြင်း အလုပ်များကို စီစဉ်ပေးသည်
* **Retry logic** — သက်တမ်းတိုးရန် ကြိုးစားမှုများ ပျက်ကွက်ပါက အသိဉာဏ်ရှိစွာ ပြန်လည်ကြိုးစားပေးသည်
* **Email အကြောင်းကြားချက်များ** — မကြာမီ သက်တမ်းတိုးမည့်အကြောင်း၊ အောင်မြင်စွာ သက်တမ်းတိုးခြင်း၊ ပျက်ကွက်ခြင်းနှင့် သက်တမ်းကုန်ဆုံးခြင်းများအတွက် ဖွဲ့စည်းနိုင်သော အကြောင်းကြားချက်များ

#### ဖောက်သည် DNS စီမံခန့်ခွဲမှု
* **Self-service DNS** — ဖောက်သည်များသည် ၎င်းတို့၏ Dashboard မှ A, AAAA, CNAME, MX, TXT စသည့် DNS မှတ်တမ်းများကို ထည့်ခြင်း၊ တည်းဖြတ်ခြင်းနှင့် ဖျက်ခြင်းများ ပြုလုပ်နိုင်သည်။
* **Domain အခြေအနေ** — အချိန်နှင့်တပြေးညီ မှတ်ပုံတင်အခြေအနေ၊ သက်တမ်းကုန်ဆုံးရက်များနှင့် nameserver အချက်အလက်များ
* **အလိုအလျောက် Domain Mapping** — မှတ်ပုံတင်ထားသော သို့မဟုတ် ရှိပြီးသား ဒိုမိန်းများကို ဖောက်သည်၏ Site သို့ အလိုအလျောက် ချိတ်ဆက်ပေးသည်

#### Admin Tools
* **လက်ဖြင့် Domain မှတ်ပုံတင်ခြင်း** — Admin panel မှတစ်ဆင့် ဖောက်သည်များကိုယ်စား ဒိုမိန်းများကို မှတ်ပုံတင်ပေးနိုင်သည်
* **Domain စောင့်ကြည့်ခြင်း** — ဖောက်သည်များ၏ ဒိုမိန်းများ၊ အခြေအနေများနှင့် သက်တမ်းကုန်ဆုံးရက်များကို စာမျက်နှာတစ်ခုတည်းမှ ကြည့်ရှုနိုင်သည်
* **ပံ့ပိုးပေးသူအလိုက် Log များ** — Registrar တစ်ခုချင်းစီအတွက် သီးခြား log လမ်းကြောင်းများ (ဥပမာ- domain-seller-namecheap.log)
* **မှတ်ပုံတင်အသေးစိတ်အချက်အလက်များ** — မှတ်ပုံတင်သူ အချက်အလက်များနှင့် မှတ်ပုံတင်ဒေတာများကို core domain edit စာမျက်နှာတွင် ကြည့်ရှုနိုင်သည်

### ပံ့ပိုးပေးသော Providers များ

* **OpenSRS** — မှတ်ပုံတင်ခြင်း၊ သက်တမ်းတိုးခြင်း၊ DNS စီမံခန့်ခွဲခြင်းနှင့် WHOIS ကိုယ်ရေးကိုယ်တာတို့နှင့် အပြည့်အဝ ပေါင်းစပ်ထားသည်
* **Namecheap** — မှတ်ပုံတင်ခြင်း၊ သက်တမ်းတိုးခြင်း၊ အစုလိုက် ရရှိနိုင်မှု စစ်ဆေးခြင်းနှင့် အခမဲ့ WhoisGuard တို့နှင့် အပြည့်အဝ ပေါင်းစပ်ထားသည်
* **GoDaddy** — မှတ်ပုံတင်ခြင်းနှင့် သက်တမ်းတိုးခြင်း
* **ResellerClub** — မှတ်ပုံတင်ခြင်း၊ သက်တမ်းတိုးခြင်းနှင့် DNS စီမံခန့်ခွဲခြင်း
* **NameSilo** — မှတ်ပုံတင်ခြင်းနှင့် သက်တမ်းတိုးခြင်း
* **Enom** — မှတ်ပုံတင်ခြင်း၊ သက်တမ်းတိုးခြင်းနှင့် DNS စီမံခန့်ခွဲခြင်း

### နည်းပညာဆိုင်ရာ အသေးစိတ်အချက်များ

* **လိုအပ်ချက်:** PHP 7.4+၊ Multisite Ultimate 2.4.12+
* **Architecture:** Integration Registry pattern နှင့် Domain_Selling_Capability interface
* **Autoloading:** Jetpack autoloader with PSR-4 class mapping
* **Standards:** WordPress coding standards, strict typing, prepared statements

## Installation

1. **Multisite Ultimate v2.4.12+** ကို ထည့်သွင်းပြီး network-activated ဖြစ်ကြောင်း သေချာပါစေ။
2. ပလပ်အင်ကို `/wp-content/plugins/ultimate-multisite-domain-seller/` နေရာသို့ တင်ပါ။
3. WordPress Network Admin မှ **Plugins** တွင် ပလပ်အင်ကို Network-activate လုပ်ပါ။
4. **Setup Wizard** သည် ပထမဆုံး အသုံးပြုသည့်အခါ အလိုအလျောက် စတင်မည်ဖြစ်သည် — အောက်ပါအဆင့်များကို လိုက်နာပါ-
   * သင့် domain registrar ကို ချိတ်ဆက်ပါ (OpenSRS သို့မဟုတ် Namecheap)
   * ချိတ်ဆက်မှုကို စမ်းသပ်ပါ
   * ရရှိနိုင်သော TLD များနှင့် လက်ကားဈေးနှုန်းများကို တင်သွင်းပါ
   * ဈေးနှုန်းအမြတ် (pricing markup) ဖြင့် သင့်ပထမဆုံး ဒိုမိန်းထုတ်ကုန်ကို ဖန်တီးပါ
5. **Ultimate Multisite > Checkout Forms** တွင် သင်၏ checkout form အတွက် **Domain Selection** field ကို ထည့်ပါ။

## မကြာခဏ မေးလေ့ရှိသော မေးခွန်းများ

### Multisite Ultimate ဘယ် version လိုအပ်ပါသလဲ။

Version 2.4.12 သို့မဟုတ် အထက်။ ဤ addon သည် WP Ultimo v1.x သို့မဟုတ် 2.4.12 အောက်ရှိ Ultimate Multisite version များနှင့် တွဲဖက်အသုံးပြုနိုင်ခြင်း မရှိပါ။

### ဘယ် domain providers တွေကို ပံ့ပိုးပေးပါသလဲ။

Registrar ခြောက်ခုကို အပြည့်အဝ ပံ့ပိုးပေးထားသည်- **OpenSRS**, **Namecheap**, **GoDaddy**, **ResellerClub**, **NameSilo**, နှင့် **Enom** တို့ဖြစ်သည်။ ဤ addon ၏ Integration Registry architecture သည် စိတ်ကြိုက် ပံ့ပိုးပေးသူများ ထပ်ထည့်ရန် ရိုးရှင်းစေသည် — [developer guide](/developer/integration-guide/custom-registrar) ကို ကြည့်ရှုပါ။

### WHOIS ကိုယ်ရေးကိုယ်တာ အကာအကွယ်ကို ပေးနိုင်ပါသလား။

ဟုတ်ကဲ့။ ဒိုမိန်းထုတ်ကုန်တစ်ခုစီတွင် WHOIS Privacy setting ပါရှိပြီး နည်းလမ်း ၃ ခုရှိသည်- Disabled၊ Always Included၊ သို့မဟုတ် Customer Choice။ Customer Choice အဖြစ် သတ်မှတ်ထားပါက၊ ဖောက်သည်များသည် checkout လုပ်စဉ်အတွင်း ဈေးနှုန်း (သို့မဟုတ် Namecheap ၏ WhoisGuard အတွက် "အခမဲ့") နှင့်အတူ checkbox ကို မြင်ရမည်။ ကိုယ်ရေးကိုယ်တာ အမှတ်အသား (privacy flag) ကို စာရင်းသွင်းချိန်တွင် Registrar သို့ အလိုအလျောက် ပေးပို့ပေးပါသည်။

### TLD မတူညီတဲ့ ဈေးနှုန်းတွေ သတ်မှတ်လို့ရပါသလား။

ဟုတ်ကဲ့။ မတူညီသော TLD အုပ်စုများအတွက် သီးခြားဒိုမိန်းထုတ်ကုန်များ ဖန်တီးပါ။ ထုတ်ကုန်တစ်ခုစီတွင် ၎င်း၏ကိုယ်ပိုင် အမြတ်အစွန်းအမျိုးအစား (ရာခိုင်နှုန်း၊ သတ်မှတ်အမြတ်၊ သို့မဟုတ် သတ်မှတ်ဈေးနှုန်း)၊ ပံ့ပိုးထားသော TLD များ၊ နှင့် အစမ်းဈေးနှုန်းများ ရှိနိုင်ပါသည်။

### ဖောက်သည်တွေ ဘယ်လို checkout options တွေ မြင်ရပါသလဲ။

Domain Selection field သည် ဖောက်သည်များအား တက်ဘ် ၃ ခု ပေးသည်- **Subdomain** (သင့် network ပေါ်တွင် အခမဲ့ subdomain အသုံးပြုခြင်း)၊ **Register New Domain** (ဒိုမိန်းအသစ် ရှာဖွေပြီး မှတ်ပုံတင်ခြင်း)၊ နှင့် **Existing Domain** (၎င်းတို့ပိုင်ဆိုင်ပြီးသား ဒိုမိန်းကို ချိတ်ဆက်ခြင်း)။ သင်ဖွင့်ပေးထားသော နည်းလမ်းများသာ ပေါ်လာမည်ဖြစ်သည်။

### အလိုအလျောက် သက်တမ်းတိုးတာကို ပံ့ပိုးပေးပါသလား။

ဟုတ်ကဲ့။ ဒိုမိန်းများကို အလိုအလျောက် သက်တမ်းတိုးရန် သတ်မှတ်နိုင်ပြီး ဖောက်သည်၏ အဖွဲ့ဝင်အဆင့်အတန်းနှင့် ချည်နှောင်ထားသည်။ သက်တမ်းတိုးငွေပေးချေမှုများကို ဖောက်သည်၏ လက်ရှိ payment gateway မှတစ်ဆင့် လုပ်ဆောင်ပေးပါသည်။

### ဖောက်သည်တွေ သူတို့ရဲ့ DNS records တွေကို စီမံခန့်ခွဲနိုင်ပါသလား။

ဟုတ်ကဲ့။ ဖောက်သည်များသည် A, AAAA, CNAME, MX, နှင့် TXT မှတ်တမ်းများကို ထည့်ခြင်း၊ တည်းဖြတ်ခြင်း၊ နှင့် ဖျက်ခြင်းများ ပြုလုပ်နိုင်ရန် ၎င်းတို့၏ Dashboard တွင် DNS စီမံခန့်ခွဲမှု interface ကို ရရှိကြသည်။ DNS စီမံခန့်ခွဲမှုသည် OpenSRS, ResellerClub, နှင့် Enom တို့တွင် ရရှိနိုင်သည်။

### Setup wizard ပါပါသလား။

ဟုတ်ကဲ့။ Setup wizard သည် ပထမဆုံး အသုံးပြုသည့်အခါ အလိုအလျောက် လုပ်ဆောင်ပေးပြီး registrar ချိတ်ဆက်ခြင်း၊ TLD များ တင်သွင်းခြင်းနှင့် ပထမဆုံး ဒိုမိန်းထုတ်ကုန် ဖန်တီးခြင်းတို့ကို လမ်းညွှန်ပေးပါသည်။ addon settings မှလည်း wizard ကို ပြန်လည် run လုပ်နိုင်ပါသည်။
