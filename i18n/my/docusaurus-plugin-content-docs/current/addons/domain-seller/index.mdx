# Domain Seller

**Multisite Ultimate: Domain Seller** သည် သင့် WordPress Multisite ကွန်ရက်ကို အားကောင်းသော domain ရောင်းချရေး platform အဖြစ် ပြောင်းလဲပေးသည်။ သုံးစွဲသူများကို စာရင်းသွင်းချိန်တွင်ဖြစ်စေ၊ ၎င်းတို့၏ dashboard မှဖြစ်စေ စိတ်ကြိုက် domain များကို တိုက်ရိုက် မှတ်ပုံတင်နိုင်စေပြီး အချိန်နှင့်တပြေးညီ ရနိုင်မှုစစ်ဆေးခြင်းနှင့် ပွင့်လင်းသော ဈေးနှုန်းများကို ပေးပါသည်။

### Domain Seller ကို ဘာကြောင့် သုံးသင့်သလဲ? {#why-domain-seller}

Domain မှတ်ပုံတင်ခြင်းသည် WaaS (ဝက်ဘ်ဆိုက်ကို ဝန်ဆောင်မှုအဖြစ်) platform တိုင်းအတွက် အများဆုံးတောင်းဆိုကြသော ထပ်ဆောင်းရောင်းချမှုများထဲမှ တစ်ခုဖြစ်သည်။ သင့်သုံးစွဲသူများကို third-party registrar ထံ ပို့မည့်အစား domain များကို တိုက်ရိုက် ကမ်းလှမ်းနိုင်သည် — ထိုသို့ပြုလုပ်ခြင်းဖြင့် သင့်ကွန်ရက်အတွင်း ဝင်ငွေကို ပိုမိုထိန်းထားနိုင်ပြီး သုံးစွဲသူများအတွက် အနှောင့်အယှက်မရှိသော အတွေ့အကြုံကို ပေးနိုင်သည်။

### အဓိက လုပ်ဆောင်ချက်များ {#key-features}

#### Domain Registrar မျိုးစုံ ပံ့ပိုးမှု {#multiple-domain-registrar-support}
* **OpenSRS** — မှတ်ပုံတင်ခြင်း၊ သက်တမ်းတိုးခြင်းနှင့် စီမံခန့်ခွဲခြင်း အပြည့်အစုံ
* **Namecheap** — WhoisGuard နှင့် sandbox စမ်းသပ်မှုတို့ပါဝင်သော ပေါင်းစည်းမှု အပြည့်အစုံ
* **HostAfrica** — Domains Reseller API မှတစ်ဆင့် domain ရှာဖွေခြင်း၊ ဈေးနှုန်းသတ်မှတ်ခြင်း၊ မှတ်ပုံတင်ခြင်း၊ လွှဲပြောင်းခြင်း၊ DNS နှင့် ID ကာကွယ်မှု
* **Openprovider** — Reseller ဈေးနှုန်း၊ မှတ်ပုံတင်ခြင်း၊ သက်တမ်းတိုးခြင်း၊ လွှဲပြောင်းမှုများ၊ DNS၊ WHOIS ကိုယ်ရေးလုံခြုံမှုနှင့် TLD sync
* **Hostinger** — မျှဝေထားသော Hostinger ပေါင်းစည်းမှု token ကို အသုံးပြု၍ domain ရှာဖွေခြင်းနှင့် မှတ်ပုံတင်ခြင်း
* **တိုးချဲ့နိုင်သော Architecture** — Integration Registry pattern ပေါ်တွင် တည်ဆောက်ထားသောကြောင့် provider များ ထပ်ထည့်ရန် လွယ်ကူသည်
* **ပါဝင်ပြီးသား ချိတ်ဆက်မှု စမ်းသပ်ခြင်း** — တိုက်ရိုက်မသုံးမီ provider အထောက်အထားများနှင့် ချိတ်ဆက်နိုင်မှုကို စစ်ဆေးအတည်ပြုပါ

#### လမ်းညွှန်ပါ Setup Wizard {#guided-setup-wizard}
* **တစ်ချက်နှိပ် onboarding** — provider configuration၊ TLD import နှင့် domain product ဖန်တီးခြင်းတို့ကို အဆင့်ဆင့် လမ်းညွှန်ပေးသည်
* **အလိုအလျောက် TLD sync** — သင့်ချိတ်ဆက်ထားသော provider များမှ ရရှိနိုင်သည့် TLD အားလုံးနှင့် wholesale ဈေးနှုန်းများကို import လုပ်သည်
* **မူလ product ဖန်တီးခြင်း** — သင့်တင့်သော markup မူလတန်ဖိုးများဖြင့် အသုံးပြုရန် အသင့်ဖြစ်သော domain product ကို ဖန်တီးပေးသည်

#### ပြောင်းလွယ်ပြင်လွယ် Domain ဈေးနှုန်းသတ်မှတ်မှု {#flexible-domain-pricing}
* **Markup mode သုံးမျိုး** — ရာခိုင်နှုန်း markup၊ fixed markup သို့မဟုတ် TLD တစ်ခုစီအတွက် fixed price
* **TLD အလိုက် product များ** — TLD အုပ်စုအမျိုးမျိုး (.com/.net, .io, နိုင်ငံကုဒ်များ စသည်) အတွက် domain product များကို သီးခြားဖန်တီးပါ
* **မိတ်ဆက်ဈေးနှုန်း** — မှတ်ပုံတင်မှုအသစ်များကို ဆွဲဆောင်ရန် ပထမနှစ်အတွက် လျှော့ဈေး ကမ်းလှမ်းပါ
* **အချိန်နှင့်တပြေးညီ ဈေးနှုန်း** — ထိရောက်သော caching ဖြင့် provider များမှ live wholesale ဈေးနှုန်းများ
* **Coupon ပံ့ပိုးမှု** — Multisite Ultimate discount code များနှင့် အပြည့်အဝ ကိုက်ညီမှုရှိသည်

#### WHOIS ကိုယ်ရေးလုံခြုံမှု ကာကွယ်မှု {#whois-privacy-protection}
* **Product တစ်ခုချင်း configuration** — WHOIS ကိုယ်ရေးလုံခြုံမှုကို Disabled, Always Included, သို့မဟုတ် Customer Choice အဖြစ် သတ်မှတ်ပါ
* **သုံးစွဲသူ opt-in** — Customer Choice ဟု သတ်မှတ်ထားသောအခါ checkout checkbox တစ်ခုမှ သုံးစွဲသူများအား privacy protection ထည့်နိုင်စေသည်
* **ပြင်ဆင်သတ်မှတ်နိုင်သော ဈေးနှုန်း** — privacy add-on အတွက် တစ်နှစ်ချင်းဈေးနှုန်း သတ်မှတ်နိုင်သည်၊ သို့မဟုတ် အခမဲ့ ကမ်းလှမ်းနိုင်သည်
* **Registrar ပေါင်းစည်းမှု** — မှတ်ပုံတင်ချိန်တွင် WhoisGuard (Namecheap) သို့မဟုတ် WHOIS ကိုယ်ရေးလုံခြုံမှု (OpenSRS) ကို အလိုအလျောက် ဖွင့်ပေးသည်

#### ပေါင်းစည်းထားသော Checkout အတွေ့အကြုံ {#unified-checkout-experience}
* **Domain mode သုံးမျိုး** — သုံးစွဲသူများသည် subdomain (အခမဲ့)၊ domain အသစ် မှတ်ပုံတင်ခြင်း၊ သို့မဟုတ် ၎င်းတို့ပိုင်ဆိုင်ပြီးသား domain ကို mapping လုပ်ခြင်းတို့အနက် ရွေးချယ်နိုင်သည်
* **အချိန်နှင့်တပြေးညီ domain ရှာဖွေမှု** — စာမျက်နှာ ပြန်တင်စရာမလိုသော AJAX အခြေပြု ရနိုင်မှုစစ်ဆေးခြင်း
* **Smart အကြံပြုချက်များ** — နှစ်သက်သော domain ကို ယူထားပြီးသားဖြစ်ပါက အခြား TLD အကြံပြုချက်များ
* **ပွင့်လင်းသော ဈေးနှုန်း** — မှတ်ပုံတင်ဈေးနှုန်း၊ သက်တမ်းတိုးဈေးနှုန်းနှင့် ရွေးချယ်နိုင်သော WHOIS ကိုယ်ရေးလုံခြုံမှု အခကြေးငွေကို ရှင်းလင်းစွာ ပြသသည်
* **Registrant ဆက်သွယ်ရန် field များ** — checkout အတွင်း လိုအပ်သော registrant အချက်အလက်များ (အမည်၊ လိပ်စာ၊ ဖုန်း) ကို တစ်နေရာတည်းတွင် ကောက်ယူသည်
* **အလိုအလျောက် ဖန်တီးထားသော site URL** — ရွေးချယ်ထားသော domain မှ site လိပ်စာကို အလိုအလျောက် ထုတ်ယူသတ်မှတ်သည်

#### အလိုအလျောက် သက်တမ်းတိုးစနစ် {#auto-renewal-system}
* **Membership နှင့် ချိတ်ဆက်ထားသော သက်တမ်းတိုးမှုများ** — သုံးစွဲသူ၏ membership active ဖြစ်နေသရွေ့ domain များကို အလိုအလျောက် သက်တမ်းတိုးသည်
* **Smart အချိန်ဇယားဆွဲခြင်း** — Domain သက်တမ်းကုန်ဆုံးရက်များအပေါ် မူတည်၍ renewal job များကို queue ထဲ ထည့်သည်
* **ပြန်လည်ကြိုးစားမှု logic** — မအောင်မြင်သော သက်တမ်းတိုးကြိုးစားမှုများအတွက် ထိရောက်သော ပြန်လည်ကြိုးစားမှုများ
* **Email အသိပေးချက်များ** — လာမည့် သက်တမ်းတိုးမှုများ၊ အောင်မြင်သော သက်တမ်းတိုးမှုများ၊ မအောင်မြင်မှုများနှင့် သက်တမ်းကုန်ဆုံးမှုများအတွက် ပြင်ဆင်သတ်မှတ်နိုင်သော သတိပေးချက်များ

#### သုံးစွဲသူ DNS စီမံခန့်ခွဲမှု {#customer-dns-management}
* **ကိုယ်တိုင်ဝန်ဆောင်မှု DNS** — သုံးစွဲသူများသည် ၎င်းတို့၏ dashboard မှ DNS record များ (A, AAAA, CNAME, MX, TXT) ကို ထည့်နိုင်၊ ပြင်နိုင်၊ ဖျက်နိုင်သည်
* **Domain အခြေအနေ** — အချိန်နှင့်တပြေးညီ မှတ်ပုံတင်အခြေအနေ၊ သက်တမ်းကုန်ဆုံးရက်များနှင့် nameserver အချက်အလက်
* **အလိုအလျောက် domain mapping** — မှတ်ပုံတင်ထားသော သို့မဟုတ် ရှိပြီးသား domain များကို သုံးစွဲသူ၏ site နှင့် အလိုအလျောက် mapping လုပ်သည်

#### Admin ကိရိယာများ {#admin-tools}
* **Manual domain မှတ်ပုံတင်ခြင်း** — admin panel မှ သုံးစွဲသူများကိုယ်စား domain များကို မှတ်ပုံတင်ပါ
* **Domain စောင့်ကြည့်ခြင်း** — သုံးစွဲသူ domain အားလုံး၊ အခြေအနေများနှင့် သက်တမ်းကုန်ဆုံးရက်များကို စာမျက်နှာတစ်ခုတည်းမှ ကြည့်ပါ
* **Provider အလိုက် logging** — registrar တစ်ခုချင်းစီအတွက် သီးခြား log channel များ (ဥပမာ၊ domain-seller-namecheap.log)
* **မှတ်ပုံတင်အသေးစိတ်များ** — core domain edit page တွင် registrant အချက်အလက်နှင့် မှတ်ပုံတင် data ကို ကြည့်ပါ

### ပံ့ပိုးထားသော Provider များ {#supported-providers}

* **OpenSRS** — မှတ်ပုံတင်ခြင်း၊ သက်တမ်းတိုးခြင်း၊ DNS စီမံခန့်ခွဲမှုနှင့် WHOIS ကိုယ်ရေးလုံခြုံမှုတို့ပါဝင်သော ပေါင်းစည်းမှု အပြည့်အစုံ
* **Namecheap** — မှတ်ပုံတင်ခြင်း၊ သက်တမ်းတိုးခြင်း၊ batch ရနိုင်မှုစစ်ဆေးခြင်းနှင့် အခမဲ့ WhoisGuard တို့ပါဝင်သော ပေါင်းစည်းမှု အပြည့်အစုံ
* **HostAfrica** — မှတ်ပုံတင်ခြင်း၊ သက်တမ်းတိုးခြင်း၊ လွှဲပြောင်းမှုများ၊ DNS စီမံခန့်ခွဲမှု၊ EPP code ရယူခြင်း၊ registrar lock နှင့် ID ကာကွယ်မှု
* **Openprovider** — မှတ်ပုံတင်ခြင်း၊ သက်တမ်းတိုးခြင်း၊ လွှဲပြောင်းမှုများ၊ DNS စီမံခန့်ခွဲမှု၊ EPP code ရယူခြင်း၊ registrar lock၊ WHOIS ကိုယ်ရေးလုံခြုံမှုနှင့် reseller TLD sync
* **Hostinger** — မှတ်ပုံတင်ခြင်း၊ domain ရှာဖွေကြည့်ခြင်း၊ nameserver update များ၊ registrar lock နှင့် WHOIS ကိုယ်ရေးလုံခြုံမှု; သက်တမ်းတိုးမှုများကို Hostinger auto-renew က ကိုင်တွယ်သည်
* **GoDaddy** — မှတ်ပုံတင်ခြင်းနှင့် သက်တမ်းတိုးခြင်း
* **ResellerClub** — မှတ်ပုံတင်ခြင်း၊ သက်တမ်းတိုးခြင်းနှင့် DNS စီမံခန့်ခွဲမှု
* **NameSilo** — မှတ်ပုံတင်ခြင်းနှင့် သက်တမ်းတိုးခြင်း
* **Enom** — မှတ်ပုံတင်ခြင်း၊ သက်တမ်းတိုးခြင်းနှင့် DNS စီမံခန့်ခွဲမှု

### နည်းပညာအသေးစိတ်များ {#technical-details}

* **လိုအပ်သည်:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **Architecture:** Domain_Selling_Capability interface ပါဝင်သော Integration Registry pattern
* **Autoloading:** PSR-4 class mapping ပါဝင်သော Jetpack autoloader
* **စံနှုန်းများ:** WordPress coding standards၊ strict typing၊ prepared statements

## တပ်ဆင်ခြင်း {#installation}

1. **Multisite Ultimate v2.4.12+** ကို ထည့်သွင်းထားပြီး network-activated ဖြစ်နေကြောင်း သေချာစစ်ဆေးပါ
2. plugin ကို `/wp-content/plugins/ultimate-multisite-domain-seller/` သို့ တင်ပါ
3. WordPress Network Admin ရှိ **plugin များ** မှ plugin ကို network-activate လုပ်ပါ
4. ပထမဆုံး activation တွင် **စတင်ပြင်ဆင်မှု Wizard** သည် အလိုအလျောက် စတင်ပါမည် — အောက်ပါအဆင့်များကို လိုက်နာပါ:
   * သင့် domain registrar ကို ချိတ်ဆက်ပါ
   * ချိတ်ဆက်မှုကို စမ်းသပ်ပါ
   * ရရှိနိုင်သော TLD များနှင့် wholesale ဈေးနှုန်းများကို import လုပ်ပါ
   * ဈေးနှုန်း markup ပါသော သင့် ပထမဆုံး domain product ကို ဖန်တီးပါ
5. **Ultimate Multisite > ငွေရှင်းလွှာများ** တွင် သင့် checkout form ထဲသို့ **Domain ရွေးချယ်မှု** field ကို ထည့်ပါ

## မေးလေ့ရှိသော မေးခွန်းများ {#frequently-asked-questions}

### Multisite Ultimate ၏ မည်သည့် version လိုအပ်ပါသလဲ? {#what-version-of-multisite-ultimate-is-required}

Version 2.4.12 သို့မဟုတ် ထို့အထက် လိုအပ်သည်။ ဤ addon သည် WP Ultimo v1.x သို့မဟုတ် 2.4.12 အောက်ရှိ Ultimate Multisite version များနှင့် မကိုက်ညီပါ။

### မည်သည့် domain provider များကို ထောက်ပံ့ထားပါသလဲ? {#which-domain-providers-are-supported}

registrar ကိုးခုကို ထောက်ပံ့ထားသည်: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo**, နှင့် **Enom**။ registrar API များက မတူညီသော operation များကို ဖော်ပြပေးသောကြောင့် လုပ်ဆောင်နိုင်စွမ်းများသည် provider အလိုက် ကွဲပြားသည်။ provider တစ်ခုချင်းစီ၏ အကျဉ်းချုပ်အတွက် setup guide ကို ကြည့်ပါ။ addon ၏ Integration Registry ဖွဲ့စည်းပုံကြောင့် custom provider များ ထည့်ရန် လွယ်ကူသည် — [developer guide](/developer/integration-guide/custom-registrar) ကို ကြည့်ပါ။

### WHOIS privacy protection ပေးနိုင်ပါသလား? {#can-i-offer-whois-privacy-protection}

ပေးနိုင်ပါသည်။ domain product တစ်ခုစီတွင် mode သုံးမျိုးပါသော WHOIS Privacy setting ရှိသည်: Disabled, Always Included, သို့မဟုတ် Customer Choice။ Customer Choice အဖြစ် သတ်မှတ်ထားပါက customer များသည် checkout အတွင်း ဈေးနှုန်းပါသော checkbox ကို မြင်ရမည် (သို့မဟုတ် Namecheap ၏ WhoisGuard အတွက် "အခမဲ့")။ registration လုပ်ချိန်တွင် privacy flag ကို registrar ထံ အလိုအလျောက် ပို့ပေးသည်။

### TLD များအလိုက် ဈေးနှုန်း မတူအောင် သတ်မှတ်နိုင်ပါသလား? {#can-i-set-different-pricing-for-different-tlds}

သတ်မှတ်နိုင်ပါသည်။ မတူညီသော TLD group များအတွက် သီးခြား domain product များ ဖန်တီးပါ။ product တစ်ခုစီတွင် ၎င်း၏ ကိုယ်ပိုင် markup type (ရာခိုင်နှုန်း၊ fixed markup၊ သို့မဟုတ် fixed price)၊ ထောက်ပံ့ထားသော TLD များနှင့် introductory pricing ရှိနိုင်သည်။

### checkout တွင် customer များ မည်သည့်ရွေးချယ်စရာများကို မြင်ရပါသလဲ? {#what-checkout-options-do-customers-see}

Domain ရွေးချယ်မှု field သည် customer များအား tab သုံးခု ပေးသည်: **Subdomain** (သင့် network ပေါ်ရှိ အခမဲ့ subdomain ကို အသုံးပြုရန်), **Domain အသစ် မှတ်ပုံတင်ရန်** (domain အသစ်ကို ရှာဖွေပြီး မှတ်ပုံတင်ရန်), နှင့် **ရှိပြီးသား Domain** (သူတို့ ပိုင်ဆိုင်ပြီးသား domain ကို map လုပ်ရန်)။ သင် enable လုပ်ထားသော mode များသာ ပေါ်လာမည်။

### automatic renewal များကို ထောက်ပံ့ပါသလား? {#does-it-support-automatic-renewals}

ထောက်ပံ့ပါသည်။ Domain များကို auto-renew အဖြစ် သတ်မှတ်နိုင်ပြီး customer ၏ membership status နှင့် ချိတ်ဆက်ထားသည်။ renewal payment များကို customer ၏ ရှိပြီးသား payment gateway မှတစ်ဆင့် လုပ်ဆောင်သည်။

### customer များသည် သူတို့၏ DNS record များကို စီမံခန့်ခွဲနိုင်ပါသလား? {#can-customers-manage-their-dns-records}

စီမံခန့်ခွဲနိုင်ပါသည်။ customer များသည် dashboard တွင် A, AAAA, CNAME, MX, နှင့် TXT record များ ထည့်ရန်၊ ပြင်ရန်၊ ဖျက်ရန် DNS စီမံခန့်ခွဲမှု interface ကို ရရှိသည်။ DNS စီမံခန့်ခွဲမှုကို OpenSRS, ResellerClub, Enom, HostAfrica, နှင့် Openprovider တို့တွင် ရရှိနိုင်သည်။ hosted domain များအတွက် Hostinger DNS ကို core Hostinger domain-mapping integration က ကိုင်တွယ်သည်။

### setup wizard ရှိပါသလား? {#is-there-a-setup-wizard}

ရှိပါသည်။ setup wizard သည် ပထမဆုံး activation တွင် အလိုအလျောက် လည်ပတ်ပြီး registrar ချိတ်ဆက်ခြင်း၊ TLD များ import လုပ်ခြင်းနှင့် သင့် ပထမဆုံး domain product ဖန်တီးခြင်းတို့ကို အဆင့်လိုက် လမ်းညွှန်ပေးသည်။ wizard ကို addon settings မှလည်း ထပ်မံ run နိုင်သည်။
