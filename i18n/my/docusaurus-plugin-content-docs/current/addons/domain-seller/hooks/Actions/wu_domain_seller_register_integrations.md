# Action: wu_domain_seller_register_integrations

အခြား plugin များက domain seller integration အသစ်များကို မှတ်ပုံတင်နိုင်စေရန် ခွင့်ပြုပါသည်။

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $registry | `\WP_Ultimo\Integrations\Integration_Registry` | ပေါင်းစပ်မှု မှတ်ပုံတင်စနစ် (The integration registry) ဖြစ်ပါသည်။ |
