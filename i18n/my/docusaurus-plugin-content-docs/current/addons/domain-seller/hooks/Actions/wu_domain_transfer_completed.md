# Action: wu_domain_transfer_completed

ဒိုမိန်း လွှဲပြောင်းမှု ပြီးမြောက်သွားတဲ့အခါမှာ ဒီ action ကို ပစ်လွှတ်ပါတယ်။

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $domain | `\WP_Ultimo\Models\Domain` | ဒိုမိန်း အရာဝတ္ထု (domain object) ဖြစ်ပါတယ်။ |
| $transfer_data | `array` | လွှဲပြောင်းမှု ပြီးမြောက်တဲ့ အချက်အလက်တွေပါ။ |
