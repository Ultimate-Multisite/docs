# Support Agents (ပံ့ပိုးကူညီသူများ) {#support-agents}

တစ်ခုလုံးကို ကိုယ်တိုင် စီမံခန့်ခွဲရတာက မလွယ်တဲ့အလုပ်ပါ။ Network က ကြီးလာတာနဲ့အမျှ ပံ့ပိုးမှု (support) အတွက် လိုအပ်ချက်တွေ ပိုများလာမှာပါ။ အလုပ်ဝန်ပိတာကို လျော့ကျစေဖို့ ကူညီပေးမယ့်သူတွေကို ခေါ်ယူတာက ကောင်းတဲ့အကြံပါ။ ဒါပေမဲ့ အခွင့်အရေး အပြည့်အဝ ပေးလိုက်တာက စွန့်စားရပြီး Network ကို ထိခိုက်စေနိုင်ပါတယ်။ ဒီနေရာမှာ Support Agent add-on က အသုံးဝင်လာမှာပါ။

ဒီ add-on က Support Agent လို့ခေါ်တဲ့ WordPress user အမျိုးအစားအသစ်တစ်ခုကို ဖန်တီးခွင့်ပေးပြီး၊ အဲဒီ user ဘယ်လိုခွင့်ပြုချက်တွေ ရှိရမလဲဆိုတာကို အပြည့်အဝ ထိန်းချုပ်နိုင်စေပါတယ်။ ခွင့်ပြုချက်ပေးတာက toggle button တစ်ခုကို ဖွင့်လိုက်သလို လွယ်ကူပြီး၊ ဖယ်ရှားဖို့ ဆုံးဖြတ်လိုက်ရင်လည်း အဲဒီခလုတ်ကို ပိတ်လိုက်ရုံပါပဲ။

## Features (အင်္ဂါရပ်များ) {#features}

- WordPress နဲ့ Ultimate Multisite ရဲ့ ခွင့်ပြုချက်တွေကို အသေးစိတ် စာရင်းပြုစုထားခြင်း။
- အသုံးပြုခွင့်ကို အလွယ်တကူ ဖွင့်/ပိတ် လုပ်နိုင်ခြင်း။
- Agent အကောင့်တွေကို အကန့်အသတ်မရှိ ဖန်တီးနိုင်ခြင်း။

## Installation (တပ်ဆင်ခြင်း) {#installation}

1. 'ultimate-multisite-support-agents' ကို '/wp-content/plugins/' directory ထဲသို့ တင်ပါ။
2. WordPress ရဲ့ 'Plugins' menu ကနေ plugin ကို activate လုပ်ပါ။
3. plugin ကို စနစ်တကျ တပ်ဆင်နိုင်ဖို့ အဆင့်ဆင့် လမ်းညွှန် (Wizard) ကို လိုက်နာပါ။
