# Support Tickets {#support-tickets}

**Support Tickets** သည် Ultimate Multisite အတွက် အစွမ်းထက်တဲ့ Addon တစ်ခုဖြစ်ပြီး၊ သင့် WordPress network တစ်ခုလုံးမှာ အပြည့်အစုံ စနစ်ကျတဲ့ Support Ticket စနစ်တစ်ခုကို ထည့်ပေးမှာ ဖြစ်ပါတယ်။ SaaS ပလက်ဖောင်းများ၊ WaaS (Website as a Service) ပေးသူများ၊ ဒါမှမဟုတ် network အဆင့်ရော၊ တစ်ဦးချင်း site အဆင့်မှာပါ Customer Support ပေးဖို့ လိုအပ်တဲ့ multisite network များအတွက် အကောင်းဆုံးပါပဲ။

## Support Tickets ကို ဘာကြောင့် ရွေးချယ်သင့်သလဲ။ {#why-choose-support-tickets}

* **အဆင့်အမျိုးမျိုး ပံ့ပိုးမှု (Multi-Level Support)**: Network စီမံခန့်ခွဲသူများက network တစ်ခုလုံးအတွက် ticket များကို ကိုင်တွယ်ပေးပြီး၊ site ပိုင်ရှင်များက သူတို့ရဲ့ ကိုယ်ပိုင် customer support များကို စီမံခန့်ခွဲနိုင်ပါတယ်။
* **ကျွမ်းကျင်တဲ့ လုပ်ငန်းစဉ် (Professional Workflow)**: ဦးစားပေးအဆင့်များ၊ အခြေအနေများ၊ နှင့် တာဝန်ပေးအပ်မှုများဖြင့် ticket ၏ ဘဝသံသရာတစ်ခုလုံးကို စီမံခန့်ခွဲနိုင်ပါတယ်။
* **Customer အတွေ့အကြုံ (Customer Experience)**: အသုံးပြုရလွယ်ကူပြီး၊ စတိုင်ကျတဲ့၊ responsive interface တစ်ခုဖြစ်လို့ customer တွေ အကူအညီရဖို့ လွယ်ကူစေပါတယ်။
* **လုံခြုံပြီး ယုံကြည်စိတ်ချရခြင်း (Secure & Reliable)**: WordPress ရဲ့ လုံခြုံရေး အကောင်းဆုံး အလေ့အကျင့်တွေနဲ့ အကောင်းဆုံး database စွမ်းဆောင်ရည်တွေနဲ့ တည်ဆောက်ထားပါတယ်။
* **ချောမွေ့စွာ ပေါင်းစပ်နိုင်ခြင်း (Seamless Integration)**: Ultimate Multisite ရဲ့ customer နဲ့ site စီမံခန့်ခွဲမှုတွေနဲ့ အပြည့်အဝ အဆင်ပြေစွာ အလုပ်လုပ်ပါတယ်။

## အဓိက အင်္ဂါရပ်များ (Key Features) {#key-features}

**အပြည့်အစုံ Ticket စီမံခန့်ခွဲမှု (Complete Ticket Management)**

* အလွယ်တကူ ကိုးကားနိုင်မယ့် ထူးခြားတဲ့ ticket နံပါတ်ပေးစနစ်
* စီမံနိုင်တဲ့ ဦးစားပေးအဆင့်များ (Low, Normal, High, Critical)
* အပြည့်အစုံ အခြေအနေ လုပ်ငန်းစဉ် (Open, In Progress, Waiting, Resolved, Closed)
* Ticket အမျိုးအစားခွဲခြင်း (General, Technical, Billing, Feature Request, Bug Report)
* အလိုအလျောက် ခွဲဝေပေးနိုင်တဲ့ (optional auto-assignment) စနစ်နဲ့ support agent တွေဆီ ticket များကို ခွဲဝေပေးနိုင်ခြင်း

**အကြောင်းအရာ ဆက်စပ် စကားပြောဆိုမှုများ (Threaded Conversations)**

* ticket အကြောင်းပြန်မှုများအတွက် အပြည့်အစုံ စကားပြောဆိုမှု ဆက်စပ်ခြင်း (Full conversation threading)
* ဝန်ထမ်းနဲ့ customer တို့ရဲ့ အကြောင်းပြန်မှုတွေအတွက် သီးခြား စတိုင်လ်ကျတဲ့ ပုံစံများ
* customer တွေ မမြင်နိုင်တဲ့ ဝန်ထမ်းရဲ့ ကိုယ်ပိုင်မှတ်စုများ (Private staff notes)
* အချိန်နဲ့တပြေးညီ AJAX ဖြင့် အလုပ်လုပ်သော interface
* ပုံစံထည့်သွင်းမှု ရွေးချယ်စရာများပါတဲ့ ခေတ်မီ text editor

**ဖိုင်တွဲများ (File Attachments)**

* drag-and-drop ပံ့ပိုးမှုနဲ့ လုံခြုံတဲ့ ဖိုင်တင်ခြင်း
* စီမံနိုင်တဲ့ ဖိုင်အမျိုးအစားနဲ့ အရွယ်အစား ကန့်သတ်ချက်များ
* အန္တရာယ်ရှိတဲ့ ဖိုင်တွေကို သိရှိနိုင်တဲ့ အဆင့်မြင့် လုံခြုံရေး
* ဝင်ရောက်ခွင့် ထိန်းချုပ်မှုများပါတဲ့ ကာကွယ်ထားတဲ့ download များ

**ဉာဏ်ရည်ထက်မြက်သော အကြောင်းကြားချက်များ (Smart Notifications)**

* ticket အသစ်များ၊ အကြောင်းပြန်မှုများ၊ နှင့် အခြေအနေပြောင်းလဲမှုများအတွက် အလိုအလျောက် email အကြောင်းကြားချက်များ
* စိတ်ကြိုက် ပြုလုပ်နိုင်သော email template များ
* ဝန်ထမ်းနဲ့ customer များအတွက် အခန်းကဏ္ဍအလိုက် အကြောင်းကြားချက်များ
* ပုဂ္ဂိုလ်ရေးအရ ပိုမိုကောင်းမွန်စေရန် dynamic template variables များ

**အစွမ်းထက်သော Admin Interface (Powerful Admin Interface)**

* စီမံခန့်ခွဲမှုနဲ့ စစ်ထုတ်မှုများပါတဲ့ အဆင့်မြင့် ticket စာရင်း
* အခြေအနေ၊ ဦးစားပေးအဆင့်၊ နှင့် ခွဲဝေပေးမှုများကို လျင်မြန်စွာ ပြင်ဆင်နိုင်ခြင်း
* ထိရောက်စွာ စီမံခန့်ခွဲနိုင်ရန် အစုလိုက် လုပ်ဆောင်မှုများ (Bulk actions)
* စွမ်းဆောင်ရည် အတိုင်းအတာများပါတဲ့ Statistics dashboard
* အပြည့်အစုံ စနစ်တကျ ချိန်ညှိမှု panel

**လှပသော Customer Portal (Beautiful Customer Portal)**

* စက်ပစ္စည်းအားလုံးမှာ ကြည်လင်ပြတ်သားပြီး responsive interface
* Mobile-first ဒီဇိုင်း
* အဆင့်မြင့် ရှာဖွေမှုနှင့် စစ်ထုတ်မှုများ
* ပုံစံထည့်သွင်းမှု toolbar ပါတဲ့ ခေတ်မီ editor
* အသုံးပြုရလွယ်ကူတဲ့ ticket တင်သွင်းမှု ပုံစံများ

**ပြောင်းလွယ်ပြင်လွယ်ရှိသော ပေါင်းစပ်မှု (Flexible Integration)**

* shortcodes များ သို့မဟုတ် Ultimate Multisite UI elements များကို အသုံးပြုခြင်း
* သင့် site နေရာတိုင်းမှာ ticket ပုံစံများနှင့် စာရင်းများကို ပြသနိုင်ခြင်း
* multisite network ဖွဲ့စည်းပုံနှင့် ချောမွေ့စွာ အလုပ်လုပ်ခြင်း
* hooks နှင့် filters များဖြင့် ချဲ့ထွင်နိုင်ခြင်း

## ဘယ်သူတွေအတွက် အသင့်တော်ဆုံးလဲ။ {#perfect-for}

* Customer support လိုအပ်တဲ့ SaaS ပလက်ဖောင်းများ
* Client site များစွာကို စီမံခန့်ခွဲတဲ့ WaaS ပေးသူများ
* အဆင့်အလိုက် ပံ့ပိုးမှုရှိတဲ့ Membership site များ
* မတူညီတဲ့ brand များစွာကို စီမံခန့်ခွဲတဲ့ Network စီမံခန့်ခွဲသူများ
* Customer service ပေးတဲ့ Site ပိုင်ရှင်များ

## တပ်ဆင်ခြင်း (Installation) {#installation}

1. Ultimate Multisite plugin ကို သင့် WordPress network မှာ တပ်ဆင်ပြီး အသက်သွင်းထားကြောင်း သေချာပါစေ။
2. Addon ဖိုင်များကို သင့် `/wp-content/plugins/` directory ထဲသို့ တင်ပါ။
3. WordPress ရဲ့ 'Plugins' menu ကနေ Network အဆင့်မှာ plugin ကို အသက်သွင်းပါ။
4. သင့်ရဲ့ setting တွေကို ချိန်ညှိဖို့ Network Admin > Support Tickets ကို သွားပါ။

## မကြာခဏ မေးလေ့ရှိသော မေးခွန်းများ (Frequently Asked Questions) {#frequently-asked-questions}

## ၎င်းသည် Ultimate Multisite ကို လိုအပ်ပါသလား။ {#does-this-require-ultimate-multisite}

ဟုတ်ကဲ့၊ ဤ Addon သည် Ultimate Multisite ကို တပ်ဆင်ပြီး အသက်သွင်းထားရန် လိုအပ်ပါသည်။ ၎င်းသည် Ultimate Multisite ၏ customer နှင့် site စီမံခန့်ခွဲမှု အင်္ဂါရပ်များနှင့် နက်နက်ရှိုင်းရှိုင်း ပေါင်းစပ်ထားပါသည်။

## Site ပိုင်ရှင်များက သူတို့ရဲ့ ကိုယ်ပိုင် customer support ကို ကိုင်တွယ်နိုင်ပါသလား။ {#can-site-owners-handle-their-own-customer-support}

လုံးဝရပါတယ်။ Site ပိုင်ရှင်များသည် ၎င်းတို့၏ customer များကို ပံ့ပိုးမှုပေးရန် ticket စနစ်ကို အသုံးပြုနိုင်ပြီး၊ network စီမံခန့်ခွဲသူများက network တစ်ခုလုံးအတွက် support ticket များကို ကိုင်တွယ်နိုင်ပါတယ်။

## Customer များက ဘယ်လို ticket တင်သွင်းရပါသလဲ။ {#how-do-customers-submit-tickets}

shortcodes များ သို့မဟုတ် Ultimate Multisite UI elements များကို အသုံးပြု၍ သင့် site နေရာတိုင်းမှာ ticket တင်သွင်းမှု ပုံစံများကို ထည့်သွင်းနိုင်ပါတယ်။ Customer များက ပုံစံကို ဖြည့်ပြီး သူတို့ရဲ့ တောင်းဆိုချက်ကို တင်သွင်းရုံပါပဲ။

## ticket အခြေအနေများနှင့် ဦးစားပေးအဆင့်များကို စိတ်ကြိုက် ပြုလုပ်နိုင်ပါသလား။ {#can-i-customize-the-ticket-statuses-and-priorities}

ဟုတ်ကဲ့! သင့်ရဲ့ support လုပ်ငန်းစဉ်နဲ့ ကိုက်ညီအောင် ဦးစားပေးအဆင့်များ၊ အခြေအနေများ၊ နှင့် ticket အမျိုးအစားများကို network admin setting များမှာ အပြည့်အဝ စိတ်ကြိုက် ပြုလုပ်နိုင်ပါတယ်။

## ဖိုင်တွဲများ ပံ့ပိုးမှုပေးပါသလား။ {#does-it-support-file-attachments}

ဟုတ်ကဲ့၊ customer တွေရော ဝန်ထမ်းတွေပါ ticket များနှင့် အကြောင်းပြန်မှုများမှာ ဖိုင်တွဲများ တင်နိုင်ပါတယ်။ လုံခြုံရေးအတွက် ခွင့်ပြုထားတဲ့ ဖိုင်အမျိုးအစားနဲ့ အရွယ်အစား ကန့်သတ်ချက်တွေကို သင် အပြည့်အဝ ထိန်းချုပ်နိုင်ပါတယ်။

## အကြောင်းကြားချက်များကို ဘယ်လို ကိုင်တွယ်သလဲ။ {#how-are-notifications-handled}

စနစ်က ticket အသစ်များ၊ အကြောင်းပြန်မှုများ၊ နှင့် အခြေအနေပြောင်းလဲမှုများအတွက် email အကြောင်းကြားချက်များကို အလိုအလျောက် ပေးပို့ပေးပါတယ်။ email template အားလုံးကို setting တွေမှာ စိတ်ကြိုက် ပြုလုပ်နိုင်ပါတယ်။

## ticket များကို သီးသန့် support agent တွေဆီ ခွဲဝေပေးနိုင်ပါသလား။ {#can-i-assign-tickets-to-specific-support-agents}

ဟုတ်ကဲ့၊ ticket များကို agent များဆီ လက်နဲ့ ခွဲဝေပေးနိုင်သလို၊ ticket များကို အလိုအလျောက် ခွဲဝေပေးနိုင်ရန် auto-assignment ကိုလည်း ဖွင့်ထားနိုင်ပါတယ်။

## ဝန်ထမ်းတွေအတွက် ကိုယ်ပိုင်မှတ်စုများ ရှိပါသလား။ {#are-there-private-notes-for-staff}

ဟုတ်ကဲ့၊ ဝန်ထမ်းအဖွဲ့ဝင်များသည် customer များ မမြင်နိုင်ဘဲ အခြားဝန်ထမ်းအဖွဲ့ဝင်များသာ မြင်နိုင်သော ကိုယ်ပိုင်အကြောင်းပြန်မှုများကို ထည့်သွင်းနိုင်ပါတယ်။

## mobile အတွက် အဆင်ပြေပါသလား။ {#is-it-mobile-friendly}

လုံးဝရပါတယ်။ interface တစ်ခုလုံးကို mobile-first ချဉ်းကပ်မှုနဲ့ တည်ဆောက်ထားတာဖြစ်ပြီး စက်ပစ္စည်းအားလုံးမှာ လှပစွာ အလုပ်လုပ်ပါတယ်။

## shortcodes တွေနဲ့ ticket စာရင်းနဲ့ ပုံစံတွေကို ပြသလို့ရပါသလား။ {#can-i-display-ticket-lists-and-forms-using-shortcodes}

ဟုတ်ကဲ့! ရရှိနိုင်တဲ့ shortcodes များမှာ-
* `[wu_support_tickets]` - customer ticket စာရင်းကို ပြသရန်
* `[wu_support_ticket_form]` - ticket အသစ် တင်သွင်းမှု ပုံစံကို ပြသရန်
* `[wu_support_ticket_view ticket_id="123"]` - သီးသန့် ticket တစ်ခုကို ပြသရန်

## ဘယ်လို ဖိုင်အမျိုးအစားတွေကို တင်လို့ရပါသလဲ။ {#what-file-types-can-be-uploaded}

ခွင့်ပြုထားတဲ့ ဖိုင်အမျိုးအစားတွေကို setting တွေမှာ စီမံနိုင်ပါတယ်။ အန္တရာယ်ရှိတဲ့ ဖိုင်တွေကို ကာကွယ်ဖို့ အဆင့်မြင့် လုံခြုံရေး စစ်ဆေးမှုတွေ ပါဝင်ပါတယ်။

## ticket အချက်အလက်တွေကို export လုပ်လို့ရပါသလား။ {#can-i-export-ticket-data}

ဟုတ်ကဲ့၊ Addon မှာ ticket အချက်အလက်နဲ့ စာရင်းအချက်အလက်တွေကို export လုပ်နိုင်တဲ့ စွမ်းရည်တွေ ပါဝင်ပါတယ်။
