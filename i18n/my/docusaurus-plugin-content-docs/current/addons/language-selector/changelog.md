# ဘာသာစကား ရွေးချယ်စက် ပြောင်းလဲမှု မှတ်တမ်း {#language-selector-changelog}

Version: 1.1.3 - Released on 2026-05-05
* ပြင်ဆင်မှု (Fix): Input variables များကို ယခုအခါ သန့်စင်ပေးထားပြီး၊ third-party plugins များနှင့် ပဋိပက္ခမဖြစ်စေရန် custom hook names များကို ရှေ့တွင် တွဲထားပေးလိုက်ပါသည်။

* ပိုမိုကောင်းမွန်စေမှု (Improved): တွဲဖက်မှု အဆင့်မြှင့်တင်မှုများနှင့် code အရည်အသွေး မြှင့်တင်မှုများ ပြုလုပ်ထားပါသည်။

Version: 1.1.0 - Released on 2025-09-28
* Prefix အမည်ကို ultimate-multisite အဖြစ် ပြောင်းလဲခြင်း၊ text domain ကို အဆင့်မြှင့်ခြင်းနှင့် version ကို အဆင့်မြှင့်ခြင်းများ ပြုလုပ်ထားပါသည်။
