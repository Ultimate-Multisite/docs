# Filter: wu_apc_get_meta_content_default_content

Developer တွေအနေနဲ့ default value တွေကို filter လုပ်နိုင်အောင် လုပ်ပေးတာပါ။

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $default_value | `string` | default value ကို ဖော်ပြပါတယ်။ |
| $meta_type | `string` | ရယူမယ့် meta data ရဲ့ အမျိုးအစားကို ဖော်ပြပါတယ်။ |
| $meta_key | `string` | meta field ရဲ့ နာမည်ကို ဖော်ပြပါတယ်။ |

### Since

- 1.4.0
### Source

[inc/wu-apc-functions.php](https://github.com/Ultimate-Multisite/ultimate-multisite-admin-page-creator/blob/main/inc/wu-apc-functions.php#L68) မှာ ၆၈ လိုင်းမှာ သတ်မှတ်ထားပါတယ်။

## Returns
အသစ်ဖြစ်တဲ့ default value ကို ပြန်ပေးပါတယ်။
