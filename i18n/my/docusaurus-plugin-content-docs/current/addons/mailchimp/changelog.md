# Mailchimp ပြောင်းလဲမှုမှတ်တမ်း {#mailchimp-changelog}

ဗားရှင်း- ၁.၁.၁ - ၂၀၂၅-၀၉-၂၈ တွင် ထုတ်ဝေခဲ့သည်

* prefix အမည်ရှေ့ဆက်ကို ultimate-multisite အဖြစ် ပြောင်းလဲခဲ့သည်၊ စာသားနယ်ပယ် (text domain) ကို အပ်ဒိတ်လုပ်ခဲ့သည်၊ ဗားရှင်းနံပါတ်ကို တိုးမြှင့်ခဲ့သည်။

ဗားရှင်း ၁.၁.၀ - ၂၀၁၈/၀၃/၀၅
- ပြင်ဆင်ခဲ့သည်- အသေးစားပြဿနာများနှင့် အမှားများစွာကို ပြင်ဆင်ခဲ့သည်။
- ထည့်သွင်းခဲ့သည်- MailChimp အုပ်စုများမှတစ်ဆင့် စာရင်းတစ်ခုတည်းနှင့် segment တစ်ခုကို အသုံးပြုနိုင်သည့် စွမ်းဆောင်နိုင်မှု။
