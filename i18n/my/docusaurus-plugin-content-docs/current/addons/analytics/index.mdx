# အချက်အလက် စစ်ဆေးခြင်း (Analytics) {#analytics}

ပရော်ဖက်ရှင်နယ် SaaS အချက်အလက် စစ်ဆေးမှု Dashboard တစ်ခုဖြစ်ပြီး၊ MRR၊ ဖောက်သည်များ ထွက်ခွာနှုန်း (churn)၊ ဖောက်သည် ကျန်းမာရေး အခြေအနေနှင့် Funnel Tracking တို့ကို အမှန်တကယ် အချက်အလက်များဖြင့် စောင့်ကြည့်နိုင်ပါသည်။
