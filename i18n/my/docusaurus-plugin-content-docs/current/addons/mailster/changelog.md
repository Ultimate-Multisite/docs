# Mailster ပြောင်းလဲမှု မှတ်တမ်း

### 1.0.0
* ပထမဆုံး ထုတ်ဝေမှု
* စာရင်းသွင်းခြင်း သို့မဟုတ် ငွေပေးချေမှုပြုလုပ်သည့်အခါ အလိုအလျောက် စာရင်းသွင်းသူများ စုစည်းမှု (sync)
* ထုတ်ကုန်အလိုက် စာရင်းခွဲများ သတ်မှတ်နိုင်ခြင်း
* GDPR လိုက်နာမှုအတွက် Checkbox အခြေခံ opt-in စနစ်
* Double opt-in ပံ့ပိုးမှု
* အကျယ်ပြန့်ဆုံး field mapping စနစ်
* အသေးစိတ် မှတ်တမ်းတင်ခြင်း (Detailed logging)
