# Mailster {#mailster}

Ultimate Multisite Mailster Integration addon ကို အသုံးပြုပြီး သင့်ရဲ့ မัลတီဆိုရ် နက်ဝက်ကို စွမ်းအားပြည့် အီးမေးလ်စာရင်းတည်ဆောက်ရေးစက်ကြီးတစ်ခုအဖြစ် ပြောင်းလဲလိုက်ပါ။ အသစ်စာရင်းသွင်းသူများ သို့မဟုတ် ဝယ်ယူသူများ ပေါ်လာတဲ့အခါ သူတို့ကို Mailster အီးမေးလ်စာရင်းတွေထဲကို အလိုအလျောက်ထည့်ပေးမှာဖြစ်လို့ သင့်ရဲ့ အဖွဲ့ဝင်မှုပလက်ဖောင်းနဲ့ အီးမေးလ်စျေးကွက်ရှာဖွေရေးဗျူဟာတွေကြား အပြစ်အနာအပျက်မရှိ ချိတ်ဆက်မှုတစ်ခုကို ဖန်တီးပေးပါလိမ့်မယ်။

## အဓိကအင်္ဂါရပ်များ (Key Features) {#key-features}

**အလိုအလျောက် စာရင်းသွင်းသူများ ပေါင်းစပ်ခြင်း (Automatic Subscriber Sync)**
* စာရင်းသွင်းတဲ့အခါ ဝယ်သူတွေကို Mailster စာရင်းတွေထဲကို အလိုအလျောက်ထည့်ပေးခြင်း
* စာရင်းသွင်းချိန်မှာ ချက်ချင်းပေါင်းစပ်မလား၊ ဒါမှမဟုတ် ငွေပေးချေမှု အတည်ပြုတဲ့အထိ စောင့်ဆိုင်းမလား ရွေးချယ်နိုင်ခြင်း
* ကိုယ်တိုင်လုပ်ဆောင်စရာမလိုဘဲ အီးမေးလ်စာရင်းတွေကို အမြဲတမ်း အသစ်အဆန်းနဲ့ ခေတ်မီနေစေခြင်း

**ပြောင်းလွယ်ပြင်လွယ်ရှိသော စာရင်းစီမံခန့်ခွဲမှု (Flexible List Management)**
* ဝယ်သူအသစ်အားလုံးအတွက် ကမ္ဘာလုံးဆိုင်ရာ (global) စံစာရင်းများကို သတ်မှတ်နိုင်ခြင်း
* ပစ်မှတ်ထားတဲ့ စည်းရုံးလှုပ်ရှားမှုတွေအတွက် ထုတ်ကုန်အလိုက် စံစာရင်းများကို ပြောင်းလဲသတ်မှတ်နိုင်ခြင်း
* Mailster ရဲ့ ရင်းနှီးကျွမ်းဝင်တဲ့ interface ကို အသုံးပြုပြီး စာရင်းခွဲဝေမှုတွေကို အလွယ်တကူ စီမံခန့်ခွဲနိုင်ခြင်း

**GDPR စံနှုန်းများနှင့် ကိုက်ညီသော စာရင်းသွင်းခွင့်ရယူမှု ရွေးချယ်စရာများ (GDPR-Compliant Opt-In Options)**
* အလိုအလျောက် စာရင်းသွင်းခြင်း (automatic subscription) ဒါမှမဟုတ် Checkbox အခြေခံ သဘောတူညီမှုရယူခြင်း (checkbox-based consent) ကို ရွေးချယ်နိုင်ခြင်း
* အတည်ပြုထားတဲ့ စာရင်းသွင်းမှုတွေအတွက် Double Opt-in ကို ဖွင့်ပေးနိုင်ခြင်း
* သင့်ရဲ့ စံနှုန်းလိုက်နာမှု လိုအပ်ချက်တွေပေါ်မူတည်ပြီး Opt-in Checkbox ကို ပြသ/ဖျောက်ထားနိုင်ခြင်း

**ဉာဏ်ရည်ထက်မြက်သော အချက်အလက် ပေါင်းစပ်ခြင်း (Smart Data Mapping)**
* ဝယ်သူနာမည်နဲ့ ဆက်သွယ်ရန် အချက်အလက်တွေကို အလိုအလျောက် ပေါင်းစပ်ပေးခြင်း
* ငွေတောင်းခံလိပ်စာတွေကို စာရင်းသွင်းသူ (subscriber) field တွေနဲ့ ပေါင်းစပ်ပေးခြင်း
* ပစ်မှတ်ထားတဲ့ စည်းရုံးလှုပ်ရှားမှုတွေအတွက် စာရင်းသွင်းတဲ့ရက်တွေကို မှတ်တမ်းတင်ပေးခြင်း

**ထုတ်ကုန်အဆင့် ထိန်းချုပ်မှု (Product-Level Control)**
* ထုတ်ကုန်အသီးသီးအတွက် စာရင်းအမျိုးမျိုး ခွဲဝေပေးနိုင်ခြင်း
* ဝယ်သူတွေ ဘာဝယ်လဲဆိုတာအပေါ် အခြေခံပြီး ပစ်မှတ်ထားတဲ့ စည်းရုံးလှုပ်ရှားမှုတွေ ဖန်တီးနိုင်ခြင်း
* လိုအပ်တဲ့အခါမှာ Mailster ကို သီးသန့်ထုတ်ကုန်တွေအတွက် ပိတ်ထားနိုင်ခြင်း

**ကျွမ်းကျင်ပရော်ဖက်ရှင်နယ် အတွေ့အကြုံ (Professional Experience)**
* စာရင်းရွေးချယ်မှုအတွက် Native Mailster UI ကို ပေါင်းစပ်ထားခြင်း
* ပြဿနာဖြေရှင်းခြင်းနဲ့ အတည်ပြုစစ်ဆေးမှုအတွက် အသေးစိတ် မှတ်တမ်းတင်ထားခြင်း (Detailed logging)
* ရှိပြီးသား စာရင်းသွင်းသူတွေကို အပြစ်အနာအပျက်မရှိ အဆင့်မြှင့်တင်ပေးခြင်း

## ဘယ်သူတွေအတွက် အသင့်တော်ဆုံးလဲ (Perfect For) {#perfect-for}

* အီးမေးလ်စာရင်းတွေ တည်ဆောက်နေတဲ့ အဖွဲ့ဝင်မှုဆိုဒ်ပိုင်ရှင်များ
* ဝယ်သူအသစ်တွေကို ဂရုစိုက်မြှင့်တင်လိုတဲ့ SaaS ပလက်ဖောင်းများ
* ကျောင်းသားတွေကို စာရင်းသွင်းမှုအပေါ် အခြေခံပြီး ခွဲခြားလိုတဲ့ သင်တန်းဖန်တီးသူများ
* Ultimate Multisite ကို Mailster နဲ့ အသုံးပြုနေတဲ့ မည်သူမဆို

## ဘယ်လိုအလုပ်လုပ်သလဲ (How It Works) {#how-it-works}

1. သင့်ရဲ့ Ultimate Multisite network ပေါ်မှာ addon ကို ထည့်သွင်းပြီး အသက်သွင်းပါ။
2. Ultimate Multisite settings ထဲမှာ ကမ္ဘာလုံးဆိုင်ရာ စံစာရင်းတွေကို စီစဉ်ပေးပါ။
3. လိုအပ်ရင် ထုတ်ကုန်အလိုက် စံစာရင်းတွေနဲ့ အစားထိုးသတ်မှတ်ပါ။
4. သင့်ရဲ့ Opt-in mode နဲ့ စာရင်းသွင်းချိန်ကို ရွေးချယ်ပါ။
5. သင့်ရဲ့ Mailster စာရင်းတွေ အလိုအလျောက် ကြီးထွားလာတာကို ကြည့်လိုက်ရုံပါပဲ!

## ထည့်သွင်းခြင်း (Installation) {#installation}

1. plugin ဖိုင်တွေကို သင့်ရဲ့ Ultimate Multisite addons directory ထဲကို တင်ပါ။
2. Mailster ကို သင့်ရဲ့ main site ပေါ်မှာ ထည့်သွင်းပြီး အသက်သွင်းထားကြောင်း သေချာပါစေ။
3. Ultimate Multisite addons စာမျက်နှာကနေ addon ကို အသက်သွင်းပါ။
4. စီစဉ်ဖွဲ့စည်းဖို့အတွက် Settings > Mailster Integration ကို သွားပါ။
5. သင့်ရဲ့ စံစာရင်းတွေကို ရွေးချယ်ပြီး Opt-in ဦးစားပေးမှုတွေကို စီစဉ်ပေးပါ။

## မကြာခဏမေးလေ့ရှိသော မေးခွန်းများ (Frequently Asked Questions) {#frequently-asked-questions}

## ဒီ addon က Ultimate Multisite မရှိဘဲ အလုပ်လုပ်ပါသလား။ {#does-this-addon-work-without-ultimate-multisite}

မရပါဘူး။ ဒီ addon ဟာ Ultimate Multisite ကို အသုံးပြုဖို့ လိုအပ်ပါတယ်။ Mailster ကို Ultimate Multisite ရဲ့ ဝယ်သူနဲ့ ထုတ်ကုန်စနစ်နဲ့ ပေါင်းစပ်ဖို့ အထူးဒီဇိုင်းထုတ်ထားတာ ဖြစ်ပါတယ်။

## Site အားလုံးမှာ Mailster ထည့်သွင်းထားဖို့ လိုပါသလား။ {#do-i-need-mailster-installed-on-all-sites}

မလိုပါဘူး။ Mailster ကို သင့်ရဲ့ main network site ပေါ်မှာသာ ထည့်သွင်းပြီး အသက်သွင်းထားဖို့ လိုပါတယ်။ ဒီ addon က သားမြစ်ဆိုဒ် (subsites) အားလုံးကနေ အလိုအလျောက် ချိတ်ဆက်ပေးမှာပါ။

## ထုတ်ကုန်အမျိုးမျိုးအတွက် စာရင်းအမျိုးမျိုး ခွဲဝေပေးလို့ရပါသလား။ {#can-i-assign-different-lists-to-different-products}

ရပါတယ်! ကမ္ဘာလုံးဆိုင်ရာ စံစာရင်းတွေကို သတ်မှတ်ပြီး၊ ထုတ်ကုန်တစ်ခုချင်းစီအပေါ်မှာ အစားထိုးသတ်မှတ်နိုင်ပါတယ်။ ဒါက ဝယ်သူတွေ ဘာဝယ်လဲဆိုတာအပေါ် အခြေခံပြီး ပစ်မှတ်ထားတဲ့ အီးမေးလ်စည်းရုံးလှုပ်ရှားမှုတွေ ဖန်တီးနိုင်စေပါတယ်။

## ဝယ်သူတစ်ယောက်က Mailster မှာ ရှိပြီးသားဆိုရင် ဘာဖြစ်မလဲ။ {#what-happens-if-a-customer-already-exists-in-mailster}

ပုံမှန်အားဖြင့်၊ ဒီ addon က ရှိပြီးသား စာရင်းသွင်းသူတွေကို အဆင့်မြှင့်တင်ပေးပြီး စာရင်းအသစ်တွေထဲကို ထပ်ထည့်ပေးမှာပါ။ ရှိပြီးသား စာရင်းသွင်းသူတွေကို ကျော်လွန်ချင်တယ်ဆိုရင် settings ထဲမှာ ဒါကို ပိတ်ထားနိုင်ပါတယ်။

## ဒါက GDPR နဲ့ ကိုက်ညီပါသလား။ {#is-this-gdpr-compliant}

ဟုတ်ပါတယ်! ဒီ addon မှာ Checkbox အခြေခံ Opt-in mode ပါဝင်ပြီး ဝယ်သူတွေက အီးမေးလ်တွေလက်ခံဖို့ ရှင်းရှင်းလင်းလင်း သဘောတူညီမှုပေးရပါတယ်။ ဒါဟာ သင့်ရဲ့ သင့်လျော်တဲ့ ကိုယ်ရေးကိုယ်တာမူဝါဒတွေနဲ့ ပေါင်းစပ်လိုက်တဲ့အခါ GDPR လိုအပ်ချက်တွေကို ဖြည့်ဆည်းပေးပါတယ်။

## Double Opt-in ကို အသုံးပြုလို့ရပါသလား။ {#can-i-use-double-opt-in}

လုံးဝရပါတယ်! Double opt-in setting ကို ဖွင့်ပေးလိုက်ရုံနဲ့ Mailster က စာရင်းသွင်းမှုတွေကို အသက်မသွင်းခင် အတည်ပြုတဲ့ အီးမေးလ်တွေ ပို့ပေးမှာ ဖြစ်ပါတယ်။
