# AffiliateWP

Multisite Ultimate: AffiliateWP Integration

သင့်ရဲ့ Ultimo Network ရဲ့ client အခြေခံကို ချဲ့ထွင်ဖို့အတွက် စွမ်းအားပြည့် AffiliateWP ကို အသုံးပြုလိုက်ပါ။

## Installation {#installation}

၁။ 'ultimate-multisite-git ' ကို '/wp-content/plugins/' directory ထဲကို upload လုပ်ပါ။
၂။ WordPress ရဲ့ 'Plugins' menu ကနေ plugin ကို activate လုပ်ပါ။
၃။ plugin ကို စနစ်တကျ တည်ဆောက်နိုင်ဖို့အတွက် အဆင့်ဆင့် လမ်းညွှန် (Wizard) ကို လိုက်နာပါ။

## Usage Guide {#usage-guide}

[AffiliateWP](https://affiliatewp.com/) ဟာ သင့်ရဲ့ online လုပ်ငန်းအတွက် affiliate program တစ်ခု တည်ဆောက်တဲ့အခါ အဓိက အသုံးပြုရတဲ့ ဖြေရှင်းချက်တစ်ခု ဖြစ်လာခဲ့ပါပြီ။ Ultimate Multisite: AffiliateWP ကတော့ သင့်ရဲ့ premium network အတွက်လည်း အလားတူအရာတွေကို တည်ဆောက်နိုင်စေမယ့် တံတားတစ်ခုပါပဲ။

## Getting Started {#getting-started}

ပထမဆုံးအနေနဲ့၊ သင်ဟာ [**AffiliateWP**](https://affiliatewp.com/) ကို **active** လုပ်ထားဖို့ **လိုအပ်ပါတယ်**! သင်ဟာ **ပေါင်းစပ်ထားတဲ့ plugin** မရှိဘူးဆိုရင် ကျွန်တော်တို့ရဲ့ add-on ကို အသုံးပြုနိုင်မှာ မဟုတ်ပါဘူး။ သင့်ရဲ့ client တွေက သူတို့ရဲ့ site တွေမှာလည်း အသုံးပြုခွင့် ပေးဖို့ စီစဉ်ထားတာ မဟုတ်ဘူးဆိုရင်လည်း AffiliateWP ကို သင့်ရဲ့ main site ပေါ်မှာပဲ active လုပ်ထားသင့်ပါတယ်။

ဒါ့အပြင်၊ **Ultimate Multisite** နဲ့ **Ultimate Multisite: AffiliateWP** နှစ်ခုလုံးကို **network တစ်ခုလုံးမှာ** **activate** လုပ်ထားဖို့လည်း လိုအပ်ပါတယ်။

## Adding Ultimate Multisite as an AffiliateWP Integration {#adding-ultimate-multisite-as-an-affiliatewp-integration}

သင့်ရဲ့ main site မှာ Affiliates Settings ကို သွားပြီး၊ Integrations tab မှာ Ultimate Multisite ကို အမှန်ခြစ်ပေးပါ။ ဒါက [AffiliateWP](https://affiliatewp.com/) ကို Ultimate Multisite sign-up flow ကနေ လာတဲ့ ဝင်ရောက်မှုတွေနဲ့ conversion တွေကို စတင် track လုပ်ဖို့ ပြောပြတာပါပဲ။

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_အောက်ဆုံးက Ultimate Multisite box ကို အမှန်ခြစ်ပြီး save လုပ်လိုက်ပါ_

ဒါဆိုရင် လိုအပ်တဲ့ အရာတွေ အားလုံးနီးပါး ပြီးသွားပါပြီ။ referral link တွေကနေ လာတဲ့ ဝင်ရောက်မှုတွေကို track လုပ်ပေးမှာဖြစ်ပြီး၊ အဲဒီ user က နောက်ပိုင်းမှာ ပထမဆုံး ငွေပေးချေမှု လုပ်တဲ့အခါ conversion တစ်ခုကို ဖန်တီးပေးမှာဖြစ်ပြီး၊ အဲဒီ conversion အတွက် တာဝန်ယူတဲ့ affiliate ဆီကို ကော်မရှင်တစ်ခုကို ခွဲပေးထားမှာ ဖြစ်ပါတယ်။

ဒါပေမဲ့၊ အသစ်ဖန်တီးထားတဲ့ user ဆီက ရရှိတဲ့ ပထမဆုံး ငွေပေးချေမှုကိုပဲ track လုပ်ပေးမှာသာ ဖြစ်ပါတယ်။ ပုံမှန် referral ငွေပေးချေမှုတွေ ရရှိစေဖို့အတွက် [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) လိုအပ်မှာ ဖြစ်ပါတယ်။

## Enabling Recurring Referrals {#enabling-recurring-referrals}

[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) ကို install လုပ်ပြီးတဲ့နောက်မှာ Affiliates Settings ရဲ့ Recurring Referrals tab ကို သွားပြီး၊ recurring referrals တွေကို အသုံးပြုဖို့ enable လုပ်ပေးပါ။

<!-- Screenshot unavailable: AffiliateWP Recurring Referrals settings with Enable Recurring Referrals toggle -->

_အလွန်ရိုးရှင်းပါတယ်- Recurring Referrals ကို Enable လုပ်လိုက်ရုံပါပဲ!_

ထပ်ပြောရရင်၊ ဒါဆိုရင် လိုအပ်တဲ့ အရာတွေ အားလုံး ပြီးသွားပါပြီ!

အခုဆိုရင်၊ referral လုပ်ထားတဲ့ account တစ်ခုမှာ ငွေပေးချေမှု အသစ်တစ်ခု ရရှိတိုင်း၊ တာဝန်ယူတဲ့ affiliate ဆီကို သူ့ရဲ့ အချိုးကျတဲ့ အစိတ်အပိုင်းကို အလိုအလျောက် ရရှိသွားမှာပါ!
