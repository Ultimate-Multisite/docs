# AffiliateWP ပြောင်းလဲမှုမှတ်တမ်း {#affiliatewp-changelog}

Version: 2.0.1 - 2025-09-28 တွင် ထုတ်ဝေခဲ့သည်

* ရှေ့ဆက်စကားလုံး (prefix) ကို ultimate-multisite အဖြစ် ပြောင်းလဲခဲ့သည်၊ စာသားနယ်ပယ် (text domain) ကို အဆင့်မြှင့်ခဲ့သည်၊ နှင့် version နံပါတ်ကို တိုးမြှင့်ခဲ့သည်။

Version: 2.0.0 - 2022-08-05 တွင် ထုတ်ဝေခဲ့သည်

* ultimo payments ကို အခြေခံ၍ အလုပ်လုပ်နိုင်ရန် plugin ကို ပြန်လည်တည်ဆောက်ခဲ့သည်။

Version: 2.0.0-beta.2 - 2022-01-21 တွင် ထုတ်ဝေခဲ့သည်

* အတွင်းပိုင်း: hooks နှင့် filters ထုတ်ပေးစက် (generator) ကို ထည့်သွင်းခဲ့သည်။
* အတွင်းပိုင်း: developer များ အသုံးပြုရလွယ်ကူစေရန် Multisite Ultimate stubs များကို ထည့်သွင်းခဲ့သည်။
* ပြင်ဆင်ခဲ့သည်: ပစ္စည်းတွန်းလှည်း (cart) တစ်ခုတွင် discount code မရှိသည့်အခါ ဖြစ်ပေါ်ခဲ့သော အလွန်ပြင်းထန်သည့် အမှား (Fatal error) ကို ပြင်ဆင်ခဲ့သည်။
* ပြင်ဆင်ခဲ့သည်: ကြည့်ရှုမှု (Visits) များကို မှန်ကန်စွာ ပြောင်းလဲမှု မလုပ်နိုင်သည့် ပြဿနာကို ပြင်ဆင်ခဲ့သည်။

Version 2.0.0-beta.1 - 2021-12-29 တွင် ထုတ်ဝေခဲ့သည်

* ၂.၀ အခြေခံ ဗားရှင်းကို ပထမဆုံးအကြိမ် ထုတ်ဝေခဲ့သည်။
