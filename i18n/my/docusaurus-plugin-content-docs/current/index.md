# Ultimate Multisite Documentation {#ultimate-multisite-documentation}

Ultimate Multisite documentation ကို ကြိုဆိုပါတယ်။ Ultimate Multisite က WordPress Multisite ကို အပြည့်အဝ Website as a Service (WaaS) ပလက်ဖောင်းအဖြစ် ပြောင်းလဲပေးနိုင်ပါတယ်။

## Documentation Sections {#documentation-sections}

### [User Guide](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

သင့်ရဲ့ WaaS platform ကို တည်ဆောက်ရန်၊ စနစ်ထည့်သွင်းရန်နှင့် စီမံခန့်ခွဲရန် လိုအပ်တဲ့ အရာအားလုံးကို ဖော်ပြထားပါတယ်။ Installation၊ payment gateways များ၊ domain mapping၊ client management စသည်တို့ကို အကျုံးဝင်ပါတယ်။

### [Developer Documentation](/developer/hooks) {#developer-documentation}

Ultimate Multisite ပေါ်မှာ တည်ဆောက်မယ့် developer တွေအတွက် နည်းပညာဆိုင်ရာ အကိုးအကားစာအုပ် ဖြစ်ပါတယ်။ hooks reference၊ REST API documentation နဲ့ addon development guide တွေ ပါဝင်ပါတယ်။

### [Addons](/addons) {#addons}

Ultimate Multisite ကို WooCommerce integration၊ analytics၊ domain selling စတဲ့ အပိုဆောင်း feature တွေနဲ့ ချဲ့ထွင်ပေးတဲ့ အခကြေးငွေပေးရတဲ့ addon plugin တွေအတွက် documentation ဖြစ်ပါတယ်။

## Getting Help {#getting-help}

- [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) -- မေးခွန်းများ မေးမြန်းပြီး အကြံဉာဏ်များ မျှဝေနိုင်ပါတယ်
- [GitHub Issues](https://github.com/Ultimate-Multisite/ultimate-multisite/issues) -- အမှားအယွင်းများကို တင်ပြနိုင်ပါတယ်
- [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) -- core plugin ကို ဒေါင်းလုဒ်လုပ်ရန်
