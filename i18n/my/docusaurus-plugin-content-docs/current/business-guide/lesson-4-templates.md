# သင်ခန်းစာ ၄- အထူးပြု Template များ တည်ဆောက်ခြင်း

Template များသည် သင်၏ အထူးပြု တန်ဖိုးကို ဖော်ပြသည့် အဓိက အစိတ်အပိုင်းများ ဖြစ်ပါသည်။ စက်ဘက်ရုံပိုင်ရှင်တစ်ဦးသည် စာရင်းသွင်းပြီးသည်နှင့် မိမိ၏ စက်ဘက်ရုံအကြောင်း အချိန်ဇယားများ၊ လေ့ကျင့်ရေးဆရာများ၏ ပရိုဖိုင်များနှင့် သင့်လျော်သော ပုံရိပ်များ ပါဝင်သည့် စက်ဘက်ရုံဝဘ်ဆိုဒ်ကဲ့သို့ မြင်ရပါက၊ ဘာမှမရှိသော စာမျက်နှာဖြူကို မြင်ရသူထက် နေထိုင်ရန် ပိုမိုစိတ်အားထက်သန်ပါသည်။

## ကျွန်ုပ်တို့ ရပ်ခဲ့သည့်နေရာ {#where-we-left-off}

Ultimate Multisite ကို ထည့်သွင်းပြီး စနစ်တကျ ဖွဲ့စည်းထားသည့် FitSite network တစ်ခုကို ကျွန်ုပ်တို့ ရရှိထားပြီး ဖြစ်ပါသည်။ ယခုအခါ FitSite ကို ကျန်းမာရေးနှင့် အားကစားလုပ်ငန်းများအတွက် အထူးထုတ်လုပ်ထားသည်ဟု ခံစားရစေမည့် Template များကို တည်ဆောက်သွားရမှာ ဖြစ်ပါတယ်။

## အထူးပြု Template များ ဘာကြောင့် အရေးပါသလဲ {#why-niche-templates-matter}

ယေဘူယ Template များသည် သင့်ဖောက်သည်များအား ခက်ခဲသည့် အလုပ်ကို လုပ်ဆောင်စေရသည်- မည်သည့်စာမျက်နှာများ လိုအပ်သည်၊ မည်သည့်အကြောင်းအရာဖွဲ့စည်းပုံက အဆင်ပြေသည်၊ နှင့် မည်သို့ပုံစံဖြစ်အောင် လုပ်ရမည်ကို ကိုယ်တိုင်ရှာဖွေရခြင်း ဖြစ်သည်။ အထူးပြု Template များက ထိုအခက်အခဲကို ပပျောက်စေပါသည်။

FitSite သို့ စာရင်းသွင်းသည့် စက်ဘက်ရုံပိုင်ရှင်တစ်ဦးသည် အောက်ပါအရာများကို မြင်တွေ့ရသင့်သည်-

- စက်ဘက်ရုံဝဘ်ဆိုဒ်ကဲ့သို့ မြင်ရသည့် အဓိကစာမျက်နှာ (Homepage)
- အတန်းများ၊ အချိန်ဇယားများ၊ လေ့ကျင့်ရေးဆရာများနှင့် ဈေးနှုန်းများအတွက် စာမျက်နှာများ ကြိုတင်ဖန်တီးပေးထားခြင်း
- ၎င်းတို့၏ လုပ်ငန်းနှင့် ကိုက်ညီသော ပုံရိပ်များနှင့် နေရာယူထားသည့် အကြောင်းအရာများ (Placeholder content)
- ကျန်းမာရေးနှင့် အားကစားလုပ်ငန်းအတွက် ပရော်ဖက်ရှင်နယ်ကျပြီး အမှတ်တံဆိပ်နှင့် ကိုက်ညီသော ဒီဇိုင်း

သူတို့သည် ၎င်းတို့၏ အသေးစိတ်အချက်အလက်များကို ဖြည့်စွက်ရုံသာ ဖြစ်သည်။ အစမှ ပြန်စရခြင်း မရှိပါ။

## Template များ စီစဉ်ခြင်း {#planning-your-templates}

တည်ဆောက်ခြင်းမပြုမီ၊ မည်သည့် Template များကို ပေးအပ်မည်ကို ဆုံးဖြတ်ပါ။ FitSite အတွက် ကျွန်ုပ်တို့သည် Template သုံးခုကို ဖန်တီးသွားမည်ဖြစ်သည်-

### Template ၁- စက်ဘက်ရုံ အခြေခံ (Studio Essential) {#template-1-studio-essential}

သေးငယ်သော စက်ဘက်ရုံများနှင့် ကိုယ်ပိုင်လေ့ကျင့်ရေးဆရာများအတွက်။

- ပုံရိပ်အကြီး၊ အတန်းအကျဉ်းချုပ်များနှင့် လုပ်ဆောင်ရန် ခေါ်ဆိုမှု (Call to action) ပါဝင်သည့် **အဓိကစာမျက်နှာ (Homepage)**
- စက်ဘက်ရုံ၏ ဇာတ်လမ်းနှင့် ရည်ရွယ်ချက်ပါဝင်သည့် **အကြောင်း (About)** စာမျက်နှာ
- အချိန်ဇယားပုံစံပါဝင်သည့် **အတန်းများ (Classes)** စာမျက်နှာ
- ပရိုဖိုင်ကတ်များပါဝင်သည့် **လေ့ကျင့်ရေးဆရာများ (Trainers)** စာမျက်နှာ
- တည်နေရာမြေပုံနှင့် ဖောင်ပါဝင်သည့် **ဆက်သွယ်ရန် (Contact)** စာမျက်နှာ
- ကျန်းမာရေးနှင့် အားကစားနှင့် သင့်လျော်သော အရောင်အသွေးစနစ်ပါရှိသည့် **ရှင်းလင်းပြီး ခေတ်မီသော ဒီဇိုင်း**

### Template ၂- အားကစားရုံ ပရို (Gym Pro) {#template-2-gym-pro}

ဝန်ဆောင်မှုများ ပိုမိုများပြားသည့် အထင်ကရ အားကစားရုံများအတွက်။

- Studio Essential တွင်ပါရှိသည့် အရာအားလုံး၊ ထို့အပြင်-
- ဈေးနှုန်းဇယားပါဝင်သည့် **အဖွဲ့ဝင်အဖြစ် စာမျက်နှာ (Membership)**
- အဆောက်အအုံဓာတ်ပုံများအတွက် **ပြခန်း (Gallery)** စာမျက်နှာ
- ကျန်းမာရေးအကြံပြုချက်များနှင့် သတင်းများအတွက် **ဘလော့ဂ် (Blog)** အပိုင်း
- အဓိကစာမျက်နှာပေါ်ရှိ **အထောက်အကရံများ (Testimonials)** အပိုင်း
- **ပိုမိုထင်ရှားသော အမှတ်တံဆိပ် (Branding)** ရွေးချယ်မှုများ

### Template ၃- ကျန်းမာရေးကွင်းဆက် (Fitness Chain) {#template-3-fitness-chain}

နေရာများစွာရှိသော လုပ်ငန်းများအတွက်။

- Gym Pro တွင်ပါရှိသည့် အရာအားလုံး၊ ထို့အပြင်-
- နေရာများစွာစာရင်းပါဝင်သည့် **တည်နေရာများ (Locations)** စာမျက်နှာ
- **ခွဲရုံး/တည်နေရာ** အတွက် အထူးစာမျက်နှာ Template
- နေရာအလိုက် အသေးစိတ်အချက်အလက်ပါဝင်သည့် **ဗဟိုချုပ်ကိုင်မှုရှိသော အမှတ်တံဆိပ် (Centralized branding)**
- နေရာအနှံ့ရှိ **ဝန်ထမ်းစာရင်း (Staff directory)**

## Template ဝဘ်ဆိုဒ်တစ်ခု တည်ဆောက်ခြင်း {#building-a-template-site}

Ultimate Multisite တွင် Template ဆိုသည်မှာ ဖောက်သည်အသစ်များ၏ ဝဘ်ဆိုဒ်များ မည်သို့ပုံစံဖြစ်စေလိုသည်ကို စနစ်တကျ ဖွဲ့စည်းထားသည့် WordPress ဝဘ်ဆိုဒ်တစ်ခုသာ ဖြစ်သည်။ ၎င်းကို မည်သို့ဖန်တီးရမည်ကို အောက်ပါအတိုင်း ဖော်ပြထားသည်-

### အဆင့် ၁- Template ဝဘ်ဆိုဒ်ကို ဖန်တီးခြင်း {#step-1-create-the-template-site}

၁။ သင့် network admin တွင် **Sites > Add New** သို့ သွားပါ။
၂။ `template-studio-essential` ဟူ၍ နာမည်ပေးကာ ဝဘ်ဆိုဒ်တစ်ခု ဖန်တီးပါ။
၃။ ဤဝဘ်ဆိုဒ်သည် သင်၏ အလုပ်လုပ်မည့် ပုံစံဘောင် (canvas) ဖြစ်လာမည်။

### အဆင့် ၂- Theme ကို ထည့်သွင်းပြီး ဖွဲ့စည်းခြင်း {#step-2-install-and-configure-the-theme}

Template ဝဘ်ဆိုဒ်၏ dashboard သို့ ပြောင်းပြီး-

၁။ ကျန်းမာရေးနှင့် အားကစားလုပ်ငန်းများအတွက် သင့်လျော်သော Theme တစ်ခုကို အသက်သွင်းပါ။
၂။ Theme ၏ ဆက်တင်များ၊ အရောင်များနှင့် စာလုံးပုံစံများကို ဖွဲ့စည်းပါ။
၃။ ကျန်းမာရေးနှင့် အားကစားနှင့် သင့်လျော်သော လမ်းကြောင်းများပါရှိသည့် Header နှင့် Footer များကို စနစ်တကျ တည်ဆောက်ပါ။

:::tip Theme ရွေးချယ်မှု
ကျန်းမာရေးနှင့် အားကစားအတွက် ကောင်းမွန်စွာ မြင်ရနိုင်သော်လည်း၊ သင့်ဖောက်သည်များ မကိုင်တွယ်နိုင်လောက်အောင် ရှုပ်ထွေးမှုမရှိသော Theme ကို ရွေးချယ်ပါ။ Astra၊ GeneratePress သို့မဟုတ် Kadence ကဲ့သို့သော Theme များသည် ပေါ့ပါးပြီး၊ စိတ်ကြိုက်ပြင်ဆင်နိုင်ကာ၊ အထောက်အပံ့ကောင်းများ ပေးထားသောကြောင့် ကောင်းမွန်စွာ အလုပ်လုပ်ပါသည်။
:::

### အဆင့် ၃- စာမျက်နှာများ ဖန်တီးခြင်း {#step-3-create-the-pages}

အောက်ပါအရာများဖြင့် စာမျက်နှာတစ်ခုချင်းစီကို တည်ဆောက်ပါ-

- ကျန်းမာရေးနှင့် အားကစားနှင့် သဘာဝကျစွာ ဖတ်ရသည့် **နေရာယူထားသည့် အကြောင်းအရာ (Placeholder content)** ("Welcome to [Studio Name]" ဟု ရေးသားရမည်၊ "Lorem ipsum" မဟုတ်ပါ)
- ကျန်းမာရေးနှင့် အားကစားလှုပ်ရှားမှုများကို ပြသသည့် အခမဲ့ ဓာတ်ပုံဘဏ်များမှ **နေရာယူထားသည့် ပုံရိပ်များ (Placeholder images)**
- စာမျက်နှာတည်ဆောက်သူ (page builder) သို့မဟုတ် block editor ကို အသုံးပြု၍ **လုပ်ဆောင်နိုင်သော ပုံစံများ (Functional layouts)**

နေရာယူထားသည့် အကြောင်းအရာများကို ဖြစ်နိုင်သမျှ ညွှန်ကြားချက်ပေးသည့် ပုံစံမျိုး ဖြစ်အောင် ရေးသားပါ။ ယေဘူယ filler text များအစား၊ "ဤနေရာကို သင့်စက်ဘက်ရုံအကြောင်း အကျဉ်းချုပ်ဖော်ပြချက်နှင့် မည်သည့်အရာက ထူးခြားစေသည်ကို ရေးထည့်ပါ။ သင်၏ လေ့ကျင့်ရေးအတွေးအခေါ်၊ အတွေ့အကြုံနှစ်များ သို့မဟုတ် ဖောက်သည်များ မည်သည့်အရာကို မျှော်လင့်နိုင်သည်ကို ဖော်ပြပါ။" ကဲ့သို့သော အကြောင်းအရာများကို ရေးသားပါ။

### အဆင့် ၄- Plugins များ ဖွဲ့စည်းခြင်း {#step-4-configure-plugins}

ကျန်းမာရေးစက်ဘက်ရုံများ လိုအပ်သည့် Plugins များကို ထည့်သွင်းပြီး အသက်သွင်းပါ-

- ဘွတ်ကင် (booking) သို့မဟုတ် အချိန်ဇယားဆွဲသည့် plugin (သင့်အစီအစဉ်အဆင့်အတွက် သက်ဆိုင်ပါက)
- ဆက်သွယ်ရန် ဖောင် plugin
- SEO plugin (ကျန်းမာရေးနှင့် အားကစားနှင့် သက်ဆိုင်သော မူရင်းဆက်တင်များဖြင့် ကြိုတင်ဖွဲ့စည်းထားသည်)

### အဆင့် ၅- Template အဖြစ် သတ်မှတ်ခြင်း {#step-5-mark-as-template}

၁။ **Ultimate Multisite > Sites** သို့ သွားပါ။
၂။ Template ဝဘ်ဆိုဒ်ကို တည်းဖြတ်ပါ။
၃။ **Site Template** ခလုတ်ကို ဖွင့်ပါ။

သင်ပေးလိုသည့် Template တစ်ခုချင်းစီအတွက် ဤလုပ်ငန်းစဉ်ကို ထပ်ခါထပ်ခါ လုပ်ဆောင်ပါ။

## Template အရည်အသွေး စစ်ဆေးမှုစာရင်း (Template Quality Checklist) {#template-quality-checklist}

Template တစ်ခုကို အသုံးပြုနိုင်သည်ဟု မဆုံးဖြတ်မီ၊ အောက်ပါအချက်များကို စစ်ဆေးပါ။

- [ ] စာမျက်နှာအားလုံး မှန်ကန်စွာ တင်နေပြီး ပရော်ဖက်ရှင်နယ်ကျမှု ရှိခြင်း
- [ ] နေရာယူထားသည့် အကြောင်းအရာများသည် အထောက်အကူဖြစ်ပြီး အထူးပြုလုပ်ငန်းနှင့် ကိုက်ညီခြင်း
- [ ] ပုံရိပ်များသည် သင့်လျော်ပြီး လိုင်စင်ရရှိမှု မှန်ကန်ခြင်း
- [ ] မိုဘိုင်းဖုန်းတွင် စာမျက်နှာအားလုံး အဆင်ပြေစွာ အလုပ်လုပ်ခြင်း (Mobile responsiveness)
- [ ] လမ်းကြောင်းဖွဲ့စည်းမှုသည် ယုယမှုရှိပြီး ပြည့်စုံခြင်း
- [ ] ဆက်သွယ်ရန် ဖောင်များသည် အလုပ်လုပ်ခြင်း
- [ ] ချိုးကျနေသော link များ သို့မဟုတ် ပျောက်ဆုံးနေသော အစိတ်အပိုင်းများ မရှိခြင်း
- [ ] စာမျက်နှာတင်နှုန်းသည် လက်ခံနိုင်ဖွယ်ရှိခြင်း

## ယခုသင်ခန်းစာတွင် ကျွန်ုပ်တို့ တည်ဆောက်ခဲ့သည်များ {#what-we-built-this-lesson}

- မတူညီသော ကျန်းမာရေးလုပ်ငန်း အရွယ်အစားများအတွက် ဒီဇိုင်းထုတ်ထားသည့် **အထူးပြု Template သုံးခု**
- ပလက်ဖောင်းတစ်ခုလုံးကို အထူးထုတ်လုပ်ထားသည်ဟု ခံစားရစေသည့် **ကျန်းမာရေးနှင့် အားကစားနှင့် သင့်လျော်သော အကြောင်းအရာနှင့် ပုံရိပ်များ**
- ဖောက်သည်များအား စိတ်ကြိုက်ပြင်ဆင်မှုများ လုပ်ဆောင်ရန် လမ်းညွှန်ပေးသည့် **ညွှန်ကြားချက်ပေးသည့် နေရာယူထားသည့် အကြောင်းအရာများ**
- Template များသည် အသုံးပြုရန် အသင့်ဖြစ်ကြောင်း သေချာစေသည့် **အရည်အသွေး စစ်ဆေးမှုစာရင်း**
