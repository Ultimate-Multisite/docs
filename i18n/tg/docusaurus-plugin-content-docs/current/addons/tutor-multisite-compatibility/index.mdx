# Tutor Multisite Compatibility

Плагини WordPress, ки барои Tutor LMS дар муҳитҳои бисёрсомона ислоҳҳои мутобиқат илова мекунад.

## Тавсиф {#description}

Ин плагин ислоҳҳои муҳими мутобиқатро барои иҷрои Tutor LMS дар шабакаҳои бисёрсомонаи WordPress таъмин мекунад. Он номнависии корбарон, идоракунии омӯзгорон, маҳдудиятҳои курс ва дигар функсияҳои хоси бисёрсомонаро идора мекунад.

## Хусусиятҳо {#features}

- **Идоракунии ҷадвалҳои пойгоҳи додаҳо**: Ба таври худкор ҷадвалҳои гумшудаи Tutor LMS-ро, ки бо сабаби маҳдудиятҳои калиди хориҷии бисёрсомона эҷод намешаванд, муайян ва эҷод мекунад
- **Номнависии корбарони бисёрсомона**: Коркардкунандагони фармоишии AJAX барои идоракунии номнависии корбарон дар саросари сомонаҳои шабака
- **Идоракунии нақши омӯзгор**: Таъйин ва идоракунии худкори нақши омӯзгор дар муҳитҳои бисёрсомона
- **Маҳдудиятҳои курс**: Ҳамгироӣ бо Ultimate Multisite барои идоракунии маҳдудиятҳои эҷоди курс ва funnel бар асоси нақшаҳои сомона
- **Намоишҳои фармоишии маъмурӣ**: Рӯйхати омӯзгорон ва интерфейси идоракунии мутобиқ ба бисёрсомона
- **Абзорҳои маъмурӣ**: Интерфейси идоракунии пойгоҳи додаҳо барои маъмурони шабака ва сомона

## Талабот {#requirements}

- WordPress 5.0+
- PHP 7.4+
- Плагини Tutor LMS (ба таври худкор тавассути сарлавҳаи `Requires Plugins` маҷбур карда мешавад)
- Насби WordPress Multisite
- Плагини Ultimate Multisite (барои функсияи маҳдудиятҳои курс)

## Насб {#installation}

1. Файлҳои плагинро ба `/wp-content/plugins/tutor-multisite-compatibility/` боргузорӣ кунед
2. Боварӣ ҳосил кунед, ки Tutor LMS насб ва фаъол шудааст
3. Плагинро тавассути менюи 'Плагинҳо' дар WordPress фаъол кунед
4. Плагин ба таври худкор бо Tutor LMS ҳамгиро мешавад

## Функсияҳое, ки аз MU plugin кӯчонида шудаанд {#functionality-moved-from-mu-plugin}

Ин плагин функсияҳоеро дар бар мегирад, ки қаблан дар `wp-content/mu-plugins/daves.php` буданд:

- Маҳдудиятҳои тағйири ҳолати курс
- Маҳдудиятҳои эҷоди курс
- Ҳамгироӣ ва маҳдудиятҳои CartFlows
- Идоракунии нақши омӯзгори Tutor
- Идоракунии номнависии корбарони бисёрсомона
- Намоиши ID-и маҳсулоти курс

## Идоракунии пойгоҳи додаҳо {#database-management}

### Мушкилот {#the-problem}

Дар муҳитҳои бисёрсомонаи WordPress, эҷоди ҷадвалҳои Tutor LMS барои ҷадвалҳое, ки ба ҷадвали корбарон маҳдудиятҳои калиди хориҷӣ доранд, ноком мешавад. Ин аз он сабаб рух медиҳад, ки:

- Ҷадвалҳои хоси сомона пешояндҳое мисли `wp_2_tutor_carts` истифода мебаранд
- Калидҳои хориҷӣ ба `wp_2_users` ишора мекунанд (ки вуҷуд надорад)
- Ҷадвали воқеии корбарон `wp_users` аст (дар тамоми шабака муштарак)

### Ҳал {#the-solution}

Ин плагин ба таври худкор:

1. **Ҷадвалҳои гумшударо муайян мекунад**: Ҷадвалҳоеро месанҷад, ки бо сабаби маҳдудиятҳои калиди хориҷӣ эҷод нашудаанд
2. **Ҷадвалҳоро бо истинодҳои дуруст эҷод мекунад**: Номи дурусти ҷадвали корбарро истифода мебарад (`wp_users` дар бисёрсомона)
3. **Сенарияҳои гуногунро идора мекунад**:
   - Фаъолсозӣ/азнавфаъолсозии плагин
   - Эҷоди сомонаи нав дар шабака
   - Санҷиши дастии ҷадвал тавассути интерфейси маъмурӣ

### Ҷадвалҳои таъсирдида {#affected-tables}

- `tutor_coupon_usages` - Калиди хориҷӣ ба ҷадвали корбаронро дар бар мегирад
- `tutor_carts` - Калиди хориҷӣ ба ҷадвали корбаронро дар бар мегирад
- `tutor_notification_preferences` - Ҷадвали TutorPro бо калиди хориҷӣ ба ҷадвали корбарон

### Интерфейси маъмурӣ {#admin-interface}

Ба абзорҳои идоракунии пойгоҳи додаҳо дар ин ҷо дастрасӣ пайдо кунед:

- **Network Admin**: Танзимот → Tutor Multisite DB
- **Site Admin**: Абзорҳо → Tutor Multisite DB

Интерфейс ба шумо имкон медиҳад:
- Ҷадвалҳои гумшударо дар тамоми сомонаҳо бинед
- Эҷоди ҷадвалҳоро дастӣ оғоз кунед
- Танзимоти пойгоҳи додаҳоро санҷед
- Маълумоти муфассали ҳолатро бинед
