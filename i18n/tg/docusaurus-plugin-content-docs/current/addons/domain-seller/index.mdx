# Domain Seller {#domain-seller}

**Multisite Ultimate: Domain Seller** шабакаи WordPress Multisite-и шуморо ба платформаи пурқуввати фурӯши доменҳо табдил медиҳад. Ба муштариён имкон диҳед, ки доменҳои фармоиширо бевосита ҳангоми сабтином ё аз Dashboard-и худ сабт кунанд, бо санҷиши дастрасӣ дар вақти воқеӣ ва нархгузории шаффоф.

### Чаро Domain Seller? {#why-domain-seller}

Сабти домен яке аз бештар дархостшавандаҳои иловагӣ барои ҳар платформаи WaaS (Сомона ҳамчун хизматрасонӣ) мебошад. Ба ҷои фиристодани муштариёни худ ба сабткунандаи тарафи сеюм, шумо метавонед доменҳоро бевосита пешниҳод кунед — дар шабакаи худ даромади бештарро нигоҳ дошта, ҳамзамон ба муштариён таҷрибаи бе монеа диҳед.

### Хусусиятҳои асосӣ {#key-features}

#### Дастгирии чандин сабткунандаи домен {#multiple-domain-registrar-support}
* **OpenSRS** — Сабт, тамдид ва идоракунии пурра
* **Namecheap** — Ҳамгироии пурра бо WhoisGuard ва озмоиши sandbox
* **HostAfrica** — Ҷустуҷӯи домен, нархгузорӣ, сабт, интиқол, DNS ва ҳифзи ID тавассути Domains Reseller API
* **Openprovider** — Нархгузории reseller, сабт, тамдид, интиқолҳо, DNS, махфияти WHOIS ва ҳамоҳангсозии TLD
* **Hostinger** — Ҷустуҷӯ ва сабти домен бо истифода аз token-и ҳамгироии муштараки Hostinger
* **Меъмории васеъшаванда** — Бар асоси намунаи Integration Registry сохта шудааст, ки илова кардани провайдерҳои бештарро осон мекунад
* **Санҷиши пайвасти дарунсохт** — Пеш аз фаъолсозии зинда эътиборномаи провайдер ва пайвастшавиро тасдиқ кунед

#### Устоди роҳнамои танзим {#guided-setup-wizard}
* **Оғози кор бо як клик** — Шуморо аз танзими провайдер, воридоти TLD ва эҷоди маҳсулоти домен мегузаронад
* **Ҳамоҳангсозии худкори TLD** — Ҳамаи TLD-ҳои дастрас ва нархгузории яклухтро аз провайдерҳои пайвастшудаи шумо ворид мекунад
* **Эҷоди маҳсулоти пешфарз** — Маҳсулоти домени омода барои истифода бо пешфарзҳои оқилонаи изофанарх эҷод мекунад

#### Нархгузории чандири домен {#flexible-domain-pricing}
* **Се ҳолати изофанарх** — Изофанархи фоизӣ, изофанархи собит ё нархи собит барои ҳар TLD
* **Маҳсулоти махсус барои TLD** — Барои гурӯҳҳои гуногуни TLD маҳсулоти домени ҷудогона эҷод кунед (.com/.net, .io, рамзҳои кишварҳо ва ғ.)
* **Нархгузории шиносоӣ** — Барои ҷалби сабтҳои нав нархи тахфифшудаи соли аввалро пешниҳод кунед
* **Нархгузории вақти воқеӣ** — Нархгузории яклухти зинда аз провайдерҳо бо кэшкунии ҳушманд
* **Дастгирии купон** — Мутобиқати пурра бо рамзҳои тахфифи Multisite Ultimate

#### Ҳифзи махфияти WHOIS {#whois-privacy-protection}
* **Танзим барои ҳар маҳсулот** — Махфияти WHOIS-ро ба Ғайрифаъол, Ҳамеша дохилшуда ё Интихоби муштарӣ таъин кунед
* **Розигии муштарӣ** — Ҳангоми гузоштан ба Интихоби муштарӣ, қуттии интихоб дар пардохт ба муштариён имкон медиҳад, ки ҳифзи махфиятро илова кунанд
* **Нархгузории танзимшаванда** — Барои иловаи махфият нархи солона таъин кунед ё онро ройгон пешниҳод кунед
* **Ҳамгироии сабткунанда** — Ҳангоми сабт WhoisGuard (Namecheap) ё махфияти WHOIS (OpenSRS)-ро худкор фаъол мекунад

#### Таҷрибаи ягонаи пардохт {#unified-checkout-experience}
* **Се ҳолати домен** — Муштариён байни зердомен (ройгон), сабти домени нав ё пайваст кардани домени мавҷудае, ки аллакай доранд, интихоб мекунанд
* **Ҷустуҷӯи домен дар вақти воқеӣ** — Санҷиши дастрасӣ бо AJAX бе боркунии дубораи саҳифа
* **Пешниҳодҳои ҳушманд** — Пешниҳодҳои TLD-и алтернативӣ вақте домени дилхоҳ гирифта шудааст
* **Нархгузории шаффоф** — Намоиши равшани нархи сабт, нархи тамдид ва ҳаққи ихтиёрии махфияти WHOIS
* **Майдонҳои тамоси сабтшаванда** — Маълумоти зарурии сабтшавандаро (ном, суроға, телефон) дар дохили раванди пардохт ҷамъ мекунад
* **URL-и худкор тавлидшудаи сомона** — Суроғаи сомона аз домени интихобшуда худкор ҳосил мешавад

#### Системаи тамдиди худкор {#auto-renewal-system}
* **Тамдидҳои вобаста ба узвият** — Доменҳо ҳангоми фаъол будани узвияти муштарӣ худкор тамдид мешаванд
* **Ҷадвалбандии ҳушманд** — Корҳои тамдид бар асоси санаҳои анҷоми муҳлати домен ба навбат гузошта мешаванд
* **Мантиқи кӯшиши дубора** — Кӯшишҳои дубораи ҳушманд барои талошҳои нокомшудаи тамдид
* **Огоҳиномаҳои почтаи электронӣ** — Ҳушдорҳои танзимшаванда барои тамдидҳои наздик, тамдидҳои муваффақ, нокомиҳо ва анҷоми муҳлатҳо

#### Идоракунии DNS аз ҷониби муштарӣ {#customer-dns-management}
* **DNS-и худхизматрасон** — Муштариён метавонанд сабтҳои DNS-ро (A, AAAA, CNAME, MX, TXT) аз Dashboard-и худ илова, таҳрир ва нест кунанд
* **Вазъи домен** — Вазъи сабт дар вақти воқеӣ, санаҳои анҷоми муҳлат ва маълумоти nameserver
* **Пайвасткунии худкори домен** — Доменҳои сабтшуда ё мавҷуда худкор ба сомонаи муштарӣ пайваст карда мешаванд

#### Абзорҳои администратор {#admin-tools}
* **Сабти дастии домен** — Аз панели администратор доменҳоро аз номи муштариён сабт кунед
* **Мониторинги домен** — Ҳамаи доменҳои муштариён, вазъиятҳо ва санаҳои анҷоми муҳлатро аз як саҳифа бинед
* **Сабтгирии махсуси провайдер** — Каналҳои ҷудогонаи сабт барои ҳар сабткунанда (масалан, domain-seller-namecheap.log)
* **Ҷузъиёти сабт** — Маълумоти сабтшаванда ва додаҳои сабтро дар саҳифаи асосии таҳрири домен бинед

### Провайдерҳои дастгиришаванда {#supported-providers}

* **OpenSRS** — Ҳамгироии пурра бо сабт, тамдид, идоракунии DNS ва махфияти WHOIS
* **Namecheap** — Ҳамгироии пурра бо сабт, тамдид, санҷиши гурӯҳии дастрасӣ ва WhoisGuard-и ройгон
* **HostAfrica** — Сабт, тамдид, интиқолҳо, идоракунии DNS, дарёфти рамзи EPP, қулфи сабткунанда ва ҳифзи ID
* **Openprovider** — Сабт, тамдид, интиқолҳо, идоракунии DNS, дарёфти рамзи EPP, қулфи сабткунанда, махфияти WHOIS ва ҳамоҳангсозии TLD-и reseller
* **Hostinger** — Сабт, ҷустуҷӯи домен, навсозии nameserver, қулфи сабткунанда ва махфияти WHOIS; тамдидҳо аз ҷониби тамдиди худкори Hostinger идора мешаванд
* **GoDaddy** — Сабт ва тамдид
* **ResellerClub** — Сабт, тамдид ва идоракунии DNS
* **NameSilo** — Сабт ва тамдид
* **Enom** — Сабт, тамдид ва идоракунии DNS

### Ҷузъиёти техникӣ {#technical-details}

* **Талаб мекунад:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **Меъморӣ:** Намунаи Integration Registry бо интерфейси Domain_Selling_Capability
* **Боркунии худкор:** Jetpack autoloader бо харитасозии синфҳои PSR-4
* **Стандартҳо:** Стандартҳои коднависии WordPress, навъбандии қатъӣ, дархостҳои омодашуда

## Насбкунӣ {#installation}

1. Боварӣ ҳосил кунед, ки **Multisite Ultimate v2.4.12+** насб ва дар шабака фаъол шудааст
2. plugin-ро ба `/wp-content/plugins/ultimate-multisite-domain-seller/` боргузорӣ кунед
3. plugin-ро аз **Plugins** дар WordPress Network Admin дар шабака фаъол кунед
4. **Setup Wizard** ҳангоми фаъолсозии аввал ба таври худкор оғоз мешавад — қадамҳоро иҷро кунед, то:
   * Сабткунандаи домени худро пайваст кунед
   * Пайвастшавиро санҷед
   * TLD-ҳои дастрас ва нархгузории яклухтро ворид кунед
   * Аввалин маҳсулоти домении худро бо иловапулии нархгузорӣ эҷод кунед
5. Майдони **Интихоби домен**-ро ба шакли checkout-и худ дар **Ultimate Multisite > Checkout Forms** илова кунед

## Саволҳои зуд-зуд додашаванда {#frequently-asked-questions}

### Кадом версияи Multisite Ultimate лозим аст? {#what-version-of-multisite-ultimate-is-required}

Версияи 2.4.12 ё баландтар. addon бо WP Ultimo v1.x ё версияҳои Ultimate Multisite поёнтар аз 2.4.12 мувофиқ нест.

### Кадом провайдерҳои домен дастгирӣ мешаванд? {#which-domain-providers-are-supported}

Нӯҳ сабткунанда дастгирӣ мешаванд: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo**, ва **Enom**. Қобилиятҳо вобаста ба провайдер фарқ мекунанд, зеро API-ҳои сабткунанда амалиётҳои гуногунро дастрас мекунанд; барои шарҳи провайдер ба провайдер ба дастури танзим нигаред. Меъмории Integration Registry-и addon илова кардани провайдерҳои фармоиширо осон мекунад — ба [дастури таҳиягар](/developer/integration-guide/custom-registrar) нигаред.

### Оё ман метавонам муҳофизати махфияти WHOIS-ро пешниҳод кунам? {#can-i-offer-whois-privacy-protection}

Бале. Ҳар як маҳсулоти домен танзими махфияти WHOIS дорад, ки се ҳолат дорад: Ғайрифаъол, Ҳамеша дохилшуда, ё Интихоби муштарӣ. Вақте ки ба Интихоби муштарӣ гузошта мешавад, муштариён ҳангоми checkout қуттии қайдро бо нарх (ё "Ройгон" барои WhoisGuard-и Namecheap) мебинанд. Нишонаи махфият ҳангоми сабт ба таври худкор ба сабткунанда фиристода мешавад.

### Оё ман метавонам барои TLD-ҳои гуногун нархгузории гуногун муқаррар кунам? {#can-i-set-different-pricing-for-different-tlds}

Бале. Барои гурӯҳҳои гуногуни TLD маҳсулоти домении ҷудогона эҷод кунед. Ҳар як маҳсулот метавонад навъи иловапулии худро (фоиз, иловапулии собит, ё нархи собит), TLD-ҳои дастгиришаванда, ва нархгузории муқаддимавӣ дошта бошад.

### Муштариён кадом имконоти checkout-ро мебинанд? {#what-checkout-options-do-customers-see}

Майдони Интихоби домен ба муштариён се ҷадвал медиҳад: **Зердомен** (истифодаи зердомени ройгон дар шабакаи шумо), **Сабти домени нав** (ҷустуҷӯ ва сабти домени нав), ва **Домени мавҷуда** (пайваст кардани домене, ки аллакай доранд). Танҳо ҳолатҳое, ки шумо фаъол мекунед, намоён мешаванд.

### Оё он навсозиҳои худкорро дастгирӣ мекунад? {#does-it-support-automatic-renewals}

Бале. Доменҳо метавонанд ба навсозии худкор гузошта шаванд ва ба ҳолати узвияти муштарӣ пайваст карда мешаванд. Пардохтҳои навсозӣ тавассути дарвозаи пардохти мавҷудаи муштарӣ коркард мешаванд.

### Оё муштариён метавонанд сабтҳои DNS-и худро идора кунанд? {#can-customers-manage-their-dns-records}

Бале. Муштариён дар dashboard-и худ интерфейси идоракунии DNS мегиранд, то сабтҳои A, AAAA, CNAME, MX, ва TXT-ро илова, таҳрир ва нест кунанд. Идоракунии DNS бо OpenSRS, ResellerClub, Enom, HostAfrica, ва Openprovider дастрас аст. DNS-и Hostinger барои доменҳои hosted тавассути ҳамгироии асосии domain-mapping-и Hostinger коркард мешавад.

### Оё setup wizard вуҷуд дорад? {#is-there-a-setup-wizard}

Бале. setup wizard ҳангоми фаъолсозии аввал ба таври худкор иҷро мешавад ва шуморо дар пайваст кардани сабткунанда, ворид кардани TLD-ҳо, ва эҷоди аввалин маҳсулоти домении шумо роҳнамоӣ мекунад. Шумо инчунин метавонед wizard-ро аз танзимоти addon дубора иҷро кунед.
