# Тағйироти Metered Plans

Версияи 1.1.0 - Нашр шуд дар 2026-05-05
- Нав: ҳисоббарории AI token барои зерсайтҳои multisite — истифодаи AI token-ро дар саросари сайтҳои муштариён бо нархҳои танзимшавандаи барои ҳар token пайгирӣ ва ҳисоббарорӣ кунед
- Нав: иҷборисозии пайвасткунакҳо бо кашфи динамикии маҳдудият ва сабти мустақим аз нав сохта шуд, ки дақиқии вақти воқеиро дар саросари ҳамаи пайвасткунакҳо таъмин мекунад
- Ислоҳ: навсозиҳои ҷадвали пойгоҳи додаҳо акнун бо таърифҳои схемаи BerlinDB дуруст мутобиқ карда шудаанд ва нокомиҳои навсозиро дар насбҳои нав пешгирӣ мекунанд
- Ислоҳ: callback-ҳои навсозии пойгоҳи додаҳо ба формати дуруст табдил дода шуданд, ки нокомиҳои хомӯши навсозиро ҳал мекунад
- Ислоҳ: арзишҳои касрӣ акнун дар майдони вуруди AI Usage Overage Markup қабул карда мешаванд
- Ислоҳ: хатогиҳои марговар ва масъалаҳои дукарата оғозшавӣ ҳангоми боршавии plugin ҳал шуданд
- Беҳтаршуда: скрипти check-env npm илова шуд, то муҳитҳои таҳиягарон дар иҷрои аввал худтанзим шаванд
