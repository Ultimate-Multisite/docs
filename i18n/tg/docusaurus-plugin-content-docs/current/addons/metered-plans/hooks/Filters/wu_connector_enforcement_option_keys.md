# Филтр: wu_connector_enforcement_option_keys

Рӯйхати калидҳои имконоти провайдери AI-ро, ки аз сайти асосӣ маҷбуран татбиқ мешаванд, филтр кунед.

Калидҳои имконотро барои plugin-ҳои провайдери AI-и фармоишӣ ё тарафи сеюм илова кунед, то танзимоти онҳо низ аз сайти асосӣ дар зерсайтҳо мерос гирифта шаванд.

## Параметрҳо {#parameters}

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $keys | `string[]` | Номҳои калидҳои имконот барои маҷбуран татбиқ кардан (аллакай калидҳои connector-и ба таври динамикӣ ошкоршуда ва EXTRA_PROVIDER_OPTIONS-ро дар бар мегирад). |

### Аз версия {#since}

- 1.2.0
