# Чандиҷорагирӣ

Зерсайтҳоро байни серверҳо ва пойгоҳҳои додаҳо барои иҷроиш ва амният тақсим кунед

## Нуктаҳои барҷастаи нашри версияи 1.2.0 {#version-120-release-highlights}

Multi-Tenancy 1.2.0 пояи иҷорагири соҳибихтиёрро барои равандҳои ҷудокунии истеҳсолӣ илова мекунад:

- Воридшавии худкори иҷорагир бе ҳолат бо боздидҳои SSO кӯтоҳмуддат ва ҳифзшуда аз такрор.
- Ҷудокунии пойгоҳи дода ва системаи файлӣ барои ҳар зерсайт барои иҷорагирони соҳибихтиёр.
- Қобилияти чандиҷорагирии Hostinger барои таъминкунии иҷорагир бо огоҳӣ аз мизбон.
- Дарвозаҳои санҷиши муҳоҷирати соҳибихтиёр барои санҷишҳои додаҳои кӯҳна ва коркарди фиристодани асинхронӣ.
- Пайвасткунии танзимшавандаи мизбони пойгоҳи дода дар ҳамон мошин барои насбҳои Bedrock, FrankenPHP ва localhost.
- Беҳбудиҳои роҳандозии ибтидоии иҷорагир, нависандаи пойгоҳи дода, роҳандозии ибтидоии схема ва таъминкунии корбар.
- Барҳамдиҳии соҳибихтиёри бехатартар, ки маълумоти эътибории иҷорагирро пок мекунад.

Ба роҳнамоҳои корбар барои [раванди муҳоҷирати соҳибихтиёр](/user-guide/getting-started/sovereign-migration-workflow), [ҷудокунии чандиҷорагирӣ](/user-guide/configuration/multi-tenancy-isolation) ва [амалиёти иҷорагири соҳибихтиёр](/user-guide/administration/sovereign-tenant-operations) нигаред.

## Талабот {#requirements}

- WordPress 5.3 ё баландтар
- PHP 7.4 ё баландтар
- Плагини Multisite Ultimate (фаъол)

## Насб {#installation}

1. Файлҳои addon-ро ба директорияи `/wp-content/plugins/`-и худ боргузорӣ кунед
2. Плагинро тавассути менюи 'Plugins' дар WordPress фаъол кунед
3. Танзимоти addon-ро дар мудири шабака танзим кунед

## Таҳия {#development}

### Танзим {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Санҷиш {#testing}

Addon чаҳорчӯбаи фарогири санҷишро дар бар мегирад:

- **Санҷишҳои воҳидӣ**: Санҷишҳои асосёфта ба PHPUnit бо маҷмӯаи санҷишии WordPress
- **Синфи асосии санҷиш**: Усулҳои ёрирасонро барои сенарияҳои маъмули санҷиш таъмин мекунад
- **Фарогирии санҷиш**: Барои пайгирии фарогирии код танзим шудааст
- **Санҷиши multisite**: Санҷишҳо дар муҳити multisite иҷро мешаванд

### Стандартҳои код {#code-standards}

- **Стандартҳои коднависии WordPress**: Тавассути PHPCS иҷро карда мешаванд
- **Мутобиқат бо PHP 7.4+**: Хусусиятҳои муосири PHP дастгирӣ мешаванд
- **Таҳлили статикӣ**: Ҳамгироии PHPStan барои санҷиши навъҳо
- **Навсозии код**: Қоидаҳои Rector барои навсозиҳои PHP

### Системаи сохтан {#build-system}

- **Коркарди дороиҳо**: Хурдсозӣ барои файлҳои CSS/JS
- **Тарҷума**: Эҷоди файли POT
- **Бастабандӣ**: Эҷоди бойгонӣ барои тақсимот
- **Ҳолати таҳия**: Дороиҳои хурднашуда барои ислоҳи хатогиҳо

## Танзимот {#configuration}

Addon имконоти гуногуни танзимотро дастгирӣ мекунад, ки тавассути интерфейси мудири шабака дастрасанд.

### Танзимоти иҷорагири соҳибихтиёр {#sovereign-tenant-configuration}

Барои иҷорагирони соҳибихтиёр, пойгоҳи додаи иҷорагир, решаи системаи файлӣ, пайвасткунии мизбони пойгоҳи дода ва қобилияти мизбонро пеш аз равона кардани трафик ба иҷорагир танзим кунед. Пас аз ҳар тағйироте, ки ба маълумоти эътибории пойгоҳи дода, пайвасткунии мизбон, масирдиҳӣ ё таъминкунии корбар таъсир мерасонад, санҷиши муҳоҷиратро иҷро кунед.

`localhost`-ро танҳо вақте истифода баред, ки муҳити иҷроии иҷорагир ва иҷозатҳои пойгоҳи дода дастрасӣ тавассути socket-и маҳаллӣ ё ҳамон мошинро интизор бошанд. `127.0.0.1` ё номи мизбони хизматрасониро вақте истифода баред, ки он мизбони иҷозатдодаи танзимшуда аз ҷониби провайдери шумост.

### Қобилияти Hostinger {#hostinger-capability}

Қобилияти чандиҷорагирии Hostinger бо токени муштараки ҳамгироии Hostinger кор мекунад. Тасдиқ кунед, ки домен дар hPanel мавҷуд аст, DNS ба ҳисоби Hostinger ишора мекунад ва танзимоти пойгоҳи додаи иҷорагир пеш аз санҷиш бо hPanel мувофиқат мекунанд.

## Hook-ҳо ва филтрҳо {#hooks-and-filters}

### Амалҳо {#actions}

- `ultimate-multisite-addon-template_init` - Пас аз оғозкунии addon иҷро мешавад
- `ultimate-multisite-addon-template_loaded` - Пас аз бор шудани ҳамаи файлҳои addon иҷро мешавад

### Филтрҳо {#filters}

- `ultimate-multisite-addon-template_settings` - Тағйир додани танзимоти addon
- `ultimate-multisite-addon-template_enabled` - Бознависии ҳолати фаъол/ғайрифаъоли addon
