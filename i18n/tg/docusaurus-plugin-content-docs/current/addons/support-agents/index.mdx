# Агентҳои дастгирӣ

Идора кардани тамоми шабака ба танҳоӣ кори осон нест. Вақте ки шабакаи шумо рушд мекунад, шумо афзоиши талабот ба дастгириро мебинед. Ҷалб кардани одамоне, ки ба кам кардани бори кори шумо кӯмак мекунанд, фикри хуб хоҳад буд. Аммо додани дастрасӣ/иҷозати пурра хатарнок аст ва метавонад шабакаи шуморо зери хатар гузорад. Маҳз дар ҳамин ҷо ин иловаи Агентҳои дастгирӣ ба кор меояд.

Ин илова ба шумо имкон медиҳад, ки навъи нави корбари WordPress-ро бо номи Агенти дастгирӣ эҷод кунед ва ба шумо назорати пурра медиҳад, ки ин корбар кадом иҷозатҳоро дорад. Додани иҷозат ҳамон қадар осон аст, ки тугмаи гузаришро фаъол кунед, ва агар қарор кунед онро хориҷ намоед, онро хомӯш кунед.

## Хусусиятҳо

- Рӯйхати ҷузъбандишудаи иҷозатҳои WordPress ва Ultimate Multisite
- Дастрасиро ба осонӣ фаъол ва ғайрифаъол кунед
- Шумораи номаҳдуди ҳисобҳои агент эҷод кунед

## Насбкунӣ

1. 'ultimate-multisite-support-agents'-ро ба директорияи '/wp-content/plugins/' боргузорӣ кунед
2. Иловаро тавассути менюи 'Иловаҳо' дар WordPress фаъол кунед
3. Барои танзим кардани илова Устоди қадам ба қадамро пайравӣ кунед
