# Superdav AI Agent

Superdav AI Agent як ёвари пешрафтаи AI барои WordPress аст, ки Gratis AI Agent-ро бо малакаҳои огоҳ аз мавзӯъ, қобилиятҳои беҳтаршудаи ҷустуҷӯи интернетӣ, хусусиятҳои созандаи сайт ва ҳамгироиҳои ихтиёрии худкорсозӣ васеъ мекунад. Версияи 1.18.0 абзорҳои хондани Google Calendar, харитасозии мухотабон ва иштирокчиён, дарвозаҳои тасдиқи инсонӣ, бартарафсозии такрори ёдрасҳо, огоҳиномаҳои SMS-и TextBee, таъмини идорашавандаи хидмати Superdav ва бастаи ҳамроҳи Advanced-ро, ки барои равандҳои кории таҳиягарони боэътимод алоҳида паҳн мешавад, илова мекунад.

## Хусусиятҳои асосӣ {#key-features}

- **Малакаҳои огоҳ аз мавзӯъ** — роҳнамоӣ ва қобилиятҳоро бар асоси мавзӯи фаъоли WordPress ва plugin-ҳои насбшудаи шумо ба таври худкор мутобиқ мекунад
- **Якчанд таъминкунандаи ҷустуҷӯи интернетӣ** — байни Brave Search ва Tavily барои натиҷаҳои ғанитари ҷустуҷӯ ва қобилиятҳои пешрафтаи таҳқиқот интихоб кунед
- **Формаҳои тамоси созандаи сайт** — формаҳои тамоси пурра коршоямро мустақиман аз интерфейси чат бо истифода аз plugin-и формаи насбшудаи худ эҷод кунед
- **Созандаи беҳтаршудаи сайт** — нақшаҳои беҳтаршудаи чандмарҳилавии сохтани сайт бо кашфи беҳтари plugin ва барқарорсозӣ аз хатоҳо
- **Малакаҳои дарунсохти мавзӯъ** — роҳнамоии махсус барои Block Themes, Classic Themes, Kadence Blocks ва Kadence Theme
- **Ҳамгироии беҳтаршудаи WooCommerce** — барои эътимоднокӣ ва мутобиқати беҳтар ба истифодаи API-ҳои аслии WooCommerce бозсозӣ шудааст
- **Муайянкунии худкори таъминкунанда** — plugin-ҳо ва мавзӯъҳои дастрасро ҳангоми фаъол ё ғайрифаъол шудани онҳо ба таври худкор муайян ва навсозӣ мекунад
- **Эҷоди логотипи фармоишӣ** — логотипҳои SVG-и фармоиширо бо поксозии худкори бехатар барои фазои номҳо эҷод ва ҷойгир кунед
- **Санҷиши контрасти палитра** — ҷуфтҳои рангро барои мутобиқат ба дастрасии WCAG пеш аз татбиқ ба мавзӯи худ санҷед
- **Кашфи бо акс беҳтаршуда** — ҳангоми мусоҳибаи кашфи Theme Builder аксҳо боргузорӣ кунед, то ба самти тарроҳӣ таъсир расонанд
- **Менюҳои меҳмоннавозӣ** — саҳифаҳои сохторёфтаи менюи хӯрок ва нӯшокиҳоро барои тарабхонаҳо, қаҳвахонаҳо ва тиҷоратҳои меҳмоннавозӣ эҷод кунед
- **Пешнамоишҳои мизи корӣ ва мобилӣ** — тарҳи худро дар дастгоҳҳои мизи корӣ ва мобилӣ ҳангоми интихоби самти тарроҳӣ пешнамоиш кунед
- **Харошидани сайтҳои сатҳи 1** — sd-ai-agent/site-scrape акнун ба таври пешфарз барои таҳлили вебсайтҳои рақибон ва илҳомбахш дастрас аст
- **Абзорҳои хондани Google Calendar** — тақвимҳо ва рӯйдодҳои танзимшударо хонед, то худкорсозиҳо тавонанд дар бораи заминаи ҷадвали дарпешистода мулоҳиза кунанд
- **Харитасозии мухотабон ва иштирокчиён** — иштирокчиёни тақвимро пеш аз эҷоди кори пайгирӣ бо корбарон ва мухотабони маълуми сайт мувофиқ кунед
- **Дарвозаҳои тасдиқи инсонӣ** — худкорсозиҳои ҳассосро барои бозбинӣ пеш аз фиристодани паёмҳо, идомаи кор ё ворид кардани тағйирот таваққуф кунед
- **Сабтҳои ёдрас ва бартарафсозии такрор** — ёдрасҳои аллакай фиристодашударо пайгирӣ кунед, то иҷроҳои такрории худкорсозӣ ҳамон қабулкунандаро ду бор огоҳ накунанд
- **Огоҳиномаҳои SMS-и TextBee** — вақте ки таъминкунандаи SMS фаъол аст, огоҳиномаҳои паёми матнии танзимшударо тавассути TextBee фиристед
- **Таъмини идорашавандаи хидмати Superdav** — сайтҳои дастгиришавандаро ба endpoint-ҳои мизбонии Superdav бо танзими худкор ба ҷойи танзими дастии endpoint пайваст кунед
- **Бастаи ҳамроҳи Advanced** — абзорҳои алоҳида паҳншавандаи таҳиягари боэътимод барои равандҳои кории системаи файлӣ, пойгоҳи додаҳо, WP-CLI, паҳнкунандаи REST, созандаи plugin, аксгирии git, идоракунии корбар ва санҷиши меъёрӣ

## Талабот {#requirements}

- WordPress 6.0 ё болотар
- PHP 8.1 ё болотар
- WordPress Multisite (дар шабака фаъолшуда) ё насби яксайтӣ
- Plugin-и пайвасткунандаи фаъоли таъминкунандаи AI (Anthropic, OpenAI ё мувофиқ)
- Gratis AI Agent v1.9.0 ё болотар (Superdav AI Agent Gratis AI Agent-ро васеъ мекунад)

## Насб {#installation}

1. Файлҳои plugin-ро ба `/wp-content/plugins/superdav-ai-agent` боргузорӣ кунед
2. Plugin-ро тавассути экрани **Plugins** дар WordPress фаъол кунед
3. Боварӣ ҳосил кунед, ки Gratis AI Agent низ фаъол аст
4. Ба **Gratis AI Agent → Settings** гузаред ва таъминкунандаи ҷустуҷӯи интернетӣ, пайвасти хидмати идорашавандаи Superdav, Google Calendar ё маълумоти эътибории TextBee-и худро мувофиқи зарурат танзим кунед

Бастаи WordPress.org хусусиятҳои асосии Superdav AI Agent-ро дар бар мегирад. Бастаи ҳамроҳи Advanced барои муҳитҳое, ки дар онҳо абзорҳои таҳиягари боэътимод ба таври ошкор насб ва иҷозат дода шудаанд, алоҳида сохта ва паҳн карда мешавад.

## Малакаҳои огоҳ аз мавзӯъ {#theme-aware-skills}

Superdav AI Agent мавзӯи фаъол ва plugin-ҳои насбшудаи шуморо ба таври худкор муайян мекунад ва малакаи мувофиқи огоҳ аз мавзӯъро ба заминаи ёвар бор мекунад. Малакаҳои дастрас инҳоро дар бар мегиранд:

| Малака | Ба чӣ татбиқ мешавад |
|---|---|
| **Block Themes** | Мавзӯъҳое, ки меъмории блокмеҳвари WordPress (Таҳрири пурраи сайт)-ро истифода мебаранд |
| **Classic Themes** | Мавзӯъҳои анъанавии WordPress, ки қолабҳои PHP ва муҳаррири классикиро истифода мебаранд |
| **Kadence Blocks** | Сайтҳое, ки plugin-и Kadence Blocks фаъол доранд |
| **Kadence Theme** | Сайтҳое, ки Kadence theme ҳамчун мавзӯи фаъол доранд |

Барои роҳнамоии муфассал оид ба ҳар як малака ба [Малакаҳои огоҳ аз мавзӯъ](../../user-guide/configuration/theme-aware-skills) нигаред.

## Таъминкунандагони ҷустуҷӯи интернетӣ {#internet-search-providers}

Superdav AI Agent v1.10.0 **Tavily**-ро ҳамчун алтернатива ба Brave Search барои дархостҳои ҷустуҷӯи интернетӣ илова мекунад. Ҳар ду таъминкунанда натиҷаҳои ҷустуҷӯи вақти воқеиро пешниҳод мекунанд ва метавонанд дар **Gratis AI Agent → Settings → Advanced** танзим шаванд.

| Таъминкунанда | Хусусиятҳо | Танзим |
|---|---|---|
| **Brave Search** | Натиҷаҳои ҷустуҷӯи зуд ва мутамарказ ба махфият | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Натиҷаҳои ҷустуҷӯи бойтар, қобилиятҳои пешрафтаи таҳқиқот | [Tavily API](https://tavily.com/) |

Барои дастурҳои танзим ба [Ҷустуҷӯи интернет](../../user-guide/configuration/internet-search) нигаред.

## Равандҳои тақвим, тасдиқ ва огоҳинома {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 равандҳои автоматикунониро берун аз вазифаҳои муҳтаво ва сохтани сомона васеъ мекунад:

- **Абзорҳои Google Calendar** ба агент имкон медиҳанд, ки ҳангоми дастрас будани маълумоти эътибории тақвим тақвимҳои танзимшуда ва ҷузъиёти рӯйдодҳоро бихонад.
- **Ёварҳои тамос ва иштироккунандагон** пеш аз омода кардани ёдрасҳо ё вазифаҳои пайгирӣ иштироккунандагони рӯйдодро ба корбарони WordPress ё тамосҳои маълум мутобиқ мекунанд.
- **Дарвозаҳои тасдиқи инсонӣ** ба автоматикунонӣ имкон медиҳанд, ки то баррасии амали пешниҳодшуда аз ҷониби корбари ваколатдор таваққуф кунад.
- **Сабтҳои ёдрас** ҳангоми кӯшиши дубора ё ба таври такрорӣ ба нақша гирифтани автоматикунонӣ аз ёдрасҳои такрории тақвим ё огоҳинома пешгирӣ мекунанд.
- **Дастгирии TextBee SMS** ҳангоми танзим шудани таъминкунандаи TextBee огоҳиномаҳои паёми матниро фаъол мекунад.

Ин абзорҳоро барои кумакҳои бо ҷадвал огоҳ, пайгирии иштироккунандагон ва равандҳои ёдрас истифода баред, ки пеш аз фиристодани паёмҳои ҳассос ҳанӯз баррасии инсонро талаб мекунанд.

## Хусусиятҳои Theme Builder {#theme-builder-features}

Theme Builder дар Superdav AI Agent v1.16.0 қобилиятҳои нави пурқувватро дар бар мегирад:

### Мусоҳибаи кашф бо боргузории акс {#discovery-interview-with-photo-upload}

Мусоҳибаи кашфи Theme Builder акнун қадами боргузории аксро дар бар мегирад, ки дар он шумо метавонед илҳоми визуалӣ пешниҳод кунед. Аксҳои бренди худ, сомонаҳои рақибон ё илҳоми тарроҳиро боргузорӣ кунед, то ба Theme Builder барои фаҳмидани самти тарроҳии шумо кумак расонед.

Барои роҳнамоии муфассал ба [Мусоҳибаи кашф](./theme-builder/discovery-interview.md) нигаред.

### Самти тарроҳӣ бо пешнамоишҳои мизи корӣ ва мобилӣ {#design-direction-with-desktop-and-mobile-previews}

Пеш аз он ки Theme Builder theme-и пурраи шуморо тавлид кунад, тарроҳии худро ҳам дар дастгоҳҳои мизи корӣ ва ҳам мобилӣ баррасӣ кунед. Пешнамоишҳои вақти воқеиро дар чандин нуқтаи мутобиқшаванда бинед.

Барои роҳнамоии муфассал ба [Самти тарроҳӣ](./theme-builder/design-direction.md) нигаред.

### Менюҳои меҳмоннавозӣ {#hospitality-menus}

Theme Builder акнун метавонад барои тарабхонаҳо, қаҳвахонаҳо, барҳо ва дигар тиҷоратҳои меҳмоннавозӣ саҳифаҳои сохторёфтаи менюи ғизо ва нӯшокиҳоро тавлид кунад. Менюҳо нархгузорӣ, тавсифҳо, маълумоти парҳезӣ ва ҳушдорҳои аллергенҳоро дар бар мегиранд.

Барои роҳнамоии муфассал ба [Менюҳои меҳмоннавозӣ](./theme-builder/hospitality-menus.md) нигаред.

## Танзимот {#configuration}

Ҳама танзимот дар **Gratis AI Agent → Settings** дар маъмурияти WordPress ҷойгиранд. Superdav AI Agent имконоти зерини танзимро илова мекунад:

| Танзим | Тавсиф |
|---|---|
| **Таъминкунандаи ҷустуҷӯи интернет** | Байни Brave Search ва Tavily интихоб кунед (ихтиёрӣ) |
| **Tavily API Key** | API key барои ҷустуҷӯи интернетии Tavily (агар Tavily интихоб шуда бошад) |
| **Хидмати идорашавандаи Superdav** | Сомонаҳои дастгиришавандаро ба хидмати мизбонишудаи Superdav бо таъминкунии худкори endpoint пайваст кунед |
| **Маълумоти эътибории Google Calendar** | Дастрасии тақвимро танзим кунед, то агент тақвимҳо ва рӯйдодҳоро бихонад |
| **Таъминкунандаи TextBee SMS** | Маълумоти эътибории TextBee-ро барои равандҳои тасдиқшудаи огоҳиномаи SMS танзим кунед |

## Бастаи ҳамроҳи пешрафта {#advanced-companion-package}

Superdav AI Agent v1.18.0 бастаи ҳамроҳи Advanced-ро илова мекунад, ки барои равандҳои кори боэътимоди таҳиягарон алоҳида паҳн мешавад. Ин қобилиятҳо дар бастаи асосии WordPress.org дохил карда нашудаанд. Бастаи ҳамроҳро танҳо дар сомонаҳое насб кунед, ки маъмурон ба таври возеҳ ба дастрасии агент ба абзорҳои таҳиягар эътимод доранд.

Абзорҳои Advanced амалиётҳои низоми файлӣ, дастрасӣ ба пойгоҳи додаҳо, WP-CLI, равандҳои dispatcher-и REST, василаҳои plugin builder, snapshot-ҳои git, ёварҳои идоракунии корбарон ва абзорҳои benchmark-ро дар бар мегиранд. Ин абзорҳоро танҳо ба маъмурони боэътимод маҳдуд нигоҳ доред ва пеш аз фаъол кардани онҳо дар сомонаҳои истеҳсолӣ танзими дарвозаҳои тасдиқро баррасӣ кунед.

## Беҳбудиҳо дар v1.18.0 {#improvements-in-v1180}

- **Мутобиқсозии тамосҳо ва иштироккунандагон** — иштироккунандагони тақвимро бо корбарон ва тамосҳои сомона мувофиқ кунед
- **Дарвозаҳои тасдиқи инсонӣ** — автоматикунониҳои ҳассосро пеш аз идома барои баррасӣ таваққуф диҳед
- **Бартарафсозии такрори ёдрасҳо** — ёдрасҳоро сабт кунед, то аз огоҳиномаҳои такрорӣ ҳангоми кӯшишҳои дубора ё иҷроҳои такрорӣ пешгирӣ шавад
- **Таъминкунандаи TextBee SMS** — пас аз тасдиқҳои зарурӣ огоҳиномаҳои паёми матнии танзимшударо фиристед

- **Паҳнкунии ҳамроҳи Advanced** — абзорҳои таҳиягари боэътимодро аз сохти асосии WordPress.org ҷудогона нашр кунед
