# Тавлиди логои SVG

Қобилияти **Тавлиди логои SVG** ба Theme Builder имкон медиҳад, ки логоҳои фармоишии SVG-ро мустақиман дар сайти WordPress-и шумо бо поксозии худкори бехатар барои namespace эҷод ва ҷойгир кунад.

## Шарҳи умумӣ {#overview}

Ин қобилият логоҳои графикаи вектории миқёспазир (SVG)-ро бар асоси самти брендинги сайти шумо ва афзалиятҳои тарроҳӣ эҷод мекунад. SVG-ҳои эҷодшуда ба таври худкор поксозӣ мешаванд, то барои истифода дар WordPress бехатар бошанд ва ҳамзамон якпорчагии визуалиро нигоҳ доранд.

## Параметрҳо {#parameters}

| Параметр | Навъ | Ҳатмӣ | Тавсиф |
|-----------|------|----------|-------------|
| `site_name` | string | Ҳа | Номи сайт, ки барои он лого эҷод мешавад |
| `style` | string | Ҳа | Услуби тарроҳӣ (масалан, "муосир", "классикӣ", "минималистӣ", "бозигарона") |
| `colors` | array | Не | Массиви кодҳои ранги hex барои истифода дар лого (масалан, `["#678233", "#ffffff"]`) |
| `width` | number | Не | Паҳноии SVG бо пиксел (пешфарз: 200) |
| `height` | number | Не | Баландии SVG бо пиксел (пешфарз: 200) |
| `include_text` | boolean | Не | Оё номи сайт ҳамчун матн дар лого дохил карда шавад ё не (пешфарз: true) |

## Формати баромад {#output-format}

Ин қобилият сатри SVG-ро бо сохтори зерин бармегардонад:

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## Рафтори поксозии SVG {#svg-sanitisation-behaviour}

SVG-ҳои эҷодшуда аз поксозии худкори бехатар барои namespace мегузаранд, то:

- **Хориҷ кардани атрибутҳои ноамн** — коркардгарони рӯйдод, скриптҳо ва атрибутҳои эҳтимолан хатарнокро ҳазф мекунад
- **Нигоҳ доштани namespace-ҳо** — namespace-ҳои SVG (xmlns, xlink)-ро барои намоиши дуруст нигоҳ медорад
- **Санҷидани сохтор** — мутобиқати SVG-ро ба стандартҳои W3C таъмин мекунад
- **Рамзгузории entity-ҳо** — аломатҳои махсусро дар муҳтавои матнӣ дуруст escape мекунад
- **Хориҷ кардани истинодҳои беруна** — ҷадвалҳои услуби беруна ва истинодҳои тасвирро ҳазф мекунад

Ин кафолат медиҳад, ки SVG барои ҷойгиркунии мустақим дар WordPress бехатар аст ва поксозии иловагӣ талаб намекунад.

## Намунаи истифода {#usage-example}

**Prompt:**
```
Барои стартапи технологии ман бо номи "CloudSync" бо истифода аз рангҳои кабуд ва сафед як логои муосир эҷод кунед.
```

**Натиҷа:**
Ин қобилият логои SVG эҷод мекунад, ки:
- Номи сайт "CloudSync"-ро дар бар мегирад
- Нақшаи рангҳои кабуд ва сафеди муайяншударо истифода мебарад
- Аз принсипҳои тарроҳии муосир пайравӣ мекунад
- Ба таври худкор поксозӣ шудааст ва барои истифода омода аст

## Интегратсия бо Theme Builder {#integration-with-theme-builder}

Ҳангоми истифодаи интихоби самти тарроҳии Theme Builder, қобилияти Тавлиди логои SVG:

1. Самти тарроҳӣ ва палитраи рангҳои шуморо таҳлил мекунад
2. Логои фармоишии SVG эҷод мекунад, ки ба афзалиятҳои шумо мувофиқ аст
3. Логоро ба таври худкор ба қисми сарлавҳа/брендинги сайти шумо ҷойгир мекунад
4. SVG-ро ҳамчун логои фармоишӣ дар медиаи WordPress нигоҳ медорад

## Таҷрибаҳои беҳтарин {#best-practices}

- **Афзалиятҳои услубиро равшан пешниҳод кунед** — услуби тарроҳии дилхоҳатонро тавсиф кунед (муосир, классикӣ, бозигарона ва ғайра)
- **Рангҳоро муайян кунед** — барои ҳамоҳангӣ рангҳои бренди худро дохил кунед
- **Намоишро санҷед** — санҷед, ки лого дар андозаҳои гуногуни экран дуруст нишон дода мешавад
- **Минбаъд фармоишӣ кунед** — аз абзорҳои фармоишисозии логотипи WordPress барои танзими андоза ва ҷойгиршавӣ истифода баред

## Маҳдудиятҳо {#limitations}

- Логоҳои SVG ҳамчун графикаи статикӣ эҷод мешаванд (аниматсияшуда нестанд)
- Логоҳои мураккаб бо унсурҳои зиёд метавонанд такмили дастиро талаб кунанд
- Шрифтҳои фармоишӣ дастгирӣ намешаванд; матн аз шрифтҳои система истифода мебарад
- Андозаҳои хеле калон ё хеле хурд метавонанд ба сифат таъсир расонанд
