# Эҷоди меню

Қобилияти **Эҷоди меню** менюҳои навигатсионии WordPress-ро бо дастгирии барчаспҳои ҷудогонаи навигатсионӣ, ки аз унвонҳои саҳифа фарқ мекунанд, эҷод мекунад.

## Шарҳи умумӣ

Ин қобилият функсионалияти стандартии эҷоди менюро бо имконияти муайян кардани параметри `navigation_label` васеъ мекунад. Ин ба шумо имкон медиҳад менюҳое эҷод кунед, ки дар онҳо барчаспи нишон додашуда дар навигатсия аз унвони саҳифа фарқ мекунад ва дар сохтори сайт ва таҷрибаи корбар чандирии бештар фароҳам меорад.

## Параметрҳо

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | string | Yes | Номи меню, масалан `Primary Navigation` |
| `location` | string | No | Ҷойгиршавии theme барои таъин кардани ин меню ба он, масалан `primary` |
| `navigation_label` | string | No | Барчаспе, ки дар навигатсия нишон дода мешавад (ҷудо аз унвони саҳифа) |

## Арзиши баргардонидашаванда

```json
{
  "success": true,
  "menu_id": 7,
  "name": "Primary Navigation",
  "navigation_label": "Main Menu"
}
```

## Барчаспи навигатсия нисбат ба унвони саҳифа

Параметри `navigation_label` ба шумо имкон медиҳад номи дохилии менюро аз барчаспе, ки ба корбарон нишон дода мешавад, ҷудо кунед:

- **`name`** — Шиносандаи дохилии меню, ки WordPress истифода мебарад (масалан, "Primary Navigation")
- **`navigation_label`** — Барчаспе, ки ба боздидкунандагони сайт дар навигатсия нишон дода мешавад (масалан, "Main Menu")

Ин муфид аст, вақте ки:
- Қоидаи номгузории дохилии шумо аз барчаспҳои рӯ ба корбар фарқ мекунад
- Шумо дар навигатсия нисбат ба панели идоракунӣ барчаспҳои кӯтоҳтар мехоҳед
- Шумо бояд чанд забонро бо дарозии гуногуни барчаспҳо дастгирӣ кунед
- Шумо барои минтақаҳои мушаххас ё гурӯҳҳои корбарон меню месозед

## Намунаҳои истифода

### Намунаи 1: Менюи одӣ бо барчаспи навигатсия

**Prompt:**
```
Менюи навигатсияи асосиро бо номи "Main Navigation" ва бо барчаспи навигатсияи "Menu" эҷод кунед.
```

**Натиҷа:**
```json
{
  "success": true,
  "menu_id": 7,
  "name": "Main Navigation",
  "navigation_label": "Menu"
}
```

### Намунаи 2: Меню барои ҷойгиршавии мушаххаси Theme

**Prompt:**
```
Менюи поёнаро бо номи "Footer Links" ва бо барчаспи навигатсияи "Quick Links" эҷод кунед ва онро ба ҷойгиршавии поён таъин кунед.
```

**Натиҷа:**
```json
{
  "success": true,
  "menu_id": 8,
  "name": "Footer Links",
  "navigation_label": "Quick Links",
  "location": "footer"
}
```

## Ҳамгироӣ бо Theme Builder

Ҳангоми истифодаи Theme Builder, қобилияти Эҷоди меню:

1. Ҷойгиршавиҳои дастраси менюи theme-ро ба таври худкор муайян мекунад
2. Менюҳоро бо барчаспҳои мувофиқи навигатсия барои тарҳи шумо эҷод мекунад
3. Менюҳоро ба ҷойгиршавиҳои дурусти theme таъин мекунад
4. Илова кардани бандҳои менюро пас аз эҷод дастгирӣ мекунад

## Қобилиятҳои марбут

- **`add_menu_item`** — Илова кардани бандҳо ба менюи мавҷуда
- **`update_menu`** — Тағйири номи меню ё аз нав таъин кардани он ба ҷойгиршавии theme
- **`delete_menu`** — Нест кардани меню аз сайти шумо

## Таҷрибаҳои беҳтарин

- **Барчаспҳои равшани навигатсияро истифода баред** — барчаспҳоро барои корбарон кӯтоҳ ва тавсифӣ нигоҳ доред
- **Ҷойгиршавиҳои theme-ро мувофиқ кунед** — менюҳоро барои намоиши дуруст ба ҷойгиршавии дурусти theme таъин кунед
- **Сохтори менюро ба нақша гиред** — пеш аз эҷоди бандҳо иерархияи менюи худро муайян кунед
- **Мутобиқшавиро санҷед** — боварӣ ҳосил кунед, ки менюҳо дар дастгоҳҳои мобилӣ дуруст намоиш дода мешаванд
- **Барчаспҳоро маҳаллӣ кунед** — барои версияҳои забонии гуногун барчаспҳои гуногуни навигатсияро истифода баред

## Маҳдудиятҳо

- Барчаспҳои навигатсия танҳо барои намоиш мебошанд; `name`-и дохилӣ ҳамоно барои шиносоии WordPress истифода мешавад
- Дастгирии theme фарқ мекунад; на ҳамаи theme-ҳо ҳамаи ҷойгиршавиҳои менюро дастгирӣ мекунанд
- Бандҳои меню бояд пас аз эҷоди меню ҷудогона илова карда шаванд
- Тағйир додани барчаспи навигатсия навсозии менюро талаб мекунад
