# Таърихи тағйирот {#changelog}

## 1.18.0 — Нашр шуд дар 2026-06-29 {#1180--released-on-2026-06-29}

### Нав {#new}

- **Асбобҳои Google Calendar** — тақвимҳо ва рӯйдодҳои танзимшударо барои худкорсозиҳои огоҳ аз ҷадвал хонед
- **Мутобиқсозии тамосҳо ва ёварҳои иштирокчиён** — иштирокчиёни тақвимро бо корбарон ва тамосҳои сайт мутобиқ кунед
- **Дарвозаҳои тасдиқи инсонӣ ва сабтҳои ёдрасӣ** — худкорсозиҳоро барои баррасӣ таваққуф диҳед ва аз огоҳиномаҳои такрорӣ пешгирӣ кунед
- **Таъминкунандаи SMS-и TextBee** — огоҳиномаҳои паёмакҳои танзимшударо тавассути TextBee фиристед
- **Бастаи ҳамроҳи пешрафта** — асбобҳои низоми файлҳои таҳиягари боэътимод, пойгоҳи додаҳо, WP-CLI, паҳнкунандаи REST, созандаи plugin, аксбардории git, идоракунии корбарон ва санҷиши нишондиҳандаҳоро, ки аз сохтани WordPress.org ҷудогона тақсим мешаванд, илова кунед

### Такмилёфта {#improved}

- **Танзими хидмати идорашавандаи Superdav** — endpoint-ҳои хидмати мизбоншуда ва таъмини худкори пайвастшавиро барои сайтҳои дастгиришаванда илова кунед
- **Бастабандии нашр** — ZIP-ҳои ҷудогонаи ҳаста ва Advanced созед, ҳар дуро дар GitHub нашр кунед ва ба WordPress.org танҳо бастаи ҳастаро фиристед

### Ислоҳшуда {#fixed}

- **Эътимоднокии дархости AI** — интихоби модел, анҷоми вақти дархост, модели пешфарзи танзим, коркарди матни reasoning ва роҳнамоии такрори кӯшишро барои фарохониҳои нодурусти асбоб беҳтар кунед
- **Мустаҳкамсозии тақвим ва ёдрасӣ** — token-ҳои Google Calendar ва ҳазфи такрори ёдрасиҳоро мустаҳкам кунед
- **Оғози кор ва идомаи тасдиқҳо** — оғозҳои оғози кор дар frontend ва идомаи қобилиятҳои тасдиқшударо ислоҳ кунед
- **Масъалаҳои баррасии бастабандии WordPress.org** — фикру мулоҳизаҳои баррасии бастабандиро барои нашри ҳаста ҳал кунед

## 1.16.0 — Нашр шуд дар 2026-05-20 {#1160--released-on-2026-05-20}

### Нав {#new-1}

- **Қобилияти тавлиди Logo SVG** — Theme Builder акнун метавонад Logo SVG-ҳои фармоиширо бо поксозии амн барои namespace тавлид ва ҷойгир кунад
- **Боргузории акс дар мусоҳибаи кашф** — мусоҳибаи кашфи Theme Builder акнун қадами боргузории аксро барои заминаи бойтари тарроҳӣ дар бар мегирад
- **Қобилияти санҷиши контрасти палитра** — ҷуфтҳои рангро пеш аз татбиқ ба theme барои мутобиқат ба WCAG санҷед
- **Менюҳои меҳмондорӣ** — Theme Builder акнун метавонад барои тиҷоратҳои меҳмондорӣ саҳифаҳои сохторёфтаи менюи хӯрок ва нӯшокиҳоро тавлид кунад
- **Намоиши пешакии desktop ва mobile** — ҳангоми интихоби самти тарроҳӣ тарҳи худро дар дастгоҳҳои desktop ва mobile пешакӣ бинед
- **Параметри барчаспи паймоиш** — қобилияти эҷоди меню акнун `navigation_label`-и ҷудогонаеро дастгирӣ мекунад, ки аз унвони саҳифа фарқ мекунад
- **Дастрасии асбоби сатҳи 1** — sd-ai-agent/site-scrape акнун асбоби сатҳи 1 аст, ки ба таври пешфарз дар Theme Builder дастрас мебошад

### Ислоҳшуда {#fixed-1}

- **Кэши AI Client** — акнун бо transients барои пойдории байни дархостҳо пуштибонӣ мешавад ва аз гумшавии додаҳо дар вазифаҳои дарозмуддати agent пешгирӣ мекунад
- **Пайвандҳои амали сатри plugin** — барои равшанӣ ислоҳ ва аз нав номгузорӣ шуданд

## 1.10.0 — Нашр шуд дар 2026-05-05 {#1100--released-on-2026-05-05}

### Нав {#new-2}

- **Ҷустуҷӯи интернетии Tavily** — Tavily-ро ҳамчун таъминкунандаи ҷустуҷӯ барои натиҷаҳои бойтари ҷустуҷӯи интернетӣ дар баробари Brave Search илова кунед
- **Маҳоратҳои дарунсохти огоҳ аз theme** — роҳнамоҳои маҳорати Block Themes, Classic Themes, Kadence Blocks ва Kadence Theme акнун бо plugin пешниҳод мешаванд
- **Қобилияти варақаи тамоси созандаи сайт** — варақаи тамосро мустақиман аз интерфейси чат ба ҳар саҳифа илова кунед

### Такмилёфта {#improved-1}

- **Ҳамгироии WooCommerce аз нав сохторбандӣ шуд** — акнун барои эътимоднокӣ ва мутобиқати беҳтар API-ҳои маҳаллии WooCommerce-ро истифода мебарад
- **Рӯйхати таъминкунандагон худкор нав мешавад** — вақте ки ҳар гуна plugin фаъол ё ғайрифаъол мешавад

### Ислоҳшуда {#fixed-2}

- **Қобилияти navigate-to** — ҳалқаи беохири бозборкуниро дар баъзе саҳифаҳои admin ислоҳ кард
- **Қобилияти list-posts** — акнун номҳои категория ва tag-ро ба slug-ҳо дуруст ҳал мекунад
- **Фармонҳои WP-CLI** — тахаллусҳои namespace-и гумшударо пас аз як сохторбандии қаблӣ барқарор кард
- **Худкорсозии рӯйдод** — сайтҳоеро, ки дар онҳо ҷадвалҳои худкорсозӣ ҳанӯз сохта нашудаанд, нармона коркард мекунад
- **Қобилияти memory-save** — акнун дар созандаи дастури система префикси дурусти namespace-ро истифода мебарад
- **Натиҷаҳои асбоби Scalar** — акнун пеш аз баргардондан ба AI дуруст печонида мешаванд
- **Омори истифода** — акнун ҳангоми навсозӣ аз версияҳои кӯҳнатар формати куҳнаи калиди қобилиятро дуруст коркард мекунад
