# Самти дизайн

Қадами **Самти дизайн** ба шумо имкон медиҳад, ки пеш аз он ки Theme Builder мавзӯи комилро эҷод кунад, дизайни визуалии мавзӯи худро бозбинӣ, такмил ва ниҳоӣ кунед.

## Шарҳи умумӣ

Пас аз анҷом додани Мусоҳибаи кашф, Theme Builder самти дизайни шуморо бо **намоиши пешакии desktop ва mobile** пешниҳод мекунад. Ин ба шумо имкон медиҳад, ки пеш аз тасдиқи мавзӯъ дақиқ бубинед, ки дизайни шумо дар дастгоҳҳои гуногун чӣ гуна менамояд.

## Ҷузъҳои самти дизайн

### 1. Палитраи рангҳо

Нақшаи рангҳои мавзӯи шумо, аз ҷумла:

- **Ранги асосӣ** — ранги асосии бренд, ки барои тугмаҳо, пайвандҳо ва таъкидҳо истифода мешавад
- **Ранги дуюмдараҷа** — ранги пурракунанда барои гуногунӣ
- **Ранги таъкидӣ** — ранги барҷаста барои унсурҳои муҳим
- **Рангҳои бетараф** — хокистарӣ ва сафед барои пасзаминаҳо ва матн
- **Ранги матн** — ранги асосии матн барои хонданпазирӣ

### 2. Типография

Интихоби ҳуруфҳо барои:

- **Ҳуруфи сарлавҳа** — барои унвонҳои саҳифа ва сарлавҳаҳои бахшҳо истифода мешавад
- **Ҳуруфи матни асосӣ** — барои матни параграф ва муҳтаво истифода мешавад
- **Андозаҳои ҳуруф** — андозабандии мутобиқшаванда барои андозаҳои гуногуни экран
- **Баландии сатр** — фосила байни сатрҳо барои хонданпазирӣ
- **Вазнҳои ҳуруф** — навъҳои ғафс, муқаррарӣ ва сабук

### 3. Тарҳ ва фосилагузорӣ

- **Паҳнои контейнер** — паҳнои ҳадди аксар барои муҳтаво
- **Фосилаҳои дохилӣ ва ҳошияҳо** — фосила дар атрофи унсурҳо
- **Системаи шабака** — тарҳи сутунӣ барои дизайни мутобиқшаванда
- **Фосилагузории ҷузъҳо** — масофа байни унсурҳои UI

### 4. Унсурҳои визуалӣ

- **Услубҳои тугма** — дизайнҳои тугмаҳои асосӣ, дуюмдараҷа ва сеюмдараҷа
- **Дизайнҳои корт** — тарҳҳо барои блокҳои муҳтаво
- **Нишонаҳо** — услуб ва андозаи нишона
- **Сарҳадҳо ва сояҳо** — умқи визуалии нозук
- **Эффектҳои hover** — услубдиҳии ҳолати интерактивӣ

## Намоиши пешакӣ

### Намоиши пешакии desktop

Намоиши пешакии desktop дизайни шуморо дар паҳнои пурра нишон медиҳад (одатан 1200px ё васеътар):

- Менюи пурраи паймоиш
- Тарҳи пурра бо ҳамаи сутунҳо
- Типография ва фосилагузории калон
- Ҳама унсурҳои визуалӣ дар андозаи пурра

### Намоиши пешакии mobile

Намоиши пешакии mobile дизайни шуморо дар паҳнои mobile нишон медиҳад (одатан 375px):

- Паймоиши мутобиқшаванда (менюи hamburger)
- Тарҳи яксутунӣ
- Типография ва фосилагузории танзимшуда
- Андозаҳои тугмаҳои мувофиқ барои ламс

### Нуқтаҳои шикасти мутобиқшаванда

Theme Builder намоишҳои пешакиро дар ин нуқтаҳои шикаст рендер мекунад:

| Дастгоҳ | Паҳно | Намоиши пешакӣ |
|--------|-------|---------|
| **Mobile** | 375px | Телефони амудӣ |
| **Tablet** | 768px | Планшети уфуқӣ |
| **Desktop** | 1200px | Desktop-и паҳнои пурра |
| **Large Desktop** | 1920px | Намоиши ултравасеъ |

## Такмил додани дизайни худ

### Ворид кардани тағйирот

Шумо метавонед самти дизайни худро бо роҳҳои зерин такмил диҳед:

1. **Танзими рангҳо** — ҳар гуна рангро дар палитра иваз кунед
2. **Иваз кардани ҳуруфҳо** — шрифтҳои гуногунро интихоб кунед
3. **Тағйир додани фосилагузорӣ** — фосилаҳои дохилӣ ва ҳошияҳоро танзим кунед
4. **Навсозии тарҳ** — паҳнои контейнерҳо ва сутунҳои шабакаро иваз кунед
5. **Фармоишисозии унсурҳо** — услубҳои тугма, дизайнҳои корт ва ғайраро тағйир диҳед

### Навсозиҳои намоиши пешакӣ

Тағйирот дар вақти воқеӣ инъикос мешаванд:

- Намоиши пешакии desktop фавран нав мешавад
- Намоиши пешакии mobile фавран нав мешавад
- Ҳама нуқтаҳои шикасти мутобиқшаванда нав мешаванд
- Шумо метавонед байни намоишҳои пешакӣ гузаред, то тағйиротро санҷед

### Намуди муқоиса

Самти дизайни худро бо инҳо муқоиса кунед:

- **Дизайни аслӣ** — бинед, ки чӣ тағйир ёфт
- **Дизайнҳои рақибон** — бо сомонаҳои илҳомбахш муқоиса кунед
- **Версияҳои қаблӣ** — ба самтҳои дизайни пешина баргардед

## Тасдиқи самти дизайн

Вақте ки аз дизайни худ қаноатманд ҳастед:

1. **Ҳар ду намоиши пешакиро бозбинӣ кунед** — санҷед, ки desktop ва mobile хуб менамоянд
2. **Ҳама рангҳоро санҷед** — контраст ва дастрасиро таъмин кунед
3. **Типографияро озмоиш кунед** — хонданпазириро дар ҳамаи андозаҳо санҷед
4. **Тарҳро тасдиқ кунед** — фосилагузорӣ ва ҳамоҳангиро санҷед
5. **Дизайнро тасдиқ кунед** — ба эҷоди мавзӯъ гузаред

## Қадамҳои баъдӣ

Пас аз тасдиқи самти дизайни худ:

1. Theme Builder мавзӯи пурраи шуморо эҷод мекунад
2. Мавзӯъ дар сомонаи WordPress-и шумо насб карда мешавад
3. Шумо метавонед бо истифода аз танзимгари WordPress минбаъд фармоишӣ кунед
4. Ба [Менюҳои меҳмондорӣ](./hospitality-menus.md) ё дигар хусусиятҳо гузаред

## Таҷрибаҳои беҳтарин

- **Дар дастгоҳҳои воқеӣ озмоиш кунед** — агар имкон бошад, телефонҳо ва планшетҳои воқеиро истифода баред
- **Хонданпазириро санҷед** — мутмаин шавед, ки матн дар ҳамаи андозаҳо хонданпазир аст
- **Контрастро санҷед** — барои дастрасӣ аз [Санҷиши контрасти палитра](../abilities/validate-palette-contrast.md) истифода баред
- **Иҷроишро ба назар гиред** — тасвирҳо ва ҳуруфҳоро барои суръат оптимизатсия кунед
- **Барои муҳтаво нақша гиред** — мутмаин шавед, ки тарҳ бо муҳтавои воқеии шумо кор мекунад

## Ҳалли мушкилот

### Намоиши пешакӣ нав намешавад

- Саҳифаро навсозӣ кунед
- Кэши браузерро пок кунед
- Браузери дигарро санҷед
- Пайвасти интернетро санҷед

### Рангҳо гуногун менамоянд

- Танзимоти ранги мониторро санҷед
- Дар дастгоҳҳои гуногун санҷед
- Контрасти рангро бо абзорҳои дастрасӣ санҷед
- Симуляторҳои нобиноии рангро ба назар гиред

### Мушкилоти типография

- Санҷед, ки файлҳои ҳуруф бор мешаванд
- Андозаи ҳуруфро дар нуқтаҳои шикасти гуногун санҷед
- Бо муҳтавои воқеӣ озмоиш кунед
- Дарозии сатрро барои хонданпазирӣ ба назар гиред
