# Мусоҳибаи Discovery

**Мусоҳибаи Discovery** қадами аввал дар раванди тарроҳии Theme Builder аст. Он маълумотро дар бораи сайти шумо, тиҷорати шумо ва афзалиятҳои тарроҳии шумо ҷамъ мекунад, то theme-и шахсисозишуда эҷод шавад.

## Шарҳи умумӣ

Мусоҳибаи Discovery шуморо тавассути силсилаи саволҳо роҳнамоӣ мекунад, то бренди шумо, аудиторияи ҳадаф ва ҳадафҳои тарроҳиро фаҳмад. Дар версияи 1.16.0, мусоҳиба ҳоло **қадами боркунии акс**-ро дар бар мегирад, ки ба шумо имкон медиҳад барои тарроҳии худ илҳоми визуалӣ пешниҳод кунед.

## Қадамҳои мусоҳиба

### 1. Маълумоти тиҷорат

- Номи сайт ва шиор
- Навъи тиҷорат ё соҳа
- Ҳадафҳои асосии тиҷорат
- Тавсифи аудиторияи ҳадаф

### 2. Афзалиятҳои тарроҳӣ

- Нақшаи рангҳои афзалиятнок
- Услуби тарроҳӣ (муосир, классикӣ, минималистӣ, бозича, ва ғайра)
- Вебсайтҳо ё брендҳои илҳомбахш
- Унсурҳои тарроҳии ҳатмӣ

### 3. Боркунии акс (нав дар v1.16.0)

Як ё чанд аксро бор кунед, то ба самти тарроҳӣ таъсир расонад:

#### Форматҳои қабулшаванда

| Формат | Пасванд | Тафсилот |
|--------|-----------|---------|
| **JPEG** | `.jpg`, `.jpeg` | Барои аксҳо тавсия мешавад |
| **PNG** | `.png` | Шаффофиятро дастгирӣ мекунад |
| **WebP** | `.webp` | Формати муосир, андозаи хурдтари файл |
| **GIF** | `.gif` | Танҳо тасвирҳои статикӣ (бе аниматсия) |

#### Талаботи файл

- **Андозаи максималии файл:** 10 MB
- **Андозаҳои минималӣ:** 400 × 300 пиксел
- **Андозаҳои тавсияшуда:** 1200 × 800 пиксел ё калонтар
- **Таносуби тарафҳо:** Ҳар гуна таносуб қобили қабул аст

#### Чӣ гуна аксҳо ба самти тарроҳӣ таъсир мерасонанд

Аксҳое, ки шумо бор мекунед, ба Theme Builder кӯмак мекунанд фаҳмад:

- **Палитраи рангҳо** — рангҳои бартаридошта ҷудо карда шуда, дар theme истифода мешаванд
- **Услуби визуалӣ** — муосир, рустикӣ, минималистӣ, зинда ва ғайра
- **Ҳолат ва фазо** — касбӣ, эҷодӣ, дӯстона, боҳашамат ва ғайра
- **Унсурҳои тарроҳӣ** — нақшҳо, бофтҳо ва услубҳои композитсионӣ
- **Илҳоми типография** — услубҳои шрифт, ки theme-и визуалиро пурра мекунанд

### 4. Сохтори муҳтаво

- Саҳифаҳои асосии зарурӣ (Home, About, Services, Contact ва ғайра)
- Бахши блог ё хабарҳо лозим аст
- Функсионалии тиҷорати электронӣ лозим аст
- Хусусиятҳои махсус ё ҳамгироиҳо

### 5. Баррасӣ ва тасдиқ

- Хулосаи ҳамаи маълумоти пешниҳодшуда
- Тасдиқ пеш аз оғози тавлиди тарроҳӣ

## Беҳтарин амалҳо барои боркунии акс

### Чиро бор кардан

- **Илҳоми бренд** — логотипҳо, намунаҳои ранг ё дастурҳои бренд
- **Сайтҳои рақибон** — скриншотҳои сайтҳое, ки ба шумо маъқуланд
- **Тахтаҳои ҳолат** — маҷмӯаҳои тасвирҳо, ки эстетикаи дилхоҳи шуморо ифода мекунанд
- **Аксҳои маҳсулот** — агар маҳсулоте барои намоиш дошта бошед
- **Аксҳои даста ё офис** — барои брендингӣ шахсӣ ё касбӣ
- **Тасвирҳои тарзи зиндагӣ** — намояндаи аудиторияи ҳадафи шумо

### Аз чӣ худдорӣ кардан

- **Тасвирҳои пастсифат** — аксҳои хира ё пикселшуда
- **Тасвирҳои номарбут** — аксҳое, ки бренди шуморо ифода намекунанд
- **Тасвирҳои пурматн** — скриншотҳо бо матни зиёд
- **Маводи дорои ҳуқуқи муаллиф** — боварӣ ҳосил кунед, ки барои истифодаи тасвирҳо ҳуқуқ доред
- **GIF-ҳои аниматсионӣ** — танҳо тасвирҳои статикӣ коркард мешаванд

### Маслиҳатҳо барои натиҷаҳои беҳтарин

1. **2-3 тасвир бор кунед** — тасвирҳои сершумор контексти беҳтар медиҳанд
2. **Аксҳои баландсифатро истифода баред** — тасвирҳои равшан ва хуб равшаншуда беҳтарин кор мекунанд
3. **Гуногунӣ дохил кунед** — омехтаи ранг, услуб ва мавзӯъ
4. **Контекст пешниҳод кунед** — кӯтоҳ шарҳ диҳед, ки чаро ҳар тасвирро интихоб кардед
5. **Мушаххас бошед** — агар як унсури муайян ба шумо маъқул бошад, онро нишон диҳед

## Натиҷаи самти тарроҳӣ

Пас аз мусоҳибаи Discovery, Theme Builder тавлид мекунад:

- **Палитраи рангҳо** — рангҳои асосӣ, дуюмдараҷа ва таъкидӣ
- **Типография** — оилаҳои шрифт барои сарлавҳаҳо ва матни бадан
- **Услуби layout** — фосилагузорӣ, ҳамоҳангсозӣ ва ҷойгиркунии компонентҳо
- **Унсурҳои визуалӣ** — нишонаҳо, нақшҳо ва унсурҳои ороишӣ
- **Тахтаи ҳолат** — намояндагии визуалии самти тарроҳӣ

## Қадамҳои баъдӣ

Пас аз анҷом додани мусоҳибаи Discovery:

1. Самти тарроҳии тавлидшударо баррасӣ кунед
2. Фикру мулоҳиза диҳед ё тағйирот дархост кунед
3. Ба қадами [Самти тарроҳӣ](./design-direction.md) гузаред
4. Theme Builder theme-и фармоишии шуморо эҷод мекунад
