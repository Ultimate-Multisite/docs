# Менюҳои меҳмондорӣ

Хусусияти **Менюҳои меҳмондорӣ** ба Theme Builder имкон медиҳад, ки саҳифаҳои сохторёфтаи менюи ғизо ва нӯшокиҳоро мустақиман дар сомонаи WordPress-и шумо эҷод ва ҷойгир кунад.

## Шарҳи умумӣ {#overview}

Theme Builder акнун метавонад барои тиҷоратҳои меҳмондорӣ, аз ҷумла тарабхонаҳо, қаҳвахонаҳо, барҳо ва хизматрасониҳои кейтеринг саҳифаҳои менюи касбӣ ва сохторёфта эҷод кунад. Ин менюҳо пурра ба тарҳи сомонаи шумо ҳамгиро мешаванд ва метавонанд ба осонӣ навсозӣ ва идора карда шаванд.

## Навъҳои дастгиришавандаи меҳмондорӣ {#supported-hospitality-types}

- **Тарабхонаҳо** — менюҳои хӯрокхӯрии пуррахизмат
- **Қаҳвахонаҳо** — менюҳои қаҳва ва таомҳои сабук
- **Барҳо ва толорҳои истироҳат** — менюҳои нӯшокиҳо ва пешхӯракҳо
- **Нонвойхонаҳо** — менюҳои шириниҳои хамирӣ ва нон
- **Хизматрасониҳои кейтеринг** — имконоти меню барои чорабиниҳо
- **Фургонҳои ғизо** — менюҳои хизматрасонии сайёри ғизо
- **Заводҳои пиво ва шаробсозӣ** — менюҳои нӯшокиҳо бо тавсифҳо

## Сохтори меню {#menu-structure}

### Категорияҳои меню {#menu-categories}

Менюҳо ба категорияҳо ташкил карда мешаванд:

- **Пешхӯракҳо** — оғозкунандаҳо ва порсияҳои хурд
- **Таомҳои асосӣ** — хӯрокҳои асосӣ
- **Гарнирҳо** — иловаҳо ва сабзавот
- **Шириниҳо** — лаззатҳои ширин
- **Нӯшокиҳо** — нӯшокиҳо (спиртӣ ва ғайриспиртӣ)
- **Пешниҳодҳои махсус** — пешниҳодҳои ҳаррӯза ё мавсимӣ

### Формати банди меню {#menu-item-format}

Ҳар як банди меню дар бар мегирад:

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### Майдонҳои банд {#item-fields}

| Майдон | Навъ | Тавсиф |
|-------|------|-------------|
| `name` | string | Номи таом ё нӯшокӣ |
| `description` | string | Тавсифи муфассали банд |
| `price` | string | Нарх (бо асъор форматшуда) |
| `dietary_info` | array | Хусусиятҳои парҳезӣ (Vegan, Gluten-Free ва ғайра) |
| `allergens` | array | Аллергенҳои маъмул (чормағзҳо, маҳсулоти баҳрии садафдор ва ғайра) |
| `availability` | string | Кай дастрас аст (ҳаррӯза, мавсимӣ ва ғайра) |

## Эҷоди менюҳои меҳмондорӣ {#creating-hospitality-menus}

### Қадами 1: Пешниҳоди маълумоти меню {#step-1-provide-menu-information}

Ба Theme Builder дар бораи менюи худ маълумот диҳед:

```
Барои тарабхонаи итолиёвии ман менюи тарабхона бо пешхӯракҳо,
таомҳои паста, таомҳои асосӣ ва шириниҳо эҷод кунед. Нархҳо ва тавсифҳоро дохил кунед.
```

### Қадами 2: Theme Builder менюро эҷод мекунад {#step-2-theme-builder-generates-menu}

Theme Builder:

1. Саҳифаи менюи сохторёфта эҷод мекунад
2. Онро тавре тарҳрезӣ мекунад, ки ба мавзӯи сомонаи шумо мувофиқат кунад
3. Бандҳоро ба категорияҳо ташкил мекунад
4. Нархҳо ва тавсифҳоро формат мекунад
5. Маълумоти парҳезӣ ва аллергенҳоро илова мекунад

### Қадами 3: Баррасӣ ва фармоишӣ кардан {#step-3-review-and-customize}

Шумо метавонед:

1. Бандҳои меню ва нархҳоро таҳрир кунед
2. Категорияҳоро илова ё хориҷ кунед
3. Бандҳоро дар дохили категорияҳо аз нав тартиб диҳед
4. Тавсифҳо ва маълумоти парҳезиро навсозӣ кунед
5. Услуб ва ҷойгиркуниро танзим кунед

## Имконоти намоиши меню {#menu-display-options}

### Саҳифаи пурраи меню {#full-menu-page}

Саҳифаи махсусе, ки менюи пурраи шуморо намоиш медиҳад:

- Аз рӯи категория ташкилшуда
- Қобили ҷустуҷӯ ва филтркунӣ
- Ҷойгиркунии мувофиқ барои чоп
- Тарҳи мутобиқ ба мобилӣ

### Виджети меню {#menu-widget}

Бахшҳои менюро дар саҳифаҳои дигар ҷойгир кунед:

- Бандҳои баргузида дар саҳифаи асосӣ
- Пешниҳодҳои ҳаррӯза дар панели канорӣ
- Менюи нӯшокиҳо дар саҳифаи бар
- Намоиши шириниҳо дар поёнсаҳифа

### PDF-и меню {#menu-pdf}

Менюи PDF-и қобили боргирӣ эҷод кунед:

- Форматкунии касбӣ
- Сифати омода барои чоп
- Тасвирҳо ва тавсифҳоро дар бар мегирад
- Барои мубодила ва фиристодан бо email осон

## Маълумоти парҳезӣ ва аллергенҳо {#dietary-and-allergen-information}

### Хусусиятҳои парҳезӣ {#dietary-attributes}

Бандҳоро бо маълумоти парҳезӣ қайд кунед:

- **Vegan** — бе маҳсулоти ҳайвонот
- **Гиёҳхорӣ** — бе гӯшт
- **Gluten-Free** — барои бемории селиак бехатар
- **Бе маҳсулоти ширӣ** — бе маҳсулоти шир
- **Бе чормағз** — бе чормағзҳои дарахтӣ ё арахис
- **Камкарбогидрат** — карбогидратҳои камшуда
- **Сафедаи баланд** — бой аз сафеда

### Огоҳии аллергенҳо {#allergen-warnings}

Аллергенҳои маъмулро дохил кунед:

- **Чормағзҳо** — чормағзҳои дарахтӣ ва арахис
- **Маҳсулоти баҳрии садафдор** — харчангшаклон ва моллюскҳо
- **Моҳӣ** — ҳамаи намудҳои моҳӣ
- **Маҳсулоти ширӣ** — шир ва маҳсулоти шир
- **Тухмҳо** — тухми мурғ
- **Соя** — маҳсулоти соя
- **Gluten** — гандум ва ғалладонаҳои дорои gluten
- **Кунҷид** — тухмӣ ва равғани кунҷид

## Идоракунии меню {#menu-management}

### Навсозии нархҳо {#updating-prices}

Нархҳои менюро ба осонӣ навсозӣ кунед:

1. Ба саҳифаи меню гузаред
2. "Таҳрири меню"-ро клик кунед
3. Нархҳои бандҳоро навсозӣ кунед
4. Тағйиротро захира кунед
5. Тағйирот фавран дар сомонаи шумо пайдо мешаванд

### Илова кардани бандҳои мавсимӣ {#adding-seasonal-items}

Вариантҳои мавсимии меню эҷод кунед:

1. Нусхаи нави меню эҷод кунед
2. Бандҳои мавсимиро илова кунед
3. Бандҳоро ҳамчун "Мавсимӣ" қайд кунед
4. Санаҳои дастрасиро ба нақша гиред
5. Дар мавсим ба таври худкор намоиш диҳед

### Идоракунии пешниҳодҳои махсус {#managing-specials}

Пешниҳодҳои махсуси ҳаррӯза ё ҳафтаинаро намоиш диҳед:

1. Категорияи "Пешниҳодҳои махсус" эҷод кунед
2. Бандҳоро бо санаҳои дастрасӣ илова кунед
3. Пешниҳодҳои махсусро дар саҳифаи асосӣ таъкид кунед
4. Ҳар рӯз ё ҳар ҳафта навсозӣ кунед
5. Пешниҳодҳои махсуси кӯҳнаро бойгонӣ кунед

## Ҳамгироӣ бо Theme Builder {#integration-with-theme-builder}

Ҳангоми истифодаи Theme Builder барои сомонаҳои меҳмондорӣ:

1. **Шинохти худкори меню** — муайян мекунад, ки оё шумо тиҷорати ғизо/нӯшокӣ ҳастед
2. **Эҷоди саҳифаи меню** — саҳифаҳои менюи касбӣ эҷод мекунад
3. **Мутобиқати тарҳ** — менюҳо ба мавзӯи сомонаи шумо мувофиқат мекунанд
4. **Оптимизатсияи мобилӣ** — менюҳо дар телефонҳо зебо намоиш дода мешаванд
5. **Оптимизатсияи SEO** — менюҳо барои муҳаррикҳои ҷустуҷӯ мувофиқанд

## Таҷрибаҳои беҳтарин {#best-practices}

### Тарҳи меню {#menu-design}

- **Ташкили равшан** — сохтори мантиқии категорияҳо
- **Тавсифҳои хонданӣ** — иштиҳоовар ва иттилоотӣ
- **Нархгузории яксон** — асъор ва форматгузории равшан
- **Аксҳои касбӣ** — тасвирҳои ғизо бо сифати баланд
- **Фазои холӣ** — саҳифаро аз ҳад зиёд пур накунед

### Муҳтаво {#content}

- **Тавсифҳои дақиқ** — бандҳоро дақиқ тавсиф кунед
- **Таъкид кардани пешниҳодҳои махсус** — пешниҳодҳои махсусро намоён кунед
- **Дохил кардани аллергенҳо** — ҳамеша аллергенҳоро рӯйхат кунед
- **Мунтазам навсозӣ кардан** — нархҳо ва бандҳоро ҷорӣ нигоҳ доред
- **Истифодаи забони иштиҳоовар** — бандҳоро болаззат садо диҳед

### Дастрасӣ {#accessibility}

- **Ҳуруфҳои хонданӣ** — аз ҳуруфҳои равшан ва осонхон истифода баред
- **Контрасти кофӣ** — боварӣ ҳосил кунед, ки матн хонданист
- **Нишонаҳои парҳезӣ** — имконоти парҳезиро равшан қайд кунед
- **Огоҳии аллергенҳо** — аллергенҳоро ба таври намоён нишон диҳед
- **Мутобиқ ба мобилӣ** — дар ҳамаи дастгоҳҳо санҷед
