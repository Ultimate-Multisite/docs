# Абзорҳо ва танзимот {#tools-and-configuration}

Theme Builder маҷмӯи абзорҳоро барои таҳлил, тарҳрезӣ ва сохтани сайти WordPress-и шумо истифода мебарад. Дар версияи 1.16.0, **sd-ai-agent/site-scrape акнун абзори Tier 1 аст**, ки онро дар Theme Builder ба таври пешфарз дастрас мекунад. Superdav AI Agent v1.18.0 инчунин бастаи Advanced companion-ро, ки ҷудогона паҳн мешавад, барои равандҳои кории таҳиягарони боэътимод муаррифӣ мекунад; ин абзорҳо қисми бастаи асосии WordPress.org нестанд ва бояд ҷудогона насб ва иҷозат дода шаванд.

## Абзорҳои дастрас {#available-tools}

### Абзорҳои Tier 1 (ҳамеша дастрас) {#tier-1-tools-always-available}

Абзорҳои Tier 1 дар Theme Builder бе ягон танзимоти иловагӣ ба таври пешфарз дастрасанд.

#### sd-ai-agent/site-scrape {#sd-ai-agentsite-scrape}

**Мақсад:** Таҳлили вебсайтҳои мавҷуда барои гирифтани илҳоми тарроҳӣ, сохтори муҳтаво ва коркард.

**Қобилиятҳо:**

- **Таҳлили вебсайт** — ҷамъоварӣ ва таҳлили вебсайтҳои рақибон ё илҳомбахш
- **Истихроҷи тарроҳӣ** — муайян кардани рангҳо, шрифтҳо ва намунаҳои ҷойгиркунӣ
- **Сохтори муҳтаво** — фаҳмидани созмондиҳӣ ва иерархияи саҳифа
- **Муайянкунии хусусиятҳо** — муайян кардани plugin-ҳо ва коркард
- **Таҳлили иҷроиш** — санҷиши суръати саҳифа ва беҳинасозӣ
- **Таҳлили SEO** — баррасии мета-тегҳо ва додаҳои сохторёфта

**Истифода:**

```
Тарроҳии example.com-ро таҳлил кунед ва онро ҳамчун илҳом барои сайти ман истифода баред.
```

**Он чӣ истихроҷ мекунад:**

- Палитраи рангҳо ва истифодаи рангҳо
- Типография (шрифтҳо ва андозаҳо)
- Сохтори ҷойгиркунӣ ва фосилагузорӣ
- Намунаҳои паймоиш
- Созмондиҳии муҳтаво
- Истифодаи тасвирҳо ва расонаҳо
- Унсурҳои интерактивӣ
- Мутобиқшавӣ ба мобилӣ

**Маҳдудиятҳо:**

- Сайтҳои бо парол муҳофизатшударо ҷамъоварӣ карда наметавонад
- robots.txt ва сиёсатҳои сайтро риоя мекунад
- Шояд муҳтавои динамикиро сабт накунад
- Сайтҳои дорои JavaScript-и зиёд метавонанд таҳлили маҳдуд дошта бошанд
- Таҳлили сайтҳои калон метавонад вақти бештар гирад

### Абзорҳои Tier 2 (ихтиёрӣ) {#tier-2-tools-optional}

Абзорҳои Tier 2 вақте дастрасанд, ки дар танзимоти Theme Builder ба таври возеҳ фаъол карда шаванд.

#### Таҳлилҳои пешрафта {#advanced-analytics}

Метрикаҳои муфассали иҷроишро пешниҳод мекунад:

- Вақтҳои боршавии саҳифа
- Core Web Vitals
- Холи SEO
- Холи дастрасӣ
- Холи беҳтарин амалияҳо

#### Беҳинасози муҳтаво {#content-optimizer}

Таҳлил мекунад ва беҳбудиҳоро пешниҳод мекунад барои:

- Хондашавандагӣ
- Беҳинасозии SEO
- Истифодаи калидвожаҳо
- Сохтори муҳтаво
- Ҷойгиркунии даъват ба амал

### Абзорҳои Tier 3 (Premium) {#tier-3-tools-premium}

Абзорҳои Tier 3 танзимоти иловагӣ ё хусусиятҳои premium талаб мекунанд.

#### Генератори муҳтавои AI {#ai-content-generator}

Муҳтаво тавлид мекунад барои:

- Тавсифҳои маҳсулот
- Саҳифаҳои хизматрасонӣ
- Навиштаҳои блог
- Тавсифҳои мета
- Матни даъват ба амал

#### Абзорҳои тарроҳии пешрафта {#advanced-design-tools}

Қобилиятҳои пешрафтаи тарроҳиро пешниҳод мекунад:

- Тавлиди CSS-и фармоишӣ
- Эҷоди аниматсия
- Тарроҳии унсури интерактивӣ
- Назарияи пешрафтаи ранг
- Беҳинасозии типография

### Абзорҳои Advanced Companion (бастаи ҷудогона) {#advanced-companion-tools-separate-package}

Бастаи Advanced companion ҷудо аз нашри асосии Superdav AI Agent паҳн мешавад. Он барои муҳитҳои таҳиягарон ва соҳибони сайтҳои боэътимод пешбинӣ шудааст, ки дар он маъмурон ба таври возеҳ ба агент иҷозат медиҳанд, ки аз абзорҳои таъсири баланд истифода барад.

Қобилиятҳои Advanced companion метавонанд инҳоро дар бар гиранд:

- **Абзорҳои системаи файлӣ** — санҷиш ва тағйири файлҳои тасдиқшуда дар муҳитҳои боэътимод
- **Абзорҳои пойгоҳи додаҳо** — дархост ё навсозии додаҳои сайт ҳангоми иҷозати возеҳ
- **Абзорҳои WP-CLI** — иҷрои фармонҳои нигоҳдорӣ ва санҷиши WordPress
- **Абзорҳои фиристандаи REST** — даъвати endpoint-ҳои сабтшудаи REST тавассути равандҳои кории фиристодани назоратшаванда
- **Абзорҳои созандаи plugin** — сохтани қолаб ва такроркорӣ бар коди plugin
- **Абзорҳои snapshot-и Git** — эҷоди snapshot-ҳо пеш аз амалҳои хатарноки таҳиягар
- **Абзорҳои идоракунии корбарон** — кумак дар баррасӣ ё танзими сабтҳои корбарон дар ҷойҳое, ки иҷозат дода шудааст
- **Абзорҳои benchmark** — иҷрои benchmark-ҳои иҷроиш ё қобилият барои баррасии рушд

Инҳоро ҳамчун абзорҳои Theme Builder, ки барои ҳар як насб умуман дастрасанд, ҳуҷҷатгузорӣ накунед. Онҳо танҳо вақте дастрасанд, ки бастаи Advanced companion насб, фаъол ва ба маъмурони боэътимод маҳдуд карда шуда бошад.

## Танзими абзорҳо {#tool-configuration}

### Фаъол кардани абзорҳо {#enabling-tools}

Барои фаъол кардани абзорҳои иловагӣ дар Theme Builder:

1. Ба **Gratis AI Agent → Танзимот** гузаред
2. Ба **Theme Builder → Абзорҳо** равед
3. Абзорҳоро мувофиқи зарурат фаъол/ғайрифаъол кунед
4. Танзимотро захира кунед

### Иҷозатҳои абзорҳо {#tool-permissions}

Баъзе абзорҳо иҷозат талаб мекунанд:

- **Ҷамъоварии сайт** — дастрасӣ ба интернет талаб мекунад
- **Таҳлилҳо** — ҳамгироии Google Analytics талаб мекунад
- **Тавлиди муҳтаво** — калидҳои API талаб мекунад
- **Хусусиятҳои пешрафта** — метавонанд обунаи premium талаб кунанд
- **Абзорҳои Advanced companion** — бастаи Advanced-и ҷудогона паҳншаванда ва эътимоди возеҳи маъмурро талаб мекунанд

### Калидҳои API ва маълумоти эътиборӣ {#api-keys-and-credentials}

Калидҳои API-ро барои абзорҳое танзим кунед, ки онҳоро талаб мекунанд:

1. Ба **Gratis AI Agent → Танзимот → Калидҳои API** равед
2. Барои ҳар як абзор маълумоти эътибориро ворид кунед
3. Пайвастшавиро санҷед
4. Ба таври бехатар захира кунед

### Насби бастаи Advanced Companion {#installing-the-advanced-companion-package}

Бастаи Advanced companion ҷудо аз ZIP-и асосии WordPress.org бароварда мешавад. Онро танҳо аз канали расмии паҳнкунии лоиҳа барои нашр насб кунед, сипас дастрасиро ба маъмурони боэътимод маҳдуд кунед. Пеш аз фаъол кардани абзорҳои системаи файлӣ, пойгоҳи додаҳо, WP-CLI, фиристандаи REST, созандаи plugin, идоракунии корбарон ё benchmark дар сайти истеҳсолӣ, дарвозаҳои тасдиқи инсонӣ ва рафтори snapshot-и git-ро баррасӣ кунед.

## Истифодаи sd-ai-agent/site-scrape {#using-sd-ai-agentsite-scrape}

### Истифодаи асосӣ {#basic-usage}

Аз Theme Builder хоҳиш кунед, ки вебсайтро таҳлил кунад:

```
Тарроҳии сайти рақиби маро дар competitor.com таҳлил кунед
ва барои сайти ман беҳбудиҳои тарроҳиро пешниҳод кунед.
```

### Таҳлили мушаххас {#specific-analysis}

Навъҳои мушаххаси таҳлилро дархост кунед:

```
Палитраи рангҳоро аз example.com истихроҷ кунед ва онро ҳамчун илҳом истифода баред.
```

```
Сохтори паймоиши example.com-ро таҳлил кунед ва созмондиҳии монандро
ба сайти ман татбиқ кунед.
```

```
Мутобиқшавии мобилии example.com-ро санҷед ва итминон ҳосил кунед, ки сайти ман
ба ҳамон андоза мутобиқшаванда аст.
```

### Таҳлили муқоисавӣ {#comparison-analysis}

Якчанд сайтро муқоиса кунед:

```
Тарроҳиҳои site1.com ва site2.com-ро муқоиса кунед ва тарроҳие эҷод кунед,
ки беҳтарин унсурҳои ҳар дуро муттаҳид мекунад.
```

## Маҳдудиятҳо ва мулоҳизаҳои абзорҳо {#tool-limitations-and-considerations}

### Маҳдудкунии суръат {#rate-limiting}

- Scraping барои пешгирии серборшавии сервер бо маҳдудияти суръат анҷом дода мешавад
- Дархостҳои сершумор ба ҳамон site метавонанд маҳдуд карда шаванд
- Site-ҳои калон метавонанд барои таҳлил вақти бештар гиранд

### Дақиқии муҳтаво {#content-accuracy}

- Муҳтавои динамикӣ метавонад пурра сабт нашавад
- Муҳтавое, ки бо JavaScript намоиш дода мешавад, метавонад нопурра бошад
- Додаҳои вақти воқеӣ метавонанд инъикос нашаванд
- Баъзе муҳтаво метавонад паси пардохтдеворҳо бошад

### Мулоҳизаҳои ҳуқуқӣ ва ахлоқӣ {#legal-and-ethical-considerations}

- robots.txt ва сиёсатҳои site-ро эҳтиром кунед
- Муҳтавои ҳифзшуда бо ҳуқуқи муаллифро барои бознашр scrape накунед
- Таҳлилро барои илҳом истифода баред, на барои нусхабардорӣ
- Санҷед, ки барои истифодаи муҳтавои истихроҷшуда ҳуқуқ доред
- Шартҳои хизматрасонии site-ҳои таҳлилшударо риоя кунед

### Таъсир ба иҷроиш {#performance-impact}

- Scraping-и site-ҳои калон метавонад вақт гирад
- Scraping-ҳои ҳамзамони сершумор метавонанд сусттар бошанд
- Пайвастшавии шабака ба суръат таъсир мерасонад
- Маҷмӯаҳои калони додаҳо метавонанд коркарди бештар талаб кунанд

## Беҳтарин амалияҳо {#best-practices}

### Истифодаи таҳлили site {#using-site-analysis}

1. **Site-ҳои сершуморро таҳлил кунед** — аз чанд манбаъ илҳом гиред
2. **Ба сохтор диққат диҳед** — намунаҳои тарҳбандӣ ва ташкилро омӯзед
3. **Рангҳоро истихроҷ кунед** — палитраҳои рангро ҳамчун нуқтаҳои оғоз истифода баред
4. **Типографияро омӯзед** — омезишҳои ҳуруферо, ки писандатонанд, муайян кунед
5. **Навигатсияро баррасӣ кунед** — ташкилёбии менюро фаҳмед

### Scraping-и ахлоқӣ {#ethical-scraping}

1. **robots.txt-ро эҳтиром кунед** — дастурҳои site-ро риоя кунед
2. **Муҳтаворо нусхабардорӣ накунед** — таҳлилро танҳо барои илҳом истифода баред
3. **Ҳуқуқҳоро санҷед** — боварӣ ҳосил кунед, ки метавонед муҳтавои истихроҷшударо истифода баред
4. **Манбаъҳоро нишон диҳед** — ба манбаъҳои илҳом эътибор диҳед
5. **Шартҳоро риоя кунед** — шартҳои хизматрасонии site-ро иҷро кунед

### Ба ҳадди аксар расонидани самаранокии абзор {#maximizing-tool-effectiveness}

1. **Мушаххас бошед** — навъҳои мушаххаси таҳлилро дархост кунед
2. **Контекст диҳед** — ҳадафи site-и худро шарҳ диҳед
3. **Интизориҳоро муқаррар кунед** — ҳадафҳои тарҳи худро тавсиф кунед
4. **Такрор кунед** — бар асоси натиҷаҳо такмил диҳед
5. **Абзорҳоро якҷо истифода баред** — барои таҳлили ҳамаҷониба аз абзорҳои сершумор истифода баред

## Ҳалли мушкилот {#troubleshooting}

### Site Scrape намешавад {#site-wont-scrape}

- Санҷед, ки site барои умум дастрас аст ё не
- Боварӣ ҳосил кунед, ки robots.txt scraping-ро иҷозат медиҳад
- Site-и дигарро санҷед
- Пайвасти интернетро санҷед
- Агар мушкил идома ёбад, бо дастгирӣ тамос гиред

### Таҳлили нопурра {#incomplete-analysis}

- Site метавонад муҳтавои динамикӣ дошта бошад
- JavaScript метавонад пурра намоиш дода нашавад
- Site-ҳои калон метавонанд аз вақт берун шаванд
- Ба ҷои ин, саҳифаҳои мушаххасро таҳлил кунед
- Навъҳои мушаххаси таҳлилро дархост кунед

### Иҷроиши суст {#slow-performance}

- Site-ҳои калон барои таҳлил вақти бештар мегиранд
- Scraping-ҳои ҳамзамони сершумор сусттаранд
- Пайвастшавии шабака ба суръат таъсир мерасонад
- Дар соатҳои камбор санҷед
- Аввал бахшҳои хурдтарро таҳлил кунед
