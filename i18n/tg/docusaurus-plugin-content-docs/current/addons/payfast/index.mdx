# PayFast

Ultimate Multisite: Payfast Gateway

Payfast — шлюзи зуд ва бехатари пардохтҳои онлайн дар Африқои Ҷанубӣ —-ро ҳамчун имкони пардохти Ultimate Multisite илова мекунад.

## Насб

1. 'ultimate-multisite-payfast'-ро ба директорияи '/wp-content/plugins/' боргузорӣ кунед
2. Плагинро тавассути менюи 'Plugins' дар WordPress фаъол кунед
3. Барои танзими плагин Wizard-и қадам ба қадамро пайравӣ кунед

## Дастури истифода

PayFast як коркардкунандаи пешсафи пардохтҳои онлайн барои тиҷоратҳо дар Африқои Ҷанубӣ мебошад. Он барои кӯмак ба тоҷирони онлайн дар рушд бахшида шудааст, то амалиётҳоро осон ва бехатар кунад. Бо дархости корбари Ultimate Multisite, мо қарор додем иловагие эҷод кунем, ки ҳамгироиро бо PayFast имконпазир месозад. Ин мақола нишон медиҳад, ки чӣ гуна PayFast-ро ҳамчун шлюзи пардохти худ танзим кунед, то дар шабакаи Ultimate Multisite-и худ пардохт қабул намоед.

Қадами аввал насб кардани илова аст.

# Чӣ гуна иловаи Payfast-ро насб кардан мумкин аст:

Ба саҳифаи иловаҳо гузаред ва иловаи Ultimate Multisite: Payfast Gateway-ро интихоб кунед.

![Саҳифаи иловаҳо](/img/admin/addons-page.png)

Равзанаи илова пайдо мешавад. Ба Install Now клик кунед.

![Саҳифаи иловаҳо бо имкони насб](/img/admin/addons-page.png)

Пас аз анҷоми насб, шумо ба саҳифаи плагинҳо равона карда мешавед. Дар ин ҷо танҳо ба Network Activate клик кунед ва иловаи _Payfast_ дар шабакаи шумо фаъол мешавад.

# Танзими PayFast

Пас аз насби илова, акнун метавонед ба танзимоти Ultimate Multisite гузаред ва танзими PayFast-ро оғоз кунед.

Дар зери саҳифаи **танзимоти Ultimate Multisite**, шумо метавонед ҷадвали "Пардохтҳо"-ро пайдо кунед. Дар зери ин ҷадвал, шумо бояд "**Payfast** "-ро ҳамроҳ бо тугмаи гузариш барои фаъол ё ғайрифаъол кардани ин шлюзи пардохт бубинед.

![Танзимоти шлюзи пардохт](/img/config/settings-payment-gateways.png)Ба пеш рафта, тугмаи гузаришро клик кунед, то онро фаъол созед. Сипас он қуттиро васеъ мекунад, то майдонҳои гуногунеро нишон диҳад, ки шумо бояд пур кунед, ба монанди **Mechant ID, Merchant Key ва PassPhrase.** Шумо метавонед ин маълумотро аз Account-и **Payfast**-и худ гиред

<!-- Скриншот дастнорас аст: танзимоти васеъшудаи PayFast, ки майдонҳои Merchant ID, Merchant Key ва PassPhrase-ро нишон медиҳад -->

Қадами навбатӣ ворид шудан ба Account-и PayFast-и худ барои гирифтани маълумоти лозим аст.

Пас аз ворид шудан, шумо метавонед фавран эътиборномаҳои лозимиро аз dashboard-и Account-и **Payfast**-и худ бинед.

<!-- Скриншот дастнорас аст: dashboard-и Account-и PayFast, ки эътиборномаҳои Merchant ID ва Merchant Key-ро нишон медиҳад -->Шумо метавонед эътиборномаро оддӣ нусха гирифта гузоред ва ҷузъиёти онро дар танзимоти Ultimate Multisite-и худ пур кунед. Сипас тугмаи **«Захира кардани танзимот** »-ро клик кунед, то тағйирот захира шаванд.

<!-- Скриншот дастнорас аст: танзимоти PayFast-и Ultimate Multisite бо эътиборномаҳои пуршуда ва тугмаи захира -->
