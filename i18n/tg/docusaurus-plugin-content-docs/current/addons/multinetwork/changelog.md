# Тағйиротномаи Multi-Network

### 1.3.0 {#130}
* Нав: Пешнамоишгари қолаби шабака — қолабҳои шабакаро пеш аз харид бо панели зинда аз назар гузаронед ва пешнамоиш кунед.
* Нав: Интихобгари қолаби шабака ҳангоми checkout — муштариён метавонанд дар ҷараёни signup checkout қолаби шабакаро интихоб кунанд.
* Нав: Cross-network magic link SSO — муштариён метавонанд тавассути аутентификатсияи magic-link ба зер-шабакаҳо бе мушкил дастрасӣ пайдо кунанд.
* Нав: Клонкунии медиаи шабака — медиаи сайт ҳангоми клон кардани шабакаҳо аз қолаб дуруст такрор карда мешавад.
* Ислоҳ: Клонкунии шабака акнун бе addon-и multi-tenancy кор мекунад; имтиёзҳои super admin баъд аз клон дубора дода мешаванд.
* Ислоҳ: Эҷоди шабака акнун бо WooCommerce ва ҳамаи дарвозаҳои пардохт мувофиқ аст.
* Ислоҳ: Рақами порт акнун ба тавлиди домени эҳтиётӣ дохил карда мешавад.
* Ислоҳ: CSS-и интихобгари қолаб ба stylesheet-и enqueued интиқол дода шуд, то дар ҳамаи theme-ҳо дуруст намоиш дода шавад.
* Ислоҳ: Муҳофизат аз калиди таърифнашудаи `network_id` дар доираи дархости сайт.
* Ислоҳ: Санҷиши иҷозати шабака пеш аз ивазкунии контекст хатогиҳои хусусияти ҷадвали `wpdb`-ро пешгирӣ мекунад.
