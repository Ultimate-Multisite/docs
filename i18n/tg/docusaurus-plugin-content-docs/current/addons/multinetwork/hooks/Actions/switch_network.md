# Амал: switch_network {#action-switchnetwork}

Ҳангоми иваз шудани контексти шабакаи ҷорӣ иҷро мешавад.

## Параметрҳо {#parameters}

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $new_network_id | `int` | ID-и шабакае, ки ба он иваз карда мешавад. |
| $old_network_id | `int` | ID-и шабакаи қаблан ҷорӣ. |
