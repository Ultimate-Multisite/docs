# Action: move_site

Пас аз он иҷро мешавад, ки сайт ба шабакаи нав кӯчонида шудааст.

## Параметрҳо {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $site_id | `int` | ID-и сайте, ки кӯчонида шудааст. |
| $old_network_id | `int` | ID-и шабакаи аслӣ барои сайт. |
| $new_network_id | `int` | ID-и шабакае, ки сайт ба он кӯчонида шудааст. |

### Аз {#since}

- 1.3.0
