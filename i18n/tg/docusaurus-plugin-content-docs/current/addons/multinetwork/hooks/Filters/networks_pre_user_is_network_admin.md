# Филтр: networks_pre_user_is_network_admin

Шабакаҳоеро, ки корбар маъмури онҳо аст, филтр мекунад, то равандро кӯтоҳ кунад.

## Параметрҳо {#parameters}

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $ | `array\|bool\|null` | Рӯйхати ID-ҳои шабака ё false. Ҳар чизе ғайр аз null равандро кӯтоҳ мекунад. |
| $ | `int` | ID-и корбар, ки барои он шабакаҳо бояд баргардонида шаванд. |
