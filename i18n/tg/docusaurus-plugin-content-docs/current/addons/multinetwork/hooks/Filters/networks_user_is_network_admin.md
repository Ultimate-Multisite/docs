| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $ | `array\|bool` | Рӯйхати ID-ҳои шабака ё false, агар барои корбар шабакае набошад. |
| $ | `int` | ID-и корбар, ки шабакаҳо бояд барои он баргардонида шаванд. |
