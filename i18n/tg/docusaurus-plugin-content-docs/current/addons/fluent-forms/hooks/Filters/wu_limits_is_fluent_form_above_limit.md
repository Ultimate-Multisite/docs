# Филтр: wu_limits_is_fluent_form_above_limit

Месанҷад, ки оё навъи додашудаи форма дар ин plan иҷозат дода шудааст ё не. Ба таҳиягарони plugin имкон медиҳад, ки арзиши баргардондашавандаро филтр кунанд

## Параметрҳо {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $value | `bool` | Оё навъи форма аз маҳдудият боло аст ё не |
| $form_type | `string` | Навъи формае, ки санҷида мешавад |
| $form_count | `int` | Шумораи ҷории формаҳо |
| $quota | `int` | Квотаи иҷозатдодашуда |
