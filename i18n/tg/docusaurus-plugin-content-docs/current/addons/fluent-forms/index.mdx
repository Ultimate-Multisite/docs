# Fluent Forms {#fluent-forms}

Ин add-on имкон медиҳад, ки барои Fluent Forms дар Multisite Ultimate дар маҳсулотҳо, узвиятҳо ё сомонаҳо маҳдудиятҳо илова карда шаванд.
Fluent Forms ба ҷойи навъҳои фармоишии навиштаҳо ҷадвалҳои фармоиширо истифода мебарад, бинобар ин имконоти муқаррарии маҳдудсозӣ бо Fluent Forms кор намекунанд.

## Хусусиятҳо {#features}

- Илова кардани маҳдудиятҳо барои Fluent Forms ё Conversational Fluent Forms
- Муштариён ҳангоми расидан ба маҳдудият паёми дӯстона мегиранд, ки онҳоро ба навсозии нақшаашон даъват мекунад
