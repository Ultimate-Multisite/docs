# Бисёрасъорӣ

Ultimate Multisite: Бисёрасъорӣ ба шабакаи шумо имкон медиҳад, ки пардохтҳоро бо чандин асъор қабул кунад. Он нархҳоро бо истифода аз қурбҳои зинда ё дастӣ ба таври худкор табдил медиҳад, ба шумо иҷозат медиҳад барои ҳар асъор нархҳои дақиқи ҳар маҳсулотро таъин кунед ва асъори афзалиятноки ҳар меҳмонро тавассути геолокатсия муайян мекунад — ҳама аз як Dashboard-и маъмурӣ идора мешавад.

## Хусусиятҳои асосӣ {#key-features}

- **Табдили худкори қурби асъор** — нархҳо аз асъори асосии шумо бо истифода аз қурбҳое, ки аз провайдери интихобкардаи шумо гирифта мешаванд, табдил дода мешаванд
- **Бознависи нархгузорӣ барои ҳар маҳсулот** — барои ҳар маҳсулот нархҳои дақиқи махсуси асъорро таъин кунед, то табдили худкорро гузаред
- **Майдони интихоби асъор ҳангоми сабтином** — ба муштариён имкон медиҳад, ки ҳангоми пардохт/сабтином асъори афзалиятноки худро интихоб кунанд
- **Чандин провайдери қурби асъор** — ECB, Frankfurter, Stripe ё қурбҳои пурра дастӣ
- **Муайянкунии асъор бар асоси геолокатсия** — асъореро, ки ба кишвари меҳмон мувофиқ аст, ба таври худкор пешакӣ интихоб мекунад
- **Dashboard-и маъмурӣ** — асъорҳои фаъол, қурбҳои асъор ва танзимоти провайдерро аз як ҷой идора кунед

## Талабот {#requirements}

- Ultimate Multisite 2.0.0 ё болотар
- Насби WordPress Multisite
- PHP 7.4 ё болотар

## Насб {#installation}

1. Файлҳои плагинро ба `/wp-content/plugins/ultimate-multisite-multi-currency` боргузорӣ кунед
2. Плагинро тавассути экрани **Плагинҳо** дар WordPress фаъол кунед (Фаъолсозии шабакавӣ)
3. Ба **Ultimate Multisite → Бисёрасъорӣ** гузаред, то Dashboard-и маъмуриро кушоед
4. Асъорҳоеро, ки мехоҳед қабул кунед, илова намоед ва провайдери қурби асъори худро танзим кунед

## Танзимот {#configuration}

### Dashboard-и маъмурӣ {#admin-dashboard}

Dashboard-и маъмурии Бисёрасъорӣ (**Ultimate Multisite → Бисёрасъорӣ**) ҷойи марказӣ барои ин корҳост:

- Илова, фаъол ё ғайрифаъол кардани асъорҳо
- Интихоби провайдери қурби асъор ва навсозии қурбҳо ҳангоми дархост
- Баррасии қурбҳои ҷорӣ барои ҳамаи асъорҳои фаъол
- Таъини қурбҳои дастӣ, вақте ки провайдерҳои худкор мувофиқ нестанд

### Провайдерҳои қурби асъор {#exchange-rate-providers}

| Провайдер | Тавсиф |
|---|---|
| **ECB** | Бонки марказии Аврупо — қурбҳои ройгони ҳаррӯза барои шабакаҳои бар EUR асосёфта |
| **Frankfurter** | API-и оинавии ECB бо сарчашмаи кушода — ройгон, бе зарурати калиди API |
| **Stripe** | Қурбҳои зиндаи FX-и Stripe-ро истифода мебарад — ҳамгироии фаъоли Stripe талаб мешавад |
| **Дастӣ** | Шумо қурбҳоро мустақим ворид мекунед; даъватҳои API-и беруна нест |

Провайдери худро дар **Ultimate Multisite → Бисёрасъорӣ → Танзимот → Провайдери қурби асъор** интихоб кунед. Қурбҳоро дар ҳар вақт дастӣ нав кардан мумкин аст ё барои навсозии худкор ба ҷадвал гузоштан мумкин аст.

### Бознависи нархгузорӣ барои ҳар маҳсулот {#per-product-pricing-overrides}

Вақте ки табдили худкор ба қадри кофӣ дақиқ нест, шумо метавонед барои асъори мушаххас дар ҳар маҳсулот нархи дақиқ таъин кунед:

1. Муҳаррири маҳсулотро кушоед (**Ultimate Multisite → Маҳсулот → Таҳрир**)
2. Ба бахши **Нархгузории асъор** поён ҳаракат кунед
3. Барои ҳар асъоре, ки мехоҳед бознависӣ кунед, нархи дақиқро ворид кунед
4. Маҳсулотро захира кунед

Маҳсулоте, ки бознависӣ надоранд, нархи табдилёфтаро аз асъори асосии шумо истифода мебаранд.

### Майдони интихоби асъор ҳангоми сабтином {#currency-selector-signup-field}

Майдони интихоби асъор метавонад ба ҷараёни сабтином/пардохти шумо илова карда шавад, то муштариён ҳангоми бақайдгирӣ асъори афзалиятноки худро интихоб кунанд:

1. Ба **Ultimate Multisite → Бисёрасъорӣ → Танзимот** гузаред
2. **Намоиш додани интихобкунандаи асъор ҳангоми сабтином**-ро фаъол кунед
3. Ихтиёрӣ, асъори пешфарзро таъин кунед ё иҷозат диҳед, ки геолокатсия арзиши пешакӣ интихобшударо муайян кунад

Асъори интихобшуда бо обуна нигоҳ дошта мешавад ва барои ҳамаи ҳисоббарориҳои минбаъда истифода мегардад.

### Муайянкунии асъор бар асоси геолокатсия {#geolocation-based-currency-detection}

Вақте ки муайянкунии геолокатсия фаъол аст, addon кишвари меҳмонро ба асъор мувофиқ мекунад ва онро дар майдони интихоби асъор пешакӣ интихоб менамояд. Барои фаъол кардан:

1. Ба **Ultimate Multisite → Бисёрасъорӣ → Танзимот** гузаред
2. **Муайян кардани асъор аз ҷойгиршавии меҳмон**-ро фаъол кунед
3. Кишварҳоро ба асъорҳо дар ҷадвали **Харитаи геолокатсия** мувофиқ кунед

Агар барои кишвари меҳмон ягон мувофиқкунӣ вуҷуд надошта бошад, асъори асосии шабака ҳамчун арзиши эҳтиётӣ истифода мешавад.

## Саволҳои зуд-зуд додашаванда {#frequently-asked-questions}

### Агар провайдери қурби асъор дастнорас бошад, чӣ мешавад? {#what-happens-if-an-exchange-rate-provider-is-unavailable}

Addon ба қурбҳои охирине, ки бомуваффақият гирифта шудаанд, бармегардад. Агар ҳеҷ гоҳ ягон қурб гирифта нашуда бошад, он ба қурбҳои дастие, ки шумо танзим кардаед, бармегардад. Тавсия мешавад, ки қурбҳои дастиро ҳамчун шабакаи бехатарӣ танзим кунед.

### Оё ман метавонам барои асъорҳои гуногун провайдерҳои гуногун истифода барам? {#can-i-use-different-providers-for-different-currencies}

Не — дар як вақт барои ҳамаи асъорҳо як провайдер фаъол аст. Агар ба шумо назорати пурра бар қурбҳои инфиродӣ лозим бошад, **Дастӣ**-ро истифода баред.

### Оё геолокатсия плагини ҷудогона талаб мекунад? {#does-geolocation-require-a-separate-plugin}

Плагини берунаи геолокатсия талаб намешавад. Addon муайянкунии дарунсохти IP-ба-кишвари WordPress-ро истифода мебарад. Барои дақиқии баландтар, хидмати махсуси геолокатсия метавонад тавассути hook-ҳои филтри пешниҳодшуда ҳамгиро карда шавад.

### Оё бознависиҳои ҳар маҳсулот пеш аз андоз татбиқ мешаванд ё баъд аз он? {#are-per-product-overrides-applied-before-or-after-tax}

Бознависиҳои ҳар маҳсулот нархи пеш аз андозро дар асъори ҳадаф таъин мекунанд. Андоз болои нархи бознависӣ бо истифода аз танзимоти мавҷудаи андози шумо ҳисоб карда мешавад.
