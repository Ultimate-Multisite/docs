# Сабти тағйироти ҳамгироии WooCommerce {#woocommerce-integration-changelog}

Версияи 2.2.0 - Нашр шуд дар 2026-07-01
* Нав: Маблағҳои андози Ultimate Multisite акнун дар checkout ҳамчун сатрҳои ҷудогонаи ҳаққи WooCommerce пайдо мешаванд, ки ҷамъҳои андозро пеш аз пардохт равшантар мекунанд.
* Нав: Танзими ихтиёрии "Узвиятҳоро ҳангоми навсозии ноком фавран боздоред" ва филтри `wu_woo_suspend_on_payment_failure` барои сайтҳое илова шуд, ки мехоҳанд боздории навсозии ноком дар равзанаи кӯшиши дубораи WooCommerce Subscriptions анҷом шавад.
* Ислоҳ: Ҳолатҳои WooCommerce Subscription, ки пас аз навсозиҳои ноком ё барқароршуда метавонистанд бо узвиятҳои Ultimate Multisite ҳамоҳанг набошанд, ҳамоҳанг карда шуданд.
* Ислоҳ: Асъори мағозаи WooCommerce ба рӯйхати асъорҳои Ultimate Multisite илова шуд, вақте ки он набуд.
* Ислоҳ: Тафсилоти ҳисоббарории муштарӣ ҳангоми равона кардани обуначиён ба checkout-и WooCommerce нигоҳ дошта шуд.
* Беҳтар шуд: Мутобиқат бо Jetpack Autoloader 5 илова шуд.
* Беҳтар шуд: Эҷоди бастаи нашр тоза карда шуд, то zip-ҳои GitHub ва marketplace аз директорияҳои staging-и дохилӣ ва файлҳои рушд канорагирӣ кунанд.

* Ислоҳ: Санҷида шуд, ки оё обуна вуҷуд дорад, то дар ҷараёни бекоркунӣ хатогиҳо пешгирӣ шаванд.
