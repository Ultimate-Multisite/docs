# Таърихи тағйироти ҳамоҳангсозии муҳтаво {#content-sync-changelog}

### Version 1.1.0 - Нашршуда дар 2025-12-31 {#version-110---released-on-2025-12-31}
* Нав: Дастгирӣ барои ҳамоҳангсозии ҳамаи навъҳои навиштаҳо (навиштаҳо, саҳифаҳо, навъҳои фармоишии навиштаҳо)
* Нав: Филтркунии навъи навишта дар қадами интихоби муҳтаво
* Нав: Қадами интихоби сайти манбаъ - ҳамоҳангсозӣ аз ҳар сайт, на танҳо сайти асосӣ
* Нав: Пешинтихоби сайти фаръӣ барои сайтҳои қолабӣ
* Нав: Баровардани медиа аз post_content барои ҳамаи навъҳои навиштаҳо
* Нав: Ҳамоҳангсозии таксономияҳо (категорияҳо, барчаспҳо, таксономияҳои фармоишӣ)
* Тағйир ёфт: Ном аз "Elementor Template Sync" ба "Content Sync" иваз карда шуд
* Тағйир ёфт: Қадами "Select Templates" ба "Select Data" иваз карда шуд
* Беҳтар шуд: Коди махсуси Elementor акнун танҳо барои навъи навиштаи elementor_library иҷро мешавад

### Version 1.0.0 - Нашршуда дар 2025-12-19 {#version-100---released-on-2025-12-19}
* Нашри ибтидоӣ
* Ёвари чандқадамӣ барои интихоби қолаб ва сайт
* Коркарди бастаӣ бо бозхӯрди пешрафт дар вақти воқеӣ
* Ҳамоҳангсозии пурраи медиа бо бозхаритасозии худкори ID
* Интегратсияи танзимот бо Ultimate Multisite
* Дастгирии истиснои сайт тавассути танзимоти шабака
* Поккунии кэши Elementor пас аз ҳамоҳангсозӣ
* Нигоҳдории таксономияи қолаб
