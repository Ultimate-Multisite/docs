# Content Sync {#content-sync}

Multisite Ultimate: Content Sync

Ин addon барои Multisite Ultimate ба маъмурони шабака имкон медиҳад, ки муҳтаворо аз ҳар гуна сайт ба сайтҳои дигари шабака ҳамоҳанг созанд. Он ҳамаи навъҳои навиштаҳоро, аз ҷумла навиштаҳо, саҳифаҳо, навъҳои фармоишии навишта ва қолабҳои Elementor дастгирӣ мекунад. Ин addon интерфейси ҷодугари чандқадамиро барои интихоби муҳтаво ва сайтҳои ҳадаф, бо коркарди гурӯҳӣ ва бозхӯрди пешрафт дар вақти воқеӣ дар бар мегирад.

## Хусусиятҳо {#features}

* Интихоби муҳтаво: Шабакаи визуалии кортҳо бо тасвирҳои хурд ва филтркунии навъи навишта
* Ҳамаи навъҳои навишта: Ҳамоҳангсозии навиштаҳо, саҳифаҳо, маҳсулот, қолабҳои Elementor ва ҳар гуна навъи фармоишии навишта
* Интихоби сайт: Интихоб кунед, ки муҳтаво ба кадом сайтҳо ҳамоҳанг карда шавад, бо хориҷ кардани сайти манбаъ ва пешинтихоби сайти фарзанд
* Коркарди гурӯҳӣ: Ба 10 сайт дар як вақт бо навсозиҳои пешрафт ҳамоҳанг мекунад
* Ҳамоҳангсозии расона: Ба таври худкор файлҳои расонаиро (тасвирҳо, видеоҳо) аз муҳтавои навишта ба сайтҳои ҳадаф нусха мекунад ва дубора харита мекунад
* Ҳамоҳангсозии таксономия: Категорияҳо, барчаспҳо ва таксономияҳои фармоиширо нигоҳ медорад
* Дастгирии Elementor: Дастгирии пурраи қолабҳои Elementor бо бозтавлиди CSS ва дубора харитасозии ID-и қолаб
* Ҳалли ихтилоф: Имкони иваз кардани муҳтавои мавҷуда ё гузаронидани он
* Идоракунии кэш: Пас аз ҳамоҳангсозӣ кэши Elementor-ро дар сайтҳои ҳадаф ба таври худкор пок мекунад

## Талабот {#requirements}

* WordPress 5.3 ё баландтар
* PHP 7.4 ё баландтар
* Plugin-и Multisite Ultimate (фаъол)
* Plugin-и Elementor (ихтиёрӣ, барои ҳамоҳангсозии қолаби Elementor зарур аст)

## Насб {#installation}

1. 'ultimate-multisite-content-sync'-ро ба директорияи '/wp-content/plugins/' боргузорӣ кунед
2. Plugin-ро тавассути менюи 'Plugins' дар WordPress Network Admin-и худ фаъол кунед
3. Танзимотро дар Ultimate Multisite → Танзимот → Content Sync танзим кунед
4. Ба Ultimate Multisite → Content Sync гузаред, то ҳамоҳангсозиро оғоз кунед
