# Эҷодкунандаи саҳифаи маъмурӣ {#admin-page-creator}

## Муҳтаво {#content}
Эҷодкунандаи саҳифаи маъмурӣ ба шумо имкон медиҳад, ки ба осонӣ дар дохили Dashboard-и маъмурии WordPress саҳифаҳои нави маъмурӣ ва зерсаҳифаҳо илова кунед. Онро метавон барои эҷоди саҳифаи хушомадгӯӣ ё саҳифаи омӯзишӣ истифода бурд, ки метавонад ҳамчун саҳифаи вуруд барои корбарони нав хизмат кунад. Шумо метавонед созандагони саҳифа, монанди Beaver Builder, Elementor, Brizy ва Oxygen-ро истифода баред, то саҳифаҳоро бе душворӣ тарҳрезӣ кунед.

Он инчунин дастгирии HTML, CSS ва JavaScript-и фармоишӣ, ҳамчунин назорати дастрасиро дар асоси нақша ва нақши корбар пешниҳод мекунад!

## Хусусиятҳо {#features}
* Дастгирии HTML ва CSS-и фармоишӣ
* Дастгирии JavaScript
* Ҳолати муҳтавои PHP
* Иҷозат дар асоси нақши корбар
* Дастгирии созандагони саҳифа

## Насб {#installation}

1. **ultimate-multisite-admin-page-creator**-ро ба директорияи **/wp-content/plugins/** боргузорӣ кунед
2. plugin-ро тавассути менюи **Plugins** дар WordPress Network Admin-и худ фаъол кунед

## Дастури истифода {#usage-guide}

Ҳарчанд WordPress илова кардани муҳтаворо ба қисми пеши сайтҳо воқеан осон мекунад, эҷоди саҳифаҳо ва зерсаҳифаҳо дар тарафи маъмурӣ баъзе малакаҳои асосии коднависиро талаб мекунад. Ultimate Multisite: Эҷодкунандаи саҳифаи маъмурӣ ҳадаф дорад ҳамон осонии истифодаи қисми пешро ба қисми пушти сайтҳои шабакаи шумо биёрад.

## Чаро Эҷодкунандаи саҳифаи маъмуриро истифода бурдан лозим аст? {#why-use-admin-page-creator}

Шояд шумо мехоҳед дар панели маъмурии муштариёни худ саҳифаи фармоишӣ бо омӯзишҳо пешниҳод кунед, шояд мехоҳед саҳифаи фармоишӣ бо саволҳои маъмул ва виҷети тамос илова кунед. Имкониятҳо бо Эҷодкунандаи саҳифаи маъмурӣ беохиранд, ва маҳз ҳаминро мо дар омӯзиши имрӯза баррасӣ мекунем.

## Эҷоди аввалин саҳифаи маъмурии фармоишии худ {#creating-your-first-custom-admin-page}

Пас аз насб ва фаъол кардани Ultimate Multisite: Эҷодкунандаи саҳифаи маъмурӣ, ба менюи маъмурии шабакаи худ равед ва ба Саҳифаҳои маъмурӣ гузаред.

![Рӯйхати саҳифаҳои маъмурии фармоишӣ дар Network Admin](/img/admin/admin-pages-list.png)

_Рӯйхати саҳифаҳои маъмурии фармоишӣ дар Network Admin_

Шумо рӯйхати ҳамаи саҳифаҳои маъмуриро, ки то ҳоло эҷод кардаед, хоҳед дид (азбаски ин боздиди аввалини шумост, рӯйхат холӣ хоҳад буд). Давом диҳед ва **Илова кардани саҳифаи нави маъмурӣ**-ро пахш кунед. Шумо ба вироишгари саҳифаи маъмурӣ равона карда мешавед.

Ин унвонест, ки дар болои саҳифа намоиш дода мешавад, на нишонаи менюи маъмурӣ.

## Имконоти меню {#menu-options}

Дар паҳлӯ, дар meta-box-и Имконоти умумӣ, шумо метавонед навъи меню, нишона ва бисёр чизҳои дигарро танзим кунед.

### Навъҳои меню {#menu-types}

Саҳифаи нави маъмурии фармоишии шумо метавонад ду навъи гуногуни меню дошта бошад: менюи сатҳи боло ва зерменю.

Саҳифаи менюи сатҳи боло, тавре ки аз номаш бармеояд, саҳифаи навэҷодшудаи шуморо ба менюи маъмурӣ дар паҳлӯи саҳифаҳои пешфарзи WordPress илова мекунад.

Агар шумо варианти сатҳи болоро интихоб кунед, инчунин метавонед тартиби меню ва нишонаи менюро интихоб кунед.

Агар хоҳед ин саҳифаро саҳифаи зерменю созед, шумо метавонед саҳифаи волидайниеро интихоб кунед, ки ин саҳифа ба он пайваст шавад. Шумо метавонед саҳифаи фармоишии худро ба саҳифаҳои пешфарзи WordPress ё ба саҳифаҳои маъмурии фармоишии сатҳи болои худ илова кунед.

<!-- Скриншот дастнорас аст: Имконоти навъи менюи Эҷодкунандаи саҳифаи маъмурӣ бо танзимоти нишона, тартиб ва тасвирча -->

_Банди менюро барои ин Саҳифаи фармоишӣ пурра фармоишӣ кунед_

<!-- Скриншот дастнорас аст: Интихоби саҳифаи волидайни зерменю дар Эҷодкунандаи саҳифаи маъмурӣ -->

_Шумо инчунин метавонед ин саҳифаро ҳамчун зерменюи саҳифаҳои дигари меню, аз ҷумла саҳифаҳои фармоишии сатҳи болои худатон, илова кунед._

## Имконоти пешрафта {#advanced-options}

Шумо бояд тавонед ҳама гуна муҳтаворо ба саҳифаҳои маъмурии фармоишии худ илова кунед. Ин қоидаҳо ва файлҳои CSS, JavaScript ва инчунин китобхонаҳои берунаро дар бар мегирад. Шумо метавонед ин корро бо истифода аз ҷадвали Имконоти пешрафта анҷом диҳед.

### CSS-и фармоишӣ {#custom-css}

Дар ҷадвали CSS, шумо метавонед коди CSS-и фармоишии худро илова кунед, инчунин файлҳои CSS-ро аз сайтҳои дурдаст ворид намоед (монанди CDN-ҳо, Google Fonts ва ғайра).

<!-- Скриншот дастнорас аст: Вироишгари CSS-и фармоишии Эҷодкунандаи саҳифаи маъмурӣ бо воридкунии stylesheet-и беруна -->

_CSS-и фармоишӣ низ дастгирӣ мешавад. Шумо инчунин метавонед файлҳои услуби берунаро дохил кунед!_

### JavaScript-и фармоишӣ {#custom-javascript}

Ҳамин чиз барои JavaScript низ дуруст аст. Шумо метавонед китобхонаҳои берунаро барои истифода дар коди JavaScript-и фармоишии худ илова кунед.

<!-- Скриншот дастнорас аст: Вироишгари JavaScript-и фармоишии Эҷодкунандаи саҳифаи маъмурӣ бо воридкунии китобхонаи беруна -->

_Коди JavaScript-и худро илова кунед!_

## Иҷозатҳо {#permissions}

Ҳамчунин дар Имконоти пешрафта, шумо метавонед интихоб кунед, ки кадом зерсайтҳо ба ин саҳифа дастрасӣ хоҳанд дошт. Ин муфид аст, агар шумо, масалан, ният дошта бошед ба сатҳҳои гуногуни нақша муҳтавои истисноии гуногун пешниҳод кунед.

<!-- Скриншот дастнорас аст: Танзимоти иҷозатҳои Эҷодкунандаи саҳифаи маъмурӣ барои назорати дастрасӣ дар асоси нақша ва нақш -->

_Шумо бар он, ки ин саҳифа дар куҷо намоён мешавад, назорати пурра доред_

**Муҳим** : Барои намоиш дода шудани саҳифа ҳар ду шарт бояд иҷро шаванд. Ин маънои онро дорад, ки агар корбар мизоҷи як нақшаи муайян бошад, аммо яке аз нақшҳои интихобшударо надошта бошад (ё баръакс), саҳифа дар панели ӯ пайдо намешавад.

## Фаъол {#active}

Агар хоҳед саҳифаи фармоишии admin-ро бе нест кардани он ё бе дахолат ба иҷозатҳо ғайрифаъол кунед, танҳо имконоти Active-ро аз интихоб хориҷ кунед.

Ғайрифаъол кардани саҳифаи сатҳи болоӣ ба таври худкор ҳамаи зерсаҳифаҳои ба он саҳифаи волидайн пайвастшударо ғайрифаъол мекунад.
