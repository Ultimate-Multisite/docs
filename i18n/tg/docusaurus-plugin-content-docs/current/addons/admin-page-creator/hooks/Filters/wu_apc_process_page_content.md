| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $content | `string` | Муҳтаво пас аз ҷойгузиниҳо. |

| $to_replace | `array` | Массиве, ки ҷойнишинҳоро дар бар мегирад. |
| $placeholder_count | `int` | Шумораи ҷойнишинҳои ёфтшуда. |
