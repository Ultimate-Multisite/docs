# Gratis AI Agent

Gratis AI Agent як ёвари автономии WordPress бо AI аст, ки метавонад тавассути дастурҳои забони табиӣ сомонаҳои пурраи WordPress-ро ба нақша гирад, созад ва идора кунад. Он маҷмӯи сохторёфтаи **қобилиятҳо**-ро пешниҳод мекунад — амалҳои ҷудогона ва санҷишпазире, ки агент метавонад онҳоро фаро хонад — ва ҳама чизро аз идоракунии муҳтаво то мутобиқсозии низоми тарроҳӣ фаро мегирад. Версияи 1.9.0 қобилиятҳои эҷоди муҳтаво, бозбинии визуалии саҳифа тавассути скриншотҳои ҷониби муштарӣ, панҷ профили дарунсохти агент ва парчамҳои хусусиятро барои назорати дастрасӣ ва брендсозӣ илова мекунад.

## Хусусиятҳои асосӣ

- **Низоми қобилиятҳо** — феҳристи модулӣ ва тавсеапазири амалҳое, ки агент метавонад дар насби WordPress-и шумо фаро хонад
- **Идоракунии муҳтаво** — эҷоди навиштаҳо ва саҳифаҳо (аз ҷумла таъйини қолабҳои саҳифа), эҷоди гурӯҳии якчанд навишта дар як фарохонӣ, гузоштани тасвирҳои барҷаста ва сохтани формаҳои тамос
- **Бозбинии визуалӣ** — гирифтани скриншотҳои саҳифаҳои зинда, муқоисаи ҳолатҳои пеш/пас ва гирифтани бозбинии тарроҳии тавлидшуда бо AI, ки ҷойгиршавӣ, типография, ранг ва дастрасиро фаро мегирад
- **Панҷ агенти дарунсохт** — Муаллифи муҳтаво, Созандаи сомона, Студияи тарроҳӣ, Менеҷери плагин ва Ёвари дастгирӣ; тавассути Интихобгари агент дар сарлавҳаи чат ивазшаванда
- **Парчамҳои хусусият** — ивазкунакҳои назорати дастрасӣ ва брендсозӣ дар Settings → Feature Flags барои маҳдудиятҳои нақш, номгузории white-label ва мутобиқсозии виҷет
- **Идоракунии намудҳои фармоишии навишта** — сабт, рӯйхат кардан ва нест кардани намудҳои фармоишии навишта бо нигоҳдории доимӣ дар байни бозоғозиҳо
- **Идоракунии таксономияи фармоишӣ** — сабт, рӯйхат кардан ва нест кардани таксономияҳои фармоишӣ бо нигоҳдории доимӣ
- **Қобилиятҳои низоми тарроҳӣ** — ворид кардани CSS-и фармоишӣ, идоракунии намунаҳои интихобшудаи блок, гузоштани логотипи сомона ва татбиқи пештанзимҳои theme.json
- **Идоракунии сабкҳои глобалӣ** — хондан ва навиштани арзишҳои theme.json барои палитраҳои ранг, типография, фосилагузорӣ ва бештар
- **Идоракунии менюи паймоиш** — эҷод ва навсозии менюҳои паймоиш ва ҷузъҳои онҳо ба таври барномавӣ
- **Идоракунии имконот** — хондан ва навиштани имконоти WordPress бо blocklist-и бехатарии дарунсохт барои ҳифзи танзимоти муҳим
- **Феҳристи қобилиятҳои насбшаванда** — кашф ва фаъолсозии бастаҳои иловагии қобилиятҳо бе тағйир додани файлҳои асосии плагин
- **Ҳамоҳангсозии созандаи сомона v2** — нақшаҳои бисёрқадамии сохтани сомона бо кашфи плагин, пайгирии пешрафт ва барқарорсозӣ аз хато
- **Фармони санҷиши WP-CLI** — иҷрои маҷмӯаи санҷиши қобилиятҳои агент аз сатри фармон барои CI ва арзёбии модел

## Талабот

- WordPress 6.0 ё болотар
- PHP 8.1 ё болотар
- WordPress Multisite (дар шабака фаъолшуда) ё насби яксомона
- Плагини пайвасткунандаи фаъоли таъминкунандаи AI (Anthropic, OpenAI ё мувофиқ)

## Насб

1. Файлҳои плагинро ба `/wp-content/plugins/gratis-ai-agent` боргузорӣ кунед
2. Плагинро тавассути экрани **Плагинҳо** дар WordPress фаъол кунед
3. Ба **Gratis AI Agent → Settings** гузаред ва таъминкунандаи AI-и худро пайваст кунед
4. (Ихтиёрӣ) **Феҳристи қобилиятҳо**-ро барои насби бастаҳои иловагии қобилиятҳо бубинед

## Оғози зуд

Пас аз фаъолсозӣ ва пайваст шудан ба таъминкунандаи AI, панели **Gratis AI Agent**-ро дар лавҳаи канории идоракунии WordPress кушоед. Дастури забони табииро ворид кунед ва агент қадамҳои лозимиро бо истифода аз қобилиятҳои дастрасаш ба нақша мегирад ва иҷро мекунад.

**Намунаи дастурҳо:**

```
Як навъи фармоишии навиштаи "Портфолио" бо таксономияи "Категорияи лоиҳа" эҷод кунед.
```

```
Палитраи ранги ҳолати торикро ба мавзӯъ ворид кунед ва ранги асосиро ба #1a1a2e гузоред.
```

```
Сомонаи тарабхонаро бо менюи онлайн, формаи фармоиш ва саҳифаи тамос созед.
```

## Шарҳи қобилиятҳо

Қобилиятҳо амалҳои атомие мебошанд, ки агент метавонад иҷро кунад. Ҳар қобилият як синфи сабтшудаи PHP аст, ки схемаеро (параметрҳо ва намудҳои бозгашт) пешниҳод мекунад, ки агент онро ҳангоми иҷро мехонад. Қобилиятҳо ба соҳаҳои функсионалӣ гурӯҳбандӣ шудаанд:

| Соҳа | Қобилиятҳо |
|---|---|
| **Идоракунии муҳтаво** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **Бозбинии визуалӣ** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **Намудҳои фармоишии навишта** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **Таксономияҳои фармоишӣ** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **Низоми тарроҳӣ** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **Сабкҳои глобалӣ** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **Менюҳои паймоиш** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **Идоракунии имконот** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **Қобилиятҳои насбшаванда** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **Созандаи сомона** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

Барои ҳуҷҷатгузории пурраи параметрҳо ва намунаҳои истифода ба [Маълумотномаи қобилиятҳо](./abilities) нигаред.

## WP-CLI

Gratis AI Agent оилаи фармонҳои `wp gratis-ai-agent`-ро барои иҷрои санҷишҳо ва идоракунии қобилиятҳо аз сатри фармон дар бар мегирад. Барои рӯйхати пурраи фармонҳо ба [Маълумотномаи WP-CLI](./wp-cli) нигаред.

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## Танзимот

Ҳамаи танзимот дар зери **Gratis AI Agent → Settings** дар идоракунии WordPress ҷойгиранд:

| Танзим | Тавсиф |
|---|---|
| **Таъминкунандаи AI** | Пайвасткунандаи фаъоли таъминкунандаи AI-ро интихоб ва танзим кунед |
| **Модел** | Модели забонеро интихоб кунед, ки барои ҷаласаҳои агент истифода мешавад |
| **Ҳадди аксари токенҳо** | Буҷети токен барои ҳар дархост (пешфарз 4096) |
| **Вақти анҷоми қобилият** | Ҳадди аксари сонияҳое, ки як қобилияти ҷудогона метавонад иҷро шавад (пешфарз 30) |
| **Сабти ислоҳи хато** | Фарохониҳои қобилият ва андешаронии агентро ба сабти ислоҳи хатои WordPress сабт кунед |

## Blocklist-и бехатарии идоракунии имконот

Қобилияти идоракунии имконот дорои blocklist-и дарунсохт аст, ки ба агент аз хондан ё навиштани имконоти ҳассоси WordPress монеъ мешавад. Blocklist-и пешфарз инҳоро дар бар мегирад:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (имконоти URL — барои инҳо аз абзори махсуси муҳоҷират истифода баред)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (калидҳои махфӣ)
- `active_plugins`, `template` (фаъолсозии plugin/theme — аз қобилиятҳои махсус истифода баред)
- Маълумоти эътибории даргоҳҳои пардохт ва калидҳои API, ки аз ҷониби plugin-ҳои насбшуда сабт шудаанд

Вурудҳои иловагӣ метавонанд тавассути филтри `gratis_ai_agent_options_blocklist` илова карда шаванд.

## Hook-ҳо ва филтрҳо

### Амалҳо

- `gratis_ai_agent_ability_registered` — пас аз сабт шудани қобилият иҷро мешавад, номи синфи қобилиятро мегирад
- `gratis_ai_agent_plan_started` — вақте иҷро мешавад, ки нақшаи созандаи сайт оғоз ба иҷро мешавад
- `gratis_ai_agent_plan_completed` — вақте иҷро мешавад, ки нақша бо муваффақият ба анҷом мерасад
- `gratis_ai_agent_plan_failed` — вақте иҷро мешавад, ки нақша бо хатои ислоҳнопазир рӯ ба рӯ мешавад

### Филтрҳо

- `gratis_ai_agent_abilities` — рӯйхати қобилиятҳои сабтшударо пеш аз он ки агент онро бор кунад, тағйир диҳед
- `gratis_ai_agent_options_blocklist` — blocklist-и бехатарии имконотро васеъ ё кам кунед
- `gratis_ai_agent_benchmark_questions` — ба маҷмӯаи арзёбӣ саволҳои санҷишии фармоишӣ илова кунед
- `gratis_ai_agent_plan_plugins` — рӯйхати кашфи plugin-ҳоро ҳангоми тавлиди нақшаи сайт тағйир диҳед
