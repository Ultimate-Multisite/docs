# Мудири плагинҳо ва мавзӯъҳо

Multisite Ultimate: Мудири плагинҳо ва мавзӯъҳо

Маълумоти мета (унвон, муаллиф, тасвири хурд ва тавсиф)-и плагинҳо ва мавзӯъҳои насбшударо таҳрир ё пинҳон кунед. Барои плагинҳо ва мавзӯъҳо категорияҳо эҷод кунед, то корбарони шумо онҳоро филтр карда тавонанд, ва ба панели корбарони худ саҳифаи зебои фармоишии плагинҳоро илова кунед.

Барои истифодаи плагин, шумо метавонед танҳо ба саҳифаҳои плагинҳо ё мавзӯъҳои худ дар маъмури шабака гузаред ва "Таҳрири маълумоти плагин/мавзӯъ"-ро клик кунед.

## Насбкунӣ {#installation}

1. 'ultimate-multisite-plugin-and-theme-manager'-ро ба директорияи '/wp-content/plugins/' боргузорӣ кунед
2. Плагинро тавассути менюи 'Плагинҳо' дар WordPress Network Admin-и худ фаъол кунед

## Роҳнамои истифода {#usage-guide}

Ultimate Multisite: Мудири плагинҳо ва мавзӯъҳо як иловаи пуриқтидори **премиум** барои Ultimate Multisite мебошад, ки ба шумо назорати пурра медиҳад, ки корбарони шабакаи шумо имконоти плагин ва мавзӯъро дар шабакаи шумо чӣ гуна мебинанд ва бо онҳо чӣ гуна ҳамкорӣ мекунанд.

Ин илова се ҳадаф дорад:

  1. Он ба шумо имкон медиҳад, ки метамаълумоти ҳам плагинҳо ва ҳам мавзӯъҳоро пурра тағйир диҳед. Ин ном, тавсиф, номи муаллиф ва ҳатто тасвирҳои хурдро дар бар мегирад!

  2. Он ба шумо имкон медиҳад, ки плагинҳо ва мавзӯъҳоро категориябандӣ кунед, то корбарони шумо онҳоро ба осонӣ ҷудо карда тавонанд.

  3. Он ба шумо имкон медиҳад, ки саҳифаи кӯҳна ва дилгиркунандаи плагинҳоро бо саҳифаи нави монанд ба мавзӯъҳо, бо ҷудокунӣ аз рӯи категорияҳо, иваз кунед.

Пас аз боргирӣ ва насби он, дар саҳифаи Танзимоти Ultimate Multisite ҷадвали нав - _Танзимоти илова_ \- намоён мешавад.
