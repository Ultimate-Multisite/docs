# Сабти тағйироти тикетҳои дастгирӣ

### 1.0.4 - 2026-05-05
* Беҳтар шуд: Директорияи vendor/ аз пайгирии Git хориҷ карда шуд (аллакай бо .gitignore фаро гирифта шудааст), ки андозаи анборро кам мекунад
* Беҳтар шуд: То WordPress 7.0 санҷида шуд

### 1.0.3 - 2026-05-01
* Ислоҳ: Қабули арзишҳои null дар танзимгарҳои модели тикети nullable
* Ислоҳ: Нигоҳдории ҳамаи тикетҳо дар ҷадвалҳои глобалии сатҳи тамоми шабака барои дастгирии дурусти multisite
* Ислоҳ: Пинҳон кардани майдонҳои кормандон ва ислоҳи равонасозии саҳифаи холӣ дар формаи тикети нави муштарӣ
* Ислоҳ: Иваз кардани фарохонӣҳои муайяннашудаи add_meta() бо update_meta()-и дуруст барои нигоҳдории метамаълумот
* Ислоҳ: Иваз кардани санҷиши capability-и сабтнашуда бо wu_view_all_support_tickets-и дуруст
* Ислоҳ: Илова кардани коркардгарҳои AJAX-и намерасида барои вазъи тикет, таъйинкунӣ ва амалҳои таҳрири зуд
* Ислоҳ: Дуруст кардани номи метод барои ошкорсозии посухи кормандон дар намоишҳои тикет
* Ислоҳ: Якҷо кардани коркардгарҳои такрории посухҳо ва ҳамоҳанг кардани номҳои амали nonce
* Ислоҳ: Илова кардани намоиши frontend-и намерасида барои shortcode-и [wu_submit_ticket]
* Ислоҳ: Илова кардани сутуни user_id-и намерасида ва ислоҳи методи Support_Ticket::get_user_id()
* Ислоҳ: Хориҷ кардани нохунаки дукаратаи зиёдатӣ дар теги имконоти филтри афзалият
* Ислоҳ: Илова кардани панели идоракунии тикетҳои network admin барои super admins
* Беҳтар шуд: Якҷо кардани CSS-и admin ба як stylesheet-и берунии ягона
* Беҳтар шуд: Хориҷ кардани зерменюи Settings аз менюи admin-и subsite
* Беҳтар шуд: Боркунии шартии дороиҳои frontend танҳо дар саҳифаҳои тикетҳои дастгирӣ
* Беҳтар шуд: Гузаронидани autoloader-и plugin ҳангоми он ки autoloader-и решаи Bedrock вобастагиҳоро аллакай бор кардааст

### 1.0.2 - 2025-12-11
* Илова шуд: Функсионалияти пурраи намоиши тикет барои admin ва frontend
* Илова шуд: Коркардгари AJAX барои фиристодани посухҳои тикет
* Илова шуд: Дастгирии функсионалияти посух ба тикет бо коркарди дурусти форма
* Илова шуд: Намоиши дурусти огоҳӣ барои фиристодани тикетҳо ва посухҳо дар admin
* Илова шуд: Пайвасткунии худкори тикет бо корбари ҷорӣ барои фиристодани мустақили муштарӣ
* Илова шуд: Такмили амният барои пешгирӣ аз иваз кардани соҳибияти тикет аз ҷониби муштариён
* Илова шуд: Функсияҳои ёрирасони намерасида (wu_format_date, wu_user_can_view_ticket ва ғайра)
* Илова шуд: Боргирӣ ва коркарди дурусти замимаҳои файлӣ
* Илова шуд: Системаи огоҳиномаи email барои посухҳои тикет ва тағйироти вазъ
* Ислоҳ шуд: Формаи посухи тикет акнун ID-и зарурии тикетро дар бар мегирад
* Ислоҳ шуд: Масъалаҳои намоёнии огоҳӣ дар намоишҳои тикети admin
* Ислоҳ шуд: Хатоҳои синтаксисӣ дар функсияҳои тикети дастгирӣ
* Беҳтар шуд: Сохтори пойгоҳи додаҳо бо классҳои дурусти query ва schema барои посухҳо ва замимаҳо
* Беҳтар шуд: Муҳоҷират ба хусусиятҳои воқеӣ ба ҷои атрибутҳо барои моделҳои тикет
