# Material WP

Интерфейси маъмурии WordPress солҳост яксон аст. Аммо оё ягон бор фикр кардаед, ки версияи интерфейси маъмурӣ бо Material Design чӣ гуна менамояд? 

Material WP як қолаби маъмурӣ аст, ки барои додани ҳамин намуди зоҳирӣ ва эҳсос таҳия шудааст ва ба шумо кӯмак мекунад, ки ба муштариёни худ Dashboard-и бештар фармоишишуда пешниҳод намоед.

## Хусусиятҳо {#features}

- Дастгирии CSS-и фармоишӣ
- Танзимоти осони услубдиҳӣ
- Азнавтартибдиҳии меню
- Воридот/содироти осон

## Ҳуҷҷатҳо {#documentation}

**Фармоишисозии панели маъмурии худ бо Material WP**

## Насб {#installation}

1. 'material-wp'-ро ба директорияи '/wp-content/plugins/' боргузорӣ кунед
2. Плагинро тавассути менюи 'Плагинҳо' дар WordPress фаъол кунед
