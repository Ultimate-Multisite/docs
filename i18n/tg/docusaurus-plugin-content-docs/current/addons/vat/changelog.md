# Таърихи тағйироти VAT {#vat-changelog}

Версия 1.0.7 - Нашр шуд дар 2026-02-03

* Ислоҳ: Рақамҳои VAT-и Олмон (DE) бинобар хатогиҳои VIES SOAP service MS_MAX_CONCURRENT_REQ аз санҷиш намегузаштанд. Аз SOAP API-и кӯҳна ба EU VIES REST API-и расмӣ гузаронида шуд, то санҷиш боэътимодтар шавад.
* Ислоҳ: Ҳоло рақамҳои VAT вақте ки хидмати VIES муваққатан дастнорас аст, ба ҷойи радкунии нодуруст қабул карда мешаванд. Барои баррасии баъдӣ сабти журнал эҷод мешавад.
* Ислоҳ: Майдони VAT/Tax ID ҳоло барои ҳамаи кишварҳо намоён аст, на танҳо барои аъзои ИА. Tax ID-ҳои ғайриИА (масалан, рақамҳои Swiss CHE) барои намоиш дар ҳисобнома бе санҷиши VIES нигоҳ дошта мешаванд.
* Ислоҳ: Ҷустуҷӯи префикси VAT барои Юнон (GR), Монако (MC) ва Ҷазираи Мэн (IM) бинобар мантиқи нодурусти ҷустуҷӯ дар массив вайрон буд.
* Такмил: Вобастагии нолозими таҳияи wp-cli-bundle хориҷ карда шуд, ки ҳангоми истифода дар баробари иловаи Site Exporter метавонист хатогиҳои марговар ба вуҷуд орад.
* Такмил: Маҷмӯаи фарогири санҷишҳои воҳидӣ илова шуд (53 санҷиш).

Версия 1.0.6 - Нашр шуд дар 2026-01-25

* Ислоҳ: Муомилоти дохилии B2B ҳоло VAT-ро дуруст ҳисоб мекунад. Мувофиқи қоидаҳои VAT-и ИА, reverse charge танҳо ба муомилоти фаромарзии B2B татбиқ мешавад, на вақте ки кишвари муштарӣ бо кишвари ширкат якхела аст.

Версия 1.0.5 - Нашр шуд дар 2026-01-22

* Ислоҳ: Илова бор намешуд ё дуруст кор намекард.
* Тағйир: Манбаи маълумоти меъёри VAT аз euvatrates.com ба анбори ibericode/vat-rates гузаронида шуд, то маълумот боэътимодтар ва фаъолона нигоҳдорӣ шавад.
* Ислоҳ: Хатои чопии арзиши option-и super_reduced_rates ислоҳ шуд.
* Такмил: Барои коркарди рамзи кишвар санҷишҳои null илова шуданд, то аз хатогиҳо пешгирӣ шавад.
* Тағйир: Файлҳои тарҷумаи ҳамроҳшуда ба фоидаи навсозиҳои худкор тавассути Traduttore хориҷ карда шуданд.
