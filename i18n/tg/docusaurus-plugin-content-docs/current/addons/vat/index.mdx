# VAT

Multisite Ultimate: VAT-и Аврупо

Ба коркарди пешфарзи андози Multisite Ultimate хусусиятҳо ва абзорҳои иловагӣ илова мекунад, то VAT-и Аврупоро беҳтар дастгирӣ кунад.
Ба таври худкор меъёрҳои EU VAT-ро ба саҳифаи Меъёрҳои андоз аз илова мекунад

## Насб

1. 'ultimate-multisite-vat'-ро ба директорияи '/wp-content/plugins/' боргузорӣ кунед
2. Плагинро тавассути менюи 'Плагинҳо' дар WordPress фаъол созед
3. Ба Ultimate Multisite > Танзимот > Андозҳо равед ва EU VAT-ро фаъол созед.
4. Ба Меъёрҳои андоз равед, дар поён Навсозии меъёрҳои EU VAT-ро пахш кунед ва Меъёрҳои андозро захира кунед.
5. Иҷро шуд!
