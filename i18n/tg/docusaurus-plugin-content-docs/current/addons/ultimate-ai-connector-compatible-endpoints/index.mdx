# Пайвасткунандаи Ultimate AI барои нуқтаҳои ниҳоии мувофиқ

Пайвасткунандаи Ultimate AI барои нуқтаҳои ниҳоии мувофиқ API-ҳои провайдерҳои AI-и мувофиқ бо OpenAI-ро бо шабакаи multisite-и WordPress-и шумо ҳамгиро мекунад. Якчанд провайдери AI-ро танзим кунед — ҳар яке бо URL-и нуқтаи ниҳоӣ, калиди API ва модели пешфарзи худ — ва дархостҳоро бо гузариши эҳтиётии худкор ҳангоми дастнорас будани провайдер масир диҳед.

## Хусусиятҳои асосӣ {#key-features}

- **Дастгирии чандпровайдерӣ**: Якчанд корти провайдери AI илова кунед, ҳар яке бо URL-и нуқтаи ниҳоӣ, калиди API ва модели пешфарзи худ
- **Кашида ҷобаҷогузорӣ кардан**: Провайдерҳоро аз рӯи афзалият бо истифода аз дастаки нигоҳдорӣ дар ҳар корт тартиб диҳед; дархостҳо провайдерҳоро бо ҳамин тартиб истифода мебаранд
- **Масирдиҳии худкори гузариши эҳтиётӣ**: Агар провайдери асосӣ дастнорас бошад, дархостҳо ба таври худкор ба провайдери танзимшудаи навбатӣ мегузаранд
- **Танзимоти ҷудогона барои ҳар провайдер**: Ҳар провайдер мустақилона бо маълумоти эътимоднокӣ ва интихоби модели худ танзим мешавад
- **Интерфейси танзимот дар асоси кортҳо**: Бар пояи компонентҳои устувори Card-и WordPress 6.9+ сохта шудааст (CardBody, CardHeader, CardDivider)
- **Кушодашавии худкор ҳангоми илова**: Кортҳои нави провайдер ба таври худкор кушода мешаванд, то шумо тавонед фавран тафсилотро пур кунед
- **Бекоркунии кэши скрипт**: Скриптҳои провайдер ҳангоми навсозии плагин аз кэш берун карда мешаванд, то бор шудани версияи охирин таъмин гардад

## Талабот {#requirements}

- WordPress 5.3 ё болотар
- PHP 7.4 ё болотар
- Плагини Ultimate Multisite (фаъол)
- Нуқтаи ниҳоии API-и мувофиқ бо OpenAI (худмизбон ё тарафи сеюм)

## Насб {#installation}

1. Файлҳои addon-ро ба директорияи `/wp-content/plugins/`-и худ боргузорӣ кунед
2. Плагинро тавассути менюи «Плагинҳо» дар WordPress фаъол кунед
3. Ба **Ultimate Multisite → AI Connector** гузаред, то addon-ро танзим кунед

## Танзимот {#configuration}

### Танзими чандпровайдерӣ {#multi-provider-setup}

Экрани танзимот тарҳбандии бар асоси кортҳоро истифода мебарад, ки дар он ҳар корт як провайдери AI-ро нишон медиҳад. Шумо метавонед ба қадри лозим провайдер илова кунед.

#### Илова кардани провайдер {#adding-a-provider}

1. Дар мудири шабака ба **Ultimate Multisite → AI Connector** равед
2. **Илова кардани провайдер**-ро пахш кунед
3. Корти нави провайдер пайдо мешавад ва ба таври худкор кушода мешавад
4. Тафсилоти провайдерро пур кунед:
   - **Номи провайдер** — тамғае барои шинохтани ин провайдер (масалан, "OpenAI", "Mistral", "Local Ollama")
   - **URL-и нуқтаи ниҳоӣ** — URL-и асосии API-и мувофиқ бо OpenAI (масалан, `https://api.openai.com/v1`)
   - **Калиди API** — калиди API-и шумо барои ин провайдер
   - **Модели пешфарз** — шиносаи модел, ки барои ин провайдер ба таври пешфарз истифода мешавад (масалан, `gpt-4o`, `mistral-small`)
5. **Захира кардани танзимот**-ро пахш кунед

#### Ҳазф кардани провайдер {#removing-a-provider}

Барои нест кардани он, тугмаи **Ҳазф кардан**-ро дар ҳар корти провайдер пахш кунед. Барои татбиқи тағйирот танзимотро захира кунед.

#### Аз нав тартиб додани провайдерҳо {#reordering-providers}

Нишонаи нигоҳдорӣ (⣿)-ро дар тарафи чапи ҳар корти провайдер кашида, онро аз нав тартиб диҳед. Тартиб афзалияти гузариши эҳтиётиро муайян мекунад: дархостҳо ба провайдери аввал фиристода мешаванд ва агар дархост ноком шавад, ба провайдерҳои баъдӣ мегузаранд.

### Масирдиҳии гузариши эҳтиётӣ {#fallback-routing}

Вақте дархост ба провайдери асосӣ ноком мешавад — бинобар анҷоми муҳлат, маҳдудияти суръат ё қатъи хизматрасонӣ — пайвасткунанда ба таври худкор бо провайдери навбатии рӯйхат дубора кӯшиш мекунад. Ин раванд дар рӯйхат идома меёбад, то вақте ки як провайдер муваффақ шавад ё ҳамаи провайдерҳо санҷида шаванд.

| Вазъият | Рафтор |
|----------|-----------|
| Провайдери асосӣ ҷавоб медиҳад | Дархост ба таври муқаррарӣ хизматрасонӣ мешавад |
| Провайдери асосӣ ноком мешавад | Дархост бо провайдери навбатӣ дубора кӯшиш мешавад |
| Ҳамаи провайдерҳо ноком мешаванд | Хато ба даъваткунанда баргардонида мешавад |

Гузариши эҳтиётӣ барои плагин ё theme-и даъваткунанда шаффоф аст — онҳо ё ҷавоби муваффақ мегиранд ё хатои ниҳоӣ, новобаста аз он ки чанд провайдер санҷида шуд.

### Маълумотномаи танзимоти провайдер {#provider-configuration-reference}

| Майдон | Ҳатмӣ | Тавсиф |
|-------|----------|-------------|
| Номи провайдер | Ҳа | Тамғаи намоишӣ барои ин провайдер |
| URL-и нуқтаи ниҳоӣ | Ҳа | URL-и асосии API-и мувофиқ бо OpenAI |
| Калиди API | Ҳа | Калиди аутентификатсия барои ин провайдер |
| Модели пешфарз | Ҳа | Шиносаи модел, ки вақте дар дархост модел нишон дода нашудааст, истифода мешавад |

## Бартарафсозии мушкилот {#troubleshooting}

### Ягон провайдер номбар нашудааст {#no-providers-are-listed}

Агар рӯйхати провайдерҳо баъд аз захира кардан холӣ бошад, тасдиқ кунед, ки:

- Шумо пас аз илова кардани провайдерҳо **Захира кардани танзимот**-ро пахш кардед
- Браузери шумо саҳифаи танзимотро аз захира шудан манъ накардааст (консоли браузерро барои хатоҳои JavaScript санҷед)
- Плагин фаъол аст ва шумо дар экрани мудири шабака ҳастед

### Дархостҳо барои ҳамаи провайдерҳо ноком мешаванд {#requests-are-failing-for-all-providers}

- Санҷед, ки URL-и нуқтаи ниҳоии ҳар провайдер аз сервери шумо дастрас аст (на танҳо аз браузери шумо)
- Санҷед, ки ҳар калиди API дуруст аст ва муҳлаташ нагузаштааст
- Ҳолати маҳдудияти суръат ва квотаи провайдерро дар dashboard-и онҳо баррасӣ кунед
- Сабткунии ислоҳи хатои WordPress-ро (`WP_DEBUG_LOG`) фаъол кунед, то паёмҳои муфассали хатогӣ аз пайвасткунанда сабт шаванд

### Провайдер барои гузариши эҳтиётӣ истифода намешавад {#a-provider-is-not-being-used-for-fallback}

Кашида ҷобаҷогузорӣ кардан тартиби гузариши эҳтиётиро тағйир медиҳад. Тасдиқ кунед, ки провайдере, ки интизор доред дархостҳои гузариши эҳтиётиро қабул кунад, дар рӯйхат дуруст ҷойгир шудааст ва URL-и нуқтаи ниҳоӣ ва калиди API-и он дурустанд.

### Корти нави провайдер ба таври худкор кушода намешавад {#new-provider-card-does-not-expand-automatically}

Кушодашавии худкор талаб мекунад, ки JavaScript дар браузери шумо фаъол бошад. Агар корт ҳангоми пахш кардан кушода нашавад, консоли браузерро барои хатоҳо санҷед. Муҳтавои корт ҳамеша бо пахш кардани сарлавҳаи корт дастрас аст, ҳатто агар кушодашавии худкор фаъол нашавад.
