# Сабтҳои тағйироти Ultimate AI Connector for Compatible Endpoints {#ultimate-ai-connector-for-compatible-endpoints-changelog}

## Версияи 2.0.0 — Нашршуда дар 2026-04-24 {#version-200--released-on-2026-04-24}

* Нав: Дастгирии чандпровайдерӣ — якчанд endpoint-и AI-ро танзим кунед ва дархостҳоро бо fallback-и худкор байни провайдерҳо масир диҳед.
* Ислоҳ: Интегратсияи SDK-и чандпровайдерӣ бо ID-ҳои дурусти провайдер, URL-ҳои бақайдгирӣ ва доираи HTTP filter барои ҳар провайдер.
* Ислоҳ: Корти нави провайдер ҳоло ҳангоми илова шудан худкор васеъ мешавад; тозакунии кэши script ҳангоми навсозии plugin.
* Ислоҳ: Namespace-и динамикии class-и провайдер барои eval() ва рафтори худкор-васеъшавӣ барои провайдерҳои навиловашуда.
* Ислоҳ: Барои мутобиқат бо WordPress 6.9+ аз компонентҳои устувори Card/CardBody/CardHeader/CardDivider истифода мекунад (дигар таҷрибавӣ нестанд).
* Ислоҳ: DragHandle-и номуайянро бо нишонаи unicode grip барои кашида аз нав тартиб додани провайдерҳо иваз кард.
* Ислоҳ: Дархости HTTP-и масдудкунандаро, ки дар ҳар бор бор шудани саҳифа иҷро мешуд, бартараф кард.
* Беҳтаршуда: Равандҳои кории GitHub Actions ба Node.js 24 навсозӣ шуданд.
