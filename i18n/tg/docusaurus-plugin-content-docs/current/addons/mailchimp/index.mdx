# Mailchimp

Ultimate Multisite: ҳамгироии Mailchimp

Шабакаи Ultimate Multisite-и худро бо Mailchimp-и худ ҳамгиро кунед. Шумо метавонед нақшаҳои муайянро ба рӯйхатҳои Mailchimp пайваст кунед ва ҳангоми сабтином муштариёнро ба таври худкор ба он рӯйхатҳо илова намоед.

## Насб {#installation}

1. 'ultimate-multisite-mailchimp'-ро ба директорияи '/wp-content/plugins/' боргузорӣ кунед
2. Плагинро тавассути менюи 'Плагинҳо' дар WordPress Network Admin-и худ фаъол кунед

## Дастури истифода {#usage-guide}

Ҳамоҳангсозии почтаҳои электронии муштариён бо рӯйхатҳои MailChimp-и шумо бо замимаи Ultimate Multisite: MailChimp Integration-и мо воқеан осон аст. Ин дастур ҳадаф дорад ба шумо кӯмак кунад, ки ҳама чизро танзим намоед!

## Гирифтани API key {#getting-an-api-key}

Пеш аз он ки ба танзимоти гуногуни дастрас гузарем, ба мо як MailChimp API key лозим аст. Ин ба Ultimate Multisite имкон медиҳад, ки бо MailChimp API иртибот кунад, то рӯйхатҳо ва гурӯҳҳоро дарёфт намояд ва ҳангоми сабтиноми корбарон суроғаҳои почтаи электрониро ба он рӯйхатҳо ва гурӯҳҳо илова кунад.

Барои гирифтани MailChimp API key-и худ ин дастурро пайгирӣ кунед: [Дар бораи API Keys | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

## Реҷаи ҳамгироӣ {#integration-mode}

Замимаи MailChimp Integration ду «реҷаи ҳамгироӣ»-ро пешниҳод мекунад, ки маънояш ин аст, ки ду роҳи гуногуни истифодаи замима барои ҳамоҳангсозии суроғаҳои почтаи электронӣ вуҷуд дорад.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_Ду реҷаи ҳамгироӣ барои чандирии бештар!_

### Реҷаи 1: Рӯйхатҳои сершумор {#mode-1-multiple-lists}

Реҷаи пешфарз аз **Рӯйхатҳои сершумор** истифода мебарад. Ин реҷа ба шумо имкон медиҳад, ки барои ҳар яке аз нақшаҳо якчанд рӯйхати MailChimp-ро интихоб кунед. Агар шумо корбарони худро ба рӯйхатҳо ҷудо карда бошед, ин беҳтарин интихоб аст.

Масалан, фарз мекунем, ки шумо дар Account-и MailChimp-и худ 3 рӯйхати гуногун доред: **Newsletter** , **Plan A** , ва **Plan B**. Шумо мехоҳед ҳамаи корбарон, новобаста аз нақшаашон, инчунин ба рӯйхати Newsletter дохил карда шаванд, то навсозиҳои шуморо аз даст надиҳанд.

Барои ба ин ноил шудан, шумо метавонед реҷаи Multiple Lists-ро истифода баред, то ҳар ду рӯйхати **Newsletter** ва **Plan A**-ро ба Plan A MailChimp Settings, ва **Newsletter** ва **Plan B**-ро ба Plan B MailChimp Settings илова кунед.

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_Рӯйхатҳои пешфарзро барои Account-ҳои нав ва барои Account-ҳои бекоршуда интихоб кунед_

### Реҷаи 2: Як рӯйхат + гурӯҳҳои сершумор {#mode-2-single-list--multiple-groups}

MailChimp ба шумо имкон медиҳад рӯйхати худро ба гурӯҳҳои хурдтари корбарон, ки **Гурӯҳҳо** ном доранд, ҷудо кунед. Ин ба шумо назорати бештар медиҳад, зеро метавонед маъракаро ба зермаҷмӯи мушаххаси муштариёни худ фиристед, бе он ки нусхаҳои сершумори ҳамон маъракаро дар MailChimp эҷод кунед (ки баъдан таҳлили нишондиҳандаҳои он маъракаро душвор мекунад).

Масалан, шумо метавонед як рӯйхат бо номи **My Network Subscribers,** дошта бошед ва дар дохили он гурӯҳҳо барои **Plan A** ва **Plan B** дошта бошед.

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_Рӯйхатҳо ва гурӯҳҳои пешфарзро интихоб кунед, агар аз реҷаи ҳамгироии «як рӯйхат + гурӯҳҳои сершумор» истифода баред_

## Рӯйхатҳо ва гурӯҳҳои пешфарз {#default-lists-and-groups}

Пас аз он ки интихоб мекунед кадом реҷаи ҳамгироӣ барои ҳолати истифодаи мушаххаси шумо мувофиқтар аст, вақти интихоби рӯйхатҳо (ва гурӯҳҳо, вобаста ба реҷа)-и пешфарз мерасад. Рӯйхати пешфарз вақте истифода мешавад, ки нақшаи додашуда рӯйхати мушаххаси интихобшуда надошта бошад.

Шумо инчунин метавонед рӯйхати пешфарзро барои Account-ҳои бекоршуда интихоб кунед. Ҳар вақте ки корбар обунаи худро қатъ мекунад ва Account-и худро аз шабакаи шумо хориҷ менамояд, суроғаи почтаи электронии ӯ ба рӯйхате, ки барои Account-ҳои бекоршуда таъин шудааст, кӯчонида мешавад.

## Фаъол кардани обунаи дукарата {#enabling-double-subscription}

Дар ҳамон саҳифаи танзимот, шумо метавонед **Обунаи дукарата**-ро фаъол кунед. Ин интихоб корбарро ҳам ба рӯйхати пешфарзи интихобшуда ВА ҳам ба рӯйхате, ки барои нақшаи он корбар интихоб шудааст, илова мекунад. Агар ин интихоб фаъол набошад, корбар танҳо ба рӯйхате, ки барои нақшаи корбар интихоб шудааст, илова карда мешавад. Агар барои он нақшаи мушаххас ҳеҷ чиз таъин нашуда бошад, пас корбар ба рӯйхати пешфарзи интихобшуда илова карда мешавад.

<!-- Screenshot unavailable: MailChimp double subscription toggle setting -->

_Шумо метавонед корбарони худро ҳангоми сабтином ҳам ба рӯйхати пешфарз ва ҳам ба рӯйхати нақша обуна кунед_

## Интихоби рӯйхатҳо ва гурӯҳҳо барои ҳар яке аз нақшаҳо {#selecting-lists-and-groups-for-each-of-the-plans}

Ба **Нақшаҳо**-и худ равед ва нақшаеро, ки мехоҳед таҳрир кунед, интихоб намоед. Дар хотир доред, ки дар мета-қуттии **Имконоти пешрафта**, ҷадвали нав бо номи MailChimp дастрас мешавад.

Дар он ҷадвал, шумо метавонед барои он нақша якчанд рӯйхат ё як рӯйхат ва гурӯҳҳои сершуморро (вобаста ба реҷаи ҳамгироӣ) интихоб кунед.

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_Рӯйхатҳо ва гурӯҳҳоеро, ки мехоҳед барои ҳар яке аз нақшаҳои худ истифода баред, интихоб кунед_

Вақте ки корбари нав барои он нақша сабтином мекунад, суроғаи почтаи электронии ӯ ба рӯйхатҳо ва гурӯҳҳои интихобшуда илова карда мешавад!
