# Дарси 10: Рӯзи оғоз

Ҳама чиз сохта шудааст. Пеш аз он ки дарҳоро боз кунед, ин дарс рӯйхати санҷиши пеш аз оғозро қадам ба қадам мегузарад, то мутмаин шавед, ки ҳеҷ чиз вайрон, гумшуда ё шармандакунанда нест.

## Дар куҷо монда будем {#where-we-left-off}

FitSite қолабҳо, нақшаҳо, checkout, брендинг, onboarding ва нархгузориро пурра танзим кардааст. Акнун мо месанҷем, ки ҳама чиз кор мекунад ва ба таври зинда оғоз мекунем.

## Рӯйхати санҷиши пеш аз оғоз {#pre-launch-checklist}

Ҳар як бандро иҷро кунед. Ҳеҷ кадомро нагузаронед.

### Инфрасохтори платформа {#platform-infrastructure}

- [ ] Hosting устувор аст ва зери сарборӣ хуб кор мекунад
- [ ] Wildcard SSL фаъол аст ва ҳамаи зердоменҳо тавассути HTTPS хизмат мекунанд
- [ ] Харитасозии домен кор мекунад -- сохтани як сайт ва харитасозии домени фармоиширо санҷед
- [ ] Нусхаҳои эҳтиётӣ танзим ва санҷида шудаанд (ҳадди ақал якеро барқарор кунед, то тасдиқ кунед)
- [ ] Мониторинг фаъол аст -- агар платформа аз кор монад, шумо хоҳед донист

### Қолабҳо {#templates}

- [ ] Ҳар се қолаб дар сайтҳои нав дуруст бор мешаванд
- [ ] Мундариҷаи ҷойнишон муфид аст ва хатои имлоӣ надорад
- [ ] Ҳама тасвирҳо иҷозатномаи дуруст доранд (бе нишонаҳои обии аксҳои саҳомӣ)
- [ ] Мутобиқшавӣ ба мобилӣ дар ҳар саҳифаи қолаб кор мекунад
- [ ] Суръати боршавии саҳифа қобили қабул аст (бо абзоре мисли GTmetrix ё PageSpeed Insights санҷед)
- [ ] Дар ягон қолаб пайвандҳои шикаста ё дороиҳои гумшуда нестанд

### Нақшаҳо ва маҳсулотҳо {#plans-and-products}

- [ ] Ҳар се нақша фаъол ва намоёнанд
- [ ] Тавсифҳои нақшаҳо дақиқ ва ба соҳаи мушаххас мувофиқанд
- [ ] Нархгузорӣ дуруст аст (моҳона ва солона)
- [ ] Пардохтҳои setup дар нақшаҳои дуруст танзим шудаанд
- [ ] Давраи озмоишӣ дар нақшаи Starter кор мекунад
- [ ] Order bumps ҳангоми checkout дуруст пайдо мешаванд
- [ ] Маҳдудиятҳои plugin ва theme барои ҳар нақша дуруст татбиқ мешаванд

### Ҷараёни checkout {#checkout-flow}

- [ ] Барои ҳар нақша як сабти пурраи санҷиширо анҷом диҳед (режими пардохти санҷиширо истифода баред)
- [ ] Интихоби қолабҳо қолабҳои дурустро барои ҳар нақша нишон медиҳад
- [ ] Пардохт бо муваффақият коркард мешавад
- [ ] Муштарӣ пас аз сабтином номаи хушомад мегирад
- [ ] Сайти нав бо қолаби дуруст сохта мешавад
- [ ] Муштарӣ метавонад фавран ба сайти нави худ ворид шавад
- [ ] Рамзҳои тахфиф дуруст кор мекунанд

### Брендинг {#branding}

- [ ] Саҳифаи воридшавӣ брендинги FitSite-ро нишон медиҳад
- [ ] Dashboard-и админ брендинги FitSite-ро нишон медиҳад
- [ ] Ҳама имейлҳои системавӣ брендинги FitSite ва забони махсуси фитнесро истифода мебаранд
- [ ] Ҳисобномаҳо бо тафсилоти тиҷорати шумо дуруст намоиш дода мешаванд
- [ ] Сайти маркетингӣ зинда аст ва ба формаи checkout пайванд медиҳад

### Onboarding {#onboarding}

- [ ] Виҷети Quick Start дар dashboard-ҳои муштариёни нав пайдо мешавад
- [ ] Ҳама пайвандҳои Quick Start ба саҳифаҳои дуруст ишора мекунанд
- [ ] Пайдарпаии номаи хушомад танзим ва санҷида шудааст
- [ ] Мақолаҳои пойгоҳи дониш нашр шудаанд ва дастрасанд
- [ ] Саҳифаи Account маълумоти дурусти нақша ва имконоти upgrade-ро нишон медиҳад

### Ҳуқуқӣ ва тиҷоратӣ {#legal-and-business}

- [ ] Шартҳои хизматрасонӣ нашр шудаанд ва аз checkout пайванд доранд
- [ ] Сиёсати махфият нашр шудааст ва дастрас аст
- [ ] Сиёсати баргардонидани маблағ муайян ва ҳуҷҷатгузорӣ шудааст
- [ ] Шахси ҳуқуқии тиҷоратӣ барои гирифтани пардохтҳо таъсис дода шудааст
- [ ] Дарвозаи пардохт дар режими зинда аст (на режими санҷишӣ)
- [ ] Танзимоти андоз барои қаламрави ҳуқуқии шумо дуруст аст

## Оғози нарм ё оғози қатъӣ {#soft-launch-vs-hard-launch}

Оғози думарҳилавиро баррасӣ кунед:

### Марҳилаи 1: Оғози нарм {#phase-1-soft-launch}

5-10 соҳиби студияи фитнесро даъват кунед, ки пеш аз оғози оммавӣ сабтином кунанд. Инҳо муштариёни бетаи шумо мебошанд. Ба онҳо тахфифи назаррас пешниҳод кунед (50% тахфиф барои тамоми умр, ё 3 моҳ ройгон) дар ивази:

- Бозخӯрди ростқавлона дар бораи таҷрибаи сабтином ва onboarding
- Иҷозат барои истифодаи сайти онҳо ҳамчун намунаи намоишӣ
- Гузориш додани ҳар гуна хатогиҳо ё мушкилоте, ки дучор мешаванд

Ин ба шумо бозخӯрди воқеии муштариён ва сайтҳои зинда медиҳад, то пеш аз кушодан ба омма онҳоро намоиш диҳед.

### Марҳилаи 2: Оғози оммавӣ {#phase-2-public-launch}

Пас аз ворид кардани бозخӯрди оғози нарм:

- Дарвозаи пардохтро ба режими зинда гузаронед
- Сайти маркетингии худро нашр кунед
- Ҷалби муштариёнро оғоз кунед (Дарси 11)
- Дар каналҳои марбут ба саноати фитнес эълон кунед

## Амалҳои рӯзи оғоз {#launch-day-actions}

Дар рӯзе, ки ба омма мебароед:

1. **Ба пардохтҳои зинда гузаред** -- режими санҷиширо дар дарвозаи пардохти худ ғайрифаъол кунед
2. **Боз як бор санҷед** -- бо пардохти воқеӣ як сабтиноми пурраи санҷишӣ анҷом диҳед (баъд ба худ маблағро баргардонед)
3. **Бодиққат назорат кунед** -- хатогиҳо, сабтиномҳои ноком ё мушкилоти пардохтро пайгирӣ кунед
4. **Дастрас бошед** -- муштариёни воқеии аввалини шумо шояд ба кӯмак ниёз дошта бошанд, ва ҷавоби зуд эътимод месозад
5. **Кӯтоҳ ҷашн гиред** -- сипас ба кор баргардед

## Чӣ метавонад нодуруст шавад {#what-can-go-wrong}

Барои инҳо омода бошед:

- **Мушкилоти дарвозаи пардохт**: Тамоси дастгирии провайдери дарвозаи худро омода дошта бошед
- **Мушкилоти сертификати SSL**: Бидонед, ки чӣ гуна сертификатҳоро санҷед ва нав кунед
- **Нокомии расонидани имейл**: Санҷед, ки имейлҳо воқеан мерасанд (ҷузвдонҳои спамро санҷед)
- **Иҷроиш зери сарборӣ**: Агар афзоиши ногаҳонии трафик гиред, бидонед, ки чӣ гуна hosting-и худро васеъ кунед
- **Сардаргумии муштариён**: Пойгоҳи дониш ва каналҳои дастгирии худро омода дошта бошед

## Он чизе ки дар ин дарс сохтем {#what-we-built-this-lesson}

- **Рӯйхати ҳамаҷонибаи санҷиши пеш аз оғоз**, ки инфрасохтор, мундариҷа, пардохтҳо ва масъалаҳои ҳуқуқиро фаро мегирад
- **Стратегияи оғози нарм** барои гирифтани бозخӯрди воқеӣ пеш аз баромадан ба омма
- **Равандҳои рӯзи оғоз** барои ба таври зинда баромадан бо итминон
- **Нақшаи эҳтиётӣ** барои мушкилоти маъмули рӯзи оғоз
