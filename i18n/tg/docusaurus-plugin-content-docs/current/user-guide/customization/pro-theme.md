# Тағйири панел администраторӣ бо PRO Theme {#customizing-your-admin-panel-with-pro-theme}

Ин аст, ки ҳангори корҳо бо муштабиён, мо интихоб мекунем, ки ба онҳо таҷрибаи махсус дар дашбор администри WordPress пешниҳод кунем. Агар шумо оフтани назар барои назари ва ҳиссият профессионалтар ҷустӣ мекунед, PRO Theme метавонад ба шумо кӯмак расонад.

PRO Theme як темаи администраторӣ аст, ки барои кӯмак вам дар тағйири дашборҳои муштабиёнаи худ таҳия шудааст. Ин инчунин қисми асосии плагини премиум Ultimate Multisite-ист.

Баъди он ки онро насб кардед, ба саҳифаи _Plugins_ равед ва барои фаъол кардани он дар тармоҳ (Network) худро клик кунед.

<!-- Скриншот мавҷуд нест: саҳифаи WP Plugins, ки дасти фаъоли кардани PRO Theme-ро барои Тармоҳ нишон медиҳад -->

_Онро дар саҳифаи Plugins-и худ ба таври тармоҳ фаъол кунед._

**ЭТО НОТА:** _Агар шумо уни фақат коркардаро илова кунед, он муштариённинг интерфейсида фақат ушбу фойдаланувчига темани қўшиш учун белгилаш мумкин бўлган вариант сифати кўрсатилган._

Энди сизнинг администратор интерфейсингиз тўлиқ қайта лойиҳаланди! Сонини чап томонда "PRO Theme" деб номланган янги меню ҳам бор.
