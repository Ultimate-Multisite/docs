# Қобилияти Theme Builder: Таҳия ва Фаъолсозӣ кардани Themeҳои Блок {#theme-builder-abilities-scaffold-and-activate-block-themes}

Superdav AI Agent v1.12.0 ду қобилияти қавиро тақдим мекунад, ки имкон медиҳанд шумо тапряин (custom) block themes-ро мустақиман аз интерфейси чат таҳия ва ворид кунед.

## Обзор {#overview}

Қобилияти **scaffold-block-theme** ва **activate-theme** ба агентоҳо имкон медиҳанд:
- Themeҳои блок пурраванд ва омода барои истеҳсолотро барварои тасвирҳои шумо таҳия намоянд.
- Themeҳоро бе даст ворид кардани корбар, мустақиман дар сайти худ фаъол кунанд.
- Тасмимҳои тарҳреро бо ройхати раҳнамоӣ таҳия намоянд ва маҷмӯаи назаррас (cohesive) созад.

## Таҳияи Theme Block {#scaffold-block-theme}

Қобилияти **scaffold-block-theme** як theme block-и WordPress-ро бо сохтори пурраи theme, аз ҷумла:

- конфигуратсияи `theme.json` бо токенҳои дизайн (design tokens)
- файлоҳо барои قالب‌های رایج (common layouts)
- стил‌ها و вариационҳои بلاки хосиятӣ (custom block styles and variations)
- метаданбарии тема ва اعلامیهҳо барои дастгирӣ (theme metadata and support declarations)

### Чӣ тавр даъват кардан (How to Invoke) {#how-to-invoke}

Дар муошират бо Агент Superdav AI дар чат-и худ, шумо метавонед тақозои таҳияи темаро пешниҳод кунед:

```
"Таҳияи тему بلاки бо номи 'Modern Agency' бо схема ранги сабз ва оқай (blue and white), типографикае бе sans-serif, ва тарҳێکی мутахассис."
```

Агент ин корҳоро иҷро мекунад:
1. Аз муошират тавзеҳотҳои дизайн-и шуморо ҷамъ орад
2. Ҳамаи сохтори темуро таҳия кунад
3. Ҳамаи файлоҳои зарурӣ барои темаро сохта ва таҳия мекунад
4. Темаро барои фаъол шудан омода мекунад

### Натиҷаи интизолшаванда (Expected Output) {#expected-output}

Вақте ки қобилият ба таври муваффақ иҷро карда шавад, шумо инҳоро мебинед:

✓ Theme "Modern Agency" scaffolded successfully
  Location: /wp-content/themes/modern-agency/
  Colors: Primary #0066CC, Secondary #FFFFFF
  Typography: Inter (sans-serif)
  Status: Ready to activate

## Активация темы {#activate-theme}

Возможность **activate-theme** позволяет переключить ваш сайт на только что созданную или существующую блочную тему.

### Как вызвать {#how-to-invoke-1}

После создания темы вы можете активировать ее немедленно:

```
"Активировать тему Modern Agency"
```

Или активировать любую существующую тему:

```
"Переключиться на тему Twentytwentyfour"
```

### Ожидаемый результат {#expected-output-1}

При успешной активации:

- Тсвӣ тасдиқ кардани темаи фаъол
- Номи темаи қаблӣ (барои шавқат)
- Сардодаи сайт, ки ин омодалик дар он ҷо аст
- Ҳама гуна ноттибҳо барои инфитишоти тема

Мисол:
```
✓ Тема ба таври муваффақ фаъол карда шуд
  Темаи фаъол: Modern Agency
  Темаи қаблӣ: Twentytwentyfour
  Дар сардодаи ин сайт фаъол аст: https://yoursite.com
  Нотаб: Барои тасдиқи тақсимоти онҳо ба саҳифаи асосии худ равед.
```

## Интеграция: Навиштани ва Темаро Фаъол кардан (Workflow) {#workflow-scaffold-and-activate}

Ин кор通ӣ одатан ду қобилиятро муттаҳид мекунад:

1. **Ҷониббарии тема**: "Темаи блоК барои саҳифаи асосии SaaS-и ман сохтаред"
2. **Агент темаро ҷониббар менамояд**: Файлҳо ва токенҳои тарҳро таҳия мекунад
3. **Баррадани муҳокила ва такмил**: Агар лозим бошад, дар бораи тағйирот дар тарҳ сухан гӯед
4. **Фаъол кардан**: "Интегратсияро ҳоло фаъол кунед"
5. **Тасдиқ**: Ба сайти худ равед, то тасдиқ кунед, ки тарҳи нав фаъол карда шудааст

## Токенҳои Тарҳ ва Ташкил (Design Tokens and Customization) {#design-tokens-and-customization}

Темаҳои scaffolded аз дизайн-токеҳои WordPress (аз ҳисоби `theme.json`) барои:

- **Рангҳо**: Палитраи асосӣ, дуюм, акцентӣ, худкор
- **Типографика**: FontFamilyҳо, аҳамиятҳо, вазъияҳо, баландшавии қаторҳо
- **Фарқҳо (Spacing)**: Padding, margin, масофаҳо (gap scales)
- **Ҳадислаҳо (Borders)**: Токенаҳои радиус ва ширини ҳаддиҳо

Ин токенҳо дар `theme.json` марказӣ шудаанд, ки тағйир додани тамоми системаи дизайн шуморо аз як файл осон мекунад.

## Маҳдудиятҳо ва қайдҳо {#limitations-and-notes}

- Темаҳо дар папка `/wp-content/themes/` scaffold карда мешаванд ва бояд аз қоидаҳои номини WordPress пайравӣ кунанд
- Фаъолсозӣ (Activation) ба иҷозатҳои мувофиқ дар сайти WordPress шумо ниёз дорад
- Коди PHP-и худкор дар темаҳо кам аст; барои функсияҳои мураккаб аз плагиноҳо истифода баред
- Темаҳои Block беҳтарин кор мекунанд бо WordPress 5.9 ва нокиш

## Ҳалли мушкилот (Troubleshooting) {#troubleshooting}

**Тема пас аз scaffolding пайдо نمی‌شود**
- Барои тасдиқ кардан, ки дастии (directory) тема вуҷуд дорад ва иҷозатҳои мувофиқро дорад.
- Таъкид кунед, ки `theme.json` формати JSON дуруст аст.
- Ишмоли номи тема ба темаҳои мавҷуда тавонониши (conflict) накунад.

**Фаоли шаваӣ натиҷа дода нед**
- Тасдиқ кунед, ки шумо иҷозатҳои администратор доред.
- Барои тасдиқ кардан, ки дастии тема ба WordPress хондашаванда аст.
- Логҳои хатои WordPress-ро барои маълумот баррасӣ кунед.

**Design tokensҳо мегиранд (apply) нашаванд**
- Формати `theme.json` дуруст аст ё не, тасдиқ кунед.
- Ҳамаи плагинҳои кэшинг (caching plugins)-ро тоза кунед.
- Таъкид кунед, ки версияи WordPress-и шумо аз токенҳое, ки истифода мебаред, дастгирӣ мекунад.

## Қадамҳои минбаъда {#next-steps}

Баъди фаъолисозии темаи худ, шумо метавонед:
- аз **Design System Aesthetics skill** барои такмили типографика, ранги ва фасолатноки (spacing) истифода баред.
- тартиби навъи блокларро тавассути WordPress block editor хусусиятҳои мушаххасӣ тағйир диҳед.
- CSS махсусро дар файли `style.css`и тема илова кунед.
- шаблонҳои блоки махсусан барои навъҳои гуногуни саҳифаҳо созед.
