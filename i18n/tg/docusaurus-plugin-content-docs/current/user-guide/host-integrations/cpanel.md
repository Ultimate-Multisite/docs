# Интеграция с cPanel

## Обзор {#overview}
cPanel — это одна из самых популярных панелей управления веб-хостингом, используемая многими хостинг-провайдерами как общими, так и выделенными. Эта интеграция позволяет автоматически синхронизировать домены между Ultimate Multisite и cPanel, что дает вам возможность автоматически добавлять псевдонимы доменов (aliases) и поддомены в вашу учетную запись cPanel.

## Возможности {#features}
- Автоматическое создание дополнительных доменов (addon domain) в cPanel
- Автоматическое создание поддоменов в cPanel (для установок с поддоменными мультисайтами)
- Удаление домена при удалении настроек сопоставления (mappings)

## Требования {#requirements}
Следующие константы должны быть определены в вашем файле `wp-config.php`:

```php
define('WU_CPANEL_USERNAME', 'your_cpanel_username');
define('WU_CPANEL_PASSWORD', 'your_cpanel_password');
define('WU_CPANEL_HOST', 'your_cpanel_host');
```

Опционально, вы также можете определить:

```php
define('WU_CPANEL_PORT', 2083); // По умолчанию 2083
define('WU_CPANEL_ROOT_DIR', '/public_html'); // По умолчанию /public_html
```

## Инструкции по настройке {#setup-instructions}

### 1. Получите учетные данные cPanel {#1-get-your-cpanel-credentials}

1. Получите свое имя пользователя и пароль cPanel от вашего хостинг-провайдера.
2. Определите свой хост cPanel (обычно это `cpanel.yourdomain.com` или `yourdomain.com:2083`).

### 2. Добавьте константы в wp-config.php {#2-add-constants-to-wp-configphp}

Добавьте следующие константы в ваш файл `wp-config.php`:

```php
define('WU_CPANEL_USERNAME', 'your_cpanel_username');
define('WU_CPANEL_PASSWORD', 'your_cpanel_password');
define('WU_CPANEL_HOST', 'your_cpanel_host');
```

Опционально, вы можете настроить порт и корневой каталог:

```php
define('WU_CPANEL_PORT', 2083); // Измените, если ваш cPanel использует другой порт
define('WU_CPANEL_ROOT_DIR', '/public_html'); // Измените, если корень ваших документов отличается
```

### 3. Включите интеграцию {#3-enable-the-integration}

1. Дар администринг WordPress-и худUltimate Multisite равите > Settings баровед
2. Барои "Domain Mapping" (Сопоставление доменов) таб равите
3. Пойдем вниз к "Host Integrations" (Интеграции хостинга)
4. cPanel integration-ро включите
5. Нажмите "Save Changes" (Сохранить изменения)

## Дар чӣ гузашта аст {#how-it-works}

### Addon Domains (Доменҳои иловагӣ) {#addon-domains}

Вақте ки домен дар Ultimate Multisite сопостав карда мешавад:

1. Интегратсия дар API-и cPanel дархости пешниҳоди домен ҳамчун домени иловагӣ месозад
2. Домен барои нишон додани роҳи асосии (root directory) шумо таҳия карда мешавад
3. Вақте ки сопостав кардани домен бартараф карда мешавад, интегратсия ин домени иловагӣ аз cPanel бартараф мекунад

### Subdomains (Поддоменҳо) {#subdomains}

Барои намудани Multisite бо поддоменҳо, вақте ки сайт нав сохта мешавад:

1. Интегратсия қисми поддоменро аз домени пурра гирифта мегирад
2. Он дар API-и cPanel дархости пешниҳоди поддоменро месозад
3. Поддомен барои нишон додани роҳи асосии (root directory) шумо таҳия карда мешавад

## Нотҳои муҳим {#important-notes}

- Интегратсия аз API2-и cPanel барои муошират бо ҳисоби cPanel-и шумо истифода мебарад
- Ҳисоби cPanel-и шумо бояд иҷозатҳо барои илова кардани доменҳои иловагӣ ва поддоменҳо дошта бошад
- Баъзе пешгирвандагони хостинг метавонанд миқдори доменҳои иловагӣ ё поддоменҳоро, ки шумо метавонед сохтаред, маҳдуд кунанд
- Интегратсия барои таҳияи DNS (DNS configuration) кор намекунад; инчунин шумо бояд доменҳоро ба нишон додани IP-и сервер хонаш ишора диҳед

## Танқисоти мушкилот (Troubleshooting) {#troubleshooting}

### Masolati пайвастшавии API {#api-connection-issues}
- Таъкид кунед, ки истинод ва пароли cPanel-и шумо дуруст аст
- Муайян кунед, ки хостинг cPanel-и шумо дуруст ва дастрас аст
- Ишфоҳ кунед, ки ҳисоби cPanel-и шумо иҷозатҳои зарурӣ дорад

### Домен нагозошта шуд {#domain-not-added}
- Логҳои Ultimate Multisite-ро барои ҳама гуна хатоҳо тафтиш кунед
- Таъкид кунед, ки домен аз cPanel аллакай дода нашудааст
- Ишфоҳ кунед, ки ҳисоби cPanel-и шумо ба маҳдуди доменҳои иловагӣ ё поддоменҳо расидааст

### Masalalari Sertifikатҳои SSL {#ssl-certificate-issues}

- Интеграция истеҳсоли сертификатҳои SSL-ро идора намекунад.
- Барои истеҳсоли сертификатҳои SSL барои доменҳои худ, шумо бояд аз воситаҳои SSL/TLSи cPanel ё хусусияти AutoSSL истифода баред.
- Барои ин, метавонед аз хидмаҳое мемоносозед, ба монанди Let's Encrypt бо AutoSSL-и cPanel.
