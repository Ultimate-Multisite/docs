# Мувофидабаи Миёнандагон ва Рӯйхати Масъалаҳо

Gratis AI Agent v1.5.0 системаи мувофиқи он барои фидбэк ва гузориш дар бораи масъалаҳоро пешниҳод мекунад, ки ирсоли интихосони охирин корбар (end users) аз ҷониби худашон ба таври доимӣ имкон медиҳад, ки ҷавоиби номувофиқро нишон диҳанд ва мушкилотро мустақиман аз интерфейси чат гузориш намоянд. Ин система аз идораи розиатӣ (consent management), командаи гузориши автоматикӣ ва тасони мушкилот аз ҷониби AI дар паҳнпайваст (backend) иборат аст.

## Датти "Норози" (Thumbs-Down Button)

Ҳар як пиёшае, ки аз ҷониби касеи ёрдаи AI фиристода мешавад, дафтери **датт "норози"** (👎) нишон дода мешавад. Вақте корбар ба он клик мекунад, он метавонад ҷавоибро ҳамчун номувофиқ ё нодуруст қайд намояд.

- Ин дафт дар ҳолати баррафстани курсор (hover) пасли ҳар як пиёшаи ёрда меояд.
- Клики ба он **Modal-и Розиатӣ барои Фидбэк** (Feedback Consent Modal) ройгон мекунад.
- Ин фидбэк бо пайвасткунии пайвастаи муозино ва пиёшаи мушаххас тартиби муозино мебошад.

## Modal-и Розиатӣ барои Фидбэк (Feedback Consent Modal)

Вақте корбар ба дафти "норози" клик мекунад, пеш аз фиристодани ҳар як маълумот, як modal-и розиатӣ пайдо мешавад. Ин modal:

- Маълумотҳоеро, ки ба даст оварда мешаванд (қисми муозино, контексти браузер), шарҳ медиҳад.
- Аз корбар хоҳиш мекунад, ки онро барои сахт кардани ин маълумотро тасдиқ кунад.
- Як майдони матни озодӣ барои корбарон мегузорад, то онҳо тавсифи мушкилотро диданд бивашанд.
- Варианти **"Ислоҳ" (Cancel)**-ро пешниҳод мекунад, ки бе фиристодан онро бартарӣ кунад.

Ҳатто доди фидбэк то вақте корбар худаш тасдиқ намояд, қайд карда намешавад.

## Баннер Фидбэки Автоматик (Auto-Prompt Feedback Banner)

Дар охири муозино, ёрда метавонад **баннери фидбэки автоматик**-ро нишон диҳад — як пассаи напурсозӣ, ки аз он пурсида мешавад, ки сессия кӯмак кардааст ё нест.

Ин баннер мувофиқи дароз ва натиҷаҳои муозино, беавтоматик пайдо мешавад. Он ба ҳамон равиши фидбэки дафти "норози" пайваст аст. Корбарон метавонанд без пешниҳоди фидбэк онро бартарӣ кунанд.

## Командаи /report-issue (Slash Command)

Корбарон метавонанд бо навиштани `/report-issue` дар ворид кардани чат, ройх овардани равиши фидбэки мувофиқро ба таври доимӣ иҷро кунанд. Ин команда:

- Мӯҳарти Иҷозат барои фидбэк (Feedback Consent Modal) дарҳол кушода мешавад.
- Тавсифи матн бо контексти муозираи мувофиқ ба тасвири мавҷуи гуфтугӣ пур карда мешавад.
- Изохиёти иловагӣ ба модарӣ (submission) пеш аз фиристодан ба кор мегиранд.

Команда `/report-issue` дар ҳамаи режими чат (inline, floating widget, full-screen) мавҷуд аст.

## AI-Assisted Triage (Тақсимкунандаи Ҷавоби Илмӣ)

Фидбэки пешниҳодшуда ба қатри тақсимкунандаи AI равона карда мешавад, ки:

- Ҳар як гунаи лоиҳаро тақсим мекунад (хато фактии, ҷавоби номувофиқ, мувофиқ накардани мувофиқат ва ғайра).
- Метамаълумотҳои мувофиқи муозираи муозираи гуфтугӣ илова мекунад.
- Хулосаи тақсимкунандаро ба **Feedback Endpoint** (аз [Settings > Advanced](#settings-advanced) дидаред) равона мекунад.

Ин вақти баррасии рухдонии инсонӣ-ро аз он кам мекунад, дар ҳоле ки мушкилоти муҳимтарин аввал нишон дода мешаванд.

## Settings > Advanced {#settings-advanced} (Ҳусусиятҳо > Муассир)

Барои фаъол кардани пахши фидбэк, ин сохторҳоро дар **Gratis AI Agent → Settings → Advanced** тартиб диҳед:

| Field | Description (Тавсиф) |
|---|---|
| **Feedback Endpoint URL** | URL-и он ки дар он дархостҳои POST бо маҷмӯаҳои фидбэк (JSON) қабул карда мешаванд. |
| **Feedback API Key** | Токен Bearer, ки барои тасдиқи пешниҳоди фидбэк дар занҷираи `Authorization` фиристода мешавад. |

Ҳайсиятҳои ҳардуи матнро худкор (blank) гузоред, то ҷамъоварии фидбэк ба даст овардашуда нест. Барои истифодаи дастурҳои фидбэки инсонҳо боқӣ мемонад, аммо пешниҳоди онҳо равона карда намешаванд.

:::tip
Маҳаллаи фидбэк бояд ба таври JSON бо ҳад аз `message_id`, `conversation_id`, `feedback_text` ва `triage_category` лоиҳаҳо қабул намояд. Ба ҳуҷҷатҳои пешниҳоди он равонаи муқарраршуда барои схема интизорид мебинед.
:::
