# Поиска дар Интернет

Gratis AI Agent v1.5.0 қобилияти **Internet Search** (Поиск в Интернете) -ро илова кун муошир иттилотни веб-сайтидан даровида гирифта омода месозад. Инчуна он аз [Brave Search API](https://brave.com/search/api/) ё [Tavily API](https://tavily.com/) дастгирӣ мекунад.

## Чӣ гуна кор мекунад {#how-it-works}

Вақте ки поиск дар Интернет фаъол карда мешавад, ассистент ҳангоми он ки маълум кунад, ки савол ба иттилооти ҳозира ё бехории беруна ниёз дорад — масалан, рӯзҳои охир, нархҳои зинда ё ҳуҷҷатҳое, ки аз вақти омӯзиши модели тағйир ёфтаанд — ӯ аз ҷониби шумораи муайян кардашудаи пешгирии поиска (search provider) истафода мекунад.

Натиҷаҳо дар вақти воқеӣ гирифта шуда, пеш аз он ки ассистент ҷавоб диҳад, ба контексти ӯ ворид мешаванд. Ассистент индоза мекунад, ки он аз натиҷаҳои поиск барои ҷавоби савол истифода кардааст.

## Фаъол кардани Internet Search {#enabling-internet-search}

Поиска дар Интернет барои дониши API-и пешгирии интихобшудаи шумо лозим аст. Барои таҳияи он:

1. Ба **Gratis AI Agent → Settings → Advanced** равед.
2. Ҳамаҷонибаи **Internet Search Provider** (Пешгирии Internet Search) -ро пайдо кунед ва яна **Brave Search** ё **Tavily**-ро интихоб кунед.
3. API key-и худ-ро дар майдони мувофиқ ворид кунед. Линкиҳо барои дониши ҳар як майдон паён мешаванд.
4. Ба **Save Settings** (Ҳамаҷонибаи танзимҳо)-барнавор кунед.

Баъди он ки key-ро савдо кардаед, қобилияти Internet Search омода барои ассистент мебошад.

## Brave Search {#brave-search}

### Гирифтани API Key-и Brave Search {#obtaining-a-brave-search-api-key}

1. Ба [Brave Search API page](https://brave.com/search/api/) равед.
2. Барои намунаи (plan) ройгонӣ кунед. Дар сатҳи бепул бо маҳдудият муайян кардани дарвозаҳо ҳар моҳ мавҷуд аст.
3. API key-и худ-ро аз дашбори муаллифи Brave Search нусхабардоред.
4. Онро ба майдони **Brave Search API Key** дар танзимҳои Gratis AI Agent ирсолеш кунед.

### Маҳдудиятҳои истифода {#usage-limits}

Использование оплачивается Brave Search на основе количества выполненных запросов. Каждый ответ ИИ, который инициирует поиск, считается одним запросом. Следите за своим использованием в [панели разработчика Brave Search](https://brave.com/search/api/), чтобы избежать неожиданных списаний.

## Tavily {#tavily}

Superdav AI Agent v1.10.0 добавляет **Tavily** как альтернативный провайдер поиска в интернете, предлагая более богатые результаты поиска и расширенные возможности для исследований.

### Получение ключа API Tavily {#obtaining-a-tavily-api-key}

1. Перейдите на [страницу API Tavily](https://tavily.com/).
2. Зарегистрируйтесь. Доступен бесплатный уровень с лимитом запросов в месяц.
3. Скопируйте свой ключ API из панели управления Tavily.
4. Вставьте его в поле **Tavily API Key** в настройках Gratis AI Agent.

### Лимиты использования {#usage-limits-1}

Использование оплачивается Tavily на основе количества выполненных вызовов API. Каждый ответ ИИ, который инициирует поиск, считается одним вызовом. Следите за своим использованием в [панели управления Tavily](https://tavily.com/), чтобы избежать неожиданных списаний.

## Отключение поиска в интернете {#disabling-internet-search}

Удалите ключ API из поля активного провайдера поиска и сохраните изменения. Возможность поиска в интернете больше не будет предлагаться ассистенту.

:::note
Поиск в интернете добавляет задержку ответам, потому что ассистент должен дождаться результатов поиска перед генерацией ответа. Для задач, требующих высокой оперативности, рассмотрите, нужен ли поиск в реальном времени или достаточно встроенных знаний ассистента.
:::
