# Идоракун кардани لینکҳои муштара барои нақшаҳо (Plans)

Ultimate Multisite ба шумо имконият медиҳад, ки لینکҳои нақшаҳоро бо истилоҳот ва муштариёнатон худатон сарфа кунед. Барои ин, ба ҳар як саҳифаи тасвири (edit pages) нақшаҳои худ равед ва ба дафтарчаи "_Click to copy the Shareable Link_" клик кунед.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_Дар саҳифаи тасвири Нақша (Plan's Edit Page), метавонед дафтарчаи сохтани لینکҳои муштараро пайдо кунед._

Бош доред, ки ин механизм аз таври стандарт ҳисоби корӣ барои сохтани لینک истифода мебарад. Ин маънои онро дорад, ки агар ҳисоби корнии воситаи шумо моҳӣ бошад, масалан, дар охир, шумо як لینک ба ин монанд доред: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Агар шумо хоҳед, ки لینکҳои муштара ба таври артома (quarterly) муайян шавад, бояд 3-ро дар охир илова кунед:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Барои интихоби нақшаи _premium_ барои дараҷаи солӣ, шумо 1 ва калимаро илова мекунед:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Дар ин ҷо ёд гиред, ки шумо танҳо як لینک барои таври ҳисоби корӣ варзиёти гуногун надошта метавонед. Шумо бояд барои ҳар як дараҷаи ҳисоби корнии худ як لینک мушаххас сарфа кунед.

Вақте ки шумо ин لینکро ба муштариёнатон пешниҳод мекунед, онҳо ба қадами аввали равиши қайд (signup flow) ирсоли мешаванд.
