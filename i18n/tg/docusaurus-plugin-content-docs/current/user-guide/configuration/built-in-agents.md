# Agent-ҳои дарунсохт

Gratis AI Agent v1.9.0 панҷ agent-и дарунсохтро пешниҳод мекунад, ки ҳар кадом бо маҷмӯаи мутамаркази абзорҳо, system prompt-и мутобиқшуда ва пешниҳодҳои оғозӣ барои вазифаҳои маъмул дар он соҳа пешакӣ танзим шудаанд. Гузариш байни agent-ҳо он чизеро, ки ёвар метавонад анҷом диҳад ва тарзи посухдиҳии онро тағйир медиҳад — бе ҳеҷ гуна танзим аз ҷониби шумо. Superdav AI Agent v1.18.0 метавонад ба ин равандҳои корӣ абзорҳои вобаста ба ҷадвал, сабтҳои ёдрасӣ, дарвозаҳои тасдиқ ва огоҳиномаҳои SMS илова кунад, вақте ки ҳамгироиҳои марбут танзим шудаанд.

## Agent чист?

Ҳар agent як профили танзимоти номдор аст, ки инҳоро муттаҳид мекунад:

- **Абзорҳо** — қобилиятҳое, ки agent иҷозати фаро хондани онҳоро дорад (масалан, Content Writer ба қобилиятҳои эҷоди навиштаҳо дастрасӣ дорад; Design Studio ба қобилиятҳои CSS ва theme.json дастрасӣ дорад)
- **System prompt** — дастурҳое, ки оҳанг, афзалиятҳо ва маҳдудиятҳои agent-ро муқаррар мекунанд
- **Пешниҳодҳо** — prompt-ҳои пешакӣ навишташуда, ки дар интерфейси чат нишон дода мешаванд, то ба шумо барои зуд оғоз кардан кумак кунанд

## Дастрасӣ ба Agent Picker

1. Панели **Gratis AI Agent**-ро дар панели канории мудири WordPress кушоед.
2. Дар кунҷи болоии чапи сарлавҳаи чат **нишонаи agent**-ро клик кунед (нишона барои инъикоси agent-и фаъол тағйир меёбад).
3. **Agent Picker** ҳамчун рӯйпӯши form-table кушода мешавад. Ҳар agent бо нишона, ном ва тавсифи яксатрааш рӯйхат мешавад.
4. Барои фаъол кардани он сатри agent-ро клик кунед. Сарлавҳаи чат фавран нав мешавад.

Шумо инчунин метавонед дар миёни суҳбат agent-ҳоро иваз кунед — system prompt-и agent-и нав аз паёми навбатӣ эътибор пайдо мекунад.

## Панҷ Agent-и дарунсохт

### Content Writer

**Тамаркуз:** Эҷод ва таҳрири навиштаҳо, саҳифаҳо ва формаҳои тамос.

**Абзорҳои дастрас:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`. Бо фаъол будани ҳамгироиҳои Superdav AI Agent v1.18.0, контексти тақвими танзимшуда, дарвозаҳои тасдиқ, ёдраскуниҳо ва абзорҳои огоҳиномаи SMS низ метавонанд барои равандҳои кории тасдиқшуда дастрас бошанд.

**Чиро хуб анҷом медиҳад:**
- Таҳия ва нашри навиштаҳои блог аз мухтасар ё нақша
- Эҷоди гурӯҳҳои саҳифаҳои фуруд барои сайти нав
- Сохтани формаҳои тамос ва дархост
- Танзими тасвирҳои барҷаста барои навиштаҳо аз URL ё ҷустуҷӯ
- Таҳияи паёмҳои пайгирии чорабинӣ аз контексти танзимшудаи Google Calendar, сипас таваққуф барои тасдиқ пеш аз фиристодани огоҳиномаҳо

**Пешниҳодҳои оғозӣ:**
- *Як навиштаи блогии 500-калимагӣ дар бораи манфиатҳои WordPress multisite нависед.*
- *Саҳифаҳои Дар бораи мо, Хизматҳо ва Тамосро эҷод карда, онҳоро нашр кунед.*
- *Ба саҳифаи Тамос як формаи дархости фармоиш илова кунед.*
- *Барои иштирокчиёни чорабинии тақвимии танзимшудаи фардо як ёдрасӣ таҳия кунед ва пеш аз фиристодани он барои тасдиқ интизор шавед.*

### Site Builder

**Тамаркуз:** Эҷоди вебсайт аз аввал то охир аз як prompt.

**Абзорҳои дастрас:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`. Бо Superdav AI Agent v1.18.0, абзорҳои танзимшудаи managed-service, тасдиқ, ёдрасӣ, тақвим ва SMS метавонанд дар ҷое дастрас бошанд, ки мудирон онҳоро фаъол мекунанд.

**Чиро хуб анҷом медиҳад:**
- Тавлиди нақшаи бисёрмарҳилавии сохтани сайт барои навъи тиҷорати тавсифшуда
- Иҷрои ҳар марҳила ба таври худкор — сохтор, муҳтаво, паймоиш, дизайн
- Барқарор шудан аз хатоҳо дар миёнаи нақша бе зарурати дахолати дастӣ
- Насби plugin-ҳои тавсияшуда ҳамчун қисми сохт
- Эҷоди формаҳои тамос мустақиман аз интерфейси чат (Superdav AI Agent v1.10.0+)
- Ҳамоҳангсозии ёдраскуниҳои оғоз ё пайгирии иштирокчиён бе огоҳиномаҳои такрорӣ, вақте ки дарвозаҳои тасдиқ ва сабтҳои ёдрасӣ фаъоланд

**Пешниҳодҳои оғозӣ:**
- *Як сайти портфолиои аксбардорӣ бо навъи навиштаи галерея, саҳифаи фармоиш ва формаи тамос созед.*
- *Як вебсайти тарабхона бо менюи онлайн, соатҳои корӣ ва формаи дархости фармоиши миз эҷод кунед.*
- *Як сайти машварати фрилансӣ бо саҳифаҳои хизматрасонӣ, бахши портфолио ва блог роҳандозӣ кунед.*
- *Бо истифода аз site builder ба саҳифаи Тамос формаи тамос илова кунед.*
- *Пас аз тасдиқ шудани рӯйхати санҷиши оғози сайт, ба тамоси ҷонибдори манфиатдори танзимшуда ёдрасии SMS фиристед.*

### Design Studio

**Тамаркуз:** Фармоишисозии визуалӣ — рангҳо, типография, CSS ва намунаҳои блок.

**Абзорҳои дастрас:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Чиро хуб анҷом медиҳад:**
- Татбиқи preset-ҳои номдори theme (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- Танзими дақиқи типография ва палитраҳои ранги глобалӣ тавассути theme.json
- Ворид кардани CSS-и фармоишӣ барои тағйироти махсуси бренд
- Гирифтани screenshot-и саҳифа ва баррасии он барои масъалаҳои дизайн

**Пешниҳодҳои оғозӣ:**
- *Preset-и warm-editorial-ро татбиқ кунед ва сипас ранги асосиро ба #2d6a4f гузоред.*
- *Аз саҳифаи асосӣ screenshot гиред ва бигӯед, ки чиро беҳтар мекардед.*
- *Як намунаи блоки hero-и аз нав истифодашаванда бо тасвири пасзаминаи пурпаҳно ва сарлавҳаи марказшуда эҷод кунед.*

### Plugin Manager

**Тамаркуз:** Кашф, насб ва идоракунии WordPress plugin-ҳо.

**Абзорҳои дастрас:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Чиро хуб анҷом медиҳад:**
- Тавсия додани беҳтарин plugin барои мавриди истифодаи тавсифшуда
- Насб кардани бастаҳои қобилият аз registry
- Аз назар гузаронидани феҳристи қобилиятҳои дастрас аз рӯи категория

**Пешниҳодҳои оғозӣ:**
- *Беҳтарин plugin барои феҳристи membership кадом аст?*
- *Бастаи қобилиятҳои WooCommerce-ро насб кунед.*
- *Ҳама бастаҳои қобилияти ecommerce-и дастрасро нишон диҳед.*

### Support Assistant

**Тамаркуз:** Посух додан ба саволҳо дар бораи муҳтавои сайт, танзимот ва конфигуратсияи WordPress.

**Абзорҳои дастрас:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Он чӣ корҳоро хуб анҷом медиҳад:**
- Ҷустуҷӯи танзимот ва имконоти ҷории сайт
- Шарҳ додани он ки кадом навъҳои навиштаҳо, таксономияҳо ва менюҳо дар сайт танзим шудаанд
- Посух додан ба саволҳои «ин танзимот чӣ кор мекунад?» тавассути хондани арзишҳои зинда
- Хизмат кардан ҳамчун қабати ташхисии танҳо-хонданӣ пеш аз ворид кардани тағйирот

**Пешниҳодҳои оғозӣ:**
- *Кадом plugin-ҳо ва танзимот ҳоло дар ин сайт фаъоланд?*
- *Ҳамаи навъҳои навиштаҳои фармоишии сабтшударо дар ин сайт номбар кунед.*
- *Кадом менюҳои паймоиш вуҷуд доранд ва онҳо дар куҷо таъин шудаанд?*

## Ҳамгироиҳои автоматсозии Superdav

Вақте ки ҳамгироиҳои Superdav AI Agent v1.18.0 танзим шудаанд, агентҳои дарунсохт метавонанд дар ҷараёнҳои кории автоматсозии бехатартари бо ҷадвал огоҳ иштирок кунанд:

- **Абзорҳои хондани Google Calendar** ба агентҳо имкон медиҳанд, ки пеш аз таҳияи кори пайгирӣ тақвимҳо ва рӯйдодҳои танзимшударо тафтиш кунанд.
- **Мутобиқсозии тамосҳо ва иштирокчиён** кӯмак мекунад, ки иштирокчиёни рӯйдод ба корбарони WordPress ё тамосҳои маълум мувофиқ карда шаванд.
- **Дарвозаҳои тасдиқи инсонӣ** амалҳои ҳассосро то он даме таваққуф медиҳанд, ки корбари ваколатдор онҳоро баррасӣ ва тасдиқ кунад.
- **Сабтҳои ёдрасӣ** ҳангоми кӯшиши дубора ё такрори вазифаҳои ҷадвалшуда аз огоҳиномаҳои такрорӣ пешгирӣ мекунанд.
- **Огоҳиномаҳои SMS-и TextBee** паёмҳои матнии танзимшударо танҳо вақте мефиристанд, ки эътимодномаҳои SMS ва иҷозатҳои ҷараёни корӣ фаъол бошанд.

Ҷараёни кории тавсияшуда: аз агент хоҳиш кунед, ки паём ё амалро омода кунад, дархости тасдиқро баррасӣ кунед, сипас иҷозат диҳед, ки амали тасдиқшуда идома ёбад. Барои ёдрасиҳои такроршаванда, несткунии такрории ёдрасиро фаъол нигоҳ доред, то ҳамон рӯйдод ё тамос такроран огоҳ карда нашавад.

## Фармоишии агентҳо

Ҳар як агенти дарунсохт метавонад тавассути филтри `gratis_ai_agent_agents` васеъ ё иваз карда шавад.

### Илова кардани дастури системавии фармоишӣ ба агенти мавҷуда

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### Сабт кардани агенти нав

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

Агенти нав фавран пас аз иҷро шудани филтр дар Agent Picker пайдо мешавад.

### Хориҷ кардани агенти дарунсохт

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
