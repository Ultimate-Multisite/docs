# Агенти вбудовані

Gratis AI Agent v1.9.0 постаёт с пятью встроенными агентами, каждый из которых предварительно настроен на определённый набор инструментов, индивидуальный системный промпт и стартовые предложения, соответствующие общим задачам в этой области. Переключение между агентами меняет то, что может делать ассистент и как он отвечает — без каких-либо настроек с вашей стороны.

## Что такое Агент?

Каждый агент — это именованный профиль конфигурации, который объединяет:

- **Инструменты (Tools)** — способности, которые разрешено вызывать агенту (например, у Контент-писателя есть доступ к функциям создания контента; в Дизайн-студии есть доступ к CSS и theme.json).
- **Системный промпт (System prompt)** — инструкции, которые задают тон, приоритеты и ограничения поведения агента.
- **Предложения (Suggestions)** — заранее написанные запросы, отображаемые в интерфейсе чата, чтобы помочь вам быстро начать работу.

## Доступ к Выбору Агентов (Agent Picker)

1. Откройте панель **Gratis AI Agent** в боковой панели админки WordPress.
2. Нажмите на **иконку агента** в левом верхнем углу заголовка чата (иконка меняется, чтобы отразить активного агента).
3. **Agent Picker** открывается как всплывающая таблица-форма. Каждый агент перечислен со своей иконкой, именем и однострочным описанием.
4. Нажмите на строку с агентом, чтобы активировать его. Заголовок чата обновляется немедленно.

Вы также можете переключаться между агентами в процессе разговора — системный промпт нового агента начинает действовать с berikutnya сообщения.

## Пять Встроенных Агентов

### Контент-писатель (Content Writer)

**Фокус:** Создание и редактирование постов, страниц и форм обратной связи.

**Доступные инструменты:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`

**Чӣ чӣ хуб мекунад:**
- Навиштани варситот (blog posts) аз як тасвир ё нақшаи мавзӯъ
- Эҷоди гурӯҳҳо аз саҳифаҳои Landing Page барои сайтҳои нав
- Сохтани формаҳои иртибот ва саволҳо

**Тавсияҳои оғозӣ:**
- *Як варситоти 500 калимаи дар бораи афзалиятҳои WordPress multisite нависед.*
- *Саҳифаҳои "Дар бораи мо", "Хизматрасониҳо" ва "Иртибот" созед ва онҳоро нашр кунед.*
- *Формаи савол барои марраҷ (booking enquiry form) ба саҳифаи Иртибот илова кунед.*

---

### Site Builder (Сохтани сайт)

**Мақсад:** Сохтани вебсайт аз як фармоиш.

**Инструментҳо:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`

**Чӣ чӣ хуб мекунад:**
- Навиштани нақшаи сохтани сайт аз бисёр марҳила барои тиҷорати тавсифишуда
- Иҷрои ҳар як марҳила аз ҷониби худ — сохтор, мундариҷа, навигация, дизайн
- Ҳангоми корро дар марҳилаҳо аз хатоҳо баргардошта ва бе сарчашмаи инсонӣ
- Насобати плагиниҳои тавсияшуда ҳамчун қисми сохтани сайт

**Тавсияҳои оғозӣ:**
- *Як сайти портфолиои гӯштаро бо навъи post turi Gallery, саҳифаи марраҷ ва формаи иртибот созед.*
- *Сайти ресторонро бо менюи онлайн, соати кушода ва формаи савол барои брон кардани дарсҳо (table-booking) созед.*
- *Сайти машваратҳои мустақилро бо саҳифаҳои хизматрасониҳо, қисми портфолио ва як варситот созед.*
- *Барои иртибот ба саҳифаи Иртибот аз Site Builder истифода баред.*

---

### Design Studio (Студии дизайн)

**Мақсад:** Тағйири назаррасоӣ — ранги, типографика, CSS ва намунаҳои блоК.

**Инструментҳое ки дар мавҷудонд:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Чӣ гуна онҳо хуб кор мекунанд:**
- Нақшаҳои мавзӯи (theme presets) бо номи муайянро мегиранд (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog).
- Типografiя ва палитраи ранги глобалӣ тавассути theme.json хусусият дода мешаванд.
- CSS махсуси брендиро барои ивазҳо инъобат мекунанд (override).
- Скриншот аз саҳифа гирифта ва онро барои мушкилоти тарҳӣ баррасӣ кардан.

**Тавсияҳои оғоз:**
- *Нақшаи warm-editorial-ро мегиранд ва баъд ранги асосӣ (primary colour) -ро ба #2d6a4f тағйир диҳед.*
- *Скриншот аз саҳифаи асосӣ гиред ва ба ман гӯед, ки шумо чӣ гуна онро беҳтар мекунед.*
- *Нақшаи hero мувофиқ бо фони пурра (full-width background image) ва сарсавдои марказӣ (centred heading) созед.*

---

### Plugin Manager (Дираи плагинҳо)

**Мақсад:** Кашф, инсталл ва идоракунии плагинҳои WordPress.

**Инструментҳои мавҷуда:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Чӣ гуна онҳо хуб кор мекунанд:**
- Плагини беҳтаринро барои ҳолати ибораташуда тавсия медиҳанд.
- Блокҳои (ability packs) инсталл аз реестр мегиранд.
- Каталогии мувофиқӣ дар як категорияҳо ҷудо карда мешаванд.

**Тавсияҳои оғоз:**
- *Чӣ беҳтарин плагин барои дароҷаи аъзои (membership directory) аст?*
- *Блокҳои WooCommerce abilities pack-ро инсталл кунед.*
- *Ҳамаи блокҳои мувофиқи иқтисодият (ecommerce ability packs) -ро намӣ кунед.*

---

### Support Assistant (Ёрдараи дастгирӣ)

**Мақсад:**Ҷавоб додан ба саволҳо дар бораи мундариҷаи сатҳ, танзимҳо ва тартиби WordPress.

**Инструментҳои мавҷуда:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Чӣ чӣ хуб мекунад:**
- Пайваст кардани танзимот ва имкониятҳои ҳолати мавҷруд сайт.
- Тавзеҳ додани он ки кадом навъҳои пост (post types), таксиномаҳо (taxonomies) ва менюҳо дар сайт тартиб ёфтаанд.
- Ба ҷавоби саволҳои "ин танзим чӣ кор мекунад?" бо хондаи арзишҳои рӯйхати ҳаётӣ.
- Дар кадом ҳолат ҳамчун қабаи гузаришҳо, як қадамаи диагностик (diagnostic layer) бахшидан.

**Паршниҳо барои оғоз:**
- *Кадом плагиниҳо ва танзимҳо дар ин сайт фаъол мебошанд?*
- *Ҳамаи навъҳои постҳои хосиятӣ (custom post types) ки дар ин сайт қайд шудаанд, ройхат кунед.*
- *Кадом менюҳои навигация вуҷуд доранд ва онҳо кадом ҷойгир карда шудаанд?*

## Ташкил ва тағйир додани агентаҳо (Customising Agents)

Ҳар як агентҳои воридшуда метавонаи аз ҳисоби филтр `gratis_ai_agent_agents` паст карда ё ҷойгир карда шаванд.

### Додани системаи тавсияи хосиятӣ ба агенти мавҷуда

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nҲамеш бо забони англиси британӣ навишта ва аз икораи Оксфорд истифода кунед.";
    }
    return $agents;
} );
```

### Қайд кардани агент нав

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Постҳо ва саҳифаҳоро барои коркарди маҳсулот (search engines) оптимизат мекунад.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'Шумо SEO Specialist-и бузукӣ ҳастед. Ба оптимизатсияи калидҳо, тавсифҳои мета ва маҷмӯаи маълумот (structured data) тамаркуз кунед.',
        'suggestions'   => [
            'Сарлавҳа ва тавсифи мета-тасвири саҳифаро барои пангашт ҳамаи 5 пост аз нав баҳо دهید.',
            'Барои пяти постҳои охирин, тавсия додани беҳтар кардани тагги (title tag) намудад.'
        ],
    ];
    return $agents;
} );
```

Ночуви агент дар ҷои фароҷи Agent Picker дар лаҳзаи фаъолият фильтр баъд пайдо мешавад.

### Ҳаттоки агенти сохташударо аз рӯй овардан

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
