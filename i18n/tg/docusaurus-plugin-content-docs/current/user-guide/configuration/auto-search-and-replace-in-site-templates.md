# Автоматический поиск и замена в шаблонах сайта (v2) {#auto-search-and-replace-in-site-templates-v2}

_**Этот туториал требует WP UItimo версии 2.x.**_

Одна из самых мощных функций Ultimate Multisite — это возможность добавлять произвольный текст, цвет и поля для выбора в форму регистрации. Как только мы захватим эти данные, сможем использовать их для предварительного заполнения контента в определенных частях выбранного шаблона сайта. Затем, когда новый сайт будет опубликован, Ultimate Multisite заменит плейсхолдеры на фактическую информацию, введенную во время регистрации.

Например, вы можете создавать свои шаблонные сайты с использованием плейсхолдеров. Плейсхолдеры нужно оборачивать двойными фигурными скобками — `{{placeholder_name}}`.

Затем вы можете просто добавить соответствующее поле регистрации для захвата этих данных.

Ваш клиент сможет заполнить это поле во время регистрации.

Ultimate Multisite затем автоматически заменит плейсхолдеры данными, предоставленными клиентом.

## **Решение проблемы "шаблон полный плейсхолдеров"** {#solving-the-template-full-of-placeholders-problem}

Все это здорово, но мы сталкиваемся с неприятной проблемой: теперь наши шаблоны сайтов — которые могут посещать наши клиенты — заполнены уродливыми плейсхолдерами, которые мало что рассказывают.

Чтобы решить эту проблему, мы предлагаем опцию установки фиктивных значений для плейсхолдеров и используем эти значения для поиска и замены их содержимого на шаблонных сайтах во время посещения вашими клиентами.

Вы можете получить доступ к редактору плейсхолдеров, перейдя по пути **Ultimate Multisite > Settings > Sites**, прокрутив до области Site Template Options (Параметры шаблона сайта) и затем нажав на ссылку **Edit Placeholders** (Редактировать плейсхолдеры).

![Область Site Template Options в настройках Sites](/img/config/settings-sites-templates-section.png)

Это переведет вас в редактор содержимого плейсхолдеров, где вы сможете добавлять плейсхолдеры и их соответствующее содержимое.

![Навзираҳои шаблонҳо редактор](https://example.com/img/config/settings-sites-templates-section.png)
