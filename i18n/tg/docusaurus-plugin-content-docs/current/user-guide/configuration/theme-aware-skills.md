# Маҳорати, мувофиқ ба тема {#theme-aware-skills}

Superdav AI Agent v1.10.0 чор маҳорати нав ва махсусиро пешниҳод мекунад, ки бо таври автоматик ба темуи WordPress-и фаъол шумо мутобиқ мешаванд. Ин маҳоратҳо роҳнамои ва қобилиятҳои махсусро таъмин мекунанд, ки ба сохтори ва хусусиятҳои темаи шумо мувофиқ аст.

## Маҳорати мувофиқ ба тема чист? {#what-are-theme-aware-skills}

Маҳорати мувофиқ ба тема онҳо базаҳои донишӣ ва маҷмӯаҳои воситаҳои пешгӯӣ мебошанд, ки бо таври автоматик аз рӯи тему WordPress-и фаъол дар сайти шумо интихоб карда мешаванд. Вақте шумо темуро тағйир медиҳед, маҳорати мавҷудаи ассистент мувофиқ мешавад — ҳатто танзими рухрии махсус талаб карда намешавад.

Ҳар як маҳорат шаомӯи инҳо дорад:

- **Документатсияи махсуси тема** — роҳнамоӣ дар бораи истифода ва таҳияи мувофиқшудаи тема
- **Эстеме блок ва патернҳо (Block and pattern references)** — блоклари мавҷуда, патернҳо ва танзимотҳои тарҳ
- **Намунаҳои таҳия** — қисмҳои кодӣ ва намунаҳои танзимоти барои vazifаҳои маъмул
- **Азбаҳои беҳтарин (Best practices)** — тавсияҳо барои сохтори ва раванди кори тема

## Маҳоратҳои мувофиқ ба тема мавҷуда {#available-theme-aware-skills}

### Темаҳои блок (Block Themes) {#block-themes}

**Барои кадом темаҳо мебошад:** Темаҳое, ки аз сохтори блоклавии WordPress (Full Site Editing) истифода мебаранд.

Маҳорати Block Themes роҳнамоӣ медиҳад дар бораи:

- Таҳия ва тағйири шаблонҳо бо истифода аз редактори блок
- Коркард бо патернҳои блок ва блоклари мувофиқ (reusable blocks)
- Танзими устуворҳо (global styles) ва танзимотҳои `theme.json`
- Истифодаи блоклари тема ва вариационҳо
- Сохтани патернҳои блокӣ барои сайти худ

**Бо таври автоматик фаъол мешавад, вақте:** Темаи фаъоли шумо тему блокрост (шамперо `block-templates` дастгирӣ мекунад).

### Темаҳои классик (Classic Themes) {#classic-themes}

**Барои кадом темаҳо мебошад:** Темаҳои анъанавии WordPress, ки аз PHP templates ва редактори классик истифода мебаранд.

Маҳорати Classic Themes роҳнамои инҳоро медиҳад:

* Касб с файлоҳои PHP template ва hookҳо
* Тағйир додани намуди тема тавассути Customizer
* Истифодаи масоҳати widget ва sidebarҳо
* Тағйири CSS ва рушди child theme
* Фаҳмидани иерархияи тема ва шаблонҳои (template tags)

**Автоматик фаъол мешавад вақте:** Темаи фаъоли шумо классикӣ (non-block) бошад.

### Kadence Blocks {#kadence-blocks}

**Барои кадом сайтҳо мебошад:** Сайтиҳое, ки аз плагини Kadence Blocks барои тарҳре ба даст овардашудаи блокҳои мураккаб истифода мекунанд.

Маҳорати Kadence Blocks ройхати масоҳати пешрафтаи Kadence-ро фароҳ меконад:

* Истифодаи ройхати пешрафтаи Kadence (Hero, Testimonials, Pricing ва ғайра)
* Ташкили калитуби блокҳои Kadence ва танзимоти Responsive
* Сохтани тарҳҳои махсус бо истифода аз масоҳати grid ва container-ҳои Kadence
* Интегратсияи блокҳои Kadence бо темаи шумо
* Истифодаи системаи дизайн ва presetҳои Kadence

**Автоматик фаъол мешавад вақте:** Плагини Kadence Blocks дар сайти шумо фаъол باشد.

### Kadence Theme {#kadence-theme}

**Барои кадом сайтҳо мебошад:** Сайтиҳое, ки аз темаи Kadence барои тарҳре ба даст овардашудаи блокҳо ва тағйир додани он истифода мекунанд.

Маҳорати Kadence Theme ройхати ин корро фароҳ меконад:

* Тағйири темуи Kadence тавассути глобал styles ва theme.json
* Истифодаи намунаҳои (patterns) ва шаблонҳои (templates) блокҳои дар дохили Kadence
* Ташкили танзимоти ва опсияҳои темаи Kadence
* Сохтани тарҳҳои махсус бо истифода аз системаи дизайн-и Kadence
* Интегратсияи Kadence бо плагиниҳо ва воситаҳои маъмул

**Автоматик фаъол мешавад вақте:** Темаи Kadence темаи фаъоли шумо باشد.

## Чӣ тавр маҳоратҳо интихоб карда мешаванд {#how-skills-are-selected}

Assistent (ё касби ёрӣ) ҳар як паём ба темуи фаъоли ва плагиниҳои иловашудаи шуморо автоматик муайян мекунад. Агар маҳорати мувофиқ бо тема мавҷуд бошад, он автоматик ба контексти Assistent ворид мешавад. Вам лозим нест, ки маҳоратҳоро худатон фаъол кунед ё тағйир диҳед.

### Маҳоратҳои бисёр (Multiple Skills) {#multiple-skills}

Агар баъзе маҳоратҳо ба сайт равона бошанд (масалан, агар шумо ҳам Kadence Blocks ва Kadence Theme-ро фаъол дошта бошед), касби ёрӣ ҳамаи маҳоратҳои мувофиқро дорад ва метавонад аз ҳар яке аз онҳо роҳнамоии онҳоро ишора кунад.

### Тағйир додани маҳорат (Switching Themes) {#switching-themes}

Вақте ки шумо маҳорати фаъоли худро тағйир медиҳед, маҳоратҳои мавҷудаи ёрӣ дар паёмҳои минбаъда аз ҷониби ассистент автоматик омода мешаванд. Масалан:

1. Шумо аз маҳорати **Block Themes** (Маҳорати Маҷмӯаи Блокҳо) бо маҳорат фаъол дошта сурат мегиред.
2. Шумо ба маҳорати классикӣ тағйир медиҳед.
3. Дар паёми дигари худ, маҳорати **Classic Themes** (Маҳорати Маҳорати Классикӣ) автоматик омода карда мешавад ва маҳорати **Block Themes** аз мавҷудияти он ба кор мегирад.

## Истифодаи маҳоратҳои ҳассоси маҳорат (Using Theme-Aware Skills) {#using-theme-aware-skills}

Барои истифодаи маҳорати ҳассоси маҳорат, инсоф танҳо дар интерфейси чат тавсиф кунед, ки шумо мехоҳед чӣ корро анҷом диҳед. Ассистент автоматик роҳнамоии маҳорати мувофиқро ишора мекунад.

### Мисолҳои пурсиш (Example Prompts) {#example-prompts}

**Барои Block Themes:**
> "Секции геройро бо тасвири фони ва матни марказӣ бо истифода аз намунаҳои блокҳо созед."

**Барои Classic Themes:**
> "Бо истифода аз child theme, як мисоли махсуси зонаи виджетро ба болои паҳн (sidebar) илова кунед."

**Барои Kadence Blocks:**
> "Секции тасвирҳои мубоддатро бо истифода аз блок Testimonials-и Kadence созед."

**Барои Kadence Theme:**
> "Тавозуни тарҳрезӣ ва стили менюи навигатро тағйир диҳед."

Ассистент роҳнамоии махсуси маҳорат ва мисолҳои кодро пешниҳод мекунад, ки ба маҳорати фаъоли ва плагинотҳои шумо мувофиқ аст.

:::note
Маҳоратҳои ҳассоси маҳорат дар Superdav AI Agent v1.10.0 ва нокиш аз он қаблии он омода мебошанд. Барои он ҳатто танҳо таҷҳизот ё конфигуратсияи иловагӣ лозим нест.
:::
