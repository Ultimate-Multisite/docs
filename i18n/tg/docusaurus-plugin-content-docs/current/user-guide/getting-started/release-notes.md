# Нотаҳои релиз {#release-notes}

## Нӯшаи 2.13.0 — Равшан карда шуд дар 2026-06-05 {#version-2130--released-on-2026-06-05}

- Навъи: Добавлен поддерживаемый суверенный арендатор для потоков учетной записи клиента, оформления заказа, выставления счетов, сайта, шаблонов и сопоставления доменов, чтобы сети арендаторов могли направлять клиентов обратно на основной сайт для управляемых действий.
- Навъи: Добавлены проверки учетных данных при продлении подписок, чтобы шлюзы отключали автопродление, если отсутствует сохраненное соглашение о выставлении счетов, подписка или токен хранилища (vault token).
- Навъи: Добалена публикация с проверкой HMAC для ожидаемого создания сайта, чтобы процесс предоставления ресурсов на хостах, где задержки в фоновых заданиях возможны, стал более надежным.
- Навъи: Добавлены точки расширения для URL SSO, базовых доменов форм оформления заказа и автоматического создания записей домена.
- Исправление: SSO теперь работает надежнее между сопоставленными доменами, анонимными посещениями брокеров, выходами из системы и конфликтами зависимостей плагинов.
- Исправление: Теперь ожидаемое создание сайта восстанавливается после устаревших флагов публикации и предотвращает застревание клиентов на экране создания сайта.
- Исправление: Записи домена больше не создаются для общих базовых доменов форм оформления заказа, а неиспользуемые фоновые задания провайдера хостинга пропускаются, если нет активной интеграции.
- Исправление: Краевые случаи для оформления заказа, адреса выставления счетов, сброса пароля, проверки электронной почты, переключения шаблонов, туров и панели клиента больше не блокируют обычные потоки клиентов.
- Исправление: Рассылка электронных писем теперь сохраняет конфиденциальность получателей, избегая при этом фатальных ошибок SMTP/плагинов при сбоях списков получателей или транспорта почты.
- Исправление: Краевые случаи для продления членства, отображения сроков действия и сбора платежей теперь предотвращают немедленное истечение срока действия, сбои или пропуск обязательных платежей.
- Улучшено: Совместимость с WordPress протестирована до версии 7.0, производственные ассеты Vue пересобираются из источников npm, а покрытие Cypress end-to-end теперь включает больше потоков оформления заказа, настройки SSO и шлюзов.

## Нውсияи 2.12.0 — Барои коркарда дар 2026-05-15 {#version-2120--released-on-2026-05-15}

- Нав: Hostinger (hPanel) ҳамчун пешниҳоди хости дастги кард шуд ва пайваст кардани доменҳо низ ишора мешавад

- Намудачу: Имои почтаҳои BCC ин ҳол аз истифодаи занҷираи мактубҳое, ки нашоҳида намешавад, ба таври нофаҳмӣ барои пешгирии нишон додани манзили гирифтанчур
- Намудачу: Санаи паёмбарӣ (membership expiration date) ҳангори савдо кардани арзиши бесанаат, аз коррупсия намешавад

- Намудачу: Рексаи SSO дар сайтиҳое, ки домени онҳо тағйир дода шудааст, ҳоло барои пешгирии даврҳои беосори редирект (redirect loops) маҳдуд карда мешавад
- Намудачу: Интихоби тасвир аз Wizard барои гузариш ба модели маълумот дар пазираи асли дуруст нав карда мешавад
- Намудачу: CLI-и Site Exporter ин танзими пешбудиар (default) дурусти иртибозии сайт шабакаро сақлашро ҳифз мекунад
- Беҳтар шуд: wp-cli-и банд аз пакет плагин ба кор дароварда шуд, ки ҳаҷми плагинро кам кардааст

- Исправление: Опции пустой идентификации сайта сохраняются при сохранении.
