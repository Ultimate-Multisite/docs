# Жолиби миграцияи ҳуҷҷатҳо {#sovereign-migration-workflow}

Ultimate Multisite: Multi-Tenancy 1.2.0 барои гузариш аз як подмони стандарт ба инфрасорори лойхатҳои суверени (sovereign tenant) варори тасдиқи миграцияро мегирад.

## Пеш аз оғоз {#before-you-start}

Тасдиқ кунед, ки шумо:

- Як бэкапи ҳолати мавҷудаи сайтро доред.
- Addon Multi-Tenancy фаъол аст.
- База данныхи лойхати мақсадӣ, корзиши файлони асосии он ва домени омода ҳастанд.
- Пайварии хости база данных (database host binding) доред, ки ба муҳити мақсадӣ мувофиқ аст.
- Дастрасӣ доред барои иҷрои командаҳои WP-CLI барои шабака.

## Интеграцияи пайваста (Recommended workflow) {#recommended-workflow}

1. База данных ва файлони лойхати мақсадӣ омодасозии онро анҷом диҳед.
2. Настроёни муҳити алоҳидаи лойхат (tenant isolation settings)-ро қайд кунед ё навсоз кунед.
3. Миграцияи лойхат-ро иҷро кунед.
4. Касбор ё тасдиқи корбарони лойхатҳоро анҷом диҳед.
5. Корнишҳои мигратсияи асинх (async migration jobs)-ро ба итмом расонед.
6. Тасдиқи миграцияи суверени-ро иҷро кунед.
7. Ба лойхат бо SSO раведдан кунед.
8. Фақат пас аз тасдиқи муваффақият, DNS ё маршрутиро тағйир диҳед.

## Постҳои тасдиқ (Verification gates) {#verification-gates}

Ин пайвастаи тасдиқ миграцияро аз якчанд ҷанба мебарасонад:

- Мавҷудияти схемаи лойхат ва имконияти навиштани он аз ҷониби навиписандаи база данныхи лойхат.
- Додани хости база данных (database host) мувофиқ бо иҷозатҳо (grants).
- Мавҷудияти корбарони лойхат ва мувофиқати он бо шумораи инсталлушн корбарони суверени назаррасошуда.
- Имконияти ба итмом расонидани қатои корҳои асинх (async push queue).
- Накишҳои давлатии собиаи шабака (legacy network-side data paths) барои лойхати суверени аллакай зарур нест.

Ҳар як намуди номуваффақиятро ҳамчун манъатгари пеш аз оғози кор ҳатфидаред. Мушкилоти база данных, корбар, қатои ё маршрутиро дар ҷои муайян ислоҳ кунед ва пас аз он тасдиқро боз иҷро кунед пеш аз расонидани лойхат ба муштариён.

## Визити аввали истеҳсолии (First production visit) {#first-production-visit}

Баъди расонидани тасдиқ, барои визити администратори аввалия, аз экрани идоракунии сайт **Visit (SSO)**-ро истифода баред. Ин маршрути лойхат, дараҷаи нигоҳдории токен SSO, ҷойгир кардани мағзи манбаъ (origin pinning) ва таъмини корбар аз ҷониби самти лойхат дар як қадами назоратшаванда тасдиқ мекунад.
