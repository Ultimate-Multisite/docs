# Эътимобилеи Назорати Дарёшӣ (Site Builder Mode) - Эътибор ба Назорат

**Эътимобилеи Site Builder дар Superdav AI Agent v1.12.0 аз навсозӣ шудааст.** Агар шумо аз модаи Site Builder истифода мекардед, бояд барои таҳияи тема ва таъсири сайт ба **Setup Assistant agent** гузаред.

## Чӣ Меъни Эҷод Шуда? {#what-happened}

### Модаи Site Builder (Қадим) {#site-builder-mode-legacy}

Модаи Site Builder як интерфейси бо асоси ройгон (wizard-based) буд барои:

- Эҷоди сайтҳо аз шаблонҳо
- Ташкили калитувҳои асосӣ
- Интихоби тема
- Тарзи таъмини мувофиқ ба онҳо

### Чӣ Мезои Он? {#what-replaced-it}

**Setup Assistant agent** ҳоло ҳамаи функсияҳои Site Builder-ро бо:

- Ташкилоти мувофиқтар ва ройгон аз ҷониби агент
- Имкониятҳои беҳтари таҳияи тема
- Ҳамwerking бо раванди оғоз (onboarding) Theme Builder
- Хотраи доимӣ (`site_brief` memory) барои сессияҳои оянда

## Агар Шумо аз Модаи Site Builder истифода мекардед {#if-you-were-using-site-builder-mode}

### Сайтҳо Азбод Маъқул Нанд {#your-sites-are-safe}

- Сайтҳои мавҷуда, ки бо модаи Site Builder сохта шудаанд, навтарин кор мекунанд
- Ҳеч як маълумот ё тағйири сайт ба вуҷуд намеояд
- Шумо метавонед идома дидани идоракунии сайтҳоро ба таври муқаррарӣ идома диҳед

### Ба Setup Assistant Agent Гузаред {#migrate-to-setup-assistant-agent}

Барои таъсири навсайт ё тағйироти тема, аз Setup Assistant agent истифода баред:

```
"Help me set up a new site"
```

ё

```
"Start the Theme Builder onboarding"
```

Setup Assistant agent ҳамин функсияҳоро бо ройгонӣ бештар пешниҳод мекунад.

## Муқоиса: Site Builder vs. Setup Assistant {#comparison-site-builder-vs-setup-assistant}

| Хусусият | Site Builder (Навсозӣ шуд) | Setup Assistant (Нав) |
|---------|----------------------|----------------------|
| Усули таъсир | Формаи ройгон (Wizard form) | Муозасоти агент (Agent conversation) |
| Интихоби тема | Шаблонҳои пеш аз муқарраршуда | Таҳияи худкор (Custom generation) |
| Таҳия (Customization) | Имкониятҳои маҳдуд | Системаи тарҳро ба таври пурра |
| Site brief | Наст карда ме шуд | Хотраи доимӣ (Persistent memory) |
| Сессияҳои оянда | Ишфои мунтазам | Истифодаи `site_brief` аз ҷониби худи он |
| Ройгонӣ | Раванди муқарраршуда | Муозасоти мутобиқ (Adaptive conversation) |

## Гузариш ба Setup Assistant Agent {#migrating-to-setup-assistant-agent}

### Барои Сайтҳои Нав {#for-new-sites}

Вместо использования режима Site Builder:

1. Запрос: "Помогите мне настроить новый сайт"
2. Агент Помощник по настройке (Setup Assistant agent) проведет вас через:
   - Цель и задачи сайта
   - Целевая аудитория
   - Идентичность бренда
   - Генерация темы
   - Начальная конфигурация

### Для существующих сайтов {#for-existing-sites}

Если у вас уже есть сайт, созданный в режиме Site Builder:

1. Вы можете продолжать использовать его как есть
2. Чтобы обновить тему, запросите: "Переделать мой сайт"
3. Агент Помощник по настройке поможет вам создать новую тему
4. Данные вашего сайта остаются без изменений

### Для изменений темы {#for-theme-changes}

Вместо выбора темы в режиме Site Builder:

1. Запрос: "Изменить мою тему"
2. Агент Помощник по настройке сделает следующее:
   - Спросит о ваших дизайнерских предпочтениях
   - Сгенерирует кастомную тему
   - Активирует ее на вашем сайте

## Ключевые различия {#key-differences}

### Режим Site Builder {#site-builder-mode}

```
1. Выберите шаблон
2. Выберите тему
3. Настройте базовые параметры
4. Готово
```

### Агент Помощник по настройке (Setup Assistant Agent) {#setup-assistant-agent}

```
1. Опишите цель вашего сайта
2. Определите вашу целевую аудиторию
3. Выберите дизайнерские предпочтения
4. Агент генерирует кастомную тему
5. Агент активирует тему
6. Краткое описание сайта сохраняется для будущих сессий
```

## Преимущества Агента Помощника по настройке {#benefits-of-setup-assistant-agent}

### Больше гибкости {#more-flexible}

- Описывайте ваш сайт на естественном языке
- Получайте индивидуальные рекомендации
- Адаптируйтесь к вашим конкретным потребностям

### Лучшая кастомизация {#better-customization}

- Генерация кастомной темы
- Решения системы дизайна
- Постоянные дизайн-токены

### Постоянная память {#persistent-memory}

- Ваше site_brief сохраняется
- Будущие агенты понимают ваш сайт
- Не нужно повторять информацию о настройке

### Интегрированный рабочий процесс {#integrated-workflow}

- Онбординг Theme Builder
- Навык эстетики Дизайн-системы
- Возможность управления видимостью (Ability Visibility controls)
- Все работает слаженно

## Устранение неполадок {#troubleshooting}

### Я не могу найти режим Site Builder {#i-cant-find-site-builder-mode}

Режим Site Builder удален. Используйте вместо него Агента Помощника по настройке:

"Мароси ман сайт навсозӣ кунам"

### Ман мехоҳам аз Site Builder як сайти нав созеям {#i-want-to-recreate-a-site-from-site-builder}

Шумо метавонед онро бо ассистенти Setup Assistant дуборасозед:

1. Дархост: "Маро ба сохтани саҳифа нав кӯмак кунед"
2. Мақсади саҳифаи қадими худро ва тарҳонаш тавсиф конӣ
3. Ассистент як темаи монанд тавлид мекунад
4. `site_brief` шумо барои истифодаи оянда сақлана мешавад

### Сайти Site Builder-и мавҷудаи ман кор намекунад {#my-existing-site-builder-site-isnt-working}

Сайти сохташуда бо модаи Site Builder байн боқӣ мемонад кор кардан. Агар шумо мушкилот доред:

1. Барои он текшиш кунед, ки темаи шумо ҳамеша фаъол аст
2. Тасдиқ конӣ, ки плагиниҳо иҷозат дода шудаанд
3. Логҳои хатои WordPress-ро текшиш конӣ
4. Агар мушкилот давом накунад, бо дастгирӣ пиёшавӣ кунед

### Ман метавонам аз шаблонҳои қадими Site Builder истифода барам? {#can-i-still-use-my-old-site-builder-templates}

Шаблонҳои Site Builder интихоб карда нашуданд. Аммо:

- Сайти мавҷудаи шумо кор мекунанд
- Шумо метавонед сайти монандро бо ассистенти Setup Assistant дуборасозед
- Ассистент имкониятҳои бештар барои таҳия пешниҳод мекунад

## Қадамҳои навбатии амалӣ {#next-steps}

1. **Барои сайтномаҳои нав**: аз ассистенти Setup Assistant истифода баред
2. **Барои сайти мавҷуда**: онҳоро ба ҳолати онҳо идома диҳед
3. **Барои тағйири тема**: барои кӯмак аз ассистенти Setup Assistant дархост конӣ кунед
4. **Барои такмили тарҳон**: аз қобилияти Design System Aesthetics истифода баред
