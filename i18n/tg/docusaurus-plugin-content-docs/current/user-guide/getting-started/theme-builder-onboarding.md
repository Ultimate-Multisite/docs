# Рушёи Обудӣ барои Theme Builder {#theme-builder-onboarding-flow}

Superdav AI Agent v1.12.0 омодааст, ки **рушёи обудӣ (onboarding flow) Theme Builder**-ро пешниҳод мекунад, ки ба шумо дар интихоби аввалин таъсисолот, як блок-темаи худкорро таҳия кардан ёрӣ мерасонад. Ин модаи қадимии Site Builder-ро бо муносибатҳои мувофиқтар ва ёрӣ аз ҷониби агент иваз мекунад.

## Рушёи Обудӣ Theme Builder чист? {#what-is-the-theme-builder-onboarding-flow}

Рушёи обудӣ Theme Builder як визуалӣ (interactive) wizard аст, ки:

- Шуморо дар қарорҳои тарҳӣ (рангҳо, типографика, тартиби саҳифа) роҳнамоӣ мекунад
- Аз алоҳидаи мақсадҳои визуалӣ (visual identity preferences)и сайти шуморо ба даст меорад
- Як Theme Builder-и худкорро ки барои ниёзҳои шумо мувофиқ аст, таҳия мекунад
- Вақте он пурра шуд, модаи тему-ро автоматикӣ фаъол мекунад

Ин рушё аз ҷониби **Setup Assistant agent** таъмин шудааст, ки саволҳои равшанӣ мепурсад ва теме-ро ба таври пайдарпай (incrementally) сохта мерасонад.

## Оғози Рушёи Обудӣ Theme Builder {#starting-the-theme-builder-onboarding}

### Навъи Интихоб (First-Run Setup) {#first-run-setup}

Вақте шумо аввалин бор Superdav AI Agent-ро дар интишори навтарин WordPress оғоз мекунед, шумо инҳоро мебинед:

```
Бахшаш аз Superdav AI Agent!

Шумо чӣ гузаронидан мехоҳед?
1. Таҳияи теме худкор (Theme Builder)
2. Истифодаи тему мавҷуда
3. Ҳоло барои интихоб саркард
```

Барои ворид шудан ба рушёи обудӣ Theme Builder, **"Build a custom theme"**-ро интихоб кунед.

### Фаъолисозии Рушмӣ (Manual Activation) {#manual-activation}

Шумо метавонед дар ҳар вақт бо дархост:

```
"Start the Theme Builder onboarding"
```

ё

```
"Help me create a custom theme"
```

рушёи обудӣ Theme Builder-ро оғоз кунед.

## Қадамҳои Рушёи Обудӣ {#the-onboarding-steps}

### Қадами 1: Интихоби Навъ (Mode Selection) {#step-1-mode-selection}

Setup Assistant agent дар бораи афзалиёти шумо пурсед:

```
Шумо мехоҳед теме-ро чӣ гуна созед?
- Роҳнанда (Ман саволҳоро мепурсам ва барои шумо онро созам)
- Аз таҷриба (Ман имкониятҳоро нишон медиҳам ва шумо қарор медиҳед)
```

**Навъи роҳнанда (Guided mode)** барои аксаки корکارҳо пайваста тарҳре рекоменду мешавад; агент ба таври асосӣ дар асоси соҳа ва мақсадҳои шумо тавсияҳои тарҳӣ медиҳад.

### Қадами 2: Муайян кардани Саҳифа (Site Specification) {#step-2-site-specification}

Ба шумо дар бораи сайти худ пурсида хоҳанд шуд:

- **Мақсади сайт**: Электрон савдо (E-commerce), блог, портфолио, SaaS ва ҳ.к.
- **Ҳадафи мақсадган аудитория**: Кишилар сифатидан кимлар келиб тушади?
- **Рангҳои бренд**: Асосий ва иккинчи ранги (ёки "ман билишмайман")
- **Устуворлик (Tone)**: Мутахассис, равон, қизиқарли, минималист ва ҳ.к.

Ин маълумотлар сизнинг **site_brief** хотирасига сақланади ва агентлар келаси қаторларда буни ишора қилишади.

### Қадам 3: Дизайн тизими қарорлари (Design System Decisions) {#step-3-design-system-decisions}

Агент сизга дизайн токенлари танлашда йўл кўрсатади:

- **Типографика**: Шрифт оиласи (serif, sans-serif, monospace) ва ўлчамлар шкаласи
- **Ранг палитраси**: Асосий, иккинчи, акцент ва юқори ранги
- **Масофалар (Spacing)**: Қисқа, оддий ёки кенг лойализациялар
- **Ҳаракат (Motion)**: Анимациялар ва ўтиш эффекти (хоҳласангиз)

### Қадам 4: Тема ишлаб чиқариш (Theme Generation) {#step-4-theme-generation}

Setup Assistant агент сизнинг махсус блок темасини қуйидагилар билан тузиб беради:

- Барча дизайн токенларини ўз ичига олган `theme.json`
- Одатдаги лойализациялар учун Блок шаблонлари (хона саҳифаси, блог, контакт)
- Дизайн тизимингизга мос блок услублари
- Тема метамаънолари ва WordPress даярлиги декларациялари

### Қадам 5: Фаоллик ва текширув (Activation and Verification) {#step-5-activation-and-verification}

Тема автоматик фаол қилинади ва сиз қуйидагиларни кўрасиз:

```
✓ Сизнинг махсус темам ҳозираи салоҳиятда!
  Тема номи: [Сайт Номингиз] Темаси
  Рангҳо: [Асосий] / [Иккинчи]
  Типографика: [Шрифт оиласи]

  Дизайни янги кўриш учун сайтга кириб боринг.
```

Кейин сиз теманинг тўғри кўринга олганини текшириш учун сайтга кириб боришингиз мумкин.

## Сайт спецификаси ва site_brief хотираси (Memory) {#site-specification-and-sitebrief-memory}

Кириш пайтида, агент сизнинг сайтинг спецификасини **site_brief** хотираси категориясида қайтади. Бу қуйидагиларни ўз ичига олади:

- Сайт мақсади ва устуворли
- Ҳадафи мақсадган аудитория
- Бренд ранги ва устуворлик (Tone)
- Дизайн афзалликлари
- Мазмун тузилмаси

### Нега site_brief муҳим? {#why-sitebrief-matters}

Келаси қаторларда агентлар сизнинг site_brief хотирасингизни қуйидаги мақсадда ишора қилишади:

### Ҳифзи мувофиқӣ дар тағйирот {#viewing-your-sitebrief}

*   Намунаҳои дизайнро ҳангори тағйирҳо ба таври якхела сабт кунед.
*   Хусусиятҳоеро пешниҳод кунед, ки ба мақсади сайт шумо мувофиқат мекунанд.
*   Тавсияҳоро бо фаҳмиши контекст пешниҳод кунед.
*   Саволҳои гузаштаи таъсир кунандаро такрор накунед.

### Намудани хулосаи сайт (site_brief) {#customizing-after-onboarding}

Шумо метавонед аз агент пурсед:

```
"Show me my site brief"
```

ё

```
"What do you know about my site?"
```

Агент тасвири муайян кардашудаи муайянкунандаи сайт-и шуморо нишон медиҳад.

## Ташкил додан пас аз оғайриат (Onboarding) {#use-the-design-system-aesthetics-skill}

Баъди пурра шудан оғайриат Theme Builder, шумо метавонед:

### Истифодаи малака Зарбукии Дизайни Системӣ (Design System Aesthetics Skill) {#edit-themejson-directly}

Тавсияҳои дизайнро тақозо кунед:

```
"Refine the typography to be more modern"
```

ё

```
"Adjust the color palette to be warmer"
```

Малака **Зарбукии Дизайни Системӣ** шуморо ба тағйиротҳои мушаххас дида месозад.

### Тасвири мустақими theme.json-ро тағйир додан {#create-custom-block-templates}

Барои корбарони пешрафта, барои тағйир додани:

*   Color tokens (пелтаҳои ранги)
*   Typography scales (масштабҳои нусхаи матн)
*   Spacing values (арзишҳои фасолати)
*   Border and shadow definitions (таърифи бортир ва саёҳонҳо)

файл `/wp-content/themes/[theme-name]/theme.json` тағйир додаред.

### Эҷоди шаблонҳои блокии худкор (Custom Block Templates) {#comparison-old-vs-new-onboarding}

Аз муҳаррираи блок WordPress барои эҷоди шаблонҳои худкор барои:

*   Homepage layouts (тағйироти навъи саҳифаи асосӣ)
*   Blog post pages (саҳифаҳои мақолаҳо)
*   Product pages (саҳифаҳои маҳсулот)
*   Contact forms (формаҳои иртибот)

## Муқоиса: Қадри қадим ва нав {#troubleshooting}

| Хусусият | Site Builder (Қадим) | Theme Builder (Нав) |
| :--- | :--- | :--- |
| Усули таҳия | Форма-бази асосӣ | Муозасоти роҳнамои агент |
| Таҳияи тема | Шаблонҳои маҳдуд | Қалоби мувофиқ (scaffolding) |
| Дизайни токенҳо | Кишиш рушмӣ | Гӯёбарии роҳнамоӣ |
| Мувофиқат | Варианти муқаррарӣ | Тағйирёб |
| Тағйирот дар оянда | На ташаккул мекард | Дар site_brief сабт мешавад |

## Ҳалли мушкилот (Troubleshooting) {#next-steps}

**Оғайриат ба пурра наорамад**
*   Оғайриатро дигар боз оғоз кунед: "Start the Theme Builder onboarding"
*   Барои тасдиқ кардан, ки инсталлат WordPress-и шумо мувофиқ аст.
*   Тасдиқ намоед, ки агент Setup Assistant фаъол карда шудааст.

**Мой site_brief не используется**
- Убедитесь, что у агента есть доступ к памяти
- Попросите агента «вспомнить мой site brief» (recall my site brief)
- Проверьте, включена ли память в ваших настройках

**Сгенерированная тема не соответствует моим предпочтениям**
- Используйте навык Design System Aesthetics для доработки
- Попросите агента «сгенерировать тему с [конкретными изменениями]» (regenerate the theme with [specific changes])
- Редактируйте файл theme.json напрямую для точного контроля

## Следующие шаги

После завершения обучения Theme Builder:

1. **Проверьте свой сайт**: Зайдите на свой сайт, чтобы увидеть новую тему
2. **Уточните дизайн**: Используйте навык Design System Aesthetics для внесения правок
3. **Добавьте контент**: Начните создавать контент для вашего сайта
4. **Изучите возможности**: Узнайте о других возможностях агента, таких как scaffold-block-theme и activate-theme
