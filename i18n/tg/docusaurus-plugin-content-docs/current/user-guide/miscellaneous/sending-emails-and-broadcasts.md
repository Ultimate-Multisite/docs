# Арسالи Эймол ва Броадкастҳо (v2)

_**НОТА МУҲИМ: Ин мақола ба Ultimate Multisite версияи 2.x меруудид.**_

Ultimate Multisite ҳамчун хусусияти дорад, ки имкон медиҳад шумо метавонед бо фиристодани поймандираи имодашуда ба корбар ё гурӯҳи корбарон эймол бисозед ва инчунин нотироти дар дашбори администратори онҳо барои паҳн кардани эълонҳо фиристонед.

## Памнокҳои администраторӣ (admin notices) - Роҳбарии корбарро бо Броадкастҳо таҳия кунед

Бо истифода аз хусусияти broadcast-и Ultimate Multisite, шумо метавонед **номкоти администраторӣ** (admin notices) -ро ба дашбори администратори подсайт корбарони худ илова кунед.

Ин хеле муфид аст агар шумо лозим бошед инкорҳоеро сурат намонед, мисли таъмини хизматрасонии система ё пешниҳоди маҳсулот ё хизматрасонии нав ба корбарони мавҷудаи худ. Ин шу тавр аст ки нотироти администраторӣ дар дашбори корбарони шумо намояд.

<!-- Screenshot unavailable: Broadcast admin notice shown on a customer's subsite dashboard -->

Барои оғози инкор администраторӣ, ба дашбори администратори шабакаи худ равед ва дар менюи **Ultimate Multisite**, шумо опсияи **Broadcasts**-ро мебинед.

![Broadcasts list page in Ultimate Multisite admin](/img/admin/broadcasts-list.png)

Шумо инкорҳои мавҷударо низ метавонед тағйир диҳед:

![Broadcast edit interface](/img/admin/broadcast-edit.png)

Ин саҳифаи дастрасӣ, бахши **Add Broadcast**-ро дар боло расонед.

Ин фурӯзи ҷониби Add broadcast modal window-ро оварда мерасонад, ки дар он метавонед интихоб кунед ки шумо кадом намуди broadcast-ро мехоҳед фиристонед.

Барои ин, **Message**-ро интихоб кунед ва ба тугмаи **Next Step** расонед.

![Add broadcast modal with the Message option selected](/img/admin/broadcast-add-message.png)

Саҳифаи дигари аз шумо хоҳиш мекунад ки либоси **Target customer** (Корбарони мақсадӣ) ё **Target product** (Маҳсулоти мақсадӣ)-ро интихоб кунед. Эътибор доред, ки шумо метавонед бештар аз як корбар ё бештар аз як маҳсулот-ро интихоб намоед.

Барои талосни ё кормандиё ё маҳсулот ҷудо кардан, шумо бояд калимаро дар ҳар як матнгу (field) нависед.

Дар сохтори **Message type** метавонед ранги огоҳӣ (notice) -ро интихоб кунед. Ин ба аҳамияти паст аз фиристодани шумо мерасонад.

Баъд метавонед **Next Step**-ро расоне.

![Target customers, target product and message type fields for a Message broadcast](/img/admin/broadcast-message-targets.png)

Шумо метавонед дар ҷониби иҷрои матн (subject) ва мундариҷаи матни (content/message) ки мехоҳед ба кормандони фиристодан пешниҳод кунед, бо ворид кардани мавзӯъ ва мундариҷаи он оғоз кунед.

![Broadcast message subject and content editor on the compose step](/img/admin/broadcast-edit.png)

Баъди сохтани матн, шумо метавонед ба тугмаи **Send** фиристодад.

Ва ин ҳаст. Огоҳии администратор дарҳол дар дашбор (dashboard) кормандони шумо нишон дода мешавад.

## Пайвасткунии почта ба муштариёни худро

Бо истифода аз хусусияти broadcast-и Ultimate Multisite, шумо метавонед пойгоҳи мавзӯъро ба кормандонтанаш фиристоданро бисозед. Шумо имконият доред ки почтаро танҳо ба кормандони муайян ё гурӯҳи муайяни кормандон асос назар ба маҳсулот ё сарият (plan) ки онҳо дар он тартиб карда шудаанд, нишон диҳед.

Барои оғози фиристодани почта, ба дашбори администратори шабакаи худ равед ва дар менюи Ultimate Multisite, шумо имкония **Broadcast**-ро мебинед.

![Broadcasts list page used as the starting point for an email broadcast](/img/admin/broadcasts-list.png)

Ин аз ин саҳифа, тугмаи **Add broadcast**-ро дар боло расоне.

Ин модале (modal window)-и Add broadcast ошкор мешавад, ки дар он метавонед интихоб кунед ки кадом намуди фиристодани матнро мехоҳед фиристонед. Барои ин иҷро кунед ва **Email**-ро интихоб кунед, баъди ин тугмаи **Next Step**-ро расоне.

![Add broadcast modal with the Email option selected](/img/admin/broadcast-add-email.png)

Модалеи нав аз шумо дар бораи **Target customer** ё **Target product** пурсид мешавад. Эътибор диҳед, ки шумо метавонед бештар аз як кормандон ё бештар аз як маҳсулотро интихоб кунед.

Барои талошу дар бораи ҳисоби корбар ё маҳсулот, шумо бояд калимаро дар дохили майдон оғоз карда нависед.

Вақте ки шумо аудиторияи ҳадафи худро интихоб кардед, метавонед **Next Step**-ро расонид.

![Интихоби муштариён ва маҳсулоти ҳадафӣ барои паёми электронӣ](/img/admin/broadcast-email-targets.png)

Маҷмӯаи нав ин ҷо место аст, ки шумо метавонед таҳияи пойгоҳи имоилии худ бо навиштани мавзӯъ ва мундаро (мундаро/мазмун) ки мехоҳед ба корбарон фиристонед оғоз кунед.

<!-- Screenshot unavailable: Ҳуҷҷатгузори мавзӯъ ва мундарои паёми электронӣ дар қадами таҳия -->

Баъди сохтани паём, шумо метавонед ба тугмаи **Send** фиристод.

Ва ин чӣ қадар осон аст, ки паёмро ба корбарони худ бо истифода аз хусусияти broadcast (паёми массовӣ) фиристонед.

## Паёми системавӣ (System emails)

Паёми системавӣ дар Ultimate Multisite онҳоянд **эъломотҳои автоматики** ки система пас аз амалҳои муайян, ба монанди қайдши корбар, пардохт, маппи ошиқона ва ғайра фиристод. Ин паёмҳо метавонанд аз танзимоти Ultimate Multisite тағйир ё тартиб дода шаванд. Он инчунин ҳамчун хусусияти муайян меояд, ки иҷрои танзимотҳои мавҷудаи дигар Ultimate Multisite-ро ба даст оварӣ ва импорт кардани онҳоро иҷро мекунад.

### Ҳадасоти кор аз навсозӣ ва импорт (Resetting & Importing)

Версияҳои нави Ultimate Multisite, инчуна ҳамчун add-ons, метавонанд ва хоҳанд ки одатан паёми электронӣ навиро ба таври муайян фиристонанд.

Барои пешгирии ташдидҳо ва дигар масъалаҳо, **момем мавзӯъҳои нав이ро ҳамчун Паёми Системавӣ дар инсталлии шумо автоматикӣ илова намекунем**, агар онҳо ба коркарди дуруст функсияи муайян ҳатмӣ набошанд.

Аммо, супер админҳо ва агентаҳо метавонанд ин паёме нав сохташударо тавассути импортёр (importer tool) импорт кунанд. Ин раванд як паёми системавӣ бо мундаро ва танзимоти мавзӯи нави месозад, ки ба супер админ имкон медиҳад онҳо ҳама гуна тағйиротро бигзоранд ё онҳоро чуно онҳо сар карда шаванд.

#### Чӣ тавр паёми системавӣ импорт кунем (How to import system emails)

Барои расонидаи Ultimate Multisite Settings баргарӣ равед ва ба табби **Emails** равед.

![Emails tab in Ultimate Multisite settings showing the System Emails section](/img/config/settings-emails-tab.png)

Сар карда, дар паҳнба (sidebar), ба дафтори **Customize System Emails** фиристоред.

<!-- Screenshot unavailable: Customize System Emails button on the System Emails sidebar panel -->

Дар саҳифаи System Emails, шумо дафтори **Reset & Import**-ро дар боло мебинед. Барои он фиристодан модале (modal window) барои импорт ва расонидани мувофиқ аст.

![Reset or Import action button on the System Emails admin page](/img/admin/system-emails-reset-import.png)

Баъд, шумо метавонед танзимоти Import Emails-ро ба таври иҷозат диҳед ва бинед ки кадом email-ҳои система барои импорт мавҷуданд.

<!-- Screenshot unavailable: Reset and Import modal with the Import Emails options expanded -->

#### Расонидани System Emails

Баъзан шумо мефаҳсонед, ки тағйироти худро ба як шаблон emailӣ барои шумо кор намекунанд ва хоҳед онро ба **holati воқеи** (default state) он аз нав расонед.

Дар чунин ҳолатҳо, шумо ду танзим доред: шумо метавонед содда email-и системаро ифрот диҳед ва онро бо истифода аз дастурҳо дар боло импорт кунед (ки ин маълумотҳои фиристодан ва дигаронро мебарорад), ки ин усул камтарин аҳамият дорад.

Ё шумо метавонед **Reset & Import tool**-ро барои расонидани он шаблон email-и истифода баред.

Барои расонидани як шаблон email, шумо метавонед қадамҳои болоро иҷро кунед то ба Reset & Import tool расонед ва пас, танзимоти **Reset**-ро фаъол карда, email-ҳоеро интихоб намоед ки мехоҳед онҳоро ба мундариҷаи воқеи худ аз нав расонед.
