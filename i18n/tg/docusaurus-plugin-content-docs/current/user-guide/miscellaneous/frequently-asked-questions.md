# Общие вопросы

Вот несколько **Часто задаваемых вопросов**, которые часто возникают при запросах в поддержку.

  * **Что такое Ultimate Multisite?**

Ultimate Multisite — это плагин для WordPress, созданный для установок WordPress multisite. Он превращает вашу установку WordPress в премиальную сеть сайтов, похожую на [WordPress.com](https://WordPress.com), позволяя клиентам создавать сайты по ежемесячной, квартальной или годовой подписке (вы также можете создавать бесплатные тарифы).

  * **Будет ли Ultimate Multisite работать без Multisite?**

Нет. Ultimate Multisite требует наличия WordPress multisite, подкаталога или домена уровня поддомена.

  * **Какие требования к Ultimate Multisite?**

Ultimate Multisite требует:

**Версия WordPress** : v5.3+ (Рекомендуется: Последняя стабильная версия)
**Версия PHP** : 7.4.x (Поддержка версий > 8.0 скоро появится)
**Версия MySQL** : v5+ (Рекомендуется: Версия 5.6, если у вашего хостинг-провайдера нет версии 8.0)

_Также необходимо активировать Multisite (поддомен или подкаталог)._ [Как установить WordPress Multisite](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **Будет ли Ultimate Multisite работать с Multisite с подкаталогами?**

Да. Ultimate Multisite поддерживает как установки в подкаталогах, так и установки в подкаталогах.

  * **Будет ли Ultimate Multisite работать с Multisite с поддоменами?**

Да. Ultimate Multisite поддерживает как установки в подкаталогах, так и установки на поддоменах.

  * **Могу ли я использовать Ultimate Multisite для ограничения того, какие плагины и темы могут использовать пользователи?**

Да. С помощью Ultimate Multisite вы можете выбрать, какие плагины и темы доступны каждому тарифу в вашей сети.

  * **Могу ли я использовать Ultimate Multisite для ограничения количества записей, которые пользователь может создать на своем сайте?**

Да, с Ultimate Multisite вы можете установить квоты постов для **какого угодно типа поста** на всей вашей сети. Вы также можете ограничить количество пользователей, которых администратор может пригласить, **объем дискового пространства** и многое другое.

  * **Поддерживает ли Ultimate Multisite Шаблоны сайтов (Site Templates)?**

Да. С Ultimate Multisite вы можете создавать столько Шаблонов сайтов, сколько вам нужно. Шаблоны сайтов — это не темы, это обычные сайты в вашей сети. Вы можете активировать плагины, темы и настраивать этот Шаблон сайта любым удобным способом. Если пользователь выбирает этот шаблон, его сайт будет создан как копия этого шаблона со всеми настройками.

Вы также можете установить определенный шаблон для конкретного тарифа (плана). С помощью этого каждый пользователь, который подпишется на этот план, получит свой сайт в виде дубликата этого Шаблона сайта.

  * **Интегрируется ли Ultimate Multisite с WooCommerce?**

Да!

Если вы хотите, чтобы пользователи вашей сети имели доступ к WooCommerce для создания магазинов, это возможно с помощью Ultimate Multisite. Вы даже можете ограничить количество продуктов, которые ваши клиенты смогут создавать на каждом уровне тарифа (плана).

Что касается подписок для Ultimate Multisite, вы можете использовать [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/), чтобы разрешить вашим клиентам Ultimate Multisite платить с помощью платежных шлюзов WooCommerce на вашем основном сайте. Таким образом, вы сможете воспользоваться огромной библиотекой платежных шлюзов, разработанных для WooCommerce, и использовать их для выставления счетов вашим клиентам Ultimate Multisite.

  * **Есть ли способ мигрировать подписки с WPMUDEV Pro Sites?**

Ultimate Multisite интихобёрӣ ҳоло ба шумо [инструменти мусоидаткунанда барои мигратсия] (https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) дар шабака Pro Sites-и шумо пешниҳод меконад.

* **Чаро табби «Allowed Plugins» дар саҳифаи тасдики Плани ман хусусият кам аст?**

Ultimate Multisite танҳо иҷозат дорад, ки дастрасӣ ба плагиналҳое пешниҳод карда шаванд ё рад карда шаванд, ки дар сабақи сабака шумо фаъол (network-activated) набошанд. Вақте ки як плагинал фаъол мешавад, он ба табиати худ, ҳангоми интихоб, дар ҳамаи сайти сабаки шумо автоматик маҷдӯз карда мешавад.

Агар шумо هیچ плагиналҳоро дар табби «Allowed Plugins» набинисед, санҷед, ки плагинали лозимшуда фаъол аст ё нест.

* **Оё ман метавонам тарҳи худ барои ҷадвали нарх (Pricing Table) -ро истифода барам?**

Ҳа. Аз версияи 1.4.0 ба кор омада, Ultimate Multisite дастгирии тағйир додани шаблонҳо (template overrides)-ро дорад.

* **Оё Ultimate Multisite бо WP Engine кор мекунад?**

Ҳа. Ultimate Multisite ҳамеша аз ҷониби WP Engine дастгирӣ мешуд, аммо барои кор карда намудани маппинг домен (domain mapping) аз ҷониби администратор қадамҳои иловагии мустақил талаб мешуд (дохили доменро дар панели WP Engine илова кардан). **Ин ҳоло ба таври нав версияи 1.6.0 намудааст.** Ultimate Multisite бо API-и WP Engine мустақиман ҳамкорӣ мекунад ва доменҳо пас аз маппинг автоматик ба панел илова карда мешаванд.

* **Шумо кадом пешниҳоди хостиро барои хостинг тавсия медиҳед?**

Ultimate Multisite метавонад бо ҳар як пешниҳоди хостинг кор кунад. Фақат фарқи он аст, ки баъзе ширкатҳои хостинг, мисли Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane ва [Runcloud.io](http://Runcloud.io), ҳамгирӣ барои қобилияти маппинг доменро мустақилтар доранд, ки онҳо аз авс (out of the box) кор мекунанд. Барои дигар хостингҳо, шумо метавонед маппинг доменро худатон таҳия кунед.
