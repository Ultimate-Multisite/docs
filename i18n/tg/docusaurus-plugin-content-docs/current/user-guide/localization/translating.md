# Тарҷумаи Ultimate Multisite {#translating-ultimate-multisite}

Ultimate Multisite ба забони инглиз таҳия шудааст, аммо плагин пурра таржимашаванда аст ва мо фаровон кӯмаки таржимаро қабул мекунем.

## Тарҷумаҳои асосии плагин (Core Plugin Translations) {#core-plugin-translations}

Плагини асосии Ultimate Multisite аз ҷониби платформаи расмии **WordPress.org** тасвир шудааст:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

Ин системаи тарҷумаи стандарт ҷомеаи WordPress аст. Вақте тарҷумаҳо ба мисли арзиши тасдиқи муайян расиданд, онҳо аз рӯи омодасозии плагини WordPress барои ҳамаи корбарони Ultimate Multisite автоматикӣ тақсим мешаванд.

**Мо ба шумо ниёз дорем!** Агар шумо забони дигари инглизро гоҳар дошта бошед, лутфан дар тарҷумаҳо иштирок намоед. Ҳатто тарҷумаи якчанд матн низ ёрӣ мерасонад. Барои оғоз кардан:

1. Ба [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/) равед.
2. Забони худро интихоб кунед.
3. Ба "Contribute Translation" (Тарҷумаи иштирок додан) клик кунед.
4. Вам ҳисоби озод WordPress.org лозим аст.

Агар шумо хоҳед, ки барои забони худ редактори таржима шавед (ки метавонад тарҷумаҳоро тасдиқ намояд), ба [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) пайваст шунед.

## Тарҷумаҳои плагини иловагӣ (Addon Translations) {#addon-translations}

Тарҷумаи плагиниҳо барои иловагӣ дар ҷои дигар тасвир карда мешаванд:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Ҳоло тарҷумаҳои иловагӣ ба таври сунъи зина (AI-generated) ҳамчун намуна оғоз мебошанд. Онҳо ҳамаи забонҳои дастгирӣ-ро дар бар мегиранд, аммо метавонанд ифодаи номутабиқ ё носохтсияти дошта бошанд. Мо тарҷумаварони инсонӣ (human translators) -ро барои баррасӣ ва таклифи беҳтар кардани онҳо ба таври табиӣ ва дурусттар истифода мебарем.

## Тарҷумаҳои маҳаллӣ (Local Translations) {#local-translations}

Агар шумо ба ҷои сахтсозии он (upstream) барои лоиҳаи худ тарҷума кунед, ин ду имконият доред:

### Истифодаи Poedit {#using-poedit}

Шумо метавонед аз додани барномаи桌面ӣ **[Poedit](https://poedit.net)** барои тарҷумаи файли `.pot` истифода баред:

1. Poedit-ро аз [poedit.net](https://poedit.net) дидаред.
2. Файли `.pot`-ро аз `wp-content/plugins/ultimate-multisite/lang/` кушод.
3. Мазмунҳо (strings) дар интерфейси Poedit тарҷума кунед.
4. Савдо кунед -- Poedit файллари `.po` ва `.mo` -ро таҳия мекунад.
5. Файлҳои намудашударо дар папка `lang` ҷойгир шавед.

### Истифодаи Loco Translate {#using-loco-translate}

Плагини WordPress-и **[Loco Translate](https://wordpress.org/plugins/loco-translate/)** ба шумо имкон медиҳад, ки бе ҳатмлоти воситаҳои桌面ӣ мустақиман аз панеڵی администратор тарҷума кунед. Он барои таҳия ва муттаҳии файллари таржимаро барои шумо идора мекунад.
