# رویداد: Регистрация учетной записи через Zapier {#event-register-an-account-via-zapier}

В статье [Интеграция Ultimate Multisite с Zapier](zapier.md) мы обсуждали, как использовать Zapier для выполнения различных действий в Ultimate Multisite на основе триггеров и событий. В этой статье мы покажем, как можно интегрировать сторонние приложения. Мы будем использовать Google Sheets в качестве источника данных и отправлять информацию в Ultimate Multisite для регистрации учетной записи.

Прежде чем подключать Zapier, перейдите в **Ultimate Multisite > Settings > API & Webhooks** и убедитесь, что API включен. Скопируйте API Key и API Secret с этого экрана, когда Zapier запросит учетные данные аккаунта Ultimate Multisite.

![Настройки API и вебхуков с ключом API, секретом API и опцией Включить API](/img/admin/settings-api-webhooks.png)

Сначала вам нужно создать **Google Sheet** в вашем Google Drive. Убедитесь, что вы правильно определили каждую колонку, чтобы позже могли легко сопоставить данные.

После создания Google Sheets вы можете войти в свой аккаунт Zapier и начать создавать zap.

В поле поиска для **"App event"** выберите **"Google Sheets"**.

Затем в поле "**Event**" выберите **"New spreadsheet row"** (Новая строка в таблице) и нажмите "**Continue**" (Продолжить).

Следующий шаг попросит вас выбрать **Google Account**, где хранится Google Sheet. Просто убедитесь, что указан правильный аккаунт Google.

В разделе **"Set up trigger"** вам нужно будет выбрать и указать ту Google Таблицу и лист, которые вы будете использовать в качестве источника данных. Просто заполните это и нажмите "**Continue**".

Далее следует "**test your trigger**" (протестировать ваш триггер), чтобы убедиться, что ваша Google Sheet правильно подключена.

Если тест успешен, вы должны увидеть результат, показывающий некоторые значения из ваших таблиц. Нажмите "**Continue**", чтобы продолжить.

Қадамҳои минбаъда ин аст, ки дуюмро таъмин кунед, ки ҳангори Ultimate Multisite корбар (account) сохта ё қайд мекунад. Дар майдони ҷудоварӣ (search field), **"Ultimate Multisite(2.0.2)"**-ро интихоб кунед.

Дар майдони **"Event"**, **"Register an Account in Ultimate Multisite"**-ро интихоб кунед ва ба тугмане, ки **"Continue"** (Даом ҷвариш) меравад, фиристоред.

Дар майдони **"Set up an action"**, шумо метавонед барои маълумоти муштариён, аъзогиҳо (memberships), маҳсулот ва ғайра тасвирҳои гунокиро бинед. Шумо метавонед арзишҳоро дар Google Sheet-и худ пайваст карда, онҳоро ба майдони мувофиқ ишора кунед, ки бояд дар онҳо пур шаванд (ба мисли сахтгӯи паст).

Баъди пайваст кардани арзишҳо, шумо метавонед ин корро санҷед.
