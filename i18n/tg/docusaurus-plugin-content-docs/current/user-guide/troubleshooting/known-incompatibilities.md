# Нокии мувофиатҳо бо плагинотҳои дигар {#known-incompatibilities-with-other-plugins}

Баъди он ки Ultimate Multisite озод карда мешавад ва фаъол карда мешавад, шумо метавонед ба масъалаҳо воқеӣ расонед аз номуайян карда нашавандагӣ бо плагинотҳои дигар.

Мо дар ҳар як навбатии нави онро барои кам кардани ин рӯйхат кор мекунем, аммо агар ягон плагин дар ин ҷо фароҳамъдад шавад, шумо метавонед онро безабон кунед, то Ultimate Multisite Network-и пурра коркарда дорои бошад.

Ин рӯйхати онҳо ва он ки шумо бояд дар ин ҳолат чӣ кор кунед:

**Баъзе плагинотҳои WPMU DEV** Баъзе плагинотҳое, ки аз ҷониби WPMU DEV нигоҳ намудашавандагӣ нест, маълум аст, ки онҳо равиши қайд кардани Ultimate Multisite-и ройгонро дар қадами охир (кардор наметавонад ҳисобот кунад) қатъ мекунанд. Ин рӯйхати плагинотҳоест, ки ин масъалаҳоро ба вуҷуд меоранд:

  * Signup Language: _Ultimate Multisite додани он аст, ки ҳамин намуди корро дорад ва номи ӯ Ultimate Multisite: Language Switcher аст._

  * Set Blog Description on Blog Creation: _Ин дар Ultimate Multisite сохта шудааст, аз ин рӯ, ба плагиноти иловагӣ ниёз надорад._

  * New Blog Templates: _Ин дар Ultimate Multisite сохта шудааст, аз ин рӯ, ба плагиноти иловагӣ ниёз надорад._

  * Set Password on Multisite Blog Creation: _Ин дар Ultimate Multisite сохта шудааст, аз ин рӯ, ба плагиноти иловагӣ ниёз надорад._

  * Signup TOS: _Ин дар Ultimate Multisite сохта шудааст, аз ин рӯ, ба плагиноти иловагӣ ниёз надорад._

  * Anti-Splog: Ҳарчанд он камтар қувват дорад, он низ _дар Ultimate Multisite сохта шудааст._

**Нотаъиба** : Ҳамаи дигар плагинотҳои WPMU Dev метавонанд ба таври маъмул дар паҳлӯи Ultimate Multisite истифода шаванд. Мисолҳоянд _Smush_ , _Forminator_ , _Defender_, ва ғайра.
