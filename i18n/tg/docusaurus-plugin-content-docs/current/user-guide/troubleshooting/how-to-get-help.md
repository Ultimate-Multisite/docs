# Чӣ тавр ёрӣ пайдо кардан

Шумо лицензияи возмӯадои худро доред ва ба он дастрасӣ ба дастаи мо барои кӯмак расонида шудааст, ки дар ҳалли мушкилоти имконпазир дар ояти танзим кардани Ultimate Multisite ёрӣ мерасонад.

Ин дарс дар бораи каналҳои дастгирии мавҷуда барои дордорони лицензия ва ин маълумоти гуногуни ки бояд доред, то бо ҳамроҳ бо дастаи дастгирии мо бо истифода аз онро пайваст кунед, фароҳ аст.

## Сайти ёрӣ (Help Site) {#help-site}

Шумо метавонед ба мақола ва дарсҳои гуногуни сайти ёрӣ расондед, ки дар ҳамин пайваст [ба ин пайваст](https://community.ultimatemultisite.com/) ҷойгир аст.

Мо истифодаи корбаронро барои шинос шудан бо хусусият ва функсияҳои асосии Ultimate Multisite тавсия медиҳем. Мақолаҳои "Оғоз" (Getting Started) дар сайти ёрӣ мавҷуданд ва ҷавобҳо ба саволҳои маъмул, аз қабили чӣ тавр плагиниҳоро инстадан ё чӣ тавр шウェイҳои масофаро паҳн кардан мумкин аст, дар онҳо ҷойгир мешаванд.

## Алоқа бо дастгирӣ (Contact Support) {#contact-support}

Агар шумо ба ҳеҷ гуна мушкилӣ расондед ва натавонед ҷавобро аз сайти ёрӣ пайдо кунед, метавонед ба дастаи дастгирии мо тамос гиред: яъне бо клик кардан ба иконка пушти сухан (speech bubble icon), ки дар паҳлӯи қафтирҳои паст-расои сайти мо ҷойгир аст, ё бо фиристодани почта ба `support@ultimatemultisite.com`.

Вақте ки барои мушкилоти техникӣ бо дастгирӣ алоқа мекунед, лутфан тасвири муфассали мушкилоти худро ҳамроҳ гиред ва скриншотҳо, номаҳои хато ё ҳар гуна маълумоти дигарро пешниҳод кунед, ки ба одамон дастгирии мо кӯмак мекунад, то онҳо мушкилоти шуморо ба осон фаҳм намоянд ва арзёбӣ намоянд. Шумо инчунин лицензияи худ-ро бояд пешниҳод кунед. Фақат дордорони лицензияи фаъол барои дастгирӣ ҳуқуқ доранд.

## Сайти ҷамъият (Community Site) {#community-site}

[Сайти ҷамъияти мо](https://community.ultimatemultisite.com/) макони муносибат ва ҳамкорӣ барои истифодабарандагони Ultimate Multisite мебошад. Он ҳамчун системаи иттилоот амал мекунад, ки аъзоҳо метавонанд паст кунанд, дар баррасии муҳокилаҳо иштирок намоянд, машвратнома диҳанд, ҳамкорӣ кунанд ё аз ҷомеа барои ҳалли мушкилот ҳамроҳ шаванд. Барои ошкор кардани макони махфӣ/саҳифаҳо бояд лицензияи Ultimate Multisite-и худро пешниҳод намоед.
