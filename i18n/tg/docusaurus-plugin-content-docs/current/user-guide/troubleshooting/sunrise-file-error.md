# Номишидаи Настанон кардани Файли Sunrise {#error-installing-the-sunrise-file}

Файли sunrise.php файли махсусиест, ки WordPress ҳангоми оғозиш (bootstrap) худро мехоҳад ҷудо кунад. Чтобы WordPress тавоноӣ дошта бошад файл `sunrise.php`-ро пайдо кунад, он бояд дар папка **wp-content** ҷойгир буда бошад.

Вақте шумо Ultimate Multisite-ро фаъол мекунед ва аз ройани танзимкунанда (setup wizard) ба монанди он, ки дар сахтсозии скриншот доред, Ultimate Multisite кӯшиш мекунад файл `sunrise.php`-и моро ба папка wp-content нусхабардорӣ кунад.

Аз рӯи аксария вақт, мо метавонем файлро муваффақият нусхабардорӣ намоем ва ҳама чиз дуруст кор мекунад. Аммо, агар ба таври дохилӣ чизе дуруст танзим нашуда бошад (масалан, иҷозатҳои папка), шумо метавонед ба ҳолати мушоҳида расонед, ки Ultimate Multisite наметавонад файлро нусхабардорӣ намояд.

Агар шумо нотариъи хатои Ultimo-ро бихонед, мебинед, ки ин ҳамон чиз дар ин ҷо рӯй додааст: **Sunrise copy failed** (Нусхабардории Sunrise намуваффақат гузашт).

Барои ислоҳ кардани он, шумо метавонед содда файл `sunrise.php`-ро дохили папка плагини wp-ultimo нусхабардорӣ кунед ва онро ба папка wp-content-и худ пайваст кунед. Баъди ин, саҳифаи танзимкунандаро муаррифӣ (reload) кунед ва санҷишҳо бояд дуруст бошанд.

Ҳангоми ин, метавонад лозим ояд ки иҷозатҳои папкаи худ аз рӯи умумии онро санҷед, то дар охир ба шумо мушкилот наоред (на танҳо бо Ultimate Multisite, балки бо плагини ва темаҳои дигар низ).

**Health Check tool** (Инчунин воситаи санҷиши саломатӣ) ки қисми WordPress аст (инро метавонед аз **admin panel > Tools > Health Check** дар саҳифаи асосии сайт-и худ пайдо кунед), қобилияти он дорад, ки ба шумо гӯяд, ки оё иҷозатҳои папкаи худро ба арзишҳое танзим карданете, ки метавонанд ба WordPress мушкилот баривед.
