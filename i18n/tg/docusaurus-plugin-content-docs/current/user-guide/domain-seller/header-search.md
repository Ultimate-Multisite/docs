# Ҷустуҷӯи домен дар сарлавҳа

Ин танзимро вақте истифода баред, ки мехоҳед формаи хурде дар сарлавҳа дошта бошед, ки ҷустуҷӯи доменро оғоз мекунад ва сипас ба муштарӣ имкон медиҳад аз доменҳои дастрас дар дохили саҳифаи пардохти Ultimate Multisite интихоб кунад.

## Талабот {#requirements}

- Ultimate Multisite дар шабака фаъол бошад.
- Multisite Ultimate Domain Seller дар шабака фаъол бошад.
- Ҳадди ақал як маҳсулоти фаъоли сабти домен бо:
  - `domain_provider` ба таъминотгари танзимшуда таъин шуда бошад.
  - TLD-ҳои дастгиришаванда танзим шуда бошанд, масалан `com`, `net` ва `org`.
- Формаи пардохти дуруст, ки майдони **Интихоби домен**-ро дар бар мегирад.

## Формаи пардохт {#checkout-form}

1. Формаи пардохтеро, ки саҳифаи сабтином истифода мебарад, эҷод ё таҳрир кунед.
2. Майдонҳои одии ҳатмии пардохт/Account-ро, аз ҷумла **Номи корбар**, илова кунед. Формаи пардохте, ки танҳо майдони доменро дар бар мегирад, аз тарафи санҷиши Ultimate Multisite рад карда мешавад.
3. Майдони **Интихоби домен** илова кунед.
4. Ҳолати Интихоби доменро ба **Танҳо сабт** гузоред, вақте ки ҷараён бояд ба доменҳои сабтшуда тамаркуз кунад, на ба зердоменҳои ройгон ё доменҳои мавҷуда.
5. Маҳсулоти сабти доменро ба он майдон таъин кунед.

Саҳифаи сабтином бояд формаи пардохтро намоиш диҳад, масалан:

```text
[wu_checkout slug="domain-form"]
```

## Формаи сарлавҳа {#header-form}

Дар сарлавҳаи сомона як формаи хурди `GET` илова кунед, ки истилоҳи ҷустуҷӯи воридшударо ба саҳифаи пардохт ҳамчун `domain_name` мефиристад:

```html
<form class="ums-header-domain-search" action="/register/" method="get">
  <label class="screen-reader-text" for="ums-header-domain-search-input">Search for a domain</label>
  <input id="ums-header-domain-search-input" name="domain_name" type="text" placeholder="Find your domain" autocomplete="off">
  <button type="submit">Search</button>
</form>
```

Дар JavaScript-и фармоишии сарлавҳа доменро пешакӣ интихоб накунед. Сарлавҳа бояд танҳо истилоҳи ҷустуҷӯро гузаронад. Скрипти пардохти Domain Seller `?domain_name=example`-ро мехонад, қуттии ҷустуҷӯи пардохтро пур мекунад ва ҷустуҷӯи дастрасиро иҷро мекунад, то муштарӣ аз доменҳои баргардонидашуда интихоб карда тавонад.

## Рафтори пешбинишуда {#expected-behaviour}

Ҷустуҷӯи `example` дар сарлавҳа бояд инро кушояд:

```text
/register/?domain_name=example
```

Сипас саҳифаи пардохт бояд натиҷаҳои интихобшавандаро намоиш диҳад, ба монанди:

- `example.com`
- `example.net`
- `example.org` дастнорас
- TLD-ҳои дигари дастгиришавандаи таъминотгар

Пас аз интихоби натиҷаи дастрас, хулосаи фармоиш бояд маҳсулоти сабти домен ва номи домени интихобшударо дар бар гирад.

## Санҷиш {#verification}

1. Саҳифаи асосиро кушоед.
2. Номи холисро ҷустуҷӯ кунед, масалан `example`.
3. Тасдиқ кунед, ки URL-и пардохт `?domain_name=example`-ро дар бар мегирад.
4. Тасдиқ кунед, ки майдони домен дар пардохт `example`-ро дар бар мегирад.
5. Тасдиқ кунед, ки рӯйхати интихоби доменҳо пайдо мешавад.
6. Барои домени дастрас **Интихоб**-ро клик кунед.
7. Тасдиқ кунед, ки хулосаи фармоиш `Domain Registration - example.com` ё домени интихобшударо дар бар мегирад.

## Ислоҳи мушкилот {#troubleshooting}

- Агар ягон рӯйхат пайдо нашавад, ҷадвали шабакаи браузерро барои `admin-ajax.php?action=wu_domain_search` тафтиш кунед ва тасдиқ кунед, ки он `domains` ё `results`-и холӣ набударо бармегардонад.
- Агар формаи пардохт ҳангоми захиракунӣ аз санҷиш нагузарад, майдонҳои ҳатмии Account, ба монанди **Номи корбар**, илова кунед.
- Агар интихоби домен сабадро нав накунад, тасдиқ кунед, ки `window.wu_checkout_form` вуҷуд дорад ва дороиҳои пардохти Domain Seller дар саҳифаи пардохт бор шудаанд.
