# Хукҳо ва филтрҳои Ultimate Multisite

Ultimate Multisite аз як маҷмӯи функсияҳо ва саволҳои база данных иборат аст. Он хукҳо (hooks) ва филтрҳо (filters) дорад, ки ба шумо имкон медиҳанд кодҳои мавҷударо тағйир диҳед ё кодҳои нав илова кунед.

Мо рӯйхати ин хукҳо ва филтрҳоро ҳамчун манбаи маълумот сохтарем, то корбарони пешрафта метавонанд шабакаи худсони худро бештар таҳия намоянд. Шумо метавонед ин рӯйхат аз Github бо клик кардани линки поёнро дарёвед.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
