# Веб-сайтҳои муштаби шумо (v2) {#your-clients-websites-v2}

Баъди он ки дар шабакаи шумо ройгон шуд, муштабои шумо метавонанд ба дашбор ихсоси худро ворид шаванд.

Илова ба саҳифаи **Account** (Ҳисобот), ки он ҷояст, ки онҳо иттилооти муҳим дар бораи пардохтҳо, гирифтани аъзоӣ, доменҳо, маҳдудоти нақша ва дигаронро мебийнаваранд, онҳо ба саҳифаи **My Sites** (Веб-сайтҳои ман) низ дастрасӣ доранд, ки дар он метавонанд ҳамаи веб-сайтҳои худро бибийнаваранд ва идора кунанд. Ҳамаи он ки бояд онҳо кунанд, ин аст, ки дар дашбор ояти "My Sites" (Веб-сайтҳои ман) -ро клик кунанд.

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

Ҳамаи веб-сайтҳои онҳо нишон дода мешаванд, ва дар паёи ҳар кадомина ба онҳо дастури **Manage** (Идора кардан) мавҷуд аст. Агар онро клик кунанд, онҳо ба **dashboard-i in website махсус** (дашбор-и ин веб-сайт мушаххас) равона мешаванд.

Онҳо инчунин метавонанд бибийнаваранд ки кадом яке аз веб-сайтҳои онҳо **primary** (асосии) аст барои нақшаҳое, ки имконият медиҳанд ки онҳо бештар аз як веб-сайт дошта бошанд.

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

Ин назари пурраи интерфейси тағйир додани саҳифаи сайт аст:

![Site edit full interface](/img/admin/site-edit-full.png)

Агар онҳо хоҳанд ки веб-сайти асосии худро тағйир диворанд, бояд ба саҳифаи Account (Ҳисобот) бо клик кардани менюи **Account** раванд.

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

Онҳо инчунин метавонанд маълумоти муштабони худро бибийнаваранд ва тағйир диворанд:

![Customer edit interface](/img/admin/customer-edit.png)

Баъди он, онҳо бояд ба **Change Default Site** (Тағйири саҳифаи восита) клик кунанд.

![Change Default Site button on account page](/img/account-page/change-default-site.png)

Пардози папочка пайдо мешавад. Ба рӯй рӯйди кор (drop-down list) клик кунед ва интихоб намоед, ки кадом веб-сайтро мехоҳед асосии он созед.

![Change Default Site dropdown selection](/img/account-page/change-default-site-dropdown.png)

Баъди он, фақат ба **Change Default Site** (Тағйири саҳифаи восита) клик кунед ва кор хутёр!
