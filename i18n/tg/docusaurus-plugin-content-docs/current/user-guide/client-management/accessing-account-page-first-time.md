# Чӣ тавр ман ба страницу ҳисоби хуп барои аввали варзиш мерасонам?

Бахшами шуморо дар Ultimate Multisite!

Барои дастрасӣ ба страницу ҳисоби худ, аз линки истифода кунед, ки пас аз хари плагини мо дар почтаи шумо расонида шудааст:

<!-- Скриншот мавҷуд нест: Почтаи страница ҳисоб бо линки дастрасӣ -->

Аз почтае, ки ҳангори харол куфтаед ва пароли худро аз он ба даст овардед, истифода ба онҳо кунед.

<!-- Скриншот мавҷуд нест: Страҳи воридши номбандӣ бо майдони почта ва пароль -->

Аз он сабаки ки ин аввали мароси муаррифи ин ҳисоб аст, аз шумо хоҳиш мекунанд, ки пароли худро тағйир диҳед. Онро тағйир диҳед ва ба тугмаи _Save_ (Савдо) фиристонед.

<!-- Скриншот мавҷуд нест: Формаи тағйири пароль бо тугмаи Save -->

Дар страницу ҳисоби худ, шумо метавонед ҳамаи маълумоти ҳисоби худро идора кунед, аз ҷумла ҳисоботдиҳӣ (billings), таҷдидҳо (renewals), бекор кардани хизматрасонӣ (cancellations) ва бисёр чизҳои дигар.
