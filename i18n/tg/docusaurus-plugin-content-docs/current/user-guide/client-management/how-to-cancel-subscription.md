# Чӣ тариқат манбаъи худро бартарӣ карданро қатъ мекунам? {#how-can-i-cancel-my-subscription}

Агар шумо лицензияи Ultimate Multisite дошта бошед, метавонед аз ҳар вақти хоҳед, таъмини муддати онро қатъ кунед. Фақат қадамҳои зеринро иҷро кунед:

<!-- Screenshot unavailable: Account page email with access link -->

Имэли худро, ки ҳангоми харид додани он пешниҳод кардаед, ва пароли худро истифода баред.

<!-- Screenshot unavailable: Login page with email and password fields -->

Ин экрани аввали саҳифаи ҳисоби шумост:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

Барои қатъ кардани муддати таъмини худ, дар менюи чап, ба **Renewing & Billing** равед.

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

Барои кушодани пӯшии ҷониби сарфақӣ (side window), ба ҳаракат аз паҳлӯй (arrow) дар паҳлӯй равед. Пас, шумо бояд он 옵сияро интихоб намоед, ки **auto-renew**-ро қатъ мекунад.

Система як паёми тасдиқро ба шумо нишон медиҳад.

Агар шумо аз қатъ кардани муддати таъмини худ амман мешумонед, фақат ба тугмаи _**Cancel Renewals**_ клик кунед. Баъди ин амал, аз шумо хоҳиш мекунанд, ки як талоши хурд (survey) ҷавоб диҳед.

Барои хушбахтӣ! Муддати таъмини шумо ба таври автоматик таҷдид намешавад.

**Шумо дойдед кали хеле муқаррарро то вақти пасари муддати таъмини худ**. Агар шумо хоҳед, ки муддати таъмини худро боз оғоз кунед, бояд инро ба таври рушдӣ (manually) иҷро намоед.
