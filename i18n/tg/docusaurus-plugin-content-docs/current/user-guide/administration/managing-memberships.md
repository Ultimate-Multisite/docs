# Идоракунии членӣ (Managing Memberships) {#managing-memberships}

Дар Ultimate Multisite шумо метавонед ҳамаи ҳатмӯзи (subscriptions) ва дастрасӣ ба онҳоро дар шабакаи худ идора кунед. Шумо метавонед маълумотҳои ҳатмӯзиро, аз ҷумла намудаҳо ва маҳсулот, шумораи сайтҳо, миқдори ҳисоботдиҳӣ ва тавридани он, таърихи пардохт ва соатҳои воқеаро баррасӣ ва тағйир диҳед.

Ба ин саҳифа барои дастрасӣ ба таббии **Memberships** дар менюи Ultimate Multisite равед.

![Memberships list page](/img/admin/memberships-list.png)

Дар саҳифаи Memberships шумо метавонед рӯйхати муштариёни худ ва маҳсулотҳои пайвастаро, ки ба ҳисоботи онҳо тааллуқ доранд, бинед. Шумо метавонед ҳатмӯзиҳоро аз рӯи ҳолати онҳо фаил (active), интизори (pending), дар паст (on hold), гузашташуда (expired) ё бекор (cancelled) филтр кунед.

Узатиҳои озод ҳамчун узатиҳои **ҳаёти муқаррарӣ** ҳисобида мешаванд. Онҳо санаи гузаштани вақти надоранд ва аз нав гӯштан автоматик ба он намеояд. Агар намуди озод пас аз муддати таҷриба ё муддати муайян ба даст оварӣ шавад, аз маҳсулоти пуст бо муддати таҷриба истифода баред, узатиро ба таври рушд ислоҳ кунед, ё вақте ки дастрасии он корбар бояд тағйир пайдо кунад, муштариёнро ба намуди дигар гузаронед.

Барои дастрасӣ ба саҳифаи тасвири узатиро раведдан, ба узатиҳо маълумотҳои иҷозат (маълумоти ҳисобот), маҳдудиятҳои сайт ё зиёдаш ва ғайраро тағир додан мумкин аст.

![Membership edit interface](/img/admin/membership-edit.png)

Ин намуди пурраи саҳифаи ислоҳ кардани узатиро инҷо медорад:

![Membership edit full interface](/img/admin/membership-edit-full.png)

Шумо инчунин узатиҳои навро, ки дар шабакаи шумо пеш аз ба регистратсияи онҳо қайд шудаанд, илова карда метавонед. Ба **Add Membership** клик кунед ва маълумотҳоро тартиб диҳед.

![Add Membership modal](/img/admin/membership-add-modal.png)

Синхер ба **Создание членства** клик кунед. Чандрои навтарин дар рӯйхати членҳо пайдо мешавад.
