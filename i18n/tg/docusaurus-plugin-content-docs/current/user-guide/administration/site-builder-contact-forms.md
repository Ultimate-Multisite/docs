# Формаҳои истиоли контактӣ дар сохтани сайт

Superdav AI Agent v1.10.0 имконияро барои сохтани формаҳои истиоли контактӣ мустақиман аз интерфейси чат бо истифори агент-и Site Builder медиҳад. Ин ба шумо имкон медиҳад, формаҳои истиоли пурра коркардаро ба ҳар як саҳифа илова кунед, бе он ки аз чат равед.

## Обзор {#overview}

Имконияи форма истиоли контактӣ дар Site Builder (`create_contact_form`) автоматикӣ плагини формаи иловашудаи шуморо муайян мекунад ва формати истиоли истиоли контактӣ бо қобилиятҳои табии он плагинро сохта месозад. Плагинҳои формаи дастгирӣ аз инҳо намоён мешаванд:

- Contact Form 7
- WPForms
- Fluent Forms
- Gravity Forms

Ассистент автоматикӣ беҳтарин плагини формаро дар сайти шумо интихоб карда ва формати истиоли контактӣ мувофиқ ба он плагин таҳия мекунад.

## Сохтани Формаи Истиоли Контактӣ {#creating-a-contact-form}

Барои сохтани формаи истиоли контактӣ бо истифори Site Builder:

1. Дар администри WordPress панеڵی чати **Gratis AI Agent**-ро кушоед.
2. Ба агент-и **Site Builder** таваҷҷуҳ шавед, ки ин кор аз иконтари агент дар сатри сарбанд чат аст.
3. Формаи истиоли контактиро, ки мехоҳед созед, тавсиф кунед. Масалан:

   > "Формаи истиоли контактӣ бо майдонҳои ном, почта, паём ва рақами телефонро ба саҳифа Контакт илова кунед."

   Ё соддатар:

   > "Формаи истиоли контактӣ барои саҳифа Контакт созед."

4. Site Builder формаи истиоли контактӣ-ро таҳия карда, коди қисм (shortcode) -ро пешниҳод мекунад, ки он омода барои илова шавад.

## Истифодаи Коди Қисми Таҳияшуда {#using-the-generated-shortcode}

Баъди он ки Site Builder формаи истиоли контактӣ-ро созад, он коди қисм (shortcode) -ро пешниҳод мекунад (масалан, `[contact-form-7 id="123"]`). Шумо метавонед:

1. **Онро дар саҳифа ё пост илова кунед** — Коди қисмро нусхабардорӣ карда, онро дар ҳар як саҳифа ё пост бо истифори block editor ё classic editor паст кунед.
2. **Онро тавассути Site Builder илова кунед** — Барои ин, аз Site Builder хоҳиш кунед, ки формаро ба саҳифаи муайян автоматикӣ илова намояд:

   > "Формаи истиоли контактӣ-ро ба саҳифа Контакт илова кунед."

3. **Использование его в шаблоне** — Если вы с PHP дружите, можете добавить этот shortcode в файл шаблона темы.

## Настройка контактной формы {#customizing-the-contact-form}

После того как Site Builder создаст контактную форму, вы сможете настроить ее еще больше:

### Через интерфейс чата {#via-the-chat-interface}

Попросите Site Builder изменить форму:

> "Обновите контактную форму, чтобы добавить поле 'Тема' и сделать поле сообщения обязательным."

Site Builder обновит форму и вернет обновленный shortcode.

### Через административный интерфейс плагина формы {#via-the-form-plugins-admin-interface}

Вы также можете настроить форму напрямую в настройках вашего плагина формы:

1. Перейдите в **Contact Form 7** (или ваш установленный плагин формы) в админке WordPress.
2. Найдите форму, созданную Site Builder.
3. Отредактируйте поля формы, правила проверки и уведомления по электронной почте по мере необходимости.

## Плагины форм и совместимость {#form-plugins-and-compatibility}

Site Builder автоматически определяет, какой плагин формы установлен на вашем сайте, и использует его для создания контактной формы. Если установлено несколько плагинов форм, Site Builder отдает приоритет им в следующем порядке:

1. Contact Form 7
2. WPForms
3. Fluent Forms
4. Gravity Forms

Если ни один из этих плагинов не установлен, Site Builder порекомендует установить один перед созданием контактной формы.

## Уведомления по электронной почте {#email-notifications}

Контактные формы, созданные Site Builder, по умолчанию настроены на отправку уведомлений по электронной почте администратору сайта. Вы можете изменить адрес получателя и сообщение уведомления:

1. Перейдите в административный интерфейс вашего плагина формы.
2. Найдите форму, созданную Site Builder.
3. Отредактируйте настройки уведомлений по электронной почте.

Для получения подробных инструкций обратитесь к документации вашего плагина формы:

- [Email Notifications for Contact Form 7](https://contactform7.com/docs/)
- [Email Notifications for WPForms](https://wpforms.com/docs/)
- [Email Notifications for Fluent Forms](https://fluentforms.com/docs/)
- [Email Notifications for Gravity Forms](https://docs.gravityforms.com/)

## Мисолҳои истифодаи онҳо (Example Use Cases) {#example-use-cases}

### Формаи истиозати мустамот (Simple Contact Form) {#simple-contact-form}

> "Формаи истиозати мустамотии соддаро бо майдони ном, почта ва паём тасвир кунед."

### Формаи бисёри қадамҳо (Multi-Step Form) {#multi-step-form}

> "Формаи истиозати мустамотӣ тасвир кунед, ки барои қадами аввал маълумоти иртибот ва барои қадами дуюм паём ва раванди алоқаро танзим мекунад."

### Форма бо маҷмӯи логикаи шартӣ (Form with Conditional Logic) {#form-with-conditional-logic}

> "Формаи истиозати мустамотӣ тасвир кунед, ки рӯйхатҳои гуноки майдонҳоро вобаста ба интихоби корбар дар меню (dropdown) нишон медиҳад."

### Форма бо интиқоли файл (Form with File Upload) {#form-with-file-upload}

> "Формаи истиозати мустамотӣ тасвир кунед, ки иҷозат медиҳад, ки корбарон як файл ё ҳуҷатро онро интиқол намоянд."

:::note
Таҳияи формаи истиозати мустамот дар Superdav AI Agent версияи v1.10.0 ва баъди он мавҷудаст. Барои истифодаи ин қобилият, агент Site Builder бояд фаъол бошад.
