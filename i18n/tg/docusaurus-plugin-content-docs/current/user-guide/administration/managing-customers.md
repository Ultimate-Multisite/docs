# Идоракунӣ аз муштаъриҳо {#managing-customers}

Саҳифаи **Customers** рӯйхати аъзони қайдшуда дар шабакаи шуморо нишон медиҳад. Ин саҳифа маълумоти асосии корбар, аз ҷумла ном, адрес электронной почта, охиргони воридшавӣ ва пайвастакунандаи **switch to** (инчунин: тасвир кардани муштариён) - ки иҷозат медиҳад ба ҳайси администратори сайти подзӯҳи корбар ба он ворид шавед.

![Customers list page](/img/admin/customers-list.png)

Барои дидани маълумоти муфассали оид ба як муштари, дар паTextInput-и **Manage** (Идоракунӣ) паст рӯйхати муштари, клик кунед.

Дар саҳифаи муфассали муштари, блоки **At a Glance** (Ба таври назаррас) маълумоти ID муштари, вақти охиргони воридшавӣ ва маблағи умумии сарфшудаи ин муштариро нишон медиҳад.

Блоки **Customer Options** (Варианти муштари) имкон медиҳад ки шумо ин муштариро ҳамчун **VIP** (маркази таги, ки пайдо кардани онро осон мекунад) муайян кунед. Табибаи **Billing Info** (Маълумоти ҳисоботдиҳӣ) маълумоти ҳисоботдиҳиро аз ҷумла почта, адрес ва кишвар нишон медиҳад.

Блок **Fields** номи корбарони муаррифӣ, почта ва имконият медиҳад, ки статуси **Email Verification**-ро тағйир диҳед.

Пас аз блок Customer Options шумо чор блоки иловагӣ мебинед:

- **Memberships** — Ҳуҷҷатҳои (subscription) ҳозираи корбарон, ки онҳо ба онҳо дастрас аст
- **Payments** — Таърихи пардохти корбар
- **Sites** — Сайтиҳое, ки ин корбар соҳиб аст
- **Events** — Вақтгузорӣ (events) бо тааллуқ ба ин корбар (тасдиқи почта, тағйироти ҳисобот ва ғайра)

Барои расонидани саҳифаи муфассали як элемент, ба иконка стрелка (→) пасли он клик кунед. Масалан, клик кардан ба стрелкаи пасли як сайт шуморо ба саҳифаи тасвири сайт мебарад, ки дар он метавонед пайвастагии дастрасии аъзо (membership association), турии сайт ва статуси фаолиашро тағйир диҳед.

Ҳамин хел ба аъзои корӣ ва пардохтҳо низ меояд — ҳар кадоме аз онҳо ба саҳмҳои муфассал мегузарад, ки дар он шумо метавонед статуси аъзои корӣ, муштаби асосӣ, маҳсулотҳо, садои ройхати пардохт ва ҳатто боздории пулро тартиб диҳед.

Дар паҳони рақамӣ (прасти) саҳифаи муштаби кормандон ба шумо инҳо ҷойгир мешаванд:

- **Ҳолати воридшавии охир ва IP** — Вақти воридшавии охир, адрес IP ва кишвар
- **Замонаҳо (Timestamps)** — Вақти сохта шудани ҳисоби муштаби кормандон ва илова кардани навбатии он
- **Номиёни гузариш** — Имконият барои ба таври доимӣ муштаби кормандони гузаронидашударо ба кор даровардани (Delete)

:::warning
Барои кормандони гузаронидашуда, ба таври доимӣ ва баргардории он имкон надорад.
:::
