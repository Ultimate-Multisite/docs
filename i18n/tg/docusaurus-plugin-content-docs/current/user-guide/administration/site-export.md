# Экспорт сайта

Панель администрирования **Site Export** позволяет сетевым администраторам упаковать один сайт или всю сеть в загружаемый архив для миграции, резервного копирования или передачи задач.

## Экспорт одного сайта {#exporting-one-site}

Перейдите в **Ultimate Multisite > Site Export** и выберите **Generate new Site Export**. Выберите подсайт, который хотите экспортировать, а затем решите, нужно ли включать в архив загрузки (uploads), плагины и темы.

Когда экспорт завершится, скачайте ZIP из списка **Existing Exports**. Экспортные ZIP-файлы теперь включают самозапускаемый `index.php` и файл `readme.txt`, чтобы архив можно было загрузить на новый хостинг и запустить без предварительной установки отдельного плагина импортера.

## Экспорт всей сети {#exporting-the-whole-network}

Используйте **Network Export** на странице Site Export, когда вам нужен один архив, содержащий все подсайты в сети. Это полезно перед миграциями хостов, учениями по восстановлению после катастроф или перестройками на тестовом стенде, где каждый подсайт должен перемещаться вместе.

Поскольку сетевой экспорт может быть намного больше, чем экспорт одного сайта, запускайте его в период низкой нагрузки и убедитесь, что целевое хранилище имеет достаточно свободного места для загрузок, плагинов, тем и сгенерированных ZIP-файлов.

### Пакеты импорта сети (Network Import Bundles) {#network-import-bundles}

Начиная с Ultimate Multisite 2.12.0, Site Exporter может генерировать **network import bundles** — специальные архивы, разработанные для упрощенного восстановления сайта во всей сети. Сетевой пакет импорта содержит все необходимые файлы и метаданные для восстановления нескольких сайтов на новой сетевой установке.

#### Генерация пакета импорта сети {#generating-a-network-import-bundle}

1. Барои раврасидани ба **Ultimate Multisite > Site Export** равад баронед
2. Барои интихоби **Generate new Network Export** клик кунед
3. Барои навсозии на хулосаи экспорт, **Network Import Bundle**-ро электор кунед
4. Интихоб кунед ки озодҳо (uploads), плагинҳо ва темаҳо (themes) дар он аз ҳама меоянд
5. Барои интихоби навсозии, **Generate** клик кунед
6. Барои дидани хулосаи ZIP аз рӯй рӯйростӣ ба рӯй **Existing Exports**

#### Раврасонидан аз Network Import Bundle {#restoring-from-a-network-import-bundle}

Барои раврасонидани сайти аз пакет импорти сети:

1. Ultimate Multisite-ро дар хости мақсадӣ инстаред
2. Дараҷаи гузариш (setup wizard) -ро барои multisite тамом кунед
3. Ба **Ultimate Multisite > Site Export** дар шабакаи нав равед
4. Барои импорти, **Import Network Bundle** клик кунед
5. Файли ZIP-и пакет импорти сетиро интиқол диҳед (upload)
6. Дараҷаи импортро аз рӯй экранҳо иҷро кунед
7. Раванди импорт ҳама сатҳо, мундариҷа ва тадбирҳои онҳоро ба навсоз мебарад

Network import bundles инҳоро ҳифз мекунанд:
- Мундариҷаи сайт (постҳо, саҳифаҳо, навсозгориҳои махсус)
- Тадбирҳо ва танзимҳои сайт
- Навти кормандон ва иҷозатҳо (roles and permissions)
- Плагинҳо ва темаҳо (агар дар пакет омада бошанд)
- Омадашудаҳои медиа (agar dar bundle o'shod boshad)
- Джадвали доираи дастрасӣ ва маълумотҳои махсуси база данных

## Инсталл кардани ZIP-и экспортӣ, ки худаш идора мекард {#installing-a-self-booting-export-zip}

Барои раврасонидани ZIP-и мустақилли, ки дар хости нав омадааст:

1. Мундариҷаи ZIP-и экспортшударо ба кор истифодашавандаи веб (web root) мақсадӣ интиқол диҳед
2. `index.php` -ро ки импорти шуд, дар браузер кушоед
3. Дараҷаи инсталлро аз пакет экспортшуда иҷро кунед
4. Пеш аз бартарии файлони муваққата, **readme.txt**-и пакети экспортшударо барои нотходдаҳои махсуси экспорт баррасӣ кунед

Барои инсталл ва ҷузъи импорти махсуси addonҳо, [Site Exporter addon documentation](/addons/site-exporter)-ро биёвед.

Барои инструменти таҳияи сайт (single-site tool), ки дар Ultimate Multisite 2.9.0 омадааст, ба [Export & Import](/user-guide/administration/export-import) биравед.
