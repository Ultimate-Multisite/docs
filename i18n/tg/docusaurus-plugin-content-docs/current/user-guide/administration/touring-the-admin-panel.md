# Гурӯҳи администри (Admin Panel)

Ultimate Multisite мехоҳад ба таври имкони мумкин ва беифаҳмо ин аст, аммо бо инструменти қадрии он, он метавонад барои доираи ройхати ройгонӣ кӯтоби ройгонӣ (guided tour) пешниҳоди кор кунад. Биёед ба сахнаҳои администри равед варӣ бирасем, то шумо ором шавед.

## Dashboard (Панели дохилгарӣ)

**Dashboard**-и Ultimate Multisite ҳисобот ва таҳлили асосиро нишон медиҳад, аз ҷумла даромадҳо, фаъолияти сатҳи вебсайт, олами аъзони ройгонӣ, шумораи меъбодон ва маълумоти географикӣ.

![Network Dashboard full page](/img/admin/network-dashboard-full-page.png)

## Checkout Forms (Формаҳои пардохт)

**Checkout Forms** шуморо имконият медиҳанд, ки сахнаҳои ройгонӣ варзишӣ (registration pages) худро таҳия кунед. Шумо метавонед сохторҳоро тағйир диҳед, классон ва скриптҳои худро ворид намоед ва дастрасӣ ба асоси кишти визитёр ё макони он тақсим карда шавад.

![Checkout Forms list](/img/admin/checkout-forms-list.png)

## Products (Маҳсулот)

Қисми **Products** ҷоест, ки шумо барои шабакаи худ намунаҳои гуногуни сарчашмаҳо, бандҳо ва хизматрасониҳоро таҳия мекунед. Дар сатҳи маҳсулот нархҳо, давраҳои ҳисоббарии боби муайян ва маҳдудиятҳо ва қисмҳо (quotas) нишон диҳед.

![Products list](/img/admin/products-list.png)

## Memberships (Аъзогирӣ)

Сахнаи **Memberships** ҳамаи ройгонӣ дар шабакаи шуморо нишон медиҳад. Тафсилоти ройгонии худро, аз ҷумла сатҳиҳо, маҳсулотҳо, миқдори ҳисоббарии боби муайян ва давраҳо, таърихи пардохт ва вақтҳои (timestamps) тафсир кунед ва тағйир диҳед.

![Memberships list](/img/admin/memberships-list.png)

## Payments (Пардохтҳо)

Саҳифаи **Payments** намудани мушкилоти пардохтро дар тамоми шабакаи шумо ба таври тез нишон медиҳад ва маълумоти муаммоти транзаксияҳо, аз ҷумла маҳсулот ва миқдори мушаххас.

![Payments list](/img/admin/payments-list.png)

## Customers

Саҳифаи **Customers** ҳамаи аъзони қайдшуда дар шабакаи шуморо нишон медиҳад, аз интихоби маълумоти асосии корбар аз он аз ном, адрес электронной почта, охиргони воридшавӣ ва пайвастагии "switch to" барои воридшавӣ ба подсайт-и он.

![Customers list](/img/admin/customers-list.png)

## Sites

Саҳифаи **Sites** ҳамаи подсайтро дар шабакаи шумо, аз ҷумла шаблонҳои сайт ва сайти муаллифи корбарро нишон медиҳад. Шумо ба таври осон доменҳои намудашуда, маҳдудиҳо ва квотаҳо, плагинотҳо ва темаҳо, инчунин тасвирҳои сайтро идора мекунед.

![Sites list](/img/admin/sites-list.png)

## Domains

Саҳифаи **Domains** ба доменҳои махсуси муайяншуда барои подсайтҳо хос аст. Дар ҳайси супер администратор, шумо метавонед номҳои доменҳои махсусро худаш илова ё нишон диҳед ва сардоштҳои DNS ва логҳоро бинад.

![Domains list](/img/admin/domains-list.png)

## Discount Codes

**Discount Codes** имкон медиҳанд, ки шумо кодҳои купонро барои пешниҳоди тахфир ба корбарони охири худ созед. Арзиши тахфирро муайян карда ва корбарон ё гурӯҳҳои мушаххаси мақсадӣ-ро муайян кунед.

![Discount Codes list](/img/admin/discount-codes-list.png)

## Broadcasts

**Broadcasts** воситаи фароҳам овардани эълонҳо ё фиристодани паймондагони махфият ба корбарони подсайт-и шумост. Гурӯҳҳои мушаххаси корбарро бар асоси нақша ё маҳсулоти ки онҳо ба он тавсея мегиранд, муайян кунед.

![Broadcasts list](/img/admin/broadcasts-list.png)

## Settings

Саҳифаи **Settings** ҷоест, ки шумо Ultimate Multisite — садори қайдшуда, садори пардохт, API ва webhooks, нақша кардани доменҳо ва дигар пайвастшавӣ-ҳоро тартиб медиҳед.

تنظیمات اتصال هوش مصنوعی لیست کردنِ مجموعه‌های ارائه‌دهندگان OAuth پشتیبانی‌شده فعلی: Anthropic Max، OpenAI ChatGPT/Codex و Google AI Pro هستند. هر کارت ارائه‌دهنده به سوپر ادمین‌ها اجازه می‌دهد حساب‌ها را متصل کنند، حساب‌های ذخیره‌شده را به‌روزرسانی نمایند، حساب‌ها را با ایمیل حذف کنند و در صورت مسدود شدن بازوجه مرورگر توسط محیط sandbox، از روش جایگزین OAuth دستی استفاده کنند. گزینه‌های تنظیمات Cursor Pro از پنل ادمین حذف شده‌اند.

حساب‌های ChatGPT/Codex از قابلیت استفاده از ابزار پشتیبانی‌شده توسط کانکتور پشتیبانی می‌کنند که در آن عملیات اجازه استفاده از ابزار را می‌دهد؛ بنابراین، گردش کارهای مدیریتی که به عملیات‌های پشتیبانی‌شده توسط کانکتور وابسته هستند می‌توانند پس از اتصال حساب OpenAI از رفتار ابزار Codex استفاده کنند.

## رویدادها (Events)

صفحه **رویدادها** سوابق تمام رویدادها و لاگ‌ها در شبکه شما را نگه می‌دارد. این صفحه فعالیت‌هایی مانند تغییر طرح، ثبت‌نام و سایر حرکات را ردیابی می‌کند که برای نظارت بر سایت‌های چندگانه شما مفید است.

![لیست رویدادها](/img/admin/events-list.png)

## وب‌هوک‌ها (Webhooks)

**وب‌هوک‌ها** به شما اجازه می‌دهند داده‌ها را به یک برنامه دیگر ارسال کنید. این قابلیت برای ارسال داده از Ultimate Multisite به پلتفرم‌هایی مانند Zapier مفید است.

![لیست وب‌هوک‌ها](/img/admin/webhooks-list.png)
