# Танзимоти Gratis AI Agent

Экрани **Settings → Advanced** дар Gratis AI Agent барои ҳамгироиҳои backend конфигуратсияи сатҳи мудирро фароҳам меорад. Ин саҳифа ирсоли feedback, калидҳои таъминкунандаи ҷустуҷӯ, танзими хидмати идорашавандаи Superdav, идораҳои Google Calendar, танзимоти SMS-и TextBee ва парчамҳои имкониятҳои сатҳи тамоми шабакаро ҳуҷҷатгузорӣ мекунад.

## Дастрасӣ ба танзимот

1. Дар мудирияти WordPress, ба **Gratis AI Agent → Settings** гузаред.
2. Ҷадвали **Advanced**-ро клик кунед.

## Конфигуратсияи endpoint-и feedback

Endpoint-и feedback ҳар вақте ки корбар тавассути тугмаи thumbs-down, баннери auto-prompt ё фармони `/report-issue` feedback мефиристад, аз AI agent дархостҳои POST қабул мекунад.

| Майдон | Тавсиф |
|---|---|
| **Feedback Endpoint URL** | URL-е, ки фиристодани feedback-ро ҳамчун дархостҳои HTTP POST бо баданаи JSON қабул мекунад. |
| **Feedback API Key** | bearer token-е, ки дар `Authorization` header-и ҳар дархости feedback фиристода мешавад. Агар endpoint-и шумо аутентификатсия талаб накунад, холӣ гузоред. |

### Баданаи JSON-и интизоршаванда

Endpoint-и feedback-и шумо бояд баданаи JSON-ро бо ҳадди ақал майдонҳои зерин қабул кунад:

```json
{
  "message_id": "msg_abc123",
  "conversation_id": "conv_xyz789",
  "feedback_text": "The answer was incorrect about pricing.",
  "triage_category": "factual_error"
}
```

Вобаста ба контексти суҳбат, дар payload майдонҳои иловагӣ низ метавонанд ҳузур дошта бошанд.

### Қиматҳои `triage_category`

Қабати triage-и AI пеш аз ирсоли payload яке аз қиматҳои зеринро ба `triage_category` таъин мекунад:

| Қимат | Маъно |
|---|---|
| `factual_error` | Assistant иттилооти воқеии нодуруст пешниҳод кард. |
| `unhelpful_answer` | Ҷавоб аз ҷиҳати техникӣ дуруст буд, аммо муфид набуд. |
| `inappropriate_content` | Ҷавоб дорои муҳтавое буд, ки набояд ба корбарон нишон дода шавад. |
| `other` | Feedback ба категорияи маълум мувофиқат накард. |

### Аутентификатсия

Агар endpoint-и шумо аутентификатсия талаб кунад, майдони **Feedback API Key**-ро ба bearer token-и худ таъин кунед. Agent мефиристад:

```
Authorization: Bearer <your-api-key>
```

Агар майдони **Feedback API Key** холӣ бошад, ягон `Authorization` header фиристода намешавад.

### Ғайрифаъол кардани ҷамъоварии feedback

Ҳар ду майдони **Feedback Endpoint URL** ва **Feedback API Key**-ро холӣ гузоред. Тугмаи thumbs-down ва UI-и feedback барои корбарон намоён мемонанд, аммо фиристодашудаҳо ба ягон хидмати беруна ирсол намешаванд.

## Brave Search API Key

Инчунин дар ҷадвали **Advanced**, майдони **Brave Search API Key** қобилияти [Ҷустуҷӯи интернет](../configuration/internet-search)-ро фаъол мекунад.

| Майдон | Тавсиф |
|---|---|
| **Brave Search API Key** | Калиди API-и шумо аз dashboard-и таҳиягарони Brave Search. Барои фаъол кардани ҷустуҷӯи интернет дар AI assistant зарур аст. |

Нишони майдон пайванди кликшаванда ба саҳифаи сабтиноми Brave Search API-ро дар бар мегирад. Барои ғайрифаъол кардани ҷустуҷӯи интернет холӣ гузоред.

Барои ҳуҷҷатгузории корбари ниҳоӣ оид ба ин имконият, ба [Ҷустуҷӯи интернет](../configuration/internet-search) нигаред.

## Хидмати идорашавандаи Superdav

Superdav AI Agent v1.18.0 endpoint-ҳои хидмати идорашавандаи Superdav ва provisioning-и худкори пайвастшавиро барои сайтҳои дастгиришаванда илова мекунад. Вақте ки сайти шумо бояд ба таъминкунандаи hosted ба ҷойи endpoint-и хидмати ба таври дастӣ конфигуратсияшуда пайваст шавад, ин идораҳоро истифода баред.

| Майдон | Тавсиф |
|---|---|
| **Managed Superdav Service** | Пайвастшавӣ ба хидмати hosted Superdav-ро барои сайтҳои дастгиришаванда фаъол мекунад. |
| **Provision Connection** | Provisioning-и худкори endpoint ва маълумоти эътимодномаро оғоз мекунад. Инро пас аз тасдиқ кардани он истифода баред, ки сайт бояд таъминкунандаи идорашавандаро истифода кунад. |
| **Service Endpoint / Connection Status** | Пас аз provisioning, endpoint ё ҳолати пайвастшавии ҷориро нишон медиҳад. |

Пас аз provisioning, танзимотро захира кунед ва пеш аз такя кардан ба workflow-ҳои хидмати идорашаванда ҳолати пайвастшавиро санҷед. Агар provisioning ноком шавад, роҳнамоии такрори намоишдодашударо баррасӣ кунед ва тасдиқ намоед, ки сайт иҷозати истифодаи таъминкунандаи hosted-ро дорад.

## Конфигуратсияи Google Calendar

Вақте ки имкониятҳои тақвими Superdav AI Agent v1.18.0 фаъоланд, agent метавонад тақвимҳои конфигуратсияшуда ва ҷузъиёти рӯйдодҳоро хонад. Абзорҳои тақвим ба хондан нигаронида шудаанд ва барои ёдраскуниҳои бо ҷадвал огоҳ, пайгирии иштирокчиён ва мувофиқсозии тамосҳо муфиданд.

| Майдон | Тавсиф |
|---|---|
| **Google Calendar Credentials** | Маълумоти эътимоднома ё пайвастшавии token-ро, ки барои хондани маълумоти тақвим зарур аст, нигоҳ медорад. |
| **Calendar Selection** | Маҳдуд мекунад, ки agent кадом тақвимҳои конфигуратсияшударо метавонад баррасӣ кунад. |
| **Calendar Connection Status** | Тасдиқ мекунад, ки оё маълумоти эътимодномаи ҷорӣ метавонад тақвимҳо ва рӯйдодҳоро хонад. |

Маълумоти эътимодномаи тақвимро танҳо ба тақвимҳое маҳдуд нигоҳ доред, ки agent ба онҳо ниёз дорад. Агар ҳолат token-и муҳлаташ гузаштаеро нишон диҳад, аз нав пайваст кунед ё маълумоти эътимодномаро иваз намоед.

## Огоҳиномаҳои SMS-и TextBee

Superdav AI Agent v1.18.0 TextBee-ро ҳамчун таъминкунандаи SMS барои workflow-ҳои огоҳиномаи конфигуратсияшуда илова мекунад. Огоҳиномаҳои SMS бояд барои паёмҳои ҳассос ё рӯ ба корбар бо дарвозаҳои тасдиқи инсонӣ ҷуфт карда шаванд.

| Майдон | Тавсиф |
|---|---|
| **TextBee API Key** | Дархостҳоро ба таъминкунандаи SMS-и TextBee аутентификатсия мекунад. |
| **TextBee Device / Sender** | Вақте ки таъминкунанда талаб мекунад, фиристанда ё дастгоҳи TextBee-ро, ки барои паёмҳои баромад истифода мешавад, интихоб мекунад. |
| **SMS Notifications Enabled** | Ба workflow-ҳои тасдиқшуда иҷозат медиҳад, ки огоҳиномаҳои паёми матнӣ фиристанд. Барои пешгирӣ аз фиристодани SMS ғайрифаъол гузоред. |

Паёми санҷиширо танҳо ба рақаме фиристед, ки ба мудир тааллуқ дорад, сипас пеш аз фаъол кардани ёдраскуниҳои ҷадвалшуда ё рӯ ба иштирокчиён рафтори дарвозаи тасдиқро тасдиқ кунед.

## Парчамҳои имкониятҳо

Инчунин дар v1.9.0 муаррифӣ шудааст, ҷадвали **Settings → Feature Flags** калидҳои ивазкунандаро барои функсионалияти ихтиёрӣ фароҳам меорад. Ҳар як парчам дар сатҳи тамоми шабака ё фаъол аст ё ғайрифаъол; дар айни замон override барои ҳар сайт вуҷуд надорад.

### Дастрасӣ ба парчамҳои имкониятҳо

1. Дар мудирияти WordPress, ба **Gratis AI Agent → Settings** гузаред.
2. Ҷадвали **Feature Flags**-ро клик кунед.

### Парчамҳои идоракунии дастрасӣ

| Парчам | Пешфарз | Тавсиф |
|---|---|---|
| **Маҳдуд кардан ба мудирон** | Хомӯш | Вақте фаъол аст, танҳо корбароне бо нақши `administrator` метавонанд панели чати AI Agent-ро кушоянд. Ҳама нақшҳои дигар ба ҷойи он паёми "Бо мудири худ тамос гиред"-ро мебинанд. |
| **Маҳдуд кардан ба Network Admins** | Хомӯш | Вақте дар шабакаи бисёрсомона фаъол аст, танҳо Super Admins метавонанд agent-ро истифода баранд. Мудирони сомонаҳои алоҳида баста мешаванд. Агар ҳар ду фаъол бошанд, бар "Маҳдуд кардан ба мудирон" афзалият дорад. |
| **Иҷозат додани дастрасии Subscriber** | Хомӯш | Вақте фаъол аст, корбароне бо нақши `subscriber` метавонанд интерфейси чатро истифода баранд, аммо ба қобилиятҳои танҳо-хондан маҳдуд мешаванд (бе эҷоди навишта ё тағйири танзимот). |
| **Ғайрифаъол кардан барои ғайриаъзоён** | Хомӯш | Бо ҳолати узвияти Ultimate Multisite ҳамгиро мешавад. Вақте фаъол аст, чат барои сомонаҳое, ки узвияти фаъол надоранд, пинҳон мешавад. |

### Парчамҳои брендсозӣ

| Парчам | Пешфарз | Тавсиф |
|---|---|---|
| **Пинҳон кардани поёни "Powered by Gratis AI Agent"** | Хомӯш | Сатри нисбати брендиро, ки дар поёни виҷети чат нишон дода мешавад, нест мекунад. Барои роҳандозиҳои white-label тавсия мешавад. |
| **Номи фармоишии Agent** | *(холӣ)* | Нишони пешфарзи "Gratis AI Agent"-ро дар сарлавҳаи чат ва менюи мудир бо номи маҳсулоти худатон иваз мекунад. Барои истифодаи пешфарз холӣ гузоред. |
| **Пинҳон кардани интихобгари Agent** | Хомӯш | Вақте фаъол аст, корбарон наметавонанд байни панҷ agent-и дарунсохт гузаранд. Agent-и ҷорӣ ба он чизе собит мешавад, ки ҳамчун пешфарз дар Settings → General танзим шудааст. |
| **Истифодаи нишонаи сомона ҳамчун аватари чат** | Хомӯш | Нишонаи пешфарзи AI-ро дар сарлавҳаи виҷети чат бо нишонаи сомонаи WordPress иваз мекунад (дар Appearance → Customize → Site Identity танзим мешавад). |

### Парчамҳои амнияти худкорсозӣ

Superdav AI Agent v1.18.0 дарвозаҳои тасдиқи инсонӣ ва сабтҳои ёдрасро барои иҷроҳои худкорсозии бехатартар ҷорӣ мекунад. Ин идораҳо вобаста ба бастаи насбшуда метавонанд дар парчамҳои хусусият ё танзимоти пешрафтаи худкорсозӣ пайдо шаванд.

| Парчам | Пешфарз | Тавсиф |
|---|---|---|
| **Талаб кардани тасдиқи инсонӣ** | Тавсия мешавад фаъол | Худкорсозиҳои ҳассосро то вақте таваққуф мекунад, ки корбари ваколатдор амали пешниҳодшударо баррасӣ ва тасдиқ кунад. |
| **Пешгирии такрори ёдрасҳо** | Фаъол | Ёдрасҳои фиристодашударо сабт мекунад, то кӯшишҳои такрорӣ ё иҷроҳои банақшагирифташуда огоҳиномаҳои такрорӣ нафиристанд. |
| **Фаъол кардани абзорҳои тақвим** | Хомӯш то танзим шудан | Ба agent иҷозат медиҳад тақвимҳо ва рӯйдодҳои танзимшудаи Google-ро бихонад. |
| **Фаъол кардани огоҳиномаҳои SMS** | Хомӯш то танзим шудан | Ба равандҳои кории тасдиқшуда иҷозат медиҳад, пас аз захира шудани маълумоти эътимод, огоҳиномаҳои SMS-и TextBee фиристанд. |

### Татбиқи тағйирот

Пас аз иваз кардани ҳар гуна парчам **Save Settings**-ро клик кунед. Тағйирот фавран эътибор пайдо мекунанд — тоза кардани кеш ё аз нав фаъолсозии plugin лозим нест.
