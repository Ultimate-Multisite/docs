# Филтр: wu_password_require_number

Филтр мекунад, ки оё дар гузарвожаҳо рақамҳо талаб карда шаванд.

## Параметрҳо

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | Оё рақамҳо талаб карда шаванд. Пешфарз true аст, вақте ки қоидаҳо иҷро карда мешаванд. |
| $defender_active | `bool` | Оё Defender Pro Strong Password фаъол аст. |

- 2.4.0
