# Филтр: wu_get_site_domain_and_path

Ба таҳиягарон имкон медиҳад, ки ҷуфтҳои домен/масирро тағйир диҳанд.

Ин метавонад барои як қатор чизҳо муфид бошад, масалан, татбиқи ягон навъи ҳалли муҳити озмоишӣ, серверҳои гуногун ва ғайра.

## Параметрҳо {#parameters}

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $d | `object` | Объекти ҷорӣ, ки калидҳои домен ва масирро дар бар мегирад. |
| $path_or_subdomain | `string` | Масир/зердомене, ки аслан ба функсия дода шудааст. |

### Аз версияи {#since}

- 2.0.0
### Манбаъ {#source}

Дар [`inc/functions/site.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/site.php#L235) дар сатри 235 муайян шудааст


## Бозмегардонад {#returns}
Объекте, ки калидҳои домен ва масирро дар бар мегирад.
