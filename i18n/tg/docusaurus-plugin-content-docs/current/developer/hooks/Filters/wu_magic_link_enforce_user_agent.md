# Филтр: wu_magic_link_enforce_user_agent

Филтр мекунад, ки оё санҷиши user agent маҷбурӣ шавад.

Ба false гузоред, то token-ҳо дар browser-ҳо/дастгоҳҳои гуногун кор кунанд. Ин амниятро коҳиш медиҳад, аммо истифодабариро осонтар мекунад.

## Параметрҳо

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $enforce | `bool` | Оё мувофиқати user agent маҷбурӣ шавад. |

### Аз версияи

- 2.0.0
### Манбаъ

Дар [`inc/sso/class-magic-link.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/sso/class-magic-link.php#L410) дар сатри 410 муайян шудааст.
