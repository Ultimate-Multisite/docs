# Филтр: wu_current_set_customer {#filter-wucurrentsetcustomer}

|------|------|-------------|
| $customer | `\WP_Ultimo\Models\Customer` | Муштарии ҷорӣ барои таъин кардан. |
| $ | `self` | Намунаи класси Current. |
