# Филтр: wu_billing_address_fields {#filter-wubillingaddressfields}

Ба таҳиягарони plugin иҷозат медиҳад, ки майдонҳои суроғаи ҳисоббаробаркуниро филтр кунанд.

## Параметрҳо {#parameters}

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $fields | `array` | Массиви суроғаи ҳисоббаробаркунӣ. |
| $zip_only | `bool` | Агар ба мо танҳо индекси почта ва кишвар лозим бошад. |
