# Филтр: wu_sso_enabled {#filter-wussoenabled}

Қобилияти вуруди ягонаи байнидомениро фаъол/ғайрифаъол мекунад.

Ин арзишро филтр кунед, то вуруди ягонаро комилан хомӯш кунед, ё ба ҷои он онро шартан фаъол созед.

## Параметрҳо {#parameters}

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $enabled | `bool` | Оё SSO бояд фаъол бошад? True барои фаъол, false-ish барои хомӯш. |

### Аз версия {#since}

- 2.0.11
### Манбаъ {#source}

Дар [`inc/sso/class-sso.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/sso/class-sso.php#L110) дар сатри 110 муайян шудааст


## Бармегардонад {#returns}
Оё SSO фаъол аст ё не.
