# Filter: wu_contains_element {#filter-wucontainselement}

Ба таҳиягарон иҷозат медиҳад, ки натиҷаҳои ҷустуҷӯи ибтидоиро тағйир диҳанд.

Ин барои созандагони тарафи сеюм ва монанди инҳо муфид аст.

## Параметрҳо {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $contains_elements | `bool` | Агар элемент дар муҳтаво мавҷуд бошад. |
| $content | `string` | Муҳтавое, ки санҷида мешавад. |
| $element | `self` | Элементи ҷорӣ. |
| $post | `null\|\WP_Post` | post барои санҷиш. |
