# Филтр: wu_documentation_get_link {#filter-wudocumentationgetlink}

Ба таҳиягарони plugin иҷозат медиҳад, ки пайвандҳоро филтр кунанд.

Маълум нест, ки ин чӣ гуна муфид буда метавонад, аммо доштани он зарар надорад

## Параметрҳо {#parameters}

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $link | `string` | Пайванди сабтшуда |
| $slug | `string` | Slug-е, ки барои гирифтани пайванд истифода мешавад |
| $default_link | `string` | Пайванди пешфарзи сабтшуда |

### Аз версияи {#since}

- 1.7.0
