# Филтр: wu_apply_plan_limits {#filter-wuapplyplanlimits}

Ба таҳиягарони плагин иҷозат медиҳад, ки маҳдудиятҳоро кӯтоҳмасир кунанд.

Шумо метавонед ин филтрро барои иҷрои ҳар гуна код пеш аз оғоз шудани ҳар кадом аз маҳдудиятҳо истифода баред. Агар филтри шумо ягон арзиши truthy баргардонад, раванд идома меёбад; агар ягон арзиши falsy баргардонад, код бармегардад ва ҳеҷ кадоме аз hook-ҳои зер иҷро намешаванд.
