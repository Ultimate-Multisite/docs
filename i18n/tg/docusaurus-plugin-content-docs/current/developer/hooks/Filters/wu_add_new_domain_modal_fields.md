# Филтр: wu_add_new_domain_modal_fields

Майдонҳоро барои модали илова кардани домени нав филтр мекунад.

Ба addon-ҳо (масалан, Domain Seller) имкон медиҳад, ки майдонҳои бақайдгирии доменро тағйир диҳанд ё иваз кунанд.

## Параметрҳо

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $fields | `array` | Майдонҳои форма. |
