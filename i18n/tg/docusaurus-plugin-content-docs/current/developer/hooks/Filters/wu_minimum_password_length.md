# Филтр: wu_minimum_password_length {#filter-wuminimumpasswordlength}

Танҳо вақте татбиқ мешавад, ки wu_enforce_password_rules true бошад.

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $min_length | `int` | Дарозии ҳадди ақали парол. Пешфарз 12 (бо Defender Pro мувофиқ аст). |
| $defender_active | `bool` | Оё Defender Pro Strong Password фаъол аст. |
