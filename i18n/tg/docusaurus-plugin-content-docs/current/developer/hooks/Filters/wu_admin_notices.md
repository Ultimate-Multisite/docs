# Филтр: wu_admin_notices

Ба таҳиягарон иҷозат медиҳад, ки огоҳиҳои админро, ки аз ҷониби Ultimate Multisite илова шудаанд, филтр кунанд.

## Параметрҳо

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $notices | `array` | Рӯйхати огоҳиҳо барои он панели мушаххас. |
| $all_notices | `array` | Рӯйхати огоҳиҳои иловашуда, аз рӯйи панел ҷудошуда. |
| $panel | `string` | Панел барои гирифтани огоҳиҳо. |
| $filter | `string` | Агар огоҳиҳои пӯшидашаванда филтр шуда бошанд. |
| $dismissed_messages | `array` | Рӯйхати калидҳои огоҳии пӯшидашуда. |

### Аз версияи

- 2.0.0
### Манбаъ

Дар [`inc/class-admin-notices.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-admin-notices.php#L121) дар сатри 121 муайян шудааст
