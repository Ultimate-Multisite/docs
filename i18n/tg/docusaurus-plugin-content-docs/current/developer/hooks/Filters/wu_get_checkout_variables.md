# Филтр: wu_get_checkout_variables {#filter-wugetcheckoutvariables}

Ба таҳиягарони plugin иҷозат медиҳад, ки pre-set-ҳои саҳифаи checkout-ро филтр кунанд.

Эҳтиёт бошед, key-ҳои гумшуда метавонанд checkout-ро дар front-end пурра вайрон кунанд.

## Параметрҳо {#parameters}

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $variables | `array` | Тағйирёбандаҳои маҳалисозишуда. |
| $checkout | `\Checkout` | class-и checkout. |

### Аз версияи {#since}

- 2.0.0
### Манбаъ {#source}

Дар [`inc/checkout/class-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout.php#L1970) дар сатри 1970 муайян шудааст


## Бармегардонад {#returns}
Массиви нави тағйирёбандаҳо.
