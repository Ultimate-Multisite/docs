# Филтр: wp_ultimo_render_vars {#filter-wpultimorendervars}

Ба таҳиягарони plugin иҷозат медиҳад, ки ба контексти render ба таври умумӣ тағйирёбандаи иловагӣ илова кунанд.

## Параметрҳо {#parameters}

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $args | `array` | Массиве, ки тағйирёбандаҳои аз ҷониби фарохонии render гузаронидашударо дар бар мегирад. |
| $view | `string` | Номи намуде, ки бояд render шавад. |
| $default_view | `string` | Номи fallback_view |

### Аз версияи {#since}

- 2.0.0
### Манбаъ {#source}

Дар [`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L31) дар сатри 31 муайян шудааст
