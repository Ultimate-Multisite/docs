# Филтр: wu_country_get_cities {#filter-wucountrygetcities}

Рӯйхати шаҳрҳоро барои як иёлат дар як кишвар бармегардонад.

## Параметрҳо {#parameters}

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $cities | `array` | Рӯйхати номҳои шаҳрҳои иёлат. Калидҳо вуҷуд надоранд. |
| $country_code | `string` | Рамзи дуҳарфии ISO барои кишвар. |
| $state_code | `string` | Рамзи дуҳарфии ISO барои иёлат. |
| $current_country | `\WP_Ultimo\Country\Country` | Намунаи синфи ҷорӣ. |

### Аз версияи {#since}

- 2.0.11
### Манбаъ {#source}

Дар [`inc/country/class-country.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/country/class-country.php#L146) дар сатри 146 муайян шудааст


## Бармегардонад {#returns}
Рӯйхати филтршудаи иёлатҳо.
