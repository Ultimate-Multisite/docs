# Филтр: get_site_url_for_previewer

Ба таҳиягарони plugin иҷозат медиҳад, ки URL-и истифодашаванда дар previewer-ро филтр кунанд

## Параметрҳо {#parameters}

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $domain | `string` | Домени пешфарз, ки ҳоло истифода мешавад; барои тағйирдиҳиҳо муфид аст |
| $domain_options | `array` | Рӯйхати ҳамаи имконоти домен, ки дар Ultimate Multisite Settings -&gt; Network Settings -&gt; Domain Options ворид шудаанд |

### Аз замон {#since}

- 1.7.2
### Манбаъ {#source}

Дар [`inc/checkout/class-legacy-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-legacy-checkout.php#L812) дар сатри 812 муайян шудааст


## Бармегардонад {#returns}
Домени нав барои истифода
