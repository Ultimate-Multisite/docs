# Филтр: wu_upgrade_to_unlock_url

Ба таҳиягарон иҷозат медиҳад, ки URL-и гузариш барои кушоданро тағйир диҳанд

## Параметрҳо {#parameters}

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $url | `string` | URL-и гузариш. |
| $args | `array` | Модул ва навъи маҳдудияте, ки ба гузариш ниёз дорад. |
