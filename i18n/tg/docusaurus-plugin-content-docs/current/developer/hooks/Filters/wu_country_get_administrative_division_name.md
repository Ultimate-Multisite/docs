# Filter: wu_country_get_administrative_division_name

Номи зебои зербахшҳои маъмурии кишварро бармегардонад.

## Параметрҳо

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $name | `string` | Номи бахш. Одатан чизе монанди иёлат, вилоят, минтақа ва ғайра. |
| $country_code | `string` | Коди дуҳарфаи ISO барои кишвар. |
| $state_code | `string` | Коди дуҳарфаи ISO барои иёлат. |
| $ucwords | `bool` | агар мо калимаҳоро бо ҳарфҳои калон нависем. |
| $current_country | `\WP_Ultimo\Country\Country` | Намунаи синфи ҷорӣ. |
