# Амал: wu_page_load

Ба таҳиягарони плагин иҷозат медиҳад, ки ба саҳифаҳои мо hook-ҳои иловагӣ илова кунанд.

## Параметрҳо {#parameters}

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $id | `string` | ID-и ин саҳифа. |
| $page_hook | `string` | Hook-и саҳифаи ин саҳифа. |
| $admin_page | `self` | Намунаи саҳифа. |

### Аз версия {#since}

- 1.8.2
- 2.0.4: Параметри сеюм илова шуд: намунаи саҳифа.
