# Амал: wu_handle_customer_meta_fields

Ба таҳиягарони plugin иҷозат медиҳад, ки агар лозим бошад, meta data-ро бо роҳҳои гуногун захира кунанд.

## Параметрҳо {#parameters}

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $meta_repository | `array` | Рӯйхати meta fields, бо сохтори key =&gt; value. |
| $customer | `\Customer` | Объекти муштарии Ultimate Multisite. |
| $checkout | `\Checkout` | Синфи пардохт. |

### Аз версияи {#since}

- 2.0.0
