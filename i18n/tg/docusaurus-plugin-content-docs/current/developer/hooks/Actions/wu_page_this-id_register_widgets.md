# Action: wu_page_\{$this->id\}_register_widgets

Пас аз сабти виджетҳо барои ин саҳифа иҷро мешавад.

The dynamic portion of the hook name, `$this-&gt;id`, refers to the page id.

## Параметрҳо

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $id | `string` | id-и саҳифа. |
| $page_hook | `string` | hook-и саҳифа. |
| $page | `object` | объекти саҳифа. |
