# Амал: wu_model_post_save {#action-wumodelpostsave}

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $model | `string` | Слаги модел. |
| $data | `array` | Додаҳои объект, ки нигоҳ дошта мешаванд, сериализатсияшуда. |
| $data_unserialized | `array` | Додаҳои объект, ки нигоҳ дошта мешаванд. |
| $this | `\Base_Model` | Намунаи объект. |
