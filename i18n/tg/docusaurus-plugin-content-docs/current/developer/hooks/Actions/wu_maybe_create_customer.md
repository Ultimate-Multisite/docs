# Амал: wu_maybe_create_customer

Ба таҳиягарони plugin иҷозат медиҳад, ки ҳангоми илова шудани муштарӣ корҳои иловагӣ анҷом диҳанд.

Масалан, ин ҷо мо hook-ҳоро барои илова кардани customer-&gt;user ба сайти асосӣ низ илова мекунем.

## Параметрҳо

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $customer | `\Customer` | Муштарие, ки эҳтимол сохта шудааст. |
| $checkout | `\Checkout` | Синфи checkout-и ҷорӣ. |

### Аз версияи

- 2.0.0
### Манбаъ

Дар [`inc/checkout/class-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout.php#L1156) дар сатри 1156 муайян шудааст
