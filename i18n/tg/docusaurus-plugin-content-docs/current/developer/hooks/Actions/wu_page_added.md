# Амал: wu_page_added {#action-wupageadded}

Ба таҳиягарони plugin иҷозат медиҳад, ки ҳангоми сабти саҳифаҳо корҳои иловагӣ иҷро кунанд.

Бар хилофи wu_page_load, ки танҳо вақте иҷро мешавад, ки саҳифаи мушаххас дида мешавад, ин hook ҳангоми сабт барои ҳар як саҳифаи admin, ки бо истифода аз коди Ultimate Multisite илова мешавад, иҷро мегардад.

## Параметрҳо {#parameters}

| Ном | Навъ | Тавсиф |
|------|------|-------------|
| $page_id | `string` | ID-и ин саҳифа. |
| $page_hook | `string` | Номи hook-и ин саҳифа. |

### Аз версияи {#since}

- 2.0.0
