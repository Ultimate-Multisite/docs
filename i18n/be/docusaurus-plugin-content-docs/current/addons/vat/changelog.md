# Журнал змены VAT {#vat-changelog}

Version 1.0.7 - Выпушчана 2026-02-03

* Выпраўка: Номеры VAT для Германіі (DE) не праходзілі валідацыю з-за памылак MS_MAX_CONCURRENT_REQ ў сэрвісе VIES SOAP. Мы пераключыліся з старачнага SOAP API на афіцыйны EU VIES REST API для больш надзейнай валідацыі.
* Выпраўка: Нават калі сэрвіс VIES на некаторы час недоступны, номеры VAT цяпер прымаюцца, а не адхіляюцца памылка. Будзе створана запіс у журнале для пазнейнага прагляду.
* Выпраўка: Пале VAT/Ідэнтіфікатар падатку цяпер бачна для ўсіх краін, а не толькі для членаў ЕС. Падатковыя ідэнтіфікатары, якія не ўваходзяць у ЕС (напрыклад, швейцарскія номеры CHE), захоўваюцца для адбывання ў рахунку-фактуры без валідацыі VIES.
* Выпраўка: Пашук прафікса VAT для Грэцыі (GR), Манако (MC) і Акланда (IM) быў зламаны з-за невернай лагікі пошуку ў масыві.
* Удосканаленне: Выбыў непазбежны dev-забіч wp-cli-bundle, які мог выклікаць фатальных памылак пры выкарыстанні разам з аддаткам Site Exporter.
* Удосканаленне: Дададзены паўнавартасны набор юніт-тэставання (53 тэсты).

Version 1.0.6 - Выпушчана 2026-01-25

* Выпраўка: Тураенныя B2B-транзакцыі цяпер правільна адлічваюць VAT. Згодна з падатковым правядзеннем ЕС, механізм "адверхнега падатку" (reverse charge) распаколькі ўжываецца толькі для памежавых B2B-транзакцый, а не калі краіна кліента збіваецца з краінай кампаніі.

Version 1.0.5 - Выпушчана 2026-01-22

* Выпраўка: Аддатак не загружаецца або не працуе належным чынам.
* Змена: Пачатклік VAT-ставак пераключыўся з euvatrates.com на рэпазітарый ibericode/vat-rates для больш надзейных і актыўна падтрымліваных дадзеных.
* Выпраўка: Выправлена памылка ў значэнні опцыі super_reduced_rates.
* Удосканаленне: Дададзены праверка на нуль (null checks) для апрацоўкі кода краіны, каб zapрэварыць памылкі.
* Змена: Выбылі ўпакоўкі перакладу ў годзе аўтаматычнага абнаўлення праз Traduttore.

Version: 1.0.3 - Выпушчана 2025-09-28

* Змена: Паўторнае імя (prefix) змянена на ultimate-multisite; абноўлены тэкстовы домен; павышэнне версіі.

Version 1.0.0-beta.4 - 2021-09-24

* Дададзены: фільтр wp_ultimo_skip_network_active_check для наладжання, заснаваных на mu-plugins;
