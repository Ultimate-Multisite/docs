# Журнал змены GoCardless {#gocardless-changelog}

Версія: 1.0.3 - Выпушчана 2026-05-05
* Выправілі: Сіты ў стані чакання, якія засталіся (orphaned pending sites), цяпер вычышчаюцца, калі кансалюецца абонентскасць GoCardless, калі яна знаходзіцца ў стані чакання або ўтрымкі.
* Удосканалена: Плагін-аўталоадэра прапускаецца, калі аўталоадэра Bedrock root ўжо загрузіў залежнасці.

Версія: 1.0.1 - Выпушчана 2025-09-28

* Зменены прэфікс на ultimate-multisite; абноўлены тэкстовы домен; павышэнне версіі.

2021-09-24 - Версія 1.0.0-beta.4

* Дададзена: фільтр wp_ultimo_skip_network_active_check для наладжання, заснаваных на mu-plugins;

2021-06-16 - Версія 1.0.0-beta.3

* Выправілі: Апрацоўвальнікі webhook-аўповедамленняў, якія не маглі пацвердзіць падпісаванні (signatures);

2021-06-14 - Версія 1.0.0-beta.2

* Удосканалена: Публікацыя сайту пасля стварэння мандата GoCardless;

Версія 1.0.0-beta.1 - Пачатковая версія
