# Журнал змены AffiliateWP

Версія: 2.0.1 - Выпушчана 2025-09-28

* Змена прафікса на ultimate-multisite; абнаўленне тэкстовай сферы; павышэнне нумара версіі.

Версія: 2.0.0 - Выпушчана 2022-08-05

* Перабудова плагіна для працы ў аснове ultimo payments

Версія: 2.0.0-beta.2 - Выпушчана 2022-01-21

* Унутранае: Дададзены генератар хукаў і фільтраў;
* Унутранае: Дададзены стэб Multisite Ultimate для паляпшэння працы разработнікаў;
* Выправлена: Катастрафічная памылка, калі на канкрэтным кошыку адсутні купон зніжкі;
* Выправлена: Наведванні, якія не канвертуюцца належным чынам;

Версія 2.0.0-beta.1 - Выпушчана 2021-12-29

* Пачатковае выпусканне версіі, аснованай на 2.0;
