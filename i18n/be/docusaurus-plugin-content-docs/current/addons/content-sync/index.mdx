# Сінхронizacja кантэнту

Multisite Ultimate: Сінхронizacja кантэнту

Гэты аддатак для Multisite Ultimate дазваляе адміністратарам сеткі синхронізаваць кантэнт з аднаго сайта на іншыя сайты ў сетцы. Ён падтрымлівае ўсе тыпы пасты, уключаючы пасты, страўнцы, асобныя тыпы пасты і шаблоны Elementor. Аддатак уключае пакрокавы візард-інтерфейс для выбару кантэнту і тэлесных сайтаў, з папартійнай прапрацоўкай і зваротнай сувязясцю пра прагрэс у рэальным часе.

## Функцыі

* **Выбар кантэнту:** Візуальная сетка картак з мінітурамі і фільтрацыяй па тыпах пасты.
* **Усе тыпы пасты:** Синхронizacja пастов, страўнцый, прадуктаў, шаблонаў Elementor і любых асобных тыпаў пасты.
* **Выбар сайтаў:** Выбар сайтаў, на якія трэба синхронізаваць кантэнт, з выключэннем сайта-источніка і папярэднім выбарам пад-сайтаў.
* **Папартійная прапрацоўка:** Синхронizacja на 10 сайтаў адначасова з абнаўленнем прагрэсу.
* **Сінхронizacja мультимедыя:** Аўтаматычна капіруе і пераадрэгёўвае мультимедыяфайлы (карціны, відэа) з кантэнту пастов на тэлесны сайты.
* **Сінхронizacja таксономіі:** Захаваўвае катэгорыі, тэгі і асобныя таксономіі.
* **Падтрымка Elementor:** Поўная падтрымка шаблонаў Elementor з рэгенерацыяй CSS і пераадрэгёўваннем ID шаблону.
* **Выпрацоўка канфліктаў:** Варыясць замясціць існуючы кантэнт або праскочыць яго.
* **Кіраванне кэшам:** Аўтаматычна очишчае кэш Elementor на тэлесных сайтах пасля сінхроннізацыі.

## Патрабаванні

* WordPress 5.3 або больш высокі
* PHP 7.4 або больш высокі
* Плагін Multisite Ultimate (актыўны)
* Плагін Elementor (неабязавальна, але патрабаваны для сінхронізацыі шаблонаў Elementor)

## Устаноўка

1. Загрузіце 'ultimate-multisite-content-sync' у каталог '/wp-content/plugins/'
2. Актывуйце плагін праз меню 'Plugins' у адмініструванні вашай сеткі WordPress.
3. Наладзьце параметры у Ultimate Multisite → Settings → Content Sync
4. Пходзьце па шляху Ultimate Multisite → Content Sync, каб пачаць сінхронizaваць.
