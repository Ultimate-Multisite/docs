# Filter: wu_apc_should_display_admin_menu

Дазваляе адміністратарым схаваць страніцы адмін-меню, калі яны завялічылі ўсе наладкі кастымізацыі.

## Параметры

| Назва | Тып | Апісан |
|------|------|-------------|
| $should_display_admin_menu | `bool` | Калі нам трэба адбыць страніцы. |
| $network_active | `bool` | Праверка, ці актыўны адмін-панель плагіна на ўзняч-уровні. |
