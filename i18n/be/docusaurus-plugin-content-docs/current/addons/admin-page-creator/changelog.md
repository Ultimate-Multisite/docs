# Admin Page Creator Changelog

### Version 1.9.0 - Released on 2026-01-18 {#version-190---released-on-2026-01-18}

* Выправлена: Катастрафічныя памылкі пры актывацыі.
* Рэфактарызавана: Класы вытатня адночнасці з крынікам кантэнту, каб выкарыстоўваць Singleton trait з належнай ініцыялізацыяй роўных конструктараў.
* Выправлена: Памылкі кампіляцыі шаблону Vue.js, якія мечылі селектар тыпу кантэнту.
* Выправлена: Адсутнасць ў дадзеных Vue (template_id, external_link_url_embedable, external_link_url_checking).
* Выправлена: Рэдатар TinyMCE не ініцыялізуецца належным чынам пры загрузцы старонкі.
* Выправлена: Файл падтрымкі Divi не загружаецца аўтаматычна праз Composer.
* Выправлена: Пашкодзваны PHP-сінтакс у класе падтрымкі Divi.
* Улепшана: Арганізацыя і мініфікацыя JavaScript-актываў.
* Улепшана: Сумяшчальнасць з найлепшымі версіямі WordPress і генератараў старонкі.
