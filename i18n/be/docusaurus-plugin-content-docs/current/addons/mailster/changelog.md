# Журнал змены Mailster {#mailster-changelog}

### 1.0.0 {#100}
* Первы выпусканне
* Аўтаматычная синхронізацыя падпіскаў пры рэгістрацыі або плацменце
* Пазначаныя спісы для канкрэтных прадуктаў
* Падпіска праз праверку (checkbox) для адпаведнасці GDPR
* Падтрымка двойнай падпіскі (Double opt-in)
* Агульнае мапіяванне паляў
* Падрабяжная фіксацыя (журналуванне)
