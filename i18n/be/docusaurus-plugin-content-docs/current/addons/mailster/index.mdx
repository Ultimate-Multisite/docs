# Mailster

Ператварыце свой multisite-сетвар у магутную машыну па збору адраслаў электроннай пошты з аддонкам Ultimate Multisite Mailster Integration. Аўтаматычна дадавайце новых кліентаў да спісаў Mailster, калі яны рэгіструюцца або робяць пакупкі, ствараючы адным крокам злучэнне паміж вашай платформой для канчатковых карыстальнікаў і стратэгіей маркетынгу па электроннай пошце.

## Асноўныя функцыі {#key-features}

**Аўтаматычная синхронізацыя адраслаў**
* Аўтаматычна дадае кліентаў у спісы Mailster, калі яны рэгіструюцца
* Выбірайце, адсинхронізаваць адразу па рэгістрацыі ці пачакаць пацвердzenia плацёжа
* Патрымлівайце свае спісы электроннай пошты свежымі і актуальнымі без ручных дзеянняў

**Гнуткае кіраванне спісамі**
* Устаноўце галаўныя спісы па спілканню для ўсіх новых кліентаў
* Пазбегаце галаўных спісаў, выкарыстоўваючы спісы, спецыфічныя для прадукту, для мэтанакіраваных кампаній
* Легальна кіруйце прызначаннем спісаў у знаёмым інтэрфейсе Mailster

**Варыянты падпіскі, адпаведныя GDPR**
* Выбірайце паміж аўтаматычнай падпіскай або згодай праз праверку (checkbox)
* Актывуйце двойную падпіску (double opt-in) для пацверджаных падпісаў
* Паказвайце/схаваляйце праверку падпіскі ў залежнасці ад вашага патрабавання адпаведнасці

**Разумны мапінг дадзеных**
* Аўтаматычна синхронізацыя іменаў кліентаў і кантактнай інфармацыі
* Мапінг адрасыў выстаўвання рахунку на поля падпісателя
* Адлік даты рэгістрацыі для мэтанакіраваных кампаній

**Кантроль на ўзроўні прадукту**
* Прызначайце розныя спісы для розных прадуктаў
* Стварайце мэтанакіраваныя кампаніі на аснове таго, што пакупаюць кліенты
* Выключайце Mailster для пэўных прадуктаў, калі гэта неабходна

**Падлецкі досвед**
* Паўнавартасная інтэграцыя UI Mailster для выбару спісаў
* Падрабяжны журналіраванне для выяўлення і праверкі праблем
* Беспяднае абнаўленне існуючых падпісаў

## Ідэальна падыходзіць для {#perfect-for}

* Уласнікаў сайтаў-членаў, якія ствараюць спісы электроннай пошты
* Платформ SaaS, якія жадаюць пакіраваць новых кліентаў
* Twіракоў каурсаў, якія сегментуюць студэнтаў па рэгістрацыі
* Каго і аднога, хто выкарыстоўвае Ultimate Multisite з Mailster

## Як гэта працуе {#how-it-works}

1. Усталюйце і актывуйце аддонак на сетвар Ultimate Multisite
2. Наладзьце галаўныя спісы па спілканню ў настройках Ultimate Multisite
3. Па важнасці, перазбегайце галаўныя спісы спісамі, спецыфічнымі для прадукту
4. Выберыце рэжым падпіскі і час падпіскі
5. Гэтай жайце, як вашы спісы Mailster аўтаматычна растуць!

## Устаноўка {#installation}

1. Загрузіце файлы плагіна ў каталог аддонкаў Ultimate Multisite
2. Убедзіцеся, што Mailster ўсталяваны і актыўны на вашым галоўным сайце
3. Актывуйце аддонак праз адкаранне аддонкаў Ultimate Multisite
4. Прыходзьце на Settings > Mailster Integration для канфігурацыі
5. Выберыце галаўныя спісы і наладзьце перавагі падпіскі

## Часта задаваныя пытанні {#frequently-asked-questions}

## Ці працуе гэты аддонак без Ultimate Multisite? {#does-this-addon-work-without-ultimate-multisite}

Нятучна, гэты аддонак патрабуе Ultimate Multisite для працы. Ён спецыфікачна створаны для інтэграцыі Mailster з сістэмаю кліентаў і прадуктаў Ultimate Multisite.

## Ці патрабую я Mailster, усталяванага на ўсіх сайтах? {#do-i-need-mailster-installed-on-all-sites}

Нятучна, Mailster патрабуе толькі ўсталявання і актывацыі на вашым галоўным сетварным сайце. Аддонак аўтаматычна злучыцца з ім з усіх субсайтаў.

## Ці магу я прызначаць розныя спісы для розных прадуктаў? {#can-i-assign-different-lists-to-different-products}

Так! Вы можаце ўсталяваць галаўныя спісы па спілканню, а потым перазбегаць іх на ўзроўні кожнага прадукту. Гэта дазваляе вам ствараць вельмі мэтанакіраваныя кампаніі па электроннай пошце на аснове таго, што пакупаюць кліенты.

## Што стане гады, калі кліент ужо існую ў Mailster? {#what-happens-if-a-customer-already-exists-in-mailster}

Па спілканню, аддонак абнавіць існуючых падпісаў і дадасць іх да новых спісаў. Вы можаце выключыць гэта ў настройках, калі перакупіце памынуць існуючых падпісаў.

## Ці адпавядае гэта GDPR? {#is-this-gdpr-compliant}

Так! Аддонак уключае рэжым падпіскі праз праверку (checkbox), дзе кліенты павінны адкрыта дазволіць атрымліваць электронную пашту. Гэта адпавядае патрабаванням GDPR у спалучэнні з належнай палітыкай прыватнасці.

## Ці магу я выкарыстоўваць двойную падпіску (double opt-in)? {#can-i-use-double-opt-in}

Абсалютны гарант! Актывуйце настаўленне двойнай падпіскі, і Mailster адшлюшыць пацвердныя паведамленні па электроннай пошце, перш чым актываваць падпіскі.
