Version 2.0.1 - Выпушчана 2023-08-09

* Дададзена: Сумяшчальнасць з Woocommerce 7.9.0.
* Дададзена: Сумяшчальнасць з Woocommerce Subscriptions 5.3.0.
* Дададзена: Падтрымка абнаўлення членства.
* Дададзена: Паведамленні пра траіалы і плату за наладку ў Woocommerce.
* Дададзена: Ідэнтыфікацыя прадуктаў Ultimate Multisite Woocommerce па мета-значэнні.
* Дададзена: Устаўлена адрачыннае выпраўленне для адзначэння ўсіх прадуктаў Woocommerce, звязаных з Ultimate Multisite.
* Дададзена: Вылучаны прадукты, створаныя Ultimate Multisite, з спісу Woocommerce.
* Удосканаленне: Ствараны непаўторны купон Woocommerce для прымянення да кошыка.
* Удосканаленне: Адпраўлена паўторнае адзначэнне на прадукт Woocommerce.
* Удосканаленне: Дададзена адзначэнне паўторнага адзначэння на прадукт Woocommerce.
* Удосканаленне: Забеспячана тып прадукту ў checkout.
* Выпраўка: Ахаванне статусу членства падчас працэсу спадкамлення.
* Выпраўка: Праверана, ці існуюць падпіскі, каб пазбегнуць памылак падчас працэсу скасавання.
* Выпраўка: Дададзена дата пачатку падпіскі для выкарыстання ў падпісках Woocommerce.
* Внутранае: Выкарыстана выбудоўвальны працэс PHP 8.1.
