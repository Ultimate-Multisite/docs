# Журнал змен інтэграцыі WooCommerce {#woocommerce-integration-changelog}

Версія 2.2.0 - Выпушчана 2026-07-01
* Новае: сумы падаткаў Ultimate Multisite цяпер паказваюцца як асобныя радкі збораў WooCommerce пры афармленні замовы, што робіць агульныя сумы падаткаў больш зразумелымі перад аплатай.
* Новае: дададзена неабавязковая налада "Неадкладна прыпыняць сяброўствы пры няўдалых падаўжэннях" і фільтр `wu_woo_suspend_on_payment_failure` для сайтаў, якія хочуць прыпыняць сяброўства пры няўдалым падаўжэнні падчас акна паўторных спроб WooCommerce Subscriptions.
* Выпраўленне: узгоднены статусы WooCommerce Subscription, якія маглі заставацца несінхранізаванымі з сяброўствамі Ultimate Multisite пасля няўдалых або адноўленых падаўжэнняў.
* Выпраўленне: валюта крамы WooCommerce дададзена ў спіс валют Ultimate Multisite, калі яе там не было.
* Выпраўленне: захаваны плацежныя звесткі кліента пры перанакіраванні падпісчыкаў на афармленне замовы WooCommerce.
* Палепшана: дададзена сумяшчальнасць з Jetpack Autoloader 5.
* Палепшана: ачышчана генерацыя пакета выпуску, каб zip-архівы GitHub і маркетплэйса пазбягалі ўкладзеных прамежкавых каталогаў і файлаў распрацоўкі.

Версія 2.0.6 - Выпушчана 2026-01-16
* Палепшанне: уключаны асноўныя падпіскі ў дадатак. Больш не патрабуе пашырэння Woocommerce Subscriptinos.

Версія 2.0.5 - Выпушчана 2026-01-09
* Палепшанне: загрузка перакладаў з glotpress API.
* Выпраўленне: фатальная памылка ў некаторых канструктарах старонак.
* Выпраўленне: бясконцае перанакіраванне, калі кліент з'яўляецца ўдзельнікам галоўнага сайта.

Версія 2.0.4 - Выпушчана 2025-11-14
* Дададзена: пераклады для значна большай колькасці моў.
* Зменена: перайменавана ў Ultimate Multisite: Woocommerce Integration.
* Дададзена: сумяшчальнасць з Woocommerce 10.2.1.
* Дададзена: сумяшчальнасць з Woocommerce Subscriptions 7.7.0.
* Выпраўленне: сумяшчальнасць з PHP 8.4
* Выпраўленне: перанакіраванне, калі не існуе старонкі WC account.

Версія 2.0.3 - Выпушчана 2025-08-13
* Зменена: уключаны аўтаматычныя абнаўленні з новым маркетплэйсам.

Версія 2.0.2 - Выпушчана 2025-07-05
* Зменена: перайменавана ў Multisite Ultimate: Woocommerce Integration.
* Дададзена: сумяшчальнасць з Woocommerce 9.8.1.
* Дададзена: сумяшчальнасць з Woocommerce Subscriptions 7.3.0.
* Выпраўленне: скасаванне падпіскі кліентам.
* Выпраўленне: фатальная памылка пры выкарыстанні блока афармлення замовы.
* Палепшанне: цяпер сумяшчальна з высокапрадукцыйнымі карыстальніцкімі табліцамі заказаў Woocommerce.
* Выпраўленне: скасаванне пры афармленні замовы WooCommerce усё яшчэ можа павысіць узровень сяброўства.

Версія 2.0.1 - Выпушчана 2023-08-09

* Дададзена: сумяшчальнасць з Woocommerce 7.9.0.
* Дададзена: сумяшчальнасць з Woocommerce Subscriptions 5.3.0.
* Дададзена: падтрымка абнаўленняў сяброўства.
* Дададзена: паведамленні пра пробныя перыяды і платы за наладжванне ў Woocommerce.
* Дададзена: ідэнтыфікацыя прадуктаў Ultimate Multisite Woocommerce праз мета-значэнне.
* Дададзена: устаўлена аднаразовае выпраўленне, каб пазначыць усе прадукты Woocommerce, звязаныя з Ultimate Multisite.
* Дададзена: прадукты, створаныя Ultimate Multisite, выдалены са спіса Woocommerce.
* Палепшанне: створана непаўторная зніжка Woocommerce для прымянення да кошыка.
* Палепшанне: адноўлена паўторная зніжка для прадукту Woocommerce.
* Палепшанне: дададзена метка паўторнай зніжкі да прадукту Woocommerce.
* Палепшанне: забяспечаны тып прадукту пры афармленні замовы.
* Выпраўленне: захаваны статус сяброўства падчас працэсу паніжэння.
* Выпраўленне: праверана, ці існуе падпіска, каб пазбегнуць памылак падчас працэсу скасавання.
* Выпраўленне: дададзена дата пачатку падпіскі для выкарыстання ў падпісках Woocommerce.
* Унутранае: рэалізаваны новы працэс зборкі PHP 8.1.
