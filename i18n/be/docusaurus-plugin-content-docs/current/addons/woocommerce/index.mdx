# Інтеграцыя з WooCommerce

Інтеграцыя з WooCommerce працуе вельмі падобна да таго, як працуе наш Manual Gateway. Яна прапануе шмат гнуткасці для карыстальнікаў, дазвольваючы ім выбраць любой з метадаў аплатам, доступных у вашай ўстаноўцы WooCommerce. І ўсё проста: калі білінгавы перыяд карыстальніка скінчыцца, падпіска карыстальніка будзе ўстаноўлена на пазір. У той жа час будзе створана замоўленне ў ўстаноўцы WooCommerce на галоўнай сайце сеткі. Пасля гэтага карыстальнік атрымлівае рахунак-фактуру на свой электронны адрас, з спасылкай для аплаты. Калі аплата атрымліваецца, падпіска пановляецца.

## Функцыі

- Легкая інтэграцыя паміж Ultimate Multisite і WooCommerce
- Несколько метадаў аплаты
- Абсталяваны аўтаматычны працэс з WooCommerce Subscriptions

## Дакументацыя

**Наладка інтэграцыі з WooCommerce**

**Пашырэнне вашага выбару аплатаў з Ultimate Multisite: Інтеграцыя з WooCommerce**

## Устаноўка

1. Загрузіце 'ultimate-multisite-woocommerce' у каталог '/wp-content/plugins/'
2. Актывуйце плагін праз меню 'Plugins' у WordPress
3. Выкончыце крок за крокам Паказальнік для налады плагіна

## Паўдарожжа па выкарыстанні

_**ВАЖНАЕ ПАМЫНКА: Гэты артыкул адносіцца да версіі Ultimate Multisite 2.x.**_

_**УВАГА:** Ultimate Multisite: WooCommerce Integration патрабуе, каб WooCommerce быў актываваны, як мінімум, на вашым галоўным сайце._

Мы ведаем, што _Stripe_ і _PayPal_ недаступныя ў некаторых краінах, што абмяжоўвае або перашкаджае карыстальнікам Ultimate Multisite эфектыўна выкарыстоўваць наш плагін. Таму мы стварылі **додатак для інтэграцыі _WooCommerce_**.

_WooCommerce_ — вельмі папулярны плагін для электроннай търгоўні. Разработнікі па ўсім свеце стварылі дадаткі для інтэграцыі розных шлюзов аплаты ў яго. Мы выкарыстовалі гэта, каб пашырыць шлюзы аплаты, якія вы можаце выкарыстоўваць у сістэме білінгу Ultimate Multisite.

## Наладка

Сперш вам трэба ўсталяваць дадатак WooCommerce Integration, які **свабодны для ліцэнзіяваных карыстальнікаў**. Каб яго ўсталяваць, націсніце на Ultimate Multisite ў вашым супер-адмінскім дашбордзе і выберыце **Add-ons**.

![Add-ons page](/img/admin/addons-page.png)

Тут вы можаце знайсці ўсе аддатки Ultimate Multisite. Націсніце на дадатак **Ultimate Multisite: WooCommerce Integration**.

![Add-ons page showing WooCommerce Integration](/img/admin/addons-page.png)

Адчыніцца акно з дэталямі дадатку. Просто націсніце на **Install Now** *.*

<!-- Screenshot unavailable: WooCommerce Integration addon install popup with Install Now button -->

Пасля ўстаноўкі вы будзеце перакінуты на старонку плагінаў. Тут проста націсніце на **Network Activate**, і дадатак _WooCommerce_ будзе актываваны на вашай сетцы.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for WooCommerce Integration -->

Пасля актывацыі, калі ў вас яшчэ не ўсталяваны і не актываваны плагін WooCommerce на вашым сайце, **вы атрымаеце нагадванне**.

<!-- Screenshot unavailable: Admin notice reminding to install and activate WooCommerce plugin -->

Пасля ўсталявання вы ўбачыце адпаведны варыянт у настройках аплаты Ultimate Multisite. Пасля гэтага вы можаце **актываваць** варыянт _WooCommerce_ з адтуды.

![Payment gateway settings](/img/config/settings-payment-gateways.png)

Вы можаце накіраваць некаторыя базавыя налады, такія як адказны для адбывання на фронт-энды і інструкцыі для паказа карыстальнікам падчас аплаты. Пасля гэтага вы можаце прайсці на **налады _WooCommerce_ вашага галоўнага сайту**, каб актываваць і накіраваць шлюз аплаты, які вы аддаеце перавагу.

## Як кіраваць аплатай

Інтеграцыя з WooCommerce працуе вельмі падобна да таго, як працуе наш **Manual Gateway**. Недалік у тым, што кожны плацёж павінен быць аплачаны ўручную па канцы білінгавай перыяду. Вось як працуе гэты працэс:

  * Калі білінгавы цыкл карыстальніка скінчыцца, кандыдатства ўстаноўваецца **на пазір**. У той жа час будзе створана замоўленне ў ўстаноўцы _WooCommerce_ на галоўным сайце вашай сеткі.

  * Карыстальнікі атрымліваюць рахунак-фактуру на свой электронны адрас, з спасылкай для аплаты (гэтая спасылка таксама з'яўляецца ў іх гісторыі білінгу з дашборда падсайту). Гэтая спасылка перанакіроўвае карыстальніка на **checkout** _**WooCommerce**_, дзе карыстальнікі змогуць выбраць адзін з **даступных шлюзов _WooCommerce_** для праходжання аплаты.

  * Калі аплата атрымліваецца, кандыдатства **пановляецца**, а плацёж фігуруе ў Ultimate Multisite, і замоўленне ўстаноўваецца як заканчэнне.

### Некаторыя важныя пазначкі:

  * Пачатковая функцыянальнасць Ultimate Multisite ўсё яшчэ працуе з гэтай інтэграцыяй. Напрыклад, **выкарыстанне паваротнай аплаты** з экрана **Упраўленне падпіскамі** будзе адпраўляць камунікацыю з WooCommerce для выкарыスタння паваротнай аплаты з выкарыстаннем выбранага шлюзу аплаты.

  * Ultimate Multisite: WooCommerce Integration залежыць ад WP-Cron для працы. Гэта вельмі добра працуе ў большасці выпадкаў, але калі на сайце не шмат трафіку, cron не будзе паведамляць пра праверку запланаваных падзей, і падбыццё будзе прапушчана, што можа выклікаць запаDomainMask працы.

## Плагін Woocommerce Subscriptions

Ручная ініцыятыва аплаты кожны месяц не ідэальна для карыстальнікаў, бо падпіска павінна аўтаматычна пановляцца кожны білінгавы цыкл. Таму мы распрацавалі дадатак WooCommerce Integration, каб **падтрымліваць плагін WooCommerce Subscriptions**. Гэта аўтаматызуе аплату, што робіць транзакцыю больш лёгкай для вашага карыстальніка.

Усё, што вам трэба зрабіць, гэта **усталяваць** [плагін Woocommerce Subscriptions](https://woocommerce.com/products/woocommerce-subscriptions/) і каб ён быў **актываваны на вашым галоўным сайце**. Пасля гэтага вярэцеся на налады аплаты Ultimate Multisite і **актывуйце інтэграцыю**.

І гэта ўсё. Вашым карыстальнікам больш не будзе неабходна ўручную праводзіць аплату па канцы білінгавай перыяду.

Акрамя таго, што ў вашай галоўнай сетцы будзе створана замоўленне ў _WooCommerce_, будзе створана таксама падпіска для кожнага канта.
