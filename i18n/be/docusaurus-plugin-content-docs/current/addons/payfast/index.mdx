# PayFast {#payfast}

Multisite Ultimate: Payfast Gateway

Дадае Payfast — хуткі і бяспечны онлайн-плацёжныя шлюз у Південнай Афрыцы — як плацёжную опцыю ў Multisite Ultimate.

## Устаноўка {#installation}

1. Загрузіце 'ultimate-multisite-payfast' у каталог '/wp-content/plugins/'
2. Актывуйце plugin праз меню 'Plugins' у WordPress
3. Выкончыце ўсе крокі Wizard, каб наладзіць plugin

## Usage Guide {#usage-guide}

PayFast — вядомы онлайн-плацёжны працэсар для прадпрыемстваў у Південнай Афрыцы. Ён прызначаны для дапамогі онлайн-торгначальнікам у росненні, гарантуючы беспарэрыўныя і бяспечныя транзакцыі. Па веданні патрабавання карыстальнікаў Ultimate Multisite, мы прынялі рашэнне стварыць аддон, які ўможліць інтэграцыю з PayFast. У гэтай артыкуле мы пакажам, як наладзіць PayFast ў якасці вашага плацёжнага шлюзу для атрымання плацёжаў у вашай сетцы Ultimate Multisite.

Першы крок — усталяваць аддон.

# How to install the Payfast add-on: {#how-to-install-the-payfast-add-on}

Перайце на старонку аддонаў і выберыце аддон Ultimate Multisite: Payfast Gateway.

![Add-ons page](/img/admin/addons-page.png)

Аддон адчыніцца ў акне. Націсніце на Install Now.

![Add-ons page with install option](/img/admin/addons-page.png)

Пасля ўстаноўкі вас перанаправяць на старонку plugins. Тут проста націсніце на Network Activate, і аддон _Payfast_ будзе актываваны ў вашай сетцы.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# Setting Up PayFast {#setting-up-payfast}

Пасля ўстаноўкі аддону вы можаце прайсці ў налады Ultimate Multisite і пачаць канфігураваць PayFast.

На старонцы **Ultimate Multisite settings** вы знайце тэчку "Payments". У гэтай тэчцы вы павінны ўбачыць "**Payfast** " разам з кнопкай пераключэння (toggle button), каб актываваць або выключыць гэты плацёжны шлюз.

![Payment gateway settings](/img/config/settings-payment-gateways.png)Пайдзеце і націсніце на кнопку пераключэння, каб актываваць яго. Тады гэта ўвеличиць пале, каб адказаць розным поліям, якія вам трэба зарэгістраваць, такія як **Mechant ID, Merchant Key і PassPhrase.** Гэтую інфармацыю вы можаце атрымаць у сваім аккаунце **Payfast**

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Наступны крок — увайсці ў свой аккаунт PayFast, каб атрымаць неабходную інфармацыю.

Upon logging in, you can immediately see the credentials you need from your **Payfast** account dashboard.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->Вы можаце проста skopiраваць і ўставіць ўлігі ў налады Ultimate Multisite. Пасля гэтага націсніце кнопку **"Save settings** ", каб захаваць змены.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

І гэта ўсё, ваша інтэграцыя плацёжаў PayFast завершана!
