# Журнал змены Ultimate AI Connector для сумяшчальных кінтпаінтаў {#ultimate-ai-connector-for-compatible-endpoints-changelog}

## Версія 2.0.0 — Выпушчана 2026-04-24 {#version-200--released-on-2026-04-24}

* Ноўнасць: Падтрымка некалькіх пастаўшчыкоў — магчыма сканфігураваць некалькі AI-кінтпаінтаў і маршрутываць запыты з аўтаматычным рэзерваваннем паміж пастаўшчыкамі.
* Выпраўка: Інтеграцыя SDK для некалькіх пастаўшчыкоў з правільнымі ID пастаўшчыкоў, URL-адрэсам рэгістрацыі і абмежаваннем HTTP-фільтра па пастаўшчыку.
* Выпраўка: Новая картка пастаўшчыка цяпер аўтаматычна разкрываецца пры дадаванні; пазбегуванне кеша сцэнікай пры абноўленні plugin.
* Выпраўка: Дынамічны пацёўка класа пастаўшчыка для eval() і аўтаматычнае разкрыванне для новых дададзеных пастаўшчыкоў.
* Выпраўка: Выкарыстоўваюць стабільныя кампаненты Card/CardBody/CardHeader/CardDivider (ужо не ээкспериментальныя) для сумяшчальнасці з WordPress 6.9+.
* Выпраўка: Замяшчаны невызначаны DragHandle на унікодны іконку хвату для перапарадкавання пастаўшчыкоў змяшчэннем.
* Выпраўка: Улінічана блокуючы HTTP-запыт, які выконваўся пры кожным загружанні старонкі.
* Удосканалена: Працоўныя параграмы GitHub Actions абноўлены да Node.js 24.
