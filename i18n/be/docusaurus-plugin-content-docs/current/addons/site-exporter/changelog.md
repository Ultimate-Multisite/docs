# Журнал змены Site Exporter

Version: 1.0.6 - Released on 2026-05-11
* Новы: Экспорт сайта цяпер пастаўляе ў пакет self-booting index.php, таму ZIP можна ўсталяваць на свежы хаст без асобнай установки plugin.
* Новы: Экспорт сеткі дазваляе адміністратарым выкарыстоўваць увесь падсайты ў адным архіві з адкімнікі Site Export.

Version: 1.0.5 - Released on 2026-05-05
* Выправілі: Дададзена __callStatic да WP_CLI polyfill stub, каб апрацаваць не-mocked метады, такія як get_config, zapрэўшаючы збоі тэставай бутстрапцыі.

Version: 1.0.4 - Released on 2025-11-25

* Выправілі сумяшчальнасць з Ultimate Multisite 2.4.

Version: 1.0.2 - Released on 2025-09-28

* Змянена прафікс на ultimate-multisite; абноўлены тэкстовы домен; павышэнне версіі.

### Version 1.0.1 - Released on 2023-08-09 {#version-101---released-on-2023-08-09}

* Улепшана: Зараз гарантуе атрыманне правільнага шляху файла імпарта.
* Выправілі: Выключаны кеш объекта, каб zapрэўсці памылкі падчас імпарту.
* Выправілі: Выправлена праблема з непрацаваннем спасылкі на загрузку plugin імпарта.

### Version 1.0.0 - Released on 2022-12-23 {#version-100---released-on-2022-12-23}

* Выправілі: Папап-ап формы экспорту не загружаецца;
* Выправілі: Файлы Javascript не загружаецца на старонцы экспорту;
* Унутранае: Новы працэс збіст;

### Version 1.0.0-beta.4 - Released on 2022-01-13 {#version-100-beta4---released-on-2022-01-13}

* Унутранае: Дададзена генерацыя хукаў і фільтраў;
