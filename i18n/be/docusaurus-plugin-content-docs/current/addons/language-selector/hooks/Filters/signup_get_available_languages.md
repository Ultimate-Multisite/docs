# Фільтр: signup_get_available_languages

Фільтруе спіс даступных моў для рэгістрацыі наперарнага сайту.

Пепраўка пустага масыва ў гэты хук выключыць выказванне гэтага напыдання ў форме рэгістрацыі, і будзе выкарыстоўвацца мова па замаўчанні пры стварэнні сайту. Мовы, якія яшчэ не ўсталяваны, будуць вынабраныя.

## Параметры {#parameters}

| Назва | Тып | Апісан |
|------|------|-------------|
| $available_languages | `array` | Даступныя мовы. |
