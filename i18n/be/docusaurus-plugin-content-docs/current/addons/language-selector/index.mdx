# Выбар мовы

Ultimate Multisite: Выбар мовы пры рэгістрацыі

Дадайце поле выбару мовы ў процэс рэгістрацыі.

## Устаноўка {#installation}

1. Загрузіце 'ultimate-multisite-language-selector' у каталог '/wp-content/plugins/'
2. Актывуйце плагін праз меню 'Plugins' у адмініструванні вашай WordPress Network

## Паказальнік выкарыстання {#usage-guide}

З гэтым бясплатным аддаткам вы зможаце даць сваім карыстальнікам магчымасць выбраць, якій мове яны аддаюць перавагу на пачатку рэгістрацыі і на спілкуванні ўлігіну.

Пасля [актывацыі вашай ліцензіі](../../user-guide/getting-started/installing-ultimate-multisite.md) вы зможаце ўсталяваць аддаткі Ultimate Multisite, на якія дае вам доступ ваша ліцэнцыя.

Зайдзіце на адкастрыні налады Ultimate Multisite, а потым, у меню слева, націсніце на тэга _Add-ons_.

![Add-ons page](/img/admin/addons-page.png)

_Зайдзіце на адкастрыню Add-ons, каб пачаць усталяваць Language Selector_

Там вы знайце ўсе аддаткі, якія мы прапануем.

Бясплатныя аддаткі даступныя ўсім нашых кліентам, незалежна ад плана падпіскі, які яны купілі.

Націсніце на кнопку _Install it Now!_ пад мінітурай аддатка. Пасля гэтага актывуйце яго на адкастрыні Plugins.

![Add-ons page with install button](/img/admin/addons-page.png)

_Націсніце на кнопку, каб усталяваць аддатак_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_Актывуйце яго на адкастрыні Plugins_

Цяпер зайдзіце ў меню Налады Ultimate Multisite і ўвесці тэга _Network Settings_. Там, на дне адкастрыні, вы знайце новы раздзел, названы _Languages_.

![Settings page](/img/config/settings-general.png)

_Паявіўся новы раздзел!_

Вы можаце пакінуць яго пустым, каб адказаць усім мовах як магчымасці, або выбраць, якія з іх вы хочаце зрабіць даступнымі. Мовы — гэта тыя, якія вы ўсталявалі ў сваю WordPress-сістэму.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_Спіс моў, даступных у гэтай WP-сістэме_

Пасля захавання ўкіштованняў, якія вы зрабілі, меню выбару з выбраных вами моў будзе адказана на адкастрыні ўлігіну, а таксама ў процэсе рэгістрацыі.

![Login and Registration settings](/img/config/settings-login-registration.png)

## Налады мовы праз URL {#setting-the-language-via-the-url}

Вы таксама можаце змяніць мову адкастрыні, прайшоўшы міну парамэтр локалі ў URL. Калі вы хочаце перакінуцца на адкастрыню рэгістрацыі італьянскай мовы, напрыклад, дадайце ?locale=it_IT у URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Для працы гэтага ў вашай WordPress-сетварцы павінна быць усталявана мова.
