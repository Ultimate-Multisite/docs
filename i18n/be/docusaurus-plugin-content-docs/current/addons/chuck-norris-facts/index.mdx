# Факты чака Норрыса

Дадайце трохі гумару ў свае штодзённыя адміні stratumныя задачы ў WordPress з аддокам **Ultimate Multisite - Chuck Norris Facts**. Пасля актывацыі вы будзеце бачыць выпадковы факт пра чака Норрыса ў верхнім правым кутку кожнай адміні stratumнай старонкі.

Ідэальна, каб павеселіць вашу каманду, палепшыць настaйджэнне падчас доўгіх адміні stratumных сесій, або проста адзначыць легендарную веліч чака Норрыса.

## Прычоны, па якіх факты чака Норрыса? {#why-chuck-norris-facts}

* **Палепшыць настaйджэнне ў камандзе**: Дадайце веселы элемент ў адміні stratumванне вашай сеткі
* **Разбіць лёд**: Выдатны размоваклік для новых членаў каманды
* **Павышэнне прадуктыўнасці**: Шматка смеху можа палепшыць канцэнтрацыю і прадуктыўнасць
* **Нестарэльны гумар**: Факты пра чака Норрыса ніколі не старэюць

## Асноўныя функцыі {#key-features}

**110+ Легендарных фактаў**
Выдатная зборнік класічных жартів і фактаў пра чака Норрыса, якія адзначаюць яго легендарнасць.

**Невыяўляльнае знаходжанне**
Факты з'яўляюцца ў адміні stratumнай бары ў невыяўляльны спосаб, які не перашкаджае вашаму працаваму парачку.

**Гатовы для перакладу**
Усе факты завалены ў функцыі перакладу, што робіць лёгкім локалізацыю для вашай мовы або стварэнне спецыяльных фактаў для вашай каманды.

**Поўная рэагатыўнасць**
Факты правільна знаходзяцца на настільных, планшэтных і мабільных экранах — чака Норрыс выдатны на ўсіх прыладах.

**Падтрымка RTL**
Поўная падтрымка для моў з правага налева, бо чака Норрыс перавышае моўныя бар'еры.

**Сумяшчальны рэдактар блокаў**
Факты інтэлігентна схаваны ў рэдактары блокаў, каб адмовіцца ад канфліктаў інтэрфейсу.

**З акцэнтам на даступнасць**
Уключае належны тэкст для экранных рыдэраў для поўнай адпаведнасці патрабаванням да даступнасці.

## Ідэальна для {#perfect-for}

* Адміні stratumтары сеткі Multisite, якія жадаюць дадаць індывідуальнасці
* Рэалізатары, якія цэняць добры гумар
* Каго попаў патрабуе штодзённую дозу велічы чака Норрыса
* Сайты, якія жадаюць стварыць веселы, паzyтыўны працавальны парадак

## Устаноўка {#installation}

1. Забяспечце, каб плагін Ultimate Multisite быў усталяваны і актываваны ў вашай сетцы WordPress
2. Загрузіце файлы аддока ў каталог `/wp-content/plugins/`
3. Актывуйце плагін на сеткі праз меню 'Plugins' у WordPress
4. Насладжвайцеся выпадковыми фактамі пра чака Норрыса на кожнай адміні stratumнай старонцы!

Не патрабуе канфігурацыі — проста актывуйце і пасмехайцеся!

## Часта задаваныя пытанні {#frequently-asked-questions}

## Ці патрабуе гэта Ultimate Multisite? {#does-this-require-ultimate-multisite}

Хоць распрацаваны як аддок Ultimate Multisite, плагін працуе незалежна і можа працаваць на любой устаноўцы WordPress.

## Ці магу я дадаць свой уласны факты? {#can-i-add-my-own-facts}

Так! Факты гатовыя для перакладу. Вы можаце стварыць свой уласны файл `.po` са спецыяльнымі фактамі і пакласці яго ў каталог `/lang`. Або вы можаце прафільтраваць масыў фактаў праграмна.

## Ці паступоўвае гэта маё адміні stratumванне? {#does-this-slow-down-my-admin}

Ні ўвесам! Плагін вельмі лёгкі і не мае ўплыву на прадуктыўнасць. Ён толькі знаходзіць тэкст — без знешніх выклікаў API або цяжкай прапрацоўкі.

## Ці магу я выключыць яго на певных старонках? {#can-i-disable-it-on-certain-pages}

Факты аўтаматычна схаваныя на старонках, дзе яны могуць перашкаджаць (як-от рэдактар блокаў). Вы таксама можаце выкарыстоўваць хукі WordPress, каб наладзіць, дзе факты з'яўляюцца.

## Ці адпавядаюць факты ўсім аўдыторыя? {#are-the-facts-appropriate-for-all-audiences}

Так! Усе факты — гэта чысты, сімейны гумар, які адзначае легендарны статус чака Норрыса.

## Ці магу я перакласці факты на другую мову? {#can-i-translate-the-facts-to-another-language}

Абсалютутна! Усе факты выкарыстоўваюць функцыі перакладу WordPress. Стварыце файл перакладу для вашай мовы і пакласціце яго ў каталог `/lang`. Тэкстовая сферы: `ultimate-multisite-chuck-norris-facts`

## Ці працуе гэта з multisite? {#does-it-work-with-multisite}

Так! Ён спецыяльна распрацаваны для сеткавых Multisite і можа актывавацца на сеткі.
