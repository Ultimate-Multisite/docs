# Mailchimp

Ultimate Multisite: Інстыляцыя з Mailchimp

Звяжыце вашу сетку Ultimate Multisite з Mailchimp. Вы можаце звязаць пэўныя планы з спісамі Mailchimp і аўтаматычна дадаваць падпіскаўшых да гэтых спісаў пры рэгістрацыі.

## Устаноўка

1. Загрузіце 'ultimate-multisite-mailchimp' у каталог '/wp-content/plugins/'
2. Актывуйце plugin праз меню 'Plugins' у адмініструванні вашай WordPress Network

## Паўдарожжа па выкарыстанні

Сінханізацыя адраслаў падпіскаў з вашымі спісамі MailChimp вельмі лёгкая з нашым Add-on Ultimate Multisite: MailChimp Integration. Гэты паўдарожжа адначасова паможа вам усе ўсё наладзіць!

## Атримання API ключа

Перш чым мы пачнем разглядаць розныя налады, нам патрэбен API ключ MailChimp. Гэта дазволіць Ultimate Multisite спілкувацца з API MailChimp, каб атрымаць спісы, групы і дадаваць адраслевыя адрасы ў гэтыя спісы і групы, калі карыстальнікі рэгіструюцца.

Паслядоўце гэтым паўдарожжу, каб атрымаць свой API ключ MailChimp: [About API Keys | Mailchimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

_Дадайце свой API ключ MailChimp у_ _**Налады Ultimate Multisite - > Налады Add-on -> MailChimp**_

## Рэжым інтэграцыі

Add-on MailChimp Integration прапануе два "рэжымы інтэграцыі", што азначае, што ёсць два розныя спосабы выкарыстаць аддон для сінханізацыі адраслевых адрасаў.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_Два рэжымы інтэграцыі для вялікай гнуткасці!_

### Рэжым 1: Некалькі спісаў (Multiple Lists)

Паўднёўны рэжым выкарыстоўвае **Некалькі спісаў (Multiple Lists)**. Гэты рэжым дазволіць вам выбраць некалькі спісаў MailChimp для кожнага плана. Гэта лепшы варыянт, калі вы падільняеце сваіх карыстальнікаў на розныя спісы.

Напрыклад, уявіце, што ў вас ёсць 3 розныя спісы на вашым аккаунце MailChimp: **Newsletter** , **Plan A** , і **Plan B**. Вы хочаце, каб усе карыстальнікі, незалежна ад плана, былі ў спісе Newsletter, каб яны не прагаварылі вашыя навіны.

Каб гэта дасягнуць, вы можаце выкарыстоўваць рэжым Некалькі спісаў, каб дадаць спісы **Newsletter** і **Plan A** у Налады MailChimp для Plan A, і **Newsletter** і **Plan B** у Налады MailChimp для Plan B.

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_Выберыце спісы па паўднёўных для новых і адмененых аккаунтаў_

### Рэжым 2: Адральны спіс + Некалькі групаў (Single List + Multiple Groups)

MailChimp дазваляе вам падільніць ваш спіс на меншыя групы карыстальнікаў, якія называюцца **Групы (Groups)**. Гэта дазваляе вам мець больш кантролю, бо вы можаце адправіць кампанію ў пэўны падбор карыстальнікаў, не ствараючы некалькі копій адной і той жа кампаніі ў MailChimp (што стварае цяжкасць з аналізай метрык гэтай кампаніі пазней).

Напрыклад, вы можаце мець адзін спіс пад назвай **My Network Subscribers**, і ўнутры яго мець групы для **Plan A** і **Plan B**.

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_Выберыце спісы і групы па паўднёўных, калі выкарыстоўваеце рэжым інтэграцыі "аддарльны спіс + некалькі групаў"_

## Паўднёўныя спісы і групы

Пасля таго, як вы выберыце, які рэжым інтэграцыі больш падыходзіць да вашага канкрэтнага выпадку выкарыстання, час выбраць паўднёўныя спісы (і групы, залежна ад рэжыму). Паўднёўны спіс будзе выкарыстованы, калі пэўны план не мае выбранага спісу.

Вы таксама можаце выбраць паўднёўны спіс для адмененых аккаунтаў. Колі падпіска карыстальніка заканчваецца і ён адказваецца ад вашай сеткі, яго адраслевы адрас будзе перамешчаны ў спіс, вызначаны для адмененых аккаунтаў.

## Аказанне двойнай падпіскі

На той жа адказной старонцы вы зможаце актываваць **Двойную падпіску (Double Subscription)**. Гэты варыянт дадасць карыстальніка як да паўднёўнага спісу, так і да спісу, вызначанага для плана карыстальніка. Калі гэты варыянт не актываваны, карыстальнік будзе дададзены толькі да спісу, вызначанага для плана карыстальніка. Калі для гэтага канкрэтнага плана нічога не вызначана, то карыстальнік будзе дададзены да паўднёўнага спісу, вызначанага.

_Вашыя карыстальнікі могуць быць падпісаны як да паўднёўнага спісу, так і да спісу плана пры рэгістрацыі_

## Выбар спісаў і групаў для кожнага плана

Зайдзіце ў **Планы (Plans)** і выберыце план, які вы хочаце рэдагаваць. Зверніце ўвагу, што ў мета-бокс **Advanced Options** будзе даступны новы таб, названы MailChimp.

На гэтым табе вы зможаце выбраць некалькі спісаў або аддарльны спіс і некалькі групаў (залежна ад рэжыму інтэграцыі) для гэтага плана.

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_Выберыце спісы і групы, якія вы хочаце выкарыстоўваць для кожнага плана_

Калі новы карыстальнік рэгіструецца па гэтым плане, яго адраслевы адрас будзе дададзены да выбраных спісаў і групаў!
