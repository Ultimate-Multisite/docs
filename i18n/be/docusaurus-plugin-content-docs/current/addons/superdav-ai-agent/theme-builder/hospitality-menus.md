# Меню для гаспітальнасці

Функцыя **Меню для гаспітальнасці** дазваляе Theme Builder ствараць і ўстаўляць структураваныя меню стравы і напояў непасрэдна на ваш сайт WordPress.

## Агляд {#overview}

Theme Builder цяпер можа ствараць прафесійныя, структураваныя меню для гаспітальных прадпрыемстваў, у тым ліку рэстараканаў, кафе, барэў і кейтеринг-служаванняў. Гэтыя меню будзяць у поўнай меры інтэграваны ў дызайн вашага сайта і іх лёгка адрэдагаваць і кіраваць.

## Падтрымкі гаспітальных тыпаў {#supported-hospitality-types}

- **Рэстараканы** — меню для ўжывання ў поўнай паслузе
- **Кафе** — меню кавы і лёгкіх снекаў
- **Бары і лаунжы** — меню напояў і закусок
- **Пекарень** — меню выпічкі і хлёба
- **Кейтеринг-служаванні** — варыянты меню для падзей
- **Фуд-тракі** — меню мабільнага харчавання
- **Прыварні і ўінэры** — меню напояў з апісаннямі

## Структура меню {#menu-structure}

### Катэгорыі меню {#menu-categories}

Меню арганізаваны па катэгорыях:

- **Закускі** — стартары і невялікія тарты
- **Асноўныя стравы** — галоўныя блюда
- **Гарніры** — суправаджняльныя стравы і веганскія стравы
- **Дэсерты** — сласткія лакомства
- **Напоі** — напоі (алкалястычныя і неалкалястычныя)
- **Спецыялі** — штодзённыя або сезонныя спецыялі

### Формат пункту меню {#menu-item-format}

Кажны пункт меню ўключае:

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### Паля пункту {#item-fields}

| Пале | Тып | Апісанне |
|-------|------|-------------|
| `name` | string | Назва стравы або напою |
| `description` | string | Падрабязнае апісанне пункту |
| `price` | string | Ціна (фарматована з валютай) |
| `dietary_info` | array | Дыетарычныя атратывы (Vegan, Gluten-Free і г. д.) |
| `allergens` | array | Паўторныя аллергены (Горэхи, Ракастры і г. д.) |
| `availability` | string | Калі даступны (Daily, Seasonal і г. д.) |

## Стваৰэнне меню для гаспітальнасці {#creating-hospitality-menus}

### Крок 1: Надаць інфармацыю пра меню {#step-1-provide-menu-information}

Паведамце Theme Builder пра сваё меню:

```
Create a restaurant menu for my Italian restaurant with appetizers,
pasta dishes, main courses, and desserts. Include prices and descriptions.
```

### Крок 2: Theme Builder стварае меню {#step-2-theme-builder-generates-menu}

Theme Builder:

1. Стварае структураванае меню.
2. Афармляе яго ў адпаведнасці з тэмай вашага сайта.
3. Арганізуе пункты па катэгорыях.
4. Фарматуе цэны і апісанні.
5. Дадае інфармацыю пра дыетарычныя абмежаванні і аллергены.

### Крок 3: Пагледжаць і набілічнаваць {#step-3-review-and-customize}

Вы можаце:

1. Рэдагаваць пункты меню і цэны.
2. Дадаваць або выдаляць катэгорыі.
3. Пазначаць парядок пунктаў унутры катэгорый.
4. Актуалізаваць апісанні і дыетарычную інфармацыю.
5. Наладжваць стыль і раскладку.

## Варыянты выказання меню {#menu-display-options}

### Поўнае меню на адным спілкуванні {#full-menu-page}

Выкверэнены спілкуванні, што адлюстроўвае ўсё ваша меню:

- Арганізована па катэгорыях.
- Магчыма знайсці і адфільтраваць.
- Афармленне для друку.
- Дызайн, адаптаваны для мабільных тэлефонаў.

### Віджэт меню {#menu-widget}

Устаўляйце секцыі меню на іншыя спілкуванні:

- Выдадныя пункты на галоўнай старонцы.
- Штодзённыя спецыялі ў бокавой палосе.
- Меню напояў на спілкуванні бара.
- Выстава дэсертаў у падвароўні.

### PDF меню {#menu-pdf}

Стварае спайдляжальны PDF-файл меню:

- Прафесійны фарматаванне.
- Якасць, гатовая для друку.
- Уключае малюнкі і апісанні.
- Легальна падільіцца і адправіць па электроннай пошце.

## Дыетарычныя абмежаванні і аллергены {#dietary-and-allergen-information}

### Дыетарычныя атратывы {#dietary-attributes}

Адзначайце пункты дыетарычнай інфармацыяй:

- **Vegan** — без жывёльных прадукт або
- **Vegetarian** — без мяса
- **Gluten-Free** — бяспечна для людзей з целіакыяй
- **Dairy-Free** — без маладочных прадукт або
- **Nut-Free** — без древых горэхаў або арашідёў
- **Low-Carb** — з нізкім узроўнем вуглядоў
- **High-Protein** — багаты на белок

### Папераджанні пра аллергены {#allergen-warnings}

Уключайце паўторныя аллергены:

- **Nuts** — древыя горэхі і арашідёў
- **Shellfish** — ракастры і маллюскі
- **Fish** — усе віды рыбы
- **Dairy** — маlačё і маладочные прадукт
- **Eggs** — курыныя яйцы
- **Soy** — соевыя прадукт
- **Gluten** — крэмень і збіжжа, якія ўтрымліваюць глютен
- **Sesame** — сесамовыя зярэнь і маста

## Кіраванне меню {#menu-management}

### Актуалізацыя цэнаў {#updating-prices}

Лёгка актуалізуйце цэны ў меню:

1. Пходзьце на спілкуванне меню.
2. Націсніце "Edit Menu".
3. Актуалізуйце цэны для пунктаў.
4. Захаваць змены.
5. Змены неадкладна з'яўляюцца на вашым сайце.

### Дадаванне сезонных пунктаў {#adding-seasonal-items}

Стварайце сезонныя варыянты меню:

1. Стварыце новую версію меню.
2. Дадавайце сезонныя пункты.
3. Адзначайце пункты як "Seasonal".
4. Наладзьце даты даступнасці.
5. Аўтаматычна адлюстроўвацца пакуль сезон.

### Кіраванне спецыялямі {#managing-specials}

Адлюстроўвайце штодзённыя або штонедельныя спецыялі:

1. Стварыце катэгорыю "Specials".
2. Дадавайце пункты з датамі даступнасці.
3. Выдзейніце спецыялі на галоўнай старонцы.
4. Актуалізуйце штодня або штонедзельнікам.
5. Архавіруйце старыя спецыялі.

## Інтэграцыя з Theme Builder {#integration-with-theme-builder}

Калі вы выкарыстоўваеце Theme Builder для гаспітальных сайтаў:

1. **Automatic menu detection** — выяўляе, ці ў вас прадпрыемства харчавання/напояў.
2. **Menu page creation** — стварае прафесійныя спілкуванні меню.
3. **Design matching** — меню адпавядаюць тэме вашага сайта.
4. **Mobile optimization** — меню выглядаюць прыгожна на тэлефонах.
5. **SEO optimization** — меню зручныя для пошукавых сіл.

## Найлепшыя практыкі {#best-practices}

### Дызайн меню {#menu-design}

- **Чытыльная арганізацыя** — лагічная структура катэгорый.
- **Падрабязныя апісанні** — апетыядныя і інфармацыйныя.
- **Пастаянная цэнаванне** — ясная валюта і фарматаванне.
- **Парафесійныя малюнкі** — малюнкі ежы высокай якасці.
- **Пустыя месцы** — не перагружайце спілкуванне.

### Кантэнт {#content}

- **Адпаведныя апісанні** — апісвайце пункты правільна.
- **Выдзейніце спецыялі** — павялічце ўвагу на спецыялі.
- **Уключайце аллергены** — заўсёды пералічвайце аллергены.
- **Регулярна актуалізуйце** — падтрымлівайце цэны і пункты ў курсе.
- **Выкарыстоўвайце апетыядную мову** — нехай пункты гукляюць смачна.

### Доступнасць {#accessibility}

- **Чытыльны шрыфт** — выкарыстоўвайце ясны, лёгка чытальны тэкст.
- **Падостаткова контрасту** — гарантуйце, што тэкст лёгка чытальны.
- **Дыетарычныя маркіроўкі** — ясна адзначайце дыетарычныя варыянты.
- **Папераджанні пра аллергены** — выдзейніце аллергены.
- **Адаптаваны для мабільных** — праверка на ўсіх прыладах.
