# Superdav AI Agent {#superdav-ai-agent}

Superdav AI Agent — гэта прасунуты AI-памочнік для WordPress, які пашырае Gratis AI Agent навыкамі з улікам тэмы, палепшанымі магчымасцямі інтэрнэт-пошуку, функцыямі канструктара сайтаў і дадатковымі інтэграцыямі аўтаматызацыі. Версія 1.18.0 дадае інструменты чытання Google Calendar, супастаўленне кантактаў і ўдзельнікаў, шлюзы адабрэння чалавекам, дэдублікацыю напамінаў, SMS-апавяшчэнні TextBee, кіраванае разгортванне сэрвісу Superdav і асобна распаўсюджваны спадарожны пакет Advanced для давераных працоўных працэсаў распрацоўшчыкаў.

## Асноўныя магчымасці {#key-features}

- **Навыкі з улікам тэмы** — аўтаматычна адаптуе рэкамендацыі і магчымасці на аснове вашай актыўнай тэмы WordPress і ўсталяваных плагінаў
- **Некалькі пастаўшчыкоў інтэрнэт-пошуку** — выбірайце паміж Brave Search і Tavily для багацейшых вынікаў пошуку і прасунутых магчымасцей даследавання
- **Кантактныя формы канструктара сайтаў** — стварайце цалкам функцыянальныя кантактныя формы непасрэдна з інтэрфейсу чата, выкарыстоўваючы ўсталяваны плагін формаў
- **Палепшаны канструктар сайтаў** — удасканаленыя шматфазныя планы стварэння сайта з лепшым выяўленнем плагінаў і аднаўленнем пасля памылак
- **Убудаваныя навыкі тэм** — спецыялізаваныя рэкамендацыі для блокавых тэм, класічных тэм, Kadence Blocks і Kadence Theme
- **Палепшаная інтэграцыя WooCommerce** — перапрацавана для выкарыстання ўласных API WooCommerce дзеля лепшай надзейнасці і сумяшчальнасці
- **Аўтаматычнае выяўленне пастаўшчыкоў** — аўтаматычна выяўляе і абнаўляе даступныя плагіны і тэмы, калі яны актывуюцца або дэактывуюцца
- **Генерацыя ўласнага лагатыпа** — генеруйце і ўбудоўвайце ўласныя SVG-лагатыпы з аўтаматычнай бяспечнай для прастор імёнаў санітызацыяй
- **Праверка кантрасту палітры** — правярайце пары колераў на адпаведнасць патрабаванням даступнасці WCAG перад ужываннем да вашай тэмы
- **Выяўленне з выкарыстаннем фота** — загружайце фота падчас інтэрв’ю выяўлення Theme Builder, каб паўплываць на кірунак дызайну
- **Меню для сферы гасціннасці** — генеруйце структураваныя старонкі меню ежы і напояў для рэстаранаў, кавярняў і бізнесаў сферы гасціннасці
- **Папярэдні прагляд на настольных і мабільных прыладах** — праглядайце свой дызайн на настольных і мабільных прыладах падчас выбару кірунку дызайну
- **Скрапінг сайтаў Tier 1** — sd-ai-agent/site-scrape цяпер даступны па змаўчанні для аналізу сайтаў канкурэнтаў і крыніц натхнення
- **Інструменты чытання Google Calendar** — чытайце наладжаныя календары і падзеі, каб аўтаматызацыі маглі ўлічваць кантэкст будучага раскладу
- **Супастаўленне кантактаў і ўдзельнікаў** — супастаўляйце ўдзельнікаў календара з вядомымі карыстальнікамі сайта і кантактамі перад генерацыяй наступных дзеянняў
- **Шлюзы адабрэння чалавекам** — прыпыняйце адчувальныя аўтаматызацыі для праверкі перад адпраўкай паведамленняў, аднаўленнем працы або ўнясеннем змен
- **Запісы напамінаў і дэдублікацыя** — адсочвайце напаміны, якія ўжо былі адпраўлены, каб паўторныя запускі аўтаматызацыі не апавяшчалі таго самага атрымальніка двойчы
- **SMS-апавяшчэнні TextBee** — адпраўляйце наладжаныя тэкставыя апавяшчэнні праз TextBee, калі пастаўшчык SMS уключаны
- **Кіраванае разгортванне сэрвісу Superdav** — падключайце падтрымліваемыя сайты да размешчаных Superdav endpoint-аў з аўтаматычнай наладкай замест ручной канфігурацыі endpoint-а
- **Спадарожны пакет Advanced** — асобна распаўсюджваныя інструменты для давераных распрацоўшчыкаў для працоўных працэсаў файлавай сістэмы, базы даных, WP-CLI, REST-дыспетчара, канструктара плагінаў, git-здымкаў, кіравання карыстальнікамі і бенчмаркінгу

## Патрабаванні {#requirements}

- WordPress 6.0 або вышэй
- PHP 8.1 або вышэй
- WordPress Multisite (актываваны ў сетцы) або ўсталёўка на адзін сайт
- Актыўны плагін-злучальнік AI-пастаўшчыка (Anthropic, OpenAI або сумяшчальны)
- Gratis AI Agent v1.9.0 або вышэй (Superdav AI Agent пашырае Gratis AI Agent)

## Усталяванне {#installation}

1. Загрузіце файлы плагіна ў `/wp-content/plugins/superdav-ai-agent`
2. Актывуйце плагін праз экран **Плагіны** у WordPress
3. Пераканайцеся, што Gratis AI Agent таксама актываваны
4. Перайдзіце ў **Gratis AI Agent → Settings** і наладзьце свайго пастаўшчыка інтэрнэт-пошуку, падключэнне да кіраванага сэрвісу Superdav, Google Calendar або ўліковыя даныя TextBee паводле патрэбы

Пакет WordPress.org змяшчае асноўныя магчымасці Superdav AI Agent. Спадарожны пакет Advanced збіраецца і распаўсюджваецца асобна для асяроддзяў, дзе інструменты давераных распрацоўшчыкаў яўна ўсталяваны і аўтарызаваны.

## Навыкі з улікам тэмы {#theme-aware-skills}

Superdav AI Agent аўтаматычна выяўляе вашу актыўную тэму і ўсталяваныя плагіны, загружаючы адпаведны навык з улікам тэмы ў кантэкст памочніка. Даступныя навыкі ўключаюць:

| Навык | Да чаго ўжываецца |
|---|---|
| **Блокавыя тэмы** | Тэмы, якія выкарыстоўваюць блокавую архітэктуру WordPress (поўнае рэдагаванне сайта) |
| **Класічныя тэмы** | Традыцыйныя тэмы WordPress, якія выкарыстоўваюць PHP-шаблоны і класічны рэдактар |
| **Kadence Blocks** | Сайты з актыўным плагінам Kadence Blocks |
| **Kadence Theme** | Сайты з тэмай Kadence як актыўнай тэмай |

Глядзіце [Навыкі з улікам тэмы](../../user-guide/configuration/theme-aware-skills) для падрабязных рэкамендацый па кожным навыку.

## Пастаўшчыкі інтэрнэт-пошуку {#internet-search-providers}

Superdav AI Agent v1.10.0 дадае **Tavily** як альтэрнатыву Brave Search для запытаў інтэрнэт-пошуку. Абодва пастаўшчыкі прапануюць вынікі пошуку ў рэальным часе і могуць быць наладжаны ў **Gratis AI Agent → Settings → Advanced**.

| Пастаўшчык | Магчымасці | Наладжванне |
|---|---|---|
| **Brave Search** | Хуткія вынікі пошуку з акцэнтам на прыватнасць | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Больш насычаныя вынікі пошуку, пашыраныя даследчыя магчымасці | [Tavily API](https://tavily.com/) |

Глядзіце [Інтэрнэт-пошук](../../user-guide/configuration/internet-search) для інструкцый па канфігурацыі.

## Рабочыя працэсы календара, зацвярджэння і апавяшчэнняў {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 пашырае рабочыя працэсы аўтаматызацыі за межы задач са змесцівам і стварэннем сайтаў:

- **Інструменты Google Calendar** дазваляюць агенту чытаць наладжаныя календары і падрабязнасці падзей, калі даступныя ўліковыя даныя календара.
- **Дапаможнікі для кантактаў і ўдзельнікаў** супастаўляюць удзельнікаў падзей з карыстальнікамі WordPress або вядомымі кантактамі перад падрыхтоўкай напамінаў ці наступных задач.
- **Шлюзы зацвярджэння чалавекам** дазваляюць аўтаматызацыі прыпыніцца, пакуль упаўнаважаны карыстальнік не разгледзіць прапанаванае дзеянне.
- **Запісы напамінаў** прадухіляюць дубляванне календарных або апавяшчальных напамінаў, калі аўтаматызацыя паўтараецца або плануецца неаднаразова.
- **Падтрымка TextBee SMS** уключае апавяшчэнні тэкставымі паведамленнямі, калі пастаўшчык TextBee наладжаны.

Выкарыстоўвайце гэтыя інструменты для дапамогі з улікам раскладу, наступных дзеянняў з удзельнікамі і рабочых працэсаў напамінаў, якія ўсё яшчэ патрабуюць праверкі чалавекам перад адпраўкай адчувальных паведамленняў.

## Канфігурацыя {#configuration}

Усе налады знаходзяцца ў **Gratis AI Agent → Settings** у адміністраванні WordPress. Superdav AI Agent дадае наступныя параметры канфігурацыі:

| Налада | Апісанне |
|---|---|
| **Пастаўшчык інтэрнэт-пошуку** | Выберыце паміж Brave Search і Tavily (неабавязкова) |
| **Tavily API Key** | API-ключ для інтэрнэт-пошуку Tavily (калі выбраны Tavily) |
| **Кіраваны сэрвіс Superdav** | Падключайце падтрыманыя сайты да размешчанага сэрвісу Superdav з аўтаматычным прадастаўленнем endpoint-аў |
| **Уліковыя даныя Google Calendar** | Наладзьце доступ да календара, каб агент мог чытаць календары і падзеі |
| **Пастаўшчык TextBee SMS** | Наладзьце ўліковыя даныя TextBee для зацверджаных рабочых працэсаў SMS-апавяшчэнняў |

## Пашыраны спадарожны пакет {#advanced-companion-package}

Superdav AI Agent v1.18.0 дадае асобна распаўсюджваны пашыраны спадарожны пакет для давераных рабочых працэсаў распрацоўшчыкаў. Гэтыя магчымасці не ўваходзяць у асноўны пакет WordPress.org. Усталёўвайце спадарожны пакет толькі на сайтах, дзе адміністратары відавочна давяраюць агенту доступ да інструментаў распрацоўшчыка.

Пашыраныя інструменты ўключаюць аперацыі з файлавай сістэмай, доступ да базы даных, WP-CLI, рабочыя працэсы REST-дыспетчара, утыліты builder-а plugin-аў, git-здымкі, дапаможнікі кіравання карыстальнікамі і інструменты тэставання прадукцыйнасці. Абмяжоўвайце гэтыя інструменты даверанымі адміністратарамі і правярайце канфігурацыю шлюзаў зацвярджэння перад уключэннем іх на вытворчых сайтах.

## Паляпшэнні ў v1.10.0 {#improvements-in-v1100}

- **Інтэграцыя WooCommerce перапрацавана** — цяпер выкарыстоўвае ўласныя API WooCommerce для большай надзейнасці і сумяшчальнасці
- **Аўтаабнаўленне спіса правайдараў** — аўтаматычна вызначае, калі плагіны актывуюцца або дэактывуюцца, і абнаўляе спіс даступных правайдараў
- **Палепшаны канструктар сайтаў** — палепшаныя шматфазныя планы стварэння сайта з лепшым аднаўленнем пасля памылак
- **Навыкі з улікам тэмы** — чатыры новыя ўбудаваныя навыкі, якія аўтаматычна адаптуюцца да вашай актыўнай тэмы

## Выпраўленні памылак у v1.10.0 {#bug-fixes-in-v1100}

- Выпраўлена здольнасць `list-posts`, якая некарэктна пераўтварала назвы катэгорый і тэгаў у slugs

- Выпраўлена, каб скалярныя вынікі інструментаў карэктна абгортваліся перад вяртаннем у AI
