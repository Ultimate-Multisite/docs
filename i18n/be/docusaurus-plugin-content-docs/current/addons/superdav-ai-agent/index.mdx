# Superdav AI Agent

Superdav AI Agent — гэта павышаны па асабістаму асістэнце WordPress, які пашырае функцыянальнасць Gratis AI Agent, дадаўшы ў ім уменне, спасыланё на тэму (theme-aware skills), палепшаныя магчымасці пошуку ў Інтэрнэце і ўзмацненыя функцыі стваральніка сайта (site builder). У версіі 1.16.0 дадалі стварэнне карысных логоў, праверку кантрасту палітраў, загрузку выяваў падчас апытуванні ў канцэпцыі, стварэнне структураваных меню для гаспiтальнасці, а таксама рэндэрынг папярэдняга прагляду на настольных і мабільных прыладах у Theme Builder.

## Асноўныя функцыі

- **Уменне, спасыланё на тэму** — аўтаматычна апасноўвае паказанні і магчымасці на аснове актыўнай тэмы WordPress і ўсталяваных плагінаў
- **Некалькі правайдэнтаў пошуку ў Інтэрнэце** — выбірайце паміж Brave Search і Tavily для больш багатых вынікаў пошуку і пашырэных рэдакцыйных магчымасцей
- **Формы кантактаў у стваральніку сайта** — стварайце цалкам працаваныя формы кантактаў непасрэдна з чат-інтерфейсу, выкарыстоўваючы ўсталяваны плагін для форм
- **Палепшаны стварльнік сайта** — палепшаныя планы стварэння сайта ў некалькіх этапах з лепшым выяўленнем плагінаў і аднаўленнем памылак
- **Вбудаваныя уменні тэмы** — спецыялізаваныя паказанні для Block Themes, Classic Themes, Kadence Blocks і Kadence Theme
- **Палепшаная інтэграцыя з WooCommerce** — перапрацавана для выкарыстання ўстакоўкі WooCommerce API, што гарантуе лепшую надзейнасць і сумяшчальнасць
- **Аўтаматычнае выяўленне правайдэнтаў** — аўтаматычна выяўляе і абнаўляе даступныя плагіны і тэмы, калі яны актывуюцца або дэактывуюцца
- **Стваранне карысных логоў** — стварэнне і ўбудоўванне карысных SVG-логоў з аўтаматычнай санітызацыяй, бяспечнай для імястоў
- **Праверка кантрасту палітраў** — праверка парыў кантоў на выкананне патрабаванняў WCAG да канчатковага прымянення ў тэме
- **Апытанне з выявамі** — загружайце выявы падчас апытування ў Theme Builder, каб пазбегнуць накіравання дызайну
- **Меню для гаспiтальнасці** — стварэнне структураваных старон меню з ежы і напояў для рэстараканаў, кафе і гаспiтальных прадпрыемстваў
- **Папярэднія прагляды на настольных і мабільных прыладах** — праглядзіце свой дызайн на настольных і мабільных прыладах падчас выбару накіравання дызайну
- **Scraping сайта ўзроўню 1** — sd-ai-agent/site-scrape цяпер па спілкуванні ў стандартным рэжыме для аналізу сайтаў канкурэнтаў і крыніў натхнення

## Патрабаванні

- WordPress 6.0 або больш высокі
- PHP 8.1 або больш высокі
- WordPress Multisite (актываваны на сетцы) або ўстаноўка для адной тэмы
- Актыўны плагін-каннеКТар для AI-провайдэнта (Anthropic, OpenAI або сумяшчальны)
- Gratis AI Agent v1.9.0 або больш высокі (Superdav AI Agent пашырае функцыянальнасць Gratis AI Agent)

## Устаноўка

1. Загрузіце файлы плагіна ў `/wp-content/plugins/superdav-ai-agent`
2. Актывуйце плагін праз экран **Plugins** у WordPress
3. Убедзіцеся, што таксама актываваны Gratis AI Agent
4. Айдзіце ў **Gratis AI Agent → Settings** і наладзьце свой правайдэнта пошуку ў Інтэрнэце (неабязава)

## Пачатак працы

Пасля актывацыі Superdav AI Agent аўтаматычна палепшае вашу ўстаноўку Gratis AI Agent, дадаўшы ў яе уменні, спасыланё на тэму, і адпаведныя магчымасці. Адкраніце панэль **Gratis AI Agent** у бокавым меню адмінастрацыі WordPress і пачніце выкарыстоўваць новыя функцыі.

**Прыклады інструкцый:**

```
Дадайце форму кантактаў на старонку "Кантакты".
```

```
Пашукайце ў вебе найлепшыя сучасныя правілы бяспекі WordPress.
```

```
Стварыце секцыю "герой" выкарыстоўваючы блёк-патэрны актыўнай тэмы.
```

## Уменні, спасыланё на тэму

Superdav AI Agent аўтаматычна выяўляе вашу актыўную тэму і ўсталяваныя плагіны, загружаючы адпаведныя уменні, спасыланё на тэму, у канцэст асістэнта. Даступныя уменні ўключаюць:

| Уменне | Падыходзіць для |
|---|---|
| **Block Themes** | Тэмы, якія выкарыстоўваюць апаратную архітэктуру WordPress на аснове блёкаў (Full Site Editing) |
| **Classic Themes** | Традыцыйныя тэмы WordPress, якія выкарыстоўваюць PHP-шаблоны і класічны рэдактар |
| **Kadence Blocks** | Сайты з актыўным плагінам Kadence Blocks |
| **Kadence Theme** | Сайты з тэмай Kadence ў якасці актыўнай тэмы |

Для больш падрабязнага паказання кожнага ўмення спадыліце [Theme-Aware Skills](../../user-guide/configuration/theme-aware-skills).

## Провайдэнты пошуку ў Інтэрнэце

Superdav AI Agent v1.10.0 дадаў **Tavily** як альтэрнатыву Brave Search для пошуку ў Інтэрнэце. Аба правайдэнта выконваюць пошук у рэальным часе і іх можна наладзіць у **Gratis AI Agent → Settings → Advanced**.

| Провайдэнт | Функцыі | Наладка |
|---|---|---|
| **Brave Search** | Шматпрытрыўны пошук, сканцэнтраваны на прыватнасці | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Больш багаты пошук, пашырэныя рэдакцыйныя магчымасці | [Tavily API](https://tavily.com/) |

Для інструкцый па канфігурацыі спадыліце [Internet Search](../../user-guide/configuration/internet-search).

## Формы кантактаў у стваральніку сайта

Агенты стваральніка сайта цяпер могуць ствараць формы кантактаў непасрэдна з чат-інтерфейсу. Проста папросіце стваральніка сайта стварыць форму кантактаў, і ён аўтаматычна выяўліць ваш ўсталяваны плагін для форм (Contact Form 7, WPForms, Fluent Forms або Gravity Forms) і згенеруе гатоў-до-выкарыстання форму.

Для падрабязнага паказання спадыліце [Site Builder Contact Forms](../../user-guide/administration/site-builder-contact-forms).

## Функцыі Theme Builder

Theme Builder у Superdav AI Agent v1.16.0 уключае магутныя новыя магчымасці:

### Апытанне з загрузкай выяваў

Апытанне Theme Builder цяпер уключае этап загрузкі выявы, дзе вы можаце прадастаўляць візуальнае натхненне. Загрузіце выявы вашага брэнда, сайтаў канкурэнтаў або дызайнерскага натхнення, каб дапамагчы Theme Builder зразумець ваш дызайнерскі накіраванне.

Для падрабязнага паказання спадыліце [Discovery Interview](./theme-builder/discovery-interview.md).

### Дызайнерскае накіраванне з папярэдніямі праглядамі на настольных і мабільных прыладах

Паглядзіце свой дызайн на настольных і мабільных прыладах, перш чым Theme Builder згенеруе вашу канчатковую тэму. Паглядзіце рэальны прагляд у некалькіх рэастытыўных кропкавых памербах.

Для падрабязнага паказання спадыліце [Design Direction](./theme-builder/design-direction.md).

### Меню для гаспiтальнасці

Theme Builder цяпер можа ствараць структураваныя старонкі меню з ежы і напояў для рэстараканаў, кафе, бар і іншых гаспiтальных прадпрыемстваў. Меню ўключаюць цану, апісаванні, інфармацыю пра рагітванне і папярэджанні пра аллергены.

Для падрабязнага паказання спадыліце [Hospitality Menus](./theme-builder/hospitality-menus.md).

### Новыя магчымасці

- **[Generate Logo SVG](./abilities/generate-logo-svg.md)** — стварэнне карысных SVG-логоў з аўтаматычнай санітызацыяй
- **[Validate Palette Contrast](./abilities/validate-palette-contrast.md)** — праверка парыў кантоў на выкананне патрабаванняў WCAG
- **[Create Menu](./abilities/create-menu.md)** — стварэнне навігацыйных меню з адрознімі тэгамі навігацыі

Для інфармацыі пра даступныя інструменты і інструмент scraping сайта ўзроўню 1 спадыліце [Tools and Configuration](./theme-builder/tools.md).

## Канфігурацыя

Усе налады знаходзяцца ў **Gratis AI Agent → Settings** у адмінастрацыі WordPress. Superdav AI Agent дадае наступныя варыянты канфігурацыі:

| Наладка | Апісан |
|---|---|
| **Internet Search Provider** | Выбірайце паміж Brave Search і Tavily (неабязава) |
| **Tavily API Key** | API-ключ для пошуку ў Інтэрнэце Tavily (калі выбрана Tavily) |

## Палепшэнні ў v1.16.0

- **Кэш AI Client** — цяпер падтрымліваецца праз transients для працягласці паміж запытамі, zapрэўваючы страту даных падчас доўгапрацяглых задачаў агента
- **Інструмент scraping сайта ўзроўню 1** — sd-ai-agent/site-scrape цяпер па спілкуванні ў стандартным рэжыме ў Theme Builder для аналізу сайтаў канкурэнтаў і крыніў натхнення
- **Пасылкі дзеянняў у радку плагінаў** — выпраўлены і перададзены імя для яскрасці

## Палепшэнні ў v1.10.0

- **Перапрацаваная інтэграцыя з WooCommerce** — цяпер выкарыстоўвае ўстакоўкі WooCommerce API, што гарантуе лепшую надзейнасць і сумяшчальнасць
- **Аўтаматычнае абнаўленне спісу правайдэнтаў** — аўтаматычна выяўляе, калі плагіны актывуюцца або дэактывуюцца, і абнаўляе даступны спіс правайдэнтаў
- **Палепшаны стварльнік сайта** — палепшаныя планы стварэння сайта ў некалькіх этапах з лепшым аднаўленнем памылак
- **Уменні, спасыланё на тэму** — чатыры новыя ўбудаваныя ўменні, якія аўтаматычна апасноўваюць вашу актыўную тэму

## Выпраўленыя памылкі ў v1.10.0

- Выпраўлена ўменне `navigate-to`, якое выклікае нескінчныя перазагрузкі на некаторых адмінастрацыйных старонках
- Выпраўлена ўменне `list-posts`, якое не правільна выяўляе імены катэгорый і тэга ў slugs
- Выпраўленыя каманды WP-CLI, якія не маюць псевдонімаў імястоў пасля папярэдняй перапрацоўкі
- Выпраўлена аўтаматычнае выкананне, якое грацыёзна адносіцца да сайтаў, дзе табліцы аўтаматызацыі яшчэ не створаны
- Выпраўлена ўменне `memory-save`, якое выкарыстоўвае правільны прэфікс імястоў у стваральніку інструкцый сістэмы
- Выпраўленыя вынікі скалярных інструментаў, якія правільна ўвапнаваўваюцца перад вяратам AI
- Выпраўленыя статыстыстыкі выкарыстання, якія правільна адноўляюць пастарэльны фармат ключа ў павышэнні з старых версій
