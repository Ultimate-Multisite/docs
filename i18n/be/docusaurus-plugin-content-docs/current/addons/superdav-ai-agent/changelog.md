# Changelog

## 1.16.0 — Выпушчана 2026-05-20

### Ноўнасці

- **Выкарыстоўваць SVG для лого** — Theme Builder цяпер можа ствараць і ўстаўляць карыстальніцкія SVG-лого з бяспечным (namespace-safe) санітызацыяй
- **Загрузка вывучэння ў інтар뷰 адкрыцця** — Theme Builder discovery interview цяпер уключае этап загрузкі вывучэння для больш багатага кантексту дызайну
- **Праверка кантрасту палітра** — дазваляе праверыць пары колераў на адпаведнасць WCAG, перш чым замясціць іх у тэму
- **Меню для гаспрыяльнасці** — Theme Builder цяпер можа ствараць структураваныя адкажванні (меню) з ежытай і напойвай.
- **Падгляд на настольным і мабільным дывайсах** — Падглядай свой дызайн на настольных і мабільных дывайсах падчас выбару nakірэння дызайну
- **Параметр тэга навігацыі** — Функцыя «Стварыць меню» цяпер падтрымлівае асобны `navigation_label`, які адрозніваецца ад спасылкі на старонку
- **Даступ да інструментаў Т1** — sd-ai-agent/site-scrape цяпер з'яўляецца інструментам Т1, які даступны па спецыяльнасці ў Theme Builder

### Выпраўлена

- **Кэш кліента AI** — цяпер падтрымліваецца праз transients для працягласці паміж запытамі, што zaprabiegt страце дадзеных падчас доўгапрацяглых задачаў агента
- **Пасылкі дзеянняў у радку плагіна** — выпраўлена і перададзена для больш яснага фарматавання

## 1.10.0 — Выпушчана 2026-05-05

### Ноўнасці

- **Пошук у інтэрнэце Tavily** — дададзены Tavily як пошукавы провайдар для больш багатых вынікаў пошуку ў інтэрнэце побач з Brave Search
- **Вбудаваныя навыкі, адукаваныя тэмай** — Block Themes, Classic Themes, Kadence Blocks і Kadence Theme skill guides цяпер пастаўляюцца ў пастаўку з плагінам
- **Функцыя формы кантактаў у стваральніку сайта** — дадавайце форму кантактаў на любыя старонкі непасрэдна з чат-інтерфейсу

### Уляпнена

- **Інтыгрэйцыя WooCommerce рэфактарызавана** — цяпер выкарыстоўвае ўбудаваныя API WooCommerce для больш высокай надзейнасці і сумяшчальнасці
- **Спіс провайдэраў аўтаматычна абнаўляецца** — калі актывуюць або дэактывуюць якіпа\\кі плагін

### Выпраўлена

- **Функцыя navigate-to** — выпраўлены нескінчны цыкл перазагрузкі на некаторых адмінковых старонках
- **Функцыя list-posts** — цяпер правільна вызначае імены катэгорый і тэга ў slugs
- **Команды WP-CLI** — адпраўлены адсутныя псевдонімы (aliases) пасля папярэдняга рэфактыביע
- **Аўтаматызацыя падзей** — грацыёзна апрацоўвае сайты, дзе табліцы аўтаматызацыі яшчэ не створаны
- **Функцыя memory-save** — цяпер выкарыстоўвае правільны прэфікс namespace ў стваральніку сістэмавых інструкцый
- **Вынікі інструмента Scalar** — цяпер правільна ўваджаюцца, перш чым быць вяртаным AI
- **Статыстыка выкарыстання** — цяпер правільна апрацоўвае стары фармат ключа функцыі падчас апгрэйду з старэйшых версій
