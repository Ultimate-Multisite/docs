# Журнал змяненняў

## 1.18.0 — Выпушчана 2026-06-29

### Новае

- **Інструменты Google Calendar** — чытайце наладжаныя календары і падзеі для аўтаматызацый з улікам раскладу
- **Супастаўленне кантактаў і дапаможныя сродкі для ўдзельнікаў** — супастаўляйце ўдзельнікаў календара з карыстальнікамі і кантактамі сайта
- **Шлюзы чалавечага зацвярджэння і запісы напамінаў** — прыпыняйце аўтаматызацыі для праверкі і пазбягайце дублікатаў апавяшчэнняў
- **Пастаўшчык SMS TextBee** — адпраўляйце наладжаныя тэкставыя апавяшчэнні праз TextBee
- **Пашыраны спадарожны пакет** — дадае інструменты даверанага распрацоўшчыка для файлавай сістэмы, базы даных, WP-CLI, дыспетчара REST, канструктара плагінаў, здымкаў git, кіравання карыстальнікамі і тэстаў прадукцыйнасці, якія распаўсюджваюцца асобна ад зборкі WordPress.org

### Палепшана

- **Наладжванне кіраванага сэрвісу Superdav** — дададзеныя канчатковыя пункты размешчанага сэрвісу і аўтаматычнае стварэнне падключэнняў для падтрымліваемых сайтаў
- **Пакаванне рэлізу** — збіраць асобныя ZIP-файлы ядра і Advanced, публікаваць абодва на GitHub і адпраўляць на WordPress.org толькі пакет ядра

### Выпраўлена

- **Надзейнасць AI-запытаў** — палепшаны выбар мадэлі, тайм-аўты запытаў, стандартная мадэль наладжвання, апрацоўка тэксту разважанняў і рэкамендацыі паўторных спроб для недапушчальных выклікаў інструментаў
- **Умацаванне календара і напамінаў** — узмоцнены токены Google Calendar і дэдуплікацыя напамінаў
- **Аднаўленне анбордынгу і зацвярджэнняў** — выпраўлены запускі анбордынгу на frontend і аднаўленні пацверджаных магчымасцей
- **Пытанні праверкі пакавання WordPress.org** — улічаны водгукі праверкі пакавання для рэлізу ядра

## 1.16.0 — Выпушчана 2026-05-20

### Новае

- **Магчымасць генерацыі Logo SVG** — Theme Builder цяпер можа генераваць і ўбудоўваць карыстальніцкія SVG-лагатыпы з санітызацыяй, бяспечнай для прастор імёнаў
- **Загрузка фота ў інтэрв’ю выяўлення** — інтэрв’ю выяўлення Theme Builder цяпер уключае крок загрузкі фота для багацейшага дызайнерскага кантэксту
- **Магчымасць праверкі кантрасту палітры** — правярайце пары колераў на адпаведнасць WCAG перад ужываннем да тэмы
- **Меню для гасціннасці** — Theme Builder цяпер можа генераваць структураваныя старонкі меню ежы і напояў для бізнесаў у сферы гасціннасці
- **Рэндэрынг папярэдняга прагляду для настольных і мабільных прылад** — праглядайце свой дызайн на настольных і мабільных прыладах падчас выбару дызайнерскага напрамку
- **Параметр подпісу навігацыі** — магчымасць Create Menu цяпер падтрымлівае асобны `navigation_label`, адрозны ад загалоўка старонкі
- **Даступнасць інструментаў узроўню 1** — sd-ai-agent/site-scrape цяпер з’яўляецца інструментам узроўню 1, даступным па змаўчанні ў Theme Builder

### Выпраўлена

- **Кэш AI Client** — цяпер падтрымліваецца transient-амі для захавання паміж запытамі, што прадухіляе страту даных падчас працяглых задач агента
- **Спасылкі дзеянняў у радку плагіна** — выпраўленыя і перайменаваныя для яснасці

## 1.10.0 — Выпушчана 2026-05-05

### Новае

- **Інтэрнэт-пошук Tavily** — дададзены Tavily як пастаўшчык пошуку для багацейшых вынікаў інтэрнэт-пошуку разам з Brave Search
- **Убудаваныя навыкі з улікам тэмы** — кіраўніцтвы па навыках Block Themes, Classic Themes, Kadence Blocks і Kadence Theme цяпер пастаўляюцца разам з плагінам
- **Магчымасць кантактнай формы для канструктара сайтаў** — дадавайце кантактную форму на любую старонку непасрэдна з інтэрфейсу чата

### Палепшана

- **Рэфактарынг інтэграцыі WooCommerce** — цяпер выкарыстоўвае ўласныя API WooCommerce для лепшай надзейнасці і сумяшчальнасці
- **Спіс пастаўшчыкоў абнаўляецца аўтаматычна** — калі любы плагін актывуецца або дэактывуецца

### Выпраўлена

- **Магчымасць navigate-to** — выпраўлены бясконцы цыкл перазагрузкі на некаторых старонках адміністратара
- **Магчымасць list-posts** — цяпер правільна вызначае назвы катэгорый і тэгаў у slugs
- **Каманды WP-CLI** — адноўлены адсутныя псеўданімы прасторы імёнаў пасля папярэдняга рэфактарынгу
- **Аўтаматызацыя падзей** — карэктна апрацоўвае сайты, дзе табліцы аўтаматызацыі яшчэ не створаныя
- **Магчымасць memory-save** — цяпер выкарыстоўвае правільны прэфікс прасторы імёнаў у канструктары сістэмных інструкцый
- **Скалярныя вынікі інструментаў** — цяпер правільна абгортваюцца перад вяртаннем у AI
- **Статыстыка выкарыстання** — цяпер правільна апрацоўвае састарэлы фармат ключа магчымасці пры абнаўленні са старых версій
