# Multi-Currency

Ultimate Multisite: Multi-Currency дазваляе вашым сеткам прымацаць плацменты ў некалькіх валютах. Яно аўтаматычна канвертуе цаны, выкарыстоўваючы жывыя або ўручную ўстаноўленыя курсы валют, дазваляе вам ўсталяваць точныя цаны для кожнага прадукту ў любой валюце і выяўляе пераважвае валюту кожнага адвідувача праз геолокацыю — усё гэта кіруецца з адной адмін-панелі.

## Асноўныя функцыі {#key-features}

- **Аўтаматычная канверсія курсу валют** — цаны канвертуюцца з вашай асноўнай валюты, выкарыстоўваючы курсы, атрыманыя ад выбранага провайдэра.
- **Пазбегненне аўтаматычнай канверсіі для прадуктаў** — вы можаце ўсталяваць точныя цаны для канкрэтнай валюты на любы прадукт, каб праскочыць аўтаматычную канверсію.
- **Пале выбару валюты пры рэгістрацыі** — дазваляе кліентам выбраць пераважвае валюту падчас рэгістрацыі/аплатам.
- **Некалькі провайдэраў курсу валют** — ECB, Frankfurter, Stripe або ўсега ўручную ўстаноўленыя курсы.
- **Выяўленне валюты на аснове геолокацыі** — аўтаматычна выбірае валюту, якая адпавядае краіне адвідувача.
- **Адмін-панель** — кіруйце актыўнымі валютамі, курсамі і напрамленнямі провайдэраў з аднаго месца.

## Патрабаванні {#requirements}

- Ultimate Multisite 2.0.0 або больш высокія
- Устаноўка WordPress Multisite
- PHP 7.4 або больш высокі

## Устаноўка {#installation}

1. Загрузіце файлы плагіна ў `/wp-content/plugins/ultimate-multisite-multi-currency`
2. Актывуйце плагін праз экран **Plugins** у WordPress (Network Activate)
3. Праййдзіце па **Ultimate Multisite → Multi-Currency**, каб адкрыць адмін-панель
4. Дадайце валюты, якія вы хочаце прымацаць, і наладзьце свой провайдэра курсу валют

## Наладка {#configuration}

### Адмін-панель {#admin-dashboard}

Адмін-панель Multi-Currency (**Ultimate Multisite → Multi-Currency**) — гэта асноўнае месца для:

- Дадавання, актывацыі або выключэння валют
- Выбору провайдэра курсу валют і абнаўлення курса па патрабаванні
- Пагледжання бягучых курсаў для ўсіх актыўных валют
- Устаноўлення ручных курсаў, калі аўтаматычныя провайдэры не падыходзяць

### Провайдэры курсу валют {#exchange-rate-providers}

| Провайдэра | Апісан |
|---|---|
| **ECB** | Еўрапейскі$}; Банк — бясплатныя штоднёўныя курсы для сеткавых структураў, аснованых на EUR |
| **Frankfurter** | Адкрыты выцэнны API-адзеркаванне ECB — бясплатна, не патрабуе API-ключа |
| **Stripe** | Выкарыстоўвае жывыя курсы валют Stripe — патрабуе актываванага інтэграцыі Stripe |
| **Manual** | Вы ўводзіце курсы ўручную; не патрабуе знешніх выклікаў API |

Выберыце свой провайдэра ў **Ultimate Multisite → Multi-Currency → Settings → Exchange Rate Provider**. Курсы можна абнавіць на маманты ў любы час або nakрэStackNavigator на аўтаматычнае абнаўленне.

### Пазбегненне аўтаматычнай канверсіі для прадуктаў {#per-product-pricing-overrides}

Калі аўтаматычная канверсія не дастаткова точная, вы можаце ўсталяваць точную цану для канкрэтнай валюты на любы прадукт:

1. Адкраніце рэдактар прадукту (**Ultimate Multisite → Products → Edit**)
2. Зробіце скрол да секцыі **Currency Pricing**
3. Увядзіце точную цану для кожнай валюты, якую вы хочаце пераўзынчыць
4. Захаваць прадукт

Прадукты без пераўзынчэння выкарыстоўваюць канвертаваную цану з вашай асноўнай валюты.

### Пале выбару валюты пры рэгістрацыі {#currency-selector-signup-field}

Пале выбару валюты можна дадаць у ваш лад рэгістрацыі/аплатам, каб кліенты выбіралі пераважвае валюту падчас рэгістрацыі:

1. Праййдзіце па **Ultimate Multisite → Multi-Currency → Settings**
2. Актывуйце **Show currency selector on signup**
3. Па жаданню ўсталюйце пачатковую валюту або дапусціце геолокацыю вызначыць выбранае значэнне

Выбраная валюта захаваецца ў падпіску і выкарыстоўваецца для ўсіх наступных расліцованняў.

### Выяўленне валюты на аснове геолокацыі {#geolocation-based-currency-detection}

Калі актывавана выяўленне геолокацыі, аддон спалучае краіну адвідувача з валютай і аўтаматычна выбірае яе ў палі выбару валюты. Каб актываваць:

1. Праййдзіце па **Ultimate Multisite → Multi-Currency → Settings**
2. Актывуйце **Detect currency from visitor location**
3. Спалучайце краіны з валютамі ў табліцы **Geolocation Mapping**

Калі для краіны адвідувача няма спалучэння, выкарыстоўваецца асноўная валюта сеткі ў якасці запаснага варыянту.

## Часта задаваныя пытанні {#frequently-asked-questions}

### Што стане гады, калі провайдэра курсу валют няма? {#what-happens-if-an-exchange-rate-provider-is-unavailable}

Аддон праходзіць на апошнія паспяхова атрыманыя курсы. Калі курсы ніколі не атрымліваліся, ён праходзіць на ручныя курсы, якія вы наладзілі. Рэкамендуецца наладзіць ручныя курсы ў якасці страхавальнай сітка.

### Ці магу я выкарыстоўваць розныя провайдэры для розных валют? {#can-i-use-different-providers-for-different-currencies}

Нятучна — для ўсіх валют актыўны адзін провайдэра. Выкарыстоўвайце **Manual**, калі вам патрэбна поўная кантроль над кожнымі курсамі.

### Ці патрабуе геолокацыя асобнага плагіна? {#does-geolocation-require-a-separate-plugin}

Не патрабуе знешнага плагіна геолокацыі. Аддон выкарыстоўвае ўбудаваны WordPress-механізм выяўлення IP-адрэса ў краіне. Для больш высокай дакладнасці можна інтэграваць выкверэненне геолокацыі праз прадастаўленыя хукі фільтрацыі.

### Ці ўсталёўваюцца пераўзынчэнні прадуктаў да ці па податках? {#are-per-product-overrides-applied-before-or-after-tax}

Пераўзынчэнні прадуктаў ўсталёўваюць цану *пера* податках у мэтах валюты. Падатак вылічваюцца ад пераўзынчэння з выкарыстаннем вашай існуючай канфігурацыі податкаў.
