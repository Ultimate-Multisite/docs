# Loco Translate

Аддон Loco Translate для Ultimate Multisite дае магчымасць перакладу, спецыфічную для кожнага сайта ў вашай сетцы WordPress multisite. Калі вам калі-некалі трэба было, каб розныя сайты ў вашай сетцы мелі розныя пераклады аднаго інага плагіна або тэмы, гэты аддон — ідэальнае рашэнне.

**Што робіць гэты аддон**

Гэты аддон пазбежна інтэгруе Ultimate Multisite з папулярным плагінам Loco Translate, дазваляючы кожнаму субсайту ў вашай сетцы захаваць ўласныя карыстальніцкія пераклады. Нягледзячы на тое, ці вы ўпрацоўваеце платформу для некалькіх рэгіёнаў, абыт адрозныя моўныя с>)польнасці, ці вам проста патрэбна гнуткасць перакладу па ўсей сетцы, гэты аддон робіць гэта лёгка.

**Ідэальна для:**

* Бізнесу некалькіх рэгіёнаў з рознымі патрабаваннямі локалізацыі
* Навучальных устаноў, якія абслугоўваюць розныя с>)польнасці
* SaaS-платформаў, якія прапануюць многамоўную падтрымку
* Веб-агенцый, якія кіруюць некалькімі сайтамі кліентаў
* Любой сеткі multisite, якая патрабуе гнуткасці перакладу

## Устаноўка

1. Убедзіцеся, што Ultimate Multisite ўсталяваны і актываваны на сетцы.
2. Усталюйце і актывуйце плагін Loco Translate.
3. Загрузіце гэты аддон у каталог плагінаў.
4. На сетцы актывуйце аддон Loco Translate Integration.
5. Пачніце ствараць пераклады, спецыфічныя для сайта, праз Loco Translate на любы субсайт.

І гэта ўсё! Інтэграцыя працуе аўтаматычна без неабходнай дадатковай канфігурацыі.

## Часта задаваныя пытанні

## Ці патрабуецца ад канфігурацыі пасля ўстаноўкі?

Ня. Інтэграцыя працуе аўтаматычна. Як толькі аддон і Loco Translate будуць актыўныя, субсайты змогуць неадкладна пачынаць ствараць ўласныя карыстальніцкія пераклады.

## Што стане гады, калі субсайт не стварае карыстальніцкія пераклады?

Сайт аўтаматычна выкарыстоўае пераклады для ўсей сеткі ў якасці рэзервацыйнага варыянта. Нічога не зламаецца, і ўсё працуе пазбежна.

## Ці могуць субсайты атрымаць доступ да перакладаў для ўсей сеткі?

Звычайныя адміністратары сайта бачаць і рэдагуюць толькі пераклады ўласнага сайта. Суперадміністратары могуць атрымаць доступ як да перакладаў для ўсей сеткі, так і да перакладаў для канкрэтных сайтаў, залежна ад таго, які сайт яны праглядаюць.

## Ці працуе гэта з плагінамі і тэмамі?

Так! Карыстальніцкія пераклады спецыфічныя для сайта працуюць як для плагінаў WordPress, так і для тэмаў.

## Ці замедліць гэта маю сетцы multisite?

Ня. Аддон выкарыстоўвае эфектыўную загрузку файлаў, якая праверяе пераклады спецыфічныя для сайта толькі калі гэта неабходна. Нема заўважальнага ўплыву на прадукцыйнасць.

## Ці магу я skopyраваць пераклады для ўсей сеткі на канкрэтны сайт у якасці пачатковай кропкі?

Так! Пакуль вы рэдагуеце пераклады ў Loco Translate, вы можаце лёгка пачаць з перакладаў для ўсей сеткі і наладзіць іх для кожнага сайта.

## Ці працуе гэта з WooCommerce і іншымі плагінамі?

Абсалютна! Любы плагін або тэма, якая выкарыстоўвае стандартныя функцыі перакладу WordPress, працуе з карыстальніцкімі перакладамі спецыфічным для сайта.
