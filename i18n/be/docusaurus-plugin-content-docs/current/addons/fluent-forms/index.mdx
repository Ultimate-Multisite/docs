# Fluent Forms

Гэты ад-он дазваляе дадаць ліміты да Fluent Forms на Multisite Ultimate у прадуктах, канчатках або сайтах.
Fluent Forms выкарыстоўвае карыстальніцкія табліцы замест карыстальніцкіх пастыўтаў, таму звычайныя варыянты лімітавання не працуюць з Fluent Forms.

## Функцыі {#features}

- Дадаць ліміты для Fluent Forms або Conversational Fluent Forms
- Кліенты атрымліваюць прывітанны паведамленне, якое запрашае іх абнавіць свой план, калі яны дасягаюць ліміту.
