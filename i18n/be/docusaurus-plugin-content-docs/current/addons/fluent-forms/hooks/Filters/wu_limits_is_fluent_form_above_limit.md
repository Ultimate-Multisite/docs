# Фільтр: wu_limits_is_fluent_form_above_limit

Паطلعяе, ці дазваляецца ўказаны тып формы на гэтым плане. Дазваляе раробнікам плагінаў фільтраваць павернутую каштоўнасць.

## Паметры {#parameters}

| Назва | Тып | Апісан |
|------|------|-------------|
| $value | `bool` | Калі тып формы перавышае ліміт, ці не перавышае |
| $form_type | `string` | Тып формы, які праверка |
| $form_count | `int` | Бядучы лік форм |
| $quota | `int` | Дапушчальны квотаж |
