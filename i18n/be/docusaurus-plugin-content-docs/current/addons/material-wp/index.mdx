# Material WP {#material-wp}

Інтерфейс адмінастрацыі WordPress не змяняўся ўжо вельмі даўга. Але калісьці задумваліся, як выглядае адмінастрацыйны інтэрфейс у версіі Material Design?

Material WP — гэта тэма для адмінастрацыі, распрацаваная, каб даць вам гэты выгляд і адчуванне, дапамагаючы вам прадастаўляць сваім кліентам больш карыстальніка адпавядачны Dashboard.

## Функцыі {#features}

- Падтрымка карыстальніка CSS
- Простыя налады стылізацыі
- Паўторная парадкаванне меню
- Просты імпарт/экспорт

## Дакументацыя {#documentation}

**Карыстальніцкія налады адмінастрацыйнай панэлі з Material WP**

## Устаноўка {#installation}

1. Загрузіце 'material-wp' у каталог '/wp-content/plugins/'
2. Актывуйце плагін праз меню 'Plugins' у WordPress
