# Action: switch_network {#action-switchnetwork}

Выклікаецца, калі пераключаецца актыўны контекст сеткі.

## Паметрыкі {#parameters}

| Назва | Тып | Апісан |
|------|------|-------------|
| $new_network_id | `int` | ID сеткі, да якой пераключаюцца. |
| $old_network_id | `int` | ID сеткі, якая была актыўнай раней. |

- 1.3.0
