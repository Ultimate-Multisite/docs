# Фільтр: network_options_list

Фільтруе спіс сеткавых напрамленняў (options), якія залегіяць ад домену і шляху сеткі.

## Параметры

| Name | Type | Description |
|------|------|-------------|
| $network_options | `array` | Спіс іменаў сеткавых напрамленняў. |

### З

- 1.3.0
