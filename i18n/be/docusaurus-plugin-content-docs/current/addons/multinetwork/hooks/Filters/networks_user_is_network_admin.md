# Фільтр: networks_user_is_network_admin {#filter-networksuserisnetworkadmin}

Фільтруе сеткі, дзе карыстальнік з адміністратарам.

## Параметры {#parameters}

| Назва | Тып | Апісан |
|------|------|-------------|
| $ | `array\|bool` | Спіс ID сеткі або `false`, калі для карыстальніка няма сеткі. |
| $ | `int` | ID карыстальніка, для якіх трэба павернуць сеткі. |
