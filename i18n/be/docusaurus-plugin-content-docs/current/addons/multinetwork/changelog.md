# Журнал змены для многіх сетак

### 1.3.0
* Ноўнасць: Прэвіюар шаблонаў сетак — прагляд і прэглед шаблонаў сетак на жывым панілі, перш аплаты.
* Ноўнасць: Выбар шаблону сеткі на касёр — кліенты могуць выбраць шаблон сеткі падчас працэсу рэгістрацыі на касёр.
* Ноўнасць: SSO па магічным спасылку паміж сеткамі — кліенты могуць атрымаць доступ да падсетак безпрапынна праз аўтэнтыкацыю па магічным спасылку.
* Ноўнасць: Клонаванне медыя сетак — медыя сайту правільна дубліруюцца пры клонаванні сетак з шаблону.
* Выпраўленне: Клонаванне сетак цяпер працуе без аддонка мальта-тэнансі; прывілегіі суперадміністратара аднаўляюцца пасля клонавання.
* Выпраўленне: Стваৰэнне сетак цяпер сумяшчальна з WooCommerce і ўсімі шлюзамі аплаты.
* Выпраўленне: Партальны нумар цяпер уключаецца ў генерацыю рэзерваўных доменных імемен.
* Выпраўленне: CSS для выбару шаблону перамешчаны ў запынены (enqueued) стыль, каб правільна адбыць на ўсіх тэмах.
* Выпраўленне: Захавана адсутнасць ключа `network_id` у аб'ёме zapytaniya сайту.
* Выпраўленне: Прыверка правіль на сетку перад пераключэннем кантексту zapytaniya zapякоўвае памылкі ў ўласцівасцях табліцы `wpdb`.
