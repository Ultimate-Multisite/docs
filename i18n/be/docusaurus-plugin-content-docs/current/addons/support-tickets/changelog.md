# Журнал змены (Changelog) для Тікет-сістэмы падтрымкі

### 1.0.4 - 2026-05-05 {#104---2026-05-05}
* Палепшана: Выбыў паказчык `vendor/` з адліку Git (ужо ахоўваецца праз .gitignore), што зменшыла памер рэпазіторыя.
* Палепшана: Пратэсціравана да WordPress 7.0.

### 1.0.3 - 2026-05-01 {#103---2026-05-01}
* Выправлена: Ахаваецца нулявыя значэнні ў сеттары для нулявых тэматычных мадэляў.
* Выправлена: Усе тэматы ўстаноўлены ў глобальныя, сеткавыя табліцы для надлежай наймультысайт падтрымкі.
* Выправлена: Схавана паля прапераўнікаў і выправлена пераадрэктыванне на пусты экран на форме "новая тэматы" для кліентаў.
* Выправлена: Замяшчае выклікі `add_meta()` без вызначэння на правільныя `update_meta()` для захоўвання метададзеных.
* Выправлена: Замяшчае праверку кабілетата, які не зарэгістраваны, на правільную `wu_view_all_support_tickets`.
* Выправлена: Дададзеныя адсутныя AJAX-апрацоўнікі для статусу тэматы, прызначання і дзеяння "швидка-рэдагаванне".
* Выправлена: Выпраўлена назва метаду для выяўлення адказу прапераўніка ў пагледжанні тэматы.
* Выправлена: Карэкцыя дублікатаў апрацоўнікаў адказаў і вырівнанне назваў дзеянняў nonce.
* Выправлена: Дададзена адсутная фронтэнд-пагледжання для шорткада [wu_submit_ticket].
* Выправлена: Дададзена адсутнае пале `user_id` і выпраўлена метад `Support_Ticket::get_user_id()`.
* Выправлена: Выбыў зайвы двойны апостаў у тэгу варыянту фільтрацыі прыярытэта.
* Выправлена: Дададзена панялічжанне кіравання тэматы для адміністрацыі сеткі для супер-адмінаў.
* Палепшана: Карэкцыя адмінавай CSS ў адзін вынешні стыль.
* Палепшана: Выбыў падменю "Налады" з адмінавага меню падсайта.
* Палепшана: Умоўнае загружаванне фронтэнд-актываў толькі на страных тэматы падтрымкі.
* Палепшана: Прыпыняе працу аўтозагрузніка плагіна, калі аўтозагрузнік карэнь Bedrock ужо загрузіў залежнасці.

### 1.0.2 - 2025-12-11 {#102---2025-12-11}
* Дададзена: Функцыянальнасць поўнага прагляду тэматы для адміна і фронтэнда.
* Дададзена: AJAX-апрацоўнік для адпраўлення адказаў на тэматы.
* Дададзена: Падтрымка функцыянальнасці адказаў на тэматы з надлежай апрацоўкай формы.
* Дададзена: Надлежачна выяўленне паведамленняў для адпраўлення тэматы і адказаў у адміна.
* Дададзена: Аўтаматычная прывязка тэматы да бягучага карыстальніка пры самоадправленні тэматы кліентам.
* Дададзена: Палепшаная бяспека, каб zapрэчыць кліентам перавыканне ўласніцтва тэматы.
* Дададзена: Адсутныя дапаможнікі функцыі (wu_format_date, wu_user_can_view_ticket і г.д.).
* Дададзена: Надлежачнае спампоўленне і апрацоўванне файлаў.
* Дададзена: Сістэма паведамленняў па электроннай пошце для адказаў на тэматы і змены статусу.
* Выправлена: Форма адказа на тэматы ўключае цяпер неабходны ID тэматы.
* Выправлена: Праблемы з прыкрэйленнем паведамленняў у адмінавагі пагледжанні тэматы.
* Выправлена: Синтаксісныя памылкі ў функцыях тэматы падтрымкі.
* Палепшана: Структура базы дадзеных з надлежай апрацоўкай запыту і класамі схемы для адказаў і прыкрэсных файлаў.
* Палепшана: Міграцыя на рэальные ўметнічыкі (properties) замест атратываў (attributes) для тэматычных мадэляў.

### 1.0.1 - 2025-09-28 {#101---2025-09-28}

* Выправлена: Невялікія выпраўленні памылак і палепшыльніцтва.
