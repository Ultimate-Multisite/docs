# Мультикастова система Hostinger {#hostinger-multi-tenancy}

Ultimate Multisite: Multi-Tenancy 1.2.0 дадае функцыянальнасць мультикастовай сістэмы Hostinger, што дазваляе хоสต์ваныя домены ўчаствовать у працы па стварэнні суверэйнага асобнага прастору (tenant) паралельно з існуючай інтэграцыяй мапіравання домейна Hostinger.

Выкарыстоўвайце цю функцыю, калі вы карыстаецеся доменамі асобных прастораў (tenants) і ізаляванай інфраструктурай праз hPanel Hostinger.

## Заметкі па наладцы {#setup-notes}

1. Наладзіце асноўную інтэграцыю з Hostinger у **Ultimate Multisite > Settings > Host Integrations**.
2. Падтвердзіце, што токен API Hostinger можа адпраўляць кіраванне ці для цілевай домены або поддомена.
3. Акраніруйце (Enable) аддон Multi-Tenancy.
4. Наладзіце стратэгію ізаляцыі прастораў перад публікацыяй прастора.
5. Выконайте ворхладнавы працэс пераасунення (migration verification workflow) перад пасылкай продакцыйным трафікам у прастор.
