# Размяшчэнне рэкламы ў вашай сетцы з Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector — гэта **бесплатны дадатак**, які дае вам інструменты для размяшчэння рэкламы на вашых планах, яшчэ з іх адкаднага экрану рэдактара.

Пасля ўстаноўкі Ultimate Multisite: Ad Injector зайдзіце на свой экран _Plugins_, каб актываваць яго.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Націсніце кнопку Network Activate_

Як толькі дадатак будзе актываваны, у мета-бокс _Advanced Options_ на экране рэдактара плана з'явіцца новая опцыя пад назвай _Ad Settings_.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_Цяпер вы можаце рэдактаваць налады рэкламы для кожнага плана_

На гэтай вкладцы вы зможаце дадаць коды рэкламы для пасты і страных у фронт-энды, да і пасля кантэнту.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Пасля ўстаўлення кода рэкламы, ён з'явіцца на пастах і страных гэтага канкрэтнага плана._

Вы таксама можаце ўключыць рэкламу ў шапку Адмін-страніц для карыстальнікаў гэтага плана.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Зайдзіце ў опцыю Admin Panel Ad Code, калі хочаце адбыць рэкламу на бэк-энд карыстальніка_

### Па за межамі рэкламы

Вы таксама можаце выкарыстоўваць Ultimate Multisite: Ad Injector, каб прапанаваць падтрымку для планаў высокага ўзроўню.

Проста зайдзіце ў опцыю _Admin Panel Ads_ і дадайце HTML-код для вашага інструмента падтрымкі.
