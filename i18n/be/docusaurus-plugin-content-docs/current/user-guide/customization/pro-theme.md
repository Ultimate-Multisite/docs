# Кастымізацыя вашай адміннай панэлі з PRO Theme

Часта, калі мы працуем з кліентамі, нам трэба прадастаўляць ім кастымізаваны досвед у адмін-панэлі (dashboard) WordPress. Калі вы шукаеце больш прафесійны выгляд і пачуццё, PRO Theme можа вам дапамагчы.

PRO Theme — гэта адмін-тэма, распрацаваная, каб дапамагчы вам пры кастымізацыі адмін-панэлі (Dashboard) вашага кліента. А яна таксама ўваходзіць у ядро прэміум-плагінаў Ultimate Multisite.

Пасля ўстаноўкі зайдзіце на сваю старонку _Plugins_ і націсніце, каб актываваць яго на вашай Сеці (Network).

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Актывуйце яго на ўсім уровне Сеці на старонцы Plugins_

_**ПАМЫНКА:**_ _Калі вы яго проста ўстаніруеце, ён з'яўляецца для вашага кліента як варыянт, які можна адзначыць, каб дадаць тэму толькі для гэтага карыстальніка._

І цяпер ваша адмін-інтерфейс цалкам перароблены! У вас таксама з'яўляецца новае меню ў левай слупцы, названае PRO Theme.
