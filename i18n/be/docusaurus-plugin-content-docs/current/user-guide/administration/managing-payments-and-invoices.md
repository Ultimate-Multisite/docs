# Кіраванне Плацментамі і Паўторнымі Арахуваннямі

## Налады Плацментаў {#payment-settings}

Перш чым вы пачнеце прымаць плацменты, вам трэба наладзіць налады, звязаныя з плацментамі. Айдзіце ў **Ultimate Multisite → Settings** і націсніце на тэab **Payment**.

![Payment settings tab](/img/admin/settings-payments-top.png)

Вось паўнавартасны прагляд старонкі налад для плацментаў:

![Payment settings full page](/img/admin/settings-payments-full.png)

### Агульныя Налады Плацментаў {#general-payment-options}

У агульных налад вы можаце наладзіць:

- **Currency** — Асноўная валюта, якая выкарыстоўваецца для транзакцый
- **Currency Position** — Калі знаходзіцца сімвал валюты (пера/пасля сумы)

![Payment gateway settings](/img/admin/settings-payments-options.png)

### Плацменныя Варота (Payment Gateways) {#payment-gateways}

Ultimate Multisite падтрымлівае некалькі плацменных вартаў. Вы можаце актываваць і наладзіць кожнае з іх на тэab Налад для плацментаў.

![Payment gateway configuration](/img/admin/settings-payments-gateways.png)

Даступныя вартаў уключаюць:

- **Stripe** — Плацменты карたй па вартай Stripe
- **PayPal** — Плацменты PayPal
- **Manual** — Для офлайн або спецыяльнай апрацоўкі плацментаў

Каждая варта ўласны спічкванне для налад, дзе вы ўводзіце API ключы і іншыя налады.

![Additional gateway settings](/img/admin/settings-payments-gateways-2.png)

### Рэжым Sandbox (Sandbox Mode) {#sandbox-mode}

Вы можаце актываваць **Sandbox Mode**, каб пратэставаць ваша інтэграванне плацментаў, перш чым вы запусціце яго ў практыку. Калі рэжым sandbox актыўны, не будзе зроблены жодны сапраўдны злік.

## Прагляд Плацментаў {#viewing-payments}

Айдзіце на старонку **Payments** унутры Ultimate Multisite, каб паглядзець усе транзакцыі па ўсей вашай сетцы.

![Payments list](/img/admin/payments-list.png)

Вы можаце фільтраваць плацменты па статусе (завершаны, у чаканні, не паспяховы, адказны) і шукаць канкрэтныя транзакцыі.

Націсніце на плацмент, каб паглядзець яго паўнавартаныя дэталі, уключаючы пастаянныя пачаткі, звязаны канчатк, інфармацыю кліента і дадзеныя плацменнага варта.

## Паўторныя Арахуванні (Invoices) {#invoices}

Ultimate Multisite можа аўтаматычна генерываць паўторныя рахункі пасля плацментаў. Вы можаце наладзіць шаблон паўторнага рахунка і фармату нумерацыі з Налад для плацментаў.

Варыянты налад паўторнага рахунка ўключаюць:

- **Назва і адрас кампаніі**, якія адлюстраваныя на паўторных рахунках
- **Формат і паслідок нумерацыі** паўторнага рахунка
- **Лого**, якое адлюстраваецца ў заголоўкі паўторнага рахунка
- **Спецыяльны тэкст у падпіску** для ўмоваў, намет або правяднай інфармацыі

Каб наладзіць шаблон паўторнага рахунка, айдзіце ў **Ultimate Multisite → Settings → Payment** і знайце налады, звязаныя з паўторнымі рахункамі.
