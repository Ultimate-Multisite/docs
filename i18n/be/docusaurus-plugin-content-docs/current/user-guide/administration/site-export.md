# Экспорт сайта

Адміністрацыйная адкарона **Экспорт сайта** дазваляе адміністратарам сеткі спакавацца адзін сайт або ўсю сетку ў выягодны для загрузкі архів для міграцыі, рэзервацыі ці перадачы праекта.

## Экспорт аднаго сайта {#exporting-one-site}

Зайдзіце ў **Ultimate Multisite > Site Export** і выберыце **Generate new Site Export**. Выберыце субсайт, які хочаце экспортаваць, а потым паказаць, ці павінен архів ўключаць загрузкі (uploads), plugins і themes.

Пасля завяршэння экспорту загрузіце ZIP з спісу **Existing Exports**. Экспорт ZIP-файлы цяпер уключаюць самозапускальны `index.php` і `readme.txt`, таму архів можна загрузіць на свежы хаст і запусціць, не ўсталёўваючы перш аддечны plugin для імпорту.

## Экспорт усёй сеткі {#exporting-the-whole-network}

Выкарыстоўвайце **Network Export** на адкароне Site Export, калі вам патрэбен адзін архів, які ўтрымлівае ўсе субсайты ў сетцы. Гэта вельмі карысна перад міграцыяй хаста, практыкаваннем аднаўлення пасля катастрофы або пабудовай стэйдзінга, дзе кожны субсайт павінен «путоваць» разам.

Паколькі экспорт сеткі можа быць значна больш вялікім, чым экспорт аднаго сайта, прагоніце яго ў час невялікай трафіка і пацверце, што мэтавае сховішча мае дастаткова вольнага месца для загрузкі, plugins, themes і сгенераваных ZIP-файлаў.

### Паўвязкі імпорту сеткі (Network Import Bundles) {#network-import-bundles}

Пакуль Ultimate Multisite 2.12.0, Site Exporter можа ствараць **network import bundles** — спецыялізаваныя архівы, распрацаваныя для аптымізаванага аднаўлення сайта на ўсім уровне сеткі. Network import bundle змяшчае ўсе неабходныя файлы і метададзеныя для аднаўлення некалькіх сайтаў на новую сеткавую ўстаноўку.

#### Стваৰэнне Network Import Bundle {#generating-a-network-import-bundle}

1. Зайдзіце ў **Ultimate Multisite > Site Export**
2. Націсніце **Generate new Network Export**
3. Выберыце **Network Import Bundle** як тып экспорту
4. Паказаць, ці ўключаць загрузкі, plugins і themes
5. Націсніце **Generate**
6. Загрузіце bundle ZIP з спісу **Existing Exports**

#### Аднаўленне з Network Import Bundle {#restoring-from-a-network-import-bundle}

Калі вам трэба аднавіць сайты з network import bundle:

1. Усталюйце Ultimate Multisite на ваш мэтавы хаст
2. Завершыце мастацкую ўстановку мвулітэсайту
3. Зайдзіце ў **Ultimate Multisite > Site Export** на новай сетцы
4. Націсніце **Import Network Bundle**
5. Загрузіце файл ZIP network import bundle
6. Выконвайце інструкцыі па імпорце на экране
7. Процес імпорту аднавіць усі сайты, іх кантэнт і канфігурацыі

Network import bundles захаValueOf:
- Кантэнт сайта (публікацыі, страніцы, кастомныя типы публікацый)
- Налады і опцыі сайта
- Ролі і правільдоступ карыстальнікаў
- Plugins і themes (калі яны ўключаны ў bundle)
- Загрузкі медыя (калі яны ўключаны ў bundle)
- Кастомныя базы дадзеных і дадзеныя

## Устаноўка самозапускальнага export ZIP {#installing-a-self-booting-export-zip}

Калі вам трэба аднавіць самозапускальны ZIP на свежы хасце:

1. Загрузіце вміст экспортаванага ZIP на мэтавы корань вебасайта (web root).
2. Адкраніце загружаны `index.php` у браўзер.
3. Выконвайце інструкцыі па ўсталёўцы, якія з'яўляюцца часткай спакаванага экспорта.
4. Пагледзьце `readme.txt` ў bundle, каб знайсці спецыфічныя для экспорту заўвагі, перш чым выдаліць عвязальныя файлы.

Для дэталяў па ўсталёўцы і імпорце для адналення аддоткаў (addon-specific installation and import details) спадывіце [Site Exporter addon documentation](/addons/site-exporter).

Для інструмента аднаго сайта, дадаванага ў Ultimate Multisite 2.9.0, спадывіце [Export & Import](/user-guide/administration/export-import).
