# Формы кантактаў Site Builder

Superdav AI Agent v1.10.0 дадае магчымасць ствараць формы кантактаў непасрэдна з чат-інтерфейсу выкарыстоўваючы агента Site Builder. Гэта дазваляе вам дадаць функцыянальныя формы кантактаў на любыя страўны, не пакідаjąчы чату.

## Агляд

Магчымасць стварэння формы кантактаў Site Builder (`create_contact_form`) аўтаматычна выяўляе ўсталяваны плъгін для форм і стварае форму кантактаў, выкарыстоўваючы ўласныя магчымасці гэтага плъгіна. Падтрымкаваныя плъгіны для форм уключаюць:

- Contact Form 7
- WPForms
- Fluent Forms
- Gravity Forms

Асістэнт аўтаматычна выбірае найлепшы доступны плъгін для форм на вашым сайце і генеравае форму кантактаў, адыладжаную пад патрэбы яго.

## Ствараэнне формы кантактаў

Каб стварыць форму кантактаў выкарыстоўваючы Site Builder:

1. Адкраніце чат-панель **Gratis AI Agent** у адмініструванні WordPress.
2. Перайце ў агента **Site Builder**, націснуўшы на значок агента у загалоўным рэжыме чату.
3. Апішыце форму кантактаў, якую вы хочаце стварыць. Напрыклад:

   > "Дадай форму кантактаў на страўну 'Кантакты' з полямі для імя, электронная адрас, паведамленне і тэлефон."

   Або проста:

   > "Стварыць форму кантактаў для страўны 'Кантакты'."

4. Site Builder згенэруе форму кантактаў і паверне шорткад, які гатовы для ўбудоўкі.

## Выкарыстанне згенераванага шорткада

Пасля таго, як Site Builder стварыць форму кантактаў, ён паверне шорткад (напрыклад, `[contact-form-7 id="123"]`). Вы можаце:

1. **Убудаваць яго на страўне або ў паст** — Скопіруйце шорткад і ўстаўце яго на любы страўны або ў паст выкарыстоўваючы рэдактар блокаў або класічны рэдактар.
2. **Дадаць яго праз Site Builder** — Папросіце Site Builder аўтаматычна дадаць форму на канкрэтную страўну:

   > "Дадай форму кантактаў на страўну 'Кантакты'."

3. **Выкарыстоўваць яго ў шаблоне** — Калі вы ведаеце PHP, вы можаце дадаць шорткад у файл шаблону тэмы.

## Наладка формы кантактаў

Пасля таго, як Site Builder стварыць форму кантактаў, вы можаце далей яе наладзіць:

### Праз чат-інтерфейс

Папросіце Site Builder адрэдагаваць форму:

> "Актуалізуй форму кантактаў, каб дадаць поле 'Тэма' і зрабіць поле 'Паведамленне' абавязковым."

Site Builder актуалізуе форму і паверне абноўлены шорткад.

### Праз адмін-панель плъгіна для форм

Вы таксама можаце наладзіць форму непасрэдна ў настройках вашага плъгіна для форм:

1. Зайдзіце ў **Contact Form 7** (або ў ваш ўсталяваны плъгін для форм) у адмініструванні WordPress.
2. Знайдзіце форму, створаную Site Builder.
3. Адрэдагуйце поля формы, правілыя валідацыі і паведамленні па электроннай пошце па патрабаванні.

## Плъгіны для форм і сумяшчальнасць

Site Builder аўтаматычна выяўляе, які плъгін для форм усталяваны на вашым сайце, і выкарыстоўвае яго для стварэння формы кантактаў. Калі ўсталяваны некалькі плъгінаў для форм, Site Builder прымагвае прыярытэт у наступным парадку:

1. Contact Form 7
2. WPForms
3. Fluent Forms
4. Gravity Forms

Калі не усталяваны жодны з гэтых плъгінаў, Site Builder рэкамендуе ўсталяваць адзін, перш чым ствараць форму кантактаў.

## Паведамленні па электроннай пошце

Формы кантактаў, створаныя Site Builder, па спецыятацыі адпраўляюць паведамленні па электроннай пошце адміністратары сайту. Вы можаце наладзіць адрас электроннай пошты атрымальніка і тэкст паведамлення:

1. Зайдзіце ў адмін-панель вашага плъгіна для форм.
2. Знайдзіце форму, створаную Site Builder.
3. Адрэдагуйце налады паведамлення па электроннай пошце.

Для больш падрабязнай інфармацыі спасылайцеся на дакументацыю вашага плъгіна для форм:

- [Contact Form 7 Email Notifications](https://contactform7.com/docs/)
- [WPForms Email Notifications](https://wpforms.com/docs/)
- [Fluent Forms Email Notifications](https://fluentforms.com/docs/)
- [Gravity Forms Email Notifications](https://docs.gravityforms.com/)

## Прыкладныя выпадкі выкарыстання

### Простая форма кантактаў

> "Стварыць простую форму кантактаў з полямі для імя, электронная адрас і паведамленне."

### Форма ў некалькі крокаў

> "Стварыць форму кантактаў з першым крокам для кантактаў і другим крокам для паведамлення і пераканаўчага спілкування."

### Форма з кандыцыянальнай лагікай

> "Стварыць форму кантактаў, якая паказвае розныя поля залежна ад выбару карыстальніка ў выпадаўным меню."

### Форма з загрузкай файла

> "Стварыць форму кантактаў, якая дазваляе карыстальнікам загрузіць файл або прыдатак."

:::note
Ствараэнне формы кантактаў доступна ў Superdav AI Agent v1.10.0 і па-сля. Для выкарыстання гэтай магчымасці павінен быць актыўным агент Site Builder.
:::
