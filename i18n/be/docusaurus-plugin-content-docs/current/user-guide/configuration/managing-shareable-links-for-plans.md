# Управление пажартаваўчымі спасылкімі для планаў {#managing-shareable-links-for-plans}

Ultimate Multisite дае вам магчымасць адпраўляць спасылкі на планы вашаму кліентам. Для гэтага вам трэба зайсці на любую адрэдагаванне страночку плана і націснуць кнопку _Click to copy the Shareable Link_.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_На страночцы рэдагавання плана вы знайдзеце кнопку для стварэння пажартаваўчых спасылкіў_

Зверніце ўвагу, што гэты механізм выкарыстоўвае пачатковую варту спілкування для стварэння спасылкі. Гэта азначае, што калі ваша пачатковая варта — гэта месяц, напрыклад, у выніку вы атрымаеце спасылку, якая выглядае так: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Калі вы хочаце, каб спасылка пажартаваўчым па спілкуванні па задынэ, вам трэба дадаць 3 у канцы.

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Каб выбраць план _premium_ на гадачны графік, вам трэба дадаць 1, а затым слова:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Па-заўвага, што вы не можаце мець толькі адну спасылку для розных вартаў спілкування. Вам трэба адпраўляць адну канкрэтную спасылку для кожнай вашый варты спілкування.

Як толькі вы адпраўляеце спасылку сваім кліентам, яна перанаправіць іх на першы этап вашага працэсу рэгістрацыі.
