# Вбудаваныя Агенты

Gratis AI Agent v1.9.0 пастаўляе пяць вбудаваных агентаў. Кожны з іх папярэдні канфігураваны з сканцёнаванай набор тэхналогій (tools), спецыяльным системным прамтам (system prompt) і пачатковыми прапаنوкамі, якія адпавядаюць частым задачам у гэтай сферы. Перайход паміж агентамі змяняе тое, што можа рабіць асістэнт і як ён адказвае — і ўсё гэта без вашага канфігуравання.

## Што гэта за Агент?

Кожны агент — гэта названы профіль канфігурацыі, які спалучае:

- **Tools** — здольнасці, якія агент мае права выкарыстоўваць (напрыклад, Content Writer мае доступ да здольнасцей стварэння пасты; Design Studio мае доступ да здольнасцей CSS і theme.json)
- **System prompt** — інструкцыі, якія вызначаюць тональнасць, прыярытэты і абмежаванні агента
- **Suggestions** — папярэджанне напісаныя прамты, якія знаходзяцца ў чат-інтерфейсе, каб дапамагчы вам хутка пачаць працаваць

## Як выбраць Агента

1. Адкраніце панэль **Gratis AI Agent** у бокавой панэлі адміні strata WordPress.
2. Націсніце на **ікану агента** у верхнем левым кутку загаłówка чату (ікана змяняецца, паказваючы актыўнага агента).
3. Адкрыецца **Agent Picker** у выглядзе накладкі-табліцы. Кожны агент пералічваецца з яго іканай, назвай і адным-радковым апісаннем.
4. Націсніце на радкі агента, каб актываваць яго. Загалывык чату неадкладна абнаўляецца.

Вы таксама можаце пераключаць агентаў падчас размовы — системны прамт новага агента ўступае ў сілу з наступнага паведамлення.

## Пяць Вбудаваных Агентаў

### Content Writer

**Сканцэнтраванасць:** Стваৰэнне і рэдагаванне пастаў, страных і формы кантактаў.

**Даступныя tools:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`

**Што ён робіць добра:**
- Напісаць і апублікаваць блог-пасты па брыфе або планаванні
- Ствараць патыкі страных для новай сайта
- Будаваць формы кантактаў і запытаў
- Устаноўваць асноўныя імавыя файлы па спасылцы або па пошуку

**Пачатковыя прапановы:**
- *Напішыць блог-паст на 500 словаў пра перавагі WordPress multisite.*
- *Стварыць странец "Пра нас", "Паслугі" і "Кантакты" і апублікаваць іх.*
- *Дадаць форму запыту на заброніраванне на странец "Кантакты".*

---

### Site Builder

**Сканцэнтраванасць:** Стваৰэнне сайта ад пачатку да канца па сапраўдным прамце.

**Даступныя tools:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`

**Што ён робіць добра:**
- Генераваць планаванне стварэння сайта ў некалькіх фазах для апісанага бізнесу
- Выконваць кожную фазу самастойна — структура, кантэнт, навігацыя, дызайн
- Адновіраць працэс памылках падчас планавання без неабходнасці ручных дзеянняў
- Устаноўваць рэкамендаваныя plugins ў кантэксце стварэння
- Ствараць формы кантактаў непасрэдна з чат-інтерфейсу (Superdav AI Agent v1.10.0+)

**Пачатковыя прапановы:**
- *Стварыць сайт-портфоліё для мастака з пастом-галерайкай, страной заброніравання і фармай кантактаў.*
- *Стварыць сайт рэстарану з онлайн-меню, гадзіннікам працы і фармай запыту на заброніраванне сталі.*
- *Наладзіць сайт фрылансавага кансультанта з странам паслуг, секцыяй портфоліё і блогом.*
- *Дадаць форму кантактаў на странец "Кантакты" выкарыстоўваючы Site Builder.*

---

### Design Studio

**Сканцэнтраванасць:** Візуальнае касміванне — колеры, тыпографія, CSS і патэрныкі блокаў.

**Даступныя tools:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Што ён робіць добра:**
- Выкарыстоўваць названыя тэматычныя прасеты (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- Дыктэраваць глобальную тыпографіку і палітра колераў праз theme.json
- Встаўляць кастомны CSS для перакрыццяў, спецыфічных для брэнда
- Зрабіць скріншот страных і правесці іх на дызайнныя праблемы

**Пачатковыя прапановы:**
- *Выкарыстоўваць прасет warm-editorial, а потым ўсталяваць асноўны колер на #2d6a4f.*
- *Зрабіць скріншот галоўнай старонкі і сказаць мне, што б вы палепшылі.*
- *Стварыць перавыкавальны патэрнак гера-блоку з фонавым малюнкам на ўсю шырыню і цэнтрававаным загалізаваннем.*

---

### Plugin Manager

**Сканцэнтраванасць:** Выяўленне, устаноўка і кіраванне WordPress plugins.

**Даступныя tools:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Што ён робіць добра:**
- Рэкамендаваць лепшы plugin для апісаванага выкарыстання
- Устаноўваць пакіеты здольнасцей з рэгістры
- Прагляд даступнага каталогу здольнасцей па катэгорыях

**Пачатковыя прапановы:**
- *Які лепшы plugin для каталогу абонентства?*
- *Усталяваць пакіет здольнасцей WooCommerce.*
- *Паказаць мне ўсе даступныя пакіеты здольнасцей ecommerce.*

---

### Support Assistant

**Сканцэнтраванасць:** Адказванне на пытанні пра кантэнт сайта, налады і канфігурацыю WordPress.

**Даступныя tools:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Што ён робіць добра:**
- Пашук бягучых налад і опцый сайта
- Выказванне, якія post types, taxonomies і menus наладжаваны на сайце
- Адказванне на пытанні "што робіць гэтая наладка?" праз чытанне жывых значэнняў
- Выступленне ў ролі рэжыму-аператара дыягнастыкі, які не змяняе нічога, перш чым што-небудь змяняць

**Пачатковыя прапановы:**
- *Які plugins і налады зараз актыўныя на гэтым сайце?*
- *Вылічыць усе кастомныя post types, зарэгістраваныя на гэтым сайце.*
- *Які навігацыйныя меню існуюць і куды яны прызначаны?*

## Касміванне Агентаў

Кажны вбудаваны агент можна пашырыць або замяніць праз фільтр `gratis_ai_agent_agents`.

### Дадаванне кастомнага системнага прамту да існуючага агента

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### Рэгістрацыя новага агента

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

Новы агент з'яўляецца ў Agent Picker неадкладна пасля прагона фільтра.

### Выбынванне вбудаванага агента

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
