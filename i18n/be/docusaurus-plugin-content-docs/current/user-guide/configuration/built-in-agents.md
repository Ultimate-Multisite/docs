# Убудаваныя агенты

Gratis AI Agent v1.9.0 пастаўляецца з пяццю ўбудаванымі агентамі, кожны з якіх загадзя наладжаны з мэтанакіраваным наборам інструментаў, адаптаваным сістэмным запытам і пачатковымі прапановамі, падабранымі пад распаўсюджаныя задачы ў гэтай вобласці. Пераключэнне паміж агентамі змяняе тое, што памочнік можа рабіць і як ён адказвае — без якой-небудзь канфігурацыі з вашага боку. Superdav AI Agent v1.18.0 можа дадаваць у гэтыя працоўныя працэсы інструменты з улікам раскладу, запісы напамінаў, шлюзы зацвярджэння і SMS-апавяшчэнні, калі адпаведныя інтэграцыі наладжаны.

## Што такое агент? {#what-is-an-agent}

Кожны агент — гэта названы профіль канфігурацыі, які аб’ядноўвае:

- **Інструменты** — магчымасці, якія агенту дазволена выклікаць (напрыклад, аўтар кантэнту мае доступ да магчымасцей стварэння запісаў; дызайн-студыя мае доступ да магчымасцей CSS і theme.json)
- **Сістэмны запыт** — інструкцыі, якія задаюць тон, прыярытэты і абмежаванні агента
- **Прапановы** — загадзя напісаныя запыты, якія паказваюцца ў інтэрфейсе чата, каб дапамагчы вам хутка пачаць

## Доступ да выбару агента {#accessing-the-agent-picker}

1. Адкрыйце панэль **Gratis AI Agent** у бакавой панэлі адміністратара WordPress.
2. Націсніце **значок агента** ў левым верхнім куце загалоўка чата (значок змяняецца, каб адлюстроўваць актыўнага агента).
3. **Выбар агента** адкрываецца як накладка з формай-табліцай. Кожны агент паказаны са сваім значком, назвай і апісаннем у адзін радок.
4. Націсніце радок агента, каб актываваць яго. Загаловак чата абнаўляецца адразу.

Вы таксама можаце пераключаць агентаў пасярод размовы — сістэмны запыт новага агента пачынае дзейнічаць з наступнага паведамлення.

## Пяць убудаваных агентаў {#the-five-built-in-agents}

### Аўтар кантэнту {#content-writer}

**Фокус:** Стварэнне і рэдагаванне запісаў, старонак і кантактных форм.

**Даступныя інструменты:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`. Калі ўключаны інтэграцыі Superdav AI Agent v1.18.0, для зацверджаных працоўных працэсаў таксама могуць быць даступныя наладжаны кантэкст календара, шлюзы зацвярджэння, напаміны і інструменты SMS-апавяшчэнняў.

**Што ён робіць добра:**
- Стварэнне чарнавікоў і публікацыя запісаў блога з брыфа або плана
- Стварэнне пакетаў мэтавых старонак для новага сайта
- Стварэнне кантактных форм і форм запытаў
- Устаноўка галоўных выяў для запісаў з URL або пошуку
- Стварэнне чарнавікоў паведамленняў для далейшага кантакту пасля падзеі з наладжанага кантэксту Google Calendar, затым прыпыненне для зацвярджэння перад адпраўкай апавяшчэнняў

**Пачатковыя прапановы:**
- *Напішыце запіс у блог на 500 слоў пра перавагі WordPress multisite.*
- *Стварыце старонкі «Пра нас», «Паслугі» і «Кантакты» і апублікуйце іх.*
- *Дадайце форму запыту браніравання на старонку «Кантакты».*
- *Падрыхтуйце чарнавік напаміну для ўдзельнікаў заўтрашняй наладжанай падзеі календара і пачакайце зацвярджэння перад яго адпраўкай.*

### Канструктар сайта {#site-builder}

**Фокус:** Скразное стварэнне вэб-сайта з аднаго запыту.

**Даступныя інструменты:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`. З Superdav AI Agent v1.18.0 наладжаныя інструменты кіраванага сэрвісу, зацвярджэння, напамінаў, календара і SMS могуць быць даступныя там, дзе адміністратары іх уключаць.

**Што ён робіць добра:**
- Генерацыя шматфазнага плана стварэння сайта для апісанага тыпу бізнесу
- Аўтаномнае выкананне кожнай фазы — структура, кантэнт, навігацыя, дызайн
- Аднаўленне пасля памылак пасярод плана без неабходнасці ручнога ўмяшання
- Усталяванне рэкамендаваных плагінаў як часткі зборкі
- Стварэнне кантактных форм непасрэдна з інтэрфейсу чата (Superdav AI Agent v1.10.0+)
- Каардынацыя напамінаў пра запуск або далейшага кантакту з удзельнікамі без дублявання апавяшчэнняў, калі ўключаны шлюзы зацвярджэння і запісы напамінаў

**Пачатковыя прапановы:**
- *Стварыце сайт партфоліа фатографа з тыпам запісаў галерэі, старонкай браніравання і кантактнай формай.*
- *Стварыце вэб-сайт рэстарана з анлайн-меню, гадзінамі працы і формай запыту браніравання століка.*
- *Наладзьце сайт фрыланс-кансультанта са старонкамі паслуг, раздзелам партфоліа і блогам.*
- *Дадайце кантактную форму на старонку «Кантакты» з дапамогай канструктара сайта.*
- *Пасля зацвярджэння кантрольнага спіса запуску сайта адпраўце SMS-напамін наладжанаму кантакту зацікаўленага боку.*

### Дызайн-студыя {#design-studio}

**Фокус:** Візуальная кастамізацыя — колеры, тыпаграфіка, CSS і блокавыя шаблоны.

**Даступныя інструменты:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Што яна робіць добра:**
- Прымяненне названых прадустановак theme (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- Дакладная настройка глабальнай тыпаграфікі і каляровых палітраў праз theme.json
- Устаўка карыстальніцкага CSS для пераазначэнняў, спецыфічных для брэнда
- Стварэнне здымка экрана старонкі і агляд яго на прадмет праблем дызайну

**Пачатковыя прапановы:**
- *Прымяніце прадустаноўку warm-editorial, а потым задайце асноўны колер #2d6a4f.*
- *Зрабіце здымак экрана галоўнай старонкі і скажыце, што вы б палепшылі.*
- *Стварыце шматразовы блокавы шаблон hero з фонавым відарысам на ўсю шырыню і загалоўкам па цэнтры.*

### Менеджар плагінаў {#plugin-manager}

**Фокус:** Пошук, усталяванне і кіраванне плагінамі WordPress.

**Даступныя інструменты:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Што ён робіць добра:**
- Рэкамендацыя найлепшага плагіна для апісанага выпадку выкарыстання
- Усталяванне пакетаў магчымасцей з рэестра
- Прагляд даступнага каталога магчымасцей па катэгорыях

**Пачатковыя прапановы:**
- *Які найлепшы плагін для каталога членства?*
- *Усталюйце пакет магчымасцей WooCommerce.*
- *Пакажыце мне ўсе даступныя пакеты магчымасцей электроннай камерцыі.*

### Памочнік падтрымкі {#support-assistant}

**Фокус:** Адказы на пытанні пра кантэнт сайта, налады і канфігурацыю WordPress.

**Даступныя інструменты:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Што ён робіць добра:**
- Шукае бягучыя налады і параметры сайта
- Тлумачыць, якія тыпы запісаў, таксаноміі і меню наладжаны на сайце
- Адказвае на пытанні «што робіць гэтая налада?», чытаючы актуальныя значэнні
- Служыць дыягнастычным пластом толькі для чытання перад унясеннем змен

**Пачатковыя прапановы:**
- *Якія плагіны і налады цяпер актыўныя на гэтым сайце?*
- *Пералічыце ўсе карыстальніцкія тыпы запісаў, зарэгістраваныя на гэтым сайце.*
- *Якія навігацыйныя меню існуюць і дзе яны прызначаны?*

## Інтэграцыі аўтаматызацыі Superdav {#superdav-automation-integrations}

Калі інтэграцыі Superdav AI Agent v1.18.0 наладжаны, убудаваныя агенты могуць удзельнічаць у больш бяспечных працоўных працэсах аўтаматызацыі з улікам раскладу:

- **Інструменты чытання Google Calendar** дазваляюць агентам праглядаць наладжаныя календары і падзеі перад падрыхтоўкай наступных дзеянняў.
- **Супастаўленне кантактаў і ўдзельнікаў** дапамагае звязваць удзельнікаў падзей з карыстальнікамі WordPress або вядомымі кантактамі.
- **Шлюзы чалавечага зацвярджэння** прыпыняюць адчувальныя дзеянні, пакуль упаўнаважаны карыстальнік не праверыць і не пацвердзіць іх.
- **Запісы напамінаў** прадухіляюць дубляванне апавяшчэнняў, калі запланаваныя заданні паўтараюцца або запускаюцца паўторна.
- **SMS-апавяшчэнні TextBee** адпраўляюць наладжаныя тэкставыя паведамленні толькі тады, калі ўліковыя дадзеныя SMS і дазволы працоўнага працэсу ўключаны.

Рэкамендаваны працоўны працэс: папрасіце агента падрыхтаваць паведамленне або дзеянне, праверце запыт на зацвярджэнне, а потым дазвольце зацверджанаму дзеянню працягнуцца. Для перыядычных напамінаў пакідайце дэдуплікацыю напамінаў уключанай, каб адна і тая ж падзея або кантакт не атрымлівалі апавяшчэнні паўторна.

## Наладжванне агентаў {#customising-agents}

Кожнага ўбудаванага агента можна пашырыць або замяніць праз фільтр `gratis_ai_agent_agents`.

### Даданне карыстальніцкага сістэмнага prompt да існуючага агента {#adding-a-custom-system-prompt-to-an-existing-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### Рэгістрацыя новага агента {#registering-a-new-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

Новы агент з’яўляецца ў выбаршчыку агентаў адразу пасля запуску фільтра.

### Выдаленне ўбудаванага агента {#removing-a-built-in-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
