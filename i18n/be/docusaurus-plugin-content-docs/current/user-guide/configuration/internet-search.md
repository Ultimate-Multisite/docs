# Internet Search {#internet-search}

Gratis AI Agent v1.5.0 дадаў функцыю **Internet Search**, якая дазваляе асістэнту AI атрымліваць актуальную інфармацыю з Інтэрнэту падчас размовы. Гэты функрыяльнасць працуе на аснове [Brave Search API](https://brave.com/search/api/) або [Tavily API](https://tavily.com/).

## How It Works {#how-it-works}

Калі ўключана пошук у Інтэрнэце, асістэнт можа аўтаматычна адпраўляць запыт да вашага канфігураванага пошукавага провайдэра, калі вызначае, што пытанне патрабуе актуальнай або ўнешняй інфармацыі — напрыклад, недавнія навіны, жывыя цаны або дакументацыя, якая магла змяніцца з даты навучэння мадэлі.

Вынікі атрымліваюцца ў рэальным часе і ўстаўляюцца ў кантэкст асістэнта, перш чым ён згенеруе адказ. Асістэнт паказвае, калі ён выкарыстоўваў вынікі пошуку для адказа на пытанне.

## Enabling Internet Search {#enabling-internet-search}

Для пошуку ў Інтэрнэце патрабуецца API key ад вашага выбранага пошукавага провайдэра. Каб гэта канфігураваць:

1. Зайдзіце ў **Gratis AI Agent → Settings → Advanced**.
2. Знайдзіце выпадальны спіс **Internet Search Provider** і выберыце **Brave Search** або **Tavily**.
3. Увярце свой API key ў адпаведным полі. Ссылкі на рэгістрацыі адлюстроўваюцца побач з кожным полі.
4. Націсніце **Save Settings**.

Пасля захавання ключа функцыя пошуку ў Інтэрнэце аўтаматычна становіцца даступнай для асістэнта.

## Brave Search {#brave-search}

### Obtaining a Brave Search API Key {#obtaining-a-brave-search-api-key}

1. Адвідайце [Brave Search API page](https://brave.com/search/api/).
2. Зарэгістравацца на план. Даступны бясплатны ўзровень з лімітам запытаў на месяц.
3. Скопіруйце свой API key з дашборда для разработніка Brave Search.
4. Устаўце яго ў поле **Brave Search API Key** у настройках Gratis AI Agent.

### Usage Limits {#usage-limits}

Выкарыстанне выстаўляецца па рахунку Brave Search на аснове колькасці адпраўленых запытаў. Кожны адказ AI, які выклікае пошук, лічыцца адным запытам. Адслеживайце сваё выкарыстанне ў [Brave Search developer dashboard](https://brave.com/search/api/), каб пазбегнуць нечакаваных скіраванняў.

## Tavily {#tavily}

Superdav AI Agent v1.10.0 дадаў **Tavily** як альтэрнатыўнага провайдэра пошуку ў Інтэрнэце, прапануючы больш багатыя вынікі пошуку і пашыраныя рэдакцыйныя магчымасці.

### Obtaining a Tavily API Key {#obtaining-a-tavily-api-key}

1. Адвідайце [Tavily API page](https://tavily.com/).
2. Зарэгістравацца на кант. Даступны бясплатны ўзровень з лімітам запытаў на месяц.
3. Скопіруйце свой API key з дашборда Tavily.
4. Устаўце яго ў поле **Tavily API Key** у настройках Gratis AI Agent.

### Usage Limits {#usage-limits-1}

Выкарыстанне выстаўляецца па рахунку Tavily на аснове колькасці выклікаў API. Кожны адказ AI, які выклікае пошук, лічыцца адным выклікам. Адслеживайце сваё выкарыстанне ў [Tavily dashboard](https://tavily.com/), каб пазбегнуць нечакаваных скіраванняў.

## Disabling Internet Search {#disabling-internet-search}

Умастуйце API key з поля актыўнага пошукавага провайдэра і захавайце. Функцыя пошуку ў Інтэрнэце больш не будзе прапаноўвацца асістэнту.

:::note
Пошук ў Інтэрнэце павялічвае праходжанне адказаў, бо асістэнт павінен пачакаць вынікаў пошуку, перш чым згенеруй адказ. Для выпадкаў, якія патрабуюць хуткага адказу, разбярэйцеся, ці неабходны рэальны пошук, ці дастаткова ўбудаваныя веды асістэнта.
:::
