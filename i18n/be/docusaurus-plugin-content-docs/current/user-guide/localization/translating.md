# Пераклад Ultimate Multisite {#translating-ultimate-multisite}

Ultimate Multisite распрацаваны англійскай мовай, але plugin цалкам перакладны, і мы актыўна вітае ўдзел у перакладах.

## Пераклады асноўнага plugin {#core-plugin-translations}

Асноўны plugin Ultimate Multisite перакладаецца праз афіцыйную **платформу перакладу WordPress.org**:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

Гэта стандартная сістэма перакладу ў спільноце WordPress. Калі пераклады дасягнуць дастатковага ўзроўню пацверджання, яны аўтаматычна распыляюцца ўсім карыстальнікам Ultimate Multisite праз абнаўленні plugin WordPress.

**Нам патрэбна ваша дапамога!** Калі вы ведаеце мову, адрознічную ад англійскай, прасбуйце падумаць пра ўдзел у перакладах. Нават пераклад некалькіх тэкставых фрагментаў (strings) дапаможа. каб пачаць:

1. Навесці [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)
2. Выбраць сваю мову
3. Націснуць "Contribute Translation"
4. Вам спатрэбіцца бясплатны акаунт WordPress.org

Калі вы хочаце стаць рэдактарам перакладу для сваёй мовы (і мець магчымасць пацвярджаць пераклады), напішыце нам праз [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions).

## Пераклады аддоўных plugin (Addon Translations) {#addon-translations}

Пераклады аддоўных plugin перакладаюцца асобна тут:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Пераклады аддоўных plugin на дадзены момант згенераваны выкарыстоўвайце штучны інтэлект (AI) як выхадная кропка. Яны пакрываюць усі падтрымліваныя мовы, але могуць містжваць незручны фразевы склад або невярнасці. Мы вітае людскіх перакладаўцаў, якія могуць прагледжаць і прапанаваць паляпшэнні, каб зрабіць іх больш натуральнымі і правільнымі.

## Пераклады на ўзроўні месца (Local Translations) {#local-translations}

Калі вы пераканаўваacieся, што пераклад патрэбны толькі для вашага ўласнага праекта, а не для ўдзелу ў агульнай спільноце, ёсць дзве варыянты:

### Выкарыстанне Poedit {#using-poedit}

Вы можаце выкарыстоўваць настольны праграмістскі пакет **[Poedit](https://poedit.net)** для перакладу файла `.pot`:

1. Замясціце Poedit з [poedit.net](https://poedit.net)
2. Адкраніце файл `.pot` з пакету `wp-content/plugins/ultimate-multisite/lang/`
3. Перакладзіце тэкставыя фрагменты ў інтэрфейсе Poedit
4. Захаваць — Poedit стварае файлы `.po` і `.mo`
5. Памясціце выхадныя файлы ў папку `lang`

### Выкарыстанне Loco Translate {#using-loco-translate}

Plugin **[Loco Translate](https://wordpress.org/plugins/loco-translate/)** для WordPress дазваляе вам перакладаць непасрэдна з адмінчай панэлі без выкарыстання настольных інструментаў. Ён сам займаецца стварэннем і збірайцеляй перакладачных файлаў.
