# Папраўванне пачатку спісу краін Ultimate Multisite {#re-ordering-the-ultimate-multisite-country-list}

Выпадальны спіс для выбару краіны паводзіць па ўкарыўкавым паводле англійскіх варыянтаў назваў краін. Гэта можа азначаць, што ваша рэзідэнцыйная краіна і/або краіна большасці вашага кліентаў можа быць захавана ў сярэдзіне вельмі доўгага спісу краін.

Каб дапамагчы вам з гэтай праблемай і дадаць краіну вашага выбару ў пачатак спісу, вы можаце выкарыстоўваць наступны фрагмент кода (вы можаце дадаць яго ў mu-плагін):

add_filter('wu_get_countries', function($countries) {

// Змяніце DE на двухлітарны код вашай краіны

// Прыклад для Германіі.

unset($countries['DE']);

// Змяніце DE на двухлітарны код вашай краіны

// і назву вашай краіны.

// Прыклад для Германіі.

$countries = array_merge(array( 'DE' => __('Germany', 'ultimate-multisite'), ), $countries);

return $countries;

});
