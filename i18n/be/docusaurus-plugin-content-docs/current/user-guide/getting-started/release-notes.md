# Заўвагі да выпуску

## Версія 2.13.0 — Выпушчана 2026-06-05 {#version-2130--released-on-2026-06-05}

- Новае: Дададзена падтрымка суверэнных арандатараў для патокаў акаўнта кліента, афармлення замовы, білінгу, сайта, рахунка, пераключэння шаблонаў і прывязкі даменаў, каб сеткі арандатараў маглі накіроўваць кліентаў назад на галоўны сайт для кіраваных дзеянняў.
- Новае: Дададзены праверкі ўліковых даных падаўжэння для рэгулярных сяброўстваў, каб шлюзы маглі адключаць аўтападаўжэнне, калі захаваная білінгавая дамова, падпіска або vault token адсутнічае.
- Новае: Дададзена публікацыя праз loopback з праверкай HMAC для стварэння сайтаў у чаканні, каб зрабіць падрыхтоўку сайта пасля афармлення замовы больш надзейнай на хостах, дзе фонавыя заданні затрымліваюцца.
- Новае: Дададзены пункты пашырэння для распрацоўшчыкаў для SSO URL-адрасоў, базавых даменаў формы афармлення замовы і аўтаматычнага стварэння даменных запісаў.
- Выпраўленне: SSO стаў больш надзейным на прывязаных даменах, пры ананімных наведваннях брокера, выхадзе з сістэмы і канфліктах залежнасцей паміж plugin-амі.
- Выпраўленне: Стварэнне сайта ў чаканні цяпер аднаўляецца пасля састарэлых сцягоў публікацыі і не пакідае кліентаў заблакіраванымі на экране стварэння сайта.
- Выпраўленне: Даменныя запісы больш не ствараюцца для агульных базавых даменаў формы афармлення замовы, а невыкарыстоўваныя фонавыя заданні пастаўшчыка хостынгу прапускаюцца, калі няма актыўнай інтэграцыі.
- Выпраўленне: Крайнія выпадкі афармлення замовы, білінгавага адраса, скіду пароля, праверкі email, пераключэння шаблонаў, тураў і кліенцкай панэлі кіравання больш не блакіруюць звычайныя кліенцкія патокі.
- Выпраўленне: Масавыя email-паведамленні цяпер захоўваюць атрымальнікаў прыватнымі, пазбягаючы фатальных памылак SMTP/plugin, калі спісы атрымальнікаў або паштовыя транспарты даюць збой.
- Выпраўленне: Крайнія выпадкі падаўжэння сяброўства, паказу заканчэння тэрміну і збору плацяжоў цяпер пазбягаюць неадкладнага заканчэння, збояў або прапушчаных абавязковых плацяжоў.
- Палепшана: Сумяшчальнасць з WordPress праверана да версіі 7.0, production-рэсурсы Vue перасабраны з npm-крыніц, а Cypress end-to-end-пакрыццё цяпер правярае больш патокаў афармлення замовы, наладжвання, SSO і шлюзаў.
