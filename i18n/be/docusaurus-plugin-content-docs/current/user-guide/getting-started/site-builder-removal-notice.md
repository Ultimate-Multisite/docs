# Паведамленне пра адключэнне рэжыму Site Builder

**Рэжым Site Builder адключаны ў Superdav AI Agent v1.12.0.** Калі вы выкарыстоўвалі рэжым Site Builder, вам трэба пераходзіць на **Setup Assistant agent** для стварэння тэмаў і наладзіцыі сайта.

## Што адбылося?

### Site Builder Mode (Стары рэжым)

Site Builder mode быў візардам-інтерфейсам для:

- Стваৰэння сайтаў з тэмпатаў
- Наладзіцыі асноўных параметрў
- Выбору тэмы
- Наладзіцыі першага кантэнту

### Што яго замяściла?

**Setup Assistant agent** цяпер адпраўляе ўсе функцыянальнасці Site Builder, прапануючы:

- Больш гнуткае наладзіванне па кедарэнні агента
- Палепшаныя магчымасці кастылязацыі тэмы
- Інтеграцыю з Theme Builder onboarding
- Пастаянную памяць site_brief для будучых сесій

## Калі вы выкарыстоўвалі Site Builder Mode

### Вашы сайты ў бяспецы

- Існуючыя сайты, створаныя ў рэжыме Site Builder mode, працягваюць працаваць
- Не было страты дадзеных або перапынку працы сайта
- Вы можаце працягваць кіраваць сваімі сайтамі ў звычайным рэжыме

### Пераход на Setup Assistant Agent

Для наладзіцыі новых сайтаў або змены тэмы выкарыстоўвайце Setup Assistant agent:

```
"Help me set up a new site"
```

або

```
"Start the Theme Builder onboarding"
```

Setup Assistant agent прадугледжвае такую ж функцыянальнасць, але з больш вялікай гнуткасцю.

## Параўнанне: Site Builder vs. Setup Assistant

| Функцыя | Site Builder (Адключаны) | Setup Assistant (Новы) |
|---------|----------------------|----------------------|
| Спосаб наладзівання | Форма-візарды | Размова з агентам |
| Выбар тэмы | Падготваныя тэмпаты | Кастылёванае стварэнне |
| Кастылязацыя | Аменныя магчымасці | Поўны дызайн-сістэма |
| Site brief | Не захаваецца | Пастаянная памяць |
| Будучыя сесіі | Паўторнае наладзіванне | Выкарыстанне захаванага site_brief |
| Гнуткасць | Фіксаваны працэс | Адаптыўная размова |

## Пераход на Setup Assistant Agent

### Для новых сайтаў

Замест выкарыстання Site Builder mode:

1. Запыт: "Help me set up a new site"
2. Setup Assistant agent паведоміць вас праз:
   - Мета і мэты сайта
   - Цільовую аўдыторыю
   - Брэндавую ідэнтычнасць
   - Генерацыю тэмы
   - Пачатковае наладзіванне

### Для існуючых сайтаў

Калі ў вас ёсць існуючы сайт, створаны ў Site Builder mode:

1. Вы можаце працягваць яго выкарыстоўваць, як ёсць
2. Каб абнавіць тэму, запыце: "Redesign my site"
3. Setup Assistant agent дапаможа вам стварыць новую тэму
4. Дадзеныя вашага сайта застаюцца незменненымі

### Для змены тэмы

Замест выбару тэмы ў Site Builder mode:

1. Запыт: "Change my theme"
2. Setup Assistant agent:
   - Паспытае пра вашыя дызайн-параметры
   - Згенеруе кастылёваную тэму
   - Актывуе яе на вашым сайце

## Асноўныя адрозненні

### Site Builder Mode

```
1. Выбраць тэмпат
2. Выбраць тэму
3. Наладзіць асноўныя параметры
4. Гатова
```

### Setup Assistant Agent

```
1. Опісаць мэту сайта
2. Вызначыць цільовую аўдыторыю
3. Выбраць дызайн-параметры
4. Агент генеруе кастылёваную тэму
5. Агент актывуе тэму
6. Site brief захаваецца для будучых сесій
```

## Прыдостажанні Setup Assistant Agent

### Больш гнуткае

- Опішыце свой сайт натуральнай мовай
- Атрывайце кастылёваныя рэкамендацыі
- Адаптуецца да вашых канкрэтных патрабаванняў

### Палепшаная кастылязацыя

- Генерацыя кастылёванай тэмы
- Рэшэнні дызайн-сістэмы
- Пастаянныя дызайн-токены

### Пастаянная памяць

- Ваш site_brief захаваецца
- Будучыя агенты разумеюць ваш сайт
- Не трэба па паўторнаму наладзіванню інфармацыі

### Інтегрована працэс

- Theme Builder onboarding
- Design System Aesthetics skill
- Ability Visibility controls
- Усё працуе разам без перапынкаў

## Выяўленне праблем

### Я не знаходжу Site Builder mode

Site Builder mode адключаны. Выкарыстоўвайце Setup Assistant agent:

```
"Help me set up a new site"
```

### Я хочу адлюстраваць сайт з Site Builder

Вы можаце адлюстраваць яго з Setup Assistant agent:

1. Запыт: "Help me set up a new site"
2. Опішыце мэту і дызайн вашага першапачатковага сайта
3. Агент згенеруе падобную тэму
4. Ваш site_brief будзе захаваны для будучых адношэнняў

### Май існуючы сайт Site Builder не працуе

Існуючыя сайты, створаныя ў Site Builder mode, працягваюць працаваць. Калі вы адчуваеце праблемы:

1. Праверце, што ваша тэма яшчэ актыўная
2. Упэўніцеся, што вашы плагіны ўключаны
3. Праверце журналы памылак WordPress
4. Звярніцеся на падтрымку, калі праблемы працягваюць

### Ці магу я яшчэ выкарыстоўваць старыя тэмпаты Site Builder?

Тэмпаты Site Builder больш не даступныя. Аднак:

- Вашы існуючыя сайты працягваюць працаваць
- Вы можаце адлюстраваць падобныя сайты з Setup Assistant agent
- Setup Assistant agent прадугледжвае больш магчымасцей кастылязацыі

## Наступныя крокі

1. **Для новых сайтаў**: Выкарыстоўвайце Setup Assistant agent
2. **Для існуючых сайтаў**: Працягвайце выкарыстоўваць іх, як ёсць
3. **Для змены тэмы**: Запыце дапамогі ў Setup Assistant agent
4. **Для ўдасканалення дызайну**: Выкарыстоўвайце Design System Aesthetics skill

## Звязаныя тэмы

- **Theme Builder Onboarding**: Паведомічнае наладзіванне для кастылёваных тэмаў
- **Setup Assistant Agent**: Наладзіванне сайта па кедарэнні агента
- **Site Specification Skill**: Вызначэнне мэтаў і аўдыторыі сайта
- **Design System Aesthetics Skill**: Удасканаленне візуальнай ідэнтычнасці сайта
