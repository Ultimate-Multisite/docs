# Процес пачатку працы Theme Builder

Superdav AI Agent v1.12.0 прадстаўляе кераваны **працэс пачатку працы Theme Builder**, які дапамагае вам стварыць кастомны блоковы тэма падчас першага наладжання. Гэта замяняе стары рэжым Site Builder на больш гнуткае падыходжанне з памошчю агента.

## Што гэта за працэс пачатку працы Theme Builder?

Процэс пачатку працы Theme Builder — гэта интерактыўны паўначалёўчык, які:

- Праўводзіць вас праз рашэнні па дызайне (коlrы, тыпографія, раскладка)
- Запісвае вашую візуальную ідэнтычнасць сайта
- Гэнэруе кастомны блоковы тэма, адыладжаны пад вашыя патрэбы
- Апрацоўвае тэму аўтаматычна пасля завяршэння

Гэты працэс працуе на аснове **Setup Assistant agent**, які задае ўдакладняльваючыя пытанні і паступова буе вашу тэму.

## Пачатак працёсу пачатку працы Theme Builder

### Наладжанне пры першым запуску

Калі вы ўпершыню запускаеце Superdav AI Agent на новым WordPress-устаноўленні, вы ўбачыце:

```
Welcome to Superdav AI Agent!

What would you like to do?
1. Build a custom theme (Theme Builder)
2. Use an existing theme
3. Skip setup for now
```

Выберыце **"Build a custom theme"**, каб пачаць працэс пачатку працы Theme Builder.

### Ручная актывацыя

Вы таксама можаце пачаць працэс пачатку працы Theme Builder у любы час, пакуль запытуючы:

```
"Start the Theme Builder onboarding"
```

або

```
"Help me create a custom theme"
```

## Крокі пачатку працы

### Крок 1: Выбар рэжыму

Setup Assistant agent запытвае вас пра перавагу:

```
How would you like to build your theme?
- Guided (I'll ask questions and build it for you)
- Hands-on (I'll show you options and you decide)
```

**Guided mode** рэкамендуецца для большасці карыстальнікаў; агент прапануе рэкамендацыі па дызайне на аснове вашай індустрыйі і мэтаў.

### Крок 2: Спецыфікацыя сайта

Вас запытаюць пра ваш сайт:

- **Site purpose**: E-commerce, blog, portfolio, SaaS і г. д.
- **Target audience**: Кога вашы наведвальнікі?
- **Brand colors**: Асноўныя і другояя колеры (або "Я не ўпэўнены")
- **Tone**: Прафесійны, творчы, гумарыстычны, мінімалістычны і г. д.

Гэтыя інфармацыі запісваюцца ў вашу памяць **site_brief**, на якую агенты спасылаюцца ў будучых сесіянах.

### Крок 3: Рашэнні па дызайне

Агент праводзіць вас праз выбар дызайнерскіх токенаў:

- **Typography**: Семія шрыфтоў (serif, sans-serif, monospace) і маштаб памерёў
- **Color palette**: Асноўныя, другояя, акцэнтныя і нейтральныя колеры
- **Spacing**: Кампактныя, звычайныя або пазбегальныя раскладкі
- **Motion**: Анімацыі і пераходы (калі гэта неабходна)

### Крок 4: Гэнэрацыя тэмы

Setup Assistant agent стварае каркас вашай кастомнай блоковы тэмы, уключаючы:

- `theme.json`, які ўтварэнне ўсіх вашага дызайнерскіх токенаў
- Блоковы шаблоны для агульных раскладкі (гомашэйд, блог, кантакты)
- Кастомныя стилі блокаў, якія адпавядаюць вашай дызайнерскай сістэме
- Метададзеныя тэмы і дэкларацыі падтрымкі WordPress

### Крок 5: Актывацыя і праверка

Тэма актывуецца аўтаматычна, і вы ўбачыце:

```
✓ Your custom theme is now live!
  Theme name: [Your Site Name] Theme
  Colors: [Primary] / [Secondary]
  Typography: [Font Family]

  Visit your site to see the new design.
```

Пасля гэтага вы можаце адвідаць свой сайт, каб праверыць, што тэма адбываецца правільна.

## Спецыфікацыя сайта і памяць site_brief

Падчас пачатку працы агент запісвае спецыфікацыю вашага сайта ў катэгорыю памяці **site_brief**. Гэта ўключае:

- Мэту і мэты сайта
- Цільовую аўдыторыю
- Колеры і тон брэнда
- Дызайнерскія перавагі
- Структура кантэнту

### Чаму важна site_brief

У будучых сесіянах агенты спасылаюцца на ваш site_brief, каб:

- Захаваць дызайнерскую адпаведнасць падчас змен
- Прапанаваць функцыі, адпаведныя мэце вашага сайта
- Надаць рэкамендацыі, якія ўлічваюць кантэкст
- Пазбегнуць паўторнага задавання пытанні па наладжанні

### Пагляд вашага site_brief

Вы можаце запытаць агента:

```
"Show me my site brief"
```

або

```
"What do you know about my site?"
```

Агент адкажа вам, паказваючы захаваную спецыфікацыю вашага сайта.

## Кастымізацыя пасля пачатку працы

Пасля завяршэння пачатку працы Theme Builder вы можаце:

### Выкарыстоўваць навык Design System Aesthetics

Запытаць узмацненне дызайну:

```
"Refine the typography to be more modern"
```

або

```
"Adjust the color palette to be warmer"
```

**Design System Aesthetics skill** праводзіць вас праз мэтанакіраваныя абнаўленні дызайну.

### Рэдагаваць theme.json непасрэдна

Для пашыраваных карыстальнікаў рэдагаванне `/wp-content/themes/[theme-name]/theme.json` дазволіць адрэгуляваць:

- Токены колераў
- Маштабы тыпографікі
- Значэнні паміжкрэсных прастудзяў
- Вызначэнні рамкі і ценяў

### Ствараць кастомныя блоковы шаблоны

Выкарыстоўвайце рэдактара блокаў WordPress, каб стварыць кастомныя шаблоны для:

- Раскладкі гомашэйд
- Стравак блога
- Стравак прадуктаў
- Кантактаў

## Параўнанне: Стары vs. Новы пачатак працы

| Feature | Site Builder (Legacy) | Theme Builder (New) |
|---------|----------------------|-------------------|
| Setup method | Wizard-based form | Agent-guided conversation |
| Theme generation | Limited templates | Custom scaffolding |
| Design tokens | Manual entry | Guided decisions |
| Flexibility | Fixed options | Customizable |
| Future updates | Not referenced | Stored in site_brief |

## Выяўленне праблем

**Процэс пачатку працы не завяршыўся**
- Пачніце працэс зноў: "Start the Theme Builder onboarding"
- Праверце, што ваша ўстаноўка WordPress абноўлена
- Упэўніцеся, што Setup Assistant agent актыўваны

**Май site_brief не выкарыстоўваецца**
- Патвердзіце, што ў агента ёсць доступ да памяці
- Папытайце агента "recall my site brief"
- Праверце, што памяць актыўвана ў наступствах

**Гэрагенэраваная тэма не адпавядае маім перавагам**
- Выкарыстоўце Design System Aesthetics skill, каб гэта ўдасканаліць
- Папытайце агента "regenerate the theme with [specific changes]"
- Рэдагаваць theme.json непасрэдна для дакладнага кантролю

## Наступныя крокі

Пасля завяршэння пачатку працы Theme Builder:

1. **Праверце свой сайт**: Адвідайце свой сайт, каб паглядзець новую тэму
2. **Удасканальце дызайн**: Выкарыстоўвайце Design System Aesthetics skill для абрэгулення
3. **Дадайце кантэнт**: Пачніце ствараць кантэнт сайта
4. **Вывучце магчымасці**: Пазнайце іншыя магчымасці агента, такія як scaffold-block-theme і activate-theme
