# Інтеграцыя WPMU DEV

## Агляд
WPMU DEV — гэта комплексная платформа WordPress, якая прапануе хостынг, плагіны і паслугі для сайтаў WordPress. Гэтая інтэграцыя дазваляе аўтаматычна стракаваць домены і кіраваць сертыфікатам SSL паміж Ultimate Multisite і хостынг-акаунтом WPMU DEV.

## Функцыі
- Аўтаматычнае стракаванне доменаў
- Кіраванне сертыфікатам SSL
- Пашыренаныя спробы праверкі сертыфікатаў SSL

## Патрабаванні
Інтэграцыя аўтаматычна выяўляе, ці хостыруе вы на WPMU DEV, і выкарыстоўвае ўбудаваны API. Калі вы хостыруеце на WPMU DEV, дадатковая канфігурацыя не патрэбна.

Інтэграцыя праверыць прысутнасць канстанты `WPMUDEV_HOSTING_SITE_ID`, якая аўтаматычна вызначаецца пры хостыінгу на WPMU DEV.

## Інструкцыі па наладзе

### 1. Праверка хостынгу WPMU DEV

Калі вы хостыруеце на WPMU DEV, неабходныя канстанты павінны ўжо быць вызначаны. Праверце, што:

1. Канстанта `WPMUDEV_HOSTING_SITE_ID` вызначана ў вашай асяроддзі
2. У вас ёсць актыўная абонентская плата WPMU DEV з доступом да API

### 2. Актывацыя інтэграцыі

1. У адмініструванні WordPress заййміцеся ў Ultimate Multisite > Settings
2. Пайміце на тэбестрані "Domain Mapping"
3. Зробіце скрол да "Host Integrations"
4. Актывуйце інтэграцыю WPMU DEV
5. Націсніце "Save Changes"

## Як гэта працуе

### Стракаванне доменаў

Калі домен стракаваны ў Ultimate Multisite:

1. Інтэграцыя выкарыстоўвае WPMU DEV API, каб дадаць домен да вашага хостынг-акаунту
2. Яна таксама аўтаматычна дадае версію з www
3. WPMU DEV займаецца канфігурацыяй домена і выдачай сертыфікатаў SSL

### Кіраванне сертыфікатам SSL

Інтэграцыя настроена на павышэнне колькасці спробаў праверкі сертыфікатаў SSL для хостынгу WPMU DEV, бо на выдачу і ўстаноўку сертыфікатаў SSL можа прайсці некаторы час. Па ўстаноўцы яна паспрабуе да 10 разаў для праверкі сертыфікатаў SSL, што больш, чым стандартныя 5 спробаў.

## Важнае заўважванне

### Выбыванне домена
У цяперар WPMU DEV API не прадугледжвае спосабу выдалення доменаў. Калі стракаванне домена выбывае ў Ultimate Multisite, домен застанецца ў вашай хостынг-панелі WPMU DEV. Калі гэта неабходна, вам трэба выдаліць яго ўручную ў адмініструванні хостынгу WPMU DEV.

### Аўтэнтычныя API
Інтэграцыя выкарыстоўвае ключ API WPMU DEV, які захоўваецца ў базе дадзеных WordPress як опцыя `wpmudev_apikey`. Гэта аўтаматычна наладжаецца, калі вы падключаеце свой сайт да WPMU DEV.

## Выяўленне праблем

### Праблемы з падключэннем API
- Праверце, што ваш сайт належным чынам падключаны да WPMU DEV
- Праверце, што опцыя `wpmudev_apikey` вызначана ў базе дадзеных WordPress
- Убедзіцеся, што ваша абонентская плата WPMU DEV актыўная

### Праблемы з сертыфікатам SSL
- WPMU DEV можа патрабаваць некаторы час для выдачы сертыфікатаў SSL (завычайна 5-15 хвілін)
- Інтэграцыя настроена на праверку да 10 разаў для сертыфікатаў SSL
- Калі сертыфікаты SSL па-ранейшым не выдаляюцца пасля некалькіх спробаў, звярніцеся ў службу падтрымкі WPMU DEV

### Домен не дададзены
- Праверце журналы Ultimate Multisite на наяўнасць памылак
- Убедзіцеся, што домен яшчэ не дададзены ў WPMU DEV
- Убедзіцеся, што ваш хостынг-план WPMU DEV падтрымлівае колькасць доменаў, якія вы дадаеце
