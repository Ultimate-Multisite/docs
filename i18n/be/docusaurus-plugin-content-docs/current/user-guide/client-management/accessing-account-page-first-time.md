# Як мне ўпершыню атрымаць доступ да адкаднай? {#how-can-i-access-my-account-page-for-the-first-time}

Вітаем у Ultimate Multisite!

Атклікацеся на адкаднай праз спасылку, якую вы атрымалі ў сваёй электроннай рассылцы пасля куплі нашага плагіна:

<!-- Screenshot unavailable: Account page email with access link -->

Выкарыстоўвайце электронную адрактую, якую вы пакінулі падчас куплі, і пароль, які вы атрымалі ў сваёй рассылцы.

<!-- Screenshot unavailable: Login page with email and password fields -->

Калі гэта ваш першы раз, калі вы ўваходзіце ў гэты адказны кабінет, вас папросуць змяніць пароль. Змяніце яго і націсніце кнопку _Save_.

<!-- Screenshot unavailable: Change password form with Save button -->

На адкаднай вы можаце кіраваць усімі дэталямі вашага адкаднай, такія як рахункі, пановленні, адмена і многа іншых рэчаў.
