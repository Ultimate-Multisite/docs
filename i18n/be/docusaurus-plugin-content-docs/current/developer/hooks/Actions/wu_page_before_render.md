# Дія: wu_page_before_render

Дозволяє розробникам плагінів додавати додатковий контент перед тим, як ми виведемо сторінку на екран.

## Параметри

| Ім'я | Тип | Опис |
|------|------|-------------|
| $page_id | `string` | Ідентифікатор цієї сторінки. |
| $page | `object` | Об'єкт сторінки. |

### Зверніться до

- 1.8.2
