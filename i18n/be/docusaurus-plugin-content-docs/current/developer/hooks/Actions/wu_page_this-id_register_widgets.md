# Action: wu_page_\{$this->id\}_register_widgets

Выклікаецца пасля рэгістрацыі віджытаў для гэтай старонкі.

Дынамічная частка іме напалугі, `$this->id`, адносіцца да ID старонкі.

## Паметры

| Назва | Тып | Апісан |
|------|------|-------------|
| $id | `string` | ID старонкі. |
| $page_hook | `string` | Hook старонкі. |
| $page | `object` | Объект старонкі. |
