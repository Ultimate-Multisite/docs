# Action: wu_before_light_ajax

У некаторых выпадках нам трэба загрузіць дадатковыя функцыянальнасці для апрацоўкі дзеяўлення.

Гэты Action можна выкарыстоўваць для загрузкі дадатковых залежнасцей, калі гэта неабходна.
