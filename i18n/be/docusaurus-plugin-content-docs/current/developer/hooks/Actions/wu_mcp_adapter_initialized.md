# Дзея: wu_mcp_adapter_initialized

Выклікаецца пасля таго, як MCP adapter будзе ініцыялізаваны.

Дазваляе іншым plugins і тэмам рэгіструйваць свае ўласныя магчымасці.

## Паметры {#parameters}

| Назва | Тып | Апісан |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | Экземпляры адпавядання MCP adapter. |

### З {#since}

- 2.5.0
