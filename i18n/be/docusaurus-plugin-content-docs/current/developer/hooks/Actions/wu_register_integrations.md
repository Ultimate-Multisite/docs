# Action: wu_register_integrations

Выконваецца, каб дазволіць інтыгрэцыйным модулям зарэгістравацца.

## Параметры {#parameters}

| Назва | Тып | Опіс |
|------|------|-------------|
| $registry | `\Integration_Registry` | Экземплярык рэгістры. |
