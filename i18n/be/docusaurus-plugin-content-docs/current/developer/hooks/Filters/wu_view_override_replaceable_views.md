# Filter: wu_view_override_replaceable_views

Дазваляе рэалізатары дадаць адпаведныя паказальнікі ў спіс замяняемых.

Будзьце ўважлівы, бо дапушчэнне дадатковых паказальнікаў можа прывесці да загружкі устарэлых капій замест версій Ultimate Multisite.

## Параметры {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $replaceable_views | `array` | Спіс дазволеных паказальнікаў. |

### Since {#since}

- 2.0.0
### Source {#source}

Вызначаны ў [`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L50) у 50-й пасылцы
