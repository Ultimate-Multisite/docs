# Фільтр: wu_stripe_existing_product_id {#filter-wustripeexistingproductid}

Фільтруе ID прадукту, які трэба праверыць. Калі ён існуюць, новая падпіска будзе выкарыстоўваць гэты прадукт.

## Паметры {#parameters}

| Назва | Тып | Апісан |
|------|------|-------------|
| $product_id | `string` | ID прадукту Stripe, які трэба праверыць. |
| $name | `string` | Назва прадукту. |
