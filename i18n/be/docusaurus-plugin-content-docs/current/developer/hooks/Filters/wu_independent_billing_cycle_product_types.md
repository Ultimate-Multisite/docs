# Filter: wu_independent_billing_cycle_product_types {#filter-wuindependentbillingcycleproducttypes}

Фільтруе тыпы прадуктаў, якія маюць самастойна вядзенне (або "самакрэўны") цыкл выстаўлення рахункаў.

Прадукты з самастойна вядзеннем цыклу выстаўлення рахункаў не прымушана павінны адпавядаць перыяду выстаўлення рахункаў плана ў кошыку. Гэта вельмі карысна для прадуктаў, такіх як рэгістрацыя домена, якія заўсёды выстаўляюцца на год, незалежна ад таго, ці план месячны, ці годны.

## Параметры {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | Массив тыпаў прадуктаў, якія маюць самастойна вядзенне цыклу выстаўлення рахункаў. |
