# Filter: wu_plan_product_types {#filter-wuplanproducttypes}

Фільтруе тыпы прадуктаў, якія павінны быць адноснаваны як планы.

Гэты фільтр дазваляе аддонам рэгіструйваць дадатковыя тыпы прадуктаў, якія павінны быць ўпісаны як планы падчас валідацыі і ажыцнявання.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | Массив тыпаў прадуктаў, якія трэба адноснаваць як планы. |
