# Фільтр: wu_get_site_domain_and_path

Дазваляе рэалізавальнікам маніпуляваць пары (пары) домена і шляху.

Гэта можа быць карысным для многіх рэчаў, напрыклад, для рэалізацыі некаторых стэйдзінг-ра🍌шэнняў, працы з рознымі sérversy і г.д.

## Параметры {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $d | `object` | Bieжачыя доўга, які ўтрымлівае ключы домена і шляху. |
| $path_or_subdomain | `string` | Арыгінальны шлях/субдомен, перададзены ў функцыю. |

### Since {#since}

- 2.0.0
### Source {#source}

Вызначаны ў [`inc/functions/site.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/site.php#L235) на 235-й пасылцы.


## Returns {#returns}
Об'ект, які ўтрымлівае ключы домена і шляху.
