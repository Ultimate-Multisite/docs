# Фільтр: wu_sso_url {#filter-wussourl}

Фільтруе згенераваныя SSO URL перад тым, як яны вяртаюцца для міждаменных дзеянняў кліента.

Выкарыстоўвайце гэты фільтр, калі інтэграцыі трэба дадаць давераны кантэкст да SSO-спасылкі суверэннага арандатара або замяніць URL брокера, захоўваючы праверку токена Ultimate Multisite.

## Параметры {#parameters}

| Назва | Тып | Апісанне |
|------|------|-------------|
| $sso_url | `string` | Згенераваны SSO URL. |
| $user | `WP_User` | Карыстальнік, які будзе аўтэнтыфікаваны праз наведванне SSO. |
| $site_id | `int` | ID мэтавага сайта для наведвання. |
| $redirect_to | `string` | URL прызначэння пасля паспяховай праверкі SSO. |

### З версіі {#since}

- 2.13.0

### Крыніца {#source}

Вызначана ў `inc/sso/class-sso.php`.


## Вяртае {#returns}

Адфільтраваны SSO URL.
