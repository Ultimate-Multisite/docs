# Фільтр: wu_pre_save_settings

Дазваляе рэалізавальнікам фільтраваць налады перад zapisom праз Ultimate Multisite.

## Параметры {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $settings | `array` | Налады, якія трэба zapisaць. |
| $settings_to_save | `array` | Новыя налады, якія трэба дадаць. |
| $saved_settings | `array` | Налады, якія ўжо zapісаны. |

### З версіі {#since}

- 2.0.18
