# Filter: wu_checkout_validation_aliases

Дазваляе раробнікам плагінаў дадаваць карыстальніцкія (кастомныя) псеўнімы адноўленні ў валідатары формы.

## Параметры {#parameters}

| Назва | Тып | Апісан |
|------|------|-------------|
| $validation_aliases | `array` | Массив з ідэнтыфікатарамі (id) па адноўленні (alias). |
| $checkout | `\Checkout` | Клас адкалення (checkout). |
