# Фільтр: wu_checkout_validation_rules

Дазваляе раробнікам плагінаў фільтраваць правілы валідацыі.

## Параметры {#parameters}

| Назва | Тып | Апісан |
|------|------|-------------|
| $validation_rules | `array` | Правілы валідацыі, якія трэба выкарыстоўваць. |
| $checkout | `\Checkout` | Клас карыстальніка. |
