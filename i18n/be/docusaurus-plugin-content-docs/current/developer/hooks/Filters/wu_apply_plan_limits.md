# Filter: wu_apply_plan_limits

Дазваляе раробнікам плагінаў прапыніць абмежаванні.

Вы можаце выкарыстоўваць гэты filter, каб выканаць любы код да таго, як пачнуць працаваць якія-не і адно з абмежаванняў. Калі filter вяртае любое істиннае значэнне (truthy value), працэс прайдзе далі; калі ён вяртае любое неістиннае значэнне (falsy value), код спыніцца, і ні адно з хукаў, якія ніжэй, не выхадзе.
