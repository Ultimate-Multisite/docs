# Дакументацыя Ultimate Multisite

ДобягаДобяга ў дакументацыю Ultimate Multisite. Ultimate Multisite ператварае WordPress Multisite ў поўную платформу Website as a Service (WaaS).

## Раздзелы дакументацыі

### [User Guide](/user-guide/getting-started/ultimate-multisite-101)

Усё, што вам патрэбна, каб наладзіць, сканфігураваць і кіраваць вашай платформой WaaS. Пакрывае ўстаноўку, шлюзы плацёжаў, мапінг доменных імен, кіраванне кліентамі і больш.

### [Developer Documentation](/developer/hooks)

Тэхнічны рэчнік для праграмістаў, якія ствараюць на базе Ultimate Multisite. Уключае рэчнік па хуках (hooks), дакументацыю REST API і параўды па распрацоўцы аддонакóў (addons).

### [Addons](/addons)

Дакументацыя па платных аддонаквых плагінах (addon plugins), якія пашыраюць Ultimate Multisite дадатковыми функцыяналамі, такімі як інтэграцыя з WooCommerce, аналітыка, продаж доменных імен і больш.
