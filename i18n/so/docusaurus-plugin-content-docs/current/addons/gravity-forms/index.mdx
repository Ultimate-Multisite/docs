# Gravity Forms

**Khibradda diiwaangelinta isticmaalahaaga ku beddel awoodda Gravity Forms**

Si habsami leh ugu dhex-dar dhisaha foomamka ugu horumarsan adduunka Multisite Ultimate si aad u abuurto foomam diiwaangelin qurux badan, beddelid sare leh oo kordhiya is-diiwaangelinta xubinnimadaada isla markaana kordhiya dakhliga.

## 🌟 Astaamaha Muhiimka ah

### 📋 **Dhisidda Foomka Sare**
- **Foomam tallaabooyin badan leh** oo loogu talagalay socodka diiwaangelin adag
- **Logic shuruudeed** si loo muujiyo/loo qariyo fields si firfircoon
- **File uploads** loogu talagalay dukumiintiyada iyo sawirrada
- **Noocyada field hodanka ah**: saxiixyo, xisaabinno, xulashada product
- **Foomam mobile-responsive** ah oo si fiican ugu muuqda qalab kasta

### 🔗 **Xidhitaanka Xogta oo Habsami Leh**
- Interface **xidhitaanka field muuqaal ah**
- **Macluumaadka macmiilka**: magacyo, email, faahfaahinta billing
- **Xogta xubinnimada**: plans, muddada, fields gaar ah
- **Abuurista site**: cinwaanno, URLs, xulashada template
- **Ururinta xog gaar ah** oo loogu talagalay baahiyahaaga gaarka ah

### 💰 **Habaynta Lacag-bixinta oo Dabacsan**
- **Gudbin checkout**: U gudbid habsami leh bogagga lacag-bixinta Multisite Ultimate
- **Isdhexgalka WooCommerce**: Ka faa'iidayso boqolaal payment gateways
- Taageerada **tijaabooyin bilaash ah iyo xayeysiisyo**
- **Qaansheegad otomaatig ah** iyo abuurista rasiidh

### 🎨 **Khibrad Isticmaale oo Xirfad leh**
- **Bogag xaqiijin gaar ah** oo leh merge tags
- **Ogeysiisyo email otomaatig ah** oo loogu talagalay macaamiisha iyo admins
- **Tilmaamayaasha horumarka** ee foomamka tallaabooyinka badan leh
- **Hubinta khaladka** oo leh farriimo caawinaya
- **Joogteynta brand-ka** oo la jaanqaadaysa naqshadda site-kaaga

### 🛡️ **Amni Heer-Shirkadeed**
- **Ilaalinta spam-ka** oo leh ikhtiyaarrada CAPTCHA ee ku dhex-jira
- **Hubinta xogta** iyo nadiifinta
- Dhaqannada ugu wanaagsan ee **amniga WordPress**
- Astaamaha **u hoggaansanaanta GDPR**
- **File uploads ammaan ah** iyo kaydin
