# PayFast {#payfast}

Multisite Ultimate: Albaabka Payfast

Waxay ku darsataa Payfast - albaab lacag-bixineed oo online ah, degdeg ah oo ammaan ah kuna yaal Koonfur Afrika - si uu u noqdo ikhtiyaar lacag-bixineed oo Multisite Ultimate ah.

## Rakibid {#installation}

1. Ku rar 'ultimate-multisite-payfast' galka '/wp-content/plugins/'
2. Ka hawlgeli plugin-ka adigoo adeegsanaya menu-ga 'Plugins' ee WordPress
3. Raac Wizard-ka tallaabo-tallaabo ah si aad plugin-ka u dejiso

## Hagaha Isticmaalka {#usage-guide}

PayFast waa farsameeye lacag-bixineed online ah oo hormuud u ah ganacsiyada Koonfur Afrika. Waxaa u heellan inuu ka caawiyo ganacsatada online-ka ah inay koraan iyadoo macaamilada laga dhigayo kuwo fudud oo ammaan ah. Sida ku cad codsiga isticmaalaha Ultimate Multisite, waxaan go’aansannay inaan abuurno addon suurtogelinaya isku-darka PayFast. Maqaalkani wuxuu muujin doonaa sida PayFast loogu dejiyo albaabkaaga lacag-bixinta si aad lacag uga hesho shabakaddaada Ultimate Multisite.

Tallaabada ugu horreysa waa in la rakibo add on-ka.

# Sida loo rakibo add-on-ka Payfast: {#how-to-install-the-payfast-add-on}

Tag bogga add-ons-ka oo dooro add-on-ka Ultimate Multisite: Payfast Gateway.

![Bogga add-ons-ka](/img/admin/addons-page.png)

Daaqadda add-on-ka ayaa soo bixi doonta. Guji Install Now.

![Bogga add-ons-ka oo leh ikhtiyaarka rakibidda](/img/admin/addons-page.png)

Marka rakibiddu dhammaato, waxaa laguu wareejin doonaa bogga plugins-ka. Halkan, kaliya guji Network Activate waxaana add-on-ka _Payfast_ laga hawlgelin doonaa shabakaddaada.

# Dejinta PayFast {#setting-up-payfast}

Ka dib markaad rakibto add-on-ka, hadda waxaad aadi kartaa dejimaha Ultimate Multisite oo bilaabi kartaa habaynta PayFast.

Hoosta bogga **dejimaha Ultimate Multisite**, waxaad ka heli kartaa tab-ka "Payments". Hoosta tab-kan, waa inaad aragtaa "**Payfast** " oo la socda badhan toggle ah si aad u hawlgeliso ama u damiso albaabkan lacag-bixineed.

![Dejimaha albaabka lacag-bixinta](/img/config/settings-payment-gateways.png)Hore u soco oo guji badhanka toggle-ka si aad u hawlgeliso. Kadib wuxuu ballaarin doonaa sanduuqa si uu u muujiyo field-yo kala duwan oo aad u baahan tahay inaad buuxiso sida **Mechant ID, Merchant Key iyo PassPhrase.** Waxaad macluumaadkan ka heli kartaa **Payfast** Account-kaaga

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Tallaabada xigta waa inaad gasho Account-kaaga PayFast si aad u hesho macluumaadka aad u baahan tahay.

Markaad gasho, waxaad isla markiiba ka arki kartaa aqoonsiyada aad u baahan tahay dashboard-ka **Payfast** Account-kaaga.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->Si fudud ayaad u nuqulan kartaa oo u dhajin kartaa aqoonsiga, kadibna waxaad buuxin kartaa faahfaahinta dejimahaaga Ultimate Multisite. Kadib guji badhanka **"Save settings** " si aad u kaydiso isbeddellada.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Waa intaas, isku-darka lacag-bixinta PayFast waa dhammeystiran yahay!
