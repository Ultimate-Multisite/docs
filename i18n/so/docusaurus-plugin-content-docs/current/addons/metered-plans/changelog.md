# Isbeddellada Qorshayaasha Mitirka Leh

Nooca 1.1.0 - La sii daayay 2026-05-05
- Cusub: Biil-gareynta AI token ee multisite subsites — la soco oo biil ka qaad isticmaalka AI token guud ahaan site-yada macaamiisha iyadoo la adeegsanayo qiimayaal per-token la habeyn karo
- Cusub: Xoojinta connector-ka oo dib loogu dhisay ogaanshaha xadka firfircoon iyo write-through, taasoo xaqiijinaysa saxnaanta waqtiga-dhabta ah guud ahaan dhammaan connector-rada
- Hagaajin: Casriyeynta miisaska database-ka hadda si sax ah ayay ula jaanqaadaysaa qeexitaannada BerlinDB schema, taasoo ka hortagaysa guuldarrooyinka casriyeynta ee rakibaadaha cusub
- Hagaajin: Database upgrade callbacks waxaa loo beddelay qaabka saxda ah, taasoo xallinaysa guuldarrooyin casriyeyn oo aamusan
- Hagaajin: Qiimayaal jajab ah hadda waa laga aqbalaa goobta gelinta AI Usage Overage Markup
- Hagaajin: La xalliyay khaladaadka fatal-ka ah iyo arrimaha laba-jeer-bilowga marka plugin-ku raro
- La hagaajiyay: Waxaa lagu daray check-env npm script si deegaannada horumariyayaashu iskood isu habeeyaan marka ugu horreysa ee la socodsiiyo
