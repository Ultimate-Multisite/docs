# Filter: wu_connector_enforcement_option_keys

Shaandhee liiska furayaasha ikhtiyaarka ee bixiyaha AI ee laga dhaqan-gelinayo goobta weyn.

Ku dar furayaasha ikhtiyaarka ee plugins bixiye AI gaar ah ama dhinac saddexaad ah si dejintooda sidoo kale looga dhaxlo goobta weyn ee subsites.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $keys | `string[]` | Magacyada furayaasha ikhtiyaarka ee la dhaqan-gelinayo (horey waxaa ugu jira furayaasha connector ee si firfircoon loo helay iyo EXTRA_PROVIDER_OPTIONS). |
