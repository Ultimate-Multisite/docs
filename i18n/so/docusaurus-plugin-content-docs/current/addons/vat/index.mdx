# VAT

Multisite Ultimate: VAT-ka Yurub

Waxay ku dartaa sifooyin iyo qalab dheeraad ah maareynta canshuuraha caadiga ah ee Multisite Ultimate si loo taageero VAT-ka Yurub si ka wanaagsan.
Si otomaatig ah ayay heerarka EU VAT ugu dartaa bogga Heerarka Canshuurta laga bilaabo

## Rakibid {#installation}

1. Ku rar 'ultimate-multisite-vat' galka '/wp-content/plugins/'
2. Ka hawlgeli fidiyaha adigoo adeegsanaya liiska 'Fidiyeyaasha' ee WordPress
3. Tag Ultimate Multisite > Dejinta > Canshuuraha oo Daar EU VAT.
4. Guji Heerarka Canshuurta ka dibna Cusboonaysii Heerarka EU VAT ee hoose oo Kaydi Heerarka Canshuurta.
5. Waa dhammaatay!
