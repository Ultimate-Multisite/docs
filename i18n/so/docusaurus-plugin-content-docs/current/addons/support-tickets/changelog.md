# Diiwaanka Isbeddellada Tigidhada Taageerada

### 1.0.4 - 2026-05-05
* La hagaajiyey: Laga saaray galka vendor/ la socodka Git (horey waxaa u daboolay .gitignore), taas oo yaraynaysa cabbirka kaydka
* La hagaajiyey: Waxaa la tijaabiyey ilaa WordPress 7.0

### 1.0.3 - 2026-05-01
* La saxay: Aqbal qiimayaasha null ee setters-ka moodelka tigidhka nullable
* La saxay: Ku kaydi dhammaan tigidhada jadwallada guud ee shabakadda oo dhan si loo helo taageero multisite sax ah
* La saxay: Qari beeraha shaqaalaha oo hagaaji wareejinta bogga madhan ee foomka tigidh-cusub ee macmiilka
* La saxay: Ku beddel wicitaannada add_meta() ee aan la qeexin update_meta() sax ah si loo kaydiyo metadata
* La saxay: Ku beddel hubinta awoodda aan diiwaangashanayn wu_view_all_support_tickets sax ah
* La saxay: Ku dar AJAX handlers maqan oo loogu talagalay xaaladda tigidhka, meeleynta, iyo ficillada quick-edit
* La saxay: Sax magaca habka ee ogaanshaha jawaabta shaqaalaha ee aragtiyada tigidhka
* La saxay: Isku gee handlers-ka jawaabta ee nuqulka ah oo waafaji magacyada ficillada nonce
* La saxay: Ku dar aragtida frontend ee maqan ee shortcode-ka [wu_submit_ticket]
* La saxay: Ku dar tiirka user_id ee maqan oo sax habka Support_Ticket::get_user_id()
* La saxay: Ka saar calaamadda double-quote dheeraadka ah ee tag-ga ikhtiyaarka shaandhada mudnaanta
* La saxay: Ku dar guddi maareynta tigidhada network admin oo loogu talagalay super admins
* La hagaajiyey: Isku gee CSS-ka admin-ka hal stylesheet dibadeed
* La hagaajiyey: Ka saar Settings submenu liiska admin-ka subsite-ka
* La hagaajiyey: Si shuruudaysan u rar frontend assets kaliya bogagga tigidhada taageerada
* La hagaajiyey: Ka bood plugin autoloader marka Bedrock root autoloader horey u raray dependencies

### 1.0.2 - 2025-12-11
* Lagu daray: Shaqaynta buuxda ee daawashada tigidhada ee maamule iyo frontend
* Lagu daray: AJAX handler loogu talagalay gudbinta jawaabaha tigidhada
* Lagu daray: Taageero shaqaynta jawaabta tigidhka oo leh maarayn foom oo sax ah
* Lagu daray: Soo bandhigid ogeysiis sax ah oo loogu talagalay gudbinta tigidhada iyo jawaabaha gudaha admin
* Lagu daray: Isku-xirid tigidh oo otomaatig ah oo lala sameeyo isticmaalaha hadda jira ee gudbinta iskiis ee macmiilka
* Lagu daray: Xoojinta amniga si looga hortago in macaamiishu beddelaan lahaanshaha tigidhka
* Lagu daray: Functions caawiye oo maqan (wu_format_date, wu_user_can_view_ticket, iwm.)
* Lagu daray: Soo dejin iyo maarayn lifaaq fayl oo sax ah
* Lagu daray: Nidaamka ogeysiiska iimaylka ee jawaabaha tigidhada iyo isbeddellada xaaladda
* La saxay: Foomka jawaabta tigidhka hadda wuxuu ku jiraa aqoonsiga tigidhka ee lagama maarmaanka ah
* La saxay: Arrimaha muuqashada ogeysiiska ee aragtiyada tigidhada admin
* La saxay: Khaladaadka syntax ee functions-ka tigidhada taageerada
* La hagaajiyey: Qaab-dhismeedka database-ka oo leh query iyo schema classes sax ah oo loogu talagalay jawaabaha iyo lifaaqyada
* La hagaajiyey: U guurid properties dhab ah halkii attributes loogu talagalay moodelada tigidhada
