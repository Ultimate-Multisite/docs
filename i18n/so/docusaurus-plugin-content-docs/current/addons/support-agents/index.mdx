# Support Agents

Maareynta shabakad dhan kaligaa ma aha hawl fudud. Marka shabakaddaadu bilowdo inay korto, waxaad bilaabi doontaa inaad aragto baahi taageero oo sii kordheysa. Waxay ahaan doontaa fikrad wanaagsan in la keeno dad kaa caawin doona inaad yareyso culayska shaqadaada. Si kastaba ha ahaatee, bixinta gelitaan/oggolaansho buuxda waa khatar waxayna waxyeelleyn kartaa shabakaddaada. Halkaas ayay ka soo gelaysaa add-on-kan Support Agent.

Add-on-kan wuxuu kuu oggolaanayaa inaad abuurto nooc cusub oo isticmaale WordPress ah oo la yiraahdo Support Agent, wuxuuna ku siinayaa xakameyn buuxda oo ku saabsan oggolaanshaha uu isticmaalahaasi leeyahay. Bixinta oggolaansho waa u fudud sida shididda badhan toggle ah, damintiisuna sidoo kale haddii aad go'aansato inaad ka saarto.

## Tilmaamo {#features}

- Liis faahfaahsan oo oggolaanshaha WordPress & Ultimate Multisite ah
- Si fudud u suurageli & u jooji gelitaanka
- Abuur tiro aan xadidnayn oo Account-yada Agent ah

## Rakibid {#installation}

1. Soo geli 'ultimate-multisite-support-agents' galka '/wp-content/plugins/'
2. Ka hawlgeli plugin-ka adigoo adeegsanaya menu-ga 'Plugins' ee WordPress
3. Raac Wizard-ka talaabo-talaabo ah si aad u dejiso plugin-ka
