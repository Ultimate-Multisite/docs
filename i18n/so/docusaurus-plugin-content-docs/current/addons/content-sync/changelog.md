# Diiwaanka Isbeddellada Content Sync

### Nooca 1.1.0 - La sii daayay 2025-12-31 {#version-110---released-on-2025-12-31}
* Cusub: Taageero loogu talagalay isku-xiridda dhammaan noocyada qoraallada (qoraallo, bogag, noocyo qoraal oo gaar ah)
* Cusub: Shaandhaynta nooca qoraalka ee tallaabada xulashada nuxurka
* Cusub: Tallaabada xulashada site-ka isha - ka isku-xir site kasta, ma aha oo keliya site-ka ugu weyn
* Cusub: Hore-u-xulashada site-ka ilmaha ee site-yada template-ka
* Cusub: Ka-soo-saarista warbaahinta `post_content` ee dhammaan noocyada qoraallada
* Cusub: Isku-xiridda taxonomiyada (qaybo, tags, taxonomiyo gaar ah)
* La beddelay: Magaca waxaa laga beddelay "Elementor Template Sync" waxaana loo bixiyay "Content Sync"
* La beddelay: Tallaabada "Xulo Templates" waxaa loo beddelay "Xulo Xog"
* La hagaajiyay: Koodhka gaarka u ah Elementor hadda wuxuu u shaqeeyaa oo keliya nooca qoraalka `elementor_library`

### Nooca 1.0.0 - La sii daayay 2025-12-19 {#version-100---released-on-2025-12-19}
* Sii-dayntii ugu horreysay
* Wizard tallaabooyin badan leh oo loogu talagalay xulashada template-ka iyo site-ka
* Habayn qaybo-qaybo ah oo leh feedback horumar oo waqtiga-dhabta ah
* Isku-xir warbaahineed oo buuxa oo leh dib-u-habayn ID oo otomaatig ah
* Is-dhexgalka dejimaha ee Ultimate Multisite
* Taageerada ka-reebitaanka site-ka iyada oo loo marayo dejimaha shabakadda
* Nadiifinta kaydka Elementor ka dib isku-xirka
* Ilaalinta taxonomy-ga template-ka
