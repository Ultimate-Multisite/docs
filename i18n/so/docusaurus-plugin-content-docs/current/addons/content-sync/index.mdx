# Iswaafajinta Mawduuca {#content-sync}

Multisite Ultimate: Iswaafajinta Mawduuca

Ku-darkan loogu talagalay Multisite Ultimate wuxuu u oggolaanayaa maamulayaasha shabakadda inay mawduuc ka iswaafajiyaan goob kasta una gudbiyaan goobaha kale ee shabakadda. Wuxuu taageeraa dhammaan noocyada qoraallada oo ay ku jiraan qoraallo, bogag, noocyada qoraallada gaarka ah, iyo qaab-dhismeedyada Elementor. Ku-darku wuxuu ka kooban yahay is-dhexgal hage tallaabooyin badan leh oo lagu dooranayo mawduuca iyo goobaha bartilmaameedka, iyadoo leh habayn koox-koox ah iyo jawaab-celin horumar oo waqtiga-dhabta ah.

## Astaamaha {#features}

* Xulashada Mawduuca: Shabakad kaarar muuqaal ah oo leh sawirro yaryar iyo shaandhayn nooca qoraalka ah
* Dhammaan Noocyada Qoraallada: Iswaafaji qoraallo, bogag, badeecooyin, qaab-dhismeedyada Elementor, iyo nooc kasta oo qoraal gaar ah
* Xulashada Goobta: Dooro goobaha loo iswaafajinayo mawduuca, iyadoo laga reebayo goobta isha ah iyo hore-u-xulashada goobta ilmaha ah
* Habayn Koox-koox ah: Wuxuu hal mar u iswaafajiyaa 10 goobood isagoo bixinaya cusboonaysiin horumar
* Iswaafajinta Warbaahinta: Si toos ah ayuu u koobiyeeyaa una dib-u-khariidadeeyaa faylasha warbaahinta (sawirro, fiidiyowyo) laga bilaabo mawduuca qoraalka ilaa goobaha bartilmaameedka
* Iswaafajinta Taxonomy: Wuxuu ilaaliyaa qaybaha, tags, iyo taxonomies gaar ah
* Taageerada Elementor: Taageero buuxda oo loogu talagalay qaab-dhismeedyada Elementor oo leh dib-u-abuurista CSS iyo dib-u-khariidadaynta aqoonsiga qaab-dhismeedka
* Xallinta Isku-dhaca: Ikhtiyaar lagu beddelayo mawduuca jira ama lagu dhaafayo
* Maareynta Cache: Si toos ah ayuu u nadiifiyaa cache-ka Elementor ee goobaha bartilmaameedka ka dib iswaafajinta

## Shuruudaha {#requirements}

* WordPress 5.3 ama ka sarreeya
* PHP 7.4 ama ka sarreeya
* Multisite Ultimate plugin (firfircoon)
* Elementor plugin (ikhtiyaari, waxaa looga baahan yahay iswaafajinta qaab-dhismeedka Elementor)

## Rakibidda {#installation}

1. U geli 'ultimate-multisite-content-sync' galka '/wp-content/plugins/'
2. Ka hawlgeli plugin-ka adigoo adeegsanaya menu-ga 'Kudarro' ee WordPress Network Admin-kaaga
3. Habee dejimaha gudaha Ultimate Multisite → Dejimaha → Iswaafajinta Mawduuca
4. U gudub Ultimate Multisite → Iswaafajinta Mawduuca si aad u bilowdo iswaafajinta
