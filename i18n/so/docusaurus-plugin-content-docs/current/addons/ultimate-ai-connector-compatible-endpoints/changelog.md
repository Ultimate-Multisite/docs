# Ultimate AI Connector ee Changelog-ka Endpoints La Jaanqaadi Kara

## Version 2.0.0 — La sii daayay 2026-04-24

* Cusub: Taageero bixiyeyaal badan — habee endpoints AI badan oo codsiyada u jihay fallback otomaatig ah oo dhex mara bixiyeyaasha.
* Hagaajin: Isdhexgalka SDK ee bixiyeyaal badan oo leh provider IDs sax ah, registration URLs, iyo xadidaynta HTTP filter ee bixiye kasta.
* Hagaajin: Kaarka bixiye cusub hadda si otomaatig ah ayuu u fidaa marka lagu daro; nadiifinta script cache marka plugin la cusboonaysiiyo.
* Hagaajin: Namespace fasalka bixiye ee firfircoon ee eval() iyo hab-dhaqanka is-fidinta otomaatiga ah ee bixiyeyaasha cusub ee lagu daray.
* Hagaajin: Waxay isticmaashaa components Card/CardBody/CardHeader/CardDivider oo deggan (mar dambe tijaabo ma aha) si ay ula jaanqaaddo WordPress 6.9+.
* Hagaajin: DragHandle aan la qeexin waxaa lagu beddelay calaamad unicode grip ah oo loogu talagalay jiididda bixiye si dib loogu habeeyo.
* Hagaajin: Waxaa la tirtiray codsi HTTP oo xannibaya kaas oo kici jiray mar kasta oo bog la raro.
* La hagaajiyay: GitHub Actions workflows waxaa loo cusboonaysiiyay Node.js 24.
