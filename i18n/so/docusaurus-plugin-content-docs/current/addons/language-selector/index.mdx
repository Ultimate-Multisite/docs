# Xulashada Luqadda {#language-selector}

Ultimate Multisite: Xulashada Luqadda ee Is-diiwaangelinta

Ku dar field xulashada luqadda socodka is-diiwaangelinta.

## Rakibid {#installation}

1. U rar 'ultimate-multisite-language-selector' galka '/wp-content/plugins/'
2. Ka hawlgeli plugin-ka adigoo adeegsanaya menu-ga 'Plugins' ee WordPress Network Admin-kaaga

## Hagaha Isticmaalka {#usage-guide}

Add-on-kan bilaashka ah waxaad ugu oggolaan kartaa isticmaaleyaashaada inay doortaan luqadda ay rabaan inay ku arkaan socodka is-diiwaangelinta iyo bogga gelitaanka.

Kadib [markaad hawlgeliso liisankaaga](../../user-guide/getting-started/installing-ultimate-multisite.md) waxaad awoodi doontaa inaad rakibto add-ons-ka Ultimate Multisite ee liisankaagu kuu siinayo gelitaan.

Tag bogga dejimaha Ultimate Multisite ka dibna, menu-ga bidixda ku yaal, guji calaamadda _Add-ons_.

![Bogga add-ons](/img/admin/addons-page.png)

_Tag bogga Add-ons si aad u bilowdo rakibidda Language Selector_

Halkaas, waxaad ka heli doontaa dhammaan add-ons-ka aan bixino.

Add-ons-ka bilaashka ah waxay diyaar u yihiin dhammaan macaamiisheena, iyadoon loo eegin plan-ka subscription-ka ay iska diiwaangeliyeen.

Guji badhanka _Install it Now!_ ee hoos yaal sawirka yar ee add-on-ka. Kadib ka hawlgeli bogga Plugins.

![Bogga add-ons oo leh badhanka rakibidda](/img/admin/addons-page.png)

_Guji badhanka si aad u rakibto add-on-ka_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_Ka hawlgeli boggaaga Plugins_

Hadda, tag menu-ga Settings ee Ultimate Multisite oo geli tab-ka _Network Settings_. Halkaas, qaybta hoose ee bogga, waxaad ka heli doontaa qayb cusub oo la yiraahdo _Languages_.

![Bogga dejimaha](/img/config/settings-general.png)

_Qayb gebi ahaan cusub ayaa soo baxaysa!_

Waad ka tegi kartaa mid madhan si aad dhammaan luqadaha ugu muujiso xulashooyin ahaan, ama waxaad dooran kartaa kuwa aad rabto inaad diyaar ka dhigto. Luqaduhu waa kuwa aad ku rakibtay dejintaada WordPress.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_Liiska luqadaha laga heli karo rakibidda WP-kan_

Kadib markaad kaydiso isbeddellada aad samaysay, menu xulasho oo leh luqadaha aad dooratay ayaa lagu soo bandhigi doonaa bogga Login iyo sidoo kale Sign-up Flow.

## Dejinta Luqadda iyadoo la adeegsanayo URL-ka {#setting-the-language-via-the-url}

Waxaad sidoo kale beddeli kartaa luqadda bog adigoo parameter-ka locale u gudbinaya URL-ka. Haddii aad rabto inaad u wareejiso bog is-diiwaangelin oo Talyaani ah, tusaale ahaan, ku dar ?locale=it_IT URL-ka.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Luqaddu waa inay ku rakibnaataa shabakaddaada WordPress si tani u shaqeyso.
