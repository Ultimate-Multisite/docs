# Shaandheeye: signup_get_available_languages {#filter-signupgetavailablelanguages}

Wuxuu shaandheeyaa liiska luqadaha la heli karo ee isdiiwaangelinta goobta dhinaca hore.

Gudbinta array madhan hook-kan waxay joojin doontaa soo saarista dejinta ee foomka isdiiwaangelinta, waxaana la adeegsan doonaa luqadda caadiga ah marka la abuurayo goobta. Luqadaha aan hore loo rakibin waa laga saari doonaa.

## Halbeegyada {#parameters}

| Magac | Nooc | Sharaxaad |
|------|------|-------------|
| $available_languages | `array` | Luqadaha la heli karo. |
