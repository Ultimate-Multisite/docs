# Action: wu_page_added {#action-wupageadded}

U oggolow horumariyeyaasha plugin inay waxyaabo dheeraad ah socodsiiyaan marka bogag la diiwaangeliyo.

Si ka duwan wu_page_load, oo kaliya shaqeeya marka bog gaar ah la arkayo, hook-kan wuxuu shaqeeyaa xilliga diiwaangelinta bog kasta oo admin ah oo lagu darayo iyadoo la adeegsanayo code-ka WP Ultimo.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $page_id | `string\|\WP_Ultimo_Site_Exporter\Admin_Pages\Base_Admin_Page` | ID-ga boggan. |
| $page_hook | `string\|\WP_Ultimo_Site_Exporter\Admin_Pages\Base_Admin_Page` | Hook-ka loo sameeyay boggan. |

- 2.0.0
