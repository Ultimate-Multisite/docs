# Jiheynta Naqshadda

Tallaabada **Jiheynta Naqshadda** waxay kuu oggolaanaysaa inaad dib u eegto, sifayso, oo dhammaystirto naqshadda muuqaalka ee theme-kaaga ka hor inta Theme Builder uusan abuurin theme-ka dhammaystiran.

## Dulmar

Kadib markaad dhammaystirto Discovery Interview, Theme Builder wuxuu soo bandhigayaa jiheynta naqshaddaada oo leh **soo-bandhig horudhac ah oo desktop iyo mobile ah**. Tani waxay kuu oggolaanaysaa inaad si sax ah u aragto sida naqshaddaadu uga muuqan doonto qalab kala duwan ka hor intaadan go'aansan theme-ka.

## Qaybaha Jiheynta Naqshadda

### 1. Palette-ka Midabbada

Nidaamka midabbada ee theme-kaaga, oo ay ku jiraan:

- **Midabka koowaad** — midabka ugu weyn ee astaanta, loona isticmaalo badhamada, xiriiriyeyaasha, iyo lahjadaha
- **Midabka labaad** — midab dhammaystir ah oo kala-duwanaansho bixiya
- **Midabka lahjadda** — midab muujin ah oo loogu talagalay walxaha muhiimka ah
- **Midabbada dhexdhexaadka ah** — cawl iyo caddaan loogu talagalay asalka iyo qoraalka
- **Midabka qoraalka** — midabka qoraalka koowaad ee akhris-fududeynta

### 2. Typography

Xulashooyinka farta ee loogu talagalay:

- **Farta cinwaanka** — loo isticmaalo cinwaannada bogagga iyo madax-qaybyada
- **Farta jirka** — loo isticmaalo qoraalka faqradda iyo nuxurka
- **Cabbirrada farta** — cabbirro jawaab-celin leh oo ku habboon cabbirrada shaashadaha kala duwan
- **Dhererka xariiqda** — kala-fogaanshaha u dhexeeya xariiqyada si loo fududeeyo akhriska
- **Miisaannada farta** — kala-duwanaansho dhumuc leh, caadi ah, iyo khafiif ah

### 3. Layout iyo Kala-fogaansho

- **Ballaca weelka** — ballaca ugu badan ee nuxurka
- **Padding iyo margins** — kala-fogaanshaha ku wareegsan walxaha
- **Nidaamka grid** — layout tiirar ah oo loogu talagalay naqshad jawaab-celin leh
- **Kala-fogaanshaha qaybaha** — masaafada u dhexeysa walxaha UI

### 4. Walxaha Muuqaalka

- **Qaababka badhanka** — naqshadaha badhanka koowaad, labaad, iyo saddexaad
- **Naqshadaha kaarka** — layout-yada loogu talagalay baloogyada nuxurka
- **Astaamo** — qaabka iyo cabbirka astaanta
- **Xuduudaha iyo hoosaska** — qoto-dheer muuqaal oo khafiif ah
- **Saamaynta hover** — qaabaynta xaaladda isdhexgalka

## Soo-bandhigga Horudhaca

### Horudhaca Desktop

Horudhaca desktop wuxuu muujinayaa naqshaddaada iyadoo ballac buuxa ah (badanaa 1200px ama ka ballaaran):

- Liiska navigation oo buuxa
- Layout dhammaystiran oo leh dhammaan tiirarka
- Typography weyn iyo kala-fogaansho
- Dhammaan walxaha muuqaalka oo cabbir buuxa ah

### Horudhaca Mobile

Horudhaca mobile wuxuu muujinayaa naqshaddaada iyadoo ballac mobile ah (badanaa 375px):

- Navigation jawaab-celin leh (liiska hamburger)
- Layout hal-tiir ah
- Typography iyo kala-fogaansho la hagaajiyay
- Cabbirrada badhanka oo ku habboon taabashada

### Breakpoints Jawaab-celin Leh

Theme Builder wuxuu soo bandhigaa horudhacyo breakpoints-kan:

| Qalab | Ballac | Horudhac |
|--------|-------|---------|
| **Mobile** | 375px | Taleefan portrait ah |
| **Tablet** | 768px | Tablet landscape ah |
| **Desktop** | 1200px | Desktop ballac buuxa ah |
| **Desktop Weyn** | 1920px | Shaashad ultra-wide ah |

## Sifaynta Naqshaddaada

### Samaynta Hagaajinno

Waxaad sifayn kartaa jiheynta naqshaddaada adigoo:

1. **Hagaajinaya midabbada** — beddel midab kasta oo ku jira palette-ka
2. **Beddelaya faraha** — dooro typefaces kala duwan
3. **Wax-ka-beddelaya kala-fogaanshaha** — hagaaji padding iyo margins
4. **Cusboonaysiinaya layout-ka** — beddel ballacyada weelka iyo tiirarka grid-ka
5. **Habeynaya walxaha** — wax ka beddel qaababka badhanka, naqshadaha kaarka, iwm.

### Cusboonaysiinta Horudhaca

Isbeddelladu waxay ka muuqdaan waqtiga-dhabta ah:

- Horudhaca desktop si degdeg ah ayuu u cusboonaysiiyaa
- Horudhaca mobile si degdeg ah ayuu u cusboonaysiiyaa
- Dhammaan breakpoints-ka jawaab-celinta leh waa la cusboonaysiiyaa
- Waxaad u kala wareegi kartaa horudhacyada si aad u xaqiijiso isbeddellada

### Muuqaalka Isbarbardhigga

Isbarbardhig jiheynta naqshaddaada la:

- **Naqshaddii asalka ahayd** — arag waxa isbeddelay
- **Naqshadaha tartamayaasha** — la barbardhig goobaha dhiirrigelinta
- **Noocyadii hore** — ku noqo jiheynno naqshadeed oo hore

## Ansixinta Jiheynta Naqshadda

Markaad ku qanacdo naqshaddaada:

1. **Dib u eeg labada horudhac** — xaqiiji in desktop iyo mobile ay si fiican u muuqdaan
2. **Hubi dhammaan midabbada** — xaqiiji isbarbardhigga iyo helitaanka
3. **Tijaabi typography** — xaqiiji akhris-fududeynta dhammaan cabbirrada
4. **Xaqiiji layout-ka** — hubi kala-fogaanshaha iyo isku-aaddinta
5. **Ansixi naqshadda** — u gudub abuurista theme-ka

## Tallaabooyinka Xiga

Kadib markaad ansixiso jiheynta naqshaddaada:

1. Theme Builder wuxuu abuuraa theme-kaaga dhammaystiran
2. Theme-ka waxaa lagu rakibaa site-kaaga WordPress
3. Waxaad sii habeyn kartaa adigoo isticmaalaya WordPress customizer
4. U gudub [Menus-ka Hospitality](./hospitality-menus.md) ama astaamo kale

## Hababka Ugu Fiican

- **Ku tijaabi qalab dhab ah** — isticmaal taleefanno iyo tablet-yo dhab ah haddii ay suurtagal tahay
- **Hubi akhris-fududeynta** — xaqiiji in qoraalku yahay mid la akhrin karo dhammaan cabbirrada
- **Xaqiiji isbarbardhigga** — isticmaal [Xaqiiji Isbarbardhigga Palette-ka](../abilities/validate-palette-contrast.md) si loo hubiyo helitaanka
- **Tixgeli waxqabadka** — u hagaaji sawirrada iyo faraha xawaare
- **U qorshee nuxurka** — xaqiiji in layout-ku la shaqeeyo nuxurkaaga dhabta ah

## Xallinta Dhibaatooyinka

### Horudhacu Ma Cusboonaysiinayo

- Dib u cusboonaysii bogga
- Nadiifi cache-ka browser-ka
- Isku day browser kale
- Hubi xiriirka internet-ka

### Midabbadu Si Kala Duwan Ayay U Muuqdaan

- Hubi dejinta midabka monitor-ka
- Ku tijaabi qalab kala duwan
- Ku xaqiiji isbarbardhigga midabka qalabka helitaanka
- Tixgeli simulators-ka indho-la'aanta midabka

### Dhibaatooyinka Typography

- Xaqiiji in faylasha fartu ay soo rarayaan
- Hubi cabbirka farta breakpoints kala duwan
- Ku tijaabi nuxur dhab ah
- Tixgeli dhererka xariiqda si loo fududeeyo akhriska
