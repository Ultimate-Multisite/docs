# Superdav AI Agent

Superdav AI Agent waa caawiye WordPress AI horumarsan oo ballaariya Gratis AI Agent isagoo ku daraya xirfado theme-la-socda, awoodaha raadinta internetka oo la xoojiyay, astaamo dhisaha goobta, iyo isku-xiryo otomaatig ikhtiyaari ah. Nooca 1.18.0 wuxuu ku darayaa qalabka akhriska Google Calendar, khariidaynta xiriirrada iyo ka-qaybgalayaasha, albaabbada oggolaanshaha aadanaha, ka-saaridda nuqullada xusuusiyeyaasha, ogeysiisyada TextBee SMS, bixinta adeegga Superdav ee la maareeyo, iyo xirmo wehel Advanced ah oo si gooni ah loo qaybiyo oo loogu talagalay socodsiinta shaqooyinka horumariyayaasha la aamini karo.

## Astaamaha Muhiimka ah

- **Xirfado theme-la-socda** — si otomaatig ah ayuu u waafajiyaa hagidda iyo awoodaha iyadoo lagu salaynayo theme-kaaga WordPress ee firfircoon iyo plugins-ka rakiban
- **Bixiyeyaal raadinta internetka oo badan** — kala dooro Brave Search iyo Tavily si aad u hesho natiijooyin raadin oo hodan ah iyo awoodaha cilmi-baaris horumarsan
- **Foomamka xiriirka dhisaha goobta** — ka samee foomam xiriir oo si buuxda u shaqeeya si toos ah interface-ka sheekada adigoo adeegsanaya plugin-ka foomka ee rakiban
- **Dhisaha goobta oo la xoojiyay** — qorshayaal dhisid goob oo marxalado badan leh oo la hagaajiyay, helitaan plugin oo ka wanaagsan, iyo ka-soo-kabasho khaladaad
- **Xirfado theme oo ku dhex jira** — hagid gaar ah oo loogu talagalay Block Themes, Classic Themes, Kadence Blocks, iyo Kadence Theme
- **Isku-xirka WooCommerce oo la hagaajiyay** — dib ayaa loo habeeyay si loo adeegsado APIs-ka WooCommerce ee asalka ah si loo helo isku-halayn iyo waafaqid ka wanaagsan
- **Ogaanshaha bixiyaha otomaatigga ah** — si otomaatig ah ayuu u ogaadaa una cusboonaysiiyaa plugins iyo themes la heli karo marka la hawlgeliyo ama la demiyaa
- **Abuurista astaanta gaarka ah** — abuur oo geli astaamo SVG gaar ah oo leh nadiifin otomaatig ah oo namespace-safe ah
- **Xaqiijinta isbarbardhigga palette-ka** — hubi lammaanaha midabbada si ay u waafaqaan helitaanka WCAG ka hor inta aan lagu dabaqin theme-kaaga
- **Sahamin sawirro lagu xoojiyay** — soo rar sawirro inta lagu jiro wareysiga sahaminta Theme Builder si ay u saameeyaan jihada naqshadeynta
- **Liisaska martigelinta** — abuur bogag liis cunto iyo cabitaan oo qaabaysan oo loogu talagalay makhaayadaha, kafateeriyada, iyo ganacsiyada martigelinta
- **Horudhacyo desktop iyo mobile** — horay u eeg naqshaddaada qalabka desktop iyo mobile inta lagu jiro xulashada jihada naqshadeynta
- **Xoqidda goobta Heerka 1** — sd-ai-agent/site-scrape hadda si caadi ah ayaa loo heli karaa si loogu falanqeeyo mareegaha tartamayaasha iyo kuwa dhiirrigelinta
- **Qalabka akhriska Google Calendar** — akhri calendars iyo dhacdooyinka la habeeyay si otomaatiggu uga fikiri karo macnaha jadwalka soo socda
- **Khariidaynta xiriirrada iyo ka-qaybgalayaasha** — waafaji ka-qaybgalayaasha calendar-ka isticmaalayaasha goobta iyo xiriirrada la yaqaan ka hor inta aan la abuurin shaqo daba-gal ah
- **Albaabbada oggolaanshaha aadanaha** — hakad geli otomaatigyo xasaasi ah si dib loogu eego ka hor inta aan fariimo la dirin, shaqo dib loo bilaabin, ama isbeddello la samayn
- **Diiwaannada xusuusiyeyaasha iyo ka-saaridda nuqullada** — la soco xusuusiyeyaasha hore loo diray si orodyada otomaatigga ee soo noqnoqda aysan isla qaataha laba jeer u ogeysiin
- **Ogeysiisyada TextBee SMS** — dir ogeysiisyada farriin-qoraal ee la habeeyay iyada oo loo marayo TextBee marka bixiye SMS la hawlgeliyo
- **Bixinta adeegga Superdav ee la maareeyo** — ku xir goobaha la taageero endpoints Superdav ee la martigeliyay, iyadoo dejin otomaatig ah la adeegsanayo halkii laga isticmaali lahaa habayn endpoint oo gacanta ah
- **Xirmada wehelka Advanced** — qalab horumariye la aamini karo oo si gooni ah loo qaybiyo oo loogu talagalay filesystem, database, WP-CLI, REST dispatcher, dhisaha plugin, git snapshot, maareynta isticmaalayaasha, iyo socodsiinta shaqooyinka benchmark

## Shuruudaha

- WordPress 6.0 ama ka sarreeya
- PHP 8.1 ama ka sarreeya
- WordPress Multisite (network-activated) ama rakibid single-site ah
- Plugin xiriiriye bixiye AI firfircoon (Anthropic, OpenAI, ama mid la jaanqaadi kara)
- Gratis AI Agent v1.9.0 ama ka sarreeya (Superdav AI Agent wuxuu ballaariyaa Gratis AI Agent)

## Rakibidda

1. Soo rar faylasha plugin-ka ` /wp-content/plugins/superdav-ai-agent`
2. Ka hawlgeli plugin-ka shaashadda **Plugins** ee WordPress
3. Hubi in Gratis AI Agent sidoo kale la hawlgeliyay
4. U gudub **Gratis AI Agent → Settings** oo habeey bixiyahaaga raadinta internetka, xiriirka adeegga Superdav ee la maareeyo, Google Calendar, ama aqoonsiyada TextBee sida loo baahdo

Xirmada WordPress.org waxay ka kooban tahay astaamaha aasaasiga ah ee Superdav AI Agent. Xirmada wehelka Advanced si gooni ah ayaa loo dhisaa loona qaybiyaa deegaannada halkaas oo qalabka horumariye la aamini karo si cad loogu rakibo loona oggolaado.

## Xirfado Theme-La-Socda

Superdav AI Agent si otomaatig ah ayuu u ogaadaa theme-kaaga firfircoon iyo plugins-ka rakiban, isagoo ku shubaya xirfadda theme-la-socota ee ku habboon macnaha caawiyaha. Xirfadaha la heli karo waxaa ka mid ah:

| Xirfad | Waxa Lagu Dabaqo |
|---|---|
| **Block Themes** | Themes adeegsada qaab-dhismeedka WordPress ee block-ku-saleysan (Full Site Editing) |
| **Classic Themes** | Themes WordPress dhaqameed oo adeegsada templates PHP iyo editor-kii classic ahaa |
| **Kadence Blocks** | Goobo leh plugin-ka Kadence Blocks oo firfircoon |
| **Kadence Theme** | Goobo leh Kadence theme oo ah theme-ka firfircoon |

Eeg [Xirfado Theme-La-Socda](../../user-guide/configuration/theme-aware-skills) si aad u hesho hagid faahfaahsan oo ku saabsan xirfad kasta.

## Bixiyeyaasha Raadinta Internetka

Superdav AI Agent v1.10.0 wuxuu ku darayaa **Tavily** beddel ahaan Brave Search ee su'aalaha raadinta internetka. Labada bixiye waxay bixiyaan natiijooyin raadin waqtiga-dhabta ah waxaana lagu habayn karaa **Gratis AI Agent → Settings → Advanced**.

| Bixiye | Astaamo | Dejinta |
|---|---|---|
| **Brave Search** | Natiijooyin raadin oo degdeg ah, diiradda saaraya asturnaanta | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Natiijooyin raadin oo hodan ah, awoodaha cilmi-baaris horumarsan | [Tavily API](https://tavily.com/) |

Eeg [Raadinta Internet-ka](../../user-guide/configuration/internet-search) si aad u hesho tilmaamaha qaabeynta.

## Jadwalka, Ansixinta, iyo Socodsiinta Ogeysiisyada

Superdav AI Agent v1.18.0 wuxuu ballaariyaa socodsiinnada otomaatiga ah kana gudbiyaa hawlaha nuxurka iyo dhisidda goobta:

- **Qalabka Google Calendar** waxay agent-ka u oggolaadaan inuu akhriyo jadwallada la qaabeeyey iyo faahfaahinta dhacdooyinka marka aqoonsiyada jadwalku diyaar yihiin.
- **Kaaliyeyaasha xiriirka iyo ka-qaybgalayaasha** waxay ka-qaybgalayaasha dhacdada ku waafajiyaan isticmaaleyaasha WordPress ama xiriirrada la yaqaan ka hor inta aan la diyaarin xusuusino ama hawlo daba-gal ah.
- **Albaabbada ansixinta aadanaha** waxay u oggolaadaan otomaatig inuu istaago ilaa isticmaale la oggolaaday uu dib u eego tallaabada la soo jeediyey.
- **Diiwaannada xusuusinta** waxay ka hortagaan xusuusino jadwal ama ogeysiisyo nuqul ah marka otomaatigga dib loo tijaabiyo ama si soo noqnoqota loo jadwaleeyo.
- **Taageerada TextBee SMS** waxay suuragelisaa ogeysiisyada farriin-qoraalka marka bixiyaha TextBee la qaabeeyo.

U adeegso qalabkan caawimaad jadwal-og, daba-galka ka-qaybgalayaasha, iyo socodsiinnada xusuusinta ee weli u baahan dib u eegis aadanaha ka hor inta aan la dirin farriimo xasaasi ah.

## Foomamka Xiriirka ee Dhisaha Goobta

Agent-ka Dhisaha Goobta hadda wuxuu ka abuuri karaa foomamka xiriirka si toos ah interface-ka sheekada. Si fudud u weydii Dhisaha Goobta inuu abuuro foom xiriir, wuxuuna si otomaatig ah u ogaan doonaa plugin-ka foomka ee aad rakibtay (Contact Form 7, WPForms, Fluent Forms, ama Gravity Forms) wuxuuna soo saari doonaa foom diyaar u ah in la isticmaalo.

Eeg [Foomamka Xiriirka ee Dhisaha Goobta](../../user-guide/administration/site-builder-contact-forms) si aad u hesho hagitaan faahfaahsan.

## Astaamaha Theme Builder

Theme Builder ee Superdav AI Agent v1.16.0 wuxuu ku jiraa awooddo cusub oo xooggan:

### Wareysiga Ogaanshaha oo leh Soo-gelinta Sawir

Wareysiga ogaanshaha ee Theme Builder hadda wuxuu ku jiraa tallaabo sawir lagu soo geliyo halkaas oo aad ku bixin karto dhiirrigelin muuqaal ah. Soo geli sawirro astaantaada, goobaha tartamayaasha, ama dhiirrigelinta naqshadda si aad uga caawiso Theme Builder inuu fahmo jihada naqshaddaada.

Eeg [Wareysiga Ogaanshaha](./theme-builder/discovery-interview.md) si aad u hesho hagitaan faahfaahsan.

### Jihada Naqshadda oo leh Muuqaallo Hore oo Desktop iyo Mobile ah

Dib u eeg naqshaddaada labadaba qalabka desktop iyo mobile ka hor inta Theme Builder uusan soo saarin theme-kaaga dhammaystiran. Arag muuqaal-horyo waqtiga-dhabta ah oo ku jira meelaha kala-jebinta jawaab-celinta ee kala duwan.

Eeg [Jihada Naqshadda](./theme-builder/design-direction.md) si aad u hesho hagitaan faahfaahsan.

### Liisaska Martigelinta

Theme Builder hadda wuxuu soo saari karaa bogag liis cunto iyo cabitaan oo qaabaysan oo loogu talagalay maqaayadaha, kafateeriyada, baararka, iyo ganacsiyada kale ee martigelinta. Liisasku waxay ka kooban yihiin qiimeyn, sharaxaad, macluumaad cunto, iyo digniino xasaasiyadeed.

Eeg [Liisaska Martigelinta](./theme-builder/hospitality-menus.md) si aad u hesho hagitaan faahfaahsan.

### Awooddo Cusub

- **[Xaqiiji Isbarbardhigga Palette](./abilities/validate-palette-contrast.md)** — hubi lammaanaha midabada si ay ugu hoggaansamaan WCAG
- **[Abuur Menu](./abilities/create-menu.md)** — abuur menus hagid oo leh calaamado hagid oo kala duwan

Eeg [Qalabka iyo Qaabeynta](./theme-builder/tools.md) si aad u hesho macluumaad ku saabsan qalabka la heli karo iyo qalabka Tier 1 site-scrape.

## Qaabeynta

Dhammaan dejimuhu waxay ku jiraan **Gratis AI Agent → Settings** gudaha maamulka WordPress. Superdav AI Agent wuxuu ku daraa xulashooyinka qaabeynta ee soo socda:

| Dejinta | Sharaxaad |
|---|---|
| **Bixiyaha Raadinta Internet-ka** | Ka dooro Brave Search iyo Tavily (ikhtiyaari) |
| **Tavily API Key** | API key loogu talagalay raadinta internet-ka Tavily (haddii Tavily la doorto) |
| **Adeegga Superdav ee La Maamulo** | Ku xir goobaha la taageero adeegga Superdav ee martigashan iyadoo endpoint-yada si otomaatig ah loo diyaarinayo |
| **Aqoonsiyada Google Calendar** | Qaabee gelitaanka jadwalka si agent-ku u akhriyi karo jadwallada iyo dhacdooyinka |
| **Bixiyaha TextBee SMS** | Qaabee aqoonsiyada TextBee ee socodsiinnada ogeysiiska SMS ee la ansixiyey |

## Xirmada Wehelka Horumarsan

Superdav AI Agent v1.18.0 wuxuu ku daraa xirmo wehel Horumarsan oo si gaar ah loo qaybiyo oo loogu talagalay socodsiinnada horumariye lagu kalsoon yahay. Awoodahan laguma dhex daro xirmada asaasiga ah ee WordPress.org. Ku rakib xirmada wehelka kaliya goobaha ay maamulayaashu si cad ugu kalsoon yihiin agent-ka inuu galo qalabka horumariyaha.

Qalabka Horumarsan waxaa ka mid ah hawlgallada nidaamka faylasha, gelitaanka database-ka, WP-CLI, socodsiinnada REST dispatcher, utilities-ka dhisaha plugin, sawirrada git, kaaliyeyaasha maamulka isticmaalaha, iyo qalabka benchmark. Ku xaddid qalabkan maamulayaal lagu kalsoon yahay oo dib u eeg qaabeynta albaabka-ansixinta ka hor inta aan lagu awoodsiin goobaha production.

- La hagaajiyay amarrada WP-CLI oo ka maqnaa aliases namespace ka dib dib-u-habayn hore
- La hagaajiyay otomaatiga dhacdooyinka oo si deggan ula tacaalaya goobaha aan weli la abuurin jadwallada otomaatiga
- La hagaajiyay awoodda `memory-save` oo adeegsanaysa horgalaha namespace saxda ah ee dhisaha tilmaanta nidaamka
- La hagaajiyay natiijooyinka qalabka scalar oo si sax ah loo duubayo ka hor inta aan loo celin AI
- La hagaajiyay tirakoobka isticmaalka oo si sax ah ula tacaalaya qaabka furaha awoodda ee dhaxalka ah marka laga cusboonaysiinayo noocyo hore
