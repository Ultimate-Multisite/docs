# Shaandheeye: wp_ultimo_woocommerce_checkout_fields_to_clean

Waxay u oggolaanaysaa horumariyeyaasha inay shaandheeyaan fields-ka caadiga ah ee cinwaanka billing-ka WooCommerce.

Maadaama Ultimate Multisite leeyahay fields-ka cinwaanka billing-ka, waxaa macquul noqon karta in la saaro fields-ka caadiga ah ee WooCommerce ku muujiyo foomka checkout-ka. Sida caadiga ah, qiime looma gudbiyo liiskan, sidaas darteed dhammaan fields-ka lagu habeeyay WooCommerce si loo muujiyo waa la hayaa.

### Tan iyo {#since}

- 2.0.0
### Isha {#source}

Waxaa lagu qeexay [`inc/gateways/class-woocommerce-gateway.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-woocommerce/blob/main/inc/gateways/class-woocommerce-gateway.php#L277) xariiqda 277


## Soo celinta {#returns}
Liiska fields-ka la saarayo. Qiimayaasha la heli karo waxaa ka mid ah billing_first_name, billing_last_name, billing_company, billing_country, billing_address_1, billing_address_2, billing_city, billing_state, billing_postcode, billing_email, iyo billing_phone.
