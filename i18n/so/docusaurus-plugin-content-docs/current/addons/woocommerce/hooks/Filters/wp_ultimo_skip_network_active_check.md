# Shaandheeye: wp_ultimo_skip_network_active_check {#filter-wpultimoskipnetworkactivecheck}

U oggolow horumariyeyaasha inay si gaaban uga gudbaan hubinta kicinta shabakadda.

Tani waa waxtar marka la isticmaalayo dejinno ku salaysan composer iyo dejinno kale oo caado ah, sida Bedrock, tusaale ahaan, halka isticmaalka plugin-nada sida mu-plugins uu yahay caadiga.

## Halbeegyada {#parameters}

| Magac | Nooc | Sharaxaad |
|------|------|-------------|
| $skip_network_activation_check | `bool` | Haddii aan ka boodno hubinta iyo in kale, qiimaha caadiga ahi waa false. |

### Tan iyo {#since}

- 2.0.0
### Isha {#source}

Waxaa lagu qeexay [`inc/class-requirements.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-woocommerce/blob/main/inc/class-requirements.php#L272) safka 272


## Soo celinta {#returns}
true haddii aad rabto inaad ka booddo hubinta, haddii kale false.
