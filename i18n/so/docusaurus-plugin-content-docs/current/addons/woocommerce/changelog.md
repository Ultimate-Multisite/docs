# Diiwaanka Isbeddellada Isdhexgalka WooCommerce

Nooca 2.2.0 - La sii daayay 2026-07-01
* Cusub: Qaddarrada canshuurta Ultimate Multisite hadda waxay ka muuqdaan checkout-ka sidii khadad fee WooCommerce oo kala gaar ah, taas oo wadarta canshuurta ka dhigaysa mid ka cad ka hor lacag-bixinta.
* Cusub: Waxaa lagu daray dejin la doorto oo ah "Suspend Memberships Immediately on Failed Renewals" iyo filter-ka `wu_woo_suspend_on_payment_failure` loogu talagalay mareegaha doonaya hakinta cusboonaysiinta fashilantay inta lagu jiro daaqadda isku-dayga WooCommerce Subscriptions.
* Hagaajin: Waxaa la iswaafajiyay xaaladaha WooCommerce Subscription ee ka bixi karay isku-dubbaridka memberships-ka Ultimate Multisite ka dib cusboonaysiin fashilantay ama soo kabatay.
* Hagaajin: Waxaa lagu daray lacagta dukaanka WooCommerce liiska lacagaha Ultimate Multisite markii ay ka maqnayd.
* Hagaajin: Waxaa la ilaaliyay faahfaahinta biil-garaynta macmiilka marka subscribers loo wareejinayo checkout-ka WooCommerce.
* La hagaajiyay: Waxaa lagu daray waafaqid Jetpack Autoloader 5.
* La hagaajiyay: Waxaa la nadiifiyay abuurista xirmada sii-deynta si zips-ka GitHub iyo marketplace ay uga fogaadaan galal staging oo is-dhexyaal ah iyo faylasha horumarinta.
