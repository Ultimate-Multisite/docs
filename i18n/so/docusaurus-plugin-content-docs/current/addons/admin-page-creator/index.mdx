# Admin Page Creator

## Nuxur
Admin Page Creator wuxuu kuu oggolaanayaa inaad si fudud ugu darto bogag maamul oo cusub iyo bogag-hoosaadyo gudaha WordPress admin dashboard. Waxaa loo isticmaali karaa in lagu abuuro bog soo-dhaweyn ama bog casharro ah oo u shaqayn kara sidii bog degitaan oo loogu talagalay isticmaaleyaasha cusub. Waxaad isticmaali kartaa dhisaha bogga sida Beaver Builder, Elementor, Brizy, iyo Oxygen si aad bogagga ugu naqshadayso dhib la’aan.

Waxa kale oo uu bixiyaa taageero custom HTML, CSS iyo JavaScript, iyo sidoo kale xakamaynta gelitaanka oo ku salaysan qorshe iyo doorka isticmaalaha!

## Astaamo
* Taageer Custom HTML iyo CSS
* Taageero JavaScript
* Habka Nuxurka PHP
* Oggolaansho ku salaysan doorka isticmaalaha
* Taageero dhisayaasha bogga

## Hagaha Isticmaalka

In kasta oo WordPress uu ka dhigayo mid aad u fudud in nuxur lagu daro dhinaca hore ee goobaha, abuurista bogag iyo bogag-hoosaadyo dhinaca maamulka waxay u baahan tahay xirfado aasaasi ah oo coding ah. Ultimate Multisite: Admin Page Creator wuxuu ujeedkiisu yahay inuu isla fudaydkaas isticmaalka ee dhinaca hore u keeno dhinaca dambe ee goobaha shabakaddaada.

## Maxaad u isticmaashaa Admin Page Creator?

Waxaa laga yaabaa inaad rabto inaad bog gaar ah ku bixiso qaybta maamulka ee macaamiishaada oo leh casharro, ama laga yaabee inaad rabto inaad ku darto bog gaar ah oo leh su’aalo inta badan la isweydiiyo iyo widget xiriir. Suurtagalnimadu waa aan dhammaad lahayn marka la adeegsanayo Admin Page Creator, waana taas waxa aan ku sahamin doonno casharka maanta.

## Abuurista Boggaaga Maamul ee Gaar ah ee ugu horreeya

Ka dib markaad rakibto oo hawlgeliso Ultimate Multisite: Admin Page Creator, u gudub menu-ga maamulka shabakaddaada oo tag Admin Pages.

![Liiska Bogagga Maamul ee Gaar ah ee ku jira Network Admin](/img/admin/admin-pages-list.png)

Waxaad arki doontaa liis dhammaan bogagga maamulka ee aad ilaa hadda abuurtay (maadaama tani tahay booqashadaadii ugu horreysay, liisku wuu madhnaan doonaa). Hore u soco oo guji **Ku dar Bog Maamul oo cusub**. Waxaa laguu wareejin doonaa tifaftiraha Bogga Maamulka.

### Dooriyaha Nooca Nuxurka

Ultimate Multisite: Admin Page Creator wuxuu ku siinayaa laba ikhtiyaar marka ay timaaddo nuxurka bogga: waxaad isticmaali kartaa tifaftiraha caadiga ah ee WordPress WYSIWYG ama waxaad u wareegi kartaa tifaftiraha HTML.

Haddii aadan aqoon u lahayn HTML, ku ekaaw tifaftiraha caadiga ah ee WordPress. Waxaad awoodi doontaa inaad wax ka beddesho nuxurka bogga sida aad ugu samayn lahayd Post ama Page caadi ah.

## Ikhtiyaarrada Menu-ga

Dhinaca, meta-box-ka General Options, waxaad awoodi doontaa inaad dejiso nooca menu-ga, summadda, iyo wax badan oo kale.

### Noocyada Menu-ga

Boggaaga cusub ee maamul ee gaarka ah wuxuu yeelan karaa laba nooc oo menu ah: menu heer-sare ah iyo submenu.

Bog menu heer-sare ah, sida magacu tilmaamayo, wuxuu boggaaga cusub ee aad abuurtay ku dari doonaa menu-ga maamulka agtiisa bogagga caadiga ah ee WordPress.

Haddii aad doorato ikhtiyaarka heer-sare, waxaad sidoo kale awoodi doontaa inaad doorato kala-horraynta menu-ga iyo icon menu ah.

Haddii aad rabto inaad boggan ka dhigto bog sub-menu ah, waxaad awoodi doontaa inaad doorato bog waalid ah oo boggan lagu xiro. Waxaad boggaaga gaarka ah ku dari kartaa bogagga caadiga ah ee WordPress ama bogaggaaga maamul ee gaarka ah ee heer-sare ah.

<!-- Screenshot unavailable: Admin Page Creator menu type options with label, order, and icon settings -->

_Si buuxda u habee sheyga menu-ga ee Boggan Gaar ah_

## Ikhtiyaarro Sare

Waa inaad awood u yeelataa inaad dhammaan noocyada nuxurka ku darto bogaggaaga maamul ee gaarka ah. Taas waxaa ka mid ah xeerarka CSS iyo faylasha, JavaScript, iyo maktabado dibadeed sidoo kale. Taas waxaad ku samayn kartaa adigoo adeegsanaya Tab-ka Advanced Options.

### Custom CSS

Tab-ka CSS, waxaad ku dari kartaa code-kaaga custom CSS, sidoo kale waxaad ka soo dejin kartaa faylal CSS goobo fog (sida CDNs, Google Fonts, iwm).

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_Custom CSS sidoo kale waa la taageeraa. Waxaad sidoo kale ku dari kartaa faylal style dibadeed!_

### Custom JavaScript

Isla taas ayaa u ansax ah JavaScript. Waxaad ku dari kartaa maktabado dibadeed si aad ugu isticmaasho code-kaaga custom JavaScript.

_Ku dar code-kaaga JavaScript!_

## Oggolaanshooyin

Sidoo kale gudaha Advanced Options, waxaad awoodi doontaa inaad doorato sub-sites-ka heli doona gelitaanka boggan. Tani waa waxtar haddii aad qorshaynayso inaad nuxur gaar ah oo kala duwan u adeegto heerar qorshe oo kala duwan, tusaale ahaan.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Waxaad si buuxda u maamuli kartaa meesha boggani ka soo muuqan doono_

**Muhiim** : Labada shuruudoodba waa in la buuxiyaa si bogga loo soo bandhigo. Taas macnaheedu waa haddii isticmaaluhu yahay macmiil ka tirsan qorshe gaar ah, laakiin aanu lahayn mid ka mid ah doorarka la xushay (ama lidkeeda) boggu kama soo muuqan doono guddigooda.

## Firfircoon

Haddii aad rabto inaad joojiso bog maamul gaar loo sameeyay adigoon tirtirin ama faragelin oggolaanshaha, kaliya ka saar calaamadda ikhtiyaarka Firfircoon.

Joojinta bog heer-sare ah waxay si otomaatig ah u joojin doontaa dhammaan bogagga-hoosaadka ku xiran bogga waalidka ah.
