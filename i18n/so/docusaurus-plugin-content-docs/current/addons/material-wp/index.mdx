# Material WP

Is-dhexgalka maamulka WordPress muddo dheer ayuu isku mid ahaa. Laakiin waligaa ma isweydiisay sida nooca Material Design ee is-dhexgalka maamulka uu u ekaan lahaa? 

Material WP waa theme maamul oo loo sameeyay inuu ku siiyo muuqaalkaas iyo dareenkaas, isagoo kaa caawinaya inaad macaamiishaada siiso Dashboard si ka badan loo habeeyay.

## Sifooyinka {#features}

- Taageerada CSS gaar ah
- Dejinta qaabeynta oo fudud
- Dib-u-habeynta menu-ga
- Soo dejin/dhoofin fudud

## Dukumeenti {#documentation}

**Ku habeynta Guddi-maamulkaaga Material WP**

## Rakibid {#installation}

1. U rar 'material-wp' galka '/wp-content/plugins/'
2. Ka hawlgeli plugin-ka adigoo adeegsanaya menu-ga 'Plugins' ee WordPress

## Hagaha Isticmaalka {#usage-guide}

Is-dhexgalka maamulka WordPress muddo dheer ayuu isku mid ahaa. Laakiin waligaa ma isweydiisay sida nooca Material Design ee is-dhexgalka maamulka uu u ekaan lahaa?
