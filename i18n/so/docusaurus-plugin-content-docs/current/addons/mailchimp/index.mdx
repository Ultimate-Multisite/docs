# Mailchimp {#mailchimp}

Multisite Ultimate: Isku-xirka Mailchimp

Ku xiro shabakaddaada Multisite Ultimate Mailchimp-kaaga. Waxaad qorshayaal gaar ah ku xiri kartaa Liisaska Mailchimp oo si otomaatig ah ugu dari kartaa macaamiisha liisaskaas marka ay isdiiwaangeliyaan.

## Rakibid {#installation}

1. Ku rar 'ultimate-multisite-mailchimp' galka '/wp-content/plugins/'
2. Ka hawlgeli plugin-ka menu-ga 'Plugins' ee WordPress Network Admin-kaaga

## Hagaha Isticmaalka {#usage-guide}

Isku-waafajinta emails-ka macaamiishaada liisaskaaga MailChimp aad bay ugu fududahay Ultimate Multisite: Kordhinta Isku-xirka MailChimp. Casharkani wuxuu ujeedkiisu yahay inuu kaa caawiyo inaad wax walba dejisato!

## Helitaanka furaha API {#getting-an-api-key}

Kahor inta aanan bilaabin inaan galno dejinnada kala duwan ee la heli karo, waxaan u baahanahay furaha API ee MailChimp. Tani waxay Ultimate Multisite u oggolaan doontaa inuu la hadlo MailChimp API si uu u soo helo liisas, kooxo, oo ugu daro cinwaannada email-ka liisaskaas iyo kooxahaas marka isticmaalayaashu isdiiwaangeliyaan.

Raac casharkan si aad u hesho furahaaga API ee MailChimp: [Ku saabsan Furayaasha API | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

<!-- Screenshot unavailable: MailChimp API key settings in Ultimate Multisite Add-on Settings -->

_Ku dar furahaaga API ee MailChimp gudaha_ _**Dejinnada Ultimate Multisite - > Dejinnada Kordhinta -> MailChimp**_

## Habka Isku-xirka {#integration-mode}

Kordhinta Isku-xirka MailChimp waxay bixisaa laba “hab oo isku-xir”, taas oo macnaheedu yahay inay jiraan laba hab oo kala duwan oo aad u isticmaali karto kordhinta si aad u isku-waafajiso cinwaannada email-ka.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_Laba hab oo isku-xir ah oo bixiya dabacsanaan ka badan!_

### Habka 1: Liisas Badan {#mode-1-multiple-lists}

Habka caadiga ah wuxuu adeegsadaa **Liisas Badan**. Habkani wuxuu kuu oggolaan doonaa inaad qorshe kasta u doorato liisas badan oo MailChimp ah. Tani waa xulashada ugu fiican haddii aad isticmaalayaashaada u kala qaybiso liisas kala duwan.

Tusaale ahaan, aan ka soo qaadno inaad Account-kaaga MailChimp ku leedahay 3 liis oo kala duwan: **Newsletter** , **Plan A** , iyo **Plan B**. Waxaad rabtaa in dhammaan isticmaalayaasha, iyadoon loo eegin qorshahooda, sidoo kale lagu daro liiska Newsletter, si aanay u seegin cusboonaysiintaada.

Si taas loo gaaro, waxaad isticmaali kartaa habka Liisas Badan si aad labada liis **Newsletter** iyo **Plan A** ugu darto Dejinnada MailChimp ee Plan A, iyo **Newsletter** iyo **Plan B** Dejinnada MailChimp ee Plan B.

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_Dooro liisaska caadiga ah ee Account-yada cusub iyo Account-yada la joojiyey_

### Habka 2: Hal Liis + Kooxo Badan {#mode-2-single-list--multiple-groups}

MailChimp wuxuu kuu oggolaanayaa inaad liiskaaga u kala qaybiso kooxo yaryar oo isticmaalayaal ah oo loo yaqaan **Kooxo**. Tani waxay kuu oggolaanaysaa inaad yeelato xakameyn dheeraad ah maadaama aad olole u diri karto qayb gaar ah oo ka mid ah macaamiishaada adigoon u baahnayn inaad MailChimp ku abuurto nuqullo badan oo isla ololaha ah (taas oo adkaynaysa in markii dambe la falanqeeyo cabbirrada ololahaas).

Tusaale ahaan, waxaad yeelan kartaa hal liis oo la yiraahdo **My Network Subscribers,** waxaana gudaha ku jira kooxo loogu talagalay **Plan A** iyo **Plan B**.

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_Dooro liisaska iyo kooxaha caadiga ah, haddii aad isticmaalayso habka isku-xirka “hal liis + kooxo badan”_

## Liisaska iyo Kooxaha Caadiga ah {#default-lists-and-groups}

Kadib markaad doorato habka isku-xirka ee ugu macquulsan xaaladdaada isticmaalka gaarka ah, waa waqtigii la dooran lahaa liisaska caadiga ah (iyo kooxaha, iyadoo ku xiran habka). Liiska caadiga ah ayaa la isticmaali doonaa marka qorshe la bixiyay uusan lahayn Liis gaar ah oo la doortay.

Waxaad sidoo kale dooran kartaa liiska caadiga ah ee Account-yada la joojiyey. Mar kasta oo isticmaale uu joojiyo subscription-kiisa ama keeda oo uu ka saaro Account-kiisa ama keeda shabakaddaada, cinwaankiisa ama keeda email-ka waxaa loo wareejin doonaa liiska loo dejiyey Account-yada la joojiyey.

## Hawlgelinta Isdiiwaangelinta Laba-jibbaaran {#enabling-double-subscription}

Bogga isla dejinnada, waxaad awoodi doontaa inaad hawlgeliso **Isdiiwaangelinta Laba-jibbaaran**. Xulashadani waxay isticmaalaha ku dari doontaa labadaba liiska caadiga ah ee la doortay IYO liiska loo doortay qorshaha isticmaalahaas. Haddii xulashadan aan la hawlgelin, isticmaalaha waxaa kaliya lagu dari doonaa liiska loo doortay qorshaha isticmaalaha. Haddii midna aan loo dejin qorshahaas gaarka ah, markaas isticmaalaha waxaa lagu dari doonaa liiska caadiga ah ee la doortay.

<!-- Screenshot unavailable: MailChimp double subscription toggle setting -->

_Waxaad ka dhigi kartaa isticmaalayaashaada inay ku biiraan labadaba liiska caadiga ah iyo liiska qorshaha marka ay isdiiwaangeliyaan_

## Doorashada Liisaska iyo Kooxaha mid kasta oo ka mid ah Qorshayaasha {#selecting-lists-and-groups-for-each-of-the-plans}

Tag **Qorshayaasha**-kaaga oo dooro qorshaha aad rabto inaad wax ka beddesho. Ogow in meta-box-ka **Xulashooyinka Sare**, tab cusub oo la yiraahdo MailChimp la heli doono.

Tab-kaas, waxaad awoodi doontaa inaad doorato liisas badan ama hal liis iyo kooxo badan (iyadoo ku xiran habka isku-xirka) qorshahaas.

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_Dooro Liisaska iyo Kooxaha aad rabto inaad u isticmaasho mid kasta oo ka mid ah qorshayaashaada_

Marka isticmaale cusub uu iska diiwaangeliyo qorshahaas, cinwaankiisa email-ka waxaa lagu dari doonaa liisaska iyo kooxaha la doortay!
