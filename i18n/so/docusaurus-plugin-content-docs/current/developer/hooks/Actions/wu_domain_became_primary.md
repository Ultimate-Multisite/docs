# Ficil: wu_domain_became_primary

Wuxuu dhacaa marka domain uu noqdo domain-ka aasaasiga ah ee site.

Ficilkan waxaa la kiciyaa marka calaamadda primary_domain ee domain loo dejiyo true, ha ahaato marka la abuurayo domain cusub oo aasaasi ah ama marka la cusboonaysiinayo domain jira si uu u noqdo aasaasi.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $domain | `\WP_Ultimo\Models\Domain` | Domain-ka noqday aasaasi. |
| $blog_id | `int` | Aqoonsiga blog ee site-ka ay saameysay. |
| $was_new | `bool` | Haddii kani yahay domain cusub oo la abuuray. |

- 2.0.0
