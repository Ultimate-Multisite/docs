# Ficil: wu_page_added

U oggolow horumariyeyaasha plugin-ka inay socodsiiyaan waxyaabo dheeraad ah marka bogagga la diiwaangeliyo.

Si ka duwan wu_page_load, oo kaliya shaqeeya marka bog gaar ah la arkayo, hook-kan wuxuu shaqeeyaa xilliga diiwaangelinta bog kasta oo admin ah oo lagu darayo iyadoo la adeegsanayo koodhka Ultimate Multisite.

## Halbeegyada {#parameters}

| Magac | Nooc | Sharaxaad |
|------|------|-------------|
| $page_id | `string` | ID-ga boggan. |
| $page_hook | `string` | Magaca hook-ka ee boggan. |

- 2.0.0
