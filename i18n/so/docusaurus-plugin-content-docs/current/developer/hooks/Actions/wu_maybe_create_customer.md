# Action: wu_maybe_create_customer {#action-wumaybecreatecustomer}

Halkan waa meesha aan ku darno hooks-ka loogu darayo customer-&gt;user-ka site-ka weyn sidoo kale, tusaale ahaan.

|------|------|-------------|
