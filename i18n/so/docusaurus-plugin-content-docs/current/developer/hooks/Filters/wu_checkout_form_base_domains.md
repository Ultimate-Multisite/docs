# Filter: wu_checkout_form_base_domains {#filter-wucheckoutformbasedomains}

Wuxuu shaandheeyaa base domains-ka checkout-form ee la wadaago oo aan noqon mapped-domain records.

Isticmaal filter-kan marka is-dhexgal uu bixiyo base domains dheeraad ah oo loogu talagalay beeraha **Site URL** ee checkout form. Domains-ka uu soo celiyo filter-kan waxaa loola dhaqmaa sidii hosts diiwaangelin la wadaago halkii ay ka ahaan lahaayeen custom domains u gaar ah site kasta.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $domains | `array` | Shared base domains laga soo ururiyey checkout form configuration. |

### Since {#since}

- 2.13.0

### Source {#source}

Waxaa lagu qeexay `inc/functions/domain.php`.


## Returns {#returns}

Array ka kooban checkout-form base domains la caadiyeeyey.
