# Sifeeye: wu_minimum_password_length {#filter-wuminimumpasswordlength}

Sifee dhererka ugu yar ee password-ka.

Waxaa la dhaqan geliyaa oo keliya marka wu_enforce_password_rules uu yahay true.

## Halbeegyada {#parameters}

| Magac | Nooc | Sharaxaad |
|------|------|-------------|
| $min_length | `int` | Dhererka ugu yar ee password-ka. Default 12 (waxay la mid tahay Defender Pro). |
| $defender_active | `bool` | Haddii Defender Pro Strong Password uu firfircoon yahay. |

### Laga bilaabo {#since}

- 2.4.0
