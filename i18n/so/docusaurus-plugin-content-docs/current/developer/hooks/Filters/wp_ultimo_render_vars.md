# Shaandheeye: wp_ultimo_render_vars {#filter-wpultimorendervars}

U oggolow horumariyeyaasha plugin inay ku daraan doorsoome dheeraad ah macnaha render guud ahaan.

## Halbeegyada {#parameters}

| Magac | Nooc | Sharaxaad |
|------|------|-------------|
| $args | `array` | Array ka kooban doorsoomayaasha uu gudbiyo wicitaanka render. |
| $view | `string` | Magaca muuqaalka la render-gareynayo. |
| $default_view | `string` | Magaca fallback_view |
