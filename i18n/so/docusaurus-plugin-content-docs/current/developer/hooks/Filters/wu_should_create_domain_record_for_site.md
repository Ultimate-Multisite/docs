# Shaandhee: wu_should_create_domain_record_for_site

Wuxuu shaandheeyaa in Ultimate Multisite ay u baahan tahay inay u abuurto diiwaan domain goob cusub oo la abuuray.

Adeegso shaandheeyahan si aad u joojiso ama dib ugu dhigto abuurista tooska ah ee diiwaanka domain ee goobaha isticmaala domain saldhig ah oo foom checkout la wadaago, host gudaha ah, ama domain uu integration kale si gaar ah u maamuli doono.

## Halbeegyada {#parameters}

| Magac | Nooc | Sharaxaad |
|------|------|-------------|
| $create | `bool` | In diiwaanka domain la abuuri doono iyo in kale. |
| $site | `WP_Site` | Shayga goobta cusub ee la abuuray. |

### Tan iyo {#since}

- 2.13.0

### Isha {#source}

Waxaa lagu qeexay `inc/functions/domain.php`.


## Soo-celinta {#returns}

Boolean muujinaya in diiwaanka domain la abuuro iyo in kale.
