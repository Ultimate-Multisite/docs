# Filter: wu_use_domain_mapping {#filter-wuusedomainmapping}

Go'aami in mapping la isticmaali karo iyo in kale

Caadi ahaan, waxaad rabtaa inaad oggolaato oo keliya mappings firfircoon in la isticmaalo. Si kastaba ha ahaatee, haddii aad rabto inaad isticmaasho loojig ka horumarsan, ama aad oggolaato domains aan firfircoonayn in sidoo kale la map-gareeyo, halkan ka filter garee.

## Parameters {#parameters}

| Magac | Nooc | Sharaxaad |
|------|------|-------------|
| $is_active | `bool` | Mapping-ka ma in loola dhaqmaa sidii mid firfircoon? |
| $mapping | `\Domain` | Mapping-ka aan baarayno |
| $domain | `string` |  |
