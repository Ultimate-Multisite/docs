# Sifeeye: wu_bypass_unset_current_user {#filter-wubypassunsetcurrentuser}

U oggolow horumariyeyaasha inay ka gudbaan koodhka ka-saarista isticmaalaha hadda.

Soo celinta wax kasta oo aan ahayn null waxay ka gudbi doontaa ka-saarista isticmaalaha hadda ee soo galay. Tani waxay faa'iido u yeelan kartaa xaalado qaarkood, tusaale ahaan, marka lala macaamilayo goobo-hoosaadyo loo isticmaalayo sidii goleyaal maamul.

## Halbeegyada {#parameters}

| Magac | Nooc | Sharaxaad |
|------|------|-------------|
| $null_or_bypass | `mixed` | Null si loo sii wado, wax kasta oo kale si looga gudbo. |
| $current_user | `false\|\WP_User` | Shayga isticmaalaha hadda. |
