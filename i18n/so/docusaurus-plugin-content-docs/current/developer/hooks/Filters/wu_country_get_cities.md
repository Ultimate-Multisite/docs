# Sifeeye: wu_country_get_cities

Waxay soo celisaa liiska magaalooyinka ee gobol ku jira waddan.

## Halbeegyada {#parameters}

| Magac | Nooc | Sharaxaad |
|------|------|-------------|
| $cities | `array` | Liiska magacyada magaalooyinka gobolka. Furayaal ma jiraan. |
| $country_code | `string` | Koodhka ISO ee laba-xaraf ah ee waddanka. |
| $state_code | `string` | Koodhka ISO ee laba-xaraf ah ee gobolka. |
| $current_country | `\WP_Ultimo\Country\Country` | Nuqul ka tirsan fasalka hadda. |
