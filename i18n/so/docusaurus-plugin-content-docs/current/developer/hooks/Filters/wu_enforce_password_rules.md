# Filter: wu_enforce_password_rules

Filter in la dhaqan-gelinayo xeerar sir dheeraad ah.

Marka ay true tahay, waxay dhaqan-gelisaa dhererka ugu yar iyo shuruudaha xarfaha. Si toos ah ayaa loo hawlgeliyaa marka dejinta "Super Strong" la doorto ama marka astaanta Strong Password ee Defender Pro ay firfircoon tahay.

## Xuduudaha {#parameters}

| Magac | Nooc | Sharaxaad |
|------|------|-------------|
| $enforce_rules | `bool` | In la dhaqan-gelinayo xeerar dheeraad ah iyo in kale. |
| $strength_setting | `string` | Qiimaha dejinta admin-ka. |
| $defender_active | `bool` | In Defender Pro Strong Password uu firfircoon yahay iyo in kale. |
