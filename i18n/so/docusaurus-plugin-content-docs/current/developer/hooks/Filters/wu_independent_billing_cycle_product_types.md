# Shaandheeye: wu_independent_billing_cycle_product_types {#filter-wuindependentbillingcycleproducttypes}

Shaandhee noocyada alaabta leh wareegyo biil-bixin oo madax-bannaan.

Alaabta leh wareegyo biil-bixin oo madax-bannaan laguma qasbo inay la mid noqdaan muddada biil-bixinta plan-ka ee gaadhiga. Tani waxay faa'iido u leedahay alaabta sida diiwaangelinta domain-ka oo had iyo jeer sannadle loo dallaco, iyadoo aan loo eegin in plan-ku yahay bille ama sannadle.

## Halbeegyada {#parameters}

| Magac | Nooc | Sharaxaad |
|------|------|-------------|
| $types | `array` | Array ah noocyada alaabta leh wareegyo biil-bixin oo madax-bannaan. |
