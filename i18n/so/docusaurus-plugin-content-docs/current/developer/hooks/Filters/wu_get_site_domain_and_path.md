# Shaandheeye: wu_get_site_domain_and_path {#filter-wugetsitedomainandpath}

U oggolow horumariyeyaasha inay wax ka beddelaan lammaaneyaasha domain/path.

Tani waxay faa'iido u yeelan kartaa waxyaabo badan, sida hirgelinta nooc ka mid ah xal staging, serverro kala duwan, iwm.

## Parameterrada {#parameters}

| Magac | Nooc | Sharaxaad |
|------|------|-------------|
| $d | `object` | object-ka hadda jira ee ka kooban furayaasha domain iyo path. |
| $path_or_subdomain | `string` | path/subdomain-kii asalka ahaa ee loo gudbiyey function-ka. |

### Laga bilaabo {#since}

- 2.0.0
### Isha {#source}

Waxaa lagu qeexay [`inc/functions/site.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/site.php#L235) safka 235


## Soo celinta {#returns}
object ka kooban furayaasha domain iyo path.
