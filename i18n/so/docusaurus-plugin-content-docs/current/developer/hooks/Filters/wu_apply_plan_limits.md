# Shaandheeye: wu_apply_plan_limits

U oggolow horumariyeyaasha plugin inay si kooban uga gudbaan xaddidaadaha.

Waxaad isticmaali kartaa shaandheeyahan si aad u socodsiiso code kasta oo aad rabto ka hor inta aan mid ka mid ah xaddidaadaha la bilaabin. Haddii shaandheeyahaagu soo celiyo qiime run u dhigma, habsocodku wuu sii socon doonaa; haddii uu soo celiyo qiime been u dhigma, code-ku wuu soo noqon doonaa, midna hooks-ka hoose ma socon doono.
