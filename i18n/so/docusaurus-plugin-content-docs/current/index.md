# Dukumentiga Ultimate Multisite

Ku soo dhawoow dukumentiga Ultimate Multisite. Ultimate Multisite wuxuu WordPress Multisite u beddelaa madal Website as a Service (WaaS) oo dhamaystiran.

## Qaybaha Dukumentiga {#documentation-sections}

### [Hagaha Isticmaalaha](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

Wax kasta oo aad u baahan tahay si aad u dejiso, u habayso, una maamusho madashaada WaaS. Waxay daboolaysaa rakibidda, marinnada lacag-bixinta, domain mapping, maamulka macmiilka, iyo wax ka badan.

### [Dukumentiga Horumariyaha](/developer/hooks) {#developer-documentation}

Tixraac farsamo oo loogu talagalay horumariyeyaasha wax ku dhisaya Ultimate Multisite. Waxaa ku jira tixraaca hooks, dukumentiga REST API, iyo hagayaasha horumarinta addon.

### [Kordhino](/addons) {#addons}

Dukumenti loogu talagalay addon plugins lacag leh oo Ultimate Multisite ku kordhiya astaamo dheeraad ah sida isdhexgalka WooCommerce, falanqaynta, iibinta domain, iyo wax ka badan.

- [Arrimaha GitHub](https://github.com/Ultimate-Multisite/ultimate-multisite/issues) -- Soo sheeg cillado
- [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) -- Soo dejiso plugin-ka aasaasiga ah
