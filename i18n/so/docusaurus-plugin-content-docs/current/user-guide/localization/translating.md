# Tarjuma Ultimate Multisite {#translating-ultimate-multisite}

Ultimate Multisite waa la horumaray English, laakiin plugin-ka waxaa si buuxda loo tarjumi karaa, waxaana si firfircoon ugu dhiirigelinayaa waxyaabaha tarjumidda.

## Tarjumaada Core Plugin-ka {#core-plugin-translations}

Core Ultimate Multisite plugin-ka waxaa la tarjumaya iyadoo la isticmaalayo **platform-ka tarjumaadda rasmiga ah ee WordPress.org**:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

Tani waa nidaamka tarjumaadda caadiga ah ee bulshada WordPress. Marka tarjumaaddu gaarto heer ogeysiis ku filan, waxay si toos ah ugu qaybsanaysaa dhammaan isticmaalayaasha Ultimate Multisite iyadoo la cusboonaysiiyo plugin-ka WordPress.

**Waxaan u baahanahay caawintaada!** Haddii aad luuqad kale oo aan English ahayn, fadlan ka fikir inaad ka qaybgalto tarjumaadaha. Tarjumaadda qaar ayaa ka caawinaysa. Si aad u bilowdo:

1. booqo [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)
2. dooro luqaddaada
3. riix "Contribute Translation" (Tarjumaad ka qaybgal)
4. waxaad u baahan doontaa account bilaash ah oo WordPress.org

Haddii aad rabto inaad noqoto editor tarjumaado ee luqaddaada (oo awood u leh inuu ogeysiisii tarjumaadaha), la xiriir via [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions).

## Tarjumaada Addon-yada {#addon-translations}

Addon plugins waxaa si gaar ah loo tarjumaa meesha:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Tarjumaadaha addon-yada hadda waxaa sameeyay AI si ay u noqdaan bilowga. Waxay daboolayaan dhammaan luuqadaha taageerada lahayd, laakiin waxay ka koobnaan karaa weedho aan si wanaagsan u muuqda ama qalad. Waxaan dhiirigelinayaa tarjumaadayaasha bani'aadamka inay eegaan oo hagaajinta ay soo jeedaan si ay noqdaan kuwo dabiici ah oo sax ah.

## Tarjumaadaha Lagu Sameeyay Goobtaada {#local-translations}

Haddii aad doorbidayso inaad tarjumo si gaar ah loogu talagalay mashruucaaga adiga oo aan la qaybsan upstream, waxaa jira laba fursad:

### Isticmaalka Poedit {#using-poedit}

Waxaad isticmaali kartaa barnaamijka desktop-ka ah ee **[Poedit](https://poedit.net)** si aad u tarjunto faylka `.pot`:

1. Soo dhaweey Poedit ka [poedit.net](https://poedit.net)
2. Fur faylka `.pot` ee ku yaalla `wp-content/plugins/ultimate-multisite/lang/`
3. Tarjumo qoraallada (strings) ee interface-ka Poedit
4. Hayso -- Poedit wuxuu abuuraa faylalka `.po` iyo `.mo`
5. Dhig faylalka natiijada folder-ka `lang`

### Isticmaalka Loco Translate {#using-loco-translate}

Plugin-ka **[Loco Translate](https://wordpress.org/plugins/loco-translate/)** ee WordPress wuxuu kuu oggolaanayaa inaad si toos ah uga tarjunto dashboard-ka admin-ka adiga oo aan u baahnayn qalab desktop ah. Wuxuu kuu maamulaa samaynta iyo isku-dhafka faylalka tarjuma ee aad.
