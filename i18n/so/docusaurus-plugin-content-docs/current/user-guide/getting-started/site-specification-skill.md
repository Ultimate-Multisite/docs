# Xirfadda Aqoonsiga Website-ga (Site Specification Skill)

**Xirfadda Aqoonsiga Website-ga (Site Specification skill)** waa hab qaabaysan oo aad ku qorto hadafyada website-kaaga, dadka aad u jeeddo, iyo aqoonsiga calaamadda (brand identity). Macluumaadkaas waxaa lagu kaydyaa xusuusta **site_brief**-kaaga, kaas oo agensi-yadu waxay isticmaalaan dhammaan wixii la soo dhaafay si ay bixiyaan caawimaad isku mid ah oo fahmaya macnaha goobta aad ku jirto.

## Waa maxay Site Specification? {#what-is-site-specification}

Site Specification waa habka qorista (documentation) ee:

- **Ujeeddada website-ka**: Waa maxay waxa website-kaagu sameeyo iyo sababta uu u jiro
- **Dadka la bartilmaameeyay (Target audience)**: Cidda soo booqanaya website-kaaga iyo waxyaabaha ay u baahan yihiin
- **Aqoonsiga calaamadda (Brand identity)**: Midabada, qaabka hadalka (tone), iyo qaabka muuqaalka ah ee aad isticmaasho
- **Ujeeddooyinka ganacsiga**: Waa maxay guusha ay u muuqato website-kaaga
- **Qaababka macluumaadka (Content structure)**: Sida website-kaagu loo habeeyay

Aqoonsigan wuxuu noqonayaa **site_brief**-kaaga, xusuus in sii jooga oo agensi-yadu isticmaalaan si ay u fahmaan goobta aad ku jirto.

## Maxay Tahay In La Isticmaalo Site Specification? {#why-use-site-specification}

### Isku Midnimada Xilliyada (Consistency Across Sessions) {#consistency-across-sessions}

Haddii aan lahayn site_brief, waxaad u baahan doontaa inaad dib u sharaxdo ujeeddada website-kaaga muddo kasta oo aad bilowdo session cusub. Marka aad haysato, agensi-yadu waxay si degdeg ah u fahmi doonaan:

- Ujeeddooyinka iyo dadka la bartilmaameeyay ee website-kaaga
- Midabada iyo qaabka hadalka (tone) calaamaddaada
- Qaababka macluumaadka (content structure)
- Ujeeddada ganacsigaaga

### Talooyin Siiro oo Wanaagsan (Better Recommendations) {#better-recommendations}

Agensi-yadu waxay isticmaalaan site_brief-kaaga si ay:

- Isku talo bixiyaan astaamo (features) la jaanqaadaya ujeeddada website-kaaga
- Isku talo bixiyaan qaabab macluumaad oo ku habboon go'aankaaga
- Soo jeedin qaab-dhismeedyo muuqaal ah oo isku mid ah calaamaddaada
- Ka fogaadaan in ay soo jeediyaan astaamo aan la jaanqaadan

### Bilowga Degdegga ah (Faster Onboarding) {#faster-onboarding}

Agensi-yada cusub (ama agensi-yada session-ka cusub) waxay si dhakhso ah u horreebaa iyagoo akhrinaya site_brief-kaaga halkii ay ka weydiin lahayd su'aalo caddeyn.

## Bilowga Site Specification {#initiating-site-specification}

## Habka Qorshaha Goobta (The Site Specification Process) {#the-site-specification-process}

### Tallaabada 1: Ujeeddada Goobta (Site Purpose) {#step-1-site-purpose}

Agent-ku wuxuu weydiinayaa:

```
Maxay tahay ujeeddada ugu muhiimsan ee goobtaada?
- Doorka iibka carruurta (E-commerce store)
- Blog ama meel lagu soo dhigo macluumaad (content site)
- Portfolio ama muujinta shaqooyinka
- SaaS application
- Bulsho ama forum
- Kale: [sharax]
```

Waxaad dooran kartaa qayb ama sharaxi ujeeddadaada.

### Tallaabada 2: Marka Lagu Dhigan Doonayso (Target Audience) {#step-2-target-audience}

```
Marka maxay yihiin dadka ugu muhiimsan ee aad la hadlayso?
- Macaamiisha / bulshada guud
- Xirfadlayaasha ganacsiga
- Barnaamijiyeyaasha / isticmaalayaasha farsamada (technical users)
- Ardayda / waxbarashayaasha
- Kale: [sharax]

Maxay yihiin baahiyadooda ugu muhiimsan?
```

### Tallaabada 3: Macnaha Brandka (Brand Identity) {#step-3-brand-identity}

```
Waa midabada brandkaaga?
- Midabka hore (Primary color): [doonista midabka ama hex code]
- Midabka labaad (Secondary color): [doonista midabka ama hex code]
- Midabka xoojinta (Accent color): [ixtira ah]

Sidee u qeexaysaa qaabka codsiga brandkaaga?
- Xirfad leh / korporada
- Abuuraysan / fanka
- Ciyaar badan / fudud
- Yar / casri ah
- Diirran / saaxiib ah
```

### Tallaabada 4: Ujeedooyinka Ganacsiga (Business Goals) {#step-4-business-goals}

```
Maxay tahay guusha goobtaada?
- Soo soo jiidashada macaamiisha (Generate leads)
- Iibinta alaabta
- Dhisidda bulsho
- Wasaarida aqoonta
- Xaqiijinta awoodda (Establish authority)
- Kale: [sharax]

Waa micne muhiimkaagu?
- Lacagta soo gashiga (Revenue)
- Iskaashiga isticmaalayaasha (User engagement)
- Gaarka macluumaadka la gaarsiiyo (Content reach)
- Isbeddellada macaamiisha (Conversions)
- Kale
```

### Tallaabada 5: Qaababka Macluumaadka (Content Structure) {#step-5-content-structure}

```
Sidee u habaysan tahay macluumaadkaaga?
- Flat (dhammaan macluumaadka si isku mid ah)
- Isku xiran (categories iyo qaybo kale)
- Waqtiga (blog-style)
- Ku salaysan alaabta (catalog)
- Kale: [sharax]

Noocyada macluumaadka aad isticmaasho waa maxay?
- Maqaallada blogga (Blog posts)
- Bogga alaabta (Product pages)
- Baaritaannada shaqada (Case studies)
- Qoraalka farsamada (Documentation)
- Saddaxda muuqaalka (Videos)
- Kale
```

## Sida Agas-askagu U Isticmaalo site_brief-kaaga {#how-agents-use-sitebrief}

### Talooyin Naqshadaynta (Design Recommendations) {#design-recommendations}

Marka aad weydiiso isbeddellada naqshadaynta, agas-askagu wuxuu isticmaalaa site_brief-kaaga:

```
Adiga: "Redesign my homepage"
Agas-askagu: "Marka la eegayo site brief-kaaga, waxaad u bartay haween da'da 25-45 jir ah oo xiisaynaya modaawada mustaqbalka leh. Waxaan abuuri doonaa naqshad xoogga saaraysa alaabta qurxinta gacanta iyo qiimaha mustaqbalka leh ee aad leedahay."
```

### Soo Jeedinta Faa'iidada (Feature Suggestions) {#feature-suggestions}

Agas-askagu wuxuu soo jeedinayaa faa'iidooyin la jaanqaadaya ujeeddadaada:

```
Adiga: "What features should I add?"
Agas-askagu: "Haddii ay tahay dukaan e-commerce oo diiradda saaraya xiriirka macaamiisha, waxaan ku talinayaa: barnaamij lagu xoojiyo xiriirka macaamiisha (loyalty program), dib u eegista macaamiisha, newsletter email ah, iyo soo jeedinta alaabta."
```

### Qaababka Macluumaadka (Content Structure) {#content-structure}

Agas-askagu wuxuu soo jeedinayaa habka loo qaabeeyo macluumaadka iyadoo la isticmaalayo qaabka aad qorsheysay:

## گرما-باش (Best Practices) {#best-practices}

### وردی و دقیق باشید (Be Specific) {#be-specific}

"همه مخاطبان" 대신, আপনার আসল শ্রোতাদের বর্ণনা করুন:

- ✓ "২৫-৪৫ বছর বয়সী নারীরা, যারা টেকসই ফ্যাশন পছন্দ করেন"
- ✗ "সবাই"

### নিয়মিত আপডেট করুন (Update Regularly) {#update-regularly}

আপনার সাইট যত উন্নত হবে, আপনার site_brief আপডেট করতে ভুলবেন না:

- যখন আপনি নতুন কোনো শ্রোতাদের দিকে মনোযোগ দেবেন
- যখন আপনি নতুন পণ্যের লাইন যোগ করবেন
- যখন আপনার ব্র্যান্ডের পরিচয় পরিবর্তন হবে
- যখন আপনার ব্যবসার লক্ষ্য বদলাবে

### একই শব্দ ব্যবহার করুন (Use Consistent Terminology) {#use-consistent-terminology}

প্রতিটি সেশনে একই শব্দ ব্যবহার করুন:

- ✓ সবসময় "sustainable jewelry" বলুন (যেমন "eco-friendly jewelry" বা "green products" নয়)
- ✓ শ্রোতাদের প্রতি ধারাবাহিকভাবে একই উপায়ে উল্লেখ করুন

### প্রেক্ষাপট যোগ করুন (Include Context) {#include-context}

এজেন্টদের আপনার সিদ্ধান্ত বুঝতে সাহায্য করার জন্য পটভূমি দিন:

- "আমরা এমন পেশাদারদের লক্ষ্য করছি যারা দামের চেয়ে গুণমানকে বেশি গুরুত্ব দেয়"
- "আমাদের শ্রোতারা প্রযুক্তি সম্পর্কে অভিজ্ঞ এবং আধুনিক ডিজাইন আশা করে"
- "আমরা একটি বুটস্ট্র্যাপ স্টার্টআপ, তাই আমাদের সাশ্রয়ী সমাধান দরকার"

## থিম বিল্ডার অনবোর্ডিংয়ের সাথে সম্পর্ক (Relationship to Theme Builder Onboarding) {#relationship-to-theme-builder-onboarding}

Site Specification দক্ষতাটি **Theme Builder onboarding flow** এর মধ্যে যুক্ত করা হয়েছে। যখন আপনি অনবোর্ডিং শেষ করবেন, তখন আপনি যে তথ্যগুলো দিয়েছেন তার ভিত্তিতে আপনার site_brief স্বয়ংক্রিয়ভাবে তৈরি হয়ে যাবে।

আপনি যদি চান তবে Site Specification স্বাধীনভাবেও চালাতে পারেন যদি আপনি চান:

- প্রাথমিক সেটআপের পরে আপনার স্পেসিফিকেশন উন্নত করতে
- আপনার সাইট বিকশিত হওয়ার সাথে সাথে আপনার site brief আপডেট করতে
- Theme Builder শুরু করার আগে একটি বিস্তারিত স্পেসিফিকেশন তৈরি করতে

## সমস্যা সমাধান (Troubleshooting) {#troubleshooting}

**আমার site_brief ব্যবহার করা হচ্ছে না**
- নিশ্চিত করুন যে এজেন্টের মেমোরিতে অ্যাক্সেস আছে
- এজেন্টকে "আমার site brief মনে করিয়ে দিতে" বলুন
- আপনার সেটিংস-এ মেমরি চালু আছে কিনা তা পরীক্ষা করুন

**Waxaan rabaa inaan dib u bilaabo meel cusub oo `site_brief` cusub**
- Weydii agentka: "Fura site brief-kayga oo bilow cusub"
- Markaas dib u fuli Site Specification

**Agentku wuxuu siinayaa talooyin aan la jaanqaadin site_brief-kaygu ahayn**
- Weydii agentka inuu "dib u eego site brief-kayga"
- Cusboonaysii site_brief-ka haddii uu ka soo horjeedo (outdated)
- Bixinta macluumaad dheeri ah talooyinkaaga

## Tallaabooyinka Xiga {#next-steps}

Marka aad qorsheysid site specification-kaaga:

1. **Isticmaal Theme Builder**: Samee theme gaar ah oo ku salaysan site_brief-kaaga
2. **Dib u eeg Naqshadaynta (Refine Design)**: Isticmaal xirfadda Design System Aesthetics si aad u qabato shaqada naqshadaynta faahfaahanka leh
3. **Qorshedo Waxyaabaha La Qabsanayo**: Weydii agentka talooyin ku saabsan qaab dhismeedka waxyaabaha la qabsanayo (content structure)
4. **Dhis Waxyaabaha Muhiimka ah**: Codsi ka samee waxyaabaha muhiimka ah ee la jaanqaadaya hadafka ganacsigaaga
