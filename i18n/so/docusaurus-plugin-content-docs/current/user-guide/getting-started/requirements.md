# Shuruudda Ultimate Multisite

Marka hore oo aad is weydiin karto ka hor intaadan iibsato Ultimate Multisite waa maxay shuruudaha loo baahan yahay inaan plugin-kaas u fuliyo. Waa hagaag, la'aanta, waxaa jira shuruudo aasaasi ah oo kaliya oo loogu talagalay inaan fulino Ultimate Multisite: **magaca domain-ka** (domain name), **bixiyaha hosting-ka** wanaagsan, iyo **warbixinta WordPress Multisite**. Waa intaas oo dhan aad u baahan tahay!

## Magaca Domain-ka {#domain-name}

Marka hore, aan ka hadalno magaca domain-ka.

Marka aad dooranayso magaca domain-ka, xusuusnow waxa gaarka ah eeada (niche) iyo waxa aad samayn doonto Ultimate Multisite-ka: Ma rabtaa inaad samayso hay'ad (agency)? Ma rabtaa inaad iibiso websites la mid ah dukumeentiyada shoocaha? Mise ma rabtaa inaan bixiyo hosting-ka WordPress? Wax walba waa suurtagal iyadoo la isticmaalayo Ultimate Multisite.

Plugin-keena wuxuu bixiyaa adeegyo guud (subscription services), ururinta lacagta, foomamka checkout-ka, voucher-yada ruxa (discount vouchers), iyo isgaarsiinta macaamiisha. Waxaa jira shaqo badan oo loo baahan yahay si loo install gareeyo, loo hagaajiyo (configure) oo loo ilaaliyo WordPress Multisite, taasoo Ultimate Multisite ka fududaynaysa.

Niche kasta oo aad doorato waxaa la maamuli karaa iyadoo la isticmaalayo plugin-keena haddii aad iibiso websites sida adeeg! Hubi inaad doorato magaca domain-ka oo la jaanqaadaya qorshaha ganacsigaaga.

## Bixiyaha Hosting-ka (Hosting Provider) {#hosting-provider}

Hadda oo aad heshay magaca domain-ka soo jiidashada ah ee muujinaya niche-kaaga iyo ganacsigaaga, waxaad u baahan doontaa inaad doorato shirkad hosting wanaagsan si aad u kaydsato shabakadahaaga (network) Ultimate Multisite-ka. Waxaa jira fursado wanaagsan oo suuqa ka ah. Qaar ka mid ah bixiyaha hosting-ka waxay si fiican ugu habaysadaan Ultimate Multisite, taasoo noqon karto arrin muhiim ah iyadoo ku xiran waxa aad qorshaynayso inaad dhisato.

Ultimate Multisite-ku dhammaan hosting providers ku shaqeyn doonaa. Waxa kaliya ee kala duwanaanshaha waa in qaar ka mid ah shirkadaha hosting sida _WPMU DEV Hosting_, _WP Engine_, _Closte_, _Cloudways_, _Gridpane_ iyo _Runcloud_ ay si xooggan ula xiriiraan mapaynta domain-ka. Qaybsi kale, waxaad u baahan kartaa inaad si gacanta ah ugu mapayso custom domains-ka. Waa fikrad wanaagsan in aad hesho hosting provider oo taageera AutoSSL (Automatic SSL). _Closte_, _Runcloud_ iyo _cPanel_ waxay taageeraa arrintan.

Marka la eego qorshaha server-ka, ma jirto qaab gaar ah oo ku xiran sababta, sababtoo ah wuxuu ku xiran yahay astaamaha iyo shaqooyinka aad bixisay iyo cabbirka shabakadahaaga. Shared hosting (hosting la wadaag) waxay noqon kartaa bilaawga wanaagsan, laakiin marka shabakadahaagu bilaaba inay koroddoonaadaan, waxaa laga yaabaa inaad u baahan doonto VPS gaar ah.

## WordPress Multisite {#wordpress-multisite}

Hadda, aan ka hadalno wax yar oo ku saabsan installation-ka WordPress Multisite.

WordPress Multisite wuxuu kuu oggolaanayaa inaad hesho shabakad (network) oo websites ah oo ku jira installation kaliya. Tani waa astaamood la dhisay (built-in), laakiin ma shaqeeyo si toos ah (not active by default). Waxaad u baahan tahay inaad furto astaamadan gudaha installation-ka WordPress iyo control panel-ka web hosting-kaaga.

In aad hesho WordPress oo leh Multisite la furay ** waa aasaaska Ultimate Multisite**. Maadaama isticmaalka ugu weyn ee Ultimate Multisite uu yahay inuu ka caawiyo isticmaalayaasha inay maamulaan iyo iibiyaan websites-ka iyagoo adeegsada (as a service), tani waa inay la samaysaa WordPress oo leh Multisite la furay.

Waxaan leenahay **maqal qoto dheer** oo tilmaamaya tallaabo tallaabo ah waxa loo baahan yahay si aad u furto astaamaha Multisite-ka installation-ka WordPress-kaaga.

Marka aad Multisite ku furto WordPress, wuxuu diyaar u yahay in la install gareeyo Ultimate Multisite. Si aad si sax ah ugu install gareyso Ultimate Multisite, waa inaad haysataa:

  * WordPress: v5.3+ (La talinayaa: Version-ka ugu dambeeya ee stable)

  * PHP: 7.4.x (Taageerada > 8.0 ayaa soo socota)

  * MySQL: v5+ (La talinayaa: 5.6, haddii aanad heyn 8.0 oo ku jira hosting provider-kaaga)

Tani waa shuruudaha software-ka oo kaliya ee Ultimate Multisite.

Xusuusnow in Ultimate Multisite uu u shaqayn karo PHP 8.0 laakiin waxaan ku talinaynaa inaad si wanaagsan ugu isticmaasho PHP 7.4.x.

Sidoo kale, waxaan ku talinaynaa inaad si fiican ugu install gareysid **main domains** (domain-yada koowaad), ma aha subdomains ama subdirectories. Waxaa laga yaaba inuu shaqeeyo subdomains/subdirectories, laakiin waxaa laga yaaba inuu soo saaro dhibaatooyin.

Si aad hesho hagitaan faahfaahsan oo ku saabsan sida loo install gareeyo Ultimate Multisite, **ku riix halkan**.
