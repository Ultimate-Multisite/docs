# Waxyaalad la'aanta la'aanta ah oo ay jiraan Plugin-yada kale {#known-incompatibilities-with-other-plugins}

Markaad install gareysid oo aad furto (activate) Ultimate Multisite, waxaad la kulmi kartaa dhibaatooyin sababta ay u yeeshaan isku dhafka (incompatibilities) ee aan la xalayn mid ka mid ah plugin-yada kale.

Waxaan si firfircoon ugu shaqaynaynaa inaan yareyno liiskaas markasta oo la soo saar kasta, laakiin haddii plugin ay ku jirto halkan, waxaa laga yaabaa inaad u baahan tahay inaad ka furto (deactivate) si aad hesho Ultimate Multisite Network oo si buuxda u shaqaynaya.

Waa kan liiska iyo waxa aad samayn karto xaaladahaas:

**Qaar ka mid ah plugin-yada WPMU DEV** Qayb ka mid ah plugin-yada _iyaba lahayd_ (no longer maintained) ee WPMU DEV waxaa loo yaqaanaa inay sababtaan in habka diiwaangelinta Ultimate Multisite uu joojiyo tallaabada ugu dambeysa (oo isticmaaluhu awood u la'aado inuu abuuro account). Waa kan liiska plugin-yada sababta ay keeni karaan arrintan:

  * Signup Language: _Ultimate Multisite wuxuu leeyahay add-on oo bixiya habka isku midka ah, oo loo yaqaano Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Tani waa qayb ka mid ah Ultimate Multisite, sidaasina ma baahanin plugin kale._

  * New Blog Templates: _Tani waa qayb ka mid ah Ultimate Multisite, sidaasina ma baahanin plugin kale._

  * Set Password on Multisite Blog Creation: _Tani waa qayb ka mid ah Ultimate Multisite, sidaasina ma baahanin plugin kale._

  * Signup TOS: _Tani waa qayb ka mid ah Ultimate Multisite, sidaasina ma baahanin plugin kale._

  * Anti-Splog: In kasta oo ay awood la'aanti badan tahay, waxay sidoo kale _waa qayb ka mid ah Ultimate Multisite._

**Fiiro Gaar Ah** : Plugin-yada WPMU Dev ee kale dhammaantood waxay si caadi ah u isticmaali karaan Ultimate Multisite. Tusaalo waxaa ka mid ah _Smush_, _Forminator_, _Defender,_ iwm.
