# Khaladinta faylka Sunrise-ga

Faylka sunrise.php waa fayl gaar ah oo WordPress uu raadinayo marka uu isku dhafka (bootstraps) u samaynayo. Si WordPress-ka uu awoodi karo inuu ogaado faylka sunrise.php, wuxuu u baahan yahay inuu ku yaalo folder-ka **wp-content**.

Marka aad furto Ultimate Multisite oo aad raacdo hagaha (setup wizard) sida kan aad aragtay screenshot-ka, Ultimate Multisite wuxuu isku dayaa inuu nuqul sameeyo faylkaaga sunrise.php folder-ka wp-content.

<!-- Screenshot aanu hayn: bogga hagaha oo muujinaya tallaabada soo saarista sunrise.php -->

Inta badan, waxaan ku guuleynaa inaan si guuleystiran u nuqulno faylka oo dhammaan waxyaabaha shaqeeyaan. Si kastaba ha ahaatee, haddii wax aan si sax ah loo dejin (sida xaddidda furitaanka folder-ka), waxaa laga yaabaa inaad la kulanto xaalad aysan awoodin Ultimate Multisite inuu nuqul sameeyo faylka.

Haddii aad akhrisid farriinta qaladka Ultimo si aad u ogaato waxa dhacay, waxaad arki doontaa in ay tahay tanas oo dhacday: **Sunrise copy failed** (Nuqulka Sunrise-ga ayaa fashilmay).

<!-- Screenshot aanu hayn: Farriinta qaladka oo muujinaysa "Sunrise copy failed" -->

Si aad u hagaajisid, waxaad si fudud nuqul sameyn kartaa faylka sunrise.php ee ku jira folder-ka wp-ultimo plugin-kaaga oo aad ku dhejin karto folder-ka wp-content-kaaga. Marka aad samayso tani, dib u soo gal bogga hagaha (wizard page) oo hubinta waa inay si sax ah u socotaa.

<!-- Screenshot aanu hayn: Maamulka faylka oo muujinaya sunrise.php gudaha folder-ka wp-ultimo plugin --> Haddii ay tahay arrintan, waxaa laga yaaba in la baahdo in la hubiyo xaddidda furitaanka folder-kaaga si looga fogaado dhibaatooyinka mustaqbalka (ama kaliya Ultimate Multisite aysiina, laakiin sidoo kale plugin-yo iyo themes kale).

**Health Check tool** (Aaladda Hubinta Caafimaadka) ee ku jirta WordPress (waxaad u heli kartaa **admin panel > Tools > Health Check** boggaaga asalka ah) waxay kuu sheegi kartaa haddii xaddidda furitaanka folder-kaagu ay tahay qiimo abuuraya dhibaatooyin WordPress-ka.
