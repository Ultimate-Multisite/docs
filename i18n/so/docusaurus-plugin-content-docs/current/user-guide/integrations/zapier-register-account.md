# وقفو: бүртгэл gacanta Zapier-ka loo isticmaal {#event-register-an-account-via-zapier}

Maqaalka [Integrating Ultimate Multisite with Zapier](zapier.md) ku bilowaynaa sida loo isticmaalo Zapier si loogu sameeyo waxyaabo kala duwan oo ka mid ah Ultimate Multisite iyadoo la adeegsanayo triggers iyo events. Maqaalkan, waxaan kuu sheegi doonaa sida aad ula isdhegaysan karto apps-ka saddexaad (3rd party applications). Waxaan u isticmaalaynaa Google Sheets oo ah ilaha macluumaadka, ka dibna waxaan u dirnaa Ultimate Multisite si loo sameeyo бүртagaan.

Ka hor intaadan la xiriirin Zapier, tag **Ultimate Multisite > Settings > API & Webhooks** oo hubi in API-ga uu furan yahay (enabled). Marka Zapier ku weydiiyo macluumaadka ka mid ah account-ka Ultimate Multisite, nuqul qaadarka API Key iyo API Secret ee screen-kan.

![API and Webhooks settings with API Key, API Secret, and Enable API options](/img/admin/settings-api-webhooks.png)

Marka hore, waxaad u baahan tahay inaad samayso **Google Sheet** oo ku jirta Google Drive-kaaga. Hubi inaad si sax ah u qeexdo qof kasta oo column-ka ah si aad dalkii macluumaadka ugu soo gali karto si fudud.

Marka aad samayso Google sheet, waxaad galaysaa account-ka Zapier-kaaga adigu waxayna bilaawdaa inaad samayso zap cusub (zap).

Duraanka search field ee **"App event"** ku dooro **"Google Sheets"**.

Ka dib, dalka "**Event** " ku dooro **"New spreadsheet row**" oo riix "**Continue**".

Tallaabada xigta waxay weydiin doontaa inaad doorato **Google Account**-kaas Google Sheet-ka lagu kaydsan yahay. Sidaa darteed, hubi in account-ka Google saxda ah uu la xiriir yahay.

Duraanka **"Set up trigger"**, waxaad u baahan doontaa inaad doorato oo aad qeexdo google spreadsheet iyo worksheet-ka aad isticmaali doonto halkaas oo macluumaadka ka soo socdo. Si kastaba ha ahaatee, si fudud ku buuxiso kuwaas oo riix "**Continue**".

Tallaabada xigta waa **"test your trigger"** si aad u hubiso in Google Sheet-kaaga si sax ah loo xiriiray.

Haddii imtixaankaagu uu guuleysto, waxaad arki doontaa natiijada oo muujinaysa qaar ka mid ah qiimaha laga soo qaaday spreadsheets-kaaga. Riix "**Continue**" si aad u sii socoto.

Tallaabta xigta waa inaad dejisid ficilka labaad ee abuuraya ama diiirta (register) account-ka. Marka aad doorato meesha raadinta (search field), dooro "**Ultimate Multisite(2.0.2)** ".

Dagaalka "**Event**" (Maadoowga) ku dooro "**Register an Account in Ultimate Multisite**", ka dibna riix badhka "**Continue**".

Dagaalka "**Set up an action**" (Dejinta ficilka), waxaad arki doontid meelo kala duwan oo loogu talagalay macluumaadka macaamiisha, xubnaha (memberships), badeecadaha (products), iwm. Waxaad ku xiraysaa qiimaha aad ka soo qaadatay google sheet-kaaga iyo inaad u siiso meelka saxda ah halka ay la buuxinayaan sida ay muujisada hoose u aragayaan sawirkaas.

Marka aad xirto qiimaha, waxaad isku dayi kartaa in aad ficilka (action) ku baaro.
