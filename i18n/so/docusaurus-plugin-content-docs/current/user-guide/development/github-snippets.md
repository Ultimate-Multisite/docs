# sida isticdaalka Ultimate Multisite snippets-kaaga ku jira GitHub repository-gaaga

GitHub repository-gaagga waxaa ku jira code snippets ah oo ay si badan u codsadaan isticmaalayaasha Ultimate Multisite ee doonaya inay ku dariyaan waxyaabo yaryar sida ku darid script-ka Google Analytics bogagga isdiiwaangelinta ama qarinta meta box-ka laga soo qaado admin dashboard-ka.

Maqaalkan wuxuu kuu sheegi doonaa sida loo isticmaalo ama meesha lagu dhigo koodhkan si gaar ah.

Waxaad ka heli kartaa snippets-ka linkii hoose.

https://github.com/next-press/wp-ultimo-snippets/

Waxaa jira 2 hab oo aad ku dari karto koodhka:

  1. faylka functions.php ee theme-kaaga.

  2. Must-Use Plugins (mu-plugins).

# sida loo dariyo snippet-ka faylka functions.php ee theme-kaaga.

  1. Tagashada admin dashboard-ka WordPress Network-kaaga oo aando Themes > Theme Editor (Waxaad arki doontaa screenshot-ka hoose).

  2. Bogga "Edit Themes"-ka, hubi in aad dooratay theme-kaaga socda ee meesha uu ku yaalo dropdown-ka ee dhinaca sare ee dhabarkaaga (#3 ee screenshot-ka hoose).

  3. Guji faylka functions.php oo ku jira qaybta "Theme Files" si aad u soo dejiso faylka. Hoos u soo saar (scroll down) oo ku dheji snippet-ka Ultimate Multisite-ka laga helo GitHub repository-gaaga.

# sida loo sameeyo Must-Use Plugins (mu-plugins)

WordPress wuxuu leeyahay astaamaha kuu oggolaanaya inaad soo dejiso habab cusub oo la yiraahdo "Must-Use Plugins", ama si kooban "mu-plugins".

Mu-plugins gaarka ah ee khasbadda ah waxaa lagu soo dejinayaa ka hor dhammaan plugins-ka caadiga ah, waxayna ma lahayn fursad loo xirmo (deactivate). Network multisite-ka, koodhka ku jira mu-plugins-kaas wuxuu la soo socdaa dhammaan bogagga ku jira installation-kaaga.

1\. Isticmaal FTP ama SSH si aad u heli karto faylka filesystem-ka WordPress-kaaga.

2. WordPress installation-ka `wp-content` directory-ga gudaha ku samayso folder cusub oo la yiraahdo: `mu-plugins`.

<!-- Sawirka ma lahayn: File manager-ka wuxuu muujinayaa directory-ga wp-content iyo folder-ka mu-plugins -->

3. Computer-kaaga, Notepad ama code editor kasta oo aad isticmaasho ku samayso fayl PHP cusub oo la yiraahdo `wu-snippet.php`.

4. Ultimate Multisite code snippet-ka aad ka helisay GitHub repository-ga ku dheji faylkaas oo ku kaydi. Waxaad sidoo kale kugu dari kartaa koodkan sare ee code snippet-ka si aad u magacawdo mu plugin-kaaga.
