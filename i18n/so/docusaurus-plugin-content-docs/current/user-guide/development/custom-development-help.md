# Sidaa Lagu Soo Helid Caawimaad Horumar Ah Gaar ah oo Ka Shaqeeya Codeable? {#how-to-get-custom-development-help-from-codeable}

Mararka qaarkood, si aan u samayno hababka shirkadda (network flows) sida aan doonayno si sax ah, waxaan u baahanahay hagaajin gaar ah. Haddii aad tan raadinayso, waxaan ku talinaynaa [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), meel ay ka heli karto horumarsan oo leh aqoon ballaaran oo ku saabsan Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) waa platformka isku xirta macaamiisha iyo developer-ka WordPress ee aqoonta lahayd, kuwaasoo awood u leh inay ka shaqeeyaan mashaariic kasta oo cabbir ah. Waxay leeyihiin hab codsi si taxadar leh oo ay hubiyaan in aad la isku xirto khabiirro gaar ah oo ku aqoon leh hawsha aad rabto inaad qabato.

### Sida Loo Sameeyo Mashruuc (Project) Codeable-ka? {#how-to-create-a-project-in-codeable}

Marka hore, tag [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) oo riix badhanka **Start a Project** (Bilow Mashruuc).

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Hadda, si aad u soo gudbiso mashruuc, riix dhinaca bidix ee **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

In "I need help with" (Waxaan u baahanahay caawimaad), dooro **Customizing** (Hagaajinta):

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

In "My" (Midigga), dooro **Plugin** (Addon):

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Markaas riix badhanka **Continue** (Dabac).

<!-- Screenshot unavailable: Codeable project form Continue button -->

Hadda, waa waqtiga aad sharaxdo mashruucaaga iyo baahiyahaaga habka ugu wanaagsan ee aad u samayn karto. Isku day inaad si badan macluumaad uga bixiso!

<!-- Screenshot unavailable: Codeable project description form -->

Si ka caawin lahayd qiimaynta (pricing), jawaab weydiisii soo socda.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Kadib, haddii aanad wel haysan account [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) ahayn, samee mid oo kaaga soo abuuro oo guji badhanka **Publish My Project** (Ballaburinta Mashruuca Miyga).

<!-- Sawirka ma laha: Abuurista account Codeable iyo badhanka Publish My Project -->

Marka aad ballaburiso mashruuca [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) ah, platform-ku wuxuu kuu xiri doonaa horumariyeyaasha (developers) kuwaas oo ku habboon aqoonta loo baahan yahay mashruucaaga.

Habkani wuxuu kaa caawin doonaa inaad hesho qofka saxda ee shaqadaada, kaas oo lahayd tajiradda WordPress iyo Ultimate Multisite!

Qiimaha ugu yar ee Codeable waa $70–$120 USD hal saac. Tixgelinta (estimates) waxay isbeddeli doontaa iyadoo ku xiran heerka shaqada, adagga farsamada, iyo mudnaanta mashruuca.

Marka aad hesho tixgelintaas, waxaad awoodaysaa inaad u horumariso (hire) horumariyaha ugu habboon baahiyahaaga oo aad wada bilaabato mashruuca!
