# را성이שkaads-ka-network-ka-ad-sare-Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector waa **add-on bilaash ah** oo si aad u hesho qalabka lagu dhigo xayaysiiso (ads) plans-kaaga isagoo ka soo qaadaya bogga wax ka beddelka.

Markaad installay Ultimate Multisite: Ad Injector, tag bogga _Plugins_ ee aad ku samaysay si aad u fuliso (activate).

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Riix badhanka Network Activate_

Marka add-on-ka la fuliyo, _Advanced Options_ meta box-ka bogga wax ka beddelka plans-kaaga wuxuu leeyahay dooro cusub oo la yiraahdo _Ad Settings_.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_Hadda waxaad dib u samayn kartaa ad settings-ka qaaliga ah ee plans-kaaga._

Tab-kaas, waxaad awoodaysaa inaad ku dartid koodka xayaysiiska (ad codes) post-yada iyo pages-kaanka front-end-ka, hore iyo dambe ee macluumaadka.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Marka aad ku dheeshay koodka xayaysiiskaaga, wuxuu si cad u muuqdaa post-yada iyo pages-kaankaas ee plan-kaas gaarka ah._

Waxaad sidoo kale samayn kartaa in xayaysiis la darsado header-ka Admin Pages-ka oo ay isticmaala yihiin isticmaalayaasha plan-kaas.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Tag option-ka Ad Code ee Admin Panel haddii aad rabto inaad xayaysiis la muuqato back-end-ka isticmaalayaashaada._

### Ka Bakran Xayaysiiska (Beyond Advertising)

Waxaad sidoo kale isticmaali kartaa Ultimate Multisite: Ad Injector si aad u bixiso taageero plans-ka sare ee lacagta.

Si kaliya tag option-ka _Admin Panel Ads_ oo ku dartid koodka HTML-ka qalabkaaga taageerada (support tool).
