# De-activation of Membership (v2) {#canceling-a-membership-v2}

_**Fadlan Muhiim Ah: Qoraalkan wuxuu la xiriira Ultimate Multisite version 2.x.**_

In kastoo de-activationka socdaaga (customer side) ay fududahay, waa sidan aad u samayn karto.

Haddii isticmaaluhu doonayo inuu ka saaro aabbiisa (membership), wuxuu ugu horreeya raadinayaa bogga **Account**-kiis.

Markaas, wuxuu qaadanayaa **Danger Zone**-ka, ka dibna wuxuu riixayaa **Delete Account**.

![Account page Danger Zone with Delete Account button](/img/account-page/danger-zone.png)

Waxaa soo bandhigay module oo weydiinaya xaqiijinta. Wuxuu qabtaa (toggle) door kasta oo ah **Confirm Account Deletion** oo wuxuuna riixayaa **Delete Account**.

![Confirm Account Deletion dialog](/img/account-page/delete-account-confirm.png)

Tani waxay ka saari doontaa isticmaalaha website-kaaga, laguna xiraya aabbiisa ee shirkaddaada (network), sidoo kale laga saari doono boggiisa shirkaddaada.

![Logged out after account deletion](/img/account-page/logged-out-after-delete.png)

![Membership canceled confirmation](/img/account-page/membership-canceled.png)

**Muhiim:** Waxaad aragaysaa in tallaabadaas aysan ka saarin doonin macaamiisha shirkaddaada, sababtoo ah macmiilku wuxuu noqon karaa leh aabbiro badan ama la'aadin aabbi koi.

![Customer still exists in the network after cancellation](/img/admin/customer-after-cancel.png)

![Customer with no active memberships](/img/admin/memberships-customer-empty.png)
