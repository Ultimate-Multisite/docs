# Website-yada ee Macaamiishaaga (v2) {#your-clients-websites-v2}

Shirkaddaadu ku kaydsan marka aad ku gasho shabakadaha, macaamiishaadu waxay heli karaan dashboard-kooda.

Marka kale oo ay helaan bogga **Account** (Xogta), halkaas oo ay arki karaan macluumaadka muhiimka ah ee ku saabsan lacag bixinta, xubnaha, domain-yada, xaddidda qorshaha iyo waxyaabo kale, waxay sidoo kale heli karaan bogga **My Sites** (Website-yadeeyda), halkaas oo ay arki karaan oo maamuli karaan dhammaan website-yadeedooda. Waxay kaliya inay ku riixaan My Sites-ka dashboard-kooda.

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

Waxaa lagu muujin doonaa dhammaan website-yadeedooda, iyadoo la haysto badhan **Manage** (Maamul) oo si toos ah hoose qabow kasta. Haddii ay ku riixaan, waxay u socdaan **dashboard-ka website-gaas gaarka ah**.

Waxay sidoo kale arki karaan website-keedii ugu muhiimsan ee qorshaha haddii uu kuu oggolaado inaad leedahay in aad leedahay website badan.

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

Halkan waa aragti buuxda oo ku saabsan interface-ka wax ka beddelka website-ka:

![Site edit full interface](/img/admin/site-edit-full.png)

Haddii ay doonayaan inay beddelaan website-keedii ugu muhiimsan, waxay u baahan yihiin inay aadaan bogga Account iyagoo riixaya menu-ga **Account**.

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

Waxay sidoo kale arki karaan oo wax ka beddeli karaan faahfaahinta macaamiisha:

![Customer edit interface](/img/admin/customer-edit.png)

Markaas, waa inay riixaan **Change Default Site** (Beddel Website-ka Hoose).

![Change Default Site button on account page](/img/account-page/change-default-site.png)

Waxaa soo muuqan doonaa daaqad pop-up ah. Ku riix liiska drop-down-ka oo dooro website-ka aad u rabto inaad samayso midka ugu muhiimsan.

![Change Default Site dropdown selection](/img/account-page/change-default-site-dropdown.png)

Markaas, kaliya ku riix **Change Default Site** (Beddel Website-ka Hoose) oo aad dhammaystirtay!

![Xaqiijinta isbeddelka goobta caadiga ah](/img/account-page/change-default-site-confirm.png)
