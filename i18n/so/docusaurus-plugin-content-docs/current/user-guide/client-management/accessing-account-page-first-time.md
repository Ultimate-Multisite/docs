# Sida sida aad u heli karto macluumaadkaaga ugu horreeya?

Soo dhowaanshaha Ultimate Multisite!

Sida loo galo boggaaga macluumaadka (account page), isticmaal xiriirka aad ka heshay email-kaaga ka dib markii aad iibsat plugin-keena:

<!-- Screenshot unavailable: Email-ka sida uu u muuqdo, isagoo ku jira xiriir lagu galo bogga account -->

Isticmaal email-ka aad bixisay inta aad iibsat waxaana haysataa oo markii aad heshay erayada sirta (password) ee email-kaaga.

<!-- Screenshot unavailable: Bogga galista (Login page) isagoo leh meel lagu qoro email-ka iyo erayada sirta -->

Maadaama tani tahay markii ugu horreysay ee aad ku galay macluumaadkaaga, waxa ay weydiin doonaan inaad beddesho erayada sirta. Beddelo oo riix badhka _Save_ (Wax-soo-saar).

<!-- Screenshot unavailable: Qaabka beddelista erayada sirta isagoo leh badhka Save -->

Boggaaga macluumaadka, waxaad maamuli kartaa dhammaan faahfaahinta macluumaadkaaga sida la-bixinta (billings), dib u soo celinta (renewals), ka hortagga (cancellations), iyo waxyaabo badan oo kale.
