# Filter: wu_upgrade_to_unlock_url

## پیرامیٹرز {#parameters}

| نام | قسم | تفصیل |
|------|------|-------------|
| $url | `string` | اپ گریڈ کا URL۔ |
| $args | `array` | ماڈیول اور حد کی قسم جس کو اپ گریڈ کرنے کی ضرورت ہے۔ |
