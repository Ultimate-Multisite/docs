# Filter: wu_plan_product_types {#filter-wuplanproducttypes}

ان پروڈکٹ ٹائپس کو فلٹر کریں جنہیں پلانز کے طور پر سمجھا جانا چاہیے۔

یہ فلٹر ایڈ آنز کو اضافی پروڈکٹ ٹائپس رجسٹر کرنے کی اجازت دیتا ہے جنہیں تصدیق (validation) اور الگ تھلگ کرنے (segregation) کے عمل میں پلانز کے طور پر پہچانا جانا چاہیے۔
