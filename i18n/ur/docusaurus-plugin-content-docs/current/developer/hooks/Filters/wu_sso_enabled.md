# Filter: wu_sso_enabled

کراس-ڈومین سنگل-سائن-آن (single-sign-on) کی سہولت کو فعال یا غیر فعال کرنا۔

اس ویلیو کو فلٹر کر کے سنگل-سائن-آن کو مکمل طور پر بند کیا جا سکتا ہے، یا اسے کسی خاص شرط کے تحت فعال کیا جا سکتا ہے۔

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $enabled | `bool` | کیا SSO کو فعال کرنا ہے؟ آن کے لیے `True`، اور آف کے لیے `false`۔ |
