# Filter: wu_independent_billing_cycle_product_types

ان پروڈکٹ ٹائپس کو فلٹر کرتا ہے جن کے بلنگ سائیکل آزاد (independent) ہوتے ہیں۔

جن پروڈکٹس کے بلنگ سائیکل آزاد ہوتے ہیں، انہیں کارٹ میں پلان کی بلنگ مدت سے ملنے پر مجبور نہیں کیا جاتا۔ یہ ان پروڈکٹس کے لیے مفید ہے جیسے کہ ڈومین رجسٹریشن، جو چاہے پلان ماہانہ ہو یا سالانہ، ہمیشہ سالانہ بلنگ کرتی ہیں۔

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | ان پروڈکٹ ٹائپس کی ایک اریے (array) جن کے بلنگ سائیکل آزاد ہوتے ہیں۔ |
