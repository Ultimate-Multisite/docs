# Filter: wu_add_new_domain_modal_fields

یہ فلٹر نیا ڈومین شامل کرنے والے موڈل (modal) کے فیلڈز کو فلٹر کرتا ہے۔

یہ ایڈ آنز (addons) کو (جیسے Domain Seller) ڈومین رجسٹریشن کے فیلڈز کو تبدیل کرنے یا ان کی جگہ نئے فیلڈز شامل کرنے کی اجازت دیتا ہے۔

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $fields | `array` | فارم کے فیلڈز۔ |
