مجھے یقین نہیں کہ یہ کس طرح مفید ہو سکتا ہے، لیکن اسے رکھنا نقصان دہ نہیں ہے۔

## پیرامیٹرز

| Name | Type | Description |
|------|------|-------------|
| $link | `string` | رجسٹر کیا گیا لنک |

| $default_link | `string` | رجسٹر کیا گیا ڈیفالٹ لنک |
