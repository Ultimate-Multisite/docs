# Filter: wu_view_override_replaceable_views

یہ ڈویلپرز کو replaceable list میں مزید فولڈرز شامل کرنے کی اجازت دیتا ہے۔

احتیاط برتیں، کیونکہ مزید فولڈرز کی اجازت دینے سے Ultimate Multisite کے ورژن کے بجائے پرانی (out-of-date) کاپیاں لوڈ ہو سکتی ہیں۔

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $replaceable_views | `array` | اجازت دیے گئے فولڈرز کی فہرست۔ |

### Since

- 2.0.0
### Source

[`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L50) میں لائن 50 پر ڈیفائن کیا گیا ہے۔
