# Filter: wp_ultimo_render_vars

پلاگ ان ڈویلپرز کو رینڈر کنٹیکسٹ میں عالمی سطح پر اضافی ویری ایبل شامل کرنے کی اجازت دیتا ہے۔

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $args | `array` | وہ ویری ایبلز جو رینڈر کال کے ذریعے پاس کیے گئے ہیں، ان کا ایک اریے (Array)۔ |
| $view | `string` | اس ویو کا نام جسے رینڈر کیا جانا ہے۔ |
| $default_view | `string` | ڈیفالٹ ویو (fallback_view) کا نام۔ |
