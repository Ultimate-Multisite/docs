# Filter: wu_password_require_uppercase {#filter-wupasswordrequireuppercase}

یہ فلٹر طے کرتا ہے کہ پاسورڈ میں بڑے حروف (uppercase letters) درکار ہیں یا نہیں۔

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | یہ طے کرتا ہے کہ بڑے حروف درکار ہیں یا نہیں۔ جب اصول لاگو کیے جاتے ہیں تو ڈیفالٹ طور پر یہ درست (true) ہوتا ہے۔ |
| $defender_active | `bool` | یہ طے کرتا ہے کہ کیا Defender Pro Strong Password فعال (active) ہے۔ |
