# Action: wu_before_light_ajax {#action-wubeforelightajax}

کچھ صورتوں میں، ہمیں ایکشنز کو ہینڈل کرنے کے لیے اضافی فنکشنلٹی (extra juice) لوڈ کرنے کی ضرورت پڑ سکتی ہے۔

اس ایکشن کا استعمال ضرورت پڑنے پر اضافی dependencies لوڈ کرنے کے لیے کیا جا سکتا ہے۔
