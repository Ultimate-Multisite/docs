# Action: wu_page_\{$this->id\}_register_widgets

یہ ایکشن اس وقت فعال ہوتا ہے جب اس پیج کے لیے ویجیٹس رجسٹر ہو جاتے ہیں۔

ہوک کے نام کا متحرک حصہ، `$this->id`, پیج کی آئی ڈی (page id) کو ظاہر کرتا ہے۔

## پیرامیٹرز (Parameters)

| Name | Type | Description |
|------|------|-------------|
| $id | `string` | پیج کی آئی ڈی۔ |
| $page_hook | `string` | پیج کا ہک۔ |
| $page | `object` | پیج آبجیکٹ۔ |
