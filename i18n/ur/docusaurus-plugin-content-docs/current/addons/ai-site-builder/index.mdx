# AI Site Builder {#ai-site-builder}

ultimate-multisite-ai-site-builder

## Requirements {#requirements}

- WordPress 5.3 یا اس سے زیادہ
- PHP 7.4 یا اس سے زیادہ
- Multisite Ultimate plugin (active)

## Installation {#installation}

1. addon files کو اپنے `/wp-content/plugins/` directory میں upload کریں۔
2. WordPress میں 'Plugins' menu کے ذریعے plugin کو activate کریں۔
3. network admin میں addon کی settings configure کریں۔

## Development {#development}

### Setup {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Testing {#testing}

اس addon میں ایک مکمل testing framework شامل ہے:

- **Unit Tests**: WordPress test suite کے ساتھ PHPUnit پر مبنی ٹیسٹ
- **Base Test Class**: عام test scenarios کے لیے helper methods فراہم کرتا ہے
- **Test Coverage**: code coverage کو track کرنے کے لیے configure کیا گیا ہے
- **Multisite Testing**: ٹیسٹ multisite environment میں چلتے ہیں

### Code Standards {#code-standards}

- **WordPress Coding Standards**: PHPCS کے ذریعے نافذ کیے گئے
- **PHP 7.4+ Compatibility**: جدید PHP features کو سپورٹ کرتا ہے
- **Static Analysis**: type checking کے لیے PHPStan کا انضمام (integration)
- **Code Modernization**: PHP upgrades کے لیے Rector rules

### Build System {#build-system}

- **Asset Processing**: CSS/JS files کے لیے Minification
- **Translation**: POT file generation
- **Packaging**: distribution کے لیے archive creation
- **Development Mode**: debugging کے لیے Unminified assets

## Configuration {#configuration}

یہ addon network admin interface کے ذریعے قابل رسائی مختلف configuration options کو سپورٹ کرتا ہے۔

## Hooks and Filters {#hooks-and-filters}

### Actions {#actions}

- `ultimate-multisite-ai-site-builder_init` - addon initialization کے بعد trigger ہوتا ہے
- `ultimate-multisite-ai-site-builder_loaded` - تمام addon files load ہونے کے بعد trigger ہوتا ہے

### Filters {#filters}

- `ultimate-multisite-ai-site-builder_settings` - addon settings کو modify کرتا ہے
- `ultimate-multisite-ai-site-builder_enabled` - addon کے enable/disable status کو override کرتا ہے
