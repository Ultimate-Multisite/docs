# Support Agents

پورے نیٹ ورک کو خود سنبھالنا کوئی آسان کام نہیں ہے۔ جیسے جیسے آپ کا نیٹ ورک بڑھنا شروع ہوتا ہے، آپ کو سپورٹ کی بڑھتی ہوئی مانگ نظر آنے لگے گی۔ یہ ایک اچھا خیال ہوگا کہ ایسے لوگوں کو شامل کیا جائے جو آپ کے کام کا بوجھ کم کر سکیں۔ تاہم، مکمل رسائی یا اجازت دینا خطرناک ہو سکتا ہے اور آپ کے نیٹ ورک کو نقصان پہنچا سکتا ہے۔ یہیں پر یہ Support Agent add-on کام آتا ہے۔

یہ ایڈ-آن آپ کو Support Agent نام کا ایک نیا قسم کا WordPress صارف (user) بنانے کی اجازت دیتا ہے، اور آپ کو اس صارف کی اجازتوں (permissions) پر مکمل کنٹرول دیتا ہے۔ اجازت دینا اتنا آسان ہے جتنا کہ ایک ٹوگل بٹن آن کرنا، اور اگر آپ اسے ہٹانا چاہیں تو اسے بند کرنا بھی اتنا ہی آسان ہے۔

## Features {#features}

- WordPress اور Ultimate Multisite کی اجازتوں (permissions) کی تفصیل وار فہرست
- آسانی سے رسائی کو فعال اور غیر فعال کرنا
- غیر محدود تعداد میں Agent اکاؤنٹس بنانا

## Installation {#installation}

1. 'ultimate-multisite-support-agents' کو '/wp-content/plugins/' ڈائریکٹری میں اپ لوڈ کریں۔
2. WordPress میں 'Plugins' مینیو کے ذریعے پلگ ان کو ایکٹیویٹ کریں۔
3. پلگ ان کو سیٹ اپ کرنے کے لیے مرحلہ وار ویزرڈ (Wizard) کی پیروی کریں۔
