# زبان کا سلیکٹر (Language Selector) {#language-selector}

Multisite Ultimate: سائن اپ پر زبان کا سلیکٹر

سائن اپ کے عمل (sign-up flow) میں ایک زبان کا سلیکٹر فیلڈ شامل کریں۔

## انسٹالیشن (Installation) {#installation}

1. 'ultimate-multisite-language-selector' کو '/wp-content/plugins/' ڈائریکٹری میں اپ لوڈ کریں۔
2. اپنے WordPress Network Admin میں 'Plugins' مینیو کے ذریعے پلگ ان کو ایکٹیویٹ کریں۔

## استعمال کی رہنمائی (Usage Guide) {#usage-guide}

اس مفت ایڈ-آن کے ساتھ، آپ اپنے صارفین کو یہ منتخب کرنے دے سکتے ہیں کہ وہ سائن اپ کے عمل (sign-up flow) اور لاگ ان پیج پر کون سی زبان دیکھنا چاہتے ہیں۔

[اپنے لائسنس کو ایکٹیویٹ کرنے] کے بعد، آپ وہ Ultimate Multisite ایڈ-آنز انسٹال کر سکیں گے جن تک آپ کا لائسنس رسائی دیتا ہے۔

اپنے Ultimate Multisite سیٹنگز پیج پر جائیں اور پھر، بائیں طرف موجود مینیو میں، _Add-ons_ لیبل پر کلک کریں۔

![Add-ons page](/img/admin/addons-page.png)

_Language Selector انسٹال کرنا شروع کرنے کے لیے Add-ons پیج پر جائیں_

وہاں، آپ کو وہ تمام ایڈ-آنز ملیں گے جو ہم پیش کرتے ہیں۔

مفت ایڈ-آنز ہمارے تمام صارفین کے لیے دستیاب ہیں، چاہے انہوں نے کس سبسکرپشن پلان پر سائن اپ کیا ہو۔

ایڈ-آن کے تھمب نیل کے نیچے دیے گئے _Install it Now!_ بٹن پر کلک کریں۔ پھر اسے Plugins پیج پر ایکٹیویٹ کریں۔

![Add-ons page with install button](/img/admin/addons-page.png)

_ایڈ-آن انسٹال کرنے کے لیے بٹن پر کلک کریں_

_یہ اپنے Plugins پیج پر ایکٹیویٹ کریں_

اب، Ultimate Multisite کے سیٹنگز مینیو پر جائیں اور _Network Settings_ ٹیب میں داخل ہوں۔ وہاں، پیج کے نچلے حصے میں، آپ کو _Languages_ نام کا ایک نیا سیکشن ملے گا۔

![Settings page](/img/config/settings-general.png)

_ایک بالکل نیا سیکشن ظاہر ہو گیا!_

آپ اسے خالی چھوڑ کر تمام زبانوں کو آپشنز کے طور پر دکھا سکتے ہیں یا وہ منتخب کر سکتے ہیں جو آپ دستیاب کرنا چاہتے ہیں۔ یہ زبانیں وہ ہیں جو آپ نے اپنے WordPress سیٹ اپ میں انسٹال کی ہیں۔

## URL کے ذریعے زبان سیٹ کرنا {#setting-the-language-via-the-url}

آپ URL میں لوکل (locale) پیرامیٹر کو بائی پاس کرتے ہوئے بھی کسی پیج کی زبان تبدیل کر سکتے ہیں۔ مثال کے طور پر، اگر آپ意大利 (Italian) میں رجسٹر پیج پر ری ڈائریکٹ کرنا چاہتے ہیں، تو URL میں ?locale=it_IT شامل کریں۔

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

اس کے کام کرنے کے لیے زبان آپ کے WordPress نیٹ ورک پر انسٹال ہونی چاہیے۔
