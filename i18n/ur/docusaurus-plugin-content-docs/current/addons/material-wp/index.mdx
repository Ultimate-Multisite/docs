# Material WP

WordPress کا ایڈمن انٹرفیس بہت عرصے سے ایک جیسا ہے۔ لیکن کیا آپ نے کبھی سوچا ہے کہ اگر اس کا Material Design ورژن کیسا لگے گا؟

Material WP ایک ایڈمن تھیم ہے جو آپ کو وہ لُک اور فیل دینے کے لیے تیار کیا گیا ہے، جس سے آپ اپنے کلائنٹس کو زیادہ کسٹمائزڈ Dashboard فراہم کر سکتے ہیں۔

## خصوصیات {#features}

- کسٹم CSS سپورٹ
- آسان اسٹائلنگ سیٹنگز
- مینو کو دوبارہ ترتیب دینا (Menu reordering)
- آسان امپورٹ/ایکسپورٹ

## دستاویزات {#documentation}

**Material WP کے ساتھ اپنے ایڈمن پینل کو کسٹمائز کرنا**

## تنصیب {#installation}

1. 'material-wp' کو '/wp-content/plugins/' ڈائریکٹری میں اپ لوڈ کریں۔
2. WordPress میں 'Plugins' مینو کے ذریعے پلگ ان کو فعال (Activate) کریں۔

## استعمال کی گائیڈ {#usage-guide}

WordPress کا ایڈمن انٹرفیس بہت عرصے سے ایک جیسا ہے۔ لیکن کیا آپ نے کبھی سوچا ہے کہ اگر اس کا Material Design ورژن کیسا لگے گا؟

Material WP ایک ایڈمن تھیم ہے جو آپ کو وہ لُک اور فیل دینے کے لیے تیار کیا گیا ہے، جس سے آپ اپنے کلائنٹس کو زیادہ کسٹمائزڈ Dashboard فراہم کر سکتے ہیں۔ Material WP، Ultimate Multisite کے پریمیئم پلگ انز کور کا حصہ بھی ہے۔

**اسے انسٹال کرنے** کے بعد، اپنی _Plugins_ پیج پر جائیں اور اسے اپنے Network پر فعال (activate) کرنے کے لیے کلک کریں۔
