# ملٹی-نیٹ ورک تبدیلیوں کی فہرست

### 1.3.0 {#130}
* نیا: نیٹ ورک ٹیمپلیٹ پریویوَر — خریدنے سے پہلے لائیو پینل کے ساتھ نیٹ ورک ٹیمپلیٹس کو براؤز اور پریویو کریں۔
* نیا: چیک آؤٹ پر نیٹ ورک ٹیمپلیٹ پیکر — صارفین سائن اپ چیک آؤٹ کے دوران نیٹ ورک ٹیمپلیٹ کا انتخاب کر سکتے ہیں۔
* نیا: کراس-نیٹ ورک میجک لنک SSO — صارفین میجک لنک آتھنٹیکیشن کے ذریعے سب-نیٹورکس تک آسانی سے رسائی حاصل کر سکتے ہیں۔
* نیا: نیٹ ورک میڈیا کلوننگ — ٹیمپلیٹ سے نیٹورکس کو کلون کرتے وقت سائٹ کا میڈیا صحیح طریقے سے ڈپلیکیٹ ہو جاتا ہے۔
* درست کیا گیا: اب نیٹ ورک کلوننگ ملٹی-ٹینینسی ایڈ آن کے بغیر بھی کام کرتی ہے؛ کلون کے بعد سپر ایڈمن کی سہولیات دوبارہ دے دی جاتی ہیں۔
* درست کیا گیا: نیٹ ورک کی تخلیق اب WooCommerce اور تمام ادائیگی کے گیٹ وے کے ساتھ مطابقت رکھتی ہے۔
* درست کیا گیا: فیل بیک ڈومین جنریشن میں اب پورٹ نمبر شامل ہے۔
* درست کیا گیا: تمام تھیمز پر صحیح رینڈرنگ کے لیے ٹیمپلیٹ پیکر CSS کو ایک انکیو (enqueued) کردہ اسٹائل شیٹ میں منتقل کیا گیا ہے۔
* درست کیا گیا: سائٹ کوئری کے دائرہ کار میں undefined `network_id` کی کلید کے خلاف گارڈ لگایا گیا۔
* درست کیا گیا: کنٹیکسٹ سوئچنگ سے پہلے نیٹ ورک کی اجازت کی جانچ `wpdb` ٹیبل پراپرٹی کی غلطیوں کو روکتی ہے۔
