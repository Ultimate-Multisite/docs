# Action: switch_network

یہ عمل (Action) اس وقت چلتا ہے جب موجودہ نیٹ ورک کا سیاق و سباق (context) تبدیل کیا جاتا ہے۔

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $new_network_id | `int` | اس نیٹ ورک کی آئی ڈی جس پر تبدیل کیا جا رہا ہے۔ |
| $old_network_id | `int` | پہلے موجودہ نیٹ ورک کی آئی ڈی۔ |
