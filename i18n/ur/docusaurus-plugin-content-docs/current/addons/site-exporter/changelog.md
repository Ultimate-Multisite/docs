# سائٹ ایکسپورٹر کا Changelog

Version: 1.0.6 - ریلیز ہوا 2026-05-11
* نیا: اب سائٹ ایکسپورٹس میں ایک خودکار آغاز کرنے والا index.php شامل ہوتا ہے تاکہ ZIP کو الگ سے plugin انسٹال کیے بغیر نئے ہوسٹ پر انسٹال کیا جا سکے۔
* نیا: نیٹ ورک ایکسپورٹ ایڈمنسٹریٹرز کو سائٹ ایکسپورٹ ایڈمن پیج سے تمام سب سائٹس کو ایک ہی آرکائیو میں ایکسپورٹ کرنے کی سہولت دیتا ہے۔

Version: 1.0.5 - ریلیز ہوا 2026-05-05
* درست کیا: ٹیسٹ بوٹسٹریپ کی ناکامیوں کو روکنے کے لیے get_config جیسے غیر موک شدہ طریقوں کو سنبھالنے کے لیے WP_CLI polyfill stub میں __callStatic شامل کیا گیا۔

Version: 1.0.4 - ریلیز ہوا 2025-11-25

* درست کیا: Ultimate Multisite 2.4 کے ساتھ مطابقت (compatibility) کو درست کیا ہے۔

* نام کی تبدیلی (prefix) کو ultimate-multisite میں تبدیل کیا؛ ٹیکسٹ ڈومین کو اپ ڈیٹ کیا؛ ورژن بڑھایا۔
