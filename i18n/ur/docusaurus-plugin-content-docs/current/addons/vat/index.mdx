# VAT

Multisite Ultimate: European VAT

یہ Multisite Ultimate کی ڈیفالٹ ٹیکس ہینڈلنگ میں اضافی فیچرز اور ٹولز شامل کرتا ہے تاکہ یورپی VAT کو بہتر طریقے سے سپورٹ کیا جا سکے۔ یہ خود بخود Tax Rates پیج پر EU VAT ریٹس شامل کر دیتا ہے۔

## Installation

1. 'ultimate-multisite-vat' کو '/wp-content/plugins/' ڈائریکٹری میں اپ لوڈ کریں۔
2. WordPress میں 'Plugins' مینیو کے ذریعے پلگ ان کو ایکٹیویٹ کریں۔
3. Ultimate Multisite > Settings > Taxes پر جائیں اور EU Vat کو Enable کریں۔
4. Tax Rates پر کلک کریں، نیچے دیے گئے EU VAT Rates کو اپ ڈیٹ کریں، اور Tax Rates کو Save کریں۔
5. ہو گیا۔!
