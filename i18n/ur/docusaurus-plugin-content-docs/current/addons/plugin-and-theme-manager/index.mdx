# Plugin & Theme Manager

Multisite Ultimate: Plugin اور Theme Manager

انسٹال شدہ plugins اور themes کی meta information (title, author, thumbnail, اور description) کو ایڈٹ یا چھپا دیں۔ plugins اور themes کے لیے categories بنائیں تاکہ آپ کے صارفین انہیں فلٹر کر سکیں اور اپنے صارفین کے panel میں ایک خوبصورت کسٹم Plugins page شامل کر سکیں۔

اس plugin کو استعمال کرنے کے لیے، آپ بس اپنے network admin پر Plugins یا Themes کے پیجز پر جائیں اور "Edit Plugin/Theme info" پر کلک کریں۔

## انسٹالیشن

1. 'ultimate-multisite-plugin-and-theme-manager' کو '/wp-content/plugins/' directory میں اپ لوڈ کریں۔
2. اپنے WordPress Network Admin میں 'Plugins' مینیو کے ذریعے plugin کو ایکٹیویٹ کریں۔

## استعمال کی رہنمائی

Ultimate Multisite: Plugin اور Theme Manager ایک طاقتور **premium** Ultimate Multisite add-on ہے جو آپ کو اس طرح مکمل کنٹرول دیتا ہے کہ آپ کے network کے صارفین آپ کے network پر plugin اور theme کے آپشنز کو کیسے دیکھتے ہیں اور ان کے ساتھ تعامل کرتے ہیں۔

یہ add-on تین مقاصد پورے کرتا ہے:

  1. یہ آپ کو plugins اور themes دونوں کی metadata کو مکمل طور پر تبدیل کرنے دیتا ہے۔ اس میں نام، تفصیل (description)، مصنف کا نام (author name)، اور یہاں تک کہ تھمبنیلز بھی شامل ہیں!

  2. یہ آپ کو plugins اور themes کو categories میں تقسیم کرنے دیتا ہے، تاکہ آپ کے صارفین انہیں آسانی سے ترتیب دے سکیں۔

  3. یہ آپ کو پرانے، بورنگ plugins page کو ایک نئے themes جیسے page سے بدلنے کا آپشن دیتا ہے، جس میں categories کے ذریعے ترتیب دینے کی سہولت بھی شامل ہے۔

اسے ڈاؤن لوڈ اور انسٹال کرنے کے بعد، Ultimate Multisite Settings page پر ایک نیا ٹیب - _Add-on Settings_ - نظر آئے گا۔
