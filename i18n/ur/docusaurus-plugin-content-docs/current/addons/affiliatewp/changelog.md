# AffiliateWP کی تبدیلیاں (Changelog)

ورژن: 2.0.1 - ریلیز ہوا 2025-09-28

* پری فکس کو ultimate-multisite میں تبدیل کیا؛ ٹیکسٹ ڈومین کو اپ ڈیٹ کیا؛ اور ورژن میں اضافہ کیا ہے۔

* ultimo پیمنٹس پر مبنی کام کرنے کے لیے پلگ ان کی دوبارہ ساخت (rebuild) کی گئی۔

ورژن 2.0.0-beta.2 - ریلیز ہوا 2022-01-21

* اندرونی: ہکس اور فلٹرز جنریٹر (hooks and filters generator) شامل کیے گئے؛
* اندرونی: ڈویلپر کی سہولت کے لیے Multisite Ultimate سٹब्स شامل کیے گئے؛
* ٹھیک کیا: جب کسی کارٹ پر کوئی ڈسکاؤنٹ کوڈ موجود نہ ہو تو فیتل ایرر (Fatal error) آ جاتا تھا۔
* ٹھیک کیا: وزٹ (Visits) صحیح طریقے سے کنورٹ نہیں ہو رہے تھے۔
