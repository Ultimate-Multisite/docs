# AffiliateWP

Multisite Ultimate: AffiliateWP کا انٹیگریشن (Integration)

اپنے Ultimo Network کے کلائنٹ بیس کو بڑھانے کے لیے طاقتور AffiliateWP کا استعمال کریں!

## Installation

1. 'ultimate-multisite-git ' کو '/wp-content/plugins/' ڈائریکٹری میں اپ لوڈ کریں۔
2. WordPress میں 'Plugins' مینیو کے ذریعے پلگ ان کو ایکٹیویٹ کریں۔
3. پلگ ان کو سیٹ کرنے کے لیے مرحلہ وار ویزرڈ (Wizard) کی پیروی کریں۔

## Usage Guide

[AffiliateWP](https://affiliatewp.com/) آپ کے آن لائن کاروبار کے لیے افیلیئیٹ پروگرام بنانے کا سب سے بہترین حل بن چکا ہے۔ Ultimate Multisite: AffiliateWP وہ پل (bridge) ہے جو آپ کو اپنے پریمیئم نیٹ ورک کے لیے بھی ایسا ہی کچھ بنانے کے قابل بنائے گا۔

## Getting Started

سب سے پہلے، آپ کے پاس [**AffiliateWP**](https://affiliatewp.com/) **ایکٹیویٹ** ہونا **ضروری** ہے! اگر آپ کے پاس **وہ پلگ ان نہیں ہے جو اس کے ساتھ انٹیگریٹ ہوتا ہے**، تو آپ ہمارا ایڈ آن استعمال نہیں کر پائیں گے۔ اگر آپ اپنے کلائنٹس کو بھی اپنی سائٹس پر اسے استعمال کرنے کی اجازت نہیں دینا چاہتے، تو AffiliateWP صرف آپ کی مرکزی سائٹ پر ایکٹیویٹ ہونا چاہیے۔

مزید برآں، آپ کو **Ultimate Multisite** اور **Ultimate Multisite: AffiliateWP** دونوں کو پورے نیٹ ورک میں **ایکٹیویٹ** کرنے کی ضرورت ہوگی۔

## Adding Ultimate Multisite as an AffiliateWP Integration

اپنی مرکزی سائٹ پر، Affiliates Settings پر جائیں، اور Integrations ٹیب میں، Ultimate Multisite کو چیک کریں۔ یہ [AffiliateWP](https://affiliatewp.com/) کو بتائے گا کہ وہ Ultimate Multisite کے سائن اپ فلو سے آنے والی وزٹس اور کنورژنز کو ٹریک کرنا شروع کر دے۔

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_نیچے Ultimate Multisite باکس کو چیک کریں اور سیو کریں_

بس اتنا ہی کرنا ہے۔ ریفرل لنکس سے آنے والی وزٹس کو ٹریک کیا جائے گا، اور جب وہ صارف بعد میں اپنا پہلا پیمنٹ کرے گا، تو کنورژن بن جائے گا اور اس کنورژن کے ذمہ دار افیلیئیٹ کے لیے کمیشن رکھ دیا جائے گا۔

تاہم، صرف نئے بنائے گئے صارف سے موصول ہونے والا پہلا پیمنٹ ٹریک کیا جائے گا۔ بار بار ہونے والے ریفرل پیمنٹس کو فعال کرنے کے لیے، آپ کو [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) کی ضرورت ہوگی۔

## Enabling Recurring Referrals

[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) کو انسٹال کرنے کے بعد، Affiliates Settings کے Recurring Referrals ٹیب پر جائیں، اور بار بار ہونے والے ریفرلز کے استعمال کو فعال کریں۔
