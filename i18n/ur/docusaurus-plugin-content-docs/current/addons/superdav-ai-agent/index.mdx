# Superdav AI Agent

Superdav AI Agent ایک جدید WordPress AI معاون ہے جو Gratis AI Agent کو تھیم سے باخبر مہارتوں، بہتر انٹرنیٹ تلاش کی صلاحیتوں، سائٹ بلڈر خصوصیات، اور اختیاری خودکاری انضمامات کے ساتھ وسعت دیتا ہے۔ ورژن 1.18.0 میں Google Calendar پڑھنے کے اوزار، رابطہ اور شرکا کی نقشہ بندی، انسانی منظوری کے دروازے، یاددہانی کی تکرار ختم کرنا، TextBee SMS اطلاعات، منظم Superdav سروس کی فراہمی، اور قابلِ اعتماد ڈویلپر ورک فلو کے لیے الگ سے تقسیم کیا گیا Advanced ساتھی پیکیج شامل کیا گیا ہے۔

## اہم خصوصیات

- **تھیم سے باخبر مہارتیں** — آپ کے فعال WordPress تھیم اور نصب شدہ پلگ اِنز کی بنیاد پر رہنمائی اور صلاحیتوں کو خودکار طور پر ڈھالتی ہیں
- **متعدد انٹرنیٹ تلاش فراہم کنندگان** — زیادہ بھرپور تلاش کے نتائج اور جدید تحقیق کی صلاحیتوں کے لیے Brave Search اور Tavily میں سے انتخاب کریں
- **سائٹ بلڈر رابطہ فارم** — اپنے نصب شدہ فارم پلگ اِن کا استعمال کرتے ہوئے چیٹ انٹرفیس سے براہِ راست مکمل طور پر فعال رابطہ فارم بنائیں
- **بہتر سائٹ بلڈر** — بہتر پلگ اِن دریافت اور خرابی سے بحالی کے ساتھ بہتر کثیر مرحلہ سائٹ بنانے کے منصوبے
- **بلٹ اِن تھیم مہارتیں** — Block Themes، Classic Themes، Kadence Blocks، اور Kadence Theme کے لیے خصوصی رہنمائی
- **بہتر WooCommerce انضمام** — بہتر قابلِ اعتمادیت اور مطابقت کے لیے مقامی WooCommerce APIs استعمال کرنے کی خاطر ازسرِ نو بنایا گیا
- **خودکار فراہم کنندہ شناخت** — دستیاب پلگ اِنز اور تھیمز کو فعال یا غیر فعال کیے جانے پر خودکار طور پر شناخت اور تازہ کرتا ہے
- **حسبِ ضرورت لوگو بنانا** — خودکار نام اسپیس محفوظ صفائی کے ساتھ حسبِ ضرورت SVG لوگو بنائیں اور شامل کریں
- **پیلیٹ تضاد کی توثیق** — اپنے تھیم پر لاگو کرنے سے پہلے WCAG رسائی پذیری کی پابندی کے لیے رنگوں کے جوڑوں کی جانچ کریں
- **تصویر سے بہتر دریافت** — ڈیزائن کی سمت پر اثر انداز ہونے کے لیے Theme Builder دریافت انٹرویو کے دوران تصاویر اپ لوڈ کریں
- **مہمان نوازی مینیو** — ریستورانوں، کیفے، اور مہمان نوازی کے کاروباروں کے لیے منظم کھانے اور مشروبات کے مینیو صفحات بنائیں
- **ڈیسک ٹاپ اور موبائل پیش منظر** — ڈیزائن کی سمت کے انتخاب کے دوران اپنے ڈیزائن کا ڈیسک ٹاپ اور موبائل آلات پر پیش منظر دیکھیں
- **درجہ 1 سائٹ اسکریپنگ** — sd-ai-agent/site-scrape اب حریف اور ترغیبی ویب سائٹس کے تجزیے کے لیے بطور ڈیفالٹ دستیاب ہے
- **Google Calendar پڑھنے کے اوزار** — ترتیب دیے گئے کیلنڈرز اور واقعات پڑھیں تاکہ خودکاریاں آنے والے شیڈول کے تناظر کے بارے میں استدلال کر سکیں
- **رابطہ اور شرکا کی نقشہ بندی** — فالو اَپ کام بنانے سے پہلے کیلنڈر شرکا کو معروف سائٹ صارفین اور روابط سے ملائیں
- **انسانی منظوری کے دروازے** — پیغامات بھیجنے، کام دوبارہ شروع کرنے، یا تبدیلیاں کرنے سے پہلے حساس خودکاریوں کو جائزے کے لیے روکیں
- **یاددہانی ریکارڈز اور تکرار ختم کرنا** — ان یاددہانیوں کا سراغ رکھیں جو پہلے ہی بھیجی جا چکی ہیں تاکہ دہرائی گئی خودکاری کارروائیاں اسی وصول کنندہ کو دوبارہ اطلاع نہ دیں
- **TextBee SMS اطلاعات** — جب کوئی SMS فراہم کنندہ فعال ہو تو TextBee کے ذریعے ترتیب دی گئی ٹیکسٹ پیغام اطلاعات بھیجیں
- **منظم Superdav سروس کی فراہمی** — دستی endpoint ترتیب کے بجائے خودکار سیٹ اپ کے ساتھ معاونت یافتہ سائٹس کو میزبان Superdav endpoints سے جوڑیں
- **Advanced ساتھی پیکیج** — فائل سسٹم، ڈیٹا بیس، WP-CLI، REST ڈسپیچر، پلگ اِن بلڈر، git اسنیپ شاٹ، صارف-انتظام، اور بینچ مارک ورک فلو کے لیے الگ سے تقسیم کیے گئے قابلِ اعتماد ڈویلپر اوزار

## تقاضے

- WordPress 6.0 یا اس سے زیادہ
- PHP 8.1 یا اس سے زیادہ
- WordPress Multisite (نیٹ ورک پر فعال) یا سنگل سائٹ تنصیب
- ایک فعال AI فراہم کنندہ کنیکٹر پلگ اِن (Anthropic، OpenAI، یا مطابقت رکھنے والا)
- Gratis AI Agent v1.9.0 یا اس سے زیادہ (Superdav AI Agent، Gratis AI Agent کو وسعت دیتا ہے)

## تنصیب

1. پلگ اِن فائلیں `/wp-content/plugins/superdav-ai-agent` پر اپ لوڈ کریں
2. WordPress میں **پلگ اِنز** اسکرین کے ذریعے پلگ اِن فعال کریں
3. یقینی بنائیں کہ Gratis AI Agent بھی فعال ہے
4. **Gratis AI Agent → Settings** پر جائیں اور ضرورت کے مطابق اپنا انٹرنیٹ تلاش فراہم کنندہ، منظم Superdav سروس کنکشن، Google Calendar، یا TextBee اسناد ترتیب دیں

WordPress.org پیکیج میں بنیادی Superdav AI Agent خصوصیات شامل ہیں۔ Advanced ساتھی پیکیج ان ماحولوں کے لیے الگ سے بنایا اور تقسیم کیا جاتا ہے جہاں قابلِ اعتماد ڈویلپر اوزار واضح طور پر نصب اور مجاز کیے جاتے ہیں۔

## فوری آغاز

فعال ہونے کے بعد، Superdav AI Agent آپ کی Gratis AI Agent تنصیب کو تھیم سے باخبر مہارتوں اور اضافی صلاحیتوں کے ساتھ خودکار طور پر بہتر بناتا ہے۔ WordPress منتظم سائیڈ بار میں **Gratis AI Agent** پینل کھولیں اور نئی خصوصیات استعمال کرنا شروع کریں۔

**مثالی ہدایات:**

```
رابطہ صفحے میں ایک رابطہ فارم شامل کریں۔
```

```
تازہ ترین WordPress سیکیورٹی بہترین طریقوں کے لیے ویب تلاش کریں۔
```

```
فعال تھیم کے بلاک پیٹرنز استعمال کرتے ہوئے ایک ہیرو سیکشن بنائیں۔
```

```
کل کے ترتیب دیے گئے Google Calendar واقعات کا جائزہ لیں اور شرکا کے لیے یاددہانیاں مسودہ کریں۔
```

```
میرے پیغام منظور کرنے کے بعد ایک SMS یاددہانی بھیجیں۔
```

## تھیم سے باخبر مہارتیں

Superdav AI Agent خودکار طور پر آپ کے فعال تھیم اور نصب شدہ پلگ اِنز کی شناخت کرتا ہے، اور معاون کے تناظر میں مناسب تھیم سے باخبر مہارت لوڈ کرتا ہے۔ دستیاب مہارتوں میں شامل ہیں:

| مہارت | لاگو ہوتا ہے |
|---|---|
| **Block Themes** | WordPress بلاک پر مبنی (Full Site Editing) ساخت استعمال کرنے والے تھیمز |
| **Classic Themes** | PHP ٹیمپلیٹس اور کلاسک ایڈیٹر استعمال کرنے والے روایتی WordPress تھیمز |
| **Kadence Blocks** | وہ سائٹس جن پر Kadence Blocks پلگ اِن فعال ہے |
| **Kadence Theme** | وہ سائٹس جن میں Kadence theme فعال تھیم کے طور پر موجود ہے |

ہر مہارت پر تفصیلی رہنمائی کے لیے [تھیم سے باخبر مہارتیں](../../user-guide/configuration/theme-aware-skills) دیکھیں۔

## انٹرنیٹ تلاش فراہم کنندگان

Superdav AI Agent v1.10.0 انٹرنیٹ تلاش کے سوالات کے لیے Brave Search کے متبادل کے طور پر **Tavily** شامل کرتا ہے۔ دونوں فراہم کنندگان حقیقی وقت کے تلاش نتائج پیش کرتے ہیں اور انہیں **Gratis AI Agent → Settings → Advanced** میں ترتیب دیا جا سکتا ہے۔

| فراہم کنندہ | خصوصیات | سیٹ اپ |
|---|---|---|
| **Brave Search** | تیز، رازداری پر مرکوز تلاش کے نتائج | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | زیادہ بھرپور تلاش کے نتائج، جدید تحقیقی صلاحیتیں | [Tavily API](https://tavily.com/) |

ترتیب کی ہدایات کے لیے [انٹرنیٹ تلاش](../../user-guide/configuration/internet-search) دیکھیں۔

## کیلنڈر، منظوری، اور اطلاع کے ورک فلو

Superdav AI Agent v1.18.0 خودکاری کے ورک فلو کو مواد اور سائٹ بنانے کے کاموں سے آگے بڑھاتا ہے:

- **Google Calendar ٹولز** ایجنٹ کو ترتیب دیے گئے کیلنڈرز اور ایونٹ کی تفصیلات پڑھنے دیتے ہیں جب کیلنڈر کی اسناد دستیاب ہوں۔
- **رابطہ اور شریکِ تقریب مددگار** یاددہانیوں یا فالو اپ کاموں کی تیاری سے پہلے ایونٹ کے شرکا کو WordPress صارفین یا معلوم رابطوں سے ملاتے ہیں۔
- **انسانی منظوری کے گیٹس** کسی خودکاری کو اس وقت تک روکنے دیتے ہیں جب تک ایک مجاز صارف مجوزہ کارروائی کا جائزہ نہ لے۔
- **یاددہانی ریکارڈز** اس وقت نقلی کیلنڈر یا اطلاع کی یاددہانیوں کو روکتے ہیں جب خودکاری دوبارہ آزمائی جائے یا بار بار شیڈول کی جائے۔
- **TextBee SMS سپورٹ** اس وقت ٹیکسٹ پیغام اطلاعات کو فعال کرتی ہے جب TextBee فراہم کنندہ ترتیب دیا گیا ہو۔

ان ٹولز کو شیڈول سے باخبر معاونت، شریکِ تقریب فالو اپ، اور یاددہانی ورک فلو کے لیے استعمال کریں جن میں حساس پیغامات بھیجنے سے پہلے اب بھی انسانی جائزہ درکار ہو۔

## سائٹ بلڈر رابطہ فارمز

سائٹ بلڈر ایجنٹ اب چیٹ انٹرفیس سے براہِ راست رابطہ فارمز بنا سکتا ہے۔ بس سائٹ بلڈر سے رابطہ فارم بنانے کو کہیں، اور یہ آپ کے نصب شدہ فارم plugin (Contact Form 7، WPForms، Fluent Forms، یا Gravity Forms) کا خودکار طور پر پتا لگائے گا اور استعمال کے لیے تیار فارم تیار کرے گا۔

تفصیلی رہنمائی کے لیے [سائٹ بلڈر رابطہ فارمز](../../user-guide/administration/site-builder-contact-forms) دیکھیں۔

## Theme Builder خصوصیات

Superdav AI Agent v1.16.0 میں Theme Builder طاقتور نئی صلاحیتیں شامل کرتا ہے:

### تصویر اپ لوڈ کے ساتھ دریافت انٹرویو

Theme Builder دریافت انٹرویو میں اب تصویر اپ لوڈ کا مرحلہ شامل ہے جہاں آپ بصری تحریک فراہم کر سکتے ہیں۔ اپنے برانڈ، مدمقابل سائٹس، یا ڈیزائن تحریک کی تصاویر اپ لوڈ کریں تاکہ Theme Builder آپ کی ڈیزائن سمت کو سمجھ سکے۔

تفصیلی رہنمائی کے لیے [دریافت انٹرویو](./theme-builder/discovery-interview.md) دیکھیں۔

### ڈیسک ٹاپ اور موبائل پیش نظاروں کے ساتھ ڈیزائن سمت

Theme Builder کے آپ کی مکمل theme تیار کرنے سے پہلے ڈیسک ٹاپ اور موبائل دونوں آلات پر اپنے ڈیزائن کا جائزہ لیں۔ متعدد ریسپانسیو بریک پوائنٹس پر حقیقی وقت کے پیش نظارے دیکھیں۔

تفصیلی رہنمائی کے لیے [ڈیزائن سمت](./theme-builder/design-direction.md) دیکھیں۔

### مہمان نوازی مینیو

Theme Builder اب ریستورانوں، کیفے، بارز، اور دیگر مہمان نوازی کاروباروں کے لیے منظم خوراک اور مشروبات کے مینیو صفحات تیار کر سکتا ہے۔ مینیو میں قیمتیں، وضاحتیں، غذائی معلومات، اور الرجی تنبیہات شامل ہوتی ہیں۔

تفصیلی رہنمائی کے لیے [مہمان نوازی مینیو](./theme-builder/hospitality-menus.md) دیکھیں۔

### نئی صلاحیتیں

- **[Logo SVG تیار کریں](./abilities/generate-logo-svg.md)** — خودکار صفائی کے ساتھ حسبِ ضرورت SVG لوگو بنائیں
- **[پیلیٹ کنٹراسٹ کی توثیق کریں](./abilities/validate-palette-contrast.md)** — WCAG تعمیل کے لیے رنگوں کے جوڑوں کی جانچ کریں
- **[مینیو بنائیں](./abilities/create-menu.md)** — جداگانہ نیویگیشن لیبلز کے ساتھ نیویگیشن مینیو بنائیں

دستیاب ٹولز اور Tier 1 سائٹ سکریپ ٹول کے بارے میں معلومات کے لیے [ٹولز اور ترتیب](./theme-builder/tools.md) دیکھیں۔

## ترتیب

تمام ترتیبات WordPress ایڈمن میں **Gratis AI Agent → Settings** کے تحت ہیں۔ Superdav AI Agent درج ذیل ترتیب کے اختیارات شامل کرتا ہے:

| ترتیب | وضاحت |
|---|---|
| **انٹرنیٹ تلاش فراہم کنندہ** | Brave Search اور Tavily میں سے انتخاب کریں (اختیاری) |
| **Tavily API Key** | Tavily انٹرنیٹ تلاش کے لیے API کلید (اگر Tavily منتخب ہو) |
| **Managed Superdav Service** | معاونت یافتہ سائٹس کو خودکار endpoint فراہمی کے ساتھ ہوسٹڈ Superdav سروس سے جوڑیں |
| **Google Calendar اسناد** | کیلنڈر رسائی ترتیب دیں تاکہ ایجنٹ کیلنڈرز اور ایونٹس پڑھ سکے |
| **TextBee SMS فراہم کنندہ** | منظور شدہ SMS اطلاع ورک فلو کے لیے TextBee اسناد ترتیب دیں |

## جدید ساتھی پیکیج

Superdav AI Agent v1.18.0 قابلِ اعتماد ڈویلپر ورک فلو کے لیے الگ سے تقسیم کیا جانے والا جدید ساتھی پیکیج شامل کرتا ہے۔ یہ صلاحیتیں WordPress.org بنیادی پیکیج میں شامل نہیں ہیں۔ ساتھی پیکیج صرف ان سائٹس پر نصب کریں جہاں منتظمین ایجنٹ کو ڈویلپر ٹولنگ تک رسائی دینے پر واضح طور پر اعتماد کرتے ہوں۔

جدید ٹولز میں فائل سسٹم عملیات، ڈیٹابیس رسائی، WP-CLI، REST ڈسپیچر ورک فلو، plugin بلڈر یوٹیلٹیز، git اسنیپ شاٹس، صارف نظم و نسق مددگار، اور بینچ مارک ٹولنگ شامل ہیں۔ ان ٹولز کو قابلِ اعتماد منتظمین تک محدود رکھیں اور پروڈکشن سائٹس پر انہیں فعال کرنے سے پہلے منظوری گیٹ ترتیب کا جائزہ لیں۔

## v1.18.0 میں بہتریاں

- **Google Calendar سپورٹ** — شیڈول سے باخبر خودکاریوں کے لیے ترتیب دیے گئے کیلنڈرز اور ایونٹس پڑھیں
- **رابطہ اور شریکِ تقریب میپنگ** — کیلنڈر شرکا کو سائٹ صارفین اور رابطوں سے ملائیں
- **انسانی منظوری کے گیٹس** — جاری رکھنے سے پہلے جائزے کے لیے حساس خودکاریاں روکیں
- **یاددہانی کی نقل سے بچاؤ** — دوبارہ کوششوں یا مکرر چلانے پر نقلی اطلاعات سے بچنے کے لیے یاددہانیاں ریکارڈ کریں
- **TextBee SMS فراہم کنندہ** — مطلوبہ منظوریوں کے بعد ترتیب دی گئی ٹیکسٹ پیغام اطلاعات بھیجیں
- **Managed Superdav سروس سیٹ اپ** — معاونت یافتہ سائٹس کے لیے ہوسٹڈ سروس endpoints خودکار طور پر فراہم کریں
- **جدید ساتھی تقسیم** — قابلِ اعتماد ڈویلپر ٹولز کو WordPress.org بنیادی build سے الگ شائع کریں
- **اعتماد پذیری کی اصلاحات** — ماڈل انتخاب، درخواست ٹائم آؤٹس، ڈیفالٹ سیٹ اپ ماڈل ہینڈلنگ، استدلال متن ہینڈلنگ، دوبارہ کوشش رہنمائی، کیلنڈر token مضبوطی، یاددہانی مضبوطی، آن بورڈنگ لانچز، تصدیق شدہ صلاحیت دوبارہ شروع ہونا، اور WordPress.org پیکیجنگ جائزہ مسائل کو بہتر بنائیں
