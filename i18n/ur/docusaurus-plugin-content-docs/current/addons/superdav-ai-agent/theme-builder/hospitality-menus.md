# ہاسپیٹیلٹی مینیوز

**Hospitality Menus** فیچر Theme Builder کو آپ کی WordPress سائٹ میں براہ راست منظم کھانے اور مشروبات کے مینیو پیجز بنانے اور ایمبیڈ کرنے کی سہولت دیتا ہے۔

## جائزہ (Overview)

Theme Builder اب ہاسپیٹیلٹی کاروباروں، جن میں ریستوران، کیفے، بار، اور کیٹرنگ سروسز شامل ہیں، کے لیے پیشہ ورانہ، منظم مینیو پیجز بنا سکتا ہے۔ یہ مینیوز آپ کی سائٹ کے ڈیزائن میں مکمل طور پر مربوط ہوتے ہیں اور انہیں آسانی سے اپ ڈیٹ اور مینج کیا جا سکتا ہے۔

## سپورٹ شدہ ہاسپیٹیلٹی کی اقسام (Supported Hospitality Types)

- **Restaurants** — مکمل سروس والے ڈائننگ مینیوز
- **Cafes** — کافی اور ہلکے کھانے کے مینیوز
- **Bars and Lounges** — مشروبات اور ایپٹائزر کے مینیوز
- **Bakeries** — پیسٹری اور بریڈ کے مینیوز
- **Catering Services** — ایونٹ مینیو کے آپشنز
- **Food Trucks** — موبائل فوڈ سروس مینیوز
- **Breweries and Wineries** — تفصیلات کے ساتھ مشروبات کے مینیوز

## مینیو کا ڈھانچہ (Menu Structure)

### مینیو کیٹیگریز (Menu Categories)

مینیوز کو کیٹیگریز میں منظم کیا جاتا ہے:

- **Appetizers** — سٹارٹرز اور چھوٹے پلیٹس
- **Entrees** — مین کورسز
- **Sides** — ساتھ میں پیش کیے جانے والے کھانے اور سبزیاں
- **Desserts** — میٹھے پکوان
- **Beverages** — مشروبات (الکحل والے اور الکحل سے پاک)
- **Specials** — روزانہ یا موسمی خصوصی پیشکش

### مینیو آئٹم کا فارمیٹ (Menu Item Format)

ہر مینیو آئٹم میں شامل ہوتا ہے:

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### آئٹم فیلڈز (Item Fields)

| Field | Type | Description |
|-------|------|-------------|
| `name` | string | ڈش یا مشروب کا نام |
| `description` | string | آئٹم کی تفصیلی وضاحت |
| `price` | string | قیمت (کرنسی کے ساتھ فارمیٹ شدہ) |
| `dietary_info` | array | غذائی خصوصیات (Vegan, Gluten-Free, وغیرہ) |
| `allergens` | array | عام الرجنز (Nuts, Shellfish, وغیرہ) |
| `availability` | string | کب دستیاب ہے (Daily, Seasonal, وغیرہ) |

## ہاسپیٹیلٹی مینیوز بنانا (Creating Hospitality Menus)

### مرحلہ 1: مینیو کی معلومات فراہم کریں (Provide Menu Information)

Theme Builder کو اپنے مینیو کے بارے میں بتائیں:

```
Create a restaurant menu for my Italian restaurant with appetizers,
pasta dishes, main courses, and desserts. Include prices and descriptions.
```

### مرحلہ 2: Theme Builder مینیو تیار کرتا ہے (Theme Builder Generates Menu)

Theme Builder:

1. ایک منظم مینیو پیج بناتا ہے
2. اسے آپ کی سائٹ کے تھیم کے مطابق ڈیزائن کرتا ہے
3. آئٹمز کو کیٹیگریز میں ترتیب دیتا ہے
4. قیمتوں اور تفصیلات کو فارمیٹ کرتا ہے
5. غذائی اور الرجن کی معلومات شامل کرتا ہے

### مرحلہ 3: جائزہ لیں اور اپنی مرضی کے مطابق بنائیں (Review and Customize)

آپ یہ کر سکتے ہیں:

1. مینیو آئٹمز اور قیمتیں ایڈٹ کرنا
2. کیٹیگریز شامل یا ہٹانا
3. کیٹیگریز کے اندر آئٹمز کو دوبارہ ترتیب دینا
4. تفصیلات اور غذائی معلومات کو اپ ڈیٹ کرنا
5. اسٹائلنگ اور لے آؤٹ کو ایڈجسٹ کرنا

## مینیو دکھانے کے آپشنز (Menu Display Options)

### مکمل مینیو پیج (Full Menu Page)

ایک مخصوص پیج جو آپ کا مکمل مینیو دکھاتا ہے:

- کیٹیگری کے لحاظ سے منظم
- سرچ اور فلٹر کرنے کے قابل
- پرنٹ کے لیے موزوں لے آؤٹ
- موبائل پر بہترین نظر آنے والا ڈیزائن

### مینیو وِجٹ (Menu Widget)

مینیو سیکشنز کو دیگر پیجز پر ایمبیڈ کریں:

- ہوم پیج پر نمایاں آئٹمز
- سائی بار پر روزانہ کی خصوصی پیشکش
- بار پیج پر مشروبات کا مینیو
- فوٹر میں ڈیزرٹ کا شوکیس

### مینیو پی ڈی ایف (Menu PDF)

ایک ڈاؤن لوڈ کے قابل پی ڈی ایف مینیو تیار کریں:

- پیشہ ورانہ فارمیٹنگ
- پرنٹ کے لیے تیار معیار
- تصاویر اور تفصیلات شامل
- شیئر اور ای میل کرنے میں آسان

## غذائی اور الرجن کی معلومات (Dietary and Allergen Information)

### غذائی خصوصیات (Dietary Attributes)

آئٹمز کو غذائی معلومات سے نشان زد کریں:

- **Vegan** — کوئی جانوروں کے مصنوعات نہیں
- **Vegetarian** — کوئی گوشت نہیں
- **Gluten-Free** — سیلیق کی بیماری کے لیے محفوظ
- **Dairy-Free** — کوئی دودھ کے مصنوعات نہیں
- **Nut-Free** — کوئی درخت کے میوہ جات یا مونگ پھلی نہیں
- **Low-Carb** — کم کاربوہائیڈریٹ
- **High-Protein** — پروٹین سے بھرپور

### الرجن وارننگز (Allergen Warnings)

عام الرجنز شامل کریں:

- **Nuts** — درخت کے میوہ جات اور مونگ پھلی
- **Shellfish** — کرسٹیشینز اور مالوکسک
- **Fish** — تمام مچھلی کی اقسام
- **Dairy** — دودھ اور دودھ کے مصنوعات
- **Eggs** — مرغی کے انڈے
- **Soy** — سویا مصنوعات
- **Gluten** — گندم اور گلوٹین پر مشتمل اناج
- **Sesame** — کیندی کے بیج اور تیل

## مینیو کا انتظام (Menu Management)

### قیمتیں اپ ڈیٹ کرنا (Updating Prices)

مینیو کی قیمتوں کو آسانی سے اپ ڈیٹ کریں:

1. مینیو پیج پر جائیں
2. "Edit Menu" پر کلک کریں
3. آئٹمز کی قیمتیں اپ ڈیٹ کریں
4. تبدیلیاں محفوظ کریں
5. تبدیلیاں فوراً آپ کی سائٹ پر نظر آئیں گی

### موسمی آئٹمز شامل کرنا (Adding Seasonal Items)

موسمی مینیو کی مختلف اقسام بنائیں:

1. ایک نیا مینیو ورژن بنائیں
2. موسمی آئٹمز شامل کریں
3. آئٹمز کو "Seasonal" کے طور پر نشان زد کریں
4. دستیابی کی تاریخیں طے کریں
5. موسم کے دوران خود بخود دکھایا جائے گا

### خصوصی پیشکشوں کا انتظام (Managing Specials)

روزانہ یا ہفتہ وار خصوصی پیشکشیں دکھائیں:

1. ایک "Specials" کیٹیگری بنائیں
2. دستیابی کی تاریخوں کے ساتھ آئٹمز شامل کریں
3. ہوم پیج پر خصوصی پیشکشوں کو نمایاں کریں
4. روزانہ یا ہفتہ وار اپ ڈیٹ کریں
5. پرانے خصوصی پیشکشوں کو آرکائیو کریں

## Theme Builder کے ساتھ انضمام (Integration with Theme Builder)

جب آپ ہاسپیٹیلٹی سائٹس کے لیے Theme Builder استعمال کرتے ہیں:

1. **Automatic menu detection** — یہ پہچانتا ہے کہ آپ فوڈ/بیوریج کا کاروبار ہیں یا نہیں
2. **Menu page creation** — پیشہ ورانہ مینیو پیجز تیار کرتا ہے
3. **Design matching** — مینیوز آپ کی سائٹ کے تھیم سے مطابقت رکھتے ہیں
4. **Mobile optimization** — مینیوز فون پر خوبصورتی سے دکھائے جاتے ہیں
5. **SEO optimization** — مینیوز سرچ انجن کے لیے دوستانہ ہوتے ہیں

## بہترین طریقے (Best Practices)

### مینیو ڈیزائن (Menu Design)

- **واضح تنظیم** — منطقی کیٹیگری ڈھانچہ
- **پڑھنے میں آسان تفصیلات** — بھوک بڑھانے والی اور معلوماتی
- **مستقل قیمتیں** — واضح کرنسی اور فارمیٹنگ
- **پیشہ ور تصاویر** — اعلیٰ معیار کی کھانے کی تصاویر
- **خالی جگہ (Whitespace)** — پیج کو زیادہ بھرے نہ

### مواد (Content)

- **درست تفصیلات** — آئٹمز کی درست وضاحت کریں
- **خصوصی پیشکشوں کو نمایاں کریں** — خصوصی پیشکشوں کو نمایاں بنائیں
- **الرجنز شامل کریں** — ہمیشہ الرجنز کی فہرست بنائیں
- **باقاعدگی سے اپ ڈیٹ کریں** — قیمتوں اور آئٹمز کو تازہ رکھیں
- **بھوک بڑھانے والی زبان استعمال کریں** — آئٹمز کو مزیدار لگنے دیں۔

### رسائی (Accessibility)

- **پڑھنے میں آسان فونٹس** — واضح، قابلِ قراءت ٹائپ فیس استعمال کریں
- **کافی کنٹراسٹ** — یقینی بنائیں کہ متن پڑھنے کے قابل ہو
- **غذائی لیبلز** — غذائی آپشنز کو واضح طور پر نشان زد کریں
- **الرجن وارننگز** — الرجنز کو نمایاں طور پر دکھائیں
- **موبائل کے لیے موزوں** — تمام ڈیوائسز پر ٹیسٹ کریں
