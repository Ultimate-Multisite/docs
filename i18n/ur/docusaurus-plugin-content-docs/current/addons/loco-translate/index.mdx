# Loco Translate {#loco-translate}

Ultimate Multisite: Loco Translate Integration addon آپ کے WordPress multisite network میں site-specific translation کی صلاحیت لاتا ہے۔ اگر آپ کو کبھی یہ ضرورت پڑی ہے کہ آپ کے network کے مختلف sites کو ایک ہی plugin یا theme کی مختلف translations چاہیے، تو یہ addon بہترین حل ہے۔

**یہ Addon کیا کرتا ہے**

یہ addon Ultimate Multisite کو مشہور Loco Translate plugin کے ساتھ آسانی سے جوڑ دیتا ہے، جس سے آپ کے network کے ہر subsite کو اپنی منفرد translations برقرار رکھنے کی سہولت ملتی ہے۔ چاہے آپ ایک multi-regional platform چلا رہے ہوں، مختلف زبانوں کے کمیونٹی کو سروس دے رہے ہوں، یا صرف اپنے network میں translation کی لچک کی ضرورت ہو، یہ addon اسے بہت آسان بنا دیتا ہے۔

**کس کے لیے بہترین ہے:**

* مختلف localization کی ضروریات والے multi-regional کاروبار
* متنوع کمیونٹیوں کو سروس دینے والے تعلیمی ادارے
* multilingual support پیش کرنے والے SaaS platforms
* متعدد client sites کو manage کرنے والی web agencies
* کسی بھی multisite network کو جس میں translation flexibility کی ضرورت ہو

## تنصیب (Installation) {#installation}

1. یقینی بنائیں کہ Ultimate Multisite install اور network activated ہے۔
2. Loco Translate plugin install اور activate کریں۔
3. اس addon کو اپنے plugins directory میں upload کریں۔
4. Loco Translate Integration addon کو network level پر activate کریں۔
5. کسی بھی subsite پر Loco Translate کے ذریعے site-specific translations بنانا شروع کریں۔

بس! یہ integration بغیر کسی اضافی configuration کے خود بخود کام کرنا شروع کر دیتا ہے۔

## اکثر پوچھے گئے سوالات (Frequently Asked Questions) {#frequently-asked-questions}

## کیا تنصیب کے بعد مجھے کچھ configure کرنے کی ضرورت ہے؟ {#do-i-need-to-configure-anything-after-installation}

نہیں! یہ integration خودکار (automatic) ہے۔ ایک بار جب addon اور Loco Translate دونوں active ہو جائیں گے، تو subsites فوراً اپنی منفرد translations بنانا شروع کر سکتے ہیں۔

## اگر کوئی subsite custom translations نہ بنائے تو کیا ہوگا؟ {#what-happens-if-a-subsite-doesnt-create-custom-translations}

site خود بخود network-wide translations کو fallback کے طور پر استعمال کرے گا۔ کچھ بھی خراب نہیں ہوگا، اور سب کچھ آسانی سے کام کرے گا۔

## کیا subsites network-wide translations تک رسائی حاصل کر سکتے ہیں؟ {#can-subsites-access-network-wide-translations}

عام site admins صرف اپنے site کی translations دیکھ اور ایڈٹ کر سکتے ہیں۔ Super admins کو یہ دیکھنے پر منحصر ہوتا ہے کہ وہ کون سا site دیکھ رہے ہیں، وہ network-wide اور site-specific دونوں translations تک رسائی حاصل کر سکتے ہیں۔

## کیا یہ plugins اور themes دونوں کے ساتھ کام کرتا ہے؟ {#does-this-work-with-both-plugins-and-themes}

جی ہاں! Site-specific translations WordPress plugins اور themes دونوں کے لیے کام کرتی ہیں۔

## کیا اس سے میرے multisite network کی رفتار کم ہو جائے گی؟ {#will-this-slow-down-my-multisite-network}

نہیں. یہ addon efficient file loading استعمال کرتا ہے جو صرف ضرورت پڑنے پر site-specific translations کو چیک کرتا ہے۔ اس سے کارکردگی (performance) پر کوئی قابلِ ذکر اثر نہیں پڑتا۔

## کیا میں network translations کو ایک starting point کے طور پر کسی خاص site میں copy کر سکتا ہوں؟ {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

جی ہاں! Loco Translate میں translations ایڈٹ کرتے وقت، آپ آسانی سے network translations سے شروع کر سکتے ہیں اور ہر site کے لیے انہیں customize کر سکتے ہیں۔

## کیا یہ WooCommerce اور دیگر plugins کے ساتھ کام کرتا ہے؟ {#does-this-work-with-woocommerce-and-other-plugins}

بالکل! کوئی بھی plugin یا theme جو standard WordPress translation functions استعمال کرتا ہے، وہ site-specific translations کے ساتھ کام کرے گا۔
