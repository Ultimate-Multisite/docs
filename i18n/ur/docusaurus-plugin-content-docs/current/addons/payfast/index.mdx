# PayFast {#payfast}

Multisite Ultimate: Payfast Gateway

یہ Payfast کو شامل کرتا ہے - جو جنوبی افریقہ میں ایک تیز اور محفوظ آن لائن ادائیگیوں کا گیٹ وے ہے - اسے Multisite Ultimate کے ادائیگی کے آپشن کے طور پر۔

## انسٹالیشن {#installation}

1. 'ultimate-multisite-payfast' کو '/wp-content/plugins/' ڈائریکٹری میں اپ لوڈ کریں۔
2. WordPress میں 'Plugins' مینیو کے ذریعے پلگ ان کو ایکٹیویٹ کریں۔
3. پلگ ان کو سیٹ اپ کرنے کے لیے مرحلہ وار ویزرڈ (Wizard) کی پیروی کریں۔

## استعمال کی گائیڈ {#usage-guide}

PayFast جنوبی افریقہ میں کاروباروں کے لیے ایک بڑا آن لائن ادائیگی کا پروسیسر ہے۔ یہ لین دین کو ہموار اور محفوظ بنا کر آن لائن تاجروں کو بڑھنے میں مدد دینے کے لیے وقف ہے۔ Ultimate Multisite صارف کی درخواست کے مطابق، ہم نے ایک ایڈ آن بنانے کا فیصلہ کیا ہے جو PayFast کے ساتھ انٹیگریشن کو ممکن بنائے گا۔ یہ آرٹیکل دکھاتا ہے کہ آپ اپنے Ultimate Multisite نیٹ ورک میں ادائیگی وصول کرنے کے لیے PayFast کو اپنے ادائیگی کے گیٹ وے کے طور پر کیسے سیٹ اپ کریں۔

پہلا قدم ایڈ آن کو انسٹال کرنا ہے۔

# Payfast ایڈ آن کو کیسے انسٹال کریں: {#how-to-install-the-payfast-add-on}

ایڈ آنز کے پیج پر جائیں اور Ultimate Multisite: Payfast Gateway ایڈ آن کو منتخب کریں۔

![Add-ons page](/img/admin/addons-page.png)

ایڈ آن کی ونڈو کھل جائے گی۔ 'Install Now' پر کلک کریں۔

![Add-ons page with install option](/img/admin/addons-page.png)

انسٹالیشن مکمل ہونے کے بعد، آپ کو پلگ انز کے پیج پر ری ڈائریکٹ کیا جائے گا۔ یہاں، صرف 'Network Activate' پر کلک کریں اور _Payfast_ ایڈ آن آپ کے نیٹ ورک پر ایکٹیویٹ ہو جائے گا۔

# PayFast کو سیٹ اپ کرنا {#setting-up-payfast}

ایڈ آن انسٹال کرنے کے بعد، آپ اب Ultimate Multisite کی سیٹنگز میں جا کر PayFast کو کنفیگر کرنا شروع کر سکتے ہیں۔

**Ultimate Multisite settings** پیج کے تحت، آپ کو "Payments" ٹیب مل جائے گا۔ اس ٹیب کے تحت، آپ کو "Payfast" ملے گا جس کے ساتھ اس ادائیگی کے گیٹ وے کو فعال یا غیر فعال کرنے کے لیے ایک ٹوگل بٹن ہوگا۔

![Payment gateway settings](/img/config/settings-payment-gateways.png)آگے بڑھیں اور اسے فعال کرنے کے لیے ٹوگل بٹن پر کلک کریں۔ اس کے بعد یہ باکس پھیل جائے گا اور مختلف فیلڈز دکھائے گا جنہیں آپ کو بھرنے کی ضرورت ہے، جیسے کہ **Merchant ID, Merchant Key اور PassPhrase۔** آپ یہ معلومات اپنے **Payfast** اکاؤنٹ سے حاصل کر سکتے ہیں۔

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

اگلا قدم وہ معلومات حاصل کرنے کے لیے اپنے PayFast اکاؤنٹ میں لاگ ان کرنا ہے۔

Upon logging in, you can immediately see the credentials you need from your **Payfast** account dashboard.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->آپ بس کریڈینشلز کو کاپی اور پیسٹ کر کے اپنی Ultimate Multisite کی سیٹنگز میں تفصیلات بھر سکتے ہیں۔ پھر تبدیلیوں کو محفوظ کرنے کے لیے **"Save settings** " بٹن پر کلک کریں۔

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->
