# Filter: wu_limits_is_fluent_form_above_limit {#filter-wulimitsisfluentformabovelimit}

یہ چیک کرتا ہے کہ کیا کوئی دی گئی فارم کی قسم اس پلان پر استعمال کرنے کی اجازت ہے۔ یہ پلگ ان ڈویلپرز کو ریٹرن ویلیو کو فلٹر کرنے کی سہولت دیتا ہے۔

## پیرامیٹرز {#parameters}

| Name | Type | تفصیل |
|------|------|-------------|
| $value | `bool` | اگر فارم کی قسم حد سے زیادہ ہے یا نہیں |
| $form_type | `string` | جس فارم کی قسم کو چیک کیا جا رہا ہے |
| $form_count | `int` | موجودہ فارم کی تعداد |
| $quota | `int` | اجازت یافتہ کوٹا |
