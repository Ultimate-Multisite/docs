# Fluent Forms

یہ ایڈ آن (add-on) Multisite Ultimate پر Multisite کے اندر پروڈکٹس، ممبرشپس یا سائٹس میں Fluent Forms پر حدیں (limits) لگانے کو ممکن بناتا ہے۔
Fluent Forms، کسٹم پوسٹ ٹائپس (custom post types) کے بجائے کسٹم ٹیبلز (custom tables) استعمال کرتا ہے، اس لیے عام حد بندی کے آپشنز Fluent Forms کے ساتھ کام نہیں کرتے۔

## خصوصیات (Features)

- Fluent Forms یا Conversational Fluent Forms کے لیے حدیں (limits) شامل کریں۔
- جب صارفین حد تک پہنچ جاتے ہیں تو انہیں ایک دوستانہ پیغام ملتا ہے جو انہیں اپنا پلان اپ گریڈ کرنے کی ترغیب دیتا ہے۔
