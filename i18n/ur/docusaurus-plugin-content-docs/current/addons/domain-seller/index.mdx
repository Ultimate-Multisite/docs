# ڈومین فروخت کنندہ

**Multisite Ultimate: ڈومین فروخت کنندہ** آپ کے WordPress Multisite نیٹ ورک کو ایک طاقتور ڈومین فروخت کرنے والے پلیٹ فارم میں بدل دیتا ہے۔ صارفین کو سائن اپ کے دوران یا اپنے Dashboard سے براہِ راست حسبِ ضرورت ڈومینز رجسٹر کرنے دیں، ریئل ٹائم دستیابی کی جانچ اور شفاف قیمتوں کے ساتھ۔

### ڈومین فروخت کنندہ کیوں؟ {#why-domain-seller}

ڈومین رجسٹریشن کسی بھی WaaS (ویب سائٹ بطور سروس) پلیٹ فارم کے لیے سب سے زیادہ مطلوب اضافی فروختوں میں سے ایک ہے۔ اپنے صارفین کو کسی تیسرے فریق رجسٹرار کی طرف بھیجنے کے بجائے، آپ براہِ راست ڈومینز پیش کر سکتے ہیں — یوں زیادہ آمدنی اپنے نیٹ ورک میں رکھتے ہوئے صارفین کو بے رکاوٹ تجربہ دیتے ہیں۔

### اہم خصوصیات {#key-features}

#### متعدد ڈومین رجسٹرارز کی سپورٹ {#multiple-domain-registrar-support}
* **OpenSRS** — مکمل رجسٹریشن، تجدید، اور انتظام
* **Namecheap** — WhoisGuard اور sandbox ٹیسٹنگ کے ساتھ مکمل انضمام
* **HostAfrica** — Domains Reseller API کے ذریعے ڈومین تلاش، قیمتیں، رجسٹریشن، منتقلی، DNS، اور ID تحفظ
* **Openprovider** — ری سیلر قیمتیں، رجسٹریشن، تجدید، منتقلیاں، DNS، WHOIS رازداری، اور TLD ہم آہنگی
* **Hostinger** — مشترکہ Hostinger انضمامی ٹوکن استعمال کرتے ہوئے ڈومین تلاش اور رجسٹریشن
* **قابلِ توسیع ساخت** — Integration Registry pattern پر مبنی، جس سے مزید فراہم کنندگان شامل کرنا آسان ہو جاتا ہے
* **بلٹ اِن کنکشن ٹیسٹنگ** — لائیو جانے سے پہلے فراہم کنندہ کی اسناد اور رابطہ کاری کی توثیق کریں

#### رہنمائی والا سیٹ اپ وزرڈ {#guided-setup-wizard}
* **ایک کلک آن بورڈنگ** — فراہم کنندہ کی ترتیب، TLD درآمد، اور ڈومین پروڈکٹ بنانے کے عمل میں آپ کی رہنمائی کرتا ہے
* **خودکار TLD ہم آہنگی** — آپ کے منسلک فراہم کنندگان سے تمام دستیاب TLDs اور تھوک قیمتیں درآمد کرتا ہے
* **ڈیفالٹ پروڈکٹ تخلیق** — معقول مارک اپ ڈیفالٹس کے ساتھ استعمال کے لیے تیار ڈومین پروڈکٹ بناتا ہے

#### لچکدار ڈومین قیمتیں {#flexible-domain-pricing}
* **تین مارک اپ طریقے** — فیصدی مارک اپ، مقررہ مارک اپ، یا ہر TLD کے لیے مقررہ قیمت
* **TLD مخصوص پروڈکٹس** — مختلف TLD گروپس (.com/.net، .io، ملکی کوڈز، وغیرہ) کے لیے الگ ڈومین پروڈکٹس بنائیں
* **تعارفی قیمتیں** — نئی رجسٹریشنز کو راغب کرنے کے لیے پہلے سال کی رعایتی قیمت پیش کریں
* **ریئل ٹائم قیمتیں** — ذہین کیشنگ کے ساتھ فراہم کنندگان سے براہِ راست تھوک قیمتیں
* **کوپن سپورٹ** — Multisite Ultimate رعایتی کوڈز کے ساتھ مکمل مطابقت

#### WHOIS رازداری کا تحفظ {#whois-privacy-protection}
* **ہر پروڈکٹ کی ترتیب** — WHOIS رازداری کو غیر فعال، ہمیشہ شامل، یا صارف کا انتخاب مقرر کریں
* **صارف کی آپٹ اِن** — جب صارف کا انتخاب مقرر ہو، تو checkout چیک باکس صارفین کو رازداری کا تحفظ شامل کرنے دیتا ہے
* **قابلِ ترتیب قیمتیں** — رازداری add-on کے لیے سالانہ قیمت مقرر کریں، یا اسے مفت پیش کریں
* **رجسٹرار انضمام** — رجسٹریشن کے وقت خودکار طور پر WhoisGuard (Namecheap) یا WHOIS رازداری (OpenSRS) فعال کرتا ہے

#### متحد checkout تجربہ {#unified-checkout-experience}
* **ڈومین کے تین طریقے** — صارفین ذیلی ڈومین (مفت)، نیا ڈومین رجسٹر کرنے، یا پہلے سے اپنی ملکیت میں موجود ڈومین میپ کرنے میں سے انتخاب کرتے ہیں
* **ریئل ٹائم ڈومین تلاش** — AJAX سے چلنے والی دستیابی کی جانچ، بغیر صفحہ دوبارہ لوڈ کیے
* **ذہین تجاویز** — جب مطلوبہ ڈومین لیا جا چکا ہو تو متبادل TLD تجاویز
* **شفاف قیمتیں** — رجسٹریشن قیمت، تجدید قیمت، اور اختیاری WHOIS رازداری فیس کی واضح نمائش
* **رجسٹرنٹ رابطہ فیلڈز** — checkout کے دوران درکار رجسٹرنٹ معلومات (نام، پتہ، فون) اسی جگہ جمع کرتا ہے
* **خودکار طور پر بنایا گیا سائٹ URL** — سائٹ کا پتہ منتخب کردہ ڈومین سے خودکار طور پر اخذ کیا جاتا ہے

#### خودکار تجدیدی نظام {#auto-renewal-system}
* **رکنیت سے منسلک تجدیدیں** — جب تک صارف کی رکنیت فعال ہو، ڈومینز خودکار طور پر تجدید ہوتے ہیں
* **ذہین شیڈولنگ** — تجدیدی کام ڈومین کی میعاد ختم ہونے کی تاریخوں کی بنیاد پر قطار میں لگائے جاتے ہیں
* **دوبارہ کوشش کی منطق** — ناکام تجدیدی کوششوں کے لیے ذہین دوبارہ کوششیں
* **ای میل اطلاعات** — آنے والی تجدیدوں، کامیاب تجدیدوں، ناکامیوں، اور میعاد ختم ہونے کے لیے قابلِ ترتیب الرٹس

#### صارف DNS انتظام {#customer-dns-management}
* **سیلف سروس DNS** — صارفین اپنے Dashboard سے DNS ریکارڈز (A, AAAA, CNAME, MX, TXT) شامل، ترمیم، اور حذف کر سکتے ہیں
* **ڈومین اسٹیٹس** — ریئل ٹائم رجسٹریشن اسٹیٹس، میعاد ختم ہونے کی تاریخیں، اور نیم سرور معلومات
* **خودکار ڈومین میپنگ** — رجسٹرڈ یا موجودہ ڈومینز خودکار طور پر صارف کی سائٹ سے میپ کیے جاتے ہیں

#### ایڈمن ٹولز {#admin-tools}
* **دستی ڈومین رجسٹریشن** — ایڈمن پینل سے صارفین کی طرف سے ڈومینز رجسٹر کریں
* **ڈومین مانیٹرنگ** — ایک صفحے سے تمام صارف ڈومینز، اسٹیٹسز، اور میعاد ختم ہونے کی تاریخیں دیکھیں
* **فراہم کنندہ مخصوص لاگنگ** — ہر رجسٹرار کے لیے الگ لاگ چینلز (مثلاً، domain-seller-namecheap.log)
* **رجسٹریشن تفصیلات** — بنیادی ڈومین ترمیمی صفحے پر رجسٹرنٹ معلومات اور رجسٹریشن ڈیٹا دیکھیں

### معاون فراہم کنندگان {#supported-providers}

* **OpenSRS** — رجسٹریشن، تجدید، DNS انتظام، اور WHOIS رازداری کے ساتھ مکمل انضمام
* **Namecheap** — رجسٹریشن، تجدید، بیچ دستیابی جانچ، اور مفت WhoisGuard کے ساتھ مکمل انضمام
* **HostAfrica** — رجسٹریشن، تجدید، منتقلیاں، DNS انتظام، EPP کوڈ بازیافت، رجسٹرار لاک، اور ID تحفظ
* **Openprovider** — رجسٹریشن، تجدید، منتقلیاں، DNS انتظام، EPP کوڈ بازیافت، رجسٹرار لاک، WHOIS رازداری، اور ری سیلر TLD ہم آہنگی
* **Hostinger** — رجسٹریشن، ڈومین تلاش، نیم سرور اپ ڈیٹس، رجسٹرار لاک، اور WHOIS رازداری؛ تجدیدیں Hostinger auto-renew کے ذریعے سنبھالی جاتی ہیں
* **GoDaddy** — رجسٹریشن اور تجدید
* **ResellerClub** — رجسٹریشن، تجدید، اور DNS انتظام
* **NameSilo** — رجسٹریشن اور تجدید
* **Enom** — رجسٹریشن، تجدید، اور DNS انتظام

### تکنیکی تفصیلات {#technical-details}

* **درکار ہے:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **ساخت:** Domain_Selling_Capability interface کے ساتھ Integration Registry pattern
* **Autoloading:** PSR-4 class mapping کے ساتھ Jetpack autoloader
* **معیارات:** WordPress coding standards، strict typing، prepared statements

## تنصیب {#installation}

1. یقینی بنائیں کہ **Multisite Ultimate v2.4.12+** نصب ہے اور نیٹ ورک پر فعال ہے
2. plugin کو `/wp-content/plugins/ultimate-multisite-domain-seller/` میں اپ لوڈ کریں
3. WordPress Network Admin میں **Plugins** سے plugin کو نیٹ ورک پر فعال کریں
4. پہلی activation پر **Setup Wizard** خودکار طور پر شروع ہوتا ہے — ان مراحل پر عمل کریں تاکہ:
   * اپنے domain registrar کو connect کریں
   * connection کو test کریں
   * دستیاب TLDs اور wholesale pricing درآمد کریں
   * pricing markup کے ساتھ اپنا پہلا domain product بنائیں
5. **Ultimate Multisite > Checkout Forms** میں اپنے checkout form میں **Domain Selection** field شامل کریں

## اکثر پوچھے گئے سوالات {#frequently-asked-questions}

### Multisite Ultimate کا کون سا version درکار ہے؟ {#what-version-of-multisite-ultimate-is-required}

Version 2.4.12 یا اس سے زیادہ۔ یہ addon WP Ultimo v1.x یا 2.4.12 سے کم Ultimate Multisite versions کے ساتھ compatible نہیں ہے۔

### کون سے domain providers supported ہیں؟ {#which-domain-providers-are-supported}

نو registrars supported ہیں: **OpenSRS**، **Namecheap**، **HostAfrica**، **Openprovider**، **Hostinger**، **GoDaddy**، **ResellerClub**، **NameSilo**، اور **Enom**۔ صلاحیتیں provider کے لحاظ سے مختلف ہوتی ہیں کیونکہ registrar APIs مختلف operations ظاہر کرتی ہیں؛ provider-by-provider جائزے کے لیے setup guide دیکھیں۔ addon کا Integration Registry architecture custom providers شامل کرنا آسان بناتا ہے — [developer guide](/developer/integration-guide/custom-registrar) دیکھیں۔

### کیا میں WHOIS privacy protection پیش کر سکتا ہوں؟ {#can-i-offer-whois-privacy-protection}

جی ہاں۔ ہر domain product میں تین modes کے ساتھ WHOIS Privacy setting ہوتی ہے: Disabled، Always Included، یا Customer Choice۔ جب اسے Customer Choice پر set کیا جائے تو customers checkout کے دوران price (یا Namecheap کے WhoisGuard کے لیے "Free") کے ساتھ checkbox دیکھتے ہیں۔ privacy flag registration کے وقت خودکار طور پر registrar کو pass کر دیا جاتا ہے۔

### کیا میں مختلف TLDs کے لیے مختلف pricing set کر سکتا ہوں؟ {#can-i-set-different-pricing-for-different-tlds}

جی ہاں۔ مختلف TLD groups کے لیے الگ domain products بنائیں۔ ہر product کا اپنا markup type (percentage، fixed markup، یا fixed price)، supported TLDs، اور introductory pricing ہو سکتی ہے۔

### customers کو checkout کے کون سے options نظر آتے ہیں؟ {#what-checkout-options-do-customers-see}

Domain Selection field customers کو تین tabs دیتی ہے: **Subdomain** (اپنے network پر free subdomain استعمال کریں)، **Register New Domain** (نیا domain search اور register کریں)، اور **Existing Domain** (ایسا domain map کریں جو وہ پہلے سے own کرتے ہیں)۔ صرف وہ modes ظاہر ہوں گے جنہیں آپ enable کریں گے۔

### کیا یہ automatic renewals کو support کرتا ہے؟ {#does-it-support-automatic-renewals}

جی ہاں۔ Domains کو auto-renew پر set کیا جا سکتا ہے اور یہ customer's membership status سے tied ہوتے ہیں۔ Renewal payments customer کے existing payment gateway کے ذریعے process کی جاتی ہیں۔

### کیا customers اپنے DNS records manage کر سکتے ہیں؟ {#can-customers-manage-their-dns-records}

جی ہاں۔ Customers کو اپنے dashboard میں DNS management interface ملتا ہے تاکہ A، AAAA، CNAME، MX، اور TXT records add، edit، اور delete کر سکیں۔ DNS management OpenSRS، ResellerClub، Enom، HostAfrica، اور Openprovider کے ساتھ available ہے۔ hosted domains کے لیے Hostinger DNS core Hostinger domain-mapping integration کے ذریعے handle کیا جاتا ہے۔

### کیا کوئی setup wizard ہے؟ {#is-there-a-setup-wizard}

جی ہاں۔ setup wizard پہلی activation پر خودکار طور پر چلتا ہے اور آپ کو registrar connect کرنے، TLDs import کرنے، اور اپنا پہلا domain product بنانے کے مراحل سے گزارتا ہے۔ آپ addon settings سے wizard دوبارہ بھی چلا سکتے ہیں۔
