# Content Sync {#content-sync}

Multisite Ultimate: Content Sync

Multisite Ultimate کے لیے یہ addon network administrators کو network کے کسی بھی site سے content کو دوسرے sites پر sync کرنے کی سہولت دیتا ہے۔ یہ posts, pages, custom post types، اور Elementor templates سمیت تمام post types کو سپورٹ کرتا ہے۔ اس addon میں content منتخب کرنے اور target sites کا انتخاب کرنے کے لیے ایک multi-step wizard interface شامل ہے، جس میں batch processing اور real-time progress feedback بھی ملتا ہے۔

## Features {#features}

* Content Selection: Thumbnails اور post type filtering کے ساتھ visual card grid۔
* All Post Types: posts, pages, products, Elementor templates، اور کسی بھی custom post type کو sync کریں۔
* Site Selection: منتخب کریں کہ content کو کن sites پر sync کرنا ہے، جس میں source site کو exclude کرنا اور child site کو پہلے سے select کرنا شامل ہے۔
* Batch Processing: ایک وقت میں 10 sites پر sync کرتا ہے اور progress updates فراہم کرتا ہے۔
* Media Sync: post content سے media files (images, videos) کو خود بخود copy کرتا ہے اور target sites پر unmap کر دیتا ہے۔
* Taxonomy Sync: categories, tags, اور custom taxonomies کو محفوظ رکھتا ہے۔
* Elementor Support: CSS regeneration اور template ID remapping کے ساتھ Elementor templates کی مکمل سپورٹ۔
* Conflict Resolution: موجودہ content کو replace کرنے یا اسے skip کرنے کا آپشن۔
* Cache Management: sync کے بعد target sites پر Elementor cache کو خود بخود clear کرتا ہے۔

## Requirements {#requirements}

* WordPress 5.3 or higher
* PHP 7.4 or higher
* Multisite Ultimate plugin (active)
* Elementor plugin (optional, required for Elementor template sync)

## Installation {#installation}

1. 'ultimate-multisite-content-sync' کو '/wp-content/plugins/' directory میں upload کریں۔
2. اپنے WordPress Network Admin میں 'Plugins' menu کے ذریعے plugin کو activate کریں۔
3. Ultimate Multisite → Settings → Content Sync میں settings configure کریں۔
4. Syncing شروع کرنے کے لیے Ultimate Multisite → Content Sync پر جائیں۔
