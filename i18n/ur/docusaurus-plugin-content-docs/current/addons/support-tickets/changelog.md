# سپورٹ ٹکٹس کی تبدیلیوں کا ریکارڈ {#support-tickets-changelog}

### 1.0.4 - 2026-05-05 {#104---2026-05-05}
* بہتر کیا: `vendor/` ڈائریکٹری کو Git ٹریکنگ سے ہٹا دیا (یہ پہلے ہی `.gitignore` سے کور ہے)، جس سے ریپوزٹری کا سائز کم ہو گیا۔
* بہتر کیا: WordPress 7.0 تک ٹیسٹ کیا گیا۔

### 1.0.3 - 2026-05-01 {#103---2026-05-01}
* ٹھیک کیا: قابلِ نل ٹکٹ ماڈل سیٹرز میں نل ویلیوز کو قبول کیا گیا۔
* ٹھیک کیا: مناسب ملٹی سائٹ سپورٹ کے لیے تمام ٹکٹس کو گلوبل نیٹ ورک-وائڈ ٹیبلز میں اسٹور کیا گیا۔
* ٹھیک کیا: سٹاف فیلڈز کو چھپایا گیا اور کسٹمر کے نئے ٹکٹ فارم پر بلینک پیج ریڈائریکٹ کو ٹھیک کیا گیا۔
* ٹھیک کیا: میٹا ڈیٹا اسٹوریج کے لیے غیر متعین `add_meta()` کالز کو درست `update_meta()` سے تبدیل کیا گیا۔
* ٹھیک کیا: غیر رجسٹرڈ کیپیبلٹی چیک کو درست `wu_view_all_support_tickets` سے تبدیل کیا گیا۔
* ٹھیک کیا: ٹکٹ کی اسٹیٹس، اسائنمنٹ، اور کوئیک-ایڈٹ ایکشنز کے لیے گمشدہ AJAX ہینڈلرز شامل کیے گئے۔
* ٹھیک کیا: ٹکٹ ویوز میں سٹاف رسپانس ڈیٹیکشن کے لیے میتھڈ کا نام درست کیا گیا۔
* ٹھیک کیا: ڈپلیکیٹ ریپلائی ہینڈلرز کو کنسولیڈیٹ کیا گیا اور نانس ایکشن کے ناموں کو ہم آہنگ کیا گیا۔
* ٹھیک کیا: `[wu_submit_ticket]` شارٹ کوڈ کے لیے گمشدہ فرنٹ اینڈ ویو شامل کیا گیا۔
* ٹھیک کیا: گمشدہ `user_id` کالم شامل کیا گیا اور `Support_Ticket::get_user_id()` میتھڈ کو ٹھیک کیا گیا۔
* ٹھیک کیا: پرائیورٹی فلٹر آپشن ٹیگ میں اضافی ڈبل کوٹ کو ہٹا دیا گیا۔
* ٹھیک کیا: سپر ایڈمنز کے لیے نیٹ ورک ایڈمن ٹکٹ مینجمنٹ پینل شامل کیا گیا۔
* بہتر کیا: ایڈمن CSS کو ایک سنگل ایکسٹرنل سٹائل شیٹ میں کنسولیڈیٹ کیا گیا۔
* بہتر کیا: سب سائیٹ ایڈمن مینیو سے سیٹنگز سب مینیو کو ہٹا دیا گیا۔
* بہتر کیا: فرنٹ اینڈ ایسٹس کو صرف سپورٹ ٹکٹ پیجز پر کنڈیشنل طور پر لوڈ کیا گیا۔
* بہتر کیا: جب Bedrock روٹ آٹOLOADER پہلے ہی ڈیپینڈنسیز لوڈ کر چکا ہو تو پلگ ان آٹOLOADER کو چھوڑ دیا گیا۔

### 1.0.2 - 2025-12-11 {#102---2025-12-11}
* شامل کیا: ایڈمن اور فرنٹ اینڈ کے لیے مکمل ٹکٹ دیکھنے کی فعالیت۔
* شامل کیا: ٹکٹ ریپلائی جمع کرانے کے لیے AJAX ہینڈلر۔
* شامل کیا: مناسب فارم ہینڈلنگ کے ساتھ ٹکٹ ریپلائی کی فعالیت کے لیے سپورٹ۔
* شامل کیا: ایڈمن میں ٹکٹ جمع کرانے اور ریپلائی کے لیے مناسب نوٹس ڈسپلے۔
* شامل کیا: کسٹمر کی اپنی طرف سے جمع کرائی گئی معلومات کے لیے موجودہ صارف کے ساتھ خودکار ٹکٹ ایسوسی ایشن۔
* شامل کیا: کسٹمرز کو ٹکٹ کی ملکیت کو اووررائڈ کرنے سے روکنے کے لیے سیکیورٹی بہتری۔
* شامل کیا: گمشدہ ہیلپر فنکشنز (`wu_format_date`, `wu_user_can_view_ticket`, وغیرہ)۔
* شامل کیا: مناسب فائل اٹیچمنٹ ڈاؤن لوڈ اور ہینڈلنگ۔
* شامل کیا: ٹکٹ ریپلائی اور اسٹیٹس میں تبدیلی کے لیے ای میل نوٹیفیکیشن سسٹم۔
* ٹھیک کیا: ٹکٹ ریپلائی فارم میں اب ضروری ٹکٹ ID شامل ہے۔
* ٹھیک کیا: ایڈمن ٹکٹ ویوز میں نوٹس کی مرئیت کے مسائل۔
* ٹھیک کیا: سپورٹ ٹکٹ فنکشنز میں Syntax غلطیاں۔
* بہتر کیا: ریپلائیز اور اٹیچمنٹس کے لیے مناسب کوئری اور اسکیما کلاسز کے ساتھ ڈیٹا بیس کی ساخت۔
* بہتر کیا: ٹکٹ ماڈلز کے لیے اٹریبیوٹس کے بجائے حقیقی پراپرٹیز پر مائیگریشن۔
