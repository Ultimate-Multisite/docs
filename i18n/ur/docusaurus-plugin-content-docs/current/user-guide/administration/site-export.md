# سائٹ ایکسپورٹ {#site-export}

**سائٹ ایکسپورٹ** ایڈمن پیج نیٹ ورک ایڈمنسٹریٹرز کو ایک سائٹ، یا پورے نیٹ ورک کو، ڈاؤن لوڈ کے قابل آرکائیو میں پیک کرنے کی سہولت دیتا ہے تاکہ اسے منتقل (migrate) کیا جا سکے، بیک اپ لیا جا سکے، یا کسی اور کو حوالے (hand-off) کیا جا سکے۔

## ایک سائٹ ایکسپورٹ کرنا {#exporting-one-site}

**Ultimate Multisite > Site Export** پر جائیں اور **Generate new Site Export** کا انتخاب کریں۔ اس سب-سائٹ کا انتخاب کریں جسے آپ ایکسپورٹ کرنا چاہتے ہیں، اور پھر یہ منتخب کریں کہ کیا آرکائیو میں اپلوڈز، پلگ انز، اور تھیمز شامل ہونے چاہئیں۔

جب ایکسپورٹ مکمل ہو جائے گا، تو **Existing Exports** لسٹ سے ZIP ڈاؤن لوڈ کریں۔ ایکسپورٹ ZIPs میں اب ایک خود چلنے والا `index.php` اور ایک `readme.txt` شامل ہوتا ہے، تاکہ آرکائیو کو پہلے سے کوئی الگ امپورٹر پلگ ان انسٹال کیے بغیر، ایک نئے ہوسٹ پر اپ لوڈ کرکے شروع کیا جا سکے۔

## پورا نیٹ ورک ایکسپورٹ کرنا {#exporting-the-whole-network}

اگر آپ کو ایک ہی آرکائیو چاہیے جس میں نیٹ ورک کی تمام سب-سائٹس شامل ہوں، تو سائٹ ایکسپورٹ پیج پر **Network Export** استعمال کریں۔ یہ ہوسٹ کی منتقلی (host migrations)، ڈیزاسٹر-ریکوری کی مشقوں، یا سٹیجنگ کو دوبارہ بنانے کے حالات میں مفید ہے جہاں ہر سب-سائٹ کو ایک ساتھ سفر کرنے کی ضرورت ہو۔

چونکہ نیٹ ورک ایکسپورٹ ایک سنگل-سائٹ ایکسپورٹ سے کہیں زیادہ بڑا ہو سکتا ہے، اس لیے اسے کم ٹریفک والے وقت میں چلائیں اور اس بات کی تصدیق کریں کہ ٹارگٹ سٹوریج میں اپلوڈز، پلگ انز، تھیمز، اور بنائے گئے ZIP فائلوں کے لیے کافی خالی جگہ ہے۔

### نیٹ ورک امپورٹ بنڈلز {#network-import-bundles}

Ultimate Multisite 2.12.0 سے شروع ہو کر، Site Exporter **نیٹ ورک امپورٹ بنڈلز** تیار کر سکتا ہے — یہ خصوصی آرکائیو ہوتے ہیں جو نیٹ ورک کی سطح پر سائٹ کی بحالی (restoration) کو آسان بناتے ہیں۔ ایک نیٹ ورک امپورٹ بنڈل میں متعدد سائٹس کو ایک نئے نیٹ ورک انسٹالیشن پر بحال کرنے کے لیے تمام ضروری فائلیں اور میٹا ڈیٹا شامل ہوتا ہے۔

#### نیٹ ورک امپورٹ بنڈل تیار کرنا {#generating-a-network-import-bundle}

1. **Ultimate Multisite > Site Export** پر جائیں
2. **Generate new Network Export** پر کلک کریں
3. ایکسپورٹ کی قسم کے طور پر **Network Import Bundle** کا انتخاب کریں
4. یہ منتخب کریں کہ اپلوڈز، پلگ انز، اور تھیمز شامل ہونے چاہئیں یا نہیں
5. **Generate** پر کلک کریں
6. **Existing Exports** لسٹ سے بنڈل ZIP ڈاؤن لوڈ کریں

#### نیٹ ورک امپورٹ بنڈل سے بحال کرنا {#restoring-from-a-network-import-bundle}

سائٹس کو نیٹ ورک امپورٹ بنڈل سے بحال کرنے کے لیے:

1. اپنے ٹارگٹ ہوسٹ پر Ultimate Multisite انسٹال کریں۔
2. ملٹی سائٹ سیٹ اپ ویزرڈ مکمل کریں۔
3. نئے نیٹ ورک پر **Ultimate Multisite > Site Export** پر جائیں
4. **Import Network Bundle** پر کلک کریں
5. نیٹ ورک امپورٹ بنڈل ZIP فائل اپ لوڈ کریں۔
6. اسکرین پر دکھائی دینے والی امپورٹ ہدایات پر عمل کریں۔
7. امپورٹ کا عمل تمام سائٹس، ان کے مواد، اور کنفیگریشنز کو بحال کر دے گا۔

نیٹ ورک امپورٹ بنڈلز یہ چیزیں محفوظ رکھتے ہیں:
- سائٹ کا مواد (پوسٹس، پیجز، کسٹم پوسٹ ٹائپس)
- سائٹ کی سیٹنگز اور آپشنز
- صارف کے کردار اور اجازتیں
- پلگ انز اور تھیمز (اگر بنڈل میں شامل ہیں)
- میڈیا اپلوڈز (اگر بنڈل میں شامل ہیں)
- کسٹم ڈیٹا بیس ٹیبلز اور ڈیٹا

## ایک خود چلنے والا ایکسپورٹ ZIP انسٹال کرنا {#installing-a-self-booting-export-zip}

ایک نئے ہوسٹ پر خود چلنے والا ZIP بحال کرنے کے لیے:

1. ایکسپورٹ شدہ ZIP مواد کو ٹارگٹ ویب روٹ پر اپ لوڈ کریں۔
2. براؤزر میں اپ لوڈ شدہ `index.php` کھولیں۔
3. بنڈل شدہ ایکسپورٹ پیکج سے اسکرین پر دکھائی دینے والی انسٹالر ہدایات پر عمل کریں۔
4. عارضی فائلیں ہٹانے سے پہلے ایکسپورٹ سے متعلق نوٹس کے لیے بنڈل شدہ `readme.txt` کا جائزہ لیں۔

addon-specific انسٹالیشن اور امپورٹ تفصیلات کے لیے، [Site Exporter addon documentation](/addons/site-exporter) دیکھیں۔

Ultimate Multisite 2.9.0 میں شامل کردہ سنگل-سائٹ ٹول کے لیے، [Export & Import](/user-guide/administration/export-import) دیکھیں۔
