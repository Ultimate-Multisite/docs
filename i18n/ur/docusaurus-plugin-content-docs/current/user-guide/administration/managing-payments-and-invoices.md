# ادائیگیوں اور رسیدوں کا نظم {#managing-payments-and-invoices}

## ادائیگی کی ترتیبات {#payment-settings}

ادائیگیاں وصول کرنا شروع کرنے سے پہلے، آپ کو ادائیگی سے متعلق ترتیبات کو تشکیل دینا ہوگا۔ **Ultimate Multisite → Settings** پر جائیں اور **Payment** ٹیب پر کلک کریں۔

![ادائیگی کی ترتیبات کا ٹیب](/img/admin/settings-payments-top.png)

یہاں ادائیگی کی ترتیبات کے صفحے کا مکمل منظر ہے:

![ادائیگی کی ترتیبات کا مکمل صفحہ](/img/admin/settings-payments-full.png)

### عمومی ادائیگی کے اختیارات {#general-payment-options}

عمومی ترتیبات میں آپ تشکیل دے سکتے ہیں:

- **کرنسی** — لین دین کے لیے استعمال ہونے والی پہلے سے طے شدہ کرنسی
- **کرنسی کی پوزیشن** — کرنسی کا نشان کہاں ظاہر ہوتا ہے (رقم سے پہلے/بعد)

![ادائیگی گیٹ وے کی ترتیبات](/img/admin/settings-payments-options.png)

### ادائیگی گیٹ ویز {#payment-gateways}

Ultimate Multisite متعدد ادائیگی گیٹ ویز کو سپورٹ کرتا ہے۔ آپ Payment ترتیبات کے ٹیب سے ہر ایک کو فعال اور تشکیل دے سکتے ہیں۔

![ادائیگی گیٹ وے کی تشکیل](/img/admin/settings-payments-gateways.png)

دستیاب گیٹ ویز میں شامل ہیں:

- **Stripe** — Stripe کے ذریعے کریڈٹ کارڈ ادائیگیاں
- **PayPal** — PayPal ادائیگیاں
- **دستی** — آف لائن یا حسبِ ضرورت ادائیگی پراسیسنگ کے لیے

ہر گیٹ وے کا اپنا تشکیل سیکشن ہوتا ہے جہاں آپ API keys اور دیگر ترتیبات درج کرتے ہیں۔

![اضافی گیٹ وے ترتیبات](/img/admin/settings-payments-gateways-2.png)

### Sandbox Mode {#sandbox-mode}

لائیو ہونے سے پہلے اپنی ادائیگی انضمام کی جانچ کے لیے آپ **Sandbox Mode** فعال کر سکتے ہیں۔ جب sandbox mode فعال ہو تو کوئی حقیقی چارجز نہیں کیے جائیں گے۔

## ادائیگیاں دیکھنا {#viewing-payments}

اپنے نیٹ ورک بھر میں تمام لین دین دیکھنے کے لیے Ultimate Multisite کے تحت **Payments** صفحے پر جائیں۔

![ادائیگیوں کی فہرست](/img/admin/payments-list.png)

آپ ادائیگیوں کو حیثیت کے لحاظ سے فلٹر کر سکتے ہیں (مکمل، زیرِ التوا، ناکام، واپس کی گئی) اور مخصوص لین دین تلاش کر سکتے ہیں۔

کسی ادائیگی پر کلک کریں تاکہ اس کی مکمل تفصیلات دیکھ سکیں، جن میں لائن آئٹمز، متعلقہ ممبرشپ، صارف کی معلومات، اور ادائیگی گیٹ وے ڈیٹا شامل ہیں۔

## رسیدیں {#invoices}

Ultimate Multisite ادائیگیوں کے لیے خودکار طور پر رسیدیں بنا سکتا ہے۔ آپ Payment ترتیبات سے رسید کے template اور نمبرنگ format کو حسبِ ضرورت بنا سکتے ہیں۔

رسید کو حسبِ ضرورت بنانے کے اختیارات میں شامل ہیں:

- **کمپنی کا نام اور پتہ** جو رسیدوں پر دکھایا جاتا ہے
- **رسید نمبرنگ** کا format اور sequence
- **لوگو** جو رسید کے ہیڈر میں دکھایا جاتا ہے
- **حسبِ ضرورت فوٹر متن** شرائط، نوٹس، یا قانونی معلومات کے لیے

رسید کے template کو حسبِ ضرورت بنانے کے لیے، **Ultimate Multisite → Settings → Payment** پر جائیں اور رسید سے متعلق ترتیبات تلاش کریں۔
