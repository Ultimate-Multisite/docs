# Admin Panel کا دورہ

Ultimate Multisite کا مقصد زیادہ سے زیادہ سادہ اور خود واضح ہونا ہے، لیکن اتنے طاقتور ٹول کے ساتھ ایک رہنمائی والا دورہ مددگار ہو سکتا ہے۔ آئیے Admin صفحات سے گزرتے ہیں تاکہ آپ کو سمت سمجھنے میں مدد ملے۔

## Dashboard

Ultimate Multisite **Dashboard** بنیادی رپورٹس اور تجزیات دکھاتا ہے، جن میں آمدنیاں، site سرگرمیاں، membership کی بڑھوتری، visitor کی تعداد، اور جغرافیائی ڈیٹا شامل ہیں۔

![Ultimate Multisite Dashboard نچلا حصہ](/img/admin/um-dashboard-bottom.png)

![Network Dashboard مکمل صفحہ](/img/admin/network-dashboard-full-page.png)

## Checkout Forms

**Checkout Forms** آپ کو custom registration صفحات بنانے کی لچک دیتے ہیں۔ آپ fields کو customize کر سکتے ہیں، custom classes اور scripts شامل کر سکتے ہیں، اور visitor کے ملک یا مقام کی بنیاد پر رسائی محدود کر سکتے ہیں۔

![Checkout Forms فہرست](/img/admin/checkout-forms-list.png)

## Products

**Products** سیکشن وہ جگہ ہے جہاں آپ اپنے network کے لیے مختلف plans، packages، اور services بناتے ہیں۔ product سطح پر prices، billing frequencies، اور limits & quotas متعین کریں۔

![Products فہرست](/img/admin/products-list.png)

## Memberships

**Memberships** صفحہ آپ کے network میں تمام subscriptions دکھاتا ہے۔ plans، products، billing amounts اور frequencies، payment history، اور timestamps سمیت subscription details کا جائزہ لیں اور ترمیم کریں۔

![Memberships فہرست](/img/admin/memberships-list.png)

## Payments

**Payments** صفحہ آپ کے پورے network میں payment history کا فوری منظر فراہم کرتا ہے، جس میں specific products اور amounts سمیت detailed transaction information شامل ہوتی ہے۔

![Payments فہرست](/img/admin/payments-list.png)

## Customers

**Customers** صفحہ آپ کے network میں registered تمام members کو basic user information کے ساتھ دکھاتا ہے، جس میں نام، email address، last login، اور ان کے subsite میں login کرنے کے لیے "switch to" link شامل ہے۔

![Customers فہرست](/img/admin/customers-list.png)

## Sites

**Sites** صفحہ آپ کے network میں تمام subsites کی فہرست دیتا ہے، جس میں site templates اور customer-owned sites شامل ہیں۔ mapped domains، limits & quotas، plugins اور themes، اور site thumbnails کو آسانی سے manage کریں۔

![Sites فہرست](/img/admin/sites-list.png)

## Domains

**Domains** صفحہ subsites سے mapped custom domains کے لیے مخصوص ہے۔ super admin کے طور پر، آپ custom domain names کو manually add یا map کر سکتے ہیں اور DNS records اور logs دیکھ سکتے ہیں۔

![Domains فہرست](/img/admin/domains-list.png)

## Discount Codes

**Discount Codes** آپ کو اپنے end-users کو discounts پیش کرنے کے لیے coupon codes بنانے دیتے ہیں۔ discount value متعین کریں اور specific users یا groups کو target کریں۔

![Discount Codes فہرست](/img/admin/discount-codes-list.png)

## Broadcasts

**Broadcasts** announcements جاری کرنے یا اپنے subsite users کو private messages بھیجنے کا ایک tool ہے۔ جس plan یا product کو انہوں نے subscribe کیا ہے، اس کی بنیاد پر specific user groups کو target کریں۔

![Broadcasts فہرست](/img/admin/broadcasts-list.png)

## Settings

**Settings** صفحہ وہ جگہ ہے جہاں آپ Ultimate Multisite کو configure کرتے ہیں — registration settings، payments، API اور webhooks، domain mapping، اور دیگر integrations۔

AI connector settings فی الحال supported OAuth provider pools کی فہرست دیتے ہیں: Anthropic Max، OpenAI ChatGPT/Codex، اور Google AI Pro۔ ہر provider card super admins کو accounts connect کرنے، saved accounts refresh کرنے، email کے ذریعے accounts remove کرنے، اور جب sandboxed environment browser redirect کو block کرے تو manual OAuth fallback استعمال کرنے دیتا ہے۔ Cursor Pro setup options کو admin panel سے ہٹا دیا گیا ہے۔

ChatGPT/Codex accounts connector-backed tool use کو support کرتے ہیں جہاں operation tools کی اجازت دیتا ہے، اس لیے connector-supported operations پر depend کرنے والے admin workflows، OpenAI account connect ہونے کے بعد Codex tool behavior استعمال کر سکتے ہیں۔

## Events

**Events** صفحہ آپ کے network میں تمام events اور logs کا record رکھتا ہے۔ یہ plan changes، signups، اور دیگر movements جیسی activities کو track کرتا ہے — آپ کے multisite کی نگرانی کے لیے مفید ہے۔

![Events فہرست](/img/admin/events-list.png)

## Webhooks

**Webhooks** آپ کو data کسی دوسری application تک پہنچانے کی اجازت دیتے ہیں۔ Ultimate Multisite سے Zapier جیسے platforms کو data بھیجنے کے لیے مفید۔

![Webhooks فہرست](/img/admin/webhooks-list.png)
