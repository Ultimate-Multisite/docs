# ڈیش بورڈ کا جائزہ

Ultimate Multisite کا اپنا ایک مخصوص **Dashboard** ہے جو Ultimate Multisite مینو کے تحت ملتا ہے، اور یہ Network Dashboard میں بھی ویجٹس شامل کرتا ہے۔ اس مضمون میں ہم دونوں جگہوں پر موجود چیزوں کا جائزہ لیں گے۔

## Network Dashboard

Ultimate Multisite نیٹ ورک لیول کے ڈیش بورڈ میں کارآمد ویجٹس شامل کرتا ہے جو **My Sites → Network Admin → Dashboard** پر موجود ہے۔

![Ultimate Multisite ویجٹس کے ساتھ Network Dashboard](/img/admin/network-dashboard-full.png)

ہر ویجٹ کیا فراہم کرتا ہے:

**First Steps:** یہ نئی انسٹالیشن پر ظاہر ہوتا ہے اور آپ کو سیٹ اپ کے کام مکمل کرنے کی یاد دہانی کراتا ہے جیسے ادائیگی کا طریقہ ترتیب دینا اور ٹیسٹ اکاؤنٹ بنانا۔

![First Steps ویجٹ](/img/admin/network-dashboard-first-steps.png)

**Summary:** دن کی سائن اپس اور آمدنی کی رپورٹ دکھاتا ہے۔ MRR (Monthly Recurring Revenue) آمدنی پیدا کرنے والی ممبرشپس رکھنے والے صارفین کی متوقع کل رقم ظاہر کرتا ہے۔

![Summary ویجٹ](/img/admin/network-dashboard-summary.png)

**Activity Stream:** Ultimate Multisite کی سرگرمیوں کی رپورٹ دکھاتا ہے — سائن اپس، منسوخیاں، سائٹس کی تخلیق، حذف اور دیگر سرگرمیاں۔

![Activity Stream ویجٹ](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** نیٹ ورک میں صارفین اور سائٹس کا مختصر خلاصہ دکھاتا ہے اور نئی سائٹس یا صارفین بنانے کے شارٹ کٹس فراہم کرتا ہے۔

![Right Now ویجٹ](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** Ultimate Multisite کی تازہ ترین خبریں لاتا اور دکھاتا ہے جن میں اپ ڈیٹس، اہم bugs، اور سیکیورٹی فکسز شامل ہیں۔

![News ویجٹ](/img/admin/network-dashboard-news.png)

## Ultimate Multisite Dashboard

جہاں network dashboard پورے نیٹ ورک کے بارے میں جامع معلومات پیش کرتا ہے، وہیں Ultimate Multisite dashboard (جو Ultimate Multisite مینو کے اوپری حصے میں ہے) سروس کے بارے میں معلومات فراہم کرتا ہے۔

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Monthly Recurring Revenue Growth:** 12 ماہ کی مدت میں آمدنی کی ماہانہ تفصیل دکھاتا ہے۔ نئی آمدنی نیلے رنگ میں جبکہ منسوخیاں سرخ رنگ میں ظاہر ہوتی ہیں۔

![MRR Growth چارٹ](/img/admin/um-dashboard-mrr.png)

**Date-range Support:** ڈیش بورڈ کا ڈیٹا وقت کی مدت کے لحاظ سے فلٹر کریں تاکہ آپ مطلوبہ معلومات پر توجہ دے سکیں۔ تاریخ کی حد نیچے دیے گئے ویجٹس میں دکھائی جانے والی معلومات کو کنٹرول کرتی ہے۔

![تاریخ کی حد اور ڈیش بورڈ کے اعدادوشمار](/img/admin/um-dashboard-date-range.png)

ڈیش بورڈ کے ویجٹس میں شامل ہیں:

- **Revenue** — منتخب مدت کی کل آمدنی MRR اور ہر پروڈکٹ کی تفصیل کے ساتھ
- **New Memberships** — منتخب مدت میں کتنی نئی ممبرشپس بنائی گئیں
- **Signups by Countries** — سائن اپس کی جغرافیائی تقسیم
- **Signups by Forms** — کتنے صارفین نے ہر checkout فارم کے ذریعے سائن اپ کیا بمقابلہ ایڈمنز نے شامل کیے
- **Most Visited Sites** — آپ کے نیٹ ورک کی سب سے زیادہ دیکھی جانے والی سائٹس (اس کے لیے **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** پر وزٹس کی گنتی فعال کرنا ضروری ہے)

![ڈیش بورڈ کے اعدادوشمار](/img/admin/um-dashboard-stats.png)
