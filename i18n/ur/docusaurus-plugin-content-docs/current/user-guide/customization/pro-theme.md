# PRO Theme کے ساتھ اپنے Admin Panel کو حسب ضرورت بنائیں

اکثر جب ہم کلائنٹس کے لیے کام کرتے ہیں تو ہمیں WordPress کے admin dashboard میں انہیں ایک مخصوص تجربہ فراہم کرنے کی ضرورت پڑتی ہے۔ اگر آپ زیادہ پیشہ ورانہ شکل و صورت تلاش کر رہے ہیں تو PRO Theme آپ کی مدد کر سکتا ہے۔

PRO Theme ایک admin theme ہے جو آپ کے کلائنٹس کے Dashboard کو حسب ضرورت بنانے میں مدد کے لیے تیار کیا گیا ہے۔ اور یہ Ultimate Multisite کے premium plugins کا بنیادی حصہ بھی ہے۔

اسے install کرنے کے بعد، اپنے _Plugins_ صفحے پر جائیں اور اپنے Network پر اسے activate کرنے کے لیے کلک کریں۔

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_اپنے Plugins صفحے پر اسے network-wide طور پر activate کریں_

_**نوٹ:**_ _اگر آپ اسے صرف install کرتے ہیں تو یہ آپ کے کلائنٹ کے پاس ایک آپشن کے طور پر دکھائی دے گا جسے منتخب کر کے theme صرف اس صارف کے لیے شامل کیا جا سکتا ہے۔_

اب، آپ کا admin interface مکمل طور پر نئے انداز میں ڈیزائن ہو گیا ہے! آپ کے پاس بائیں کالم میں PRO Theme کے نام سے ایک نیا menu بھی ہے۔
