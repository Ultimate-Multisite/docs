# Ultimate Multisite: Ad Injector کے ذریعے اپنے نیٹ ورک پر اشتہارات لگانا

Ultimate Multisite: Ad Injector ایک **مفت ایڈ آن** ہے جو آپ کو اپنے پلانز کی ترمیمی صفحے سے ہی اشتہارات لگانے کے ٹولز فراہم کرتا ہے۔

Ultimate Multisite: Ad Injector انسٹال کرنے کے بعد، اسے ایکٹیویٹ کرنے کے لیے اپنے _Plugins_ صفحے پر جائیں۔

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Network Activate بٹن پر کلک کریں_

ایڈ آن ایکٹیو ہونے کے بعد، پلان کی ترمیمی سکرین پر _Advanced Options_ میٹا باکس میں _Ad Settings_ نام کا ایک نیا آپشن آ جائے گا۔

![پلان کی ترمیمی سکرین پر Ad Settings](/img/config/product-edit-full.png)

_اب آپ اپنے ہر پلان کے لیے اشتہارات کی سیٹنگز میں ترمیم کر سکتے ہیں_

اس ٹیب میں، آپ فرنٹ اینڈ پر پوسٹس اور پیجز کے لیے مواد سے پہلے اور بعد میں اشتہارات کے کوڈز شامل کر سکتے ہیں۔

آپ اس پلان کے صارفین کے لیے ایڈمن پیجز کے ہیڈر میں بھی اشتہارات شامل کر سکتے ہیں۔
