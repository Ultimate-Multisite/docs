# پلانز کے لیے قابلِ اشتراک لنکس کا انتظام

Ultimate Multisite آپ کو اپنے کلائنٹس کے ساتھ پلان کے لنکس براہ راست شیئر کرنے کی سہولت فراہم کرتا ہے۔ ایسا کرنے کے لیے، اپنے کسی بھی پلان کے ایڈٹ پیج پر جائیں اور _Click to copy the Shareable Link_ بٹن پر کلک کریں۔

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_پلان کے ایڈٹ پیج پر، آپ کو قابلِ اشتراک لنکس بنانے کا بٹن مل جائے گا_

یاد رکھیں کہ یہ طریقہ لنک بنانے کے لیے ڈیفالٹ بلنگ فریکوئنسی استعمال کرتا ہے۔ اس کا مطلب یہ ہے کہ اگر آپ کی ڈیفالٹ بلنگ ماہانہ ہے، تو آخر میں آپ کو کچھ ایسا لنک ملے گا: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

اگر آپ چاہتے ہیں کہ قابلِ اشتراک لنک سہ ماہی کے لیے ڈیفالٹ ہو، تو آپ کو آخر میں 3 لگانا ہوگا

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

_premium_ پلان کو سالانہ شیڈول پر منتخب کرنے کے لیے، آپ کو 1 اور پھر لفظ year لگانا ہوگا:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

ذہن میں رکھیں کہ آپ مختلف بلنگ فریکوئنسیز کے لیے صرف ایک لنک نہیں رکھ سکتے۔ آپ کو اپنی ہر بلنگ فریکوئنسی کے لیے ایک علیحدہ لنک شیئر کرنا ہوگا۔

جب آپ اپنے صارفین کے ساتھ لنک شیئر کریں گے، تو یہ انہیں آپ کے سائن اپ فلو کے پہلے مرحلے پر لے جائے گا۔
