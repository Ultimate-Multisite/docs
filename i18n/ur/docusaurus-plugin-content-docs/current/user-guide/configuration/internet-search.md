# انٹرنیٹ سرچ

Gratis AI Agent v1.5.0 میں **Internet Search** کی صلاحیت شامل کی گئی ہے، جو AI اسسٹنٹ کو گفتگو کے دوران ویب سے تازہ ترین معلومات حاصل کرنے کی اجازت دیتی ہے۔ یہ فیچر [Brave Search API](https://brave.com/search/api/) یا [Tavily API](https://tavily.com/) کے ذریعے کام کرتا ہے۔

## یہ کیسے کام کرتا ہے

جب انٹرنیٹ سرچ فعال ہوتی ہے، تو اسسٹنٹ خود بخود آپ کے سیٹ کردہ سرچ فراہم کنندہ (search provider) سے پوچھ گچھ کرتا ہے جب اسے یہ معلوم ہوتا ہے کہ سوال کے لیے موجودہ یا بیرونی معلومات کی ضرورت ہے — مثال کے طور پر، حالیہ خبریں، براہ راست قیمتیں، یا کوئی دستاویزات جو ماڈل کی تربیت کی تاریخ کے بعد تبدیل ہو سکتی ہوں۔

نتائج کو حقیقی وقت (real time) میں حاصل کیا جاتا ہے اور جواب تیار کرنے سے پہلے اسسٹنٹ کے سیاق و سباق (context) میں شامل کر دیا جاتا ہے۔ اسسٹنٹ اس بات کی نشاندہی کرتا ہے کہ اس نے سوال کا جواب دینے کے لیے سرچ کے نتائج استعمال کیے ہیں۔

## انٹرنیٹ سرچ کو فعال کرنا

انٹرنیٹ سرچ کے لیے آپ کے منتخب کردہ سرچ فراہم کنندہ سے ایک API key کی ضرورت ہوتی ہے۔ اسے ترتیب دینے کے لیے:

1. **Gratis AI Agent → Settings → Advanced** پر جائیں۔
2. **Internet Search Provider** ڈراپ ڈاؤن کو تلاش کریں اور یا تو **Brave Search** یا **Tavily** کا انتخاب کریں۔
3. متعلقہ فیلڈ میں اپنی API key درج کریں۔ ہر فیلڈ کے ساتھ سائن اپ لنکس دکھائے گئے ہیں۔
4. **Save Settings** پر کلک کریں۔

ایک بار جب key محفوظ ہو جاتی ہے، تو انٹرنیٹ سرچ کی صلاحیت خود بخود اسسٹنٹ کے لیے دستیاب ہو جاتی ہے۔

## Brave Search

### Brave Search API Key حاصل کرنا

1. [Brave Search API page](https://brave.com/search/api/) پر جائیں۔
2. ایک پلان کے لیے سائن اپ کریں۔ ایک ماہانہ درخواست کی حد کے ساتھ ایک مفت ٹائر (free tier) دستیاب ہے۔
3. Brave Search developer dashboard سے اپنی API key کاپی کریں۔
4. اسے Gratis AI Agent کی سیٹنگز میں **Brave Search API Key** فیلڈ میں پیسٹ کریں۔

### استعمال کی حدود (Usage Limits)

استعمال کا بل Brave Search کی طرف سے کی گئی پوچھ گچھ کی تعداد کی بنیاد پر کیا جاتا ہے۔ ہر AI جواب جو سرچ کو ٹرگر کرتا ہے، اسے ایک پوچھ گچھ (query) شمار کیا جاتا ہے۔ غیر متوقع چارجز سے بچنے کے لیے اپنے استعمال کی نگرانی [Brave Search developer dashboard](https://brave.com/search/api/) میں کریں۔

## Tavily

Superdav AI Agent v1.10.0 نے **Tavily** کو ایک متبادل انٹرنیٹ سرچ فراہم کنندہ کے طور پر شامل کیا ہے، جو زیادہ بھرپور سرچ نتائج اور ایڈوانس ریسرچ کی صلاحیتیں پیش کرتا ہے۔

### Tavily API Key حاصل کرنا

1. [Tavily API page](https://tavily.com/) پر جائیں۔
2. ایک اکاؤنٹ کے لیے سائن اپ کریں۔ ایک ماہانہ درخواست کی حد کے ساتھ ایک مفت ٹائر (free tier) دستیاب ہے۔
3. Tavily dashboard سے اپنی API key کاپی کریں۔
4. اسے Gratis AI Agent کی سیٹنگز میں **Tavily API Key** فیلڈ میں پیسٹ کریں۔

### استعمال کی حدود (Usage Limits)

استعمال کا بل Tavily کی طرف سے کی گئی API کالز کی تعداد کی بنیاد پر کیا جاتا ہے۔ ہر AI جواب جو سرچ کو ٹرگر کرتا ہے، اسے ایک کال شمار کیا جاتا ہے۔ غیر متوقع چارجز سے بچنے کے لیے اپنے استعمال کی نگرانی [Tavily dashboard](https://tavily.com/) میں کریں۔

## انٹرنیٹ سرچ کو غیر فعال کرنا

API key کو فعال سرچ فراہم کنندہ کے فیلڈ سے ہٹا دیں اور محفوظ کریں۔ انٹرنیٹ سرچ کی صلاحیت اب اسسٹنٹ کو پیش نہیں کی جائے گی۔

:::note
انٹرنیٹ سرچ جوابات میں تاخیر (latency) کا اضافہ کرتی ہے کیونکہ اسسٹنٹ کو جواب دینے سے پہلے سرچ کے نتائج کا انتظار کرنا پڑتا ہے۔ وقت کے لحاظ سے حساس استعمال کے کیسز کے لیے، غور کریں کہ کیا حقیقی وقت کی سرچ ضروری ہے یا کیا اسسٹنٹ کا اپنا موجودہ علم کافی ہے۔
:::
