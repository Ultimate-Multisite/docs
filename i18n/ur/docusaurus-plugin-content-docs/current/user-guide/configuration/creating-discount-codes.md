# ڈسکاؤنٹ کوڈز بنانا (v2)

_**اہم نوٹ: یہ مضمون Ultimate Multisite ورژن 2.x کے لیے ہے۔**_

Ultimate Multisite کے ذریعے آپ اپنے گاہکوں کو ان کی سبسکرپشنز پر رعایت دینے کے لیے ڈسکاؤنٹ کوڈز بنا سکتے ہیں۔ اور انہیں بنانا بالکل آسان ہے!

## ڈسکاؤنٹ کوڈز بنانا اور ترمیم کرنا

ڈسکاؤنٹ کوڈ بنانے یا اس میں ترمیم کرنے کے لیے **Ultimate Multisite > Discount Codes** پر جائیں۔

![](/img/config/discount-codes-list.png)

یہاں آپ کو ان ڈسکاؤنٹ کوڈز کی فہرست نظر آئے گی جو آپ نے پہلے سے بنائے ہیں۔

آپ نیا کوپن بنانے کے لیے **Add Discount** **Code** پر کلک کر سکتے ہیں یا موجودہ کوڈز میں ترمیم کرنے کے لیے ان پر ماؤس لے جائیں اور **Edit** پر کلک کریں۔

![ڈسکاؤنٹ کوڈز کی فہرست ہوور ایکشنز کے ساتھ](/img/config/discount-codes-list.png)

![Add Discount Code بٹن](/img/config/discount-codes-list.png)

آپ کو اس صفحے پر بھیج دیا جائے گا جہاں آپ اپنا کوپن کوڈ بنائیں یا اس میں ترمیم کریں گے۔ اس مثال میں ہم ایک نیا کوڈ بنائیں گے۔

![ڈسکاؤنٹ کوڈ ترمیم کا صفحہ](/img/config/discount-codes-list.png)

آئیں یہاں دستیاب ترتیبات پر نظر ڈالتے ہیں:

**Enter Discount Code:** یہ صرف آپ کے ڈسکاؤنٹ کوڈ کا نام ہے۔ یہ وہ کوڈ نہیں ہے جو آپ کے گاہکوں کو چیک آؤٹ فارم پر استعمال کرنا ہوگا۔

**Description:** یہاں آپ مختصراً بیان کر سکتے ہیں کہ یہ کوپن کس مقصد کے لیے ہے۔

![ڈسکاؤنٹ کوڈ کا نام اور تفصیل کے خانے](/img/config/discount-codes-list.png)

**Coupon code:** یہاں آپ وہ کوڈ متعین کریں گے جو آپ کے گاہکوں کو چیک آؤٹ کے دوران درج کرنا ہوگا۔

![کوپن کوڈ کا خانہ](/img/config/discount-codes-list.png)

**Discount:** یہاں آپ اپنے ڈسکاؤنٹ کوڈ کے لیے **فیصد** یا **مقررہ رقم** میں سے کوئی ایک منتخب کر سکتے ہیں۔

![ڈسکاؤنٹ فیصد یا مقررہ رقم کی ترتیب](/img/config/discount-codes-list.png)

**Apply to renewals:** اگر یہ آپشن بند ہے تو یہ ڈسکاؤنٹ کوڈ صرف **پہلی ادائیگی** پر لاگو ہوگا۔ باقی تمام ادائیگیوں پر کوئی رعایت نہیں ملے گی۔ اگر یہ آپشن آن ہے تو ڈسکاؤنٹ کوڈ تمام آنے والی ادائیگیوں پر لاگو ہوگا۔

**Setup fee discount:** اگر یہ آپشن بند ہے تو کوپن کوڈ **سیٹ اپ فیس پر کوئی رعایت نہیں دے گا**۔ اگر یہ آپشن آن ہے تو آپ وہ رعایت (فیصد یا مقررہ رقم) مقرر کر سکتے ہیں جو یہ کوپن کوڈ آپ کے پلانز کی سیٹ اپ فیس پر لاگو کرے گا۔

![Apply to renewals اور setup fee discount کے آپشنز](/img/config/discount-codes-list.png)

**Active:** اس کوپن کوڈ کو دستی طور پر فعال یا غیر فعال کریں۔

![ڈسکاؤنٹ کوڈ کے لیے Active ٹوگل](/img/config/discount-codes-list.png)

**Advanced Optio** **ns** کے تحت درج ذیل ترتیبات ہیں:

**Limit uses:**

  * **Uses:** یہاں آپ دیکھ سکتے ہیں کہ ڈسکاؤنٹ کوڈ کتنی بار استعمال ہوا۔

  * **Max uses:** اس سے یہ حد مقرر ہوگی کہ صارفین اس ڈسکاؤنٹ کوڈ کو کتنی بار استعمال کر سکتے ہیں۔ مثال کے طور پر، اگر آپ یہاں 10 لکھیں تو کوپن صرف 10 بار استعمال ہو سکے گا۔ اس حد کے بعد کوپن کوڈ مزید استعمال نہیں ہو سکے گا۔

![Limit uses کی ترتیب uses اور max uses کے خانوں کے ساتھ](/img/config/discount-codes-list.png)**Start & expiration dates:** یہاں آپ اپنے کوپن کے لیے شروع ہونے کی تاریخ اور/یا ختم ہونے کی تاریخ مقرر کر سکتے ہیں۔

![شروع اور اختتام کی تاریخ کے خانے](/img/config/discount-codes-list.png)

**Limit products:** اگر آپ **Select products** کو آن کریں تو آپ کی تمام مصنوعات آپ کو دکھائی جائیں گی۔ آپ دستی طور پر منتخب کر سکتے ہیں (آن یا آف کر کے) کہ کون سی مصنوعات اس کوپن کوڈ کو قبول کر سکتی ہیں۔ جو مصنوعات یہاں آف ہیں ان پر اگر آپ کے گاہک یہ کوپن کوڈ استعمال کریں گے تو کوئی تبدیلی نظر نہیں آئے گی۔

![Limit products کے انتخاب کے ٹوگلز](/img/config/discount-codes-list.png)

یہ تمام آپشنز مقرر کرنے کے بعد اپنا کوپن محفوظ کرنے کے لیے **Save Discount Code** پر کلک کریں اور بس ہو گیا!

![Save Discount Code بٹن](/img/config/discount-codes-list.png)

کوپن اب آپ کی فہرست میں ہے اور وہاں سے آپ اسے **ترمیم یا حذف** کر سکتے ہیں۔

![فہرست میں ڈسکاؤنٹ کوڈ ترمیم اور حذف کے آپشنز کے ساتھ](/img/config/discount-codes-list.png)

### URL پیرامیٹرز کا استعمال:

اگر آپ اپنی پرائسنگ ٹیبلز کو حسب ضرورت بنانا چاہتے ہیں یا اپنی ویب سائٹ کے لیے ایک خوبصورت کوپن کوڈ صفحہ بنانا چاہتے ہیں اور چیک آؤٹ فارم پر ڈسکاؤنٹ کوڈ خودکار طور پر لاگو کرنا چاہتے ہیں، تو آپ یہ URL پیرامیٹرز کے ذریعے کر سکتے ہیں۔

سب سے پہلے آپ کو اپنے پلان کا شیئر کرنے والا لنک حاصل کرنا ہوگا۔ اس کے لیے **Ultimate Multisite > Products** پر جائیں اور کوئی پلان منتخب کریں۔

**Click to Copy Shareable Link** بٹن پر کلک کریں۔ اس سے آپ کو اس مخصوص پلان کا شیئر کرنے والا لنک مل جائے گا۔ ہمارے معاملے میں یہ لنک [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/)_ تھا۔_

![شیئر کرنے والے لنک کے بٹن کے ساتھ پروڈکٹ کا صفحہ](/img/config/products-list.png)

اس مخصوص پلان پر اپنا ڈسکاؤنٹ کوڈ لاگو کرنے کے لیے، URL میں بس **?discount_code=XXX** پیرامیٹر شامل کریں۔ جہاں **XXX** کوپن کوڈ ہے۔

ہماری اس مثال میں، ہم اس مخصوص مصنوعات پر کوپن کوڈ **50OFF** لاگو کریں گے۔

اس مخصوص پلان کے لیے 50OFF ڈسکاؤنٹ کوڈ لاگو کرنے کے بعد URL اس طرح نظر آئے گا: [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/) _**?discount_code=50OFF**_۔
