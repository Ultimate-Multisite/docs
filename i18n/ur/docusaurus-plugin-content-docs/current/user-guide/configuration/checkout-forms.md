# چیک آؤٹ فارمز

چیک آؤٹ فارمز نئے گاہکوں کو حاصل کرنے کے لیے مختلف طریقے آزمانے کا ایک آسان اور لچکدار طریقہ ہیں۔

Ultimate Multisite 2.0 میں ایک چیک آؤٹ فارم ایڈیٹر موجود ہے جو آپ کو جتنے چاہیں فارم بنانے کی سہولت دیتا ہے، مختلف فیلڈز، مختلف پروڈکٹس وغیرہ کے ساتھ۔

اس فیچر تک رسائی کے لیے، بائیں سائڈ بار میں Checkout Forms مینو پر جائیں۔

![چیک آؤٹ فارمز کی فہرست](/img/config/checkout-forms-list.png)

اس صفحے پر آپ اپنے تمام چیک آؤٹ فارمز دیکھ سکتے ہیں۔

اگر آپ نیا فارم بنانا چاہتے ہیں، تو صفحے کے اوپر Add Checkout Form پر کلک کریں۔

آپ شروع کرنے کے لیے تین میں سے ایک آپشن منتخب کر سکتے ہیں: single step، multi-step یا blank۔ پھر Go to the Editor پر کلک کریں۔

![چیک آؤٹ فارم ایڈیٹر](/img/config/checkout-form-editor.png)

اس کے علاوہ، آپ اپنے موجودہ فارمز کو ان کے نام کے نیچے موجود آپشنز پر کلک کر کے ایڈٹ یا ڈپلیکیٹ کر سکتے ہیں۔ وہاں آپ کو فارم کا shortcode کاپی کرنے یا فارم کو حذف کرنے کے آپشنز بھی ملیں گے۔

![چیک آؤٹ فارم ہوور ایکشنز](/img/config/checkout-form-hover-actions.png)

### چیک آؤٹ فارم کی تدوین

آپ مختلف مقاصد کے لیے چیک آؤٹ فارمز بنا سکتے ہیں۔ اس مثال میں ہم ایک رجسٹریشن فارم پر کام کریں گے۔

چیک آؤٹ فارم ایڈیٹر میں جانے کے بعد، اپنے فارم کو ایک نام دیں (جو صرف اندرونی حوالے کے لیے استعمال ہوگا) اور ایک slug (جو shortcodes بنانے وغیرہ کے لیے استعمال ہوتا ہے)۔

![چیک آؤٹ فارم ایڈیٹر](/img/config/checkout-form-editor.png)

فارمز steps اور fields پر مشتمل ہوتے ہیں۔ آپ Add New Checkout Step پر کلک کر کے نیا step شامل کر سکتے ہیں۔

![نیا چیک آؤٹ اسٹیپ شامل کریں](/img/config/checkout-form-add-step.png)

ماڈل ونڈو کے پہلے ٹیب میں، اپنے فارم کے step کا مواد بھریں۔ اسے ایک ID، نام اور تفصیل دیں۔ یہ چیزیں زیادہ تر اندرونی استعمال کے لیے ہیں۔

![چیک آؤٹ فارم اسٹیپ](/img/config/checkout-form-step.png)

اس کے بعد، step کی visibility سیٹ کریں۔ آپ "Always show"، "Only show for logged in users" یا "Only show for guests" میں سے کوئی ایک منتخب کر سکتے ہیں۔

![چیک آؤٹ فارم اسٹیپ](/img/config/checkout-form-step.png)

آخر میں، step کا سٹائل کنفیگر کریں۔ یہ اختیاری فیلڈز ہیں۔

![چیک آؤٹ فارم اسٹیپ](/img/config/checkout-form-step.png)

اب ہمارے پہلے step میں فیلڈز شامل کرنے کا وقت ہے۔ بس Add New Field پر کلک کریں اور جس قسم کا سیکشن چاہیں وہ منتخب کریں۔

![فیلڈز کے ساتھ چیک آؤٹ فارم اسٹیپ](/img/config/checkout-form-step.png)

ہر فیلڈ کے مختلف پیرامیٹرز ہوتے ہیں جو بھرنے ہوتے ہیں۔ اس پہلی انٹری کے لیے، ہم "Username" فیلڈ منتخب کریں گے۔

![چیک آؤٹ فارم اسٹیپ](/img/config/checkout-form-step.png)

![چیک آؤٹ فارم اسٹیپ](/img/config/checkout-form-step.png)

![چیک آؤٹ فارم اسٹیپ](/img/config/checkout-form-step.png)

آپ جتنے چاہیں steps اور fields شامل کر سکتے ہیں۔ اپنے پروڈکٹس دکھانے کے لیے تاکہ گاہک کوئی ایک منتخب کر سکیں، Pricing Table فیلڈ استعمال کریں۔ اگر آپ اپنے کلائنٹس کو ٹیمپلیٹ منتخب کرنے دینا چاہتے ہیں، تو Template Selection فیلڈ شامل کریں۔ اور اسی طرح۔

_**نوٹ:** چیک آؤٹ فارم بنانے کے لیے username، email، password، site title، site URL، order summary، payment، اور submit button لازمی فیلڈز ہیں۔_

جب آپ اپنے چیک آؤٹ فارم پر کام کر رہے ہوں، آپ ہمیشہ Preview بٹن استعمال کر کے دیکھ سکتے ہیں کہ آپ کے کلائنٹس کو فارم کیسا نظر آئے گا۔ آپ موجودہ صارف یا وزیٹر کے طور پر دیکھنے کے درمیان بھی تبدیلی کر سکتے ہیں۔

![چیک آؤٹ فارم محفوظ کریں](/img/config/checkout-form-save.png)

![چیک آؤٹ فارم ایڈیٹر](/img/config/checkout-form-editor.png)

آخر میں، Advanced Options میں آپ "Thank You" صفحے کا پیغام کنفیگر کر سکتے ہیں، conversions ٹریک کرنے کے لیے snippets شامل کر سکتے ہیں، اپنے چیک آؤٹ فارم میں custom CSS شامل کر سکتے ہیں یا اسے مخصوص ممالک تک محدود کر سکتے ہیں۔

![ایڈوانسڈ آپشنز](/img/config/checkout-form-advanced.png)

آپ دائیں کالم میں یہ آپشن ٹوگل کر کے اپنے چیک آؤٹ فارم کو دستی طور پر فعال یا غیر فعال بھی کر سکتے ہیں، یا فارم کو مستقل طور پر حذف کر سکتے ہیں۔

![ایکٹیو ٹوگل](/img/config/checkout-form-active.png)

اپنا چیک آؤٹ فارم محفوظ کرنا نہ بھولیں!

![محفوظ کریں بٹن](/img/config/checkout-form-save.png)

اپنے فارم کا shortcode حاصل کرنے کے لیے Generate Shortcode پر کلک کریں اور ماڈل ونڈو میں دکھائے گئے نتیجے کو کاپی کریں۔

![شارٹ کوڈ کے ساتھ محفوظ کریں بٹن](/img/config/checkout-form-save.png)
