# میں اپنی سبسکرپشن کیسے منسوخ کر سکتا/سکتی ہوں؟

اگر آپ کے پاس Ultimate Multisite لائسنس ہے، تو آپ جب چاہیں اس کی تجدید منسوخ کر سکتے ہیں۔ بس نیچے دیے گئے مراحل پر عمل کریں:


<!-- Screenshot unavailable: Account page email with access link -->

خریداری کے وقت فراہم کردہ ای میل اور اپنا پاس ورڈ استعمال کریں۔

<!-- Screenshot unavailable: Login page with email and password fields -->

یہ آپ کے اکاؤنٹ پیج کی ابتدائی سکرین ہے:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

اپنی سبسکرپشن منسوخ کرنے کے لیے، بائیں طرف کے مینو میں _Renewing & Billing_ پر جائیں۔

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

سائیڈ ونڈو کھولنے کے لیے دائیں طرف کے تیر پر کلک کریں۔ پھر، آپ کو وہ آپشن منتخب کرنا ہوگا جو **auto-renew** کو بند کر دے گا۔

سسٹم آپ کو ایک تصدیقی پیغام دکھائے گا۔

اگر آپ واقعی اپنی سبسکرپشن منسوخ کرنا چاہتے ہیں، تو بس _**Cancel Renewals**_ بٹن پر کلک کریں۔ اس کے بعد، آپ سے ایک مختصر سروے کا جواب دینے کو کہا جائے گا۔

ہو گیا! آپ کی سبسکرپشن خود بخود تجدید نہیں ہوگی۔

**آپ کی سبسکرپشن ختم ہونے تک آپ کے پاس ایک درست key رہے گی**۔ اگر آپ اپنی سبسکرپشن دوبارہ فعال کرنا چاہتے ہیں، تو آپ کو یہ دستی طور پر کرنا ہوگا۔
