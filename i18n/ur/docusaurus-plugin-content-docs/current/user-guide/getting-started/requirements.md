# Ultimate Multisite کی ضروریات

Ultimate Multisite خریدنے سے پہلے آپ کے ذہن میں پہلا سوال یہ آ سکتا ہے کہ ہمارا plugin چلانے کے لیے کیا ضروریات ہیں۔ خوشی کی بات یہ ہے کہ Ultimate Multisite چلانے کے لیے صرف چند بنیادی چیزیں درکار ہیں: ایک **ڈومین نام**، ایک اچھا **ہوسٹنگ فراہم کنندہ** اور WordPress **Multisite** انسٹالیشن۔ بس اتنا ہی کافی ہے!

## ڈومین نام

سب سے پہلے، ڈومین نام کے بارے میں بات کرتے ہیں۔

ڈومین نام کا انتخاب کرتے وقت، یہ ذہن میں رکھیں کہ آپ کا مخصوص شعبہ کیا ہے اور آپ Ultimate Multisite کے ساتھ کیا بنانا چاہتے ہیں: کیا آپ ایک ایجنسی بنانا چاہتے ہیں؟ آن لائن دکان جیسی ویب سائٹس بیچنا چاہتے ہیں؟ یا شاید WordPress ویب ہوسٹنگ؟ Ultimate Multisite کے ساتھ سب کچھ ممکن ہے۔

ہمارا plugin سبسکرپشن سروسز، ادائیگی وصولی، چیک آؤٹ فارمز، ڈسکاؤنٹ واؤچرز اور صارفین کے ساتھ رابطے کی سہولیات فراہم کرتا ہے۔ WordPress Multisite کو انسٹال کرنے، ترتیب دینے اور چلانے میں درکار زیادہ تر کام Ultimate Multisite آسان بنا دیتا ہے۔

آپ جو بھی شعبہ منتخب کریں، ہمارے plugin کی مدد سے اسے سنبھالا جا سکتا ہے، بشرطیکہ آپ ویب سائٹس بطور سروس بیچ رہے ہوں! یقینی بنائیں کہ آپ ایسا ڈومین نام منتخب کریں جو آپ کے کاروباری منصوبے سے مطابقت رکھتا ہو۔

## ہوسٹنگ فراہم کنندہ

اب جبکہ آپ کے پاس ایک دلکش ڈومین نام ہے جو آپ کے شعبے اور کاروبار کی نمائندگی کرتا ہے، آپ کو Ultimate Multisite کے ساتھ اپنا نیٹ ورک ہوسٹ کرنے کے لیے ایک اچھی ویب ہوسٹنگ کمپنی کا انتخاب کرنا ہوگا۔ مارکیٹ میں کچھ اچھے اختیارات موجود ہیں۔ کچھ ہوسٹنگ فراہم کنندگان Ultimate Multisite کے ساتھ بہتر انضمام رکھتے ہیں، جو آپ کے منصوبوں کے لحاظ سے فیصلہ کن عنصر ہو سکتا ہے۔

Ultimate Multisite کسی بھی ہوسٹنگ فراہم کنندہ پر کام کرنا چاہیے۔ فرق صرف یہ ہے کہ کچھ ہوسٹنگ کمپنیاں جیسے _WPMU DEV Hosting_، _WP Engine_، _Closte_، _Cloudways_، _Gridpane_ اور _Runcloud_ ڈومین میپنگ کے لیے بہتر انضمام رکھتی ہیں۔ دیگر ہوسٹنگ کے لیے، آپ کو حسب ضرورت ڈومینز دستی طور پر میپ کرنے کی ضرورت ہو سکتی ہے۔ ایسا ہوسٹنگ فراہم کنندہ لینا بھی اچھا خیال ہے جو AutoSSL کی سہولت فراہم کرتا ہو۔ _Closte_، _Runcloud_ اور _cPanel_ اس کی سہولت دیتے ہیں۔

سرور کی تفصیلات کے بارے میں کوئی مخصوص فارمولا نہیں ہے کیونکہ یہ آپ کی پیش کردہ خصوصیات اور فعالیت اور آپ کے نیٹ ورک کے سائز پر منحصر ہوگا۔ شیئرڈ ہوسٹنگ شروع کرنے کے لیے اچھی ہو سکتی ہے، لیکن جیسے ہی آپ کا نیٹ ورک بڑھنے لگے، آپ کو شاید ایک مخصوص VPS کی ضرورت ہوگی۔

## WordPress Multisite

اب، WordPress Multisite انسٹالیشن کے بارے میں تھوڑی بات کرتے ہیں۔

WordPress Multisite آپ کو ایک ہی انسٹالیشن پر سائٹس کا نیٹ ورک رکھنے کی اجازت دیتا ہے۔ یہ ایک بلٹ ان فیچر ہے، لیکن بذریعہ ڈیفالٹ فعال نہیں ہوتا۔ آپ کو اپنی WordPress انسٹالیشن اور ویب ہوسٹنگ کنٹرول پینل میں اس فیچر کو فعال کرنا ہوگا۔

Multisite فعال WordPress رکھنا **Ultimate Multisite کی بنیاد ہے**۔ چونکہ Ultimate Multisite کا بنیادی مقصد صارفین کو ویب سائٹس بطور سروس منظم کرنے اور بیچنے میں مدد کرنا ہے، اس لیے یہ Multisite فعال WordPress پر کرنا ضروری ہے۔

ہمارے پاس **ایک تفصیلی مضمون** ہے جو قدم بہ قدم سکھاتا ہے کہ آپ کی WordPress انسٹالیشن پر Multisite فیچر فعال کرنے کے لیے کیا کرنا ہے۔

WordPress پر Multisite فعال کرنے کے بعد، Ultimate Multisite انسٹال کرنے کا وقت ہے۔ Ultimate Multisite کو درست طریقے سے انسٹال کرنے کے لیے، آپ کے پاس یہ ہونا ضروری ہے:

  * WordPress: v5.3+ (تجویز کردہ: تازہ ترین مستحکم ورژن)

  * PHP: 7.4.x (> 8.0 کی سہولت جلد آ رہی ہے)

  * MySQL: v5+ (تجویز کردہ: 5.6، اگر آپ کے ہوسٹنگ فراہم کنندہ پر 8.0 دستیاب نہ ہو)

Ultimate Multisite کے لیے یہی واحد سافٹ ویئر ضروریات ہیں۔

یاد رکھیں کہ Ultimate Multisite شاید PHP 8.0 کے ساتھ کام کرے لیکن ہم اسے PHP 7.4.x کے ساتھ چلانے کی تجویز دیتے ہیں۔

اس کے علاوہ، ہم اسے **مین ڈومینز** پر انسٹال کرنے کی تجویز دیتے ہیں، سب ڈومینز یا سب ڈائریکٹریز پر نہیں۔ یہ سب ڈومینز/سب ڈائریکٹریز پر کام کر سکتا ہے، لیکن کچھ غلطیاں پیش آ سکتی ہیں۔

Ultimate Multisite انسٹال کرنے کی تفصیلی گائیڈ کے لیے، **یہاں کلک کریں**۔
