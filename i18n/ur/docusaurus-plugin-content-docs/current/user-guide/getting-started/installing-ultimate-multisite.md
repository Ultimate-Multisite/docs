# Ultimate Multisite کی تنصیب

:::note
یہ ٹیوٹوریل فرض کرتا ہے کہ آپ نے پہلے سے WordPress Multisite انسٹال اور کنفیگر کر رکھا ہے۔ یہ سیکھنے کے لیے WP Beginner کا [یہ ٹیوٹوریل](https://www.wpbeginner.com/glossary/multisite/) دیکھیں۔
:::

## Plugin کی تنصیب

Ultimate Multisite مفت میں [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) پر دستیاب ہے۔

اپنے **Network Admin Dashboard** سے **Plugins → Add New Plugin** پر جائیں۔

![Network Admin Add New Plugin صفحہ](/img/installation/add-new-plugin.png)

**"Ultimate Multisite"** تلاش کریں (بالکل درست نتیجے کے لیے quotes کے ساتھ) اور یہ پہلے نتیجے کے طور پر ظاہر ہوگا۔ **Install Now** پر کلک کریں۔

![Ultimate Multisite دکھاتے ہوئے تلاش کے نتائج](/img/installation/search-ultimate-multisite.png)

انسٹال ہونے کے بعد، اپنے پورے نیٹ ورک پر plugin کو فعال کرنے کے لیے **Network Activate** پر کلک کریں۔

![Plugin انسٹال شدہ Network Activate بٹن کے ساتھ](/img/installation/plugin-installed.png)

فعال ہونے کے بعد، آپ خود بخود Setup Wizard پر منتقل ہو جائیں گے۔

![Plugin فعال اور wizard پر منتقل](/img/installation/plugin-activated.png)

## Setup Wizard

Setup Wizard تقریباً 10 منٹ میں Ultimate Multisite کو ترتیب دینے میں آپ کی رہنمائی کرے گا۔

### خوش آمدید

شروع کرنے کے لیے **Get Started** پر کلک کریں۔

![Setup Wizard خوش آمدید اسکرین](/img/installation/wizard-welcome.png)

### تنصیب سے پہلے کی جانچ

یہ مرحلہ آپ کے سسٹم کی معلومات اور WordPress کی تنصیب کو چیک کرتا ہے تاکہ یقینی بنایا جا سکے کہ یہ Ultimate Multisite کی ضروریات پوری کرتا ہے۔ اگر سب کچھ ٹھیک ہے تو **Go to the Next Step** پر کلک کریں۔

![سسٹم کی ضروریات دکھاتی تنصیب سے پہلے کی جانچ](/img/installation/wizard-pre-install-checks.png)

### تنصیب

انسٹالر ضروری ڈیٹابیس ٹیبلز بنائے گا اور `sunrise.php` فائل انسٹال کرے گا جو Ultimate Multisite کے کام کرنے کے لیے ضروری ہے۔ آگے بڑھنے کے لیے **Install** پر کلک کریں۔

![ڈیٹابیس ٹیبلز اور sunrise.php دکھاتا تنصیب کا مرحلہ](/img/installation/wizard-installation.png)

### آپ کی کمپنی

اپنی کمپنی کی معلومات بھریں اور اپنی ڈیفالٹ کرنسی سیٹ کریں۔ یہ معلومات آپ کے پورے WaaS پلیٹ فارم میں استعمال ہوں گی۔ مکمل ہونے پر **Continue** پر کلک کریں۔

![آپ کی کمپنی کی ترتیب کا مرحلہ](/img/installation/wizard-your-company.png)

### ڈیفالٹ مواد

یہ مرحلہ آپ کو پہلے سے بنے ہوئے ٹیمپلیٹس، پروڈکٹس، اور دیگر ابتدائی مواد انسٹال کرنے دیتا ہے۔ یہ Ultimate Multisite کی خصوصیات سے واقف ہونے کا بہترین طریقہ ہے۔ ڈیفالٹ مواد شامل کرنے کے لیے **Install** پر کلک کریں، یا اگر آپ شروع سے شروع کرنا چاہتے ہیں تو اس مرحلے کو چھوڑ دیں۔

![ڈیفالٹ مواد کی تنصیب کا مرحلہ](/img/installation/wizard-default-content.png)

### تجویز کردہ Plugins

اختیاری طور پر تجویز کردہ معاون plugins انسٹال کریں۔ انہیں شامل کرنے کے لیے **Install** پر کلک کریں یا جاری رکھنے کے لیے چھوڑ دیں۔

![تجویز کردہ plugins کا مرحلہ](/img/installation/wizard-recommended-plugins.png)

### تیار!

بس! آپ کی Ultimate Multisite کی تنصیب مکمل ہو گئی۔ اب آپ **Network Admin Dashboard** سے اپنا Website as a Service پلیٹ فارم بنانا شروع کر سکتے ہیں۔

![سیٹ اپ مکمل - تیار اسکرین](/img/installation/wizard-ready.png)

![Ultimate Multisite فعال کے ساتھ Network Admin Dashboard](/img/installation/network-dashboard.png)

آگے بڑھیں اور مزے کریں!
