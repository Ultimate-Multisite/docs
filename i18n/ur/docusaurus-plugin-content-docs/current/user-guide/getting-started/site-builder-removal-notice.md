# سائٹ بلڈر موڈ کی ہٹانے کی اطلاع

**Superdav AI Agent v1.12.0 میں سائٹ بلڈر موڈ (Site Builder mode) کو ہٹا دیا گیا ہے۔** اگر آپ سائٹ بلڈر موڈ استعمال کر رہے تھے، تو تھیم بنانے اور سائٹ کی سیٹ اپ کے لیے آپ کو **سیٹ اپ اسسٹنٹ ایجنٹ (Setup Assistant agent)** پر منتقل (migrate) ہونا چاہیے۔

## کیا ہوا؟ {#what-happened}

### سائٹ بلڈر موڈ (قدیم) {#site-builder-mode-legacy}

سائٹ بلڈر موڈ ایک ویزرڈ پر مبنی انٹرفیس تھا جو مندرجہ ذیل کاموں کے لیے استعمال ہوتا تھا:

- ٹیمپلیٹس سے سائٹس بنانا
- بنیادی سیٹنگز کی ترتیب دینا
- تھیم کا انتخاب کرنا
- ابتدائی مواد (initial content) سیٹ اپ کرنا

### اس کی جگہ کیا آیا؟ {#what-replaced-it}

اب **سیٹ اپ اسسٹنٹ ایجنٹ** تمام سائٹ بلڈر کی فعالیت کو سنبھالتا ہے، جس میں یہ خصوصیات شامل ہیں:

- زیادہ لچکدار، ایجنٹ کی رہنمائی میں سیٹ اپ
- تھیم کی بہتر کسٹمائزیشن کے آپشنز
- تھیم بلڈر آن بورڈنگ کے ساتھ انٹیگریشن
- مستقبل کے سیشنز کے لیے مستقل `site_brief` میموری

## اگر آپ سائٹ بلڈر موڈ استعمال کر رہے تھے {#if-you-were-using-site-builder-mode}

### آپ کی سائٹس محفوظ ہیں {#your-sites-are-safe}

- سائٹ بلڈر موڈ سے بننے والی موجودہ سائٹس کام کرتی رہیں گی
- کوئی ڈیٹا کا نقصان یا سائٹ میں خلل نہیں پڑے گا
- آپ اپنی سائٹس کو معمول کے مطابق مینج کرنا جاری رکھ سکتے ہیں

### سیٹ اپ اسسٹنٹ ایجنٹ پر منتقل ہوں {#migrate-to-setup-assistant-agent}

نئی سائٹ سیٹ اپ یا تھیم میں تبدیلی کے لیے، سیٹ اپ اسسٹنٹ ایجنٹ استعمال کریں:

```
"Help me set up a new site"
```

یا

```
"Start the Theme Builder onboarding"
```

سیٹ اپ اسسٹنٹ ایجنٹ زیادہ لچک کے ساتھ وہی فعالیت فراہم کرتا ہے۔

## موازنہ: سائٹ بلڈر بمقابلہ سیٹ اپ اسسٹنٹ {#comparison-site-builder-vs-setup-assistant}

| خصوصیت | سائٹ بلڈر (ہٹایا گیا) | سیٹ اپ اسسٹنٹ (نیا) |
|---------|----------------------|----------------------|
| سیٹ اپ کا طریقہ | ویزرڈ فارم | ایجنٹ کی گفتگو (Agent conversation) |
| تھیم کا انتخاب | پہلے سے طے شدہ ٹیمپلیٹس | کسٹم جنریشن |
| کسٹمائزیشن | محدود آپشنز | مکمل ڈیزائن سسٹم |
| سائٹ کا خلاصہ (Site brief) | محفوظ نہیں ہوتا | مستقل میموری |
| مستقبل کے سیشنز | سیٹ اپ دہرانا پڑتا ہے | محفوظ `site_brief` استعمال ہوتا ہے |
| لچک | مقررہ ورک فلو | موافق گفتگو (Adaptive conversation) |

## سیٹ اپ اسسٹنٹ ایجنٹ پر منتقل ہونا {#migrating-to-setup-assistant-agent}

### نئی سائٹس کے لیے {#for-new-sites}

سائٹ بلڈر موڈ استعمال کرنے کے بجائے:

1. درخواست کریں: "Help me set up a new site"
2. سیٹ اپ اسسٹنٹ ایجنٹ آپ کی رہنمائی کرے گا:
   - سائٹ کا مقصد اور اہداف
   - ہدف کا سامعین (Target audience)
   - برانڈ کی شناخت (Brand identity)
   - تھیم کی جنریشن
   - ابتدائی ترتیب (Initial configuration)

### موجودہ سائٹس کے لیے {#for-existing-sites}

اگر آپ کے پاس سائٹ بلڈر موڈ سے کوئی موجودہ سائٹ ہے:

1. آپ اسے جیسا ہے ویسا استعمال کرتے رہ سکتے ہیں
2. تھیم کو اپ ڈیٹ کرنے کے لیے، درخواست کریں: "Redesign my site"
3. سیٹ اپ اسسٹنٹ ایجنٹ آپ کو ایک نیا تھیم بنانے میں مدد کرے گا
4. آپ کے سائٹ کے ڈیٹا میں کوئی تبدیلی نہیں آئے گی

### تھیم میں تبدیلی کے لیے {#for-theme-changes}

سائٹ بلڈر موڈ کے تھیم کے انتخاب کے بجائے:

1. درخواست کریں: "Change my theme"
2. سیٹ اپ اسسٹنٹ ایجنٹ:
   - آپ کی ڈیزائن کی ترجیحات کے بارے میں پوچھے گا
   - ایک کسٹم تھیم تیار کرے گا
   - اسے آپ کی سائٹ پر فعال (Activate) کرے گا

## اہم فرق {#key-differences}

### سائٹ بلڈر موڈ {#site-builder-mode}

```
1. ایک ٹیمپلیٹ منتخب کریں
2. ایک تھیم منتخب کریں
3. بنیادی سیٹنگز ترتیب دیں
4. مکمل
```

### سیٹ اپ اسسٹنٹ ایجنٹ {#setup-assistant-agent}

```
1. اپنی سائٹ کا مقصد بیان کریں
2. اپنے ہدف کے سامعین کی وضاحت کریں
3. ڈیزائن کی ترجیحات کا انتخاب کریں
4. ایجنٹ کسٹم تھیم تیار کرتا ہے
5. ایجنٹ تھیم فعال کرتا ہے
6. مستقبل کے سیشنز کے لیے سائٹ کا خلاصہ محفوظ ہوتا ہے
```

## سیٹ اپ اسسٹنٹ ایجنٹ کے فوائد {#benefits-of-setup-assistant-agent}

### زیادہ لچکدار {#more-flexible}

- اپنی سائٹ کو عام زبان میں بیان کریں
- کسٹم سفارشات حاصل کریں
- آپ کی مخصوص ضروریات کے مطابق ڈھلتا ہے

### بہتر کسٹمائزیشن {#better-customization}

- کسٹم تھیم جنریشن
- ڈیزائن سسٹم کے فیصلے
- مستقل ڈیزائن ٹوکنز (Persistent design tokens)

### مستقل میموری {#persistent-memory}

- آپ کا `site_brief` محفوظ ہوتا ہے
- مستقبل کے ایجنٹ آپ کی سائٹ کو سمجھیں گے
- سیٹ اپ کی معلومات دہرانے کی ضرورت نہیں

### مربوط ورک فلو {#integrated-workflow}

- تھیم بلڈر آن بورڈنگ
- ڈیزائن سسٹم ایسٹھیٹکس سکل (Design System Aesthetics skill)
- قابلِ ظاہریات کنٹرولز کی صلاحیت (Ability Visibility controls)
- سب کچھ بغیر کسی رکاوٹ کے مل جل کر کام کرتا ہے

## ٹرابل شوٹنگ {#troubleshooting}

### مجھے سائٹ بلڈر موڈ نہیں مل رہا {#i-cant-find-site-builder-mode}

سائٹ بلڈر موڈ کو ہٹا دیا گیا ہے۔ اس کے بجائے سیٹ اپ اسسٹنٹ ایجنٹ استعمال کریں۔:

```
"Help me set up a new site"
```

### میں سائٹ بلڈر سے سائٹ دوبارہ بنانا چاہتا ہوں {#i-want-to-recreate-a-site-from-site-builder}

آپ اسے سیٹ اپ اسسٹنٹ ایجنٹ کے ساتھ دوبارہ بنا سکتے ہیں:

1. درخواست کریں: "Help me set up a new site"
2. اپنی اصل سائٹ کا مقصد اور ڈیزائن بیان کریں
3. ایجنٹ ایک ملتی جلتی تھیم تیار کرے گا
4. آپ کا `site_brief` مستقبل کے حوالہ کے لیے محفوظ ہو جائے گا

### میری موجودہ سائٹ بلڈر سائٹ کام نہیں کر رہی {#my-existing-site-builder-site-isnt-working}

سائٹ بلڈر موڈ سے بننے والی موجودہ سائٹس کام کرتی رہیں گی۔ اگر آپ کو کوئی مسئلہ آ رہا ہے:

1. چیک کریں کہ آپ کی تھیم ابھی بھی فعال ہے
2. تصدیق کریں کہ آپ کے پلگ انز (plugins) فعال ہیں
3. ورڈپریس ایرر لاگز (WordPress error logs) چیک کریں
4. اگر مسائل برقرار رہیں تو سپورٹ سے رابطہ کریں

### کیا میں اپنے پرانے سائٹ بلڈر ٹیمپلیٹس استعمال کر سکتا ہوں؟ {#can-i-still-use-my-old-site-builder-templates}

سائٹ بلڈر ٹیمپلیٹس اب دستیاب نہیں ہیں۔ تاہم:

- آپ کی موجودہ سائٹس کام کرتی رہیں گی
- آپ سیٹ اپ اسسٹنٹ ایجنٹ کے ساتھ ملتی جلتی سائٹس دوبارہ بنا سکتے ہیں
- سیٹ اپ اسسٹنٹ ایجنٹ زیادہ کسٹمائزیشن کے آپشنز فراہم کرتا ہے

## اگلے اقدامات {#next-steps}

1. **نئی سائٹس کے لیے**: سیٹ اپ اسسٹنٹ ایجنٹ استعمال کریں
2. **موجودہ سائٹس کے لیے**: انہیں جیسا ہے ویسا استعمال کرتے رہیں
3. **تھیم میں تبدیلی کے لیے**: سیٹ اپ اسسٹنٹ ایجنٹ سے مدد مانگیں
4. **ڈیزائن کو بہتر بنانے کے لیے**: ڈیزائن سسٹم ایسٹھیٹکس سکل استعمال کریں

- **Site Specification Skill**: اپنی سائٹ کے اہداف اور سامعین کی وضاحت کرنا
