# اکثر پوچھے جانے والے سوالات

یہاں کچھ **اکثر پوچھے جانے والے سوالات** ہیں جو عام طور پر سپورٹ کی درخواستوں میں آتے ہیں۔

  * **Ultimate Multisite کیا ہے؟**

Ultimate Multisite ایک WordPress plugin ہے، جو WordPress multisite installs کے لیے بنایا گیا ہے۔ یہ آپ کے WordPress install کو سائٹس کے ایک پریمیم نیٹ ورک میں تبدیل کر دیتا ہے – جیسے [WordPress.com](https://WordPress.com) – جو صارفین کو ماہانہ، سہ ماہی، یا سالانہ فیس کے ذریعے سائٹس بنانے کی اجازت دیتا ہے (آپ مفت پلانز بھی بنا سکتے ہیں)۔

  * **کیا Ultimate Multisite بغیر Multisite کے کام کرے گا؟**

نہیں۔ Ultimate Multisite کے لیے WordPress multisite، subdirectory، یا subdomain ضروری ہے۔

  * **Ultimate Multisite کے لیے کیا تقاضے ہیں؟**

Ultimate Multisite کے لیے درکار ہے:

**WordPress Version** : v5.3+ (تجویز کردہ: آخری مستحکم ورژن)  
**PHP Version** : 7.4.x (8.0 سے اوپر کے لیے سپورٹ جلد آ رہی ہے)  
**MySQL Version** : v5+ (تجویز کردہ: ورژن 5.6، اگر آپ کے hosting provider پر 8.0 دستیاب نہیں ہے)

_Multisite (subdomain یا subdirectory) کو بھی فعال کرنا ضروری ہے۔_ [WordPress Multisite کیسے انسٹال کریں](/user-guide/getting-started/how-to-install-wordpress-multisite)۔

  * **کیا Ultimate Multisite Subdirectories والے Multisite کے ساتھ کام کرے گا؟**

ہاں۔ Ultimate Multisite subdirectory installs اور subdirectory installs دونوں کو سپورٹ کرتا ہے۔

  * **کیا Ultimate Multisite Subdomains والے Multisite کے ساتھ کام کرے گا؟**

ہاں۔ Ultimate Multisite subdirectory installs اور subdomain installs دونوں کو سپورٹ کرتا ہے۔

  * **کیا میں Ultimate Multisite کو صارفین کے لیے Plugins اور Themes کو محدود کرنے کے لیے استعمال کر سکتا ہوں؟**

ہاں۔ Ultimate Multisite کے ساتھ آپ منتخب کر سکتے ہیں کہ آپ کے نیٹ ورک پر ہر پلان کے لیے کون سے plugins اور themes دستیاب ہوں۔

  * **کیا میں Ultimate Multisite کو صارف کی اپنی سائٹ پر پوسٹس کی تعداد محدود کرنے کے لیے استعمال کر سکتا ہوں؟**

ہاں۔ Ultimate Multisite کے ساتھ آپ اپنے نیٹ ورک پر **ہر دستیاب post type** کے لیے پوسٹ کوٹہ مقرر کر سکتے ہیں۔ آپ ان صارفین کی تعداد بھی محدود کر سکتے ہیں جنہیں ایڈمن مدعو کر سکتا ہے، **ڈسک سپیس کی مقدار،** اور بہت کچھ۔

  * **کیا Ultimate Multisite Site Templates کو سپورٹ کرتا ہے؟**

ہاں۔ Ultimate Multisite کے ساتھ آپ جتنے چاہیں Site Templates بنا سکتے ہیں۔ Site Templates themes نہیں ہیں، یہ آپ کے نیٹ ورک پر عام سائٹس ہیں۔ آپ plugins، themes فعال کر سکتے ہیں، اور اس Site Template کو جیسے چاہیں customize کر سکتے ہیں۔ اگر کوئی صارف وہ template منتخب کرتا ہے، تو اس کی سائٹ اس template کی کاپی کے طور پر بنائی جائے گی، تمام customizations کے ساتھ۔

آپ کسی خاص template کو کسی پلان کے لیے بھی سیٹ کر سکتے ہیں۔ ایسا کرنے سے، اس پلان پر سائن اپ کرنے والے ہر صارف کی سائٹ اس Site Template کی نقل کے طور پر بنائی جائے گی۔

  * **کیا Ultimate Multisite WooCommerce کے ساتھ integrate ہوتا ہے؟**

ہاں!

اگر آپ چاہتے ہیں کہ آپ کے نیٹ ورک کے صارفین کے لیے WooCommerce دستیاب ہو، جس سے وہ اسٹورز سیٹ اپ کر سکیں، تو یہ Ultimate Multisite کے ساتھ ممکن ہے۔ آپ ہر پلان ٹیئر پر اپنے صارفین کی بنائی جانے والی پروڈکٹس کی تعداد بھی محدود کر سکتے ہیں۔

Ultimate Multisite کے لیے سبسکرپشن پیمنٹس کے لحاظ سے، آپ [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) استعمال کر سکتے ہیں تاکہ آپ کے Ultimate Multisite صارفین آپ کی main site پر WooCommerce gateways کے ذریعے ادائیگی کر سکیں۔ اس طرح، آپ WooCommerce کے لیے بنائے گئے payment gateways کی بڑی لائبریری سے فائدہ اٹھا سکتے ہیں اور انہیں اپنے Ultimate Multisite صارفین کو بل کرنے کے لیے استعمال کر سکتے ہیں۔

  * **کیا WPMUDEV Pro Sites سے سبسکرپشنز منتقل کرنے کا کوئی طریقہ ہے؟**

ہاں، Ultimate Multisite اب آپ کو اپنے Pro Sites نیٹ ورک کو منتقل کرنے میں مدد کے لیے [ایک مددگار ٹول](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) فراہم کرتا ہے۔

  * **میرے Plan Edit Page پر "Allowed Plugins" ٹیب خالی کیوں ہے؟**

Ultimate Multisite آپ کو صرف ان plugins تک رسائی کی اجازت یا انکار دینے دیتا ہے جو آپ کے نیٹ ورک پر network-activated نہیں ہیں۔ جب کوئی plugin network-activated ہوتا ہے، تو اپنی فطرت کے مطابق، یہ آپ کے نیٹ ورک کی ہر سائٹ پر بطور ڈیفالٹ لوڈ ہو گا۔

اگر آپ کو "Allowed Plugins" ٹیب پر کوئی plugins نظر نہیں آ رہے، تو چیک کریں کہ مطلوبہ plugin network-active ہے یا نہیں۔

  * **کیا میں اپنا خود کا Pricing Table ڈیزائن استعمال کر سکتا ہوں؟**

ہاں۔ ورژن 1.4.0 سے، Ultimate Multisite template overrides کو سپورٹ کرتا ہے۔

  * **کیا Ultimate Multisite WP Engine کے ساتھ کام کرتا ہے؟**

ہاں۔ Ultimate Multisite ہمیشہ سے WP Engine کے ساتھ سپورٹ تھا، لیکن domain mapping کو کام کرنے کے لیے ایڈمن کی طرف سے دستی اضافی اقدامات کی ضرورت تھی (WP Engine پینل میں domain شامل کرنا)۔ **ورژن 1.6.0 سے یہ اب ضروری نہیں رہا۔** Ultimate Multisite براہ راست WP Engine API کے ساتھ integrate ہوتا ہے اور mapping کے بعد domains خودکار طور پر پینل میں شامل ہو جاتے ہیں۔

  * **آپ کون سا hosting provider تجویز کرتے ہیں؟**

Ultimate Multisite کسی بھی hosting provider کے ساتھ کام کرنا چاہیے۔ فرق صرف یہ ہے کہ کچھ hosting کمپنیاں جیسے Closte، Serverpilot، WPMUDEV، WP Engine، Cloudways، Gridpane اور [Runcloud.io](http://Runcloud.io) کے پاس domain mapping کی صلاحیتوں کے لیے گہرا integration ہے تو یہ فوری طور پر کام کرتی ہیں۔ دوسری hostings کے لیے، آپ کو domain mapping دستی طور پر سیٹ اپ کرنا پڑ سکتا ہے۔
