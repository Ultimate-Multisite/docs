# ای میلز اور براڈکاسٹس بھیجنا (v2)

_**اہم نوٹ: یہ مضمون Ultimate Multisite ورژن 2.x کے بارے میں ہے۔**_

Ultimate Multisite میں ایک ایسی خصوصیت موجود ہے جو آپ کو اپنے صارفین سے رابطہ کرنے کی سہولت دیتی ہے۔ آپ کسی مخصوص صارف یا صارفین کے گروپ کو ای میل بھیج سکتے ہیں، اور ساتھ ہی ان کے admin dashboard پر اعلانات نشر کرنے کے لیے نوٹسز بھی بھیج سکتے ہیں۔

## براڈکاسٹس کے ذریعے اپنے صارفین کے dashboard پر admin نوٹسز شامل کریں

Ultimate Multisite کی broadcast خصوصیت استعمال کرتے ہوئے، آپ اپنے صارفین کی subsite کے admin dashboard پر **admin نوٹسز** شامل کر سکتے ہیں۔

یہ خصوصیت اس وقت بہت کارآمد ہے جب آپ کو کوئی اعلان کرنا ہو، جیسے سسٹم کی دیکھ بھال یا اپنے موجودہ صارفین کو نئی مصنوعات یا خدمات پیش کرنا۔ آپ کے صارف کے dashboard پر admin نوٹس کچھ اس طرح نظر آئے گا۔

![صارف کے dashboard پر دکھایا گیا Admin notice broadcast](/img/admin/broadcasts-list.png)

admin نوٹس شروع کرنے کے لیے، اپنے network admin dashboard پر جائیں اور **Ultimate Multisite** مینو کے نیچے، آپ کو **Broadcasts** کا آپشن ملے گا۔

![Ultimate Multisite admin میں Broadcasts مینو](/img/admin/broadcasts-list.png)

اس صفحے سے، اوپر **Add Broadcast** بٹن پر کلک کریں۔

اس سے Add broadcast کی modal ونڈو کھلے گی جہاں آپ یہ منتخب کر سکتے ہیں کہ آپ کس قسم کا broadcast بھیجنا چاہتے ہیں۔

آگے بڑھیں اور **Message** منتخب کریں پھر **Next Step** بٹن پر کلک کریں۔

![Message type منتخب کے ساتھ Add broadcast modal](/img/admin/broadcasts-list.png)

اگلی ونڈو آپ سے **Target customer** یا **Target product** کے بارے میں پوچھے گی۔ یاد رہے کہ آپ ایک سے زیادہ صارفین یا ایک سے زیادہ مصنوعات منتخب کر سکتے ہیں۔

کسی صارف اکاؤنٹ یا مصنوعات کی تلاش کے لیے آپ کو فیلڈ میں keyword ٹائپ کرنا شروع کرنا ہوگا۔

**Message type** فیلڈ میں، آپ نوٹس کا رنگ منتخب کر سکتے ہیں۔ اس سے آپ کے پیغام کی فوری ضرورت پر زور دیا جائے گا۔

پھر آپ **Next Step** پر کلک کر سکتے ہیں۔

![براڈکاسٹ کے لیے ہدف صارف اور مصنوعات کا انتخاب](/img/admin/broadcasts-list.png)

اگلی ونڈو میں آپ موضوع اور وہ مواد/پیغام درج کر کے اپنا پیغام لکھنا شروع کر سکتے ہیں جو آپ صارفین کو براڈکاسٹ کرنا چاہتے ہیں۔

![براڈکاسٹ پیغام کا موضوع اور مواد ایڈیٹر](/img/admin/broadcasts-list.png)

اپنا پیغام تیار کرنے کے بعد، آپ **Send** بٹن دبا سکتے ہیں۔

بس اتنا ہی ہے۔ admin نوٹس فوری طور پر آپ کے صارف کے dashboard پر ظاہر ہو جائے گا۔

## اپنے صارفین کو ای میلز بھیجیں

Ultimate Multisite کی broadcast خصوصیت استعمال کرتے ہوئے، آپ اپنے صارفین کو ای میل بھیج سکتے ہیں۔ آپ کے پاس یہ اختیار ہے کہ ای میل صرف مخصوص صارفین کو بھیجیں یا ان صارفین کے گروپ کو ہدف بنائیں جو کسی خاص مصنوعات یا پلان کے سبسکرائبر ہیں۔

ای میل براڈکاسٹ شروع کرنے کے لیے، اپنے network admin dashboard پر جائیں اور Ultimate Multisite مینو کے نیچے، آپ کو Broadcast کا آپشن ملے گا۔

![Ultimate Multisite admin میں Broadcasts صفحہ](/img/admin/broadcasts-list.png)

اس صفحے سے، اوپر **Add broadcast** بٹن پر کلک کریں۔

اس سے Add broadcast کی modal ونڈو کھلے گی جہاں آپ یہ منتخب کر سکتے ہیں کہ آپ کس قسم کا broadcast بھیجنا چاہتے ہیں۔ آگے بڑھیں اور **Email** منتخب کریں پھر **Next Step** بٹن پر کلک کریں۔

![Email type منتخب کے ساتھ Add broadcast modal](/img/admin/broadcasts-list.png)

اگلی ونڈو آپ سے **Target customer** یا **Target product** کے بارے میں پوچھے گی۔ یاد رہے کہ آپ ایک سے زیادہ صارفین یا ایک سے زیادہ مصنوعات منتخب کر سکتے ہیں۔

کسی صارف اکاؤنٹ یا مصنوعات کی تلاش کے لیے آپ کو فیلڈ میں keyword ٹائپ کرنا شروع کرنا ہوگا۔

اپنی ہدف آڈینس منتخب کرنے کے بعد، آپ **Next Step** پر کلک کر سکتے ہیں۔

![ای میل براڈکاسٹ کے لیے ہدف صارف اور مصنوعات کا انتخاب](/img/admin/broadcasts-list.png)

اگلی ونڈو میں آپ موضوع اور وہ مواد/پیغام درج کر کے اپنی ای میل لکھنا شروع کر سکتے ہیں جو آپ صارفین کو بھیجنا چاہتے ہیں۔

![ای میل براڈکاسٹ کا موضوع اور مواد ایڈیٹر](/img/admin/broadcasts-list.png)

اپنا پیغام تیار کرنے کے بعد، آپ **Send** بٹن دبا سکتے ہیں۔

اور broadcast خصوصیت استعمال کرتے ہوئے اپنے صارفین کو ای میل بھیجنا اتنا آسان ہے۔

## سسٹم ای میلز

Ultimate Multisite میں سسٹم ای میلز وہ **خودکار اطلاعات** ہیں جو سسٹم کی طرف سے مخصوص کارروائیوں کے بعد بھیجی جاتی ہیں جیسے رجسٹریشن، ادائیگی، domain mapping وغیرہ۔ ان ای میلز کو Ultimate Multisite کی ترتیبات سے ایڈٹ یا تبدیل کیا جا سکتا ہے۔ اس میں ایک خصوصیت بھی ہے جو آپ کو کسی دوسری Ultimate Multisite انسٹالیشن سے موجودہ ترتیبات ری سیٹ اور درآمد کرنے کی اجازت دیتی ہے۔

### ری سیٹ کرنا اور درآمد کرنا

نئے Ultimate Multisite ورژنز، اور ساتھ ہی add-ons، وقتاً فوقتاً نئی ای میلز رجسٹر کر سکتے ہیں اور کریں گے۔

تنازعات اور دیگر مسائل سے بچنے کے لیے، **ہم نئے ای میل ٹیمپلیٹس کو آپ کی انسٹالیشن پر سسٹم ای میلز کے طور پر خود بخود شامل نہیں کریں گے**، جب تک کہ وہ کسی خاص خصوصیت کے درست کام کرنے کے لیے انتہائی ضروری نہ ہوں۔

تاہم، super admins اور agents ان نئی رجسٹرڈ ای میلز کو importer ٹول کے ذریعے درآمد کر سکتے ہیں۔ یہ عمل نئے ای میل ٹیمپلیٹ کے مواد اور ترتیب کے ساتھ ایک نئی سسٹم ای میل بنائے گا، جس سے super admin کو اجازت ملے گی کہ وہ جو تبدیلیاں چاہیں کریں یا انہیں جوں کا توں رکھیں۔

#### سسٹم ای میلز کیسے درآمد کریں

اپنے Ultimate Multisite Settings صفحے پر جائیں اور **Emails** ٹیب پر جائیں۔

![Ultimate Multisite settings میں Emails ٹیب](/img/config/settings-emails.png)

پھر، سائیڈبار پر، **Customize System Emails** بٹن پر کلک کریں۔

![سائیڈبار پر Customize System Emails بٹن](/img/config/settings-emails.png)

System Emails صفحے پر، آپ کو اوپر **Reset & Import** ایکشن بٹن نظر آئے گا۔ اس بٹن پر کلک کرنے سے import اور reset کی modal ونڈو کھلے گی۔

![System Emails صفحے پر Reset and Import ایکشن بٹن](/img/config/settings-emails.png)

پھر، آپ Import Emails آپشنز کو ٹوگل کر کے دیکھ سکتے ہیں کہ کون سی سسٹم ای میلز درآمد کے لیے دستیاب ہیں۔

![دستیاب سسٹم ای میلز دکھاتے Import Emails آپشنز](/img/config/settings-emails.png)

#### سسٹم ای میلز ری سیٹ کرنا

کبھی کبھی، آپ کو احساس ہوگا کہ کسی ای میل ٹیمپلیٹ میں آپ نے جو تبدیلیاں کی ہیں وہ اب آپ کے لیے کام نہیں کر رہیں اور آپ اسے **ڈیفالٹ حالت** میں واپس لانا چاہیں گے۔

ایسے حالات میں، آپ کے پاس دو اختیارات ہیں: آپ آسانی سے سسٹم ای میل حذف کر کے اسے واپس درآمد کر سکتے ہیں (اوپر دی گئی ہدایات استعمال کرتے ہوئے) - لیکن یہ send metrics اور دیگر چیزیں مٹا دے گا، جو اس طریقے کو کم پسندیدہ بناتا ہے۔

یا آپ اس ای میل ٹیمپلیٹ کو ری سیٹ کرنے کے لیے **Reset & Import tool** استعمال کر سکتے ہیں۔

کسی ای میل ٹیمپلیٹ کو ری سیٹ کرنے کے لیے، آپ اوپر دیے گئے مراحل کی پیروی کر سکتے ہیں جب تک آپ Reset & Import tool تک نہ پہنچ جائیں، اور پھر، **Reset** آپشن کو ٹوگل کریں اور وہ ای میلز منتخب کریں جنہیں آپ ان کے ڈیفالٹ مواد پر واپس لانا چاہتے ہیں۔

![ای میل ٹیمپلیٹس کو ڈیفالٹس پر بحال کرنے کے لیے Reset آپشن](/img/config/settings-emails.png)
