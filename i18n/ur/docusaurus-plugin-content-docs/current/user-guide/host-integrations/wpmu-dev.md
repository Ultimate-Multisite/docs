# WPMU DEV انٹیگریشن

## جائزہ
WPMU DEV ایک مکمل WordPress پلیٹ فارم ہے جو WordPress سائٹس کے لیے ہوسٹنگ، پلگ انز، اور خدمات فراہم کرتا ہے۔ یہ انٹیگریشن Ultimate Multisite اور WPMU DEV ہوسٹنگ کے درمیان خودکار ڈومین سنکنگ اور SSL سرٹیفکیٹ مینجمنٹ کو ممکن بناتا ہے۔

## خصوصیات
- خودکار ڈومین سنکنگ
- SSL سرٹیفکیٹ مینجمنٹ
- SSL سرٹیفکیٹ کی توسیعی تصدیقی کوششیں

## ضروریات
یہ انٹیگریشن خود بخود پتہ لگا لیتا ہے کہ آیا آپ WPMU DEV پر ہوسٹ کر رہے ہیں اور بلٹ ان API استعمال کرتا ہے۔ اگر آپ WPMU DEV پر ہوسٹنگ کر رہے ہیں تو کسی اضافی ترتیب کی ضرورت نہیں۔

یہ انٹیگریشن `WPMUDEV_HOSTING_SITE_ID` constant کی موجودگی چیک کرتا ہے، جو WPMU DEV پر ہوسٹنگ کرتے وقت خود بخود متعین ہو جاتا ہے۔

## سیٹ اپ ہدایات

### 1. WPMU DEV ہوسٹنگ کی تصدیق

اگر آپ WPMU DEV پر ہوسٹنگ کر رہے ہیں، تو ضروری constants پہلے سے متعین ہونے چاہیئیں۔ تصدیق کریں کہ:

1. `WPMUDEV_HOSTING_SITE_ID` constant آپ کے ماحول میں متعین ہے
2. آپ کے پاس API رسائی کے ساتھ فعال WPMU DEV رکنیت ہے

### 2. انٹیگریشن کو فعال کریں

1. اپنے WordPress ایڈمن میں، Ultimate Multisite > Settings پر جائیں
2. "Domain Mapping" ٹیب پر جائیں
3. "Host Integrations" تک نیچے سکرول کریں
4. WPMU DEV انٹیگریشن کو فعال کریں
5. "Save Changes" پر کلک کریں

## یہ کیسے کام کرتا ہے

### ڈومین سنکنگ

جب Ultimate Multisite میں کوئی ڈومین میپ کیا جاتا ہے:

1. انٹیگریشن WPMU DEV API استعمال کرتے ہوئے ڈومین کو آپ کے ہوسٹنگ اکاؤنٹ میں شامل کرتا ہے
2. یہ خود بخود ڈومین کا www ورژن بھی شامل کر دیتا ہے
3. WPMU DEV ڈومین کنفیگریشن اور SSL سرٹیفکیٹ جاری کرنے کا انتظام کرتا ہے

### SSL سرٹیفکیٹ مینجمنٹ

یہ انٹیگریشن WPMU DEV ہوسٹنگ کے لیے SSL سرٹیفکیٹ کی تصدیقی کوششوں کی تعداد بڑھانے کے لیے ترتیب دیا گیا ہے، کیونکہ SSL سرٹیفکیٹس جاری اور انسٹال ہونے میں کچھ وقت لگ سکتا ہے۔ بنیادی طور پر، یہ SSL سرٹیفکیٹ کی تصدیق کے لیے 10 بار تک کوشش کرے گا، جبکہ معیاری 5 کوششیں ہوتی ہیں۔

## اہم نکات

### ڈومین ہٹانا

فی الحال، WPMU DEV API ڈومینز ہٹانے کا طریقہ فراہم نہیں کرتا۔ جب Ultimate Multisite میں ڈومین میپنگ ہٹائی جاتی ہے، تو ڈومین آپ کے WPMU DEV ہوسٹنگ اکاؤنٹ میں موجود رہے گا۔ اگر ضروری ہو تو آپ کو اسے WPMU DEV ہوسٹنگ ڈیش بورڈ سے دستی طور پر ہٹانا ہوگا۔

### API توثیق

یہ انٹیگریشن WPMU DEV API key استعمال کرتا ہے جو آپ کے WordPress ڈیٹابیس میں `wpmudev_apikey` option کے طور پر محفوظ ہے۔ یہ خود بخود سیٹ اپ ہو جاتا ہے جب آپ اپنی سائٹ کو WPMU DEV سے جوڑتے ہیں۔

## مسائل حل کرنا

### API کنکشن کے مسائل
- تصدیق کریں کہ آپ کی سائٹ WPMU DEV سے صحیح طرح جڑی ہوئی ہے
- چیک کریں کہ `wpmudev_apikey` option آپ کے WordPress ڈیٹابیس میں سیٹ ہے
- یقینی بنائیں کہ آپ کی WPMU DEV رکنیت فعال ہے

### SSL سرٹیفکیٹ کے مسائل
- WPMU DEV کو SSL سرٹیفکیٹس جاری کرنے میں کچھ وقت لگ سکتا ہے (عام طور پر 5-15 منٹ)
- انٹیگریشن SSL سرٹیفکیٹس کے لیے 10 بار تک چیک کرنے کے لیے ترتیب دیا گیا ہے
- اگر متعدد کوششوں کے بعد بھی SSL سرٹیفکیٹس جاری نہیں ہو رہے، تو WPMU DEV سپورٹ سے رابطہ کریں

### ڈومین شامل نہیں ہوا
- کسی بھی غلطی کے پیغامات کے لیے Ultimate Multisite logs چیک کریں
- تصدیق کریں کہ ڈومین پہلے سے WPMU DEV میں شامل نہیں ہے
- یقینی بنائیں کہ آپ کا WPMU DEV ہوسٹنگ پلان آپ کے شامل کیے جانے والے ڈومینز کی تعداد کو سپورٹ کرتا ہے
