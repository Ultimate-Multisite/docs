# Ultimate Multisite کی دستاویزات {#ultimate-multisite-documentation}

Ultimate Multisite کی دستاویزات میں خوش آمدید۔ Ultimate Multisite WordPress Multisite کو ایک مکمل Website as a Service (WaaS) پلیٹ فارم میں بدل دیتا ہے۔

## دستاویزات کے سیکشنز {#documentation-sections}

### [User Guide](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

آپ کو اپنا WaaS پلیٹ فارم سیٹ اپ کرنے، کنفیگر کرنے اور مینج کرنے کے لیے ہر چیز یہاں ملے گی۔ اس میں انسٹالیشن، پیمنٹ گیٹ وے، ڈومین میپنگ، کلائنٹ مینجمنٹ، اور بہت کچھ شامل ہے۔

### [Developer Documentation](/developer/hooks) {#developer-documentation}

یہ ان ڈویلپرز کے لیے ایک تکنیکی حوالہ ہے جو Ultimate Multisite پر کام کر رہے ہیں۔ اس میں hooks کا حوالہ، REST API کی دستاویزات، اور ایڈ آن ڈویلپمنٹ گائیڈز شامل ہیں۔

### [Addons](/addons) {#addons}

یہ ادا شدہ ایڈ آن پلگ انز کی دستاویزات ہیں جو Ultimate Multisite کو اضافی خصوصیات، جیسے WooCommerce انٹیگریشن، اینالیٹکس، ڈومین بیچنا، اور بہت کچھ، کے ساتھ بڑھاتے ہیں۔

## مدد حاصل کرنا {#getting-help}

- [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) -- سوالات پوچھیں اور خیالات کا تبادلہ کریں
- [GitHub Issues](https://github.com/Ultimate-Multisite/ultimate-multisite/issues) -- بگ (bugs) کی رپورٹ کریں
- [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) -- بنیادی پلگ ان ڈاؤن لوڈ کریں
