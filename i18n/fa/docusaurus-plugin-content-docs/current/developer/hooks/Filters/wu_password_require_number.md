# Filter: wu_password_require_number

فیلتر تعیین می‌کند که آیا رمز عبور باید شامل عدد باشد یا خیر.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | آیا نیاز به عدد است یا خیر. پیش‌فرض آن در صورت اعمال قوانین، `true` است. |
| $defender_active | `bool` | اینکه آیا افزونه Defender Pro Strong Password فعال است یا خیر. |

### Since {#since}

- 2.4.0
