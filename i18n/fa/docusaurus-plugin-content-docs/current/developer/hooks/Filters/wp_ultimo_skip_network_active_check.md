# فیلتر: wp_ultimo_skip_network_active_check

این فیلتر به توسعه‌دهندگان اجازه می‌دهد تا این بررسی را نادیده بگیرند (short-circuit).

این کار زمانی مفید است که از تنظیمات سفارشی و مبتنی بر composer، مانند Bedrock، استفاده می‌کنید؛ محیط‌هایی که در آن‌ها استفاده از پلاگین‌ها به صورت mu-plugins استاندارد است.

### از زمان

- 2.0.0
### منبع

در [`inc/class-requirements.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-requirements.php#L256) در خط ۲۵۶ تعریف شده است.
