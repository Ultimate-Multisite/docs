# Filter: wu_password_require_special

فیلتر تعیین می‌کند که آیا رمزهای عبور باید شامل کاراکترهای ویژه باشند یا خیر.

## پارامترها {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | آیا نیاز به کاراکترهای ویژه است. پیش‌فرض آن true است، زمانی که قوانین اعمال شده باشند. |
| $defender_active | `bool` | آیا Defender Pro Strong Password فعال است یا خیر. |

### از نسخه {#since}

- 2.4.0
