# Filter: wu_limits_is_post_above_limit

بررسی می‌کند که آیا یک نوع نوشته (post type) مشخص شده در این پلن مجاز است یا خیر. این قابلیت به توسعه‌دهندگان پلاگین اجازه می‌دهد تا مقدار بازگشتی را فیلتر کنند.

## پارامترها {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $value | `bool` | اگر نوع نوشته غیرفعال باشد یا خیر. |
