# فیلتر: wu_sso_url {#filter-wussourl}

URLهای SSO تولیدشده را پیش از بازگردانده‌شدن برای کنش‌های مشتری بین‌دامنه‌ای فیلتر می‌کند.

زمانی از این فیلتر استفاده کنید که یک یکپارچه‌سازی نیاز دارد زمینهٔ قابل‌اعتماد را به یک لینک SSO برای مستأجر حاکمیتی اضافه کند یا URL کارگزار را جایگزین کند، در حالی که اعتبارسنجی توکن Ultimate Multisite را حفظ می‌کند.

## پارامترها {#parameters}

| نام | نوع | توضیح |
|------|------|-------------|
| $sso_url | `string` | URL تولیدشدهٔ SSO. |
| $user | `WP_User` | کاربری که با بازدید SSO احراز هویت خواهد شد. |
| $site_id | `int` | شناسهٔ سایت مقصد برای بازدید. |
| $redirect_to | `string` | URL مقصد پس از اعتبارسنجی موفق SSO. |

### از نسخه {#since}

- 2.13.0

### منبع {#source}

در `inc/sso/class-sso.php` تعریف شده است.


## بازگشت‌ها {#returns}

URL فیلترشدهٔ SSO.
