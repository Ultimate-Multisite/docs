# Filter: wu_password_require_uppercase

این فیلتر تعیین می‌کند که آیا در رمز عبور، استفاده از حروف بزرگ (Uppercase) الزامی باشد یا خیر.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | تعیین اینکه آیا حروف بزرگ الزامی است. پیش‌فرض آن true است وقتی که قوانین اعمال شده باشند. |
| $defender_active | `bool` | تعیین اینکه آیا افزونه Defender Pro Strong Password فعال است یا خیر. |

### Since {#since}

- 2.4.0
