# Filter: wu_view_override_replaceable_views

به توسعه‌دهندگان اجازه می‌دهد تا پوشه‌های اضافی به لیست قابل تعویض اضافه کنند.

مراقب باشید، زیرا اجازه دادن به پوشه‌های اضافی ممکن است باعث شود که نسخه‌های قدیمی به جای نسخه‌های Ultimate Multisite بارگذاری شوند.

## پارامترها {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $replaceable_views | `array` | لیست پوشه‌های مجاز. |

### از {#since}

- 2.0.0
### منبع {#source}

در [inc/functions/template.php](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L50) در خط ۵۰ تعریف شده است.
