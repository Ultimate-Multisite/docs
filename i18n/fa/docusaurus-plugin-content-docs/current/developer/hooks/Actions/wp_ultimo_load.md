# Action: wp_ultimo_load

هنگامی که تمام وابستگی‌ها بارگذاری شدند، فعال می‌شود.

به توسعه‌دهندگان پلاگین اجازه می‌دهد تا قابلیت‌های جدیدی اضافه کنند. برای مثال، پشتیبانی از ارائه‌دهندگان هاستینگ جدید و غیره.

### از نسخه

- 2.0.0
