# Action: wu_mcp_adapter_initialized {#action-wumcpadapterinitialized}

این اکشن پس از مقداردهی اولیه (initialize) آداپتور MCP اجرا می‌شود.

این اکشن به افزونه‌ها و قالب‌های دیگر اجازه می‌دهد تا قابلیت‌های مخصوص به خود را ثبت کنند.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | نمونه (instance) آداپتور MCP. |

### Since {#since}

- 2.5.0
