# Action: wu_register_integrations {#action-wuregisterintegrations}

این اکشن برای اینکه اجازه دهد افزونه‌های یکپارچه‌سازی (integrations) خودشان را ثبت کنند، اجرا می‌شود.

هسته (Core) افزونه‌های یکپارچه‌سازی خود را در اینجا ثبت می‌کند.

## پارامترها {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $registry | `\Integration_Registry` | نمونه (instance) رجیستری. |

### از نسخه {#since}

- 2.5.0
