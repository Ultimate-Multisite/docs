# جستجوی اینترنتی

Gratis AI Agent v1.5.0 یک قابلیت **جستجوی اینترنتی** اضافه می‌کند که به دستیار هوش مصنوعی اجازه می‌دهد در طول مکالمه، اطلاعات به‌روز را از وب بازیابی کند. این قابلیت توسط [Brave Search API](https://brave.com/search/api/) یا [Tavily API](https://tavily.com/) پشتیبانی می‌شود.

## نحوه کارکرد {#how-it-works}

وقتی جستجوی اینترنتی فعال شود، دستیار می‌تواند به طور خودکار از ارائه‌دهنده جستجوی پیکربندی شده شما سؤال کند، زمانی که تشخیص دهد یک سؤال به اطلاعات جاری یا خارجی نیاز دارد — مثلاً اخبار اخیر، قیمت‌های زنده، یا مستنداتی که ممکن است از تاریخ قطع آموزش مدل تغییر کرده باشند.

نتایج به صورت بلادرنگ (real time) بازیابی شده و قبل از اینکه دستیار پاسخی تولید کند، در زمینه (context) آن تزریق می‌شود. دستیار زمانی که از نتایج جستجو برای پاسخ دادن به یک سؤال استفاده کرده باشد، این موضوع را نشان می‌دهد.

## فعال‌سازی جستجوی اینترنتی {#enabling-internet-search}

جستجوی اینترنتی نیاز به یک کلید API از ارائه‌دهنده جستجوی انتخابی شما دارد. برای پیکربندی آن، مراحل زیر را دنبال کنید:

1. به **Gratis AI Agent → Settings → Advanced** بروید.
2. منوی کشویی **Internet Search Provider** را پیدا کرده و یا **Brave Search** یا **Tavily** را انتخاب کنید.
3. کلید API خود را در فیلد مربوطه وارد کنید. لینک‌های ثبت‌نام در کنار هر فیلد نمایش داده می‌شوند.
4. روی **Save Settings** کلیک کنید.

پس از ذخیره کلید، قابلیت جستجوی اینترنتی به طور خودکار برای دستیار در دسترس خواهد بود.

## Brave Search {#brave-search}

### دریافت کلید API برای Brave Search {#obtaining-a-brave-search-api-key}

1. از [Brave Search API page](https://brave.com/search/api/) دیدن کنید.
2. برای یک پلن ثبت‌نام کنید. یک سطح رایگان با محدودیت درخواست ماهانه در دسترس است.
3. کلید API خود را از داشبورد توسعه‌دهنده Brave Search کپی کنید.
4. آن را در فیلد **Brave Search API Key** در تنظیمات Gratis AI Agent جای‌گذاری کنید.

### محدودیت‌های استفاده {#usage-limits}

Brave Search بر اساس تعداد جستجوهای انجام شده، هزینه استفاده را محاسبه می‌کند. هر پاسخ هوش مصنوعی که باعث فعال شدن جستجو شود، یک جستجو محسوب می‌شود. برای جلوگیری از هزینه‌های غیرمنتظره، استفاده خود را در [Brave Search developer dashboard](https://brave.com/search/api/) رصد کنید.

## Tavily {#tavily}

Superdav AI Agent v1.10.0، **Tavily** را به عنوان یک ارائه‌دهنده جایگزین جستجوی اینترنتی اضافه می‌کند که نتایج جستجوی غنی‌تر و قابلیت‌های تحقیقاتی پیشرفته‌ای را ارائه می‌دهد.

### دریافت کلید API برای Tavily {#obtaining-a-tavily-api-key}

1. از [Tavily API page](https://tavily.com/) دیدن کنید.
2. برای یک حساب کاربری ثبت‌نام کنید. یک سطح رایگان با محدودیت درخواست ماهانه در دسترس است.
3. کلید API خود را از داشبورد Tavily کپی کنید.
4. آن را در فیلد **Tavily API Key** در تنظیمات Gratis AI Agent جای‌گذاری کنید.

### محدودیت‌های استفاده {#usage-limits-1}

Tavily بر اساس تعداد فراخوانی‌های API، هزینه استفاده را محاسبه می‌کند. هر پاسخ هوش مصنوعی که باعث فعال شدن جستجو شود، یک فراخوانی محسوب می‌شود. برای جلوگیری از هزینه‌های غیرمنتظره، استفاده خود را در [Tavily dashboard](https://tavily.com/) رصد کنید.

## غیرفعال‌سازی جستجوی اینترنتی {#disabling-internet-search}

کلید API را از فیلد ارائه‌دهنده جستجوی فعال حذف کرده و ذخیره کنید. قابلیت جستجوی اینترنتی دیگر برای دستیار ارائه نخواهد شد.

:::note
جستجوی اینترنتی باعث تأخیر در پاسخ‌ها می‌شود، زیرا دستیار باید قبل از تولید پاسخ، منتظر نتایج جستجو بماند. برای موارد استفاده حساس به زمان، در نظر بگیرید که آیا جستجوی بلادرنگ ضروری است یا اینکه دانش داخلی دستیار کافی است.
:::
