# Agents داخلی

Gratis AI Agent v1.9.0 همراه با پنج agent داخلی عرضه می‌شود که هرکدام با مجموعه‌ای متمرکز از ابزارها، یک system prompt اختصاصی، و پیشنهادهای آغازین متناسب با کارهای رایج در آن حوزه از پیش پیکربندی شده‌اند. جابه‌جایی بین agentها تعیین می‌کند assistant چه کارهایی می‌تواند انجام دهد و چگونه پاسخ می‌دهد — بدون هیچ پیکربندی از طرف شما. Superdav AI Agent v1.18.0 می‌تواند ابزارهای آگاه از زمان‌بندی، رکوردهای یادآوری، دروازه‌های تأیید، و اعلان‌های SMS را زمانی که یکپارچه‌سازی‌های مرتبط پیکربندی شده‌اند به این جریان‌های کاری اضافه کند.

## Agent چیست؟

هر agent یک پروفایل پیکربندی نام‌گذاری‌شده است که این موارد را ترکیب می‌کند:

- **ابزارها** — توانایی‌هایی که agent مجاز است فراخوانی کند (مثلاً یک Content Writer به توانایی‌های ایجاد نوشته دسترسی دارد؛ یک Design Studio به توانایی‌های CSS و theme.json دسترسی دارد)
- **System prompt** — دستورالعمل‌هایی که لحن، اولویت‌ها و محدودیت‌های agent را تعیین می‌کنند
- **پیشنهادها** — promptهای از پیش نوشته‌شده که در رابط چت نمایش داده می‌شوند تا به شما کمک کنند سریع شروع کنید

## دسترسی به Agent Picker

1. پنل **Gratis AI Agent** را در نوار کناری مدیریت WordPress باز کنید.
2. روی **آیکن agent** در بالا-چپ سربرگ چت کلیک کنید (آیکن تغییر می‌کند تا agent فعال را نشان دهد).
3. **Agent Picker** به‌صورت یک پوشش form-table باز می‌شود. هر agent با آیکن، نام، و توضیح یک‌خطی خود فهرست شده است.
4. روی ردیف یک agent کلیک کنید تا فعال شود. سربرگ چت بلافاصله به‌روزرسانی می‌شود.

همچنین می‌توانید در میانهٔ گفتگو agentها را جابه‌جا کنید — system prompt مربوط به agent جدید از پیام بعدی اعمال می‌شود.

## پنج Agent داخلی

### Content Writer

**تمرکز:** ایجاد و ویرایش نوشته‌ها، صفحه‌ها، و فرم‌های تماس.

**ابزارهای موجود:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`. با فعال بودن یکپارچه‌سازی‌های Superdav AI Agent v1.18.0، زمینهٔ تقویم پیکربندی‌شده، دروازه‌های تأیید، یادآوری‌ها، و ابزارهای اعلان SMS نیز ممکن است برای جریان‌های کاری تأییدشده در دسترس باشند.

**کارهایی که خوب انجام می‌دهد:**
- تهیهٔ پیش‌نویس و انتشار نوشته‌های وبلاگ از روی brief یا outline
- ایجاد دسته‌ای از صفحه‌های فرود برای یک سایت جدید
- ساخت فرم‌های تماس و استعلام
- تنظیم تصاویر شاخص روی نوشته‌ها از طریق URL یا جستجو
- تهیهٔ پیش‌نویس پیام‌های پیگیری رویداد از زمینهٔ Google Calendar پیکربندی‌شده، سپس توقف برای تأیید پیش از ارسال اعلان‌ها

**پیشنهادهای آغازین:**
- *یک نوشتهٔ وبلاگ ۵۰۰ کلمه‌ای دربارهٔ مزایای WordPress multisite بنویس.*
- *یک صفحهٔ دربارهٔ ما، خدمات، و تماس ایجاد کن و آن‌ها را منتشر کن.*
- *یک فرم استعلام رزرو به صفحهٔ تماس اضافه کن.*
- *برای شرکت‌کنندگان رویداد تقویم پیکربندی‌شدهٔ فردا یک یادآوری پیش‌نویس کن و پیش از ارسال آن منتظر تأیید بمان.*

### Site Builder

**تمرکز:** ایجاد کامل وب‌سایت از ابتدا تا انتها با یک prompt واحد.

**ابزارهای موجود:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`. با Superdav AI Agent v1.18.0، ابزارهای پیکربندی‌شدهٔ خدمات مدیریت‌شده، تأیید، یادآوری، تقویم، و SMS ممکن است در جایی که مدیران آن‌ها را فعال می‌کنند در دسترس باشند.

**کارهایی که خوب انجام می‌دهد:**
- تولید یک برنامهٔ چندمرحله‌ای ساخت سایت برای نوع کسب‌وکار توصیف‌شده
- اجرای مستقل هر مرحله — ساختار، محتوا، ناوبری، طراحی
- بازیابی از خطاها در میانهٔ برنامه بدون نیاز به مداخلهٔ دستی
- نصب pluginهای پیشنهادی به‌عنوان بخشی از فرایند ساخت
- ایجاد فرم‌های تماس مستقیماً از رابط چت (Superdav AI Agent v1.10.0+)
- هماهنگ‌سازی یادآوری‌های راه‌اندازی یا پیگیری شرکت‌کنندگان بدون اعلان‌های تکراری وقتی دروازه‌های تأیید و رکوردهای یادآوری فعال باشند

**پیشنهادهای آغازین:**
- *یک سایت نمونه‌کار عکاسی با نوع نوشتهٔ گالری، یک صفحهٔ رزرو، و یک فرم تماس بساز.*
- *یک وب‌سایت رستوران با منوی آنلاین، ساعت‌های کاری، و یک فرم استعلام رزرو میز ایجاد کن.*
- *یک سایت مشاورهٔ آزادکاری با صفحه‌های خدمات، بخش نمونه‌کار، و یک وبلاگ راه‌اندازی کن.*
- *با استفاده از site builder یک فرم تماس به صفحهٔ تماس اضافه کن.*
- *پس از تأیید چک‌لیست راه‌اندازی سایت، یک یادآوری SMS برای مخاطب ذی‌نفع پیکربندی‌شده ارسال کن.*

### Design Studio

**تمرکز:** سفارشی‌سازی بصری — رنگ‌ها، تایپوگرافی، CSS، و الگوهای بلوک.

**ابزارهای موجود:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**کارهایی که خوب انجام می‌دهد:**
- اعمال presetهای نام‌گذاری‌شدهٔ theme (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- تنظیم دقیق تایپوگرافی سراسری و پالت‌های رنگی از طریق theme.json
- تزریق CSS سفارشی برای بازنویسی‌های مخصوص برند
- گرفتن اسکرین‌شات از یک صفحه و بررسی آن از نظر مسائل طراحی

**پیشنهادهای آغازین:**
- *preset warm-editorial را اعمال کن و سپس رنگ اصلی را روی #2d6a4f تنظیم کن.*
- *از صفحهٔ اصلی یک اسکرین‌شات بگیر و بگو چه چیزهایی را بهبود می‌دادی.*
- *یک الگوی بلوک hero قابل استفادهٔ مجدد با تصویر پس‌زمینهٔ تمام‌عرض و heading وسط‌چین ایجاد کن.*

### Plugin Manager

**تمرکز:** کشف، نصب، و مدیریت pluginهای WordPress.

**ابزارهای موجود:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**کارهایی که خوب انجام می‌دهد:**
- پیشنهاد بهترین plugin برای یک مورد کاربرد توصیف‌شده
- نصب بسته‌های توانایی از registry
- مرور کاتالوگ توانایی‌های موجود بر اساس دسته‌بندی

**پیشنهادهای آغازین:**
- *بهترین plugin برای یک فهرست membership چیست؟*
- *بستهٔ توانایی‌های WooCommerce را نصب کن.*
- *همهٔ بسته‌های توانایی ecommerce موجود را نشان بده.*

### Support Assistant

**تمرکز:** پاسخ دادن به پرسش‌ها دربارهٔ محتوای سایت، تنظیمات، و پیکربندی WordPress.

**ابزارهای موجود:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**چه کارهایی را خوب انجام می‌دهد:**
- جست‌وجوی تنظیمات و گزینه‌های فعلی سایت
- توضیح اینکه چه انواع نوشته، طبقه‌بندی‌ها و منوهایی روی سایت پیکربندی شده‌اند
- پاسخ به پرسش‌های «این تنظیم چه کاری انجام می‌دهد؟» با خواندن مقادیر زنده
- عمل‌کردن به‌عنوان یک لایهٔ تشخیصی فقط‌خواندنی پیش از ایجاد تغییرات

**پیشنهادهای شروع:**
- *در حال حاضر چه افزونه‌ها و تنظیماتی روی این سایت فعال هستند؟*
- *همهٔ انواع نوشتهٔ سفارشی ثبت‌شده روی این سایت را فهرست کن.*
- *چه منوهای ناوبری‌ای وجود دارند و به کجا اختصاص داده شده‌اند؟*

## یکپارچه‌سازی‌های خودکارسازی Superdav

وقتی یکپارچه‌سازی‌های Superdav AI Agent v1.18.0 پیکربندی شوند، عامل‌های داخلی می‌توانند در گردش‌کارهای خودکارسازی امن‌تر و آگاه از زمان‌بندی مشارکت کنند:

- **ابزارهای خواندن Google Calendar** به عامل‌ها اجازه می‌دهند پیش از پیش‌نویس‌کردن کارهای پیگیری، تقویم‌ها و رویدادهای پیکربندی‌شده را بررسی کنند.
- **نگاشت مخاطب و شرکت‌کننده** کمک می‌کند شرکت‌کنندگان رویداد با کاربران WordPress یا مخاطبان شناخته‌شده تطبیق داده شوند.
- **دروازه‌های تأیید انسانی** اقدامات حساس را تا زمانی که یک کاربر مجاز آن‌ها را بازبینی و تأیید کند متوقف می‌کنند.
- **رکوردهای یادآوری** هنگام تلاش مجدد یا تکرار کارهای زمان‌بندی‌شده، از اعلان‌های تکراری جلوگیری می‌کنند.
- **اعلان‌های پیامکی TextBee** فقط زمانی پیام‌های متنی پیکربندی‌شده را ارسال می‌کنند که اعتبارنامه‌های پیامک و مجوزهای گردش‌کار فعال باشند.

گردش‌کار پیشنهادی: از عامل بخواهید پیام یا اقدام را آماده کند، درخواست تأیید را بازبینی کنید، سپس اجازه دهید اقدام تأییدشده از سر گرفته شود. برای یادآوری‌های تکرارشونده، حذف تکرار یادآوری را فعال نگه دارید تا همان رویداد یا مخاطب بارها مطلع نشود.

## سفارشی‌سازی عامل‌ها

هر عامل داخلی را می‌توان از طریق فیلتر `gratis_ai_agent_agents` گسترش داد یا جایگزین کرد.

### افزودن یک اعلان سیستمی سفارشی به یک عامل موجود

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### ثبت یک عامل جدید

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

عامل جدید بلافاصله پس از اجرای فیلتر در انتخاب‌گر عامل ظاهر می‌شود.

### حذف یک عامل داخلی

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
