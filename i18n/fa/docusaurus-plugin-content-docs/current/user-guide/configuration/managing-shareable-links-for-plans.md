# مدیریت لینک‌های قابل اشتراک‌گذاری برای پلن‌ها

Ultimate Multisite این امکان را به شما می‌دهد که لینک پلن‌ها را مستقیماً با مشتریانتان به اشتراک بگذارید. برای این کار، به صفحه ویرایش هر یک از پلن‌هایتان بروید و روی دکمه _Click to copy the Shareable Link_ کلیک کنید.

![صفحه ویرایش محصول با دکمه لینک قابل اشتراک‌گذاری](/img/config/product-edit-full.png)

_در صفحه ویرایش پلن، دکمه ایجاد لینک‌های قابل اشتراک‌گذاری را پیدا می‌کنید_

توجه داشته باشید که این سازوکار از دوره پرداخت پیش‌فرض برای ساخت لینک استفاده می‌کند. یعنی اگر دوره پرداخت پیش‌فرض شما ماهانه باشد، در نهایت لینکی مثل این خواهید داشت: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

اگر می‌خواهید لینک قابل اشتراک‌گذاری به‌صورت پیش‌فرض روی پرداخت سه‌ماهه تنظیم شود، باید عدد 3 را به انتهای آن اضافه کنید

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

برای انتخاب پلن _premium_ با دوره پرداخت سالانه، عدد 1 و سپس کلمه year را اضافه کنید:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

در نظر داشته باشید که نمی‌توانید یک لینک واحد برای دوره‌های پرداخت مختلف داشته باشید. باید برای هر دوره پرداخت، یک لینک جداگانه به اشتراک بگذارید.

پس از به اشتراک گذاشتن لینک با مشتریانتان، آن‌ها به اولین مرحله از فرآیند ثبت‌نام هدایت می‌شوند.
