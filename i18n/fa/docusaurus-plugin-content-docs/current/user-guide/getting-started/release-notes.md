# یادداشت‌های انتشار

## نسخه 2.13.0 — منتشرشده در 2026-06-05

- جدید: پشتیبانی از مستأجر مستقل برای جریان‌های حساب مشتری، پرداخت، صورت‌حساب، سایت، فاکتور، تغییر قالب، و نگاشت دامنه اضافه شد تا شبکه‌های مستأجر بتوانند مشتریان را برای اقدامات مدیریت‌شده به سایت اصلی برگردانند.
- جدید: بررسی‌های اعتبارنامه تمدید برای عضویت‌های تکرارشونده اضافه شد تا درگاه‌ها بتوانند تمدید خودکار را هنگامی که توافق صورت‌حساب ذخیره‌شده، اشتراک، یا توکن خزانه وجود ندارد غیرفعال کنند.
- جدید: انتشار لوپ‌بک تأییدشده با HMAC برای ایجاد سایت‌های در انتظار اضافه شد تا آماده‌سازی از پرداخت تا سایت روی میزبان‌هایی که کارهای پس‌زمینه در آن‌ها با تأخیر انجام می‌شود قابل‌اعتمادتر شود.
- جدید: نقاط توسعه برای توسعه‌دهندگان جهت URLهای SSO، دامنه‌های پایه فرم پرداخت، و ایجاد خودکار رکورد دامنه اضافه شد.
- رفع مشکل: SSO در دامنه‌های نگاشت‌شده، بازدیدهای ناشناس کارگزار، خروج، و تداخل‌های وابستگی بین افزونه‌ها قابل‌اعتمادتر است.
- رفع مشکل: ایجاد سایت در انتظار اکنون از پرچم‌های انتشار کهنه بازیابی می‌شود و از گیر کردن مشتریان در صفحه ایجاد سایت جلوگیری می‌کند.
- رفع مشکل: رکوردهای دامنه دیگر برای دامنه‌های پایه فرم پرداخت مشترک ایجاد نمی‌شوند، و کارهای پس‌زمینه ارائه‌دهنده میزبانِ استفاده‌نشده هنگامی که هیچ یکپارچه‌سازی فعالی وجود ندارد نادیده گرفته می‌شوند.
- رفع مشکل: موارد لبه‌ای پرداخت، نشانی صورت‌حساب، بازنشانی رمز عبور، تأیید ایمیل، تغییر قالب، تورها، و Dashboard مشتری دیگر جریان‌های عادی مشتری را مسدود نمی‌کنند.
- رفع مشکل: ایمیل‌های پخش اکنون گیرندگان را خصوصی نگه می‌دارند و در عین حال هنگام شکست فهرست گیرندگان یا انتقال‌دهنده‌های ایمیل، از خطاهای مرگبار SMTP/افزونه جلوگیری می‌کنند.
- رفع مشکل: موارد لبه‌ای تمدید عضویت، نمایش انقضا، و دریافت پرداخت اکنون از انقضای فوری، خرابی، یا پرداخت‌های الزامیِ از‌دست‌رفته جلوگیری می‌کنند.
- بهبود: سازگاری WordPress تا نسخه 7.0 آزموده شده است، دارایی‌های Vue تولیدی از منابع npm دوباره ساخته شده‌اند، و پوشش سرتاسری Cypress اکنون جریان‌های بیشتری از پرداخت، راه‌اندازی، SSO، و درگاه را آزمایش می‌کند.
