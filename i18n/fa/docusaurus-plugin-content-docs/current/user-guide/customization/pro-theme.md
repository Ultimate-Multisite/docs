# سفارشی‌سازی پنل مدیریت با PRO Theme

اغلب در پروژه‌های مشتریان، نیاز داریم که تجربه‌ای سفارشی در داشبورد مدیریت WordPress به آن‌ها ارائه دهیم. اگر به دنبال ظاهری حرفه‌ای‌تر و جذاب‌تر هستید، PRO Theme می‌تواند به شما کمک کند.

PRO Theme یک قالب مدیریت است که برای کمک به شما در سفارشی‌سازی داشبورد مشتریانتان طراحی شده است. همچنین بخشی از افزونه‌های اصلی نسخه پریمیوم Ultimate Multisite است.

پس از نصب، به صفحه _Plugins_ بروید و روی گزینه فعال‌سازی در شبکه کلیک کنید.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_آن را در صفحه Plugins به صورت شبکه‌ای فعال کنید_

_**توجه:**_ _اگر فقط آن را نصب کنید، برای مشتریانتان به عنوان گزینه‌ای نمایش داده می‌شود که می‌توانند انتخاب کنند تا قالب فقط برای همان کاربر اعمال شود._

حالا رابط مدیریت شما کاملاً بازطراحی شده است! همچنین یک منوی جدید در ستون سمت چپ به نام PRO Theme دارید.
