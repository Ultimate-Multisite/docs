# قرار دادن تبلیغات در شبکه شما با Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector یک **افزونه رایگان** است که ابزارهای لازم برای قرار دادن تبلیغات روی پلن‌هایتان را مستقیماً از صفحه ویرایش آن‌ها در اختیار شما می‌گذارد.

پس از نصب Ultimate Multisite: Ad Injector، به صفحه _افزونه‌ها_ بروید و آن را فعال کنید.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_روی دکمه فعال‌سازی شبکه کلیک کنید_

وقتی افزونه فعال شد، در کادر _گزینه‌های پیشرفته_ در صفحه ویرایش پلن، گزینه جدیدی به نام _تنظیمات تبلیغات_ اضافه می‌شود.

![تنظیمات تبلیغات در صفحه ویرایش پلن](/img/config/product-edit-full.png)

_حالا می‌توانید تنظیمات تبلیغات را برای هر یک از پلن‌هایتان ویرایش کنید_

در این تب، می‌توانید کدهای تبلیغاتی را برای نوشته‌ها و برگه‌ها در بخش نمایشی سایت، قبل و بعد از محتوا اضافه کنید.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_پس از وارد کردن کد تبلیغتان، تبلیغ روی نوشته‌ها و برگه‌های آن پلن خاص نمایش داده می‌شود._

همچنین می‌توانید تبلیغات را در هدر صفحات مدیریت برای کاربران آن پلن قرار دهید.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_اگر می‌خواهید تبلیغات را در پنل مدیریت کاربرانتان نمایش دهید، به گزینه کد تبلیغات پنل مدیریت بروید_

### فراتر از تبلیغات

همچنین می‌توانید از Ultimate Multisite: Ad Injector برای ارائه پشتیبانی به پلن‌های سطح بالا استفاده کنید.

کافیست به گزینه _تبلیغات پنل مدیریت_ بروید و کد HTML ابزار پشتیبانی خود را اضافه کنید.

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_پشتیبانی مشتری را به پلن‌های خاص اضافه کنید_

به این ترتیب کاربران می‌توانند مستقیماً از پنل مدیریت خود با شما در ارتباط باشند!
