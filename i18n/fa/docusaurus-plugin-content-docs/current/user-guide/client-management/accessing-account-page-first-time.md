# چگونه می‌توانم برای اولین بار به صفحه حساب کاربری خود دسترسی پیدا کنم؟

به Ultimate Multisite خوش آمدید!

از طریق لینکی که پس از خرید افزونه در ایمیل دریافت کردید، می‌توانید به صفحه حساب کاربری خود دسترسی پیدا کنید:

<!-- Screenshot unavailable: Account page email with access link -->

از ایمیلی که هنگام خرید وارد کرده‌اید و رمز عبوری که در ایمیل دریافت کرده‌اید استفاده کنید.

<!-- Screenshot unavailable: Login page with email and password fields -->

از آنجا که این اولین بار است که وارد این حساب می‌شوید، از شما خواسته می‌شود رمز عبور خود را تغییر دهید. رمز عبور جدید را وارد کنید و روی دکمه _ذخیره_ کلیک کنید.

<!-- Screenshot unavailable: Change password form with Save button -->

در صفحه حساب کاربری، می‌توانید تمام جزئیات حساب خود را مدیریت کنید؛ از جمله صورتحساب‌ها، تمدیدها، لغو اشتراک‌ها و بسیاری موارد دیگر.
