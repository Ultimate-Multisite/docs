# هوک‌ها و فیلترهای Ultimate Multisite

Ultimate Multisite از تعداد زیادی توابع و کوئری‌های دیتابیس تشکیل شده است. این افزونه دارای هوک‌ها و فیلترهایی است که به شما امکان می‌دهد کدهای جدید به توابع موجود اضافه کنید یا آن‌ها را تغییر دهید.

ما فهرستی از این هوک‌ها و فیلترها را به عنوان مرجعی برای کمک به کاربران پیشرفته تهیه کرده‌ایم تا بتوانند شبکه خود را بیشتر سفارشی‌سازی کنند. با کلیک روی لینک زیر می‌توانید به این فهرست در Github دسترسی پیدا کنید.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
