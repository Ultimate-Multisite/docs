# چگونه از Codeable کمک برای توسعه سفارشی بگیریم؟

گاهی اوقات، واقعاً به سفارشی‌سازی‌های خاصی نیاز داریم تا جریان‌های کاری شبکه‌مان دقیقاً همان‌طور که می‌خواهیم کار کنند. اگر به دنبال چنین چیزی هستید، [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) را پیشنهاد می‌کنیم؛ جایی که می‌توانید توسعه‌دهندگان باتجربه و متخصص در Ultimate Multisite پیدا کنید.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) پلتفرمی است که مشتریان را به توسعه‌دهندگان فریلنسر حرفه‌ای WordPress متصل می‌کند؛ توسعه‌دهندگانی که توانایی کار روی پروژه‌های با هر اندازه‌ای را دارند. این پلتفرم فرآیند بررسی دقیقی دارد تا مطمئن شود شما با متخصصانی که دقیقاً برای کار مورد نظرتان واجد شرایط هستند، همکار می‌شوید.

### چگونه در Codeable پروژه ایجاد کنیم؟

ابتدا به [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) بروید و روی دکمه **Start a Project** کلیک کنید.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

حالا برای ثبت پروژه، در سمت چپ روی **Start a Project** کلیک کنید.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

در بخش "I need help with"، گزینه **Customizing** را انتخاب کنید:

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

در بخش "My"، گزینه **Plugin** را انتخاب کنید:

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

سپس روی دکمه **Continue** کلیک کنید.

<!-- Screenshot unavailable: Codeable project form Continue button -->

حالا وقت آن است که پروژه و نیازهایتان را به بهترین شکل ممکن توضیح دهید. سعی کنید تا حد امکان اطلاعات کاملی ارائه دهید!

<!-- Screenshot unavailable: Codeable project description form -->

به سؤالات زیر پاسخ دهید تا در تعیین قیمت کمک شود.

<!-- Screenshot unavailable: Codeable project pricing questions -->

پس از آن، اگر هنوز حساب کاربری در [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) ندارید، یکی بسازید و روی دکمه **Publish My Project** کلیک کنید.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

بعد از انتشار پروژه‌تان در [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE)، این پلتفرم شما را به توسعه‌دهندگانی متصل می‌کند که مهارت‌های لازم برای پروژه‌تان را دارند.

این فرآیند به شما کمک می‌کند فرد مناسب برای کارتان را پیدا کنید؛ کسی که تجربه کار با WordPress و Ultimate Multisite را دارد!

حداقل نرخ Codeable بین ۷۰ تا ۱۲۰ دلار آمریکا در ساعت است. تخمین هزینه بر اساس حجم کار، پیچیدگی و فوریت پروژه تغییر می‌کند.

پس از دریافت تخمین هزینه، می‌توانید توسعه‌دهنده‌ای که بهترین گزینه برای نیازهایتان است را استخدام کنید و پروژه‌تان را با هم شروع کنید!
