# گزارش تغییرات مالیات بر ارزش افزوده (VAT) {#vat-changelog}

Version 1.0.7 - تاریخ انتشار 2026-02-03

* رفع اشکال: شماره‌های VAT آلمان (DE) به دلیل خطاهای MS_MAX_CONCURRENT_REQ سرویس VIES SOAP، در اعتبارسنجی ناموفق بودند. ما از API قدیمی SOAP به API رسمی VIES REST اتحادیه اروپا برای اعتبارسنجی مطمئن‌تر تغییر یافتیم.
* رفع اشکال: اکنون، در صورت عدم دسترسی موقت به سرویس VIES، شماره‌های VAT به جای رد شدن اشتباه، پذیرفته می‌شوند. یک ورودی لاگ برای بررسی‌های بعدی ایجاد می‌شود.
* رفع اشکال: فیلد VAT/شناسه مالیاتی اکنون برای همه کشورها، نه فقط اعضای اتحادیه اروپا، قابل مشاهده است. شناسه‌های مالیاتی غیر اتحادیه اروپا (مانند شماره‌های CHE سوئیس) برای نمایش فاکتور، بدون نیاز به اعتبارسنجی VIES، ذخیره می‌شوند.
* رفع اشکال: جستجوی پیشوند VAT برای یونان (GR)، موناکو (MC) و جزیره من (IM) به دلیل منطق نادرست جستجوی آرایه، از کار افتاده بود.
* بهبود: وابستگی توسعه‌دهنده (dev dependency) غیرضروری wp-cli-bundle که ممکن بود هنگام استفاده با افزونه Site Exporter باعث خطاهای مهلک شود، حذف شد.
* بهبود: مجموعه کامل تست واحد (Unit Test Suite) (۵۳ تست) اضافه شد.

Version 1.0.6 - تاریخ انتشار 2026-01-25

* رفع اشکال: تراکنش‌های داخلی B2B اکنون VAT را به درستی محاسبه می‌کنند. طبق قوانین VAT اتحادیه اروپا، مالیات معکوس (reverse charge) فقط برای تراکنش‌های فرامرزی B2B اعمال می‌شود، نه زمانی که کشور مشتری با کشور شرکت مطابقت داشته باشد.

Version 1.0.5 - تاریخ انتشار 2026-01-22

* رفع اشکال: افزونه بارگذاری یا کار نمی‌کرد.
* تغییر: منبع داده نرخ VAT از euvatrates.com به مخزن ibericode/vat-rates تغییر یافت تا داده‌ای قابل اعتمادتر و فعال‌تر نگهداری شود.
* رفع اشکال: غلط املایی مقدار گزینه super_reduced_rates تصحیح شد.
* بهبود: بررسی‌های null برای مدیریت کدهای کشور اضافه شد تا از بروز خطا جلوگیری شود.
* تغییر: فایل‌های ترجمه بسته‌بندی شده حذف شدند و به جای آن‌ها، به‌روزرسانی خودکار از طریق Traduttore استفاده می‌شود.
