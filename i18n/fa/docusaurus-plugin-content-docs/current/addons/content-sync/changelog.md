# گزارش تغییرات Content Sync {#content-sync-changelog}

### نسخه 1.1.0 - تاریخ انتشار 2025-12-31 {#version-110---released-on-2025-12-31}
* جدید: پشتیبانی از همگام‌سازی تمام پست تایپ‌ها (پست‌ها، صفحات، پست تایپ‌های سفارشی)
* جدید: فیلتر کردن پست تایپ در مرحله انتخاب محتوا
* جدید: مرحله انتخاب سایت منبع - امکان همگام‌سازی از هر سایتی، نه فقط سایت اصلی
* جدید: پیش‌انتخاب سایت فرزند برای سایت‌های قالب‌بندی
* جدید: استخراج محتوای رسانه‌ای از post_content برای تمام پست تایپ‌ها
* جدید: همگام‌سازی تاکسونومی (دسته‌بندی‌ها، برچسب‌ها، تاکسونومی‌های سفارشی)
* تغییر یافت: نام آن از "Elementor Template Sync" به "Content Sync" تغییر یافت
* تغییر یافت: مرحله "Select Templates" به "Select Data" تغییر نام داد
* بهبود یافت: کدهای مخصوص Elementor اکنون فقط برای پست تایپ elementor_library اجرا می‌شوند

### نسخه 1.0.0 - تاریخ انتشار 2025-12-19 {#version-100---released-on-2025-12-19}
* اولین انتشار
* ویزارد چند مرحله‌ای برای انتخاب قالب و سایت
* پردازش دسته‌ای با بازخورد پیشرفت بلادرنگ
* همگام‌سازی کامل رسانه‌ای با نگاشت مجدد خودکار شناسه (ID)
* یکپارچه‌سازی تنظیمات با Ultimate Multisite
* پشتیبانی از حذف سایت از شبکه از طریق تنظیمات شبکه
* پاک‌سازی کش Elementor پس از همگام‌سازی
* حفظ تاکسونومی قالب
