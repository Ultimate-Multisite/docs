# MailChimp {#mailchimp}

Ultimate Multisite: اتصال به MailChimp

شبکه Multisite Ultimate خود را به MailChimp متصل کنید. شما می‌توانید طرح‌های خاصی را به لیست‌های MailChimp مرتبط کرده و با ثبت نام کاربران، آن‌ها را به طور خودکار به آن لیست‌ها اضافه کنید.

## نصب {#installation}

۱. فایل 'ultimate-multisite-mailchimp' را در دایرکتوری '/wp-content/plugins/' آپلود کنید.
۲. افزونه را از طریق منوی 'Plugins' در پنل مدیریت شبکه WordPress خود فعال کنید.

## راهنمای استفاده {#usage-guide}

همگام‌سازی ایمیل‌های مشترکین شما با لیست‌های MailChimp با افزونه اتصال Ultimate Multisite: MailChimp Integration ما بسیار آسان است. هدف این آموزش این است که همه چیز را برای شما آماده کند!

## دریافت کلید API {#getting-an-api-key}

قبل از اینکه وارد تنظیمات مختلف شویم، به یک کلید API (API key) از MailChimp نیاز داریم. این کلید به Ultimate Multisite اجازه می‌دهد تا با API مربوط به MailChimp ارتباط برقرار کرده و لیست‌ها، گروه‌ها را بازیابی کرده و آدرس‌های ایمیل را هنگام ثبت نام کاربران به آن لیست‌ها و گروه‌ها اضافه کند.

برای دریافت کلید API خود، این آموزش را دنبال کنید: [About API Keys | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

<!-- Screenshot unavailable: MailChimp API key settings in Ultimate Multisite Add-on Settings -->

_کلید API مربوط به MailChimp خود را در_ _**تنظیمات Ultimate Multisite - > Add-on Settings -> MailChimp**_ _اضافه کنید._

## حالت اتصال (Integration Mode) {#integration-mode}

افزونه اتصال MailChimp دو "حالت اتصال" ارائه می‌دهد، به این معنی که دو روش مختلف برای استفاده از این افزونه جهت همگام‌سازی آدرس‌های ایمیل وجود دارد.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_دو حالت اتصال برای انعطاف‌پذیری بیشتر!_

### حالت ۱: لیست‌های چندگانه (Multiple Lists) {#mode-1-multiple-lists}

حالت پیش‌فرض از **لیست‌های چندگانه** استفاده می‌کند. این حالت به شما امکان می‌دهد که برای هر طرح، چندین لیست MailChimp را انتخاب کنید. اگر کاربران خود را در لیست‌های مختلفی دسته‌بندی می‌کنید، این بهترین گزینه است.

برای مثال، فرض کنید در حساب MailChimp شما ۳ لیست مختلف وجود دارد: **Newsletter**، **Plan A** و **Plan B**. شما می‌خواهید که همه کاربران، صرف نظر از طرحی که دارند، در لیست Newsletter نیز قرار بگیرند تا از به‌روزرسانی‌های شما باخبر باشند.

برای رسیدن به این هدف، می‌توانید از حالت Multiple Lists استفاده کنید تا هم لیست‌های **Newsletter** و **Plan A** را در تنظیمات MailChimp مربوط به Plan A و هم **Newsletter** و **Plan B** را در تنظیمات MailChimp مربوط به Plan B اضافه کنید.

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_لیست‌های پیش‌فرض را برای حساب‌های جدید و حساب‌های لغو شده انتخاب کنید_

### حالت ۲: یک لیست + گروه‌های چندگانه (Single List + Multiple Groups) {#mode-2-single-list--multiple-groups}

MailChimp به شما اجازه می‌دهد تا لیست خود را به گروه‌های کوچک‌تری از کاربران به نام **Groups** تقسیم کنید. این کار به شما کنترل بیشتری می‌دهد، زیرا می‌توانید کمپینی را برای زیرمجموعه خاصی از مشترکین خود ارسال کنید بدون اینکه مجبور باشید چندین نسخه از همان کمپین را در MailChimp ایجاد کنید (که تحلیل معیارهای آن کمپین را بعداً دشوار می‌کند).

برای مثال، شما می‌توانید یک لیست واحد به نام **My Network Subscribers** داشته باشید و درون آن گروه‌هایی برای **Plan A** و **Plan B** قرار دهید.

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_لیست‌ها و گروه‌های پیش‌فرض را انتخاب کنید، در صورت استفاده از حالت اتصال "یک لیست + گروه‌های چندگانه"_

## لیست‌ها و گروه‌های پیش‌فرض {#default-lists-and-groups}

پس از اینکه تعیین کردید کدام حالت اتصال برای مورد استفاده خاص شما منطقی‌تر است، زمان انتخاب لیست‌های پیش‌فرض (و گروه‌ها، بسته به حالت) فرا می‌رسد. لیست پیش‌فرض زمانی استفاده می‌شود که یک طرح خاص، لیست مشخصی انتخاب نکرده باشد.

شما همچنین می‌توانید لیست پیش‌فرض را برای حساب‌های لغو شده انتخاب کنید. هر زمان که کاربری اشتراک خود را لغو کرده و حساب خود را از شبکه شما حذف کند، آدرس ایمیل او به لیست تعیین شده برای حساب‌های لغو شده منتقل خواهد شد.

## فعال‌سازی اشتراک دوگانه (Double Subscription) {#enabling-double-subscription}

در همین صفحه تنظیمات، می‌توانید **Double Subscription** را فعال کنید. این گزینه، کاربر را هم به لیست پیش‌فرض انتخاب شده و هم به لیست انتخاب شده برای طرح او اضافه می‌کند. اگر این گزینه فعال نباشد، کاربر فقط به لیست انتخاب شده برای طرح او اضافه می‌شود. اگر برای آن طرح خاصی چیزی تعیین نشده باشد، کاربر به لیست پیش‌فرض انتخاب شده اضافه خواهد شد.

<!-- Screenshot unavailable: MailChimp double subscription toggle setting -->

_شما می‌توانید کاربران خود را هم در لیست پیش‌فرض و هم در لیست طرح، هنگام ثبت نام، مشترک کنید_

## انتخاب لیست‌ها و گروه‌ها برای هر یک از طرح‌ها {#selecting-lists-and-groups-for-each-of-the-plans}

به بخش **Plans** بروید و طرحی را که می‌خواهید ویرایش کنید، انتخاب نمایید. توجه داشته باشید که در باکس متا **Advanced Options**، تب جدیدی به نام MailChimp در دسترس خواهد بود.

در آن تب، می‌توانید لیست‌های چندگانه یا یک لیست و گروه‌های چندگانه (بسته به حالت اتصال) را برای آن طرح انتخاب کنید.

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_لیست‌ها و گروه‌هایی که می‌خواهید برای هر یک از طرح‌هایتان استفاده کنید را انتخاب کنید_

هنگامی که کاربر جدیدی برای آن طرح ثبت نام می‌کند، آدرس ایمیل او به لیست‌ها و گروه‌های انتخاب شده اضافه خواهد شد!
