# سازنده سایت هوش مصنوعی

ultimate-multisite-ai-site-builder

## پیش‌نیازها {#requirements}

- WordPress نسخه ۵.۳ یا بالاتر
- PHP نسخه ۷.۴ یا بالاتر
- افزونه Ultimate Multisite (فعال باشد)

## نصب {#installation}

۱. فایل‌های افزونه را در مسیر `/wp-content/plugins/` آپلود کنید.
۲. افزونه را از طریق منوی 'Plugins' در وردپرس فعال کنید.
۳. تنظیمات افزونه را در بخش مدیریت شبکه (network admin) پیکربندی کنید.

## توسعه {#development}

### راه‌اندازی {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### ساختار پروژه {#project-structure}

```
ultimate-multisite-ai-site-builder/
├── inc/                          # کلاس‌ها و فایل‌های شامل PHP
│   ├── class-multisite-ultimate-updater.php  # مدیریت به‌روزرسانی
│   └── class-ai_site_builder-main.php             # عملکرد اصلی
├── tests/                        # تست‌های واحد (Unit tests)
│   ├── bootstrap.php            # بوت‌استرپ تست
│   ├── class-*-base.php         # کلاس پایه تست
│   └── class-*-test.php         # موارد تست
├── views/                       # فایل‌های قالب (Template files)
├── assets/                      # دارایی‌های استاتیک
│   ├── css/                     # فایل‌های استایل (Stylesheets)
│   ├── js/                      # فایل‌های جاوااسکریپت
│   └── images/                  # تصاویر
├── lang/                        # فایل‌های زبان
├── composer.json                # وابستگی‌های PHP
├── package.json                 # وابستگی‌های Node.js
├── phpunit.xml.dist             # پیکربندی PHPUnit
├── .phpcs.xml.dist              # پیکربندی استانداردهای کدنویسی
└── rector.php                   # قوانین مدرن‌سازی کد
```

### تست‌گیری {#testing}

این افزونه شامل یک چارچوب تست جامع است:

- **تست‌های واحد (Unit Tests)**: تست‌های مبتنی بر PHPUnit با مجموعه تست وردپرس
- **کلاس پایه (Base Test Class)**: متدهای کمکی برای سناریوهای تست رایج فراهم می‌کند
- **پوشش تست (Test Coverage)**: برای ردیابی پوشش کد پیکربندی شده است
- **تست‌گیری چندسایتی (Multisite Testing)**: تست‌ها در محیط چندسایتی اجرا می‌شوند

### استانداردهای کدنویسی {#code-standards}

- **استانداردهای کدنویسی وردپرس**: با PHPCS اعمال می‌شود
- **سازگاری با PHP 7.4+**: از ویژگی‌های مدرن PHP پشتیبانی می‌کند
- **تحلیل استاتیک (Static Analysis)**: یکپارچه‌سازی PHPStan برای بررسی نوع داده (type checking)
- **مدرن‌سازی کد**: قوانین Rector برای ارتقاء PHP

### سیستم ساخت (Build System) {#build-system}

- **پردازش دارایی‌ها (Asset Processing)**: کوچک‌سازی (Minification) برای فایل‌های CSS/JS
- **ترجمه (Translation)**: تولید فایل POT
- **بسته‌بندی (Packaging)**: ایجاد آرشیو برای توزیع
- **حالت توسعه (Development Mode)**: دارایی‌های بدون کوچک‌سازی برای اشکال‌زدایی (debugging)

## پیکربندی {#configuration}

این افزونه از گزینه‌های پیکربندی مختلفی پشتیبانی می‌کند که از طریق رابط مدیریت شبکه (network admin) قابل دسترسی هستند.

## هوک‌ها و فیلترها {#hooks-and-filters}

### اکشن‌ها (Actions) {#actions}

- `ultimate-multisite-ai-site-builder_init` - پس از مقداردهی اولیه افزونه اجرا می‌شود
- `ultimate-multisite-ai-site-builder_loaded` - پس از بارگذاری تمام فایل‌های افزونه اجرا می‌شود

### فیلترها (Filters) {#filters}

- `ultimate-multisite-ai-site-builder_settings` - تغییر تنظیمات افزونه
- `ultimate-multisite-ai-site-builder_enabled` - بازنویسی وضعیت فعال/غیرفعال بودن افزونه
