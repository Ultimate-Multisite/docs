# Plugin & Theme Manager

Multisite Ultimate: Plugin and Theme Manager

اطلاعات متا (عنوان، نویسنده، تصویر کوچک و توضیحات) پلاگین‌ها و تم‌های نصب شده را ویرایش یا مخفی کنید. برای پلاگین‌ها و تم‌ها دسته‌بندی ایجاد کنید تا به کاربران اجازه دهید آن‌ها را فیلتر کنند و یک صفحه پلاگین سفارشی و زیبا به پنل کاربری آن‌ها اضافه کنید.

برای استفاده از این پلاگین، کافی است به صفحات پلاگین‌ها یا تم‌های بخش مدیریت شبکه (network admin) خود بروید و روی "Edit Plugin/Theme info" کلیک کنید.

## نصب {#installation}

1. 'ultimate-multisite-plugin-and-theme-manager' را در دایرکتوری '/wp-content/plugins/' آپلود کنید
2. پلاگین را از طریق منوی 'Plugins' در بخش مدیریت شبکه WordPress خود فعال کنید

## راهنمای استفاده {#usage-guide}

Ultimate Multisite: Plugin and Theme Manager یک افزونه (add-on) قدرتمند **پریمیوم** برای Ultimate Multisite است که کنترل کاملی به شما می‌دهد که کاربران شبکه شما چگونه با تنظیمات پلاگین و تم در شبکه شما تعامل داشته و آن‌ها را مشاهده کنند.

این افزونه سه کارکرد اصلی دارد:

  1. به شما امکان می‌دهد تا اطلاعات متا (metadata) هم پلاگین‌ها و هم تم‌ها را به طور کامل تغییر دهید. این شامل نام، توضیحات، نام نویسنده و حتی تصاویر کوچک (thumbnails) است!

  2. به شما امکان می‌دهد تا پلاگین‌ها و تم‌ها را دسته‌بندی کنید تا کاربران شما بتوانند به راحتی آن‌ها را مرتب کنند.

  3. به شما این گزینه را می‌دهد که صفحه پلاگین‌های قدیمی و خسته‌کننده را با یک صفحه جدید شبیه به تم‌ها که قابلیت مرتب‌سازی بر اساس دسته‌بندی دارد، جایگزین کنید.

پس از دانلود و نصب، یک تب جدید به نام _Add-on Settings_ در صفحه تنظیمات Ultimate Multisite نمایش داده خواهد شد.
