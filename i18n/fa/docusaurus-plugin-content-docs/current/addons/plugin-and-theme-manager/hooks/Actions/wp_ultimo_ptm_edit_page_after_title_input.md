# Action: wp_ultimo_ptm_edit_page_after_title_input

به توسعه‌دهندگان پلاگین اجازه می‌دهد تا اطلاعات اضافی زیر فیلد ورودی متن اضافه کنند.

## پارامترها

| Name | Type | Description |
|------|------|-------------|
| $ | `object` | شیئی که اطلاعات را در خود نگه می‌دارد |
| $ | `\WU_Page` | نمونه (instance) صفحه Ultimate Multisite |

### از نسخه

- 1.8.2
### منبع

- در [`views/base/edit.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-plugin-and-theme-manager/blob/main/views/base/edit.php#L95) در خط ۹۵ تعریف شده است.
