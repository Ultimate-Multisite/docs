# فیلتر: wp_ultimo_skip_network_active_check

### منبع

در [inc/class-requirements.php](https://github.com/Ultimate-Multisite/ultimate-multisite-plugin-and-theme-manager/blob/main/inc/class-requirements.php#L227) در خط ۲۲۷ تعریف شده است.
