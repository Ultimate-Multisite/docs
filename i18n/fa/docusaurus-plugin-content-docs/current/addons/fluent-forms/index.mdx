# Fluent Forms

این افزونه امکان اضافه کردن محدودیت به Fluent Forms در محصولات، عضویت‌ها یا سایت‌ها در Ultimate Multisite را فراهم می‌کند.
Fluent Forms به جای استفاده از Custom Post Type، از جداول سفارشی (custom tables) استفاده می‌کند، بنابراین گزینه‌های محدودیت‌گذاری معمولی با Fluent Forms کار نمی‌کنند.

## ویژگی‌ها

- اضافه کردن محدودیت برای Fluent Forms یا Conversational Fluent Forms
- وقتی مشتری به محدودیت برسد، یک پیام دوستانه دریافت می‌کند که او را تشویق به ارتقاء پلن خود می‌کند.
