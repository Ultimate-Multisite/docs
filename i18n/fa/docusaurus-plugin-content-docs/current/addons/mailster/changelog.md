# گزارش تغییرات Mailster

### ۱.۰.۰
* انتشار اولیه
* همگام‌سازی خودکار مشترکین هنگام ثبت‌نام یا پرداخت
* تخصیص لیست‌های مخصوص هر محصول
* گزینه انتخاب (Opt-in) با چک‌باکس برای رعایت قوانین GDPR
* پشتیبانی از دابل آپت-این (Double opt-in)
* نگاشت (Mapping) جامع فیلدها
* لاگ‌برداری (ثبت وقایع) دقیق
