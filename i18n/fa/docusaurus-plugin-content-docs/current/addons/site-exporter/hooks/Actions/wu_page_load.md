# Action: wu_page_load

به توسعه‌دهندگان پلاگین اجازه می‌دهد تا هوک‌های اضافی به صفحات ما اضافه کنند.

## پارامترها {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $page_id | `string\|\WP_Ultimo_Site_Exporter\Admin_Pages\Base_Admin_Page` | شناسه (ID) این صفحه. |
| $page_hook | `string\|\WP_Ultimo_Site_Exporter\Admin_Pages\Base_Admin_Page` | هوک (hook) که برای این صفحه تولید شده است. |
