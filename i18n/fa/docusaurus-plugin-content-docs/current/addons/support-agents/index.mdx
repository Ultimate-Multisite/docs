# نمایندگان پشتیبانی

مدیریت یک شبکه کامل به تنهایی کار آسانی نیست. با رشد شبکه شما، تقاضای پشتیبانی هم افزایش پیدا می‌کند. بهتر است افرادی را استخدام کنید که به کاهش بار کاری شما کمک کنند. با این حال، دادن دسترسی/مجوز کامل خطرناک است و ممکن است شبکه شما را به خطر بیندازد. اینجاست که افزونه Support Agent وارد عمل می‌شود.

این افزونه به شما امکان می‌دهد که نوع جدیدی از کاربر وردپرس به نام Support Agent ایجاد کنید و کنترل کاملی بر مجوزهای این کاربر داشته باشید. دادن مجوز به سادگی روشن کردن یک دکمه (toggle button) است و اگر تصمیم بگیرید آن را حذف کنید، کافی است آن را خاموش کنید.

## ویژگی‌ها

- لیست تفصیلی مجوزهای وردپرس و Ultimate Multisite
- فعال و غیرفعال کردن دسترسی با سهولت
- ایجاد تعداد نامحدودی حساب کاربری Agent

## نصب

1. فایل 'ultimate-multisite-support-agents' را در دایرکتوری '/wp-content/plugins/' آپلود کنید.
2. افزونه را از طریق منوی 'Plugins' در وردپرس فعال کنید.
3. برای راه‌اندازی افزونه، مراحل گام به گام ویزارد (Wizard) را دنبال کنید.
