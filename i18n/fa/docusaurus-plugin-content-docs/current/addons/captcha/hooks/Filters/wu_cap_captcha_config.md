فیلتر تنظیمات Cap Captcha.

این فیلتر به توسعه‌دهندگان اجازه می‌دهد تا تنظیمات Cap Captcha را سفارشی‌سازی کنند.

## پارامترها {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $config | `array` | آرایه تنظیمات Cap. |
| $level | `string` | سطح امنیتی (fast, medium, max). |
