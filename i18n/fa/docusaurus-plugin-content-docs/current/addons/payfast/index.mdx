# PayFast

Ultimate Multisite: دروازه پرداخت Payfast

این افزونه، Payfast - که یک دروازه پرداخت آنلاین سریع و امن در آفریقای جنوبی است - را به عنوان یکی از گزینه‌های پرداخت Ultimate Multisite اضافه می‌کند.

## نصب

۱. 'ultimate-multisite-payfast' را در دایرکتوری '/wp-content/plugins/' آپلود کنید.
۲. افزونه را از طریق منوی 'Plugins' در WordPress فعال کنید.
۳. مراحل گام به گام ویزارد (Wizard) را دنبال کنید تا افزونه را تنظیم نمایید.

## راهنمای استفاده

PayFast یک پردازشگر پرداخت آنلاین پیشرو برای کسب‌وکارها در آفریقای جنوبی است. این سرویس متعهد به کمک به فروشندگان آنلاین برای رشد است و این کار را با ساده و امن کردن تراکنش‌ها انجام می‌دهد. طبق درخواست کاربران Ultimate Multisite، ما تصمیم گرفتیم یک افزونه (addon) بسازیم که اتصال به PayFast را ممکن کند. این مقاله نشان می‌دهد که چگونه PayFast را به عنوان دروازه پرداخت خود در شبکه Ultimate Multisite دریافت کنید.

اولین قدم نصب این افزونه است.

# نحوه نصب افزونه Payfast:

به صفحه افزونه‌ها (add-ons) بروید و افزونه Ultimate Multisite: Payfast Gateway را انتخاب کنید.

![Add-ons page](/img/admin/addons-page.png)

پنجره افزونه باز می‌شود. روی Install Now کلیک کنید.

![Add-ons page with install option](/img/admin/addons-page.png)

پس از اتمام نصب، به صفحه افزونه‌ها (plugins) هدایت می‌شوید. در اینجا، کافی است روی Network Activate کلیک کنید تا افزونه _Payfast_ در شبکه شما فعال شود.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# راه‌اندازی PayFast

پس از نصب افزونه، اکنون می‌توانید به تنظیمات Ultimate Multisite بروید و شروع به پیکربندی PayFast کنید.

در صفحه **Ultimate Multisite settings**، می‌توانید تب "Payments" را پیدا کنید. در این تب، باید "Payfast" را به همراه یک دکمه سوییچ (toggle button) برای فعال یا غیرفعال کردن این دروازه پرداخت پیدا کنید.

![Payment gateway settings](/img/config/settings-payment-gateways.png)و دکمه سوییچ را بزنید تا آن را فعال کنید. سپس، این کار کادر را باز کرده و فیلدهای مختلفی را برای شما نمایش می‌دهد که باید آن‌ها را پر کنید، مانند **Merchant ID, Merchant Key و PassPhrase.** شما می‌توانید این اطلاعات را از حساب **Payfast** خود دریافت کنید.

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

گام بعدی ورود به حساب PayFast شما برای دریافت اطلاعات مورد نیاز است.

Upon logging in, you can immediately see the credentials you need from your **Payfast** account dashboard.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->شما به سادگی می‌توانید این اعتبارنامه‌ها را کپی و در تنظیمات Ultimate Multisite خود وارد کنید. سپس روی دکمه **"Save settings"** کلیک کنید تا تغییرات ذخیره شوند.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

همین است، ادغام پرداخت PayFast شما کامل شد!
