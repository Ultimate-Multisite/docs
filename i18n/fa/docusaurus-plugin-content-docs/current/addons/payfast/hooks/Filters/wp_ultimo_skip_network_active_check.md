# فیلتر: wp_ultimo_skip_network_active_check {#filter-wpultimoskipnetworkactivecheck}

این فیلتر به توسعه‌دهندگان اجازه می‌دهد تا این بررسی را دور بزنند (short-circuit).

این کار زمانی مفید است که از تنظیمات سفارشی و مبتنی بر کامپوزر (composer-based) استفاده می‌کنید، مانند Bedrock، که در این حالت استفاده از پلاگین‌ها به عنوان mu-plugins امری عادی است.

### از زمان {#since}

- 2.0.0
### منبع {#source}

در [`inc/class-requirements.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-payfast/blob/main/inc/class-requirements.php#L229) در خط ۲۲۹ تعریف شده است.
