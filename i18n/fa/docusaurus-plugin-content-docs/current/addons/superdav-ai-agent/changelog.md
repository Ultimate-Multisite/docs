# گزارش تغییرات

## ۱.۱۶.۰ — تاریخ انتشار ۲۰۲۶-۰۵-۲۰

### جدید

- **قابلیت تولید لوگوی SVG** — اکنون Theme Builder می‌تواند لوگوی SVG سفارشی را با پاک‌سازی ایمن در فضای نام (namespace-safe sanitisation) تولید و جاسازی کند.
- **آپلود عکس در مصاحبه کشف (discovery interview)** — مصاحبه کشف Theme Builder اکنون شامل یک مرحله آپلود عکس برای زمینه طراحی غنی‌تر است.
- **قابلیت اعتبارسنجی کنتراست پالت** — قبل از اعمال پالت رنگ به یک تم، جفت رنگ‌ها را برای مطابقت با استانداردهای WCAG بررسی کنید.
- **منوهای هتلی** — Theme Builder اکنون می‌تواند صفحات منوی غذایی و نوشیدنی ساختاریافته را برای کسب‌وکارهای هتلی تولید کند.
- **رندر پیش‌نمایش دسکتاپ و موبایل** — در حین انتخاب جهت طراحی، طرح خود را روی دستگاه‌های دسکتاپ و موبایل پیش‌نمایش کنید.
- **پارامتر برچسب ناوبری** — قابلیت Create Menu اکنون از یک `navigation_label` مجزا که جدا از عنوان صفحه است، پشتیبانی می‌کند.
- **قابلیت ابزار سطح ۱** — sd-ai-agent/site-scrape اکنون یک ابزار سطح ۱ است که به طور پیش‌فرض در Theme Builder در دسترس است.

### رفع اشکال

- **کش (Cache) کلاینت AI** — اکنون با استفاده از transients پشتیبانی می‌شود تا پایداری بین درخواست‌ها حفظ شود و از از دست رفتن داده‌ها در وظایف طولانی‌مدت عامل جلوگیری شود.
- **لینک‌های اکشن سطر پلاگین** — برای وضوح، تصحیح و تغییر نام داده شدند.

## ۱.۱۰.۰ — تاریخ انتشار ۲۰۲۶-۰۵-۰۵

### جدید

- **جستجوی اینترنت Tavily** — Tavily را به عنوان یک ارائه‌دهنده جستجو برای نتایج جستجوی اینترنتی غنی‌تر، در کنار Brave Search، اضافه کنید.
- **مهارت‌های داخلی آگاه به تم** — Block Themes، Classic Themes، Kadence Blocks و راهنماهای مهارت Kadence Theme اکنون همراه با پلاگین عرضه می‌شوند.
- **قابلیت فرم تماس سازنده سایت** — یک فرم تماس را مستقیماً از رابط چت به هر صفحه‌ای اضافه کنید.

### بهبود یافته

- **بازسازی یکپارچه‌سازی WooCommerce** — اکنون از APIهای بومی WooCommerce برای قابلیت اطمینان و سازگاری بهتر استفاده می‌کند.
- **به‌روزرسانی خودکار لیست ارائه‌دهندگان** — هنگام فعال یا غیرفعال کردن هر پلاگینی.

### رفع اشکال

- **قابلیت navigate-to** — حلقه رفرش بی‌نهایت (infinite reload loop) در برخی صفحات ادمین رفع شد.
- **قابلیت list-posts** — اکنون نام دسته‌بندی و برچسب‌ها را به درستی به اسلاگ (slug) تبدیل می‌کند.
- **دستورات WP-CLI** — نام‌های مستعار (alias) فضای نام از دست رفته پس از یک بازسازی قبلی، بازیابی شدند.
- **اتوماسیون رویداد** — سایت‌هایی که هنوز جداول اتوماسیون در آن‌ها ایجاد نشده‌اند را به طور مناسب مدیریت می‌کند.
- **قابلیت memory-save** — اکنون از پیشوند فضای نام صحیح در سازنده دستورالعمل سیستم استفاده می‌کند.
- **نتایج ابزار Scalar** — اکنون قبل از بازگرداندن به AI، به درستی بسته‌بندی می‌شوند.
- **آمار استفاده** — اکنون فرمت کلید قابلیت قدیمی را هنگام ارتقا از نسخه‌های قدیمی‌تر به درستی مدیریت می‌کند.
