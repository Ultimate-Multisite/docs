# Superdav AI Agent

Superdav AI Agent یک دستیار هوش مصنوعی پیشرفته برای وردپرس است که قابلیت‌های آگاه به تم (theme-aware skills)، جستجوی اینترنتی پیشرفته‌تر و ویژگی‌های بهبودیافته ساخت سایت (site builder) را به Gratis AI Agent اضافه می‌کند. نسخه ۱.۱۶.۰ شامل تولید لوگوی سفارشی، اعتبارسنجی کنتراست پالت رنگ، آپلود عکس در مصاحبه کشف (discovery interview)، تولید منوی هتلداری ساختاریافته، و رندر پیش‌نمایش دسکتاپ/موبایل در Theme Builder است.

## ویژگی‌های کلیدی

- **مهارت‌های آگاه به تم (Theme-aware skills)** — راهنمایی و قابلیت‌ها را بر اساس تم فعال وردپرس و پلاگین‌های نصب شده شما، به صورت خودکار تنظیم می‌کند.
- **تأمین‌کنندگان جستجوی اینترنتی متعدد** — برای نتایج جستجوی غنی‌تر و قابلیت‌های تحقیقاتی پیشرفته، بین Brave Search و Tavily انتخاب کنید.
- **فرم‌های تماس ساخت سایت** — با استفاده از پلاگین فرم نصب شده، فرم‌های تماس کاملاً کاربردی را مستقیماً از رابط چت ایجاد کنید.
- **ساخت سایت پیشرفته** — برنامه‌های ساخت سایت چند مرحله‌ای بهبود یافته با کشف پلاگین بهتر و بازیابی خطای بهتر.
- **مهارت‌های داخلی تم (Built-in theme skills)** — راهنمایی تخصصی برای Block Themes، Classic Themes، Kadence Blocks و Kadence Theme.
- **یکپارچه‌سازی بهبودیافته WooCommerce** — بازسازی شده تا از APIهای بومی WooCommerce برای قابلیت اطمینان و سازگاری بهتر استفاده کند.
- **تشخیص خودکار تأمین‌کننده** — پلاگین‌ها و تم‌های موجود را هنگام فعال یا غیرفعال شدن، به صورت خودکار تشخیص داده و به‌روزرسانی می‌کند.
- **تولید لوگوی سفارشی** — لوگوهای SVG سفارشی را با پاک‌سازی خودکار فضای نام (namespace-safe sanitisation) تولید و جاسازی کنید.
- **اعتبارسنجی کنتراست پالت رنگ** — قبل از اعمال رنگ‌ها روی تم شما، جفت رنگ‌ها را برای انطباق با دسترسی‌پذیری WCAG بررسی کنید.
- **کشف با عکس** — برای تأثیرگذاری بر جهت‌گیری طراحی، عکس آپلود کنید در طول مصاحبه کشف Theme Builder.
- **منوهای هتلداری** — صفحات منوی غذا و نوشیدنی ساختاریافته را برای رستوران‌ها، کافه‌ها و کسب‌وکارهای هتلداری تولید کنید.
- **پیش‌نمایش دسکتاپ و موبایل** — در طول انتخاب جهت‌گیری طراحی، طرح خود را روی دستگاه‌های دسکتاپ و موبایل پیش‌نمایش کنید.
- **خزش سایت سطح ۱ (Tier 1 site scraping)** — sd-ai-agent/site-scrape اکنون به طور پیش‌فرض برای تحلیل وب‌سایت‌های رقیب و الهام‌بخش در دسترس است.

## پیش‌نیازها

- WordPress ۶.۰ یا بالاتر
- PHP ۸.۱ یا بالاتر
- WordPress Multisite (فعال در شبکه) یا نصب تک‌سایتی
- یک پلاگین کانکتور AI فعال (Anthropic، OpenAI، یا سازگار)
- Gratis AI Agent v1.9.0 یا بالاتر (Superdav AI Agent، Gratis AI Agent را گسترش می‌دهد)

## نصب

۱. فایل‌های پلاگین را در `/wp-content/plugins/superdav-ai-agent` آپلود کنید.
۲. پلاگین را از صفحه **Plugins** در وردپرس فعال کنید.
۳. مطمئن شوید که Gratis AI Agent نیز فعال شده باشد.
۴. به **Gratis AI Agent → Settings** بروید و تأمین‌کننده جستجوی اینترنتی خود را پیکربندی کنید (اختیاری).

## شروع سریع

پس از فعال‌سازی، Superdav AI Agent به طور خودکار نصب Gratis AI Agent شما را با مهارت‌های آگاه به تم و قابلیت‌های اضافی تقویت می‌کند. پنل **Gratis AI Agent** را در نوار کناری مدیریت وردپرس باز کرده و از ویژگی‌های جدید استفاده کنید.

**مثال دستورات:**

```
Add a contact form to the Contact page.
```

```
Search the web for the latest WordPress security best practices.
```

```
Create a hero section using the active theme's block patterns.
```

## مهارت‌های آگاه به تم (Theme-Aware Skills)

Superdav AI Agent به طور خودکار تم فعال و پلاگین‌های نصب شده شما را تشخیص داده و مهارت آگاه به تم مناسب را در زمینه دستیار بارگذاری می‌کند. مهارت‌های موجود عبارتند از:

| Skill | Applies To |
|---|---|
| **Block Themes** | تم‌هایی که از معماری مبتنی بر بلوک (Full Site Editing) وردپرس استفاده می‌کنند |
| **Classic Themes** | تم‌های سنتی وردپرس که از قالب‌های PHP و ویرایشگر کلاسیک استفاده می‌کنند |
| **Kadence Blocks** | سایت‌هایی که پلاگین Kadence Blocks فعال دارند |
| **Kadence Theme** | سایت‌هایی که از تم Kadence به عنوان تم فعال استفاده می‌کنند |

برای راهنمایی دقیق در مورد هر مهارت، به [Theme-Aware Skills](../../user-guide/configuration/theme-aware-skills) مراجعه کنید.

## تأمین‌کنندگان جستجوی اینترنتی (Internet Search Providers)

Superdav AI Agent v1.10.0، **Tavily** را به عنوان جایگزینی برای Brave Search برای کوئری‌های جستجوی اینترنتی اضافه کرده است. هر دو تأمین‌کننده نتایج جستجوی بلادرنگ ارائه می‌دهند و می‌توانند در **Gratis AI Agent → Settings → Advanced** پیکربندی شوند.

| Provider | Features | Setup |
|---|---|---|
| **Brave Search** | نتایج جستجوی سریع و متمرکز بر حریم خصوصی | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | نتایج جستجوی غنی‌تر، قابلیت‌های تحقیقاتی پیشرفته | [Tavily API](https://tavily.com/) |

برای دستورالعمل‌های پیکربندی، به [Internet Search](../../user-guide/configuration/internet-search) مراجعه کنید.

## فرم‌های تماس ساخت سایت (Site Builder Contact Forms)

عامل Site Builder اکنون می‌تواند فرم‌های تماس را مستقیماً از رابط چت ایجاد کند. کافی است از Site Builder بخواهید که یک فرم تماس ایجاد کند، و این عامل به طور خودکار پلاگین فرم نصب شده شما (Contact Form 7، WPForms، Fluent Forms، یا Gravity Forms) را تشخیص داده و یک فرم آماده استفاده تولید می‌کند.

برای راهنمایی دقیق، به [Site Builder Contact Forms](../../user-guide/administration/site-builder-contact-forms) مراجعه کنید.

## ویژگی‌های Theme Builder

Theme Builder در Superdav AI Agent v1.16.0 شامل قابلیت‌های قدرتمند جدیدی است:

### مصاحبه کشف با آپلود عکس (Discovery Interview with Photo Upload)

مصاحبه کشف Theme Builder اکنون شامل مرحله آپلود عکس است که می‌توانید الهام بصری ارائه دهید. برای کمک به Theme Builder در درک جهت‌گیری طراحی شما، عکس‌های برند، سایت‌های رقیب یا الهام‌بخش را آپلود کنید.

برای راهنمایی دقیق، به [Discovery Interview](./theme-builder/discovery-interview.md) مراجعه کنید.

### جهت‌گیری طراحی با پیش‌نمایش دسکتاپ و موبایل (Design Direction with Desktop and Mobile Previews)

قبل از اینکه Theme Builder تم کامل شما را تولید کند، طرح خود را هم روی دستگاه‌های دسکتاپ و هم موبایل بررسی کنید. پیش‌نمایش‌های بلادرنگ را در چندین نقطه شکست (responsive breakpoints) مشاهده کنید.

برای راهنمایی دقیق، به [Design Direction](./theme-builder/design-direction.md) مراجعه کنید.

### منوهای هتلداری (Hospitality Menus)

Theme Builder اکنون می‌تواند صفحات منوی غذا و نوشیدنی ساختاریافته را برای رستوران‌ها، کافه‌ها، بارها و سایر کسب‌وکارهای هتلداری تولید کند. منوها شامل قیمت‌گذاری، توضیحات، اطلاعات رژیمی و هشدارهای آلرژن هستند.

برای راهنمایی دقیق، به [Hospitality Menus](./theme-builder/hospitality-menus.md) مراجعه کنید.

### قابلیت‌های جدید

- **[Generate Logo SVG](./abilities/generate-logo-svg.md)** — ایجاد لوگوهای SVG سفارشی با پاک‌سازی خودکار
- **[Validate Palette Contrast](./abilities/validate-palette-contrast.md)** — بررسی جفت رنگ‌ها برای انطباق با WCAG
- **[Create Menu](./abilities/create-menu.md)** — ایجاد منوهای ناوبری با برچسب‌های ناوبری متمایز

برای اطلاعات در مورد ابزارهای موجود و ابزار خزش سایت سطح ۱، به [Tools and Configuration](./theme-builder/tools.md) مراجعه کنید.

## پیکربندی (Configuration)

تمام تنظیمات در **Gratis AI Agent → Settings** در مدیریت وردپرس قرار دارند. Superdav AI Agent گزینه‌های پیکربندی زیر را اضافه می‌کند:

| Setting | Description |
|---|---|
| **Internet Search Provider** | انتخاب بین Brave Search و Tavily (اختیاری) |
| **Tavily API Key** | کلید API برای جستجوی اینترنتی Tavily (در صورت انتخاب Tavily) |

## بهبودها در v1.16.0

- **AI Client cache** — اکنون توسط transients پشتیبانی می‌شود تا پایداری بین درخواست‌ها حفظ شود و از از دست رفتن داده‌ها در وظایف طولانی مدت عامل جلوگیری کند.
- **ابزار خزش سایت سطح ۱ (Tier 1 site-scrape tool)** — sd-ai-agent/site-scrape اکنون به طور پیش‌فرض در Theme Builder برای تحلیل وب‌سایت‌های رقیب و الهام‌بخش در دسترس است.
- **لینک‌های اکشن سطر پلاگین** — برای وضوح تصحیح و تغییر نام داده شدند.

## بهبودها در v1.10.0

- **بازسازی یکپارچه‌سازی WooCommerce** — اکنون از APIهای بومی WooCommerce برای قابلیت اطمینان و سازگاری بهتر استفاده می‌کند.
- **به‌روزرسانی خودکار لیست تأمین‌کنندگان** — به طور خودکار تشخیص می‌دهد که پلاگین‌ها فعال یا غیرفعال شده‌اند و لیست تأمین‌کننده موجود را به‌روز می‌کند.
- **ساخت سایت پیشرفته** — برنامه‌های ساخت سایت چند مرحله‌ای بهبود یافته با بازیابی خطای بهتر.
- **مهارت‌های آگاه به تم** — چهار مهارت داخلی جدید که به طور خودکار با تم فعال شما تطبیق می‌یابند.

## رفع اشکالات در v1.10.0

- رفع قابلیت `navigate-to` که باعث حلقه‌های بارگذاری بی‌نهایت در برخی صفحات مدیریت می‌شد.
- رفع قابلیت `list-posts` که نام دسته‌بندی و برچسب‌ها را به درستی به اسلاگ تبدیل نمی‌کرد.
- رفع دستورات WP-CLI که پس از یک بازسازی قبلی، نام‌های فضای نام (namespace aliases) را از دست می‌دادند.
- رفع اینکه اتوماسیون رویدادها به طور مناسب سایت‌هایی را که هنوز جداول اتوماسیون برای آن‌ها ایجاد نشده است، مدیریت می‌کرد.
- رفع قابلیت `memory-save` که از پیشوند فضای نام صحیح در سازنده دستورالعمل سیستم استفاده می‌کرد.
- رفع اینکه نتایج ابزار اسکالر (scalar tool) قبل از بازگرداندن به AI به درستی بسته‌بندی می‌شدند.
- رفع اینکه آمار استفاده، فرمت کلید قابلیت قدیمی را هنگام ارتقا از نسخه‌های قدیمی‌تر به درستی مدیریت می‌کرد.
