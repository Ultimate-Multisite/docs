# Superdav AI Agent {#superdav-ai-agent}

Superdav AI Agent یک دستیار پیشرفته هوش مصنوعی برای WordPress است که Gratis AI Agent را با مهارت‌های آگاه از قالب، قابلیت‌های جست‌وجوی اینترنتی بهبودیافته، ویژگی‌های سایت‌ساز، و ادغام‌های اختیاری خودکارسازی گسترش می‌دهد. نسخه 1.18.0 ابزارهای خواندن Google Calendar، نگاشت مخاطب و شرکت‌کننده، دروازه‌های تأیید انسانی، حذف تکرار یادآورها، اعلان‌های پیامکی TextBee، فراهم‌سازی سرویس مدیریت‌شده Superdav، و یک بسته همراه پیشرفته که به‌صورت جداگانه برای جریان‌های کاری توسعه‌دهندگان مورداعتماد توزیع می‌شود را اضافه می‌کند.

## ویژگی‌های کلیدی {#key-features}

- **مهارت‌های آگاه از قالب** — راهنمایی‌ها و قابلیت‌ها را بر اساس قالب WordPress فعال و افزونه‌های نصب‌شده شما به‌صورت خودکار تطبیق می‌دهد
- **چند ارائه‌دهنده جست‌وجوی اینترنتی** — برای نتایج جست‌وجوی غنی‌تر و قابلیت‌های پژوهشی پیشرفته، بین Brave Search و Tavily انتخاب کنید
- **فرم‌های تماس سایت‌ساز** — فرم‌های تماس کاملاً کاربردی را مستقیماً از رابط گفت‌وگو با استفاده از افزونه فرم نصب‌شده خود ایجاد کنید
- **سایت‌ساز بهبودیافته** — برنامه‌های ساخت سایت چندمرحله‌ای بهبودیافته با کشف بهتر افزونه‌ها و بازیابی خطا
- **مهارت‌های داخلی قالب** — راهنمایی تخصصی برای قالب‌های بلوکی، قالب‌های کلاسیک، Kadence Blocks، و Kadence Theme
- **ادغام بهبودیافته WooCommerce** — برای اطمینان‌پذیری و سازگاری بهتر، بازطراحی‌شده تا از APIهای بومی WooCommerce استفاده کند
- **تشخیص خودکار ارائه‌دهنده** — افزونه‌ها و قالب‌های دردسترس را هنگام فعال یا غیرفعال شدن، به‌صورت خودکار تشخیص داده و تازه‌سازی می‌کند
- **تولید لوگوی سفارشی** — لوگوهای SVG سفارشی را با پاک‌سازی خودکار و ایمن از نظر فضای نام تولید و جاسازی کنید
- **اعتبارسنجی کنتراست پالت** — پیش از اعمال روی قالب خود، جفت‌رنگ‌ها را از نظر انطباق با دسترس‌پذیری WCAG بررسی کنید
- **کشف بهبودیافته با عکس** — هنگام مصاحبه کشف Theme Builder عکس بارگذاری کنید تا بر جهت‌گیری طراحی تأثیر بگذارد
- **منوهای مهمان‌نوازی** — صفحات منوی ساختاریافته غذا و نوشیدنی را برای رستوران‌ها، کافه‌ها، و کسب‌وکارهای مهمان‌نوازی تولید کنید
- **پیش‌نمایش دسکتاپ و موبایل** — هنگام انتخاب جهت‌گیری طراحی، طراحی خود را روی دستگاه‌های دسکتاپ و موبایل پیش‌نمایش کنید
- **خزش سایت سطح ۱** — sd-ai-agent/site-scrape اکنون به‌صورت پیش‌فرض برای تحلیل وب‌سایت‌های رقیب و الهام‌بخش در دسترس است
- **ابزارهای خواندن Google Calendar** — تقویم‌ها و رویدادهای پیکربندی‌شده را بخوانید تا خودکارسازی‌ها بتوانند درباره زمینه برنامه زمانی پیشِ‌رو استدلال کنند
- **نگاشت مخاطب و شرکت‌کننده** — پیش از تولید کارهای پیگیری، شرکت‌کنندگان تقویم را با کاربران و مخاطبان شناخته‌شده سایت تطبیق دهید
- **دروازه‌های تأیید انسانی** — خودکارسازی‌های حساس را پیش از ارسال پیام‌ها، ازسرگیری کار، یا ایجاد تغییرات برای بازبینی متوقف کنید
- **سوابق یادآور و حذف تکرار** — یادآورهایی را که قبلاً ارسال شده‌اند پیگیری کنید تا اجرای تکراری خودکارسازی‌ها به یک گیرنده دوبار اطلاع‌رسانی نکند
- **اعلان‌های پیامکی TextBee** — هنگامی که یک ارائه‌دهنده پیامک فعال است، اعلان‌های پیام متنی پیکربندی‌شده را از طریق TextBee ارسال کنید
- **فراهم‌سازی سرویس مدیریت‌شده Superdav** — به‌جای پیکربندی دستی نقطه پایانی، سایت‌های پشتیبانی‌شده را با راه‌اندازی خودکار به نقاط پایانی میزبانی‌شده Superdav متصل کنید
- **بسته همراه پیشرفته** — ابزارهای توسعه‌دهنده مورداعتماد که به‌صورت جداگانه برای جریان‌های کاری سیستم فایل، پایگاه داده، WP-CLI، توزیع‌کننده REST، سازنده افزونه، snapshot گیت، مدیریت کاربر، و بنچمارک توزیع می‌شوند

## الزامات {#requirements}

- WordPress 6.0 یا بالاتر
- PHP 8.1 یا بالاتر
- نصب WordPress چندسایته (فعال‌شده در شبکه) یا تک‌سایته
- یک افزونه رابط ارائه‌دهنده هوش مصنوعی فعال (Anthropic، OpenAI، یا سازگار)
- Gratis AI Agent v1.9.0 یا بالاتر (Superdav AI Agent، Gratis AI Agent را گسترش می‌دهد)

## نصب {#installation}

1. فایل‌های افزونه را در `/wp-content/plugins/superdav-ai-agent` بارگذاری کنید
2. افزونه را از طریق صفحه **افزونه‌ها** در WordPress فعال کنید
3. مطمئن شوید Gratis AI Agent نیز فعال شده است
4. به **Gratis AI Agent → تنظیمات** بروید و در صورت نیاز، ارائه‌دهنده جست‌وجوی اینترنتی، اتصال سرویس مدیریت‌شده Superdav، Google Calendar، یا اطلاعات اعتبارسنجی TextBee خود را پیکربندی کنید

بسته WordPress.org شامل ویژگی‌های اصلی Superdav AI Agent است. بسته همراه پیشرفته برای محیط‌هایی که ابزارهای توسعه‌دهنده مورداعتماد به‌صراحت نصب و مجاز شده‌اند، به‌صورت جداگانه ساخته و توزیع می‌شود.

## شروع سریع {#quick-start}

پس از فعال‌سازی، Superdav AI Agent نصب Gratis AI Agent شما را به‌صورت خودکار با مهارت‌های آگاه از قالب و قابلیت‌های اضافی بهبود می‌دهد. پنل **Gratis AI Agent** را در نوار کناری مدیریت WordPress باز کنید و استفاده از ویژگی‌های جدید را آغاز کنید.

**دستورالعمل‌های نمونه:**

```
یک فرم تماس به صفحه تماس اضافه کن.
```

```
وب را برای تازه‌ترین بهترین روش‌های امنیت WordPress جست‌وجو کن.
```

```
با استفاده از الگوهای بلوکی قالب فعال، یک بخش قهرمان ایجاد کن.
```

```
رویدادهای Google Calendar پیکربندی‌شده فردا را بررسی کن و یادآورها را برای شرکت‌کنندگان پیش‌نویس کن.
```

```
پس از تأیید پیام توسط من، یک یادآور پیامکی ارسال کن.
```

## مهارت‌های آگاه از قالب {#theme-aware-skills}

Superdav AI Agent به‌صورت خودکار قالب فعال و افزونه‌های نصب‌شده شما را تشخیص می‌دهد و مهارت آگاه از قالب مناسب را در زمینه دستیار بارگذاری می‌کند. مهارت‌های در دسترس شامل موارد زیر هستند:

| مهارت | قابل اعمال برای |
|---|---|
| **قالب‌های بلوکی** | قالب‌هایی که از معماری بلوک‌محور WordPress (ویرایش کامل سایت) استفاده می‌کنند |
| **قالب‌های کلاسیک** | قالب‌های سنتی WordPress که از الگوهای PHP و ویرایشگر کلاسیک استفاده می‌کنند |
| **Kadence Blocks** | سایت‌هایی که افزونه Kadence Blocks در آن‌ها فعال است |
| **Kadence Theme** | سایت‌هایی که قالب فعال آن‌ها Kadence Theme است |

برای راهنمایی دقیق درباره هر مهارت، [مهارت‌های آگاه از قالب](../../user-guide/configuration/theme-aware-skills) را ببینید.

## ارائه‌دهندگان جست‌وجوی اینترنتی {#internet-search-providers}

Superdav AI Agent v1.10.0، **Tavily** را به‌عنوان جایگزینی برای Brave Search جهت پرس‌وجوهای جست‌وجوی اینترنتی اضافه می‌کند. هر دو ارائه‌دهنده نتایج جست‌وجوی بی‌درنگ ارائه می‌دهند و می‌توانند در **Gratis AI Agent → تنظیمات → پیشرفته** پیکربندی شوند.

| ارائه‌دهنده | ویژگی‌ها | راه‌اندازی |
|---|---|---|
| **Brave Search** | نتایج جست‌وجوی سریع و متمرکز بر حریم خصوصی | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | نتایج جست‌وجوی غنی‌تر، قابلیت‌های پژوهشی پیشرفته | [Tavily API](https://tavily.com/) |

برای دستورالعمل‌های پیکربندی، [جست‌وجوی اینترنتی](../../user-guide/configuration/internet-search) را ببینید.

## گردش‌کارهای تقویم، تأیید و اعلان {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 گردش‌کارهای خودکارسازی را فراتر از وظایف محتوا و ساخت سایت گسترش می‌دهد:

- **ابزارهای Google Calendar** به عامل اجازه می‌دهند وقتی اعتبارنامه‌های تقویم در دسترس هستند، تقویم‌های پیکربندی‌شده و جزئیات رویداد را بخواند.
- **کمک‌کننده‌های مخاطب و شرکت‌کننده** پیش از آماده‌سازی یادآوری‌ها یا وظایف پیگیری، شرکت‌کنندگان رویداد را به کاربران WordPress یا مخاطبان شناخته‌شده نگاشت می‌کنند.
- **دروازه‌های تأیید انسانی** به یک خودکارسازی اجازه می‌دهند تا زمانی که یک کاربر مجاز اقدام پیشنهادی را بررسی کند، متوقف شود.
- **رکوردهای یادآوری** وقتی یک خودکارسازی دوباره تلاش می‌شود یا به‌طور مکرر زمان‌بندی می‌شود، از ایجاد یادآوری‌های تکراری تقویم یا اعلان جلوگیری می‌کنند.
- **پشتیبانی SMS TextBee** وقتی ارائه‌دهنده TextBee پیکربندی شده باشد، اعلان‌های پیامکی را فعال می‌کند.

از این ابزارها برای کمک آگاه از زمان‌بندی، پیگیری شرکت‌کنندگان، و گردش‌کارهای یادآوری استفاده کنید که همچنان پیش از ارسال پیام‌های حساس به بازبینی انسانی نیاز دارند.

برای راهنمایی تفصیلی، [فرم‌های تماس سایت‌ساز](../../user-guide/administration/site-builder-contact-forms) را ببینید.

## پیکربندی {#configuration}

همه تنظیمات در بخش **Gratis AI Agent → تنظیمات** در مدیریت WordPress قرار دارند. Superdav AI Agent گزینه‌های پیکربندی زیر را اضافه می‌کند:

| تنظیم | توضیح |
|---|---|
| **ارائه‌دهنده جست‌وجوی اینترنتی** | بین Brave Search و Tavily انتخاب کنید (اختیاری) |
| **Tavily API Key** | کلید API برای جست‌وجوی اینترنتی Tavily (اگر Tavily انتخاب شده باشد) |
| **سرویس Superdav مدیریت‌شده** | سایت‌های پشتیبانی‌شده را با فراهم‌سازی خودکار endpoint به سرویس میزبانی‌شده Superdav متصل کنید |
| **اعتبارنامه‌های Google Calendar** | دسترسی تقویم را پیکربندی کنید تا عامل بتواند تقویم‌ها و رویدادها را بخواند |
| **ارائه‌دهنده SMS TextBee** | اعتبارنامه‌های TextBee را برای گردش‌کارهای اعلان SMS تأییدشده پیکربندی کنید |

## بسته همراه پیشرفته {#advanced-companion-package}

Superdav AI Agent v1.18.0 یک بسته همراه پیشرفته با توزیع جداگانه برای گردش‌کارهای توسعه‌دهنده مورد اعتماد اضافه می‌کند. این قابلیت‌ها در بسته هسته WordPress.org گنجانده نشده‌اند. بسته همراه را فقط روی سایت‌هایی نصب کنید که مدیران به‌صراحت به عامل اعتماد دارند تا به ابزارهای توسعه‌دهنده دسترسی داشته باشد.

ابزارهای پیشرفته شامل عملیات سیستم فایل، دسترسی پایگاه داده، WP-CLI، گردش‌کارهای dispatch کننده REST، ابزارهای کمکی سازنده plugin، snapshotهای git، کمک‌کننده‌های مدیریت کاربر، و ابزارهای benchmark هستند. این ابزارها را محدود به مدیران مورد اعتماد نگه دارید و پیش از فعال‌سازی آن‌ها در سایت‌های تولید، پیکربندی دروازه تأیید را بازبینی کنید.

- **توزیع همراه پیشرفته** — انتشار ابزارهای توسعه‌دهنده مورد اعتماد جدا از ساخت هسته WordPress.org

- **حافظه پنهان کلاینت هوش مصنوعی** — اکنون برای پایداری بین درخواست‌ها با transientها پشتیبانی می‌شود و از از دست رفتن داده‌ها در وظایف طولانی‌مدت agent جلوگیری می‌کند
- **ابزار خزش سایت Tier 1** — `sd-ai-agent/site-scrape` اکنون به‌صورت پیش‌فرض در Theme Builder برای تحلیل وب‌سایت‌های رقیب و الهام‌بخش در دسترس است
- **پیوندهای کنش ردیف افزونه** — برای وضوح بیشتر اصلاح و تغییر نام داده شدند

## بهبودها در v1.10.0 {#improvements-in-v1100}

- **بازطراحی یکپارچه‌سازی WooCommerce** — اکنون برای قابلیت اطمینان و سازگاری بهتر از APIهای بومی WooCommerce استفاده می‌کند
- **نوسازی خودکار فهرست ارائه‌دهندگان** — به‌طور خودکار تشخیص می‌دهد چه زمانی افزونه‌ها فعال یا غیرفعال می‌شوند و فهرست ارائه‌دهندگان در دسترس را به‌روزرسانی می‌کند
- **سازنده سایت بهبودیافته** — برنامه‌های ساخت سایت چندمرحله‌ای با بازیابی بهتر از خطا بهبود یافتند
- **مهارت‌های آگاه از پوسته** — چهار مهارت داخلی جدید که به‌طور خودکار با پوسته فعال شما سازگار می‌شوند
