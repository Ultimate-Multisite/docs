# مصاحبه کشف (Discovery Interview)

**مصاحبه کشف** اولین گام در فرآیند طراحی Theme Builder است. این مصاحبه اطلاعاتی درباره سایت، کسب‌وکار و ترجیحات طراحی شما جمع‌آوری می‌کند تا یک قالب (Theme) شخصی‌سازی شده ایجاد شود.

## نمای کلی (Overview) {#overview}

مصاحبه کشف شما را از طریق مجموعه‌ای از سؤالات راهنمایی می‌کند تا برند، مخاطب هدف و اهداف طراحی شما را درک کند. در نسخه v1.16.0، این مصاحبه شامل یک مرحله **آپلود عکس** شده است که به شما امکان می‌دهد الهام بصری برای طراحی خود ارائه دهید.

## مراحل مصاحبه {#interview-steps}

### ۱. اطلاعات کسب‌وکار {#1-business-information}

- نام سایت و شعار (Tagline)
- نوع کسب‌وکار یا صنعت
- اهداف اصلی کسب‌وکار
- توصیف مخاطب هدف

### ۲. ترجیحات طراحی {#2-design-preferences}

- طرح رنگ مورد علاقه
- سبک طراحی (مدرن، کلاسیک، مینیمال، بازیگوش و غیره)
- وب‌سایت‌ها یا برندهای الهام‌بخش
- عناصر طراحی ضروری

### ۳. آپلود عکس (جدید در v1.16.0) {#3-photo-upload-new-in-v1160}

برای تأثیرگذاری بر جهت‌گیری طراحی، یک یا چند عکس آپلود کنید:

#### فرمت‌های پذیرفته شده {#accepted-formats}

| Format | Extension | Details |
|--------|-----------|---------|
| **JPEG** | `.jpg`, `.jpeg` | برای عکس‌های عکاسی توصیه می‌شود |
| **PNG** | `.png` | از شفافیت (Transparency) پشتیبانی می‌کند |
| **WebP** | `.webp` | فرمت مدرن با حجم فایل کمتر |
| **GIF** | `.gif` | فقط تصاویر استاتیک (بدون انیمیشن) |

#### الزامات فایل {#file-requirements}

- **حداکثر حجم فایل:** ۱۰ مگابایت
- **حداقل ابعاد:** ۴۰۰ × ۳۰۰ پیکسل
- **ابعاد پیشنهادی:** ۱۲۰۰ × ۸۰۰ پیکسل یا بزرگتر
- **نسبت ابعاد:** هر نسبتی قابل قبول است

#### عکس‌ها چگونه بر جهت‌گیری طراحی تأثیر می‌گذارند {#how-photos-influence-design-direction}

عکس‌هایی که آپلود می‌کنید به Theme Builder کمک می‌کنند تا موارد زیر را درک کند:

- **پالت رنگی** — رنگ‌های غالب استخراج شده و در قالب استفاده می‌شوند
- **سبک بصری** — مدرن، روستیک، مینیمال، پر جنب و جوش و غیره
- **حس و حال و جو** — حرفه‌ای، خلاقانه، دوستانه، لوکس و غیره
- **عناصر طراحی** — الگوها، بافت‌ها و سبک‌های ترکیب‌بندی
- **الهام تایپوگرافی** — سبک‌های فونتی که با تم بصری مکمل باشند

### ۴. ساختار محتوا {#4-content-structure}

- صفحات اصلی مورد نیاز (خانه، درباره ما، خدمات، تماس با ما و غیره)
- نیاز به بخش وبلاگ یا اخبار
- نیاز به قابلیت تجارت الکترونیک (E-commerce)
- ویژگی‌ها یا یکپارچه‌سازی‌های ویژه

### ۵. بازبینی و تأیید {#5-review-and-confirmation}

- خلاصه تمام اطلاعات ارائه شده
- تأیید قبل از شروع تولید طراحی

## بهترین شیوه‌ها برای آپلود عکس {#photo-upload-best-practices}

### چه چیزی آپلود کنید {#what-to-upload}

- **الهام برند** — لوگوها، پالت رنگی، یا دستورالعمل‌های برند
- **سایت‌های رقیب** — اسکرین‌شات‌هایی از سایت‌هایی که تحسین می‌کنید
- **Mood boards** — مجموعه‌ای از تصاویر که زیبایی‌شناسی مورد نظر شما را نشان می‌دهد
- **عکس‌های محصول** — اگر محصولی برای نمایش دارید
- **عکس‌های تیم یا دفتر کار** — برای برندسازی شخصی یا حرفه‌ای
- **تصاویر سبک زندگی** — که مخاطب هدف شما را نشان می‌دهند

### از چه چیزی باید پرهیز کنید {#what-to-avoid}

- **تصاویر با کیفیت پایین** — عکس‌های تار یا پیکسلی
- **تصاویر نامرتبط** — عکس‌هایی که برند شما را نشان نمی‌دهند
- **تصاویر متنی سنگین** — اسکرین‌شات‌هایی با متن زیاد
- **مواد دارای حق چاپ** — مطمئن شوید که حق استفاده از تصاویر را دارید
- **GIFهای متحرک** — فقط تصاویر استاتیک پردازش می‌شوند

### نکات برای بهترین نتیجه {#tips-for-best-results}

۱. **۲ تا ۳ عکس آپلود کنید** — تصاویر متعدد، زمینه بهتری ارائه می‌دهند
۲. **از عکس‌های با کیفیت بالا استفاده کنید** — تصاویر واضح و با نور مناسب بهترین کارایی را دارند
۳. **تنوع داشته باشید** — ترکیبی از رنگ، سبک و موضوع
۴. **زمینه ارائه دهید** — به طور خلاصه توضیح دهید که چرا هر عکس را انتخاب کرده‌اید
۵. **مشخص باشید** — اگر عنصر خاصی را دوست دارید، به آن اشاره کنید

## خروجی جهت‌گیری طراحی {#design-direction-output}

پس از مصاحبه کشف، Theme Builder موارد زیر را تولید می‌کند:

- **پالت رنگی** — رنگ‌های اصلی، ثانویه و تأکیدی
- **تایپوگرافی** — خانواده‌های فونت برای عناوین و متن اصلی
- **سبک چیدمان** — فاصله، تراز و چیدمان اجزا
- **عناصر بصری** — آیکون‌ها، الگوها و عناصر تزئینی
- **Mood board** — نمایش بصری جهت‌گیری طراحی

## گام‌های بعدی {#next-steps}

پس از تکمیل مصاحبه کشف:

۱. جهت‌گیری طراحی تولید شده را بررسی کنید
۲. بازخورد دهید یا درخواست تغییرات کنید
۳. به مرحله [Design Direction](./design-direction.md) بروید
۴. Theme Builder قالب سفارشی شما را ایجاد می‌کند
