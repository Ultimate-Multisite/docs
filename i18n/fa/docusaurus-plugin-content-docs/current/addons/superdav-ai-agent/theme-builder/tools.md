# ابزارها و تنظیمات

Theme Builder مجموعه‌ای از ابزارها را برای تحلیل، طراحی و ساخت سایت وردپرسی شما استفاده می‌کند. در نسخه ۱.۱۶.۰، **sd-ai-agent/site-scrape اکنون یک ابزار سطح ۱ (Tier 1) است** و به همین دلیل به صورت پیش‌فرض در Theme Builder در دسترس قرار گرفته است.

## ابزارهای موجود

### ابزارهای سطح ۱ (Tier 1) (همیشه در دسترس)

ابزارهای سطح ۱ بدون نیاز به هیچ تنظیمات اضافی، به صورت پیش‌فرض در Theme Builder در دسترس هستند.

#### sd-ai-agent/site-scrape

**هدف:** تحلیل وب‌سایت‌های موجود برای استخراج الهام طراحی، ساختار محتوا و عملکرد.

**قابلیت‌ها:**

- **تحلیل وب‌سایت:** خزش (scrape) و تحلیل وب‌سایت‌های رقیب یا الهام‌بخش
- **استخراج طراحی:** شناسایی رنگ‌ها، فونت‌ها و الگوهای چیدمان
- **ساختار محتوا:** درک سازماندهی و سلسله مراتب صفحات
- **تشخیص ویژگی:** شناسایی پلاگین‌ها و قابلیت‌ها
- **تحلیل عملکرد:** بررسی سرعت و بهینه‌سازی صفحه
- **تحلیل سئو (SEO):** بررسی تگ‌های متا و داده‌های ساختاریافته

**نحوه استفاده:**

```
Analyze the design of example.com and use it as inspiration for my site.
```

**آنچه استخراج می‌کند:**

- پالت رنگ و نحوه استفاده از رنگ‌ها
- تایپوگرافی (فونت‌ها و اندازه‌ها)
- ساختار چیدمان و فاصله‌گذاری
- الگوهای ناوبری (Navigation)
- سازماندهی محتوا
- استفاده از تصاویر و رسانه‌ها
- عناصر تعاملی
- واکنش‌گرایی در موبایل (Mobile responsiveness)

**محدودیت‌ها:**

- نمی‌تواند سایت‌های محافظت شده با رمز عبور را خزش کند
- به فایل robots.txt و سیاست‌های سایت احترام می‌گذارد
- ممکن است محتوای پویا (dynamic content) را ثبت نکند
- سایت‌های سنگین جاوا اسکریپت ممکن است تحلیل محدودی داشته باشند
- سایت‌های بزرگ ممکن است زمان بیشتری برای تحلیل نیاز داشته باشند

### ابزارهای سطح ۲ (Tier 2) (اختیاری)

ابزارهای سطح ۲ زمانی در دسترس هستند که به صراحت در تنظیمات Theme Builder فعال شوند.

#### Advanced Analytics (تحلیل پیشرفته)

معیارهای عملکرد دقیقی ارائه می‌دهد:

- زمان بارگذاری صفحه
- Core Web Vitals
- امتیاز سئو (SEO score)
- امتیاز دسترسی‌پذیری (Accessibility score)
- امتیاز بهترین شیوه‌ها (Best practices score)

#### Content Optimizer (بهینه‌ساز محتوا)

برای موارد زیر تحلیل کرده و بهبودهایی پیشنهاد می‌دهد:

- خوانایی (Readability)
- بهینه‌سازی سئو
- استفاده از کلمات کلیدی
- ساختار محتوا
- قرارگیری فراخوان به اقدام (Call-to-action placement)

### ابزارهای سطح ۳ (Tier 3) (پرمیوم)

ابزارهای سطح ۳ نیاز به تنظیمات اضافی یا ویژگی‌های پریمیوم دارند.

#### AI Content Generator (تولیدکننده محتوای هوش مصنوعی)

محتوا برای موارد زیر تولید می‌کند:

- توضیحات محصول
- صفحات خدمات
- پست‌های وبلاگ
- توضیحات متا (Meta descriptions)
- متن فراخوان به اقدام (Call-to-action copy)

#### Advanced Design Tools (ابزارهای طراحی پیشرفته)

قابلیت‌های طراحی پیشرفته‌ای را فراهم می‌کند:

- تولید CSS سفارشی
- ایجاد انیمیشن
- طراحی عناصر تعاملی
- نظریه رنگ پیشرفته
- بهینه‌سازی تایپوگرافی

## پیکربندی ابزار

### فعال‌سازی ابزارها

برای فعال‌سازی ابزارهای اضافی در Theme Builder:

۱. به **Gratis AI Agent → Settings** بروید
۲. به **Theme Builder → Tools** بروید
۳. ابزارها را در صورت نیاز روشن/خاموش کنید
۴. تنظیمات را ذخیره کنید

### مجوزهای ابزار

برخی ابزارها نیاز به مجوز دارند:

- **Site scraping (خزش سایت):** نیاز به دسترسی اینترنت دارد
- **Analytics (تحلیل):** نیاز به یکپارچه‌سازی Google Analytics دارد
- **Content generation (تولید محتوا):** نیاز به کلیدهای API دارد
- **Advanced features (ویژگی‌های پیشرفته):** ممکن است نیاز به اشتراک پریمیوم داشته باشد

### کلیدهای API و اعتبارنامه‌ها

برای ابزارهایی که نیاز به آن‌ها دارند، کلیدهای API را پیکربندی کنید:

۱. به **Gratis AI Agent → Settings → API Keys** بروید
۲. اعتبارنامه‌ها را برای هر ابزار وارد کنید
۳. اتصال را تست کنید
۴. به صورت امن ذخیره کنید

## استفاده از sd-ai-agent/site-scrape

### استفاده پایه

از Theme Builder بخواهید یک وب‌سایت را تحلیل کند:

```
Analyze the design of my competitor's site at competitor.com
and suggest design improvements for my site.
```

### تحلیل خاص

درخواست انواع خاصی از تحلیل:

```
Extract the color palette from example.com and use it as inspiration.
```

```
Analyze the navigation structure of example.com and apply similar
organization to my site.
```

```
Check the mobile responsiveness of example.com and ensure my site
is equally responsive.
```

### تحلیل مقایسه‌ای

مقایسه چندین سایت:

```
Compare the designs of site1.com and site2.com and create a design
that combines the best elements of both.
```

## محدودیت‌ها و ملاحظات ابزار

### محدودیت نرخ (Rate Limiting)

- خزش برای جلوگیری از بار اضافی سرور، محدود می‌شود
- درخواست‌های متعدد به یک سایت ممکن است محدود شوند (throttled)
- سایت‌های بزرگ ممکن است زمان بیشتری برای تحلیل نیاز داشته باشند

### دقت محتوا

- ممکن است محتوای پویا به طور کامل ثبت نشود
- محتوای رندر شده با جاوا اسکریپت ممکن است ناقص باشد
- داده‌های بلادرنگ (real-time) ممکن است منعکس نشوند
- ممکن است برخی محتواها پشت دیوار پرداخت (paywalls) باشند

### ملاحظات قانونی و اخلاقی

- به robots.txt و سیاست‌های سایت احترام بگذارید
- محتوای دارای حق نشر را برای بازتولید خزش نکنید
- از تحلیل برای الهام گرفتن استفاده کنید، نه کپی کردن
- تأیید کنید که حق استفاده از محتوای استخراج شده را دارید
- شرایط خدمات سایت‌های تحلیل شده را رعایت کنید

### تأثیر عملکردی

- خزش سایت‌های بزرگ ممکن است زمان‌بر باشد
- خزش‌های همزمان متعدد ممکن است کندتر باشند
- اتصال شبکه بر سرعت تأثیر می‌گذارد
- مجموعه داده‌های بزرگ ممکن است نیاز به پردازش بیشتری داشته باشند

## بهترین شیوه‌ها

### استفاده از تحلیل سایت

۱. **تحلیل چندین سایت** — الهام را از چندین منبع جمع‌آوری کنید
۲. **تمرکز بر ساختار** — الگوهای چیدمان و سازماندهی را یاد بگیرید
۳. **استخراج رنگ‌ها** — از پالت‌های رنگی به عنوان نقطه شروع استفاده کنید
۴. **مطالعه تایپوگرافی** — ترکیب فونت‌هایی که دوست دارید را شناسایی کنید
۵. **بررسی ناوبری** — سازماندهی منو را درک کنید

### خزش اخلاقی

۱. **احترام به robots.txt** — دستورالعمل‌های سایت را دنبال کنید
۲. **کپی کردن محتوا نکنید** — از تحلیل فقط برای الهام استفاده کنید
۳. **حقوق را تأیید کنید** — مطمئن شوید که حق استفاده از محتوای استخراج شده را دارید
۴. **منبع‌دهی کنید** — منابع الهام را ذکر کنید
۵. **قوانین را رعایت کنید** — از شرایط خدمات سایت پیروی کنید

### به حداکثر رساندن اثربخشی ابزار

۱. **مشخص باشید** — درخواست انواع خاصی از تحلیل کنید
۲. **بافتار ارائه دهید** — هدف سایت خود را توضیح دهید
۳. **انتظارات را تعیین کنید** — اهداف طراحی خود را شرح دهید
۴. **تکرار کنید** — بر اساس نتایج اصلاح کنید
۵. **ابزارها را ترکیب کنید** — برای تحلیل جامع از چندین ابزار استفاده کنید

## عیب‌یابی

### سایت خزش نمی‌شود

- بررسی کنید که آیا سایت از نظر عمومی قابل دسترسی است
- تأیید کنید که robots.txt اجازه خزش می‌دهد
- یک سایت متفاوت را امتحان کنید
- اتصال اینترنت را بررسی کنید
- اگر مشکل ادامه داشت، با پشتیبانی تماس بگیرید

### تحلیل ناقص

- ممکن است سایت حاوی محتوای پویا باشد
- ممکن است جاوا اسکریپت به طور کامل رندر نشده باشد
- سایت‌های بزرگ ممکن است زمان‌بندی (timeout) شوند
- سعی کنید به جای آن، صفحات خاصی را تحلیل کنید
- انواع تحلیل خاصی را درخواست کنید

### عملکرد کند

- تحلیل سایت‌های بزرگ زمان بیشتری می‌برد
- خزش‌های همزمان متعدد کندتر هستند
- اتصال شبکه بر سرعت تأثیر می‌گذارد
- سعی کنید در ساعات کم‌ترافیک کار کنید
- ابتدا بخش‌های کوچک‌تر را تحلیل کنید
