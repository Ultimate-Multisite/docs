# ابزارها و پیکربندی

Theme Builder از مجموعه‌ای از ابزارها برای تحلیل، طراحی و ساخت سایت WordPress شما استفاده می‌کند. در نسخه 1.16.0، **sd-ai-agent/site-scrape اکنون یک ابزار Tier 1 است** و به‌صورت پیش‌فرض در Theme Builder در دسترس قرار دارد. Superdav AI Agent v1.18.0 همچنین یک بسته همراه Advanced را که به‌صورت جداگانه توزیع می‌شود، برای گردش‌کارهای توسعه‌دهنده مورد اعتماد معرفی می‌کند؛ این ابزارها بخشی از بسته هسته WordPress.org نیستند و باید جداگانه نصب و مجاز شوند.

## ابزارهای موجود

### ابزارهای Tier 1 (همیشه در دسترس)

ابزارهای Tier 1 به‌صورت پیش‌فرض در Theme Builder بدون هیچ پیکربندی اضافی در دسترس هستند.

#### sd-ai-agent/site-scrape

**هدف:** تحلیل وب‌سایت‌های موجود برای استخراج الهام طراحی، ساختار محتوا و عملکرد.

**قابلیت‌ها:**

- **تحلیل وب‌سایت** — خزش و تحلیل وب‌سایت‌های رقیب یا الهام‌بخش
- **استخراج طراحی** — شناسایی رنگ‌ها، فونت‌ها و الگوهای چیدمان
- **ساختار محتوا** — درک سازمان‌دهی و سلسله‌مراتب صفحه
- **تشخیص ویژگی‌ها** — شناسایی پلاگین‌ها و عملکردها
- **تحلیل عملکرد** — بررسی سرعت صفحه و بهینه‌سازی
- **تحلیل SEO** — بازبینی برچسب‌های متا و داده‌های ساختاریافته

**استفاده:**

```
طراحی example.com را تحلیل کن و از آن به‌عنوان الهام برای سایت من استفاده کن.
```

**آنچه استخراج می‌کند:**

- پالت رنگ و نحوه استفاده از رنگ‌ها
- تایپوگرافی (فونت‌ها و اندازه‌ها)
- ساختار چیدمان و فاصله‌گذاری
- الگوهای ناوبری
- سازمان‌دهی محتوا
- استفاده از تصاویر و رسانه
- عناصر تعاملی
- واکنش‌گرایی موبایل

**محدودیت‌ها:**

- نمی‌تواند سایت‌های محافظت‌شده با رمز عبور را خزش کند
- به robots.txt و سیاست‌های سایت احترام می‌گذارد
- ممکن است محتوای پویا را ثبت نکند
- سایت‌هایی که به‌شدت متکی به JavaScript هستند ممکن است تحلیل محدودی داشته باشند
- تحلیل سایت‌های بزرگ ممکن است زمان بیشتری ببرد

### ابزارهای Tier 2 (اختیاری)

ابزارهای Tier 2 زمانی در دسترس هستند که به‌صراحت در تنظیمات Theme Builder فعال شده باشند.

#### تحلیل‌های پیشرفته

شاخص‌های عملکرد دقیق ارائه می‌دهد:

- زمان‌های بارگذاری صفحه
- Core Web Vitals
- امتیاز SEO
- امتیاز دسترس‌پذیری
- امتیاز بهترین رویه‌ها

#### بهینه‌ساز محتوا

برای موارد زیر تحلیل و پیشنهاد بهبود ارائه می‌دهد:

- خوانایی
- بهینه‌سازی SEO
- استفاده از کلمات کلیدی
- ساختار محتوا
- جای‌گذاری فراخوان اقدام

### ابزارهای Tier 3 (Premium)

ابزارهای Tier 3 به پیکربندی اضافی یا ویژگی‌های premium نیاز دارند.

#### تولیدکننده محتوای AI

برای موارد زیر محتوا تولید می‌کند:

- توضیحات محصول
- صفحه‌های خدمات
- نوشته‌های وبلاگ
- توضیحات متا
- متن فراخوان اقدام

#### ابزارهای طراحی پیشرفته

قابلیت‌های طراحی پیشرفته ارائه می‌دهد:

- تولید CSS سفارشی
- ایجاد انیمیشن
- طراحی عنصر تعاملی
- نظریه رنگ پیشرفته
- بهینه‌سازی تایپوگرافی

### ابزارهای همراه Advanced (بسته جداگانه)

بسته همراه Advanced جدا از انتشار هسته Superdav AI Agent توزیع می‌شود. این بسته برای محیط‌های توسعه‌دهنده و مالک سایتِ مورد اعتماد در نظر گرفته شده است که در آن‌ها مدیران به‌صراحت به agent اجازه می‌دهند از ابزارهای با اثرگذاری بالا استفاده کند.

قابلیت‌های همراه Advanced می‌توانند شامل موارد زیر باشند:

- **ابزارهای فایل‌سیستم** — بررسی و تغییر فایل‌های تأییدشده در محیط‌های مورد اعتماد
- **ابزارهای پایگاه داده** — پرس‌وجو یا به‌روزرسانی داده‌های سایت هنگام مجوزدهی صریح
- **ابزارهای WP-CLI** — اجرای فرمان‌های نگهداری و بازرسی WordPress
- **ابزارهای توزیع‌کننده REST** — فراخوانی endpointهای REST ثبت‌شده از طریق گردش‌کارهای توزیع کنترل‌شده
- **ابزارهای سازنده پلاگین** — ایجاد چارچوب اولیه و تکرار روی کد پلاگین
- **ابزارهای snapshot در Git** — ایجاد snapshot پیش از عملیات‌های پرریسک توسعه‌دهنده
- **ابزارهای مدیریت کاربر** — کمک به بازبینی یا تنظیم رکوردهای کاربر در موارد مجاز
- **ابزارهای benchmark** — اجرای benchmarkهای عملکرد یا قابلیت برای بازبینی توسعه

این موارد را به‌عنوان ابزارهای عموماً در دسترس Theme Builder برای هر نصب مستند نکنید. آن‌ها فقط زمانی در دسترس هستند که بسته همراه Advanced نصب، فعال و به مدیران مورد اعتماد محدود شده باشد.

## پیکربندی ابزار

### فعال‌سازی ابزارها

برای فعال‌سازی ابزارهای اضافی در Theme Builder:

1. به **Gratis AI Agent → Settings** بروید
2. به **Theme Builder → Tools** بروید
3. ابزارها را در صورت نیاز روشن/خاموش کنید
4. تنظیمات را ذخیره کنید

### مجوزهای ابزار

برخی ابزارها به مجوز نیاز دارند:

- **خزش سایت** — به دسترسی اینترنت نیاز دارد
- **تحلیل‌ها** — به یکپارچه‌سازی Google Analytics نیاز دارد
- **تولید محتوا** — به کلیدهای API نیاز دارد
- **ویژگی‌های پیشرفته** — ممکن است به اشتراک premium نیاز داشته باشد
- **ابزارهای همراه Advanced** — به بسته Advanced که جداگانه توزیع می‌شود و اعتماد صریح مدیر نیاز دارند

### کلیدهای API و اعتبارنامه‌ها

کلیدهای API را برای ابزارهایی که به آن‌ها نیاز دارند پیکربندی کنید:

1. به **Gratis AI Agent → Settings → API Keys** بروید
2. اعتبارنامه‌ها را برای هر ابزار وارد کنید
3. اتصال را آزمایش کنید
4. به‌صورت امن ذخیره کنید

### نصب بسته همراه Advanced

بسته همراه Advanced جدا از فایل ZIP هسته WordPress.org منتشر می‌شود. آن را فقط از کانال رسمی توزیع پروژه برای انتشار نصب کنید، سپس دسترسی را به مدیران مورد اعتماد محدود کنید. پیش از فعال‌سازی ابزارهای فایل‌سیستم، پایگاه داده، WP-CLI، توزیع‌کننده REST، سازنده پلاگین، مدیریت کاربر یا benchmark روی یک سایت تولیدی، دروازه‌های تأیید انسانی و رفتار snapshot در git را بازبینی کنید.

## استفاده از sd-ai-agent/site-scrape

### استفاده پایه

از Theme Builder بخواهید یک وب‌سایت را تحلیل کند:

```
طراحی سایت رقیب من در competitor.com را تحلیل کن
و بهبودهای طراحی برای سایت من پیشنهاد بده.
```

### تحلیل مشخص

انواع مشخصی از تحلیل را درخواست کنید:

```
پالت رنگ example.com را استخراج کن و از آن به‌عنوان الهام استفاده کن.
```

```
ساختار ناوبری example.com را تحلیل کن و سازمان‌دهی مشابهی را
روی سایت من اعمال کن.
```

```
واکنش‌گرایی موبایل example.com را بررسی کن و مطمئن شو سایت من
به همان اندازه واکنش‌گرا است.
```

### تحلیل مقایسه‌ای

چند سایت را مقایسه کنید:

```
طراحی‌های site1.com و site2.com را مقایسه کن و طراحی‌ای ایجاد کن
که بهترین عناصر هر دو را ترکیب کند.
```

## محدودیت‌ها و ملاحظات ابزار

### محدودسازی نرخ

- استخراج با محدودیت نرخ انجام می‌شود تا از بارگذاری بیش از حد سرور جلوگیری شود
- چندین درخواست به یک سایت ممکن است محدود شوند
- سایت‌های بزرگ ممکن است برای تحلیل زمان بیشتری ببرند

### دقت محتوا

- محتوای پویا ممکن است به‌طور کامل ثبت نشود
- محتوای رندرشده با JavaScript ممکن است ناقص باشد
- داده‌های لحظه‌ای ممکن است منعکس نشوند
- برخی محتواها ممکن است پشت دیوارهای پرداخت باشند

### ملاحظات قانونی و اخلاقی

- به robots.txt و سیاست‌های سایت احترام بگذارید
- محتوای دارای حق نشر را برای بازتولید استخراج نکنید
- از تحلیل برای الهام گرفتن استفاده کنید، نه کپی کردن
- مطمئن شوید حق استفاده از محتوای استخراج‌شده را دارید
- شرایط خدمات سایت‌های تحلیل‌شده را رعایت کنید

### تأثیر بر عملکرد

- استخراج سایت‌های بزرگ ممکن است زمان ببرد
- چندین استخراج هم‌زمان ممکن است کندتر باشد
- اتصال شبکه بر سرعت اثر می‌گذارد
- مجموعه‌داده‌های بزرگ ممکن است به پردازش بیشتری نیاز داشته باشند

## بهترین روش‌ها

### استفاده از تحلیل سایت

1. **چندین سایت را تحلیل کنید** — از چندین منبع الهام جمع‌آوری کنید
2. **بر ساختار تمرکز کنید** — الگوهای چیدمان و سازمان‌دهی را یاد بگیرید
3. **رنگ‌ها را استخراج کنید** — از پالت‌های رنگی به‌عنوان نقاط شروع استفاده کنید
4. **تایپوگرافی را بررسی کنید** — ترکیب‌های فونتی‌ای را که دوست دارید شناسایی کنید
5. **ناوبری را مرور کنید** — سازمان‌دهی منو را درک کنید

### استخراج اخلاقی

1. **به robots.txt احترام بگذارید** — دستورالعمل‌های سایت را دنبال کنید
2. **محتوا را کپی نکنید** — فقط از تحلیل برای الهام گرفتن استفاده کنید
3. **حقوق را بررسی کنید** — مطمئن شوید می‌توانید از محتوای استخراج‌شده استفاده کنید
4. **منابع را ذکر کنید** — به منابع الهام اعتبار بدهید
5. **شرایط را دنبال کنید** — با شرایط خدمات سایت مطابقت داشته باشید

### به حداکثر رساندن اثربخشی ابزار

1. **مشخص باشید** — انواع مشخصی از تحلیل را درخواست کنید
2. **زمینه ارائه دهید** — هدف سایت خود را توضیح دهید
3. **انتظارات را تعیین کنید** — اهداف طراحی خود را توصیف کنید
4. **تکرار کنید** — بر اساس نتایج پالایش کنید
5. **ابزارها را ترکیب کنید** — از چندین ابزار برای تحلیل جامع استفاده کنید

## عیب‌یابی

### سایت استخراج نمی‌شود

- بررسی کنید که آیا سایت به‌صورت عمومی قابل دسترسی است
- بررسی کنید که robots.txt اجازه استخراج می‌دهد
- سایت دیگری را امتحان کنید
- اتصال اینترنت را بررسی کنید
- اگر مشکل ادامه داشت، با پشتیبانی تماس بگیرید

### تحلیل ناقص

- سایت ممکن است محتوای پویا داشته باشد
- JavaScript ممکن است به‌طور کامل رندر نشود
- سایت‌های بزرگ ممکن است با پایان زمان مواجه شوند
- به‌جای آن، تحلیل صفحات مشخص را امتحان کنید
- انواع مشخصی از تحلیل را درخواست کنید

### عملکرد کند

- تحلیل سایت‌های بزرگ زمان بیشتری می‌برد
- چندین استخراج هم‌زمان کندتر هستند
- اتصال شبکه بر سرعت اثر می‌گذارد
- در ساعات کم‌ترافیک امتحان کنید
- ابتدا بخش‌های کوچک‌تر را تحلیل کنید
