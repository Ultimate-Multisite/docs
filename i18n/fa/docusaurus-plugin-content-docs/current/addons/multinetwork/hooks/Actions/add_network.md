# Action: add_network

پس از اضافه شدن یک شبکه جدید، فراخوانی می‌شود.

## پارامترها

| نام | نوع | توضیحات |
|------|------|-------------|
| $new_network_id | `int` | شناسه شبکه‌ای که اضافه شده است. |
| $r | `array` | آرایه کامل و مرتبط از آرگومان‌های شبکه. |

### از نسخه

- 1.3.0
### منبع

در [`inc/functions/network.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-multinetwork/blob/main/inc/functions/network.php#L1304) در خط ۱۳۰۴ تعریف شده است.
