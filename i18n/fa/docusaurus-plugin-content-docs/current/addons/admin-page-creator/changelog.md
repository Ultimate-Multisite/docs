### نسخه 1.9.0 - تاریخ انتشار 2026-01-18 {#version-190---released-on-2026-01-18}

* رفع شد: خطاهای مهلک هنگام فعال‌سازی.
* بازسازی شد: کلاس‌های منبع محتوا برای استفاده از ویژگی Singleton با مقداردهی اولیه صحیح سازنده والد.
* رفع شد: خطاهای کامپایل Vue.js که مانع رندر شدن انتخاب‌کننده نوع محتوا می‌شد.
* رفع شد: حذف شدن ویژگی‌های داده Vue (template_id, external_link_url_embedable, external_link_url_checking).
* رفع شد: عدم مقداردهی اولیه صحیح ویرایشگر TinyMCE هنگام بارگذاری صفحه.
* رفع شد: فایل پشتیبانی Divi که از طریق Composer بارگذاری نمی‌شد.
* رفع شد: وجود نحو PHP خراب در کلاس پشتیبانی Divi.
* بهبود یافت: سازماندهی و کوچک‌سازی (minification) دارایی‌های جاوااسکریپت.
* بهبود یافت: سازگاری با آخرین نسخه‌های WordPress و Page Builderها.
