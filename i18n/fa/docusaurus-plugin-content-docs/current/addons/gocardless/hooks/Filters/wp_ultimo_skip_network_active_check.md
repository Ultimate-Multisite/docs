# فیلتر: wp_ultimo_skip_network_active_check

این فیلتر به توسعه‌دهندگان اجازه می‌دهد تا این بررسی را نادیده بگیرند (short-circuit کنند).

این زمانی مفید است که از تنظیمات سفارشی و مبتنی بر composer، مانند Bedrock، استفاده می‌کنید؛ محیط‌هایی که در آن‌ها استفاده از پلاگین‌ها به عنوان mu-plugins معمول است.

### از زمان

- 2.0.0
### منبع

در [inc/class-requirements.php](https://github.com/Ultimate-Multisite/ultimate-multisite-gocardless/blob/main/inc/class-requirements.php#L229) در خط ۲۲۹ تعریف شده است.
