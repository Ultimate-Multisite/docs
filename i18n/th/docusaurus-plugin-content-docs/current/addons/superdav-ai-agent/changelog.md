# บันทึกการเปลี่ยนแปลง

## 1.16.0 — เผยแพร่เมื่อ 2026-05-20

### คุณสมบัติใหม่

- **ความสามารถในการสร้างโลโก้ SVG** — Theme Builder สามารถสร้างและฝังโลโก้ SVG แบบกำหนดเองได้ พร้อมการทำ sanitisation ที่ปลอดภัย
- **การอัปโหลดรูปภาพในส่วนสัมภาษณ์เพื่อค้นหา (discovery interview)** — Theme Builder discovery interview ตอนนี้มีขั้นตอนการอัปโหลดรูปภาพ เพื่อให้ได้บริบทการออกแบบที่สมบูรณ์ยิ่งขึ้น
- **ความสามารถในการตรวจสอบความเปรียบต่างของสี (Validate Palette Contrast)** — ตรวจสอบคู่สีเพื่อให้เป็นไปตามมาตรฐาน WCAG ก่อนนำไปใช้กับธีม
- **เมนูสำหรับธุรกิจบริการ (Hospitality menus)** — Theme Builder สามารถสร้างหน้าเมนูอาหารและเครื่องดื่มที่มีโครงสร้างสำหรับธุรกิจบริการได้แล้ว
- **การแสดงตัวอย่างบนเดสก์ท็อปและมือถือ (Desktop and mobile preview rendering)** — ดูตัวอย่างการออกแบบของคุณบนอุปกรณ์เดสก์ท็อปและมือถือในระหว่างการเลือกทิศทางการออกแบบ
- **พารามิเตอร์ป้ายกำกับเมนู (Navigation label parameter)** — ความสามารถในการสร้างเมนู (Create Menu) รองรับ `navigation_label` ที่แยกต่างหากจากชื่อหน้า
- **ความพร้อมใช้งานของเครื่องมือ Tier 1** — sd-ai-agent/site-scrape เป็นเครื่องมือ Tier 1 ที่พร้อมใช้งานโดยค่าเริ่มต้นใน Theme Builder

### แก้ไขข้อบกพร่อง

- **แคชของ AI Client** — ตอนนี้ใช้ transients สำหรับการเก็บแคช เพื่อให้ข้อมูลคงอยู่ระหว่างการเรียกใช้ (cross-request persistence) ป้องกันการสูญหายของข้อมูลในงานของ agent ที่รันเป็นเวลานาน
- **ลิงก์การดำเนินการในแถวของ Plugin** — แก้ไขและเปลี่ยนชื่อเพื่อความชัดเจน

## 1.10.0 — เผยแพร่เมื่อ 2026-05-05

### คุณสมบัติใหม่

- **การค้นหาอินเทอร์เน็ตด้วย Tavily** — เพิ่ม Tavily เป็นผู้ให้บริการค้นหา (search provider) เพื่อให้ได้ผลการค้นหาอินเทอร์เน็ตที่สมบูรณ์ยิ่งขึ้น นอกเหนือจาก Brave Search
- **ทักษะในตัวที่รองรับธีม (Theme-aware built-in skills)** — Block Themes, Classic Themes, Kadence Blocks, และ Kadence Theme skill guides มาพร้อมกับ Plugin แล้ว
- **ความสามารถในการสร้างแบบฟอร์มติดต่อด้วย Site builder** — เพิ่มแบบฟอร์มติดต่อ (contact form) ในหน้าใดก็ได้โดยตรงจากหน้าต่างแชท

### ปรับปรุง

- **การปรับปรุง WooCommerce integration** — ตอนนี้ใช้ WooCommerce APIs แบบ Native เพื่อความน่าเชื่อถือและความเข้ากันได้ที่ดีขึ้น
- **รายการผู้ให้บริการจะรีเฟรชโดยอัตโนมัติ** — เมื่อมีการเปิดใช้งานหรือปิดใช้งาน Plugin ใด ๆ

### แก้ไขข้อบกพร่อง

- **ความสามารถ navigate-to** — แก้ไขลูปการโหลดซ้ำไม่สิ้นสุด (infinite reload loop) ในหน้า Admin บางหน้า
- **ความสามารถ list-posts** — แก้ไขการระบุชื่อหมวดหมู่ (category) และแท็ก (tag) ให้เป็น slug อย่างถูกต้อง
- **คำสั่ง WP-CLI** — กู้คืน namespace aliases ที่หายไปหลังจากการปรับปรุงครั้งก่อน
- **ระบบอัตโนมัติของ Event** — จัดการอย่างราบรื่นสำหรับเว็บไซต์ที่ยังไม่ได้สร้างตารางสำหรับระบบอัตโนมัติ (automation tables)
- **ความสามารถ memory-save** — ใช้ namespace prefix ที่ถูกต้องในตัวสร้างคำสั่งระบบ (system instruction builder)
- **ผลลัพธ์ของ Scalar tool** — ห่อผลลัพธ์ของ Scalar tool อย่างถูกต้องก่อนส่งกลับไปยัง AI
- **สถิติการใช้งาน (Usage statistics)** — จัดการรูปแบบคีย์ ability แบบเก่า (legacy ability key format) อย่างถูกต้องเมื่ออัปเกรดจากเวอร์ชันก่อนหน้า
