# บันทึกการเปลี่ยนแปลง

## 1.18.0 — เผยแพร่เมื่อ 2026-06-29

### ใหม่

- **เครื่องมือ Google Calendar** — อ่านปฏิทินและเหตุการณ์ที่กำหนดค่าไว้สำหรับระบบอัตโนมัติที่รับรู้ตารางเวลา
- **การแมปผู้ติดต่อและตัวช่วยผู้เข้าร่วม** — จับคู่ผู้เข้าร่วมปฏิทินกับผู้ใช้และผู้ติดต่อของไซต์
- **ประตูอนุมัติโดยมนุษย์และบันทึกการแจ้งเตือน** — หยุดระบบอัตโนมัติชั่วคราวเพื่อการตรวจสอบและหลีกเลี่ยงการแจ้งเตือนซ้ำ
- **ผู้ให้บริการ SMS TextBee** — ส่งการแจ้งเตือนข้อความตัวอักษรที่กำหนดค่าไว้ผ่าน TextBee
- **แพ็กเกจคู่หูขั้นสูง** — เพิ่มเครื่องมือระบบไฟล์ ฐานข้อมูล WP-CLI ตัวกระจาย REST ตัวสร้างปลั๊กอิน สแนปช็อต git การจัดการผู้ใช้ และเบนช์มาร์กสำหรับนักพัฒนาที่เชื่อถือได้ ซึ่งแจกจ่ายแยกจากบิลด์ WordPress.org

### ปรับปรุง

- **การตั้งค่าบริการ Superdav แบบมีการจัดการ** — เพิ่ม endpoint ของบริการที่โฮสต์และการจัดเตรียมการเชื่อมต่ออัตโนมัติสำหรับไซต์ที่รองรับ
- **การแพ็กเกจรุ่นเผยแพร่** — สร้าง ZIP แยกสำหรับแกนหลักและ Advanced เผยแพร่ทั้งสองบน GitHub และส่งเฉพาะแพ็กเกจแกนหลักไปยัง WordPress.org

### แก้ไข

- **ความน่าเชื่อถือของคำขอ AI** — ปรับปรุงการเลือกโมเดล ระยะหมดเวลาของคำขอ โมเดลตั้งค่าเริ่มต้น การจัดการข้อความการให้เหตุผล และคำแนะนำการลองใหม่สำหรับการเรียกใช้เครื่องมือที่ไม่ถูกต้อง
- **การเสริมความแข็งแกร่งของปฏิทินและการแจ้งเตือน** — เสริมความแข็งแกร่งของโทเค็น Google Calendar และการลดการแจ้งเตือนซ้ำ
- **การเริ่มต้นใช้งานและการกลับมาทำงานต่อหลังอนุมัติ** — แก้ไขการเปิดการเริ่มต้นใช้งานฝั่งหน้าเว็บและการกลับมาทำงานต่อของความสามารถที่ยืนยันแล้ว
- **ปัญหาการตรวจสอบแพ็กเกจของ WordPress.org** — จัดการข้อเสนอแนะจากการตรวจสอบแพ็กเกจสำหรับรุ่นแกนหลัก

## 1.16.0 — เผยแพร่เมื่อ 2026-05-20

### ใหม่

- **ความสามารถสร้าง Logo SVG** — ตอนนี้ Theme Builder สามารถสร้างและฝัง SVG โลโก้แบบกำหนดเองพร้อมการทำความสะอาดที่ปลอดภัยต่อ namespace
- **การอัปโหลดรูปภาพในการสัมภาษณ์ค้นพบ** — ตอนนี้การสัมภาษณ์ค้นพบของ Theme Builder มีขั้นตอนอัปโหลดรูปภาพเพื่อให้บริบทการออกแบบสมบูรณ์ยิ่งขึ้น
- **ความสามารถตรวจสอบคอนทราสต์ของพาเลตต์** — ตรวจสอบคู่สีกับข้อกำหนด WCAG ก่อนนำไปใช้กับธีม
- **เมนูสำหรับธุรกิจบริการ** — ตอนนี้ Theme Builder สามารถสร้างหน้าเมนูอาหารและเครื่องดื่มแบบมีโครงสร้างสำหรับธุรกิจบริการ
- **การเรนเดอร์ตัวอย่างบนเดสก์ท็อปและมือถือ** — ดูตัวอย่างการออกแบบของคุณบนอุปกรณ์เดสก์ท็อปและมือถือระหว่างการเลือกทิศทางการออกแบบ
- **พารามิเตอร์ป้ายกำกับการนำทาง** — ตอนนี้ความสามารถ Create Menu รองรับ `navigation_label` ที่แยกต่างหากจากชื่อหน้า
- **ความพร้อมใช้งานของเครื่องมือระดับ 1** — ตอนนี้ sd-ai-agent/site-scrape เป็นเครื่องมือระดับ 1 ที่พร้อมใช้งานโดยค่าเริ่มต้นใน Theme Builder

### แก้ไข

- **แคช AI Client** — ตอนนี้รองรับด้วย transients เพื่อการคงอยู่ข้ามคำขอ ป้องกันการสูญหายของข้อมูลในงาน agent ที่ทำงานยาวนาน
- **ลิงก์การดำเนินการแถวปลั๊กอิน** — แก้ไขและเปลี่ยนชื่อเพื่อความชัดเจน

## 1.10.0 — เผยแพร่เมื่อ 2026-05-05

### ใหม่

- **การค้นหาอินเทอร์เน็ต Tavily** — เพิ่ม Tavily เป็นผู้ให้บริการค้นหาเพื่อผลการค้นหาอินเทอร์เน็ตที่สมบูรณ์ยิ่งขึ้นควบคู่กับ Brave Search
- **ทักษะในตัวที่รับรู้ธีม** — ตอนนี้คู่มือทักษะ Block Themes, Classic Themes, Kadence Blocks และ Kadence Theme มาพร้อมกับปลั๊กอิน
- **ความสามารถแบบฟอร์มติดต่อของตัวสร้างไซต์** — เพิ่มแบบฟอร์มติดต่อไปยังหน้าใดก็ได้โดยตรงจากอินเทอร์เฟซแชต

### ปรับปรุง

- **ปรับโครงสร้างการผสานรวม WooCommerce ใหม่** — ตอนนี้ใช้ WooCommerce APIs ดั้งเดิมเพื่อความน่าเชื่อถือและความเข้ากันได้ที่ดีขึ้น
- **รายการผู้ให้บริการรีเฟรชโดยอัตโนมัติ** — เมื่อมีการเปิดใช้งานหรือปิดใช้งานปลั๊กอินใดๆ

### แก้ไข

- **ความสามารถ navigate-to** — แก้ไขลูปโหลดซ้ำไม่สิ้นสุดในหน้าผู้ดูแลระบบบางหน้า
- **ความสามารถ list-posts** — ตอนนี้แปลงชื่อหมวดหมู่และแท็กเป็น slug ได้อย่างถูกต้อง
- **คำสั่ง WP-CLI** — กู้คืน alias ของ namespace ที่หายไปหลังจากการปรับโครงสร้างก่อนหน้า
- **ระบบอัตโนมัติเหตุการณ์** — จัดการไซต์ที่ยังไม่ได้สร้างตารางระบบอัตโนมัติอย่างราบรื่น
- **ความสามารถ memory-save** — ตอนนี้ใช้คำนำหน้า namespace ที่ถูกต้องในตัวสร้างคำสั่งระบบ
- **ผลลัพธ์เครื่องมือแบบ Scalar** — ตอนนี้ถูกห่ออย่างถูกต้องก่อนส่งกลับไปยัง AI
- **สถิติการใช้งาน** — ตอนนี้จัดการรูปแบบคีย์ความสามารถแบบเดิมได้อย่างถูกต้องเมื่ออัปเกรดจากเวอร์ชันเก่า
