# Superdav AI Agent

Superdav AI Agent คือผู้ช่วย AI สำหรับ WordPress ขั้นสูงที่ขยาย Gratis AI Agent ด้วยทักษะที่รับรู้ธีม ความสามารถในการค้นหาอินเทอร์เน็ตที่ดียิ่งขึ้น ฟีเจอร์ตัวสร้างไซต์ และการผสานการทำงานอัตโนมัติแบบเลือกใช้ได้ เวอร์ชัน 1.18.0 เพิ่มเครื่องมืออ่าน Google Calendar, การจับคู่ผู้ติดต่อและผู้เข้าร่วม, ด่านการอนุมัติโดยมนุษย์, การตัดรายการแจ้งเตือนซ้ำ, การแจ้งเตือน SMS ผ่าน TextBee, การจัดเตรียมบริการ Superdav ที่มีการจัดการ และแพ็กเกจคู่เสริม Advanced ที่เผยแพร่แยกต่างหากสำหรับเวิร์กโฟลว์ของนักพัฒนาที่เชื่อถือได้

## ฟีเจอร์หลัก {#key-features}

- **ทักษะที่รับรู้ธีม** — ปรับคำแนะนำและความสามารถโดยอัตโนมัติตามธีม WordPress ที่ใช้งานอยู่และปลั๊กอินที่ติดตั้ง
- **ผู้ให้บริการค้นหาอินเทอร์เน็ตหลายราย** — เลือกระหว่าง Brave Search และ Tavily เพื่อผลลัพธ์การค้นหาที่สมบูรณ์ยิ่งขึ้นและความสามารถในการวิจัยขั้นสูง
- **แบบฟอร์มติดต่อของตัวสร้างไซต์** — สร้างแบบฟอร์มติดต่อที่ใช้งานได้เต็มรูปแบบโดยตรงจากอินเทอร์เฟซแชท โดยใช้ปลั๊กอินแบบฟอร์มที่คุณติดตั้ง
- **ตัวสร้างไซต์ที่ปรับปรุงแล้ว** — แผนการสร้างไซต์หลายระยะที่ดีขึ้น พร้อมการค้นหาปลั๊กอินและการกู้คืนข้อผิดพลาดที่ดียิ่งขึ้น
- **ทักษะธีมในตัว** — คำแนะนำเฉพาะทางสำหรับ Block Themes, Classic Themes, Kadence Blocks และ Kadence Theme
- **การผสาน WooCommerce ที่ปรับปรุงแล้ว** — ปรับโครงสร้างใหม่ให้ใช้ API ดั้งเดิมของ WooCommerce เพื่อความน่าเชื่อถือและความเข้ากันได้ที่ดียิ่งขึ้น
- **การตรวจจับผู้ให้บริการอัตโนมัติ** — ตรวจจับและรีเฟรชปลั๊กอินและธีมที่พร้อมใช้งานโดยอัตโนมัติเมื่อมีการเปิดใช้งานหรือปิดใช้งาน
- **การสร้างโลโก้แบบกำหนดเอง** — สร้างและฝังโลโก้ SVG แบบกำหนดเอง พร้อมการทำความสะอาดให้ปลอดภัยต่อ namespace โดยอัตโนมัติ
- **การตรวจสอบคอนทราสต์ของพาเลตต์** — ตรวจสอบคู่สีให้เป็นไปตามข้อกำหนดการเข้าถึง WCAG ก่อนนำไปใช้กับธีมของคุณ
- **การค้นพบที่เสริมด้วยภาพถ่าย** — อัปโหลดภาพถ่ายระหว่างการสัมภาษณ์ค้นพบของ Theme Builder เพื่อมีอิทธิพลต่อทิศทางการออกแบบ
- **เมนูสำหรับธุรกิจบริการ** — สร้างหน้าเมนูอาหารและเครื่องดื่มแบบมีโครงสร้างสำหรับร้านอาหาร คาเฟ่ และธุรกิจบริการ
- **ตัวอย่างเดสก์ท็อปและมือถือ** — ดูตัวอย่างการออกแบบของคุณบนอุปกรณ์เดสก์ท็อปและมือถือระหว่างการเลือกทิศทางการออกแบบ
- **การดึงข้อมูลไซต์ระดับ 1** — sd-ai-agent/site-scrape พร้อมใช้งานตามค่าเริ่มต้นแล้วสำหรับการวิเคราะห์เว็บไซต์คู่แข่งและเว็บไซต์แรงบันดาลใจ
- **เครื่องมืออ่าน Google Calendar** — อ่านปฏิทินและเหตุการณ์ที่กำหนดค่าไว้ เพื่อให้ระบบอัตโนมัติสามารถให้เหตุผลจากบริบทกำหนดการที่กำลังจะมาถึงได้
- **การจับคู่ผู้ติดต่อและผู้เข้าร่วม** — จับคู่ผู้เข้าร่วมในปฏิทินกับผู้ใช้ไซต์และผู้ติดต่อที่รู้จักก่อนสร้างงานติดตามผล
- **ด่านการอนุมัติโดยมนุษย์** — หยุดระบบอัตโนมัติที่ละเอียดอ่อนไว้ชั่วคราวเพื่อให้ตรวจสอบก่อนส่งข้อความ ทำงานต่อ หรือทำการเปลี่ยนแปลง
- **บันทึกการแจ้งเตือนและการตัดรายการซ้ำ** — ติดตามการแจ้งเตือนที่ส่งไปแล้ว เพื่อให้การรันระบบอัตโนมัติซ้ำไม่แจ้งเตือนผู้รับรายเดิมสองครั้ง
- **การแจ้งเตือน SMS ผ่าน TextBee** — ส่งการแจ้งเตือนข้อความที่กำหนดค่าไว้ผ่าน TextBee เมื่อเปิดใช้งานผู้ให้บริการ SMS
- **การจัดเตรียมบริการ Superdav ที่มีการจัดการ** — เชื่อมต่อไซต์ที่รองรับกับ endpoint ของ Superdav ที่โฮสต์ไว้ พร้อมการตั้งค่าอัตโนมัติแทนการกำหนดค่า endpoint ด้วยตนเอง
- **แพ็กเกจคู่เสริม Advanced** — เครื่องมือสำหรับนักพัฒนาที่เชื่อถือได้ซึ่งเผยแพร่แยกต่างหากสำหรับเวิร์กโฟลว์ระบบไฟล์ ฐานข้อมูล WP-CLI ตัวกระจาย REST ตัวสร้างปลั๊กอิน สแนปช็อต git การจัดการผู้ใช้ และการวัดประสิทธิภาพ

## ข้อกำหนด {#requirements}

- WordPress 6.0 หรือสูงกว่า
- PHP 8.1 หรือสูงกว่า
- WordPress แบบหลายไซต์ (เปิดใช้งานระดับเครือข่าย) หรือการติดตั้งแบบไซต์เดียว
- ปลั๊กอินตัวเชื่อมต่อผู้ให้บริการ AI ที่ใช้งานอยู่ (Anthropic, OpenAI หรือที่เข้ากันได้)
- Gratis AI Agent v1.9.0 หรือสูงกว่า (Superdav AI Agent ขยาย Gratis AI Agent)

## การติดตั้ง {#installation}

1. อัปโหลดไฟล์ปลั๊กอินไปยัง `/wp-content/plugins/superdav-ai-agent`
2. เปิดใช้งานปลั๊กอินผ่านหน้าจอ **ปลั๊กอิน** ใน WordPress
3. ตรวจสอบให้แน่ใจว่า Gratis AI Agent ถูกเปิดใช้งานด้วย
4. ไปที่ **Gratis AI Agent → การตั้งค่า** และกำหนดค่าผู้ให้บริการค้นหาอินเทอร์เน็ต การเชื่อมต่อบริการ Superdav ที่มีการจัดการ Google Calendar หรือข้อมูลประจำตัว TextBee ตามต้องการ

แพ็กเกจ WordPress.org มีฟีเจอร์หลักของ Superdav AI Agent ส่วนแพ็กเกจคู่เสริม Advanced ถูกสร้างและเผยแพร่แยกต่างหากสำหรับสภาพแวดล้อมที่มีการติดตั้งและอนุญาตเครื่องมือสำหรับนักพัฒนาที่เชื่อถือได้อย่างชัดเจน

## เริ่มต้นอย่างรวดเร็ว {#quick-start}

เมื่อเปิดใช้งานแล้ว Superdav AI Agent จะเพิ่มประสิทธิภาพการติดตั้ง Gratis AI Agent ของคุณโดยอัตโนมัติด้วยทักษะที่รับรู้ธีมและความสามารถเพิ่มเติม เปิดแผง **Gratis AI Agent** ในแถบด้านข้างผู้ดูแลระบบ WordPress แล้วเริ่มใช้ฟีเจอร์ใหม่

**ตัวอย่างคำสั่ง:**

```
เพิ่มแบบฟอร์มติดต่อไปยังหน้าติดต่อ
```

```
ค้นหาเว็บสำหรับแนวทางปฏิบัติที่ดีที่สุดด้านความปลอดภัย WordPress ล่าสุด
```

```
สร้างส่วน hero โดยใช้รูปแบบบล็อกของธีมที่ใช้งานอยู่
```

```
ตรวจสอบเหตุการณ์ Google Calendar ที่กำหนดค่าไว้ของวันพรุ่งนี้ และร่างการแจ้งเตือนสำหรับผู้เข้าร่วม
```

```
ส่งการแจ้งเตือน SMS หลังจากฉันอนุมัติข้อความ
```

## ทักษะที่รับรู้ธีม {#theme-aware-skills}

Superdav AI Agent ตรวจจับธีมที่ใช้งานอยู่และปลั๊กอินที่ติดตั้งโดยอัตโนมัติ แล้วโหลดทักษะที่รับรู้ธีมที่เหมาะสมเข้าสู่บริบทของผู้ช่วย ทักษะที่พร้อมใช้งานประกอบด้วย:

| ทักษะ | ใช้กับ |
|---|---|
| **Block Themes** | ธีมที่ใช้สถาปัตยกรรมแบบบล็อกของ WordPress (Full Site Editing) |
| **Classic Themes** | ธีม WordPress แบบดั้งเดิมที่ใช้เทมเพลต PHP และตัวแก้ไขแบบคลาสสิก |
| **Kadence Blocks** | ไซต์ที่เปิดใช้งานปลั๊กอิน Kadence Blocks |
| **Kadence Theme** | ไซต์ที่ใช้ธีม Kadence เป็นธีมที่ใช้งานอยู่ |

ดู [ทักษะที่รับรู้ธีม](../../user-guide/configuration/theme-aware-skills) สำหรับคำแนะนำโดยละเอียดเกี่ยวกับแต่ละทักษะ

## ผู้ให้บริการค้นหาอินเทอร์เน็ต {#internet-search-providers}

Superdav AI Agent v1.10.0 เพิ่ม **Tavily** เป็นทางเลือกแทน Brave Search สำหรับคำค้นหาอินเทอร์เน็ต ผู้ให้บริการทั้งสองรายเสนอผลลัพธ์การค้นหาแบบเรียลไทม์และสามารถกำหนดค่าได้ใน **Gratis AI Agent → การตั้งค่า → ขั้นสูง**

| ผู้ให้บริการ | คุณสมบัติ | การตั้งค่า |
|---|---|---|
| **Brave Search** | ผลการค้นหาที่รวดเร็วและเน้นความเป็นส่วนตัว | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | ผลการค้นหาที่สมบูรณ์ยิ่งขึ้น ความสามารถในการค้นคว้าขั้นสูง | [Tavily API](https://tavily.com/) |

ดู [การค้นหาอินเทอร์เน็ต](../../user-guide/configuration/internet-search) สำหรับคำแนะนำการกำหนดค่า

## เวิร์กโฟลว์ปฏิทิน การอนุมัติ และการแจ้งเตือน {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 ขยายเวิร์กโฟลว์อัตโนมัติให้ครอบคลุมมากกว่างานเนื้อหาและการสร้างไซต์:

- **เครื่องมือ Google Calendar** ช่วยให้ agent อ่านปฏิทินที่กำหนดค่าไว้และรายละเอียดกิจกรรมได้เมื่อมีข้อมูลประจำตัวของปฏิทินพร้อมใช้งาน
- **ตัวช่วยผู้ติดต่อและผู้เข้าร่วม** จับคู่ผู้เข้าร่วมกิจกรรมกับผู้ใช้ WordPress หรือผู้ติดต่อที่รู้จักก่อนเตรียมการแจ้งเตือนหรืองานติดตามผล
- **ประตูอนุมัติโดยมนุษย์** ช่วยให้ระบบอัตโนมัติหยุดชั่วคราวจนกว่าผู้ใช้ที่ได้รับอนุญาตจะตรวจสอบการดำเนินการที่เสนอ
- **ระเบียนการแจ้งเตือน** ป้องกันการแจ้งเตือนปฏิทินหรือการแจ้งเตือนซ้ำเมื่อมีการลองระบบอัตโนมัติใหม่หรือจัดกำหนดการซ้ำหลายครั้ง
- **การรองรับ TextBee SMS** เปิดใช้งานการแจ้งเตือนทางข้อความเมื่อกำหนดค่าผู้ให้บริการ TextBee แล้ว

ใช้เครื่องมือเหล่านี้สำหรับความช่วยเหลือที่คำนึงถึงกำหนดการ การติดตามผู้เข้าร่วม และเวิร์กโฟลว์การแจ้งเตือนที่ยังต้องให้มนุษย์ตรวจสอบก่อนส่งข้อความที่ละเอียดอ่อน

## แบบฟอร์มติดต่อของตัวสร้างไซต์ {#site-builder-contact-forms}

agent ตัวสร้างไซต์สามารถสร้างแบบฟอร์มติดต่อได้โดยตรงจากอินเทอร์เฟซแชทแล้ว เพียงขอให้ตัวสร้างไซต์สร้างแบบฟอร์มติดต่อ และระบบจะตรวจหา plugin แบบฟอร์มที่ติดตั้งไว้โดยอัตโนมัติ (Contact Form 7, WPForms, Fluent Forms หรือ Gravity Forms) และสร้างแบบฟอร์มที่พร้อมใช้งาน

ดู [แบบฟอร์มติดต่อของตัวสร้างไซต์](../../user-guide/administration/site-builder-contact-forms) สำหรับคำแนะนำโดยละเอียด

## คุณสมบัติของ Theme Builder {#theme-builder-features}

Theme Builder ใน Superdav AI Agent v1.16.0 มีความสามารถใหม่ที่ทรงพลัง:

### การสัมภาษณ์เพื่อค้นหาความต้องการพร้อมการอัปโหลดรูปภาพ {#discovery-interview-with-photo-upload}

ตอนนี้การสัมภาษณ์เพื่อค้นหาความต้องการของ Theme Builder มีขั้นตอนการอัปโหลดรูปภาพ ซึ่งคุณสามารถให้แรงบันดาลใจด้านภาพได้ อัปโหลดรูปภาพของแบรนด์ของคุณ ไซต์คู่แข่ง หรือแรงบันดาลใจด้านการออกแบบ เพื่อช่วยให้ Theme Builder เข้าใจทิศทางการออกแบบของคุณ

ดู [การสัมภาษณ์เพื่อค้นหาความต้องการ](./theme-builder/discovery-interview.md) สำหรับคำแนะนำโดยละเอียด

### ทิศทางการออกแบบพร้อมตัวอย่างเดสก์ท็อปและมือถือ {#design-direction-with-desktop-and-mobile-previews}

ตรวจทานการออกแบบของคุณทั้งบนอุปกรณ์เดสก์ท็อปและมือถือก่อนที่ Theme Builder จะสร้าง theme ที่สมบูรณ์ของคุณ ดูตัวอย่างแบบเรียลไทม์ที่จุดเปลี่ยน responsive หลายระดับ

ดู [ทิศทางการออกแบบ](./theme-builder/design-direction.md) สำหรับคำแนะนำโดยละเอียด

### เมนูธุรกิจบริการอาหารและเครื่องดื่ม {#hospitality-menus}

ตอนนี้ Theme Builder สามารถสร้างหน้ารายการอาหารและเครื่องดื่มที่มีโครงสร้างสำหรับร้านอาหาร คาเฟ่ บาร์ และธุรกิจบริการอื่น ๆ ได้ เมนูประกอบด้วยราคา คำอธิบาย ข้อมูลโภชนาการตามข้อจำกัดด้านอาหาร และคำเตือนสารก่อภูมิแพ้

ดู [เมนูธุรกิจบริการอาหารและเครื่องดื่ม](./theme-builder/hospitality-menus.md) สำหรับคำแนะนำโดยละเอียด

### ความสามารถใหม่ {#new-abilities}

- **[สร้างโลโก้ SVG](./abilities/generate-logo-svg.md)** — สร้างโลโก้ SVG แบบกำหนดเองพร้อมการล้างข้อมูลอัตโนมัติ
- **[ตรวจสอบคอนทราสต์ของพาเลต](./abilities/validate-palette-contrast.md)** — ตรวจสอบคู่สีเพื่อให้เป็นไปตาม WCAG
- **[สร้างเมนู](./abilities/create-menu.md)** — สร้างเมนูนำทางพร้อมป้ายกำกับการนำทางที่แตกต่างกัน

ดู [เครื่องมือและการกำหนดค่า](./theme-builder/tools.md) สำหรับข้อมูลเกี่ยวกับเครื่องมือที่มีให้ใช้งานและเครื่องมือดึงข้อมูลไซต์ Tier 1

## การกำหนดค่า {#configuration}

การตั้งค่าทั้งหมดอยู่ภายใต้ **Gratis AI Agent → Settings** ในผู้ดูแล WordPress Superdav AI Agent เพิ่มตัวเลือกการกำหนดค่าดังต่อไปนี้:

| การตั้งค่า | คำอธิบาย |
|---|---|
| **ผู้ให้บริการค้นหาอินเทอร์เน็ต** | เลือกระหว่าง Brave Search และ Tavily (ไม่บังคับ) |
| **คีย์ API ของ Tavily** | คีย์ API สำหรับการค้นหาอินเทอร์เน็ตของ Tavily (หากเลือก Tavily) |
| **บริการ Superdav ที่มีการจัดการ** | เชื่อมต่อไซต์ที่รองรับกับบริการ Superdav ที่โฮสต์ไว้ พร้อมการจัดเตรียม endpoint อัตโนมัติ |
| **ข้อมูลประจำตัว Google Calendar** | กำหนดค่าการเข้าถึงปฏิทินเพื่อให้ agent อ่านปฏิทินและกิจกรรมได้ |
| **ผู้ให้บริการ TextBee SMS** | กำหนดค่าข้อมูลประจำตัว TextBee สำหรับเวิร์กโฟลว์การแจ้งเตือน SMS ที่ได้รับอนุมัติ |

## แพ็กเกจคู่หูขั้นสูง {#advanced-companion-package}

Superdav AI Agent v1.18.0 เพิ่มแพ็กเกจคู่หูขั้นสูงที่แจกจ่ายแยกต่างหากสำหรับเวิร์กโฟลว์นักพัฒนาที่เชื่อถือได้ ความสามารถเหล่านี้ไม่ได้รวมอยู่ในแพ็กเกจหลักของ WordPress.org ติดตั้งแพ็กเกจคู่หูเฉพาะบนไซต์ที่ผู้ดูแลระบบไว้วางใจ agent อย่างชัดเจนให้เข้าถึงเครื่องมือสำหรับนักพัฒนา

เครื่องมือขั้นสูงประกอบด้วยการดำเนินการระบบไฟล์ การเข้าถึงฐานข้อมูล WP-CLI เวิร์กโฟลว์ตัวจัดการ REST ยูทิลิตีตัวสร้าง plugin สแนปช็อต git ตัวช่วยจัดการผู้ใช้ และเครื่องมือวัดประสิทธิภาพ จำกัดเครื่องมือเหล่านี้ไว้เฉพาะผู้ดูแลระบบที่เชื่อถือได้ และตรวจทานการกำหนดค่า approval-gate ก่อนเปิดใช้งานบนไซต์ที่ใช้งานจริง

## การปรับปรุงใน v1.18.0 {#improvements-in-v1180}

- **การรองรับ Google Calendar** — อ่านปฏิทินและกิจกรรมที่กำหนดค่าไว้สำหรับระบบอัตโนมัติที่คำนึงถึงกำหนดการ
- **การจับคู่ผู้ติดต่อและผู้เข้าร่วม** — จับคู่ผู้เข้าร่วมปฏิทินกับผู้ใช้ไซต์และผู้ติดต่อ
- **ประตูอนุมัติโดยมนุษย์** — หยุดระบบอัตโนมัติที่ละเอียดอ่อนชั่วคราวเพื่อการตรวจทานก่อนดำเนินการต่อ
- **การป้องกันการแจ้งเตือนซ้ำ** — บันทึกการแจ้งเตือนเพื่อหลีกเลี่ยงการแจ้งเตือนซ้ำในการลองใหม่หรือการรันซ้ำ
- **ผู้ให้บริการ TextBee SMS** — ส่งการแจ้งเตือนทางข้อความที่กำหนดค่าไว้หลังจากได้รับการอนุมัติที่จำเป็น
- **การตั้งค่าบริการ Superdav ที่มีการจัดการ** — จัดเตรียม endpoint บริการที่โฮสต์ไว้โดยอัตโนมัติสำหรับไซต์ที่รองรับ
- **การแจกจ่ายคู่หูขั้นสูง** — เผยแพร่เครื่องมือสำหรับนักพัฒนาที่เชื่อถือได้แยกจากบิลด์หลักของ WordPress.org
- **การแก้ไขความน่าเชื่อถือ** — ปรับปรุงการเลือกโมเดล ระยะหมดเวลาคำขอ การจัดการโมเดลตั้งค่าเริ่มต้น การจัดการข้อความเหตุผล คำแนะนำการลองใหม่ การเสริมความปลอดภัย token ปฏิทิน การเสริมความแข็งแรงการแจ้งเตือน การเริ่มต้น onboarding การกลับมาทำงานของความสามารถที่ยืนยันแล้ว และปัญหาการตรวจทานแพ็กเกจ WordPress.org

## การปรับปรุงใน v1.16.0 {#improvements-in-v1160}

- **แคช AI Client** — ตอนนี้รองรับด้วย transients เพื่อคงอยู่ข้ามคำขอ ป้องกันการสูญเสียข้อมูลในงาน agent ที่ทำงานยาวนาน
- **เครื่องมือดึงข้อมูลเว็บไซต์ Tier 1** — sd-ai-agent/site-scrape พร้อมใช้งานเป็นค่าเริ่มต้นใน Theme Builder สำหรับวิเคราะห์เว็บไซต์คู่แข่งและเว็บไซต์อ้างอิงแรงบันดาลใจ
- **ลิงก์การกระทำในแถว plugin** — แก้ไขและเปลี่ยนชื่อให้ชัดเจนขึ้น

## การปรับปรุงใน v1.10.0 {#improvements-in-v1100}

- **ปรับโครงสร้างการผสานการทำงานกับ WooCommerce ใหม่** — ตอนนี้ใช้ WooCommerce APIs แบบดั้งเดิมเพื่อความน่าเชื่อถือและความเข้ากันได้ที่ดีขึ้น
- **รีเฟรชรายการผู้ให้บริการอัตโนมัติ** — ตรวจจับโดยอัตโนมัติเมื่อ plugins ถูกเปิดใช้งานหรือปิดใช้งาน และอัปเดตรายการผู้ให้บริการที่มีอยู่
- **ตัวสร้างเว็บไซต์ที่ปรับปรุงแล้ว** — ปรับปรุงแผนการสร้างเว็บไซต์หลายระยะพร้อมการกู้คืนจากข้อผิดพลาดที่ดีขึ้น
- **ทักษะที่รับรู้ theme** — ทักษะในตัวใหม่ 4 รายการที่ปรับให้เข้ากับ theme ที่ใช้งานอยู่ของคุณโดยอัตโนมัติ

## การแก้ไขข้อบกพร่องใน v1.10.0 {#bug-fixes-in-v1100}

- แก้ไขความสามารถ `navigate-to` ที่ทำให้เกิดลูปโหลดซ้ำไม่สิ้นสุดบนหน้า admin บางหน้า
- แก้ไขความสามารถ `list-posts` ที่ไม่สามารถแปลงชื่อหมวดหมู่และแท็กเป็น slugs ได้อย่างถูกต้อง
- แก้ไขคำสั่ง WP-CLI ที่ขาด namespace aliases หลังจากการปรับโครงสร้างก่อนหน้า
- แก้ไข event automation ให้จัดการเว็บไซต์ที่ยังไม่ได้สร้างตาราง automation ได้อย่างราบรื่น
- แก้ไขความสามารถ `memory-save` ให้ใช้ namespace prefix ที่ถูกต้องในตัวสร้างคำสั่งระบบ
- แก้ไขผลลัพธ์ของเครื่องมือแบบ scalar ให้ถูกห่อหุ้มอย่างถูกต้องก่อนส่งกลับไปยัง AI
- แก้ไขสถิติการใช้งานให้จัดการรูปแบบคีย์ความสามารถแบบเดิมได้อย่างถูกต้องเมื่ออัปเกรดจากเวอร์ชันเก่า
