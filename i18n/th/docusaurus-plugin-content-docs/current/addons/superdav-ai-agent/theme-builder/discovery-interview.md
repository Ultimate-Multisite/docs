# การสัมภาษณ์เพื่อค้นหาแนวคิด

**การสัมภาษณ์เพื่อค้นหาแนวคิด (Discovery Interview)** คือขั้นตอนแรกในกระบวนการสร้าง Theme Builder โดยจะรวบรวมข้อมูลเกี่ยวกับเว็บไซต์ ธุรกิจ และความชอบด้านการออกแบบของคุณ เพื่อสร้างธีมที่ปรับให้เข้ากับคุณโดยเฉพาะ

## ภาพรวม {#overview}

การสัมภาษณ์เพื่อค้นหาแนวคิดจะนำคุณผ่านชุดคำถามเพื่อทำความเข้าใจเกี่ยวกับแบรนด์ กลุ่มเป้าหมาย และเป้าหมายด้านการออกแบบของคุณ ในเวอร์ชัน 1.16.0 การสัมภาษณ์นี้ได้เพิ่ม **ขั้นตอนการอัปโหลดรูปภาพ** ซึ่งช่วยให้คุณสามารถให้แรงบันดาลใจทางภาพสำหรับการออกแบบได้

## ขั้นตอนการสัมภาษณ์ {#interview-steps}

### 1. ข้อมูลธุรกิจ {#1-business-information}

- ชื่อเว็บไซต์และคำขวัญ
- ประเภทธุรกิจหรืออุตสาหกรรม
- เป้าหมายทางธุรกิจหลัก
- คำอธิบายกลุ่มเป้าหมาย

### 2. ความชอบด้านการออกแบบ {#2-design-preferences}

- ชุดสีที่ต้องการ
- สไตล์การออกแบบ (เช่น โมเดิร์น คลาสสิก มินิมอล สนุกสนาน ฯลฯ)
- เว็บไซต์หรือแบรนด์ที่เป็นแรงบันดาลใจ
- องค์ประกอบการออกแบบที่ต้องมี

### 3. การอัปโหลดรูปภาพ (ใหม่ใน v1.16.0) {#3-photo-upload-new-in-v1160}

อัปโหลดรูปภาพหนึ่งรูปหรือมากกว่า เพื่อเป็นแนวทางในการออกแบบ:

#### รูปแบบที่รองรับ {#accepted-formats}

| รูปแบบ | ส่วนขยาย | รายละเอียด |
|--------|-----------|---------|
| **JPEG** | `.jpg`, `.jpeg` | แนะนำสำหรับภาพถ่าย |
| **PNG** | `.png` | รองรับความโปร่งใส (Transparency) |
| **WebP** | `.webp` | รูปแบบที่ทันสมัย ขนาดไฟล์เล็ก |
| **GIF** | `.gif` | สำหรับภาพนิ่งเท่านั้น (ไม่รองรับแอนิเมชัน) |

#### ข้อกำหนดไฟล์ {#file-requirements}

- **ขนาดไฟล์สูงสุด:** 10 MB
- **ขนาดขั้นต่ำ:** 400 × 300 พิกเซล
- **ขนาดที่แนะนำ:** 1200 × 800 พิกเซล หรือใหญ่กว่า
- **อัตราส่วนภาพ:** ยอมรับทุกอัตราส่วน

#### รูปภาพมีอิทธิพลต่อแนวทางการออกแบบอย่างไร {#how-photos-influence-design-direction}

รูปภาพที่คุณอัปโหลดจะช่วยให้ Theme Builder เข้าใจ:

- **ชุดสี (Color palette)** — จะมีการดึงสีหลักและสีรองมาใช้ในธีม
- **สไตล์ภาพ (Visual style)** — เช่น โมเดิร์น แบบชนบท มินิมอล สดใส ฯลฯ
- **อารมณ์และบรรยากาศ (Mood and atmosphere)** — เช่น มืออาชีพ สร้างสรรค์ เป็นมิตร หรูหรา ฯลฯ
- **องค์ประกอบการออกแบบ (Design elements)** — เช่น ลวดลาย พื้นผิว และสไตล์การจัดองค์ประกอบ
- **แรงบันดาลใจด้านตัวอักษร (Typography inspiration)** — รูปแบบฟอนต์ที่เข้ากับธีมภาพ

### 4. โครงสร้างเนื้อหา {#4-content-structure}

- หน้าหลักที่ต้องการ (หน้าแรก, เกี่ยวกับเรา, บริการ, ติดต่อเรา ฯลฯ)
- ต้องการส่วนบล็อกหรือข่าวสารหรือไม่
- ต้องการฟังก์ชันอีคอมเมิร์ซหรือไม่
- คุณสมบัติพิเศษหรือการเชื่อมต่อที่ต้องการ

### 5. การทบทวนและยืนยัน {#5-review-and-confirmation}

- สรุปข้อมูลทั้งหมดที่ให้มา
- การยืนยันก่อนที่การสร้างการออกแบบจะเริ่มต้น

## แนวทางปฏิบัติที่ดีที่สุดในการอัปโหลดรูปภาพ {#photo-upload-best-practices}

### สิ่งที่ควรอัปโหลด {#what-to-upload}

- **แรงบันดาลใจจากแบรนด์** — โลโก้, ตัวอย่างสี, หรือแนวทางของแบรนด์
- **เว็บไซต์คู่แข่ง** — ภาพหน้าจอของเว็บไซต์ที่คุณชื่นชอบ
- **Mood boards** — ชุดภาพที่รวบรวมเพื่อแสดงถึงสุนทรียภาพที่คุณต้องการ
- **รูปภาพผลิตภัณฑ์** — หากคุณมีสินค้าที่ต้องการนำเสนอ
- **รูปภาพทีมงานหรือสำนักงาน** — สำหรับการสร้างแบรนด์ส่วนตัวหรือองค์กร
- **ภาพไลฟ์สไตล์** — ที่แสดงถึงกลุ่มเป้าหมายของคุณ

### สิ่งที่ควรหลีกเลี่ยง {#what-to-avoid}

- **รูปภาพคุณภาพต่ำ** — รูปภาพที่เบลอหรือแตกเป็นพิกเซล
- **รูปภาพที่ไม่เกี่ยวข้อง** — รูปภาพที่ไม่สื่อถึงแบรนด์ของคุณ
- **รูปภาพที่มีข้อความมากเกินไป** — ภาพหน้าจอที่มีข้อความจำนวนมาก
- **สื่อที่มีลิขสิทธิ์** — ตรวจสอบให้แน่ใจว่าคุณมีสิทธิ์ในการใช้รูปภาพเหล่านั้น
- **GIF แบบเคลื่อนไหว** — ประมวลผลเฉพาะภาพนิ่งเท่านั้น

### เคล็ดลับเพื่อให้ได้ผลลัพธ์ที่ดีที่สุด {#tips-for-best-results}

1. **อัปโหลด 2-3 รูปภาพ** — รูปภาพหลายรูปจะให้บริบทที่ดีกว่า
2. **ใช้รูปภาพคุณภาพสูง** — รูปภาพที่ชัดเจนและมีแสงสว่างเพียงพอจะให้ผลลัพธ์ที่ดีที่สุด
3. **ใส่ความหลากหลาย** — ผสมผสานสี สไตล์ และหัวข้อที่แตกต่างกัน
4. **ให้บริบท** — อธิบายสั้น ๆ ว่าทำไมคุณถึงเลือกรูปภาพแต่ละรูป
5. **ระบุให้ชัดเจน** — หากคุณชอบองค์ประกอบใดเป็นพิเศษ ให้ชี้จุดนั้น ๆ

## ผลลัพธ์แนวทางการออกแบบ {#design-direction-output}

หลังจากเสร็จสิ้นการสัมภาษณ์เพื่อค้นหาแนวคิด Theme Builder จะสร้าง:

- **ชุดสี (Color palette)** — สีหลัก สีรอง และสีเน้น
- **ตัวอักษร (Typography)** — ตระกูลฟอนต์สำหรับหัวข้อและเนื้อหา
- **สไตล์เลย์เอาต์ (Layout style)** — ระยะห่าง การจัดแนว และการจัดวางองค์ประกอบ
- **องค์ประกอบภาพ (Visual elements)** — ไอคอน ลวดลาย และองค์ประกอบตกแต่ง
- **Mood board** — การแสดงภาพแนวทางการออกแบบ

## ขั้นตอนต่อไป {#next-steps}

หลังจากเสร็จสิ้นการสัมภาษณ์เพื่อค้นหาแนวคิด:

1. ตรวจสอบแนวทางการออกแบบที่สร้างขึ้น
2. ให้ข้อเสนอแนะหรือขอการปรับปรุง
3. ดำเนินการไปยังขั้นตอน [Design Direction](./design-direction.md)
4. Theme Builder จะสร้างธีมที่ปรับแต่งให้คุณ
