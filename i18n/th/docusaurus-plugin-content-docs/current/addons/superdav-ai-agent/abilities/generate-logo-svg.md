# สร้างโลโก้ SVG

ความสามารถ **Generate Logo SVG** ช่วยให้ Theme Builder สามารถสร้างและฝังโลโก้ SVG แบบกำหนดเองลงในเว็บไซต์ WordPress ของคุณได้โดยตรง พร้อมการทำความสะอาดชื่อเนมสเปซที่ปลอดภัยโดยอัตโนมัติ

## ภาพรวม {#overview}

ความสามารถนี้จะสร้างโลโก้แบบเวกเตอร์กราฟิก (SVG) ที่ปรับขนาดได้ โดยอิงจากแนวทางการสร้างแบรนด์และความชอบในการออกแบบของเว็บไซต์คุณ SVG ที่สร้างขึ้นจะถูกทำความสะอาด (sanitised) โดยอัตโนมัติเพื่อให้แน่ใจว่าปลอดภัยสำหรับการใช้งานใน WordPress พร้อมทั้งยังคงความสมบูรณ์ทางภาพไว้

## พารามิเตอร์ {#parameters}

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `site_name` | string | Yes | ชื่อเว็บไซต์ที่ต้องการสร้างโลโก้ |
| `style` | string | Yes | รูปแบบการออกแบบ (เช่น "modern", "classic", "minimalist", "playful") |
| `colors` | array | No | อาร์เรย์ของรหัสสีแบบ hex ที่จะใช้ในโลโก้ (เช่น `["#678233", "#ffffff"]`) |
| `width` | number | No | ความกว้างของ SVG เป็นพิกเซล (ค่าเริ่มต้น: 200) |
| `height` | number | No | ความสูงของ SVG เป็นพิกเซล (ค่าเริ่มต้น: 200) |
| `include_text` | boolean | No | ระบุว่าจะรวมชื่อเว็บไซต์เป็นข้อความในโลโก้หรือไม่ (ค่าเริ่มต้น: true) |

## รูปแบบผลลัพธ์ {#output-format}

ความสามารถนี้จะส่งคืนสตริง SVG ที่มีโครงสร้างดังนี้:

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## พฤติกรรมการทำความสะอาด SVG {#svg-sanitisation-behaviour}

SVG ที่สร้างขึ้นจะผ่านการทำความสะอาดชื่อเนมสเปซที่ปลอดภัยโดยอัตโนมัติเพื่อ:

- **ลบแอตทริบิวต์ที่ไม่ปลอดภัย** — ลบ event handlers, scripts, และแอตทริบิวต์ที่อาจเป็นอันตราย
- **รักษาชื่อเนมสเปซ** — คงไว้ซึ่ง SVG namespaces (xmlns, xlink) เพื่อการแสดงผลที่ถูกต้อง
- **ตรวจสอบโครงสร้าง** — ทำให้แน่ใจว่า SVG เป็นไปตามมาตรฐาน W3C
- **เข้ารหัสเอนทิตี้** — การหลบหนีอักขระพิเศษในเนื้อหาข้อความอย่างเหมาะสม
- **ลบการอ้างอิงภายนอก** — ลบสไตล์ชีตภายนอกและการอ้างอิงรูปภาพ

สิ่งนี้ช่วยให้มั่นใจได้ว่า SVG นั้นปลอดภัยที่จะฝังลงใน WordPress ได้โดยตรงโดยไม่จำเป็นต้องมีการทำความสะอาดเพิ่มเติม

## ตัวอย่างการใช้งาน {#usage-example}

**Prompt:**
```
Generate a modern logo for my tech startup called "CloudSync" using blue and white colors.
```

**Result:**
ความสามารถนี้จะสร้างโลโก้ SVG ที่:
- รวมชื่อเว็บไซต์ "CloudSync"
- ใช้ชุดสีน้ำเงินและขาวตามที่ระบุ
- เป็นไปตามหลักการออกแบบที่ทันสมัย
- ถูกทำความสะอาดโดยอัตโนมัติและพร้อมใช้งาน

## การผสานรวมกับ Theme Builder {#integration-with-theme-builder}

เมื่อใช้การเลือกทิศทางการออกแบบของ Theme Builder ความสามารถ Generate Logo SVG จะ:

1. วิเคราะห์ทิศทางการออกแบบและชุดสีของคุณ
2. สร้างโลโก้ SVG แบบกำหนดเองที่ตรงกับความต้องการของคุณ
3. ฝังโลโก้ลงในส่วนหัว/แบรนดิ้งของเว็บไซต์คุณโดยอัตโนมัติ
4. จัดเก็บ SVG เป็นโลโก้แบบกำหนดเองใน WordPress media

## แนวทางปฏิบัติที่ดีที่สุด {#best-practices}

- **ระบุความชอบด้านสไตล์ให้ชัดเจน** — อธิบายรูปแบบการออกแบบที่คุณต้องการ (เช่น modern, classic, playful เป็นต้น)
- **ระบุสี** — ใส่สีแบรนด์ของคุณเพื่อให้เกิดความสม่ำเสมอ
- **ทดสอบการแสดงผล** — ตรวจสอบว่าโลโก้แสดงผลอย่างถูกต้องในขนาดหน้าจอที่แตกต่างกัน
- **ปรับแต่งเพิ่มเติม** — ใช้เครื่องมือปรับแต่งโลโก้ของ WordPress เพื่อปรับขนาดและการจัดวาง

## ข้อจำกัด {#limitations}

- โลโก้ SVG ถูกสร้างเป็นกราฟิกแบบคงที่ (ไม่ใช่อะนิเมชัน)
- โลโก้ที่ซับซ้อนซึ่งมีองค์ประกอบจำนวนมากอาจต้องมีการปรับแต่งด้วยตนเอง
- ไม่รองรับฟอนต์แบบกำหนดเอง; ข้อความจะใช้ฟอนต์ของระบบ
- ขนาดที่ใหญ่หรือเล็กเกินไปอาจส่งผลต่อคุณภาพ
