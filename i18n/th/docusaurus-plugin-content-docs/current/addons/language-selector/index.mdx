# ตัวเลือกภาษา (Language Selector)

Ultimate Multisite: ตัวเลือกภาษาบนหน้าลงทะเบียน

เพิ่มช่องเลือกภาษาในขั้นตอนการลงทะเบียน

## การติดตั้ง

1. อัปโหลด 'ultimate-multisite-language-selector' ไปยังไดเรกทอรี '/wp-content/plugins/'
2. เปิดใช้งาน plugin ผ่านเมนู 'Plugins' ใน WordPress Network Admin ของคุณ

## คู่มือการใช้งาน

ด้วย add-on ฟรีตัวนี้ คุณสามารถให้ผู้ใช้เลือกภาษาที่ต้องการให้แสดงผลได้ทั้งในขั้นตอนการลงทะเบียนและหน้าเข้าสู่ระบบ

หลังจาก [เปิดใช้งาน license ของคุณ](../../user-guide/getting-started/installing-ultimate-multisite.md) แล้ว คุณจะสามารถติดตั้ง Ultimate Multisite add-ons ที่ license ของคุณอนุญาตให้เข้าถึงได้

ไปที่หน้าการตั้งค่า Ultimate Multisite ของคุณ จากนั้นที่เมนูด้านซ้าย ให้คลิกที่ป้ายกำกับ _Add-ons_

![Add-ons page](/img/admin/addons-page.png)

_ไปที่หน้า Add-ons เพื่อเริ่มติดตั้ง Language Selector_

ที่นั่น คุณจะพบ add-ons ทั้งหมดที่เรามีให้

add-ons ฟรีมีให้ลูกค้าของเราทุกคน ไม่ว่าพวกเขาจะสมัครแผนบริการใดก็ตาม

คลิกปุ่ม _Install it Now!_ ที่อยู่ใต้ภาพย่อของ add-on จากนั้นให้เปิดใช้งานในหน้า Plugins

![Add-ons page with install button](/img/admin/addons-page.png)

_คลิกปุ่มเพื่อติดตั้ง add-on_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_เปิดใช้งานในหน้า Plugins ของคุณ_

ตอนนี้ ให้ไปที่เมนู Settings ของ Ultimate Multisite และเลือกแท็บ _Network Settings_ ที่ด้านล่างของหน้า คุณจะพบส่วนใหม่ที่ชื่อว่า _Languages_

![Settings page](/img/config/settings-general.png)

_ปรากฏส่วนใหม่ขึ้นมา!_

คุณสามารถปล่อยให้ว่างไว้เพื่อแสดงภาษาทั้งหมดเป็นตัวเลือก หรือเลือกภาษาที่คุณต้องการให้ใช้งานได้ ภาษาเหล่านี้คือภาษาที่คุณติดตั้งไว้ใน WordPress setup ของคุณ

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_รายการภาษาที่มีให้ใช้งานในการติดตั้ง WP นี้_

หลังจากบันทึกการเปลี่ยนแปลงแล้ว เมนูตัวเลือกที่มีภาษาที่คุณเลือกจะแสดงบนหน้า Login และในขั้นตอนการลงทะเบียนด้วย

![Login and Registration settings](/img/config/settings-login-registration.png)

_ตัวเลือกภาษาบนหน้าเข้าสู่ระบบ_

![Registration page](/img/config/registration-page.png)

_ตัวเลือกภาษาในขั้นตอนการลงทะเบียน_

## การตั้งค่าภาษาผ่าน URL

นอกจากนี้ คุณยังสามารถเปลี่ยนภาษาของหน้าได้โดยการข้ามพารามิเตอร์ locale ไปยัง URL ตัวอย่างเช่น หากคุณต้องการเปลี่ยนเส้นทางไปยังหน้าลงทะเบียนเป็นภาษาอิตาลี ให้เพิ่ม ?locale=it_IT เข้าไปใน URL

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

ภาษาดังกล่าวจะต้องถูกติดตั้งในเครือข่าย WordPress ของคุณจึงจะใช้งานได้
