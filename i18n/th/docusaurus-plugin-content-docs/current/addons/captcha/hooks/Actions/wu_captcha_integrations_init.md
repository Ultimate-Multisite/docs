# Action: wu_captcha_integrations_init

จะทำงานหลังจากที่การเชื่อมต่อ (integrations) ทั้งหมดถูกเริ่มต้นเรียบร้อยแล้ว

การเชื่อมต่อจากภายนอก (Third-party integrations) สามารถใช้ hook ตรงนี้เพื่อเพิ่มการเชื่อมต่อของตัวเองได้
