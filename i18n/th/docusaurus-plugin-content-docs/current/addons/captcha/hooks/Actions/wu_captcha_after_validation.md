# Action: wu_captcha_after_validation

| Name | Type | Description |
|------|------|-------------|
| $result | `array` | ผลลัพธ์ของการตรวจสอบ (validation result) |
| $token | `string` | โทเค็นที่ถูกตรวจสอบ |
