# Action: add_network {#action-addnetwork}

จะทำงานหลังจากที่มีการเพิ่มเครือข่ายใหม่สำเร็จแล้ว

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $new_network_id | `int` | ID ของเครือข่ายที่ถูกเพิ่มเข้ามา |
| $r | `array` | อาร์เรย์แบบ associative ที่มีข้อมูลเครือข่ายทั้งหมด |

### Since {#since}

- 1.3.0
