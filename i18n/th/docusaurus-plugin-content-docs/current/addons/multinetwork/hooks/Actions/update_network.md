# Action: update_network

จะถูกเรียกใช้หลังจากที่มีการอัปเดตเครือข่าย (network) ที่มีอยู่แล้ว

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $network_id | `int` | ID ของเครือข่ายที่ถูกเพิ่มเข้ามา |
| $args | `array` | อาร์เรย์ที่เก็บคู่คีย์และค่าของอาร์กิวเมนต์เครือข่าย |

### Since {#since}

- 1.3.0
