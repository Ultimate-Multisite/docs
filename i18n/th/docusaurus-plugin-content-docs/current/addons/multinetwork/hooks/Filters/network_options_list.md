# Filter: network_options_list {#filter-networkoptionslist}

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $network_options | `array` | รายชื่อตัวเลือกเครือข่ายที่ต้องการ (list of network option names). |
