# บันทึกการเปลี่ยนแปลง Multi-Network {#multi-network-changelog}

### 1.3.0 {#130}
* เพิ่ม: ตัวแสดงตัวอย่างเทมเพลตเครือข่าย — สามารถเรียกดูและดูตัวอย่างเทมเพลตเครือข่ายได้แบบสดๆ ก่อนทำการซื้อ
* เพิ่ม: ตัวเลือกเทมเพลตเครือข่ายที่หน้าชำระเงิน — ลูกค้าสามารถเลือกเทมเพลตเครือข่ายได้ในขั้นตอนการสมัครสมาชิกและการชำระเงิน
* เพิ่ม: SSO ด้วย Magic Link ข้ามเครือข่าย — ลูกค้าสามารถเข้าถึงเครือข่ายย่อยต่างๆ ได้อย่างราบรื่นผ่านการยืนยันตัวตนด้วย Magic Link
* เพิ่ม: การโคลนสื่อเครือข่าย — สื่อของเว็บไซต์จะถูกทำซ้ำอย่างถูกต้องเมื่อโคลนเครือข่ายจากเทมเพลต
* แก้ไข: การโคลนเครือข่ายตอนนี้ใช้งานได้โดยไม่ต้องใช้ add-on multi-tenancy; สิทธิ์ผู้ดูแลระบบสูงสุดจะถูกคืนให้หลังจากการโคลน
* แก้ไข: การสร้างเครือข่ายตอนนี้ใช้งานได้กับ WooCommerce และเกตเวย์การชำระเงินทั้งหมด
* แก้ไข: ตอนนี้ได้รวมหมายเลขพอร์ตไว้ในการสร้างโดเมนสำรองแล้ว
* แก้ไข: ย้าย CSS ของตัวเลือกเทมเพลตไปไว้ใน stylesheet ที่ถูก enqueue เพื่อให้แสดงผลถูกต้องในทุกธีม
* แก้ไข: ป้องกันการใช้คีย์ `network_id` ที่ไม่ได้กำหนดขอบเขตใน site query
* แก้ไข: การตรวจสอบสิทธิ์เครือข่ายก่อนการสลับบริบท (context switching) ป้องกันข้อผิดพลาดคุณสมบัติตาราง `wpdb`
