# Domain Seller {#domain-seller}

**Multisite Ultimate: Domain Seller** เปลี่ยนเครือข่าย WordPress Multisite ของคุณให้เป็นแพลตฟอร์มขายโดเมนที่ทรงพลัง ให้ลูกค้าจดทะเบียนโดเมนแบบกำหนดเองได้โดยตรงระหว่างการสมัครใช้งานหรือจาก Dashboard ของตน พร้อมการตรวจสอบความพร้อมใช้งานแบบเรียลไทม์และราคาที่โปร่งใส

### ทำไมต้องใช้ Domain Seller? {#why-domain-seller}

การจดทะเบียนโดเมนเป็นหนึ่งในข้อเสนอขายเพิ่มที่มีผู้ร้องขอมากที่สุดสำหรับแพลตฟอร์ม WaaS (เว็บไซต์ในรูปแบบบริการ) ใด ๆ แทนที่จะส่งลูกค้าของคุณไปยังผู้รับจดทะเบียนภายนอก คุณสามารถเสนอขายโดเมนได้โดยตรง — ทำให้รายได้อยู่ในเครือข่ายของคุณมากขึ้น พร้อมมอบประสบการณ์ที่ราบรื่นให้ลูกค้า

### คุณสมบัติหลัก {#key-features}

#### รองรับผู้รับจดทะเบียนโดเมนหลายราย {#multiple-domain-registrar-support}
* **OpenSRS** — การจดทะเบียน การต่ออายุ และการจัดการเต็มรูปแบบ
* **Namecheap** — การผสานรวมครบถ้วนกับ WhoisGuard และการทดสอบใน sandbox
* **HostAfrica** — การค้นหาโดเมน ราคา การจดทะเบียน การโอน DNS และการป้องกัน ID ผ่าน Domains Reseller API
* **Openprovider** — ราคาสำหรับผู้ค้าปลีก การจดทะเบียน การต่ออายุ การโอน DNS ความเป็นส่วนตัว WHOIS และการซิงค์ TLD
* **Hostinger** — การค้นหาและจดทะเบียนโดเมนโดยใช้โทเค็นการผสานรวม Hostinger ที่ใช้ร่วมกัน
* **สถาปัตยกรรมที่ขยายได้** — สร้างบนรูปแบบ Integration Registry ทำให้ง่ายต่อการเพิ่มผู้ให้บริการเพิ่มเติม
* **การทดสอบการเชื่อมต่อในตัว** — ตรวจสอบข้อมูลรับรองและการเชื่อมต่อของผู้ให้บริการก่อนเปิดใช้งานจริง

#### ตัวช่วยตั้งค่าแบบมีคำแนะนำ {#guided-setup-wizard}
* **การเริ่มใช้งานในคลิกเดียว** — นำคุณผ่านการกำหนดค่าผู้ให้บริการ การนำเข้า TLD และการสร้างผลิตภัณฑ์โดเมน
* **การซิงค์ TLD อัตโนมัติ** — นำเข้า TLD ที่พร้อมใช้งานทั้งหมดและราคาขายส่งจากผู้ให้บริการที่เชื่อมต่อของคุณ
* **การสร้างผลิตภัณฑ์เริ่มต้น** — สร้างผลิตภัณฑ์โดเมนที่พร้อมใช้งานพร้อมค่าเริ่มต้นของส่วนเพิ่มราคาที่เหมาะสม

#### การตั้งราคาโดเมนที่ยืดหยุ่น {#flexible-domain-pricing}
* **โหมดส่วนเพิ่มราคาสามแบบ** — ส่วนเพิ่มราคาแบบเปอร์เซ็นต์ ส่วนเพิ่มราคาแบบคงที่ หรือราคาคงที่ต่อ TLD
* **ผลิตภัณฑ์เฉพาะ TLD** — สร้างผลิตภัณฑ์โดเมนแยกสำหรับกลุ่ม TLD ต่าง ๆ (.com/.net, .io, รหัสประเทศ ฯลฯ)
* **ราคาแนะนำช่วงแรก** — เสนอราคาปีแรกแบบลดราคาเพื่อดึงดูดการจดทะเบียนใหม่
* **ราคาแบบเรียลไทม์** — ราคาขายส่งสดจากผู้ให้บริการพร้อมการแคชอัจฉริยะ
* **รองรับคูปอง** — เข้ากันได้เต็มรูปแบบกับรหัสส่วนลด Multisite Ultimate

#### การป้องกันความเป็นส่วนตัว WHOIS {#whois-privacy-protection}
* **การกำหนดค่าต่อผลิตภัณฑ์** — ตั้งค่าความเป็นส่วนตัว WHOIS เป็น ปิดใช้งาน, รวมไว้เสมอ หรือ ลูกค้าเลือกเอง
* **การเลือกเข้าร่วมของลูกค้า** — เมื่อตั้งค่าเป็น ลูกค้าเลือกเอง ช่องทำเครื่องหมายในการชำระเงินจะให้ลูกค้าเพิ่มการป้องกันความเป็นส่วนตัวได้
* **ราคาที่กำหนดค่าได้** — ตั้งราคาต่อปีสำหรับส่วนเสริมความเป็นส่วนตัว หรือเสนอให้ฟรี
* **การผสานรวมกับผู้รับจดทะเบียน** — เปิดใช้งาน WhoisGuard (Namecheap) หรือความเป็นส่วนตัว WHOIS (OpenSRS) โดยอัตโนมัติในเวลาจดทะเบียน

#### ประสบการณ์การชำระเงินแบบรวมศูนย์ {#unified-checkout-experience}
* **โหมดโดเมนสามแบบ** — ลูกค้าเลือกระหว่างซับโดเมน (ฟรี), การจดทะเบียนโดเมนใหม่ หรือการแมปโดเมนที่มีอยู่ซึ่งตนเป็นเจ้าของอยู่แล้ว
* **การค้นหาโดเมนแบบเรียลไทม์** — การตรวจสอบความพร้อมใช้งานที่ขับเคลื่อนด้วย AJAX โดยไม่ต้องโหลดหน้าใหม่
* **คำแนะนำอัจฉริยะ** — คำแนะนำ TLD ทางเลือกเมื่อโดเมนที่ต้องการถูกใช้ไปแล้ว
* **ราคาที่โปร่งใส** — แสดงราคาจดทะเบียน ราคาต่ออายุ และค่าธรรมเนียมความเป็นส่วนตัว WHOIS แบบเลือกได้อย่างชัดเจน
* **ช่องข้อมูลติดต่อผู้จดทะเบียน** — รวบรวมข้อมูลผู้จดทะเบียนที่จำเป็น (ชื่อ ที่อยู่ โทรศัพท์) แบบในหน้าเดียวระหว่างการชำระเงิน
* **URL เว็บไซต์ที่สร้างอัตโนมัติ** — ที่อยู่เว็บไซต์จะถูกสร้างโดยอัตโนมัติจากโดเมนที่เลือก

#### ระบบต่ออายุอัตโนมัติ {#auto-renewal-system}
* **การต่ออายุที่ผูกกับสมาชิกภาพ** — โดเมนจะต่ออายุโดยอัตโนมัติขณะที่สมาชิกภาพของลูกค้ายังใช้งานอยู่
* **การจัดกำหนดการอัจฉริยะ** — งานต่ออายุจะถูกจัดคิวตามวันหมดอายุของโดเมน
* **ตรรกะการลองใหม่** — การลองใหม่อย่างชาญฉลาดสำหรับความพยายามต่ออายุที่ล้มเหลว
* **การแจ้งเตือนทางอีเมล** — การแจ้งเตือนที่กำหนดค่าได้สำหรับการต่ออายุที่กำลังจะมาถึง การต่ออายุสำเร็จ ความล้มเหลว และการหมดอายุ

#### การจัดการ DNS ของลูกค้า {#customer-dns-management}
* **DNS แบบบริการตนเอง** — ลูกค้าสามารถเพิ่ม แก้ไข และลบระเบียน DNS (A, AAAA, CNAME, MX, TXT) จาก Dashboard ของตน
* **สถานะโดเมน** — สถานะการจดทะเบียนแบบเรียลไทม์ วันหมดอายุ และข้อมูล nameserver
* **การแมปโดเมนอัตโนมัติ** — โดเมนที่จดทะเบียนหรือโดเมนที่มีอยู่จะถูกแมปไปยังเว็บไซต์ของลูกค้าโดยอัตโนมัติ

#### เครื่องมือผู้ดูแลระบบ {#admin-tools}
* **การจดทะเบียนโดเมนด้วยตนเอง** — จดทะเบียนโดเมนในนามของลูกค้าจากแผงผู้ดูแลระบบ
* **การติดตามโดเมน** — ดูโดเมนของลูกค้าทั้งหมด สถานะ และวันหมดอายุจากหน้าเดียว
* **การบันทึกเฉพาะผู้ให้บริการ** — ช่องบันทึกแยกสำหรับผู้รับจดทะเบียนแต่ละราย (เช่น domain-seller-namecheap.log)
* **รายละเอียดการจดทะเบียน** — ดูข้อมูลผู้จดทะเบียนและข้อมูลการจดทะเบียนบนหน้าแก้ไขโดเมนหลัก

### ผู้ให้บริการที่รองรับ {#supported-providers}

* **OpenSRS** — การผสานรวมเต็มรูปแบบกับการจดทะเบียน การต่ออายุ การจัดการ DNS และความเป็นส่วนตัว WHOIS
* **Namecheap** — การผสานรวมเต็มรูปแบบกับการจดทะเบียน การต่ออายุ การตรวจสอบความพร้อมใช้งานแบบชุด และ WhoisGuard ฟรี
* **HostAfrica** — การจดทะเบียน การต่ออายุ การโอน การจัดการ DNS การดึงรหัส EPP การล็อกผู้รับจดทะเบียน และการป้องกัน ID
* **Openprovider** — การจดทะเบียน การต่ออายุ การโอน การจัดการ DNS การดึงรหัส EPP การล็อกผู้รับจดทะเบียน ความเป็นส่วนตัว WHOIS และการซิงค์ TLD สำหรับผู้ค้าปลีก
* **Hostinger** — การจดทะเบียน การค้นหาโดเมน การอัปเดต nameserver การล็อกผู้รับจดทะเบียน และความเป็นส่วนตัว WHOIS; การต่ออายุจัดการโดยการต่ออายุอัตโนมัติของ Hostinger
* **GoDaddy** — การจดทะเบียนและการต่ออายุ
* **ResellerClub** — การจดทะเบียน การต่ออายุ และการจัดการ DNS
* **NameSilo** — การจดทะเบียนและการต่ออายุ
* **Enom** — การจดทะเบียน การต่ออายุ และการจัดการ DNS

### รายละเอียดทางเทคนิค {#technical-details}

* **ต้องใช้:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **สถาปัตยกรรม:** รูปแบบ Integration Registry พร้อมอินเทอร์เฟซ Domain_Selling_Capability
* **การโหลดอัตโนมัติ:** Jetpack autoloader พร้อมการแมปคลาส PSR-4
* **มาตรฐาน:** มาตรฐานการเขียนโค้ด WordPress, การกำหนดชนิดอย่างเคร่งครัด, prepared statements

## การติดตั้ง {#installation}

1. ตรวจสอบให้แน่ใจว่าได้ติดตั้งและเปิดใช้งานบนเครือข่าย **Multisite Ultimate v2.4.12+** แล้ว
2. อัปโหลด plugin ไปยัง `/wp-content/plugins/ultimate-multisite-domain-seller/`
3. เปิดใช้งาน plugin บนเครือข่ายจาก **Plugins** ใน WordPress Network Admin
4. **Setup Wizard** จะเริ่มทำงานโดยอัตโนมัติเมื่อเปิดใช้งานครั้งแรก — ทำตามขั้นตอนเพื่อ:
   * เชื่อมต่อผู้รับจดทะเบียนโดเมนของคุณ
   * ทดสอบการเชื่อมต่อ
   * นำเข้า TLD ที่พร้อมใช้งานและราคาขายส่ง
   * สร้าง product โดเมนแรกของคุณพร้อมส่วนเพิ่มราคา
5. เพิ่ม field **Domain Selection** ลงในฟอร์ม checkout ของคุณใน **Ultimate Multisite > Checkout Forms**

## คำถามที่พบบ่อย {#frequently-asked-questions}

### ต้องใช้ Multisite Ultimate เวอร์ชันใด? {#what-version-of-multisite-ultimate-is-required}

เวอร์ชัน 2.4.12 หรือสูงกว่า addon นี้ไม่เข้ากันได้กับ WP Ultimo v1.x หรือ Ultimate Multisite เวอร์ชันต่ำกว่า 2.4.12

### รองรับผู้ให้บริการโดเมนรายใดบ้าง? {#which-domain-providers-are-supported}

รองรับผู้รับจดทะเบียนเก้าราย: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo** และ **Enom** ความสามารถจะแตกต่างกันไปตามผู้ให้บริการ เนื่องจาก API ของผู้รับจดทะเบียนเปิดเผยการดำเนินการที่ต่างกัน โปรดดูคู่มือการตั้งค่าสำหรับภาพรวมแยกตามผู้ให้บริการ สถาปัตยกรรม Integration Registry ของ addon ทำให้การเพิ่มผู้ให้บริการแบบกำหนดเองทำได้อย่างตรงไปตรงมา — ดู [คู่มือนักพัฒนา](/developer/integration-guide/custom-registrar)

### ฉันสามารถเสนอการป้องกันความเป็นส่วนตัว WHOIS ได้หรือไม่? {#can-i-offer-whois-privacy-protection}

ได้ product โดเมนแต่ละรายการมีการตั้งค่าความเป็นส่วนตัว WHOIS พร้อมสามโหมด: ปิดใช้งาน, รวมไว้เสมอ หรือให้ลูกค้าเลือก เมื่อกำหนดเป็นให้ลูกค้าเลือก ลูกค้าจะเห็นช่องทำเครื่องหมายระหว่าง checkout พร้อมราคา (หรือ "ฟรี" สำหรับ WhoisGuard ของ Namecheap) แฟล็กความเป็นส่วนตัวจะถูกส่งต่อไปยังผู้รับจดทะเบียนโดยอัตโนมัติในเวลาจดทะเบียน

### ฉันสามารถตั้งราคาที่แตกต่างกันสำหรับ TLD ต่าง ๆ ได้หรือไม่? {#can-i-set-different-pricing-for-different-tlds}

ได้ สร้าง product โดเมนแยกต่างหากสำหรับกลุ่ม TLD ต่าง ๆ product แต่ละรายการสามารถมีประเภทส่วนเพิ่มราคาของตัวเอง (เปอร์เซ็นต์, ส่วนเพิ่มแบบคงที่ หรือราคาคงที่), TLD ที่รองรับ และราคาแนะนำช่วงแรก

### ลูกค้าเห็นตัวเลือก checkout อะไรบ้าง? {#what-checkout-options-do-customers-see}

field Domain Selection ให้ลูกค้ามีสามแท็บ: **Subdomain** (ใช้โดเมนย่อยฟรีบนเครือข่ายของคุณ), **Register New Domain** (ค้นหาและจดทะเบียนโดเมนใหม่) และ **Existing Domain** (แมปโดเมนที่พวกเขาเป็นเจ้าของอยู่แล้ว) เฉพาะโหมดที่คุณเปิดใช้งานเท่านั้นที่จะปรากฏ

### รองรับการต่ออายุอัตโนมัติหรือไม่? {#does-it-support-automatic-renewals}

ได้ โดเมนสามารถตั้งค่าให้ต่ออายุอัตโนมัติได้ และจะผูกกับสถานะ membership ของลูกค้า การชำระเงินต่ออายุจะถูกประมวลผลผ่านเกตเวย์การชำระเงินที่มีอยู่ของลูกค้า

### ลูกค้าสามารถจัดการระเบียน DNS ของตนได้หรือไม่? {#can-customers-manage-their-dns-records}

ได้ ลูกค้าจะได้รับอินเทอร์เฟซจัดการ DNS ใน dashboard ของตนเพื่อเพิ่ม แก้ไข และลบระเบียน A, AAAA, CNAME, MX และ TXT การจัดการ DNS พร้อมใช้งานกับ OpenSRS, ResellerClub, Enom, HostAfrica และ Openprovider ส่วน DNS ของ Hostinger สำหรับโดเมนที่โฮสต์จะจัดการโดยการผสานรวมการแมปโดเมน Hostinger หลัก

### มี setup wizard หรือไม่? {#is-there-a-setup-wizard}

มี setup wizard จะทำงานโดยอัตโนมัติเมื่อเปิดใช้งานครั้งแรก และแนะนำคุณผ่านการเชื่อมต่อผู้รับจดทะเบียน การนำเข้า TLD และการสร้าง product โดเมนแรกของคุณ คุณยังสามารถเรียกใช้ wizard อีกครั้งได้จากการตั้งค่า addon
