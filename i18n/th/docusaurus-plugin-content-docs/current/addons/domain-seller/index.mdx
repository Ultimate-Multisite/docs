# Domain Seller

**Multisite Ultimate: Domain Seller** จะเปลี่ยนเครือข่าย WordPress Multisite ของคุณให้กลายเป็นแพลตฟอร์มที่ทรงพลังสำหรับการขายโดเมน ทำให้ลูกค้าสามารถลงทะเบียนโดเมนแบบกำหนดเองได้โดยตรงระหว่างการสมัครสมาชิกหรือจากแดชบอร์ดของตนเอง พร้อมการตรวจสอบความพร้อมใช้งานแบบเรียลไทม์และราคาที่โปร่งใส

### ทำไมต้องใช้ Domain Seller?

การจดทะเบียนโดเมนเป็นหนึ่งในบริการเสริม (upsell) ที่ลูกค้าต้องการมากที่สุดสำหรับแพลตฟอร์ม WaaS (Website as a Service) ไม่ว่าจะเป็นการส่งลูกค้าของคุณไปยังผู้รับจดทะเบียนภายนอก คุณสามารถเสนอโดเมนได้โดยตรง ซึ่งจะช่วยให้คุณเก็บรายได้ไว้ในเครือข่ายของคุณได้มากขึ้น พร้อมมอบประสบการณ์ที่ราบรื่นให้กับลูกค้า

### คุณสมบัติหลัก

#### รองรับผู้ให้บริการโดเมนหลายราย
* **OpenSRS** — รองรับการจดทะเบียน ต่ออายุ และจัดการโดเมนได้อย่างเต็มรูปแบบ
* **Namecheap** — เชื่อมต่อได้อย่างสมบูรณ์แบบกับ WhoisGuard และการทดสอบแบบ Sandbox
* **สถาปัตยกรรมที่ขยายได้** — สร้างขึ้นบนรูปแบบ Integration Registry ทำให้ง่ายต่อการเพิ่มผู้ให้บริการรายอื่น ๆ
* **การทดสอบการเชื่อมต่อในตัว** — ตรวจสอบข้อมูลประจำตัวและสถานะการเชื่อมต่อของผู้ให้บริการก่อนใช้งานจริง

#### ตัวช่วยตั้งค่าแบบมีขั้นตอน (Guided Setup Wizard)
* **การเริ่มต้นใช้งานด้วยคลิกเดียว** — นำทางคุณตลอดการตั้งค่าผู้ให้บริการ, การนำเข้า TLD, และการสร้างผลิตภัณฑ์โดเมน
* **การซิงค์ TLD อัตโนมัติ** — นำเข้า TLD และราคาขายส่งทั้งหมดจากผู้ให้บริการที่คุณเชื่อมต่อไว้
* **การสร้างผลิตภัณฑ์เริ่มต้น** — สร้างผลิตภัณฑ์โดเมนที่พร้อมใช้งานทันที พร้อมค่า Markup เริ่มต้นที่เหมาะสม

#### การกำหนดราคาโดเมนที่ยืดหยุ่น
* **โหมด Markup สามแบบ** — กำหนด Markup เป็นเปอร์เซ็นต์, กำหนดราคาคงที่, หรือราคาคงที่ต่อ TLD
* **ผลิตภัณฑ์เฉพาะ TLD** — สร้างผลิตภัณฑ์โดเมนแยกกันสำหรับกลุ่ม TLD ที่แตกต่างกัน (.com/.net, .io, โค้ดประเทศ ฯลฯ)
* **ราคาแนะนำ** — เสนอราคาปีแรกที่ลดพิเศษเพื่อดึงดูดการลงทะเบียนใหม่
* **ราคาแบบเรียลไทม์** — แสดงราคาขายส่งสดจากผู้ให้บริการ พร้อมระบบแคชอัจฉริยะ
* **รองรับคูปอง** — เข้ากันได้เต็มรูปแบบกับรหัสส่วนลดของ Multisite Ultimate

#### การป้องกันความเป็นส่วนตัว WHOIS
* **การตั้งค่าต่อผลิตภัณฑ์** — กำหนดความเป็นส่วนตัว WHOIS เป็น Disabled, Always Included, หรือ Customer Choice
* **การเลือกโดยลูกค้า** — เมื่อตั้งค่าเป็น Customer Choice ช่องทำเครื่องหมายในการชำระเงินจะให้ลูกค้าเพิ่มการป้องกันความเป็นส่วนตัว
* **การกำหนดราคาได้** — กำหนดราคาต่อปีสำหรับส่วนเสริมความเป็นส่วนตัว หรือเสนอให้ฟรี
* **การเชื่อมต่อกับ Registrar** — เปิดใช้งาน WhoisGuard (Namecheap) หรือ WHOIS privacy (OpenSRS) โดยอัตโนมัติเมื่อมีการลงทะเบียน

#### ประสบการณ์การชำระเงินที่ครบวงจร
* **สามโหมดโดเมน** — ลูกค้าสามารถเลือกได้ระหว่าง Subdomain (ฟรี), การลงทะเบียนโดเมนใหม่, หรือการแมปโดเมนที่มีอยู่แล้วที่ลูกค้าเป็นเจ้าของ
* **การค้นหาโดเมนแบบเรียลไทม์** — การตรวจสอบความพร้อมใช้งานด้วย AJAX โดยไม่ต้องโหลดหน้าใหม่
* **คำแนะนำอัจฉริยะ** — เสนอ TLD ทางเลือกเมื่อโดเมนที่ต้องการถูกใช้งานแล้ว
* **ราคาที่โปร่งใส** — แสดงราคาการลงทะเบียน, ราคาต่ออายุ, และค่าธรรมเนียม WHOIS privacy (ถ้ามี) อย่างชัดเจน
* **ช่องข้อมูลผู้ลงทะเบียน** — รวบรวมข้อมูลผู้ลงทะเบียนที่จำเป็น (ชื่อ, ที่อยู่, โทรศัพท์) ในหน้าชำระเงิน
* **URL ไซต์ที่สร้างอัตโนมัติ** — ที่อยู่เว็บไซต์จะถูกกำหนดโดยอัตโนมัติจากโดเมนที่เลือก

#### ระบบต่ออายุอัตโนมัติ
* **การต่ออายุที่ผูกกับสมาชิกภาพ** — โดเมนจะต่ออายุโดยอัตโนมัติในขณะที่สมาชิกภาพของลูกค้ายังใช้งานอยู่
* **การจัดตารางเวลาอัจฉริยะ** — งานต่ออายุจะถูกจัดคิวตามวันหมดอายุของโดเมน
* **ตรรกะการลองใหม่ (Retry logic)** — พยายามต่ออายุอย่างชาญฉลาดเมื่อการต่ออายุล้มเหลว
* **การแจ้งเตือนทางอีเมล** — การแจ้งเตือนที่กำหนดค่าได้สำหรับการต่ออายุที่กำลังจะมาถึง, การต่ออายุที่สำเร็จ, ความล้มเหลว, และการหมดอายุ

#### การจัดการ DNS ของลูกค้า
* **DNS แบบบริการตนเอง** — ลูกค้าสามารถเพิ่ม, แก้ไข, และลบระเบียน DNS (A, AAAA, CNAME, MX, TXT) ได้จากแดชบอร์ดของตนเอง
* **สถานะโดเมน** — สถานะการลงทะเบียน, วันหมดอายุ, และข้อมูล nameserver แบบเรียลไทม์
* **การแมปโดเมนอัตโนมัติ** — โดเมนที่ลงทะเบียนหรือโดเมนที่มีอยู่จะถูกแมปไปยังเว็บไซต์ของลูกค้าโดยอัตโนมัติ

#### เครื่องมือสำหรับผู้ดูแลระบบ (Admin Tools)
* **การลงทะเบียนโดเมนด้วยตนเอง** — ลงทะเบียนโดเมนในนามของลูกค้าจากแผงผู้ดูแลระบบ
* **การตรวจสอบโดเมน** — ดูโดเมนทั้งหมดของลูกค้า, สถานะ, และวันหมดอายุได้จากหน้าเดียว
* **การบันทึกข้อมูลเฉพาะผู้ให้บริการ** — ช่องบันทึกแยกสำหรับผู้รับจดทะเบียนแต่ละราย (เช่น domain-seller-namecheap.log)
* **รายละเอียดการลงทะเบียน** — ดูข้อมูลผู้ลงทะเบียนและข้อมูลการจดทะเบียนในหน้าแก้ไขโดเมนหลัก

### ผู้ให้บริการที่รองรับ

* **OpenSRS** — เชื่อมต่อเต็มรูปแบบสำหรับการจดทะเบียน, ต่ออายุ, การจัดการ DNS, และความเป็นส่วนตัว WHOIS
* **Namecheap** — เชื่อมต่อเต็มรูปแบบสำหรับการจดทะเบียน, ต่ออายุ, การตรวจสอบความพร้อมใช้งานแบบกลุ่ม, และ WhoisGuard ฟรี
* **GoDaddy** — การจดทะเบียนและต่ออายุ
* **ResellerClub** — การจดทะเบียน, ต่ออายุ, และการจัดการ DNS
* **NameSilo** — การจดทะเบียนและต่ออายุ
* **Enom** — การจดทะเบียน, ต่ออายุ, และการจัดการ DNS

### รายละเอียดทางเทคนิค

* **ต้องใช้:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **สถาปัตยกรรม:** Integration Registry pattern พร้อมด้วย Domain_Selling_Capability interface
* **Autoloading:** Jetpack autoloader พร้อมการแมปคลาส PSR-4
* **มาตรฐาน:** มาตรฐานการเขียนโค้ดของ WordPress, strict typing, prepared statements

## การติดตั้ง

1. ตรวจสอบให้แน่ใจว่าได้ติดตั้ง **Multisite Ultimate v2.4.12+** และเปิดใช้งานในระดับเครือข่ายแล้ว
2. อัปโหลดปลั๊กอินไปยัง `/wp-content/plugins/ultimate-multisite-domain-seller/`
3. เปิดใช้งานปลั๊กอินในระดับเครือข่ายจากเมนู **Plugins** ใน WordPress Network Admin
4. **Setup Wizard** จะเปิดใช้งานโดยอัตโนมัติในการเปิดใช้งานครั้งแรก — ทำตามขั้นตอนเพื่อ:
   * เชื่อมต่อผู้รับจดทะเบียนโดเมนของคุณ (OpenSRS หรือ Namecheap)
   * ทดสอบการเชื่อมต่อ
   * นำเข้า TLD และราคาขายส่งที่มีอยู่
   * สร้างผลิตภัณฑ์โดเมนแรกของคุณพร้อมการกำหนด Markup ราคา
5. เพิ่มฟิลด์ **Domain Selection** ลงในแบบฟอร์มการชำระเงินของคุณใน **Ultimate Multisite > Checkout Forms**

## คำถามที่พบบ่อย

### ต้องใช้ Multisite Ultimate เวอร์ชันใด?

เวอร์ชัน 2.4.12 หรือสูงกว่า ปลั๊กอินเสริมนี้ไม่รองรับ WP Ultimo v1.x หรือ Ultimate Multisite เวอร์ชันที่ต่ำกว่า 2.4.12

### รองรับผู้ให้บริการโดเมนรายใดบ้าง?

รองรับผู้รับจดทะเบียนครบถ้วน 6 ราย ได้แก่: **OpenSRS**, **Namecheap**, **GoDaddy**, **ResellerClub**, **NameSilo**, และ **Enom** สถาปัตยกรรม Integration Registry ของปลั๊กอินเสริมนี้ทำให้การเพิ่มผู้ให้บริการแบบกำหนดเองเป็นเรื่องง่าย — ดู [developer guide](/developer/integration-guide/custom-registrar)

### ฉันสามารถเสนอการป้องกันความเป็นส่วนตัว WHOIS ได้หรือไม่?

ได้ค่ะ แต่ละผลิตภัณฑ์โดเมนมีการตั้งค่า WHOIS Privacy พร้อมโหมดสามแบบ: Disabled, Always Included, หรือ Customer Choice เมื่อตั้งค่าเป็น Customer Choice ลูกค้าจะเห็นช่องทำเครื่องหมายในหน้าชำระเงินพร้อมราคา (หรือ "ฟรี" สำหรับ WhoisGuard ของ Namecheap) โดยแฟล็กความเป็นส่วนตัวจะถูกส่งไปยังผู้รับจดทะเบียนโดยอัตโนมัติเมื่อมีการลงทะเบียน

### ฉันสามารถกำหนดราคาที่แตกต่างกันสำหรับ TLD ต่างๆ ได้หรือไม่?

ได้ค่ะ คุณสามารถสร้างผลิตภัณฑ์โดเมนแยกกันสำหรับกลุ่ม TLD ที่แตกต่างกัน แต่ละผลิตภัณฑ์สามารถมีประเภท Markup ของตัวเอง (เปอร์เซ็นต์, กำหนดราคาคงที่, หรือราคาคงที่) TLD ที่รองรับ และราคาแนะนำ

### ลูกค้าจะเห็นตัวเลือกการชำระเงินอะไรบ้าง?

ฟิลด์ Domain Selection จะให้ลูกค้าสามแท็บ: **Subdomain** (ใช้ Subdomain ฟรีในเครือข่ายของคุณ), **Register New Domain** (ค้นหาและลงทะเบียนโดเมนใหม่), และ **Existing Domain** (แมปโดเมนที่ลูกค้าเป็นเจ้าของอยู่แล้ว) จะแสดงเฉพาะโหมดที่คุณเปิดใช้งานเท่านั้น

### รองรับการต่ออายุอัตโนมัติหรือไม่?

ได้ค่ะ โดเมนสามารถตั้งค่าให้ต่ออายุอัตโนมัติและผูกกับสถานะสมาชิกภาพของลูกค้า การชำระเงินต่ออายุจะดำเนินการผ่าน Payment Gateway ที่ลูกค้ามีอยู่

### ลูกค้าสามารถจัดการระเบียน DNS ของตนเองได้หรือไม่?

ได้ค่ะ ลูกค้าจะได้รับอินเทอร์เฟซการจัดการ DNS ในแดชบอร์ดเพื่อเพิ่ม, แก้ไข, และลบระเบียน A, AAAA, CNAME, MX, และ TXT การจัดการ DNS สามารถใช้ได้กับ OpenSRS, ResellerClub, และ Enom

### มี Setup Wizard หรือไม่?

มีค่ะ Setup wizard จะทำงานโดยอัตโนมัติในการเปิดใช้งานครั้งแรก และจะนำทางคุณตลอดการเชื่อมต่อผู้รับจดทะเบียน, การนำเข้า TLD, และการสร้างผลิตภัณฑ์โดเมนแรก คุณยังสามารถรัน wizard ซ้ำได้จากส่วนการตั้งค่าของปลั๊กอินเสริม
