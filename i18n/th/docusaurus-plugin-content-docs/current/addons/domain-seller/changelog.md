# บันทึกการเปลี่ยนแปลงของ Domain Seller

Version 1.1.0 - เผยแพร่เมื่อ 2026-05-08
- ใหม่: เพิ่มฟังก์ชันการสร้าง DNS record (add_dns_record) สำหรับผู้ให้บริการจดโดเมนอย่าง ResellerClub, Enom, และ OpenSRS
- แก้ไข: ตัวแยกวิเคราะห์ (parser) DNS record เริ่มต้น สามารถรองรับโทเค็น {DOMAIN} และ {SITE_URL} ได้แล้ว
- แก้ไข: ชื่อ slug ของฟิลด์การเลือกโดเมนในหน้าชำระเงินถูกกำหนดขอบเขต (namespaced) เพื่อป้องกันการชนกับ site_url

Version 1.0.8 - เผยแพร่เมื่อ 2026-05-07
- แก้ไข: การคำนวณราคาโดเมนของ ResellerClub ตอนนี้จะดึงราคาต้นทุนแบบเรียลไทม์จาก API endpoint ที่ถูกต้องแล้ว

Version 1.0.7 - เผยแพร่เมื่อ 2026-05-06
* แก้ไข: การทดสอบการเชื่อมต่อ (test_connection) ของ ResellerClub ส่งพารามิเตอร์ tlds ที่จำเป็น (#224) แล้ว

Version 1.0.6 - เผยแพร่เมื่อ 2026-05-05
* แก้ไข: การจดโดเมนของ ResellerClub ทำงานได้อย่างถูกต้องแล้ว ด้วยการปรับปรุงการจัดการ API response และการกำหนดเส้นทางผู้ให้บริการตาม registry
* แก้ไข: ปรับปรุงประสบการณ์ผู้ใช้ (UX) ของหน้าจัดการโดเมน (Register Domain admin page)
* ลบ: การเชื่อมต่อกับ registrar CyberPanel

Version 1.0.5 - เผยแพร่เมื่อ 2026-04-02
* ใหม่: การเชื่อมต่อกับ registrar GoDaddy สำหรับการจดและจัดการโดเมน
* ใหม่: การเชื่อมต่อกับ registrar NameSilo
* ใหม่: การเชื่อมต่อกับ registrar ResellerClub
* ใหม่: การตรวจสอบและแมปโดเมนที่ส่ง SES โดยอัตโนมัติเมื่อมีการซื้อโดเมน
* แก้ไข: ป้องกันไม่ให้ค่าคงที่ (constants) ของ plugin Guard ถูกนิยามซ้ำในสภาพแวดล้อมทดสอบ
* แก้ไข: MySQL flags word-split ได้อย่างถูกต้องใน install-wp-tests.sh

* **ปรับปรุง:** คำอธิบายโดยรวมในหน้า settings ดีขึ้น (ค้นหา TLDs, การต่ออายุ, DNS, การแจ้งเตือน)

* **ปรับปรุง:** API ราคาของ Namecheap ใช้พารามิเตอร์ที่ถูกต้องและการแยกวิเคราะห์ response
* **ปรับปรุง:** การจัดเก็บ TLD แบบรวมศูนย์ใน network option เดียว
* **ปรับปรุง:** การบันทึกกิจกรรมโดเมนสำหรับการเปลี่ยนแปลง DNS, การโอนย้าย, และการใช้ config
* **ปรับปรุง:** การซิงค์ TLD เต็มรูปแบบสำหรับ OpenSRS โดยใช้ IANA master list พร้อมการตรวจสอบแบบ batch
* **ปรับปรุง:** การซิงค์ TLD เต็มรูปแบบสำหรับ Namecheap ด้วยการร้องขอ API แบบ paginated

* **ปรับปรุง:** หน้า settings ที่มีการตั้งค่า DNS และการโอนย้าย
* **ปรับปรุง:** หมายเลขโทรศัพท์ถูกจัดรูปแบบเป็น +CC.NNN ของ registrar โดยอัตโนมัติ
* **ปรับปรุง:** การตรวจสอบฟิลด์โทรศัพท์จะลบอักขระจัดรูปแบบก่อนส่งข้อมูล
* **ปรับปรุง:** อัปเกรดข้อกำหนดเวอร์ชันเป็น Ultimate Multisite 2.4.12 พร้อมการแจ้งเตือนที่ชัดเจนขึ้น
* **ปรับปรุง:** CI workflow ใช้ checkout ที่เหมาะสมสำหรับทั้งปลั๊กอินเสริมและปลั๊กอินหลัก
* **ปรับปรุง:** prepare_registrant_info() อ่านจาก user meta ที่บันทึกใน checkout โดยมี billing address เป็นค่าสำรอง
* **แก้ไข:** AJAX ของการค้นหาโดเมนล้มเหลวสำหรับผู้ใช้ที่ไม่ได้ล็อกอินในระหว่างขั้นตอนการซื้อ
* **แก้ไข:** AJAX ของการคำนวณราคาโดเมนล้มเหลวสำหรับผู้ใช้ที่ไม่ได้ล็อกอินในระหว่างขั้นตอนการซื้อ

* **แก้ไข:** Timeout ของ API sandbox ของ Namecheap สั้นเกินไป
* **แก้ไข:** ข้อความปุ่ม Select ของการค้นหาโดเมนไม่แสดงบนพื้นหลังสีเขียว
* **แก้ไข:** การจดโดเมนล้มเหลวด้วยข้อผิดพลาด "RegistrantFirstName is Missing" เนื่องจากข้อมูลติดต่อขาดหายไป
* **แก้ไข:** สร้าง record โดเมนด้วย blog_id=0 เมื่อ site ยังไม่มีอยู่ ณ เวลาที่ checkout
* **แก้ไข:** การตั้งค่า TLD เริ่มต้นถูกส่งกลับเป็น string แทนที่จะเป็น array ที่ถูกแยกวิเคราะห์
* **ลบ:** หน้า admin การจัดการโดเมนแบบเดี่ยว — ตอนนี้จัดการผ่านหน้าโดเมนหลักแล้ว
