# บันทึกการเปลี่ยนแปลง Domain Seller {#domain-seller-changelog}

Version 1.3.0 - เผยแพร่เมื่อ 2026-06-02
- ใหม่: เพิ่มคำเตือนใน network-admin เมื่อยอดคงเหลือของตัวแทนจำหน่าย HostAfrica ต่ำเกินไป
- ใหม่: เพิ่มการแมปอัตโนมัติของโดเมนที่ลงทะเบียนใหม่ไปยังเว็บไซต์เครือข่าย
- แก้ไข: ใช้ข้อกำหนดของฟิลด์ผู้ลงทะเบียนเฉพาะเมื่อลงทะเบียนโดเมนใหม่เท่านั้น
- แก้ไข: ทำให้การแจ้งเตือนการตรวจสอบยอดคงเหลือสามารถปิดได้
- แก้ไข: รับรองว่ารายละเอียดการเรียกเก็บเงินของผู้ลงทะเบียน WooCommerce จะถูกเก็บรักษาไว้
- แก้ไข: บังคับใช้ข้อกำหนดข้อมูลติดต่อของผู้ลงทะเบียนระหว่างการลงทะเบียน
- แก้ไข: ป้องกันไม่ให้ผลิตภัณฑ์ลงทะเบียนโดเมนถูกสร้างด้วยส่วนเพิ่มราคา 0%
- แก้ไข: เก็บรักษาการเลือกโดเมนและราคาไว้ตลอดกระบวนการเซสชันชำระเงิน
- แก้ไข: ปรับปรุงการแสดงสกุลเงินของราคาค่าโดเมน HostAfrica
- แก้ไข: ปรับปรุงพฤติกรรม form-action ของการชำระเงินเพื่อป้องกันความไม่ตรงกันของ query-var ในแกน WP
- ปรับปรุง: ลิงก์เอกสารการกำหนดค่าตัวแทนจำหน่าย HostAfrica ในคำแนะนำการตั้งค่า

Version 1.2.0 - เผยแพร่เมื่อ 2026-05-25
- ใหม่: เพิ่ม HostAfrica เป็นการผสานการขายโดเมน พร้อมการรองรับการชำระเงิน ตัวช่วยตั้งค่า การค้นหา TLD/ราคา การลงทะเบียน การต่ออายุ การโอน nameserver, DNS, EPP code, registrar lock และ ID protection
- ใหม่: เพิ่ม Openprovider เป็นการผสานการขายโดเมน พร้อมการรองรับราคาตัวแทนจำหน่าย การลงทะเบียน การต่ออายุ การโอน nameserver, DNS, EPP code, registrar lock, WHOIS privacy และการซิงก์ TLD
- ใหม่: เพิ่ม Hostinger เป็นการผสานการขายโดเมน โดยใช้ Hostinger API token ที่ใช้ร่วมกันจากการผสานหลัก สำหรับการตรวจสอบความพร้อมใช้งาน การลงทะเบียน การอัปเดต nameserver, registrar lock และ WHOIS privacy
- ปรับปรุง: เพิ่ม docblocks ของ action และ filter สำหรับวงจรชีวิตโดเมน เพื่อเป็นคำแนะนำการผสานสำหรับนักพัฒนา
- ปรับปรุง: อัปเดตเมทาดาทาความเข้ากันได้ของ plugin เป็น WordPress 7.0 ใน readme ของ addon
- ปรับปรุง: อัปเดตเทมเพลตการวางแผนที่ใช้สำหรับการประสานงานรุ่นที่จะมาถึง

Version 1.1.0 - เผยแพร่เมื่อ 2026-05-08

- แก้ไข: ตัวแยกวิเคราะห์ระเบียน DNS เริ่มต้นตอนนี้รองรับโทเค็น {DOMAIN} และ {SITE_URL}

Version 1.0.1 - เผยแพร่เมื่อ 2026-02-27

* **ใหม่:** เครื่องมือนำเข้า TLD สำหรับการจัดการราคาแบบจำนวนมาก
* **ใหม่:** รองรับราคาช่วงแนะนำสำหรับผลิตภัณฑ์โดเมน
* **ใหม่:** ชุดทดสอบ E2E ด้วย Cypress
* **ใหม่:** เทมเพลตอีเมลสำหรับการแจ้งเตือนวงจรชีวิตโดเมน
* **ใหม่:** ช่องที่อยู่ผู้จดทะเบียนในโมดัลการจดทะเบียนโดเมนฝั่งผู้ดูแลระบบ โดยกรอกล่วงหน้าจากการตั้งค่า
* **ใหม่:** อินเทอร์เฟซจัดการ DNS ของลูกค้า พร้อมรองรับการเพิ่ม แก้ไข และลบระเบียน
* **ใหม่:** ตัวเลือกชำระเงิน "นำโดเมนของคุณเองมาใช้" พร้อมการแมปโดเมนอัตโนมัติ
* **ใหม่:** สร้าง URL ของเว็บไซต์โดยอัตโนมัติจากชื่อโดเมนระหว่างชำระเงิน
* **ใหม่:** การกำหนดค่า nameserver และระเบียน DNS เริ่มต้นในการตั้งค่า
* **ใหม่:** รายละเอียดการจดทะเบียนโดเมนและการจัดการ DNS บนหน้าหลักสำหรับแก้ไขโดเมน
* **ใหม่:** ตัวช่วยตั้งค่าสร้างผลิตภัณฑ์โดเมนเริ่มต้นโดยอัตโนมัติ พร้อมค่าเริ่มต้นที่เหมาะสม
* **ใหม่:** ซิงก์ TLD อัตโนมัติรายวันผ่าน cron ในผู้ให้บริการที่กำหนดค่าไว้ทั้งหมด
* **ใหม่:** การปกป้องความเป็นส่วนตัว WHOIS พร้อมการกำหนดค่ารายผลิตภัณฑ์ (เปิดเสมอ, ลูกค้าเลือกได้, หรือปิดใช้งาน)
* **ใหม่:** ช่องทำเครื่องหมายความเป็นส่วนตัว WHOIS ในหน้าชำระเงิน พร้อมการแสดงราคาและรองรับโหมดมืด
* **ใหม่:** หน้า Register Domain สำหรับผู้ดูแลระบบ สำหรับการจดทะเบียนโดเมนด้วยตนเอง
* **ใหม่:** การอัปเดต plugin อัตโนมัติผ่านเซิร์ฟเวอร์อัปเดตของ Ultimate Multisite
* **ใหม่:** แท็บตัวกรองประเภทผลิตภัณฑ์โดเมนในตารางรายการผลิตภัณฑ์ พร้อมสไตล์ป้ายสีม่วง
* **ใหม่:** ช่องข้อมูลติดต่อผู้จดทะเบียน (ชื่อ, ที่อยู่, เมือง, รัฐ, รหัสไปรษณีย์, ประเทศ, โทรศัพท์) ในฟอร์มชำระเงินโดเมน
* **ใหม่:** การตรวจสอบความถูกต้องของช่องผู้จดทะเบียนก่อนเรียก registrar API พร้อมข้อความข้อผิดพลาดที่ชัดเจน
* **ใหม่:** ช่อง log เฉพาะผู้ให้บริการสำหรับเหตุการณ์การจดทะเบียนโดเมน (เช่น domain-seller-namecheap.log)
* **ใหม่:** ช่องข้อมูลติดต่อผู้จดทะเบียนในฟอร์มหลักสำหรับการลงทะเบียน/สมัครและชำระเงิน (แสดงเมื่อจดทะเบียนโดเมน)
* **ปรับปรุง:** แทนที่ช่องค้นหาโดเมนในหน้าชำระเงินด้วยช่องเลือกโดเมนแบบรวม ที่รองรับแท็บโดเมนย่อย, จดทะเบียน, และโดเมนที่มีอยู่
* **ปรับปรุง:** การตั้งค่าผลิตภัณฑ์โดเมนแสดงแบบอินไลน์บนหน้าแก้ไขผลิตภัณฑ์ผ่านระบบวิดเจ็ตหลัก
* **ปรับปรุง:** ข้อมูลโดเมนของลูกค้าเชื่อมเข้ากับวิดเจ็ตแมปโดเมนหลักแทน metabox แบบแยกต่างหาก
* **ปรับปรุง:** ตัวช่วยนำเข้า TLD ถูกทำให้ง่ายขึ้นเป็นการซิงก์คลิกเดียวจากผู้ให้บริการทั้งหมด
* **ปรับปรุง:** การตรวจสอบความพร้อมใช้งานโดเมนของ Namecheap ใช้การเรียก API แบบชุดเพื่อค้นหาได้เร็วขึ้น
* **ปรับปรุง:** API ราคาของ Namecheap ใช้พารามิเตอร์และการแยกวิเคราะห์การตอบกลับที่ถูกต้อง
* **ปรับปรุง:** รวมศูนย์การจัดเก็บ TLD ไว้ในตัวเลือกเครือข่ายเดียว
* **ปรับปรุง:** การบันทึกกิจกรรมโดเมนสำหรับการเปลี่ยนแปลง DNS, การโอนย้าย, และการใช้การกำหนดค่า
* **ปรับปรุง:** การซิงก์ TLD แบบเต็มสำหรับ OpenSRS โดยใช้รายการหลักของ IANA พร้อมการตรวจสอบแบบชุด
* **ปรับปรุง:** การซิงก์ TLD แบบเต็มสำหรับ Namecheap พร้อมคำขอ API แบบแบ่งหน้า
* **ปรับปรุง:** แทนที่คลาสผู้ให้บริการแบบเดิมด้วยรูปแบบ Integration Registry
* **ปรับปรุง:** แผงการตั้งค่าพร้อมการกำหนดค่า DNS และการโอนย้าย
* **ปรับปรุง:** หมายเลขโทรศัพท์ถูกจัดรูปแบบอัตโนมัติเป็นรูปแบบ registrar +CC.NNN
* **ปรับปรุง:** การตรวจสอบช่องโทรศัพท์จะตัดอักขระการจัดรูปแบบออกก่อนส่ง
* **ปรับปรุง:** เพิ่มข้อกำหนดเวอร์ชันเป็น Ultimate Multisite 2.4.12 พร้อมประกาศที่ชัดเจนขึ้น
* **ปรับปรุง:** เวิร์กโฟลว์ CI ใช้ checkout ที่เหมาะสมสำหรับทั้ง addon และ core plugin
* **ปรับปรุง:** prepare_registrant_info() อ่านจากเมตาผู้ใช้ที่บันทึกจาก checkout โดยมีที่อยู่สำหรับเรียกเก็บเงินเป็นตัวสำรอง
* **แก้ไข:** AJAX ค้นหาโดเมนล้มเหลวสำหรับผู้ใช้ที่ไม่ได้เข้าสู่ระบบระหว่างชำระเงิน
* **แก้ไข:** AJAX ราคาโดเมนล้มเหลวสำหรับผู้ใช้ที่ไม่ได้เข้าสู่ระบบระหว่างชำระเงิน
* **แก้ไข:** ข้อผิดพลาดร้ายแรงจากการประกาศคลาส Spyc ซ้ำเมื่อเรียกใช้คำสั่ง WP-CLI
* **แก้ไข:** timeout ของ Namecheap sandbox API สั้นเกินไป
* **แก้ไข:** ข้อความปุ่ม Select ในการค้นหาโดเมนมองไม่เห็นบนพื้นหลังสีเขียว
* **แก้ไข:** การจดทะเบียนโดเมนล้มเหลวด้วยข้อผิดพลาด "RegistrantFirstName is Missing" เนื่องจากข้อมูลติดต่อขาดหาย
* **แก้ไข:** ระเบียนโดเมนถูกสร้างด้วย blog_id=0 เมื่อเว็บไซต์ยังไม่มีอยู่ ณ เวลาชำระเงิน
* **แก้ไข:** การตั้งค่า TLD เริ่มต้นถูกส่งกลับเป็นสตริงแทนอาร์เรย์ที่แยกวิเคราะห์แล้ว
* **นำออก:** หน้า Domain Management สำหรับผู้ดูแลระบบแบบแยกต่างหาก — ตอนนี้จัดการผ่านหน้าโดเมนหลักแล้ว
