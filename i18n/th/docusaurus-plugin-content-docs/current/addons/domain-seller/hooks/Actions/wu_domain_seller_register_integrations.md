# Action: wu_domain_seller_register_integrations {#action-wudomainsellerregisterintegrations}

## พารามิเตอร์ {#parameters}

|------|------|-------------|
| $registry | `\WP_Ultimo\Integrations\Integration_Registry` | ระบบลงทะเบียนการเชื่อมต่อ |
