# บันทึกการเปลี่ยนแปลง VAT {#vat-changelog}

Version 1.0.7 - เผยแพร่เมื่อ 2026-02-03

* **แก้ไข:** การตรวจสอบหมายเลข VAT ของเยอรมนี (DE) ล้มเหลวเนื่องจากข้อผิดพลาดของบริการ VIES SOAP (MS_MAX_CONCURRENT_REQ) เราจึงเปลี่ยนจากการใช้ SOAP API แบบเดิม มาใช้ EU VIES REST API อย่างเป็นทางการแทน เพื่อให้การตรวจสอบมีความน่าเชื่อถือมากขึ้น
* **แก้ไข:** ตอนนี้ระบบจะยอมรับหมายเลข VAT แม้ว่าบริการ VIES จะไม่พร้อมใช้งานชั่วคราว แทนที่จะปฏิเสธอย่างผิดพลาด ระบบจะสร้างรายการบันทึก (log entry) ไว้ให้ตรวจสอบภายหลัง
* **แก้ไข:** ช่องหมายเลข VAT/Tax ID จะแสดงสำหรับทุกประเทศ ไม่ใช่แค่ประเทศสมาชิก EU เท่านั้น หมายเลขภาษีของประเทศนอก EU (เช่น หมายเลข CHE ของสวิตเซอร์แลนด์) จะถูกจัดเก็บไว้เพื่อแสดงบนใบแจ้งหนี้ แม้ว่าจะไม่มีการตรวจสอบ VIES ก็ตาม
* **แก้ไข:** การค้นหาคำนำหน้า VAT สำหรับกรีซ (GR), โมนาโก (MC), และเกาะแมน (IM) มีปัญหาเนื่องจากตรรกะการค้นหาอาร์เรย์ที่ไม่ถูกต้อง
* **ปรับปรุง:** ลบ dependency ของ wp-cli-bundle ที่ไม่จำเป็นออกไป ซึ่งอาจทำให้เกิดข้อผิดพลาดร้ายแรงเมื่อใช้งานร่วมกับส่วนเสริม Site Exporter
* **ปรับปรุง:** เพิ่มชุดการทดสอบหน่วย (unit test suite) ที่ครอบคลุม (53 tests)

Version 1.0.6 - เผยแพร่เมื่อ 2026-01-25

* **แก้ไข:** การทำธุรกรรม B2B ภายในประเทศจะมีการเรียกเก็บ VAT อย่างถูกต้องแล้ว ตามกฎ VAT ของ EU การเรียกเก็บแบบ reverse charge จะใช้ได้เฉพาะกับการทำธุรกรรม B2B ข้ามพรมแดนเท่านั้น ไม่ใช่กรณีที่ประเทศของลูกค้าตรงกับประเทศของบริษัท

Version 1.0.5 - เผยแพร่เมื่อ 2026-01-22

* **แก้ไข:** ส่วนเสริมไม่โหลดหรือทำงานไม่ถูกต้อง
* **เปลี่ยนแปลง:** เปลี่ยนแหล่งข้อมูลอัตรา VAT จาก euvatrates.com เป็น repository ibericode/vat-rates เพื่อให้ได้ข้อมูลที่เชื่อถือได้และมีการดูแลอย่างต่อเนื่องมากขึ้น
* **แก้ไข:** แก้ไขคำผิดของค่าตัวเลือก super_reduced_rates
* **ปรับปรุง:** เพิ่มการตรวจสอบค่า null สำหรับการจัดการรหัสประเทศ เพื่อป้องกันข้อผิดพลาด
* **เปลี่ยนแปลง:** ลบไฟล์แปลภาษาที่รวมมาให้ (bundled translation files) และเปลี่ยนไปใช้การอัปเดตอัตโนมัติผ่าน Traduttore
