# Action: wp_ultimo_gocardless_load {#action-wpultimogocardlessload}

จะถูกเรียกใช้เมื่อโหลดส่วนประกอบที่จำเป็น (dependencies) ทั้งหมดเสร็จสมบูรณ์

ช่วยให้ผู้พัฒนา plugin สามารถเพิ่มฟังก์ชันการทำงานใหม่ๆ ได้ ตัวอย่างเช่น การเพิ่มการรองรับผู้ให้บริการ Hosting รายใหม่ เป็นต้น
