# บันทึกการเปลี่ยนแปลง (Changelog) ของ Chuck Norris Facts {#chuck-norris-facts-changelog}

### 1.0.1 - 2025-11-03 {#101---2025-11-03}
* เพิ่ม: ฟังก์ชันการอัปเดตอัตโนมัติผ่านเซิร์ฟเวอร์อัปเดตของ Ultimate Multisite
* ปรับปรุง: ทดสอบใช้งานได้ถึง WordPress 6.6
* ปรับปรุง: ปรับปรุงโครงสร้าง plugin ด้วย composer autoloader
* ปรับปรุง: เพิ่มความเข้ากันได้กับ WordPress เวอร์ชันใหม่ๆ

### 1.0.0 - 2024-10-21 {#100---2024-10-21}
* การเปิดตัวครั้งแรก
* มีข้อเท็จจริงเกี่ยวกับ Chuck Norris มากกว่า 110 ข้อ
* รองรับการแปลภาษาเต็มรูปแบบ
* ดีไซน์ที่รองรับทุกขนาดหน้าจอ (Responsive design)
* รองรับภาษาแบบ RTL (Right-to-Left)
* รองรับการใช้งานกับ Block editor
* ฟีเจอร์การเข้าถึง (Accessibility) พร้อมรองรับ Screen Reader
