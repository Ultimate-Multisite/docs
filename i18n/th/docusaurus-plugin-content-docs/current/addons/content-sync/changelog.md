# บันทึกการเปลี่ยนแปลง Content Sync

### เวอร์ชัน 1.1.0 - เผยแพร่เมื่อ 2025-12-31
* ใหม่: รองรับการซิงค์ประเภทโพสต์ทั้งหมด (posts, pages, custom post types)
* ใหม่: การกรองประเภทโพสต์ในขั้นตอนการเลือกเนื้อหา
* ใหม่: ขั้นตอนการเลือกเว็บไซต์ต้นทาง - สามารถซิงค์จากเว็บไซต์ใดก็ได้ ไม่ใช่แค่เว็บไซต์หลัก
* ใหม่: การเลือกเว็บไซต์ลูกล่วงหน้าสำหรับเว็บไซต์แม่แบบ
* ใหม่: การดึงสื่อจาก post_content สำหรับประเภทโพสต์ทั้งหมด
* ใหม่: การซิงค์แท็กซอนโนมี (categories, tags, custom taxonomies)
* เปลี่ยนแปลง: เปลี่ยนชื่อจาก "Elementor Template Sync" เป็น "Content Sync"
* เปลี่ยนแปลง: เปลี่ยนชื่อขั้นตอน "Select Templates" เป็น "Select Data"
* ปรับปรุง: โค้ดเฉพาะของ Elementor จะทำงานเฉพาะกับประเภทโพสต์ elementor_library เท่านั้น

### เวอร์ชัน 1.0.0 - เผยแพร่เมื่อ 2025-12-19
* การเปิดตัวครั้งแรก
* ตัวช่วยแบบหลายขั้นตอนสำหรับการเลือกแม่แบบและเว็บไซต์
* การประมวลผลแบบกลุ่มพร้อมการแจ้งความคืบหน้าแบบเรียลไทม์
* การซิงค์สื่อแบบสมบูรณ์พร้อมการแมป ID ใหม่โดยอัตโนมัติ
* การรวมเข้ากับส่วนตั้งค่าของ Ultimate Multisite
* รองรับการยกเว้นเว็บไซต์ผ่านการตั้งค่าเครือข่าย
* การล้างแคช Elementor หลังการซิงค์
* การคงไว้ซึ่งแท็กซอนโนมีของแม่แบบ
