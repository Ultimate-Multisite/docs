# Mailchimp

Ultimate Multisite: การเชื่อมต่อกับ Mailchimp

คุณสามารถเชื่อมต่อเครือข่าย Ultimate Multisite ของคุณเข้ากับ Mailchimp ได้ คุณสามารถผูกแผนบริการบางอย่างเข้ากับ Mailchimp Lists และเพิ่มผู้สมัครสมาชิกเข้าสู่รายการเหล่านั้นโดยอัตโนมัติเมื่อมีการลงทะเบียน

## การติดตั้ง {#installation}

1. อัปโหลด 'ultimate-multisite-mailchimp' ไปยังไดเรกทอรี '/wp-content/plugins/'
2. เปิดใช้งานปลั๊กอินผ่านเมนู 'Plugins' ใน WordPress Network Admin ของคุณ

## คู่มือการใช้งาน {#usage-guide}

การซิงค์อีเมลของผู้สมัครสมาชิกเข้ากับรายการ MailChimp ของคุณนั้นง่ายมากด้วย Add-on Ultimate Multisite: MailChimp Integration ของเรา บทช่วยสอนนี้มีจุดมุ่งหมายเพื่อช่วยให้คุณตั้งค่าทุกอย่างได้อย่างสมบูรณ์!

## การรับ API key {#getting-an-api-key}

ก่อนที่เราจะเริ่มดูการตั้งค่าต่างๆ ที่มีให้ เราจำเป็นต้องมี MailChimp API key สิ่งนี้จะช่วยให้ Ultimate Multisite สามารถสื่อสารกับ MailChimp API เพื่อดึงรายการ (lists) กลุ่ม (groups) และเพิ่มที่อยู่อีเมลไปยังรายการและกลุ่มเหล่านั้นเมื่อผู้ใช้ลงทะเบียน

ทำตามบทช่วยสอนนี้เพื่อรับ MailChimp API key ของคุณเอง: [About API Keys | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

<!-- Screenshot unavailable: MailChimp API key settings in Ultimate Multisite Add-on Settings -->

_โปรดเพิ่ม MailChimp API key ของคุณที่_ _**Ultimate Multisite Settings - > Add-on Settings -> MailChimp**_

## โหมดการเชื่อมต่อ (Integration Mode) {#integration-mode}

Add-on MailChimp Integration มี "โหมดการเชื่อมต่อ" ให้เลือก 2 โหมด ซึ่งหมายความว่ามี 2 วิธีที่แตกต่างกันที่คุณสามารถใช้ Add-on นี้เพื่อซิงค์ที่อยู่อีเมลได้

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_สองโหมดการเชื่อมต่อเพื่อความยืดหยุ่นที่มากขึ้น!_

### โหมดที่ 1: Multiple Lists {#mode-1-multiple-lists}

โหมดเริ่มต้นใช้ **Multiple Lists** โหมดนี้จะช่วยให้คุณสามารถเลือกรายการ MailChimp ได้หลายรายการสำหรับแต่ละแผนบริการ นี่คือตัวเลือกที่ดีที่สุดหากคุณต้องการแบ่งผู้ใช้ของคุณออกเป็นรายการต่างๆ

ตัวอย่างเช่น สมมติว่าคุณมีรายการที่แตกต่างกัน 3 รายการในบัญชี MailChimp ของคุณ ได้แก่ **Newsletter** , **Plan A** , และ **Plan B** คุณต้องการให้ผู้ใช้ทุกคน ไม่ว่าจะใช้แผนใด ก็ยังถูกรวมอยู่ในรายการ Newsletter เพื่อไม่ให้พลาดการอัปเดตของคุณ

ในการทำเช่นนั้น คุณสามารถใช้โหมด Multiple Lists เพื่อเพิ่มรายการ **Newsletter** และ **Plan A** ในการตั้งค่า MailChimp ของ Plan A และเพิ่ม **Newsletter** และ **Plan B** ในการตั้งค่า MailChimp ของ Plan B

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_เลือกรายการเริ่มต้นสำหรับบัญชีใหม่และบัญชีที่ยกเลิก_

### โหมดที่ 2: Single List + Multiple Groups {#mode-2-single-list--multiple-groups}

MailChimp อนุญาตให้คุณแบ่งรายการของคุณออกเป็นกลุ่มย่อยของผู้ใช้ที่เรียกว่า **Groups** สิ่งนี้ช่วยให้คุณควบคุมได้มากขึ้น เพราะคุณสามารถส่งแคมเปญไปยังกลุ่มผู้สมัครสมาชิกย่อยเฉพาะกลุ่มได้ โดยไม่จำเป็นต้องสร้างแคมเปญหลายชุดบน MailChimp (ซึ่งทำให้การวิเคราะห์เมตริกของแคมเปญนั้นๆ ในภายหลังทำได้ยาก)

ตัวอย่างเช่น คุณสามารถมีรายการเดียวชื่อว่า **My Network Subscribers** และภายในรายการนั้นมีกลุ่มสำหรับ **Plan A** และ **Plan B**

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_เลือกรายการและกลุ่มเริ่มต้น หากใช้โหมดการเชื่อมต่อแบบ “single list + multiple groups”_

## รายการและกลุ่มเริ่มต้น (Default Lists and Groups) {#default-lists-and-groups}

หลังจากที่คุณเลือกโหมดการเชื่อมต่อที่เหมาะสมกับกรณีการใช้งานของคุณแล้ว ก็ถึงเวลาเลือกรายการเริ่มต้น (และกลุ่ม ขึ้นอยู่กับโหมด) รายการเริ่มต้นจะถูกใช้เมื่อแผนบริการที่กำหนดไม่มีการเลือกรายการเฉพาะเจาะจง

คุณยังสามารถเลือกรายการเริ่มต้นสำหรับบัญชีที่ยกเลิกได้ด้วย ทุกครั้งที่ผู้ใช้ยกเลิกการสมัครสมาชิกและลบบัญชีออกจากเครือข่ายของคุณ ที่อยู่อีเมลของพวกเขาจะถูกย้ายไปยังรายการที่ตั้งค่าไว้สำหรับบัญชีที่ยกเลิก

## การเปิดใช้งาน Double Subscription {#enabling-double-subscription}

ในหน้าการตั้งค่าเดียวกัน คุณจะสามารถเปิดใช้งาน **Double Subscription** ตัวเลือกนี้จะเพิ่มผู้ใช้เข้าสู่ทั้งรายการเริ่มต้นที่เลือก **และ** รายการที่เลือกสำหรับแผนบริการของผู้ใช้นั้นๆ หากไม่ได้เปิดใช้งานตัวเลือกนี้ ผู้ใช้จะถูกเพิ่มเข้าเฉพาะรายการที่เลือกสำหรับแผนบริการเท่านั้น หากไม่มีการตั้งค่าสำหรับแผนบริการนั้นๆ เลย ผู้ใช้จะถูกเพิ่มเข้าสู่รายการเริ่มต้นที่เลือกแทน

<!-- Screenshot unavailable: MailChimp double subscription toggle setting -->

_คุณสามารถให้ผู้ใช้ของคุณสมัครสมาชิกทั้งรายการเริ่มต้นและรายการของแผนบริการได้เมื่อลงทะเบียน_

## การเลือกรายการและกลุ่มสำหรับแต่ละแผนบริการ {#selecting-lists-and-groups-for-each-of-the-plans}

ไปที่ **Plans** ของคุณและเลือกแผนบริการที่คุณต้องการแก้ไข โปรดทราบว่าใน meta-box **Advanced Options** จะมีแท็บใหม่ชื่อ MailChimp ให้ใช้งาน

ในแท็บนั้น คุณจะสามารถเลือกรายการหลายรายการ หรือรายการเดียวและกลุ่มหลายกลุ่ม (ขึ้นอยู่กับโหมดการเชื่อมต่อ) สำหรับแผนบริการนั้นๆ

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_เลือกรายการและกลุ่มที่คุณต้องการใช้สำหรับแต่ละแผนบริการของคุณ_

เมื่อผู้ใช้ใหม่ลงทะเบียนสำหรับแผนบริการนั้น ที่อยู่อีเมลของพวกเขาจะถูกเพิ่มเข้าสู่รายการและกลุ่มที่เลือกโดยอัตโนมัติ!
