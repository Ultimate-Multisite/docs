# AI Site Builder

ultimate-multisite-ai-site-builder

## ข้อกำหนดเบื้องต้น {#requirements}

- WordPress 5.3 หรือสูงกว่า
- PHP 7.4 หรือสูงกว่า
- ปลั๊กอิน Ultimate Multisite (ต้องเปิดใช้งาน)

## การติดตั้ง {#installation}

1. อัปโหลดไฟล์ addon ไปยังไดเรกทอรี `/wp-content/plugins/`
2. เปิดใช้งานปลั๊กอินผ่านเมนู 'Plugins' ใน WordPress
3. ตั้งค่า addon ในส่วนของ Network Admin

## การพัฒนา {#development}

### การตั้งค่า {#setup}

```bash
# ติดตั้ง dependencies
composer install
npm install

# รันการทดสอบ
npm run test

# ตรวจสอบมาตรฐานโค้ด
vendor/bin/phpcs
vendor/bin/phpstan

# แก้ไขปัญหาด้านรูปแบบโค้ด
vendor/bin/phpcbf

# สร้างโค้ดสำหรับ Production
npm run build
```

### การทดสอบ {#testing}

Addon นี้มาพร้อมกับเฟรมเวิร์กการทดสอบที่ครอบคลุม:

- **Unit Tests**: การทดสอบที่ใช้ PHPUnit และชุดทดสอบของ WordPress
- **Base Test Class**: มีเมธอดช่วยเหลือสำหรับสถานการณ์การทดสอบทั่วไป
- **Test Coverage**: ตั้งค่าเพื่อติดตามความครอบคลุมของโค้ด
- **Multisite Testing**: การทดสอบที่รันในสภาพแวดล้อมแบบ Multisite

### มาตรฐานโค้ด {#code-standards}

- **WordPress Coding Standards**: บังคับใช้ผ่าน PHPCS
- **PHP 7.4+ Compatibility**: รองรับฟีเจอร์ PHP สมัยใหม่
- **Static Analysis**: การรวม PHPStan เพื่อตรวจสอบประเภทข้อมูล (Type Checking)
- **Code Modernization**: กฎของ Rector สำหรับการอัปเกรด PHP

### ระบบ Build {#build-system}

- **Asset Processing**: การย่อขนาด (Minification) สำหรับไฟล์ CSS/JS
- **Translation**: การสร้างไฟล์ POT
- **Packaging**: การสร้างไฟล์ Archive สำหรับการแจกจ่าย
- **Development Mode**: ไฟล์ assets ที่ยังไม่ย่อขนาดสำหรับการดีบัก

## การตั้งค่า {#configuration}

Addon นี้รองรับตัวเลือกการตั้งค่าที่หลากหลาย ซึ่งสามารถเข้าถึงได้ผ่านหน้า Network Admin

## Hooks และ Filters {#hooks-and-filters}

### Actions {#actions}

- `ultimate-multisite-ai-site-builder_init` - ถูกเรียกใช้หลังจากเริ่มต้น addon
- `ultimate-multisite-ai-site-builder_loaded` - ถูกเรียกใช้หลังจากโหลดไฟล์ addon ทั้งหมด

### Filters {#filters}

- `ultimate-multisite-ai-site-builder_settings` - ใช้สำหรับแก้ไขการตั้งค่าของ addon
- `ultimate-multisite-ai-site-builder_enabled` - ใช้สำหรับกำหนดสถานะเปิด/ปิดของ addon แทนค่าเดิม
