# บันทึกการเปลี่ยนแปลง Metered Plans {#metered-plans-changelog}

Version 1.1.0 - เผยแพร่เมื่อ 2026-05-05
- ใหม่: การเรียกเก็บเงินตามโทเค็น AI สำหรับ subsites ของ multisite — ติดตามและเรียกเก็บเงินการใช้โทเค็น AI ทั่วทั้งไซต์ของลูกค้า พร้อมอัตราต่อโทเค็นที่สามารถตั้งค่าได้
- ใหม่: ปรับปรุงการบังคับใช้ Connector ใหม่ด้วยการค้นหาขีดจำกัดแบบไดนามิก (dynamic limit discovery) และ write-through ทำให้มั่นใจได้ถึงความแม่นยำแบบเรียลไทม์ในทุก Connector
- แก้ไข: การอัปเกรดตารางฐานข้อมูลตอนนี้สอดคล้องกับคำจำกัดความของ schema ของ BerlinDB อย่างถูกต้อง ป้องกันความล้มเหลวในการอัปเกรดเมื่อติดตั้งใหม่
- แก้ไข: แปลง callback ของการอัปเกรดฐานข้อมูลเป็นรูปแบบที่ถูกต้อง แก้ไขปัญหาความล้มเหลวในการอัปเกรดแบบเงียบ (silent upgrade failures)
- แก้ไข: ตอนนี้ยอมรับค่าทศนิยม (Fractional values) ในช่องป้อนข้อมูลส่วนเพิ่มการใช้งานเกินของ AI (AI Usage Overage Markup)
- แก้ไข: แก้ไขข้อผิดพลาดร้ายแรง (fatal errors) และปัญหาการเริ่มต้นค่าซ้ำซ้อน (double-initialization issues) เมื่อโหลดปลั๊กอิน
- ปรับปรุง: เพิ่มสคริปต์ npm check-env เพื่อให้สภาพแวดล้อมสำหรับนักพัฒนา (developer environments) สามารถกำหนดค่าด้วยตัวเองได้ในการรันครั้งแรก
