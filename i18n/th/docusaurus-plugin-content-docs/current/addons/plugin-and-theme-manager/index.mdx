# Plugin & Theme Manager {#plugin--theme-manager}

Ultimate Multisite: Plugin and Theme Manager

สามารถแก้ไขหรือซ่อนข้อมูลเมตา (เช่น ชื่อเรื่อง, ผู้เขียน, รูปย่อ, และคำอธิบาย) ของปลั๊กอินและธีมที่ติดตั้งได้ นอกจากนี้ยังสามารถสร้างหมวดหมู่สำหรับปลั๊กอินและธีม เพื่อให้ผู้ใช้ของคุณสามารถกรองข้อมูลได้ และเพิ่มหน้า Plugins ที่สวยงามและปรับแต่งได้ในส่วนผู้ใช้ของคุณ

หากต้องการใช้ปลั๊กอินนี้ คุณสามารถไปที่หน้า Plugins หรือ Themes ในส่วนผู้ดูแลระบบเครือข่าย (network admin) แล้วคลิกที่ "Edit Plugin/Theme info"

## การติดตั้ง {#installation}

1. อัปโหลด 'ultimate-multisite-plugin-and-theme-manager' ไปยังไดเรกทอรี '/wp-content/plugins/'
2. เปิดใช้งานปลั๊กอินผ่านเมนู 'Plugins' ในส่วนผู้ดูแลระบบเครือข่าย WordPress ของคุณ

## คู่มือการใช้งาน {#usage-guide}

Ultimate Multisite: Plugin and Theme Manager เป็นส่วนเสริม (add-on) ของ Ultimate Multisite ที่ทรงพลังระดับ **Premium** ซึ่งช่วยให้คุณควบคุมได้อย่างเต็มที่ว่าผู้ใช้ในเครือข่ายของคุณจะเห็นและโต้ตอบกับตัวเลือกปลั๊กอินและธีมในเครือข่ายของคุณอย่างไร

ส่วนเสริมนี้มีวัตถุประสงค์หลัก 3 ประการ:

  1. ช่วยให้คุณสามารถเปลี่ยนข้อมูลเมตาของทั้งปลั๊กอินและธีมได้อย่างสมบูรณ์ ซึ่งรวมถึงชื่อเรื่อง, คำอธิบาย, ชื่อผู้เขียน, และรูปย่อด้วย!

  2. ช่วยให้คุณสามารถจัดหมวดหมู่ปลั๊กอินและธีม เพื่อให้ผู้ใช้ของคุณสามารถจัดเรียงข้อมูลได้อย่างง่ายดาย

  3. ให้ทางเลือกในการแทนที่หน้า Plugins แบบเก่าที่น่าเบื่อ ด้วยหน้าใหม่ที่คล้ายกับหน้า Themes ซึ่งมีการจัดเรียงตามหมวดหมู่

หลังจากดาวน์โหลดและติดตั้งแล้ว จะมีแท็บใหม่ชื่อว่า _Add-on Settings_ ปรากฏขึ้นบนหน้า Ultimate Multisite Settings
