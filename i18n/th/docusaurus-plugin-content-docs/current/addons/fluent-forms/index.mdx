# Fluent Forms

Add-on นี้ช่วยให้คุณสามารถกำหนดขีดจำกัด (limits) ให้กับ Fluent Forms บน Multisite Ultimate ไม่ว่าจะใช้กับสินค้า (products), การเป็นสมาชิก (memberships) หรือเว็บไซต์ (sites) ก็ได้
เนื่องจาก Fluent Forms ใช้ custom tables แทน custom post types ตัวเลือกการจำกัดแบบปกติจึงไม่สามารถใช้งานได้กับ Fluent Forms

## คุณสมบัติ {#features}

- เพิ่มขีดจำกัดสำหรับ Fluent Forms หรือ Conversational Fluent Forms
- ลูกค้าจะได้รับข้อความที่เป็นมิตรเพื่อแจ้งให้พวกเขาทราบถึงการอัปเกรดแผนเมื่อถึงขีดจำกัดที่กำหนด
