# บันทึกการเปลี่ยนแปลงของ Site Exporter {#site-exporter-changelog}

Version: 1.0.6 - เผยแพร่เมื่อ 2026-05-11
* ใหม่: การส่งออกเว็บไซต์ตอนนี้จะรวมไฟล์ index.php ที่สามารถบูตได้ด้วยตัวเอง ทำให้สามารถติดตั้งไฟล์ ZIP นี้บนโฮสต์ใหม่ได้เลย โดยไม่จำเป็นต้องติดตั้งปลั๊กอินแยกต่างหาก
* ใหม่: การส่งออกเครือข่ายช่วยให้ผู้ดูแลระบบสามารถส่งออกไซต์ย่อยทั้งหมดในไฟล์เก็บถาวรเดียวได้จากหน้าผู้ดูแลระบบ Site Export

Version: 1.0.5 - เผยแพร่เมื่อ 2026-05-05
* แก้ไข: เพิ่ม __callStatic ให้กับ WP_CLI polyfill stub เพื่อจัดการกับเมธอดที่ไม่ได้จำลอง (unmocked methods) เช่น get_config ซึ่งช่วยป้องกันความล้มเหลวในการบูตระบบทดสอบ

Version: 1.0.4 - เผยแพร่เมื่อ 2025-11-25

* แก้ไข: แก้ไขความเข้ากันได้กับ Ultimate Multisite 2.4

Version: 1.0.2 - เผยแพร่เมื่อ 2025-09-28

* เปลี่ยนชื่อ prefix เป็น ultimate-multisite; อัปเดต text domain; และเพิ่มเวอร์ชัน

### Version 1.0.1 - เผยแพร่เมื่อ 2023-08-09 {#version-101---released-on-2023-08-09}

* ปรับปรุง: ตอนนี้จะมั่นใจได้ว่าได้เส้นทางไฟล์ (file path) ของไฟล์นำเข้าที่ถูกต้อง
* แก้ไข: ปิดใช้งาน object cache เพื่อป้องกันข้อผิดพลาดระหว่างการนำเข้า
* แก้ไข: แก้ไขปัญหาที่ลิงก์ดาวน์โหลดปลั๊กอินนำเข้าไม่ทำงาน

### Version 1.0.0 - เผยแพร่เมื่อ 2022-12-23 {#version-100---released-on-2022-12-23}

* แก้ไข: ป๊อปอัปฟอร์มการส่งออกไม่โหลด;
* แก้ไข: ไฟล์ Javascript ไม่โหลดบนหน้าการส่งออก;
* ภายใน: กระบวนการสร้างใหม่;

### Version 1.0.0-beta.4 - เผยแพร่เมื่อ 2022-01-13 {#version-100-beta4---released-on-2022-01-13}

* ภายใน: เพิ่มตัวสร้าง hooks และ filters;
* ภายใน: เพิ่ม stubs ของ WP Ultimo เพื่อเพิ่มคุณภาพชีวิตให้กับนักพัฒนา;
