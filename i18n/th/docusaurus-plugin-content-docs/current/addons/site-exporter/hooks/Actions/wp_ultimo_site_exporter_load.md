# Action: wp_ultimo_site_exporter_load {#action-wpultimositeexporterload}

จะทำงานเมื่อมีการโหลดส่วนประกอบ (dependencies) ทั้งหมดเรียบร้อยแล้ว

ช่วยให้นักพัฒนาปลั๊กอินสามารถเพิ่มฟังก์ชันการทำงานใหม่ ๆ ได้ ตัวอย่างเช่น การรองรับผู้ให้บริการโฮสติ้งรายใหม่ เป็นต้น

### ตั้งแต่ {#since}

- 1.0.0
