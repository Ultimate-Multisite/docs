# เจ้าหน้าที่สนับสนุน

การบริหารเครือข่ายทั้งหมดด้วยตัวคนเดียวไม่ใช่เรื่องง่ายเลย เมื่อเครือข่ายของคุณเริ่มขยายตัว คุณจะเริ่มเห็นความต้องการในการสนับสนุนที่เพิ่มขึ้น การจ้างคนมาช่วยแบ่งเบาภาระงานของคุณจึงเป็นความคิดที่ดี อย่างไรก็ตาม การให้สิทธิ์การเข้าถึง/สิทธิ์เต็มรูปแบบนั้นมีความเสี่ยงและอาจทำให้เครือข่ายของคุณเสียหายได้ นี่คือจุดที่ Add-on Support Agent ตัวนี้เข้ามาช่วย

Add-on นี้ช่วยให้คุณสามารถสร้างผู้ใช้ประเภทใหม่ใน WordPress ที่เรียกว่า Support Agent และให้คุณควบคุมสิทธิ์ของผู้ใช้นั้นได้อย่างสมบูรณ์ การให้สิทธิ์ทำได้ง่าย ๆ เพียงแค่เปิดปุ่มสลับ (toggle button) และหากคุณตัดสินใจที่จะยกเลิกสิทธิ์ ก็สามารถปิดได้ทันที

## คุณสมบัติ

*   รายการสิทธิ์การใช้งานของ WordPress และ Ultimate Multisite แบบละเอียด
*   เปิด/ปิดการเข้าถึงได้อย่างง่ายดาย
*   สร้างบัญชี Agent ได้ไม่จำกัด

## การติดตั้ง

1.  อัปโหลด 'ultimate-multisite-support-agents' ไปยังไดเรกทอรี '/wp-content/plugins/'
2.  เปิดใช้งานปลั๊กอินผ่านเมนู 'Plugins' ใน WordPress
3.  ทำตามขั้นตอนของ Wizard เพื่อตั้งค่าปลั๊กอิน
