# บันทึกการเปลี่ยนแปลง Loco Translate

### 1.0.1 {#101}
* ปรับปรุงความเข้ากันได้กับ Loco Translate เวอร์ชันล่าสุด
* เพิ่มการควบคุมการเข้าถึงเพื่อความปลอดภัยที่ดีขึ้น
* ปรับปรุงประสิทธิภาพ

### 1.0.0 {#100}
* การเปิดตัวครั้งแรก
* ไดเรกทอรีการแปลเฉพาะเว็บไซต์
* ระบบลำดับความสำคัญของการแปล
* การใช้งานระบบควบคุมการเข้าถึง
* การรวมเข้ากับ UI ของ Loco Translate อย่างราบรื่น
