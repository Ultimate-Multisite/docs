# Loco Translate

Addon Loco Translate สำหรับ Ultimate Multisite ช่วยให้คุณสามารถจัดการการแปลเฉพาะเว็บไซต์ได้ในเครือข่าย WordPress multisite ของคุณ หากคุณเคยต้องการให้เว็บไซต์ต่างๆ ในเครือข่ายของคุณมีการแปลที่แตกต่างกันสำหรับปลั๊กอินหรือธีมเดียวกัน Addon นี้คือคำตอบที่สมบูรณ์แบบ

**Addon นี้ทำอะไรได้บ้าง**

Addon นี้จะผสานรวม Ultimate Multisite เข้ากับปลั๊กอิน Loco Translate ที่ได้รับความนิยมได้อย่างราบรื่น ทำให้แต่ละ subsite ในเครือข่ายของคุณสามารถเก็บการแปลที่กำหนดเองได้ด้วยตัวมันเอง ไม่ว่าคุณจะกำลังสร้างแพลตฟอร์มสำหรับหลายภูมิภาค ให้บริการชุมชนภาษาที่แตกต่างกัน หรือเพียงแค่ต้องการความยืดหยุ่นในการแปลทั่วทั้งเครือข่าย Addon นี้ก็จะทำให้เรื่องเหล่านี้เป็นเรื่องง่าย

**เหมาะสำหรับ:**

* ธุรกิจหลายภูมิภาคที่มีความต้องการด้านการแปลที่แตกต่างกัน
* สถาบันการศึกษาที่ให้บริการชุมชนที่หลากหลาย
* แพลตฟอร์ม SaaS ที่รองรับหลายภาษา
* เอเจนซี่เว็บไซต์ที่ดูแลหลายไซต์ของลูกค้า
* เครือข่าย multisite ใดๆ ที่ต้องการความยืดหยุ่นในการแปล

## การติดตั้ง {#installation}

1. ตรวจสอบให้แน่ใจว่าได้ติดตั้ง Ultimate Multisite และเปิดใช้งานเครือข่ายแล้ว
2. ติดตั้งและเปิดใช้งานปลั๊กอิน Loco Translate
3. อัปโหลด Addon นี้ไปยังไดเรกทอรีปลั๊กอินของคุณ
4. เปิดใช้งาน Addon Loco Translate Integration ในระดับเครือข่าย
5. เริ่มสร้างการแปลเฉพาะเว็บไซต์ผ่าน Loco Translate บน subsite ใดก็ได้

แค่นี้ก็เสร็จแล้ว! ระบบจะทำงานโดยอัตโนมัติโดยไม่จำเป็นต้องมีการตั้งค่าเพิ่มเติมใดๆ

## คำถามที่พบบ่อย {#frequently-asked-questions}

## ฉันต้องตั้งค่าอะไรเพิ่มเติมหลังการติดตั้งหรือไม่? {#do-i-need-to-configure-anything-after-installation}

ไม่เลย! ระบบจะทำงานอัตโนมัติ เมื่อทั้ง Addon และ Loco Translate ถูกเปิดใช้งานแล้ว subsite ก็สามารถเริ่มสร้างการแปลที่กำหนดเองได้ทันที

## จะเกิดอะไรขึ้นถ้า subsite ไม่ได้สร้างการแปลที่กำหนดเอง? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

เว็บไซต์จะใช้การแปลระดับเครือข่ายเป็นตัวสำรองโดยอัตโนมัติ ไม่มีอะไรพัง และทุกอย่างยังคงทำงานได้อย่างราบรื่น

## subsite สามารถเข้าถึงการแปลระดับเครือข่ายได้หรือไม่? {#can-subsites-access-network-wide-translations}

ผู้ดูแลระบบเว็บไซต์ทั่วไปจะเห็นและแก้ไขเฉพาะการแปลของเว็บไซต์ตัวเองเท่านั้น ส่วนผู้ดูแลระบบระดับสูง (Super admins) สามารถเข้าถึงได้ทั้งการแปลระดับเครือข่ายและการแปลเฉพาะเว็บไซต์ ขึ้นอยู่กับว่าพวกเขากำลังดูเว็บไซต์ไหนอยู่

## สิ่งนี้ใช้ได้กับทั้งปลั๊กอินและธีมหรือไม่? {#does-this-work-with-both-plugins-and-themes}

ได้แน่นอน! การแปลเฉพาะเว็บไซต์ใช้ได้ทั้งกับปลั๊กอินและธีมของ WordPress

## สิ่งนี้จะทำให้เครือข่าย multisite ของฉันช้าลงหรือไม่? {#will-this-slow-down-my-multisite-network}

ไม่ครับ Addon นี้ใช้การโหลดไฟล์ที่มีประสิทธิภาพ ซึ่งจะตรวจสอบการแปลเฉพาะเว็บไซต์ก็ต่อเมื่อจำเป็นเท่านั้น จึงไม่มีผลกระทบต่อประสิทธิภาพที่สังเกตได้

## ฉันสามารถคัดลอกการแปลระดับเครือข่ายไปยังเว็บไซต์เฉพาะเพื่อใช้เป็นจุดเริ่มต้นได้หรือไม่? {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

ได้ครับ! ขณะที่คุณกำลังแก้ไขการแปลใน Loco Translate คุณสามารถเริ่มต้นด้วยการแปลระดับเครือข่ายและปรับแต่งให้เหมาะกับแต่ละเว็บไซต์ได้อย่างง่ายดาย

## สิ่งนี้ใช้ได้กับ WooCommerce และปลั๊กอินอื่นๆ หรือไม่? {#does-this-work-with-woocommerce-and-other-plugins}

แน่นอนครับ! ปลั๊กอินหรือธีมใดๆ ที่ใช้ฟังก์ชันการแปลมาตรฐานของ WordPress จะสามารถทำงานร่วมกับการแปลเฉพาะเว็บไซต์ได้
