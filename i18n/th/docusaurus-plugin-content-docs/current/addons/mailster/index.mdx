# Mailster {#mailster}

เปลี่ยนเครือข่าย multisite ของคุณให้เป็นเครื่องมือสร้างรายชื่ออีเมลที่ทรงพลังด้วยปลั๊กอิน Ultimate Multisite Mailster Integration ตัวเสริมนี้ ระบบจะเพิ่มลูกค้าใหม่เข้าสู่รายชื่ออีเมล Mailster โดยอัตโนมัติเมื่อพวกเขาลงทะเบียนหรือซื้อสินค้า ทำให้การเชื่อมต่อระหว่างแพลตฟอร์มสมาชิกและกลยุทธ์การตลาดอีเมลของคุณเป็นไปอย่างราบรื่น

## คุณสมบัติหลัก {#key-features}

**การซิงค์สมาชิกอัตโนมัติ**
* เพิ่มลูกค้าเข้าสู่รายชื่อ Mailster โดยอัตโนมัติเมื่อพวกเขาลงทะเบียน
* เลือกว่าจะซิงค์ทันทีเมื่อลงทะเบียน หรือรอการยืนยันการชำระเงิน
* ทำให้รายชื่ออีเมลของคุณสดใหม่และเป็นปัจจุบันโดยไม่ต้องทำด้วยตนเอง

**การจัดการรายชื่อที่ยืดหยุ่น**
* กำหนดรายชื่อเริ่มต้นทั่วโลกสำหรับลูกค้าใหม่ทุกคน
* สามารถกำหนดรายชื่อเฉพาะสินค้าเพื่อแทนที่ค่าเริ่มต้นสำหรับการทำแคมเปญที่เจาะจง
* จัดการการกำหนดรายชื่อได้อย่างง่ายดายด้วย interface ที่คุ้นเคยของ Mailster

**ตัวเลือกการ Opt-in ที่สอดคล้องกับ GDPR**
* เลือกระหว่างการสมัครสมาชิกอัตโนมัติ หรือการยินยอมโดยใช้ช่องทำเครื่องหมาย (checkbox)
* เปิดใช้งาน double opt-in สำหรับการสมัครสมาชิกที่ได้รับการยืนยัน
* แสดง/ซ่อนช่องทำเครื่องหมาย opt-in ตามข้อกำหนดด้านการปฏิบัติตามกฎระเบียบของคุณ

**การแมปข้อมูลอัจฉริยะ**
* ซิงค์ชื่อลูกค้าและข้อมูลติดต่อโดยอัตโนมัติ
* แมปที่อยู่สำหรับเรียกเก็บเงินไปยังช่องข้อมูลสมาชิก
* ติดตามวันที่ลงทะเบียนเพื่อทำแคมเปญที่เจาะจง

**การควบคุมระดับสินค้า**
* กำหนดรายชื่อที่แตกต่างกันให้กับสินค้าที่แตกต่างกัน
* สร้างแคมเปญที่เจาะจงตามสินค้าที่ลูกค้าซื้อ
* ปิดใช้งาน Mailster สำหรับสินค้าบางรายการเมื่อจำเป็น

**ประสบการณ์ระดับมืออาชีพ**
* การรวม UI ของ Mailster แบบ Native สำหรับการเลือกรายชื่อ
* การบันทึกข้อมูลโดยละเอียดสำหรับการแก้ไขปัญหาและการตรวจสอบ
* การอัปเดตสมาชิกที่มีอยู่ได้อย่างราบรื่น

## เหมาะสำหรับ {#perfect-for}

* เจ้าของเว็บไซต์สมาชิกที่ต้องการสร้างรายชื่ออีเมล
* แพลตฟอร์ม SaaS ที่ต้องการดูแลลูกค้าใหม่
* ผู้สร้างคอร์สที่ต้องการแบ่งกลุ่มนักเรียนตามการลงทะเบียน
* ทุกคนที่ใช้ Ultimate Multisite ร่วมกับ Mailster

## วิธีการทำงาน {#how-it-works}

1. ติดตั้งและเปิดใช้งาน addon บนเครือข่าย Ultimate Multisite ของคุณ
2. กำหนดรายชื่อเริ่มต้นทั่วโลกในหน้าการตั้งค่า Ultimate Multisite
3. (ทางเลือก) กำหนดรายชื่อเฉพาะสินค้าเพื่อแทนที่ค่าเริ่มต้น
4. เลือกรูปแบบการ opt-in และช่วงเวลาการสมัครสมาชิกของคุณ
5. ดูรายชื่อ Mailster ของคุณเติบโตขึ้นโดยอัตโนมัติ!

## การติดตั้ง {#installation}

1. อัปโหลดไฟล์ปลั๊กอินไปยังไดเรกทอรี addons ของ Ultimate Multisite
2. ตรวจสอบให้แน่ใจว่า Mailster ได้รับการติดตั้งและเปิดใช้งานบนเว็บไซต์หลักของคุณ
3. เปิดใช้งาน addon ผ่านหน้า Ultimate Multisite addons
4. ไปที่ Settings > Mailster Integration เพื่อกำหนดค่า
5. เลือกรายชื่อเริ่มต้นของคุณและกำหนดค่าความต้องการ opt-in

## คำถามที่พบบ่อย {#frequently-asked-questions}

## addon นี้ทำงานได้โดยไม่ต้องใช้ Ultimate Multisite หรือไม่? {#does-this-addon-work-without-ultimate-multisite}

ไม่ได้ addon นี้ต้องใช้ Ultimate Multisite ในการทำงาน มันถูกออกแบบมาโดยเฉพาะเพื่อรวม Mailster เข้ากับระบบลูกค้าและสินค้าของ Ultimate Multisite

## ฉันจำเป็นต้องติดตั้ง Mailster บนทุกไซต์หรือไม่? {#do-i-need-mailster-installed-on-all-sites}

ไม่จำเป็น Mailster ต้องการเพียงแค่ติดตั้งและเปิดใช้งานบนเว็บไซต์หลักของเครือข่ายเท่านั้น addon จะเชื่อมต่อกับเว็บไซต์นั้นโดยอัตโนมัติจาก subsites ทั้งหมด

## ฉันสามารถกำหนดรายชื่อที่แตกต่างกันให้กับสินค้าที่แตกต่างกันได้หรือไม่? {#can-i-assign-different-lists-to-different-products}

ได้แน่นอน! คุณสามารถกำหนดรายชื่อเริ่มต้นทั่วโลก จากนั้นจึงกำหนดรายชื่อเฉพาะสินค้าเพื่อแทนที่ค่าเริ่มต้นได้ วิธีนี้ช่วยให้คุณสร้างแคมเปญอีเมลที่เจาะจงสูงตามสินค้าที่ลูกค้าซื้อ

## จะเกิดอะไรขึ้นถ้าลูกค้ามีอยู่ใน Mailster แล้ว? {#what-happens-if-a-customer-already-exists-in-mailster}

โดยค่าเริ่มต้น addon จะอัปเดตสมาชิกที่มีอยู่และเพิ่มพวกเขาเข้าสู่รายชื่อใหม่ คุณสามารถปิดใช้งานส่วนนี้ในการตั้งค่าได้หากคุณต้องการข้ามสมาชิกที่มีอยู่

## สิ่งนี้สอดคล้องกับ GDPR หรือไม่? {#is-this-gdpr-compliant}

ได้แน่นอน! addon นี้มีโหมด opt-in ที่ใช้ช่องทำเครื่องหมาย ซึ่งลูกค้าต้องให้ความยินยอมอย่างชัดเจนในการรับอีเมล ซึ่งเป็นไปตามข้อกำหนดของ GDPR เมื่อรวมกับนโยบายความเป็นส่วนตัวที่เหมาะสม

## ฉันสามารถใช้ double opt-in ได้หรือไม่? {#can-i-use-double-opt-in}

ได้อย่างแน่นอน! เปิดใช้งานการตั้งค่า double opt-in และ Mailster จะส่งอีเมลยืนยันก่อนที่จะเปิดใช้งานการสมัครสมาชิก
