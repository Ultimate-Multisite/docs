# บันทึกการเปลี่ยนแปลง Mailster

### 1.0.0 {#100}
* เปิดตัวครั้งแรก
* การซิงค์สมาชิกอัตโนมัติเมื่อมีการสมัครหรือชำระเงิน
* การกำหนดรายชื่อ (list assignments) เฉพาะสำหรับผลิตภัณฑ์
* การเลือกเข้าร่วม (opt-in) ด้วยช่องทำเครื่องหมายเพื่อการปฏิบัติตามข้อกำหนด GDPR
* รองรับการยืนยันตัวตนสองขั้นตอน (Double opt-in)
* การแมปฟิลด์ที่ครอบคลุม
* การบันทึกรายละเอียด (Detailed logging)
