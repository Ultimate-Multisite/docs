# AffiliateWP

Multisite Ultimate: การผสานรวมกับ AffiliateWP

ใช้ AffiliateWP ที่ทรงพลังเพื่อขยายฐานลูกค้าของ Ultimo Network ของคุณ!

## การติดตั้ง

1. อัปโหลด 'ultimate-multisite-git ' ไปยังไดเรกทอรี '/wp-content/plugins/'
2. เปิดใช้งานปลั๊กอินผ่านเมนู 'Plugins' ใน WordPress
3. ทำตามขั้นตอนของตัวช่วย (Wizard) เพื่อตั้งค่าปลั๊กอิน

## คู่มือการใช้งาน

[AffiliateWP](https://affiliatewp.com/) ได้กลายเป็นโซลูชันหลักสำหรับการสร้างโปรแกรมพันธมิตรสำหรับธุรกิจออนไลน์ของคุณ Ultimate Multisite: AffiliateWP คือสะพานที่ช่วยให้คุณสร้างสิ่งเดียวกันนี้สำหรับเครือข่ายระดับพรีเมียมของคุณได้

## เริ่มต้นใช้งาน

ก่อนอื่นเลย คุณ**ต้องเปิดใช้งาน** [**AffiliateWP**](https://affiliatewp.com/) **ก่อน**! คุณจะไม่สามารถใช้ add-on ของเราได้หากคุณไม่มี**ปลั๊กอินที่เชื่อมต่อกับมัน** ในกรณีที่คุณไม่ได้วางแผนที่จะให้ลูกค้าของคุณใช้บนเว็บไซต์ของพวกเขาด้วย AffiliateWP ควรเปิดใช้งานเฉพาะบนเว็บไซต์หลักของคุณเท่านั้น

นอกจากนี้ คุณจะต้องเปิดใช้งานทั้ง **Ultimate Multisite** และ **Ultimate Multisite: AffiliateWP** **ทั่วทั้งเครือข่าย** ด้วย

## การเพิ่ม Ultimate Multisite เป็นการผสานรวมกับ AffiliateWP

บนเว็บไซต์หลักของคุณ ให้ไปที่ Affiliates Settings และที่แท็บ Integrations ให้ทำเครื่องหมายที่ Ultimate Multisite สิ่งนี้จะแจ้งให้ [AffiliateWP](https://affiliatewp.com/) เริ่มติดตามการเข้าชมและการแปลง (conversions) จากขั้นตอนการลงทะเบียนของ Ultimate Multisite

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_ทำเครื่องหมายที่ช่อง Ultimate Multisite ด้านล่างแล้วบันทึก_

แค่นี้ก็พอแล้ว การเข้าชมที่มาจากลิงก์แนะนำ (referral links) จะถูกติดตาม และเมื่อผู้ใช้รายนั้นชำระเงินครั้งแรก การแปลง (conversion) จะถูกสร้างขึ้น และจะมีการกันค่าคอมมิชชันไว้ให้กับพันธมิตรที่รับผิดชอบการแปลงนั้น

อย่างไรก็ตาม จะมีการติดตามเฉพาะการชำระเงินครั้งแรกที่ได้รับจากผู้ใช้ที่สร้างขึ้นใหม่เท่านั้น หากต้องการเปิดใช้งานการชำระเงินแบบแนะนำซ้ำ (recurring referral payments) คุณจะต้องใช้ [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## การเปิดใช้งานการแนะนำซ้ำ (Recurring Referrals)

หลังจากติดตั้ง [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) แล้ว ให้ไปที่แท็บ Recurring Referrals ใน Affiliates Settings และเปิดใช้งานการแนะนำซ้ำ

<!-- Screenshot unavailable: AffiliateWP Recurring Referrals settings with Enable Recurring Referrals toggle -->

_ง่ายแสนง่าย: เพียงแค่เปิดใช้งาน Recurring Referrals คุณก็พร้อมใช้งานแล้ว!_

อีกครั้ง ย้ำว่าแค่นี้ก็พอแล้ว!

ตอนนี้ ทุกครั้งที่มีการชำระเงินใหม่ในบัญชีที่ถูกแนะนำมา พันธมิตรที่รับผิดชอบก็จะได้รับส่วนแบ่งของตัวเองโดยอัตโนมัติ!
