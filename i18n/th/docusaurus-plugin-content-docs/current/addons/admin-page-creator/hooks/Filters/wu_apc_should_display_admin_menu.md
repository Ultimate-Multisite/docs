# Filter: wu_apc_should_display_admin_menu

ฟิลเตอร์นี้ช่วยให้ผู้ดูแลระบบสามารถซ่อนหน้าเมนูแอดมินได้ เมื่อพวกเขาปรับแต่งทุกอย่างเสร็จเรียบร้อยแล้ว

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $should_display_admin_menu | `bool` | ค่านี้กำหนดว่าเราต้องการให้แสดงหน้าเมนูหรือไม่ |
| $network_active | `bool` | ตรวจสอบว่าปลั๊กอินกำลังทำงานในโหมดแอดมินเครือข่ายหรือไม่ |
