# บันทึกการเปลี่ยนแปลงของ Admin Page Creator

### Version 1.9.0 - Released on 2026-01-18 {#version-190---released-on-2026-01-18}

* แก้ไข: ข้อผิดพลาดร้ายแรงที่เกิดขึ้นตอนเปิดใช้งาน
* ปรับปรุงโครงสร้าง: ปรับปรุงคลาสแหล่งที่มาของเนื้อหาให้ใช้ Singleton trait พร้อมการกำหนดค่า constructor ของ parent อย่างถูกต้อง
* แก้ไข: ข้อผิดพลาดในการคอมไพล์เทมเพลตของ Vue.js ซึ่งทำให้ตัวเลือกประเภทเนื้อหาไม่แสดงผล
* แก้ไข: ข้อมูล property ของ Vue ที่ขาดหายไป (template_id, external_link_url_embedable, external_link_url_checking)
* แก้ไข: ตัวแก้ไข TinyMCE ไม่เริ่มต้นทำงานอย่างถูกต้องเมื่อโหลดหน้า
* แก้ไข: ไฟล์สนับสนุน Divi ไม่ถูกโหลดอัตโนมัติผ่าน Composer
* แก้ไข: ไวยากรณ์ PHP ที่เสียหายในคลาสสนับสนุน Divi
* ปรับปรุง: การจัดระเบียบและย่อขนาด (minification) ของ asset JavaScript
* ปรับปรุง: เพิ่มความเข้ากันได้กับ WordPress และเวอร์ชัน Page Builder ล่าสุด
