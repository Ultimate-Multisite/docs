# Material WP

หน้าผู้ดูแลระบบของ WordPress ดูเหมือนจะเหมือนเดิมมานานมากแล้ว คุณเคยสงสัยไหมว่าถ้ามันเป็นเวอร์ชัน Material Design มันจะหน้าตาเป็นอย่างไร?

Material WP คือธีมสำหรับแอดมินที่พัฒนาขึ้นเพื่อให้คุณได้รูปลักษณ์และความรู้สึกแบบนั้น ช่วยให้คุณสามารถมอบ Dashboard ที่ปรับแต่งได้มากขึ้นให้กับลูกค้าของคุณ

## คุณสมบัติ {#features}

- รองรับ Custom CSS
- การตั้งค่าสไตล์ที่ง่าย
- การจัดเรียงเมนูใหม่
- การนำเข้า/ส่งออกที่ง่าย

## Documentation {#documentation}

**การปรับแต่งแผงผู้ดูแลระบบของคุณด้วย Material WP**

## Installation {#installation}

1. อัปโหลด 'material-wp' ไปยังไดเรกทอรี '/wp-content/plugins/'
2. เปิดใช้งานปลั๊กอินผ่านเมนู 'Plugins' ใน WordPress

## Usage Guide {#usage-guide}

หน้าผู้ดูแลระบบของ WordPress ดูเหมือนจะเหมือนเดิมมานานมากแล้ว คุณเคยสงสัยไหมว่าถ้ามันเป็นเวอร์ชัน Material Design มันจะหน้าตาเป็นอย่างไร?

Material WP คือธีมสำหรับแอดมินที่พัฒนาขึ้นเพื่อให้คุณได้รูปลักษณ์และความรู้สึกแบบนั้น ช่วยให้คุณสามารถมอบ Dashboard ที่ปรับแต่งได้มากขึ้นให้กับลูกค้าของคุณ Material WP ยังเป็นส่วนหนึ่งของปลั๊กอินหลักระดับพรีเมียมของ Ultimate Multisite

หลังจาก **ติดตั้งแล้ว** ให้ไปที่หน้า _Plugins_ ของคุณแล้วคลิกเพื่อเปิดใช้งานบน Network ของคุณ
