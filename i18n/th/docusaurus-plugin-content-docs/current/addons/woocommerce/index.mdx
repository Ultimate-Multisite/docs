# การผสานรวมกับ WooCommerce

WooCommerce Integration ทำงานในลักษณะที่คล้ายคลึงกับ Manual Gateway ของเรามาก มันมอบความยืดหยุ่นอย่างมากให้กับผู้ใช้ของคุณ ทำให้พวกเขาสามารถเลือกรูปแบบการชำระเงินใดก็ได้ที่มีอยู่ใน WooCommerce ที่ติดตั้งไว้ และมันก็ง่ายมาก: เมื่อรอบบิลของผู้ใช้สิ้นสุดลง การสมัครสมาชิกของผู้ใช้จะถูกระงับไว้ ในขณะเดียวกัน ระบบจะสร้างคำสั่งซื้อบน WooCommerce ที่ติดตั้งบนเว็บไซต์หลักของเครือข่าย ผู้ใช้จะได้รับใบแจ้งหนี้ทางอีเมล พร้อมลิงก์สำหรับดำเนินการชำระเงิน เมื่อได้รับชำระเงินแล้ว การสมัครสมาชิกก็จะได้รับการต่ออายุ

## คุณสมบัติเด่น

- การผสานรวมที่ง่ายดายระหว่าง Ultimate Multisite และ WooCommerce
- รองรับวิธีการชำระเงินหลายรูปแบบ
- กระบวนการอัตโนมัติเต็มรูปแบบด้วย WooCommerce Subscriptions

## เอกสารประกอบ

**การตั้งค่า WooCommerce Integration**

**การขยายตัวเลือกการชำระเงินของคุณด้วย Ultimate Multisite: WooCommerce Integration**

## การติดตั้ง

1. อัปโหลด 'ultimate-multisite-woocommerce' ไปยังไดเรกทอรี '/wp-content/plugins/'
2. เปิดใช้งานปลั๊กอินผ่านเมนู 'Plugins' ใน WordPress
3. ทำตามขั้นตอนของตัวช่วย (Wizard) เพื่อตั้งค่าปลั๊กอิน

## คู่มือการใช้งาน

_**หมายเหตุสำคัญ: บทความนี้อ้างอิงถึง Ultimate Multisite เวอร์ชัน 2.x**_

_**ข้อควรระวัง:** Ultimate Multisite: WooCommerce Integration ต้องอาศัยการเปิดใช้งาน WooCommerce อย่างน้อยบนเว็บไซต์หลักของคุณ_

เราเข้าใจว่า _Stripe_ และ _PayPal_ อาจไม่พร้อมใช้งานในบางประเทศ ซึ่งจำกัดหรือขัดขวางไม่ให้ผู้ใช้ Ultimate Multisite ใช้ปลั๊กอินของเราได้อย่างมีประสิทธิภาพ ดังนั้นเราจึงสร้าง **ส่วนเสริม (add-on) เพื่อผสานรวมกับ _WooCommerce_**

_WooCommerce_ เป็นปลั๊กอินอีคอมเมิร์ซที่ได้รับความนิยมมาก นักพัฒนาทั่วโลกได้สร้างส่วนเสริมเพื่อผสานรวมเกตเวย์การชำระเงินที่แตกต่างกันเข้ากับปลั๊กอินนี้ เราจึงใช้ประโยชน์จากจุดนี้เพื่อขยายเกตเวย์การชำระเงินที่คุณสามารถใช้ได้ในระบบการเรียกเก็บเงินของ Ultimate Multisite

## การตั้งค่า

ก่อนอื่น คุณต้องติดตั้งส่วนเสริม WooCommerce Integration ซึ่ง **ฟรีสำหรับผู้ใช้ที่มีใบอนุญาต** ในการติดตั้ง ให้คลิกที่ Ultimate Multisite บน super admin dashboard ของคุณ และเลือก **Add-ons**

![Add-ons page](/img/admin/addons-page.png)

ที่นี่คุณจะพบส่วนเสริม Ultimate Multisite ทั้งหมด คลิกที่ส่วนเสริม **Ultimate Multisite: WooCommerce Integration**

![Add-ons page showing WooCommerce Integration](/img/admin/addons-page.png)

หน้าต่างจะปรากฏขึ้นพร้อมรายละเอียดของส่วนเสริม เพียงคลิกที่ **Install Now** *.*

<!-- Screenshot unavailable: WooCommerce Integration addon install popup with Install Now button -->

หลังจากติดตั้งเสร็จแล้ว คุณจะถูกนำทางไปยังหน้าปลั๊กอิน ที่นี่ให้คลิกที่ **Network Activate** และส่วนเสริม _WooCommerce_ ก็จะถูกเปิดใช้งานบนเครือข่ายของคุณ

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for WooCommerce Integration -->

หลังจากเปิดใช้งานแล้ว หากคุณยังไม่ได้ติดตั้งและเปิดใช้งานปลั๊กอิน WooCommerce บนเว็บไซต์ของคุณ **คุณจะได้รับการแจ้งเตือน**

<!-- Screenshot unavailable: Admin notice reminding to install and activate WooCommerce plugin -->

เมื่อติดตั้งแล้ว คุณจะเห็นตัวเลือกเพิ่มเติมภายใต้การตั้งค่าการชำระเงินของ Ultimate Multisite จากนั้นคุณสามารถ **เปิดใช้งาน** ตัวเลือก _WooCommerce_ ได้จากตรงนั้น

![Payment gateway settings](/img/config/settings-payment-gateways.png)

คุณสามารถกำหนดค่าพื้นฐานบางอย่างได้ เช่น ชื่อที่แสดงผลในส่วนหน้า (front end) และคำแนะนำเพื่อนำทางผู้ใช้ของคุณในขั้นตอนการชำระเงิน จากนั้นคุณสามารถไปที่ **การตั้งค่า WooCommerce ของเว็บไซต์หลัก** เพื่อเปิดใช้งานและกำหนดค่าเกตเวย์การชำระเงินที่คุณต้องการใช้

## วิธีการจัดการการชำระเงิน

WooCommerce Integration ทำงานในลักษณะที่คล้ายคลึงกับ **Manual Gateway** ของเรา ข้อเสียคือทุกการชำระเงินจะต้องชำระด้วยตนเองเมื่อสิ้นสุดรอบบิล นี่คือขั้นตอนการทำงาน:

  * เมื่อรอบบิลของผู้ใช้สิ้นสุดลง การเป็นสมาชิกจะถูกระงับไว้ ในขณะเดียวกัน ระบบจะสร้างคำสั่งซื้อบน _WooCommerce_ ที่ติดตั้งบนเว็บไซต์หลักของเครือข่ายของคุณ

  * ผู้ใช้จะได้รับใบแจ้งหนี้ทางอีเมล พร้อมลิงก์สำหรับดำเนินการชำระเงิน (ลิงก์นี้จะปรากฏในประวัติการเรียกเก็บเงินจาก dashboard ของ subsite ของพวกเขาด้วย) ลิงก์นั้นจะนำผู้ใช้ไปยังหน้า **checkout ของ _WooCommerce_** ซึ่งผู้ใช้จะสามารถเลือก **เกตเวย์ _WooCommerce_ ที่พร้อมใช้งาน** เพื่อดำเนินการชำระเงิน

  * เมื่อได้รับชำระเงินแล้ว การเป็นสมาชิกจะได้รับการ **ต่ออายุ** การชำระเงินจะถูกบันทึกใน Ultimate Multisite และคำสั่งซื้อจะถูกตั้งค่าให้เสร็จสมบูรณ์

### ข้อควรทราบที่สำคัญบางประการ:

  * ฟังก์ชันการทำงานเริ่มต้นของ Ultimate Multisite ยังคงใช้งานได้กับการผสานรวมนี้ ตัวอย่างเช่น การ **คืนเงิน** จากหน้า **Subscription Management** จะสื่อสารกับ WooCommerce เพื่อดำเนินการคืนเงินโดยใช้รูปแบบการชำระเงินที่เลือก

  * Ultimate Multisite: WooCommerce Integration ต้องอาศัย WP-Cron ในการทำงาน ซึ่งโดยปกติแล้วจะทำงานได้ดีมาก แต่หากเว็บไซต์มีการเข้าชมต่ำ ระบบ cron จะไม่ถูกกระตุ้นให้ตรวจสอบกิจกรรมที่กำหนดเวลา และกิจกรรมนั้นก็จะพลาดไป ซึ่งอาจทำให้เกิดความล่าช้าในการดำเนินการได้

## ปลั๊กอิน WooCommerce Subscriptions

การดำเนินการชำระเงินด้วยตนเองทุกเดือนไม่เหมาะสำหรับผู้ใช้ปลายทาง เนื่องจากโดยปกติแล้วการสมัครสมาชิกควรจะต่ออายุโดยอัตโนมัติในทุกรอบบิล ดังนั้นเราจึงออกแบบส่วนเสริม WooCommerce Integration เพื่อ **รองรับปลั๊กอิน WooCommerce Subscriptions** สิ่งนี้จะช่วยทำให้การชำระเงินเป็นไปโดยอัตโนมัติ ทำให้การทำธุรกรรมง่ายขึ้นสำหรับผู้ใช้ปลายทางของคุณ

สิ่งที่คุณต้องทำคือ **ติดตั้ง** [ปลั๊กอิน WooCommerce Subscriptions](https://woocommerce.com/products/woocommerce-subscriptions/) และให้ **เปิดใช้งานบนเว็บไซต์หลักของคุณ** จากนั้นกลับไปที่การตั้งค่าการชำระเงินของ Ultimate Multisite และ **เปิดใช้งานการผสานรวม**

และแค่นี้ก็เรียบร้อย ผู้ใช้ปลายทางของคุณจะไม่จำเป็นต้องชำระเงินด้วยตนเองเมื่อสิ้นสุดรอบบิลของพวกเขาอีกต่อไป

นอกเหนือจากการสร้างคำสั่งซื้อบนเว็บไซต์หลักของคุณด้วย _WooCommerce_ แล้ว ระบบยังจะสร้างการสมัครสมาชิกสำหรับแต่ละบัญชีด้วย
