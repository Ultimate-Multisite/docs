# บันทึกการเปลี่ยนแปลง WooCommerce Integration

Version 2.0.6 - เผยแพร่เมื่อ 2026-01-16
* การปรับปรุง: รวม core subscriptions เข้าไปใน addon แล้ว ไม่จำเป็นต้องใช้ Woocommerce Subscriptions extension อีกต่อไป

Version 2.0.5 - เผยแพร่เมื่อ 2026-01-09
* การปรับปรุง: โหลดคำแปลจาก glotpress API
* การแก้ไข: แก้ไขข้อผิดพลาดร้ายแรง (Fatal error) ที่เกิดขึ้นใน Page Builder บางตัว
* การแก้ไข: แก้ปัญหาการเปลี่ยนเส้นทางแบบวนลูปไม่รู้จบ (Infinite redirect) เมื่อลูกค้าเป็นสมาชิกของเว็บไซต์หลัก

Version 2.0.4 - เผยแพร่เมื่อ 2025-11-14
* เพิ่มเติม: คำแปลสำหรับภาษาจำนวนมากขึ้น
* เปลี่ยนแปลง: เปลี่ยนชื่อเป็น Ultimate Multisite: Woocommerce Integration
* เพิ่มเติม: รองรับกับ Woocommerce 10.2.1
* เพิ่มเติม: รองรับกับ Woocommerce Subscriptions 7.7.0
* การแก้ไข: รองรับกับ PHP 8.4
* การแก้ไข: แก้ปัญหาการเปลี่ยนเส้นทางเมื่อไม่มีหน้าบัญชี WC

Version 2.0.3 - เผยแพร่เมื่อ 2025-08-13
* เปลี่ยนแปลง: เปิดใช้งานการอัปเดตอัตโนมัติด้วย marketplace ใหม่

Version 2.0.2 - เผยแพร่เมื่อ 2025-07-05
* เปลี่ยนแปลง: เปลี่ยนชื่อเป็น Multisite Ultimate: Woocommerce Integration
* เพิ่มเติม: รองรับกับ Woocommerce 9.8.1
* เพิ่มเติม: รองรับกับ Woocommerce Subscriptions 7.3.0
* การแก้ไข: แก้ปัญหาการยกเลิกการสมัครสมาชิกโดยลูกค้า
* การแก้ไข: แก้ข้อผิดพลาดร้ายแรงเมื่อใช้ checkout block
* การปรับปรุง: ตอนนี้รองรับกับตารางคำสั่งซื้อแบบกำหนดเองประสิทธิภาพสูงของ Woocommerce
* การแก้ไข: การยกเลิกที่หน้า checkout ของ WooCommerce ยังสามารถอัปเกรดสมาชิกได้

Version 2.0.1 - เผยแพร่เมื่อ 2023-08-09

* เพิ่มเติม: รองรับกับ Woocommerce 7.9.0
* เพิ่มเติม: รองรับกับ Woocommerce Subscriptions 5.3.0
* เพิ่มเติม: รองรับการอัปเดตสมาชิก
* เพิ่มเติม: เพิ่มการแจ้งเตือนเกี่ยวกับช่วงทดลองใช้และค่าธรรมเนียมการตั้งค่าใน Woocommerce
* เพิ่มเติม: ระบุผลิตภัณฑ์ Woocommerce ของ Ultimate Multisite ด้วย meta value
* เพิ่มเติม: ใส่การแก้ไขชั่วคราวเพื่อทำเครื่องหมายผลิตภัณฑ์ Woocommerce ทั้งหมดที่เกี่ยวข้องกับ Ultimate Multisite
* เพิ่มเติม: ลบผลิตภัณฑ์ที่สร้างโดย Ultimate Multisite ออกจากรายการ Woocommerce
* การปรับปรุง: สร้างส่วนลด Woocommerce แบบไม่เกิดขึ้นซ้ำเพื่อใช้กับตะกร้าสินค้า
* การปรับปรุง: กู้คืนส่วนลดแบบเกิดขึ้นซ้ำไปยังผลิตภัณฑ์ Woocommerce
* การปรับปรุง: เพิ่มป้ายกำกับส่วนลดแบบเกิดขึ้นซ้ำไปยังผลิตภัณฑ์ Woocommerce
* การปรับปรุง: ตรวจสอบประเภทผลิตภัณฑ์ที่หน้า checkout
* การแก้ไข: คงสถานะสมาชิกไว้ระหว่างกระบวนการลดระดับ (downgrade)
* การแก้ไข: ตรวจสอบว่ามีการสมัครสมาชิกอยู่หรือไม่ เพื่อหลีกเลี่ยงข้อผิดพลาดระหว่างกระบวนการยกเลิก
* การแก้ไข: เพิ่มวันที่เริ่มการสมัครสมาชิกเพื่อใช้ใน Woocommerce subscriptions
* ภายใน: ใช้กระบวนการ build PHP 8.1 ใหม่

Version 2.0.0 - เขียนใหม่ทั้งหมด

* เพิ่มเติม: วิธีการยกเลิกกระบวนการเพื่อลบการสมัครสมาชิก woo เมื่อเปลี่ยน gateway หรือยกเลิกสมาชิก;
* เพิ่มเติม: ตัวจัดการสำหรับการลดและอัปเกรดสมาชิก;
* การปรับปรุง: โหลด dependencies ของ woocommerce บนฟอร์มอัปเดตลูกค้าใน subsites เพื่อให้สามารถอัปเดตบัญชีได้;
* การปรับปรุง: แก้ไขการโหลด Woocommerce cart หากไม่มีอยู่;
* การปรับปรุง: ตรวจสอบให้แน่ใจว่าเราอยู่บนตารางของ main site เมื่อดำเนินการ checkout;
* การปรับปรุง: ทำให้คำสั่งซื้อการต่ออายุ Ultimo อิงตามมูลค่าคำสั่งซื้อการสมัครสมาชิกของ Woocommerce ไม่ใช่จากการชำระเงินครั้งล่าสุด;
* การแก้ไข: ไปที่ลิงก์ปุ่ม WU Membership;
* การแก้ไข: ตั้งค่าคำสั่งซื้อ Ultimo เป็นชำระเงินเมื่อการต่ออายุ woocommerce subscriptions ได้ชำระเงินแล้ว;
* Build: เพิ่ม MPB เป็น builder

Version 2.0.0-beta-5 - เผยแพร่เมื่อ 2022-01-21

* ภายใน: เพิ่ม hooks และ filters generator;
* ภายใน: เพิ่ม Ultimate Multisite stubs เพื่อคุณภาพชีวิตของนักพัฒนา;
* แก้ไข: ป้องกันการสร้างผลิตภัณฑ์หลายรายการเมื่อไม่จำเป็น;

Version 2.0.0-beta.4 - 2021-09-23

* การแก้ไข: กำหนดให้ WooCommerce ต้องอยู่ในสถานะ network active แทนที่จะเป็นแค่ main site;
* การปรับปรุง: เพิ่ม filter เพื่อให้ add-on สามารถใช้เป็น mu-plugin ได้

Version 2.0.0-beta.3 - 2021-05-28

* การแก้ไข: การควบคุมการเข้าถึง dashboard เข้มงวดเกินไป;
* การปรับปรุง: เพิ่มลิงก์ช่วยเหลือ WooCommerce ไปที่เมนูหลักของ Ultimate Multisite;

Version 2.0.0-beta.2 - 2021-05-04

* การปรับปรุง: สร้างการชำระเงินที่รอดำเนินการ (pending payments) บน Ultimo เมื่อสร้างคำสั่งซื้อการต่ออายุ WCS;
* การปรับปรุง: กรอกข้อมูลช่องทางการเรียกเก็บเงินด้วยข้อมูลลูกค้า Ultimate Multisite;
* การปรับปรุง: เพิ่มช่องทางการเรียกเก็บเงินสำหรับ gateway ต่างๆ;

Version 2.0.0-beta.1 - 2021-05-04

* การปล่อยรุ่น beta ครั้งแรก
