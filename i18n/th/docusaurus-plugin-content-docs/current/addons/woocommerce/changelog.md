# บันทึกการเปลี่ยนแปลงการผสานการทำงาน WooCommerce

เวอร์ชัน 2.2.0 - เผยแพร่เมื่อ 2026-07-01
* ใหม่: ยอดภาษีของ Ultimate Multisite จะแสดงเป็นบรรทัดค่าธรรมเนียม WooCommerce แยกต่างหากที่ checkout แล้ว ทำให้ยอดรวมภาษีชัดเจนขึ้นก่อนชำระเงิน
* ใหม่: เพิ่มการตั้งค่าแบบเลือกเปิดใช้งาน "ระงับ Memberships ทันทีเมื่อการต่ออายุล้มเหลว" และตัวกรอง `wu_woo_suspend_on_payment_failure` สำหรับไซต์ที่ต้องการระงับเมื่อการต่ออายุล้มเหลวในช่วงหน้าต่างลองใหม่ของ WooCommerce Subscriptions
* แก้ไข: ปรับสถานะ WooCommerce Subscription ที่อาจไม่ซิงก์กับ membership ของ Ultimate Multisite หลังจากการต่ออายุล้มเหลวหรือกู้คืนได้
* แก้ไข: เพิ่มสกุลเงินของร้านค้า WooCommerce ไปยังรายการสกุลเงินของ Ultimate Multisite เมื่อไม่มีอยู่
* แก้ไข: เก็บรายละเอียดการเรียกเก็บเงินของลูกค้าไว้เมื่อเปลี่ยนเส้นทางผู้สมัครสมาชิกไปยัง WooCommerce checkout
* ปรับปรุง: เพิ่มความเข้ากันได้กับ Jetpack Autoloader 5
* ปรับปรุง: จัดระเบียบการสร้างแพ็กเกจเผยแพร่เพื่อให้ไฟล์ zip ของ GitHub และ marketplace หลีกเลี่ยงไดเรกทอรี staging ซ้อนกันและไฟล์สำหรับการพัฒนา

เวอร์ชัน 2.0.6 - เผยแพร่เมื่อ 2026-01-16
* ปรับปรุง: รวม subscription หลักไว้ใน addon ไม่ต้องใช้ส่วนขยาย Woocommerce Subscriptinos อีกต่อไป

เวอร์ชัน 2.0.5 - เผยแพร่เมื่อ 2026-01-09
* ปรับปรุง: โหลดคำแปลจาก glotpress API
* แก้ไข: ข้อผิดพลาดร้ายแรงใน page builder บางตัว
* แก้ไข: การเปลี่ยนเส้นทางไม่สิ้นสุดเมื่อลูกค้าเป็นสมาชิกของไซต์หลัก

เวอร์ชัน 2.0.4 - เผยแพร่เมื่อ 2025-11-14
* เพิ่ม: คำแปลสำหรับภาษาอื่น ๆ อีกมากมาย
* เปลี่ยนแปลง: เปลี่ยนชื่อเป็น Ultimate Multisite: Woocommerce Integration
* เพิ่ม: ความเข้ากันได้กับ Woocommerce 10.2.1
* เพิ่ม: ความเข้ากันได้กับ Woocommerce Subscriptions 7.7.0
* แก้ไข: ความเข้ากันได้กับ PHP 8.4
* แก้ไข: เปลี่ยนเส้นทาง แต่เมื่อไม่มีหน้า Account ของ WC

เวอร์ชัน 2.0.3 - เผยแพร่เมื่อ 2025-08-13
* เปลี่ยนแปลง: เปิดใช้งานการอัปเดตอัตโนมัติกับ marketplace ใหม่

เวอร์ชัน 2.0.2 - เผยแพร่เมื่อ 2025-07-05
* เปลี่ยนแปลง: เปลี่ยนชื่อเป็น Multisite Ultimate: Woocommerce Integration
* เพิ่ม: ความเข้ากันได้กับ Woocommerce 9.8.1
* เพิ่ม: ความเข้ากันได้กับ Woocommerce Subscriptions 7.3.0
* แก้ไข: การยกเลิก subscription โดยลูกค้า
* แก้ไข: ข้อผิดพลาดร้ายแรงเมื่อใช้ checkout block
* ปรับปรุง: ตอนนี้เข้ากันได้กับตารางคำสั่งซื้อแบบกำหนดเองประสิทธิภาพสูงของ Woocommerce
* แก้ไข: การยกเลิกที่ WooCommerce checkout ยังสามารถอัปเกรด membership ได้

เวอร์ชัน 2.0.1 - เผยแพร่เมื่อ 2023-08-09

* เพิ่ม: ความเข้ากันได้กับ Woocommerce 7.9.0
* เพิ่ม: ความเข้ากันได้กับ Woocommerce Subscriptions 5.3.0
* เพิ่ม: รองรับการอัปเดต membership
* เพิ่ม: ประกาศเกี่ยวกับช่วงทดลองใช้และค่าธรรมเนียมการตั้งค่าใน Woocommerce
* เพิ่ม: การระบุผลิตภัณฑ์ Ultimate Multisite Woocommerce ด้วยค่า meta
* เพิ่ม: แทรกการแก้ไขแบบครั้งเดียวเพื่อทำเครื่องหมายผลิตภัณฑ์ Woocommerce ทั้งหมดที่เกี่ยวข้องกับ Ultimate Multisite
* เพิ่ม: นำผลิตภัณฑ์ที่ Ultimate Multisite สร้างออกจากรายการ Woocommerce
* ปรับปรุง: สร้างส่วนลด Woocommerce แบบไม่เกิดซ้ำเพื่อใช้กับตะกร้าสินค้า
* ปรับปรุง: กู้คืนส่วนลดแบบเกิดซ้ำไปยังผลิตภัณฑ์ Woocommerce
* ปรับปรุง: เพิ่มป้ายกำกับส่วนลดแบบเกิดซ้ำไปยังผลิตภัณฑ์ Woocommerce
* ปรับปรุง: ตรวจสอบให้แน่ใจว่าประเภทผลิตภัณฑ์ถูกต้องบน checkout
* แก้ไข: รักษาสถานะ membership ระหว่างกระบวนการดาวน์เกรด
* แก้ไข: ตรวจสอบว่ามี subscription อยู่หรือไม่เพื่อหลีกเลี่ยงข้อผิดพลาดระหว่างกระบวนการยกเลิก
* แก้ไข: เพิ่มวันที่เริ่มต้น subscription เพื่อใช้ใน Woocommerce subscriptions
* ภายใน: ใช้กระบวนการ build ใหม่สำหรับ PHP 8.1

เวอร์ชัน 2.0.0 - เขียนใหม่ทั้งหมด

* เพิ่ม: เมธอดประมวลผลการยกเลิกเพื่อลบ woo subscription เมื่อเปลี่ยน gateway หรือยกเลิก membership;
* เพิ่ม: Handler เพื่อดาวน์เกรดและอัปเกรด memberships;
* ปรับปรุง: โหลด dependency ของ woocommerce ในฟอร์มอัปเดตลูกค้าในไซต์ย่อยเพื่อให้สามารถอัปเดต account ได้;
* ปรับปรุง: โหลดตะกร้า Woocommerce ให้ถูกต้องหากไม่มีอยู่;
* ปรับปรุง: ตรวจสอบให้แน่ใจว่าเราอยู่บนตารางของไซต์หลักเมื่อประมวลผล checkout;
* ปรับปรุง: ทำให้คำสั่งซื้อต่ออายุของ Ultimo อิงตามมูลค่าคำสั่งซื้อ subscription ของ Woocommerce และไม่ใช่จากการชำระเงินล่าสุด;
* แก้ไข: ลิงก์ปุ่มไปยัง WU Membership;
* แก้ไข: ตั้งคำสั่งซื้อ Ultimo เป็นชำระแล้วเมื่อการต่ออายุ Woocommerce subscriptions ชำระแล้ว;
* Build: เพิ่ม MPB เป็น builder;
