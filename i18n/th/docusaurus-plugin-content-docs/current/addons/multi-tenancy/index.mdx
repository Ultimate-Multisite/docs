# Multi-Tenancy

กระจายซับไซต์ (subsites) ไปยังเซิร์ฟเวอร์และฐานข้อมูลที่แตกต่างกัน เพื่อเพิ่มประสิทธิภาพและความปลอดภัย

## ข้อกำหนดเบื้องต้น

- WordPress 5.3 หรือสูงกว่า
- PHP 7.4 หรือสูงกว่า
- ปลั๊กอิน Multisite Ultimate (ต้องเปิดใช้งาน)

## การติดตั้ง

1. อัปโหลดไฟล์ addon ไปยังไดเรกทอรี `/wp-content/plugins/` ของคุณ
2. เปิดใช้งานปลั๊กอินผ่านเมนู 'Plugins' ใน WordPress
3. กำหนดค่าการตั้งค่าของ addon ในส่วนผู้ดูแลระบบเครือข่าย (network admin)

## การพัฒนา

### การตั้งค่า

```bash
# ติดตั้ง dependencies
composer install
npm install

# รันการทดสอบ
npm run test

# รันการตรวจสอบมาตรฐานโค้ด
vendor/bin/phpcs
vendor/bin/phpstan

# แก้ไขปัญหาด้านรูปแบบโค้ด
vendor/bin/phpcbf

# Build สำหรับ Production
npm run build
```

### การทดสอบ

Addon นี้มาพร้อมกับเฟรมเวิร์กการทดสอบที่ครอบคลุม:

- **Unit Tests**: การทดสอบที่ใช้ PHPUnit ร่วมกับ WordPress test suite
- **Base Test Class**: มีเมธอดช่วยเหลือสำหรับสถานการณ์การทดสอบทั่วไป
- **Test Coverage**: ตั้งค่าเพื่อติดตามความครอบคลุมของโค้ด (Code Coverage)
- **Multisite Testing**: การทดสอบที่รันในสภาพแวดล้อมแบบ Multisite

### มาตรฐานโค้ด

- **WordPress Coding Standards**: บังคับใช้ผ่าน PHPCS
- **PHP 7.4+ Compatibility**: รองรับฟีเจอร์ PHP สมัยใหม่
- **Static Analysis**: การรวม PHPStan เพื่อตรวจสอบประเภทข้อมูล (Type Checking)
- **Code Modernization**: กฎของ Rector สำหรับการอัปเกรด PHP

### ระบบ Build

- **Asset Processing**: การย่อขนาด (Minification) สำหรับไฟล์ CSS/JS
- **Translation**: การสร้างไฟล์ POT
- **Packaging**: การสร้างไฟล์เก็บถาวร (Archive) สำหรับการแจกจ่าย
- **Development Mode**: ไฟล์ที่ไม่ได้ย่อขนาด (Unminified) สำหรับการดีบัก (Debugging)

## การตั้งค่า (Configuration)

Addon นี้รองรับตัวเลือกการตั้งค่าที่หลากหลาย ซึ่งสามารถเข้าถึงได้ผ่านหน้าผู้ดูแลระบบเครือข่าย (network admin interface)

## Hooks และ Filters

### Actions (การกระทำ)

- `ultimate-multisite-addon-template_init` - ทำงานหลังจากเริ่มต้น Addon
- `ultimate-multisite-addon-template_loaded` - ทำงานหลังจากโหลดไฟล์ Addon ทั้งหมด

### Filters (ตัวกรอง)

- `ultimate-multisite-addon-template_settings` - ใช้แก้ไขการตั้งค่าของ Addon
- `ultimate-multisite-addon-template_enabled` - ใช้กำหนดสถานะเปิด/ปิดของ Addon ใหม่
