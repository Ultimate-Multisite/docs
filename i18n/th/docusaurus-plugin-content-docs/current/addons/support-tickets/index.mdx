# Support Tickets

**Support Tickets** เป็น addon ที่ทรงพลังสำหรับ Ultimate Multisite ซึ่งเพิ่มระบบตั๋วสนับสนุน (support ticket system) ที่ครอบคลุมให้กับเครือข่าย WordPress ของคุณ เหมาะอย่างยิ่งสำหรับแพลตฟอร์ม SaaS, ผู้ให้บริการ WaaS (Website as a Service), และเครือข่าย multisite ที่ต้องการให้บริการลูกค้าทั้งในระดับเครือข่ายและระดับเว็บไซต์ย่อย

## ทำไมต้องเลือก Support Tickets?

* **การสนับสนุนหลายระดับ (Multi-Level Support)**: ผู้ดูแลระบบเครือข่ายจัดการตั๋วในระดับเครือข่าย ในขณะที่เจ้าของเว็บไซต์จัดการการสนับสนุนลูกค้าของตนเอง
* **ขั้นตอนการทำงานระดับมืออาชีพ (Professional Workflow)**: บริหารจัดการวงจรชีวิตของตั๋วได้อย่างสมบูรณ์ ตั้งแต่การกำหนดลำดับความสำคัญ สถานะ และการมอบหมายงาน
* **ประสบการณ์ลูกค้า (Customer Experience)**: อินเทอร์เฟซที่สวยงามและตอบสนองได้ง่าย ทำให้ลูกค้าสามารถขอความช่วยเหลือได้อย่างสะดวก
* **ปลอดภัยและเชื่อถือได้ (Secure & Reliable)**: สร้างขึ้นตามแนวทางปฏิบัติที่ดีที่สุดด้านความปลอดภัยของ WordPress และปรับปรุงประสิทธิภาพฐานข้อมูลให้เหมาะสม
* **การผสานรวมที่ราบรื่น (Seamless Integration)**: ทำงานร่วมกับระบบจัดการลูกค้าและเว็บไซต์ของ Ultimate Multisite ได้อย่างสมบูรณ์แบบ

## คุณสมบัติหลัก (Key Features)

**การจัดการตั๋วที่สมบูรณ์ (Complete Ticket Management)**

* ระบบหมายเลขตั๋วที่ไม่ซ้ำกัน เพื่อการอ้างอิงที่ง่ายดาย
* ระดับความสำคัญที่ปรับแต่งได้ (Low, Normal, High, Critical)
* ขั้นตอนการทำงานของสถานะที่ครอบคลุม (Open, In Progress, Waiting, Resolved, Closed)
* การจัดหมวดหมู่ตั๋ว (General, Technical, Billing, Feature Request, Bug Report)
* การมอบหมายตั๋วให้กับเจ้าหน้าที่สนับสนุน โดยมีตัวเลือกการมอบหมายอัตโนมัติ

**การสนทนาแบบต่อเนื่อง (Threaded Conversations)**

* การสนทนาแบบต่อเนื่องเต็มรูปแบบสำหรับการตอบกลับตั๋ว
* การจัดรูปแบบที่แตกต่างกันสำหรับการตอบกลับของเจ้าหน้าที่และลูกค้า
* บันทึกส่วนตัวของเจ้าหน้าที่ที่ลูกค้าไม่สามารถเห็นได้
* อินเทอร์เฟซที่ขับเคลื่อนด้วย AJAX แบบเรียลไทม์
* ตัวแก้ไขข้อความที่ทันสมัยพร้อมตัวเลือกการจัดรูปแบบ

**การแนบไฟล์ (File Attachments)**

* การอัปโหลดไฟล์ที่ปลอดภัยพร้อมรองรับการลากและวาง (drag-and-drop)
* ข้อจำกัดประเภทและขนาดไฟล์ที่ปรับแต่งได้
* ความปลอดภัยขั้นสูงด้วยการตรวจจับไฟล์ที่เป็นอันตราย
* การดาวน์โหลดที่ได้รับการป้องกันพร้อมการควบคุมการเข้าถึง

**การแจ้งเตือนอัจฉริยะ (Smart Notifications)**

* การแจ้งเตือนทางอีเมลอัตโนมัติสำหรับตั๋วใหม่ การตอบกลับ และการเปลี่ยนแปลงสถานะ
* เทมเพลตอีเมลที่ปรับแต่งได้
* การแจ้งเตือนตามบทบาทสำหรับเจ้าหน้าที่และลูกค้า
* ตัวแปรเทมเพลตแบบไดนามิกสำหรับข้อความส่วนบุคคล

**อินเทอร์เฟซผู้ดูแลระบบที่ทรงพลัง (Powerful Admin Interface)**

* รายการตั๋วขั้นสูงพร้อมการจัดเรียงและตัวกรอง
* การแก้ไขด่วนสำหรับสถานะ ความสำคัญ และการมอบหมายงาน
* การดำเนินการแบบกลุ่ม (Bulk actions) เพื่อการจัดการที่มีประสิทธิภาพ
* แดชบอร์ดสถิติพร้อมตัวชี้วัดประสิทธิภาพ
* แผงการตั้งค่าที่ครอบคลุม

**พอร์ทัลลูกค้าที่สวยงาม (Beautiful Customer Portal)**

* อินเทอร์เฟซที่สะอาดตาและตอบสนองได้บนทุกอุปกรณ์
* การออกแบบที่เน้นมือถือเป็นหลัก (Mobile-first design)
* การค้นหาและตัวกรองขั้นสูง
* ตัวแก้ไขที่ทันสมัยพร้อมแถบเครื่องมือการจัดรูปแบบ
* แบบฟอร์มการส่งตั๋วที่ใช้งานง่าย

**การผสานรวมที่ยืดหยุ่น (Flexible Integration)**

* ใช้ shortcodes หรือองค์ประกอบ UI ของ Ultimate Multisite
* แสดงแบบฟอร์มและรายการตั๋วได้ทุกที่บนเว็บไซต์ของคุณ
* ทำงานร่วมกับโครงสร้างเครือข่าย multisite ได้อย่างราบรื่น
* ขยายได้ด้วย hooks และ filters

## เหมาะสำหรับ (Perfect For)

* แพลตฟอร์ม SaaS ที่ต้องการการสนับสนุนลูกค้า
* ผู้ให้บริการ WaaS ที่จัดการเว็บไซต์ของลูกค้าหลายราย
* เว็บไซต์สมาชิกที่มีการสนับสนุนหลายระดับ
* ผู้ดูแลระบบเครือข่ายที่จัดการแบรนด์หลายแบรนด์
* เจ้าของเว็บไซต์ที่ให้บริการลูกค้า

## การติดตั้ง (Installation)

1. ตรวจสอบให้แน่ใจว่าได้ติดตั้งและเปิดใช้งานปลั๊กอิน Ultimate Multisite บนเครือข่าย WordPress ของคุณแล้ว
2. อัปโหลดไฟล์ addon ไปยังไดเรกทอรี `/wp-content/plugins/` ของคุณ
3. เปิดใช้งานปลั๊กอินในระดับเครือข่ายผ่านเมนู 'Plugins' ใน WordPress
4. ไปที่ Network Admin > Support Tickets เพื่อกำหนดค่าการตั้งค่าของคุณ

## คำถามที่พบบ่อย (Frequently Asked Questions)

## ต้องใช้ Ultimate Multisite หรือไม่?

ใช่ addon นี้ต้องใช้ Ultimate Multisite ที่ติดตั้งและเปิดใช้งานอยู่ มันมีการผสานรวมอย่างลึกซึ้งกับคุณสมบัติการจัดการลูกค้าและเว็บไซต์ของ Ultimate Multisite

## เจ้าของเว็บไซต์สามารถจัดการการสนับสนุนลูกค้าของตนเองได้หรือไม่?

แน่นอนครับ/ค่ะ! เจ้าของเว็บไซต์สามารถใช้ระบบตั๋วเพื่อให้บริการลูกค้าของตนเองได้ ในขณะที่ผู้ดูแลระบบเครือข่ายสามารถจัดการตั๋วสนับสนุนในระดับเครือข่ายได้

## ลูกค้าส่งตั๋วได้อย่างไร?

คุณสามารถเพิ่มแบบฟอร์มส่งตั๋วได้ทุกที่บนเว็บไซต์ของคุณโดยใช้ shortcodes หรือองค์ประกอบ UI ของ Ultimate Multisite ลูกค้าเพียงแค่กรอกแบบฟอร์มและส่งคำขอของพวกเขา

## ฉันสามารถปรับแต่งสถานะและความสำคัญของตั๋วได้หรือไม่?

ได้ครับ/ค่ะ! คุณสามารถปรับแต่งระดับความสำคัญ สถานะ และประเภทตั๋วได้อย่างเต็มที่ในการตั้งค่าผู้ดูแลระบบเครือข่าย เพื่อให้ตรงกับขั้นตอนการทำงานด้านการสนับสนุนของคุณ

## รองรับการแนบไฟล์หรือไม่?

ได้ครับ/ค่ะ ทั้งลูกค้าและเจ้าหน้าที่สามารถแนบไฟล์ไปยังตั๋วและการตอบกลับได้ คุณสามารถควบคุมประเภทไฟล์และขีดจำกัดขนาดที่อนุญาตได้อย่างเต็มที่เพื่อความปลอดภัย

## การแจ้งเตือนถูกจัดการอย่างไร?

ระบบจะส่งการแจ้งเตือนทางอีเมลโดยอัตโนมัติสำหรับตั๋วใหม่ การตอบกลับ และการเปลี่ยนแปลงสถานะ คุณสามารถปรับแต่งเทมเพลตอีเมลทั้งหมดได้ในการตั้งค่า

## ฉันสามารถมอบหมายตั๋วให้กับเจ้าหน้าที่สนับสนุนที่เฉพาะเจาะจงได้หรือไม่?

ได้ครับ/ค่ะ คุณสามารถมอบหมายตั๋วให้กับเจ้าหน้าที่ด้วยตนเอง หรือเปิดใช้งานการมอบหมายอัตโนมัติเพื่อกระจายตั๋วโดยอัตโนมัติ

## มีบันทึกส่วนตัวสำหรับเจ้าหน้าที่หรือไม่?

ได้ครับ/ค่ะ เจ้าหน้าที่สามารถเพิ่มการตอบกลับส่วนตัวที่มองเห็นได้เฉพาะเจ้าหน้าที่คนอื่นเท่านั้น ไม่ใช่ลูกค้า

## ใช้งานบนมือถือได้หรือไม่?

แน่นอนครับ/ค่ะ! อินเทอร์เฟซทั้งหมดถูกสร้างขึ้นด้วยแนวคิด Mobile-first และทำงานได้อย่างสวยงามบนทุกอุปกรณ์

## ฉันสามารถแสดงรายการและแบบฟอร์มตั๋วโดยใช้ shortcodes ได้หรือไม่?

ได้ครับ/ค่ะ shortcodes ที่มีให้ใช้:
* `[wu_support_tickets]` - แสดงรายการตั๋วของลูกค้า
* `[wu_support_ticket_form]` - แสดงแบบฟอร์มส่งตั๋วใหม่
* `[wu_support_ticket_view ticket_id="123"]` - แสดงตั๋วเฉพาะรายการ

## สามารถอัปโหลดไฟล์ประเภทใดได้บ้าง?

คุณสามารถกำหนดประเภทไฟล์ที่อนุญาตได้ในการตั้งค่า ระบบมีการสแกนความปลอดภัยขั้นสูงเพื่อป้องกันไฟล์ที่เป็นอันตราย

## ฉันสามารถส่งออกข้อมูลตั๋วได้หรือไม่?

ได้ครับ/ค่ะ addon นี้มีคุณสมบัติการส่งออกข้อมูลสำหรับข้อมูลสถิติและข้อมูลตั๋ว
