# บทที่ 7: ทำให้เป็นของคุณเอง

ลูกค้าของคุณไม่ควรรู้สึกว่ากำลังใช้ "plugin WordPress ตัวใดตัวหนึ่ง" พวกเขาควรรู้สึกว่ากำลังใช้ FitSite ซึ่งเป็นแพลตฟอร์มที่สร้างมาเพื่ออุตสาหกรรมของพวกเขาโดยเฉพาะ บทเรียนนี้จะครอบคลุมเรื่องการสร้างแบรนด์ (branding), การทำ White-labeling, และการทำให้แพลตฟอร์มนี้รู้สึกเหมือนเป็นผลิตภัณฑ์ที่สมบูรณ์

## จุดที่เราทำไปแล้ว

FitSite มีขั้นตอนการชำระเงิน (checkout flow) ที่ใช้งานได้จริง ซึ่งนำเจ้าของสตูดิโอฟิตเนสตั้งแต่การเลือกแผนไปจนถึงเว็บไซต์ที่ใช้งานได้จริง ตอนนี้เราจะทำให้ประสบการณ์ทั้งหมดรู้สึกเหมือนเป็นผลิตภัณฑ์ที่มีแบรนด์ที่สอดคล้องกัน

## โดเมนแพลตฟอร์มของคุณ

รากฐานของแบรนด์คุณคือโดเมน (domain) สำหรับ FitSite:

- **โดเมนหลัก**: `fitsite.com` (เว็บไซต์ทางการตลาดและรากของเครือข่าย)
- **เว็บไซต์ลูกค้า**: `studioname.fitsite.com` (subdomains)
- **โดเมนที่กำหนดเอง**: ลูกค้าที่ใช้แผน Growth และ Pro สามารถเชื่อมโยงโดเมนของตัวเองได้

### การตั้งค่าโดเมนของคุณ

1. ลงทะเบียนโดเมนแพลตฟอร์มของคุณ
2. ชี้โดเมนไปยังผู้ให้บริการโฮสติ้งของคุณ
3. กำหนดค่า Wildcard DNS (`*.fitsite.com`) สำหรับ subdomains ของลูกค้า
4. ตรวจสอบให้แน่ใจว่า Wildcard SSL ทำงานอยู่

ดู [วิธีตั้งค่า Domain Mapping](/user-guide/domain-mapping/how-to-configure-domain-mapping) สำหรับคำแนะนำโดยละเอียด

## การทำ White-Labeling ประสบการณ์ Admin

เมื่อเจ้าของสตูดิโอฟิตเนสเข้าสู่ระบบ Dashboard ของเว็บไซต์ พวกเขาควรเห็นแบรนด์ของคุณ ไม่ใช่แบรนด์ของ WordPress หรือ Ultimate Multisite

### หน้าเข้าสู่ระบบที่กำหนดเอง (Custom Login Page)

ปรับแต่งหน้าเข้าสู่ระบบของ WordPress เพื่อแสดง:

- โลโก้ FitSite ของคุณ
- ภาพพื้นหลังที่เหมาะสมกับวงการฟิตเนส
- สีแบรนด์ของคุณ

### การสร้างแบรนด์ Dashboard

ใช้ addon [Admin Page Creator](/addons/admin-page-creator) หรือ CSS แบบกำหนดเองเพื่อ:

- แทนที่โลโก้ WordPress ด้วยโลโก้ FitSite ของคุณ
- ปรับแต่งชุดสีของ Admin ให้เข้ากับแบรนด์ของคุณ
- เพิ่ม Widget บน Dashboard ที่กำหนดเอง พร้อมลิงก์ด่วนและแหล่งข้อมูลช่วยเหลือที่เฉพาะเจาะจงสำหรับวงการฟิตเนส

### หน้า Admin ที่กำหนดเอง

พิจารณาการสร้างหน้า Admin ที่กำหนดเองเพื่อแสดงการดำเนินการที่เกี่ยวข้องที่สุดสำหรับเจ้าของสตูดิโอฟิตเนส:

- "แก้ไขตารางเรียนของคุณ"
- "อัปเดตโปรไฟล์เทรนเนอร์"
- "ดูเว็บไซต์ของคุณ"

วิธีนี้ช่วยลดช่วงการเรียนรู้ได้โดยการนำการดำเนินการที่เกี่ยวข้องกับเฉพาะทางมาไว้ด้านหน้า แทนที่จะฝังไว้ในเมนูมาตรฐานของ WordPress

## การสร้างแบรนด์ในการสื่อสารของคุณ

อีเมล ใบแจ้งหนี้ และการแจ้งเตือนทุกอย่างควรเสริมสร้างแบรนด์ของคุณ

### อีเมลระบบ (System Emails)

ไปที่ **Ultimate Multisite > Settings > Emails** และปรับแต่งอีเมลระบบทั้งหมด:

- **ชื่อผู้ส่ง (From name)**: FitSite
- **อีเมลผู้ส่ง (From email)**: hello@fitsite.com
- **เทมเพลตอีเมล (Email templates)**: ใช้สีและโลโก้แบรนด์ของคุณ
- **ภาษา (Language)**: เน้นเรื่องฟิตเนสตลอดทั้งฉบับ

อีเมลสำคัญที่ต้องปรับแต่ง:

| อีเมล | เวอร์ชันทั่วไป | เวอร์ชัน FitSite |
|-------|----------------|-----------------|
| ยินดีต้อนรับ | "เว็บไซต์ใหม่ของคุณพร้อมแล้ว" | "เว็บไซต์สตูดิโอฟิตเนสของคุณเปิดใช้งานแล้ว" |
| ใบเสร็จการชำระเงิน | "ได้รับชำระเงินแล้ว" | "ยืนยันการชำระเงินค่าบริการ FitSite" |
| ทดลองใช้ใกล้หมด | "การทดลองใช้ของคุณกำลังจะหมดลง" | "การทดลองใช้ FitSite ของคุณจะหมดใน 3 วัน — รักษาเว็บไซต์สตูดิโอของคุณให้ใช้งานได้" |

### ใบแจ้งหนี้ (Invoices)

ปรับแต่งเทมเพลตใบแจ้งหนี้ด้วย:

- โลโก้และสีแบรนด์ FitSite ของคุณ
- รายละเอียดธุรกิจของคุณ
- ชื่อผลิตภัณฑ์ที่เฉพาะเจาะจงสำหรับวงการฟิตเนส (ไม่ใช่รหัสแผนทั่วไป)

## เว็บไซต์สำหรับลูกค้า (Customer-Facing Site)

โดเมนหลักของคุณ (`fitsite.com`) จำเป็นต้องมีเว็บไซต์ทางการตลาดที่ขายแพลตฟอร์มนี้ ซึ่งแยกต่างหากจากระบบ Admin ของ Ultimate Multisite -- นี่คือหน้าตาของธุรกิจคุณต่อสาธารณะ

หน้าสำคัญ:

- **หน้าแรก (Homepage)**: ข้อเสนอคุณค่าที่ชัดเจนสำหรับธุรกิจฟิตเนส
- **คุณสมบัติ (Features)**: FitSite ทำอะไรได้บ้าง ในแง่ของฟิตเนส
- **ราคา (Pricing)**: แผนทั้งสามของคุณ พร้อมการเปรียบเทียบคุณสมบัติที่เฉพาะเจาะจงสำหรับวงการ
- **ตัวอย่าง (Examples)**: แสดงเว็บไซต์ที่สร้างบนแพลตฟอร์ม
- **สมัครใช้งาน (Sign Up)**: ลิงก์ไปยังแบบฟอร์มชำระเงินของคุณ

:::tip เว็บไซต์ทางการตลาดของคุณสามารถเป็น Network Site ได้
สร้างเว็บไซต์ทางการตลาดของคุณเป็นเว็บไซต์ภายในเครือข่ายของคุณเอง วิธีนี้ช่วยให้คุณจัดการได้จาก Dashboard เดียวกัน และแสดงให้เห็นถึงความสามารถของแพลตฟอร์มของคุณเอง
:::

## โดเมนที่กำหนดเองสำหรับลูกค้า

สำหรับลูกค้าที่ใช้แผนที่มีโดเมนที่กำหนดเอง ให้จัดทำเอกสารกระบวนการนี้ให้ชัดเจน:

1. ลูกค้าลงทะเบียนหรือโอนโดเมนของตนไปยังผู้รับจดทะเบียน
2. ลูกค้าอัปเดต DNS เพื่อชี้ไปยังแพลตฟอร์มของคุณ (ระบุ record ที่แน่นอน)
3. Ultimate Multisite จะจัดการการทำ Domain Mapping และ SSL

สร้างบทความช่วยเหลือหรือรายการความรู้ (knowledge base entry) สำหรับกระบวนการนี้โดยเฉพาะ โดยเขียนสำหรับเจ้าของสตูดิโอฟิตเนสที่ไม่ได้มีความรู้ด้านเทคนิค

## เครือข่าย FitSite จนถึงตอนนี้

```
FitSite Network
├── WordPress Multisite (subdomain mode)
├── Ultimate Multisite (configured + branded)
├── Platform Domain (fitsite.com + wildcard SSL)
├── Site Templates (Studio Essential, Gym Pro, Fitness Chain)
├── Products (Starter, Growth, Pro + Order Bumps)
├── Checkout Flow (niche-specific, tested)
├── Branding
│   ├── Custom login page
│   ├── Branded admin dashboard
│   ├── Niche-specific system emails
│   ├── Branded invoices
│   └── Marketing site on fitsite.com
└── Ready for onboarding flow (next lesson)
```

## สิ่งที่เราสร้างในบทเรียนนี้

- **Platform domain และ DNS** ถูกตั้งค่าสำหรับการสร้างแบรนด์
- **Admin ที่ทำ White-label** พร้อมการสร้างแบรนด์ FitSite ทั่วทั้งระบบ
- **การสื่อสารที่ปรับแต่งแล้ว** — อีเมล ใบแจ้งหนี้ และการแจ้งเตือนทั้งหมดเป็นแบรนด์เดียวกัน
- **เว็บไซต์ทางการตลาด** ที่ขาย FitSite ให้กับเจ้าของสตูดิโอฟิตเนส
- **เอกสารสำหรับโดเมนที่กำหนดเอง** สำหรับลูกค้าที่ต้องการโดเมนของตัวเอง
