# ภาพรวม Dashboard

Ultimate Multisite มี **Dashboard** เฉพาะอยู่ภายใต้เมนู Ultimate Multisite และยังเพิ่ม widget ไปยัง Network Dashboard ด้วย บทความนี้จะอธิบายสิ่งที่คุณจะเห็นในทั้งสองส่วน

## Network Dashboard

Ultimate Multisite เพิ่ม widget ที่มีประโยชน์ไปยัง dashboard ระดับเครือข่าย ซึ่งอยู่ที่ **My Sites → Network Admin → Dashboard**

![Network Dashboard พร้อม widget ของ Ultimate Multisite](/img/admin/network-dashboard-full.png)

แต่ละ widget มีฟังก์ชันดังนี้:

**First Steps:** จะปรากฏเมื่อติดตั้งใหม่ เพื่อเตือนให้คุณทำขั้นตอนการตั้งค่าให้เสร็จสมบูรณ์ เช่น การกำหนดวิธีการชำระเงินและการสร้างบัญชีทดสอบ

![widget First Steps](/img/admin/network-dashboard-first-steps.png)

**Summary:** รายงานจำนวนการสมัครสมาชิกและรายได้ประจำวัน ค่า MRR (รายได้ประจำเดือน) แสดงยอดรวมที่คาดการณ์จากลูกค้าที่มีสมาชิกภาพที่สร้างรายได้

![widget Summary](/img/admin/network-dashboard-summary.png)

**Activity Stream:** รายงานเหตุการณ์ต่างๆ ใน Ultimate Multisite — การสมัครสมาชิก การยกเลิก การสร้างเว็บไซต์ การลบ และกิจกรรมอื่นๆ

![widget Activity Stream](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** แสดงสรุปย่อของผู้ใช้และเว็บไซต์ในเครือข่าย พร้อมทางลัดสำหรับสร้างเว็บไซต์หรือผู้ใช้ใหม่

![widget Right Now](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** ดึงและแสดงข่าวล่าสุดของ Ultimate Multisite รวมถึงการอัปเดต บั๊กสำคัญ และการแก้ไขด้านความปลอดภัย

![widget News](/img/admin/network-dashboard-news.png)

## Dashboard ของ Ultimate Multisite

ในขณะที่ network dashboard แสดงข้อมูลภาพรวมของเครือข่ายทั้งหมด dashboard ของ Ultimate Multisite (อยู่ที่ระดับบนสุดของเมนู Ultimate Multisite) จะแสดงข้อมูลเกี่ยวกับบริการ

![Dashboard ของ Ultimate Multisite](/img/admin/um-dashboard-top.png)

**Monthly Recurring Revenue Growth:** แสดงรายละเอียดรายได้รายเดือนในช่วง 12 เดือน รายได้ใหม่จะแสดงเป็นสีน้ำเงิน ส่วนการยกเลิกจะแสดงเป็นสีแดง

![กราฟ MRR Growth](/img/admin/um-dashboard-mrr.png)

**Date-range Support:** กรองข้อมูล dashboard ตามช่วงเวลาเพื่อเน้นข้อมูลที่คุณต้องการ ช่วงวันที่จะควบคุมสิ่งที่แสดงใน widget ด้านล่าง

![ช่วงวันที่และสถิติ dashboard](/img/admin/um-dashboard-date-range.png)

widget ของ dashboard ประกอบด้วย:

- **Revenue** — รายได้รวมในช่วงเวลานั้น พร้อม MRR และรายละเอียดแยกตามผลิตภัณฑ์
- **New Memberships** — จำนวนสมาชิกภาพใหม่ที่สร้างขึ้นในช่วงเวลาที่เลือก
- **Signups by Countries** — การแยกแยะการสมัครสมาชิกตามภูมิศาสตร์
- **Signups by Forms** — จำนวนลูกค้าที่สมัครผ่านแบบฟอร์ม checkout แต่ละแบบ เทียบกับที่ผู้ดูแลระบบเพิ่มให้
- **Most Visited Sites** — เว็บไซต์ที่มีผู้เข้าชมมากที่สุดในเครือข่ายของคุณ (ต้องเปิดใช้งานการนับการเข้าชมที่ **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**)

![สถิติ Dashboard](/img/admin/um-dashboard-stats.png)
