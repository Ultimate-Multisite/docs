# การจัดการเว็บไซต์

เว็บไซต์ (หรือเว็บไซต์ย่อย) คือหัวใจหลักของธุรกิจ WaaS ของคุณ Ultimate Multisite มีเว็บไซต์ 3 ประเภท:

- **เว็บไซต์ของลูกค้า** — เว็บไซต์ที่เป็นของลูกค้าแต่ละราย
- **Site Templates** — เว็บไซต์ที่กำหนดไว้ล่วงหน้าให้ลูกค้าเลือกใช้เป็นจุดเริ่มต้น
- **เว็บไซต์หลัก** — เว็บไซต์หลักของเครือข่ายคุณ

## การดูเว็บไซต์

ไปที่ **Ultimate Multisite → Sites** เพื่อดูเว็บไซต์ย่อยทั้งหมดในเครือข่ายของคุณ แต่ละเว็บไซต์จะมีป้ายกำกับประเภท และคุณสามารถกรองดูได้ตาม All Sites, Customer-Owned, Templates หรือ Pending

![หน้ารายการเว็บไซต์](/img/admin/sites-list.png)

## การเพิ่มเว็บไซต์ใหม่

คลิกปุ่ม **Add Site** เพื่อสร้างเว็บไซต์ใหม่ คุณจะต้องระบุข้อมูลต่อไปนี้:

- **Site Title** — ชื่อของเว็บไซต์ใหม่
- **Site URL** — โดเมน/path สำหรับเว็บไซต์
- **Site Type** — ระบุว่าเป็นเว็บไซต์ของลูกค้า, template หรือเว็บไซต์ทั่วไป

ตัวเลือก **Copy Site** ให้คุณสร้างเว็บไซต์ใหม่โดยใช้ site template ที่มีอยู่เป็นต้นแบบ เมื่อเปิดใช้งานตัวเลือกนี้ คุณสามารถเลือก template ที่ต้องการใช้เป็นจุดเริ่มต้น อย่าลืมเปิดใช้งาน **Copy Media on Duplication** เพื่อรวมไฟล์สื่อด้วย

## การจัดการเว็บไซต์ที่มีอยู่

คลิก **Manage** บนเว็บไซต์ใดก็ได้เพื่อเปิดหน้า **Edit Site** ซึ่งคุณจะพบข้อมูลดังนี้:

### ข้อมูลพื้นฐาน

ชื่อเว็บไซต์ ประเภท site ID และคำอธิบาย นอกจากนี้คุณยังจะเห็นโดเมนที่เชื่อมต่อ membership ที่เกี่ยวข้อง และบัญชีลูกค้าที่เป็นเจ้าของเว็บไซต์

### ตัวเลือกเว็บไซต์

กำหนดค่าความสามารถและขีดจำกัดของเว็บไซต์:

- **ขีดจำกัดการเข้าชม** — จำนวนการเข้าชมเว็บไซต์สูงสุด
- **ขีดจำกัดบัญชีผู้ใช้** — จำกัดตามบทบาทผู้ใช้แต่ละบทบาท
- **พื้นที่ดิสก์** — โควตาพื้นที่จัดเก็บสำหรับเว็บไซต์
- **โดเมนของตัวเอง** — เปิดใช้งาน domain mapping สำหรับเว็บไซต์นี้
- **การแสดงผล plugin และ theme** — ควบคุมว่า plugin และ theme ใดจะแสดง ซ่อน หรือเปิดใช้งานไว้ล่วงหน้า

โดยค่าเริ่มต้น เว็บไซต์จะใช้ขีดจำกัดที่กำหนดไว้ในระดับ membership การตั้งค่าขีดจำกัดในระดับเว็บไซต์จะแทนที่การตั้งค่าของ membership

### ความเชื่อมโยง

ใต้ตัวเลือกเว็บไซต์ คุณจะพบข้อมูลเกี่ยวกับ:

- **โดเมนที่เชื่อมต่อ** กับเว็บไซต์
- **Membership** ที่เว็บไซต์อยู่ภายใต้
- **บัญชีลูกค้า** ที่เชื่อมโยงกับเว็บไซต์

### แถบด้านขวา

ทางด้านขวาคุณสามารถ:

- **เปิด/ปิดใช้งานเว็บไซต์** ด้วยปุ่มสลับ
- **เปลี่ยนประเภทเว็บไซต์** หรือโอนความเป็นเจ้าของ
- **ตั้งรูปภาพ/รูปย่อของเว็บไซต์** (แสดงบนหน้าเว็บสำหรับ site templates)
- **ลบเว็บไซต์** อย่างถาวร

:::warning
การลบเว็บไซต์ไม่สามารถกู้คืนได้ เว็บไซต์และเนื้อหาทั้งหมดจะถูกลบออกอย่างถาวร
:::
