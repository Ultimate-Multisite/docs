# การจัดการการชำระเงินและใบแจ้งหนี้ {#managing-payments-and-invoices}

## การตั้งค่าการชำระเงิน {#payment-settings}

ก่อนที่คุณจะเริ่มรับการชำระเงิน คุณจะต้องกำหนดค่าการตั้งค่าที่เกี่ยวข้องกับการชำระเงิน ไปที่ **Ultimate Multisite → Settings** แล้วคลิกแท็บ **Payment**

![แท็บการตั้งค่าการชำระเงิน](/img/admin/settings-payments-top.png)

นี่คือมุมมองแบบเต็มของหน้าการตั้งค่าการชำระเงิน:

![หน้าการตั้งค่าการชำระเงินแบบเต็ม](/img/admin/settings-payments-full.png)

### ตัวเลือกการชำระเงินทั่วไป {#general-payment-options}

ในการตั้งค่าทั่วไป คุณสามารถกำหนดค่าได้ดังนี้:

- **สกุลเงิน** — สกุลเงินเริ่มต้นที่ใช้สำหรับธุรกรรม
- **ตำแหน่งสกุลเงิน** — ตำแหน่งที่สัญลักษณ์สกุลเงินปรากฏ (ก่อน/หลังจำนวนเงิน)

![การตั้งค่า Payment gateway](/img/admin/settings-payments-options.png)

### Payment Gateways {#payment-gateways}

Ultimate Multisite รองรับ Payment gateway หลายรายการ คุณสามารถเปิดใช้งานและกำหนดค่าแต่ละรายการได้จากแท็บการตั้งค่า Payment

![การกำหนดค่า Payment gateway](/img/admin/settings-payments-gateways.png)

Gateway ที่มีให้ใช้งาน ได้แก่:

- **Stripe** — การชำระเงินด้วยบัตรเครดิตผ่าน Stripe
- **PayPal** — การชำระเงินผ่าน PayPal
- **Manual** — สำหรับการประมวลผลการชำระเงินแบบออฟไลน์หรือแบบกำหนดเอง

แต่ละ Gateway มีส่วนการกำหนดค่าของตัวเอง ซึ่งคุณป้อน API keys และการตั้งค่าอื่น ๆ

![การตั้งค่า Gateway เพิ่มเติม](/img/admin/settings-payments-gateways-2.png)

### โหมด Sandbox {#sandbox-mode}

คุณสามารถเปิดใช้งาน **โหมด Sandbox** เพื่อทดสอบการเชื่อมต่อการชำระเงินของคุณก่อนใช้งานจริง เมื่อโหมด Sandbox ทำงานอยู่ จะไม่มีการเรียกเก็บเงินจริง

## การดูการชำระเงิน {#viewing-payments}

ไปที่หน้า **Payments** ใต้ Ultimate Multisite เพื่อดูธุรกรรมทั้งหมดทั่วทั้งเครือข่ายของคุณ

![รายการการชำระเงิน](/img/admin/payments-list.png)

คุณสามารถกรองการชำระเงินตามสถานะ (เสร็จสมบูรณ์, รอดำเนินการ, ล้มเหลว, คืนเงินแล้ว) และค้นหาธุรกรรมที่ต้องการได้

คลิกที่การชำระเงินเพื่อดูรายละเอียดทั้งหมด รวมถึงรายการบรรทัด membership ที่เกี่ยวข้อง ข้อมูลลูกค้า และข้อมูล Payment gateway

## ใบแจ้งหนี้ {#invoices}

Ultimate Multisite สามารถสร้างใบแจ้งหนี้สำหรับการชำระเงินได้โดยอัตโนมัติ คุณสามารถปรับแต่งเทมเพลตใบแจ้งหนี้และรูปแบบการกำหนดหมายเลขได้จากการตั้งค่า Payment

ตัวเลือกการปรับแต่งใบแจ้งหนี้ ได้แก่:

- **ชื่อและที่อยู่บริษัท** ที่แสดงบนใบแจ้งหนี้
- **การกำหนดหมายเลขใบแจ้งหนี้** รูปแบบและลำดับ
- **โลโก้** ที่แสดงบนส่วนหัวของใบแจ้งหนี้
- **ข้อความส่วนท้ายแบบกำหนดเอง** สำหรับข้อกำหนด หมายเหตุ หรือข้อมูลทางกฎหมาย

หากต้องการปรับแต่งเทมเพลตใบแจ้งหนี้ ให้ไปที่ **Ultimate Multisite → Settings → Payment** แล้วมองหาการตั้งค่าที่เกี่ยวข้องกับใบแจ้งหนี้
