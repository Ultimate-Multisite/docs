# การจัดการการชำระเงินและใบแจ้งหนี้

## ตั้งค่าการชำระเงิน

ก่อนที่คุณจะเริ่มรับการชำระเงิน คุณจะต้องตั้งค่าที่เกี่ยวข้องกับการชำระเงินก่อน ไปที่ **Ultimate Multisite → Settings** แล้วคลิกที่แท็บ **Payment**

![แท็บตั้งค่าการชำระเงิน](/img/admin/settings-payments-top.png)

### ตัวเลือกการชำระเงินทั่วไป

ในการตั้งค่าทั่วไป คุณสามารถกำหนดค่าต่อไปนี้:

- **Currency** — สกุลเงินเริ่มต้นที่ใช้ในการทำธุรกรรม
- **Currency Position** — ตำแหน่งของสัญลักษณ์สกุลเงิน (ก่อน/หลังจำนวนเงิน)

![การตั้งค่า payment gateway](/img/admin/settings-payments-options.png)

### Payment Gateways

Ultimate Multisite รองรับ payment gateway หลายตัว คุณสามารถเปิดใช้งานและตั้งค่าแต่ละตัวได้จากแท็บ Payment settings

![การตั้งค่า payment gateway](/img/admin/settings-payments-gateways.png)

Gateway ที่พร้อมใช้งานได้แก่:

- **Stripe** — การชำระเงินผ่านบัตรเครดิตด้วย Stripe
- **PayPal** — การชำระเงินผ่าน PayPal
- **Manual** — สำหรับการชำระเงินแบบออฟไลน์หรือแบบกำหนดเอง

แต่ละ gateway จะมีส่วนการตั้งค่าของตัวเอง ซึ่งคุณจะต้องกรอก API keys และการตั้งค่าอื่นๆ

![การตั้งค่า gateway เพิ่มเติม](/img/admin/settings-payments-gateways-2.png)

### โหมด Sandbox

คุณสามารถเปิดใช้ **Sandbox Mode** เพื่อทดสอบระบบการชำระเงินก่อนใช้งานจริง เมื่อเปิดโหมด sandbox จะไม่มีการเรียกเก็บเงินจริง

## ดูรายการการชำระเงิน

ไปที่หน้า **Payments** ภายใต้ Ultimate Multisite เพื่อดูธุรกรรมทั้งหมดในเครือข่ายของคุณ

![รายการการชำระเงิน](/img/admin/payments-list.png)

คุณสามารถกรองการชำระเงินตามสถานะ (completed, pending, failed, refunded) และค้นหาธุรกรรมที่ต้องการได้

คลิกที่การชำระเงินเพื่อดูรายละเอียดทั้งหมด รวมถึงรายการสินค้า, สมาชิกภาพที่เกี่ยวข้อง, ข้อมูลลูกค้า และข้อมูล payment gateway

## ใบแจ้งหนี้

Ultimate Multisite สามารถสร้างใบแจ้งหนี้โดยอัตโนมัติสำหรับการชำระเงิน คุณสามารถปรับแต่งเทมเพลตใบแจ้งหนี้และรูปแบบหมายเลขได้จาก Payment settings

ตัวเลือกการปรับแต่งใบแจ้งหนี้ได้แก่:

- **ชื่อบริษัทและที่อยู่** ที่แสดงบนใบแจ้งหนี้
- **รูปแบบหมายเลขใบแจ้งหนี้** และลำดับ
- **โลโก้** ที่แสดงในส่วนหัวของใบแจ้งหนี้
- **ข้อความส่วนท้ายที่กำหนดเอง** สำหรับเงื่อนไข, หมายเหตุ หรือข้อมูลทางกฎหมาย

หากต้องการปรับแต่งเทมเพลตใบแจ้งหนี้ ให้ไปที่ **Ultimate Multisite → Settings → Payment** แล้วมองหาการตั้งค่าที่เกี่ยวกับใบแจ้งหนี้
