# แบบฟอร์มติดต่อด้วย Site Builder {#site-builder-contact-forms}

Superdav AI Agent v1.10.0 เพิ่มความสามารถในการสร้างแบบฟอร์มติดต่อได้โดยตรงจากหน้าต่างแชท โดยใช้ Site Builder agent ทำให้คุณสามารถเพิ่มแบบฟอร์มติดต่อที่ใช้งานได้จริงลงในหน้าใดก็ได้โดยไม่ต้องออกจากหน้าแชท

## ภาพรวม {#overview}

ความสามารถในการสร้างแบบฟอร์มติดต่อของ Site Builder (`create_contact_form`) จะตรวจจับ plugin สำหรับสร้างฟอร์มที่คุณติดตั้งไว้โดยอัตโนมัติ และสร้างแบบฟอร์มติดต่อโดยใช้ความสามารถดั้งเดิมของ plugin นั้นๆ โดยรองรับ plugin สำหรับสร้างฟอร์มดังนี้:

- Contact Form 7
- WPForms
- Fluent Forms
- Gravity Forms

ผู้ช่วยจะเลือก plugin สำหรับสร้างฟอร์มที่ดีที่สุดที่มีอยู่ในเว็บไซต์ของคุณโดยอัตโนมัติ และสร้างแบบฟอร์มติดต่อที่ปรับให้เข้ากับ plugin นั้นๆ

## การสร้างแบบฟอร์มติดต่อ {#creating-a-contact-form}

ในการสร้างแบบฟอร์มติดต่อโดยใช้ Site Builder:

1. เปิดแผงแชท **Gratis AI Agent** ในหน้าผู้ดูแลระบบ WordPress
2. สลับไปที่ agent **Site Builder** โดยคลิกที่ไอคอน agent ที่ส่วนหัวของแชท
3. อธิบายแบบฟอร์มติดต่อที่คุณต้องการสร้าง ตัวอย่างเช่น:

   > "เพิ่มแบบฟอร์มติดต่อในหน้า Contact โดยมีช่องสำหรับชื่อ อีเมล ข้อความ และเบอร์โทรศัพท์"

   หรือเพียงแค่:

   > "สร้างแบบฟอร์มติดต่อสำหรับหน้า Contact"

4. Site Builder จะสร้างแบบฟอร์มติดต่อและส่ง shortcode ที่พร้อมนำไปใช้งานให้คุณ

## การใช้ Shortcode ที่สร้างขึ้น {#using-the-generated-shortcode}

หลังจากที่ Site Builder สร้างแบบฟอร์มติดต่อแล้ว มันจะส่ง shortcode กลับมาให้คุณ (ตัวอย่างเช่น, `[contact-form-7 id="123"]`) คุณสามารถ:

1. **ฝังในหน้าหรือโพสต์** — คัดลอก shortcode แล้ววางลงในหน้าหรือโพสต์ใดก็ได้โดยใช้ block editor หรือ classic editor
2. **เพิ่มผ่าน Site Builder** — ขอให้ Site Builder เพิ่มแบบฟอร์มไปยังหน้าเฉพาะโดยอัตโนมัติ:

   > "เพิ่มแบบฟอร์มติดต่อลงในหน้า Contact"

3. **ใช้ใน template** — หากคุณคุ้นเคยกับ PHP คุณสามารถเพิ่ม shortcode นี้ลงในไฟล์ template ของ theme ได้

## การปรับแต่งแบบฟอร์มติดต่อ {#customizing-the-contact-form}

หลังจากที่ Site Builder สร้างแบบฟอร์มติดต่อแล้ว คุณสามารถปรับแต่งเพิ่มเติมได้:

### ผ่านหน้าต่างแชท {#via-the-chat-interface}

ขอให้ Site Builder แก้ไขแบบฟอร์ม:

> "อัปเดตแบบฟอร์มติดต่อเพื่อเพิ่มช่องหัวข้อ (subject) และกำหนดให้ช่องข้อความ (message) เป็นช่องที่ต้องกรอก"

Site Builder จะอัปเดตแบบฟอร์มและส่ง shortcode ที่อัปเดตแล้วกลับมาให้คุณ

### ผ่านหน้าผู้ดูแลระบบของ Form Plugin {#via-the-form-plugins-admin-interface}

คุณยังสามารถปรับแต่งแบบฟอร์มได้โดยตรงในการตั้งค่าของ form plugin ของคุณ:

1. ไปที่ **Contact Form 7** (หรือ form plugin ที่คุณติดตั้ง) ในหน้าผู้ดูแลระบบ WordPress
2. ค้นหาแบบฟอร์มที่สร้างโดย Site Builder
3. แก้ไขช่องฟอร์ม กฎการตรวจสอบ และการแจ้งเตือนทางอีเมลตามที่จำเป็น

## Form Plugins และความเข้ากันได้ {#form-plugins-and-compatibility}

Site Builder จะตรวจจับโดยอัตโนมัติว่ามีการติดตั้ง form plugin ใดบนเว็บไซต์ของคุณ และใช้ plugin นั้นในการสร้างแบบฟอร์มติดต่อ หากมีการติดตั้ง form plugin หลายตัว Site Builder จะให้ความสำคัญตามลำดับนี้:

1. Contact Form 7
2. WPForms
3. Fluent Forms
4. Gravity Forms

หากไม่มีการติดตั้ง plugin เหล่านี้เลย Site Builder จะแนะนำให้คุณติดตั้งก่อนจึงจะสามารถสร้างแบบฟอร์มติดต่อได้

## การแจ้งเตือนทางอีเมล {#email-notifications}

แบบฟอร์มติดต่อที่สร้างโดย Site Builder จะถูกตั้งค่าให้ส่งการแจ้งเตือนทางอีเมลไปยังผู้ดูแลระบบเว็บไซต์โดยค่าเริ่มต้น คุณสามารถปรับแต่งที่อยู่อีเมลผู้รับและการแจ้งเตือนได้:

1. ไปที่หน้าผู้ดูแลระบบของ form plugin ของคุณ
2. ค้นหาแบบฟอร์มที่สร้างโดย Site Builder
3. แก้ไขการตั้งค่าการแจ้งเตือนทางอีเมล

สำหรับคำแนะนำโดยละเอียด โปรดดูเอกสารของ form plugin ของคุณ:

- [Contact Form 7 Email Notifications](https://contactform7.com/docs/)
- [WPForms Email Notifications](https://wpforms.com/docs/)
- [Fluent Forms Email Notifications](https://fluentforms.com/docs/)
- [Gravity Forms Email Notifications](https://docs.gravityforms.com/)

## ตัวอย่างการใช้งาน {#example-use-cases}

### แบบฟอร์มติดต่ออย่างง่าย {#simple-contact-form}

> "สร้างแบบฟอร์มติดต่ออย่างง่ายที่มีช่องสำหรับชื่อ อีเมล และข้อความ"

### แบบฟอร์มหลายขั้นตอน {#multi-step-form}

> "สร้างแบบฟอร์มติดต่อที่มีขั้นตอนแรกสำหรับข้อมูลติดต่อ และขั้นตอนที่สองสำหรับข้อความและวิธีการติดต่อที่ต้องการ"

### แบบฟอร์มพร้อมเงื่อนไข (Conditional Logic) {#form-with-conditional-logic}

> "สร้างแบบฟอร์มติดต่อที่แสดงช่องต่างๆ ที่แตกต่างกันตามการเลือกของผู้ใช้ในเมนูแบบเลื่อนลง"

### แบบฟอร์มพร้อมการอัปโหลดไฟล์ {#form-with-file-upload}

> "สร้างแบบฟอร์มติดต่อที่อนุญาตให้ผู้ใช้อัปโหลดไฟล์หรือเอกสารแนบ"

:::note
การสร้างแบบฟอร์มติดต่อมีให้ใช้ใน Superdav AI Agent v1.10.0 และเวอร์ชันที่ใหม่กว่า Site Builder agent ต้องเปิดใช้งานจึงจะใช้ความสามารถนี้ได้
:::
