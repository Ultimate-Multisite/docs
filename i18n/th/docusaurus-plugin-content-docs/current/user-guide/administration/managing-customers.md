# การจัดการลูกค้า

หน้า **Customers** แสดงรายชื่อสมาชิกที่ลงทะเบียนในเครือข่ายของคุณ หน้านี้แสดงข้อมูลพื้นฐานของผู้ใช้ ได้แก่ ชื่อ อีเมล การเข้าสู่ระบบครั้งล่าสุด และลิงก์ **switch to** ที่ให้คุณเข้าสู่ระบบในไซต์ย่อยของผู้ใช้ในฐานะผู้ดูแลระบบไซต์

![Customers list page](/img/admin/customers-list.png)

คลิก **Manage** ด้านล่างการ์ดของลูกค้าเพื่อดูข้อมูลรายละเอียดของลูกค้ารายนั้น

ในหน้ารายละเอียดลูกค้า บล็อก **At a Glance** จะแสดงรหัสลูกค้า เวลาเข้าสู่ระบบครั้งล่าสุด และยอดเงินรวมที่ลูกค้ารายนี้ใช้จ่าย

บล็อก **Customer Options** ให้คุณตั้งค่าลูกค้ารายนี้เป็น **VIP** (แท็กที่ช่วยให้ค้นหาได้ง่ายขึ้น) แท็บ **Billing Info** แสดงข้อมูลการเรียกเก็บเงิน เช่น อีเมล ที่อยู่ และประเทศ

บล็อก **Fields** แสดงชื่อผู้ใช้ อีเมลของลูกค้า และให้คุณเปลี่ยนสถานะ **Email Verification** ได้

ด้านล่างบล็อก Customer Options คุณจะพบบล็อกเพิ่มเติมอีกสี่บล็อก:

- **Memberships** — การสมัครสมาชิกปัจจุบันที่ลูกค้าสมัครไว้
- **Payments** — ประวัติการชำระเงินของลูกค้า
- **Sites** — ไซต์ที่ลูกค้ารายนี้เป็นเจ้าของ
- **Events** — เหตุการณ์ที่เกี่ยวข้องกับลูกค้ารายนี้ (การยืนยันอีเมล การเปลี่ยนแปลงข้อมูลการเรียกเก็บเงิน ฯลฯ)

คลิกไอคอนลูกศร (→) ข้างรายการใดก็ได้เพื่อไปยังหน้ารายละเอียดของรายการนั้น ตัวอย่างเช่น การคลิกลูกศรข้างไซต์จะพาคุณไปยังหน้าแก้ไขไซต์ ซึ่งคุณสามารถปรับเปลี่ยนการเชื่อมโยงสมาชิก ประเภทไซต์ และสถานะการใช้งานได้

เช่นเดียวกันกับ memberships และ payments — แต่ละรายการจะลิงก์ไปยังหน้ารายละเอียดที่คุณสามารถแก้ไขสถานะสมาชิก ลูกค้าที่เชื่อมโยง ผลิตภัณฑ์ การตั้งค่า payment gateway และแม้แต่ดำเนินการคืนเงินได้

ทางด้านขวาของหน้าลูกค้า คุณจะพบ:

- **Last Login & IPs** — เวลาเข้าสู่ระบบครั้งล่าสุด IP address และประเทศ
- **Timestamps** — วันที่สร้างบัญชีลูกค้าและวันที่แก้ไขล่าสุด
- **Delete** — ตัวเลือกสำหรับลบลูกค้าอย่างถาวร

:::warning
การลบลูกค้าเป็นการดำเนินการถาวรและไม่สามารถยกเลิกได้
:::
