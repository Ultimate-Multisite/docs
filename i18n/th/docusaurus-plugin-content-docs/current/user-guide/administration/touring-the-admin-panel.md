# ทัวร์ชมแผงควบคุมแอดมิน

Ultimate Multisite ออกแบบมาให้ใช้งานง่ายและเข้าใจได้ด้วยตัวเอง แต่ด้วยความสามารถที่หลากหลายของเครื่องมือนี้ การมีคู่มือนำทางก็ช่วยได้มาก มาดูหน้าแอดมินแต่ละส่วนกันเพื่อให้คุณคุ้นเคยกับระบบ

## Dashboard

**Dashboard** ของ Ultimate Multisite แสดงรายงานและข้อมูลวิเคราะห์เบื้องต้น รวมถึงรายได้ กิจกรรมของเว็บไซต์ การเติบโตของสมาชิก จำนวนผู้เข้าชม และข้อมูลทางภูมิศาสตร์

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

## Checkout Forms

**Checkout Forms** ให้ความยืดหยุ่นในการสร้างหน้าลงทะเบียนแบบกำหนดเอง คุณสามารถปรับแต่งฟิลด์ต่างๆ เพิ่ม class และ script ที่ต้องการ รวมถึงจำกัดการเข้าถึงตามประเทศหรือตำแหน่งที่ตั้งของผู้เยี่ยมชม

![Checkout Forms list](/img/admin/checkout-forms-list.png)

## Products

ส่วน **Products** คือที่สำหรับสร้างแผน แพ็กเกจ และบริการต่างๆ สำหรับเครือข่ายของคุณ กำหนดราคา รอบการเรียกเก็บเงิน และข้อจำกัด & โควตาในระดับสินค้า

![Products list](/img/admin/products-list.png)

## Memberships

หน้า **Memberships** แสดงการสมัครสมาชิกทั้งหมดในเครือข่ายของคุณ ดูและแก้ไขรายละเอียดการสมัครสมาชิก รวมถึงแผน สินค้า จำนวนเงินและรอบการเรียกเก็บเงิน ประวัติการชำระเงิน และข้อมูลวันเวลา

![Memberships list](/img/admin/memberships-list.png)

## Payments

หน้า **Payments** แสดงภาพรวมประวัติการชำระเงินทั่วทั้งเครือข่ายอย่างรวดเร็ว พร้อมข้อมูลธุรกรรมโดยละเอียด รวมถึงสินค้าและจำนวนเงินที่เฉพาะเจาะจง

![Payments list](/img/admin/payments-list.png)

## Customers

หน้า **Customers** แสดงสมาชิกทั้งหมดที่ลงทะเบียนในเครือข่ายของคุณ พร้อมข้อมูลผู้ใช้เบื้องต้น ได้แก่ ชื่อ อีเมล การเข้าสู่ระบบครั้งล่าสุด และลิงก์ "switch to" สำหรับเข้าสู่ระบบ subsite ของพวกเขา

![Customers list](/img/admin/customers-list.png)

## Sites

หน้า **Sites** แสดงรายการ subsite ทั้งหมดในเครือข่ายของคุณ รวมถึง site template และเว็บไซต์ที่ลูกค้าเป็นเจ้าของ จัดการโดเมนที่แมป ข้อจำกัด & โควตา plugin และ theme รวมถึงภาพขนาดย่อของเว็บไซต์ได้อย่างง่ายดาย

![Sites list](/img/admin/sites-list.png)

## Domains

หน้า **Domains** มีไว้สำหรับจัดการโดเมนที่กำหนดเองซึ่งแมปกับ subsite โดยเฉพาะ ในฐานะ super admin คุณสามารถเพิ่มหรือแมปชื่อโดเมนที่กำหนดเองด้วยตนเอง และดูระเบียน DNS และ log ต่างๆ

![Domains list](/img/admin/domains-list.png)

## Discount Codes

**Discount Codes** ช่วยให้คุณสร้างรหัสคูปองเพื่อเสนอส่วนลดให้กับผู้ใช้ปลายทาง ระบุมูลค่าส่วนลดและกำหนดเป้าหมายไปยังผู้ใช้หรือกลุ่มที่เฉพาะเจาะจง

![Discount Codes list](/img/admin/discount-codes-list.png)

## Broadcasts

**Broadcasts** เป็นเครื่องมือสำหรับส่งประกาศหรือข้อความส่วนตัวไปยังผู้ใช้ subsite ของคุณ กำหนดเป้าหมายกลุ่มผู้ใช้เฉพาะตามแผนหรือสินค้าที่พวกเขาสมัครใช้งาน

![Broadcasts list](/img/admin/broadcasts-list.png)

## Settings

หน้า **Settings** คือที่สำหรับกำหนดค่า Ultimate Multisite — การตั้งค่าการลงทะเบียน การชำระเงิน API และ webhooks การแมปโดเมน และการเชื่อมต่ออื่นๆ

![Settings page](/img/admin/settings-general.png)

## Events

หน้า **Events** เก็บบันทึกเหตุการณ์และ log ทั้งหมดในเครือข่ายของคุณ ติดตามกิจกรรมต่างๆ เช่น การเปลี่ยนแผน การสมัคร และความเคลื่อนไหวอื่นๆ — มีประโยชน์สำหรับการตรวจสอบ multisite ของคุณ

![Events list](/img/admin/events-list.png)

## Webhooks

**Webhooks** ช่วยให้คุณส่งข้อมูลไปยังแอปพลิเคชันอื่นได้ มีประโยชน์สำหรับการส่งข้อมูลจาก Ultimate Multisite ไปยังแพลตฟอร์มอย่าง Zapier

![Webhooks list](/img/admin/webhooks-list.png)
