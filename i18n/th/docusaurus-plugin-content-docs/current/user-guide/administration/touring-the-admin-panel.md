# สำรวจ Admin Panel

Ultimate Multisite มีเป้าหมายให้ใช้งานง่ายและเข้าใจได้ด้วยตัวเองมากที่สุด แต่ด้วยเครื่องมือที่ทรงพลังเช่นนี้ การมีคำแนะนำพาชมย่อมช่วยได้ มาดูหน้าผู้ดูแลระบบต่าง ๆ เพื่อช่วยให้คุณเริ่มต้นได้อย่างมั่นใจ

## Dashboard {#dashboard}

**Dashboard** ของ Ultimate Multisite แสดงรายงานและการวิเคราะห์พื้นฐาน รวมถึงรายได้ กิจกรรมของเว็บไซต์ การเติบโตของสมาชิก จำนวนผู้เข้าชม และข้อมูลทางภูมิศาสตร์

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

นี่คือมุมมองแบบเต็มของ Dashboard:

![หน้าเต็มของ Ultimate Multisite Dashboard](/img/admin/um-dashboard-full.png)

และส่วนล่างของ Dashboard:

![ส่วนล่างของ Ultimate Multisite Dashboard](/img/admin/um-dashboard-bottom.png)

นี่คือมุมมองแบบเต็มของ Dashboard เครือข่ายด้วย:

![หน้าเต็มของ Network Dashboard](/img/admin/network-dashboard-full-page.png)

## ฟอร์มชำระเงิน {#checkout-forms}

**ฟอร์มชำระเงิน** ช่วยให้คุณมีความยืดหยุ่นในการสร้างหน้าลงทะเบียนแบบกำหนดเอง คุณสามารถปรับแต่งช่องข้อมูล ใส่คลาสและสคริปต์แบบกำหนดเอง และจำกัดการเข้าถึงตามประเทศหรือตำแหน่งที่ตั้งของผู้เข้าชมได้

![รายการฟอร์มชำระเงิน](/img/admin/checkout-forms-list.png)

## ผลิตภัณฑ์ {#products}

ส่วน **ผลิตภัณฑ์** คือที่ที่คุณสร้างแผน แพ็กเกจ และบริการต่าง ๆ สำหรับเครือข่ายของคุณ กำหนดราคา ความถี่ในการเรียกเก็บเงิน และขีดจำกัดและโควตาที่ระดับผลิตภัณฑ์

![รายการผลิตภัณฑ์](/img/admin/products-list.png)

## สมาชิกภาพ {#memberships}

หน้า **สมาชิกภาพ** แสดงการสมัครสมาชิกทั้งหมดในเครือข่ายของคุณ ตรวจสอบและแก้ไขรายละเอียดการสมัครสมาชิก รวมถึงแผน ผลิตภัณฑ์ ยอดเงินและความถี่ในการเรียกเก็บเงิน ประวัติการชำระเงิน และเวลาในการบันทึก

![รายการสมาชิกภาพ](/img/admin/memberships-list.png)

## การชำระเงิน {#payments}

หน้า **การชำระเงิน** ให้มุมมองอย่างรวดเร็วของประวัติการชำระเงินทั่วทั้งเครือข่ายของคุณ พร้อมข้อมูลธุรกรรมโดยละเอียด รวมถึงผลิตภัณฑ์และจำนวนเงินเฉพาะ

![รายการการชำระเงิน](/img/admin/payments-list.png)

## ลูกค้า {#customers}

หน้า **ลูกค้า** แสดงสมาชิกทั้งหมดที่ลงทะเบียนในเครือข่ายของคุณ พร้อมข้อมูลผู้ใช้พื้นฐาน รวมถึงชื่อ ที่อยู่อีเมล การเข้าสู่ระบบล่าสุด และลิงก์ "สลับไปยัง" สำหรับเข้าสู่ระบบเว็บไซต์ย่อยของพวกเขา

![รายการลูกค้า](/img/admin/customers-list.png)

## เว็บไซต์ {#sites}

หน้า **เว็บไซต์** แสดงรายการเว็บไซต์ย่อยทั้งหมดในเครือข่ายของคุณ รวมถึงเทมเพลตเว็บไซต์และเว็บไซต์ที่ลูกค้าเป็นเจ้าของ จัดการโดเมนที่แมปไว้ ขีดจำกัดและโควตา ปลั๊กอินและธีม รวมถึงภาพตัวอย่างเว็บไซต์ได้อย่างง่ายดาย

![รายการเว็บไซต์](/img/admin/sites-list.png)

## โดเมน {#domains}

หน้า **โดเมน** มีไว้สำหรับโดเมนแบบกำหนดเองที่แมปกับเว็บไซต์ย่อย ในฐานะผู้ดูแลระบบระดับสูง คุณสามารถเพิ่มหรือแมปชื่อโดเมนแบบกำหนดเองด้วยตนเอง และดูระเบียน DNS และบันทึกได้

![รายการโดเมน](/img/admin/domains-list.png)

## โค้ดส่วนลด {#discount-codes}

**โค้ดส่วนลด** ช่วยให้คุณสร้างโค้ดคูปองเพื่อมอบส่วนลดให้ผู้ใช้ปลายทางของคุณ ระบุมูลค่าส่วนลดและกำหนดเป้าหมายผู้ใช้หรือกลุ่มเฉพาะได้

![รายการโค้ดส่วนลด](/img/admin/discount-codes-list.png)

## การประกาศ {#broadcasts}

**การประกาศ** เป็นเครื่องมือสำหรับเผยแพร่ประกาศหรือส่งข้อความส่วนตัวถึงผู้ใช้เว็บไซต์ย่อยของคุณ กำหนดเป้าหมายกลุ่มผู้ใช้เฉพาะตามแผนหรือผลิตภัณฑ์ที่พวกเขาสมัครใช้งานอยู่

![รายการการประกาศ](/img/admin/broadcasts-list.png)

## การตั้งค่า {#settings}

หน้า **การตั้งค่า** คือที่ที่คุณกำหนดค่า Ultimate Multisite — การตั้งค่าการลงทะเบียน การชำระเงิน API และ webhooks การแมปโดเมน และการเชื่อมต่ออื่น ๆ

การตั้งค่าตัวเชื่อมต่อ AI แสดงรายการพูลผู้ให้บริการ OAuth ที่รองรับในปัจจุบัน ได้แก่ Anthropic Max, OpenAI ChatGPT/Codex และ Google AI Pro การ์ดผู้ให้บริการแต่ละรายการช่วยให้ผู้ดูแลระบบระดับสูงเชื่อมต่อบัญชี รีเฟรชบัญชีที่บันทึกไว้ ลบบัญชีตามอีเมล และใช้ทางเลือก OAuth แบบแมนนวลเมื่อสภาพแวดล้อมแบบ sandbox บล็อกการเปลี่ยนเส้นทางของเบราว์เซอร์ ตัวเลือกการตั้งค่า Cursor Pro ถูกนำออกจาก Admin Panel แล้ว

บัญชี ChatGPT/Codex รองรับการใช้เครื่องมือที่ขับเคลื่อนโดยตัวเชื่อมต่อเมื่อการดำเนินการอนุญาตให้ใช้เครื่องมือ ดังนั้นเวิร์กโฟลว์ของผู้ดูแลระบบที่พึ่งพาการดำเนินการที่ตัวเชื่อมต่อรองรับจึงสามารถใช้พฤติกรรมเครื่องมือของ Codex ได้หลังจากเชื่อมต่อบัญชี OpenAI แล้ว

## เหตุการณ์ {#events}

หน้า **เหตุการณ์** เก็บบันทึกเหตุการณ์และล็อกทั้งหมดในเครือข่ายของคุณ โดยติดตามกิจกรรมต่าง ๆ เช่น การเปลี่ยนแผน การสมัครใช้งาน และความเคลื่อนไหวอื่น ๆ — มีประโยชน์สำหรับการติดตาม multisite ของคุณ

![รายการเหตุการณ์](/img/admin/events-list.png)

## Webhooks {#webhooks}

**Webhooks** ช่วยให้คุณส่งข้อมูลไปยังแอปพลิเคชันอื่นได้ มีประโยชน์สำหรับการส่งข้อมูลจาก Ultimate Multisite ไปยังแพลตฟอร์มอย่าง Zapier

![รายการ Webhooks](/img/admin/webhooks-list.png)
