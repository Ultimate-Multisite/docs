# การจัดการ Memberships {#managing-memberships}

ใน Ultimate Multisite คุณสามารถจัดการ memberships และเข้าถึง subscriptions ทั้งหมดบนเครือข่ายของคุณได้ ตรวจสอบและแก้ไขรายละเอียด subscription เช่น plan และผลิตภัณฑ์ จำนวนไซต์ ยอดเรียกเก็บและความถี่ ประวัติการชำระเงิน และประทับเวลา

หากต้องการเข้าถึงหน้านี้ ให้ไปที่แท็บ **Memberships** ในเมนู Ultimate Multisite

![หน้ารายการ Memberships](/img/admin/memberships-list.png)

บนหน้า Memberships คุณสามารถดูรายการลูกค้าของคุณและผลิตภัณฑ์ที่เชื่อมโยงกับ Account ของพวกเขาได้ คุณสามารถกรอง memberships ตามสถานะ: ใช้งานอยู่, รอดำเนินการ, พักไว้, หมดอายุ หรือยกเลิกแล้ว

memberships ฟรีจะถือเป็น memberships แบบ **ตลอดชีพ** จะไม่มีวันหมดอายุและจะไม่หมดอายุโดยอัตโนมัติ หาก plan ฟรีควรสิ้นสุดหลังช่วงทดลองใช้หรือช่วงเวลาที่กำหนด ให้ใช้ผลิตภัณฑ์แบบชำระเงินพร้อมช่วงทดลองใช้ ยกเลิก membership ด้วยตนเอง หรือย้ายลูกค้าไปยัง plan อื่นเมื่อสิทธิ์การเข้าถึงของพวกเขาควรเปลี่ยนแปลง

คลิกที่ membership เพื่อเข้าถึงหน้าที่คุณสามารถแก้ไขรายละเอียด membership ได้ — ข้อมูลการเรียกเก็บเงิน ขีดจำกัดไซต์หรือการเข้าชม และอื่นๆ

![อินเทอร์เฟซแก้ไข Membership](/img/admin/membership-edit.png)

นี่คือมุมมองเต็มของหน้าแก้ไข membership:

![อินเทอร์เฟซแก้ไข Membership แบบเต็ม](/img/admin/membership-edit-full.png)

คุณยังสามารถเพิ่ม memberships ใหม่ที่เชื่อมโยงกับผู้ใช้ที่ลงทะเบียนไว้ก่อนหน้านี้บนเครือข่ายของคุณได้ คลิก **Add Membership** และกำหนดค่ารายละเอียด

![โมดัลเพิ่ม Membership](/img/admin/membership-add-modal.png)

จากนั้นคลิก **Create Membership** membership ใหม่จะปรากฏในรายการ memberships.
