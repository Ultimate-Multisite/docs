# การจัดการสมาชิก

ใน Ultimate Multisite คุณสามารถจัดการสมาชิกและเข้าถึงการสมัครสมาชิกทั้งหมดบนเครือข่ายของคุณได้ ตรวจสอบและแก้ไขรายละเอียดการสมัครสมาชิก เช่น แผนและผลิตภัณฑ์ จำนวนเว็บไซต์ ยอดเรียกเก็บเงินและความถี่ในการชำระ ประวัติการชำระเงิน และการประทับเวลา

หากต้องการเข้าถึงหน้านี้ ให้ไปที่แท็บ **Memberships** ในเมนู Ultimate Multisite

![หน้ารายการสมาชิก](/img/admin/memberships-list.png)

ในหน้า Memberships คุณจะเห็นรายชื่อลูกค้าและผลิตภัณฑ์ที่เชื่อมโยงกับบัญชีของพวกเขา คุณสามารถกรองสมาชิกตามสถานะได้ ไม่ว่าจะเป็น active, pending, on hold, expired หรือ cancelled

คลิกที่สมาชิกเพื่อเข้าสู่หน้าที่คุณสามารถแก้ไขรายละเอียดสมาชิกได้ เช่น ข้อมูลการเรียกเก็บเงิน ขีดจำกัดเว็บไซต์หรือการเข้าชม และอื่นๆ

คุณยังสามารถเพิ่มสมาชิกใหม่ที่เชื่อมโยงกับผู้ใช้ที่ลงทะเบียนไว้ก่อนหน้านี้บนเครือข่ายของคุณได้ คลิก **Add Membership** แล้วกำหนดค่ารายละเอียด

![หน้าต่างเพิ่มสมาชิก](/img/admin/membership-add-modal.png)

จากนั้นคลิก **Create Membership** สมาชิกใหม่จะปรากฏในรายการสมาชิก
