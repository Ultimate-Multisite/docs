# ทำความเข้าใจแดชบอร์ด Ultimate Multisite

ก่อนที่เราจะลงลึกถึงตัวเลือกการกำหนดค่าขั้นสูงและการใช้งาน Ultimate Multisite มาดูแดชบอร์ดกันก่อน

เช่นเดียวกับ plugin WordPress อื่นๆ Ultimate Multisite จะสร้างเมนูในระดับเครือข่ายชื่อ **Ultimate Multisite** พร้อมกับเมนูย่อยหลายรายการอยู่ข้างใต้

## วิดเจ็ตแดชบอร์ดระดับเครือข่าย

Ultimate Multisite เพิ่มวิดเจ็ตที่มีประโยชน์หลายตัวเข้าไปในแดชบอร์ดระดับเครือข่าย คุณสามารถเข้าถึงแดชบอร์ดนี้ได้โดยไปที่ **My Sites → Network Admin → Dashboard** วิดเจ็ตเหล่านี้ออกแบบมาเพื่อให้ผู้ดูแลระบบเครือข่ายเข้าถึงข้อมูลและการดำเนินการที่ใช้บ่อยได้อย่างสะดวก

![แดชบอร์ดเครือข่ายพร้อมวิดเจ็ต Ultimate Multisite](/img/admin/network-dashboard-full.png)

### ขั้นตอนแรก

![วิดเจ็ตขั้นตอนแรก](/img/admin/network-dashboard-first-steps.png)

วิดเจ็ตนี้จะปรากฏเมื่อติดตั้ง Ultimate Multisite ใหม่ เพื่อเตือนผู้ดูแลระบบเครือข่ายให้ทำงานเพิ่มเติม เช่น ตั้งค่าวิธีการชำระเงินและสร้างบัญชีทดสอบ

### สรุปภาพรวม

![วิดเจ็ตสรุปภาพรวม](/img/admin/network-dashboard-summary.png)

วิดเจ็ตสรุปภาพรวมแสดงจำนวนการสมัครและรายได้ของวันนั้นๆ MRR (รายได้ประจำรายเดือน) แสดงยอดรวมที่คาดการณ์จากลูกค้าที่มีสมาชิกภาพที่สร้างรายได้

### สตรีมกิจกรรม

![วิดเจ็ตสตรีมกิจกรรม](/img/admin/network-dashboard-activity-stream.png)

วิดเจ็ตนี้รายงานเหตุการณ์ต่างๆ ทั่วทั้ง Ultimate Multisite เหตุการณ์เหล่านี้รวมถึงการสมัคร การยกเลิก การสร้างไซต์ การลบ และกิจกรรมอื่นๆ

สามารถดูรายละเอียดเพิ่มเติมได้ในส่วน Events ของเอกสารประกอบ

### ตอนนี้

![วิดเจ็ตตอนนี้](/img/admin/network-dashboard-right-now.png)

วิดเจ็ตอรรถประโยชน์นี้แสดงสรุปย่อของผู้ใช้และไซต์ในเครือข่าย ทางลัดด้านบนช่วยให้สร้างไซต์หรือผู้ใช้ใหม่ได้ด้วยคลิกเดียว

### ข่าวสารและการสนทนา

![วิดเจ็ตข่าวสารและการสนทนา](/img/admin/network-dashboard-news.png)

วิดเจ็ตนี้ดึงและแสดงข่าวสารล่าสุดของ Ultimate Multisite คอยติดตามวิดเจ็ตนี้เพื่อรับข้อมูลเกี่ยวกับการอัปเดต บั๊กสำคัญ และการแก้ไขด้านความปลอดภัย

## แดชบอร์ด Ultimate Multisite

ในขณะที่แดชบอร์ดระดับเครือข่ายแสดงข้อมูลภาพรวมของเครือข่ายทั้งหมด แดชบอร์ด Ultimate Multisite (อยู่ที่ระดับบนสุดของเมนู Ultimate Multisite) จะแสดงข้อมูลเกี่ยวกับบริการ

![แดชบอร์ด Ultimate Multisite](/img/admin/um-dashboard-top.png)

### การเติบโตของรายได้ประจำรายเดือน

กราฟ MRR (รายได้ประจำรายเดือน) แสดงรายละเอียดรายได้รายเดือนในช่วงเวลา 12 เดือน

![กราฟการเติบโต MRR](/img/admin/um-dashboard-mrr.png)

รายได้ใหม่แสดงเป็นสีน้ำเงิน ในขณะที่การยกเลิกแสดงเป็นสีแดง

### การเลือกช่วงวันที่

ขึ้นอยู่กับจำนวนลูกค้าและกิจกรรม สถิติอาจมากเกินไปจนดูไม่ไหว การเลือกช่วงวันที่ที่มีในแดชบอร์ดช่วยให้ผู้ดูแลระบบเครือข่ายโฟกัสเฉพาะข้อมูลและช่วงเวลาที่ต้องการได้

![ตัวเลือกช่วงวันที่และสถิติ](/img/admin/um-dashboard-date-range.png)

ช่วงวันที่ควบคุมระยะเวลาของข้อมูลที่แสดงในวิดเจ็ตด้านล่าง ได้แก่ รายได้ สมาชิกภาพใหม่ การสมัครแยกตามประเทศ การสมัครแยกตามฟอร์ม และไซต์ที่มีผู้เข้าชมมากที่สุด

![วิดเจ็ตสถิติแดชบอร์ด](/img/admin/um-dashboard-stats.png)

:::tip
เพื่อให้วิดเจ็ต **Most Visited Sites** ทำงานได้ ให้เปิดใช้งานการนับการเข้าชมที่ **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**
:::
