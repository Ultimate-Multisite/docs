# การจัดการอีเมลระบบ {#managing-system-emails}

Ultimate Multisite ส่งอีเมลระบบต่าง ๆ สำหรับเหตุการณ์ เช่น การยืนยันการสมัคร ใบเสร็จการชำระเงิน และการเปลี่ยนแปลงสมาชิก คุณสามารถปรับแต่งเทมเพลตอีเมลเหล่านี้ให้เข้ากับแบรนด์ของคุณได้

## การดูอีเมลระบบ {#viewing-system-emails}

ไปที่ **Ultimate Multisite → Settings** แล้วคลิกแท็บ **Emails** เพื่อดูการตั้งค่าที่เกี่ยวข้องกับอีเมล

![แท็บ Settings - Emails](/img/admin/settings-emails.png)

จากตรงนั้น คุณสามารถเข้าถึงหน้า **System Emails** เพื่อดูเทมเพลตอีเมลทั้งหมดได้

![รายการ System Emails](/img/admin/system-emails-list.png)

คุณยังสามารถแก้ไขเทมเพลตอีเมลแต่ละรายการได้โดยใช้ตัวแก้ไขอีเมลระบบ:

![ตัวแก้ไขอีเมลระบบ](/img/admin/system-email-editor.png)

## การนำเข้าเทมเพลตอีเมลใหม่ {#importing-new-email-templates}

เวอร์ชันใหม่และ add-on ของ Ultimate Multisite อาจลงทะเบียนเทมเพลตอีเมลใหม่ เพื่อป้องกันความขัดแย้ง เทมเพลตใหม่จะไม่ถูกเพิ่มโดยอัตโนมัติ — คุณต้องนำเข้าเองด้วยตนเอง

วิธีนำเข้าเทมเพลตอีเมลใหม่:

1. ไปที่หน้า **System Emails**
2. คลิกปุ่ม **Reset & Import** ที่ด้านบนของหน้า
3. เปิดตัวเลือก **Import Emails** เพื่อดูเทมเพลตที่มีให้ใช้งาน
4. เลือกเทมเพลตที่คุณต้องการนำเข้าและยืนยัน

## การรีเซ็ตเทมเพลตอีเมล {#resetting-email-templates}

หากคุณได้ปรับแต่งเทมเพลตอีเมลแล้วและต้องการย้อนกลับไปเป็นเนื้อหาเริ่มต้น คุณมีสองตัวเลือก:

1. **ลบและนำเข้าใหม่** — ลบอีเมลระบบและนำเข้าอีกครั้ง (การทำเช่นนี้จะลบสถิติการส่ง)
2. **รีเซ็ตผ่านเครื่องมือ** — ใช้เครื่องมือ **Reset & Import** เปิดตัวเลือก **Reset** และเลือกอีเมลที่คุณต้องการรีเซ็ต

ตัวเลือก Reset เป็นตัวเลือกที่แนะนำ เพราะจะคงสถิติการส่งและข้อมูลเมตาอื่น ๆ ไว้ พร้อมกับกู้คืนเนื้อหาเริ่มต้นกลับมา
