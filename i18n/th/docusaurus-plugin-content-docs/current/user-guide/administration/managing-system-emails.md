# การจัดการอีเมลระบบ

Ultimate Multisite จะส่งอีเมลระบบต่างๆ สำหรับเหตุการณ์ต่างๆ เช่น การยืนยันการสมัคร ใบเสร็จการชำระเงิน และการเปลี่ยนแปลงสมาชิกภาพ คุณสามารถปรับแต่งเทมเพลตอีเมลเหล่านี้ให้ตรงกับแบรนด์ของคุณได้

## การดูอีเมลระบบ

ไปที่ **Ultimate Multisite → Settings** แล้วคลิกที่แท็บ **Emails** เพื่อดูการตั้งค่าที่เกี่ยวข้องกับอีเมล

![Settings - Emails tab](/img/admin/settings-emails.png)

จากนั้น คุณสามารถเข้าถึงหน้า **System Emails** เพื่อดูเทมเพลตอีเมลทั้งหมด

![System Emails list](/img/admin/system-emails-list.png)

## การนำเข้าเทมเพลตอีเมลใหม่

Ultimate Multisite เวอร์ชันใหม่และ add-on ต่างๆ อาจลงทะเบียนเทมเพลตอีเมลใหม่ เพื่อป้องกันปัญหาความขัดแย้ง เทมเพลตใหม่จะไม่ถูกเพิ่มโดยอัตโนมัติ — คุณต้องนำเข้าด้วยตนเอง

วิธีนำเข้าเทมเพลตอีเมลใหม่:

1. ไปที่หน้า **System Emails**
2. คลิกปุ่ม **Reset & Import** ที่ด้านบนของหน้า
3. เปิดตัวเลือก **Import Emails** เพื่อดูเทมเพลตที่มีให้เลือก
4. เลือกเทมเพลตที่ต้องการนำเข้าและยืนยัน

## การรีเซ็ตเทมเพลตอีเมล

หากคุณปรับแต่งเทมเพลตอีเมลไปแล้วและต้องการกลับไปใช้เนื้อหาเริ่มต้น คุณมีสองทางเลือก:

1. **ลบแล้วนำเข้าใหม่** — ลบอีเมลระบบและนำเข้าใหม่อีกครั้ง (วิธีนี้จะลบสถิติการส่งด้วย)
2. **รีเซ็ตผ่านเครื่องมือ** — ใช้เครื่องมือ **Reset & Import** เปิดตัวเลือก **Reset** และเลือกอีเมลที่ต้องการรีเซ็ต

แนะนำให้ใช้ตัวเลือก Reset เพราะจะเก็บรักษาสถิติการส่งและข้อมูลเมตาอื่นๆ ไว้ ในขณะที่กู้คืนเนื้อหาเริ่มต้น
