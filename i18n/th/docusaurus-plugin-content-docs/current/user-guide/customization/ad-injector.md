# การวางโฆษณาบนเครือข่ายของคุณด้วย Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector เป็น**ส่วนเสริมฟรี**ที่ให้เครื่องมือสำหรับวางโฆษณาบนแผนของคุณได้โดยตรงจากหน้าแก้ไข

หลังจากติดตั้ง Ultimate Multisite: Ad Injector แล้ว ให้ไปที่หน้า _Plugins_ เพื่อเปิดใช้งาน

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_คลิกปุ่ม Network Activate_

เมื่อส่วนเสริมเปิดใช้งานแล้ว กล่อง _Advanced Options_ ในหน้าแก้ไขแผนจะมีตัวเลือกใหม่ชื่อ _Ad Settings_

![Ad Settings บนหน้าแก้ไขแผน](/img/config/product-edit-full.png)

_ตอนนี้คุณสามารถแก้ไขการตั้งค่าโฆษณาสำหรับแต่ละแผนของคุณได้แล้ว_

ในแท็บนั้น คุณสามารถเพิ่มโค้ดโฆษณาสำหรับโพสต์และหน้าในส่วนหน้าเว็บ ทั้งก่อนและหลังเนื้อหา

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_หลังจากใส่โค้ดโฆษณาแล้ว โฆษณาจะแสดงบนโพสต์และหน้าของแผนนั้นๆ_

คุณยังสามารถแสดงโฆษณาบนส่วนหัวของหน้า Admin สำหรับผู้ใช้ในแผนนั้นได้อีกด้วย

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_ไปที่ตัวเลือก Admin Panel Ad Code หากคุณต้องการแสดงโฆษณาในหน้าหลังบ้านของผู้ใช้_

### นอกเหนือจากการโฆษณา

คุณยังสามารถใช้ Ultimate Multisite: Ad Injector เพื่อให้บริการสนับสนุนแก่แผนระดับสูงได้อีกด้วย

เพียงไปที่ตัวเลือก _Admin Panel Ads_ แล้วเพิ่มโค้ด HTML สำหรับเครื่องมือสนับสนุนของคุณ

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_เพิ่มการสนับสนุนลูกค้าให้กับแผนที่ต้องการ_

ด้วยวิธีนี้ ผู้คนสามารถติดต่อคุณได้โดยตรงจาก Admin Panel ของพวกเขา!
