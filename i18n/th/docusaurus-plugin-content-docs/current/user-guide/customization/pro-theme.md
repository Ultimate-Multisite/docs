# การปรับแต่ง Admin Panel ด้วย PRO Theme

บ่อยครั้งเวลาทำงานให้ลูกค้า เรามักต้องการมอบประสบการณ์ที่ปรับแต่งเฉพาะใน admin dashboard ของ WordPress ให้กับลูกค้า หากคุณกำลังมองหารูปลักษณ์ที่ดูเป็นมืออาชีพมากขึ้น PRO Theme อาจช่วยคุณได้

PRO Theme เป็น admin theme ที่พัฒนาขึ้นเพื่อช่วยคุณปรับแต่ง Dashboard ให้กับลูกค้า และยังเป็นส่วนหนึ่งของ plugin หลักระดับ premium ของ Ultimate Multisite อีกด้วย

หลังจากติดตั้งแล้ว ให้ไปที่หน้า _Plugins_ และคลิกเปิดใช้งานบน Network ของคุณ

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_เปิดใช้งานแบบ network-wide ในหน้า Plugins ของคุณ_

_**หมายเหตุ:**_ _หากคุณแค่ติดตั้งโดยไม่ได้เปิดใช้งานแบบ network-wide มันจะแสดงเป็นตัวเลือกในหน้าของลูกค้า ซึ่งสามารถเลือกเพื่อเพิ่ม theme เฉพาะสำหรับผู้ใช้คนนั้นได้_

ตอนนี้ หน้า admin interface ของคุณได้รับการออกแบบใหม่ทั้งหมดแล้ว! คุณยังมีเมนูใหม่ในคอลัมน์ด้านซ้ายที่ชื่อว่า PRO Theme
