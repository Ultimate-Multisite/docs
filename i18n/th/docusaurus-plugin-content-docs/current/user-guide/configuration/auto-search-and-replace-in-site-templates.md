# ค้นหาและแทนที่อัตโนมัติในเทมเพลตเว็บไซต์ (v2)

_**บทช่วยสอนนี้ต้องใช้ WP UItimo เวอร์ชัน 2.x**_

หนึ่งในฟีเจอร์ที่ทรงพลังที่สุดของ Ultimate Multisite คือความสามารถในการเพิ่มฟิลด์ข้อความ สี และตัวเลือกแบบกำหนดเองลงในแบบฟอร์มลงทะเบียน เมื่อเราบันทึกข้อมูลนั้นแล้ว เราสามารถใช้ข้อมูลนั้นเพื่อเติมเนื้อหาล่วงหน้าในบางส่วนของเทมเพลตเว็บไซต์ที่เลือก จากนั้นเมื่อเว็บไซต์ใหม่ถูกเผยแพร่ Ultimate Multisite จะแทนที่ตัวแทนข้อความด้วยข้อมูลจริงที่กรอกระหว่างการลงทะเบียน

ตัวอย่างเช่น คุณสามารถทำเว็บไซต์เทมเพลตของคุณด้วยตัวแทนข้อความได้ ควรเพิ่มตัวแทนข้อความโดยล้อมด้วยวงเล็บปีกกาคู่ - {{placeholder_name}}

จากนั้น คุณสามารถเพิ่มฟิลด์ลงทะเบียนที่ตรงกันเพื่อบันทึกข้อมูลนั้นได้อย่างง่ายดาย

จากนั้นลูกค้าของคุณจะสามารถกรอกฟิลด์นั้นระหว่างการลงทะเบียนได้

จากนั้น Ultimate Multisite จะแทนที่ตัวแทนข้อความด้วยข้อมูลที่ลูกค้าให้มาโดยอัตโนมัติ

## **การแก้ปัญหา "เทมเพลตที่เต็มไปด้วยตัวแทนข้อความ"**

ทั้งหมดนั้นยอดเยี่ยมมาก แต่เราพบปัญหาที่ไม่น่าดู: ตอนนี้เทมเพลตเว็บไซต์ของเรา - ที่ลูกค้าของเราสามารถเข้าชมได้ - เต็มไปด้วยตัวแทนข้อความที่ไม่น่าดูและไม่ได้บอกอะไรมากนัก

เพื่อแก้ปัญหานั้น เรามีตัวเลือกให้ตั้งค่าปลอมสำหรับตัวแทนข้อความ และเราใช้ค่าเหล่านั้นเพื่อค้นหาและแทนที่เนื้อหาของตัวแทนข้อความบนเว็บไซต์เทมเพลตในขณะที่ลูกค้าของคุณกำลังเข้าชม

คุณสามารถเข้าถึงตัวแก้ไขตัวแทนข้อความของเทมเพลตได้โดยไปที่ **Ultimate Multisite > Settings > Sites** เลื่อนลงไปยังพื้นที่ตัวเลือกเทมเพลตเว็บไซต์ แล้วคลิกลิงก์ **แก้ไขตัวแทนข้อความ**

![พื้นที่ตัวเลือกเทมเพลตเว็บไซต์ในหน้าการตั้งค่าเว็บไซต์](/img/config/settings-sites-templates-section.png)

การดำเนินการนั้นจะพาคุณไปยังตัวแก้ไขเนื้อหาของตัวแทนข้อความ ซึ่งคุณสามารถเพิ่มตัวแทนข้อความและเนื้อหาที่เกี่ยวข้องได้

![จุดเริ่มต้นของตัวแก้ไขตัวแทนข้อความของเทมเพลต](/img/config/settings-sites-templates-section.png)
