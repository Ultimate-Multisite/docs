# ระบบค้นหาและแทนที่อัตโนมัติในเทมเพลตเว็บไซต์ (v2)

_**บทเรียนนี้ต้องใช้ WP UItimo เวอร์ชัน 2.x**_

หนึ่งในฟีเจอร์ที่ทรงพลังที่สุดของ Ultimate Multisite คือความสามารถในการเพิ่มฟิลด์ข้อความ สี และตัวเลือกต่างๆ ลงในฟอร์มลงทะเบียนได้อย่างอิสระ เมื่อเก็บข้อมูลเหล่านั้นได้แล้ว เราสามารถนำมาใช้เติมเนื้อหาล่วงหน้าในส่วนต่างๆ ของเทมเพลตเว็บไซต์ที่เลือกไว้ จากนั้นเมื่อเว็บไซต์ใหม่ถูกเผยแพร่ Ultimate Multisite จะแทนที่ตัวแทนข้อความ (placeholder) ด้วยข้อมูลจริงที่ผู้ใช้กรอกไว้ตอนลงทะเบียน

ตัวอย่างเช่น คุณสามารถสร้างเทมเพลตเว็บไซต์พร้อมตัวแทนข้อความได้ โดยตัวแทนข้อความควรครอบด้วยวงเล็บปีกกาคู่ เช่น {{placeholder_name}}

จากนั้น คุณเพียงเพิ่มฟิลด์ลงทะเบียนที่ตรงกันเพื่อเก็บข้อมูลนั้น

ลูกค้าของคุณจะสามารถกรอกฟิลด์นั้นได้ระหว่างการลงทะเบียน

Ultimate Multisite จะแทนที่ตัวแทนข้อความด้วยข้อมูลที่ลูกค้ากรอกไว้โดยอัตโนมัติ

## **แก้ปัญหา "เทมเพลตเต็มไปด้วยตัวแทนข้อความ"**

ทุกอย่างดูดี แต่เราก็พบปัญหาที่น่ารำคาญ: ตอนนี้เทมเพลตเว็บไซต์ของเรา — ซึ่งลูกค้าสามารถเข้าชมได้ — เต็มไปด้วยตัวแทนข้อความที่ดูไม่สวยและไม่สื่อความหมายมากนัก

เพื่อแก้ปัญหานี้ เรามีตัวเลือกให้ตั้งค่าข้อความตัวอย่างสำหรับตัวแทนข้อความ และเราจะใช้ค่าเหล่านั้นในการค้นหาและแทนที่เนื้อหาบนเทมเพลตเว็บไซต์ขณะที่ลูกค้าของคุณกำลังเยี่ยมชม

คุณสามารถเข้าถึงหน้าแก้ไขตัวแทนข้อความของเทมเพลตได้โดยไปที่ **Ultimate Multisite > Settings > Sites** จากนั้นคลิกลิงก์ Edit Placeholders ที่แถบด้านข้าง

![ลิงก์ Edit Placeholders ในแถบด้านข้างของการตั้งค่า Sites](/img/config/settings-sites.png)

คุณจะเข้าสู่หน้าแก้ไขเนื้อหาตัวแทนข้อความ ซึ่งคุณสามารถเพิ่มตัวแทนข้อความและค่าที่ต้องการได้

![หน้าแก้ไขเนื้อหาตัวแทนข้อความพร้อมชื่อและค่าของ placeholder](/img/config/settings-sites.png)
