# Agents ในตัว

Gratis AI Agent v1.9.0 มาพร้อม agents ในตัวห้ารายการ แต่ละรายการถูกตั้งค่าล่วงหน้าด้วยชุดเครื่องมือที่เน้นเฉพาะด้าน system prompt ที่ปรับแต่งแล้ว และคำแนะนำเริ่มต้นที่สอดคล้องกับงานทั่วไปในด้านนั้น ๆ การสลับระหว่าง agents จะเปลี่ยนสิ่งที่ผู้ช่วยทำได้และวิธีที่ผู้ช่วยตอบกลับ — โดยไม่ต้องกำหนดค่าใด ๆ จากคุณ Superdav AI Agent v1.18.0 สามารถเพิ่มเครื่องมือที่คำนึงถึงกำหนดการ บันทึกการเตือนความจำ ประตูอนุมัติ และการแจ้งเตือน SMS ให้กับเวิร์กโฟลว์เหล่านี้เมื่อมีการกำหนดค่าการผสานการทำงานที่เกี่ยวข้อง

## Agent คืออะไร? {#what-is-an-agent}

แต่ละ agent คือโปรไฟล์การกำหนดค่าที่มีชื่อ ซึ่งรวม:

- **เครื่องมือ** — ความสามารถที่ agent ได้รับอนุญาตให้เรียกใช้ (เช่น Content Writer มีสิทธิ์เข้าถึงความสามารถในการสร้างโพสต์; Design Studio มีสิทธิ์เข้าถึงความสามารถด้าน CSS และ theme.json)
- **System prompt** — คำสั่งที่กำหนดโทน ลำดับความสำคัญ และข้อจำกัดของ agent
- **คำแนะนำ** — prompt ที่เขียนไว้ล่วงหน้าซึ่งแสดงในอินเทอร์เฟซแชทเพื่อช่วยให้คุณเริ่มต้นได้อย่างรวดเร็ว

## การเข้าถึง Agent Picker {#accessing-the-agent-picker}

1. เปิดแผง **Gratis AI Agent** ในแถบด้านข้างผู้ดูแลระบบ WordPress
2. คลิก **ไอคอน agent** ที่มุมซ้ายบนของส่วนหัวแชท (ไอคอนจะเปลี่ยนเพื่อสะท้อน agent ที่ใช้งานอยู่)
3. **Agent Picker** จะเปิดเป็นโอเวอร์เลย์แบบ form-table แต่ละ agent จะแสดงพร้อมไอคอน ชื่อ และคำอธิบายหนึ่งบรรทัด
4. คลิกแถวของ agent เพื่อเปิดใช้งาน ส่วนหัวแชทจะอัปเดตทันที

คุณยังสามารถสลับ agents ระหว่างการสนทนาได้ — system prompt ของ agent ใหม่จะมีผลตั้งแต่ข้อความถัดไป

## Agents ในตัวทั้งห้า {#the-five-built-in-agents}

### Content Writer {#content-writer}

**จุดเน้น:** การสร้างและแก้ไขโพสต์ หน้า และแบบฟอร์มติดต่อ

**เครื่องมือที่มีให้ใช้:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types` เมื่อเปิดใช้งานการผสานการทำงานของ Superdav AI Agent v1.18.0 อาจมีเครื่องมือสำหรับบริบทปฏิทินที่กำหนดค่าแล้ว ประตูอนุมัติ การเตือนความจำ และการแจ้งเตือน SMS ให้ใช้งานในเวิร์กโฟลว์ที่ได้รับอนุมัติด้วย

**สิ่งที่ทำได้ดี:**
- ร่างและเผยแพร่โพสต์บล็อกจากบทสรุปหรือโครงร่าง
- สร้างชุดของหน้าแลนดิ้งสำหรับเว็บไซต์ใหม่
- สร้างแบบฟอร์มติดต่อและแบบฟอร์มสอบถาม
- ตั้งค่ารูปภาพเด่นบนโพสต์จาก URL หรือการค้นหา
- ร่างข้อความติดตามผลอีเวนต์จากบริบท Google Calendar ที่กำหนดค่าไว้ จากนั้นหยุดรอการอนุมัติก่อนส่งการแจ้งเตือน

**คำแนะนำเริ่มต้น:**
- *เขียนโพสต์บล็อก 500 คำเกี่ยวกับประโยชน์ของ WordPress multisite*
- *สร้างหน้าเกี่ยวกับเรา บริการ และติดต่อ แล้วเผยแพร่*
- *เพิ่มแบบฟอร์มสอบถามการจองลงในหน้าติดต่อ*
- *ร่างการเตือนความจำสำหรับผู้เข้าร่วมในอีเวนต์ปฏิทินที่กำหนดค่าไว้ของวันพรุ่งนี้ และรอการอนุมัติก่อนส่ง*

---

### Site Builder {#site-builder}

**จุดเน้น:** การสร้างเว็บไซต์แบบครบวงจรจาก prompt เดียว

**เครื่องมือที่มีให้ใช้:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form` ด้วย Superdav AI Agent v1.18.0 อาจมีเครื่องมือสำหรับบริการที่จัดการ การอนุมัติ การเตือนความจำ ปฏิทิน และ SMS ที่กำหนดค่าแล้วให้ใช้เมื่อผู้ดูแลระบบเปิดใช้งาน

**สิ่งที่ทำได้ดี:**
- สร้างแผนการสร้างเว็บไซต์หลายระยะสำหรับประเภทธุรกิจที่อธิบายไว้
- ดำเนินการแต่ละระยะโดยอัตโนมัติ — โครงสร้าง เนื้อหา การนำทาง การออกแบบ
- กู้คืนจากข้อผิดพลาดระหว่างแผนโดยไม่ต้องมีการแทรกแซงด้วยตนเอง
- ติดตั้ง plugins ที่แนะนำเป็นส่วนหนึ่งของการสร้าง
- สร้างแบบฟอร์มติดต่อโดยตรงจากอินเทอร์เฟซแชท (Superdav AI Agent v1.10.0+)
- ประสานงานการเตือนความจำการเปิดตัวหรือการติดตามผลผู้เข้าร่วมโดยไม่มีการแจ้งเตือนซ้ำ เมื่อเปิดใช้งานประตูอนุมัติและบันทึกการเตือนความจำ

**คำแนะนำเริ่มต้น:**
- *สร้างเว็บไซต์พอร์ตโฟลิโอการถ่ายภาพพร้อมประเภทโพสต์แกลเลอรี หน้าการจอง และแบบฟอร์มติดต่อ*
- *สร้างเว็บไซต์ร้านอาหารพร้อมเมนูออนไลน์ เวลาเปิดทำการ และแบบฟอร์มสอบถามการจองโต๊ะ*
- *ตั้งค่าเว็บไซต์ที่ปรึกษาอิสระพร้อมหน้าบริการ ส่วนพอร์ตโฟลิโอ และบล็อก*
- *เพิ่มแบบฟอร์มติดต่อไปยังหน้าติดต่อโดยใช้ site builder*
- *หลังจากเช็กลิสต์การเปิดตัวเว็บไซต์ได้รับการอนุมัติแล้ว ให้ส่งการเตือนความจำ SMS ไปยังผู้ติดต่อผู้มีส่วนได้ส่วนเสียที่กำหนดค่าไว้*

### Design Studio {#design-studio}

**จุดเน้น:** การปรับแต่งด้านภาพ — สี ตัวพิมพ์ CSS และ block patterns

**เครื่องมือที่มีให้ใช้:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**สิ่งที่ทำได้ดี:**
- ใช้ค่าพรีเซ็ต theme ที่มีชื่อ (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- ปรับแต่งตัวพิมพ์ส่วนกลางและพาเลตสีอย่างละเอียดผ่าน theme.json
- แทรก CSS แบบกำหนดเองสำหรับการเขียนทับเฉพาะแบรนด์
- ถ่ายภาพหน้าจอของหน้าและตรวจทานปัญหาด้านการออกแบบ

**คำแนะนำเริ่มต้น:**
- *ใช้พรีเซ็ต warm-editorial แล้วตั้งค่าสีหลักเป็น #2d6a4f*
- *ถ่ายภาพหน้าจอของหน้าแรกและบอกฉันว่าคุณจะปรับปรุงอะไร*
- *สร้าง hero block pattern ที่ใช้ซ้ำได้พร้อมภาพพื้นหลังเต็มความกว้างและหัวเรื่องกึ่งกลาง*

### Plugin Manager {#plugin-manager}

**จุดเน้น:** การค้นพบ ติดตั้ง และจัดการ WordPress plugins

**เครื่องมือที่มีให้ใช้:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**สิ่งที่ทำได้ดี:**
- แนะนำ plugin ที่ดีที่สุดสำหรับกรณีใช้งานที่อธิบายไว้
- ติดตั้ง ability packs จากรีจิสทรี
- เรียกดูแค็ตตาล็อกความสามารถที่มีอยู่ตามหมวดหมู่

**คำแนะนำเริ่มต้น:**
- *plugin ที่ดีที่สุดสำหรับไดเรกทอรีสมาชิกคืออะไร?*
- *ติดตั้งชุดความสามารถ WooCommerce*
- *แสดงชุดความสามารถอีคอมเมิร์ซที่มีทั้งหมดให้ฉันดู*

---

### Support Assistant {#support-assistant}

**จุดเน้น:** การตอบคำถามเกี่ยวกับเนื้อหาเว็บไซต์ การตั้งค่า และการกำหนดค่า WordPress

**เครื่องมือที่มีให้ใช้:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**สิ่งที่ทำได้ดี:**
- ค้นหาการตั้งค่าและตัวเลือกปัจจุบันของ site
- อธิบายว่า post types, taxonomies และเมนูใดถูกกำหนดค่าไว้บน site
- ตอบคำถาม "การตั้งค่านี้ทำอะไร?" โดยอ่านค่าจริงแบบสด
- ทำหน้าที่เป็นชั้นการวินิจฉัยแบบอ่านอย่างเดียวก่อนทำการเปลี่ยนแปลง

**คำแนะนำเริ่มต้น:**
- *plugins และการตั้งค่าใดกำลังใช้งานอยู่บน site นี้ในขณะนี้?*
- *แสดงรายการ custom post types ทั้งหมดที่ลงทะเบียนไว้บน site นี้*
- *มีเมนูนำทางใดบ้าง และถูกกำหนดไว้ที่ไหน?*

## การผสานการทำงานอัตโนมัติของ Superdav {#superdav-automation-integrations}

เมื่อมีการกำหนดค่าการผสานการทำงานของ Superdav AI Agent v1.18.0 agents ในตัวสามารถมีส่วนร่วมในเวิร์กโฟลว์อัตโนมัติที่ปลอดภัยยิ่งขึ้นและรับรู้กำหนดการได้:

- **เครื่องมืออ่าน Google Calendar** ช่วยให้ agents ตรวจสอบปฏิทินและกิจกรรมที่กำหนดค่าไว้ก่อนร่างงานติดตามผล
- **การจับคู่ผู้ติดต่อและผู้เข้าร่วม** ช่วยจับคู่ผู้เข้าร่วมกิจกรรมกับผู้ใช้ WordPress หรือผู้ติดต่อที่รู้จัก
- **ประตูการอนุมัติจากมนุษย์** หยุดการดำเนินการที่ละเอียดอ่อนไว้ชั่วคราวจนกว่าผู้ใช้ที่ได้รับอนุญาตจะตรวจสอบและยืนยัน
- **ระเบียนการแจ้งเตือน** ป้องกันการแจ้งเตือนซ้ำเมื่อ scheduled jobs ลองใหม่หรือทำซ้ำ
- **การแจ้งเตือน TextBee SMS** ส่งข้อความตัวอักษรที่กำหนดค่าไว้เฉพาะเมื่อมีการเปิดใช้ข้อมูลรับรอง SMS และสิทธิ์ของเวิร์กโฟลว์

เวิร์กโฟลว์ที่แนะนำ: ขอให้ agent เตรียมข้อความหรือการดำเนินการ ตรวจสอบพรอมป์การอนุมัติ จากนั้นอนุญาตให้การดำเนินการที่ได้รับอนุมัติกลับมาทำงานต่อ สำหรับการแจ้งเตือนที่เกิดซ้ำ ให้เปิดใช้การลบรายการซ้ำของการแจ้งเตือนไว้ เพื่อไม่ให้กิจกรรมหรือผู้ติดต่อเดียวกันถูกแจ้งซ้ำหลายครั้ง

## การปรับแต่ง Agents {#customising-agents}

agent ในตัวแต่ละตัวสามารถขยายหรือแทนที่ได้ผ่านตัวกรอง `gratis_ai_agent_agents`

### การเพิ่ม system prompt แบบกำหนดเองให้กับ agent ที่มีอยู่ {#adding-a-custom-system-prompt-to-an-existing-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### การลงทะเบียน agent ใหม่ {#registering-a-new-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

agent ใหม่จะปรากฏใน Agent Picker ทันทีหลังจากตัวกรองทำงาน

### การลบ agent ในตัว {#removing-a-built-in-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
