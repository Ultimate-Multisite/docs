# ขั้นตอนการสมัครสมาชิก (v2)

_**หมายเหตุสำคัญ: บทความนี้อ้างอิงถึง Ultimate Multisite เวอร์ชัน 2.x**_

ผู้ใช้สามารถสมัครสมาชิกเข้าร่วมเครือข่ายของคุณได้หลายวิธี ไม่ว่าจะผ่านแบบฟอร์มสมัครสมาชิก หรือใช้ลิงก์ที่แชร์ได้ซึ่งเลือกแพลนไว้ล่วงหน้า ในบทความนี้ เราจะแสดงให้คุณเห็นว่าลูกค้าสามารถสมัครสมาชิกบนเครือข่ายของคุณได้อย่างไรผ่านช่องทางต่างๆ ที่มี และจะเกิดอะไรขึ้นหลังจากที่พวกเขาสมัครสมาชิกแล้ว

## การใช้แบบฟอร์มสมัครสมาชิก:

นี่คือขั้นตอนการสมัครสมาชิกแบบมาตรฐาน คุณสร้างหน้าสมัครสมาชิกที่มี **checkout form** และหน้านี้จะเป็นที่ที่ลูกค้าของคุณจะเข้ามาเพื่อสมัครสมาชิกบนเครือข่ายและสมัครใช้แพลน คุณสามารถมีหน้าสมัครสมาชิกหลายหน้าได้ โดยแต่ละหน้าก็มีแบบฟอร์มสมัครสมาชิกที่แตกต่างกันได้ตามต้องการ

หน้าเริ่มต้นสำหรับการสมัครสมาชิกคือ [_**yourdomain.com/register**_](http://yourdomain.com/register) แต่คุณสามารถเปลี่ยนได้ตลอดเวลาที่ **Ultimate Multisite > Settings > Login & Registration > Default Registration Page**

เมื่อผู้ใช้เข้ามาที่หน้าสมัครสมาชิกของคุณ (โดยปกติจะคลิกที่ปุ่ม **Sign in** หรือ **Buy now**) พวกเขาจะเห็นแบบฟอร์มสมัครสมาชิกของคุณที่นั่น

![แบบฟอร์มสมัครสมาชิกที่แสดงบนหน้าสมัครสมาชิก](/img/config/checkout-forms-list.png)

สิ่งที่พวกเขาต้องทำคือกรอกข้อมูลในช่องที่จำเป็นทั้งหมด เช่น อีเมล ชื่อผู้ใช้ รหัสผ่าน เป็นต้น แล้วชำระเงินสำหรับแพลน หรือยืนยันที่อยู่อีเมลหากพวกเขาสมัครแพลนฟรี หรือแพลนแบบชำระเงินที่มีช่วงทดลองใช้โดยไม่ต้องกรอกข้อมูลการชำระเงิน

ในหน้า "Thank you" พวกเขาจะเห็นข้อความแจ้งว่าต้องยืนยันที่อยู่อีเมลหรือไม่ หรือเว็บไซต์ของพวกเขาเปิดใช้งานแล้วและสามารถเริ่มใช้งานได้เลย

![หน้า Thank You หลังจากสมัครสมาชิก](/img/config/checkout-form-editor.png)

หากจำเป็นต้องยืนยันที่อยู่อีเมล พวกเขาจะต้องไปที่กล่องจดหมายอีเมลและคลิกที่ลิงก์ยืนยัน เว็บไซต์ของพวกเขาจะไม่ถูกเปิดใช้งานหากไม่ได้ยืนยันที่อยู่อีเมล

หากพวกเขาสมัครแพลนแบบชำระเงิน หรือการยืนยันอีเมลไม่ได้เป็นข้อบังคับบนเครือข่ายของคุณ เว็บไซต์ของพวกเขาจะถูกเปิดใช้งานทันทีหลังจาก checkout และจะเห็นลิงก์สำหรับเข้าสู่ระบบไปยัง dashboard ของพวกเขา

![เว็บไซต์เปิดใช้งานแล้วพร้อมลิงก์เข้าสู่ระบบไปยัง dashboard](/img/config/checkout-form-editor.png)

## การใช้ลิงก์ที่แชร์ได้:

ขั้นตอนการสมัครสมาชิกผ่านลิงก์ที่แชร์ได้นั้นโดยพื้นฐานแล้วเหมือนกับแบบฟอร์มสมัครสมาชิก ข้อแตกต่างเพียงอย่างเดียวคือเมื่อใช้ลิงก์ที่แชร์ได้ ลูกค้าของคุณสามารถมีผลิตภัณฑ์หรือ website template ที่เลือกไว้ล่วงหน้าบน checkout form (ดูหัวข้อ Pre-selecting products and templates via URL parameters) หรืออาจมีรหัสคูปองที่เพิ่มไว้แล้ว (ดูหัวข้อ Using URL Parameters)

ขั้นตอนการสมัครสมาชิกจะเหมือนเดิม: พวกเขาจะต้องกรอกชื่อ ชื่อผู้ใช้ ที่อยู่อีเมล ชื่อเว็บไซต์และหัวข้อ เป็นต้น แต่แพลนหรือ site template จะถูกเลือกไว้ล่วงหน้าให้พวกเขาแล้ว

### การสมัครสมาชิกด้วยการชำระเงินแบบ Manual:

หากคุณไม่ต้องการใช้ PayPal, Stripe หรือ payment gateway อื่นๆ ที่ Ultimate Multisite หรือ add-on integrations เสนอให้ คุณสามารถใช้การชำระเงินแบบ manual สำหรับลูกค้าของคุณได้ ด้วยวิธีนี้ คุณสามารถสร้างใบแจ้งหนี้ให้พวกเขาชำระเงินผ่านระบบชำระเงินที่คุณต้องการหลังจากที่พวกเขาสมัครสมาชิกบนเครือข่ายของคุณ

ขั้นตอนการสมัครสมาชิกจะเหมือนกับด้านบนทุกประการ แต่บนหน้าสมัครสมาชิก ลูกค้าของคุณจะเห็นข้อความแจ้งว่าพวกเขาจะได้รับอีเมลพร้อมคำแนะนำเพิ่มเติมเพื่อทำการชำระเงินให้เสร็จสมบูรณ์

![ข้อความการชำระเงินแบบ manual ระหว่างการสมัครสมาชิก](/img/config/settings-payment-gateways.png)

และหลังจากการสมัครสมาชิกเสร็จสมบูรณ์ พวกเขาจะเห็นคำแนะนำการชำระเงินที่คุณตั้งไว้ (และจะได้รับทางอีเมลด้วย)

![คำแนะนำการชำระเงินที่แสดงหลังจากสมัครสมาชิก](/img/config/settings-payment-gateways.png)

คำแนะนำการชำระเงินสามารถเปลี่ยนได้ที่ **Ultimate Multisite > Settings > Payments** หลังจากเปิดตัวเลือกการชำระเงินแบบ **Manual**:

![ตัวเลือก Manual payment พร้อมช่องคำแนะนำการชำระเงิน](/img/config/settings-payment-gateways.png)

หลังจากที่ลูกค้าของคุณทำการชำระเงินแบบ manual เสร็จสิ้นและส่งการยืนยันให้คุณ คุณต้อง **ยืนยันการชำระเงินด้วยตนเอง** เพื่อเปิดใช้งานสมาชิกภาพและเว็บไซต์ของลูกค้า

ในการทำเช่นนี้ ให้ไปที่ **Ultimate Multisite > Payments** และค้นหาการชำระเงินของลูกค้า สถานะควรยังคงแสดงเป็น **Pending**

![รายการ Payments พร้อมการชำระเงินแบบ manual ที่รอดำเนินการ](/img/admin/payments-list.png)

คลิกที่หมายเลขการชำระเงินและคุณจะสามารถเปลี่ยนสถานะเป็น **Completed** ได้

![หน้ารายละเอียดการชำระเงิน](/img/admin/payments-list.png)

![การเปลี่ยนสถานะการชำระเงินเป็น Completed](/img/admin/payments-list.png)

หลังจากเปลี่ยนสถานะเป็น **Completed** คุณจะเห็นข้อความ **Activate membership** ให้เปิดตัวเลือกนี้เพื่อเปิดใช้งานสมาชิกภาพและเว็บไซต์ที่เชื่อมโยงกับลูกค้ารายนี้ จากนั้นคลิก **Save Payment**

![ตัวเลือก Activate membership และปุ่ม Save Payment](/img/admin/payments-list.png)

ตอนนี้ลูกค้าของคุณควรสามารถเข้าถึง dashboard และฟีเจอร์ทั้งหมดที่พวกเขาสมัครใช้งานได้แล้ว
