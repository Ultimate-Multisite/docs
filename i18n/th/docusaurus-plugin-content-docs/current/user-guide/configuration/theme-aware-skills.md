# ทักษะที่รู้เรื่องธีม (Theme-Aware Skills)

Superdav AI Agent v1.10.0 มาพร้อมกับทักษะในตัวใหม่ที่รู้เรื่องธีมถึงสี่อย่าง ซึ่งจะปรับตัวให้เข้ากับ WordPress theme ที่คุณใช้งานอยู่โดยอัตโนมัติ ทักษะเหล่านี้จะให้คำแนะนำและความสามารถเฉพาะทางที่ปรับให้เข้ากับสถาปัตยกรรมและฟีเจอร์ของธีมของคุณโดยเฉพาะ

## ทักษะที่รู้เรื่องธีมคืออะไร?

ทักษะที่รู้เรื่องธีมคือฐานความรู้และชุดเครื่องมือที่ถูกตั้งค่าไว้ล่วงหน้า ซึ่งผู้ช่วย AI จะเลือกใช้โดยอัตโนมัติ โดยอิงจาก WordPress theme ที่กำลังใช้งานอยู่ในเว็บไซต์ของคุณ เมื่อคุณเปลี่ยนธีม ทักษะที่ผู้ช่วยสามารถใช้ได้ก็จะอัปเดตโดยอัตโนมัติ ไม่ต้องตั้งค่าด้วยตนเองเลย

แต่ละทักษะประกอบด้วย:

- **เอกสารเฉพาะธีม (Theme-specific documentation)** — คำแนะนำในการใช้งานและปรับแต่งธีม
- **การอ้างอิงบล็อกและรูปแบบ (Block and pattern references)** — บล็อก รูปแบบ (patterns) และตัวเลือกการออกแบบที่มีให้ใช้
- **ตัวอย่างการปรับแต่ง (Customization examples)** — ตัวอย่างโค้ดสั้นๆ และรูปแบบการตั้งค่าสำหรับงานทั่วไป
- **แนวทางปฏิบัติที่ดีที่สุด (Best practices)** — คำแนะนำสำหรับสถาปัตยกรรมและขั้นตอนการทำงานของธีม

## ทักษะที่รู้เรื่องธีมที่มีให้ใช้

### Block Themes

**ใช้ได้กับ:** ธีมที่ใช้สถาปัตยกรรมแบบบล็อกของ WordPress (Full Site Editing)

ทักษะ Block Themes จะให้คำแนะนำเกี่ยวกับ:

- การสร้างและแก้ไข template โดยใช้ block editor
- การทำงานกับ block patterns และ reusable blocks
- การปรับแต่ง global styles และการตั้งค่า theme.json
- การใช้ theme blocks และ variations
- การสร้าง custom block patterns สำหรับเว็บไซต์ของคุณ

**เปิดใช้งานโดยอัตโนมัติเมื่อ:** ธีมที่ใช้งานของคุณเป็น block theme (รองรับฟีเจอร์ `block-templates`)

### Classic Themes

**ใช้ได้กับ:** ธีม WordPress แบบดั้งเดิมที่ใช้ PHP templates และ classic editor

ทักษะ Classic Themes จะให้คำแนะนำเกี่ยวกับ:

- การทำงานกับไฟล์ template ของ PHP และ hooks
- การปรับแต่งรูปลักษณ์ของธีมผ่าน Customizer
- การใช้ widget areas และ sidebars
- การแก้ไข CSS และการพัฒนา child theme
- การทำความเข้าใจ theme hierarchy และ template tags

**เปิดใช้งานโดยอัตโนมัติเมื่อ:** ธีมที่ใช้งานของคุณเป็น classic (non-block) theme

### Kadence Blocks

**ใช้ได้กับ:** เว็บไซต์ที่ใช้ปลั๊กอิน Kadence Blocks สำหรับการออกแบบแบบบล็อกขั้นสูง

ทักษะ Kadence Blocks จะให้คำแนะนำเกี่ยวกับ:

- การใช้ไลบรารี block ขั้นสูงของ Kadence (เช่น Hero, Testimonials, Pricing ฯลฯ)
- การตั้งค่า Kadence block และตัวเลือก responsive
- การสร้าง layout แบบกำหนดเองด้วย grid และ container blocks ของ Kadence
- การรวม Kadence blocks เข้ากับธีมของคุณ
- การใช้ประโยชน์จาก design system และ presets ของ Kadence

**เปิดใช้งานโดยอัตโนมัติเมื่อ:** ปลั๊กอิน Kadence Blocks ถูกเปิดใช้งานบนเว็บไซต์ของคุณ

### Kadence Theme

**ใช้ได้กับ:** เว็บไซต์ที่ใช้ธีม Kadence สำหรับการออกแบบและการปรับแต่งแบบบล็อก

ทักษะ Kadence Theme จะให้คำแนะนำเกี่ยวกับ:

- การปรับแต่งธีม Kadence ผ่าน global styles และ theme.json
- การใช้ block patterns และ templates ที่มาพร้อมกับ Kadence
- การตั้งค่าและตัวเลือกของธีม Kadence
- การสร้างดีไซน์แบบกำหนดเองด้วย design system ของ Kadence
- การรวม Kadence เข้ากับปลั๊กอินและเครื่องมือยอดนิยม

**เปิดใช้งานโดยอัตโนมัติเมื่อ:** ธีม Kadence เป็นธีมที่ใช้งานของคุณ

## ทักษะถูกเลือกอย่างไร

ผู้ช่วยจะตรวจจับธีมที่ใช้งานและปลั๊กอินที่ติดตั้งโดยอัตโนมัติในทุกข้อความ หากมีทักษะที่รู้เรื่องธีมที่ตรงกัน ทักษะนั้นจะถูกโหลดเข้าสู่บริบทของผู้ช่วยโดยอัตโนมัติ คุณไม่จำเป็นต้องเปิดหรือสลับทักษะด้วยตนเอง

### หลายทักษะ

หากมีหลายทักษะที่ใช้ได้กับเว็บไซต์ของคุณ (ตัวอย่างเช่น หากคุณเปิดใช้งานทั้ง Kadence Blocks และ Kadence Theme) ผู้ช่วยจะสามารถเข้าถึงทักษะที่เกี่ยวข้องทั้งหมด และสามารถอ้างอิงคำแนะนำจากแต่ละทักษะได้

### การสลับธีม

เมื่อคุณเปลี่ยนธีมที่ใช้งาน ทักษะที่ผู้ช่วยสามารถใช้ได้จะอัปเดตโดยอัตโนมัติในข้อความถัดไป ตัวอย่างเช่น:

1. คุณกำลังใช้ block theme และมีทักษะ **Block Themes** เปิดใช้งานอยู่
2. คุณสลับไปใช้ classic theme
3. ในข้อความถัดไป **Classic Themes** จะถูกโหลดโดยอัตโนมัติ และทักษะ **Block Themes** จะไม่สามารถใช้งานได้อีกต่อไป

## การใช้ทักษะที่รู้เรื่องธีม

หากต้องการใช้ทักษะที่รู้เรื่องธีม เพียงแค่บรรยายสิ่งที่คุณต้องการทำในหน้าต่างแชท ผู้ช่วยจะอ้างอิงคำแนะนำจากทักษะที่เหมาะสมให้คุณโดยอัตโนมัติ

### ตัวอย่างคำสั่ง (Example Prompts)

**สำหรับ Block Themes:**
> "สร้าง hero section ที่มีพื้นหลังเป็นรูปภาพและข้อความอยู่ตรงกลางโดยใช้ block patterns"

**สำหรับ Classic Themes:**
> "เพิ่ม widget area แบบกำหนดเองที่ sidebar โดยใช้ child theme"

**สำหรับ Kadence Blocks:**
> "สร้างส่วน testimonials โดยใช้ Kadence Testimonials block"

**สำหรับ Kadence Theme:**
> "ปรับแต่ง layout ของ header และการจัดรูปแบบเมนูนำทาง"

ผู้ช่วยจะให้คำแนะนำเฉพาะธีมและตัวอย่างโค้ดที่ปรับให้เข้ากับธีมและปลั๊กอินที่คุณใช้งานอยู่

:::note
ทักษะที่รู้เรื่องธีมจะพร้อมใช้งานโดยอัตโนมัติใน Superdav AI Agent v1.10.0 และเวอร์ชันที่ใหม่กว่า ไม่จำเป็นต้องมีการตั้งค่าหรือกำหนดค่าเพิ่มเติม
:::
