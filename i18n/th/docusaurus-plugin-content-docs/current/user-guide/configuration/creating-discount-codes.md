# การสร้างรหัสส่วนลด (v2)

_**หมายเหตุสำคัญ: บทความนี้อ้างอิงถึง Ultimate Multisite เวอร์ชัน 2.x**_

ด้วย Ultimate Multisite คุณสามารถสร้างรหัสส่วนลดเพื่อมอบส่วนลดให้กับลูกค้าสำหรับการสมัครสมาชิกของพวกเขาได้ และการสร้างก็ทำได้ง่ายมาก!

## การสร้างและแก้ไขรหัสส่วนลด

หากต้องการสร้างหรือแก้ไขรหัสส่วนลด ให้ไปที่ **Ultimate Multisite > Discount Codes**

![](/img/config/discount-codes-list.png)

ที่นั่นคุณจะเห็นรายการรหัสส่วนลดที่คุณได้สร้างไว้แล้ว

คุณสามารถคลิกที่ **Add Discount** **Code** เพื่อสร้างคูปองใหม่ หรือจะแก้ไขรหัสที่มีอยู่โดยเลื่อนเมาส์ไปวางเหนือรายการนั้นแล้วคลิก **Edit**

![รายการรหัสส่วนลดพร้อมตัวเลือกเมื่อเลื่อนเมาส์ไปวาง](/img/config/discount-codes-list.png)

![ปุ่ม Add Discount Code](/img/config/discount-codes-list.png)

คุณจะถูกนำไปยังหน้าที่ใช้สร้างหรือแก้ไขรหัสคูปองของคุณ ในตัวอย่างนี้ เราจะสร้างรหัสใหม่

![หน้าแก้ไขรหัสส่วนลด](/img/config/discount-codes-list.png)

มาดูการตั้งค่าต่างๆ ที่มีให้ใช้งานกัน:

**Enter Discount Code:** นี่คือชื่อของรหัสส่วนลดของคุณ ไม่ใช่รหัสที่ลูกค้าจะต้องใช้ในแบบฟอร์มชำระเงิน

**Description:** ที่นี่คุณสามารถอธิบายสั้นๆ ว่าคูปองนี้มีไว้เพื่ออะไร

![ช่องกรอกชื่อและคำอธิบายรหัสส่วนลด](/img/config/discount-codes-list.png)

**Coupon code:** ที่นี่คือจุดที่คุณกำหนดรหัสที่ลูกค้าจะต้องกรอกในขั้นตอนการชำระเงิน

![ช่องกรอก Coupon code](/img/config/discount-codes-list.png)

**Discount:** ที่นี่คุณสามารถตั้งค่าได้ทั้ง **เปอร์เซ็นต์** หรือ **จำนวนเงินคงที่** สำหรับรหัสส่วนลดของคุณ

![การตั้งค่าส่วนลดแบบเปอร์เซ็นต์หรือจำนวนคงที่](/img/config/discount-codes-list.png)

**Apply to renewals:** หากปิดตัวเลือกนี้ รหัสส่วนลดจะถูกใช้กับ **การชำระเงินครั้งแรก** เท่านั้น การชำระเงินครั้งต่อๆ ไปจะไม่ได้รับส่วนลด หากเปิดตัวเลือกนี้ รหัสส่วนลดจะมีผลกับการชำระเงินทุกครั้งในอนาคต

**Setup fee discount:** หากปิดตัวเลือกนี้ รหัสคูปองจะ **ไม่ให้ส่วนลดสำหรับค่าธรรมเนียมการตั้งค่า** ของคำสั่งซื้อ หากเปิดตัวเลือกนี้ คุณสามารถกำหนดส่วนลด (เปอร์เซ็นต์หรือจำนวนคงที่) ที่รหัสคูปองนี้จะใช้กับค่าธรรมเนียมการตั้งค่าของแผนต่างๆ ได้

![ตัวเลือก Apply to renewals และ setup fee discount](/img/config/discount-codes-list.png)

**Active:** เปิดหรือปิดใช้งานรหัสคูปองนี้ด้วยตนเอง

![ปุ่มเปิด/ปิดสถานะ Active ของรหัสส่วนลด](/img/config/discount-codes-list.png)

ใน **Advanced Optio** **ns** เรามีการตั้งค่าดังต่อไปนี้:

**Limit uses:**

  * **Uses:** ที่นี่คุณสามารถดูได้ว่ารหัสส่วนลดถูกใช้ไปกี่ครั้งแล้ว

  * **Max uses:** ตัวเลือกนี้จะจำกัดจำนวนครั้งที่ผู้ใช้สามารถใช้รหัสส่วนลดนี้ได้ ตัวอย่างเช่น หากคุณใส่ 10 ที่นี่ คูปองจะใช้ได้เพียง 10 ครั้ง หลังจากถึงขีดจำกัดนี้แล้ว รหัสคูปองจะไม่สามารถใช้ได้อีก

![การตั้งค่า Limit uses พร้อมช่อง uses และ max uses](/img/config/discount-codes-list.png)**Start & expiration dates:** ที่นี่คุณมีตัวเลือกในการเพิ่มวันที่เริ่มต้นและ/หรือวันหมดอายุให้กับคูปองของคุณ

![ช่องกรอกวันที่เริ่มต้นและวันหมดอายุ](/img/config/discount-codes-list.png)

**Limit products:** หากคุณเปิด **Select products** ผลิตภัณฑ์ทั้งหมดของคุณจะแสดงให้เห็น คุณสามารถเลือกด้วยตนเอง (โดยการเปิดหรือปิด) ว่าผลิตภัณฑ์ใดสามารถใช้รหัสคูปองนี้ได้ ผลิตภัณฑ์ที่ถูกปิดไว้จะไม่มีการเปลี่ยนแปลงใดๆ หากลูกค้าพยายามใช้รหัสคูปองนี้กับผลิตภัณฑ์เหล่านั้น

![ปุ่มเปิด/ปิดการเลือก Limit products](/img/config/discount-codes-list.png)

หลังจากตั้งค่าตัวเลือกทั้งหมดเหล่านี้แล้ว ให้คลิกที่ **Save Discount Code** เพื่อบันทึกคูปองของคุณ เท่านี้ก็เสร็จเรียบร้อย!

![ปุ่ม Save Discount Code](/img/config/discount-codes-list.png)

ตอนนี้คูปองอยู่ในรายการของคุณแล้ว และจากตรงนั้น คุณสามารถคลิกเพื่อ **แก้ไขหรือลบ** ได้

![รหัสส่วนลดในรายการพร้อมตัวเลือกแก้ไขและลบ](/img/config/discount-codes-list.png)

### การใช้ URL Parameters:

หากคุณต้องการปรับแต่งตารางราคาหรือสร้างหน้ารหัสคูปองที่สวยงามสำหรับเว็บไซต์ของคุณ และต้องการใช้รหัสส่วนลดกับแบบฟอร์มชำระเงินโดยอัตโนมัติ คุณสามารถทำได้ผ่าน URL parameters

ขั้นแรก คุณต้องได้รับลิงก์แชร์สำหรับแผนของคุณ โดยไปที่ **Ultimate Multisite > Products** และเลือกแผน

คลิกที่ปุ่ม **Click to Copy Shareable Link** ซึ่งจะให้ลิงก์แชร์สำหรับแผนนี้โดยเฉพาะ ในกรณีของเรา ลิงก์แชร์ที่ได้รับคือ [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/)_._

![หน้าผลิตภัณฑ์พร้อมปุ่มลิงก์แชร์](/img/config/products-list.png)

หากต้องการใช้รหัสส่วนลดกับแผนนี้โดยเฉพาะ เพียงเพิ่ม parameter **?discount_code=XXX** ต่อท้าย URL โดยที่ **XXX** คือรหัสคูปอง

ในตัวอย่างของเราที่นี่ เราจะใช้รหัสคูปอง **50OFF** กับผลิตภัณฑ์นี้โดยเฉพาะ

URL สำหรับแผนนี้พร้อมกับรหัสส่วนลด 50OFF ที่ใช้แล้วจะมีลักษณะดังนี้: [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/) _**?discount_code=50OFF**_
