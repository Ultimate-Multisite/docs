# การสร้างรหัสส่วนลด (v2)

_**หมายเหตุสำคัญ: บทความนี้อ้างอิงถึง Ultimate Multisite เวอร์ชัน 2.x**_

ด้วย Ultimate Multisite คุณสามารถสร้างรหัสส่วนลดเพื่อมอบส่วนลดให้ลูกค้าของคุณสำหรับการสมัครสมาชิกของพวกเขาได้ และการสร้างก็ทำได้ง่าย!

## การสร้างและแก้ไขรหัสส่วนลด

หากต้องการสร้างหรือแก้ไขรหัสส่วนลด ให้ไปที่ **Ultimate Multisite > รหัสส่วนลด**

![รายการรหัสส่วนลด — สถานะว่างก่อนที่จะมีรหัสใด ๆ](/img/config/discount-codes-empty.png)

ที่นั่นคุณจะมีรายการรหัสส่วนลดที่คุณสร้างไว้แล้ว

คุณสามารถคลิกที่ **เพิ่มรหัส** **ส่วนลด** เพื่อสร้างคูปองใหม่ หรือคุณสามารถแก้ไขรายการที่มีอยู่ได้โดยเลื่อนเมาส์ไปเหนือรายการนั้นแล้วคลิก **แก้ไข**

![รายการรหัสส่วนลดพร้อมการกระทำเมื่อเลื่อนเมาส์ แสดงลิงก์แก้ไขและลบ](/img/config/discount-codes-list-hover.png)

![ปุ่มเพิ่มรหัสส่วนลดในส่วนหัวของหน้า](/img/config/discount-codes-add-button.png)

คุณจะถูกเปลี่ยนเส้นทางไปยังหน้าที่คุณจะสร้างหรือแก้ไขรหัสคูปองของคุณ ในตัวอย่างนี้ เราจะสร้างรายการใหม่

![หน้าการแก้ไขรหัสส่วนลดพร้อมทุกส่วนที่มองเห็นได้](/img/config/discount-code-edit.png)

มาดูการตั้งค่าที่มีอยู่ที่นี่กัน:

**ป้อนรหัสส่วนลด:** นี่เป็นเพียงชื่อของรหัสส่วนลดของคุณ นี่ไม่ใช่รหัสที่ลูกค้าของคุณจะต้องใช้ในแบบฟอร์มชำระเงิน

**คำอธิบาย:** ที่นี่ คุณสามารถอธิบายสั้น ๆ ได้ว่าคูปองนี้มีไว้เพื่ออะไร

![ช่องชื่อและคำอธิบายรหัสส่วนลดที่ด้านบนของหน้าแก้ไข](/img/config/discount-code-description.png)

คุณยังสามารถดูรหัสส่วนลดว่าเปิดใช้งานหรือไม่เปิดใช้งานได้:

![สถานะเปิดใช้งานของรหัสส่วนลด](/img/config/discount-code-active.png)

**รหัสคูปอง:** ที่นี่คือที่ที่คุณกำหนดรหัสที่ลูกค้าของคุณจะต้องป้อนระหว่างการชำระเงิน

![ช่องรหัสคูปองที่ลูกค้าป้อนรหัสตอนชำระเงิน](/img/config/discount-code-coupon-field.png)

**ส่วนลด:** ที่นี่ คุณสามารถตั้งค่าส่วนลดเป็น **เปอร์เซ็นต์** หรือ **จำนวนเงินคงที่** สำหรับรหัสส่วนลดของคุณได้

![การตั้งค่าจำนวนส่วนลดพร้อมดรอปดาวน์เปอร์เซ็นต์หรือจำนวนเงินคงที่](/img/config/discount-code-amount.png)

**ใช้กับการต่ออายุ:** หากปิดตัวเลือกนี้ รหัสส่วนลดนี้จะถูกใช้กับ **การชำระเงินครั้งแรก** เท่านั้น การชำระเงินอื่น ๆ ทั้งหมดจะไม่มีส่วนลด หากเปิดตัวเลือกนี้ รหัสส่วนลดจะใช้ได้กับการชำระเงินในอนาคตทั้งหมด

**ส่วนลดค่าติดตั้ง:** หากปิดตัวเลือกนี้ รหัสคูปองจะ **ไม่ให้ส่วนลดใด ๆ สำหรับค่าติดตั้ง** ของคำสั่งซื้อ หากเปิดตัวเลือกนี้ คุณสามารถตั้งค่าส่วนลด (เปอร์เซ็นต์หรือจำนวนเงินคงที่) ที่รหัสคูปองนี้จะใช้กับค่าติดตั้งของแผนของคุณได้

![ตัวเลือกสวิตช์ใช้กับการต่ออายุและส่วนลดค่าติดตั้ง](/img/config/discount-code-renewals.png)

**เปิดใช้งาน:** เปิดใช้งานหรือปิดใช้งานรหัสคูปองนี้ด้วยตนเอง

![สวิตช์เปิดใช้งานเพื่อเปิดหรือปิดรหัสส่วนลดด้วยตนเอง](/img/config/discount-code-active.png)

ภายใต้ **ตัวเลือกขั้นสูง** เรามีการตั้งค่าต่อไปนี้:

![ตัวเลือกขั้นสูงของรหัสส่วนลด](/img/config/discount-code-advanced.png)

**จำกัดการใช้งาน:**

  * **การใช้งาน:** ที่นี่ คุณสามารถดูได้ว่ารหัสส่วนลดถูกใช้ไปกี่ครั้งแล้ว

  * **จำนวนการใช้งานสูงสุด:** สิ่งนี้จะจำกัดจำนวนครั้งที่ผู้ใช้สามารถใช้รหัสส่วนลดนี้ได้ ตัวอย่างเช่น หากคุณใส่ 10 ที่นี่ คูปองจะใช้ได้เพียง 10 ครั้งเท่านั้น หลังจากถึงขีดจำกัดนี้แล้ว จะไม่สามารถใช้รหัสคูปองได้อีก

![การตั้งค่าจำกัดการใช้งานพร้อมจำนวนการใช้งานปัจจุบันและช่องจำนวนการใช้งานสูงสุด](/img/config/discount-code-limit-uses.png)

**วันที่เริ่มต้นและวันหมดอายุ:** ที่นี่คุณจะมีตัวเลือกในการเพิ่มวันที่เริ่มต้นและ/หรือวันหมดอายุให้กับคูปองของคุณ

![ช่องวันที่เริ่มต้นและวันหมดอายุสำหรับกำหนดเวลารหัสส่วนลด](/img/config/discount-code-dates.png)

**จำกัดผลิตภัณฑ์:** หากคุณเปิด **เลือกผลิตภัณฑ์** ผลิตภัณฑ์ทั้งหมดของคุณจะแสดงให้คุณเห็น คุณจะมีตัวเลือกในการเลือกด้วยตนเอง (โดยเปิดหรือปิดสวิตช์) ว่าผลิตภัณฑ์ใดสามารถรับรหัสคูปองนี้ได้ ผลิตภัณฑ์ที่ถูกปิดไว้ที่นี่จะไม่แสดงการเปลี่ยนแปลงใด ๆ หากลูกค้าของคุณพยายามใช้รหัสคูปองนี้กับผลิตภัณฑ์เหล่านั้น

![ส่วนจำกัดผลิตภัณฑ์พร้อมสวิตช์เปิดปิดแยกตามผลิตภัณฑ์](/img/config/discount-code-limit-products.png)

หลังจากตั้งค่าตัวเลือกทั้งหมดเหล่านี้แล้ว ให้คลิกที่ **บันทึกรหัสส่วนลด** เพื่อบันทึกคูปองของคุณ เท่านี้ก็เสร็จเรียบร้อย!

![ปุ่มบันทึกรหัสส่วนลดที่ด้านล่างของหน้าแก้ไข](/img/config/discount-code-save.png)

ตอนนี้คูปองอยู่ในรายการของคุณแล้ว และจากที่นั่น คุณสามารถคลิกเพื่อ **แก้ไขหรือลบ** ได้

![แถวรหัสส่วนลดในรายการพร้อมการกระทำแก้ไขและลบเมื่อเลื่อนเมาส์](/img/config/discount-codes-list-hover.png)

### การใช้พารามิเตอร์ URL:

หากคุณต้องการปรับแต่งตารางราคาของคุณ หรือสร้างหน้ารหัสคูปองที่สวยงามสำหรับเว็บไซต์ของคุณ และต้องการใช้รหัสส่วนลดกับแบบฟอร์มชำระเงินของคุณโดยอัตโนมัติ คุณสามารถทำได้ผ่านพารามิเตอร์ URL

ก่อนอื่น คุณต้องรับลิงก์ที่แชร์ได้สำหรับแผนของคุณ เมื่อต้องการทำเช่นนี้ ให้ไปที่ **Ultimate Multisite > ผลิตภัณฑ์** และเลือกแผน

คลิกที่ปุ่ม **คลิกเพื่อคัดลอกลิงก์ที่แชร์ได้** ซึ่งจะให้ลิงก์ที่แชร์ได้สำหรับแผนเฉพาะนี้แก่คุณ ในกรณีของเรา ลิงก์ที่แชร์ได้ที่ได้รับคือ [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/)_._

![หน้าผลิตภัณฑ์พร้อมปุ่มลิงก์ที่แชร์ได้](/img/config/products-list.png)

หากต้องการใช้รหัสส่วนลดของคุณกับแผนเฉพาะนี้ เพียงเพิ่มพารามิเตอร์ **?discount_code=XXX** ไปยัง URL โดยที่ **XXX** คือรหัสคูปอง

ในตัวอย่างของเราที่นี่ เราจะใช้รหัสคูปอง **50OFF** กับผลิตภัณฑ์เฉพาะนี้

URL สำหรับแผนเฉพาะนี้และเมื่อใช้รหัสส่วนลด 50OFF แล้วจะมีลักษณะดังนี้: [_**mynetworkdomain.com/register/premium/**_](http://mynetworkdomain.com/register/premium/) _**?discount_code=50OFF**_
