# การค้นหาบนอินเทอร์เน็ต

Gratis AI Agent v1.5.0 เพิ่มความสามารถ **การค้นหาบนอินเทอร์เน็ต** ซึ่งช่วยให้ผู้ช่วย AI สามารถดึงข้อมูลล่าสุดจากเว็บได้ระหว่างการสนทนา ความสามารถนี้ขับเคลื่อนโดย [Brave Search API](https://brave.com/search/api/) หรือ [Tavily API](https://tavily.com/)

## วิธีการทำงาน {#how-it-works}

เมื่อเปิดใช้งานการค้นหาบนอินเทอร์เน็ต ผู้ช่วยจะสามารถสอบถามผู้ให้บริการค้นหาที่คุณตั้งค่าไว้โดยอัตโนมัติ เมื่อมันพิจารณาว่าคำถามนั้นต้องการข้อมูลปัจจุบันหรือข้อมูลภายนอก — เช่น ข่าวล่าสุด ราคาแบบเรียลไทม์ หรือเอกสารที่อาจมีการเปลี่ยนแปลงไปจากข้อมูลที่ใช้ในการฝึกฝนของโมเดล

ผลลัพธ์จะถูกดึงมาแบบเรียลไทม์และถูกนำไปใส่ในบริบทของผู้ช่วยก่อนที่มันจะสร้างคำตอบ ผู้ช่วยจะแจ้งให้ทราบเมื่อมีการใช้ผลการค้นหาเพื่อตอบคำถาม

## การเปิดใช้งานการค้นหาบนอินเทอร์เน็ต {#enabling-internet-search}

การค้นหาบนอินเทอร์เน็ตจำเป็นต้องใช้คีย์ API จากผู้ให้บริการค้นหาที่คุณเลือก ในการตั้งค่า:

1. ไปที่ **Gratis AI Agent → Settings → Advanced**
2. ค้นหาเมนูแบบเลื่อนลง **Internet Search Provider** และเลือก **Brave Search** หรือ **Tavily**
3. ป้อนคีย์ API ของคุณในช่องที่เกี่ยวข้อง มีลิงก์สำหรับลงทะเบียนแสดงอยู่ถัดจากแต่ละช่อง
4. คลิก **Save Settings**

เมื่อบันทึกคีย์แล้ว ความสามารถในการค้นหาบนอินเทอร์เน็ตก็จะพร้อมใช้งานสำหรับผู้ช่วยโดยอัตโนมัติ

## Brave Search {#brave-search}

### การรับคีย์ API ของ Brave Search {#obtaining-a-brave-search-api-key}

1. ไปที่ [Brave Search API page](https://brave.com/search/api/)
2. ลงทะเบียนแผนบริการ มีระดับฟรีให้ใช้งานพร้อมข้อจำกัดการร้องขอรายเดือน
3. คัดลอกคีย์ API ของคุณจากหน้า dashboard ของ Brave Search developer
4. วางคีย์ลงในช่อง **Brave Search API Key** ในการตั้งค่า Gratis AI Agent

### ข้อจำกัดการใช้งาน {#usage-limits}

Brave Search จะคิดค่าบริการตามจำนวนการสอบถามที่ทำไป การตอบกลับของ AI แต่ละครั้งที่กระตุ้นให้เกิดการค้นหาจะนับเป็น 1 การสอบถาม โปรดตรวจสอบการใช้งานของคุณใน [Brave Search developer dashboard](https://brave.com/search/api/) เพื่อหลีกเลี่ยงค่าใช้จ่ายที่ไม่คาดคิด

## Tavily {#tavily}

Superdav AI Agent v1.10.0 เพิ่ม **Tavily** เป็นผู้ให้บริการค้นหาบนอินเทอร์เน็ตทางเลือก ซึ่งให้ผลการค้นหาที่ละเอียดกว่าและความสามารถในการวิจัยขั้นสูง

### การรับคีย์ API ของ Tavily {#obtaining-a-tavily-api-key}

1. ไปที่ [Tavily API page](https://tavily.com/)
2. ลงทะเบียนบัญชี มีระดับฟรีให้ใช้งานพร้อมข้อจำกัดการร้องขอรายเดือน
3. คัดลอกคีย์ API ของคุณจาก Tavily dashboard
4. วางคีย์ลงในช่อง **Tavily API Key** ในการตั้งค่า Gratis AI Agent

### ข้อจำกัดการใช้งาน {#usage-limits-1}

Tavily จะคิดค่าบริการตามจำนวนการเรียกใช้ API การตอบกลับของ AI แต่ละครั้งที่กระตุ้นให้เกิดการค้นหาจะนับเป็น 1 การเรียกใช้ โปรดตรวจสอบการใช้งานของคุณใน [Tavily dashboard](https://tavily.com/) เพื่อหลีกเลี่ยงค่าใช้จ่ายที่ไม่คาดคิด

## การปิดใช้งานการค้นหาบนอินเทอร์เน็ต {#disabling-internet-search}

ลบคีย์ API ออกจากช่องผู้ให้บริการค้นหาที่ใช้งานอยู่แล้วบันทึก การค้นหาบนอินเทอร์เน็ตจะไม่ถูกนำเสนอให้ผู้ช่วยอีกต่อไป

:::note
การค้นหาบนอินเทอร์เน็ตจะเพิ่มความหน่วง (latency) ให้กับการตอบกลับ เนื่องจากผู้ช่วยต้องรอผลการค้นหาก่อนที่จะสร้างคำตอบ สำหรับกรณีที่ต้องการความรวดเร็ว ควรพิจารณาว่าจำเป็นต้องมีการค้นหาแบบเรียลไทม์หรือไม่ หรือความรู้ที่มีอยู่ภายในของผู้ช่วยก็เพียงพอแล้ว
:::
