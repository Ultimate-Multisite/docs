# การจัดการลิงก์แชร์สำหรับแผนบริการ

Ultimate Multisite มีตัวเลือกให้คุณแชร์ลิงก์แผนบริการโดยตรงกับลูกค้าของคุณได้ วิธีการคือ ไปที่หน้าแก้ไขแผนบริการใดก็ได้ แล้วคลิกปุ่ม _Click to copy the Shareable Link_

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_ในหน้าแก้ไขแผนบริการ คุณจะพบปุ่มสำหรับสร้างลิงก์แชร์_

โปรดทราบว่ากลไกนี้ใช้รอบการชำระเงินเริ่มต้นในการสร้างลิงก์ หมายความว่าถ้ารอบการชำระเงินเริ่มต้นของคุณเป็นรายเดือน ในท้ายที่สุดคุณจะได้ลิงก์แบบนี้: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

หากคุณต้องการให้ลิงก์แชร์เป็นรอบรายไตรมาสเป็นค่าเริ่มต้น คุณต้องเพิ่มเลข 3 ต่อท้าย

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

หากต้องการเลือกแผน _premium_ แบบรายปี ให้เพิ่มเลข 1 ตามด้วยคำว่า:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

โปรดจำไว้ว่าคุณไม่สามารถใช้ลิงก์เดียวสำหรับรอบการชำระเงินที่แตกต่างกันได้ คุณควรแชร์ลิงก์เฉพาะสำหรับแต่ละรอบการชำระเงินของคุณ

เมื่อคุณแชร์ลิงก์ให้ลูกค้าแล้ว ลิงก์จะพาพวกเขาไปยังขั้นตอนแรกของกระบวนการสมัครสมาชิก
