# บันทึกการเผยแพร่

## เวอร์ชัน 2.13.0 — เผยแพร่เมื่อ 2026-06-05 {#version-2130--released-on-2026-06-05}

- ใหม่: เพิ่มการรองรับ tenant แบบ sovereign สำหรับขั้นตอน customer account, checkout, billing, site, invoice, การสลับ template และ domain mapping เพื่อให้เครือข่าย tenant สามารถนำลูกค้ากลับไปยัง site หลักสำหรับการดำเนินการที่มีการจัดการได้
- ใหม่: เพิ่มการตรวจสอบ credential สำหรับการต่ออายุของ membership แบบเกิดซ้ำ เพื่อให้ gateway สามารถปิดใช้งานการต่ออายุอัตโนมัติเมื่อไม่มี billing agreement, subscription หรือ vault token ที่บันทึกไว้
- ใหม่: เพิ่มการเผยแพร่แบบ loopback ที่ตรวจสอบด้วย HMAC สำหรับการสร้าง site ที่รอดำเนินการ เพื่อให้การจัดเตรียมจาก checkout ไปยัง site น่าเชื่อถือยิ่งขึ้นบนโฮสต์ที่งานเบื้องหลังล่าช้า
- ใหม่: เพิ่มจุดขยายสำหรับนักพัฒนาสำหรับ URL ของ SSO, โดเมนฐานของฟอร์ม checkout และการสร้างระเบียนโดเมนอัตโนมัติ
- แก้ไข: SSO น่าเชื่อถือยิ่งขึ้นในโดเมนที่แมปแล้ว, การเข้าชม broker แบบไม่ระบุตัวตน, การออกจากระบบ และความขัดแย้งของ dependency ระหว่าง plugin
- แก้ไข: การสร้าง site ที่รอดำเนินการตอนนี้กู้คืนจากแฟล็กการเผยแพร่ที่ค้างได้ และหลีกเลี่ยงการปล่อยให้ลูกค้าติดอยู่บนหน้าจอการสร้าง site
- แก้ไข: ไม่มีการสร้างระเบียนโดเมนสำหรับโดเมนฐานของฟอร์ม checkout ที่ใช้ร่วมกันอีกต่อไป และข้ามงานเบื้องหลังของผู้ให้บริการโฮสต์ที่ไม่ได้ใช้เมื่อไม่มี integration ที่ใช้งานอยู่
- แก้ไข: กรณีขอบของ checkout, ที่อยู่ billing, การรีเซ็ตรหัสผ่าน, การยืนยันอีเมล, การสลับ template, ทัวร์ และ dashboard ลูกค้า จะไม่บล็อกขั้นตอนปกติของลูกค้าอีกต่อไป
- แก้ไข: อีเมล broadcast ตอนนี้เก็บผู้รับเป็นส่วนตัว พร้อมหลีกเลี่ยงข้อผิดพลาดร้ายแรงของ SMTP/plugin เมื่อรายการผู้รับหรือระบบส่งเมลล้มเหลว
- แก้ไข: กรณีขอบของการต่ออายุ membership, การแสดงวันหมดอายุ และการเรียกเก็บเงิน ตอนนี้หลีกเลี่ยงการหมดอายุทันที, การแครช หรือการพลาดการชำระเงินที่จำเป็น
- ปรับปรุง: ทดสอบความเข้ากันได้ของ WordPress ถึงเวอร์ชัน 7.0, สร้าง asset ของ Vue สำหรับ production ใหม่จากแหล่ง npm และการครอบคลุม end-to-end ของ Cypress ตอนนี้ทดสอบขั้นตอน checkout, setup, SSO และ gateway มากขึ้น

## เวอร์ชัน 2.12.0 — เผยแพร่เมื่อ 2026-05-15 {#version-2120--released-on-2026-05-15}

- ใหม่: เพิ่ม Hostinger (hPanel) เป็นผู้ให้บริการโฮสต์ที่รองรับ พร้อม integration สำหรับ domain mapping
- ใหม่: Site Exporter ตอนนี้จัดการ bundle การนำเข้าเครือข่ายเพื่อให้การกู้คืน site ทั้งเครือข่ายเป็นไปอย่างคล่องตัว
- แก้ไข: อีเมล broadcast แบบ BCC ตอนนี้ใช้ header แบบ undisclosed-recipients เพื่อป้องกันการเปิดเผยที่อยู่ผู้รับ
- แก้ไข: วันที่หมดอายุของ membership ไม่เสียหายอีกต่อไปเมื่อบันทึกด้วยค่าที่ไม่ใช่วันที่
- แก้ไข: การอัปเดต membership ของ Stripe ตอนนี้ล้างส่วนลดได้อย่างถูกต้องโดยไม่เรียกใช้ API deleteDiscount ที่เลิกใช้แล้ว
- แก้ไข: การ redirect ของ SSO บน site ที่มี domain mapping ตอนนี้ถูกจำกัดเพื่อป้องกัน redirect loop แบบไม่สิ้นสุด
- แก้ไข: การเลือกในตัวเลือกภาพของ setup wizard ตอนนี้อัปเดตโมเดลข้อมูลพื้นฐานได้อย่างถูกต้อง
- แก้ไข: Site Exporter CLI ตอนนี้รักษาการเลือก site เครือข่ายเริ่มต้นที่ถูกต้องไว้
- ปรับปรุง: ลบ wp-cli ที่รวมมาออกจากแพ็กเกจ plugin เพื่อลดขนาด plugin

## เวอร์ชัน 2.11.0 — เผยแพร่เมื่อ 2026-05-11 {#version-2110--released-on-2026-05-11}

- ใหม่: การส่งออก site ตอนนี้รวม `index.php` ที่บูตตัวเองได้ เพื่อให้สามารถติดตั้งไฟล์ ZIP บนโฮสต์ใหม่ได้โดยไม่ต้องติดตั้ง plugin แยกต่างหาก
- ใหม่: การส่งออกเครือข่ายช่วยให้ผู้ดูแลระบบส่งออก subsite ทั้งหมดใน archive เดียวจากหน้าแอดมิน Site Export
- ใหม่: ตัวสลับ plan Allow Site Templates ตอนนี้บังคับใช้ผ่าน fallback chain โดยจำกัดความพร้อมใช้งานของ template ตามขีดจำกัดของ plan ได้อย่างถูกต้อง
- ใหม่: ตัวแก้ไขฟอร์ม checkout เตือนเมื่อเพิ่ม product โดยไม่ได้กำหนด field ที่จำเป็น
- ใหม่: แท็บการตั้งค่า Import/Export ตอนนี้อธิบายขอบเขตของตัวเองอย่างชัดเจนและลิงก์ตรงไปยังเครื่องมือ Site Export

## เวอร์ชัน 2.10.0 — เผยแพร่เมื่อ 2026-05-05 {#version-2100--released-on-2026-05-05}

- แก้ไข: ตรวจสอบความถูกต้องของ input สำหรับ override site ก่อนบันทึก
- แก้ไข: ตอนนี้แสดง prompt ที่อยู่ billing เมื่อที่อยู่ว่าง

- แก้ไข: โหลด Currents แบบ lazy-loaded ก่อน init hook เพื่อป้องกันปัญหา timing
- แก้ไข: ใช้ path ของ SSO ที่ถูก filter ในทุกขั้นตอนการเข้าสู่ระบบ
- แก้ไข: ตัวเลือกตัวตนของ site ที่ว่างถูกเก็บรักษาไว้เมื่อบันทึก

## เวอร์ชัน 2.9.0 — เผยแพร่เมื่อ 2026-04-30 {#version-290--released-on-2026-04-30}

- ใหม่: เพิ่มการส่งออกและนำเข้า single-site ภายใต้ **เครื่องมือ > ส่งออกและนำเข้า**
- แก้ไข: ไฟล์ ZIP ที่ส่งออกตอนนี้ให้บริการผ่าน endpoint ดาวน์โหลดที่ผ่านการยืนยันตัวตน
- แก้ไข: แก้ไขความเสี่ยง SQL injection และปัญหา query ใน query การส่งออก/นำเข้าที่รอดำเนินการ
- แก้ไข: site ที่รอดำเนินการไม่ถูกเผยแพร่เมื่อแอดมินยืนยันอีเมลลูกค้าด้วยตนเอง
- แก้ไข: ล้างระเบียน pending_site ที่กำพร้าเมื่อไม่มี membership
- แก้ไข: แก้ไข padding ของ nav การตั้งค่าและการนำทาง anchor การค้นหา
- แก้ไข: site ที่รอดำเนินการตอนนี้แสดงก่อนในมุมมอง All Sites
- แก้ไข: เพิ่ม header User-Agent ของผู้ให้บริการภาพหน้าจอ (mShots) เพื่อป้องกันข้อผิดพลาด 403
- แก้ไข: แก้ไข dependency วงกลมของตารางเวลา cron สำหรับการนำเข้า
- แก้ไข: ทำให้ Tour IDs เป็นรูปแบบ underscore ใน keys การตั้งค่าผู้ใช้
- ปรับปรุง: ตอนนี้ใช้ ZipArchive แทน Alchemy/Zippy เพื่อความเข้ากันได้ที่ดียิ่งขึ้น
