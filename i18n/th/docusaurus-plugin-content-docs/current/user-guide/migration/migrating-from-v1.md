# การย้ายข้อมูลจาก V1

## Ultimate Multisite ได้เปลี่ยนจากเวอร์ชัน 1.x มาสู่เวอร์ชัน 2.x แล้ว

Ultimate Multisite เวอร์ชัน 2.0 ขึ้นไปเป็นการเขียนโค้ดใหม่ทั้งหมด หมายความว่าเวอร์ชันเก่าและเวอร์ชันใหม่แทบไม่มีส่วนที่เหมือนกันเลย ด้วยเหตุนี้ เมื่ออัปเกรดจาก 1.x เป็น 2.x ข้อมูลของคุณจะต้องถูกย้ายไปยังรูปแบบที่เวอร์ชันใหม่สามารถอ่านได้

โชคดีที่ Ultimate Multisite 2.0+ **มีตัวย้ายข้อมูลในตัว** ที่สามารถตรวจจับข้อมูลจากเวอร์ชันเก่าและแปลงเป็นรูปแบบใหม่ได้ การย้ายข้อมูลนี้จะเกิดขึ้นระหว่าง **ตัวช่วยตั้งค่า** ของเวอร์ชัน 2.0+

บทเรียนนี้จะอธิบายวิธีการทำงานของตัวย้ายข้อมูล สิ่งที่ต้องทำเมื่อเกิดข้อผิดพลาด และวิธีแก้ไขปัญหาที่อาจเกิดขึ้นระหว่างกระบวนการนี้

_**สำคัญ: ก่อนเริ่มอัปเกรดจากเวอร์ชัน 1.x เป็นเวอร์ชัน 2.0 โปรดสร้างข้อมูลสำรองของฐานข้อมูลเว็บไซต์ของคุณก่อน**_

## ขั้นตอนแรก

ขั้นตอนแรกคือดาวน์โหลดไฟล์ .zip ของ plugin และติดตั้งเวอร์ชัน 2.0 บนหน้า dashboard ผู้ดูแลเครือข่ายของคุณ

หลังจากที่คุณ[ติดตั้งและเปิดใช้งานเวอร์ชัน 2.0](../getting-started/installing-ultimate-multisite.md) ระบบจะตรวจจับโดยอัตโนมัติว่า Multisite ของคุณกำลังใช้งานเวอร์ชันเก่าอยู่ และคุณจะเห็นข้อความนี้ที่ด้านบนของหน้า plugin

_**หมายเหตุ:** หากคุณติดตั้ง Ultimate Multisite 1.x บน Multisite ของคุณอยู่แล้ว คุณจะมีตัวเลือกให้แทนที่ plugin ด้วยเวอร์ชันที่คุณเพิ่งดาวน์โหลด กรุณาคลิก **Replace current with uploaded**_

<!-- Screenshot unavailable: Plugin page showing option to replace v1 with uploaded v2 version -->

หน้าถัดไปจะแจ้งให้คุณทราบว่ามี add-on เก่าตัวไหนบ้างที่ติดตั้งอยู่พร้อมกับเวอร์ชัน 1.x พร้อมคำแนะนำว่าเวอร์ชันที่คุณใช้อยู่นั้นเข้ากันได้กับเวอร์ชัน 2.0 หรือไม่ หรือคุณต้องติดตั้ง add-on เวอร์ชันอัปเกรดหลังจากการย้ายข้อมูล

<!-- Screenshot unavailable: Message on the top of the plugins page about updating to v2.0, with list of add-ons that need updating -->

เมื่อคุณพร้อมที่จะดำเนินการต่อ คุณสามารถคลิกปุ่มที่เขียนว่า **Visit the Installer to finish the upgrade**

<!-- Screenshot unavailable: Button saying Visit the Installer to finish the upgrade -->

จากนั้นจะพาคุณไปยังหน้าตัวช่วยติดตั้งพร้อมข้อความต้อนรับ คุณเพียงแค่คลิก **Get Started** เพื่อไปยังหน้าถัดไป

<!-- Screenshot unavailable: Setup Wizard welcoming page with Get Started button -->

หลังจากคลิก **Get Started** จะนำคุณไปยังหน้าตรวจสอบก่อนติดตั้ง ซึ่งจะแสดงข้อมูลระบบและการติดตั้ง WordPress ของคุณ และบอกว่าเป็นไปตาม**ข้อกำหนดของ Ultimate Multisite** หรือไม่

<!-- Screenshot unavailable: Pre-install Checks page showing system requirements confirmation -->

ขั้นตอนถัดไปคือกรอก license key ของ Ultimate Multisite และเปิดใช้งาน plugin เพื่อให้แน่ใจว่าฟีเจอร์ทั้งหมดรวมถึง add-on จะพร้อมใช้งานบนเว็บไซต์ของคุณ

<!-- Screenshot unavailable: License activation page with license key input and Agree and Activate button -->

หลังจากกรอก key แล้ว คลิก **Agree & Activate**

หลังจากเปิดใช้งาน license แล้ว คุณสามารถเริ่มการติดตั้งจริงได้โดยคลิก **Install** ในหน้าถัดไป ซึ่งจะสร้างไฟล์และฐานข้อมูลที่จำเป็นสำหรับการทำงานของเวอร์ชัน 2.0 โดยอัตโนมัติ

## มาถึงขั้นตอนการย้ายข้อมูล

ตัวย้ายข้อมูลมีระบบความปลอดภัยในตัวที่จะตรวจสอบ multisite ทั้งหมดของคุณเพื่อให้แน่ใจว่าข้อมูล Ultimate Multisite ทั้งหมดสามารถย้ายได้โดยไม่มีปัญหา คลิกปุ่ม **Run Check** เพื่อเริ่มกระบวนการ

<!-- Screenshot unavailable: Migration page with Run Check button to verify data can be converted -->

หลังจากตรวจสอบเสร็จ จะมีผลลัพธ์สองแบบ: **มีข้อผิดพลาด** หรือ **ไม่มีข้อผิดพลาด**

### มีข้อผิดพลาด

หากคุณได้รับข้อความแจ้งข้อผิดพลาด คุณจะต้องติดต่อทีมสนับสนุนของเราเพื่อให้ช่วยแก้ไขข้อผิดพลาด อย่าลืม**แนบ log ข้อผิดพลาด** เมื่อสร้างตั๋วสนับสนุน คุณสามารถดาวน์โหลด log หรือคลิกลิงก์ที่เขียนว่า contact our support team ซึ่งจะเปิด widget ช่วยเหลือทางด้านขวาของหน้าจอพร้อมกรอกข้อมูลให้คุณโดยอัตโนมัติ รวมถึง log ข้อผิดพลาดในส่วนคำอธิบาย

_**เนื่องจากระบบพบข้อผิดพลาด คุณจะไม่สามารถดำเนินการย้ายข้อมูลไปยังเวอร์ชัน 2.0 ได้ คุณสามารถย้อนกลับไปใช้เวอร์ชัน 1.x เพื่อให้เครือข่ายของคุณทำงานต่อไปได้จนกว่าข้อผิดพลาดจะได้รับการแก้ไข**_

### ไม่มีข้อผิดพลาด

หากระบบไม่พบข้อผิดพลาดใดๆ คุณจะเห็นข้อความสำเร็จและปุ่ม **Migrate** ที่ด้านล่างซึ่งจะให้คุณดำเนินการย้ายข้อมูลต่อได้ ในหน้านี้จะมีการเตือนให้คุณสร้างข้อมูลสำรองของฐานข้อมูลก่อนดำเนินการต่อ ซึ่งเราแนะนำอย่างยิ่ง คลิก **Migrate** หากคุณมีข้อมูลสำรองแล้ว
