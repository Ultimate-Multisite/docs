# การตั้งค่าการชำระเงินด้วยตนเอง (v2)

_**หมายเหตุสำคัญ: บทความนี้อ้างอิงถึง Ultimate Multisite เวอร์ชัน 2.x**_

การชำระเงินด้วยตนเองเป็นวิธีที่ช่วยให้คุณเสนอช่องทางการชำระเงินอื่นๆ ในกรณีที่ **Stripe** หรือ **PayPal** ไม่พร้อมใช้งานสำหรับผู้ใช้ของคุณ อาจเป็นการโอนเงินผ่านธนาคาร หรือช่องทางการชำระเงินอื่นๆ ที่ผู้ใช้ของคุณสามารถใช้ได้ในท้องถิ่น

## วิธีเปิดใช้งานการชำระเงินด้วยตนเอง

การตั้งค่าการชำระเงินด้วยตนเองทำได้ง่ายมาก คุณเพียงแค่เปิดใช้งานในส่วนของช่องทางการชำระเงิน และใส่คำแนะนำโดยละเอียดว่าผู้ใช้ควรส่งการชำระเงินอย่างไร

ก่อนอื่น ไปที่ **Ultimate Multisite > Settings > Payments** ใต้หัวข้อ **Payment Gateways** ให้เปิด **Manual** คุณจะเห็นกล่อง **Payment Instructions** ปรากฏขึ้นมา

เพิ่มข้อมูลที่ลูกค้าต้องใช้ในการชำระเงินลงในกล่องนี้ เช่น รายละเอียดบัญชีธนาคารและอีเมลของคุณ เพื่อให้ลูกค้าสามารถส่งหลักฐานการชำระเงินให้คุณได้

![ปุ่มเปิดปิดช่องทางการชำระเงินด้วยตนเองและกล่องคำแนะนำการชำระเงิน](/img/config/settings-payment-gateways.png)

หลังจากตั้งค่าเสร็จแล้ว เพียงคลิก **Save Settings** ก็เรียบร้อย เมื่อผู้ใช้ลงทะเบียนเข้าเครือข่ายของคุณ พวกเขาจะเห็นข้อความแจ้งว่าจะได้รับคำแนะนำของคุณเพื่อดำเนินการซื้อให้เสร็จสมบูรณ์

![ข้อความการชำระเงินด้วยตนเองที่แสดงระหว่างการลงทะเบียน](/img/config/settings-payment-gateways.png)

และพวกเขายังจะได้รับข้อความในหน้า **Thank You** พร้อมคำแนะนำการชำระเงินของคุณด้วย

![หน้า Thank You แสดงคำแนะนำการชำระเงิน](/img/config/settings-payment-gateways.png)

## การยืนยันการชำระเงินด้วยตนเอง

หากต้องการยืนยันการชำระเงินด้วยตนเอง ให้ไปที่เมนู **Payments** ในแถบด้านซ้าย ที่นี่คุณจะเห็นการชำระเงินทั้งหมดในเครือข่ายของคุณพร้อมรายละเอียด รวมถึง **สถานะ** การชำระเงินด้วยตนเองจะมีสถานะ **Pending** เสมอ จนกว่าคุณจะเปลี่ยนด้วยตนเอง

![รายการการชำระเงินแสดงการชำระเงินด้วยตนเองที่รอดำเนินการ](/img/admin/payments-list.png)

เข้าสู่หน้ารายละเอียดการชำระเงินโดยคลิกที่ **reference code** ในหน้านี้คุณจะเห็นรายละเอียดทั้งหมดของการชำระเงินที่รอดำเนินการ เช่น reference ID, สินค้า, timestamps และอื่นๆ

![หน้ารายละเอียดการชำระเงินพร้อม reference code และสินค้า](/img/admin/payments-list.png)

ในคอลัมน์ด้านขวา คุณสามารถเปลี่ยนสถานะการชำระเงินได้ การเปลี่ยนเป็น **Completed** และ **เปิดตัวเลือก Activate Membership** จะทำให้เว็บไซต์ของลูกค้าใช้งานได้และสมาชิกภาพของพวกเขาจะเปิดใช้งาน

![การเปลี่ยนสถานะการชำระเงินเป็น Completed พร้อมปุ่มเปิดปิด Activate Membership](/img/admin/payments-list.png)
