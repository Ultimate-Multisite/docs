# การตั้งค่าการชำระเงินแบบ Manual (v2) {#setting-up-manual-payments-v2}

_**หมายเหตุสำคัญ: บทความนี้อ้างอิงถึง Ultimate Multisite เวอร์ชัน 2.x**_

การชำระเงินแบบ Manual เป็นวิธีให้คุณเสนอวิธีชำระเงินอื่น ๆ ในกรณีที่ **Stripe** หรือ **PayPal** ไม่พร้อมใช้งานสำหรับผู้ใช้ของคุณ อาจเป็นการโอนเงินผ่านธนาคาร หรือวิธีชำระเงินอื่นใดที่ผู้ใช้ของคุณสามารถใช้ได้ในพื้นที่ของตน

## วิธีเปิดใช้งานการชำระเงินแบบ Manual {#how-to-enable-manual-payments}

การตั้งค่าการชำระเงินแบบ Manual ทำได้ง่ายมาก คุณเพียงต้องเปิดใช้งานใต้เกตเวย์การชำระเงิน และใส่คำแนะนำโดยละเอียดว่าผู้ใช้ควรส่งการชำระเงินอย่างไร

ก่อนอื่น ไปที่ **Ultimate Multisite > Settings > Payments** ใต้ **Payment Gateways** ให้เปิด **Manual** คุณจะเห็นว่ากล่อง **Payment Instructions** จะปรากฏขึ้นให้คุณ

เพิ่มข้อมูลที่ลูกค้าของคุณต้องใช้เพื่อชำระเงินลงในกล่องนี้ อาจเป็นรายละเอียดบัญชีธนาคารของคุณและอีเมลของคุณ เพื่อให้ลูกค้าสามารถส่งหลักฐานยืนยันการชำระเงินมาให้คุณได้ เป็นต้น

![สวิตช์เกตเวย์การชำระเงินแบบ Manual พร้อมพื้นที่ข้อความ Payment Instructions](/img/config/manual-gateway-expanded.png)

นี่คืออินเทอร์เฟซการตั้งค่าเกตเวย์แบบ Manual:

![การตั้งค่าเกตเวย์แบบ Manual](/img/config/manual-gateway-settings.png)

หลังจากตั้งค่าแล้ว เพียงคลิก **Save Settings** ก็เสร็จเรียบร้อย เมื่อผู้ใช้ลงทะเบียนกับเครือข่ายของคุณ พวกเขาจะเห็นข้อความแจ้งว่าพวกเขาจะได้รับคำแนะนำของคุณเพื่อดำเนินการซื้อให้เสร็จสมบูรณ์

![ข้อความยืนยันการลงทะเบียนที่บอกผู้ใช้ว่าพวกเขาจะได้รับคำแนะนำการชำระเงิน](/img/frontend/registration-manual-notice.png)

และพวกเขาจะได้รับข้อความบนหน้า **Thank You** ของคุณพร้อมคำแนะนำการชำระเงินของคุณด้วย

<!-- Screenshot unavailable: Thank You page showing payment instructions after checkout -->

## การยืนยันการชำระเงินแบบ Manual {#confirming-manual-payments}

หากต้องการยืนยันการชำระเงินแบบ Manual ให้ไปที่เมนู **Payments** บนแถบด้านซ้าย ที่นั่นคุณจะเห็นการชำระเงินทั้งหมดในเครือข่ายของคุณและรายละเอียดต่าง ๆ รวมถึง **สถานะ** ของแต่ละรายการ การชำระเงินแบบ Manual จะมีสถานะ **Pending** เสมอจนกว่าคุณจะเปลี่ยนด้วยตนเอง

![รายการ Payments ที่แสดงการชำระเงินแบบ Manual ที่รอดำเนินการ](/img/admin/payments-list.png)

เข้าสู่หน้าการชำระเงินโดยคลิก **รหัสอ้างอิง** ในหน้านี้คุณจะเห็นรายละเอียดทั้งหมดของการชำระเงินที่รอดำเนินการ เช่น ID อ้างอิง ผลิตภัณฑ์ เวลาประทับ และอื่น ๆ

![หน้ารายละเอียดการชำระเงินที่แสดงรหัสอ้างอิง ผลิตภัณฑ์ และยอดรวม](/img/admin/payment-edit.png)

ในคอลัมน์ด้านขวา คุณสามารถเปลี่ยนสถานะของการชำระเงินได้ การเปลี่ยนเป็น **Completed** และ **เปิดตัวเลือก Activate Membership** จะเปิดใช้งาน site ของลูกค้าของคุณ และ membership ของพวกเขาจะมีสถานะใช้งานอยู่

![หน้าแก้ไขการชำระเงินที่ตั้งค่า Status เป็น Completed และเปิดสวิตช์ Activate Membership](/img/admin/payment-activate-membership.png)
