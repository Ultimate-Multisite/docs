# การจัดการภาษี {#tax-handling}

Ultimate Multisite มีโมดูลการจัดเก็บภาษีใน plugin หลักของเรา ดังนั้นหากคุณต้องการจัดเก็บภาษีการขายสำหรับแผน แพ็กเกจ และบริการของคุณ คุณสามารถทำได้อย่างง่ายดายโดยไม่ต้องติดตั้ง add-ons ใด ๆ

สำหรับบริษัทที่ตั้งอยู่ในยุโรป เรามี **add-on** ที่เพิ่มเครื่องมือและคุณสมบัติเพื่อ **รองรับ VAT** ให้เป็นไปตามข้อกำหนดได้ดียิ่งขึ้น

Ultimate Multisite ไม่ได้ยื่นหรือส่งภาษีแทนคุณให้รัฐบาล เราเพียงช่วยให้คุณจัดเก็บภาษีที่เหมาะสมในขณะทำธุรกรรมเท่านั้น **คุณยังคงต้องนำส่งภาษีด้วยตนเอง**

## การเปิดใช้งานการจัดเก็บภาษี {#enabling-tax-collection}

การจัดเก็บภาษีไม่ได้เปิดใช้งานโดยค่าเริ่มต้น หากต้องการเปิดใช้งาน คุณต้องไปที่ **Ultimate Multisite > Settings > Taxes** แล้วสลับเพื่อเปิดใช้งานการตั้งค่า Enable Taxes

![สวิตช์ Enable Taxes ที่ด้านบนของหน้าการตั้งค่าภาษี](/img/config/settings-taxes-enable.png)

นี่คือมุมมองแบบเต็มของหน้าการตั้งค่าภาษี:

![หน้าการตั้งค่าภาษีแบบเต็ม](/img/config/settings-taxes-full.png)

คุณยังสามารถดูการตั้งค่าภาษีสำหรับผลิตภัณฑ์แต่ละรายการได้:

![การตั้งค่าภาษีสำหรับผลิตภัณฑ์](/img/config/settings-taxes.png)

### ไม่รวมภาษีเทียบกับรวมภาษี {#tax-excluded-vs-tax-included}

โดยค่าเริ่มต้น ราคาผลิตภัณฑ์ทั้งหมดของคุณไม่รวมภาษี ซึ่งหมายความว่าภาษี **ไม่ได้รวมอยู่** ในราคาของผลิตภัณฑ์ หากเราพิจารณาว่าลูกค้าควรชำระภาษีสำหรับการซื้อรายการใดรายการหนึ่ง เราจะเพิ่มภาษี **เพิ่มเข้าไป** จากยอดรวมย่อย

หากคุณต้องการให้รวมภาษีไว้ในราคาของผลิตภัณฑ์ คุณสามารถทำได้โดยเปิดใช้งานการตั้งค่า **Inclusive Tax**

![แถวสวิตช์ Inclusive Tax ใต้การตั้งค่า Enable Taxes](/img/config/settings-taxes-inclusive.png)

อย่าลืม **บันทึก** การเปลี่ยนแปลงที่คุณทำไว้

###

## การสร้างอัตราภาษี {#creating-tax-rates}

หลังจากที่คุณเปิดใช้งานการจัดเก็บภาษีแล้ว คุณจะต้องสร้างอัตราภาษีสำหรับสถานที่เฉพาะโดยใช้ตัวแก้ไขอัตราภาษีของเรา

คุณสามารถเข้าถึงตัวแก้ไขได้โดยคลิกปุ่ม **Manage Tax Rates** บนแถบด้านข้างของหน้าการตั้งค่าภาษี

![ลิงก์ Manage Tax Rates ในแผง Tax Rates บนหน้าการตั้งค่า](/img/config/settings-taxes-manage-rates.png)

ในหน้าตัวแก้ไขอัตราภาษี คุณสามารถเพิ่มอัตราภาษีใหม่ได้โดยคลิกปุ่ม **Add new Row**

![ตารางตัวแก้ไขอัตราภาษีพร้อมปุ่ม Add new Row ที่ด้านบน](/img/config/tax-rates-editor.png)

คุณจะต้องกำหนด **ชื่อ** ให้กับอัตราภาษีแต่ละรายการ (ใช้บนใบแจ้งหนี้) จากนั้นคุณสามารถเลือก **ประเทศ** (จำเป็น), **รัฐ,** และ **เมือง** (ทั้งสองอย่างเป็นทางเลือก) ที่จะเรียกเก็บภาษีนี้ สุดท้าย ให้เพิ่ม **อัตราภาษีเป็นเปอร์เซ็นต์**

### หมวดหมู่ภาษี {#tax-categories}

คุณยังสามารถสร้างหมวดหมู่ภาษีหลายรายการ เพื่อเพิ่มอัตราภาษีที่แตกต่างกันสำหรับผลิตภัณฑ์แต่ละประเภท

คลิกเพื่อ **Add new Tax Category** จากนั้นเขียนชื่อหมวดหมู่ของคุณแล้วกด **Create**

![ปุ่ม Add new Tax Category ที่ด้านบนของตัวแก้ไขอัตราภาษี](/img/config/tax-categories-add.png)

![ช่องกรอก Tax Category Name ใน modal สร้างหมวดหมู่](/img/config/tax-categories-create-modal.png)

หากต้องการเรียกดูหมวดหมู่ ให้คลิก **Switch** แล้วเลือกหมวดหมู่ที่คุณต้องการเพิ่มภาษีใหม่

![ปุ่มดรอปดาวน์ Switch เพื่อเปลี่ยนระหว่างหมวดหมู่ภาษี](/img/config/tax-categories-switch.png)

![ดรอปดาวน์ตัวเลือกหมวดหมู่ภาษีที่แสดงหมวดหมู่ที่มีอยู่](/img/config/tax-categories-select.png)

คุณสามารถตั้งค่าหมวดหมู่ภาษีสำหรับผลิตภัณฑ์เฉพาะได้โดยไปที่ **Product edit page** แล้วไปที่แท็บ Taxes

![แท็บภาษีของผลิตภัณฑ์พร้อมหมวดหมู่ภาษีและสวิตช์ taxable](/img/config/product-taxes.png)

ในหน้าจอเดียวกันนั้น คุณสามารถปิดสวิตช์ **Is Taxable?** เพื่อแจ้งให้ Ultimate Multisite ทราบว่าไม่ควรจัดเก็บภาษีสำหรับผลิตภัณฑ์นั้น

## การรองรับ VAT ของยุโรป {#european-vat-support}

ดังที่กล่าวไว้ก่อนหน้านี้ เรามี add-on สำหรับลูกค้าใน EU ที่มีข้อกำหนดเพิ่มเติมเนื่องจากระเบียบ VAT ของยุโรป

เครื่องมือ VAT ของเราช่วยในเรื่องสำคัญสองสามอย่าง:

  * โหลดอัตรา VAT ของ EU ได้ง่าย;

  * การเก็บและตรวจสอบหมายเลข VAT - และการคิดภาษีแบบย้อนกลับสำหรับนิติบุคคลที่ได้รับยกเว้น VAT (เช่น บริษัทที่มีหมายเลข VAT ที่ถูกต้อง);

หากต้องการติดตั้ง add-on นั้น ให้ไปที่ **Ultimate Multisite > Settings** แล้วคลิกลิงก์แถบด้านข้าง **Check our Add-ons**

![แถบด้านข้างของหน้าการตั้งค่าพร้อมลิงก์ Check our Add-ons](/img/config/settings-taxes-addons-link.png)

คุณจะถูกเปลี่ยนเส้นทางไปยังหน้า add-ons ของเรา ที่นั่น คุณสามารถค้นหา **Ultimate Multisite VAT add-on** และติดตั้งได้

<!-- Screenshot unavailable: VAT add-on tile on the Add-ons page -->

<!-- Screenshot unavailable: VAT add-on Install Now dialog -->

จากนั้น ไปที่ **Network Admin > Plugins** และเปิดใช้งาน add-on นั้นทั่วทั้งเครือข่าย

<!-- Screenshot unavailable: Network Activate action for the VAT add-on on the Plugins page -->

หากคุณกลับไปที่ **Tax Settings tab** คุณจะเห็นตัวเลือกใหม่ที่พร้อมใช้งาน สลับตัวเลือก **Enable VAT Support** เพื่อเปิดใช้งานเครื่องมือ VAT ใหม่ อย่าลืม **บันทึก** การตั้งค่าของคุณ!

<!-- Screenshot unavailable: Enable VAT Support toggle in the Tax settings after add-on activation -->

### การดึงอัตราภาษี VAT {#pulling-on-vat-tax-rates}

หนึ่งในเครื่องมือที่การผสานรวมของเราเพิ่มเข้ามาคือความสามารถในการโหลดอัตราภาษีสำหรับรัฐสมาชิก EU ซึ่งสามารถทำได้โดยไปที่หน้าตัวแก้ไขอัตราภาษีหลังจากเปิดใช้งานการรองรับ EU VAT

ที่ด้านล่างของหน้า คุณจะเห็นตัวเลือกการดึง VAT การเลือกประเภทอัตราและคลิกปุ่ม **Update EU VAT Rates** จะดึงข้อมูลและเติมตารางโดยอัตโนมัติด้วยอัตราภาษีสำหรับแต่ละรัฐสมาชิก EU จากนั้นคุณเพียงต้องบันทึกเท่านั้น

![ปุ่ม Update EU VAT Rates ที่ด้านล่างของตัวแก้ไขอัตราภาษี](/img/config/tax-rates-vat-pull.png)

คุณยังสามารถแก้ไขค่าได้หลังจากดึงข้อมูลแล้ว เมื่อต้องการทำ ให้แก้ไขบรรทัดในตารางที่คุณต้องการ แล้วคลิกเพื่อบันทึกค่าใหม่

### การตรวจสอบ VAT {#vat-validation}

เมื่อเปิดใช้งานการรองรับ VAT แล้ว Ultimate Multisite จะเพิ่มช่องเพิ่มเติมลงในแบบฟอร์ม checkout ใต้ช่องที่อยู่สำหรับเรียกเก็บเงิน ช่องนี้จะแสดงเฉพาะสำหรับลูกค้าที่อยู่ใน EU เท่านั้น

<!-- Screenshot unavailable: VAT Number field on the frontend checkout form below the billing address -->

จากนั้น Ultimate Multisite จะตรวจสอบความถูกต้องของหมายเลข VAT และหากผลลัพธ์ระบุว่าถูกต้อง กลไก reverse charge จะถูกนำมาใช้ และอัตราภาษีจะถูกตั้งเป็น 0% สำหรับคำสั่งซื้อนั้น
