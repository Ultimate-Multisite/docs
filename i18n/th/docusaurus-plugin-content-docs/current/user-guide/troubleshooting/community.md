# วิธีเข้าร่วม Ultimate Multisite Community

Ultimate Multisite Community เป็นพื้นที่ที่คุณสามารถพูดคุยกับผู้ใช้ Ultimate Multisite คนอื่นๆ และแบ่งปันเคล็ดลับ ประสบการณ์ และอื่นๆ อีกมากมาย หากต้องการเข้าถึง Ultimate Multisite Community ให้ไปที่ https://community.ultimatemultisite.com/home

ที่หน้าต้อนรับ ให้คลิกที่ลิงก์ "**creating your account here**"

<!-- Screenshot unavailable: Community welcome page with creating your account link -->

คุณจะถูกนำไปยังหน้าลงทะเบียนของเรา ตอนนี้คุณต้องกรอกชื่อผู้ใช้ อีเมล และรหัสผ่าน ทำ Captcha แล้วคลิกปุ่ม **Register**

<!-- Screenshot unavailable: Community register page with username, email, password fields and Captcha -->

หลังจากลงทะเบียนแล้ว ก็ถึงเวลาสร้างโปรไฟล์ของคุณ ในหน้านี้คุณสามารถอัปโหลดรูปภาพ กรอกชื่อเต็ม เขียนประวัติย่อ และอื่นๆ นอกจากนี้ยังสามารถเพิ่มลิงก์โซเชียลมีเดียและตั้งค่าสิทธิ์การเข้าถึงสำหรับบัญชีของคุณได้อีกด้วย

<!-- Screenshot unavailable: Community profile creation page with photo upload and bio fields -->

ตอนนี้คุณอยู่ในช่อง "Introductions" ซึ่งคุณสามารถแนะนำตัวเองและทำความรู้จักกับสมาชิกคนอื่นๆ ได้ คุณสามารถสร้างโพสต์ได้โดยคลิกที่ปุ่ม **New Post**

<!-- Screenshot unavailable: Community Introductions channel with New Post button -->

ช่องต่างๆ ในส่วน Lounge และช่อง "Dev Logs" เปิดให้ทุกคนเข้าถึงได้ อย่างไรก็ตาม คุณจะต้องได้รับสิทธิ์พิเศษเพื่อเข้าถึงช่องอื่นๆ ในชุมชน

<!-- Screenshot unavailable: Community channels showing Lounge section and Dev Logs channel -->

คุณจะได้รับสิทธิ์การเข้าถึงนี้เมื่อคุณให้ license key ของ Ultimate Multisite ที่ยังใช้งานอยู่กับเรา โดยคลิกที่ลิงก์ช่องส่วนตัวใดก็ได้ แล้วคลิกปุ่ม **Unlock**

<!-- Screenshot unavailable: Community private channel with Unlock button -->

คุณจะถูกนำไปยังหน้าปลดล็อกการเข้าถึง กรอก license key ของ Ultimate Multisite แล้วคลิกปุ่ม Unlock Access:

<!-- Screenshot unavailable: Community unlock access page with license key input -->

License ของคุณจะถูกตรวจสอบและคุณจะสามารถเข้าถึงช่องส่วนตัวได้ กระบวนการนี้อาจใช้เวลาถึง 24 ชั่วโมง ดังนั้นให้คลิกลิงก์เพื่อกลับไปยังชุมชนและเพลิดเพลินกับพื้นที่อื่นๆ ในขณะที่เราดำเนินการคำขอของคุณ

<!-- Screenshot unavailable: Community license validation confirmation page -->
