# ปลั๊กอินที่ใช้งานร่วมกันไม่ได้

หลังจากติดตั้งและเปิดใช้งาน Ultimate Multisite คุณอาจพบปัญหาจากความไม่เข้ากันกับปลั๊กอินอื่นๆ ที่ยังไม่ได้รับการแก้ไข

เรากำลังพยายามลดรายการนี้ให้น้อยลงในทุกๆ เวอร์ชันใหม่ที่ออกมา แต่หากปลั๊กอินใดอยู่ในรายการนี้ คุณอาจต้องปิดการใช้งานเพื่อให้ Ultimate Multisite Network ทำงานได้อย่างสมบูรณ์

นี่คือรายการปลั๊กอินที่มีปัญหาและสิ่งที่คุณควรทำในแต่ละกรณี:

**ปลั๊กอินบางตัวจาก WPMU DEV** ปลั๊กอินบางตัวที่ WPMU DEV _ไม่ได้ดูแล_ แล้ว อาจทำให้ขั้นตอนการลงทะเบียนของ Ultimate Multisite หยุดค้างในขั้นตอนสุดท้าย (ผู้ใช้ไม่สามารถสร้างบัญชีได้) นี่คือรายการปลั๊กอินที่ทำให้เกิดปัญหาเหล่านี้:

  * Signup Language: _Ultimate Multisite มี add-on ที่ให้ฟังก์ชันเดียวกัน ชื่อว่า Ultimate Multisite: Language Switcher_

  * Set Blog Description on Blog Creation: _ฟีเจอร์นี้มีอยู่ใน Ultimate Multisite แล้ว จึงไม่จำเป็นต้องใช้ปลั๊กอินเพิ่มเติม_

  * New Blog Templates: _ฟีเจอร์นี้มีอยู่ใน Ultimate Multisite แล้ว จึงไม่จำเป็นต้องใช้ปลั๊กอินเพิ่มเติม_

  * Set Password on Multisite Blog Creation: _ฟีเจอร์นี้มีอยู่ใน Ultimate Multisite แล้ว จึงไม่จำเป็นต้องใช้ปลั๊กอินเพิ่มเติม_

  * Signup TOS: _ฟีเจอร์นี้มีอยู่ใน Ultimate Multisite แล้ว จึงไม่จำเป็นต้องใช้ปลั๊กอินเพิ่มเติม_

  * Anti-Splog: แม้จะมีความสามารถน้อยกว่า แต่_ฟีเจอร์นี้ก็มีอยู่ใน Ultimate Multisite เช่นกัน_

**หมายเหตุ** : ปลั๊กอินอื่นๆ จาก WPMU Dev สามารถใช้งานร่วมกับ Ultimate Multisite ได้ตามปกติ ตัวอย่างเช่น _Smush_ , _Forminator_ , _Defender_ เป็นต้น
