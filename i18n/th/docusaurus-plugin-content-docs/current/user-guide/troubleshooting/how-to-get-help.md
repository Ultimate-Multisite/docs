# วิธีขอความช่วยเหลือ

เมื่อคุณได้รับ License key ที่ถูกต้องแล้ว คุณจะสามารถติดต่อทีมงานของเราเพื่อขอความช่วยเหลือในการแก้ไขปัญหาต่างๆ ที่อาจพบระหว่างการตั้งค่า Ultimate Multisite ได้

บทเรียนนี้จะอธิบายช่องทางการสนับสนุนที่มีให้สำหรับผู้ถือ license รวมถึงข้อมูลที่คุณต้องเตรียมไว้เมื่อต้องการติดต่อทีมสนับสนุนของเรา

## เว็บไซต์ช่วยเหลือ

คุณสามารถเข้าถึงบทความและบทแนะนำต่างๆ ได้จากเว็บไซต์ช่วยเหลือของเราตามลิงก์ด้านล่าง

[)

เราแนะนำให้ผู้ใช้ทำความคุ้นเคยกับฟีเจอร์และฟังก์ชันพื้นฐานของ Ultimate Multisite บทความ "เริ่มต้นใช้งาน" มีให้อ่านบนเว็บไซต์ช่วยเหลือของเรา และคุณยังสามารถค้นหาคำตอบสำหรับคำถามที่พบบ่อยได้ที่นั่นเช่นกัน เช่น วิธีติดตั้ง add-on วิธีเชื่อมต่อ payment gateway เป็นต้น

## ติดต่อฝ่ายสนับสนุน

หากคุณติดปัญหาและไม่พบคำตอบจากเว็บไซต์ช่วยเหลือของเรา คุณสามารถติดต่อทีมสนับสนุนได้โดยคลิกไอคอนกรอบข้อความที่มุมขวาล่างของเว็บไซต์ หรือส่งอีเมลโดยตรงไปที่ support@ultimatemultisite.com

เมื่อติดต่อฝ่ายสนับสนุนเกี่ยวกับปัญหาทางเทคนิค กรุณาอธิบายปัญหาของคุณอย่างละเอียด พร้อมแนบภาพหน้าจอ ข้อความ error หรือข้อมูลใดๆ ที่จะช่วยให้ทีมสนับสนุนเข้าใจและประเมินปัญหาของคุณได้ง่ายขึ้น คุณต้องแจ้ง license key ของคุณด้วย เฉพาะผู้ถือ license ที่ยังใช้งานอยู่เท่านั้นที่มีสิทธิ์ได้รับการสนับสนุน

## เว็บไซต์ชุมชน

[เว็บไซต์ชุมชน](https://community.ultimatemultisite.com/) ของเราเป็นพื้นที่ให้ผู้ใช้ Ultimate Multisite ได้พูดคุยแลกเปลี่ยนกัน ที่นี่เป็นแหล่งข้อมูลที่สมาชิกสามารถโพสต์ แสดงความคิดเห็นในหัวข้อสนทนา ให้คำแนะนำ ร่วมมือกัน หรือขอความช่วยเหลือจากกลุ่มได้ คุณจะต้องใช้ license key ของ Ultimate Multisite เพื่อเข้าถึงพื้นที่/หน้าส่วนตัว
