# Hook และ Filter ของ Ultimate Multisite

Ultimate Multisite ประกอบด้วยฟังก์ชันและการ query ฐานข้อมูลจำนวนมาก มี hook และ filter ที่ช่วยให้คุณสามารถแก้ไขหรือเพิ่มโค้ดใหม่เข้าไปในฟังก์ชันที่มีอยู่ได้

เราได้รวบรวมรายการ hook และ filter เหล่านี้ไว้เป็นแหล่งอ้างอิงสำหรับผู้ใช้ขั้นสูงที่ต้องการปรับแต่งเครือข่ายของตนเพิ่มเติม คุณสามารถเข้าถึงรายการนี้ได้จาก Github โดยคลิกลิงก์ด้านล่าง

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
