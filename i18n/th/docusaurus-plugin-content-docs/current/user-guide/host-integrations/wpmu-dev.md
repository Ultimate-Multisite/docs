# การเชื่อมต่อกับ WPMU DEV

## ภาพรวม
WPMU DEV เป็นแพลตฟอร์ม WordPress ครบวงจรที่ให้บริการ hosting, plugins และบริการต่างๆ สำหรับเว็บไซต์ WordPress การเชื่อมต่อนี้ช่วยให้สามารถซิงค์โดเมนและจัดการใบรับรอง SSL ระหว่าง Ultimate Multisite และ WPMU DEV hosting ได้โดยอัตโนมัติ

## ฟีเจอร์
- ซิงค์โดเมนอัตโนมัติ
- จัดการใบรับรอง SSL
- เพิ่มจำนวนครั้งในการตรวจสอบใบรับรอง SSL

## ความต้องการของระบบ
ระบบจะตรวจจับโดยอัตโนมัติว่าคุณใช้ hosting ของ WPMU DEV หรือไม่ และจะใช้ API ที่มีอยู่ในตัว หากคุณใช้ hosting ของ WPMU DEV ไม่จำเป็นต้องตั้งค่าเพิ่มเติมใดๆ

ระบบจะตรวจสอบการมีอยู่ของค่าคงที่ `WPMUDEV_HOSTING_SITE_ID` ซึ่งจะถูกกำหนดโดยอัตโนมัติเมื่อใช้ hosting ของ WPMU DEV

## คำแนะนำการตั้งค่า

### 1. ตรวจสอบ WPMU DEV Hosting

หากคุณใช้ hosting ของ WPMU DEV ค่าคงที่ที่จำเป็นควรถูกกำหนดไว้แล้ว ตรวจสอบให้แน่ใจว่า:

1. ค่าคงที่ `WPMUDEV_HOSTING_SITE_ID` ถูกกำหนดไว้ในระบบของคุณ
2. คุณมีสมาชิกภาพ WPMU DEV ที่ใช้งานอยู่และสามารถเข้าถึง API ได้

### 2. เปิดใช้งานการเชื่อมต่อ

1. ในหน้าผู้ดูแลระบบ WordPress ไปที่ Ultimate Multisite > Settings
2. ไปที่แท็บ "Domain Mapping"
3. เลื่อนลงไปที่ "Host Integrations"
4. เปิดใช้งานการเชื่อมต่อ WPMU DEV
5. คลิก "Save Changes"

## วิธีการทำงาน

### การซิงค์โดเมน

เมื่อมีการ map โดเมนใน Ultimate Multisite:

1. ระบบจะใช้ WPMU DEV API เพื่อเพิ่มโดเมนไปยังบัญชี hosting ของคุณ
2. ระบบจะเพิ่มโดเมนเวอร์ชัน www โดยอัตโนมัติด้วย
3. WPMU DEV จะจัดการการตั้งค่าโดเมนและออกใบรับรอง SSL ให้

### การจัดการใบรับรอง SSL

การเชื่อมต่อนี้ถูกตั้งค่าให้เพิ่มจำนวนครั้งในการตรวจสอบใบรับรอง SSL สำหรับ WPMU DEV hosting เนื่องจากอาจต้องใช้เวลาในการออกและติดตั้งใบรับรอง SSL โดยค่าเริ่มต้น ระบบจะพยายามตรวจสอบใบรับรอง SSL สูงสุด 10 ครั้ง เทียบกับมาตรฐาน 5 ครั้ง

## หมายเหตุสำคัญ

### การลบโดเมน

ปัจจุบัน WPMU DEV API ไม่มีวิธีการลบโดเมน เมื่อการ map โดเมนถูกลบออกใน Ultimate Multisite โดเมนจะยังคงอยู่ในบัญชี WPMU DEV hosting ของคุณ คุณจะต้องลบออกด้วยตนเองจาก dashboard ของ WPMU DEV hosting หากจำเป็น

### การยืนยันตัวตน API

การเชื่อมต่อนี้ใช้ WPMU DEV API key ที่เก็บไว้ในฐานข้อมูล WordPress ของคุณในตัวเลือก `wpmudev_apikey` ซึ่งจะถูกตั้งค่าโดยอัตโนมัติเมื่อคุณเชื่อมต่อเว็บไซต์กับ WPMU DEV

## การแก้ไขปัญหา

### ปัญหาการเชื่อมต่อ API
- ตรวจสอบว่าเว็บไซต์ของคุณเชื่อมต่อกับ WPMU DEV อย่างถูกต้อง
- ตรวจสอบว่าตัวเลือก `wpmudev_apikey` ถูกตั้งค่าในฐานข้อมูล WordPress ของคุณ
- ตรวจสอบให้แน่ใจว่าสมาชิกภาพ WPMU DEV ของคุณยังใช้งานอยู่

### ปัญหาใบรับรอง SSL
- WPMU DEV อาจใช้เวลาในการออกใบรับรอง SSL (โดยปกติ 5-15 นาที)
- ระบบถูกตั้งค่าให้ตรวจสอบใบรับรอง SSL สูงสุด 10 ครั้ง
- หากใบรับรอง SSL ยังไม่ถูกออกหลังจากพยายามหลายครั้ง ให้ติดต่อฝ่ายสนับสนุนของ WPMU DEV

### โดเมนไม่ถูกเพิ่ม
- ตรวจสอบบันทึกของ Ultimate Multisite เพื่อดูข้อความแสดงข้อผิดพลาด
- ตรวจสอบว่าโดเมนยังไม่ได้ถูกเพิ่มใน WPMU DEV
- ตรวจสอบให้แน่ใจว่าแผน WPMU DEV hosting ของคุณรองรับจำนวนโดเมนที่คุณต้องการเพิ่ม
