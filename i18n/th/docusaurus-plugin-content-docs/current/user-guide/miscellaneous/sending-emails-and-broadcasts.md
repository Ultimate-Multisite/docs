# การส่งอีเมลและการประกาศ (v2)

_**หมายเหตุสำคัญ: บทความนี้อ้างอิงถึง Ultimate Multisite เวอร์ชัน 2.x**_

Ultimate Multisite มีฟีเจอร์ที่ช่วยให้คุณสื่อสารกับลูกค้าได้ โดยการส่งอีเมลไปยังผู้ใช้เฉพาะรายหรือกลุ่มผู้ใช้ รวมถึงการส่งประกาศบนหน้าแดชบอร์ดของผู้ดูแลระบบเพื่อแจ้งข่าวสาร

## เพิ่มประกาศบนแดชบอร์ดของลูกค้าด้วยฟีเจอร์ Broadcasts

ฟีเจอร์ Broadcast ของ Ultimate Multisite ช่วยให้คุณเพิ่ม **admin notices** บนแดชบอร์ดของ subsite ของผู้ใช้ได้

ฟีเจอร์นี้มีประโยชน์มากเมื่อคุณต้องการประกาศเรื่องต่างๆ เช่น การปิดปรับปรุงระบบ หรือเสนอผลิตภัณฑ์และบริการใหม่ให้กับผู้ใช้ที่มีอยู่ ตัวอย่างการแสดงผล admin notice บนแดชบอร์ดของผู้ใช้จะมีลักษณะดังนี้

![Admin notice broadcast ที่แสดงบนแดชบอร์ดของลูกค้า](/img/admin/broadcasts-list.png)

ในการเริ่มต้นสร้าง admin notice ให้ไปที่แดชบอร์ด network admin และภายใต้เมนู **Ultimate Multisite** คุณจะพบตัวเลือก **Broadcasts**

![เมนู Broadcasts ใน Ultimate Multisite admin](/img/admin/broadcasts-list.png)

จากหน้านี้ ให้คลิกปุ่ม **Add Broadcast** ที่ด้านบน

หน้าต่าง Add broadcast จะปรากฏขึ้น ให้คุณเลือกประเภทของ broadcast ที่ต้องการส่ง

เลือก **Message** แล้วคลิกปุ่ม **Next Step**

![หน้าต่าง Add broadcast เมื่อเลือกประเภท Message](/img/admin/broadcasts-list.png)

หน้าต่างถัดไปจะให้คุณเลือก **Target customer** หรือ **Target product** คุณสามารถเลือกผู้ใช้หรือผลิตภัณฑ์ได้มากกว่าหนึ่งรายการ

ในการค้นหาบัญชีผู้ใช้หรือผลิตภัณฑ์ ให้เริ่มพิมพ์คำค้นหาในช่องที่กำหนด

ในช่อง **Message type** คุณสามารถเลือกสีของประกาศได้ ซึ่งจะช่วยเน้นความเร่งด่วนของข้อความ

จากนั้นคลิก **Next Step**

![การเลือก Target customer และ product สำหรับ broadcast](/img/admin/broadcasts-list.png)

หน้าต่างถัดไปคือที่ที่คุณสามารถเริ่มเขียนข้อความ โดยกรอกหัวข้อและเนื้อหาที่ต้องการส่งถึงผู้ใช้

![ตัวแก้ไขหัวข้อและเนื้อหาของ Broadcast](/img/admin/broadcasts-list.png)

หลังจากสร้างข้อความเสร็จแล้ว ให้คลิกปุ่ม **Send**

เพียงเท่านี้ก็เรียบร้อย admin notice จะแสดงบนแดชบอร์ดของผู้ใช้ทันที

## ส่งอีเมลถึงลูกค้า

ฟีเจอร์ Broadcast ของ Ultimate Multisite ช่วยให้คุณส่งอีเมลถึงผู้ใช้ได้ คุณมีตัวเลือกในการส่งอีเมลไปยังผู้ใช้เฉพาะราย หรือกำหนดกลุ่มเป้าหมายตามผลิตภัณฑ์หรือแผนที่พวกเขาสมัครใช้งาน

ในการเริ่มต้น email broadcast ให้ไปที่แดชบอร์ด network admin และภายใต้เมนู Ultimate Multisite คุณจะพบตัวเลือก Broadcast

![หน้า Broadcasts ใน Ultimate Multisite admin](/img/admin/broadcasts-list.png)

จากหน้านี้ ให้คลิกปุ่ม **Add broadcast** ที่ด้านบน

หน้าต่าง Add broadcast จะปรากฏขึ้น ให้คุณเลือกประเภทของ broadcast ที่ต้องการส่ง เลือก **Email** แล้วคลิกปุ่ม **Next Step**

![หน้าต่าง Add broadcast เมื่อเลือกประเภท Email](/img/admin/broadcasts-list.png)

หน้าต่างถัดไปจะให้คุณเลือก **Target customer** หรือ **Target product** คุณสามารถเลือกผู้ใช้หรือผลิตภัณฑ์ได้มากกว่าหนึ่งรายการ

ในการค้นหาบัญชีผู้ใช้หรือผลิตภัณฑ์ ให้เริ่มพิมพ์คำค้นหาในช่องที่กำหนด

เมื่อเลือกกลุ่มเป้าหมายแล้ว ให้คลิก **Next Step**

![การเลือก Target customer และ product สำหรับ email broadcast](/img/admin/broadcasts-list.png)

หน้าต่างถัดไปคือที่ที่คุณสามารถเริ่มเขียนอีเมล โดยกรอกหัวข้อและเนื้อหาที่ต้องการส่งถึงผู้ใช้

![ตัวแก้ไขหัวข้อและเนื้อหาของ Email broadcast](/img/admin/broadcasts-list.png)

หลังจากสร้างข้อความเสร็จแล้ว ให้คลิกปุ่ม **Send**

นี่คือวิธีง่ายๆ ในการส่งอีเมลถึงผู้ใช้ของคุณโดยใช้ฟีเจอร์ broadcast

## อีเมลระบบ

อีเมลระบบใน Ultimate Multisite คือ **การแจ้งเตือนอัตโนมัติ** ที่ระบบส่งหลังจากการดำเนินการบางอย่าง เช่น การลงทะเบียน การชำระเงิน การ mapping โดเมน เป็นต้น อีเมลเหล่านี้สามารถแก้ไขได้จากการตั้งค่า Ultimate Multisite นอกจากนี้ยังมีฟีเจอร์ที่ช่วยให้คุณรีเซ็ตและนำเข้าการตั้งค่าที่มีอยู่จากการติดตั้ง Ultimate Multisite อื่นได้

### การรีเซ็ตและนำเข้า

เวอร์ชันใหม่ของ Ultimate Multisite รวมถึง add-on ต่างๆ สามารถและจะลงทะเบียนอีเมลใหม่เป็นระยะ

เพื่อป้องกันความขัดแย้งและปัญหาต่างๆ **เราจะไม่เพิ่มเทมเพลตอีเมลใหม่เป็น System Emails ในการติดตั้งของคุณโดยอัตโนมัติ** ยกเว้นกรณีที่จำเป็นต่อการทำงานที่ถูกต้องของฟีเจอร์นั้นๆ

อย่างไรก็ตาม super admin และ agent สามารถนำเข้าอีเมลที่ลงทะเบียนใหม่ผ่านเครื่องมือ importer ได้ กระบวนการนี้จะสร้าง system email ใหม่พร้อมเนื้อหาและการกำหนดค่าของเทมเพลตอีเมลใหม่ ทำให้ super admin สามารถแก้ไขตามต้องการหรือใช้งานตามที่เป็นอยู่

#### วิธีนำเข้า system emails

ไปที่หน้า Ultimate Multisite Settings และเลือกแท็บ **Emails**

![แท็บ Emails ในการตั้งค่า Ultimate Multisite](/img/config/settings-emails.png)

จากนั้นที่ sidebar ให้คลิกปุ่ม **Customize System Emails**

![ปุ่ม Customize System Emails บน sidebar](/img/config/settings-emails.png)

ในหน้า System Emails คุณจะเห็นปุ่ม **Reset & Import** ที่ด้านบน การคลิกปุ่มนี้จะเปิดหน้าต่างสำหรับนำเข้าและรีเซ็ต

![ปุ่ม Reset and Import บนหน้า System Emails](/img/config/settings-emails.png)

จากนั้นคุณสามารถเปิดตัวเลือก Import Emails เพื่อดูว่ามี system emails ใดบ้างที่พร้อมนำเข้า

![ตัวเลือก Import Emails แสดง system emails ที่พร้อมนำเข้า](/img/config/settings-emails.png)

#### การรีเซ็ต System Emails

บางครั้งคุณอาจพบว่าการเปลี่ยนแปลงที่ทำกับเทมเพลตอีเมลไม่ตรงกับความต้องการอีกต่อไป และต้องการรีเซ็ตกลับเป็น **ค่าเริ่มต้น**

ในกรณีเช่นนี้ คุณมีสองตัวเลือก: คุณสามารถลบ system email แล้วนำเข้าใหม่ (ตามคำแนะนำด้านบน) ซึ่งจะลบข้อมูลสถิติการส่งและข้อมูลอื่นๆ ทำให้วิธีนี้ไม่ค่อยเหมาะสม

หรือคุณสามารถใช้ **เครื่องมือ Reset & Import** เพื่อรีเซ็ตเทมเพลตอีเมลนั้น

ในการรีเซ็ตเทมเพลตอีเมล ให้ทำตามขั้นตอนด้านบนจนกระทั่งถึงเครื่องมือ Reset & Import จากนั้นเปิดตัวเลือก **Reset** และเลือกอีเมลที่ต้องการรีเซ็ตกลับเป็นเนื้อหาเริ่มต้น

![ตัวเลือก Reset เพื่อคืนค่าเทมเพลตอีเมลเป็นค่าเริ่มต้น](/img/config/settings-emails.png)
