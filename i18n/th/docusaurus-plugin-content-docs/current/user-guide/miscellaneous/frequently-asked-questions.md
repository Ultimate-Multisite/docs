# คำถามที่พบบ่อย

นี่คือ**คำถามที่พบบ่อย**ที่มักถูกถามเข้ามาในการขอความช่วยเหลือ

  * **Ultimate Multisite คืออะไร?**

Ultimate Multisite เป็น plugin สำหรับ WordPress ที่ออกแบบมาสำหรับการติดตั้งแบบ multisite โดยเฉพาะ ซึ่งจะเปลี่ยน WordPress ของคุณให้กลายเป็นเครือข่ายเว็บไซต์ระดับพรีเมียม เหมือนกับ [WordPress.com](https://WordPress.com) ทำให้ลูกค้าสามารถสร้างเว็บไซต์ผ่านการชำระค่าบริการรายเดือน รายไตรมาส หรือรายปี (คุณยังสามารถสร้างแผนฟรีได้ด้วย)

  * **Ultimate Multisite ใช้งานได้โดยไม่ต้องเปิด Multisite ไหม?**

ไม่ได้ Ultimate Multisite ต้องใช้งานร่วมกับ WordPress multisite แบบ subdirectory หรือ subdomain เท่านั้น

  * **ความต้องการของระบบสำหรับ Ultimate Multisite มีอะไรบ้าง?**

Ultimate Multisite ต้องการ:

**เวอร์ชัน WordPress** : v5.3+ (แนะนำ: เวอร์ชันเสถียรล่าสุด)  
**เวอร์ชัน PHP** : 7.4.x (การรองรับ > 8.0 จะมาเร็วๆ นี้)  
**เวอร์ชัน MySQL** : v5+ (แนะนำ: เวอร์ชัน 5.6 ในกรณีที่โฮสติ้งของคุณยังไม่รองรับ 8.0)

_ต้องเปิดใช้งาน Multisite (subdomain หรือ subdirectory) ด้วย_ [วิธีติดตั้ง WordPress Multisite](/user-guide/getting-started/how-to-install-wordpress-multisite)

  * **Ultimate Multisite ใช้งานได้กับ Multisite แบบ Subdirectory ไหม?**

ได้ Ultimate Multisite รองรับทั้งการติดตั้งแบบ subdirectory และ subdomain

  * **Ultimate Multisite ใช้งานได้กับ Multisite แบบ Subdomain ไหม?**

ได้ Ultimate Multisite รองรับทั้งการติดตั้งแบบ subdirectory และ subdomain

  * **ฉันสามารถใช้ Ultimate Multisite เพื่อจำกัดว่าผู้ใช้จะใช้ Plugin และ Theme ใดได้บ้างไหม?**

ได้ ด้วย Ultimate Multisite คุณสามารถเลือกได้ว่า plugin และ theme ใดบ้างที่จะพร้อมใช้งานในแต่ละแผนบนเครือข่ายของคุณ

  * **ฉันสามารถใช้ Ultimate Multisite เพื่อจำกัดจำนวนโพสต์ที่ผู้ใช้สามารถสร้างบนเว็บไซต์ของตนได้ไหม?**

ได้ ด้วย Ultimate Multisite คุณสามารถกำหนดโควต้าโพสต์สำหรับ**ทุกประเภทโพสต์ที่มี**บนเครือข่ายของคุณ คุณยังสามารถจำกัดจำนวนผู้ใช้ที่ผู้ดูแลระบบสามารถเชิญได้ **พื้นที่เก็บข้อมูล** และอื่นๆ อีกมากมาย

  * **Ultimate Multisite รองรับ Site Template ไหม?**

ได้ ด้วย Ultimate Multisite คุณสามารถสร้าง Site Template ได้มากเท่าที่ต้องการ Site Template ไม่ใช่ theme แต่เป็นเว็บไซต์ปกติบนเครือข่ายของคุณ คุณสามารถเปิดใช้งาน plugin, theme และปรับแต่ง Site Template นั้นได้ตามต้องการ หากผู้ใช้เลือก template นั้น เว็บไซต์ของเขาจะถูกสร้างเป็นสำเนาของ template นั้น พร้อมการปรับแต่งทั้งหมด

คุณยังสามารถกำหนด template เฉพาะสำหรับแผนได้ด้วย เมื่อทำเช่นนี้ ผู้ใช้ทุกคนที่สมัครในแผนนั้นจะได้เว็บไซต์ที่ถูกสร้างเป็นสำเนาของ Site Template นั้น

  * **Ultimate Multisite เชื่อมต่อกับ WooCommerce ได้ไหม?**

ได้!

หากคุณต้องการให้ผู้ใช้ในเครือข่ายของคุณสามารถใช้งาน WooCommerce เพื่อตั้งร้านค้าได้ สามารถทำได้โดยใช้ Ultimate Multisite คุณยังสามารถจำกัดจำนวนสินค้าที่ลูกค้าสามารถสร้างได้ในแต่ละระดับแผนได้อีกด้วย

สำหรับการชำระค่าสมาชิกของ Ultimate Multisite คุณสามารถใช้ [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) เพื่อให้ลูกค้า Ultimate Multisite ของคุณชำระเงินผ่านช่องทางการชำระเงินของ WooCommerce บนเว็บไซต์หลักของคุณ วิธีนี้ช่วยให้คุณเข้าถึงคลังช่องทางการชำระเงินขนาดใหญ่ที่พัฒนาขึ้นสำหรับ WooCommerce และใช้เพื่อเรียกเก็บเงินจากลูกค้า Ultimate Multisite ของคุณได้

  * **มีวิธีย้ายการสมัครสมาชิกจาก WPMUDEV Pro Sites ไหม?**

มี ตอนนี้ Ultimate Multisite มี[เครื่องมือที่ช่วยคุณในการย้ายข้อมูล](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/)จากเครือข่าย Pro Sites ของคุณ

  * **ทำไมแท็บ "Allowed Plugins" ว่างเปล่าในหน้าแก้ไขแผน?**

Ultimate Multisite จะให้คุณอนุญาตหรือปฏิเสธการเข้าถึงเฉพาะ plugin ที่ไม่ได้เปิดใช้งานแบบ network-activated บนเครือข่ายของคุณเท่านั้น เมื่อ plugin ถูกเปิดใช้งานแบบ network-activated มันจะถูกโหลดโดยค่าเริ่มต้นบนทุกเว็บไซต์ในเครือข่ายของคุณ

หากคุณไม่เห็น plugin ใดๆ แสดงในแท็บ "Allowed Plugins" ให้ตรวจสอบว่า plugin ที่ต้องการนั้นถูกเปิดใช้งานแบบ network-active หรือไม่

  * **ฉันสามารถใช้ดีไซน์ตาราง Pricing ของตัวเองได้ไหม?**

ได้ ตั้งแต่เวอร์ชัน 1.4.0 เป็นต้นไป Ultimate Multisite รองรับการ override template

  * **Ultimate Multisite ใช้งานได้กับ WP Engine ไหม?**

ได้ Ultimate Multisite ได้รับการสนับสนุนจาก WP Engine มาตลอด แต่ก่อนหน้านี้การทำ domain mapping ต้องการขั้นตอนเพิ่มเติมจากผู้ดูแลระบบ (เพิ่มโดเมนในแผงควบคุม WP Engine) **แต่ไม่จำเป็นอีกต่อไปตั้งแต่เวอร์ชัน 1.6.0** Ultimate Multisite เชื่อมต่อโดยตรงกับ WP Engine API และโดเมนจะถูกเพิ่มไปยังแผงควบคุมโดยอัตโนมัติหลังจากทำ mapping

  * **คุณแนะนำโฮสติ้งเจ้าไหน?**

Ultimate Multisite ควรใช้งานได้กับโฮสติ้งทุกเจ้า ข้อแตกต่างเพียงอย่างเดียวคือบางบริษัทโฮสติ้ง เช่น Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane และ [Runcloud.io](http://Runcloud.io) มีการเชื่อมต่อที่แน่นแฟ้นกว่าสำหรับความสามารถในการทำ domain mapping จึงใช้งานได้ทันที สำหรับโฮสติ้งอื่นๆ คุณอาจต้องตั้งค่า domain mapping ด้วยตนเอง
