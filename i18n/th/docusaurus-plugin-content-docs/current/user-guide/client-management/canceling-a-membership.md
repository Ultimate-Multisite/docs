# การยกเลิกสมาชิกภาพ (v2) {#canceling-a-membership-v2}

_**หมายเหตุสำคัญ: บทความนี้อ้างอิงถึง Ultimate Multisite เวอร์ชัน 2.x**_

การยกเลิกสมาชิกภาพทางฝั่งลูกค้าของคุณนั้นค่อนข้างง่าย

หากผู้ใช้ของคุณต้องการยกเลิกสมาชิกภาพ พวกเขาควรเข้าถึงหน้า ****Account**** ของตนก่อน

ที่นั่น พวกเขาควรคลิกไปที่ **โซนอันตราย** และคลิก **ลบ Account**

![โซนอันตรายในหน้า Account พร้อมปุ่มลบ Account](/img/account-page/danger-zone.png)

โมดูลจะเปิดขึ้นเพื่อขอการยืนยันจากพวกเขา พวกเขาควรเปิดตัวเลือก **ยืนยันการลบ Account** และคลิก **ลบ Account**

![กล่องโต้ตอบยืนยันการลบ Account](/img/account-page/delete-account-confirm.png)

การดำเนินการนี้จะนำพวกเขาออกจากระบบของไซต์ และยกเลิกสมาชิกภาพของพวกเขาบนเครือข่ายของคุณ รวมถึงลบไซต์ของพวกเขาออกจากเครือข่ายของคุณด้วย

![ออกจากระบบหลังจากลบ Account](/img/account-page/logged-out-after-delete.png)

![การยืนยันการยกเลิกสมาชิกภาพ](/img/account-page/membership-canceled.png)

**สำคัญ:** โปรดสังเกตว่าการดำเนินการนี้จะไม่ลบลูกค้าออกจากเครือข่ายของคุณ เนื่องจากลูกค้าหนึ่งรายสามารถมีสมาชิกภาพมากกว่าหนึ่งรายการ หรือไม่มีสมาชิกภาพใดที่เชื่อมโยงกับพวกเขาได้

![ลูกค้ายังคงอยู่ในเครือข่ายหลังจากการยกเลิก](/img/admin/customer-after-cancel.png)

![ลูกค้าที่ไม่มีสมาชิกภาพที่ใช้งานอยู่](/img/admin/memberships-customer-empty.png)
