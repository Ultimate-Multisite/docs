# การยกเลิกการเป็นสมาชิก (v2)

_**หมายเหตุสำคัญ: บทความนี้อ้างอิงถึง Ultimate Multisite เวอร์ชัน 2.x**_

การยกเลิกการเป็นสมาชิกในฝั่งลูกค้าของคุณทำได้ง่ายมาก

หากผู้ใช้ต้องการยกเลิกการเป็นสมาชิก พวกเขาควรเข้าไปที่หน้า **Account** ก่อน

จากนั้นให้ไปที่ **Danger Zone** แล้วคลิก **Delete Account**

![หน้า Account แสดง Danger Zone พร้อมปุ่ม Delete Account](/img/admin/memberships-list.png)

จะมีหน้าต่างปรากฏขึ้นเพื่อขอการยืนยัน พวกเขาควรเปิดตัวเลือก **Confirm Account Deletion** แล้วคลิก **Delete Account**

![หน้าต่างยืนยันการลบบัญชี](/img/admin/memberships-list.png)

การดำเนินการนี้จะทำให้พวกเขาออกจากระบบเว็บไซต์ ยกเลิกการเป็นสมาชิกบนเครือข่ายของคุณ รวมถึงลบเว็บไซต์ของพวกเขาออกจากเครือข่ายของคุณด้วย

![ออกจากระบบหลังจากลบบัญชี](/img/admin/memberships-list.png)

![การยืนยันการยกเลิกการเป็นสมาชิก](/img/admin/memberships-list.png)

**สำคัญ:** โปรดทราบว่าการดำเนินการนี้จะไม่ลบลูกค้าออกจากเครือข่ายของคุณ เนื่องจากลูกค้าหนึ่งคนสามารถมีการเป็นสมาชิกได้มากกว่าหนึ่งรายการ หรือไม่มีการเป็นสมาชิกเลยก็ได้

![ลูกค้ายังคงอยู่ในเครือข่ายหลังจากยกเลิก](/img/admin/memberships-list.png) | ![ลูกค้าที่ไม่มีการเป็นสมาชิกที่ใช้งานอยู่](/img/admin/memberships-list.png)
