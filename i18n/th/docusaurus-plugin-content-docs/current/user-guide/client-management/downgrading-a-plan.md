# การดาวน์เกรดแพลน (v2)

_**หมายเหตุสำคัญ: บทความนี้อ้างอิงถึง Ultimate Multisite เวอร์ชัน 2.x**_

การดาวน์เกรดแพลนหรือการสมัครสมาชิกเป็นสิ่งที่ลูกค้าของคุณอาจทำบ่อย หากมีงบประมาณจำกัด หรือตัดสินใจว่าไม่จำเป็นต้องใช้ทรัพยากรมากในการดูแลไซต์ย่อยของตน

## วิธีดาวน์เกรดแพลน

ลูกค้าของคุณสามารถดาวน์เกรดแพลนได้ทุกเมื่อ โดยเข้าสู่ระบบหน้าแดชบอร์ดผู้ดูแลไซต์ย่อย แล้วคลิก **Change** ใต้หน้าบัญชีของตน

![หน้าบัญชีที่มีปุ่ม Change ใต้การเป็นสมาชิก](/img/admin/memberships-list.png)

เมื่อคลิกปุ่ม **Change** ผู้ใช้/ลูกค้าจะถูกนำไปยังหน้าชำระเงิน ซึ่งสามารถเลือกแพลนที่ต้องการเปลี่ยนการสมัครสมาชิกได้

![หน้าชำระเงินแสดงตัวเลือกแพลนสำหรับการดาวน์เกรด](/img/admin/memberships-list.png)

ในตัวอย่างนี้ เรากำลังดาวน์เกรดแพลนจาก **Premium** เป็น **Free**

ในการดำเนินการต่อ ผู้ใช้เพียงแค่คลิกปุ่ม **Complete Checkout** จากนั้นระบบจะพากลับไปยังหน้าบัญชี พร้อมแสดงข้อความเกี่ยวกับการเปลี่ยนแปลงที่รอดำเนินการสำหรับการเป็นสมาชิก การเปลี่ยนแปลงจะมีผลใน**รอบการเรียกเก็บเงินถัดไป**ของลูกค้า

![หน้าบัญชีแสดงข้อความการเปลี่ยนแปลงการเป็นสมาชิกที่รอดำเนินการ](/img/admin/memberships-list.png)

### สิ่งที่เกิดขึ้นเมื่อผู้ใช้ดาวน์เกรดแพลน

สิ่งสำคัญที่ต้องทราบคือการดาวน์เกรดแพลนจะไม่เปลี่ยนแปลงการตั้งค่าที่มีอยู่ในไซต์ย่อยของผู้ใช้

ระบบจะไม่เปลี่ยนเทมเพลตไซต์โดยอัตโนมัติ เนื่องจากการเปลี่ยนเทมเพลตไซต์จะลบและรีเซ็ตไซต์ย่อยทั้งหมด เพื่อหลีกเลี่ยงการสูญเสียข้อมูลโดยไม่จำเป็น ดังนั้นพื้นที่ดิสก์ ธีม ปลั๊กอิน ฯลฯ จะยังคงอยู่เหมือนเดิม ยกเว้นโพสต์

เราเข้าใจว่าสิ่งที่คุณกังวลหลักๆ คือขีดจำกัดและโควตาที่คุณตั้งไว้ในแต่ละแพลน แต่เราต้องพิจารณาถึงความเสียหายที่อาจเกิดขึ้นกับไซต์ย่อยของผู้ใช้ หากเราลบหรือเปลี่ยนแปลงการตั้งค่าใดๆ

สำหรับโพสต์ที่เกินขีดจำกัดที่ตั้งไว้ในแพลน คุณมี 3 ตัวเลือก: **เก็บโพสต์ไว้ตามเดิม**, **ย้ายโพสต์ไปถังขยะ** หรือ **ย้ายโพสต์เป็นฉบับร่าง** คุณสามารถกำหนดค่านี้ได้ในการตั้งค่า Ultimate Multisite

![ตัวเลือกเมื่อโพสต์เกินขีดจำกัดในการตั้งค่า Ultimate Multisite](/img/config/settings-sites.png)

### สิ่งที่เกิดขึ้นกับการชำระเงิน

ในเวอร์ชัน 2.0 ไม่จำเป็นต้องปรับการชำระเงินในเรื่องของการคิดค่าใช้จ่ายตามสัดส่วนอีกต่อไป

เนื่องจากระบบจะรอให้การเป็นสมาชิกปัจจุบัน**ครบรอบการเรียกเก็บเงินก่อน** แล้วแพลน/การเป็นสมาชิกใหม่จึงจะมีผล จำนวนเงินที่เรียกเก็บสำหรับการเป็นสมาชิกใหม่จะถูกนำไปใช้และเรียกเก็บโดยอัตโนมัติในรอบการเรียกเก็บเงินถัดไป
