# ฉันจะเข้าถึงหน้าบัญชีของฉันครั้งแรกได้อย่างไร?

ยินดีต้อนรับสู่ Ultimate Multisite!

เข้าถึงหน้าบัญชีของคุณผ่านลิงก์ที่คุณได้รับทางอีเมลหลังจากซื้อ plugin ของเรา:

<!-- Screenshot unavailable: Account page email with access link -->

ใช้อีเมลที่คุณให้ไว้ตอนซื้อและรหัสผ่านที่คุณได้รับทางอีเมล

<!-- Screenshot unavailable: Login page with email and password fields -->

เนื่องจากนี่เป็นการเข้าสู่ระบบครั้งแรกของบัญชีนี้ ระบบจะขอให้คุณเปลี่ยนรหัสผ่าน เปลี่ยนรหัสผ่านแล้วคลิกปุ่ม _Save_

<!-- Screenshot unavailable: Change password form with Save button -->

ในหน้าบัญชีของคุณ คุณสามารถจัดการรายละเอียดบัญชีทั้งหมด เช่น การเรียกเก็บเงิน การต่ออายุ การยกเลิก และอื่นๆ อีกมากมาย
