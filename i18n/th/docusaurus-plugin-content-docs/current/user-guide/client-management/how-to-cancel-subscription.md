# ฉันจะยกเลิกการสมัครสมาชิกได้อย่างไร?

หากคุณมีใบอนุญาต Ultimate Multisite คุณสามารถยกเลิกการต่ออายุได้ทุกเมื่อที่ต้องการ เพียงทำตามขั้นตอนด้านล่าง:


<!-- Screenshot unavailable: Account page email with access link -->

ใช้อีเมลที่คุณให้ไว้ตอนซื้อและรหัสผ่านของคุณ

<!-- Screenshot unavailable: Login page with email and password fields -->

นี่คือหน้าจอเริ่มต้นของหน้าบัญชีของคุณ:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

หากต้องการยกเลิกการสมัครสมาชิก ให้ไปที่เมนูด้านซ้ายแล้วเลือก _Renewing & Billing_

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

คลิกลูกศรด้านขวาเพื่อเปิดหน้าต่างด้านข้าง จากนั้นเลือกตัวเลือกที่จะยกเลิก **การต่ออายุอัตโนมัติ**

ระบบจะแสดงข้อความยืนยันให้คุณ

หากคุณแน่ใจว่าต้องการยกเลิกการสมัครสมาชิก ให้คลิกปุ่ม _**Cancel Renewals**_ หลังจากดำเนินการนี้ คุณจะถูกขอให้ตอบแบบสำรวจสั้นๆ

เสร็จแล้ว! การสมัครสมาชิกของคุณจะไม่ถูกต่ออายุโดยอัตโนมัติ

**คุณจะยังมี key ที่ใช้งานได้จนกว่าการสมัครสมาชิกของคุณจะหมดอายุ** หากคุณต้องการเปิดใช้งานการสมัครสมาชิกอีกครั้ง คุณจะต้องดำเนินการด้วยตนเอง
