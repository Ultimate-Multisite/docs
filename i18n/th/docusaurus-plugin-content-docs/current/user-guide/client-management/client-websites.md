# เว็บไซต์ของลูกค้าของคุณ (v2)

หลังจากลงทะเบียนบนเครือข่ายของคุณแล้ว ลูกค้าของคุณสามารถเข้าถึง Dashboard ของตนได้

นอกจากหน้า **Account** ซึ่งพวกเขาสามารถดูข้อมูลสำคัญเกี่ยวกับการชำระเงิน การเป็นสมาชิก โดเมน ข้อจำกัดของแผน และอื่น ๆ แล้ว พวกเขายังสามารถเข้าถึงหน้า **เว็บไซต์ของฉัน** ซึ่งพวกเขาสามารถดูและจัดการเว็บไซต์ทั้งหมดของตนได้ สิ่งที่พวกเขาต้องทำคือคลิกที่ เว็บไซต์ของฉัน บน Dashboard ของตน

![ลิงก์เว็บไซต์ของฉันใน Dashboard ของลูกค้า](/img/account-page/my-sites-link.png)

พวกเขาจะเห็นเว็บไซต์ทั้งหมดของตน พร้อมปุ่ม **จัดการ** อยู่ด้านล่างของแต่ละเว็บไซต์ หากคลิกที่ปุ่มนั้น พวกเขาจะถูกเปลี่ยนเส้นทางไปยัง **Dashboard ของเว็บไซต์เฉพาะนี้**

พวกเขายังสามารถดูได้ว่าเว็บไซต์ใดของตนเป็นเว็บไซต์ **หลัก** สำหรับแผนที่อนุญาตให้มีเว็บไซต์ได้มากกว่าหนึ่งเว็บไซต์

![หน้าเว็บไซต์ของฉันที่แสดงเว็บไซต์ของลูกค้าพร้อมปุ่มจัดการ](/img/account-page/my-sites-page.png)

นี่คือมุมมองเต็มของอินเทอร์เฟซแก้ไขเว็บไซต์:

![อินเทอร์เฟซแก้ไขเว็บไซต์แบบเต็ม](/img/admin/site-edit-full.png)

หากพวกเขาต้องการเปลี่ยนเว็บไซต์หลักของตน พวกเขาต้องไปที่หน้า Account โดยคลิกที่เมนู **Account**

![ลิงก์เมนู Account ใน Dashboard ของลูกค้า](/img/account-page/account-menu.png)

คุณยังสามารถดูและแก้ไขรายละเอียดลูกค้าได้:

![อินเทอร์เฟซแก้ไขลูกค้า](/img/admin/customer-edit.png)

จากนั้น พวกเขาควรคลิกที่ **เปลี่ยนเว็บไซต์เริ่มต้น**

![ปุ่มเปลี่ยนเว็บไซต์เริ่มต้นบนหน้า Account](/img/account-page/change-default-site.png)

หน้าต่างป๊อปอัปจะปรากฏขึ้น คลิกที่รายการดรอปดาวน์และเลือกเว็บไซต์ที่คุณต้องการกำหนดให้เป็นเว็บไซต์หลัก

![การเลือกจากดรอปดาวน์เพื่อเปลี่ยนเว็บไซต์เริ่มต้น](/img/account-page/change-default-site-dropdown.png)

จากนั้น เพียงคลิกที่ **เปลี่ยนเว็บไซต์เริ่มต้น** ก็เสร็จเรียบร้อย!

![การยืนยันการเปลี่ยนเว็บไซต์เริ่มต้น](/img/account-page/change-default-site-confirm.png)
