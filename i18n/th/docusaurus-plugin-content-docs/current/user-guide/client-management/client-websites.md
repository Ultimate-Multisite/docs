# เว็บไซต์ของลูกค้า (v2)

หลังจากสมัครสมาชิกบนเครือข่ายของคุณแล้ว ลูกค้าจะสามารถเข้าถึงแดชบอร์ดของตนเองได้

นอกจากหน้า **Account** ที่ลูกค้าสามารถดูข้อมูลสำคัญเกี่ยวกับการชำระเงิน สมาชิกภาพ โดเมน ข้อจำกัดของแพลน และอื่นๆ แล้ว พวกเขายังสามารถเข้าถึงหน้า **My Sites** เพื่อดูและจัดการเว็บไซต์ทั้งหมดได้อีกด้วย เพียงแค่คลิกที่ My Sites บนแดชบอร์ด

![ลิงก์ My Sites ในแดชบอร์ดของลูกค้า](/img/admin/customers-list.png)

ระบบจะแสดงเว็บไซต์ทั้งหมด พร้อมปุ่ม **Manage** อยู่ด้านล่างแต่ละเว็บไซต์ เมื่อคลิกปุ่มนี้ ลูกค้าจะถูกนำไปยัง **แดชบอร์ดของเว็บไซต์นั้นๆ**

ลูกค้ายังสามารถดูได้ว่าเว็บไซต์ไหนเป็นเว็บไซต์ **หลัก** สำหรับแพลนที่อนุญาตให้มีเว็บไซต์มากกว่าหนึ่งแห่ง

![หน้า My Sites แสดงเว็บไซต์ของลูกค้าพร้อมปุ่ม Manage](/img/admin/customers-list.png)

หากต้องการเปลี่ยนเว็บไซต์หลัก ให้ไปที่หน้า Account โดยคลิกที่เมนู **Account**

![ลิงก์เมนู Account ในแดชบอร์ดของลูกค้า](/img/admin/customers-list.png)

จากนั้นให้คลิกที่ **Change Default Site**

![ปุ่ม Change Default Site ในหน้า Account](/img/admin/customers-list.png)

หน้าต่างป๊อปอัปจะปรากฏขึ้น คลิกที่รายการแบบเลื่อนลงและเลือกเว็บไซต์ที่ต้องการตั้งเป็นเว็บไซต์หลัก

![การเลือก Change Default Site จากรายการแบบเลื่อนลง](/img/admin/customers-list.png)

จากนั้นเพียงคลิก **Change Default Site** เป็นอันเสร็จเรียบร้อย!

![การยืนยันการเปลี่ยนเว็บไซต์หลัก](/img/admin/customers-list.png)
