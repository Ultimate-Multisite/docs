# Action: wu_mcp_adapter_initialized

จะทำงานหลังจากที่ MCP adapter ถูกเริ่มต้นเรียบร้อยแล้ว

ใช้สำหรับให้ปลั๊กอินและธีมอื่น ๆ สามารถลงทะเบียนความสามารถของตัวเองได้

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | อินสแตนซ์ของ MCP adapter |

### Since

- 2.5.0
