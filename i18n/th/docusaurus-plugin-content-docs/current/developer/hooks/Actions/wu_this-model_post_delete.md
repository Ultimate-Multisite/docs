# Action: wu_\{$this->model\}_post_delete

## พารามิเตอร์ {#parameters}

| Name | Type | คำอธิบาย |
|------|------|-------------|
| $result | `bool` | เป็น `True` หากลบวัตถุได้สำเร็จ |
| $this | `\Base_Model` | ตัวแปรที่เก็บ instance ของวัตถุนั้นๆ |
