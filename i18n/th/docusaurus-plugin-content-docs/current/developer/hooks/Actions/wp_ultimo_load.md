# Action: wp_ultimo_load {#action-wpultimoload}

จะทำงานเมื่อมีการโหลด dependencies ทั้งหมดเรียบร้อยแล้ว

ช่วยให้นักพัฒนาปลั๊กอินสามารถเพิ่มฟังก์ชันการทำงานใหม่ ๆ ได้ ตัวอย่างเช่น การรองรับผู้ให้บริการโฮสติ้งรายใหม่ เป็นต้น
