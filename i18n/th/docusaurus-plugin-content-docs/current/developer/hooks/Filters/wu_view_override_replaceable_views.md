# Filter: wu_view_override_replaceable_views

ให้ผู้พัฒนาสามารถเพิ่มโฟลเดอร์อื่น ๆ เข้าไปในรายการที่สามารถแทนที่ได้

โปรดระมัดระวัง เพราะการเพิ่มโฟลเดอร์เพิ่มเติมอาจทำให้ระบบโหลดไฟล์เวอร์ชันเก่าแทนที่จะเป็นเวอร์ชันของ Ultimate Multisite

## พารามิเตอร์

| Name | Type | คำอธิบาย |
|------|------|-------------|
| $replaceable_views | `array` | รายการโฟลเดอร์ที่อนุญาต |
