# Filter: wp_ultimo_redirect_url_after_signup {#filter-wpultimoredirecturlaftersignup}

ใช้สำหรับกำหนด URL ปลายทางที่เปลี่ยนเส้นทาง

นี่คือตัวกรอง (filter) แบบเก่า (legacy filter) พารามิเตอร์บางตัวที่ส่งมาอาจไม่พร้อมใช้งาน เช่น $site_id

### ตั้งแต่ {#since}

- 1.1.3: อนุญาตให้นักพัฒนาสามารถกำหนดค่า URL ปลายทางที่เปลี่ยนเส้นทางได้
