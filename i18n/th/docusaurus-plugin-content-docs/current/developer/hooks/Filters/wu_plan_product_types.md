# Filter: wu_plan_product_types

ใช้สำหรับกรองประเภทสินค้าที่ควรถูกนับว่าเป็นแผน (plans)

Filter นี้ช่วยให้ส่วนเสริม (addons) สามารถลงทะเบียนประเภทสินค้าเพิ่มเติมที่ควรได้รับการยอมรับว่าเป็นแผนในการตรวจสอบความถูกต้องและการจัดกลุ่ม

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | อาร์เรย์ของประเภทสินค้าที่ต้องการให้ถือว่าเป็นแผน |

- 2.3.0
