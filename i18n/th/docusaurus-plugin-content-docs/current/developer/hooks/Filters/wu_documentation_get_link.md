# Filter: wu_documentation_get_link

อนุญาตให้นักพัฒนาปลั๊กอินสามารถกรองลิงก์ได้

ไม่แน่ใจว่าฟังก์ชันนี้จะมีประโยชน์แค่ไหน แต่การมีมันไว้ก็ไม่เสียหายอะไร

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $link | `string` | ลิงก์ที่ลงทะเบียนไว้ |
| $slug | `string` | สลัก (slug) ที่ใช้ในการเรียกดูลิงก์ |
| $default_link | `string` | ลิงก์เริ่มต้นที่ลงทะเบียนไว้ |

### Since {#since}

- 1.7.0
