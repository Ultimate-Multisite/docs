# Filter: send_auth_cookies

ใช้สำหรับป้องกันไม่ให้คุกกี้สำหรับยืนยันตัวตน (auth cookies) ถูกส่งไปยังไคลเอนต์จริง

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $send | `bool` | กำหนดว่าจะส่ง auth cookies ไปยังไคลเอนต์หรือไม่ |

### Since

- 4.7.4
