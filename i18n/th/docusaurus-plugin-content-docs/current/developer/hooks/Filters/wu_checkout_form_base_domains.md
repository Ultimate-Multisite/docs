# Filter: wu_checkout_form_base_domains

กรองโดเมนฐานของแบบฟอร์มชำระเงินที่ใช้ร่วมกัน ซึ่งไม่ควรถูกสร้างเป็นระเบียนโดเมนที่แมปไว้

ใช้ filter นี้เมื่อการผสานรวมให้โดเมนฐานเพิ่มเติมสำหรับฟิลด์ **Site URL** ของแบบฟอร์มชำระเงิน โดเมนที่ส่งคืนโดย filter นี้จะถูกถือว่าเป็นโฮสต์ลงทะเบียนที่ใช้ร่วมกัน แทนที่จะเป็นโดเมนกำหนดเองรายไซต์

## พารามิเตอร์ {#parameters}

| ชื่อ | ประเภท | คำอธิบาย |
|------|------|-------------|
| $domains | `array` | โดเมนฐานที่ใช้ร่วมกันซึ่งรวบรวมจากการกำหนดค่าแบบฟอร์มชำระเงิน |

### ตั้งแต่ {#since}

- 2.13.0

### แหล่งที่มา {#source}

กำหนดไว้ใน `inc/functions/domain.php`


## ค่าที่ส่งคืน {#returns}

อาร์เรย์ของโดเมนฐานของแบบฟอร์มชำระเงินที่ปรับให้อยู่ในรูปแบบมาตรฐานแล้ว
