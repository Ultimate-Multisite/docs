# PRO Theme ব্যৱহাৰ কৰি আপোনাৰ Admin Panel ক কাস্টমাইজ কৰক {#customizing-your-admin-panel-with-pro-theme}

ক্লায়েন্টৰ কাম কৰোঁতে, কেতিয়াবা আমি আমাৰ ক্লায়েন্টলৈ WordPress-ৰ admin dashboard ত এটা কাস্টম অভিজ্ঞতা দিবলগীয়া হয়। যদি আপুনি এটা বেছি প্ৰফেশ্যনেল লুক আৰু ফিল বিচাৰিছে, তেন্তে PRO Theme আপোনাক সহায় কৰিব পাৰে।

PRO Theme হৈছে এটা admin theme, যি আপোনাক আপোনাৰ ক্লায়েন্টৰ Dashboard ক কাস্টমাইজ কৰাত সহায় কৰিবলৈ গঢ়ি তোলা হৈছে। আৰু ই Ultimate Multisite-ৰ premium plugins core-ৰো এটা অংশ।

ইটো install কৰাৰ পিছত, আপুনি আপোনাৰ _Plugins_ পৃষ্ঠাখনলৈ যাওক আৰু আপোনাৰ Network ত ইয়াক activate কৰক।

_**নোট:**_ _যদি আপুনি ইয়াক কেৱল install কৰে, তেন্তে ই আপোনাৰ ক্লায়েন্টৰ বাবে এটা অপচন হিচাপে দেখুৱাব, যাক কেৱল সেই ব্যৱহাৰকাৰীৰ বাবে থিমটো যোগ কৰিবলৈ চিহ্নিত কৰিব পাৰি._

এতিয়া, আপোনাৰ admin interface সম্পূৰ্ণৰূপে redesign কৰা হৈ গ'ল! আপোনাৰ বাঁহফালে (left column) PRO Theme নামৰ এটা নতুন মেনুও আছে।
