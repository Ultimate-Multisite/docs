# প্লেনৰ বাবে Shareable Link ব্যৱস্থাপনা {#managing-shareable-links-for-plans}

Ultimate Multisite-এ আপোনাক আপোনাৰ ক্লায়েন্টসকলক প্লেন লিংক (plan link) পোনপটীয়াকৈ ভাগ কৰাৰ সুবিধা দিয়ে। ইয়াৰ বাবে, আপোনাৰ যিকোনো এটা প্লেনৰ এডিট পেজলৈ (edit page) যাওক আৰু _Click to copy the Shareable Link_ বোৱাৰ্টনটো ক্লিক কৰক।

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_প্লেনৰ এডিট পেজত, আপুনি shareable link বনাৰ বাবে বোৱাৰ্টনটো বিচাৰি পাব_

লক্ষ্য কৰক যে এই ব্যৱস্থাটোৱে লিংকটো বনাবলৈ ডিফল্ট বিলিং ফ্ৰিকৱেঞ্চি (default billing frequency) ব্যৱহাৰ কৰে। ইয়াৰ অৰ্থ হ'ল যে যদি আপোনাৰ ডিফল্ট বিলিং মাহেকীয়া (monthly) হয়, উদাহৰণস্বৰূপে, তেন্তে শেষত আপোনাৰ লিংকটো এনেকুৱা হ'ব: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

যদি আপুনি shareable link-টোকে ট্ৰাইমেষ্ট্ৰেল (quarterly) কৰিব বিচাৰে, তেন্তে আপুনি শেষত '3' যোগ কৰিব লাগিব

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

যদি আপুনি _premium_ প্লেনটো বছৰীয়াকৈ (yearly) নিৰ্বাচন কৰিব বিচাৰে, তেন্তে আপুনি '1' আৰু তাৰ পিছত 'year' শব্দটো যোগ কৰিব লাগিব:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

মনত ৰাখিব যে বিভিন্ন বিলিং ফ্ৰিকৱেঞ্চিৰ বাবে আপুনি কেৱল এটা লিংক ৰাখিব নোৱাৰে। আপুনি আপোনাৰ প্ৰতিটো বিলিং ফ্ৰিকৱেঞ্চিৰ বাবে এটা নিৰ্দিষ্ট লিংক ভাগ কৰিব লাগিব।

এবাৰ যেতিয়া আপুনি লিংকটো আপোনাৰ কাস্টমৰসকলক ভাগ কৰে, তেতিয়া ই তেওঁলোকক আপোনাৰ সাইনআপ ফ্লো (signup flow) ৰ প্ৰথম ধাপলৈ পুনৰ স্থানান্তৰিত কৰে (redirect)।
