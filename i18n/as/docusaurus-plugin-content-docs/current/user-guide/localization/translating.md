# Ultimate Multisite অনুবাদ কৰা {#translating-ultimate-multisite}

Ultimate Multisite ইংৰাজীত বিকশিত কৰা হৈছে, কিন্তু এই plugin-টো সম্পূৰ্ণৰূপে অনুবাদ কৰিব পৰা যায় আৰু আমি অনুবাদৰ বৰঙণি স্বীকাৰ কৰোঁ।

## মূল Plugin অনুবাদ {#core-plugin-translations}

মূল Ultimate Multisite plugin-টো আনুষ্ঠানিক **WordPress.org translation platform**-ৰ জৰিয়তে অনুবাদ কৰা হয়:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

এইটো হৈছে প্ৰচলিত WordPress কমিউনিটি অনুবাদ ব্যৱস্থা। যেতিয়া অনুবাদবোৰ যথেষ্ট অনুমোদনৰ সীমা অতিক্ৰম কৰে, তেতিয়া ইয়াক WordPress plugin update-ৰ জৰিয়তে সকলো Ultimate Multisite ব্যৱহাৰকাৰীৰ ওচৰলৈ স্বয়ংক্রিয়ভাৱে বিতৰণ কৰা হয়।

**আপোনাৰ সহায়ৰ প্ৰয়োজন!** যদি আপুনি ইংৰাজী বাহিৰৰ কোনো ভাষাত কথা ক'ব পাৰে, তেন্তে অনুগ্ৰহ কৰি অনুবাদৰ বৰঙণি দিবলৈ বিবেচনা কৰক। মাত্ৰ কেইটামান বাক্য অনুবাদ কৰাও সহায়ক। আৰম্ভ কৰিবলৈ:

1. [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/) লৈ যোৱা
2. আপোনাৰ ভাষাটো বাছি লওক
3. "Contribute Translation" ক্লিক কৰক
4. আপোনাৰ এটা বিনামূলীয়া WordPress.org account হ'ব লাগিব

যদি আপুনি নিজৰ ভাষাৰ বাবে অনুবাদ এডিটর হ'ব বিচাৰে (যিয়ে অনুবাদ অনুমোদন কৰিব পাৰে), তেন্তে [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) জৰিয়তে যোগাযোগ কৰক।

## Addon অনুবাদ {#addon-translations}

Addon plugin-বোৰ পৃথকভাৱে তলত অনুবাদ কৰা হয়:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Addon অনুবাদবোৰ বৰ্তমান আৰম্ভণিৰ হিচাপে AI দ্বাৰা সৃষ্টি কৰা হৈছে। ই সকলো সমৰ্থন কৰা ভাষাক কওঁ, কিন্তু ইয়াত বেয়া বাক্য গঠন বা ভুল হ'ব পাৰে। আমি মানৱ অনুবাদকসকলক ইয়াক অধিক স্বাভাৱিক আৰু সঠিক কৰিবলৈ পুনৰীক্ষণ কৰিবলৈ আৰু উন্নত কৰিবলৈ অনুৰোধ জনাই।

## স্থানীয় অনুবাদ {#local-translations}

যদি আপুনি ওপৰৰ উৎসত বৰঙণি দিয়াৰ পৰিৱৰ্তে নিজৰ প্ৰজেক্টৰ বাবে স্থানীয়ভাৱে অনুবাদ কৰিবলৈ পছন্দ কৰে, তেন্তে দুটা বিকল্প আছে:

### Poedit ব্যৱহাৰ কৰি {#using-poedit}

আপুনি ডেস্কটপ এপ্লিকেচন **[Poedit](https://poedit.net)** ব্যৱহাৰ কৰি `.pot` ফাইলটো অনুবাদ কৰিব পাৰে:

1. [poedit.net](https://poedit.net) ৰ পৰা Poedit ডাউনলোড কৰক
2. `wp-content/plugins/ultimate-multisite/lang/` ফোল্ডাৰৰ পৰা `.pot` ফাইলটো খুলক
3. Poedit interfes-ত থকা বাক্যবোৰ অনুবাদ কৰক
4. সেভ কৰক -- Poedit ले `.po` আৰু `.mo` ফাইল সৃষ্টি কৰে
5. আউটপুট ফাইলবোৰ `lang` ফোল্ডাৰত থওক

### Loco Translate ব্যৱহাৰ কৰি {#using-loco-translate}

**[Loco Translate](https://wordpress.org/plugins/loco-translate/)** WordPress plugin-টোৱে আপোনাক কোনো ডেস্কটপ টুল নোহোৱাকৈ এডমিন প্যানেলৰ পৰা পোনপটীয়াকৈ অনুবাদ কৰিবলৈ দিয়ে। ই আপোনাৰ বাবে অনুবাদ ফাইল সৃষ্টি কৰা আৰু সংকলন কৰা ব্যৱস্থা কৰে।
