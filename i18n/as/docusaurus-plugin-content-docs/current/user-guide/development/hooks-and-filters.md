# Ultimate Multisite Hooks and filters

Ultimate Multisite বহুতো ফাংশন আৰু ডেটাবেছ কোৱাৰীৰে গঠিত। ইয়াত hooks আৰু filters আছে, যাৰ সহায়ত আপুনি থকা ফাংশনসমূহক সলনি কৰিব বা নতুন কোড যোগ কৰিব পাৰে।

আমি উন্নত ব্যৱহাৰকাৰীসকলক তেওঁলোকৰ নেটৱৰ্কক আৰু অধিক কাস্টমাইজ কৰাত সহায় কৰিবলৈ এই hooks আৰু filters-ৰ এটা তালিকা तयार কৰিছো। তলৰ লিংকত ক্লিক কৰি আপুনি সেই তালিকাখন Github बाट অ্যাক্সেছ কৰিব পাৰিব।
