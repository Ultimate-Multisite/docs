# Dashboard Overview {#dashboard-overview}

Ultimate Multisite-ৰ Ultimate Multisite মেনুৰ তলত এটা নিৰ্দিষ্ট **Dashboard** থাকে, আৰু ই Network Dashboard তও কিছুমান widget যোগ কৰে। এই article-টোৱে আপুনি দুয়োটা ঠাইতে কি কি দেখিব পাৰিব, তাৰ বিষয়ে কওঁ।

## The Network Dashboard {#the-network-dashboard}

Ultimate Multisite-এ **My Sites → Network Admin → Dashboard** ত পোৱা network-level dashboard ত ব্যৱহাৰিক widget যোগ কৰে।

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

প্ৰতিটো widget কি কি তথ্য দিয়ে, তাৰ তলত দিয়া হ'ল:

**First Steps:** নতুনকৈ install কৰাৰ পিছত এই widget-টো দেখিবলৈ পোৱা যায় আৰু ই আপোনাক পেমেন্ট পদ্ধতি কনফিগাৰ কৰা বা এটা টেস্টিং একাউণ্ট সৃষ্টি কৰাৰ দৰে সেটআপ কৰিবলগীয়া কামসমূহ সম্পন্ন কৰিবলৈ মনত পেণ কৰায়।

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Summary:** ই দিনটোৰ সাইনআপ আৰু আয়ৰ সংখ্যা প্ৰতিবেদন কৰে। MRR (Monthly Recurring Revenue) মানে হৈছে আয়-উৎপাদনশীল membership থকা ग्राहकসকলৰ বাবে এটা অনুমান কৰা মুঠ পৰিমাণ।

![Summary widget](/img/admin/network-dashboard-summary.png)

**Activity Stream:** Ultimate Multisite ত হোৱা বিভিন্ন ঘটনা — সাইনআপ, বাতিল কৰা, site সৃষ্টি, মুছা, আৰু অন্যান্য কাৰ্যকলাপৰ বিষয়ে প্ৰতিবেদন দিয়ে।

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** ই network-ত থকা ব্যৱহাৰকাৰী আৰু site-সমূহৰ এটা সংক্ষিপ্ত সামৰণি দেখুৱায়, আৰু নতুন site বা user সৃষ্টি কৰাৰ বাবে shortcut দিয়ে।

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** ই আপডেট, গুৰুত্বপূৰ্ণ bug, আৰু security fix সহ Ultimate Multisite-ৰ শেহতীয়া খবৰসমূহ প্ৰাপ্ত কৰি প্ৰদৰ্শন কৰে।

![News widget](/img/admin/network-dashboard-news.png)

## The Ultimate Multisite Dashboard {#the-ultimate-multisite-dashboard}

যিহেতু network dashboard-এ network সম্পৰ্কে এটা সামগ্ৰিক তথ্য দিয়ে, সেয়ে Ultimate Multisite dashboard (Ultimate Multisite মেনুৰ ওপৰত স্তৰত) দিয়ে সেৱিকাটোৰ বিষয়ে তথ্য।

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Monthly Recurring Revenue Growth:** ই এটা ১২-মাহৰ calendar period ত আয়ৰ মাসিক বিভাজন দেখুৱায়। নতুন আয়ক নীলা ৰঙত আৰু বাতিল হোৱা আয়ক ৰঙা ৰঙত দেখুওৱা হয়।

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**Date-range Support:** আপুনি যি তথ্যৰ ওপৰত গুৰুত্ব দিব বিচাৰে, তাৰ বাবে dashboard data-ক সময়ৰ সীমা অনুসৰি ফিল্টাৰ কৰিব পাৰে। এই date range-এ তলৰ widget-সমূহত কি দেখুৱাব লাগিব, সেয়া নিৰ্ধাৰণ কৰে।

![Date range and dashboard stats](/img/admin/um-dashboard-date-range.png)

dashboard widget-সমূহৰ ভিতৰত আছে:

- **Revenue** — MRR আৰু প্ৰডাক্ট-ভিত্তিক বিভাজন সহ সময়ছোৱাৰ মুঠ আয়
- **New Memberships** — নিৰ্বাচিত সময়ছোৱাত কেইটা নতুন membership সৃষ্টি কৰা হৈছে
- **Signups by Countries** — সাইনআপৰ ভৌগোলিক বিভাজন
- **Signups by Forms** — কিমান ग्राहक প্ৰতিটো checkout form-ৰ জৰিয়তে সাইনআপ কৰিলে বনাম admin-দ্বাই যোগ কৰিলে
- **Most Visited Sites** — আপোনাৰ network-ত আটাইতকৈ বেছি ভ্ৰমণ কৰা site-সমূহ (যিটো **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** ত visit counting enable কৰাৰ প্ৰয়োজন)
