# সাইটসমূহ ব্যৱস্থাপনা কৰা

সাইট (বা সাবসাইট) হৈছে আপোনাৰ WaaS ব্যৱসায়ৰ মূল ভিত্তি। Ultimate Multisite-ত ৩ ধৰণৰ সাইট থাকে:

- **ग्राहक-ৰ মালিকানাধীন (Customer-owned)** — নির্দিষ্ট ग्राहकসকলৰ সৈতে জড়িত সাইটসমূহ
- **সাইট টেমপ্লেট (Site Templates)** — আগতে নিৰ্ধাৰিত সাইটসমূহ, যাক ग्राहकসকলে আৰম্ভণি স্থান হিচাপে বাছি ল'ব পাৰে
- **মূল সাইট (Main Site)** — আপোনাৰ প্ৰাথমিক নেটৱৰ্ক সাইট

## সাইট দেখা (Viewing Sites)

আপোনাৰ নেটৱৰ্কৰ সকলো সাবসাইট দেখাৰ বাবে **Ultimate Multisite → Sites** লৈ যাওক। প্ৰতিটো সাইটক ধৰণ অনুসৰি লেবেল কৰা হৈছে আৰু আপুনি All Sites, Customer-Owned, Templates, বা Pending অনুসৰি ফিল্টাৰ কৰিব পাৰে।

![Sites list page](/img/admin/sites-list.png)

## নতুন সাইট যোগ কৰা (Adding a New Site)

এটা নতুন সাইট সৃষ্টি কৰিবলৈ **Add Site** বাটনত ক্লিক কৰক। আপোনাক তলত দিয়া তথ্যসমূহ প্ৰদান কৰিব লাগিব:

- **Site Title** — নতুন সাইটটোৰ নাম
- **Site URL** — সাইটটোৰ ডোমেইন/পথ (domain/path)
- **Site Type** — ই এটা গ্রাহকৰ সাইট, টেমপ্লেট, নে সাধাৰণ সাইট

**Copy Site** বিকল্পটোৱে আপোনাক এটা বিদ্যমান সাইট টেমপ্লেটৰ ওপৰত ভিত্তি কৰি নতুন সাইট সৃষ্টি কৰাত সহায় কৰে। ই সক্ষম কৰিলে, আপুনি আৰম্ভণি স্থান হিচাপে কোন টেমপ্লেট ব্যৱহাৰ কৰিব লাগে, সেইটো বাছি ল'ব পাৰে। মিডিয়া ফাইল অন্তৰ্ভুক্ত কৰিবলৈ নিশ্চিত কৰক যে **Copy Media on Duplication** সক্ষম কৰা হৈ আছে।

## বিদ্যমান সাইট ব্যৱস্থাপনা কৰা (Managing an Existing Site)

যিকোনো সাইটৰ ওপৰত **Manage** ক্লিক কৰি **Edit Site** পৃষ্ঠাখন খুলক। ইয়াত আপুনি তলত দিয়া তথ্যসমূহ বিচাৰি পাব:

### মৌলিক তথ্য (Basic Information)

সাইটৰ নাম, ধৰণ, সাইট ID, আৰু বৰ্ণনা। আপুনি ম্যাপ কৰা ডোমেইন (mapped domain), জড়িত সদস্যপদ (associated membership), আৰু সাইটটোৰ মালিক হোৱা ग्राहकৰ একাউণ্টোও ইয়াত দেখিবলৈ পাব।

### সাইট বিকল্পসমূহ (Site Options)

সাইটটোৰ ক্ষমতা আৰু সীমা নিৰ্ধাৰণ কৰক:

- **Visit limits** — সাইটলৈ যোৱাৰ সর্বোচ্চ সংখ্যা
- **User account limits** — প্ৰতিজন ব্যৱহাৰকাৰীৰ ভূমিকা অনুসৰি সীমা
- **Disk space** — সাইটটোৰ বাবে ষ্টোৰেজ কোটা
- **Custom domain** — এই সাইটটোৰ বাবে ডোমেইন ম্যাপিং সক্ষম কৰক
- **Plugin and theme visibility** — কোন কোন plugin আৰু theme দৃশ্যমান, লুকাই থোৱা, বা আগতে active কৰিব, সেয়া নিয়ন্ত্ৰণ কৰক

স্বাভাবিকভাৱে, সাইটসমূহে সদস্যপদ স্তৰত (membership level) নিৰ্ধাৰিত সীমা অনুসৰণ কৰে। সাইট স্তৰত সীমা নিৰ্ধাৰণ কৰিলে, সেয়া সদস্যপদৰ সেটিংছক ওপৰত দিয়ে (override কৰিব)।

### সম্পৰ্কসমূহ (Associations)

সাইট বিকল্পসমূহৰ তলত আপুনি তলত দিয়া তথ্যসমূহ বিচাৰি পাব:

- সাইটটোৰ সৈতে জড়িত **Mapped domains**
- সাইটটোৰ অধীনস্থ **Membership**
- সাইটটোৰ সৈতে জড়িত **Customer account**

### সোঁফাৰ বাৰ্ (Right Sidebar)

ডানফাৰ বাৰ্-ত আপুনি:

- টগল বাটনৰ সহায়ত **সাইটটো সক্ষম/নিষ্ক্ষম** কৰিব পাৰে
- **সাইটৰ ধৰণ সলনি** কৰিব পাৰে বা মালিকীস্বত্ব পুনৰ নিৰ্ধাৰণ কৰিব পাৰে
- **এটা সাইট ইমেজ/থাম্বনেইল** সেট কৰিব পাৰে (যিটো সাইট টেমপ্লেটৰ বাবে ফ্ৰণ্টএণ্ডেত দেখুৱাই হয়)
- সাইটটো স্থায়ীভাৱে **মুছিব** পাৰে

:::warning
এটা সাইট মুছাটো উলটে নিব নোৱাৰি। সাইট আৰু ইয়াৰ সকলো বিষয়বস্তু স্থায়ীভাৱে আঁতৰাই দিয়া হ'ব।
:::
