# সদস্যপদ ব্যৱস্থাপনা

Ultimate Multisite ত আপুনি সদস্যপদ ব্যৱস্থাপনা কৰিব পাৰে আৰু আপোনাৰ নেটৱৰ্কৰ সকলো সাবস্ক্রিপচন (subscriptions) একত্ৰিতভাৱে ব্যৱহাৰ কৰিব পাৰে। আপুনি প্লেন আৰু প্ৰডাক্টৰ দৰে সাবস্ক্রিপচনৰ বিৱৰণ, সাইটৰ সংখ্যা, বিলিং ধন আৰু ফ্ৰিকৱেঞ্চি (frequency), পেমেন্টৰ ইতিহাস, আৰু টাইমষ্ট্যাম্প (timestamps) আদি পৰীক্ষা কৰিব আৰু সম্পাদনা কৰিব পাৰে।

এই পৃষ্ঠাখনলৈ যাবলৈ, Ultimate Multisite মেনুৰ **Memberships** ট‍্যাবটোত যাওক।

![Memberships list page](/img/admin/memberships-list.png)

Memberships পৃষ্ঠাখনত আপুনি আপোনাৰ গ্ৰাহকসকলৰ তালিকা আৰু তেওঁলোকৰ একাউণ্টৰ সৈতে জড়িত প্ৰডাক্টসমূহ দেখা পাব। আপুনি স্থিতি (status) অনুসৰি সদস্যপদ ফিল্টাৰ কৰিব পাৰে: active, pending, on hold, expired, বা cancelled।

বিনামূলীয়া সদস্যপদক **lifetime** সদস্যপদ হিচাপে গণ্য কৰা হয়। ইয়াৰ কোনো মেয়াদ শেষ হোৱাৰ তাৰিখ নাই আৰু ই স্বয়ংক্ৰিয়ভাৱে শেষ নহ'ব। যদি কোনো বিনামূলীয়া প্লেন এটা ট্রায়েল বা নিৰ্দিষ্ট সময়ৰ পিছত শেষ হোৱা উচিত, তেন্তে এটা ট্রায়েল সহ কোনো পেইড প্ৰডাক্ট ব্যৱহাৰ কৰক, সদস্যপদটো নিজে হাতেৰে বাতিল কৰক, বা তেওঁলোকৰ প্ৰৱেশাধিকাৰ (access) সলনি হ'লে গ্ৰাহকজনক আন এটা প্লেনলৈ স্থানান্তৰ কৰক।

সদস্যপদটোত ক্লিক কৰক যাতে আপুনি সদস্যপদৰ বিৱৰণ সম্পাদনা কৰিব পাৰে — বিলিং তথ্য, সাইট বা ভিজিটৰ সীমা, আৰু ইয়াৰ বাহিৰেও আন বহুতো তথ্য।

![Membership edit interface](/img/admin/membership-edit.png)

ইয়াত সদস্যপদ সম্পাদনা পৃষ্ঠাখনৰ সম্পূৰ্ণ দৃশ্য দিয়া হৈছে:

![Membership edit full interface](/img/admin/membership-edit-full.png)

আপুনি আপোনাৰ নেটৱৰ্কত পূৰ্বে ৰেজিষ্ট্ৰেড হোৱা ব্যৱহাৰকাৰীসকলৰ সৈতে জড়িত নতুন সদস্যপদও যোগ কৰিব পাৰে। **Add Membership** ক্লিক কৰক আৰু বিৱৰণসমূহ কনফিগাৰ কৰক।

![Add Membership modal](/img/admin/membership-add-modal.png)

তাৰ পাছত **Create Membership** ক্লিক কৰক। নতুন সদস্যপদটো সদস্যপদৰ তালিকাখনত দেখা যাব।
