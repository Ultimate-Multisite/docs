# Ultimate Multisite Dashboard বুझाइ

Ultimate Multisite-ৰ অধিক উন্নত কনফিগাৰেচন অপচন আৰু কাৰ্য্যগত দিশত গভীৰভাৱে নাযোৱাৰ আগতে, আমি dashboard-টোৰ ওপৰত চমু নজৰ কৰোঁ।

বহু WordPress plugin-ৰ দৰেই, Ultimate Multisite-এ **Ultimate Multisite** নামৰ এটা network-level menu entry সৃষ্টি কৰে, যাৰ তলত কিছুমান sub-menu entry আছে।

## Network Dashboard Widgets

Ultimate Multisite-এ network-level dashboard-ত কিছুমান উপকাৰী widget যোগ কৰে। এই dashboard-টো **My Sites → Network Admin → Dashboard** লৈ গৈ পোৱা যায়। এই widget-বোৰ network administrator-ৰ বাবে তথ্য আৰু সাধাৰণ কাৰ্য্যসমূহলৈ সহজ প্ৰৱেশ যোগান ধৰিবলৈ ডিজাইন কৰা হৈছে।

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

### First Steps

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

এই widget-টো Ultimate Multisite-ৰ নতুনকৈ install কৰাৰ পিছত দেখুৱাই আহে আৰু network administrator-ক পেমেন্ট মেথড কনফিগাৰ কৰা বা এটা টেস্টিং একাউণ্ট সৃষ্টি কৰাৰ দৰে আন কাৰ্য্যসমূহ সম্পন্ন কৰিবলৈ মনত পেণ কৰায়।

### Summary

![Summary widget](/img/admin/network-dashboard-summary.png)

Summary widget-এ দিনটোৰ signups আৰু তাৰ বাবে অৰ্জন কৰা ৰেভেণ्युৰ ওপৰত প্ৰতিবেদন দিয়ে। MRR (Monthly Recurring Revenue) ত সদস্যপদৰ বাবে ৰেভেণ्यु-উৎপাদনশীল সদস্যসকলৰ মাজত এটা প্ৰত্যাশিত মুঠ যোগান দেখুৱায়।

### Activity Stream

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

এই widget-টো Ultimate Multisite-ত হোৱা বিভিন্ন ঘটনাৰ ওপৰত প্ৰতিবেদন দিয়ে। ঘটনাৰ ভিতৰত signups, cancellations, site সৃষ্টি, deletion আৰু আন কাৰ্য্যসমূহ অন্তৰ্ভুক্ত।

অধিক বিশদ বৰ্ণনাৰ বাবে অনুগ্ৰহ কৰি ডকুমেন্টেশ্যনৰ Events অংশটো চাওক।

### Right Now

![Right Now widget](/img/admin/network-dashboard-right-now.png)

এই utility widget-টোৱে network-ত থকা ব্যৱহাৰকাৰী আৰু site-সমূহৰ এটা চমু সংক্ষেপ দেখুৱায়। ওপৰত থকা সহজ shortcut-বোৰে নতুন site বা user সৃষ্টি কৰিবলৈ এটা ক্লিকৰে প্ৰৱেশ যোগান ধৰে।

### News and Discussions

![News and Discussions widget](/img/admin/network-dashboard-news.png)

এই widget-টোৱে Latest Ultimate Multisite খবৰ আনি দেখুৱায়। আপডেট, গুৰুত্বপূৰ্ণ bug বা security fix সম্পৰ্কে তথ্য পাবলৈ এই widget-টোৰ ওপৰত চকু ৰাখিব।

## Ultimate Multisite Dashboard

য'ত network-level dashboard-এ network সম্পৰ্কে সামগ্ৰিক তথ্য প্ৰদৰ্শন কৰে, সেয়ে Ultimate Multisite dashboard (Ultimate Multisite menu-ৰ ওপৰত স্তৰত অৱস্থিত) ত সেৱিকাটোৰ (service) তথ্য প্ৰদৰ্শন কৰে।

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

### Monthly Recurring Revenue Growth

MRR (Monthly Recurring Revenue) chart-এ ১২ মাহৰ ক্যালেন্ডাৰ সময়ছোৱাত ৰেভেণ्युৰ মাহেকীয়া বিভাজন দেখুৱায়।

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

নতুন ৰেভেণ्युক নীলা (blue) ৰঙত আৰু cancellation-ক ৰঙা (red) ৰঙত প্ৰতিফলিত কৰা হয়।

### Date-range Support

ব্যৱহাৰকাৰী আৰু কাৰ্য্যৰ সংখ্যাৰ ওপৰত নিৰ্ভৰ কৰি, statistics অতি বেছি হ'ব পাৰে। dashboard-ত থকা date-range support-এ network administrator-ক কেৱল প্ৰয়োজনীয় তথ্য আৰু সময়ছোৱাত মনোযোগ দিবলৈ দিয়ে।

![Date range selector and stats](/img/admin/um-dashboard-date-range.png)

এই date range-এ widget-বোৰৰ তলত প্ৰদৰ্শিত তথ্যৰ সময়ছোৱাক নিয়ন্ত্ৰণ কৰে, যাৰ ভিতৰত ৰেভেণ्यु, নতুন সদস্যপদ, দেশ অনুসৰি signups, form অনুসৰি signups, আৰু আটাইতকৈ বেছি ভ্ৰমণ কৰা site-সমূহ অন্তৰ্ভুক্ত।

![Dashboard statistics widgets](/img/admin/um-dashboard-stats.png)

:::tip
**Most Visited Sites** widget-টো কাম কৰিবলৈ, **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** ত visit counting সক্ষম কৰক।
:::
