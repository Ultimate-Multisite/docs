# চেকআউট ফিল্ড আৰু কাস্টমৰ DNS ব্যৱস্থাপনা

## ডোমেইন নিৰ্বাচন চেকআউট ফিল্ড

**Domain Selection** ফিল্ডটো এটা চেকআউট এলিমেন্ট যাৰ জৰিয়তে কাস্টমৰক তেওঁলোকৰ সাইটৰ ডোমেইন কেনেকৈ পাব, তাৰ বাবে বিকল্প দিয়ে। ডোমেইন বিক্ৰী কৰিবলৈ আপুনি ইয়াক যিকোনো চেকআউট ফৰ্মত যোগ কৰিব পাৰে।

### চেকআউট ফৰ্মত ফিল্ড যোগ কৰা

১. **Network Admin › Ultimate Multisite › Checkout Forms** লৈ যাওক
২. এটা চেকআউট ফৰ্ম খুলক বা সৃষ্টি কৰক
৩. চেকআউট এডিটৰত, **Add Field** ক্লিক কৰক
৪. ফিল্ড তালিকাৰ পৰা **Domain Selection** বাছি লওক
৫. ফিল্ড অপচনসমূহ কনফিগাৰ কৰক (তলত চাওক)
৬. ফৰ্মটো সেভ কৰক

### ফিল্ড অপচনসমূহ

**Domain modes** — কাস্টমৰক কোন কোন ট্ৰাব (tabs) দেখা হ'ব, তাৰ বাবে বাছি লওক। প্ৰতিটো মোড স্বাধীনভাৱে সক্ষম বা অক্ষম কৰিব পাৰি:

| Mode | কি কৰে |
|---|---|
| **Subdomain** | কাস্টমৰে আপোনাৰ নেটৱৰ্কৰ এটা বিনামূলীয়া সাবডোমেইন ব্যৱহাৰ কৰে (উদাহৰণস্বৰূপে, `mysite.yournetwork.com`)। কোনো পেমেন্টৰ প্ৰয়োজন নাই। |
| **Register New Domain** | কাস্টমৰে এটা নতুন ডোমেইন বিচাৰি উলিয়াই আপোনাৰ কনফিগাৰেট কৰা প্ৰোভাইডাৰৰ জৰিয়তে ইয়াক ৰেজিষ্ট্ৰে কৰে। মূল্য নিৰ্ধাৰণ কৰিবলৈ মিল থকা ডোমেইন প্ৰডাক্ট ব্যৱহাৰ কৰে। |
| **Existing Domain** | কাস্টমৰে তেওঁলোকৰ মালিকানাধীন এটা ডোমেইন ম্যাপ কৰে। কোনো ৰেজিষ্ট্ৰেচন ফি নহয়। ডোমেইনটো স্বয়ংক্রিয়ভাৱে তেওঁলোকৰ সাইটলৈ ম্যাপ কৰা হয়। |

**Default mode** — যেতিয়া তিনিওটা মোড সক্ষম কৰা হয়, তেতিয়া কোন ট্ৰাবটো প্ৰথম খুলিব। ডোমেইন ৰেজিষ্ট্ৰেচনক ঐচ্ছিক কৰি ৰাখিবলৈ ইয়াক **Subdomain** সেট কৰক, অথবা ক্ৰয়ক উৎসাহিত কৰিবলৈ **Register New Domain** সেট কৰক।

**Domain product** — ঐচ্ছিকভাৱে এই ফিল্ডটো এটা নিৰ্দিষ্ট ডোমেইন প্ৰডাক্টলৈ পিন কৰিব পাৰি। যদি ইয়াক সেট কৰা নহয়, তেন্তে এই এডননটোৱে কাস্টমৰে যি TLD বিচাৰে তাৰ ওপৰত ভিত্তি কৰি মিল থকা প্ৰডাক্টটো স্বয়ংক্রিয়ভাৱে বাছি লয়।

### Registrant contact fields

যদি কাস্টমৰে **Register New Domain** ট্ৰাবটো বাছি লয়, তেন্তে চেকআউট ফৰ্মটোৱে ইনলাইনভাৱে ৰেজিষ্ট্ৰেণ্টৰ যোগাযোগ ফিল্ড যোগ কৰে:

- First name / Last name
- Email address
- Address (line 1, city, state/province, postal code, country)
- Phone number

এইবোৰ সকলো ৰেজিষ্ট্ৰাৰৰ বাবে বাধ্যতামূলক আৰু ৰেজিষ্ট্ৰেচন API কল কৰাৰ আগতে ইয়াৰ বৈধতা পৰীক্ষা কৰা হয়। ফোন নম্বৰবোৰ স্বয়ংক্রিয়ভাৱে ৰেজিষ্ট্ৰাৰসকলে আশা কৰা `+CC.NNN` আন্তঃৰ্জাতিক ফৰম্যাটোত সাজু কৰা হয়।

### Auto-generated site URL

যদি কাস্টমৰে এটা ডোমেইন ৰেজিষ্ট্ৰে কৰে বা ম্যাপ কৰে, তেন্তে সাইট URL ফিল্ডটো স্বয়ংক্রিয়ভাৱে বাছি লোৱা ডোমেইনৰ পৰা পূৰণ কৰা হয়। কাস্টমৰসকলে এটা পৃথক URL ফিল্ড পূৰণ কৰাৰ প্ৰয়োজন নাই।

### Search behaviour

- কাস্টমৰে টাইপ কৰাৰ লগে লগে AJAX ব্যৱহাৰ কৰি ডোমেইনৰ উপলব্ধতা লাইভভাৱে পৰীক্ষা কৰা হয়
- যদি পছন্দ কৰা ডোমেইনটো উপলব্ধ নহয়, তেন্তে বিকল্প TLD সাজেশন দেখুওৱা হয়
- মূল্য লাইভভাৱে লৈ আহৰণ কৰা হয় আৰু স্পষ্টকৈ প্ৰদৰ্শিত হয় (ৰেজিষ্ট্ৰেচন মূল্য, পুনৰাবৃত্তি মূল্য, ঐচ্ছিক WHOIS privacy ফি)
- কুপন কোডবোৰ আন যিকোনো প্ৰডাক্টৰ দৰেই ডোমেইন প্ৰডাক্টলৈ প্ৰযোজ্য হয়

**Search responsiveness উন্নত কৰা:**

```php
// Slow connections ত API কল হ্ৰাস কৰিবলৈ debounce delay (milliseconds) বৃদ্ধি কৰক
add_filter('wu_domain_seller_search_delay', function($delay) {
    return 800; // default: 500
});
```

**ডোমেইন অনুসন্ধান ফৰ্মত কাস্টম ফিল্ড যোগ কৰা:**

```php
add_filter('wu_checkout_form_register_domain_form_fields', function($fields) {
    $fields['custom_note'] = [
        'type'  => 'text',
        'label' => 'Additional notes',
    ];
    return $fields;
});
```

## কাস্টমৰ DNS ব্যৱস্থাপনা

কাস্টমৰসকলে তেওঁলোকৰ ৰেজিষ্ট্ৰেড ডোমেইনৰ বাবে DNS ৰেকৰ্ডছ **My Account** পৃষ্ঠাৰ পৰা, তেওঁলোকৰ ডোমেইনৰ এণ্ট্ৰীৰ তলত ব্যৱস্থাপনা কৰিব পাৰে।

### সমৰ্থন কৰা ৰেকৰ্ড প্ৰকাৰসমূহ

| Type | ব্যৱহাৰ |
|---|---|
| **A** | হোষ্টনেমক IPv4 ঠিকনাৰ সৈতে ম্যাপ কৰে |
| **AAAA** | হোষ্টনেমক IPv6 ঠিকনাৰ সৈতে ম্যাপ কৰে |
| **CNAME** | আন এটা হোষ্টনেমলৈ ইংগিত কৰা এটা এলিয়াস (alias) সৃষ্টি কৰে |
| **MX** | মেেল এক্সচেঞ্জ ছাৰ্ভাৰ সেট কৰে |
| **TXT** | SPF, DMARC, ভেরিফিকেচন, বা আন টেক্সট ৰেকৰ্ড যোগ কৰে |

### কোন প্ৰোভাইডাৰসকলে DNS ব্যৱস্থাপনা সমৰ্থন কৰে?

DNS ব্যৱস্থাপনা (ৰেজিৰ্ড যোগ কৰা, সম্পাদনা কৰা, ডিলিট কৰা) **OpenSRS**, **ResellerClub**, আৰু **Enom** লগত উপলব্ধ। Namecheap, GoDaddy, আৰু NameSilo ডোমেইনসমূহে ষ্টেটছ আৰু মেয়াদোত্তৰ তথ্য দেখুৱায়, কিন্তু DNS ৰ ব্যৱস্থাপনাটো ৰেজিষ্ট্ৰাৰৰ কণ্ট্ৰল প্যানেলত পোনপটীয়াকৈ কৰিব লাগিব।

### ডিফল্ট DNS ৰেকৰ্ডছ

আপুনি ডিফল্ট DNS ৰেকৰ্ডছ কনফিগাৰ কৰিব পাৰে যিবোৰ ডোমেইন ৰেজিষ্ট্ৰে কৰাৰ সময়ত স্বয়ংক্রিয়ভাৱে প্ৰয়োগ হয়। **Settings › Domain Seller › Default DNS Records** লৈ যাওক।

ডিফল্ট ৰেকৰ্ড মূল্যসমূহে দুটা টোকেন সমৰ্থন কৰে:

| Token | ক'ত সলনি হয় |
|---|---|
| `{DOMAIN}` | ৰেজিষ্ট্ৰেড ডোমেইনৰ নাম (উদাহৰণস্বৰূপে, `example.com`) |
| `{SITE_URL}` | কাস্টমৰৰ সাইটৰ বাবে WordPress সাইট URL |

**উদাহৰণ — এপেক্স ডোমেইন আৰু www ক আপোনাৰ ছাৰ্ভাৰ IP লৈ নিয়া:**

```
Type: A
Name: {DOMAIN}
Value: 203.0.113.10

Type: CNAME
Name: www.{DOMAIN}
Value: {DOMAIN}
```

### Admin: DNS দেখা আৰু সম্পাদনা কৰা

Network adminসকলে **Network Admin › Ultimate Multisite › Domains** ত ডোমেইনৰ এডিট পৃষ্ঠাৰ পৰা যিকোনো কাস্টমৰ ডোমেইনৰ DNS ৰেকৰ্ড দেখা আৰু সম্পাদনা কৰিব পাৰে।
