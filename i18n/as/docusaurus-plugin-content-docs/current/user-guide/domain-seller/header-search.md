# হেডাৰ ডোমেইন সন্ধান

যেতিয়া আপুনি এটা সৰু হেডাৰ ফৰ্ম বিচাৰে যিয়ে ডোমেইন সন্ধান আৰম্ভ কৰে, তাৰ পিছত গ্ৰাহকক Ultimate Multisite চেকআউটৰ ভিতৰত উপলব্ধ ডোমেইনসমূহৰ পৰা বাছনি কৰিবলৈ দিয়ে, তেতিয়া এই ছেটআপ ব্যৱহাৰ কৰক।

## প্ৰয়োজনীয়তা

- Ultimate Multisite network-active।
- Multisite Ultimate Domain Seller network-active।
- অন্ততঃ এটা সক্ৰিয় ডোমেইন পঞ্জীয়ন সামগ্ৰী, য’ত আছে:
  - `domain_provider` এটা কনফিগাৰ কৰা প্ৰদানকাৰীলৈ ছেট কৰা।
  - সমৰ্থিত TLDs কনফিগাৰ কৰা, উদাহৰণস্বৰূপে `com`, `net`, আৰু `org`।
- **ডোমেইন নিৰ্বাচন** ক্ষেত্ৰ থকা এটা বৈধ চেকআউট ফৰ্ম।

## চেকআউট ফৰ্ম

1. পঞ্জীয়ন পৃষ্ঠাই ব্যৱহাৰ কৰা চেকআউট ফৰ্ম সৃষ্টি বা সম্পাদনা কৰক।
2. **ব্যৱহাৰকাৰী নাম**-সহ সাধাৰণ প্ৰয়োজনীয় চেকআউট/Account ক্ষেত্ৰসমূহ যোগ কৰক। কেৱল ডোমেইন ক্ষেত্ৰ থকা চেকআউট ফৰ্ম Ultimate Multisite বৈধতা পৰীক্ষাই প্ৰত্যাখ্যান কৰে।
3. এটা **ডোমেইন নিৰ্বাচন** ক্ষেত্ৰ যোগ কৰক।
4. যেতিয়া প্ৰবাহটোৱে বিনামূলীয়া উপডোমেইন বা বিদ্যমান ডোমেইনৰ সলনি পঞ্জীয়িত ডোমেইনত গুৰুত্ব দিব লাগে, তেতিয়া ডোমেইন নিৰ্বাচন ম’ড **কেৱল পঞ্জীয়ন**-লৈ ছেট কৰক।
5. সেই ক্ষেত্ৰত ডোমেইন পঞ্জীয়ন সামগ্ৰী নিৰ্ধাৰণ কৰক।

পঞ্জীয়ন পৃষ্ঠাই চেকআউট ফৰ্ম ৰেণ্ডাৰ কৰিব লাগে, উদাহৰণস্বৰূপে:

```text
[wu_checkout slug="domain-form"]
```

## হেডাৰ ফৰ্ম

ছাইটৰ হেডাৰত এটা সৰু `GET` ফৰ্ম যোগ কৰক, যিয়ে দিয়া সন্ধান শব্দটো `domain_name` হিচাপে চেকআউট পৃষ্ঠালৈ পঠিয়ায়:

```html
<form class="ums-header-domain-search" action="/register/" method="get">
  <label class="screen-reader-text" for="ums-header-domain-search-input">Search for a domain</label>
  <input id="ums-header-domain-search-input" name="domain_name" type="text" placeholder="Find your domain" autocomplete="off">
  <button type="submit">Search</button>
</form>
```

কাষ্টম হেডাৰ JavaScript-ত কোনো ডোমেইন পূৰ্বনিৰ্বাচন নকৰিব। হেডাৰে কেৱল সন্ধান শব্দটো পাছ কৰিব লাগে। Domain Seller চেকআউট স্ক্ৰিপ্টে `?domain_name=example` পঢ়ে, চেকআউট সন্ধান বাকচ পূৰণ কৰে, আৰু উপলব্ধতা সন্ধান চলায় যাতে গ্ৰাহকে ঘূৰাই দিয়া ডোমেইনসমূহৰ পৰা বাছনি কৰিব পাৰে।

## প্ৰত্যাশিত আচৰণ

হেডাৰত `example` সন্ধান কৰিলে খুলিব লাগে:

```text
/register/?domain_name=example
```

তাৰ পিছত চেকআউটে এনে নিৰ্বাচনযোগ্য ফলাফল দেখুৱাব লাগে:

- `example.com`
- `example.net`
- `example.org` উপলব্ধ নহয়
- প্ৰদানকাৰীয়ে সমৰ্থন কৰা অন্য TLDs

এটা উপলব্ধ ফলাফল বাছনি কৰাৰ পিছত, অৰ্ডাৰ সাৰাংশত ডোমেইন পঞ্জীয়ন সামগ্ৰী আৰু নিৰ্বাচিত ডোমেইন নাম থাকিব লাগে।

## যাচাই

1. মুখ্য পৃষ্ঠা খোলক।
2. এটা খালী নাম সন্ধান কৰক, উদাহৰণস্বৰূপে `example`।
3. চেকআউট URL-ত `?domain_name=example` অন্তৰ্ভুক্ত আছে বুলি নিশ্চিত কৰক।
4. চেকআউট ডোমেইন ক্ষেত্ৰত `example` আছে বুলি নিশ্চিত কৰক।
5. ডোমেইন পছন্দৰ এখন তালিকা দেখা যায় বুলি নিশ্চিত কৰক।
6. উপলব্ধ ডোমেইনৰ বাবে **নিৰ্বাচন কৰক** ক্লিক কৰক।
7. অৰ্ডাৰ সাৰাংশত `Domain Registration - example.com` বা নিৰ্বাচিত ডোমেইন আছে বুলি নিশ্চিত কৰক।

## সমস্যা সমাধান

- যদি কোনো তালিকা নেদেখায়, `admin-ajax.php?action=wu_domain_search`-ৰ বাবে ব্ৰাউজাৰৰ network টেব পৰীক্ষা কৰক আৰু ই খালী নহোৱা `domains` বা `results` ঘূৰাই দিয়ে বুলি নিশ্চিত কৰক।
- যদি সংৰক্ষণ কৰাৰ সময়ত চেকআউট ফৰ্মৰ বৈধতা পৰীক্ষা বিফল হয়, **ব্যৱহাৰকাৰী নাম**-ৰ দৰে প্ৰয়োজনীয় Account ক্ষেত্ৰসমূহ যোগ কৰক।
- যদি ডোমেইন নিৰ্বাচন কৰিলে কাৰ্ট আপডেট নহয়, `window.wu_checkout_form` আছে আৰু Domain Seller চেকআউট assets চেকআউট পৃষ্ঠাত লোড হৈছে বুলি নিশ্চিত কৰক।
