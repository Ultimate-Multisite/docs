# আপোনাৰ ক্লায়েন্টৰ ওয়েবসাইটসমূহ (v2)

আপোনাৰ নেটৱৰ্কে ৰাজিস্ট্ৰেচন কৰাৰ পিছত, আপোনাৰ গ্ৰাহকসকলে তেওঁলোকৰ ড্যাশবৰ্ডলৈ প্ৰৱেশ কৰিব পাৰে।

**Account** পৃষ্ঠাৰ উপৰিও, য'ত তেওঁলোকে তেওঁলোকৰ পেমেন্ট, মেম্বৰশ্বিপ, ডোমেইন, پلانৰ সীমাবদ্ধতা আৰু অন্যান্য বিষয়ে গুৰুত্বপূৰ্ণ তথ্য দেখা পাওঁ, তাতকৈও অধিক, তেওঁলোকে **My Sites** পৃষ্ঠাখনলৈও প্ৰৱেশ কৰিব পাৰে। এই পৃষ্ঠাখনত তেওঁলোকে তেওঁলোকৰ সকলো ওয়েবসাইট দেখা আৰু ব্যৱস্থাপনা কৰিব পাৰে। তেওঁলোকে কেৱল তেওঁলোকৰ ড্যাশবৰ্ডত My Sites ওপৰত ক্লিক কৰিব লাগিব।

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

তেওঁলোকৰ সকলো ওয়েবসাইট ইয়াত দেখুৱাই দিয়া হ'ব, আৰু প্ৰত্যেকটোৰ ঠিক তলত এটা **Manage** বাটন থাকিব। যদি তেওঁলোকে ইয়াত ক্লিক কৰে, তেন্তে তেওঁলোকক **এই নিৰ্দিষ্ট ওয়েবসাইটৰ ড্যাশবৰ্ডলৈ** পুনৰ স্থানান্তৰ কৰা হ'ব।

তেওঁলোকে আৰু এইটোও দেখা পাৰিব যে কোনটো ওয়েবসাইট তেওঁলোকৰ বাবে **primary** (মুখ্য) হ'ব, বিশেষকৈ এনে پلانৰ বাবে যিয়ে তেওঁলোকক এটাৰ অধিক ওয়েবসাইট ৰাখিবলৈ অনুমতি দিয়ে।

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

ইয়াত সাইট এডিট ইন্টাৰফেচৰ সম্পূৰ্ণ দৃশ্যটো দিয়া হ'ল:

![Site edit full interface](/img/admin/site-edit-full.png)

যদি তেওঁলোকে তেওঁলোকৰ primary ওয়েবসাইট সলনি কৰিব বিচাৰে, তেন্তে তেওঁলোকে **Account** মেনুত ক্লিক কৰি Account পৃষ্ঠাখনলৈ যাব লাগিব।

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

আপুনি গ্ৰাহকৰ বিৱৰণও (details) দেখা আৰু সম্পাদনা কৰিব পাৰে:

![Customer edit interface](/img/admin/customer-edit.png)

তাৰ পিছত, তেওঁলোকে **Change Default Site** ওপৰত ক্লিক কৰিব লাগিব।

![Change Default Site button on account page](/img/account-page/change-default-site.png)

এটা পপ-আপ উইন্ডো খুলিব। ড্রপ-ডাউন তালিকাখনত ক্লিক কৰক আৰু আপুনি কোনটো ওয়েবসাইটক primary ওয়েবসাইট বনাওঁ বিচাৰে, সেয়া বাছি লওক।

![Change Default Site dropdown selection](/img/account-page/change-default-site-dropdown.png)

তাৰ পিছত, কেৱল **Change Default Site** ওপৰত ক্লিক কৰক আৰু আপোনাৰ কাম সম্পূৰ্ণ!

![Confirming the default site change](/img/account-page/change-default-site-confirm.png)
