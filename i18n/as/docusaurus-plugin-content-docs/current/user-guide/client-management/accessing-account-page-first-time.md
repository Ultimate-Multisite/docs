# মই প্ৰথমবাৰৰ বাবে মোৰ একাউণ্ট পৃষ্ঠালৈ কেনেকৈ যাব পাৰোঁ?

Ultimate Multisite লৈ স্বাগতম!

আমাৰ plugin ক্ৰয় কৰাৰ পিছত আপুনি ইমেইলত যি লিংক পাইছে, তাৰ জৰিয়তে আপোনাৰ একাউণ্ট পৃষ্ঠালৈ প্ৰৱেশ কৰক:

<!-- Screenshot unavailable: Account page email with access link -->

ক্ৰয় কৰাৰ সময়ত আপুনি দিয়া ইমেইল আৰু আপোনাৰ ইমেইলত পোৱা পাছৱৰ্ড ব্যৱহাৰ কৰক।

<!-- Screenshot unavailable: Login page with email and password fields -->

যেহেতু আপুনি এই একাউণ্টলৈ প্ৰথমবাৰৰ বাবে লগইন কৰিছে, সেয়েহে আপোনাক আপোনাৰ পাছৱৰ্ড সলনি কৰিবলৈ কোৱা হ'ব। ইয়াক সলনি কৰি _Save_ বাটনত ক্লিক কৰক।

<!-- Screenshot unavailable: Change password form with Save button -->

আপোনাৰ একাউণ্ট পৃষ্ঠাখনত, আপুনি বিলিং, পুনৰনিৰ্মাণ (renewals), বাতিল কৰা (cancelations) আৰু অন্যান্য বহুতো বিৱৰণ ব্যৱস্থাপনা কৰিব পাৰে।
