# মই মোৰ সাবস্ক্রিপচন কেনেকৈ বাতিল কৰিব পাৰোঁ? {#how-can-i-cancel-my-subscription}

যদি আপোনাৰ Ultimate Multisite লাইচেன்ஸ் আছে, তেন্তে আপুনি যেতিয়া ইচ্ছা পুনৰীক্ষণ (renewal) বাতিল কৰিব পাৰে। তলত দিয়া পদক্ষেপসমূহ অনুসৰণ কৰক:


<!-- Screenshot unavailable: Account page email with access link -->

ক্ৰয়ৰ সময়ত দিয়া ইমেইল আইডি আৰু আপোনাৰ পাছৱৰ্ড ব্যৱহাৰ কৰক।

<!-- Screenshot unavailable: Login page with email and password fields -->

এইটো হ'ল আপোনাৰ Account Page-ৰ আৰম্ভণিৰ দৃশ্য:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

আপোনাৰ সাবস্ক্রিপচন বাতিল কৰিবলৈ, বাঁহফালে থকা মেনুৰ পৰা, _Renewing & Billing_ ত যাওক।

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

কাষৰ ফালে থকা তীব্ৰকণ (arrow) টো ক্লিক কৰি এটা সাইড উইন্ডো খুলক। তাৰ পিছত, আপুনি **auto-renew** বাতিল কৰা বিকল্পটো বাছি ল'ব লাগিব।

ব্যৱস্থাই আপোনাক এটা নিশ্চিতকৰণ (confirmation) বাৰ্তা দেখুৱাব।

যদি আপুনি নিশ্চিত যে আপুনি आफ्नो সাবস্ক্রিপচন বাতিল কৰিব বিচাৰে, তেন্তে কেৱল _**Cancel Renewals**_ বোতামত ক্লিক কৰক। এই কাৰ্যৰ পিছত, আপোনাক এটা সৰু সমীক্ষাত উত্তৰ দিবলৈ কোৱা হ'ব।

হৈ গ'ল! আপোনাৰ সাবস্ক্রিপচন স্বয়ংক্রিয়ভাৱে পুনৰীক্ষণ নহ'ব।

**আপোনাৰ সাবস্ক্রিপচনৰ মেয়াদ শেষ হোৱা পৰ্যন্ত আপোনাৰ এটা বৈধ key থাকিব।** যদি আপুনি আপোনাৰ সাবস্ক্রিপচন পুনৰ সক্ৰিয় কৰিব বিচাৰে, তেন্তে আপুনি ই মানুহে (manually) কৰিব লাগিব।
