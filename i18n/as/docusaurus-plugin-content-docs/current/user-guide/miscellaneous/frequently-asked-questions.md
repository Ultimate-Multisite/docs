# সঘনাই सोधिएका প্ৰশ্নসমূহ {#frequently-asked-questions}

সহায়ৰ অনুৰোধৰ সময়ত সঘনাই উত্থাপিত হোৱা কিছুমান **সঘনাই सोधिएका প্ৰশ্নসমূহ** তলত দিয়া হ'ল।

  * **Ultimate Multisite কি?**

Ultimate Multisite হৈছে এটা WordPress plugin, যি WordPress multisite install-ৰ বাবে বনোৱা হৈছে। ই আপোনাৰ WordPress install-ক এটা প্ৰিমিয়াম নেটৱৰ্কৰ সাইটত ৰূপান্তৰ কৰে—যেনে [WordPress.com](https://WordPress.com)—যি ক্লায়েন্টসকলক মাহে, ত্রৈমাসিকে বা বছৰৰ ধাৰাবাহিকতাৰে সাইট সৃষ্টি কৰিবলৈ দিয়ে (আপুনি বিনামূলীয়া (Free) প্লেনো সৃষ্টি কৰিব পাৰে)।

  * **Multisite নোহোৱাকৈও Ultimate Multisite কাম কৰিবনে?**

নহয়। Ultimate Multisite-ক WordPress multisite, subdirectory, বা subdomainৰ প্ৰয়োজন।

  * **Ultimate Multisite-ৰ বাবে কি কি প্ৰয়োজনীয়তা আছে?**

Ultimate Multisite-ক তলত দিয়াৰ প্ৰয়োজন:

**WordPress Version** : v5.3+ (পৰামৰ্শ দিয়া: শেহতীয়া স্থিতিশীল সংস্কৰণ)
**PHP Version** : 7.4.x (প্ৰশীৰৰ বাবে > 8.0 সমৰ্থন সোনকালে আহিব)
**MySQL Version** : v5+ (পৰামৰ্শ দিয়া: সংস্কৰণ 5.6, যদি আপোনাৰ hosting provider-ত 8.0 উপলব্ধ নহয়)

_Multisite (subdomain বা subdirectory) তো সক্রিয় কৰিব লাগিব।_ [How to Install WordPress Multisite](/user-guide/getting-started/how-to-install-wordpress-multisite)।

  * **Ultimate Multisite কি subdirectory সহ Multisite লগত কাম কৰিবনে?**

হয়। Ultimate Multisite-এ subdirectory install আৰু subdirectory install দুয়োটা সমৰ্থন কৰে।

  * **Ultimate Multisite কি subdomain সহ Multisite লগত কাম কৰিবনে?**

হয়। Ultimate Multisite-এ subdirectory install আৰু subdomain install দুয়োটা সমৰ্থন কৰে।

  * **মই কি Ultimate Multisite ব্যৱহাৰ কৰি user-সকলে কোন কোন Plugins আৰু Themes ব্যৱহাৰ কৰিব পাৰিব, সেয়া সীমিত কৰিব পাৰোঁ?**

হয়। Ultimate Multisite ব্যৱহাৰ কৰি আপুনি আপোনাৰ নেটৱৰ্কৰ প্ৰতিটো প্লেনৰ বাবে কোন কোন plugins আৰু themes উপলব্ধ হ'ব, সেয়া নিৰ্বাচন কৰিব পাৰে।

  * **মই কি Ultimate Multisite ব্যৱহাৰ কৰি user-সকলে তেওঁৰ সাইটত কেইটা post সৃষ্টি কৰিব পাৰিব, সেয়া সীমিত কৰিব পাৰোঁ?**

হয়। Ultimate Multisite ব্যৱহাৰ কৰি আপুনি আপোনাৰ নেটৱৰ্কৰ **প্ৰতিটো উপলব্ধ post type**-ৰ বাবে post quota স্থাপন কৰিব পাৰে। আপুনি ম‍্যাডমিনক কেইজনমান userক আমন্ত্ৰণ জনোৱাৰ সীমা, **disk space-ৰ পৰিমাণ**, আৰু ইয়াৰ বাহিৰেও বহুতো কথা সীমিত কৰিব পাৰে।

  * **Ultimate Multisite কি Site Templates সমৰ্থন কৰে?**

হয়। Ultimate Multisite ব্যৱহাৰ কৰি আপুনি যিমানেই Site Templates সৃষ্টি কৰিব বিচৰা, সেয়েহে কৰিব পাৰে। Site Templates হৈছে theme নহয়, ই আপোনাৰ নেটৱৰ্কৰ ওপৰত থকা সাধাৰণ সাইট। আপুনি plugins, themes, আৰু Site Template-ক যিমান ইচ্ছা কাস্টমাইজ কৰিব পাৰে। যদি কোনো user এই টেমপ্লেটটো নিৰ্বাচন কৰে, তেন্তে তেওঁৰ সাইটটো সেই টেমপ্লেটটোৰ এটা কপি হিচাপে সৃষ্টি হ'ব, আৰু সকলো কাস্টমাইজেশ্যন তাত থাকিব।

আপুনি এটা প্লেনৰ বাবে এটা নিৰ্দিষ্ট টেমপ্লেটও সেট কৰিব পাৰে। এনে কৰাৰ দ্বাৰা, সেই প্লেনত নাম লেখোৱা প্ৰতিজন userৰ সাইটটো সেই Site Template-ৰ এটা নকল হিচাপে সৃষ্টি হ'ব।

  * **Ultimate Multisite কি WooCommerce লগত সংহতি কৰে?**

হয়!

যদি আপুনি বিচাৰে যে আপোনাৰ নেটৱৰ্কৰ user-সকলে WooCommerce ব্যৱহাৰ কৰিব পাৰে, যাতে তেওঁলোকে ষ্ট’ৰ স্থাপন কৰিব পাৰে, তেন্তে Ultimate Multisite ব্যৱহাৰ কৰি সেয়া সম্ভৱ। আপুনি আনহাতে আপোনাৰ ক্লায়েন্টসকলক প্ৰতিটো প্লেন টীৰত কেইটা প্ৰোডাক্ট সৃষ্টি কৰিবলৈ সীমাও দি যাব পাৰে।

Ultimate Multisite-ৰ subscription payment-ৰ ক্ষেত্ৰত, আপুনি [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) ব্যৱহাৰ কৰিব পাৰে যাতে আপোনাৰ Ultimate Multisite ক্লায়েন্টসকলে আপোনাৰ মূল সাইটত WooCommerce gateway ব্যৱহাৰ কৰি ধন পৰিশোধ কৰিব পাৰে। এনেদৰে, আপুনি WooCommerce-ৰ বাবে বিকশিত হোৱা বিশাল payment gateway-ৰ লাইব্ৰেৰীত প্ৰৱেশ কৰিব পাৰে আৰু আপোনাৰ Ultimate Multisite ক্লায়েন্টসকলক বিল কৰিবলৈ ব্যৱহাৰ কৰিব পাৰে।

  * **WPMUDEV Pro Sites ৰ পৰা subscription স্থানান্তৰ কৰাৰ কোনো উপায় আছে নেকি?**

হয়, Ultimate Multisite এ এতিয়া আপোনাৰ Pro Sites network-ক স্থানান্তৰ কৰাত সহায় কৰিবলৈ [a helpful tool to get you to migrate](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) এটা সহায়ক ট্ৰুল অফাৰ কৰে।

  * **মোৰ Plan Edit Page-ত “Allowed Plugins” ট্ৰাৱেলটো খালি কিয় আছে?**

Ultimate Multisite কেৱল আপোনাৰ নেটৱৰ্কত network-activated নোহোৱা plugins-লৈ প্ৰৱেশ কৰিবলৈ অনুমতি দিব বা নাকচ কৰিব পাৰে। যেতিয়া কোনো plugin network-activated হয়, তেতিয়া স্বাভাৱিকভাৱে, ই আপোনাৰ নেটৱৰ্কৰ প্ৰতিটো সাইটত ডিফল্টভাৱে লোড হয়।

যদি আপুনি “Allowed Plugins” ট্ৰাৱেলত কোনো plugin তালিকাভুক্ত নহয়, তেন্তে পৰীক্ষা কৰক যে কাৰণীয় plugin টো network-active নে নাই।

  * **মই মোৰ নিজৰ Pricing Table design ব্যৱহাৰ কৰিব পাৰোঁ নেকি?**

হয়। ১.৪.০ সংস্কৰণৰ পৰা, Ultimate Multisite-এ template overrides সমৰ্থন কৰে।

  * **Ultimate Multisite কি WP Engine লগত কাম কৰে?**

হয়। Ultimate Multisite সদায়ৱে WP Engine-ৰ দ্বাৰা সমৰ্থন কৰা হৈ আহিছে, কিন্তু domain mapping কৰিবলৈ ম্যানুৱেলভাৱে কিছুমান অতিৰিক্ত পদক্ষেপ ল'ব লাগিছিল (domain টো WP Engine panel-ত যোগ কৰিব লাগিছিল)। **১.৬.০ সংস্কৰণৰ পৰা এইটো সমস্যা নহ'ল।** Ultimate Multisite-এ পোনপটীয়াকৈ WP Engine API লগত সংহতি কৰে আৰু mapping হোৱাৰ পিছত domain টো স্বয়ংক্রিয়ভাৱে panel-ত যোগ হয়।

  * **আপুনি কোন hosting provider-ক পৰামৰ্শ দিয়ে?**

Ultimate Multisite কোনো ধৰণৰ hosting provider লগত কাম কৰিব লাগে। এটামাত্ৰ পাৰ্থক্য হ'ল যে Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane আৰু [Runcloud.io](http://Runcloud.io) जस्ता কিছুমান hosting কোম্পানীৰ domain mapping capability-ৰ বাবে অধিক নিবিড় সংহতি আছে, সেয়েহে সিহঁত 'out of the box' কাম কৰে। আন hosting-ৰ বাবে, আপুনি domain mapping ম্যানুৱেলভাৱে স্থাপন কৰিব লাগিব।
