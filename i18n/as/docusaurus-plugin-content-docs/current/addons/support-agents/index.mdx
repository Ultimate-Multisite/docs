# Support Agents

এটা সম্পূৰ্ণ নেটৱৰ্ক নিজে পৰিচালনা কৰাটো সহজ কাম নহয়। যেতিয়া আপোনাৰ নেটৱৰ্ক ডাঙৰ হ'বলৈ ধৰে, তেতিয়া আপুনি সহায়ৰ বাবে বাঢ়ি অহা চাহিদা অনুভৱ কৰিব। আপোনাৰ কামৰ চাপ কম কৰাত সহায় কৰিব পৰা লোকক লৈ অহাটো এটা ভাল ধাৰণা হ'ব। কিন্তু, সম্পূৰ্ণ অ্যাক্সেছ বা অনুমতি (permission) দিয়াটো বিপদজনক হ'ব পাৰে আৰু ই আপোনাৰ নেটৱৰ্কক ক্ষতিগ্ৰস্ত কৰিব পাৰে। এই ঠাইতে এই Support Agent add-on-টোৰ প্ৰয়োজন।

এই add-on-টোৱে আপোনাক Support Agent নামৰ এটা নতুন ধৰণৰ WordPress user সৃষ্টি কৰিবলৈ দিয়ে, আৰু সেই user-এ কি কি অনুমতি (permissions) পাব, তাৰ ওপৰত আপোনাক সম্পূৰ্ণ নিয়ন্ত্ৰণ দিয়ে। অনুমতি দিয়াটো এটা toggle button অন কৰাৰ দৰে সহজ, আৰু যদি আপুনি ইয়াক আঁতৰাবলৈ সিদ্ধান্ত লয়, তেন্তে ইয়াক অফ কৰাটোও একেবাৰে সহজ।

## Features

- WordPress আৰু Ultimate Multisiteৰ অনুমতিৰ তালিকা (Itemized list)
- সহজে অ্যাক্সেছ সক্ষম কৰা আৰু অক্ষম কৰা (Enable & disable access)
- সীমাহীন সংখ্যক Agent account সৃষ্টি কৰিব পাৰে

## Installation

1. 'ultimate-multisite-support-agents' ফাইলটো '/wp-content/plugins/' ডাইৰেক্টৰীত আপলোড কৰক।
2. WordPress-ৰ 'Plugins' মেনুৰ জৰিয়তে plugin টো অ্যাক্টিভেট কৰক।
3. plugin টো সেট কৰিবলৈ ধাপে ধাপে Wizard অনুসৰণ কৰক।
