# Gratis AI Agent

Gratis AI Agent হৈছে এটা স্বয়ংসম্পূর্ণ (autonomous) WordPress AI সহায়ক, যিয়ে সাধাৰণ ভাষাৰ নিৰ্দেশনাৰ জৰিয়তে সম্পূৰ্ণ WordPress সাইট পৰিকল্পনা কৰিব পাৰে, নিৰ্মাণ কৰিব পাৰে আৰু ব্যৱস্থাপনা কৰিব পাৰে। ইয়াৰ জৰিয়তে এটা সুসংগঠিত **ক্ষমতাসমূহ (abilities)**ৰ সেট প্ৰকাশ কৰে — যিবোৰ বিচ্ছিন্ন, পৰীক্ষা কৰিব পৰা কাৰ্যকলাপ যাৰ জৰিয়তে সহায়কজনৰ দ্বাৰা বিভিন্ন কাম কৰিব পাৰি। এই ক্ষমতাৰ পৰিসৰত কন্টেন্ট ব্যৱস্থাপনাৰ পৰা আৰম্ভ কৰি ডিজাইন ছিষ্টেম কাস্টমাইজেশ্যনলৈ সকলো ধৰণৰ কাম সামৰি লৈছে। Version 1.9.0 ত কন্টেন্ট সৃষ্টি কৰাৰ ক্ষমতা, ক্লায়েন্ট-সাইড স্ক্ৰিনশ্বটৰ জৰিয়তে ভিজুৱেল পেজ ৰিভিউ, পাঁচটা বিল্ট-ইন এজেণ্ট প্ৰোফাইল, আৰু অ্যাক্সেছ কণ্ট্ৰ'ল আৰু ব্ৰেণ্ডিংৰ বাবে ফিচাৰ ফ্লাগ যোগ কৰা হৈছে।

## মূল বৈশিষ্ট্যসমূহ (Key Features) {#key-features}

- **Abilities system** — এটা মডুলৰ, বৰ্ধিত কৰিব পৰা কাৰ্যকলাপৰ ক্যাটালগ, যাৰ জৰিয়তে সহায়কজনৰ দ্বাৰা আপোনাৰ WordPress installaționত কাম কৰিব পাৰি।
- **Content Management** — পোষ্ট আৰু পেজ সৃষ্টি কৰা (পেজ টেমপ্লেট নিৰ্ধাৰণ কৰাও সামৰি), এটা কলত বহু পোষ্ট একলগাকৈ সৃষ্টি কৰা, ফিচাৰ্ড ইমেজ সেট কৰা, আৰু কন্টাক্ট ফৰ্ম নিৰ্মাণ কৰা।
- **Visual Review** — লাইভ পেজৰ স্ক্ৰিনশ্বট লওঁ, পূৰ্বে আৰু বৰ্তমানৰ (before/after) অৱস্থাৰ পাৰ্থক্য দেখুৱাওঁ, আৰু লেআউট, টাইপ'গ্ৰাফি, ৰং, আৰু অ্যাক্সেছিবিলিটি সামৰি লৈ এটা AI-উৰ্জিত ডিজাইন ৰিভিউ লাভ কৰা।
- **পাঁচটা বিল্ট-ইন এজেণ্ট** — Content Writer, Site Builder, Design Studio, Plugin Manager, আৰু Support Assistant; যিটো চ্যাট হেডেৰত থকা Agent Pickerৰ জৰিয়তে সলনি কৰিব পাৰি।
- **Feature flags** — Settings → Feature Flags ত অ্যাক্সেছ কণ্ট্ৰ'ল আৰু ব্ৰেণ্ডিং টগল, যিটো ৰোল ৰেষ্ট্ৰিকচন, হোৱাইট-লেবেল নাম আৰু widget কাস্টমাইজেশ্যনৰ বাবে ব্যৱহাৰ কৰা হয়।
- **Custom Post Type management** — কাস্টম পোষ্ট টাইপ ৰেজিষ্টাৰ কৰা, তালিকাভুক্ত কৰা, আৰু ডিলিট কৰা, যিটো restart কৰাৰ পিছতো সংৰক্ষিত থাকে।
- **Custom Taxonomy management** — কাস্টম ট্যাক্সোনমি ৰেজিষ্টাৰ কৰা, তালিকাভুক্ত কৰা, আৰু ডিলিট কৰা, যিটো সংৰক্ষিত থাকে।
- **Design System abilities** — কাস্টম CSS ইনজেক্ট কৰা, কিউৰেটেড ব্লক প‍্যাটাৰ্ণ ব্যৱস্থাপনা কৰা, সাইট লোগো সেট কৰা, আৰু theme.json প্ৰিসেট লগোৱা।
- **Global Styles management** — কালৰ প‍্যালেট, টাইপ'গ্ৰাফি, স্পেসিং আদিৰ বাবে theme.json মান পঢ়া আৰু লিখা।
- **Navigation Menu management** — নেভিগেশ্যন মেনু আৰু ইয়াৰ আইটেমসমূহ প্ৰগ্ৰামাটিকভাৱে সৃষ্টি আৰু আপডেট কৰা।
- **Options Management** — WordPress options পঢ়া আৰু লিখা, যিটো গুৰুত্বপূৰ্ণ ছেটিংছক ৰক্ষা কৰিবলৈ এটা বিল্ট-ইন সেফটি ব্লকলিস্টৰ সৈতে।
- **Installable Abilities Registry** — মূল plugin ফাইলসমূহ সলনি নকৰাকৈ অধিক ক্ষমতাৰ প‍্যাক (ability packs) আৱিষ্কাৰ কৰা আৰু active কৰা।
- **Site Builder Orchestration v2** — plugin আৱিষ্কাৰ, প্ৰগ্ৰেছ ট্ৰেকিং, আৰু error recovery সহ বহু-ধাপৰ সাইট বিল্ড পৰিকল্পনা।
- **WP-CLI benchmark command** — CI আৰু model evaluationৰ বাবে কাণ্ডৰ পৰা সহায়কজনৰ ক্ষমতাৰ benchmark suite চলক।

## প্ৰয়োজনীয়তা (Requirements) {#requirements}

- WordPress 6.0 বা তাতকৈ অধিক
- PHP 8.1 বা তাতকৈ অধিক
- WordPress Multisite (network-activated) বা single-site installațion
- এটা active AI provider connector plugin (Anthropic, OpenAI, বা সামঞ্জস্যপূৰ্ণ)

## install কৰা (Installation) {#installation}

1. plugin ফাইলসমূহক `/wp-content/plugins/gratis-ai-agent` मा upload কৰক।
2. WordPressৰ **Plugins** স্ক্ৰীণৰ জৰিয়তে plugin টি activate কৰক।
3. **Gratis AI Agent → Settings** লৈ যোৱা আৰু আপোনাৰ AI provider সংযোগ কৰক।
4. (ঐচ্ছিক) অধিক ক্ষমতাৰ প‍্যাক install কৰিবলৈ **Abilities Registry** ब्राउজ কৰক।

## আৰম্ভণি (Quick Start) {#quick-start}

একবাৰ activate কৰা আৰু AI provider লগত সংযোগ কৰাৰ পিছত, WordPress admin sidebar ত **Gratis AI Agent** প‍্যানেলটো খুলক। এটা সাধাৰণ ভাষাৰ নিৰ্দেশনা টাইপ কৰক আৰু সহায়কজনৰ উপলব্ধ ক্ষমতা ব্যৱহাৰ কৰি প্ৰয়োজনীয় পদক্ষেপসমূহ পৰিকল্পনা কৰি কাৰ্যকৰী কৰিব।

**উদাহৰণৰ নিৰ্দেশনা:**

```
Create a "Portfolio" custom post type with a "Project Category" taxonomy.
```

```
Inject a dark-mode colour palette into the theme and set the primary colour to #1a1a2e.
```

```
Build a restaurant website with an online menu, booking form, and contact page.
```

## Abilities Overview {#abilities-overview}

Abilities হৈছে সেইবোৰ মৌলিক কাৰ্যকলাপ যি সহায়কজনৰ দ্বাৰা কৰিব পাৰি। প্ৰতিটো ability এটা register কৰা PHP class, যিয়ে এটা schema (parameters আৰু return types) প্ৰকাশ কৰে, যিটো সহায়কজনৰ দ্বাৰা runtime ত পঢ়ি লোৱা হয়। ক্ষমতা সমূহক বিভিন্ন কাৰ্যিক অঞ্চলত ভাগ কৰা হৈছে:

| Area | Abilities |
|---|---|
| **Content Management** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **Visual Review** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **Custom Post Types** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **Custom Taxonomies** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **Design System** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **Global Styles** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **Navigation Menus** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **Options Management** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **Installable Abilities** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **Site Builder** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

সম্পূৰ্ণ parameter documentation আৰু ব্যৱহাৰৰ উদাহৰণৰ বাবে [Abilities Reference](./abilities) চাওক।

## WP-CLI {#wp-cli}

Gratis AI Agent ত এটা `wp gratis-ai-agent` command family অন্তৰ্ভুক্ত কৰা হৈছে, যাৰ জৰিয়তে কাণ্ডৰ পৰা benchmark চলক আৰু abilities ব্যৱস্থাপনা কৰিব পাৰি। সম্পূৰ্ণ command তালিকাৰ বাবে [WP-CLI Reference](./wp-cli) চাওক।

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## Configuration {#configuration}

সকলো ছেটিংছ WordPress admin ত **Gratis AI Agent → Settings** ত উপলব্ধ:

| Setting | Description |
|---|---|
| **AI Provider** | active AI provider connector বাছি লওঁ আৰু configure কৰক |
| **Model** | সহায়কজনৰ সেশনৰ বাবে ব্যৱহাৰ কৰিবলগীয়া ভাষা মডেল বাছি লওঁ |
| **Max Tokens** | প্ৰতি-অনুগ্ৰহ (per-request) token budget (default 4096) |
| **Ability Timeout** | এটা individual ability কিমান সময়লৈ (seconds) চলিব পাৰে (default 30) |
| **Debug Logging** | ability কল আৰু সহায়কজনৰ যুক্তি (reasoning) WordPress debug log ত লগ কৰক |

## Options Management Safety Blocklist {#options-management-safety-blocklist}

Options Management ability ত এটা বিল্ট-ইন blocklist আছে, যিয়ে সহায়কজনক সংবেদনশীল WordPress options পঢ়া বা লিখিব নিদিয়ে। ডিফল্ট blocklist ত তলত দিয়াবোৰ সামৰি লৈছে:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (URL options — এইবোৰৰ বাবে এটা ডেডিকেটেড migration tool ব্যৱহাৰ কৰক)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (secret keys)
- `active_plugins`, `template` (plugin/theme activation — ডেডিকেটেড abilities ব্যৱহাৰ কৰক)
- Payment gateway credentials আৰু API keys যিবোটা plugin লৈ install কৰা হৈছে।

অতিরিক্ত এণ্ট্ৰীসমূহ `gratis_ai_agent_options_blocklist` filterৰ জৰিয়তে যোগ কৰিব পাৰি।

## Hooks and Filters {#hooks-and-filters}

### Actions {#actions}

- `gratis_ai_agent_ability_registered` — এটা ability register হোৱাৰ পিছত trigger হয়, ability class name লাভ কৰে।
- `gratis_ai_agent_plan_started` — এটা site builder plan execute হ'বলৈ আৰম্ভ কৰাৰ সময়ত trigger হয়।
- `gratis_ai_agent_plan_completed` — এটা plan সফলভাৱে শেষ হোৱাৰ সময়ত trigger হয়।
- `gratis_ai_agent_plan_failed` — এটা plan ক কোনো recover কৰিব নোৱাৰা error হোৱাৰ সময়ত trigger হয়।

### Filters {#filters}

- `gratis_ai_agent_abilities` — সহায়কজনক load কৰাৰ আগতে register কৰা ability list মডিফাই কৰা।
- `gratis_ai_agent_options_blocklist` — options safety blocklist বৃদ্ধি কৰা বা হ্ৰাস কৰা।
- `gratis_ai_agent_benchmark_questions` — evaluation suite ত কাস্টম benchmark প্ৰশ্ন যোগ কৰা।
- `gratis_ai_agent_plan_plugins` — site plan generationৰ সময়ত plugin আৱিষ্কাৰ তালিকা মডিফাই কৰা।
