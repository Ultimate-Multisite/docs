১. অ্যাডন ফাইলবোৰ আপোনাৰ `/wp-content/plugins/` ডাইৰেক্টৰীত আপলোড কৰক।

এই অ্যাডনটোৱে এটা বিস্তৃত টেস্টিং framework অন্তৰ্ভুক্ত কৰে:

- **Base Test Class**: সাধাৰণ টেস্টিং scenario-ৰ বাবে helper methods যোগান ধৰে।
- **Test Coverage**: code coverage ট্ৰেক কৰিবলৈ configure কৰা হৈছে।
- **Multisite Testing**: টেস্টিংবোৰ multisite environment ত চলি।

### Actions {#actions}

- `ultimate-multisite-ai-site-builder_init` - অ্যাডন initialize হোৱাৰ পাছত trigger হয়।
- `ultimate-multisite-ai-site-builder_loaded` - সকলো অ্যাডন ফাইল load হোৱাৰ পাছত trigger হয়।

### Filters {#filters}

- `ultimate-multisite-ai-site-builder_settings` - অ্যাডনৰ settings modify কৰিব পাৰে।
- `ultimate-multisite-ai-site-builder_enabled` - অ্যাডন enable/disable status-ক override কৰিব পাৰে।
