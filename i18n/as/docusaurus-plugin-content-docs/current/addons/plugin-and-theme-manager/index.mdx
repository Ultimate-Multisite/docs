# Plugin আৰু Theme Manager {#plugin--theme-manager}

Multisite Ultimate: Plugin আৰু Theme Manager

install কৰা plugin আৰু theme-ৰ মেটা তথ্য (title, author, thumbnail আৰু description) সম্পাদনা কৰক বা লুকুৱাই ৰাখক। আপোনাৰ ব্যৱহাৰকাৰীসকলক ফিল্টাৰ কৰিবলৈ plugin আৰু theme-ৰ বাবে কেটাগৰী (categories) সৃষ্টি কৰক আৰু আপোনাৰ ব্যৱহাৰকাৰীৰ প্যানেলত এটা ধুনীয়া কাস্টম Plugins পৃষ্ঠা যোগ কৰক।

এই plugin ব্যৱহাৰ কৰিবলৈ, আপুনি কেৱল নিজৰ network admin-ৰ Plugins বা Themes পৃষ্ঠালৈ যোৱা আৰু "Edit Plugin/Theme info" मा ক্লিক কৰিব লাগিব।

## Installation (ইনষ্টলেচন) {#installation}

1. 'ultimate-multisite-plugin-and-theme-manager' ফাইলটো '/wp-content/plugins/' ডাইৰেক্টৰীত আপলোড কৰক।
2. আপোনাৰ WordPress Network Admin-ৰ 'Plugins' মেনুৰ জৰিয়তে plugin টো Activate কৰক।

## Usage Guide (ব্যৱহাৰৰ নির্দেশিকা) {#usage-guide}

Ultimate Multisite: Plugin and Theme Manager হৈছে এটা শক্তিশালী **premium** Ultimate Multisite add-on, যিয়ে আপোনাৰ network-ৰ ব্যৱহাৰকাৰীসকলে plugin আৰু theme অপচনবোৰ কেনেকৈ দেখিব আৰু কেনেকৈ ব্যৱহাৰ কৰিব, তাৰ ওপৰত সম্পূৰ্ণ নিয়ন্ত্ৰণ দিয়ে।

এই add-on-এ তিনিটা কাম কৰে:

  1. ই আপোনাক plugin আৰু theme দুয়োটাৰ মেটা তথ্য সম্পূৰ্ণৰূপে সলনি কৰিবলৈ দিয়ে। ইয়াৰ ভিতৰত নাম, description, author নাম, আৰু thumbnail-ও অন্তৰ্ভুক্ত!

  2. ই আপোনাক plugin আৰু theme-ক কেটাগৰী কৰিবলৈ দিয়ে, যাতে আপোনাৰ ব্যৱহাৰকাৰীসকলে ইয়াক সহজে সৰ্বগঠন (sort) কৰিব পাৰে।

  3. ই আপোনাক পুৰণি, একঘেয়ে plugins পৃষ্ঠাটোক কেটাগৰী অনুসৰি সৰ্বগঠন কৰা এটা নতুন themes-ৰ দৰে পৃষ্ঠাৰে সলনি কৰাৰ বিকল্প দিয়ে।

ই ডাউনলোড আৰু install কৰাৰ পিছত, Ultimate Multisite Settings পৃষ্ঠাখনত এটা নতুন ট্ৰেব - _Add-on Settings_ - দেখুৱাব।
