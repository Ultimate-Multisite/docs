# Filter: wu_cap_captcha_config

Cap Captcha কনফিগাৰেচন ফিল্টৰ কৰে।

এই ফিল্টৰটোৱে ডেভেলপাৰসকলক Cap Captcha ছেটিংছ কাস্টমাইজ কৰিবলৈ অনুমতি দিয়ে।

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $config | `array` | Cap কনফিগাৰেচন আৰেই। |
| $level | `string` | সুৰক্ষাৰ স্তৰ (fast, medium, max)। |
