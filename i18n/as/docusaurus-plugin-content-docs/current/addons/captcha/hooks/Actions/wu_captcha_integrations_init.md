# Action: wu_captcha_integrations_init {#action-wucaptchaintegrationsinit}

সকলো ইন্টিগ্ৰেচন (integrations) ইনিশিয়ালাইজ (initialized) হোৱাৰ পিছত এই অ্যাকশনটো (action) ট্ৰিগাৰ হয়।

ট্ৰিড-পাৰ্টি ইন্টিগ্ৰেচনहरू (Third-party integrations) ইয়াত হুক (hook) কৰি নিজৰ ইন্টিগ্ৰেচন যোগ কৰিব পাৰে।

- 1.2.0
