# Loco Translate

Ultimate Multisite: Loco Translate Integration addon ले আপোনাৰ WordPress multisite network-লৈ site-specific translation capability লৈ আহে। যদি আপুনি কেতিয়াবা অনুভৱ কৰিছে যে আপোনাৰ network-ৰ বিভিন্ন site-লৈ এটা একেটা plugin বা theme-ৰ বিভিন্ন translation হ'ব লাগিব, তেন্তে এই addon টো এটা উপযুক্ত সমাধান।

**এই Addon টো কি কৰে**

এই addon টো Ultimate Multisite-ক জনপ্রিয় Loco Translate plugin লগত সহজে সংযোগ কৰে, যাৰ ফলত আপোনাৰ network-ৰ প্ৰতিটো subsite-য়ে নিজৰ নিজস্ব custom translation ৰখাৰ সুবিধা পায়। আপুনি যদি এটা multi-regional platform চলাইছে, বিভিন্ন ভাষাৰ কমিউনিটিক সেৱা আগবঢ়াইছে, বা কেৱল আপোনাৰ network-ত translation flexibilityৰ প্ৰয়োজন, তেন্তে এই addon টোৱে এই কামটো অতি সহজ কৰি তোলে।

**কৰণীয়:**

* বিভিন্ন localization প্ৰয়োজন থকা multi-regional business
* বৈচিত্ৰ্যময় কমিউনিটিক সেৱা আগবঢ়োৱা শিক্ষামূলক প্ৰতিষ্ঠান
* বহুভাষিক সমৰ্থন (multilingual support) থকা SaaS platform
* বহু client site সংস্থাপন কৰা web agency
* translation flexibilityৰ প্ৰয়োজন থকা যিকোনো multisite network

## Installation

1. নিশ্চিত কৰক যে Ultimate Multisite install আৰু network activate হৈ আছে।
2. Loco Translate plugin install আৰু activate কৰক।
3. এই addon টো আপোনাৰ plugins directory मा upload কৰক।
4. Loco Translate Integration addon টো network-level-ত activate কৰক।
5. যিকোনো subsite-ৰ ওপৰত Loco Translate ব্যৱহাৰ কৰি site-specific translation বনাওঁ আৰম্ভ কৰক।

এইটোহে! কোনো এটা অতিৰিক্ত configuration নহয়, এই integration টো স্বয়ংক্রিয়ভাৱে কাম কৰে।

## Frequently Asked Questions

## কি লাগিছে install কৰাৰ পিছত কোনো configuration কৰিব লাগিবনে?

না! এই integration টো স্বয়ংক্রিয়। addon আৰু Loco Translate দুয়োটাই active হোৱাৰ লগে লগে, subsite-সমূহে নিজৰ custom translation বনাওঁ আৰম্ভ কৰিব পাৰে।

## যদি কোনো subsite-য়ে custom translation ননাওঁ কৰে भने কি হ'ব?

site টো স্বয়ংক্রিয়ভাৱে network-wide translation ব্যৱহাৰ কৰিব (fallback)। কোনো ধৰণৰ সমস্যা নহয়, আৰু সকলো সহজে কাম কৰে।

## subsite-সমূহে network-wide translation ব্যৱহাৰ কৰিব পাৰেনে?

সাধাৰণ site admin-সকলে কেৱল নিজৰ site-ৰ translationহে দেখা আৰু সম্পাদনা কৰিব পাৰে। Super admin-সকলে তেওঁ কোনটো site চাই আছে তাৰ ওপৰত নিৰ্ভৰ কৰি network-wide আৰু site-specific দুয়োটা translation ব্যৱহাৰ কৰিব পাৰে।

## ই plugin আৰু theme দুয়োটাতে কাম কৰে নে?

হয়! Site-specific translation WordPress plugin আৰু theme দুয়োটাতে কাম কৰে।

## ই মোৰ multisite network-ক ধীৰ কৰি তুলিবনে?

না। এই addon টো efficient file loading ব্যৱহাৰ কৰে, যিয়ে কেৱল প্ৰয়োজন হোৱাৰ সময়তে site-specific translation-ৰ বাবে check কৰে। ইয়াৰ কোনো লক্ষণীয় performance impact নাই।

## মই network translation-ক এটা starting point হিচাপে এটা specific site-লৈ copy কৰিব পাৰেনে?

হয়! Loco Translate-ত translation সম্পাদনা কৰোঁতে, আপুনি সহজে network translation লৈ আৰম্ভ কৰি প্ৰতিটো site-ৰ বাবে ইয়াক customize কৰিব পাৰে।

## ই WooCommerce আৰু আন plugin-সমূহত কাম কৰে নে?

নিশ্চয়! যিকোনো plugin বা theme যিয়ে standard WordPress translation functions ব্যৱহাৰ কৰে, সেইবোৰ site-specific translation-ৰ সৈতে কাম কৰিব।
