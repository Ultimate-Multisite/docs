# PayFast

Multisite Ultimate: Payfast Gateway

ই South Africa-ৰ এটা দ্ৰুত আৰু সুৰক্ষিত অনলাইন পেমেন্ট গেটৱে (online payments gateway) – Payfast – लाई Multisite Ultimate-ৰ এটা পেমেন্ট বিকল্প হিচাপে যোগ কৰে।

## স্থাপন (Installation) {#installation}

১. 'ultimate-multisite-payfast' ফাইলটো '/wp-content/plugins/' ডাইৰেক্টৰিটোত আপলোড কৰক।
২. WordPress-ৰ 'Plugins' মেনুৰ মাজেৰে plugin টো Activate কৰক।
৩. plugin টো সেট কৰিবলৈ ধাপে ধাপে Wizard অনুসৰণ কৰক।

## ব্যৱহাৰৰ নির্দেশিকা (Usage Guide) {#usage-guide}

PayFast হৈছে South Africa-ৰ ব্যৱসায়-ব্যৱসায়ৰ বাবে এটা প্ৰമുഖ অনলাইন পেমেন্ট প্ৰচেছৰ। ই অনলাইন ভ্ৰেচকলৈ (merchants) লেনদেনক সহজ আৰু সুৰক্ষিত কৰি তেওঁলোকক ডাঙৰকাবলৈ নিবেদিত। Ultimate Multisite ব্যৱহাৰকাৰীৰ অনুৰোধ মৰ্মে, আমি এটা add-on সৃষ্টি কৰাৰ সিদ্ধান্ত লৈছো, যাৰ দ্বাৰা PayFast লগত ইন্টিগ্ৰেট কৰিব পৰা যাব। এই article-টোৱে দেখুৱাই যে আপোনাৰ Ultimate Multisite network-ত পেমেন্ট গ্ৰহণ কৰিবলৈ PayFast কেনেকৈ এটা পেমেন্ট গেটৱে হিচাপে সেট কৰিব।

প্ৰথম পদক্ষেপটো হ'ল add-on টো install কৰা।

# Payfast add-on কেনেকৈ install কৰিব:

add-ons page मा যাওক আৰু Ultimate Multisite: Payfast Gateway add-on টো select কৰক।

![Add-ons page](/img/admin/addons-page.png)

add-on উইন্ডোটা খুলি আহিব। 'Install Now' मा ক্লিক কৰক।

![Add-ons page with install option](/img/admin/addons-page.png)

install কৰা শেষ হোৱাৰ পিছত, আপোনাক plugins page मा redirect কৰা হ'ব। ইয়াত, কেৱল 'Network Activate' मा ক্লিক কৰক আৰু _Payfast_ add-on টো আপোনাৰ network मा activate হৈ যাব।

# PayFast সেট কৰা (Setting Up PayFast)

add-on install কৰাৰ পিছত, আপুনি এতিয়া Ultimate Multisite settings मा গৈ PayFast কনফিগাৰ কৰিব পাৰে।

**Ultimate Multisite settings** page-ৰ তলত, আপুনি "Payments" ট্যাবটো বিচাৰি পাব। এই ট্যাবৰ তলত, আপুনি "Payfast" আৰু এই পেমেন্ট গেটৱেটো enable বা disable কৰিবলৈ এটা toggle button পাব।

![Payment gateway settings](/img/config/settings-payment-gateways.png)আগুৱাই গৈ toggle button টো ক্লিক কৰি ইয়াক enable কৰক। তাৰ পিছত, ইয়াৰ ভিতৰত বিভিন্ন field খুলি আহিব, যিবোৰ আপুনি ভৰাব লাগিব, যেনে **Mechant ID, Merchant Key, আৰু PassPhrase।** আপুনি এই তথ্যবোৰ আপোনাৰ **Payfast** account बाट পাব।

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

পৰৱৰ্তী পদক্ষেপটো হ'ল আপোনাৰ PayFast account লগত login কৰা, যাতে আপোনাক প্ৰয়োজনীয় তথ্য পোৱা যায়।

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->আপুনি কেৱল credential টো copy আৰু paste কৰি আপোনাৰ Ultimate Multisite settings मा details ভৰাব পাৰে। তাৰ পিছত, পৰিৱৰ্তনসমূহ সেভ কৰিবলৈ **"Save settings** " button টো ক্লিক কৰক।
