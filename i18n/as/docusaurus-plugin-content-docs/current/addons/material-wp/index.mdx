# Material WP

WordPress-ৰ admin interface বহু দিনৰ পৰা একেই আছে। কিন্তু আপুনি কেতিয়াবা ভাবিছেনে যে Material Design-ৰ এটা admin interface কেনে হ'ব?

Material WP হৈছে এটা admin theme যি আপোনাক সেই look and feel দিবলৈ তৈয়াৰ কৰা হৈছে, যাৰ জৰিয়তে আপুনি আপোনাৰ clients-লৈ এটা বেছি customize কৰা Dashboard দিব পাৰে।

## বৈশিষ্ট্যসমূহ

- Custom CSS সমৰ্থন
- সহজ styling সেটিংছ
- মেনু পুনৰ্গঠন (Menu reordering)
- সহজ import/export

## ডকুমেন্টেশ্যন

**Material WP ব্যৱহাৰ কৰি আপোনাৰ Admin Panel customize কৰক**

## স্থাপন

1. 'material-wp' ফাইলটো '/wp-content/plugins/' ডাইৰেক্টৰীত upload কৰক
2. WordPress-ৰ 'Plugins' মেনুৰ জৰিয়তে plugin টো activate কৰক

## ব্যৱহাৰৰ নির্দেশিকা

WordPress-ৰ admin interface বহু দিনৰ পৰা একেই আছে। কিন্তু আপুনি কেতিয়াবা ভাবিছেনে যে Material Design-ৰ এটা admin interface কেনে হ'ব?

Material WP হৈছে এটা admin theme যি আপোনাক সেই look and feeling দিবলৈ তৈয়াৰ কৰা হৈছে, যাৰ জৰিয়তে আপুনি আপোনাৰ clients-লৈ এটা বেছি customize কৰা Dashboard দিব পাৰে। Material WP হৈছে Ultimate Multisite-ৰ premium plugins core-ৰো এটা অংশ।
