# Filter: ultimate_multisite_content_sync_url

ডেভেলপমেন্টৰ বাবে প্লাগইনৰ মূল স্থান (plugin root) লৈ আপেক্ষিক পথ (relative path) ব্যৱস্থাপনা কৰিবলৈ অনুমতি দিয়ে।

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $dir | `string` | প্লাগইনৰ মূল স্থানৰ লগত আপুনি যি পথ (path) অ্যাক্সেচ কৰিব বিচাৰে। |

### Since

- 1.0.0
### Source

[`inc/class-helper.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-content-sync/blob/main/inc/class-helper.php#L52) ত ৫২ নং শাৰীত সংজ্ঞায়িত কৰা হৈছে।


## Returns
প্লাগইনৰ মূল স্থানৰ লগত আপেক্ষিক পথ।
