# কন্টেন্ট Sync

Multisite Ultimate: Content Sync

Multisite Ultimate-ৰ বাবে এই Addon-টোৱে নেটৱৰ্ক প্রশাসকসকলক (network administrators) এটা সাইটৰ কন্টেন্ট আন সাইটলৈ সিনক্ৰনাইজ কৰাত সহায় কৰে। ই পোষ্ট, পেজ, কাস্টম পোষ্ট প্ৰকাৰ (custom post types), আৰু Elementor টেমপ্লেট সহ সকলো পোষ্ট প্ৰকাৰক সমৰ্থন কৰে। এই Addon-টোৱে কন্টেন্ট বাছনি কৰা আৰু লক্ষ্য সাইটসমূহ (target sites) নিৰ্বাচন কৰিবলৈ এটা বহু-ধাপীয়া উইজৰ্ড interfase, ছেচ প্ৰচেছিং (batch processing), আৰু ৰিয়েল-টাইম প্ৰগ্ৰেছ ফিডবেকৰ সুবিধা দিয়ে।

## বৈশিষ্ট্যসমূহ (Features) {#features}

* কন্টেন্ট বাছনি: থাম্বনেইল আৰু পোষ্ট প্ৰকাৰ ফিল্টাৰিং সহ ভিজুৱেল কাৰ্ড গ্ৰিড।
* সকলো পোষ্ট প্ৰকাৰ: পোষ্ট, পেজ, প্ৰডাক্ট, Elementor টেমপ্লেট, আৰু যিকোনো কাস্টম পোষ্ট প্ৰকাৰ সিনক্ৰনাইজ কৰিব পাৰে।
* সাইট বাছনি: কন্টেন্ট কি সাইটলৈ সিনক্ৰনাইজ কৰিব লাগে, সেয়া বাছনি কৰিব পাৰে। ইয়াত উৎস সাইট (source site) বাদ দিয়া আৰু চাইল্ড সাইট (child site) আগতে বাছনি কৰাৰ সুবিধা আছে।
* ছেচ প্ৰচেছিং: এটা সময়ত ১০ টা সাইটলৈ সিনক্ৰনাইজ কৰে আৰু প্ৰগ্ৰেছ আপডেট দিয়ে।
* মিডিয়া সিনক্ৰনাইজ: পোষ্ট কন্টেন্টৰ পৰা মিডিয়া ফাইলসমূহ (ছবি, ভিডিঅ') স্বয়ংক্ৰিয়ভাৱে কপি কৰে আৰু লক্ষ্য সাইটলৈ পুনৰ স্থান নিৰ্ধাৰণ কৰে (remaps)।
* ট্যাক্সোনমি সিনক্ৰনাইজ: কেটাগৰী, ট্যাগ, আৰু কাস্টম ট্যাক্সোনমি সংৰক্ষিত কৰি ৰাখে।
* Elementor সমৰ্থন: CSS পুনৰ সৃষ্টি (regeneration) আৰু টেমপ্লেট ID পুনৰ স্থান নিৰ্ধাৰণ (remapping) সহ Elementor টেমপ্লেটৰ সম্পূৰ্ণ সমৰ্থন।
* সংঘাত সমাধান (Conflict Resolution): বিদ্যমান কন্টেন্ট সলনি কৰিব নে এয়া এৰি যাব, তাৰ বিকল্প।
* কেছ ব্যৱস্থাপনা (Cache Management): সিনক্ৰনাইজ হোৱাৰ পিছত লক্ষ্য সাইটসমূহত Elementor কেছ স্বয়ংক্ৰিয়ভাৱে মুচি দিয়ে।

## প্ৰয়োজনীয়তা (Requirements) {#requirements}

* WordPress 5.3 বা তাতকৈ অধিক
* PHP 7.4 বা তাতকৈ অধিক
* Multisite Ultimate plugin (সক্ৰিয়)
* Elementor plugin (ঐচ্ছিক, Elementor টেমপ্লেট সিনক্ৰনাইজ কৰিবলৈ প্ৰয়োজনীয়)

## স্থাপন (Installation) {#installation}

1. 'ultimate-multisite-content-sync' ফাইলটো '/wp-content/plugins/' ডাইৰেক্টৰীত আপলোড কৰক।
2. আপোনাৰ WordPress Network Admin-ৰ 'Plugins' মেনুৰ মাজেৰে plugin টো Activate কৰক।
3. Ultimate Multisite → Settings → Content Sync ত সেটিংস কনফিগাৰ কৰক।
4. সিনক্ৰনাইজ কৰিবলৈ Ultimate Multisite → Content Sync লৈ যাওক।
