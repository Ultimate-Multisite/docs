# মাল্টি-নেটৱৰ্ক চেঞ্জলগ

### 1.3.0
* নতুন: নেটৱৰ্ক টেমপ্লেট প্ৰিভিউৰ — ক্ৰয় কৰাৰ আগতে লাইভ প্যানেলৰ সৈতে নেটৱৰ্ক টেমপ্লেট চাওঁক আৰু প্ৰিভিউ কৰক।
* নতুন: চেকাউটে নেটৱৰ্ক টেমপ্লেট পিকাৰ — গ্রাহকসকলে সাইনআপ চেকাউটে নেটৱৰ্ক টেমপ্লেট বাছি ল'ব পাৰিব।
* নতুন: ক্ৰ'ছ-নেটৱৰ্ক মেজিক লিংক SSO — গ্রাহকসকলে মেজিক-লিংক অথেন্টিকেচনৰ জৰিয়তে সহজে উপ-নেটৱৰ্কসমূহলৈ প্ৰৱেশ কৰিব পাৰিব।
* নতুন: নেটৱৰ্ক মিডিয়া ক্লোনিং — টেমপ্লেটৰ পৰা নেটৱৰ্ক ক্লোন কৰোঁতে সাইট মিডিয়া সঠিকভাৱে ডুপ্লিকেট হয়।
* সংশোধন: মাল্টি-টেনেন্সী এড-অন নোহোৱাকৈও এতিয়া নেটৱৰ্ক ক্লোনিং কৰিব পাৰি; ক্লোন কৰাৰ পিছত ছুপাৰ এডমিন প্ৰৱilegিয়া পুনৰ দিয়া হয়।
* সংশোধন: নেটৱৰ্ক সৃষ্টি কৰাটো এতিয়া WooCommerce আৰু সকলো পেমেন্ট গেটৱেৰৰ সৈতে সামঞ্জস্যপূৰ্ণ।
* সংশোধন: ফলব্যাক ডোমেইন জেনারেচনত এতিয়া প’ৰ্ট নম্বৰো অন্তৰ্ভুক্ত কৰা হৈছে।
* সংশোধন: টেমপ্লেট পিকাৰ CSS সকলো থীমাত সঠিক ৰেণ্ডাৰিংৰ বাবে এটা এংকুৱেড ষ্টাইলশীটলৈ স্থানান্তৰ কৰা হৈছে।
* সংশোধন: সাইট কোৱাৰী স্কোপত undefined `network_id` কি চিকাৰ কৰা হৈছে।
* সংশোধন: কনটেক্সট সলনি কৰাৰ আগতে নেটৱৰ্ক অনুমতি পৰীক্ষা কৰিলে `wpdb` টেবুল প্ৰ’পৰ্টিৰ ভুল নহয়।
