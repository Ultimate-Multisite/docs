# Multi-Network {#multi-network}

Ultimate Multisite: Multinetwork addon ले আপোনাৰ Ultimate Multisite platfrom-ক শক্তিশালী multi-network functionality লৈ বঢ়াই দিয়ে। এই addon-এ আপোনাৰ কাস্টমৰসকলক কেৱল একক site ব্যৱস্থাপনা কৰাই নহয়, সৰু-বৰ সকলো WordPress multisite network ক ক্ৰয় কৰিবলৈ আৰু ব্যৱস্থাপনা কৰিবলৈ সক্ষম কৰে।

## মূল বৈশিষ্ট্যসমূহ (Key Features) {#key-features}

* **Network Provisioning**: আপোনাৰ কাস্টমৰসকলৰ বাবে সম্পূৰ্ণ WordPress multisite network স্বয়ংক্রিয়ভাৱে সৃষ্টি কৰা আৰু কনফিগাৰ কৰা।
* **Network Templates**: কাস্টমৰসকলক সাইনআপ কৰাৰ আগতে বিভিন্ন প্ৰস্তুত network template চাবলৈ, প্ৰিভিউ কৰিবলৈ আৰু বাছি ল'বলৈ সুযোগ দিয়ে।
* **Network Management**: multi-network environment-ৰ বাবে সম্পূৰ্ণ ব্যৱস্থাপনা ক্ষমতা।
* **Isolated Networks**: প্ৰতিজন কাস্টমৰ network নিজৰ ডেটাবেছ টেবুল আৰু কনফিগাৰেটচন লৈ স্বাধীনভাৱে কাম কৰে।
* **Cross-Network SSO**: magic-link authentication ব্যৱহাৰ কৰি কাস্টমৰসকলক তেওঁলোকৰ sub-network সমূহলৈ সহজে প্ৰৱেশ কৰাৰ সুবিধা দিয়ে।
* **Scalable Architecture**: অপটিমাইজড resource allocation লৈ একাধিক network ক দক্ষতাৰে সামৰি লয়।
* **Integration**: Ultimate Multisite-ৰ subscription আৰু billing system লগত সহজে সংমিশ্ৰণ কৰে।

## ব্যৱহাৰৰ ক্ষেত্ৰ (Use Cases) {#use-cases}

* **White Label Hosting**: কাস্টমৰসকলক তেওঁলোকৰ নিজা ব্ৰেণ্ডৰ multisite network প্ৰদান কৰা।
* **Agency Solutions**: এজেঞ্চি সকলক এটা মাত্ৰ platfromৰ পৰা একাধিক ক্লায়েন্টৰ network ব্যৱস্থাপনা কৰাত সক্ষম কৰা।
* **Enterprise Deployments**: জটিল multi-network প্ৰয়োজনীয়তা থকা বৃহৎ সংস্থাসমূহক সমৰ্থন কৰা।
* **Education Platforms**: বিভিন্ন স্কুল বা বিভাগৰ বাবে পৃথক network সৃষ্টি কৰা।

## প্ৰয়োজনীয়তা (Requirements) {#requirements}

* Ultimate Multisite 2.0.0 বা তাতকৈ অধিক
* WordPress Multisite installation
* PHP 7.4 বা তাতকৈ অধিক
* MySQL 5.6 বা তাতকৈ অধিক

## স্থাপন (Installation) {#installation}

1. প্লাগইন ফাইলসমূহ `/wp-content/plugins/ultimate-multisite-multinetwork` ডাইৰেক্টৰীত আপলোড কৰক।
2. WordPress-ৰ 'Plugins' স্ক্ৰীণৰ মাজেৰে প্লাগইনটো active কৰক।
3. Ultimate Multisite → Settings মাজেৰে multi-network setting কনফিগাৰ কৰক।

## Network Templates {#network-templates}

Version 1.3.0 ले নতুন network ক্ৰয় কৰা কাস্টমৰসকলৰ বাবে এটা template discovery আৰু selection flow যোগ কৰিছে। যেতিয়া আপুনি বিচাৰে যে প্ৰতিজন নতুন কাস্টমৰ network এটা প্ৰস্তুত কাঠামো, ডিজাইন, কন্টেন্ট লাইব্ৰেৰী, বা vertical-specific setup লৈ আৰম্ভ কৰক, তেতিয়া network template ব্যৱহাৰ কৰক।

### Template previewer {#template-previewer}

এই network template previewer-এ কাস্টমৰসকলক checkout কৰাৰ আগতে উপলব্ধ template সমূহ চাবলৈ দিয়ে। প্ৰতিটো template এটা live preview panel-ত খোলা যায়, যাৰ ফলত কাস্টমৰসকলে network ক্ৰয় কৰাৰ আগতে layout আৰু কন্টেন্টৰ তুলনাকৰণ কৰিব পাৰে।

কাস্টমৰসকলক সহায় কৰিবলৈ previewer ব্যৱহাৰ কৰক:

* উপলব্ধ প্ৰতিটো network template-ৰ look আৰু structure পৰীক্ষা কৰিবলৈ।
* ক্ৰয় প্ৰবাহ (purchase flow) এৰি নোহোৱাকৈ একাধিক template ৰ তুলনাকৰণ কৰিবলৈ।
* সাইনআপ কৰাৰ আগতে তেওঁলোকৰ ব্যৱহাৰৰ ক্ষেত্ৰৰ সৈতে খাপ খোৱা template বাছি ল'বলৈ।

template নাম আৰু বৰ্ণনা কাস্টমৰ-বান্ধৱী (customer-friendly) ৰাখক। এই previewer-এ আটাইতকৈ বেছি উপকাৰী হয় যেতিয়া কাস্টমৰসকলে ইয়াৰ উদ্দেশ্য ব্যৱহাৰৰ ক্ষেত্ৰ (intended use case) বুজিব পাৰে, যেনে: এজেঞ্চি network, স্কুল network, franchise network, বা white-label hosting bundle।

### Checkout-ত Template picker {#template-picker-at-checkout}

template picker-ক signup checkout flow-ত এটা field হিচাপে যোগ কৰিব পাৰি। কনফিগাৰ কৰাৰ লগে লগে, কাস্টমৰসকলে checkout কৰাৰ সময়ত তেওঁলোকে বিচৰা network template বাছি লয়, আৰু নতুন network provision কৰাৰ সময়ত সেই বাছনি কৰা template টো ব্যৱহাৰ কৰা হয়।

checkout flow-ত picker ব্যৱহাৰ কৰিবলৈ:

1. যি network সমূহক template হিচাপে অফাৰ কৰিব বিচাৰে, সেয়া প্ৰস্তুত কৰক।
2. প্ৰাসংগিক signup বা checkout form-ত Multi-Network template selection field-টো enable কৰক।
3. কোন কোন template এই checkout flow-ৰ বাবে উপলব্ধ হ'ব, সেয়া বাছি লওক।
4. checkout flow publish কৰক আৰু এটা signup কৰি পৰীক্ষা কৰক যে বাছনি কৰা template টো নতুন network-লৈ প্ৰয়োগ হৈছে নেকি।

যদি এটা checkout flow-এ মাত্ৰ এটা network product অফাৰ কৰে, তেন্তে picker-এ এতিয়াও এটা guided onboarding step হিচাপে উপকাৰী হ'ব পাৰে। multi-plan funnels-ৰ বাবে, কেৱল সেই template সমূহহে প্ৰদৰ্শিত কৰক যিবোৰ বাছনি কৰা product বা plan লগত সামঞ্জস্যপূৰ্ণ।

## Cross-Network Magic Link SSO {#cross-network-magic-link-sso}

Cross-network magic link SSO কাস্টমৰসকলক তেওঁলোকৰ প্ৰাথমিক account area আৰু sub-network সমূহৰ মাজত ম্যানুৱেলভাৱে লগইন নকৰাকৈ যাবলৈ দিয়ে। যেতিয়া কোনো কাস্টমৰে এটা cross-network access link অনুসৰণ কৰে, Ultimate Multisite এটা সময়-সীমাবদ্ধ magic link সৃষ্টি কৰে আৰু কাস্টমৰক গন্তব্য network-ত authenticate কৰে।

এনে ব্যৱস্থা কৰিলে সেই কাস্টমৰসকলৰ অভিজ্ঞতা উন্নত হয় যিয়ে এটাৰ অধিক network ব্যৱস্থাপনা কৰে বা ক্ৰয় কৰাৰ ঠিক পিছতে নতুনকৈ clone কৰা template network সমূহলৈ প্ৰৱেশ কৰিবলগীয়া হয়।

Admin checklist:

* নিশ্চিত কৰক যে WordPress multisite installation সমগ্ৰ জুড়ে Multi-Network addon active আছে।
* নিশ্চিত কৰক যে কাস্টমৰসকলক cross-network link দিয়াৰ আগতে সঠিক network access আছে।
* network domain আৰু mapped domain সঠিকভাৱে কনফিগাৰ কৰক যাতে magic link টো কাৰ কাৰণে গন্তব্য network লৈ যায়।
* network, domain, বা checkout setting সলনি কৰাৰ পিছত কাস্টমৰ dashboard আৰু যিকোনো post-checkout access button পৰীক্ষা কৰক।

Magic link সমূহক লগইন link হিচাপে গণ্য কৰিব লাগে। ইয়াৰ সৃষ্টি কৰা হৈছে সহজ কাস্টমৰ প্ৰৱেশৰ বাবে আৰু ইয়াক উদ্দেশ্যিত session বাহিৰত forward বা reuse কৰিব নালাগে।

## সঘনে কৰা প্ৰশ্নসমূহ (Frequently Asked Questions) {#frequently-asked-questions}

### multi-network কি? {#what-is-multinetwork}

Multi-network হৈছে এটা WordPress feature যিয়ে আপোনাক এটা মাত্ৰ WordPress installation ওপৰত একাধিক স্বাধীন multisite network চলাই দিবলৈ দিয়ে।

### ই সাধাৰণ multisite ৰ পৰা কেনেকৈ বেলেগ? {#how-is-this-different-from-regular-multisite}

সাধাৰণ multisite-এ এটা networkৰ ভিতৰত একাধিক site সক্ষম কৰে। Multi-network-এ একাধিক network সক্ষম কৰে, যিটো প্ৰতিটোৰ নিজৰ site থাকে, আৰু সকলোৱে এটা WordPress installationৰ ভিতৰত থাকে।

### ইয়াৰ বাবে বিশেষ server configuration কিহৰহে প্রয়োজন? {#does-this-require-special-server-configuration}

হয়, multi-network-ৰ বাবে নিৰ্দিষ্ট server setup আৰু configurationৰ প্ৰয়োজন। দবাৰী setup নিৰ্দেশনাৰ বাবে অনুগ্ৰহ কৰি documentation পৰামৰ্শ কৰক।
