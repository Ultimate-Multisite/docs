# Domain Seller

**Multisite Ultimate: Domain Seller**-এ আপোনাৰ WordPress Multisite নেটৱৰ্কক এটা শক্তিশালী ড’মেইন বিক্ৰী মঞ্চলৈ ৰূপান্তৰ কৰে। গ্ৰাহকসকলক ছাইন-আপৰ সময়ত বা তেওঁলোকৰ Dashboard-ৰ পৰা পোনপটীয়াকৈ কাষ্টম ড’মেইন পঞ্জীয়ন কৰিবলৈ দিয়ক, ৰিয়েল-টাইম উপলব্ধতা পৰীক্ষা আৰু স্বচ্ছ মূল্য নিৰ্ধাৰণৰ সৈতে।

### Domain Seller কিয়?

ড’মেইন পঞ্জীয়ন যিকোনো WaaS (Website as a Service) মঞ্চৰ বাবে আটাইতকৈ বেছি অনুৰোধ কৰা আপচেলসমূহৰ এটা। আপোনাৰ গ্ৰাহকসকলক তৃতীয়-পক্ষৰ registrar-লৈ পঠোৱাৰ সলনি, আপুনি পোনপটীয়াকৈ ড’মেইন আগবঢ়াব পাৰে — গ্ৰাহকসকলক এটা বাধাহীন অভিজ্ঞতা দিয়াৰ লগতে আপোনাৰ নেটৱৰ্কত অধিক ৰাজহ ৰাখি।

### মুখ্য বৈশিষ্ট্যসমূহ

#### একাধিক ড’মেইন Registrar সমৰ্থন
* **OpenSRS** — সম্পূৰ্ণ পঞ্জীয়ন, নবীকৰণ, আৰু পৰিচালনা
* **Namecheap** — WhoisGuard আৰু sandbox পৰীক্ষণৰ সৈতে সম্পূৰ্ণ ইন্টিগ্ৰেচন
* **HostAfrica** — Domains Reseller API-ৰ জৰিয়তে ড’মেইন সন্ধান, মূল্য নিৰ্ধাৰণ, পঞ্জীয়ন, স্থানান্তৰ, DNS, আৰু ID সুৰক্ষা
* **Openprovider** — Reseller মূল্য নিৰ্ধাৰণ, পঞ্জীয়ন, নবীকৰণ, স্থানান্তৰ, DNS, WHOIS গোপনীয়তা, আৰু TLD sync
* **Hostinger** — শ্বেয়াৰ কৰা Hostinger integration token ব্যৱহাৰ কৰি ড’মেইন সন্ধান আৰু পঞ্জীয়ন
* **সম্প্ৰসাৰণযোগ্য Architecture** — Integration Registry pattern-ৰ ওপৰত নিৰ্মিত, যিয়ে অধিক providers যোগ কৰাটো সহজ কৰে
* **Built-in Connection Testing** — live কৰাৰ আগতে provider credentials আৰু সংযোগ বৈধ কৰক

#### নিৰ্দেশিত Setup Wizard
* **এক-ক্লিক onboarding** — provider configuration, TLD import, আৰু domain product সৃষ্টি কৰাৰ মাজেৰে আপোনাক আগবঢ়াই নিয়ে
* **স্বয়ংক্ৰিয় TLD sync** — আপোনাৰ সংযুক্ত providers-ৰ পৰা সকলো উপলব্ধ TLD আৰু wholesale pricing import কৰে
* **ডিফল্ট product সৃষ্টি** — যুক্তিসংগত markup defaults-ৰ সৈতে ব্যৱহাৰৰ বাবে সাজু domain product সৃষ্টি কৰে

#### নমনীয় ড’মেইন মূল্য নিৰ্ধাৰণ
* **তিনিটা markup mode** — শতাংশ markup, fixed markup, বা প্ৰতি TLD-ত fixed price
* **TLD-নিৰ্দিষ্ট products** — বিভিন্ন TLD group (.com/.net, .io, দেশৰ code, ইত্যাদি)ৰ বাবে পৃথক domain products সৃষ্টি কৰক
* **প্ৰাৰম্ভিক মূল্য নিৰ্ধাৰণ** — নতুন পঞ্জীয়ন আকৰ্ষণ কৰিবলৈ ৰেহাইযুক্ত প্ৰথম বছৰৰ মূল্য আগবঢ়াওক
* **ৰিয়েল-টাইম মূল্য নিৰ্ধাৰণ** — বুদ্ধিমান caching-ৰ সৈতে providers-ৰ পৰা live wholesale pricing
* **Coupon সমৰ্থন** — Multisite Ultimate discount codes-ৰ সৈতে সম্পূৰ্ণ সামঞ্জস্য

#### WHOIS গোপনীয়তা সুৰক্ষা
* **প্ৰতি-product configuration** — WHOIS গোপনীয়তা Disabled, Always Included, বা Customer Choice হিচাপে ছেট কৰক
* **গ্ৰাহকৰ opt-in** — Customer Choice হিচাপে ছেট কৰিলে, checkout checkbox-এ গ্ৰাহকসকলক গোপনীয়তা সুৰক্ষা যোগ কৰিবলৈ দিয়ে
* **কনফিগাৰ কৰিব পৰা মূল্য নিৰ্ধাৰণ** — privacy add-on-ৰ বাবে প্ৰতি বছৰৰ মূল্য ছেট কৰক, বা বিনামূলীয়াকৈ আগবঢ়াওক
* **Registrar integration** — পঞ্জীয়নৰ সময়ত স্বয়ংক্ৰিয়ভাৱে WhoisGuard (Namecheap) বা WHOIS গোপনীয়তা (OpenSRS) সক্ষম কৰে

#### একীভূত Checkout অভিজ্ঞতা
* **তিনিটা ড’মেইন mode** — গ্ৰাহকসকলে subdomain (বিনামূলীয়া), নতুন ড’মেইন পঞ্জীয়ন, বা তেওঁলোকে ইতিমধ্যে মালিকানাধীন বিদ্যমান ড’মেইন mapping কৰাৰ মাজৰ পৰা বাছি লয়
* **ৰিয়েল-টাইম ড’মেইন সন্ধান** — page reload নোহোৱাকৈ AJAX-চালিত উপলব্ধতা পৰীক্ষা
* **স্মাৰ্ট পৰামৰ্শ** — পছন্দৰ ড’মেইন লোৱা থাকিলে বিকল্প TLD পৰামৰ্শ
* **স্বচ্ছ মূল্য নিৰ্ধাৰণ** — পঞ্জীয়ন মূল্য, নবীকৰণ মূল্য, আৰু ঐচ্ছিক WHOIS গোপনীয়তা মাচুলৰ স্পষ্ট প্ৰদৰ্শন
* **Registrant contact fields** — checkout-ৰ সময়ত inline-ভাৱে প্ৰয়োজনীয় registrant তথ্য (নাম, ঠিকনা, ফোন) সংগ্ৰহ কৰে
* **স্বয়ং-উৎপন্ন site URL** — বাছনি কৰা ড’মেইনৰ পৰা site ঠিকনা স্বয়ংক্ৰিয়ভাৱে আহৰণ কৰা হয়

#### Auto-Renewal System
* **Membership-সংলগ্ন নবীকৰণ** — গ্ৰাহকৰ membership সক্ৰিয় হৈ থাকিলে ড’মেইনসমূহ স্বয়ংক্ৰিয়ভাৱে নবীকৰণ হয়
* **স্মাৰ্ট scheduling** — ড’মেইনৰ expiry dates-ৰ ওপৰত ভিত্তি কৰি renewal jobs queue কৰা হয়
* **Retry logic** — ব্যৰ্থ renewal attempts-ৰ বাবে বুদ্ধিমান retries
* **Email notifications** — আগন্তুক নবীকৰণ, সফল নবীকৰণ, ব্যৰ্থতা, আৰু expirations-ৰ বাবে কনফিগাৰ কৰিব পৰা alerts

#### গ্ৰাহকৰ DNS পৰিচালনা
* **Self-service DNS** — গ্ৰাহকসকলে তেওঁলোকৰ Dashboard-ৰ পৰা DNS records (A, AAAA, CNAME, MX, TXT) যোগ, সম্পাদনা, আৰু মচিব পাৰে
* **ড’মেইন স্থিতি** — ৰিয়েল-টাইম পঞ্জীয়ন স্থিতি, expiry dates, আৰু nameserver তথ্য
* **স্বয়ংক্ৰিয় domain mapping** — পঞ্জীয়িত বা বিদ্যমান ড’মেইনসমূহ স্বয়ংক্ৰিয়ভাৱে গ্ৰাহকৰ site-লৈ mapped কৰা হয়

#### Admin Tools
* **মেনুৱেল ড’মেইন পঞ্জীয়ন** — admin panel-ৰ পৰা গ্ৰাহকসকলৰ হৈ ড’মেইন পঞ্জীয়ন কৰক
* **ড’মেইন monitoring** — একে পৃষ্ঠাৰ পৰা সকলো গ্ৰাহকৰ ড’মেইন, স্থিতি, আৰু expiry dates চাওক
* **Provider-নিৰ্দিষ্ট logging** — প্ৰতি registrar-ৰ বাবে পৃথক log channels (যেনে, domain-seller-namecheap.log)
* **পঞ্জীয়নৰ বিৱৰণ** — core domain edit page-ত registrant তথ্য আৰু পঞ্জীয়ন data চাওক

### সমৰ্থিত Providers

* **OpenSRS** — পঞ্জীয়ন, নবীকৰণ, DNS পৰিচালনা, আৰু WHOIS গোপনীয়তাৰ সৈতে সম্পূৰ্ণ ইন্টিগ্ৰেচন
* **Namecheap** — পঞ্জীয়ন, নবীকৰণ, batch availability checking, আৰু বিনামূলীয়া WhoisGuard-ৰ সৈতে সম্পূৰ্ণ ইন্টিগ্ৰেচন
* **HostAfrica** — পঞ্জীয়ন, নবীকৰণ, স্থানান্তৰ, DNS পৰিচালনা, EPP code retrieval, registrar lock, আৰু ID সুৰক্ষা
* **Openprovider** — পঞ্জীয়ন, নবীকৰণ, স্থানান্তৰ, DNS পৰিচালনা, EPP code retrieval, registrar lock, WHOIS গোপনীয়তা, আৰু reseller TLD sync
* **Hostinger** — পঞ্জীয়ন, domain lookup, nameserver updates, registrar lock, আৰু WHOIS গোপনীয়তা; নবীকৰণসমূহ Hostinger auto-renew-এ পৰিচালনা কৰে
* **GoDaddy** — পঞ্জীয়ন আৰু নবীকৰণ
* **ResellerClub** — পঞ্জীয়ন, নবীকৰণ, আৰু DNS পৰিচালনা
* **NameSilo** — পঞ্জীয়ন আৰু নবীকৰণ
* **Enom** — পঞ্জীয়ন, নবীকৰণ, আৰু DNS পৰিচালনা

### কাৰিকৰী বিৱৰণ

* **প্ৰয়োজন:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **Architecture:** Domain_Selling_Capability interface-ৰ সৈতে Integration Registry pattern
* **Autoloading:** PSR-4 class mapping-ৰ সৈতে Jetpack autoloader
* **Standards:** WordPress coding standards, strict typing, prepared statements

## Installation

1. **Multisite Ultimate v2.4.12+** ইনষ্টল কৰা আৰু network-activated হোৱা নিশ্চিত কৰক
2. plugin-টো `/wp-content/plugins/ultimate-multisite-domain-seller/`-লৈ আপলোড কৰক
3. WordPress Network Admin-ৰ **Plugins**-ৰ পৰা plugin-টো network-activate কৰক
4. প্ৰথম activation-ত **Setup Wizard** স্বয়ংক্ৰিয়ভাৱে আৰম্ভ হয় — এই পদক্ষেপসমূহ অনুসৰণ কৰক:
   * আপোনাৰ domain registrar সংযোগ কৰক
   * সংযোগটো পৰীক্ষা কৰক
   * উপলব্ধ TLD-সমূহ আৰু wholesale pricing আমদানি কৰক
   * pricing markup সহ আপোনাৰ প্ৰথম domain product সৃষ্টি কৰক
5. **Ultimate Multisite > Checkout Forms**-ত আপোনাৰ checkout form-ত **Domain Selection** field যোগ কৰক

## সততে সোধা প্ৰশ্নসমূহ

### Multisite Ultimate-ৰ কোনটো version প্ৰয়োজন?

Version 2.4.12 বা তাতকৈ ওপৰৰ। addon-টো WP Ultimo v1.x বা 2.4.12-ৰ তলৰ Ultimate Multisite version-সমূহৰ সৈতে compatible নহয়।

### কোনবোৰ domain provider সমৰ্থিত?

নটা registrar সমৰ্থিত: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo**, আৰু **Enom**। registrar API-সমূহে বেলেগ-বেলেগ operation প্ৰকাশ কৰাৰ বাবে provider অনুসৰি ক্ষমতা ভিন্ন হয়; provider-অনুযায়ী overview-ৰ বাবে setup guide চাওক। addon-ৰ Integration Registry architecture-এ custom provider যোগ কৰাটো সৰল কৰে — [developer guide](/developer/integration-guide/custom-registrar) চাওক।

### মই WHOIS privacy protection প্ৰদান কৰিব পাৰোঁনে?

হয়। প্ৰতিটো domain product-ত তিনিটা mode সহ WHOIS Privacy setting থাকে: Disabled, Always Included, বা Customer Choice। Customer Choice-লৈ set কৰিলে, customer-সকলে checkout-ৰ সময়ত price (বা Namecheap-ৰ WhoisGuard-ৰ বাবে "Free") সহ এটা checkbox দেখে। registration সময়ত privacy flag স্বয়ংক্ৰিয়ভাৱে registrar-লৈ পাছ কৰা হয়।

### মই বেলেগ TLD-ৰ বাবে বেলেগ pricing set কৰিব পাৰোঁনে?

হয়। বেলেগ TLD group-ৰ বাবে পৃথক domain product সৃষ্টি কৰক। প্ৰতিটো product-ৰ নিজৰ markup type (percentage, fixed markup, বা fixed price), supported TLD-সমূহ, আৰু introductory pricing থাকিব পাৰে।

### checkout options-ত customer-সকলে কি দেখে?

Domain Selection field-এ customer-সকলক তিনিটা tab দিয়ে: **Subdomain** (আপোনাৰ network-ত এটা বিনামূলীয়া subdomain ব্যৱহাৰ কৰক), **Register New Domain** (নতুন domain search আৰু register কৰক), আৰু **Existing Domain** (তেওঁলোকে ইতিমধ্যে মালিকানাধীন domain এটা map কৰক)। কেৱল আপুনি enable কৰা mode-সমূহহে দেখা যাব।

### ই automatic renewal সমৰ্থন কৰেনে?

হয়। Domain-সমূহ auto-renew-লৈ set কৰিব পাৰি আৰু customer-ৰ membership status-ৰ সৈতে সংযুক্ত থাকে। Renewal payment-সমূহ customer-ৰ existing payment gateway-ৰ জৰিয়তে process কৰা হয়।

### customer-সকলে তেওঁলোকৰ DNS records manage কৰিব পাৰেনে?

হয়। customer-সকলে তেওঁলোকৰ dashboard-ত A, AAAA, CNAME, MX, আৰু TXT record যোগ, edit, আৰু delete কৰিবলৈ DNS management interface পায়। DNS management OpenSRS, ResellerClub, Enom, HostAfrica, আৰু Openprovider-ৰ সৈতে উপলব্ধ। hosted domain-ৰ বাবে Hostinger DNS core Hostinger domain-mapping integration-ৰ দ্বাৰা handle কৰা হয়।

### setup wizard আছেনে?

হয়। setup wizard-টো প্ৰথম activation-ত স্বয়ংক্ৰিয়ভাৱে চলি আপোনাক registrar সংযোগ কৰা, TLD-সমূহ import কৰা, আৰু আপোনাৰ প্ৰথম domain product সৃষ্টি কৰাৰ মাজেৰে লৈ যায়। আপুনি addon settings-ৰ পৰা wizard-টো পুনৰ চলাবও পাৰে।
