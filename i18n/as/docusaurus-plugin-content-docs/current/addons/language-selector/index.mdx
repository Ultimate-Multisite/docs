# ভাষা নির্বাচক (Language Selector)

Multisite Ultimate: Sign-up मा ভাষা নির্বাচক

সাইন-আপ ফ্লো (sign-up flow)-ত এটা ভাষা নির্বাচক ফিল্ড যোগ কৰক।

## স্থাপন (Installation) {#installation}

১. 'ultimate-multisite-language-selector' ফাইলটো '/wp-content/plugins/' ডাইৰেক্টৰীত আপলোড কৰক।
২. আপোনাৰ WordPress Network Admin-ৰ 'Plugins' মেনুৰ মাজেৰে প্লাগইনটো অ্যাক্টিভেট কৰক।

## ব্যৱহাৰৰ নির্দেশিকা (Usage Guide) {#usage-guide}

এই বিনামূলীয়া এড-অন (add-on) ব্যৱহাৰ কৰি আপুনি আপোনাৰ ব্যৱমponentসকলক (users) সাইন-আপ ফ্লো আৰু লগইন পেজত কোন ভাষাটো বিচাৰে, সেইটো বাছি ল'বলৈ দিব পাৰে।

[আপোনাৰ লাইচেঞ্চ অ্যাক্টিভেট কৰাৰ পিছত](../../user-guide/getting-started/installing-ultimate-multisite.md) আপুনি সেইবোৰ Ultimate Multisite এড-অন (add-ons) ইনষ্টল কৰিব পাৰিব, যিবোৰ আপোনাৰ লাইচেঞ্চৰ দ্বাৰা আপোনাক প্ৰৱেশাধিকাৰ দিয়া হৈছে।

আপোনাৰ Ultimate Multisite settings পেজলৈ যাওক আৰু তাৰ পিছত, বাঁহফালে থকা মেনুৰ পৰা _Add-ons_ লেবেলটো ক্লিক কৰক।

![Add-ons page](/img/admin/addons-page.png)

_Language Selector ইনষ্টল কৰিবলৈ Add-ons পেজলৈ যাওক_

তাত, আপুনি আমি যিমানবোৰ এড-অন (add-ons) আগবঢ়াইছো, সেইবোৰ সকলোবোৰ পাব।

বিনামূলীয়া এড-অনবোৰ আমাৰ সকলো গ্ৰাহকৰ বাবে উপলব্ধ, তেওঁলোকে কোনটো সাবস্ক্রিপচন প্লেনত সাইন-আপ কৰিছিল তাৰ ওপৰত নিৰ্ভৰ নকৰে।

এড-অনৰ থাম্বনেইলৰ তলত থকা _Install it Now!_ বাটনটো ক্লিক কৰক। তাৰ পিছত Plugins পেজত ইয়াক অ্যাক্টিভেট কৰক।

![Add-ons page with install button](/img/admin/addons-page.png)

_এড-অনটো ইনষ্টল কৰিবলৈ বাটনটো ক্লিক কৰক_

_আপোনাৰ Plugins পেজত ইয়াক অ্যাক্টিভেট কৰক_

![Settings page](/img/config/settings-general.png)

_এটা সম্পূৰ্ণ নতুন অংশ দেখা গ'ল!_

আপুনি ইয়াক খালিও এৰি যাব পাৰে যাতে সকলো ভাষা অপচন হিচাপে দেখুৱাই, অথবা আপুনি কোনবোৰ ভাষা উপলব্ধ কৰিব বিচাৰে, সেইবোৰ বাছি ল'ব পাৰে। এই ভাষাবোৰ আপোনাৰ WordPress সেটআপত ইনষ্টল কৰা ভাষা।

## URL মাজেৰে ভাষা সেট কৰা {#setting-the-language-via-the-url}

আপুনি URL-ৰ locale পাৰামিটাৰক বাইপাস কৰিও এটা পেজৰ ভাষা সলনি কৰিব পাৰে। উদাহৰণস্বৰূপে, যদি আপুনি ইতালিয়ানত এটা ৰেজিষ্টাৰ পেজলৈ ৰিডাইৰেক্ট কৰিব বিচাৰে, তেন্তে URL-ত ?locale=it_IT যোগ কৰক।

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

ইয়াৰ কাৰ্য্য কৰিবলৈ ভাষাটো আপোনাৰ WordPress নেটৱৰ্কত ইনষ্টল কৰাটো প্ৰয়োজনীয়।
