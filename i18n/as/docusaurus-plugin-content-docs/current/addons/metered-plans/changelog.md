# মিটৰ্ড প্লেনছৰ পৰিৱৰ্তনৰ তালিকা {#metered-plans-changelog}

Version 1.1.0 - মুকলি কৰা হৈছিল 2026-05-05
- নতুন: মাল্টিসাইট সাব-সাইটসমূহৰ বাবে AI টোকেন বিলিং — কাস্টমৰ সাইটসমূহত AI টোকেন ব্যৱহাৰৰ অনুসৰণ কৰা আৰু কনফিগাৰেবল টোকেন হৰাত বিল কৰা হ'ব।
- নতুন: কানেক্টৰ এনফৰসমেন্টক ডাইনামিক লিমিট ডিসক’আৰী আৰু রাইট-থ্ৰুৰ সৈতে পুনৰ নিৰ্মাণ কৰা হৈছে, যাৰ ফলত সকলো কানেক্টৰত রিয়েল-টাইম সঠিকতা নিশ্চিত হয়।
- ফিক্স: ডেটাবেছ টেবুল আপগ্ৰেডসমূহ এতিয়া BerlinDB স্কিমা সংজ্ঞাৰ সৈতে সঠিকভাৱে সংগতিপূৰ্ণ হৈছে, যাৰ ফলত নতুন ইনষ্টলত আপগ্ৰেড কৰাৰ সমস্যা নহ'ব।
- ফিক্স: ডেটাবেছ আপগ্ৰেড কলবেকসমূহ সঠিক ফৰম্যাটোত পৰিৱৰ্তন কৰা হৈছে, যাৰ ফলত নিৰৱচ্ছিন্ন আপগ্ৰেডৰ সমস্যা সমাধান হ'ল।
- ফিক্স: AI Usage Overage Markup ইনপুট ফিল্ডত এতিয়া ভগ্নাংশ মান (fractional values) গ্ৰহণ কৰা হয়।
- ফিক্স: প্লাগইন লোড কৰাৰ সময়ত হোৱা ভয়ানক ভুল (fatal errors) আৰু ডাবল-ইনশ্বালাইজেচন সমস্যা সমাধান কৰা হৈছে।
- উন্নত: ডেভেলপাৰ এনভায়রনমেণ্টৰ বাবে check-env npm স্ক্ৰিপ্ট যোগ কৰা হৈছে, যাতে প্ৰথমবাৰৰ পৰা ই নিজে কনফিগাৰ কৰিব পাৰে।
