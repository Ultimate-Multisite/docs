# WooCommerce Integration পৰিৱৰ্তন-লগ

সংস্কৰণ 2.2.0 - 2026-07-01 তাৰিখে মুকলি কৰা হৈছে
* নতুন: Ultimate Multisite কৰৰ পৰিমাণসমূহ এতিয়া চেকআউটত সুকীয়া WooCommerce মাচুল শাৰী হিচাপে দেখা যায়, যাৰ ফলত পেমেণ্টৰ আগতে কৰৰ মুঠ পৰিমাণ অধিক স্পষ্ট হয়।
* নতুন: WooCommerce Subscriptions পুনৰ-চেষ্টাৰ সময়সীমাৰ ভিতৰত ব্যৰ্থ-নবীকৰণ স্থগিতকৰণ বিচৰা ছাইটসমূহৰ বাবে এটা অপ্ট-ইন "ব্যৰ্থ নবীকৰণত সদস্যতাসমূহ তৎক্ষণাৎ স্থগিত কৰক" ছেটিং আৰু `wu_woo_suspend_on_payment_failure` ফিল্টাৰ যোগ কৰা হৈছে।
* সংশোধন: ব্যৰ্থ বা পুনৰুদ্ধাৰ হোৱা নবীকৰণৰ পিছত Ultimate Multisite সদস্যতাসমূহৰ সৈতে অসামঞ্জস্য হৈ থাকিব পৰা WooCommerce Subscription অৱস্থাসমূহ মিলোৱা হৈছে।
* সংশোধন: অনুপস্থিত থাকিলে Ultimate Multisite মুদ্ৰা তালিকাত WooCommerce ষ্টোৰ মুদ্ৰা যোগ কৰা হৈছে।
* সংশোধন: চাবস্ক্ৰাইবাৰসকলক WooCommerce চেকআউটলৈ পুনৰনিৰ্দেশ কৰাৰ সময়ত গ্ৰাহকৰ বিলিং বিৱৰণসমূহ সংৰক্ষণ কৰা হৈছে।
* উন্নত: Jetpack Autoloader 5-ৰ সৈতে সামঞ্জস্য যোগ কৰা হৈছে।
* উন্নত: ৰিলিজ পেকেজ তৈয়াৰকৰণ পৰিষ্কাৰ কৰা হৈছে যাতে GitHub আৰু মাৰ্কেটপ্লেচ zip-সমূহে nested staging ডাইৰেক্টৰি আৰু উন্নয়ন ফাইল এৰাই চলে।
