# Action: wp_ultimo_woocommerce_load

এই অ্যাকশনটো তেতিয়া ট্ৰيগাৰ হয় যেতিয়া সকলো ॲড-অন ডিপেণ্ডেন্সি (add-on dependencies) লোড হৈ যায়।

এইটোৱে প্লাগইন ডেভেলপাৰসকলক নতুন ফাংচনালিটি যোগ কৰিবলৈ অনুমতি দিয়ে।
