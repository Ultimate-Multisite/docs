# Action: wu_page_enqueue_scripts {#action-wupageenqueuescripts}

ইয়াত এটা কাস্টম অ্যাকশ্বন যোগ কৰা হৈছে, যাৰ ফলত আমি প্ৰয়োজন অনুসৰি স্ক্রিপ্ট আৰু ষ্টাইল লোড কৰিব পাৰোঁ আৰু নিশ্চিত কৰিব পাৰোঁ যে ইহঁত কেৱল সঠিক পৃষ্ঠাৰ ওপৰতহে লোড হয়।
