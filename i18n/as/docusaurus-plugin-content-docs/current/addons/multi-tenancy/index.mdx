# বহু-ভাড়াটীয়া ব্যৱস্থা

কাৰ্যক্ষমতা আৰু সুৰক্ষাৰ বাবে উপ-ছাইটসমূহ ছাৰ্ভাৰ আৰু ডাটাবেছসমূহৰ মাজত বিতৰণ কৰক

## Version 1.2.0 মুক্তিৰ মুখ্য দিশসমূহ {#version-120-release-highlights}

Multi-Tenancy 1.2.0-এ উৎপাদন বিচ্ছিন্নকৰণ কাৰ্যপ্ৰবাহৰ বাবে স্বায়ত্তশাসিত ভাড়াটীয়াৰ ভিত্তি যোগ কৰে:

- অল্পকালীন, পুনৰাবৃত্তি-সুৰক্ষিত SSO ভ্ৰমণৰ সৈতে অৱস্থাহীন ভাড়াটীয়া স্বয়ংক্ৰিয় লগইন।
- স্বায়ত্তশাসিত ভাড়াটীয়াৰ বাবে প্ৰতিটো উপ-ছাইটভিত্তিক ডাটাবেছ আৰু ফাইলছিষ্টেম বিচ্ছিন্নকৰণ।
- host-সচেতন ভাড়াটীয়া provisioning-ৰ বাবে Hostinger বহু-ভাড়াটীয়া ক্ষমতা।
- পুৰণি ডাটাৰ পৰীক্ষা আৰু async push প্ৰক্ৰিয়াকৰণৰ বাবে স্বায়ত্তশাসিত migration যাচাইকৰণ gate।
- Bedrock, FrankenPHP, আৰু localhost install-ৰ বাবে কনফিগাৰযোগ্য একে-মেচিন ডাটাবেছ host binding।
- ভাড়াটীয়া bootstrap, ডাটাবেছ writer, schema bootstrap, আৰু ব্যৱহাৰকাৰী provisioning উন্নতি।
- ভাড়াটীয়াৰ credentials পৰিষ্কাৰ কৰা অধিক নিৰাপদ স্বায়ত্তশাসিত teardown।

[স্বায়ত্তশাসিত migration কাৰ্যপ্ৰবাহ](/user-guide/getting-started/sovereign-migration-workflow), [বহু-ভাড়াটীয়া বিচ্ছিন্নকৰণ](/user-guide/configuration/multi-tenancy-isolation), আৰু [স্বায়ত্তশাসিত ভাড়াটীয়া কাৰ্যকলাপ](/user-guide/administration/sovereign-tenant-operations)-ৰ বাবে ব্যৱহাৰকাৰী guideসমূহ চাওক।

## প্ৰয়োজনীয়তা {#requirements}

- WordPress 5.3 বা তাতকৈ উচ্চ
- PHP 7.4 বা তাতকৈ উচ্চ
- Multisite Ultimate প্লাগইন (সক্ৰিয়)

## ইনষ্টলেচন {#installation}

1. addon ফাইলসমূহ আপোনাৰ `/wp-content/plugins/` ডাইৰেক্টৰীত আপল'ড কৰক
2. WordPress-ত 'Plugins' মেনুৰ জৰিয়তে প্লাগইন সক্ৰিয় কৰক
3. নেটৱৰ্ক admin-ত addon ছেটিংছ কনফিগাৰ কৰক

## কনফিগাৰেচন {#configuration}

addon-এ নেটৱৰ্ক admin interface-ৰ জৰিয়তে প্ৰবেশযোগ্য বিভিন্ন কনফিগাৰেচন বিকল্প সমৰ্থন কৰে।

### স্বায়ত্তশাসিত ভাড়াটীয়া কনফিগাৰেচন {#sovereign-tenant-configuration}

স্বায়ত্তশাসিত ভাড়াটীয়াৰ বাবে, traffic ভাড়াটীয়ালৈ route কৰাৰ আগতে ভাড়াটীয়া ডাটাবেছ, ফাইলছিষ্টেম root, ডাটাবেছ host binding, আৰু host ক্ষমতা কনফিগাৰ কৰক। ডাটাবেছ credentials, host binding, routing, বা ব্যৱহাৰকাৰী provisioning-ত প্ৰভাৱ পেলোৱা প্ৰতিটো পৰিৱৰ্তনৰ পাছত migration যাচাইকৰণ চলাওক।

ভাড়াটীয়া runtime আৰু ডাটাবেছ grantসমূহে local socket বা একে-মেচিন access আশা কৰিলেহে `localhost` ব্যৱহাৰ কৰক। আপোনাৰ provider-এ কনফিগাৰ কৰা grant host সেইটোৱেই হ'লে `127.0.0.1` বা এটা service hostname ব্যৱহাৰ কৰক।

### Hostinger ক্ষমতা {#hostinger-capability}

Hostinger বহু-ভাড়াটীয়া ক্ষমতাই shared Hostinger integration token-ৰ সৈতে কাম কৰে। যাচাইকৰণৰ আগতে domain hPanel-ত উপস্থিত আছে, DNS-এ Hostinger account-লৈ point কৰে, আৰু ভাড়াটীয়া ডাটাবেছ ছেটিংছ hPanel-ৰ সৈতে মিলে বুলি নিশ্চিত কৰক।

## Hook আৰু Filterসমূহ {#hooks-and-filters}

### Actionসমূহ {#actions}

- `ultimate-multisite-addon-template_init` - addon initialization-ৰ পাছত চলোৱা হয়
- `ultimate-multisite-addon-template_loaded` - সকলো addon ফাইল load হোৱাৰ পাছত চলোৱা হয়

### Filterসমূহ {#filters}

- `ultimate-multisite-addon-template_settings` - addon ছেটিংছ সলনি কৰক
- `ultimate-multisite-addon-template_enabled` - addon enable/disable স্থিতি override কৰক
