# Multi-Tenancy

কাৰ্যক্ষমতা আৰু নিৰাপত্তা বৰ্ধৰণৰ বাবে সাব-সাইটসমূহ বিভিন্ন ছাৰ্ভাৰ আৰু ডেটাবেছত বিতৰণ কৰে।

## প্ৰয়োজনীয়তা (Requirements)

- WordPress ৫.৩ বা তাৰ অধিক
- PHP ৭.৪ বা তাৰ অধিক
- Multisite Ultimate plugin (সক্ৰিয়)

## স্থাপন (Installation)

১. addons ফাইলসমূহ আপোনাৰ `/wp-content/plugins/` ডাইৰেক্টৰীত আপলোড কৰক।
২. WordPress-ৰ 'Plugins' মেনুৰ মাজেৰে প্লাগইনটো সক্ৰিয় কৰক।
৩. নেটৱৰ্ক এডমিনত addons ছেটিংসমূহ কনফিগাৰ কৰক।

## বিকাশ (Development)

### ছেটআপ (Setup)

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### পৰীক্ষা (Testing)

এই addons ত এটা বিস্তৃত পৰীক্ষা ব্যৱস্থা অন্তৰ্ভুক্ত কৰা হৈছে:

- **Unit Tests**: WordPress test suite সহ PHPUnit-ভিত্তিক পৰীক্ষা।
- **Base Test Class**: সাধাৰণ পৰীক্ষাৰ দৃশ্যপটৰ বাবে সহায়ক পদ্ধতি (helper methods) প্ৰদান কৰে।
- **Test Coverage**: কোড কভাৰেজ ট্ৰেক কৰিবলৈ কনফিগাৰ কৰা।
- **Multisite Testing**: পৰীক্ষাসমূহ এটা multisite পৰিৱেশত চলি।

### কোড মানদণ্ড (Code Standards)

- **WordPress Coding Standards**: PHPCS ৰ মাজেৰে বলবৎ কৰা।
- **PHP ৭.৪+ Compatibility**: আধুনিক PHP বৈশিষ্ট্যসমূহ সমৰ্থন কৰে।
- **Static Analysis**: টাইপ চেকিংৰ বাবে PHPStan ইন্টিগ্ৰেচন।
- **Code Modernization**: PHP আপগ্রেডৰ বাবে Rector নিয়ম।

### বিল্ড ব্যৱস্থা (Build System)

- **Asset Processing**: CSS/JS ফাইলসমূহৰ বাবে মিনিফিকাচন (Minification)।
- **Translation**: POT ফাইলৰ সৃষ্টি।
- **Packaging**: বিতৰণৰ বাবে আৰ্কাইভ সৃষ্টি।
- **Development Mode**: ডিবাগিংৰ বাবে মিনিফিকা নোহোৱা asset।

## কনফিগাৰেচন (Configuration)

এই addons ত নেটৱৰ্ক এডমিন ইন্টাৰফেচৰ মাজেৰে উপলব্ধ বিভিন্ন কনফিগাৰেচন বিকল্প সমৰ্থন কৰে।

## হুক আৰু ফিল্টাৰ (Hooks and Filters)

### Actions (কাৰ্য)

- `ultimate-multisite-addon-template_init` - addons ইনিশ্বিয়ালাইজেশ্যনৰ পিছত ট্ৰিগাৰ হয়।
- `ultimate-multisite-addon-template_loaded` - সকলো addons ফাইল লোড হোৱাৰ পিছত ট্ৰিগাৰ হয়।

### Filters (ফিল্টাৰ)

- `ultimate-multisite-addon-template_settings` - addons ছেটিং মডিফাই কৰিবলৈ।
- `ultimate-multisite-addon-template_enabled` - addons সক্ষম/নিষ্ক্ষম স্থিতি ওপৰ কৰিবলৈ।
