# সঁজুলি আৰু কনফিগাৰেচন

Theme Builder-এ আপোনাৰ WordPress ছাইট বিশ্লেষণ, ডিজাইন আৰু নিৰ্মাণ কৰিবলৈ একাধিক সঁজুলিৰ সমষ্টি ব্যৱহাৰ কৰে। সংস্কৰণ 1.16.0-ত, **sd-ai-agent/site-scrape এতিয়া স্তৰ ১ সঁজুলি**, যাৰ ফলত ই Theme Builder-ত ডিফল্টভাৱে উপলব্ধ। Superdav AI Agent v1.18.0-এ বিশ্বাসযোগ্য ডেভেলপাৰ কাৰ্যপ্ৰবাহৰ বাবে পৃথকে বিতৰণ কৰা Advanced companion package-ও পৰিচয় কৰাইছে; সেই সঁজুলিসমূহ WordPress.org core package-ৰ অংশ নহয় আৰু পৃথকে ইনষ্টল আৰু অনুমোদন কৰিব লাগিব।

## উপলব্ধ সঁজুলি

### স্তৰ ১ সঁজুলি (সদায় উপলব্ধ)

স্তৰ ১ সঁজুলি কোনো অতিৰিক্ত কনফিগাৰেচন নোহোৱাকৈ Theme Builder-ত ডিফল্টভাৱে উপলব্ধ।

#### sd-ai-agent/site-scrape

**উদ্দেশ্য:** ডিজাইন অনুপ্ৰেৰণা, কন্টেন্ট গঠন, আৰু কাৰ্যক্ষমতা উলিয়াবলৈ বিদ্যমান ৱেবছাইট বিশ্লেষণ কৰা।

**সক্ষমতাসমূহ:**

- **ৱেবছাইট বিশ্লেষণ** — প্ৰতিযোগী বা অনুপ্ৰেৰণামূলক ৱেবছাইট scrape আৰু বিশ্লেষণ কৰা
- **ডিজাইন নিষ্কাশন** — ৰং, ফন্ট, আৰু লেআউট পেটাৰ্ন চিনাক্ত কৰা
- **কন্টেন্ট গঠন** — পৃষ্ঠাৰ সংগঠন আৰু স্তৰবিন্যাস বুজা
- **বৈশিষ্ট্য চিনাক্তকৰণ** — প্লাগইন আৰু কাৰ্যক্ষমতা চিনাক্ত কৰা
- **কাৰ্যক্ষমতা বিশ্লেষণ** — পৃষ্ঠাৰ গতি আৰু অনুকূলন পৰীক্ষা কৰা
- **SEO বিশ্লেষণ** — meta tags আৰু structured data পৰ্যালোচনা কৰা

**ব্যৱহাৰ:**

```
example.com-ৰ ডিজাইন বিশ্লেষণ কৰক আৰু মোৰ ছাইটৰ বাবে ইয়াক অনুপ্ৰেৰণা হিচাপে ব্যৱহাৰ কৰক।
```

**ই কি উলিয়ায়:**

- ৰঙৰ পেলেট আৰু ৰঙৰ ব্যৱহাৰ
- টাইপোগ্ৰাফী (ফন্ট আৰু আকাৰ)
- লেআউট গঠন আৰু ফাঁক
- নেভিগেচন পেটাৰ্ন
- কন্টেন্ট সংগঠন
- ছবি আৰু মিডিয়া ব্যৱহাৰ
- ইণ্টাৰেক্টিভ উপাদান
- মোবাইল responsiveness

**সীমাবদ্ধতাসমূহ:**

- পাছৱৰ্ড-সুৰক্ষিত ছাইট scrape কৰিব নোৱাৰে
- robots.txt আৰু ছাইট নীতি মানি চলে
- ডাইনামিক কন্টেন্ট ধৰা নপৰিব পাৰে
- JavaScript-ভাৰী ছাইটৰ বিশ্লেষণ সীমিত হ’ব পাৰে
- ডাঙৰ ছাইট বিশ্লেষণ কৰিবলৈ অধিক সময় লাগিব পাৰে

### স্তৰ ২ সঁজুলি (ঐচ্ছিক)

স্তৰ ২ সঁজুলি Theme Builder settings-ত স্পষ্টভাৱে সক্ৰিয় কৰিলে উপলব্ধ হয়।

#### Advanced Analytics

বিশদ কাৰ্যক্ষমতা মেট্ৰিক প্ৰদান কৰে:

- পৃষ্ঠা লোড সময়
- Core Web Vitals
- SEO স্ক’ৰ
- Accessibility স্ক’ৰ
- Best practices স্ক’ৰ

#### Content Optimizer

নিম্নলিখিত বিষয়ৰ বাবে বিশ্লেষণ আৰু উন্নতিৰ পৰামৰ্শ দিয়ে:

- পঠনযোগ্যতা
- SEO অনুকূলন
- Keyword ব্যৱহাৰ
- কন্টেন্ট গঠন
- Call-to-action স্থাপন

### স্তৰ ৩ সঁজুলি (প্ৰিমিয়াম)

স্তৰ ৩ সঁজুলিৰ বাবে অতিৰিক্ত কনফিগাৰেচন বা প্ৰিমিয়াম বৈশিষ্ট্যৰ প্ৰয়োজন।

#### AI Content Generator

নিম্নলিখিতৰ বাবে কন্টেন্ট সৃষ্টি কৰে:

- প্ৰডাক্ট বৰ্ণনা
- সেৱা পৃষ্ঠা
- ব্লগ পোষ্ট
- Meta descriptions
- Call-to-action কপি

#### Advanced Design Tools

উন্নত ডিজাইন সক্ষমতা প্ৰদান কৰে:

- Custom CSS generation
- Animation creation
- Interactive element design
- Advanced color theory
- Typography optimization

### Advanced Companion Tools (পৃথক পেকেজ)

Advanced companion package core Superdav AI Agent release-ৰ পৰা পৃথকে বিতৰণ কৰা হয়। ই বিশ্বাসযোগ্য ডেভেলপাৰ আৰু ছাইট-মালিক পৰিবেশৰ বাবে উদ্দেশ্য কৰা হৈছে, য’ত প্ৰশাসকসকলে agent-ক উচ্চ-প্ৰভাৱৰ tooling ব্যৱহাৰ কৰিবলৈ স্পষ্টভাৱে অনুমতি দিয়ে।

Advanced companion সক্ষমতাসমূহত অন্তৰ্ভুক্ত হ’ব পাৰে:

- **Filesystem tools** — বিশ্বাসযোগ্য পৰিবেশত অনুমোদিত ফাইল পৰিদৰ্শন আৰু সংশোধন কৰা
- **Database tools** — স্পষ্টভাৱে অনুমোদন থাকিলে ছাইট ডাটা query বা update কৰা
- **WP-CLI tools** — WordPress maintenance আৰু inspection commands চলোৱা
- **REST dispatcher tools** — নিয়ন্ত্ৰিত dispatch workflows-ৰ জৰিয়তে registered REST endpoints কল কৰা
- **Plugin builder tools** — plugin code scaffold আৰু iterate কৰা
- **Git snapshot tools** — বিপজ্জনক ডেভেলপাৰ কাৰ্যৰ আগতে snapshots সৃষ্টি কৰা
- **User-management tools** — অনুমতি থকা ঠাইত user records পৰ্যালোচনা বা সামঞ্জস্য কৰাত সহায় কৰা
- **Benchmark tools** — development review-ৰ বাবে performance বা capability benchmarks চলোৱা

এইবোৰক প্ৰতিটো ইনষ্টলেচনৰ বাবে সাধাৰণতে উপলব্ধ Theme Builder সঁজুলি হিচাপে নথিভুক্ত নকৰিব। কেৱল Advanced companion package ইনষ্টল, সক্ৰিয়, আৰু বিশ্বাসযোগ্য প্ৰশাসকৰ বাবে সীমাবদ্ধ কৰিলে ইয়াৰ উপলব্ধতা থাকে।

## সঁজুলি কনফিগাৰেচন

### সঁজুলি সক্ৰিয় কৰা

Theme Builder-ত অতিৰিক্ত সঁজুলি সক্ৰিয় কৰিবলৈ:

1. **Gratis AI Agent → Settings**-লৈ যাওক
2. **Theme Builder → Tools**-লৈ যাওক
3. প্ৰয়োজন অনুসৰি সঁজুলি on/off কৰক
4. settings সংৰক্ষণ কৰক

### সঁজুলিৰ অনুমতি

কিছু সঁজুলিৰ অনুমতিৰ প্ৰয়োজন:

- **ছাইট scraping** — ইণ্টাৰনেট প্ৰৱেশাধিকাৰ প্ৰয়োজন
- **Analytics** — Google Analytics integration প্ৰয়োজন
- **কন্টেন্ট generation** — API keys প্ৰয়োজন
- **উন্নত বৈশিষ্ট্য** — premium subscription প্ৰয়োজন হ’ব পাৰে
- **Advanced companion tools** — পৃথকে বিতৰণ কৰা Advanced package আৰু স্পষ্ট administrator trust প্ৰয়োজন

### API Keys আৰু Credentials

প্ৰয়োজন হোৱা সঁজুলিৰ বাবে API keys কনফিগাৰ কৰক:

1. **Gratis AI Agent → Settings → API Keys**-লৈ যাওক
2. প্ৰতিটো সঁজুলিৰ বাবে credentials প্ৰৱিষ্ট কৰক
3. সংযোগ পৰীক্ষা কৰক
4. সুৰক্ষিতভাৱে সংৰক্ষণ কৰক

### Advanced Companion Package ইনষ্টল কৰা

Advanced companion package WordPress.org core ZIP-ৰ পৰা পৃথকে মুক্তি দিয়া হয়। ইয়াক কেৱল release-ৰ বাবে official project distribution channel-ৰ পৰা ইনষ্টল কৰক, তাৰ পিছত বিশ্বাসযোগ্য প্ৰশাসকৰ বাবে access সীমাবদ্ধ কৰক। production ছাইটত filesystem, database, WP-CLI, REST dispatcher, plugin builder, user-management, বা benchmark tools সক্ৰিয় কৰাৰ আগতে human approval gates আৰু git snapshot behaviour পৰ্যালোচনা কৰক।

## sd-ai-agent/site-scrape ব্যৱহাৰ কৰা

### মৌলিক ব্যৱহাৰ

Theme Builder-ক এটা ৱেবছাইট বিশ্লেষণ কৰিবলৈ কওক:

```
competitor.com-ত থকা মোৰ প্ৰতিযোগীৰ ছাইটৰ ডিজাইন বিশ্লেষণ কৰক
আৰু মোৰ ছাইটৰ বাবে ডিজাইন উন্নতিৰ পৰামৰ্শ দিয়ক।
```

### নিৰ্দিষ্ট বিশ্লেষণ

নিৰ্দিষ্ট ধৰণৰ বিশ্লেষণ অনুৰোধ কৰক:

```
example.com-ৰ পৰা ৰঙৰ পেলেট উলিয়াওক আৰু ইয়াক অনুপ্ৰেৰণা হিচাপে ব্যৱহাৰ কৰক।
```

```
example.com-ৰ নেভিগেচন গঠন বিশ্লেষণ কৰক আৰু সদৃশ
সংগঠন মোৰ ছাইটত প্ৰয়োগ কৰক।
```

```
example.com-ৰ মোবাইল responsiveness পৰীক্ষা কৰক আৰু মোৰ ছাইট
সমানেই responsive হোৱাটো নিশ্চিত কৰক।
```

### তুলনামূলক বিশ্লেষণ

একাধিক ছাইট তুলনা কৰক:

```
site1.com আৰু site2.com-ৰ ডিজাইন তুলনা কৰক আৰু দুয়োটাৰে সৰ্বশ্ৰেষ্ঠ উপাদান
একত্ৰিত কৰা এটা ডিজাইন সৃষ্টি কৰক।
```

## সঁজুলিৰ সীমাবদ্ধতা আৰু বিবেচনা

### Rate Limiting

- চাৰ্ভাৰৰ অতিভাৰ ৰোধ কৰিবলৈ scraping হাৰ-সীমাবদ্ধ কৰা হয়
- একে site-লৈ একাধিক অনুৰোধ throttled হ’ব পাৰে
- ডাঙৰ site-সমূহ বিশ্লেষণ কৰিবলৈ অধিক সময় লাগিব পাৰে

### বিষয়বস্তুৰ সঠিকতা

- গতিশীল বিষয়বস্তু সম্পূৰ্ণকৈ ধৰা নপৰিব পাৰে
- JavaScript-rendered বিষয়বস্তু অসম্পূৰ্ণ হ’ব পাৰে
- বাস্তৱ-সময়ৰ তথ্য প্ৰতিফলিত নোহ’ব পাৰে
- কিছুমান বিষয়বস্তু paywall-ৰ আঁৰত থাকিব পাৰে

### আইনী আৰু নৈতিক বিবেচনা

- robots.txt আৰু site নীতিসমূহ সন্মান কৰক
- পুনৰ প্ৰকাশৰ বাবে কপিৰাইটযুক্ত বিষয়বস্তু scrape নকৰিব
- নকলৰ বাবে নহয়, প্ৰেৰণাৰ বাবে বিশ্লেষণ ব্যৱহাৰ কৰক
- উলিওৱা বিষয়বস্তু ব্যৱহাৰ কৰাৰ অধিকাৰ আপোনাৰ আছে নে নাই যাচাই কৰক
- বিশ্লেষণ কৰা site-সমূহৰ সেৱাৰ চৰ্ত মানি চলক

### কাৰ্যক্ষমতাৰ প্ৰভাৱ

- ডাঙৰ site scrape কৰিবলৈ সময় লাগিব পাৰে
- একেলগে একাধিক scrape ধীৰ হ’ব পাৰে
- নেটৱৰ্ক সংযোগে গতিৰ ওপৰত প্ৰভাৱ পেলায়
- ডাঙৰ dataset-সমূহৰ অধিক প্ৰক্ৰিয়াকৰণৰ প্ৰয়োজন হ’ব পাৰে

## উত্তম পদ্ধতিসমূহ

### site বিশ্লেষণ ব্যৱহাৰ কৰা

1. **একাধিক site বিশ্লেষণ কৰক** — কেইবাটাও উৎসৰ পৰা প্ৰেৰণা সংগ্ৰহ কৰক
2. **গঠনৰ ওপৰত গুৰুত্ব দিয়ক** — layout আৰু সংগঠনৰ ধৰণসমূহ শিকক
3. **ৰং উলিয়াওক** — ৰঙৰ palette-সমূহ আৰম্ভণি বিন্দু হিচাপে ব্যৱহাৰ কৰক
4. **typography অধ্যয়ন কৰক** — আপুনি ভাল পোৱা font সংমিশ্ৰণ চিনাক্ত কৰক
5. **navigation পৰ্যালোচনা কৰক** — menu সংগঠন বুজক

### নৈতিক scraping

1. **robots.txt সন্মান কৰক** — site নিৰ্দেশনা মানি চলক
2. **বিষয়বস্তু নকল নকৰিব** — কেৱল প্ৰেৰণাৰ বাবে বিশ্লেষণ ব্যৱহাৰ কৰক
3. **অধিকাৰ যাচাই কৰক** — উলিওৱা বিষয়বস্তু ব্যৱহাৰ কৰিব পাৰে বুলি নিশ্চিত কৰক
4. **উৎসৰ স্বীকৃতি দিয়ক** — প্ৰেৰণাৰ উৎসসমূহক কৃতিত্ব দিয়ক
5. **চৰ্ত মানি চলক** — site-ৰ সেৱাৰ চৰ্ত মানি চলক
