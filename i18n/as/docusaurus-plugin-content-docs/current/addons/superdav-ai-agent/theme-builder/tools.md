# ট্ৰুল আৰু কনফিগাৰেচন

Theme Builder-এ আপোনাৰ WordPress সাইট বিশ্লেষণ কৰিবলৈ, ডিজাইন কৰিবলৈ আৰু নিৰ্মাণ কৰিবলৈ এটা প্ৰকাৰৰ ট্ৰুলৰ ব্যৱহাৰ কৰে। ১.১৬.০ সংস্কৰণত, **sd-ai-agent/site-scrape এতিয়া এটা Tier 1 ট্ৰুল**, যাৰ ফলত ই Theme Builder-ত ডিফল্টভাৱে উপলব্ধ হৈছে।

## উপলব্ধ ট্ৰুলসমূহ

### Tier 1 ট্ৰুলসমূহ (সদায় উপলব্ধ)

Tier 1 ট্ৰুলসমূহ কোনো এটা অতিৰিক্ত কনফিগাৰেচনৰ প্ৰয়োজন নোহোৱাকৈ Theme Builder-ত ডিফল্টভাৱে উপলব্ধ।

#### sd-ai-agent/site-scrape

**উদ্দেশ্য:** ডিজাইনৰ অনুপ্ৰেৰণা, বিষয়বস্তুৰ গঠন আৰু কাৰ্যক্ষমতা আৱদ্ধ কৰিবলৈ বিদ্যমান ওয়েবসাইটসমূহ বিশ্লেষণ কৰে।

**ক্ষমতা:**

- **Website analysis** — প্ৰতিদ্বন্দ্বী বা অনুপ্ৰেৰণামূলক ওয়েবসাইটসমূহ স্ক্ৰেপ আৰু বিশ্লেষণ কৰে
- **Design extraction** — ৰং, ফণ্ট আৰু লেআউটৰ ধৰণ চিনাক্ত কৰে
- **Content structure** — পৃষ্ঠাৰ সংগঠন আৰু অনুক্ৰম বুজাই দিয়ে
- **Feature detection** — প্লাগইন আৰু কাৰ্যক্ষমতা চিনাক্ত কৰে
- **Performance analysis** — পৃষ্ঠাৰ গতি আৰু অপটিমাইজেশ্যন পৰীক্ষা কৰে
- **SEO analysis** — মেটা ট্যাগ আৰু ছট্ৰাকচাৰ্ড ডেটা পৰীক্ষা কৰে

**প্ৰয়োগ:**

```
Analyze the design of example.com and use it as inspiration for my site.
```

**কি আৱদ্ধ কৰে:**

- ৰংৰ প্যালেট আৰু ৰং ব্যৱহাৰ
- টাইপ’গ্ৰাফি (ফণ্ট আৰু আকাৰ)
- লেআউট গঠন আৰু স্পেছিং
- নেভিগেশ্যনৰ ধৰণ
- বিষয়বস্তুৰ সংগঠন
- ছবি আৰু মিডিয়া ব্যৱহাৰ
- ইন্টাৰেক্টিভ এলিমেন্ট
- মোবাইল ৰেসপনসিভনেছ

**সীমাবদ্ধতা:**

- পাছৱৰ্ড-ৰাঙিত সাইট স্ক্ৰেপ কৰিব নোৱাৰে
- robots.txt আৰু সাইট নীতি মানি চলে
- গতিশীল বিষয়বস্তু (dynamic content) আৱদ্ধ কৰিব নোৱাৰে
- JavaScript-ভিত্তিক সাইটসমূহত সীমিত বিশ্লেষণ হ'ব পাৰে
- ডাঙৰ সাইটসমূহ বিশ্লেষণ কৰিবলৈ বেছি সময় লাগিব পাৰে

### Tier 2 ট্ৰুলসমূহ (ঐচ্ছিক)

Tier 2 ট্ৰুলসমূহ Theme Builder ছেটিংছত স্পষ্টকৈ সক্ষম কৰিলেহে উপলব্ধ হয়।

#### Advanced Analytics

বিশদ কাৰ্যক্ষমতাৰ মেট্ৰিক প্ৰদান কৰে:

- Page load times
- Core Web Vitals
- SEO score
- Accessibility score
- Best practices score

#### Content Optimizer

তলৰ বাবে বিশ্লেষণ কৰে আৰু উন্নতিৰ বাবে পৰামৰ্শ দিয়ে:

- Readability
- SEO optimization
- Keyword usage
- Content structure
- Call-to-action placement

### Tier 3 ট্ৰুলসমূহ (প্ৰিমিয়াম)

Tier 3 ট্ৰুলসমূহৰ বাবে অতিৰিক্ত কনফিগাৰেচন বা প্ৰিমিয়াম বৈশিষ্ট্যৰ প্ৰয়োজন।

#### AI Content Generator

তলৰ বাবে বিষয়বস্তু সৃষ্টি কৰে:

- Product descriptions
- Service pages
- Blog posts
- Meta descriptions
- Call-to-action copy

#### Advanced Design Tools

উন্নত ডিজাইন ক্ষমতা প্ৰদান কৰে:

- Custom CSS generation
- Animation creation
- Interactive element design
- Advanced color theory
- Typography optimization

## ট্ৰুল কনফিগাৰেচন

### ট্ৰুল সক্ষম কৰা

Theme Builder-ত অতিৰিক্ত ট্ৰুল সক্ষম কৰিবলৈ:

1. **Gratis AI Agent → Settings** লৈ যাওক
2. **Theme Builder → Tools** লৈ যাওক
3. প্ৰয়োজন অনুসৰি ট্ৰুলসমূহ অন/অফ কৰক
4. ছেটিংছ সেভ কৰক

### ট্ৰুল অনুমতি (Tool Permissions)

কিছুমান ট্ৰুলৰ বাবে অনুমতিৰ প্ৰয়োজন:

- **Site scraping** — ইন্টারনেট প্ৰৱেশৰ প্ৰয়োজন
- **Analytics** — Google Analytics ইন্টিগ্ৰেচনৰ প্ৰয়োজন
- **Content generation** — API keys ৰ প্ৰয়োজন
- **Advanced features** — প্ৰিমিয়াম সাবস্ڪ্ৰিপচনৰ প্ৰয়োজন হ'ব পাৰে

### API Keys আৰু Credentials

যি ট্ৰুলসমূহত API keysৰ প্ৰয়োজন, তাৰ বাবে কনফিগাৰ কৰক:

1. **Gratis AI Agent → Settings → API Keys** লৈ যাওক
2. প্ৰতিটো ট্ৰুলৰ বাবে credentials প্ৰৱেশ কৰক
3. সংযোগ পৰীক্ষা কৰক
4. নিৰাপদভাৱে সেভ কৰক

## sd-ai-agent/site-scrape ব্যৱহাৰ কৰা

### মৌলিক ব্যৱহাৰ

Theme Builder-ক এটা ওয়েবসাইট বিশ্লেষণ কৰিবলৈ কওক:

```
Analyze the design of my competitor's site at competitor.com
and suggest design improvements for my site.
```

### নিৰ্দিষ্ট বিশ্লেষণ

নিৰ্দিষ্ট ধৰণৰ বিশ্লেষণৰ বাবে অনুৰোধ কৰক:

```
Extract the color palette from example.com and use it as inspiration.
```

```
Analyze the navigation structure of example.com and apply similar
organization to my site.
```

```
Check the mobile responsiveness of example.com and ensure my site
is equally responsive.
```

### তুলনা বিশ্লেষণ

বহুটা সাইটৰ তুলনা কৰক:

```
Compare the designs of site1.com and site2.com and create a design
that combines the best elements of both.
```

## ট্ৰুলৰ সীমাবদ্ধতা আৰু বিবেচনা

### Rate Limiting

- ছাৰ্ৱাৰৰ অধিক চাপ এৰাবলৈ স্ক্ৰেপিংয়ে rate-limited কৰা হয়
- একেটা সাইটলৈ বহুতো অনুৰোধ কৰিলে সীমিত কৰিব পাৰে (throttled)
- ডাঙৰ সাইটসমূহ বিশ্লেষণ কৰিবলৈ বেছি সময় লাগিব পাৰে

### বিষয়বস্তুৰ সঠিকতা (Content Accuracy)

- গতিশীল বিষয়বস্তু সম্পূৰ্ণৰূপে আৱদ্ধ নহ'ব পাৰে
- JavaScript-ৰ দ্বাৰা ৰেণ্ডাৰ কৰা বিষয়বস্তু অসম্পূৰ্ণ হ'ব পাৰে
- বাস্তৱ-সময়ৰ তথ্য প্ৰতিফলিত নহ'ব পাৰে
- কিছুমান বিষয়বস্তু পে-ৱলৰ আঁৰত থাকিব পাৰে

### আইনী আৰু নৈতিক বিবেচনা

- robots.txt আৰু সাইট নীতি মানি চলা
- পুনৰ প্ৰতিলিপি কৰিবলৈ কপিৰাইট কৰা বিষয়বস্তু স্ক্ৰেপ নকৰিব
- বিশ্লেষণক অনুপ্ৰেৰণা হিচাপে ব্যৱহাৰ কৰক, নকল কৰিবলৈ নহয়
- আৱদ্ধ বিষয়বস্তু ব্যৱহাৰ কৰাৰ অধিকাৰ আপোনাৰ আছে নে নাই, সেয়া পৰীক্ষা কৰক
- বিশ্লেষণ কৰা সাইটসমূহৰ সেৱাৰ চৰমশ্ৰুতি (terms of service) অনুসৰণ কৰক

### কাৰ্যক্ষমতাৰ প্ৰভাৱ (Performance Impact)

- ডাঙৰ সাইট স্ক্ৰেপ কৰিবলৈ সময় লাগিব পাৰে
- একে সময়তে বহুতো স্ক্ৰেপ কৰিলে বেছি সময় লাগিব পাৰে
- নেটৱৰ্ক সংযোগে গতি প্ৰভাৱিত কৰে
- ডাঙৰ ডেটা ছেটৰ বাবে অধিক প্ৰক্ৰিয়াকৰণৰ প্ৰয়োজন হ'ব পাৰে

## উত্তম অভ্যাস (Best Practices)

### সাইট বিশ্লেষণ ব্যৱহাৰ কৰা

1. **বহুটা সাইট বিশ্লেষণ কৰক** — বিভিন্ন উৎসৰ পৰা অনুপ্ৰেৰণা গ্ৰহণ কৰক
2. **গঠনৰ ওপৰত গুৰুত্ব দিন** — লেআউট আৰু সংগঠনৰ ধৰণ শিকক
3. **ৰং আৱদ্ধ কৰক** — ৰংৰ প্যালেটক আৰম্ভণি বিন্দু হিচাপে ব্যৱহাৰ কৰক
4. **টাইপ’গ্ৰাফি অধ্যয়ন কৰক** — আপোনাৰ ভাল লাগি থকা ফণ্টৰ সংমিশ্ৰণ চিনাক্ত কৰক
5. **নেভিগেশ্যন পৰীক্ষা কৰক** — মেনুৰ সংগঠন বুজি লওক

### নৈতিক স্ক্ৰেপিং

1. **robots.txt মানক** — সাইটৰ নির্দেশিকা অনুসৰণ কৰক
2. **বিষয়বস্তু কপি নকৰিব** — কেৱল অনুপ্ৰেৰণাৰ বাবে বিশ্লেষণ ব্যৱহাৰ কৰক
3. **অধিকাৰ পৰীক্ষা কৰক** — আৱদ্ধ বিষয়বস্তু ব্যৱহাৰ কৰিব পাৰিবনে, সেয়া নিশ্চিত কৰক
4. **উৎস উল্লেখ কৰক** — অনুপ্ৰেৰণামূলক উৎসক কৃতিত্ব দিয়ক
5. **শৰ্ত মানক** — সাইটৰ সেৱাৰ চৰমশ্ৰুতিৰ সৈতে সন্মত হ'ক

### ট্ৰুলৰ কাৰ্যকৰীতা বৃদ্ধি কৰা

1. **নিৰ্দিষ্ট হ'ক** — নিৰ্দিষ্ট ধৰণৰ বিশ্লেষণৰ বাবে অনুৰোধ কৰক
2. **প্ৰাসংগিকতা প্ৰদান কৰক** — আপোনাৰ সাইটৰ উদ্দেশ্য ব্যাখ্যা কৰক
3. **অপেক্ষাৰ বুলি ঘোষণা কৰক** — আপোনাৰ ডিজাইনৰ লক্ষ্য বৰ্ণনা কৰক
4. **পুনৰাবৃত্তি কৰক** — ফলাফলৰ ওপৰত ভিত্তি কৰি সংশোধন কৰক
5. **ট্ৰুল সংমিশ্ৰণ কৰক** — বিস্তৃত বিশ্লেষণৰ বাবে বহু ট্ৰুল ব্যৱহাৰ কৰক

## সমস্যা সমাধান (Troubleshooting)

### সাইট স্ক্ৰেপ নহ'ব

- সাইটটো জনপ্ৰযোজ্যভাৱে উপলব্ধ নেকি, সেয়া পৰীক্ষা কৰক
- robots.txt স্ক্ৰেপিং কৰিবলৈ অনুমতি দিয়ে নেকি, সেয়া নিশ্চিত কৰক
- আন এটা সাইট চেষ্টা কৰক
- ইন্টারনেট সংযোগ পৰীক্ষা কৰক
- সমস্যাটো চলি থাকিলে সহায়কৰ সৈতে যোগাযোগ কৰক

### অসম্পূৰ্ণ বিশ্লেষণ

- সাইটত গতিশীল বিষয়বস্তু থাকিব পাৰে
- JavaScript সম্পূৰ্ণৰূপে ৰেণ্ডাৰ নহ'ব পাৰে
- ডাঙৰ সাইটসমূহে timeout হ'ব পাৰে
- তাৰ পৰিৱৰ্তে নিৰ্দিষ্ট পৃষ্ঠা বিশ্লেষণ কৰিবলৈ চেষ্টা কৰক
- নিৰ্দিষ্ট বিশ্লেষণ ধৰণ অনুৰোধ কৰক

### ধীৰ কাৰ্যক্ষমতা

- ডাঙৰ সাইট বিশ্লেষণ কৰিবলৈ বেছি সময় লাগে
- একে সময়তে বহুতো স্ক্ৰেপ কৰিলে বেছি সময় লাগে
- নেটৱৰ্ক সংযোগে গতি প্ৰভাৱিত কৰে
- ব্যস্ত সময়ৰ বাহিৰত চেষ্টা কৰক
- প্ৰথমে সৰু অংশ বিশ্লেষণ কৰক
