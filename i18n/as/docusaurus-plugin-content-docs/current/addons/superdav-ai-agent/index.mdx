# Superdav AI Agent

Superdav AI Agent হৈছে এটা উন্নত WordPress AI সহায়ক, যিয়ে Gratis AI Agent लाई থীম-সচেতন দক্ষতা (theme-aware skills), উন্নত ইন্টাৰনেট অনুসন্ধান ক্ষমতা, আৰু উন্নত সাইট বিল্ডাৰ বৈশিষ্ট্যৰ সৈতে সম্প্ৰসাৰণ কৰে। Version 1.16.0 ত কাস্টম লোগো জেনারেচন, প্যালেট কনট্ৰাষ্ট ভ্যালিডেশ্যন, ডিসকভারি সাক্ষাৎকাৰত ফটো আপলোড, ষ্ট্ৰাকচাৰ্ড হোষ্টেলිටি মেনু জেনারেচন, আৰু Theme Builder ত ডেস্কটপ/মোবাইল প্ৰিভিউ ৰেণ্ডাৰিং যোগ কৰা হৈছে।

## মূল বৈশিষ্ট্যসমূহ (Key Features)

- **থীম-সচেতন দক্ষতা (Theme-aware skills)** — আপোনাৰ active WordPress থীম আৰু install কৰা প্লাগইনৰ ওপৰত ভিত্তি কৰি স্বয়ংক্রিয়ভাৱে পথপ্ৰদৰ্শক আৰু ক্ষমতাৰ সলনি কৰে।
- **বহু ইন্টাৰনেট অনুসন্ধান প্ৰদানকাৰী (Multiple internet search providers)** — অধিক সমৃদ্ধ অনুসন্ধান ফলাফল আৰু উন্নত গৱেষণা ক্ষমতাৰ বাবে Brave Search আৰু Tavilyৰ মাজত বাছি লওঁ।
- **সাইট বিল্ডাৰৰ যোগাযোগ ফৰ্ম (Site builder contact forms)** — আপোনাৰ install কৰা ফৰ্ম প্লাগইন ব্যৱহাৰ কৰি চ্যাট ইন্টাৰফেচৰ পৰাই সম্পূৰ্ণ কাৰ্য্যক্ষম যোগাযোগ ফৰ্ম সৃষ্টি কৰক।
- **উন্নত সাইট বিল্ডাৰ (Enhanced site builder)** — উন্নত প্লাগইন আৱিষ্কাৰ আৰু ভুল সংশোধনৰ সৈতে উন্নত বহু-ধাপৰ সাইট বিল্ড پلان।
- **বিল্ট-ইন থীম দক্ষতা (Built-in theme skills)** — Block Themes, Classic Themes, Kadence Blocks, আৰু Kadence Themeৰ বাবে বিশেষ পথপ্ৰদৰ্শক।
- **উন্নত WooCommerce ইন্টিগ্ৰেচন (Improved WooCommerce integration)** — উন্নত বিশ্বাসযোগ্যতা আৰু সামঞ্জস্যৰ বাবে native WooCommerce API ব্যৱহাৰ কৰিবলৈ পুনৰ গঠন কৰা হৈছে।
- **স্বয়ংক্রিয় প্ৰদানকাৰী আৱিষ্কাৰ (Automatic provider detection)** — প্লাগইন বা থীম activate বা deactivate কৰা হৈছে বুলি স্বয়ংক্রিয়ভাৱে আৱদ্ধ আৰু উপলব্ধ প্লাগইন আৰু থীমসমূহ Refresh কৰে।
- **কাস্টম লোগো জেনারেচন (Custom logo generation)** — স্বয়ংক্রিয় namespace-safe sanitisation সৈতে কাস্টম SVG লোগো জেনারেট আৰু embed কৰক।
- **প্যালেট কনট্ৰাষ্ট ভ্যালিডেশ্যন (Palette contrast validation)** — আপোনাৰ থীমলৈ প্ৰয়োগ কৰাৰ আগতে WCAG সহজলভ্যতাৰ নিয়ম অনুসৰি ৰঙৰ জোড়া পৰীক্ষা কৰক।
- **ফটো-উন্নত ডিসকভারি (Photo-enhanced discovery)** — থীম বিল্ডাৰৰ ডিসকভারি সাক্ষাৎকাৰৰ সময়ত ফটো আপলোড কৰক যাতে ডিজাইনৰ দিশ নিৰ্ধাৰণ কৰাত সহায় হয়।
- **হোষ্টেলිටি মেনু (Hospitality menus)** — ресторан, কেফে, আৰু হোষ্টেলිටি ব্যৱসায়ৰ বাবে গঠনমূলক খাদ্য আৰু পানীয় মেনু পৃষ্ঠা জেনারেট কৰক।
- **ডেস্কটপ আৰু মোবাইল প্ৰিভিউ (Desktop and mobile previews)** — ডিজাইনৰ দিশ নিৰ্ধাৰণ কৰাৰ সময়ত ডেস্কটপ আৰু মোবাইল ডিভাইচত আপোনাৰ ডিজাইনৰ প্ৰিভিউ লওক।
- **Tier 1 সাইট স্ক্ৰেপিং (Tier 1 site scraping)** — sd-ai-agent/site-scrape এতিয়া ডিফল্টভাৱে উপলব্ধ, যিয়ে প্ৰতিদ্বন্দ্বী আৰু অনুপ্ৰেৰণামূলক ওয়েবসাইট বিশ্লেষণ কৰে।

## প্ৰয়োজনীয়তা (Requirements)

- WordPress 6.0 বা তাতকৈ অধিক
- PHP 8.1 বা তাতকৈ অধিক
- WordPress Multisite (network-activated) বা single-site installation
- এটা active AI provider connector plugin (Anthropic, OpenAI, বা সামঞ্জস্যপূৰ্ণ)
- Gratis AI Agent v1.9.0 বা তাতকৈ অধিক (Superdav AI Agent ले Gratis AI Agent लाई সম্প্ৰসাৰণ কৰে)

## স্থাপন (Installation)

1. প্লাগইন ফাইলসমূহ `/wp-content/plugins/superdav-ai-agent` লৈ আপলোড কৰক।
2. WordPressৰ **Plugins** স্ক্ৰীণৰ মাজেৰে প্লাগইনটো activate কৰক।
3. নিশ্চিত কৰক যে Gratis AI Agentো activate কৰা হৈ আছে।
4. **Gratis AI Agent → Settings** লৈ যোৱা আৰু আপোনাৰ ইন্টাৰনেট অনুসন্ধান প্ৰদানকাৰী (optional) কনফিগাৰ কৰক।

## আৰম্ভণি (Quick Start)

activate কৰাৰ পিছত, Superdav AI Agent স্বয়ংক্রিয়ভাৱে থীম-সচেতন দক্ষতা আৰু আন ক্ষমতাৰ সৈতে আপোনাৰ Gratis AI Agent installation लाई উন্নত কৰে। WordPress admin sidebar ত **Gratis AI Agent** প্যানেলটো খুলক আৰু নতুন বৈশিষ্ট্যসমূহ ব্যৱহাৰ কৰিবলৈ আৰম্ভ কৰক।

**উদাহৰণৰ নিৰ্দেশনা (Example instructions):**

```
Add a contact form to the Contact page.
```

```
Search the web for the latest WordPress security best practices.
```

```
Create a hero section using the active theme's block patterns.
```

## থীম-সচেতন দক্ষতা (Theme-Aware Skills)

Superdav AI Agent স্বয়ংক্রিয়ভাৱে আপোনাৰ active থীম আৰু install কৰা প্লাগইন আৱিষ্কাৰ কৰে, আৰু সহায়কৰ context ত উপযুক্ত থীম-সচেতন দক্ষতা লোড কৰে। উপলব্ধ দক্ষতা সমূহৰ ভিতৰত আছে:

| Skill | Applies To |
|---|---|
| **Block Themes** | থীম যিয়ে WordPress block-based (Full Site Editing) architecture ব্যৱহাৰ কৰে |
| **Classic Themes** | PHP টেমপ্লেট আৰু classic editor ব্যৱহাৰ কৰা পৰম্পৰাগত WordPress থীম |
| **Kadence Blocks** | যি সাইটত Kadence Blocks প্লাগইন active আছে |
| **Kadence Theme** | যি সাইটত Kadence থীম active থীম হিচাপে আছে |

প্ৰতিটো দক্ষতাৰ ওপৰত বিশদ পথপ্ৰদৰ্শকৰ বাবে [Theme-Aware Skills](../../user-guide/configuration/theme-aware-skills) চাওক।

## ইন্টাৰনেট অনুসন্ধান প্ৰদানকাৰী (Internet Search Providers)

Superdav AI Agent v1.10.0 ত ইন্টাৰনেট অনুসন্ধান প্ৰশ্নৰ বাবে Brave Search ৰ বিকল্প হিচাপে **Tavily** যোগ কৰা হৈছে। দুয়োটা প্ৰদানকাৰীয়ে real-time অনুসন্ধান ফলাফল আগবঢ়ায় আৰু ইয়াক **Gratis AI Agent → Settings → Advanced** ত কনফিগাৰ কৰিব পাৰি।

| Provider | Features | Setup |
|---|---|---|
| **Brave Search** | দ্ৰুত, privacy-focused অনুসন্ধান ফলাফল | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | অধিক সমৃদ্ধ অনুসন্ধান ফলাফল, উন্নত গৱেষণা ক্ষমতা | [Tavily API](https://tavily.com/) |

কনফিগাৰেশ্যন নিৰ্দেশনাৰ বাবে [Internet Search](../../user-guide/configuration/internet-search) চাওক।

## সাইট বিল্ডাৰ যোগাযোগ ফৰ্ম (Site Builder Contact Forms)

Site Builder agent এতিয়া চ্যাট ইন্টাৰফেচৰ পৰাই যোগাযোগ ফৰ্ম সৃষ্টি কৰিব পাৰে। কেৱল Site Builder ক যোগাযোগ ফৰ্ম এটা সৃষ্টি কৰিবলৈ কওক, আৰু ই স্বয়ংক্রিয়ভাৱে আপোনাৰ install কৰা ফৰ্ম প্লাগইন (Contact Form 7, WPForms, Fluent Forms, বা Gravity Forms) আৱিষ্কাৰ কৰি ব্যৱহাৰ কৰিবলৈ সাজু ফৰ্ম এটা জেনারেট কৰিব।

বিশদ পথপ্ৰদৰ্শকৰ বাবে [Site Builder Contact Forms](../../user-guide/administration/site-builder-contact-forms) চাওক।

## থীম বিল্ডাৰ বৈশিষ্ট্যসমূহ (Theme Builder Features)

Superdav AI Agent v1.16.0 ত Theme Builder-ত শক্তিশালী নতুন ক্ষমতা যোগ কৰা হৈছে:

### ফটো আপলোড সহ ডিসকভারি সাক্ষাৎকাৰ (Discovery Interview with Photo Upload)

Theme Builderৰ ডিসকভারি সাক্ষাৎকাৰত এতিয়া ফটো আপলোডৰ এটা ধাপ আছে, য'ত আপুনি দৃশ্যমান অনুপ্ৰেৰণা (visual inspiration) প্ৰদান কৰিব পাৰে। আপোনাৰ ব্ৰেণ্ড, প্ৰতিদ্বন্দ্বী সাইট, বা ডিজাইন অনুপ্ৰেৰণাৰ ফটো আপলোড কৰক যাতে Theme Builder ক আপোনাৰ ডিজাইনৰ দিশ বুজিবলৈ সহায় হয়।

বিশদ পথপ্ৰদৰ্শকৰ বাবে [Discovery Interview](./theme-builder/discovery-interview.md) চাওক।

### ডেস্কটপ আৰু মোবাইল প্ৰিভিউ সহ ডিজাইনৰ দিশ (Design Direction with Desktop and Mobile Previews)

Theme Builder-এ আপোনাৰ সম্পূৰ্ণ থীম জেনারেট কৰাৰ আগতে ডেস্কটপ আৰু মোবাইল দুয়োটা ডিভাইচত আপোনাৰ ডিজাইনটো পৰীক্ষা কৰক। বিভিন্ন responsive breakpoints ত real-time previews চাওক।

বিশদ পথপ্ৰদৰ্শকৰ বাবে [Design Direction](./theme-builder/design-direction.md) চাওক।

### হোষ্টেলිටি মেনু (Hospitality Menus)

Theme Builder এতিয়া ресторан, কেফে, বাৰ, আৰু অন্যান্য হোষ্টেলිටি ব্যৱসায়ৰ বাবে গঠনমূলক খাদ্য আৰু পানীয় মেনু পৃষ্ঠা জেনারেট কৰিব পাৰে। মেনুসমূহত মূল্য, বৰ্ণনা, খাদ্যৰ সংবেদনশীল তথ্য (dietary information), আৰু Allergen warnings অন্তৰ্ভুক্ত থাকে।

বিশদ পথপ্ৰদৰ্শকৰ বাবে [Hospitality Menus](./theme-builder/hospitality-menus.md) চাওক।

### নতুন ক্ষমতা (New Abilities)

- **[Generate Logo SVG](./abilities/generate-logo-svg.md)** — স্বয়ংক্রিয় sanitisation সৈতে কাস্টম SVG লোগো সৃষ্টি কৰক।
- **[Validate Palette Contrast](./abilities/validate-palette-contrast.md)** — WCAG অনুপালনৰ বাবে ৰঙৰ জোড়া পৰীক্ষা কৰক।
- **[Create Menu](./abilities/create-menu.md)** — পৃথক নেভিগেশ্যন লেবেল সহ নেভিগেশ্যন মেনু সৃষ্টি কৰক।

উপলব্ধ টুল আৰু Tier 1 site-scrape টুলৰ তথ্যৰ বাবে [Tools and Configuration](./theme-builder/tools.md) চাওক।

## কনফিগাৰেশ্যন (Configuration)

সকলো সেটিং WordPress admin ত **Gratis AI Agent → Settings** ত উপলব্ধ। Superdav AI Agent ले তলত দিয়া কনফিগাৰেশ্যন বিকল্পসমূহ যোগ কৰে:

| Setting | Description |
|---|---|
| **Internet Search Provider** | Brave Search আৰু Tavilyৰ মাজত বাছি লওঁ (optional) |
| **Tavily API Key** | Tavily ইন্টাৰনেট অনুসন্ধানৰ বাবে API key (যদি Tavily বাছি লয়) |

## v1.16.0 ত উন্নতি (Improvements in v1.16.0)

- **AI Client cache** — এতিয়া cross-request persistence ৰ বাবে transients দ্বাৰা সমৰ্থন কৰা হয়, যাৰ ফলত দীঘলীয়া agent কাৰ্য্যত তথ্য হেৰুৱাই নযায়।
- **Tier 1 site-scrape tool** — sd-ai-agent/site-scrape এতিয়া Theme Builder ত ডিফল্টভাৱে উপলব্ধ, যিয়ে প্ৰতিদ্বন্দ্বী আৰু অনুপ্ৰেৰণামূলক ওয়েবসাইট বিশ্লেষণ কৰে।
- **Plugin row action links** — স্পষ্টতাৰ বাবে সংশোধন কৰা আৰু নাম সলনি কৰা হৈছে।

## v1.10.0 ত উন্নতি (Improvements in v1.10.0)

- **WooCommerce integration refactored** — উন্নত বিশ্বাসযোগ্যতা আৰু সামঞ্জস্যৰ বাবে এতিয়া native WooCommerce API ব্যৱহাৰ কৰে।
- **Provider list auto-refresh** — স্বয়ংক্রিয়ভাৱে আৱিষ্কাৰ কৰে যে প্লাগইন activate বা deactivate কৰা হৈছে আৰু উপলব্ধ প্ৰদানকাৰী তালিকা আপডেট কৰে।
- **Enhanced site builder** — উন্নত বহু-ধাপৰ সাইট বিল্ড پلان, উন্নত error recovery সৈতে।
- **Theme-aware skills** — চাৰিটা নতুন বিল্ট-ইন দক্ষতা যিয়ে স্বয়ংক্রিয়ভাৱে আপোনাৰ active থীমৰ লগত খাপ খুৱাই লয়।

## v1.10.0 ত বাগ সংশোধন (Bug Fixes in v1.10.0)
