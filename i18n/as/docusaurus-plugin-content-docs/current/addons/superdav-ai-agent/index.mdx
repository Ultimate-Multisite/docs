# Superdav AI Agent

Superdav AI Agent এটা উন্নত WordPress AI সহায়ক যিয়ে Gratis AI Agent-ক theme-সচেতন দক্ষতা, উন্নত internet সন্ধান ক্ষমতা, site builder বৈশিষ্ট্য, আৰু ঐচ্ছিক automation সংযোজনৰ সৈতে সম্প্ৰসাৰিত কৰে। সংস্কৰণ 1.18.0-এ Google Calendar পঢ়া সঁজুলি, যোগাযোগ আৰু অংশগ্ৰহণকাৰী মেপিং, মানৱ অনুমোদন গেট, reminder পুনৰাবৃত্তি আঁতৰোৱা, TextBee SMS জাননী, পৰিচালিত Superdav সেৱা provisioning, আৰু বিশ্বাসযোগ্য developer workflow-ৰ বাবে পৃথকে বিতৰণ কৰা Advanced companion package যোগ কৰে।

## মুখ্য বৈশিষ্ট্যসমূহ

- **Theme-সচেতন দক্ষতা** — আপোনাৰ সক্ৰিয় WordPress theme আৰু স্থাপিত plugins-ৰ ওপৰত ভিত্তি কৰি নিৰ্দেশনা আৰু ক্ষমতাসমূহ স্বয়ংক্ৰিয়ভাৱে খাপ খুৱায়
- **বহু internet সন্ধান providers** — অধিক সমৃদ্ধ সন্ধান ফলাফল আৰু উন্নত গৱেষণা ক্ষমতাৰ বাবে Brave Search আৰু Tavily-ৰ মাজত বাছনি কৰক
- **Site builder যোগাযোগ forms** — আপোনাৰ স্থাপিত form plugin ব্যৱহাৰ কৰি chat interface-ৰ পৰাই সম্পূৰ্ণ কার্যক্ষম যোগাযোগ forms সৃষ্টি কৰক
- **উন্নত site builder** — উন্নত plugin আৱিষ্কাৰ আৰু ত্ৰুটি পুনৰুদ্ধাৰৰ সৈতে উন্নত বহু-পর্যায়ৰ site build plans
- **অন্তৰ্নিৰ্মিত theme দক্ষতা** — Block Themes, Classic Themes, Kadence Blocks, আৰু Kadence Theme-ৰ বাবে বিশেষ নিৰ্দেশনা
- **উন্নত WooCommerce সংযোজন** — অধিক নিৰ্ভৰযোগ্যতা আৰু compatibility-ৰ বাবে native WooCommerce APIs ব্যৱহাৰ কৰিবলৈ refactor কৰা হৈছে
- **স্বয়ংক্ৰিয় provider চিনাক্তকৰণ** — plugins আৰু themes সক্ৰিয় বা নিষ্ক্ৰিয় কৰা হলে উপলব্ধ plugins আৰু themes স্বয়ংক্ৰিয়ভাৱে চিনাক্ত আৰু refresh কৰে
- **Custom logo generation** — স্বয়ংক্ৰিয় namespace-safe sanitisation-ৰ সৈতে custom SVG logos সৃষ্টি আৰু embed কৰক
- **Palette contrast validation** — আপোনাৰ theme-ত প্ৰয়োগ কৰাৰ আগতে WCAG accessibility compliance-ৰ বাবে ৰঙৰ যোৰ পৰীক্ষা কৰক
- **Photo-enhanced discovery** — design direction-ত প্ৰভাৱ পেলাবলৈ Theme Builder discovery সাক্ষাৎকাৰৰ সময়ত photos upload কৰক
- **Hospitality menus** — restaurants, cafes, আৰু hospitality businesses-ৰ বাবে structured food আৰু beverage menu pages সৃষ্টি কৰক
- **Desktop আৰু mobile previews** — design-direction নিৰ্বাচনৰ সময়ত desktop আৰু mobile devices-ত আপোনাৰ design preview কৰক
- **Tier 1 site scraping** — competitor আৰু inspiration websites বিশ্লেষণৰ বাবে sd-ai-agent/site-scrape এতিয়া default হিচাপে উপলব্ধ
- **Google Calendar পঢ়া সঁজুলি** — configured calendars আৰু events পঢ়ক যাতে automations-এ আগন্তুক schedule context সম্পৰ্কে যুক্তি কৰিব পাৰে
- **যোগাযোগ আৰু অংশগ্ৰহণকাৰী মেপিং** — follow-up work সৃষ্টি কৰাৰ আগতে calendar participants-ক জনা site users আৰু contacts-ৰ সৈতে মিলাওক
- **মানৱ অনুমোদন গেট** — messages পঠিওৱা, work পুনৰাৰম্ভ কৰা, বা পৰিৱৰ্তন কৰাৰ আগতে review-ৰ বাবে sensitive automations স্থগিত ৰাখক
- **Reminder records আৰু deduplication** — ইতিমধ্যে পঠিওৱা reminders track কৰক যাতে repeat automation runs-এ একেজন recipient-ক দুবাৰ notify নকৰে
- **TextBee SMS জাননী** — SMS provider enabled থাকিলে TextBee-ৰ জৰিয়তে configured text-message notifications পঠিয়াওক
- **Managed Superdav service provisioning** — manual endpoint configuration-ৰ সলনি automatic setup-ৰ সৈতে supported sites-ক hosted Superdav endpoints-ৰ সৈতে সংযোগ কৰক
- **Advanced companion package** — filesystem, database, WP-CLI, REST dispatcher, plugin builder, git snapshot, user-management, আৰু benchmark workflows-ৰ বাবে পৃথকে বিতৰণ কৰা trusted-developer tools

## প্ৰয়োজনীয়তা

- WordPress 6.0 বা তাতকৈ উচ্চ
- PHP 8.1 বা তাতকৈ উচ্চ
- WordPress Multisite (network-activated) বা single-site installation
- এটা সক্ৰিয় AI provider connector plugin (Anthropic, OpenAI, বা compatible)
- Gratis AI Agent v1.9.0 বা তাতকৈ উচ্চ (Superdav AI Agent-এ Gratis AI Agent সম্প্ৰসাৰিত কৰে)

## স্থাপন

1. plugin files `/wp-content/plugins/superdav-ai-agent`-লৈ upload কৰক
2. WordPress-ৰ **Plugins** screen-ৰ জৰিয়তে plugin সক্ৰিয় কৰক
3. Gratis AI Agent-ও সক্ৰিয় আছে বুলি নিশ্চিত কৰক
4. **Gratis AI Agent → Settings**-লৈ যাওক আৰু প্ৰয়োজন অনুসৰি আপোনাৰ internet search provider, managed Superdav service connection, Google Calendar, বা TextBee credentials configure কৰক

WordPress.org package-ত core Superdav AI Agent বৈশিষ্ট্যসমূহ অন্তৰ্ভুক্ত। Advanced companion package সেই পৰিৱেশসমূহৰ বাবে পৃথকে build আৰু distribute কৰা হয় য’ত trusted developer tools স্পষ্টভাৱে install আৰু authorise কৰা থাকে।

## দ্ৰুত আৰম্ভণি

সক্ৰিয় হোৱাৰ পাছত, Superdav AI Agent-এ theme-সচেতন দক্ষতা আৰু অতিৰিক্ত ক্ষমতাসমূহৰ সৈতে আপোনাৰ Gratis AI Agent installation স্বয়ংক্ৰিয়ভাৱে উন্নত কৰে। WordPress admin sidebar-ত **Gratis AI Agent** panel খোলক আৰু নতুন বৈশিষ্ট্যসমূহ ব্যৱহাৰ আৰম্ভ কৰক।

**উদাহৰণ নিৰ্দেশনা:**

```
Contact page-ত এটা যোগাযোগ form যোগ কৰক।
```

```
শেহতীয়া WordPress security best practices-ৰ বাবে web-ত সন্ধান কৰক।
```

```
সক্ৰিয় theme-ৰ block patterns ব্যৱহাৰ কৰি এটা hero section সৃষ্টি কৰক।
```

```
কাইলৈৰ configured Google Calendar events review কৰক আৰু attendees-ৰ বাবে reminders draft কৰক।
```

```

```

## Theme-সচেতন দক্ষতা

Superdav AI Agent-এ আপোনাৰ সক্ৰিয় theme আৰু স্থাপিত plugins স্বয়ংক্ৰিয়ভাৱে চিনাক্ত কৰে, আৰু assistant-ৰ context-ত উপযুক্ত theme-সচেতন skill load কৰে। উপলব্ধ skills-ত অন্তৰ্ভুক্ত:

| Skill | প্ৰযোজ্য ক্ষেত্ৰ |
|---|---|
| **Block Themes** | WordPress block-based (Full Site Editing) architecture ব্যৱহাৰ কৰা themes |
| **Classic Themes** | PHP templates আৰু classic editor ব্যৱহাৰ কৰা পৰম্পৰাগত WordPress themes |
| **Kadence Blocks** | Kadence Blocks plugin সক্ৰিয় থকা sites |
| **Kadence Theme** | active theme হিচাপে Kadence theme থকা sites |

প্ৰতিটো skill-ৰ ওপৰত বিস্তৃত নিৰ্দেশনাৰ বাবে [Theme-সচেতন দক্ষতা](../../user-guide/configuration/theme-aware-skills) চাওক।

## Internet Search Providers

Superdav AI Agent v1.10.0-এ internet search queries-ৰ বাবে Brave Search-ৰ বিকল্প হিচাপে **Tavily** যোগ কৰে। দুয়োটা providers-এ real-time search results প্ৰদান কৰে আৰু **Gratis AI Agent → Settings → Advanced**-ত configure কৰিব পাৰি।

| প্ৰদানকাৰী | বৈশিষ্ট্যসমূহ | ছেটআপ |
|---|---|---|
| **Brave Search** | দ্ৰুত, গোপনীয়তা-কেন্দ্ৰিক সন্ধান ফলাফল | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | অধিক সমৃদ্ধ সন্ধান ফলাফল, উন্নত গৱেষণা সক্ষমতা | [Tavily API](https://tavily.com/) |

কনফিগাৰেশ্যন নিৰ্দেশনাৰ বাবে [ইণ্টাৰনেট সন্ধান](../../user-guide/configuration/internet-search) চাওক।

## কেলেণ্ডাৰ, অনুমোদন, আৰু জাননী ৱৰ্কফ্ল’সমূহ

Superdav AI Agent v1.18.0-এ সমল আৰু site-নিৰ্মাণ কাৰ্যৰ বাহিৰলৈ অটোমেশ্যন ৱৰ্কফ্ল’সমূহ বিস্তাৰ কৰে:

- **Google Calendar সঁজুলিসমূহে** calendar credentials উপলব্ধ থাকিলে agent-ক কনফিগাৰ কৰা কেলেণ্ডাৰ আৰু ইভেণ্টৰ বিৱৰণ পঢ়িবলৈ দিয়ে।
- **যোগাযোগ আৰু অংশগ্ৰহণকাৰী সহায়কসমূহে** স্মাৰক বা অনুসৰণ কাৰ্য প্ৰস্তুত কৰাৰ আগতে ইভেণ্টৰ অংশগ্ৰহণকাৰীসকলক WordPress ব্যৱহাৰকাৰী বা পৰিচিত যোগাযোগৰ সৈতে মিলাই দিয়ে।
- **মানৱ অনুমোদন গেটসমূহে** প্ৰস্তাৱিত কাৰ্য এজন অনুমোদিত ব্যৱহাৰকাৰীয়ে পৰ্যালোচনা কৰা পৰ্যন্ত অটোমেশ্যনক ৰখাই থ’বলৈ অনুমতি দিয়ে।
- **স্মাৰক ৰেকৰ্ডসমূহে** অটোমেশ্যন পুনৰ চেষ্টা কৰা বা বাৰে বাৰে সময়সূচী কৰা হ’লে দ্বৈত কেলেণ্ডাৰ বা জাননী স্মাৰক ৰোধ কৰে।
- **TextBee SMS সমৰ্থনে** TextBee প্ৰদানকাৰী কনফিগাৰ কৰা থাকিলে পাঠ-বার্তা জাননী সক্ৰিয় কৰে।

সূচী-সচেতন সহায়, অংশগ্ৰহণকাৰী অনুসৰণ, আৰু সংবেদনশীল বাৰ্তা পঠিওৱাৰ আগতে এতিয়াও মানৱ পৰ্যালোচনা প্ৰয়োজন হোৱা স্মাৰক ৱৰ্কফ্ল’ৰ বাবে এই সঁজুলিসমূহ ব্যৱহাৰ কৰক।

## Site Builder যোগাযোগ ফৰ্ম

Site Builder agent-এ এতিয়া chat interface-ৰ পৰা পোনপটীয়াকৈ যোগাযোগ ফৰ্ম সৃষ্টি কৰিব পাৰে। Site Builder-ক কেৱল এটা যোগাযোগ ফৰ্ম সৃষ্টি কৰিবলৈ কওক, আৰু ই আপোনাৰ ইনষ্টল কৰা form plugin (Contact Form 7, WPForms, Fluent Forms, বা Gravity Forms) স্বয়ংক্ৰিয়ভাৱে চিনাক্ত কৰি ব্যৱহাৰৰ বাবে সাজু এটা ফৰ্ম সৃষ্টি কৰিব।

বিস্তৃত নিৰ্দেশনাৰ বাবে [Site Builder যোগাযোগ ফৰ্ম](../../user-guide/administration/site-builder-contact-forms) চাওক।

## Theme Builder বৈশিষ্ট্যসমূহ

Superdav AI Agent v1.16.0-ৰ Theme Builder-ত শক্তিশালী নতুন সক্ষমতাসমূহ অন্তৰ্ভুক্ত আছে:

### ফটো আপলোডসহ আৱিষ্কাৰ সাক্ষাৎকাৰ

Theme Builder আৱিষ্কাৰ সাক্ষাৎকাৰত এতিয়া এটা ফটো আপলোড স্তৰ অন্তৰ্ভুক্ত, য’ত আপুনি দৃশ্যগত অনুপ্ৰেৰণা দিব পাৰে। Theme Builder-এ আপোনাৰ ডিজাইন দিশ বুজিবলৈ সহায় পাবলৈ আপোনাৰ brand, প্ৰতিদ্বন্দ্বী site, বা ডিজাইন অনুপ্ৰেৰণাৰ ফটো আপলোড কৰক।

বিস্তৃত নিৰ্দেশনাৰ বাবে [আৱিষ্কাৰ সাক্ষাৎকাৰ](./theme-builder/discovery-interview.md) চাওক।

### Desktop আৰু Mobile পূৰ্বদৰ্শনৰ সৈতে ডিজাইন দিশ

Theme Builder-এ আপোনাৰ সম্পূৰ্ণ theme সৃষ্টি কৰাৰ আগতে desktop আৰু mobile ডিভাইচ দুয়োটাতে আপোনাৰ ডিজাইন পৰ্যালোচনা কৰক। একাধিক responsive breakpoint-ত real-time পূৰ্বদৰ্শন চাওক।

বিস্তৃত নিৰ্দেশনাৰ বাবে [ডিজাইন দিশ](./theme-builder/design-direction.md) চাওক।

### আতিথ্য মেনুসমূহ

Theme Builder-এ এতিয়া ৰেস্তোৰাঁ, কেফে, বাৰ, আৰু অন্যান্য আতিথ্য ব্যৱসায়ৰ বাবে গঠনযুক্ত খাদ্য আৰু পানীয় মেনু পৃষ্ঠা সৃষ্টি কৰিব পাৰে। মেনুসমূহত মূল্য, বিৱৰণ, খাদ্য-সম্পৰ্কীয় তথ্য, আৰু allergen সতৰ্কবাণী অন্তৰ্ভুক্ত থাকে।

বিস্তৃত নিৰ্দেশনাৰ বাবে [আতিথ্য মেনুসমূহ](./theme-builder/hospitality-menus.md) চাওক।

- **[Palette Contrast বৈধ কৰক](./abilities/validate-palette-contrast.md)** — WCAG অনুপালনৰ বাবে ৰঙৰ যোৰ পৰীক্ষা কৰক
- **[মেনু সৃষ্টি কৰক](./abilities/create-menu.md)** — সুকীয়া navigation label-সহ navigation মেনু সৃষ্টি কৰক

উপলব্ধ সঁজুলি আৰু Tier 1 site-scrape সঁজুলিৰ বিষয়ে তথ্যৰ বাবে [সঁজুলি আৰু কনফিগাৰেশ্যন](./theme-builder/tools.md) চাওক।

## কনফিগাৰেশ্যন

সকলো ছেটিং WordPress admin-ৰ **Gratis AI Agent → Settings**-ৰ অধীনত আছে। Superdav AI Agent-এ তলৰ কনফিগাৰেশ্যন বিকল্পসমূহ যোগ কৰে:

| ছেটিং | বিৱৰণ |
|---|---|
| **Internet Search Provider** | Brave Search আৰু Tavily-ৰ মাজৰ পৰা বাছনি কৰক (ঐচ্ছিক) |
| **Tavily API Key** | Tavily ইণ্টাৰনেট সন্ধানৰ বাবে API key (যদি Tavily বাছনি কৰা হয়) |
| **Managed Superdav Service** | স্বয়ংক্ৰিয় endpoint provisioning-সহ সমৰ্থিত site-সমূহক hosted Superdav service-ৰ সৈতে সংযোগ কৰক |
| **Google Calendar Credentials** | agent-এ কেলেণ্ডাৰ আৰু ইভেণ্ট পঢ়িব পৰাকৈ কেলেণ্ডাৰ অভিগম কনফিগাৰ কৰক |
| **TextBee SMS Provider** | অনুমোদিত SMS জাননী ৱৰ্কফ্ল’ৰ বাবে TextBee credentials কনফিগাৰ কৰক |

## উন্নত সহচৰ পেকেজ

Superdav AI Agent v1.18.0-এ বিশ্বাসযোগ্য developer ৱৰ্কফ্ল’ৰ বাবে পৃথকে বিতৰণ কৰা Advanced সহচৰ পেকেজ যোগ কৰে। এই সক্ষমতাসমূহ WordPress.org core পেকেজত bundled নহয়। প্ৰশাসকসকলে agent-ক developer tooling-ত অভিগমৰ বাবে স্পষ্টভাৱে বিশ্বাস কৰা site-সমূহতহে সহচৰ পেকেজ ইনষ্টল কৰক।

Advanced সঁজুলিসমূহত filesystem কাৰ্যসমূহ, database অভিগম, WP-CLI, REST dispatcher ৱৰ্কফ্ল’, plugin builder utility, git snapshot, user-management সহায়ক, আৰু benchmark tooling অন্তৰ্ভুক্ত। এই সঁজুলিসমূহ বিশ্বাসযোগ্য প্ৰশাসকসকলৰ মাজতে সীমাবদ্ধ ৰাখক আৰু production site-ত সক্ষম কৰাৰ আগতে approval-gate কনফিগাৰেশ্যন পৰ্যালোচনা কৰক।
