# Action: wp_ultimo_affiliatewp_load {#action-wpultimoaffiliatewpload}

যেতিয়া সকলো ডিপেণ্ডেন্সি (dependencies) লোড হৈ যায়, তেতিয়া এই অ্যাকশনটো ট্ৰিগাৰ হয়।

প্লাগইন ডেভেলপাৰসকলক ইয়াৰ জৰিয়তে নতুন ফাংচনালিটি (functionality) যোগ কৰিবলৈ সুবিধা দিয়ে। উদাহৰণস্বৰূপে, নতুন হোস্টিং প্ৰোভাইডেৰ (Hosting providers) সমৰ্থন যোগ কৰিব পাৰি, ইত্যাদি।
