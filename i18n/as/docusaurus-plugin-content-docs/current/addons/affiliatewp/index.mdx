# AffiliateWP

Multisite Ultimate: AffiliateWP Integration

আপোনাৰ Ultimo Network-ৰ ক্লায়েন্ট বেছ বৃদ্ধি কৰিবলৈ শক্তিশালী AffiliateWP ব্যৱহাৰ কৰক!

## Installation {#installation}

১. 'ultimate-multisite-git ' ফাইলটো '/wp-content/plugins/' ডাইৰেক্টৰীত আপলোড কৰক।
২. WordPress-ৰ 'Plugins' মেনুৰ মাজেৰে প্লাগইনটো অ্যাক্টিভেট কৰক।
৩. প্লাগইনটো সেট কৰিবলৈ ধাপে ধাপে (step by step) Wizard অনুসৰণ কৰক।

## Usage Guide {#usage-guide}

[AffiliateWP](https://affiliatewp.com/) আপোনাৰ অনলাইন ব্যৱসায়ৰ বাবে এটা affiliate program গঢ়ি তোলাৰ সময়ত এটা প্ৰয়োজনীয় সমাধান হৈ পৰিছে। Ultimate Multisite: AffiliateWP হৈছে এটা সেতুবন্ধন যিয়ে আপোনাক আপোনাৰ প্ৰিমিয়াম নেটৱৰ্কৰ বাবে একেটা বস্তু গঢ়ি তুলিবলৈ সক্ষম কৰিব।

## Getting Started {#getting-started}

প্ৰথমে, আপোনাৰ ওলাই [**AffiliateWP**](https://affiliatewp.com/) **অ্যাক্টিভ** হোৱাটো **প্ৰয়োজনীয়**! যদি আপোনাৰ ওলাই **ইয়াৰ সৈতে ইন্টিগ্রেট হোৱা প্লাগইন** নাই, তেন্তে আপুনি আমাৰ add-on ব্যৱহাৰ কৰিব নোৱাৰিব। যদি আপুনি বিচাৰে যে আপোনাৰ ক্লায়েন্টসকলেও ই তেওঁলোকৰ সাইটত ব্যৱহাৰ কৰিব, তেন্তে AffiliateWP কেৱল আপোনাৰ মূল সাইটতহে অ্যাক্টিভ থাকিব।

ইয়াৰ উপৰিও, আপোনাৰ **Ultimate Multisite** আৰু **Ultimate Multisite: AffiliateWP** দুয়োটাই **network-wide** **অ্যাক্টিভেটেড** হোৱাটো প্ৰয়োজন।

## Adding Ultimate Multisite as an AffiliateWP Integration {#adding-ultimate-multisite-as-an-affiliatewp-integration}

আপোনাৰ মূল সাইটত, 'Affiliates Settings'লৈ যাওক, আৰু 'Integrations' ট্ৰিখনত Ultimate Multisite চেক কৰক। ইয়াৰ ফলত [AffiliateWP](https://affiliatewp.com/) लाई Ultimate Multisite sign-up flow ৰ পৰা ভিজিট আৰু conversion (পරිৱৰ্তন) ট্ৰেক কৰিবলৈ আৰম্ভ কৰিব বুলি জনাওঁ।

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_তলত থকা Ultimate Multisite বক্সটো চেক কৰি সেভ কৰক_

ইমানহে কৰিব লাগিব। referral links ৰ পৰা অহা ভিজিটবোৰ ট্ৰেক কৰা হ'ব আৰু যেতিয়া সেই ব্যৱমৃতৱে পিছত তেওঁৰ প্ৰথম পেমেন্ট কৰিব, তেতিয়া conversion এটা সৃষ্টি হ'ব আৰু সেই conversionৰ বাবে দায়ী affiliate-লৈ এটা কমমিছন থৈ দিয়া হ'ব।

যদিও, কেৱল নতুন সৃষ্টি হোৱা ব্যৱমৃতৰ পৰা পোৱা প্ৰথম পেমেন্টটোৱহে ট্ৰেক কৰা হ'ব। recurring referral payments সক্ষম কৰিবলৈ, আপোনাৰ [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) প্ৰয়োজন হ'ব।

## Enabling Recurring Referrals {#enabling-recurring-referrals}

[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) install কৰাৰ পিছত, 'Affiliates Settings'ৰ 'Recurring Referrals' ট্ৰিখনলৈ যাওক, আৰু recurring referrals ব্যৱহাৰ কৰক।

_যিমান সহজ: Recurring Referrals সক্ষম কৰক আৰু আপুনি সাজু!_

আকৌ এবাৰ, ইমানহে কৰিব লাগিব!

এতিয়া, যেতিয়া এটা referred account मा নতুন পেমেন্ট পোৱা যায়, তেতিয়া দায়ী affiliate-লৈ স্বয়ংক্রিয়ভাৱে তেওঁৰ নিজৰ অংশটো লাভ কৰিব!
