# Support Tickets

**Support Tickets** হৈছে Ultimate Multisite-ৰ এটা শক্তিশালী addon, যিয়ে আপোনাৰ WordPress network-লৈ এটা সম্পূৰ্ণ সাপোর্ট টিকেট ব্যৱস্থা যোগ কৰে। ই SaaS platforms, WaaS (Website as a Service) প্ৰদানকাৰীসকল, আৰু multisite network-সমূহৰ বাবে উপযুক্ত, যিয়ে network আৰু ব্যক্তিগত site দুয়োটা স্তৰত ग्राहकৰ সহায় আগবঢ়াব লাগিব।

## Support Tickets কিয় বাছি ল'ব?

* **বহু-স্তৰৰ সহায় (Multi-Level Support)**: network प्रशासকসকলে network-ব্যাপী টিকেটসমূহ ব্যৱস্থ কৰে, আনহাতে siteৰ মালিকসকলে তেওঁলোকৰ নিজৰ ग्राहक সহায় ব্যৱস্থা কৰে।
* **পেশাদাৰী কার্যপ্ৰবাহ (Professional Workflow)**: প্ৰয়োজনীয়তা (priorities), স্থিতি (statuses), আৰু দায়িত্ব (assignments) সহ টিকেটৰ সম্পূৰ্ণ জীৱনচক্ৰ ব্যৱস্থাপনা।
* **ग्राहकৰ অভিজ্ঞতা (Customer Experience)**: এটা ধুনীয়া, responsive interface যিয়ে ग्राहकসকলক সহায় ল'বলৈ সহজ কৰি তোলে।
* **নিৰাপদ আৰু বিশ্বাসী (Secure & Reliable)**: WordPress-ৰ নিৰাপত্তা উত্তম প্ৰথা আৰু অনুকূলিত ডেটাবেছ প্ৰদৰ্শনৰে নিৰ্মাণ কৰা।
* **সহজ সংহতি (Seamless Integration)**: Ultimate Multisite-ৰ client আৰু site ব্যৱস্থাপনাৰ সৈতে নিখুঁতভাৱে কাম কৰে।

## মূল বৈশিষ্ট্যসমূহ (Key Features)

**সম্পূৰ্ণ টিকেট ব্যৱস্থাপনা (Complete Ticket Management)**

* সহজে উল্লেখ কৰিবলৈ অনন্য টিকেট নম্বৰিং ব্যৱস্থা।
* কনফিগাৰেবল প্ৰয়োজনীয়তা স্তৰ (Low, Normal, High, Critical)।
* বিস্তৃত স্থিতি কার্যপ্ৰবাহ (Open, In Progress, Waiting, Resolved, Closed)।
* টিকেটৰ বৰ্গীকৰণ (General, Technical, Billing, Feature Request, Bug Report)।
* ঐচ্ছিক auto-assignment সহ টিকেটসমূহক সহায় কৰ্মীসকলক (support agents) লৈ দিয়া।

**থ্ৰীডেড কথোপকথন (Threaded Conversations)**

* টিকেটৰ উত্তৰৰ বাবে সম্পূৰ্ণ কথোপকথন থ্ৰীডিং।
* কৰ্মচাৰী আৰু ग्राहकৰ উত্তৰৰ বাবে পৃথক শৈলী।
* ব্যক্তিগত কৰ্মচাৰী নোট যাৰ বাবে ग्राहकসকলে দেখা নোৱাৰে।
* real-time AJAX-ভিত্তিক interface।
* formatting বিকল্প সহ আধুনিক text editor।

**ফাইল সংযুক্তি (File Attachments)**

* drag-and-drop সমৰ্থন সহ নিৰাপদ ফাইল আপলোড।
* কনফিগাৰেবল ফাইল ধৰণ আৰু আকাৰৰ সীমাবদ্ধতা।
* ক্ষতিকাৰক ফাইল ধৰণৰ পৰা উন্নত নিৰাপত্তা।
* প্ৰৱেশ নিয়ন্ত্ৰণ সহ সুরক্ষিত ডাউনলোড।

**স্মাৰ্ট notificatio**

* নতুন টিকেট, উত্তৰ, আৰু স্থিতি পৰিৱৰ্তনৰ বাবে স্বয়ংক্রিয় ইমেইল সতৰ্কবাণী।
* কাস্টমাইজযোগ্য ইমেইল টেমপ্লেট।
* কৰ্মচাৰী আৰু ग्राहकসকলৰ বাবে role-based notificatio।
* ব্যক্তিগতকৰণ কৰা বাৰ্তাৰ বাবে dynamic template variables।

**শক্তিশালী Admin Interface**

* sorting আৰু filtering সহ উন্নত টিকেট তালিকা।
* স্থিতি, প্ৰয়োজনীয়তা, আৰু দায়িত্বৰ বাবে দ্ৰুত সম্পাদনা (Quick edit)।
* দক্ষ ব্যৱস্থাপনাৰ বাবে bulk actions।
* প্ৰদৰ্শন সূচক সহ statistics dashboard।
* সম্পূৰ্ণ সেটিংছ প্যানেল।

**ধুনীয়া Customer Portal**

* সকলো ডিভাইচত clean, responsive interface।
* Mobile-first design।
* উন্নত অনুসন্ধান আৰু filtering।
* formatting toolbar সহ আধুনিক editor।
* ব্যৱহাৰ কৰিবলৈ সহজ টিকেট জমা দিয়া forms।

**নমনীয় সংহতি (Flexible Integration)**

* shortcodes বা Ultimate Multisite UI elements ব্যৱহাৰ কৰক।
* আপোনাৰ site-ৰ যিকোনো ঠাইত টিকেট forms আৰু তালিকা প্ৰদৰ্শন কৰক।
* multisite network structure লগত নিৰবিচ্ছিন্নভাৱে কাম কৰে।
* hooks আৰু filters লগত সম্প্রসারণযোগ্য।

## ক'ৰ বাবে উপযুক্ত (Perfect For)

* ग्राहक সহায়ৰ প্ৰয়োজন হোৱা SaaS platforms।
* বহুতো client site ব্যৱস্থাপনা কৰা WaaS providers।
* স্তৰীকৃত সহায় থকা Membership sites।
* বহুতো brand ব্যৱস্থাপনা কৰা network प्रशासক।
* ग्राहक সেৱা আগবঢ়োৱা siteৰ মালিক।

## স্থাপন (Installation)

1. নিশ্চিত কৰক যে Ultimate Multisite plugin আপোনাৰ WordPress network-ত install আৰু activate কৰা হৈছে।
2. addon ফাইলসমূহ আপোনাৰ `/wp-content/plugins/` directory লৈ আপলোড কৰক।
3. WordPress-ৰ 'Plugins' মেনুৰ মাজেৰে network-ত plugin টি activate কৰক।
4. আপোনাৰ সেটিংছ কনফিগাৰ কৰিবলৈ Network Admin > Support Tickets লৈ যাওক।

## সঘনে কৰা প্ৰশ্নসমূহ (Frequently Asked Questions)

## ইয়াৰ বাবে Ultimate Multisite কি প্ৰয়োজন?

হয়, এই addon-টো Ultimate Multisite install আৰু active হোৱাটো প্ৰয়োজন। ই Ultimate Multisite-ৰ client আৰু site ব্যৱস্থাপনা বৈশিষ্ট্যসমূহৰ সৈতে গভীৰভাৱে সংহতি কৰে।

## কি siteৰ মালিকসকলে তেওঁলোকৰ নিজৰ ग्राहक সহায় ব্যৱস্থা কৰিব পাৰেনে?

নিশ্চয়! siteৰ মালিকসকলে তেওঁলোকৰ নিজৰ ग्राहकসকলক সহায় আগবঢ়াবলৈ টিকেট ব্যৱস্থা ব্যৱহাৰ কৰিব পাৰে, আনহাতে network प्रशासকসকলে network-ব্যাপী সহায় টিকেটসমূহ ব্যৱস্থ কৰিব পাৰে।

## ग्राहकসকলে কেনেকৈ টিকেট জমা কৰে?

আপুনি shortcodes বা Ultimate Multisite UI elements ব্যৱহাৰ কৰি আপোনাৰ site-ৰ যিকোনো ঠাইত টিকেট জমা দিয়া forms যোগ কৰিব পাৰে। ग्राहकসকলে কেৱল form টো ভৰাই তেওঁলোকৰ অনুৰোধ জমা কৰে।

## মই টিকেটৰ স্থিতি আৰু প্ৰয়োজনীয়তা কাস্টমাইজ কৰিব পাৰেনে?

হয়! আপুনি আপোনাৰ সহায় কার্যপ্ৰবাহৰ সৈতে মিলিবলৈ network admin settings ত প্ৰয়োজনীয়তা স্তৰ, স্থিতি, আৰু টিকেট ধৰণসমূহ সম্পূৰ্ণ কাস্টমাইজ কৰিব পাৰে।

## ইয়াৰ বাবে ফাইল সংযুক্তি সমৰ্থন কৰে নেকি?

হয়, ग्राहक আৰু কৰ্মচাৰী দুয়োটাৰ বাবে টিকেট আৰু উত্তৰত ফাইল সংযুক্তি কৰিব পাৰি। নিৰাপত্তাৰ বাবে আপোনাৰ অনুমোদিত ফাইল ধৰণ আৰু আকাৰৰ সীমাৰ ওপৰত সম্পূৰ্ণ নিয়ন্ত্ৰণ আছে।

## notificatio কেনেকৈ ব্যৱস্থাপনা কৰা হয়?

এই ব্যৱস্থাই নতুন টিকেট, উত্তৰ, আৰু স্থিতি পৰিৱৰ্তনৰ বাবে স্বয়ংক্রিয় ইমেইল notificatio প্ৰেণ কৰে। আপুনি সেটিংছত সকলো ইমেইল টেমপ্লেট কাস্টমাইজ কৰিব পাৰে।

## মই টিকেটসমূহক নির্দিষ্ট সহায় কৰ্মীসকলক লৈ যাব পাৰেনে?

হয়, আপুনি টিকেটসমূহক কৰ্মীসকলক হাতেৰে লৈ যাব পাৰে বা টিকেটসমূহ স্বয়ংক্রিয়ভাৱে বিতৰণ কৰিবলৈ auto-assignment সক্ষম কৰিব পাৰে।

## কৰ্মচাৰীসকলৰ বাবে ব্যক্তিগত নোট আছে নেকি?

হয়, কৰ্মচাৰী সদস্যসকলে ব্যক্তিগত উত্তৰ যোগ কৰিব পাৰে যিবোৰ কেৱল আন কৰ্মচাৰী সদস্যসকলৰ বাবেহে দৃশ্যমান, ग्राहकসকলৰ বাবে নহয়।

## ই mobile-friendly নেকি?

নিশ্চয়! সমগ্ৰ interface টো mobile-first approach লৈ নিৰ্মাণ কৰা হৈছে আৰু সকলো ডিভাইচত ধুনীয়াভাৱে কাম কৰে।

## মই shortcodes ব্যৱহাৰ কৰি টিকেট তালিকা আৰু forms প্ৰদৰ্শন কৰিব পাৰেনে?

হয়! উপলব্ধ shortcodes:
* `[wu_support_tickets]` - ग्राहकৰ টিকেট তালিকা প্ৰদৰ্শন কৰে
* `[wu_support_ticket_form]` - নতুন টিকেট জমা দিয়া form প্ৰদৰ্শন কৰে
* `[wu_support_ticket_view ticket_id="123"]` - নির্দিষ্ট টিকেট প্ৰদৰ্শন কৰে

## কি ফাইল ধৰণ আপলোড কৰিব পাৰি?

আপুনি সেটিংছত অনুমোদিত ফাইল ধৰণ কনফিগাৰ কৰিব পাৰে। এই ব্যৱস্থাত ক্ষতিকাৰক ফাইলৰ বিৰুদ্ধে ৰক্ষা কৰিবলৈ উন্নত নিৰাপত্তা স্কেনিং অন্তৰ্ভুক্ত।

## মই টিকেট ডেটা export কৰিব পাৰেনে?

হয়, addon-টোৱে টিকেট ডেটা আৰু statistics-ৰ বাবে export ক্ষমতা অন্তৰ্ভুক্ত কৰিছে।
