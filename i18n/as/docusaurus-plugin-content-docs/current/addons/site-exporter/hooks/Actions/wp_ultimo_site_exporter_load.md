# Action: wp_ultimo_site_exporter_load

এই অ্যাকশনটো তেতিয়া ট্ৰিগাৰ হয় যেতিয়া সকলো প্ৰয়োজনীয়তা (dependencies) লোড হৈ যায়।

প্লাগইন ডেভেলপাৰসকলক নতুন কাৰ্য্যক্ষমতা (functionality) যোগ কৰিবলৈ অনুমতি দিয়ে। উদাহৰণস্বৰূপে, নতুন হোস্টিং প্ৰোভাইডেৰৰ বাবে সমৰ্থন যোগ কৰা ইত্যাদি।
