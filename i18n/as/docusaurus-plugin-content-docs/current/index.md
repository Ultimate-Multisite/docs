# Ultimate Multisiteৰ দস্তাবেজ

Ultimate Multisiteৰ দস্তাবেজলৈ স্বাগতম। Ultimate Multisite-এ WordPress Multisite-ক এটা সম্পূৰ্ণ Website as a Service (WaaS) প্ল্যাটফৰ্মত ৰূপান্তৰিত কৰে।

## দস্তাবেজৰ অংশসমূহ {#documentation-sections}

### [প্ৰেমীৰ গাইড](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

আপোনাৰ WaaS প্ল্যাটফৰ্ম সেট কৰিবলৈ, কনফিগাৰ কৰিবলৈ আৰু ব্যৱস্থাপনা কৰিবলৈ প্ৰয়োজনীয় সকলো তথ্য ইয়াত পোৱা যায়। ইয়াৰত ইনষ্টলেচন, পেমেন্ট গেটৱে, ডোমেইন ম্যাপিং, ক্লায়েন্ট ব্যৱস্থাপনা আৰু ইয়াৰ বাহিৰে আন বিষয়বোৰ কওঁ।

### [ডেভেলপৰ ডকছ](/developer/hooks) {#developer-documentation}

Ultimate Multisite ওপৰত কাম কৰা ডেভেলপাৰসকলৰ বাবে এটা প্ৰযুক্তিগত তথ্যকোষ। ইয়াৰত hooks ৰ তথ্যকোষ, REST API ডক্তৰিন, আৰু addon ডেভেলপমেণ্ট গাইডবোৰ অন্তৰ্ভুক্ত।

### [অ্যাড-অনসমূহ](/addons) {#addons}

পেইড addon plugin-ৰ বাবে দস্তাবেজ, যিয়ে Ultimate Multisite-ক WooCommerce ইন্টিগ্ৰেচন, বিশ্লেষণ (analytics), ডোমেইন বিক্ৰী আৰু ইয়াৰ দৰে আন বৈশিষ্ট্যৰে বঢ়াই তোলে।
