# পাঠ ৪: Niche Templates নিৰ্মাণ কৰা {#lesson-4-building-niche-templates}

Templates হৈছে আপোনাৰ বিশেষ মূল্যৰ মূল ভিত্তি। এজন ফিটনেস ষ্টুডিঅ’ৰ মালিক যিয়ে সাইন আপ কৰে আৰু তেওঁৰ ওচৰত এনে এটা сайт দেখা পায় যিটো ইতিমধ্যে এটা ফিটনেস ওয়েবসাইটৰ দৰে লাগিছে — য'ত ক্লাছৰ সময়সূচী, প্ৰশিক্ষকৰ প্ৰোফাইল আৰু সঠিক ছবি আছে — তেওঁ এটা খালি ক্যানভাস দেখি থকা মানুহজনক ধৰি ৰখিবলৈ বহুগুণে বেছি প্ৰৱণ।

## আমি ক'ত থমকিছিলো {#where-we-left-off}

Ultimate Multisite install আৰু configure কৰি এটা কাৰ্য্যক্ষম FitSite network আছে। এতিয়া আমি সেই Templates সমূহ নিৰ্মাণ কৰিম যাৰ বাবে FitSite ফিটনেস ব্যৱসায়ৰ বাবে বিশেষভাৱে নিৰ্মাণ কৰা বুলি অনুভৱ কৰে।

## Niche Templates কিয় গুৰুত্বপূৰ্ণ {#why-niche-templates-matter}

সাধাৰণ Templates সমূহ আপোনাৰ গ্ৰাহকসকলক কঠিন কাম কৰিবলৈ বাধ্য কৰে: তেওঁলোকক কি পৃষ্ঠাৰ প্ৰয়োজন, কি বিষয়বস্তুৰ গঠন কাৰ্যকৰী, আৰু তেওঁলোকৰ উদ্যোগৰ বাবে কেনেকৈ সঠিকভাৱে দেখুৱাব পাৰি—এইবোৰ বিচাৰি উলিয়াবলৈ। Niche Templates সমূহ এই বাধা বা জটিলতা আঁতৰাই দিয়ে।

FitSite লৈ সাইন আপ কৰা এজন ফিটনেস ষ্টুডিঅ’ৰ মালিকক তলৰ কথাখিনি দেখা উচিত:

- এটা হোমপেজ যিটো এটা ফিটনেস ষ্টুডিঅ’ৰ ওয়েবসাইটৰ দৰে লাগিছে
- ক্লাছ, সময়সূচী, প্ৰশিক্ষক আৰু মূল্যৰ বাবে পৃষ্ঠা ইতিমধ্যে সৃষ্টি কৰা হৈছে
- ছবি আৰু নমুনা বিষয়বস্তু যিটো তেওঁলোকৰ উদ্যোগৰ সৈতে মিল আছে
- এটা ডিজাইন যিটো ফিটনেছৰ বাবে পেছাদাৰী আৰু ব্র্যান্ডৰ সৈতে মিল থকা অনুভৱ দিয়ে

তেওঁলোকে তেওঁলোকৰ বিৱৰণ ভৰাই দিয়ে। তেওঁলোকে শূন্যৰ পৰা আৰম্ভ নকৰে।

## আপোনাৰ Templates পৰিকল্পনা কৰা {#planning-your-templates}

নিৰ্মাণ কৰাৰ আগতে, আপুনি কি Templates আগবঢ়াব লাগিব, সেয়া সিদ্ধান্ত লওক। FitSite-ৰ বাবে, আমি তিনিটা নিৰ্মাণ কৰিম:

### Template 1: Studio Essential {#template-1-studio-essential}

সৰু ষ্টুডিঅ’ আৰু ব্যক্তিগত প্ৰশিক্ষকসকলৰ বাবে।

- হীৰো ছবি, ক্লাছৰ হাইলাইট আৰু কাৰ্য্য কৰিবলৈ আহ্বান (Call to action) থকা **হোমপেজ**
- ষ্টুডিঅ’ৰ কাহিনী আৰু লক্ষ্য থকা **About** পৃষ্ঠা
- সময়সূচী বিন্যাস থকা **Classes** পৃষ্ঠা
- প্ৰোফাইল কাৰ্ড থকা **Trainers** পৃষ্ঠা
- স্থানৰ মানচিত্ৰ আৰু ফৰ্ম থকা **Contact** পৃষ্ঠা
- ফিটনেছৰ বাবে উপযুক্ত ৰংৰ স্কিম সহ **clean, modern design**

### Template 2: Gym Pro {#template-2-gym-pro}

বেছি সেৱা থকা প্ৰতিষ্ঠিত জিমৰ বাবে।

- Studio Essential-ত থকা সকলো বস্তু, লগতে:
- মূল্যৰ টেবুল থকা **Membership** পৃষ্ঠা
- সুবিধাৰ ছবিৰ বাবে **Gallery** পৃষ্ঠা
- ফিটনেছ টিপ আৰু বাতৰিৰ বাবে **Blog** অংশ
- হোমপেজে **Testimonials** অংশ
- **বেছি প্ৰকট ব্র্যান্ডিং** বিকল্প

### Template 3: Fitness Chain {#template-3-fitness-chain}

বহু স্থান থকা ব্যৱসায়ৰ বাবে।

- Gym Pro-ত থকা সকলো বস্তু, লগতে:
- বহু সুবিধাৰ তালিকা থকা **Locations** পৃষ্ঠা
- **Franchise/location** উপ-পৃষ্ঠা Template
- স্থান-ভিত্তিক বিৱৰণ সহ **Centralized branding**
- স্থানসমূহৰ বাবে **Staff directory**

## এটা Template Site নিৰ্মাণ কৰা {#building-a-template-site}

Ultimate Multisite-ত, এটা Template মানে কেৱল এটা WordPress site যিটো ধৰণৰ নতুন গ্ৰাহকৰ сайт হ'ব, তাৰ বাবে configure কৰা হৈছে। ইয়াক কেনেকৈ সৃষ্টি কৰিব পাৰি, তাৰ পদ্ধতি তলত দিয়া হ'ল:

### Step 1: Template Site সৃষ্টি কৰা {#step-1-create-the-template-site}

1. আপোনাৰ network admin-ত **Sites > Add New** লৈ যাওক
2. `template-studio-essential` নামৰ এটা site সৃষ্টি কৰক
3. এই সাইটটোৱে আপোনাৰ কাৰ্য্যক্ষম ক্যানভাস হৈ পৰে

### Step 2: Theme install আৰু configure কৰা {#step-2-install-and-configure-the-theme}

Template site dashboard-লৈ সলনি হ'ক আৰু:

1. ফিটনেস ব্যৱসায়ৰ বাবে উপযুক্ত এটা theme activate কৰক
2. theme setting, color, আৰু typography configure কৰক
3. ফিটনেছৰ বাবে উপযুক্ত navigation সহ header আৰু footer set কৰক

:::tip Theme Selection
এটা এনে theme বাছি লওক যিটো ফিটনেছৰ বাবে ভাল লাগিব বুলি যথেষ্ট নমনীয়, কিন্তু ইমান জটিল নহ'ব যে আপোনাৰ গ্ৰাহকসকলে ইয়াক ব্যৱস্থাপনা কৰিব নোৱাৰে। Astra, GeneratePress, বা Kadence जस्ता theme সমূহ ভাল কাৰণে যে সিহঁত lightweight, customizable, আৰু ভালদৰে সমৰ্থন কৰা।
:::

### Step 3: Pages সৃষ্টি কৰা {#step-3-create-the-pages}

তলৰ বিষয়বস্তুৰে প্ৰতিটো পৃষ্ঠা নিৰ্মাণ কৰক:

- **Placeholder content** যিটো ফিটনেছৰ বাবে স্বাভাৱিকভাৱে পঢ়ি যায় ("Welcome to [Studio Name]" নহয় "Lorem ipsum")
- ফিটনেছ কাৰ্য্যকলাপ দেখুৱাই থকা বিনামূলীয়া স্টক ফটো স্থানৰ পৰা **Placeholder images**
- page builder বা block editor ব্যৱহাৰ কৰি **Functional layouts**

Placeholder content ক যিমান সম্ভৱ শিক্ষণীয় কৰি তুলক। সাধাৰণ filler text কোৱাৰ সলনি, এনে বিষয়বস্তু লিখক: "এইটো আপোনাৰ ষ্টুডিঅ’ আৰু ই কেনেকৈ বিশেষ বুলি এটা চমু বিৱৰণৰে সলনি কৰক। আপোনাৰ প্ৰশিক্ষণৰ দৰ্শন, বছৰৰ অভিজ্ঞতা, বা গ্ৰাহকসকলে কি আশা কৰিব পাৰে, সেয়া উল্লেখ কৰক।"

### Step 4: Plugins configure কৰা {#step-4-configure-plugins}

ফিটনেস ষ্টুডিঅ’ৰ বাবে প্ৰয়োজনীয় plugins install আৰু activate কৰক:

- এটা booking বা scheduling plugin (যদি আপোনাৰ plan tier-লৈ લાગু হয়)
- এটা contact form plugin
- এটা SEO plugin (ফিটনেছ-সম্পৰ্কীয় ডিফল্টৰ লগত pre-configured)

### Step 5: Template হিচাপে চিহ্নিত কৰা {#step-5-mark-as-template}

1. **Ultimate Multisite > Sites** লৈ যাওক
2. template site টি edit কৰক
3. **Site Template** toggle টি enable কৰক

আপুনি যি Templates আগবঢ়াব বিচাৰে, তাৰ বাবে এই প্ৰক্ৰিয়াটো পুনৰাবৃত্তি কৰক।

## Template Quality Checklist {#template-quality-checklist}

এটা Template উপলব্ধ কৰাৰ আগতে, তলৰ কথাখিনি পৰীক্ষা কৰক:

- [ ] সকলো পৃষ্ঠা সঠিকভাৱে load হয় আৰু পেছাদাৰী দেখায়
- [ ] Placeholder content সহায়ক আৰু niche-specific হয়
- [ ] ছবি উপযুক্ত আৰু সঠিকভাৱে লাইচেঞ্চ কৰা হয়
- [ ] Mobile responsiveness সকলো পৃষ্ঠাৰ ওপৰত কাৰ্যকৰী হয়
- [ ] Navigation লজিক আৰু সম্পূৰ্ণ হয়
- [ ] Contact forms কাৰ্য্যক্ষম হয়
- [ ] কোনো ভাঙি যোৱা link বা হেৰুওৱা asset নাই
- [ ] Page load speed গ্রহণযোগ্য হ'ব লাগে

## FitSite Network এ এতিয়ালৈকে কি আছে {#the-fitsite-network-so-far}

```
FitSite Network
├── WordPress Multisite (subdomain mode)
├── Ultimate Multisite (configured)
├── Hosting with wildcard SSL
├── Domain mapping configured
├── Site Templates
│   ├── Studio Essential (small studios, trainers)
│   ├── Gym Pro (established gyms)
│   └── Fitness Chain (multi-location)
└── Ready for plan configuration (next lesson)
```

## এই পাঠত আমি কি নিৰ্মাণ কৰিলোঁ {#what-we-built-this-lesson}

- বিভিন্ন ফিটনেস ব্যৱসায়ৰ আকাৰৰ বাবে ডিজাইন কৰা **তিনিটা niche-specific Templates**
- **ফিটনেছ-উপযুক্ত বিষয়বস্তু আৰু ছবি** যিয়ে প্ল্যাটফৰ্মটোক বিশেষভাৱে নিৰ্মাণ কৰা বুলি অনুভৱ কৰায়
- গ্ৰাহকসকলক কাস্টমাইজেশ্যনৰ মাজেৰে পথ প্ৰদৰ্শন কৰা **শিক্ষণীয় Placeholder content**
- Templates সমূহ production-ready কৰিবলৈ এটা **quality checklist**
