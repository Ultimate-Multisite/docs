# Action: wu_site_created {#action-wusitecreated}

কোনো স্থান (site) প্ৰথমবাৰৰ বাবে সৃষ্টি হোৱাৰ পিছত এই অ্যাকশ্বনটো টোপনি হয়।

যদি কোনো টেমপ্লেটৰ পৰা ইয়াক নকল কৰা হয়, তেন্তে ই টোপনি নহয়।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $data | `array` | যি বস্তুৰ তথ্য সঞ্চয় কৰা হ'ব। |
| $site | `\Site` | বস্তুৰ প্ৰতিলিপি (object instance)। |
