# Action: wu_page_after_render

প্লাগইন ডেভেলপাৰসকলক আমি পৃষ্ঠাখন প্ৰিন্ট কৰাৰ পিছত অধিক বিষয়বস্তু যোগ কৰিবলৈ অনুমতি দিয়ে।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $page_id | `string` | এই পৃষ্ঠাখনৰ আইডি (ID)। |
| $page | `object` | পৃষ্ঠাৰ বস্তু (page object)। |

- 1.8.2
