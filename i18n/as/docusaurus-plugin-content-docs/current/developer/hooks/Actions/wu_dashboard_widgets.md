# Action: wu_dashboard_widgets {#action-wudashboardwidgets}

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $tab | `string` | বৰ্তমানৰ টাবটো। |
| $screen | `\WP_Screen` | স্ক্ৰীণ অবজেক্টটো। |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Ultimate Multisite এডমিন পেজৰ এটা প্ৰতিলিপি। |
