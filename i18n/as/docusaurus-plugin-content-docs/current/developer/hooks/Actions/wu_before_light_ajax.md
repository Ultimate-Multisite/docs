# Action: wu_before_light_ajax

কেতিয়াবা, অ্যাকশনসমূহ সঠিকভাৱে সামলিবলৈ আমাক কিছুমান অতিৰিক্ত সহায়ৰ প্ৰয়োজন হ'ব পাৰে।

এই অ্যাকশনটোৰ ব্যৱহাৰ কৰি প্ৰয়োজন অনুসৰি অতিৰিক্ত ডিপেণ্ডেন্সি (dependencies) লোড কৰিব পাৰি।
