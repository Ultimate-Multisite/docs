# Action: wu_maybe_create_customer {#action-wumaybecreatecustomer}

কাস্টমাৰ যোগ কৰা হয় যেতিয়া প্লাগইন ডেভেলপাৰসকলক অধিক কাম কৰিবলৈ অনুমতি দিয়ে।

উদাহৰণস্বৰূপে, কাস্টমাৰক ইউজাৰলৈ মূল সাইটতো যোগ কৰিবলৈ আমি ইয়াত হুক যোগ কৰোঁ।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $customer | `\Customer` | যি কাস্টমাৰ হয়তো সৃষ্টি কৰা হৈছে। |
| $checkout | `\Checkout` | বৰ্তমানৰ চেকআউট ক্লাছ। |
