# Action: wu_dashboard_\{$this->tab\}_widgets {#action-wudashboardthis-tabwidgets}

প্লাগইন ডেভেলপাৰসকলক নেটৱৰ্ক ড্যাশবৰ্ড প্যানেলত উইজেট যোগ কৰিবলৈ অনুমতি দিয়ে।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $tab | `string` | বৰ্তমানৰ টেবে। |
| $screen | `\WP_Screen` | স্ক্ৰীণ অবজেক্ট। |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Ultimate Multisite এডমিন পেজৰ প্ৰতিলিপি (instance)। |
