# Filter: wu_independent_billing_cycle_product_types

যি প্রোডাক্ট টাইপৰ স্বাধীন বিলিং চক্ৰ (independent billing cycles) আছে, সেইবোৰ ফিল্টাৰ কৰে।

যি প্রোডাক্টবোৰৰ স্বাধীন বিলিং চক্ৰ থাকে, সেইবোৰ ক'ৰ্টত (cart) থকা پلانৰ বিলিং সময়ৰ সৈতে মিলিবলগীয়া নহয়। এইটো ডোমেইন ৰেজিষ্ট্ৰেচনৰ দৰে প্রোডাক্টৰ বাবে উপকাৰী, যিবোৰ پلانটো মাহিক নে বৰ্ষিক হ'ব তাৰ পৰিৱৰ্তে সদায় বৰ্ষিক বিলিং হয়।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | স্বাধীন বিলিং চক্ৰ থকা প্রোডাক্ট টাইপৰ এটা তালিকা (Array)। |
