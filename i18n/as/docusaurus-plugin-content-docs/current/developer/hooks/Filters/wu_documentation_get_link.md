# Filter: wu_documentation_get_link {#filter-wudocumentationgetlink}

প্লাগইন ডেভেলপাৰসকলক লিংকসমূহ ফিল্টাৰ কৰিবলৈ অনুমতি দিয়ে।

ই কিমান উপযোগী হ'ব, মই নিশ্চিত নহওঁ, কিন্তু ইয়াক থকাটো কোনো ক্ষতি নহ'ব।

## পৰামিটারসমূহ {#parameters}

| Name | Type | বৰ্ণনা |
|------|------|-------------|
| $link | `string` | ৰেজিষ্ট্ৰেড লিংক |
| $slug | `string` | লিংকটো পুনৰুদ্ধাৰ কৰিবলৈ ব্যৱহৃত 'slug' |
| $default_link | `string` | ৰেজিষ্ট্ৰেড ডিফল্ট লিংক |
