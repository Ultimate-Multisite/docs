# Filter: wp_ultimo_render_vars

প্লাগইন ডেভেলপাৰসকলক গ্লোবেলভাৱে ৰেণ্ডাৰ কনটেক্সটলৈ (render context) অতিৰিক্ত ভৰিয়েবল (variable) যোগ কৰিবলৈ অনুমতি দিয়ে।

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $args | `array` | ৰেণ্ডাৰ কলৰ দ্বাৰা প্ৰেৰণ কৰা ভৰিয়েবল ধৰি ৰখা আৰ্ৰে। |
| $view | `string` | যি ভিউটো ৰেণ্ডাৰ কৰিব লাগিব তাৰ নাম। |
| $default_view | `string` | ফলব্যাক ভিউটোৰ নাম। |

### Since

- 2.0.0
### Source

[`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L31) ত ৩১ নং শাৰীত সংজ্ঞায়িত কৰা হৈছে।
