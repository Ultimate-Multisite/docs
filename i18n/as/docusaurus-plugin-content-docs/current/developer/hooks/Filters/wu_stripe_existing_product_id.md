# Filter: wu_stripe_existing_product_id {#filter-wustripeexistingproductid}

|------|------|-------------|
| $product_id | `string` | পৰীক্ষা কৰিবলগীয়া Stripe প্ৰডাক্টৰ ID। |
| $name | `string` | প্ৰডাক্টৰ নাম। |
