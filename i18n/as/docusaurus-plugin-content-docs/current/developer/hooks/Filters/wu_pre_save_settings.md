## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $settings | `array` | সেভ কৰিবলগীয়া ছেটিংছ। |
| $settings_to_save | `array` | যোগ কৰিবলগীয়া নতুন ছেটিংছ। |
| $saved_settings | `array` | সেভ কৰা বৰ্তমানৰ ছেটিংছ। |
