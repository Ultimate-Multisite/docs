# Filter: wu_setup_wizard {#filter-wusetupwizard}

ডেভেলপমেন্টৰ বাবে এটা সেটআপ উইজৰ্ডত (setup wizard) আৰু কিছুমান পদক্ষেপ (steps) যোগ কৰিবলৈ অনুমতি দিয়ে।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $sections | `array` | বৰ্তমান থকা অংশসমূহ (Current sections)। |
| $is_migration | `bool` | যদি এইটো এটা মাইগ্ৰেচন (migration) হ'ব নে নহয়। |
| $wizard | `object` | বৰ্তমানৰ উইজৰ্ডৰ প্ৰতিলিপিটো (The current instance)। |
