# Filter: wu_plan_product_types

যি সামগ্ৰীৰ প্ৰকাৰক প্লেন (plan) হিচাপে গণ্য কৰিব লাগে, সেইবোৰ ফিল্টৰ কৰে।

এই ফিল্টৰটোৱে এডঅনক (addons) এনে কিছুমান সামগ্ৰীৰ প্ৰকাৰ যোগ কৰিবলৈ অনুমতি দিয়ে, যাক বৈধতা (validation) আৰু পৃথকীকৰণ (segregation)ৰ সময়ত প্লেন হিচাপে চিনাক্ত কৰা হ'ব।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | প্লেন হিচাপে গণ্য কৰিবলগীয়া সামগ্ৰীৰ প্ৰকাৰৰ আৰ্ৰে। |
