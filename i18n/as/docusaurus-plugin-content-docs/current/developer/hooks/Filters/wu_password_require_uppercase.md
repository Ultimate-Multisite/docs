# Filter: wu_password_require_uppercase

পাসৱৰ্ডত ডাঙৰ আখৰ (uppercase letters) প্ৰয়োজন নে নাই, সেয়া ফিল্টৰ কৰে।

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | পাসৱৰ্ডত ডাঙৰ আখৰৰ প্ৰয়োজন নে নাই। যদি নিয়ম প্ৰয়োগ কৰা হয়, তেন্তে ডিফল্টভাৱে 'true' থাকে। |
| $defender_active | `bool` | Defender Pro Strong Password সক্ৰিয় হৈ আছে নে নাই। |
