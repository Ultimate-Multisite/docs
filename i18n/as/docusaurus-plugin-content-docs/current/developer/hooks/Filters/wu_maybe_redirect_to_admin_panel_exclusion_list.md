# Filter: wu_maybe_redirect_to_admin_panel_exclusion_list {#filter-wumayberedirecttoadminpanelexclusionlist}

স্বয়ংক্রিয় পুনৰ-নিৰ্দেশ (auto-redirect) নহ'বলৈ কিছুমান parameters-ৰ বৰ্জন তালিকা (exclusion list) সৃষ্টি কৰে।

এইটো প্ৰয়োজনীয় কাৰণ, নহ'লে পেজ বিল্ডাৰটোৱে লগইন পেজটো সংৰ্যাস (edit) কৰিব নোৱাৰিব যেতিয়া ইয়াক সংজ্ঞায়িত কৰা হয়।

### Since {#since}

- 2.0.4

[`inc/checkout/class-checkout-pages.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout-pages.php#L357) ফাইলৰ ৩5৭ নং শাৰীত সংজ্ঞায়িত কৰা হৈছে।
