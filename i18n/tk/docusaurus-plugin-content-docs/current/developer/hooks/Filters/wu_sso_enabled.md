# Filter: wu_sso_enabled

Domenler arasynda ýeke giriş mümkinçiliklerini işjeňleşdiriň/öçüriň.

Ýeke girişini doly öçürmek ýa-da onuň ýerine şertli işjeňleşdirmek üçin bu bahany süzgüçden geçiriň.

## Parametrler {#parameters}

| Ady | Görnüşi | Beýany |
|------|------|-------------|
| $enabled | `bool` | SSO işjeň bolmalymy? Açmak üçin true, öçürmek üçin false-a meňzeş baha. |

- 2.0.11
