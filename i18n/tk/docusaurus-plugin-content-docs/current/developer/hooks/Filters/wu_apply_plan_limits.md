# Süzgüç: wu_apply_plan_limits

Goşmaça döredijilere çäklendirmeleri aýlanyp geçmäge rugsat beriň.

Bu süzgüji çäklendirmeleriň islendigi başlatylmazdan öň erkin kody işletmek üçin ulanyp bilersiňiz. Eger süzgüjiňiz hakykat hökmünde bahalandyrylýan islendik bahany gaýtarsa, proses dowam eder; eger ýalan hökmünde bahalandyrylýan islendik bahany gaýtarsa, kod yzyna gaýdar we aşakdaky hook-laryň hiç biri işledilmez.
