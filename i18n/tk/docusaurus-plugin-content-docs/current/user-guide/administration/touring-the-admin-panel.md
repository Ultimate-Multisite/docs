# Admin Paneli Üzümleme {#touring-the-admin-panel}

Ultimate Multisite mümkün olduğunca basit we özünü açıkladı edar, bu güce sahip bir araçla size rehberlik etmek mümkündür. Sizden rahat etmeniz için admin sayfalarını birlikte inceleyelim.

## Dashboard (Panel) {#dashboard}

Ultimate Multisite **Dashboard** girileri, gelirler, site faaliyeti, üye artışı, ziyaretçi sayıları we coğrafi veriler bilen temel raporlar we analizleri gösterir.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

İşte dashboard'un tam görünümü:

![Ultimate Multisite Dashboard full page](/img/admin/um-dashboard-full.png)

We dashboard'un alt bölümü:

![Ultimate Multisite Dashboard bottom section](/img/admin/um-dashboard-bottom.png)

Ayrıca ağ dashboard'unun tam görünüm şudur:

![Network Dashboard full page](/img/admin/network-dashboard-full-page.png)

## Checkout Formları (Ödeme Formları) {#checkout-forms}

**Checkout Forms**, size özel kayıt sayfaları oluşturma esnekliği verir. Alanları özelleştirebilir, özel sınıflar we scriptler ekleyebilir we ziyaretçi ülkesi veya konumuna göre erişimi kısıtlayabilirsiniz.

![Checkout Forms list](/img/admin/checkout-forms-list.png)

## Ürünler (Products) {#products}

**Products** bölümü, ağınız için farklı planlar, paketler we hizmetler oluşturduğunuz yerdir. Fiyatları, faturalandırma sıklıklarını we ürün seviyesinde limitleri ve kotaları belirleyin.

![Products list](/img/admin/products-list.png)

## Üyelikler (Memberships) {#memberships}

**Memberships** sayfası ağınızdaki tüm abonelikleri gösterir. Planlar, ürünler, fatura miktarları we sıklıkları, ödeme geçmişi we zaman damgaları dahil olmak üzere abonelik ayrıntılarını gözden geçirin ve düzenleyin.

![Memberships list](/img/admin/memberships-list.png)

## Ödemeler (Payments) {#payments}

**Payments** səhifesi bütün şəbəkəniz üzrə ödəniş tarixi haqqında sürətli baxış təmin edir və xüsusi məhsullar və məbləğlər daxil olmaqla ətraflı əməliyyat məlumatları ilə gəlir.

![Payments list](/img/admin/payments-list.png)

## Müştərilər (Customers) {#customers}

**Müştərilər** səhifəsi şəbəkənizdə qeydiyyatdan keçmiş bütün üzvləri ad, e-poçt ünvanı, son daxil olma tarixi və onların alt saytına daxil olmaq üçün "keç" (switch to) linki daxil olmaq daxil olmaqla əsas istifadəçi məlumatları ilə göstərir.

![Customers list](/img/admin/customers-list.png)

## Saytlar (Sites) {#sites}

**Saytlar** səhifəsi şəbəkənizdəki bütün alt saytları, sayt şablonlarını və müştəri tərəfindən sahib olan saytları siyahıya salır. Xəritə edilmiş domenləri, limitləri və quota-ları, pluginləri və temaları, sayt miniatürlərini asanlıqla idarə edin.

![Sites list](/img/admin/sites-list.png)

## Domenlər (Domains) {#domains}

**Domenlər** səhifəsi alt saytlara bağlı xüsusi domenləri sadalayan yerdir. Super admin olaraq, siz xüsusi domen adlarını əllə əlavə edə və ya xəritə qoya və DNS qeydlərini və logları görə bilərsiniz.

![Domains list](/img/admin/domains-list.png)

## Azins Kodları (Discount Codes) {#discount-codes}

**Azins Kodları** son istifadəçilərə endirim təklif etmək üçün kupon kodları yaratmağa imkan verir. Endirim dəyərini müəyyən edin və xüsusi istifadəçiləri və ya qrupları hədəfləyin.

![Discount Codes list](/img/admin/discount-codes-list.png)

## Yayımlar (Broadcasts) {#broadcasts}

**Yayımlar** alt sayt istifadəçilərinə elanlar göndərmək və ya onlara şəxsi mesajlar göndərmək üçün bir alətdir. Plan və ya müştəri olan məhsul əsasında müəyyən istifadəçi qruplarını hədəfləyin.

![Broadcasts list](/img/admin/broadcasts-list.png)

## Ayarlar (Settings) {#settings}

**Ayarlar** səhifəsi Ultimate Multisite-ı konfiqurasiya etdiyiniz yerdir — qeydiyyat ayarları, ödənişlər, API və webhooks, domen xəritələnməsi və digər inteqrasiyalar.

AI connector sozlamalarynda şu vaqt qo'llab-quvvatlanayotgan OAuth provayder havzolari ro'yxati: Anthropic Max, OpenAI ChatGPT/Codex va Google AI Pro. Har bir provayder kartasi super administratorlarga hisob-kitoblarni ulashish, saqlangan hisob-kitoblarni yangilash, elektron pochta orqali hisob-kitoblarni olib tashlash va brauzer yo'naltirishini bloklaydigan sandboxed muhitda qo'llaniladigan qo'lda OAuth o'zlashtirish (manual OAuth fallback) funksiyasidan foydalanish imkonini beradi. Cursor Pro sozlamalari admin panelidan olib tashlangan.

ChatGPT/Codex hisob-kitoblari connector-yordamida vositalardan foydalanishni qo'llab-quvvatlaydi, bu yerda operatsiya vositalarni ruxsat etsa, shuning uchun OpenAI hisob-kitobi ulanganidan so'ng connector yordamida qo'llaniladigan operatsiyalariga bog'liq admin ish oqimlar Codex vosita xarakteridan foydalanish imkonini beradi.

## Hodisalari (Events) {#events}

**Hodisalari (Events)** sahifasi tarmohingizdagi barcha hodisalar va yozuvlarni saqlab qoladi. U reja o'zgarishlari, ro'yxatdan olinishlar va boshqa harakatlar kabi faoliyatni kuzatib boradi — bu sizning multisite-ingizni nazorat qilish uchun foydali.

![Events list](/img/admin/events-list.png)

## Webhooklar {#webhooks}

**Webhooklar (Webhooks)** ma'lumotlarni boshqa ilovaga yuborish imkonini beradi. Ultimate Multisite dan Zapier kabi platformalarga ma'lumotlarni yuborish uchun foydali.

![Webhooks list](/img/admin/webhooks-list.png)
