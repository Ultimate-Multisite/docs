# Üylyk (Memberships) Başdaresini Dolandyradyk {#managing-memberships}

Ultimate Multisite-de siz şebekäňizdäki ähli abonementleri we üyelikleri dolandyradyk bilersiňiz. Abonamentleri, planlary we önüleri, sazlaryny, tozlary we üpjünçilik maslahatlaryny, ötek töläýiňizi we wagt işaretlerini gözden geçiripli we redaktir edip bilersiňiz.

Bu sahka bilen işlemegi üçin Ultimate Multisite menüsindäki **Memberships** (Üylykler) sekýiisine gitmeli.

![Memberships list page](/img/admin/memberships-list.png)

Memberships sahhasynda siz müşderilerini we olaryň hasaplary bilen baglanyşykly önüleri görä bir sanaw alarsyňyz. Üyelikleri statusy bilen filtrlenebilir: aktif, bekleýän, tozlarynda, sonu gelip biljek ýa-da ýapyk (cancelled).

Beb membershipler **ömür boyu** membershipler hasaplanýar. Olaryň son gizipe gelme sany ýok we awtomatiki ýagdaýdan durnyp bilmezler. Eger bep plan bir synapdan ýa-da belli bir wagtyndan soň dowam etmeli bolsa, synap bilen gelen toýdurylan ötdürişi bolan tölenji maglumat bilen, membershipi manual ýapyşdyryp ýa-da girizimi üýtgetmek gerek bolan wagtda müşderini başga planlara geçirmek üçin ulanyň.

Membershipi açyp, onuň detallerini — hasaplaýjy maglumatlary, sahypa ýa-da zynjyryş limitlerini we beýleki zatlary — redaktir etmegi bilerli sahny özüňizden tapyň.

![Membership edit interface](/img/admin/membership-edit.png)

Membershipi redaktir etmegiň doly görnüşi aşakdaky ýerde:

![Membership edit full interface](/img/admin/membership-edit-full.png)

![Membership modalyny goşmak](/img/admin/membership-add-modal.png)
