# Dashboard Görnüşi {#dashboard-overview}

Ultimate Multisite menüsünün altında xüsusi bir **Dashboard** barada var, aynı zamanda Network Dashboard'a da widgetlar əlavə edir. Bu məqalədə hər iki yerdə nəyi görəcəyinizi izah edirik.

## Network Dashboard {#the-network-dashboard}

Ultimate Multisite, **My Sites → Network Admin → Dashboard** yolunda tapılan network səviyyəli dashboard-a faydalı widgetlar əlavə edir.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

Hər bir widget nəyi təqdim etdiyini aşağıdakılar:

**İlk Addımlar (First Steps):** Yeni quraşdırmada görünür və sizə ödəniş üsulunu konfiqurasiya etmək və test hesabı yaratmaq kimi qurulma tapşırıqlarını tamamlama xatırladır.

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Xülasə (Summary):** Gün ərzində qeydiyyat sayını və gəliri göstərir. MRR (Aylıq Təkrar Gəlir), gəlir yaradan üzvlər üzrə proqnozlaşdırılan ümumi məbləği göstərir.

![Summary widget](/img/admin/network-dashboard-summary.png)

**Fəaliyyət Axını (Activity Stream):** Ultimate Multisite üzrə baş verən hadisələri – qeydiyyatlar, ləğv edilmələr, saytların yaradılması və silinməsi, digər fəaliyyətləri bildirir.

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**İndi (Right Now):** Networkdəki istifadəçilər və saytlar haqqında qısa bir xülasə göstərir və yeni saytlar və ya istifadəçilər yaratmaq üçün qısayollar təqdim edir.

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**Xəbərlər və Müzakirələr (News & Discussions):** Ultimate Multisite ilə bağlı ən son xəbərləri, yenilikləri, kritik səhvləri və təhlükəsizlik düzəlişlərini çəkir və göstərir.

![News widget](/img/admin/network-dashboard-news.png)

## Ultimate Multisite Dashboard {#the-ultimate-multisite-dashboard}

Network dashboard bütün network haqqında ümumi məlumatları təqdim edərkən, Ultimate Multisite dashboard (Ultimate Multisite menüsünün ən yuxarı səviyyəsində) xidmətlə bağlı məlumatları göstərir.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Aylık Tekrarlanan Gelir Büyümesi (Monthly Recurring Revenue Growth):** Bu, 12 aylık takvim dönemi boyunca gelirin aylık dökümünü gösterir. Yeni gelir mavi renkte gösterilirken iptaller kırmızı olarak görünür.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**Tarih Aralığı Desteği:** İhtiyacınız olan bilgiye odaklanmak için dashboard verilerini zaman dilimine göre filtreleyin. Tarih aralığı, aşağıdaki widget'larda neyin gösterileceğini kontrol eder.

![Date range and dashboard stats](/img/admin/um-dashboard-date-range.png)

Dashboard widget'ları şunları içerir:

- **Gelir (Revenue)** — Dönem için toplam gelir, MRR ve ürün bazında dökümlerle birlikte
- **Yeni Üyelikler (New Memberships)** — Seçilen dönemde kaç yeni üyelik oluşturulduğu
- **Ülkelere Göre Kayıtlar (Signups by Countries)** — Kayıtların coğrafi dökümü
- **Formlara Göre Kayıtlar (Signups by Forms)** — Her bir ödeme formu üzerinden kaç müşterinin kayıt olduğu ve yöneticiler tarafından eklenenler arasındaki fark
- **En Çok Ziyaret Edilen Siteler (Most Visited Sites)** — Ağınızdaki en çok ziyaret edilen siteler (Ziyaret sayımını **Ultimate Multisite → Ayarlar → Siteler → Ziyaret Sınırını ve Sayımı Etkinleştir** kısmından etkinleştirmeniz gerekir)

![Dashboard statistics](/img/admin/um-dashboard-stats.png)
