# Sunrise Dosyasını Quraşma Xataly

sunrise.php dosyası bu, WordPress özünü başlatýan wagtarda gözleýän aýratyn bir fayl. WordPress-i sunrise.php dosyasyny tapyp bilmek üçin, ol dosya **wp-content** klitpiniň içinde bolmaly.

Ultimate Multisite-y aktivleşdirip ekranlardaky ýaly sozlamany geçip geçende, Ultimate Multisite biziň sunrise.php faylyny wp-content klitpiniň içine köçürmegi sabyşar.

<!-- Ekran resmini tapyp bilmezden: Sunrise.php quraşma aşamyny görkezýän sozlama wizardini görkezýär -->

Çoğunlyk wagtda, biz fayly äýtdyrmagy hemme zat işlediğinde baglanyşykly ýa-da her zat düzgün işleýär. Ýöne, eger bir zat düzgün sozlamalansa (meselem, klitpiniň girizilmegi), Ultimate Multisite fayly köçürip bilmezlik bilen bir wariantyň ýüzüne çykyp biler.

Ultimo size berýän xata mesageni okap gelse, bu ýerde näme bolanyny görkezýär: **Sunrise copy failed** (Sunrise nusgalap geçmekde tapawut boldy).

<!-- Ekran resmini tapyp bilmezden: Sunrise nusgalap geçmekde tapawut boldy diýen xata mesageni görkezýär -->

Bu soragy çözmek üçin, siz wp-ultimo plugin klitpiniň içinde ýerleşýän sunrise.php faylyny almagy we onuň wp-content klitpiniň içine ýazmagy bilen çözip bilersiňiz. Bu işi geçirdikten soň, wizard sahyny ýa-da ýen ýükleň we barlaglar geçmeli.

<!-- Ekran resmini tapyp bilmezden: wp-ultimo plugin klitpiniň içinde sunrise.php faylyny görkezýän fayl menejeri --> Her halda, gelejekde sorunlar bolmamasyny üpjün etmek üçin (Ultimate Multisite bilen däl-de, beýleki pluginler we theme-ler bilen hem), klitpiniň girizilmegi barada umumy barlagdan geçmek zerurdyr.

WordPress-iň bir bölegi bolan **Health Check tool** (olaryň esasy sahypasynda **admin panel > Tools > Health Check** arkaly giriş bilersiňiz) klitpiniň girizilmegi WordPress bilen problem döredip bilerlikde bolmadyk klitpileri görkezýär.
