## Gurnama {#installation}

## Dili URL arkaly bellemek {#setting-the-language-via-the-url}

Şeýle hem locale parametrini URL-e geçirip, sahypanyň dilini üýtgedip bilersiňiz. Mysal üçin, italýan dilindäki registrasiýa sahypasyna ugrukdyrmak isleseňiz, URL-e ?locale=it_IT goşuň.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Munuň işlemegi üçin dil WordPress toruňyzda gurnalan bolmaly.
