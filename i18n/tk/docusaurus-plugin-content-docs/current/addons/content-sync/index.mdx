# Mazmun sinhronlaşdyrmasy

Multisite Ultimate: Mazmun sinhronlaşdyrmasy

Multisite Ultimate üçin bu goşundy tor administratorlaryna islendik saýtdan tordaky beýleki saýtlara mazmuny sinhronlaşdyrmaga mümkinçilik berýär. Ol ýazgylar, sahypalar, ýörite post görnüşleri we Elementor şablonlary ýaly ähli post görnüşlerini goldaýar. Goşundy mazmuny we nyşan saýtlary saýlamak üçin köp basgançakly ussat interfeýsini, toparlaýyn işläp taýýarlamagy we hakyky wagtda ösüş boýunça seslenmäni öz içine alýar.

## Aýratynlyklar

* Mazmun saýlamak: Kiçi suratlar we post görnüşi boýunça süzgüç bilen wizual kartoçka tory
* Ähli post görnüşleri: Ýazgylary, sahypalary, önümleri, Elementor şablonlaryny we islendik ýörite post görnüşini sinhronlaşdyrýar
* Saýt saýlamak: Çeşme saýty çykarmak we çaga saýtlary öňünden saýlamak bilen mazmunyň haýsy saýtlara sinhronlaşdyryljakdygyny saýlaň
* Toparlaýyn işläp taýýarlamak: Ösüş täzelenmeleri bilen bir gezekde 10 saýta çenli sinhronlaşdyrýar
* Media sinhronlaşdyrmasy: Post mazmunyndaky media faýllary (suratlar, wideolar) awtomatiki göçürýär we nyşan saýtlara täzeden baglaýar
* Taksonomiýa sinhronlaşdyrmasy: Kategoriýalary, bellikleri we ýörite taksonomiýalary saklaýar
* Elementor goldawy: CSS-i täzeden döretmek we şablon ID-lerini täzeden baglamak bilen Elementor şablonlary üçin doly goldaw
* Çaknyşygy çözmek: Bar bolan mazmuny çalyşmak ýa-da ony geçmek mümkinçiligi
* Keş dolandyryşy: Sinhronlaşdyrmadan soň nyşan saýtlarda Elementor keşini awtomatiki arassalaýar

## Talaplar

* WordPress 5.3 ýa-da has ýokary
* PHP 7.4 ýa-da has ýokary
* Multisite Ultimate plugin-i (işjeň)
* Elementor plugin-i (islege bagly, Elementor şablon sinhronlaşdyrmasy üçin zerur)

## Gurnama

1. 'ultimate-multisite-content-sync'-i '/wp-content/plugins/' katalogyna ýükläň
2. WordPress Network Admin-iňizdäki 'Pluginler' menýusy arkaly plugin-i işjeňleşdiriň
3. Ultimate Multisite → Sazlamalar → Mazmun sinhronlaşdyrmasy bölüminde sazlamalary düzüň
4. Sinhronlaşdyrmagy başlamak üçin Ultimate Multisite → Mazmun sinhronlaşdyrmasy bölümine geçiň
