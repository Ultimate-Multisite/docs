# VAT

Ultimate Multisite: Ýewropa VAT

Ýewropa VAT-y has gowy goldamak üçin Ultimate Multisite-niň adaty salgyt dolandyryşyna goşmaça aýratynlyklary we gurallary goşýar.
ÝB VAT stawkalaryny Salgyt stawkalary sahypasyna awtomatiki goşýar

## Gurnama

1. 'ultimate-multisite-vat'-y '/wp-content/plugins/' katalogyna ýükläň
2. WordPress-däki 'Pluginler' menýusy arkaly plugini işjeňleşdiriň
3. Ultimate Multisite > Sazlamalar > Salgytlar bölümine gidiň we ÝB VAT-y işjeňleşdiriň.
4. Salgyt stawkalaryna basyň we aşakdaky ÝB VAT stawkalaryny täzeläň hem-de Salgyt stawkalaryny ýazdyryň.
5. Tamam!
