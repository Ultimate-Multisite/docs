# 第四課：建立利基模板

模板是您利基價值主張的核心。如果一位健身工作室老闆註冊後，看到一個看起來就像健身網站的網站——裡面已經有課程時間表、教練資料和適當的圖片——他比看到一片空白畫布時，更有可能留下來。

## 我們上次停在哪裡

我們已經建立了一個安裝並配置了 Ultimate Multisite 的可運作的 FitSite 網路。現在，我們要建立那些讓 FitSite 感覺專為健身業務量身打造的模板。

## 為什麼利基模板很重要

通用模板會迫使您的客戶做艱難的工作：找出他們需要哪些頁面、什麼內容結構是可行的，以及如何讓它看起來符合他們的行業。利基模板消除了這種摩擦。

註冊 FitSite 的健身工作室老闆應該看到：

- 一個看起來像健身工作室網站的首頁
- 已經建立的課程、時間表、教練和定價頁面
- 與其行業相符的圖片和佔位內容
- 一個感覺專業且符合健身品牌風格的設計

他們只需要填入自己的細節。他們不需要從零開始。

## 規劃您的模板

在開始建立之前，請決定要提供哪些模板。對於 FitSite，我們將創建三個：

### 模板一：工作室基礎版 (Studio Essential)

適用於小型工作室和個人教練。

- 包含英雄圖片、課程亮點和行動呼籲的**首頁**
- 包含工作室故事和使命的**關於**頁面
- 包含時間表佈局的**課程**頁面
- 包含個人資料卡片的**教練**頁面
- 包含地點地圖和表單的**聯絡**頁面
- **乾淨、現代的設計**，搭配適合健身的配色方案

### 模板二：健身館專業版 (Gym Pro)

適用於擁有更多服務的成熟健身館。

- 包含工作室基礎版的所有內容，外加：
- 包含定價表的**會員**頁面
- 用於設施照片的**相簿**頁面
- 用於健身技巧和新聞的**部落格**區塊
- 首頁上的**客戶評價**區塊
- **更突出的品牌**選項

### 模板三：健身連鎖版 (Fitness Chain)

適用於多地點營運的企業。

- 包含健身館專業版的所有內容，外加：
- 包含多個設施列表的**地點**頁面
- **加盟/地點**子頁面模板
- 包含地點特定細節的**集中化品牌**管理
- 跨地點的**員工名錄**

## 建立模板網站

在 Ultimate Multisite 中，模板就是一個按照您希望新客戶網站呈現的方式配置的 WordPress 網站。以下是如何創建一個：

### 步驟 1：創建模板網站

1. 在您的網路管理員介面中，導航至 **Sites > Add New**
2. 創建一個名為 `template-studio-essential` 的網站
3. 這個網站將成為您的工作畫布

### 步驟 2：安裝和配置主題

切換到模板網站儀表板，然後：

1. 啟用一個適合健身業務的主題
2. 配置主題設定、顏色和字體
3. 設定包含適合健身風格導航的頁首和頁尾

:::tip 主題選擇
選擇一個足夠靈活、適合健身但又不會過於複雜，讓您的客戶能夠管理的模板。像 Astra、GeneratePress 或 Kadence 這樣的主題表現良好，因為它們輕量、可自定義且支援完善。
:::

### 步驟 3：創建頁面

使用以下內容建立每個頁面：

- **佔位內容**：內容必須自然地與健身相關（例如：「歡迎來到 [工作室名稱]」，而不是「Lorem ipsum」）
- **佔位圖片**：使用免費圖庫網站的圖片，展示健身活動
- **功能佈局**：使用頁面建立器或區塊編輯器來建立

盡可能地讓佔位內容具有指導性。不要使用通用填充文字，而是寫入像：「請將此處替換為您工作室的簡短描述，以及讓它與眾不同的地方。請提及您的訓練理念、年資或客戶可以期待什麼。」這樣的內容。

### 步驟 4：配置外掛程式 (Plugins)

安裝並啟用健身工作室需要的外掛程式：

- 預約或排程外掛程式（如果適用於您的方案層級）
- 聯絡表單外掛程式
- SEO 外掛程式（預先配置了與健身相關的預設值）

### 步驟 5：標記為模板

1. 導航至 **Ultimate Multisite > Sites**
2. 編輯該模板網站
3. 啟用 **Site Template** 開關

對您想要提供的每個模板重複此過程。

## 模板品質檢查清單

在將模板公開之前，請驗證以下項目：

- [ ] 所有頁面都能正確加載且看起來專業
- [ ] 佔位內容有幫助且具備利基特定性
- [ ] 圖片適當且擁有適當的授權
- [ ] 所有頁面的行動裝置響應式設計都正常運作
- [ ] 導航邏輯清晰且完整
- [ ] 聯絡表單可以正常運作
- [ ] 沒有斷開的連結或遺失的資源
- [ ] 頁面加載速度可以接受

## FitSite 網路現狀

```
FitSite Network
├── WordPress Multisite (subdomain mode)
├── Ultimate Multisite (configured)
├── Hosting with wildcard SSL
├── Domain mapping configured
├── Site Templates
│   ├── Studio Essential (small studios, trainers)
│   ├── Gym Pro (established gyms)
│   └── Fitness Chain (multi-location)
└── Ready for plan configuration (next lesson)
```

## 本課所學

- **三個利基特定的模板**，針對不同規模的健身業務
- **適合健身的內容和圖片**，讓平台感覺專為此目的打造
- **指導客戶進行自定義的教學佔位內容**
- **一份品質檢查清單**，確保模板已準備好投入生產
