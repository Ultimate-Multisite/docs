# Fluent Forms

這個附加元件讓您可以在 Ultimate Multisite 的產品、會員資格或網站上為 Fluent Forms 設定使用限制。由於 Fluent Forms 使用的是自訂資料表（custom tables），而不是自訂文章類型（custom post types），因此一般的限制選項無法與 Fluent Forms 搭配使用。

## 功能特色 {#features}

- 為 Fluent Forms 或 Conversational Fluent Forms 設定使用限制
- 當客戶達到限制時，會收到友善的訊息，提醒他們升級方案
