# Material WP

WordPress 的管理介面多年來一直沒有變化。但您是否曾想過，如果它採用 Material Design 的風格會是什麼樣子？

Material WP 是一款開發的管理主題，旨在為您提供這種外觀和使用感受，幫助您為您的客戶提供更具客製化的 Dashboard。

## Features {#features}

- 自訂 CSS 支援
- 簡單的樣式設定
- 選單重新排序
- 輕鬆匯入/匯出

## Documentation {#documentation}

**使用 Material WP 自訂您的管理面板**

## Installation {#installation}

1. 將 'material-wp' 上傳到 '/wp-content/plugins/' 目錄
2. 透過 WordPress 的「Plugins」選單啟用此外掛

## Usage Guide {#usage-guide}

WordPress 的管理介面多年來一直沒有變化。但您是否曾想過，如果它採用 Material Design 的風格會是什麼樣子？

Material WP 是一款開發的管理主題，旨在為您提供這種外觀和使用感受，幫助您為您的客戶提供更具客製化的 Dashboard。Material WP 也是 Ultimate Multisite 高級外掛核心的一部分。

在**安裝它**之後，請前往您的 _Plugins_ 頁面，並在您的 Network 上點擊啟用。
