# 更新日誌

## 1.18.0 — 於 2026-06-29 發佈 {#1180--released-on-2026-06-29}

### 新增 {#new}

- **Google Calendar 工具** — 讀取已設定的日曆與事件，以支援具排程感知的自動化
- **聯絡人對應與參與者輔助工具** — 將日曆參與者與網站使用者及聯絡人配對
- **人工核准關卡與提醒記錄** — 暫停自動化以供審核，並避免重複通知
- **TextBee SMS 供應商** — 透過 TextBee 傳送已設定的簡訊通知
- **進階配套套件** — 新增可信開發者檔案系統、資料庫、WP-CLI、REST 分派器、plugin 建構器、git 快照、使用者管理與基準測試工具，並與 WordPress.org 版本分開發佈

### 改進 {#improved}

- **受管理的 Superdav 服務設定** — 為支援的網站新增託管服務 endpoint 與自動連線佈建
- **發佈打包** — 建置獨立的核心與進階 ZIP，兩者皆發佈至 GitHub，且僅將核心套件傳送至 WordPress.org

### 修正 {#fixed}

- **AI 請求可靠性** — 改善模型選擇、請求逾時、預設設定模型、推理文字處理，以及無效工具呼叫的重試指引
- **日曆與提醒強化** — 強化 Google Calendar token 與提醒去重
- **入門流程與核准恢復** — 修正前端入門流程啟動與已確認能力的恢復
- **WordPress.org 打包審核問題** — 處理核心發佈版本的打包審核回饋

## 1.16.0 — 於 2026-05-20 發佈 {#1160--released-on-2026-05-20}

### 新增 {#new-1}

- **產生 Logo SVG 能力** — Theme Builder 現在可以產生並嵌入自訂 Logo SVG，並具備命名空間安全的清理處理
- **探索訪談中的照片上傳** — Theme Builder 探索訪談現在包含照片上傳步驟，以提供更豐富的設計脈絡
- **驗證調色盤對比能力** — 在套用到 theme 前，檢查色彩配對是否符合 WCAG
- **餐旅選單** — Theme Builder 現在可以為餐旅業務產生結構化的食物與飲品選單頁面
- **桌面與行動裝置預覽算繪** — 在設計方向選擇期間，於桌面與行動裝置上預覽你的設計
- **導覽標籤參數** — Create Menu 能力現在支援與頁面標題分開的獨立 `navigation_label`
- **Tier 1 工具可用性** — sd-ai-agent/site-scrape 現在是 Theme Builder 中預設可用的 Tier 1 工具

### 修正 {#fixed-1}

- **AI 用戶端快取** — 現在由 transients 支援跨請求持久化，防止長時間執行的 agent 任務發生資料遺失
- **Plugin 列動作連結** — 已修正並重新命名以提升清晰度

## 1.10.0 — 於 2026-05-05 發佈 {#1100--released-on-2026-05-05}

### 新增 {#new-2}

- **Tavily 網際網路搜尋** — 新增 Tavily 作為搜尋供應商，與 Brave Search 一同提供更豐富的網際網路搜尋結果
- **具 theme 感知的內建技能** — Block Themes、Classic Themes、Kadence Blocks 與 Kadence Theme 技能指南現在隨 plugin 一併提供
- **網站建構器聯絡表單能力** — 可直接從聊天介面將聯絡表單新增至任何頁面

### 改進 {#improved-1}

- **WooCommerce 整合重構** — 現在使用原生 WooCommerce API，以提升可靠性與相容性
- **供應商清單自動重新整理** — 當任何 plugin 啟用或停用時

### 修正 {#fixed-2}

- **navigate-to 能力** — 修正在某些管理頁面上的無限重新載入迴圈
- **list-posts 能力** — 現在能正確將分類與標籤名稱解析為代稱
- **WP-CLI 命令** — 在先前重構後恢復遺失的命名空間別名
- **事件自動化** — 對於尚未建立自動化資料表的網站，現在可妥善處理
- **memory-save 能力** — 現在於系統指令建構器中使用正確的命名空間前綴
- **純量工具結果** — 現在會在傳回給 AI 前正確包裝
- **使用統計** — 現在從舊版本升級時，可正確處理舊版能力鍵格式
