# 更新日誌

## 1.16.0 — 發布日期 2026-05-20

### 新增功能

- **生成 Logo SVG 功能** — Theme Builder 現在可以生成並嵌入帶有命名空間安全清理的自訂 Logo SVG
- **探索式訪談中的照片上傳** — Theme Builder 的探索式訪談現在包含照片上傳步驟，以提供更豐富的設計背景資訊
- **驗證調色盤對比度功能** — 在將顏色應用到主題之前，可以檢查顏色組合是否符合 WCAG 標準
- **餐飲菜單** — Theme Builder 現在可以為餐飲相關企業生成結構化的食品和飲料菜單頁面
- **桌面和行動裝置預覽渲染** — 在選擇設計方向時，可以在桌面和行動裝置上預覽您的設計
- **導航標籤參數** — Create Menu 功能現在支援一個獨立於頁面標題的 `navigation_label`
- **Tier 1 工具可用性** — sd-ai-agent/site-scrape 現在是一個 Tier 1 工具，預設在 Theme Builder 中可用

### 修復問題

- **AI 客戶端快取** — 現在使用臨時資料 (transients) 進行跨請求持久化，防止在長時間運行的 Agent 任務中丟失資料
- **Plugin 行操作連結** — 已修正並重新命名，使其更清晰

## 1.10.0 — 發布日期 2026-05-05

### 新增功能

- **Tavily 網路搜尋** — 在 Brave Search 之外，新增 Tavily 作為搜尋提供者，以提供更豐富的網路搜尋結果
- **主題感知內建技能** — Block Themes、Classic Themes、Kadence Blocks 和 Kadence Theme 的技能指南現在已包含在插件中
- **網站建構器聯絡表單功能** — 可直接從聊天介面將聯絡表單新增到任何頁面

### 改進功能

- **WooCommerce 整合重構** — 現在使用原生的 WooCommerce API，提高了可靠性和相容性
- **提供者列表自動刷新** — 當任何插件被啟用或停用時，列表會自動刷新

### 修復問題

- **navigate-to 功能** — 修復了某些管理頁面上的無限重新載入迴圈
- **list-posts 功能** — 現在能正確地將分類和標籤名稱解析為 slug
- **WP-CLI 指令** — 在先前重構後，恢復了遺失的命名空間別名
- **事件自動化** — 對於尚未建立自動化資料表的網站，現在能優雅地處理
- **memory-save 功能** — 現在在系統指令建構器中使用正確的命名空間前綴
- **Scalar 工具結果** — 現在在返回給 AI 之前，會正確地進行包裝
- **使用統計數據** — 在從舊版本升級時，現在能正確處理舊的功能鍵格式
