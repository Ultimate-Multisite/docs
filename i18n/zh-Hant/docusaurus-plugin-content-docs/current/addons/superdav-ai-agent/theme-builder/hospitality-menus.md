# 餐飲菜單 {#hospitality-menus}

**餐飲菜單**功能讓 Theme Builder 能夠直接在您的 WordPress 網站中生成並嵌入結構化的餐飲菜單頁面。

## 概覽 {#overview}

Theme Builder 現在可以為餐飲業（包括餐廳、咖啡館、酒吧和餐飲服務）創建專業、結構化的菜單頁面。這些菜單與您的網站設計完全整合，並且可以輕鬆地更新和管理。

## 支援的餐飲類型 {#supported-hospitality-types}

- **餐廳** — 全方位服務的餐飲菜單
- **咖啡館** — 咖啡和輕食菜單
- **酒吧和休息室** — 飲品和開胃菜菜單
- **麵包店** — 糕點和麵包菜單
- **餐飲服務** — 活動菜單選項
- **餐車** — 移動餐飲服務菜單
- **酒莊和釀酒廠** — 帶有描述的飲品菜單

## 菜單結構 {#menu-structure}

### 菜單類別 {#menu-categories}

菜單會根據類別進行組織：

- **開胃菜** — 前菜和小點心
- **主菜** — 主要餐點
- **配菜** — 搭配和蔬菜
- **甜點** — 甜點
- **飲品** — 飲料（酒精和非酒精）
- **特色菜** — 當日或季節限定特色菜

### 菜單項目格式 {#menu-item-format}

每個菜單項目都包含：

```json
{
  "name": "烤鮭魚",
  "description": "搭配檸檬奶油醬的新鮮大西洋鮭魚",
  "price": "$24.95",
  "dietary_info": ["無麩質", "高蛋白"],
  "allergens": ["魚"],
  "availability": "每日供應"
}
```

#### 項目欄位 {#item-fields}

| Field | Type | Description |
|-------|------|-------------|
| `name` | string | 菜餚或飲品名稱 |
| `description` | string | 項目詳細描述 |
| `price` | string | 價格（需格式化貨幣） |
| `dietary_info` | array | 飲食屬性（純素、無麩質等） |
| `allergens` | array | 常見過敏原（堅果、貝類等） |
| `availability` | string | 可用時間（每日、季節性等） |

## 創建餐飲菜單 {#creating-hospitality-menus}

### 步驟 1：提供菜單資訊 {#step-1-provide-menu-information}

告訴 Theme Builder 您的菜單內容：

```
為我的義式餐廳創建一份菜單：包含開胃菜、義大利麵主食、主菜和甜點。請包含價格和描述。
```

### 步驟 2：Theme Builder 生成菜單 {#step-2-theme-builder-generates-menu}

Theme Builder 會：

1. 創建一個結構化的菜單頁面
2. 設計使其與您的網站主題相匹配
3. 將項目組織到不同的類別中
4. 格式化價格和描述
5. 添加飲食和過敏原資訊

### 步驟 3：審閱和自訂 {#step-3-review-and-customize}

您可以：

1. 編輯菜單項目和價格
2. 添加或移除類別
3. 重新排序類別內的項目
4. 更新描述和飲食資訊
5. 調整樣式和佈局

## 菜單顯示選項 {#menu-display-options}

### 完整菜單頁面 {#full-menu-page}

一個專門展示您完整菜單的頁面：

- 依類別組織
- 可搜尋和篩選
- 適合列印的佈局
- 響應手機設計

### 菜單小工具 (Menu Widget) {#menu-widget}

將菜單部分嵌入到其他頁面：

- 網站首頁的特色項目
- 側邊欄的每日特價
- 酒吧頁面的飲品菜單
- 頁腳的甜點展示區

### 菜單 PDF {#menu-pdf}

生成可下載的 PDF 菜單：

- 專業的格式化
- 適合列印的品質
- 包含圖片和描述
- 易於分享和電子郵件傳送

## 飲食和過敏原資訊 {#dietary-and-allergen-information}

### 飲食屬性 {#dietary-attributes}

標記帶有飲食資訊的項目：

- **純素 (Vegan)** — 不含任何動物產品
- **素食 (Vegetarian)** — 不含肉類
- **無麩質 (Gluten-Free)** — 適合乳糜瀉患者
- **無乳製品 (Dairy-Free)** — 不含牛奶產品
- **無堅果 (Nut-Free)** — 不含樹堅果或花生
- **低碳水 (Low-Carb)** — 碳水化合物含量較低
- **高蛋白 (High-Protein)** — 富含蛋白質

### 過敏原警告 {#allergen-warnings}

包含常見過敏原：

- **堅果 (Nuts)** — 樹堅果和花生
- **貝類 (Shellfish)** — 甲殼類和軟體動物
- **魚 (Fish)** — 所有魚類物種
- **乳製品 (Dairy)** — 牛奶和乳製品
- **蛋 (Eggs)** — 雞蛋
- **大豆 (Soy)** — 大豆製品
- **麩質 (Gluten)** — 小麥和含麩質穀物
- **芝麻 (Sesame)** — 芝麻籽和油

## 菜單管理 {#menu-management}

### 更新價格 {#updating-prices}

輕鬆更新菜單價格：

1. 導航到菜單頁面
2. 點擊「編輯菜單」
3. 更新項目價格
4. 儲存變更
5. 變更會立即顯示在您的網站上

### 添加季節性項目 {#adding-seasonal-items}

創建季節性菜單變體：

1. 創建一個新的菜單版本
2. 添加季節性項目
3. 將項目標記為「季節限定」
4. 設定供應日期
5. 在季節期間自動顯示

### 管理特色菜 {#managing-specials}

展示每日或每週的特色菜：

1. 創建一個「特色菜」類別
2. 添加帶有供應日期的項目
3. 在首頁突出顯示特色菜
4. 每日或每週更新
5. 歸檔舊的特色菜

## 與 Theme Builder 的整合 {#integration-with-theme-builder}

當您使用 Theme Builder 為餐飲網站時：

1. **自動菜單偵測** — 識別您是否為餐飲相關業務
2. **菜單頁面創建** — 生成專業的菜單頁面
3. **設計匹配** — 菜單與您的網站主題保持一致
4. **行動裝置優化** — 菜單在手機上顯示美觀
5. **SEO 優化** — 菜單對搜尋引擎友好

## 最佳實踐 {#best-practices}

### 菜單設計 {#menu-design}

- **清晰的組織** — 邏輯清晰的類別結構
- **易讀的描述** — 誘人且資訊豐富
- **一致的定價** — 貨幣和格式清晰
- **專業的照片** — 高品質的食物圖片
- **留白** — 不要讓頁面過於擁擠

### 內容 {#content}

- **準確的描述** — 準確描述項目內容
- **突出特色菜** — 使特色菜更引人注目
- **包含過敏原** — 務必列出過敏原
- **定期更新** — 保持價格和項目內容的最新
- **使用誘人的語言** — 讓項目聽起來美味可口

### 無障礙性 {#accessibility}

- **易讀的字體** — 使用清晰、易讀的字體
- **足夠的對比度** — 確保文字易於閱讀
- **飲食標籤** — 清楚標記飲食選項
- **過敏原警告** — 醒目地顯示過敏原資訊
- **行動裝置友善** — 在所有設備上進行測試
