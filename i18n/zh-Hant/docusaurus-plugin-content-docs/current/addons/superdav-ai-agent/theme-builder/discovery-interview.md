# 探索訪談

**探索訪談 (Discovery Interview)** 是 Theme Builder 設計流程的第一步。它會收集關於您的網站、業務和設計偏好等資訊，從而為您打造一個個人化的主題。

## 概述 {#overview}

探索訪談會引導您回答一系列問題，以了解您的品牌、目標受眾和設計目標。在 1.16.0 版本中，訪談新增了**照片上傳步驟**，讓您可以提供視覺靈感，作為設計的參考。

## 訪談步驟 {#interview-steps}

### 1. 業務資訊 {#1-business-information}

- 網站名稱和標語
- 業務類型或行業
- 主要業務目標
- 目標受眾描述

### 2. 設計偏好 {#2-design-preferences}

- 偏好的配色方案
- 設計風格（現代、經典、極簡、活潑等）
- 靈感網站或品牌
- 必備的設計元素

### 3. 照片上傳（v1.16.0 新增） {#3-photo-upload-new-in-v1160}

上傳一張或多張照片，以影響設計方向：

#### 可接受格式 {#accepted-formats}

| 格式 | 擴充字元 | 詳細說明 |
|--------|-----------|---------|
| **JPEG** | `.jpg`, `.jpeg` | 適用於照片的最佳選擇 |
| **PNG** | `.png` | 支援透明度 |
| **WebP** | `.webp` | 現代格式，檔案較小 |
| **GIF** | `.gif` | 僅支援靜態圖片（不支援動畫） |

#### 檔案要求 {#file-requirements}

- **最大檔案大小：** 10 MB
- **最小尺寸：** 400 × 300 像素
- **建議尺寸：** 1200 × 800 像素或更大
- **長寬比：** 任何比例皆可

#### 照片如何影響設計方向 {#how-photos-influence-design-direction}

您上傳的照片有助於 Theme Builder 了解：

- **配色方案 (Color palette)** — 會提取主色調並用於主題設計
- **視覺風格 (Visual style)** — 現代、鄉村風、極簡、充滿活力等
- **氛圍和氣質 (Mood and atmosphere)** — 專業、創意、友善、奢華等
- **設計元素 (Design elements)** — 圖案、紋理和構圖風格
- **字體靈感 (Typography inspiration)** — 與視覺主題相輔相成的字體風格

### 4. 內容結構 {#4-content-structure}

- 所需的主要頁面（首頁、關於我們、服務、聯絡我們等）
- 是否需要部落格或新聞區塊
- 是否需要電子商務功能
- 特殊功能或整合需求

### 5. 回顧與確認 {#5-review-and-confirmation}

- 所有提供資訊的總結
- 在開始生成設計前進行的確認

## 照片上傳最佳實踐 {#photo-upload-best-practices}

### 應上傳的內容 {#what-to-upload}

- **品牌靈感** — 標誌、色塊或品牌指南
- **競爭對手網站** — 您欣賞的網站截圖
- **情緒板 (Mood boards)** — 代表您理想美學的圖片合集
- **產品照片** — 如果您有需要展示的產品
- **團隊或辦公室照片** — 用於個人或專業品牌建立
- **生活風格圖片** — 代表您的目標受眾

### 應避免的內容 {#what-to-avoid}

- **低品質圖片** — 模糊或像素化的照片
- **不相關的圖片** — 與您的品牌不相關的照片
- **文字過多的圖片** — 包含大量文字的截圖
- **受版權保護的材料** — 請確保您有權使用這些圖片
- **動畫 GIF** — 僅處理靜態圖片

### 最佳結果提示 {#tips-for-best-results}

1. **上傳 2-3 張圖片** — 多張圖片能提供更好的上下文參考
2. **使用高品質照片** — 清晰、光線充足的圖片效果最佳
3. **包含多樣性** — 混合不同顏色、風格和主題的圖片
4. **提供上下文** — 簡短描述您選擇每張圖片的原因
5. **具體化** — 如果您喜歡某個特定元素，請明確指出

## 設計方向輸出 {#design-direction-output}

完成探索訪談後，Theme Builder 會生成：

- **配色方案 (Color palette)** — 主色、輔色和強調色
- **字體 (Typography)** — 標題和內文的字體家族
- **佈局風格 (Layout style)** — 間距、對齊和組件排列
- **視覺元素 (Visual elements)** — 圖標、圖案和裝飾元素
- **情緒板 (Mood board)** — 設計方向的視覺呈現

## 後續步驟 {#next-steps}

完成探索訪談後：

1. 回顧生成的設計方向
2. 提供回饋或要求調整
3. 進入 [Design Direction](./design-direction.md) 步驟
4. Theme Builder 將為您創建客製化主題
