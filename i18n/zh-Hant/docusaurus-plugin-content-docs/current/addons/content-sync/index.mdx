# 內容同步

Multisite Ultimate：內容同步

此附加元件適用於 Multisite Ultimate，讓網路管理員能夠將任何站點的內容同步到網路中的其他站點。它支援所有文章類型，包括文章、頁面、自訂文章類型和 Elementor 範本。此附加元件包含一個多步驟嚮導介面，用於選擇內容和目標站點，並提供批次處理和即時進度回饋。

## 功能特色 {#features}

*   **內容選擇：** 視覺化卡片網格，支援縮圖和文章類型篩選
*   **所有文章類型：** 可同步文章、頁面、產品、Elementor 範本以及任何自訂文章類型
*   **站點選擇：** 可選擇要同步內容的目標站點，並支援排除來源站點和預選子站點
*   **批次處理：** 一次可同步至 10 個站點，並提供進度更新
*   **媒體同步：** 自動將文章內容中的媒體檔案（圖片、影片）複製並重新映射到目標站點
*   **分類同步：** 保留分類、標籤和自訂分類法
*   **Elementor 支援：** 對 Elementor 範本提供完整支援，包含 CSS 再生成和範本 ID 重新映射
*   **衝突解決：** 提供覆蓋現有內容或跳過內容的選項
*   **快取管理：** 在同步後，自動清除目標站點上的 Elementor 快取

## 需求條件 {#requirements}

*   WordPress 5.3 或更高版本
*   PHP 7.4 或更高版本
*   Multisite Ultimate 插件（必須啟用）
*   Elementor 插件（可選，但同步 Elementor 範本時需要）

## 安裝 {#installation}

1.  將 'ultimate-multisite-content-sync' 上傳到 '/wp-content/plugins/' 目錄
2.  透過 WordPress 網路管理員的「外掛程式」選單啟用此插件
3.  在 Ultimate Multisite → 設定 → 內容同步中配置設定
4.  導航至 Ultimate Multisite → 內容同步以開始同步
