# Admin Page Creator 更新日誌

### Version 1.9.0 - 於 2026-01-18 發布 {#version-190---released-on-2026-01-18}

* 修復：啟用時出現的致命錯誤。
* 重構：內容來源類現在使用單例特徵 (Singleton trait)，並正確初始化父建構子。
* 修復：Vue.js 模板編譯錯誤，導致內容類型選擇器無法渲染。
* 修復：缺少 Vue 資料屬性 (template_id, external_link_url_embedable, external_link_url_checking)。
* 修復：TinyMCE 編輯器在頁面載入時無法正常初始化。
* 修復：Divi 支援檔案未透過 Composer 自動載入。
* 修復：Divi 支援類別中的 PHP 語法已損壞。
* 改進：JavaScript 資源的組織和最小化。
* 改進：與最新版本的 WordPress 和頁面建構器相容。
