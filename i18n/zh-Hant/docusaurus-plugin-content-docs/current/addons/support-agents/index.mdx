# 支援代理人 {#support-agents}

自己管理整個網路可不是一件容易的事。隨著您的網路規模逐漸擴大，您會發現支援需求也越來越高。這時候，招募一些人來協助您分擔工作負擔會是非常好的主意。然而，給予這些人完全的存取權限風險很高，可能會危害到您的網路安全。這就是「支援代理人」附加元件的作用所在。

透過這個附加元件，您可以建立一種名為「支援代理人」（Support Agent）的新型 WordPress 使用者，並且可以完全控制該使用者擁有的權限範圍。授予權限就像按下開關按鈕一樣簡單，如果之後您決定移除權限，關閉它也同樣容易。

## 功能特色 {#features}

*   包含 WordPress 和 Ultimate Multisite 的權限項目清單
*   輕鬆啟用和停用存取權限
*   可建立無限數量的代理人帳戶

## 安裝 {#installation}

1.  將 'ultimate-multisite-support-agents' 上傳到 `/wp-content/plugins/` 目錄
2.  透過 WordPress 的「外掛」（Plugins）選單啟用此外掛
3.  依照逐步嚮導（Wizard）的指示完成外掛的設定
