# 計費方案更新日誌

版本 1.1.0 - 發布日期 2026-05-05
- 新增：針對多站點子站的 AI Token 計費功能 — 可以追蹤並根據可配置的每 Token 費率對客戶站點的 AI Token 用量進行計費。
- 新增：連接器強制執行功能重構，採用動態限制發現和寫入式同步，確保所有連接器都能達到即時準確性。
- 修復：資料庫表格升級現在已正確對齊 BerlinDB 的結構定義，防止新安裝時發生升級失敗。
- 修復：資料庫升級回呼已轉換為正確格式，解決了靜默升級失敗問題。
- 修復：現在接受 AI 用量超額加價輸入欄位中的小數值。
- 修復：解決了插件加載時的致命錯誤和重複初始化問題。
- 改善：新增了 check-env npm script，讓開發環境在首次運行時自動配置。
