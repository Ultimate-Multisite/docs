# PayFast

Multisite Ultimate: Payfast 支付閘道

此插件為 Multisite Ultimate 增加了 PayFast 支付選項。PayFast 是南非一個快速且安全的線上支付閘道。

## 安裝 {#installation}

1. 將 'ultimate-multisite-payfast' 上傳到 '/wp-content/plugins/' 目錄
2. 透過 WordPress 的「外掛」選單啟用此插件
3. 依照步驟引導式的嚮導（Wizard）設定插件

## 使用指南 {#usage-guide}

PayFast 是南非企業領先的線上支付處理商。它致力於透過讓交易流程順暢且安全，來幫助線上商家成長。根據 Ultimate Multisite 使用者的需求，我們決定創建一個附加元件，以實現與 PayFast 的整合。本文將展示如何將 PayFast 設定為您的支付閘道，以便在您的 Ultimate Multisite 網路中接收款項。

第一步是安裝此附加元件。

# 如何安裝 Payfast 附加元件：

前往附加元件頁面，選擇 Ultimate Multisite: Payfast Gateway 附加元件。

![Add-ons page](/img/admin/addons-page.png)

附加元件視窗會彈出。點擊「立即安裝」（Install Now）。

![Add-ons page with install option](/img/admin/addons-page.png)

安裝完成後，您將被重定向到外掛頁面。在這裡，只需點擊「網路啟用」（Network Activate），_Payfast_ 附加元件就會在您的網路中啟用。

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# 設定 PayFast

安裝附加元件後，您現在可以前往 Ultimate Multisite 設定，開始配置 PayFast。

在 **Ultimate Multisite 設定** 頁面，您可以在「支付」（Payments）分頁找到相關選項。在這個分頁下，您應該會看到「**Payfast**」，旁邊有一個開關按鈕，用於啟用或停用此支付閘道。

![Payment gateway settings](/img/config/settings-payment-gateways.png)請點擊開關按鈕來啟用它。接著，它會展開一個區塊，顯示您需要填寫的各種欄位，例如 **Merchant ID**、**Merchant Key** 和 **PassPhrase**。您可以從您的 **Payfast** 帳戶獲取這些資訊。

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

接下來的步驟是登入您的 PayFast 帳戶，以獲取所需的資訊。

登入後，您可以在 **Payfast** 帳戶儀表板上立即看到您需要的憑證。

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->您只需複製並貼上這些憑證，然後將詳細資訊填寫到您的 Ultimate Multisite 設定中。接著點擊 **「儲存設定」** 按鈕即可儲存變更。

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

就是這樣，您的 PayFast 支付整合就完成了！
