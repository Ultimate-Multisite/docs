## 🚦 工作流程 {#-how-it-works}

### 1. **設計表單** {#1-design-your-form}
使用 Gravity Forms 的拖放建構器，創建完美的註冊體驗。

### 2. **映射欄位** {#2-map-your-fields}
使用我們的視覺化映射介面，將表單欄位連接到 Multisite Ultimate 的資料點。

### 3. **配置支付** {#3-configure-payment}
在結帳重定向或 WooCommerce 整合之間進行選擇，以處理支付。

### 4. **上線營運** {#4-go-live}
發布您的表單，開始將訪客轉化為付費客戶！
