# AffiliateWP

Multisite Ultimate：AffiliateWP 整合

利用功能強大的 AffiliateWP 來擴大您的 Ultimo Network 客戶群！

## 安裝 {#installation}

1. 將 'ultimate-multisite-git ' 上傳到 `/wp-content/plugins/` 目錄
2. 透過 WordPress 的「外掛」選單啟用此外掛
3. 依照步驟引導式的嚮導（Wizard）設定外掛

## 使用指南 {#usage-guide}

[AffiliateWP](https://affiliatewp.com/) 已成為為您的線上業務建立聯盟行銷計畫的首選解決方案。Ultimate Multisite: AffiliateWP 則是能讓您為您的專業網路建立相同功能的橋樑。

## 開始使用 {#getting-started}

首先，您**必須啟用** [**AffiliateWP**](https://affiliatewp.com/)！如果沒有**整合此外掛的插件**，您將無法使用我們的附加元件。如果您的客戶不打算在他們自己的網站上使用此功能，那麼 AffiliateWP 應該只在您的主網站上啟用。

此外，您還需要將 **Ultimate Multisite** 和 **Ultimate Multisite: AffiliateWP** **在整個網路範圍內都啟用**。

## 將 Ultimate Multisite 作為 AffiliateWP 整合 {#adding-ultimate-multisite-as-an-affiliatewp-integration}

在您的主網站上，前往「聯盟行銷設定」（Affiliates Settings），在「整合」（Integrations）分頁中，勾選 Ultimate Multisite。這會告訴 [AffiliateWP](https://affiliatewp.com/) 開始追蹤來自 Ultimate Multisite 註冊流程的造訪和轉換。

<!-- Screenshot unavailable: AffiliateWP Integrations settings tab with Ultimate Multisite checkbox -->

_在底部勾選 Ultimate Multisite，然後儲存_

基本上就這麼簡單了。來自推薦連結的造訪將會被追蹤，當該用戶稍後完成首次付款時，轉換就會被建立，並且佣金會撥給負責該轉換的聯盟行銷人員。

不過，只有新創建用戶的首次付款會被追蹤。若要啟用週期性推薦付款，您需要 [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## 啟用週期性推薦 {#enabling-recurring-referrals}

安裝了 [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) 後，請前往「聯盟行銷設定」的「週期性推薦」（Recurring Referrals）分頁，並啟用週期性推薦功能。

<!-- Screenshot unavailable: AffiliateWP Recurring Referrals settings with Enable Recurring Referrals toggle -->

_簡單到不行：啟用週期性推薦，您就大功告成了！_

再次強調，就是這麼簡單！

現在，每當推薦帳戶收到新的付款時，負責的聯盟行銷人員都會自動獲得應有的分成！
