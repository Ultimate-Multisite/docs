# 網路多站點更新日誌

### 1.3.0
* 新增：網路範本預覽器 — 在購買前，即可透過即時面板瀏覽和預覽網路範本。
* 新增：結帳頁面網路範本選擇器 — 客戶在註冊結帳流程中可以選擇網路範本。
* 新增：跨網路魔術連結 SSO — 客戶可以透過魔術連結認證，無縫存取子網路。
* 新增：網路媒體克隆 — 從範本克隆網路時，網站媒體會正確複製。
* 修復：現在即使沒有多租戶外掛，網路克隆也能正常運作；超級管理員權限會在克隆後重新賦予。
* 修復：網路建立現在與 WooCommerce 和所有付款閘道相容。
* 修復：埠號現在包含在備用網域生成中。
* 修復：範本選擇器 CSS 已移動到排隊樣式表，確保所有主題都能正確渲染。
* 修復：防止在 site query 範圍內出現未定義的 `network_id` 鍵。
* 修復：在切換上下文前檢查網路權限，避免了 `wpdb` 資料表屬性錯誤。
