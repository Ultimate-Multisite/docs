# Action: update_network

當現有的網路已更新後觸發。

## 參數

| 名稱 | 類型 | 描述 |
|------|------|-------------|
| $network_id | `int` | 已新增網路的 ID。 |
| $args | `array` | 包含網路參數的關聯陣列。 |
