# 網域銷售工具 {#domain-seller}

**Multisite Ultimate: Domain Seller** 會將你的 WordPress Multisite 網路轉變為強大的網域銷售平台。讓客戶在註冊期間或從他們的 Dashboard 直接註冊自訂網域，並提供即時可用性檢查與透明定價。

### 為什麼選擇網域銷售工具？ {#why-domain-seller}

網域註冊是任何 WaaS（網站即服務）平台最常被要求的加購項目之一。你可以直接提供網域，而不是將客戶導向第三方註冊商——如此可讓更多收入留在你的網路中，同時為客戶提供順暢無阻的體驗。

### 主要功能 {#key-features}

#### 支援多個網域註冊商 {#multiple-domain-registrar-support}
* **OpenSRS** — 完整的註冊、續約與管理
* **Namecheap** — 與 WhoisGuard 和沙盒測試完整整合
* **HostAfrica** — 透過 Domains Reseller API 提供網域搜尋、定價、註冊、轉移、DNS 與身分保護
* **Openprovider** — 經銷商定價、註冊、續約、轉移、DNS、WHOIS 隱私與 TLD 同步
* **Hostinger** — 使用共用的 Hostinger 整合 token 進行網域搜尋與註冊
* **可擴充架構** — 建立於 Integration Registry 模式之上，讓新增更多提供者變得容易
* **內建連線測試** — 在正式上線前驗證提供者憑證與連線能力

#### 引導式設定精靈 {#guided-setup-wizard}
* **一鍵入門** — 引導你完成提供者設定、TLD 匯入與網域產品建立
* **自動 TLD 同步** — 從已連接的提供者匯入所有可用 TLD 與批發定價
* **預設產品建立** — 使用合理的預設加價建立可立即使用的網域產品

#### 彈性網域定價 {#flexible-domain-pricing}
* **三種加價模式** — 百分比加價、固定加價，或每個 TLD 固定價格
* **TLD 專屬產品** — 為不同 TLD 群組（.com/.net、.io、國碼等）建立獨立的網域產品
* **首年優惠定價** — 提供折扣首年價格以吸引新的註冊
* **即時定價** — 來自提供者的即時批發定價，並具備智慧快取
* **優惠券支援** — 與 Multisite Ultimate 折扣代碼完全相容

#### WHOIS 隱私保護 {#whois-privacy-protection}
* **按產品設定** — 將 WHOIS 隱私設定為停用、永遠包含，或客戶選擇
* **客戶選擇加入** — 設為客戶選擇時，checkout 核取方塊可讓客戶新增隱私保護
* **可設定定價** — 為隱私加購項目設定每年價格，或免費提供
* **註冊商整合** — 在註冊時自動啟用 WhoisGuard（Namecheap）或 WHOIS 隱私（OpenSRS）

#### 統一的 Checkout 體驗 {#unified-checkout-experience}
* **三種網域模式** — 客戶可在子網域（免費）、註冊新網域，或對應既有且已擁有的網域之間選擇
* **即時網域搜尋** — 由 AJAX 驅動的可用性檢查，無需重新載入頁面
* **智慧建議** — 偏好的網域已被註冊時，提供替代 TLD 建議
* **透明定價** — 清楚顯示註冊價格、續約價格與可選的 WHOIS 隱私費用
* **註冊人聯絡欄位** — 在 checkout 期間即時收集必要的註冊人資訊（姓名、地址、電話）
* **自動產生的網站 URL** — 網站位址會從所選網域自動衍生

#### 自動續約系統 {#auto-renewal-system}
* **與會員資格綁定的續約** — 客戶的會員資格有效期間，網域會自動續約
* **智慧排程** — 續約作業會依據網域到期日排入佇列
* **重試邏輯** — 對失敗的續約嘗試進行智慧重試
* **Email 通知** — 可設定即將續約、續約成功、失敗與到期的提醒

#### 客戶 DNS 管理 {#customer-dns-management}
* **自助式 DNS** — 客戶可從 Dashboard 新增、編輯與刪除 DNS 記錄（A、AAAA、CNAME、MX、TXT）
* **網域狀態** — 即時註冊狀態、到期日與 nameserver 資訊
* **自動網域對應** — 已註冊或既有網域會自動對應到客戶的網站

#### Admin 工具 {#admin-tools}
* **手動網域註冊** — 從 admin panel 代表客戶註冊網域
* **網域監控** — 從單一頁面檢視所有客戶網域、狀態與到期日
* **提供者專屬記錄** — 每個註冊商都有獨立記錄通道（例如 domain-seller-namecheap.log）
* **註冊詳細資料** — 在核心網域編輯頁面檢視註冊人資訊與註冊資料

### 支援的提供者 {#supported-providers}

* **OpenSRS** — 完整整合註冊、續約、DNS 管理與 WHOIS 隱私
* **Namecheap** — 完整整合註冊、續約、批次可用性檢查與免費 WhoisGuard
* **HostAfrica** — 註冊、續約、轉移、DNS 管理、EPP code 擷取、註冊商鎖定與身分保護
* **Openprovider** — 註冊、續約、轉移、DNS 管理、EPP code 擷取、註冊商鎖定、WHOIS 隱私與經銷商 TLD 同步
* **Hostinger** — 註冊、網域查詢、nameserver 更新、註冊商鎖定與 WHOIS 隱私；續約由 Hostinger 自動續約處理
* **GoDaddy** — 註冊與續約
* **ResellerClub** — 註冊、續約與 DNS 管理
* **NameSilo** — 註冊與續約
* **Enom** — 註冊、續約與 DNS 管理

### 技術詳細資料 {#technical-details}

* **需求：** PHP 7.4+、Multisite Ultimate 2.4.12+
* **架構：** 具有 Domain_Selling_Capability 介面的 Integration Registry 模式
* **自動載入：** Jetpack autoloader 搭配 PSR-4 類別對應
* **標準：** WordPress coding standards、strict typing、prepared statements

## 安裝 {#installation}

1. 確保已安裝並網路啟用 **Multisite Ultimate v2.4.12+**
2. 將 plugin 上傳至 `/wp-content/plugins/ultimate-multisite-domain-seller/`
3. 從 WordPress Network Admin 的 **Plugins** 網路啟用 plugin
4. **Setup Wizard** 會在首次啟用時自動啟動 — 依照步驟：
   * 連接你的網域註冊商
   * 測試連線
   * 匯入可用的 TLD 與批發價格
   * 建立你的第一個網域 product，並設定價格加成
5. 在 **Ultimate Multisite > Checkout Forms** 中將 **Domain Selection** 欄位新增至你的 checkout 表單

## 常見問題 {#frequently-asked-questions}

### 需要哪個版本的 Multisite Ultimate？ {#what-version-of-multisite-ultimate-is-required}

版本 2.4.12 或更高。此 addon 不相容於 WP Ultimo v1.x 或低於 2.4.12 的 Ultimate Multisite 版本。

### 支援哪些網域提供者？ {#which-domain-providers-are-supported}

支援九個註冊商：**OpenSRS**、**Namecheap**、**HostAfrica**、**Openprovider**、**Hostinger**、**GoDaddy**、**ResellerClub**、**NameSilo** 和 **Enom**。功能會因提供者而異，因為註冊商 API 會公開不同的操作；請參閱設定指南，了解各提供者的概覽。此 addon 的 Integration Registry 架構讓新增自訂提供者變得很直接 — 請參閱[開發者指南](/developer/integration-guide/custom-registrar)。

### 我可以提供 WHOIS 隱私保護嗎？ {#can-i-offer-whois-privacy-protection}

可以。每個網域 product 都有一個 WHOIS Privacy 設定，包含三種模式：停用、永遠包含，或客戶選擇。設為客戶選擇時，客戶會在 checkout 期間看到一個核取方塊，並顯示價格（或 Namecheap 的 WhoisGuard 顯示「免費」）。隱私標記會在註冊時自動傳遞給註冊商。

### 我可以為不同 TLD 設定不同價格嗎？ {#can-i-set-different-pricing-for-different-tlds}

可以。為不同 TLD 群組建立個別的網域 product。每個 product 都可以有自己的加成類型（百分比、固定加成或固定價格）、支援的 TLD，以及 introductory pricing。

### 客戶會看到哪些 checkout 選項？ {#what-checkout-options-do-customers-see}

Domain Selection 欄位會提供客戶三個分頁：**Subdomain**（在你的網路上使用免費子網域）、**Register New Domain**（搜尋並註冊新網域），以及 **Existing Domain**（對應他們已擁有的網域）。只有你啟用的模式會顯示。

### 它支援自動續約嗎？ {#does-it-support-automatic-renewals}

是。網域可以設定為自動續約，並與客戶的 membership 狀態綁定。續約付款會透過客戶現有的付款閘道處理。

### 客戶可以管理他們的 DNS 記錄嗎？ {#can-customers-manage-their-dns-records}

可以。客戶會在自己的 dashboard 中取得 DNS 管理介面，可新增、編輯和刪除 A、AAAA、CNAME、MX 和 TXT 記錄。DNS 管理可用於 OpenSRS、ResellerClub、Enom、HostAfrica 和 Openprovider。託管網域的 Hostinger DNS 由核心 Hostinger 網域對應整合處理。

### 有設定精靈嗎？ {#is-there-a-setup-wizard}

有。設定精靈會在首次啟用時自動執行，並引導你連接註冊商、匯入 TLD，以及建立你的第一個網域 product。你也可以從 addon 設定重新執行精靈。
