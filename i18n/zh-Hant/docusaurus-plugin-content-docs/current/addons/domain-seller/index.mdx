# Domain Seller

**Multisite Ultimate: Domain Seller** 能將您的 WordPress Multisite 網路，轉變為一個強大的網域銷售平台。讓客戶可以在註冊時或從儀表板直接註冊自訂網域，並提供即時的可用性檢查和透明的定價。

### 為什麼需要 Domain Seller？

網域註冊是任何 WaaS（網站即服務）平台最常被要求增加的附加服務之一。您不需要將客戶導向第三方註冊商，而是可以直接提供網域服務——這樣不僅能讓您的網路保留更多收入，也能為客戶提供無縫的體驗。

### 主要功能

#### 多重網域註冊商支援
* **OpenSRS** — 提供完整的註冊、續期和管理功能
* **Namecheap** — 與 WhoisGuard 和沙盒測試進行完整整合
* **可擴展架構** — 基於整合註冊表模式（Integration Registry pattern）構建，讓您輕鬆增加更多服務商
* **內建連線測試** — 在正式上線前，驗證服務商的憑證和連線狀態

#### 導引式設定嚮導
* **一鍵上手** — 帶您完成服務商配置、TLD 匯入和網域產品創建的整個流程
* **自動 TLD 同步** — 從您連接的服務商處匯入所有可用的 TLD 和批發價格
* **預設產品創建** — 創建一個即用型的網域產品，並預設合理的加價（markup）

#### 彈性網域定價
* **三種加價模式** — 百分比加價、固定加價，或每個 TLD 的固定價格
* **特定 TLD 產品** — 為不同的 TLD 群組（例如 .com/.net、.io、國家代碼等）創建獨立的網域產品
* **入門價格** — 提供折扣的第一年價格，以吸引新的註冊客戶
* **即時定價** — 來自服務商的即時批發價格，並具備智慧快取功能
* **優惠券支援** — 與 Multisite Ultimate 的折扣代碼完全相容

#### WHOIS 隱私保護
* **按產品配置** — 設定 WHOIS 隱私為「停用」、「總是包含」或「客戶選擇」
* **客戶選擇加入** — 當設定為「客戶選擇」時，結帳頁面會提供一個勾選框，讓客戶選擇是否增加隱私保護
* **可配置價格** — 為隱私附加服務設定每年的價格，或提供免費選項
* **註冊商整合** — 在註冊時自動啟用 WhoisGuard (Namecheap) 或 WHOIS 隱私 (OpenSRS)

#### 統一結帳體驗
* **三種網域模式** — 客戶可選擇子網域（免費）、註冊新網域，或映射他們已擁有的現有網域
* **即時網域搜尋** — 採用 AJAX 技術進行可用性檢查，無需重新載入頁面
* **智慧建議** — 當首選網域已被佔用時，提供替代 TLD 建議
* **透明定價** — 清晰顯示註冊價格、續期價格和可選的 WHOIS 隱私費用
* **註冊人聯絡資訊欄位** — 在結帳過程中，即時收集所需的註冊人資訊（姓名、地址、電話）
* **自動生成網站 URL** — 網站地址會根據所選網域自動推導

#### 自動續期系統
* **綁定會員的續期** — 只要客戶的會員狀態活躍，網域就會自動續期
* **智慧排程** — 續期任務會根據網域到期日期排隊處理
* **重試邏輯** — 對於續期失敗的嘗試，提供智慧重試機制
* **電子郵件通知** — 可配置關於即將到期、成功續期、失敗和到期的警報通知

#### 客戶 DNS 管理
* **自助式 DNS** — 客戶可以在儀表板中新增、編輯和刪除 DNS 記錄（A, AAAA, CNAME, MX, TXT）
* **網域狀態** — 即時的註冊狀態、到期日期和名稱伺服器資訊
* **自動網域映射** — 註冊或現有的網域會自動映射到客戶的網站

#### 管理員工具
* **手動網域註冊** — 從管理面板為客戶註冊網域
* **網域監控** — 從一個頁面查看所有客戶的網域、狀態和到期日期
* **服務商特定日誌** — 為每個註冊商設置獨立的日誌通道（例如 domain-seller-namecheap.log）
* **註冊詳情** — 在核心網域編輯頁面查看註冊人資訊和註冊數據

### 支援的服務商

* **OpenSRS** — 提供註冊、續期、DNS 管理和 WHOIS 隱私的完整整合
* **Namecheap** — 提供註冊、續期、批次可用性檢查和免費 WhoisGuard 的完整整合
* **GoDaddy** — 註冊和續期
* **ResellerClub** — 註冊、續期和 DNS 管理
* **NameSilo** — 註冊和續期
* **Enom** — 註冊、續期和 DNS 管理

### 技術細節

* **要求:** PHP 7.4+，Multisite Ultimate 2.4.12+
* **架構:** 整合註冊表模式（Integration Registry pattern）與 Domain_Selling_Capability 介面
* **自動載入:** 使用 Jetpack autoloader 搭配 PSR-4 類別映射
* **標準:** WordPress 編碼標準、嚴格類型檢查、預處理語句

## 安裝

1. 確保已安裝並在網路層級啟用 **Multisite Ultimate v2.4.12+**
2. 將外掛程式上傳到 `/wp-content/plugins/ultimate-multisite-domain-seller/`
3. 在 WordPress 網路管理員的 **外掛程式** 中進行網路啟用
4. 首次啟用時，**設定嚮導** 會自動啟動——請按照步驟：
   * 連接您的網域註冊商（OpenSRS 或 Namecheap）
   * 測試連線
   * 匯入可用的 TLD 和批發價格
   * 使用定價加成創建您的第一個網域產品
5. 在 **Ultimate Multisite > 結帳表單** 中，新增 **網域選擇** 欄位

## 常見問題

### 需要哪個版本的 Multisite Ultimate？

版本 2.4.12 或更高。此附加元件不相容於 WP Ultimo v1.x 或低於 2.4.12 的 Ultimate Multisite 版本。

### 支援哪些網域服務商？

完全支援六個註冊商：**OpenSRS**、**Namecheap**、**GoDaddy**、**ResellerClub**、**NameSilo** 和 **Enom**。此附加元件的整合註冊表架構使其很容易增加自訂服務商——請參閱 [開發者指南](/developer/integration-guide/custom-registrar)。

### 我可以提供 WHOIS 隱私保護嗎？

可以。每個網域產品都有一個 WHOIS 隱私設定，包含三種模式：停用、總是包含或客戶選擇。當設定為「客戶選擇」時，客戶在結帳頁面會看到一個勾選框，並顯示價格（或 Namecheap WhoisGuard 的「免費」）。隱私標記會在註冊時自動傳遞給註冊商。

### 我可以為不同的 TLD 設定不同的價格嗎？

可以。為不同的 TLD 群組創建獨立的網域產品。每個產品可以擁有自己的加價類型（百分比、固定加價或固定價格）、支援的 TLD 和入門價格。

### 客戶在結帳時會看到哪些選項？

網域選擇欄位會為客戶提供三個分頁：**子網域**（使用您網路上的免費子網域）、**註冊新網域**（搜尋並註冊新網域），和 **現有網域**（映射他們已擁有的網域）。只有您啟用的模式才會顯示。

### 它是否支援自動續期？

是的。網域可以設定自動續期，並且與客戶的會員狀態綁定。續期付款會透過客戶現有的支付閘道處理。

### 客戶可以管理他們的 DNS 記錄嗎？

是的。客戶可以在儀表板中獲得一個 DNS 管理介面，用於新增、編輯和刪除 A、AAAA、CNAME、MX 和 TXT 記錄。DNS 管理功能適用於 OpenSRS、ResellerClub 和 Enom。

### 有設定嚮導嗎？

是的。設定嚮導會在首次啟用時自動運行，引導您完成連接註冊商、匯入 TLD 和創建第一個網域產品的過程。您也可以從附加元件設定中重新運行嚮導。
