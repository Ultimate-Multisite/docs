# 網域銷售變更記錄

版本 1.3.0 - 發布於 2026-06-02
- 新增：當 HostAfrica 經銷商餘額過低時，新增 network-admin 警告
- 新增：新增將新註冊網域自動對應到網路網站的功能
- 修正：僅在註冊新網域時套用註冊人欄位需求
- 修正：讓監控餘額通知可關閉
- 修正：確保 WooCommerce 註冊人帳單詳細資料會被保留
- 修正：在註冊期間強制執行註冊人聯絡資料需求
- 修正：防止以 0% 加價建立網域註冊商品
- 修正：在 checkout 工作階段流程中保留網域選擇與定價
- 修正：改進 HostAfrica 網域定價的幣別顯示
- 修正：改進 checkout 表單動作行為，以防止 WP-core query-var 不相符
- 改進：在設定指引中連結 HostAfrica 經銷商組態文件

版本 1.2.0 - 發布於 2026-05-25
- 新增：新增 HostAfrica 作為網域銷售整合，支援 checkout、設定精靈、查詢、TLD/定價、註冊、續約、移轉、名稱伺服器、DNS、EPP code、註冊商鎖定與 ID 保護
- 新增：新增 Openprovider 作為網域銷售整合，支援經銷商定價、註冊、續約、移轉、名稱伺服器、DNS、EPP code、註冊商鎖定、WHOIS 隱私與 TLD 同步
- 新增：新增 Hostinger 作為網域銷售整合，使用核心整合中的共用 Hostinger API token 進行可用性檢查、註冊、名稱伺服器更新、註冊商鎖定與 WHOIS 隱私
- 改進：新增網域生命週期動作與篩選器 docblock，提供開發者整合指引
- 改進：在 addon readme 中將 plugin 相容性中繼資料更新為 WordPress 7.0
- 改進：更新用於即將發布版本協調的規劃範本

版本 1.1.0 - 發布於 2026-05-08
- 新增：已為 ResellerClub、Enom 與 OpenSRS 註冊商實作 DNS 記錄建立（add_dns_record）
- 修正：預設 DNS Records 剖析器現在可容許 {DOMAIN} 與 {SITE_URL} token
- 修正：網域選擇 checkout 欄位 slug 已加上命名空間，以防止與 site_url 衝突

版本 1.0.8 - 發布於 2026-05-07
- 修正：ResellerClub 網域定價現在會從正確的 API endpoint 擷取即時成本價格

版本 1.0.7 - 發布於 2026-05-06
* 修正：ResellerClub test_connection 會傳送必要的 tlds 參數 (#224)

版本 1.0.6 - 發布於 2026-05-05
* 修正：ResellerClub 網域註冊現在可透過改進的 API 回應處理與由註冊局驅動的提供者路由正確運作
* 修正：Register Domain admin 頁面 UX 改進
* 移除：CyberPanel 註冊商整合

版本 1.0.5 - 發布於 2026-04-02
* 新增：GoDaddy 註冊商整合，用於網域註冊與管理
* 新增：NameSilo 註冊商整合
* 新增：ResellerClub 註冊商整合
* 新增：在網域購買與對應時自動驗證 SES 寄送網域
* 修正：在測試環境中防止 plugin 常數被重新定義
* 修正：install-wp-tests.sh 中的 MySQL 旗標正確進行字詞分割

* **修正：** 部分缺少的 css 資產

版本 1.0.2 - 發布於 2026-03-01
* **改進：** 從設定頁面移除全域加價設定 — 定價現在完全以每個商品為單位
* **改進：** 在設定頁面新增「管理網域商品」連結，方便快速導覽
* **改進：** 網域商品設定的欄位描述與工具提示更清楚（catch-all 與 TLD-specific、加價類型、 introductory pricing）
