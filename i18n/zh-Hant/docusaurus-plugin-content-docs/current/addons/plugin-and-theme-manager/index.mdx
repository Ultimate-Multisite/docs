# 外掛與主題管理員

Multisite Ultimate：外掛與主題管理員

您可以編輯或隱藏已安裝外掛和主題的元資料（包括標題、作者、縮圖和描述）。您還可以為外掛和主題建立分類，讓您的用戶能夠篩選它們，並為您的用戶面板新增一個精美的自訂外掛頁面。

若要使用此外掛，您只需前往網路管理員介面上的「外掛」或「主題」頁面，然後點擊「編輯外掛/主題資訊」即可。

## 安裝 {#installation}

1. 將 'ultimate-multisite-plugin-and-theme-manager' 上傳到 '/wp-content/plugins/' 目錄
2. 透過 WordPress 網路管理員的「外掛」選單啟用此外掛

## 使用指南 {#usage-guide}

Ultimate Multisite：外掛與主題管理員是一個功能強大的 **付費** Ultimate Multisite 附加元件，它讓您能完全控制您的網路用戶如何查看和與網路上的外掛和主題選項互動。

此附加元件具有三個主要功能：

  1. 它讓您可以完全更改外掛和主題的元資料。這包括名稱、描述、作者名稱，甚至縮圖！

  2. 它讓您可以對外掛和主題進行分類，方便您的用戶輕鬆地進行整理。

  3. 它提供了一個選項，讓您可以將舊的、單調的外掛頁面，替換成一個類似主題頁面、並可依分類進行排序的新頁面。

下載並安裝後，Ultimate Multisite 設定頁面會新增一個標籤頁——_Add-on Settings_。
