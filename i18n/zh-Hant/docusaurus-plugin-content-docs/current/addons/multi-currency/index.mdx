# 多幣度

Ultimate Multisite: Multi-Currency 讓您的網路商店能夠接受多種貨幣的付款。它能使用即時或手動的匯率自動轉換價格，讓您為任何產品設定特定貨幣的精確價格，並透過地理位置偵測每位訪客偏好的貨幣 — 所有功能都可以在單一的管理儀表板中管理。

## 主要功能

- **自動匯率轉換** — 價格會使用從您選擇的提供者獲取的匯率，從您的基礎貨幣進行轉換。
- **產品單價覆寫** — 您可以在任何產品上設定特定貨幣的精確價格，以繞過自動轉換。
- **貨幣選擇器註冊欄位** — 讓客戶在結帳/註冊時選擇他們偏好的貨幣。
- **多種匯率提供者** — 可選擇 ECB、Frankfurter、Stripe 或完全手動設定的匯率。
- **基於地理位置的貨幣偵測** — 自動預先選擇與訪客國家相符的貨幣。
- **管理儀表板** — 從一個地方管理活躍的貨幣、匯率和提供者設定。

## 要求

- Ultimate Multisite 2.0.0 或更高版本
- WordPress Multisite 安裝
- PHP 7.4 或更高版本

## 安裝

1. 將外掛檔案上傳到 `/wp-content/plugins/ultimate-multisite-multi-currency`。
2. 透過 WordPress 的 **Plugins** 畫面啟用外掛（網路啟用）。
3. 導航到 **Ultimate Multisite → Multi-Currency** 以開啟管理儀表板。
4. 新增您想要接受的貨幣，並設定您的匯率提供者。

## 設定

### 管理儀表板

Multi-Currency 管理儀表板（**Ultimate Multisite → Multi-Currency**）是以下功能的中心位置：

- 新增、啟用或停用貨幣。
- 選擇匯率提供者，並按需刷新匯率。
- 查看所有活躍貨幣的當前匯率。
- 當自動提供者不適用時，設定手動匯率。

### 匯率提供者

| 提供者 | 說明 |
|---|---|
| **ECB** | 歐洲中央銀行 — 適用於以歐元為基礎的網路的免費每日匯率。 |
| **Frankfurter** | 開源的 ECB 鏡像 API — 免費，無需 API key。 |
| **Stripe** | 使用 Stripe 的即時外匯匯率 — 需要啟用 Stripe 整合。 |
| **手動** | 您直接輸入匯率；無需外部 API 呼叫。 |

請在 **Ultimate Multisite → Multi-Currency → Settings → Exchange Rate Provider** 下選擇您的提供者。匯率可以隨時手動刷新，或設定自動更新。

### 產品單價覆寫

當自動轉換的價格不夠精確時，您可以為任何產品設定特定貨幣的精確價格：

1. 開啟產品編輯器（**Ultimate Multisite → Products → Edit**）。
2. 捲動到 **Currency Pricing** 區塊。
3. 為您想要覆寫的每種貨幣輸入精確價格。
4. 儲存產品。

沒有覆寫的產品將使用從您的基礎貨幣轉換而來的價格。

### 貨幣選擇器註冊欄位

您可以將貨幣選擇器欄位新增到您的註冊/結帳流程中，讓客戶在註冊時選擇他們偏好的貨幣：

1. 前往 **Ultimate Multisite → Multi-Currency → Settings**。
2. 啟用 **Show currency selector on signup**。
3. 可選地設定預設貨幣，或讓地理位置決定預先選定的值。

所選的貨幣會與訂閱資訊一起儲存，並用於所有後續的計費。

### 基於地理位置的貨幣偵測

當啟用地理位置偵測時，此附加元件會將訪客的國家映射到一種貨幣，並在貨幣選擇器欄位中預先選定它。若要啟用：

1. 前往 **Ultimate Multisite → Multi-Currency → Settings**。
2. 啟用 **Detect currency from visitor location**。
3. 在 **Geolocation Mapping** 表格中將國家映射到貨幣。

如果訪客的國家沒有映射，系統將使用網路的基礎貨幣作為備用選項。

## 常見問題

### 如果匯率提供者無法使用，會發生什麼事？

此附加元件會回退到上次成功獲取的匯率。如果從未獲取過匯率，則會回退到您設定的手動匯率。建議您設定手動匯率作為安全網。

### 我可以為不同的貨幣使用不同的提供者嗎？

不可以 — 所有貨幣目前只啟用一個提供者。如果您需要完全控制單個匯率，請使用 **Manual**。

### 地理位置偵測需要額外的外掛嗎？

不需要外部的地理位置外掛。此附加元件使用 WordPress 內建的 IP 到國家偵測功能。若需要更高的準確性，可以透過提供的 filter hooks 整合專用的地理位置服務。

### 產品單價覆寫是在稅前還是稅後計算的？

產品單價覆寫設定的是目標貨幣的稅前價格。稅金會使用您現有的稅務設定，在覆寫價格的基礎上計算。
