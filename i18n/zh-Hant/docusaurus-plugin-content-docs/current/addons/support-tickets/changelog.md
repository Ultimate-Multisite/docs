# 支援票務更新日誌

### 1.0.4 - 2026-05-05 {#104---2026-05-05}
* 改進：移除了 vendor/ 目錄的 Git 追蹤 (此目錄已由 .gitignore 排除)，從而減小了儲存庫大小
* 改進：測試支援到 WordPress 7.0

### 1.0.3 - 2026-05-01 {#103---2026-05-01}
* 修復：接受可空票務模型設定器中的 null 值
* 修復：將所有票務儲存在全域網路級別的資料表，以確保正確的多站點支援
* 修復：隱藏工作人員欄位，並修復客戶新票務表單上的空白頁面重定向問題
* 修復：將 undefined add_meta() 呼叫替換為正確的 update_meta() 以儲存元資料
* 修復：將未註冊的 capability 檢查替換為正確的 wu_view_all_support_tickets
* 修復：新增票務狀態、指派和快速編輯動作的缺失 AJAX 處理器
* 修復：修正票務檢視頁面中工作人員回覆偵測的方法名稱
* 修復：整合重複的回覆處理器，並統一 nonce 動作名稱
* 修復：新增 [wu_submit_ticket] shortcode 的缺失前端檢視頁面
* 修復：新增 user_id 欄位，並修復 Support_Ticket::get_user_id() 方法
* 修復：移除優先級篩選選項標籤中的多餘雙引號
* 修復：為超級管理員新增網路管理票務管理面板
* 改進：將管理員 CSS 整合到單一外部樣式表
* 改進：移除子站點管理選單中的「設定」子選單
* 改進：僅在支援票務頁面上條件性載入前端資源

### 1.0.2 - 2025-12-11 {#102---2025-12-11}
* 新增：管理員和前端的完整票務檢視功能
* 新增：票務回覆提交的 AJAX 處理器
* 新增：支援票務回覆功能，並包含正確的表單處理
* 新增：管理員在票務提交和回覆時的正確通知顯示
* 新增：為客戶自行提交的票務，自動關聯到當前使用者
* 新增：增強安全性，防止客戶覆蓋票務所有權
* 新增：缺失的輔助函數 (wu_format_date, wu_user_can_view_ticket 等)
* 新增：正確的文件附件下載和處理
* 新增：票務回覆和狀態變更的電子郵件通知系統
* 修復：票務回覆表單現在包含必要的票務 ID
* 修復：管理員票務檢視頁面中的通知可見性問題
* 修復：支援票務函數中的語法錯誤
* 改進：資料庫結構，為回覆和附件提供了正確的查詢和 Schema 類別
* 改進：將票務模型從使用屬性（attributes）遷移到使用真實屬性（real properties）
