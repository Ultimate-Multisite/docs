# Mailster

使用 Ultimate Multisite Mailster Integration addon，將您的 multisite 網路轉變為強大的電子郵件名單建立機器。當客戶註冊或完成購買時，系統會自動將他們新增到您的 Mailster 電子郵件名單中，從而為您的會員平台和電子郵件行銷策略建立無縫的連結。

## 主要功能

**自動訂閱者同步**
* 當客戶註冊時，自動將他們新增到 Mailster 名單
* 可選擇在註冊時立即同步，或等待付款確認後再同步
* 無需手動操作，讓您的電子郵件名單保持最新和活躍

**彈性名單管理**
* 為所有新客戶設定全域預設名單
* 可使用產品特定的名單覆蓋預設值，進行目標性行銷活動
* 可利用 Mailster 熟悉的介面輕鬆管理名單指派

**符合 GDPR 的選擇加入選項**
* 可選擇自動訂閱或基於勾選框的同意機制
* 啟用雙重選擇加入（double opt-in）以確認訂閱
* 根據您的合規要求，顯示或隱藏選擇加入勾選框

**智慧資料對應**
* 自動同步客戶姓名和聯絡資訊
* 將帳單地址對應到訂閱者欄位
* 追蹤註冊日期，用於目標性行銷活動

**產品級控制**
* 為不同的產品指派不同的名單
* 根據客戶購買的商品，建立目標性行銷活動
* 在需要時，可為特定產品停用 Mailster

**專業體驗**
* 內建 Mailster UI 介面，用於名單選擇
* 詳細的日誌記錄，用於故障排除和驗證
* 對現有訂閱者進行無縫更新

## 適用對象

* 正在建立電子郵件名單的會員網站擁有者
* 希望培育新客戶的 SaaS 平台
* 根據學員註冊情況進行分組的課程創作者
* 所有使用 Ultimate Multisite 並搭配 Mailster 的用戶

## 工作原理

1. 在您的 Ultimate Multisite 網路中安裝並啟用此 addon
2. 在 Ultimate Multisite 設定中配置全域預設名單
3. 可選擇使用產品特定的名單覆蓋預設值
4. 選擇您的選擇加入模式和訂閱時機
5. 觀看您的 Mailster 名單自動增長！

## 安裝

1. 將 plugin 檔案上傳到您的 Ultimate Multisite addons 目錄
2. 確保 Mailster 已在您的主站點安裝並啟用
3. 透過 Ultimate Multisite addons 頁面啟用此 addon
4. 導航至 Settings > Mailster Integration 進行配置
5. 選擇您的預設名單並配置選擇加入偏好

## 常見問題

## 這個 addon 是否可以在沒有 Ultimate Multisite 的情況下使用？

不，此 addon 需要 Ultimate Multisite 才能運作。它專門設計用於將 Mailster 與 Ultimate Multisite 的客戶和產品系統整合。

## 我是否需要在所有網站上都安裝 Mailster？

不，Mailster 只需要在您的主網路站點安裝並啟用即可。此 addon 將會自動從所有子站點連接到它。

## 我可以為不同的產品指派不同的名單嗎？

可以！您可以設定全域預設名單，然後在每個產品層級進行覆蓋。這讓您能夠根據客戶的購買內容，建立高度目標化的電子郵件行銷活動。

## 如果客戶已經存在於 Mailster 中會發生什麼？

預設情況下，此 addon 會更新現有的訂閱者，並將他們新增到新的名單中。如果您偏好跳過現有訂閱者，可以在設定中停用此功能。

## 這符合 GDPR 規範嗎？

是的！此 addon 包含基於勾選框的選擇加入模式，客戶必須明確同意接收電子郵件。結合適當的隱私權政策，這符合 GDPR 的要求。

## 我可以使用雙重選擇加入（double opt-in）嗎？

當然可以！啟用 double opt-in 設定後，Mailster 會在啟用訂閱前發送確認電子郵件。
