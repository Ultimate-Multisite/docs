# 過濾器：wu_cap_captcha_config

過濾 Cap Captcha 的設定。

讓開發人員可以客製化 Cap Captcha 的各種設定。

## 參數

| 名稱 | 類型 | 說明 |
|------|------|-------------|
| $config | `array` | Cap 的設定陣列。 |
| $level | `string` | 安全等級（fast、medium、max）。 |

### 適用版本

- 1.2.0
