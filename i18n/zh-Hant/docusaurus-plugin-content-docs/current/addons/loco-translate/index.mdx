# Loco Translate

Ultimate Multisite: Loco Translate 整合外掛為您的 WordPress 網站網路帶來了針對特定站點的翻譯功能。如果您曾需要網路中的不同站點對同一外掛或主題擁有不同的翻譯版本，那麼這個外掛就是完美的解決方案。

**此外掛的功能**

此外掛能讓 Ultimate Multisite 與流行的 Loco Translate 外掛完美結合，讓您的網路中的每個子站點都能維持其獨立的自訂翻譯。無論您是經營一個多區域平台、服務不同的語言社群，還是只是需要在整個網路中擁有翻譯彈性，此外掛都能讓您輕鬆實現。

**適用場景：**

* 具有不同本地化需求的跨區域企業
* 為多元社群提供服務的教育機構
* 提供多語言支援的 SaaS 平台
* 管理多個客戶網站的網頁代理商
* 任何需要翻譯彈性的多站點網路

## 安裝

1. 確保已安裝並啟用了 Ultimate Multisite
2. 安裝並啟用 Loco Translate 外掛
3. 將此外掛上傳到您的外掛目錄
4. 在網路層級啟用 Loco Translate 整合外掛
5. 即可透過 Loco Translate 在任何子站點開始建立特定站點的翻譯。

就是這麼簡單！整合功能會在後台自動運作，無需任何額外的設定。

## 常見問題

## 安裝後需要設定什麼嗎？

不需要！整合是自動的。只要外掛和 Loco Translate 都處於啟用狀態，子站點就能立即開始建立自己的自訂翻譯。

## 如果子站點沒有建立自訂翻譯會怎樣？

該站點會自動使用網路層級的翻譯作為備用方案。不會有任何錯誤，所有功能都能順暢運作。

## 子站點可以存取網路層級的翻譯嗎？

一般站點管理員只能看到和編輯自己站點的翻譯。超級管理員（Super admin）可以根據他們正在查看的站點，存取網路層級和特定站點的翻譯。

## 這適用於外掛和主題嗎？

是的！特定站點的翻譯適用於 WordPress 的外掛和主題。

## 這會拖慢我的多站點網路嗎？

不會。此外掛使用了高效的檔案載入機制，只有在需要時才會檢查特定站點的翻譯。因此，不會有明顯的效能影響。

## 我可以將網路翻譯複製到特定站點作為起點嗎？

可以！當您在 Loco Translate 中編輯翻譯時，您可以輕鬆地從網路翻譯開始，並為每個站點進行客製化。

## 這適用於 WooCommerce 和其他外掛嗎？

當然！任何使用標準 WordPress 翻譯功能的外掛或主題，都能與特定站點的翻譯功能協同工作。
