## 安裝步驟 {#installation}

1. 確保您的 WordPress 網路已安裝並啟用了 Ultimate Multisite plugin
2. 將附加元件檔案上傳到您的 `/wp-content/plugins/` 目錄
3. 透過 WordPress 的「Plugins」選單進行網路啟用
4. 在每個管理頁面享受隨機的查克·諾里斯笑話吧！

無需任何設定——只需啟用，然後微笑！

## 常見問題 {#frequently-asked-questions}

## 這需要 Ultimate Multisite 嗎？ {#does-this-require-ultimate-multisite}

雖然它設計為 Ultimate Multisite 附加元件，但該 plugin 具備獨立運作的能力，可以在任何 WordPress 安裝上運行。

## 我可以添加自己的笑話嗎？ {#can-i-add-my-own-facts}

可以！這些笑話是可翻譯的。您可以創建包含自定義笑話的 `.po` 檔案，並將其放置在 `/lang` 目錄中。或者，您也可以透過程式碼過濾（filter）笑話陣列。

## 這會讓我的管理介面變慢嗎？ {#does-this-slow-down-my-admin}

完全不會！這個 plugin 極為輕量，對效能沒有任何影響。它只顯示文字——沒有外部 API 呼叫或繁重的處理。

## 我可以在某些頁面禁用它嗎？ {#can-i-disable-it-on-certain-pages}

在可能造成干擾的頁面（例如 Block Editor）上，笑話會自動隱藏。您也可以使用 WordPress hooks 來自定義笑話出現的位置。

## 這些笑話適合所有觀眾嗎？ {#are-the-facts-appropriate-for-all-audiences}

是的！所有笑話都是乾淨、適合家庭的幽默內容，頌揚查克·諾里斯傳奇的地位。

## 我可以將笑話翻譯成其他語言嗎？ {#can-i-translate-the-facts-to-another-language}

當然可以！所有笑話都使用了 WordPress 的翻譯函數。請為您的語言創建一個翻譯檔案，並將其放置在 `/lang` 目錄中。文字域（Text domain）：`ultimate-multisite-chuck-norris-facts`

## 它與 multisite 協同工作嗎？ {#does-it-work-with-multisite}

是的！它專為 multisite 網路設計，並且可以進行網路啟用。
