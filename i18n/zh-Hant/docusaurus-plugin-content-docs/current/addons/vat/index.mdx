# VAT

Multisite Ultimate：歐洲增值稅 (VAT)

此插件為 Multisite Ultimate 的預設稅務處理功能增加了額外的功能和工具，以更好地支援歐洲的增值稅 (VAT)。
它會自動將歐盟的增值稅率新增到「稅率」頁面。

## 安裝

1. 將 'ultimate-multisite-vat' 上傳到 `/wp-content/plugins/` 目錄
2. 透過 WordPress 的「外掛」選單啟用此插件
3. 前往 Ultimate Multisite > 設定 > 稅務，並啟用 EU Vat。
4. 點擊「稅率」，然後在底部更新歐盟增值稅率，最後儲存稅率。
5. 完成！
