# 多租戶架構 (Multi-Tenancy)

為了提升效能和安全性，將子站點分散到不同的伺服器和資料庫中。

## 必要條件

- WordPress 5.3 或更高版本
- PHP 7.4 或更高版本
- Multisite Ultimate 外掛 (已啟用)

## 安裝

1. 將附加元件檔案上傳到您的 `/wp-content/plugins/` 目錄
2. 透過 WordPress 的「外掛」選單啟用外掛
3. 在網路管理介面中設定附加元件的選項

## 開發

### 設定

```bash
# 安裝依賴項
composer install
npm install

# 執行測試
npm run test

# 執行程式碼規範檢查
vendor/bin/phpcs
vendor/bin/phpstan

# 修復程式碼風格問題
vendor/bin/phpcbf

# 為生產環境建置
npm run build
```

### 專案結構

```
ultimate-multisite-addon-template/
├── inc/                          # PHP 類別和包含檔案
│   ├── class-multisite-ultimate-updater.php  # 更新處理器
│   └── class-multi_tenancy-main.php             # 主要功能
├── tests/                        # 單元測試
│   ├── bootstrap.php            # 測試啟動器
│   ├── class-*-base.php         # 基礎測試類別
│   └── class-*-test.php         # 測試案例
├── views/                       # 範本檔案
├── assets/                      # 靜態資源
│   ├── css/                     # 樣式表
│   ├── js/                      # JavaScript 檔案
│   └── images/                  # 圖片
├── lang/                        # 語言檔案
├── composer.json                # PHP 依賴項
├── package.json                 # Node.js 依賴項
├── phpunit.xml.dist             # PHPUnit 設定
├── .phpcs.xml.dist              # 程式碼規範設定
└── rector.php                   # 程式碼現代化規則
```

### 測試

此附加元件包含一套完整的測試框架：

- **單元測試 (Unit Tests)**：基於 PHPUnit 且包含 WordPress 測試套件
- **基礎測試類別 (Base Test Class)**：提供用於常見測試場景的輔助方法
- **測試覆蓋率 (Test Coverage)**：已設定追蹤程式碼覆蓋率
- **多站點測試 (Multisite Testing)**：測試會在多站點環境中運行

### 程式碼規範

- **WordPress 編碼規範 (WordPress Coding Standards)**：透過 PHPCS 執行強制檢查
- **PHP 7.4+ 相容性**：支援現代 PHP 功能
- **靜態分析 (Static Analysis)**：整合 PHPStan 進行型別檢查
- **程式碼現代化 (Code Modernization)**：使用 Rector 規則進行 PHP 升級

### 建置系統

- **資源處理 (Asset Processing)**：對 CSS/JS 檔案進行最小化 (Minification)
- **翻譯 (Translation)**：生成 POT 檔案
- **封裝 (Packaging)**：建立用於發佈的壓縮檔案
- **開發模式 (Development Mode)**：提供未最小化的資源，方便除錯

## 設定

此附加元件支援多種配置選項，您可以在網路管理介面中存取這些選項。

## Hooks 和 Filters

### Actions (動作)

- `ultimate-multisite-addon-template_init` - 在附加元件初始化後觸發
- `ultimate-multisite-addon-template_loaded` - 在所有附加元件檔案載入後觸發

### Filters (過濾器)

- `ultimate-multisite-addon-template_settings` - 修改附加元件設定
- `ultimate-multisite-addon-template_enabled` - 覆寫附加元件的啟用/停用狀態
