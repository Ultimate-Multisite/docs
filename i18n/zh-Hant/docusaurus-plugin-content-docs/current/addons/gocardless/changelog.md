# GoCardless 更新日誌 {#gocardless-changelog}

Version: 1.0.3 - 發布日期 2026-05-05
* 修復：當 GoCardless 會籍在待處理或暫停狀態下被取消時，系統會自動清理那些孤立的待處理網站。
* 改進：當 Bedrock 根自動載入器已載入依賴項時，插件自動載入器將會跳過執行。

Version: 1.0.1 - 發布日期 2025-09-28

* 將前綴名稱改為 ultimate-multisite；更新文字域；並提升版本號。

2021-09-24 - Version 1.0.0-beta.4

* 新增：為基於 mu-plugins 的設置，加入了 filter wp_ultimo_skip_network_active_check 過濾器；

2021-06-16 - Version 1.0.0-beta.3

* 修復：Webhook 處理器無法確認簽名時的問題；

2021-06-14 - Version 1.0.0-beta.2

* 改進：在建立 GoCardless 授權後，網站會自動發布；

Version 1.0.0-beta.1 - 初始版本
