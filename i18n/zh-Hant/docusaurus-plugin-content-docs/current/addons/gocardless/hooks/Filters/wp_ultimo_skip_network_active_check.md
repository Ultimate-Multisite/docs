# 篩選器：wp_ultimo_skip_network_active_check

允許開發人員跳過（short-circuit）此檢查。

當您使用基於 composer 的或其他客製化設定（例如 Bedrock）時，這會很有用，因為在這些環境中，使用 mu-plugins 是常見的做法。

- 2.0.0
