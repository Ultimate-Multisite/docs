# Ultimate Multisite 文件說明

歡迎來到 Ultimate Multisite 文件說明。Ultimate Multisite 將 WordPress Multisite 轉型為完整的網站即服務 (WaaS) 平台。

## 文件區塊

### [User Guide](/user-guide/getting-started/ultimate-multisite-101)

您需要了解所有設定、配置和管理 WaaS 平台的一切內容。內容涵蓋安裝、支付閘道、網域映射、客戶管理等等。

### [Developer Documentation](/developer/hooks)

這是為在 Ultimate Multisite 上開發的工程師提供的技術參考資料。內容包含 hooks 參考、REST API 文件，以及 addon 開發指南。

### [Addons](/addons)

這是關於付費 addon plugins 的文件說明，這些插件可以為 Ultimate Multisite 增加額外的功能，例如 WooCommerce 整合、分析功能、網域銷售等等。

## 尋求協助

- [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) -- 提出問題和分享想法
- [GitHub Issues](https://github.com/Ultimate-Multisite/ultimate-multisite/issues) -- 報告錯誤
- [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) -- 下載核心 plugin
