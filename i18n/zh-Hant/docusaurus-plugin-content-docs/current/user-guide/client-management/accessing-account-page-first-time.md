# 如何第一次登入我的帳戶頁面？

歡迎使用 Ultimate Multisite！

購買我們的 plugin 後，您會收到一封電子郵件，請透過郵件中的連結進入您的帳戶頁面：

<!-- Screenshot unavailable: Account page email with access link -->

使用您購買時填寫的電子郵件地址，以及郵件中提供的密碼登入。

<!-- Screenshot unavailable: Login page with email and password fields -->

由於這是您第一次登入此帳戶，系統會要求您變更密碼。請設定新密碼後點擊 _儲存_ 按鈕。

<!-- Screenshot unavailable: Change password form with Save button -->

在帳戶頁面中，您可以管理所有帳戶相關資訊，包括帳單、續訂、取消訂閱等各項設定。
