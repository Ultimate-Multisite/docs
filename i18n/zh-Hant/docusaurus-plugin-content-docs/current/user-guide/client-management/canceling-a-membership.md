# 取消會員資格（v2）

_**重要提醒：本文適用於 Ultimate Multisite 2.x 版本。**_

對您的客戶來說，取消會員資格的流程相當簡單。

如果您的用戶想要取消會員資格，他們應該先進入自己的 **帳戶** 頁面。

接著，前往 **危險區域**，然後點擊 **刪除帳戶**。

![帳戶頁面中危險區域的刪除帳戶按鈕](/img/admin/memberships-list.png)

系統會彈出一個視窗，要求用戶確認操作。用戶需要開啟 **確認刪除帳戶** 選項，然後點擊 **刪除帳戶**。

![確認刪除帳戶對話框](/img/admin/memberships-list.png)

這個操作會將用戶登出網站、取消他們在您網路上的會員資格，並從您的網路中刪除他們的網站。

![帳戶刪除後的登出畫面](/img/admin/memberships-list.png)

![會員資格已取消的確認訊息](/img/admin/memberships-list.png)

**重要：** 請注意，這個操作不會將客戶從您的網路中完全移除，因為一個客戶可以擁有多個會員資格，也可以沒有任何會員資格。

![取消後客戶仍存在於網路中](/img/admin/memberships-list.png) | ![沒有有效會員資格的客戶](/img/admin/memberships-list.png)  
---|---
