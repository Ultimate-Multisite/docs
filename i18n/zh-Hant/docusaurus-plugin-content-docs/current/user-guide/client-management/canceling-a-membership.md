# 取消會員資格 (v2) {#canceling-a-membership-v2}

_**重要注意事項：本文適用於 Ultimate Multisite 2.x 版本。**_

在你的客戶端取消會員資格相當容易。

如果你的使用者想取消他們的會員資格，他們應先存取自己的 ****Account**** 頁面。

在那裡，他們應點擊前往 **危險區域**，然後點擊 **刪除 Account**。

![含有刪除 Account 按鈕的 Account 頁面危險區域](/img/account-page/danger-zone.png)

一個模組會開啟並要求他們確認。他們應開啟 **確認刪除 Account** 選項，然後點擊 **刪除 Account**。

![確認刪除 Account 對話框](/img/account-page/delete-account-confirm.png)

這會讓他們登出網站，並取消他們在你網路中的會員資格，同時也會從你的網路中刪除他們的網站。

![刪除 Account 後已登出](/img/account-page/logged-out-after-delete.png)

![會員資格已取消確認](/img/account-page/membership-canceled.png)

**重要：**請注意，此動作不會將客戶從你的網路中排除，因為一位客戶可以擁有一個以上的會員資格，或沒有任何與其連結的會員資格。

![取消後客戶仍存在於網路中](/img/admin/customer-after-cancel.png)

![沒有有效會員資格的客戶](/img/admin/memberships-customer-empty.png)
