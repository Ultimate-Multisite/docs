# 如何取消我的訂閱？

如果您擁有 Ultimate Multisite 授權，您可以隨時取消續訂。只需按照以下步驟操作：


<!-- Screenshot unavailable: Account page email with access link -->

使用您購買時提供的電子郵件和密碼登入。

<!-- Screenshot unavailable: Login page with email and password fields -->

這是您的帳戶頁面初始畫面：

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

若要取消訂閱，請在左側選單中前往 _Renewing & Billing_。

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

點擊右側的箭頭開啟側邊視窗，然後選擇取消**自動續訂**的選項。

系統會顯示確認訊息。

如果您確定要取消訂閱，只需點擊 _**Cancel Renewals**_ 按鈕。完成此操作後，系統會請您填寫一份簡短的問卷調查。

完成！您的訂閱將不會自動續訂。

**在訂閱到期之前，您的授權金鑰仍然有效**。如果您之後想重新啟用訂閱，需要手動操作。
