# Ultimate Multisite Hooks 與 filters

Ultimate Multisite 由大量的函式和資料庫查詢所組成。它提供了 hooks 和 filters，讓你可以修改現有函式或新增自己的程式碼。

我們整理了這些 hooks 和 filters 的清單，作為進階使用者進一步自訂網站網路的參考資源。你可以點擊下方連結，從 Github 存取這份清單。
