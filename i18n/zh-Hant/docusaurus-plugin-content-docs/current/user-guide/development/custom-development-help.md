# 如何從 Codeable 獲得客製化開發協助？

有時候，我們真的需要一些很棒的客製化功能，讓網站網絡的流程完全符合我們的需求。如果這正是你所需要的，我們推薦 [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE)，這是一個能找到具備 Ultimate Multisite 專業知識的經驗豐富開發者的平台。

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) 是一個連結客戶與合格 WordPress 自由接案開發者的平台，能夠處理任何規模的專案。他們有嚴謹的申請流程，確保你能配對到特別適合你需求的專家。

### 如何在 Codeable 建立專案？

首先，前往 [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) 並點擊 **Start a Project** 按鈕。

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

接著，要提交專案，請點擊左側的 **Start a Project**。

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

在「I need help with」中，選擇 **Customizing**：

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

在「My」中，選擇 **Plugin**：

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

然後點擊 **Continue** 按鈕。

<!-- Screenshot unavailable: Codeable project form Continue button -->

現在，是時候盡可能詳細地描述你的專案和需求了。試著提供越多資訊越好！

<!-- Screenshot unavailable: Codeable project description form -->

回答以下問題以協助報價。

<!-- Screenshot unavailable: Codeable project pricing questions -->

之後，如果你還沒有 [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) 帳號，請先建立一個，然後點擊 **Publish My Project** 按鈕。

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

一旦你發布了 [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) 專案，平台會為你配對符合專案所需技能的開發者。

這個流程將幫助你找到適合你工作的人選，這些人都具備 WordPress 和 Ultimate Multisite 的經驗！

Codeable 的最低費率為每小時 70–120 美元。報價會根據工作範圍、複雜度和專案的急迫性而有所不同。

一旦你收到報價，就可以聘請最符合你需求的開發者，一起展開你的專案！
