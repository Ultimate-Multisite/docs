# 使用 Ultimate Multisite: Ad Injector 在你的網路上放置廣告

Ultimate Multisite: Ad Injector 是一個**免費附加元件**，讓你可以直接從方案編輯頁面在各個方案中放置廣告。

安裝 Ultimate Multisite: Ad Injector 後，前往你的 _Plugins_ 頁面啟用它。

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_點擊 Network Activate 按鈕_

附加元件啟用後，方案編輯畫面的 _Advanced Options_ 區塊會出現一個新選項，叫做 _Ad Settings_。

![方案編輯畫面上的 Ad Settings](/img/config/product-edit-full.png)

_現在你可以為每個方案編輯廣告設定了_

在這個分頁中，你可以為前台的文章和頁面新增廣告程式碼，放置在內容之前或之後。

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_插入你的廣告程式碼後，廣告就會顯示在該方案的文章和頁面上。_

你也可以在該方案使用者的管理頁面頁首加入廣告。

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_如果你想在使用者的後台顯示廣告，請前往 Admin Panel Ad Code 選項_

### 不只是廣告

你也可以使用 Ultimate Multisite: Ad Injector 為高階方案提供客戶支援服務。

只要前往 _Admin Panel Ads_ 選項，加入你的客服工具 HTML 程式碼即可。

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_為特定方案加入客戶支援功能_

這樣一來，使用者就可以直接從他們的管理面板聯繫你！
