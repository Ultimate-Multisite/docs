# 使用 PRO Theme 自訂你的管理面板

在接案工作中，我們經常需要為客戶打造專屬的 WordPress 管理後台體驗。如果你正在尋找更專業的外觀和操作感受，PRO Theme 或許能幫上忙。

PRO Theme 是一款管理介面主題，專門設計來協助你自訂客戶的 Dashboard。它同時也是 Ultimate Multisite 進階外掛核心的一部分。

安裝完成後，前往 _Plugins_ 頁面，點擊在你的網路上啟用它。

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_在 Plugins 頁面全網站啟用_

_**注意：**_ _如果你只是單純安裝而不在網路上啟用，它會在客戶端顯示為一個選項，讓使用者可以選擇是否只為自己套用這個主題。_

現在，你的管理介面已經完全煥然一新！左側欄位也會出現一個新的選單叫做 PRO Theme。
