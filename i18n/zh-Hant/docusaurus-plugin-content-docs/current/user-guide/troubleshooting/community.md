# 如何加入 Ultimate Multisite 社群

Ultimate Multisite 社群是一個讓你與其他 Ultimate Multisite 使用者交流、分享技巧和經驗的空間。要進入 Ultimate Multisite 社群，請前往 https://community.ultimatemultisite.com/home。

在歡迎頁面上，點擊「**在此建立帳號**」連結。

<!-- Screenshot unavailable: Community welcome page with creating your account link -->

你會被導向我們的註冊頁面。現在你需要輸入使用者名稱、電子郵件和密碼。完成 Captcha 驗證後，點擊 **Register** 按鈕。

<!-- Screenshot unavailable: Community register page with username, email, password fields and Captcha -->

註冊完成後，接下來要建立你的個人檔案。在這個頁面上，你可以上傳照片、輸入全名、撰寫簡短的自我介紹等等。此外，也可以新增社群媒體連結，並設定帳號的隱私權限。

<!-- Screenshot unavailable: Community profile creation page with photo upload and bio fields -->

現在你已經進入「Introductions」頻道，你可以在這裡自我介紹，也能認識其他成員。點擊 **New Post** 按鈕就可以發佈你的貼文。

<!-- Screenshot unavailable: Community Introductions channel with New Post button -->

Lounge 區塊和「Dev Logs」頻道對所有人開放。不過，如果要存取社群中的其他頻道，你需要取得特別權限。

<!-- Screenshot unavailable: Community channels showing Lounge section and Dev Logs channel -->

只要提供有效的 Ultimate Multisite 授權金鑰，我們就會授予你存取權限。請點擊任一私人頻道的連結，然後點擊 **Unlock** 按鈕。

<!-- Screenshot unavailable: Community private channel with Unlock button -->

你會被導向解鎖存取權限的頁面。輸入你的 Ultimate Multisite 授權金鑰，然後點擊 Unlock Access 按鈕：

<!-- Screenshot unavailable: Community unlock access page with license key input -->

你的授權金鑰經過驗證後，就可以存取私人頻道了。這個過程最多需要 24 小時，所以請點擊連結返回社群，在我們處理你的申請期間，歡迎你先探索其他公開空間。

<!-- Screenshot unavailable: Community license validation confirmation page -->
