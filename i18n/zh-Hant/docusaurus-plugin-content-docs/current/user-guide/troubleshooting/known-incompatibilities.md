# 與其他外掛的已知相容性問題

安裝並啟用 Ultimate Multisite 後，您可能會遇到與其他外掛之間尚未解決的相容性問題。

我們正積極努力在每次新版本發布時縮減這份清單，但如果某個外掛被列在這裡，您可能需要停用它，才能讓 Ultimate Multisite Network 完整運作。

以下是這些外掛的清單，以及您應該如何處理：

**部分 WPMU DEV 外掛** 某些 WPMU DEV _已停止維護_ 的外掛，已知會導致 Ultimate Multisite 的註冊流程在最後一步卡住（使用者無法建立帳號）。以下是會造成這些問題的外掛清單：

  * Signup Language：_Ultimate Multisite 有一個提供相同功能的附加元件，叫做 Ultimate Multisite: Language Switcher。_

  * Set Blog Description on Blog Creation：_這個功能已內建於 Ultimate Multisite，不需要額外的外掛。_

  * New Blog Templates：_這個功能已內建於 Ultimate Multisite，不需要額外的外掛。_

  * Set Password on Multisite Blog Creation：_這個功能已內建於 Ultimate Multisite，不需要額外的外掛。_

  * Signup TOS：_這個功能已內建於 Ultimate Multisite，不需要額外的外掛。_

  * Anti-Splog：雖然功能較為精簡，但這也_已內建於 Ultimate Multisite。_

**注意**：所有其他 WPMU Dev 外掛都可以與 Ultimate Multisite 正常搭配使用。例如 _Smush_、_Forminator_、_Defender_ 等。
