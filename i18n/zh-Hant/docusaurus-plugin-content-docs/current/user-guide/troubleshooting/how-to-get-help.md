# 如何取得協助

您已取得有效的授權金鑰，隨附的還有我們團隊的支援服務，協助您解決在設定 Ultimate Multisite 過程中可能遇到的問題。

本課程將介紹授權持有者可使用的支援管道，以及聯繫支援團隊時需要準備哪些資訊。

## 說明網站

您可以從我們的說明網站瀏覽各種文章和教學，連結如下。

[)

我們鼓勵使用者熟悉 Ultimate Multisite 的基本特色與功能。說明網站上有「新手入門」系列文章，常見問題的解答也能在那裡找到，例如如何安裝附加元件、如何整合付款閘道等。

## 聯繫支援團隊

如果您遇到困難，且無法在說明網站找到答案，可以透過以下方式聯繫我們的支援團隊：點擊網站右下角的對話泡泡圖示，或直接寄送電子郵件至 support@ultimatemultisite.com。

聯繫支援團隊反映技術問題時，請務必提供問題的詳細描述，並附上螢幕截圖、錯誤訊息，或任何有助於支援人員快速理解和評估問題的資訊。您也需要提供授權金鑰，僅限有效授權持有者才能獲得支援服務。

## 社群網站

我們的[社群網站](https://community.ultimatemultisite.com/)是 Ultimate Multisite 使用者彼此交流的園地。這是一個資訊平台，成員可以在此發文、參與討論、提供建議、互相合作或集思廣益。您需要提供 Ultimate Multisite 授權金鑰才能解鎖私人空間或頁面。
