# 管理客戶

**客戶**頁面顯示已在您的網路中註冊的會員列表。此頁面顯示基本的使用者資訊，包括姓名、電子郵件地址、最後登入時間，以及一個**切換至**連結，讓您能以網站管理員的身分登入該使用者的子網站。

![客戶列表頁面](/img/admin/customers-list.png)

點擊客戶卡片下方的**管理**，即可查看該客戶的詳細資訊。

在客戶詳細資訊頁面中，**總覽**區塊會顯示客戶 ID、最後登入時間，以及該客戶的消費總金額。

**客戶選項**區塊可讓您將此客戶設定為 **VIP**（一個方便日後搜尋的標籤）。**帳單資訊**分頁則顯示帳單相關資訊，如電子郵件、地址和國家。

**欄位**區塊會顯示客戶的使用者名稱、電子郵件，並可讓您變更**電子郵件驗證**狀態。

在客戶選項區塊下方，您還會看到四個額外的區塊：

- **會員方案** — 客戶目前訂閱的方案
- **付款紀錄** — 客戶的付款歷史
- **網站** — 此客戶擁有的網站
- **事件** — 與此客戶相關的事件（電子郵件驗證、帳單變更等）

點擊任何項目旁邊的箭頭圖示（→），即可前往其詳細資訊頁面。例如，點擊網站旁的箭頭會帶您前往網站編輯頁面，您可以在那裡修改會員方案關聯、網站類型和啟用狀態。

會員方案和付款紀錄也是同樣的操作方式——每個項目都連結至詳細資訊頁面，您可以在該頁面編輯會員方案狀態、關聯的客戶、產品、付款閘道設定，甚至處理退款。

在客戶頁面的右側，您會看到：

- **最後登入與 IP** — 最後登入時間、IP 位址和國家
- **時間戳記** — 客戶帳號的建立時間和最後修改時間
- **刪除** — 永久刪除客戶的選項

:::warning
刪除客戶是永久性的操作，無法復原。
:::
