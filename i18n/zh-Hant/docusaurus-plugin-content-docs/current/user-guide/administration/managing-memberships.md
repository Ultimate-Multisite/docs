# 管理會員資格

在 Ultimate Multisite 中，您可以管理會員資格並存取網路上的所有訂閱項目。您可以檢視和編輯訂閱詳細資訊，包括方案和產品、網站數量、帳單金額和頻率、付款記錄以及時間戳記。

若要存取此頁面，請前往 Ultimate Multisite 選單中的 **Memberships** 分頁。

![會員資格列表頁面](/img/admin/memberships-list.png)

在 Memberships 頁面中，您可以看到客戶列表以及與其帳戶相關聯的產品。您可以依狀態篩選會員資格：啟用中、待處理、暫停、已過期或已取消。

點擊任一會員資格，即可進入編輯頁面，修改會員資格的詳細資訊，例如帳單資訊、網站或訪客數量限制等。

您也可以為先前已在網路上註冊的使用者新增會員資格。點擊 **Add Membership** 並設定相關詳細資訊。

![新增會員資格彈出視窗](/img/admin/membership-add-modal.png)

接著點擊 **Create Membership**。新的會員資格將會顯示在會員資格列表中。
