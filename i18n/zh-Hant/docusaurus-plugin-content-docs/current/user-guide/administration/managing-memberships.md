# 管理會員資格 {#managing-memberships}

在 Ultimate Multisite 中，你可以管理會員資格，並存取你網路上的所有訂閱。檢視並編輯訂閱詳細資料，例如方案和產品、網站數量、帳單金額與頻率、付款紀錄，以及時間戳記。

若要存取此頁面，請前往 Ultimate Multisite 選單中的 **Memberships** 分頁。

![會員資格清單頁面](/img/admin/memberships-list.png)

在 Memberships 頁面上，你可以查看客戶清單，以及與其帳戶相關聯的產品。你可以依狀態篩選會員資格：啟用中、待處理、保留中、已過期或已取消。

免費會員資格會被視為**終身**會員資格。它們不會收到到期日期，也不會自動到期。如果免費方案應在試用或固定期間後結束，請使用含試用的付費產品、手動取消會員資格，或在其存取權限應變更時將客戶移至其他方案。

點擊某個會員資格，即可存取可編輯會員資格詳細資料的頁面——帳單資訊、網站或造訪限制，以及更多內容。

![會員資格編輯介面](/img/admin/membership-edit.png)

以下是會員資格編輯頁面的完整檢視：

![會員資格編輯完整介面](/img/admin/membership-edit-full.png)

你也可以新增與先前在你網路上註冊的使用者相關聯的新會員資格。點擊 **Add Membership** 並設定詳細資料。

![新增 Membership 對話框](/img/admin/membership-add-modal.png)

然後點擊 **Create Membership**。新的會員資格會出現在會員資格清單中。
