# 控制台總覽

Ultimate Multisite 在 Ultimate Multisite 選單下有專屬的**控制台**，同時也在網站網路控制台中新增了小工具。本文將介紹這兩個地方的功能。

## 網站網路控制台

Ultimate Multisite 會在網站網路層級的控制台中加入實用的小工具，位置在 **我的網站 → 網路管理員 → 控制台**。

![含有 Ultimate Multisite 小工具的網站網路控制台](/img/admin/network-dashboard-full.png)

以下是各個小工具的功能說明：

**初始設定：** 在全新安裝時顯示，提醒你完成設定任務，例如設定付款方式和建立測試帳號。

![初始設定小工具](/img/admin/network-dashboard-first-steps.png)

**摘要：** 顯示當日的註冊人數和收入。MRR（每月經常性收入）是根據有付費會員資格的客戶預估的總收入。

![摘要小工具](/img/admin/network-dashboard-summary.png)

**活動串流：** 顯示 Ultimate Multisite 的各種事件，包括註冊、取消、網站建立、刪除及其他活動。

![活動串流小工具](/img/admin/network-dashboard-activity-stream.png)

**即時狀態：** 顯示網路中使用者和網站的簡要摘要，並提供建立新網站或使用者的快捷方式。

![即時狀態小工具](/img/admin/network-dashboard-right-now.png)

**最新消息與討論：** 擷取並顯示 Ultimate Multisite 的最新消息，包括更新、重大錯誤和安全性修正。

![最新消息小工具](/img/admin/network-dashboard-news.png)

## Ultimate Multisite 控制台

網站網路控制台呈現的是整個網路的整體資訊，而 Ultimate Multisite 控制台（位於 Ultimate Multisite 選單的頂層）則呈現服務相關的資訊。

![Ultimate Multisite 控制台](/img/admin/um-dashboard-top.png)

**每月經常性收入成長：** 以 12 個月為週期，顯示每月的收入明細。新收入以藍色顯示，取消則以紅色顯示。

![MRR 成長圖表](/img/admin/um-dashboard-mrr.png)

**日期範圍支援：** 依時間區間篩選控制台資料，聚焦在你需要的資訊。日期範圍會影響下方小工具顯示的內容。

![日期範圍與控制台統計資料](/img/admin/um-dashboard-date-range.png)

控制台小工具包括：

- **收入** — 該期間的總收入，包含 MRR 和各產品的明細
- **新會員** — 在選定期間內建立的新會員數量
- **依國家區分的註冊** — 註冊的地理分布
- **依表單區分的註冊** — 透過各結帳表單註冊的客戶數量，以及由管理員手動新增的數量
- **最多人造訪的網站** — 你網路中最多人造訪的網站（需要在 **Ultimate Multisite → 設定 → 網站 → 啟用造訪限制與計數** 中啟用造訪計數功能）

![控制台統計資料](/img/admin/um-dashboard-stats.png)
