# 設定參考

本頁面追蹤影響 Ultimate Multisite 日常管理和近期行為變更的設定。

## 其他選項

**其他選項**區域出現在 **Ultimate Multisite > 設定 > 登入與註冊** 下。

| 設定 | 描述 |
|---|---|
| **Enable Jumper** | 在管理介面中顯示 Jumper 快速導航工具。您可以使用它直接跳轉到 Ultimate Multisite 畫面、網路物件和支援的管理目的地。如果您不希望看到此快捷方式，請停用此選項。 |

## 錯誤報告和遙測

先前用於選擇是否開啟的錯誤報告設定已從設定頁面移除。匿名遙測功能已停用，且不再有 UI 開關來啟用它。

如果您維護內部運行手冊或設定頁面的截圖，請移除所有提及舊版錯誤報告選用欄位的內容，以免管理員尋找一個已不存在的設定。

## 匯入/匯出設定

**匯入/匯出**設定分頁描述了它控制哪些設定，並直接連結到 **Ultimate Multisite > 站點匯出**，用於站點和網路備份。請使用此設定分頁進行匯入/匯出配置；若要處理單一站點的匯出/匯入流程，請使用 **工具 > 匯出與匯入**；當您需要完整的網路匯出備份時，請使用站點匯出工具。
