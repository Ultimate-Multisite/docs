# 管理方案的可分享連結

Ultimate Multisite 提供您直接與客戶分享方案連結的功能。若要使用此功能，請前往任一方案的編輯頁面，點擊「_點擊複製可分享連結_」按鈕。

![產品編輯頁面與可分享連結按鈕](/img/config/product-edit-full.png)

_在方案編輯頁面中，您可以找到建立可分享連結的按鈕_

請注意，此機制會使用預設的帳單週期來產生連結。舉例來說，如果您的預設帳單週期是每月，最後產生的連結會像這樣：[_**https://network.com/register/premium/**_](https://network.com/register/premium/)

如果您希望可分享連結預設為每季方案，需要在結尾加上 3

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

若要選擇 _premium_ 方案的年繳週期，請加上 1 再加上 year 這個字：

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

請注意，您無法用單一連結涵蓋不同的帳單週期。每種帳單週期都需要分享各自專屬的連結。

當您將連結分享給客戶後，連結會引導他們前往註冊流程的第一步。
