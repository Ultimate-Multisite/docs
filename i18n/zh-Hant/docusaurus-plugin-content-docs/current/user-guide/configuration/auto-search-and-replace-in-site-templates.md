# 網站模板自動搜尋與取代功能 (v2)

_**本教學適用於 WP UItimo 2.x 版本。**_

Ultimate Multisite 最強大的功能之一，就是能在註冊表單中新增任意的文字欄位、顏色選擇器和下拉選單。取得這些資料後，我們可以用來預先填入使用者所選網站模板的特定內容。當新網站發佈時，Ultimate Multisite 會自動將預留位置替換為註冊時填寫的實際資訊。

舉例來說，你可以在模板網站中加入預留位置。預留位置需要用雙大括號包圍，格式為 {{placeholder_name}}。

接著，只要新增對應的註冊欄位來收集這些資料即可。

你的客戶在註冊過程中就能填寫這些欄位。

Ultimate Multisite 會自動將預留位置替換為客戶提供的資料。

## **解決「模板充滿預留位置」的問題**

以上功能固然很棒，但我們會遇到一個麻煩：現在你的網站模板——客戶可能會瀏覽的頁面——充滿了看起來不太美觀、也讓人摸不著頭緒的預留位置。

為了解決這個問題，我們提供了為預留位置設定假值的選項。當客戶瀏覽模板網站時，系統會使用這些假值來搜尋並取代預留位置的內容。

你可以前往 **Ultimate Multisite > Settings > Sites**，然後點擊側邊欄的 Edit Placeholders 連結，即可進入模板預留位置編輯器。

![Sites 設定側邊欄中的 Edit Placeholders 連結](/img/config/settings-sites.png)

這會帶你進入預留位置內容編輯器，你可以在這裡新增預留位置及其對應的內容。

![預留位置內容編輯器，顯示預留位置名稱與數值](/img/config/settings-sites.png)
