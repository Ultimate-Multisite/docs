# 結帳表單

結帳表單是一種簡單又靈活的方式，讓你可以嘗試不同的做法來轉換新客戶。

Ultimate Multisite 2.0 提供結帳表單編輯器，讓你可以建立任意數量的表單，包含不同的欄位、提供的產品等。

要使用這個功能，請點選左側邊欄的「結帳表單」選單。

![結帳表單清單](/img/config/checkout-forms-list.png)

在這個頁面，你可以看到所有已建立的結帳表單。

如果想要建立新的表單，只要點選頁面上方的「新增結帳表單」即可。

你可以選擇以下三種選項作為起始點：單一步驟、多步驟或空白表單。然後，點選「前往編輯器」。

![結帳表單編輯器](/img/config/checkout-form-editor.png)

你也可以點選表單名稱下方的選項來編輯或複製現有的表單。在那裡，你還可以找到複製表單短代碼或刪除表單的選項。

![結帳表單懸停動作](/img/config/checkout-form-hover-actions.png)

### 編輯結帳表單

你可以為不同目的建立結帳表單。在這個範例中，我們將製作一個註冊表單。

進入結帳表單編輯器後，為你的表單命名（僅供內部參考使用）並設定一個 slug（例如用於建立短代碼）。

![結帳表單編輯器](/img/config/checkout-form-editor.png)

表單由步驟和欄位組成。你可以點選「新增結帳步驟」來加入新步驟。

![新增結帳步驟](/img/config/checkout-form-add-step.png)

在彈出視窗的第一個分頁，填寫表單步驟的內容。設定 ID、名稱和描述。這些項目主要供內部使用。

![結帳表單步驟](/img/config/checkout-form-step.png)

接下來，設定步驟的顯示條件。你可以選擇「總是顯示」、「僅對已登入使用者顯示」或「僅對訪客顯示」。

![結帳表單步驟](/img/config/checkout-form-step.png)

最後，設定步驟的樣式。這些是選填欄位。

![結帳表單步驟](/img/config/checkout-form-step.png)

現在，是時候在第一個步驟中新增欄位了。只要點選「新增欄位」，然後選擇你想要的區塊類型。

![含有欄位的結帳表單步驟](/img/config/checkout-form-step.png)

每個欄位都有不同的參數需要填寫。在這個第一個項目中，我們將選擇「使用者名稱」欄位。

![結帳表單步驟](/img/config/checkout-form-step.png)

![結帳表單步驟](/img/config/checkout-form-step.png)

![結帳表單步驟](/img/config/checkout-form-step.png)

你可以根據需要新增任意數量的步驟和欄位。要顯示產品讓客戶選擇，請使用「價格表」欄位。如果你想讓客戶選擇範本，請新增「範本選擇」欄位。依此類推。

_**注意：** 使用者名稱、電子郵件、密碼、網站標題、網站網址、訂單摘要、付款和送出按鈕是建立結帳表單的必填欄位。_

在製作結帳表單時，你可以隨時使用「預覽」按鈕來查看客戶會看到的表單樣貌。你也可以切換以現有使用者或訪客身分檢視。

![結帳表單儲存](/img/config/checkout-form-save.png)

![結帳表單編輯器](/img/config/checkout-form-editor.png)

最後，在「進階選項」中，你可以設定「感謝頁面」的訊息、新增追蹤轉換的程式碼片段、為結帳表單新增自訂 CSS，或將表單限制在特定國家顯示。

![進階選項](/img/config/checkout-form-advanced.png)

你也可以在右側欄位切換選項來手動啟用或停用結帳表單，或永久刪除表單。

![啟用切換](/img/config/checkout-form-active.png)

別忘了儲存你的結帳表單！

![儲存按鈕](/img/config/checkout-form-save.png)

要取得表單的短代碼，請點選「產生短代碼」，然後複製彈出視窗中顯示的結果。

![儲存按鈕與短代碼](/img/config/checkout-form-save.png)
