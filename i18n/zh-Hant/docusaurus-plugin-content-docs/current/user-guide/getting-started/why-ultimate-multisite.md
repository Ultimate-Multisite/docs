# 為什麼選擇 Ultimate Multisite

如果你找到了這篇文章，可能是聽說我們的 plugin 能幫助你實現商業構想。希望在閱讀完這篇文章後，你能找到所需的答案。

## 什麼是 Ultimate Multisite？

Ultimate Multisite 是一款 WordPress Multisite plugin，讓你能夠為客戶提供 WaaS（Website as a Service，網站即服務）。透過我們的 plugin，你可以設定多種訂閱方案，讓客戶支付定期費用，在你的 Multisite 網路中託管網站。

Ultimate Multisite 支援多種付款管道（例如 WooCommerce 或 Stripe），還能建立網站範本，讓新用戶註冊時省去大量麻煩。此外，你也可以對應網域、自訂註冊表單，為客戶打造個人化的體驗。

而這只是開始，你還可以透過 add-ons 和 webhooks 擴充更多功能。

## 其他功能

Ultimate Multisite 還提供：

  * 與 Managed WordPress 主機服務商整合的 Domain Mapping 功能：當新網域完成對應時，Ultimate Multisite 會自動將對應的網域加入 WP Engine、Cloudways、cPanel、RunCloud、Closte 等主機的管理面板，不需要手動處理。在某些情況下，我們甚至會自動為你取得 SSL 憑證。

  * Webhooks：Ultimate Multisite 擁有強大的 Webhook 系統，讓你的網路能與各種不同的服務整合。我們甚至提供與 Zapier 的直接整合，讓你的網路能連接平台上超過 1000 種應用程式和服務。

  * 強大的 Admin Pages，讓你用喜愛的頁面編輯器協助客戶：Ultimate Multisite 有一個實用的 add-on 叫做 Admin Pages Creator，讓你使用 Elementor、Beaver Builder 和 Brizy 範本，在客戶的 dashboard 中新增完全自訂的管理頁面。

  * 兩種不同的 Admin Themes，自訂客戶的 dashboard 體驗：請參考 Material WP 和 PRO。

還有更進階的功能：

  * 支援 Template Overrides：透過 Ultimate Multisite，你可以用自己的版本取代註冊流程的預設 HTML 範本，不必擔心修改 plugin 核心檔案。

  * Search and Replace API：你也可以在網站範本中加入像 {{site_name}} 這樣的預留位置，並使用 Search and Replace API，在用戶註冊後自動將這些預留位置替換為客戶提供的資訊。

## 使用情境

現在讓我們來看看 Ultimate Multisite 的幾種使用情境。

### 情境 1：設計公司

一般來說，設計公司的核心專長在於網站設計，而主機託管或行銷等服務則是附加項目。

對設計公司而言，Ultimate Multisite 的最大價值在於能夠在單一平台上託管和管理多個網站。如果公司的設計已標準化採用特定主題，例如 GeneratePress、Astra、OceanWP 等，還可以運用 Ultimate Multisite 的功能，自動為每個新網站啟用這些主題。

透過 Ultimate Multisite 的設定，你可以針對每個產品方案嚴格控管 plugins 和主題——決定要顯示或隱藏哪些項目，以及新網站建立時的啟用狀態。

有了 Ultimate Multisite，設計公司可以安心專注於他們最擅長的事——打造出色的網站。

### 情境 2：垂直領域服務商

有句老話說：「專注做好一件事。」對許多專業人士來說，這意味著圍繞單一核心概念來開發產品或服務。

如果你專注於發展特定領域的網站網路，可能會希望在共同的架構和平台上提供服務。這時網站範本就派上用場了！

網站範本是 Ultimate Multisite 的創新功能之一——範本網站已經安裝並啟用了主題、安裝並啟用了必要的 plugins，還建立了範例文章或頁面。當客戶基於範本建立新網站時，範本的內容和設定會自動複製到新網站上。

對於垂直領域的網站和服務供應商來說，這是無可比擬的優勢——能夠立即建立一個配備自訂 plugins 和設計、隨時可用的網站。客戶只需提供最少的資訊就能完成服務。

### 情境 3：WordPress 網站託管

託管 WordPress 網站的方式有很多種，但很少像單純提供預先安裝好 WordPress 的網頁空間那麼簡單。這是因為要提供有價值的服務，需要考量和整合許多面向。

Ultimate Multisite 在這個領域表現出色，為 WordPress 網站託管提供了完整的一站式解決方案。方案中包含訂閱服務、收款、結帳表單、折扣券和客戶通訊等核心機制。

正確安裝、設定和維護 WordPress Multisite 所需的大量整合工作，都由 Ultimate Multisite 代為處理，網站管理員只需要專注於與自身服務或領域相關的事項，例如產品方案、定價和服務內容。

對於想要與 Ultimate Multisite 整合的開發者，方案也提供了完整的 RESTful API 和用於事件通知的 Webhooks。

不需要依賴大量外部 plugins 和授權，Ultimate Multisite 就能提供功能豐富的解決方案，可與 Wix、Squarespace、[WordPress.com](https://WordPress.com) 等平台相媲美。
