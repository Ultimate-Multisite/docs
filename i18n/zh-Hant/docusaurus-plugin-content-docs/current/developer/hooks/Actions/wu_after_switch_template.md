# 動作：wu_after_switch_template

允許外掛開發者在使用者或超級管理員切換網站範本後，執行相關的函數動作。

## 參數 {#parameters}

| 名稱 | 類型 | 描述 |
|------|------|-------------|
| $id | `int` | 網站 ID |
