# 過濾器：wp_ultimo_redirect_url_after_signup

用於設定重新導向的 URL。

這是一個舊版過濾器。傳遞的某些參數可能不可用，例如 `$site_id`。

### 自 {#since}

- 1.1.3 版本起：允許開發人員過濾重新導向的 URL。
### 來源 {#source}

定義於 [`inc/checkout/class-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout.php#L2450) 的第 2450 行
