# Filter: wu_independent_billing_cycle_product_types {#filter-wuindependentbillingcycleproducttypes}

過濾具有獨立計費週期的產品類型。

具有獨立計費週期的產品，不會被迫與購物車中的方案計費週期相匹配。這對於網域註冊等產品特別有用，因為這些產品無論方案是按月還是按年計費，其計費週期總是固定的。

## 參數 {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | 包含獨立計費週期的產品類型陣列。 |

### 可用自 {#since}

- 2.5.0
