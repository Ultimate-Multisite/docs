# 篩選器：wu_view_override_replaceable_views

讓開發人員可以新增可替換清單中的資料夾。

請務必小心，因為新增資料夾可能會導致系統載入過時的副本，而不是 Ultimate Multisite 的版本。

## 參數

| 名稱 | 類型 | 描述 |
|------|------|-------------|
| $replaceable_views | `array` | 允許的資料夾清單。 |

### 自

- 2.0.0
### 來源
