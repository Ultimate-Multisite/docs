# 過濾器：wu_add_new_domain_modal_fields

用於過濾新增網域彈出視窗的欄位。

這讓外掛（例如 Domain Seller）可以修改或替換網域註冊的欄位。

## 參數 {#parameters}

| 名稱 | 類型 | 描述 |
|------|------|-------------|
| $fields | `array` | 表單欄位陣列。 |

### 可用版本 {#since}

- 2.1.0
