# 過濾器：wu_current_{$type}_get_manage_url {#filter-wucurrenttypegetmanageurl}

讓開發人員可以修改管理站點的 URL 參數。

## 參數 {#parameters}

| 名稱 | 類型 | 描述 |
|------|------|-------------|
| $manage_site_url | `string` | 管理站點的 URL。 |
| $id | `int` | 站點 ID。 |
| $site_hash | `string` | 站點雜湊值。 |

### 從版本 {#since}

- 2.0.9
### 來源 {#source}

定義於 [`inc/class-current.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-current.php#L214) 的第 214 行


## 回傳值 {#returns}
修改後的管理 URL。
