# Kòman Customize Panèl Admin ou ak PRO Theme la

Souvan lè nou ap fè travay pou kliyan yo, nou gen bezwen bay yo yon eksperyans espesyal nan dashboard WordPress la. Si ou ap chèche yon aparans ki pi pwofesyonèl, PRO Theme ka ede w.

PRO Theme se yon theme admin ki devlope pou ede w lè w ap personalize Dashboard kliyan ou yo. Li se tou pati nan kò plugin premium Ultimate Multisite la.

Mwen anpeche li epi ale sou paj _Plugins_ ou a epi klike pou aktive l sou rezo ou an.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Aktive l sou tout rezo ou sou paj Plugins_

_**NOT**:_ _Si ou jis mete l, li ap parèt nan kliyan ou yo kòm yon opsyon ki ka mete pou ajoute theme a sèlman pou sa itilizatè a._

Kounye a, ou gen entèfas admin ou totalman redizain! Ou gen yon nou menu sou kolòn mou nan rele PRO Theme.
