# Tradiks Ultimate Multisite

Ultimate Multisite se devlope an Angla, men plugin la totalman translatib ab, e nou ak swete kontribisyon tradiksyon.

## Tradiksyon Plugin Kòr (Core) {#core-plugin-translations}

Plugin kòr Ultimate Multisite a tradwi atravè platfòm tradiksyon ofisyèl **WordPress.org**:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

Sa se sistèm tradiksyon kominote WordPress la estanda. Lè tradiksyon yo rive nan yon nivo apwoblase ki sufis, yo otomatikman distribye a tout itilizat Ultimate Multisite atravè mizajou plugin WordPress.

**Nou bezwen èd ou!** Si ou pale yon lang lòt pase Angla, tanpri konsidere kontribisyon tradiksyon. Men tradwi kèk estrikti ede tou. Pou kòmanse:

1. Ale sou [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)
2. Chwazi lang ou vle
3. Klike sou "Contribute Translation" (Kontribye Tradiksyon)
4. Ou pral bezwen yon kont WordPress.org gratis

Si ou ta renmen vin yon editeur tradiksyon pou lang ou a (ki ka apwouve tradiksyon yo), kontakte nou atravè [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions).

## Tradiksyon Addon (Addon) {#addon-translations}

Plugin addon yo tradwi sepilman nan:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Tradiksyon addon yo kounye a genyen AI ki te jenere yo kòm yon kòmansman. Yo kouvri tout lang ki sipòte men yo ka gen fraz ki pa natirèl oswa enprécis. Si ou préférez créer des traductions localement pour votre propre projet plutôt que de contribuer en amont, il y a deux options :

### Utiliser Poedit {#local-translations}

Vous pouvez utiliser l'application de bureau **[Poedit](https://poedit.net)** pour traduire le fichier `.pot` :

1. Téléchargez Poedit depuis [poedit.net](https://poedit.net)
2. Ouvrez le fichier `.pot` situé dans `wp-content/plugins/ultimate-multisite/lang/`
3. Traduisez les chaînes dans l'interface de Poedit
4. Enregistrez -- Poedit génère des fichiers `.po` et `.mo`
5. Placez les fichiers de sortie dans le dossier `lang`

### Utiliser Loco Translate {#using-poedit}

Le plugin WordPress **[Loco Translate](https://wordpress.org/plugins/loco-translate/)** vous permet de traduire directement depuis le panneau d'administration sans avoir besoin d'outils de bureau. Il s'occupe de créer et de compiler les fichiers de traduction pour vous.
