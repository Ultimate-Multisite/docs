# VAT

Multisite Ultimate: VAT Ewopeyen

Ajoute fonksyonalite ak zouti anplis nan jesyon taks default Multisite Ultimate pou pi byen sipòte VAT Ewopeyen.
Ajoute otomatikman pousantaj VAT EU yo nan paj Pousantaj taks yo apati

## Enstalasyon {#installation}

1. Telechaje 'ultimate-multisite-vat' nan repètwa '/wp-content/plugins/' la
2. Aktive ekstansyon an atravè meni 'Ekstansyon' nan WordPress
3. Ale nan Ultimate Multisite > Paramèt > Taks epi aktive VAT EU.
4. Ale klike sou Pousantaj taks epi Mizajou Pousantaj VAT EU yo anba a epi Sove Pousantaj taks yo.
5. Fini!
