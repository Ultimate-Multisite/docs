# Action: move_site {#action-movesite}

Deklanche apre yo fin deplase yon sit nan yon nouvo rezo.

## Paramèt {#parameters}

| Non | Tip | Deskripsyon |
|------|------|-------------|
| $site_id | `int` | ID sit yo te deplase a. |
| $old_network_id | `int` | ID rezo orijinal la pou sit la. |
| $new_network_id | `int` | ID rezo yo te deplase sit la ladan an. |
