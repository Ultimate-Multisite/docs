# Material WP

Entèfas admin WordPress la rete menm jan depi lontan. Men èske ou janm mande tèt ou kijan yon vèsyon Material Design nan entèfas admin nan ta sanble? 

Material WP se yon tèm admin ki devlope pou ba ou aparans ak santiman sa a, pandan l ap ede ou bay kliyan ou yo yon Dashboard ki pi pèsonalize.

## Karakteristik

- Sipò CSS pèsonalize
- Paramèt estil fasil
- Reòganizasyon meni
- Enpòtasyon/ekspòtasyon fasil

## Dokimantasyon

**Pèsonalize Panèl Admin ou ak Material WP**

## Enstalasyon

1. Telechaje 'material-wp' nan anyè '/wp-content/plugins/' la
2. Aktive plugin nan atravè meni 'Plugins' nan WordPress
