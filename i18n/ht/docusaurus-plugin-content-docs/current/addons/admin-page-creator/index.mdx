# Kreyatè Paj Admin {#admin-page-creator}

## Kontni {#content}
Kreyatè paj admin ap pèmèt ou ajoute fasilman nouvo paj admin ak sou-paj nan dashboard admin WordPress la. Yo ka itilize li pou kreye yon paj akèy oswa yon paj leson patikilye ki ka sèvi kòm paj aterisaj pou nouvo itilizatè yo. Ou ka itilize yon page builder tankou Beaver Builder, Elementor, Brizy, ak Oxygen pou konsevwa paj yo san traka.

Li ofri tou sipò pou HTML, CSS ak JavaScript pèsonalize, ansanm ak kontwòl aksè ki baze sou plan ak wòl itilizatè!

## Karakteristik {#features}
* Sipòte HTML ak CSS Pèsonalize
* Sipò JavaScript
* Mòd Kontni PHP
* Pèmisyon ki baze sou wòl Itilizatè
* Sipò pou page builders

## Dokimantasyon {#documentation}
[Kreye Paj Admin ak Sou-paj Pèsonalize avèk Multisite Ultimate: Kreyatè Paj Admin](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## Enstalasyon {#installation}

1. Telechaje **ultimate-multisite-admin-page-creator** nan repètwa **/wp-content/plugins/** la
2. Aktive plugin nan atravè meni **Plugins** nan WordPress Network Admin ou a

## Gid Itilizasyon {#usage-guide}

Pandan WordPress rann li vrèman fasil pou ajoute kontni sou pati devan sit yo, kreye paj ak sou-paj sou bò admin nan mande kèk konpetans debaz nan kòd. Ultimate Multisite: Kreyatè Paj Admin vize pote menm fasilite itilizasyon pati devan an nan pati dèyè sit rezo ou yo.

## Poukisa itilize Kreyatè Paj Admin? {#why-use-admin-page-creator}

Petèt ou vle ofri yon paj pèsonalize sou panèl admin kliyan ou yo ak leson patikilye, petèt ou vle ajoute yon paj pèsonalize ak FAQ ak yon widget kontak. Posiblite yo san limit ak Kreyatè Paj Admin, e se sa nou pral eksplore nan leson patikilye jodi a.

_Kreye yon Paj Leson Patikilye ak yon playlist YouTube entegre posib kounye a!_

## Kreye premye Paj Admin Pèsonalize ou {#creating-your-first-custom-admin-page}

Apre ou fin enstale epi aktive Ultimate Multisite: Kreyatè Paj Admin, ale nan meni admin rezo ou a epi ale nan Paj Admin yo.

![Lis Paj Admin Pèsonalize sou Network Admin](/img/admin/admin-pages-list.png)

_Lis Paj Admin Pèsonalize sou Network Admin nan_

Ou pral wè yon lis tout paj admin ou te kreye jiskaprezan (kòm se premye vizit ou, lis la ap vid). Kontinye epi klike sou **Ajoute nouvo Paj Admin**. Y ap redirije ou nan editè Paj Admin nan.

## Opsyon Meni {#menu-options}

Sou bò a, nan meta-box Opsyon Jeneral la, w ap kapab konfigire kalite meni an, etikèt la, ak anpil lòt bagay.

### Kalite Meni {#menu-types}

Nouvo paj admin pèsonalize ou a ka gen de kalite meni diferan: meni nivo siperyè ak soumeni.

Yon paj meni nivo siperyè, jan non an sijere, ap ajoute paj ou fèk kreye a nan meni admin nan ansanm ak paj WordPress pa defo yo.

Si ou chwazi opsyon nivo siperyè a, w ap kapab chwazi lòd meni an ak yon ikòn meni tou.

Si ou vle fè paj sa a vin yon paj soumeni, w ap kapab chwazi yon paj paran pou atache paj sa a ladan. Ou ka ajoute paj pèsonalize ou a nan paj WordPress pa defo oswa nan paj admin pèsonalize nivo siperyè ou yo.

<!-- Kapti ekran pa disponib: Opsyon kalite meni Kreyatè Paj Admin ak paramèt etikèt, lòd, ak ikòn -->

_Pèsonalize nèt atik meni an pou Paj Pèsonalize sa a_

<!-- Kapti ekran pa disponib: Seleksyon paj paran soumeni Kreyatè Paj Admin -->

_Ou ka ajoute paj sa a tou kòm yon soumeni lòt paj meni, ki gen ladan pwòp paj pèsonalize nivo siperyè pa ou yo._

## Opsyon Avanse {#advanced-options}

Ou ta dwe kapab ajoute tout kalite kontni nan paj admin pèsonalize ou yo. Sa gen ladan règ ak fichye CSS, JavaScript, ak bibliyotèk ekstèn tou. Ou ka fè sa lè ou itilize Onglet Opsyon Avanse a.

### CSS Pèsonalize {#custom-css}

Sou onglet CSS la, ou ka ajoute pwòp kòd CSS pèsonalize ou, ansanm ak enpòte fichye CSS soti nan sit aleka (tankou CDN, Google Fonts, elatriye).

<!-- Kapti ekran pa disponib: Editè CSS pèsonalize Kreyatè Paj Admin ak enpòtasyon stylesheet ekstèn -->

_CSS pèsonalize sipòte tou. Ou ka enkli fichye estil ekstèn tou!_

### JavaScript Pèsonalize {#custom-javascript}

Menm bagay la valab pou JavaScript. Ou ka ajoute bibliyotèk ekstèn pou itilize nan kòd JavaScript pèsonalize ou.

<!-- Kapti ekran pa disponib: Editè JavaScript pèsonalize Kreyatè Paj Admin ak enpòtasyon bibliyotèk ekstèn -->

_Ajoute pwòp kòd JavaScript ou!_

## Pèmisyon {#permissions}

Tou nan Opsyon Avanse yo, w ap kapab chwazi ki sou-sit k ap gen aksè ak paj sa a. Sa itil si w ap planifye sèvi kontni eksklizif diferan bay diferan nivo plan, pa egzanp.

<!-- Kapti ekran pa disponib: Paramèt pèmisyon Kreyatè Paj Admin pou kontwòl aksè ki baze sou plan ak wòl -->

_Ou gen kontwòl total sou kote paj sa a pral parèt_

**Enpòtan** : Toude kondisyon yo dwe satisfè pou paj la parèt. Sa vle di si itilizatè a se yon kliyan yon plan espesifik, men li pa gen youn nan wòl yo chwazi yo (oswa okontrè), paj la p ap parèt sou panel li.

## Aktif {#active}

Si ou vle dezaktive yon paj admin pèsonalize san ou pa efase li oswa san ou pa manyen otorizasyon yo, jis dezaktive opsyon Aktif la.

Lè ou dezaktive yon paj nivo prensipal, sa ap dezaktive otomatikman tout sou-paj ki atache ak paj paran sa a.
