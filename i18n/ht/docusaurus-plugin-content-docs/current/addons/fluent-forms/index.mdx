# Fluent Forms {#fluent-forms}

Ekstansyon sa a fè li posib pou ajoute limit pou Fluent Forms sou Multisite Ultimate nan pwodwi, abònman manm oswa sit.
Fluent Forms sèvi ak tab pèsonalize olye de kalite pòs pèsonalize, kidonk opsyon limit nòmal yo pa mache ak Fluent Forms.
