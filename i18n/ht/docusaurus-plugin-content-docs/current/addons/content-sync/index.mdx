# Senkronizasyon Kontni {#content-sync}

Ultimate Multisite: Senkronizasyon Kontni

Addon sa a pou Ultimate Multisite pèmèt administratè rezo yo senkronize kontni soti nan nenpòt sit pou ale sou lòt sit nan rezo a. Li sipòte tout kalite pòs, tankou pòs, paj, kalite pòs pèsonalize, ak modèl Elementor. Addon an gen ladan yon entèfas asistan an plizyè etap pou chwazi kontni ak sit sib yo, avèk tretman an pakèt ak fidbak pwogrè an tan reyèl.

## Fonksyonalite {#features}

* Seleksyon Kontni: Griy kat vizyèl ak imaj miniati ak filtraj selon kalite pòs
* Tout Kalite Pòs: Senkronize pòs, paj, pwodwi, modèl Elementor, ak nenpòt kalite pòs pèsonalize
* Seleksyon Sit: Chwazi sou ki sit pou senkronize kontni an, avèk eksklizyon sit sous la ak pre-seleksyon sit pitit yo
* Tretman an Pakèt: Senkronize sou 10 sit alafwa avèk mizajou pwogrè
* Senkronizasyon Medya: Kopi epi remape otomatikman fichye medya yo (imaj, videyo) soti nan kontni pòs la pou ale sou sit sib yo
* Senkronizasyon Taksonomi: Konsève kategori, etikèt, ak taksonomi pèsonalize yo
* Sipò Elementor: Sipò konplè pou modèl Elementor avèk rejenerasyon CSS ak remapaj ID modèl yo
* Rezolisyon Konfli: Opsyon pou ranplase kontni ki deja egziste oswa sote li
* Jesyon Kach: Efase kach Elementor otomatikman sou sit sib yo apre senkronizasyon

## Kondisyon {#requirements}

* WordPress 5.3 oswa pi wo
* PHP 7.4 oswa pi wo
* Ultimate Multisite plugin (aktif)
* Elementor plugin (opsyonèl, obligatwa pou senkronizasyon modèl Elementor)

## Enstalasyon {#installation}

1. Telechaje 'ultimate-multisite-content-sync' nan repètwa '/wp-content/plugins/' la
2. Aktive plugin nan atravè meni 'Plugins' nan Admin Rezo WordPress ou a
3. Konfigire paramèt yo nan Ultimate Multisite → Paramèt → Senkronizasyon Kontni
4. Ale nan Ultimate Multisite → Senkronizasyon Kontni pou kòmanse senkronize
