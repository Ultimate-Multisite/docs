# Filtè: wu_magic_link_enforce_ip

Filtre si pou aplike verifikasyon adrès IP.

Mete sou false pou pèmèt tokens yo mache soti nan rezo diferan. Sa diminye sekirite men ogmante fasilite itilizasyon (pa egzanp, pou itilizatè mobil k ap chanje rezo).

## Paramèt {#parameters}

| Non | Tip | Deskripsyon |
|------|------|-------------|
| $enforce | `bool` | Si pou aplike korespondans adrès IP. |
