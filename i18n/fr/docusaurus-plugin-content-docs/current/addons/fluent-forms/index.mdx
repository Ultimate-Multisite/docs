# Fluent Forms

Cet add-on permet d'ajouter des limites aux Fluent Forms sur Multisite Ultimate dans les produits, les adhésions ou les sites.  
Fluent Forms utilise des tables personnalisées au lieu de types de publication personnalisés, donc les options de limitation normales ne fonctionnent pas avec Fluent Forms.

## Fonctionnalités

- Ajouter des limites pour Fluent Forms ou Conversational Fluent Forms  
- Les clients reçoivent un message convivial les incitant à mettre à niveau leur plan lorsqu'ils atteignent la limite
