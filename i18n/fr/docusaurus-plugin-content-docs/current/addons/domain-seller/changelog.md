### Version 2.0.0 - 2024-XX-XX

**Réécriture majeure pour Multisite Ultimate v2**

* **Nouveau :** Réécriture complète avec une architecture moderne PHP 7.4+
* **Nouveau :** Intégration transparente avec le système de paiement Multisite Ultimate v2
* **Nouveau :** Gestion des produits de domaine avec options de tarification flexibles
* **Nouveau :** Architecture de support multi-fournisseurs de domaine
* **Nouveau :** Intégration de l'auto-renouvellement et de l'abonnement
* **Nouveau :** Interface de gestion des domaines client
* **Nouveau :** Surveillance des domaines admin et journaux
* **Nouveau :** Support des coupons pour les produits de domaine
* **Nouveau :** Gestion complète des paramètres
* **Nouveau :** Base de code extensible conviviale pour les développeurs
* **Amélioré :** Fournisseur OpenSRS mis à jour avec prise en charge complète des fonctionnalités
* **Amélioré :** Interface utilisateur moderne cohérente avec Multisite Ultimate v2
* **Corrigé :** Tout le code v1 obsolète mis à jour aux normes v2
* **Supprimé :** Compatibilité héritée v1 (changement majeur)
