# Filtre : wu_domain_seller_search_delay

Filtre le délai de debounce de la recherche de domaine en millisecondes.

Augmentez cette valeur pour réduire les appels API sur les connexions lentes.

## Paramètres

| Nom | Type | Description |
|------|------|-------------|
| $delay | `int` | Délai de debounce en millisecondes. Par défaut : 500. |
