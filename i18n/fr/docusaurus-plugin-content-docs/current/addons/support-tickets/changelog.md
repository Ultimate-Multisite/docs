# Journal des tickets de support

### 1.0.2 - 2025-12-11
* Ajouté : Fonctionnalité complète de visualisation des tickets pour l'administrateur et le frontend
* Ajouté : Gestionnaire AJAX pour les soumissions de réponses aux tickets
* Ajouté : Prise en charge de la fonctionnalité de réponse aux tickets avec gestion correcte du formulaire
* Ajouté : Affichage correct des notifications pour les soumissions et réponses aux tickets dans l'administration
* Ajouté : Association automatique du ticket avec l'utilisateur actuel pour les soumissions autonomes des clients
* Ajouté : Amélioration de la sécurité pour empêcher les clients de modifier la propriété du ticket
* Ajouté : Fonctions d'assistance manquantes (wu_format_date, wu_user_can_view_ticket, etc.)
* Ajouté : Téléchargement et gestion corrects des pièces jointes de fichiers
* Ajouté : Système de notification par e‑mail pour les réponses aux tickets et les changements de statut
* Corrigé : Le formulaire de réponse aux tickets inclut désormais l'ID de ticket nécessaire
* Corrigé : Problèmes de visibilité des notifications dans les vues de tickets d'administration
* Corrigé : Erreurs de syntaxe dans les fonctions de tickets de support
* Amélioré : Structure de la base de données avec des classes de requêtes et de schémas appropriées pour les réponses et les pièces jointes
* Amélioré : Migration vers des propriétés réelles au lieu d'attributs pour les modèles de tickets
