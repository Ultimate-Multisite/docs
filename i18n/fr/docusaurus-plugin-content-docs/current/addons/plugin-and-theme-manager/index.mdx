# Gestionnaire d’extensions et de thèmes

Ultimate Multisite : Gestionnaire d’extensions et de thèmes

Modifiez ou masquez les informations méta (titre, auteur, miniature et description) des extensions et thèmes installés. Créez des catégories pour les extensions et les thèmes afin de permettre à vos utilisateurs de les filtrer et ajoutez une belle page Extensions personnalisée au panneau de vos utilisateurs.

Pour utiliser l’extension, vous pouvez simplement accéder à vos pages Extensions ou Thèmes dans l’administration réseau et cliquer sur « Modifier les infos de l’extension/du thème ».

## Installation {#installation}

1. Téléversez 'ultimate-multisite-plugin-and-theme-manager' dans le répertoire '/wp-content/plugins/'
2. Activez l’extension via le menu 'Extensions' dans votre administration réseau WordPress

## Guide d’utilisation {#usage-guide}

Ultimate Multisite : Gestionnaire d’extensions et de thèmes est un module complémentaire Ultimate Multisite **premium** puissant qui vous donne un contrôle total sur la manière dont les utilisateurs de votre réseau voient les options d’extensions et de thèmes sur votre réseau et interagissent avec elles.

Ce module complémentaire a trois objectifs :

  1. Il vous permet de modifier complètement les métadonnées des extensions comme des thèmes. Cela inclut le nom, la description, le nom de l’auteur et même les miniatures !

  2. Il vous permet de catégoriser les extensions et les thèmes, afin de permettre à vos utilisateurs de les trier facilement.

  3. Il vous donne la possibilité de remplacer l’ancienne page des extensions, ennuyeuse, par une nouvelle page semblable à celle des thèmes avec un tri par catégories.

Après l’avoir téléchargé et installé, un nouvel onglet - _Réglages du module complémentaire_ \- apparaîtra sur la page des réglages Ultimate Multisite.
