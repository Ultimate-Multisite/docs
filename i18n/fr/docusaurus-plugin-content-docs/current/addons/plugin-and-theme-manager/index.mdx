# Plugin & Theme Manager

Multisite Ultimate: Plugin and Theme Manager

Modifiez ou masquez les métadonnées (titre, auteur, vignette et description) des plugins et thèmes installés. Créez des catégories pour les plugins et thèmes afin de permettre à vos utilisateurs de les filtrer et ajoutez une page d'Plugins personnalisée et attrayante à votre panneau d'utilisateurs.

Pour utiliser le plugin, vous pouvez simplement naviguer vers vos pages Plugins ou Themes dans l'administration de votre réseau et cliquer sur "Edit Plugin/Theme info".

## Installation

1. Upload 'ultimate-multisite-plugin-and-theme-manager' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in your WordPress Network Admin

## Usage Guide

Ultimate Multisite: Plugin and Theme Manager est un add-on **premium** puissant d'Ultimate Multisite qui vous donne un contrôle total sur la façon dont vos utilisateurs du réseau voient et interagissent avec les options de plugin et de thème sur votre réseau.

This add-on serves three purposes:

  1. Il vous permet de modifier complètement les métadonnées des plugins et des thèmes. Cela inclut le nom, la description, le nom de l'auteur et même les vignettes !
  2. Il vous permet de catégoriser les plugins et les thèmes, afin que vos utilisateurs puissent les trier facilement.
  3. Il vous donne la possibilité de remplacer l'ancienne page de plugins ennuyeuse par une nouvelle page de type thème avec un tri par catégories.

After downloading and installing it, a new tab - _Add-on Settings_ \- apparaîtra sur la page des paramètres d'Ultimate Multisite.
