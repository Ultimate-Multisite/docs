# Synchronisation de contenu

Multisite Ultimate : Synchronisation de contenu

Ce module pour Multisite Ultimate permet aux administrateurs réseau de synchroniser le contenu de n'importe quel site vers d'autres sites du réseau. Il prend en charge tous les types de publication, y compris les articles, les pages, les types de publication personnalisés et les modèles Elementor. Le module comprend une interface d'assistant à plusieurs étapes pour sélectionner le contenu et les sites cibles, avec un traitement par lots et un retour d'information en temps réel.

## Fonctionnalités

* Sélection de contenu : grille de cartes visuelles avec vignettes et filtrage par type de publication
* Tous les types de publication : synchroniser les articles, les pages, les produits, les modèles Elementor et tout type de publication personnalisé
* Sélection de sites : choisir les sites vers lesquels synchroniser le contenu, avec exclusion du site source et pré-sélection des sites enfants
* Traitement par lots : synchronise vers 10 sites à la fois avec des mises à jour de progression
* Synchronisation des médias : copie et re-map automatiquement les fichiers multimédias (images, vidéos) du contenu des publications vers les sites cibles
* Synchronisation des taxonomies : préserve les catégories, les étiquettes et les taxonomies personnalisées
* Support Elementor : prise en charge complète des modèles Elementor avec régénération CSS et re-mappage des ID de modèle
* Résolution de conflits : option de remplacer le contenu existant ou de le sauter
* Gestion du cache : efface automatiquement le cache Elementor sur les sites cibles après la synchronisation

## Exigences

* WordPress 5.3 ou supérieur
* PHP 7.4 ou supérieur
* Extension Multisite Ultimate (active)
* Extension Elementor (facultatif, requise pour la synchronisation des modèles Elementor)

## Installation

1. Téléchargez 'ultimate-multisite-content-sync' dans le répertoire '/wp-content/plugins/'
2. Activez l'extension via le menu 'Extensions' dans l'admin réseau WordPress
3. Configurez les paramètres dans Ultimate Multisite → Paramètres → Synchronisation de contenu
4. Accédez à Ultimate Multisite → Synchronisation de contenu pour commencer la synchronisation
