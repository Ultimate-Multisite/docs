# Action: switch_network

Se déclenche lorsque le contexte réseau actuel est changé.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $new_network_id | `int` | ID du réseau vers lequel on passe. |
| $old_network_id | `int` | ID du réseau précédemment actif. |

- 1.3.0
