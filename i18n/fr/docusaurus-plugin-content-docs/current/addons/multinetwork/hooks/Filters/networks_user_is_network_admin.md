# Filtre : networks_user_is_network_admin

Filtre les réseaux dont l'utilisateur est l'administrateur.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $ | `array\|bool` | Liste des ID de réseaux ou false si aucun réseau pour l'utilisateur. |
| $ | `int` | ID de l'utilisateur pour lequel les réseaux doivent être retournés. |
