# Filtre: wu_apc_should_display_admin_menu

Permet aux administrateurs de masquer les pages du menu d'administration, s'ils ont terminé le processus de personnalisation.

## Paramètres

| Nom | Type | Description |
|------|------|-------------|
| $should_display_admin_menu | `bool` | Si nous devons afficher les pages. |
| $network_active | `bool` | Vérifier si le plugin est actif en tant qu'administrateur réseau. |
