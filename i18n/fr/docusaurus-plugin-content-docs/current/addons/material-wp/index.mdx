# Material WP {#material-wp}

L’interface d’administration WordPress est restée la même depuis des lustres. Mais vous êtes-vous déjà demandé à quoi ressemblerait une version Material Design de l’interface d’administration ? 

Material WP est un thème d’administration développé pour vous offrir cette apparence et cette expérience, en vous aidant à fournir à vos clients un Dashboard plus personnalisé.

## Fonctionnalités {#features}

- Prise en charge du CSS personnalisé
- Réglages de style simples
- Réorganisation du menu
- Importation/exportation faciles

## Documentation {#documentation}

**Personnaliser votre panneau d’administration avec Material WP**

## Installation {#installation}

1. Téléversez 'material-wp' dans le répertoire '/wp-content/plugins/'
2. Activez l’extension via le menu 'Extensions' dans WordPress

## Guide d’utilisation {#usage-guide}

L’interface d’administration WordPress est restée la même depuis des lustres. Mais vous êtes-vous déjà demandé à quoi ressemblerait une version Material Design de l’interface d’administration ?

Material WP est un thème d’administration développé pour vous offrir cette apparence et cette sensation, en vous aidant à fournir à vos clients un Dashboard plus personnalisé. Material WP fait également partie du noyau d’extensions premium d’Ultimate Multisite.

Après **l’avoir installé**, accédez à votre page _Extensions_ et cliquez pour l’activer sur votre réseau.
