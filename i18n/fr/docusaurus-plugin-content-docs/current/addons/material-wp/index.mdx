# Material WP

L'interface d'administration WordPress est restée la même depuis longtemps. Mais vous êtes-vous déjà demandé à quoi ressemblerait une version de l'interface d'administration en Material Design ?

Material WP est un thème d'administration développé pour vous offrir cette apparence et cette sensation, vous aidant à fournir à vos clients un tableau de bord plus personnalisé.

## Fonctionnalités

- Support CSS personnalisé
- Paramètres de style faciles
- Réorganisation du menu
- Import/Export faciles

## Documentation

**Personnalisation de votre panneau d'administration avec Material WP**

## Installation

1. Téléchargez 'material-wp' dans le répertoire '/wp-content/plugins/'
2. Activez le plugin via le menu 'Plugins' dans WordPress

## Guide d'utilisation

L'interface d'administration WordPress est restée la même depuis longtemps. Mais vous êtes-vous déjà demandé à quoi ressemblerait une version de l'interface d'administration en Material Design ?

Material WP est un thème d'administration développé pour vous offrir cette apparence et cette sensation, vous aidant à fournir à vos clients un tableau de bord plus personnalisé. Material WP fait également partie du cœur des plugins premium d'Ultimate Multisite.

**Après l'installer**, allez sur votre page _Plugins_ et cliquez pour l'activer sur votre réseau.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-WgljuQxy9a.png)

_Activez-le à l'échelle du réseau sur votre page Plugins_

_**NOTE :**_ _Si vous l'activez simplement, il apparaîtra chez votre client comme une option pouvant être cochée pour ajouter le thème uniquement à cet utilisateur._

Maintenant, votre interface d'administration est complètement repensée ! Vous avez également un nouveau menu dans la colonne de gauche appelé Material WP

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-l9ivnUahyp.png)

_Voici l'apparence Material Design sur votre installation WP_
