# Site Exporter

## Contenu

Nous comprenons le besoin de pouvoir déplacer des sous-sites au sein de votre réseau vers une installation mono-site et vice versa si votre client décide de migrer son site web. L'addon Site Exporter et Importer facilite cette tâche. Cet outil exporte et importe des sites à l'aide de fichiers zip qui peuvent également servir de sauvegarde.

## Fonctionnalités

* Exporter et importer des sites
* Interface conviviale
* Créer une sauvegarde du site

## Documentation

**Comment exporter et importer un site**

## Installation

1. Téléversez **ultimate-multisite-site-exporter** dans le répertoire **/wp-content/plugins/**
2. Activez l'extension via le menu **Plugins** dans WordPress
3. Suivez l'assistant pas à pas pour configurer l'extension

## Comment exporter un site

Pour exporter un site depuis votre réseau, allez sur **Export & Import** dans le menu Ultimate Multisite et, dans l'onglet **Export Sites**, cliquez sur **Generate new Site Export**.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-_bBztobh1xIFoj0Zo3zo7bx842u3XFMgpc2b8kiHmGmBU6RjuxlSOSY4ZNzaWTPOJrb0TIW_Q0Dv9WvT0yD-WwOkZ7bR36Ij5aJa_l19cDTfuj9DADQqI5dGy3eIeUmoQA=s0)

Recherchez le site que vous souhaitez exporter et sélectionnez les éléments à inclure dans le fichier exporté. Vous pouvez inclure thèmes, extensions et téléchargements. Si votre site est trop volumineux, vous pouvez choisir d'exécuter le processus d'export en arrière-plan.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-fk-i4RP0uZFj8kgfcQeRfAqWtCBelOjpOQARFfeBisVkeRSc1Qywq6Dpy66MnB6oBbnvdfJM359LdRSgqCLU1tNozo1u1OFH5svxtld8TZAXHY-CPGh92gWz9HQ8nlUcjg=s0)

Cliquez sur le bouton **Export Site**. Votre site exporté sera disponible dans la liste **Existing Exports**. Cliquez sur le bouton **Download ZIP** pour télécharger le fichier.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-mouH16ZCUuNaUg0PwF2JmMhhsKPJOztau21QIeB7DXqAcX4VI5FL_itxvnKmFo439xuGUWsntM3doG6yBkjS2gi6bNTjg-kLHC19m19br5xsgBl0XH8hy5LvuPN1uNtx9g=s0)

## Comment importer un site

Pour importer un site, allez dans l'onglet **Import a Site** et téléversez le fichier .zip du site que vous souhaitez importer. Vous pouvez modifier l'URL vers une nouvelle ou conserver l'ancienne en laissant le champ vide. Vous pouvez également cocher la case pour supprimer le fichier .zip de votre installation après l'import. Cliquez ensuite sur **Begin Import**.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-Uq6wNJ4TPTuAtSJqKunxSJEn5cli7vbwYGgL1G3fcHwdvU7gSwW_J4xlBAtZ1fyVbDU7Pupi8uNFhw1Lzm4S2fjVTkbQoz7fiZPIc1qiy8qj3iGt-MNVlyLWmIZXFxPLYA=s0)

Le processus peut prendre un peu de temps. Vous verrez un import en attente dans la liste sous l'importateur.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-kMkwSY3bYrE5vjfJtLviYImhNBn_PwmuamaRDlQKS9rGwIy3a_jXV0glMNTU8fBmV6IKkNN7obXNQFqNaAJtAAbWDftzX76-qdCDKCrqdru_9taciFCayE9FNszZJTODvA=s0)

Vous pouvez arrêter le processus en cliquant sur **Cancel Import**.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-oGmtCq7br6Y9GAfw_Ch53TP8nkJ0hw77sQJpSCQ7fEj5qk2lJo3WuMl7NYqB9c8SmORis53bWHILwGLxUTbpv4trWhxibu-yu5DNu6yNWXfkRo0bcnAZJ5F2K1hYIMH6PQ=s0)

Une fois le processus terminé, votre site est déjà disponible sur votre réseau.
