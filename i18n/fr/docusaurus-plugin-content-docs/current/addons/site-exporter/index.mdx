# Site Exporter

## Contenu

Nous comprenons le besoin de pouvoir déplacer des sous-sites au sein de votre réseau vers une installation mono-site et vice versa si votre client décide de migrer son site web. L'addon Site Exporter et Importer facilite cette tâche. Cet outil exporte et importe des sites à l'aide de fichiers zip qui peuvent également servir de sauvegarde.

## Fonctionnalités

* Exporter et importer des sites
* Interface conviviale
* Créer une sauvegarde du site

## Documentation

**Comment exporter et importer un site**

## Installation

1. Téléversez **ultimate-multisite-site-exporter** dans le répertoire **/wp-content/plugins/**
2. Activez l'extension via le menu **Plugins** dans WordPress
3. Suivez l'assistant pas à pas pour configurer l'extension

## Guide d'utilisation

Avec Ultimate Multisite, vous pouvez exporter un site depuis votre réseau ou importer un site vers votre installation multisite ou mono-site.

Pour démarrer le processus, allez sur notre page d'add-ons et installez l'addon Ultimate Multisite: Site Exporter & Importer.!

To start the process, go to our add-ons page and install the Ultimate Multisite: Site Exporter & Importer add-on.![Add-ons page](/img/admin/addons-page.png)

![Add-ons page with install option](/img/admin/addons-page.png)

Allez sur votre page Extensions et cliquez sur **Network Activate** l'addon que vous venez d'installer.!

Go to your Plugins page and click to **Network Activate** the add-on you have just installed.<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Site Exporter addon -->

L'option **Export & Import** se trouve dans le menu Ultimate Multisite, à gauche.

## Comment exporter un site

Pour exporter un site depuis votre réseau, allez sur **Export & Import** dans le menu Ultimate Multisite et, dans l'onglet **Export Sites**, cliquez sur **Generate new Site Export**.

<!-- Screenshot unavailable: Export Sites tab with Generate new Site Export button -->

Recherchez le site que vous souhaitez exporter et sélectionnez les éléments à inclure dans le fichier exporté. Vous pouvez inclure thèmes, extensions et téléchargements. Si votre site est trop volumineux, vous pouvez choisir d'exécuter le processus d'export en arrière-plan.

Search for the site you want to export and select the items you want to include in the exported file. You can include themes, plugins, and uploads. If your site is too big, you can choose to run the export process in the background.<!-- Screenshot unavailable: Site export options with site search, include themes/plugins/uploads checkboxes -->

Cliquez sur le bouton **Export Site**. Votre site exporté sera disponible dans la liste **Existing Exports**. Cliquez sur le bouton **Download ZIP** pour télécharger le fichier.

Click the **Export Site** button. Your exported site will be available on the **Existing Exports** list. Click the **Download ZIP** button to download the file.<!-- Screenshot unavailable: Existing Exports list with Download ZIP button -->

## Comment importer un site

Pour importer un site, allez dans l'onglet **Import a Site** et téléversez le fichier .zip du site que vous souhaitez importer. Vous pouvez modifier l'URL vers une nouvelle ou conserver l'ancienne en laissant le champ vide. Vous pouvez également cocher la case pour supprimer le fichier .zip de votre installation après l'import. Cliquez ensuite sur **Begin Import**.

<!-- Screenshot unavailable: Import a Site tab with zip upload field and Begin Import button -->

Le processus peut prendre un peu de temps. Vous verrez un import en attente dans la liste sous l'importateur.

<!-- Screenshot unavailable: Pending import in the list below the importer -->

Vous pouvez arrêter le processus en cliquant sur **Cancel Import**.

<!-- Screenshot unavailable: Cancel Import button to stop the import process -->

Une fois le processus terminé, votre site est déjà disponible sur votre réseau.

## Comment importer un site sur une installation unique

Pour importer un site sur votre installation unique, installez d'abord et activez l'importateur sur votre site. Vous pouvez trouver le lien de téléchargement dans l'onglet exporter.

<!-- Screenshot unavailable: Exporter tab with link to download single-site importer -->

Vous pouvez trouver l'importateur dans le menu **Tools**, à gauche, sous l'élément **Export & Import**.

<!-- Screenshot unavailable: Single-site Tools menu showing Export and Import submenu -->

Téléversez le fichier .zip du site que vous souhaitez importer. Vous pouvez modifier l'URL vers une nouvelle ou conserver l'ancienne en laissant le champ vide. Vous pouvez également cocher la case pour supprimer le fichier .zip de votre installation après l'import. Cliquez ensuite sur **Begin Import**.

<!-- Screenshot unavailable: Single-site import form with zip upload and Begin Import button -->

Le processus peut prendre un peu de temps. Vous verrez un import en attente dans la liste sous l'importateur.

<!-- Screenshot unavailable: Single-site pending import in the list -->

Vous pouvez arrêter le processus en cliquant sur **Cancel Import**.

<!-- Screenshot unavailable: Cancel Import button to stop the import process -->

Une fois le processus terminé, votre site sera remplacé par le site que vous venez d'importer.
