# Ultimate Multisite Modules complémentaires

Addons extend Ultimate Multisite with additional features and integrations. Most addons are paid and can be purchased from [ultimatemultisite.com](https://ultimatemultisite.com).

## Installation

Les modules complémentaires peuvent être installés de deux façons :

1. **Via la page des modules complémentaires** -- Accédez à la page de gestion des modules complémentaires de votre site dans l'administration WordPress
2. **Téléchargement manuel** -- Téléchargez depuis [votre compte](https://ultimatemultisite.com/my-account/downloads/) et téléversez via Plugins > Ajouter
