# AffiliateWP

Multisite Ultimate: AffiliateWP Integration

Utilisez la puissante AffiliateWP pour développer la base de clients de votre réseau Ultimo !

## Installation

1. Téléchargez 'ultimate-multisite-git ' dans le répertoire '/wp-content/plugins/'  
2. Activez le plugin via le menu 'Plugins' dans WordPress  
3. Suivez l'assistant pas à pas pour configurer le plugin  

## Usage Guide

AffiliateWP est devenu la solution de référence pour créer un programme d'affiliation pour votre entreprise en ligne.  
Ultimate Multisite: AffiliateWP est le pont qui vous permettra de créer la même chose pour votre réseau premium.

## Getting Started

Tout d'abord, vous **devez avoir** [**AffiliateWP**](https://affiliatewp.com/) **actif** ! Vous ne pourrez pas utiliser notre add-on si vous n'avez pas **le plugin qui s'intègre avec lui**. Si vous ne prévoyez pas de laisser vos clients l'utiliser sur leurs sites également, AffiliateWP doit être actif uniquement sur votre site principal.  

De plus, vous aurez besoin que **Ultimate Multisite** et **Ultimate Multisite: AffiliateWP** soient **activés** **à l'échelle du réseau**.

## Adding Ultimate Multisite as an AffiliateWP Integration

Sur votre site principal, accédez aux Paramètres des Affiliés, et dans l'onglet Intégrations, cochez Ultimate Multisite. Cela indiquera à [AffiliateWP](https://affiliatewp.com/) de commencer à suivre les visites et les conversions provenant du flux d'inscription à Ultimate Multisite.

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-602126536867724dfc6f0a54-zkWh3TFms-1F9087E6-928C-4043-94B9-F85966AF05DA.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-602126536867724dfc6f0a54-zkWh3TFms-1F9087E6-928C-4043-94B9-F85966AF05DA.png)

_Cochez la case Ultimate Multisite en bas et enregistrez_

C'est à peu près tout ce qu'il faut. Les visites provenant de liens de parrainage seront suivies et lorsque cet utilisateur effectuera son premier paiement, la conversion sera créée et une commission sera mise de côté pour l'affilié responsable de la conversion.  

Seul le premier paiement reçu du nouvel utilisateur sera suivi, cependant. Pour activer les paiements de parrainage récurrents, vous aurez besoin du [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/).

## Enabling Recurring Referrals

Après avoir installé le [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/), accédez à l'onglet Paramètres des Affiliés Références Récurrentes et activez l'utilisation des références récurrentes.

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-602126536867724dfc6f0a54-bmSN5h7Lk-4697DC4E-ABB9-4C34-BFD8-97E4B62C261C.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-602126536867724dfc6f0a54-bmSN5h7Lk-4697DC4E-ABB9-4C34-BFD8-97E4B62C261C.png)

_Simple comme ça : activez les Références Récurrentes et vous êtes prêt !_

Encore une fois, c'est tout ce qu'il faut !

Maintenant, chaque fois qu'un nouveau paiement est reçu sur un compte référé, l'affilié responsable recevra automatiquement sa part.
