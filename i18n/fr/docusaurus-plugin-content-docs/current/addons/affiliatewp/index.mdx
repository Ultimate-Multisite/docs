# AffiliateWP {#affiliatewp}

Ultimate Multisite : intégration AffiliateWP

Utilisez le puissant AffiliateWP pour développer la base de clients de votre réseau Ultimo !

## Installation {#installation}

1. Téléversez 'ultimate-multisite-git ' dans le répertoire '/wp-content/plugins/'
2. Activez le plugin via le menu 'Extensions' dans WordPress
3. Suivez l’assistant étape par étape pour configurer le plugin

## Guide d’utilisation {#usage-guide}

[AffiliateWP](https://affiliatewp.com/) est devenu la solution de référence pour créer un programme d’affiliation pour votre activité en ligne. Ultimate Multisite: AffiliateWP est la passerelle qui vous permettra de créer la même chose pour votre réseau premium.

## Bien démarrer {#getting-started}

Tout d’abord, vous **devez avoir** [**AffiliateWP**](https://affiliatewp.com/) **actif** ! Vous ne pourrez pas utiliser notre module complémentaire si vous n’avez pas **le plugin qui s’intègre avec lui**. Si vous ne prévoyez pas de laisser vos clients l’utiliser également sur leurs sites, AffiliateWP ne doit être actif que sur votre site principal.

Vous devrez également avoir **Ultimate Multisite** et **Ultimate Multisite: AffiliateWP** **activés** **sur l’ensemble du réseau**.

## Ajouter Ultimate Multisite comme intégration AffiliateWP {#adding-ultimate-multisite-as-an-affiliatewp-integration}

Sur votre site principal, rendez-vous dans les réglages Affiliés et, dans l’onglet Intégrations, cochez Ultimate Multisite. Cela indiquera à [AffiliateWP](https://affiliatewp.com/) de commencer à suivre les visites et les conversions issues du parcours d’inscription Ultimate Multisite.

<!-- Capture d’écran indisponible : onglet des réglages d’intégrations AffiliateWP avec case à cocher Ultimate Multisite -->

_Cochez la case Ultimate Multisite en bas et enregistrez_

C’est à peu près tout ce qu’il faut. Les visites provenant de liens de parrainage seront suivies et, lorsque cet utilisateur effectuera plus tard son premier paiement, la conversion sera créée et une commission sera mise de côté pour l’affilié responsable de la conversion.

Cependant, seul le premier paiement reçu du nouvel utilisateur créé sera suivi. Pour activer les paiements de parrainage récurrents, vous aurez besoin du [module complémentaire AffiliateWP Recurring Referrals](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## Activer les parrainages récurrents {#enabling-recurring-referrals}

Après avoir installé le [module complémentaire AffiliateWP Recurring Referrals](https://affiliatewp.com/add-ons/pro/recurring-referrals/), rendez-vous dans l’onglet Parrainages récurrents des réglages Affiliés, puis activez l’utilisation des parrainages récurrents.

<!-- Capture d’écran indisponible : réglages AffiliateWP Recurring Referrals avec bouton d’activation des parrainages récurrents -->

_Aussi simple que possible : activez les parrainages récurrents et tout est prêt !_

Encore une fois, c’est tout ce qu’il faut !

Désormais, chaque fois qu’un nouveau paiement est reçu sur un compte parrainé, l’affilié responsable recevra automatiquement sa propre part !
