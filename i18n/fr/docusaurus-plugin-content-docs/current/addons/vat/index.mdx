# TVA

Multisite Ultimate : TVA européenne

Ajoute des fonctionnalités et des outils supplémentaires au traitement des taxes par défaut de Multisite Ultimate pour mieux prendre en charge la TVA européenne.
Automatically adds EU VAT rates to the Tax Rates page from

## Installation

1. Téléchargez 'ultimate-multisite-vat' dans le répertoire '/wp-content/plugins/'
2. Activez le plugin via le menu 'Plugins' dans WordPress
3. Accédez à Ultimate Multisite > Paramètres > Taxes et activez la TVA UE.
4. Cliquez sur Taux de taxe et mettez à jour les taux de TVA UE en bas, puis enregistrez les taux de taxe.
5. Terminé!
