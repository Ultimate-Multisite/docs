# PayFast

Multisite Ultimate: Payfast Gateway

Ajoute Payfast - une passerelle de paiement en ligne rapide et sécurisée en Afrique du Sud - comme option de paiement pour Multisite Ultimate.

## Installation

1. Upload 'ultimate-multisite-payfast' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow the step by step Wizard to set the plugin up

## Usage Guide

PayFast est un processeur de paiement en ligne de premier plan pour les entreprises en Afrique du Sud. Il est dédié à aider les marchands en ligne à se développer en rendant les transactions fluides et sécurisées. Suite à la demande d'un utilisateur d'Ultimate Multisite, nous avons décidé de créer un addon qui rendra l'intégration avec PayFast possible. Cet article montre comment configurer PayFast comme votre passerelle de paiement pour recevoir des paiements dans votre réseau Ultimate Multisite.

La première étape consiste à installer l'addon.

# Comment installer l'addon Payfast:

Allez à la page des addons et sélectionnez l'addon Ultimate Multisite: Payfast Gateway.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-BmLWqj4yjt.png)

La fenêtre de l'addon s'ouvrira. Cliquez sur Installer maintenant.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-1jpCakOHNy.png)

Une fois l'installation terminée, vous serez redirigé vers la page des plugins. Ici, cliquez simplement sur Activer sur le réseau et l'addon _Payfast_ sera activé sur votre réseau.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-49OQHBwPxk.png)

# Configuration de PayFast

Après avoir installé l'addon, vous pouvez maintenant accéder aux paramètres d'Ultimate Multisite et commencer à configurer PayFast.

Sur la page **Ultimate Multisite settings**, vous pouvez trouver l'onglet « Payments ». Sous cet onglet, vous devriez voir « **Payfast** » accompagné d'un bouton bascule pour activer ou désactiver cette passerelle de paiement.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-07bPkaGJka.png)Cliquez sur le bouton bascule pour l'activer. Il développera alors la boîte pour afficher différents champs que vous devez remplir, tels que **Mechant ID, Merchant Key et PassPhrase**. Vous pouvez obtenir ces informations depuis votre compte **Payfast**.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-pocxO2VPfM.png)

L'étape suivante consiste à vous connecter à votre compte PayFast pour obtenir les informations dont vous avez besoin.

Une fois connecté, vous pouvez immédiatement voir les informations d'identification dont vous avez besoin depuis le tableau de bord de votre compte **Payfast**.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-KW0WnsxHIW.png)Vous pouvez simplement copier et coller les informations d'identification et remplir les détails dans vos paramètres Ultimate Multisite. Ensuite, cliquez sur le bouton **« Save settings »** pour enregistrer les modifications.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-5MIrG21r3e.png)

C'est tout, votre intégration de paiement PayFast est terminée!
