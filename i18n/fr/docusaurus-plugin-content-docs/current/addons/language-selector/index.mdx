# Sélecteur de langue

Multisite Ultimate : sélecteur de langue à l’inscription

Ajoutez un champ de sélection de langue au flux d’inscription.

## Installation

1. Téléversez 'ultimate-multisite-language-selector' dans le répertoire '/wp-content/plugins/'
2. Activez le plugin via le menu 'Extensions' dans votre administration réseau WordPress

## Guide d’utilisation

Avec ce module complémentaire gratuit, vous pouvez laisser vos utilisateurs choisir la langue qu’ils souhaitent voir dans le flux d’inscription et sur la page de connexion.

Après avoir [activé votre licence](../../user-guide/getting-started/installing-ultimate-multisite.md), vous pourrez installer les modules complémentaires Ultimate Multisite auxquels votre licence vous donne accès.

Accédez à votre page de réglages Ultimate Multisite, puis, dans le menu de gauche, cliquez sur l’intitulé _Modules complémentaires_.

![Page des modules complémentaires](/img/admin/addons-page.png)

_Accédez à la page des modules complémentaires pour commencer à installer le sélecteur de langue_

Vous y trouverez tous les modules complémentaires que nous proposons.

Les modules complémentaires gratuits sont disponibles pour tous nos clients, quel que soit le plan d’abonnement auquel ils ont souscrit.

Cliquez sur le bouton _Installez-le maintenant !_ sous la vignette du module complémentaire. Activez-le ensuite sur la page Extensions.

![Page des modules complémentaires avec bouton d’installation](/img/admin/addons-page.png)

_Cliquez sur le bouton pour installer le module complémentaire_

<!-- Capture d’écran indisponible : page des extensions WP affichant le bouton d’activation réseau pour le module complémentaire Sélecteur de langue -->

_Activez-le sur votre page Extensions_

Maintenant, accédez au menu Réglages d’Ultimate Multisite et ouvrez l’onglet _Réglages réseau_. Là, en bas de la page, vous trouverez une nouvelle section appelée _Langues_.

![Page des réglages](/img/config/settings-general.png)

_Une toute nouvelle section apparaît !_

Vous pouvez la laisser vide pour afficher toutes les langues comme options, ou sélectionner celles que vous souhaitez rendre disponibles. Les langues sont celles que vous avez installées dans votre configuration WordPress.

<!-- Capture d’écran indisponible : liste déroulante de sélection de langue affichant les langues disponibles dans l’installation WP -->

_Liste des langues disponibles sur cette installation WP_

Après avoir enregistré les modifications que vous avez apportées, un menu de sélection avec les langues que vous avez choisies s’affichera sur la page de connexion ainsi que dans le flux d’inscription.

![Réglages de connexion et d’inscription](/img/config/settings-login-registration.png)

_Sélecteur de langue sur la page de connexion_

![Page d’inscription](/img/config/registration-page.png)

_Sélecteur de langue dans le flux d’inscription_

## Définir la langue via l’URL

Vous pouvez également changer la langue d’une page en passant le paramètre locale dans l’URL. Si vous souhaitez rediriger vers une page d’inscription en italien, par exemple, ajoutez ?locale=it_IT à l’URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

La langue doit être installée sur votre réseau WordPress pour que cela fonctionne.
