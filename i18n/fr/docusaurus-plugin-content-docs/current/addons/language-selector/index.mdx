# Sélecteur de langue

Multisite Ultimate : Sélecteur de langue lors de l'inscription

Ajoutez un champ de sélection de langue au flux d'inscription.

## Installation

1. Téléchargez 'ultimate-multisite-language-selector' dans le répertoire '/wp-content/plugins/'
2. Activez le plugin via le menu 'Plugins' dans l'administration réseau WordPress

## Guide d'utilisation

Avec cette extension gratuite, vous pouvez permettre à vos utilisateurs de choisir la langue qu'ils souhaitent voir dans le flux d'inscription et sur la page de connexion.

Après avoir [activé votre licence](../../user-guide/getting-started/installing-ultimate-multisite.md), vous pourrez installer les extensions Ultimate Multisite auxquelles votre licence vous donne accès.

Accédez à la page des paramètres d'Ultimate Multisite, puis, dans le menu de gauche, cliquez sur l'étiquette _Add-ons_.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-rzVqJCVpY1.png)

_Accédez à la page Add-ons pour commencer l'installation du Sélecteur de langue_

Vous y trouverez toutes les extensions que nous proposons.

Les extensions gratuites sont disponibles pour tous nos clients, quel que soit le plan d'abonnement auquel ils se sont inscrits.

Cliquez sur le bouton _Install it Now!_ sous la vignette de l'extension. Puis activez-la sur la page Plugins.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-bJgpO3QvLh.png)

_Cliquez sur le bouton pour installer l'extension_

_![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-P5ogwZKKDy.png)_

_Activez-la sur votre page Plugins_

Ensuite, accédez au menu Paramètres d'Ultimate Multisite et ouvrez l'onglet _Network Settings_. En bas de la page, vous trouverez une nouvelle section appelée _Languages_.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-CaH7BHDh50.png)

_Une toute nouvelle section apparaît !_

Vous pouvez le laisser vide pour afficher toutes les langues comme options ou sélectionner celles que vous souhaitez rendre disponibles. Les langues sont celles que vous avez installées dans votre configuration WordPress.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-PGljAd4C24.png)

_Liste des langues disponibles sur cette installation WP_

Après avoir enregistré les modifications, un menu de sélection avec les langues que vous avez choisies s'affichera sur la page de connexion ainsi que sur le flux d'inscription.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-qUnJL7JHxY.png)

_Sélecteur de langue sur la page de connexion_

_![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-HnKLDvxuWU.png)_

_Sélecteur de langue sur le flux d'inscription_

## Définir la langue via l'URL

Vous pouvez également changer la langue d'une page en contournant le paramètre locale dans l'URL. Si vous souhaitez rediriger vers une page d'inscription en italien, par exemple, ajoutez ?locale=it_IT à l'URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

La langue doit être installée sur votre réseau WordPress pour que cela fonctionne.
