# Mailchimp

Multisite Ultimate : Intégration Mailchimp

Intégrez votre réseau Multisite Ultimate à votre Mailchimp. Vous pouvez lier certains plans aux Listes Mailchimp et ajouter automatiquement les abonnés à ces listes lors de l'inscription.

## Installation

1. Téléchargez 'ultimate-multisite-mailchimp' dans le répertoire '/wp-content/plugins/'.
2. Activez le plugin via le menu 'Plugins' dans votre Admin Réseau WordPress.

## Usage Guide

Synchroniser les e-mails des abonnés avec vos Listes Mailchimp est vraiment simple grâce à notre Add-on Intégration Mailchimp pour Ultimate Multisite. Ce tutoriel a pour but de vous aider à tout mettre en place !

## Getting an API key

Avant de commencer à explorer les différents paramètres disponibles, nous avons besoin d'une clé API Mailchimp. Cela permettra à Ultimate Multisite de communiquer avec l'API Mailchimp pour récupérer les listes, les groupes, et ajouter les adresses e-mail à ces listes et groupes lorsque les utilisateurs s'inscrivent.

Suivez ce tutoriel pour obtenir votre propre clé API Mailchimp : [À propos des clés API | Mailchimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-zimbKYFZo-B4D1E28C-1203-4E9A-AC7B-D8E9B5659B93.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-zimbKYFZo-B4D1E28C-1203-4E9A-AC7B-D8E9B5659B93.png)

_Ajoutez votre clé API Mailchimp sur_ _**Ultimate Multisite Settings - > Add-on Settings -> MailChimp**_

## Integration Mode

Le add-on Intégration Mailchimp propose deux « modes d’intégration », ce qui signifie qu’il existe deux façons différentes d’utiliser l’add-on pour synchroniser les adresses e-mail.

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-mc8dVzW5S-B7A9495E-05B8-4FBB-8954-80EBF4A9CB3B.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-mc8dVzW5S-B7A9495E-05B8-4FBB-8954-80EBF4A9CB3B.png)

_Deux modes d’intégration pour plus de flexibilité !_

### Mode 1: Multiple Lists

Le mode par défaut utilise **Multiple Lists**. Ce mode vous permet de sélectionner plusieurs listes Mailchimp pour chacun des plans. C’est la meilleure option si vous segmentez vos utilisateurs en différentes listes.

Par exemple, supposons que vous avez 3 listes différentes sur votre compte Mailchimp : **Newsletter**, **Plan A** et **Plan B**. Vous souhaitez que tous les utilisateurs, quel que soit leur plan, soient également inclus dans la liste Newsletter, afin qu'ils ne manquent pas vos mises à jour.

Pour y parvenir, vous pouvez utiliser le mode Multiple Lists pour ajouter les deux listes **Newsletter** et **Plan A** aux Paramètres Mailchimp du Plan A, et **Newsletter** et **Plan B** aux Paramètres Mailchimp du Plan B.

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-kZS8-S_eh-4145C1B8-B903-4DFF-9D92-AF7E0A61445E.png)](https://s3.amazonaws.com/helpscout.net/docs/assets/6017c85715d41b7c717cdcf9/images/6021265efb34b55df443e4ad/60212659ac2f834ec53865f3-kZS8-S%5Feh-4145C1B8-B903-4DFF-9D92-AF7E0A61445E.png)

_Sélectionnez les listes par défaut pour les nouveaux comptes et les comptes annulés_

### Mode 2: Single List + Multiple Groups

Mailchimp vous permet de segmenter votre liste en groupes plus petits d'utilisateurs appelés **Groups**. Cela vous donne plus de contrôle, car vous pouvez envoyer une campagne à un sous-ensemble spécifique de vos abonnés sans avoir à créer plusieurs copies de la même campagne sur Mailchimp (ce qui rend difficile l'analyse des métriques de cette campagne par la suite).

Par exemple, vous pouvez avoir une seule liste appelée **My Network Subscribers**, et à l'intérieur y avoir des groupes pour **Plan A** et **Plan B**.

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-tOsErbJy6-A468DB31-B539-493E-AF5F-F44A4F90A974.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-tOsErbJy6-A468DB31-B539-493E-AF5F-F44A4F90A974.png)

_Sélectionnez les listes et groupes par défaut, si vous utilisez le mode d’intégration « single list + multiple groups »_

## Default Lists and Groups

Une fois que vous avez choisi le mode d’intégration qui convient le mieux à votre cas d'utilisation, il est temps de sélectionner les listes par défaut (et les groupes, selon le mode). La liste par défaut sera utilisée lorsqu'un plan donné ne possède pas de liste spécifique sélectionnée.

Vous pouvez également sélectionner la liste par défaut pour les comptes annulés. Chaque fois qu'un utilisateur met fin à son abonnement et supprime son compte de votre réseau, son adresse e-mail sera déplacée vers la liste définie pour les comptes annulés.

## Enabling Double Subscription

Sur la même page de paramètres, vous pourrez activer **Double Subscription**. Cette option ajoutera l'utilisateur à la fois à la liste par défaut sélectionnée ET à la liste sélectionnée pour le plan de l'utilisateur. Si cette option n'est pas activée, l'utilisateur ne sera ajouté qu'à la liste sélectionnée pour son plan. S'il n'y a aucune liste définie pour ce plan particulier, l'utilisateur sera ajouté à la liste par défaut sélectionnée.

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-OPUDRnO94-5D4C267A-14EB-43B4-B86F-0328D330B521.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-OPUDRnO94-5D4C267A-14EB-43B4-B86F-0328D330B521.png)

_Vous pouvez faire souscrire vos utilisateurs à la fois à la liste par défaut et à la liste du plan lors de l'inscription_

## Selecting Lists and Groups for each of the Plans

Accédez à vos **Plans** et sélectionnez le plan que vous souhaitez modifier. Notez que dans la meta-box **Advanced Options**, un nouvel onglet appelé Mailchimp sera disponible.

Dans cet onglet, vous pourrez sélectionner plusieurs listes ou une seule liste et plusieurs groupes (selon le mode d’intégration) pour ce plan.

[![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-BMHjubpN5-C3183A30-687C-4837-8350-842CF959BB06.png)](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-60212659ac2f834ec53865f3-BMHjubpN5-C3183A30-687C-4837-8350-842CF959BB06.png)

_Sélectionnez les Listes et Groupes que vous souhaitez utiliser pour chacun de vos plans_

Lorsqu'un nouvel utilisateur s'inscrit à ce plan, son adresse e-mail sera ajoutée aux listes et groupes sélectionnés !
