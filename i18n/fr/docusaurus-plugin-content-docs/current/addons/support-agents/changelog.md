# Support Agents Changelog

Version: 1.0.13 - Released on 2025-09-28

* Renommer le préfixe en ultimate-multisite ; mettre à jour le domaine de texte ; mise à jour de la version.

Version 1.0.12 - Released on 2025-09-17

* Corrigé : compatibilité avec Ultimate Multisite 2.4.4

Version 1.0.11 - Released on 2023-11-21

* Corrigé : compatibilité avec Ultimate Multisite 2.3.0

Version 1.0.10 - Released on 2023-08-30

* Corrigé : suppression en masse des Agents de support

Version 1.0.9 - Released on 2023-08-09

* Ajouté : Ajout de la capacité de lire les formulaires de paiement.
* Corrigé : mise en page du formulaire de permission des Agents de support;
* Corrigé : message d'avertissement sur le formulaire de suppression d'un agent de support;
* Corrigé : masquage des widgets du tableau de bord réseau pour les utilisateurs non autorisés.
* Corrigé : correction des capacités d'édition et de suppression des Agents de support.
* Corrigé : correction du processus de création des Agents de support.
* Corrigé : empêché d'ajouter un client comme agent de support.
* Corrigé : redirection des utilisateurs non autorisés vers la page d'administration d'édition de l'agent de support.
* Corrigé : amélioration de la mise en page du formulaire de permission des Agents de support.
* Interne : mise en place d'un nouveau processus de construction.

Version 1.0.8 - Released on 2022-01-21

* Interne : ajout d'un générateur de hooks et filtres;
* Interne : ajout de stubs Ultimate Multisite pour la qualité de vie des développeurs;
* Ajouté : capacités de formulaire de paiement;
* Corrigé : incohérence du nom de capacité entre les formulaires Ultimate Multisite et les options de ce module;

Version 1.0.7 - Released on 2021-12-02

* Corrigé : suppression des Agents de support ne fonctionnait pas;
* Corrigé : suppression en masse des Agents de support ne fonctionnait pas;

Version 1.0.6 - Released on 2021-11-22

* Corrigé : ajout d'un nouveau modal ne fonctionnait pas après les optimisations de performance sur le cœur;
* Corrigé : vue grid-item présente sur le cœur et non sur le module;

Version 1.0.5 - Released on 2021-10-31

* Corrigé : petites incompatibilités avec PHP 8;
* Amélioré : wp_ultimo_skip_network_active_check utilise désormais wu_is_must_use();

Version 1.0.4 - 2021-09-24

* Ajouté : filtre wp_ultimo_skip_network_active_check pour les configurations basées sur mu-plugins;

Version 1.0.3 - Released on 2021-08-29

* Ajouté : prise en charge de l'API REST - les agents de support peuvent être créés via REST;

Version 1.0.2 - Released on 2021-07-26

* Corrigé : mise à jour des permissions des agents ne fonctionnait pas;

Version 1.0.1 - Released on 2021-05-28

* Ajouté : notification lorsque l'utilisateur est déjà super administrateur;
* Ajouté : permission pour autoriser le changement d'utilisateur;
* Amélioré : meilleurs champs de formulaire;

Version 1.0.0 - Initial Release
