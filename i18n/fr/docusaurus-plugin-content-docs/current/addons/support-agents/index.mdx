# Agents de support

Gérer un réseau entier seul n'est pas une tâche facile. Au fur et à mesure que votre réseau grandit, vous constaterez une demande croissante de support. Il sera judicieux d'engager des personnes qui vous aideront à alléger votre charge de travail. Cependant, accorder un accès/une permission complète est risqué et peut compromettre votre réseau. C'est là que ce module d'Agent de support intervient.

## Fonctionnalités

- Liste détaillée des permissions WordPress & Ultimate Multisite
- Activer et désactiver l'accès facilement
- Créer un nombre illimité de comptes d'Agent

## Installation

1. Upload 'ultimate-multisite-support-agents' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow the step by step Wizard to set the plugin up
