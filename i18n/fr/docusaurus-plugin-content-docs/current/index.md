# Ultimate Multisite Documentation

Bienvenue dans la documentation Ultimate Multisite. Ultimate Multisite transforme WordPress Multisite en une plateforme complète de Website as a Service (WaaS).

## Sections de la documentation

### [Guide de l'utilisateur](/user-guide/getting-started/ultimate-multisite-101)

Tout ce dont vous avez besoin pour configurer, gérer et administrer votre plateforme WaaS. Couvre l'installation, les passerelles de paiement, la cartographie de domaine, la gestion des clients, et plus encore.

### [Documentation pour les développeurs](/developer/hooks)

Référence technique pour les développeurs construisant sur Ultimate Multisite. Comprend la référence des hooks, la documentation de l'API REST, et les guides de développement d'addons.

### [Addons](/addons)

Documentation des plugins addons payants qui étendent Ultimate Multisite avec des fonctionnalités supplémentaires telles que l'intégration WooCommerce, l'analyse, la vente de domaines, et plus encore.
