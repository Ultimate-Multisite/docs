# Comment obtenir de l'aide pour le développement personnalisé via Codeable ?

Parfois, nous avons vraiment besoin d'une personnalisation géniale pour que nos flux réseau soient exactement comme nous le souhaitons. Si c'est ce que vous recherchez, nous vous recommandons [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), un endroit où vous pouvez trouver des développeurs expérimentés spécialisés dans Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) est une plateforme qui met en relation les clients avec des développeurs freelance WordPress qualifiés, capables de travailler sur des projets de toute taille. Ils ont un processus de candidature méticuleux pour s'assurer que vous êtes jumelé avec des experts spécifiquement qualifiés pour la tâche que vous avez en tête.

### Comment créer un projet sur Codeable ?

Tout d'abord, allez sur [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) et cliquez sur le bouton **Start a Project**.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Ensuite, pour soumettre un projet, cliquez sur le côté gauche sur **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

Dans « I need help with », choisissez **Customizing** :

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

Dans « My », choisissez **Plugin** :

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Ensuite, cliquez sur le bouton **Continue**.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Maintenant, il est temps de décrire votre projet et vos besoins de la meilleure façon possible. Essayez de fournir autant d'informations que possible !

<!-- Screenshot unavailable: Codeable project description form -->

Répondez aux questions suivantes pour aider à la tarification.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Ensuite, si vous n'avez pas encore de compte [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), créez-en un et cliquez sur le bouton **Publish My Project**.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

Une fois que vous avez publié votre projet sur [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), la plateforme vous mettra en relation avec des développeurs correspondant aux compétences requises pour votre projet.

Ce processus vous aidera à trouver la bonne personne pour votre travail, qui possède de l'expérience avec WordPress et Ultimate Multisite !

Le tarif minimum de Codeable est de 70 à 120 USD par heure. Les estimations varieront en fonction de la portée du travail, de la complexité et de l'urgence du projet.

Une fois que vous avez reçu votre estimation, vous pourrez engager le développeur qui correspond le mieux à vos besoins et lancer votre projet ensemble !
