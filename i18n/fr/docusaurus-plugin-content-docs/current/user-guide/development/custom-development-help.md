# Comment obtenir de l’aide pour du développement personnalisé avec Codeable ?

Parfois, nous avons vraiment besoin d’une personnalisation remarquable pour que notre réseau fonctionne exactement comme nous le souhaitons. Si c’est ce que vous recherchez, nous recommandons [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), un endroit où vous pouvez trouver des développeurs expérimentés ayant une expertise Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) est une plateforme qui met en relation des clients avec des développeurs freelance WordPress qualifiés, capables de travailler sur des projets de toute taille. Ils disposent d’un processus de candidature minutieux afin de s’assurer que vous êtes mis en relation avec des experts spécifiquement qualifiés pour la tâche que vous avez en tête.

### Comment créer un projet dans Codeable ? {#how-to-create-a-project-in-codeable}

Tout d’abord, allez sur [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) et cliquez sur le bouton **Start a Project**.

<!-- Capture d’écran indisponible : page d’accueil Codeable avec le bouton Start a Project -->

Maintenant, pour soumettre un projet, cliquez à gauche sur **Start a Project**.

<!-- Capture d’écran indisponible : page de soumission de projet Codeable avec l’option Start a Project -->

Dans « I need help with », choisissez **Customizing** :

<!-- Capture d’écran indisponible : formulaire de projet Codeable avec Customizing sélectionné -->

Dans « My », choisissez **Plugin** :

<!-- Capture d’écran indisponible : formulaire de projet Codeable avec Plugin sélectionné -->

Cliquez ensuite sur le bouton **Continue**.

<!-- Capture d’écran indisponible : bouton Continue du formulaire de projet Codeable -->

Maintenant, il est temps de décrire votre projet et vos besoins du mieux possible. Essayez de donner autant d’informations que possible !

<!-- Capture d’écran indisponible : formulaire de description de projet Codeable -->

Répondez aux questions suivantes pour aider à établir le prix.

<!-- Capture d’écran indisponible : questions de tarification du projet Codeable -->

Après cela, si vous n’avez pas encore de compte [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), créez-en un et cliquez sur le bouton **Publish My Project**.

<!-- Capture d’écran indisponible : création de compte Codeable et bouton Publish My Project -->

Une fois que vous avez publié votre projet [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), la plateforme vous mettra en relation avec des développeurs dont les compétences correspondent à celles requises pour votre projet.

Ce processus vous aidera à trouver la bonne personne pour votre mission, qui a de l’expérience avec WordPress et Ultimate Multisite !

Le tarif minimum de Codeable est de 70 à 120 $ USD par heure. Les estimations varieront en fonction de l’étendue du travail, de la complexité et de l’urgence d’un projet.

Une fois que vous aurez reçu votre estimation, vous pourrez engager le développeur qui correspond le mieux à vos besoins et lancer votre projet ensemble !
