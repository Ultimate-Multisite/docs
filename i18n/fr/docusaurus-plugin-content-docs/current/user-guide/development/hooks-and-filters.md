# Ultimate Multisite Crochets et filtres

Ultimate Multisite est composé d'un grand nombre de fonctions et de requêtes de base de données. Il possède des crochets et des filtres qui vous permettront de modifier ou d'ajouter du nouveau code aux fonctions existantes.

Nous avons créé une liste de ces crochets et filtres comme référence pour aider les utilisateurs avancés à personnaliser davantage leur réseau. Vous pouvez accéder à la liste depuis Github en cliquant sur le lien ci-dessous.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
