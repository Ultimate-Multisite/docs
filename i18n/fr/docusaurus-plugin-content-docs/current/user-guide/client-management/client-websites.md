# Les sites Web de votre client (v2)

Après leur inscription sur votre réseau, vos clients peuvent accéder à leur Dashboard.

En plus de la page **Account**, où ils peuvent voir des informations importantes concernant leurs paiements, abonnements, domaines, limitations de plan et plus encore, ils ont également accès à la page **Mes sites**, où ils peuvent voir et gérer tous leurs sites Web. Il leur suffit de cliquer sur Mes sites dans leur Dashboard.

![Lien Mes sites dans le Dashboard du client](/img/account-page/my-sites-link.png)

Tous leurs sites Web leur seront affichés, avec un bouton **Gérer** juste en dessous de chacun d’eux. S’ils cliquent dessus, ils seront redirigés vers le **Dashboard de ce site Web spécifique**.

Ils peuvent également voir lequel de leurs sites Web est le site **principal** pour les plans qui leur permettent d’avoir plus d’un site Web.

![Page Mes sites affichant les sites Web des clients avec des boutons Gérer](/img/account-page/my-sites-page.png)

Voici une vue complète de l’interface de modification du site :

![Interface complète de modification du site](/img/admin/site-edit-full.png)

S’ils veulent changer leur site Web principal, ils doivent se rendre sur la page Account en cliquant sur le menu **Account**.

![Lien du menu Account dans le Dashboard du client](/img/account-page/account-menu.png)

Vous pouvez également consulter et modifier les détails du client :

![Interface de modification du client](/img/admin/customer-edit.png)

Ensuite, ils doivent cliquer sur **Changer le site par défaut**.

![Bouton Changer le site par défaut sur la page Account](/img/account-page/change-default-site.png)

Une fenêtre contextuelle apparaîtra. Cliquez sur la liste déroulante et sélectionnez le site Web que vous souhaitez définir comme site Web principal.

![Sélection dans la liste déroulante Changer le site par défaut](/img/account-page/change-default-site-dropdown.png)

Ensuite, cliquez simplement sur **Changer le site par défaut** et c’est terminé !

![Confirmation du changement du site par défaut](/img/account-page/change-default-site-confirm.png)
