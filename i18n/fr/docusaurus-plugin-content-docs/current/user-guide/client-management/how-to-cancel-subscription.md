# Comment puis-je annuler mon abonnement ?

Si vous avez une licence Ultimate Multisite, vous pouvez annuler son renouvellement à tout moment. Il vous suffit de suivre les étapes ci-dessous :


<!-- Screenshot unavailable: Account page email with access link -->

Utilisez l’adresse e-mail que vous avez fournie lors de l’achat et votre mot de passe.

<!-- Screenshot unavailable: Login page with email and password fields -->

Voici l’écran initial de votre page Account :

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

Pour annuler votre abonnement, dans le menu à gauche, allez dans _Renewing & Billing._

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

Cliquez sur la flèche à droite pour ouvrir une fenêtre latérale. Ensuite, vous devez sélectionner l’option qui annulera le **auto-renew**.

Le système vous affichera un message de confirmation.

Si vous êtes sûr de vouloir annuler votre abonnement, cliquez simplement sur le bouton _**Cancel Renewals**_. Après cette action, il vous sera demandé de répondre à une courte enquête.

Terminé ! Votre abonnement ne sera pas renouvelé automatiquement.

**Vous disposerez d’une clé valide jusqu’à l’expiration de votre abonnement**. Si vous souhaitez réactiver votre abonnement, vous devrez le faire manuellement.
