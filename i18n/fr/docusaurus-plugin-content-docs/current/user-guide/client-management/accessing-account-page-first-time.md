Bienvenue sur Ultimate Multisite !

Accédez à votre page de compte via le lien que vous avez reçu dans votre e‑mail après l’achat de notre plugin :

<!-- Screenshot unavailable: Account page email with access link -->

Utilisez l’e‑mail que vous avez fourni lors de l’achat et le mot de passe que vous avez reçu dans votre e‑mail.

<!-- Screenshot unavailable: Login page with email and password fields -->

Comme c’est la première fois que vous vous connectez à ce compte, il vous sera demandé de changer votre mot de passe. Modifiez-le et cliquez sur le bouton _Save_.

<!-- Screenshot unavailable: Change password form with Save button -->

Sur votre page de compte, vous pouvez gérer tous les détails de votre compte tels que les facturations, les renouvellements, les annulations, et bien d’autres choses.
