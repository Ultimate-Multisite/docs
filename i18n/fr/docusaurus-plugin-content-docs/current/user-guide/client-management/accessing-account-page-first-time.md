# Comment accéder à ma page de compte pour la première fois ? {#how-can-i-access-my-account-page-for-the-first-time}

Bienvenue dans Ultimate Multisite !

Accédez à votre page de compte via le lien que vous avez reçu dans votre e-mail après avoir acheté notre plugin :

<!-- Capture d’écran indisponible : e-mail de page Account avec lien d’accès -->

Utilisez l’e-mail que vous avez fourni lors de l’achat et le mot de passe que vous avez reçu dans votre e-mail.

<!-- Capture d’écran indisponible : page de connexion avec champs e-mail et mot de passe -->

Comme c’est la première fois que vous vous connectez à cet Account, il vous sera demandé de changer votre mot de passe. Changez-le et cliquez sur le bouton _Enregistrer_.

<!-- Capture d’écran indisponible : formulaire de changement de mot de passe avec bouton Enregistrer -->

Sur votre page de compte, vous pouvez gérer toutes les informations de votre compte, comme les facturations, les renouvellements, les annulations et bien d’autres choses.
