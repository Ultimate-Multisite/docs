# Questions fréquemment posées

Voici quelques **Questions fréquemment posées** qui surgissent souvent lors des demandes de support.

  * **Qu'est-ce qu'Ultimate Multisite ?**

Ultimate Multisite est un plugin WordPress, conçu pour les installations multisite WordPress, qui transforme votre installation WordPress en un réseau premium de sites – comme [WordPress.com](https://WordPress.com) – permettant aux clients de créer des sites via des frais mensuels, trimestriels ou annuels (vous pouvez également créer des plans gratuits).

  * **Ultimate Multisite fonctionnera-t-il sans Multisite ?**

Non. Ultimate Multisite nécessite WordPress multisite, sous-répertoire ou sous-domaine.

  * **Quelles sont les exigences pour Ultimate Multisite ?**

Ultimate Multisite nécessite :

**WordPress Version** : v5.3+ (Recommandé : Dernière version stable)  
**PHP Version** : 7.4.x (Support > 8.0 à venir bientôt)  
**MySQL Version** : v5+ (Recommandé : Version 5.6, au cas où vous n'auriez pas la 8.0 disponible chez votre hébergeur)

_Multisite (sous-domaine ou sous-répertoire) doit également être activé._ [Comment installer WordPress Multisite](https://support.delta.nextpress.co/hc/wp-ultimo/articles/1677127280-how-do-i-install-word_press-multisite).

  * **Ultimate Multisite fonctionnera-t-il avec Multisite avec des sous-répertoires ?**

Oui. Ultimate Multisite prend en charge les installations en sous-répertoire et les installations en sous-répertoire.

  * **Ultimate Multisite fonctionnera-t-il avec Multisite avec des sous-domaines ?**

Oui. Ultimate Multisite prend en charge les installations en sous-répertoire et les installations en sous-domaine.

  * **Puis-je utiliser Ultimate Multisite pour limiter les plugins et thèmes que les utilisateurs peuvent utiliser ?**

Oui. Avec Ultimate Multisite, vous pouvez sélectionner quels plugins et thèmes sont disponibles pour chaque plan sur votre réseau.

  * **Puis-je utiliser Ultimate Multisite pour limiter le nombre de publications qu'un utilisateur peut créer sur son site ?**

Oui. Avec Ultimate Multisite, vous pouvez établir des quotas de publications pour **chaque type de publication disponible** sur votre réseau. Vous pouvez également limiter le nombre d'utilisateurs que l'administrateur peut inviter, la **quantité d'espace disque**, et bien plus encore.

  * **Ultimate Multisite prend-il en charge les modèles de site ?**

Oui. Avec Ultimate Multisite, vous pouvez créer autant de Modèles de site que vous le souhaitez. Les Modèles de site ne sont pas des thèmes, ce sont des sites normaux sur votre réseau. Vous pouvez activer des plugins, des thèmes, et personnaliser ce Modèle de site de toutes les manières que vous souhaitez. Si un utilisateur sélectionne ce modèle, son site sera créé comme une copie de ce modèle, avec toutes les personnalisations en place.

Vous pouvez également définir un modèle particulier pour un plan. En faisant cela, chaque utilisateur qui s'inscrit à ce plan aura son site créé comme un duplicata de ce Modèle de site.

  * **Ultimate Multisite s'intègre-t-il avec WooCommerce ?**

Oui !

Si vous souhaitez que vos utilisateurs réseau disposent de WooCommerce, leur permettant de créer des boutiques, c'est possible avec Ultimate Multisite. Vous pouvez même limiter le nombre de produits que vos clients pourront créer à chaque niveau de plan.

En ce qui concerne les paiements d'abonnement pour Ultimate Multisite, vous pouvez utiliser le [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) pour permettre à vos clients Ultimate Multisite de payer via les passerelles WooCommerce sur votre site principal. De cette façon, vous pouvez exploiter la vaste bibliothèque de passerelles de paiement développées pour WooCommerce et les utiliser pour facturer vos clients Ultimate Multisite.

  * **Existe-t-il un moyen de migrer les abonnements depuis WPMUDEV Pro Sites ?**

Oui, Ultimate Multisite vous propose désormais [un outil utile pour vous aider à migrer](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) votre réseau Pro Sites.

  * **Pourquoi l'onglet « Plugins autorisés » est-il vide sur ma page d'édition de plan ?**

Ultimate Multisite ne vous permet d'autoriser ou de refuser l'accès qu'aux plugins non activés sur le réseau. Lorsqu'un plugin est activé sur le réseau, par nature, il sera chargé par défaut sur chaque site de votre réseau.

Si vous ne voyez aucun plugin répertorié dans l'onglet « Plugins autorisés », vérifiez si le plugin souhaité est activé sur le réseau ou non.

  * **Puis-je utiliser ma propre conception de tableau de prix ?**

Oui. Depuis la version 1.4.0, Ultimate Multisite prend en charge les remplacements de modèles.

  * **Ultimate Multisite fonctionne-t-il avec WP Engine ?**

Oui. Ultimate Multisite était toujours pris en charge par WP Engine, mais faire fonctionner la cartographie de domaine nécessitait des étapes supplémentaires manuelles de l'administrateur (ajout du domaine au panneau WP Engine). **Cela n'est plus le cas depuis la version 1.6.0.** Ultimate Multisite s'intègre directement à l'API WP Engine et les domaines sont ajoutés automatiquement au panneau après la cartographie.

  * **Quel hébergeur recommandez-vous ?**

Ultimate Multisite devrait fonctionner avec n'importe quel hébergeur. La seule différence est que certaines entreprises d'hébergement comme Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane et [Runcloud.io](http://Runcloud.io) disposent d'une intégration plus poussée pour les capacités de cartographie de domaine, ce qui les rend prêtes à l'emploi. Pour les autres hébergeurs, vous devrez peut-être configurer la cartographie de domaine manuellement.
