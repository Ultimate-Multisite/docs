# Traduire Ultimate Multisite

Ultimate Multisite est développé en anglais, mais le plugin est entièrement traduisible et nous accueillons activement les contributions de traduction.

## Traductions du plugin principal

Le plugin principal Ultimate Multisite est traduit via la plateforme officielle **WordPress.org plateforme de traduction** :

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

Ceci est le système de traduction communautaire standard de WordPress. Lorsque les traductions atteignent un seuil d'approbation suffisant, elles sont automatiquement distribuées à tous les utilisateurs d'Ultimate Multisite via les mises à jour du plugin WordPress.

**Nous avons besoin de votre aide!** Si vous parlez une langue autre que l'anglais, veuillez envisager de contribuer des traductions. Même traduire quelques chaînes aide. Pour commencer :

1. Visit [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)
2. Sélectionnez votre langue
3. Cliquez sur "Contribuer à la traduction"
4. Vous aurez besoin d'un compte gratuit WordPress.org

Si vous souhaitez devenir éditeur de traduction pour votre langue (capable d'approuver les traductions), contactez via [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions).

## Traductions des addons

Les plugins addons sont traduits séparément à :

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Les traductions des addons sont actuellement générées par IA comme point de départ. Elles couvrent toutes les langues prises en charge mais peuvent contenir des formulations maladroites ou des inexactitudes. Nous invitons les traducteurs humains à les examiner et à proposer des améliorations pour les rendre plus naturelles et précises.

## Traductions locales

Si vous préférez créer des traductions localement pour votre propre projet plutôt que de contribuer en amont, il existe deux options :

### Utilisation de Poedit

Vous pouvez utiliser l'application de bureau **[Poedit](https://poedit.net)** pour traduire le fichier `.pot` :

1. Téléchargez Poedit depuis [poedit.net](https://poedit.net)
2. Ouvrez le fichier `.pot` depuis `wp-content/plugins/ultimate-multisite/lang/`
3. Traduisez les chaînes dans l'interface Poedit
4. Enregistrez -- Poedit génère les fichiers `.po` et `.mo`
5. Placez les fichiers de sortie dans le dossier `lang`

### Utilisation de Loco Translate

Le plugin WordPress **[Loco Translate](https://wordpress.org/plugins/loco-translate/)** vous permet de traduire directement depuis le panneau d'administration sans aucun outil de bureau. Il s'occupe de créer et compiler les fichiers de traduction pour vous.
