# Placer des publicités sur votre réseau avec Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector est un **add-on gratuit** qui vous donne les outils pour placer des publicités sur vos plans directement depuis leur page de modification.

Après avoir installé Ultimate Multisite: Ad Injector, allez sur votre page _Plugins_ pour l’activer.

<!-- Capture d’écran indisponible : page WP Plugins affichant le bouton Network Activate pour l’addon Ad Injector -->

_Cliquez sur le bouton Network Activate_

Une fois l’add-on actif, la boîte méta _Advanced Options_ sur l’écran de modification du plan comportera une nouvelle option appelée _Ad Settings_.

![Ad Settings sur l’écran de modification du plan](/img/config/product-edit-full.png)

_Vous pouvez maintenant modifier les paramètres des publicités pour chacun de vos plans_

Dans cet onglet, vous pourrez ajouter les codes publicitaires pour les articles et les pages dans le front-end, avant et après le contenu.

<!-- Capture d’écran indisponible : champs de code publicitaire pour les articles et les pages avant le contenu -->

<!-- Capture d’écran indisponible : exemple de publicité affichée sur une page d’article pour un plan spécifique -->

_Après avoir inséré le code de votre publicité, elle s’affichera sur les articles et les pages de ce plan spécifique._

Vous pouvez également inclure des publicités dans l’en-tête des Admin Pages pour les utilisateurs de ce plan.

<!-- Capture d’écran indisponible : option Admin Panel Ad Code dans les paramètres du plan -->

<!-- Capture d’écran indisponible : exemple de publicité affichée dans le panneau d’administration de l’utilisateur -->

_Allez à l’option Admin Panel Ad Code si vous souhaitez afficher des publicités dans le back-end de votre utilisateur_

### Au-delà de la publicité

Vous pouvez également utiliser Ultimate Multisite: Ad Injector pour offrir une assistance aux plans de niveau supérieur.

Allez simplement à l’option _Admin Panel Ads_ et ajoutez le code HTML de votre outil d’assistance.

<!-- Capture d’écran indisponible : option Admin Panel Ads avec le code HTML de l’outil d’assistance -->

<!-- Capture d’écran indisponible : widget d’assistance client affiché dans le panneau d’administration pour un plan spécifique -->

_Ajoutez une assistance client à des plans spécifiques_

Ainsi, les utilisateurs peuvent vous contacter directement depuis leur Admin Panel !
