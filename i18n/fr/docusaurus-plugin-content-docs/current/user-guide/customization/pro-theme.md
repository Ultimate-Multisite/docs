# Personnaliser votre panneau d’administration avec PRO Theme

Souvent, lors de travaux pour des clients, nous devons offrir à nos clients une expérience personnalisée dans le Dashboard d’administration de WordPress. Si vous recherchez une apparence et une expérience plus professionnelles, PRO Theme pourrait vous aider.

PRO Theme est un thème d’administration développé pour vous aider à personnaliser le Dashboard de vos clients. Il fait également partie du cœur des plugins premium d’Ultimate Multisite.

Après l’avoir installé, allez sur votre page _Plugins_ et cliquez pour l’activer sur votre réseau.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Activez-le à l’échelle du réseau sur votre page Plugins_

_**REMARQUE :**_ _Si vous venez de l’installer, il apparaîtra chez votre client comme une option pouvant être cochée pour ajouter le thème uniquement à cet utilisateur._

Maintenant, votre interface d’administration est entièrement repensée ! Vous disposez également d’un nouveau menu dans la colonne de gauche appelé PRO Theme.
