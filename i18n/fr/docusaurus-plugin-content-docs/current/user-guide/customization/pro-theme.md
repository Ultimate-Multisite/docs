# Personnaliser votre panneau d'administration avec le thème PRO

Souvent, lorsqu'on travaille pour des clients, on a besoin de leur offrir une expérience personnalisée dans le tableau de bord d'administration de WordPress. Si vous recherchez un look et une sensation plus professionnels, le thème PRO peut vous aider.

Le thème PRO est un thème d'administration développé pour vous aider à personnaliser le tableau de bord de vos clients. Il fait également partie du cœur des plugins premium d'Ultimate Multisite.

Après l'avoir installé, allez sur votre page _Plugins_ et cliquez pour l'activer sur votre réseau.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Activez-le à l'échelle du réseau sur votre page Plugins_

_**NOTE :**_ _Si vous l'installez simplement, il apparaîtra chez votre client comme une option pouvant être cochée pour ajouter le thème uniquement à cet utilisateur._

Maintenant, votre interface d'administration est complètement repensée ! Vous disposez également d'un nouveau menu dans la colonne de gauche appelé PRO Theme.

<!-- Screenshot unavailable: PRO Theme redesigned admin panel interface -->

_Un tout nouveau look pour votre panneau d'administration_
