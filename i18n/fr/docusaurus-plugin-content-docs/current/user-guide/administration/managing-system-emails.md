# Gestion des e-mails système

Ultimate Multisite envoie divers e-mails système pour des événements tels que les confirmations d'inscription, les reçus de paiement et les changements d'adhésion. Vous pouvez personnaliser ces modèles d'e-mails pour correspondre à votre marque.

## Affichage des e-mails système

Accédez à **Ultimate Multisite → Settings** et cliquez sur l'onglet **Emails** pour voir les paramètres liés aux e-mails.

![Paramètres - Onglet Emails](/img/admin/settings-emails.png)

À partir de là, vous pouvez accéder à la page **System Emails** pour voir tous les modèles d'e-mails.

![Liste des e-mails système](/img/admin/system-emails-list.png)

## Importation de nouveaux modèles d'e-mails

Les nouvelles versions d'Ultimate Multisite et les extensions peuvent enregistrer de nouveaux modèles d'e-mails. Pour éviter les conflits, les nouveaux modèles ne sont pas ajoutés automatiquement — vous devez les importer manuellement.

Pour importer de nouveaux modèles d'e-mails :

1. Allez à la page **System Emails**
2. Cliquez sur le bouton **Reset & Import** en haut de la page
3. Activez l'option **Import Emails** pour voir les modèles disponibles
4. Sélectionnez les modèles que vous souhaitez importer et confirmez

## Réinitialisation des modèles d'e-mails

Si vous avez personnalisé un modèle d'e-mails et souhaitez revenir au contenu par défaut, vous avez deux options :

1. **Supprimer et réimporter** — Supprimez l'e-mail système et réimportez-le (cela efface les métriques d'envoi)
2. **Réinitialiser via l'outil** — Utilisez l'outil **Reset & Import**, activez l'option **Reset**, et sélectionnez les e-mails que vous souhaitez réinitialiser

L'option Reset est préférée car elle conserve les métriques d'envoi et d'autres métadonnées tout en restaurant le contenu par défaut.
