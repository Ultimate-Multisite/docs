# Gestion des clients

La page **Customers** affiche la liste des membres enregistrés dans votre réseau. Cette page affiche les informations de base de l'utilisateur, y compris le nom, l'adresse e‑mail, la dernière connexion, et un lien **switch to** qui vous permet de vous connecter au sous‑site de l'utilisateur en tant qu'administrateur du site.

![Page de liste des clients](/img/admin/customers-list.png)

Cliquez sur **Manage** sous la carte d'un client pour voir les informations détaillées sur ce client.

Sur la page de détail du client, le bloc **At a Glance** affiche l'ID du client, le temps de la dernière connexion et le montant total dépensé par ce client.

Le bloc **Customer Options** vous permet de définir ce client comme **VIP** (un tag pour le rendre plus facile à trouver). L'onglet **Billing Info** affiche les informations de facturation telles que l'e‑mail, l'adresse et le pays.

Le bloc **Fields** affiche le nom d'utilisateur du client, l'e‑mail, et vous permet de modifier le statut **Email Verification**.

Sous le bloc Customer Options, vous trouverez quatre blocs supplémentaires :

- **Memberships** — Abonnements actuels auxquels le client est abonné
- **Payments** — L'historique des paiements du client
- **Sites** — Sites que possède ce client
- **Events** — Événements liés à ce client (vérification d'e‑mail, changements de facturation, etc.)

Cliquez sur l'icône flèche (→) à côté de tout élément pour naviguer vers sa page de détail. Par exemple, cliquer sur la flèche à côté d'un site vous amène à la page d'édition du site où vous pouvez modifier l'association d'abonnement, le type de site et le statut actif.

Il en va de même pour les abonnements et les paiements — chacun mène à des pages de détail où vous pouvez modifier le statut d'abonnement, le client associé, les produits, les paramètres de passerelle de paiement, et même traiter des remboursements.

Sur le côté droit de la page client, vous trouverez :

- **Last Login & IPs** — Heure de la dernière connexion, adresse IP et pays
- **Timestamps** — Quand le compte client a été créé et modifié pour la dernière fois
- **Delete** — Option de supprimer définitivement le client

:::warning
Supprimer un client est permanent et ne peut pas être annulé.
:::
