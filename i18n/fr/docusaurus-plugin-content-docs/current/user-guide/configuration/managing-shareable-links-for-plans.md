# Gérer les liens partageables pour les plans {#managing-shareable-links-for-plans}

Ultimate Multisite vous offre la possibilité de partager des liens de plan directement avec vos clients. Pour ce faire, accédez à l’une des pages de modification de vos plans et cliquez sur le bouton _Cliquer pour copier le lien partageable_.

![Page de modification du produit avec bouton de lien partageable](/img/config/product-edit-full.png)

_Sur la page de modification du plan, vous pouvez trouver le bouton pour créer des liens partageables_

Notez que ce mécanisme utilise la fréquence de facturation par défaut pour générer le lien. Cela signifie que si votre facturation par défaut est mensuelle, par exemple, vous aurez au final un lien comme celui-ci : [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Si vous souhaitez que le lien partageable utilise par défaut une fréquence trimestrielle, vous devrez ajouter 3 à la fin

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Pour sélectionner le plan _premium_ avec une périodicité annuelle, vous ajouteriez 1 puis le mot :

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Gardez à l’esprit que vous ne pouvez pas avoir un seul lien pour différentes fréquences de facturation. Vous devez partager un lien spécifique pour chacune de vos fréquences de facturation.

Une fois que vous partagez le lien avec vos clients, il les redirigera vers la première étape de votre parcours d’inscription.
