# Gestion des liens partageables pour les plans

Ultimate Multisite vous offre la possibilité de partager directement les liens de plan avec vos clients. Pour ce faire, accédez à la page d'édition de l'un de vos plans et cliquez sur le bouton _Cliquez pour copier le lien partageable_.

![](https://wp-ultimo-space.fra1.cdn.digitaloceanspaces.com/hs-file-2DJoMvapXh.png)

_On la page d'édition du plan, vous trouverez le bouton pour créer des liens partageables_

Notez que ce mécanisme utilise la fréquence de facturation par défaut pour construire le lien. Cela signifie que si votre facturation par défaut est mensuelle, par exemple, vous obtiendrez finalement un lien comme celui-ci : [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Si vous souhaitez que le lien partageable soit par défaut trimestriel, vous devrez ajouter 3 à la fin

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Pour sélectionner le plan _premium_ sur un calendrier annuel, vous ajouteriez 1 puis le mot :

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Gardez à l'esprit que vous ne pouvez pas avoir un seul lien pour différentes fréquences de facturation. Vous devez partager un lien spécifique pour chacune de vos fréquences de facturation.

Une fois que vous partagez le lien avec vos clients, il les redirigera vers la première étape de votre flux d'inscription.
