# Action: wu_page_before_render

Permet aux développeurs de plugins d'ajouter du contenu supplémentaire avant d'imprimer la page.

## Parameters {#parameters}

| Nom | Type | Description |
|------|------|-------------|
| $page_id | `string` | L'identifiant de cette page. |
| $page | `object` | L'objet de la page. |
