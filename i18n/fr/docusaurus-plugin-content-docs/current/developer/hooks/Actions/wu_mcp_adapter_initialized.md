# Action: wu_mcp_adapter_initialized

Se déclenche après l'initialisation de l'adaptateur MCP.

Permet à d'autres plugins et thèmes d'enregistrer leurs propres capacités.

## Paramètres

| Nom | Type | Description |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | L'instance de l'adaptateur MCP. |
