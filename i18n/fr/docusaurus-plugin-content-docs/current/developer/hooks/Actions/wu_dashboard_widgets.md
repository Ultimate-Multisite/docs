# Action: wu_dashboard_widgets

Permet aux développeurs de plugins d'ajouter des widgets au panneau du tableau de bord réseau.

## Parameters

| Nom | Type | Description |
|------|------|-------------|
| $tab | `string` | L'onglet actuel. |
| $screen | `\WP_Screen` | L'objet écran. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Instance de la page d'administration Ultimate Multisite. |

### Depuis

- 2.0.0
