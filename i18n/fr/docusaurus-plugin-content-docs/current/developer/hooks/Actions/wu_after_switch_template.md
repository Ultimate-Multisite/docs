# Action: wu_after_switch_template

Permet aux développeurs de plugins d'accrocher des fonctions après qu'un utilisateur ou un super administrateur ait changé le modèle du site

## Parameters

| Nom | Type | Description |
|------|------|-------------|
| $id | `int` | ID du site |
