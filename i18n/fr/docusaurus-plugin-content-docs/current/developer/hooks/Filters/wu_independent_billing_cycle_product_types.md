# Filter: wu_independent_billing_cycle_product_types

Filtrer les types de produits qui ont des cycles de facturation indépendants.

Les produits avec des cycles de facturation indépendants ne sont pas contraints de correspondre à la période de facturation du plan dans le panier. Cela est utile pour des produits comme les enregistrements de domaine qui facturent toujours annuellement, indépendamment du fait que le plan soit mensuel ou annuel.

## Paramètres

| Nom | Type | Description |
|------|------|-------------|
| $types | `array` | Tableau des types de produits avec des cycles de facturation indépendants. |
