# Filter: wu_view_override_replaceable_views

Permet aux développeurs d’ajouter des dossiers supplémentaires à la liste remplaçable.

Soyez prudent, car l’ajout de dossiers supplémentaires peut entraîner le chargement de copies obsolètes au lieu des versions Ultimate Multisite.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $replaceable_views | `array` | Liste des dossiers autorisés. |
