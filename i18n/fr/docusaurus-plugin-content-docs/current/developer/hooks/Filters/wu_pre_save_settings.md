# Filtre: wu_pre_save_settings

Autoriser les développeurs à filtrer les paramètres avant la sauvegarde par Ultimate Multisite.

## Paramètres

| Nom | Type | Description |
|------|------|-------------|
| $settings | `array` | Les paramètres à sauvegarder. |
| $settings_to_save | `array` | Les nouveaux paramètres à ajouter. |
| $saved_settings | `array` | Les paramètres actuels sauvegardés. |

### Depuis

- 2.0.18
