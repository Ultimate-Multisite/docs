# Filter: wu_plan_product_types

Filtrer les types de produit qui doivent être traités comme des plans.

Ce filtre permet aux modules complémentaires d'enregistrer des types de produit supplémentaires qui doivent être reconnus comme des plans lors de la validation et de la segmentation.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | Tableau des types de produit à traiter comme des plans. |

- 2.3.0
