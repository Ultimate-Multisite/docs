# Žinomi nekompatybiai su kitais pluginais {#known-incompatibilities-with-other-plugins}

Po instalavimo ir aktyvuojame Ultimate Multisite, galite susitikti su problemomis dėl nereikalaujamų nekompatabilitietų su kitais pluginais.

Mes aktyviai dirbiname, kad šią sąrašą mažinant kas naujo atnaujinimo metu, bet jei pluginas yra skirbtiame šiuose, jums gali reikėti jį deaktivuoti, kad turėtumėte visiškai veikiančią Ultimate Multisite Tęstėlę.

Štai jų sąrašas ir ką turėtumėte padaryti tokiuose savakartuose:

**Kai kurie WPMU DEV pluginai** Kai kurie _darbiniai_ nekompatybėmis su WPMU DEV, jie žinomi dėl to, kad gali sukelti Ultimate Multisite registracijos procesą užbaigimo metu (naudotojas negali suk creare paskyros). Štai sąrašas pluginų, kurie vyriausčia šiais problemomis:

  * Signup Language: _Ultimate Multisite turi papildomą funkciją, kuri yra ta sama, vadinama Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Tai įtraukta Ultimate Multisite, todėl nėra reikalinga jokių papildomo plugino._

  * New Blog Templates: _Tai įtraukta Ultimate Multisite, todėl nėra reikalinga jokių papildomo plugino._

  * Set Password on Multisite Blog Creation: _Tai įtraukta Ultimate Multisite, todėl nėra reikalinga jokių papildomo plugino._

  * Signup TOS: _Tai įtraukta Ultimate Multisite, todėl nėra reikalinga jokių papildomo plugino._

  * Anti-Splog: Nors jis yra mažiau galingas, jis taip pat _įtrauktas Ultimate Multisite._

**Pastaba** : Visi kiti WPMU Dev pluginai gali naudoti normaliai su Ultimate Multisite. Pavyzdžiai – _Smush_, _Forminator_, _Defender_ ir kitų.
