# Administrator Panel Personalizavimas PRO Temu su Jautra {#customizing-your-admin-panel-with-pro-theme}

Daugeli atvejų klientų darbui mes turime nustatyti, kad norime suteikti savo klientams pritaikytą patirtį WordPress administravimo panelioje. Jei ieškote profesionalaus vaizdo ir jutimo, PRO Theme gali jums padėti.

PRO Theme yra administravimo temai, kurį pateikiamas, kad padėtum jūsų klientų Dashboard personalizuoti. Jis taip pat yra Ultimate Multisite premium plugino pagrindinė dalis.

Po įdarymo, eikite į savo _Plugins_ puslapį ir paspauskite „Aktyvuoti“ (Activate) jūsų tinklu.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Aktyvuokite temą visame tinklu savo Plugins puslapyje._

_**PASTABA:**_ _Jei jis įdarytumėte tik, jis pasirodys jūsų klientams kaip pasirinkimas, kurį galima pažymėti, kad tema būtų nustatoma tik šiam vartotojui._

Dabar turite visiškai pernaujatiną administravimo interfeis! Kiekviename puslapyje istautas naujas sąrašas kairėje skiltyje vadinamas PRO Theme.

<!-- Screenshot unavailable: PRO Theme redesigned admin panel interface -->
