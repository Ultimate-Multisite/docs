# Jūsų klientų svetainės (v2) {#your-clients-websites-v2}

Registruotinoje jūsų tinkluose, jūsų klientai gali pasiekti savo dashboardą.

Be **Account** puslapio, kur jie gali pamatyti svarbų informacijų dėl savo mokėjimų, nuskaitų, domenų, planų apribojimų ir kitų dalykų, jie taip pat turi prieigą prie svetainės **My Sites**, kurje jie gali pamatyti ir valdyti visus savo svetainis. Visi, ką reikia sukleisti, tai paspausti ant My Sites savo dashboardo.

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

Jums bus parodyti visi jūsų svetainiai, su **Manage** mygtuku tiesiog po kiekvieno iš jų. Jei ieškinsite ant jo, jie bus nukreipti į šio specifinio svetainės **dashboardą**.

Jie galite pamatyti, kuris iš jūsų svetaini yra **pagrindinis**, jei planai leidžia turėti daugiau nei vieną svetainį.

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

Štai visas svetainės redaguojimo interfeiso vaizdas:

![Site edit full interface](/img/admin/site-edit-full.png)

Jei jie nori pakeisti pagrindinį svetainį, jie turi eiti į Account puslapį paspaudus meniu **Account**.

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

Jie taip pat gali pamatyti ir redaguoti klientų duomenis:

![Customer edit interface](/img/admin/customer-edit.png)

Tada jie turėtų paspausti **Change Default Site**.

![Change Default Site button on account page](/img/account-page/change-default-site.png)

Pasirodys pop-up langas. Paspauskite išskirtinį sąrašą ir pasirinkite, kurį svetainį norite padaryti pagrindiniu svetaini.

![Change Default Site dropdown selection](/img/account-page/change-default-site-dropdown.png)

Tada, paspaudę **Change Default Site**, jūsų darbai yra atlikti!

![Patvirtinimas pagrindinio vietos keitimo](/img/account-page/change-default-site-confirm.png)
