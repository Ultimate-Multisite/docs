# Kaip pirmą kartą pasiekti savo paskyros puslapį? {#how-can-i-access-my-account-page-for-the-first-time}

Sveiki! Šis Ultimate Multisite!

Pasiekit savo paskyros puslapį per tą linką, kurį gauta el. paštu po pirkimo mūsų plugino:

<!-- Screenshot unavailable: Account page email with access link -->

Naudokite el. pašto adresą, kurį pateikėte pirkimo metu, ir slaptažodį, kuris buvo nurodytas jūsų el. pašte.

<!-- Screenshot unavailable: Login page with email and password fields -->

Kadangi tai jūsų pirmasis laikas prisijungti į šią paskyrą, jums bus pateikta siuntimas pakeisti slaptažodį. Pakeiskite jį ir spauskite mygtuką _Save_ (Įsiminti).

<!-- Screenshot unavailable: Change password form with Save button -->

Jūsų paskyros puslapyje galite valdyti visus savo paskyros duomenis, tokius kaip skaičiavimas, atnaujinimai, atšaukimai ir daug kitų dalykų.
