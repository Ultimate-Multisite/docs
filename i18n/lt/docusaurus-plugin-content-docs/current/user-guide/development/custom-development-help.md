# Kaip pritaikyti savo projektą su Codeable?

Kai kurtai mums reikalinga puiki individualizavimas, kad mūsų tinklo srautai būtų tikėtiai taip, kaip mes norime. Jei tai yra tas, kas jums reikia, mes siunčiame [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), vietą, kur galite rasti patirties turiančius programavimus specialistus su Ultimate Multisite patirtimi.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) yra platforma, diegti klientus su kvalifikuoti WordPress freelancerius, kurie gali dirbti projektų bet kokio dydžio. Jie turi atsargų pateikimo procesą, kad užtikrintų, kad būsite su specialistais, tiesiogiai kvalifikuoti jūsų planuotam užduobiui.

### Kaip nustatyti projektą Codeable?

Pirmiausia, eikite į [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) ir paspauskite mygtuką **Start a Project** (Pradėti projektą).

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Dabar, kad pateiktumėte projektą, paspauskite laukimo kairėje **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

Vykdant „I need help with“ (Man reikia pagalbos su), pasirinkite **Customizing** (Pritaikymas):

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

Vykdant „My“ (Mano), pasirinkite **Plugin** (Platina):

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Tada paspauskite mygtuką **Continue** (Tęsti).

<!-- Screenshot unavailable: Codeable project form Continue button -->

Dabar atvykime laiku aprašyti savo projektą ir pateikti geriausi būdą, kurį galite. Pradėkite pateikti kuo daugiau informacijos!

<!-- Screenshot unavailable: Codeable project description form -->

Atsakykite šiuos klausimus, kad padėtumėte nustatyti kainą.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Po toje, jei turėtum naujieną [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) paskyrą, kur ją turite, sukurkite ją ir paspauskite mygtuką **Publish My Project** (Pateikti savo projektą).

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

Kai pateiksite savo [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) projektą, platforma sujungia jus su programstais, kurie atitinka jūsų projekto reikalaujamą įgūdžių paketą.

Šis proces padės rasti tinkamą žmogų jūsų darbu, kuris turėtų patirtį su WordPress ir Ultimate Multisite!

Codeable minimalus laipsnis yra 70–120 USD per valandą. Estimatai keisim priklausomai nuo darbo apribojimo, sudėtingumo ir projektu skubumo.

Kai gausite savo estimatą, galėsite įsiprenasti programstui, kuris labiausiai tinka jūsų poreikiams, ir kartu pradėti projektą!
