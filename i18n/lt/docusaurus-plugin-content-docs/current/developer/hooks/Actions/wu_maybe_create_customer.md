# Veiksmas: wu_maybe_create_customer

Leidžia įskiepių kūrėjams atlikti papildomus veiksmus, kai klientas pridedamas.

Štai čia pridedame kabliukus, skirtus kliento-&gt;naudotojo pridėjimui ir prie pagrindinės svetainės, pavyzdžiui.

## Parametrai {#parameters}

| Pavadinimas | Tipas | Aprašymas |
|------|------|-------------|
| $customer | `\Customer` | Klientas, kuris galbūt buvo sukurtas. |
| $checkout | `\Checkout` | Dabartinė checkout klasė. |
