# Filtras: wu_view_override_replaceable_views

Leidžia kūrėjams pridėti papildomų aplankų prie pakeičiamųjų sąrašo.

Būkite atsargūs, nes leidus papildomus aplankus gali būti įkeltos pasenusios kopijos vietoje Ultimate Multisite versijų.

## Parametrai

| Pavadinimas | Tipas | Aprašymas |
|------|------|-------------|
| $replaceable_views | `array` | Leidžiamų aplankų sąrašas. |
