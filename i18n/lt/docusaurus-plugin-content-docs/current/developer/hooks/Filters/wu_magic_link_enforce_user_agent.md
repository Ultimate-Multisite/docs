# Filtras: wu_magic_link_enforce_user_agent

Filtruoja, ar taikyti naudotojo agento patvirtinimą.

Nustatykite į false, kad žetonai veiktų skirtingose naršyklėse / įrenginiuose. Tai sumažina saugumą, bet padidina patogumą.

## Parametrai

| Pavadinimas | Tipas | Aprašymas |
|------|------|-------------|
| $enforce | `bool` | Ar taikyti naudotojo agento atitikimą. |
