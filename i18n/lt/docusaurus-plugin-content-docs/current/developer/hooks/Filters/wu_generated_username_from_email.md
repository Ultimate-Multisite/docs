# Filtras: wu_generated_username_from_email {#filter-wugeneratedusernamefromemail}

Filtruoti sugeneruotą kliento naudotojo vardą.

## Parametrai {#parameters}

| Pavadinimas | Tipas | Aprašymas |
|------|------|-------------|
| $username | `string` | Sugeneruotas naudotojo vardas. |
| $email | `string` | Naujo kliento el. pašto adresas. |
| $new_user_args | `array` | Naujo naudotojo argumentų masyvas, galbūt įskaitant vardą ir pavardę. |
| $suffix | `string` | Prie naudotojo vardo pridedama eilutė, kad jis būtų unikalus. |
