# Material WP {#material-wp}

WordPress administravimo sąsaja jau seniai yra tokia pati. Bet ar kada susimąstėte, kaip atrodytų administravimo sąsajos Material Design versija? 

Material WP yra administravimo tema, sukurta suteikti jums tokią išvaizdą ir pojūtį, padedanti jūsų klientams pasiūlyti labiau pritaikytą Dashboard.

## Funkcijos {#features}

- Pasirinktinio CSS palaikymas
- Paprasti stiliaus nustatymai
- Meniu perrikiavimas
- Paprastas importavimas / eksportavimas

## Dokumentacija {#documentation}

**Administravimo skydelio pritaikymas naudojant Material WP**

## Diegimas {#installation}

1. Įkelkite 'material-wp' į '/wp-content/plugins/' katalogą
2. Aktyvuokite įskiepį per WordPress meniu 'Įskiepiai'
