# Fluent Forms

Šis priedas leidžia pridėti Fluent Forms apribojimus Multisite Ultimate produktuose, narystėse arba svetainėse.
Fluent Forms naudoja pasirinktines lenteles vietoje pasirinktinių įrašų tipų, todėl įprastos apribojimų parinktys su Fluent Forms neveikia.

## Funkcijos {#features}

- Pridėkite apribojimus Fluent Forms arba Conversational Fluent Forms
- Klientai gauna draugišką pranešimą, raginantį juos atnaujinti savo planą, kai pasiekia limitą
