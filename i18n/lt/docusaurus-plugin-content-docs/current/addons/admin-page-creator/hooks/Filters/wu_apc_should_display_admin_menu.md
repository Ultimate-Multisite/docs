## Parametrai {#parameters}

| Pavadinimas | Tipas | Aprašymas |
|------|------|-------------|
| $should_display_admin_menu | `bool` | Ar reikia rodyti puslapius. |
| $network_active | `bool` | Patikrinti, ar plugin yra aktyvus tinklo administravime. |
