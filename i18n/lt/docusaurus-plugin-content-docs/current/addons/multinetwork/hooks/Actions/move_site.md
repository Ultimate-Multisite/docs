# Veiksmas: move_site

Suveikia po to, kai svetainė perkeliama į naują tinklą.

## Parametrai {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $site_id | `int` | Perkeltos svetainės ID. |
| $old_network_id | `int` | Pradinio svetainės tinklo ID. |
| $new_network_id | `int` | Tinklo, į kurį svetainė perkelta, ID. |
