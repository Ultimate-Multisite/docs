# VAT

Multisite Ultimate: Europos VAT

Prideda papildomų funkcijų ir įrankių prie Multisite Ultimate numatytojo mokesčių tvarkymo, kad būtų geriau palaikomas Europos VAT.
Automatiškai prideda ES VAT tarifus į mokesčių tarifų puslapį iš

## Diegimas {#installation}

1. Įkelkite 'ultimate-multisite-vat' į '/wp-content/plugins/' katalogą
2. Aktyvuokite įskiepį per 'Įskiepiai' meniu WordPress
3. Eikite į Ultimate Multisite > Nustatymai > Mokesčiai ir įjunkite ES VAT.
4. Spustelėkite Mokesčių tarifai, tada apačioje Atnaujinti ES VAT tarifus ir Išsaugoti mokesčių tarifus.
5. Atlikta!
