# Palaikymo agentai

Valdyti visą tinklą vienam nėra lengva užduotis. Kai jūsų tinklas pradės augti, pradėsite matyti didėjantį palaikymo poreikį. Gera mintis bus pasitelkti žmonių, kurie padės sumažinti jūsų darbo krūvį. Tačiau suteikti visišką prieigą / leidimus yra rizikinga ir gali pakenkti jūsų tinklui. Būtent čia praverčia šis palaikymo agentų priedas.

Šis priedas leidžia sukurti naujo tipo WordPress naudotoją, vadinamą palaikymo agentu, ir suteikia jums visišką kontrolę, kokius leidimus tas naudotojas turi. Suteikti leidimą taip paprasta, kaip įjungti perjungiklį, o jei nuspręsite jį pašalinti, tereikia jį išjungti.

## Funkcijos {#features}

- Išsamus WordPress ir Ultimate Multisite leidimų sąrašas
- Lengvai įjunkite ir išjunkite prieigą
- Kurkite neribotą agentų paskyrų skaičių

## Diegimas {#installation}

1. Įkelkite 'ultimate-multisite-support-agents' į '/wp-content/plugins/' katalogą
2. Aktyvinkite įskiepį per WordPress meniu 'Įskiepiai'
3. Vykdykite nuoseklų vedlį, kad nustatytumėte įskiepį
