# Dokumentace Ultimate Multisite

Vítejte v dokumentaci Ultimate Multisite. Ultimate Multisite transformuje WordPress Multisite na plošnou platformu Website as a Service (WaaS).

## Sekce dokumentace {#documentation-sections}

### [Průvodce uživatele](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

Zde najdete vše, co potřebujete k nastavení, konfiguraci a správě vaší WaaS platformy. Pokrývá instalaci, platební brány, mapování domén, správu klientů a další.

### [Dokumentace pro vývojáře](/developer/hooks) {#developer-documentation}

Technická reference pro vývojáře, kteří pracují s Ultimate Multisite. Zahrnuje přehled hooků, dokumentaci REST API a průvodce vývojem addonů.

### [Addony](/addons) {#addons}

Dokumentace pro placené addony (doplňkové pluginy), které rozšíří Ultimate Multisite o další funkce, jako je integrace s WooCommerce, analytika, prodej domén a další.

## Jak získat pomoc {#getting-help}

- [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) -- Zkuste otázky a sdílejte nápady
- [GitHub Issues](https://github.com/Ultimate-Multisite/ultimate-multisite/issues) -- Hlášení chyb
- [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) -- Stáhněte základní plugin
