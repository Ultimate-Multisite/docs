# Filtrační metoda: wu_wp_cli_enabled_sub_commands {#filter-wuwpclienabledsubcommands}

Filtruje, které podkómaní (sub\_commands) jsou pro tuto entitu povolena.

## Parametry {#parameters}

| Název | Typ | Popis |
|------|------|-------------|
| $sub\_commands | `array` | Výchozí podkómaní. |
| $command\_base | `string` | Základní slovo použité v příkazu hned po kořenovém. |
| $this | `\Base_Manager` | Instancí objektu. |
