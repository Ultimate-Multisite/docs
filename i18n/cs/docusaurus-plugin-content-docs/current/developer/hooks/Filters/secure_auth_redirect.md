# Filter: secure_auth_redirect {#filter-secureauthredirect}

Určuje, zda má být použito bezpečné přesměrování při ověřování.

## Parametry {#parameters}

| Název | Typ | Popis |
|------|------|-------------|
| $secure | `bool` | Zda má být použito bezpečné přesměrování při ověřování. Výchozí hodnota je false. |

### Od verze {#since}

- 3.1.0
