# Filter: wu_password_require_uppercase {#filter-wupasswordrequireuppercase}

Filtr určuje, zda je v pravidlech pro hesla vyžadováno použití velkých písmen.

## Parametry {#parameters}

| Název | Typ | Popis |
|------|------|-------------|
| $require | `bool` | Zda je vyžadováno použití velkých písmen. Výchozí hodnota je `true`, pokud jsou pravidla aktivní. |
| $defender_active | `bool` | Zda je aktivní plugin Defender Pro Strong Password. |

- 2.4.0
