# Filter: send_auth_cookies

Umožňuje zabránit, aby se autentizační cookies skutečně odeslaly klientovi.

## Parameters

| Name | Type | Popis |
|------|------|-------------|
| $send | `bool` | Určuje, zda se autentizační cookies odesílat mají klientovi. |
