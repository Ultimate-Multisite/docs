# Filtr: wu_password_require_special

Filtr určuje, zda je v pravidlech vyžadováno použití speciálních znaků v heslech.

## Parametry {#parameters}

| Název | Typ | Popis |
|------|------|-------------|
| $require | `bool` | Čím vyžadovat speciální znaky. Výchozí hodnota je true, když jsou pravidla vynucována. |
| $defender_active | `bool` | Čím je aktivní Defender Pro Strong Password. |
