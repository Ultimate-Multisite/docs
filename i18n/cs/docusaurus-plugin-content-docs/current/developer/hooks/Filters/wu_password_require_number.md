# Filter: wu_password_require_number

Filtr určuje, zda je v heslech vyžadováno použití číslic.

## Parametry {#parameters}

| Název | Typ | Popis |
|------|------|-------------|
| $require | `bool` | Zda je vyžadováno použití číslic. Výchozí hodnota je pravda, pokud jsou vynucovány pravidla. |
| $defender_active | `bool` | Zda je aktivní Defender Pro Strong Password. |
