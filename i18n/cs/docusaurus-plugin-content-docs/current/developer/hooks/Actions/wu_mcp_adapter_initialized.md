# Akce: wu_mcp_adapter_initialized

Spustí se po inicializaci MCP adaptéru.

Umožňuje jiným pluginům a tématům zaregistrovat vlastní schopnosti.

## Parametry

| Název | Typ | Popis |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | Instancí MCP adaptéru. |
