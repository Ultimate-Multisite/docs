# Akce: wu_site_created

Spouští se, když je vytvořeno nové místo (site) poprvé.

Nejsou spouštěny, pokud je místo duplikováno z šablonky.

## Parametry

| Název | Typ | Popis |
|------|------|-------------|
| $data | `array` | Objektová data, která budou uložena. |
| $site | `\Site` | Instancí objektu. |
