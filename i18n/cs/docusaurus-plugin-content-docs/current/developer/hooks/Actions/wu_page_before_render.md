# Akce: wu_page_before_render

Umožňuje vývojářům pluginů přidat další obsah před vykreslením stránky.

## Parametry {#parameters}

| Název | Typ | Popis |
|------|------|-------------|
| $page_id | `string` | ID této stránky. |
| $page | `object` | Objekt stránky. |

### Od verze {#since}

- 1.8.2
