# Kompatibilita Tutor Multisite {#tutor-multisite-compatibility}

Plugin pro WordPress, který přidává opravy kompatibility pro Tutor LMS v prostředí multisite.

## Popis {#description}

Tento plugin poskytuje základní opravy kompatibility pro spouštění Tutor LMS na síti WordPress multisite. Řídí uživatelskou registraci, správu instruktorů, limity kurzů a další funkce specifické pro multisite.

## Funkce {#features}

- **Správa tabulek databáze**: Automaticky detekuje a vytváří chybějící tabulky Tutor LMS, které selhávají kvůli omezením cizích klíčů multisite
- **Registrace uživatelů multisite**: Vlastní AJAX ovladače pro správu registrácie uživatelů napříč síti webů
- **Správa rolí instruktorů**: Automatické přiřazování a správa rolí instruktorů v prostředí multisite
- **Limity kurzů**: Integrace s Ultimate Multisite pro správu limitů při vytváření kurzů a funneli na základě plánů webu
- **Vlastní zobrazení v administraci**: Panel pro seznam a správu instruktorů, který je uvědomělen multisite
- **Administrativní nástroje**: Uživatelské rozhraní pro správu databáze pro administrátory sítě i webu

## Požadavky {#requirements}

- WordPress 5.0+
- PHP 7.4+
- Plugin Tutor LMS (automaticky vynuceno v hlavičce `Requires Plugins`)
- Instalace WordPress Multisite
- Plugin Ultimate Multisite (pro funkcionalitu limitů kurzů)

## Instalace {#installation}

1. Nahraj soubory pluginu do adresáře `/wp-content/plugins/tutor-multisite-compatibility/`
2. Ujistěte se, že je Tutor LMS nainstalován a aktivován
3. Aktivujte plugin v menu 'Plugins' v WordPressu
4. Plugin se automaticky integruje s Tutor LMS

- Zobrazení ID produktů kurzů

## Správa databáze {#database-management}

### Problém {#the-problem}

V prostředí WordPress multisite selhává vytváření tabulek Tutor LMS pro tabulky s omezeními cizích klíčů na tabulku uživatelů. Děje se to proto, že:

- Tabulky specifické pro web používají prefixy jako `wp_2_tutor_carts`
- Cizí klíče odkazují na `wp_2_users` (která neexistuje)
- Skutečná tabulka uživatelů je `wp_users` (sdílená napříč sítí)

### Řešení {#the-solution}

Tento plugin automaticky:

1. **Detekuje chybějící tabulky**: Kontroluje tabulky, které selhaly při vytváření kvůli omezením cizích klíčů
2. **Vytváří tabulky s правиlnými odkazy**: Používá správné jméno tabulky uživatelů (`wp_users` v multisite)
3. **Řídí více scénářů**:
   - Aktivace/reaktivace pluginu
   - Vytvoření nového webu v síti
   - Ruční kontrola tabulek přes administraci

### Dotyčné tabulky {#affected-tables}

- `tutor_coupon_usages` - Obsahuje cizí klíč na tabulku uživatelů
- `tutor_carts` - Obsahuje cizí klíč na tabulku uživatelů
- `tutor_notification_preferences` - Tabulka TutorPro s cizím klíčem na tabulku uživatelů

### Administrační rozhraní {#admin-interface}

Přístup k nástrojům pro správu databáze na:

- **Admin sítě**: Nastavení → Tutor Multisite DB
- **Admin webu**: Nástroje → Tutor Multisite DB

Rozhraní vám umožňuje:
- Zobrazit chybějící tabulky napříč všemi weby
- Ručně spustit vytváření tabulek
- Kontrolovat konfiguraci databáze
- Zobrazit podrobné stavové informace
