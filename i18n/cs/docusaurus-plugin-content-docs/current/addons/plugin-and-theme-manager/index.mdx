# Plugin & Theme Manager

Multisite Ultimate: Plugin a Theme Manager

Upravte nebo skryjte meta údaje (název, autor, miniatura a popis) nainstalovaných pluginů a témat. Vytvořte kategorie pro pluginy a témata, aby vaši uživatelé mohli filtrovat a přidat krásní vlastní stránku s pluginy do panelu uživatelů.

Pro použití pluginu stačí navigovat na stránky Pluginy nebo Témata v administraci sítě a kliknout na „Upravit informace o pluginu/tématu“.

## Instalace

1. Nahrajte 'ultimate-multisite-plugin-and-theme-manager' do adresáře '/wp-content/plugins/'
2. Aktivujte plugin v menu 'Plugins' v administraci sítě WordPress

## Manuál použití

Ultimate Multisite: Plugin a Theme Manager je výkonný **premium** doplněk Ultimate Multisite, který vám dává plnou kontrolu nad tím, jak vaši uživatelé sítě vidí a s jakým způsobem interagují s možnostmi pluginů a témat na vaší síti.

Tento doplněk slouží k tří účelům:

  1. Umožňuje vám kompletně změnit meta údaje jak pluginů, tak témat. Zahrnuje to název, popis, jméno autora a dokonce i miniatury!

  2. Umožňuje vám kategorizovat pluginy a témata, aby vaši uživatelé mohli snadno filtrovat.

  3. Dává vám možnost nahradit starou, nudnou stránku s pluginy novou stránkou podobnou tématům, s tříděním podle kategorií.

Po stažení a instalaci se na stránce Ultimate Multisite Settings objeví nová záložka – _Add-on Settings_.
