# Material WP

Administrační rozhraní WordPressu je už věky stejné. Ale někdy jste se divokali, jak by vypadalo rozhraní v designu Material Design?

Material WP je admin téma, které bylo vyvinuto, aby vám poskytlo ten vzhled a pocit, a pomáhá vám nabídnout vašim klientům více přizpůsobený Dashboard.

## Funkce {#features}

- Podpora vlastního CSS
- Snadné nastavení stylu
- Přerovnání menu
- Snadný import/export

## Dokumentace {#documentation}

**Přizpůsobení vašeho administrátorského panelu pomocí Material WP**

## Instalace {#installation}

1. Nahrajte 'material-wp' do adresáře '/wp-content/plugins/'
2. Aktivujte plugin pomocí menu 'Plugins' v WordPressu

## Manuál použití {#usage-guide}

Administrační rozhraní WordPressu je už věky stejné. Ale někdy jste se divokali, jak by vypadalo rozhraní v designu Material Design?
