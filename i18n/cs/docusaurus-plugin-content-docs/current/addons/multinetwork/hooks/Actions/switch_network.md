# Action: switch_network

Spustí se, když se změní aktuální kontext sítě.

## Parametry

| Název | Typ | Popis |
|------|------|-------------|
| $new_network_id | `int` | ID sítě, na kterou se přepíná. |
| $old_network_id | `int` | ID předchozí aktuální sítě. |
