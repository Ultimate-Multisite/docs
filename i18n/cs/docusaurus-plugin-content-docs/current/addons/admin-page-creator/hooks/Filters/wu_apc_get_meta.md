# Filter: wu_apc_get_meta {#filter-wuapcgetmeta}

Umožňuje vývojářům filtrovat finální hodnoty a přidávat nové zpracovatelské případy.

## Parametry {#parameters}

| Název | Typ | Popis |
|------|------|-------------|
| $default_value | `string` | Výchozí hodnota. |
| $meta_type | `string` | Typ meta údajů, které mají být získány. |
| $meta_key | `string` | Název meta pole. |

### Od {#since}

- 1.4.0

Novou výchozí hodnotu.
