# Filter: wu_apc_should_display_admin_menu {#filter-wuapcshoulddisplayadminmenu}

## Parametry {#parameters}

| Název | Typ | Popis |
|------|------|-------------|
| $should_display_admin_menu | `bool` | Zkontroluje, zda je nutné zobrazit stránky. |
| $network_active | `bool` | Zkontroluje, zda je plugin aktivní na úrovni sítě. |
