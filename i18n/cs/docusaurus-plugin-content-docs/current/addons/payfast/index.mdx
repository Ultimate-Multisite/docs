# PayFast {#payfast}

Multisite Ultimate: Payfast Gateway

Přidává PayFast – rychlou a bezpečnou online platební bránu v Jižní Africe – jako platební možnost pro Multisite Ultimate.

## Instalace {#installation}

1. Nahrajte 'ultimate-multisite-payfast' do adresáře '/wp-content/plugins/'
2. Aktivujte plugin v menu 'Plugins' v WordPressu
3. Postupujte podle průvodce krok za krokem pro nastavení pluginu

## Manuál použití {#usage-guide}

PayFast je přední online platební zpracovatel pro podniky v Jižní Africe. Je věnován pomoci online obchodníkům růstem tím, že zajišťuje bezproblémové a bezpečné transakce. Na základě požadavku uživatelů Ultimate Multisite jsme se rozhodli vytvořit add-on, který umožní integraci s PayFast. Tento článek ukazuje, jak nastavit PayFast jako vaši platební bránu, abyste mohli přijímat platby v vaší síti Ultimate Multisite.

Prvním krokem je instalace add-onu.

# Jak nainstalovat add-on Payfast: {#how-to-install-the-payfast-add-on}

Přejděte na stránku add-onů a vyberte add-on Ultimate Multisite: Payfast Gateway.

![Add-ons page](/img/admin/addons-page.png)

Otevře se okno add-onů. Klikněte na Install Now (Nainstalovat nyní).

![Add-ons page with install option](/img/admin/addons-page.png)

Po dokončení instalace budete přesměrováni na stránku pluginů. Zde stačí kliknout na Network Activate (Aktivovat síť) a add-on _Payfast_ bude aktivován pro vaši síť.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# Nastavení PayFast {#setting-up-payfast}

Po instalaci add-onu můžete nyní jít do nastavení Ultimate Multisite a začít konfigurovat PayFast.

Na stránce **Ultimate Multisite settings** najdete záložku "Payments" (Platby). Pod touto záložkou byste měli najít **"Payfast"** spolu s přepínači, který umožňuje zapnout nebo vypnout tuto platební bránu.

![Payment gateway settings](/img/config/settings-payment-gateways.png)Pokračujte a klikněte na přepínač, abyste ji aktivovali. Poté se rozbalí pole, které zobrazí různé pole, která je třeba vyplnit, jako jsou **Merchant ID, Merchant Key a PassPhrase.** Tyto informace získáte ze svého účtu **Payfast**

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Dalším krokem je přihlášení do vašeho účtu PayFast, abyste získali potřebné informace.

Po přihlášení uvidíte v rámci vašeho **Payfast** dashboardu okamžitě údaje, které potřebujete.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->Stačí jednoduše zkopírovat a vložit tyto údaje a vyplnit detaily ve vašich nastaveních Ultimate Multisite. Poté klikněte na tlačítko **"Save settings"** (Uložit nastavení), abyste změny uložili.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

To je to, vaše integrace platební brány PayFast je dokončena!
