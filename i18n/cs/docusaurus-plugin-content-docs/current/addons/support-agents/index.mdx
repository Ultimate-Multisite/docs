# Agenti podpory

Spravovat celou síť sám je náročný úkol. Jak vaše síť začne růst, uvidíte rostoucí poptávku po podpoře. Bude dobré, když si přivete lidi, kteří vám sníží zátěž. Nicméně, udělit plný přístup/oprávnění je riskantní a může ohrozit vaši síť. Zde vstupuje do hry tento addon pro Agenty podpory.

Tento addon vám umožňuje vytvořit nový typ uživatele WordPress nazvaný Agent podpory a dává vám plnou kontrolu nad oprávněními, která tento uživatel má. Udělit oprávnění je tak snadné, jako zapnout přepínač (toggle button), a vypnout ho, pokud se rozhodnete o jeho odstranění.

## Funkce

* Detailní seznam oprávnění WordPressu a Ultimate Multisite
* Snadné povolení a zakázání přístupu
* Vytváření neomezeného počtu účtů agentů

## Instalace

1. Nahrajte 'ultimate-multisite-support-agents' do adresáře '/wp-content/plugins/'.
2. Aktivujte plugin v menu 'Plugins' v WordPressu.
3. Postupujte podle průvodce (Wizard) krok za krokem, abyste plugin nastavili.
