# Fluent Forms

Tento add-on umožňuje přidat limity pro Fluent Forms na Multisite Ultimate v produktech, členstvích nebo na stránkách.
Fluent Forms používá vlastní tabulky namísto vlastních typů příspěvků, takže standardní možnosti omezení nefungují s Fluent Forms.

## Funkce

- Přidání limitů pro Fluent Forms nebo Conversational Fluent Forms
- Zákazníci dostanou přátelskou zprávu, která je vyzve k upgradu svého plánu, když dosáhnou limitu.
