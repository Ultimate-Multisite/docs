# Superdav AI Agent

Superdav AI Agent je pokročilý WordPress AI asistent, který rozšiřuje Gratis AI Agent o dovednosti zohledňující šablonu, vylepšené možnosti vyhledávání na internetu, funkce pro tvorbu webu a volitelné integrační automatizace. Verze 1.18.0 přidává nástroje pro čtení Google Calendar, mapování kontaktů a účastníků, brány pro lidské schválení, deduplikaci připomenutí, TextBee SMS oznámení, zřizování spravované služby Superdav a samostatně distribuovaný doprovodný balíček Advanced pro důvěryhodné vývojářské pracovní postupy.

## Klíčové funkce {#key-features}

- **Dovednosti zohledňující šablonu** — automaticky přizpůsobují pokyny a možnosti podle vaší aktivní WordPress šablony a nainstalovaných pluginů
- **Více poskytovatelů internetového vyhledávání** — vyberte si mezi Brave Search a Tavily pro bohatší výsledky vyhledávání a pokročilé možnosti rešerše
- **Kontaktní formuláře pro tvorbu webu** — vytvářejte plně funkční kontaktní formuláře přímo z chatovacího rozhraní pomocí nainstalovaného pluginu pro formuláře
- **Vylepšená tvorba webu** — zdokonalené vícefázové plány sestavení webu s lepším vyhledáváním pluginů a zotavením po chybách
- **Vestavěné dovednosti pro šablony** — specializované pokyny pro blokové šablony, klasické šablony, Kadence Blocks a Kadence Theme
- **Vylepšená integrace WooCommerce** — přepracováno tak, aby používalo nativní WooCommerce API pro lepší spolehlivost a kompatibilitu
- **Automatická detekce poskytovatelů** — automaticky detekuje a obnovuje dostupné pluginy a šablony při jejich aktivaci nebo deaktivaci
- **Generování vlastního loga** — generujte a vkládejte vlastní SVG loga s automatickou sanitizací bezpečnou pro jmenné prostory
- **Ověření kontrastu palety** — kontrolujte dvojice barev z hlediska souladu s přístupností WCAG před jejich použitím v šabloně
- **Objevování vylepšené fotografiemi** — nahrávejte fotografie během úvodního rozhovoru v Theme Builder, aby ovlivnily směr designu
- **Menu pro pohostinství** — generujte strukturované stránky jídelních a nápojových lístků pro restaurace, kavárny a podniky v pohostinství
- **Náhledy pro počítač a mobil** — prohlížejte si svůj design na počítačích a mobilních zařízeních během výběru směru designu
- **Scraping webů úrovně 1** — sd-ai-agent/site-scrape je nyní ve výchozím nastavení dostupný pro analýzu konkurenčních a inspiračních webů
- **Nástroje pro čtení Google Calendar** — čtou nakonfigurované kalendáře a události, aby automatizace mohly zohlednit kontext nadcházejícího harmonogramu
- **Mapování kontaktů a účastníků** — páruje účastníky kalendáře se známými uživateli a kontakty webu před generováním následné práce
- **Brány pro lidské schválení** — pozastavují citlivé automatizace ke kontrole před odesláním zpráv, obnovením práce nebo provedením změn
- **Záznamy připomenutí a deduplikace** — sledují připomenutí, která již byla odeslána, aby opakované běhy automatizace neupozornily stejného příjemce dvakrát
- **TextBee SMS oznámení** — odesílají nakonfigurovaná oznámení textovou zprávou přes TextBee, když je povolen poskytovatel SMS
- **Zřizování spravované služby Superdav** — připojuje podporované weby k hostovaným koncovým bodům Superdav s automatickým nastavením místo ruční konfigurace koncových bodů
- **Doprovodný balíček Advanced** — samostatně distribuované nástroje pro důvěryhodné vývojáře pro práci se souborovým systémem, databází, WP-CLI, REST dispečerem, tvorbou pluginů, git snapshoty, správou uživatelů a benchmarky

## Požadavky {#requirements}

- WordPress 6.0 nebo vyšší
- PHP 8.1 nebo vyšší
- WordPress Multisite (aktivovaný v síti) nebo instalace jednoho webu
- Aktivní plugin konektoru poskytovatele AI (Anthropic, OpenAI nebo kompatibilní)
- Gratis AI Agent v1.9.0 nebo vyšší (Superdav AI Agent rozšiřuje Gratis AI Agent)

## Instalace {#installation}

1. Nahrajte soubory pluginu do `/wp-content/plugins/superdav-ai-agent`
2. Aktivujte plugin prostřednictvím obrazovky **Pluginy** ve WordPress
3. Ujistěte se, že je aktivován také Gratis AI Agent
4. Přejděte na **Gratis AI Agent → Nastavení** a podle potřeby nakonfigurujte svého poskytovatele internetového vyhledávání, připojení ke spravované službě Superdav, Google Calendar nebo přihlašovací údaje TextBee

Balíček WordPress.org obsahuje základní funkce Superdav AI Agent. Doprovodný balíček Advanced je sestavován a distribuován samostatně pro prostředí, kde jsou výslovně nainstalovány a autorizovány důvěryhodné vývojářské nástroje.

Po aktivaci Superdav AI Agent automaticky vylepší vaši instalaci Gratis AI Agent o dovednosti zohledňující šablonu a další možnosti. Otevřete panel **Gratis AI Agent** v postranním panelu administrace WordPress a začněte používat nové funkce.

**Příklad pokynů:**

```
Přidej kontaktní formulář na stránku Kontakt.
```


```
Zkontroluj zítřejší nakonfigurované události Google Calendar a připrav návrhy připomenutí pro účastníky.
```

```
Pošli SMS připomenutí poté, co zprávu schválím.
```

## Dovednosti zohledňující šablonu {#theme-aware-skills}

Superdav AI Agent automaticky detekuje vaši aktivní šablonu a nainstalované pluginy a načte odpovídající dovednost zohledňující šablonu do kontextu asistenta. Dostupné dovednosti zahrnují:

| Dovednost | Vztahuje se na |
|---|---|
| **Blokové šablony** | Šablony používající blokovou architekturu WordPress (Full Site Editing) |
| **Klasické šablony** | Tradiční WordPress šablony používající PHP šablony a klasický editor |
| **Kadence Blocks** | Weby s aktivním pluginem Kadence Blocks |
| **Kadence Theme** | Weby se šablonou Kadence jako aktivní šablonou |

Podrobné pokyny ke každé dovednosti najdete v části [Dovednosti zohledňující šablonu](../../user-guide/configuration/theme-aware-skills).

## Poskytovatelé internetového vyhledávání {#internet-search-providers}

Superdav AI Agent v1.10.0 přidává **Tavily** jako alternativu k Brave Search pro dotazy internetového vyhledávání. Oba poskytovatelé nabízejí výsledky vyhledávání v reálném čase a lze je nakonfigurovat v **Gratis AI Agent → Nastavení → Pokročilé**.

| Poskytovatel | Funkce | Nastavení |
|---|---|---|
| **Brave Search** | Rychlé výsledky vyhledávání se zaměřením na soukromí | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Bohatší výsledky vyhledávání, pokročilé možnosti průzkumu | [Tavily API](https://tavily.com/) |

Pokyny ke konfiguraci najdete v části [Internetové vyhledávání](../../user-guide/configuration/internet-search).

## Pracovní postupy kalendáře, schvalování a oznámení {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 rozšiřuje automatizační pracovní postupy nad rámec úloh souvisejících s obsahem a tvorbou webů:

- **Nástroje Google Calendar** umožňují agentovi číst nakonfigurované kalendáře a podrobnosti událostí, když jsou k dispozici přihlašovací údaje kalendáře.
- **Pomocníci pro kontakty a účastníky** mapují účastníky událostí na uživatele WordPress nebo známé kontakty před přípravou připomenutí nebo následných úloh.
- **Brány lidského schválení** umožňují automatizaci pozastavit, dokud oprávněný uživatel nezkontroluje navrhovanou akci.
- **Záznamy připomenutí** zabraňují duplicitním připomenutím z kalendáře nebo oznámením, když je automatizace opakována nebo opakovaně plánována.
- **Podpora TextBee SMS** umožňuje oznámení textovými zprávami, když je nakonfigurován poskytovatel TextBee.

Používejte tyto nástroje pro asistenci zohledňující harmonogram, následnou komunikaci s účastníky a pracovní postupy připomenutí, které před odesláním citlivých zpráv stále vyžadují lidskou kontrolu.

## Kontaktní formuláře Tvůrce webu {#site-builder-contact-forms}

Podrobné pokyny najdete v části [Kontaktní formuláře Tvůrce webu](../../user-guide/administration/site-builder-contact-forms).

## Konfigurace {#configuration}

Všechna nastavení jsou v administraci WordPress v části **Gratis AI Agent → Nastavení**. Superdav AI Agent přidává následující možnosti konfigurace:

| Nastavení | Popis |
|---|---|
| **Poskytovatel internetového vyhledávání** | Vyberte mezi Brave Search a Tavily (volitelné) |
| **Tavily API Key** | API klíč pro internetové vyhledávání Tavily (pokud je vybráno Tavily) |
| **Spravovaná služba Superdav** | Připojte podporované weby k hostované službě Superdav s automatickým zřizováním endpointů |
| **Přihlašovací údaje Google Calendar** | Nakonfigurujte přístup ke kalendáři, aby agent mohl číst kalendáře a události |
| **Poskytovatel TextBee SMS** | Nakonfigurujte přihlašovací údaje TextBee pro schválené pracovní postupy SMS oznámení |

## Pokročilý doprovodný balíček {#advanced-companion-package}

Superdav AI Agent v1.18.0 přidává samostatně distribuovaný pokročilý doprovodný balíček pro důvěryhodné vývojářské pracovní postupy. Tyto možnosti nejsou součástí základního balíčku WordPress.org. Nainstalujte doprovodný balíček pouze na webech, kde administrátoři výslovně důvěřují agentovi, že může přistupovat k vývojářským nástrojům.

Pokročilé nástroje zahrnují operace se souborovým systémem, přístup k databázi, WP-CLI, pracovní postupy REST dispečera, nástroje pro tvorbu pluginů, snímky git, pomocníky pro správu uživatelů a benchmarkovací nástroje. Omezte tyto nástroje na důvěryhodné administrátory a před jejich povolením na produkčních webech zkontrolujte konfiguraci schvalovacích bran.

## Opravy chyb ve v1.10.0 {#bug-fixes-in-v1100}

- Opravena schopnost `navigate-to`, která na některých stránkách administrace způsobovala nekonečné smyčky opětovného načítání
- Opravena schopnost `list-posts`, která správně nepřeváděla názvy kategorií a tagů na slugy
- Opraveny příkazy WP-CLI, kterým po předchozím refaktoringu chyběly aliasy jmenného prostoru
- Opravena automatizace událostí, aby korektně zpracovávala weby, kde tabulky automatizace ještě nejsou vytvořeny
- Opravena schopnost `memory-save`, aby v builderu systémových instrukcí používala správný prefix jmenného prostoru
- Opraveno správné zabalení skalárních výsledků nástrojů před jejich vrácením AI
- Opraveno správné zpracování staršího formátu klíče schopnosti ve statistikách využití při upgradu ze starších verzí
