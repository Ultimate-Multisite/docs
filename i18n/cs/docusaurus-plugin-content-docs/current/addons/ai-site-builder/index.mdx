# AI Site Builder

ultimate-multisite-ai-site-builder

## Požadavky {#requirements}

- WordPress 5.3 nebo vyšší
- PHP 7.4 nebo vyšší
- Multisite Ultimate plugin (aktivní)

## Instalace {#installation}

1. Nahrání souborů addonu do adresáře `/wp-content/plugins/`
2. Aktivujte plugin pomocí menu „Plugins“ v WordPressu
3. Konfigurujte nastavení addonu v administraci sítě

## Vývoj {#development}

### Nastavení {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Struktura projektu {#project-structure}

```
ultimate-multisite-ai-site-builder/
├── inc/                          # PHP třídy a vložení
│   ├── class-multisite-ultimate-updater.php  # Handler pro aktualizace
│   └── class-ai_site_builder-main.php             # Hlavní funkcionalita
├── tests/                        # Unit testy
│   ├── bootstrap.php            # Bootstrap pro testy
│   ├── class-*-base.php         # Základní třída pro testy
│   └── class-*-test.php         # Testovací případy
├── views/                       # Soubory šablon
├── assets/                      # Statické zdroje
│   ├── css/                     # Stylové soubory
│   ├── js/                      # JavaScript soubory
│   └── images/                  # Obrázky
├── lang/                        # Jazykové soubory
├── composer.json                # PHP závislosti
├── package.json                 # Závislosti Node.js
├── phpunit.xml.dist             # Konfigurace PHPUnit
├── .phpcs.xml.dist              # Konfigurace standardů kódu
└── rector.php                   # Pravidla modernizace kódu
```

### Testování {#testing}

Addon obsahuje komplexní rámec pro testování:

- **Unit Tests**: Testy založené na PHPUnit s testovacím balíkem WordPressu
- **Base Test Class**: Poskytuje pomocné metody pro běžné testovací scénáře
- **Test Coverage**: Konfigurováno pro sledování pokrytí kódu
- **Multisite Testing**: Testy běží v prostředí Multisite

### Standardy kódu {#code-standards}

- **WordPress Coding Standards**: Udržováno pomocí PHPCS
- **PHP 7.4+ Compatibility**: Podporuje moderní funkce PHP
- **Static Analysis**: Integrace PHPStan pro kontrolu typů
- **Code Modernization**: Pravidla Rectoru pro aktualizace PHP

### Systém sestavení {#build-system}

- **Asset Processing**: Minifikace souborů CSS/JS
- **Translation**: Generování souborů POT
- **Packaging**: Vytváření archivu pro distribuci
- **Development Mode**: Neminifikované zdroje pro ladění

## Konfigurace {#configuration}

Addon podporuje různé možnosti konfigurace, které jsou dostupné prostřednictvím rozhraní administrace sítě.

## Hooks a filtry {#hooks-and-filters}

### Akce (Actions) {#actions}

- `ultimate-multisite-ai-site-builder_init` - Spustí se po inicializaci addonu
- `ultimate-multisite-ai-site-builder_loaded` - Spustí se po načtení všech souborů addonu

### Filtry (Filters) {#filters}

- `ultimate-multisite-ai-site-builder_settings` - Změna nastavení addonu
- `ultimate-multisite-ai-site-builder_enabled` - Přepsání stavu povolení/zakázání addonu
