# Známé nekompatibility s jinými pluginy

Po instalaci a aktivaci Ultimate Multisite se můžete setkat s problémy způsobenými dosud nevyřešenými nekompatibilitami s jinými pluginy.

Aktivně pracujeme na tom, abychom tento seznam s každou novou verzí zkracovali. Pokud je však plugin uveden v tomto seznamu, možná ho budete muset deaktivovat, aby vaše síť Ultimate Multisite fungovala správně.

Zde je seznam těchto pluginů a co byste měli v daných případech udělat:

**Některé pluginy od WPMU DEV** Některé pluginy, které WPMU DEV _již neudržuje_, způsobují zastavení registračního procesu Ultimate Multisite v posledním kroku (uživatel nemůže vytvořit účet). Zde je seznam pluginů, které tyto problémy způsobují:

  * Signup Language: _Ultimate Multisite nabízí doplněk se stejnou funkcionalitou, který se jmenuje Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Tato funkce je součástí Ultimate Multisite, takže není potřeba žádný další plugin._

  * New Blog Templates: _Tato funkce je součástí Ultimate Multisite, takže není potřeba žádný další plugin._

  * Set Password on Multisite Blog Creation: _Tato funkce je součástí Ultimate Multisite, takže není potřeba žádný další plugin._

  * Signup TOS: _Tato funkce je součástí Ultimate Multisite, takže není potřeba žádný další plugin._

  * Anti-Splog: I když v méně rozsáhlé podobě, i tato funkce je _součástí Ultimate Multisite._

**Poznámka**: Všechny ostatní pluginy od WPMU Dev můžete používat spolu s Ultimate Multisite bez problémů. Například _Smush_, _Forminator_, _Defender_ a další.
