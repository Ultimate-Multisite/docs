# Jak se poprvé přihlásím na stránku svého účtu?

Vítejte v Ultimate Multisite!

Na stránku svého účtu se dostanete přes odkaz, který vám přišel e-mailem po zakoupení pluginu:

<!-- Screenshot unavailable: Account page email with access link -->

Použijte e-mail, který jste zadali při nákupu, a heslo, které jste obdrželi e-mailem.

<!-- Screenshot unavailable: Login page with email and password fields -->

Protože se do tohoto účtu přihlašujete poprvé, budete vyzváni ke změně hesla. Změňte ho a klikněte na tlačítko _Uložit_.

<!-- Screenshot unavailable: Change password form with Save button -->

Na stránce svého účtu můžete spravovat všechny údaje o účtu, jako jsou fakturace, obnovení předplatného, zrušení a mnoho dalšího.
