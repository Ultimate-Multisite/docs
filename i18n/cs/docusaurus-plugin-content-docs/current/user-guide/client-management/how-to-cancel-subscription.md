# Jak mohu zrušit své předplatné?

Pokud máte licenci Ultimate Multisite, můžete kdykoli zrušit její obnovování. Stačí postupovat podle následujících kroků:


<!-- Screenshot unavailable: Account page email with access link -->

Použijte e-mail, který jste zadali při nákupu, a své heslo.

<!-- Screenshot unavailable: Login page with email and password fields -->

Toto je úvodní obrazovka vašeho účtu:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

Pro zrušení předplatného přejděte v menu vlevo na _Renewing & Billing._

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

Klikněte na šipku vpravo, čímž otevřete postranní okno. Poté vyberte možnost, která zruší **automatické obnovování**.

Systém vám zobrazí potvrzovací zprávu.

Pokud jste si jisti, že chcete zrušit předplatné, stačí kliknout na tlačítko _**Cancel Renewals**_. Po této akci budete požádáni o vyplnění krátkého dotazníku.

Hotovo! Vaše předplatné se nebude automaticky obnovovat.

**Platný licenční klíč budete mít až do vypršení vašeho předplatného**. V případě, že budete chtít předplatné znovu aktivovat, budete to muset udělat ručně.
