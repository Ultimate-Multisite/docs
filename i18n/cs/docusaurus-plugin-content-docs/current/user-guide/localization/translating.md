# Překlady Ultimate Multisite

Ultimate Multisite je vyvíjen v angličtině, ale plugin je plně přeložitelný a příspěvky k překladům vítáme.

## Překlady základního pluginu

Základní plugin Ultimate Multisite se překládá prostřednictvím oficiální **překladatelské platformy WordPress.org**:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

Jedná se o standardní komunitní překladatelský systém WordPressu. Když překlady dosáhnou dostatečného počtu schválení, jsou automaticky distribuovány všem uživatelům Ultimate Multisite prostřednictvím aktualizací pluginu ve WordPressu.

**Potřebujeme vaši pomoc!** Pokud mluvíte jiným jazykem než anglicky, zvažte prosím přispění k překladům. Pomůže i překlad několika řetězců. Jak začít:

1. Navštivte [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)
2. Vyberte svůj jazyk
3. Klikněte na „Contribute Translation"
4. Budete potřebovat bezplatný účet na WordPress.org

Pokud byste se chtěli stát editorem překladů pro svůj jazyk (s možností schvalovat překlady), ozvěte se nám přes [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions).

## Překlady doplňků

Doplňkové pluginy se překládají samostatně na:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Překlady doplňků jsou momentálně generovány pomocí AI jako výchozí bod. Pokrývají všechny podporované jazyky, ale mohou obsahovat neobratné formulace nebo nepřesnosti. Uvítáme, když je lidští překladatelé zkontrolují a navrhnou vylepšení, aby zněly přirozeněji a přesněji.

## Lokální překlady

Pokud dáváte přednost vytváření překladů lokálně pro vlastní projekt místo přispívání do hlavního repozitáře, máte dvě možnosti:

### Použití Poedit

Můžete použít desktopovou aplikaci **[Poedit](https://poedit.net)** k překladu souboru `.pot`:

1. Stáhněte si Poedit z [poedit.net](https://poedit.net)
2. Otevřete soubor `.pot` z `wp-content/plugins/ultimate-multisite/lang/`
3. Přeložte řetězce v rozhraní Poedit
4. Uložte -- Poedit vygeneruje soubory `.po` a `.mo`
5. Výstupní soubory umístěte do složky `lang`

### Použití Loco Translate

WordPress plugin **[Loco Translate](https://wordpress.org/plugins/loco-translate/)** vám umožňuje překládat přímo z administračního panelu bez jakýchkoli desktopových nástrojů. Postará se za vás o vytvoření a kompilaci překladových souborů.
