# Jak získat pomoc s vlastním vývojem od Codeable?

Někdy opravdu potřebujeme skvělé úpravy, aby naše síťové procesy fungovaly přesně tak, jak chceme. Pokud přesně tohle hledáte, doporučujeme [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) – místo, kde najdete zkušené vývojáře se znalostí Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) je platforma, která propojuje zákazníky s kvalifikovanými WordPress freelance vývojáři schopnými pracovat na projektech jakékoli velikosti. Mají důkladný výběrový proces, který zajistí, že vás spojí s odborníky přímo kvalifikovanými pro úkol, který máte na mysli.

### Jak vytvořit projekt v Codeable?

Nejprve přejděte na [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) a klikněte na tlačítko **Start a Project**.

Pro odeslání projektu klikněte vlevo na **Start a Project**.

V „I need help with" vyberte **Customizing**:

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

V „My" vyberte **Plugin**:

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Poté klikněte na tlačítko **Continue**.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Nyní je čas co nejlépe popsat váš projekt a vaše potřeby. Snažte se uvést co nejvíce informací!

<!-- Screenshot unavailable: Codeable project description form -->

Odpovězte na následující otázky, které pomohou s určením ceny.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Poté, pokud ještě nemáte účet na [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), vytvořte si ho a klikněte na tlačítko **Publish My Project**.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

Jakmile svůj projekt na [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) publikujete, platforma vás spojí s vývojáři, kteří odpovídají požadovaným dovednostem pro váš projekt.

Tento proces vám pomůže najít správnou osobu pro vaši práci, která má zkušenosti s WordPress a Ultimate Multisite!

Minimální sazba Codeable je 70–120 USD za hodinu. Odhady se budou lišit podle rozsahu práce, složitosti a naléhavosti projektu.

Jakmile obdržíte svůj odhad, budete moci najmout vývojáře, který nejlépe vyhovuje vašim potřebám, a společně odstartovat váš projekt!
