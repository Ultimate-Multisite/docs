# Vkládání reklam do vaší sítě pomocí Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector je **bezplatný doplněk**, který vám umožňuje vkládat reklamy do vašich plánů přímo z jejich editační stránky.

Po instalaci Ultimate Multisite: Ad Injector přejděte na stránku _Pluginy_ a aktivujte ho.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Klikněte na tlačítko Aktivovat pro celou síť_

Jakmile je doplněk aktivní, v meta boxu _Pokročilé možnosti_ na editační stránce plánu se objeví nová možnost _Nastavení reklam_.

![Nastavení reklam na editační stránce plánu](/img/config/product-edit-full.png)

_Nyní můžete upravovat nastavení reklam pro každý z vašich plánů_

Na této záložce můžete přidat reklamní kódy pro příspěvky a stránky ve front-endu, před obsahem i za ním.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Po vložení reklamního kódu se reklama zobrazí na příspěvcích a stránkách daného plánu._

Reklamy můžete také zobrazovat v záhlaví administračních stránek pro uživatele daného plánu.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Přejděte na možnost Reklamní kód v administraci, pokud chcete zobrazovat reklamy v back-endu vašich uživatelů_

### Nejen reklamy

Ultimate Multisite: Ad Injector můžete využít také k nabídce podpory pro vyšší tarify.

Stačí přejít na možnost _Reklamy v administraci_ a přidat HTML kód vašeho nástroje pro podporu.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Přidejte zákaznickou podporu ke konkrétním plánům_

Takto vás mohou lidé kontaktovat přímo ze své administrace!
