# Přizpůsobení administrace pomocí PRO Theme

Při práci pro klienty často potřebujeme připravit vlastní vzhled administrace WordPressu. Pokud hledáte profesionálnější vzhled a dojem, PRO Theme vám s tím může pomoci.

PRO Theme je téma pro administraci vyvinuté tak, aby vám usnadnilo přizpůsobení nástěnky pro vaše klienty. Je také součástí prémiových pluginů Ultimate Multisite.

Po instalaci přejděte na stránku _Pluginy_ a aktivujte ho pro celou síť.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Aktivujte ho pro celou síť na stránce Pluginy_

_**POZNÁMKA:**_ _Pokud ho pouze nainstalujete, zobrazí se vašim klientům jako volba, kterou mohou zaškrtnout a přidat téma pouze pro svůj účet._

Nyní máte administrační rozhraní kompletně přepracované! V levém sloupci se také objevila nová položka menu s názvem PRO Theme.
