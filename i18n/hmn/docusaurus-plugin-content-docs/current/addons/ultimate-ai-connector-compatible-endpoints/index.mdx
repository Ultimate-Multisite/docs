# Tus Ultimate AI Connector rau Cov Chaw Kawg Uas Sib Haum

Tus Ultimate AI Connector rau Cov Chaw Kawg Uas Sib Haum txuas cov AI provider APIs uas sib haum nrog OpenAI rau koj lub WordPress multisite network. Teeb tau ntau AI provider — txhua tus muaj nws tus kheej endpoint URL, API key, thiab default model — thiab xa requests nrog automatic fallback thaum ib tus provider siv tsis tau.

## Cov Nta Tseem Ceeb

- **Txhawb ntau tus provider**: Ntxiv tau ntau daim card rau AI provider, txhua daim muaj nws tus kheej endpoint URL, API key, thiab default model
- **Luag kom rov teeb kev txiav txim**: Teeb providers raws qhov tseem ceeb los ntawm kev siv tus grip handle ntawm txhua daim card; requests yuav siv providers raws li kev txiav txim ntawd
- **Automatic fallback routing**: Yog tus provider tseem ceeb siv tsis tau, requests yuav cia li fallback mus rau tus provider tom ntej uas tau teeb tseg
- **Kev teeb tsa ib leeg rau txhua tus provider**: Txhua tus provider raug teeb cais nrog nws tus kheej credentials thiab kev xaiv model
- **Settings UI raws card**: Ua los ntawm WordPress 6.9+ Card components uas ruaj khov (CardBody, CardHeader, CardDivider)
- **Cia li nthuav thaum ntxiv**: Daim card provider tshiab yuav nthuav cia li, kom koj sau tau cov ntsiab lus tam sim ntawd
- **Script cache busting**: Provider scripts raug cache-busted thaum hloov kho plugin kom paub tseeb tias version tshiab tshaj raug load

## Yam Yuav Tsum Muaj

- WordPress 5.3 lossis siab dua
- PHP 7.4 lossis siab dua
- Ultimate Multisite qhov txuas ntxiv (qhib siv)
- Ib qho API endpoint uas sib haum nrog OpenAI (host koj tus kheej lossis los ntawm lwm tus)

## Kev Nruab

1. Upload cov addon files mus rau koj phau `/wp-content/plugins/`
2. Qhib qhov txuas ntxiv los ntawm zaub mov 'Cov Txuas Ntxiv' hauv WordPress
3. Mus rau **Ultimate Multisite → AI Connector** kom teeb tsa addon

## Kev Teeb Tsa

### Kev Teeb Ntau Tus Provider

Lub vijtsam teeb tsa siv layout raws card uas txhua daim card sawv cev rau ib tus AI provider. Koj ntxiv tau ntau tus provider npaum li koj xav tau.

#### Ntxiv Ib Tus Provider

1. Mus rau **Ultimate Multisite → AI Connector** hauv network admin
2. Nias **Ntxiv Provider**
3. Ib daim provider card tshiab tshwm thiab nthuav cia li
4. Sau cov ntsiab lus provider:
   - **Provider Name** — ib daim label los txheeb tus provider no (piv txwv "OpenAI", "Mistral", "Local Ollama")
   - **Endpoint URL** — base URL ntawm API uas sib haum nrog OpenAI (piv txwv `https://api.openai.com/v1`)
   - **API Key** — koj tus API key rau tus provider no
   - **Default Model** — tus model identifier uas yuav siv raws default rau tus provider no (piv txwv `gpt-4o`, `mistral-small`)
5. Nias **Txuag Settings**

#### Tshem Ib Tus Provider

Nias lub pob **Tshem** ntawm ib daim provider card twg kom rho tawm nws. Txuag settings kom siv qhov hloov ntawd.

#### Rov Teeb Kev Txiav Txim Providers

Luag lub grip icon (⣿) nyob sab laug ntawm ib daim provider card twg kom rov teeb nws qhov kev txiav txim. Qhov kev txiav txim txiav txim siab fallback priority: requests raug xa mus rau thawj tus provider thiab fallback mus rau cov provider tom qab yog ib request ua tsis tiav.

### Fallback Routing

Thaum ib request mus rau tus provider tseem ceeb ua tsis tiav — vim timeout, rate limit, lossis service outage — tus connector yuav cia li sim dua nrog tus provider tom ntej hauv daim list. Qhov no txuas ntxiv nqis hauv daim list mus txog thaum ib tus provider ua tiav lossis tag nrho providers twb raug sim lawm.

| Qhov xwm txheej | Kev coj ua |
|----------|-----------|
| Tus provider tseem ceeb teb | Request raug pab raws li ib txwm |
| Tus provider tseem ceeb ua tsis tiav | Request raug sim dua nrog tus provider tom ntej |
| Tag nrho providers ua tsis tiav | Xa error rov qab rau tus hu |

Fallback yog pob tshab rau qhov txuas ntxiv lossis theme uas hu — lawv yuav tau txais ib qho response ua tiav lossis ib qho error kawg, txawm tau sim pes tsawg tus provider los xij.

## Kev Kho Teeb Meem

### Tsis muaj providers nyob hauv list

Yog daim provider list khoob tom qab txuag, xyuas kom meej tias:

- Koj twb nias **Txuag Settings** tom qab ntxiv providers lawm
- Koj lub browser tsis tau thaiv nplooj settings kom txuag tsis tau (xyuas browser console rau JavaScript errors)
- Qhov txuas ntxiv qhib siv thiab koj nyob rau ntawm network admin screen

### Requests ua tsis tiav rau tag nrho providers

- Tshawb xyuas tias txhua tus provider qhov endpoint URL mus txog tau ntawm koj server (tsis yog tsuas yog ntawm koj lub browser xwb)
- Xyuas tias txhua tus API key siv tau thiab tseem tsis tau tas sijhawm
- Saib tus provider qhov rate-limit thiab quota status hauv lawv dashboard
- Qhib WordPress debug logging (`WP_DEBUG_LOG`) kom khaws tau cov lus error ntxaws los ntawm tus connector

### Ib tus provider tsis raug siv rau fallback

Drag-to-reorder hloov fallback order. Xyuas kom tus provider uas koj xav kom txais fallback requests nyob rau qhov chaw yog hauv daim list thiab tias nws qhov endpoint URL thiab API key yog lawm.

### Daim provider card tshiab tsis nthuav cia li

Auto-expand yuav tsum muaj JavaScript qhib siv hauv koj lub browser. Yog daim card tsis nthuav thaum nias, xyuas browser console rau errors. Cov ntsiab lus hauv daim card yeej nkag tau tas li los ntawm kev nias card header txawm auto-expand tsis trigger los xij.
