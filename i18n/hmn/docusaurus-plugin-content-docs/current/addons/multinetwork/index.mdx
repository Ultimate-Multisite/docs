# Multi-Network

Ultimate Multisite: Multinetwork addon nthuav koj lub platform Ultimate Multisite nrog kev ua haujlwm multinetwork muaj zog. Addon no cia koj cov neeg siv khoom yuav thiab tswj tau tag nrho cov WordPress multisite networks, tsis yog tsuas yog ib lub site xwb.

## Cov Nta Tseem Ceeb {#key-features}

* **Kev Tsim Network**: Tsim thiab teeb tsa tag nrho cov WordPress multisite networks rau koj cov neeg siv khoom tau txais
* **Network Templates**: Cia cov neeg siv khoom xauj saib, saib ua ntej, thiab xaiv ib lub network template uas twb npaj ua ntej lawm ua ntej signup
* **Kev Tswj Network**: Muaj peev xwm tswj tag nrho rau cov ib puag ncig multinetwork
* **Networks Sib Cais**: Txhua tus neeg siv khoom lub network khiav nws tus kheej nrog nws cov database tables thiab kev teeb tsa
* **Cross-Network SSO**: Muab kev nkag mus yooj yim rau cov neeg siv khoom mus rau lawv cov sub-networks nrog magic-link authentication
* **Architecture Uas Nthuav Tau**: Tswj ntau lub networks tau zoo nrog kev faib peev txheej uas kho kom zoo
* **Kev Sib Txuas**: Sib txuas tau yooj yim nrog Ultimate Multisite cov subscription thiab billing systems

## Cov Kev Siv {#use-cases}

* **White Label Hosting**: Muab rau cov neeg siv khoom lawv tus kheej cov multisite networks uas muaj lawv hom npe
* **Agency Solutions**: Pab kom agencies tswj tau ntau tus client networks los ntawm ib lub platform xwb
* **Enterprise Deployments**: Txhawb cov koom haum loj uas muaj cov kev xav tau multi-network nyuaj
* **Education Platforms**: Tsim cov networks sib cais rau cov tsev kawm lossis departments sib txawv

## Yam Yuav Tsum Muaj {#requirements}

* Ultimate Multisite 2.0.0 lossis siab dua
* WordPress Multisite installation
* PHP 7.4 lossis siab dua
* MySQL 5.6 lossis siab dua

## Kev Nruab {#installation}

1. Upload cov plugin files mus rau directory `/wp-content/plugins/ultimate-multisite-multinetwork`
2. Activate lub plugin hauv qhov screen 'Plugins' hauv WordPress
3. Teeb tsa multinetwork settings los ntawm Ultimate Multisite → Settings

## Network Templates {#network-templates}

Version 1.3.0 ntxiv ib txoj kev tshawb nrhiav thiab xaiv template rau cov neeg siv khoom uas yuav networks tshiab. Siv network templates thaum koj xav kom txhua tus neeg siv khoom lub network tshiab pib ntawm ib qho qauv, design, content library, lossis kev teeb tsa raws ib hom lag luam uas twb npaj lawm.

### Tus saib template ua ntej {#template-previewer}

Tus network template previewer cia cov neeg siv khoom xauj saib cov templates uas muaj ua ntej checkout. Txhua lub template qhib tau hauv ib lub live preview panel, kom cov neeg siv khoom piv tau layouts thiab content ua ntej lawv yuav ib lub network.

Siv tus previewer los pab cov neeg siv khoom:

* Saib xyuas qhov tsos thiab qauv ntawm txhua lub network template uas muaj
* Piv ntau lub templates yam tsis tas tawm ntawm txoj kev yuav
* Xaiv lub template uas phim lawv qhov kev siv tshaj plaws ua ntej signup

Khaws template names thiab descriptions kom phooj ywg rau cov neeg siv khoom. Tus previewer pab tau tshaj thaum cov neeg siv khoom nkag siab tau qhov kev siv uas npaj tseg, xws li agency networks, school networks, franchise networks, lossis white-label hosting bundles.

### Tus xaiv template thaum checkout {#template-picker-at-checkout}

Tus template picker ntxiv tau rau hauv signup checkout flow ua ib daim field. Thaum teeb tsa lawm, cov neeg siv khoom xaiv lub network template uas lawv xav tau thaum checkout, thiab lub template uas xaiv ntawd yuav raug siv thaum lawv lub network tshiab raug provisioned.

Yuav siv tus picker hauv checkout flow:

1. Npaj cov networks uas koj xav muab ua templates.
2. Enable Multinetwork template selection field hauv signup lossis checkout form uas cuam tshuam.
3. Xaiv seb cov templates twg yuav muaj rau checkout flow ntawd.
4. Publish checkout flow thiab sim ib qho signup kom paub tseeb tias lub template uas xaiv raug siv rau lub network tshiab.

Yog ib qho checkout flow tsuas muaj ib qho network product xwb, tus picker tseem pab tau ua ib kauj ruam guided onboarding. Rau multi-plan funnels, tsuas tso tawm cov templates uas compatible nrog product lossis plan uas xaiv xwb.

## Cross-Network Magic Link SSO {#cross-network-magic-link-sso}

Cross-network magic link SSO cia cov neeg siv khoom txav ntawm lawv thaj chaw Account tseem ceeb thiab sub-networks yam tsis tas sign in dua los ntawm tes. Thaum ib tus neeg siv khoom nias ib qho cross-network access link, Ultimate Multisite tsim ib qho magic link uas muaj sijhawm txwv thiab authenticate tus neeg siv khoom mus rau destination network.

Qhov no txhim kho kev siv rau cov neeg siv khoom uas tswj ntau dua ib lub network lossis cov uas yuav tsum nkag mus rau template networks tshiab uas cloned tam sim tom qab yuav.

Admin checklist:

* Xyuas kom meej tias Multinetwork addon active thoob plaws WordPress multisite installation.
* Xyuas kom cov neeg siv khoom muaj network access raug ua ntej lawv tau txais cross-network links.
* Khaws network domains thiab mapped domains kom teeb tsa raug kom magic links resolve mus rau destination network uas npaj tseg.
* Kuaj customer dashboard thiab txhua lub post-checkout access buttons tom qab hloov network, domain, lossis checkout settings.

Magic links yuav tsum raug saib zoo li login links. Lawv tsim los rau kev nkag tau yooj yim rau cov neeg siv khoom thiab yuav tsum tsis txhob forward lossis rov siv sab nraum session uas npaj tseg.

## Cov Lus Nug Nquag Nug {#frequently-asked-questions}

### Multinetwork yog dab tsi? {#what-is-multinetwork}

Multinetwork yog ib qho WordPress feature uas cia koj khiav ntau lub multisite networks ywj pheej rau ntawm tib qho WordPress installation.

### Qhov no txawv li cas ntawm multisite ib txwm? {#how-is-this-different-from-regular-multisite}

Multisite ib txwm cia muaj ntau lub sites hauv ib lub network. Multinetwork cia muaj ntau lub networks, txhua lub muaj lawv tus kheej sites, tag nrho nyob hauv tib qho WordPress installation.

### Qhov no puas xav tau server configuration tshwj xeeb? {#does-this-require-special-server-configuration}

Yog, multinetwork xav tau server setup thiab configuration tshwj xeeb. Thov saib documentation rau cov lus qhia setup kom ntxaws.
