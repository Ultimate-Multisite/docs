# Kev Xam Phaj Tshawb Nrhiav

**Kev Xam Phaj Tshawb Nrhiav** yog kauj ruam thawj zaug hauv Theme Builder tus txheej txheem tsim qauv. Nws sau ntaub ntawv txog koj lub vev xaib, kev lag luam, thiab yam qauv uas koj nyiam kom tsim tau ib lub theme uas phim koj.

## Kev Piav Qhia Dav Dav {#overview}

Kev Xam Phaj Tshawb Nrhiav coj koj mus teb ib co lus nug kom nkag siab txog koj lub brand, pab neeg hom phiaj, thiab cov hom phiaj tsim qauv. Hauv version 1.16.0, kev xam phaj tam sim no muaj ib **kauj ruam upload duab** uas tso cai rau koj muab kev tshoov siab pom tau rau koj tus qauv.

## Cov Kauj Ruam Xam Phaj {#interview-steps}

### 1. Ntaub Ntawv Kev Lag Luam {#1-business-information}

- Lub npe vev xaib thiab tagline
- Hom kev lag luam lossis kev lag luam teb
- Cov hom phiaj kev lag luam tseem ceeb
- Kev piav qhia txog pab neeg hom phiaj

### 2. Yam Qauv Uas Nyiam {#2-design-preferences}

- Cov xim uas nyiam siv
- Hom qauv tsim (niaj hnub, classic, minimalist, lom zem, thiab lwm yam)
- Cov vev xaib lossis brand uas muab kev tshoov siab
- Cov ntsiab qauv uas yuav tsum muaj

### 3. Upload Duab (Tshiab hauv v1.16.0) {#3-photo-upload-new-in-v1160}

Upload ib daim lossis ntau daim duab kom pab coj txoj kev tsim qauv:

#### Cov Format Uas Txais {#accepted-formats}

| Format | Extension | Cov Ntsiab Lus |
|--------|-----------|---------|
| **JPEG** | `.jpg`, `.jpeg` | Pom zoo rau cov duab thaij |
| **PNG** | `.png` | Txhawb transparency |
| **WebP** | `.webp` | Format niaj hnub, file me dua |
| **GIF** | `.gif` | Tsuas yog duab nyob ruaj xwb (tsis muaj animation) |

#### Cov Cai Rau File {#file-requirements}

- **File loj tshaj plaws:** 10 MB
- **Qhov loj yam tsawg kawg:** 400 × 300 pixels
- **Qhov loj uas pom zoo:** 1200 × 800 pixels lossis loj dua
- **Aspect ratio:** Ratio twg los yeej txais tau

#### Duab Pab Cuam Tshuam Txoj Kev Tsim Qauv Li Cas {#how-photos-influence-design-direction}

Cov duab uas koj upload pab Theme Builder nkag siab txog:

- **Color palette** — rho tawm cov xim tseem ceeb thiab siv rau hauv theme
- **Hom kev pom** — niaj hnub, rustic, minimalist, ci ntsa iab, thiab lwm yam.
- **Kev xav thiab huab cua** — professional, muaj tswv yim, phooj ywg, luxurious, thiab lwm yam.
- **Cov ntsiab tsim qauv** — qauv, textures, thiab hom kev teeb tso ua ke
- **Kev tshoov siab typography** — hom font uas haum nrog lub theme pom

### 4. Txheej Txheem Cov Ntsiab Lus {#4-content-structure}

- Cov nplooj ntawv tseem ceeb uas xav tau (Home, About, Services, Contact, thiab lwm yam)
- Puas xav tau seem blog lossis xov xwm
- Puas xav tau e-commerce functionality
- Cov yam ntxwv tshwj xeeb lossis integrations

### 5. Tshuaj Xyuas thiab Paub Meej {#5-review-and-confirmation}

- Cov ntsiab lus xaus ntawm txhua yam ntaub ntawv uas tau muab
- Kev paub meej ua ntej pib tsim qauv

## Cov Kev Xyaum Zoo Rau Upload Duab {#photo-upload-best-practices}

### Yam Yuav Upload {#what-to-upload}

- **Kev tshoov siab rau brand** — logos, color swatches, lossis brand guidelines
- **Cov vev xaib sib tw** — screenshots ntawm cov vev xaib uas koj nyiam
- **Mood boards** — pawg duab uas sawv cev rau qhov aesthetic uas koj xav tau
- **Duab khoom** — yog tias koj muaj cov khoom los nthuav qhia
- **Duab pab pawg lossis chaw ua haujlwm** — rau personal lossis professional branding
- **Duab lifestyle** — sawv cev rau koj pab neeg hom phiaj

### Yam Yuav Zam {#what-to-avoid}

- **Duab tsis zoo** — duab plooj lossis pixelated
- **Duab tsis cuam tshuam** — duab uas tsis sawv cev rau koj lub brand
- **Duab muaj ntawv ntau dhau** — screenshots uas muaj ntawv ntau
- **Khoom muaj copyright** — nco ntsoov tias koj muaj cai siv cov duab
- **Animated GIFs** — tsuas yog cov duab nyob ruaj thiaj raug ua haujlwm

### Tswv Yim Kom Tau Txiaj Ntsig Zoo Tshaj {#tips-for-best-results}

1. **Upload 2-3 daim duab** — ntau daim duab muab context zoo dua
2. **Siv duab zoo** — duab meej thiab ci zoo ua haujlwm zoo tshaj
3. **Muab ntau hom** — sib xyaw xim, style, thiab yam ntsiab lus
4. **Muab context** — piav luv luv tias vim li cas koj xaiv txhua daim duab
5. **Qhia kom meej** — yog tias koj nyiam ib qho ntsiab tshwj xeeb, taw qhia nws

## Qhov Tso Tawm Rau Txoj Kev Tsim Qauv {#design-direction-output}

Tom qab Kev Xam Phaj Tshawb Nrhiav, Theme Builder tsim:

- **Color palette** — cov xim primary, secondary, thiab accent
- **Typography** — cov tsev neeg font rau headings thiab body text
- **Hom layout** — spacing, alignment, thiab kev teeb component
- **Cov ntsiab pom** — icons, patterns, thiab cov ntsiab dai kom zoo nkauj
- **Mood board** — kev sawv cev pom tau ntawm txoj kev tsim qauv
