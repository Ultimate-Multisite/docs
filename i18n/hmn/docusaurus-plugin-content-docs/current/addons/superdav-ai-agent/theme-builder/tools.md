# Cov Cuab Yeej thiab Kev Teeb Tsa

Theme Builder siv ib pawg cuab yeej los tshuaj xyuas, tsim qauv, thiab tsim koj lub WordPress site. Hauv version 1.16.0, **sd-ai-agent/site-scrape tam sim no yog cuab yeej Qib 1**, ua rau nws muaj siv tau ua ntej hauv Theme Builder. Superdav AI Agent v1.18.0 kuj qhia ib pob Advanced companion uas faib tawm nyias rau cov txheej txheem ua haujlwm ntawm developer uas ntseeg tau; cov cuab yeej ntawd tsis yog ib feem ntawm WordPress.org core package thiab yuav tsum muab nruab thiab tso cai siv nyias.

## Cov Cuab Yeej Muaj Siv Tau {#available-tools}

### Cov Cuab Yeej Qib 1 (Muaj Siv Tau Tas Li) {#tier-1-tools-always-available}

Cov cuab yeej Qib 1 muaj siv tau ua ntej hauv Theme Builder yam tsis tas muaj kev teeb tsa ntxiv.

#### sd-ai-agent/site-scrape {#sd-ai-agentsite-scrape}

**Lub hom phiaj:** Tshuaj xyuas cov website uas twb muaj lawm kom rho tau kev tshoov siab rau qauv, qauv txheej txheem ntsiab lus, thiab functionality.

**Peev xwm:**

- **Kev tshuaj xyuas website** — scrape thiab tshuaj xyuas cov website sib tw lossis cov website tshoov siab
- **Kev rho qauv** — txheeb xyuas xim, font, thiab layout patterns
- **Tus qauv ntsiab lus** — nkag siab kev npaj page thiab hierarchy
- **Kev tshawb nrhiav feature** — txheeb xyuas plugins thiab functionality
- **Kev tshuaj xyuas performance** — kuaj page speed thiab optimization
- **Kev tshuaj xyuas SEO** — tshuaj xyuas meta tags thiab structured data

**Kev siv:**

```
Tshuaj xyuas tus qauv ntawm example.com thiab siv nws ua kev tshoov siab rau kuv site.
```

**Yam Nws Rho Tau:**

- Color palette thiab kev siv xim
- Typography (fonts thiab sizes)
- Layout structure thiab spacing
- Navigation patterns
- Kev npaj ntsiab lus
- Kev siv images thiab media
- Interactive elements
- Mobile responsiveness

**Kev txwv:**

- Scrape tsis tau cov site uas tiv thaiv nrog password
- Hwm robots.txt thiab site policies
- Tej zaum yuav tsis ntes tau dynamic content
- Cov site uas siv JavaScript-heavy tej zaum yuav tshuaj xyuas tau tsawg
- Cov site loj tej zaum yuav siv sijhawm ntev dua los tshuaj xyuas

### Cov Cuab Yeej Advanced Companion (Pob Nyias) {#advanced-companion-tools-separate-package}

Pob Advanced companion raug faib tawm nyias ntawm qhov core Superdav AI Agent release. Nws yog npaj rau developer thiab site-owner environments uas ntseeg tau, qhov uas administrators tso cai meej rau tus agent siv high-impact tooling.

Cov peev xwm Advanced companion tuaj yeem muaj xws li:

- **Filesystem tools** — tshuaj xyuas thiab kho cov files uas tau pom zoo hauv environments uas ntseeg tau
- **Database tools** — query lossis update site data thaum tso cai meej lawm
- **WP-CLI tools** — khiav WordPress maintenance thiab inspection commands
- **REST dispatcher tools** — hu registered REST endpoints los ntawm controlled dispatch workflows
- **Plugin builder tools** — scaffold thiab iterate rau plugin code
- **Git snapshot tools** — tsim snapshots ua ntej developer operations uas muaj kev pheej hmoo
- **User-management tools** — pab review lossis kho user records qhov twg tso cai
- **Benchmark tools** — khiav performance lossis capability benchmarks rau development review

Tsis txhob sau cov no ua cov Theme Builder tools uas muaj siv dav rau txhua installation. Lawv muaj siv tau tsuas yog thaum pob Advanced companion raug nruab, raug qhib, thiab raug txwv rau administrators uas ntseeg tau.

## Kev Teeb Tsa Cuab Yeej {#tool-configuration}

### Qhib Cov Cuab Yeej {#enabling-tools}

Txhawm rau qhib cov cuab yeej ntxiv hauv Theme Builder:

1. Mus rau **Gratis AI Agent → Settings**
2. Mus rau **Theme Builder → Tools**
3. Toggle cov cuab yeej on/off raws li xav tau
4. Save settings

### Kev Tso Cai Rau Cuab Yeej {#tool-permissions}

Qee cov cuab yeej xav tau permissions:

- **Site scraping** — xav tau internet access
- **Analytics** — xav tau Google Analytics integration
- **Content generation** — xav tau API keys
- **Advanced features** — tej zaum yuav xav tau premium subscription
- **Advanced companion tools** — xav tau pob Advanced uas faib tawm nyias thiab kev ntseeg meej ntawm administrator

### API Keys thiab Credentials {#api-keys-and-credentials}

Teeb API keys rau cov cuab yeej uas xav tau lawv:

1. Mus rau **Gratis AI Agent → Settings → API Keys**
2. Sau credentials rau txhua lub cuab yeej
3. Kuaj connection
4. Save kom ruaj ntseg

### Nruab Pob Advanced Companion {#installing-the-advanced-companion-package}

Pob Advanced companion raug tso tawm nyias ntawm WordPress.org core ZIP. Nruab nws tsuas yog los ntawm official project distribution channel rau qhov release ntawd, ces txwv kev nkag mus rau administrators uas ntseeg tau. Review human approval gates thiab git snapshot behaviour ua ntej qhib filesystem, database, WP-CLI, REST dispatcher, plugin builder, user-management, lossis benchmark tools ntawm production site.

## Kev Txwv thiab Yam Yuav Tau Xav Txog Rau Cuab Yeej {#tool-limitations-and-considerations}

### Rate Limiting {#rate-limiting}

- Kev scraping raug txwv tus nqi kom tiv thaiv tsis pub server hnyav dhau
- Ntau qhov kev thov mus rau tib lub vev xaib tej zaum yuav raug qeeb
- Cov vev xaib loj tej zaum yuav siv sijhawm ntev dua los tshuaj xyuas

### Kev raug ntawm cov ntsiab lus {#content-accuracy}

- Cov ntsiab lus uas hloov pauv tas li tej zaum yuav tsis raug khaws tau tag nrho
- Cov ntsiab lus uas JavaScript tsim tawm tej zaum yuav tsis tiav
- Cov ntaub ntawv real-time tej zaum yuav tsis raug qhia
- Qee cov ntsiab lus tej zaum yuav nyob tom qab paywalls

### Kev xav txog kev cai lij choj thiab kev coj ncaj ncees {#legal-and-ethical-considerations}

- Hwm robots.txt thiab cov cai ntawm lub vev xaib
- Tsis txhob scrape cov ntsiab lus uas muaj copyright los rov tsim dua
- Siv kev tshuaj xyuas ua kev tshoov siab, tsis yog luam
- Xyuas kom tseeb tias koj muaj cai siv cov ntsiab lus uas rho tawm tau
- Ua raws cov terms of service ntawm cov vev xaib uas raug tshuaj xyuas

### Kev cuam tshuam rau kev ua haujlwm {#performance-impact}

- Kev scraping cov vev xaib loj tej zaum yuav siv sijhawm
- Ntau qhov scraping tib lub sijhawm tej zaum yuav qeeb dua
- Kev txuas network cuam tshuam rau kev ceev
- Cov ntaub ntawv loj tej zaum yuav xav tau kev ua haujlwm ntxiv

## Cov kev coj zoo tshaj {#best-practices}

### Siv kev tshuaj xyuas vev xaib {#using-site-analysis}

1. **Tshuaj xyuas ntau lub vev xaib** — sau kev tshoov siab los ntawm ntau qhov chaw
2. **Tsom rau qauv** — kawm cov qauv layout thiab kev teeb tsa
3. **Rho cov xim** — siv cov color palettes ua qhov chaw pib
4. **Kawm typography** — txheeb xyuas cov kev sib xyaw font uas koj nyiam
5. **Tshuaj xyuas kev taw qhia** — nkag siab kev teeb tsa menu

### Kev scraping raws kev coj ncaj ncees {#ethical-scraping}

1. **Hwm robots.txt** — ua raws cov lus qhia ntawm lub vev xaib
2. **Tsis txhob luam cov ntsiab lus** — siv kev tshuaj xyuas rau kev tshoov siab xwb
3. **Xyuas cov cai** — xyuas kom meej tias koj siv tau cov ntsiab lus uas rho tawm tau
4. **Qhia qhov chaw** — muab credit rau cov chaw tshoov siab
5. **Ua raws cov terms** — ua raws cov terms of service ntawm lub vev xaib

### Ua kom cuab yeej muaj txiaj ntsig tshaj {#maximizing-tool-effectiveness}

1. **Hais kom meej** — thov cov hom kev tshuaj xyuas tshwj xeeb
2. **Muab ntsiab lus tom qab** — piav qhia lub hom phiaj ntawm koj lub vev xaib
3. **Teeb kev cia siab** — piav koj cov hom phiaj tsim qauv
4. **Rov kho dua** — kho kom zoo raws li cov txiaj ntsig
5. **Sib txuas cov cuab yeej** — siv ntau lub cuab yeej rau kev tshuaj xyuas kom dav

## Kev daws teeb meem {#troubleshooting}

### Lub vev xaib tsis kam scrape {#site-wont-scrape}

- Xyuas seb lub vev xaib puas qhib rau pej xeem nkag tau
- Xyuas kom paub tias robots.txt tso cai rau scraping
- Sim lwm lub vev xaib
- Xyuas kev txuas internet
- Hu rau support yog qhov teeb meem tseem muaj

### Kev tshuaj xyuas tsis tiav {#incomplete-analysis}

- Lub vev xaib tej zaum yuav muaj cov ntsiab lus dynamic
- JavaScript tej zaum yuav tsis raug rendered tag nrho
- Cov vev xaib loj tej zaum yuav timeout
- Sim tshuaj xyuas cov nplooj ntawv tshwj xeeb hloov
- Thov cov hom kev tshuaj xyuas tshwj xeeb

### Kev ua haujlwm qeeb {#slow-performance}

- Cov vev xaib loj siv sijhawm ntev dua los tshuaj xyuas
- Ntau qhov scraping tib lub sijhawm qeeb dua
- Kev txuas network cuam tshuam rau kev ceev
- Sim thaum tsis yog cov sijhawm muaj neeg siv coob
- Tshuaj xyuas cov ntu me ua ntej
