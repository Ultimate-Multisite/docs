# Superdav AI Agent {#superdav-ai-agent}

Superdav AI Agent yog ib tug WordPress AI assistant qib siab uas nthuav Gratis AI Agent nrog kev txawj paub theme, peev xwm tshawb nrhiav internet zoo dua, cov yam ntxwv tsim vev xaib, thiab kev txuas automation uas xaiv tau. Version 1.18.0 ntxiv cov cuab yeej nyeem Google Calendar, kev map contact thiab attendee, cov rooj vag kom tib neeg pom zoo, kev deduplication rau reminder, TextBee SMS notifications, kev npaj managed Superdav service, thiab ib pob Advanced companion uas faib nyias rau cov trusted developer workflows.

## Cov Yam Ntxwv Tseem Ceeb {#key-features}

- **Kev txawj paub theme** — cia li hloov cov lus qhia thiab peev xwm raws li koj lub WordPress theme uas tab tom siv thiab cov plugins uas tau nruab
- **Ntau tus provider tshawb nrhiav internet** — xaiv ntawm Brave Search thiab Tavily kom tau cov txiaj ntsig tshawb nrhiav nplua nuj dua thiab peev xwm research qib siab
- **Cov contact forms rau tus tsim vev xaib** — tsim cov contact forms uas ua haujlwm tiav ncaj qha ntawm chat interface siv koj lub form plugin uas tau nruab
- **Tus tsim vev xaib uas kho kom zoo dua** — cov phiaj xwm tsim vev xaib ntau theem uas txhim kho nrog kev nrhiav plugin zoo dua thiab rov qab los ntawm yuam kev
- **Kev txawj theme uas muaj los hauv** — cov lus qhia tshwj xeeb rau Block Themes, Classic Themes, Kadence Blocks, thiab Kadence Theme
- **WooCommerce integration uas kho kom zoo dua** — rov kho kom siv WooCommerce APIs native kom ntseeg tau dua thiab sib haum zoo dua
- **Kev kuaj provider cia li ua** — cia li kuaj thiab refresh cov plugins thiab themes uas muaj thaum lawv raug qhib lossis kaw
- **Kev tsim logo custom** — tsim thiab embed custom SVG logos nrog automatic namespace-safe sanitisation
- **Kev txheeb xyuas palette contrast** — xyuas cov khub xim kom ua raws WCAG accessibility ua ntej siv rau koj lub theme
- **Kev tshawb pom uas muaj duab pab** — upload duab thaum Theme Builder discovery interview kom cuam tshuam rau txoj kev tsim design
- **Cov hospitality menus** — tsim cov nplooj menu zaub mov thiab dej haus uas muaj qauv rau restaurants, cafes, thiab cov lag luam hospitality
- **Desktop thiab mobile previews** — preview koj lub design rau desktop thiab mobile devices thaum xaiv design direction
- **Tier 1 scraping vev xaib** — sd-ai-agent/site-scrape tam sim no muaj los ntawm default rau kev txheeb xyuas competitor thiab inspiration websites
- **Cov cuab yeej nyeem Google Calendar** — nyeem cov calendars thiab events uas tau configure kom automations thiaj xav tau txog lub schedule context uas yuav los
- **Kev map contact thiab attendee** — phim cov neeg koom calendar rau cov neeg siv vev xaib thiab contacts uas paub ua ntej tsim follow-up work
- **Cov rooj vag kom tib neeg pom zoo** — nres cov automations uas sensitive rau review ua ntej xa messages, rov pib ua haujlwm, lossis hloov pauv
- **Cov reminder records thiab deduplication** — taug qab cov reminders uas twb tau xa lawm kom automation khiav dua tsis txhob ceeb toom tib tus recipient ob zaug
- **TextBee SMS notifications** — xa cov text-message notifications uas tau configure dhau TextBee thaum muaj SMS provider qhib
- **Managed Superdav service provisioning** — txuas cov vev xaib uas txhawb nqa rau hosted Superdav endpoints nrog automatic setup es tsis yog configure endpoint manually
- **Advanced companion package** — cov trusted-developer tools uas faib nyias rau filesystem, database, WP-CLI, REST dispatcher, plugin builder, git snapshot, user-management, thiab benchmark workflows

## Yam Yuav Tsum Muaj {#requirements}

- WordPress 6.0 lossis siab dua
- PHP 8.1 lossis siab dua
- WordPress Multisite (network-activated) lossis single-site installation
- Ib lub AI provider connector plugin uas active (Anthropic, OpenAI, lossis compatible)
- Gratis AI Agent v1.9.0 lossis siab dua (Superdav AI Agent nthuav Gratis AI Agent)

## Kev Nruab {#installation}

1. Upload cov plugin files rau `/wp-content/plugins/superdav-ai-agent`
2. Qhib lub plugin dhau ntawm **Plugins** screen hauv WordPress
3. Xyuas kom Gratis AI Agent kuj tau qhib lawm
4. Mus rau **Gratis AI Agent → Settings** thiab configure koj tus internet search provider, managed Superdav service connection, Google Calendar, lossis TextBee credentials raws li xav tau

Lub WordPress.org package muaj cov core Superdav AI Agent features. Lub Advanced companion package raug build thiab faib nyias rau cov environments uas cov trusted developer tools raug nruab thiab tso cai meej.
