## Webmail SSO thiab Magic-Link Login {#webmail-sso-and-magic-link-login}

Qhib webmail access tsuas rau providers uas koj twb teeb tsa required API credentials lawm xwb. Yog action ploj rau ib tug provider, ua ntej xyuas tias provider yog active, configured, thiab supports webmail login links.
