# Tus Tsim Nplooj Admin

## Cov Ntsiab Lus
Tus tsim nplooj admin yuav tso cai rau koj yooj yim ntxiv cov nplooj admin tshiab thiab cov nplooj me nyob hauv WordPress admin dashboard. Nws siv tau los tsim nplooj txais tos lossis nplooj qhia siv uas tuaj yeem ua nplooj pib rau cov neeg siv tshiab. Koj siv tau cov page builder xws li Beaver Builder, Elementor, Brizy, thiab Oxygen los tsim cov nplooj yam tsis muaj teeb meem.

Nws tseem muaj kev txhawb rau HTML, CSS thiab JavaScript uas kho tau, nrog rau kev tswj kev nkag raws li plan thiab user role!

## Cov Nta
* Txhawb HTML thiab CSS uas kho tau
* Txhawb JavaScript
* PHP Content Mode
* Kev tso cai raws li User role
* Txhawb rau page builders

## Kev Txhim Kho

1. Upload **ultimate-multisite-admin-page-creator** mus rau hauv directory **/wp-content/plugins/**
2. Qhib lub plugin ntawm menu **Plugins** hauv koj WordPress Network Admin

## Kev Qhia Siv

Thaum WordPress ua rau yooj yim heev ntxiv cov ntsiab lus rau sab pem hauv ntej ntawm cov site, kev tsim nplooj thiab nplooj me nyob sab admin yuav tsum muaj me ntsis kev txawj sau code. Ultimate Multisite: Tus Tsim Nplooj Admin lub hom phiaj yog coj tib yam kev yooj yim ntawm sab pem hauv ntej mus rau sab tom qab ntawm koj cov network sites.

## Vim li cas thiaj siv Tus Tsim Nplooj Admin?

Tej zaum koj xav muab ib nplooj uas kho tau rau hauv koj cov client lub vaj huam sib luag admin nrog cov kev qhia, tej zaum koj xav ntxiv ib nplooj uas kho tau nrog FAQs thiab ib contact widget. Cov peev xwm muaj tsis kawg nrog Tus Tsim Nplooj Admin, thiab qhov ntawd yog yam peb yuav tshawb hauv zaj qhia hnub no.

## Tsim koj thawj Nplooj Admin Uas Kho Tau

Tom qab txhim kho thiab qhib Ultimate Multisite: Tus Tsim Nplooj Admin, mus rau koj lub network admin menu thiab mus rau Admin Pages.

![Daim npe Nplooj Admin Uas Kho Tau ntawm Network Admin](/img/admin/admin-pages-list.png)

Koj yuav pom daim npe ntawm txhua nplooj admin uas koj tau tsim los txog tam sim no (vim qhov no yog koj thawj zaug tuaj saib, daim npe yuav khoob). Mus ntxiv thiab nyem **Ntxiv Nplooj Admin tshiab**. Koj yuav raug xa mus rau tus editor Nplooj Admin.

## Kev Xaiv Menu

Nyob sab, hauv General Options meta-box, koj yuav tuaj yeem teeb hom menu, daim ntawv lo, thiab ntau yam ntxiv.

### Hom Menu

Koj nplooj admin tshiab uas kho tau muaj tau ob hom menu sib txawv: top-level menu thiab submenu.

Ib nplooj top-level menu, raws li lub npe qhia, yuav ntxiv koj nplooj uas nyuam qhuav tsim rau hauv admin menu nrog rau cov nplooj WordPress qub.

Yog koj xaiv qhov top-level option, koj yuav xaiv tau menu order thiab ib menu icon thiab.

Yog koj xav ua kom nplooj no ua ib nplooj sub-menu, koj yuav xaiv tau ib parent page los txuas nplooj no rau. Koj ntxiv tau koj nplooj uas kho tau rau cov nplooj WordPress qub lossis rau koj cov nplooj admin uas kho tau hom top-level.

## Kev Xaiv Qib Siab

Koj yuav tsum muaj peev xwm ntxiv txhua hom ntsiab lus rau koj cov nplooj admin uas kho tau. Qhov ntawd suav nrog CSS rules thiab files, JavaScript, thiab external libraries thiab. Koj ua tau qhov ntawd los ntawm kev siv Advanced Options Tab.

### CSS Uas Kho Tau

Nyob rau CSS tab, koj ntxiv tau koj tus kheej custom CSS code, nrog rau import CSS files los ntawm cov site sab nraud (xws li CDNs, Google Fonts, thiab lwm yam).

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_Custom CSS kuj tau txais kev txhawb thiab. Koj kuj muab tau cov style files sab nraud los siv!_

### JavaScript Uas Kho Tau

Tib yam no kuj siv tau rau JavaScript. Koj ntxiv tau external libraries los siv rau hauv koj custom JavaScript code.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Ntxiv koj tus kheej JavaScript code!_

## Kev Tso Cai

Tsis tas li hauv Advanced Options, koj yuav xaiv tau cov sub-sites twg yuav nkag tau rau nplooj no. Qhov no pab tau yog koj npaj yuav muab cov ntsiab lus tshwj xeeb sib txawv rau cov plan tiers sib txawv, piv txwv li.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Koj muaj kev tswj tag nrho seb nplooj no yuav tshwm qhov twg_

**Tseem ceeb** : Ob qho kev mob no yuav tsum ua tau raws mas nplooj ntawv thiaj yuav tshwm. Qhov no txhais tias yog tus neeg siv yog ib tug client ntawm ib qho plan twg, tab sis tsis muaj ib qho ntawm cov roles uas tau xaiv (lossis rov qab) ces nplooj ntawv yuav tsis tshwm ntawm lawv lub panel.

## Qhib Siv

Yog koj xav kaw ib nplooj ntawv admin uas tsim kho tshwj xeeb yam tsis tas rho tawm nws lossis cuam tshuam rau cov permissions, cia li tshem qhov cim ntawm qhov kev xaiv Qhib Siv xwb.

Thaum kaw ib nplooj ntawv qib-siab tshaj plaws lawm, nws yuav cia li kaw tag nrho cov nplooj ntawv me uas txuas nrog nplooj ntawv niam ntawd.
