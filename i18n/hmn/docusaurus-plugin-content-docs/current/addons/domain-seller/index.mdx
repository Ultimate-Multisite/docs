# Domain Seller {#domain-seller}

**Multisite Ultimate: Domain Seller** hloov koj lub WordPress Multisite network ua ib lub platform muag domain muaj zog. Cia cov neeg siv sau npe domain raws lawv xav tau ncaj qha thaum sign-up los yog ntawm lawv lub Dashboard, nrog kev kuaj xyuas muaj nyob tiag-tiag thiab nqi meej pem.

### Vim li cas thiaj siv Domain Seller? {#why-domain-seller}

Kev sau npe domain yog ib qho upsell uas cov neeg thov tshaj plaws rau txhua lub WaaS (Website as a Service) platform. Es tsis txhob xa koj cov neeg siv mus rau ib tus registrar sab nraud, koj tuaj yeem muab domain ncaj qha — khaws tau nyiaj tau los ntau dua hauv koj lub network thaum muab kev siv uas tsis muaj kev khuam rau cov neeg siv.

### Cov yam ntxwv tseem ceeb {#key-features}

#### Txhawb ntau tus Domain Registrar {#multiple-domain-registrar-support}
* **OpenSRS** — Sau npe, rov txuas hnub nyoog, thiab tswj kom tiav
* **Namecheap** — Kev txuas ua haujlwm tiav nrog WhoisGuard thiab kev sim sandbox
* **HostAfrica** — Tshawb nrhiav domain, nqi, sau npe, hloov mus, DNS, thiab ID protection ntawm Domains Reseller API
* **Openprovider** — Nqi reseller, sau npe, rov txuas hnub nyoog, hloov mus, DNS, WHOIS privacy, thiab TLD sync
* **Hostinger** — Tshawb nrhiav thiab sau npe domain siv Hostinger integration token uas sib koom siv
* **Extensible Architecture** — Tsim rau saum Integration Registry pattern, ua rau yooj yim ntxiv provider ntau ntxiv
* **Kev sim kev txuas uas muaj hauv-system** — Xyuas kom provider credentials thiab kev txuas ua haujlwm raug ua ntej tso live

#### Guided Setup Wizard {#guided-setup-wizard}
* **One-click onboarding** — Coj koj mus teeb provider configuration, import TLD, thiab tsim domain product
* **Automatic TLD sync** — Import tag nrho TLDs uas muaj thiab wholesale pricing los ntawm koj cov providers uas txuas lawm
* **Kev tsim product default** — Tsim ib qho domain product npaj siv nrog markup defaults uas tsim nyog

#### Kev teeb nqi domain yoog tau {#flexible-domain-pricing}
* **Peb hom markup** — Percentage markup, fixed markup, los yog fixed price rau ib TLD
* **Products tshwj xeeb rau TLD** — Tsim domain products cais rau pab TLD sib txawv (.com/.net, .io, country codes, thiab lwm yam)
* **Nqi pib qhia** — Muab tus nqi xyoo thawj txo qis los nyiam kev sau npe tshiab
* **Nqi tiag-tiag** — Wholesale pricing live los ntawm providers nrog caching ntse
* **Txhawb coupon** — Sib haum tag nrho nrog Multisite Ultimate discount codes

#### WHOIS Privacy Protection {#whois-privacy-protection}
* **Configuration raws product** — Teeb WHOIS privacy ua Disabled, Always Included, los yog Customer Choice
* **Customer opt-in** — Thaum teeb ua Customer Choice, ib lub checkbox thaum them nyiaj cia cov neeg siv ntxiv privacy protection
* **Nqi kho tau** — Teeb tus nqi ib xyoos rau privacy add-on, los yog muab dawb
* **Registrar integration** — Qhib WhoisGuard (Namecheap) los yog WHOIS privacy (OpenSRS) tau txais thaum txog sijhawm sau npe

#### Kev them nyiaj ua ke zoo ib yam {#unified-checkout-experience}
* **Peb hom domain** — Cov neeg siv xaiv ntawm subdomain (dawb), sau npe domain tshiab, los yog mapping domain uas twb muaj lawm uas lawv yog tus tswv
* **Tshawb nrhiav domain tiag-tiag** — Kev kuaj xyuas muaj nyob uas siv AJAX yam tsis tas reload nplooj ntawv

* **Registrant contact fields** — Sau cov ntaub ntawv registrant uas yuav tsum muaj (npe, chaw nyob, xov tooj) inline thaum them nyiaj
* **Site URL tsim tau txais** — Chaw nyob site raug tsim tau txais los ntawm domain uas xaiv

#### Auto-Renewal System {#auto-renewal-system}
* **Kev rov txuas hnub nyoog khi nrog membership** — Domains rov txuas hnub nyoog tau txais thaum tus neeg siv membership tseem active
* **Kev teem sijhawm ntse** — Renewal jobs raug queue raws hnub domain tas sij hawm

#### Customer DNS Management {#customer-dns-management}

## Installation {#installation}

1. Nco ntsoov tias **Multisite Ultimate v2.4.12+** twb raug nruab thiab network-activated lawm
2. Upload lub plugin mus rau `/wp-content/plugins/ultimate-multisite-domain-seller/`
3. Network-activate lub plugin los ntawm **Plugins** hauv WordPress Network Admin
4. **Setup Wizard** yuav qhib nws tus kheej thaum activate thawj zaug — ua raws cov kauj ruam kom:
   * Txuas koj tus domain registrar
   * Kuaj qhov kev txuas
   * Import cov TLD uas muaj thiab tus nqi wholesale
   * Tsim koj thawj domain khoom nrog pricing markup
5. Ntxiv daim teb **Domain Selection** rau koj daim checkout form hauv **Ultimate Multisite > Checkout Forms**

## Cov Lus Nug Uas Nquag Nug {#frequently-asked-questions}

### Yuav tsum muaj Multisite Ultimate version twg? {#what-version-of-multisite-ultimate-is-required}

Version 2.4.12 lossis siab dua. Lub addon tsis sib haum nrog WP Ultimo v1.x lossis Ultimate Multisite versions qis dua 2.4.12.

### Cov domain providers twg raug txhawb? {#which-domain-providers-are-supported}

Muaj cuaj tus registrars raug txhawb: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo**, thiab **Enom**. Peev xwm txawv raws provider vim registrar APIs nthuav tawm cov operations sib txawv; saib setup guide kom pom overview ib provider-rau-ib-provider. Lub addon tus Integration Registry architecture ua rau yooj yim ntxiv custom providers — saib [developer guide](/developer/integration-guide/custom-registrar).

### Kuv puas tuaj yeem muab WHOIS privacy protection? {#can-i-offer-whois-privacy-protection}

Tau. Txhua domain khoom muaj WHOIS Privacy setting nrog peb hom: Disabled, Always Included, lossis Customer Choice. Thaum teeb rau Customer Choice, customers yuav pom ib lub checkbox thaum checkout nrog tus nqi (lossis "Free" rau Namecheap's WhoisGuard). Lub privacy flag yuav raug xa mus rau registrar cia li thaum lub sijhawm registration.

### Kuv puas teeb tau tus nqi sib txawv rau TLDs sib txawv? {#can-i-set-different-pricing-for-different-tlds}

Tau. Tsim domain khoom cais rau TLD groups sib txawv. Txhua khoom muaj tau nws tus kheej hom markup (percentage, fixed markup, lossis fixed price), cov TLD uas txhawb, thiab introductory pricing.

### Customers pom cov kev xaiv checkout dab tsi? {#what-checkout-options-do-customers-see}

Daim teb Domain Selection muab peb tabs rau customers: **Subdomain** (siv ib subdomain dawb hauv koj lub network), **Register New Domain** (nrhiav thiab register ib domain tshiab), thiab **Existing Domain** (map ib domain uas lawv twb muaj lawm). Tsuas yog cov hom uas koj enable xwb thiaj yuav tshwm.

### Nws puas txhawb automatic renewals? {#does-it-support-automatic-renewals}

Tau. Domains tuaj yeem raug teeb kom auto-renew thiab raug khi rau customer tus membership status. Renewal payments raug process los ntawm customer lub payment gateway uas twb muaj lawm.

### Customers puas tswj tau lawv cov DNS records? {#can-customers-manage-their-dns-records}

Tau. Customers tau ib qho DNS management interface hauv lawv dashboard kom ntxiv, kho, thiab rho tawm A, AAAA, CNAME, MX, thiab TXT records. DNS management muaj nrog OpenSRS, ResellerClub, Enom, HostAfrica, thiab Openprovider. Hostinger DNS rau hosted domains raug tswj los ntawm core Hostinger domain-mapping integration.

### Puas muaj setup wizard? {#is-there-a-setup-wizard}

Muaj. Setup wizard khiav nws tus kheej thaum activate thawj zaug thiab coj koj mus txuas ib registrar, import TLDs, thiab tsim koj thawj domain khoom. Koj kuj rov khiav tau wizard los ntawm addon settings.
