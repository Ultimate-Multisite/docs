# Tus Xaiv Lus {#language-selector}

Multisite Ultimate: Tus Xaiv Lus thaum Sau Npe

Ntxiv ib daim teb xaiv lus rau hauv txoj kev sau npe.

## Kev teeb tsa {#installation}

1. Upload 'ultimate-multisite-language-selector' mus rau hauv directory '/wp-content/plugins/'
2. Qhib plugin los ntawm menu 'Plugins' hauv koj WordPress Network Admin

## Phau Ntawv Qhia Siv {#usage-guide}

Nrog qhov add-on pub dawb no, koj cia tau koj cov neeg siv xaiv hom lus uas lawv xav pom hauv txoj kev sau npe thiab ntawm nplooj ntawv nkag mus.

Tom qab [qhib koj daim license](../../user-guide/getting-started/installing-ultimate-multisite.md) lawm, koj yuav nruab tau cov add-ons ntawm Ultimate Multisite uas koj daim license pub koj nkag tau.

Mus rau koj nplooj ntawv teeb tsa Ultimate Multisite, ces ntawm menu sab laug, nyem daim label _Add-ons_.

![Nplooj ntawv Add-ons](/img/admin/addons-page.png)

_Mus rau nplooj ntawv Add-ons kom pib nruab Tus Xaiv Lus_

Ntawm ntawd, koj yuav pom tag nrho cov add-ons uas peb muab.

Cov add-ons pub dawb muaj rau txhua tus neeg siv khoom ntawm peb, txawm lawv sau npe rau hom subscription plan twg los xij.

Nyem lub pob _Install it Now!_ hauv qab daim thumbnail ntawm add-on ntawd. Ces qhib nws ntawm nplooj ntawv Plugins.

![Nplooj ntawv Add-ons uas muaj lub pob nruab](/img/admin/addons-page.png)

_Nyem lub pob kom nruab add-on_

_Qhib nws ntawm koj nplooj ntawv Plugins_

Tam sim no, mus rau menu Settings ntawm Ultimate Multisite thiab nkag rau tab _Network Settings_. Nyob ntawd, hauv qab kawg ntawm nplooj ntawv, koj yuav pom ib seem tshiab hu ua _Languages_.

## Teeb Lus los ntawm URL {#setting-the-language-via-the-url}

Koj kuj hloov tau hom lus ntawm ib nplooj ntawv los ntawm kev xa locale parameter mus rau URL. Yog koj xav redirect mus rau nplooj ntawv sau npe ua lus Italian, piv txwv li, ntxiv ?locale=it_IT rau URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Hom lus ntawd yuav tsum raug nruab rau hauv koj WordPress network kom qhov no ua haujlwm tau.
