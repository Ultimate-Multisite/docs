# Material WP

WordPress kev sib tshuam tswj hwm yeej zoo ib yam tau ntev lawm. Tab sis koj puas tau xav tias ib qho kev sib tshuam tswj hwm qauv Material Design yuav zoo li cas? 

Material WP yog ib lub admin theme uas tsim los muab qhov tsos thiab kev xav ntawd rau koj, pab koj muab Dashboard uas kho tau raws li xav tau ntau dua rau koj cov neeg siv khoom.

## Cov nta

- Txhawb Custom CSS
- Teeb styling tau yooj yim
- Rov teeb kev txiav txim ntawm ntawv qhia zaub mov
- Import/export tau yooj yim

## Ntaub ntawv qhia

**Kho koj Admin Panel nrog Material WP**

## Kev teeb tsa

1. Upload 'material-wp' rau hauv '/wp-content/plugins/' directory
2. Qhib qhov txuas ntxiv los ntawm 'Cov txuas ntxiv' menu hauv WordPress

## Phau ntawv qhia siv

WordPress kev sib tshuam tswj hwm yeej zoo ib yam tau ntev lawm. Tab sis koj puas tau xav tias ib qho kev sib tshuam tswj hwm qauv Material Design yuav zoo li cas?
