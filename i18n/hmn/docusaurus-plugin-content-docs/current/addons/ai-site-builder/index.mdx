# AI Site Builder

ultimate-multisite-ai-site-builder

## Cov kev yuav tsum muaj

- WordPress 5.3 lossis siab dua
- PHP 7.4 lossis siab dua
- Multisite Ultimate plugin (qhib siv lawm)

## Kev teeb tsa

1. Upload cov ntaub ntawv ntawm qhov ntxiv mus rau koj phau ntawv `/wp-content/plugins/`
2. Qhib siv plugin los ntawm menu 'Cov Plugin' hauv WordPress
3. Teeb tsa cov kev xaiv ntawm qhov ntxiv hauv tus thawj tswj network

## Kev tsim kho

### Kev teeb tsa

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Kev xeem

Qhov ntxiv no muaj ib lub moj khaum kev xeem uas dav:

- **Kev Xeem Unit**: Cov kev xeem raws PHPUnit nrog WordPress test suite
- **Chav Kawm Xeem Pib**: Muab cov helper methods rau cov xwm txheej xeem uas siv ntau
- **Kev Ntsuas Kev Npog Xeem**: Teeb tsa los taug qab code coverage
- **Kev Xeem Multisite**: Cov kev xeem khiav hauv ib puag ncig multisite

### Cov qauv code

- **WordPress Coding Standards**: Yuam siv los ntawm PHPCS
- **Kev sib haum nrog PHP 7.4+**: Txhawb cov nta PHP tshiab
- **Kev tshuaj xyuas static**: Kev sib txuas PHPStan rau kev xyuas hom
- **Kev kho code kom tshiab**: Cov cai Rector rau kev upgrade PHP

### Txheej txheem build

- **Kev ua cov asset**: Minification rau cov ntaub ntawv CSS/JS
- **Kev txhais lus**: Tsim ntaub ntawv POT
- **Kev ntim pob**: Tsim archive rau kev faib tawm
- **Hom kev tsim kho**: Cov asset uas tsis tau minify rau kev debug

## Kev teeb tsa configuration

Qhov ntxiv txhawb ntau yam kev xaiv configuration uas nkag tau los ntawm tus interface ntawm tus thawj tswj network.

## Hooks thiab Filters

### Actions

- `ultimate-multisite-ai-site-builder_init` - Tso tawm tom qab pib qhov ntxiv lawm
- `ultimate-multisite-ai-site-builder_loaded` - Tso tawm tom qab txhua cov ntaub ntawv ntawm qhov ntxiv raug loaded lawm

### Filters

- `ultimate-multisite-ai-site-builder_settings` - Hloov cov settings ntawm qhov ntxiv
- `ultimate-multisite-ai-site-builder_enabled` - Override qhov xwm txheej qhib/kaw ntawm qhov ntxiv
