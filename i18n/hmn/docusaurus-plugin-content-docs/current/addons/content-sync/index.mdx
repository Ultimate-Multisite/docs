# Kev Sync Cov Ntsiab Lus

Multisite Ultimate: Kev Sync Cov Ntsiab Lus

Qhov txuas ntxiv no rau Multisite Ultimate tso cai rau cov thawj tswj kev sib txuas kom synchronize cov ntsiab lus ntawm ib lub vev xaib twg mus rau lwm cov vev xaib hauv kev sib txuas. Nws txhawb txhua hom ntawv tshaj tawm xws li cov ntawv tshaj tawm, nplooj ntawv, hom ntawv tshaj tawm tshwj xeeb, thiab Elementor templates. Qhov txuas ntxiv no muaj ib qho wizard interface ntau kauj ruam rau xaiv cov ntsiab lus thiab cov vev xaib hom phiaj, nrog batch processing thiab kev qhia kev nce qib tam sim ntawd.

## Cov Nta

* Kev Xaiv Cov Ntsiab Lus: Daim card grid pom tau nrog thumbnails thiab kev lim raws hom ntawv tshaj tawm
* Txhua Hom Ntawv Tshaj Tawm: Sync cov ntawv tshaj tawm, nplooj ntawv, cov khoom, Elementor templates, thiab txhua hom ntawv tshaj tawm tshwj xeeb
* Kev Xaiv Vev Xaib: Xaiv cov vev xaib uas yuav sync cov ntsiab lus mus rau, nrog kev cais tawm lub vev xaib qhov chaw los thiab kev xaiv ua ntej cov vev xaib me
* Batch Processing: Sync mus rau 10 lub vev xaib ib zaug nrog kev hloov tshiab txog kev nce qib
* Media Sync: Luam thiab remap cov ntaub ntawv media (duab, videos) ntawm cov ntsiab lus hauv ntawv tshaj tawm mus rau cov vev xaib hom phiaj
* Taxonomy Sync: Khaws categories, tags, thiab custom taxonomies cia
* Elementor Support: Txhawb Elementor templates tag nrho nrog kev tsim CSS dua tshiab thiab kev remap template ID
* Kev Daws Teeb Meem Sib Tsoo: Muaj kev xaiv los hloov cov ntsiab lus uas twb muaj lawm lossis hla nws
* Kev Tswj Cache: Rho Elementor cache tawm ntawm cov vev xaib hom phiaj tom qab sync tas

## Yam Yuav Tsum Muaj

* WordPress 5.3 lossis siab dua
* PHP 7.4 lossis siab dua
* Multisite Ultimate plugin (qhib lawm)
* Elementor plugin (tsis yuam, yuav tsum muaj rau Elementor template sync)

## Kev Nruab

1. Upload 'ultimate-multisite-content-sync' rau hauv '/wp-content/plugins/' directory
2. Qhib lub plugin ntawm 'Plugins' menu hauv koj WordPress Network Admin
3. Kho cov chaw teeb hauv Ultimate Multisite → Settings → Content Sync
4. Mus rau Ultimate Multisite → Content Sync kom pib sync
