## Rate Limiting {#rate-limiting}

Ultimate Multisite: Captcha v1.5.0 qhia ib hard-stop rate limiter uas tiv thaiv koj network ntawm brute-force thiab card-testing attacks.

### Nws Ua Haujlwm Li Cas {#how-it-works}

Rate limiter suav txhua qhov GET thiab POST request ntawm cov chaw uas captcha tiv thaiv:

* **WordPress login endpoints** — wp-login, register, lost-password, comments
* **WooCommerce checkout** — my-account, checkout, pay-for-order
* **Ultimate Multisite** — checkout, inline-login

Thaum ib tus qhua tshaj rate limit, lub plugin teb nrog:

* **HTTP 429 (Too Many Requests)** status code
* **Retry-After** header qhia thaum twg rov sim dua
* **Randomised tarpit sleep** (1–5 vib nas this raws default, hard-capped ntawm 15 vib nas this) kom ua rau attackers qeeb

### Configuration {#configuration}

#### Tarpit Timing {#tarpit-timing}

Tswj qhov randomised delay uas siv rau rate-limited requests:

* **`cap_rate_limit_tarpit_min`** — Yam tsawg kawg sleep duration hauv vib nas this (default: 1)
* **`cap_rate_limit_tarpit_max`** — Yam siab tshaj sleep duration hauv vib nas this (default: 5, hard-capped ntawm 15)

Cov settings no muaj nyob hauv **Ultimate Multisite → Settings → Captcha**.

### Kev Nthuav Tau {#extensibility}

#### IP Whitelist Filter {#ip-whitelist-filter}

Zam cov trusted IP ranges ntawm rate limiting:

```php
add_filter( 'wu_cap_rate_limit_whitelist_ip', function( $is_whitelisted, $ip ) {
    // Whitelist your monitoring service
    if ( $ip === '203.0.113.42' ) {
        return true;
    }
    return $is_whitelisted;
}, 10, 2 );
```

#### Pre-Block Action {#pre-block-action}

Khiav custom logic tam sim ua ntej xa ib rate-limit block:

```php
add_action( 'wu_cap_rate_limit_will_block', function( $ip, $surface ) {
    // Log the block, notify admins, or update custom metrics
    error_log( "Rate limit triggered for $ip on $surface" );
}, 10, 2 );
```

## Client IP Detection & Trusted Proxies {#client-ip-detection--trusted-proxies}

Ultimate Multisite: Captcha v1.5.0 qhia spoof-resistant client-IP detection. Txoj method `Captcha_Core::get_client_ip()` tam sim no yog lub hauv paus tseeb nkaus xwb rau visitor IP attribution thoob plaws:

* Cov yuam sij bucket rau kev txwv tus nqi
* Tus muab Captcha `remoteip` (reCAPTCHA, hCaptcha `siteverify`)
* Cov IP hash rau kev txheeb cais

### Qauv Kev Ntseeg Siab {#trust-model}

Qhov txuas ntxiv no yuam siv ib qib kev ntseeg IP uas nruj heev:

1. **REMOTE_ADDR** — Ib txwm ntseeg siab (IP ntawm tus peer uas txuas ncaj qha)
2. **CF-Connecting-IP** — Tsuas lees siv thaum:
   - Tus peer uas txuas ncaj qha nyob hauv Cloudflare IP range tam sim no
   - `cap_trust_cloudflare_headers` tau qhib (default: OFF)
3. **X-Forwarded-For** — Tsuas lees siv thaum:
   - Tus peer uas txuas ncaj qha nyob hauv koj daim trusted-proxy list uas admin teeb
   - Lub header raug parse ntawm sab xis mus rau sab laug, hla cov trusted/Cloudflare hops

### Kev Teeb Tsa {#configuration-1}

#### Kev Ntseeg Cloudflare Header {#cloudflare-header-trust}

Qhib kev ntseeg rau Cloudflare lub `CF-Connecting-IP` header:

* **`cap_trust_cloudflare_headers`** — Teeb rau ON kom xaiv siv kev ntseeg `CF-Connecting-IP`
  - Default: OFF (tsis qhib)
  - Qhov txuas ntxiv no los nrog ib daim Cloudflare CIDR snapshot uas twb bundled lawm
  - CIDR ranges raug refresh txhua lub lim tiam ntawm wp-cron
  - Siv bundled fallback yog tias refresh tsis tiav

#### Kev Teeb Tsa Trusted Proxy {#trusted-proxy-configuration}

Txhais koj cov proxy thiab load-balancer nyob pem hauv ntej:

* **`cap_trusted_proxies`** — Thaj ntawv rau CIDRs lossis IPs ncaj qha (ib kab ib qho)
  - Tso comments tau (cov kab pib nrog `#`)
  - Yog tsis muaj qhov setting no, `X-Forwarded-For` yuav raug tsis quav ntsej txawm tias rate limiter twb qhib lawm
  - Piv txwv:
    ```
    # Our load balancer
    192.0.2.0/24

    # Backup proxy
    198.51.100.50
    ```

#### Kev Tshawb Pom Nws Tus Kheej Thaum Qhib Thawj Zaug {#first-enable-auto-detection}

Thaum koj qhib rate limiter thawj zaug, qhov txuas ntxiv no yuav tshawb pom koj qhov Cloudflare thiab proxy posture uas zoo li yog tiag, thiab qhia ib daim admin notice uas nias ib zaug tau "Siv cov settings uas tshawb pom".

* Qhov txuas ntxiv no **yeej tsis overwrite** koj cov nqi uas twb khaws tseg lawm
* Yog tias traffic tom qab qhia tias koj qhov config tsis phim qhov muaj tiag lawm (piv txwv, Cloudflare hloov CIDR ranges), ib daim mismatch notice uas tshem tsis tau yuav qhia qhov update uas pom zoo

### Vim Li Cas Qhov No Tseem Ceeb {#why-this-matters}

Ib qho qauv kev ntseeg IP uas nruj yuav kaw tau ib txoj kev spoofing tseem ceeb: cov request ncaj qha mus rau origin-server uas nqa ib lub `CF-Connecting-IP` header cuav yav dhau los yuav raug muab tso rau bucket raws IP cuav ntawd, tsis yog tus peer tiag. Qhov no tseem ceeb tshwj xeeb rau:

* **Kev txwv tus nqi** — Cov neeg tawm tsam yuav bypass limits tsis tau los ntawm kev spoof IPs
* **Kev txheeb cais** — Koj cov metrics kev tawm tsam qhia IPs ntawm cov neeg tuaj xyuas tiag, tsis yog cov cuav
* **Kev txheeb xyuas Captcha** — Cov providers tau txais IP raug ntawm tus neeg tuaj xyuas rau kev ntsuas kev pheej hmoo
