# Mailster

Hloov koj lub multisite network mus ua ib lub tshuab muaj zog rau kev tsim email list nrog Ultimate Multisite Mailster Integration addon. Ntxiv cov neeg yuav khoom tshiab rau hauv koj cov Mailster email lists tau txais thaum lawv sau npe lossis yuav khoom, tsim kom muaj kev txuas tsis tu ncua ntawm koj lub membership platform thiab koj txoj kev npaj email marketing.

## Cov yam ntxwv tseem ceeb {#key-features}

**Sync cov neeg sau npe tau txais**
* Ntxiv cov neeg yuav khoom rau hauv Mailster lists tau txais thaum lawv sau npe
* Xaiv kom sync tam sim thaum sau npe lossis tos kom paub tseeb tias tau them nyiaj lawm
* Khaws koj cov email lists tshiab thiab hloov tshiab tas li yam tsis tas ua manually

**Kev tswj list uas yoog tau**
* Teem global default lists rau txhua tus neeg yuav khoom tshiab
* Override cov default nrog product-specific lists rau cov campaign uas tsom tau meej
* Tswj list assignments tau yooj yim nrog Mailster lub interface uas paub siv lawm

**Cov kev xaiv opt-in uas raws GDPR**
* Xaiv ntawm automatic subscription lossis kev tso cai raws checkbox
* Qhib double opt-in rau cov subscriptions uas tau lees paub
* Qhia/zais opt-in checkbox raws li koj cov kev xav tau compliance

**Kev mapping ntaub ntawv ntse**
* Sync cov npe neeg yuav khoom thiab ntaub ntawv tiv tauj tau txais
* Map billing addresses rau subscriber fields
* Taug qab hnub sau npe rau cov campaign uas tsom tau meej

**Kev tswj ntawm qib product**
* Muab lists txawv rau products txawv
* Tsim cov campaign uas tsom tau meej raws li yam uas cov neeg yuav khoom yuav
* Disable Mailster rau products tshwj xeeb thaum xav tau

**Kev paub siv zoo li kws tshaj lij**
* Native Mailster UI integration rau kev xaiv list
* Kev logging ntxaws rau troubleshooting thiab verification
* Hloov tshiab cov subscribers uas twb muaj lawm yam tsis tu ncua

## Zoo meej rau {#perfect-for}

* Cov tswv membership site uas tab tom tsim email lists
* SaaS platforms uas xav nurture cov neeg yuav khoom tshiab

## Nws ua haujlwm li cas {#how-it-works}

1. Nruab thiab qhib lub addon rau ntawm koj lub Ultimate Multisite network
2. Configure global default lists hauv Ultimate Multisite settings
3. Yog xav tau, override nrog product-specific lists
4. Xaiv koj hom opt-in thiab lub sijhawm subscription
5. Saib koj cov Mailster lists loj hlob tau txais!

## Kev nruab {#installation}

1. Upload cov plugin files mus rau koj lub Ultimate Multisite addons directory
2. Xyuas kom Mailster raug nruab thiab active rau ntawm koj lub main site
3. Qhib lub addon hauv Ultimate Multisite addons page
4. Mus rau Settings > Mailster Integration los configure
5. Xaiv koj cov default lists thiab configure opt-in preferences

## Cov lus nug uas nquag nug {#frequently-asked-questions}

## Lub addon no puas ua haujlwm tsis muaj Ultimate Multisite? {#does-this-addon-work-without-ultimate-multisite}

Tsis tau, lub addon no yuav tsum muaj Ultimate Multisite thiaj ua haujlwm tau. Nws raug tsim tshwj xeeb los integrate Mailster nrog Ultimate Multisite lub customer thiab product system.

## Kuv puas yuav tsum nruab Mailster rau txhua sites? {#do-i-need-mailster-installed-on-all-sites}

Tsis tas, Mailster tsuas yog yuav tsum raug nruab thiab activated rau ntawm koj lub main network site xwb. Lub addon yuav automatically txuas mus rau nws los ntawm txhua subsites.

## Kuv puas muab tau lists txawv rau products txawv? {#can-i-assign-different-lists-to-different-products}

Tau kawg! Koj tuaj yeem teem global default lists, ces override lawv raws ib product zuj zus. Qhov no tso cai rau koj tsim email campaigns uas tsom tau zoo heev raws li yam uas cov neeg yuav khoom yuav.

## Yuav ua li cas yog ib tus neeg yuav khoom twb muaj nyob hauv Mailster lawm? {#what-happens-if-a-customer-already-exists-in-mailster}

Raws default, lub addon yuav update cov subscribers uas twb muaj lawm thiab ntxiv lawv rau hauv lists tshiab. Koj tuaj yeem disable qhov no hauv settings yog koj nyiam hla cov subscribers uas twb muaj lawm.

## Qhov no puas raws GDPR? {#is-this-gdpr-compliant}

Raws! Lub addon muaj checkbox-based opt-in mode uas cov neeg yuav khoom yuav tsum tso cai meej meej kom tau txais emails. Qhov no ua tau raws GDPR requirements thaum siv nrog privacy policies uas tsim nyog.

## Kuv puas siv tau double opt-in? {#can-i-use-double-opt-in}

Tau kawg! Qhib double opt-in setting thiab Mailster yuav xa confirmation emails ua ntej qhib subscriptions.
