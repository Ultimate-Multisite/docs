# Support Tickets

**Support Tickets** yog ib qho addon muaj zog rau Ultimate Multisite uas ntxiv ib lub tshuab support ticket puv ntoob rau koj lub WordPress network. Zoo kawg rau SaaS platforms, cov neeg muab WaaS (Website as a Service), thiab multisite networks uas yuav tsum muab kev pab txhawb rau cov neeg siv khoom ntawm ob theem: network thiab site nyias tus.

## Vim Li Cas Xaiv Support Tickets? {#why-choose-support-tickets}

* **Kev Pab Txhawb Ntau Theem**: Network admins saib xyuas tickets thoob plaws network thaum site owners tswj lawv tus kheej kev pab txhawb rau cov neeg siv khoom
* **Kev Ua Haujlwm Zoo Li Kws Tshaj Lij**: Tswj tag nrho lub neej ntawm ticket nrog priorities, statuses, thiab assignments
* **Kev Paub Siv Rau Neeg Siv Khoom**: Interface zoo nkauj thiab teb haum txhua qhov screen uas ua rau cov neeg siv khoom thov kev pab tau yooj yim
* **Ruaj Ntseg & Ntseeg Tau**: Tsim nrog WordPress cov kev coj ua zoo tshaj plaws rau kev ruaj ntseg thiab kho kom database performance khiav tau zoo
* **Kev Sib Txuas Du Lug**: Ua haujlwm zoo kawg nrog Ultimate Multisite kev tswj customer thiab site

## Cov Nta Tseem Ceeb {#key-features}

**Kev Tswj Ticket Puv Ntoob**

* Lub tshuab muab lej ticket tshwj xeeb kom nrhiav tau yooj yim
* Kho tau priority levels (Low, Normal, High, Critical)
* Status workflow puv ntoob (Open, In Progress, Waiting, Resolved, Closed)
* Kev faib ticket ua pawg (General, Technical, Billing, Feature Request, Bug Report)
* Muab tickets rau support agents nrog kev auto-assignment xaiv tau

**Kev Sib Tham Muaj Xov**

* Muaj conversation threading tag nrho rau ticket replies
* Kev tsim qauv sib txawv rau staff thiab customer responses
* Private staff notes uas customers pom tsis tau
* Interface siv AJAX ua haujlwm tam sim ntawd
* Text editor tshiab nrog formatting options

**File Attachments**

* Upload file ruaj ntseg nrog kev txhawb drag-and-drop
* Kho tau kev txwv file type thiab size
* Kev ruaj ntseg siab nrog malicious file detection
* Downloads raug tiv thaiv nrog access controls

**Kev Ceeb Toom Ntse**

* Xa email alerts cia li mus rau new tickets, replies, thiab status changes
* Kho tau email templates
* Notifications raws role rau staff thiab customers
* Template variables hloov tau rau cov lus uas phim tus neeg

**Admin Interface Muaj Zog**

* Ticket list siab nrog sorting thiab filtering
* Quick edit rau status, priority, thiab assignments
* Bulk actions rau kev tswj kom ua tau sai
* Statistics dashboard nrog performance metrics
* Settings panel puv ntoob

**Customer Portal Zoo Nkauj**

* Interface huv, responsive rau txhua lub cuab yeej
* Tsim mobile-first
* Advanced search thiab filtering
* Editor tshiab nrog formatting toolbar
* Ticket submission forms siv yooj yim

**Kev Sib Txuas Yooj Yim**

* Siv shortcodes lossis Ultimate Multisite UI elements
* Muab ticket forms thiab lists tso rau txhua qhov chaw ntawm koj lub site
* Ua haujlwm du lug nrog multisite network structure
* Nthuav tau nrog hooks thiab filters

## Zoo Kawg Rau {#perfect-for}

## Kev Tsim Nruab {#installation}

1. Xyuas kom Ultimate Multisite plugin twb raug nruab thiab qhib rau koj lub WordPress network lawm
2. Upload cov addon files mus rau koj lub `/wp-content/plugins/` directory
3. Network activate lub plugin los ntawm 'Plugins' menu hauv WordPress
4. Mus rau Network Admin > Support Tickets los configure koj cov settings

## Cov Lus Nug Nquag Nug {#frequently-asked-questions}

## Qhov no puas yuav tsum muaj Ultimate Multisite? {#does-this-require-ultimate-multisite}

Yog, addon no yuav tsum muaj Ultimate Multisite nruab thiab active. Nws sib txuas tob nrog Ultimate Multisite cov nta kev tswj customer thiab site.

## Site owners puas tswj tau lawv tus kheej customer support? {#can-site-owners-handle-their-own-customer-support}

Tau kawg! Site owners siv tau lub ticket system los muab support rau lawv tus kheej customers, thaum network admins tswj tau support tickets thoob plaws network.

## Customers xa tickets li cas? {#how-do-customers-submit-tickets}

Koj ntxiv tau ticket submission forms rau txhua qhov chaw ntawm koj lub site siv shortcodes lossis Ultimate Multisite UI elements. Customers tsuas sau daim form thiab xa lawv qhov request xwb.

## Kuv puas kho tau ticket statuses thiab priorities? {#can-i-customize-the-ticket-statuses-and-priorities}

Tau! Koj kho tau tag nrho priority levels, statuses, thiab ticket types hauv network admin settings kom phim koj support workflow.

## Nws puas txhawb file attachments? {#does-it-support-file-attachments}

Yog, ob tog customers thiab staff puav leej attach tau files rau tickets thiab responses. Koj muaj kev tswj tag nrho rau allowed file types thiab size limits kom muaj kev ruaj ntseg.

## Notifications raug tswj li cas? {#how-are-notifications-handled}

Lub system cia li xa email notifications rau new tickets, replies, thiab status changes. Koj kho tau tag nrho email templates hauv settings.

## Kuv puas assign tau tickets rau support agents tshwj xeeb? {#can-i-assign-tickets-to-specific-support-agents}

Tau, koj manually assign tau tickets rau agents lossis qhib auto-assignment kom faib tickets cia li mus.

## Puas muaj private notes rau staff? {#are-there-private-notes-for-staff}

Yog, staff members ntxiv tau private replies uas tsuas pom rau lwm tus staff members xwb, tsis pom rau customers.

## Nws puas haum mobile? {#is-it-mobile-friendly}

Tau kawg! Tag nrho interface yog tsim raws mobile-first approach thiab ua haujlwm zoo nkauj rau txhua lub cuab yeej.

## Kuv puas tso tau ticket lists thiab forms siv shortcodes? {#can-i-display-ticket-lists-and-forms-using-shortcodes}

Tau! Shortcodes muaj:
* `[wu_support_tickets]` - Qhia customer ticket list
* `[wu_support_ticket_form]` - Qhia new ticket submission form
* `[wu_support_ticket_view ticket_id="123"]` - Qhia ticket tshwj xeeb

## Upload tau file types twg? {#what-file-types-can-be-uploaded}

Koj configure tau allowed file types hauv settings. Lub system muaj advanced security scanning los tiv thaiv malicious files.

## Kuv puas export tau ticket data? {#can-i-export-ticket-data}

Tau, addon no muaj export capabilities rau ticket data thiab statistics.
