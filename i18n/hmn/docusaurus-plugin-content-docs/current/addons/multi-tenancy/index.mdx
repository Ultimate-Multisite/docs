# Kev Xauj Ntau Tus

Faib cov chaw me mus thoob cov servers thiab databases kom ua haujlwm ceev dua thiab muaj kev nyab xeeb dua

## Cov ntsiab lus tseem ceeb ntawm Version 1.2.0 tso tawm {#version-120-release-highlights}

Multi-Tenancy 1.2.0 ntxiv lub hauv paus neeg xauj ywj pheej rau cov txheej txheem cais kev tsim tiag:

- Kev nkag rau neeg xauj yam tsis khaws xeev nrog SSO kev mus saib uas kav luv thiab tiv thaiv replay.
- Kev cais database thiab filesystem rau ib qho chaw me rau cov neeg xauj ywj pheej.
- Hostinger peev xwm kev xauj ntau tus rau kev npaj neeg xauj uas paub host.
- Cov rooj vag kuaj xyuas kev tsiv teb tsaws ywj pheej rau kev kuaj cov ntaub ntawv qub thiab kev ua async push.
- Kev khi database host uas teeb tau rau tib lub tshuab rau Bedrock, FrankenPHP, thiab localhost installs.
- Kev txhim kho tenant bootstrap, database writer, schema bootstrap, thiab kev npaj neeg siv.
- Kev tshem tawm ywj pheej uas nyab xeeb dua uas ntxuav cov ntaub ntawv pov thawj ntawm neeg xauj.

Saib cov lus qhia rau neeg siv rau [txheej txheem kev tsiv teb tsaws ywj pheej](/user-guide/getting-started/sovereign-migration-workflow), [kev cais kev xauj ntau tus](/user-guide/configuration/multi-tenancy-isolation), thiab [kev khiav haujlwm neeg xauj ywj pheej](/user-guide/administration/sovereign-tenant-operations).

## Yam yuav tsum muaj {#requirements}

- WordPress 5.3 lossis siab dua
- PHP 7.4 lossis siab dua
- Multisite Ultimate plugin (qhib siv)

## Kev teeb tsa {#installation}

1. Upload cov ntaub ntawv addon mus rau koj phau `/wp-content/plugins/`
2. Qhib lub plugin los ntawm daim ntawv qhia 'Plugins' hauv WordPress
3. Teeb cov chaw addon hauv network admin

### Kev sim {#testing}

Lub addon muaj ib lub moj khaum kev sim uas dav:

- **Kev sim Unit**: Cov kev sim raws PHPUnit nrog WordPress test suite

- **Test Coverage**: Teeb kom taug qab code coverage
- **Kev sim Multisite**: Cov kev sim khiav hauv ib puag ncig multisite

- **WordPress Coding Standards**: Yuam siv los ntawm PHPCS

- **Kev hloov code kom tshiab**: Rector rules rau PHP upgrades

## Kev teeb tsa {#configuration}

Lub addon txhawb ntau yam kev xaiv configuration uas nkag tau los ntawm network admin interface.

### Kev teeb tsa neeg xauj ywj pheej {#sovereign-tenant-configuration}

Rau cov neeg xauj ywj pheej, teeb tus neeg xauj database, filesystem root, database host binding, thiab host capability ua ntej routing traffic mus rau tus neeg xauj. Khiav kev kuaj xyuas kev tsiv teb tsaws tom qab txhua qhov kev hloov uas cuam tshuam rau database credentials, host binding, routing, lossis user provisioning.

Siv `localhost` tsuas yog thaum tenant runtime thiab database grants cia siab rau local socket lossis kev nkag ntawm tib lub tshuab. Siv `127.0.0.1` lossis service hostname thaum qhov ntawd yog grant host uas koj provider tau teeb.

### Hostinger capability {#hostinger-capability}

Hostinger multi-tenancy capability ua haujlwm nrog shared Hostinger integration token. Paub meej tias domain muaj nyob hauv hPanel, DNS taw mus rau Hostinger account, thiab tenant database settings phim hPanel ua ntej kev kuaj xyuas.

## Hooks thiab Filters {#hooks-and-filters}

### Filters {#filters}
