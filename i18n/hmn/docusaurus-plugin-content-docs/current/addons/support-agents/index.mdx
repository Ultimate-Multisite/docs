# Cov Neeg Sawv Cev Txhawb Nqa

Kev tswj ib lub network tag nrho koj ib leeg tsis yog ib txoj haujlwm yooj yim. Thaum koj lub network pib loj hlob, koj yuav pib pom tias kev thov kev txhawb nqa nce ntxiv. Nws yuav yog ib lub tswv yim zoo los coj cov neeg uas yuav pab koj txo koj txoj haujlwm. Txawm li ntawd los, muab kev nkag/kev tso cai tag nrho yog ib qho muaj kev pheej hmoo thiab tej zaum yuav ua rau koj lub network tsis ruaj ntseg. Qhov ntawd yog qhov uas qhov Support Agent add-on no los pab.

Qhov add-on no tso cai rau koj tsim ib hom WordPress user tshiab hu ua Support Agent, thiab muab kev tswj tag nrho rau koj txog cov kev tso cai uas tus user ntawd muaj. Kev muab kev tso cai yooj yim ib yam li qhib ib lub toggle button, thiab kaw nws yog tias koj txiav txim siab tshem tawm.

## Cov yam ntxwv

- Daim ntawv teev cais ntawm WordPress & Ultimate Multisite kev tso cai
- Qhib & kaw kev nkag tau yooj yim
- Tsim tau Agent accounts tsis txwv tus naj npawb

## Kev teeb tsa

1. Upload 'ultimate-multisite-support-agents' mus rau hauv '/wp-content/plugins/' directory
2. Activate qhov plugin los ntawm 'Plugins' menu hauv WordPress
3. Ua raws Wizard ib kauj ruam zuj zus los teeb tsa qhov plugin
