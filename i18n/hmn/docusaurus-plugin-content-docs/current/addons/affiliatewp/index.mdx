# AffiliateWP

Multisite Ultimate: Kev sib txuas AffiliateWP

Siv AffiliateWP uas muaj zog los pab kom cov neeg siv ntawm koj lub Ultimate Multisite Network loj hlob!

## Kev teeb tsa {#installation}

1. Upload 'ultimate-multisite-git ' rau hauv directory '/wp-content/plugins/'
2. Qhib plugin los ntawm menu 'Plugins' hauv WordPress
3. Ua raws tus pab teeb tsa ib kauj ruam zuj zus los teeb plugin kom tiav

## Phau ntawv qhia siv {#usage-guide}

[AffiliateWP](https://affiliatewp.com/) tau dhau los ua txoj kev daws teeb meem uas neeg nyiam siv thaum tsim ib qho affiliate program rau koj lub lag luam online. Ultimate Multisite: AffiliateWP yog tus choj uas yuav pab kom koj tsim tau tib yam ntawd rau koj lub network premium.

## Pib siv {#getting-started}

Ua ntej tshaj, koj **yuav tsum muaj** [**AffiliateWP**](https://affiliatewp.com/) **qhib lawm**! Koj yuav siv tsis tau peb lub add-on yog tias koj tsis muaj **lub plugin uas nws sib txuas nrog**. Yog tias koj tsis npaj yuav cia koj cov client siv nws rau ntawm lawv cov site thiab, AffiliateWP yuav tsum qhib tsuas yog rau koj lub site tseem ceeb xwb.

Tsis tas li ntawd, koj yuav tsum muaj ob qho **Ultimate Multisite** thiab **Ultimate Multisite: AffiliateWP** **qhib** **thoob plaws network**.

## Ntxiv Ultimate Multisite ua ib qho AffiliateWP Kev sib txuas {#adding-ultimate-multisite-as-an-affiliatewp-integration}

Ntawm koj lub site tseem ceeb, mus rau Affiliates Settings, thiab ntawm tab Integrations, xaiv Ultimate Multisite. Qhov no yuav qhia rau [AffiliateWP](https://affiliatewp.com/) kom pib taug qab kev mus saib thiab kev hloov pauv los ntawm Ultimate Multisite txoj kev sau npe.

<!-- Duab screenshot tsis muaj: AffiliateWP Integrations settings tab nrog Ultimate Multisite checkbox -->

_Xaiv lub thawv Ultimate Multisite nyob hauv qab thiab save_

Qhov ntawd yeej yog txhua yam uas yuav tsum ua xwb. Kev mus saib uas los ntawm cov referral links yuav raug taug qab, thiab thaum tus user ntawd tom qab them nws thawj zaug, kev hloov pauv yuav raug tsim thiab ib qho nyiaj commission yuav raug muab cais cia rau tus affiliate uas yog tus ua rau muaj kev hloov pauv ntawd.

Tiamsis tsuas yog thawj qhov kev them nyiaj uas tau txais los ntawm tus user tshiab uas tsim los xwb thiaj yuav raug taug qab. Yog xav qhib kev them referral rov ua dua, koj yuav tsum muaj [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## Qhib Recurring Referrals {#enabling-recurring-referrals}

Tom qab teeb tsa [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/), mus rau Affiliates Settings Recurring Referrals tab, thiab qhib kev siv recurring referrals.

<!-- Duab screenshot tsis muaj: AffiliateWP Recurring Referrals settings nrog Enable Recurring Referrals toggle -->

_Yooj yim npaum li no xwb: Qhib Recurring Referrals ces txhua yam npaj txhij lawm!_

Ib zaug ntxiv, qhov ntawd yog txhua yam uas yuav tsum ua xwb!

Tam sim no, txhua zaus muaj ib qho kev them nyiaj tshiab tau txais rau ntawm ib qho account uas raug xa los, tus affiliate uas yog tus lav yuav tau txais nws feem nyiaj ncaj qha!
