# ຍ້ག་ເປັນຫຍັງ Ultimate Multisite

ຖ້າເຈົ້າໄດ້ມາຮອດບົດຄວາມນີ້, ເຈົ້າອາດຈະໄດ້ຍິນວ່າ plugin ຂອງພວກເຮົາອາດເປັນຄຳຕອບທີ່ຈະເຮັດໃຫ້ແນວຄິດທຸລະກິດຂອງເຈົ້າເປັນຈິງ. ພວກເຮົາຫວັງວ່າເຈົ້າຈະໄດ້ຄຳຕອບທີ່ຕ້ອງການໃນສິ້ນສຸດຂອງບົດຄວາມນີ້.

## Ultimate Multisite ແມ່ນຫຍັງ?

Ultimate Multisite ແມ່ນ WordPress Multisite plugin ຫນຶ່ງ ທີ່ເຮັດໃຫ້ເຈົ້າສາມາດສະເໜີບໍລິການ WaaS (Website as a Service) ແກ່ລູກຄ້າໄດ້. ດ້ວຍ plugin ຂອງພວກເຮົາ, ເຈົ້າສາມາດມີ subscription tiers ຕ່າງໆ ແລະ ໃຫ້ລູກຄ້າຈ່າຍຄ່າທຳນຽມເປັນປະຈຳເພື່ອໃຫ້ມີເວັບໄຊຢູ່ເທິງ network Multisite ຂອງເຈົ້າ.

Ultimate Multisite ຊ່ວຍໃຫ້ເຈົ້າສາມາດຮັບການຈ່າຍເງິນຜ່ານ gateway ທີ່ແຕກຕ່າງກັນ (ເຊັ່ນ: WooCommerce ຫຼື Stripe), ສ້າງ site templates ເພື່ອເຮັດໃຫ້ຊີວິດຂອງເຈົ້າສະດວກຂຶ້ນເມື່ອມີຜູ້ໃຊ້ໃໝ່ລົງທະບຽນໃນ network ຂອງເຈົ້າ, map domains, ແລະ ເຮັດ form ລົງທະບຽນໃຫ້ເປັນສ່ວນຕົວເພື່ອສ້າງປະສົບການທີ່ປັບແຕ່ງມາສຳລັບລູກຄ້າຂອງເຈົ້າ.

ແລະ ນີ້ພຽງແຕ່ເລີ່ມຕົ້ນເທົ່ານັ້ນ ເພາະເຈົ້າສາມາດຂະຫຍາຍຟັງຊັນຂອງມັນດ້ວຍ add-ons ແລະ webhooks ໄດ້ອີກ.

## ຄຸນສົມບັດອື່ນໆ

Ultimate Multisite ຍັງມີ:

  * ການເຊື່ອມຕໍ່ Domain Mapping ກັບ Managed WordPress hosting providers: Ultimate Multisite ຈະເພີ່ມ domain mapping ອັດຕະໂນມັດເຂົ້າໃນ admin panels ຂອງ WP Engine, Cloudways, cPanel, RunCloud, Closte ແລະ ອື່ນໆ ເມື່ອມີ domain ໃໝ່ຖືກ map, ເພື່ອບໍ່ໃຫ້ເຈົ້າຕ້ອງເຮັດດ້ວຍຕົນເອງ. ໃນບາງກໍລະນີ, ພວກເຮົາແມ່ນຈະສັ່ງໃຫ້ມັນດຶງ SSL certificate ມາໃຫ້ເຈົ້າເລີຍ.

  * Webhooks: Ultimate Multisite ມີລະບົບ Webhook ທີ່ຊົງພະລັງ ເຊິ່ງອະນຸຍາດໃຫ້ເຈົ້າສາມາດເຊື່ອມຕໍ່ network ຂອງເຈົ້າກັບ service ຕ່າງໆໄດ້ຫຼາຍຢ່າງ. ພວກເຮົາແມ່ນຈະສະເໜີການເຊື່ອມຕໍ່ໂດຍກົງກັບ Zapier, ເຊິ່ງເຮັດໃຫ້ network ຂອງເຈົ້າສາມາດເຂົ້າເຖິງ app ແລະ service ຫຼາຍກວ່າ 1000+ ອັນທີ່ມີຢູ່ໃນ platform ຂອງເຂົາເຈົ້າ.

* ຫນ້າ Admin ທີ່ຊົງພະລັງ ເພື່ອຊ່ວຍໃຫ້ລູກຄ້າຂອງທ່ານໃຊ້ page builder ທີ່ມັກ: Ultimate Multisite ມີ add-on ທີ່ສະດວກ ຊື່ວ່າ Admin Pages Creator, ມັນເຮັດໃຫ້ເຈົ້າສາມາດເພີ່ມ Admin Pages ທີ່ປັບແຕ່ງໄດ້ຢ່າງເຕັມຮູບແບບເຂົ້າໃນ dashboard ຂອງລູກຄ້າໂດຍໃຊ້ template ຕ່າງໆ ເຊັ່ນ Elementor, Beaver Builder, ແລະ Brizy.

* ສອງ Admin Theme ແຕກຕ່າງກັນເພື່ອປັບແຕ່ງປະສົບການ dashboard ຂອງລູກຄ້າ: Check Material WP ແລະ PRO.

ແລະ ສິ່ງທີ່ສັບສົນກວ່າ:

* ການສະໜັບສະໜູນ Template Overrides: ກັບ Ultimate Multisite, ເຈົ້າສາມາດປ່ຽນ template HTML ມາດຕະຖານສຳລັບ flow ການລົງທະບຽນ (signup flow) ດ້ວຍຂອງເຈົ້າເອງໄດ້ໂດຍບໍ່ຕ້ອງກັງວົນວ່າຈະຕ້ອງແກ້ໄຂ file ຫຼັກຂອງ plugin.

* Search and Replace API: ເຈົ້າສາມາດເພີ່ມ annotation ໃນ site templates ດ້ວຍ placeholders ເຊັ່ນ {{site_name}} ແລະ ໃຊ້ Search and Replace API ເພື່ອ auto-populate placeholders เหล่านั้นດ້ວຍຂໍ້ມູນທີ່ລູກຄ້າໃຫ້ມາ ຫຼັງຈາກລົງທະບຽນ, ອັດຕະໂນມັດ.

## ກໍລະນີການນຳໃຊ້ (Use cases)

ດຽວນີ້ແມ່ນເວລາທີ່ຈະເຫັນ use cases ທີ່ເປັນໄປໄດ້ຂອງ Ultimate Multisite.

### ກໍລະນີທີ 1: ອົງກອນການບໍລິການ (An Agency)

ໂດຍທົ່ວໄປແລ້ວ, ທັກສະຫຼັກຂອງອົງກອນການບໍລິການແມ່ນຢູ່ໃນການອອກແບບ website ທີ່ມີດ້ານຕ່າງໆ ເຊັ່ນ ການໂຮດ hosting ຫຼື ການຕະຫຼາດທີ່ຖືກລາຍຊື່ເປັນບໍລິການເພີ່ມເຕີມ.

ສຳລັບອົງກອນການບໍລິການ, Ultimate Multisite ນຳສະເໜີຂໍ້ສະເໜີທີ່ມີຄຸນຄ່າຢ່າງແທ້ຈິງໃນຄວາມສາມາດຂອງມັນທີ່ຈະໂຮດ ແລະ ຈັດການ website ຫຼາຍໆ ເຊິ່ງ platform ດຽວ. ອັນນີ້ຍັງເປັນໄປໄດ້ຫຼາຍກວ່າສຳລັບອົງກອນທີ່ເຮັດໃຫ້ design ຂອງເຂົາເຈົ້າເປັນມາດຕະຖານໃນ theme ໃດໜຶ່ງ ເຊັ່ນ GeneratePress, Astra, OceanWP, ຫຼື ອື່ນໆ ສາມາດໃຊ້ຄວາມສາມາດຂອງ Ultimate Multisite ເພື່ອເປີດໃຊ້ theme เหล่าນີ້ໂດຍອັດຕະໂນມັດສຳລັບ website ໃໝ່ແຕ່ລະໂຕ.

ການຄວບຄຸມທີ່ເຂັ້ມງວດກ່ຽວກັບ plugin ແລະ themes ຈະຖືກຮັກສາໃນລະດັບ product ແຕ່ລະໂຕ ຜ່ານການຕັ້ງຄ່າຂອງ Ultimate Multisite ເຊິ່ງເຮັດໃຫ້ສາມາດເຮັດໃຫ້ plugin ແລະ themes ມີໃຫ້ໃຊ້ ຫຼື ຊ່ອນໄວ້ໄດ້ ລວມທັງສະຖານະການ activate ເມື່ອສ້າງ website ໃໝ່.

Agencies ཚོ་ Ultimate Multisite ལུ་ཁ་གསལ་བའི་སྐྱིད་པའི་རེ་བ་འདི་ཐོབ་ཐရයි. ཁོང་ཚོས་ཁོང་ཚོའི་ལས་ཀྱི་ལེགས་ཤོམ་ - website མཛེས། བྱེད་ཆོག་པ་ཚུགས་ནི་འགོ་བཙུགས་ཐབས་འདི་དང་འགྲོ་ཡོད་པའི་སྐོར་ལུ་.

### Case 2: Niche Provider (ལས་ཀ་ཁྱིམ་ཅིག་)

"གཅིག་གི་ལས་ཀ་བྱས་པ་དང་, དེ་ལེགས་པར་བྱེད་པ་" ཟེར་བ་古い བརྡ་ཞིག་ཡོད་པ་. སློབ་གྲྭ་མང་པོ་ཚུ་ལྟར་, དོན་ཚན་གཅིག་ཁར་ product (ຜະລິດຕະພັນ) ཡང་ན་ service (ལས་ཀ་) འདི་བསྐྲུན་པ་འདི་འདྲ་བྱེད་དགོས་པ་ཨིན།

ཁྱོད་ཀྱིས་ website ཚུ་ གཅིག་ཁར་ focus བྱེད་བའི་ network ཞིག་བསྐྲུགས་པའི་ལས་ཀ་བྱེད་མི་ཚོ་ཡིན་ན་, common framework ( പൊതുဝ frame work) དང་ platform (plattform) འདི་བརྟེན་ནི་ service ག་ར་བསྤྲོད་པ་འདི་ལུ་ help ཆེ་ཐོབ་. އެདྲའི་སྐབས་ལུ་ site templates ཚུ་ help ཆེ་ཐོབ་.

Ultimate Multisite གིས་ innovative (新) feature ཞིག་ཡོད་པ་, template site འདི་. template site ဆိုတာ theme (theme) ຕິດຕັ້ງབྱས་པ་དང་ activate (activate) བྱས་པ་, လိုအပ်တဲ့ plugins ཚུ་ ຕິດຕັ້ງབྱས་པ་དང་ activate བྱས་པ་, ຕົວຢ່າງ posts (post) ཚུ་ཡང་བཟོ་ထားတာ ཚུ་ inclusion (ඇතුළත්) ཡོད་པའི་ site ཅིག་ཨིན། Customer གིས་ template བརྟེན་ནི་ new site ཅིག་བསྐྲུན་တဲ့အခါ, template ལས་བཏོན་པའི་ content (내용) དང་ settings (သတ်မှတ်ချက်) ཚུ་ new site ཅིག་ལུ་ copy (မိတ္တူ) བྱིན་ཐொཕ་.

Niche sites དང་ service ཚུ་ ပေးတဲ့ provider ཚོ་ལུ་, website ཅིག་ ချက်ချင်း ready (အသင့်တော်) ဖြစ်အောင် custom plugins နဲ့ design တွေနဲ့ ချက်ချင်းབཟོ་နိုင်တဲ့ အားသာချက်ကြီးတစ်ခုကို ယူရတယ်။ Customer གིས་ service ပြီးမြောက်ဖို့အတွက် input (ထည့်သွင်းမှု) နည်းနည်းပဲ ပေးရတော့တယ်။

### Case 3: WordPress Web Hosting (WordPress Website Hosting)

WordPress sites တွေ host လုပ်ဖို့ နည်းလမ်းပေါင်းများစွာ ရှိပေမယ့်, customer ལུ་ WordPress version တစ်ခုကို pre-installed (ကြိုတင်တပ်ဆင်ထားပြီး) web space (website နေရာ) ပေးတာလောက် လွယ်ကူတဲ့ နည်းလမ်း မရှိပါဘူး။ ဘာလို့လဲဆိုတော့ meaningful service (အဓိပ္ပာယ်ရှိတဲ့ ဝန်ဆောင်မှု) တစ်ခုပေးဖို့အတွက် decision (ဆုံးဖြတ်ချက်) နဲ့ consideration (စဉ်းစားစရာတွေ) အများကြီးကို ပေါင်းစပ်ဖို့ လိုအပ်လို့ပါ။

Ultimate Multisite က WordPress sites တွေကို host လုပ်ဖို့ comprehensive turnkey solution (အပြည့်အစုံ ready ဖြစ်တဲ့ ဖြေရှင်းနည်း) တစ်ခု ပေးခြင်းအားဖြင့် ဒီနေရာမှာ ထူးချွန်ပါတယ်။ ဒီsolution ထဲမှာ subscription services (လစဉ်ကြေးပေးမှု ဝန်ဆောင်မှု), payment collection (ငွေကောက်ခံမှု), checkout forms (ငွေပေးချေမှုပုံစံများ), discount vouchers (လျှော့စျေး ကူပွန်များ) နဲ့ customer communications (Customer နဲ့ ဆက်သွယ်ရေး) အတွက် လိုအပ်တဲ့ core mechanisms (အဓိက ယန္တရားများ) တွေ ပါဝင်ပါတယ်။
