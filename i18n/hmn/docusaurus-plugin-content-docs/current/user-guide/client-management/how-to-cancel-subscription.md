# ང་གིས་ unsubscribe (subscription) ག་རེ་བསྒྲུབ་ཡོད།

ཁྱེད་ཀྱིས་ Ultimate Multisite license ཡོད་ན་, ཁྱེད་རང་གི་ renewal (ໃໝ່བྱེད་རྒྱུ་) ག་ཚད་ ཞུས་ནས་ བཅོལ་ཆོག་པའི་སྐབས་ ག་ཚད་ བཅོལ་ཆོག་པ་ལྟར་ བཅོལ་ཆོག་ཐུབ་တယ်။ འདི་ལྟར་ ခြေလှမ်းརིམ་ཁ་གཉིས་ཀྱི་རིམ་པ་བསྟན་ན།

ཁྱེད་ཀྱིས་ ဝယ်ယူསྐབས་ བསྐྲུན་པའི་ email དང་ password (密码) ག་རེ་བེད་སྤྱོད་བྱེད་དགོས་པ་དེ་བེད་སྤྱོད་བྱེད་ပါ။

འདི་ནི་ཁྱེད་ཀྱི་ Account Page འདིའི་སྐབས་ཀྱི་ initial screen (བཞག་སྟངས་) ཡིན།

ཁྱེད་ཀྱི་ subscription བཅོལ་ཆོག་པའི་ཆེད་དུ་, མགོ་བོའི་ལམ་རྒྱུན་ཐོག་ནས་ (menu on the left), _Renewing & Billing_ (ໃໝ່བྱེད་པ་དང་ བརྩི་འགྲོ་) ལ་འགྲོ་ပါ။

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

ညာဘက်དུ་ཡོད་པའི་ arrow (མཚན་ indicative) ལ་བཀྱང་ནས་ side window (ཕྱོགས་གཅིག་ཏུ་བསྒྲུབ་བྱེད་པའི་បង្អួច) འགོ་བཙུགས་ပါ။ དེ་ནས་, **auto-renew** (རང་མ་གྲུབ་སྐྱོང་བྱེད་རྒྱུའི་) option (ເລືອກབྱེད་ཐབས་) ལ་བཀོལ་ཆོག་པ་དེ་བསྒྲུབ་བྱེད་དགོས་པ་ལེགས་པོ་ཡོད།

System (ལག་རྟགས་) ནས་ confirmation message (ຢືນຢັນབྱེད་པའི་བརྡ་) ཞིག་ སྟོན་རྒྱུ་ཡིན།

ཁྱེད་ཀྱིས་ subscription བཅོལ་ཆོག་པ་དེ་ངེས་པར་དུ་བྱེད་དགོས་པ་ཡིན་ན་, _**Cancel Renewals**_ (ໃໝ່བྱེད་པ་བཅོལ་) button (བཀྲལ་) ལ་བཀྲལ་ပါ။ འདི་བྱས་ནས་, quick survey (མཁོ་སྤྲོད་ཆུང་ཙམ་) ཞིག་ ལན་འདྲི་རྒྱུ་ཡོང་།

Done! ཁྱེད་ཀྱི་ subscription རང་མ་གྲུབ་སྐྱོང་བྱེད་པར་བཅོལ་ཆོག་པ་རེད།

**ཁྱེད་ཀྱི་ key (code) ནི་ subscription ໝົດ காலཚད་ བཞིན་ valid (ལེགས་པའི་) 状態 (status) འདྲ་རྒྱུ་ཡིན།** ཁྱེད་ཀྱིས་ subscription ག་རེ་བསྐྱར་བྱེད་ཆོག་པ་ཡོད་ན་, manual (ལས་ཀྱི་ཐོག་ནས་) བྱས་དགོས་པ་དེ་བསྟེན་དགོས་པོ་ཡོད།
