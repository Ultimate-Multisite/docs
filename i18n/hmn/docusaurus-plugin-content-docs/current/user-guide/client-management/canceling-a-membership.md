# бһҖбһ¶бһҡбһӣбһ»бһ”бһҹбһҳбһ¶бһҮбһ·бһҖбһ—бһ¶бһ– (v2)

_**бһ…бҹҶбһҺбһ¶бҹҶбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ– бһўбһҸбҹ’бһҗбһ”бһ‘бһ“бҹҒбҹҮбһҹбҹҶбһҠбҹ…бһӣбһҫ Ultimate Multisite version 2.xбҹ”**_

бһҖбһ¶бһҡбһӣбһ»бһ”бһҹбһҳбһ¶бһҮбһ·бһҖбһ—бһ¶бһ–бһ“бҹ…бһҒбһ¶бһ„бһўбһҸбһ·бһҗбһ·бһҮбһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһӮбһәбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҺбһ¶бһҹбҹӢбҹ”

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ…бһ„бҹӢбһӣбһ»бһ”бһҹбһҳбһ¶бһҮбһ·бһҖбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒ бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹӮбһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ‘бҹҶбһ–бҹҗбһҡ ****Account**** бһҮбһ¶бһҳбһ»бһ“бһҹбһ·бһ“бҹ”

бһ“бҹ…бһ‘бһёбһ“бҹ„бҹҮ бһ–бһҪбһҖбһӮбҹҒбһҸбҹ’бһҡбһјбһңбһ…бһ»бһ…бһ‘бҹ… **Danger Zone** бһ бһҫбһҷбһ…бһ»бһ…бһӣбһҫ **Delete Account**бҹ”

![Account page Danger Zone with Delete Account button](/img/account-page/danger-zone.png)

бһҳбҹүбһјбһҢбһ»бһӣбһҳбһҪбһҷбһ“бһ№бһ„бһ”бһҫбһҖбһЎбһҫбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһ»бҹҶбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ–бһёбһ–бһҪбһҖбһӮбҹҒбҹ” бһ–бһҪбһҖбһӮбҹҒбһӮбһҪбһҡбһҸбҹӮбһ”бһ·бһ‘бһҮбһҳбҹ’бһҡбһҫбһҹ **Confirm Account Deletion** бһ бһҫбһҷбһ…бһ»бһ…бһӣбһҫ **Delete Account**бҹ”

![Confirm Account Deletion dialog](/img/account-page/delete-account-confirm.png)

бһҡбһҝбһ„бһ“бҹҒбҹҮбһ“бһ№бһ„бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһ–бһҪбһҖбһӮбҹҒбһ…бҹҒбһүбһ–бһёбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ“бһ·бһ„бһӣбһ»бһ”бһҹбһҳбһ¶бһҮбһ·бһҖбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҺбҹ’бһҸбһ¶бһүбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҖбҹҸбһҠбһјбһ…бһҮбһ¶бһӣбһ»бһ”бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒбһ…бҹҒбһүбһ–бһёбһ”бһҺбҹ’бһҸбһ¶бһүбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹӮбһҡбҹ”
