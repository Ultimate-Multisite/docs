# ໃສ່ໂຄສະນາໃນເຄືອຂ່າຍຂອງທ່ານດ້ວຍ Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector ແມ່ນ **add-on ຟຣີ** ທີ່ໃຫ້ເຄື່ອງມືແກ້ໄຂໂຄສະນາໃນແຜນການຕ່າງໆໄດ້ໂດຍກົງຈາກໜ້າແກ້ໄຂຂອງມັນ.

ຫຼັງຈາກຕິດຕັ້ງ Ultimate Multisite: Ad Injector ແລ້ວ, ໄປທີ່ໜ້າ _Plugins_ ຂອງທ່ານເພື່ອເປີດໃຊ້ງານມັນ.

<!-- ຮູບພາບບໍ່ມີໃຫ້ເບິ່ງ: ໜ້າ WP Plugins ສະແດງປຸ່ມ Network Activate ສຳລັບ add-on Ad Injector -->

_ກົດປຸ່ມ Network Activate_

ເມື່ອ add-on ເປີດໃຊ້ງານແລ້ວ, ບ່ອນ _Advanced Options_ ໃນໜ້າແກ້ໄຂແຜນການ (plan edit screen) ຈະມີຕົວເລືອກໃໝ່ຊື່ວ່າ _Ad Settings_.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_ດຽວນີ້ເຈົ້າສາມາດແກ້ໄຂການຕັ້ງຄ່າໂຄສະນາສຳລັບແຕ່ລະແຜນຂອງເຈົ້າໄດ້ເລີຍ_

ໃນ tab ນັ້ນ, ເຈົ້າຈະສາມາດເພີ່ມ ad codes ສຳລັບ posts ແລະ pages ໃນ front-end ໄດ້, ທັງຢູ່ກ່ອນແລະຫຼັງເນື້ອຫາ.

<!-- ຮູບພາບບໍ່ມີໃຫ້ເບິ່ງ: ຕົວຢ່າງການສະແດງໂຄສະນາໃນໜ້າ post ສຳລັບແຜນໃດໜຶ່ງ -->

_ຫຼັງຈາກທີ່ເຈົ້າໄດ້ໃສ່ code ໂຄສະນາຂອງເຈົ້າແລ້ວ, ມັນຈະສະແດງຢູ່ໃນ posts ແລະ pages ຂອງແຜນນັ້ນ._

ເຈົ້າສາມາດມີໂຄສະນາຢູ່ໃນ header ຂອງ Admin Pages ສຳລັບຜູ້ໃຊ້ຂອງແຜນນັ້ນໄດ້ອີກດ້ວຍ.

<!-- ຮູບພາບບໍ່ມີໃຫ້ເບິ່ງ: ຕົວເລືອກ Ad Code ໃນ Admin Panel ຂອງແຜນການຕັ້ງຄ່າ -->

<!-- ຮູບພາບບໍ່ມີໃຫ້ເບິ່ງ: ຕົວຢ່າງການສະແດງໂຄສະນາໃນ user admin panel -->

_ໄປທີ່ຕົວເລືອກ Admin Panel Ad Code ຖ້າເຈົ້າຕ້ອງການສະແດງໂຄສະນາໃນ back-end ຂອງຜູ້ໃຊ້._

### ເກີນກວ່າການໂຄສະນາ (Beyond Advertising) {#beyond-advertising}

ເຈົ້າສາມາດໃຊ້ Ultimate Multisite: Ad Injector ເພື່ອສະໜອງການຊ່ວຍເຫຼືອໃຫ້ກັບແຜນທີ່ມີລະດັບສູງໄດ້ອີກ.

ພຽງແຕ່ໄປທີ່ຕົວເລືອກ _Admin Panel Ads_ ແລະ ເພີ່ມ HTML code ສຳລັບ support tool ຂອງເຈົ້າໃສ່.
