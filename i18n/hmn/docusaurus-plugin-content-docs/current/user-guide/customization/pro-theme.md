# ປັບແຕ່ງ Admin Panel ຂອງທ່ານດ້ວຍ PRO Theme

ເມື່ອເຮົາເຮັດວຽກໃຫ້ລູກຄ້າເລື້ອຍໆ, ເຮົາກໍມັກຈະໄດ້ຮັບຄວາມຕ້ອງການທີ່ຈະສະໜອງປະສົບການທີ່ເປັນແບບສະເພາະໃນ dashboard ຂອງ WordPress ໃຫ້ລູກຄ້າ. ຖ້າເຈົ້າກໍາລັງຊອກຫາຮູບລັກສະນະທີ່ເບິ່ງເປັນມືອາຊີບກວ່າ, PRO Theme ອາດຈະຊ່ວຍເຈົ້າໄດ້.

PRO Theme ແມ່ນ theme ສຳລັບ admin interface ທີ່ສ້າງຂຶ້ນມາເພື່ອຊ່ວຍໃຫ້ເຈົ້າໃນການປັບແຕ່ງ Dashboard ຂອງລູກຄ້າ. ແລະ ມັນຍັງເປັນສ່ວນໜຶ່ງຂອງ core plugin ພ्रीमियम ຂອງ Ultimate Multisite ນໍາອີກ.

ຫຼັງຈາກຕິດຕັ້ງແລ້ວ, ເຈົ້າໄປທີ່ໜ້າ _Plugins_ ຂອງເຈົ້າ ແລ້ວຄລິກເພື່ອ activate ມັນໃນ Network ຂອງເຈົ້າ.

_**ໝາຍເຫດ:**_ _ຖ້າເຈົ້າພຽງແພ້ຕິດຕັ້ງມັນ, ມັນຈະສະແດງເປັນຕົວເລືອກໃຫ້ລູກຄ້າຂອງເຈົ້າ ເພື່ອຕິກໃສ່ເພື່ອນຳໃຊ້ theme ນັ້ນກັບຜູ້ໃຊ້ນັ້ນເທົ່ານັ້ນ._

ດຽວນີ້, interface admin ຂອງເຈົ້າ ກໍຖືກອອກແບບໃໝ່ທັງໝົດແລ້ວ! ເຈົ້າຈະມີເມນູໃໝ່ຢູ່ຄໍຊ້າຍທີ່ເອີ້ນວ່າ PRO Theme.
