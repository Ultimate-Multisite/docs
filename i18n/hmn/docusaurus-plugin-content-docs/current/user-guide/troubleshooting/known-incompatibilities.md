# ຄວາມບໍ່ເຂົ້າກັນທີ່ຮູ້ຈັກກັບ Plugin ອື່ນໆ

ຫຼັງຈາກຕິດຕັ້ງ ແລະ ເປີດໃຊ້ງານ Ultimate Multisite ແລ້ວ, ເຈົ້າອາດຈະ gặp ปัญหาເນື່ອງຈາກມັນບໍ່ເຂົ້າກັນກັບ plugin ອື່ນໆທີ່ຍັງບໍ່ໄດ້ແກ້ໄຂ.

ພວກເຮົາກຳລັງເຮັດວຽກຢ່າງຕັ້ງໜ້າເພື່ອຫຼຸດລົງຂອງລາຍການນີ້ໃນທຸກໆການປ່ອຍໃໝ່, ແຕ່ຖ້າ plugin ໃດຢູ່ໃນລາຍການນີ້, ເຈົ້າອາດຈະຕ້ອງປິດມັນ (deactivate) ເພື່ອໃຫ້ Ultimate Multisite Network ເຮັດວຽກໄດ້ເຕັມທີ່.

ນີ້ແມ່ນລາຍການຂອງພວກມັນ ແລະ ສິ່ງທີ່ເຈົ້າຄວນເຮັດໃນກໍລະນີເຫຼົ່ານີ້:

**Plugin WPMU DEV ບາງຊະນິດ** Plugin ບາງໂຕທີ່ _ບໍ່ໄດ້ຮັບການຮັກສາ_ ໂດຍ WPMU DEV ແມ່ນຮູ້ຈັກວ່າເຮັດໃຫ້ຂັ້ນຕອນການລົງທະບຽນ Ultimate Multisite ຢຸດຢູ່ສະຖານທີ່ສຸດ (ຜູ້ໃຊ້ບໍ່ສາມາດສ້າງບັນຊີໄດ້). ນີ້ແມ່ນລາຍການຂອງ plugin ທີ່ເຮັດໃຫ້ເກີດບັນຫາເຫຼົ່ານັ້ນ:

  * Signup Language: _Ultimate Multisite ມີ add-on ທີ່ໃຫ້ຟັງຊັນດຽວກັນ, ຊື່ວ່າ Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _ສິ່ງນີ້ຖືກສ້າງເຂົ້າມາໃນ Ultimate Multisite ແລ້ວ, ສະນັ້ນບໍ່ຈຳເປັນຕ້ອງມີ plugin ເພີ່ມເຕີມ._

  * New Blog Templates: _ສິ່ງນີ້ຖືກສ້າງເຂົ້າມາໃນ Ultimate Multisite ແລ້ວ, ສະນັ້ນບໍ່ຈຳເປັນຕ້ອງມີ plugin ເພີ່ມເຕີມ._

  * Set Password on Multisite Blog Creation: _ສິ່ງນີ້ຖືກສ້າງເຂົ້າມາໃນ Ultimate Multisite ແລ້ວ, ສະນັ້ນບໍ່ຈຳເປັນຕ້ອງມີ plugin ເພີ່ມເຕີມ._

  * Signup TOS: _ສິ່ງນີ້ຖືກສ້າງເຂົ້າມາໃນ Ultimate Multisite ແລ້ວ, ສະນັ້ນບໍ່ຈຳເປັນຕ້ອງມີ plugin ເພີ່ມເຕີມ._

  * Anti-Splog: ເຖິງແມ່ນວ່າຈະບໍ່ມີພະລັງຫຼາຍເທົ່າກັນ, ສິ່ງນີ້ກໍ _ຖືກສ້າງເຂົ້າມາໃນ Ultimate Multisite ແລ້ວ._

**ໝາຍເຫດ**: Plugin WPMU Dev ອື່ນໆທັງໝົດສາມາດໃຊ້ໄດ້ປົກກະຕິພ້ອມກັບ Ultimate Multisite ເຊັ່ນ: _Smush_ , _Forminator_ , _Defender,_ ແລະ ອື່ນໆ.
