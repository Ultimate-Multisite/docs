# ຖາມເລື້ອຍໆ (Frequently Asked Questions) {#frequently-asked-questions}

ນີ້ແມ່ນຄຳຖາມທີ່ມັກຈະຖືກຖາມເວລາຂໍຄວາມຊ່ວຍເຫຼືອຕ່າງໆ.

  * **Ultimate Multisite ແມ່ນຫຍັງ?**

Ultimate Multisite ແມ່ນ plugin ຂອງ WordPress, ເຊິ່ງສ້າງມາເພື່ອຕິດຕັ້ງ WordPress multisite, ມັນຈະປ່ຽນການຕິດຕັ້ງ WordPress ຂອງທ່ານໃຫ້ເປັນເຄືອຂ່າຍເວັບໄຊທີ່ພຣີມຽມ – ຄືກັບ [WordPress.com](https://WordPress.com) – ເຮັດໃຫ້ລູກຄ້າສາມາດສ້າງເວັບໄຊໄດ້ຜ່ານຄ່າທຳນຽບປະຈຳເດືອນ, ປະຈຳໄຕມາດ, ຫຼື ປະຈຳປີ (ທ່ານຍັງສາມາດສ້າງ Free plans ໄດ້ອີກ).

  * **Ultimate Multisite ຈະເຮັດວຽກໂດຍບໍ່ຕ້ອງມີ Multisite ໄດ້ບໍ່?**

ບໍ່ໄດ້. Ultimate Multisite ຮຽກຮ້ອງໃຫ້ມີ WordPress multisite, subdirectory, ຫຼື subdomain.

  * **ຂໍ້ກຳນົດສຳລັບ Ultimate Multisite ແມ່ນຫຍັງ?**

Ultimate Multisite ຮຽກຮ້ອງໃຫ້ມີ:

**WordPress Version** : v5.3+ (ແນະນຳ: ໃຊ້ເວີຊັນທີ່ໝັ້ນຄົງສຸດທ້າຍ)
**PHP Version** : 7.4.x (ຈະມີການສະໜັບສະໜູນ > 8.0 ໃນໄວໆນີ້)
**MySQL Version** : v5+ (ແນະນຳ: ເວີຊັນ 5.6, ໃນກໍລະນີທີ່ທ່ານບໍ່ມີ 8.0 ຢູ່ໃນ hosting provider ຂອງທ່ານ)

_Multisite (subdomain ຫຼື subdirectory) ກໍຕ້ອງໄດ້ຖືກເປີດໃຊ້ງານດ້ວຍ._ [ວິທີຕິດຕັ້ງ WordPress Multisite](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **Ultimate Multisite ສາມາດເຮັດວຽກກັບ Multisite ທີ່ມີ Subdirectories ໄດ້ບໍ່?**

ໄດ້. Ultimate Multisite ຮອງຮັບທັງການຕິດຕັ້ງແບບ subdirectory ແລະ subdirectory.

  * **Ultimate Multisite ສາມາດເຮັດວຽກກັບ Multisite ທີ່ມີ Subdomains ໄດ້ບໍ່?**

ໄດ້. Ultimate Multisite ຮອງຮັບທັງການຕິດຕັ້ງແບບ subdirectory ແລະ subdomain.

  * **ຂ້ອຍສາມາດໃຊ້ Ultimate Multisite ເພື່ອຈຳກັດວ່າຜູ້ໃຊ້ສາມາດໃຊ້ Plugin ແລະ Themes ໃດໄດ້ແດ່ໃນເວັບໄຊຂອງເຂົາເຈົ້າໄດ້ບໍ່?**

ໄດ້. ດ້ວຍ Ultimate Multisite, ທ່ານສາມາດເລືອກໄດ້ວ່າ plugin ແລະ themes ໃດທີ່ມີໃຫ້ແຕ່ລະ plan ໃນເຄືອຂ່າຍຂອງທ່ານ.

  * **ຂ້ອຍສາມາດໃຊ້ Ultimate Multisite ເພື່ອຈຳກັດຈຳນວນ post ທີ່ຜູ້ໃຊ້ສາມາດສ້າງໃນເວັບໄຊຂອງເຂົາເຈົ້າໄດ້ບໍ່?**

ඔව්. Ultimate Multisite එකෙන් ඔයාගේ ජාලයේ තියෙන **සෑම post type එකකටම** post quotas (post ප්‍රමාණය සීමා කිරීම) හදන්න පුළුවන්. පරිපාලකයන් ආරාධනා කරන්න පුළුවන් පරිශීලක ගණන, **ดิสก์ ස්පේස් ප්‍රමාණය**, සහ තවත් බොහෝ දේවල් සීමා කරන්නත් පුළුවන්.

* **Ultimate Multisite එක Site Templates (සයිට් සැකිලි) වලට සහය දක්වනවාද?**

ඔව්. Ultimate Multisite එකෙන් ඔයාට ඕන තරම් Site Templates හදාගන්න පුළුවන්. Site Templates කියන්නේ themes (තේමා) නෙවෙයි, ඒක ඔයාගේ ජාලයේ තියෙන සාමාන්‍ය වෙබ් අඩවි (sites) වගේ. ඔයාට plugins, themes activate කරන්නත්, ඒ Site Template එක ඕන විදිහකට customize කරන්නත් පුළුවන්. යම් පරිශීලකයෙක් ඒ template එක තෝරුවොත්, ඒ template එකේ හැම වෙනස්කමක්ම සහිතව අලුත් copy එකක් විදිහට එයාගේ site එක හදනවා.

ඔයාට plan එකකට විශේෂ template එකක් set කරන්නත් පුළුවන්. ඒක කළාම, ඒ plan එකට ලියාපදිංචි වන හැම පරිශීලකයෙකුටම ඒ Site Template එකේ duplicate එකක් විදිහට site එකක් හදනවා.

* **Ultimate Multisite WooCommerce එක්ක integrate වෙනවද?**

ඔව්!

ඔයාගේ ජාලයේ users ලාට WooCommerce තියෙන්න ඕන නම්, ඔවුන්ට stores setup කරන්න ඉඩ දීමට, Ultimate Multisite භාවිතා කරලා ඒක කරන්න පුළුවන්. ඒ වගේම ඔයාට හැම plan tier එකකටම client ලා හදන්න පුළුවන් product ගණන සීමා කරන්නත් පුළුවන්.

Ultimate Multisite සඳහා subscription payments සම්බන්ධයෙන්, ඔයාට [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) භාවිතා කරලා Ultimate Multisite client ලාට ඔයාගේ ප්‍රධාන site එකේ තියෙන WooCommerce gateways පාවිච්චි කරලා ගෙවන්න ඉඩ දෙන්න පුළුවන්. ඒ විදිහට, WooCommerce සඳහා දියුණු කළ Payment Gateways වල විශාල library එකකට ඔයාට සම්බන්ධ වෙලා, ඔයාගේ Ultimate Multisite client ලාට බිල් කරන්න ඒවා පාවිච්චි කරන්න පුළුවන්.

* **WPMUDEV Pro Sites වලින් subscription migrate කරන්න ක්‍රමයක් තියෙනවද?**

Yes, Ultimate Multisite ལྷག་པར་དུ་གནས་ཚུལ་འདི་བཀྲལ་བ་ཡོད་པ་རེད། [your Pro Sites network-སྤྲོད་རྒྱུ་སྐྱིད་ཆེད་ཀྱི་ឧបករណ៍](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) ཞིག་ཡོད།

  * **"Allowed Plugins" tab ངའི་ Plan Edit Page-ལས་ག་རེ་མི་འགྲོ་བ་လဲ?**

Ultimate Multisite ནི་ཁྱེད་ཀྱི་ network-ལ་ activate (བསྐྱར་ཆུ་) མ་བྱས་པའི་ plugin တွေကို ཡིན། ཡིན་ན་ཡང་，plugin ཞིག་ network-ལ་activate བྱས་པ་ཡིན་ན་，དེའི་རང་བཞིན་ལྟར་ network-ལ་ཡོད་པའི་ site ཚོ་ཐམས་ཅད་ཀྱི་ default-འོག་ནས་ load (བསྐྱར་ཆུ་) བྱེད་ཀྱི་ཡོད།

"Allowed Plugins" tab-ལས་plugin ག་རེ་མི་ལོན་པ་མི་བལྟན་ဘူးဆိုရင်，ཁྱེད་ཀྱིས་དགའ་བའི་ plugin དེ་ network-ལ་active (བསྐྱར་ཆུ་) བྱས་ཡོད་པ་མ་རེད། ཡང་ན་མ་བྱས་པའི་སྐོར་ལྟ་རྒྱུར་བལྟ་དགོས་ཡོད།

  * **ངའི་ Pricing Table design རང་ཉིད་ཀྱིས་བེད་སྤྱོད་ཆོག་လား?**

ཆོག។ version 1.4.0 ནས་བཟུང་，Ultimate Multisite-a template overrides (template-སོ་སོར་བཅོས་) တွေကို support (རྒྱབ་སྐྱོར་) བྱེད་ཀྱི་ཡོད།

  * **Ultimate Multisite-a WP Engine ກັບལེན་ཆོག་လား?**

ཆོག។ Ultimate Multisite-a WP Engine-ལས་རྒྱབ་སྐྱོར་བྱས་ཡོད་པ་རེད། ཡིན་ནའང་，domain mapping (domain-སོ་སོར་བསྒྱུར་བཅོས་) ལས་ལས་လုပ်ဖို့အတွက် admin-a manual extra steps (手動གཞི་རྩའི་ག extras) ག་རེ་བྱེད་དགོས་པ་མ་རེད། **version 1.6.0 ནས་བཟུང་འདྲ་མིན་ཡོད་པ་མ་རེད།** Ultimate Multisite-a WP Engine API ກັບ တိုက်ရိုက် integrate (ເຊື່ອມKết) བྱེད་ཀྱི་ཡོད། mapping བྱས་པ་ལས་domain ཚོ་ panel-ལ་အလိုအလျောက် ထည့်ပေးလိမ့်မယ်။

  * **ཁྱེད་རང་ག་རེ་བཀའ་གནང་လဲ?**

Ultimate Multisite-a 任何 hosting provider (host-སྤྲོད་མཁན་) ກັບལེན་ཆོག་པ་ཡིན། གཅིག་པུ་དོན་དུ་ Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane နဲ့ [Runcloud.io](http://Runcloud.io) ལྟ་བུའི་ hosting company ཚོ་ domain mapping capabilities (domain-སོ་སོར་བསྒྱུར་བཅོས་ཀྱི་စွမ်းဆောင်ရည်) དེ་ལ་ tighter integration (གནས་ཚད་ཆེན་པོ་) ཡོད་པ་མ་རེད། ཁོ་ཚོ་က out of the box (မူလལས་) འགྲོ་ཐུབ། ཡང་ན་ other hostings ཚོ་ལ་ domain mapping ကို manual-སྒྲིག་လုပ်ဖို့ လိုအပ်နိုင်တယ်။
