# ལག་བསികལམའི་དཔྱད་རིམ་ (Touring the Admin Panel)

Ultimate Multisite ньང་ཚོ་གཞན་ལ་ཤེས་ཐུབ་པ་དང་འགྲོ་བ་སླ་བར་བྱེད་ཀྱི་ཡོད་པ་རེད། ཡིན་ནའང་,  මෙལོང་ཆེ་တဲ့ ឧបករណ៍ཅིག་ཡག་པོ་ཡོད་པས་ཁྱེད་རང་ལ་ག导བསྐྱོད་ (guided tour) བྱེད་ཐུབ་တယ်။  administrate page ཚོ་ལ་ལམ་བར་སྟོན་བྱེད་རྒྱུ་ནི་ཁྱེད་རང་ལ་འགྲོ་བའི་སྐབས་དེ་ལ help བྱེད་ཆོག་པ་རེད།

## Dashboard (ལས་ཀ་གཙོ་བོའི་ལག་བསྟན་) {#dashboard}

Ultimate Multisite-ta **Dashboard** нь ဝင်ငွေ, website ལས་བྱེད་སྒོ་ཚང་མས་, membership འཕེལ་རྒྱས་, visitor count, geographical data ཚུལ་མཐོང་རྒྱུ་ཡོད་པའི་ basic reports ཚོ་བལྟ་སྐྱོད་ཆེ་བ་ཞིག་ཡོད།

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

Dashboard འདི་གི་ முழு view (full view) འདིན།:

![Ultimate Multisite Dashboard full page](/img/admin/um-dashboard-full.png)

და dashboard འདི་གི་အောက်ပိုင်းအပိုင်း (bottom section):

![Ultimate Multisite Dashboard bottom section](/img/admin/um-dashboard-bottom.png)

Network dashboard འདི་ཡung view ཚང་མས་ལྟ་སྐྱོད་ཆེ་བ་ཞིག་ཡོད།:

![Network Dashboard full page](/img/admin/network-dashboard-full-page.png)

## Checkout Forms (ငွေပေးချေမှုပုံစံများ) {#checkout-forms}

**Checkout Forms** ནི་ཁྱེད་རང་ལ་ custom registration pages ཚོ་བཟོ་နိုင်တဲ့ flexibility ཞིག་ပေးတယ်။ fields တွေကို customize လုပ်လို့ရတယ်၊ custom classes နဲ့ scripts တွေ ထည့်လို့ရတယ်၊ ပြီးတော့ visitor གི་རྒྱལ་ཁབ་ (country) သို့မဟုတ် location ལ་အခြေခံပြီး access တွေကို ကန့်သတ်လို့ရတယ်။

![Checkout Forms list](/img/admin/checkout-forms-list.png)

## Products (ကုန်ပစ္စည်းများ) {#products}

**Products** section ནི་ཁྱེད་རང་တို့ network အတွက် plan, package, နဲ့ services တွေ ဘယ်လို ထုတ်မလဲဆိုတာ သတ်မှတ်တဲ့နေရာပဲ။ product level ལ་ဈေးနှုန်းတွေ၊ billing frequency တွေနဲ့ limits & quotas တွေကို သတ်မှတ်နိုင်တယ်။

![Products list](/img/admin/products-list.png)

## Memberships (အဖွဲ့ဝင်များ) {#memberships}

**Memberships** page ནི་ཁྱེད་རང་တို့ network ထဲက subscription ཚོ་အားလုံးကို ပြသပေးတယ်။ plans, products, billing amounts နဲ့ frequencies, payment history, timestamp တွေ အပါအဝင် subscription details တွေကို review လုပ်ပြီး edit လုပ်လို့ရတယ်။

![Memberships list](/img/admin/memberships-list.png)

## Payments (ငွေပေးချေမှုများ) {#payments}

ទំព័រ **Payments** ផ្តល់នូវការមើលលឿននៃប្រវត្តិការបង់ប្រាក់សម្រាប់បណ្តាញទាំងមូលរបស់អ្នក ដោយមានព័ត៌មានប្រតិបត្តិការលម្អិត រួមទាំងផលិតផល និងចំនួនទឹកប្រាក់ជាក់លាក់។

![Payments list](/img/admin/payments-list.png)

## Customers {#customers}

ទំព័រ **Customers** បង្ហាញសមាជិកទាំងអស់ដែលបានចុះឈ្មោះនៅក្នុងបណ្តាញរបស់អ្នក រួមជាមួយព័ត៌មានអ្នកប្រើប្រាស់មូលដ្ឋាន រួមមាន ឈ្មោះ, ប្រវត្តិរូបអ៊ីម៉ែល, ការចូលប្រើចុងក្រោយ, និងតំណ "switch to" ដើម្បីចូលទៅកាន់ subsites របស់ពួកគេ។

![Customers list](/img/admin/customers-list.png)

## Sites {#sites}

ទំព័រ **Sites** រាយបញ្ជី subsites ទាំងអស់នៅក្នុងបណ្តាញរបស់អ្នក រួមទាំង site templates និង subsites ដែលជាកម្មសិទ្ធិរបស់អតិថិជន។ អ្នកអាចគ្រប់គ្រង mapped domains, limits & quotas, plugins និង themes, និង site thumbnails បានយ៉ាងងាយស្រួល។

![Sites list](/img/admin/sites-list.png)

## Domains {#domains}

ទំព័រ **Domains** ផ្តោតលើ custom domains ដែលត្រូវបានភ្ជាប់ទៅនឹង subsites។ ជា super admin អ្នកអាចបន្ថែម ឬ map custom domain names ដោយដៃ និងមើល DNS records និង logs បាន។

![Domains list](/img/admin/domains-list.png)

## Discount Codes {#discount-codes}

**Discount Codes** អនុញ្ញាតឱ្យអ្នកបង្កើត coupon codes ដើម្បីផ្តល់ការបញ្ចុះតម្លៃដល់ end-users របស់អ្នក។ កំណត់តម្លៃនៃការបញ្ចុះតម្លៃ និងកំណត់គោលដៅទៅកាន់អ្នកប្រើប្រាស់ ឬក្រុមជាក់លាក់ណាមួយ។

![Discount Codes list](/img/admin/discount-codes-list.png)

## Broadcasts {#broadcasts}

**Broadcasts** គឺជាឧបករណ៍សម្រាប់ផ្សព្វផ្សាយការជូនដំណឹង ឬផ្ញើសារឯកជនទៅកាន់អ្នកប្រើប្រាស់ subsites របស់អ្នក។ កំណត់ក្រុមអ្នកប្រើប្រាស់ជាក់លាក់ដោយផ្អែកលើផែនការ (plan) ឬផលិតផលដែលពួកគេបានចុះឈ្មោះ។

![Broadcasts list](/img/admin/broadcasts-list.png)

## Settings {#settings}

ទំព័រ **Settings** គឺជាកន្លែងដែលអ្នកកំណត់រចនាសម្ព័ន្ធ Ultimate Multisite — ការកំណត់នៃការចុះឈ្មោះ, payments, API និង webhooks, domain mapping, និងការរួមបញ្ចូលផ្សេងៗទៀត។

ການຕັ້ງຄ່າ AI connector ຍັງສະໜັບສະໜູນ OAuth provider pools ທີ່ຮອງຮັບໃນປັດຈຸບັນ: Anthropic Max, OpenAI ChatGPT/Codex, ແລະ Google AI Pro. ແຕ່ລະ card ຂອງ provider ຈະເຮັດໃຫ້ super admins ສາມາດເຊື່ອມຕໍ່ account, ເຮັດ refresh account ທີ່ບັນທຶກໄວ້, ລຶບ account ໂດຍ email, ແລະ ໃຊ້ manual OAuth fallback ເມື່ອ environment sandboxed ຕັດການ redirect browser. ຕົວເລືອກ setup ຂອງ Cursor Pro ໄດ້ຖືກເອົາອອກຈາກ admin panel ແລ້ວ.

ChatGPT/Codex accounts ຮອງຮັບການໃຊ້ tool use ທີ່ອີງໃສ່ connector, ເຊິ່ງເຮັດໃຫ້ operation ສາມາດໃຊ້ tools ໄດ້, ສະນັ້ນ workflow ຂອງ admin ທີ່ເພິ່ງພາການ operation ທີ່ຮອງຮັບໂດຍ connector ສາມາດໃຊ້ Codex tool behavior ໄດ້ຫຼັງຈາກເຊື່ອມຕໍ່ account OpenAI ແລ້ວ.

## Events {#events}

ໜ້າ **Events** ຈະເກັບບັນທຶກ events ແລະ logs ທັງໝົດໃນ network ຂອງເຈົ້າ. ມັນຕິດຕາມກິດຈະກໍາຕ່າງໆ ເຊັ່ນ: ການປ່ຽນແຜນ, ການລົງທະບຽນ, ແລະ ການເຄື່ອນໄຫວອື່ນໆ — ມີປະໂຫຍດສຳລັບການຕິດຕາມ multisite ຂອງເຈົ້າ.

![Events list](/img/admin/events-list.png)

## Webhooks {#webhooks}

**Webhooks** ອະນຸຍໃຫ້ເຈົ້າສົ່ງ data ໄປຫາ application ອື່ນ. ມີປະໂຫຍດສຳລັບການສົ່ງ data ຈາກ Ultimate Multisite ໄປຫາ platform ເຊັ່ນ Zapier.

![Webhooks list](/img/admin/webhooks-list.png)
