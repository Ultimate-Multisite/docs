# མཐུན་རིམ་དཔྱད་བརྟག་ (Dashboard Overview)

Ultimate Multisite ལུ་ **Ultimate Multisite** ເມནུ་འོག་ལས་ **Dashboard** གཅིག་ཁར་ཡོད་པ་མ་ཚད་ Network Dashboard ལ་ widget ཚོ་ཡང་བཞག་ཆོག. དཔྱད་རྟག་འདི་ནང་ ཁྱེད་ཀྱིས་གཉིས་ཀ་ལས་ ဘာတွေເຫັນཚུགས་པའི་སྐོར་བཤད་ཡོད་པ་ཨིན།

## Network Dashboard (Network དཔྱད་རྟག་)

Ultimate Multisite ནང་མཐུན་རིམ་ level དཔྱད་རྟག་ (My Sites → Network Admin → Dashboard) ལུ་ ཕན་ཐོགས་ཆེ་བའི་ widgets ཚོ་འཇུག་པ་ཨིན།

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

Widget ཚུ་གཅིག་གིས་ ဘာတွေ བརྗོད་ཡོད་လဲ:

**First Steps (གནམ་སྟོན་གྱི་གནས་ཚུལ་):** 新 installation (ལམ་འདི་གཅིག་ཁར་བསྐྱར་བཞག་པ་) ཚུ་ལུ་ ປະກົດཐོབ་ནང་, Payment method  configurate (ལག་རྩལ་འཇུག་པ་) དང་ test account (དཔྱད་རྟག་ account) བཟོ་ཚུགས་པ་ལས་ བཏོན་མི་ཚུ་ བསྟན་ཡོད་པ་ཨིན།

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Summary (សរុប):** ថ្ងៃགཅིག་ནང་ signups (མི་ཚོ་ចូលབསྐྱོད་པའི་အရေအတွက်) དང་ revenue (ប្រាក់ဝင်) တွေကို བརྗོད་ཡོད་པ་ཨིན། MRR (Monthly Recurring Revenue) ནི་ customer ཚུ་ལས་ ມາཔའི་ membership တွေက total projected amount (សរុប བརྩིས་པའི་ចំនួន) ག་དྲン འོང་སྲིད་ ག་དྲン འོང་སྲིད་ කියලා བརྗོད་ཡོད།

![Summary widget](/img/admin/network-dashboard-summary.png)

**Activity Stream (פעលགནས་ཚུལ་):** Ultimate Multisite ནང་ 起こတဲ့ events ཚུ་ལས་ བརྗོད་ཡོད་པ་ཨིན། signups, cancellations (បោះបង់), site creation (網站បង្កើត), deletions (削除), MenuBar ផ្សេងៗ ទៀត។

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**Right Now (ឥឡូវ):** Network ནང་ ཡོད་པའི་ users (អ្នកប្រើប្រាស់) និង sites (website) ཚུ་ ខ្លីៗ សរុប བརྗོད་ཡོད་པ་མ་ཚད་ new sites ឬ users ཚོ་ បង្កើតဖို့ shortcut (ផ្លូវលឿន) တွေ ཡང་មាន។

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**News & Discussions (ព័ត៌មាន និង ពិភាក្សា):** Ultimate Multisite ནང་ ថ្មីបំផុត news တွေကို ទាញយកပြီး បង្ហាញပေးတယ် — updates, critical bugs (កំហុសធ្ងន់ធ្ងរ), and security fixes (ការជួសជុលសុវត្ថិភាព) ដូចជា។

![News widget](/img/admin/network-dashboard-news.png)

## The Ultimate Multisite Dashboard (Ultimate Multisite དཔྱད་རྟག་)

Network dashboard ནི་ network ទាំងមូលនៃ information တွေကို បង្ហាញပေးပေမယ့်, Ultimate Multisite dashboard (Ultimate Multisite menu ទីតាំងខាងលើ) គឺ service របស់វាསྐོར་ ព័ត៌មានတွေကို បង្ហាញပေးပါတယ်။

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Monthly Recurring Revenue Growth:** ມັນສະແດງລາຍລະອຽດລາຍເດືອນຂອງລາຍຮັບໃນໄລຍະເວລາປະຈຳປີ 12 ເດືອນ. ລາຍຮັບໃໝ່ຈະສະແດງເປັນສີຟ້າ ແລະ ການຍົກເລີກຈະສະແດງເປັນສີແດງ.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**ການຮອງຮັບຊ່ວງວັນທີ:** ກັ່ນຕອງຂໍ້ມູນຂອງ dashboard ຕາມໄລຍະເວລາເພື່ອເນັ້ນຂໍ້ມູນທີ່ເຈົ້າຕ້ອງການ. ຊ່ວງວັນທີນີ້ຈະຄວບຄຸມວ່າຈະສະແດງຫຍັງຢູ່ໃນ widgets ຂ້າງລຸ່ມ.

![Date range and dashboard stats](/img/admin/um-dashboard-date-range.png)

Widgets ຂອງ dashboard ລວມມີ:

- **Revenue (ລາຍຮັບ)** — ລາຍຮັບທັງໝົດໃນໄລຍະເວລານັ້ນພ້ອມກັບ MRR ແລະ ການແບ່ງລາຍລະອຽດຕາມຜະລິດຕະພັນ
- **New Memberships (ການສະໝັກສະມາຊິກໃໝ່)** — ມີການສ້າງສະມາຊິກໃໝ່ຈັກຄົນໃນໄລຍະເວລາທີ່ເລືອກ
- **Signups by Countries (ການສະໝັກໂດຍປະເທດ)** — ການແບ່ງຂໍ້ມູນທາງພູມສາດຂອງການສະໝັກ
- **Signups by Forms (ການສະໝັກຕາມແບບຟອມ)** — ມີລູກຄ້າຈັກຄົນທີ່ສະໝັກຜ່ານແບບຟອມ checkout ແຕ່ລະແບບ ເທຽບກັບຜູ້ທີ່ເພີ່ມໂດຍ admin
- **Most Visited Sites (ໄຊທ໌ທີ່ຖືກເຂົ້າເຖິງຫຼາຍທີ່ສຸດ)** — ໄຊທ໌ທີ່ຖືກເຂົ້າເຖິງຫຼາຍທີ່ສຸດໃນເຄືອຂ່າຍຂອງເຈົ້າ (ຕ້ອງເປີດໃຊ້ການນັບການເຂົ້າເຖິງຢູ່ທີ່ **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**)
