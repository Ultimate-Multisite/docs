# ការគ្រប់គ្រងអតិថិជន (Managing Customers) {#managing-customers}

ទំព័រ **Customers** នឹងបង្ហាញបញ្ជីសមាជិកទាំងអស់ដែលបានចុះឈ្មោះនៅក្នុងបណ្តាញរបស់អ្នក។ ទំព័រនេះបង្ហាញព័ត៌មានមូលដ្ឋានរបស់អ្នកប្រើប្រាស់ រួមមាន ឈ្មោះ, ប្រវត្តិរូបអ៊ីម៉ែល, ការចូលប្រើប្រាស់ចុងក្រោយ, និងតំណ **switch to** ដែលអនុញ្ញាតឱ្យអ្នកចូលទៅកាន់ subsites របស់អតិថិជននោះក្នុងនាមជា site administrator។

![Customers list page](/img/admin/customers-list.png)

ចុចលើ **Manage** នៅក្រោមប័ណ្ណអតិថិជនណាមួយ ដើម្បីមើលព័ត៌មានលម្អិតអំពីអតិថិជននោះ។

នៅលើទំព័រព័ត៌មានលម្អិតរបស់អតិថិជន ប្លុក **At a Glance** នឹងបង្ហាញ Customer ID, ពេលវេលាចូលប្រើប្រាស់ចុងក្រោយ, និងចំនួនទឹកប្រាក់សរុបដែលអតិថិជននេះបានចំណាយ។

ប្លុក **Customer Options** អនុញ្ញាតឱ្យអ្នកកំណត់អតិថិជននេះជា **VIP** (គឺជាស្លាកមួយដើម្បីធ្វើឱ្យពួកគេងាយស្រួលរក)។ ផ្ទាំង **Billing Info** បង្ហាញព័ត៌មានគិតថ្លៃដូចជា អ៊ីម៉ែល, អាសយដ្ឋាន, និងប្រទេស។

ប្លុក **Fields** បង្ហាញឈ្មោះអ្នកប្រើប្រាស់អតិថិជន, អ៊ីម៉ែល, ហើយអនុញ្ញាតឱ្យអ្នកផ្លាស់ប្តូរស្ថានភាព **Email Verification**។

ខាងក្រោមប្លុក Customer Options អ្នកនឹងឃើញប្លុកបន្ថែមចំនួនបួន៖

- **Memberships** — ការជាសមាជិកបច្ចុប្បន្នដែលអតិថិជនបានចុះឈ្មោះ
- **Payments** — ប្រវត្តិការទូទាត់របស់អតិថិជន
- **Sites** — គេហទំព័រដែលអតិថិជននេះជាម្ចាស់
- **Events** — សកម្មភាពដែលទាក់ទងនឹងអតិថិជននេះ (ដូចជា email verification, ការផ្លាស់ប្តូរការគិតថ្លៃ ជាដើម)

ចុចលើរូបតំណាងព្រួញ (→) នៅជាប់នឹងធាតុណាមួយដើម្បីចូលទៅកាន់ទំព័រព័ត៌មានលម្អិតរបស់វា។ ឧទាហរណ៍ ចុចលើព្រួញនៅជាប់នឹងគេហទំព័រមួយ នឹងនាំអ្នកទៅទំព័រแก้ไขគេហទំព័រ ដែលអ្នកអាចកែប្រែការភ្ជាប់សមាជិកភាព, ប្រភេទគេហទំព័រ, និងស្ថានភាពសកម្ម។

ដូចគ្នានេះដែរ វាអនុវត្តចំពោះ memberships និង payments — នីមួយៗតំណភ្ជាប់ទៅកាន់ទំព័រព័ត៌មានលម្អិតដែលអ្នកអាចកែសម្រួលស្ថានភាពសមាជិកភាព, អតិថិជនដែលពាក់ព័ន្ធ, ផលិតផល, ការកំណត់ payment gateway, និងសូម្បីតែដំណើរការការ Refund។

នៅខាងស្តាំនៃទំព័រអតិថិជន អ្នកនឹងឃើញ៖

- **Last Login & IPs** — མུ་མཐུད་བསྐྱོད་ අවසන් වෙတဲ့ ཚེས་, IP address, དང་རྒྱལ་ཁབ་
- **Timestamps** — Customer account འདི་སྐྲ་བྱུང་ và མཇུག་བསྒྱུར་ཡོང་た ཚེས་
- **Delete** — Customer རང་ཉིད་ཀྱིས་གནོད་སྐྱོན་བྱེད་ཐབས་ཀྱི་ཡོད།
