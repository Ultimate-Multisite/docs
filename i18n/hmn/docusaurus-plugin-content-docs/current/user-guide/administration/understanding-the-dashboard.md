# ຄວາມເຂົ້າໃຈກ່ຽວກັບ Ultimate Multisite Dashboard

ກ່ອນที่เราຈະເຈາະເລິກເຂົ້າໄປໃນຕົວເລືອກການຕັ້ງຄ່າທີ່ສູງຂຶ້ນ ແລະ ແນວທາງການເຮັດວຽກຂອງ Ultimate Multisite, ເຊິ່ງມາເບິ່ງ dashboard ກ່ອນ.

ຄືກັບ plugin WordPress ຫຼາຍຢ່າງ, Ultimate Multisite ສ້າງເມນູລະດັບ network (ເຄືອຂ່າຍ) ຊື່ວ່າ **Ultimate Multisite** ພ້ອມກັບເມນູຮອງຢູ່ໃຕ້.

## Widgets ໃນ Network Dashboard {#network-dashboard-widgets}

Ultimate Multisite ຈະເພີ່ມ widgets ທີ່ເປັນປະໂຫຍດສຳລັບ dashboard ລະດັບ network. Dashboard ນີ້ສາມາດຊອກເຫັນໄດ້ໂດຍການໄປທີ່ **My Sites → Network Admin → Dashboard**. Widgets ເຫຼົ່ານີ້ຖືກອອກແບບມາເພື່ອໃຫ້ເຂົ້າເຖິງຂໍ້ມູນ ແລະ ການເຮັດວຽກທົ່ວໄປໄດ້ງ່າຍສຳລັບຜູ້ບໍລິຫານເຄືອຂ່າຍ.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

### ຂັ້ນຕອນທໍາອິດ (First Steps) {#first-steps}

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

Widget ນີ້ຈະປາກົດຂຶ້ນໃນການຕິດຕັ້ງ Ultimate Multisite ໃໝ່ ແລະ ເປັນການເຕືອນຜູ້ບໍລິຫານເຄືອຂ່າຍໃຫ້ເຮັດວຽກເພີ່ມເຕີມ ເຊັ່ນ: ການຕັ້ງຄ່າວິທີການຈ່າຍເງິນ ແລະ ການສ້າງບັນຊີທົດສອບ.

### ສະຫຼຸບ (Summary) {#summary}

![Summary widget](/img/admin/network-dashboard-summary.png)

Widget Summary ຈະລາຍງານຈຳນວນການສະໝັກໃຊ້ ແລະ ລາຍຮັບຂອງມັນໃນມື້ນັ້ນ. MRR (Monthly Recurring Revenue) ຈະຊີ້ໃຫ້ເຫັນຍອດລວມທີ່ຄາດຫວັງທົ່ວລູກຄ້າທີ່ມີການເປັນສະມາຊິກທີ່ສ້າງລາຍໄດ້.

### ສະຖິຕິກິດຈະກຳ (Activity Stream) {#activity-stream}

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

Widget ນີ້ຈະລາຍງານເຫດການທັງໝົດໃນ Ultimate Multisite. ເຫດການຕ່າງໆປະກອບມີ ການສະໝັກໃຊ້, ການຍົກເລີກ, ການສ້າງໄຊທ໌, ການລຶບ, ແລະ ກິດຈະກຳອື່ນໆ.

ກະລຸນາເບິ່ງສ່ວນ Events ໃນ documentation ເພື່ອຂໍ້ຄວາມທີ່ລະອຽດກວ່າ.

### ຕອນນີ້ (Right Now) {#right-now}

![Right Now widget](/img/admin/network-dashboard-right-now.png)

Widget no hmo no tsoom zoo li tsoom qhia txog cov neeg thiab cov site hauv network. Cov qhia tseem ceeb ntawm lub siab uas thiaj tsim cov site lossis neeg tshiab los yog tsim cov neeg tshiab.

### News and Discussions (Txhawm Hauv Kev Qhia thiab Thov) {#news-and-discussions}

![News and Discussions widget](/img/admin/network-dashboard-news.png)

Cov qhia no yuav rooj thiab tso cov news tshiab ntawm Ultimate Multisite. Tsis txhob tso cov qhia no los xyuas rau kev hlu lom, cov bugs tseem ceeb, thiab cov kev tsim kev pabcuam txog am ninh.

## Ultimate Multisite Dashboard (Lub Siab Ua Ultimate Multisite) {#ultimate-multisite-dashboard}

Thaum lub dashboard ntawm network tso cov ntaub ntawv zoo li qhov uas muaj hauv network, lub dashboard ntawm Ultimate Multisite (uau los ntawm lub menu ntawm Ultimate Multisite) tso cov ntaub ntawv txog kev pabcuam.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

### Monthly Recurring Revenue Growth (Kev Tsim Kev Txhim Kho ntawm Cov Tsis Txhob Tso Kev Txhim Kho Hmo) {#monthly-recurring-revenue-growth}

Cov chart MRR tso cov ntaub ntawv txog kev tsim kho hmo hauv lub sijhawm neeg 12 hnub.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

Cov kev tsim kho tshiab yuav tso cov ntsuam lam, thiab cov kev txhob tso yuav tso cov ntsuam tsawv.

### Date-range Support (Kev Thia Lub Sijhawm Neeg) {#date-range-support}

Thaum muaj neeg uas tsim hauv cov neeg thiab cov txhais tau, cov ntaub ntawv yuav zoo li kho siab heev. Cov kev thia lub sijhawm neeg uas muaj hauv dashboard cho yuav tso cov ntaub ntawv thiab lub sijhawm neeg uas xav txhua administrator ntawm network tsim kom tsuas tso cov ntaub ntawv thiab lub sijhawm neeg uas xav txhua.

![Date range selector and stats](/img/admin/um-dashboard-date-range.png)

Lub sijhawm neeg yuam kawm lub sijhawm uas cov qhia hauv cov widget uas thiaj los ntawm nws, loj hmo, kev tsim kho hmo, kev tsim neeg tshiab, kev tsim neeg tshiab los ntawm txoj rhoum, thiab cov site uas muaj tshaj tawm tshaj.

![Dashboard statistics widgets](/img/admin/um-dashboard-stats.png)

:::tip
Ua txhua yam los ua kom widget **Most Visited Sites** zoo, hais rau visit counting hauv **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**.
:::
