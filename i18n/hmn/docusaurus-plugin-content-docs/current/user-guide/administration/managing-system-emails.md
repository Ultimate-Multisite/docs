# ការគ្រប់គ្រងអ៊ីមែលប្រព័ន្ធ (Managing System Emails)

Ultimate Multisite នឹងផ្ញើអ៊ីមែលជាច្រើនសម្រាប់ព្រឹត្តិការណ៍ផ្សេងៗ ដូចជាការបញ្ជាក់ការចុះឈ្មោះ, សំបុត្រទទួលលុយ, និងការផ្លាស់ប្តូរភាពជាសមាជិក។ អ្នកអាចកែសម្រួល قالب (template) អ៊ីមែលទាំងនេះឱ្យត្រូវនឹងម៉ាករបស់អ្នកបាន។

## ការមើលអ៊ីមែលប្រព័ន្ធ (Viewing System Emails) {#viewing-system-emails}

ចូលទៅកាន់ **Ultimate Multisite → Settings** ហើយចុចលើแท็บ **Emails** ដើម្បីមើលការកំណត់ដែលទាក់ទងនឹងអ៊ីមែល។

![Settings - Emails tab](/img/admin/settings-emails.png)

ពីទីនោះ អ្នកអាចចូលទៅកាន់ទំព័រ **System Emails** ដើម្បីមើល قالبអ៊ីមែលទាំងអស់។

![System Emails list](/img/admin/system-emails-list.png)

អ្នកក៏អាចកែសម្រួល قالبអ៊ីមែលនីមួយៗបានដោយប្រើ system email editor៖

![System email editor](/img/admin/system-email-editor.png)

## ការនាំចូល قالبអ៊ីមែលថ្មី (Importing New Email Templates) {#importing-new-email-templates}

កំណែថ្មីនៃ Ultimate Multisite និង add-ons អាចនឹងបង្កើត قالبអ៊ីមែលថ្មីៗ។ ដើម្បីការពារការប៉ះទង្គិចគ្នា អ្នកមិនអាចបន្ថែម قالبថ្មីដោយស្វ័យប្រវត្តិបានទេ — អ្នកត្រូវនាំចូលវាដោយដៃ។

ដើម្បីនាំចូល قالبអ៊ីមែលថ្មី៖

1. ទៅកាន់ទំព័រ **System Emails**
2. ចុចលើប៊ូតុង **Reset & Import** នៅខាងលើទំព័រ
3. បើកជម្រើស **Import Emails** ដើម្បីមើល قالبដែលមាន។
4. ជ្រើសរើស قالبដែលអ្នកចង់នាំចូល ហើយបញ្ជាក់ការជ្រើសរើស។

## ការកំណត់ឡើងវិញនូវ قالبអ៊ីមែល (Resetting Email Templates) {#resetting-email-templates}

ប្រសិនបើអ្នកបានកែសម្រួល قالبអ៊ីមែលហើយ ហើយចង់ត្រឡប់ទៅខ្លឹមសារដើមវិញ អ្នកមានជម្រើសពីរ៖

1. **លុបហើយនាំចូលឡើងវិញ** — លុប system email ចេញ ហើយនាំចូលវាម្ដងទៀត (វានឹងលុបស្ថិតិការផ្ញើ)
2. **កំណត់ឡើងវិញតាមរយៈឧបករណ៍** — ប្រើឧបករណ៍ **Reset & Import**, បើកជម្រើស **Reset**, ហើយជ្រើសរើសអ៊ីមែលដែលអ្នកចង់កំណត់ឡើងវិញ។

ជម្រើស Reset គឺល្អជាងព្រោះវារក្សាស្ថិតិការផ្ញើ និងទិន្នន័យផ្សេងទៀតទុក ខណៈពេលដែលបានស្តារខ្លឹមសារដើមឡើងវិញ។
