# ការគ្រប់គ្រងតំណដែលអាចចែករំលែកបានសម្រាប់ផែនការ (Plans)

Ultimate Multisite ផ្តល់ឱ្យអ្នកនូវជម្រើសមួយដើម្បីចែករំលែកតំណផែនការដោយផ្ទាល់ទៅកាន់អតិថិជនរបស់អ្នក។ ដើម្បីធ្វើបែបនេះ សូមចូលទៅកាន់ទំព័រស្វែងរក (edit pages) របស់ផែនការណាមួយរបស់អ្នក ហើយចុចលើប៊ូតុង _Click to copy the Shareable Link_។

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_នៅលើទំព័រស្វែងរកផែនការ (Plan's Edit Page) អ្នកអាចរកឃើញប៊ូតុងសម្រាប់បង្កើតតំណដែលអាចចែករំលែកបាន។_

សូមចំណាំថា យន្តការនេះប្រើប្រាស់ភាពញឹកញាប់នៃការគិតថ្លៃ (billing frequency) តាមលំនាំដើមដើម្បីបង្កើតតំណ។ មានន័យថា ប្រសិនបើការគិតថ្លៃលំនាំដើមរបស់អ្នកគឺប្រចាំខែ (monthly) ឧទាហរណ៍ នៅទីបំផុតអ្នកនឹងមានតំណដូចនេះ៖ [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

ប្រសិនបើអ្នកចង់ឱ្យតំណដែលអាចចែករំលែកបានកំណត់ជាប្រចាំត្រីមាស (quarterly) អ្នកត្រូវបន្ថែមលេខ 3 នៅខាងចុង៖

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

ដើម្បីជ្រើសរើសផែនការ _premium_ តាមកាលវិភាគប្រចាំឆ្នាំ (yearly schedule) អ្នកត្រូវបន្ថែមលេខ 1 រួចបន្តដោយពាក្យ៖

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

សូមចងចាំថា អ្នកមិនអាចមានតំណតែមួយសម្រាប់ភាពញឹកញាប់នៃការគិតថ្លៃផ្សេងៗគ្នាបានទេ។ អ្នកគួរតែចែករំលែកតំណជាក់លាក់មួយសម្រាប់ភាពញឹកញាប់នៃការគិតថ្លៃនីមួយៗរបស់អ្នក។

នៅពេលអ្នកចែករំលែកតំណនោះទៅកាន់អតិថិជនរបស់អ្នក វានឹងបញ្ជូនពួកគេទៅជំហានដំបូងនៃដំណើរការចុះឈ្មោះ (signup flow) របស់អ្នក។
