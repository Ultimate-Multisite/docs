# Ultimate Multisite བསྒྱུར་བ། (Ultimate Multisite Translation) {#translating-ultimate-multisite}

Ultimate Multisite нь English-ལས་བརྩམས་ཆོག, ແຕ່ plugin འདི་ཡག་པོ་བསྒྱུར་ཆོག་པ་དང་། ང་ཚོས་བསྒྱུར་བའི་བཅའ་ལ་དངོས་སུ་བཀྲོལ་བ་ཡོད་པ་རེད།

## Core Plugin Translations (Plugin གཙོ་བོའི་བསྒྱུར་བ) {#core-plugin-translations}

Ultimate Multisite plugin གཙོ་བོའི་ཆ་ parte-ལས་ official **WordPress.org translation platform** ལམ་ནས་བསྒྱུར་བྱེད་ཀྱི་ཡོད།

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

འདི་ནི་ WordPress ཚོགས་པའི་ standard translation system རེད། བསྒྱུར་བ་ཚོས་ approval threshold (བཀྲལ་བའི་གོ་སྐབས་) ལྷག་པ་བྱུང་ན་ WordPress plugin updates ལམ་ནས་Ultimate Multisite users ཚུ་ཡོངས་ལ་ automatic-y བསྐྱོད་ཆོག

**ང་ཚོས་ཁྱེད་རང་གི་བརྩོན་འགྲོ་དགོས།** English ལས་གཞན་པའི་སྐད་ཡིག་ཤེས་པ་ཅིག་ཡོད་ན་བསྒྱུར་བ་བཀྲལ་ནས་བསྐྱོད་ཆོག་པའི་སྐོར་ལས་བརྩོན་འགྲོ་བྱེད་ཐབས་དེ་ཚོ་བཀྲལ་ནས་བརྩོན་འགྲོ་བྱེད་ཆོག ཚིག་མང་པོ་བསྒྱུར་བ་ཡིན་ན་ཡང་ཕན་ཐོགས་ཆེ། བཞག་རྒྱུ་སྐབས་:

1. [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/) ལམ་ནས་བརྩོན་འགྲོ་བྱེད་
2. ཁྱེད་ཀྱི་སྐད་ཡིག་བསྐྱར་ជ្រើសរើស
3. "Contribute Translation" (བསྒྱུར་བ་བཀྲལ་) ནང་བཀླག་
4. WordPress.org account ຟຣີ གཅིག་བསྐྲུན་དགོས།

ཁྱེད་རང་གི་སྐད་ཡིག་གི་ translation editor (བསྒྱུར་བ་བཀྲལ་ཐག་བཅད་ཆོག་པའི་) ሆነན་འདོད་ན་ [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) ལམ་ནས་ຕິດຕໍ່བྱེད་ཆོག

## Addon Translations (Addon བསྒྱུར་བ་) {#addon-translations}

Addon plugins တွေကို གཞན་དྲ་ཐོག་ནས་བསྒྱུར་བྱེད་ཀྱི་ཡོད།

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Addon translations ཚུ་ད་ལྟའི་時点ནས་AI-y བསྐྲུན་པ་ཙམ་རེད། འདི་ཚོ་ support བྱེད་ཐུབ་པའི་སྐད་ཡིག་ཚང་མས་བསྒྲུབ་ထားပေမယ့် ཚིག་གི་ပုံစံ (phrasing) འཁྱོངས་པའི་ཆ་ཤས་དང་ accuracy (ត្រឹមត្រូវမှု) តិចតួច ཡོད་ཐོワイ། ང་ཚོས་ human translators ཚོ་བལྟ་སྐྱོང་བྱེད་ပြီး ཚིག་གི་ပုံစံတွေကို ལྷག་པ་དང་ accuracy འདྲ་བ་བྱེད་ཆོག་པའི་建議 (suggestions) ཞུ་རྒྱུར་བཀྲལ་ཡོད་རེད།

## Local Translations (Local བསྒྱུར་བ་) {#local-translations}

ຖ້າເຈົ້າຢາກສ້າງຄຳແປເອງສຳລັບໂຄງການຂອງເຈົ້າ ແທນທີ່ຈະສົ່ງໄປໃຫ້ຄົນອື່ນ, ມີສອງທາງເລືອກ:

### ການໃຊ້ Poedit {#using-poedit}

ເຈົ້າສາມາດໃຊ້ໂປຣແກຣມ desktop ຊື່ **[Poedit](https://poedit.net)** ເພື່ອແປໄຟລ໌ `.pot` ໄດ້:

1. ດາວໂຫຼດ Poedit ຈາກ [poedit.net](https://poedit.net)
2. ເປີດໄຟລ໌ `.pot` ທີ່ຢູ່ໃນ thư mục `wp-content/plugins/ultimate-multisite/lang/`
3. ແປຂໍ້ຄວາມ (strings) ໃນ interface ຂອງ Poedit
4. ເກັບຮັກສາ -- Poedit ຈະສ້າງໄຟລ໌ `.po` ແລະ `.mo` ໃຫ້ເຈົ້າໂດຍອັດຕະໂນມັດ
5. ວາງໄຟລ໌ທີ່ໄດ້ຮັບຜົນມາໄວ້ໃນ folder `lang`

### ການໃຊ້ Loco Translate {#using-loco-translate}

Plugin **[Loco Translate](https://wordpress.org/plugins/loco-translate/)** ຂອງ WordPress ອະນຸຍາດໃຫ້ເຈົ້າແປໂດຍກົງຈາກ admin panel ໂດຍບໍ່ຕ້ອງໃຊ້ເຄື່ອງມື desktop ເພື່ອເຮັດຫຍັງເລີຍ. ມັນຈະຈັດການສ້າງ ແລະ ຄompile (compile) ໄຟລ໌ແປໃຫ້ເຈົ້າເອງ.
