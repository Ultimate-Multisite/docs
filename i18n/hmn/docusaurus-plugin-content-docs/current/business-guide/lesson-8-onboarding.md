# ບົດຮຽນທີ 8: ການຕ້ອນຮັບລູກຄ້າໃໝ່

ການເຮັດໃຫ້ລູກຄ້າສະໝັກເຂົ້າລະບົບໄດ້ແມ່ນພຽງເຄິ່ງໜຶ່ງຂອງວຽກເທົ່ານັ້ນ. ຖ້າເຂົາເຈົ້າເຂົ້າສູ່ລະບົບແລ້ວ, ຮູ້ສຶກວ່າຂໍ້ມູນຫຼາຍເກີນໄປ ແລະ ບໍ່ກັບມາອີກ, ທ່ານໄດ້ສູນເສຍເຂົາເຈົ້າແລ້ວ. ບົດຮຽນນີ້ອອກແບບປະສົບການທີ່ຈະປ່ຽນການສະໝັກໃໝ່ໃຫ້ເປັນລູກຄ້າທີ່ເຄື່ອນໄຫວ ແລະ ມີສ່ວນຮ່ວມ.

## ເຮົາເຮັດມາຮອດໃສ?

FitSite ມີການຈັດຮູບແບບ (branding) ທີ່ພ້ອມໃຊ້ງານແລ້ວໃນຂັ້ນຕອນການຈ່າຍເງິນ. ດຽວນີ້, ພວກເຮົາຈະມາສຸມໃສ່ສິ່ງທີ່ຈະເກີດຂຶ້ນຫຼັງຈາກເຈົ້າຂອງສະຕູຟິດສຳເລັດການສະໝັກ ແລະ ເຂົ້າມາໃນເວັບໄຊທ໌ໃໝ່ຂອງເຂົາເຈົ້າເປັນຄັ້ງທຳອິດ.

## ເປັນຫຍັງການຕ້ອນຮັບ (Onboarding) ຈຶ່ງສຳຄັນ?

30 ນາທີທໍາອິດຫຼັງຈາກສະໝັກແມ່ນກຳນົດວ່າລູກຄ້າຈະຢູ່ຕໍ່ ຫຼື ຈະໜີໄປ. ເຈົ້າຂອງສະຕູຟິດທີ່:

- ເຂົ້າລະບົບແລ້ວ ແລະ ເຫັນເວັບໄຊທ໌ທີ່ເບິ່ງຄືກັບເວັບໄຊຟິດສຳເລັດແລ້ວ → ຢູ່ຕໍ່
- ຮູ້ຢ່າງຊັດເຈນວ່າຕ້ອງເຮັດຫຍັງຕໍ່ໄປ → ຢູ່ຕໍ່
- ຮູ້ສຶກ lạcໃນ dashboard ຂອງ WordPress ທົ່ວໄປ → ອອກໄປ

Template ສ່ວນຕົວຂອງເຈົ້າ (ບົດຮຽນທີ 4) ຈະຈັດການກັບຂໍ້ທຳອິດ. ບົດຮຽນນີ້ຈະຈັດການກັບຂໍ້ທີສອງ.

## ປະສົບການເຂົ້າສູ່ລະບົບຄັ້ງທໍາອິດ

### Dashboard Widget ຕ້ອນຮັບ

ສ້າງ dashboard widget ທີ່ເປັນແບບສະເພາະເພື່ອທັກທາຍລູກຄ້າໃໝ່ ແລະ ຊ່ວຍເຫຼືອເຂົາເຈົ້າໃນການຕັ້ງຄ່າ. ອັນນີ້ຄວນຈະສະແດງອອກຢ່າງໂດດເດັ່ນເມື່ອເຂົາເຈົ້າເຂົ້າສູ່ລະບົບເປັນຄັ້ງທໍາອິດ.

**FitSite Quick Start:**

1. **ເພີ່ມຊື່ສະຕູ ແລະ ໂລໂກ້ຂອງເຈົ້າ** -- ເຊື່ອມຕໍ່ໄປທີ່ Customizer ຫຼື ການຕັ້ງຄ່າ Site Identity
2. **ອັບເດດຕາຕະລາງຮຽນຂອງເຈົ້າ** -- ເຊື່ອມຕໍ່ໂດຍກົງກັບ editor ຂອງໜ້າ Classes
3. **ເພີ່ມຄູຝຶກຂອງເຈົ້າ** -- ເຊື່ອມຕໍ່ໄປທີ່ editor ຂອງໜ້າ Trainers
4. **ຕັ້ງລາຍລະອຽດການຕິດຕໍ່ຂອງເຈົ້າ** -- ເຊື່ອມຕໍ່ໄປທີ່ editor ຂອງໜ້າ Contact
5. **ເບິ່ງຕົວຢ່າງເວັບໄຊທ໌ຂອງເຈົ້າ** -- ເຊື່ອມຕໍ່ໄປທີ່ frontend

### བདེ་འཇགས་པའི་ Dashboard སོ་སོལ་གཏོང་།

ཕྱི་མའི་བཀའ་རྩིས་བྱེད་མཁན་ཚོས་ WordPress menu ནང་གི་ཆ་ཚང་ཚུལ་ཐམས་ཅད་མཐོང་རྒྱུ་གལ་ཆེ་བ་མིན་པར་བལྟ་དགོས་པ་མ་རེད། འདིའི་སྐོར་ལས་བསམ་བློ་བཏང་རོགས།

- ហាត់སྟངས་ཀྱི་སྒྲིག་བཅོས་དང་འབྲེལ་མེད་པའི་ menu item ཚབ་ནས་བཞག་པ་ (ཡོད་ན་ཡང་) (དཔེར་ན། Comment ཚེ་བཀྲལ་མེད་པའི་སྐབས་)
- menu རྣམ་པ་བསྒྱུར་ནས་བཀའ་རྩིས་མང་ཤོས་དང་འབྲེལ་བ་ཆེ་བའི་ item ཚུ་གི་གོ་བོར་བཞག་པ་
- niche (དཔེར་ན། "Your Studio" ལྟར་) མཐོང་རྒྱུ་གི་ label ཚུ་བཟོ་ནས་བཞག་པ་ ("Appearance" ལྟར་མ་རེད།)

[Plugin & Theme Manager](/addons/plugin-and-theme-manager) addon ནི་ customer ཚོས་ག་རེ་མཐོང་རྒྱུའི་སྐོར་ལ་བསྒྲིགས་པར་གནང་ཆོག་པ་ཡོད་པས་ཁྱེད་རང་ལུ་ཕན་ཐོགས་འབྱུང་།

## Welcome Email Sequence (Email ཚོགས་པོ་)

Welcome email དེ་གཅིག་མ་རེད། customer ཚོ་གོང་མའི་བདག་སྐྱིད་སྤྲོད་བྱེད་རྒྱུའི་ཆེད་དུ་ sequence ཞིག་བཟོ་དགོས་པ་ལས་འགལ་བ་མེད་པར་བཀའ་རྩིས་བྱེད་དགོས།

### Email 1: Welcome (Signup བྱས་ནས་ཡང་ཤིན་མི་འདྲ་)

- Subject: "FitSite ལ་ཕེབས་པའི་སྐྱིད་པོ་བཏང་། -- ཁྱེད་ཀྱི་ studio website གྲ་སྒྲིག་བྱས་པར་གྲུབ་པ་རེད།"
- Content: Login link, quick start steps (གནམ་མཐོང་ཚོད་འཇགས་པའི་ཆ་ལག་), help resources ལ་བལྟ་རྒྱུ་ link
- Tone: སྐྱིད་པོ་དང་སྤྲོད། fitness དང་འབྲེལ་བ་ཡོད་པའི་སྐད་ཆ་

### Email 2: Quick Wins (ཉིན་མོ་དང་པོ་)

- Subject: "FitSite ནང་གི་གཙོ་བོ་ 3 ຢ່າງ"
- Content: ཁྱེད་ཀྱི་ logo ထည့်དགོས་པ་, homepage hero image འགོ་བར་བཅོས་དགོས་པ་, class schedule འགོད་དགོས་པ་
- ག་རེ་ click བྱས་ནི་ཡོད་མའི་ screenshots ཚུ་ထည့်དགོས།

### Email 3: Make It Yours (ཉིན་མོ་གསུམ་)

- Subject: "ཁྱེད་ཀྱི་ fitness site ལས་གཞན་འཕགས་པ་བཟོ་"
- Content: colors བཅོས་ནིའི་སྐོར་ལ་བརྟེན་པ།, trainer photos ထည့်དགོས་པ་, studio story བྲིས་དགོས་པ་
- platform ལ་ཡོད་པའི་ fitness site ཚུ་གི་དཔེ་དོན་ (examples) ལ་ link

### Email 4: Go Live (ཉིན་མོ་བདུན་)

- ຫົວຂໍ້: "ພ້ອມແລ້ວທີ່ຈະແບ່ງປັນ FitSite ຂອງທ່ານໃຫ້ໂລກຮູ້ບໍ?"
- ເນື້ອໃນ: ລາຍການກວດສອບສິ່ງທີ່ຄວນຢັ້ງຢືນກ່ອນທີ່ຈະແບ່ງປັນ, ວິທີເຊື່ອມຕໍ່ domain ທີ່ເປັນຂອງເຈົ້າ (ຖ້າໃຊ້ Growth/Pro), ຄຳແນະນຳການແບ່ງປັນຜ່ານສື່ສັງຄົມ

## ແຫຼ່ງຂໍ້ມູນຊ່ວຍເຫຼືອ

ສ້າງເນື້ອໃນຊ່ວຍເຫຼືອທີ່ສະເພາະເຈາະຈົງກັບ niche (ຄວາມຊ່ຽວຊານ) ເພື່ອຕອບຄຳຖາມທີ່ເຈົ້າຂອງສະຕູການອອກກຳລັງກາຍສອບຖາມແທ້ໆ:

### ບົດຄວາມໃນ Knowledge Base

- "ວິທີອັບເດດຕາຕະລາງຮຽນ"
- "ການເພີ່ມ ແລະ ແກ້ໄຂຂໍ້ມູນຄູຝຶກ (trainer profiles)"
- "ການປ່ຽນໂລໂກ້ ແລະ ສີຂອງສະຕູເຈົ້າ"
- "ການເຊື່ອມຕໍ່ domain ຂອງເຈົ້າເອງ" (ສຳລັບລູກຄ້າ Growth/Pro)
- "ການເພີ່ມ widget ການຈອງເວລາເຂົ້າໃນເວັບໄຊທ໌ຂອງເຈົ້າ"

ຂຽນບົດຄວາມເຫຼົ່ານີ້ໃຫ້ຜູ້ໃຊ້ທີ່ບໍ່ແມ່ນນັກພັດທະນາອ່ານໄດ້. ໃຊ້ຮູບພາບປະກອບ. ຫຼີກລ່ຽງການໃຊ້ຄຳສັບສະເພາະຂອງ WordPress.

### ຄູ່ມືວິດີໂອ (Video Walkthroughs)

ບັນທຶກຈໍລະທຽນສັ້ນໆ (2-3 ນາທີ) ສະແດງໃຫ້ເຫັນ:

- ການເຂົ້າສູ່ລະບົບຄັ້ງທຳອິດ ແລະ ການណែនាំ
- ການແກ້ໄຂໜ້າຫຼັກ (homepage)
- ການອັບເດດຕາຕະລາງຮຽນ
- ການເພີ່ມຄູຝຶກໃໝ່

ວິດີໂອນີ້ບໍ່ຈຳເປັນຕ້ອງເປັນການຜະລິດທີ່ສວຍງາມ. ສິ່ງທີ່ສຳຄັນແມ່ນຄວາມຊັດເຈນ, ມີປະໂຫຍດ ແລະ ເປັນຂອງ niche ຂອງເຈົ້າ.

## ການວັດແທກຄວາມສຳເລັດໃນການເຂົ້າຮ່ວມ (Measuring Onboarding Success)

ຕິດຕາມຕົວຊີ້ວັດເຫຼົ່ານີ້ເພື່ອຮູ້ວ່າການເຂົ້າຮ່ວມຂອງເຈົ້າໄດ້ເຮັດວຽກບໍ່:

- **Activation rate**: ເປີເຊັນຂອງຜູ້ລົງທະບຽນທີ່ປ່ຽນແປງໄຊທ໌ຂອງເຂົາເຈົ້າໃນອາທິດທຳອິດ?
- **First-week logins**: ລູກຄ້າໃໝ່ເຂົ້າສູ່ລະບົບຈັກເທື່ອໃນອາທິດທຳອິດຂອງເຂົາເຈົ້າ?
- **Support tickets from new customers**: ຈຳນວນສູງໝາຍຄວາມວ່າການເຂົ້າຮ່ວມຂອງເຈົ້າຂາດຫຍັງແດ່.
- **Trial-to-paid conversion**: ຖ້າເຈົ້າສະເໜີການທົດລອງ, ມີຈັກເປີເຊັນທີ່ປ່ຽນເປັນການຈ່າຍເງິນ?

## ສິ່ງທີ່ພວກເຮົາສ້າງໃນບົດຮຽນນີ້ (What We Built This Lesson)

- **ການສອນເຂົ້າລະບົບຄັ້ງທໍາອິດທີ່ມີຄູ່ມື** ພ້ອມກັບ Quick Start widget

- **ຊັບພະຍາກອນຊ່ວຍເຫຼືອສະເພາະ niche** ຂຽນມາສຳລັບເຈົ້າຂອງສະຕູຟິດເນສທີ່ບໍ່ມີຄວາມຮູ້ດ້ານເຕັກນິກ
