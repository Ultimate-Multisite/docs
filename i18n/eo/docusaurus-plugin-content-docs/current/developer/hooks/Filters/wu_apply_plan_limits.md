# Filtrilo: wu_apply_plan_limits

Permesas al plugin-programistoj mallongcirkviti la limigojn.

Vi povas uzi ĉi tiun filtrilon por ruli arbitran kodon antaŭ ol iu ajn el la limoj estos iniciatita. Se via filtrilo redonas iun ajn veran valoron, la procezo plu iros; se ĝi redonas iun ajn falsan valoron, la kodo revenos kaj neniu el la subaj hooks ruliĝos.
