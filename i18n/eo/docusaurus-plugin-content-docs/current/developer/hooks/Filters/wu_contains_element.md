# Filtro: wu_contains_element {#filter-wucontainselement}

Permesas al programistoj ŝanĝi la rezultojn de la komenca serĉo.

Tio utilas por triaj konstruiloj kaj similaj iloj.

## Parametroj {#parameters}

| Nomo | Tipo | Priskribo |
|------|------|-------------|
| $contains_elements | `bool` | Ĉu la elemento estas enhavata en la enhavo. |
| $content | `string` | La enhavo ekzamenata. |
| $element | `self` | La aktuala elemento. |
| $post | `null\|\WP_Post` | afiŝo por kontroli. |
