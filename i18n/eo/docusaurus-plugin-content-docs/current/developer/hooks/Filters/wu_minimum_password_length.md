# Filtrilo: wu_minimum_password_length

Filtru la minimuman pasvortan longon.

Devigata nur kiam wu_enforce_password_rules estas true.

## Parametroj {#parameters}

| Nomo | Tipo | Priskribo |
|------|------|-------------|
| $min_length | `int` | Minimuma pasvorta longo. Defaŭlte 12 (kongruas kun Defender Pro). |
| $defender_active | `bool` | Ĉu Defender Pro Strong Password estas aktiva. |

- 2.4.0
