# Action: wu_page_\{$this->id\}_register_widgets {#action-wupagethis-idregisterwidgets}

Ekfunkcias post kiam widgets estas registritaj por ĉi tiu paĝo.

The dynamic portion of the hook name, `$this-&gt;id`, refers to the page id.

|------|------|-------------|
| $id | `string` | La id de la paĝo. |
| $page_hook | `string` | La hoko de la paĝo. |
| $page | `object` | La objekto de la paĝo. |
