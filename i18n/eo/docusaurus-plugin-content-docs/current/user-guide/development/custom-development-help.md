# Kiel ak vi alti devlopi help de Codeable? {#how-to-get-custom-development-help-from-codeable}

Gare foje, ni vi tre necese jaka bon personalizajnaj por fari niaj retoj la esakite la manko. Se tio estas tio, ni rekomendas Codeable (https://codeable.io/partners/wpultimo/?ref=7wHwE), loko kie vi povas trovi ekspertajn devlopantojn kun Ultimate Multisite ekspertizo.

Codeable (https://codeable.io/partners/wpultimo/?ref=7wHwE) estas platformo, kiu konektas klientojn kun kvalifikitaj WordPress freelance devlopantoj, kiuj povas labori sur projektachj de ĉirkaŭ iu ajn granda. Ili havas meticulan aplikadon por certigi, ke vi estas korespondita kun ekspertaj specifike kvalifikitaj por la tasko, kiun vi havas en la manon.

### Kiel krei projekton en Codeable? {#how-to-create-a-project-in-codeable}

Unue, diretete al Codeable (https://codeable.io/partners/wpultimo/?ref=7wHwE) kaj kliku sur butonon **Start a Project** (Krei Projekton).

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Sekve, por envipli projekton, kliku en la malan lado al **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

En "I need help with" (Mi bezonas helpaj kun), elektu **Customizing** (Personalizi) :

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

En "My" (Mia), elektu **Plugin** (Plugino) :

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Tamen kliku sur butonon **Continue** (Kontinu).

<!-- Screenshot unavailable: Codeable project form Continue button -->

Sekve, venas tempo deskribi vian projektion kaj bezonas la plej bonan manjon. Provi doni iliajn informojn kiel eblee multe!

<!-- Screenshot unavailable: Codeable project description form -->

Rondigu la sekvantajn demandojn por helpi kun precoj.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Ansek, se vi ne havas [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) konton, kreu unu kaj klikas la butonon **Publish My Project**.

<!-- Skrinomaterial ne estas disponabla: Kreo Codeable konton kaj Publish My Project butono -->

Kiam vi publikigas vian [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) projekton, la platformo konektas vin kun programistoj, kiuj korespondas al la skintan neceson por via projektaj laboro.

Ĉi tiu procezo helbus vi trovi la ĝustan homon por via laboro, kiu havas esperen konceson kun WordPress kaj Ultimate Multisite!

La minimuma tarifo de Codeable estas $70–$120 USD per horo. Estimo ŝanĝiĝos laŭ la fokuso de la laboro, la komplekseco kaj la urgenco de la projekto.

Kiam vi ricevas vian estimon, vi alcastu la programistadon, kiu plej bone korespondas al viaj bezonoj kaj komencu via projekton kun li!
