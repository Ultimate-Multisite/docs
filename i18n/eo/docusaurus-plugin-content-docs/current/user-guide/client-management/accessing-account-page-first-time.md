# Kiel mi la vôtre konta paĝon pri valoro fton?

Bienvenue en Ultimate Multisite!

Akcesu al vôtre konta paĝo per linian ki vi ricevis en vian e-mail post kaufi nôt plugin:

<!-- Screenshot unavailable: Account page email with access link -->

Uzu la e-mail, ki vi donis pandan la aĉet, kaj la ŝlosilon, ki vi ricevis en vian e-mail.

<!-- Screenshot unavailable: Login page with email and password fields -->

Ĉar tio estas vôla unua fojto al ĉi tiu konto, vi demandos ŝanĝi vian ŝlosilon. Ŝanĝu ĝin kaj kliku sur la butono _Save_ (Konservi).

<!-- Screenshot unavailable: Change password form with Save button -->

Sur vian konta paĝo, vi povas administra ĉiuj vian konta detalojn kiel fakturaj transakcioj, renovacioj, kanceluoj, kaj multaj aliaj thingoj.
