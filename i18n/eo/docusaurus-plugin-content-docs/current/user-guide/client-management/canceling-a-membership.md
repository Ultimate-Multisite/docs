# Membro Canceli (v2)

_**IMPORTANTE NOTA: Ĉi tiu artikolo referas al Ultimate Multisite versio 2.x.**_

Canceli la membraon de via klientaj lado estas relativfacile.

Se via uzanto volas canceli sian membraon, ili devus predawn aliri sian ****Konton**** (Account) paĝon.

Tial, ili devus kliki al **Perikla Zonon** (Danger Zone), kaj kliki al **Dela Konton** (Delete Account).

![Perikla Zona de la Paĝo de Konto kun butono Deleta Konton](/img/account-page/danger-zone.png)

Un ramo (module) ŝtonĝos, demandante sian konfirmon. Ili devus aktivi togglin la opcion **Konfimi Deletion de Konto** (Confirm Account Deletion) kaj kliki al **Dela Konton**.

![Dialogo de Konfimo de Deletion de Konto](/img/account-page/delete-account-confirm.png)

Tio logas ilin el la sitio kaj canceli sian membraon sur via retoj, kiel ankaŭ dele sian site de via retoj.

![Ellogita post deletion de konto](/img/account-page/logged-out-after-delete.png)

![Konfimo de Membrao Cancela](/img/account-page/membership-canceled.png)

**Importante:** Notu, ke ĉi tiu akcio ne ekskludus la klienton el via rero, ĉar klient povas havi pli ol un membraon aŭ neniu membraon linkita al ili.

![Kliento estas tute en la rero post cancelo](/img/admin/customer-after-cancel.png)

![Kliento sen aktiva membraoj](/img/admin/memberships-customer-empty.png)
