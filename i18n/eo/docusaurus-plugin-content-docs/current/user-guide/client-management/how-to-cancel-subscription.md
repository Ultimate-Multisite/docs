# Kiel mi povas kanceli min la subskribon? {#how-can-i-cancel-my-subscription}

Se vi havas licencon Ultimate Multisite, vi povas kanceli ĝian renovejon kiam vi volas. Simple sekvu la paŝojn al unuoparte:

<!-- Screenshot unavailable: Account page email with access link -->

Uzu la e-mailon, kiun vi donis pri la aĉeto, kaj via ŝlosilon.

<!-- Screenshot unavailable: Login page with email and password fields -->

Tio estas via initialej skurtaco de la Konta Paĝo:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

Por kanceli vian subskribon, sur la menuko en la malpare, dirigi al _Renewing & Billing._ (Renovejo kaj Fakturaprocesado.)

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

Klapu la arrowon de la dreto por ofiĉi vin flankajn fenestrion. Tiam vi devas selekti la opcion, kiu kancelu la **auto-renovejon**.

La sistema montros al vi konfirmaĵon.

Se vi estas certa, ke vi volas kanceli vian subskribon, klapu la butono _**Cancel Renewals**_ (Kancelu Renovejon). Post ĉi tiu akcio, vi demandos respondi rapidej sur un shorta envesto.

Fini! Via subskribo ne renoveĝos aŭtomate.

**Vi havos validan klavon ĝis tiam ke via subskribo ekspiras**. Se vi volas reactivați vian subskribon, vi devus fari tio manuele.
