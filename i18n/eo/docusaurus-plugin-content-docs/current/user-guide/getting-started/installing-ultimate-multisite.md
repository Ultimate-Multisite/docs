# Instalado de Ultimate Multisite

:::note
Ĉi tiu tutoro asumas, ke vi 이미 havas WordPress Multisite instalitan kaj konfiguratitan. Por lerni kiel ti tio fariĝas, renkontu [ti ĉi tutoran](https://www.wpbeginner.com/glossary/multisite/) de WP Beginner.
:::

## Instalado de la Plugino

Ultimate Multisite estas liberaj por [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/).

De via **Network Admin Dashboard**, diru al **Plugins → Add New Plugin**.

![Pagino "Add New Plugin" de Network Admin](/img/installation/add-new-plugin.png)

Serĉu **"Ultimate Multisite"** (con citoj por preciza korespondenco), kaj ĝi aparos kiel la unua rezulto. Kliku sur **Install Now**.

![Serĉaj rezultoj montras Ultimate Multisite](/img/installation/search-ultimate-multisite.png)

Unbeinfakte instalita, kliku sur **Network Activate**, por aktivi la pluginon en via tuta rano.

![Plugin instalita per butono Network Activate](/img/installation/plugin-installed.png)

Post aktiva, vi estu aŭtomate redirekti al la Setup Wizard.

![Plugin aktiva kaj redirekti al wizard](/img/installation/plugin-activated.png)

## Setup Wizard

La Setup Wizard guidos vin per konfiguri Ultimate Multisite en ĉirkaŭ 10 minutoj.

### Bienvenon

Kliku sur **Get Started**, por komenci.

![Bienvena skrinco de la Setup Wizard](/img/installation/wizard-welcome.png)

### Pre-instalaj Kontoj

Ĉi tiu paŝo kontrolas vian sisteminformon kaj WordPress instaliron, por certigi, ke ĝi plibidis Ultimate Multisite's rakbezonojn. Se ĉio ŝajnas bone, kliku sur **Go to the Next Step**.

![Pre-instalaj kontroloj montras sistemrakbezonojn](/img/installation/wizard-pre-install-checks.png)

:::note Buton "Network Activate" (v2.6.1+)
Se Ultimate Multisite estis instalita, sed **neaktivita ak la reto** — pa ekzemple, se vi klikis sur **Activate** (single-site) aldatne **Network Activate** de la skriĉo de plugin'oj de reto — la paŝto Pre-install Checks detektos tion kaj prezentas butonon **Network Activate**.

Klikado sur **Network Activate** aktive la pluginon sur tut viaj multisite reto aŭtomatik. Unfoje aktiva, la wizard kontinue normalan al la paŝto instalado. Vi ne bezonas fori la wizard por elteni la aktivita staton.
:::

### Instalado

La instalisto kreos necesajn bazdatoblasojn kaj instalas la filon `sunrise.php`, kiun Ultimate Multisite bezonas funkcii por operi. Klikas sur **Install** por procedi.

![Instalado paŝto montranta bazdatoblasojn kaj sunrise.php](/img/installation/wizard-installation.png)

### Via Kompanio

Plenpusu viaj kompania informojn kaj definu vian defaultmonetaron. Ĉi tiu informeco wirdu uzata tut sur via WaaS platformo. Klikas sur **Continue** kiam vi finas.

![Via Kompanio konfigurita paŝto](/img/installation/wizard-your-company.png)

### Default Kontenaj

Tiu paŝto permesas instali prédefinitajn mallonojn, produktojn kaj aliaj komencajn kontentojn. Ĉi tio estas granda maniero akiri familian konsciencon kun la funkcioj de Ultimate Multisite. Klikas sur **Install** por aĉi la defaultan kontenon, aŭ pasu ĉi tiun paŝton se vi preferas komenci de nulo.

![Instalado paŝto de defaultaj kontentoj](/img/installation/wizard-default-content.png)

### Rekomanditaj Plugin'oj

Opционаle instalu rekommanditajn kompanĝ plugin'ojn. Klikas sur **Install** por aĉi ilin aŭ pasu por kontinui.

![Paŝto de rekommanditaj plugin'oj](/img/installation/wizard-recommended-plugins.png)

### Listata!

Tio estas tio! Via Ultima Multisite instalado estas kompleta. Nun vi povas komenci konstrui vian Vikiviblon kiel Serva Platforma per la **Network Admin Dashboard**.

![Instalo kompleta - Parega skriĉo](/img/installation/wizard-ready.png)

![Network Admin Dashboard kun Ultima Multisite aktiva](/img/installation/network-dashboard.png)

Komenci ĝui!
