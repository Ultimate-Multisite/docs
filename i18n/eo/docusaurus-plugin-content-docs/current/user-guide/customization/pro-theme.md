# Ĉmiziigo de Via Administra Panelo kun PRO Temo {#customizing-your-admin-panel-with-pro-theme}

Multaj foj estas necesas doni al via klientoj pli specialan sperton en la administran panelo de WordPress, kiam vi faras klientajn laborojn. Se vi serĉas pli profesian ŝtoncon kaj senton, PRO Temo povas helpi vin.

PRO Temo estas administran temo, kiun oni sviluppis por helpi vin pri modifi la Dashboard de via klientoj. Ĝi ankaŭ estas parto de la bazaj plugin-partoj de Ultimate Multisite.

Post instalado, diru al via `_Plugins_` paĝo kaj kliku por aktivi ĝin sur via Retejo (Network).

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Aktivi ĝin sur la tuta Retejo sur via _Plugins_ paĝo_

_**NOTO:**_ _Se vi nur instalas ĝin, ĝi montros sur via klientoj kiel opcio, kiun oni povas markui por aĉeti la temon nur al tiu uzanto._

Nove, vi havas vian administran interfacon komplete redizajnit! Vi ankaŭ havas novan menuan en la maltravena kolono nomita PRO Theme.
