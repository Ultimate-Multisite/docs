# Subtenaj Agentoj

Administri tutan reton memstare ne estas facila tasko. Dum via reto komencas kreski, vi komencos vidi kreskantan postulon je subteno. Estos bona ideo venigi homojn, kiuj helpos vin malpezigi vian laborŝarĝon. Tamen, doni plenan aliron/permeson estas riske kaj povas endanĝerigi vian reton. Ĝuste tie ĉi tiu aldonaĵo Subtenaj Agentoj utilas.

Ĉi tiu aldonaĵo permesas al vi krei novan tipon de WordPress-uzanto nomatan Subtena Agento, kaj donas al vi plenan regon pri kiujn permesojn tiu uzanto havas. Doni permeson estas tiel facile kiel ŝalti baskulbutonon, kaj malŝalti ĝin se vi decidas forigi ĝin.

## Trajtoj {#features}

- Detaligita listo de WordPress & Ultimate Multisite-permesoj
- Ebligi & malebligi aliron facile
- Krei senliman nombron da Agent-kontoj

## Instalado {#installation}

1. Alŝutu 'ultimate-multisite-support-agents' al la dosierujo '/wp-content/plugins/'
2. Aktivigu la kromprogramon per la menuo 'Kromprogramoj' en WordPress
3. Sekvu la paŝon-post-paŝan Sorĉiston por agordi la kromprogramon
