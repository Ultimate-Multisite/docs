# Enhava Sinkronigo

Multisite Ultimate: Enhava Sinkronigo

Ĉi tiu aldonaĵo por Multisite Ultimate permesas al retaj administrantoj sinkronigi enhavon de iu ajn retejo al aliaj retejoj en la reto. Ĝi subtenas ĉiujn afiŝotipojn, inkluzive de afiŝoj, paĝoj, propraj afiŝotipoj kaj Elementor-ŝablonoj. La aldonaĵo inkluzivas plurpaŝan sorĉistan interfacon por elekti enhavon kaj celajn retejojn, kun aro-prilaborado kaj realtempaj progresaj reagoj.

## Trajtoj

* Enhava Elekto: Vida kartokrado kun bildetoj kaj filtrado laŭ afiŝotipo
* Ĉiuj Afiŝotipoj: Sinkronigu afiŝojn, paĝojn, produktojn, Elementor-ŝablonojn kaj ajnan propran afiŝotipon
* Reteja Elekto: Elektu al kiuj retejoj sinkronigi enhavon, kun ekskludo de fonta retejo kaj antaŭelekto de idaj retejoj
* Aro-Prilaborado: Sinkronigas al 10 retejoj samtempe kun progresaj ĝisdatigoj
* Media Sinkronigo: Aŭtomate kopias kaj remapas mediajn dosierojn (bildojn, filmetojn) de afiŝa enhavo al celaj retejoj
* Taksonomia Sinkronigo: Konservas kategoriojn, etikedojn kaj proprajn taksonomiojn
* Elementor-Subteno: Plena subteno por Elementor-ŝablonoj kun CSS-regenerado kaj ŝablona ID-remapado
* Konflikta Solvado: Opcio anstataŭigi ekzistantan enhavon aŭ preterlasi ĝin
* Kaŝmemora Administrado: Aŭtomate malplenigas la Elementor-kaŝmemoron ĉe celaj retejoj post sinkronigo

## Postuloj

* WordPress 5.3 aŭ pli alta
* PHP 7.4 aŭ pli alta
* Multisite Ultimate plugin (aktiva)
* Elementor plugin (nedeviga, postulata por sinkronigo de Elementor-ŝablonoj)

## Instalado

1. Alŝutu 'ultimate-multisite-content-sync' al la dosierujo '/wp-content/plugins/'
2. Aktivigu la plugin per la menuo 'Kromprogramoj' en via administrilo de WordPress-reto
3. Agordu agordojn en Ultimate Multisite → Agordoj → Enhava Sinkronigo
4. Iru al Ultimate Multisite → Enhava Sinkronigo por komenci sinkronigi
