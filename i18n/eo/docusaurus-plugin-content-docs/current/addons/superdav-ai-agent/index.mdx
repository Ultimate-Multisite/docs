# Superdav AI Agent

Superdav AI Agent estas altnivela WordPress AI-asistanto, kiu etendas Gratis AI Agent per etos-konsciaj kapabloj, plibonigitaj interretaj serĉkapabloj, funkcioj de reteja konstruilo, kaj laŭvolaj aŭtomatigaj integriĝoj. Versio 1.18.0 aldonas ilojn por legi Google Calendar, mapadon de kontaktoj kaj partoprenantoj, homajn aprobajn pordegojn, deduplikadon de memorigiloj, TextBee-SMS-sciigojn, proviziadon de administrata Superdav-servo, kaj aparte distribuatan Altnivelan akompanan pakaĵon por fidataj programistaj laborfluoj.

## Ĉefaj Funkcioj

- **Etos-konsciaj kapabloj** — aŭtomate adaptas gvidadon kaj kapablojn laŭ via aktiva WordPress-etoso kaj instalitaj kromaĵoj
- **Multoblaj interretaj serĉprovizantoj** — elektu inter Brave Search kaj Tavily por pli riĉaj serĉrezultoj kaj altnivelaj esplorkapabloj
- **Kontaktformularoj de reteja konstruilo** — kreu plene funkciajn kontaktformularojn rekte el la babila interfaco uzante vian instalitan formularan kromaĵon
- **Plibonigita reteja konstruilo** — plibonigitaj plurfazaj retejo-konstruaj planoj kun pli bona malkovro de kromaĵoj kaj erara reakiro
- **Enkonstruitaj etoskapabloj** — specialigita gvidado por Blokaj Etosoj, Klasikaj Etosoj, Kadence Blocks, kaj Kadence Theme
- **Plibonigita WooCommerce-integriĝo** — refaktorita por uzi denaskajn WooCommerce-API-ojn por pli bona fidindeco kaj kongrueco
- **Aŭtomata detekto de provizantoj** — aŭtomate detektas kaj refreŝigas disponeblajn kromaĵojn kaj etosojn kiam ili estas aktivigitaj aŭ malaktivigitaj
- **Propra generado de emblemoj** — generu kaj enkorpigu proprajn SVG-emblemojn kun aŭtomata nomspac-sekura purigado
- **Validigo de paletra kontrasto** — kontrolu kolorparojn por WCAG-alirebleca konformeco antaŭ ol apliki ilin al via etoso
- **Foto-plibonigita malkovro** — alŝutu fotojn dum la malkovra intervjuo de Theme Builder por influi la projektan direkton
- **Gastigaj menuoj** — generu strukturitajn paĝojn de manĝaĵaj kaj trinkaĵaj menuoj por restoracioj, kafejoj, kaj gastigaj entreprenoj
- **Labortablaj kaj poŝtelefonaj antaŭrigardoj** — antaŭrigardu vian dezajnon sur labortablaj kaj poŝtelefonaj aparatoj dum elekto de dezajna direkto
- **Nivelo-1 reteja skrapado** — sd-ai-agent/site-scrape nun disponeblas defaŭlte por analizi konkurantajn kaj inspirajn retejojn
- **Iloj por legi Google Calendar** — legu agorditajn kalendarojn kaj eventojn por ke aŭtomatigoj povu rezoni pri venonta horara kunteksto
- **Mapado de kontaktoj kaj partoprenantoj** — kongruigu kalendarajn partoprenantojn kun konataj retejaj uzantoj kaj kontaktoj antaŭ ol generi sekvan laboron
- **Homaj aprobaj pordegoj** — paŭzigu sentemajn aŭtomatigojn por revizio antaŭ ol sendi mesaĝojn, rekomenci laboron, aŭ fari ŝanĝojn
- **Memorigilaj registroj kaj deduplikado** — spuru memorigilojn jam senditajn, por ke ripetaj aŭtomatigaj ruloj ne sciigu la saman ricevonton dufoje
- **TextBee-SMS-sciigoj** — sendu agorditajn tekstmesaĝajn sciigojn per TextBee kiam SMS-provizanto estas ebligita
- **Proviziado de administrata Superdav-servo** — konektu subtenatajn retejojn al gastigitaj Superdav-finpunktoj kun aŭtomata agordo anstataŭ mana finpunkta agordo
- **Altnivela akompana pakaĵo** — aparte distribuitaj fidata-programistaj iloj por dosiersistemo, datumbazo, WP-CLI, REST-dispeĉilo, kromaĵa konstruilo, git-momentfoto, uzanto-administrado, kaj mezurkomparaj laborfluoj

## Postuloj

- WordPress 6.0 aŭ pli alta
- PHP 8.1 aŭ pli alta
- WordPress Multisite (ret-aktivigita) aŭ unu-reteja instalado
- Aktiva kromaĵo de AI-provizanta konektilo (Anthropic, OpenAI, aŭ kongrua)
- Gratis AI Agent v1.9.0 aŭ pli alta (Superdav AI Agent etendas Gratis AI Agent)

## Instalado

1. Alŝutu la kromaĵajn dosierojn al `/wp-content/plugins/superdav-ai-agent`
2. Aktivigu la kromaĵon per la ekrano **Kromaĵoj** en WordPress
3. Certigu, ke Gratis AI Agent ankaŭ estas aktivigita
4. Iru al **Gratis AI Agent → Agordoj** kaj agordu vian interretan serĉprovizanton, administratan Superdav-servan konekton, Google Calendar, aŭ TextBee-akreditaĵojn laŭbezone

La WordPress.org-pakaĵo enhavas la kernajn funkciojn de Superdav AI Agent. La Altnivela akompana pakaĵo estas konstruita kaj distribuata aparte por medioj kie fidataj programistaj iloj estas eksplicite instalitaj kaj rajtigitaj.

## Rapida Komenco

Post aktivigo, Superdav AI Agent aŭtomate plibonigas vian instalaĵon de Gratis AI Agent per etos-konsciaj kapabloj kaj aldonaj ebloj. Malfermu la panelon **Gratis AI Agent** en la administra flankbreto de WordPress kaj komencu uzi la novajn funkciojn.

**Ekzemplaj instrukcioj:**

```
Aldonu kontaktformularon al la paĝo Kontakto.
```

```
Serĉu la reton pri la plej novaj plej bonaj praktikoj por WordPress-sekureco.
```

```
Kreu heroan sekcion uzante la blokajn ŝablonojn de la aktiva etoso.
```

```
Reviziu la agorditajn Google Calendar-eventojn de morgaŭ kaj verku memorigilojn por partoprenantoj.
```

```
Sendu SMS-memorigilon post kiam mi aprobos la mesaĝon.
```

## Etos-Konsciaj Kapabloj

Superdav AI Agent aŭtomate detektas vian aktivan etoson kaj instalitajn kromaĵojn, ŝargante la taŭgan etos-konscian kapablon en la kuntekston de la asistanto. Disponeblaj kapabloj inkluzivas:

| Kapablo | Aplikiĝas Al |
|---|---|
| **Blokaj Etosoj** | Etosoj uzantaj la blok-bazitan arkitekturon de WordPress (Plena Reteja Redaktado) |
| **Klasikaj Etosoj** | Tradiciaj WordPress-etosoj uzantaj PHP-ŝablonojn kaj la klasikan redaktilon |
| **Kadence Blocks** | Retejoj kun la kromaĵo Kadence Blocks aktiva |
| **Kadence Theme** | Retejoj kun la etoso Kadence kiel la aktiva etoso |

Vidu [Etos-Konsciaj Kapabloj](../../user-guide/configuration/theme-aware-skills) por detala gvidado pri ĉiu kapablo.

## Interretaj Serĉprovizantoj

Superdav AI Agent v1.10.0 aldonas **Tavily** kiel alternativon al Brave Search por interretaj serĉpetoj. Ambaŭ provizantoj ofertas realtempajn serĉrezultojn kaj povas esti agorditaj en **Gratis AI Agent → Agordoj → Altnivela**.

| Provizanto | Trajtoj | Agordo |
|---|---|---|
| **Brave Search** | Rapidaj, privatec-fokusitaj serĉrezultoj | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Pli riĉaj serĉrezultoj, altnivelaj esplorkapabloj | [Tavily API](https://tavily.com/) |

Vidu [Interreta Serĉo](../../user-guide/configuration/internet-search) por agordaj instrukcioj.

## Kalendaraj, Aprobaj, kaj Sciigaj Laborfluoj

Superdav AI Agent v1.18.0 vastigas aŭtomatigajn laborfluojn preter enhavaj kaj retejo-konstruaj taskoj:

- **Google Calendar-iloj** permesas al la agento legi agorditajn kalendarojn kaj eventodetalojn kiam kalendaraj akreditaĵoj disponeblas.
- **Kontaktaj kaj partoprenantaj helpiloj** mapigas eventajn partoprenantojn al WordPress-uzantoj aŭ konataj kontaktoj antaŭ ol prepari memorigilojn aŭ sekvajn taskojn.
- **Homaj aprobaj pordegoj** permesas al aŭtomatigo paŭzi ĝis rajtigita uzanto revizias la proponitan agon.
- **Memorigilaj rikordoj** malhelpas duoblajn kalendarajn aŭ sciigajn memorigilojn kiam aŭtomatigo estas reprovitaj aŭ planitaj ripete.
- **TextBee SMS-subteno** ebligas tekstmesaĝajn sciigojn kiam la TextBee-provizanto estas agordita.

Uzu ĉi tiujn ilojn por horar-konscia helpo, partoprenanta sekvado, kaj memorigilaj laborfluoj kiuj ankoraŭ postulas homan revizion antaŭ ol sentemaj mesaĝoj estas senditaj.

## Agordo

Ĉiuj agordoj estas sub **Gratis AI Agent → Agordoj** en la WordPress-admino. Superdav AI Agent aldonas la jenajn agordajn opciojn:

| Agordo | Priskribo |
|---|---|
| **Interreta Serĉa Provizanto** | Elektu inter Brave Search kaj Tavily (nedeviga) |
| **Tavily API Key** | API key por Tavily-interreta serĉo (se Tavily estas elektita) |
| **Administrata Superdav-Servo** | Konektu subtenatajn retejojn al la gastigita Superdav-servo kun aŭtomata endpoint-provizado |
| **Google Calendar-Akreditaĵoj** | Agordu kalendaran aliron por ke la agento povu legi kalendarojn kaj eventojn |
| **TextBee SMS-Provizanto** | Agordu TextBee-akreditaĵojn por aprobitaj SMS-sciigaj laborfluoj |

## Altnivela Kunula Pakaĵo

Superdav AI Agent v1.18.0 aldonas aparte distribuitan Altnivelan kunulan pakaĵon por fidindaj programistaj laborfluoj. Ĉi tiuj kapabloj ne estas kunigitaj en la WordPress.org kerna pakaĵo. Instalu la kunulan pakaĵon nur sur retejoj kie administrantoj eksplicite fidas la agenton aliri programistajn ilojn.

Altnivelaj iloj inkluzivas dosiersistemajn operaciojn, datumbazan aliron, WP-CLI, REST-dispeĉilajn laborfluojn, plugin-konstruajn utilaĵojn, git-momentkopiojn, uzant-administrajn helpilojn, kaj rendimentomezurajn ilojn. Tenu ĉi tiujn ilojn limigitaj al fidindaj administrantoj kaj reviziu aprobo-pordegan agordon antaŭ ol ebligi ilin sur produktadaj retejoj.

## Plibonigoj en v1.18.0

- **Google Calendar-subteno** — legi agorditajn kalendarojn kaj eventojn por horar-konsciaj aŭtomatigoj
- **Kontakta kaj partoprenanta mapigado** — kongruigi kalendarajn partoprenantojn al reteja uzantoj kaj kontaktoj
- **Homaj aprobaj pordegoj** — paŭzigi sentemajn aŭtomatigojn por revizio antaŭ daŭrigo
- **Memorigila deduobligo** — registri memorigilojn por eviti duoblajn sciigojn ĉe reprovoj aŭ ripetaj ruloj
- **TextBee SMS-provizanto** — sendi agorditajn tekstmesaĝajn sciigojn post postulataj aproboj
- **Agordo de administrata Superdav-servo** — provizi gastigitajn servajn endpoints aŭtomate por subtenataj retejoj
- **Altnivela kunula distribuo** — publikigi fidind-programistajn ilojn aparte de la WordPress.org kerna konstruaĵo
- **Fidindecaj korektoj** — plibonigi modelan elekton, petajn tempolimojn, defaŭltan agordan modeltraktadon, rezonad-tekstan traktadon, reprovan gvidadon, kalendaran token-fortikigon, memorigilan fortikigon, enkondukajn lanĉojn, konfirmitajn kapablo-rekomencojn, kaj WordPress.org-pakajn reviziajn problemojn

## Plibonigoj en v1.16.0

- **AI Client kaŝmemoro** — nun subtenata de transients por trans-peta persisto, malhelpante datumperdon dum longe rulataj agentaj taskoj
- **Nivelo 1 reteja skrapilo** — sd-ai-agent/site-scrape nun disponeblas defaŭlte en Theme Builder por analizi konkurantajn kaj inspirajn retejojn
- **Plugin vico-agaj ligiloj** — korektitaj kaj renomitaj por klareco

## Plibonigoj en v1.10.0

- **WooCommerce integriĝo refaktorita** — nun uzas denaskajn WooCommerce API-ojn por pli bona fidindeco kaj kongrueco
- **Aŭtomata refreŝigo de provizanto-listo** — aŭtomate detektas kiam plugin-oj estas aktivigitaj aŭ malaktivigitaj kaj ĝisdatigas la disponeblan provizanto-liston
- **Plibonigita reteja konstruilo** — plibonigitaj plurfazaj retej-konstruaj planoj kun pli bona erar-reakiro
- **Theme-konsciaj kapabloj** — kvar novaj enkonstruitaj kapabloj kiuj aŭtomate adaptiĝas al via aktiva theme

## Cimo-korektoj en v1.10.0

- Korektis kapablon `navigate-to`, kiu kaŭzis senfinajn reŝargajn buklojn sur kelkaj administraj paĝoj
- Korektis kapablon `list-posts`, kiu ne ĝuste solvis kategoriajn kaj etikedajn nomojn al slugs
- Korektis WP-CLI-komandojn al kiuj mankis namespace aliases post antaŭa refaktoro
- Korektis eventan aŭtomatigon por gracie trakti retejojn kie aŭtomatigaj tabeloj ankoraŭ ne estas kreitaj
- Korektis kapablon `memory-save` uzantan la ĝustan namespace prefix en la sistema instrukci-konstruilo
- Korektis scalar tool results por esti ĝuste envolvitaj antaŭ ol esti resenditaj al la AI
- Korektis uzad-statistikojn por ĝuste trakti la heredan kapablo-ŝlosilan formaton dum ĝisdatigo el pli malnovaj versioj
