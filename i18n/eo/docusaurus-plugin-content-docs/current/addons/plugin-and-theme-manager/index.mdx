# Administrilo de Kromprogramoj kaj Etosoj {#plugin--theme-manager}

Multisite Ultimate: Administrilo de Kromprogramoj kaj Etosoj

Redaktu aŭ kaŝu la metainformojn (titolo, aŭtoro, miniaturbildo kaj priskribo) de instalitaj kromprogramoj kaj etosoj. Kreu kategoriojn por kromprogramoj kaj etosoj por permesi al viaj uzantoj filtri ilin kaj aldoni belan propran paĝon Kromprogramoj al la panelo de viaj uzantoj.

Por uzi la kromprogramon, vi povas simple navigi al viaj paĝoj Kromprogramoj aŭ Etosoj en via reta administrado kaj alklaki "Redakti informojn de Kromprogramo/Etoso".

## Instalado {#installation}

1. Alŝutu 'ultimate-multisite-plugin-and-theme-manager' al la dosierujo '/wp-content/plugins/'
2. Aktivigu la kromprogramon per la menuo 'Kromprogramoj' en via WordPress reta administrado

## Uzgvidilo {#usage-guide}

Ultimate Multisite: Administrilo de Kromprogramoj kaj Etosoj estas potenca **premium** Ultimate Multisite aldonaĵo, kiu donas al vi plenan regon super la maniero kiel viaj retaj uzantoj vidas kaj interagas kun la opcioj de kromprogramoj kaj etosoj en via reto.

Ĉi tiu aldonaĵo servas tri celojn:

  1. Ĝi ebligas al vi tute ŝanĝi la metadatumojn de kaj kromprogramoj kaj etosoj. Tio inkluzivas nomon, priskribon, aŭtoran nomon, kaj eĉ miniaturbildojn!

  2. Ĝi ebligas al vi kategoriigi kromprogramojn kaj etosojn, por permesi al viaj uzantoj facile ordigi ilin.

  3. Ĝi donas al vi la opcion anstataŭigi la malnovan, enuigan paĝon de kromprogramoj per nova etos-simila paĝo kun ordigo laŭ kategorioj.

Post elŝutado kaj instalado de ĝi, nova langeto - _Agordoj de aldonaĵo_ \- aperos sur la paĝo de Ultimate Multisite Agordoj.
