# Fluent Forms

Ĉi tiu aldonaĵo ebligas aldoni limojn al Fluent Forms en Multisite Ultimate en produktoj, membrecoj aŭ retejoj.
Fluent Forms uzas proprajn tabelojn anstataŭ propraj afiŝotipoj, do la normalaj limigaj opcioj ne funkcias kun Fluent Forms.

## Funkcioj

- Aldonu limojn por Fluent Forms aŭ Conversational Fluent Forms
- Klientoj ricevas afablan mesaĝon instigantan ilin ĝisdatigi sian planon kiam ili atingas la limon
