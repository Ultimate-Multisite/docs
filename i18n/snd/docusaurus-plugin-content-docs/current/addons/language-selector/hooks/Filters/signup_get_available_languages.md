# Filter: signup_get_available_languages

فرنٽ-اينڊ سائيٽ سائن اپس لاءِ موجود ٻولين جي فهرست کي فلٽر ڪري ٿو.

هن hook ڏانهن خالي array موڪلڻ سان signup form تي setting جي آئوٽ پٽ کي بند ڪيو ويندو، ۽ سائيٽ ٺاهڻ وقت default ٻولي استعمال ٿيندي. جيڪي ٻوليون اڳ ۾ نصب ٿيل نه هونديون، سي هٽايون وينديون.

## Parameter

| Name | Type | Description |
|------|------|-------------|
| $available_languages | `array` | موجود ٻوليون. |
