# ٻولي چونڊيندڙ {#language-selector}

Ultimate Multisite: سائن اپ تي ٻولي چونڊيندڙ

سائن اپ وهڪري ۾ ٻولي چونڊڻ جو خانو شامل ڪريو.

## انسٽاليشن {#installation}

1. 'ultimate-multisite-language-selector' کي '/wp-content/plugins/' ڊائريڪٽري ۾ اپلوڊ ڪريو
2. پنهنجي WordPress نيٽ ورڪ انتظاميا ۾ 'پلگ اِنز' مينيو ذريعي پلگ اِن کي فعال ڪريو

## استعمال جي رهنمائي {#usage-guide}

هن مفت اضافي سان توهان پنهنجي استعمال ڪندڙن کي اهو چونڊڻ ڏئي سگهو ٿا ته اهي سائن اپ وهڪري ۽ لاگ اِن صفحي تي ڪهڙي ٻولي ڏسڻ چاهين ٿا.

[پنهنجو لائسنس فعال ڪرڻ](../../user-guide/getting-started/installing-ultimate-multisite.md) کان پوءِ توهان Ultimate Multisite جا اهي اضافا انسٽال ڪري سگهندا جن تائين توهان جو لائسنس رسائي ڏئي ٿو.

پنهنجي Ultimate Multisite سيٽنگن واري صفحي تي وڃو ۽ پوءِ، کاٻي پاسي واري مينيو ۾، _اضافا_ ليبل تي ڪلڪ ڪريو.

![اضافن جو صفحو](/img/admin/addons-page.png)

_ٻولي چونڊيندڙ انسٽال ڪرڻ شروع ڪرڻ لاءِ اضافن واري صفحي تي وڃو_

اتي، توهان کي اسان جا سڀ پيش ڪيل اضافا ملندا.

مفت اضافا اسان جي سڀني گراهڪن لاءِ موجود آهن، ڀلي اهي ڪنهن به سبسڪرپشن منصوبي تي سائن اپ ڪيا هجن.

اضافي جي ٿمب نيل هيٺ _هاڻي انسٽال ڪريو!_ بٽڻ تي ڪلڪ ڪريو. پوءِ ان کي پلگ اِنز صفحي تي فعال ڪريو.

![انسٽال بٽڻ سان اضافن جو صفحو](/img/admin/addons-page.png)

_اضافي کي انسٽال ڪرڻ لاءِ بٽڻ تي ڪلڪ ڪريو_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_ان کي پنهنجي پلگ اِنز صفحي تي فعال ڪريو_

هاڻي، Ultimate Multisite جي سيٽنگن واري مينيو ڏانهن وڃو ۽ _نيٽ ورڪ سيٽنگون_ ٽيب ۾ داخل ٿيو. اتي، صفحي جي تري ۾، توهان کي _ٻوليون_ نالي هڪ نئون حصو ملندو.

![سيٽنگن جو صفحو](/img/config/settings-general.png)

_هڪ بلڪل نئون حصو ظاهر ٿئي ٿو!_

توهان ان کي خالي ڇڏي سگهو ٿا ته جيئن سڀ ٻوليون اختيارن طور ڏيکارجن، يا اهي چونڊي سگهو ٿا جيڪي توهان موجود ڪرڻ چاهيو ٿا. ٻوليون اهي آهن جيڪي توهان پنهنجي WordPress سيٽ اپ ۾ انسٽال ڪيون آهن.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_هن WP انسٽاليشن تي موجود ٻولين جي فهرست_

توهان ڪيل تبديليون محفوظ ڪرڻ کان پوءِ، توهان چونڊيل ٻولين سان هڪ چونڊ مينيو لاگ اِن صفحي تي ۽ سائن اپ وهڪري تي پڻ ڏيکاريو ويندو.

![لاگ اِن ۽ رجسٽريشن سيٽنگون](/img/config/settings-login-registration.png)

_لاگ اِن صفحي تي ٻولي چونڊيندڙ_

![رجسٽريشن صفحو](/img/config/registration-page.png)

_سائن اپ وهڪري تي ٻولي چونڊيندڙ_

## URL ذريعي ٻولي سيٽ ڪرڻ {#setting-the-language-via-the-url}

توهان URL ڏانهن locale پيرا ميٽر موڪلي ڪنهن صفحي جي ٻولي پڻ تبديل ڪري سگهو ٿا. جيڪڏهن توهان مثال طور اطالوي ۾ رجسٽر صفحي ڏانهن ريڊائريڪٽ ڪرڻ چاهيو ٿا، ته URL ۾ ?locale=it_IT شامل ڪريو.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

هن جي ڪم ڪرڻ لاءِ ٻولي توهان جي WordPress نيٽ ورڪ تي انسٽال ٿيل هجڻ گهرجي.
