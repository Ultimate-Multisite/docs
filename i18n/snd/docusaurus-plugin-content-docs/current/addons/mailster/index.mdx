# Mailster {#mailster}

Ultimate Multisite Mailster انٽيگريشن اضافي سان پنهنجي multisite نيٽ ورڪ کي طاقتور اي ميل فهرست ٺاهڻ واري مشين ۾ تبديل ڪريو. جڏهن نوان گراهڪ سائن اپ ڪن يا خريداريون ڪن، تڏهن انهن کي پاڻمرادو پنهنجي Mailster اي ميل فهرستن ۾ شامل ڪريو، جنهن سان توهان جي ميمبرشپ پليٽفارم ۽ اي ميل مارڪيٽنگ حڪمت عملي جي وچ ۾ بي رڪاوٽ ڳانڍاپو ٺهي ٿو.

## اهم خاصيتون {#key-features}

**پاڻمرادو سبسڪرائيبر هم وقت سازي**
* جڏهن گراهڪ سائن اپ ڪن، تڏهن انهن کي پاڻمرادو Mailster فهرستن ۾ شامل ڪريو
* سائن اپ تي فوري هم وقت سازي چونڊيو يا ادائگي جي تصديق جو انتظار ڪريو
* هٿرادو مداخلت کان سواءِ پنهنجي اي ميل فهرستن کي تازو ۽ اپڊيٽ رکو

**لچڪدار فهرست انتظام**
* سڀني نون گراهڪن لاءِ عالمي ڊفالٽ فهرستون مقرر ڪريو
* نشاني بڻايل مهمن لاءِ پروڊڪٽ-مخصوص فهرستن سان ڊفالٽ کي اوور رائيڊ ڪريو
* Mailster جي واقف انٽرفيس سان فهرست تفويضون آساني سان سنڀاليو

**GDPR-مطابق Opt-In اختيار**
* پاڻمرادو سبسڪرپشن يا چيڪ باڪس تي ٻڌل رضامندي مان چونڊيو
* تصديق ٿيل سبسڪرپشنز لاءِ ڊبل opt-in فعال ڪريو
* پنهنجي تعميل جي گهرجن جي بنياد تي opt-in چيڪ باڪس ڏيکاريو/لڪايو

**هوشيار ڊيٽا ميپنگ**
* گراهڪن جا نالا ۽ رابطا معلومات پاڻمرادو هم وقت ڪريو
* بلنگ پتا سبسڪرائيبر فيلڊن سان ميپ ڪريو
* نشاني بڻايل مهمن لاءِ سائن اپ جون تاريخون ٽريڪ ڪريو

**پروڊڪٽ-سطح ڪنٽرول**
* مختلف پروڊڪٽس کي مختلف فهرستون تفويض ڪريو
* گراهڪن جي خريد ڪيل شين جي بنياد تي نشاني بڻايل مهمون ٺاهيو
* ضرورت پوڻ تي مخصوص پروڊڪٽس لاءِ Mailster غير فعال ڪريو

**پيشاور تجربو**
* فهرست چونڊ لاءِ اصلي Mailster UI انٽيگريشن
* مسئلا حل ڪرڻ ۽ تصديق لاءِ تفصيلي لاگنگ
* موجود سبسڪرائيبرن جون بي رڪاوٽ اپڊيٽون

## هنن لاءِ بهترين {#perfect-for}

* ميمبرشپ سائيٽ مالڪ جيڪي اي ميل فهرستون ٺاهي رهيا آهن
* SaaS پليٽفارم جيڪي نون گراهڪن جي پرورش ڪرڻ چاهين ٿا
* ڪورس ٺاهيندڙ جيڪي داخلا جي بنياد تي شاگردن کي ورهائي رهيا آهن
* هر اهو ماڻهو جيڪو Mailster سان Ultimate Multisite هلائي رهيو آهي

## اهو ڪيئن ڪم ڪري ٿو {#how-it-works}

1. پنهنجي Ultimate Multisite نيٽ ورڪ تي اضافو انسٽال ۽ فعال ڪريو
2. Ultimate Multisite سيٽنگن ۾ عالمي ڊفالٽ فهرستون ترتيب ڏيو
3. اختياري طور پروڊڪٽ-مخصوص فهرستن سان اوور رائيڊ ڪريو
4. پنهنجو opt-in موڊ ۽ سبسڪرپشن ٽائمنگ چونڊيو
5. پنهنجي Mailster فهرستن کي پاڻمرادو وڌندي ڏسو!

## انسٽاليشن {#installation}

1. پلگ ان فائلون پنهنجي Ultimate Multisite اضافن واري ڊائريڪٽري ۾ اپلوڊ ڪريو
2. يقيني بڻايو ته Mailster توهان جي مکيه سائيٽ تي انسٽال ۽ فعال آهي
3. Ultimate Multisite اضافن واري صفحي ذريعي اضافو فعال ڪريو
4. ترتيب ڏيڻ لاءِ سيٽنگون > Mailster انٽيگريشن ڏانهن وڃو
5. پنهنجون ڊفالٽ فهرستون چونڊيو ۽ opt-in ترجيحون ترتيب ڏيو

## اڪثر پڇيا ويندڙ سوال {#frequently-asked-questions}

## ڇا هي اضافو Ultimate Multisite کان سواءِ ڪم ڪري ٿو؟ {#does-this-addon-work-without-ultimate-multisite}

نه، هن اضافي کي ڪم ڪرڻ لاءِ Ultimate Multisite گهربل آهي. اهو خاص طور تي Mailster کي Ultimate Multisite جي گراهڪ ۽ پروڊڪٽ سسٽم سان ضم ڪرڻ لاءِ ٺاهيو ويو آهي.

## ڇا مون کي سڀني سائيٽن تي Mailster انسٽال ڪرڻ جي ضرورت آهي؟ {#do-i-need-mailster-installed-on-all-sites}

نه، Mailster کي صرف توهان جي مکيه نيٽ ورڪ سائيٽ تي انسٽال ۽ فعال ٿيڻ جي ضرورت آهي. اضافو سڀني سب سائيٽن مان پاڻمرادو ان سان ڳنڍجي ويندو.

## ڇا مان مختلف پروڊڪٽس کي مختلف فهرستون تفويض ڪري سگهان ٿو؟ {#can-i-assign-different-lists-to-different-products}

ها! توهان عالمي ڊفالٽ فهرستون مقرر ڪري سگهو ٿا، پوءِ انهن کي هر پروڊڪٽ جي بنياد تي اوور رائيڊ ڪري سگهو ٿا. اهو توهان کي گراهڪن جي خريد ڪيل شين جي بنياد تي انتهائي نشاني بڻايل اي ميل مهمون ٺاهڻ جي اجازت ڏئي ٿو.

## جيڪڏهن ڪو گراهڪ اڳ ئي Mailster ۾ موجود هجي ته ڇا ٿيندو؟ {#what-happens-if-a-customer-already-exists-in-mailster}

ڊفالٽ طور، اضافو موجود سبسڪرائيبرن کي اپڊيٽ ڪندو ۽ انهن کي نون فهرستن ۾ شامل ڪندو. جيڪڏهن توهان موجود سبسڪرائيبرن کي ڇڏي ڏيڻ پسند ڪريو ٿا ته توهان سيٽنگن ۾ هن کي غير فعال ڪري سگهو ٿا.

## ڇا اهو GDPR مطابق آهي؟ {#is-this-gdpr-compliant}

ها! اضافي ۾ چيڪ باڪس تي ٻڌل opt-in موڊ شامل آهي جتي گراهڪن کي اي ميلون حاصل ڪرڻ لاءِ واضح رضامندي ڏيڻ لازمي آهي. مناسب رازداري پاليسين سان گڏ هي GDPR گهرجون پوريون ڪري ٿو.

## ڇا مان ڊبل opt-in استعمال ڪري سگهان ٿو؟ {#can-i-use-double-opt-in}

بلڪل! ڊبل opt-in سيٽنگ فعال ڪريو ۽ Mailster سبسڪرپشنز فعال ڪرڻ کان اڳ تصديقي اي ميلون موڪليندو.
