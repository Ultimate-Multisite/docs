# عمل: switch_network

تڏهن هلندو آهي جڏهن موجوده نيٽ ورڪ جو تناظر مٽايو وڃي.

## پيراميٽر {#parameters}

| نالو | قسم | وضاحت |
|------|------|-------------|
| $new_network_id | `int` | ان نيٽ ورڪ جي سڃاڻپ جنهن ڏانهن مٽايو پيو وڃي. |
| $old_network_id | `int` | اڳ ۾ موجوده نيٽ ورڪ جي سڃاڻپ. |
