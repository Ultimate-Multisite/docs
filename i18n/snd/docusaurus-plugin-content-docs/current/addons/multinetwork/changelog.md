### 1.3.0
* نئون: نيٽ ورڪ ٽيمپليٽ پري ويوئر — خريد ڪرڻ کان اڳ لائيو پينل سان نيٽ ورڪ ٽيمپليٽس برائوز ۽ پري ويو ڪريو.
* نئون: چيڪ آئوٽ تي نيٽ ورڪ ٽيمپليٽ چونڊيندڙ — گراهڪ سائن اپ چيڪ آئوٽ وهڪري دوران نيٽ ورڪ ٽيمپليٽ چونڊي سگهن ٿا.
* نئون: ڪراس-نيٽ ورڪ ميجڪ لنڪ SSO — گراهڪ ميجڪ لنڪ تصديق ذريعي سب-نيٽ ورڪس تائين بنا رنڊڪ رسائي حاصل ڪري سگهن ٿا.
* نئون: نيٽ ورڪ ميڊيا ڪلوننگ — ٽيمپليٽ مان نيٽ ورڪس ڪلون ڪرڻ وقت سائيٽ ميڊيا صحيح نموني نقل ٿئي ٿي.
* درست: نيٽ ورڪ ڪلوننگ هاڻي multi-tenancy addon کان سواءِ ڪم ڪري ٿي؛ ڪلون کان پوءِ سپر ايڊمن اختيار ٻيهر ڏنا وڃن ٿا.
* درست: نيٽ ورڪ ٺاهڻ هاڻي WooCommerce ۽ سڀني ادائيگي گيٽ ويز سان مطابقت رکي ٿو.
* درست: fallback domain ٺاهڻ ۾ هاڻي پورٽ نمبر شامل آهي.
* درست: سڀني themes تي صحيح رينڊرنگ لاءِ ٽيمپليٽ چونڊيندڙ CSS کي enqueued stylesheet ڏانهن منتقل ڪيو ويو.
* درست: سائيٽ query scope ۾ غير متعين `network_id` key کان بچاءُ.
* درست: context switching کان اڳ نيٽ ورڪ permission check `wpdb` ٽيبل property errors کي روڪي ٿو.
