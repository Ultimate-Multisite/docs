# Material WP

WordPress جو انتظامي انٽرفيس سالن کان ساڳيو رهيو آهي. پر ڇا توهان ڪڏهن سوچيو آهي ته انتظامي انٽرفيس جو Material Design نسخو ڪيئن لڳندو؟ 

Material WP هڪ انتظامي theme آهي جيڪو توهان کي اهو ڏيک ۽ احساس ڏيڻ لاءِ ٺاهيو ويو آهي، جنهن سان توهان پنهنجي گراهڪن کي وڌيڪ حسب ضرورت Dashboard ڏئي سگهو ٿا.

## خاصيتون {#features}

- Custom CSS جي سهائتا
- انداز سازي جون آسان سيٽنگون
- مينيو جي ٻيهر ترتيب
- آسان درآمد/برآمد

## دستاويز {#documentation}

**Material WP سان پنهنجو انتظامي پينل حسب ضرورت بڻائڻ**

## تنصيب {#installation}

1. 'material-wp' کي '/wp-content/plugins/' ڊائريڪٽري ۾ اپلوڊ ڪريو
2. WordPress ۾ 'پلگ انز' مينيو ذريعي plugin کي فعال ڪريو
