# حمايتي ٽڪيٽون

**حمايتي ٽڪيٽون** Ultimate Multisite لاءِ هڪ طاقتور addon آهي، جيڪو توهان جي WordPress نيٽورڪ ۾ هڪ جامع حمايتي ٽڪيٽ نظام شامل ڪري ٿو. SaaS پليٽفارمن، WaaS (Website as a Service) مهيا ڪندڙن، ۽ multisite نيٽورڪن لاءِ ڀرپور آهي، جن کي نيٽورڪ ۽ الڳ سائيٽ ٻنهي سطحن تي گراهڪ حمايت فراهم ڪرڻي هجي.

## حمايتي ٽڪيٽون ڇو چونڊيو؟ {#why-choose-support-tickets}

* **گهڻ-سطحي حمايت**: نيٽورڪ منتظم نيٽورڪ-وسيع ٽڪيٽون سنڀالين ٿا، جڏهن ته سائيٽ مالڪ پنهنجي گراهڪ حمايت منظم ڪن ٿا
* **پيشاورانه ڪم جو وهڪرو**: ترجيحن، حالتن، ۽ تفويضن سان مڪمل ٽڪيٽ زندگي چڪر جو انتظام
* **گراهڪ تجربو**: خوبصورت، responsive انٽرفيس، جيڪو گراهڪن لاءِ مدد حاصل ڪرڻ آسان بڻائي ٿو
* **محفوظ ۽ ڀروسي جوڳو**: WordPress سيڪيورٽي جي بهترين عملن سان ٺهيل ۽ بهتر ڪيل ڊيٽابيس ڪارڪردگي
* **بي رڪاوٽ انضمام**: Ultimate Multisite جي گراهڪ ۽ سائيٽ انتظام سان مڪمل نموني ڪم ڪري ٿو

## اهم خاصيتون {#key-features}

**مڪمل ٽڪيٽ انتظام**

* آسان حوالي لاءِ منفرد ٽڪيٽ نمبرنگ نظام
* ترتيب ڏئي سگهجندڙ ترجيحي سطحون (گهٽ، عام، وڌيڪ، نازڪ)
* جامع حالت وارو ڪم جو وهڪرو (کليل، جاري، انتظار ۾، حل ٿيل، بند)
* ٽڪيٽ درجابندي (عام، فني، بلنگ، خاصيت جي درخواست، بگ رپورٽ)
* اختياري خودڪار تفويض سان حمايتي ايجنٽن کي ٽڪيٽون تفويض ڪريو

**ڌاڳي وار ڳالهيون**

* ٽڪيٽ جوابن لاءِ مڪمل گفتگو جي threading
* عملي ۽ گراهڪ جوابن لاءِ الڳ انداز
* عملي جا خانگي نوٽ، جيڪي گراهڪ نٿا ڏسي سگهن
* حقيقي وقت AJAX-طاقتور انٽرفيس
* فارميٽنگ اختيارن سان جديد متن ايڊيٽر

**فائل منسلڪات**

* drag-and-drop حمايت سان محفوظ فائل اپلوڊ
* ترتيب ڏئي سگهجندڙ فائل قسم ۽ سائيز جون پابنديون
* نقصانڪار فائل سڃاڻپ سان اڳڀري سيڪيورٽي
* رسائي ڪنٽرولن سان محفوظ ڊائونلوڊ

**هوشيار اطلاعون**

* نون ٽڪيٽن، جوابن، ۽ حالت تبديلين لاءِ خودڪار اي ميل خبرداريون
* ترتيب ڏئي سگهجندڙ اي ميل ٽيمپليٽ
* عملي ۽ گراهڪن لاءِ ڪردار-بنياد اطلاعون
* ذاتي پيغامن لاءِ dynamic ٽيمپليٽ variables

**طاقتور منتظم انٽرفيس**

* ترتيب ڏيڻ ۽ فلٽرنگ سان اڳڀري ٽڪيٽ فهرست
* حالت، ترجيح، ۽ تفويضن لاءِ تڪڙو ايڊٽ
* موثر انتظام لاءِ bulk عمل
* ڪارڪردگي metrics سان انگ اکرن وارو Dashboard
* جامع سيٽنگون پينل

**خوبصورت گراهڪ پورٽل**

* سڀني ڊوائيسن تي صاف، responsive انٽرفيس
* موبائل-اول ڊيزائن
* اڳڀري ڳولا ۽ فلٽرنگ
* فارميٽنگ toolbar سان جديد ايڊيٽر
* استعمال ۾ آسان ٽڪيٽ جمع ڪرائڻ جا فارم

**لچڪدار انضمام**

* shortcodes يا Ultimate Multisite UI عنصر استعمال ڪريو
* پنهنجي سائيٽ تي ڪٿي به ٽڪيٽ فارم ۽ فهرستون ڏيکاريو
* multisite نيٽورڪ ڍانچي سان بي رڪاوٽ ڪم ڪري ٿو
* hooks ۽ filters سان وڌائي سگهجندڙ

## هنن لاءِ ڀرپور {#perfect-for}

* گراهڪ حمايت جي ضرورت رکندڙ SaaS پليٽفارم
* ڪيترين ئي ڪلائنٽ سائيٽن کي منظم ڪندڙ WaaS مهيا ڪندڙ
* سطح وار حمايت سان membership سائيٽون
* ڪيترن ئي brands کي منظم ڪندڙ نيٽورڪ منتظم
* گراهڪ سروس فراهم ڪندڙ سائيٽ مالڪ

## انسٽاليشن {#installation}

1. يقيني بڻايو ته Ultimate Multisite plugin توهان جي WordPress نيٽورڪ تي انسٽال ۽ فعال ٿيل آهي
2. addon فائلون پنهنجي `/wp-content/plugins/` ڊائريڪٽري ۾ اپلوڊ ڪريو
3. WordPress ۾ 'Plugins' مينيو ذريعي plugin کي نيٽورڪ فعال ڪريو
4. پنهنجي سيٽنگون ترتيب ڏيڻ لاءِ Network Admin > Support Tickets ڏانهن وڃو

## اڪثر پڇيا ويندڙ سوال {#frequently-asked-questions}

## ڇا هن لاءِ Ultimate Multisite گهربل آهي؟ {#does-this-require-ultimate-multisite}

ها، هن addon لاءِ Ultimate Multisite جو انسٽال ۽ فعال هجڻ ضروري آهي. اهو Ultimate Multisite جي گراهڪ ۽ سائيٽ انتظام خاصيتن سان گهري نموني ضم ٿئي ٿو.

## ڇا سائيٽ مالڪ پنهنجي گراهڪ حمايت سنڀالي سگهن ٿا؟ {#can-site-owners-handle-their-own-customer-support}

بلڪل! سائيٽ مالڪ ٽڪيٽ نظام استعمال ڪري پنهنجي گراهڪن کي حمايت فراهم ڪري سگهن ٿا، جڏهن ته نيٽورڪ منتظم نيٽورڪ-وسيع حمايتي ٽڪيٽون سنڀالي سگهن ٿا.

## گراهڪ ٽڪيٽون ڪيئن جمع ڪرائين ٿا؟ {#how-do-customers-submit-tickets}

توهان shortcodes يا Ultimate Multisite UI عنصرن کي استعمال ڪندي پنهنجي سائيٽ تي ڪٿي به ٽڪيٽ جمع ڪرائڻ جا فارم شامل ڪري سگهو ٿا. گراهڪ صرف فارم ڀريندا ۽ پنهنجي درخواست جمع ڪرائيندا.

## ڇا مان ٽڪيٽ حالتون ۽ ترجيحون ترتيب ڏئي سگهان ٿو؟ {#can-i-customize-the-ticket-statuses-and-priorities}

ها! توهان پنهنجي حمايتي ڪم جي وهڪري سان ملائڻ لاءِ نيٽورڪ منتظم سيٽنگن ۾ ترجيحي سطحون، حالتون، ۽ ٽڪيٽ قسم مڪمل طور ترتيب ڏئي سگهو ٿا.

## ڇا اهو فائل منسلڪات جي حمايت ڪري ٿو؟ {#does-it-support-file-attachments}

ها، گراهڪ ۽ عملي ٻنهي ٽڪيٽن ۽ جوابن سان فائلون منسلڪ ڪري سگهن ٿا. سيڪيورٽي لاءِ اجازت ڏنل فائل قسمن ۽ سائيز حدن تي توهان جو مڪمل ڪنٽرول آهي.

## اطلاعن کي ڪيئن سنڀاليو وڃي ٿو؟ {#how-are-notifications-handled}

نظام نون ٽڪيٽن، جوابن، ۽ حالت تبديلين لاءِ پاڻمرادو اي ميل اطلاعون موڪلي ٿو. توهان سيٽنگن ۾ سڀ اي ميل ٽيمپليٽ ترتيب ڏئي سگهو ٿا.

## ڇا مان مخصوص حمايتي ايجنٽن کي ٽڪيٽون تفويض ڪري سگهان ٿو؟ {#can-i-assign-tickets-to-specific-support-agents}

ها، توهان هٿ سان ٽڪيٽون ايجنٽن کي تفويض ڪري سگهو ٿا يا ٽڪيٽون پاڻمرادو ورهائڻ لاءِ خودڪار تفويض فعال ڪري سگهو ٿا.

## ڇا عملي لاءِ خانگي نوٽ آهن؟ {#are-there-private-notes-for-staff}

ها، عملي جا ميمبر خانگي جواب شامل ڪري سگهن ٿا، جيڪي صرف ٻين عملي ميمبرن کي نظر ايندا، گراهڪن کي نه.

## ڇا اهو موبائل-دوست آهي؟ {#is-it-mobile-friendly}

بلڪل! سڄو انٽرفيس موبائل-اول طريقي سان ٺهيل آهي ۽ سڀني ڊوائيسن تي خوبصورتي سان ڪم ڪري ٿو.

## ڇا مان shortcodes استعمال ڪري ٽڪيٽ فهرستون ۽ فارم ڏيکاري سگهان ٿو؟ {#can-i-display-ticket-lists-and-forms-using-shortcodes}

ها! موجود shortcodes:
* `[wu_support_tickets]` - گراهڪ ٽڪيٽ فهرست ڏيکاريو
* `[wu_support_ticket_form]` - نئين ٽڪيٽ جمع ڪرائڻ جو فارم ڏيکاريو
* `[wu_support_ticket_view ticket_id="123"]` - مخصوص ٽڪيٽ ڏيکاريو

## ڪهڙا فائل قسم اپلوڊ ڪري سگهجن ٿا؟ {#what-file-types-can-be-uploaded}

توهان سيٽنگن ۾ اجازت ڏنل فائل قسم ترتيب ڏئي سگهو ٿا. نظام ۾ نقصانڪار فائلن کان بچاءَ لاءِ اڳڀري سيڪيورٽي scanning شامل آهي.

## ڇا مان ٽڪيٽ ڊيٽا export ڪري سگهان ٿو؟ {#can-i-export-ticket-data}

ها، addon ۾ ٽڪيٽ ڊيٽا ۽ انگ اکرن لاءِ export صلاحيتون شامل آهن.
