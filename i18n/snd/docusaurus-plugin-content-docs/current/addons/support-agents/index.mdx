# سهائتا ايجنٽس

سڄي نيٽ ورڪ کي پاڻ سنڀالڻ سولو ڪم ناهي. جيئن توهان جو نيٽ ورڪ وڌڻ شروع ٿيندو، توهان کي سهائتا جي وڌندڙ گهرج نظر اچڻ لڳندي. اهو سٺو خيال هوندو ته اهڙا ماڻهو شامل ڪيا وڃن جيڪي توهان جو ڪم جو بار گهٽ ڪرڻ ۾ مدد ڪن. بهرحال، مڪمل رسائي/اجازت ڏيڻ خطري وارو آهي ۽ توهان جي نيٽ ورڪ کي خطري ۾ وجهي سگهي ٿو. اتي هي سهائتا ايجنٽ add-on ڪم اچي ٿو.

هي add-on توهان کي WordPress استعمال ڪندڙ جو هڪ نئون قسم ٺاهڻ جي اجازت ڏئي ٿو، جنهن کي سهائتا ايجنٽ سڏيو وڃي ٿو، ۽ توهان کي ان استعمال ڪندڙ وٽ موجود اجازتن تي مڪمل ضابطو ڏئي ٿو. اجازت ڏيڻ ايترو ئي آسان آهي جيترو هڪ ٽوگل بٽڻ آن ڪرڻ، ۽ جيڪڏهن توهان ان کي هٽائڻ جو فيصلو ڪريو ته ان کي آف ڪرڻ.

## خاصيتون

- WordPress ۽ Ultimate Multisite اجازتن جي شق وار فهرست
- آساني سان رسائي فعال ۽ غير فعال ڪريو
- ايجنٽ Accountن جو لامحدود انگ ٺاهيو

## انسٽاليشن

1. 'ultimate-multisite-support-agents' کي '/wp-content/plugins/' ڊائريڪٽري ۾ اپلوڊ ڪريو
2. WordPress ۾ 'پلگ انز' مينيو ذريعي پلگ ان فعال ڪريو
3. پلگ ان کي سيٽ اپ ڪرڻ لاءِ قدم بہ قدم مددگار جي پيروي ڪريو
