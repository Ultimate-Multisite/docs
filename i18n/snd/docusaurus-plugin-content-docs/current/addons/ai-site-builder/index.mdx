# AI سائيٽ بلڊر

ultimate-multisite-ai-site-builder

## گهرجون

- WordPress 5.3 يا وڌيڪ
- PHP 7.4 يا وڌيڪ
- Multisite Ultimate پلگ اِن (فعال)

## انسٽاليشن

1. addon فائلون پنهنجي `/wp-content/plugins/` ڊائريڪٽري ۾ اپلوڊ ڪريو
2. WordPress ۾ 'Plugins' مينيو ذريعي پلگ اِن فعال ڪريو
3. نيٽ ورڪ ايڊمن ۾ addon سيٽنگون ترتيب ڏيو

## ڊولپمينٽ

### سيٽ اپ

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### منصوبي جي جوڙجڪ

```
ultimate-multisite-ai-site-builder/
├── inc/                          # PHP ڪلاسون ۽ شامل فائلون
│   ├── class-multisite-ultimate-updater.php  # اپڊيٽ هينڊلر
│   └── class-ai_site_builder-main.php             # مکيه ڪارڪردگي
├── tests/                        # يونٽ ٽيسٽ
│   ├── bootstrap.php            # ٽيسٽ بوٽ اسٽريپ
│   ├── class-*-base.php         # بنيادي ٽيسٽ ڪلاس
│   └── class-*-test.php         # ٽيسٽ ڪيس
├── views/                       # ٽيمپليٽ فائلون
├── assets/                      # جامد اثاثا
│   ├── css/                     # اسٽائل شيٽون
│   ├── js/                      # JavaScript فائلون
│   └── images/                  # تصويرون
├── lang/                        # ٻولي فائلون
├── composer.json                # PHP انحصار
├── package.json                 # Node.js انحصار
├── phpunit.xml.dist             # PHPUnit ترتيب
├── .phpcs.xml.dist              # ڪوڊ معيارن جي ترتيب
└── rector.php                   # ڪوڊ جديد ڪرڻ جا قاعدا
```

### ٽيسٽنگ

addon ۾ هڪ جامع ٽيسٽنگ فريم ورڪ شامل آهي:

- **يونٽ ٽيسٽ**: WordPress ٽيسٽ سوٽ سان PHPUnit تي ٻڌل ٽيسٽ
- **بنيادي ٽيسٽ ڪلاس**: عام ٽيسٽ حالتن لاءِ مددگار طريقا فراهم ڪري ٿو
- **ٽيسٽ ڪوريج**: ڪوڊ ڪوريج ٽريڪ ڪرڻ لاءِ ترتيب ڏنل
- **ملٽي سائيٽ ٽيسٽنگ**: ٽيسٽ ملٽي سائيٽ ماحول ۾ هلن ٿا

### ڪوڊ معيار

- **WordPress ڪوڊنگ معيار**: PHPCS ذريعي لاڳو ڪيل
- **PHP 7.4+ مطابقت**: جديد PHP خاصيتون سپورٽ ٿيل
- **جامد تجزيو**: قسم چيڪنگ لاءِ PHPStan انٽيگريشن
- **ڪوڊ جديد ڪرڻ**: PHP اپگريڊن لاءِ Rector قاعدا

### بلڊ سسٽم

- **اثاثن جي پروسيسنگ**: CSS/JS فائلن لاءِ منيفڪيشن
- **ترجمو**: POT فائل ٺاهڻ
- **پيڪيجنگ**: ورڇ لاءِ آرڪائيو ٺاهڻ
- **ڊولپمينٽ موڊ**: ڊيبگنگ لاءِ اڻ منيفاءِ ٿيل اثاثا

## ترتيب

addon مختلف ترتيب اختيارن کي سپورٽ ڪري ٿو، جيڪي نيٽ ورڪ ايڊمن انٽرفيس ذريعي پهچ لائق آهن.

## Hooks ۽ Filters

### عمل

- `ultimate-multisite-ai-site-builder_init` - addon جي شروعات کان پوءِ هلندو آهي
- `ultimate-multisite-ai-site-builder_loaded` - سڀ addon فائلون لوڊ ٿيڻ کان پوءِ هلندو آهي

### Filters

- `ultimate-multisite-ai-site-builder_settings` - addon سيٽنگون تبديل ڪريو
- `ultimate-multisite-ai-site-builder_enabled` - addon جي فعال/غير فعال حالت کي اوور رائيڊ ڪريو
