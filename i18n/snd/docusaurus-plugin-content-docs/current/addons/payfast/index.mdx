# PayFast {#payfast}

Ultimate Multisite: Payfast ادائيگي گيٽ وي

Payfast - ڏکڻ آفريڪا ۾ هڪ تيز ۽ محفوظ آن لائن ادائيگين جو گيٽ وي - کي Ultimate Multisite جي ادائيگي واري اختيار طور شامل ڪري ٿو.

## انسٽاليشن {#installation}

1. 'ultimate-multisite-payfast' کي '/wp-content/plugins/' ڊاريڪٽري ۾ اپلوڊ ڪريو
2. WordPress ۾ 'Plugins' مينيو ذريعي پلگ اِن کي فعال ڪريو
3. پلگ اِن کي سيٽ اپ ڪرڻ لاءِ قدم بہ قدم Wizard تي عمل ڪريو

## استعمال جي رهنمائي {#usage-guide}

PayFast ڏکڻ آفريڪا ۾ ڪاروبارن لاءِ هڪ اڳواڻ آن لائن ادائيگي پروسيسر آهي. اهو آن لائن واپارين کي ٽرانزيڪشنز کي بي رڪاوٽ ۽ محفوظ بڻائي وڌڻ ۾ مدد ڏيڻ لاءِ وقف آهي. Ultimate Multisite استعمال ڪندڙ جي درخواست موجب، اسان هڪ ايڊ آن ٺاهڻ جو فيصلو ڪيو جيڪو PayFast سان انٽيگريشن کي ممڪن بڻائيندو. هي مضمون ڏيکاريندو ته توهان جي Ultimate Multisite نيٽ ورڪ ۾ ادائيگي وصول ڪرڻ لاءِ PayFast کي توهان جي ادائيگي گيٽ وي طور ڪيئن سيٽ اپ ڪجي.

پهريون قدم ايڊ آن انسٽال ڪرڻ آهي.

# Payfast ايڊ آن ڪيئن انسٽال ڪجي: {#how-to-install-the-payfast-add-on}

ايڊ آنز صفحي تي وڃو ۽ Ultimate Multisite: Payfast Gateway ايڊ آن چونڊيو.

![ايڊ آنز صفحو](/img/admin/addons-page.png)

ايڊ آن ونڊو کلي ايندي. Install Now تي ڪلڪ ڪريو.

![انسٽال اختيار سان ايڊ آنز صفحو](/img/admin/addons-page.png)

انسٽاليشن مڪمل ٿيڻ کان پوءِ، توهان کي پلگ اِنز صفحي ڏانهن موڪليو ويندو. هتي، صرف Network Activate تي ڪلڪ ڪريو ۽ _Payfast_ ايڊ آن توهان جي نيٽ ورڪ تي فعال ٿي ويندو.

# PayFast سيٽ اپ ڪرڻ {#setting-up-payfast}

ايڊ آن انسٽال ڪرڻ کان پوءِ، هاڻي توهان Ultimate Multisite سيٽنگز ڏانهن وڃي PayFast کي ترتيب ڏيڻ شروع ڪري سگهو ٿا.

**Ultimate Multisite settings** صفحي هيٺ، توهان "Payments" ٽيب ڳولي سگهو ٿا. هن ٽيب هيٺ، توهان کي ادائيگي گيٽ وي کي فعال يا غير فعال ڪرڻ لاءِ ٽوگل بٽڻ سان گڏ "**Payfast** " ڏسڻ گهرجي.

![ادائيگي گيٽ وي سيٽنگز](/img/config/settings-payment-gateways.png)اڳتي وڌو ۽ ان کي فعال ڪرڻ لاءِ ٽوگل بٽڻ تي ڪلڪ ڪريو. پوءِ اهو باڪس کي وڌائيندو ته جيئن مختلف فيلڊز ڏيکاري جن کي توهان کي ڀرڻو آهي، جهڙوڪ **Mechant ID, Merchant Key ۽ PassPhrase.** توهان هي معلومات پنهنجي **Payfast** Account مان حاصل ڪري سگهو ٿا

<!-- اسڪرين شاٽ دستياب ناهي: PayFast جون وڌايل سيٽنگز Merchant ID, Merchant Key، ۽ PassPhrase فيلڊز ڏيکاري رهيون آهن -->

ايندڙ قدم اها معلومات حاصل ڪرڻ لاءِ توهان جي PayFast Account ۾ لاگ اِن ڪرڻ آهي.

لاگ اِن ڪرڻ تي، توهان پنهنجي **Payfast** Account Dashboard مان فوري طور اهي سندون ڏسي سگهو ٿا جيڪي توهان کي گهرجن.

<!-- اسڪرين شاٽ دستياب ناهي: PayFast Account Dashboard Merchant ID ۽ Merchant Key سندون ڏيکاري رهيو آهي -->توهان سادي طرح سند ڪاپي ۽ پيسٽ ڪري سگهو ٿا ۽ پنهنجي Ultimate Multisite سيٽنگز ۾ تفصيل ڀري سگهو ٿا. پوءِ تبديلين کي محفوظ ڪرڻ لاءِ **"سيٽنگز محفوظ ڪريو** " بٽڻ تي ڪلڪ ڪريو.

<!-- اسڪرين شاٽ دستياب ناهي: Ultimate Multisite PayFast سيٽنگز سندن سان ڀريل ۽ محفوظ ڪرڻ وارو بٽڻ -->

بس ايترو، توهان جي PayFast ادائيگي انٽيگريشن مڪمل آهي!
