# Content Sync تبديلي لاگ {#content-sync-changelog}

### ورزن 1.1.0 - 2025-12-31 تي جاري ڪيو ويو {#version-110---released-on-2025-12-31}
* نئون: سڀني پوسٽ قسمن کي sync ڪرڻ لاءِ سهائتا (پوسٽون، صفحا، custom post types)
* نئون: مواد چونڊڻ واري قدم ۾ پوسٽ قسم فلٽرنگ
* نئون: ماخذ سائيٽ چونڊڻ وارو قدم - رڳو مکيه سائيٽ نه، پر ڪنهن به سائيٽ مان sync ڪريو
* نئون: template سائيٽن لاءِ ٻار سائيٽ اڳواٽ چونڊ
* نئون: سڀني پوسٽ قسمن لاءِ post_content مان ميڊيا ڪڍڻ
* نئون: Taxonomy syncing (ڪيٽيگريون، tags، custom taxonomies)
* تبديل ٿيل: "Elementor Template Sync" مان "Content Sync" ۾ نالو تبديل ڪيو ويو
* تبديل ٿيل: "Select Templates" قدم جو نالو "Select Data" رکيو ويو
* بهتر ٿيل: Elementor-خاص ڪوڊ هاڻي رڳو elementor_library پوسٽ قسم لاءِ هلندو آهي
