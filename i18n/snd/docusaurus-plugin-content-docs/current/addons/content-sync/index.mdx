# مواد هم وقت سازي

Multisite Ultimate: مواد هم وقت سازي

Multisite Ultimate لاءِ هي ايڊآن نيٽ ورڪ منتظمين کي اجازت ڏئي ٿو ته نيٽ ورڪ ۾ ڪنهن به سائيٽ مان مواد ٻين سائيٽن ڏانهن هم وقت ساز ڪن. اهو سڀني پوسٽ قسمن کي سپورٽ ڪري ٿو، جن ۾ پوسٽون، صفحا، ڪسٽم پوسٽ قسم، ۽ Elementor ٽيمپليٽ شامل آهن. ايڊآن ۾ مواد ۽ هدف سائيٽن جي چونڊ لاءِ گهڻ-مرحلي وزرڊ انٽرفيس شامل آهي، بيچ پروسيسنگ ۽ حقيقي وقت جي اڳڀرائي واري موٽ سان.

## خاصيتون {#features}

* مواد جي چونڊ: ٿمبنيلز ۽ پوسٽ قسم فلٽرنگ سان بصري ڪارڊ گرڊ
* سڀ پوسٽ قسم: پوسٽون، صفحا، پراڊڪٽس، Elementor ٽيمپليٽس، ۽ ڪو به ڪسٽم پوسٽ قسم هم وقت ساز ڪريو
* سائيٽ جي چونڊ: چونڊيو ته مواد ڪهڙين سائيٽن ڏانهن هم وقت ساز ڪرڻو آهي، ماخذ سائيٽ جي خارج ڪرڻ ۽ ٻار سائيٽ جي اڳواٽ چونڊ سان
* بيچ پروسيسنگ: اڳڀرائي اپڊيٽس سان هڪ وقت ۾ 10 سائيٽن ڏانهن هم وقت ساز ڪري ٿو
* ميڊيا هم وقت سازي: پوسٽ مواد مان ميڊيا فائلون (تصويرون، وڊيوز) پاڻمرادو نقل ڪري هدف سائيٽن ڏانهن ٻيهر ميپ ڪري ٿو
* ٽيڪسانامي هم وقت سازي: ڪيٽيگريون، ٽيگ، ۽ ڪسٽم ٽيڪساناميون برقرار رکي ٿو
* Elementor سپورٽ: CSS ٻيهر ٺاهڻ ۽ ٽيمپليٽ ID ٻيهر ميپنگ سان Elementor ٽيمپليٽس لاءِ مڪمل سپورٽ
* تڪرار جو حل: موجوده مواد کي بدلائڻ يا ان کي ڇڏي ڏيڻ جو اختيار
* ڪيش انتظام: هم وقت سازي کان پوءِ هدف سائيٽن تي Elementor ڪيش پاڻمرادو صاف ڪري ٿو

## گهرجون {#requirements}

* WordPress 5.3 يا وڌيڪ
* PHP 7.4 يا وڌيڪ
* Multisite Ultimate plugin (فعال)
* Elementor plugin (اختياري، Elementor ٽيمپليٽ هم وقت سازي لاءِ گهربل)

## انسٽاليشن {#installation}

1. 'ultimate-multisite-content-sync' کي '/wp-content/plugins/' ڊاريڪٽري ۾ اپلوڊ ڪريو
2. پنهنجي WordPress نيٽ ورڪ ايڊمن ۾ 'پلگ انز' مينيو ذريعي plugin فعال ڪريو
3. Ultimate Multisite → سيٽنگون → مواد هم وقت سازي ۾ سيٽنگون ترتيب ڏيو
4. هم وقت سازي شروع ڪرڻ لاءِ Ultimate Multisite → مواد هم وقت سازي ڏانهن وڃو
