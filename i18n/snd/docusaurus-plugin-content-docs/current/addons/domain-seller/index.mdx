# ڊومين وڪرو ڪندڙ {#domain-seller}

**Multisite Ultimate: Domain Seller** توهان جي WordPress Multisite نيٽ ورڪ کي هڪ طاقتور ڊومين وڪرو ڪندڙ پليٽفارم ۾ بدلائي ٿو. گراهڪن کي سائن اپ دوران يا پنهنجي Dashboard مان سڌو سنئون ڪسٽم ڊومين رجسٽر ڪرڻ ڏيو، حقيقي وقت ۾ دستيابي جي جاچ ۽ شفاف قيمتن سان.

### Domain Seller ڇو؟ {#why-domain-seller}

ڊومين رجسٽريشن ڪنهن به WaaS (Website as a Service) پليٽفارم لاءِ سڀ کان وڌيڪ گهرج ڪيل اپ سيلز مان هڪ آهي. پنهنجي گراهڪن کي ٽئين ڌر جي رجسٽرار ڏانهن موڪلڻ بدران، توهان سڌو سنئون ڊومين پيش ڪري سگهو ٿا — پنهنجي نيٽ ورڪ ۾ وڌيڪ آمدني رکندي، جڏهن ته گراهڪن کي بنا رڪاوٽ تجربو ڏيو ٿا.

### اهم خاصيتون {#key-features}

#### ڪيترن ئي ڊومين رجسٽرارن جي سپورٽ {#multiple-domain-registrar-support}
* **OpenSRS** — مڪمل رجسٽريشن، تجديد، ۽ انتظام
* **Namecheap** — WhoisGuard ۽ سينڊباڪس جاچ سان مڪمل انٽيگريشن
* **HostAfrica** — Domains Reseller API ذريعي ڊومين ڳولا، قيمتون، رجسٽريشن، منتقلي، DNS، ۽ ID تحفظ
* **Openprovider** — ريسيلر قيمتون، رجسٽريشن، تجديد، منتقليون، DNS، WHOIS رازداري، ۽ TLD سنڪ
* **Hostinger** — گڏيل Hostinger انٽيگريشن ٽوڪن استعمال ڪندي ڊومين ڳولا ۽ رجسٽريشن
* **وسعت لائق اڏاوت** — Integration Registry نموني تي ٺهيل، جنهن سان وڌيڪ فراهم ڪندڙ شامل ڪرڻ آسان آهي
* **بلٽ اِن ڪنيڪشن جاچ** — لائيو وڃڻ کان اڳ فراهم ڪندڙ جي سندن ۽ ڪنيڪٽوٽي جي تصديق ڪريو

#### رهنمائي ڪيل سيٽ اپ وزرڊ {#guided-setup-wizard}
* **هڪ ڪلڪ سان شروعات** — فراهم ڪندڙ جي ترتيب، TLD درآمد، ۽ ڊومين پراڊڪٽ ٺاهڻ ۾ توهان جي رهنمائي ڪري ٿو
* **خودڪار TLD سنڪ** — توهان جي ڳنڍيل فراهم ڪندڙن مان سڀ دستياب TLDs ۽ هول سيل قيمتون درآمد ڪري ٿو
* **ڊفالٽ پراڊڪٽ ٺاهڻ** — مناسب مارڪ اپ ڊفالٽس سان استعمال لاءِ تيار ڊومين پراڊڪٽ ٺاهي ٿو

#### لچڪدار ڊومين قيمتون {#flexible-domain-pricing}
* **ٽي مارڪ اپ موڊ** — سيڪڙو مارڪ اپ، مقرر مارڪ اپ، يا هر TLD لاءِ مقرر قيمت
* **TLD-مخصوص پراڊڪٽس** — مختلف TLD گروپن (.com/.net، .io، ملڪ ڪوڊز، وغيره) لاءِ الڳ ڊومين پراڊڪٽس ٺاهيو
* **تعارفي قيمتون** — نين رجسٽريشنز کي متوجه ڪرڻ لاءِ پهرين سال جي رعايتي قيمت پيش ڪريو
* **حقيقي وقت جون قيمتون** — ذهين ڪيشنگ سان فراهم ڪندڙن کان لائيو هول سيل قيمتون
* **ڪوپن سپورٽ** — Multisite Ultimate رعايتي ڪوڊز سان مڪمل مطابقت

#### WHOIS رازداري تحفظ {#whois-privacy-protection}
* **هر پراڊڪٽ جي ترتيب** — WHOIS رازداري کي غير فعال، هميشه شامل، يا گراهڪ جي چونڊ تي سيٽ ڪريو
* **گراهڪ آپٽ اِن** — جڏهن گراهڪ جي چونڊ تي سيٽ هجي، ته چيڪ آئوٽ چيڪ باڪس گراهڪن کي رازداري تحفظ شامل ڪرڻ ڏئي ٿو
* **ترتيب ڏئي سگهڻ واري قيمت** — رازداري ايڊ آن لاءِ هر سال جي قيمت مقرر ڪريو، يا ان کي مفت پيش ڪريو
* **رجسٽرار انٽيگريشن** — رجسٽريشن وقت پاڻمرادو WhoisGuard (Namecheap) يا WHOIS رازداري (OpenSRS) فعال ڪري ٿو

#### متحد چيڪ آئوٽ تجربو {#unified-checkout-experience}
* **ٽي ڊومين موڊ** — گراهڪ سب ڊومين (مفت)، نئون ڊومين رجسٽر ڪرڻ، يا اڳ ۾ پنهنجي ملڪيت واري موجوده ڊومين کي ميپ ڪرڻ مان چونڊين ٿا
* **حقيقي وقت ڊومين ڳولا** — صفحي کي ٻيهر لوڊ ڪرڻ کانسواءِ AJAX-طاقت واري دستيابي جاچ
* **ذهين صلاحون** — جڏهن پسنديده ڊومين ورتل هجي ته متبادل TLD صلاحون
* **شفاف قيمتون** — رجسٽريشن قيمت، تجديد قيمت، ۽ اختياري WHOIS رازداري فيس جو صاف ڏيک
* **رجسٽرنٽ رابطي جا فيلڊ** — چيڪ آئوٽ دوران گهربل رجسٽرنٽ معلومات (نالو، پتو، فون) اِن لائن گڏ ڪري ٿو
* **خودڪار ٺهيل سائيٽ URL** — سائيٽ پتو چونڊيل ڊومين مان پاڻمرادو نڪتل هوندو آهي

#### خودڪار تجديد نظام {#auto-renewal-system}
* **ميمبرشپ سان ڳنڍيل تجديدون** — گراهڪ جي ميمبرشپ فعال هئڻ دوران ڊومين پاڻمرادو تجديد ٿين ٿا
* **ذهين شيڊيولنگ** — تجديد نوڪريون ڊومين جي ختم ٿيڻ وارين تاريخن جي بنياد تي قطار ۾ لڳايون وڃن ٿيون
* **ٻيهر ڪوشش منطق** — ناڪام تجديد ڪوششن لاءِ ذهين ٻيهر ڪوششون
* **اي ميل اطلاع** — ايندڙ تجديدن، ڪامياب تجديدن، ناڪامين، ۽ ختم ٿيڻ بابت ترتيب ڏئي سگهڻ وارا الرٽس

#### گراهڪ DNS انتظام {#customer-dns-management}
* **پاڻمرادو خدمت DNS** — گراهڪ پنهنجي Dashboard مان DNS ريڪارڊز (A, AAAA, CNAME, MX, TXT) شامل، ايڊٽ، ۽ حذف ڪري سگهن ٿا
* **ڊومين حيثيت** — حقيقي وقت رجسٽريشن حيثيت، ختم ٿيڻ جون تاريخون، ۽ نيم سرور معلومات
* **خودڪار ڊومين ميپنگ** — رجسٽر ٿيل يا موجوده ڊومين پاڻمرادو گراهڪ جي سائيٽ سان ميپ ڪيا وڃن ٿا

#### ايڊمن اوزار {#admin-tools}
* **دستي ڊومين رجسٽريشن** — ايڊمن پينل مان گراهڪن جي طرفان ڊومين رجسٽر ڪريو
* **ڊومين مانيٽرنگ** — هڪ صفحي مان سڀ گراهڪ ڊومين، حيثيتون، ۽ ختم ٿيڻ جون تاريخون ڏسو
* **فراهم ڪندڙ-مخصوص لاگنگ** — هر رجسٽرار لاءِ الڳ لاگ چينلز (مثال طور، domain-seller-namecheap.log)
* **رجسٽريشن تفصيل** — بنيادي ڊومين ايڊٽ صفحي تي رجسٽرنٽ معلومات ۽ رجسٽريشن ڊيٽا ڏسو

### سپورٽ ٿيل فراهم ڪندڙ {#supported-providers}

* **OpenSRS** — رجسٽريشن، تجديد، DNS انتظام، ۽ WHOIS رازداري سان مڪمل انٽيگريشن
* **Namecheap** — رجسٽريشن، تجديد، بيچ دستيابي جاچ، ۽ مفت WhoisGuard سان مڪمل انٽيگريشن
* **HostAfrica** — رجسٽريشن، تجديد، منتقليون، DNS انتظام، EPP ڪوڊ حاصل ڪرڻ، رجسٽرار لاڪ، ۽ ID تحفظ
* **Openprovider** — رجسٽريشن، تجديد، منتقليون، DNS انتظام، EPP ڪوڊ حاصل ڪرڻ، رجسٽرار لاڪ، WHOIS رازداري، ۽ ريسيلر TLD سنڪ
* **Hostinger** — رجسٽريشن، ڊومين lookup، نيم سرور اپڊيٽس، رجسٽرار لاڪ، ۽ WHOIS رازداري؛ تجديدون Hostinger خودڪار تجديد ذريعي سنڀاليون وڃن ٿيون
* **GoDaddy** — رجسٽريشن ۽ تجديد
* **ResellerClub** — رجسٽريشن، تجديد، ۽ DNS انتظام
* **NameSilo** — رجسٽريشن ۽ تجديد
* **Enom** — رجسٽريشن، تجديد، ۽ DNS انتظام

### ٽيڪنيڪي تفصيل {#technical-details}

* **گهرجي ٿو:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **اڏاوت:** Domain_Selling_Capability انٽرفيس سان Integration Registry نمونو
* **Autoloading:** PSR-4 ڪلاس ميپنگ سان Jetpack autoloader
* **معيار:** WordPress ڪوڊنگ معيار، سخت ٽائپنگ، تيار ڪيل statements

## انسٽاليشن {#installation}

1. پڪ ڪريو ته **Multisite Ultimate v2.4.12+** انسٽال ٿيل ۽ نيٽورڪ-فعال ٿيل آهي
2. plugin کي `/wp-content/plugins/ultimate-multisite-domain-seller/` تي اپلوڊ ڪريو
3. WordPress نيٽورڪ ايڊمن ۾ **Plugins** مان plugin کي نيٽورڪ-فعال ڪريو
4. **سيٽ اپ وزارڊ** پهرين فعال ٿيڻ تي پاڻمرادو شروع ٿئي ٿو — هنن قدمن تي عمل ڪريو:
   * پنهنجي ڊومين رجسٽرار کي ڳنڍيو
   * ڪنيڪشن جي جاچ ڪريو
   * دستياب TLDs ۽ هول سيل قيمتون درآمد ڪريو
   * قيمتي مارڪ اپ سان پنهنجو پهريون ڊومين پراڊڪٽ ٺاهيو
5. **Ultimate Multisite > چيڪ آئوٽ فارم** ۾ پنهنجي چيڪ آئوٽ فارم ۾ **ڊومين چونڊ** فيلڊ شامل ڪريو

## اڪثر پڇيا ويندڙ سوال {#frequently-asked-questions}

### Multisite Ultimate جو ڪهڙو ورزن گهربل آهي؟ {#what-version-of-multisite-ultimate-is-required}

ورزن 2.4.12 يا ان کان وڌيڪ. addon WP Ultimo v1.x يا 2.4.12 کان هيٺ Ultimate Multisite ورزنن سان مطابقت نٿو رکي.

### ڪهڙا ڊومين فراهم ڪندڙ سهائتا ٿيل آهن؟ {#which-domain-providers-are-supported}

نو رجسٽرار سهائتا ٿيل آهن: **OpenSRS**، **Namecheap**، **HostAfrica**، **Openprovider**، **Hostinger**، **GoDaddy**، **ResellerClub**، **NameSilo**، ۽ **Enom**. صلاحيتون فراهم ڪندڙ موجب مختلف ٿين ٿيون، ڇاڪاڻ⁠تہ رجسٽرار APIs مختلف عمل ظاهر ڪن ٿيون؛ هر فراهم ڪندڙ جي جائزي لاءِ سيٽ اپ گائيڊ ڏسو. addon جي Integration Registry جوڙجڪ ڪسٽم فراهم ڪندڙ شامل ڪرڻ کي سولو بڻائي ٿي — [developer guide](/developer/integration-guide/custom-registrar) ڏسو.

### ڇا مان WHOIS رازداري تحفظ پيش ڪري سگهان ٿو؟ {#can-i-offer-whois-privacy-protection}

ها. هر ڊومين پراڊڪٽ وٽ WHOIS رازداري سيٽنگ آهي، جنهن جا ٽي موڊ آهن: غيرفعال، هميشه شامل، يا گراهڪ جي چونڊ. جڏهن گراهڪ جي چونڊ تي سيٽ ڪيو وڃي، ته گراهڪ چيڪ آئوٽ دوران قيمت سان گڏ هڪ چيڪ باڪس ڏسن ٿا (يا Namecheap جي WhoisGuard لاءِ "مفت"). رازداري فليگ رجسٽريشن وقت پاڻمرادو رجسٽرار ڏانهن موڪليو ويندو آهي.

### ڇا مان مختلف TLDs لاءِ مختلف قيمتون مقرر ڪري سگهان ٿو؟ {#can-i-set-different-pricing-for-different-tlds}

ها. مختلف TLD گروپن لاءِ الڳ ڊومين پراڊڪٽس ٺاهيو. هر پراڊڪٽ جو پنهنجو مارڪ اپ قسم (سيڪڙو، مقرر مارڪ اپ، يا مقرر قيمت)، سهائتا ٿيل TLDs، ۽ تعارفي قيمت ٿي سگهي ٿي.

### گراهڪ چيڪ آئوٽ جا ڪهڙا اختيار ڏسن ٿا؟ {#what-checkout-options-do-customers-see}

ڊومين چونڊ فيلڊ گراهڪن کي ٽي ٽيب ڏئي ٿو: **سب ڊومين** (توهان جي نيٽورڪ تي مفت سب ڊومين استعمال ڪريو)، **نئون ڊومين رجسٽر ڪريو** (نئون ڊومين ڳوليو ۽ رجسٽر ڪريو)، ۽ **موجوده ڊومين** (اهڙو ڊومين ميپ ڪريو جيڪو انهن وٽ اڳ ۾ ئي آهي). صرف اهي موڊ ظاهر ٿيندا جيڪي توهان فعال ڪندا.

### ڇا اهو پاڻمرادو تجديدن کي سهائتا ڪري ٿو؟ {#does-it-support-automatic-renewals}

ها. ڊومينز کي آٽو-رينيول تي سيٽ ڪري سگهجي ٿو ۽ اهي گراهڪ جي رڪنيت جي حالت سان ڳنڍيل هوندا آهن. تجديد جون ادائگيون گراهڪ جي موجوده ادائيگي گيٽ وي ذريعي پروسيس ڪيون وينديون آهن.

### ڇا گراهڪ پنهنجا DNS ريڪارڊ منظم ڪري سگهن ٿا؟ {#can-customers-manage-their-dns-records}

ها. گراهڪن کي پنهنجي Dashboard ۾ DNS انتظام انٽرفيس ملي ٿو، جنهن سان اهي A، AAAA، CNAME، MX، ۽ TXT ريڪارڊ شامل، ايڊٽ، ۽ حذف ڪري سگهن ٿا. DNS انتظام OpenSRS، ResellerClub، Enom، HostAfrica، ۽ Openprovider سان دستياب آهي. ميزباني ٿيل ڊومينز لاءِ Hostinger DNS بنيادي Hostinger ڊومين-ميپنگ انٽيگريشن ذريعي سنڀاليو ويندو آهي.

### ڇا سيٽ اپ وزارڊ آهي؟ {#is-there-a-setup-wizard}

ها. سيٽ اپ وزارڊ پهرين فعال ٿيڻ تي پاڻمرادو هلندو آهي ۽ توهان کي رجسٽرار ڳنڍڻ، TLDs درآمد ڪرڻ، ۽ پنهنجو پهريون ڊومين پراڊڪٽ ٺاهڻ جي عمل مان گذاريندو آهي. توهان addon سيٽنگن مان وزارڊ ٻيهر به هلائي سگهو ٿا.
