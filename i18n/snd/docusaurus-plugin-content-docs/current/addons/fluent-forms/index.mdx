# Fluent Forms

هي add-on Multisite Ultimate تي مصنوعات، رڪنيتن يا سائيٽن ۾ Fluent Forms لاءِ حدون شامل ڪرڻ ممڪن بڻائي ٿي.
Fluent Forms ڪسٽم پوسٽ قسمن جي بدران ڪسٽم ٽيبلون استعمال ڪري ٿو، تنهنڪري عام حد لڳائڻ وارا اختيار Fluent Forms سان ڪم نٿا ڪن.

## خاصيتون {#features}

- Fluent Forms يا Conversational Fluent Forms لاءِ حدون شامل ڪريو
- گراهڪن کي حد تي پهچڻ وقت پنهنجو پلان اپگريڊ ڪرڻ لاءِ هڪ دوستاڻو پيغام ملي ٿو
