# Filter: wu_limits_is_fluent_form_above_limit

چڪاس ڪري ٿو ته ڏنل فارم جو قسم هن پلان تي اجازت يافته آهي يا نه. plugin ڊولپرن کي واپسي جي قيمت فلٽر ڪرڻ جي اجازت ڏئي ٿو

## پيرا ميٽر {#parameters}

| نالو | قسم | وضاحت |
|------|------|-------------|
| $value | `bool` | ڇا فارم جو قسم حد کان مٿي آهي يا نه |
| $form_type | `string` | اهو فارم جو قسم جنهن جي چڪاس ڪئي پئي وڃي |
| $form_count | `int` | موجوده فارم ڳڻپ |
| $quota | `int` | اجازت ڏنل ڪوٽا |
