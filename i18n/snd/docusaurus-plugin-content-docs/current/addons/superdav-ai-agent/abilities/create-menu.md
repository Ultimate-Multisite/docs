# مينيو ٺاهيو

**مينيو ٺاهيو** صلاحيت WordPress نيويگيشن مينيوز ٺاهي ٿي، جن ۾ صفحي جي عنوانن کان الڳ نيويگيشن ليبلز جي سهولت هوندي آهي.

## جائزو {#overview}

هي صلاحيت معياري مينيو ٺاهڻ واري ڪارڪردگي کي `navigation_label` پيراميٽر بيان ڪرڻ جي قابليت سان وڌائي ٿي. هي توهان کي اهڙا مينيوز ٺاهڻ جي اجازت ڏئي ٿي جتي نيويگيشن ۾ ڏيکاريل ليبل صفحي جي عنوان کان مختلف هجي، جنهن سان سائيٽ جي جوڙجڪ ۽ استعمال ڪندڙ جي تجربي ۾ وڌيڪ لچڪ ملي ٿي.

## پيراميٽر {#parameters}

| پيراميٽر | قسم | گهربل | وضاحت |
|-----------|------|----------|-------------|
| `name` | string | ها | مينيو جو نالو، مثال طور `Primary Navigation` |
| `location` | string | نه | Theme جي جڳهه جنهن کي هي مينيو مقرر ڪيو وڃي، مثال طور `primary` |
| `navigation_label` | string | نه | نيويگيشن ۾ ڏيکارڻ لاءِ ليبل (صفحي جي عنوان کان الڳ) |

## واپسي قدر {#return-value}

```json
{
  "success": true,
  "menu_id": 7,
  "name": "Primary Navigation",
  "navigation_label": "Main Menu"
}
```

## نيويگيشن ليبل بمقابله صفحي جو عنوان {#navigation-label-vs-page-title}

`navigation_label` پيراميٽر توهان کي اندروني مينيو نالي کي استعمال ڪندڙن کي ڏيکاريل ليبل کان الڳ ڪرڻ جي اجازت ڏئي ٿو:

- **`name`** — WordPress پاران استعمال ٿيندڙ اندروني مينيو سڃاڻپ ڪندڙ (مثال طور، "Primary Navigation")
- **`navigation_label`** — نيويگيشن ۾ سائيٽ جي دورو ڪندڙن کي ڏيکاريل ليبل (مثال طور، "Main Menu")

هي تڏهن ڪارائتو آهي جڏهن:
- توهان جو اندروني نالو رکڻ جو طريقو استعمال ڪندڙن کي ڏيکاريل ليبلز کان مختلف هجي
- توهان ايڊمن پينل جي ڀيٽ ۾ نيويگيشن ۾ ننڍا ليبل چاهيو
- توهان کي مختلف ليبل ڊيگهن سان ڪيترين ٻولين جي سهولت ڏيڻ جي ضرورت هجي
- توهان مخصوص علائقن يا استعمال ڪندڙ گروپن لاءِ مينيوز ٺاهي رهيا آهيو

## استعمال جا مثال {#usage-examples}


## Theme Builder سان انضمام {#integration-with-theme-builder}

Theme Builder استعمال ڪندي، مينيو ٺاهڻ واري صلاحيت:

1. موجود Theme مينيو جڳهن کي پاڻمرادو سڃاڻي ٿي
2. توهان جي ڊيزائن لاءِ مناسب نيويگيشن ليبلز سان مينيوز ٺاهي ٿي
3. مينيوز کي صحيح Theme جڳهن تي مقرر ڪري ٿي
4. ٺاهڻ کان پوءِ مينيو شيون شامل ڪرڻ جي سهولت ڏئي ٿي

## لاڳاپيل صلاحيتون {#related-abilities}

- **`add_menu_item`** — موجود مينيو ۾ شيون شامل ڪريو
- **`update_menu`** — مينيو جو نالو مٽايو يا ان کي Theme جڳهه تي ٻيهر مقرر ڪريو
- **`delete_menu`** — پنهنجي سائيٽ مان مينيو هٽايو

## بهترين طريقا {#best-practices}

- **صاف نيويگيشن ليبلز استعمال ڪريو** — استعمال ڪندڙن لاءِ ليبلز مختصر ۽ وضاحتي رکو
- **Theme جڳهن سان ملائي رکو** — صحيح ڏيک لاءِ مينيوز کي صحيح Theme جڳهه تي مقرر ڪريو
- **مينيو جوڙجڪ جي رٿابندي ڪريو** — شيون ٺاهڻ کان اڳ پنهنجي مينيو درجابندي جو فيصلو ڪريو
- **جوابي ڏيک جي جاچ ڪريو** — پڪ ڪريو ته مينيوز موبائل ڊوائيسز تي صحيح ڏيکارجن ٿا
- **ليبلز کي مقامي بڻايو** — مختلف ٻولي ورجنز لاءِ مختلف نيويگيشن ليبلز استعمال ڪريو

## حدون {#limitations}

- نيويگيشن ليبلز صرف ڏيک لاءِ آهن؛ اندروني `name` اڃا به WordPress سڃاڻپ لاءِ استعمال ٿئي ٿو
- Theme جي سهولت مختلف ٿي سگهي ٿي؛ سڀ Themes سڀني مينيو جڳهن جي سهولت نٿا ڏين
- مينيو ٺاهڻ کان پوءِ مينيو شيون الڳ سان شامل ڪرڻيون پونديون
- نيويگيشن ليبل تبديل ڪرڻ لاءِ مينيو کي اپڊيٽ ڪرڻو پوندو
