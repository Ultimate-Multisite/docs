# لوگو SVG ٺاهيو

**لوگو SVG ٺاهيو** صلاحيت Theme Builder کي اجازت ڏئي ٿي ته اهو توهان جي WordPress سائيٽ ۾ سڌو سنئون ڪسٽم لوگو SVGs ٺاهي ۽ شامل ڪري، خودڪار namespace-safe صفائي سان.

## جائزو {#overview}

هي صلاحيت توهان جي سائيٽ جي برانڊنگ رخ ۽ ڊزائن ترجيحن جي بنياد تي scalable vector graphics (SVG) لوگوز ٺاهي ٿي. ٺاهيل SVGs پاڻمرادو صاف ڪيا ويندا آهن ته جيئن اهي WordPress ۾ استعمال لاءِ محفوظ هجن، جڏهن ته بصري سالميت برقرار رهي.

## پيرا ميٽر {#parameters}

| پيرا ميٽر | قسم | گهربل | وضاحت |
|-----------|------|----------|-------------|
| `site_name` | string | ها | اها سائيٽ جو نالو جنهن لاءِ لوگو ٺاهڻو آهي |
| `style` | string | ها | ڊزائن جو انداز (مثال طور، "جديد"، "ڪلاسيڪي"، "گهٽ ۾ گهٽ"، "رانديل") |
| `colors` | array | نه | لوگو ۾ استعمال ڪرڻ لاءِ hex رنگ ڪوڊن جي فهرست (مثال طور، `["#678233", "#ffffff"]`) |
| `width` | number | نه | SVG ويڪر pixels ۾ (ڊفالٽ: 200) |
| `height` | number | نه | SVG اوچائي pixels ۾ (ڊفالٽ: 200) |
| `include_text` | boolean | نه | ڇا لوگو ۾ سائيٽ جو نالو متن طور شامل ڪجي (ڊفالٽ: true) |

## آئوٽ پٽ فارميٽ {#output-format}

هي صلاحيت هيٺين ساخت سان هڪ SVG string واپس ڪري ٿي:

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## SVG صفائي جو رويو {#svg-sanitisation-behaviour}

ٺاهيل SVGs پاڻمرادو namespace-safe صفائي مان گذرن ٿا ته جيئن:

- **غير محفوظ خاصيتون هٽايون** — event handlers، scripts، ۽ امڪاني طور خطرناڪ خاصيتون ختم ڪري ٿو
- **namespaces برقرار رکي** — صحيح rendering لاءِ SVG namespaces (xmlns, xlink) برقرار رکي ٿو
- **ساخت جي تصديق ڪري** — يقيني بڻائي ٿو ته SVG، W3C معيارن سان مطابقت رکي ٿو
- **entities encode ڪري** — متن جي مواد ۾ خاص اکرن کي صحيح نموني escape ڪري ٿو
- **ٻاهريون حوالا هٽائي** — ٻاهريون stylesheets ۽ تصويري حوالا ختم ڪري ٿو

هي يقيني بڻائي ٿو ته SVG اضافي صفائي جي ضرورت کان سواءِ سڌو WordPress ۾ شامل ڪرڻ لاءِ محفوظ آهي.

## استعمال جو مثال {#usage-example}

**Prompt:**
```
منهنجي tech startup "CloudSync" لاءِ نيرو ۽ اڇو رنگ استعمال ڪندي هڪ جديد لوگو ٺاهيو.
```

**نتيجو:**
هي صلاحيت هڪ SVG لوگو ٺاهي ٿي جيڪو:
- سائيٽ جو نالو "CloudSync" شامل ڪري ٿو
- مخصوص نيرو ۽ اڇو رنگ اسڪيم استعمال ڪري ٿو
- جديد ڊزائن اصولن تي عمل ڪري ٿو
- پاڻمرادو صاف ٿيل ۽ استعمال لاءِ تيار آهي

## Theme Builder سان انضمام {#integration-with-theme-builder}

جڏهن Theme Builder جي ڊزائن-رخ چونڊ استعمال ڪئي وڃي، لوگو SVG ٺاهڻ واري صلاحيت:

1. توهان جي ڊزائن رخ ۽ رنگ palette جو تجزيو ڪري ٿي
2. توهان جي ترجيحن سان ملندڙ ڪسٽم SVG لوگو ٺاهي ٿي
3. لوگو کي پاڻمرادو توهان جي سائيٽ جي header/branding علائقي ۾ شامل ڪري ٿي
4. SVG کي WordPress media ۾ ڪسٽم لوگو طور محفوظ ڪري ٿي

## بهترين طريقا {#best-practices}

- **صاف انداز جون ترجيحون ڏيو** — جيڪو ڊزائن انداز توهان چاهيو ٿا اهو بيان ڪريو (جديد، ڪلاسيڪي، رانديل، وغيره)
- **رنگ مخصوص ڪريو** — مستقل مزاجي لاءِ پنهنجا brand رنگ شامل ڪريو
- **rendering آزمائيو** — تصديق ڪريو ته لوگو مختلف screen sizes تي صحيح ڏيکاري ٿو
- **وڌيڪ ڪسٽمائيز ڪريو** — سائيز ۽ جڳهه ترتيب ڏيڻ لاءِ WordPress جا لوگو customization اوزار استعمال ڪريو

## حدون {#limitations}

- SVG لوگوز static graphics طور ٺاهيا ويندا آهن (animated نه)
- گهڻن عنصرن وارا پيچيده لوگوز manual refinement گهرندا هوندا
- ڪسٽم fonts سپورٽ ناهن؛ متن system fonts استعمال ڪري ٿو
- تمام وڏا يا تمام ننڍا dimensions معيار تي اثر وجهي سگهن ٿا
