# هاسپيٽلٽي مينيوز

**هاسپيٽلٽي مينيوز** خصوصيت Theme Builder کي اجازت ڏئي ٿي ته اهو توهان جي WordPress سائيٽ ۾ منظم کاڌي ۽ مشروبات جي مينيو صفحن کي سڌو ٺاهي ۽ شامل ڪري.

## جائزو

Theme Builder هاڻي ريسٽورنٽن، ڪيفيز، بارن، ۽ ڪيٽرنگ خدمتن سميت هاسپيٽلٽي ڪاروبارن لاءِ پيشاور، منظم مينيو صفحا ٺاهي سگهي ٿو. اهي مينيو توهان جي سائيٽ جي ڊيزائن ۾ مڪمل طور ضم ٿيل آهن ۽ آساني سان اپڊيٽ ۽ منظم ڪري سگهجن ٿا.

## سپورٽ ٿيل هاسپيٽلٽي قسم

- **ريسٽورنٽس** — مڪمل خدمت وارا ڊائننگ مينيو
- **ڪيفيز** — ڪافي ۽ هلڪي کاڌي جا مينيو
- **بارز ۽ لائونجز** — مشروبات ۽ شروعاتي کاڌن جا مينيو
- **بيڪريز** — پيسٽري ۽ ماني جا مينيو
- **ڪيٽرنگ خدمتون** — تقريبن جي مينيو جا اختيار
- **فوڊ ٽرڪس** — موبائل کاڌي جي خدمت جا مينيو
- **بريوريز ۽ وائنريز** — وضاحتن سان مشروبات جا مينيو

## مينيو جي جوڙجڪ

### مينيو ڪيٽيگريون

مينيو ڪيٽيگرين ۾ منظم ڪيا ويندا آهن:

- **شروعاتي کاڌا** — شروع لاءِ ۽ ننڍيون پليٽون
- **مکيه کاڌا** — مکيه ڪورس
- **ساٿ وارا کاڌا** — گڏ کائڻ واريون شيون ۽ ڀاڄيون
- **مٺايون** — مٺا طعام
- **مشروبات** — پيئڻ جون شيون (الڪوحل وارا ۽ غير الڪوحل)
- **خاص شيون** — روزاني يا موسمي خاص شيون

### مينيو آئٽم فارميٽ

هر مينيو آئٽم ۾ شامل آهي:

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### آئٽم فيلڊز

| فيلڊ | قسم | وضاحت |
|-------|------|-------------|
| `name` | string | طعام يا مشروب جو نالو |
| `description` | string | آئٽم جي تفصيلي وضاحت |
| `price` | string | قيمت (ڪرنسي سان فارميٽ ٿيل) |
| `dietary_info` | array | غذائي خاصيتون (Vegan، Gluten-Free، وغيره) |
| `allergens` | array | عام الرجن (Nuts، Shellfish، وغيره) |
| `availability` | string | ڪڏهن موجود آهي (Daily، Seasonal، وغيره) |

## هاسپيٽلٽي مينيوز ٺاهڻ

### قدم 1: مينيو جي معلومات ڏيو

Theme Builder کي پنهنجي مينيو بابت ٻڌايو:

```
منهنجي اطالوي ريسٽورنٽ لاءِ شروعاتي کاڌن،
پاستا طعامن، مکيه ڪورسز، ۽ مٺاين سان ريسٽورنٽ مينيو ٺاهيو. قيمتون ۽ وضاحتون شامل ڪريو.
```

### قدم 2: Theme Builder مينيو ٺاهي ٿو

Theme Builder:

1. منظم مينيو صفحو ٺاهي ٿو
2. ان کي توهان جي سائيٽ جي theme سان ملائڻ لاءِ ڊيزائن ڪري ٿو
3. آئٽمن کي ڪيٽيگرين ۾ منظم ڪري ٿو
4. قيمتن ۽ وضاحتن کي فارميٽ ڪري ٿو
5. غذائي ۽ الرجن معلومات شامل ڪري ٿو

### قدم 3: جائزو وٺو ۽ ڪسٽمائيز ڪريو

توهان ڪري سگهو ٿا:

1. مينيو آئٽمن ۽ قيمتن کي ايڊٽ ڪريو
2. ڪيٽيگريون شامل يا ختم ڪريو
3. ڪيٽيگرين اندر آئٽمن کي ٻيهر ترتيب ڏيو
4. وضاحتون ۽ غذائي معلومات اپڊيٽ ڪريو
5. اسٽائلنگ ۽ لي آئوٽ کي ترتيب ڏيو

## مينيو ڏيکارڻ جا اختيار

### مڪمل مينيو صفحو

توهان جو مڪمل مينيو ڏيکاريندڙ هڪ مخصوص صفحو:

- ڪيٽيگري موجب منظم
- ڳولڻ ۽ فلٽر ڪرڻ لائق
- پرنٽ لاءِ موزون لي آئوٽ
- موبائل-رسپانسو ڊيزائن

### مينيو وجيٽ

ٻين صفحن تي مينيو سيڪشن شامل ڪريو:

- هوم پيج تي نمايان آئٽم
- سائيڊبار تي روزاني خاص شيون
- بار صفحي تي مشروبات جو مينيو
- فوٽر ۾ مٺاين جي نمائش

### مينيو PDF

ڊائون لوڊ ڪرڻ لائق PDF مينيو ٺاهيو:

- پيشاور فارميٽنگ
- پرنٽ لاءِ تيار معيار
- تصويرون ۽ وضاحتون شامل آهن
- شيئر ۽ اي ميل ڪرڻ آسان

## غذائي ۽ الرجن معلومات

### غذائي خاصيتون

آئٽمن کي غذائي معلومات سان نشان لڳايو:

- **Vegan** — جانورن جون ڪا به شيون نه
- **Vegetarian** — گوشت نه
- **Gluten-Free** — سيليئڪ بيماري لاءِ محفوظ
- **Dairy-Free** — کير جون شيون نه
- **Nut-Free** — وڻن جا نٽ يا مونگڦليون نه
- **Low-Carb** — گهٽ ڪاربوهائيڊريٽس
- **High-Protein** — پروٽين سان ڀرپور

### الرجن چتاءَ

عام الرجن شامل ڪريو:

- **Nuts** — وڻن جا نٽ ۽ مونگڦليون
- **Shellfish** — ڪرسٽيشين ۽ مولسڪ
- **Fish** — مڇيءَ جا سڀ قسم
- **Dairy** — کير ۽ کير جون شيون
- **Eggs** — ڪڪڙ جا آنا
- **Soy** — سويا جون شيون
- **Gluten** — ڪڻڪ ۽ Gluten رکندڙ اناج
- **Sesame** — تل جا ٻج ۽ تيل

## مينيو مينيجمينٽ

### قيمتون اپڊيٽ ڪرڻ

مينيو قيمتون آساني سان اپڊيٽ ڪريو:

1. مينيو صفحي ڏانهن وڃو
2. "مينيو ايڊٽ ڪريو" تي ڪلڪ ڪريو
3. آئٽمن لاءِ قيمتون اپڊيٽ ڪريو
4. تبديليون محفوظ ڪريو
5. تبديليون توهان جي سائيٽ تي فوري طور ظاهر ٿينديون

### موسمي آئٽم شامل ڪرڻ

موسمي مينيو تبديليون ٺاهيو:

1. نئون مينيو ورزن ٺاهيو
2. موسمي آئٽم شامل ڪريو
3. آئٽمن کي "موسمي" طور نشان لڳايو
4. دستيابي جون تاريخون شيڊيول ڪريو
5. موسم دوران پاڻمرادو ڏيکاريو

### خاص شين جو انتظام

روزاني يا هفتيوار خاص شيون ڏيکاريو:

1. "خاص شيون" ڪيٽيگري ٺاهيو
2. دستيابي جي تاريخن سان آئٽم شامل ڪريو
3. هوم پيج تي خاص شين کي نمايان ڪريو
4. روزانو يا هفتيوار اپڊيٽ ڪريو
5. پراڻيون خاص شيون آرڪائيو ڪريو

## Theme Builder سان انٽيگريشن

هاسپيٽلٽي سائيٽن لاءِ Theme Builder استعمال ڪندي:

1. **پاڻمرادو مينيو سڃاڻپ** — سڃاڻي ٿو ته ڇا توهان کاڌي/مشروبات جو ڪاروبار آهيو
2. **مينيو صفحي جي تخليق** — پيشاور مينيو صفحا ٺاهي ٿو
3. **ڊيزائن ملائڻ** — مينيو توهان جي سائيٽ جي theme سان ملن ٿا
4. **موبائل بهتري** — مينيو فونن تي سهڻي نموني ڏيکارين ٿا
5. **SEO بهتري** — مينيو سرچ-انجن دوست آهن

## بهترين طريقا

### مينيو ڊيزائن

- **واضح تنظيم** — منطقي ڪيٽيگري جوڙجڪ
- **پڙهڻ لائق وضاحتون** — دلڪش ۽ معلوماتي
- **هڪجهڙي قيمت بندي** — واضح ڪرنسي ۽ فارميٽنگ
- **پيشاور تصويرون** — اعليٰ معيار جون کاڌي جون تصويرون
- **خالي جڳهه** — صفحي کي گهڻو نه ڀريو

### مواد

- **درست وضاحتون** — آئٽمن کي درست نموني بيان ڪريو
- **خاص شين کي نمايان ڪريو** — خاص شين کي نمايان بڻايو
- **الرجن شامل ڪريو** — هميشه الرجن جي فهرست ڏيو
- **باقاعدي اپڊيٽ ڪريو** — قيمتون ۽ آئٽم تازه رکو
- **دلڪش ٻولي استعمال ڪريو** — آئٽمن کي لذيذ محسوس ڪرائيندڙ بڻايو

### رسائي

- **پڙهڻ لائق فونٽس** — صاف، پڙهڻ ۾ آسان ٽائيپ فيس استعمال ڪريو
- **ڪافي ڪنٽراسٽ** — پڪ ڪريو ته متن پڙهڻ لائق آهي
- **غذائي ليبلز** — غذائي اختيارن کي صاف نشان لڳايو
- **الرجن چتاءَ** — الرجن کي نمايان نموني ڏيکاريو
- **موبائل-دوست** — سڀني ڊوائيسز تي ٽيسٽ ڪريو
