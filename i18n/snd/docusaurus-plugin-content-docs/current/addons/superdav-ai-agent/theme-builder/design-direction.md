# ڊيزائن جي سمت

**ڊيزائن جي سمت** وارو قدم توهان کي اجازت ڏئي ٿو ته Theme Builder مڪمل ٿيم ٺاهڻ کان اڳ توهان پنهنجي ٿيم جي بصري ڊيزائن جو جائزو وٺو، ان کي بهتر ڪريو، ۽ حتمي ڪريو.

## جائزو {#overview}

Discovery Interview مڪمل ڪرڻ کان پوءِ، Theme Builder توهان جي ڊيزائن جي سمت کي **ڊيسڪ ٽاپ ۽ موبائل پريويو رينڊرنگ** سان پيش ڪري ٿو. هي توهان کي اجازت ڏئي ٿو ته ٿيم تي عمل ڪرڻ کان اڳ مختلف ڊوائيسز تي توهان جي ڊيزائن ڪيئن نظر ايندي، اهو بلڪل ڏسي سگهو.

## ڊيزائن جي سمت جا جزا {#design-direction-components}

### 1. رنگن جي پيليٽ {#1-color-palette}

توهان جي ٿيم جو رنگي منصوبو، جنهن ۾ شامل آهن:

- **بنيادي رنگ** — بٽڻن، لنڪن، ۽ نمايان حصن لاءِ استعمال ٿيندڙ مکيه برانڊ رنگ
- **ثانوي رنگ** — تنوع لاءِ مڪمل ڪندڙ رنگ
- **نمايان رنگ** — اهم عنصرن لاءِ نمايان ڪرڻ وارو رنگ
- **غير جانبدار رنگ** — پسمنظرن ۽ متن لاءِ ڀورا ۽ اڇا رنگ
- **متن جو رنگ** — پڙهڻ ۾ آساني لاءِ بنيادي متن جو رنگ

### 2. ٽائپوگرافي {#2-typography}

فونٽ چونڊون هنن لاءِ:

- **هيڊنگ فونٽ** — صفحي جي عنوانن ۽ سيڪشن هيڊرن لاءِ استعمال ٿيندو
- **باڊي فونٽ** — پيراگراف متن ۽ مواد لاءِ استعمال ٿيندو
- **فونٽ سائيزون** — مختلف اسڪرين سائيزن لاءِ ريسپانسو سائيزنگ
- **لائن اوچائي** — پڙهڻ ۾ آساني لاءِ لائينن جي وچ ۾ فاصلو
- **فونٽ وزن** — بولڊ، عام، ۽ هلڪيون تبديليون

### 3. لي آئوٽ ۽ فاصلا {#3-layout-and-spacing}

- **ڪنٽينر ويڪر** — مواد لاءِ وڌ ۾ وڌ ويڪر
- **پيڊنگ ۽ مارجن** — عنصرن جي چوڌاري فاصلو
- **گرڊ سسٽم** — ريسپانسو ڊيزائن لاءِ ڪالمن لي آئوٽ
- **جزن جو فاصلو** — UI عنصرن جي وچ ۾ فاصلو

### 4. بصري عنصر {#4-visual-elements}

- **بٽڻ انداز** — بنيادي، ثانوي، ۽ ٽرشري بٽڻ ڊيزائن
- **ڪارڊ ڊيزائنون** — مواد بلاڪن لاءِ لي آئوٽ
- **آئڪنز** — آئڪن جو انداز ۽ سائيزنگ
- **بارڊر ۽ ڇانوَ** — نفيس بصري کوٽائي
- **هوور اثرات** — انٽرايڪٽو حالت جي اسٽائلنگ

## پريويو رينڊرنگ {#preview-rendering}

### ڊيسڪ ٽاپ پريويو {#desktop-preview}

ڊيسڪ ٽاپ پريويو توهان جي ڊيزائن کي مڪمل ويڪر تي ڏيکاري ٿو (عام طور تي 1200px يا وڌيڪ):

- مڪمل نيويگيشن مينو
- سڀني ڪالمن سان مڪمل لي آئوٽ
- وڏي ٽائپوگرافي ۽ فاصلو
- سڀ بصري عنصر مڪمل سائيز تي

### موبائل پريويو {#mobile-preview}

موبائل پريويو توهان جي ڊيزائن کي موبائل ويڪر تي ڏيکاري ٿو (عام طور تي 375px):

- ريسپانسو نيويگيشن (همبرگر مينو)
- هڪ ڪالمن وارو لي آئوٽ
- ترتيب ڏنل ٽائپوگرافي ۽ فاصلو
- ٽچ لاءِ مناسب بٽڻ سائيزون

### ريسپانسو بريڪ پوائنٽس {#responsive-breakpoints}

Theme Builder هنن بريڪ پوائنٽس تي پريويوز رينڊر ڪري ٿو:

| ڊوائيس | ويڪر | پريويو |
|--------|-------|---------|
| **موبائل** | 375px | پورٽريٽ فون |
| **ٽيبلٽ** | 768px | لينڊ اسڪيپ ٽيبلٽ |
| **ڊيسڪ ٽاپ** | 1200px | مڪمل ويڪر ڊيسڪ ٽاپ |
| **وڏو ڊيسڪ ٽاپ** | 1920px | الٽرا وائڊ ڊسپلي |

## پنهنجي ڊيزائن کي بهتر ڪرڻ {#refining-your-design}

### تبديليون ڪرڻ {#making-adjustments}

توهان پنهنجي ڊيزائن جي سمت کي هن ريت بهتر ڪري سگهو ٿا:

1. **رنگ ترتيب ڏيڻ** — پيليٽ ۾ ڪنهن به رنگ کي تبديل ڪريو
2. **فونٽ تبديل ڪرڻ** — مختلف ٽائپ فيس چونڊيو
3. **فاصلا تبديل ڪرڻ** — پيڊنگ ۽ مارجن ترتيب ڏيو
4. **لي آئوٽ اپڊيٽ ڪرڻ** — ڪنٽينر ويڪرون ۽ گرڊ ڪالمن تبديل ڪريو
5. **عنصرن کي ڪسٽمائيز ڪرڻ** — بٽڻ انداز، ڪارڊ ڊيزائنون، وغيره تبديل ڪريو.

### پريويو اپڊيٽس {#preview-updates}

تبديليون حقيقي وقت ۾ ظاهر ٿين ٿيون:

- ڊيسڪ ٽاپ پريويو فوري طور اپڊيٽ ٿئي ٿو
- موبائل پريويو فوري طور اپڊيٽ ٿئي ٿو
- سڀ ريسپانسو بريڪ پوائنٽس اپڊيٽ ٿين ٿا
- توهان تبديليون تصديق ڪرڻ لاءِ پريويوز جي وچ ۾ مٽائي سگهو ٿا

### ڀيٽ وارو ڏيک {#comparison-view}

پنهنجي ڊيزائن جي سمت کي هنن سان ڀيٽيو:

- **اصل ڊيزائن** — ڏسو ڇا تبديل ٿيو
- **مد مقابل ڊيزائنون** — الهامي سائيٽن سان ڀيٽيو
- **اڳيون ورزنون** — اڳوڻين ڊيزائن سمتن ڏانهن واپس وڃو

## ڊيزائن جي سمت جي منظوري {#design-direction-approval}

جڏهن توهان پنهنجي ڊيزائن سان مطمئن ٿيو:

1. **ٻنهي پريويوز جو جائزو وٺو** — تصديق ڪريو ته ڊيسڪ ٽاپ ۽ موبائل سٺا نظر اچن ٿا
2. **سڀ رنگ چيڪ ڪريو** — ڪنٽراسٽ ۽ رسائي کي يقيني بڻايو
3. **ٽائپوگرافي آزمائيو** — سڀني سائيزن تي پڙهڻ جي آساني تصديق ڪريو
4. **لي آئوٽ جي تصديق ڪريو** — فاصلو ۽ الائنمينٽ چيڪ ڪريو
5. **ڊيزائن منظور ڪريو** — ٿيم ٺاهڻ ڏانهن اڳتي وڌو

## اڳيان قدم {#next-steps}

پنهنجي ڊيزائن جي سمت منظور ڪرڻ کان پوءِ:

1. Theme Builder توهان جو مڪمل ٿيم ٺاهي ٿو
2. ٿيم توهان جي WordPress سائيٽ تي انسٽال ٿئي ٿو
3. توهان WordPress ڪسٽمائيزر استعمال ڪندي وڌيڪ ڪسٽمائيز ڪري سگهو ٿا
4. [Hospitality Menus](./hospitality-menus.md) يا ٻين خصوصيتن ڏانهن اڳتي وڌو

## بهترين طريقا {#best-practices}

- **حقيقي ڊوائيسز تي آزمائيو** — ممڪن هجي ته اصل فون ۽ ٽيبلٽ استعمال ڪريو
- **پڙهڻ جي آساني چيڪ ڪريو** — يقيني بڻايو ته متن سڀني سائيزن تي پڙهڻ لائق آهي
- **ڪنٽراسٽ جي تصديق ڪريو** — رسائي لاءِ [Validate Palette Contrast](../abilities/validate-palette-contrast.md) استعمال ڪريو
- **ڪارڪردگي تي غور ڪريو** — رفتار لاءِ تصويرون ۽ فونٽ بهتر ڪريو
- **مواد لاءِ منصوبو ٺاهيو** — يقيني بڻايو ته لي آئوٽ توهان جي اصل مواد سان ڪم ڪري ٿو

## مسئلا حل ڪرڻ {#troubleshooting}

### پريويو اپڊيٽ نٿو ٿئي {#preview-not-updating}

- صفحو ريفريش ڪريو
- برائوزر ڪيش صاف ڪريو
- ٻيو برائوزر آزمائيو
- انٽرنيٽ ڪنيڪشن چيڪ ڪريو

### رنگ مختلف نظر اچن ٿا {#colors-look-different}

- مانيٽر جي رنگ سيٽنگون چيڪ ڪريو
- مختلف ڊوائيسز تي آزمائيو
- رسائي وارن اوزارن سان رنگ ڪنٽراسٽ جي تصديق ڪريو
- رنگن جي انڌائپ سميوليٽرز تي غور ڪريو

### ٽائپوگرافي جا مسئلا {#typography-issues}

- تصديق ڪريو ته فونٽ فائلون لوڊ ٿي رهيون آهن
- مختلف بريڪ پوائنٽس تي فونٽ سائيز چيڪ ڪريو
- اصل مواد سان آزمائيو
- پڙهڻ جي آساني لاءِ لائن جي ڊيگهه تي غور ڪريو
