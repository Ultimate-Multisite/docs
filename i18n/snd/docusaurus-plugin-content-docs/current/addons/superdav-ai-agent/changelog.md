# تبديلي نامو {#changelog}

## 1.18.0 — 2026-06-29 تي جاري ٿيو {#1180--released-on-2026-06-29}

### نئون {#new}

- **Google Calendar اوزار** — شيڊول-واقف خودڪارين لاءِ ترتيب ڏنل ڪئلينڊر ۽ واقعا پڙهو
- **رابطن جي ميپنگ ۽ شرڪت ڪندڙ مددگار** — ڪئلينڊر شرڪت ڪندڙن کي سائيٽ استعمال ڪندڙن ۽ رابطن سان ملائي ٿو
- **انساني منظوري جا دروازا ۽ ياد ڏياريندڙ رڪارڊ** — جائزي لاءِ خودڪارين کي روڪيو ۽ ورجائتي اطلاعن کان بچو
- **TextBee SMS مهيا ڪندڙ** — TextBee ذريعي ترتيب ڏنل متن-پيغام اطلاع موڪليو
- **ترقي يافته ساٿي پيڪيج** — قابل اعتماد-ڊولپر فائل سسٽم، ڊيٽابيس، WP-CLI، REST ڊسپيچر، plugin بلڊر، git سنيپ شاٽ، استعمال ڪندڙ-انتظام، ۽ بينچ مارڪ اوزار شامل ڪريو جيڪي WordPress.org بلڊ کان الڳ ورهايا ويا آهن

### بهتر ڪيو ويو {#improved}

- **منظم Superdav خدمت سيٽ اپ** — سپورٽ ٿيل سائيٽن لاءِ ميزباني ڪيل خدمت endpoints ۽ خودڪار ڪنيڪشن فراهمي شامل ڪريو
- **رليز پيڪيجنگ** — الڳ core ۽ Advanced ZIPs ٺاهيو، ٻئي GitHub تي شايع ڪريو، ۽ صرف core پيڪيج WordPress.org ڏانهن موڪليو

### درست ڪيو ويو {#fixed}

- **AI درخواست جي ڀروسي جوڳي هجڻ** — ماڊل چونڊ، درخواست ٽائيم آئوٽس، ڊفالٽ سيٽ اپ ماڊل، reasoning-text سنڀال، ۽ غلط اوزار ڪالن لاءِ ٻيهر ڪوشش جي رهنمائي بهتر ڪريو
- **ڪئلينڊر ۽ ياد ڏياريندڙ مضبوط ڪرڻ** — Google Calendar tokens ۽ ياد ڏياريندڙ deduplication کي مضبوط ڪريو
- **شروعاتي رهنمائي ۽ منظوري ٻيهر شروع ٿيڻ** — frontend شروعاتي رهنمائي لانچز ۽ تصديق ٿيل ability ٻيهر شروع ٿيڻ درست ڪريو
- **WordPress.org پيڪيجنگ جائزو مسئلا** — core رليز لاءِ پيڪيجنگ جائزي جي راءِ کي حل ڪريو

## 1.16.0 — 2026-05-20 تي جاري ٿيو {#1160--released-on-2026-05-20}

### نئون {#new-1}

- **Logo SVG ٺاهڻ جي ability** — Theme Builder هاڻي namespace-محفوظ صفائي سان ڪسٽم لوگو SVGs ٺاهي ۽ embed ڪري سگهي ٿو
- **دريافتي انٽرويو ۾ تصوير اپلوڊ** — Theme Builder دريافتي انٽرويو ۾ هاڻي وڌيڪ ڀرپور ڊزائن تناظر لاءِ تصوير اپلوڊ قدم شامل آهي
- **Palette Contrast تصديق ڪرڻ جي ability** — theme تي لاڳو ڪرڻ کان اڳ WCAG پيروي لاءِ رنگن جي جوڙن کي چيڪ ڪريو
- **مهمان نوازي جا مينيو** — Theme Builder هاڻي مهمان نوازي جي ڪاروبارن لاءِ منظم کاڌي ۽ مشروب مينيو صفحا ٺاهي سگهي ٿو
- **ڊيسڪ ٽاپ ۽ موبائل پيش منظر رينڊرنگ** — ڊزائن-رخ چونڊ دوران پنهنجي ڊزائن کي ڊيسڪ ٽاپ ۽ موبائل ڊوائيسز تي پيش منظر ۾ ڏسو
- **نيويگيشن ليبل پيراميٽر** — Create Menu ability هاڻي صفحي جي عنوان کان الڳ هڪ جدا `navigation_label` کي سپورٽ ڪري ٿي
- **Tier 1 اوزار دستيابي** — sd-ai-agent/site-scrape هاڻي Tier 1 اوزار آهي جيڪو Theme Builder ۾ ڊفالٽ طور دستياب آهي

- **Plugin قطار عمل لنڪس** — وضاحت لاءِ درست ۽ نالا تبديل ڪيا ويا

## 1.10.0 — 2026-05-05 تي جاري ٿيو {#1100--released-on-2026-05-05}

### نئون {#new-2}

- **Tavily انٽرنيٽ ڳولا** — Brave Search سان گڏ وڌيڪ ڀرپور انٽرنيٽ ڳولا نتيجن لاءِ Tavily کي ڳولا مهيا ڪندڙ طور شامل ڪريو

- **Scalar اوزار نتيجا** — هاڻي AI ڏانهن واپس ڪرڻ کان اڳ صحيح طور wrapped ڪيا وڃن ٿا
