# Superdav AI Agent

Superdav AI Agent هڪ ترقي يافته WordPress AI مددگار آهي جيڪو Gratis AI Agent کي theme-آگاهه صلاحيتن، بهتر ڪيل انٽرنيٽ ڳولا جي قابليتن، site builder خاصيتن، ۽ اختياري خودڪارڪاري انٽيگريشنز سان وڌائي ٿو. نسخو 1.18.0 Google Calendar پڙهڻ جا اوزار، رابطا ۽ شرڪت ڪندڙ ميپنگ، انساني منظوري جا دروازا، ياد ڏياريندڙن جي ٻيهر نه ورجاءُ، TextBee SMS اطلاع، منظم Superdav خدمت provisioning، ۽ ڀروسي لائق developer workflows لاءِ الڳ ورهايل Advanced ساٿي package شامل ڪري ٿو.

## اهم خاصيتون

- **Theme-آگاهه صلاحيتون** — توهان جي فعال WordPress theme ۽ نصب ٿيل plugins جي بنياد تي رهنمائي ۽ قابليتون پاڻمرادو ترتيب ڏئي ٿو
- **گهڻا انٽرنيٽ ڳولا فراهم ڪندڙ** — وڌيڪ ڀرپور ڳولا نتيجن ۽ ترقي يافته تحقيق جي قابليتن لاءِ Brave Search ۽ Tavily مان چونڊيو
- **Site builder رابطا forms** — پنهنجي نصب ٿيل form plugin استعمال ڪندي chat interface مان سڌو مڪمل طور ڪم ڪندڙ رابطا forms ٺاهيو
- **بهتر ڪيل site builder** — بهتر plugin دريافت ۽ غلطي بحالي سان سڌريل گهڻ-مرحلا site build منصوبا
- **بلٽ-اِن theme صلاحيتون** — Block Themes، Classic Themes، Kadence Blocks، ۽ Kadence Theme لاءِ خاص رهنمائي
- **بهتر WooCommerce انٽيگريشن** — وڌيڪ ڀروسي ۽ مطابقت لاءِ native WooCommerce APIs استعمال ڪرڻ لاءِ ٻيهر ترتيب ڏنل
- **خودڪار فراهم ڪندڙ سڃاڻپ** — جڏهن plugins ۽ themes فعال يا غير فعال ڪيا وڃن ته موجود plugins ۽ themes کي پاڻمرادو سڃاڻي ۽ تازو ڪري ٿو
- **ڪسٽم logo ٺاهڻ** — خودڪار namespace-محفوظ صفائي سان ڪسٽم SVG logos ٺاهيو ۽ شامل ڪريو
- **Palette contrast تصديق** — پنهنجي theme تي لاڳو ڪرڻ کان اڳ WCAG رسائي مطابقت لاءِ رنگن جي جوڙن کي چيڪ ڪريو
- **تصويرن سان بهتر دريافت** — Theme Builder discovery interview دوران ڊزائن جي رخ تي اثر وجهڻ لاءِ تصويرون upload ڪريو
- **مهمان نوازي menus** — restaurants، cafes، ۽ مهمان نوازي ڪاروبارن لاءِ جوڙجڪ وارا کاڌي ۽ مشروبات menu صفحا ٺاهيو
- **Desktop ۽ mobile previews** — design-direction چونڊ دوران desktop ۽ mobile devices تي پنهنجي ڊزائن جو preview ڏسو
- **Tier 1 site scraping** — sd-ai-agent/site-scrape هاڻي مقابلي ڪندڙ ۽ inspiration websites جي تجزيي لاءِ default طور موجود آهي
- **Google Calendar پڙهڻ جا اوزار** — ترتيب ڏنل calendars ۽ events پڙهو ته جيئن automations ايندڙ schedule context بابت سوچ ويچار ڪري سگهن
- **رابطا ۽ شرڪت ڪندڙ ميپنگ** — follow-up ڪم ٺاهڻ کان اڳ calendar شرڪت ڪندڙن کي سڃاتل site users ۽ contacts سان ملايو
- **انساني منظوري جا دروازا** — پيغام موڪلڻ، ڪم ٻيهر شروع ڪرڻ، يا تبديليون ڪرڻ کان اڳ حساس automations کي جائزي لاءِ روڪيو
- **ياد ڏياريندڙ records ۽ ٻيهر نه ورجاءُ** — اهي ياد ڏياريندڙ ٽريڪ ڪريو جيڪي اڳ ۾ موڪليا ويا آهن ته جيئن ورجائيندڙ automation runs ساڳئي وصول ڪندڙ کي ٻه ڀيرا اطلاع نه ڏين
- **TextBee SMS اطلاع** — جڏهن SMS فراهم ڪندڙ فعال هجي ته TextBee ذريعي ترتيب ڏنل text-message اطلاع موڪليو
- **منظم Superdav خدمت provisioning** — manual endpoint configuration بدران خودڪار setup سان حمايت ٿيل sites کي hosted Superdav endpoints سان ڳنڍيو
- **Advanced ساٿي package** — filesystem، database، WP-CLI، REST dispatcher، plugin builder، git snapshot، user-management، ۽ benchmark workflows لاءِ الڳ ورهايل ڀروسي لائق developer اوزار

## گهرجون

- WordPress 6.0 يا ان کان مٿي
- PHP 8.1 يا ان کان مٿي
- WordPress Multisite (network-activated) يا single-site installation
- هڪ فعال AI provider connector plugin (Anthropic، OpenAI، يا compatible)
- Gratis AI Agent v1.9.0 يا ان کان مٿي (Superdav AI Agent، Gratis AI Agent کي وڌائي ٿو)

## Installation

1. plugin files کي `/wp-content/plugins/superdav-ai-agent` تي upload ڪريو
2. WordPress ۾ **Plugins** screen ذريعي plugin کي activate ڪريو
3. پڪ ڪريو ته Gratis AI Agent به activated آهي
4. **Gratis AI Agent → Settings** ڏانهن وڃو ۽ ضرورت مطابق پنهنجو internet search provider، managed Superdav service connection، Google Calendar، يا TextBee credentials configure ڪريو

WordPress.org package ۾ بنيادي Superdav AI Agent خاصيتون شامل آهن. Advanced ساٿي package انهن environments لاءِ الڳ build ۽ distribute ڪيو ويندو آهي جتي ڀروسي لائق developer tools واضح طور install ۽ authorise ڪيا ويندا آهن.

## تڪڙو آغاز

فعال ٿيڻ کان پوءِ، Superdav AI Agent توهان جي Gratis AI Agent installation کي theme-آگاهه صلاحيتن ۽ اضافي قابليتن سان پاڻمرادو بهتر ڪري ٿو. WordPress admin sidebar ۾ **Gratis AI Agent** panel کوليو ۽ نيون خاصيتون استعمال ڪرڻ شروع ڪريو.

**مثالي هدايتون:**

```
Contact صفحي تي contact form شامل ڪريو.
```

```
تازين WordPress security best practices لاءِ web ڳوليو.
```

```
فعال theme جي block patterns استعمال ڪندي hero section ٺاهيو.
```

```
سڀاڻي جي ترتيب ڏنل Google Calendar events جو جائزو وٺو ۽ شرڪت ڪندڙن لاءِ reminders جو مسودو ٺاهيو.
```

```
جڏهن مان message approve ڪريان ته SMS reminder موڪليو.
```

## Theme-آگاهه صلاحيتون

Superdav AI Agent توهان جي فعال theme ۽ نصب ٿيل plugins کي پاڻمرادو سڃاڻي ٿو، ۽ مناسب theme-آگاهه صلاحيت کي مددگار جي context ۾ load ڪري ٿو. موجود صلاحيتن ۾ شامل آهن:

| صلاحيت | ڪهڙن تي لاڳو ٿئي ٿي |
|---|---|
| **Block Themes** | WordPress block-based (Full Site Editing) architecture استعمال ڪندڙ themes |
| **Classic Themes** | PHP templates ۽ classic editor استعمال ڪندڙ روايتي WordPress themes |
| **Kadence Blocks** | اهي sites جن ۾ Kadence Blocks plugin فعال آهي |
| **Kadence Theme** | اهي sites جن ۾ Kadence theme فعال theme طور آهي |

هر صلاحيت بابت تفصيلي رهنمائي لاءِ [Theme-آگاهه صلاحيتون](../../user-guide/configuration/theme-aware-skills) ڏسو.

## انٽرنيٽ ڳولا فراهم ڪندڙ

Superdav AI Agent v1.10.0، internet search queries لاءِ Brave Search جي متبادل طور **Tavily** شامل ڪري ٿو. ٻئي فراهم ڪندڙ real-time search results پيش ڪن ٿا ۽ **Gratis AI Agent → Settings → Advanced** ۾ configure ڪري سگهجن ٿا.

| مهيا ڪندڙ | خاصيتون | سيٽ اپ |
|---|---|---|
| **Brave Search** | تيز، رازداري تي ڌيان ڏيندڙ ڳولا جا نتيجا | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | وڌيڪ ڀرپور ڳولا جا نتيجا، ترقي يافته تحقيق جون صلاحيتون | [Tavily API](https://tavily.com/) |

ترتيب جي هدايتن لاءِ [انٽرنيٽ ڳولا](../../user-guide/configuration/internet-search) ڏسو.

## ڪئلينڊر، منظوري، ۽ اطلاع ورڪ فلو

Superdav AI Agent v1.18.0 خودڪاري ورڪ فلو کي مواد ۽ سائٽ ٺاهڻ وارن ڪمن کان اڳتي وڌائي ٿو:

- **Google Calendar اوزار** agent کي ترتيب ڏنل ڪئلينڊر ۽ واقعي جا تفصيل پڙهڻ ڏين ٿا جڏهن ڪئلينڊر سندون موجود هجن.
- **رابطي ۽ شرڪت ڪندڙن جا مددگار** ياد ڏيارڻ يا پٺيان ڪارروائي وارا ڪم تيار ڪرڻ کان اڳ واقعي جي شرڪت ڪندڙن کي WordPress استعمال ڪندڙن يا ڄاتل رابطن سان ملائين ٿا.
- **انساني منظوري جا دروازا** هڪ خودڪاري کي روڪڻ ڏين ٿا جيستائين ڪو بااختيار استعمال ڪندڙ تجويز ڪيل عمل جو جائزو نه وٺي.
- **ياد ڏياريندڙ رڪارڊ** جڏهن خودڪاري ٻيهر ڪوشش ڪئي وڃي يا بار بار شيڊول ڪئي وڃي ته نقل ڪئلينڊر يا اطلاع ياد ڏياريندڙن کي روڪين ٿا.
- **TextBee SMS سپورٽ** جڏهن TextBee مهيا ڪندڙ ترتيب ڏنل هجي ته ٽيڪسٽ-پيغام اطلاعن کي فعال ڪري ٿي.

اهي اوزار شيڊول کان باخبر مدد، شرڪت ڪندڙن جي پٺيان ڪارروائي، ۽ ياد ڏياريندڙ ورڪ فلو لاءِ استعمال ڪريو، جن ۾ حساس پيغام موڪلڻ کان اڳ اڃا به انساني جائزو گهربل هجي.

## سائٽ بلڊر رابطي جا فارم

سائٽ بلڊر agent هاڻي چيٽ انٽرفيس مان سڌو رابطي جا فارم ٺاهي سگهي ٿو. بس سائٽ بلڊر کي رابطي جو فارم ٺاهڻ لاءِ چئو، ۽ اهو پاڻمرادو توهان جو انسٽال ٿيل فارم plugin (Contact Form 7، WPForms، Fluent Forms، يا Gravity Forms) سڃاڻيندو ۽ استعمال لاءِ تيار فارم ٺاهيندو.

تفصيلي رهنمائي لاءِ [سائٽ بلڊر رابطي جا فارم](../../user-guide/administration/site-builder-contact-forms) ڏسو.

## ترتيب

سڀ سيٽنگون WordPress ايڊمن ۾ **Gratis AI Agent → Settings** هيٺ آهن. Superdav AI Agent هيٺيان ترتيب جا اختيار شامل ڪري ٿو:

| سيٽنگ | وضاحت |
|---|---|
| **Internet Search Provider** | Brave Search ۽ Tavily مان چونڊيو (اختياري) |
| **Tavily API Key** | Tavily انٽرنيٽ ڳولا لاءِ API key (جيڪڏهن Tavily چونڊيو ويو آهي) |
| **Managed Superdav Service** | خودڪار endpoint فراهمي سان سپورٽ ڪيل سائٽن کي hosted Superdav خدمت سان ڳنڍيو |
| **Google Calendar Credentials** | ڪئلينڊر پهچ ترتيب ڏيو ته جيئن agent ڪئلينڊر ۽ واقعا پڙهي سگهي |
| **TextBee SMS Provider** | منظور ٿيل SMS اطلاع ورڪ فلو لاءِ TextBee سندون ترتيب ڏيو |

## ترقي يافته ساٿي پيڪيج

Superdav AI Agent v1.18.0 ڀروسي لائق developer ورڪ فلو لاءِ الڳ ورهايل ترقي يافته ساٿي پيڪيج شامل ڪري ٿو. اهي صلاحيتون WordPress.org بنيادي پيڪيج ۾ گڏ شامل نه آهن. ساٿي پيڪيج رڳو انهن سائٽن تي انسٽال ڪريو جتي منتظم واضح طور agent تي developer tooling تائين پهچ لاءِ ڀروسو ڪن ٿا.

ترقي يافته اوزارن ۾ filesystem عمل، database پهچ، WP-CLI، REST dispatcher ورڪ فلو، plugin builder utilities، git snapshots، استعمال ڪندڙ-انتظام مددگار، ۽ benchmark tooling شامل آهن. انهن اوزارن کي ڀروسي لائق منتظمن تائين محدود رکو ۽ production سائٽن تي انهن کي فعال ڪرڻ کان اڳ approval-gate ترتيب جو جائزو وٺو.

- event automation درست ڪئي ته انهن سائيٽن کي سٺي نموني سنڀالي جتي automation tables اڃا ٺاهيا نه ويا آهن
- `memory-save` صلاحيت درست ڪئي جيڪا system instruction builder ۾ صحيح namespace prefix استعمال ڪري ٿي
- scalar tool نتيجا درست ڪيا ته AI ڏانهن واپس موٽائڻ کان اڳ صحيح نموني wrap ٿين
- استعمال جا انگ اکر درست ڪيا ته پراڻن نسخن مان upgrade تي legacy ability key format کي صحيح نموني سنڀالن
