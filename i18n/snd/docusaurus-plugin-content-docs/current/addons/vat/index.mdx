# VAT

Multisite Ultimate: يورپي VAT

يورپي VAT کي بهتر سهارو ڏيڻ لاءِ Multisite Ultimate جي ڊفالٽ ٽيڪس سنڀال ۾ اضافي خاصيتون ۽ اوزار شامل ڪري ٿو.
خودڪار طور EU VAT شرحون ٽيڪس شرحن واري صفحي ۾ شامل ڪري ٿو، مان

## انسٽاليشن {#installation}

1. 'ultimate-multisite-vat' کي '/wp-content/plugins/' ڊائريڪٽري ۾ اپلوڊ ڪريو
2. WordPress ۾ 'پلگ انز' مينيو ذريعي plugin کي فعال ڪريو
3. Ultimate Multisite > سيٽنگون > ٽيڪسز ڏانهن وڃو ۽ EU VAT فعال ڪريو.
4. ٽيڪس شرحن تي ڪلڪ ڪريو ۽ هيٺان EU VAT شرحون اپڊيٽ ڪريو ۽ ٽيڪس شرحون محفوظ ڪريو.
5. ٿي ويو!
