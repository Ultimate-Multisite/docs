# Mailchimp

Multisite Ultimate: Mailchimp انضمام

پنهنجي Multisite Ultimate نيٽ ورڪ کي پنهنجي Mailchimp سان ضم ڪريو. توهان ڪجهه منصوبن کي Mailchimp فهرستن سان ڳنڍي سگهو ٿا ۽ سائن اپ تي سبسڪرائبرن کي پاڻمرادو انهن فهرستن ۾ شامل ڪري سگهو ٿا.

## انسٽاليشن

1. 'ultimate-multisite-mailchimp' کي '/wp-content/plugins/' ڊائريڪٽري ۾ اپلوڊ ڪريو
2. پنهنجي WordPress Network Admin ۾ 'Plugins' مينيو ذريعي پلگ ان کي فعال ڪريو

## استعمال جي رهنمائي

پنهنجن سبسڪرائبرن جون اي ميلون توهان جي MailChimp فهرستن سان هم وقت ڪرڻ اسان جي Ultimate Multisite: MailChimp Integration Add-on سان واقعي آسان آهي. هن رهنمائي جو مقصد توهان کي سڀ ڪجهه سيٽ ڪرڻ ۾ مدد ڪرڻ آهي!

## API key حاصل ڪرڻ

ان کان اڳ جو اسين موجود مختلف سيٽنگن ۾ وڃون، اسان کي MailChimp API key جي ضرورت آهي. هي Ultimate Multisite کي MailChimp API سان ڳالهائڻ جي اجازت ڏيندو ته جيئن فهرستون، گروپ، ۽ اي ميل پتا حاصل ڪري، ۽ جڏهن يوزر سائن اپ ڪن ته انهن اي ميل پتن کي انهن فهرستن ۽ گروپن ۾ شامل ڪري.

پنهنجي MailChimp API key حاصل ڪرڻ لاءِ هن رهنمائي تي عمل ڪريو: [API Keys بابت | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

## انضمام موڊ

MailChimp Integration add-on ٻه “انضمام موڊ” پيش ڪري ٿو، جنهن جو مطلب آهي ته اي ميل پتن کي هم وقت ڪرڻ لاءِ توهان add-on کي ٻن مختلف طريقن سان استعمال ڪري سگهو ٿا.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_وڌيڪ لچڪ لاءِ ٻه انضمام موڊ!_

### موڊ 1: گهڻيون فهرستون

ڊفالٽ موڊ **گهڻيون فهرستون** استعمال ڪري ٿو. هي موڊ توهان کي هر منصوبي لاءِ گهڻيون MailChimp فهرستون چونڊڻ جي اجازت ڏيندو. جيڪڏهن توهان پنهنجن يوزرن کي مختلف فهرستن ۾ ورهايو ٿا ته هي بهترين اختيار آهي.

مثال طور، فرض ڪريو ته توهان جي MailChimp Account تي 3 مختلف فهرستون آهن: **خبرنامو** ، **منصوبو A** ، ۽ **منصوبو B**. توهان چاهيو ٿا ته سڀ يوزر، انهن جي منصوبي کان قطع نظر، خبرنامي جي فهرست ۾ پڻ شامل هجن، ته جيئن اهي توهان جون تازه ڪاريون نه وڃائين.

اهو حاصل ڪرڻ لاءِ، توهان Multiple Lists موڊ استعمال ڪري سگهو ٿا ته جيئن Plan A MailChimp Settings ۾ ٻئي فهرستون **خبرنامو** ۽ **منصوبو A** شامل ڪيون وڃن، ۽ Plan B MailChimp Settings ۾ **خبرنامو** ۽ **منصوبو B** شامل ڪيون وڃن.

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_نوان اڪائونٽن ۽ منسوخ ٿيل اڪائونٽن لاءِ ڊفالٽ فهرستون چونڊيو_

### موڊ 2: هڪ فهرست + گهڻا گروپ

MailChimp توهان کي پنهنجي فهرست کي يوزرن جي ننڍن گروپن ۾ ورهائڻ جي اجازت ڏئي ٿو، جن کي **گروپ** چيو وڃي ٿو. هي توهان کي وڌيڪ ڪنٽرول ڏئي ٿو، ڇاڪاڻ ته توهان پنهنجي سبسڪرائبرن جي ڪنهن مخصوص حصي ڏانهن مهم موڪلي سگهو ٿا، بغير MailChimp تي ساڳي مهم جون گهڻيون ڪاپيون ٺاهڻ جي (جنهن سان پوءِ ان مهم جا انگ اکر تجزيو ڪرڻ ڏکيو ٿي پوي ٿو).

مثال طور، توهان وٽ **منهنجي نيٽ ورڪ جا سبسڪرائبر،** نالي هڪ ئي فهرست ٿي سگهي ٿي، ۽ ان جي اندر **منصوبو A** ۽ **منصوبو B** لاءِ گروپ هجن.

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_جيڪڏهن “هڪ فهرست + گهڻا گروپ” انضمام موڊ استعمال ڪري رهيا آهيو ته ڊفالٽ فهرستون ۽ گروپ چونڊيو_

## ڊفالٽ فهرستون ۽ گروپ

جڏهن توهان چونڊي وٺو ته توهان جي خاص استعمال واري صورت لاءِ ڪهڙو انضمام موڊ وڌيڪ مناسب آهي، تڏهن ڊفالٽ فهرستون (۽ موڊ تي دارومدار رکندي گروپ) چونڊڻ جو وقت آهي. ڊفالٽ فهرست تڏهن استعمال ٿيندي جڏهن ڪنهن ڏنل منصوبي لاءِ ڪا مخصوص فهرست چونڊيل نه هوندي.

توهان منسوخ ٿيل اڪائونٽن لاءِ پڻ ڊفالٽ فهرست چونڊي سگهو ٿا. جڏهن به ڪو يوزر پنهنجي رڪنيت ختم ڪري ۽ پنهنجو اڪائونٽ توهان جي نيٽ ورڪ مان هٽائي، ان جو اي ميل پتو منسوخ ٿيل اڪائونٽن لاءِ مقرر ڪيل فهرست ڏانهن منتقل ڪيو ويندو.

## ٻٽي رڪنيت فعال ڪرڻ

ساڳئي سيٽنگن واري صفحي تي، توهان **ٻٽي رڪنيت** فعال ڪري سگهندا. هي اختيار يوزر کي چونڊيل ڊفالٽ فهرست ۽ ان يوزر جي منصوبي لاءِ چونڊيل فهرست ٻنهي ۾ شامل ڪندو. جيڪڏهن هي اختيار فعال نه آهي، ته يوزر صرف يوزر جي منصوبي لاءِ چونڊيل فهرست ۾ شامل ٿيندو. جيڪڏهن ان خاص منصوبي لاءِ ڪا فهرست مقرر نه آهي، ته پوءِ يوزر چونڊيل ڊفالٽ فهرست ۾ شامل ٿيندو.

<!-- Screenshot unavailable: MailChimp double subscription toggle setting -->

_توهان سائن اپ تي پنهنجن يوزرن کي ڊفالٽ فهرست ۽ منصوبي جي فهرست ٻنهي ۾ سبسڪرائب ڪرائي سگهو ٿا_

## هر منصوبي لاءِ فهرستون ۽ گروپ چونڊڻ

پنهنجن **منصوبن** ڏانهن وڃو ۽ اهو منصوبو چونڊيو جنهن کي توهان ايڊٽ ڪرڻ چاهيو ٿا. نوٽ ڪريو ته **Advanced Options** meta-box تي MailChimp نالي هڪ نئون ٽيب موجود هوندو.

ان ٽيب تي، توهان ان منصوبي لاءِ گهڻيون فهرستون يا هڪ فهرست ۽ گهڻا گروپ (انضمام موڊ تي دارومدار رکندي) چونڊي سگهندا.

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_اهي فهرستون ۽ گروپ چونڊيو جيڪي توهان پنهنجن هر منصوبي لاءِ استعمال ڪرڻ چاهيو ٿا_

جڏهن ڪو نئون يوزر ان منصوبي لاءِ سائن اپ ڪندو، ته ان جو اي ميل پتو چونڊيل فهرستن ۽ گروپن ۾ شامل ڪيو ويندو!
