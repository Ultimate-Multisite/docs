# WooCommerce انٽيگريشن {#woocommerce-integration}

WooCommerce انٽيگريشن بلڪل ساڳئي طريقي سان ڪم ڪري ٿي جيئن اسان جو دستي گيٽ وي ڪم ڪري ٿو. اها توهان جي واهپيدارن کي تمام گهڻي لچڪ ڏئي ٿي، جنهن سان اهي توهان جي WooCommerce انسٽال تي موجود ڪنهن به ادائيگي فارم کي چونڊي سگهن ٿا. ۽ اهو سادو آهي: جيئن ئي واهپيدار جي بلنگ مدت ختم ٿئي ٿي، واهپيدار جي رڪنيت روڪي وڃي ٿي. ساڳئي وقت، نيٽ ورڪ جي مکيه سائيٽ جي WooCommerce انسٽال تي هڪ آرڊر ٺاهيو ويندو. پوءِ واهپيدار کي پنهنجي اي ميل ۾ انوائس ملي ٿي، جنهن ۾ ادائيگي ڪرڻ لاءِ لنڪ هوندي آهي. جيئن ئي ادائيگي ملي ٿي، رڪنيت نئين ڪئي وڃي ٿي.

## خاصيتون {#features}

- Ultimate Multisite ۽ WooCommerce جي وچ ۾ آسان انٽيگريشن
- ڪيترائي ادائيگي طريقا موجود
- شامل ٿيل رڪنيتن جي بنيادي حصي کي استعمال ڪندي خودڪار بار بار بلنگ، اختياري WooCommerce Subscriptions plugin جي بهتري سان
- Checkout جا ڪل مقدار Ultimate Multisite ٽيڪسن کي الڳ، غير-ٽيڪس لائق WooCommerce فيس لائينن طور شامل ڪن ٿا، ته جيئن گراهڪ ادائيگي کان اڳ ٽيڪس رقمون جائزو وٺي سگهن

## دستاويز {#documentation}

**WooCommerce انٽيگريشن سيٽ اپ ڪرڻ**

**Ultimate Multisite سان پنهنجي ادائيگي اختيارن کي وڌائڻ: WooCommerce انٽيگريشن**

## انسٽاليشن {#installation}

1. 'ultimate-multisite-woocommerce' کي '/wp-content/plugins/' ڊائريڪٽري ۾ اپلوڊ ڪريو
2. WordPress ۾ 'Plugins' مينيو ذريعي plugin کي فعال ڪريو
3. plugin کي سيٽ اپ ڪرڻ لاءِ قدم بہ قدم Wizard جي پيروي ڪريو

## استعمال جي رهنمائي {#usage-guide}

_**اهم نوٽ: هي مضمون Ultimate Multisite ورزن 2.x بابت آهي.**_

_**ڌيان:** Ultimate Multisite: WooCommerce انٽيگريشن لاءِ ضروري آهي ته WooCommerce گهٽ ۾ گهٽ توهان جي مکيه سائيٽ تي فعال هجي._

اسان سمجهون ٿا ته _Stripe_ ۽ _PayPal_ ڪجهه ملڪن ۾ موجود نه آهن، جنهن سان Ultimate Multisite واهپيدارن لاءِ اسان جو plugin اثرائتي نموني استعمال ڪرڻ محدود يا ڏکيو ٿي پوي ٿو. تنهنڪري اسان **_WooCommerce_ کي انٽيگريٽ ڪرڻ لاءِ هڪ add-on** ٺاهيو.

_WooCommerce_ هڪ تمام مشهور eCommerce plugin آهي. سڄي دنيا جي ڊولپرن ان سان مختلف ادائيگي گيٽ ويز انٽيگريٽ ڪرڻ لاءِ add-ons ٺاهيا. اسان هن جو فائدو وٺي Ultimate Multisite بلنگ سسٽم ۾ استعمال ٿيندڙ ادائيگي گيٽ ويز کي وڌايو.

## ان کي سيٽ اپ ڪرڻ {#setting-it-up}

پهريان، توهان کي WooCommerce Integration add-on انسٽال ڪرڻو پوندو، جيڪو **لائسنس يافته واهپيدارن لاءِ مفت** آهي. ان کي انسٽال ڪرڻ لاءِ، پنهنجي سپر ايڊمن Dashboard تي Ultimate Multisite تي ڪلڪ ڪريو ۽ **Add-ons** چونڊيو.

![Add-ons صفحو](/img/admin/addons-page.png)

هتي توهان سڀ Ultimate Multisite add-ons ڳولي سگهو ٿا. **Ultimate Multisite: WooCommerce انٽيگريشن** add-on تي ڪلڪ ڪريو.

![WooCommerce انٽيگريشن ڏيکاريندڙ Add-ons صفحو](/img/admin/addons-page.png)

add-on جي تفصيلن سان هڪ ونڊو ظاهر ٿيندي. بس **Install Now** تي ڪلڪ ڪريو *.*

انسٽاليشن مڪمل ٿيڻ کان پوءِ، توهان کي plugins صفحي ڏانهن ريڊائريڪٽ ڪيو ويندو. هتي، بس **Network Activate** تي ڪلڪ ڪريو ۽ _WooCommerce_ add-on توهان جي نيٽ ورڪ تي فعال ٿي ويندو.

![WooCommerce فعال ٿيل Ultimate Multisite ادائيگي گيٽ وي سيٽنگون](/img/addons/woocommerce-payment-gateway.png)

توهان ڪجهه بنيادي سيٽنگون ترتيب ڏئي سگهو ٿا، جيئن ڏيک نالو جيڪو front end ۾ ظاهر ٿيندو ۽ توهان جي واهپيدارن کي انهن جي checkout وهڪري ۾ رهنمائي ڪرڻ لاءِ هدايتون. پوءِ توهان پنهنجي **مکيه سائيٽ جي _WooCommerce_ سيٽنگن** ڏانهن وڃي سگهو ٿا ته جيئن توهان جيڪو ادائيگي گيٽ وي استعمال ڪرڻ چاهيو ٿا، ان کي فعال ۽ ترتيب ڏئي سگهو.

### Checkout ٽيڪس فيس لائينون {#checkout-tax-fee-lines}

جڏهن Ultimate Multisite checkout ۾ ٽيڪس شامل هجن، انٽيگريشن هر Ultimate Multisite ٽيڪس رقم کي WooCommerce checkout ۾ الڳ، غير-ٽيڪس لائق فيس لائين طور شامل ڪري ٿي. اهو ادائيگي کان اڳ ٽيڪس جو ڪل مقدار ظاهر رکندو آهي، جڏهن ته WooCommerce کي ٽيڪس فيس تي ٻيهر ٽيڪس لڳائڻ کان روڪيندو آهي.

## ادائيگي کي ڪيئن منظم ڪجي {#how-to-manage-the-payment}

WooCommerce انٽيگريشن بلڪل ساڳئي طريقي سان ڪم ڪري ٿي جيئن اسان جو **دستي گيٽ وي** ڪم ڪري ٿو. نقصان اهو آهي ته هر ادائيگي بلنگ مدت جي آخر ۾ هٿ سان ادا ڪرڻي پوندي. وهڪرو هن ريت ڪم ڪري ٿو:

  * جيئن ئي واهپيدار جو بلنگ چڪر ختم ٿئي ٿو، رڪنيت **روڪيل** ڪئي وڃي ٿي. ساڳئي وقت، توهان جي نيٽ ورڪ جي مکيه سائيٽ تي _WooCommerce_ انسٽال ۾ هڪ آرڊر ٺاهيو ويندو.

  * پوءِ واهپيدارن کي پنهنجي اي ميل ۾ انوائس ملي ٿي، جنهن ۾ ادائيگي ڪرڻ لاءِ لنڪ هوندي آهي (اهو لنڪ انهن جي subsite Dashboard مان انهن جي بلنگ تاريخ ۾ پڻ ظاهر ٿيندو آهي). اهو لنڪ واهپيدار کي _**WooCommerce**_ **checkout** ڏانهن ريڊائريڪٽ ڪري ٿو، جتي واهپيدار پنهنجي ادائيگي عمل ۾ آڻڻ لاءِ **موجود _WooCommerce_ گيٽ ويز** مان هڪ چونڊي سگهندا.

  * جيئن ئي ادائيگي ملي ٿي، رڪنيت **نئين** ڪئي وڃي ٿي، ادائيگي Ultimate Multisite تي لاگ ڪئي وڃي ٿي ۽ آرڊر مڪمل طور سيٽ ڪيو وڃي ٿو.

### ڪجهه اهم نوٽس: {#some-important-notes}

  * Ultimate Multisite جي ڊفالٽ ڪارڪردگي اڃا به هن انٽيگريشن فارم سان ڪم ڪري ٿي. مثال طور، **واپسيون جاري ڪرڻ** **رڪنيت انتظام** اسڪرين مان WooCommerce سان رابطو ڪندو ته چونڊيل ادائيگي فارم استعمال ڪندي واپسي جاري ڪئي وڃي.

  * Ultimate Multisite: WooCommerce انٽيگريشن ڪم ڪرڻ لاءِ WP-Cron تي ڀاڙي ٿي. هي گهڻي ڀاڱي تمام سٺو ڪم ڪري ٿو، پر جيڪڏهن سائيٽ تي ٽريفڪ گهٽ هجي ته پوءِ cron کي طئي ٿيل واقعن لاءِ چيڪ ڪرڻ جي ترغيب نٿي ملي ۽ واقعو رهجي وڃي ٿو، جنهن سان عملدرآمد ۾ ديرون ٿي سگهن ٿيون.

## WooCommerce Subscriptions Plugin {#woocommerce-subscriptions-plugin}

هر مهيني ادائيگي هٿ سان شروع ڪرڻ آخرين-استعمال ڪندڙن لاءِ مناسب ناهي، ڇاڪاڻ ته سبسڪرپشنز کان اميد هوندي آهي ته اهي هر بلنگ چڪر ۾ پاڻمرادو ٻيهر هلن. WooCommerce Integration add-on هاڻي بنيادي سبسڪرپشنز ڪوڊ کي گڏ ڪري ٿو، تنهنڪري ورجائيندڙ پراڊڪٽس هميشه ورجائيندڙ بلنگ لاءِ سبسڪرپشنز استعمال ڪن ٿا.

هن انٽيگريشن ذريعي ورجائيندڙ پراڊڪٽس وڪڻڻ لاءِ توهان کي الڳ [WooCommerce Subscriptions plugin](https://woocommerce.com/products/woocommerce-subscriptions/) خريد يا انسٽال ڪرڻ جي **ضرورت ناهي**. جيڪڏهن اهو plugin خريد، انسٽال، ۽ توهان جي مکيه سائيٽ تي موجود آهي، ته Ultimate Multisite ان کي استعمال ڪندو ۽ WooCommerce اضافي سبسڪرپشن مينيجمينٽ خاصيتون ظاهر ڪندو. جڏهن اهو موجود نه هوندو، ته گڏ ڪيل سبسڪرپشنز core اڃا به ورجائيندڙ پراڊڪٽس کي سنڀاليندو.

![WooCommerce سبسڪرپشن انٽيگريشن سيٽنگون](/img/addons/woocommerce-subscription-integration.png)

۽ بس ايترو ئي آهي. توهان جي آخرين-استعمال ڪندڙن کي ورجائيندڙ پراڊڪٽس لاءِ پنهنجي بلنگ چڪر جي آخر ۾ هاڻي هٿ سان ادائيگي ڪرڻ جي ضرورت نه پوندي.

توهان جي مکيه سائيٽ تي _WooCommerce_ هيٺ هڪ آرڊر ٺهڻ کان علاوه، اهو هر Account لاءِ هڪ سبسڪرپشن پڻ ٺاهيندو.

### ناڪام تجديد معطلي جو رويو {#failed-renewal-suspension-behavior}

ڊيفالٽ طور، ميمبرشپس فعال رهن ٿيون جڏهن WooCommerce Subscriptions پنهنجي ناڪام-ادائيگي ٻيهر ڪوشش واري شيڊول تي عمل ڪري ٿو. هي عام WooCommerce Subscriptions مهلت واري دري کي برقرار رکي ٿو، تنهنڪري بحال ٿيل تجديد سبسڪرپشن ۽ ميمبرشپ کي وقت کان اڳ رسائي روڪڻ کان سواءِ ٻيهر هم وقت ڪري سگهي ٿي.

اهي منتظم جيڪي وڌيڪ سخت رسائي ڪنٽرول چاهين ٿا، WooCommerce Integration سيٽنگن ۾ **Suspend Memberships Immediately on Failed Renewals** فعال ڪري سگهن ٿا. جڏهن هي opt-in سيٽنگ فعال هوندي، Ultimate Multisite لاڳاپيل ميمبرشپ کي جيئن ئي تجديد ادائيگي ناڪام ٿئي ٿي معطل ڪري ڇڏيندو، WooCommerce Subscriptions جي retry handling ختم ٿيڻ جو انتظار ڪرڻ بدران.

Developers ساڳئي فيصلي کي `wu_woo_suspend_on_payment_failure` filter سان override ڪري سگهن ٿا. Hook reference صفحا add-on source مان generate ٿين ٿا، تنهنڪري هي جائزو custom suspension policies لاءِ صرف filter جو نالو ظاهر ڪري ٿو.
