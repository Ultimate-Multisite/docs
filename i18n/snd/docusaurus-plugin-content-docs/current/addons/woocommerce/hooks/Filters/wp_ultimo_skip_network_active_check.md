# فلٽر: wp_ultimo_skip_network_active_check

ڊولپرن کي نيٽ ورڪ چالو ڪرڻ جي جاچ کي مختصر رستي سان ختم ڪرڻ جي اجازت ڏئي ٿو.

هي composer-based ۽ ٻين ڪسٽم جوڙجڪن، جهڙوڪ Bedrock، استعمال ڪرڻ وقت مفيد آهي، مثال طور، جتي پلگ انز کي mu-plugins طور استعمال ڪرڻ معمول آهي.

## پيرا ميٽر

| نالو | قسم | وضاحت |
|------|------|-------------|
| $skip_network_activation_check | `bool` | ڇا اسان کي جاچ ڇڏڻ گهرجي يا نه، ڊفالٽ false آهي. |

### کان

- 2.0.0
### ذريعو

[`inc/class-requirements.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-woocommerce/blob/main/inc/class-requirements.php#L272) ۾ لائن 272 تي بيان ٿيل


## موٽ
true جيڪڏهن توهان جاچ ڇڏڻ چاهيو ٿا، ٻي صورت ۾ false.
