## پيرا ميٽر {#parameters}

| نالو | قسم | وضاحت |
|------|------|-------------|
| $username | `string` | گراهڪ جو استعمال ڪندڙ نالو. |
| $email | `string` | نئين گراهڪ جو email address. |
| $new_user_args | `array` | نئين استعمال ڪندڙ جي args جو array، شايد پهريون ۽ آخري نالا شامل هجن. |
| $suffix | `string` | استعمال ڪندڙ نالي کي منفرد بڻائڻ لاءِ string شامل ڪريو. |
