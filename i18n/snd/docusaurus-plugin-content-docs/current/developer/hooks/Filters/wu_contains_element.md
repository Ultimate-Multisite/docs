# فلٽر: wu_contains_element {#filter-wucontainselement}

ڊولپرن کي شروعاتي ڳولا جا نتيجا بدلائڻ جي اجازت ڏئي ٿو.

هي ٽئين-ڌر جي بلڊرن ۽ اهڙين شين لاءِ ڪارائتو آهي.

## پيرا ميٽر {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $contains_elements | `bool` | جيڪڏهن عنصر مواد ۾ شامل هجي. |
| $content | `string` | اهو مواد جنهن جي جاچ ڪئي پئي وڃي. |
| $element | `self` | موجوده عنصر. |
| $post | `null\|\WP_Post` | جاچڻ لاءِ پوسٽ. |
