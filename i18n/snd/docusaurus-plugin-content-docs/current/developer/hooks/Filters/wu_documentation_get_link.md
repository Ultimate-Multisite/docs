# فلٽر: wu_documentation_get_link

plugin ٺاهيندڙن کي links کي فلٽر ڪرڻ جي اجازت ڏيو.

پڪ ناهي ته اهو ڪيئن فائديمند ٿي سگهي ٿو، پر اهو رکڻ ۾ ڪو نقصان ناهي

## پيرا ميٽر

| نالو | قسم | وضاحت |
|------|------|-------------|
| $link | `string` | رجسٽر ٿيل link |
| $slug | `string` | link حاصل ڪرڻ لاءِ استعمال ٿيل slug |
| $default_link | `string` | رجسٽر ٿيل default link |
