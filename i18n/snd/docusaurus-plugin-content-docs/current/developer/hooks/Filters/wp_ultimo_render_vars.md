# فلٽر: wp_ultimo_render_vars

پلگ اِن ڊولپرن کي رينڊر سياق ۾ عالمي طور اضافي variable شامل ڪرڻ جي اجازت ڏئي ٿو.

## پيرا ميٽر {#parameters}

| نالو | قسم | وضاحت |
|------|------|-------------|
| $args | `array` | اهڙو Array جنهن ۾ رينڊر ڪال ذريعي پاس ڪيل variables شامل آهن. |
| $view | `string` | رينڊر ٿيڻ واري ڏيک جو نالو. |
| $default_view | `string` | fallback_view جو نالو |
