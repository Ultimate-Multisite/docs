# فلٽر: wu_admin_notices {#filter-wuadminnotices}

ڊولپرز کي Ultimate Multisite پاران شامل ڪيل انتظامي نوٽيس فلٽر ڪرڻ جي اجازت ڏيو.

## پيرا ميٽر {#parameters}

| نالو | قسم | وضاحت |
|------|------|-------------|
| $notices | `array` | انهي خاص پينل لاءِ نوٽيسن جي فهرست. |
| $all_notices | `array` | شامل ڪيل نوٽيسن جي فهرست، پينل موجب الڳ ڪيل. |
| $panel | `string` | نوٽيس حاصل ڪرڻ لاءِ پينل. |
| $filter | `string` | جيڪڏهن dismissable نوٽيس فلٽر ڪري ڪڍيا ويا آهن. |
| $dismissed_messages | `array` | رد ڪيل نوٽيس ڪنجيون جي فهرست. |

- 2.0.0
