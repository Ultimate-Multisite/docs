# فلٽر: get_site_url_for_previewer

plugin ڊولپرن کي اجازت ڏيو ته previewer ۾ استعمال ٿيندڙ URL کي فلٽر ڪن

## پيرا ميٽر {#parameters}

| نالو | قسم | وضاحت |
|------|------|-------------|
| $domain | `string` | هن وقت استعمال ٿيندڙ ڊفالٽ ڊومين، تبديلين لاءِ ڪارائتو |
| $domain_options | `array` | Ultimate Multisite سيٽنگون -&gt; نيٽورڪ سيٽنگون -&gt; ڊومين اختيارن ۾ داخل ڪيل سڀني ڊومين اختيارن جي فهرست |

### کان وٺي {#since}

- 1.7.2

[`inc/checkout/class-legacy-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-legacy-checkout.php#L812) ۾ لائن 812 تي بيان ٿيل


## واپسيون {#returns}
استعمال ڪرڻ لاءِ نئون ڊومين
