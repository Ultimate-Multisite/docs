# Filter: wu_magic_link_enforce_user_agent

فلٽر ڪريو ته ڇا user agent جي تصديق لاڳو ڪئي وڃي.

false تي سيٽ ڪريو ته جيئن ٽوڪنز مختلف برائوزرن/ڊوائيسز تي ڪم ڪري سگهن. هي سيڪيورٽي گهٽائي ٿو پر استعمال جي آساني وڌائي ٿو.

## پيرا ميٽرز

| نالو | قسم | وضاحت |
|------|------|-------------|
| $enforce | `bool` | ڇا user agent جي ملائڻ کي لاڳو ڪيو وڃي. |
