# عمل: wu_page_load {#action-wupageload}

پلگ اِن ڊولپرز کي اسان جي صفحن ۾ اضافي هوڪس شامل ڪرڻ جي اجازت ڏيو.

## پيرا ميٽر {#parameters}

| نالو | قسم | وضاحت |
|------|------|-------------|
| $id | `string` | هن صفحي جي ID. |
| $page_hook | `string` | هن صفحي جو صفحو هوڪ. |
| $admin_page | `self` | صفحي جو instance. |

### کان وٺي {#since}

- 1.8.2
- 2.0.4: ٽيون پيرا ميٽر شامل ڪيو ويو: صفحي جو instance.
