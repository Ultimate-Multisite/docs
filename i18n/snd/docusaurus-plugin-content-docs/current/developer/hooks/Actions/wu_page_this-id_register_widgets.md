# Action: wu_page_\{$this->id\}_register_widgets {#action-wupagethis-idregisterwidgets}

هن صفحي لاءِ widgets رجسٽر ٿيڻ کان پوءِ هلندو آهي.

The dynamic portion of the hook name, `$this-&gt;id`, refers to the page id.

## پيراميٽر {#parameters}

| نالو | قسم | وضاحت |
|------|------|-------------|
| $id | `string` | صفحي جو id. |
| $page_hook | `string` | صفحي جو hook. |
| $page | `object` | صفحي جو object. |
