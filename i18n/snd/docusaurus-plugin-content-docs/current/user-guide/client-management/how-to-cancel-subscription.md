# میں اپنی سبسکرپشن کیسے منسوخ کر سکتا ہوں؟ {#how-can-i-cancel-my-subscription}

اگر آپ کے پاس Ultimate Multisite لائسنس ہے، تو آپ کسی بھی وقت اس کی تجدید (renewal) کو منسوخ کر سکتے ہیں۔ بس نیچے دیے گئے اقدامات پر عمل کریں:


<!-- Screenshot unavailable: Account page email with access link -->

وہ ای میل استعمال کریں جو آپ نے خریداری کے دوران فراہم کی تھی اور اپنا پاس ورڈ۔

<!-- Screenshot unavailable: Login page with email and password fields -->

یہ آپ کا اکاؤنٹ پیج کا ابتدائی سکرین ہے:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

اپنی سبسکرپشن منسوخ کرنے کے لیے، بائیں طرف مینو میں جائیں اور _Renewing & Billing._ پر کلک کریں۔

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

دائیں طرف موجود تیر (arrow) پر کلک کریں تاکہ ایک سائیڈ ونڈ کھل جائے۔ پھر، آپ کو وہ آپشن منتخب کرنا ہوگا جو **auto-renew** کو منسوخ کر دے۔

سسٹم آپ کو ایک تصدیقی پیغام دکھائے گا۔

اگر آپ واقعی اپنی سبسکرپشن منسوخ کرنا چاہتے ہیں، تو بس _**Cancel Renewals**_ بٹن پر کلک کریں۔ اس عمل کے بعد، آپ سے ایک مختصر سروے (survey) بھرنے کے لیے کہا جائے گا۔

ہو گیا! آپ کی سبسکرپشن خود بخود تجدید نہیں ہوگی۔

**آپ کا کی (key) تب تک درست رہے گا جب تک آپ کی سبسکرپشن ختم نہ ہو جائے۔** اگر آپ اپنی سبسکرپشن دوبارہ شروع کرنا چاہتے ہیں، تو آپ کو یہ دستی طور پر کرنا ہوگا۔
