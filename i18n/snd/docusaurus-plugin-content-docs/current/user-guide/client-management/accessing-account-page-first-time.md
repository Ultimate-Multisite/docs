# توهان پنهنجو اڪاؤنٽ پيجهو ڪيئن پهريون دفعا وصول ڪري سگهو ٿا؟

پنهنجي اڪاؤنٽ پيجهو ان لڠڪ (link) جي ذريعي وصول ڪريو جيڪا توهان کي اسان جي plugin کڻڻ کان پوءِ توهان جي اي ميل ۾ ملي وڃي:

توهان جو خريد ڪرڻ دوران ڏني وئي اي ميل ۽ ان ۾ ملي وئي پاسورڊ استعمال ڪريو.

چونکہ هي توهان لاءِ اها اڪاؤنٽ پهريون دفعا لاگ ان ڪرڻ آهي، تنهنجي پاسورڊ بدلڻ جي درخواست ڪندي وڃي. اهو بدلائي ۽ _Save_ بٽون تي ڇڪڻ وڃو.

<!-- Screenshot unavailable: Change password form with Save button -->

پنهنجي اڪاؤنٽ پيجهو، توهان تمام معلومات جنهن حساب جو متعلق آهن جهڙوڪ بلنگ (billings)، تجديد (renewals)، منسوحي (cancellations)، ۽ ڪيترائي ٻيا خيال سنڀال سگهو ٿا.
