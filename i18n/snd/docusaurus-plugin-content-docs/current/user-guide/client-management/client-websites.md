# توهان جي ڪلائين جا ويب سائٽس (v2) {#your-clients-websites-v2}

پنهنجي نڪرن تي رجسٽر ڪرڻ کان پوءِ، توهان جا گهڻا ڪلائين پنهنجو dashboard استعمال ڪري سگهن ٿا.

**Account** صفحو، جتي اهي پنهنجي دفعات، ممڪنات (memberships)، دومين (domains)، پلان جي حد ۽ ٻيا اهم معلومات ڏس سگهن ٿا، ان کان علاوه **My Sites** صفحو به انهن کي ملي ٿو، جتي اهي پنهنجي تمام ويب سائٽس کي ڏسي سگهن ۽ منظم ڪري سگهن ٿا. انهن لاءِ بس پنهنجي dashboard تي My Sites تي ڪلڪ ڪرڻ آهي.

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

اهن کي ان جي تمام ويب سائٽس ڏسي وينديون ملنديون ٿيون، ۽ انهن هر هڪ جي ساداهي هيٺ **Manage** بٽڻ موجود هوندو آهي. جيڪڏهن اهي ان تي ڪلڪ ڪن ٿا، ته اهي ان خاص ويب سائٽ جو dashboard تي ريڊيڪٽ ٿين ويندا آهن.

اهي اهو به ڏس سگهن ٿا ته انهن جي ڪهڙي ويب سائٽ پلان لاءِ **primary** آهي جيڪڏهن پلان هڪ کان وڌيڪ ويب سائٽون رکڻ جي اجازت ڏئي ٿو.

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

هي سائيٽ ايديت جو پورو ڏسيل منظر آهي:

![Site edit full interface](/img/admin/site-edit-full.png)

جيڪڏهن اهي پنهنجي primary ويب سائٽ بدلڻ چاهين، ته انهن کي **Account** مينيوي تي ڪلڪ ڪري Account صفحو تي وڃڻ گهرجي.

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

اهي اهي ڪلائين جي تفصيل ڏسڻ ۽ ان ۾ تبديلي ڪرڻ به سگهن ٿيون:

![Customer edit interface](/img/admin/customer-edit.png)

پھر، انهن کي **Change Default Site** تي ڪلڪ ڪرڻ گهرجي.

![Change Default Site button on account page](/img/account-page/change-default-site.png)

هڪ پاپ اپ ويندو. 드롭-ڊاؤن لسٹ تي ڪلڪ ڪري ان ويب سائٽ کي چنيو ته توهان ڪهڙي کي primary ويب سائٽ بڻائڻ چاهيو ٿا.

![Change Default Site dropdown selection](/img/account-page/change-default-site-dropdown.png)

پھر، بس **Change Default Site** تي ڪلڪ ڪريو ۽ توهان جو ڪم ختم!

![Default site بدلون کي تسلسلہ ڏيڻ جو تصوير](/img/account-page/change-default-site-confirm.png)
