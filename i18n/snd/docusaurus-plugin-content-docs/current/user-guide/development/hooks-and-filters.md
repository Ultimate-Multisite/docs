# Ultimate Multisite جي هوڪس ۽ فلٽر {#ultimate-multisite-hooks-and-filters}

Ultimate Multisite وڏي تعداد ۾ فنڪشنز (functions) ۽ ڊيٽابيز ڪويريز (database queries) جو مجموعو آهي. ان ۾ هوڪس (hooks) ۽ فلٽر آهن جيڪي توهان کي موجودہ فنڪشنز کي تبديل ڪرڻ يا نئين ڪوڊ شامل ڪرڻ جي اجازت ڏين ٿا.

اسان هنن هوڪس ۽ فلٽرن جي هڪ لسٹ هيٺ ڏني آهي ته advanced users کي پنهنجي نڪري (network) کي اڃا به وڌيڪ customize ڪرڻ ۾ مدد ملي سگهي. توهان Github کان ڳنڍيل لنک تي ڪلص ڪري ان لسٹ تک رسائي سگهو ٿا.
