# گهواردار استعمال ڪرڻ (Managing Customers) {#managing-customers}

**Customers** صفحو توهان جي نرسڪي (network) ۾ لچي ٿيل اراكن جي فهرست ڏئي ٿو. هي صفحو نام، ايميل adhesion، آخري لاگ-इन وقت، ۽ هڪ **switch to** لينڪ ڏئي ٿو جنه توهان کي سائيٽ جو اراكن جي انتظام ڪندڙ (site administrator) طور لاگ ان ڪرڻ جي اجازت ڏئي ٿو.

![Customers list page](/img/admin/customers-list.png)

هڪ اراڪ جي ڪارڊ تي **Manage** تي ڇڪڻ سان توهان ان اراڪ بابت تفصيلي معلومات ڏس سگهو ٿا.

اراک جي تفصيل وارن صفحا تي، **At a Glance** بلاڪ ۾ اراڪ جي ID، آخري لاگ-इन جو وقت، ۽ هن اراڪ طرف کان ڪيتري رقم خرچ ٿي چڪي آهي، اهو ڏجي ٿو.

**Customer Options** بلاڪ توهان کي هن اراڪ کي هڪ **VIP** (هڪ ٽيگ جيڪو ان کي سڌو پيدا ڪرڻ ۾ آسان بڻائي ٿو) طور مقرر ڪرڻ جي اجازت ڏئي ٿو. **Billing Info** ٽاب ۾ بلنگ معلومات جهڙوڪ ايميل، پتو، ۽ ملڪ جي معلومات ڏني وينديون آهن.

**Fields** بلاڪ ۾ اراڪ جو صارف نام (username)، ايميل، ۽ توهان کي **Email Verification** جي حالت کي تبديل ڪرڻ جي اجازت آهي.

Customer Options بلاڪ کان نیچے توهان کي چار اضافي بلاڪ پون ملندا آهن:

- **Memberships** — موجوده سبسڪريپشن جيڪي اراڪ ان سان جڙيل آهن
- **Payments** — اراڪ جي دفعي (payment) جي تاريخ
- **Sites** — اهي سائيٽس جن جو مالک هي اراڪ آهي
- **Events** — هن اراڪ سان لاڳاپيل ڪهڙا پروگرام (جنهن ۾ ايميل تصديق، بلنگ ۾ تبديلين، يا ٻيا شيون شامل آهن)

ڪو به آئڪن (→) تي ڇڪڻ سان ان جي تفصيل وارن صفحا تي ويندا آهيو. مثال طور، سائيٽ جو箭头 (arrow) ڇڪڻ کان پوءِ توهان سائيٽ ايديت جو صفحو ڏسندا آهيو جتي توهان MEMBERSHIP جو تعلق، سائيٽ جي قسم ۽ فعال حالت کي تبديل ڪري سگهو ٿا.

هي ئي MEMBERSHIPS ۽ PAYMENTS لاءِ لاگو آهي — هر هڪ تفصيل وارن صفحا تي لڳو آهي جتي توهان MEMBERSHIP جي حالت، تعلقدار اراڪ، PRODUCTS، پيمنٽ گيتواي جي ترتي بيان (payment gateway settings)، ۽ هيٺ رفي (refunds) عمل ڪرڻ بابت تبديل ڪري سگهو ٿا.

اراک جي صفحو جي ديراني طرف توهان کي ڏسڻ ۾ ملندو آهي:

* **آخر لاگ ان ۽ IP ها** — آخر ڪڏهن ڳوٺ (login) ٿيو، IP ADDRESS، ۽ ڪنڊري (country).
* **تايم سٽمپس** — جڏهن ڪسٹمر جو account ٺهيو هو ۽ آخري تعديل ڪهجي.
* **منتقلي (Delete)** — ڪسٹمر کي حتمي طور تي مڪمل طور تي مڪمل ڪرڻ جو چوڻ.

:::warning
ڪسٽمر کي ختم ڪرڻ پورو آهي ۽ ان کي واپس نه ڪري سگهجي ٿو.
:::
