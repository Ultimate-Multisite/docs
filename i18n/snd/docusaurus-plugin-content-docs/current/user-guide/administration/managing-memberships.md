# رکنیت (Memberships) کو سنبھالنا

Ultimate Multisite ۾ توهان جي نڪري (network) تي سڀني سبسڪريپشنز کي سنڀال ۽ انهن جي رسائي (access) کي منظم ڪري سگهو ٿا. توهان پنهنجي پلانز ۽ پروڊڪٽس، سائٽن جي تعداد، بلنگ جو مقدار ۽ ڪ freqency، دفعي وارو تاريخ (payment history)، ۽ تايم سٽمپس کي ٻيهر ڏسي ۽ تعديل ڪري سگهو ٿا.

ان صفحتي تي پهچڻ لاءِ، Ultimate Multisite منيو ۾ **Memberships** ٽاب تي وڃو.

![Memberships list page](/img/admin/memberships-list.png)

Memberships صفحتي تي توهان پنهنجي گهرجندڙن (customers) ۽ انهن حسابن سان جڙيل پروڊڪٽس جي فهرست ڏسي سگهو ٿا. توهان حالت (status) جي لحاظ کان رکنيت کي فلٽر ڪري سگهو ٿا: active، pending، on hold، expired، يا cancelled.

فري ممبرشپ کي **lifetime** ممبرشپ طور پر سمجهيو ويندو آهي. ان کي ڪنه ختم ٿيڻ جو تاريخ نه ڏني ويندي ۽ اهو خود بخاري (automatically) ختم نه ٿيندو. جيڪڏهن ڪا فري پلان ٹرائل يا مقرر وقت کان پوءِ ختم ٿيڻ گهرجي، ته هڪ ٹرائل وارو پيڊ پروڊڪٽ استعمال ڪريو، ممبرشپ کي هيٺ ماني طور تي ختم ڪريو (manually cancel)، يا جڏهن توهان جي رسائي بدلون گهرجي تڏهن ڪسٹمر کي ٻئي پلان تي منتقل ڪريو.

ممبرشپ تي ڪلڪ ڪري ان صفحتي تي وڃو جتي توهان ممبرشپ جي تفصيل تعديل ڪري سگهو ٿا — بلنگ معلومات، سائٽ يا وزٽ جي حد (site or visit limits)، ۽ انهيءَ طرح ٻيا گهرجن.

![Membership edit interface](/img/admin/membership-edit.png)

ممبرشپ جي تعديل صفحتي جو پورو ڏوهرو هيٺ ڏنل آهي:

![Membership edit full interface](/img/admin/membership-edit-full.png)

توهان پنهنجي نڪري تي قبضو ڪندڙ صارفين سان جڙيل نون ممبرشپ به شامل ڪري سگهو ٿا جيڪي پريشان ٿيڻ کان اڳ توهان جي نڪري تي رجسٽر ٿيا هئا. **Add Membership** تي ڪلڪ ڪري تفصيل مقرر ڪريو.

![Add Membership modal](/img/admin/membership-add-modal.png)

پھر **Create Membership** تي ڪلڪ ڪريو. نون ممبرشپ Memberships جي فهرست ۾ ظاهر ٿيندو.
