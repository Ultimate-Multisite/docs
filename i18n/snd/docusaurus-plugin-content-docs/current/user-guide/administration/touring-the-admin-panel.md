# ایڈمن پینل گھومنا (Touring the Admin Panel)

Ultimate Multisite دا آسان ۽ خود وضاحت بڻجي ٿي ته اسان چاهيون، پر ان قدر طاقتور ٽول سان، اسان هڪ گائیڊڊ ٹور رکي سگهون ٿا. آءُ توهان کي پنهنجي پياري ایڈمن پيجز تي وڃڻ ۾ مدد ڪرڻ لاءِ قدم وارو ڪريون.

##แดش بورڊ (Dashboard)

Ultimate Multisite جو **Dashboard** توهان جي آمدني، سائٽ سرگرمين، ممبره گروه جي وڌڻ، وزترن جي گنائن ۽ جغرافيي معلومات شامل رپورٽس ۽ انالٽڪس ڏيکاري ٿو.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

هيแดش بورڊ جو پورو ڏوه آهي:

![Ultimate Multisite Dashboard full page](/img/admin/um-dashboard-full.png)

۽แดش بورڊ جو نچلا حصو:

![Ultimate Multisite Dashboard bottom section](/img/admin/um-dashboard-bottom.png)

هي هم نٽ ورڪ (network)แดش بورڊ جو پورو ڏوه آهي:

![Network Dashboard full page](/img/admin/network-dashboard-full-page.png)

##چيكآؤ فارمز (Checkout Forms)

**Checkout Forms** توهان کي خود ساختن ريجيسٽري پيجن (registration pages) کڻڻ جي لچڪ ڏين ٿا. توهان فیلڊز کي ڪسائيد ڪري سگهو ٿا، خود ساختن ڪلاسز ۽ اسڪرپٽس شامل ڪري سگهو ٿا، ۽ وزتر جي ملڪ يا مقام جي بنياد تي رسائي محدود ڪري سگهو ٿا.

![Checkout Forms list](/img/admin/checkout-forms-list.png)

##پروڊڪٹس (Products)

**Products** جو حصو آهي جتي توهان پنهنجي نٽ ورڪ لاءِ مختلف پلانز، پيڪيجس ۽ خدمتون ٺاهين ٿا. پروڊڪٽ جي سطح تي قيمت، بلنگ جي فرقي ۽ حدون ۽ کوٽا مقرر ڪري سگهو ٿا.

![Products list](/img/admin/products-list.png)

##ممبره (Memberships)

**Memberships** جو پيجه توهان جي نٽ ورڪ ۾ تمام سبسڪريپشن ڏيکاري ٿو. پلانز، پروڊڪٽس، بلنگ جي مقدارن ۽ فرقي، دفعي، هيٺين تاريخ ۽ وقت جي نشانات شامل ڪري سبسڪريپشن جي تفصيل کي نظر ۽ ترمیم ڪري سگهو ٿا.

![Memberships list](/img/admin/memberships-list.png)

##پياملٽس (Payments)

**Payments** صفحہ آپ کے پورے نیٹ ورک پر ادائیگیوں کی تاریخ کا ایک فوری نظارہ فراہم کرتا ہے، جس میں مخصوص مصنوعات اور رقم سمیت تفصیلی لین دین کی معلومات شامل ہیں۔

![Payments list](/img/admin/payments-list.png)

## صارفین (Customers) {#dashboard}

**Customers** صفحہ آپ کے نیٹ ورک میں رجسٹرڈ تمام ممبران کو دکھاتا ہے جن کی بنیادی صارف کی معلومات جیسے نام، ای میل ایڈریس، آخری لاگ ان اور اپنے سب سائٹ میں لاگ ان کرنے کے لیے ایک "switch to" لنک شامل ہے۔

![Customers list](/img/admin/customers-list.png)

## سائٹس (Sites) {#checkout-forms}

**Sites** صفحہ آپ کے نیٹ ورک کی تمام سب سائٹس، بشمول سائٹ ٹیمپلیٹس اور کسٹمر کی اپنی ملکیت والی سائٹس کو لسٹ کرتا ہے۔ یہاں آپ میپ شدہ ڈومینز، حدود اور کوٹاہوں، پلگ انز اور تھیمز، اور سائٹ تھمبنائیز کا آسانی سے انتظام کر سکتے ہیں۔

![Sites list](/img/admin/sites-list.png)

## ڈومینز (Domains) {#products}

**Domains** صفحہ سب سائٹس کے ساتھ جو کسٹم ڈومینز میپ کیے گئے ہیں، ان کے لیے وقف ہے۔ ایک سپر ایڈمن کے طور پر، آپ دستی طور پر کسٹم ڈومین کے نام شامل یا میپ کر سکتے ہیں اور DNS ریکارڈز اور لاگز دیکھ سکتے ہیں۔

![Domains list](/img/admin/domains-list.png)

## رعایت کوڈز (Discount Codes) {#memberships}

**Discount Codes** آپ کو اپنے آخری صارفین کو چھوٹ دینے کے لیے کوپن کوڈز بنانے کی اجازت دیتے ہیں۔ رعایت کی قیمت بتائیں اور مخصوص صارفین یا گروہوں کو نشانہ بنائیں۔

![Discount Codes list](/img/admin/discount-codes-list.png)

## براڈکاسٹ (Broadcasts) {#payments}

**Broadcasts** آپ کے سب سائٹ کے صارفین کو اعلان کرنے یا نجی پیغامات بھیجنے کا ایک ٹول ہے۔ اپنے سبسکرپشن پلان یا پروڈکٹ کی بنیاد پر مخصوص صارف گروہوں کو نشانہ بنائیں۔

![Broadcasts list](/img/admin/broadcasts-list.png)

## سیٹنگز (Settings) {#customers}

**Settings** صفحہ وہ جگہ ہے جہاں آپ Ultimate Multisite کو ترتیب دیتے ہیں — رجسٹریشن کی سیٹنگز، ادائیگیوں، API اور ویب ہکس، ڈومین میپنگ، اور دیگر انٹیگریشنز۔

AI कनेक्टर सेटिंग्स میں، فی الحال سپورٹ ہونے والے OAuth پرووائڈر پولز یہ ہیں: Anthropic Max، OpenAI ChatGPT/Codex، اور Google AI Pro۔ ہر پرووائڈر کارڈ سے سپر ایڈمن اکاؤنٹس کو کنیکٹ کر سکتے ہیں، محفوظ شدہ اکاؤنٹس کو ریفریش کر سکتے ہیں، ای میل کے ذریعے اکاؤنٹس کو ہٹا سکتے ہیں، اور جب کوئی سینڈ باکسڈ ماحول براؤزر ری ڈائریکٹ کو بلاک کرتا ہے تو دستی OAuth فال بیک کا استعمال کر سکتے ہیں۔ Cursor Pro سیٹ اپ کے اختیارات ایڈمن پینل سے ہٹا دیے گئے ہیں۔

ChatGPT/Codex اکاؤنٹس کنیکٹر-سپورٹ شدہ ٹول یوز کی سپورٹ کرتے ہیں جہاں آپریشنز میں ٹولز کی اجازت ہوتی ہے، اس لیے وہ ایڈمن ورک فلو جو کنیکٹر-سپورٹڈ آپریشنز پر منحصر ہوتے ہیں، OpenAI اکاؤنٹ کنیکٹ ہونے کے بعد Codex ٹول بیہیویئر استعمال کر سکتے ہیں۔

## ایونٹس (Events) {#sites}

**ایونٹس** پیج آپ کے نیٹ ورک میں ہونے والے تمام واقعات اور لاگز کا ریکارڈ رکھتا ہے۔ یہ پلان کی تبدیلیوں، سائن اپس، اور دیگر حرکتوں جیسے سرگرمیوں کو ٹریک کرتا ہے — جو آپ کے ملٹی سائٹ کو مانیٹر کرنے کے لیے کارآمد ہے۔

![Events list](/img/admin/events-list.png)

## ویب ہکس (Webhooks) {#domains}

**ویب ہکس** آپ کو ڈیٹا کسی دوسرے ایپلیکیشن تک پہنچانے کی اجازت دیتے ہیں۔ Ultimate Multisite سے Zapier جیسے پلیٹ فارمز پر ڈیٹا بھیجنے کے لیے یہ بہت کارآمد ہے۔

![Webhooks list](/img/admin/webhooks-list.png)
