# تھیم کے مطابق مہارتیں (Theme-Aware Skills) {#theme-aware-skills}

Superdav AI Agent v1.10.0 میں چار نئی تھیم سے واقف ہونے والی ان بلٹ-ان مہارتوں کو شامل کیا گیا ہے جو آپ کی فعال WordPress تھیم کے مطابق خود بخود ایڈجسٹ ہوتی ہیں۔ یہ مہارتیں آپ کی تھیم کی ساخت اور خصوصیات کے مطابق مخصوص رہنمائی اور صلاحیتیں فراہم کرتی ہیں۔

## تھیم سے واقف مہارتیں کیا ہیں؟ (What Are Theme-Aware Skills?) {#what-are-theme-aware-skills}

تھیم سے واقف مہارتیں پہلے سے ترتیب دی گئی معلومات کی بنیاد پر بنائے گئے علم کے ذخیرے اور ٹول سیٹ ہوتے ہیں جنہیں AI اسسٹنٹ آپ کی سائٹ پر فعال WordPress تھیم کی بنیاد پر خود منتخب کرتا ہے۔ جب آپ تھیمز تبدیل کرتے ہیں، تو اسسٹنٹ کی دستیاب مہارتیں خود بخود اپ ڈیٹ ہو جاتی ہیں — کسی دستی ترتیب کی ضرورت نہیں ہوتی۔

ہر مہارت میں شامل ہوتا ہے:

- **تھیم کے مخصوص دستاویزات** — تھیم کو استعمال کرنے اور اپنی مرضی کے مطابق بنانے کے بارے میں رہنمائی
- **بلاک اور پیٹرن کے حوالے** — دستیاب بلاکس، پیٹرنز، اور ڈیزائن کے اختیارات
- **تخصیص کی مثالیں** — عام کاموں کے لیے کوڈ کے ٹکڑے اور کنفیگریشن کے طریقے
- **بہترین طریقے (Best practices)** — تھیم کی ساخت اور ورک فلو کے لیے سفارشات

## دستیاب تھیم سے واقف مہارتیں (Available Theme-Aware Skills) {#available-theme-aware-skills}

### بلاک تھیمز (Block Themes) {#block-themes}

**کس پر لاگو ہوتا ہے:** وہ تھیمز جو WordPress کے بلاک پر مبنی (Full Site Editing) آرکیٹیکچر استعمال کرتے ہیں۔

بلاک تھیمز کی مہارت یہ رہنمائی فراہم کرتی ہے:

- بلاک ایڈیٹر کا استعمال کرتے ہوئے ٹیمپلیٹس بنانا اور ایڈٹ کرنا
- بلاک پیٹرنز اور دوبارہ استعمال ہونے والے بلاکس کے ساتھ کام کرنا
- عالمی اسٹائلز (global styles) اور theme.json سیٹنگز کو اپنی مرضی کے مطابق بنانا
- تھیم بلاکس اور ان کی مختلف شکلوں کا استعمال کرنا
- اپنی سائٹ کے لیے کسٹم بلاک پیٹرنز بنانا

**خود بخود فعال ہوتے ہیں جب:** آپ کی فعال تھیم ایک بلاک تھیم ہو (یہ `block-templates` فیچر کو سپورٹ کرتی ہے)۔

### کلکک تھیمز (Classic Themes) {#classic-themes}

**کس پر لاگو ہوتا ہے:** روایتی WordPress تھیمز جو PHP ٹیمپلیٹس اور کلکک ایڈیٹر استعمال کرتے ہیں۔

کلکک تھیمز کی مہارت یہ رہنمائی فراہم کرتی ہے:

PHP template files ۽ hooks سان ڪم ڪرڻ
ڪسٹمائزرز (Customizer) جي ذريعي ٿيم جو ڏسڻ ۾ تبديلي ڪرڻ
وِجٽ ايرا ۽ سائیڊبارز استعمال ڪرڻ
CSS کي تبديل ڪرڻ ۽ چيلڊ ٿيم (child theme) جو ڊولپمنٽ

**آئٽيڪٽ طور تي عمل ڪيو ويندو آهي جڏهن:** توهان جو فعال ٿيم ڪلاسيڪ (non-block) ٿيم هجي.

### Kadence Blocks {#kadence-blocks}

**ڪنهن لاءِ لاڳو ٿئي ٿو:** ان سائيٽن لاءِ جيڪي advanced block-based design لاءِ Kadence Blocks plugin استعمال ڪن.

Kadence Blocks جو اهو سکل توهان کي هيٺين ڳولڻ ۾ مدد ڏيندو آهي:

- Kadence جي advanced block library (جنهن ۾ Hero، Testimonials، Pricing، etc. شامل آهن) جو استعمال
- Kadence block جي setings ۽ responsive options جو konfigurations
- Kadence جا grid ۽ container blocks سان ڪسٹم layouts جو ٺهڻ
- Kadence blocks کي توهان جي ٿيم سان جوڙڻ (integrate)
- Kadence جي design system ۽ presets جو فائدو وٺڻ

**آئٽيڪٽ طور تي عمل ڪيو ويندو آهي جڏهن:** Kadence Blocks plugin توهان جي سائيٽ تي فعال هجي.

### Kadence Theme {#kadence-theme}

**ڪنهن لاءِ لاڳو ٿئي ٿو:** ان سائيٽن لاءِ جيڪي block-based design ۽ customization لاءِ Kadence theme استعمال ڪن.

Kadence Theme جو اهو سکل توهان کي هيٺين ڳولڻ ۾ مدد ڏيندو آهي:

- global styles ۽ theme.json جي ذريعي Kadence theme جو customisation
- Kadence جا built-in block patterns ۽ templates جو استعمال
- Kadence theme جي setings ۽ options جو konfiguration
- Kadence جي design system سان ڪسٹم designs جو ٺهڻ
- popular plugins ۽ tools سان Kadence جو جوڙڻ

**آئٽيڪٽ طور تي عمل ڪيو ويندو آهي جڏهن:** Kadence theme توهان جو فعال ٿيم هجي.

## سکل ڪيئن چونڊيا ويندا آهن {#how-skills-are-selected}

اسسٹينٽ هر message تي توهان جي فعال ٿيم ۽ ان ۾ لڳايل plugins کي خود هيٺ ڏسندو آهي. جيڪڏهن ڪو مطابق ٿيم-واهه (theme-aware) سکل موجود هجي، تڏهن اهو خودي طور تي اسسٹينٽ جي context ۾ لادو ٿيندو آهي. توهان کي سکل کي manual طور تي enable ڪرڻ يا تبديل ڪرڻ جي ضرورت نه آهي.

### متعدد سکل {#multiple-skills}

اگر آپ کی سائٹ پر ایک سے زیادہ مہارتیں لاگو ہوتی ہیں (مثال کے طور پر، اگر آپ Kadence Blocks اور Kadence Theme دونوں فعال رکھتے ہیں)، تو اسسٹنٹ کو تمام قابل اطلاق مہارتوں تک رسائی حاصل ہوگی اور وہ ہر ایک سے رہنمائی کا حوالہ دے سکتا ہے۔

### تھیم تبدیل کرنا {#switching-themes}

جب آپ اپنی فعال تھیم بدلتے ہیں، تو اگلے پیغام پر اسسٹنٹ کی دستیاب مہارتیں خود بخود اپ ڈیٹ ہو جاتی ہیں۔ مثال کے طور پر:

1. آپ **Block Themes** مہارت کو فعال کرتے ہوئے بلاک تھیم استعمال کر رہے ہیں۔
2. آپ ایک کلاسک تھیم پر سوئچ کرتے ہیں۔
3. اپنے اگلے پیغام پر، **Classic Themes** مہارت خود بخود لوڈ ہو جائے گی، اور **Block Themes** مہارت دستیاب نہیں رہے گی۔

## تھیم کے مطابق مہارتوں کا استعمال {#using-theme-aware-skills}

کسی تھیم سے متعلقہ مہارت کا فائدہ اٹھانے کے لیے، بس چیٹ انٹرفیس میں یہ بیان کریں کہ آپ کیا کرنا چاہتے ہیں۔ اسسٹنٹ خود بخود مناسب مہارت کی رہنمائی کا حوالہ دے گا۔

### مثال پرامپٹس {#example-prompts}

**Block Themes کے لیے:**
> "بلاک پیٹرن کا استعمال کرتے ہوئے بیک گراؤنڈ امیج اور سینٹرڈ ٹیکسٹ والا ہیرو سیکشن بنائیں۔"

**Classic Themes کے لیے:**
> "چائلڈ تھیم کا استعمال کرتے ہوئے سائیڈبار میں ایک کسٹم وِجٹ ایریا شامل کریں۔"

**Kadence Blocks کے لیے:**
> "Kadence Testimonials بلاک کا استعمال کرتے ہوئے ایک ٹیسٹیومونٹ سیکشن بنائیں۔"

**Kadence Theme کے لیے:**
> "ہیڈر لے آؤٹ اور نیویگیشن مینو اسٹائلنگ کو کسٹمائز کریں۔"

اسسٹنٹ آپ کی فعال تھیم اور پلسٹائنز کے مطابق مخصوص رہنمائی اور کوڈ کے نمونے فراہم کرے گا۔

:::note
تھیم سے متعلقہ مہارتیں Superdav AI Agent v1.10.0 اور اس کے بعد خود بخود دستیاب ہوتی ہیں۔ کسی اضافی سیٹ اپ یا کنفیگریشن کی ضرورت نہیں ہے۔
:::
