# انٹرنیٹ سرچ (Internet Search)

Gratis AI Agent v1.5.0 ۾ هڪ **انٽرنيٽ سرچ** جي صلاحيت شامل ڪئي آهي جيڪا AI اسسٽن کي گهڻي وقت جي گفتگو دوران ويب کان تازو معلومات حاصل ڪرڻ جو موقع ڏئي ٿي. هي [Brave Search API](https://brave.com/search/api/) يا [Tavily API](https://tavily.com/) سان چلائي ٿو.

## اهو ڪيئن کام ڪري ٿو (How It Works)

جڏهن انٽرنيٽ سرچ چالو هجي، ته اسسٽن خود هيءَ وقت جو سوال پڇڻ جي صلاحيت رکي ٿي جڏهن هو محسوس ڪري ته سوال کي موجوده يا ٻاهرين معلومات جي ضرورت آهي — مثال طور، تازي خبرون، لائیو پرائسنگ، يا ڪنه سندن ماڊل جي تربيت ختم ٿيڻ کان پوءِ بدلائي سگهن ٿيون معلومات.

نتجا (Results) وقت سان گڏ حاصل ڪيا ويندا آهن ۽ جواب ڏناءو هيٺ پيش ڪرڻ کان اڳ اسسٽن جي ڪنٽيڪسٽ ۾ شامل ڪيا ويندا آهن. اسسٽن اهو ٻڌائيندي آهي ته هو سوال جو جواب ڏيڻ لاءِ سرچ رزلٽس استعمال ڪيا آهن.

## انٽرنيٽ سرچ چالو ڪرڻ (Enabling Internet Search)

انٽرنيٽ سرچ لاءِ توهان جي چونڊيل سرچ پروائيڊر کان API key گهرجي ٿي. هيٺ ڏنل طريقو سان هيءَ ترتیب ڏئي سگهجي ٿو:

1. **Gratis AI Agent → Settings → Advanced** تي وڃو.
2. **Internet Search Provider** جي 드راپڊون (dropdown) کي ڏسو ۽ يا ته **Brave Search** يا **Tavily** چونڊيو.
3. ان جو مطابق API key ان ڊيٽا فیلڊ ۾ داخل ڪريو. هر فیلڊ کان پاسي سيني-هيڪ لينڪس ڏنل هوندا آهن.
4. **Save Settings** تي ڪلڪ ڪريو.

جڏهن هڪ کي محفوظ ڪيو ويندو آهي، ته انٽرنيٽ سرچ جي صلاحيت خود بخاري اسسٽن لاءِ موجود ٿي ويندي آهي.

## Brave Search

### Brave Search API Key حاصل ڪرڻ (Obtaining a Brave Search API Key)

1. [Brave Search API page](https://brave.com/search/api/) تي وڃو.
2. هڪ پلان لاءِ سيني-هيڪ ڪريو. هڪ مفت تير (free tier) موجود آهي جنهن ۾ هر مهينتي درخواست جي حد رکي ٿي.
3. Brave Search ڊولپرزแดش بورڊ کان پنهنجي API key کي ڪاپي ڪريو.
4. ان کي Gratis AI Agent جي سٽنگز ۾ **Brave Search API Key** فیلڊ ۾ پیسٽ ڪريو.

### استعمال جي حد (Usage Limits)

استعمال پر چارجز برے سرچ (Brave Search) کی بنیاد پر ہوتی ہے، جو کہ کی گئی کوئریوں کی تعداد پر مبنی ہے۔ ہر وہ AI جواب جس سے سرچ شروع ہو تو اسے ایک کوئری سمجھا جاتا ہے۔ غیر متوقع چارجز سے بچنے کے لیے اپنے استعمال کو [Brave Search developer dashboard](https://brave.com/search/api/) میں مانیٹر کریں۔

## Tavily

Superdav AI Agent v1.10.0 نے **Tavily** کو ایک متبادل انٹرنیٹ سرچ فراہم کنندہ کے طور پر شامل کیا ہے، جو زیادہ بھرپور نتائج اور جدید تحقیق کی صلاحیتیں پیش کرتا ہے۔

### Tavily API Key حاصل کرنا

۱۔ [Tavily API page](https://tavily.com/) پر جائیں۔
۲۔ ایک اکاؤنٹ کے لیے سائن اپ کریں۔ ماہانہ درخواستوں کی حد کے ساتھ مفت ٹیر دستیاب ہے۔
۳۔ اپنے API key کو Tavily dashboard سے کاپی کریں۔
۴۔ اسے Gratis AI Agent کی سیٹنگز میں **Tavily API Key** فیلڈ میں پیسٹ کر دیں۔

### استعمال کی حدود

استعمال Tavily کی طرف سے کیے گئے API کالز کی تعداد کے حساب سے چارج کیا جاتا ہے۔ ہر وہ AI جواب جس سے سرچ شروع ہو تو اسے ایک کال سمجھا جاتا ہے۔ غیر متوقع چارجز سے بچنے کے لیے اپنے استعمال کو [Tavily dashboard](https://tavily.com/) میں مانیٹر کریں۔

## انٹرنیٹ سرچ بند کرنا

API key کو فعال سرچ فراہم کنندہ فیلڈ سے ہٹا دیں اور سیو کر دیں۔ اس سے اسسٹنٹ کو انٹرنیٹ سرچ کی صلاحیت نہیں دی جائے گی۔

:::note
انٹرنیٹ سرچ جوابات میں تاخیر (latency) شامل کرتا ہے کیونکہ جواب تیار کرنے سے پہلے اسسٹنٹ کو سرچ کے نتائج کا انتظار کرنا پڑتا ہے۔ وقت حساس استعمال کے معاملات کے لیے، غور کریں کہ کیا حقیقی وقت کی سرچ ضروری ہے یا کیا اسسٹنٹ کا اندرونی علم کافی ہے۔
:::
