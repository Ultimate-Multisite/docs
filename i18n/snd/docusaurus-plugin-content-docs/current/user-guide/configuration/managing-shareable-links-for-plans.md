# Plans لاءِ شیئر ایبل لنکس مینج کرنا {#managing-shareable-links-for-plans}

Ultimate Multisite توهان کي आफ्ना ڪلائين سان پلان جا لنڪس سڌي شيئر ڪرڻ جو چوڻ ڏئي ٿو. ان لاءِ، پنهنجي ڪنهن به پلان جي ايڊٽينگ صفحا تي وڃو ۽ '_Click to copy the Shareable Link_' بٽون تي ڪلڪ ڪريو.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_پلان جي ايڊٽينگ صفحا تي، توهان کي شيئر ایبل لنکس کڻڻ لاءِ بٽون ملي ويندي آهي_

ان جو ڳالهه آهي ته هي ميڪانيزم لنڪ کي بنائڻ لاءِ डिफلتيفيل بلنگ فريڪنسي (default billing frequency) جو استعمال ڪري ٿو. ان جو مطلب اهو آهي ته جيڪڏهن توهان جي डिफلتيفيل بلنگ مهينوتي هجي، مثال طور تي، آخر ۾ توهان کي هيٺ ڏنل طرح جو لنڪ ملندو: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

جيڪڏهن توهان چاهيو ته شيئر ایبل لنڪ ڪوارٽلي (quarterly) تي डिफلت ڪجي، ته توهان کي آخر ۾ 3 جو لفظ شامل ڪرڻو پوندو:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

پلان '_premium_' کي سالياني شيڊول (yearly schedule) تي چونڊڻ لاءِ، توهان کي 1 جو لفظ ۽ ان کان پوءِ 'year' جو لفظ شامل ڪرڻو پوندو:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

يهڙي صورت ۾ ياد رکجو ته توهان مختلف بلنگ فريڪنسين لاءِ صرف هڪ لنڪ نه بنا سکديا. توهان کي پنهنجي هر بلنگ فريڪنس لاءِ هڪ خاص لنڪ شيئر ڪرڻ گهرجي.

جڏهن به توهان ڪلائين سان لنڪ شيئر ڪري ڇڏيو، ته اهو انهن کي توهان جي سائن اپ فلو (signup flow) جو پهريون قدم ڏانهن ريڊيڪٽ ڪري ڇڏيندو.
