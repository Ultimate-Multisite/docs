# Sovereign Migration Workflow

Ultimate Multisite: Multi-Tenancy 1.2.0 ۾، اسان ڪجهه سڌار (migration verification gates) شامل ڪيا آهن جنهن سان هڪ اڪثر (standard subsite) کي سوورين ٽينينٽ انفراسٽرڪچر ۾ منتقل ڪرڻ لاءِ تصديق جو عمل شروع ٿئي ٿو.

## شروع ڪرڻ کان اڳ

تصديق ڪريو ته آپ تساهم ڏني شيون موجود آهن:

- سورس سائٽ جي موجوده بڪ اپ (backup).
- Multi-Tenancy addon فعال آهي.
- منزل ٽينينٽ جي ڈیٹا بيس، فزڪل سيستم جو ريٽ، ۽ دومين نام (domain) تيار آهي.
- هڪ اسيبيتي (binding) جيڪا منزل ماحول سان مطابقت رکي ٿي.
- نڪر (network) لاءِ WP-CLI ڪمان چلائڻ جي رسائي.
