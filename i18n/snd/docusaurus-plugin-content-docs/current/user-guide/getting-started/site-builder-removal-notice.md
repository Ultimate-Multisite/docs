# साइट بلڊر موڊ ختم ڪرڻ جو نوٽيس

**Superdav AI Agent v1.12.0 ۾ Site Builder mode ختم ڪري ڇڏي ڇڏيو آهي.** جيڪڏهن توهان Site Builder mode استعمال ڪري رهيا هئيو، تنهنجو ميغريٽ **Setup Assistant agent** ڏانهن ڪرڻ گهرجي ٿا ٿيم (theme) جي ٺاهڻ ۽ سائيٽ جو ٺهڻ لاءِ.

## के ٿي ويو؟ {#what-happened}

### Site Builder Mode (پرانا انداز) {#site-builder-mode-legacy}

Site Builder mode ان لاءِ هڪ ويزرڊ (wizard)-بنيو انٽرفيس هو:

- ٽيمپليٽس کان سائيٽون ٺاهڻ
- بنيادي ترتي بيان (basic settings) کي ترتیب ڏيڻ
- ڪا ٿيم (theme) چونڊڻ
- شروعاتي مواد (initial content) جو ٺهڻ

### ان جي جاءِ تي ڇا اچي ويو؟ {#what-replaced-it}

**Setup Assistant agent** هاڻي سڀ Site Builder جي ڪم ڪندڙ صلاحيتون سنڀالندو آهي، جنهن سان:

- وڌيڪ لچڪدار، ايجنت-گيداهو (agent-guided) ترتي بيان
- ٿيم کي بهتر انداز ۾ تبديل ڪرڻ جا خيار
- Theme Builder جي شروعاتي عمليت (onboarding) سان جوڙجڪڻ
- مستقبل جي سيسشن لاءِ پائيدار `site_brief` memory

## جيڪڏهن توهان Site Builder Mode استعمال ڪري رهيا هئيو {#if-you-were-using-site-builder-mode}

### توهان جون سائيٽون محفوظ آهن {#your-sites-are-safe}

- Site Builder mode سان ٺهي وئي موجود سائيٽون ڪم ڪرڻ جو عمل جاري رکندا
- ڪنهن به ڊيٽا جي گواستار يا سائيٽ ۾ خراب ٿيڻ نٿو پوي
- توهان عام طرح کان پنهنجي سائيٽن کي سنڀالڻ جو ڪم ڪري سگهو ٿا

### Setup Assistant Agent ڏانهن ميغريٽ ڪريو {#migrate-to-setup-assistant-agent}

نئين سائيٽ جي ترتیب لاءِ يا ٿيم ۾ تبديل ڪرڻ لاءِ، Setup Assistant agent استعمال ڪريو:

```
"Help me set up a new site"
```

يا

```
"Start the Theme Builder onboarding"
```

Setup Assistant agent وڌيڪ لچڪدار انداز ۾ وڌيڪ ڪم ڪندڙ صلاحيتون فراهم ڪري ٿو.

## موازنہ: Site Builder vs. Setup Assistant {#comparison-site-builder-vs-setup-assistant}

| خاصيت (Feature) | Site Builder (ختم ٿيل) | Setup Assistant (نئين) |
|---------|----------------------|----------------------|
| ترتیب جو طريقو (Setup method) | ويزرڊ فارم | ايجنت مکالمو (Agent conversation) |
| ٿيم چونڊڻ | پنهنجي ٽيمپليٽس | خود ساختو ٺاهڻ (Custom generation) |
| تبديل ڪرڻ | محدود خيار | پوري تصميم نظام (Full design system) |
| سائيٽ جو مختصر بيان (Site brief) | محفوظ نه ٿيو | پائيدار memory |
| مستقبل جي سيسشنون | دہرائي وترتي عمليت | محفوظ `site_brief` استعمال ڪريو |
| لچڪ (Flexibility) | ثابت ڪيل ترتي عمليت | بدلون وارو مکالمو |

## Setup Assistant Agent ڏانهن ميغريٽ ڪرڻ {#migrating-to-setup-assistant-agent}

### نئين سائيٽ لاءِ {#for-new-sites}

साइट बिल्डर موڈ استعمال کرنے کے بجائے:

1. درخواست: "مجھے ایک نئی سائٹ سیٹ اپ کرنے میں مدد کریں"
2. سیٹ اپ اسسٹنٹ ایجنٹ آپ کی رہنمائی کرے گا:
   - سائٹ کا مقصد اور اہداف
   - ہدف سامعین (Target audience)
   - برانڈ کی شناخت (Brand identity)
   - تھیم جنریشن
   - ابتدائی ترتیب

### موجودہ سائٹس کے لیے {#for-existing-sites}

اگر آپ کے پاس سائٹ بلڈر موڈ سے کوئی موجودہ سائٹ ہے:

1. آپ اسے ویسے ہی استعمال جاری رکھ سکتے ہیں
2. تھیم کو اپ ڈیٹ کرنے کے لیے، یہ درخواست کریں: "میری سائٹ کا دوبارہ ڈیزائن کرو"
3. سیٹ اپ اسسٹنٹ ایجنٹ آپ کو ایک نیا تھیم بنانے میں مدد کرے گا
4. آپ کی سائٹ کا ڈیٹا تبدیل نہیں ہوگا

### تھیم تبدیلیوں کے لیے {#for-theme-changes}

سائٹ بلڈر موڈ کے تھیم انتخاب کے بجائے:

1. درخواست: "میرا تھیم تبدیل کریں"
2. سیٹ اپ اسسٹنٹ ایجنٹ یہ کرے گا:
   - آپ کی ڈیزائن کی ترجیحات کے بارے میں پوچھیں گے
   - ایک کسٹم تھیم تیار کرے گا
   - اسے آپ کی سائٹ پر فعال (Activate) کر دے گا

## اہم فرق {#key-differences}

### سائٹ بلڈر موڈ {#site-builder-mode}

```
1. ٹیمپلیٹ کا انتخاب کریں
2. تھیم منتخب کریں
3. بنیادی سیٹنگز ترتیب دیں
4. ہو گیا
```

### سیٹ اپ اسسٹنٹ ایجنٹ {#setup-assistant-agent}

```
1. اپنی سائٹ کے مقصد کی وضاحت کریں
2. اپنے ہدف سامعین کو متعین کریں
3. ڈیزائن کی ترجیحات کا انتخاب کریں
4. ایجنٹ کسٹم تھیم تیار کرتا ہے
5. ایجنٹ تھیم فعال کرتا ہے
6. مستقبل کے سیشنز کے لیے سائٹ بریف محفوظ ہوتا ہے
```

## سیٹ اپ اسسٹنٹ ایجنٹ کے فوائد {#benefits-of-setup-assistant-agent}

### زیادہ لچکدار (More Flexible) {#more-flexible}

- اپنی سائٹ کی وضاحت قدرتی زبان میں کریں
- کسٹم سفارشات حاصل کریں
- اپنی مخصوص ضروریات کے مطابق ڈھل جائیں

### بہتر کسٹمائزیشن {#better-customization}

- کسٹم تھیم جنریشن
- ڈیزائن سسٹم کے فیصلے
- مستقل ڈیزائن ٹوکنز (Persistent design tokens)

### مستقل یادداشت (Persistent Memory) {#persistent-memory}

- آپ کا `site_brief` محفوظ رہتا ہے
- مستقبل کے ایجنٹ آپ کی سائٹ کو سمجھ جاتے ہیں
- سیٹ اپ کی معلومات دہرانے کی ضرورت نہیں پڑتی

### مربوط ورک فلو (Integrated Workflow) {#integrated-workflow}

- تھیم بلڈر آن بورڈنگ
- ڈیزائن سسٹم کی خوبصورتی کا ہنر (Design System Aesthetics skill)
- ویزیبلٹی کنٹرولز کی صلاحیت
- سب چیزیں بغیر کسی رکاوٹ کے ایک ساتھ کام کرتی ہیں

## مسائل حل کرنا (Troubleshooting) {#troubleshooting}

### مجھے سائٹ بلڈر موڈ نہیں مل رہا {#i-cant-find-site-builder-mode}

سائٹ بلڈر موڈ ہٹا دیا گیا ہے۔ اس کے بجائے سیٹ اپ اسسٹنٹ ایجنٹ کا استعمال کریں:

"مجھ کو ایک نئی سائٹ سیٹ اپ کرنے میں مدد کریں"

### میں سائٹ بلڈر سے کوئی سائٹ دوبارہ بنانا چاہتا ہوں {#i-want-to-recreate-a-site-from-site-builder}

آپ اسے سیٹ اپ اسسٹنٹ ایجنٹ کے ذریعے دوبارہ بنا سکتے ہیں:

1. درخواست: "Help me set up a new site" (مجھ کو ایک نئی سائٹ سیٹ اپ کرنے میں مدد کریں)
2. اپنی اصل سائٹ کے مقصد اور ڈیزائن کی تفصیل بتائیں۔
3. ایجنٹ آپ کے لیے ایک ملتی جلتی تھیم تیار کرے گا۔
4. آپ کا `site_brief` مستقبل کے حوالے کے لیے محفوظ ہو جائے گا۔

### میری موجودہ سائٹ بلڈر سائٹ کام نہیں کر رہی {#my-existing-site-builder-site-isnt-working}

سائٹ بلڈر موڈ سے بنائی گئی پرانی سائٹس اب بھی کام کرتی ہیں۔ اگر آپ کو کوئی مسئلہ پیش آ رہا ہے تو:

1. چیک کریں کہ آپ کی تھیم ابھی بھی فعال (active) ہے یا نہیں۔
2. تصدیق کریں کہ آپ کے plugins آن ہیں یا نہیں۔
3. WordPress ایرر لاگز (error logs) چیک کریں۔
4. اگر مسائل برقرار رہتے ہیں تو سپورٹ سے رابطہ کریں۔

### کیا میں اب بھی اپنے پرانے Site Builder ٹیمپلیٹس استعمال کر سکتا ہوں؟ {#can-i-still-use-my-old-site-builder-templates}

Site Builder ٹیمپلیٹس اب دستیاب نہیں ہیں۔ تاہم:

- آپ کی موجودہ سائٹس کام کرتی رہتی ہیں۔
- آپ سیٹ اپ اسسٹنٹ ایجنٹ کے ذریعے ملتی جلتی سائٹس دوبارہ بنا سکتے ہیں۔
- سیٹ اپ اسسٹنٹ ایجنٹ میں زیادہ کسٹمائزیشن کے اختیارات فراہم کیے جاتے ہیں۔

## اگلے اقدامات (Next Steps) {#next-steps}

1. **نئی سائٹس کے لیے**: سیٹ اپ اسسٹنٹ ایجنٹ کا استعمال کریں۔
2. **موجودہ سائٹس کے لیے**: انہیں ویسے ہی استعمال کرتے رہیں۔
3. **تھیم تبدیل کرنے کے لیے**: سیٹ اپ اسسٹنٹ ایجنٹ سے مدد طلب کریں۔
4. **ڈیزائن کو بہتر بنانے کے لیے**: Design System Aesthetics سSkill کا استعمال کریں۔
