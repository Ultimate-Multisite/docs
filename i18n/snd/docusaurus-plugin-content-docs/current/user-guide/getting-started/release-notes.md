## ورژن 2.13.0 — 2026-06-05 تائين ريليس ٿيو {#version-2130--released-on-2026-06-05}

नयाँ: ग्राहक खाता، चेकआउट, बिलिङ, साइट, इनभइस, टेम्प्लेट स्विचिंग, र डोमेन मैपिंग वर्कफ्लोका लागि सोव्हेरेन-टेनेन्ट सपोर्ट थपिएको छ ताकि टेनेन्ट नेटवर्कले व्यवस्थापन गरिएका कार्यहरूका लागि ग्राहकहरूलाई मुख्य साइटमा निर्देशित गर्न सकून्।
नयाँ: दोहोरिने सदस्यताहरूको लागि नवीकरण-क्रेडेंशियल जाँच थपिएको छ ताकि गेटवेले बचत गरिएको बिलिङ सम्झौता, सदस्यता, वा भल्ट टोकन हराएको बेला स्वतः नवीकरण बन्द गर्न सकोस्।
नयाँ: साइट निर्माणका लागि पेंडिङ हुने HMAC-verified लूपब्याक पब्लिशिङ थपिएको छ ताकि पृष्ठभूमि कामहरू ढिलो हुने होस्टहरूमा चेकआउट-देखि-साइट प्रोभिजनिङ अझ भरपर्दो बनाउन सकियोस्।
नयाँ: SSO URL, चेकआउट-फर्म बेस डोमेन, र स्वचालित डोमेन-रेकर्ड निर्माणका लागि डेभलपर एक्सटेन्सन पोइन्ट थपिएको छ।
सुधार: नक्सा गरिएका डोमेनहरूमा SSO अझ भरपर्दो बनेको छ, अनामिक ब्रोकर भ्रमण, लॉगआउट, र विभिन्न प्लगइनहरूको निर्भरता द्वन्द्वबाट पनि यसले सुधार गरेको छ।
सुधार: पेंडिङ साइट निर्माण अब पुरानो पब्लिश फ्ल्यागहरूबाट रिकभर हुन्छ र ग्राहकहरूलाई साइट-निर्माण स्क्रिनमा अड्काउनबाट बचाउँछ।
सुधार: साझा चेकआउट-फर्म बेस डोमेनहरूको लागि डोमेन रेकर्डहरू अब बनाइँदैनन्, र कुनै एकीकरण सक्रिय नभएको बेला प्रयोग नगरिएका होस्ट-प्रदायक पृष्ठभूमि कामहरू छुटाउँछ।
सुधार: चेकआउट, बिलिङ ठेगाना, पासवर्ड रिसेट, इमेल प्रमाणीकरण, टेम्प्लेट स्विचिंग, टुरहरू, र ग्राहक ड्यासबोर्डका किनारा अवस्थाहरूले सामान्य ग्राहक प्रवाहलाई अवरुद्ध गर्दैनन्।
सुधार: ब्रोडकास्ट इमेलहरू अब प्राप्तकर्ताहरूलाई गोप्य राख्छ जबकि प्राप्तकर्ता सूची वा मेल ट्रान्सपोर्ट असफल भएमा SMTP/प्लग्इन घातक त्रुटिहरूबाट बच्दछ।
सुधार: सदस्यता नवीकरण, समाप्ति प्रदर्शन, र भुक्तानी संकलनका किनारा अवस्थाहरूले तत्काल समाप्ति, क्र्यास, वा आवश्यक भुक्तानी छुट्ने कुराबाट बचाउँछ।
सुधारिएको: वर्डप्रेस अनुकूलता ७.० सम्म परीक्षण गरिएको छ, उत्पादन Vue एसेट्सलाई npm स्रोतहरूबाट पुनर्निर्माण गरिन्छ, र Cypress अन्त्य-अन्तको कभरेजले अब चेकआउट, सेटअप, SSO, र गेटवे वर्कफ्लोहरूमा अझ बढी परीक्षण गर्दछ।

## ورژن 2.12.0 — 2026-05-15 تائید ٿيو {#version-2120--released-on-2026-05-15}

- نئين: Hostinger (hPanel) کي سپورٽڊ هاستنگ پراجوائي طور تي شامل ڪيو ويو آهي جنهن ۾ دومين ميپنگ جو گهڻو شامل آهي.
- نئين: Site Exporter انٽرنيٽ جي رابطي جي بندي کي منظم ڪرڻ لاءِ اب نڪري ٿي، جنهن سان سڄي سائيٽ کي آسان تر مرمت (restoration) ڪيو جا سکڻ ۾ مدد ملدي.
- تصحيح: BCC برڊڪاسٽ اي ميلز ۾ هدا مڪمل نه ڏيکيل recipients هيدر استعمال ڪجي ٿو ته جيئن recipient جي adres کي ظاهر ڪرڻ کان روڪجي سگهجي.
- تصحيح: جب غير تاريخي قيمت سان محفوظ ڪيو ويندو آهي ته ممبر شپ جي ختم ٿيڻ جو ٽائمن ۾ خرابيء نه ٿيندي.
- تصحيح: Stripe ممبر شپ جي اپڊيٽس اب پرانے deleteDiscount API کي هلائڻ کان سواءِ گهڻو گونج (discounts) صحيح طرح صاف ڪن ٿا.
- تصحيح: دومين ميپنگ جهڙي سائيٽن تي SSO ريڊيڪٽس کي محدود ڪيو ويو آهي ته جيئن انحدار ريڊيڪٽ لوپ (infinite redirect loops) کان بچي سگهجي.
- تصحيح: Set up wizard ۾ image picker جو چونڊ اب اندر موجود data model کي صحيح طرح اپڊيٽ ڪري ٿو.
- تصحيح: Site Exporter CLI اب صحيح default network site selection کي محفوظ رکي ٿو.
- بهتر: پليجن جي سائز گھٽ ڪرڻ لاءِ، plugin package کان bundled wp-cli کي ختم ڪيو ويو آهي.
