# Ultimate Multisite: Ad Injector سان توهان جي نٽويڪ (network) تي اشتہارات وڌائڻ لاءِ

Ultimate Multisite: Ad Injector هڪ **مفت add-on** آهي جيڪو توهان کي انهن خطه جي edit page کان ئي پنهنجي plans تي اشتہارات وراجيت ڪرڻ جا اوزار ڏئي ٿو.

Ultimate Multisite: Ad Injector نصب ڪندي، ان کي فعال ڪرڻ لاءِ پنهنجي _Plugins_ page تي وڃو.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Network Activate بٽڻ تي ڪلڪ ڪريو_

جڏهن توهان add-on کي فعال ڪري ڇڏيو ٿا، ته plan edit screen تي موجود _Advanced Options_ meta box ۾ هڪ نون خيار ظاهر ٿيندو جنهن جو نالو _Ad Settings_ آهي.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_هڪ توهان هر Plan لاءِ اشتہار جي setings edit ڪري سگهو ٿا_

ان ٽاب تي، توهان پنهنجي front-end ۾ posts ۽ pages لاءِ ad codes شامل ڪري سگهو ٿا، content کان اڳي ۽ بعد ۾.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_پنهنجي اشتہار جو code داخل ڪرڻ کان پوءِ، اهو ان خاص Plan جي posts ۽ pages تي ظاهر ٿيندو._

توهان ان Plan جا users لاءِ Admin Pages جي header ۾ به اشتہارات شامل ڪري سگهو ٿا.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_جيڪڏهن توهان پنهنجي User جو back-end تي اشتہارات ڏسڻ چاهيو ٿا ته Admin Panel Ad Code option تي وڃو._

### اشتہار وڌيڪ (Beyond Advertising) {#beyond-advertising}

توهان Ultimate Multisite: Ad Injector کي high-tier plans جي مدد ڪرڻ لاءِ استعمال ڪري سگهو ٿا.

بس _Admin Panel Ads_ option تي وڃو ۽ پنهنجي support tool جو HTML code شامل ڪريو.
