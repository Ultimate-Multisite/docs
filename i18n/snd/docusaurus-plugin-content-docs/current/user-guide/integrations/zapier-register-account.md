# ایونٹ: Zapier کے ذریعے اکاؤنٹ رجسٹر کرنا {#event-register-an-account-via-zapier}

اس آرٹیکل [Integrating Ultimate Multisite with Zapier](zapier.md) میں، ہم نے بات کی تھی کہ کیسے آپ Zapier کا استعمال ٹرگرز اور ایونٹس کی بنیاد پر Ultimate Multisite کے اندر مختلف کام کرنے کے لیے کر سکتے ہیں۔ اس آرٹیکل میں، ہم دکھائیں گے کہ آپ تھرڈ پارٹی ایپلی کیشنز کو کیسے جوڑ سکتے ہیں۔ ہم ڈیٹا کے ذرائع کے طور پر Google Sheets کا استعمال کریں گے اور معلومات کو Ultimate Multisite کو بھیجیں تاکہ اکاؤنٹ رجسٹر ہو سکے۔

Zapier سے پہلے کنیکٹ کرنے کے لیے، **Ultimate Multisite > Settings > API & Webhooks** پر جائیں اور تصدیق کریں کہ API فعال ہے۔ جب Zapier آپ سے Ultimate Multisite اکاؤنٹ کی تفصیلات مانگے تو اس سکرین سے API Key اور API Secret کاپی کر لیں۔

![API and Webhooks settings with API Key, API Secret, and Enable API options](/img/admin/settings-api-webhooks.png)

سب سے پہلے، آپ کو اپنی Google Drive کے اندر ایک **Google Sheet** بنانی ہوگی۔ اس بات کو یقینی بنائیں کہ آپ ہر کالم کو ٹھیک سے ڈیفائن کریں تاکہ بعد میں ڈیٹا کو آسانی سے مپ (map) کر سکیں۔

ایک Google sheet بنانے کے بعد، آپ اپنے Zapier اکاؤنٹ میں لاگ ان کر سکتے ہیں اور ایک zap بنانا شروع کر سکتے ہیں۔

**"App event"** کے سرچ فیلڈ کے نیچے **"Google Sheets"** منتخب کریں۔

پھر "**Event** " فیلڈ کے لیے **"New spreadsheet row"** منتخب کریں اور "**Continue** " پر کلک کریں۔

اگلا قدم آپ سے ایک **Google Account** منتخب کرنے کے لیے پوچھے گا جہاں **Google Sheet** محفوظ ہے۔ لہذا بس یہ یقینی بنائیں کہ صحیح گوگل اکاؤنٹ مخصوص کیا گیا ہے۔

**"Set up trigger** " کے نیچے، آپ کو اس گوگل اسپریڈ شیٹ اور ورکشیت کا انتخاب کرنا ہوگا جس کا استعمال آپ ڈیٹا حاصل کرنے کے لیے کریں گے۔ بس اسے پُر کریں اور "**Continue** " پر کلک کریں۔

اگلا قدم یہ ہے کہ **"test your trigger** " تاکہ یہ یقینی بنایا جا سکے کہ آپ کی گوگل sheet ٹھیک سے منسلک ہے۔

اگر آپ کا ٹیسٹ کامیاب ہو جاتا ہے، تو آپ کو اپنے اسپریڈ شیٹس سے کچھ ویلیوز دکھاتے ہوئے نتیجہ نظر آنا چاہیے، آگے بڑھنے کے لیے "**Continue** " پر کلک کریں۔

نالي اگلا قدم آهي ته دوهم عمل ترتیب ڏيڻ جيڪو Ultimate Multisite ۾ ڪاڪردار جو حساب (account) کڻي يا رجسٹر ڪرڻ سان گڏ ٿيندو. سرچ فيلڊ ۾ "**Ultimate Multisite(2.0.2)** " چونڊيو.

"**Event**" جي فیلڊ تحت، "**Register an Account in Ultimate Multisite** " چونڊيو ۽ پوءِ "**Continue** " بٽڻ کي ڇڪيو.

"**Set up an action**" تحت، توهان کي ڪسٹمر جي معلومات، ممبرشپ (memberships)، پروڊڪٹس يا ٻين شين لاءِ مختلف فیلڊ ڏس ويندا. توهان گوگل شيت ۾ قيمت (values) کي مئپ ڪري سگهو ٿا ۽ انهن کي ان مناسب فیلڊ ۾ مقرر ڪري سگهو ٿا جتي انهن کي پوري ڪرڻ گهرجي جهڙوڪ هيٺ ڏنل اسڪريئن ۾ ڏيکيل آهي.

قيمت مئپ ڪرڻ کان پوءِ، توهان عمل کي ٽيسٽ ڪري سگهو ٿا.
