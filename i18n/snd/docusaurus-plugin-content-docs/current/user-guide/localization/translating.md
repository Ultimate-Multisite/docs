# Ultimate Multisite ترجمو {#translating-ultimate-multisite}

Ultimate Multisite ان English ۾ ٺهي آهي، پر هي plugin پوري طرح ترجمي لاءِ کائتو آهي ۽ اسين فعال طور تي ترجمي جو تعاون گهڻو استقبال ڪندون آهيون.

## Core Plugin Translation (مرڪزي plugin جي ترجمو) {#core-plugin-translations}

Core Ultimate Multisite plugin کي آفیشل **WordPress.org translation platform** تروضي ٿيل آهي:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

هي standard WordPress community translation system آهي. جڏهن ترجمو ڪجهه مناسب منظوري حد تائين پهچ وڃي ٿو، ته هو WordPress plugin updates جي ذريعي تمام Ultimate Multisite users کي خود بخود ڇڪيل ويندا آهن.

**اسان کي توهان جي مدد گهرجي!** جيڪڏهن توهان English کان ٻئي ڪا زبان بولندا آهيو، ته ترجمو ڏيڻ جو سوچيو. ڪجهه strings ترجمو ڪرڻ سان به مدد ملندي آهي. شروع ڪرڻ لاءِ:

1. [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/) پهچو
2. پنهنجي زبان چونڊيو
3. "Contribute Translation" ڇڪيو
4. توهان کي WordPress.org جو مفت account گهرجي

 جيڪڏهن توهان پنهنجي زبان لاءِ ترجمو Editor بڻجي ٿو (ترجمو منظور ڪرڻ جي صلاحيت رکندا)، ته [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) ذريعي رابطو ڪري سگهو ٿا.

## Addon Translations (Addon جو ترجمو) {#addon-translations}

Addon plugins کي الگ کان:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)** تي ترجمو ڪيو ويندو آهي.

Addon جي ترجمون موجوده وقت AI تائين ٺهي وئي آهي. اهي تمام سمجهيل زبانن کي کوري ٿا پر انهن ۾ ڪجهه عجیب انداز يا غلطيون ٿي سگهن ٿيون. اسين انسان ترجموڪارن کي انهن کي بهتر ۽ صحيح بڻائڻ لاءِ نظر تحصيل ڏيڻ جو استقبال ڪندون آهيون.

## Local Translations (مقامی ترجمو) {#local-translations}

جيڪڏهن توهان پنهنجي ڪو پراڪٽ لاءِ لوڪل ترجمو ڪرڻ پسند ڪريو ٿا ته هيٺيون ٻه طريقا آهن:

### Poedit استعمال ڪرڻ {#using-poedit}

توهان `.pot` فائل کي ترجمو ڪرڻ لاءِ ديسڪٽپ اپليڪيشن **[Poedit](https://poedit.net)** جو استعمال ڪري سگهو ٿا:

1. [poedit.net](https://poedit.net) کان Poedit ڈاؤن لوڈ ڪريو
2. `wp-content/plugins/ultimate-multisite/lang/` کان `.pot` فائل کي खोलो ڪريو
3. Poedit جو انٽر فيس ۾ سٽرين (strings) ترجمو ڪريو
4. محفوظ ڪريو -- Poedit `.po` ۽ `.mo` فائلون خود هيٺ ٺاهي ٿو
5. آؤٽ پيڪٽ فائلون `lang` فولڊر ۾ رکي ڏيو

### Loco Translate استعمال ڪرڻ {#using-loco-translate}

**[Loco Translate](https://wordpress.org/plugins/loco-translate/)** ورڊپريس پليجن توهان کي ڪنه به ديسڪٽپ ٽولز کان سواءِ، اڪيڊمينيشن پينل کان سڌو ترجمو ڪرڻ جي اجازت ڏئي ٿو. اهو توهان لاءِ ترجمو فائلون ٺاهڻ ۽ کمپائل ڪرڻ جو ڪم سنڀال رکي ٿو.
