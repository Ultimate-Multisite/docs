# Filtre: wu_plan_product_types

Plan olarak kabul edilmesi gereken ürün tiplerini filtreler.

Bu filtre, eklentilerin doğrulama ve ayrıştırma işlemlerinde plan olarak tanınması gereken ek ürün tipleri kaydetmesine olanak tanır.

## Parametreler {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | Plan olarak kabul edilecek ürün tiplerinin dizisi. |
