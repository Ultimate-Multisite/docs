# Filter: wu_current_{$type}_get_manage_url {#filter-wucurrenttypegetmanageurl}

Geliştiricilerin site yönetimi URL parametrelerini değiştirmesine olanak tanır.

## Parametreler {#parameters}

| Ad | Tip | Açıklama |
|------|------|-------------|
| $manage_site_url | `string` | Yönetim sitesi URL'si. |
| $id | `int` | Site kimliği (ID). |
| $site_hash | `string` | Site hash'i. |

### Kullanılmaya Başladığı Versiyon {#since}

- 2.0.9
### Kaynak {#source}

214. satırda [`inc/class-current.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-current.php#L214) içinde tanımlanmıştır.


## Dönüş Değeri {#returns}
Değiştirilmiş yönetim URL'si.
