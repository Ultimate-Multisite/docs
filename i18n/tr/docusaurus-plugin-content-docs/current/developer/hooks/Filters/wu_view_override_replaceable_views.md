# Filter: wu_view_override_replaceable_views

Geliştiricilerin, değiştirilebilir listeye ek klasörler eklemesine olanak tanır.

Dikkatli olun, çünkü ek klasörlere izin vermek, Ultimate Multisite sürümleri yerine güncel olmayan kopyaların yüklenmesine neden olabilir.

## Parametreler

| Name | Type | Description |
|------|------|-------------|
| $replaceable_views | `array` | İzin verilen klasörlerin listesi. |

### Since

- 2.0.0
### Source

[`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L50) dosyasındaki 50. satırda tanımlanmıştır.
