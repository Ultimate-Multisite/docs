# Filter: send_auth_cookies

Bu filtre, kimlik doğrulama çerezlerinin (auth cookies) gerçekten istemciye gönderilmesini engellemenize olanak tanır.

## Parametreler

| Name | Type | Açıklama |
|------|------|-------------|
| $send | `bool` | Kimlik doğrulama çerezlerinin istemciye gönderilip gönderilmeyeceği. |
