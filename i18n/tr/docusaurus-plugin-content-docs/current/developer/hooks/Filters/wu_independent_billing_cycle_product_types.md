# Filtre: wu_independent_billing_cycle_product_types

Bağımsız faturalandırma döngüsüne sahip ürün tiplerini filtreler.

Bağımsız faturalandırma döngüsüne sahip ürünler, sepette planın faturalandırma dönemine uymak zorunda değildir. Bu, plan aylık olsa da yıllık olsa da her zaman yıllık faturalandıran alan adı kayıtları gibi ürünler için kullanışlıdır.

## Parametreler {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | Bağımsız faturalandırma döngüsüne sahip ürün tiplerinin dizisi. |
