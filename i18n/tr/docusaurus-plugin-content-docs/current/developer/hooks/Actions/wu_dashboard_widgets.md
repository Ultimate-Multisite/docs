# Action: wu_dashboard_widgets

Eklenti geliştiricilerinin Ağ Kontrol Paneli'ne (Network Dashboard Panel) widget eklemesine izin verir.

## Parametreler

| Name | Type | Description |
|------|------|-------------|
| $tab | `string` | Mevcut sekmenin adını tutar. |
| $screen | `\WP_Screen` | Ekran nesnesini temsil eder. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Ultimate Multisite yönetici sayfasının bir örneğidir. |

### Since

- 2.0.0
