# Action: wu_register_integrations

Entegrasyonların kendilerini kaydetmelerine izin vermek için çağrılır.

Çekirdek (Core), entegrasyonlarını bu noktada kaydeder.

## Parametreler {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $registry | `\Integration_Registry` | Kayıt defteri örneği. |

### Versiyon {#since}

- 2.5.0
