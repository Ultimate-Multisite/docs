# Aksiyon: wu_after_switch_template

Bir kullanıcı veya süper yönetici site şablonunu değiştirdikten sonra eklenti geliştiricilerinin fonksiyonlarını bağlamasına (hook yapmasına) olanak tanır.

## Parametreler

| Name | Type | Description |
|------|------|-------------|
| $id | `int` | Site Kimliği |

### Versiyon

- 1.9.8
### Kaynak

`inc/ui/class-template-switching-element.php` dosyasında, 296. satırda tanımlanmıştır.
