# Action: wu_dashboard_{$this->tab}_widgets {#action-wudashboardthis-tabwidgets}

Eklenti geliştiricilerinin Ağ Kontrol Paneli'ne (Network Dashboard Panel) widget eklemesine izin verir.

## Parametreler {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $tab | `string` | Mevcut sekme. |
| $screen | `\WP_Screen` | Ekran nesnesi. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Ultimate Multisite yönetici sayfası örneği. |

### Since {#since}

- 2.0.0
