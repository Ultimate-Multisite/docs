# PRO Theme ile Yönetim Panelinizi Özelleştirme

Müşteri işleri yaparken çoğu zaman WordPress yönetim panelinde müşterilerimize özel bir deneyim sunmamız gerekir. Daha profesyonel bir görünüm arıyorsanız, PRO Theme tam size göre olabilir.

PRO Theme, müşterilerinizin Dashboard'unu özelleştirmenize yardımcı olmak için geliştirilmiş bir yönetim temasıdır. Aynı zamanda Ultimate Multisite'ın premium eklenti paketinin bir parçasıdır.

Kurulumdan sonra _Eklentiler_ sayfanıza gidin ve ağınızda etkinleştirmek için tıklayın.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Eklentiler sayfanızda ağ genelinde etkinleştirin_

_**NOT:**_ _Sadece kurulum yaparsanız, müşterilerinizin panelinde temayı yalnızca o kullanıcıya eklemek için işaretlenebilecek bir seçenek olarak görünecektir._

Artık yönetim arayüzünüz tamamen yenilenmiş durumda! Sol sütunda PRO Theme adında yeni bir menü de göreceksiniz.
