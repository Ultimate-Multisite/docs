# Hesap sayfama ilk kez nasıl erişebilirim?

Ultimate Multisite'a hoş geldiniz!

Eklentimizi satın aldıktan sonra e-posta ile aldığınız bağlantıyı kullanarak hesap sayfanıza erişebilirsiniz:

<!-- Screenshot unavailable: Account page email with access link -->

Satın alma sırasında verdiğiniz e-posta adresini ve e-postanızda aldığınız şifreyi kullanın.

<!-- Screenshot unavailable: Login page with email and password fields -->

Bu hesaba ilk kez giriş yaptığınız için şifrenizi değiştirmeniz istenecektir. Şifrenizi değiştirin ve _Kaydet_ butonuna tıklayın.

<!-- Screenshot unavailable: Change password form with Save button -->

Hesap sayfanızda faturalar, yenilemeler, iptal işlemleri ve daha birçok hesap detayını yönetebilirsiniz.
