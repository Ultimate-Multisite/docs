# Müşterinizin Web Siteleri (v2)

Ağınıza kaydolduktan sonra müşterileriniz Dashboard’larına erişebilir.

Ödemeleri, üyelikleri, domain’leri, plan sınırlamaları ve daha fazlasıyla ilgili önemli bilgileri görebilecekleri **Account** sayfasının yanı sıra, tüm web sitelerini görebilecekleri ve yönetebilecekleri **Sitelerim** sayfasına da erişimleri vardır. Tek yapmaları gereken Dashboard’larında Sitelerim’e tıklamaktır.

![Müşteri Dashboard’ında Sitelerim bağlantısı](/img/account-page/my-sites-link.png)

Tüm web siteleri, her birinin hemen altında bir **Yönet** düğmesiyle gösterilir. Buna tıklarlarsa, **bu belirli web sitesinin Dashboard’ına** yönlendirilirler.

Birden fazla web sitesine sahip olmalarına izin veren planlarda, web sitelerinden hangisinin **birincil** olduğunu da görebilirler.

![Yönet düğmeleriyle müşteri web sitelerini gösteren Sitelerim sayfası](/img/account-page/my-sites-page.png)

Site düzenleme arayüzünün tam görünümü burada:

![Site düzenleme tam arayüzü](/img/admin/site-edit-full.png)

Birincil web sitelerini değiştirmek isterlerse, **Account** menüsüne tıklayarak Account sayfasına gitmeleri gerekir.

![Müşteri Dashboard’ında Account menü bağlantısı](/img/account-page/account-menu.png)

Müşteri ayrıntılarını da görüntüleyebilir ve düzenleyebilirsiniz:

![Müşteri düzenleme arayüzü](/img/admin/customer-edit.png)

Ardından, **Varsayılan Siteyi Değiştir**’e tıklamalılar.

![Account sayfasında Varsayılan Siteyi Değiştir düğmesi](/img/account-page/change-default-site.png)

Bir açılır pencere görünecek. Açılır listeye tıklayın ve hangi web sitesini birincil web sitesi yapmak istediğinizi seçin.

![Varsayılan Siteyi Değiştir açılır liste seçimi](/img/account-page/change-default-site-dropdown.png)

Ardından, sadece **Varsayılan Siteyi Değiştir**’e tıklayın ve işlem tamam!

![Varsayılan site değişikliğini onaylama](/img/account-page/change-default-site-confirm.png)
