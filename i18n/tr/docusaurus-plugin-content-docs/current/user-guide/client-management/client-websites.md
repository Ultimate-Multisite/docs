# Müşterinizin Web Siteleri (v2)

Ağınıza kayıt olduktan sonra müşterileriniz kendi panellerine erişebilir.

Ödemeler, üyelikler, alan adları, plan limitleri ve daha fazlası hakkında önemli bilgileri görebilecekleri **Hesap** sayfasının yanı sıra, tüm web sitelerini görüp yönetebilecekleri **Sitelerim** sayfasına da erişebilirler. Tek yapmaları gereken panellerinde Sitelerim'e tıklamak.

![Müşteri panelindeki Sitelerim bağlantısı](/img/admin/customers-list.png)

Tüm web siteleri, her birinin hemen altında bir **Yönet** düğmesiyle birlikte görüntülenir. Bu düğmeye tıkladıklarında, **ilgili web sitesinin paneline** yönlendirilirler.

Birden fazla web sitesine sahip olmalarına izin veren planlarda, web sitelerinden hangisinin **birincil** olduğunu da görebilirler.

![Yönet düğmeleriyle müşteri web sitelerini gösteren Sitelerim sayfası](/img/admin/customers-list.png)

Birincil web sitelerini değiştirmek istiyorlarsa, **Hesap** menüsüne tıklayarak Hesap sayfasına gitmeleri gerekir.

![Müşteri panelindeki Hesap menüsü bağlantısı](/img/admin/customers-list.png)

Ardından, **Varsayılan Siteyi Değiştir** seçeneğine tıklamalıdırlar.

![Hesap sayfasındaki Varsayılan Siteyi Değiştir düğmesi](/img/admin/customers-list.png)

Bir açılır pencere belirecektir. Açılır listeye tıklayın ve birincil yapmak istediğiniz web sitesini seçin.

![Varsayılan Siteyi Değiştir açılır liste seçimi](/img/admin/customers-list.png)

Son olarak, **Varsayılan Siteyi Değiştir** düğmesine tıklayın ve işlem tamamdır!

![Varsayılan site değişikliğini onaylama](/img/admin/customers-list.png)
