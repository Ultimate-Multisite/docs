# Content Sync Değişiklik Günlüğü {#content-sync-changelog}

### Version 1.1.0 - Yayınlanma Tarihi 2025-12-31 {#version-110---released-on-2025-12-31}
* Yeni: Tüm gönderi türlerinin (gönderiler, sayfalar, özel gönderi türleri) senkronizasyonunu destekleme
* Yeni: İçerik seçimi adımında gönderi türü filtrelemesi
* Yeni: Kaynak site seçimi adımı - yalnızca ana site değil, herhangi bir siteden senkronizasyon yapabilme
* Yeni: Şablon siteleri için alt site ön seçimi
* Yeni: Tüm gönderi türleri için post_content'ten medya çıkarma
* Yeni: Taksonomi senkronizasyonu (kategoriler, etiketler, özel taksonomiler)
* Değiştirildi: "Elementor Template Sync" adından "Content Sync" adını aldı
* Değiştirildi: "Select Templates" (Şablon Seç) adımı "Select Data" (Veri Seç) olarak yeniden adlandırıldı
* İyileştirildi: Elementor'a özgü kod artık yalnızca elementor_library gönderi türü için çalışıyor

### Version 1.0.0 - Yayınlanma Tarihi 2025-12-19 {#version-100---released-on-2025-12-19}
* İlk sürüm
* Şablon ve site seçimi için çok adımlı sihirbaz
* Gerçek zamanlı ilerleme geri bildirimi ile toplu işleme
* Otomatik ID yeniden eşleme ile tam medya senkronizasyonu
* Ultimate Multisite ile ayarlar entegrasyonu
* Ağ ayarları üzerinden site hariç tutma desteği
* Senkronizasyon sonrası Elementor önbellek temizleme
* Şablon taksonomisi koruma
