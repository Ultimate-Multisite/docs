# Filtre: wu_apc_should_display_admin_menu

Yöneticilerin, özelleştirme sürecini tamamlamışlarsa yönetici menüsü sayfalarını gizlemesine izin verir.

## Parametreler

| Name | Type | Description |
|------|------|-------------|
| $should_display_admin_menu | `bool` | Sayfaları göstermemiz gerekiyorsa. |
| $network_active | `bool` | Eklentinin ağ yöneticisi (network admin) tarafından etkinleştirilip etkinleştirilmediğini kontrol eder. |

### Versiyon

- 1.4.0
