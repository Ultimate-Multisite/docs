# Aksiyon: wp_ultimo_ptm_page_edit_after_title

Eklenti geliştiricilerinin sayfa düzenleme alanına ek düğmeler eklemesine olanak tanır.

## Parametreler

| Ad | Tip | Açıklama |
|------|------|-------------|
| $ | `object` | Bilgiyi içeren nesne |
| $ | `\WU_Page` | Multisite Ultimate Sayfa örneği |

### Kullanılabilir Versiyon

- 1.8.2
### Kaynak

[`views/base/edit.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-plugin-and-theme-manager/blob/main/views/base/edit.php#L48) dosyasının 48. satırında tanımlanmıştır.
