# Filtre: wu_cap_captcha_config

Cap Captcha yapılandırmasını filtreler.

Geliştiricilerin Cap Captcha ayarlarını özelleştirmesine olanak tanır.

## Parametreler

| Name | Type | Description |
|------|------|-------------|
| $config | `array` | Cap yapılandırma dizisi. |
| $level | `string` | Güvenlik seviyesi (fast, medium, max). |
