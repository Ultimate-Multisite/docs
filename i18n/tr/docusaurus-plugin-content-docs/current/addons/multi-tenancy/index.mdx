# Çok Kiracılık

Performans ve güvenlik için alt siteleri sunucular ve veritabanları arasında dağıtın

## Sürüm 1.2.0 yayın öne çıkanları {#version-120-release-highlights}

Multi-Tenancy 1.2.0, üretim izolasyonu iş akışları için egemen kiracı temelini ekler:

- Kısa ömürlü, yeniden oynatmaya karşı korumalı SSO ziyaretleriyle durumsuz kiracı otomatik girişi.
- Egemen kiracılar için alt site başına veritabanı ve dosya sistemi izolasyonu.
- Ana makine farkındalıklı kiracı sağlama için Hostinger çok kiracılık yeteneği.
- Eski veri kontrolleri ve async push işleme için egemen migration doğrulama kapıları.
- Bedrock, FrankenPHP ve localhost kurulumları için yapılandırılabilir aynı makine veritabanı ana makine bağlaması.
- Kiracı bootstrap, veritabanı yazıcısı, şema bootstrap ve kullanıcı sağlama iyileştirmeleri.
- Kiracı kimlik bilgilerini temizleyen daha güvenli egemen sökme.

[Egemen migration iş akışı](/user-guide/getting-started/sovereign-migration-workflow), [çok kiracılık izolasyonu](/user-guide/configuration/multi-tenancy-isolation) ve [egemen kiracı işlemleri](/user-guide/administration/sovereign-tenant-operations) için kullanıcı kılavuzlarına bakın.

## Gereksinimler {#requirements}

- WordPress 5.3 veya üzeri
- PHP 7.4 veya üzeri
- Multisite Ultimate eklentisi (etkin)

## Kurulum {#installation}

1. Ek eklenti dosyalarını `/wp-content/plugins/` dizininize yükleyin
2. WordPress içindeki 'Eklentiler' menüsünden eklentiyi etkinleştirin
3. Ek eklenti ayarlarını ağ yöneticisinde yapılandırın

## Geliştirme {#development}

### Kurulum {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

npm run build
```

### Test Etme {#testing}

Ek eklenti kapsamlı bir test framework'ü içerir:

- **Birim Testleri**: WordPress test paketiyle PHPUnit tabanlı testler
- **Temel Test Sınıfı**: Yaygın test senaryoları için yardımcı yöntemler sağlar
- **Test Kapsamı**: Kod kapsamını izlemek üzere yapılandırılmıştır
- **Multisite Testi**: Testler multisite ortamında çalışır

### Kod Standartları {#code-standards}

- **WordPress Kodlama Standartları**: PHPCS aracılığıyla uygulanır
- **PHP 7.4+ Uyumluluğu**: Modern PHP özellikleri desteklenir
- **Statik Analiz**: Tür denetimi için PHPStan entegrasyonu
- **Kod Modernleştirme**: PHP yükseltmeleri için Rector kuralları

### Derleme Sistemi {#build-system}

- **Varlık İşleme**: CSS/JS dosyaları için küçültme
- **Çeviri**: POT dosyası oluşturma
- **Paketleme**: Dağıtım için arşiv oluşturma
- **Geliştirme Modu**: Hata ayıklama için küçültülmemiş varlıklar

## Yapılandırma {#configuration}

Ek eklenti, ağ yöneticisi arayüzü üzerinden erişilebilen çeşitli yapılandırma seçeneklerini destekler.

### Egemen kiracı yapılandırması {#sovereign-tenant-configuration}

Egemen kiracılar için, trafiği kiracıya yönlendirmeden önce kiracı veritabanını, dosya sistemi kökünü, veritabanı ana makine bağlamasını ve ana makine yeteneğini yapılandırın. Veritabanı kimlik bilgilerini, ana makine bağlamasını, yönlendirmeyi veya kullanıcı sağlamayı etkileyen her değişiklikten sonra migration doğrulamasını çalıştırın.

`localhost` yalnızca kiracı çalışma zamanı ve veritabanı izinleri yerel soket veya aynı makine erişimi beklediğinde kullanın. Sağlayıcınız tarafından yapılandırılan izin ana makinesi bu olduğunda `127.0.0.1` veya bir hizmet ana makine adı kullanın.

### Hostinger yeteneği {#hostinger-capability}

Hostinger çok kiracılık yeteneği, paylaşılan Hostinger entegrasyon token'ı ile çalışır. Doğrulamadan önce domain'in hPanel'de mevcut olduğunu, DNS'in Hostinger hesabına işaret ettiğini ve kiracı veritabanı ayarlarının hPanel ile eşleştiğini doğrulayın.

## Hook'lar ve Filter'lar {#hooks-and-filters}

### Action'lar {#actions}

- `ultimate-multisite-addon-template_init` - Ek eklenti başlatıldıktan sonra tetiklenir
- `ultimate-multisite-addon-template_loaded` - Tüm ek eklenti dosyaları yüklendikten sonra tetiklenir

### Filter'lar {#filters}

- `ultimate-multisite-addon-template_settings` - Ek eklenti ayarlarını değiştirin
- `ultimate-multisite-addon-template_enabled` - Ek eklenti etkin/devre dışı durumunu geçersiz kılın
