# Material WP {#material-wp}

WordPress admin arayüzü yıllardır aynı. Peki, bir Material Design versiyonunun admin arayüzünün nasıl görünebileceğini hiç merak ettiniz mi?

Material WP, size o görünümü ve hissi vermek için geliştirilmiş bir admin temasıdır ve bu sayede müşterilerinize daha özelleştirilmiş bir Dashboard sunmanıza yardımcı olur.

## Özellikler {#features}

- Özel CSS desteği
- Kolay stil ayarları
- Menü yeniden sıralama
- Kolay import/export

## Dokümantasyon {#documentation}

**Material WP ile Admin Panel'inizi Özelleştirme**

## Kurulum {#installation}

1. 'material-wp' dosyasını '/wp-content/plugins/' dizinine yükleyin
2. WordPress'teki 'Plugins' menüsünden eklentiyi etkinleştirin

## Kullanım Kılavuzu {#usage-guide}
