# Fluent Forms

Bu eklenti, Ultimate Multisite üzerinde ürünlerde, üyeliklerde veya sitelerde Fluent Forms'a limit eklemenizi sağlar.
Fluent Forms, özel gönderi türleri yerine özel tablolar kullandığı için, normal limit seçenekleri Fluent Forms ile çalışmaz.

## Özellikler {#features}

- Fluent Forms veya Conversational Fluent Forms için limit ekleme
- Müşteriler, limitlerine ulaştıklarında planlarını yükseltmeleri gerektiğini belirten dostça bir mesaj alır.
