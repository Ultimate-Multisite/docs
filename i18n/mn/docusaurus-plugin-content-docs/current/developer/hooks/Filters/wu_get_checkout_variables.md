# Шүүлтүүр: wu_get_checkout_variables {#filter-wugetcheckoutvariables}

Плагин хөгжүүлэгчдэд checkout хуудасны урьдчилан тохируулсан утгуудыг шүүх боломж олгоно.

Болгоомжтой байгаарай, дутуу түлхүүрүүд нь front-end дээрх checkout-ийг бүрэн эвдэж болзошгүй.

## Параметрүүд {#parameters}

| Нэр | Төрөл | Тайлбар |
|------|------|-------------|
| $variables | `array` | Локалчлагдсан хувьсагчид. |
| $checkout | `\Checkout` | Checkout класс. |

### Үүнээс хойш {#since}

- 2.0.0
### Эх сурвалж {#source}

[`inc/checkout/class-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout.php#L1970)-д 1970-р мөрөнд тодорхойлсон


## Буцаана {#returns}
Шинэ хувьсагчдын массив.
