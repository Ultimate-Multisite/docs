# Шүүлтүүр: wu_pre_save_settings {#filter-wupresavesettings}

Ultimate Multisite-аар хадгалахаас өмнө тохиргоонуудыг шүүх боломжийг хөгжүүлэгчдэд олгоно.

## Параметрүүд {#parameters}

| Нэр | Төрөл | Тайлбар |
|------|------|-------------|
| $settings | `array` | Хадгалагдах тохиргоонууд. |
| $settings_to_save | `array` | Нэмэх шинэ тохиргоонууд. |
| $saved_settings | `array` | Одоогоор хадгалагдсан тохиргоонууд. |

### Хойш {#since}

- 2.0.18
