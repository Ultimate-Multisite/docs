# Шүүлтүүр: get_site_url_for_previewer {#filter-getsiteurlforpreviewer}

Өргөтгөл хөгжүүлэгчдэд урьдчилан харагчид ашиглах URL-г шүүх боломж олгоно

## Параметрүүд {#parameters}

| Нэр | Төрөл | Тайлбар |
|------|------|-------------|
| $domain | `string` | Одоогоор ашиглаж буй өгөгдмөл домэйн, өөрчлөлт хийхэд хэрэгтэй |
| $domain_options | `array` | Ultimate Multisite Тохиргоо -&gt; Сүлжээний тохиргоо -&gt; Домэйн сонголтууд хэсэгт оруулсан бүх домэйн сонголтын жагсаалт |

### Хувилбараас эхлэн {#since}

- 1.7.2
### Эх сурвалж {#source}

[`inc/checkout/class-legacy-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-legacy-checkout.php#L812) дотор 812-р мөрөнд тодорхойлсон
