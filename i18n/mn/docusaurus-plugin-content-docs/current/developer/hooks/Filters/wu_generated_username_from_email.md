# Шүүлтүүр: wu_generated_username_from_email

Үүсгэсэн хэрэглэгчийн хэрэглэгчийн нэрийг шүүх.

## Параметрүүд {#parameters}

| Нэр | Төрөл | Тайлбар |
|------|------|-------------|
| $username | `string` | Үүсгэсэн хэрэглэгчийн нэр. |
| $email | `string` | Шинэ хэрэглэгчийн имэйл хаяг. |
| $new_user_args | `array` | Шинэ хэрэглэгчийн аргументуудын массив, магадгүй нэр болон овгийг багтаасан. |
| $suffix | `string` | Хэрэглэгчийн нэрийг давтагдашгүй болгохын тулд залгах мөр. |
