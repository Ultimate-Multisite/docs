# Шүүлтүүр: wu_documentation_get_link

Plugin хөгжүүлэгчдэд холбоосуудыг шүүх боломж олгоно.

Энэ нь хэрхэн хэрэгтэй байж болохыг сайн мэдэхгүй ч байхад гэмгүй.

## Параметрүүд {#parameters}

| Нэр | Төрөл | Тайлбар |
|------|------|-------------|
| $link | `string` | Бүртгэгдсэн холбоос |
| $slug | `string` | Холбоосыг авахад ашигласан slug |
| $default_link | `string` | Бүртгэгдсэн анхдагч холбоос |

### Хойш {#since}

- 1.7.0
