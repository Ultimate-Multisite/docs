# Шүүлтүүр: wu_independent_billing_cycle_product_types

Тусдаа төлбөрийн мөчлөгтэй бүтээгдэхүүний төрлүүдийг шүүнэ.

Тусдаа төлбөрийн мөчлөгтэй бүтээгдэхүүнүүд нь сагсанд төлөвлөгөөний төлбөрийн хугацаатай заавал таарах шаардлагагүй. Энэ нь төлөвлөгөө сар бүр эсвэл жил бүр байсан ч үргэлж жилээр төлбөр тооцдог домэйн бүртгэл зэрэг бүтээгдэхүүнд хэрэгтэй.

## Параметрүүд

| Нэр | Төрөл | Тайлбар |
|------|------|-------------|
| $types | `array` | Тусдаа төлбөрийн мөчлөгтэй бүтээгдэхүүний төрлүүдийн массив. |
