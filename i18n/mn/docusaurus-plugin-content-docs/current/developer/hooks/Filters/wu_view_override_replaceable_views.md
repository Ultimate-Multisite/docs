# Шүүлтүүр: wu_view_override_replaceable_views {#filter-wuviewoverridereplaceableviews}

Хөгжүүлэгчдэд солих боломжтой жагсаалтад нэмэлт хавтас нэмэхийг зөвшөөрдөг.

Болгоомжтой байгаарай, учир нь нэмэлт хавтас зөвшөөрөх нь Ultimate Multisite хувилбаруудын оронд хуучирсан хуулбаруудыг ачаалахад хүргэж болзошгүй.

## Параметрүүд {#parameters}

| Нэр | Төрөл | Тайлбар |
|------|------|-------------|
| $replaceable_views | `array` | Зөвшөөрөгдсөн хавтаснуудын жагсаалт. |

- 2.0.0
