# Шүүлтүүр: wu_magic_link_enforce_user_agent {#filter-wumagiclinkenforceuseragent}

Хэрэглэгчийн агентын баталгаажуулалтыг албадан хэрэгжүүлэх эсэхийг шүүнэ.

Токенуудыг өөр өөр хөтөч/төхөөрөмж дээр ажиллуулахын тулд false болгож тохируулна. Энэ нь аюулгүй байдлыг бууруулдаг ч ашиглахад хялбар байдлыг нэмэгдүүлдэг.

## Параметрүүд {#parameters}

| Нэр | Төрөл | Тайлбар |
|------|------|-------------|
| $enforce | `bool` | Хэрэглэгчийн агент таарч байгаа эсэхийг албадан шалгах эсэх. |
