# Шүүлтүүр: wu_add_new_domain_modal_fields

Шинэ домэйн нэмэх модалын талбаруудыг шүүнэ.

Нэмэлтүүдэд (жишээ нь Domain Seller) домэйн бүртгэлийн талбаруудыг өөрчлөх эсвэл солих боломж олгоно.

## Параметрүүд {#parameters}

| Нэр | Төрөл | Тайлбар |
|------|------|-------------|
| $fields | `array` | Формын талбарууд. |

### Хувилбараас {#since}

- 2.1.0
