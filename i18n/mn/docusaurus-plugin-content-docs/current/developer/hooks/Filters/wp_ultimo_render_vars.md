# Шүүлтүүр: wp_ultimo_render_vars {#filter-wpultimorendervars}

Plugin хөгжүүлэгчдэд render контекстэд нэмэлт хувьсагчийг глобалаар нэмэх боломж олгоно.

## Параметрүүд {#parameters}

| Нэр | Төрөл | Тайлбар |
|------|------|-------------|
| $args | `array` | render дуудлагаар дамжуулсан хувьсагчдыг агуулсан массив. |
| $view | `string` | Render хийх view-ийн нэр. |
| $default_view | `string` | fallback_view-ийн нэр |
