# Шүүлтүүр: wu_apply_plan_limits

Plugin хөгжүүлэгчдэд хязгаарлалтуудыг богино холболтоор алгасах боломж олгоно.

Та энэ filter-ийг ашиглан аливаа хязгаарлалт эхлэхээс өмнө дурын код ажиллуулж болно. Хэрэв таны filter ямар нэг truthy утга буцаавал процесс үргэлжилнэ, харин ямар нэг falsy утга буцаавал код буцаж, доорх hook-уудын аль нь ч ажиллахгүй.
