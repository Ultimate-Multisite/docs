# Шүүлтүүр: wu_username_from_email {#filter-wuusernamefromemail}

Шинэ хэрэглэгчийн хэрэглэгчийн нэрийг шүүнэ.

## Параметрүүд {#parameters}

| Нэр | Төрөл | Тайлбар |
|------|------|-------------|
| $username | `string` | Хэрэглэгчийн хэрэглэгчийн нэр. |
| $email | `string` | Шинэ хэрэглэгчийн имэйл хаяг. |
| $new_user_args | `array` | Шинэ хэрэглэгчийн аргументуудын массив, овог болон нэрийг агуулж магадгүй. |
| $suffix | `string` | Хэрэглэгчийн нэрийг давтагдашгүй болгохын тулд нэмж залгах мөр. |
