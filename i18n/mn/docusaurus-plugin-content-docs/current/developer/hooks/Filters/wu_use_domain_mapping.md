# Шүүлтүүр: wu_use_domain_mapping {#filter-wuusedomainmapping}

Маппинг ашиглах эсэхийг тодорхойлно

Ихэвчлэн та зөвхөн идэвхтэй маппингуудыг ашиглахыг зөвшөөрөхийг хүснэ. Гэхдээ хэрэв илүү дэвшилтэт логик ашиглах эсвэл идэвхгүй домэйнүүдийг ч маппинг хийхийг зөвшөөрөхийг хүсвэл энд шүүнэ үү.

## Параметрүүд {#parameters}

| Нэр | Төрөл | Тайлбар |
|------|------|-------------|
| $is_active | `bool` | Маппингийг идэвхтэй гэж үзэх үү? |
| $mapping | `\Domain` | Бидний шалгаж буй маппинг |
| $domain | `string` |  |
