# Шүүлтүүр: wu_admin_notices

Ultimate Multisite-ээр нэмэгдсэн админ мэдэгдлүүдийг шүүх боломжийг хөгжүүлэгчдэд олгоно.

## Параметрүүд

| Нэр | Төрөл | Тайлбар |
|------|------|-------------|
| $notices | `array` | Тухайн самбарын мэдэгдлүүдийн жагсаалт. |
| $all_notices | `array` | Самбараар ангилсан, нэмэгдсэн мэдэгдлүүдийн жагсаалт. |
| $panel | `string` | Мэдэгдлүүдийг авах самбар. |
| $filter | `string` | Хааж болох мэдэгдлүүд шүүгдсэн эсэх. |
| $dismissed_messages | `array` | Хаасан мэдэгдлийн түлхүүрүүдийн жагсаалт. |

### Хувилбараас

- 2.0.0
