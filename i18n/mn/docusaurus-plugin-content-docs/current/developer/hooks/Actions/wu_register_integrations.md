# Үйлдэл: wu_register_integrations {#action-wuregisterintegrations}

Интеграцууд өөрсдийгөө бүртгүүлэх боломж олгохын тулд ажиллана.

Core өөрийн интеграцуудыг энд бүртгэдэг.

## Параметрүүд {#parameters}

| Нэр | Төрөл | Тайлбар |
|------|------|-------------|
| $registry | `\Integration_Registry` | Бүртгэлийн экземпляр. |

- 2.5.0
