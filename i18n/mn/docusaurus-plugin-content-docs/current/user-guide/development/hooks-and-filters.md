# Ultimate Multisite Hooks ба filters

Ultimate Multisite нь олон функц болон өгөгдлийн хайлт (database queries)-аар бүтсэн юм. Энэ нь таны байгаа функцуудыг өөрчлөх эсвэл шинэ код нэмэх боломжийг олгодог hooks ба filters-ийг агуулдаг.

Бид энэ хувьд хэрэглэгчдэд туслах зорилгоор энэхүү hooks ба filters-ийн жагсаалтыг бэлтгэсэн. Ажлынхаа дараа илүү нарийн тохируулга хийхийг хүс는 хүмүүст энэ жагсаалтаас хандах боломжтой. Жагсаалтыг доорх холбоосуулаар Github-ээс үзэх боломжтой.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
