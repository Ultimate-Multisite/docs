# Администратор панелийг PRO Theme-ээр тохируулах

Клиенттэй ажиллах үед бид ихэвчлэн WordPress-ийн администратор талбарт өөрсдийн клиентуудад өөр, онцгой туршлага үзүүлэх шаардлагатай байдаг. Хэрэв та илүү мэргэжлийн харагдах болон мэдрэмжтэй хэрэгсэл олохыг хүсвэл PRO Theme танд туслах болно.

PRO Theme нь өөрийнхөө клиентуудын Dashboard-ийг тохируулахад тань туслах зорилгоор бүтээгдсэн администратор theme юм. Мөн энэ нь Ultimate Multisite-ийн premium plugins core-ын хэсэг юм.

Түүнийг суулгасны дараа, _Plugins_ хуудсанд очиж өөрийг таны Network дээр идэвхжүүлэхийг сонгоно.

<!-- Зураг байхгүй: WP Plugins хуудас PRO Theme-ийн Network Activate товчлуур байгааг харуулсан -->

_Plugins хуудсанд network-ээр идэвхжүүлнэ._

_**Тайлбар:**_ _Хэрэв та зөвхөн суулгавал, энэ нь таны клиентын хувьд тухайн хэрэглэгчэд л theme-ийг нэмэх боломжийг тэмдэглэж харуулах болно._

Одоо таны администратор интерфейс бүрэн шинээр загварлагдсан байна! Зүүн багананд PRO Theme гэсэн шинэ цэс бас гарч ирсэн.

<!-- Зураг байхгүй: PRO Theme-ийн шинээр загварлагдсан admin panel interface -->

_Администратор талбарын шинэ харагдац._

_Styling Settings (Загварлагын тохиргоо)_

<!-- Зураг байхгүй: PRO Theme Functionality таб -->

_Custom CSS (Өөрийн CSS-ийг нэмэх)_

<!-- Зураг байхгүй: PRO Theme Activate таб -->

Үүний тус бүрт та загварлагын зүйлсийг идэвхжүүлэх эсвэл унтраах, үйл ажиллагааг зөвшөөрөх эсвэл хориглох боломжийг олгож, таны сайтуудын network дээр клиентуудад хэрэгтэй тохиргоог өгч чадна.
