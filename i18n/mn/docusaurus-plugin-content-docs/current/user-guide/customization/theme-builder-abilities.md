# Тема Боловсруулалт: Блок Тэмыг Сэргээх ба Идэвхжүүлэх

Superdav AI Agent v1.12.0 нь чат интерфейсээс шууд өөрийн тохируулсан блок тэмуудыг үүсгэх, хэрэгжүүлэх хоёр хүчирхэг чадварыг нэвтрүүлсэн.

## Ерөнхий өгөгдөл

**scaffold-block-theme** ба **activate-theme** чадварууд нь агентуудад дараах үйлдлүүдийг хийх боломжийг олгодог:
- Таны тодорхойлсон зүйлсээр бүрэн, ажиллуулж болох блок тэмуудыг үүсгэх
- Хөтөлгөөнт дизайн шийдвэр гаргаснаар таны сайтод гараар оролцохгүйгээр тэмуудыг автоматаар идэвхжүүлэх
- Уялдуулсан дизайны өнгө аяс (visual identity) үүсгэх

## Блок Тэмыг Сэргээх (Scaffold Block Theme)

**scaffold-block-theme** чадвар нь бүрэн тэмуудын бүтцийг, дараах зүйлсийг багтаасан шинэ WordPress блок тэмыг үүсгэнэ:

- Дизайн токентай `theme.json` тохиргоо
- Ерөнхий байрлалууд (layouts) дээрх Блок загварын файлнууд
- Тохируулсан блок стил ба хувилбарууд
- Тэмуудын метаданс болон дэмжлэг үзүүлэх зарчим

### Хэрхэн дуудах вэ?

Superdav AI Agent-тай чат хийх үед та тэмуу үүсгэх хүсэлт гаргаж болно:

```
"Modern Agency" гэсэн блок тэмыг цэнхэр ба цагаан өнгөөр, sans-serif типографик ашиглан, мэргэжлийн байдлаар загварч үүсгэ."
```

Агент дараах үйлдлүүдийг гүйцэтгэнэ:
1. Чат дунд таны дизайн сонголтуудыг цуглуулах
2. Бүтэн тэмуу бүтцийг үүсгэх
3. Шаардлагатай бүх тэмуу файлуудыг бэлдэх
4. Идэвхжүүлэхэд бэлэн болгох

### Хүлээгдэж буй гаралтын хэлбэр

Чадвар амжилтанд гүйцэтгэгдэх үед та дараах зүйлсийг харах болно:

- Тэмуу сэргээгдсэн тухай баталгаа
- Тэмууны нэр болон байрлал
- Хэрэглэсэн дизайн токенуудын (өнгө, типографик, зай) хураангуй
- Идэвхжүүлэхэд бэлэн байгаа мэдээлэл

Жишээ гаралтын хэлбэр:
```
✓ "Modern Agency" тэмуу амжилтанд гүйцэтгэгдсэн
  Байрлал: /wp-content/themes/modern-agency/
  Өнгө: Үндсэн #0066CC, Хоёр дахь #FFFFFF
  Типографик: Inter (sans-serif)
  Статус: Идэвхжүүлэхэд бэлэн
```

## Тэмууг Идэвхжүүлэх (Activate Theme)

**activate-theme** чадвар нь таны сайтыг шинээр бүтээсэн эсвэл одоо байгаа блок теми руу шилжүүлдэг функц юм.

### Хэрхэн ашиглах вэ?

Темийн загварыг үүсгэсний дараа, та түүнийг шууд идэвхжүүлэх боломжтой:

```
"Modern Agency темы-г идэвхжүүлэх"
```

Эсвэл аль нэгэн одоо байгаа темы руу шилжихэд:

```
"Twentytwentyfour тема руу шилжих"
```

### Хүлээгдэж буй үр дүн

Идэвхжүүлэх амжилттай болгосон үед:

- Идэвхтэй темуйн баталгаажуулга
- Өмнөх темыйн нэр (анхны харьцуулалтад зориулсан)
- Тема одоо ажиллаж буй сайтын URL
- Тематай холбоотой ямар нэгэн тохиргооны тэмдэглэл

Жишээний үр дүн:
```
✓ Тема амжилттай идэвхжүүлэв
  Идэвхтэй тема: Modern Agency
  Өмнөх тема: Twentytwentyfour
  Ажиллаж байна: https://yoursite.com
  Тэмдэглэл: Хүснэгтийн байрлалыг шалгахад үндсэн хуудсаа үзнэ үү
```

## Ажиллагааны дараалал: Бүтээх ба Идэвхжүүлэх (Scaffold and Activate)

Ерөнхий ажиллах арга нь хоёр чадварыг хослуулдаг:

1. **Темийн бүтцийг хүсэх**: "Миний SaaS лендинг хуудсанд блок теми үүсгэ"
2. **Agent-аар темуг бүтээх**: Файл болон дизайн токенуудыг үүсгэнэ
3. **Шалгах ба сайжруулах**: Хэрэв шаардлагатай бол дизайны өөрчлөлтөө хэлэлцэх
4. **Идэвхжүүлэх**: "Одоо темы-г идэвхжүүл"
5. **Баталгаажуулах**: Шинэ дизайн ажиллаж байгаа эсэхийг сайтынхаа дээр үзээрэй

## Дизайн токен ба Тохиргоо (Design Tokens and Customization)

Бүтээсэн темы нь WordPress-ийн дизайн токенуудыг (`theme.json` ашиглан) дараах зүйлүүдэд ашигладаг:

- **Өнгө**: Үндсэн, хоёрдогч, онцгой, нейтралын палитр
- **Текстний хэлбэр (Typography)**: Font family, хэмжээ, жин, мөрний зай
- **Зайны тоо (Spacing)**: Padding, margin, gap scales
- **Хувиргалт (Borders)**: Радиус ба өргөнгийн токенууд
- **Сүүдэр (Shadows)**: Өндрүүдийн түвшин

Эдгээр токенууд `theme.json`-д төвлөрсөн байдаг тул бүх дизайн системээ нэг файлнаас хялбархан тохируулах боломжтой.

## Хязгаарлалт ба Тэмдэглэл

Темаууд `/wp-content/themes/`-д байрлалд каркасчлагддаг бөгөөд WordPress-ийн нэрлэх зааврыг дагах ёстой.
Активаци хийхийн тулд таны WordPress сайтод тохирох эрх шаардлагатай.
Тема доторх өөр PHP код бага байх ёстой; төвөгтэй функцуудыг plugin ашиглана уу.
Блок темууд WordPress 5.9 болон түүнээс дээш дээр хамгийн сайн ажилладаг.

## Асуудал шийдэх

**Каркасчласны дараа тема гарч ирэхгүй байна**
- Тема байгаа каталог (directory) байрлал, зөв эрхтэй эсэхийг шалгана уу.
- `theme.json` файл нь зөв JSON форматтай эсэхийг шалгана уу.
- Темагийн нэр одоо байгаа темуудтай зөрчилдөөгүй эсэхийг баталгаажуулна уу.

**Активаци дутуу байна**
- Та администратор эрхтэй байгаа эсэхийг баталгаажуулна уу.
- Темагийн каталог WordPress-д хандахад боломжтой (readable) байгаа эсэхийг шалгана уу.
- Дэлгэрэнгүй мэдээллийг авахын тулд WordPress-ийн алдааны логуудыг үзнэ үү.

**Design tokens хэрэгжихгүй байна**
- `theme.json`-ийн синтакс зөв байгаа эсэхийг шалгана уу.
- Ямар ч кешинг хийх plugin-ыг цэвэрлэж үзээрэй.
- Таны WordPress хувилбар ашиглаж буй tokens-тэй нийцсэн эсэхийг шалгана уу.

## Дараагийн алхамууд

Темаа активирсны дараа та:
- **Design System Aesthetics** чадварыг ашиглан типографи, өнгө, зайг сайжруулж болно.
- WordPress block editor-ээр тусгай блок (block) стилийг тохируулах боломжтой.
- Темагийн `style.css` файл дотор өөрийн CSS-ийг нэмж болно.
- Тухайн хуудасны төрлүүдэд зориулсан өөрийн блок (block) загварыг үүсгэж болно.
