# WPMU DEV Нэгтгэл

## Ерөнхий өгүүлэл
WPMU DEV бол WordPress сайтууд зориулсан хостинг, plugin болон үйлчилгээ үзүүлдэг бүрэн WordPress платформоор. Энэхүү нэгтгэл нь Ultimate Multisite болон WPMU DEV хостинг хоорондын домен синхронизуулалт болон SSL сертификатын удирдлагыг автоматаар хийгддэг.

## Онцлог
- Домены автоматаар синхронизуулах
- SSL сертификатыг удирдах
- Өргөтгөсөн SSL сертификатын шалгах оролдлого

## Шаардлага
Энэхүү нэгтгэл нь та WPMU DEV дээр хостинг хийж байгаа эсэхийг автоматаар тодорхойлдог бөгөөд дотоод API-г ашигладаг. Хэрэв та WPMU DEV дээр хостинг хийж байгаа бол ямар нэгэн нэмэлт тохиргоо хийх шаардлагагүй.

Энэхүү нэгтгэл нь WPMU DEV дээр хостинг хийх үед автоматаар тодорхойлогдох `WPMUDEV_HOSTING_SITE_ID` тогтмол байгаа эсэхийг шалгадаг.

## Тохируулах заавар

### 1. WPMU DEV Хостингийг баталгаажуулах
Хэрэв та WPMU DEV дээр хостинг хийж байгаа бол шаардлагатай тогтмол аль хэдийн тодорхойлогдсон байх ёстой. Дараах зүйлсийг шалгана уу:

1. `WPMUDEV_HOSTING_SITE_ID` тогтмол таны орчинд (environment) тодорхойлогдсон эсэхийг
2. Та API-тай идэвхтэй WPMU DEV гишүүнч байхыг

### 2. Нэгтгэлийг идэвхжүүлэх
1. WordPress admin дээрээ Ultimate Multisite > Settings руу орно.
2. "Domain Mapping" таб руу шилжин орно.
3. "Host Integrations"-д доошоо уншиж, WPMU DEV нэгтгэлийг идэвхжүүлнэ.
4. "Save Changes" (Өөрчлөлтөө хадгалах) товчийг дарна.

## Хэрхэн ажилладаг вэ

### Домен синхронизуулалт
Ultimate Multisite-т домен тохируулсан үед:

1. Нэгтгэл нь WPMU DEV API-г ашиглан домены таны хостинг аккаунтад нэмнэ.
2. Мөн доменийн www хувилбарыг автоматаар нэмдэг.
3. Домен тохиргоо болон SSL сертификат олголтыг WPMU DEV хариуцдаг.

### SSL Сертификатын удирдлага

Энэхүү холболтыг SSL сертификатуудыг олгоход болон суулгахад зарим цаг хугацаа шаардах тул WPMU DEV хостингэд SSL сертификатын шалгах оролдлого (verification attempts) тоог нэмэгдүүлэхээр тохируулсан. Анхны тохиргооноос харьцуулахад, энэ нь стандарт 5 оролдлоос илүү 10 удаа хүртэл SSL сертификатын шалгалт хийхийг оролдоно.

## Чухал анхаарах зүйлс

### Домен арилгах
Одоогоор WPMU DEV API-д домен арилгах арга байхгүй байна. Ultimate Multisite-т домен тохиргоог арилгасны дараа, тухайн домен таны WPMU DEV хостинг аккаунтын дотор үлдэнэ. Хэрэв шаардлагатай бол та өөрөө WPMU DEV хостинг дээрх dashboard-оос гаргаж авах хэрэгтэй.

### API баталгаажуулалт (API Authentication)
Энэхүү холболт нь таны WordPress-ийн database-д `wpmudev_apikey` гэсэн option-ээр хадгалагдсан WPMU DEV API key-г ашигладаг. Та сайтын өөрийгөө WPMU DEV-тэй холбоход энэ нь автоматаар тохируулна.

## Асуудал шийдэх (Troubleshooting)

### API холболтын асуудлууд
- Таны сайт WPMU DEV-тэй зөв холбогдсон эсэхийг шалгаарай.
- `wpmudev_apikey` option нь таны WordPress database-д тохируулсан эсэхийг шалгаарай.
- Таны WPMU DEV гишүүнчлэл (membership) идэвхтэй байгаа эсэхийг баталгаажуулаарай.

### SSL сертификатын асуудлууд
- WPMU DEV нь SSL сертификат олгоход зарим цаг хугацаа шаардах боломжтой (ихэвчлэн 5-15 минут).
- Холболт нь SSL сертификатуудыг хүртэлх 10 удаа шалгахээр тохируулсан.
- Хэрэв олон оролдлогын дараа ч SSL сертификат олгогдоогүй бол WPMU DEV-ийн дэмжлэгтэй холбогдоно уу.

### Домен нэмээгүй байх
- Ultimate Multisite-ийн logs-ыг алдааны мессеж байгаа эсэхийг шалгаарай.
- Тухайн домен WPMU DEV-д аль хэдийн нэмэгдсэн эсэхийг баталгаажуул.
- Таны WPMU DEV хостинг төлөвлөгөө та нэмж буй домену дэмжих боломжтой байгаа эсэхийг шалгаарай.
