# WP Engine Холбог (Integration) {#wp-engine-integration}

## Ерөнхий өгүүлэл {#overview}
WP Engine бол WordPress сайтуудад оновчтой гүйцэтгэл, аюулгүй байдал, өргөжих чадварыг хангах премиум менеждэх WordPress хостинг платформ юм. Энэхүү холбоо нь Ultimate Multisite болон WP Engine хоорондын домен синхрололтыг автоматаар хийгддэг.

## Онцлог {#features}
- Домен синхрололт (Automatic domain syncing)
- Multisite суурийн хувьд дэддомын (subdomain) дэмжлэг
- WP Engine-ийн одоо байгаа системүүдтэй бөгөөд сулгүй холбоо

## Шаардлага {#requirements}
Холбоо нь таны WP Engine дээр хостинг хийж байгаа эсэхийг автоматаар тодорхойлно бөгөөд дотоод WP Engine API-ийг ашигладаг. Хэрэв WP Engine плагин идэвхтэй байвал зөв тохируулсан бол ямар нэгэн нэмэлт тохиргоо хийх шаардлагагүй.

Гэсэн хэдий ч, та өөрөө холбоог гараар тохируулах шаардлагатай бол `wp-config.php` файлд доор дурдсан тогтмол (constant)-ийн аль нэгийг тодорхойлж болно:

```php
define('WPE_APIKEY', 'your_api_key'); // Төвлөж буй арга
// Эсвэл
define('WPE_API', 'your_api_key'); // Бусад арга
```

## Тохируулгын заавар {#setup-instructions}

### 1. WP Engine Плагванг шалгах {#1-verify-wp-engine-plugin}

Хэрэв та WP Engine дээр хостинг хийж байгаа бол WP Engine плагин аль хэдийн суулгагдаж идэвхтэй байх ёстой. Дараах зүйлсийг шалгана уу:

1. WP Engine плагин идэвхтэй эсэхийг шалгах
2. `wp-content/mu-plugins/wpengine-common/class-wpeapi.php` файл байгаа эсэхийг шалгах

### 2. Холбоог идэвхжүүлэх {#2-enable-the-integration}

1. WordPress администратор руу орж, Ultimate Multisite > Settings рүү очино
2. "Domain Mapping" (Домен тохиргоо) таб руу шилжих
3. "Host Integrations" (Хост холбоо) хэсэг рүү доошоо унагаарай
4. WP Engine холбоог идэвхжүүлэх
5. "Save Changes" (Өөрчлөлтөө хадгалах) дээр дарна

## Хэрхэн ажилладаг вэ {#how-it-works}

### Домен синхрололт {#domain-syncing}

Ultimate Multisite-т домен тохируулсан үед:

1. Холбоо нь WP Engine API-г ашиглан тухайн домены таны WP Engine суурь руу нэмнэ
2. WP Engine доменийн тохиргоо болон SSL сертификатыг гаргах ажлыг хийдэг
3. Домен тохиргоог хассан үед, холбоо нь тухайн доменыг WP Engine-ээс арилгана

### Дэддомын дэмжлэг {#subdomain-support}

Поддомен мультисайт-установок:

1. Интеграция добавляет каждый поддомен в WP Engine при создании нового сайта.
2. WP Engine сам обрабатывает настройку поддомена.
3. Когда сайт удаляется, интеграция убирает этот поддомен из WP Engine.

## Важные замечания {#important-notes}

### Дикие домены (Wildcard Domains) {#wildcard-domains}

Для установок мультисайта на поддоме рекомендуется связаться со службой поддержки WP Engine и запросить настройку дикого домена (wildcard domain). Это позволит всем поддоменам работать автоматически, не нужно добавлять каждый из них по отдельности.

### SSL-сертификаты {#ssl-certificates}

WP Engine автоматически занимается выдачей и продлением SSL-сертификатов для всех доменов, добавленных через эту интеграцию. Дополнительная настройка не требуется.

## Устранение неполадок (Troubleshooting) {#troubleshooting}

### Проблемы с подключением к API {#api-connection-issues}
- Проверьте, что плагин WP Engine активен и правильно настроен.
- Если вы вручную определяли API ключ, проверьте, правильный ли он.
- Свяжитесь со службой поддержки WP Engine, если у вас возникли проблемы с API.

### Домен не добавлен {#domain-not-added}
- Проверьте логи Ultimate Multisite на наличие каких-либо сообщений об ошибках.
- Убедитесь, что домен еще не добавлен в WP Engine.
- Убедитесь, что ваш план WP Engine поддерживает количество добавляемых вами доменов.

### Проблемы с поддоменами {#subdomain-issues}
- Если поддомены не работают, свяжитесь со службой поддержки WP Engine и запросите настройку дикого домена (wildcard domain).
- Проверьте, правильно ли настроены ваши DNS-настройки для основного домена и поддоменов.
