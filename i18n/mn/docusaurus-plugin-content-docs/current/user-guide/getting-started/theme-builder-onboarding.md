# Theme Builder-ийн Оршил Процесс (Onboarding Flow) {#theme-builder-onboarding-flow}

Superdav AI Agent v1.12.0 нь таны анхны тохируулгын үед өөрийн өвөрмөц блоктэй сэдэв (custom block theme) бүтээхэд туслах **Theme Builder-ийн оршил процессыг** нэвтрүүлсэн. Энэ нь хуучин Site Builder модыг илүү уян хатан, агент туслалтай аргаар солино.

## Theme Builder Оршил Процесс гэж юу вэ? {#what-is-the-theme-builder-onboarding-flow}

Theme Builder-ийн оршил процесс гэдэг нь интерактив тохируулгын wizards бөгөөд дараах зүйлсийг хийдэг:

- Дизайн шийдвэр гаргахад (бүцэн, типографик, байрлал) таныг удирдан чиглүүлнэ.
- Таны сайтын харагдах хэлбэрийн сонголтуудыг авдаг.
- Таны хэрэгцээнд тохирсон өөрийн блоктэй сэдэв (custom block theme)-ийг үүсгэнэ.
- Дуусгасны дараа сэдвийг автоматаар идэвхжүүлдэг.

Энэ процессыг **Setup Assistant agent** ажиллуулдаг бөгөөд тодруулсан асуултуудыг асууж, таны сэдвийг алхам бүрээр үүсгэнэ.

## Theme Builder-ийн Оршил Процессын Эхлэл {#starting-the-theme-builder-onboarding}

### Анхны Ажиллагаа (First-Run Setup) {#first-run-setup}

Та анх Superdav AI Agent-ийг шинэ WordPress-д ажиллуулахад дараах зүйлийг харна:

```
Superdav AI Agent-д тавтай морил!

Та юу хийхийг хүсэж байна вэ?
1. Өөрийн сэдэв бүтээх (Theme Builder)
2. Одоо байгаа сэдвийг ашиглах
3. Одоогийн байдлаар тохируулгыг алгасах
```

Theme Builder-ийн оршил процессыг эхлэхийн тулд **"Өөрийн сэдэв бүтээх"** гэсэн сонголтыг хийгээрэй.

### Гадаад Идэвхжүүлэх (Manual Activation) {#manual-activation}

Та ямар ч үед дараах командыг өгч Theme Builder-ийн оршил процессыг эхлүүлэх боломжтой:

```
"Start the Theme Builder onboarding"
```

эсвэл

```
"Help me create a custom theme"
```

## Оршил Процессын Алхам {#the-onboarding-steps}

### Алхам 1: Мод Сонголт (Mode Selection) {#step-1-mode-selection}

Setup Assistant agent таны сонголтыг асууна:

```
Та сэдвийг ямар аргаар бүтээхийг хүсэж байна вэ?
- Удирдан чиглүүлэх (Би асуулт асууж, таны төлөө үүнийг хийнэ)
- Өөрийн гараар хийх (Би сонголтуудыг харуулж, та шийднэ)
```

**Удирдан чиглүүлэх режим** нь ихэнх хэрэглэгчдэд тохиромжтой; агент таны салбар болон зорилгоор дизайн дээр зөвлөмж өгнө.

### Алхам 2: Сайтын Тодорхойлолт (Site Specification) {#step-2-site-specification}

Та сайтын талаар дараах асуултуудыг хариулах шаардлагатай болно:

- **Сайтны зорилго**: Электрон худалдаа (E-commerce), блог, портфолио, SaaS гэх мэт.
- **Зорилтот үзэгч**: Таны үзэгчид хэн бэ?
- **Брэндийн өнгө**: Үндсэн ба хоёрдогч өнгө (эсвэл "мэдэхгүй")
- **Тон**: Профессионал, бүтээлч, тоглоомтой, минималист гэх мэт.

Энэ мэдээллийг таны **site_brief** санах ойд хадгална бөгөөд ирээдүйн сессионнуудад агентууд үүнийг дурдах болно.

### 3-р алхам: Дизайн систем сонголтууд {#step-3-design-system-decisions}

Агент таны дизайн токенудыг сонгоход чиглүүлнэ:

- **Типографик**: Шрифт (serif, sans-serif, monospace) ба хэмжээний дараалал
- **Өнгөний палитр**: Үндсэн, хоёрдогч, онцгой ба нейтраль өнгөнүүд
- **Хуваарь**: Хүрэх байдал (compact, normal, spacious layouts)
- **Хөдөлгөөн**: Анимаци, шилжилт (хэрэв хүсвэл)

### 4-р алхам: Theme үүсгэх {#step-4-theme-generation}

Setup Assistant агент таны өөрийн тохируулсан блок темыг дараатайгаар бэлтгэнэ:

- Бүх дизайн токенууд багтсан `theme.json`
- Ерөнхий байрлалууд (үндсэн хуудас, блог, холбоо барих) -ийн блок загварууд
- Таны дизайн системтэй таарсан өөрийн блок стилүүд
- Theme metadata болон WordPress дэмжлэх зарлалт

### 5-р алхам: Идэвхжүүлэх ба шалгах {#step-5-activation-and-verification}

Тема автоматаар идэвхжүүлнэ, дараах зүйлсийг харах болно:

```
✓ Таны өөрийн тему одоо ажиллаж байна!
  Theme name: [Таны Сайтын Нэр] Theme
  Colors: [Үндсэн] / [Хоёрдогч]
  Typography: [Шрифтийн Үндэс]

  Шинэ дизайныг харахын тулд таны сайтод зочилж үзээрэй.
```

Дараа нь темы зөв харагдаж байгаа эсэхийг шалгахын тулд таны сайтод очиж үзэх боломжтой.

## Сайтын тодорхойлолт ба site_brief Санах ой {#site-specification-and-sitebrief-memory}

Оршил үеийн хамтад агент таны сайтын тодорхойлолтыг **site_brief** санах ойн ангилалд цуглуулна. Энэ нь дараах зүйлсийг багтаана:

- Сайтын зорилго ба зорилго
- Зорилтот үзэгч
- Брэндийн өнгө ба тон
- Дизайны сонголтууд
- Агуулгын бүтэц

### site_brief яага чухал вэ? {#why-sitebrief-matters}

Ирээдүйн сессионнуудад агентууд таны site_brief-ийг дурдах замаар:

Дизайн өөрчлөлт хийх үед дизайны нэгдмэл байдлыг хадгалах
Таны сайтын зорилгод нийцсэн функцуудыг санал болгох
Орлогоос хамаарах зөвлөмжүүдийг өгөх

### Site brief-ийг үзэх {#viewing-your-sitebrief}

Та дараах байдлаар асуугаач:

```
"Show me my site brief"
```

эсвэл

```
"What do you know about my site?"
```

Агент таны хадгаласан сайтын тодорхойлолтыг (site specification) харуулна.

## Onboarding дууссаны дараа өөрчлөлт хийх {#customizing-after-onboarding}

Theme Builder-ийн onboarding дууссаны дараа та дараах үйлдлүүдийг хийж болно:

### Design System Aesthetics skill-ийг ашиглах {#use-the-design-system-aesthetics-skill}

Дизайны зарим зүйлийг сайжруулахыг хүсвэл:

```
"Refine the typography to be more modern"
```

эсвэл

```
"Adjust the color palette to be warmer"
```

**Design System Aesthetics skill** танд зориулсан дизайн шинэчлelser хийхэд чиглүүлнэ.

### theme.json-ийг шууд өөрчлөх {#edit-themejson-directly}

Урьдчилсан хэрэглэгчид `/wp-content/themes/[theme-name]/theme.json`-ыг зааж, дараах зүйлсийг тохируулах боломжтой:

- Color tokens (Өнгөний утгууд)
- Typography scales (Үсгийн хэмжээний хуваарь)
- Spacing values (Хуучин зайны утгууд)
- Border and shadow definitions (Хувир ба сүүдэрлэх тодорхойлолт)

### Custom Block Templates үүсгэх {#create-custom-block-templates}

WordPress block editor ашиглан дараах зүйлсийн хувьд өөрийн загваруудыг бүтээж болно:

- Homepage layouts (Үндсэн хуудсан байрлал)
- Blog post pages (Блог постны хуудас)
- Product pages (Барааны хуудас)
- Contact forms (Холбоо мэдээллийн маягт)

## Харьцуулсан хандалт: Хуучин ба Шинэ Onboarding {#comparison-old-vs-new-onboarding}

| Онцлог | Site Builder (Угсаа) | Theme Builder (Шинэ) |
|---|---|---|
| Тохируулах арга | Wizard-based form (Үйл явцын маягт) | Agent-guided conversation (Агентийн удирдан яриа) |
| Theme үүсгэх | Limited templates (Хязгаарлагдмал загварууд) | Custom scaffolding (Өөрийн бүтцийг тохируулах боломжтой) |
| Design tokens | Manual entry (Гараар бичих) | Guided decisions (Удирдан шийдвэр гаргах) |
| Хүчтэй байдал | Fixed options (Тодорхой сонголт) | Customizable (Өөрийн тохируулгатай) |
| Ирээдүйн шинэчлэлт | Not referenced (Холбогдохгүй) | Stored in site_brief (site_brief-д хадгалагдана) |

## Асуудал шийдэх (Troubleshooting) {#troubleshooting}

**Onboarding flow дууссан**
- Flow-г дахин эхлүүлээрэй: "Start the Theme Builder onboarding"
- WordPress-ийн танилцуулгын хувилбарыг шинэчилсэн эсэхийг шалгаарай.
- Setup Assistant агент ажиллаж байгаа эсэхийг баталгаажуулаарай.

**Миний site_brief ашиглагдаагүй байна**
- Агент дээр санах ой (memory) байгаа эсэхийг баталгаажуулна уу.
- Агентид "миний site brief-ийг дурсана" гэж асуугаарай.
- Таны тохиргооноос санах ой идэвхтэй (enabled) байгаа эсэхийг шалгана уу.

**Үүсгэсэн theme миний хүссэн зүйлтэй таарахгүй байна**
- Design System Aesthetics чадварыг ашиглан үүнийг засахыг оролдог.
- Агентид "[тодорхой өөрчлөлт] -ээр theme-ийг дахин үүсгэхийг" гэж хүсэж болно.
- Нарийвчилсан хяналт тавихын тулд theme.json файлыг шууд засах хэрэгтэй.

## Дараагийн алхамууд {#next-steps}

Theme Builder-ийн сургалтыг дуусгасны дараа:

1. **Сайт шалгах**: Шинэ theme-ийг харахын тулд таны сайтод очиж үзээрэй.
2. **Дизайныг засах**: Тохируулгын хувьд Design System Aesthetics чадварыг ашиглана уу.
3. **Агуулга нэмэх**: Сайтын агуулгыг бүтээхийг эхлээрэй.
4. **Чадваруудыг судлах**: scaffold-block-theme болон activate-theme зэрэг бусад агентын чадвар тухай суралцаарай.
