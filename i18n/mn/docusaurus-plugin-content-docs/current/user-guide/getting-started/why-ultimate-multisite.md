# Яага Ultimate Multisite {#why-ultimate-multisite}

Энэ нийтлэлд хүрсэн бол та бидний плагин нь таны бизнесийн саналыг бодит болгоход туслах шийдэл байж магадгүй гэдгийг сонссон байх юм. Бид энэ нийтлэлийн төгсгөлд танд шаардлагатай хариултуудыг өгөх гэж байна.

## Ultimate Multisite гэж юу вэ? {#what-is-ultimate-multisite}

Ultimate Multisite бол WordPress-ийн Multisite плагин бөгөөд таны үйлчлүүлэгчдэд WaaS (Website as a Service - Үндсэн сайтын үйлчилгээ) санал болгох боломжийг олгодог. Бидний плагинтай та янз бүрийн хуваарийг (subscription tiers) ашиглаж, үйлчлүүлэгчдэд таны Multisite сүлжээнд хостлогдсон сайт авахын тулд тогтмол төлбөр хийхийг шаардах боломжтой.

Ultimate Multisite нь танд өөр өөр гүүрүүдийн (жишээлбэл, WooCommerce эсвэл Stripe) замаар төлбөрийг хүлээн авах, шинэ хэрэглэгч сүлжээнд бүртгэгдэх үед таны амьдралыг илүү хялбар болгох сайт загварууд (site templates) үүсгэх, доменүүдийг тохируулах болон үйлчлүүлэгчдэд өөрсдийн хувьд тохирсон туршлагыг бий болгохын тулд бүртгэлийн маягтуудыг хувиргах зэрэг үйлдлүүдийг хийхийг дэмждэг.

Мөн энэ нь зөвхөн эхлэл юм, учир нь та үүний функцийг add-on болон webhook ашиглан өргөтгөх боломжтой.

## Бусад функцууд {#other-features}

Ultimate Multisite дараах зүйлсийг санал болгодог:

  * Удирдлагын WordPress хостинг үйлчилгээтэй Домен Тохируулга (Domain Mapping integration): Ultimate Multisite нь шинэ домен тохируулагдахад WP Engine, Cloudways, cPanel, RunCloud, Closte зэрэг удирдлагын панелуудад тохируулсан доменүүдийг автоматаар нэмдэг тул та өөрөө хийх шаардлагагүй. Зарим тохиолдолд бид танд SSL сертификатын авах үйлдлийг ч үүсгэж өгдөг.

  * Webhooks: Ultimate Multisite нь таны сүлжээг олон янзын үйлчилгээтэй холбох боломжийг олгодог хүчирхэг Webhook системтэй. Бид Zapier-тэй шууд интеграц хийх боломжийг ч санал болгож байгаа бөгөөд энэ нь таны сүлжээг тэдний платформын 1000 гаруй аппликейшн ба үйлчилгээтэй холбодог.

* Хэрэглэгчдийн таны дуртай хуудас бүтээгч (page builder) ашиглан үйлчилгээ үзүүлэхэд туслах хүчирхэг Администратор хуудсууд: Ultimate Multisite-д Admin Pages Creator гэх хэрэгтэй add-on байдаг. Энэ нь таны үйлчлүүлэгчдийн dashboard дээр Elementor, Beaver Builder, Brizy шаблонуудыг ашиглан бүрэн өөрчилсөн Admin Pages-ийг нэмж оруулах боломжийг олгодог.

* Dashboard-ын туршлагыг үйлчлүүлэгчдийн хувьд тохируулдаг хоёр өөр Admin Theme: Check Material WP ба PRO.

Мөн илүү дэвшилттэй зүйлс:

* Template Overrides (Шаблонээ өөрчлөх) дэмжлэл: Ultimate Multisite-тай хамт та signup flow-ын анхны HTML шаблонуудыг өөрийн хувилбараар сольж болно. Энэ нь core plugin файлуудыг өөрчлөхөө санаа зовох шаардлагагүй байдаг.

* Search and Replace API (Олох ба солих API): Та сайт шаблонуудад {{site_name}} зэрэг placeholder-уудыг тэмдэглэж болно. Тэгээд Search and Replace API-ийг ашиглан, үйлчлүүлэгчийн signup хийсний дараа өгсөн мэдээллийг тухайн placeholder-уудад автоматаар дүүргэх боломжтой.

## Хэрэглээний жишээ (Use cases) {#use-cases}

Одоо Ultimate Multisite-ийн боломжит хэрэглээний жишээнүүдийг харцгаая.

### Жишээ 1: Агентлаг (An Agency) {#case-1-an-agency}

Агентлагийн үндсэн ур чадвар нь ихэвчлэн сайтуудыг дизайн хийх, тэдгээрийн хостинг эсвэл маркетингийг нэмэлт үйлчилгээ болгон дурдсан байдаг.

Агентлагуудад Ultimate Multisite нь нэг платформын дээр олон сайт хостлох болон удирдах чадвараараа гайхалтай утга өгдөг. Ялангуяа GeneratePress, Astra, OceanWP эсвэл бусад тодорхой theme-үүдээр дизайнгаа стандартын болгосон агентлагууд Ultimate Multisite-ийн чадварыг ашиглан шинэ сайт бүрийн хувьд эдгээр theme-уудыг автоматаар идэвхжүүлэх боломжтой.

Ultimate Multisite-ын тохиргоо замаар plugin болон theme-үүдийг тус бүрийн бүтээгдэхүүн дээр ямар нэгэн байдлаар ашиглах эсвэл нуух, мөн шинэ сайт үүсэхэд тэдгээрийг идэвхжүүлсэн эсэхийг тодорхой хянах боломжтой.

Агентлагу Ultimate Multisite-ий ашигтай байх бөгөөд тэд хамгийн сайн хийдэг зүйлээ - гайхалтай вэбсайт дизайныг бүтээж чадна гэдэгт итгэлтэй байх боломжийг олгодог.

### 2-р тохиолдол: Тусгайлан зориулсан үйлчилгээ үзүүлэгч (Niche Provider) {#case-2-niche-provider}

"Нэг зүйг сайн хийх" гэсэн хуучин ойлголт байдаг. Олон мэргэжилтнүүдэд энэ нь нэг үндсэн санаагаар бүтээл, үйлчилгээ гаргахыг хэлнэ.

Хэрэв та тусгайлан зориулсан вэбсайтуудын сүлжээг боловсруулахад анхаарч байгаа бол ерөнхий хүрээ болон платформын дээр суурилсан үйлчилгээ үзүүлэх нь илүү тохиромжтой байдаг. Энэ үед site templates (вэбсайтын загварууд) туслах боломжтой!

Ultimate Multisite-ийн шинэлэг онцлогчдын нэг бол template site юм. Энэ бол theme (сэдэв) суурилуулсан, идэвхжүүлсэн, шаардлагатай plugin-ууд суурилуулсан бөгөөд жишээний пост эсвэл хуудаснууд үүсгэсэн байдаг. Хэрэв хэрэглэгч template-аса үндэслэн шинэ сайт үүсгэвэл, template-ийн агуулга болон тохиргоо нь шинээр бүтээсэн сайтод хуулбарлагдана.

Тусгайлан зориулсан сайт болон үйлчилгээ үзүүлэгчдэд энэ нь өөртөө хэрэгтэй plugin болон дизайнтай бэлэн байх сайтыг шууд үүсгэх боломжийг харьцуулахад хамгийн давуу тал болдог. Хэрэглэгч зөвхөн хамгийн бага оролт өгнө үйлчилгээг дуусгах боломжтой.

### 3-р тохиолдол: WordPress Вэб хостинг (WordPress Web Hosting) {#case-3-wordpress-web-hosting}

WordPress сайтуудыг хостлах олон арга байдаг ч, хэрэглэгчдэд бэлэн суурилуулсан WordPress хувилбарыг вэб зай өгөхөөс илүү энгийн арга гэж ойлгох нь бага. Учир нь утга учиртай үйлчилгээ үзүүлэхийн тулд олон шийдвэр, анхаарах зүйлсийг нэгтгэх шаардлагатай байдаг.

Ultimate Multisite-ийн энэ талбарт гайхалтай үр дүн өгдөг бөгөөд WordPress сайтуудыг хостлогын бүх шийдлийг (turnkey solution) хамрах багц болгож өгдөг. Энэ шийдэлд төлбөргүй үйлчилгээ үзүүлэх, төлбөр цуглуулах, checkout forms (оройн хуудсууд), хямдралын ваучер болон хэрэглэгчтэй харилцах зэрэг үндсэн механизмууд багтсан байдаг.

WordPress Multisite-ий системий зөв суулгалт хийх, тохируулах, үргэлжлүүлэхэд шаардлагатай олон техникий ажил нь Ultimate Multisite туслах бөгөөд энэ нь сүлжээний администраторууд өөрийн үйлчилгээ эсвэл чиглэлээс хамаарах зүйлс, жишээлбэл бүтээгдэхүүний түвшин, үнэ, үйлчилгээний санал зэрэг талбаруудыг бодох шаардлагагүй байхаар хялбар болгодог.

Ultimate Multisite-тэй холбогдохыг хүс는 хөгжүүлэгчдэд энэ шийдэл нь бүрэн RESTful API болон үйл явдлын мэдэгдэл хийх Webhooks-ийг санал болгодог.

Олон гадаад plugin болон лиценз шаардахгүйгээр, Ultimate Multisite нь Wix, Squarespace, [WordPress.com](https://WordPress.com) зэрэг бусад үйлчилгээтэй харьцуулахад онцлог баялаг функцтэй шийдлийг санал болгодог.
