# Хүрэх тайлбарууд

## Хувилбар 2.13.0 — 2026-06-05-т гаргагдсан {#version-2130--released-on-2026-06-05}

Шинэ: Хэрэглэгчийн данс, төлбөлт, сайт, вэб хуудасны загвар солих, домен тохиргоо зэрэг үйл явцуудад суверен-тенантын (sovereign-tenant) дэмжлэгийг нэмсэн. Энэ нь тенантын сүлжээнд байгаа хэрэглэгчдийг удирдлагын үйлдлүүдэд үндсэн сайт руу буцаах боломжийг олгоно.
Шинэ: Давталттай гишүүнчлэлийн хувьд шинэчилсэн баримт бичиг шалгалт (renewal-credential checks) нэмсэн. Энэ нь хадгалагдаж байгаа төлбөлт, бүртгэл эсвэл vault token байхгүй үед gateway-ийг автоматаар сэргээхийг устгах боломжийг олгоно.
Шинэ: Сайт үүсэх үеийн хүлээгдэж буй (pending site creation) хувилбаруудад HMAC-т баталгаажуулсан loopback publishing нэмсэн. Энэ нь фоны дахь ажлууд хоцорсон хост дээр төлбөлт хийх үйл явцыг илүү найдвартай болгоно.
Шинэ: SSO URL, төлбөлт хийх хуудсанд суурилсан домен (checkout-form base domains), болон автоматаар домен-рекорды үүсгэх зэрэг хөгжүүлэгчдийн өргөтгөлийн цэгүүдийг нэмсэн.
Засвар: Домен тохиргоо хийсэн домен дээр SSO, аноним брокер (anonymous broker) дамжуулалт, гарах үйлдэл (logout), болон plugin-ийн хоорондох хамаарлын зөрүүтэй зэрэгт илүү найдвартай болсон.
Засвар: Одоо хүлээгдэж буй сайт үүсэлт нь хуучин publish flag-аас сэргээгдэх бөгөөд хэрэглэгчийг сайт үүсгэх дэлгэц дээр барихаас зайлсхийв.
Засвар: Хүмүүст хамгийн их ашиглагдаж буй төлбөлт хийх хуудсанд (checkout-form base domains) домен рекорды үүсгэхгүй бөгөөд интеграц идэвхгүй байх үед хэрэглэгдээгүй хост-провайдер фоны дахь ажлуудыг алгасна.
Засвар: Төлбөлт хийх, төлбөлт оруулах хаяг, нууц үг сэргээх, цахим шуудан баталгаажуулалт, загварын солих, туршилтын явц (tours), хэрэглэгчийн самбарын гадна тохиолдлууд нь ердийн хэрэглэгчийн үйлдлүүдийг саатуулахгүй болсон.
Засвар: Хүлээн авсан и-мэйлүүдийг одоо хүлээн авагчдад нууц байлгаж, и-мэйлийн жагсаалт эсвэл хаягийн дамжуулалт алдагдсан үед SMTP/plugin-ийн ноцтой алдаа гарахгүй болсон.
Засвар: Гишүүнчлэлийн шинэчлэлт, дуусах хугацааны харууллт, төлбөрийг цуглуулах зэрэг тохиолдлууд нь шууд дуусах, систем уналт, эсвэл шаардлагатай төлбөрөө оруулаагүй байх асуудлыг зайлуулж байна.
Сэргээсэн: WordPress 7.0 хүртэл тестийн хамаарал байгаа бөгөөд production Vue assets-ийг npm эх сурвалжаас дахин бүтээдэг, Cypress end-to-end хамрах хүрээнд төлбөлт хийх, тохируулах, SSO, gateway-ийн үйл явцууд илүү их шалгагдаж байна.

## 2.12.0 хувилбар — 2026-05-15-д гаргасан {#version-2120--released-on-2026-05-15}

- Шинэ: Hostinger (hPanel)-ийг домен хамааралтай интеграц шийдэлттэй дэмждэг хост провайдер болгон нэмсэн.
- Шинэ: Site Exporter одоо сүлжээний бүх сайтуудыг хурдан сэргээхийн тулд сүлжээний импорт банд (network import bundles) -ийг зохицуулж байна.
- Засвар: BCC тархалт и메илүүд одоо хүлээн авагчдын хаягийг ил тодлахгүй байхын тулд мэдэгдэхгүй хүлээн авагч (undisclosed-recipients) хэшилтийг ашиглаж байна.
- Засвар: Огнооны утга биштай хамт хадгалсан үед гишүүнч холбоотой хугацаа алдагдахгүй болно.
- Засвар: Stripe гишүүнч холбоотой шинэчлэлтүүд одоо хуучирсан deleteDiscount API-г дуудахгүйгээр хөнгөлөлтөө зөв хасах болно.
- Засвар: Домен хамааралтай сайтуудын SSO удирдамж нь хязгаарлагдан, хэт их дахин чиглүүлэх (infinite redirect loops) үйлдэлээс сэргийлж байна.
- Засвар: Setup wizard-ийн зураг сонголт одоо доорх өгөгдлийн загварыг зөв шинэчилнэ.
- Засвар: Site Exporter CLI нь анхны сүлжээний сайтын сонголтыг зөв хадгална.
- Сайжруулсан: Плагины хэмжээг багасгах зорилгоор wp-cli-ийг плагины багцаас гаргасан.

- Шинэ: Network export нь администратор нарт Site Export admin хуудаснаас бүх дэдсайтуудыг нэг архив болгон гаргах боломжийг олгоно.

- Шинэ: Checkout form editor нь шаардлагатай талбарыг тохируул지гүйгээр бүтээгдэхүүн нэмэх үед анхааруулга өгнө.
- Шинэ: Import/Export settings таб нь өөрийн хүрч буй хүрээг тодорхой тайлж, Site Export tool руу шууд холбоотой болно.
