# Сайт Үүсгэх Модтой холбоотой мэдэгдэл

**Site Builder mode нь Superdav AI Agent v1.12.0-т хасагдсан.** Хэрэв та Site Builder mode ашиглаж байсан бол, темы бүтээх болон сайт тохируулах зорилгоор **Setup Assistant agent** руу шилжих хэрэгтэй.

## Юу болсон бэ? {#what-happened}

### Site Builder Mode (Уламжлалт) {#site-builder-mode-legacy}

Site Builder mode нь дараатай интерфейс байсан бөгөөд:

- Тэмплот ашиглан сайт үүсгэх
- Үндсэн тохиргоо хийх
- Темийг сонгох
- Эхний контент тохируулах

### Юу нь орлуулж байна вэ? {#what-replaced-it}

**Setup Assistant agent** одоо дараах зүйлсийг Site Builder mode-ийн бүх үйл ажиллагааг хариуцдаг:

- Илүү уян хатан, агент заасан тохируулга
- Темийг илүү сайн өөрчлөх боломжууд
- Theme Builder-тэй холбоотой сургалтад хамрагдах
- Ирэх сессионд зориулсан `site_brief` дурын мэдээллийг хадгалах

## Хэрэв Та Site Builder Mode Ашиглаж Байсан бол {#if-you-were-using-site-builder-mode}

### Таны Сайтууд Аюулгүй байна {#your-sites-are-safe}

- Site Builder mode ашиглан үүсгэсэн одоо байгаа сайтнууд ажилласаар байх үлдэнэ
- Өгөгдөл алдагдах эсвэл сайтын ажиллагаа зогсохгүй
- Та сайтуудыг ердийн аргаар удирдахыг үргэлжлүүлж болно

### Setup Assistant Agent руу шилжих {#migrate-to-setup-assistant-agent}

Шинэ сайт тохируулах эсвэл темы өөрчлөхөд Setup Assistant agent-ийг ашиглана уу:

```
"Help me set up a new site"
```

эсвэл

```
"Start the Theme Builder onboarding"
```

Setup Assistant agent нь илүү уян хатан байдалтай хамт ижил үйлчилгээ үзүүлнэ.

## Харьцуулга: Site Builder vs. Setup Assistant {#comparison-site-builder-vs-setup-assistant}

| Онцлог | Site Builder (Хасагдсан) | Setup Assistant (Шинэ) |
|---|---|---|
| Тохируулах арга | Wizard form (Дараалалтай маягт) | Agent conversation (Агенттэй яриа) |
| Темийг сонгох | Урьдчилан тодорхойлогдсон тэмплотууд | Өөрийн бүтээсэн үүсэлт |
| Өөрчлөх боломж | Хязгаарлагдмал сонголтууд | Бүтэн дизайн систем |
| Site brief | Хадгалагдахгүй | Үргэлжилдэг санах ой (Persistent memory) |
| Ирэх сессион | Дахин тохируулах шаардлагатай | Хадгалсан site_brief-ийг ашиглана |
| Уян хатан байдал | Тодорхой ажлын урсгал | Хүчтэй хариуцтай яриа |

## Setup Assistant Agent руу шилжих {#migrating-to-setup-assistant-agent}

### Шинэ Сайтуудад {#for-new-sites}

(Энэ хэсэг нь анхны текстэнд дутуу байгаа тул, ерөнхийн тайлбарыг хийв.)

Сайт бариг (Site Builder) моды ашиглахгүй бол:

1. Асуулт: "Шинэ сайт сургуулахыг туслаач" гэж хүснэ.
2. Сургуулах туслах агент танд дараах зүйлсийг зааж өгнө:
   - Сайтын зорилго ба зорилго
   - Зорилтот үзэгчид
   - Брэндийн онцлог
   - Theme (Сайтны загвар) үүсгэх
   - Эхний тохиргоо

### Одоо байгаа сайтуудад {#for-existing-sites}

Хэрэв та Site Builder mode ашиглан хийсэн одоогийн сайттай бол:

1. Үүнийг өөрчлөгдөөгүй байдлаар үргэлж ашиглаж болно
2. Theme-ийг шинэчлэхийг хүсвэл: "Сайтыг дахин загварч" гэж хүснэ
3. Сургуулах туслах агент танд шинэ theme (загвар) үүсгэхэд тусална
4. Таны сайтын өгөгдөл өөрчлөгдөхгүй байх болно

### Theme-ийг өөрчлөхөд {#for-theme-changes}

Site Builder mode-ын theme сонгох аргаар:

1. Асуулт: "Миний theme-ийг өөрчил" гэж хүснэ
2. Сургуулах туслах агент дараах үйлдлүүдийг хийнэ:
   - Таны дизайн хэрхэн байхыг асууна
   - Өөрт тохирсон theme үүсгэнэ
   - Тухайн site дээр түүнийг идэвхжүүлнэ

## Гол ялгаанууд {#key-differences}

### Site Builder Mode {#site-builder-mode}

```
1. Template сонгох
2. Theme сонгох
3. Үндсэн тохиргоо хийх
4. Дууссан
```

### Setup Assistant Agent {#setup-assistant-agent}

```
1. Сайтын зорилгыг тайлбарлах
2. Зорилтот үзэгчдийг тодорхойлох
3. Дизайны сонголтыг хийх
4. Агент өөрт тохирсон theme үүсгэнэ
5. Агент theme-ийг идэвхжүүлнэ
6. Ирээдүйн сессионд ашиглахаар site_brief хадгална
```

## Setup Assistant Agent-ийн давуу талууд {#benefits-of-setup-assistant-agent}

### Хурдан тохирголт {#more-flexible}

- Таны сайтыг ердийн хэлээр тайлбарлах
- Өөрт тохирсон зөвлөмж авах
- Таны онцгой хэрэгцээнд тохируулах

### Сайтыг илүү сайн тохируулах {#better-customization}

- Theme үүсгэх
- Дизайны систем (Design System) шийдвэр гаргах
- Үргэлжилдэг дизайн токенууд (Persistent design tokens)

### Санаачилсан дурсамж {#persistent-memory}

- Таны site_brief хадгалагдана
- Ирээдүйн агентууд таны сайтыг ойлгоно
- Тохиргооны мэдээллийг давтах шаардахгүй

### Нэгдсэн ажлын урсгал (Integrated Workflow) {#integrated-workflow}

- Theme Builder сургалт
- Дизайны систем эстетик чадвар
- Харагдах хяналтын боломж (Ability Visibility controls)
- Бүх зүйлээ гоо сайхан хослуулна

## Асуудал шийдэх {#troubleshooting}

### Site Builder mode олохгүй байна {#i-cant-find-site-builder-mode}

Site Builder mode ашиглалтыг дуусгасан. Одоо Setup Assistant agent-ийг ашиглаарай:

"Би шинэ сайт ашиглахыг туслаарай"

### Site Builder-ээс сайт үүсгэхийг хүсэж байна {#i-want-to-recreate-a-site-from-site-builder}

Та Setup Assistant агент-ийг ашиглан түүнийг дахин үүсгэх боломжтой:

1. Асуулт: "Би шинэ сайт ашиглахыг туслаарай"
2. Өмнөх сайтын зорилго, дизайн-ыг тайлбарлаарай
3. Агент нь төстэй тему (theme) үүсгэнэ
4. Таны site_brief ирээдүйн дурсамжтай хадгалагдана

### Миний одоо байгаа Site Builder сайт ажиллахгүй байна {#my-existing-site-builder-site-isnt-working}

Site Builder модоор бүтээсэн одоогийн сайтууд ажилласаар байх үлдэнэ. Хэрэв та асуудалтай тулгарч байвал:

1. Таны тему (theme) одоо ч идэвхтэй байгаа эсэхийг шалгана уу
2. Таны plugin-үүд идэвхтэй байгаа эсэхийг баталгаажуулна уу
3. WordPress-ийн алдаа логуудыг шалгана уу
4. Асуудал үргэлжилбэл дэмжлэгт хандана уу

### Миний хуучин Site Builder загваруудыг ашиглах боломжтой юу? {#can-i-still-use-my-old-site-builder-templates}

Site Builder загварууд одоо ажиллахгүй байна. Гэсэн хэдий ч:

- Таны одоо байгаа сайтууд ажилласаар байх үлдэнэ
- Setup Assistant агент ашиглан төстэй сайт үүсгэх боломжтой
- Setup Assistant агент нь илүү олон тохиргоо (customization options) өгдөг

## Дараагийн алхамууд {#next-steps}

1. **Шинэ сайтуудад**: Setup Assistant агент-ийг ашиглана уу
2. **Одоо байгаа сайтуудад**: Тэдгээрийг одоогийн байдлаар үргэлж ашиглана уу
3. **Тема өөрчлөхөд**: Setup Assistant агент-ээс тусламж хүсье
4. **Дизайн тохиргоог сайжруулахад**: Design System Aesthetics skill-ийг ашиглана уу

- **Site Specification Skill**: Таны сайтын зорилго, үзэгчдийг тодорхойлох
