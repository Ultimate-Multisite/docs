# Танайдны Клиентейн Вэбсайтнууд (v2)

Танай сүлжээнд бүртгэл хийсний дараа танай хэрэглэгчид өөрийн dashboard-т хандах боломжтой болно.

**Account** хуудасны гадна, энд тэд төлбөр, гишүүнчлэл, домен, төлөвлөгөөний хязгаар зэрэг чухал мэдээллийг харж чадна. Тэд **My Sites** хуудсанд ч хандах боломжтой бөгөөд энд өөрийн бүх вэбсайтуудыг харах, удирдах боломжтой. Зүгээр л dashboard дээрх My Sites-ийг дарна.

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

Тэд өөрийн бүх вэбсайтуудыг харах бөгөөд тус бүрийн доор **Manage** товчлуур байрлана. Хэрэв тэд үүнийг дарвал, тэр тухайн вэбсайтын **dashboard руу** шилжин орно.

Мөн хэд хэдэн вэбсайттай байх боломжтой төлөвлөгөөнд аль нь **үндсэн (primary)** вэбсайт болохыг тэд харах боломжтой.

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

Энэ бол сайт өөрчлөх интерфейсийн бүрэн харагдах дэлгэц юм:

![Site edit full interface](/img/admin/site-edit-full.png)

Хэрэв тэд үндсэн вэбсайтаа өөрчлөхийг хүсвэл **Account** менюг дарж Account хуудас руу очих хэрэгтэй.

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

Тэд мөн хэрэглэгчийн дэлгэрэнгүй мэдээллийг харах, өөрчлөх боломжтой:

![Customer edit interface](/img/admin/customer-edit.png)

Дараа нь тэд **Change Default Site**-ыг дарна.

![Change Default Site button on account page](/img/account-page/change-default-site.png)

Нэгэн цонх гарч ирнэ. Доорх dropdown list-ийг сонгоод аль вэбсайтыг үндсэн вэбсайт болгохыг сонгоно.

![Change Default Site dropdown selection](/img/account-page/change-default-site-dropdown.png)

Дараа нь зүгээр л **Change Default Site**-ыг дарна, ба дууссан!

![Анхны сайт өөрчлөлтийг баталгаажуулах](/img/account-page/change-default-site-confirm.png)
