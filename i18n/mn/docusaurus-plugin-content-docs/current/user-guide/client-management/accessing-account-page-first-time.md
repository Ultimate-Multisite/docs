# Эхний удаа хаягт (account) өгөгдлийнхаа хуудас руу хэрхэн нэвтрэх вэ?

Ultimate Multisite-д таны хүргэсэн и-мэйл дотор байгаа холбоосыг ашиглан өөрийн бүртгэлт (account) өгөгдлийнхаа хуудас руу ороод үзээрэй:

<!-- Screenshot unavailable: Account page email with access link -->

Та худалдан авсан плагиныхоос ирсэн и-мэйл дээрх хаягт болон и-мэйлд таны өгсөн нууц үгийг ашиглана уу.

<!-- Screenshot unavailable: Login page with email and password fields -->

Энэ бол таны бүртгэлтэд анх удаа нэвтрэх тул, танд нууц үгээ өөрчлөх шаардлага гарна. Үүнийг өөрчилсөн дараа _Save_ (Х somehow) товчийг дарна уу.

<!-- Screenshot unavailable: Change password form with Save button -->

Таны бүртгэлтийн хуудаснаас та төлбөрилт, шинэчлэх, цуцлах зэрэг бүх өгөгдлийнхаа талаарх мэдээллийг удирдах боломжтой.
