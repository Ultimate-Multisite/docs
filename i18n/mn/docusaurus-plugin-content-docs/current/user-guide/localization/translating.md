# Ultimate Multisite-ийн орчуулга {#translating-ultimate-multisite}

Ultimate Multisite нь англиар хөгжсөн боловч, энэ плагин бүрэн орчуулалт хийгдэх боломжтой бөгөөд бид орчуулгын хувь нэмэр оруулахыг идэвхтэй урьдаг.

## Үндсэн Плагины Орчуулга {#core-plugin-translations}

Үндсэн Ultimate Multisite плагин нь албан ёсны **WordPress.org орчуулгын платформ**-аар орчуулагддаг:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

Энэ бол WordPress-ийн нийгмийн стандарт орчуулгын систем юм. Орчуулга нь хангалттай зөвшөөрөл авсны дараа, WordPress плагины шинэчлэлтүүд дамжуулан Ultimate Multisite ашигладаг бүх хэрэглэгчдэд автоматаар тархана.

**Бид таны туслалцад хэрэгтэй!** Хэрэв та англиас бусад хэлээр ярьдаг бол орчуулга оруулахад оролцохыг бодож үзээрэй. Бага хэсгийг ч орчуулах нь тусалдаг. Эхлэхийн тулд:

1. [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/) руу очих
2. Таны хэлээ сонгох
3. "Contribute Translation" (Орчуулга оруулах) гэсэн товчийг дарж үзэх
4. Та WordPress.org-ийн үнэгүй аккаунт шаардлагатай

Хэрэв та өөрийн хэлээр орчуулгын редактор болохыг хүсвэл (орчуулгыг зөвшөөрөх чадвартай), [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions)-аар холбогдож болно.

## Дополнийн Плагины Орчуулга (Addon Translations) {#addon-translations}

Дополнийн плагинууд нь дараах газраар тусдаа орчуулагддаг:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Одоогийн байдлаар энэ дохио (addon) орчуулга нь эхлэлийн цэг болгон AI-аар үүсгэгдсэн байна. Энэ нь дэмждэг бүх хэлд хамаарах боловч, заримдаа хүнд байдалтай илэрхийлэл эсвэл алдаатай байж болно. Бид хүний орчуулагчдыг үзэж, тэдгээрийг илүү байгалийн бөгөөд нарийн хийхэд санал өгөхөөр урьдаг.

## Орон нутгийн Орчуулгууд (Local Translations) {#local-translations}

Хэрэв та өөрийн төсөл дээр локал (local) орчуулга хийхийг хүсвэл, upstream-д хувь нэмрэхээс илүү хоёр сонголт байна:

### Poedit ашиглах {#using-poedit}

Та `.pot` файлыг орчуулахын тулд **[Poedit](https://poedit.net)** дээрх десктоп аппликейшнийг ашиглаж болно:

1. [poedit.net](https://poedit.net) -ээс Poedit-ийг татаж авна.
2. `wp-content/plugins/ultimate-multisite/lang/` доторх `.pot` файлыг нээнэ.
3. Poedit интерфейсийн доторх текстүүдийг орчуулна.
4. Хадгалах -- Poedit нь `.po` болон `.mo` файлыг үүсгэнэ.
5. Үр дүнг `lang` фолдерт байрлуулна.

### Loco Translate ашиглах {#using-loco-translate}

**[Loco Translate](https://wordpress.org/plugins/loco-translate/)** WordPress плагины нь десктоп хэрэгсэлгүйгээр администратор (admin panel) дотор шууд орчуулга хийх боломжийг олгодог. Энэ нь орчуулгын файлыг үүсгэх, компиляци хийх ажлыг таны төлөө гүйцэтгэнэ.
