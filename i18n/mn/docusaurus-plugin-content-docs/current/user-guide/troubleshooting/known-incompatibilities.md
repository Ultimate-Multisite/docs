# Бусад плагинуудтай нийцэхгүй байх боломжит асуудлууд {#known-incompatibilities-with-other-plugins}

Ultimate Multisite-ийг суулгаад идэвхжүүлсний дараа, бусад плагинуудтай шийдэгдээгүй нийцэлтэй байдлын улмаас асуудал үүсэх магадлалтай.

Бид шинээр гаргах бүр хувилбарт энэ жагсаалтыг багасгахын тулд идэвхтэй ажиллаж байгаа ч, хэрэв плагинууд энд дурдсан бол таны Ultimate Multisite сүлжээ бүрэн ажиллахын тулд тэдгээрийг идэвхгүй болгох шаардлагатай байж болно.

Энэ жагсаалт болон түүний дараа хийх ёстой зүйлсийг доор харуулав:

**Зарим WPMU DEV плагинууд** Зарим плагинууд нь одоо WPMU DEV-ийн үүрэггүй болсон ч Ultimate Multisite-ын бүртгэлийн үе шат (хэрэв хэрэглэгч данс үүсгэх боломжгүй байвал) зогсоход хүргэдэг гэдгийг мэдэж байна. Эдгээр асуудлыг үүсгэдэг плагинуудын жагсаалт:

  * Signup Language (Бүртгэл хийх хэл): _Ultimate Multisite-ын Ultimate Multisite: Language Switcher гэсэн нэмэлт функцтэй байдаг._

  * Set Blog Description on Blog Creation (Блог үүсгэх үед блогийн тайлбарыг тохируулах): _Энэ нь Ultimate Multisite-д суурилуулсан тул нэмэлт плагинууд шаардлагагүй._

  * New Blog Templates (Шинэ блогийн загварууд): _Энэ нь Ultimate Multisite-д суурилуулсан тул нэмэлт плагинууд шаардлагагүй._

  * Set Password on Multisite Blog Creation (Сүлжээний блог үүсэх үед нууц үгийг тохируулах): _Энэ нь Ultimate Multisite-д суурилуулсан тул нэмэлт плагинууд шаардлагагүй._

  * Signup TOS (Үйлчилгээний нөхцөл ба дүрэмээр бүртгэх): _Энэ нь Ultimate Multisite-д суурилуулсан тул нэмэлт плагинууд шаардлагагүй._

  * Anti-Splog: Хэдийгээр илүү хүчтэй биш ч энэ нь _Ultimate Multisite-д суурилуулсан_.

**Тайлбар**: Бусад бүх WPMU Dev плагинуудыг Ultimate Multisite-тай хамт ердийн ашиглаж болно. Жишээлбэл, _Smush_, _Forminator_, _Defender_ гэх мэт.
