# Туслах авах {#how-to-get-help}

Та таны хүчинтэй License key байна, мөн энэ нь Ultimate Multisite-ийн хэрэгжүүлэлтийн үеэр та тулгарч болох асуудлуудыг шийдвэрлэхэд бидний багийн туслалцаатай холбогдох эрхийг өгдөг.

Энэ зааврын дагуу бид License holder-уудад зориулсан дэмжлэгт байгаа суваг болон туслах баг руу хандахад та ямар мэдээллийг бэлдэх шаардлагатай талаар тайлбарласан.

## Туслах сайт (Help Site) {#help-site}

Та доорх холбоос дээр олддог бидний туслах сайтан дээр байгаа янз бүрийн нийтлэл, зааврын гарын авлагыг үзэж болно.

[)

Бид хэрэглэгчдийг Ultimate Multisite-ийн үндсэн функц болон үйл ажиллагаатай танилцахыг уриалдаг. Туслах сайтан дээр "Эхлүүлэх" (Getting Started) зааврууд байгаа бөгөөд add-on-уудыг хэрхэн суулгах, төлбөрийн хаалгыг хэрхэн нэгтгэх зэрэг ерөнхий асуултуудын хариу ч тэнд байдаг.

## Дэмжлэгийн багтай холбогдох (Contact Support) {#contact-support}

Хэрэв та тус тус ямар нэгэн асуудалтай тулгарч, туслах сайтан дээр хариултыг олоогүй бол бидний дэмжлэгийн баг руу хоёр аргаар холбогдох боломжтой: эсвэл бидний вэбсайтын доод талын баруун талд байрлах ярианы булчин (speech bubble icon)-ыг дарж, эсвэл шууд support@ultimatemultisite.com хаялгаар и-мэйл илгээх замаар.

Техникийн асуудалтай холбоотой дэмжлэгт хандахдаа та асуудлын тодорхой тайлбар, скриншот, алдааны мессеж эсвэл бидний багт асуудлыг хялбархан ойлгож үнэлэхэд туслах ямар ч мэдээллийг хамт өгөх шаардлагатай. Мөн таны license key-г заавал өгөх ёстой. Зөвхөн идэвхтэй License holder-ууд л дэмжлэг авдаг.

## Нийгмийн сайт (Community Site) {#community-site}

Бидний [community site](https://community.ultimatemultisite.com/) нь Ultimate Multisite-ийн хэрэглэгчдийн хоорондын харилцахад зориулсан талбар юм. Энэ нь гишүүд яриа өрнүүлж, тайлбарт санал өгөх, хамтран ажиллах, эсвэл олон нийтийн хүчээр мэдээлэл цуглуулах боломжийг олгодог мэдээллийн систем юм. Нууц талбай/хагас хуваарилагдсан хуудсуудаа нээхийн тулд та Ultimate Multisite-ийн license key-г өгөх шаардлагатай болно.
