# Sunrise Файл Угтгах Алдаа {#error-installing-the-sunrise-file}

sunrise.php файл нь WordPress өөрийгөө ажиллуулах үед хайж буй тусгай файл юм. WordPress-ийн sunrise.php файлыг олж чадахын тулд тэр файл **wp-content** фолдер дотор байх шаардлагатай.

Та Ultimate Multisite-ийг идэвхжүүлээд скриншотод байгаа зэрэг setup wizard-ыг ашиглахад, Ultimate Multisite бидний sunrise.php файлыг wp-content фолдер рүү хуулгахыг оролдоно.

<!-- Скриншот харагдаагүй: Sunrise.php суулгах алхамтай setup wizard-ийн хуудас -->

Ихэнх тохиолдолд бид файлыг амжилттай хуулж чадаж, бүх зүйл ажилладаг. Гэсэн хэдий ч, ямар нэгэн зүйлийг зөв тохируулсангүй (жишээлбэл, фолдерin эрх) үед Ultimate Multisite файл хуулж чадахгүй байх боломжтой тусгай нөхцөл байдал үүсэх магадлалтай.

Хэрэв Ultimo-оос өгөгдсөн алдааны мессежийг уншаагүй бол, энэ нь яг ингэж болсон гэдгийг харах болно: **Sunrise copy failed** (Sunrise хуулгах алдаа).

<!-- Скриншот харагдаагүй: Sunrise хуулгах алдааг харуулсан мессеж -->

Үүнийг засахын тулд, та зүгээр л wp-ultimo plugin фолдер доторх sunrise.php файлыг хуулж аваад, түүнийг өөрийн wp-content фолдер рүү хийж оруулаарай. Үүний дараа wizard-ийн хуудсыг дахин ачаална уу, шалгалт амжилтад хүрэх ёстой.

<!-- Скриншот харагдаагүй: wp-ultimo plugin фолдер дотор sunrise.php файлыг харуулсан файл менежер --> Ямар ч тохиолдолд, ирээдүйд асуудал үүсэхээс зайлсхийхийн тулд таны фолдерin эрхүүдийг ерөнхийдөө шалгах нь зүйтэй (Ultimate Multisite-тэй бус, бусад plugin болон theme-тэй ч).

WordPress-ийн хэсэг болох **Health Check tool** (үндсэн сайт дээр **admin panel > Tools > Health Check**-ээр хандаж болно) таны фолдерin эрхүүд WordPress-д асуудал үүсгэх боломжит утгатай байгаа эсэхийг мэдэх чадвартай.

<!-- Үндэслэх зураг байхгүй: WordPress Health Check хэрэгсэл нь хавцын эрхий төлөвийг харуулж байна -->
