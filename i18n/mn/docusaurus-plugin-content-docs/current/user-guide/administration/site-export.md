# Сайт экспортуулга {#site-export}

**Site Export** администратор хуудас нь сүлжээний администратор нарт нэг сайт, эсвэл бүх сүлжээг шилжүүлэх, дахин ашиглах (backup), эсвэл өөрөө дамжуулах ажлуудад зориулсан татаж авах боломжтой архив болгон багтаахад тусалдаг.

## Нэг сайтыг экспортоо {#exporting-one-site}

**Ultimate Multisite > Site Export** руу орж, **Generate new Site Export**-ийг сонгоно. Экспортоо хүсэж буй дэдсайтыг сонгосны дараа архивд оруулах файлууд (uploads), plugin болон theme-үүдийг багтаах эсэхийг сонгоно.

Экспорт дууссаны дараа **Existing Exports** жагсаалтаас ZIP файлыг татаж авна уу. Одоогийн Экспорт ZIP файл нь өөрөө ачаалагдах `index.php` болон `readme.txt`-ийг багтаадаг тул архив дээр шинэ хост руу ачаалж, тусгай импортер plugin-ийг урьдчилан суулгахгүйгээр шууд ажиллуулах боломжтой.

## Бүх сүлжээг экспортоо {#exporting-the-whole-network}

Сүлжээний бүх дэдсайтуудыг нэг архив дотор багтаасан файл хэрэгтэй үед Site Export хуудасны **Network Export**-ийг ашиглана уу. Энэ нь хост шилжих, галзуу дахин сэргээх туршилт хийх эсвэл бүх дэдсайт нэг дор ажиллах шаардлагатай staging (шаг тутам) үед хэрэгтэй байдаг.

Сүлжээний экспортоо нэг сайт экспортоос их байж болзошгүй тул, ачаалалт багатай цаgningдаа ажиллуулж, зорилтот хадны зай нь uploads, plugins, themes болон үүссэн ZIP файлүүдийг багтаахад хангалттай байгаа эсэхийг шалгана уу.

### Network Import Bundles (Сүлжээний импорт бүлэгүүд) {#network-import-bundles}

Ultimate Multisite 2.12.0-аас хойш Site Exporter нь **network import bundles** -ийг үүсгэх боломжтой болсон. Энэ нь сүлжээний өргөн хүрээний сайт дахин ашиглах ажлыг хялбаршуулах зориулалттай тусгай архив юм. Network import bundle дотор олон сайтыг шинэ сүлжээнд дахин суулгах шаардлагатай бүх зайлшгүй файлууд болон метаданс багтдаг.

#### Network Import Bundle үүсгэх {#generating-a-network-import-bundle}

1. **Ultimate Multisite > Site Export** рууц рууцад орно уу.
2. **Generate new Network Export** (Шинэ сүлжээний экспорт үүсгэх) дээр дарна уу.
3. Экспортын төрөл болгож **Network Import Bundle** (Сүлжээний импорт банд) -ийг сонгоно уу.
4. Захиалга, плагинууд болон темиудыг оруулах эсэхийг сонгоно уу.
5. **Generate** (Үүсгэх) дээр дарна уу.
6. **Existing Exports** (Одоо байгаа экспортууд) жагсаалтаас банд ZIP файлыг татаж авна уу.

#### Network Import Bundle-аас сэргээх {#restoring-from-a-network-import-bundle}

Сүлжээний импорт банд ашиглан сайтнуудыг сэргээхийн тулд:

1. Ultimate Multisite-ийг таны зорилтот хост дээр суулгана уу.
2. multisite setup wizard (сүлжээний тохиргооны үйлдэл) -ийг дуусгана уу.
3. Шинэ сүлжээнд **Ultimate Multisite > Site Export** руу орно уу.
4. **Import Network Bundle** (Сүлжээний банды импортлох) дээр дарна уу.
5. Сүлжээний импорт банд ZIP файлыг ачаална уу.
6. Дэлгэц дээрх импортын заавруудыг дагаж үргэлжлүүлнэ уу.
7. Импортын үйл явцад бүх сайтнууд, тэдгээрийн агуулга болон тохиргоо сэргэгдэнэ.

Network import bundles дараах зүйлсийг хадгалдаг:
- Сайтны агуулга (post, page, custom post types)
- Сайтын тохиргоо ба сонголтууд
- Хэрэглэгчийн үүрэг болон эрхүүд
- Плагинууд ба темиуд (банд дотор оруулагдсан бол)
- Медиа ачааллууд (банд дотор оруулагдсан бол)
- Өгөгдлийн хувьд өөрчлөлт хийсэн database tables болон мэдээлэл

## Өөрийгөө асаах ZIP файлыг суулгах {#installing-a-self-booting-export-zip}

Шинэ хост дээр өөрийгөө асаах (self-booting) ZIP файлыг сэргээхийн тулд:

1. Экспорт хийсэн ZIP файлын агуулгыг зорилтот web root руу ачаална уу.
2. Ачаалсан `index.php`-ийг браузероор нээнэ уу.
3. банд экспортын пакет доторх дэлгэц дээрх суулгах заавруудыг дагаж үргэлжлүүлнэ уу.
4. түр зуурын файлуудыг арилгахын өмнө банд `readme.txt`-ийг экспортын тусгай тэмдэглэгээтэй шалгана уу.

Addon-д тохирох суулгалт болон импортын дэлгэрэнгүй мэдээллийг [Site Exporter addon documentation](/addons/site-exporter) -д үзнэ үү.

Ultimate Multisite 2.9.0-т нэмэгдсэн single-site tool (нэг сайттай хэрэгсэл)-ийн талаар [Export & Import](/user-guide/administration/export-import)-ыг үзнэ үү.
