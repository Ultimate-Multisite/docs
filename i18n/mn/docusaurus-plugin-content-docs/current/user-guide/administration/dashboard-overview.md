# Дашбордын өгляд

Ultimate Multisite-д Ultimate Multisite меню доор тусгай **Дашборд** байдаг бөгөөд мөн Network Dashboard руу виджетүүдийг нэмдэг. Энэ нийтлэл нь эдгээр хоёр газраас юу харахыг тайлбарладаг.

## Network Dashboard

Ultimate Multisite-ийн хэрэгжүүлсэн тусгай виджетүүдийг **My Sites → Network Admin → Dashboard**-д байгаа network-level dashboard руу оруулдаг.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

Энэхүү виджет бүр юу өгдөж байгааг харуулав:

**First Steps (Анхны алхам):** Шинэ суулгалт хийсний дараа гарч ирдэг бөгөөд төлбөрийн аргыг тохируулах, туршилтын аккаунт үүсгэх зэрэг тохиргоо хийх ажлуудыг дуусгахыг сануулдаг.

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Summary (Товч дүгнэлт):** Өнөөдрийн бүртгэлт болон орлогоо илэрхийлдэг. MRR (Monthly Recurring Revenue - Саарлаж ирж буй орлого) нь орлого оруулах гишүүнтэй хэрэглэгчдийн нийт төлөвлөгөөний орлогыг заадаг.

![Summary widget](/img/admin/network-dashboard-summary.png)

**Activity Stream (Үйл ажиллагааны урсгал):** Ultimate Multisite дэх үйл явдлуудыг (бүртгэлт, цуцлалт, сайт үүсгэх, устгах болон бусад үйлдэл) мэдээллэдэг.

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**Right Now (Одоогийн байдал):** Network дэх хэрэглэгч болон сайтуудын товч дүнг харуулж, шинэ сайт эсвэл хэрэглэгч үүсгэхэд хүргэх хуудсан холбоотой (shortcuts) зургийг харуулдаг.

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**News & Discussions (Мэдээлэл ба Хэлэлцүүлэг):** Ultimate Multisite-ийн хамгийн сүүлийн мэдээ, шинэчлэлт, чухал алдаанууд болон аюулгүй байдлын заавал өөрчлөлтүүдийг авчирдаг.

![News widget](/img/admin/network-dashboard-news.png)

## Ultimate Multisite Dashboard

Network dashboard нь network-ийн нийт мэдээллийг харуулдаг бол, Ultimate Multisite-ийн дашборд (Ultimate Multisite менюгийн хамгийн дээд түвшинд байрлах) тухайн үйлчилгээний талаарх мэдээллийг харуулдаг.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Сарын давтам шаардлагатай орлого (Monthly Recurring Revenue Growth):** Энэ нь 12 сарын календарь хугацааны туршид орлогын сарын дүнгийн дэлгэрэнгүй харагдах хэсэг юм. Шинэ орлого цэнхэр өнгөөр, цуцлалт (cancellations) улаан өнгөөр харагдана.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**Огнооны хүрээ дэмжлэг (Date-range Support):** Та хэрэгтэй мэдээлэлд анхаарлахаар таны өгөгдлийг цагийн хугацааны үе шатанд хамаарах байдлаар шүүх боломжтой. Огнооны хүрээ нь доорх widget-уудад ямар мэдээлэл харагдахыг тохируулдаг.

![Date range and dashboard stats](/img/admin/um-dashboard-date-range.png)

Dashboard-ын widget-үүдэд дараах зүйлс багтдаг:

- **Орлого (Revenue)** — Тухайн хугацааны нийт орлого, MRR болон бүтээгдэхүүн бүрийн дүнгийн задлалтай хамт
- **Шинэ гишүүдийн бүртгэл (New Memberships)** — Сонгосон үе шатанд хэдэн шинэ гишүүнээр бүртгэл хийгдсэн бэ
- **Улс тус бүрийн бүртгэл (Signups by Countries)** — Бүртгэл хийсэн газруудын географик задлал
- **Төлбөрийн маягт бүртгэл (Signups by Forms)** — Хэрхэн олон хэрэглэгч тус тусын төлбөрийн маягтаар бүртгэл хийж байгаа болон администратор нар нэмсэн тоо
- **Хамгийн их зочилсон сайтууд (Most Visited Sites)** — Таны сүлжээнд хамгийн их зочилсон сайтууд (Visit counting-ийг **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**-д идэвхжүүлэх шаардлагатай)
