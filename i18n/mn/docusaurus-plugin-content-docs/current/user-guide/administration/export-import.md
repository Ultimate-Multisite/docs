# Экспорт ба импорт

Ultimate Multisite 2.9.0-д **Tools > Export & Import** (Хэрэгслүүр > Экспорт ба импорт) дотор нэг сайттай холбоотой экспорт ба импортын хэрэгсэл нэмсэн. Та энэ хэрэгслийг WordPress сайтыг ZIP файл болгон багтаах, тэр ZIP файлыг сэргээх эсвэл хослуулалт хийгдэж буй Ultimate Multisite болон ердийн (single-site) WordPress байрлалуудын хооронд сайтуудыг шилжүүлэх шаардлагатай үед ашиглана.

## Шаардлагатай эрхүүд {#required-permissions}

Та экспортолж эсвэл импортлох сайт дээрх WordPress **Tools** (Хэрэгслүүр) меню руу нэвтрэх боломжтой администратор хэлбэрээр логин хийх ёстой. Хэрэв та multisite сүлжээнд байгаа бол сүлжээний түвшний Ultimate Multisite хэрэгслүүдийн ашиглалтаар дэдсайтуудыг экспортолж эсвэл импортлох үед сүлжээний администратор (network administrator) хэлбэрээр логин хийх шаардлагатай.

Экспортын ZIP файлыг баталгаажуулсан татах цэгээр дамжуулан өгдөг тул татаж дуусаха хүртэл администраторын сессийг идэвхтэй байлгаж, үүсгэсэн татах URL-ийг олон нийтэд хуваалцахыг хориглоно.

## Сайтыг ZIP болгон экспортолх {#exporting-a-site-to-a-zip}

1. Та хуулах гэж буй сайт дээрх WordPress администратор руу орж, **Tools > Export & Import** (Хэрэгслүүр > Экспорт ба импорт)-д очина уу.
2. Экспортын хэсгийг нээгээд та багтаахыг хүсэж буй сайтыг сонгоно.
3. Хэрэв боломжтой бол оруулах контент, жишээлбэл, uploads (загварчилсан файл), plugins (плагинууд) ба themes (теэмүүд)-ийг сонгоно.
4. Экспортыг эхлүүлж, үйл явц дуусах хүртэл хүлээнэ уу. Том сайтнууд ZIP файл бэлэн болохын өмнө ардны фононд дуусах шаардлагатай байж болно.
5. Дууссан ZIP файлыг экспортын жагсаалтаас татаж авна уу.

ZIP файлыг аюулгүй газарт хадгална уу. Энэ нь сайтын контент, тохиргоо, медиа файл болон сонгосон код зэрэг зүйлсийг агуулж болно.

## Экспорт юу багтаадаг вэ {#what-the-export-includes}

Экспортын ZIP файл дараах зүйлсийг агуулж болно:

- Сонгосон сайтын өгөгдлийн сан (database) контент болон тохиргоо.
- Хэрэв uploads (загварчилсан файл)-ыг оруулах сонголтыг хийсэн бол, заавал оруулсан медиа файл.
- Хэрэв тухайн сонголтуудыг хийвэл, plugins (плагинууд) ба themes (теэмүүд).
- Экспорт ба импорт хэрэгсэл нь зорилтот байрлалуудад сайтыг дахин үүсгэхэд ашигладаг импортын метаданс (metadata).

Зип файлын яг хэмжээ нь медиа агуулгын хэмжээ, сонгосон plugin болон theme-үүд, мөн сайтны database-ийн хүснэгтүүдийн хэмжээнд хамаардаг.

## ZIP-ээс сайт импортлох {#importing-a-site-from-a-zip}

1. Зохиосон WordPress сайтын **Tools > Export & Import** руу орно.
2. Импортын талбайг нээгээд Export & Import tool-аар үүсгэсэн ZIP файлыг ачаална.
3. Хэрэв сайт шинэ хаягийг ашиглах ёстой бол URL-ийн оронд өөр URL-ийг оруулж болно, эсвэл анхны URL-ийг хадгалахын тулд талбайг хоосон үлдээж болно.
4. Импорт дууссаны дараа ачаалсан ZIP файлыг устгахыг сонгоно.
5. **Begin Import** (Импортыг эхлүүлэх) товчийг дарна.
6. Хүлээгдэж буй импортыг дуусах хүртэл хянах хэрэгтэй. Процессыг дуусгахын өмнө зогсоох шаардлагатай бол **Cancel Import** (Импортыг цуцлах) ашиглана.
7. Ерөнхий трафик эсвэл хэрэглэгчдийн хандалтыг зөвшөөрсөн дараа импортлагдсан сайтыг шалгахыг санал болгоно.

Нэг сайттай WordPress-ийн усталтанд ZIP файл импортлох нь одоо байгаа сайтыг импортлагдсан сайтноор сольдог. Эхлэхийн өмнө зорилтот байрлалыг бүрэн backup хийж, ижил сайттай нэг дор олон удаа импорт эхлүүлэхийг зайлуулах хэрэгтэй.

## Хязгаарлалт ба нийцтэй байдлын тэмдэглэл {#limitations-and-compatibility-notes}

Маш том файлууд эсвэл медиа сангууд нь их хэмжээний ZIP файл үүсгэж болно. Том сайтуудыг экспортер хийхэснээр PHP-ийн ачааллын хязгаар, гүйцэтгэлийн хязгаар, дискийн зай, санах ойн хязгаар болон серверээс гарах хугацааны тохиргоог шалгаарай.
Маш том медиа сангууд нь бага ачаалалтай засвар хийх үед (maintenance window) шилжүүлэх шаардлагатай байж болно.
Зорилтот WordPress-ийн суулгалт нь таарах WordPress, PHP, Ultimate Multisite, plugin болон theme хувилбаруудыг ажиллуулах ёстой.
Нэг сайттай импорт нь зорилтот сайтыг орлуулдаг. Энэ нь нэгтгэх (merge) хэрэгсэл биш юм.
Multisite-to-single-site болон single-site-to-multisite шилжилт зөвхөн зорилтот орчин экспортер хийгдсэн сайтын plugin, theme, URL-ууд болон шаардлагатай Ultimate Multisite компонентүүдийг ажиллуулах чадвартай байвал дэмждэг.
ZIP файлыг нууцлаг байлгана уу. Энэ нь экспортер хийгдсэн сайтын өгөгдөл, ачаалсан файл болон тохиргооны дэлгэрэнгүй мэдээллийг агуулж болно.

Хуучин сүлжээний түвшний экспортер ажлуудыг [Site Export](/user-guide/administration/site-export) -д үзнэ үү.
