# Хэрэглэгчдийг удирдах {#managing-customers}

**Customers** (Хэрэглэгчид) хуудас танай сүлжээнд бүртгэгдсэн гишүүдийн жагсаалтыг харуулдаг. Энэ хуудсан дээр нэр, цахим шуудангийн хаяг, хамгийн сүүлд оролцсон цаг болон **switch to** (шинэ тал руу шилжих) холбоотой линк байрладаг бөгөөд энэ нь таны хэрэглэгчийн дэдсайт руу сайт администратор гэсэн үүрэгээр нэвтрэх боломжийг олгодог.

![Customers list page](/img/admin/customers-list.png)

Хэрэглэгчийн карт доор байгаа **Manage** (Удирдах)-ыг дарж тухайн хэрэглэгчтэй холбоотой дэлгэрэнгүй мэдээллийг үзээрэй.

Хэрэглэгчийн дэлгэрэнгүй хуудсан дээр, **At a Glance** (Нэгтгэсэн харалт)-ын хэсэг нь тухайн хэрэглэгчийн ID, хамгийн сүүлд оролцсон цаг болон энэ хэрэглэгч зарцуулсан нийт мөнгөн дүнг харуулдаг.

**Customer Options** (Хэрэглэгчийн сонголт)-ын хэсэг нь тухайн хэрэглэгчийг **VIP** (тэд илүү хялбар олохын тулд өгөгдсөн тэмдэг) болгох боломжийг олгодог. **Billing Info** (Тооцооны мэдээлэл)-ын таб нь цахим шуудангийн хаяг, улс зэрэг тооцооны мэдээллийг харуулдаг.

**Fields** (Үндсэн талбарууд)-ын хэсэг нь хэрэглэгчийн нэр, цахим шуудан, болон **Email Verification** (Имэлийн баталгаажуулалт)-ыг өөрчлөх боломжийг олгодог.

Customer Options-ын доор танайд дөрвөн нэмэлт хэсэг байрлана:

- **Memberships** (Гишүүнчлэл): Хэрэглэгчээр одоо бүртгэлтэй байгаа хуваарьчилсан үйлчилгээ.
- **Payments** (Төлбөр хийх): Хэрэглэгчийн төлбөрийн түүх.
- **Sites** (Сайтнууд): Энэ хэрэглэгч эзэмшдэг сайтнууд.
- **Events** (Үйл явдлууд): Энэ хэрэглэгчтэй холбоотой үйл явдлууд (имэлийн баталгаажуулалт, тооцооны өөрчлөлт гэх мэт).

Ямар нэгэн зүйлийн хажууд байгаа талбар дээрх стрелкин дүрс (→) дээр дарж тухайн дэлгэрэнгүй хуудас руу шилжих боломжтой. Жишээлбэл, сайтын хажууд байгаа стрелкийг дарвал таны membership association (гишүүнчлэлийн холбоо), site type (сайтын төрөл) ба идэвхтэй байдлыг өөрчилж чадах сайт засварлах хуудас руу хүргэнэ.

Гишүүнчлэл болон төлбөрийн хувьд мөн адил үйлдэл хийх боломжтой — эдгээр нь membership status (гишүүнчлэлийн статусыг), холбоотой хэрэглэгчийг, бүтээгдэхүүнийг, төлбөр хийх хаалтын тохиргоог өөрчлөх боломж олгодог бөгөөд мөн буцаагдал (refund) хийх үйл явцыг гүйцэтгэх боломжийг олгодог.

Хэрэглэгчийн хуудсанд та баруун талд дараах зүйлсийг олж үзнэ:

*Сүүлийн оролцогч ба IP-адрес* — Сүүлд нэвтэрсэн цаг, IP-адрес, улс
*Цагийн тасалбарууд (Timestamps)* — Хэрэглэгчийн бүртгэл үүссэн болон сүүлд өөрчлөгдсөн цаг
*Устгах* — Хэрэглэгчийг эдгээр мэдээллийг хэвээр үлдээхгүйг баталгаажуулах хувилбар

:::warning
Хэрэглэгчийг устгах нь бүрэн боломжтой бөгөөд буцаагдах боломжгүй.
:::
