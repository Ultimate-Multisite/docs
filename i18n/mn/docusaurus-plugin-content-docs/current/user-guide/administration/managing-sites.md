# Сайтууды удирдах {#managing-sites}

Сайт (эсвэл дэдсайт) бол таны WaaS бизнесийн гол хэсэг юм. Ultimate Multisite-д 3 төрлийн сайт байдаг:

- **Хэрэглэгчийн өмчтой** — Тодорхой хэрэглэгчдэд хамаарах сайтууд
- **Сайт загварууд (Site Templates)** — Хэрэглэгчид эхлэх цэг болгон сонгож болох урьдraа заасан сайтнууд
- **Үндсэн сайт (Main Site)** — Таны үндсэн сүлжээний сайт

## Сайтуудыг харах {#viewing-sites}

Сүлжээний бүх дэдсайтуудыг үзэхийн тулд **Ultimate Multisite → Sites** руу орно. Нэг сайт нь төрлөөр тэмдэглэгддэг бөгөөд та All Sites (Бүх сайт), Customer-Owned (Хэрэглэгчийн өмчтой), Templates (Загварууд) эсвэл Pending (Хүлээгдэж буй) -ээр шүүж үзэх боломжтой.

![Sites list page](/img/admin/sites-list.png)

## Шинэ сайт нэмэх {#adding-a-new-site}

Шинэ сайт үүсгэхийн тулд **Add Site** (Сайт нэмэх) товчийг дарна. Та дараах зүйлсийг өгөх шаардлагатай:

- **Site Title** — Шинэ сайтын нэр
- **Site URL** — Сайтын домен/загвар
- **Site Type** — Энэ нь хэрэглэгчийн сайт, загвар эсвэл ердийн сайт юу болохыг тодорхойлох

**Copy Site** (Сайты хуулна) сонголт нь одоо байгаа сайт загвараас шинэ сайт үүсгэх боломжийг олгодог. Үүнийг идэвхжүүлэхэд, эхлэл болгон аль загварыг ашиглахыг сонгож болно. Медиа файлуудыг багтаахын тулд **Copy Media on Duplication** (Хуулбарлах үед медиаг хуулна) -г идэвхтэй байлгахыг анхаарна уу.

## Одоо байгаа сайтыг удирдах {#managing-an-existing-site}

Сайт дээр **Manage** (Удирдах) товчийг дарж **Edit Site** (Сайты өөрчлөх) хуудсыг нээнэ. Энд та дараах зүйлсийг олно:

### Үндсэн мэдээлэл {#basic-information}

Сайтын нэр, төрөл, site ID болон тайлбар. Түүнчлэн хамаарах домен, холбоотой гишүүний бүртгэл, болон сайтыг эзэмшиж буй хэрэглэгчийн дансыг харж болно.

### Сайт тохиргоо (Site Options) {#site-options}

Сайтын чадвар ба хязгаарлалтыг тохируулна:

- **Visit limits** — Сайта орох хамгийн их тоо
- **User account limits** — Хэрэглэгчийн үүрэг бүрийн хязгаарлалт
- **Disk space** — Сайттай холбоотой хадны зай (storage quota)
- **Custom domain** — Энэ сайттай домен хослолыг идэвхжүүлэх
- **Plugin and theme visibility** — Аль plugin болон theme-ийг харагдах, нуух эсвэл урьдчилан идэвхжүүлэхээ хэрхэн зохицуулах

Үндсэн тохиолдолд сайтууд гишүүдийн түвшинд тогтоосон хязгааруудыг дагадаг. Сайтын түвшний хязгаарыг тохируулбал гишүүдийн тохиргоог илүү өндөр байдалтай болгоно (override).

### Холбоотой зүйлс (Associations) {#associations}

Сайтны тохиргоонуудын доор дараах мэдээллийг олох боломжтой:

- **Сайтаар холбогдсон таны домен**
- **Сайт ямар гишүүдийн дор байгаа**
- **Сайтад холбогдсон хэрэглэгчийн данс (Customer account)**

### Баруун талын цэс (Right Sidebar) {#right-sidebar}

Баруун талд та дараах үйлдлүүдийг хийж болно:

- **Сайтыг идэвхжүүлэх/утаах** тоглгоны товч ашиглан
- **Сайтын төрлийг өөрчлөх** эсвэл өмчлөлөө шинээр тохируулах
- **Сайтны зураг/thumbnail (миньти)** тохируулах (сайтын загварт урьдчилсан хэсэгт харагдана)
- **Сайтыг** бүрмөсөн устгах

:::warning
Сайтыг устгах нь буцааж болохгүй. Тухайн сайт болон түүний бүх агуулга бүрмөсөн устгагдана.
:::
