# Gratis AI Agent тохиргоо {#gratis-ai-agent-settings}

Gratis AI Agent дахь **Тохиргоо → Нарийвчилсан** дэлгэц нь backend интеграцуудад зориулсан администраторын түвшний тохиргоог өгдөг. Энэ хуудас нь санал хүсэлт дамжуулах, хайлтын үйлчилгээ үзүүлэгчийн түлхүүрүүд, удирдлагатай Superdav үйлчилгээний тохируулга, Google Calendar удирдлагууд, TextBee SMS тохиргоо, мөн сүлжээ даяарх боломжийн тугуудыг баримтжуулна.

## Тохиргоонд хандах {#accessing-settings}

1. WordPress админ хэсэгт **Gratis AI Agent → Тохиргоо** руу очно уу.
2. **Нарийвчилсан** таб дээр дарна уу.

## Санал хүсэлтийн endpoint тохиргоо {#feedback-endpoint-configuration}

Санал хүсэлтийн endpoint нь хэрэглэгч эрхий хуруу доош товч, автомат сануулах баннер, эсвэл `/report-issue` командыг ашиглан санал хүсэлт илгээх бүрд AI agent-аас POST хүсэлтүүдийг хүлээн авдаг.

| Талбар | Тайлбар |
|---|---|
| **Санал хүсэлтийн Endpoint URL** | JSON биетэй HTTP POST хүсэлтээр санал хүсэлтийн илгээлтүүдийг хүлээн авах URL. |
| **Feedback API Key** | Санал хүсэлтийн хүсэлт бүрийн `Authorization` header дотор илгээгдэх bearer token. Хэрэв таны endpoint баталгаажуулалт шаарддаггүй бол хоосон орхино уу. |

### Хүлээгдэж буй JSON Payload {#expected-json-payload}

Таны санал хүсэлтийн endpoint хамгийн багадаа дараах талбаруудтай JSON биеийг хүлээн авах ёстой:

```json
{
  "message_id": "msg_abc123",
  "conversation_id": "conv_xyz789",
  "feedback_text": "The answer was incorrect about pricing.",
  "triage_category": "factual_error"
}
```

Харилцан ярианы нөхцөлөөс хамааран payload-д нэмэлт талбарууд байж болно.

### `triage_category` утгууд {#triagecategory-values}

AI ангиллын давхарга payload-г дамжуулахаас өмнө `triage_category`-д дараах утгуудын нэгийг онооно:

| Утга | Утга санаа |
|---|---|
| `factual_error` | Туслах буруу бодит мэдээлэл өгсөн. |
| `unhelpful_answer` | Хариулт техникийн хувьд зөв байсан боловч хэрэгцээтэй байгаагүй. |
| `inappropriate_content` | Хариулт хэрэглэгчдэд харуулах ёсгүй агуулга агуулсан. |
| `other` | Санал хүсэлт мэдэгдэж буй ангилалд таараагүй. |

### Баталгаажуулалт {#authentication}

Хэрэв таны endpoint баталгаажуулалт шаарддаг бол **Feedback API Key** талбарыг өөрийн bearer token-оор тохируулна уу. Agent дараахыг илгээнэ:

```
Authorization: Bearer <your-api-key>
```

Хэрэв **Feedback API Key** талбар хоосон бол `Authorization` header илгээгдэхгүй.

### Санал хүсэлт цуглуулалтыг идэвхгүй болгох {#disabling-feedback-collection}

**Санал хүсэлтийн Endpoint URL** болон **Feedback API Key** талбаруудыг хоёуланг нь хоосон орхино уу. Эрхий хуруу доош товч болон санал хүсэлтийн UI хэрэглэгчдэд харагдсан хэвээр байх боловч илгээлтүүд ямар нэгэн гадаад үйлчилгээ рүү дамжуулагдахгүй.

## Brave Search API Key {#brave-search-api-key}

Мөн **Нарийвчилсан** таб дээрх **Brave Search API Key** талбар нь [Интернет хайлт](../configuration/internet-search) боломжийг идэвхжүүлнэ.

| Талбар | Тайлбар |
|---|---|
| **Brave Search API Key** | Brave Search хөгжүүлэгчийн Dashboard-оос авсан таны API түлхүүр. AI туслах дотор интернет хайлтыг идэвхжүүлэхэд шаардлагатай. |

Талбарын шошго нь Brave Search API бүртгэлийн хуудас руу дарах боломжтой холбоос агуулна. Интернет хайлтыг идэвхгүй болгохын тулд хоосон орхино уу.

Энэ боломжийн эцсийн хэрэглэгчийн баримтжуулалтыг [Интернет хайлт](../configuration/internet-search)-аас үзнэ үү.

## Удирдлагатай Superdav үйлчилгээ {#managed-superdav-service}

Superdav AI Agent v1.18.0 нь дэмжигдсэн сайтуудад удирдлагатай Superdav үйлчилгээний endpoints болон автомат холболт бэлтгэлтийг нэмдэг. Таны сайт гараар тохируулсан үйлчилгээний endpoint-ийн оронд hosted үйлчилгээ үзүүлэгчтэй холбогдох ёстой үед эдгээр удирдлагыг ашиглана уу.

| Талбар | Тайлбар |
|---|---|
| **Удирдлагатай Superdav үйлчилгээ** | Дэмжигдсэн сайтуудад hosted Superdav үйлчилгээний холболтыг идэвхжүүлнэ. |
| **Холболт бэлтгэх** | Автомат endpoint болон итгэмжлэлийн бэлтгэлтийг эхлүүлнэ. Сайт удирдлагатай үйлчилгээ үзүүлэгчийг ашиглах ёстойг баталгаажуулсны дараа үүнийг ашиглана уу. |
| **Үйлчилгээний Endpoint / Холболтын төлөв** | Бэлтгэлтийн дараах одоогийн endpoint эсвэл холболтын төлөвийг харуулна. |

Бэлтгэлтийн дараа тохиргоог хадгалж, удирдлагатай үйлчилгээний ажлын урсгалд найдахаасаа өмнө холболтын төлөвийг шалгана уу. Хэрэв бэлтгэлт амжилтгүй болбол харуулсан дахин оролдох зааврыг хянаж, сайт hosted үйлчилгээ үзүүлэгчийг ашиглах зөвшөөрөлтэй эсэхийг баталгаажуулна уу.

## Google Calendar тохиргоо {#google-calendar-configuration}

Superdav AI Agent v1.18.0 календарийн боломжууд идэвхжсэн үед agent тохируулсан календариуд болон үйл явдлын дэлгэрэнгүйг уншиж чадна. Календарийн хэрэгслүүд унших чиглэлтэй бөгөөд хуваарь мэддэг сануулга, оролцогчийн дагалдах холбоо, холбоо барих тааруулалтад хэрэгтэй.

| Талбар | Тайлбар |
|---|---|
| **Google Calendar итгэмжлэлүүд** | Календарийн өгөгдлийг уншихад шаардлагатай итгэмжлэл эсвэл token холболтыг хадгална. |
| **Календарь сонголт** | Agent ямар тохируулсан календариудыг шалгаж болохыг хязгаарлана. |
| **Календарийн холболтын төлөв** | Одоогийн итгэмжлэлүүд календариуд болон үйл явдлуудыг уншиж чадах эсэхийг баталгаажуулна. |

Календарийн итгэмжлэлүүдийг agent-д шаардлагатай календариудаар хязгаарлаж байгаарай. Төлөв нь хугацаа дууссан token зааж байвал итгэмжлэлүүдийг дахин холбоно эсвэл солино уу.

## TextBee SMS мэдэгдлүүд {#textbee-sms-notifications}

Superdav AI Agent v1.18.0 нь тохируулсан мэдэгдлийн ажлын урсгалуудад зориулан TextBee-г SMS үйлчилгээ үзүүлэгч болгон нэмдэг. Мэдрэмтгий эсвэл хэрэглэгчид харагдах мессежүүдэд SMS мэдэгдлийг хүний зөвшөөрлийн хаалгатай хослуулах хэрэгтэй.

| Талбар | Тайлбар |
|---|---|
| **TextBee API Key** | TextBee SMS үйлчилгээ үзүүлэгч рүү илгээх хүсэлтүүдийг баталгаажуулна. |
| **TextBee төхөөрөмж / Илгээгч** | Үйлчилгээ үзүүлэгч шаардсан үед гарах мессежүүдэд ашиглах TextBee илгээгч эсвэл төхөөрөмжийг сонгоно. |
| **SMS мэдэгдлүүд идэвхтэй** | Зөвшөөрөгдсөн ажлын урсгалуудад текст мессежийн мэдэгдэл илгээхийг зөвшөөрнө. SMS илгээлтээс сэргийлэхийн тулд идэвхгүй орхино уу. |

Туршилтын мессежийг зөвхөн администраторын эзэмшдэг дугаар руу илгээгээд, дараа нь хуваарьт эсвэл оролцогчдод чиглэсэн сануулгуудыг идэвхжүүлэхээс өмнө зөвшөөрлийн хаалганы үйлдлийг баталгаажуулна уу.

## Боломжийн тугууд {#feature-flags}

Мөн v1.9.0-д танилцуулсан **Тохиргоо → Боломжийн тугууд** таб нь нэмэлт ажиллагаанд зориулсан асаах/унтраах шилжүүлэгчүүдийг өгдөг. Туг бүр сүлжээ даяар идэвхтэй эсвэл идэвхгүй байна; одоогоор сайт тус бүрээр дарж өөрчлөх боломж байхгүй.

### Боломжийн тугуудад хандах {#accessing-feature-flags}

1. WordPress админ хэсэгт **Gratis AI Agent → Тохиргоо** руу очно уу.
2. **Боломжийн тугууд** таб дээр дарна уу.

### Хандалтын хяналтын тугууд {#access-control-flags}

| Туг | Өгөгдмөл | Тайлбар |
|---|---|---|
| **Зөвхөн Administrators-д хязгаарлах** | Унтраалттай | Идэвхжүүлсэн үед зөвхөн `administrator` үүрэгтэй хэрэглэгчид AI Agent чатын самбарыг нээж чадна. Бусад бүх үүрэгтэй хэрэглэгчид оронд нь "Администратортайгаа холбогдоно уу" гэсэн зурвас харна. |
| **Зөвхөн Network Admins-д хязгаарлах** | Унтраалттай | Multisite сүлжээнд идэвхжүүлсэн үед зөвхөн Super Admins агент ашиглаж чадна. Тусдаа сайтын администраторууд хориглогдоно. Хоёуланг нь идэвхжүүлсэн бол "Зөвхөн Administrators-д хязгаарлах"-аас давуу үйлчилнэ. |
| **Subscriber хандалтыг зөвшөөрөх** | Унтраалттай | Идэвхжүүлсэн үед `subscriber` үүрэгтэй хэрэглэгчид чатын интерфэйсийг ашиглаж болох боловч зөвхөн унших боломжуудаар хязгаарлагдана (нийтлэл үүсгэх эсвэл тохиргоо өөрчлөх боломжгүй). |
| **Гишүүн бус хүмүүст идэвхгүй болгох** | Унтраалттай | Ultimate Multisite гишүүнчлэлийн төлөвтэй нэгтгэнэ. Идэвхжүүлсэн үед идэвхтэй гишүүнчлэлгүй сайтуудад чат нуугдана. |

### Брэндийн тугууд {#branding-flags}

| Туг | Өгөгдмөл | Тайлбар |
|---|---|---|
| **"Powered by Gratis AI Agent" хөлийг нуух** | Унтраалттай | Чатын виджетийн доод хэсэгт харагддаг брэндийн хамаарлын мөрийг устгана. White-label байршуулалтад зөвлөмжтэй. |
| **Тусгай агентын нэр** | *(хоосон)* | Чатын толгой хэсэг болон админ цэс дэх өгөгдмөл "Gratis AI Agent" шошгыг таны өөрийн бүтээгдэхүүний нэрээр солино. Өгөгдмөлийг ашиглах бол хоосон үлдээнэ үү. |
| **Агент сонгогчийг нуух** | Унтраалттай | Идэвхжүүлсэн үед хэрэглэгчид суурилуулсан таван агентын хооронд шилжиж чадахгүй. Одоогийн агент Settings → General-д өгөгдмөлөөр тохируулсан зүйлээр тогтмол байна. |
| **Сайтын дүрсийг чатын аватар болгон ашиглах** | Унтраалттай | Чатын виджетийн толгой хэсэг дэх өгөгдмөл AI дүрсийг WordPress сайтын дүрсээр солино (Appearance → Customize → Site Identity хэсэгт тохируулна). |

### Автоматжуулалтын аюулгүй байдлын тугууд {#automation-safety-flags}

Superdav AI Agent v1.18.0 нь илүү аюулгүй автоматжуулалтын гүйцэтгэлд зориулан хүний зөвшөөрлийн хаалтууд болон сануулгын бүртгэлүүдийг нэвтрүүлж байна. Эдгээр хяналтууд нь суулгасан багцаас хамааран функцийн тугууд эсвэл дэвшилтэт автоматжуулалтын тохиргоонд гарч ирж болно.

| Туг | Өгөгдмөл | Тайлбар |
|---|---|---|
| **Хүний зөвшөөрөл шаардах** | Зөвлөмжтэйгээр асаалттай | Эрх бүхий хэрэглэгч санал болгосон үйлдлийг хянаж баталгаажуулах хүртэл мэдрэмтгий автоматжуулалтыг түр зогсооно. |
| **Сануулгын давхардлыг арилгах** | Асаалттай | Илгээсэн сануулгуудыг бүртгэж, дахин оролдлого эсвэл хуваарьт гүйцэтгэл давхар мэдэгдэл илгээхээс сэргийлнэ. |
| **Календарын хэрэгслүүдийг идэвхжүүлэх** | Тохируулах хүртэл унтраалттай | Агентэд тохируулсан Google календарь болон үйл явдлуудыг унших боломж олгоно. |
| **SMS мэдэгдлийг идэвхжүүлэх** | Тохируулах хүртэл унтраалттай | Итгэмжлэлүүд хадгалагдсаны дараа зөвшөөрөгдсөн ажлын урсгалууд TextBee SMS мэдэгдэл илгээх боломж олгоно. |

### Өөрчлөлтүүдийг хэрэгжүүлэх {#applying-changes}

Аливаа тугийг солисны дараа **Тохиргоог хадгалах** дээр дарна уу. Өөрчлөлтүүд шууд хүчин төгөлдөр болно — кэш цэвэрлэх эсвэл plugin дахин идэвхжүүлэх шаардлагагүй.
