# Төлбөр ба Инвойсы Удирдах {#managing-payments-and-invoices}

## Төлбөрийн Тохиргоо {#payment-settings}

Төлбөр авах эхлэхээс өмнө та төлбөрт холбогдох тохиргоог тохируулах хэрэгтэй. **Ultimate Multisite → Settings** руу очиж, **Payment** (Төлбөр) таб дээр дарна уу.

![Payment settings tab](/img/admin/settings-payments-top.png)

Энд төлбөрийн тохиргоогийн бүрэн харагдах дэлгэц байна:

![Payment settings full page](/img/admin/settings-payments-full.png)

### Ерөнхий Төлбөрийн Сонголт {#general-payment-options}

Ерөнхий тохиргоо дотор та дараах зүйлсийг тохируулах боломжтой:

- **Currency** (Мөнгө): Гүнзгий төлбөр хийхэд ашиглагдах анхны мөнгө.
- **Currency Position** (Мөнгөн тэмдэгт байрлал): Мөнгөн тэмдэгт ямар байрлалд гарна вэ (тоос өмнө/хойно).

![Payment gateway settings](/img/admin/settings-payments-options.png)

### Төлбөр Хаалтын Системүүд (Payment Gateways) {#payment-gateways}

Ultimate Multisite нь олон төлбөрийн хаалтын системтэй ажилладаг. Та Payment тохиргоо таб дээрээ тус бүрийг идэвхжүүлж, тохируулах боломжтой.

![Payment gateway configuration](/img/admin/settings-payments-gateways.png)

Боломжит хаалтын системүүд:

- **Stripe** — Stripe-ээр дамжуулан кредит карт төлбөр хийх
- **PayPal** — PayPal-аар төлбөр хийх
- **Manual** (Хүчээр/Өөрөө) — Офлайн эсвэл өөрийн тохируулгатай төлбөрийг боловсруулах

Нэг хаалтын систем нь API key болон бусад тохиргоог оруулдаг өөрийн тохиргооны хэсгийг агуулдаг.

![Additional gateway settings](/img/admin/settings-payments-gateways-2.png)

### Sandbox Mode (Туршилтын режим) {#sandbox-mode}

Үнэхээр ажиллахаас өмнө төлбөр хийх холбоотойгоо туршиж үзэхэд **Sandbox Mode**-ыг идэвхжүүлэх боломжтой. Энэ режим ажиллаж байх үед ямар ч бодит төлбөр хийгддэггүй.

## Төлбөрийг Харах {#viewing-payments}

Ultimate Multisite доторх **Payments** (Төлбөр) хуудсанд та өөрийн сүлжээний бүх гүйлгээг харах боломжтой.

![Payments list](/img/admin/payments-list.png)

Та төлбөрийг статусаар (боловсруулсан, хүлээгдэж буй, алдаатай, буцаалсан) шүүх эсвэл тодорхой гүйлгээг хайж олох боломжтой.

Төлбөд товчлуур дээр дарж түүний бүрэн дэлгэрэнгүй мэдээллийг үзээрэй. Энэ нь тухайн төлбөрөөс хамаарах зүйлс, гишүүнчлэлийн холбоотой мэдээлэл, хэрэглэгчийн мэдээлэл болон төлбөрийн шүүрэн (payment gateway) өгөгдлийг багтаасан байдаг.

## Авиар (Invoices) {#invoices}

Ultimate Multisite нь төлбөр хийгдэх үед авиарыг автоматаар үүсгэж чадна. Та төлөвлөгөөний загвар болон дугаарлах хэлбэрийг Төлбөрийн тохиргоо (Payment settings) -ос өөртөө тохируулах боломжтой.

Авиарыг тохируулах сонголтууд дараах байдалтай:

- Авиар дээр харагдах **Компанийн нэр ба хаяг**
- **Авиар дугаарлалтын хэлбэр болон дараалал**
- Авиарны гарчигт (header) харагдах **Лого**
- Нөхцөл, тэмдэглэл эсвэл хууль зүйн мэдээллийн хувьд **Тохируулсан доод хэсгийн текст (Custom footer text)**

Авиарны загварыг тохируулахын тулд **Ultimate Multisite → Settings → Payment** руу очиж, авиартай холбоотой тохиргоог хайж үзээрэй.
