# Үйл явдал: Zapier-ээр хаяг оруулах {#event-register-an-account-via-zapier}

[Ultimate Multisite-ийг Zapier-тэй холбох](zapier.md) нийтлэлд бид Zapier-ийг Ultimate Multisite доторх өөр үйлдэл хийхийн тулд trigger болон event-ээс ямар ч зүйлийг хэрэглэх талаар ярьсан. Энэ нийтлэлийнхээ зорилгоор бид гуравдагч талын аппликейшнуудыг хэрхэн интеграц хийж болохыг харуулах болно. Бид өгөгдлийн эх сурвалжаар Google Sheets-ийг ашиглаж, тухайн мэдээллийг Ultimate Multisite руу оруулан хаяг оруулах үйлдлийг хийе.

Zapier-тэй холбохоос өмнө **Ultimate Multisite > Settings > API & Webhooks** руу очиж API-ийг идэвхж байгаа эсэхийг баталгаажуулна уу. Zapier Ultimate Multisite-ийн токен шаардах үед энэхүү дэлгэцээс API Key болон API Secret-ийг хуулна уу.

![API ба Webhooks тохиргоо, API Key, API Secret, API идэвхж байгаа сонголтууд](/img/admin/settings-api-webhooks.png)

Эхлээд таны Google Drive дотор **Google Sheet** үүсгэх хэрэгтэй. Дараа нь өгөгдлийг дараа нь амархан тохируулахын тулд бүх баганын утгыг зөв тодорхойлохыг анхаарна уу.

Google Sheet-ийг үүсгэсний дараа та Zapier аккаунтод нэвтэрч, zap үүсгэх эхлэлийн үйлдлийг хийж болно.

**"App event"** гэсэн хайлт хийх талбарт **"Google Sheets"**-ийг сонгоно.

Дараа нь "**Event** " талбарт **"New spreadsheet row"** (Шинэ цагийн мөр) -ийг сонгоод "**Continue**" (Үргэлжлүүлэх)-ийг дарна.

Дараагийн алхамд Google Sheet-ийн хадгалагдаж буй **Google Account**-ыг сонгох шаардлагатай болно. Тиймээс зөв Google аккаунтыг тохируулсан эсэхийг анхаарна уу.

**"Set up trigger"** дотор таны өгөгдөл ирж буй Google Sheet болон worksheet-ийг сонгох шаардлагатай болно. Зөвхөн тэдгээрийг дүүргэж, "**Continue**" (Үргэлжлүүлэх)-ийг дарна уу.

Дараа нь таны Google Sheet зөв холбогдсон эсэхийг шалгахын тулд **"test your trigger"** (trigger-ээ турших) үйлдлийг хийнэ.

Хэрэв тест амжилттай бол таны spreadsheet-ээс зарим утгууд гарч байгааг харах ёстой. Үргэлжлүүлэхийн тулд "**Continue**" (Үргэлжлүүлэх)-ийг дарна уу.

Дараагийн алхам бол Ultimate Multisite-д бүртгэл хийх эсвэл тодорхойлолт хийх хоёр дахь үйлдлийг тохируулах явдал юм. Хайлт хийх талбарт "**Ultimate Multisite(2.0.2)**"-ыг сонгоно.

"**Event**" (Үйл явдал) талбарын доор "**Register an Account in Ultimate Multisite**" (Ultimate Multisite-д бүртгэл хийх) гэсэн сонголтыг хийгээд дараа нь "**Continue**" (Үргэлжлүүлэх) товчийг дарна.

"**Set up an action**" (Үйлдэл тохируулах) хэсэгт та үйлчлүүлэгчийн өгөгдөл, гишүүнчлэл, бүтээгдэхүүн зэрэгт янз бүрийн талбарууд харагдана. Та Google Sheet-ийн утгуудыг эдгээрт тохируулаад, дэлгэцийн доор байгаа жишээний дагуу тэдгээрийг зохих талбаруудад оруулах боломжтой.

Утгуудыг тохируулсны дараа та тухайн үйлдлийг шалгах боломжтой.
