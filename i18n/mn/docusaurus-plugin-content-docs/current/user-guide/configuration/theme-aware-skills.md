# Сэдэвд-мэдлэгтэй ур чадварүүд

Superdav AI Agent v1.10.0-ийн хамт дөрвөн шинэ, WordPress-ийн таны одоо ажиллаж буй сэмийн (theme) архитектур болон функцүүдэд тохирсон тусгай зааварчилгаа болон чадваруудыг өгөх, автоматаар тохирох дөрвөн шинэ сэмийн мэдлэгтэй ур чадвар (theme-aware skills)-ыг орууллаа.

## Сэмийн Мэдлэгтэй Ур Чадвар гэж юу вэ?

Сэмийн мэдлэгтэй ур чадвар гэдэг нь таны сайтын одоо ажиллаж буй WordPress сэмийн үндсэн дээр AI туслах систем автоматаар сонгодог, өмнө нь тохируулсан мэдлэгийн сан болон хэрэгслийн баг юм. Та сэмийг солиход, туслах системийн ашиглаж болох ур чадвар автоматаар шинэчлэгдэх бөгөөд гараар ямар ч тохируулга хийх шаардлагагүй.

Нэг ур чадварт дараах зүйлс багтдаг:

- **Сэмийн тусгай зааварчилгаа** — сэмийг ашиглах, өөрчлөх талаарх заавар
- **Блок болон загварын (pattern) дурсгал** — ашиглаж болох блок, загвар, дизайн сонголтууд
- **Тохируулгын жишээ** — ердийн ажлуудад зориулсан кодны хэсг ба тохиргооны загварууд
- **Хамгийн сайн практик** — сэмийн архитектур болон ажиллах арга барилыг сайжруулах зөвлөмж

## Боломжит Сэмийн Мэдлэгтэй Ур Чадварүүд

### Блок Сэмийн (Block Themes)

**Хэрэглэх:** WordPress-ийн блок дээр суурилсан (Full Site Editing) архитектурыг ашигладаг сэмийн хувьд.

Блок Сэмийн ур чадвар нь дараах зүйлүүдэд зааварчилгаа өгнө:

- Блок редактор ашиглан загвар (template) үүсгэх, засварлах
- Блок загвар болон дахин ашиглаж болох блоктой ажиллах
- Глобал стили болон theme.json тохиргоог өөрчлөх
- Сэмийн блок болон тэдгээрийн хувилбарыг ашиглах
- Таны сайтын зориулсан өөрийн блок загварыг үүсгэх

**Автоматаар идэвхждэг:** Таны одоо ажиллаж буй сэмийн төрөл блок сэмийн байвал (block-templates функц дэмждэг).

### Классик Сэмийн (Classic Themes)

**Хэрэглэх:** PHP загвар болон классик редакторыг ашигладаг уламжлалт WordPress сэмийн хувьд.

Классик Сэмийн ур чадвар нь дараах зүйлүүдэд зааварчилгаа өгнө:

PHP шаблон файлууд болон хуулан (hooks) ашиглах
Теманы харагдацыг Customizer-ээр өөрчлөх
Widget areas болон sidebars ашиглах
CSS болон child theme хөгжүүлэлтийг өөрчлөх
Theme hierarchy болон template tags-ийг ойлгох

**Дараах үед автоматаар идэвхтэй болно:** Таны актив тему нь classic (block-гүй) тема байх үед.

### Kadence Blocks

**Хэрэглэх:** Advanced block-based design хийхийн тулд Kadence Blocks plugin ашигладаг сайтуудад хамаарттай.

Kadence Blocks skill нь дараах зүйлүүдэд чиглүүлэл өгнө:

- Kadence-ийн дэвшилтэт блок сан (Hero, Testimonials, Pricing гэх мэт) ашиглах
- Kadence block setting болон responsive сонголтуудыг тохируулах
- Kadence grid болон container blocks ашиглан өөрийн загварыг бүтээх
- Kadence block-уудыг таны темутай нэгтгэх
- Kadence-ийн design system болон presets-ийг ашиглах

**Дараах үед автоматаар идэвхтэй болно:** Таны сайтод Kadence Blocks plugin идэвхтэй байх үед.

### Kadence Theme

**Хэрэглэх:** Block-based design болон тохиргоо хийхийн тулд Kadence theme ашигладаг сайтуудад хамаарттай.

Kadence Theme skill нь дараах зүйлүүдэд чиглүүлэл өгнө:

- Global styles болон theme.json-ээр Kadence temээ тохируулах
- Kadence-ийн суурь block patterns болон templates ашиглах
- Kadence theme setting болон сонголтуудыг тохируулах
- Kadence design system-ээр өөрийн загварыг бүтээх
- Таны сайтод алдартай plugin болон tool-той Kadence-ийг нэгтгэх

**Дараах үед автоматаар идэвхтэй болно:** Kadence theme таны актив тема байх үед.

## Skills хэр сонгогдодог

Ассистент нь тухайн мессежийн дагуу таны актив тему болон татаж байгаа plugin-уудыг автоматаар илрүүлдэг. Тохирох skill байгаа бол түүнийг ассистентын context-д автоматаар оруулдаг. Та skill-ийг гараар идэвхж эсвэл солих шаардлагагүй.

### Олон skill

Хэрэв таны сайтод хэд хэдэн ур чадвар (жишээлбэл, Kadence Blocks болон Kadence Theme хоёулаа идэвхтэй байвал) хамаарах тохиолдол гарвал, туслах ассистент бүх холбогдох ур чадваруудыг ашиглаж, тэдгээрийн заавраас мэдээлэл авч болно.

### Сэмийг солих (Switching Themes)

Та идэвхтэй сэмийг өөрчилсөн үед дараагийн мессежинд ассистен утас боломжит ур чадваруудаа автоматаар шинэчилнэ. Жишээлбэл:

1. Та **Block Themes** (Блок Сэмийн) ур чадварыг идэвхтэй блок сэмийн ашиглаж байна.
2. Та классик сэмийн (classic theme)-д шилжив.
3. Дараагийн мессежинд **Classic Themes** (Классик Сэмийн) ур чадвар автоматаар ачаалагдана, харин **Block Themes** ур чадвар илүүдэхгүй болно.

## Сэмийг мэддэг ур чадварыг ашиглах (Using Theme-Aware Skills)

Сэмийг мэддэг ур чадварыг ашиглахын тулд зүгээр л чат интерфейсээр юу хийхийг хүсэхийг тайлбарлаарай. Ассистент автоматаар тохирох ур чадварын заавраас мэдээлэл авна.

### Жишээний асуултууд (Example Prompts)

**Block Themes-ийн хувьд:**
> "Блок загваруудыг ашиглан фонык зураг болон дунд нь тексттэй хөндлөнгийн хэсэг үүсгэ."

**Classic Themes-ийн хувьд:**
> "Child theme ашиглан сайтын талын (sidebar) дээр өөрийн загварын widget area нэмнэ."

**Kadence Blocks-ийн хувьд:**
> "Kadence Testimonials block-ыг ашиглан үйлчлүүлэгчдийн тайлбаруудын хэсгийг бүтээ."

**Kadence Theme-ийн хувьд:**
> "Header layout болон навигацийн менюгийн загварыг өөрчилж үзээрэй."

Ассистент таны идэвхтэй сэмийн болон плагинуудад тохирсон заавар, код жишээг өгнө.

:::note
Theme-aware skills нь Superdav AI Agent v1.10.0 ба түүнээс хойших хувилбаруудад автоматаар ашиглагдана. Ямар нэгэн нэмэлт тохируулга эсвэл тохиргоо хийх шаардлагагүй.
:::
