# Интернет хайлт (Internet Search) {#internet-search}

Gratis AI Agent v1.5.0-д чат ярианы үед сүүлийн үеийн мэдээллийг интернэтээс хайж авах **Интернет хайлт** чадварыг нэмсэн. Энэ нь [Brave Search API](https://brave.com/search/api/) эсвэл [Tavily API](https://tavily.com/) -аар ажилладаг.

## Хэрхэн ажилладаг вэ? {#how-it-works}

Интернет хайлт идэвхтэй байвал, туслах агент нь асуулт нь одоогийн эсвэл гадаад мэдээлэл шаардлагатай гэж үзсэн үед таны тохируулсан хайлт хийх үйлчилгээг автоматаар асууж авч болно. Жишээлбэл, сүүлийн үеийн мэдээ, бодит үнийн мэдээлэл эсвэл загварын сургалтаас хойш өөрчлөгдсөн баримт бичиг зэрэг юм.

Үр дүн нь бодит цагийн хугацаанд авч, хариултыг үүсгэхээс өмнө туслах агентны контекстэд оруулдаг. Агент асуултанд хариулахын тулд хайлт хийсэн гэдгээ мэдэгдэх үед мэдээлэл гаргана.

## Интернет хайлт идэвхжүүлэх {#enabling-internet-search}

Интернет хайлтын хувьд таны сонгосон хайлт хийх үйлчилгээний API key шаардлагатай. Тохируулахын тулд:

1. **Gratis AI Agent → Settings → Advanced** руу орно.
2. **Internet Search Provider** доторх dropdown-ыг олж, **Brave Search** эсвэл **Tavily** -ийг сонгоно.
3. Хаслалт хийх талбарт өөрийн API key-г оруулна. Талбарын дагуу бүртгүүлэх холбоосууд харагдана.
4. **Save Settings**-ийг дарна.

Key-г хадгалсны дараа, Internet Search чадвар автоматаар туслах агентд ашиглахаар бэлэн болно.

## Brave Search {#brave-search}

### Brave Search API Key авах {#obtaining-a-brave-search-api-key}

1. [Brave Search API page](https://brave.com/search/api/) руу очино.
2. Төлөвлөгөө (plan) авна. Хурлын хязгаартай сар бүрийн хүсэлтийн хязгаартэй үнэгүй түвшин байдаг.
3. Brave Search developer dashboard-оос API key-г хуулна.
4. Үүнийг Gratis AI Agent-ийн settings доторх **Brave Search API Key** талбарт хийн оруулна.

### Хэрэглээний хязгаар {#usage-limits}

Хэрэглээг Brave Search-ийн таамаглалтаар тоолно, хийсэн асуултны тоогоор төлөгдөнө. Асуултыг үүсгэсэн AI хариу бүр нэг асуулт гэж тооцогдоно. Хүлээгдэхгүй төлбөрөөс зайлсхийхийн тулд [Brave Search developer dashboard](https://brave.com/search/api/) дээр хэрэглээгээ хянах хэрэгтэй.

## Tavily {#tavily}

Superdav AI Agent v1.10.0 нь илүү баялаг хайлт, дэвшилтэт судалгааны чадвартай байх боломжийг олгодог альтернатив интернэт хайлтын үйлчилгээ болгон **Tavily**-г нэмж оруулсан.

### Tavily API Key авах {#obtaining-a-tavily-api-key}

1. [Tavily API page](https://tavily.com/) руу очихыг хичээгээрэй.
2. Тооцоогоо бүртгүүлнэ. Хурдан хэрэглэгчид зориулсан давтамжийн хязгаартай үнэгүй түвшн байдаг.
3. Tavily dashboard-аас API key-гээ хуулна уу.
4. Gratis AI Agent тохиргооны **Tavily API Key** талбартээ тусгана.

### Хэрэглээний хязгаарлалт {#usage-limits-1}

Хэрэглээг Tavily-ээр хийсэн API дуудлагын тоогоор төлөгдөнө. Асуултыг үүсгэсэн AI хариу бүр нэг дуудлага гэж тооцогдоно. Хүлээгдэхгүй төлбөрөөс зайлсхийхийн тулд [Tavily dashboard](https://tavily.com/) дээр хэрэглээгээ хянах хэрэгтэй.

## Интернэт хайлт унтраах {#disabling-internet-search}

API key-г идэвхтэй хайлтын үйлчилгээний талбараас арилгаж, хадгална уу. Энэ үед туслах (assistant) нь интернэт хайлт хийх боломжийг алдах болно.

:::note
Интернэт хайлт нь хариулт өгөхөөс өмнө туслах нь хайлтын үр дүнг хүлээх шаардлагатай тул хариу өгөх хугацаанд нэмэлт цаг (latency) үүсгэдэг. Цаг хугацааны чухал асуудлуудад, бодит цагийн хайлт зайлшгүй хэрэгтэй эсвэл туслахын доторх мэдлэгийг нь шалгах нь зүйтэй.
:::
