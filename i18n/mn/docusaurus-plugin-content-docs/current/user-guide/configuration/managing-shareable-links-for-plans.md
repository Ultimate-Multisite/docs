# Плануудтай холбоотой линкүүдийг удирдах {#managing-shareable-links-for-plans}

Ultimate Multisite танд планны линкүүдийг шууд үйлчлүүлэгчдэд хуваалцах боломжийг олгодог. Үүнийг хийхийн тулд таны планыйн редактор (edit) хуудас руу орж, _Shareable Link-ийг хуулбарлах_ товчийг дарна уу.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_Планын редакторт таны хуваалцах линк үүсгэх товчийг олох боломжтой._

Энэ механизм нь линкийг үүсгэхийн тулд анхны төлөх давтамжийг ашигладаг гэдгийг анхаарна уу. Энэ нь таны анхны төлөх хугацаа сарын бол, жишээлбэл, эцэст нь та ийм линктэй болох юм: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Хэрэв та хуваалцах линк нь улирлын үндсэн давтамжтай (quarterly) байхыг хүсвэл, эцэст нь 3-ийг нэмэх шаардлагатай болно:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Хэрэв та _premium_ планыг жилийн хуваарийн дагуу сонгохыг хүсвэл, 1-ийг нэмээд дараагийн үгийг оруулна:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Та анхаарч байгаа зүйл бол өөр өөр төлөх давтамжуудад нэг л линк байх боломжгүй гэдэг юм. Та өөрийн төлөх давтамжийн тус бүрт тохирох нэг тусгай линк хуваалцах ёстой.

Линкээ үйлчлүүлэгчдэд хуваалцсны дараа, тэд таны бүртгэлийн (signup flow) эхний алхам руу чиглэгдэх болно.
