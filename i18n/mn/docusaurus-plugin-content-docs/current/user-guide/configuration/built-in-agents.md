# Суурилагдсан агентууд

Gratis AI Agent v1.9.0 нь таван суурилагдсан агенттай ирдэг бөгөөд тус бүр нь тухайн чиглэлийн нийтлэг ажлуудад тохирсон төвлөрсөн хэрэгслүүдийн багц, өөрт нь тааруулсан системийн заавар, эхлэх саналуудаар урьдчилан тохируулагдсан. Агент хооронд шилжихэд туслагч юу хийж чадах болон хэрхэн хариулах нь өөрчлөгдөнө — таны зүгээс ямар ч тохиргоо хийх шаардлагагүй. Superdav AI Agent v1.18.0 нь холбогдох интеграцуууд тохируулагдсан үед эдгээр ажлын урсгалд хуваарь мэддэг хэрэгслүүд, сануулгын бүртгэлүүд, зөвшөөрлийн хаалтууд болон SMS мэдэгдлүүдийг нэмж чадна.

## Агент гэж юу вэ? {#what-is-an-agent}

Агент бүр нь дараахыг нэгтгэсэн нэртэй тохиргооны профайл юм:

- **Хэрэгслүүд** — агент дуудах эрхтэй боломжууд (жишээ нь Content Writer нь нийтлэл үүсгэх боломжуудад хандах эрхтэй; Design Studio нь CSS болон theme.json боломжуудад хандах эрхтэй)
- **Системийн заавар** — агентын өнгө аяс, тэргүүлэх чиглэл, хязгаарлалтыг тогтоох зааврууд
- **Саналууд** — хурдан эхлэхэд туслахын тулд чат интерфейс дээр харуулдаг урьдчилан бичсэн prompt-ууд

## Агент сонгогчид хандах {#accessing-the-agent-picker}

1. WordPress админ хажуугийн самбарт **Gratis AI Agent** самбарыг нээнэ үү.
2. Чатын толгой хэсгийн зүүн дээд талд байгаа **агентын дүрс** дээр дарна уу (дүрс нь идэвхтэй агентыг илэрхийлэхээр өөрчлөгдөнө).
3. **Агент сонгогч** нь маягт-хүснэгтийн давхарга хэлбэрээр нээгдэнэ. Агент бүрийг дүрс, нэр, нэг мөрийн тайлбартайгаар жагсаасан байна.
4. Идэвхжүүлэхийн тулд агентын мөр дээр дарна уу. Чатын толгой хэсэг даруй шинэчлэгдэнэ.

Та мөн харилцан ярианы дундуур агент сольж болно — шинэ агентын системийн заавар дараагийн мессежээс эхлэн хүчин төгөлдөр болно.

## Таван суурилагдсан агент {#the-five-built-in-agents}

### Агуулга бичигч {#content-writer}

**Төвлөрөл:** Нийтлэл, хуудас болон холбоо барих маягтууд үүсгэх, засварлах.

**Боломжтой хэрэгслүүд:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`. Superdav AI Agent v1.18.0 интеграц идэвхжсэн үед тохируулсан календарийн контекст, зөвшөөрлийн хаалтууд, сануулгууд болон SMS мэдэгдлийн хэрэгслүүд нь батлагдсан ажлын урсгалд мөн боломжтой байж болно.

**Сайн хийдэг зүйлс:**
- Товч эсвэл тоймоос блог нийтлэл ноороглох, нийтлэх
- Шинэ сайтад зориулж landing page-уудын багц үүсгэх
- Холбоо барих болон лавлагааны маягтууд бүтээх
- URL эсвэл хайлтаас нийтлэл дээр онцлох зураг тохируулах
- Тохируулсан Google Calendar контекстээс арга хэмжээний дараах дагах мессежүүдийг ноороглож, мэдэгдэл илгээхийн өмнө зөвшөөрөл хүлээн түр зогсох

**Эхлэх саналууд:**
- *WordPress multisite-ийн давуу талуудын тухай 500 үгтэй блог нийтлэл бич.*
- *About, Services, Contact хуудсууд үүсгээд нийтэл.*
- *Contact хуудсанд захиалгын лавлагааны маягт нэм.*
- *Маргаашийн тохируулсан календарийн арга хэмжээнд оролцогчдод сануулга ноороглоод, илгээхийн өмнө зөвшөөрөл хүлээ.*

### Сайт бүтээгч {#site-builder}

**Төвлөрөл:** Нэг prompt-оос эхлээд эцсийн хүртэл вебсайт үүсгэх.

**Боломжтой хэрэгслүүд:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`. Superdav AI Agent v1.18.0-тэй үед администраторууд идэвхжүүлсэн газарт тохируулсан managed-service, зөвшөөрөл, сануулга, календарь болон SMS хэрэгслүүд боломжтой байж болно.

**Сайн хийдэг зүйлс:**
- Тайлбарласан бизнесийн төрөлд зориулж олон үе шаттай сайт бүтээх төлөвлөгөө үүсгэх
- Үе шат бүрийг бие даан гүйцэтгэх — бүтэц, агуулга, навигац, дизайн
- Төлөвлөгөөний дундуур гарсан алдаанаас гараар оролцох шаардлагагүйгээр сэргэх
- Бүтээлтийн нэг хэсэг болгон санал болгосон plugin-уудыг суулгах
- Чат интерфейсээс шууд холбоо барих маягтууд үүсгэх (Superdav AI Agent v1.10.0+)
- Зөвшөөрлийн хаалтууд болон сануулгын бүртгэлүүд идэвхжсэн үед давхар мэдэгдэлгүйгээр нээлтийн сануулгууд эсвэл оролцогчдын дагах ажиллагааг зохицуулах

**Эхлэх саналууд:**
- *Галерейн нийтлэлийн төрөл, захиалгын хуудас, холбоо барих маягттай гэрэл зургийн портфолио сайт бүтээ.*
- *Онлайн цэс, ажиллах цаг, ширээ захиалгын лавлагааны маягттай рестораны вебсайт үүсгэ.*
- *Үйлчилгээний хуудсууд, портфолио хэсэг, блогтой чөлөөт зөвлөхийн сайт тохируул.*
- *Сайт бүтээгчийг ашиглан Contact хуудсанд холбоо барих маягт нэм.*
- *Сайтын нээлтийн шалгах жагсаалт батлагдсаны дараа тохируулсан оролцогч талын холбоо барих хүнд SMS сануулга илгээ.*

### Дизайн студи {#design-studio}

**Төвлөрөл:** Визуал өөрчлөлт — өнгө, хэв шинж, CSS болон блок загварууд.

**Боломжтой хэрэгслүүд:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Сайн хийдэг зүйлс:**
- Нэрлэсэн theme preset-үүдийг хэрэглэх (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- theme.json-оор дамжуулан глобал үсгийн хэв маяг болон өнгөний палетыг нарийвчлан тохируулах
- Брэндэд зориулсан давхар өөрчлөлтүүдэд custom CSS оруулах
- Хуудсын screenshot авч, дизайны асуудлуудыг нь хянах

**Эхлэх саналууд:**
- *warm-editorial preset-ийг хэрэглээд дараа нь үндсэн өнгийг #2d6a4f болгож тохируул.*
- *Нүүр хуудасны screenshot аваад юуг сайжруулахыг надад хэл.*
- *Бүтэн өргөн дэвсгэр зураг, голлуулсан гарчигтай дахин ашиглах боломжтой hero блок загвар үүсгэ.*

### Plugin Manager {#plugin-manager}

**Төвлөрөл:** WordPress plugin-уудыг олох, суулгах, удирдах.

**Боломжтой хэрэгслүүд:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Сайн хийдэг зүйлс:**
- Тайлбарласан хэрэглээний тохиолдолд хамгийн сайн plugin-ыг санал болгох
- Бүртгэлээс ability pack-ууд суулгах
- Боломжтой ability каталогийг ангиллаар нь үзэх

**Эхлэх саналууд:**
- *Гишүүнчлэлийн лавлахад хамгийн сайн plugin юу вэ?*
- *WooCommerce abilities pack-ыг суулга.*
- *Боломжтой бүх ecommerce ability pack-уудыг харуул.*

### Дэмжлэгийн туслах {#support-assistant}

**Төвлөрөл:** Сайтын агуулга, тохиргоо болон WordPress тохиргооны тухай асуултад хариулах.

**Боломжтой хэрэгслүүд:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Үүнийг сайн хийдэг зүйлс:**
- Одоогийн сайтын тохиргоо болон сонголтуудыг хайж олох
- Сайт дээр ямар нийтлэлийн төрлүүд, таксономи, цэснүүд тохируулагдсаныг тайлбарлах
- Амьд утгуудыг уншиж, "энэ тохиргоо юу хийдэг вэ?" гэсэн асуултад хариулах
- Өөрчлөлт хийхээс өмнө зөвхөн унших оношилгооны давхарга болж ажиллах

**Эхлэх санал:**
- *Энэ сайт дээр одоогоор ямар plugin болон тохиргоонууд идэвхтэй байна вэ?*
- *Энэ сайт дээр бүртгэгдсэн бүх custom post type-уудыг жагсаа.*
- *Ямар навигацийн цэснүүд байна, мөн тэдгээрийг хаана оноосон бэ?*

## Superdav автоматжуулалтын интеграцууд {#superdav-automation-integrations}

Superdav AI Agent v1.18.0 интеграцууд тохируулагдсан үед суурилуулсан агентууд илүү аюулгүй, хуваарьт мэдрэмжтэй автоматжуулалтын ажлын урсгалд оролцож чадна:

- **Google Calendar унших хэрэгслүүд** нь агентуудад дараагийн ажлыг боловсруулахын өмнө тохируулсан календарь болон үйл явдлуудыг шалгах боломж олгоно.
- **Холбоо барих хүн болон оролцогчийн зураглал** нь үйл явдлын оролцогчдыг WordPress хэрэглэгчид эсвэл мэдэгдэж буй холбоо барих хүмүүстэй тааруулахад тусална.
- **Хүний зөвшөөрлийн хаалтууд** нь эрх бүхий хэрэглэгч хянаж баталгаажуулах хүртэл эмзэг үйлдлүүдийг түр зогсооно.
- **Сануулгын бүртгэлүүд** нь хуваарьт ажлууд дахин оролдох эсвэл давтагдах үед давхар мэдэгдэл илгээхээс сэргийлнэ.
- **TextBee SMS мэдэгдлүүд** нь SMS итгэмжлэлүүд болон ажлын урсгалын зөвшөөрлүүд идэвхжсэн үед л тохируулсан мессежүүдийг илгээнэ.

Зөвлөмж болгож буй ажлын урсгал: агентээс мессеж эсвэл үйлдлийг бэлтгэхийг хүсэж, зөвшөөрлийн сануулгыг хянаад, дараа нь зөвшөөрөгдсөн үйлдлийг үргэлжлүүлэхийг зөвшөөрнө. Давтагдах сануулгуудад нэг ижил үйл явдал эсвэл холбоо барих хүнд дахин дахин мэдэгдэл очихоос сэргийлэхийн тулд сануулгын давхардал арилгах тохиргоог идэвхтэй байлгаарай.

## Агентуудыг өөрчлөн тохируулах {#customising-agents}

Суурилуулсан агент бүрийг `gratis_ai_agent_agents` filter-ээр өргөтгөх эсвэл солих боломжтой.

### Одоо байгаа агентэд custom system prompt нэмэх {#adding-a-custom-system-prompt-to-an-existing-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### Шинэ агент бүртгэх {#registering-a-new-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

Шинэ агент нь filter ажилласны дараа агент сонгогчид шууд гарч ирнэ.

### Суурилуулсан агент устгах {#removing-a-built-in-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
