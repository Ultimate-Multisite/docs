# Хичээл 3: Сүлжээгээ тохируулах {#lesson-3-setting-up-your-network}

Одоо бүтээх цаг. Энэ хичээлээр та Ultimate Multisite-ийг суулгаж, FitSite сүлжээний суурийг тохируулна. Эндэх бүх шийдвэрийг фитнесийн чиглэлийг бодолцон гаргана.

## Бид хаана зогссон бэ {#where-we-left-off}

Бид фитнес студиудыг өөрийн чиглэл болгон сонгож, боломжийг баталгаажуулсан. Одоо тэр санаагаа ажиллаж буй платформ болгоно.

## Hosting-оо сонгох {#choosing-your-hosting}

Таны hosting-ийн сонголт нэг вебсайтаас илүү niche платформд илүү чухал. Та нэг site байршуулж байгаа биш -- хэдэн арваас хэдэн зуун site болж өсөх сүлжээ байршуулж байна.

### Юуг анхаарах вэ {#what-to-look-for}

- **WordPress Multisite дэмжлэг**: Бүх host multisite-ийг сайн ажиллуулдаггүй
- **Wildcard SSL**: Дэд домэйнд суурилсан сүлжээнд зайлшгүй шаардлагатай
- **Өргөтгөх боломжтой нөөцүүд**: Шилжүүлэлт хийхгүйгээр өсөх зай хэрэгтэй
- **Ultimate Multisite интеграц**: Автомат домэйн холболт болон SSL нь үйл ажиллагааны ихээхэн ажлыг хэмнэнэ

### Зөвлөмж болгож буй арга {#recommended-approach}

[Тохирох providers](/user-guide/host-integrations/closte) жагсаалтаас host сонгоно уу. Эдгээрийг Ultimate Multisite-тэй туршсан бөгөөд домэйн холболт болон SSL автоматжуулалтад хэрэгтэй интеграцуудыг өгдөг.

FitSite-д бид дэд домэйны тохиргоо ашиглана. Энэ нь хэрэглэгчдийн site-ууд өөрийн домэйныг холбохоосоо өмнө эхлээд `studioname.fitsite.com` хэлбэрээр харагдана гэсэн үг.

## WordPress Multisite суулгах {#installing-wordpress-multisite}

Хэрэв танд WordPress Multisite суулгалт хараахан байхгүй бол:

1. Hosting provider дээрээ WordPress суулгана уу
2. [WordPress Multisite хэрхэн суулгах](/user-guide/getting-started/how-to-install-wordpress-multisite) гарын авлагыг дагана уу
3. Сануулах үед **дэд домэйн** тохиргоог сонгоно уу

:::tip Яагаад дэд домэйн вэ?
Дэд домэйн нь хэрэглэгч бүрийн site-д зам (`fitsite.com/studio`) биш, өөрийн тусдаа хаяг (`studio.fitsite.com`) өгдөг. Энэ нь таны хэрэглэгчдэд илүү мэргэжлийн харагдаж, permalink зөрчилдөөнөөс сэргийлнэ. Дэлгэрэнгүй харьцуулалтыг [Ultimate Multisite 101](/user-guide/getting-started/ultimate-multisite-101)-ээс үзнэ үү.
:::

## Ultimate Multisite суулгах {#installing-ultimate-multisite}

[Ultimate Multisite суулгах](/user-guide/getting-started/installing-ultimate-multisite) гарын авлагыг дагаж дараахыг хийнэ үү:

1. Plugin-ийг бүх сүлжээнд upload хийж идэвхжүүлэх
2. [Setup Wizard](/user-guide/getting-started/multisite-setup-wizard)-ийг ажиллуулах

Setup wizard-ийн үеэр FitSite чиглэлээ санаарай:

- **Валют**: Фитнес студи хэрэглэгчдийн тань ашигладаг валютаар тохируулна
- **Компанийн нэр**: "FitSite" (эсвэл таны сонгосон брэндийн нэр)
- **Компанийн лого**: Брэндийн логогоо upload хийнэ үү -- энэ нь нэхэмжлэх болон имэйлүүд дээр харагдана

## Фитнес чиглэлд тохируулах {#configuring-for-the-fitness-niche}

Ultimate Multisite суулгасны дараа эдгээр niche-д зориулсан тохиргооны сонголтуудыг хийнэ үү:

### Ерөнхий тохиргоо {#general-settings}

**Ultimate Multisite > Settings** рүү очиж тохируулна уу:

- **Site нэр**: FitSite
- **Анхдагч role**: Administrator -- фитнес студийн эзэд өөрийн site-ийн контентыг бүрэн хянах шаардлагатай
- **Бүртгэл**: Студийн эзэд өөрсдөө бүртгүүлэх боломжтой байхаар хэрэглэгчийн бүртгэлийг идэвхжүүлнэ

### Имэйлийн тохиргоо {#email-configuration}

Таны системийн имэйлүүд таны niche-ийн хэлээр ярих ёстой. **Ultimate Multisite > Settings > Emails** рүү очиж өөрчилнө үү:

- Ерөнхий "таны шинэ site" гэсэн хэллэгийг фитнесэд зориулсан мессежээр солино
- Жишээ welcome subject: "Таны фитнес студийн вебсайт бэлэн боллоо"
- Жишээ welcome body: Тэдний студи, хичээлүүд болон фитнес site-аа эхлүүлэх талаар дурдана

Бид эдгээрийг Хичээл 8 (Хэрэглэгчийг танилцуулж эхлүүлэх)-д илүү боловсронгуй болгоно, гэхдээ өнгө аясыг одоо тогтоосноор эхний test signup-ууд хүртэл niche-д зориулагдсан мэт санагдана.

### Домэйны тохиргоо {#domain-configuration}

Хэрэв тохирох hosting provider ашиглаж байгаа бол домэйн холболтыг одоо тохируулна уу:

1. **Ultimate Multisite > Settings > Domain Mapping** рүү очно уу
2. Өөрийн тодорхой host-д зориулсан интеграцын гарын авлагыг дагана уу
3. Шинэ subsite-ууд SSL-ийг автоматаар авч байгаа эсэхийг туршина уу

Ингэснээр бид дараагийн хичээлд template болон test site-ууд үүсгэж эхлэхэд бүх зүйл эхнээсээ төгсгөл хүртэл ажиллахыг баталгаажуулна.

## FitSite сүлжээ одоогоор {#the-fitsite-network-so-far}

Энэ хичээлийн төгсгөлд танд дараах зүйлс байна:

```
FitSite сүлжээ
├── WordPress Multisite (дэд домэйны горим)
├── Ultimate Multisite (суулгаж тохируулсан)
├── Wildcard SSL-тэй hosting
├── Домэйн холболт тохируулсан
├── Niche-д зориулсан имэйлийн template-ууд (эхний)
└── Site template-уудад бэлэн (дараагийн хичээл)
```

## Энэ хичээлээр бид юу бүтээсэн бэ {#what-we-built-this-lesson}

- Дэд домэйны горим дахь **ажиллаж буй WordPress Multisite** суулгалт
- FitSite брэндлэлтэйгээр **Ultimate Multisite суулгаж** тохируулсан
- Өсөж буй сүлжээнд зориулан **hosting болон SSL** тохируулсан
- Таны hosting provider-д зориулан **домэйн холболт** тохируулсан
- Эхний өдрөөс **niche-д зориулсан имэйлийн өнгө аяс** тогтоосон
