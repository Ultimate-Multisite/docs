# Хичээл 4: Ниш загварууд бүтээх {#lesson-4-building-niche-templates}

Загварууд бол таны ниш үнэ цэнийн саналын цөм юм. Бүртгүүлээд, ангийн хуваарь, дасгалжуулагчдын танилцуулга, тохирох зураглалтайгаар фитнесийн вэбсайт шиг харагдаж буй сайтыг харсан фитнес студийн эзэн хоосон хуудас харсан хүнээс хавьгүй илүү үлдэх магадлалтай.

## Бид хаана зогссон бэ {#where-we-left-off}

Бид Ultimate Multisite суулгаж, тохируулсан ажиллаж буй FitSite сүлжээтэй болсон. Одоо FitSite-ийг фитнес бизнесүүдэд зориулж бүтээсэн мэт мэдрүүлэх загваруудыг бүтээнэ.

## Ниш загварууд яагаад чухал вэ {#why-niche-templates-matter}

Ерөнхий загварууд таны хэрэглэгчдэд хүнд ажлыг тулгадаг: ямар хуудсууд хэрэгтэйг, ямар агуулгын бүтэц тохирохыг, мөн тухайн салбартаа зөв харагдуулахыг өөрсдөө тодорхойлох шаардлагатай болдог. Ниш загварууд энэ төвөгшөөлийг арилгадаг.

FitSite-д бүртгүүлсэн фитнес студийн эзэн дараахыг харах ёстой:

- Фитнес студийн вэбсайт шиг харагдах нүүр хуудас
- Ангиуд, хуваарь, дасгалжуулагчид, үнэ тарифын хуудсууд аль хэдийн үүссэн байх
- Тэдний салбарт тохирсон зураглал болон орлуулах агуулга
- Фитнесийн хувьд мэргэжлийн, брэндэд нийцсэн мэдрэмж төрүүлэх дизайн

Тэд зөвхөн өөрийн мэдээллээ бөглөнө. Тэд тэгээс эхлэхгүй.

## Загваруудаа төлөвлөх {#planning-your-templates}

Бүтээж эхлэхээсээ өмнө ямар загварууд санал болгохоо шийд. FitSite-ийн хувьд бид гурвыг үүсгэнэ:

### Загвар 1: Studio Essential {#template-1-studio-essential}

Жижиг студиуд болон хувийн дасгалжуулагчдад.

- **Нүүр хуудас** hero зураг, ангийн онцлогууд, үйлдэлд уриалгатай
- **Тухай** хуудас студийн түүх, эрхэм зорилготой
- **Ангиуд** хуудас хуваарийн байрлалтай
- **Дасгалжуулагчид** хуудас профайл картуудтай
- **Холбоо барих** хуудас байршлын газрын зураг болон маягттай
- **Цэвэр, орчин үеийн дизайн** фитнест тохирох өнгөний схемтэй

### Загвар 2: Gym Pro {#template-2-gym-pro}

Илүү олон үйлчилгээтэй тогтсон фитнес заалнуудад.

- Studio Essential-д байгаа бүх зүйл, дээр нь:
- **Гишүүнчлэл** хуудас үнийн хүснэгттэй
- **Галерей** хуудас байгууламжийн зургуудтай
- **Блог** хэсэг фитнес зөвлөгөө, мэдээнд зориулсан
- Нүүр хуудсан дээрх **сэтгэгдлүүд** хэсэг
- **Илүү тод брэндчилэл** тохиргоонууд

### Загвар 3: Fitness Chain {#template-3-fitness-chain}

Олон байршилтай үйл ажиллагаанд.

- Gym Pro-д байгаа бүх зүйл, дээр нь:
- **Байршлууд** хуудас олон байгууламжийн жагсаалттай
- **Франчайз/байршил** дэд хуудасны загвар
- Байршил тус бүрийн дэлгэрэнгүй мэдээлэлтэй **төвлөрсөн брэндчилэл**
- Байршлууд даяарх **ажилтнуудын лавлах**

## Загвар сайт бүтээх {#building-a-template-site}

Ultimate Multisite-д загвар гэдэг нь шинэ хэрэглэгчийн сайтууд ямар харагдахыг хүсэж байгаагаар тохируулсан WordPress сайт л юм. Нэгийг хэрхэн үүсгэх вэ:

### Алхам 1: Загвар сайтыг үүсгэх {#step-1-create-the-template-site}

1. Сүлжээний админ хэсэгт **Sites > Add New** руу очно
2. `template-studio-essential` нэртэй сайт үүсгэнэ
3. Энэ сайт таны ажиллах талбар болно

### Алхам 2: Сэдвийг суулгаж тохируулах {#step-2-install-and-configure-the-theme}

Загвар сайтын Dashboard руу шилжээд:

1. Фитнес бизнесүүдэд тохирох сэдвийг идэвхжүүлнэ
2. Сэдвийн тохиргоо, өнгө, үсгийн хэвийг тохируулна
3. Фитнест тохирох навигацтай header болон footer-ийг тохируулна

:::tip Сэдэв сонголт
Фитнест сайхан харагдахуйц уян хатан боловч таны хэрэглэгчид удирдаж чадахгүй болтлоо төвөгтэй биш сэдэв сонго. Astra, GeneratePress, эсвэл Kadence зэрэг сэдвүүд хөнгөн, өөрчлөх боломжтой, сайн дэмжлэгтэй тул сайн ажилладаг.
:::

### Алхам 3: Хуудсуудыг үүсгэх {#step-3-create-the-pages}

Хуудас бүрийг дараахтайгаар бүтээнэ:

- **Орлуулах агуулга** фитнест тохирсон, байгалийн уншигдах байдлаар ("Welcome to [Studio Name]" болохоос "Lorem ipsum" биш)
- **Орлуулах зургууд** фитнес үйл ажиллагаа харуулсан үнэгүй stock photo сайтуудаас
- Page builder эсвэл block editor ашигласан **ажиллагаатай байрлалууд**

Боломжтой үед орлуулах агуулгыг зааварчилгаатай болго. Ерөнхий дүүргэгч текстийн оронд: "Үүнийг студийнхээ товч тайлбар болон түүнийг юугаараа онцгой болгодгийг бичихээр солино уу. Дасгалжуулалтын философи, туршлагын жилүүд, эсвэл үйлчлүүлэгчид юу хүлээж болохыг дурд." гэх мэт агуулга бич.

### Алхам 4: Плагинуудыг тохируулах {#step-4-configure-plugins}

Фитнес студиудад хэрэгтэй плагинуудыг суулгаж идэвхжүүл:

- Захиалга эсвэл хуваарь гаргах плагин (таны plan түвшинд хамаарах бол)
- Холбоо барих маягтын плагин
- SEO плагин (фитнест хамаарах анхдагч тохиргоотой урьдчилан тохируулсан)

### Алхам 5: Загвар гэж тэмдэглэх {#step-5-mark-as-template}

1. **Ultimate Multisite > Sites** руу очно
2. Загвар сайтыг засварлана
3. **Site Template** toggle-г идэвхжүүлнэ

Санал болгохыг хүссэн загвар бүртээ энэ процессыг давт.

## Загварын чанарын шалгах жагсаалт {#template-quality-checklist}

Загварыг ашиглах боломжтой болгохоос өмнө шалга:

- [ ] Бүх хуудсууд зөв ачаалж, мэргэжлийн харагдаж байна
- [ ] Орлуулах агуулга ашигтай бөгөөд нишэд тусгайлан тохирсон байна
- [ ] Зургууд тохиромжтой бөгөөд зөв лицензтэй байна
- [ ] Бүх хуудсан дээр гар утасны дэлгэцэд дасан зохицол ажиллаж байна
- [ ] Навигац логиктой бөгөөд бүрэн байна
- [ ] Холбоо барих маягтууд ажиллаж байна
- [ ] Эвдэрсэн холбоос эсвэл дутуу asset байхгүй
- [ ] Хуудас ачаалах хурд зөвшөөрөхүйц байна

## Энэ хичээлээр бид юу бүтээв {#what-we-built-this-lesson}

- Янз бүрийн хэмжээтэй фитнес бизнесүүдэд зориулж боловсруулсан **нишэд тусгай гурван загвар**
- Платформыг зориудаар бүтээсэн мэт мэдрүүлэх **фитнест тохирох агуулга ба зураглал**
- Хэрэглэгчдийг өөрчлөн тохируулахад чиглүүлэх **зааварчилгаатай орлуулах агуулга**
- Загварууд үйлдвэрлэлд бэлэн эсэхийг баталгаажуулах **чанарын шалгах жагсаалт**
