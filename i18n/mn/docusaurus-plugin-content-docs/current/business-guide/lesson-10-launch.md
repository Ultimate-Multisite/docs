# Хичээл 10: Нээлтийн өдөр

Бүх зүйл бэлэн боллоо. Хаалгаа нээхээс өмнө энэ хичээлээр нээлтийн өмнөх шалгах жагсаалтыг үзэж, юу ч эвдэрсэн, дутуу, эсвэл ичмээр алдаатай биш гэдгийг баталгаажуулна.

## Бид хаана зогссон бэ

FitSite-д загварууд, төлөвлөгөөнүүд, төлбөр төлөх урсгал, брэндчилэл, хэрэглэгчийн эхлүүлэх үйл явц, үнэ тогтоолт бүгд тохируулагдсан. Одоо бүх зүйл ажиллаж байгааг шалгаад нийтэд нээнэ.

## Нээлтийн өмнөх шалгах жагсаалт

Зүйл бүрийг нэг бүрчлэн шалга. Юуг ч алгасаж болохгүй.

### Платформын дэд бүтэц

- [ ] Хостинг тогтвортой бөгөөд ачаалалтай үед сайн ажиллаж байна
- [ ] Wildcard SSL идэвхтэй бөгөөд бүх дэд домэйн HTTPS-ээр үйлчилж байна
- [ ] Домэйн холболт ажиллаж байна -- сайт үүсгээд custom domain холбож турш
- [ ] Нөөцлөлт тохируулагдсан бөгөөд туршигдсан (баталгаажуулахын тулд дор хаяж нэгийг сэргээ)
- [ ] Хяналт тохируулагдсан -- платформ унтарвал чи мэднэ

### Загварууд

- [ ] Гурван загвар бүгд шинэ сайтууд дээр зөв ачаалж байна
- [ ] Түр агуулга хэрэгтэй бөгөөд үсгийн алдаагүй байна
- [ ] Бүх зураг зохих лицензтэй байна (stock photo усан тэмдэггүй)
- [ ] Загварын хуудас бүр дээр гар утасны дэлгэцэд нийцтэй байдал ажиллаж байна
- [ ] Хуудас ачаалах хурд зөвшөөрөхүйц байна (GTmetrix эсвэл PageSpeed Insights зэрэг хэрэгслээр турш)
- [ ] Аль ч загвар дээр эвдэрсэн холбоос эсвэл дутуу asset байхгүй

### Төлөвлөгөөнүүд ба бүтээгдэхүүнүүд

- [ ] Гурван төлөвлөгөө бүгд идэвхтэй бөгөөд харагдаж байна
- [ ] Төлөвлөгөөний тайлбарууд үнэн зөв бөгөөд салбарын онцлогт нийцсэн байна
- [ ] Үнэ зөв байна (сарын болон жилийн)
- [ ] Эхлүүлэх төлбөрүүд зөв төлөвлөгөөнүүд дээр тохируулагдсан байна
- [ ] Туршилтын хугацаа Starter төлөвлөгөөн дээр ажиллаж байна
- [ ] Захиалгын нэмэлт саналууд төлбөр төлөх явцад зөв гарч байна
- [ ] Plugin болон theme-ийн хязгаарлалтууд төлөвлөгөө бүрээр зөв хэрэгжиж байна

### Төлбөр төлөх урсгал

- [ ] Төлөвлөгөө бүр дээр бүрэн туршилтын бүртгэл хий (туршилтын төлбөрийн горим ашигла)
- [ ] Загвар сонголт нь төлөвлөгөө бүрт зөв загваруудыг харуулж байна
- [ ] Төлбөр амжилттай боловсруулагдаж байна
- [ ] Хэрэглэгч бүртгүүлсний дараа тавтай морилно уу имэйл хүлээн авч байна
- [ ] Шинэ сайт зөв загвартайгаар үүсэж байна
- [ ] Хэрэглэгч шинэ сайтдаа шууд нэвтэрч чадна
- [ ] Хөнгөлөлтийн кодууд зөв ажиллаж байна

### Брэндчилэл

- [ ] Нэвтрэх хуудас FitSite брэндчиллийг харуулж байна
- [ ] Admin dashboard FitSite брэндчиллийг харуулж байна
- [ ] Бүх системийн имэйлүүд FitSite брэндчилэл болон фитнесийн онцлогтой хэллэг ашиглаж байна
- [ ] Нэхэмжлэхүүд танай бизнесийн мэдээлэлтэй зөв харагдаж байна
- [ ] Маркетингийн сайт нийтэд нээгдсэн бөгөөд төлбөр төлөх маягт руу холбоостой байна

### Хэрэглэгчийг эхлүүлэх үйл явц

- [ ] Quick Start виджет шинэ хэрэглэгчийн Dashboard дээр гарч байна
- [ ] Бүх Quick Start холбоосууд зөв хуудсууд руу зааж байна
- [ ] Тавтай морилно уу имэйлийн дараалал тохируулагдсан бөгөөд туршигдсан байна
- [ ] Мэдлэгийн сангийн нийтлэлүүд нийтлэгдсэн бөгөөд хандах боломжтой байна
- [ ] Account хуудас зөв төлөвлөгөөний мэдээлэл болон upgrade сонголтуудыг харуулж байна

### Хууль эрх зүй ба бизнес

- [ ] Үйлчилгээний нөхцөл нийтлэгдсэн бөгөөд төлбөр төлөх хэсгээс холбоостой байна
- [ ] Нууцлалын бодлого нийтлэгдсэн бөгөөд хандах боломжтой байна
- [ ] Буцаан олголтын бодлого тодорхойлогдсон бөгөөд баримтжуулсан байна
- [ ] Төлбөр хүлээн авахад зориулсан бизнесийн байгууллага бүртгэгдсэн байна
- [ ] Төлбөрийн гарц live mode-д байна (test mode биш)
- [ ] Татварын тохиргоо танай харьяаллын дагуу зөв байна

## Зөөлөн нээлт ба бүрэн нээлт

Хоёр үе шаттай нээлтийг бодолц:

### 1-р үе шат: Зөөлөн нээлт

Нийтийн нээлтээс өмнө 5-10 фитнес студийн эзнийг бүртгүүлэхээр урь. Эдгээр нь таны бета хэрэглэгчид. Дараах зүйлсийн хариуд тэдэнд томоохон хөнгөлөлт (насан туршдаа 50% хямдрал, эсвэл 3 сар үнэгүй) санал болго:

- Бүртгэл болон эхлүүлэх туршлагын талаар шударга санал хүсэлт
- Тэдний сайтыг жишээ үзүүлэн болгон ашиглах зөвшөөрөл
- Тааралдсан аливаа алдаа эсвэл асуудлыг мэдээлэх

Ингэснээр та нийтэд нээхээсээ өмнө бодит хэрэглэгчийн санал хүсэлт болон харуулах боломжтой амьд сайтуудтай болно.

### 2-р үе шат: Нийтийн нээлт

Зөөлөн нээлтийн санал хүсэлтийг тусгасны дараа:

- Төлбөрийн гарцыг live mode руу шилжүүл
- Маркетингийн сайтаа нийтэл
- Хэрэглэгч татах ажлыг эхлүүл (Хичээл 11)
- Холбогдох фитнесийн салбарын сувгуудаар зарла

## Нээлтийн өдрийн үйлдлүүд

Нийтэд нээх өдөр:

1. **Live төлбөр рүү шилжүүл** -- төлбөрийн гарц дээрээ test mode-г идэвхгүй болго
2. **Дахин нэг удаа баталгаажуул** -- бодит төлбөрөөр бүрэн туршилтын бүртгэл хий (дараа нь өөртөө буцаан олголт хий)
3. **Анхааралтай хяна** -- алдаа, бүтэлгүйтсэн бүртгэл, эсвэл төлбөрийн асуудлыг ажигла
4. **Бэлэн бай** -- таны анхны бодит хэрэглэгчдэд тусламж хэрэг болж магадгүй бөгөөд хурдан хариу итгэлийг бий болгодог
5. **Товчхон тэмдэглэ** -- дараа нь ажилдаа буцаж ор

## Юу буруугаар эргэж болох вэ

Дараах зүйлсэд бэлэн бай:

- **Төлбөрийн гарцын асуудлууд**: Гарцын үйлчилгээ үзүүлэгчийн дэмжлэгийн холбоо барих мэдээллийг бэлэн байлга
- **SSL сертификатын асуудлууд**: Сертификатыг хэрхэн шалгаж, шинэчлэхээ мэддэг бай
- **Имэйл хүргэлтийн алдаа**: Имэйлүүд үнэхээр ирж байгаа эсэхийг турш (spam хавтсуудыг шалга)
- **Ачаалалтай үеийн гүйцэтгэл**: Хэрэв урсгал огцом нэмэгдвэл хостингоо хэрхэн өргөтгөхөө мэддэг бай
- **Хэрэглэгчийн төөрөгдөл**: Мэдлэгийн сан болон дэмжлэгийн сувгуудаа бэлэн байлга

- **Нээлтийн өмнөх иж бүрэн шалгах жагсаалт** дэд бүтэц, агуулга, төлбөр, хууль эрх зүйг хамарсан
