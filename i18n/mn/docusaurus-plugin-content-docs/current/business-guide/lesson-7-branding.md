# Хичээл 7: Өөрийн болгох {#lesson-7-making-it-yours}

Таны хэрэглэгчид хэзээ ч "ямар нэг WordPress нэмэлт" ашиглаж байгаа мэт мэдрэх ёсгүй. Тэд өөрсдийн салбарт зориулан бүтээгдсэн платформ болох FitSite-ийг ашиглаж байна гэж мэдрэх ёстой. Энэ хичээлд брэндлэлт, white-label хийх, мөн платформыг бүтээгдэхүүн мэт мэдрүүлэх талаар өгүүлнэ.

## Бид хаана зогссон бэ {#where-we-left-off}

FitSite нь фитнес студийн эзэмшигчдийг төлөвлөгөө сонголтоос амьд сайт хүртэл хүргэдэг ажилладаг төлбөр төлөх урсгалтай болсон. Одоо бид бүх туршлагыг нэгэн цогц, брэндлэгдсэн бүтээгдэхүүн мэт мэдрүүлнэ.

## Таны платформын домэйн {#your-platform-domain}

Таны брэндийн суурь бол таны домэйн юм. FitSite-ийн хувьд:

- **Үндсэн домэйн**: `fitsite.com` (таны маркетингийн сайт болон сүлжээний үндэс)
- **Хэрэглэгчийн сайтууд**: `studioname.fitsite.com` (дэд домэйнүүд)
- **Тусгай домэйнүүд**: Growth болон Pro төлөвлөгөөтэй хэрэглэгчид өөрсдийн домэйныг холбож болно

### Домэйнээ тохируулах {#setting-up-your-domain}

1. Платформын домэйнээ бүртгүүлнэ
2. Үүнийг хостингийн үйлчилгээ үзүүлэгч рүүгээ чиглүүлнэ
3. Хэрэглэгчийн дэд домэйнүүдэд зориулж wildcard DNS (`*.fitsite.com`) тохируулна
4. Wildcard SSL идэвхтэй эсэхийг шалгана

Дэлгэрэнгүй зааврыг [Домэйн холболтыг хэрхэн тохируулах вэ](/user-guide/domain-mapping/how-to-configure-domain-mapping)-ээс үзнэ үү.

## Админ туршлагыг white-label хийх {#white-labeling-the-admin-experience}

Фитнес студийн эзэмшигч өөрийн сайтын Dashboard-д нэвтрэхэд WordPress эсвэл Ultimate Multisite брэндлэлт бус, таны брэнд харагдах ёстой.

### Тусгай нэвтрэх хуудас {#custom-login-page}

WordPress нэвтрэх хуудсыг дараах зүйлсийг харуулахаар өөрчилнө:

- Таны FitSite лого
- Фитнест тохирсон арын зураглал
- Таны брэндийн өнгөнүүд

### Dashboard брэндлэлт {#dashboard-branding}

[Admin Page Creator](/addons/admin-page-creator) addon эсвэл тусгай CSS ашиглан:

- WordPress логог өөрийн FitSite логогоор солино
- Админы өнгөний схемийг өөрийн брэндтэй нийцүүлэн өөрчилнө
- Фитнест зориулсан хурдан холбоосууд болон тусламжийн нөөцүүдтэй тусгай dashboard widget нэмнэ

### Тусгай админ хуудсууд {#custom-admin-pages}

Фитнес студийн эзэмшигчдэд хамгийн хамааралтай үйлдлүүдийг ил гаргадаг тусгай админ хуудсууд үүсгэхийг бодолцоно уу:

- "Хичээлийн хуваариа засах"
- "Дасгалжуулагчдын профайлыг шинэчлэх"
- "Сайтаа үзэх"

Ингэснээр стандарт WordPress цэсэнд нуухын оронд тухайн салбарт хамааралтай үйлдлүүдийг голд нь байрлуулж, сурах муруйг багасгана.

## Харилцаа холбоогоо брэндлэх {#branding-your-communications}

Имэйл, нэхэмжлэл, мэдэгдэл бүр таны брэндийг бататгах ёстой.

### Системийн имэйлүүд {#system-emails}

**Ultimate Multisite > Settings > Emails** рүү очиж бүх системийн имэйлүүдийг өөрчилнө:

- **Илгээгчийн нэр**: FitSite
- **Илгээгчийн имэйл**: hello@fitsite.com
- **Имэйл загварууд**: Өөрийн брэндийн өнгөнүүд болон логог ашиглана
- **Хэл**: Бүхэлдээ фитнест чиглэсэн байна

Өөрчлөх гол имэйлүүд:

| Имэйл | Ерөнхий хувилбар | FitSite хувилбар |
|-------|----------------|-----------------|
| Тавтай морил | "Таны шинэ сайт бэлэн боллоо" | "Таны фитнес студийн вэбсайт ажиллаж эхэллээ" |
| Төлбөрийн баримт | "Төлбөр хүлээн авлаа" | "FitSite захиалгын төлбөр баталгаажлаа" |
| Туршилт дуусах | "Таны туршилт удахгүй дуусна" | "Таны FitSite туршилт 3 хоногийн дараа дуусна -- студийн вэбсайтаа ажиллуулсан хэвээр байлгаарай" |

### Нэхэмжлэлүүд {#invoices}

Нэхэмжлэлийн загваруудыг дараах зүйлстэйгээр өөрчилнө:

- Таны FitSite лого болон брэндийн өнгөнүүд
- Таны бизнесийн дэлгэрэнгүй мэдээлэл
- Фитнест чиглэсэн бүтээгдэхүүний нэрс (ерөнхий төлөвлөгөөний ID биш)

## Хэрэглэгчдэд харагдах сайт {#the-customer-facing-site}

Таны үндсэн домэйн (`fitsite.com`) платформыг борлуулах маркетингийн сайттай байх хэрэгтэй. Энэ нь Ultimate Multisite сүлжээний админаас тусдаа -- таны бизнесийн олон нийтэд харагдах нүүр царай юм.

Гол хуудсууд:

- **Нүүр хуудас**: Фитнес бизнесүүдэд зориулсан тодорхой үнэ цэнийн санал
- **Онцлогууд**: FitSite юу хийдэгийг фитнесийн хэллэгээр
- **Үнэ**: Салбарт тохирсон онцлогуудын харьцуулалттай таны гурван төлөвлөгөө
- **Жишээнүүд**: Платформ дээр бүтээгдсэн сайтуудыг үзүүлэх
- **Бүртгүүлэх**: Таны төлбөр төлөх маягт руу холбосон холбоосууд

:::tip Таны маркетингийн сайт сүлжээний сайт байж болно
Маркетингийн сайтаа өөрийн сүлжээн доторх сайт болгон үүсгэнэ үү. Ингэснээр та үүнийг ижил Dashboard-оос удирдаж, өөрийн платформын боломжуудыг харуулж чадна.
:::

## Хэрэглэгчдэд зориулсан тусгай домэйн {#custom-domain-for-customers}

Тусгай домэйн багтсан төлөвлөгөөтэй хэрэглэгчдэд зориулж үйл явцыг тодорхой баримтжуулна:

1. Хэрэглэгч домэйнээ бүртгүүлэх эсвэл registrar руу шилжүүлнэ
2. Хэрэглэгч DNS-ээ таны платформ руу чиглүүлэхээр шинэчилнэ (яг шаардлагатай бичлэгүүдийг өгнө)
3. Ultimate Multisite домэйн холболт болон SSL-ийг зохицуулна

Энэ үйл явцад тусгайлан зориулсан тусламжийн нийтлэл эсвэл мэдлэгийн сангийн бичлэгийг техникийн бус фитнес студийн эзэмшигчдэд зориулан бичиж үүсгэнэ үү.

## Одоогийн байдлаарх FitSite сүлжээ {#the-fitsite-network-so-far}

```
FitSite сүлжээ
├── WordPress Multisite (дэд домэйн горим)
├── Ultimate Multisite (тохируулсан + брэндлэсэн)
├── Платформын домэйн (fitsite.com + wildcard SSL)
├── Сайтын загварууд (Studio Essential, Gym Pro, Fitness Chain)
├── Бүтээгдэхүүнүүд (Starter, Growth, Pro + Order Bumps)
├── Төлбөр төлөх урсгал (салбарт тохирсон, туршсан)
├── Брэндлэлт
│   ├── Тусгай нэвтрэх хуудас
│   ├── Брэндлэсэн админ Dashboard
│   ├── Салбарт тохирсон системийн имэйлүүд
│   ├── Брэндлэсэн нэхэмжлэлүүд
│   └── fitsite.com дээрх маркетингийн сайт
└── Onboarding урсгалд бэлэн (дараагийн хичээл)
```

## Энэ хичээлээр бид юу бүтээв {#what-we-built-this-lesson}

- **Платформын домэйн ба DNS**-ийг брэндлэгдсэн туршлагад зориулан тохируулсан
- FitSite брэндлэлт бүхэлдээ шингэсэн **white-labeled админ**
- **Өөрчилсөн харилцаа холбоо** -- имэйлүүд, нэхэмжлэлүүд, мэдэгдлүүд бүгд брэндтэй нийцсэн
- FitSite-ийг фитнес студийн эзэмшигчдэд борлуулдаг **маркетингийн сайт**
- Өөрийн домэйн хүссэн хэрэглэгчдэд зориулсан **тусгай домэйны баримтжуулалт**
