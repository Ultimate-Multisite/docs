# Ultimate Multisite баримт бичиг

Ultimate Multisite баримт бичигт тавтай морилно уу. Ultimate Multisite нь WordPress Multisite-ыг бүрэн хэмжээний Үйлчилгээ хэлбэрийн вэбсайт (WaaS) платформ болгон хувиргадаг.

## Баримт бичгийн хэсгүүд

### [Хэрэглэгчийн гарын авлага](/user-guide/getting-started/ultimate-multisite-101)

WaaS платформоо тохируулах, тохиргоо хийх, удирдахад хэрэгтэй бүх зүйл. Суулгалт, төлбөрийн гарцууд, домэйн холболт, харилцагчийн удирдлага болон бусад зүйлийг хамарна.

### [Хөгжүүлэгчийн баримт бичиг](/developer/hooks)

Ultimate Multisite дээр хөгжүүлэлт хийж буй хөгжүүлэгчдэд зориулсан техникийн лавлах. Hooks лавлах, REST API баримт бичиг болон addon хөгжүүлэлтийн гарын авлагуудыг багтаана.

### [Addons](/addons)

Ultimate Multisite-ыг WooCommerce интеграц, аналитик, домэйн борлуулалт болон бусад нэмэлт боломжуудаар өргөтгөх төлбөртэй addon plugin-уудын баримт бичиг.
