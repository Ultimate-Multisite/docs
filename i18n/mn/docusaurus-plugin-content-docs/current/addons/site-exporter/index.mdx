# Сайт экспортлогч {#site-exporter}

## Агуулга {#content}

Таны хэрэглэгч вэбсайтаа шилжүүлэхээр шийдвэл сүлжээн доторх дэд сайтуудыг нэг сайт суулгац руу, мөн эсрэг чиглэлд шилжүүлэх хэрэгцээ гардаг гэдгийг бид ойлгодог. Сайт экспортлогч ба импортлогч нэмэлт нь энэ ажлыг илүү хялбар болгодог. Энэ хэрэгсэл нь ZIP файлууд ашиглан сайтуудыг экспортолж, импортолдог бөгөөд тэдгээр нь мөн нөөц хуулбар болж чадна.

## Онцлогууд {#features}

* Сайтуудыг экспортлох ба импортлох

* Ашиглахад хялбар интерфэйс

* Сайтын нөөц хуулбар үүсгэх

## Баримтжуулалт {#documentation}
**Сайтыг хэрхэн экспортолж, импортлох вэ**

## Суулгалт {#installation}

1. **ultimate-multisite-site-exporter**-ийг **/wp-content/plugins/** лавлах руу байршуулна уу
2. WordPress доторх **Нэмэлтүүд** цэсээр нэмэлтийг идэвхжүүлнэ үү
3. Нэмэлтийг тохируулахын тулд алхам алхмаар Wizard-ыг дагана уу

## Ашиглах заавар {#usage-guide}

Ultimate Multisite ашиглан та сүлжээнээсээ сайтыг экспортлох эсвэл multisite эсвэл нэг сайт суулгац руу сайт импортлох боломжтой.

Үйл явцыг эхлүүлэхийн тулд манай нэмэлтүүдийн хуудас руу очиж Ultimate Multisite: Сайт экспортлогч ба импортлогч нэмэлтийг суулгана уу.![Нэмэлтүүдийн хуудас](/img/admin/addons-page.png)

![Суулгах сонголттой нэмэлтүүдийн хуудас](/img/admin/addons-page.png)

Plugins хуудас руугаа очоод сая суулгасан нэмэлтийг **Сүлжээнд идэвхжүүлэх** дээр дарж идэвхжүүлнэ үү.<!-- Дэлгэцийн зураг боломжгүй: WP Plugins хуудас дээр Site Exporter нэмэлтийн Network Activate товчийг харуулж байна -->

**Экспорт ба импорт** сонголтыг зүүн талын Ultimate Multisite цэснээс олж болно.

<!-- Дэлгэцийн зураг боломжгүй: Ultimate Multisite цэс Export and Import сонголтыг харуулж байна -->

## Сайтыг хэрхэн экспортлох вэ {#how-to-export-a-site}

Сүлжээнээсээ сайт экспортлохын тулд Ultimate Multisite цэсний **Экспорт ба импорт** руу очоод **Сайтуудыг экспортлох** таб дээр **Шинэ сайтын экспорт үүсгэх** дээр дарна уу.

<!-- Дэлгэцийн зураг боломжгүй: Generate new Site Export товчтой Export Sites таб -->

Экспортлохыг хүссэн сайтаа хайж, экспортлогдох файлд оруулах зүйлсээ сонгоно уу. Та themes, plugins болон uploads оруулж болно. Хэрэв таны сайт хэт том бол экспортын үйл явцыг арын горимд ажиллуулахыг сонгож болно.<!-- Дэлгэцийн зураг боломжгүй: сайтын хайлт, themes/plugins/uploads оруулах checkbox-уудтай сайтын экспортын сонголтууд -->

**Сайт экспортлох** товчийг дарна уу. Таны экспортлогдсон сайт **Одоо байгаа экспортууд** жагсаалтад харагдана. Файлыг татахын тулд **ZIP татах** товчийг дарна уу. Экспортын ZIP-үүд нь өөрөө ачаалагддаг `index.php` болон `readme.txt` агуулдаг тул тусдаа импортлогч нэмэлтийг урьдчилан суулгахгүйгээр шинэ host дээр суулгаж болно.<!-- Дэлгэцийн зураг боломжгүй: Download ZIP товчтой Existing Exports жагсаалт -->

## Бүх сүлжээг хэрхэн экспортлох вэ {#how-to-export-the-whole-network}

Бүх дэд сайтуудыг хамтад нь экспортлохын тулд Ultimate Multisite цэсний **Экспорт ба импорт** руу очоод экспортын дэлгэц дээрх **Сүлжээний экспорт** сонголтыг ашиглана уу. Үүссэн архив нь host шилжилт, staging дахин байгуулалт эсвэл сүлжээний түвшний нөөц хуулбарт зориулан бүх дэд сайтуудыг нэг ZIP дотор агуулна.

Сүлжээний экспортууд том хэмжээтэй байж болно. Тэдгээрийг ачаалал бага үед ажиллуулж, архив болон оруулсан uploads, plugins, themes-д сервер хангалттай хадгалах зайтай эсэхийг баталгаажуулна уу.

## Сайтыг хэрхэн импортлох вэ {#how-to-import-a-site}

Сайт импортлохын тулд **Сайт импортлох** таб руу очоод импортлохыг хүссэн сайтын .zip файлыг байршуулна уу. Та URL-ийг шинэ URL болгон өөрчилж болно, эсвэл зайг хоосон үлдээснээр хуучин URL-ийг хадгалж болно. Мөн импортын дараа .zip файлыг суулгацаасаа устгахын тулд хайрцгийг чагталж болно. Дараа нь **Импорт эхлүүлэх** дээр дарна уу.

<!-- Дэлгэцийн зураг боломжгүй: zip байршуулах талбар болон Begin Import товчтой Import a Site таб -->

Үйл явц бага зэрэг хугацаа авч магадгүй. Та импортлогчийн доорх жагсаалтад хүлээгдэж буй импортыг харах болно.

<!-- Дэлгэцийн зураг боломжгүй: импортлогчийн доорх жагсаалт дахь хүлээгдэж буй импорт -->

Та **Импорт цуцлах** дээр дарж үйл явцыг зогсоож болно.

<!-- Дэлгэцийн зураг боломжгүй: импортын үйл явцыг зогсоох Cancel Import товч -->

Үйл явц дууссаны дараа таны сайт сүлжээндээ аль хэдийн боломжтой болсон байна.

## Нэг суулгац дээр сайтыг хэрхэн импортлох вэ {#how-to-import-a-site-on-a-single-install}

Сайтыг нэг суулгац руугаа импортлохын тулд эхлээд импортлогчийг сайт дээрээ суулгаж идэвхжүүлнэ үү. Үүнийг татах холбоосыг экспортлогчийн таб дээрээс олж болно.

<!-- Дэлгэцийн зураг боломжгүй: нэг сайт импортлогчийг татах холбоостой Exporter таб -->

Та импортлогчийг зүүн талын **Хэрэгслүүд** цэсний **Экспорт ба импорт** дэд зүйлээс олж болно.

<!-- Дэлгэцийн зураг боломжгүй: Export and Import дэд цэсийг харуулсан нэг сайтны Tools цэс -->

Импортлохыг хүссэн сайтын .zip файлыг байршуулна уу. Та URL-ийг шинэ URL болгон өөрчилж болно, эсвэл зайг хоосон үлдээснээр хуучин URL-ийг хадгалж болно. Мөн импортын дараа .zip файлыг суулгацаасаа устгахын тулд хайрцгийг чагталж болно. Дараа нь **Импорт эхлүүлэх** дээр дарна уу.

<!-- Дэлгэцийн зураг боломжгүй: zip байршуулах болон Begin Import товчтой нэг сайтны импортын маягт -->

Үйл явц бага зэрэг хугацаа авч магадгүй. Та импортлогчийн доорх жагсаалтад хүлээгдэж буй импортыг харах болно.

<!-- Дэлгэцийн зураг боломжгүй: жагсаалт дахь нэг сайтны хүлээгдэж буй импорт -->

Та **Импорт цуцлах** дээр дарж үйл явцыг зогсоож болно.

<!-- Дэлгэцийн зураг боломжгүй: импортын үйл явцыг зогсоох Cancel Import товч -->

Үйл явц дууссаны дараа таны сайт сая импортолсон сайтаар солигдоно.
