# Сайт экспортлогчийн өөрчлөлтийн бүртгэл

Хувилбар: 1.0.6 - 2026-05-11-нд гарсан
* Шинэ: Сайтын экспортод одоо өөрөө ачаалдаг index.php багтдаг тул ZIP-ийг тусдаа plugin суулгалгүйгээр шинэ host дээр суулгаж болно
* Шинэ: Сүлжээний экспорт нь администраторуудад Site Export админ хуудаснаас бүх дэд сайтуудыг нэг архивт экспортлох боломж олгоно

Хувилбар: 1.0.5 - 2026-05-05-нд гарсан
* Засвар: get_config зэрэг mock хийгдээгүй аргуудыг боловсруулахын тулд WP_CLI polyfill stub-д __callStatic нэмсэнээр test bootstrap алдаанаас сэргийлэв

Хувилбар: 1.0.4 - 2025-11-25-нд гарсан

* Засвар: Ultimate Multisite 2.4-тэй нийцтэй болгосон.

Хувилбар: 1.0.2 - 2025-09-28-нд гарсан

* Угтварыг ultimate-multisite болгон өөрчилсөн; text domain-ийг шинэчилсэн; хувилбарыг нэмэгдүүлсэн.
