# Multi-Currency {#multi-currency}

Ultimate Multisite: Multi-Currency нь таны сүлжээнд олон валютаар төлбөр хүлээн авах боломж олгоно. Энэ нь бодит цагийн эсвэл гарын авлагын ханшаар үнийг автоматаар хөрвүүлж, аль ч валютын хувьд бүтээгдэхүүн бүрийн яг үнийг тохируулах боломж олгон, зочин бүрийн илүүд үзэх валютыг газарзүйн байршлаар илрүүлдэг — бүгдийг нэг админ Dashboard-оос удирдана.

## Гол боломжууд {#key-features}

- **Автомат валютын ханшийн хөрвүүлэлт** — таны сонгосон ханшийн нийлүүлэгчээс татсан ханшаар үнийг суурь валютаас хөрвүүлнэ
- **Бүтээгдэхүүн бүрийн үнийн давхар тохиргоо** — автомат хөрвүүлэлтийг алгасахын тулд аливаа бүтээгдэхүүн дээр валют тус бүрийн яг үнийг тохируулна
- **Бүртгэлийн валют сонгогч талбар** — checkout/бүртгэлийн үед хэрэглэгчдэд илүүд үзэх валютаа сонгох боломж олгоно
- **Олон валютын ханшийн нийлүүлэгч** — ECB, Frankfurter, Stripe, эсвэл бүрэн гарын авлагын ханш
- **Газарзүйн байршилд суурилсан валют илрүүлэлт** — зочны улстай тохирох валютыг автоматаар урьдчилан сонгоно
- **Админ Dashboard** — идэвхтэй валютууд, валютын ханш, нийлүүлэгчийн тохиргоог нэг газраас удирдана

## Шаардлагууд {#requirements}

- Ultimate Multisite 2.0.0 эсвэл түүнээс дээш
- WordPress Multisite суулгац
- PHP 7.4 эсвэл түүнээс дээш

## Суулгалт {#installation}

1. Plugin файлуудыг `/wp-content/plugins/ultimate-multisite-multi-currency` руу байршуулна
2. WordPress дахь **Plugins** дэлгэцээр plugin-ийг идэвхжүүлнэ (Network Activate)
3. Админ Dashboard-ийг нээхийн тулд **Ultimate Multisite → Multi-Currency** руу очно
4. Хүлээн авахыг хүссэн валютуудаа нэмээд валютын ханшийн нийлүүлэгчээ тохируулна

## Тохиргоо {#configuration}

### Админ Dashboard {#admin-dashboard}

Multi-Currency админ Dashboard (**Ultimate Multisite → Multi-Currency**) нь дараахыг хийх төв газар юм:

- Валют нэмэх, идэвхжүүлэх эсвэл идэвхгүй болгох
- Валютын ханшийн нийлүүлэгч сонгож, шаардлагатай үед ханшийг шинэчлэх
- Бүх идэвхтэй валютын одоогийн ханшийг шалгах
- Автомат нийлүүлэгчид тохиромжгүй үед гарын авлагын ханш тохируулах

### Валютын ханшийн нийлүүлэгчид {#exchange-rate-providers}

| Нийлүүлэгч | Тайлбар |
|---|---|
| **ECB** | European Central Bank — EUR-д суурилсан сүлжээнүүдэд зориулсан өдөр тутмын үнэгүй ханш |
| **Frankfurter** | Нээлттэй эхийн ECB mirror API — үнэгүй, API key шаардлагагүй |
| **Stripe** | Stripe-ийн бодит цагийн FX ханшийг ашиглана — идэвхтэй Stripe интеграц шаардлагатай |
| **Manual** | Та ханшийг шууд оруулна; гадаад API дуудлага байхгүй |

Нийлүүлэгчээ **Ultimate Multisite → Multi-Currency → Settings → Exchange Rate Provider** хэсгээс сонгоно. Ханшийг хүссэн үедээ гараар шинэчлэх эсвэл автоматаар шинэчлэгдэхээр хуваарьлаж болно.

### Бүтээгдэхүүн бүрийн үнийн давхар тохиргоо {#per-product-pricing-overrides}

Автомат хөрвүүлэлт хангалттай нарийвчлалгүй үед та аливаа бүтээгдэхүүн дээр тодорхой валютын яг үнийг тохируулж болно:

1. Бүтээгдэхүүний засварлагчийг нээнэ (**Ultimate Multisite → Products → Edit**)
2. **Currency Pricing** хэсэг рүү гүйлгэнэ
3. Давхар тохиргоо хийхийг хүссэн валют бүрийн яг үнийг оруулна
4. Бүтээгдэхүүнийг хадгална

Давхар тохиргоогүй бүтээгдэхүүнүүд таны суурь валютаас хөрвүүлсэн үнийг ашиглана.

### Бүртгэлийн валют сонгогч талбар {#currency-selector-signup-field}

Валют сонгогч талбарыг таны бүртгэл/checkout урсгалд нэмснээр хэрэглэгчид бүртгүүлэх үедээ илүүд үзэх валютаа сонгох боломжтой:

1. **Ultimate Multisite → Multi-Currency → Settings** руу очно
2. **Show currency selector on signup**-ийг идэвхжүүлнэ
3. Сонголтоор анхдагч валют тохируулах эсвэл газарзүйн байршлаар урьдчилан сонгосон утгыг тодорхойлуулж болно

Сонгосон валютыг subscription-тэй хамт хадгалж, дараагийн бүх нэхэмжлэлд ашиглана.

### Газарзүйн байршилд суурилсан валют илрүүлэлт {#geolocation-based-currency-detection}

Газарзүйн байршлын илрүүлэлт идэвхтэй үед addon нь зочны улсыг валюттай харгалзуулж, валют сонгогч талбарт урьдчилан сонгоно. Идэвхжүүлэхийн тулд:

1. **Ultimate Multisite → Multi-Currency → Settings** руу очно
2. **Detect currency from visitor location**-ийг идэвхжүүлнэ
3. **Geolocation Mapping** хүснэгтэд улс орнуудыг валютуудтай харгалзуулна

Хэрэв зочны улсад тохирох харгалзуулалт байхгүй бол сүлжээний суурь валютыг нөөц сонголт болгон ашиглана.

## Түгээмэл асуултууд {#frequently-asked-questions}

### Валютын ханшийн нийлүүлэгч боломжгүй бол юу болох вэ? {#what-happens-if-an-exchange-rate-provider-is-unavailable}

Addon нь хамгийн сүүлд амжилттай татсан ханш руу буцаж ашиглана. Хэрэв ханш огт татагдаж байгаагүй бол таны тохируулсан гарын авлагын ханш руу буцна. Аюулгүй байдлын нөөц болгон гарын авлагын ханш тохируулахыг зөвлөж байна.

### Би өөр өөр валютуудад өөр өөр нийлүүлэгч ашиглаж болох уу? {#can-i-use-different-providers-for-different-currencies}

Үгүй — бүх валютын хувьд нэг удаад нэг нийлүүлэгч идэвхтэй байна. Тус бүрийн ханшийг бүрэн хянах шаардлагатай бол **Manual**-ийг ашиглана уу.

### Газарзүйн байршилд тусдаа plugin шаардлагатай юу? {#does-geolocation-require-a-separate-plugin}

Гадаад газарзүйн байршлын plugin шаардлагагүй. Addon нь WordPress-ийн суурилуулсан IP-ээс-улс илрүүлэлтийг ашиглана. Илүү өндөр нарийвчлал шаардлагатай бол зориулалтын газарзүйн байршлын үйлчилгээг өгөгдсөн filter hooks-оор нэгтгэж болно.

### Бүтээгдэхүүн бүрийн давхар тохиргоо татварын өмнө эсвэл дараа хэрэгжих үү? {#are-per-product-overrides-applied-before-or-after-tax}

Бүтээгдэхүүн бүрийн давхар тохиргоо нь зорилтот валютаар татварын өмнөх үнийг тохируулна. Татварыг таны одоогийн татварын тохиргоогоор давхар тохируулсан үнэ дээр нэмж тооцно.
