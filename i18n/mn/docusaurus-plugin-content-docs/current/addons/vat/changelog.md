# VAT өөрчлөлтийн бүртгэл {#vat-changelog}

Хувилбар 1.0.7 - 2026-02-03-нд гарсан

* Засвар: German (DE) VAT дугаарууд VIES SOAP үйлчилгээний MS_MAX_CONCURRENT_REQ алдаанаас болж баталгаажуулалтад тэнцэхгүй байсан. Илүү найдвартай баталгаажуулалтын тулд хуучин SOAP API-гаас албан ёсны EU VIES REST API руу шилжүүлсэн.
* Засвар: VIES үйлчилгээ түр хугацаанд боломжгүй үед VAT дугааруудыг буруу татгалзахын оронд одоо хүлээн авдаг болсон. Дараа хянахад зориулж бүртгэлийн бичлэг үүсгэнэ.
* Засвар: VAT/Tax ID талбар одоо зөвхөн EU гишүүдэд бус бүх улсад харагддаг болсон. EU бус татварын ID-уудыг (жишээ нь Swiss CHE дугаарууд) VIES баталгаажуулалтгүйгээр нэхэмжлэл дээр харуулахын тулд хадгална.
* Засвар: Greece (GR), Monaco (MC), болон Isle of Man (IM)-ийн VAT угтвар хайлт буруу массив хайх логикоос болж эвдэрсэн байсан.
* Сайжруулалт: Site Exporter нэмэлттэй хамт ашиглахад ноцтой алдаа үүсгэж болзошгүй шаардлагагүй wp-cli-bundle хөгжүүлэлтийн хамаарлыг устгасан.
* Сайжруулалт: Иж бүрэн unit test багц нэмсэн (53 тест).

Хувилбар 1.0.6 - 2026-01-25-нд гарсан

* Засвар: Дотоодын B2B гүйлгээнд одоо VAT зөв ногдуулдаг болсон. EU VAT дүрмийн дагуу reverse charge нь зөвхөн хил дамнасан B2B гүйлгээнд үйлчилнэ, харин хэрэглэгчийн улс компанийн улстай ижил үед үйлчлэхгүй.

Хувилбар 1.0.5 - 2026-01-22-нд гарсан

* Засвар: Addon ачаалахгүй эсвэл зөв ажиллахгүй байсан.
* Өөрчлөлт: Илүү найдвартай, идэвхтэй арчилдаг өгөгдлийн тулд VAT хувь хэмжээний өгөгдлийн эх сурвалжийг euvatrates.com-оос ibericode/vat-rates repository руу шилжүүлсэн.
* Засвар: super_reduced_rates сонголтын утгын үсгийн алдааг зассан.
* Сайжруулалт: Алдаа гарахаас сэргийлэхийн тулд улсын код боловсруулахад null шалгалт нэмсэн.
* Өөрчлөлт: Traduttore-оор дамжуулан автомат шинэчлэлт хийхийн тулд багцалсан орчуулгын файлуудыг устгасан.
