# VAT

Multisite Ultimate: Европын VAT

Европын VAT-ыг илүү сайн дэмжихийн тулд Multisite Ultimate-ийн өгөгдмөл татварын боловсруулалтад нэмэлт боломжууд болон хэрэгслүүд нэмнэ.
EU VAT-ын хувь хэмжээнүүдийг Татварын хувь хэмжээ хуудсанд автоматаар нэмдэг

## Суулгалт {#installation}

1. 'ultimate-multisite-vat'-ыг '/wp-content/plugins/' санд байршуулна уу
2. WordPress дахь 'Плагинууд' цэсээр дамжуулан plugin-ийг идэвхжүүлнэ үү
3. Ultimate Multisite > Тохиргоо > Татварууд руу орж EU VAT-ыг идэвхжүүлнэ үү.
4. Татварын хувь хэмжээ дээр дарж, доод хэсэгт EU VAT-ын хувь хэмжээнүүдийг шинэчлээд Татварын хувь хэмжээнүүдийг хадгална уу.
5. Дууслаа!
