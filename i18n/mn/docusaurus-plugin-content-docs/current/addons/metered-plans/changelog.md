# Metered Plans өөрчлөлтийн бүртгэл

Хувилбар 1.1.0 - 2026-05-05-нд гарсан
- Шинэ: Олон site-ийн дэд site-уудад зориулсан AI token billing — харилцагчийн site-ууд даяар AI token хэрэглээг хянаж, тохируулж болох per-token тарифаар төлбөр тооцно
- Шинэ: Connector enforcement-ийг dynamic limit discovery болон write-through-тэй дахин бүтээж, бүх connector даяар бодит цагийн нарийвчлалыг хангана
- Засвар: Database table шинэчлэлтүүд одоо BerlinDB schema тодорхойлолтуудтай зөв нийцэж, шинэ суулгалтууд дээр шинэчлэлтийн алдаа гарахаас сэргийлнэ
- Засвар: Database upgrade callbacks зөв формат руу хөрвүүлэгдэж, чимээгүй шинэчлэлтийн алдаануудыг шийднэ
- Засвар: AI Usage Overage Markup оролтын талбарт бутархай утгуудыг одоо зөвшөөрнө
- Засвар: plugin ачаалах үед гарч байсан ноцтой алдаанууд болон давхар эхлүүлэлтийн асуудлуудыг шийдсэн
- Сайжруулсан: check-env npm script нэмсэн тул developer орчин анхны ажиллуулах үед өөрөө тохируулагдана
