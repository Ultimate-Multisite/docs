# Агуулга синк өөрчлөлтийн бүртгэл {#content-sync-changelog}

### Хувилбар 1.1.0 - 2025-12-31-нд гарсан {#version-110---released-on-2025-12-31}
* Шинэ: Бүх нийтлэлийн төрлүүдийг синк хийх дэмжлэг (нийтлэлүүд, хуудсууд, захиалгат нийтлэлийн төрлүүд)
* Шинэ: Агуулга сонгох алхам дахь нийтлэлийн төрлийн шүүлтүүр
* Шинэ: Эх сайтын сонголтын алхам - зөвхөн үндсэн сайтаас бус, дурын сайтаас синк хийх
* Шинэ: Загвар сайтуудад зориулсан дэд сайтын урьдчилсан сонголт
* Шинэ: Бүх нийтлэлийн төрлүүдэд зориулан post_content-оос медиа задлах
* Шинэ: Таксономи синк хийх (ангиллууд, шошгууд, захиалгат таксономиуд)
* Өөрчлөгдсөн: "Elementor Template Sync"-ээс "Content Sync" болгон нэрийг өөрчилсөн
* Өөрчлөгдсөн: "Select Templates" алхмыг "Select Data" болгон нэрийг өөрчилсөн
* Сайжруулсан: Elementor-той холбоотой код одоо зөвхөн elementor_library нийтлэлийн төрөлд ажиллана

### Хувилбар 1.0.0 - 2025-12-19-нд гарсан {#version-100---released-on-2025-12-19}
* Анхны хувилбар
* Загвар болон сайт сонгох олон алхамт шидтэн
* Бодит цагийн явцын санал хүсэлттэй багц боловсруулалт
* ID-г автоматаар дахин тааруулдаг бүрэн медиа синк
* Ultimate Multisite-тэй тохиргооны интеграц
* Сүлжээний тохиргоогоор дамжуулан сайтыг хасах дэмжлэг
* Синк хийсний дараа Elementor кэшийг цэвэрлэх
* Загварын таксономи хадгалалт
