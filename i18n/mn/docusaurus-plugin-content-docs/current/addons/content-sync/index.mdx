# Агуулгын синхрончлол {#content-sync}

Multisite Ultimate: Агуулгын синхрончлол

Multisite Ultimate-д зориулсан энэ нэмэлт нь сүлжээний администраторуудад сүлжээн дэх аль ч сайтаас бусад сайт руу агуулга синхрончлох боломж олгодог. Энэ нь нийтлэл, хуудас, захиалгат нийтлэлийн төрөл, Elementor загвар зэрэг бүх нийтлэлийн төрлийг дэмждэг. Нэмэлт нь агуулга болон зорилтот сайтуудыг сонгох олон алхамт шидтэний интерфэйсийг багтаасан бөгөөд багц боловсруулалт болон бодит цагийн явцын санал хүсэлттэй.

## Боломжууд {#features}

* Агуулга сонгох: Жижиг зураг болон нийтлэлийн төрлөөр шүүх боломжтой визуал картын сүлжээ
* Бүх нийтлэлийн төрөл: Нийтлэл, хуудас, бүтээгдэхүүн, Elementor загвар болон аливаа захиалгат нийтлэлийн төрлийг синхрончилно
* Сайт сонгох: Агуулгыг аль сайтууд руу синхрончлохыг сонгоно, эх сайтыг хасах болон дэд сайтуудыг урьдчилан сонгох боломжтой
* Багц боловсруулалт: Нэг удаад 10 сайт руу синхрончилж, явцын шинэчлэлт харуулна
* Медиа синхрончлол: Нийтлэлийн агуулгаас медиа файлуудыг (зураг, видео) зорилтот сайтууд руу автоматаар хуулж, дахин онооно
* Таксономи синхрончлол: Ангилал, шошго болон захиалгат таксономийг хадгална
* Elementor дэмжлэг: CSS дахин үүсгэх болон загварын ID дахин оноох боломжтой Elementor загварын бүрэн дэмжлэг
* Зөрчил шийдвэрлэх: Одоо байгаа агуулгыг солих эсвэл алгасах сонголт
* Кэш удирдлага: Синхрончлолын дараа зорилтот сайтууд дээрх Elementor кэшийг автоматаар цэвэрлэнэ

## Шаардлагууд {#requirements}

* WordPress 5.3 эсвэл түүнээс дээш
* PHP 7.4 эсвэл түүнээс дээш
* Multisite Ultimate plugin (идэвхтэй)
* Elementor plugin (сонголттой, Elementor загвар синхрончлоход шаардлагатай)

## Суулгалт {#installation}

1. 'ultimate-multisite-content-sync'-ийг '/wp-content/plugins/' лавлах руу байршуулна
2. WordPress Network Admin дахь 'Plugins' цэсээр plugin-ийг идэвхжүүлнэ
3. Ultimate Multisite → Тохиргоо → Агуулгын синхрончлол хэсэгт тохиргоог хийнэ
4. Синхрончлол эхлүүлэхийн тулд Ultimate Multisite → Агуулгын синхрончлол руу очно
