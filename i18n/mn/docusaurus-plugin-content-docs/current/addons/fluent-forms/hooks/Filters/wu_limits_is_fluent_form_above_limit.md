# Шүүлтүүр: wu_limits_is_fluent_form_above_limit {#filter-wulimitsisfluentformabovelimit}

Тухайн маягтын төрөл энэ төлөвлөгөөнд зөвшөөрөгдсөн эсэхийг шалгана. Plugin хөгжүүлэгчдэд буцаах утгыг шүүх боломж олгоно

## Параметрүүд {#parameters}

| Нэр | Төрөл | Тайлбар |
|------|------|-------------|
| $value | `bool` | Маягтын төрөл хязгаараас давсан эсэх |
| $form_type | `string` | Шалгаж буй маягтын төрөл |
| $form_count | `int` | Одоогийн маягтын тоо |
| $quota | `int` | Зөвшөөрөгдсөн квот |
