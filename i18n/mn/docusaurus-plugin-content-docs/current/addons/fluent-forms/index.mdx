# Fluent Forms {#fluent-forms}

Энэ нэмэлт нь Multisite Ultimate дээрх бүтээгдэхүүн, гишүүнчлэл эсвэл сайтуудад Fluent Forms-д хязгаар нэмэх боломж олгодог.
Fluent Forms нь custom post type-уудын оронд захиалгат хүснэгтүүд ашигладаг тул ердийн хязгаарлалтын сонголтууд Fluent Forms-т ажиллахгүй.

## Онцлогууд {#features}

- Fluent Forms эсвэл Conversational Fluent Forms-д хязгаар нэмэх
- Хэрэглэгчид хязгаарт хүрэх үед төлөвлөгөөгөө сайжруулахыг сануулсан найрсаг мессеж авна
