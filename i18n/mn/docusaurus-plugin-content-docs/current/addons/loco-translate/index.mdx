# Loco Translate

Ultimate Multisite: Loco Translate Integration нэмэлт нь таны WordPress олон сайттай сүлжээнд сайт тус бүрийн орчуулгын боломжийг авчирна. Хэрэв таны сүлжээний өөр өөр сайтуудад ижил плагин эсвэл сэдвийн өөр өөр орчуулга хэрэгтэй болж байсан бол энэ нэмэлт төгс шийдэл юм.

**Энэ нэмэлт юу хийдэг вэ**

Энэ нэмэлт нь Ultimate Multisite-г түгээмэл Loco Translate плагинтай саадгүй нэгтгэж, таны сүлжээний дэд сайт бүр өөрийн тусгай орчуулгыг хадгалах боломжтой болгоно. Та олон бүс нутгийн платформ ажиллуулж байгаа, өөр өөр хэлний нийгэмлэгүүдэд үйлчилж байгаа, эсвэл сүлжээнийхээ хэмжээнд орчуулгын уян хатан байдал хэрэгтэй байгаа эсэхээс үл хамааран энэ нэмэлт үүнийг маш хялбар болгоно.

**Дараахад төгс тохирно:**

* Өөр өөр нутагшуулалтын хэрэгцээтэй олон бүс нутгийн бизнесүүд
* Олон төрлийн нийгэмлэгүүдэд үйлчилдэг боловсролын байгууллагууд
* Олон хэлний дэмжлэг санал болгодог SaaS платформууд
* Олон үйлчлүүлэгчийн сайтыг удирддаг веб агентлагууд
* Орчуулгын уян хатан байдал шаарддаг аливаа олон сайттай сүлжээ

## Суулгалт {#installation}

1. Ultimate Multisite суулгагдсан бөгөөд сүлжээнд идэвхжүүлсэн эсэхийг шалгана уу
2. Loco Translate плагинийг суулгаж идэвхжүүлнэ үү
3. Энэ нэмэлтийг plugins лавлахдаа байршуулна уу
4. Loco Translate Integration нэмэлтийг сүлжээнд идэвхжүүлнэ үү
5. Аль ч дэд сайт дээр Loco Translate-р дамжуулан сайт тус бүрийн орчуулга үүсгэж эхлээрэй

Ингээд боллоо! Нэгтгэл нь нэмэлт тохиргоо шаардахгүйгээр автоматаар ажиллана.

## Түгээмэл асуултууд {#frequently-asked-questions}

## Суулгасны дараа ямар нэг зүйл тохируулах шаардлагатай юу? {#do-i-need-to-configure-anything-after-installation}

Үгүй! Нэгтгэл автомат. Нэмэлт болон Loco Translate хоёулаа идэвхтэй болмогц дэд сайтууд өөрсдийн тусгай орчуулгыг шууд үүсгэж эхлэх боломжтой.

## Хэрэв дэд сайт тусгай орчуулга үүсгэхгүй бол юу болох вэ? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

Сайт нь сүлжээний түвшний орчуулгыг автоматаар нөөц хувилбар болгон ашиглана. Юу ч эвдрэхгүй бөгөөд бүх зүйл саадгүй ажиллана.

## Дэд сайтууд сүлжээний түвшний орчуулгад хандаж чадах уу? {#can-subsites-access-network-wide-translations}

Энгийн сайтын администраторууд зөвхөн өөрийн сайтын орчуулгыг харж, засварлана. Супер администраторууд аль сайтыг үзэж байгаагаасаа хамааран сүлжээний түвшний болон сайт тус бүрийн орчуулгад хоёуланд нь хандах боломжтой.

## Энэ нь плагин болон сэдэв хоёуланд нь ажиллах уу? {#does-this-work-with-both-plugins-and-themes}

Тийм! Сайт тус бүрийн орчуулга нь WordPress плагин болон сэдэв хоёуланд нь ажиллана.

## Энэ миний олон сайттай сүлжээг удаашруулах уу? {#will-this-slow-down-my-multisite-network}

Үгүй. Нэмэлт нь зөвхөн шаардлагатай үед сайт тус бүрийн орчуулгыг шалгадаг үр ашигтай файл ачаалалтыг ашигладаг. Гүйцэтгэлд мэдэгдэхүйц нөлөө байхгүй.

## Сүлжээний орчуулгыг эхлэх цэг болгон тодорхой сайт руу хуулж болох уу? {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

Тийм! Loco Translate дээр орчуулга засварлах үед та сүлжээний орчуулгаар хялбархан эхлээд, сайт бүрт зориулан өөрчлөх боломжтой.

## Энэ WooCommerce болон бусад плагинтай ажиллах уу? {#does-this-work-with-woocommerce-and-other-plugins}

Мэдээж! Стандарт WordPress орчуулгын функцуудыг ашигладаг аливаа плагин эсвэл сэдэв сайт тус бүрийн орчуулгатай ажиллана.
