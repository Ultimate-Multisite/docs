# WooCommerce интеграцийн өөрчлөлтийн бүртгэл {#woocommerce-integration-changelog}

Хувилбар 2.2.0 - 2026-07-01-нд гарсан
* Шинэ: Ultimate Multisite татварын дүнгүүд одоо төлбөр төлөхөөс өмнө татварын нийлбэрийг илүү тодорхой болгохын тулд checkout дээр тусдаа WooCommerce шимтгэлийн мөрүүдээр харагдана.
* Шинэ: WooCommerce Subscriptions-ийн дахин оролдох хугацаанд амжилтгүй сунгалтын түдгэлзүүлэлт хийхийг хүссэн сайтуудад зориулж сонголтоор идэвхжүүлэх "Амжилтгүй сунгалтад гишүүнчлэлийг шууд түдгэлзүүлэх" тохиргоо болон `wu_woo_suspend_on_payment_failure` filter нэмсэн.
* Засвар: Амжилтгүй эсвэл сэргэсэн сунгалтын дараа Ultimate Multisite гишүүнчлэлүүдтэй синхронгүй үлдэж болзошгүй WooCommerce Subscription төлөвүүдийг тохируулсан.
* Засвар: WooCommerce дэлгүүрийн валют байхгүй байсан үед Ultimate Multisite валютын жагсаалтад нэмсэн.
* Засвар: Захиалагчдыг WooCommerce checkout руу чиглүүлэх үед хэрэглэгчийн нэхэмжлэхийн мэдээллийг хадгалсан.
* Сайжруулсан: Jetpack Autoloader 5-тай нийцтэй болгосон.
* Сайжруулсан: GitHub болон marketplace zip-үүд давхар staging сангууд болон хөгжүүлэлтийн файлуудаас зайлсхийхээр хувилбарын багц үүсгэлтийг цэвэрлэсэн.

Хувилбар 2.0.6 - 2026-01-16-нд гарсан
* Сайжруулалт: Үндсэн захиалгуудыг addon-д оруулсан. Одоо Woocommerce Subscriptinos extension шаардлагагүй болсон.

Хувилбар 2.0.5 - 2026-01-09-нд гарсан

* Засвар: Хэрэглэгч үндсэн сайтын гишүүн байх үед хязгааргүй redirect болдог байсан.

Хувилбар 2.0.4 - 2025-11-14-нд гарсан

* Өөрчилсөн: Ultimate Multisite: Woocommerce Integration гэж нэрийг нь өөрчилсөн.
* Нэмсэн: Woocommerce 10.2.1-тэй нийцтэй болгосон.
* Нэмсэн: Woocommerce Subscriptions 7.7.0-тэй нийцтэй болгосон.

Хувилбар 2.0.1 - 2023-08-09-нд гарсан

* Нэмсэн: Woocommerce 7.9.0-тэй нийцтэй болгосон.

* Нэмсэн: Ultimate Multisite Woocommerce бүтээгдэхүүнүүдийг meta value-аар таних боломж.
* Нэмсэн: Ultimate Multisite-тэй холбоотой бүх Woocommerce бүтээгдэхүүнийг тэмдэглэх нэг удаагийн засварыг оруулсан.
* Нэмсэн: Ultimate Multisite-ийн үүсгэсэн бүтээгдэхүүнүүдийг Woocommerce жагсаалтаас хассан.
* Сайжруулалт: Сагсанд хэрэглэх давтагдахгүй Woocommerce хөнгөлөлт үүсгэсэн.
* Сайжруулалт: Давтагдах хөнгөлөлтийг Woocommerce бүтээгдэхүүнд сэргээсэн.
* Сайжруулалт: Woocommerce бүтээгдэхүүнд давтагдах хөнгөлөлтийн шошго нэмсэн.
* Сайжруулалт: Checkout дээр бүтээгдэхүүний төрлийг баталгаажуулсан.
* Засвар: Доошлуулах явцад гишүүнчлэлийн төлөвийг хадгалсан.
* Засвар: Цуцлах явцад алдаа гарахаас зайлсхийхийн тулд захиалга байгаа эсэхийг шалгасан.
* Засвар: Woocommerce subscriptions-д ашиглах захиалгын эхлэх огноог нэмсэн.
* Дотоод: Шинэ PHP 8.1 build процесс хэрэгжүүлсэн.
