# Material WP {#material-wp}

WordPress удирдлагын интерфейс олон жилийн турш ижил хэвээр байсан. Харин удирдлагын интерфейсийн Material Design хувилбар ямар харагдах бол гэж та бодож үзсэн үү? 

Material WP нь танд тэрхүү харагдах байдал, мэдрэмжийг өгөхөөр хөгжүүлсэн удирдлагын загвар бөгөөд үйлчлүүлэгчдэдээ илүү өөрчилж тохируулсан Dashboard өгөхөд тань тусална.

## Онцлогууд {#features}

- Тусгай CSS дэмжлэг
- Хялбар загварчлалын тохиргоо
- Цэсийн дарааллыг өөрчлөх
- Хялбар импорт/экспорт

## Баримтжуулалт {#documentation}

**Material WP ашиглан Admin Panel-аа өөрчлөх**

## Суулгалт {#installation}

1. 'material-wp'-ийг '/wp-content/plugins/' директорт байршуулна уу
2. WordPress дахь 'Нэмэлтүүд' цэсээр дамжуулан нэмэлтийг идэвхжүүлнэ үү
