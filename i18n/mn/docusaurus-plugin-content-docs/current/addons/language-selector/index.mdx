# Хэл сонгогч {#language-selector}

Multisite Ultimate: Бүртгүүлэх үед хэл сонгогч

Бүртгүүлэх урсгалд хэл сонгох талбар нэмнэ.

## Суулгалт {#installation}

1. 'ultimate-multisite-language-selector'-ийг '/wp-content/plugins/' санд байршуулна уу
2. WordPress сүлжээний админы 'Plugins' цэсээр дамжуулан нэмэлтийг идэвхжүүлнэ үү

## Ашиглалтын гарын авлага {#usage-guide}

Энэ үнэгүй нэмэлтээр та хэрэглэгчдэдээ бүртгүүлэх урсгал болон нэвтрэх хуудсан дээр ямар хэлийг харахыг хүсэж байгаагаа сонгох боломж олгоно.

[Лицензээ идэвхжүүлсний](../../user-guide/getting-started/installing-ultimate-multisite.md) дараа таны лицензээр хандах эрхтэй Ultimate Multisite нэмэлтүүдийг суулгах боломжтой болно.

Ultimate Multisite тохиргооны хуудас руугаа ороод, зүүн талын цэснээс _Нэмэлтүүд_ шошгыг дарна уу.

![Нэмэлтүүдийн хуудас](/img/admin/addons-page.png)

_Language Selector суулгаж эхлэхийн тулд Нэмэлтүүдийн хуудас руу очно уу_

Тэнд та бидний санал болгодог бүх нэмэлтийг олно.

Үнэгүй нэмэлтүүд нь бүртгүүлсэн захиалгын төлөвлөгөөнөөс үл хамааран манай бүх хэрэглэгчдэд боломжтой.

Нэмэлтийн жижиг зургийн доорх _Одоо суулгах!_ товчийг дарна уу. Дараа нь Plugins хуудсан дээр идэвхжүүлнэ үү.

![Суулгах товчтой нэмэлтүүдийн хуудас](/img/admin/addons-page.png)

_Нэмэлтийг суулгахын тулд товчийг дарна уу_

<!-- Дэлгэцийн зураг байхгүй: Language Selector нэмэлтэд зориулсан Сүлжээнд идэвхжүүлэх товчийг харуулсан WordPress Plugins хуудас -->

_Үүнийг Plugins хуудсан дээрээ идэвхжүүлнэ үү_

Одоо Ultimate Multisite-ийн Тохиргооны цэс рүү ороод _Сүлжээний тохиргоо_ табыг нээнэ үү. Тэнд, хуудасны доод хэсэгт _Хэлнүүд_ нэртэй шинэ хэсгийг олно.

![Тохиргооны хуудас](/img/config/settings-general.png)

_Цоо шинэ хэсэг гарч ирнэ!_

Бүх хэлийг сонголт болгон харуулахын тулд хоосон орхиж болно, эсвэл аль хэлийг боломжтой болгохоо сонгож болно. Хэлнүүд нь таны WordPress тохиргоонд суулгасан хэлнүүд байна.

<!-- Дэлгэцийн зураг байхгүй: WordPress суулгалт дээр боломжтой хэлнүүдийг харуулсан хэл сонгох унадаг цэс -->

_Энэ WordPress суулгалт дээр боломжтой хэлнүүдийн жагсаалт_

Хийсэн өөрчлөлтөө хадгалсны дараа таны сонгосон хэлнүүдтэй сонголтын цэс Нэвтрэх хуудсан дээр болон Бүртгүүлэх урсгал дээр харагдана.

![Нэвтрэх болон бүртгэлийн тохиргоо](/img/config/settings-login-registration.png)

_Нэвтрэх хуудсан дээрх хэл сонгогч_

![Бүртгэлийн хуудас](/img/config/registration-page.png)

_Бүртгүүлэх урсгал дээрх хэл сонгогч_

## URL-аар хэлийг тохируулах {#setting-the-language-via-the-url}

Та мөн URL-д locale параметрийг дамжуулан хуудасны хэлийг өөрчилж болно. Жишээлбэл, Итали хэл дээрх бүртгэлийн хуудас руу чиглүүлэхийг хүсвэл URL-д ?locale=it_IT нэмнэ үү.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Үүнийг ажиллуулахын тулд тухайн хэл таны WordPress сүлжээнд суулгасан байх ёстой.
