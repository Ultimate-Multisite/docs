# Filter: signup_get_available_languages {#filter-signupgetavailablelanguages}

Урд талын сайтын бүртгэлд ашиглах боломжтой хэлнүүдийн жагсаалтыг шүүнэ.

Энэ hook-д хоосон массив дамжуулбал бүртгэлийн маягт дээрх тохиргооны гаралтыг идэвхгүй болгож, сайт үүсгэх үед анхдагч хэлийг ашиглана. Аль хэдийн суулгаагүй хэлнүүд хасагдана.

## Параметрүүд {#parameters}

| Нэр | Төрөл | Тайлбар |
|------|------|-------------|
| $available_languages | `array` | Ашиглах боломжтой хэлнүүд. |
