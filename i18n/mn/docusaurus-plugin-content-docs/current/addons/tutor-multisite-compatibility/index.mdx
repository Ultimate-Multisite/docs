# Tutor Multisite Compatibility {#tutor-multisite-compatibility}

Олон сайттай орчинд Tutor LMS-д нийцтэй байдлын засварууд нэмдэг WordPress plugin.

## Тайлбар {#description}

Энэ plugin нь WordPress олон сайттай сүлжээнд Tutor LMS ажиллуулахад шаардлагатай нийцтэй байдлын чухал засваруудыг өгдөг. Энэ нь хэрэглэгчийн элсэлт, багшийн удирдлага, курсын хязгаарлалт болон олон сайттай орчинд тусгайлсан бусад ажиллагааг зохицуулна.

## Онцлогууд {#features}

- **Өгөгдлийн сангийн хүснэгтийн удирдлага**: Олон сайттай орчны гадаад түлхүүрийн хязгаарлалтаас болж үүсэж чадаагүй, дутуу Tutor LMS хүснэгтүүдийг автоматаар илрүүлж үүсгэнэ
- **Олон сайттай хэрэглэгчийн элсэлт**: Сүлжээний сайтуудын хооронд хэрэглэгчийн элсэлтийг удирдах тусгай AJAX боловсруулагчид
- **Багшийн дүрийн удирдлага**: Олон сайттай орчинд багшийн дүрийг автоматаар оноож, удирдана
- **Курсын хязгаарлалтууд**: Сайтын plan-д үндэслэн курс болон funnel үүсгэх хязгаарыг удирдахад Ultimate Multisite-тай нэгтгэнэ
- **Тусгай админ харагдацууд**: Олон сайттай орчныг мэддэг багшийн жагсаалт болон удирдлагын интерфейс
- **Админ хэрэгслүүд**: Сүлжээ болон сайтын админуудад зориулсан өгөгдлийн сангийн удирдлагын интерфейс

## Шаардлагууд {#requirements}

- WordPress 5.0+
- PHP 7.4+
- Tutor LMS plugin (`Requires Plugins` header-ээр автоматаар шаардана)
- WordPress Multisite суулгац
- Ultimate Multisite plugin (курсын хязгаарлалтын ажиллагаанд)

## Суулгалт {#installation}

1. Plugin файлуудыг `/wp-content/plugins/tutor-multisite-compatibility/` руу байршуулна
2. Tutor LMS суулгагдаж, идэвхжсэн эсэхийг шалгана
3. WordPress дахь 'Plugins' цэсээр plugin-ийг идэвхжүүлнэ
4. Plugin нь Tutor LMS-тэй автоматаар нэгтгэгдэнэ

## MU Plugin-ээс шилжүүлсэн ажиллагаа {#functionality-moved-from-mu-plugin}

Энэ plugin нь өмнө нь `wp-content/mu-plugins/daves.php` дотор байсан ажиллагааг агуулна:

- Курсын төлөв өөрчлөх хязгаарлалтууд
- Курс үүсгэх хязгаарлалтууд
- CartFlows нэгтгэл болон хязгаарууд
- Tutor багшийн дүрийн удирдлага
- Олон сайттай хэрэглэгчийн элсэлтийн боловсруулалт
- Курсын product ID харуулалт

## Өгөгдлийн сангийн удирдлага {#database-management}

### Асуудал {#the-problem}

WordPress олон сайттай орчинд хэрэглэгчдийн хүснэгт рүү гадаад түлхүүрийн хязгаарлалттай хүснэгтүүдийн хувьд Tutor LMS хүснэгт үүсгэлт амжилтгүй болдог. Энэ нь дараах шалтгаантай:

- Сайт тус бүрийн хүснэгтүүд `wp_2_tutor_carts` гэх мэт угтвар ашигладаг
- Гадаад түлхүүрүүд `wp_2_users`-ийг заадаг (энэ нь байхгүй)
- Бодит хэрэглэгчдийн хүснэгт нь `wp_users` (сүлжээ даяар хуваалцдаг)

### Шийдэл {#the-solution}

Энэ plugin автоматаар:

1. **Дутуу хүснэгтүүдийг илрүүлнэ**: Гадаад түлхүүрийн хязгаарлалтаас болж үүсэж чадаагүй хүснэгтүүдийг шалгана
2. **Зөв холбоос бүхий хүснэгтүүд үүсгэнэ**: Зөв хэрэглэгчийн хүснэгтийн нэрийг ашиглана (олон сайттай орчинд `wp_users`)
3. **Олон нөхцөл байдлыг зохицуулна**:
   - Plugin идэвхжүүлэх/дахин идэвхжүүлэх
   - Сүлжээнд шинэ сайт үүсгэх
   - Админ интерфейсээр гараар хүснэгт шалгах

### Нөлөөлөлд өртөх хүснэгтүүд {#affected-tables}

- `tutor_coupon_usages` - Хэрэглэгчдийн хүснэгт рүү гадаад түлхүүр агуулна
- `tutor_carts` - Хэрэглэгчдийн хүснэгт рүү гадаад түлхүүр агуулна
- `tutor_notification_preferences` - Хэрэглэгчдийн хүснэгт рүү гадаад түлхүүртэй TutorPro хүснэгт

### Админ интерфейс {#admin-interface}

Өгөгдлийн сангийн удирдлагын хэрэгслүүдэд дараах байршлаас хандана:

- **Сүлжээний админ**: Settings → Tutor Multisite DB
- **Сайтын админ**: Tools → Tutor Multisite DB

Интерфейс нь танд дараах боломжийг олгоно:
- Бүх сайтын дутуу хүснэгтүүдийг харах
- Хүснэгт үүсгэлтийг гараар эхлүүлэх
- Өгөгдлийн сангийн тохиргоог шалгах
- Дэлгэрэнгүй төлөвийн мэдээлэл харах
