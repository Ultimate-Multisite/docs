# Домэйн борлуулагч

**Multisite Ultimate: Domain Seller** нь таны WordPress Multisite сүлжээг хүчирхэг домэйн борлуулах платформ болгон хувиргана. Хэрэглэгчид бүртгүүлэх үедээ эсвэл Dashboard-оосоо шууд өөрийн домэйныг бүртгүүлэх боломжтой бөгөөд боломжтой эсэхийг бодит цагт шалгаж, үнийг ил тод харуулна.

### Яагаад Domain Seller гэж? {#why-domain-seller}

Домэйн бүртгэл нь аливаа WaaS (Website as a Service) платформын хамгийн их хүсэлттэй нэмэлт борлуулалтын нэг юм. Хэрэглэгчдээ гуравдагч талын бүртгэгч рүү илгээхийн оронд та домэйныг шууд санал болгож болно — ингэснээр хэрэглэгчдэд саадгүй туршлага өгөхийн зэрэгцээ сүлжээндээ илүү их орлого хадгална.

### Гол боломжууд {#key-features}

#### Олон домэйн бүртгэгчийн дэмжлэг {#multiple-domain-registrar-support}
* **OpenSRS** — Бүрэн бүртгэл, сунгалт, удирдлага
* **Namecheap** — WhoisGuard болон sandbox туршилттай бүрэн интеграц
* **HostAfrica** — Domains Reseller API-гаар дамжуулан домэйн хайлт, үнэ, бүртгэл, шилжүүлэг, DNS болон ID хамгаалалт
* **Openprovider** — Дахин борлуулагчийн үнэ, бүртгэл, сунгалт, шилжүүлэг, DNS, WHOIS нууцлал болон TLD синк
* **Hostinger** — Хуваалцсан Hostinger интеграцын token ашиглан домэйн хайлт ба бүртгэл
* **Өргөтгөх боломжтой архитектур** — Integration Registry хэв маяг дээр бүтээгдсэн тул илүү олон нийлүүлэгч нэмэхэд хялбар
* **Суурилуулсан холболтын туршилт** — Нийтэд нээхээс өмнө нийлүүлэгчийн итгэмжлэл болон холболтыг баталгаажуулна

#### Чиглүүлсэн тохиргооны шидтэн {#guided-setup-wizard}
* **Нэг товшилтоор эхлүүлэх** — Нийлүүлэгчийн тохиргоо, TLD импорт болон домэйн бүтээгдэхүүн үүсгэх алхмуудаар таныг хөтөлнө
* **Автомат TLD синк** — Холбогдсон нийлүүлэгчдээс боломжтой бүх TLD болон бөөний үнийг импортолно
* **Өгөгдмөл бүтээгдэхүүн үүсгэх** — Зохистой нэмэгдэл үнийн өгөгдмөл тохиргоотой, шууд ашиглахад бэлэн домэйн бүтээгдэхүүн үүсгэнэ

#### Уян хатан домэйн үнэ {#flexible-domain-pricing}
* **Гурван нэмэгдэл үнийн горим** — Хувийн нэмэгдэл, тогтмол нэмэгдэл эсвэл TLD бүрийн тогтмол үнэ
* **TLD-д тусгайлсан бүтээгдэхүүнүүд** — Өөр өөр TLD бүлгүүдэд (.com/.net, .io, улсын кодууд гэх мэт) тусдаа домэйн бүтээгдэхүүн үүсгэнэ
* **Танилцуулах үнэ** — Шинэ бүртгэлийг татахын тулд эхний жилийн хямдруулсан үнийг санал болгоно
* **Бодит цагийн үнэ** — Ухаалаг кэшлэлтэй нийлүүлэгчдийн шууд бөөний үнэ
* **Купоны дэмжлэг** — Multisite Ultimate хөнгөлөлтийн кодуудтай бүрэн нийцнэ

#### WHOIS нууцлалын хамгаалалт {#whois-privacy-protection}
* **Бүтээгдэхүүн бүрийн тохиргоо** — WHOIS нууцлалыг Идэвхгүй, Үргэлж багтсан эсвэл Хэрэглэгчийн сонголт гэж тохируулна
* **Хэрэглэгчийн зөвшөөрлөөр нэмэх** — Хэрэглэгчийн сонголт гэж тохируулсан үед checkout дээрх чагтлах хайрцаг нь хэрэглэгчдэд нууцлалын хамгаалалт нэмэх боломж олгоно
* **Тохируулж болох үнэ** — Нууцлалын add-on-д жилийн үнэ тохируулах эсвэл үнэгүй санал болгоно
* **Бүртгэгчийн интеграц** — Бүртгэлийн үед WhoisGuard (Namecheap) эсвэл WHOIS нууцлалыг (OpenSRS) автоматаар идэвхжүүлнэ

#### Нэгдсэн checkout туршлага {#unified-checkout-experience}
* **Гурван домэйн горим** — Хэрэглэгчид дэд домэйн (үнэгүй), шинэ домэйн бүртгүүлэх эсвэл өөрсдийн аль хэдийн эзэмшдэг байгаа домэйныг холбохоос сонгоно
* **Бодит цагийн домэйн хайлт** — Хуудас дахин ачаалахгүй AJAX-д суурилсан боломжтой эсэх шалгалт
* **Ухаалаг санал болголтууд** — Сонгосон домэйн эзлэгдсэн үед өөр TLD санал болгоно
* **Ил тод үнэ** — Бүртгэлийн үнэ, сунгалтын үнэ болон сонголтот WHOIS нууцлалын төлбөрийг тодорхой харуулна
* **Бүртгүүлэгчийн холбоо барих талбарууд** — Checkout явцад шаардлагатай бүртгүүлэгчийн мэдээллийг (нэр, хаяг, утас) шууд цуглуулна
* **Автоматаар үүсгэсэн сайтын URL** — Сайтын хаягийг сонгосон домэйнээс автоматаар гаргана

#### Автомат сунгалтын систем {#auto-renewal-system}
* **Membership-тэй холбоотой сунгалтууд** — Хэрэглэгчийн membership идэвхтэй байх хугацаанд домэйнууд автоматаар сунгагдана
* **Ухаалаг хуваарьлалт** — Сунгалтын ажлуудыг домэйны дуусах огноонд үндэслэн дараалалд оруулна
* **Дахин оролдох логик** — Амжилтгүй сунгалтын оролдлогод ухаалаг дахин оролдлогууд
* **Имэйл мэдэгдлүүд** — Удахгүй болох сунгалт, амжилттай сунгалт, алдаа болон хугацаа дуусалтын тохируулж болох анхааруулгууд

#### Хэрэглэгчийн DNS удирдлага {#customer-dns-management}
* **Өөрөө үйлчлэх DNS** — Хэрэглэгчид Dashboard-оосоо DNS бичлэгүүдийг (A, AAAA, CNAME, MX, TXT) нэмэх, засах, устгах боломжтой
* **Домэйны төлөв** — Бодит цагийн бүртгэлийн төлөв, дуусах огноо болон nameserver мэдээлэл
* **Автомат домэйн холболт** — Бүртгэгдсэн эсвэл байгаа домэйнуудыг хэрэглэгчийн сайттай автоматаар холбоно

#### Админы хэрэгслүүд {#admin-tools}
* **Гараар домэйн бүртгэх** — Админ самбараас хэрэглэгчдийн өмнөөс домэйн бүртгэнэ
* **Домэйн хяналт** — Бүх хэрэглэгчийн домэйн, төлөв болон дуусах огноог нэг хуудсаас харна
* **Нийлүүлэгч тус бүрийн лог** — Бүртгэгч бүрт тусдаа лог сувгууд (жишээ нь, domain-seller-namecheap.log)
* **Бүртгэлийн дэлгэрэнгүй** — Гол домэйн засах хуудсан дээр бүртгүүлэгчийн мэдээлэл болон бүртгэлийн өгөгдлийг харна

### Дэмжигдсэн нийлүүлэгчид {#supported-providers}

* **OpenSRS** — Бүртгэл, сунгалт, DNS удирдлага болон WHOIS нууцлалтай бүрэн интеграц
* **Namecheap** — Бүртгэл, сунгалт, багц боломжтой эсэх шалгалт болон үнэгүй WhoisGuard-тай бүрэн интеграц
* **HostAfrica** — Бүртгэл, сунгалт, шилжүүлэг, DNS удирдлага, EPP код авах, бүртгэгчийн түгжээ болон ID хамгаалалт
* **Openprovider** — Бүртгэл, сунгалт, шилжүүлэг, DNS удирдлага, EPP код авах, бүртгэгчийн түгжээ, WHOIS нууцлал болон дахин борлуулагчийн TLD синк
* **Hostinger** — Бүртгэл, домэйн хайлт, nameserver шинэчлэлт, бүртгэгчийн түгжээ болон WHOIS нууцлал; сунгалтуудыг Hostinger автомат сунгалтаар зохицуулна
* **GoDaddy** — Бүртгэл ба сунгалт
* **ResellerClub** — Бүртгэл, сунгалт болон DNS удирдлага
* **NameSilo** — Бүртгэл ба сунгалт
* **Enom** — Бүртгэл, сунгалт болон DNS удирдлага

### Техникийн дэлгэрэнгүй {#technical-details}

* **Шаардлагатай:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **Архитектур:** Domain_Selling_Capability interface бүхий Integration Registry хэв маяг
* **Автоматаар ачаалах:** PSR-4 class зураглалтай Jetpack autoloader
* **Стандартууд:** WordPress кодчиллын стандартууд, хатуу typing, prepared statements

## Суулгалт {#installation}

1. **Multisite Ultimate v2.4.12+** суусан бөгөөд сүлжээнд идэвхжүүлсэн эсэхийг шалгана уу
2. plugin-ийг `/wp-content/plugins/ultimate-multisite-domain-seller/` руу байршуулна уу
3. WordPress Network Admin дахь **Plugins**-оос plugin-ийг сүлжээнд идэвхжүүлнэ үү
4. **Тохируулах шидтэн** анхны идэвхжүүлэлтийн үед автоматаар ажиллана — дараах алхмуудыг дагана уу:
   * Өөрийн домэйн бүртгэгчийг холбоно
   * Холболтыг туршина
   * Боломжтой TLD-үүд болон бөөний үнийг импортолно
   * Үнийн нэмэгдэлтэй анхны домэйн бүтээгдэхүүнээ үүсгэнэ
5. **Ultimate Multisite > Тооцооны маягтууд** дахь тооцооны маягтдаа **Домэйн сонголт** талбарыг нэмнэ үү

## Түгээмэл асуултууд {#frequently-asked-questions}

### Multisite Ultimate-ийн ямар хувилбар шаардлагатай вэ? {#what-version-of-multisite-ultimate-is-required}

2.4.12 эсвэл түүнээс дээш хувилбар. addon нь WP Ultimo v1.x эсвэл 2.4.12-оос доош Ultimate Multisite хувилбаруудтай нийцэхгүй.

### Ямар домэйн үйлчилгээ үзүүлэгчдийг дэмждэг вэ? {#which-domain-providers-are-supported}

Есөн бүртгэгч дэмжигддэг: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo**, болон **Enom**. Боломжууд нь үйлчилгээ үзүүлэгчээс хамаарч өөр байдаг, учир нь бүртгэгчийн API-ууд өөр өөр үйлдлүүдийг ил гаргадаг; үйлчилгээ үзүүлэгч тус бүрийн тоймыг тохируулах гарын авлагаас үзнэ үү. addon-ийн Integration Registry архитектур нь өөрийн үйлчилгээ үзүүлэгчдийг нэмэхийг хялбар болгодог — [хөгжүүлэгчийн гарын авлага](/developer/integration-guide/custom-registrar)-ыг үзнэ үү.

### Би WHOIS нууцлалын хамгаалалт санал болгож болох уу? {#can-i-offer-whois-privacy-protection}

Тийм. Домэйн бүтээгдэхүүн бүр WHOIS Нууцлалын тохиргоотой бөгөөд гурван горимтой: Идэвхгүй, Үргэлж багтсан, эсвэл Үйлчлүүлэгчийн сонголт. Үйлчлүүлэгчийн сонголт гэж тохируулсан үед үйлчлүүлэгчид тооцооны явцад үнэ бүхий (эсвэл Namecheap-ийн WhoisGuard-д зориулсан "Үнэгүй") checkbox харна. Нууцлалын тэмдэглэгээ нь бүртгэлийн үед бүртгэгч рүү автоматаар дамжуулагдана.

### Би өөр өөр TLD-д өөр үнэ тохируулж болох уу? {#can-i-set-different-pricing-for-different-tlds}

Тийм. Өөр өөр TLD бүлгүүдэд зориулж тусдаа домэйн бүтээгдэхүүнүүд үүсгэнэ үү. Бүтээгдэхүүн бүр өөрийн нэмэгдлийн төрөл (хувь, тогтмол нэмэгдэл, эсвэл тогтмол үнэ), дэмжигдэх TLD-үүд болон танилцуулах үнэтэй байж болно.

### Үйлчлүүлэгчид тооцооны үед ямар сонголтуудыг хардаг вэ? {#what-checkout-options-do-customers-see}

Домэйн сонголт талбар нь үйлчлүүлэгчдэд гурван таб өгдөг: **Дэд домэйн** (таны сүлжээн дээр үнэгүй дэд домэйн ашиглах), **Шинэ домэйн бүртгүүлэх** (шинэ домэйн хайж бүртгүүлэх), болон **Одоо байгаа домэйн** (аль хэдийн эзэмшдэг домэйныг холбох). Зөвхөн таны идэвхжүүлсэн горимууд л харагдана.

### Автомат сунгалтыг дэмждэг үү? {#does-it-support-automatic-renewals}

Тийм. Домэйнуудыг автоматаар сунгахаар тохируулж болох бөгөөд үйлчлүүлэгчийн membership статустай холбогдоно. Сунгалтын төлбөрүүд нь үйлчлүүлэгчийн одоо байгаа төлбөрийн гарцаар боловсруулагдана.

### Үйлчлүүлэгчид DNS бичлэгүүдээ удирдаж чадах уу? {#can-customers-manage-their-dns-records}

Тийм. Үйлчлүүлэгчид өөрсдийн dashboard-д A, AAAA, CNAME, MX, болон TXT бичлэгүүдийг нэмэх, засах, устгах DNS удирдлагын интерфэйстэй болно. DNS удирдлага нь OpenSRS, ResellerClub, Enom, HostAfrica, болон Openprovider-той боломжтой. Байршуулсан домэйнуудад зориулсан Hostinger DNS-ийг үндсэн Hostinger домэйн-холболтын интеграц зохицуулдаг.

### Тохируулах шидтэн бий юу? {#is-there-a-setup-wizard}

Тийм. Тохируулах шидтэн нь анхны идэвхжүүлэлтийн үед автоматаар ажиллаж, бүртгэгч холбох, TLD-үүд импортлох, анхны домэйн бүтээгдэхүүнээ үүсгэх алхмуудаар таныг хөтөлнө. Мөн та addon тохиргооноос шидтэнийг дахин ажиллуулж болно.
