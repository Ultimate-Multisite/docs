# Домэйн борлуулагчийн өөрчлөлтийн бүртгэл

Хувилбар 1.3.0 - 2026-06-02-нд гарсан
- Шинэ: HostAfrica дахин борлуулагчийн үлдэгдэл хэт бага болох үед network-admin анхааруулга нэмсэн
- Шинэ: Шинээр бүртгэгдсэн домэйнүүдийг сүлжээний сайт руу автоматаар зураглахыг нэмсэн
- Засвар: Зөвхөн шинэ домэйн бүртгэх үед бүртгүүлэгчийн талбарын шаардлагуудыг хэрэгжүүлдэг болгосон
- Засвар: Үлдэгдэл хянах мэдэгдлүүдийг хааж болдог болгосон
- Засвар: WooCommerce бүртгүүлэгчийн төлбөрийн мэдээллүүд хадгалагдахыг баталгаажуулсан
- Засвар: Бүртгэлийн явцад бүртгүүлэгчийн холбоо барих шаардлагуудыг хэрэгжүүлсэн
- Засвар: Домэйн бүртгэлийн бүтээгдэхүүнүүдийг 0% нэмэгдэлтэй үүсгэхээс сэргийлсэн
- Засвар: Төлбөр хийх сессийн урсгалын турш домэйн сонголтууд болон үнийг хадгалсан
- Засвар: HostAfrica домэйний үнийн валютын харагдацыг сайжруулсан
- Засвар: WP-core query-var зөрчлөөс сэргийлэхийн тулд төлбөр хийх маягтын үйлдлийн төлөвийг сайжруулсан
- Сайжруулсан: Тохиргооны зааварчилгаанд HostAfrica дахин борлуулагчийн тохиргооны баримтжуулалтыг холбосон

Хувилбар 1.2.0 - 2026-05-25-нд гарсан
- Шинэ: HostAfrica-г төлбөр хийх, тохиргооны туслах, хайлт, TLD/үнэ, бүртгэл, сунгалт, шилжүүлэг, nameserver, DNS, EPP code, registrar lock болон ID хамгаалалтын дэмжлэгтэй домэйн борлуулах интеграци болгон нэмсэн
- Шинэ: Openprovider-г дахин борлуулагчийн үнэ, бүртгэл, сунгалт, шилжүүлэг, nameserver, DNS, EPP code, registrar lock, WHOIS нууцлал болон TLD синк дэмжлэгтэй домэйн борлуулах интеграци болгон нэмсэн
- Шинэ: Hostinger-г боломжтой эсэхийг шалгах, бүртгэл, nameserver шинэчлэлт, registrar lock болон WHOIS нууцлалд зориулан үндсэн интеграциас хуваалцсан Hostinger API token ашигладаг домэйн борлуулах интеграци болгон нэмсэн
- Сайжруулсан: Хөгжүүлэгчийн интеграцийн зааварт зориулсан домэйний амьдралын мөчлөгийн action болон filter docblock-ууд нэмсэн
- Сайжруулсан: Нэмэлтийн readme доторх plugin нийцтэй байдлын мета өгөгдлийг WordPress 7.0 болгож шинэчилсэн
- Сайжруулсан: Ирэх хувилбарын зохицуулалтад ашиглагдах төлөвлөлтийн загваруудыг шинэчилсэн
