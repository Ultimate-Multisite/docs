# Mailchimp {#mailchimp}

Multisite Ultimate: Mailchimp нэгтгэл

Өөрийн Multisite Ultimate сүлжээг Mailchimp-тэйгээ нэгтгэнэ үү. Та тодорхой төлөвлөгөөнүүдийг Mailchimp жагсаалтуудтай холбож, бүртгүүлэх үед захиалагчдыг тэдгээр жагсаалтад автоматаар нэмж болно.

## Суулгалт {#installation}

1. 'ultimate-multisite-mailchimp'-ийг '/wp-content/plugins/' лавлах руу байршуулна уу
2. WordPress Network Admin доторх 'Plugins' цэсээр дамжуулан plugin-ийг идэвхжүүлнэ үү

## Ашиглах заавар {#usage-guide}

Захиалагчдын имэйлийг таны MailChimp жагсаалтуудтай синк хийх нь манай Ultimate Multisite: MailChimp Integration нэмэлтээр үнэхээр хялбар. Энэ заавар нь бүх зүйлийг тохируулахад тань туслах зорилготой!

## API түлхүүр авах {#getting-an-api-key}

Боломжтой өөр өөр тохиргоонууд руу орохоос өмнө бидэнд MailChimp API түлхүүр хэрэгтэй. Энэ нь Ultimate Multisite-д MailChimp API-тай холбогдож жагсаалт, бүлгүүдийг авах, мөн хэрэглэгчид бүртгүүлэх үед имэйл хаягуудыг тэдгээр жагсаалт, бүлгүүдэд нэмэх боломж олгоно.

Өөрийн MailChimp API түлхүүрийг авахын тулд энэ зааврыг дагана уу: [API түлхүүрийн тухай | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

<!-- Screenshot unavailable: MailChimp API key settings in Ultimate Multisite Add-on Settings -->

_MailChimp API түлхүүрээ_ _**Ultimate Multisite Settings - > Add-on Settings -> MailChimp**_ _дээр нэмнэ үү_

## Нэгтгэлийн горим {#integration-mode}

MailChimp Integration нэмэлт нь хоёр “нэгтгэлийн горим” санал болгодог бөгөөд энэ нь имэйл хаягуудыг синк хийхэд нэмэлтийг ашиглах хоёр өөр арга байна гэсэн үг юм.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_Илүү уян хатан байдалд зориулсан хоёр нэгтгэлийн горим!_

### Горим 1: Олон жагсаалт {#mode-1-multiple-lists}

Анхдагч горим нь **Олон жагсаалт** ашигладаг. Энэ горим нь төлөвлөгөө бүрд олон MailChimp жагсаалт сонгох боломж олгоно. Хэрэв та хэрэглэгчдээ өөр өөр жагсаалтаар ангилдаг бол энэ нь хамгийн сайн сонголт юм.

Жишээлбэл, таны MailChimp Account дээр **Newsletter** , **Plan A** , болон **Plan B** гэсэн 3 өөр жагсаалт байна гэж бодъё. Та бүх хэрэглэгчийг төлөвлөгөөнөөс нь үл хамааран Newsletter жагсаалтад мөн оруулахыг хүсэж байна, ингэснээр тэд таны шинэчлэлтүүдийг алдахгүй.

Үүнийг хэрэгжүүлэхийн тулд Multiple Lists горимыг ашиглан **Newsletter** болон **Plan A** жагсаалтуудыг Plan A MailChimp Settings-д, мөн **Newsletter** болон **Plan B** жагсаалтуудыг Plan B MailChimp Settings-д нэмж болно.

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_Шинэ Account-ууд болон цуцлагдсан Account-уудад зориулсан анхдагч жагсаалтуудыг сонгоно уу_

### Горим 2: Нэг жагсаалт + Олон бүлэг {#mode-2-single-list--multiple-groups}

MailChimp нь таны жагсаалтыг **Бүлгүүд** гэж нэрлэгдэх хэрэглэгчдийн жижиг бүлгүүдэд хуваах боломж олгодог. Ингэснээр та илүү их хяналттай болж, MailChimp дээр нэг кампанит ажлын олон хуулбарыг үүсгэх шаардлагагүйгээр захиалагчдынхаа тодорхой дэд хэсэгт кампанит ажил илгээж чадна (ингэвэл тухайн кампанит ажлын үзүүлэлтүүдийг дараа нь шинжлэхэд хүндрэлтэй болдог).

Жишээлбэл, танд **My Network Subscribers,** нэртэй нэг жагсаалт байж, дотор нь **Plan A** болон **Plan B**-д зориулсан бүлгүүд байж болно.

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_“нэг жагсаалт + олон бүлэг” нэгтгэлийн горимыг ашиглаж байгаа бол анхдагч жагсаалт болон бүлгүүдийг сонгоно уу_

## Анхдагч жагсаалт ба бүлгүүд {#default-lists-and-groups}

Таны тодорхой хэрэглээний тохиолдолд аль нэгтгэлийн горим илүү тохиромжтойг сонгосны дараа анхдагч жагсаалтуудыг (мөн горимоос хамааран бүлгүүдийг) сонгох цаг боллоо. Тухайн төлөвлөгөөнд тодорхой жагсаалт сонгогдоогүй үед анхдагч жагсаалтыг ашиглана.

Мөн цуцлагдсан Account-уудад зориулсан анхдагч жагсаалтыг сонгож болно. Хэрэглэгч өөрийн subscription-ийг дуусгаж, өөрийн Account-ыг таны сүлжээнээс устгах бүрд түүний имэйл хаяг цуцлагдсан Account-уудад тохируулсан жагсаалт руу шилжинэ.

## Давхар захиалгыг идэвхжүүлэх {#enabling-double-subscription}

Ижил тохиргооны хуудсан дээр та **Давхар захиалга**-ыг идэвхжүүлэх боломжтой. Энэ сонголт нь хэрэглэгчийг сонгосон анхдагч жагсаалт БОЛОН тухайн хэрэглэгчийн төлөвлөгөөнд сонгосон жагсаалтын аль алинд нэмнэ. Хэрэв энэ сонголтыг идэвхжүүлээгүй бол хэрэглэгч зөвхөн хэрэглэгчийн төлөвлөгөөнд сонгосон жагсаалтад нэмэгдэнэ. Хэрэв тухайн тодорхой төлөвлөгөөнд юу ч тохируулаагүй бол хэрэглэгч сонгосон анхдагч жагсаалтад нэмэгдэнэ.

<!-- Screenshot unavailable: MailChimp double subscription toggle setting -->

_Бүртгүүлэх үед хэрэглэгчдээ анхдагч жагсаалт болон төлөвлөгөөний жагсаалтын аль алинд захиалуулах боломжтой_

## Төлөвлөгөө бүрд жагсаалт ба бүлгүүд сонгох {#selecting-lists-and-groups-for-each-of-the-plans}

Өөрийн **Төлөвлөгөөнүүд** рүү очоод засахыг хүссэн төлөвлөгөөгөө сонгоно уу. **Advanced Options** meta-box дээр MailChimp нэртэй шинэ таб боломжтой болохыг анхаарна уу.

Тэр таб дээр тухайн төлөвлөгөөнд зориулж олон жагсаалт эсвэл нэг жагсаалт ба олон бүлэг (нэгтгэлийн горимоос хамааран) сонгох боломжтой.

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_Төлөвлөгөө бүрдээ ашиглахыг хүссэн жагсаалт ба бүлгүүдийг сонгоно уу_

Шинэ хэрэглэгч тухайн төлөвлөгөөнд бүртгүүлэх үед түүний имэйл хаяг сонгосон жагсаалт ба бүлгүүдэд нэмэгдэнэ!
