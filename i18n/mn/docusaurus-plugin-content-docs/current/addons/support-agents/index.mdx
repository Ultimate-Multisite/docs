# Дэмжлэгийн агентууд {#support-agents}

Бүхэл бүтэн сүлжээг ганцаараа удирдах нь амар ажил биш. Таны сүлжээ өсөж эхлэхийн хэрээр дэмжлэгийн эрэлт нэмэгдэж байгааг та анзаарах болно. Ажлын ачааллаа хөнгөлөхөд туслах хүмүүсийг авчрах нь сайн санаа. Гэхдээ бүрэн хандалт/зөвшөөрөл өгөх нь эрсдэлтэй бөгөөд таны сүлжээг эрсдэлд оруулж болзошгүй. Энд энэ Дэмжлэгийн агент нэмэлт хэрэг болно.

Энэ нэмэлт нь Дэмжлэгийн агент гэж нэрлэгдэх шинэ төрлийн WordPress хэрэглэгч үүсгэх боломж олгож, тухайн хэрэглэгч ямар зөвшөөрөлтэй байхыг бүрэн хянах боломжийг танд өгнө. Зөвшөөрөл олгох нь сэлгэх товчийг асаахтай адил хялбар бөгөөд устгахаар шийдвэл унтраахад л болно.

## Онцлогууд {#features}

- WordPress ба Ultimate Multisite зөвшөөрлийн зүйлчилсэн жагсаалт
- Хандалтыг хялбараар идэвхжүүлэх ба идэвхгүй болгох
- Хязгааргүй тооны агентын Account-ууд үүсгэх

## Суулгалт {#installation}

1. 'ultimate-multisite-support-agents'-ийг '/wp-content/plugins/' лавлах руу байршуулна уу
2. WordPress доторх 'Плагинууд' цэсээр дамжуулан плагин-ыг идэвхжүүлнэ үү
3. Плагин-ыг тохируулахын тулд алхам алхмаар зааварлагчийг дагана уу
