# Хэрэгслүүд ба тохиргоо

Theme Builder нь таны WordPress сайтыг шинжлэх, дизайн гаргах, бүтээхэд зориулсан хэрэгслүүдийн багцыг ашигладаг. 1.16.0 хувилбарт **sd-ai-agent/site-scrape одоо Tier 1 хэрэгсэл болсон** тул Theme Builder-д анхдагчаар ашиглах боломжтой. Superdav AI Agent v1.18.0 нь итгэмжлэгдсэн хөгжүүлэгчийн ажлын урсгалд зориулан тусад нь түгээгддэг Advanced дагалдах багцыг мөн танилцуулж байна; эдгээр хэрэгслүүд нь WordPress.org үндсэн багцын хэсэг биш бөгөөд тусад нь суулгаж, зөвшөөрөл олгох шаардлагатай.

## Боломжтой хэрэгслүүд

### Tier 1 хэрэгслүүд (Үргэлж боломжтой)

Tier 1 хэрэгслүүд нь нэмэлт тохиргоо шаардалгүйгээр Theme Builder-д анхдагчаар боломжтой.

#### sd-ai-agent/site-scrape

**Зорилго:** Одоо байгаа вэбсайтуудыг шинжилж, дизайны санаа, контентын бүтэц, ажиллагааг гаргаж авах.

**Чадавх:**

- **Вэбсайтын шинжилгээ** — өрсөлдөгч эсвэл санаа авах вэбсайтуудыг scrape хийж шинжлэх
- **Дизайны гаргалт** — өнгө, фонт, layout загваруудыг тодорхойлох
- **Контентын бүтэц** — хуудасны зохион байгуулалт ба шатлалыг ойлгох
- **Онцлог илрүүлэлт** — plugin-ууд болон ажиллагааг тодорхойлох
- **Гүйцэтгэлийн шинжилгээ** — хуудасны хурд ба оновчлолыг шалгах
- **SEO шинжилгээ** — meta tag-ууд болон бүтэцлэгдсэн өгөгдлийг хянах

**Ашиглалт:**

```
example.com-ийн дизайныг шинжилж, миний сайтад санаа болгон ашигла.
```

**Юуг гаргаж авдаг вэ:**

- Өнгөний палитр ба өнгөний хэрэглээ
- Typography (фонт ба хэмжээ)
- Layout бүтэц ба зай авалт
- Navigation загварууд
- Контентын зохион байгуулалт
- Зураг ба медиа хэрэглээ
- Интерактив элементүүд
- Гар утасны дэлгэцэд нийцэх байдал

**Хязгаарлалтууд:**

- Нууц үгээр хамгаалагдсан сайтуудыг scrape хийх боломжгүй
- robots.txt болон сайтын бодлогыг хүндэтгэнэ
- Динамик контентыг бүрэн барьж авахгүй байж магадгүй
- JavaScript ихтэй сайтуудын шинжилгээ хязгаарлагдмал байж магадгүй
- Том сайтуудыг шинжлэхэд илүү удаан хугацаа шаардагдаж магадгүй

### Tier 2 хэрэгслүүд (Сонголтоор)

Tier 2 хэрэгслүүдийг Theme Builder тохиргоонд ил тод идэвхжүүлсэн үед ашиглах боломжтой.

#### Нарийвчилсан аналитик

Гүйцэтгэлийн дэлгэрэнгүй хэмжүүрүүдийг өгнө:

- Хуудас ачаалах хугацаа
- Core Web Vitals
- SEO оноо
- Хүртээмжийн оноо
- Шилдэг туршлагын оноо

#### Контент оновчлогч

Дараах зүйлсийг шинжилж, сайжруулалт санал болгоно:

- Уншигдах байдал
- SEO оновчлол
- Түлхүүр үгийн хэрэглээ
- Контентын бүтэц
- Call-to-action байршуулалт

### Tier 3 хэрэгслүүд (Premium)

Tier 3 хэрэгслүүд нэмэлт тохиргоо эсвэл premium боломжууд шаарддаг.

#### AI контент үүсгэгч

Дараах контентыг үүсгэнэ:

- Бүтээгдэхүүний тайлбар
- Үйлчилгээний хуудсууд
- Блог нийтлэлүүд
- Meta тайлбарууд
- Call-to-action текст

#### Нарийвчилсан дизайны хэрэгслүүд

Нарийвчилсан дизайны чадавх өгнө:

- Custom CSS үүсгэх
- Animation үүсгэх
- Интерактив элементийн дизайн
- Нарийвчилсан өнгөний онол
- Typography оновчлол

### Advanced дагалдах хэрэгслүүд (Тусдаа багц)

Advanced дагалдах багц нь Superdav AI Agent-ийн үндсэн хувилбараас тусад нь түгээгддэг. Энэ нь администраторууд agent-д өндөр нөлөөтэй хэрэгслүүд ашиглахыг ил тод зөвшөөрдөг итгэмжлэгдсэн хөгжүүлэгч болон сайт эзэмшигчийн орчинд зориулагдсан.

Advanced дагалдах чадавхууд дараахыг багтааж болно:

- **Файлын системийн хэрэгслүүд** — итгэмжлэгдсэн орчинд зөвшөөрөгдсөн файлуудыг шалгаж, өөрчлөх
- **Өгөгдлийн сангийн хэрэгслүүд** — ил тод зөвшөөрсөн үед сайтын өгөгдлийг query хийх эсвэл шинэчлэх
- **WP-CLI хэрэгслүүд** — WordPress засвар үйлчилгээ болон шалгалтын командууд ажиллуулах
- **REST dispatcher хэрэгслүүд** — хяналттай dispatch ажлын урсгалаар бүртгэгдсэн REST endpoint-уудыг дуудах
- **Plugin builder хэрэгслүүд** — plugin кодын суурь бүтэц үүсгэж, давтан сайжруулах
- **Git snapshot хэрэгслүүд** — эрсдэлтэй хөгжүүлэгчийн үйлдлүүдээс өмнө snapshot үүсгэх
- **Хэрэглэгчийн удирдлагын хэрэгслүүд** — зөвшөөрөгдсөн үед хэрэглэгчийн бүртгэлүүдийг хянах эсвэл тохируулахад туслах
- **Benchmark хэрэгслүүд** — хөгжүүлэлтийн хяналтад зориулж гүйцэтгэл эсвэл чадавхын benchmark ажиллуулах

Эдгээрийг бүх суулгалтад ерөнхийдөө ашиглах боломжтой Theme Builder хэрэгслүүд гэж баримтжуулж болохгүй. Тэдгээр нь зөвхөн Advanced дагалдах багцыг суулгаж, идэвхжүүлж, итгэмжлэгдсэн администраторуудаар хязгаарласан үед боломжтой.

## Хэрэгслийн тохиргоо

### Хэрэгслүүдийг идэвхжүүлэх

Theme Builder-д нэмэлт хэрэгслүүдийг идэвхжүүлэхийн тулд:

1. **Gratis AI Agent → Settings** рүү очно уу
2. **Theme Builder → Tools** рүү очно уу
3. Шаардлагатайгаар хэрэгслүүдийг асаах/унтраах
4. Тохиргоог хадгалах

### Хэрэгслийн зөвшөөрлүүд

Зарим хэрэгсэл зөвшөөрөл шаарддаг:

- **Сайт scrape хийх** — интернет хандалт шаарддаг
- **Аналитик** — Google Analytics интеграц шаарддаг
- **Контент үүсгэх** — API түлхүүрүүд шаарддаг
- **Нарийвчилсан боломжууд** — premium subscription шаардаж магадгүй
- **Advanced дагалдах хэрэгслүүд** — тусад нь түгээгддэг Advanced багц болон администраторын ил тод итгэл шаарддаг

### API түлхүүрүүд ба нэвтрэх мэдээлэл

Шаардлагатай хэрэгслүүдэд API түлхүүрүүдийг тохируулна уу:

1. **Gratis AI Agent → Settings → API Keys** рүү очно уу
2. Хэрэгсэл бүрийн нэвтрэх мэдээллийг оруулна уу
3. Холболтыг туршина уу
4. Аюулгүйгээр хадгална уу

### Advanced дагалдах багцыг суулгах

Advanced дагалдах багц нь WordPress.org үндсэн ZIP-ээс тусад нь гардаг. Үүнийг зөвхөн тухайн хувилбарын албан ёсны төслийн түгээх сувгаас суулгаж, дараа нь хандалтыг итгэмжлэгдсэн администраторуудаар хязгаарлана уу. Үйлдвэрлэлийн сайт дээр файлын систем, өгөгдлийн сан, WP-CLI, REST dispatcher, plugin builder, хэрэглэгчийн удирдлага эсвэл benchmark хэрэгслүүдийг идэвхжүүлэхээс өмнө хүний зөвшөөрлийн хаалтууд болон git snapshot-ийн зан төлөвийг хянаарай.

## sd-ai-agent/site-scrape ашиглах

### Үндсэн ашиглалт

Theme Builder-ээс вэбсайт шинжлэхийг хүснэ үү:

```
competitor.com дээрх миний өрсөлдөгчийн сайтын дизайныг шинжилж,
миний сайтад дизайны сайжруулалт санал болго.
```

### Тодорхой шинжилгээ

Тодорхой төрлийн шинжилгээ хүснэ үү:

```
example.com-оос өнгөний палитрыг гаргаж аваад санаа болгон ашигла.
```

```
example.com-ийн navigation бүтцийг шинжилж, төстэй
зохион байгуулалтыг миний сайтад хэрэглэ.
```

```
example.com-ийн гар утасны дэлгэцэд нийцэх байдлыг шалгаж, миний сайт
ижил хэмжээнд нийцтэй эсэхийг баталгаажуул.
```

### Харьцуулсан шинжилгээ

Олон сайтыг харьцуулна уу:

```
site1.com болон site2.com-ийн дизайнуудыг харьцуулж, хоёулангийнх нь
хамгийн сайн элементүүдийг нэгтгэсэн дизайн үүсгэ.
```

## Хэрэгслийн хязгаарлалтууд ба анхаарах зүйлс

### Rate Limiting

- Скрейпинг нь серверийн хэт ачааллаас сэргийлэхийн тулд хурдны хязгаарлалттай
- Нэг сайт руу олон хүсэлт илгээвэл хязгаарлагдаж болзошгүй
- Том сайтуудыг шинжлэхэд илүү удаан хугацаа шаардагдаж магадгүй

### Агуулгын үнэн зөв байдал

- Динамик агуулгыг бүрэн авч чадахгүй байж магадгүй
- JavaScript-ээр дүрслэгдсэн агуулга дутуу байж магадгүй
- Бодит цагийн өгөгдөл тусгагдахгүй байж магадгүй
- Зарим агуулга төлбөртэй ханын цаана байж магадгүй

### Хууль эрх зүй ба ёс зүйн анхаарах зүйлс

- robots.txt болон сайтын бодлогыг хүндэтгээрэй
- Зохиогчийн эрхтэй агуулгыг дахин нийтлэх зорилгоор бүү скрейп хийгээрэй
- Шинжилгээг хуулбарлахад бус, урам авахад ашиглаарай
- Гаргаж авсан агуулгыг ашиглах эрхтэй эсэхээ баталгаажуулаарай
- Шинжилсэн сайтуудын үйлчилгээний нөхцөлийг дагаж мөрдөөрэй

### Гүйцэтгэлд үзүүлэх нөлөө

- Том сайтуудыг скрейп хийхэд хугацаа шаардагдаж магадгүй
- Олон зэрэг скрейп хийх нь илүү удаан байж магадгүй
- Сүлжээний холболт хурданд нөлөөлнө
- Том өгөгдлийн багц илүү их боловсруулалт шаардаж магадгүй

## Шилдэг туршлагууд

### Сайтын шинжилгээ ашиглах

1. **Олон сайт шинжлэх** — хэд хэдэн эх сурвалжаас урам санаа цуглуулаарай
2. **Бүтцэд төвлөрөх** — байрлал болон зохион байгуулалтын хэв маягийг сурч аваарай
3. **Өнгө гаргаж авах** — өнгөний палитруудыг эхлэх цэг болгон ашиглаарай
4. **Типограф судлах** — өөрт таалагдах үсгийн хослолуудыг тодорхойлоорой
5. **Навигацыг хянах** — цэсийн зохион байгуулалтыг ойлгоорой

### Ёс зүйтэй скрейпинг

1. **robots.txt-г хүндэтгэх** — сайтын зааврыг дагаж мөрдөөрэй
2. **Агуулга бүү хуулбарла** — шинжилгээг зөвхөн урам авахад ашиглаарай
3. **Эрхийг баталгаажуулах** — гаргаж авсан агуулгыг ашиглаж болохоо баталгаажуулаарай
4. **Эх сурвалжийг дурд** — урам авсан эх сурвалжуудад талархал өгч дурд
5. **Нөхцөлийг дагах** — сайтын үйлчилгээний нөхцөлийг мөрдөөрэй

### Хэрэгслийн үр нөлөөг хамгийн их болгох

1. **Тодорхой бай** — тодорхой төрлийн шинжилгээ хүсээрэй
2. **Хам нөхцөл өг** — сайтынхаа зорилгыг тайлбарлаарай
3. **Хүлээлтээ тогтоо** — дизайны зорилгуудаа тайлбарлаарай
4. **Давтан сайжруул** — үр дүнд үндэслэн сайжруулаарай
5. **Хэрэгслүүдийг хослуул** — иж бүрэн шинжилгээнд олон хэрэгсэл ашиглаарай

## Асуудал шийдвэрлэх

### Сайтыг скрейп хийж болохгүй байна

- Сайт нийтэд нээлттэй хандах боломжтой эсэхийг шалгаарай
- robots.txt скрейп хийхийг зөвшөөрч байгаа эсэхийг баталгаажуулаарай
- Өөр сайт туршаарай
- Интернэт холболтоо шалгаарай
- Асуудал үргэлжилбэл дэмжлэгтэй холбогдоорой

### Дутуу шинжилгээ

- Сайт динамик агуулгатай байж магадгүй
- JavaScript бүрэн дүрслэгдээгүй байж магадгүй
- Том сайтуудад хугацаа дуусаж болзошгүй
- Оронд нь тодорхой хуудсуудыг шинжилж үзээрэй
- Тодорхой шинжилгээний төрлүүд хүсээрэй

### Удаан гүйцэтгэл

- Том сайтуудыг шинжлэхэд илүү удаан хугацаа шаардагдана
- Олон зэрэг скрейп хийх нь илүү удаан
- Сүлжээний холболт хурданд нөлөөлнө
- Ачаалал багатай цагт туршаарай
- Эхлээд жижиг хэсгүүдийг шинжлээрэй
