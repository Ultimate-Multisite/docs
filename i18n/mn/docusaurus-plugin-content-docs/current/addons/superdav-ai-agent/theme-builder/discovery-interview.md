# Нээлтийн ярилцлага {#discovery-interview}

**Нээлтийн ярилцлага** нь Theme Builder-ийн дизайны үйл явц дахь эхний алхам юм. Энэ нь хувийн тохируулгатай theme үүсгэхийн тулд таны сайт, бизнес болон дизайны сонголтын талаар мэдээлэл цуглуулдаг.

## Тойм {#overview}

Нээлтийн ярилцлага нь таны брэнд, зорилтот үзэгчид болон дизайны зорилгыг ойлгохын тулд цуврал асуултаар чиглүүлнэ. 1.16.0 хувилбарт ярилцлага нь одоо таны дизайнд харааны урам өгөх боломжтой **зураг байршуулах алхам**-ыг багтаасан.

## Ярилцлагын алхмууд {#interview-steps}

### 1. Бизнесийн мэдээлэл {#1-business-information}

- Сайтын нэр ба уриа
- Бизнесийн төрөл эсвэл салбар
- Бизнесийн үндсэн зорилгууд
- Зорилтот үзэгчдийн тодорхойлолт

### 2. Дизайны сонголтууд {#2-design-preferences}

- Сонгосон өнгөний схем
- Дизайны хэв маяг (орчин үеийн, сонгодог, минималист, хөгжилтэй гэх мэт)
- Урам өгөх вебсайтууд эсвэл брэндүүд
- Заавал байх дизайны элементүүд

### 3. Зураг байршуулах (v1.16.0-д шинэ) {#3-photo-upload-new-in-v1160}

Дизайны чиглэлд нөлөөлөх нэг эсвэл хэд хэдэн зураг байршуулаарай:

#### Зөвшөөрөгдсөн форматууд {#accepted-formats}

| Формат | Өргөтгөл | Дэлгэрэнгүй |
|--------|-----------|---------|
| **JPEG** | `.jpg`, `.jpeg` | Гэрэл зурагт зөвлөмжтэй |
| **PNG** | `.png` | Тунгалаг байдлыг дэмжинэ |
| **WebP** | `.webp` | Орчин үеийн формат, файлын хэмжээ бага |
| **GIF** | `.gif` | Зөвхөн хөдөлгөөнгүй зураг (анимацгүй) |

#### Файлын шаардлагууд {#file-requirements}

- **Файлын дээд хэмжээ:** 10 MB
- **Хамгийн бага хэмжээ:** 400 × 300 пиксел
- **Зөвлөмжтэй хэмжээ:** 1200 × 800 пиксел эсвэл түүнээс том
- **Харьцаа:** Ямар ч харьцаа тохиромжтой

#### Зураг дизайны чиглэлд хэрхэн нөлөөлдөг вэ {#how-photos-influence-design-direction}

Таны байршуулах зургууд Theme Builder-д дараахыг ойлгоход тусална:

- **Өнгөний палитр** — давамгай өнгөнүүдийг гаргаж авч theme-д ашиглана
- **Харааны хэв маяг** — орчин үеийн, хөдөөгийн, минималист, тод өнгөтэй гэх мэт
- **Сэтгэл хөдлөл ба уур амьсгал** — мэргэжлийн, бүтээлч, найрсаг, тансаг гэх мэт
- **Дизайны элементүүд** — хээ, бүтэц, композици зохиомжийн хэв маягууд
- **Типографийн урам** — харааны theme-тэй зохицох үсгийн хэв маягууд

### 4. Агуулгын бүтэц {#4-content-structure}

- Шаардлагатай үндсэн хуудсууд (Нүүр, Тухай, Үйлчилгээ, Холбоо барих гэх мэт)
- Блог эсвэл мэдээний хэсэг шаардлагатай эсэх
- Цахим худалдааны ажиллагаа шаардлагатай эсэх
- Тусгай боломжууд эсвэл интеграц

### 5. Хянах ба баталгаажуулах {#5-review-and-confirmation}

- Өгсөн бүх мэдээллийн хураангуй
- Дизайн үүсгэлт эхлэхээс өмнөх баталгаажуулалт

## Зураг байршуулах шилдэг туршлагууд {#photo-upload-best-practices}

### Юу байршуулах вэ {#what-to-upload}

- **Брэндийн урам** — лого, өнгөний дээж, эсвэл брэндийн удирдамж
- **Өрсөлдөгчийн сайтууд** — таны биширдэг сайтуудын дэлгэцийн зураг
- **Mood board-ууд** — таны хүссэн гоо зүйн мэдрэмжийг илэрхийлэх зургуудын цуглуулга
- **Бүтээгдэхүүний зураг** — танилцуулах бүтээгдэхүүнтэй бол
- **Баг эсвэл оффисын зураг** — хувийн эсвэл мэргэжлийн брэнд бүрдүүлэлтэд
- **Амьдралын хэв маягийн зураг** — таны зорилтот үзэгчдийг төлөөлөх

### Юунаас зайлсхийх вэ {#what-to-avoid}

- **Чанар муутай зураг** — бүдэг эсвэл пикселжсэн зураг
- **Хамааралгүй зураг** — таны брэндийг төлөөлөхгүй зураг
- **Текст ихтэй зураг** — их хэмжээний тексттэй дэлгэцийн зураг
- **Зохиогчийн эрхтэй материал** — зургуудыг ашиглах эрхтэй эсэхээ баталгаажуул
- **Хөдөлгөөнт GIF-үүд** — зөвхөн хөдөлгөөнгүй зургуудыг боловсруулна

### Хамгийн сайн үр дүнд хүрэх зөвлөмжүүд {#tips-for-best-results}

1. **2-3 зураг байршуулах** — олон зураг илүү сайн контекст өгнө
2. **Өндөр чанартай зураг ашиглах** — тод, сайн гэрэлтүүлэгтэй зураг хамгийн сайн ажиллана
3. **Төрөл бүрийн байдал оруулах** — өнгө, хэв маяг, сэдвийн холимог
4. **Контекст өгөх** — зураг бүрийг яагаад сонгосноо товч тайлбарла
5. **Тодорхой байх** — тодорхой нэг элемент таалагдаж байвал түүнийг зааж өг

## Дизайны чиглэлийн гаралт {#design-direction-output}

Нээлтийн ярилцлагын дараа Theme Builder дараахыг үүсгэнэ:

- **Өнгөний палитр** — үндсэн, хоёрдогч болон тодотгох өнгөнүүд
- **Типографи** — гарчиг болон үндсэн текстийн үсгийн бүлүүд
- **Байршлын хэв маяг** — зай авалт, зэрэгцүүлэлт, компонентуудын зохион байгуулалт
- **Харааны элементүүд** — дүрс, хээ, чимэглэлийн элементүүд
- **Mood board** — дизайны чиглэлийн харааны дүрслэл

## Дараагийн алхмууд {#next-steps}

Нээлтийн ярилцлагыг дуусгасны дараа:

1. Үүсгэсэн дизайны чиглэлийг хянах
2. Санал хүсэлт өгөх эсвэл тохируулга хүсэх
3. [Дизайны чиглэл](./design-direction.md) алхам руу үргэлжлүүлэх
4. Theme Builder таны custom theme-ийг үүсгэнэ
