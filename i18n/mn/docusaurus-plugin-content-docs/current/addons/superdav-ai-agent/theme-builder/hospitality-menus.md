# Зочлох үйлчилгээний цэсүүд

**Зочлох үйлчилгээний цэсүүд** функц нь Theme Builder-д бүтэцлэгдсэн хоол, ундааны цэсийн хуудсуудыг шууд таны WordPress сайтад үүсгэж, оруулах боломж олгодог.

## Тойм

Theme Builder нь одоо ресторан, кафе, бар, кейтеринг үйлчилгээ зэрэг зочлох үйлчилгээний бизнесүүдэд зориулсан мэргэжлийн, бүтэцлэгдсэн цэсийн хуудсуудыг үүсгэж чадна. Эдгээр цэсүүд нь таны сайтын загварт бүрэн нэгтгэгдсэн бөгөөд хялбархан шинэчилж, удирдах боломжтой.

## Дэмжигдэх зочлох үйлчилгээний төрлүүд

- **Ресторанууд** — бүрэн үйлчилгээтэй хоолны цэсүүд
- **Кафенууд** — кофе болон хөнгөн хоолны цэсүүд
- **Бар ба лоунжууд** — ундаа болон зуушны цэсүүд
- **Талх нарийн боовны газрууд** — нарийн боов болон талхны цэсүүд
- **Кейтеринг үйлчилгээ** — арга хэмжээний цэсийн сонголтууд
- **Хоолны явуулын машин** — явуулын хоолны үйлчилгээний цэсүүд
- **Шар айрагны үйлдвэр ба дарсны үйлдвэрүүд** — тайлбартай ундааны цэсүүд

## Цэсийн бүтэц

### Цэсийн ангиллууд

Цэсүүдийг ангиллаар зохион байгуулдаг:

- **Зууш** — эхлэл хоол болон жижиг тавагнууд
- **Үндсэн хоол** — гол хоолнууд
- **Хачир** — дагалдах хачир болон хүнсний ногоо
- **Амттан** — чихэрлэг амттангууд
- **Ундаа** — ундаа (согтууруулах болон согтууруулах бус)
- **Тусгай саналууд** — өдөр тутмын эсвэл улирлын тусгай саналууд

### Цэсийн зүйлийн формат

Цэсийн зүйл бүр дараахыг агуулна:

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### Зүйлийн талбарууд

| Талбар | Төрөл | Тайлбар |
|-------|------|-------------|
| `name` | string | Хоол эсвэл ундааны нэр |
| `description` | string | Тухайн зүйлийн дэлгэрэнгүй тайлбар |
| `price` | string | Үнэ (валютаар форматласан) |
| `dietary_info` | array | Хооллолтын шинжүүд (веган, цавуулаггүй гэх мэт) |
| `allergens` | array | Түгээмэл харшил үүсгэгчид (самар, хясаа гэх мэт) |
| `availability` | string | Хэзээ боломжтой (өдөр бүр, улирлын гэх мэт) |

## Зочлох үйлчилгээний цэсүүд үүсгэх

### Алхам 1: Цэсийн мэдээлэл өгөх

Theme Builder-д цэсийнхээ талаар хэлнэ үү:

```
Миний Итали ресторанд зориулж зууш, паста хоол, үндсэн хоол,
амттантай ресторан цэс үүсгэ. Үнэ болон тайлбаруудыг оруул.
```

### Алхам 2: Theme Builder цэс үүсгэнэ

Theme Builder:

1. Бүтэцлэгдсэн цэсийн хуудас үүсгэнэ
2. Үүнийг таны сайтын theme-тэй нийцүүлэн загварчилна
3. Зүйлсийг ангиллуудад зохион байгуулна
4. Үнэ болон тайлбаруудыг форматлана
5. Хооллолтын болон харшил үүсгэгчийн мэдээлэл нэмнэ

### Алхам 3: Хянаж, өөрчлөх

Та дараахыг хийж болно:

1. Цэсийн зүйлс болон үнийг засах
2. Ангилал нэмэх эсвэл устгах
3. Ангилал доторх зүйлсийн дарааллыг өөрчлөх
4. Тайлбар болон хооллолтын мэдээллийг шинэчлэх
5. Загварчлал болон байрлалыг тохируулах

## Цэс харуулах сонголтууд

### Бүтэн цэсийн хуудас

Таны бүрэн цэсийг харуулах зориулалтын хуудас:

- Ангиллаар зохион байгуулсан
- Хайх болон шүүх боломжтой
- Хэвлэхэд тохиромжтой байрлалтай
- Гар утсанд нийцтэй дизайнтай

### Цэсийн виджет

Цэсийн хэсгүүдийг бусад хуудсанд оруулах:

- Нүүр хуудсанд онцлох зүйлс
- Хажуугийн хэсэгт өдөр тутмын тусгай саналууд
- Барын хуудсанд ундааны цэс
- Footer-т амттаны үзүүлэн

### Цэсийн PDF

Татаж авах боломжтой PDF цэс үүсгэх:

- Мэргэжлийн форматтай
- Хэвлэхэд бэлэн чанартай
- Зураг болон тайлбаруудыг агуулна
- Хуваалцах болон имэйлээр илгээхэд хялбар

## Хооллолтын болон харшил үүсгэгчийн мэдээлэл

### Хооллолтын шинжүүд

Зүйлсийг хооллолтын мэдээллээр тэмдэглэнэ:

- **Веган** — амьтны гаралтай бүтээгдэхүүнгүй
- **Цагаан хоолтон** — махгүй
- **Цавуулаггүй** — целиак өвчтэй хүмүүст аюулгүй
- **Сүүн бүтээгдэхүүнгүй** — сүүний бүтээгдэхүүнгүй
- **Самаргүй** — модны самар эсвэл газрын самаргүй
- **Нүүрс ус багатай** — нүүрс ус багасгасан
- **Уураг ихтэй** — уургаар баялаг

### Харшил үүсгэгчийн анхааруулга

Түгээмэл харшил үүсгэгчдийг оруулна:

- **Самар** — модны самар болон газрын самар
- **Хясаа төрлийн далайн хүнс** — хавч хэлбэртэн болон нялцгай биетэн
- **Загас** — бүх төрлийн загас
- **Сүүн бүтээгдэхүүн** — сүү болон сүүний бүтээгдэхүүн
- **Өндөг** — тахианы өндөг
- **Шар буурцаг** — шар буурцгийн бүтээгдэхүүн
- **Цавуулаг** — улаан буудай болон цавуулаг агуулсан үр тариа
- **Гүнжид** — гүнждийн үр болон тос

## Цэсийн удирдлага

### Үнэ шинэчлэх

Цэсийн үнийг хялбархан шинэчилнэ:

1. Цэсийн хуудас руу очно
2. "Цэс засах" дээр дарна
3. Зүйлсийн үнийг шинэчилнэ
4. Өөрчлөлтүүдийг хадгална
5. Өөрчлөлтүүд таны сайт дээр шууд харагдана

### Улирлын зүйлс нэмэх

Улирлын цэсийн хувилбарууд үүсгэнэ:

1. Шинэ цэсийн хувилбар үүсгэнэ
2. Улирлын зүйлс нэмнэ
3. Зүйлсийг "Улирлын" гэж тэмдэглэнэ
4. Боломжтой байх огноог товлоно
5. Улирлын үеэр автоматаар харуулна

### Тусгай саналууд удирдах

Өдөр тутмын эсвэл долоо хоногийн тусгай саналуудыг харуулна:

1. "Тусгай саналууд" ангилал үүсгэнэ
2. Боломжтой байх огноотой зүйлс нэмнэ
3. Нүүр хуудсанд тусгай саналуудыг онцолно
4. Өдөр бүр эсвэл долоо хоног бүр шинэчилнэ
5. Хуучин тусгай саналуудыг архивлана

## Theme Builder-тэй нэгтгэх

Зочлох үйлчилгээний сайтуудад Theme Builder ашиглах үед:

1. **Автомат цэс илрүүлэлт** — таныг хоол/ундааны бизнес эсэхийг тодорхойлно
2. **Цэсийн хуудас үүсгэх** — мэргэжлийн цэсийн хуудсууд үүсгэнэ
3. **Дизайны нийцэл** — цэсүүд таны сайтын theme-тэй нийцнэ
4. **Гар утасны оновчлол** — цэсүүд утсан дээр үзэмжтэй харагдана
5. **SEO оновчлол** — цэсүүд хайлтын системд ээлтэй байна

## Шилдэг туршлагууд

### Цэсийн дизайн

- **Тодорхой зохион байгуулалт** — логик ангиллын бүтэц
- **Уншихад хялбар тайлбарууд** — хоолны дуршил төрүүлэхүйц, мэдээлэлтэй
- **Нэгэн жигд үнэ** — тодорхой валют болон формат
- **Мэргэжлийн зурагнууд** — өндөр чанартай хоолны зургууд
- **Хоосон зай** — хуудсыг хэт бүү дүүргэ

### Агуулга

- **Үнэн зөв тайлбарууд** — зүйлсийг үнэн зөв тайлбарла
- **Тусгай саналуудыг онцлох** — тусгай саналуудыг ялгаруул
- **Харшил үүсгэгчдийг оруулах** — харшил үүсгэгчдийг үргэлж жагсаа
- **Тогтмол шинэчлэх** — үнэ болон зүйлсийг шинэ байлга
- **Хоолны дуршил төрүүлэх хэллэг ашиглах** — зүйлсийг амттай сонсогдуулах

### Хүртээмж

- **Уншихад хялбар фонт** — тод, уншигдахуйц үсгийн хэв ашигла
- **Хангалттай ялгарал** — текст уншигдахуйц эсэхийг баталгаажуул
- **Хооллолтын шошго** — хооллолтын сонголтуудыг тодорхой тэмдэглэ
- **Харшил үүсгэгчийн анхааруулга** — харшил үүсгэгчдийг тод харагдуул
- **Гар утсанд ээлтэй** — бүх төхөөрөмж дээр турш
