# Өөрчлөлтийн бүртгэл

## 1.18.0 — 2026-06-29-нд гарсан {#1180--released-on-2026-06-29}

### Шинэ {#new}

- **Google Calendar хэрэгслүүд** — хуваарьт нийцсэн автоматжуулалтад зориулан тохируулсан календарь болон үйл явдлуудыг унших
- **Холбоо барих мэдээллийн зураглал ба оролцогчийн туслахууд** — календарийн оролцогчдыг сайтын хэрэглэгчид болон холбоо барих мэдээлэлтэй тааруулах
- **Хүний зөвшөөрлийн хаалтууд ба сануулгын бүртгэлүүд** — автоматжуулалтыг хянуулахын тулд түр зогсоож, давхардсан мэдэгдлээс зайлсхийх
- **TextBee SMS үйлчилгээ үзүүлэгч** — тохируулсан текст мессежийн мэдэгдлүүдийг TextBee-ээр дамжуулан илгээх
- **Advanced дагалдах багц** — WordPress.org бүтээлээс тусад нь түгээгддэг итгэмжлэгдсэн хөгжүүлэгчийн файлын систем, өгөгдлийн сан, WP-CLI, REST дамжуулагч, plugin үүсгэгч, git агшин, хэрэглэгчийн удирдлага, жишиг хэмжилтийн хэрэгслүүдийг нэмэх

### Сайжруулсан {#improved}

- **Удирдлагатай Superdav үйлчилгээний тохиргоо** — дэмжигдсэн сайтуудад зориулсан байршуулсан үйлчилгээний endpoint-ууд болон автомат холболтын бэлтгэлийг нэмэх
- **Хувилбарын багцлалт** — core болон Advanced ZIP-үүдийг тусад нь бүтээж, хоёуланг GitHub дээр нийтлэн, зөвхөн core багцыг WordPress.org руу илгээх

### Зассан {#fixed}

- **AI хүсэлтийн найдвартай байдал** — загвар сонголт, хүсэлтийн хугацаа дуусалт, анхдагч тохиргооны загвар, reasoning-text боловсруулалт, хүчингүй tool дуудлагад зориулсан дахин оролдох зааврыг сайжруулах
- **Календарь ба сануулгын бат бөх байдал** — Google Calendar token-ууд болон сануулгын давхардлыг арилгалтыг чангатгах
- **Анхны тохиргоо ба зөвшөөрлийн үргэлжлүүлэлтүүд** — frontend анхны тохиргооны эхлэлтүүд болон баталгаажсан чадварын үргэлжлүүлэлтүүдийг засах
- **WordPress.org багцлалтын хяналтын асуудлууд** — core хувилбарын багцлалтын хяналтын санал хүсэлтийг шийдвэрлэх

## 1.16.0 — 2026-05-20-нд гарсан {#1160--released-on-2026-05-20}

### Шинэ {#new-1}

- **Logo SVG үүсгэх чадвар** — Theme Builder одоо namespace-д аюулгүй цэвэрлэгээтэйгээр захиалгат logo SVG-үүд үүсгэж, шигтгэх боломжтой
- **Танин мэдэх ярилцлагад зураг байршуулах** — Theme Builder-ийн танин мэдэх ярилцлага одоо илүү баялаг дизайны нөхцөлд зориулсан зураг байршуулах алхмыг багтаасан
- **Палеттын контрастыг баталгаажуулах чадвар** — theme-д хэрэглэхээс өмнө өнгөний хослолуудыг WCAG нийцэлтэй эсэхийг шалгах
- **Зочлох үйлчилгээний цэсүүд** — Theme Builder одоо зочлох үйлчилгээний бизнесүүдэд зориулан бүтэцтэй хоол болон ундааны цэсийн хуудсууд үүсгэх боломжтой
- **Desktop болон mobile preview дүрслэл** — дизайны чиглэл сонгох үед desktop болон mobile төхөөрөмж дээр дизайнаа preview хийх
- **Navigation label параметр** — Create Menu чадвар одоо хуудасны гарчгаас тусдаа `navigation_label`-ийг дэмждэг
- **Tier 1 tool боломжит байдал** — sd-ai-agent/site-scrape одоо Theme Builder-д анхдагчаар боломжтой Tier 1 tool болсон

### Зассан {#fixed-1}

- **AI Client cache** — одоо хүсэлт хоорондын тогтвортой хадгалалтад transients ашигладаг болж, удаан ажиллах agent даалгавруудад өгөгдөл алдагдахаас сэргийлнэ
- **Plugin мөрийн үйлдлийн холбоосууд** — ойлгомжтой болгохын тулд засаж, нэрийг нь өөрчилсөн

## 1.10.0 — 2026-05-05-нд гарсан {#1100--released-on-2026-05-05}

### Шинэ {#new-2}

- **Tavily интернет хайлт** — Brave Search-ийн зэрэгцээ илүү баялаг интернет хайлтын үр дүнд зориулан Tavily-г хайлтын үйлчилгээ үзүүлэгчээр нэмэх
- **Theme-д нийцсэн суурилуулсан чадварууд** — Block Themes, Classic Themes, Kadence Blocks, Kadence Theme чадварын гарын авлагууд одоо plugin-тэй хамт ирдэг
- **Сайт бүтээгчийн холбоо барих маягтын чадвар** — чатын интерфейсээс шууд дурын хуудсанд холбоо барих маягт нэмэх

### Сайжруулсан {#improved-1}

- **WooCommerce интеграцийг дахин зохион байгуулсан** — илүү сайн найдвартай байдал болон нийцэлд зориулан одоо native WooCommerce API-уудыг ашигладаг
- **Үйлчилгээ үзүүлэгчийн жагсаалт автоматаар шинэчлэгддэг** — аливаа plugin идэвхжих эсвэл идэвхгүй болох үед

### Зассан {#fixed-2}

- **navigate-to чадвар** — зарим admin хуудсан дээрх хязгааргүй дахин ачаалах гогцоог зассан
- **list-posts чадвар** — одоо ангилал болон tag нэрсийг slug болгон зөв тодорхойлдог
- **WP-CLI командууд** — өмнөх дахин зохион байгуулалтын дараа алга болсон namespace alias-уудыг сэргээсэн
- **Үйл явдлын автоматжуулалт** — автоматжуулалтын хүснэгтүүд хараахан үүсээгүй сайтуудыг алдаагүй зохицуулдаг
- **memory-save чадвар** — одоо системийн заавар бүтээгчид зөв namespace prefix ашигладаг
- **Scalar tool үр дүнгүүд** — одоо AI руу буцаахаас өмнө зөв боогддог
- **Ашиглалтын статистик** — одоо хуучин хувилбаруудаас upgrade хийх үед legacy чадварын түлхүүрийн форматыг зөв боловсруулдаг
