# Цэс үүсгэх

**Цэс үүсгэх** чадвар нь хуудасны гарчгаас тусдаа ялгаатай навигацийн шошгыг дэмждэг WordPress навигацийн цэсүүдийг үүсгэдэг.

## Тойм

Энэ чадвар нь `navigation_label` параметрийг зааж өгөх боломжийг нэмснээр стандарт цэс үүсгэх ажиллагааг өргөтгөнө. Ингэснээр навигацид харагдах шошго нь хуудасны гарчгаас өөр байх цэсүүдийг үүсгэх боломжтой болж, сайтын бүтэц болон хэрэглэгчийн туршлагад илүү уян хатан байдал олгоно.

## Параметрүүд

| Параметр | Төрөл | Шаардлагатай | Тайлбар |
|-----------|------|----------|-------------|
| `name` | string | Тийм | Цэсийн нэр, жишээ нь `Primary Navigation` |
| `location` | string | Үгүй | Энэ цэсийг оноох theme байршил, жишээ нь `primary` |
| `navigation_label` | string | Үгүй | Навигацид харуулах шошго (хуудасны гарчгаас ялгаатай) |

## Буцаах утга

```json
{
  "success": true,
  "menu_id": 7,
  "name": "Primary Navigation",
  "navigation_label": "Main Menu"
}
```

## Навигацийн шошго ба хуудасны гарчиг

`navigation_label` параметр нь дотоод цэсийн нэрийг хэрэглэгчдэд навигацид харагдах шошгоос тусгаарлах боломжийг олгоно:

- **`name`** — WordPress-ийн ашигладаг дотоод цэсийн танигч (жишээ нь, "Primary Navigation")
- **`navigation_label`** — Сайтын зочдод навигацид харагдах шошго (жишээ нь, "Main Menu")

Энэ нь дараах үед хэрэгтэй:
- Таны дотоод нэрлэх журам хэрэглэгчдэд харагдах шошгоос өөр байх үед
- Админ самбартай харьцуулахад навигацид илүү богино шошго ашиглахыг хүсэх үед
- Өөр өөр шошгын урттай олон хэлийг дэмжих шаардлагатай үед
- Тодорхой бүс нутаг эсвэл хэрэглэгчийн бүлгүүдэд зориулж цэсүүд бүтээж байгаа үед

## Ашиглалтын жишээнүүд

### Жишээ 1: Навигацийн шошготой энгийн цэс

**Prompt:**
```
"Main Navigation" нэртэй үндсэн навигацийн цэс үүсгээд навигацийн шошгыг "Menu" болго.
```

**Үр дүн:**
```json
{
  "success": true,
  "menu_id": 7,
  "name": "Main Navigation",
  "navigation_label": "Menu"
}
```

### Жишээ 2: Тодорхой Theme байршилд зориулсан цэс

**Prompt:**
```
"Footer Links" нэртэй хөлийн цэс үүсгээд навигацийн шошгыг "Quick Links" болгож, footer байршилд оноо.
```

**Үр дүн:**
```json
{
  "success": true,
  "menu_id": 8,
  "name": "Footer Links",
  "navigation_label": "Quick Links",
  "location": "footer"
}
```

## Theme Builder-тэй нэгтгэх

Theme Builder ашиглах үед Цэс үүсгэх чадвар нь:

1. Боломжтой theme цэсийн байршлуудыг автоматаар илрүүлнэ
2. Таны дизайнд тохирох навигацийн шошготой цэсүүд үүсгэнэ
3. Цэсүүдийг зөв theme байршлуудад онооно
4. Үүсгэсний дараа цэсийн зүйлс нэмэхийг дэмжинэ

## Холбогдох чадварууд

- **`add_menu_item`** — Одоо байгаа цэсэнд зүйлс нэмэх
- **`update_menu`** — Цэсийн нэрийг өөрчлөх эсвэл theme байршилд дахин оноох
- **`delete_menu`** — Сайтаас цэс устгах

## Шилдэг туршлагууд

- **Тодорхой навигацийн шошго ашигла** — шошгуудыг хэрэглэгчдэд зориулан товч бөгөөд тайлбарлагдахуйц байлга
- **Theme байршлуудыг тааруул** — зөв харагдуулахын тулд цэсүүдийг зөв theme байршилд оноо
- **Цэсийн бүтцийг төлөвлө** — зүйлс үүсгэхээсээ өмнө цэсийн шатлалыг шийд
- **Хариу үйлдэлтэй байдлыг турш** — цэсүүд гар утасны төхөөрөмж дээр зөв харагдаж буйг шалга
- **Шошгуудыг нутагшуул** — өөр өөр хэлний хувилбаруудад өөр навигацийн шошго ашигла

## Хязгаарлалтууд

- Навигацийн шошгууд зөвхөн харагдах зориулалттай; дотоод `name` нь WordPress танихад ашиглагдсан хэвээр байна
- Theme-ийн дэмжлэг харилцан адилгүй; бүх theme бүх цэсийн байршлыг дэмждэггүй
- Цэс үүсгэсний дараа цэсийн зүйлсийг тусад нь нэмэх шаардлагатай
- Навигацийн шошгыг өөрчлөхөд цэсийг шинэчлэх шаардлагатай
