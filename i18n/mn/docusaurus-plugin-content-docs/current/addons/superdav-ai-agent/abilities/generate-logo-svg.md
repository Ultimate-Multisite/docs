# Logo SVG үүсгэх

**Generate Logo SVG** чадвар нь Theme Builder-д автоматаар нэрийн орон зайд аюулгүй цэвэрлэгээ хийж, захиалгат logo SVG-үүдийг шууд таны WordPress сайтад үүсгэн суулгах боломж олгодог.

## Тойм {#overview}

Энэ чадвар нь таны сайтын брэндийн чиглэл болон дизайны сонголтод үндэслэн хэмжээг нь өөрчлөх боломжтой вектор график (SVG) logo үүсгэдэг. Үүсгэсэн SVG-үүдийг WordPress-д ашиглахад аюулгүй байлгахын зэрэгцээ харагдах байдлын бүрэн бүтэн байдлыг хадгалахын тулд автоматаар цэвэрлэдэг.

## Параметрүүд {#parameters}

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `site_name` | string | Yes | Logo үүсгэх сайтын нэр |
| `style` | string | Yes | Дизайны хэв маяг (жишээ нь, "modern", "classic", "minimalist", "playful") |
| `colors` | array | No | Logo-д ашиглах hex өнгөний кодын массив (жишээ нь, `["#678233", "#ffffff"]`) |
| `width` | number | No | SVG өргөн пикселээр (өгөгдмөл: 200) |
| `height` | number | No | SVG өндөр пикселээр (өгөгдмөл: 200) |
| `include_text` | boolean | No | Logo-д сайтын нэрийг текстээр оруулах эсэх (өгөгдмөл: true) |

## Гаралтын формат {#output-format}

Энэ чадвар нь дараах бүтэцтэй SVG string буцаадаг:

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## SVG цэвэрлэгээний үйлдэл {#svg-sanitisation-behaviour}

Үүсгэсэн SVG-үүдэд автоматаар нэрийн орон зайд аюулгүй цэвэрлэгээ хийгдэж, дараахыг гүйцэтгэнэ:

- **Аюултай атрибутуудыг устгах** — event handler, script болон болзошгүй аюултай атрибутуудыг арилгана
- **Нэрийн орон зайг хадгалах** — зөв дүрслэхийн тулд SVG нэрийн орон зайг (xmlns, xlink) хадгална
- **Бүтцийг баталгаажуулах** — SVG нь W3C стандарттай нийцэж байгаа эсэхийг баталгаажуулна
- **Entity-г кодлох** — текст агуулга дахь тусгай тэмдэгтүүдийг зөв escape хийнэ
- **Гадаад лавлагааг устгах** — гадаад stylesheet болон зургийн лавлагааг арилгана

Ингэснээр SVG-г нэмэлт цэвэрлэгээ шаардахгүйгээр WordPress-д шууд суулгахад аюулгүй болгоно.

## Ашиглалтын жишээ {#usage-example}

**Prompt:**
```
"CloudSync" нэртэй миний технологийн стартапад зориулж цэнхэр болон цагаан өнгө ашигласан modern logo үүсгэ.
```

**Үр дүн:**
Энэ чадвар нь дараах шинжтэй SVG logo үүсгэнэ:
- "CloudSync" сайтын нэрийг агуулна
- Заасан цэнхэр болон цагаан өнгөний схемийг ашиглана
- Modern дизайны зарчмуудыг дагана
- Автоматаар цэвэрлэгдсэн бөгөөд ашиглахад бэлэн байна

## Theme Builder-тэй нэгтгэх {#integration-with-theme-builder}

Theme Builder-ийн дизайны чиглэл сонгохыг ашиглах үед Generate Logo SVG чадвар нь:

1. Таны дизайны чиглэл болон өнгөний палитрыг шинжилнэ
2. Таны сонголтод тохирсон захиалгат SVG logo үүсгэнэ
3. Logo-г таны сайтын header/брэндийн хэсэгт автоматаар суулгана
4. SVG-г WordPress media-д захиалгат logo болгон хадгална

## Шилдэг туршлагууд {#best-practices}

- **Хэв маягийн сонголтоо тодорхой өгнө үү** — хүсэж буй дизайны хэв маягаа тайлбарлана уу (modern, classic, playful гэх мэт)
- **Өнгөө зааж өгнө үү** — тогтвортой байдлын үүднээс брэндийн өнгөнүүдээ оруулна уу
- **Дүрслэлийг шалгана уу** — logo янз бүрийн дэлгэцийн хэмжээнд зөв харагдаж байгаа эсэхийг баталгаажуулна уу
- **Цааш нь тохируулна уу** — хэмжээ болон байрлалыг өөрчлөхийн тулд WordPress-ийн logo тохируулах хэрэгслүүдийг ашиглана уу

## Хязгаарлалтууд {#limitations}

- SVG logo-нуудыг хөдөлгөөнгүй график хэлбэрээр үүсгэдэг (анимацгүй)
- Олон элементтэй төвөгтэй logo-д гараар нарийвчлан сайжруулалт хийх шаардлагатай байж болно
- Захиалгат фонт дэмжихгүй; текст нь системийн фонтуудыг ашиглана
- Маш том эсвэл маш жижиг хэмжээс нь чанарт нөлөөлж болзошгүй
