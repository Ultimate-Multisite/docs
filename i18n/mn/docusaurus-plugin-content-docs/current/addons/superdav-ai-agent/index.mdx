# Superdav AI Agent

Superdav AI Agent нь Gratis AI Agent-ийг theme-д мэдрэмжтэй ур чадвар, сайжруулсан интернет хайлтын боломжууд, сайт бүтээгчийн функцууд болон сонголтот автоматжуулалтын интеграцуудаар өргөтгөдөг дэвшилтэт WordPress AI туслах юм. 1.18.0 хувилбар нь Google Calendar унших хэрэгслүүд, контакт болон оролцогчийн зураглал, хүний зөвшөөрлийн хаалтууд, сануулгын давхардал арилгалт, TextBee SMS мэдэгдлүүд, удирдлагатай Superdav үйлчилгээний хангалт, мөн итгэмжлэгдсэн хөгжүүлэгчийн ажлын урсгалд зориулан тусад нь түгээдэг Advanced хамтрагч багцыг нэмсэн.

## Гол боломжууд {#key-features}

- **Theme-д мэдрэмжтэй ур чадварууд** — таны идэвхтэй WordPress theme болон суулгасан plugin-уудад үндэслэн зааварчилгаа, боломжуудыг автоматаар тохируулна
- **Олон интернет хайлтын нийлүүлэгч** — илүү баялаг хайлтын үр дүн, дэвшилтэт судалгааны боломжид зориулж Brave Search болон Tavily-гийн хооронд сонгоно
- **Сайт бүтээгчийн контакт маягтууд** — суулгасан маягтын plugin-оо ашиглан чат интерфэйсээс шууд бүрэн ажиллагаатай контакт маягтууд үүсгэнэ
- **Сайжруулсан сайт бүтээгч** — plugin илрүүлэлт болон алдаа сэргэлтийг сайжруулсан олон үе шаттай сайт бүтээх төлөвлөгөөнүүд
- **Суурилуулсан theme ур чадварууд** — Block Themes, Classic Themes, Kadence Blocks, Kadence Theme-д зориулсан төрөлжсөн зааварчилгаа
- **Сайжруулсан WooCommerce интеграц** — илүү найдвартай байдал, нийцэлд зориулж төрөлх WooCommerce API-уудыг ашиглахаар дахин боловсруулсан
- **Нийлүүлэгчийг автоматаар илрүүлэх** — plugin болон theme-үүд идэвхжих эсвэл идэвхгүй болох үед боломжтой plugin болон theme-үүдийг автоматаар илрүүлж шинэчилнэ
- **Захиалгат лого үүсгэх** — автоматаар нэрийн орон зайд аюулгүй цэвэрлэгээтэйгээр захиалгат SVG лого үүсгэж шигтгэнэ
- **Палитрын контраст баталгаажуулалт** — theme-дээ хэрэглэхээс өмнө өнгөний хослолуудыг WCAG хүртээмжийн нийцэлд шалгана
- **Фото зургаар сайжруулсан илрүүлэлт** — дизайны чиглэлд нөлөөлөхийн тулд Theme Builder илрүүлэлтийн ярилцлагын үеэр зураг байршуулна
- **Зочлох үйлчилгээний цэсүүд** — ресторан, кафе болон зочлох үйлчилгээний бизнесүүдэд зориулсан бүтэцлэгдсэн хоол, ундааны цэсийн хуудсууд үүсгэнэ
- **Суурин болон гар утасны урьдчилан харалтууд** — дизайны чиглэл сонгох үед дизайнаа суурин болон гар утасны төхөөрөмж дээр урьдчилан харна
- **1-р түвшний сайт хусалт** — sd-ai-agent/site-scrape нь өрсөлдөгч болон урам зориг өгөх вэбсайтуудыг шинжлэхэд одоо анхдагчаар боломжтой
- **Google Calendar унших хэрэгслүүд** — тохируулсан календарь болон үйл явдлуудыг уншиж, автоматжуулалтууд удахгүй болох хуваарийн контекстийг ойлгон ажиллах боломжтой болгоно
- **Контакт болон оролцогчийн зураглал** — дараагийн ажлыг үүсгэхээс өмнө календарийн оролцогчдыг мэдэгдэж буй сайт хэрэглэгчид болон контактуудтай тааруулна
- **Хүний зөвшөөрлийн хаалтууд** — мессеж илгээх, ажлыг үргэлжлүүлэх эсвэл өөрчлөлт хийхээс өмнө мэдрэмтгий автоматжуулалтуудыг хянахаар түр зогсооно
- **Сануулгын бүртгэлүүд ба давхардал арилгалт** — давтан автоматжуулалтын ажиллуулалтууд ижил хүлээн авагчид хоёр удаа мэдэгдэхгүй байхын тулд аль хэдийн илгээгдсэн сануулгуудыг хянана
- **TextBee SMS мэдэгдлүүд** — SMS нийлүүлэгч идэвхжсэн үед TextBee-ээр дамжуулан тохируулсан текст-мессеж мэдэгдлүүд илгээнэ
- **Удирдлагатай Superdav үйлчилгээний хангалт** — гараар endpoint тохируулахын оронд дэмжигдсэн сайтуудыг автоматаар тохируулгатай байршуулсан Superdav endpoint-уудтай холбоно
- **Advanced хамтрагч багц** — файлын систем, өгөгдлийн сан, WP-CLI, REST dispatcher, plugin бүтээгч, git snapshot, хэрэглэгч-удирдлага болон benchmark ажлын урсгалд зориулсан тусад нь түгээдэг итгэмжлэгдсэн-хөгжүүлэгчийн хэрэгслүүд

## Шаардлагууд {#requirements}

- WordPress 6.0 эсвэл түүнээс дээш
- PHP 8.1 эсвэл түүнээс дээш
- WordPress Multisite (сүлжээгээр идэвхжүүлсэн) эсвэл нэг-сайтын суулгалт
- Идэвхтэй AI нийлүүлэгч холбогч plugin (Anthropic, OpenAI, эсвэл нийцтэй)
- Gratis AI Agent v1.9.0 эсвэл түүнээс дээш (Superdav AI Agent нь Gratis AI Agent-ийг өргөтгөнө)

## Суулгалт {#installation}

1. Plugin файлуудыг `/wp-content/plugins/superdav-ai-agent` руу байршуулна
2. WordPress дахь **Plugins** дэлгэцээр plugin-ийг идэвхжүүлнэ
3. Gratis AI Agent мөн идэвхжсэн эсэхийг баталгаажуулна
4. **Gratis AI Agent → Settings** рүү очиж шаардлагын дагуу интернет хайлтын нийлүүлэгч, удирдлагатай Superdav үйлчилгээний холболт, Google Calendar, эсвэл TextBee итгэмжлэлүүдээ тохируулна

WordPress.org багц нь Superdav AI Agent-ийн үндсэн боломжуудыг агуулдаг. Advanced хамтрагч багцыг итгэмжлэгдсэн хөгжүүлэгчийн хэрэгслүүдийг тодорхой суулгаж, зөвшөөрсөн орчинд зориулан тусад нь бүтээж түгээдэг.

## Theme-д мэдрэмжтэй ур чадварууд {#theme-aware-skills}

Superdav AI Agent нь таны идэвхтэй theme болон суулгасан plugin-уудыг автоматаар илрүүлж, тохирох theme-д мэдрэмжтэй ур чадварыг туслахын контекстэд ачаална. Боломжтой ур чадварууд нь:

| Ур чадвар | Хамаарах нь |
|---|---|
| **Block Themes** | WordPress-ийн блок-д суурилсан (Бүрэн сайт засварлах) архитектурыг ашигладаг theme-үүд |
| **Classic Themes** | PHP загварууд болон классик засварлагч ашигладаг уламжлалт WordPress theme-үүд |
| **Kadence Blocks** | Kadence Blocks plugin идэвхтэй сайтууд |
| **Kadence Theme** | Идэвхтэй theme нь Kadence theme байгаа сайтууд |

Ур чадвар бүрийн дэлгэрэнгүй зааварчилгааг [Theme-д мэдрэмжтэй ур чадварууд](../../user-guide/configuration/theme-aware-skills)-аас үзнэ үү.

## Интернет хайлтын нийлүүлэгчид {#internet-search-providers}

Superdav AI Agent v1.10.0 нь интернет хайлтын асуулгуудад Brave Search-ийн өөр хувилбар болгон **Tavily**-г нэмдэг. Хоёр нийлүүлэгч хоёулаа бодит цагийн хайлтын үр дүн санал болгодог бөгөөд **Gratis AI Agent → Settings → Advanced** дотор тохируулж болно.

| Үйлчилгээ үзүүлэгч | Онцлогууд | Тохиргоо |
|---|---|---|
| **Brave Search** | Хурдан, нууцлалд төвлөрсөн хайлтын үр дүн | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Илүү баялаг хайлтын үр дүн, дэвшилтэт судалгааны боломжууд | [Tavily API](https://tavily.com/) |

Тохиргооны зааврыг [Интернет хайлт](../../user-guide/configuration/internet-search)-аас үзнэ үү.

## Хуанли, зөвшөөрөл болон мэдэгдлийн ажлын урсгалууд {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 нь автоматжуулалтын ажлын урсгалуудыг контент болон сайт бүтээх даалгавраас цааш өргөжүүлнэ:

- **Google Calendar хэрэгслүүд** нь хуанлийн нэвтрэх мэдээлэл боломжтой үед агентэд тохируулсан хуанли болон үйл явдлын дэлгэрэнгүй мэдээллийг унших боломж олгоно.
- **Холбоо барих болон оролцогчийн туслахууд** нь сануулга эсвэл дараагийн даалгавар бэлтгэхээс өмнө үйл явдлын оролцогчдыг WordPress хэрэглэгчид эсвэл танигдсан холбоо барих хүмүүстэй тааруулна.
- **Хүний зөвшөөрлийн хаалтууд** нь эрх бүхий хэрэглэгч санал болгосон үйлдлийг хянах хүртэл автоматжуулалтыг түр зогсоох боломж олгоно.
- **Сануулгын бүртгэлүүд** нь автоматжуулалтыг дахин оролдох эсвэл давтан хуваарьлах үед давхардсан хуанлийн эсвэл мэдэгдлийн сануулга үүсэхээс сэргийлнэ.
- **TextBee SMS дэмжлэг** нь TextBee үйлчилгээ үзүүлэгчийг тохируулсан үед текст мессежийн мэдэгдлийг идэвхжүүлнэ.

Эдгээр хэрэгслийг хуваарьт мэдрэмжтэй туслалцаа, оролцогчдын дараагийн холбоо, мөн эмзэг мессеж илгээхээс өмнө хүний хяналтыг шаардсан хэвээр байх сануулгын ажлын урсгалд ашиглана уу.

## Site Builder холбоо барих маягтууд {#site-builder-contact-forms}

Дэлгэрэнгүй зааврыг [Site Builder холбоо барих маягтууд](../../user-guide/administration/site-builder-contact-forms)-аас үзнэ үү.

### Зураг оруулахтай нээлтийн ярилцлага {#discovery-interview-with-photo-upload}

Theme Builder-ийн нээлтийн ярилцлага одоо зураг оруулах алхамтай болсон бөгөөд та дүрслэлийн урам өгөгч материал өгөх боломжтой. Theme Builder-т таны дизайны чиглэлийг ойлгоход туслахын тулд брэнд, өрсөлдөгчийн сайтууд эсвэл дизайны урам өгөгч зургуудыг байршуулна уу.

Дэлгэрэнгүй зааврыг [Нээлтийн ярилцлага](./theme-builder/discovery-interview.md)-аас үзнэ үү.

Theme Builder таны бүрэн theme-ийг үүсгэхээс өмнө дизайнаа desktop болон гар утасны төхөөрөмж дээр хянана уу. Олон responsive таслах цэг дээр бодит цагийн урьдчилан харалтыг үзнэ үү.

Дэлгэрэнгүй зааврыг [Дизайны чиглэл](./theme-builder/design-direction.md)-аас үзнэ үү.

### Зочлох үйлчилгээний цэсүүд {#hospitality-menus}

Theme Builder нь одоо ресторан, кафе, бар болон бусад зочлох үйлчилгээний бизнесүүдэд зориулсан бүтэцтэй хоол, ундааны цэсийн хуудсуудыг үүсгэж чадна. Цэсүүдэд үнэ, тайлбар, хоолны дэглэмийн мэдээлэл болон харшил үүсгэгчийн анхааруулга багтана.

Дэлгэрэнгүй зааврыг [Зочлох үйлчилгээний цэсүүд](./theme-builder/hospitality-menus.md)-аас үзнэ үү.

- **[Палитрын контрастыг баталгаажуулах](./abilities/validate-palette-contrast.md)** — WCAG-д нийцэж буй эсэхийг өнгөний хослолуудаар шалгана

Боломжтой хэрэгслүүд болон Tier 1 site-scrape хэрэгслийн тухай мэдээллийг [Хэрэгслүүд ба тохиргоо](./theme-builder/tools.md)-оос үзнэ үү.

## Тохиргоо {#configuration}

Бүх тохиргоо WordPress админ дахь **Gratis AI Agent → Settings** доор байна. Superdav AI Agent дараах тохиргооны сонголтуудыг нэмнэ:

| Тохиргоо | Тайлбар |
|---|---|
| **Internet Search Provider** | Brave Search болон Tavily-ийн хооронд сонгоно (заавал биш) |
| **Tavily API Key** | Tavily интернет хайлтад зориулсан API түлхүүр (Tavily сонгосон бол) |
| **Managed Superdav Service** | Дэмжигдсэн сайтуудыг автоматаар endpoint хангалттай hosted Superdav үйлчилгээнд холбоно |
| **Google Calendar Credentials** | Агент хуанли болон үйл явдлыг уншиж чадахаар хуанлийн хандалтыг тохируулна |
| **TextBee SMS Provider** | Зөвшөөрөгдсөн SMS мэдэгдлийн ажлын урсгалд зориулж TextBee нэвтрэх мэдээллийг тохируулна |

## Дэвшилтэт хамтрагч багц {#advanced-companion-package}

Superdav AI Agent v1.18.0 нь итгэмжлэгдсэн хөгжүүлэгчийн ажлын урсгалд зориулсан тусад нь түгээгддэг Advanced хамтрагч багцыг нэмнэ. Эдгээр боломжууд WordPress.org үндсэн багцад багтаагүй. Зөвхөн администраторууд агентэд хөгжүүлэгчийн хэрэгсэлд хандахыг илт итгэдэг сайтууд дээр хамтрагч багцыг суулгана уу.

Advanced хэрэгслүүдэд файлын системийн үйлдлүүд, өгөгдлийн сангийн хандалт, WP-CLI, REST dispatcher ажлын урсгалууд, plugin builder хэрэгслүүд, git агшин, хэрэглэгчийн удирдлагын туслахууд болон benchmark хэрэгслүүд багтана. Эдгээр хэрэгслийг итгэмжлэгдсэн администраторуудаар хязгаарлаж, үйлдвэрлэлийн сайтууд дээр идэвхжүүлэхээс өмнө зөвшөөрлийн хаалтын тохиргоог хянана уу.
