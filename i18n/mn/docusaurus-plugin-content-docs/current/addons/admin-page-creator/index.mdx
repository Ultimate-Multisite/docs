# Admin Page Creator {#admin-page-creator}

## Агуулга {#content}
Admin page creator нь WordPress админ Dashboard дотор шинэ админ хуудас болон дэд хуудсуудыг хялбархан нэмэх боломжийг олгоно. Үүнийг шинэ хэрэглэгчдэд зориулсан эхлэх хуудас болж үйлчлэх тавтай морилно уу хуудас эсвэл зааварчилгааны хуудас үүсгэхэд ашиглаж болно. Та Beaver Builder, Elementor, Brizy, Oxygen зэрэг page builder ашиглан хуудсуудыг төвөггүй загварчилж болно.

Мөн custom HTML, CSS болон JavaScript дэмжлэг, түүнчлэн plan болон хэрэглэгчийн role-д суурилсан хандалтын хяналтыг санал болгодог!

## Боломжууд {#features}
* Custom HTML болон CSS дэмжлэг
* JavaScript дэмжлэг
* PHP агуулгын горим
* Хэрэглэгчийн role-д суурилсан зөвшөөрөл
* Page builder-үүдийн дэмжлэг

## Суулгалт {#installation}

1. **ultimate-multisite-admin-page-creator**-ийг **/wp-content/plugins/** директорт байршуулна уу
2. WordPress сүлжээний админыхаа **Plugins** цэсээр дамжуулан plugin-ийг идэвхжүүлнэ үү

## Ашиглах заавар {#usage-guide}

WordPress нь сайтуудын урд талд агуулга нэмэхийг үнэхээр хялбар болгодог ч админ талд хуудас болон дэд хуудас үүсгэхэд зарим үндсэн код бичих ур чадвар шаардлагатай. Ultimate Multisite: Admin Page Creator нь таны сүлжээний сайтуудын арын талд урд талынхтай ижил хэрэглээний хялбар байдлыг авчрах зорилготой.

## Яагаад Admin Page Creator ашиглах вэ? {#why-use-admin-page-creator}

## Цэсний сонголтууд {#menu-options}

Хажуу талд, General Options meta-box дээр та цэсний төрөл, шошго болон бусад олон зүйлийг тохируулах боломжтой.

### Цэсний төрлүүд {#menu-types}

Таны шинэ custom админ хуудас хоёр өөр цэсний төрөлтэй байж болно: дээд түвшний цэс болон дэд цэс.

Дээд түвшний цэсний хуудас нь нэрнээсээ харахад таны шинээр үүсгэсэн хуудсыг анхдагч WordPress хуудсуудын хамт админ цэсэнд нэмнэ.

Хэрэв та дээд түвшний сонголтыг сонговол цэсний дараалал болон цэсний icon-ыг мөн сонгох боломжтой.

Хэрэв та энэ хуудсыг дэд цэсний хуудас болгохыг хүсвэл энэ хуудсыг холбох parent хуудсыг сонгох боломжтой. Та custom хуудсаа анхдагч WordPress хуудсууд эсвэл өөрийн дээд түвшний custom админ хуудсууд дээр нэмж болно.

## Дэвшилтэт сонголтууд {#advanced-options}

Та custom админ хуудсууддаа бүх төрлийн агуулга нэмэх боломжтой байх ёстой. Үүнд CSS дүрэм болон файлууд, JavaScript, мөн гадаад сангууд орно. Та үүнийг Advanced Options табыг ашиглан хийж болно.

### Custom CSS {#custom-css}

CSS таб дээр та өөрийн custom CSS кодыг нэмэхээс гадна алсын сайтуудаас CSS файлууд импортлох боломжтой (CDN, Google Fonts гэх мэт).

JavaScript-ийн хувьд ч мөн адил. Та custom JavaScript коддоо ашиглах гадаад сангуудыг нэмж болно.

_Өөрийн JavaScript кодыг нэмээрэй!_

## Зөвшөөрлүүд {#permissions}

Мөн Advanced Options дотор та аль дэд сайтууд энэ хуудсанд хандах эрхтэй байхыг сонгох боломжтой. Жишээлбэл, та өөр өөр plan шатлалуудад өөр өөр онцгой агуулга хүргэхээр төлөвлөж байгаа бол энэ нь хэрэгтэй.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Энэ хуудас хаана харагдахыг та бүрэн хянах боломжтой_

**Чухал** : Хуудсыг харуулахын тулд хоёр нөхцөл хоёулаа хангагдсан байх ёстой. Энэ нь хэрэв хэрэглэгч тухайн plan-ы үйлчлүүлэгч боловч сонгосон үүргүүдийн аль нэгийг эзэмшдэггүй (эсвэл эсрэгээрээ) бол хуудас тэдний самбар дээр харагдахгүй гэсэн үг.

## Идэвхтэй {#active}

Хэрэв та тусгай админ хуудсыг устгалгүйгээр эсвэл зөвшөөрлүүдийг өөрчлөлгүйгээр идэвхгүй болгохыг хүсвэл Active сонголтын тэмдэглэгээг зүгээр л авна уу.

Дээд түвшний хуудсыг идэвхгүй болговол тухайн эх хуудсанд холбогдсон бүх дэд хуудсыг автоматаар идэвхгүй болгоно.
