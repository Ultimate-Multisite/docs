# Filter: wu_apc_should_display_admin_menu

Тохиргооны үйл явцыг дуусгасан бол админуудад админ цэсийн хуудсуудыг нуух боломж олгоно.

## Параметрүүд {#parameters}

| Нэр | Төрөл | Тайлбар |
|------|------|-------------|
| $should_display_admin_menu | `bool` | Хуудсуудыг харуулах шаардлагатай эсэх. |
| $network_active | `bool` | plugin сүлжээний админд идэвхтэй эсэхийг шалгана. |
