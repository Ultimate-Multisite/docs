# Шүүлтүүр: wu_apc_get_meta

Хөгжүүлэгчдэд эцсийн утгуудыг шүүх, мөн шинэ боловсруулалтын тохиолдлууд нэмэх боломж олгоно.

## Параметрүүд {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $default_value | `string` | Үндсэн утга. |
| $meta_type | `string` | Татаж авах мета өгөгдлийн төрөл. |
| $meta_key | `string` | Мета талбарын нэр. |

### Хойш {#since}

- 1.4.0
