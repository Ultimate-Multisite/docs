# Нэмэлт өргөтгөл ба загварын менежер

Multisite Ultimate: Нэмэлт өргөтгөл ба загварын менежер

Суулгасан нэмэлт өргөтгөлүүд болон загваруудын мета мэдээллийг (гарчиг, зохиогч, өнгөц зураг болон тайлбар) засах эсвэл нуух. Хэрэглэгчдэдээ тэдгээрийг шүүх боломж олгохын тулд нэмэлт өргөтгөлүүд болон загваруудад ангилал үүсгэж, хэрэглэгчдийнхээ самбарт гоёмсог захиалгат Нэмэлт өргөтгөлүүд хуудсыг нэмээрэй.

Нэмэлт өргөтгөлийг ашиглахын тулд сүлжээний удирдлага дахь Нэмэлт өргөтгөлүүд эсвэл Загварууд хуудсанд очоод "Нэмэлт өргөтгөл/Загварын мэдээллийг засах" дээр дарна уу.

## Суулгалт {#installation}

1. 'ultimate-multisite-plugin-and-theme-manager'-ийг '/wp-content/plugins/' санд байршуулна уу
2. WordPress Network Admin дахь 'Нэмэлт өргөтгөлүүд' цэсээр дамжуулан нэмэлт өргөтгөлийг идэвхжүүлнэ үү

## Ашиглах гарын авлага {#usage-guide}

Ultimate Multisite: Нэмэлт өргөтгөл ба загварын менежер нь таны сүлжээний хэрэглэгчид сүлжээн дэх нэмэлт өргөтгөл болон загварын сонголтуудыг хэрхэн харж, хэрхэн харилцахыг бүрэн хянах боломж олгодог хүчирхэг **premium** Ultimate Multisite нэмэлт юм.

Энэ нэмэлт нь гурван зорилготой:

  1. Энэ нь нэмэлт өргөтгөлүүд болон загваруудын мета өгөгдлийг бүрэн өөрчлөх боломж олгоно. Үүнд нэр, тайлбар, зохиогчийн нэр, бүр өнгөц зургууд хүртэл багтана!

  2. Энэ нь нэмэлт өргөтгөлүүд болон загваруудыг ангилах боломж олгож, хэрэглэгчдэдээ тэдгээрийг хялбархан эрэмбэлэх боломж олгоно.

  3. Энэ нь хуучин, уйтгартай нэмэлт өргөтгөлүүдийн хуудсыг ангиллаар эрэмбэлэх боломжтой, загвартай төстэй шинэ хуудсаар солих сонголтыг танд олгоно.

Татаж аваад суулгасны дараа Ultimate Multisite тохиргооны хуудсанд шинэ таб - _Нэмэлтийн тохиргоо_ \- харагдана.

_Энд та Ultimate Multisite: Нэмэлт өргөтгөл ба загварын менежерийн нэмсэн глобал сонголтуудыг олно_
