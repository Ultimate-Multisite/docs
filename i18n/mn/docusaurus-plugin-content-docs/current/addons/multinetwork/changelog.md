# Multi-Network өөрчлөлтийн бүртгэл

### 1.3.0 {#130}
* Шинэ: Сүлжээний загвар урьдчилан харагч — худалдан авахаасаа өмнө шууд самбартайгаар сүлжээний загваруудыг үзэж, урьдчилан харах боломжтой.
* Шинэ: Тооцоо хийх үеийн сүлжээний загвар сонгогч — хэрэглэгчид бүртгүүлэх тооцоо хийх урсгалын үеэр сүлжээний загвар сонгох боломжтой.
* Шинэ: Сүлжээ хоорондын magic link SSO — хэрэглэгчид magic-link баталгаажуулалтаар дамжуулан дэд сүлжээнүүдэд саадгүй хандах боломжтой.
* Шинэ: Сүлжээний медиа клонлох — сүлжээнүүдийг загвараас клонлох үед сайтын медиа зөв хуулбарлагдана.
* Засвар: Сүлжээ клонлох нь одоо multi-tenancy нэмэлтгүйгээр ажиллана; клонлосны дараа super admin эрхийг дахин олгоно.
* Засвар: Сүлжээ үүсгэх нь одоо WooCommerce болон бүх төлбөрийн гарцуудтай нийцтэй.
* Засвар: Нөөц домэйн үүсгэхэд одоо портын дугаарыг багтаана.
* Засвар: Бүх theme дээр зөв дүрслэхийн тулд загвар сонгогчийн CSS-ийг дараалалд оруулсан stylesheet рүү шилжүүлсэн.
* Засвар: Сайтын асуулгын хамрах хүрээнд тодорхойлогдоогүй `network_id` түлхүүрээс хамгаална.
* Засвар: Контекст солихын өмнөх сүлжээний зөвшөөрлийн шалгалт нь `wpdb` хүснэгтийн шинж чанарын алдаанаас сэргийлнэ.
