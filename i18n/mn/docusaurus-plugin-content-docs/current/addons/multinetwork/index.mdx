# Multi-Network

Ultimate Multisite: Multinetwork addon нь таны Ultimate Multisite платформыг хүчирхэг multinetwork функцээр өргөтгөнө. Энэ addon нь таны хэрэглэгчдэд зөвхөн тусдаа сайтууд биш, бүхэл бүтэн WordPress multisite сүлжээг худалдан авч удирдах боломж олгоно.

## Гол боломжууд {#key-features}

* **Сүлжээ үүсгэн бэлтгэх**: Таны хэрэглэгчдэд зориулж бүрэн WordPress multisite сүлжээг автоматаар үүсгэж тохируулна
* **Сүлжээний загварууд**: Бүртгүүлэхээс өмнө хэрэглэгчдэд урьдчилан бэлтгэсэн сүлжээний загварыг үзэх, урьдчилан харах, сонгох боломж олгоно
* **Сүлжээний удирдлага**: Multinetwork орчны бүрэн удирдлагын боломжууд
* **Тусгаарлагдсан сүлжээнүүд**: Хэрэглэгч бүрийн сүлжээ өөрийн өгөгдлийн сангийн хүснэгтүүд болон тохиргоотойгоор бие даан ажиллана
* **Сүлжээ хоорондын SSO**: Magic-link баталгаажуулалтаар хэрэглэгчдэд дэд сүлжээнүүддээ саадгүй нэвтрэх боломж олгоно
* **Өргөтгөх боломжтой архитектур**: Нөөцийн хуваарилалтыг оновчилсноор олон сүлжээг үр дүнтэй ажиллуулна
* **Нэгдэл**: Ultimate Multisite-ийн subscription болон төлбөр тооцооны системүүдтэй саадгүй нэгтгэгдэнэ

## Хэрэглэх тохиолдлууд {#use-cases}

* **White Label Hosting**: Хэрэглэгчдэд өөрийн брэндтэй multisite сүлжээг олгоно
* **Агентлагийн шийдлүүд**: Агентлагуудад олон клиент сүлжээг нэг платформоос удирдах боломж олгоно
* **Байгууллагын нэвтрүүлэлтүүд**: Нарийн төвөгтэй олон сүлжээний шаардлагатай томоохон байгууллагуудыг дэмжинэ
* **Боловсролын платформууд**: Өөр өөр сургууль эсвэл хэлтэст зориулж тусдаа сүлжээ үүсгэнэ

## Шаардлагууд {#requirements}

* Ultimate Multisite 2.0.0 эсвэл түүнээс дээш
* WordPress Multisite суулгац
* PHP 7.4 эсвэл түүнээс дээш
* MySQL 5.6 эсвэл түүнээс дээш

## Суулгалт {#installation}

1. Plugin файлуудыг `/wp-content/plugins/ultimate-multisite-multinetwork` директорт байршуулна
2. WordPress доторх 'Plugins' дэлгэцээр plugin-ийг идэвхжүүлнэ
3. Multinetwork тохиргоог Ultimate Multisite → Settings хэсгээр тохируулна

## Сүлжээний загварууд {#network-templates}

1.3.0 хувилбар нь шинэ сүлжээ худалдан авч буй хэрэглэгчдэд зориулсан загвар илрүүлэх болон сонгох урсгалыг нэмнэ. Шинэ хэрэглэгч бүрийн сүлжээг бэлтгэсэн бүтэц, дизайн, контентын сан, эсвэл тодорхой салбарт зориулсан тохиргооноос эхлүүлэхийг хүсвэл сүлжээний загваруудыг ашиглана уу.

### Загварын урьдчилан харагч {#template-previewer}

Сүлжээний загварын урьдчилан харагч нь хэрэглэгчдэд checkout-оос өмнө боломжтой загваруудыг үзэх боломж олгоно. Загвар бүрийг шууд урьдчилан харах самбарт нээж болох тул хэрэглэгчид сүлжээ худалдан авахаасаа өмнө байрлал болон контентыг харьцуулж чадна.

Урьдчилан харагчийг хэрэглэгчдэд дараахад туслахын тулд ашиглана уу:

* Боломжтой сүлжээний загвар бүрийн харагдах байдал болон бүтцийг шалгах
* Худалдан авах урсгалаас гаралгүйгээр олон загварыг харьцуулах
* Бүртгүүлэхээс өмнө өөрийн хэрэглэх тохиолдолд хамгийн сайн тохирох загварыг сонгох

Загварын нэр болон тайлбарыг хэрэглэгчдэд ойлгомжтой байлгаарай. Хэрэглэгчид агентлагийн сүлжээ, сургуулийн сүлжээ, франчайзын сүлжээ, эсвэл white-label hosting багц зэрэг зориулалтын хэрэглэх тохиолдлыг ойлгож чадвал урьдчилан харагч хамгийн ашигтай байдаг.

### Checkout дээрх загвар сонгогч {#template-picker-at-checkout}

Загвар сонгогчийг бүртгүүлэх checkout урсгалд field хэлбэрээр нэмж болно. Тохируулсан үед хэрэглэгчид checkout хийх явцдаа хүссэн сүлжээний загвараа сонгох бөгөөд сонгосон загварыг тэдний шинэ сүлжээг үүсгэн бэлтгэхэд ашиглана.

Сонгогчийг checkout урсгалд ашиглахын тулд:

1. Загвар болгон санал болгох сүлжээнүүдээ бэлтгэнэ.
2. Холбогдох бүртгүүлэх эсвэл checkout маягтад Multinetwork загвар сонгох field-ийг идэвхжүүлнэ.
3. Тухайн checkout урсгалд аль загварууд боломжтой байхыг сонгоно.
4. Checkout урсгалыг нийтэлж, сонгосон загвар шинэ сүлжээнд хэрэглэгдэж байгааг баталгаажуулахын тулд бүртгэлийг туршина.

Хэрэв checkout урсгал зөвхөн нэг сүлжээний бүтээгдэхүүн санал болгож байвал сонгогч нь чиглүүлсэн эхлүүлэх алхам хэлбэрээр мөн ашигтай байж болно. Олон plan-тай funnel-уудад зөвхөн сонгосон бүтээгдэхүүн эсвэл plan-тай нийцтэй загваруудыг харуулна уу.

## Сүлжээ хоорондын Magic Link SSO {#cross-network-magic-link-sso}

Сүлжээ хоорондын magic link SSO нь хэрэглэгчдэд үндсэн Account хэсэг болон дэд сүлжээнүүдийн хооронд дахин гараар нэвтрэхгүйгээр шилжих боломж олгоно. Хэрэглэгч сүлжээ хоорондын нэвтрэх холбоосоор ороход Ultimate Multisite нь хугацааны хязгаартай magic link үүсгэж, хэрэглэгчийг зорьсон сүлжээнд баталгаажуулна.

Энэ нь нэгээс олон сүлжээ удирддаг эсвэл худалдан авалтын дараа шинээр хувилсан загварын сүлжээнд шууд нэвтрэх шаардлагатай хэрэглэгчдийн туршлагыг сайжруулна.

Админы шалгах жагсаалт:

* Multinetwork addon WordPress multisite суулгац даяар идэвхтэй эсэхийг баталгаажуулна.
* Хэрэглэгчдэд сүлжээ хоорондын холбоос санал болгохоос өмнө зөв сүлжээний хандалттай эсэхийг шалгана.
* Magic link-үүд төлөвлөсөн зорилтот сүлжээ рүү шийдэгдэхийн тулд сүлжээний домэйн болон mapped домэйнүүдийг зөв тохируулсан байлгана.
* Сүлжээ, домэйн, эсвэл checkout тохиргоог өөрчилсний дараа хэрэглэгчийн Dashboard болон checkout-ийн дараах нэвтрэх товчлууруудыг туршина.

Magic link-үүдийг нэвтрэх холбоос гэж үзэх ёстой. Тэдгээр нь хэрэглэгчийг саадгүй нэвтрүүлэхээр зориулагдсан бөгөөд төлөвлөсөн session-оос гадуур дамжуулах эсвэл дахин ашиглах ёсгүй.

## Түгээмэл асуултууд {#frequently-asked-questions}

### Multinetwork гэж юу вэ? {#what-is-multinetwork}

Multinetwork нь нэг WordPress суулгац дээр олон бие даасан multisite сүлжээ ажиллуулах боломж олгодог WordPress функц юм.

### Энэ нь энгийн multisite-аас юугаараа ялгаатай вэ? {#how-is-this-different-from-regular-multisite}

Энгийн multisite нь нэг сүлжээ дотор олон сайт ажиллуулах боломж олгодог. Multinetwork нь тус бүр өөрийн сайтуудтай олон сүлжээг бүгдийг нь нэг WordPress суулгац дотор ажиллуулах боломж олгодог.

### Үүнд тусгай серверийн тохиргоо шаардлагатай юу? {#does-this-require-special-server-configuration}

Тийм, multinetwork-д тодорхой серверийн setup болон тохиргоо шаардлагатай. Дэлгэрэнгүй setup зааврыг баримт бичгээс үзнэ үү.
