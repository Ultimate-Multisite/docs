# Шүүлтүүр: networks_pre_user_is_network_admin

Хэрэглэгчийн администратор болсон сүлжээнүүдийг шүүж, процессыг богиносгон тасална.

## Параметрүүд

| Нэр | Төрөл | Тайлбар |
|------|------|-------------|
| $ | `array\|bool\|null` | Сүлжээний ID-уудын жагсаалт эсвэл false. null-ээс бусад аливаа утга процессыг богиносгон тасална. |
| $ | `int` | Сүлжээнүүдийг буцаах ёстой хэрэглэгчийн ID. |
