# Saringan: wu_post_count_statuses

Ngidinan pamekar plugin ngarobah status tulisan mana anu kudu diitung. Sacara standar, tulisan anu diterbitkeun jeung pribadi diitung

## Parameter {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $post_status | `array` | Daptar status tulisan |
| $post_type | `string` | Slug jinis tulisan |

### Ti saprak {#since}

- 1.9.1
### Sumber {#source}

Ditetepkeun dina [`inc/limitations/class-limit-post-types.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/limitations/class-limit-post-types.php#L119) dina baris 119


## Mulangkeun {#returns}
Array anyar tina status tulisan
