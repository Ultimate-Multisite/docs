# Saringan: wu_minimum_password_strength

Saring kakuatan sandi minimum anu diperlukeun (skor zxcvbn).

Tingkat kakuatan: - 0, 1: Pohara lemah - 2: Lemah - 3: Sedeng - 4: Kuat (standar)

## Parameter {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $min_strength | `int` | Tingkat kakuatan minimum anu diperlukeun. |
| $strength_setting | `string` | Nilai setélan admin (medium, strong, super_strong). |
