# Filter: wu_magic_link_enforce_user_agent {#filter-wumagiclinkenforceuseragent}

Filter naha rék maksa verifikasi user agent.

Setel ka false pikeun ngidinan token jalan dina browser/alat anu béda. Ieu ngurangan kaamanan tapi ningkatkeun kagunaan.

## Parameter {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $enforce | `bool` | Naha rék maksa nyocogkeun user agent. |
