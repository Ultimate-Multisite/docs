# Saringan: wu_get_checkout_variables {#filter-wugetcheckoutvariables}

Ngidinan pamekar pangaya pikeun nyaring setélan awal kaca pamayaran.

Ati-ati, konci anu leungit bisa ngarusak lengkep pamayaran dina front-end.

## Parameter {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $variables | `array` | Variabel anu dilokalkeun. |
| $checkout | `\Checkout` | Kelas pamayaran. |

### Ti mimiti {#since}

- 2.0.0
### Sumber {#source}

Ditetepkeun dina [`inc/checkout/class-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout.php#L1970) dina garis 1970


## Mulangkeun {#returns}
Array variabel anyar.
