# Filter: wu_view_override_replaceable_views

Ngidinan pamekar nambahkeun polder tambahan kana daptar anu bisa diganti.

Ati-ati, sabab ngidinan polder tambahan bisa nyababkeun salinan anu geus teu mutakhir dimuat tibatan vérsi Ultimate Multisite.

## Parameter

| Name | Type | Description |
|------|------|-------------|
| $replaceable_views | `array` | Daptar polder anu diidinan. |

### Ti saprak

- 2.0.0
### Sumber

Ditetepkeun dina [`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L50) dina garis 50
