# Filter: wu_independent_billing_cycle_product_types

Filter jinis produk anu boga siklus tagihan mandiri.

Produk anu boga siklus tagihan mandiri henteu dipaksa pikeun nyocogkeun jeung période tagihan plan dina cart. Ieu gunana pikeun produk saperti pendaptaran domain anu salawasna ditagih taunan, teu paduli naha plan bulanan atawa taunan.

## Parameter

| Ngaran | Jinis | Pedaran |
|------|------|-------------|
| $types | `array` | Array tina jinis produk anu boga siklus tagihan mandiri. |

- 2.5.0
