# Filter: wu_get_site_domain_and_path

Ngidinan pamekar pikeun ngarobah pasangan domain/path.

Ieu bisa mangpaat pikeun sababaraha hal, saperti nerapkeun hiji rupa solusi staging, server anu béda, jst.

## Paraméter

| Ngaran | Tipe | Pedaran |
|------|------|-------------|
| $d | `object` | Object ayeuna anu ngandung konci domain jeung path. |
| $path_or_subdomain | `string` | Path/subdomain aslina anu dipasrahkeun ka fungsi. |

### Ti mimiti

- 2.0.0
### Sumber

Ditetepkeun dina [`inc/functions/site.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/site.php#L235) dina baris 235


## Balikan
Object anu ngandung konci domain jeung path.
