# Saringan: wu_sso_enabled

Aktipkeun/nonaktipkeun kamampuhan asup sakali lintas-domain.

Saring nilai ieu pikeun mareuman asup sakali sacara lengkep, atawa aktipkeun sacara kondisional gantina.

## Paraméter {#parameters}

| Ngaran | Tipe | Déskripsi |
|------|------|-------------|
| $enabled | `bool` | Naha SSO kudu diaktipkeun? True pikeun hurung, false-ish pikeun pareum. |

### Ti saprak {#since}

- 2.0.11
### Sumber {#source}

Ditetepkeun dina [`inc/sso/class-sso.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/sso/class-sso.php#L110) dina baris 110


## Balikan {#returns}
Naha SSO diaktipkeun atawa henteu.
