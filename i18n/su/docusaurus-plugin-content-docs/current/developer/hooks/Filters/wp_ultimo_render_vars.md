# Saringan: wp_ultimo_render_vars {#filter-wpultimorendervars}

Ngidinan pamekar pangaya pikeun nambahkeun variabel tambahan kana kontéks render sacara global.

## Paraméter {#parameters}

| Ngaran | Tipe | Pedaran |
|------|------|-------------|
| $args | `array` | Array anu ngandung variabel anu diteruskeun ku panggero render. |
| $view | `string` | Ngaran pintonan anu rék dirender. |
| $default_view | `string` | Ngaran fallback_view |

### Ti saprak {#since}

- 2.0.0
### Sumber {#source}
