# Dokumentasi Ultimate Multisite

Wilujeng sumping kana dokumentasi Ultimate Multisite. Ultimate Multisite ngarobah WordPress Multisite jadi platform Website as a Service (WaaS) lengkep.

## Bagian Dokumentasi

### [Pituduh Pamaké](/user-guide/getting-started/ultimate-multisite-101)

Sagala nu anjeun peryogikeun pikeun nyetél, ngonfigurasi, jeung ngatur platform WaaS anjeun. Ngawengku pamasangan, gateway pamayaran, pemetaan domain, pangaturan klien, jeung sajabana.

### [Dokumentasi Developer](/developer/hooks)

Rujukan téknis pikeun developer anu ngawangun dina Ultimate Multisite. Ngawengku rujukan hooks, dokumentasi REST API, jeung pituduh pamekaran addon.

### [Addons](/addons)

Dokumentasi pikeun plugin addon mayar anu ngalegaan Ultimate Multisite ku fitur tambahan saperti integrasi WooCommerce, analytics, ngajual domain, jeung sajabana.
