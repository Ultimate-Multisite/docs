# Ngelajengkeun Panel Admin

Ultimate Multisite téh hoyong jadi saepsonéna gampang dipikawanoh je teu perlu loba penjelasan, tapi sabab alatna kuat pisan, bisa dibantu ku tur panduan. Hayu urang jalan-jalan di halaman admin sangkan anjeun bet nyaman.

## Dashboard {#dashboard}

**Dashboard** Ultimate Multisite nunjukkeun laporan dasar je analisis, kaasup pendapatan, aktivitas situs, pertumbuhan anggota, jumlah pengunjung, je data geografis.

## Checkout Forms {#checkout-forms}

**Checkout Forms** ngahajukeun anjeun kasempetan pikeun nyieun halaman registrasi (pendaftaran) anu disesuaikan. Anjeun bisa ngarobah kolom, masukan kelas je skrip khusus, sarta ngabatas akses dumasar kaayaan nagara atawa lokasi pengunjung.

![Checkout Forms list](/img/admin/checkout-forms-list.png)

## Products {#products}

Bagian **Products** téh tempat anjeun nyieun rencana (plans), paket, je jasa-jasa béda pikeun jaringan anjeun. Tentukan harga, frekuensi tagihan, sarta batasan je kuota dina tingkat produkna.

![Products list](/img/admin/products-list.png)

## Memberships {#memberships}

Halaman **Memberships** nunjukkeun sadaya langganan (subscriptions) di jaringan anjeun. Tinjau je émutan detail langganan kaasup rencana, produk, jumlah tagihan je frekuensi, riwayat pembayaran, sarta waktu tina kajadianna.

![Memberships list](/img/admin/memberships-list.png)

## Payments {#payments}

Halaman **Payments** ngajak anjeun ningali riwayat pembayaran di sakuliah jaringan anjeun, kalayan informasi transaksi anu detail kaasup produk je jumlahna.

![Payments list](/img/admin/payments-list.png)

## Customers {#customers}

Halaman **Customers** nunjukkeun sadaya anggota anu terdaftar dina jaringan anjeun kalayan informasi dasar pengguna saperti nami, alamat email, waktu login terakhir, sarta tautan "switch to" pikeun ngalogon ka subsite maranéhna.

![Customers list](/img/admin/customers-list.png)

## Sites {#sites}

Halaman **Sites** masebutkeun sadaya subsite dina jaringan anjeun kaasup template situs je situs anu dipilik ku pelanggan. Gampangna ngatur domain anu mapet, batasan & quota, plugin je theme, sarta thumbnail situs.

![Sites list](/img/admin/sites-list.png)

## Domains {#domains}

Halaman **Domains** khusus pikeun domain kustom anu mapet ka subsite. Salaku super admin, anjeun bisa nambahan atawa mapetkeun nama domain kustom sacara manual sarta ningali DNS records je logna.

![Domains list](/img/admin/domains-list.png)

## Discount Codes {#discount-codes}

**Discount Codes** ngajak anjeun nyiptake kode kupon pikeun nawarke diskon ka panganggo akhir (end-user) anjeun. Tentukan nilai diskon je sasaran pengguna atawa kelompok anu spesifik.

![Discount Codes list](/img/admin/discount-codes-list.png)

## Broadcasts {#broadcasts}

**Broadcasts** nyaéta alat pikeun ngaluncurkeun pengumuman atawa ngirim pesan pribadi ka panganggo subsite anjeun. Sasaran kelompok pengguna anu spesifik dumasar kana rencana atawa produk anu maranéhna langganan.

![Broadcasts list](/img/admin/broadcasts-list.png)

## Settings {#settings}

Halaman **Settings** nyaéta tempat anjeun ngatur Ultimate Multisite — pengaturan registrasi, pembayaran, API je webhooks, pemetaan domain, sarta integrasi séjénna.

Pengaturan konektor AI mendaftar kumpulan penyedia OAuth yang didukung saat ini: Anthropic Max, OpenAI ChatGPT/Codex, dan Google AI Pro. Setiap kartu penyedia memungkinkan super admin untuk menghubungkan akun, menyegarkan akun yang tersimpan, menghapus akun berdasarkan email, dan menggunakan *manual OAuth fallback* ketika lingkungan yang diisolasi memblokir pengalihan browser. Opsi pengaturan Cursor Pro telah dihapus dari panel admin.

Akun ChatGPT/Codex mendukung penggunaan alat berbasis konektor di mana operasi mengizinkan alat, jadi alur kerja admin yang bergantung pada operasi yang didukung oleh konektor dapat menggunakan perilaku alat Codex setelah akun OpenAI terhubung.

## Events (Peristiwa) {#events}

Halaman **Events** menyimpan catatan semua peristiwa dan log di jaringan Anda. Ini melacak aktivitas seperti perubahan rencana, pendaftaran, dan pergerakan lainnya — berguna untuk memantau multisite Anda.

![Events list](/img/admin/events-list.png)

## Webhooks (Webhook) {#webhooks}

**Webhooks** memungkinkan Anda mengirim data ke aplikasi lain. Berguna untuk mengirim data dari Ultimate Multisite ke platform seperti Zapier.

![Webhooks list](/img/admin/webhooks-list.png)
