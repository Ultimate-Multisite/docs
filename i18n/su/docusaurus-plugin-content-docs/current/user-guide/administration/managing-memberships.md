# Ngatur Keanggotaan (Managing Memberships)

Di Ultimate Multisite, anjeun bisa ngatur keanggotaan jeung akseskeun sagala langganan di jaringan anjeun. Tinjau jeung edit detail langganan saperti rencana (plans) jeung produk (products), jumlah situs, jumlah tagihan jeung frekuensi, riwayat pembayaran, sarta stempel waktu (timestamps).

Kanggo ngakses halaman ieu, ulah lila ka tab **Memberships** dina menu Ultimate Multisite.

![Memberships list page](/img/admin/memberships-list.png)

Dina halaman Memberships, anjeun bisa ningali daftar pelanggan jeung produk anu patali jeung akun maranéhna. Anjeun bisa ngfilter keanggotaan dumasar statusna: aktif (active), ditunggu (pending), di hold, kadaluwarsa (expired), atawa dibatalkan (cancelled).

Keanggotaan gratis diperlakukan saperti keanggotaan **lifetime** (seumur hidup). Keanggotaan ieu teu meunang tanggal kadaluwarsa jeung moal bakal kadaluwarsa otomatis. Upami rencana gratis kudu eureun sanggeus mangrupa uji coba (trial) atawa periode nu tetep, nganggo produk berbayar anu aya uji coba, batalkan keanggotaan sacara manual, atawa pindah pelanggan ka rencana séjén lamun aksesna rék robah.

Klik kana keanggotaan pikeun ngakses halaman di mana anjeun bisa edit detail keanggotaan — informasi tagihan, batasan situs atawa kunjungan, jeung séjénna.

![Membership edit interface](/img/admin/membership-edit.png)

Ieu mangrupa pandangan lengkap tina halaman edit keanggotaan:

![Membership edit full interface](/img/admin/membership-edit-full.png)

Anjeun ogé bisa nambahan keanggotaan anyar anu patali jeung pengguna anu geus terdaftar di jaringan anjeun. Klik **Add Membership** (Tambah Keanggotaan) sarta atur detailna.

![Add Membership modal](/img/admin/membership-add-modal.png)

Terus klik **Create Membership** (Buat Keanggotaan). Keanggotaan anyar éta bakal muncul dina daftar keanggotaan.
