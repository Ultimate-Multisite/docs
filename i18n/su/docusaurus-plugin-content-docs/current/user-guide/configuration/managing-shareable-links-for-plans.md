# Ngaturkeun Link Shareable Keur Rencana (Plans) {#managing-shareable-links-for-plans}

Ultimate Multisite nawiskeun pilihan pikeun ngabagikeun link rencana langsung ka pelanggan anjeun. Pikeun ngalakukeun éta, mindah ka sabab edit (edit page) salah sahiji rencana anjeun sareng klik tombol _Click to copy the Shareable Link_.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_Di Halaman Edit Rencana (Plan's Edit Page), anjeun tiasa manggihan tombol pikeun nyiptake link shareable._

Perhatike yén mekanisme ieu ngagunake frekuensi tagihan (billing frequency) anu default keur ngawangun linkna. Ieu hartina, lamun tagihan default anjeun nyaéta bulanan, conto mah, di ahirna anjeun bakal meunangkeun link saperti kieu: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Upami anjeun hoyong link shareable default jadi kuwartalan (quarterly), anjeun kudu nambahan 3 di ahirna:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Pikeun milih rencana _premium_ dina jadwal taunan (yearly schedule), anjeun bakal nambahan 1 tuluy kecap:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Urang inget yén anjeun teu tiasa ngan saukur ngagaduhan hiji link pikeun frekuensi tagihan anu béda. Anjeun kedah bagikeun hiji link anu spesifik keur unggal frekuensi tagihan anjeun.

Sakali anjeun ngabagikeun link éta ka pelanggan, éta bakal mamparkeun maranéhna ka langkah kahiji dina alur pendaftar (signup flow) anjeun.
