# Website Klien Anjeun (v2)

Sanggeus daftar di jaringan anjeun, pelanggan bisa akses dashboard maranéhna.

Salian halaman **Account**, di mana maranéhna bisa ningali informasi penting ngeunaan pembayaran, keanggotaan, domain, batasan paket, jeung masih katingali loba deui, maranéhna ogé boga akses ka halaman **My Sites**, di mana maranéhna bisa ningali je ngatur sadaya website maranéhna. Sadayana anu kudu dilakukeun nyaéta klik My Sites dina dashboard maranéhna.

![Link My Sites di dashboard pelanggan](/img/account-page/my-sites-link.png)

Sadaya website maranéhna bakal ditunjukeun, je tombol **Manage** tepat di handap unggal website. Lamun maranéhna klik éta, maranéhna bakal dialihkeun ka **dashboard website spesifik** éta.

Maréhna ogé bisa ningali mana salah sahiji website maranéhna anu jadi **utama** pikeun paket anu ngidinan maranéhna boga leuwih ti hiji website.

![Halaman My Sites nunjukkeun website pelanggan je tombol Manage](/img/account-page/my-sites-page.png)

Ieu pandangan lengkap tina antarmuka edit website:

![Antarmuka edit website lengkap](/img/admin/site-edit-full.png)

Lamun maranéhna hayang ngarobah website utama, maranéhna kudu ka halaman Account ku klik menu **Account**.

![Link menu Account di dashboard pelanggan](/img/account-page/account-menu.png)

Maranéhna ogé bisa ningali je ngedit detail pelanggan:

![Antarmuka edit pelanggan](/img/admin/customer-edit.png)

Terus, maranéhna kudu klik **Change Default Site**.

![Tombol Change Default Site di halaman account](/img/account-page/change-default-site.png)

Sateuacanna aya jendela pop-up. Klik dina daftar drop-down je pilih website mana anu hayang jadi website utama.

![Pilihan dropdown Change Default Site](/img/account-page/change-default-site-dropdown.png)

Terus, cukup klik **Change Default Site** teras urusan maranéhna réngsé!

[Konfirmasi Perubahan Situs Default] ![Confirming the default site change](/img/account-page/change-default-site-confirm.png)
