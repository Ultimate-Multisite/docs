# Nginstalasi Ultimate Multisite

:::note
Tutorial ieu ngasumsike anjeun geus ngagunake WordPress Multisite anu geus diinstal je dikonfigurasi. Pikeun diajar kumaha, cek [tutorial ieu](https://www.wpbeginner.com/glossary/multisite/) ku WP Beginner.
:::

## Nginstal Plugin

Ultimate Multisite gratis diakses di [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/).

Ti **Network Admin Dashboard** anjeun, langsung ka **Plugins → Add New Plugin**.

![Halaman Network Admin Add New Plugin](/img/installation/add-new-plugin.png)

Cari **"Ultimate Multisite"** (teu lupa ku tanda kutip pikeun kecocokanna pas) je éta bakal jadi hasil kahiji. Klik **Install Now**.

![Hasil pencarian nunjukkeun Ultimate Multisite](/img/installation/search-ultimate-multisite.png)

Sanggeus diinstal, klik **Network Activate** pikeun ngaktifkeun plugin éta di sakuliah jaringan anjeun.

![Plugin diinstal kalawan tombol Network Activate](/img/installation/plugin-installed.png)

Sanggeus diaktifkeun, anjeun bakal otomatis diarahkan ka Setup Wizard.

![Plugin diaktifkeun je diarahkan ka wizard](/img/installation/plugin-activated.png)

## Setup Wizard

Setup Wizard bakal ngabimbing anjeun ngatur Ultimate Multisite dina kira-kira 10 menit.

### Welcome

Klik **Get Started** pikeun mimiti.

![Layar welcome Setup Wizard](/img/installation/wizard-welcome.png)

### Pre-install Checks

Langkah ieu bakal ngecek informasi sistem je instalasi WordPress anjeun pikeun mastikeun éta cocog je persyaratan Ultimate Multisite. Lamun sagala hal katingali hadé, klik **Go to the Next Step**.

![Pre-install checks nunjukkeun persyaratan sistem](/img/installation/wizard-pre-install-checks.png)

:::note Tombol Network Activate (v2.6.1+)
Upami Ultimate Multisite geinstal tapi **henteun belum diaktifkeun jadi jaringan** — conto, upami anjeun ngagolek **Activate** (single-site) alih ka **Network Activate** tina layar plugin jaringan — langkah Pre-install Checks bakal ngadeukeutan éta jeung nunjukkeun tombol **Network Activate**.

Ngagolek **Network Activate** bakal ngaktifkeun plugin éta di sakuliah jaringan multisite anjeun sacara otomatis. Nalika geus aktif, wizard bakal lanjut biasa ka langkah instalasi. Anjeun teu kudu ninggalke wizard pikeun ngalaksanakeon status aktivasi.
:::

### Instalasi

Installer bakal nyiptake tabel database anu dipiharep jeung nginstal file `sunrise.php` anu dibutuhkeun ku Ultimate Multisite sangkan bisa ngagancang. Klik **Install** pikeun neraskeun.

![Langkah instalasi nunjukkeun tabel database jeung sunrise.php](/img/installation/wizard-installation.png)

### Perusahaan Anjeun

Isian informasi perusahaan anjeun jeung tetepkeun mata uang default anjeun. Informasi ieu bakal dipaké di sakuliah platform WaaS anjeun. Klik **Continue** nalika geus réngsé.

![Langkah konfigurasi Perusahaan Anjeun](/img/installation/wizard-your-company.png)

### Konten Default

Langkah ieu ngajangkeun anjeun bisa nginstal template, produk, jeung konten awal anu geus ditangtuke. Ieu cara anu saé pikeun kenalan jeung fitur Ultimate Multisite. Klik **Install** pikeun nambahan konten default, atawa leuwihna lewati langkah ieu upami anjeun milih mimiti ti nol.

![Langkah instalasi konten default](/img/installation/wizard-default-content.png)

### Plugin anu Dikawasa (Recommended Plugins)

Upami hoyong, instal plugin pendamping anu dikawasa. Klik **Install** pikeun nambahan éta atawa lewati pikeun neraskeun.

![Langkah plugin anu dikawasa](/img/installation/wizard-recommended-plugins.png)

### Siap!

Alhamdulillah, parantos réngsé! Instalasi Ultimate Multisite anjeun parantos rampung. Ayeuna anjeun tiasa miara (bangun) Website anjeun salaku platform as-a-service tina **Network Admin Dashboard**.

![Setup complete - Ready screen](/img/installation/wizard-ready.png)

![Network Admin Dashboard with Ultimate Multisite active](/img/installation/network-dashboard.png)

Mulai waé, santai waé!
