# Kumaha Gabung ka Komunitas Ultimate Multisite {#how-to-join-ultimate-multisite-community}

Ultimate Multisite Community téh tempat di mana anjeun bisa ngobrol jeung pengguna Ultimate Multisite séjén sarta berbagi tips, pengalaman, jeung hal-hal séjénna. Pikeun akses ka Ultimate Multisite Community, silaturahkeun ka https://community.ultimatemultisite.com/home.

Di halaman sambutan (welcome page), mindeng ka tautan “**creating your account here** ”.

<!-- Screenshot unavailable: Halaman sambutan Komunitas dengan tautan membuat akun -->

Anjeun bakal dialihkan ka halaman registrasi kami. Ayeuna anjeun kudu ngisi username, email, jeung password. Laksanake Captcha sareng klik tombol **Register**.

<!-- Screenshot unavailable: Halaman registrasi Komunitas dengan kolom username, email, password, jeung Captcha -->

Sanggeus registrasi, waktuna kanggo ngadamel profil anjeun. Di halaman ieu, anjeun bisa ngunggah foto, ngisi nama lengkap anjeun, nuliskeun bio sakedap, jsb. Ulah lilaan link media sosial anjeun jeung atur izin (permissions) pikeun akun anjeun.

<!-- Screenshot unavailable: Halaman pembuatan profil Komunitas dengan unggah foto jeung kolom bio -->

Ayeuna anjeun parantos di saluran “Introductions”, di mana anjeun tiasa ngawanohkeun diri sareng kenalan sakedap sareng anggota séjén. Anjeun tiasa damel postingan ku klik tombol **New Post**.

<!-- Screenshot unavailable: Saluran Introductions Komunitas dengan tombol New Post -->

Saluran dina bagian Lounge je saluran “Dev Logs” terbuka kanggo sadayana. Nanging, anjeun bakal butuh izin khusus pikeun akses kana saluran séjén di komunitas.

<!-- Screenshot unavailable: Saluran Komunitas anu nunjukkeun bagian Lounge je saluran Dev Logs -->

Akses ieu bakal dipasihkeun salaku anjeun ngahapkeun kunci lisensi Ultimate Multisite aktif ka kami. Pikeun ngalakukeun éta, klik tautan saluran pribadi mana waé teras klik tombol **Unlock**.

<!-- Screenshot tidak tersedia: Channel komunitas pribadi dengan tombol Unlock -->

Anjeun bakal dialihkan ka halaman akses unlock urang. Masukkan kunci lisensi Ultimate Multisite anjeun terus klik tombol Unlock Access:

<!-- Screenshot tidak tersedia: Halaman akses unlock komunitas dengan input kunci lisensi -->

Lisensi anjeun bakal divalidasi jeung anjeun bakal bisa ngakses channel pribadi. Proses ieu bisa butuh waktu nepi ka 24 jam, jadi klik tautan pikeun balik deui ka komunitas je nikmati tempat lianna samana urang proses permintaan anjeun.

<!-- Screenshot tidak tersedia: Halaman konfirmasi validasi lisensi komunitas -->
