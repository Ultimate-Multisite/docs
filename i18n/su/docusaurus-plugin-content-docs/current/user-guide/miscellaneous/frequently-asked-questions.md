# Pertanyaan yang Sering Diajukan

Ieu sababaraha **Pertanyaan yang Sering Diajukan** anu mindeng datang dina permintaan dukungan.

  * **Naon éta Ultimate Multisite?**

Ultimate Multisite téh plugin WordPress, dikembangkeun pikeun instalasi WordPress multisite, anu ngubah instalasi WordPress anjeun jadi jaringan situs premium – siga [WordPress.com](https://WordPress.com) – di mana klien bisa nyiptake situs ku bayaran bulanan, kuartalan, atawa taunan (anjeun og bisa nyiptake paket Gratis).

  * **Naha Ultimate Multisite bakal jalan tanpa Multisite?**

Teu. Ultimate Multisite mbutuhkeun WordPress multisite, subdirectory, atawa subdomain.

  * **Naon syaratna pikeun Ultimate Multisite?**

Ultimate Multisite mbutuhkeun:

**Versi WordPress** : v5.3+ (Dipikahoyong: Versi Stabil Terakhir)
**Versi PHP** : 7.4.x (Dukungan ka > 8.0 datang deukeut)
**Versi MySQL** : v5+ (Dipikahoyong: Versi 5.6, lamun anjeun teu boga 8.0 di penyedia hosting anjeun)

_Multisite (subdomain atawa subdirectory) og kudu diaktifkeun._ [Kumaha cara Install WordPress Multisite](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **Naon naonna Ultimate Multisite bisa ngabatas plugin jeung tema nu bisa dipaké ku pengguna?**

Iya. Ku Ultimate Multisite anjeun bisa milih plugin jeung tema mana waé anu bisa dipaké pikeun unggal paket di jaringan anjeun.

  * **Naon naonna Ultimate Multisite bisa ngabatas jumlah postingan nu bisa diciptake ku hiji pengguna dina situsna?**

Iya. Dengan Ultimate Multisite, anjeun bisa bikin kuota postingan (post quotas) buat **setiap tipe post** anu aya di jaringan anjeun. Anjeun ogé bisa ngabatas jumlah pengguna admin anu bisa diundang, **jumlah ruang disk**, jeung masih loba deui.

  * **Naha Ultimate Multisite mendukung Site Templates?**

Iya. Dengan Ultimate Multisite, anjeun bisa bikin sabaraha waé Site Template anu dipikacinta. Site Template téh lain tema (theme), tapi éta téh situs biasa di jaringan anjeun. Anjeun bisa ngaktifkeun plugin, tema, jeung ngubahna Site Template éta sakumaha anjeun hayang. Lamun hiji pengguna milih template éta, situsna bakal dibikaké salinan tina template éta, kalayan sakabéan perubahan (customizations) anu geus aya.

Anjeun ogé bisa ngatentukan template tartamtu pikeun hiji rencana (plan). Ku cara éta, unggal pengguna anu daftar dina rencana éta bakal boga situsna dibikaké salinan tina Site Template éta.

  * **Naha Ultimate Multisite nyambung jeung WooCommerce?**

Iya!

Lamun anjeun hayang pengguna jaringan anjeun bisa ngagunake WooCommerce, ngidinan maranéhna ngatur toko (stores), éta bisa dilakukeun ku cara maké Ultimate Multisite. Anjeun ogé bisa ngabatas jumlah produk anu bakal bisa diciptakeun ku klien anjeun dina unggal tingkatan rencana (plan tier).

Salaku pembayaran langganan pikeun Ultimate Multisite, anjeun bisa maké [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) pikeun ngidinan klien Ultimate Multisite anjeun pikeun bayar maké gateway WooCommerce dina situs utama anjeun. Ku cara éta, anjeun bisa ngagunake perpustakaan gedé gateway pembayaran anu dikembangkeun pikeun WooCommerce jeung ngagunake éta pikeun ngitung tagihan ka klien Ultimate Multisite anjeun.

  * **Naha aya cara pikeun migrasi langganan tina WPMUDEV Pro Sites?**

Iya, Ultimate Multisite ayeuna nawiskeun anjeun [alat mantuan pikeun migrasi] Pro Sites network anjeun (https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/).

  * **Naha tab “Allowed Plugins” kosong dina Halaman Edit Plan abdi?**

Ultimate Multisite ngan ukur ngidinan atawa nolak akses ka plugin anu teu diaktifkeun (network-activated) dina network anjeun. Lamun hiji plugin geus diaktifkeun dina network, ku sifatna, éta bakal otomatis dimuat di unggal situs dina network anjeun.

Lamun anjeun teu nempo plugin naon waé dina tab “Allowed Plugins”, cek upami plugin anu dipikahoyong téh geus aktif dina network atawa henteu.

  * **Naha abdi bisa ngagunake desain Tabel Harga (Pricing Table) abdi sorangan?**

Iya. Ti versi 1.4.0, Ultimate Multisite ngajak dukung *template overrides*.

  * **Naha Ultimate Multisite bisa maké jeung WP Engine?**

Iya. Ultimate Multisite salawasna didukung ku WP Engine, tapi pikeun mapping domain jadi jalan butuh langkah tambahan manual ti admin (nambahan domain kana panel WP Engine). **Ieu geus teu kajadian deui ti versi 1.6.0.** Ultimate Multisite ngabangun langsung jeung API WP Engine sarta domain bakal otomatis ditambihan di panel sanggeus mapping.

  * **Provider hosting mana anu anjeun rekomendasikeun?**

Ultimate Multisite bisa dipaké jeung sababaraha provider hosting. Perbedaanana ngan ukur yén sababaraha perusahaan hosting saperti Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane sarta [Runcloud.io](http://Runcloud.io) miboga integrasi mapping domain anu leuwih raket jadi maranéhna geus siap langsung dipaké (out of the box). Pikeun hosting séjén, anjeun mungkin kudu ngatur mapping domain sacara manual.
