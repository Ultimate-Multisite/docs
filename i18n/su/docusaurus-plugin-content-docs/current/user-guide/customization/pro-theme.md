# Ngubah Panel Admin je PRO Theme

Seringkali pas lagi ngerjain kerja klien, kita butuh kasih pengalaman yang lebih keren di dashboard WordPress klien. Munuhunkeun tampilan anu leuwih profesional mah, PRO Theme bisa ngabantosan anjeun.

PRO Theme téh nyaéta tema admin anu dikembangkeun pikeun ngabantosan anjeun nalika ngubah Dashboard klien. Jeung éta ogé bagian tina inti plugin premium Ultimate Multisite.

Sanggeus dipasang, indit ka halaman _Plugins_ jeung klik pikeun ngaktifkeunana di Network anjeun.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Aktifkeun téh sakuliah Network di halaman Plugins._

_**CATATAN:**_ _Lamun anjeun ngan ukur dipasang, éta bakal muncul di klien anjeun salaku pilihan anu bisa ditandai pikeun nambahan tema ngan kanggo user éta._

Ayeuna, antarmuka admin anjeun geus dirancang ulang sacara total! Anjeun ogé boga menu anyar di kolom kirang kalayan ngaran PRO Theme.

<!-- Screenshot unavailable: PRO Theme redesigned admin panel interface -->

_Tampilan anyar pikeun Panel Admin anjeun._

Di menu ieu, anjeun bakal bisa ngakses lima tab:

<!-- Screenshot unavailable: PRO Theme Functionality tab -->

_Functionality_ (Fungsionalitas)

<!-- Screenshot unavailable: PRO Theme Custom CSS tab -->

_Import/Export_ (Impor/Ekspor)

_Aktifkeun PRO Theme_
