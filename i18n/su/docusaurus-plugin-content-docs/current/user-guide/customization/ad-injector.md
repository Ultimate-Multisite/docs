# Nempat Iklan di Jaringan Anjeun je Ultimate Multisite: Ad Injector {#placing-ads-on-your-network-with-ultimate-multisite-ad-injector}

Ultimate Multisite: Ad Injector téh **add-on gratis** anu ngahandurkeun anjeun alat pikeun nempat iklan dina rencana anjeun langsung ti halaman editna.

Sanggeus nginstal Ultimate Multisite: Ad Injector, indituna ka halaman _Plugins_ anjeun pikeun ngaktifkeunana.

<!-- Screenshot teu aya: Halaman WP Plugins nunjukkeun tombol Network Activate pikeun add-on Ad Injector -->

_Klik tombol Network Activate_

Lamun add-on geus aktif, kotak meta _Advanced Options_ dina layar edit rencana bakal aya pilihan anyar nyaéta _Ad Settings_.

![Ad Settings dina layar edit rencana](/img/config/product-edit-full.png)

_Ayeuna anjeun bisa ngedit pengaturan iklan pikeun unggal rencana._

Di tab éta, anjeun bakal bisa nambahan kode iklan pikeun postingan je halaman di bagian depan (front-end), saméméh je sanggeus kontenna.

<!-- Screenshot teu aya: Lapangan kode iklan pikeun postingan je halaman saméméh konten -->

<!-- Screenshot teu aya: Conto iklan anu dipajang dina halaman postingan pikeun rencana tartamtu -->

_Sanggeus ngabéréskeun kode iklan anjeun, éta bakal muncul dina postingan je halaman rencana tartamtu._

Anjeun ogé bisa nyelip iklan dina header Halaman Admin keur pangguna rencana éta.

<!-- Screenshot teu aya: Pilihan Kode Iklan Panel Admin dina pengaturan rencana -->

<!-- Screenshot teu aya: Conto iklan anu dipajang dina panel admin pangguna -->

_Indituna ka pilihan Kode Iklan Panel Admin lamun anjeun hayang nampilkeun iklan dina back-end pangguna._

### Di Luar Iklan {#beyond-advertising}

Anjeun ogé bisa ngagunake Ultimate Multisite: Ad Injector pikeun nawarkeun pangrojong keur rencana anu leuwih luhur (high-tier plans).

Cukup indituna ka pilihan _Admin Panel Ads_ je nambahan kode HTML pikeun alat pangrojong anjeun.

<!-- Screenshot teu aya: Pilihan Admin Panel Ads kalayan kode HTML alat pangrojong -->

<!-- Screenshot teu aya: Widget dukungan pelanggan ditampilkan di panel admin untuk paket tertentu -->

_Tambah dukungan pelanggan ke paket tertentu_

Dengan cara ieu, jalma ti langsung Admin Panelna bisa ngahubung ka anjeun!
