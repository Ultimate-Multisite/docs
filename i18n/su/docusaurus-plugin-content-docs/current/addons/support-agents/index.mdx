# Agén Pangrojong

Ngatur sakabéh jaringan ku sorangan téh lain tugas anu gampang. Nalika jaringan anjeun mimiti tumuwuh, anjeun bakal mimiti ningali paménta pangrojong anu beuki ningkat. Hadé lamun anjeun mawa jalma anu bakal ngabantosan ngurangan beban gawé anjeun. Tapi, méré aksés/idin pinuh téh picilakaeun sarta bisa ngabahayakeun jaringan anjeun. Di dinya add-on Agén Pangrojong ieu dipaké.

Add-on ieu ngamungkinkeun anjeun nyieun jinis pamaké WordPress anyar anu disebut Agén Pangrojong, sarta méré anjeun kadali lengkep kana idin naon waé anu dipiboga ku pamaké éta. Méré idin sagampang ngahurungkeun tombol toggle, sarta mareumanana lamun anjeun mutuskeun rék miceunna.

## Fitur

- Daptar rinci idin WordPress & Ultimate Multisite
- Aktipkeun & nonaktipkeun aksés kalayan gampang
- Jieun jumlah Account Agén tanpa wates

## Pamasangan

1. Unggah 'ultimate-multisite-support-agents' ka diréktori '/wp-content/plugins/'
2. Aktipkeun plugin ngaliwatan ménu 'Plugins' dina WordPress
3. Turutan Wizard léngkah demi léngkah pikeun nyetél plugin éta
