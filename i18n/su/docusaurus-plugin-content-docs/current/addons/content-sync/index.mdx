# Singkronisasi Eusi

Multisite Ultimate: Singkronisasi Eusi

Tambihan ieu pikeun Multisite Ultimate ngamungkinkeun administrator jaringan nyingkronkeun eusi ti situs mana waé ka situs séjén dina jaringan. Ieu ngarojong sagala jenis kiriman kaasup kiriman, kaca, jenis kiriman husus, jeung témplat Elementor. Tambihan ieu ngawengku antarmuka wizard sababaraha léngkah pikeun milih eusi jeung situs tujuan, kalayan pamrosésan batch jeung eupan balik kamajuan sacara real-time.

## Fitur

* Pamilihan Eusi: Grid kartu visual kalayan thumbnail jeung panyaring jenis kiriman
* Sagala Jenis Kiriman: Singkronkeun kiriman, kaca, produk, témplat Elementor, jeung jenis kiriman husus naon waé
* Pamilihan Situs: Pilih situs mana anu rék disingkronkeun eusina, kalayan pangaluaran situs sumber jeung pra-pamilihan situs anak
* Pamrosésan Batch: Nyingkronkeun ka 10 situs sakaligus kalayan apdet kamajuan
* Singkronisasi Média: Otomatis nyalin jeung memetakeun deui berkas média (gambar, vidéo) tina eusi kiriman ka situs tujuan
* Singkronisasi Taksonomi: Ngajaga kategori, tag, jeung taksonomi husus
* Rojongan Elementor: Rojongan pinuh pikeun témplat Elementor kalayan regenerasi CSS jeung pemetaan deui ID témplat
* Résolusi Konflik: Pilihan pikeun ngaganti eusi anu geus aya atawa ngalangkunganana
* Manajemén Cache: Otomatis mersihan cache Elementor dina situs tujuan sanggeus sinkronisasi

## Sarat

* WordPress 5.3 atawa leuwih luhur
* PHP 7.4 atawa leuwih luhur
* Pangaya Multisite Ultimate (aktip)
* Pangaya Elementor (opsional, diperlukeun pikeun sinkronisasi témplat Elementor)

## Pamasangan

1. Unggah 'ultimate-multisite-content-sync' ka diréktori '/wp-content/plugins/'
2. Aktipkeun pangaya ngaliwatan ménu 'Pangaya' dina WordPress Network Admin anjeun
3. Konfigurasikeun setélan dina Ultimate Multisite → Setélan → Singkronisasi Eusi
4. Buka Ultimate Multisite → Singkronisasi Eusi pikeun ngamimitian nyingkronkeun
