# Loco Translate

Tambihan Ultimate Multisite: Loco Translate Integration mawa kamampuh tarjamahan husus-situs ka jaringan multisite WordPress anjeun. Lamun anjeun kungsi peryogi situs anu béda dina jaringan anjeun boga tarjamahan anu béda pikeun plugin atawa téma anu sarua, tambihan ieu mangrupakeun solusi anu sampurna.

**Naon Anu Dilakukeun ku Tambihan Ieu**

Tambihan ieu ngahijikeun Ultimate Multisite sacara mulus jeung plugin Loco Translate anu populér, ngamungkinkeun unggal situs bawahan dina jaringan anjeun ngajaga tarjamahan kustomna sorangan. Boh anjeun ngajalankeun platform multi-wilayah, ngalayanan komunitas basa anu béda, atawa ngan saukur peryogi kalenturan tarjamahan di sakuliah jaringan anjeun, tambihan ieu ngajadikeunana gampang pisan.

**Sampurna Pikeun:**

* Usaha multi-wilayah kalayan kabutuhan lokalisasi anu béda
* Lembaga atikan anu ngalayanan komunitas anu rupa-rupa
* Platform SaaS anu nawarkeun rojongan multibasa
* Agénsi wéb anu ngatur sababaraha situs klien
* Jaringan multisite naon waé anu merlukeun kalenturan tarjamahan

## Pamasangan

1. Pastikeun Ultimate Multisite geus dipasang jeung diaktipkeun dina jaringan
2. Pasang jeung aktipkeun plugin Loco Translate
3. Unggah tambihan ieu ka diréktori plugins anjeun
4. Aktipkeun tambihan Loco Translate Integration dina jaringan
5. Mimitian nyieun tarjamahan husus-situs ngaliwatan Loco Translate dina situs bawahan mana waé

Sakitu waé! Integrasina jalan sacara otomatis tanpa konfigurasi tambahan naon waé.

## Patarosan anu Mindeng Ditanyakeun

## Naha kuring kudu ngonfigurasi nanaon sanggeus pamasangan?

Henteu! Integrasina otomatis. Sanggeus tambihan jeung Loco Translate duanana aktip, situs bawahan bisa langsung mimiti nyieun tarjamahan kustomna sorangan.

## Naon anu kajadian lamun hiji situs bawahan henteu nyieun tarjamahan kustom?

Situs éta bakal sacara otomatis ngagunakeun tarjamahan sakuliah jaringan salaku cadangan. Teu aya anu ruksak, jeung sagalana jalan mulus.

## Naha situs bawahan bisa ngaksés tarjamahan sakuliah jaringan?

Admin situs biasa ngan ningali jeung ngédit tarjamahan situsna sorangan. Super admin bisa ngaksés boh tarjamahan sakuliah jaringan boh tarjamahan husus-situs gumantung kana situs mana anu keur ditempo.

## Naha ieu jalan pikeun plugin jeung téma?

Leres! Tarjamahan husus-situs jalan pikeun plugin jeung téma WordPress.

## Naha ieu bakal ngalambatkeun jaringan multisite kuring?

Henteu. Tambihan ieu ngagunakeun pamuatan berkas anu éfisién anu ngan mariksa tarjamahan husus-situs nalika diperlukeun. Teu aya pangaruh kinerja anu karasa.

## Naha kuring bisa nyalin tarjamahan jaringan ka situs husus salaku titik mimiti?

Leres! Nalika ngédit tarjamahan dina Loco Translate, anjeun bisa gampang ngamimitian ku tarjamahan jaringan tuluy ngustomisasi éta pikeun unggal situs.

## Naha ieu jalan jeung WooCommerce sarta plugin séjén?

Tangtu! Plugin atawa téma naon waé anu ngagunakeun fungsi tarjamahan standar WordPress bakal jalan jeung tarjamahan husus-situs.
