# Material WP

Antarbeungeut admin WordPress geus sarua kénéh salila mangtaun-taun. Tapi naha anjeun kantos panasaran kumaha rupana lamun antarbeungeut admin miboga vérsi Material Design? 

Material WP nyaéta téma admin anu dikembangkeun pikeun masihan anjeun rupa jeung rasa éta, ngabantosan anjeun nyayogikeun Dashboard anu leuwih disaluyukeun pikeun klien anjeun.

## Fitur

- Pangrojong CSS kustom
- Setélan gaya anu gampang
- Nyusun deui ménu
- Impor/ékspor anu gampang

## Dokuméntasi

**Nyaluyukeun Panel Admin anjeun ku Material WP**

## Pamasangan

1. Unggah 'material-wp' ka diréktori '/wp-content/plugins/'
2. Aktipkeun tambihan ngaliwatan ménu 'Tambihan' dina WordPress
