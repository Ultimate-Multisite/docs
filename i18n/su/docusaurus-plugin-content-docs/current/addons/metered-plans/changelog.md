- Perbaikan: Pamajuan tabel database ayeuna leres saluyu jeung définisi skéma BerlinDB, nyegah kagagalan pamajuan dina pamasangan anyar

- Perbaikan: Nilai fraksional ayeuna ditarima dina widang input AI Usage Overage Markup
