# Domain Seller {#domain-seller}

**Multisite Ultimate: Domain Seller** ngarobah jaringan WordPress Multisite anjeun jadi platform jual domain anu kuat. Pasihan konsumén kamampuhan pikeun ngadaptarkeun domain kustom langsung nalika daptar atawa tina Dashboard maranéhna, kalayan pamariksaan kasadiaan sacara real-time jeung harga anu transparan.

### Naha Domain Seller? {#why-domain-seller}

Pendaptaran domain mangrupa salah sahiji upsell anu paling loba dipénta pikeun platform WaaS (Website as a Service) naon waé. Tibatan ngirim konsumén anjeun ka registrar pihak katilu, anjeun tiasa nawarkeun domain sacara langsung — ngajaga leuwih loba panghasilan dina jaringan anjeun bari masihan konsumén pangalaman anu lancar tanpa halangan.

### Fitur Utama {#key-features}

#### Pangrojong Sababaraha Registrar Domain {#multiple-domain-registrar-support}
* **OpenSRS** — Pendaptaran, pembaruan, jeung pangelolaan lengkep
* **Namecheap** — Integrasi lengkep jeung WhoisGuard sarta uji sandbox
* **HostAfrica** — Pilarian domain, harga, pendaptaran, transfer, DNS, jeung panyalindungan ID via Domains Reseller API
* **Openprovider** — Harga reseller, pendaptaran, pembaruan, transfer, DNS, privasi WHOIS, jeung sinkronisasi TLD
* **Hostinger** — Pilarian jeung pendaptaran domain nganggo token integrasi Hostinger anu dibagikeun
* **Arsitéktur anu tiasa dimekarkeun** — Diwangun dina pola Integration Registry, ngajadikeun gampang pikeun nambahkeun panyadia séjén
* **Uji Sambungan Bawaan** — Validasi kredénsial panyadia jeung konektivitas saméméh diluncurkeun

#### Wizard Setup anu Dipandu {#guided-setup-wizard}
* **Onboarding sakali klik** — Nuntun anjeun ngalangkungan konfigurasi panyadia, impor TLD, jeung nyieun produk domain
* **Sinkronisasi TLD otomatis** — Ngimpor sakabéh TLD anu sayogi jeung harga grosir ti panyadia anu nyambung
* **Nyieun produk standar** — Nyieun produk domain siap-paké kalayan standar markup anu asup akal

#### Harga Domain anu Fleksibel {#flexible-domain-pricing}
* **Tilu mode markup** — Markup perséntase, markup tetep, atawa harga tetep per TLD
* **Produk husus TLD** — Jieun produk domain misah pikeun kelompok TLD anu béda (.com/.net, .io, kode nagara, jsb.)
* **Harga bubuka** — Tawarkeun harga taun kahiji anu didiskon pikeun narik pendaptaran anyar
* **Harga real-time** — Harga grosir langsung ti panyadia kalayan caching anu calakan
* **Pangrojong kupon** — Kasaluyuan lengkep jeung kode diskon Multisite Ultimate

#### Panyalindungan Privasi WHOIS {#whois-privacy-protection}
* **Konfigurasi per produk** — Atur privasi WHOIS jadi Dinonaktipkeun, Salawasna Kaasup, atawa Pilihan Konsumén
* **Opt-in konsumén** — Nalika disetel ka Pilihan Konsumén, kotak centang dina prosés pamayaran ngamungkinkeun konsumén nambahkeun panyalindungan privasi
* **Harga anu tiasa dikonfigurasi** — Atur harga per taun pikeun add-on privasi, atawa tawarkeun gratis
* **Integrasi registrar** — Sacara otomatis ngaktipkeun WhoisGuard (Namecheap) atawa privasi WHOIS (OpenSRS) nalika pendaptaran

#### Pangalaman Pamayaran anu Ngahiji {#unified-checkout-experience}
* **Tilu mode domain** — Konsumén milih antara subdomain (gratis), ngadaptarkeun domain anyar, atawa memetakan domain anu geus dipiboga
* **Pilarian domain real-time** — Pamariksaan kasadiaan anu digerakkeun ku AJAX tanpa muat ulang kaca
* **Saran pinter** — Saran TLD alternatif nalika domain anu dipikahayang geus dicokot
* **Harga transparan** — Tampilan écés pikeun harga pendaptaran, harga pembaruan, jeung biaya privasi WHOIS opsional
* **Widang kontak registrant** — Ngumpulkeun info registrant anu diperlukeun (ngaran, alamat, telepon) sacara inline nalika prosés pamayaran
* **URL situs dijieun otomatis** — Alamat situs sacara otomatis diturunkeun tina domain anu dipilih

#### Sistem Pembaruan Otomatis {#auto-renewal-system}
* **Pembaruan kabeungkeut kaanggotaan** — Domain diperbarui sacara otomatis salami kaanggotaan konsumén aktip
* **Jadwal pinter** — Pagawéan pembaruan diasupkeun kana antrean dumasar kana tanggal kadaluwarsa domain
* **Logika cobian deui** — Cobian deui sacara calakan pikeun usaha pembaruan anu gagal
* **Bewara email** — Pangeling anu tiasa dikonfigurasi pikeun pembaruan anu bakal datang, pembaruan hasil, kagagalan, jeung kadaluwarsa

#### Pangolahan DNS ku Konsumén {#customer-dns-management}
* **DNS mandiri** — Konsumén tiasa nambah, ngédit, jeung mupus rékaman DNS (A, AAAA, CNAME, MX, TXT) tina Dashboard maranéhna
* **Status domain** — Status pendaptaran real-time, tanggal kadaluwarsa, jeung inpormasi nameserver
* **Pemetaan domain otomatis** — Domain anu didaptarkeun atawa anu geus aya sacara otomatis dipetakeun ka situs konsumén

#### Pakakas Admin {#admin-tools}
* **Pendaptaran domain manual** — Daptarkeun domain atas nami konsumén tina panel admin
* **Pangawasan domain** — Tingali sakabéh domain konsumén, status, jeung tanggal kadaluwarsa tina hiji kaca
* **Logging husus panyadia** — Kanal log misah pikeun unggal registrar (contona, domain-seller-namecheap.log)
* **Rincian pendaptaran** — Tingali info registrant jeung data pendaptaran dina kaca édit domain inti

### Panyadia anu Dirojong {#supported-providers}

* **OpenSRS** — Integrasi lengkep jeung pendaptaran, pembaruan, pangolahan DNS, jeung privasi WHOIS
* **Namecheap** — Integrasi lengkep jeung pendaptaran, pembaruan, pamariksaan kasadiaan batch, jeung WhoisGuard gratis
* **HostAfrica** — Pendaptaran, pembaruan, transfer, pangolahan DNS, meunangkeun kode EPP, registrar lock, jeung panyalindungan ID
* **Openprovider** — Pendaptaran, pembaruan, transfer, pangolahan DNS, meunangkeun kode EPP, registrar lock, privasi WHOIS, jeung sinkronisasi TLD reseller
* **Hostinger** — Pendaptaran, lookup domain, pembaruan nameserver, registrar lock, jeung privasi WHOIS; pembaruan diurus ku auto-renew Hostinger
* **GoDaddy** — Pendaptaran jeung pembaruan
* **ResellerClub** — Pendaptaran, pembaruan, jeung pangolahan DNS
* **NameSilo** — Pendaptaran jeung pembaruan
* **Enom** — Pendaptaran, pembaruan, jeung pangolahan DNS

### Rincian Téknis {#technical-details}

* **Meryogikeun:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **Arsitéktur:** Pola Integration Registry kalayan interface Domain_Selling_Capability
* **Autoloading:** Autoloader Jetpack kalayan pemetaan kelas PSR-4
* **Standar:** Standar coding WordPress, strict typing, prepared statements

## Instalasi {#installation}

1. Pastikeun **Multisite Ultimate v2.4.12+** geus dipasang jeung diaktipkeun dina jaringan
2. Unggah plugin ka `/wp-content/plugins/ultimate-multisite-domain-seller/`
3. Aktipkeun plugin dina jaringan tina **Plugins** di WordPress Network Admin
4. **Setup Wizard** bakal dijalankeun otomatis dina aktivasina munggaran — turutan léngkah-léngkah pikeun:
   * Nyambungkeun domain registrar anjeun
   * Nguji sambungan
   * Ngimpor TLD anu sayaga jeung harga grosir
   * Nyieun produk domain munggaran anjeun kalayan markup harga
5. Tambahkeun widang **Domain Selection** ka formulir checkout anjeun dina **Ultimate Multisite > Checkout Forms**

## Patarosan anu Sering Ditanyakeun {#frequently-asked-questions}

### Vérsi Multisite Ultimate naon anu diperlukeun? {#what-version-of-multisite-ultimate-is-required}

Vérsi 2.4.12 atawa leuwih luhur. Addon ieu teu cocog jeung WP Ultimo v1.x atawa vérsi Ultimate Multisite di handap 2.4.12.

### Panyadia domain mana waé anu dirojong? {#which-domain-providers-are-supported}

Salapan registrar dirojong: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo**, jeung **Enom**. Kamampuhan béda-béda gumantung kana panyadia sabab registrar APIs mintonkeun operasi anu béda; tingali pituduh setup pikeun gambaran per panyadia. Arsitéktur Integration Registry milik addon ngajadikeun gampang pikeun nambahkeun panyadia khusus — tingali [pituduh developer](/developer/integration-guide/custom-registrar).

### Naha kuring bisa nawarkeun panyalindungan privasi WHOIS? {#can-i-offer-whois-privacy-protection}

Leres. Unggal produk domain boga setélan WHOIS Privacy kalayan tilu modeu: Disabled, Always Included, atawa Customer Choice. Lamun disetel ka Customer Choice, palanggan bakal ningali checkbox nalika checkout kalayan harga (atawa "Free" pikeun Namecheap's WhoisGuard). Tanda privasi sacara otomatis dikirim ka registrar nalika waktos pendaptaran.

### Naha kuring bisa netepkeun harga anu béda pikeun TLD anu béda? {#can-i-set-different-pricing-for-different-tlds}

Leres. Jieun produk domain anu misah pikeun grup TLD anu béda. Unggal produk bisa boga jinis markup sorangan (perséntase, markup tetep, atawa harga tetep), TLD anu dirojong, jeung harga bubuka.

### Pilihan checkout naon anu ditingali ku palanggan? {#what-checkout-options-do-customers-see}

Widang Domain Selection masihan palanggan tilu tab: **Subdomain** (ngagunakeun subdomain gratis dina jaringan anjeun), **Register New Domain** (milarian jeung ngadaptarkeun domain anyar), jeung **Existing Domain** (memetakeun domain anu geus dipiboga ku maranéhna). Ngan modeu anu anjeun aktipkeun anu bakal mucunghul.

### Naha ieu ngadukung pembaruan otomatis? {#does-it-support-automatic-renewals}

Leres. Domain bisa disetel pikeun auto-renew jeung dikaitkeun kana status membership palanggan. Pamayaran pembaruan diprosés ngaliwatan gateway pamayaran anu geus aya milik palanggan.

### Naha palanggan bisa ngatur rékaman DNS maranéhna? {#can-customers-manage-their-dns-records}

Leres. Palanggan meunang antarmuka manajemén DNS dina dashboard maranéhna pikeun nambahkeun, ngédit, jeung mupus rékaman A, AAAA, CNAME, MX, jeung TXT. Manajemén DNS sayaga jeung OpenSRS, ResellerClub, Enom, HostAfrica, jeung Openprovider. DNS Hostinger pikeun domain anu di-host ditangani ku integrasi domain-mapping inti Hostinger.

### Naha aya setup wizard? {#is-there-a-setup-wizard}

Leres. Setup wizard jalan otomatis dina aktivasina munggaran jeung nungtun anjeun pikeun nyambungkeun registrar, ngimpor TLD, sarta nyieun produk domain munggaran anjeun. Anjeun ogé bisa ngajalankeun deui wizard tina setélan addon.
