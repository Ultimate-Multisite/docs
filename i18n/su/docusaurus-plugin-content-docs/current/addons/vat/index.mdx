# VAT

Multisite Ultimate: VAT Éropa

Nambahkeun fitur jeung pakakas tambahan kana pananganan pajeg standar Multisite Ultimate pikeun leuwih hadé ngadukung VAT Éropa.
Sacara otomatis nambahkeun tarif EU VAT kana kaca Tarif Pajeg ti

## Pamasangan

1. Unggah 'ultimate-multisite-vat' ka diréktori '/wp-content/plugins/'
2. Aktipkeun plugin ngaliwatan ménu 'Plugin' dina WordPress
3. Buka Ultimate Multisite > Setélan > Pajeg jeung aktipkeun EU Vat.
4. Klik kana Tarif Pajeg jeung Apdet Tarif EU VAT di handap, tuluy Simpen Tarif Pajeg.
5. Réngsé!
