# Pangékspor Situs

## Eusi

Kami ngartos kabutuhan pikeun bisa mindahkeun subsitus dina jaringan anjeun ka pamasangan situs tunggal jeung sabalikna lamun palanggan anjeun mutuskeun pikeun migrasi ramatloka maranéhanana. Addon pangékspor jeung pangimpor situs ngajadikeun pancén ieu leuwih gampang. Parabot ieu ngékspor jeung ngimpor situs maké file zip anu ogé bisa dijadikeun cadangan.

## Fitur

* Ékspor jeung Impor Situs

* Antarbeungeut anu gampang dipaké

* Jieun cadangan situs

## Dokuméntasi
**Kumaha Ngékspor jeung Ngimpor Situs**

## Pamasangan

1. Unggah **ultimate-multisite-site-exporter** ka diréktori **/wp-content/plugins/**
2. Aktipkeun plugin ngaliwatan ménu **Plugin** dina WordPress
3. Turutan Wizard léngkah-demi-léngkah pikeun nyetél plugin

## Pituduh Pamakéan

Kalayan Ultimate Multisite anjeun tiasa ngékspor situs tina jaringan anjeun atawa ngimpor situs ka pamasangan multisite atawa situs-tunggal anjeun.

Pikeun ngamimitian prosésna, buka kaca add-ons kami teras pasang add-on Ultimate Multisite: Pangékspor & Pangimpor Situs.![Kaca add-ons](/img/admin/addons-page.png)

![Kaca add-ons jeung pilihan pasang](/img/admin/addons-page.png)

Buka kaca Plugin anjeun teras klik pikeun **Aktipkeun Jaringan** add-on anu kakara anjeun pasang.<!-- Screenshot teu sayaga: kaca Plugin WP némbongkeun tombol Aktipkeun Jaringan pikeun addon Pangékspor Situs -->

Pilihan **Ékspor & Impor** tiasa kapanggih dina ménu Ultimate Multisite, di kénca.

<!-- Screenshot teu sayaga: ménu Ultimate Multisite némbongkeun pilihan Ékspor jeung Impor -->

## Kumaha ngékspor situs

Pikeun ngékspor situs tina jaringan anjeun, buka **Ékspor & Impor** dina ménu Ultimate Multisite sarta, dina tab **Ékspor Situs**, klik **Jieun Ékspor Situs anyar**.

<!-- Screenshot teu sayaga: tab Ékspor Situs jeung tombol Jieun Ékspor Situs anyar -->

Pilarian situs anu rék anjeun ékspor sarta pilih item anu rék diasupkeun kana file anu diékspor. Anjeun tiasa ngalebetkeun téma, plugin, jeung unggahan. Lamun situs anjeun kacida gedéna, anjeun tiasa milih pikeun ngajalankeun prosés ékspor di tukang.<!-- Screenshot teu sayaga: pilihan ékspor situs jeung pilarian situs, kotak centang kalebet téma/plugin/unggahan -->

Klik tombol **Ékspor Situs**. Situs anu diékspor ku anjeun bakal sayaga dina daptar **Ékspor Nu Aya**. Klik tombol **Unduh ZIP** pikeun ngundeur file. ZIP ékspor ngawengku `index.php` anu bisa boot sorangan jeung `readme.txt`, jadi éta bisa dipasang dina host anyar tanpa kudu masang plugin pangimpor anu misah heula.<!-- Screenshot teu sayaga: daptar Ékspor Nu Aya jeung tombol Unduh ZIP -->

## Kumaha ngékspor sakabéh jaringan

Pikeun ngékspor sadaya subsitus babarengan, buka **Ékspor & Impor** dina ménu Ultimate Multisite sarta paké pilihan **Ékspor Jaringan** dina layar ékspor. Arsip anu dijieun ngandung sadaya subsitus dina hiji ZIP pikeun migrasi host, ngawangun deui staging, atawa cadangan tingkat-jaringan.

Ékspor jaringan tiasa gedé. Jalankeun nalika patalimarga handap sarta pastikeun server boga panyimpenan anu cekap pikeun arsip jeung unggahan, plugin, sarta téma naon waé anu diasupkeun.

## Kumaha ngimpor situs

Pikeun ngimpor situs, buka tab **Impor Situs** teras unggah file .zip tina situs anu rék anjeun impor. Anjeun tiasa ngarobah URL kana anu anyar atawa ngajaga URL heubeul ku ngantepkeun rohangan kosong. Sajaba ti éta, anjeun tiasa mariksa kotak pikeun mupus file .zip tina pamasangan anjeun sanggeus impor. Teras klik **Mimitian Impor**.

<!-- Screenshot teu sayaga: tab Impor Situs jeung widang unggah zip sarta tombol Mimitian Impor -->

Prosésna tiasa nyokot sababaraha waktu. Anjeun bakal ningali impor anu keur ngantosan dina daptar di handapeun pangimpor.

<!-- Screenshot teu sayaga: impor anu keur ngantosan dina daptar di handapeun pangimpor -->

Anjeun tiasa ngeureunkeun prosés ku ngaklik **Bolaykeun Impor**.

<!-- Screenshot teu sayaga: tombol Bolaykeun Impor pikeun ngeureunkeun prosés impor -->

Sanggeus prosés réngsé, situs anjeun geus sayaga dina jaringan anjeun.

## Kumaha ngimpor situs dina pamasangan tunggal

Pikeun ngimpor situs ka pamasangan tunggal anjeun, heula pasang jeung aktipkeun pangimpor dina situs anjeun. Anjeun tiasa manggihan tautan pikeun ngundeur éta dina tab pangékspor.

<!-- Screenshot teu sayaga: tab pangékspor jeung tautan pikeun ngundeur pangimpor situs-tunggal -->

Anjeun tiasa manggihan pangimpor dina ménu **Parabot**, di kénca, dina subitem **Ékspor & Impor**.

<!-- Screenshot teu sayaga: ménu Parabot situs-tunggal némbongkeun subménu Ékspor jeung Impor -->

Unggah file .zip tina situs anu rék anjeun impor. Anjeun tiasa ngarobah URL kana anu anyar atawa ngajaga URL heubeul ku ngantepkeun rohangan kosong. Sajaba ti éta, anjeun tiasa mariksa kotak pikeun mupus file .zip tina pamasangan anjeun sanggeus impor. Teras klik **Mimitian Impor**.

<!-- Screenshot teu sayaga: formulir impor situs-tunggal jeung unggah zip sarta tombol Mimitian Impor -->

Prosésna tiasa nyokot sababaraha waktu. Anjeun bakal ningali impor anu keur ngantosan dina daptar di handapeun pangimpor.

<!-- Screenshot teu sayaga: impor situs-tunggal anu keur ngantosan dina daptar -->

Anjeun tiasa ngeureunkeun prosés ku ngaklik **Bolaykeun Impor**.

<!-- Screenshot teu sayaga: tombol Bolaykeun Impor pikeun ngeureunkeun prosés impor -->

Sanggeus prosés réngsé, situs anjeun bakal diganti ku situs anu kakara anjeun impor.
