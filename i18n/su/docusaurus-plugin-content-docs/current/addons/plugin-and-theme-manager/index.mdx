# Pangatur Tambihan & Téma {#plugin--theme-manager}

Multisite Ultimate: Pangatur Tambihan jeung Téma

Édit atawa sumputkeun informasi meta (judul, panulis, gambar leutik jeung pedaran) tina tambihan jeung téma anu geus dipasang. Jieun kategori pikeun tambihan jeung téma supaya pamaké anjeun bisa nyaringna sarta tambahkeun kaca Tambihan kustom anu éndah kana panel pamaké anjeun.

Pikeun ngagunakeun tambihan ieu, anjeun bisa langsung muka kaca Tambihan atawa Téma dina admin jaringan anjeun sarta klik "Édit info Tambihan/Téma".

## Pamasangan {#installation}

1. Unggah 'ultimate-multisite-plugin-and-theme-manager' ka diréktori '/wp-content/plugins/'
2. Aktipkeun tambihan ngaliwatan ménu 'Tambihan' dina WordPress Network Admin anjeun

## Pituduh Pamakéan {#usage-guide}

Ultimate Multisite: Pangatur Tambihan jeung Téma nyaéta add-on Ultimate Multisite **premium** anu kuat, anu masihan anjeun kadali lengkep kana cara pamaké jaringan anjeun ningali jeung berinteraksi jeung pilihan tambihan jeung téma dina jaringan anjeun.

Add-on ieu boga tilu tujuan:

  1. Ieu ngamungkinkeun anjeun ngarobah metadata tambihan jeung téma sacara lengkep. Éta ngawengku ngaran, pedaran, ngaran panulis, komo gambar leutik ogé!

  2. Ieu ngamungkinkeun anjeun ngagolongkeun tambihan jeung téma, supaya pamaké anjeun bisa gampang nyortirna.

  3. Ieu masihan anjeun pilihan pikeun ngaganti kaca tambihan anu heubeul jeung pikabosenkeun ku kaca anyar anu siga kaca téma kalayan panyortiran dumasar kategori.

Sanggeus diundeur jeung dipasang, tab anyar - _Setélan Add-on_ \- bakal némbongan dina kaca Setélan Ultimate Multisite.
