# Fluent Forms

Add-on ieu ngamungkinkeun nambahkeun wates kana Fluent Forms dina Multisite Ultimate dina produk, kaanggotaan, atawa situs.
Fluent Forms ngagunakeun tabel kustom tibatan tipe pos kustom, jadi pilihan ngawatesan anu biasa henteu jalan jeung Fluent Forms.

## Fitur

- Tambahkeun wates pikeun Fluent Forms atawa Conversational Fluent Forms
- Palanggan meunang pesen anu ramah anu ngajurung aranjeunna pikeun ningkatkeun plan nalika ngahontal wates
