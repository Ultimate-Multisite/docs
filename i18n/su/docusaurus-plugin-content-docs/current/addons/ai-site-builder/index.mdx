# Pangwangun Situs AI {#ai-site-builder}

ultimate-multisite-ai-site-builder

## Sarat {#requirements}

- WordPress 5.3 atawa leuwih luhur
- PHP 7.4 atawa leuwih luhur
- plugin Multisite Ultimate (aktip)

## Pamasangan {#installation}

1. Unggah payil addon ka diréktori `/wp-content/plugins/` anjeun
2. Aktipkeun plugin ngaliwatan ménu 'Plugins' dina WordPress
3. Konpigurasikeun setélan addon dina admin jaringan

## Pangwangunan {#development}

### Setup {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Nguji {#testing}

Addon ieu ngawengku rangka nguji anu komprehensif:

- **Unit Tests**: Tés dumasar PHPUnit kalayan WordPress test suite
- **Base Test Class**: Nyadiakeun métode bantu pikeun skenario tés umum
- **Test Coverage**: Dikonpigurasikeun pikeun ngalacak cakupan kode
- **Multisite Testing**: Tés dijalankeun dina lingkungan multisite

### Standar Kode {#code-standards}

- **WordPress Coding Standards**: Ditegakeun ngaliwatan PHPCS
- **Kasaluyuan PHP 7.4+**: Fitur PHP modéren dirojong
- **Analisis Statis**: Integrasi PHPStan pikeun mariksa tipe
- **Modernisasi Kode**: Aturan Rector pikeun pamutahiran PHP

### Sistem Ngawangun {#build-system}

- **Ngolah Aset**: Minifikasi pikeun payil CSS/JS
- **Tarjamahan**: Nyieun payil POT
- **Paket**: Nyieun arsip pikeun distribusi
- **Modeu Pangwangunan**: Aset teu diminifikasi pikeun debugging

## Konfigurasi {#configuration}

Addon ieu ngarojong rupa-rupa pilihan konfigurasi anu bisa diaksés ngaliwatan antarbeungeut admin jaringan.

## Hooks jeung Filters {#hooks-and-filters}

### Actions {#actions}

- `ultimate-multisite-ai-site-builder_init` - Dipicu sanggeus inisialisasi addon
- `ultimate-multisite-ai-site-builder_loaded` - Dipicu sanggeus sakabéh payil addon dimuat

### Filters {#filters}

- `ultimate-multisite-ai-site-builder_settings` - Robah setélan addon
- `ultimate-multisite-ai-site-builder_enabled` - Timpa status aktip/nonaktip addon
