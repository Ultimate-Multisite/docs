# Pamilih Basa {#language-selector}

Ultimate Multisite: Pamilih Basa dina Daptar

Tambahkeun widang pamilih basa kana alur daptar.

## Pamasangan {#installation}

1. Unggah 'ultimate-multisite-language-selector' ka diréktori '/wp-content/plugins/'
2. Aktipkeun plugin ngaliwatan ménu 'Plugin' dina WordPress Network Admin anjeun

## Pituduh Pamakéan {#usage-guide}

Kalayan add-on gratis ieu anjeun tiasa ngidinan pamaké anjeun milih basa mana anu rék ditempo dina alur daptar jeung dina kaca login.

Sanggeus [ngaktipkeun lisénsi anjeun](../../user-guide/getting-started/installing-ultimate-multisite.md) anjeun bakal tiasa masang add-on Ultimate Multisite anu ku lisénsi anjeun dibéré aksésna.

Buka kaca setélan Ultimate Multisite anjeun tuluy, dina ménu di kénca, klik labél _Add-on_.

![Kaca add-on](/img/admin/addons-page.png)

_Buka kaca Add-on pikeun ngamimitian masang Pamilih Basa_

Di dinya, anjeun bakal manggihan sakabéh add-on anu ku kami ditawarkeun.

Add-on gratis sadia pikeun sakabéh palanggan kami, henteu paduli rencana langganan anu maranéhna daptarkeun.

Klik tombol _Pasang Ayeuna!_ di handapeun gambar leutik add-on. Tuluy aktipkeun éta dina kaca Plugin.

![Kaca add-on jeung tombol pasang](/img/admin/addons-page.png)

_Klik tombol pikeun masang add-on_

<!-- Screenshot teu sadia: kaca Plugin WP anu némbongkeun tombol Network Activate pikeun add-on Language Selector -->

_Aktipkeun éta dina kaca Plugin anjeun_

Ayeuna, buka ménu Setélan Ultimate Multisite sarta asup kana tab _Setélan Jaringan_. Di dinya, di handapeun kaca, anjeun bakal manggihan bagian anyar anu dingaranan _Basa_.

![Kaca setélan](/img/config/settings-general.png)

_Bagian anyar pisan mucunghul!_

Anjeun tiasa ngantepkeunana kosong pikeun nembongkeun sakabéh basa salaku pilihan atawa milih mana waé anu rék disadiakeun. Basa-basa éta nyaéta basa anu geus dipasang dina setélan WordPress anjeun.

<!-- Screenshot teu sadia: dropdown pilihan basa anu némbongkeun basa anu sadia dina pamasangan WP -->

_Daptar basa anu sadia dina pamasangan WP ieu_

Sanggeus nyimpen parobahan anu anjeun jieun, ménu pilihan jeung basa anu anjeun pilih bakal dipintonkeun dina kaca Login ogé dina Alur Daptar.

![Setélan login jeung pendaptaran](/img/config/settings-login-registration.png)

_Pamilih Basa dina Kaca Login_

![Kaca pendaptaran](/img/config/registration-page.png)

_Pamilih Basa dina Alur Daptar_

## Nyetél Basa ngaliwatan URL {#setting-the-language-via-the-url}

Anjeun ogé tiasa ngarobah basa hiji kaca ku ngirimkeun parameter locale ka URL. Upami anjeun hoyong ngalihkeun ka kaca daptar dina basa Italia, contona, tambahkeun ?locale=it_IT kana URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Basa éta kudu geus dipasang dina jaringan WordPress anjeun supaya ieu tiasa jalan.
