# Saringan: signup_get_available_languages {#filter-signupgetavailablelanguages}

Nyaring daptar basa anu sayogi pikeun pendaptaran situs front-end.

Ngirim array kosong ka hook ieu bakal nganonaktipkeun kaluaran setélan dina formulir pendaptaran, sarta basa standar bakal dipaké nalika nyieun situs. Basa anu can dipasang bakal dipiceun.

## Paraméter {#parameters}

| Ngaran | Jinis | Pedaran |
|------|------|-------------|
| $available_languages | `array` | Basa anu sayogi. |
