# PayFast

Ultimate Multisite: Payfast Gateway

Nambihan Payfast - gerbang pamayaran online anu gancang jeung aman di Afrika Kidul - salaku pilihan pamayaran Ultimate Multisite.

## Pamasangan {#installation}

1. Unggah 'ultimate-multisite-payfast' ka diréktori '/wp-content/plugins/'
2. Aktipkeun pangalengkep ngaliwatan ménu 'Pangalengkep' dina WordPress
3. Turutan Wizard léngkah demi léngkah pikeun nyetél pangalengkep

## Pituduh Pamakéan {#usage-guide}

PayFast nyaéta prosésor pamayaran online unggulan pikeun usaha di Afrika Kidul. Ieu didédikasikeun pikeun mantuan padagang online tumuwuh ku ngajadikeun transaksi lancar jeung aman. Numutkeun paménta pamaké Ultimate Multisite, kami mutuskeun nyieun tambihan anu bakal ngamungkinkeun integrasi jeung PayFast. Artikel ieu bakal némbongkeun kumaha nyetél PayFast salaku gerbang pamayaran anjeun pikeun narima pamayaran dina jaringan Ultimate Multisite anjeun.

Léngkah kahiji nyaéta masang tambihan.

# Kumaha masang tambihan Payfast:

Buka kaca tambihan sarta pilih tambihan Ultimate Multisite: Payfast Gateway.

![Kaca tambihan](/img/admin/addons-page.png)

Jandéla tambihan bakal mucunghul. Klik Install Now.

![Kaca tambihan jeung pilihan pamasangan](/img/admin/addons-page.png)

Sanggeus pamasangan réngsé, anjeun bakal dialihkeun ka kaca pangalengkep. Di dieu, cukup klik Network Activate sarta tambihan _Payfast_ bakal diaktipkeun dina jaringan anjeun.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# Nyetél PayFast

Sanggeus masang tambihan, ayeuna anjeun bisa buka setélan Ultimate Multisite sarta mimiti ngonpigurasikeun PayFast.

Dina kaca **setélan Ultimate Multisite**, anjeun bisa manggihan tab "Pamayaran". Dina tab ieu, anjeun kedah ningali "**Payfast** " babarengan jeung tombol toggle pikeun ngaktipkeun atawa nganonaktipkeun gerbang pamayaran ieu.

![Setélan gerbang pamayaran](/img/config/settings-payment-gateways.png)Terus klik tombol toggle pikeun ngaktipkeunana. Teras éta bakal ngalegaan kotak pikeun némbongkeun rupa-rupa widang anu kudu anjeun eusian sapertos **Mechant ID, Merchant Key jeung PassPhrase.** Anjeun bisa meunangkeun inpormasi ieu tina akun **Payfast** anjeun

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Léngkah salajengna nyaéta asup ka akun PayFast anjeun pikeun meunangkeun inpormasi anu anjeun peryogikeun.

Saatos asup, anjeun langsung bisa ningali kredénsial anu anjeun peryogikeun tina Dashboard akun **Payfast** anjeun.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->Anjeun cukup nyalin jeung nempelkeun kredénsial sarta ngeusian rinci dina setélan Ultimate Multisite anjeun. Teras klik tombol **"Save settings** " pikeun nyimpen parobahan.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Éta waé, integrasi pamayaran PayFast anjeun parantos réngsé!
