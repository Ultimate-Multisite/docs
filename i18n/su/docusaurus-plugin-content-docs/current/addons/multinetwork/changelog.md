# Catetan Parobahan Multi-Network

### 1.3.0
* Anyar: Panilik template jaringan — kotektak jeung tilik template jaringan ku panel langsung saméméh meuli.
* Anyar: Pamilih template jaringan di checkout — palanggan bisa milih template jaringan salila alur signup checkout.
* Anyar: Cross-network magic link SSO — palanggan bisa ngaksés sub-jaringan kalayan mulus liwat auténtikasi magic-link.
* Anyar: Kloning média jaringan — média situs diduplikasi kalayan leres nalika ngloning jaringan tina template.
* Perbaikan: Kloning jaringan ayeuna jalan tanpa addon multi-tenancy; hak istiméwa super admin dibéré deui sanggeus clone.
* Perbaikan: Nyieun jaringan ayeuna cocog jeung WooCommerce sarta sakabéh gateway pamayaran.
* Perbaikan: Nomer port ayeuna diasupkeun dina nyieun domain fallback.
* Perbaikan: CSS pamilih template dipindahkeun ka stylesheet anu di-enqueue pikeun rendering anu leres dina sakabéh theme.
* Perbaikan: Jaga tina konci `network_id` anu teu ditangtukeun dina scope query situs.
* Perbaikan: Pamariksaan idin jaringan saméméh context switching nyegah kasalahan property tabel `wpdb`.
