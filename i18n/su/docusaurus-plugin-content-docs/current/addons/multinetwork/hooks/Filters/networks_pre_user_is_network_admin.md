# Saringan: networks_pre_user_is_network_admin

Nyaring network anu pamaké jadi administrator-na, pikeun motong pondok prosésna.

## Paraméter {#parameters}

| Ngaran | Tipe | Pedaran |
|------|------|-------------|
| $ | `array\|bool\|null` | Daptar ID network atawa false. Naon waé salian ti null bakal motong pondok prosésna. |
| $ | `int` | ID pamaké anu network-na kudu dipulangkeun. |
