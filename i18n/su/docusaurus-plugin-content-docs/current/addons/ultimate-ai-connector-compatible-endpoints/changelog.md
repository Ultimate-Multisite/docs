## Vérsi 2.0.0 — Dileupaskeun dina 2026-04-24 {#version-200--released-on-2026-04-24}

* Perbaikan: Integrasi multi-provider SDK kalayan provider ID anu leres, URL pandaptaran, jeung scoping filter HTTP per provider.
* Perbaikan: Kartu provider anyar ayeuna otomatis dimekarkeun nalika ditambahkeun; script cache busting nalika update plugin.
* Perbaikan: Namespace class provider dinamis pikeun eval() jeung paripolah otomatis dimekarkeun pikeun provider anu anyar ditambahkeun.
* Perbaikan: Ngagunakeun komponén Card/CardBody/CardHeader/CardDivider anu stabil (henteu deui ékspériméntal) pikeun kasaluyuan WordPress 6.9+.
* Perbaikan: Ngaganti DragHandle anu teu didefinisikeun ku ikon grip unicode pikeun provider drag-to-reorder.
* Perbaikan: Ngaleungitkeun paménta HTTP anu ngahalangan anu dijalankeun dina unggal muatan kaca.
* Ningkat: Workflow GitHub Actions di-upgrade ka Node.js 24.
