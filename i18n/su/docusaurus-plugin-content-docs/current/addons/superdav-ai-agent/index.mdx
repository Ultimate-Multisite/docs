# Superdav AI Agent

Superdav AI Agent nyaéta asisten AI WordPress canggih anu ngalegaan Gratis AI Agent ku kamampuhan anu sadar kana theme, kamampuhan milarian internét anu ditingkatkeun, fitur pangwangun situs, jeung integrasi otomatisasi opsional. Vérsi 1.18.0 nambihan pakakas maca Google Calendar, pemetaan kontak jeung pamilon, gerbang persetujuan manusa, deduplikasi panginget, béwara SMS TextBee, panyadiaan layanan Superdav anu dikokolakeun, sarta pakét pendamping Advanced anu disebarkeun misah pikeun alur gawé pamekar anu dipercaya.

## Fitur Utama

- **Kamampuhan sadar kana theme** — otomatis nyaluyukeun pituduh jeung kamampuhan dumasar kana theme WordPress aktip anjeun jeung plugin anu dipasang
- **Sababaraha panyadia milarian internét** — pilih antara Brave Search jeung Tavily pikeun hasil milarian anu leuwih beunghar jeung kamampuhan panalungtikan canggih
- **Formulir kontak pangwangun situs** — jieun formulir kontak anu fungsional lengkep langsung tina antarmuka obrolan nganggo plugin formulir anu dipasang
- **Pangwangun situs anu ditingkatkeun** — rencana ngawangun situs multi-fase anu leuwih alus kalayan panimuan plugin jeung pamulihan kasalahan anu leuwih hadé
- **Kamampuhan theme bawaan** — pituduh husus pikeun Block Themes, Classic Themes, Kadence Blocks, jeung Kadence Theme
- **Integrasi WooCommerce anu ditingkatkeun** — diréstrukturisasi pikeun ngagunakeun API WooCommerce asli demi reliabilitas jeung kasaluyuan anu leuwih hadé
- **Deteksi panyadia otomatis** — otomatis ngadeteksi jeung nyegerkeun plugin jeung theme anu sayogi nalika diaktipkeun atawa dinonaktipkeun
- **Nyieun logo kustom** — hasilkeun jeung sisipkeun logo SVG kustom kalayan sanitasi otomatis anu aman pikeun namespace
- **Validasi kontras palét** — pariksa pasangan warna pikeun patuh kana aksésibilitas WCAG saméméh diterapkeun kana theme anjeun
- **Panimuan anu diperkaya ku poto** — unggah poto nalika wawancara panimuan Theme Builder pikeun mangaruhan arah desain
- **Menu hospitality** — hasilkeun kaca menu kadaharan jeung inuman anu terstruktur pikeun réstoran, kafe, jeung usaha hospitality
- **Pratinjau desktop jeung mobile** — tingali pratinjau desain anjeun dina alat desktop jeung mobile salila milih arah desain
- **Pangikis situs Tier 1** — sd-ai-agent/site-scrape ayeuna sayogi sacara standar pikeun nganalisis ramatloka pesaing jeung inspirasi
- **Pakakas maca Google Calendar** — maca kalénder jeung acara anu dikonfigurasi supaya otomatisasi bisa mikirkeun kontéks jadwal anu bakal datang
- **Pemetaan kontak jeung pamilon** — cocogkeun pamilon kalénder jeung pamaké situs sarta kontak anu dipikawanoh saméméh ngahasilkeun pagawéan tindak lanjut
- **Gerbang persetujuan manusa** — reureuhkeun otomatisasi anu sénsitip pikeun ditinjau saméméh ngirim pesen, neraskeun pagawéan, atawa nyieun parobahan
- **Rékaman panginget jeung deduplikasi** — lacak panginget anu geus dikirim supaya jalanna otomatisasi anu diulang henteu ngabéjaan panarima anu sarua dua kali
- **Béwara SMS TextBee** — kirim béwara pesen téks anu dikonfigurasi ngaliwatan TextBee nalika panyadia SMS diaktipkeun
- **Panyadiaan layanan Superdav anu dikokolakeun** — sambungkeun situs anu dirojong ka endpoint Superdav hosted kalayan setélan otomatis tibatan konfigurasi endpoint manual
- **Pakét pendamping Advanced** — pakakas pamekar dipercaya anu disebarkeun misah pikeun alur gawé filesystem, basis data, WP-CLI, dispatcher REST, pangwangun plugin, snapshot git, manajemén pamaké, jeung benchmark

## Sarat

- WordPress 6.0 atawa leuwih luhur
- PHP 8.1 atawa leuwih luhur
- WordPress Multisite (diaktipkeun dina jaringan) atawa pamasangan single-site
- Plugin panyambung panyadia AI anu aktip (Anthropic, OpenAI, atawa anu cocog)
- Gratis AI Agent v1.9.0 atawa leuwih luhur (Superdav AI Agent ngalegaan Gratis AI Agent)

## Pamasangan

1. Unggah berkas plugin ka `/wp-content/plugins/superdav-ai-agent`
2. Aktipkeun plugin ngaliwatan layar **Plugins** dina WordPress
3. Pastikeun Gratis AI Agent ogé diaktipkeun
4. Buka **Gratis AI Agent → Settings** sarta konfigurasikeun panyadia milarian internét anjeun, sambungan layanan Superdav anu dikokolakeun, Google Calendar, atawa kredénsial TextBee sakumaha diperlukeun

Pakét WordPress.org ngandung fitur inti Superdav AI Agent. Pakét pendamping Advanced diwangun jeung disebarkeun misah pikeun lingkungan dimana pakakas pamekar dipercaya dipasang jeung diotorisasi sacara éksplisit.

## Kamampuhan Sadar kana Theme

Superdav AI Agent otomatis ngadeteksi theme aktip anjeun jeung plugin anu dipasang, ngamuat kamampuhan sadar kana theme anu luyu kana kontéks asisten. Kamampuhan anu sayogi ngawengku:

| Kamampuhan | Dilarapkeun Ka |
|---|---|
| **Block Themes** | Theme anu ngagunakeun arsitéktur dumasar blok WordPress (Full Site Editing) |
| **Classic Themes** | Theme WordPress tradisional anu ngagunakeun témplat PHP jeung éditor klasik |
| **Kadence Blocks** | Situs kalayan plugin Kadence Blocks aktip |
| **Kadence Theme** | Situs kalayan theme Kadence salaku theme aktip |

Tingali [Kamampuhan Sadar kana Theme](../../user-guide/configuration/theme-aware-skills) pikeun pituduh lengkep ngeunaan unggal kamampuhan.

## Panyadia Milarian Internét

Superdav AI Agent v1.10.0 nambihan **Tavily** minangka alternatif pikeun Brave Search pikeun pamundut milarian internét. Duanana panyadia nawarkeun hasil milarian waktu nyata sarta bisa dikonfigurasi dina **Gratis AI Agent → Settings → Advanced**.

| Panyadia | Fitur | Setelan |
|---|---|---|
| **Brave Search** | Hasil pamilarian gancang, museur kana privasi | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Hasil pamilarian leuwih beunghar, kamampuhan panalungtikan maju | [Tavily API](https://tavily.com/) |

Tingali [Pamilarian Internét](../../user-guide/configuration/internet-search) pikeun pituduh konfigurasi.

## Alur Gawé Kalénder, Persetujuan, jeung Béwara

Superdav AI Agent v1.18.0 ngalegaan alur gawé otomatisasi saluareun pancén eusi jeung ngawangun loka:

- **Pakakas Google Calendar** ngamungkinkeun agén maca kalénder anu geus dikonfigurasi jeung rinci kajadian lamun kredénsial kalénder geus sayaga.
- **Pitulung kontak jeung hadirin** nyocogkeun pamilon kajadian ka pamaké WordPress atawa kontak anu dipikawanoh saméméh nyiapkeun panginget atawa pancén tindak lanjut.
- **Gerbang persetujuan manusa** ngamungkinkeun otomatisasi eureun heula nepi ka pamaké anu boga wenang marios tindakan anu diusulkeun.
- **Catetan panginget** nyegah panginget kalénder atawa béwara anu duplikat nalika otomatisasi dicoba deui atawa dijadwalkeun sababaraha kali.
- **Pangrojong SMS TextBee** ngamungkinkeun béwara pesen téks nalika panyadia TextBee dikonfigurasi.

Paké pakakas ieu pikeun pitulung anu sadar jadwal, tindak lanjut hadirin, jeung alur gawé panginget anu masih merlukeun pamariksaan manusa saméméh pesen sénsitip dikirim.

## Konfigurasi

Sakabéh setélan aya dina **Gratis AI Agent → Settings** dina admin WordPress. Superdav AI Agent nambahkeun pilihan konfigurasi di handap:

| Setélan | Katerangan |
|---|---|
| **Panyadia Pamilarian Internét** | Pilih antara Brave Search jeung Tavily (opsional) |
| **Tavily API Key** | API key pikeun pamilarian internét Tavily (lamun Tavily dipilih) |
| **Ladenan Superdav Dikokolakeun** | Sambungkeun loka anu dirojong ka ladenan Superdav hosted kalayan panyayagiaan endpoint otomatis |
| **Kredénsial Google Calendar** | Konfigurasi aksés kalénder sangkan agén bisa maca kalénder jeung kajadian |
| **Panyadia SMS TextBee** | Konfigurasi kredénsial TextBee pikeun alur gawé béwara SMS anu geus disatujuan |

## Pakét Companion Advanced

Superdav AI Agent v1.18.0 nambahkeun pakét companion Advanced anu disebarkeun misah pikeun alur gawé pamekar anu dipercaya. Kamampuhan ieu henteu dibundel dina pakét inti WordPress.org. Pasang pakét companion ngan dina loka tempat administrator sacara écés percaya ka agén pikeun ngaksés pakakas pamekar.

Pakakas Advanced ngawengku operasi sistem berkas, aksés basis data, WP-CLI, alur gawé dispatcher REST, utilitas pangwangun plugin, snapshot git, pitulung ngokolakeun pamaké, jeung pakakas benchmark. Watesan pakakas ieu pikeun administrator anu dipercaya jeung pariksa konfigurasi gerbang persetujuan saméméh ngaktipkeunana dina loka produksi.
