# Codeable-dan Custom Geliştirme Yardımı Nasıl Alırsınız? {#how-to-get-custom-development-help-from-codeable}

Bazen ağ akışlarımızı tam istediğimiz gibi yapmak için gerçekten harika bir özelleştirmeye ihtiyacımız olur. Eğer aradığınız buysa, size [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE)'i tavsiye ederiz; burada Ultimate Multisite uzmanlığı olan deneyimli geliştiriciler bulabilirsiniz.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), müşterileri nitelikli WordPress serbest çalışanlarla birleştiren ve her türlü büyüklükteki projelerde çalışabilen bir platformdur. Sizin aklınızdaki görev için özel olarak nitelikli uzmanlarla eşleştirilmenizi sağlamak için titiz bir başvuru süreci vardır.

### Codeable'da proje nasıl oluşturulur? {#how-to-create-a-project-in-codeable}

Öncelikle [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE)'e gidin ve **Start a Project** (Proje Başlat) düğmesine tıklayın.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Şimdi, bir proje göndermek için soldaki **Start a Project**'e tıklayın.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

"I need help with" (Yardım ihtiyacım var) kısmında **Customizing** (Özelleştirme) seçeneğini seçin:

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

"My" (Benim) kısmında **Plugin** (Eklenti) seçeneğini seçin:

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Ardından **Continue** (Devam Et) düğmesine tıklayın.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Şimdi, projenizi ve ihtiyaçlarınızı en iyi şekilde anlatma zamanı. Mümkün olduğunca fazla bilgi vermeye çalışın!

<!-- Screenshot unavailable: Codeable project description form -->

Fiyatlandırma konusunda yardımcı olmak için aşağıdaki soruları yanıtlayın.

<!-- Screenshot unavailable: Codeable project pricing questions -->

А потом, если у вас еще нет аккаунта на [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), создайте его и нажмите кнопку **Publish My Project** (Мой проект опубликовать).

<!-- Скриншот недоступен: создание аккаунта Codeable и кнопка Publish My Project -->

Как только вы опубликуете свой проект на [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), платформа соединит вас с разработчиками, которые подходят по навыкам для вашего проекта.

Этот процесс поможет найти нужного человека для вашей работы, у которого есть опыт работы с WordPress и Ultimate Multisite!

Минимальная ставка Codeable составляет от $70 до $120 USD в час. Оценки могут меняться в зависимости от объема работ, сложности и срочности проекта.

Как только вы получите оценку, сможете нанять разработчика, который лучше всего подходит для ваших нужд, и начать ваш проект вместе!
