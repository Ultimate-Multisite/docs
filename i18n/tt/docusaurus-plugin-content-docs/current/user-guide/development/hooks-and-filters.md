# Ultimate Multisite Хуклары (Hooks) и Фильтрleri (Filters)

Ultimate Multisite функциялар ва база данных каулыndan турат. Онда сиз барлык функцияларга өзгөртүүләр yapmak ýa-da köpeltmek üçin хуклар (hooks) ва фильтрлер (filters) bar. Булар сизге mövcet kodlaryňizi üýtgetmek ýa-da täze kod goşmak üçin mümkinlik berýär.

Biz bu хукlar we фильtrler listini öwrenmek üçin, ýokary derejeli ulanyjylara ulgamyny has gowy özelleştirmegi kömek üçin bir referans hökmünde döredip bildik. Bu listäge Github-dan aşakdaky linke basyp girip bilersiňiz.
