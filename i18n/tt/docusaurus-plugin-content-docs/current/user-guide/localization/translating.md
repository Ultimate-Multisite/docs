# Ultimate Multisite-иң Түсирдәлештер {#translating-ultimate-multisite}

Ultimate Multisite англисче әраб ясалган, әмма плагин толуп түсирдәлешле һәм без актив булган түсирдәлешләрне кабул итәбез.

## Азырлык Плагин Түсирдәлешләре {#core-plugin-translations}

Азырлык Ultimate Multisite плагины расOfficial **WordPress.org-ның түсирдәлеш платформасы** аркылуу түсирдәлешле.

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

Бу стандарт WordPress җәмәгатьче түсирдәлеш системасы. Түсирдәлешләр жетдирелә тулы тасга кабул иткәндә, олар WordPress плагин güncelleшләре аркылуу Ultimate Multisite-ның барлык юлекле аんですが автомат түсуге исә ясала.

**Безнең көнегез кирәк!** Англисчедан башкача тел белмисезме? Түсирдәлешләрне кабул итү өчен мөмкин булган телендә түсирдәлешләрне ясарга карагыз. Көннеч сүзләр дә түсирдәлешргә ярдәм итә. Башкармаयला:

1. [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/) сайтына гапсыгыз
2. Өчендәге тилен суртыгыз
3. "Contribute Translation" (Түсирдәлеш ясарга) батыгыз
4. Siz WordPress.org-ның безнең өчен бушлай аんですが кирәк

Сезнең телен түсирдәлеш редактору булганга (түсирдәлешләрне кабул итәргә мөмкин) мөрәҗәгать etmek өчен [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions)-ы аркылуу китерәгез.

## Дорол Түсирдәлешләр (Addon Translations) {#addon-translations}

Дорол плагины башкача түсирдәлешләнгән:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Дорол түсирдәлешләре حالияттә AI-лар тарафын ясалган якын буларак башлангыч. Олар барлык тулы булган телен каплап кайтә, әмма сүзләр яки мәгълүматлар яки хаталар каралып кала. Без безнең өчен естественный түсирдәлешләрне карап чыгып, аларны артык табиий һәм дөрес булган була ба тулырсын дигән телекләрне сугларга кулларабыз.

## Мәхлис Түсирдәлешләр (Local Translations) {#local-translations}

Если вы предпочитаете делать переводы локально для своего проекта, а не вносить вклад в основной проект, есть два варианта:

### Использование Poedit {#using-poedit}

Вы можете использовать десктопное приложение **[Poedit](https://poedit.net)** для перевода файла `.pot`:

1. Скачайте Poedit с [poedit.net](https://poedit.net)
2. Откройте файл `.pot` из папки `wp-content/plugins/ultimate-multisite/lang/`
3. Переведите строки в интерфейсе Poedit
4. Сохраните -- Poedit создаст файлы `.po` и `.mo`
5. Поместите выходные файлы в папку `lang`

### Использование Loco Translate {#using-loco-translate}

Плагин WordPress **[Loco Translate](https://wordpress.org/plugins/loco-translate/)** позволяет переводить прямо из админ-панели без каких-либо настольных инструментов. Он сам занимается созданием и компиляцией файлов перевода для вас.
