# Эчке Agents

Gratis AI Agent v1.9.0 биш эчке агент белән килә; һәрберсе билгеле бер кораллар җыелмасы, махсуслаштырылган system prompt һәм шул өлкәдәге гадәти биремнәргә туры килгән башлангыч тәкъдимнәр белән алдан көйләнгән. Агентлар арасында күчү ассистентның нәрсә эшли алуын һәм ничек җавап бирүен үзгәртә — сезнең яктан бернинди көйләүсез. Superdav AI Agent v1.18.0 бәйле интеграцияләр көйләнгәндә бу эш агымнарына расписаниене исәпкә ала торган кораллар, искәртмә язмалары, раслау капкалары һәм SMS хәбәрнамәләре өсти ала.

## Agent нәрсә ул? {#what-is-an-agent}

Һәр агент — түбәндәгеләрне берләштергән исемләнгән көйләү профиле:

- **Кораллар** — агент чакыра ала торган мөмкинлекләр (мәсәлән, Content Writer язма булдыру мөмкинлекләренә керә ала; Design Studio CSS һәм theme.json мөмкинлекләренә керә ала)
- **System prompt** — агентның тонын, өстенлекләрен һәм чикләүләрен билгеләүче күрсәтмәләр
- **Тәкъдимнәр** — тиз башлап җибәрергә ярдәм итү өчен чат интерфейсында күрсәтелә торган алдан язылган promptлар

## Agent Picker ачу {#accessing-the-agent-picker}

1. WordPress идарә ян панеленнән **Gratis AI Agent** панелен ачыгыз.
2. Чат башлыгының өске сул ягындагы **агент иконасына** басыгыз (икона актив агентны күрсәтү өчен үзгәрә).
3. **Agent Picker** форма-таблица катламы буларак ачыла. Һәр агент үз иконасы, исеме һәм бер юллык тасвирламасы белән күрсәтелә.
4. Аны активлаштыру өчен агент юлына басыгыз. Чат башлыгы шунда ук яңартыла.

Сез шулай ук сөйләшү уртасында агентларны алыштыра аласыз — яңа агентның system prompt-ы киләсе хәбәрдән башлап көченә керә.

## Биш эчке Agent {#the-five-built-in-agents}

### Content Writer {#content-writer}

**Юнәлеш:** Язмалар, битләр һәм элемтә формалары булдыру һәм мөхәррирләү.

**Мөмкин кораллар:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`. Superdav AI Agent v1.18.0 интеграцияләре кабызылган булса, көйләнгән календарь контексты, раслау капкалары, искәртмәләр һәм SMS хәбәрнамә кораллары да расланган эш агымнары өчен мөмкин булырга мөмкин.

**Нәрсәне яхшы эшли:**
- Кыскача бирем яки планнан блог язмалары әзерләү һәм бастыру
- Яңа сайт өчен landing pageлар пакетларын булдыру
- Элемтә һәм сорау формалары төзү
- URL яки эзләү нәтиҗәсе буенча язмаларга төп рәсемнәр кую
- Көйләнгән Google Calendar контекстыннан чарадан соң җибәреләсе хәбәрләр әзерләү, аннары хәбәрнамәләр җибәрер алдыннан раслау өчен туктап тору

**Башлангыч тәкъдимнәр:**
- *WordPress multisite файдалары турында 500 сүзлек блог язмасы яз.*
- *About, Services һәм Contact битләрен булдыр һәм бастыр.*
- *Contact битенә броньлау соравы формасын өстә.*
- *Иртәгәге көйләнгән календарь чарасында катнашучылар өчен искәртмә әзерлә һәм аны җибәргәнче раслауны көт.*

---

### Site Builder {#site-builder}

**Юнәлеш:** Бер promptтан баштан ахыргача веб-сайт булдыру.

**Мөмкин кораллар:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`. Superdav AI Agent v1.18.0 белән, администраторлар кабызган урыннарда көйләнгән идарә ителә торган хезмәт, раслау, искәртмә, календарь һәм SMS кораллары мөмкин булырга мөмкин.

**Нәрсәне яхшы эшли:**
- Тасвирланган бизнес төре өчен күп этаплы сайт төзү планын булдыру
- Һәр этапны мөстәкыйль башкару — структура, эчтәлек, навигация, дизайн
- План уртасындагы хаталардан кул белән катнашу таләп итмичә чыгу
- Төзүнең бер өлеше буларак тәкъдим ителгән pluginнарны урнаштыру
- Чат интерфейсыннан турыдан-туры элемтә формалары булдыру (Superdav AI Agent v1.10.0+)
- Раслау капкалары һәм искәртмә язмалары кабызылганда, кабат хәбәрнамәләрсез launch искәртмәләрен яки катнашучыларга соңыннан хәбәр җибәрүне координацияләү

**Башлангыч тәкъдимнәр:**
- *Галерея язма төре, броньлау бите һәм элемтә формасы булган фотография портфолио сайты төзе.*
- *Онлайн менюсы, эш сәгатьләре һәм өстәл броньлау соравы формасы булган ресторан веб-сайты булдыр.*
- *Хезмәт битләре, портфолио бүлеге һәм блогы булган фриланс консалтинг сайты көйлә.*
- *Site builder ярдәмендә Contact битенә элемтә формасы өстә.*
- *Сайтны чыгару checklist-ы раслангач, көйләнгән кызыксынучы зат элемтәсенә SMS искәртмә җибәр.*

### Design Studio {#design-studio}

**Юнәлеш:** Визуаль көйләү — төсләр, типография, CSS һәм block patternнар.

**Мөмкин кораллар:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Нәрсәне яхшы эшли:**
- Исемләнгән theme presetларын куллану (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- theme.json аша глобаль типографияне һәм төс палитраларын нечкә көйләү
- Брендка хас өстәмә үзгәртүләр өчен custom CSS кертү
- Битнең screenshot-ын алу һәм аны дизайн мәсьәләләре буенча тикшерү

**Башлангыч тәкъдимнәр:**
- *warm-editorial preset-ын куллан, аннары төп төсне #2d6a4f итеп куй.*
- *Баш битнең screenshot-ын ал һәм нәрсәне яхшыртыр идең икәнен әйт.*
- *Тулы киңлектәге фон рәсеме һәм үзәкләштерелгән башлыгы булган кабат кулланыла торган hero block pattern булдыр.*

### Plugin Manager {#plugin-manager}

**Юнәлеш:** WordPress pluginнарын табу, урнаштыру һәм идарә итү.

**Мөмкин кораллар:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Нәрсәне яхшы эшли:**
- Тасвирланган куллану очрагы өчен иң яхшы plugin тәкъдим итү
- Реестрдан ability packлар урнаштыру
- Мөмкин ability каталогын категория буенча карау

**Башлангыч тәкъдимнәр:**
- *Membership directory өчен иң яхшы plugin нинди?*
- *WooCommerce abilities pack урнаштыр.*
- *Барлык мөмкин ecommerce ability packларны күрсәт.*

---

### Support Assistant {#support-assistant}

**Юнәлеш:** Сайт эчтәлеге, көйләүләр һәм WordPress конфигурациясе турында сорауларга җавап бирү.

**Мөмкин кораллар:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Нәрсәне яхшы башкара:**
- Агымдагы сайт көйләүләрен һәм параметрларын эзләү
- Сайтта нинди язма төрләре, таксономияләр һәм менюлар көйләнгәнен аңлату
- Тере кыйммәтләрне укып, "бу көйләү нәрсә эшли?" сорауларына җавап бирү
- Үзгәрешләр кертү алдыннан бары тик уку өчен диагностика катламы булып хезмәт итү

**Башлау өчен тәкъдимнәр:**
- *Бу сайтта хәзер нинди pluginнар һәм көйләүләр актив?*
- *Бу сайтта теркәлгән барлык махсус язма төрләрен санап чык.*
- *Нинди навигация менюлары бар һәм алар кайда билгеләнгән?*

## Superdav автоматлаштыру интеграцияләре {#superdav-automation-integrations}

Superdav AI Agent v1.18.0 интеграцияләре көйләнгәндә, эчке agentлар иминрәк, җәдвәлне исәпкә алган автоматлаштыру эш агымнарында катнаша ала:

- **Google Calendar уку кораллары** agentларга алга таба эш әзерләгәнче көйләнгән календарьларны һәм вакыйгаларны тикшерергә мөмкинлек бирә.
- **Контакт һәм катнашучы туры китерүе** вакыйга катнашучыларын WordPress кулланучыларына яки билгеле контактларга туры китерергә ярдәм итә.
- **Кеше раславы капкалары** сизгер гамәлләрне вәкаләтле кулланучы карап чыгып раслаганчы туктатып тора.
- **Искәртү язмалары** җәдвәлле эшләр кабатлап караганда яки кабатланганда дубликат хәбәрләрне булдырмый.
- **TextBee SMS хәбәрләре** көйләнгән текст хәбәрләрен SMS таныклыклары һәм эш агымы рөхсәтләре кабызылганда гына җибәрә.

Тәкъдим ителгән эш агымы: agentтан хәбәрне яки гамәлне әзерләвен сорагыз, раслау чакыруын карап чыгыгыз, аннары расланган гамәлне дәвам итәргә рөхсәт итегез. Кабатланучы искәртүләр өчен, бер үк вакыйгага яки контактка кат-кат хәбәр җибәрелмәсен өчен, искәртүләрне дубликатсызландыруны кабызылган килеш калдырыгыз.

## Agentларны көйләү {#customising-agents}

Һәр эчке agent `gratis_ai_agent_agents` filter аша киңәйтелә яки алыштырыла ала.

### Булган agentка махсус system prompt өстәү {#adding-a-custom-system-prompt-to-an-existing-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### Яңа agent теркәү {#registering-a-new-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

Яңа agent filter эшләгәннән соң шунда ук Agent Pickerда күренә.

### Эчке agentны бетерү {#removing-a-built-in-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
