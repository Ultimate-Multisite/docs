# Планов для обмена ссылками

Ultimate Multisite вам дает возможность делиться ссылками планов напрямую со своими клиентами. Чтобы это сделать, зайдите в любую страницу редактирования вашего плана и нажмите кнопку _Click to copy the Shareable Link_ (Нажмите, чтобы скопировать ссылку для обмена).

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_На странице редактирования Плана вы найдете кнопку для создания ссылок для обмена._

Обратите внимание, что этот механизм использует стандартную частоту выставления счетов для создания ссылки. Это значит, что если ваш стандартный расчет — ежемесячный, например, в итоге у вас будет такая ссылка: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Если вы хотите, чтобы ссылка для обмена по умолчанию была квартальной, вам нужно добавить 3 в конце:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Чтобы выбрать план _premium_ на годовом графике, вы добавите 1 и слово:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Помните, что у вас не может быть только одна ссылка для разных частот выставления счетов. Вам следует делиться одной конкретной ссылкой для каждой вашей частоты выставления счетов.

Как только вы поделитесь ссылкой со своими клиентами, она перенаправит их на первый шаг вашего процесса регистрации.
