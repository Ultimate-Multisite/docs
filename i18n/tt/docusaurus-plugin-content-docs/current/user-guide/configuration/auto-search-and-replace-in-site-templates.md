# Сайт Шаблондарында Авто Поиск и Замена (v2) {#auto-search-and-replace-in-site-templates-v2}

_**Этот туториал требует WP UItimo версии 2.x.**_

Ultimate Multisite-аның ең күшбәеше функциясы – ргистрация формасына арнайы текст, реңк һәм таңлаучы укылычалар (fields) коснурырга ясау. Бу мәгълүматны тапкан вакытта анда саптылган мәгълүмат белән сайт шаблоның бер bahagianларын тозlamak өчен бирергә мөмкин. Соң, янәш сайт ялгыннан чыкгач, Ultimate Multisite ошол укылычаларны ргистрациядә каралган нагыз мәгълүмат белән алмаштырып чыгары.

Мисалы, сиз шаблон сайтка укылычалар (placeholders) белән тозлап китә аласыз. Укылычалар дубль калиткалар белән — {{placeholder_name}} - белән ясала.

Соң, ошол мәгълүматны тапкан җәйге ргистрация укылычасын арттырып, анда каралган мәгълүматны тозlamak мөмкин.

Следовательно, ваш клиент сможет заполнить это поле во время регистрации.

Ultimate Multisite затем автоматически заменит плейсхолдеры данными, предоставленными клиентом.

## **Как решить проблему "шаблон, полный плейсхолдеров"** {#solving-the-template-full-of-placeholders-problem}

Все это замечательно, но у нас возникает неприятная проблема: теперь наши шаблоны сайтов — которые могут посещать наши клиенты — заполнены некрасивыми плейсхолдерами, которые мало что говорят.

Чтобы решить эту проблему, мы предлагаем опцию установки фиктивных значений для плейсхолдеров и используем эти значения для поиска и замены их содержимого на шаблонах сайтов во время посещения вашими клиентами.

Шаблон блокларын редакторуна барып китә аласыз: **Ultimate Multisite > Settings > Sites** (Уртал Сайт > Параметрләр > Сайтлар), анда Сайт Шаблоны Вариантлары (Site Template Options) bölmesин карап, **Edit Placeholders** (Блокларны Редакция) линкийне сыйлап китә аласыз.

![Sites settings page-дагы Site Template Options bölmesi](/img/config/settings-sites-templates-section.png)

Бу сизә блокларга һәм уларның мәгънәләрен (content) итеп диярлек яса ала торган редакторга китерәчәк.

![Template placeholders editor entry point](/img/config/settings-sites-templates-section.png)
