# GridPane Интеграция

## Обзор
GridPane — это специальная панель управления хостингом для WordPress, созданная для серьезных профессионалов в этой сфере. Эта интеграция позволяет автоматически синхронизировать домены и управлять SSL-сертификатами между Ultimate Multisite и GridPane.

## Возможности
- Автоматическая синхронизация доменов
- Управление SSL-сертификатами
- Автоматическая настройка константы SUNRISE

## Требования
Следующие константы должны быть определены в вашем файле `wp-config.php`:

```php
define('WU_GRIDPANE', true);
define('WU_GRIDPANE_API_KEY', 'ваш_api_ключ');
define('WU_GRIDPANE_SERVER_ID', 'ваш_server_id');
define('WU_GRIDPANE_APP_ID', 'ваш_site_id');
```

## Инструкции по настройке

### 1. Получите учетные данные API GridPane

1. Войдите в свою панель управления GridPane
2. Перейдите в "Настройки" > "API"
3. Сгенерируйте API-ключ, если у вас его еще нет
4. Скопируйте свой API-ключ

### 2. Получите ID вашего сервера и сайта

1. В панели управления GridPane перейдите в раздел "Серверы" (Servers)
2. Выберите сервер, где размещен ваш мультисайт WordPress
3. Запомните Server ID (он виден в URL или на странице деталей сервера)
4. Перейдите в "Сайты" (Sites) и выберите свой сайт WordPress
5. Запомните Site ID (он виден в URL или на странице деталей сайта)

### 3. Добавьте константы в wp-config.php

Добавьте следующие константы в ваш файл `wp-config.php`:

```php
define('WU_GRIDPANE', true);
define('WU_GRIDPANE_API_KEY', 'ваш_api_ключ');
define('WU_GRIDPANE_SERVER_ID', 'ваш_server_id');
define('WU_GRIDPANE_APP_ID', 'ваш_site_id');
```

### 4. Включите интеграцию

1. В админке WordPress перейдите в Ultimate Multisite > Настройки (Settings)
2. Перейдите на вкладку "Domain Mapping" (Сопоставление доменов)
3. Прокрутите вниз до раздела "Host Integrations" (Интеграции хостов)
4. Включите интеграцию GridPane
5. Нажмите "Save Changes" (Сохранить изменения)

## Как это работает

Когда домен сопоставляется в Ultimate Multisite:

1. Интеграция запроса отправляет в API GridPane, чтобы добавить домен на ваш сайт.
2. GridPane автоматически занимается предоставлением SSL-сертификатов.
3. Когда сопоставление домена удаляется, интеграция убирает этот домен из GridPane.

Интеграция также автоматически обрабатывает константу SUNRISE в вашем файле wp-config.php, которая нужна для корректной работы сопоставления доменов.

## Управление константой SUNRISE

Уникальная особенность интеграции с GridPane заключается в том, что она автоматически возвращает константу SUNRISE в wp-config.php к исходному значению, чтобы избежать конфликтов с собственной системой сопоставления доменов от GridPane. Это гарантирует, что обе системы могут работать вместе без проблем.

## Устранение неполадок (Troubleshooting)

### Проблемы с подключением к API
- Проверьте, правильный ли ваш API key.
- Убедитесь, что ваши сервер и ID сайта верны.
- Удостоверьтесь, что у вашей учетной записи в GridPane есть необходимые права доступа.

### Проблемы с SSL-сертификатами
- Генерация SSL-сертификатов может занять некоторое время.
- Проверьте, правильно ли ваши домены указывают на IP-адрес вашего сервера.
- Проверьте настройки SSL в GridPane для вашего сайта.

### Домен не добавлен
- Проверьте логи Ultimate Multisite на наличие каких-либо сообщений об ошибках.
- Убедитесь, что домен еще не добавлен в GridPane.
- Убедитесь, что DNS-записи вашего домена настроены правильно.
