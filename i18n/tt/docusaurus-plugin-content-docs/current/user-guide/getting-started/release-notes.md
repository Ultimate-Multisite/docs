# Выпускные заметки (Release Notes) {#release-notes}

## Версия 2.13.0 — Выпущена 5 июня 2026 г. {#version-2130--released-on-2026-06-05}

Новое: Клиентный аккаунт, оформление заказа, биллинг, сайт, инвойс, смена шаблонов и сопоставление доменов для суверенных арендаторов добавлены, чтобы сетевые арендаторы могли направлять клиентов обратно на основной сайт для управляемых действий.

Новое: Появились проверки обновлений учетных данных для повторяющихся подписок, чтобы шлюзы могли отключать автопродление, если отсутствует сохраненное соглашение об оплате, подписка или токен хранилища (vault token).

Новое: Проведена проверка публикации с использованием HMAC для создания новых сайтов в ожидании, чтобы сделать процесс настройки сайта после оформления заказа более надежным на хостингах, где задержки с фоновыми задачами могут возникать.

- Новый пункт: Добавлены точки расширения для URL SSO, базовых доменов форм оформления заказа и автоматического создания записей в домене.
- Исправление: SSO стал более надежным при работе с сопоставленными доменами, анонимными визитами через брокера, выходе из системы и конфликтами между плагинами.

Исправление: Тепер пасующий сайт создавалы (Pending site creation) восстанавливается со старых флагов публікации и избегает того, чтобы клиенты застряли на экране создания сайта.

Исправление: Облачных форм для общих доменов больше не создаются, а фоновые задания провайдеров хостинга, которые не используются при отсутствии активной интеграции, пропускаются.

Исправление: Проблемы с оформлением заказа (Checkout), адресом выставления счета (billing address), сбросом пароля, верификацией электронной почты, переключением шаблонов, турами и дашбордом клиента больше не блокируют обычные процессы для покупателей.

Исправление: Рассылать электронные письма теперь сохраняют конфиденциальность получателей, избегая при этом фатальных ошибок SMTP/плагинов, когда списки получателей или почтовые транспортные системы не работают.

Исправление: Теперь проблемы с продлением членства, отображением сроков действия и обработкой платежей устраняются. Это помогает избежать немедленного истечения срока действия, сбоев или пропущенных обязательных платежей.

- Улучшено: WordPress uyumluluğu 7.0'a kadar test edildi, üretim Vue varlıkları npm kaynaklarından yeniden oluşturuluyor ve Cypress uçtan uca kapsama alanı artık daha fazla ödeme, kurulum, SSO ve gateway akışlarını test ediyor.

## Sürüm 2.12.0 — Yayın tarihi: 2026-05-15 {#version-2120--released-on-2026-05-15}

- Новый пункт: Добавлен хостинг-провайдер Hostinger (hPanel) с интеграцией со связыванием доменов.
- Новый пункт: Экспортер сайта теперь обрабатывает пакеты импорта сети для упрощенного восстановления сайта по всей сети.

- Исправление: Теперь рассылки по BCC используют заголовок `undisclosed-recipients`, чтобы не показывать адреса получателей.
- Исправление: Дата истечения срока действия членства больше не повреждается при сохранении с некорректным значением даты.

- Исправление: Обновления членства в Stripe теперь корректно сбрасывают скидки, не вызывая устаревший API `deleteDiscount`.
- Исправление: Перенаправления SSO на сайтах с доменным маппингом теперь ограничены для предотвращения бесконечных циклов перенаправлений.

- Исправление: Выбор изображения в мастер выбора теперь корректно обновляет базовую модель данных
- Исправление: Сайт Экспортер CLI теперь сохраняет правильный выбор сетевого сайта по умолчанию

- Улучшено: Плагин пакетинен wp-cli-ны пакетtenннан ачыкылды, плагины хәзерге килмичә күчерә.

## 2.11.0 версиясы — 2026-05-11-де ачылды {#version-2110--released-on-2026-05-11}

Новое: Сайт экспортасы indi.php файлды öz-başlanýan (self-booting) görnüşde paketler, şonda ZIP faylyndan ayrı bir plugin yüklemeksiz täze hostingde goşdirip bilersiňiz.

Новое: Функция экспорта по сети позволяет администраторам экспортировать все подсайты в один архив со страницы администрирования экспорта сайта.

- Новый пункт: Теперь переключатель плана сайта (Site Templates plan toggle) работает по цепочке резервного варианта (fallback chain), что правильно ограничивает доступ к шаблонам в зависимости от лимитов плана.
- Новый пункт: Редактор формы оформления заказа (Checkout form editor) предупреждает, если вы добавляете продукт без настроенного обязательного поля.

Новое: вкладка настроек импорта/экспорта теперь четко описывает ее область применения и напрямую ведет к инструменту экспорта сайта.

## Версия 2.10.0 — Выпущена 2026-05-05 {#version-2100--released-on-2026-05-05}

- Новый: PayPal-данных ручного ввода için rehber kurulum sihirbazı, sorunsuz ödeme ağ geçidi yapılandırması için OAuth bayrak kilitlemesiyle birlikte.

- Новый пункт: Клиентский панель (customer panel) новый шаблоном переработан с использованием карточки текущего шаблона (current-template card), постоянной сетки (persistent grid) и кнопкой **Сбросить текущий шаблон** (Reset current template).
- Исправление: Переключение шаблонов больше не блокирует интерфейс при сбое AJAX.

- Исправление: Разрешения на переключение шаблонов защищены от несанкционированного доступа.
- Исправление: Ввод настроек, переопределяющих сайт, теперь проверяется перед сохранением.
- Исправление: Теперь запрос адреса выставления счета отображается, когда адрес пуст.

- Исправление: Устранены предупреждения о несовместимости "null-to-string" в PHP 8.1.
- Исправление: Текущая загрузка ленивых данных (lazy-loaded) перед хуком `init` для предотвращения проблем с таймингом.
- Исправление: Путь SSO фильтрации соблюдается во всех сценариях входа.
- Исправление: Опции пустой идентичности сайта сохраняются при сохранении.
