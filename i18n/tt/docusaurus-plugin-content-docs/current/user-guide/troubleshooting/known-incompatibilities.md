# Başqa Pluginlarga Mümkün Olmayan Uyumluluklar

Ultimate Multisite-ı quraşdırdan vaqaytaqtan vaqaytaqtan, başqa pluginlarlarla çözülmemiş uyğunsuzluklar sebäbindən problem yaşa bilmösünüz.

Biz her yeni versiyada bu ro'yxatni kamaytirish uchun faol ishlayapmiz, lekin agar biror plugin shu yerda ro'yxatdan olingan bo'lsa, to'liq funksional Ultimate Multisite tarmoqini olish uchun uni o'chirish kerak bo'lishi mumkin.

Mana ularning ro'yxati va bu holatlarda nima qilish kerakligi:

**Ba'zi WPMU DEV pluginlari** WPMU DEV tomonidan endi texnik jihatdan qo'llab-quvvatlanmaydigan ba'zi pluginlar Ultimate Multisite ro'yxatdan o'tish jarayonini oxirgi bosqichda (foydalanuvchi hisob yarata olmaydi) to'xtatishi mumkinligi ma'lum. Bu muammolarga sabab bo'ladigan pluginlar ro'yxati:

  * Signup Language: _Ultimate Multisite-da bu funksionallikni taqdim etadigan Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Bu Ultimate Multisite ichiga kiritilgan, shuning uchun qo'shimcha pluginga ehtiyoj yo'q._

  * New Blog Templates: _Bu Ultimate Multisite ichiga kiritilgan, shuning uchun qo'shimcha pluginga ehtiyoj yo'q._

  * Set Password on Multisite Blog Creation: _Bu Ultimate Multisite ichiga kiritilgan, shuning uchun qo'shimcha pluginga ehtiyoj yo'q._

  * Signup TOS: _Bu Ultimate Multisite ichiga kiritilgan, shuning uchun qo'shimcha pluginga ehtiyoj yo'q._

  * Anti-Splog: Garchi bu kam kuchliroq bo'lsa ham, u ham _Ultimate Multisite ichiga kiritilgan._

**Eslatma**: Boshqa barcha WPMU Dev pluginlarini Ultimate Multisite bilan normal tarzda ishlatish mumkin. Misollar: _Smush_ , _Forminator_ , _Defender_, va h.k.
