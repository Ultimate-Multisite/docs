# Настройки для справки

Эта страница отслеживает настройки, которые влияют на повседневное администрирование и недавние изменения в поведении Ultimate Multisite.

## Другие опции

Раздел **Other Options** (Другие опции) находится под пунктом **Ultimate Multisite > Settings > Login & Registration** (Настройки > Вход и регистрация).

| Настройка | Описание |
|---|---|
| **Enable Jumper** (Включить Переход) | Показывает инструмент быстрой навигации Jumper в административной области. Используйте его, чтобы сразу перейти к экранам Ultimate Multisite, сетевым объектам и поддерживаемым административным адресам. Отключите его, если вы не хотите видеть эту кнопку-ярлык. |

## Сообщения об ошибках и телеметрия

Настройки включения предыдущей опции отправки сообщений об ошибках были удалены со страницы настроек. Анонимная телеметрия отключена, и нет кнопки в интерфейсе для ее включения.

Если вы поддерживаете внутренние руководства (runbooks) или скриншоты для страницы настроек, удалите ссылки на старое поле включения отправки сообщений об ошибках, чтобы администраторы не искали настройку, которой больше нет.

## Настройки импорта/экспорта

Вкладка **Import/Export** (Импорт/Экспорт) описывает, какие настройки она контролирует, и напрямую связана с **Ultimate Multisite > Site Export** (Настройки сайта > Экспорт сайта) для архивов сайтов и сети. Используйте вкладку настроек для конфигурации импорта/экспорта, используйте **Tools > Export & Import** (Инструменты > Экспорт и импорт) для рабочего процесса экспорта/импорта для одного сайта, а инструмент Site Export используйте, когда вам нужен полный архив экспорта сети.

## Настройки подключения провайдера ИИ

Настройки подключения провайдера ИИ теперь показывают только поддерживаемые пулы учетных записей OAuth:

| Провайдер | Настройка |
|---|---|
| **Anthropic Max** | Подключите одну или несколько учетных записей Anthropic Max с помощью кнопки OAuth. Используйте ручной метод OAuth в качестве запасного варианта, если песочница браузера не может автоматически завершить перенаправление. |
| **OpenAI ChatGPT/Codex** | Подключите учетные записи ChatGPT через тот же рабочий процесс пула OAuth. Операции, поддерживаемые коннектором, могут использовать вызовы инструментов ChatGPT Codex после подключения учетной записи. |
| **Google AI Pro** | Подключите учетные записи Google AI Pro через OAuth, а затем обновите коннектор, если список учетных записей не обновляется немедленно. |

Cursor Pro больше не является поддерживаемым провайдером. Удалите старые внутренние скриншоты, раунбуки или шаги по адаптации, которые упоминают поля настройки Cursor Pro или пути коннекторов.

При добавлении или удалении учетных записей провайдера вводите действительный адрес электронной почты для обновляемой или удаляемой учетной записи и сохраняйте настройки провайдера перед тестированием операций с поддержкой коннектора.
