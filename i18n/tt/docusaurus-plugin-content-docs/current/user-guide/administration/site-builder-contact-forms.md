# Сайт Билдер Контактные Формы

Superdav AI Agent v1.10.0-адация позволяет создавать контактные формы прямо из чатового интерфейса с помощью агента Site Builder. Это дает вам возможность добавлять полностью рабочие контактные формы на любую страницу, не покидая чат.

## Обзор

Возможность создания контактной формы через Site Builder (`create_contact_form`) автоматически обнаруживает ваш установленный плагин для форм и создает контактную форму, используя встроенные возможности этого плагина. Поддерживаемые плагины для форм включают:

- Contact Form 7
- WPForms
- Fluent Forms
- Gravity Forms

Ассистент автоматически выбирает лучший доступный плагин для форм на вашем сайте и генерирует контактную форму, настроенную под этот плагин.

## Создание Контактной Формы

Чтобы создать контактную форму с помощью Site Builder:

1. Откройте чат-панель **Gratis AI Agent** в админке WordPress.
2. Переключитесь на агента **Site Builder**, нажав на значок агента в шапке чата.
3. Опишите контактную форму, которую вы хотите создать. Например:

   > "Добавить контактную форму на страницу Контакты с полями для имени, email, сообщения и номера телефона."

   Или просто:

   > "Создать контактную форму для страницы Контакты."

4. Site Builder сгенерирует контактную форму и вернет шорткод, готовый к вставке.

## Использование Сгенерированного Шорткода

После того как Site Builder создаст контактную форму, он вернет шорткод (например, `[contact-form-7 id="123"]`). Вы можете:

1. **Вставить его на страницу или запись** — Скопируйте шорткод и вставьте его в любую страницу или запись с помощью редактора блоков или классического редактора.
2. **Добавить его через Site Builder** — Попросите Site Builder автоматически добавить форму на определенную страницу:

   > "Добавить контактную форму на страницу Контакты."

3. **Оны шаблонда дагын кулланыңар** — Хэрэв сиз PHP-де ыңгайлы итегез, шорткодны тема шаблон файлына нэмэге исәлегез.

## Контакт формасын өзгөртүү

Сайт Билдер контакт формасын ясап биргэне, анда дагы өзгөртүчеләр дела аласыз:

### Чат интерфейсы аркылуу

Сайт Билдергә форманы өзгөртү дисез:

> "Контакт формасын тема (subject) укыйка илалы кылып һәм хабарлама укыйка мөрәҗәгать итүне обязательный (required) кыла."

Сайт Билдер форманы өзгөртә һәм өзгөртүче шорткодны кертеп керә.

### Форма плагинының административ интерфейсы аркылуу

Сез форманы милләтеп булган форма плагинының настройкаларында да икеле өзгөртүчеләр дела аласыз:

1. WordPress административсында **Contact Form 7**-гә (яки куллап калган форма плагиныгызга) кибегез.
2. Сайт Билдер тарафыннан ясалган форманы табыгыз.
3. Мәхмүд булган вакытта форма укыйкалар, валидация кагыйдәләре һәм электрон почта хабарламаларын өзгөртә аласыз.

## Форма плагины һәм юнәлешү (Compatibility)

Сайт Билдер аның сайтыгызга нич форма плагины куллап калган дип автоматдык танып, контакт формасын ясау өчен анда булган плагины куллап калат. Бик тормышта форма плагины куллап калган вакытта бирнечләр бар дисезме? Сайт Билдер төп урында бу юнәлешү кара:

1. Contact Form 7
2. WPForms
3. Fluent Forms
4. Gravity Forms

Бу плагины ничене куллап калган булмаган вакытта, контакт формасын ясаудан мухитасынан бирене куллап кала дип суглара.

## Электрон почта хабарламалары (Email Notifications)

Сайт Билдер тарафыннан ясалган контакт формалары умолчанию сайт администраторына электрон почта хабарламалары göndermәге настройлага булган. Сез укыйка адресын һәм хабарлама мәтен өзгөртә аласыз:

1. Форма плагинының административ интерфейсына кибегез.
2. Сайт Билдер тарафыннан ясалган форманы табыгыз.
3. Электрон почта хабарламалары настройкаларын өзгөртә аласыз.

Төбәк тасвирлар өчен, форма плагиныгызның документациясын карагыз:

- [Contact Form 7 Email Notifications](https://contactform7.com/docs/)
- [WPForms Email Notifications](https://wpforms.com/docs/)
- [Fluent Forms Email Notifications](https://fluentforms.com/docs/)
- [Gravity Forms Email Notifications](https://docs.gravityforms.com/)

## Примеры использования (Example Use Cases)

### Простая контактная форма (Simple Contact Form)

> "Создайте простую контактную форму с полями для имени, электронной почты и сообщения."

### Многошаговая форма (Multi-Step Form)

> "Создайте контактную форму со вторым шагом для информации о контакте и вторым шагом для сообщения и предпочтительного способа связи."

### Форма с условной логикой (Form with Conditional Logic)

> "Создайте контактную форму, которая показывает разные поля в зависимости от выбора пользователя в выпадающем списке."

### Форма с загрузкой файлов (Form with File Upload)

> "Создайте контактную форму, которая позволяет пользователям загружать файл или вложение."

:::note
Создание контактной формы доступно в Superdav AI Agent v1.10.0 и более поздних версиях. Агент Site Builder должен быть активен для использования этой возможности.
:::
