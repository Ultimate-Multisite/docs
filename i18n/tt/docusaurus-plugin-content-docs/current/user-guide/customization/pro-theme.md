# Административную панель настраивать с PRO-темой

Клиентские работы делаるとき часто нам нужно предоставить клиентам кастомный опыт в админке WordPress. Если вы ищете более профессиональный вид, PRO Theme вам может помочь.

PRO Theme это админ-тема, разработанная для помощи вам при настройке дашборда ваших клиентов. И она также является частью премиум-плагинов Ultimate Multisite core.

Установив ее, перейдите на страницу _Plugins_ и нажмите, чтобы активировать ее в вашей Сети (Network).

<!-- Скриншот недоступен: Страница WP Plugins с кнопкой Активировать для PRO Theme -->

_Активируйте ее по всей сети на странице Plugins_

_**ПРИМЕЧАНИЕ:**_ _Если вы просто установите ее, она появится у вашего клиента как опция, которую можно отметить, чтобы добавить тему только для этого пользователя._

Теперь у вас полностью переработанный интерфейс администрирования! У вас также появилось новое меню слева с названием PRO Theme.

<!-- Скриншот недоступен: Переработанный интерфейс админ-панели PRO Theme -->

_Совершенно новый вид вашей Админ-панели_

В этом меню вы получите доступ к пяти вкладкам:

<!-- Скриншот недоступен: Вкладка Настройки стилей PRO Theme -->

_Настройки стилей (Styling Settings)_

<!-- Скриншот недоступен: Вкладка Функциональность PRO Theme -->

_Функциональность (Functionality)_

<!-- Скриншот недоступен: Вкладка Пользовательский CSS PRO Theme -->

_Пользовательский CSS (Custom CSS)_

<!-- Скриншот недоступен: Вкладка Импорт/Экспорт PRO Theme -->

_Импорт/Экспорт (Import/Export)_

<!-- Скриншот недоступен: Вкладка Активировать PRO Theme -->

_Активировать PRO Theme (Activate PRO Theme)_

В каждой из них вы можете включать и отключать элементы стилизации, разрешать или блокировать функциональность, а также предоставлять вашим клиентам ту кастомизацию, которую они могут искать в вашей сети сайтов.
