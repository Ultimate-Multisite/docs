# İlk dəfə hesabım səhifəsinə necə daxil ola bilərəm?

Ultimate Multisite-a xoş gəlmisiniz!

Bizim pluginimizi aldıqdan sonra e-poçtda aldığınız link vasitəsilə hesabınıza daxil ola bilərsiniz:

<!-- Screenshot unavailable: Account page email with access link -->

Alışveriş zamanı verdiyiniz e-poçtu və e-poçtda aldığınız şifrəni istifadə edin.

<!-- Screenshot unavailable: Login page with email and password fields -->

Bu hesabdaya ilk dəfə daxil olduğunuz üçün sizdən şifrənizi dəyişməyiniz soruşulacaq. Şifrənizi dəyişin və _Save_ düyməsini basın.

<!-- Screenshot unavailable: Change password form with Save button -->

Hesab səhifəsində faturalandırma, yeniləmələr, ləğv etmələr və çox başqa şeylər kimi bütün hesab detallarınızı idarə edə bilərsiniz.
