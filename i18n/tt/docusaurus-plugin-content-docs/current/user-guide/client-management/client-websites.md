# Клиентов сайтларыңыз (v2) {#your-clients-websites-v2}

Şebekeye kaydolduktan sonra müşteriler kendi panellerine erişebilirler.

Sadece ödemeleri, üyelikleri, alan adları, plan kısıtlamaları ve daha fazlası hakkında önemli bilgileri görebildikleri **Hesap** sayfası dışında, tüm web sitelerini görebilecekleri ve yönetecekleri **Benim Sitelerim** sayfasına da erişebilirler. Yapmaları gereken tek şey panellerinde Benim Sitelerim'e tıklamaktır.

![Müşteri paneli üzerindeki Benim Sitelerim bağlantısı](/img/account-page/my-sites-link.png)

Tüm web siteleri onlara gösterilecek ve her birinin hemen altında bir **Yönet** butonu olacak. Ona tıkladıklarında, o belirli web sitesinin **paneline** yönlendirilecektir.

Ayrıca birden fazla web sitesine izin veren planlar için hangi web sitesinin **birincil** olduğunu da görebilirler.

![Müşteri web sitelerini Yönet butonları ile gösteren Benim Sitelerim sayfası](/img/account-page/my-sites-page.png)

İşte site düzenleme arayüzünün tam görünümü:

![Site düzenleme tam arayüzü](/img/admin/site-edit-full.png)

Birincil web sitesini değiştirmek istiyorlarsa, **Hesap** menüsüne tıklayarak Hesap sayfasına gitmeleri gerekir.

![Müşteri paneli üzerindeki Hesap menüsü bağlantısı](/img/account-page/account-menu.png)

Ayrıca müşteri detaylarını da görüntüleyip düzenleyebilirler:

![Müşteri düzenleme arayüzü](/img/admin/customer-edit.png)

Sonra, **Varsayılan Siteyi Değiştir**'e tıklamalılar.

![Hesap sayfasındaki Varsayılan Siteyi Değiştir butonu](/img/account-page/change-default-site.png)

Bir açılır pencere görünecektir. Açılır listede hangi web sitesini birincil site yapmak istediğinizi seçin.

![Varsayılan Siteyi Değiştir açılır menü seçimi](/img/account-page/change-default-site-dropdown.png)

Sonra, sadece **Varsayılan Siteyi Değiştir**'e tıklayın ve işiniz bitmiş olur!
