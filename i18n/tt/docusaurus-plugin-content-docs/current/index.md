# Ultimate Multisite документациясе

Ultimate Multisite документациясенә рәхим итегез. Ultimate Multisite WordPress Multisite-ны тулы Website as a Service (WaaS) платформасына әйләндерә.

## Документация бүлекләре {#documentation-sections}

### [Кулланучы кулланмасы](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

WaaS платформагызны урнаштыру, көйләү һәм идарә итү өчен кирәк булган бар нәрсә. Урнаштыруны, түләү шлюзларын, домен бәйләүне, клиентлар белән идарә итүне һәм башкаларны үз эченә ала.

### [Эшләүчеләр документациясе](/developer/hooks) {#developer-documentation}

Ultimate Multisite нигезендә эшләүче эшләүчеләр өчен техник белешмәлек. Hooks белешмәлеген, REST API документациясен һәм addon эшләү кулланмаларын үз эченә ала.

### [Addon-нар](/addons) {#addons}

Ultimate Multisite-ны WooCommerce интеграциясе, аналитика, домен сату һәм башка өстәмә мөмкинлекләр белән киңәйтә торган түләүле addon плагиннары өчен документация.

- [GitHub мәсьәләләре](https://github.com/Ultimate-Multisite/ultimate-multisite/issues) -- Хаталар турында хәбәр итегез
- [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) -- Төп плагинны йөкләп алыгыз
