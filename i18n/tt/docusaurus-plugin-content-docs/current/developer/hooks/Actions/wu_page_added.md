# Гамәл: wu_page_added

plugin эшләүчеләргә битләр теркәлгәндә өстәмә эшләр башкарырга мөмкинлек бирә.

Бары тик билгеле бер бит каралганда гына эшли торган wu_page_load белән чагыштырганда, бу hook Ultimate Multisite коды ярдәмендә өстәлә торган һәр admin бите өчен теркәү вакытында эшли.

## Параметрлар

|------|------|-------------|
