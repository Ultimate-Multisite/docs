# Filter: wu_hosting_integration_wizard_sections

Хостинг интеграциясен көйләү өчен мастер бүлекләрен фильтрлый.

Кушымталарга мастер бүлекләрен өстәргә, бетерергә яки үзгәртергә мөмкинлек бирә.

## Параметрлар {#parameters}

| Исем | Төр | Тасвирлама |
|------|------|-------------|
| $sections | `array` | Мастер бүлекләре. |
| $integration | `\WP_Ultimo\Integrations\Host_Providers\Base_Host_Provider\|\WP_Ultimo\Integrations\Integration` | Көйләнә торган интеграция. |
| $page | `\Hosting_Integration_Wizard_Admin_Page` | Мастер битенең инстанциясе. |

### Бирле {#since}

- 2.5.0
