# Сөзгеч: wu_admin_notices {#filter-wuadminnotices}

Эшләүчеләргә Ultimate Multisite тарафыннан өстәлгән идарәче белдерүләрен сөзәргә мөмкинлек бирә.

## Параметрлар {#parameters}

| Исем | Төр | Тасвирлама |
|------|------|-------------|
| $notices | `array` | Шул конкрет панель өчен белдерүләр исемлеге. |
| $all_notices | `array` | Панельләр буенча аерылган өстәлгән белдерүләр исемлеге. |
| $panel | `string` | Белдерүләрне алу өчен панель. |
| $filter | `string` | Әгәр кире кагыла торган белдерүләр сөзеп алынган булса. |
| $dismissed_messages | `array` | Кире кагылган белдерү ачкычлары исемлеге. |

### Версиядән башлап {#since}

- 2.0.0
