# Фильтр: wu_apply_plan_limits

Плагин эшләүчеләренә чикләүләрне кыска юл белән урап үтәргә мөмкинлек бирә.

Сез бу фильтрны чикләүләрнең берсе дә башланыр алдыннан теләсә нинди код эшләтү өчен куллана аласыз. Әгәр фильтрыгыз берәр truthy кыйммәт кайтарса, процесс дәвам итәчәк; әгәр ул берәр falsy кыйммәт кайтарса, код кире кайтачак һәм түбәндәге hookларның берсе дә эшләмәячәк.
