# Фильтр: wu_post_count

Плагин эшләүчеләргә гомуми санны үзгәртергә мөмкинлек бирә

## Параметрлар

| Исем | Төр | Тасвирлама |
|------|------|-------------|
| $count | `int` | Язмаларның гомуми саны |
| $post_counts | `object` | `wp_count_posts` fn тарафыннан кайтарылган WordPress объекты |
| $post_type | `string` | Язма төренең slug-ы |
