# Фильтр: wu_minimum_password_length

Минималь пароль озынлыгын фильтрлый.

Бары тик wu_enforce_password_rules true булганда гына мәҗбүри кулланыла.

## Параметрлар {#parameters}

| Исем | Төр | Тасвирлама |
|------|------|-------------|
| $min_length | `int` | Минималь пароль озынлыгы. Килешү буенча 12 (Defender Pro белән туры килә). |
| $defender_active | `bool` | Defender Pro Strong Password активмы-юкмы. |

### Кайчаннан бирле {#since}

- 2.4.0
