# Фильтр: wp_ultimo_render_vars

Плагин эшләүчеләренә күрсәтү контекстына глобаль рәвештә өстәмә үзгәреш өстәргә мөмкинлек бирә.

## Параметрлар

| Исем | Төр | Тасвирлама |
|------|------|-------------|
| $args | `array` | Күрсәтү чакыруы аша тапшырылган үзгәрешләрне үз эченә алган массив. |
| $view | `string` | Күрсәтеләчәк күренеш исеме. |
| $default_view | `string` | fallback_view исеме |

### Кайчаннан бирле

- 2.0.0
