# Сөзгеч: wu_stripe_generate_idempotency_key

Stripe түләү параметрлары белән җибәрелә торган idempotency_key кыйммәтен сөзә.

## Параметрлар

| Исем | Төр | Тасвирлама |
|------|------|-------------|
| $idempotency_key | `string` | Идемпотентлык ачкычы кыйммәте. |
| $args | `array` | Ачкычны булдырырга ярдәм итү өчен кулланыла торган аргументлар. |
| $context | `string` | Идемпотентлык ачкычы булдырыла торган контекст. |

### Версиядән

- 3.5.0
