# Фильтр: wu_pre_save_settings

Төзүчеләргә көйләүләрне Ultimate Multisite тарафыннан сакланганчы фильтрларга мөмкинлек бирә.

## Параметрлар {#parameters}

| Исем | Төр | Тасвирлама |
|------|------|-------------|
| $settings | `array` | Сакланачак көйләүләр. |
| $settings_to_save | `array` | Өстәләчәк яңа көйләүләр. |
| $saved_settings | `array` | Хәзерге сакланган көйләүләр. |

### Версиядән башлап {#since}

- 2.0.18
