# Filter: wu_documentation_get_link

Плагин эшләүчеләренә сылтамаларны фильтрларга мөмкинлек бирә.

Моның ничек файдалы була алуына ышаныч юк, әмма булуы зыян итми

## Параметрлар {#parameters}

| Исем | Төр | Тасвирлама |
|------|------|-------------|
| $link | `string` | Теркәлгән сылтама |
| $slug | `string` | Сылтаманы алу өчен кулланылган slug |
| $default_link | `string` | Теркәлгән стандарт сылтама |
