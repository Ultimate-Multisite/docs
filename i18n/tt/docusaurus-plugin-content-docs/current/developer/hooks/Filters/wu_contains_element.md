# Фильтр: wu_contains_element

Эшләүчеләргә башлангыч эзләү нәтиҗәләрен үзгәртергә мөмкинлек бирә.

Бу өченче як төзүчеләре һәм шуңа охшашлар өчен файдалы.

## Параметрлар

| Исем | Төр | Тасвирлама |
|------|------|-------------|
| $contains_elements | `bool` | Әгәр элемент эчтәлектә булса. |
| $content | `string` | Тикшерелә торган эчтәлек. |
| $element | `self` | Хәзерге элемент. |
| $post | `null\|\WP_Post` | тикшереләсе язма. |
