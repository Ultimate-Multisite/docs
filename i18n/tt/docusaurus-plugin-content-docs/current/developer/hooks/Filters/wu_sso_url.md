# Фильтр: wu_sso_url

Кросс-домен клиент гамәлләре өчен кире кайтарылганчы, ясалган SSO URL-ларын фильтрлый.

Бу фильтрны интеграция суверен-тенант SSO сылтамасына ышанычлы контекст өстәргә яки Ultimate Multisite-ның токен тикшерүен саклап, брокер URL-ын алыштырырга тиеш булганда кулланыгыз.

## Параметрлар

| Исем | Төр | Тасвирлама |
|------|------|-------------|
| $sso_url | `string` | Ясалган SSO URL. |
| $user | `WP_User` | SSO керүе аша аутентификацияләнәчәк кулланучы. |
| $site_id | `int` | Керү өчен максат сайт ID-сы. |
| $redirect_to | `string` | SSO тикшерүе уңышлы узганнан соң билгеләнгән URL. |

### Кайчаннан

- 2.13.0

### Чыганак

`inc/sso/class-sso.php` эчендә билгеләнгән.


## Кайтара

Фильтрланган SSO URL.
