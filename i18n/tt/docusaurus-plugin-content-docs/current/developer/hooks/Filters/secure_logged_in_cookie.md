# Сөзгеч: secure_logged_in_cookie

Кергән кулланучы кукие бары тик HTTPS аша гына җибәрелергә тиешме-юкмы икәнен сөзә.

## Параметрлар

| Исем | Төр | Тасвирлама |
|------|------|-------------|
| $secure_logged_in_cookie | `bool` | Кергән кулланучы кукие бары тик HTTPS аша гына җибәрелергә тиешме-юкмы. |
| $user_id | `int` | Кулланучы ID. |
| $secure | `bool` | Аутентификация кукие бары тик HTTPS аша гына җибәрелергә тиешме-юкмы. |
