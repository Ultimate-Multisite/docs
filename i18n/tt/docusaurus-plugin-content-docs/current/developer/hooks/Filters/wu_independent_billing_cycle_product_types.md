# Фильтр: wu_independent_billing_cycle_product_types

Мөстәкыйль түләү цикллары булган продукт төрләрен фильтрлый.

Мөстәкыйль түләү цикллары булган продуктлар кәрзиндә планның түләү чорына туры килергә мәҗбүр ителми. Бу, план айлык яки еллык булуына карамастан, һәрвакыт ел саен түләнә торган домен теркәүләре кебек продуктлар өчен файдалы.

## Параметрлар

| Исем | Төр | Тасвирлама |
|------|------|-------------|
| $types | `array` | Мөстәкыйль түләү цикллары булган продукт төрләре массивы. |

- 2.5.0
