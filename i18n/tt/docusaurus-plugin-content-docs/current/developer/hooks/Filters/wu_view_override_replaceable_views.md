# Фильтр: wu_view_override_replaceable_views {#filter-wuviewoverridereplaceableviews}

Эшләүчеләргә алмаштырыла торган исемлеккә өстәмә папкалар өстәргә мөмкинлек бирә.

Сак булыгыз, чөнки өстәмә папкаларга рөхсәт бирү Ultimate Multisite версияләре урынына искергән күчермәләр йөкләнүенә китерергә мөмкин.

## Параметрлар {#parameters}

| Исем | Төр | Тасвирлама |
|------|------|-------------|
| $replaceable_views | `array` | Рөхсәт ителгән папкалар исемлеге. |

### Барлыкка килгән версия {#since}

- 2.0.0
