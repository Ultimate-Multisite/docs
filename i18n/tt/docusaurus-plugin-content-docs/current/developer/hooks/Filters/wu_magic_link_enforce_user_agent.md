# Фильтр: wu_magic_link_enforce_user_agent {#filter-wumagiclinkenforceuseragent}

Кулланучы агентын тикшерүне мәҗбүри итү-итмәүне фильтрлый.

Токеннарга төрле браузерларда/җайланмаларда эшләргә рөхсәт итү өчен false итеп куегыз. Бу иминлекне киметә, ләкин куллану уңайлылыгын арттыра.

## Параметрлар {#parameters}

| Исем | Төр | Тасвирлама |
|------|------|-------------|
| $enforce | `bool` | Кулланучы агентының туры килүен мәҗбүри итәргәме. |

- 2.0.0
