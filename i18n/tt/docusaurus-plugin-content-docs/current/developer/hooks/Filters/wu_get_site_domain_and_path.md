# Filter: wu_get_site_domain_and_path

Эшләүчеләргә домен/юл парларын үзгәртергә мөмкинлек бирә.

Бу төрле максатлар өчен файдалы булырга мөмкин, мәсәлән, ниндидер staging чишелешен, төрле серверларны һ.б. гамәлгә ашыру өчен.

## Параметрлар

| Исем | Төр | Тасвирлама |
|------|------|-------------|
| $d | `object` | Домен һәм юл ачкычларын үз эченә алган хәзерге объект. |
| $path_or_subdomain | `string` | Функциягә тапшырылган башлангыч юл/субдомен. |

### Версиядән

- 2.0.0
### Чыганак

[`inc/functions/site.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/site.php#L235) эчендә 235 нче юлда билгеләнгән


## Кайтара
Домен һәм юл ачкычларын үз эченә алган объект.
