# Фильтр: wu_get_checkout_variables {#filter-wugetcheckoutvariables}

Плагин эшләүчеләргә түләүне рәсмиләштерү бите өчен алдан билгеләнгән көйләүләрне фильтрларга мөмкинлек бирә.

Сак булыгыз, югалган ачкычлар фронт-эндта түләүне рәсмиләштерүне тулысынча бозарга мөмкин.

## Параметрлар {#parameters}

| Исем | Төр | Тасвирлама |
|------|------|-------------|
| $variables | `array` | Локальләштерелгән үзгәрүчәннәр. |
| $checkout | `\Checkout` | Түләүне рәсмиләштерү классы. |

### Версиядән {#since}

- 2.0.0
### Чыганак {#source}

[`inc/checkout/class-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout.php#L1970) эчендә, 1970 нче юлда билгеләнгән

Яңа үзгәрүчәннәр массивын.
