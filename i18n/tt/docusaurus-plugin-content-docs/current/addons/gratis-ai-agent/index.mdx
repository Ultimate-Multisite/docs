# Gratis AI Agent {#gratis-ai-agent}

Gratis AI Agent — табигый телдәге күрсәтмәләр аша тулы WordPress сайтларын планлаштыра, төзи һәм идарә итә алган автоном WordPress AI ярдәмчесе. Ул **сәләтләрнең** структур җыелмасын — агент чакыра ала торган аерым, тикшерелә торган гамәлләрне — тәкъдим итә; алар эчтәлек белән идарә итүдән алып дизайн системасын көйләүгә кадәр барын да колачлый. 1.9.0 версиясе эчтәлек булдыру сәләтләрен, клиент ягындагы скриншотлар аша битне визуаль карауны, биш урнаштырылган агент профилен, шулай ук керүне контрольдә тоту һәм брендлау өчен feature flags өсти.

## Төп мөмкинлекләр {#key-features}

- **Сәләтләр системасы** — агент сезнең WordPress урнаштыруыгызда чакыра ала торган гамәлләрнең модульле, киңәйтелә торган каталогы
- **Эчтәлек белән идарә итү** — язмалар һәм битләр булдыру (бит калыпларын билгеләүне дә кертеп), бер чакыруда берничә язманы пакетлап булдыру, төп рәсемнәрне билгеләү һәм контакт формалары төзү
- **Визуаль карау** — тере битләрнең скриншотларын төшерү, элекке/соңгы халәтләрне чагыштыру һәм макет, типография, төс һәм мөмкинлеклелекне колачлаган AI тарафыннан булдырылган дизайн бәяләмәсен алу
- **Биш урнаштырылган агент** — Content Writer, Site Builder, Design Studio, Plugin Manager һәм Support Assistant; чат башлыгындагы Agent Picker аша алыштырыла
- **Feature flags** — роль чикләүләре, white-label атамасы һәм виджетны көйләү өчен Settings → Feature Flags эчендәге керүне контрольдә тоту һәм брендлау күчергечләре
- **Махсус язма төрләре белән идарә итү** — махсус язма төрләрен теркәү, исемлекләү һәм бетерү, яңадан эшләтеп җибәрүләр арасында сакланып калу белән
- **Махсус таксономияләр белән идарә итү** — махсус таксономияләрне теркәү, исемлекләү һәм бетерү, сакланып калу белән
- **Дизайн системасы сәләтләре** — махсус CSS кертү, сайлап алынган блок үрнәкләрен идарә итү, сайт логотибын билгеләү һәм theme.json пресетларын куллану
- **Глобаль стильләр белән идарә итү** — төс палитралары, типография, аралар һәм башкалар өчен theme.json кыйммәтләрен уку һәм язу
- **Навигация менюлары белән идарә итү** — навигация менюларын һәм аларның элементларын программалы рәвештә булдыру һәм яңарту
- **Көйләнмәләр белән идарә итү** — критик көйләнмәләрне саклау өчен урнаштырылган хәвефсезлек блок-исемлеге белән WordPress көйләнмәләрен уку һәм язу
- **Урнаштырыла торган сәләтләр реестры** — төп plugin файлларын үзгәртмичә өстәмә сәләт пакетларын табу һәм активлаштыру
- **Site Builder Orchestration v2** — plugin табу, алгарышны күзәтү һәм хаталардан торгызу белән күп адымлы сайт төзү планнары
- **WP-CLI benchmark командасы** — CI һәм модель бәяләү өчен агент мөмкинлекләре benchmark җыелмасын команда юлыннан эшләтү

## Таләпләр {#requirements}

- WordPress 6.0 яки югарырак
- PHP 8.1 яки югарырак
- WordPress Multisite (челтәр дәрәҗәсендә активлаштырылган) яки бер сайтлы урнаштыру
- Актив AI провайдер тоташтыргыч plugin (Anthropic, OpenAI яки туры килә торган)

## Урнаштыру {#installation}

1. Plugin файлларын `/wp-content/plugins/gratis-ai-agent` эченә йөкләгез
2. Plugin-ны WordPress эчендәге **Plugins** экраны аша активлаштырыгыз
3. **Gratis AI Agent → Settings** бүлегенә күчегез һәм AI провайдерыгызны тоташтырыгыз
4. (Ихтыяри) Өстәмә сәләт пакетларын урнаштыру өчен **Abilities Registry**-ны карагыз

## Тиз башлау {#quick-start}

Активлаштырылып, AI провайдерга тоташтырылгач, WordPress идарә ян панелендә **Gratis AI Agent** панелен ачыгыз. Табигый телдә күрсәтмә языгыз, һәм агент үзенең булган сәләтләрен кулланып кирәкле адымнарны планлаштырып башкарачак.

**Мисал күрсәтмәләр:**

```
"Portfolio" махсус язма төрен "Project Category" таксономиясе белән булдырыгыз.
```

```
Темага караңгы режим төс палитрасын кертегез һәм төп төсне #1a1a2e итеп билгеләгез.
```

```
Онлайн меню, броньлау формасы һәм контакт бите булган ресторан сайты төзегез.
```

## Сәләтләргә күзәтү {#abilities-overview}

Сәләтләр — агент башкара ала торган атомар гамәлләр. Һәр сәләт — агент эш вакытында укый торган схема (параметрлар һәм кайтару төрләре) тәкъдим итүче теркәлгән PHP классы. Сәләтләр функциональ өлкәләргә төркемләнгән:

| Өлкә | Сәләтләр |
|---|---|
| **Эчтәлек белән идарә итү** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **Визуаль карау** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **Махсус язма төрләре** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **Махсус таксономияләр** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **Дизайн системасы** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **Глобаль стильләр** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **Навигация менюлары** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **Көйләнмәләр белән идарә итү** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **Урнаштырыла торган сәләтләр** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **Site Builder** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

Тулы параметр документлары һәм куллану мисаллары өчен [Сәләтләр белешмәсен](./abilities) карагыз.

## WP-CLI {#wp-cli}

Gratis AI Agent команда юлыннан benchmark-ларны эшләтү һәм сәләтләр белән идарә итү өчен `wp gratis-ai-agent` команда гаиләсе белән килә. Тулы команда исемлеге өчен [WP-CLI белешмәсен](./wp-cli) карагыз.

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## Конфигурация {#configuration}

Барлык көйләнмәләр WordPress идарәсендә **Gratis AI Agent → Settings** астында урнашкан:

| Көйләнмә | Тасвирлама |
|---|---|
| **AI Provider** | Актив AI провайдер тоташтыргычын сайлагыз һәм көйләгез |
| **Model** | Агент сессияләре өчен кулланылачак тел моделен сайлагыз |
| **Max Tokens** | Һәр сорау өчен токен бюджеты (килешү буенча 4096) |
| **Ability Timeout** | Аерым бер сәләт эшли ала торган максималь секунд саны (килешү буенча 30) |
| **Debug Logging** | Сәләт чакыруларын һәм агент фикер йөртүен WordPress debug журналына языгыз |

## Көйләнмәләр белән идарә итү хәвефсезлек блок-исемлеге {#options-management-safety-blocklist}

Көйләнмәләр белән идарә итү мөмкинлеге агентка сизгер WordPress көйләнмәләрен укуны яки язуны тыя торган урнаштырылган блок-исемлекне үз эченә ала. Килешенгән блок-исемлек түбәндәгеләрне колачлый:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (URL көйләнмәләре — болар өчен махсус миграция коралын кулланыгыз)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (яшерен ачкычлар)
- `active_plugins`, `template` (плагин/тема активлаштыру — махсус мөмкинлекләрне кулланыгыз)
- Урнаштырылган плагиннар теркәгән түләү шлюзы таныклыклары һәм API ачкычлары

Өстәмә язмаларны `gratis_ai_agent_options_blocklist` фильтры аша өстәп була.

## Hook-лар һәм фильтрлар {#hooks-and-filters}

### Гамәлләр {#actions}

- `gratis_ai_agent_ability_registered` — мөмкинлек теркәлгәннән соң эшли, мөмкинлек классы исемен ала
- `gratis_ai_agent_plan_started` — сайт төзү планы үтәлә башлаганда эшли
- `gratis_ai_agent_plan_completed` — план уңышлы тәмамланганда эшли
- `gratis_ai_agent_plan_failed` — план төзәтеп булмый торган хатага очраганда эшли

### Фильтрлар {#filters}

- `gratis_ai_agent_abilities` — агент аны йөкләгәнче теркәлгән мөмкинлекләр исемлеген үзгәртү
- `gratis_ai_agent_options_blocklist` — көйләнмәләр иминлеге блок-исемлеген киңәйтү яки кыскарту
- `gratis_ai_agent_benchmark_questions` — бәяләү җыелмасына махсус benchmark сорауларын өстәү
- `gratis_ai_agent_plan_plugins` — сайт планы төзелгәндә плагиннарны табу исемлеген үзгәртү
