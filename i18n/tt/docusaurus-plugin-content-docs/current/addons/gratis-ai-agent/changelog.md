# Үзгәрешләр журналы

## 1.9.0 — 2026-04-28 чыгарылды {#190--released-on-2026-04-28}

### Яңа {#new}

- **`create_contact_form` мөмкинлеге** — актив форма plugin (Contact Form 7, WPForms, Fluent Forms яки Gravity Forms) ярдәмендә элемтә формасы төзи һәм теләсә кайсы язмага яки биткә урнаштырырга әзер shortcode кайтара.
- **`set_featured_image` мөмкинлеге** — булган Media Library беркетмә ID-сыннан яки ерак URL-дан язмага яки биткә төп рәсем билгели; URL бирелгәндә рәсемне автоматик рәвештә импортлый.
- **`batch_create_posts` мөмкинлеге** — бер мөмкинлек чакыруында берничә язма төзи. `create_post` белән бер үк параметрларны хуплый, һәр язма буенча уңыш/уңышсызлык турында хәбәр итә һәм өстәмә `stop_on_error` режимына ия.
- **`page_template` параметры** — `create_post` һәм `update_post` өчен өстәлде. Төзү яки яңарту вакытында PHP бит калыбы файлын (мәсәлән, `page-full-width.php`) билгели. Theme көйләнешенә кире кайту өчен `update_post`-ка буш юл тапшырыгыз.
- **Клиент ягындагы скриншот мөмкинлекләре** — `capture_screenshot`, `compare_screenshots` һәм `review_page_design`. Сервер ягындагы headless браузер аша тере битләрнең тулы яки viewport скриншотларын төшерегез, ике скриншотны чагыштырыгыз һәм төзелеш, типография, төс һәм мөмкинлеклелекне колачлаган AI тудырган дизайн күзәтүе алыгыз.
- **Биш урнаштырылган агент** — Контент язучы, Сайт төзүче, Дизайн студиясе, Plugin менеджеры һәм Ярдәм ассистенты. Һәр агентның үзенә багышланган кораллар җыелмасы, җайлаштырылган система промпты һәм башлангыч тәкъдимнәре бар. Чат башлыгындагы яңа **Агент сайлагыч** аша алыштырып була. Карагыз: [Урнаштырылган агентлар](../../user-guide/configuration/built-in-agents).
- **Функция флаглары** — керүне контрольләү күчергечләре (администраторлар белән чикләү, челтәр администраторлары белән чикләү, язылучылар керүе, әгъза булмаганнар өчен сүндерү) һәм брендлау вариантлары (footer атрибуциясен яшерү, махсус агент исеме, агент сайлагычны яшерү, сайт иконасын чат аватары итеп куллану) белән яңа **Көйләүләр → Функция флаглары** вкладкасы. Карагыз: [Gratis AI Agent көйләүләре](../../user-guide/administration/gratis-ai-agent-settings).
- **Соңгы сессияне торгызу** — чат панели хәзер бит йөкләнгәндә һәм виджет ачылганда иң соңгы сөйләшүне автоматик рәвештә яңадан йөкли, шуңа күрә битләр арасында күчкәндә контекст югалмый.
- **Plugin гамәл сылтамалары** — Көйләүләргә һәм Мөмкинлекләр реестрына тиз сылтамалар хәзер WordPress **Плагиннар → Урнаштырылган плагиннар** экранында plugin тасвирламасы астында күренә.

### Яхшыртылды {#improved}

- **Рәсем чыганагын кабат сынау** — агент хәзер йөкләү уңышсыз булганда AI тудырган рәсемгә күчкәнче көйләнгән барлык бушлай рәсем чыганакларын кабат сынап карый.
- **Модель мәгълүматы панели** — чат башлыгында һәрвакыт күренә; беренче хәбәрдән соң инде яшерелми.
- **Авто-әйләндерү тәртибе** — кулланучы укыр өчен өскә әйләндергәндә авто-әйләндерү туктап тора; йөзүче **Аска әйләндерү** төймәсе пәйда була һәм кулланучы соңгы хәбәргә җиткәч автоматик рәвештә юкка чыга.
- **Агент сайлагыч UI** — һәр агент өчен иконалар белән form-table өстәмә катламы итеп яңадан эшләнде, бу агентларны бер карашта тануны һәм алыштыруны җиңеләйтә.
- **Lazy-loaded JS өлешләре** — чат виджетының башлангыч JavaScript җыелмасы хәзер lazy-loaded өлешләргә бүленде, башлангыч җыелма күләмен 75–90% ка киметә.
- **Чат виджетын яңадан бизәү** — элекке махсус аватар урынына бердәм AI иконасы кулланыла; урнаштырылган агент системасы белән эзлекле.
- **URL сылтамалаштыру** — система хәбәрләрендә һәм хата хәбәре куыкларында күренгән URL-лар хәзер басыла торган сылтамалар итеп күрсәтелә.

### Төзәтелде {#fixed}

- **Мөмкинлекләрне табып булу** — тасвирламалар, система промпты сылтамалары һәм namespace туры килүе төзәтелде, шуңа күрә барлык мөмкинлекләр агентның кораллар исемлегендә ышанычлы күренә.
- **Провайдерлар кэшы** — провайдерлар хәзер версия санагычы аша бөтен сайт буенча кэшка салына, multisite челтәрләрендә искергән провайдер проблемаларын булдырмый.
- **`ability_invalid_output`** — 12 мөмкинлек обработчигында хәл ителде; барысы да дөрес структураланган JSON җаваплары кайтара.
- **`pending_client_tool_calls` конвейеры** — клиент ягындагы корал чакырулары дөрес тәмамлансын һәм нәтиҗәләрне модельгә кайтарсын өчен баштан ахыргача тоташтырылды.
- **Тарих тартмасы** — кире кайтарып булмый торган үзгәрешләр кире кайтару исемлегеннән чыгарылды; **Тулы тарихны карау** сылтамасы хәзер дөрес эшли.
- **Үзгәрешләр/кире кайтару системасы** — биш аерым хата төзәтелде һәм яңа admin интерфейсы астында берләштерелде.
- **Көйләүләрне саклау toast хәбәре** — snackbar белдерүе хәзер **Көйләүләрне саклау**-ка басканнан соң ышанычлы күренә.
- **Чүп савыты контекст менюсы** — әйберләрне чүп савыты күренешеннән чыкмыйча тулысынча бетереп булсын өчен **Мәңгелеккә бетерү** варианты өстәлде.
- **Төзәтү һәм кабат җибәрү** — төзәтү иконасына басу хәзер бөтен җептәге барлык хәбәрләр өчен түгел, ә бары тик басылган хәбәр өчен генә төзәтү режимына кертә.
- **Чат төзелеше биеклеге** — бит өстендә plugin кертелгән эчтәлек (admin notices, баннерлар) пәйда булганда чат панели үз биеклеген җайлаштыра, кертү өлкәсен экраннан читкә этәрүне булдырмый.

## 1.4.0 — 2026-04-09 чыгарылды {#140--released-on-2026-04-09}

### Яңа {#new-1}

- **WP-CLI benchmark command** (`wp gratis-ai-agent benchmark`) — CI торба юллары һәм модель бәяләү эш агымнары өчен command line аша Agent Capabilities v1 benchmark suite эшләтә. Аерым-сорау буенча эшләтүләрне, тәэмин итүче/модель алыштыруларын һәм JSON/CSV чыгышын хуплый.
- **Agent Capabilities v1 benchmark suite** — балл кую һәм токен/дәвамлылык хисаплары белән, тулы сәләт өслеген сынаучы катлаулы, күп-адымлы promptларның структуралаштырылган җыелмасы.
- **Custom Post Type сәләтләре** — `register_post_type`, `list_post_types`, `delete_post_type`. Теркәлүләр options таблицасы аша яңадан башлатулар арасында саклана.
- **Custom Taxonomy сәләтләре** — `register_taxonomy`, `list_taxonomies`, `delete_taxonomy`. Мәҗбүри булмаган rewrite slugлар белән иерархик һәм яссы taxonomyларны хуплый.
- **Design System сәләтләре** — `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset`. Алдан сайлап төзелгән биш presetны үз эченә ала: `minimal-dark`, `warm-editorial`, `corporate-blue`, `vibrant-startup`, `classic-blog`.
- **Global Styles сәләтләре** — `get_global_styles`, `set_global_styles`, `reset_global_styles`. WordPress Global Styles API аша theme.json кыйммәтләрен укый һәм яза.
- **Navigation Menu идарә итү сәләте** — `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus`. Эчке менюларны һәм theme урнашуына билгеләүне хуплый.
- **Options Management сәләте** — `get_option`, `set_option`, `delete_option`, `list_options`. Мөһим WordPress көйләүләрен үзгәртүдән саклаучы эчке иминлек блок-исемлеген үз эченә ала.
- **Installable Abilities Registry** — `list_available_abilities`, `install_ability`, `recommend_plugin`. WordPress pluginнар рәвешендә таратылган сәләт җыелмаларын табыгыз һәм активлаштырыгыз.
- **Site Builder Orchestration v2** — `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`. Plugin табу, адым чыгышы сылтамалары, прогресс күзәтүе һәм автоном хата торгызуы белән күп-фазалы сайт төзү планнары.
- **Ресторан сайты benchmark соравы** (`q-restaurant-website`) — CPT теркәүне, дизайн системасын, навигацияне һәм plugin табуны колачлаучы башыннан ахырына кадәр тест.
- **AI тәэмин итүче тоташтыргыч pluginнар** тизрәк җирле үсеш көйләү өчен WordPress Playground blueprintларына өстәлде.

### Яхшыртылды {#improved-1}

- README AI тәэмин итүче тоташтыргыч документациясе һәм көйләү күрсәтмәләре белән яңартылды.

### Төзәтелде {#fixed-1}

- `main` тармагында 25 PHPUnit тест уңышсызлыгы хәл ителде.
- `blueprint.json` эчендә GitHub releases URL форматы төзәтелде.
- Иске IDлар белән бәрелешләрдән саклану өчен бирем IDларын яңадан номерлау.
