# PayFast

Multisite Ultimate: Payfast Gateway

Көньяк Африкадагы тиз һәм имин онлайн түләүләр шлюзы булган Payfast-ны Multisite Ultimate түләү варианты итеп өсти.

## Урнаштыру {#installation}

1. 'ultimate-multisite-payfast' ны '/wp-content/plugins/' каталогына йөкләгез
2. Плагинны WordPress эчендәге 'Plugins' менюсы аша активлаштырыгыз
3. Плагинны көйләү өчен адымлап Wizard күрсәтмәләрен үтәгез

## Куллану кулланмасы {#usage-guide}

PayFast — Көньяк Африкадагы бизнеслар өчен әйдәп баручы онлайн түләү эшкәртүчесе. Ул транзакцияләрне җайлы һәм имин итеп, онлайн сәүдәгәрләргә үсәргә ярдәм итүгә багышланган. Ultimate Multisite кулланучысы соравы буенча, без PayFast белән интеграцияне мөмкин итәчәк өстәмә ясарга булдык. Бу мәкалә Ultimate Multisite челтәрегездә түләүләр кабул итү өчен PayFast-ны түләү шлюзы итеп ничек көйләүне күрсәтәчәк.

Беренче адым — өстәмәне урнаштыру.

Өстәмәләр битенә керегез һәм Ultimate Multisite: Payfast Gateway өстәмәсен сайлагыз.

Өстәмә тәрәзәсе ачылып чыгачак. Install Now төймәсенә басыгыз.

Урнаштыру тәмамлангач, сез плагиннар битенә юнәлтеләчәксез. Монда бары тик Network Activate төймәсенә басыгыз, һәм _Payfast_ өстәмәсе челтәрегездә активлаштырылачак.

<!-- Скриншот юк: PayFast өстәмәсе өчен Network Activate төймәсен күрсәткән WP Plugins бите -->

# PayFast көйләү

Өстәмәне урнаштырганнан соң, сез хәзер Ultimate Multisite көйләүләренә кереп, PayFast-ны көйли башлый аласыз.

**Ultimate Multisite көйләүләре** битендә "Түләүләр" вкладкасын таба аласыз. Бу вкладка астында сез бу түләү шлюзын кушу яки сүндерү өчен күчергеч төймә белән бергә "**Payfast** " күрергә тиеш.

![Түләү шлюзы көйләүләре](/img/config/settings-payment-gateways.png)Алга таба күчергеч төймәгә басып, аны кушыгыз. Аннары ул тартманы киңәйтеп, тутырырга кирәк булган төрле кырларны күрсәтәчәк, мәсәлән **Mechant ID, Merchant Key һәм PassPhrase.** Бу мәгълүматны **Payfast** Account-ыгыздан ала аласыз

<!-- Скриншот юк: Merchant ID, Merchant Key һәм PassPhrase кырларын күрсәткән PayFast киңәйтелгән көйләүләре -->

Киләсе адым — кирәкле мәгълүматны алу өчен PayFast Account-ыгызга керү.

Кергәннән соң, **Payfast** Account Dashboard-ыгызда кирәкле керү мәгълүматларын шунда ук күрә аласыз.

<!-- Скриншот юк: Merchant ID һәм Merchant Key керү мәгълүматларын күрсәткән PayFast Account Dashboard -->Керү мәгълүматларын гади генә күчереп куя аласыз һәм Ultimate Multisite көйләүләрендәге детальләрне тутыра аласыз. Аннары үзгәрешләрне саклау өчен **"Көйләүләрне саклау** " төймәсенә басыгыз.

<!-- Скриншот юк: керү мәгълүматлары тутырылган һәм Save төймәсе булган Ultimate Multisite PayFast көйләүләре -->

Менә шул, PayFast түләү интеграциягез тәмам!
