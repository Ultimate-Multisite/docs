# Mailchimp

Multisite Ultimate: Mailchimp интеграциясе

Multisite Ultimate челтәрегезне Mailchimp белән интеграцияләгез. Сез билгеле планнарны Mailchimp исемлекләренә бәйли һәм теркәлү вакытында язылучыларны шул исемлекләргә автоматик рәвештә өсти аласыз.

## Урнаштыру

1. 'ultimate-multisite-mailchimp' ны '/wp-content/plugins/' директориясенә йөкләгез
2. Плагинны WordPress Network Admin эчендәге 'Плагиннар' менюсы аша активлаштырыгыз

## Куллану кулланмасы

Язылучыларның email адресларын MailChimp исемлекләрегез белән синхронлау безнең Ultimate Multisite: MailChimp Integration өстәмәсе белән бик җиңел. Бу дәреслек сезгә барысын да көйләп чыгарга ярдәм итү өчен әзерләнгән!

## API ачкычы алу

Мөмкин булган төрле көйләүләргә күчкәнче, безгә MailChimp API ачкычы кирәк. Бу Ultimate Multisite ка MailChimp API белән элемтәгә керергә, исемлекләрне һәм төркемнәрне алырга, шулай ук кулланучылар теркәлгәндә email адресларын шул исемлекләргә һәм төркемнәргә өстәргә мөмкинлек бирәчәк.

Үз MailChimp API ачкычын алу өчен бу дәреслекне үтәгез: [API ачкычлары турында | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

_MailChimp API ачкычын монда өстәгез:_ _**Ultimate Multisite көйләүләре - > Өстәмә көйләүләре -> MailChimp**_

## Интеграция режимы

MailChimp интеграция өстәмәсе ике “интеграция режимы” тәкъдим итә, ягъни email адресларын синхронлау өчен өстәмәне куллануның ике төрле ысулы бар.

<!-- Скриншот юк: MailChimp интеграция режимын сайлау, Берничә исемлек һәм Бер исемлек вариантлары белән -->

_Зуррак сыгылмалылык өчен ике интеграция режимы!_

### Режим 1: Берничә исемлек

Килешенгән режим **Берничә исемлек** куллана. Бу режим һәр план өчен берничә MailChimp исемлеген сайларга мөмкинлек бирә. Кулланучыларыгызны төрле исемлекләргә бүләсез икән, бу иң яхшы вариант.

Мәсәлән, MailChimp аккаунтыгызда 3 төрле исемлек бар дип фаразлыйк: **Яңалыклар** , **План A** , һәм **План B**. Сез барлык кулланучыларның, планнарына карамастан, Яңалыклар исемлегенә дә кертелүен телисез, шуңа күрә алар яңартуларыгызны калдырмаячак.

Моңа ирешү өчен, Берничә исемлек режимын кулланып, **Яңалыклар** һәм **План A** исемлекләрен План A MailChimp көйләүләренә, ә **Яңалыклар** һәм **План B** исемлекләрен План B MailChimp көйләүләренә өсти аласыз.

<!-- Скриншот юк: яңа аккаунтлар һәм бетерелгән аккаунтлар өчен MailChimp килешенгән исемлекләрен сайлау -->

_Яңа аккаунтлар һәм бетерелгән аккаунтлар өчен килешенгән исемлекләрне сайлагыз_

### Режим 2: Бер исемлек + Берничә төркем

MailChimp исемлегегезне **Төркемнәр** дип аталган кечерәк кулланучы төркемнәренә бүлергә мөмкинлек бирә. Бу сезгә күбрәк контроль бирә, чөнки сез бер үк кампаниянең MailChimp та берничә күчермәсен ясамыйча, язылучыларыгызның билгеле бер өлешенә кампания җибәрә аласыз (бу соңыннан шул кампания метрикаларын анализлауны кыенлаштырыр иде).

Мәсәлән, сездә **Минем челтәр язылучылары,** дип аталган бер исемлек була ала, һәм аның эчендә **План A** һәм **План B** өчен төркемнәр була ала.

<!-- Скриншот юк: килешенгән исемлекләр өчен MailChimp бер исемлек һәм төркемнәр сайлау -->

_“Бер исемлек + берничә төркем” интеграция режимын куллансагыз, килешенгән исемлекләрне һәм төркемнәрне сайлагыз_

## Килешенгән исемлекләр һәм төркемнәр

Сезнең конкрет куллану очрагыгыз өчен кайсы интеграция режимы мәгънәлерәк икәнен сайлаганнан соң, килешенгән исемлекләрне (һәм режимга карап төркемнәрне) сайлау вакыты җитә. Килешенгән исемлек билгеле бер план өчен махсус исемлек сайланмаган очракта кулланылачак.

Сез шулай ук бетерелгән аккаунтлар өчен килешенгән исемлекне сайлый аласыз. Кулланучы үз язылуын туктатып, аккаунтын челтәрегездән бетергән саен, аның email адресы бетерелгән аккаунтлар өчен билгеләнгән исемлеккә күчереләчәк.

## Икеле язылуны кабызу

Шул ук көйләүләр битендә сез **Икеле язылу** ны кабыза алачаксыз. Бу вариант кулланучыны сайланган килешенгән исемлеккә ДӘ, шул кулланучының планы өчен сайланган исемлеккә дә өстәячәк. Әгәр бу вариант кабызылмаган булса, кулланучы бары тик кулланучының планы өчен сайланган исемлеккә генә өстәләчәк. Әгәр шул конкрет план өчен берни дә билгеләнмәгән булса, кулланучы сайланган килешенгән исемлеккә өстәләчәк.

<!-- Скриншот юк: MailChimp икеле язылу күчергече көйләве -->

_Теркәлү вакытында кулланучыларыгыз килешенгән исемлеккә дә, план исемлегенә дә язылырга мөмкин_

## Һәр план өчен исемлекләр һәм төркемнәр сайлау

**Планнар** га керегез һәм үзгәртергә теләгән планны сайлагыз. Игътибар итегез: **Киңәйтелгән параметрлар** мета-тартмасында MailChimp дип аталган яңа кыстыргыч булачак.

Шул кыстыргычта сез бу план өчен берничә исемлекне яки бер исемлек һәм берничә төркемне (интеграция режимына карап) сайлый алачаксыз.

<!-- Скриншот юк: планны үзгәртү битендә исемлекләр һәм төркемнәр сайлавын күрсәткән MailChimp кыстыргычы -->

_Һәр планыгыз өчен кулланырга теләгән исемлекләрне һәм төркемнәрне сайлагыз_

Яңа кулланучы шул планга теркәлгәндә, аның email адресы сайланган исемлекләргә һәм төркемнәргә өстәләчәк!
