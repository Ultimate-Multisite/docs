# Ярдәм агентлары

Бөтен челтәрне үзегез генә идарә итү җиңел эш түгел. Челтәрегез үсә башлагач, ярдәмгә ихтыяҗ арта баруын күрә башларсыз. Эш йөгегезне җиңеләйтергә ярдәм итәчәк кешеләрне җәлеп итү яхшы фикер булачак. Әмма тулы керү мөмкинлеге/рөхсәт бирү хәвефле һәм челтәрегезне куркыныч астына куярга мөмкин. Нәкъ менә шунда бу Ярдәм агенты өстәмәсе кирәк була.

Бу өстәмә сезгә Ярдәм агенты дип аталган яңа төр WordPress кулланучысын булдырырга мөмкинлек бирә һәм ул кулланучының нинди рөхсәтләргә ия булуын тулысынча контрольдә тотарга бирә. Рөхсәт бирү күчергеч төймәсен кабызу кебек үк җиңел, ә аны бетерергә карар итсәгез, сүндерү дә шулай ук җиңел.

## Үзенчәлекләр {#features}

- WordPress һәм Ultimate Multisite рөхсәтләренең пунктлап бирелгән исемлеге
- Керү мөмкинлеген җиңел генә кабызу һәм сүндерү
- Чикләнмәгән санда Агент Accountларын булдыру

## Урнаштыру {#installation}

1. 'ultimate-multisite-support-agents' ны '/wp-content/plugins/' директориясенә йөкләгез
2. Плагинны WordPress эчендәге 'Plugins' менюсы аша активлаштырыгыз
3. Плагинны көйләү өчен адымлап Оста күрсәтмәләрен үтәгез
