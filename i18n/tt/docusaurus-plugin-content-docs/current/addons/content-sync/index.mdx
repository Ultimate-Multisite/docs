# Content Sync

Multisite Ultimate: Content Sync

Multisite Ultimate өчен бу өстәмә челтәр администраторларына эчтәлекне теләсә кайсы сайттан челтәрдәге башка сайтларга синхронлаштырырга мөмкинлек бирә. Ул язмалар, битләр, махсус язма төрләре һәм Elementor шаблоннары кебек барлык язма төрләрен хуплый. Өстәмә эчтәлекне һәм максат сайтларны сайлау өчен күп адымлы мастер интерфейсын, пакетлы эшкәртүне һәм реаль вакытта барыш турында кире бәйләнешне үз эченә ала.

## Мөмкинлекләр {#features}

* Эчтәлек сайлау: миниатюралар һәм язма төре буенча фильтрлау белән визуаль карточкалар челтәре
* Барлык язма төрләре: язмаларны, битләрне, товарларны, Elementor шаблоннарын һәм теләсә нинди махсус язма төрен синхронлаштыра
* Сайт сайлау: эчтәлекне кайсы сайтларга синхронлаштырырга икәнен сайлагыз, чыганак сайтны чыгарып калдырып һәм бала сайтларны алдан сайлап
* Пакетлы эшкәртү: барыш яңартулары белән берьюлы 10 сайтка синхронлаштыра
* Медиа синхронлаштыру: медиа файлларны (рәсемнәр, видеолар) язма эчтәлегеннән максат сайтларга автоматик рәвештә күчерә һәм яңадан бәйли
* Таксономия синхронлаштыру: категорияләрне, тегларны һәм махсус таксономияләрне саклый
* Elementor ярдәме: CSS яңадан генерацияләү һәм шаблон ID-ларын яңадан бәйләү белән Elementor шаблоннары өчен тулы ярдәм
* Конфликтларны хәл итү: булган эчтәлекне алыштыру яки аны үткәреп җибәрү мөмкинлеге
* Кэш идарәсе: синхронлаштырганнан соң максат сайтларда Elementor кэшын автоматик рәвештә чистарта

## Таләпләр {#requirements}

* WordPress 5.3 яки югарырак
* PHP 7.4 яки югарырак
* Multisite Ultimate плагины (актив)
* Elementor плагины (ихтыяри, Elementor шаблоннарын синхронлаштыру өчен кирәк)

## Урнаштыру {#installation}

1. 'ultimate-multisite-content-sync' ны '/wp-content/plugins/' директориясенә йөкләгез
2. WordPress Network Admin эчендәге 'Plugins' менюсы аша плагинны активлаштырыгыз
3. Ultimate Multisite → Settings → Content Sync эчендә көйләүләрне конфигурацияләгез
4. Синхронлаштыруны башлау өчен Ultimate Multisite → Content Sync бүлегенә күчегез
