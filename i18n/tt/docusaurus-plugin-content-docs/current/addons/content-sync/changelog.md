# Content Sync үзгәрешләр журналы

### Version 1.1.0 - 2025-12-31 көнне чыгарылды
* Яңа: барлык язма төрләрен синхронлаштыру өчен ярдәм (язмалар, битләр, махсус язма төрләре)
* Яңа: эчтәлек сайлау адымында язма төре буенча фильтрлау
* Яңа: чыганак сайтны сайлау адымы - төп сайттан гына түгел, теләсә кайсы сайттан синхронлаштыру
* Яңа: калып сайтлар өчен бала сайтны алдан сайлау
* Яңа: барлык язма төрләре өчен post_content-тан медиа чыгару
* Яңа: таксономияләрне синхронлаштыру (категорияләр, теглар, махсус таксономияләр)
* Үзгәртелде: "Elementor Template Sync" атамасы "Content Sync" итеп үзгәртелде
* Үзгәртелде: "Select Templates" адымы "Select Data" итеп үзгәртелде
* Яхшыртылды: Elementor-га хас код хәзер elementor_library язма төре өчен генә эшли

### Version 1.0.0 - 2025-12-19 көнне чыгарылды
* Башлангыч чыгарылыш
* Калып һәм сайт сайлау өчен күп адымлы мастер
* Реаль вакыттагы барыш турында кире элемтә белән пакетлап эшкәртү
* Автоматик ID яңадан бәйләү белән тулы медиа синхронлаштыру
* Ultimate Multisite белән көйләүләр интеграциясе
* Челтәр көйләүләре аша сайтны төшереп калдыру ярдәме
* Синхронлаштырудан соң Elementor кэшын чистарту
* Калып таксономиясен саклау
