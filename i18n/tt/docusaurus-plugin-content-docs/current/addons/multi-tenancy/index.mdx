# Күп арендалылык {#multi-tenancy}

Җитештерүчәнлек һәм иминлек өчен аскы сайтларны серверлар һәм мәгълүмат базалары арасында таратыгыз

## Version 1.2.0 чыгарылышының төп яңалыклары {#version-120-release-highlights}

Multi-Tenancy 1.2.0 җитештерү изоляциясе эш агымнары өчен суверен арендатор нигезен өсти:

- Кыска гомерле, кабат кулланудан сакланган SSO керүләре белән халәтсез арендатор autologin.
- Суверен арендаторлар өчен һәр аскы сайтка аерым мәгълүмат базасы һәм файл системасы изоляциясе.
- Хостны исәпкә алып арендатор әзерләү өчен Hostinger күп арендалылык мөмкинлеге.
- Иске мәгълүматларны тикшерү һәм асинхрон этеп җибәрү эшкәртүе өчен суверен күчерүне тикшерү капкалары.
- Bedrock, FrankenPHP һәм localhost урнаштырулары өчен көйләнә торган бер үк машинадагы мәгълүмат базасы хостын бәйләү.
- Арендаторны башлангыч әзерләү, мәгълүмат базасы язучысы, схема башлангыч әзерләве һәм кулланучыларны әзерләү яхшыртулары.
- Арендатор credential-ларын чистарта торган иминрәк суверен бетерү.

[суверен күчерү эш агымы](/user-guide/getting-started/sovereign-migration-workflow), [күп арендалылык изоляциясе](/user-guide/configuration/multi-tenancy-isolation) һәм [суверен арендатор операцияләре](/user-guide/administration/sovereign-tenant-operations) өчен кулланучы кулланмаларын карагыз.

## Таләпләр {#requirements}

- WordPress 5.3 яки югарырак
- PHP 7.4 яки югарырак
- Multisite Ultimate плагин (актив)

## Урнаштыру {#installation}

1. Өстәмә файлларын сезнең `/wp-content/plugins/` директориясенә йөкләгез
2. WordPress эчендәге 'Plugins' менюсы аша плагинны активлаштырыгыз
3. Өстәмә көйләнмәләрен челтәр администраторында көйләгез

## Үстерү {#development}

### Көйләү {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Тестлау {#testing}

Өстәмә тулы тестлау фреймворкын үз эченә ала:

- **Модуль тестлары**: WordPress тест җыелмасы белән PHPUnit нигезендәге тестлар
- **Төп тест классы**: Гомуми тест сценарийлары өчен ярдәмче методлар бирә
- **Тест капланышы**: Код капланышын күзәтү өчен көйләнгән
- **Multisite тестлавы**: Тестлар multisite мохитендә эшли

### Код стандартлары {#code-standards}

- **WordPress кодлау стандартлары**: PHPCS аша мәҗбүри үтәлә
- **PHP 7.4+ ярашлылыгы**: Заманча PHP мөмкинлекләре хуплана
- **Статик анализ**: Төрләрне тикшерү өчен PHPStan интеграциясе
- **Кодны заманалаштыру**: PHP яңартулары өчен Rector кагыйдәләре

### Җыю системасы {#build-system}

- **Ресурсларны эшкәртү**: CSS/JS файллары өчен кечерәйтү
- **Тәрҗемә**: POT файлын булдыру
- **Пакетлау**: Тарату өчен архив булдыру
- **Үстерү режимы**: Төзәтү өчен кечерәйтелмәгән ресурслар

## Конфигурация {#configuration}

Өстәмә челтәр администраторы интерфейсы аша кулланыла торган төрле конфигурация вариантларын хуплый.

### Суверен арендатор конфигурациясе {#sovereign-tenant-configuration}

Суверен арендаторлар өчен, трафикны арендаторга юнәлткәнче, арендатор мәгълүмат базасын, файл системасы тамырын, мәгълүмат базасы хостын бәйләүне һәм хост мөмкинлеген көйләгез. Мәгълүмат базасы credential-ларына, хост бәйләүгә, маршрутлауга яки кулланучыларны әзерләүгә йогынты ясаган һәр үзгәрештән соң күчерүне тикшерүне эшләтегез.

`localhost` ны арендатор башкару мохите һәм мәгълүмат базасы рөхсәтләре локаль socket яки бер үк машинадан керүне көткәндә генә кулланыгыз. Сезнең тәэмин итүчегез тарафыннан көйләнгән рөхсәт хосты шул булганда `127.0.0.1` яки сервис хост исемен кулланыгыз.

### Hostinger мөмкинлеге {#hostinger-capability}

Hostinger күп арендалылык мөмкинлеге уртак Hostinger интеграция token-ы белән эшли. Тикшерү алдыннан доменның hPanel эчендә булуын, DNS-ның Hostinger account-ына күрсәтүен һәм арендатор мәгълүмат базасы көйләнмәләренең hPanel белән туры килүен раслагыз.

## Hook-лар һәм фильтрлар {#hooks-and-filters}

### Гамәлләр {#actions}

- `ultimate-multisite-addon-template_init` - Өстәмә инициализациясеннән соң эшләтелә
- `ultimate-multisite-addon-template_loaded` - Барлык өстәмә файллары йөкләнгәннән соң эшләтелә

### Фильтрлар {#filters}

- `ultimate-multisite-addon-template_settings` - Өстәмә көйләнмәләрен үзгәртү
- `ultimate-multisite-addon-template_enabled` - Өстәмәне кабызу/сүндерү халәтен үзгәртеп билгеләү
