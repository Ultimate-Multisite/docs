# Меню ясау

**Меню ясау** мөмкинлеге WordPress навигация менюларын бит исемнәреннән аерым навигация ярлыкларын хуплап ясый.

## Гомуми күзәтү {#overview}

Бу мөмкинлек стандарт меню ясау функциональлеген `navigation_label` параметрын күрсәтү мөмкинлеге белән киңәйтә. Бу сезгә навигациядә күрсәтелә торган ярлык бит исеменнән аерыла торган менюлар ясарга мөмкинлек бирә, сайт төзелешендә һәм кулланучы тәҗрибәсендә күбрәк сыгылмалылык бирә.

## Параметрлар {#parameters}

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | string | Әйе | Меню исеме, мәсәлән, `Primary Navigation` |
| `location` | string | Юк | Бу менюны билгеләү өчен тема урыны, мәсәлән, `primary` |
| `navigation_label` | string | Юк | Навигациядә күрсәтелә торган ярлык (бит исеменнән аерым) |

## Кайтарылган кыйммәт {#return-value}

```json
{
  "success": true,
  "menu_id": 7,
  "name": "Primary Navigation",
  "navigation_label": "Main Menu"
}
```

## Навигация ярлыгы һәм бит исеме {#navigation-label-vs-page-title}

`navigation_label` параметры эчке меню исемен кулланучыларга күрсәтелә торган ярлыктан аерырга мөмкинлек бирә:

- **`name`** — WordPress куллана торган эчке меню идентификаторы (мәсәлән, "Төп навигация")
- **`navigation_label`** — Навигациядә сайтка керүчеләргә күрсәтелә торган ярлык (мәсәлән, "Төп меню")

Бу түбәндәге очракларда файдалы:
- Эчке исем бирү кагыйдәгез кулланучыларга күрсәтелә торган ярлыклардан аерыла
- Навигациядә идарә панелендәге белән чагыштырганда кыскарак ярлыклар телисез
- Төрле ярлык озынлыклары белән берничә телне хупларга кирәк
- Аерым төбәкләр яки кулланучы төркемнәре өчен менюлар төзисез

## Куллану мисаллары {#usage-examples}

### 1 нче мисал: навигация ярлыгы булган гади меню {#example-1-simple-menu-with-navigation-label}

**Сорау:**
```
"Меню" навигация ярлыгы белән "Төп навигация" дип аталган төп навигация менюсы ясагыз.
```

**Нәтиҗә:**
```json
{
  "success": true,
  "menu_id": 7,
  "name": "Main Navigation",
  "navigation_label": "Menu"
}
```

### 2 нче мисал: аерым тема урыны өчен меню {#example-2-menu-for-specific-theme-location}

**Сорау:**
```
"Тиз сылтамалар" навигация ярлыгы белән "Футер сылтамалары" дип аталган футер менюсы ясагыз һәм аны футер урынына билгеләгез.
```

**Нәтиҗә:**
```json
{
  "success": true,
  "menu_id": 8,
  "name": "Footer Links",
  "navigation_label": "Quick Links",
  "location": "footer"
}
```

## Theme Builder белән интеграция {#integration-with-theme-builder}

Theme Builder кулланганда, Меню ясау мөмкинлеге:

1. Мөмкин булган тема меню урыннарын автоматик рәвештә ачыклый
2. Дизайныгыз өчен тиешле навигация ярлыклары белән менюлар ясый
3. Менюларны дөрес тема урыннарына билгели
4. Ясаганнан соң меню элементларын өстәүне хуплый

## Бәйле мөмкинлекләр {#related-abilities}

- **`add_menu_item`** — Булган менюга элементлар өстәү
- **`update_menu`** — Менюның исемен үзгәртү яки аны тема урынына яңадан билгеләү
- **`delete_menu`** — Сайтыгыздан менюны бетерү

## Иң яхшы гамәлләр {#best-practices}

- **Ачык навигация ярлыкларын кулланыгыз** — ярлыкларны кулланучылар өчен кыска һәм тасвирлаулы итеп саклагыз
- **Тема урыннарына туры китерегез** — дөрес күрсәтү өчен менюларны тиешле тема урынына билгеләгез
- **Меню төзелешен планлаштырыгыз** — элементлар ясамас борын меню иерархиясен билгеләгез
- **Җайланмаларга яраклашуны сынагыз** — менюларның мобиль җайланмаларда дөрес күрсәтелүен тикшерегез
- **Ярлыкларны локальләштерегез** — төрле тел версияләре өчен төрле навигация ярлыкларын кулланыгыз

## Чикләүләр {#limitations}

- Навигация ярлыклары бары тик күрсәтү өчен генә; эчке `name` һаман да WordPress идентификациясе өчен кулланыла
- Тема хуплавы төрлечә; барлык темалар да барлык меню урыннарын хупламый
- Меню элементлары меню ясалганнан соң аерым өстәлергә тиеш
- Навигация ярлыгын үзгәртү менюны яңартуны таләп итә
