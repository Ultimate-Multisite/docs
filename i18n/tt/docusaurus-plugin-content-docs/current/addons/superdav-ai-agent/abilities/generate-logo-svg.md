# SVG логотипын генерацияләү

**Generate Logo SVG** мөмкинлеге Theme Builder’га махсус логотип SVG’ларын турыдан-туры WordPress сайтыгызда автоматик namespace-safe sanitisation белән ясарга һәм урнаштырырга мөмкинлек бирә.

## Күзәтү

Бу мөмкинлек сайтыгызның бренд юнәлеше һәм дизайн өстенлекләре нигезендә масштаблана торган вектор графикасы (SVG) логотипларын ясый. Ясалган SVG’лар WordPress’та куллану өчен куркынычсыз булсын өчен автоматик рәвештә чистартыла, шул ук вакытта визуаль бөтенлек саклана.

## Параметрлар

| Параметр | Төр | Кирәк | Тасвирлама |
|-----------|------|----------|-------------|
| `site_name` | string | Әйе | Логотип ясалачак сайтның исеме |
| `style` | string | Әйе | Дизайн стиле (мәс., "modern", "classic", "minimalist", "playful") |
| `colors` | array | Юк | Логотипта куллану өчен hex төс кодлары массивы (мәс., `["#678233", "#ffffff"]`) |
| `width` | number | Юк | SVG киңлеге пиксельләрдә (килешү буенча: 200) |
| `height` | number | Юк | SVG биеклеге пиксельләрдә (килешү буенча: 200) |
| `include_text` | boolean | Юк | Логотипта сайт исемен текст итеп кертү-кертмәү (килешү буенча: true) |

## Чыгару форматы

Мөмкинлек түбәндәге структуралы SVG юллыгын кайтара:

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## SVG чистарту тәртибе

Ясалган SVG’лар автоматик namespace-safe sanitisation үтә, моның өчен:

- **Куркынычсыз булмаган атрибутларны бетерү** — вакыйга эшкәрткечләрен, скриптларны һәм потенциаль куркыныч атрибутларны алып ташлый
- **Namespace’ларны саклау** — дөрес күрсәтү өчен SVG namespace’ларын (xmlns, xlink) саклый
- **Структураны тикшерү** — SVG’ның W3C стандартларына туры килүен тәэмин итә
- **Entity’ларны кодлау** — текст эчтәлегендә махсус символларны дөрес экранлаштыра
- **Тышкы сылтамаларны бетерү** — тышкы стиль таблицаларын һәм рәсем сылтамаларын алып ташлый

Бу SVG’ны өстәмә чистарту таләп итмичә турыдан-туры WordPress эченә урнаштыру өчен куркынычсыз итә.

## Куллану мисалы

**Сорау:**
```
"CloudSync" дип аталган tech стартабым өчен зәңгәр һәм ак төсләр кулланып modern логотип ясагыз.
```

**Нәтиҗә:**
Мөмкинлек SVG логотипын ясый, ул:
- "CloudSync" сайт исемен үз эченә ала
- Күрсәтелгән зәңгәр һәм ак төс схемасын куллана
- Modern дизайн принципларына иярә
- Автоматик рәвештә чистартылган һәм куллануга әзер

## Theme Builder белән интеграция

Theme Builder’ның дизайн юнәлешен сайлауын кулланганда, Generate Logo SVG мөмкинлеге:

1. Дизайн юнәлешегезне һәм төс палитрагызны анализлый
2. Өстенлекләрегезгә туры килгән махсус SVG логотипын ясый
3. Логотипны автоматик рәвештә сайтыгызның башлык/брендинг өлкәсенә урнаштыра
4. SVG’ны WordPress медиа эчендә махсус логотип итеп саклый

## Иң яхшы практикалар

- **Ачык стиль өстенлекләрен бирегез** — теләгән дизайн стилен тасвирлагыз (modern, classic, playful һ.б.)
- **Төсләрне күрсәтегез** — эзлеклелек өчен бренд төсләрегезне кертегез
- **Күрсәтүне сынагыз** — логотипның төрле экран үлчәмнәрендә дөрес күрсәтелүен тикшерегез
- **Алга таба көйләгез** — зурлыкны һәм урнашуны көйләү өчен WordPress’ның логотип көйләү коралларын кулланыгыз

## Чикләүләр

- SVG логотиплары статик графика буларак ясала (анимацияләнми)
- Күп элементлы катлаулы логотиплар кулдан камилләштерүне таләп итәргә мөмкин
- Махсус шрифтлар хупланмый; текст система шрифтларын куллана
- Бик зур яки бик кечкенә үлчәмнәр сыйфатка йогынты ясарга мөмкин
