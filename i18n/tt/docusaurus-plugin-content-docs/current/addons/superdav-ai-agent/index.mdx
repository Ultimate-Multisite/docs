# Superdav AI Agent

Superdav AI Agent — Gratis AI Agent мөмкинлекләрен тема-aware күнекмәләре, көчәйтелгән интернетта эзләү мөмкинлекләре, сайт төзүче функцияләре һәм өстәмә автоматлаштыру интеграцияләре белән киңәйтә торган алдынгы WordPress AI ярдәмчесе. 1.18.0 версиясе Google Calendar уку коралларын, контакт һәм катнашучы туры китерүне, кеше раславы капкаларын, искәртмә кабатлануын бетерүне, TextBee SMS хәбәрнамәләрен, идарә ителә торган Superdav сервисын әзерләүне һәм ышанычлы эшләүче процесслары өчен аерым таратыла торган Advanced companion пакетын өсти.

## Төп мөмкинлекләр

- **Тема-aware күнекмәләре** — күрсәтмәләрне һәм мөмкинлекләрне актив WordPress темагызга һәм урнаштырылган плагиннарга карап автоматик рәвештә җайлаштыра
- **Берничә интернет эзләү тәэмин итүчесе** — баерак эзләү нәтиҗәләре һәм алдынгы тикшеренү мөмкинлекләре өчен Brave Search һәм Tavily арасыннан сайлагыз
- **Сайт төзүчедә контакт формалары** — урнаштырылган форма плагиныгызны кулланып, чат интерфейсыннан турыдан-туры тулысынча эшли торган контакт формалары булдырыгыз
- **Көчәйтелгән сайт төзүче** — плагиннарны яхшырак табу һәм хаталардан торгызу белән камилләштерелгән күп этаплы сайт төзү планнары
- **Эчке тема күнекмәләре** — Блок темалары, Классик темалар, Kadence Blocks һәм Kadence Theme өчен махсуслаштырылган күрсәтмәләр
- **Яхшыртылган WooCommerce интеграциясе** — яхшырак ышанычлылык һәм ярашлылык өчен native WooCommerce API-ларын куллануга яңадан эшләнде
- **Тәэмин итүчеләрне автоматик ачыклау** — плагиннар һәм темалар активлаштырылганда яки активсызландырылганда, мөмкин булганнарын автоматик рәвештә ачыклый һәм яңарта
- **Үз логотибыңны булдыру** — namespace-safe автоматик чистарту белән үз SVG логотипларын булдырыгыз һәм кертегез
- **Палитра контрастын тикшерү** — темагызга кулланганчы, төсләр парларын WCAG керү мөмкинлеге таләпләренә туры килүгә тикшерегез
- **Фотолар белән баетылган ачыклау** — дизайн юнәлешенә йогынты ясау өчен Theme Builder ачыклау әңгәмәсе вакытында фотолар йөкләгез
- **Кунакчыллык менюлары** — рестораннар, кафелар һәм кунакчыллык бизнеслары өчен структуралы ризык һәм эчемлек меню битләре булдырыгыз
- **Өстәл һәм мобиль алдан караулары** — дизайн юнәлешен сайлау вакытында дизайныгызны өстәл һәм мобиль җайланмаларда алдан карагыз
- **1 нче дәрәҗә сайт кыру** — sd-ai-agent/site-scrape хәзер көндәш һәм илһам сайтларын анализлау өчен килешү буенча мөмкин
- **Google Calendar уку кораллары** — көйләнгән календарьларны һәм вакыйгаларны укыгыз, шулай итеп автоматлаштырулар киләчәк расписание контекстын аңлап эш итә ала
- **Контакт һәм катнашучы туры китерү** — алга таба эшләр булдырганчы календарь катнашучыларын билгеле сайт кулланучылары һәм контактлары белән туры китерегез
- **Кеше раславы капкалары** — хәбәрләр җибәрү, эшне дәвам итү яки үзгәрешләр кертү алдыннан сизгер автоматлаштыруларны карап чыгу өчен туктатып торыгыз
- **Искәртмә язмалары һәм кабатлануны бетерү** — инде җибәрелгән искәртмәләрне күзәтегез, шулай итеп кабат автоматлаштыру эшләүләре шул ук алучыга ике тапкыр хәбәр итми
- **TextBee SMS хәбәрнамәләре** — SMS тәэмин итүчесе кушылган булганда TextBee аша көйләнгән текст-хәбәр хәбәрнамәләрен җибәрегез
- **Идарә ителә торган Superdav сервисын әзерләү** — кул белән очлык көйләү урынына, автоматик көйләү белән хупланган сайтларны хостингтагы Superdav очлыкларына тоташтырыгыз
- **Advanced companion пакеты** — файл системасы, мәгълүмат базасы, WP-CLI, REST диспетчеры, плагин төзүче, git snapshot, кулланучы-идарәсе һәм benchmark эш процесслары өчен аерым таратыла торган ышанычлы эшләүче кораллары

## Таләпләр

- WordPress 6.0 яки югарырак
- PHP 8.1 яки югарырак
- WordPress Multisite (челтәрдә активлаштырылган) яки бер сайтлы урнаштыру
- Актив AI тәэмин итүче тоташтыргыч плагины (Anthropic, OpenAI яки ярашлы)
- Gratis AI Agent v1.9.0 яки югарырак (Superdav AI Agent Gratis AI Agent мөмкинлекләрен киңәйтә)

## Урнаштыру

1. Плагин файлларын `/wp-content/plugins/superdav-ai-agent` адресына йөкләгез
2. Плагинны WordPress эчендәге **Плагиннар** экраны аша активлаштырыгыз
3. Gratis AI Agent шулай ук активлаштырылган булуын тәэмин итегез
4. **Gratis AI Agent → Settings** бүлегенә күчегез һәм кирәк булганда интернет эзләү тәэмин итүчегезне, идарә ителә торган Superdav сервис тоташуын, Google Calendar яки TextBee таныклыкларын көйләгез

WordPress.org пакеты төп Superdav AI Agent мөмкинлекләрен үз эченә ала. Advanced companion пакеты ышанычлы эшләүче кораллары ачык итеп урнаштырылган һәм рөхсәт ителгән мохитләр өчен аерым төзелә һәм таратыла.

## Тиз башлау

Активлаштырылганнан соң, Superdav AI Agent сезнең Gratis AI Agent урнаштыруыгызны тема-aware күнекмәләре һәм өстәмә мөмкинлекләр белән автоматик рәвештә көчәйтә. WordPress идарә ян панеленда **Gratis AI Agent** панелен ачыгыз һәм яңа мөмкинлекләрне куллана башлагыз.

**Мисал күрсәтмәләр:**

```
Контакт битенә контакт формасы өстә.
```

```
Иң соңгы WordPress иминлек буенча иң яхшы гамәлләрне вебтан эзлә.
```

```
Актив теманың блок үрнәкләрен кулланып hero бүлеге булдыр.
```

```
Иртәгәге көйләнгән Google Calendar вакыйгаларын карап чык һәм катнашучылар өчен искәртмәләрнең караламасын төзе.
```

```
Мин хәбәрне раслаганнан соң SMS искәртмә җибәр.
```

## Тема-Aware күнекмәләре

Superdav AI Agent актив темагызны һәм урнаштырылган плагиннарны автоматик рәвештә ачыклый, ярдәмче контекстына тиешле тема-aware күнекмәсен йөкли. Мөмкин күнекмәләргә түбәндәгеләр керә:

| Күнекмә | Кайларга кулланыла |
|---|---|
| **Блок темалары** | WordPress блок нигезендәге (Тулы сайтны мөхәррирләү) архитектурасын кулланган темалар |
| **Классик темалар** | PHP шаблоннарын һәм классик мөхәррирне кулланган традицион WordPress темалары |
| **Kadence Blocks** | Kadence Blocks плагины актив булган сайтлар |
| **Kadence Theme** | Актив тема буларак Kadence темасы булган сайтлар |

Һәр күнекмә буенча җентекле күрсәтмәләр өчен [Тема-Aware күнекмәләре](../../user-guide/configuration/theme-aware-skills) карагыз.

## Интернет эзләү тәэмин итүчеләре

Superdav AI Agent v1.10.0 интернет эзләү сораулары өчен Brave Search-ка альтернатива буларак **Tavily** өсти. Ике тәэмин итүче дә реаль вакыттагы эзләү нәтиҗәләрен тәкъдим итә һәм **Gratis AI Agent → Settings → Advanced** эчендә көйләнә ала.

| Провайдер | Мөмкинлекләр | Көйләү |
|---|---|---|
| **Brave Search** | Тиз, хосусыйлыкка юнәлтелгән эзләү нәтиҗәләре | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Баерак эзләү нәтиҗәләре, алга киткән тикшеренү мөмкинлекләре | [Tavily API](https://tavily.com/) |

Көйләү күрсәтмәләре өчен [Интернет эзләү](../../user-guide/configuration/internet-search) карагыз.

## Календарь, раслау һәм хәбәрнамә эш агышлары

Superdav AI Agent v1.18.0 автоматлаштыру эш агышларын контент һәм сайт төзү бурычларыннан тыш киңәйтә:

- **Google Calendar кораллары** агентка календарь credentialлары булганда көйләнгән календарьларны һәм вакыйга мәгълүматларын укырга мөмкинлек бирә.
- **Контакт һәм катнашучы ярдәмчеләре** искәртмәләр яки алга таба үтәләсе бурычларны әзерләгәнче, вакыйгада катнашучыларны WordPress кулланучыларына яки билгеле контактларга туры китерә.
- **Кеше раслау капкалары** автоматлаштыруга тәкъдим ителгән гамәлне вәкаләтле кулланучы карап чыкканчы туктап торырга мөмкинлек бирә.
- **Искәртмә язмалары** автоматлаштыру кабат сынап каралганда яки кабат-кабат расписаниегә куелганда, календарь яки хәбәрнамә искәртмәләренең кабатлануын булдырмый.
- **TextBee SMS ярдәме** TextBee провайдеры көйләнгәндә текст-хәбәр хәбәрнамәләрен җибәрүне мөмкин итә.

Бу коралларны расписаниене исәпкә алган ярдәм, катнашучылар белән алга таба элемтә һәм сизгер хәбәрләр җибәрелгәнче һаман да кеше тикшерүен таләп итә торган искәртмә эш агышлары өчен кулланыгыз.

## Site Builder контакт формалары

Site Builder агент хәзер контакт формаларын турыдан-туры чат интерфейсыннан төзи ала. Site Builderдан контакт формасы төзүен генә сорагыз, һәм ул урнаштырылган форма pluginыгызны (Contact Form 7, WPForms, Fluent Forms яки Gravity Forms) автоматик рәвештә ачыклап, куллануга әзер форма булдырачак.

Төгәл күрсәтмәләр өчен [Site Builder контакт формалары](../../user-guide/administration/site-builder-contact-forms) карагыз.

## Theme Builder мөмкинлекләре

Superdav AI Agent v1.16.0 эчендәге Theme Builder көчле яңа мөмкинлекләрне үз эченә ала:

### Фото йөкләү белән ачыклау әңгәмәсе

Theme Builder ачыклау әңгәмәсенә хәзер визуаль илһам бирү өчен фото йөкләү адымы керә. Theme Builderга дизайн юнәлешегезне аңларга ярдәм итү өчен брендыгыз, көндәш сайтлар яки дизайн илһамы фотоларын йөкләгез.

Төгәл күрсәтмәләр өчен [Ачыклау әңгәмәсе](./theme-builder/discovery-interview.md) карагыз.

### Desktop һәм мобиль алдан караулар белән дизайн юнәлеше

Theme Builder тулы themeыгызны булдырганчы дизайныгызны desktop һәм мобиль җайланмаларда карап чыгыгыз. Берничә responsive сыну ноктасында реаль вакыттагы алдан карауларны күрегез.

Төгәл күрсәтмәләр өчен [Дизайн юнәлеше](./theme-builder/design-direction.md) карагыз.

### Кунакчыллык менюлары

Theme Builder хәзер рестораннар, кафелар, барлар һәм башка кунакчыллык бизнеслары өчен структуралы ризык һәм эчемлек меню битләрен булдыра ала. Менюларга бәяләр, тасвирламалар, диета мәгълүматы һәм аллерген кисәтүләре керә.

Төгәл күрсәтмәләр өчен [Кунакчыллык менюлары](./theme-builder/hospitality-menus.md) карагыз.

### Яңа сәләтләр

- **[Logo SVG булдыру](./abilities/generate-logo-svg.md)** — автоматик чистарту белән махсус SVG логотиплар булдырыгыз
- **[Палитра контрастын тикшерү](./abilities/validate-palette-contrast.md)** — WCAG таләпләренә туры килү өчен төс парларын тикшерегез
- **[Меню төзү](./abilities/create-menu.md)** — аерым навигация ярлыклары белән навигация менюлары төзегез

Мөмкин кораллар һәм Tier 1 сайт-скрейп коралы турында мәгълүмат өчен [Кораллар һәм көйләү](./theme-builder/tools.md) карагыз.

## Көйләү

Барлык көйләүләр WordPress администраторында **Gratis AI Agent → Settings** астында урнашкан. Superdav AI Agent түбәндәге көйләү параметрларын өсти:

| Көйләү | Тасвирлама |
|---|---|
| **Internet Search Provider** | Brave Search һәм Tavily арасыннан сайлагыз (мәҗбүри түгел) |
| **Tavily API Key** | Tavily интернет эзләве өчен API ачкычы (әгәр Tavily сайланган булса) |
| **Managed Superdav Service** | Хупланган сайтларны автоматик endpoint тәэмин итү белән хостингтагы Superdav сервисына тоташтырыгыз |
| **Google Calendar Credentials** | Агент календарьларны һәм вакыйгаларны укый алсын өчен календарь керүен көйләгез |
| **TextBee SMS Provider** | Расланган SMS хәбәрнамә эш агышлары өчен TextBee credentialларын көйләгез |

## Алга киткән companion пакеты

Superdav AI Agent v1.18.0 ышанычлы developer эш агышлары өчен аерым таратыла торган Advanced companion пакетын өсти. Бу мөмкинлекләр WordPress.org core пакетына кертелмәгән. Companion пакетын администраторлар агентның developer коралларына керүенә ачыктан-ачык ышанган сайтларда гына урнаштырыгыз.

Advanced коралларга файл системасы операцияләре, мәгълүмат базасына керү, WP-CLI, REST dispatcher эш агышлары, plugin builder утилиталары, git snapshots, кулланучы-идарә ярдәмчеләре һәм benchmark кораллары керә. Бу коралларны ышанычлы администраторлар белән генә чикләгез һәм аларны production сайтларда кушканчы approval-gate көйләүләрен карап чыгыгыз.
