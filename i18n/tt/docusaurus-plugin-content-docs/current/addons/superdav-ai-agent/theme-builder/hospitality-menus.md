# Кунакчыллык менюлары

**Кунакчыллык менюлары** мөмкинлеге Theme Builder’га структураланган ризык һәм эчемлек меню битләрен турыдан-туры сезнең WordPress сайтына ясарга һәм урнаштырырга мөмкинлек бирә.

## Гомуми күзәтү

Theme Builder хәзер рестораннар, кафелар, барлар һәм кейтеринг хезмәтләре кебек кунакчыллык бизнеслары өчен профессиональ, структураланган меню битләре булдыра ала. Бу менюлар сайтыгыз дизайнына тулысынча интеграцияләнә һәм җиңел яңартыла һәм идарә ителә ала.

## Хупланган кунакчыллык төрләре

- **Рестораннар** — тулы хезмәт күрсәтүле ашау менюлары
- **Кафелар** — кофе һәм җиңел ризык менюлары
- **Барлар һәм лаунжлар** — эчемлек һәм кабымлык менюлары
- **Икмәк-кондитер кибетләре** — камыр ризыклары һәм икмәк менюлары
- **Кейтеринг хезмәтләре** — чара меню вариантлары
- **Фудтраклар** — күчмә ризык хезмәте менюлары
- **Сыра һәм шәраб җитештерү урыннары** — тасвирламалы эчемлек менюлары

## Меню структурасы

### Меню категорияләре

Менюлар категорияләргә оештырыла:

- **Кабымлыклар** — башлангыч ризыклар һәм кечкенә порцияләр
- **Төп ризыклар** — төп ашлар
- **Гарнирлар** — өстәмәләр һәм яшелчәләр
- **Десертлар** — татлы сыйлар
- **Эчемлекләр** — эчемлекләр (алкогольле һәм алкогольсез)
- **Махсус тәкъдимнәр** — көндәлек яки сезонлы махсус тәкъдимнәр

### Меню пункты форматы

Һәр меню пункты үз эченә ала:

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### Пункт кырлары

| Кыр | Төр | Тасвирлама |
|-------|------|-------------|
| `name` | string | Ризык яки эчемлек исеме |
| `description` | string | Пунктның җентекле тасвирламасы |
| `price` | string | Бәя (валюта белән форматланган) |
| `dietary_info` | array | Диета атрибутлары (Веган, Глютенсыз һ.б.) |
| `allergens` | array | Киң таралган аллергеннар (Чикләвекләр, Кабырчыклы диңгез продуктлары һ.б.) |
| `availability` | string | Кайчан була (Көндәлек, Сезонлы һ.б.) |

## Кунакчыллык менюларын булдыру

### 1 нче адым: меню мәгълүматын бирү

Theme Builder’га менюыгыз турында сөйләгез:

```
Итальян рестораным өчен кабымлыклар, паста ризыклары, төп ашлар
һәм десертлар белән ресторан менюсы булдыр. Бәяләр һәм тасвирламаларны керт.
```

### 2 нче адым: Theme Builder меню булдыра

Theme Builder:

1. Структураланган меню бите булдыра
2. Аны сайтыгыз темасына туры китереп бизи
3. Пунктларны категорияләргә оештыра
4. Бәяләрне һәм тасвирламаларны форматлый
5. Диета һәм аллерген мәгълүматын өсти

### 3 нче адым: карау һәм көйләү

Сез булдыра аласыз:

1. Меню пунктларын һәм бәяләрне үзгәртү
2. Категорияләр өстәү яки бетерү
3. Категорияләр эчендә пунктлар тәртибен үзгәртү
4. Тасвирламаларны һәм диета мәгълүматын яңарту
5. Стильне һәм урнашуны көйләү

## Меню күрсәтү вариантлары

### Тулы меню бите

Тулы менюыгызны күрсәтүче аерым бит:

- Категория буенча оештырылган
- Эзләнә һәм фильтрлана ала
- Басма өчен уңайлы урнашу
- Мобиль җайланмаларга яраклашкан дизайн

### Меню виджеты

Меню бүлекләрен башка битләргә урнаштырыгыз:

- Баш биттә сайланган пунктлар
- Ян панельдә көндәлек махсус тәкъдимнәр
- Бар битендә эчемлек менюсы
- Түбән өлештә десерт витринасы

### Меню PDF

Йөкләп алына торган PDF меню ясагыз:

- Профессиональ форматлау
- Басмага әзер сыйфат
- Рәсемнәр һәм тасвирламаларны үз эченә ала
- Уртаклашу һәм электрон почта аша җибәрү җиңел

## Диета һәм аллерген мәгълүматы

### Диета атрибутлары

Пунктларны диета мәгълүматы белән билгеләгез:

- **Веган** — хайван продуктлары юк
- **Вегетариан** — ит юк
- **Глютенсыз** — целиакия авыруы өчен куркынычсыз
- **Сөт продуктларыннан азат** — сөт продуктлары юк
- **Чикләвексез** — агач чикләвекләре яки арахис юк
- **Аз углеводлы** — углеводлар киметелгән
- **Югары аксымлы** — аксымга бай

### Аллерген кисәтүләре

Киң таралган аллергеннарны кертегез:

- **Чикләвекләр** — агач чикләвекләре һәм арахис
- **Кабырчыклы диңгез продуктлары** — кысласыманнар һәм моллюсклар
- **Балык** — барлык балык төрләре
- **Сөт продуктлары** — сөт һәм сөт продуктлары
- **Йомырка** — тавык йомыркалары
- **Соя** — соя продуктлары
- **Глютен** — бодай һәм глютенлы бөртеклеләр
- **Кунжут** — кунжут орлыклары һәм мае

## Меню белән идарә итү

### Бәяләрне яңарту

Меню бәяләрен җиңел яңартыгыз:

1. Меню битенә күчегез
2. "Менюны үзгәртү"гә басыгыз
3. Пунктлар өчен бәяләрне яңартыгыз
4. Үзгәрешләрне саклагыз
5. Үзгәрешләр сайтыгызда шунда ук күренә

### Сезонлы пунктлар өстәү

Сезонлы меню вариантларын булдырыгыз:

1. Яңа меню версиясе булдырыгыз
2. Сезонлы пунктлар өстәгез
3. Пунктларны "Сезонлы" дип билгеләгез
4. Мөмкинлек даталарын планлаштырыгыз
5. Сезон вакытында автомат рәвештә күрсәтегез

### Махсус тәкъдимнәр белән идарә итү

Көндәлек яки атналык махсус тәкъдимнәрне күрсәтегез:

1. "Махсус тәкъдимнәр" категориясен булдырыгыз
2. Мөмкинлек даталары булган пунктлар өстәгез
3. Баш биттә махсус тәкъдимнәрне аерып күрсәтегез
4. Көндәлек яки атналык яңартыгыз
5. Иске махсус тәкъдимнәрне архивлагыз

## Theme Builder белән интеграция

Кунакчыллык сайтлары өчен Theme Builder кулланганда:

1. **Менюны автоматик ачыклау** — сезнең ризык/эчемлек бизнесы булу-булмавын билгели
2. **Меню бите булдыру** — профессиональ меню битләре ясый
3. **Дизайнны туры китерү** — менюлар сайтыгыз темасына туры килә
4. **Мобиль оптимизация** — менюлар телефоннарда матур күрсәтелә
5. **SEO оптимизация** — менюлар эзләү системалары өчен уңайлы

## Иң яхшы гамәлләр

### Меню дизайны

- **Ачык оештыру** — логик категория структурасы
- **Укыла торган тасвирламалар** — аппетит уята торган һәм мәгълүматлы
- **Эзлекле бәяләү** — ачык валюта һәм форматлау
- **Профессиональ фотолар** — югары сыйфатлы ризык рәсемнәре
- **Буш аралар** — битне артык тутырмагыз

### Эчтәлек

- **Төгәл тасвирламалар** — пунктларны төгәл тасвирлагыз
- **Махсус тәкъдимнәрне аерып күрсәтү** — махсус тәкъдимнәрне күзгә ташланырлык итегез
- **Аллергеннарны кертү** — аллергеннарны һәрвакыт күрсәтегез
- **Даими яңарту** — бәяләрне һәм пунктларны актуаль тотыгыз
- **Аппетит уята торган тел куллану** — пунктларны тәмле яңгырашлы итегез

### Һәркем өчен куллану мөмкинлеге

- **Укыла торган шрифтлар** — ачык, җиңел укыла торган шрифтлар кулланыгыз
- **Җитәрлек контраст** — текстның укыла алуын тәэмин итегез
- **Диета билгеләре** — диета вариантларын ачык билгеләгез
- **Аллерген кисәтүләре** — аллергеннарны күренекле итеп күрсәтегез
- **Мобиль җайланмалар өчен уңайлы** — барлык җайланмаларда сынагыз
