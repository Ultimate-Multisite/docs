# Ачыклау интервьюсы {#discovery-interview}

**Ачыклау интервьюсы** — Theme Builder'ның дизайн процессындагы беренче адым. Ул шәхсиләштерелгән тема булдыру өчен сайтыгыз, бизнесыгыз һәм дизайн өстенлекләрегез турында мәгълүмат җыя.

## Гомуми күзәтү {#overview}

Ачыклау интервьюсы брендыгызны, максат аудиториягезне һәм дизайн максатларыгызны аңлау өчен сезне сораулар сериясе аша алып бара. 1.16.0 версиясендә интервью хәзер дизайныгыз өчен визуаль илһам бирергә мөмкинлек бирүче **фото йөкләү адымын** үз эченә ала.

## Интервью адымнары {#interview-steps}

### 1. Бизнес мәгълүматы {#1-business-information}

- Сайт исеме һәм слоганы
- Бизнес төре яки тармак
- Төп бизнес максатлары
- Максат аудитория тасвирламасы

### 2. Дизайн өстенлекләре {#2-design-preferences}

- Өстен күрелгән төсләр схемасы
- Дизайн стиле (заманча, классик, минималистик, уенчак һ.б.)
- Илһам бирүче вебсайтлар яки брендлар
- Булырга тиешле дизайн элементлары

### 3. Фото йөкләү (v1.16.0'да яңа) {#3-photo-upload-new-in-v1160}

Дизайн юнәлешенә йогынты ясау өчен бер яки берничә фото йөкләгез:

#### Кабул ителә торган форматлар {#accepted-formats}

| Формат | Киңәйтелмә | Детальләр |
|--------|-----------|---------|
| **JPEG** | `.jpg`, `.jpeg` | Фотосурәтләр өчен тәкъдим ителә |
| **PNG** | `.png` | Үтә күренмәлелекне хуплый |
| **WebP** | `.webp` | Заманча формат, файл күләме кечерәк |
| **GIF** | `.gif` | Бары тик статик рәсемнәр генә (анимация юк) |

#### Файл таләпләре {#file-requirements}

- **Максималь файл күләме:** 10 MB
- **Минималь үлчәмнәр:** 400 × 300 пиксель
- **Тәкъдим ителгән үлчәмнәр:** 1200 × 800 пиксель яки зуррак
- **Яклар нисбәте:** Теләсә нинди нисбәт кабул ителә

#### Фотолар дизайн юнәлешенә ничек йогынты ясый {#how-photos-influence-design-direction}

Сез йөкләгән фотолар Theme Builder'га түбәндәгеләрне аңларга ярдәм итә:

- **Төсләр палитрасы** — өстенлекле төсләр чыгарыла һәм темада кулланыла
- **Визуаль стиль** — заманча, авылча, минималистик, җанлы һ.б.
- **Кәеф һәм атмосфера** — профессиональ, иҗади, дустанә, затлы һ.б.
- **Дизайн элементлары** — бизәкләр, текстуралар һәм композиция стильләре
- **Типография илһамы** — визуаль темага туры килә торган шрифт стильләре

### 4. Эчтәлек структурасы {#4-content-structure}

- Кирәкле төп битләр (Баш бит, Безнең турында, Хезмәтләр, Элемтә һ.б.)
- Блог яки яңалыклар бүлеге кирәкме
- Электрон сәүдә функциональлеге кирәкме
- Махсус мөмкинлекләр яки интеграцияләр

### 5. Карап чыгу һәм раслау {#5-review-and-confirmation}

- Бирелгән барлык мәгълүматның кыскача җыелмасы
- Дизайн генерациясе башланганчы раслау

## Фото йөкләүнең иң яхшы практикалары {#photo-upload-best-practices}

### Нәрсә йөкләргә {#what-to-upload}

- **Бренд илһамы** — логотиплар, төс үрнәкләре яки бренд кулланмалары
- **Көндәш сайтлар** — сезгә ошаган сайтларның скриншотлары
- **Кәеф такталары** — теләгән эстетиканы күрсәтүче рәсемнәр җыелмалары
- **Продукт фотолары** — күрсәтергә продуктларыгыз булса
- **Команда яки офис фотолары** — шәхси яки профессиональ брендинг өчен
- **Тормыш стиле рәсемнәре** — максат аудиториягезне чагылдыручы

### Нәрсәдән сакланырга {#what-to-avoid}

- **Түбән сыйфатлы рәсемнәр** — тонык яки пиксельләшкән фотолар
- **Бәйләнешсез рәсемнәр** — брендыгызны чагылдырмаган фотолар
- **Тексты күп рәсемнәр** — күп текстлы скриншотлар
- **Авторлык хокукы белән сакланган материал** — рәсемнәрне куллану хокукларыгыз булуына инаныгыз
- **Анимацияле GIF'лар** — бары тик статик рәсемнәр генә эшкәртелә

### Иң яхшы нәтиҗәләр өчен киңәшләр {#tips-for-best-results}

1. **2-3 рәсем йөкләгез** — берничә рәсем яхшырак контекст бирә
2. **Югары сыйфатлы фотолар кулланыгыз** — ачык, яхшы яктыртылган рәсемнәр иң яхшы эшли
3. **Төрлелек кертегез** — төс, стиль һәм тема катнашмасы
4. **Контекст бирегез** — һәр рәсемне ни өчен сайлавыгызны кыскача тасвирлагыз
5. **Ачык булыгыз** — әгәр аерым бер элемент ошаса, аны күрсәтегез

## Дизайн юнәлеше нәтиҗәсе {#design-direction-output}

Ачыклау интервьюсыннан соң, Theme Builder түбәндәгеләрне булдыра:

- **Төсләр палитрасы** — төп, икенчел һәм акцент төсләр
- **Типография** — башлыклар һәм төп текст өчен шрифт гаиләләре
- **Макет стиле** — аралар, тигезләү һәм компонентлар урнашуы
- **Визуаль элементлар** — иконалар, бизәкләр һәм декоратив элементлар
- **Кәеф тактасы** — дизайн юнәлешенең визуаль чагылышы

## Киләсе адымнар {#next-steps}

Ачыклау интервьюсын тәмамлагач:

1. Булдырылган дизайн юнәлешен карап чыгыгыз
2. Фикер белдерегез яки төзәтмәләр сорагыз
3. [Дизайн юнәлеше](./design-direction.md) адымына күчегез
4. Theme Builder сезнең махсус темагызны булдыра
