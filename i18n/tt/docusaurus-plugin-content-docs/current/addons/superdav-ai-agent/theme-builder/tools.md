# Кораллар һәм көйләү

Theme Builder сезнең WordPress сайтыгызны анализлау, дизайнлау һәм төзү өчен кораллар җыелмасын куллана. 1.16.0 версиясендә **sd-ai-agent/site-scrape хәзер 1 нче дәрәҗә коралы**, шуңа күрә ул Theme Builder эчендә килешү буенча кулланыла. Superdav AI Agent v1.18.0 шулай ук ышанычлы эшләүче агымнары өчен аерым таратыла торган Advanced юлдаш пакетын кертә; бу кораллар WordPress.org төп пакетының өлеше түгел һәм аерым урнаштырылырга һәм рөхсәтләнергә тиеш.

## Мөмкин булган кораллар {#available-tools}

### 1 нче дәрәҗә кораллары (һәрвакыт мөмкин) {#tier-1-tools-always-available}

1 нче дәрәҗә кораллары Theme Builder эчендә өстәмә көйләүсез килешү буенча мөмкин.

#### sd-ai-agent/site-scrape {#sd-ai-agentsite-scrape}

**Максат:** Дизайн илһамын, эчтәлек структурасын һәм функциональлекне алу өчен гамәлдәге вебсайтларны анализлау.

**Мөмкинлекләр:**

- **Вебсайт анализы** — көндәш яки илһам чыганагы булган вебсайтларны җыю һәм анализлау
- **Дизайнны алу** — төсләрне, шрифтларны һәм урнаштыру үрнәкләрен ачыклау
- **Эчтәлек структурасы** — бит оешмасын һәм иерархиясен аңлау
- **Функцияләрне ачыклау** — plugin-нарны һәм функциональлекне ачыклау
- **Эшчәнлек анализы** — бит тизлеген һәм оптимальләштерүне тикшерү
- **SEO анализы** — мета-тегларны һәм структурланган мәгълүматны карау

**Куллану:**

```
example.com дизайнын анализла һәм аны минем сайтым өчен илһам буларак куллан.
```

**Ул нәрсә ала:**

- Төсләр палитрасы һәм төсләрне куллану
- Типографика (шрифтлар һәм үлчәмнәр)
- Урнаштыру структурасы һәм аралар
- Навигация үрнәкләре
- Эчтәлек оешмасы
- Рәсемнәр һәм медиа куллану
- Интерактив элементлар
- Мобиль җайлашучанлык

**Чикләүләр:**

- Серсүз белән сакланган сайтларны җыя алмый
- robots.txt һәм сайт сәясәтләрен хөрмәт итә
- Динамик эчтәлекне тулы итеп эләктермәскә мөмкин
- JavaScript күп булган сайтларда анализ чикләнгән булырга мөмкин
- Зур сайтларны анализлау озаграк вакыт алырга мөмкин

### 2 нче дәрәҗә кораллары (ихтыяри) {#tier-2-tools-optional}

2 нче дәрәҗә кораллары Theme Builder көйләүләрендә ачык итеп кабызылганда мөмкин.

#### Алга киткән аналитика {#advanced-analytics}

Җентекле эшчәнлек күрсәткечләрен бирә:

- Бит йөкләнү вакытлары
- Core Web Vitals
- SEO бәясе
- Ачыклык бәясе
- Иң яхшы тәҗрибәләр бәясе

#### Эчтәлек оптимальләштергече {#content-optimizer}

Түбәндәгеләр өчен анализлый һәм яхшыртулар тәкъдим итә:

- Уку уңайлылыгы
- SEO оптимальләштерү
- Ачкыч сүзләр куллану
- Эчтәлек структурасы
- Гамәлгә чакыру урнашуы

### 3 нче дәрәҗә кораллары (премиум) {#tier-3-tools-premium}

3 нче дәрәҗә кораллары өстәмә көйләү яки премиум функцияләр таләп итә.

#### AI эчтәлек генераторы {#ai-content-generator}

Түбәндәгеләр өчен эчтәлек ясый:

- Товар тасвирламалары
- Хезмәт битләре
- Блог язмалары
- Мета-тасвирламалар
- Гамәлгә чакыру тексты

#### Алга киткән дизайн кораллары {#advanced-design-tools}

Алга киткән дизайн мөмкинлекләрен бирә:

- Махсус CSS ясау
- Анимация булдыру
- Интерактив элемент дизайны
- Алга киткән төс теориясе
- Типографиканы оптимальләштерү

### Алга киткән юлдаш кораллар (аерым пакет) {#advanced-companion-tools-separate-package}

Алга киткән юлдаш пакет төп Superdav AI Agent чыгарылышыннан аерым таратыла. Ул администраторлар агентка зур йогынтылы коралларны кулланырга ачык итеп рөхсәт иткән ышанычлы эшләүче һәм сайт хуҗасы мохитләре өчен билгеләнгән.

Алга киткән юлдаш мөмкинлекләре түбәндәгеләрне үз эченә ала ала:

- **Файл системасы кораллары** — ышанычлы мохитләрдә рөхсәт ителгән файлларны тикшерү һәм үзгәртү
- **Мәгълүмат базасы кораллары** — ачык итеп рөхсәт ителгәндә сайт мәгълүматын сорау яки яңарту
- **WP-CLI кораллары** — WordPress хезмәт күрсәтү һәм тикшерү командаларын эшләтү
- **REST диспетчер кораллары** — контрольдәге диспетчерлык агымнары аша теркәлгән REST endpoint-ларына мөрәҗәгать итү
- **Plugin төзү кораллары** — plugin кодын әзерләү һәм камилләштерү
- **Git снимок кораллары** — куркыныч эшләүче операцияләре алдыннан снимоклар булдыру
- **Кулланучылар белән идарә итү кораллары** — рөхсәт ителгән урыннарда кулланучы язмаларын карарга яки көйләштерергә ярдәм итү
- **Benchmark кораллары** — эшләүче каравы өчен эшчәнлек яки мөмкинлек benchmark-ларын эшләтү

Боларны һәр урнаштыру өчен гомумән мөмкин булган Theme Builder кораллары итеп документлаштырмагыз. Алар бары тик Advanced юлдаш пакеты урнаштырылган, кабызылган һәм ышанычлы администраторлар белән чикләнгән очракта гына мөмкин.

## Корал көйләү {#tool-configuration}

### Коралларны кабызу {#enabling-tools}

Theme Builder эчендә өстәмә коралларны кабызу өчен:

1. **Gratis AI Agent → Settings** бүлегенә күчегез
2. **Theme Builder → Tools** бүлегенә барыгыз
3. Кирәк булганда коралларны кабызыгыз/сүндерегез
4. Көйләүләрне саклагыз

### Корал рөхсәтләре {#tool-permissions}

Кайбер кораллар рөхсәтләр таләп итә:

- **Сайт җыю** — интернетка керүне таләп итә
- **Аналитика** — Google Analytics интеграциясен таләп итә
- **Эчтәлек ясау** — API ачкычларын таләп итә
- **Алга киткән функцияләр** — премиум subscription таләп итәргә мөмкин
- **Алга киткән юлдаш кораллар** — аерым таратыла торган Advanced пакетын һәм администраторның ачык ышанычын таләп итә

### API ачкычлары һәм керү мәгълүматлары {#api-keys-and-credentials}

Аларны таләп иткән кораллар өчен API ачкычларын көйләгез:

1. **Gratis AI Agent → Settings → API Keys** бүлегенә барыгыз
2. Һәр корал өчен керү мәгълүматларын кертегез
3. Тоташуны сынагыз
4. Куркынычсыз итеп саклагыз

### Алга киткән юлдаш пакетны урнаштыру {#installing-the-advanced-companion-package}

Алга киткән юлдаш пакет WordPress.org төп ZIP файлыннан аерым чыгарыла. Аны бары тик чыгарылыш өчен рәсми проект тарату каналыннан гына урнаштырыгыз, аннары керүне ышанычлы администраторлар белән чикләгез. Җитештерү сайтында файл системасы, мәгълүмат базасы, WP-CLI, REST диспетчер, plugin төзү, кулланучылар белән идарә итү яки benchmark коралларын кабызганчы, кеше раславы капкаларын һәм git снимок тәртибен карап чыгыгыз.

## sd-ai-agent/site-scrape куллану {#using-sd-ai-agentsite-scrape}

### Төп куллану {#basic-usage}

Theme Builder-дан вебсайтны анализлауны сорагыз:

```
competitor.com адресындагы көндәшем сайтын анализла
һәм минем сайтым өчен дизайн яхшыртуларын тәкъдим ит.
```

### Аерым анализ {#specific-analysis}

Аерым анализ төрләрен сорагыз:

```
example.com сайтыннан төсләр палитрасын ал һәм аны илһам буларак куллан.
```

```
example.com навигация структурасын анализла һәм охшаш
оешманы минем сайтыма куллан.
```

```
example.com сайтының мобиль җайлашучанлыгын тикшер һәм минем сайтның
шулай ук җайлашучан булуын тәэмин ит.
```

### Чагыштыру анализы {#comparison-analysis}

Берничә сайтны чагыштырыгыз:

```
site1.com һәм site2.com дизайннарын чагыштыр һәм икесенең дә иң яхшы
элементларын берләштергән дизайн булдыр.
```

## Корал чикләүләре һәм исәпкә алулар {#tool-limitations-and-considerations}

### Тизлекне чикләү {#rate-limiting}

- Scraping сервер артык йөкләнмәсен өчен тизлек буенча чикләнә
- Бер үк сайтка берничә сорау акрынайтылырга мөмкин
- Зур сайтларны анализлау озаграк вакыт алырга мөмкин

### Эчтәлек төгәллеге {#content-accuracy}

- Динамик эчтәлек тулысынча эләктерелмәскә мөмкин
- JavaScript аша күрсәтелгән эчтәлек тулы булмаска мөмкин
- Реаль вакыт мәгълүматлары чагылмаска мөмкин
- Кайбер эчтәлек түләүле киртәләр артында булырга мөмкин

### Хокукый һәм әхлакый мәсьәләләр {#legal-and-ethical-considerations}

- robots.txt һәм сайт сәясәтләрен хөрмәт итегез
- Авторлык хокукы белән сакланган эчтәлекне кабат бастыру өчен scraping ясамагыз
- Анализны күчереп алу өчен түгел, илһам өчен кулланыгыз
- Чыгарылган эчтәлекне кулланырга хокукларыгыз булуын тикшерегез
- Анализланган сайтларның хезмәт шартларын үтәгез

### Җитештерүчәнлеккә йогынты {#performance-impact}

- Зур сайтларны scraping ясау вакыт алырга мөмкин
- Берьюлы берничә scraping әкренрәк булырга мөмкин
- Челтәр тоташуы тизлеккә тәэсир итә
- Зур мәгълүмат җыелмалары күбрәк эшкәртү таләп итәргә мөмкин

## Иң яхшы практикалар {#best-practices}

### Сайт анализын куллану {#using-site-analysis}

1. **Берничә сайтны анализлагыз** — берничә чыганактан илһам туплагыз
2. **Структурага игътибар итегез** — макет һәм оештыру үрнәкләрен өйрәнегез
3. **Төсләрне чыгарыгыз** — төс палитраларын башлангыч нокта итеп кулланыгыз
4. **Типографияне өйрәнегез** — сезгә ошаган шрифт комбинацияләрен ачыклагыз
5. **Навигацияне карап чыгыгыз** — меню оештырылуын аңлагыз

### Әхлакый scraping {#ethical-scraping}

1. **robots.txt хөрмәт итегез** — сайт күрсәтмәләрен үтәгез
2. **Эчтәлекне күчермәгез** — анализны бары тик илһам өчен генә кулланыгыз
3. **Хокукларны тикшерегез** — чыгарылган эчтәлекне куллана алуыгызга инаныгыз
4. **Чыганакларны күрсәтегез** — илһам чыганакларын билгеләп үтегез
5. **Шартларны үтәгез** — сайтның хезмәт шартларын үтәгез

### Корал нәтиҗәлелеген максимальләштерү {#maximizing-tool-effectiveness}

1. **Анык булыгыз** — конкрет анализ төрләрен сорагыз
2. **Контекст бирегез** — сайтыгызның максатын аңлатыгыз
3. **Көтүләрне билгеләгез** — дизайн максатларыгызны тасвирлагыз
4. **Кабатлап камилләштерегез** — нәтиҗәләргә нигезләнеп төзәтегез
5. **Коралларны берләштерегез** — комплекслы анализ өчен берничә корал кулланыгыз

## Проблемаларны хәл итү {#troubleshooting}

### Сайт scraping ясалмый {#site-wont-scrape}

- Сайтның җәмәгатькә ачык булуын тикшерегез
- robots.txt scraping рөхсәт итүен тикшерегез
- Башка сайтны сынап карагыз
- Интернет тоташуын тикшерегез
- Мәсьәлә дәвам итсә, ярдәм хезмәтенә мөрәҗәгать итегез

### Тулы булмаган анализ {#incomplete-analysis}

- Сайтта динамик эчтәлек булырга мөмкин
- JavaScript тулысынча күрсәтелмәскә мөмкин
- Зур сайтларда вакыт чикләве килеп чыгарга мөмкин
- Аның урынына конкрет битләрне анализлап карагыз
- Конкрет анализ төрләрен сорагыз

### Әкрен эшләү {#slow-performance}

- Зур сайтларны анализлау озаграк вакыт ала
- Берьюлы берничә scraping әкренрәк була
- Челтәр тоташуы тизлеккә тәэсир итә
- Йөкләнеш түбән вакытларда сынап карагыз
- Башта кечерәк бүлекләрне анализлагыз
