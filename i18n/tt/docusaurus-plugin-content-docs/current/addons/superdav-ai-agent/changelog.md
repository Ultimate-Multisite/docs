# Үзгәрешләр журналы

## 1.18.0 — 2026-06-29 чыгарылды {#1180--released-on-2026-06-29}

### Яңа {#new}

- **Google Calendar кораллары** — расписаниене исәпкә алган автоматлаштырулар өчен көйләнгән календарьларны һәм вакыйгаларны уку
- **Контактларны бәйләү һәм катнашучылар ярдәмчеләре** — календарь катнашучыларын сайт кулланучылары һәм контактлар белән туры китерү
- **Кеше раславы капкалары һәм искәртү язмалары** — тикшерү өчен автоматлаштыруларны туктатып тору һәм кабатланган хәбәрнамәләрдән саклану
- **TextBee SMS тәэмин итүчесе** — көйләнгән текст-хәбәр хәбәрнамәләрен TextBee аша җибәрү
- **Advanced ярдәмче пакеты** — WordPress.org җыелмасыннан аерым таратыла торган ышанычлы-эшкәртүче файл системасы, мәгълүмат базасы, WP-CLI, REST диспетчеры, плагин төзүче, git snapshot, кулланучылар белән идарә итү һәм benchmark коралларын өстәү

### Яхшыртылды {#improved}

- **Идарә ителә торган Superdav сервисын көйләү** — хостингтагы сервис endpointларын һәм хупланган сайтлар өчен автоматик тоташу әзерләүне өстәү
- **Релизны пакетлау** — аерым төп һәм Advanced ZIPлар төзү, икесен дә GitHubта бастыру һәм WordPress.orgка бары тик төп пакетны гына җибәрү

### Төзәтелде {#fixed}

- **AI соравының ышанычлылыгы** — модель сайлауны, сорау timeoutларын, килешенгән көйләү моделен, фикерләү-текст эшкәртүен һәм дөрес булмаган корал чакырулары өчен кабатлау күрсәтмәләрен яхшырту
- **Календарь һәм искәртүләрне ныгыту** — Google Calendar tokenнарын һәм искәртүләрне кабатланудан саклауны көчәйтү
- **Onboarding һәм раслаудан соң дәвам итүләр** — frontend onboarding эшләтеп җибәрүләрен һәм расланган мөмкинлекләрнең дәвам итүен төзәтү
- **WordPress.org пакетлау тикшерүе мәсьәләләре** — төп релиз өчен пакетлау тикшерүе фикерләрен исәпкә алып төзәтү

## 1.16.0 — 2026-05-20 чыгарылды {#1160--released-on-2026-05-20}

### Яңа {#new-1}

- **Logo SVG ясау мөмкинлеге** — Theme Builder хәзер namespace-имин чистарту белән махсус logo SVGлар ясый һәм урнаштыра ала
- **Ачыклау интервьюсында фото йөкләү** — Theme Builder ачыклау интервьюсы хәзер баерак дизайн контексты өчен фото йөкләү адымын үз эченә ала
- **Палитра контрастын тикшерү мөмкинлеге** — темага кулланганчы, төс парларын WCAG таләпләренә туры килүгә тикшерү
- **Кунакчыллык менюлары** — Theme Builder хәзер кунакчыллык бизнеслары өчен структураланган ризык һәм эчемлек меню битләрен ясый ала
- **Desktop һәм mobile алдан карау күрсәтүе** — дизайн юнәлешен сайлаганда дизайныгызны desktop һәм mobile җайланмаларда алдан карау
- **Навигация ярлыгы параметры** — Create Menu мөмкинлеге хәзер бит исеменнән аерым `navigation_label` параметрын хуплый
- **Tier 1 корал мөмкинлеге** — sd-ai-agent/site-scrape хәзер Theme Builderда килешенгәнчә мөмкин булган Tier 1 коралы

### Төзәтелде {#fixed-1}

- **AI Client кэшы** — хәзер сораулар арасында саклану өчен transientлар белән тәэмин ителә, бу озак эшли торган агент биремнәрендә мәгълүмат югалуын булдырмый
- **Плагин юлы гамәл сылтамалары** — төгәллек өчен төзәтелде һәм яңадан аталды

## 1.10.0 — 2026-05-05 чыгарылды {#1100--released-on-2026-05-05}

### Яңа {#new-2}

- **Tavily интернет эзләве** — Brave Search белән беррәттән баерак интернет эзләү нәтиҗәләре өчен Tavilyны эзләү тәэмин итүчесе итеп өстәү
- **Теманы исәпкә алган эчке күнекмәләр** — Block Themes, Classic Themes, Kadence Blocks һәм Kadence Theme күнекмә кулланмалары хәзер плагин белән бергә килә
- **Сайт төзүченең контакт формасы мөмкинлеге** — чат интерфейсыннан турыдан-туры теләсә кайсы биткә контакт формасы өстәү

### Яхшыртылды {#improved-1}

- **WooCommerce интеграциясе яңадан эшләнде** — хәзер яхшырак ышанычлылык һәм ярашлылык өчен натив WooCommerce APIларын куллана
- **Тәэмин итүчеләр исемлеге автоматик яңара** — теләсә кайсы плагин активлаштырылганда яки сүндерелгәндә

### Төзәтелде {#fixed-2}

- **navigate-to мөмкинлеге** — кайбер admin битләрендә чиксез яңадан йөкләнү әйләнәсе төзәтелде
- **list-posts мөмкинлеге** — хәзер категория һәм tag исемнәрен slugларга дөрес билгели
- **WP-CLI командалары** — алдагы refactorдан соң югалган namespace aliasлары торгызылды
- **Вакыйга автоматлаштыруы** — автоматлаштыру таблицалары әле булдырылмаган сайтларны тотрыклы эшкәртә
- **memory-save мөмкинлеге** — хәзер система күрсәтмәсе төзүчедә дөрес namespace префиксын куллана
- **Скаляр корал нәтиҗәләре** — хәзер AIга кайтарылганчы дөрес төрелә
- **Куллану статистикасы** — хәзер искерәк версияләрдән яңартканда элекке ability key форматын дөрес эшкәртә
