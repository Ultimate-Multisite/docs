# Фильтр: signup_get_available_languages

Алгы өлештә сайт теркәүләре өчен мөмкин булган телләр исемлеген фильтрлый.

Бу hook-ка буш массив тапшыру теркәү формасында көйләү чыгышын сүндерәчәк, һәм сайт төзегәндә килешү буенча тел кулланылачак. Инде урнаштырылмаган телләр алып ташланачак.

## Параметрлар {#parameters}

| Исем | Төр | Тасвирлама |
|------|------|-------------|
| $available_languages | `array` | Мөмкин булган телләр. |

- 4.4.0
