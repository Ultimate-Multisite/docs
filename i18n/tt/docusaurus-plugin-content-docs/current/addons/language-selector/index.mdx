# Тел сайлагыч

Multisite Ultimate: теркәлүдә тел сайлагыч

Теркәлү агымына тел сайлау кырын өстәгез.

## Урнаштыру

1. 'ultimate-multisite-language-selector' ны '/wp-content/plugins/' директориясенә йөкләгез
2. Плагинны WordPress Network Admin эчендәге 'Плагиннар' менюсы аша активлаштырыгыз

## Куллану кулланмасы

Бу бушлай өстәмә ярдәмендә сез кулланучыларыгызга теркәлү агымында һәм керү битендә нинди тел күрергә теләүләрен сайларга мөмкинлек бирә аласыз.

[Лицензиягезне активлаштырганнан](../../user-guide/getting-started/installing-ultimate-multisite.md) соң, лицензиягез сезгә керү мөмкинлеге биргән Ultimate Multisite өстәмәләрен урнаштыра алачаксыз.

Ultimate Multisite көйләүләр битенә керегез, аннары сул яктагы менюда _Өстәмәләр_ тамгасына басыгыз.

![Өстәмәләр бите](/img/admin/addons-page.png)

_Language Selector урнаштыра башлау өчен Өстәмәләр битенә керегез_

Анда сез без тәкъдим иткән барлык өстәмәләрне табарсыз.

Бушлай өстәмәләр, язылу планына карамастан, барлык клиентларыбыз өчен дә ачык.

Өстәмә миниатюрасы астындагы _Хәзер урнаштырырга!_ төймәсенә басыгыз. Аннары аны Плагиннар битендә активлаштырыгыз.

![Урнаштыру төймәсе булган өстәмәләр бите](/img/admin/addons-page.png)

_Өстәмәне урнаштыру өчен төймәгә басыгыз_

<!-- Скриншот юк: WP Плагиннар битендә Language Selector өстәмәсе өчен Network Activate төймәсе күрсәтелә -->

_Аны Плагиннар битендә активлаштырыгыз_

Хәзер Ultimate Multisite'ның Көйләүләр менюсына керегез һәм _Челтәр көйләүләре_ кыстыргычын ачыгыз. Анда, битнең аскы өлешендә, _Телләр_ дип аталган яңа бүлек табарсыз.

![Көйләүләр бите](/img/config/settings-general.png)

_Бөтенләй яңа бүлек пәйда була!_

Барлык телләрне вариантлар итеп күрсәтү өчен аны буш калдыра аласыз яки кайсыларын ачык итәргә теләвегезне сайлый аласыз. Телләр — WordPress көйләнешегездә урнаштырылган телләр.

<!-- Скриншот юк: WP урнаштыруында мөмкин булган телләрне күрсәткән тел сайлау ачылма исемлеге -->

_Бу WordPress урнаштыруында мөмкин булган телләр исемлеге_

Керткән үзгәрешләрне саклаганнан соң, сез сайлаган телләр белән сайлау менюсы Керү битендә дә, Теркәлү агымында да күрсәтеләчәк.

![Керү һәм теркәлү көйләүләре](/img/config/settings-login-registration.png)

_Керү битендә Language Selector_

![Теркәлү бите](/img/config/registration-page.png)

_Теркәлү агымында Language Selector_

## Телне URL аша көйләү

Сез шулай ук URL-га locale параметрын тапшырып, битнең телен үзгәртә аласыз. Мәсәлән, итальян телендәге теркәлү битенә юнәлтергә теләсәгез, URL-га ?locale=it_IT өстәгез.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Бу эшләсен өчен, тел WordPress челтәрегездә урнаштырылган булырга тиеш.
