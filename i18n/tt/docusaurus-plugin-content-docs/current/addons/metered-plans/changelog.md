# Үлчәүле планнар үзгәрешләр журналы

Версия 1.1.0 - 2026-05-05 чыгарылды
- Яңа: multisite субсайтлары өчен AI token хисаплау — көйләнә торган һәр token өчен тарифлар белән customer сайтлары буйлап AI token кулланылышын күзәтегез һәм исәпләгез
- Яңа: Connector мәҗбүри куллану динамик лимит табу һәм write-through белән яңадан төзелде, барлык connectorлар буенча реаль вакыттагы төгәллекне тәэмин итә
- Төзәтү: Database таблица яңартулары хәзер BerlinDB schema билгеләмәләре белән дөрес туры китерелгән, яңа урнаштыруларда яңарту хаталарын булдырмый
- Төзәтү: Database яңарту callbackлары дөрес форматка күчерелде, тавышсыз яңарту хаталарын хәл итә
- Төзәтү: AI Usage Overage Markup кертү кырында хәзер вакланмалы кыйммәтләр кабул ителә
- Төзәтү: plugin йөкләнгәндә fatal errorлар һәм икеләтә initialization мәсьәләләре хәл ителде
- Яхшыртылды: developer мохитләре беренче эшләтүдә үз-үзен көйли алсын өчен check-env npm script өстәлде
