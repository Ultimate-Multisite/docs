# Плагин һәм тема менеджеры

Ultimate Multisite: Плагин һәм тема менеджеры

Урнаштырылган плагиннар һәм темаларның мета-мәгълүматын (исем, автор, миниатюра һәм тасвирлама) үзгәртегез яки яшерегез. Кулланучыларыгыз аларны фильтрлый алсын өчен плагиннар һәм темалар өчен категорияләр төзегез һәм кулланучыларыгыз панеленә матур махсус Плагиннар битен өстәгез.

Плагинны куллану өчен, челтәр администраторыгыздагы Плагиннар яки Темалар битләренә күчеп, "Плагин/Тема мәгълүматын үзгәртү" төймәсенә басу җитә.

## Урнаштыру

1. 'ultimate-multisite-plugin-and-theme-manager' ны '/wp-content/plugins/' директориясенә йөкләгез
2. Плагинны WordPress челтәр администраторыгыздагы 'Плагиннар' менюсы аша активлаштырыгыз

## Куллану кулланмасы

Ultimate Multisite: Плагин һәм тема менеджеры — көчле **премиум** Ultimate Multisite өстәмәсе, ул челтәр кулланучыларыгызның челтәрегездәге плагин һәм тема параметрларын ничек күрүе һәм алар белән ничек эш итүе өстеннән тулы контроль бирә.

Бу өстәмә өч максатка хезмәт итә:

  1. Ул плагиннарның да, темаларның да мета-мәгълүматын тулысынча үзгәртергә мөмкинлек бирә. Моңа исем, тасвирлама, автор исеме һәм хәтта миниатюралар да керә!

  2. Ул плагиннарны һәм темаларны категорияләргә бүлергә мөмкинлек бирә, бу кулланучыларыгызга аларны җиңел генә тәртипләргә ярдәм итә.

  3. Ул иске, күңелсез плагиннар битен категорияләр буенча сортлау мөмкинлеге булган яңа, темаларга охшаган бит белән алыштыру вариантын бирә.

Аны йөкләп урнаштырганнан соң, Ultimate Multisite көйләүләре битендә яңа кыстыргыч - _Өстәмә көйләүләре_ \- күренәчәк.

_Монда сез Ultimate Multisite: Плагин һәм тема менеджеры өстәгән глобаль параметрларны табарсыз_

Челтәрегезнең _Плагиннар_ битендә сез хәзер барлык плагиннарның мета-мәгълүматын, мәсәлән исемен, авторын һәм хәтта плагин миниатюрасын үзгәртә алачаксыз.

_Сез шулай ук аерым плагин мәгълүматын күрсәтергәме-юкмы икәнен сайлый аласыз_
