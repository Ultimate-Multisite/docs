# Tutor Multisite Compatibility

Мультисайт мохитләрендә Tutor LMS өчен ярашлылык төзәтмәләре өсти торган WordPress плагины.

## Тасвирлама

Бу плагин Tutor LMS-ны WordPress мультисайт челтәрләрендә эшләтү өчен кирәкле ярашлылык төзәтмәләрен бирә. Ул кулланучы язылуын, инструкторлар белән идарә итүне, курс чикләүләрен һәм мультисайтка хас башка функцияләрне эшкәртә.

## Мөмкинлекләр

- **Мәгълүмат базасы таблицалары белән идарә итү**: Мультисайт тышкы ачкыч чикләүләре аркасында төзелми калган Tutor LMS таблицаларын автоматик рәвештә таба һәм булдыра
- **Мультисайт кулланучы язылуы**: Челтәр сайтлары буенча кулланучы язылуын идарә итү өчен махсус AJAX эшкәрткечләре
- **Инструктор роле белән идарә итү**: Мультисайт мохитләрендә инструктор ролен автоматик билгеләү һәм идарә итү
- **Курс чикләүләре**: Сайт планнарына нигезләнеп курс һәм funnel төзү чикләрен идарә итү өчен Ultimate Multisite белән интеграция
- **Махсус админ күренешләре**: Мультисайтны исәпкә алган инструкторлар исемлеге һәм идарә интерфейсы
- **Админ кораллары**: Челтәр һәм сайт администраторлары өчен мәгълүмат базасы белән идарә итү интерфейсы

## Таләпләр

- WordPress 5.0+
- PHP 7.4+
- Tutor LMS плагины (`Requires Plugins` башлыгы аша автоматик рәвештә мәҗбүри ителә)
- WordPress Multisite урнаштыруы
- Ultimate Multisite плагины (курс чикләүләре функциясе өчен)

## Урнаштыру

1. Плагин файлларын `/wp-content/plugins/tutor-multisite-compatibility/` эченә йөкләгез
2. Tutor LMS урнаштырылган һәм активлаштырылган булуын тикшерегез
3. WordPress эчендәге «Плагиннар» менюсы аша плагинны активлаштырыгыз
4. Плагин Tutor LMS белән автоматик рәвештә интеграцияләнәчәк

## MU plugin-нан күчерелгән функциональлек

Бу плагин элек `wp-content/mu-plugins/daves.php` эчендә булган функциональлекне үз эченә ала:

- Курс статусы үзгәрү чикләүләре
- Курс төзү чикләүләре
- CartFlows интеграциясе һәм чикләре
- Tutor инструктор роле белән идарә итү
- Мультисайт кулланучы язылуын эшкәртү
- Курс продукты ID күрсәтү

## Мәгълүмат базасы белән идарә итү

### Проблема

WordPress мультисайт мохитләрендә Tutor LMS таблицалары кулланучылар таблицасына тышкы ачкыч чикләүләре булган таблицалар өчен төзелми. Бу түбәндәге сәбәпләр аркасында була:

- Сайтка хас таблицалар `wp_2_tutor_carts` кебек префикслар куллана
- Тышкы ачкычлар `wp_2_users`-ка сылтама ясый (ә ул юк)
- Чын кулланучылар таблицасы — `wp_users` (челтәр буенча уртак)

### Чишелеш

Бу плагин автоматик рәвештә:

1. **Җитешмәгән таблицаларны таба**: Тышкы ачкыч чикләүләре аркасында төзелми калган таблицаларны тикшерә
2. **Дөрес сылтамалар белән таблицалар булдыра**: Дөрес кулланучы таблицасы исемен куллана (мультисайтта `wp_users`)
3. **Берничә сценарийны эшкәртә**:
   - Плагинны активлаштыру/кабат активлаштыру
   - Челтәрдә яңа сайт булдыру
   - Админ интерфейсы аша таблицаларны кулдан тикшерү

### Йогынты ясалган таблицалар

- `tutor_coupon_usages` - Кулланучылар таблицасына тышкы ачкычны үз эченә ала
- `tutor_carts` - Кулланучылар таблицасына тышкы ачкычны үз эченә ала
- `tutor_notification_preferences` - Кулланучылар таблицасына тышкы ачкычы булган TutorPro таблицасы

### Админ интерфейсы

Мәгълүмат базасы белән идарә итү коралларына моннан керегез:

- **Челтәр админы**: Көйләүләр → Tutor Multisite DB
- **Сайт админы**: Кораллар → Tutor Multisite DB

Интерфейс сезгә түбәндәгеләрне башкарырга мөмкинлек бирә:
- Барлык сайтлар буенча җитешмәгән таблицаларны карау
- Таблица булдыруны кулдан эшләтеп җибәрү
- Мәгълүмат базасы конфигурациясен тикшерү
- Җентекле статус мәгълүматын карау

## Ярдәм

Бу плагин махсус рәвештә KursoPro платформасы өчен эшләнгән. Техник ярдәм өчен эшләүчеләр төркеме белән элемтәгә керегез.
