# Ярашлы ахыргы нокталар өчен Ultimate ЯИ тоташтыргычы

Ярашлы ахыргы нокталар өчен Ultimate ЯИ тоташтыргычы OpenAI-ярашлы ЯИ тәэмин итүче APIларын сезнең WordPress multisite челтәрегез белән берләштерә. Берничә ЯИ тәэмин итүчене көйләгез — һәркайсының үз ахыргы нокта URLы, API ачкычы һәм килешенгән моделе белән — һәм тәэмин итүче мөмкин булмаганда автоматик резервка күчү белән сорауларны юнәлтегез.

## Төп мөмкинлекләр

- **Берничә тәэмин итүчене хуплау**: Берничә ЯИ тәэмин итүче карточкасын өстәгез, һәркайсының үз ахыргы нокта URLы, API ачкычы һәм килешенгән моделе белән
- **Тартып тәртипләү**: Һәр карточкадагы тоткыч ярдәмендә тәэмин итүчеләрне өстенлек буенча урнаштырыгыз; сораулар тәэмин итүчеләрне тәртип буенча куллана
- **Автоматик резервка юнәлтү**: Төп тәэмин итүче мөмкин булмаса, сораулар автоматик рәвештә чираттагы көйләнгән тәэмин итүчегә күчә
- **Һәр тәэмин итүче өчен аерым көйләү**: Һәр тәэмин итүче үз таныклыклары һәм модель сайлавы белән мөстәкыйль көйләнә
- **Карточкаларга нигезләнгән көйләүләр UI**: Тотрыклы WordPress 6.9+ Card компонентларында төзелгән (CardBody, CardHeader, CardDivider)
- **Өстәгәндә автоматик җәелү**: Яңа тәэмин итүче карточкалары автоматик рәвештә җәелә, шуңа күрә сез детальләрне шунда ук тутыра аласыз
- **Скрипт кэшын яңарту**: Плагин яңартылганда тәэмин итүче скриптлары кэштан мәҗбүри яңартыла, иң соңгы версия йөкләнсен өчен

## Таләпләр

- WordPress 5.3 яки югарырак
- PHP 7.4 яки югарырак
- Ultimate Multisite плагин (актив)
- OpenAI-ярашлы API ахыргы ноктасы (үз серверыгызда урнаштырылган яки өченче якныкы)

## Урнаштыру

1. Өстәмә файлларын `/wp-content/plugins/` каталогыгызга йөкләгез
2. Плагинны WordPress'тагы 'Плагиннар' менюсы аша активлаштырыгыз
3. Өстәмәне көйләү өчен **Ultimate Multisite → ЯИ тоташтыргычы** бүлегенә күчегез

## Көйләү

### Берничә тәэмин итүчене көйләү

Көйләү экраны карточкаларга нигезләнгән урнашуны куллана, анда һәр карточка бер ЯИ тәэмин итүчене күрсәтә. Сез кирәк кадәр тәэмин итүче өсти аласыз.

#### Тәэмин итүче өстәү

1. Челтәр администраторында **Ultimate Multisite → ЯИ тоташтыргычы** бүлегенә керегез
2. **Тәэмин итүче өстәү** төймәсенә басыгыз
3. Яңа тәэмин итүче карточкасы пәйда була һәм автоматик рәвештә җәелә
4. Тәэмин итүче детальләрен тутырыгыз:
   - **Тәэмин итүче исеме** — бу тәэмин итүчене тану өчен билге (мәсәлән, "OpenAI", "Mistral", "Local Ollama")
   - **Ахыргы нокта URLы** — OpenAI-ярашлы APIның төп URLы (мәсәлән, `https://api.openai.com/v1`)
   - **API ачкычы** — бу тәэмин итүче өчен сезнең API ачкычыгыз
   - **Килешенгән модель** — бу тәэмин итүче өчен килешү буенча кулланылачак модель идентификаторы (мәсәлән, `gpt-4o`, `mistral-small`)
5. **Көйләүләрне саклау** төймәсенә басыгыз

#### Тәэмин итүчене бетерү

Аны бетерү өчен теләсә кайсы тәэмин итүче карточкасындагы **Бетерү** төймәсенә басыгыз. Үзгәрешне куллану өчен көйләүләрне саклагыз.

#### Тәэмин итүчеләрне яңадан тәртипләү

Теләсә кайсы тәэмин итүче карточкасының сул ягындагы тоткыч билгесен (⣿) тартып, аны яңадан тәртипләгез. Тәртип резерв өстенлеген билгели: сораулар беренче тәэмин итүчегә җибәрелә һәм сорау уңышсыз булса, аннан соңгы тәэмин итүчеләргә күчә.

### Резервка юнәлтү

Төп тәэмин итүчегә сорау уңышсыз булганда — көтү вакыты чыгу, тизлек чикләве яки хезмәт өзелү сәбәпле — тоташтыргыч исемлектәге чираттагы тәэмин итүче белән автоматик рәвештә кабатлап карый. Бу тәэмин итүче уңышка ирешкәнче яки барлык тәэмин итүчеләр сынап каралганчы исемлек буенча дәвам итә.

| Сценарий | Тәртип |
|----------|-----------|
| Төп тәэмин итүче җавап бирә | Сорау гадәттәгечә хезмәтләндерелә |
| Төп тәэмин итүче уңышсыз була | Сорау чираттагы тәэмин итүче белән кабатлана |
| Барлык тәэмин итүчеләр уңышсыз була | Чакыручыга хата кайтарыла |

Резервка күчү чакыручы плагин яки тема өчен үтә күренмәле — ничә тәэмин итүче сынап каралуына карамастан, алар я уңышлы җавап, я соңгы хата ала.

### Тәэмин итүче көйләүләре белешмәсе

| Кыр | Мәҗбүри | Тасвирлама |
|-------|----------|-------------|
| Тәэмин итүче исеме | Әйе | Бу тәэмин итүче өчен күрсәтелә торган билге |
| Ахыргы нокта URLы | Әйе | OpenAI-ярашлы APIның төп URLы |
| API ачкычы | Әйе | Бу тәэмин итүче өчен аутентификация ачкычы |
| Килешенгән модель | Әйе | Сорауда модель күрсәтелмәгәндә кулланыла торган модель идентификаторы |

## Носозлыкларны төзәтү

### Тәэмин итүчеләр күрсәтелми

Саклаганнан соң тәэмин итүчеләр исемлеге буш булса, моны раслагыз:

- Тәэмин итүчеләрне өстәгәннән соң **Көйләүләрне саклау** төймәсенә бастыңыз
- Браузерыгыз көйләүләр битен саклауны блокламады (JavaScript хаталары өчен браузер консолен тикшерегез)
- Плагин актив һәм сез челтәр администраторы экранында

### Сораулар барлык тәэмин итүчеләр өчен дә уңышсыз була

- Һәр тәэмин итүченең ахыргы нокта URLы сезнең сервердан барып җитәрлек булуын тикшерегез (браузерыгыздан гына түгел)
- Һәр API ачкычы дөрес һәм вакыты чыкмаган булуын тикшерегез
- Аларның Dashboard'ында тәэмин итүченең тизлек чикләве һәм квота халәтен карагыз
- Тоташтыргычтан җентекле хата хәбәрләрен тоту өчен WordPress көйләү журналын (`WP_DEBUG_LOG`) кабызыгыз

### Тәэмин итүче резерв өчен кулланылмый

Тартып тәртипләү резерв тәртибен үзгәртә. Резерв сорауларын алырга тиеш дип көткән тәэмин итүченең исемлектә дөрес урнашканын, шулай ук аның ахыргы нокта URLы һәм API ачкычы дөрес булуын раслагыз.

### Яңа тәэмин итүче карточкасы автоматик җәелми

Автоматик җәелү өчен браузерыгызда JavaScript кабызылган булырга тиеш. Карточка басканда җәелмәсә, хаталар өчен браузер консолен тикшерегез. Автоматик җәелү эшләмәсә дә, карточка эчтәлеге һәрвакыт карточка башлыгына басып ачыла.
