# Фильтр: wu_limits_is_fluent_form_above_limit

Бирелгән форма төренең бу планда рөхсәт ителгәнме-юкмы икәнен тикшерә. Plugin эшләүчеләргә кайтарылган кыйммәтне фильтрларга мөмкинлек бирә

## Параметрлар

| Исем | Төр | Тасвирлама |
|------|------|-------------|
| $value | `bool` | Форма төре чиктән югарымы, юкмы |
| $form_type | `string` | Тикшерелә торган форма төре |
| $form_count | `int` | Хәзерге форма саны |
| $quota | `int` | Рөхсәт ителгән квота |

### Версиядән башлап

- 1.0.0
