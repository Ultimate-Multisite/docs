# Admin Page Creator

## Эчтәлек
Admin Page Creator сезгә WordPress идарә Dashboard эчендә яңа идарә битләрен һәм асбитләрне җиңел өстәргә мөмкинлек бирәчәк. Аны яңа кулланучылар өчен төшү бите булып хезмәт итә ала торган сәламләү бите яки өйрәтү бите ясау өчен кулланырга мөмкин. Битләрне мәшәкатьсез бизәү өчен Beaver Builder, Elementor, Brizy һәм Oxygen кебек бит төзүчеләрне куллана аласыз.

Ул шулай ук махсус HTML, CSS һәм JavaScript ярдәме, шулай ук план һәм кулланучы роленә нигезләнгән керү контролен тәкъдим итә!

## Мөмкинлекләр
* Махсус HTML һәм CSS ярдәме
* JavaScript ярдәме
* PHP эчтәлек режимы
* Кулланучы роленә нигезләнгән рөхсәт
* Бит төзүчеләргә ярдәм

## Урнаштыру

1. **ultimate-multisite-admin-page-creator** ны **/wp-content/plugins/** каталогына йөкләгез
2. WordPress Network Admin эчендәге **Plugins** менюсы аша плагинны активлаштырыгыз

## Куллану күрсәтмәсе

WordPress сайтларның алгы ягына эчтәлек өстәүне чыннан да бик җиңел итсә дә, идарә ягында битләр һәм асбитләр ясау кайбер төп кодлау күнекмәләрен таләп итә. Ultimate Multisite: Admin Page Creator челтәрегездәге сайтларның арткы ягына алгы яктагы шул ук куллану җиңеллеген китерүне максат итә.

## Ни өчен Admin Page Creator кулланырга?

Бәлки, сез клиентларыгызның идарә панелендә өйрәтмәләр белән махсус бит тәкъдим итәргә телисездер, бәлки, FAQ һәм элемтә виджеты булган махсус бит өстәргә телисездер. Admin Page Creator белән мөмкинлекләр чиксез, һәм без бүгенге өйрәтмәдә нәкъ шуны өйрәнәчәкбез.

## Беренче махсус идарә битегезне ясау

Ultimate Multisite: Admin Page Creator урнаштырып активлаштырганнан соң, челтәр идарә менюсына керегез һәм Admin Pages бүлегенә күчегез.

![Network Admin эчендәге махсус идарә битләре исемлеге](/img/admin/admin-pages-list.png)

_Network Admin эчендәге махсус идарә битләре исемлеге_

Сез әлегә кадәр ясаган барлык идарә битләре исемлеген күрерсез (бу сезнең беренче керүегез булганга, исемлек буш булачак). Дәвам итегез һәм **Яңа Admin Page өстәү** төймәсенә басыгыз. Сез Admin Page мөхәрриренә юнәлтеләчәксез.

## Махсус идарә бите элементлары

Admin Page мөхәррирендә битегезне көйләү өчен төрле вариантлар булачак.

### Бит исеме

Бу — битнең өске өлешендә күрсәтеләчәк исем, идарә менюсы тамгасы түгел.

### Эчтәлек төре сайлагычы

Ultimate Multisite: Admin Page Creator бит эчтәлегенә килгәндә сезгә ике вариант бирә: сез стандарт WordPress WYSIWYG мөхәррирен куллана аласыз яки HTML мөхәрриренә күчә аласыз.

HTML белән таныш булмасагыз, стандарт WordPress мөхәррирендә калыгыз. Сез бит эчтәлеген гадәти язма яки бит белән эшләгән кебек үк үзгәртә алачаксыз.

## Меню вариантлары

Ян якта, General Options мета-тартмасында, сез меню төрен, тамгасын һәм тагын күп нәрсәне көйли алачаксыз.

### Меню төрләре

Яңа махсус идарә битегезнең ике төрле меню төре булырга мөмкин: югары дәрәҗәдәге меню һәм асменю.

Югары дәрәҗәдәге меню бите, исеменнән күренгәнчә, яңа ясалган битегезне стандарт WordPress битләре белән бергә идарә менюсына өстәячәк.

Югары дәрәҗәдәге вариантны сайласагыз, меню тәртибен һәм меню иконасын да сайлый алачаксыз.

Бу битне асменю бите итәргә теләсәгез, бу битне беркетү өчен ата битне сайлый алачаксыз. Махсус битегезне стандарт WordPress битләренә яки үзегезнең югары дәрәҗәдәге махсус идарә битләрегезгә өсти аласыз.

## Алга киткән вариантлар

Сез махсус идарә битләрегезгә төрле эчтәлек өсти алырга тиеш. Моңа CSS кагыйдәләре һәм файллары, JavaScript, шулай ук тышкы китапханәләр дә керә. Моны Advanced Options кыстыргычы ярдәмендә эшли аласыз.

### Махсус CSS

CSS кыстыргычында сез үзегезнең махсус CSS кодыгызны өсти аласыз, шулай ук ерак сайтлардан CSS файлларын импортлый аласыз (CDN, Google Fonts һ.б. кебек).

### Махсус JavaScript

JavaScript өчен дә шул ук гамәлдә. Сез махсус JavaScript кодыгызда куллану өчен тышкы китапханәләр өсти аласыз.

## Рөхсәтләр

Шулай ук Advanced Options эчендә сез бу биткә кайсы ас-сайтларның керә алачагын сайлый алачаксыз. Мәсәлән, төрле план дәрәҗәләренә төрле эксклюзив эчтәлек бирергә җыенсагыз, бу файдалы.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Бу битнең кайда күрсәтеләчәген тулысынча контрольдә тотасыз_

**Мөһим** : Бит күрсәтелсен өчен ике шарт та үтәлергә тиеш. Бу шуны аңлата: әгәр кулланучы билгеле бер планның клиенты булса, әмма сайланган рольләрнең берсенә дә ия булмаса (яки киресенчә), бит аның панелендә күренмәячәк.

## Актив

Әгәр сез махсус admin битен бетермичә яки рөхсәтләргә кагылмыйча сүндерергә теләсәгез, Active опциясеннән билгене генә алыгыз.

Югары дәрәҗәдәге битне активсызландыру шул ата-ана битенә беркетелгән барлык аскы битләрне автомат рәвештә активсызландырачак.
