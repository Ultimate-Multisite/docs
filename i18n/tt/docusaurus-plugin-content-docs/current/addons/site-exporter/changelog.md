# Site Exporter үзгәрешләр журналы

Версия: 1.0.6 - 2026-05-11 чыгарылды
* Яңа: сайт экспортлары хәзер үзеннән-үзе йөкләнә торган index.php-ны бергә туплый, шуңа күрә ZIP яңа хостка аерым плагин урнаштырмыйча урнаштырыла ала
* Яңа: челтәр экспорты администраторларга Site Export идарә битеннән барлык аскы сайтларны бер архивка экспортларга мөмкинлек бирә

Версия: 1.0.5 - 2026-05-05 чыгарылды
* Төзәтү: WP_CLI polyfill stub-ка __callStatic өстәлде, get_config кебек мокланмаган методларны эшкәртү һәм тест bootstrap өзелүләрен булдырмау өчен

Версия: 1.0.4 - 2025-11-25 чыгарылды

* Ultimate Multisite 2.4 белән ярашлылыкны төзәтү.

Версия: 1.0.2 - 2025-09-28 чыгарылды

* Префиксны ultimate-multisite итеп үзгәртү; text domain яңарту; версияне күтәрү.
