# Сайт экспортлаучысы

## Эчтәлек

Клиентыгыз үз веб-сайтын күчерергә карар иткән очракта, челтәрегездәге субсайтларны бер сайт урнаштыруына һәм киресенчә күчерә алу ихтыяҗын без аңлыйбыз. Сайт экспортлаучы һәм импортлаучы өстәмәсе бу бурычны җиңеләйтә. Бу корал сайтларны zip файллары ярдәмендә экспортлый һәм импортлый; алар шулай ук резерв күчермә булып та хезмәт итә ала.

## Мөмкинлекләр

* Сайтларны экспортлау һәм импортлау

* Куллану җиңел булган интерфейс

* Сайтның резерв күчермәсен булдыру

## Документация
**Сайтны ничек экспортларга һәм импортларга**

## Урнаштыру

1. **ultimate-multisite-site-exporter** ны **/wp-content/plugins/** директориясенә йөкләгез
2. WordPress эчендәге **Плагиннар** менюсы аша плагинны активлаштырыгыз
3. Плагинны көйләү өчен адымлап Wizard күрсәтмәләрен үтәгез

## Куллану кулланмасы

Ultimate Multisite белән сез челтәрегездән сайтны экспортлый яки multisite йә single-site урнаштыруыгызга сайт импортлый аласыз.

Процессны башлау өчен, безнең өстәмәләр битенә керегез һәм Ultimate Multisite: Сайт экспортлаучысы һәм импортлаучысы өстәмәсен урнаштырыгыз.![Өстәмәләр бите](/img/admin/addons-page.png)

![Урнаштыру мөмкинлеге булган өстәмәләр бите](/img/admin/addons-page.png)

Plugins битегезгә керегез һәм әле генә урнаштырган өстәмәне **Челтәрдә активлаштыру** өчен басыгыз.<!-- Скриншот юк: WP Plugins бите Site Exporter өстәмәсе өчен Network Activate төймәсен күрсәтә -->

**Экспорт һәм импорт** опциясен сул яктагы Ultimate Multisite менюсында табып була.

## Сайтны ничек экспортларга

Челтәрегездән сайтны экспортлау өчен, Ultimate Multisite менюсындагы **Экспорт һәм импорт** бүлегенә керегез һәм **Сайтларны экспортлау** вкладкасында **Яңа сайт экспорты булдыру** төймәсенә басыгыз.

<!-- Скриншот юк: Export Sites вкладкасында Generate new Site Export төймәсе -->

Экспортларга теләгән сайтны эзләгез һәм экспортланган файлга кертергә теләгән элементларны сайлагыз. Сез темаларны, плагиннарны һәм йөкләүләрне кертә аласыз. Әгәр сайтыгыз артык зур булса, экспорт процессын фонда эшләтүне сайлый аласыз.<!-- Скриншот юк: сайт эзләү, темалар/плагиннар/йөкләүләрне кертү өчен билгеләү шакмаклары булган сайт экспорты опцияләре -->

**Сайтны экспортлау** төймәсенә басыгыз. Экспортланган сайтыгыз **Мөмкин экспортлар** исемлегендә булачак. Файлны йөкләп алу өчен **ZIP йөкләп алу** төймәсенә басыгыз. Экспорт ZIPлары үзеннән-үзе йөкләнә торган `index.php` һәм `readme.txt` файлларын үз эченә ала, шуңа күрә аларны яңа host-та аерым importer plugin урнаштырмыйча да урнаштырып була.<!-- Скриншот юк: Download ZIP төймәсе булган Existing Exports исемлеге -->

## Бөтен челтәрне ничек экспортларга

Барлык субсайтларны бергә экспортлау өчен, Ultimate Multisite менюсындагы **Экспорт һәм импорт** бүлегенә керегез һәм экспорт экранында **Челтәр экспорты** опциясен кулланыгыз. Булдырылган архив host миграцияләре, staging яңадан җыюлары яки челтәр дәрәҗәсендәге резерв күчермәләр өчен барлык субсайтларны бер ZIP эчендә саклый.

Челтәр экспортлары зур булырга мөмкин. Аларны трафик түбән булганда эшләтегез һәм серверда архив өчен дә, кертелгән йөкләүләр, плагиннар һәм темалар өчен дә җитәрлек саклау урыны булуын раслагыз.

## Сайтны ничек импортларга

Сайтны импортлау өчен, **Сайтны импортлау** вкладкасына керегез һәм импортларга теләгән сайтның .zip файлын йөкләгез. URL-ны яңасына үзгәртә аласыз яки урынны буш калдырып, иске URL-ны саклый аласыз. Шулай ук импорттан соң .zip файлны урнаштыруыгыздан бетерү өчен шакмакны билгели аласыз. Аннары **Импортны башлау** төймәсенә басыгыз.

<!-- Скриншот юк: zip йөкләү кыры һәм Begin Import төймәсе булган Import a Site вкладкасы -->

Процесс бераз вакыт алырга мөмкин. Импортлаучы астындагы исемлектә көтелгән импортны күрерсез.

<!-- Скриншот юк: импортлаучы астындагы исемлектә көтелгән импорт -->

**Импортны туктату** төймәсенә басып, процессны туктата аласыз.

<!-- Скриншот юк: импорт процессын туктату өчен Cancel Import төймәсе -->

Процесс тәмамлангач, сайтыгыз инде челтәрегездә мөмкин булачак.

## Бер урнаштыруда сайтны ничек импортларга

Сайтны бер урнаштыруыгызга импортлау өчен, башта сайтыгызда импортлаучыны урнаштырыгыз һәм активлаштырыгыз. Аны йөкләп алу сылтамасын экспортлаучы вкладкасында таба аласыз.

<!-- Скриншот юк: single-site importer йөкләп алу сылтамасы булган Exporter вкладкасы -->

Импортлаучыны сул яктагы **Кораллар** менюсында, **Экспорт һәм импорт** аскы пунктында таба аласыз.

<!-- Скриншот юк: Export and Import аскы менюсын күрсәткән single-site Tools менюсы -->

Импортларга теләгән сайтның .zip файлын йөкләгез. URL-ны яңасына үзгәртә аласыз яки урынны буш калдырып, иске URL-ны саклый аласыз. Шулай ук импорттан соң .zip файлны урнаштыруыгыздан бетерү өчен шакмакны билгели аласыз. Аннары **Импортны башлау** төймәсенә басыгыз.

<!-- Скриншот юк: zip йөкләү һәм Begin Import төймәсе булган single-site импорт формасы -->

Процесс бераз вакыт алырга мөмкин. Импортлаучы астындагы исемлектә көтелгән импортны күрерсез.

<!-- Скриншот юк: исемлектәге single-site көтелгән импорт -->

**Импортны туктату** төймәсенә басып, процессны туктата аласыз.

<!-- Скриншот юк: импорт процессын туктату өчен Cancel Import төймәсе -->

Процесс тәмамлангач, сайтыгыз әле генә импортлаган сайт белән алыштырылачак.
