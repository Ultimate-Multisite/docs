# Ярдәм мөрәҗәгатьләре

**Ярдәм мөрәҗәгатьләре** — Ultimate Multisite өчен көчле өстәмә, ул сезнең WordPress челтәренә тулы ярдәм мөрәҗәгатьләре системасын өсти. Клиентларга ярдәмне челтәр дәрәҗәсендә дә, аерым сайтлар дәрәҗәсендә дә күрсәтергә кирәк булган SaaS платформалары, WaaS (Хезмәт буларак веб-сайт) тәэмин итүчеләре һәм күпсайтлы челтәрләр өчен бик кулай.

## Ни өчен Ярдәм мөрәҗәгатьләрен сайларга?

* **Күп дәрәҗәле ярдәм**: Челтәр администраторлары бөтен челтәр буенча мөрәҗәгатьләрне эшкәртә, ә сайт хуҗалары үз клиентларына ярдәм күрсәтүне идарә итә
* **Һөнәри эш агымы**: Өстенлекләр, статуслар һәм билгеләүләр белән мөрәҗәгатьнең тулы яшәү циклы белән идарә итү
* **Клиент тәҗрибәсе**: Клиентларга ярдәм алуны җиңеләйтә торган матур, җайлашучан интерфейс
* **Имин һәм ышанычлы**: WordPress иминлегенең иң яхшы практикаларына нигезләнеп төзелгән һәм мәгълүмат базасы җитештерүчәнлеге өчен оптимальләштерелгән
* **Җайлы интеграция**: Ultimate Multisite-ның клиентлар һәм сайтлар белән идарә итүе белән камил эшли

## Төп мөмкинлекләр

**Мөрәҗәгатьләр белән тулы идарә итү**

* Җиңел сылтама өчен уникаль мөрәҗәгать номерлау системасы
* Көйләнә торган өстенлек дәрәҗәләре (Түбән, Гадәти, Югары, Кискен)
* Тулы статус эш агымы (Ачык, Эштә, Көтелә, Хәл ителгән, Ябык)
* Мөрәҗәгатьләрне категорияләү (Гомуми, Техник, Түләү, Функция соравы, Хата турында хәбәр)
* Мөрәҗәгатьләрне ярдәм агентларына билгеләү, ихтыярый автоматик билгеләү белән

**Җепле сөйләшүләр**

* Мөрәҗәгать җаваплары өчен тулы сөйләшү җепләре
* Хезмәткәрләр һәм клиент җаваплары өчен аерым стиль
* Клиентлар күрә алмый торган шәхси хезмәткәр искәрмәләре
* Реаль вакыттагы AJAX нигезендәге интерфейс
* Форматлау мөмкинлекләре булган заманча текст мөхәррире

**Файл кушымталары**

* Сөйрәп ташлау ярдәме белән имин файл йөкләү
* Көйләнә торган файл төре һәм зурлык чикләүләре
* Зарарлы файлларны ачыклау белән алдынгы иминлек
* Керү идарәләре белән сакланган йөкләп алулар

**Акыллы хәбәрнамәләр**

* Яңа мөрәҗәгатьләр, җаваплар һәм статус үзгәрешләре өчен автоматик email хәбәрләре
* Көйләнә торган email шаблоннары
* Хезмәткәрләр һәм клиентлар өчен рольгә нигезләнгән хәбәрнамәләр
* Шәхсиләштерелгән хәбәрләр өчен динамик шаблон үзгәрүчәннәре

**Көчле администратор интерфейсы**

* Сортлау һәм фильтрлау белән алдынгы мөрәҗәгатьләр исемлеге
* Статус, өстенлек һәм билгеләүләр өчен тиз үзгәртү
* Нәтиҗәле идарә итү өчен күмәк гамәлләр
* Җитештерүчәнлек күрсәткечләре белән статистика Dashboard
* Тулы көйләүләр панели

**Матур клиент порталы**

* Барлык җайланмаларда чиста, җайлашучан интерфейс
* Мобиль җайланмаларга өстенлек бирелгән дизайн
* Алдынгы эзләү һәм фильтрлау
* Форматлау кораллар панели белән заманча мөхәррир
* Куллану җиңел булган мөрәҗәгать җибәрү формалары

**Сыгылмалы интеграция**

* Шорткодлар яки Ultimate Multisite UI элементларын кулланыгыз
* Мөрәҗәгать формаларын һәм исемлекләрен сайтыгызның теләсә кайсы урынында күрсәтегез
* Күпсайтлы челтәр төзелеше белән тоткарлыксыз эшли
* Hook һәм фильтрлар белән киңәйтелә

## Кемнәр өчен кулай

* Клиентларга ярдәм кирәк булган SaaS платформалары
* Күп клиент сайтларын идарә итүче WaaS тәэмин итүчеләре
* Дәрәҗәле ярдәме булган әгъзалык сайтлары
* Күп брендлар белән идарә итүче челтәр администраторлары
* Клиентларга хезмәт күрсәтүче сайт хуҗалары

## Урнаштыру

1. Ultimate Multisite plugin сезнең WordPress челтәрендә урнаштырылган һәм активлаштырылган булуына инаныгыз
2. Өстәмә файлларын `/wp-content/plugins/` директориясенә йөкләгез
3. WordPress-та 'Plugins' менюсы аша plugin-ны челтәр күләмендә активлаштырыгыз
4. Көйләүләрегезне конфигурацияләү өчен Челтәр администраторы > Ярдәм мөрәҗәгатьләренә күчегез

## Еш бирелә торган сораулар

## Моның өчен Ultimate Multisite кирәкме?

Әйе, бу өстәмә Ultimate Multisite урнаштырылган һәм актив булуын таләп итә. Ул Ultimate Multisite-ның клиентлар һәм сайтлар белән идарә итү мөмкинлекләре белән тирән интеграцияләнә.

## Сайт хуҗалары үз клиентларына ярдәмне үзләре күрсәтә аламы?

Әлбәттә! Сайт хуҗалары мөрәҗәгатьләр системасын үз клиентларына ярдәм күрсәтү өчен куллана ала, ә челтәр администраторлары бөтен челтәр буенча ярдәм мөрәҗәгатьләрен эшкәртә ала.

## Клиентлар мөрәҗәгатьләрне ничек җибәрә?

Сез шорткодлар яки Ultimate Multisite UI элементлары ярдәмендә сайтыгызның теләсә кайсы урынына мөрәҗәгать җибәрү формаларын өсти аласыз. Клиентлар форманы гына тутыра һәм үз соравын җибәрә.

## Мөрәҗәгать статусларын һәм өстенлекләрен көйли аламмы?

Әйе! Ярдәм эш агымыгызга туры килсен өчен челтәр администраторы көйләүләрендә өстенлек дәрәҗәләрен, статусларны һәм мөрәҗәгать төрләрен тулысынча көйли аласыз.

## Ул файл кушымталарын хуплыймы?

Әйе, клиентлар да, хезмәткәрләр дә мөрәҗәгатьләргә һәм җавапларга файллар беркетә ала. Иминлек өчен рөхсәт ителгән файл төрләрен һәм зурлык чикләрен тулысынча контрольдә тотасыз.

## Хәбәрнамәләр ничек эшкәртелә?

Система яңа мөрәҗәгатьләр, җаваплар һәм статус үзгәрешләре өчен email хәбәрнамәләрен автоматик рәвештә җибәрә. Барлык email шаблоннарын көйләүләрдә үзгәртә аласыз.

## Мөрәҗәгатьләрне билгеле ярдәм агентларына билгели аламмы?

Әйе, сез мөрәҗәгатьләрне агентларга кулдан билгели яки мөрәҗәгатьләрне автоматик рәвештә бүлү өчен автоматик билгеләүне кабыза аласыз.

## Хезмәткәрләр өчен шәхси искәрмәләр бармы?

Әйе, хезмәткәрләр клиентларга түгел, бары тик башка хезмәткәрләргә генә күренә торган шәхси җаваплар өсти ала.

## Ул мобиль җайланмалар өчен уңайлымы?

Әлбәттә! Бөтен интерфейс мобиль җайланмаларга өстенлек бирелгән алым белән төзелгән һәм барлык җайланмаларда бик матур эшли.

## Мөрәҗәгать исемлекләрен һәм формаларын шорткодлар ярдәмендә күрсәтә аламмы?

Әйе! Мөмкин булган шорткодлар:
* `[wu_support_tickets]` - Клиент мөрәҗәгатьләре исемлеген күрсәтү
* `[wu_support_ticket_form]` - Яңа мөрәҗәгать җибәрү формасын күрсәтү
* `[wu_support_ticket_view ticket_id="123"]` - Билгеле мөрәҗәгатьне күрсәтү

## Нинди файл төрләрен йөкләп була?

Рөхсәт ителгән файл төрләрен көйләүләрдә конфигурацияли аласыз. Система зарарлы файллардан саклау өчен алдынгы иминлек сканлавын үз эченә ала.

## Мөрәҗәгать мәгълүматларын экспортлый аламмы?

Әйе, өстәмә мөрәҗәгать мәгълүматлары һәм статистика өчен экспорт мөмкинлекләрен үз эченә ала.
