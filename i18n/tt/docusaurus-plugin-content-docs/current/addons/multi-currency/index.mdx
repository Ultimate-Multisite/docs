# Multi-Currency

Ultimate Multisite: Multi-Currency челтәрегезгә берничә валютада түләүләр кабул итәргә мөмкинлек бирә. Ул тере яки кул белән куелган алмашу курслары ярдәмендә бәяләрне автоматик рәвештә конвертацияли, теләсә кайсы валюта өчен һәр продуктка төгәл бәяләр куярга мөмкинлек бирә, һәм геолокация аша һәр керүченең өстенлекле валютасын ачыклый — боларның барысы да бердәм админ Dashboardтан идарә ителә.

## Төп мөмкинлекләр {#key-features}

- **Алмашу курсы буенча автоматик конвертация** — бәяләр сез сайлаган провайдердан алынган курслар ярдәмендә төп валютагыздан конвертацияләнә
- **Һәр продукт өчен бәяләрне үзгәртеп кую** — автоматик конвертацияне урап узу өчен теләсә кайсы продуктта валютага хас төгәл бәяләр куегыз
- **Теркәлү өчен валюта сайлау кыры** — клиентларга checkout/теркәлү вакытында өстенлекле валюталарын сайларга мөмкинлек бирә
- **Берничә алмашу курсы провайдеры** — ECB, Frankfurter, Stripe яки тулысынча кул белән куелган курслар
- **Геолокациягә нигезләнгән валюта ачыклау** — керүченең иленә туры килгән валютаны автоматик рәвештә алдан сайлый
- **Админ Dashboard** — актив валюталарны, алмашу курсларын һәм провайдер көйләүләрен бер урыннан идарә итегез

## Таләпләр {#requirements}

- Ultimate Multisite 2.0.0 яки югарырак
- WordPress Multisite урнаштыруы
- PHP 7.4 яки югарырак

## Урнаштыру {#installation}

1. Плагин файлларын `/wp-content/plugins/ultimate-multisite-multi-currency` эченә йөкләгез
2. WordPressтагы **Плагиннар** экраны аша плагинны активлаштырыгыз (Network Activate)
3. Админ Dashboardны ачу өчен **Ultimate Multisite → Multi-Currency** бүлегенә күчегез
4. Кабул итәргә теләгән валюталарны өстәгез һәм алмашу курсы провайдерыгызны көйләгез

## Көйләү {#configuration}

### Админ Dashboard {#admin-dashboard}

Multi-Currency админ Dashboard (**Ultimate Multisite → Multi-Currency**) — түбәндәгеләр өчен үзәк урын:

- Валюталар өстәү, кабызу яки сүндерү
- Алмашу курсы провайдерын сайлау һәм курсларны таләп буенча яңарту
- Барлык актив валюталар өчен хәзерге курсларны карау
- Автоматик провайдерлар туры килмәгәндә кул белән курслар кую

### Алмашу курсы провайдерлары {#exchange-rate-providers}

| Провайдер | Тасвирлама |
|---|---|
| **ECB** | Европа Үзәк банкы — EUR нигезендәге челтәрләр өчен бушлай көндәлек курслар |
| **Frankfurter** | Ачык чыганаклы ECB көзге API — бушлай, API ачкычы кирәкми |
| **Stripe** | Stripeның тере FX курсларын куллана — актив Stripe интеграциясе кирәк |
| **Manual** | Курсларны турыдан-туры үзегез кертәсез; тышкы API чакырулары юк |

Провайдерыгызны **Ultimate Multisite → Multi-Currency → Settings → Exchange Rate Provider** астында сайлагыз. Курсларны теләсә кайсы вакытта кул белән яңартырга яки автоматик яңартуга расписание куярга мөмкин.

### Һәр продукт өчен бәяләрне үзгәртеп кую {#per-product-pricing-overrides}

Автоматик конвертация җитәрлек төгәл булмаганда, теләсә кайсы продуктта билгеле бер валюта өчен төгәл бәя куя аласыз:

1. Продукт мөхәррирен ачыгыз (**Ultimate Multisite → Products → Edit**)
2. **Currency Pricing** бүлегенә кадәр аска төшегез
3. Үзгәртеп куярга теләгән һәр валюта өчен төгәл бәяне кертегез
4. Продуктны саклагыз

Үзгәртеп кую булмаган продуктлар төп валютагыздан конвертацияләнгән бәяне куллана.

### Теркәлү өчен валюта сайлау кыры {#currency-selector-signup-field}

Валюта сайлау кырын signup/checkout агымына өстәргә мөмкин, шулай итеп клиентлар теркәлгәндә өстенлекле валюталарын сайлый:

1. **Ultimate Multisite → Multi-Currency → Settings** бүлегенә барыгыз
2. **Show currency selector on signup** кабызыгыз
3. Ихтыяри рәвештә килешенгән валютаны куегыз яки алдан сайланган кыйммәтне геолокация билгеләсен

Сайланган валюта язылу белән саклана һәм барлык алдагы хисаплаулар өчен кулланыла.

### Геолокациягә нигезләнгән валюта ачыклау {#geolocation-based-currency-detection}

Геолокация буенча ачыклау кабызылганда, addon керүченең илен валютага туры китерә һәм аны валюта сайлау кырында алдан сайлый. Кабызу өчен:

1. **Ultimate Multisite → Multi-Currency → Settings** бүлегенә барыгыз
2. **Detect currency from visitor location** кабызыгыз
3. **Geolocation Mapping** таблицасында илләрне валюталарга туры китерегез

Керүченең иле өчен туры китерү булмаса, fallback буларак челтәрнең төп валютасы кулланыла.

## Еш бирелә торган сораулар {#frequently-asked-questions}

### Алмашу курсы провайдеры мөмкин булмаса, нәрсә була? {#what-happens-if-an-exchange-rate-provider-is-unavailable}

Addon соңгы уңышлы алынган курсларга күчә. Әгәр курслар беркайчан да алынмаган булса, ул сез көйләгән кул белән куелган курсларга күчә. Иминлек чарасы буларак кул белән курсларны көйләү тәкъдим ителә.

### Төрле валюталар өчен төрле провайдерлар куллана аламмы? {#can-i-use-different-providers-for-different-currencies}

Юк — барлык валюталар өчен берьюлы бер провайдер актив була. Аерым курслар өстеннән тулы контроль кирәк булса, **Manual** кулланыгыз.

### Геолокация өчен аерым плагин кирәкме? {#does-geolocation-require-a-separate-plugin}

Тышкы геолокация плагины кирәк түгел. Addon WordPressның эчке IP-to-country ачыклавын куллана. Югарырак төгәллек өчен махсус геолокация хезмәтен бирелгән filter hookлар аша интеграцияләргә мөмкин.

### Һәр продукт өчен үзгәртеп куюлар салымга кадәрме яки салымнан соңмы кулланыла? {#are-per-product-overrides-applied-before-or-after-tax}

Һәр продукт өчен үзгәртеп куюлар максат валютада салымга кадәрге бәяне билгели. Салым сезнең гамәлдәге салым көйләүләрегез ярдәмендә үзгәртеп куелган бәя өстенә исәпләнә.
