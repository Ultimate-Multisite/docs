# Фильтр: wp_ultimo_woocommerce_checkout_fields_to_clean

Эшкәртүчеләргә WooCommerce хисап адресының стандарт кырларын фильтрларга мөмкинлек бирә.

Ultimate Multisite хисап адресы кырларына ия булганга, WooCommerce тарафыннан checkout формасында күрсәтелә торган стандарт кырларны бетерү мәгънәле булырга мөмкин. Килешү буенча, бу исемлеккә бернинди кыйммәт тапшырылмый, шуңа күрә WooCommerce’та күрсәтелергә көйләнгән барлык кырлар саклана.

### Кайчаннан

- 2.0.0
### Чыганак

[`inc/gateways/class-woocommerce-gateway.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-woocommerce/blob/main/inc/gateways/class-woocommerce-gateway.php#L277) эчендә 277 нче юлда билгеләнгән


## Кайтара
Бетереләчәк кырлар исемлеге. Мөмкин кыйммәтләр: billing_first_name, billing_last_name, billing_company, billing_country, billing_address_1, billing_address_2, billing_city, billing_state, billing_postcode, billing_email һәм billing_phone.
