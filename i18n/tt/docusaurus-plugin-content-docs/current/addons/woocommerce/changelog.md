# WooCommerce интеграциясе үзгәрешләр журналы {#woocommerce-integration-changelog}

Version 2.2.0 - 2026-07-01 чыгарылды
* Яңа: Ultimate Multisite салым суммалары хәзер түләү битендә аерым WooCommerce җыем юллары булып күренә, бу түләүгә кадәр салым нәтиҗәләрен аңлаешлырак итә.
* Яңа: WooCommerce Subscriptions кабатлап карау тәрәзәсе вакытында уңышсыз яңарту аркасында туктатып торуны теләгән сайтлар өчен "Уңышсыз яңартуларда әгъзалыкларны шунда ук туктатып тору" көйләве һәм `wu_woo_suspend_on_payment_failure` фильтры өстәлде.
* Төзәтелде: Уңышсыз яки торгызылган яңартулардан соң Ultimate Multisite әгъзалыклары белән синхрон булмый кала алган WooCommerce Subscription статуслары килештерелде.
* Төзәтелде: Җитмәгәндә Ultimate Multisite валюта исемлегенә WooCommerce кибет валютасы өстәлде.
* Төзәтелде: Язылучыларны WooCommerce түләү битенә юнәлткәндә клиентның исәп-хисап мәгълүматлары сакланды.
* Яхшыртылды: Jetpack Autoloader 5 белән ярашлылык өстәлде.
* Яхшыртылды: Чыгарылыш пакетын төзү чистартылды, шуңа күрә GitHub һәм marketplace zip-лары эчке әзерләү каталогларын һәм эшкәртү файлларын кертми.
