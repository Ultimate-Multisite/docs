# Multi-Network

Ultimate Multisite: Multinetwork өстәмәсе сезнең Ultimate Multisite платформагызны көчле күпчелтәр функцияләре белән киңәйтә. Бу өстәмә клиентларыгызга аерым сайтлар гына түгел, ә тулы WordPress мультисайт челтәрләрен сатып алырга һәм идарә итәргә мөмкинлек бирә.

## Төп мөмкинлекләр

* **Челтәр булдыру**: Клиентларыгыз өчен тулы WordPress мультисайт челтәрләрен автоматик рәвештә булдырыгыз һәм көйләгез
* **Челтәр шаблоннары**: Клиентларга теркәлү алдыннан алдан әзерләнгән челтәр шаблонын карарга, алдан күрергә һәм сайларга мөмкинлек бирегез
* **Челтәр белән идарә итү**: Күпчелтәр мохитләре өчен тулы идарә итү мөмкинлекләре
* **Аерымланган челтәрләр**: Һәр клиент челтәре үз мәгълүмат базасы таблицалары һәм конфигурациясе белән мөстәкыйль эшли
* **Челтәрләр арасындагы SSO**: Клиентларга magic-link аутентификациясе белән үз суб-челтәрләренә өзлексез керү мөмкинлеге бирегез
* **Масштаблана торган архитектура**: Оптимальләштерелгән ресурс бүлеп бирү ярдәмендә берничә челтәрне нәтиҗәле эшкәртегез
* **Интеграция**: Ultimate Multisite-ның язылу һәм исәп-хисап системалары белән өзлексез интеграцияләнә

## Куллану очраклары

* **Ак билгеле хостинг**: Клиентларга үз брендлары белән мультисайт челтәрләрен тәкъдим итегез
* **Агентлык чишелешләре**: Агентлыкларга бер платформа аша берничә клиент челтәрен идарә итәргә мөмкинлек бирегез
* **Корпоратив урнаштырулар**: Катлаулы күпчелтәр таләпләре булган зур оешмаларга ярдәм итегез
* **Мәгариф платформалары**: Төрле мәктәпләр яки бүлекләр өчен аерым челтәрләр булдырыгыз

## Таләпләр

* Ultimate Multisite 2.0.0 яки югарырак
* WordPress Multisite урнаштырылган булуы
* PHP 7.4 яки югарырак
* MySQL 5.6 яки югарырак

## Урнаштыру

1. Плагин файлларын `/wp-content/plugins/ultimate-multisite-multinetwork` директориясенә йөкләгез
2. Плагинны WordPress эчендәге 'Plugins' экраны аша активлаштырыгыз
3. Күпчелтәр көйләүләрен Ultimate Multisite → Settings аша көйләгез

## Челтәр шаблоннары

1.3.0 версиясе яңа челтәрләр сатып алучы клиентлар өчен шаблоннарны табу һәм сайлау агымын өсти. Һәр яңа клиент челтәре әзерләнгән структурадан, дизайннан, контент китапханәсеннән яки билгеле бер тармакка көйләнгән конфигурациядән башлансын дисәгез, челтәр шаблоннарын кулланыгыз.

### Шаблонны алдан карау

Челтәр шаблоннарын алдан карау коралы клиентларга түләүгә кадәр мөмкин булган шаблоннарны карарга мөмкинлек бирә. Һәр шаблон тере алдан карау панелендә ачыла ала, шуңа күрә клиентлар челтәр сатып алганчы макетларны һәм контентны чагыштыра ала.

Алдан карау коралын клиентларга түбәндәгеләрдә ярдәм итү өчен кулланыгыз:

* Һәр мөмкин булган челтәр шаблонының тышкы күренешен һәм структурасын тикшерү
* Сатып алу агымыннан чыкмыйча берничә шаблонны чагыштыру
* Теркәлү алдыннан аларның куллану очрагына иң туры килгән шаблонны сайлау

Шаблон исемнәрен һәм тасвирламаларын клиентлар өчен аңлаешлы итеп саклагыз. Алдан карау коралы клиентлар максатчан куллану очрагын аңлый алганда иң файдалы, мәсәлән, агентлык челтәрләре, мәктәп челтәрләре, франшиза челтәрләре яки ак билгеле хостинг җыелмалары.

### Түләү вакытында шаблон сайлагыч

Шаблон сайлагычны теркәлү түләү агымына кыр буларак өстәргә мөмкин. Көйләнгән очракта, клиентлар түләү вакытында үзләре теләгән челтәр шаблонын сайлый, һәм сайланган шаблон аларның яңа челтәре булдырылганда кулланыла.

Сайлагычны түләү агымында куллану өчен:

1. Шаблоннар буларак тәкъдим итәргә теләгән челтәрләрне әзерләгез.
2. Тиешле теркәлү яки түләү формасында Multinetwork шаблон сайлау кырын кабызыгыз.
3. Бу түләү агымы өчен кайсы шаблоннар мөмкин булырга тиешлеген сайлагыз.
4. Түләү агымын бастырыгыз һәм сайланган шаблонның яңа челтәргә кулланылуын раслау өчен теркәлүне сынап карагыз.

Әгәр түләү агымы бер генә челтәр продукты тәкъдим итсә, сайлагыч барыбер юнәлешле башлангыч көйләү адымы буларак файдалы булырга мөмкин. Күп планлы воронкалар өчен сайланган продукт яки план белән ярашлы шаблоннарны гына күрсәтегез.

## Челтәрләр арасындагы тылсымлы сылтама SSO

Челтәрләр арасындагы тылсымлы сылтама SSO клиентларга үзләренең төп Account өлкәсе һәм суб-челтәрләре арасында кабат кулдан кермичә күчәргә мөмкинлек бирә. Клиент челтәрләр арасындагы керү сылтамасына ияргәндә, Ultimate Multisite вакыт белән чикләнгән тылсымлы сылтама булдыра һәм клиентны максат челтәренә аутентификацияли.

Бу бердән артык челтәр белән идарә итүче яки сатып алганнан соң яңа клонланган шаблон челтәрләренә шунда ук керергә кирәк булган клиентлар өчен тәҗрибәне яхшырта.

Администратор тикшерү исемлеге:

* Multinetwork өстәмәсенең WordPress мультисайт урнаштыруында актив булуын раслагыз.
* Клиентларга челтәрләр арасындагы сылтамалар тәкъдим ителгәнче, аларның дөрес челтәргә керү хокуклары булуына инаныгыз.
* Тылсымлы сылтамалар ниятләнгән максат челтәренә барып җитсен өчен, челтәр доменнарын һәм бәйләнгән доменнарны дөрес көйләнгән килеш саклагыз.
* Челтәр, домен яки түләү көйләүләрен үзгәрткәннән соң клиент Dashboard-ын һәм түләүдән соңгы керү төймәләрен сынап карагыз.

Тылсымлы сылтамаларга керү сылтамалары итеп карарга кирәк. Алар клиентларның өзлексез керүе өчен эшләнгән һәм ниятләнгән сессиядән читтә җибәрелергә яки яңадан кулланылырга тиеш түгел.

## Еш бирелә торган сораулар

### Күпчелтәр нәрсә ул?

Күпчелтәр — бер WordPress урнаштыруында берничә мөстәкыйль мультисайт челтәрен эшләтергә мөмкинлек бирә торган WordPress мөмкинлеге.

### Бу гадәти мультисайттан ничек аерыла?

Гадәти мультисайт бер челтәр эчендә берничә сайтка мөмкинлек бирә. Күпчелтәр бер WordPress урнаштыруы эчендә һәркайсының үз сайтлары булган берничә челтәргә мөмкинлек бирә.

### Моның өчен махсус сервер конфигурациясе кирәкме?

Әйе, күпчелтәр махсус сервер көйләве һәм конфигурациясе таләп итә. Җентекле көйләү күрсәтмәләре өчен документацияне карагыз.
