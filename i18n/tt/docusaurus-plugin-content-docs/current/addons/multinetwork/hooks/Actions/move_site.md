# Гамәл: move_site {#action-movesite}

Сайт яңа челтәргә күчерелгәннән соң эшли.

## Параметрлар {#parameters}

| Исем | Төр | Тасвирлама |
|------|------|-------------|
| $site_id | `int` | Күчерелгән сайтның ID-сы. |
| $old_network_id | `int` | Сайт өчен баштагы челтәрнең ID-сы. |
| $new_network_id | `int` | Сайт күчерелгән челтәрнең ID-сы. |

- 1.3.0
