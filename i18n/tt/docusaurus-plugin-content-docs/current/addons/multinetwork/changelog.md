# Күп-челтәр үзгәрешләр журналы

### 1.3.0
* Яңа: Челтәр шаблонын алдан карау — сатып алганчы, челтәр шаблоннарын тере панельдә карап чыгыгыз һәм алдан карагыз.
* Яңа: Түләү вакытында челтәр шаблонын сайлагыч — клиентлар теркәлү түләү агымында челтәр шаблонын сайлый ала.
* Яңа: Челтәрләр арасында magic link SSO — клиентлар magic-link аутентификациясе аша суб-челтәрләргә өзлексез керә ала.
* Яңа: Челтәр медиафайлларын клонлау — шаблоннан челтәрләрне клонлаганда, сайт медиафайллары дөрес күчереп ясала.
* Төзәтү: Челтәр клонлау хәзер multi-tenancy өстәмәсеннән башка да эшли; super admin хокуклары клонлаудан соң яңадан бирелә.
* Төзәтү: Челтәр булдыру хәзер WooCommerce һәм барлык түләү шлюзлары белән ярашлы.
* Төзәтү: Фолбэк домен булдыруга хәзер порт номеры кертелә.
* Төзәтү: Шаблон сайлагыч CSS-ы барлык темаларда дөрес күрсәтү өчен enqueued stylesheet-ка күчерелде.
* Төзәтү: Сайт соравы өлкәсендә билгеләнмәгән `network_id` ачкычыннан саклау.
* Төзәтү: Контекстны алыштырганчы челтәр рөхсәтен тикшерү `wpdb` таблица үзлеге хаталарын булдырмый.
