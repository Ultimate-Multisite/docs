# AffiliateWP {#affiliatewp}

Multisite Ultimate: AffiliateWP интеграциясе

Ultimo Network клиентлар базасын үстерү өчен көчле AffiliateWP кулланыгыз!

## Урнаштыру {#installation}

1. 'ultimate-multisite-git ' ны '/wp-content/plugins/' директориясенә йөкләгез
2. Плагинны WordPress эчендәге 'Плагиннар' менюсы аша активлаштырыгыз
3. Плагинны көйләү өчен адымлап баручы Көйләү остасына иярегез

## Куллану кулланмасы {#usage-guide}

[AffiliateWP](https://affiliatewp.com/) онлайн бизнесыгыз өчен affiliate программасы төзегәндә төп чишелешкә әйләнде. Ultimate Multisite: AffiliateWP — premium челтәрегез өчен шул ук нәрсәне төзергә мөмкинлек бирәчәк күпер.

## Башлау {#getting-started}

Иң элек, сездә [**AffiliateWP**](https://affiliatewp.com/) **актив булырга тиеш**! Әгәр сездә **аның белән интеграцияләнә торган плагин** булмаса, безнең өстәмәне куллана алмаячаксыз. Клиентларыгызга аны үз сайтларында да кулланырга рөхсәт итәргә җыенмасагыз, AffiliateWP бары тик төп сайтыгызда гына актив булырга тиеш.

Шулай ук, сезгә **Ultimate Multisite** та, **Ultimate Multisite: AffiliateWP** та **бөтен челтәр буенча** **активлаштырылган** булырга кирәк.

## Ultimate Multisite ны AffiliateWP интеграциясе итеп өстәү {#adding-ultimate-multisite-as-an-affiliatewp-integration}

Төп сайтыгызда Affiliates Settings бүлегенә керегез, һәм Integrations салынмасында Ultimate Multisite ны билгеләгез. Бу [AffiliateWP](https://affiliatewp.com/) ка Ultimate Multisite теркәлү агымыннан килгән керүләрне һәм конверсияләрне күзәтә башларга кушачак.

<!-- Скриншот юк: Ultimate Multisite билгеләү шакмагы булган AffiliateWP Integrations көйләүләре салынмасы -->

_Астагы Ultimate Multisite шакмагын билгеләгез һәм саклагыз_

Моның өчен кирәк булганнарның барысы да шул диярлек. Referral сылтамаларыннан килгән керүләр күзәтеләчәк, һәм ул кулланучы соңрак беренче түләвен ясаганда, конверсия булдырылачак һәм конверсия өчен җаваплы affiliate өчен комиссия аерып куелачак.

Әмма яңа булдырылган кулланучыдан алынган беренче түләү генә күзәтеләчәк. Кабатланучы referral түләүләрен кушу өчен, сезгә [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) кирәк булачак

## Кабатланучы Referralларны кушу {#enabling-recurring-referrals}

[AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) урнаштырганнан соң, Affiliates Settings Recurring Referrals салынмасына керегез һәм кабатланучы referralлар куллануны кушыгыз.

<!-- Скриншот юк: Enable Recurring Referrals күчергече булган AffiliateWP Recurring Referrals көйләүләре -->

_Бик гади: Enable Recurring Referrals ны кушыгыз — һәм барысы да әзер!_

Тагын бер тапкыр, моның өчен кирәк булганнарның барысы да шул!

Хәзер, юллама аша килгән аккаунтта яңа түләү алынган саен, җаваплы affiliate үз өлешен автоматик рәвештә алачак!
