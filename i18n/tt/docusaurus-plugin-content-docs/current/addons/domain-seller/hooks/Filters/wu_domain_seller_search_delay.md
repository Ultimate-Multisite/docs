# Фильтр: wu_domain_seller_search_delay

Домен эзләү debounce тоткарлыгын миллисекундларда фильтрлый.

Акрын тоташуларда API чакыруларын киметү өчен бу кыйммәтне арттырыгыз.

## Параметрлар {#parameters}

| Исем | Төр | Тасвирлама |
|------|------|-------------|
| $delay | `int` | Debounce тоткарлыгы миллисекундларда. Килешенгән 500. |
