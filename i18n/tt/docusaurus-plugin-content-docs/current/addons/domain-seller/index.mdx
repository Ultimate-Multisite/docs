# Domain Seller

**Multisite Ultimate: Domain Seller** сезнең WordPress Multisite челтәрен көчле домен сату платформасына әйләндерә. Клиентларга теркәлү вакытында яки үз Dashboard аша махсус доменнарны турыдан-туры теркәргә мөмкинлек бирегез, реаль вакытта буш булуын тикшерү һәм ачык бәяләү белән.

### Ни өчен Domain Seller? {#why-domain-seller}

Домен теркәү — һәр WaaS (Website as a Service) платформасы өчен иң күп сорала торган өстәмә сатуларның берсе. Клиентларыгызны өченче як теркәүчегә җибәрү урынына, сез доменнарны турыдан-туры тәкъдим итә аласыз — челтәрегездә күбрәк керем калдырып, клиентларга тоткарлыксыз тәҗрибә бирәсез.

### Төп мөмкинлекләр {#key-features}

#### Берничә домен теркәүчене хуплау {#multiple-domain-registrar-support}
* **OpenSRS** — Тулы теркәү, озайту һәм идарә итү
* **Namecheap** — WhoisGuard һәм sandbox сынавы белән тулы интеграция
* **HostAfrica** — Domains Reseller API аша домен эзләү, бәяләү, теркәү, күчерү, DNS һәм ID саклау
* **Openprovider** — Реселлер бәяләве, теркәү, озайту, күчерүләр, DNS, WHOIS хосусыйлыгы һәм TLD синхронлау
* **Hostinger** — Уртак Hostinger интеграция token кулланып домен эзләү һәм теркәү
* **Киңәйтелә торган архитектура** — Integration Registry үрнәгенә нигезләнгән, күбрәк тәэмин итүчеләр өстәүне җиңел итә
* **Эчке тоташуны сынау** — Эшләтеп җибәргәнче тәэмин итүче таныклыкларын һәм тоташуны тикшерү

#### Юнәлешле көйләү остасы {#guided-setup-wizard}
* **Бер басу белән башлау** — Сезне тәэмин итүче көйләве, TLD импортлау һәм домен продукты булдыру аша алып бара
* **Автоматик TLD синхронлау** — Тоташтырылган тәэмин итүчеләрдән барлык мөмкин TLDларны һәм күпләп сату бәяләрен импортлый
* **Килешү буенча продукт булдыру** — Акыллы өстәмә бәя килешүләре белән кулланырга әзер домен продукты булдыра

#### Сыгылмалы домен бәяләве {#flexible-domain-pricing}
* **Өч өстәмә бәя режимы** — Процентлы өстәмә бәя, беркетелгән өстәмә бәя яки һәр TLD өчен беркетелгән бәя
* **TLD-махсус продуктлар** — Төрле TLD төркемнәре өчен аерым домен продуктлары булдырыгыз (.com/.net, .io, ил кодлары һ.б.)
* **Кереш бәяләү** — Яңа теркәүләр җәлеп итү өчен ташламалы беренче ел бәясен тәкъдим итегез
* **Реаль вакыт бәяләве** — Акыллы кэшлау белән тәэмин итүчеләрдән туры күпләп сату бәяләре
* **Купоннарны хуплау** — Multisite Ultimate ташлама кодлары белән тулы ярашлылык

#### WHOIS хосусыйлык саклавы {#whois-privacy-protection}
* **Һәр продукт өчен көйләү** — WHOIS хосусыйлыгын Сүндерелгән, Һәрвакыт кертелгән яки Клиент сайлавы итеп билгеләгез
* **Клиентның ризалык белән кушылуы** — Клиент сайлавы итеп куелганда, checkout флажогы клиентларга хосусыйлык саклавын өстәргә мөмкинлек бирә
* **Көйләнә торган бәяләү** — Хосусыйлык өстәмәсе өчен еллык бәя билгеләгез яки аны бушлай тәкъдим итегез
* **Теркәүче интеграциясе** — Теркәү вакытында WhoisGuard (Namecheap) яки WHOIS хосусыйлыгын (OpenSRS) автоматик рәвештә эшләтә

#### Бердәм checkout тәҗрибәсе {#unified-checkout-experience}
* **Өч домен режимы** — Клиентлар субдомен (бушлай), яңа домен теркәү яки инде үзләрендә булган гамәлдәге доменны бәйләү арасында сайлый
* **Реаль вакыт домен эзләве** — Битне яңартмыйча AJAX ярдәмендә буш булуын тикшерү
* **Акыллы тәкъдимнәр** — Теләнгән домен алынган булса, альтернатив TLD тәкъдимнәре
* **Ачык бәяләү** — Теркәү бәясен, озайту бәясен һәм өстәмә WHOIS хосусыйлык түләвен ачык күрсәтү
* **Теркәүче контакт кырлары** — Checkout вакытында кирәкле теркәүче мәгълүматын (исем, адрес, телефон) юл эчендә җыя
* **Автоматик ясалган сайт URL** — Сайт адресы сайланган доменнан автоматик рәвештә чыгарыла

#### Авто-озайту системасы {#auto-renewal-system}
* **Membership белән бәйләнгән озайтулар** — Клиентның membership актив булганда доменнар автоматик рәвештә озайтыла
* **Акыллы планлаштыру** — Озайту эшләре доменның гамәлдән чыгу даталарына нигезләнеп чиратка куела
* **Кабатлау логикасы** — Уңышсыз озайту омтылышлары өчен акыллы кабатлаулар
* **Email хәбәрнамәләре** — Киләчәк озайтулар, уңышлы озайтулар, уңышсызлыклар һәм гамәлдән чыгу турында көйләнә торган кисәтүләр

#### Клиент DNS идарәсе {#customer-dns-management}
* **Үз-үзеңә хезмәт күрсәтү DNS** — Клиентлар үз Dashboard аша DNS язмаларын (A, AAAA, CNAME, MX, TXT) өсти, үзгәртә һәм бетерә ала
* **Домен статусы** — Реаль вакыттагы теркәү статусы, гамәлдән чыгу даталары һәм nameserver мәгълүматы
* **Автоматик домен бәйләү** — Теркәлгән яки гамәлдәге доменнар клиент сайтына автоматик рәвештә бәйләнә

#### Admin кораллары {#admin-tools}
* **Кулдан домен теркәү** — Admin панеленнән клиентлар исеменнән доменнар теркәгез
* **Домен күзәтүе** — Барлык клиент доменнарын, статусларын һәм гамәлдән чыгу даталарын бер биттән карагыз
* **Тәэмин итүчегә махсус логлау** — Һәр теркәүче өчен аерым лог каналлары (мәс., domain-seller-namecheap.log)
* **Теркәү нечкәлекләре** — Төп домен үзгәртү битендә теркәүче мәгълүматын һәм теркәү мәгълүматларын карагыз

### Хуплана торган тәэмин итүчеләр {#supported-providers}

* **OpenSRS** — Теркәү, озайту, DNS идарәсе һәм WHOIS хосусыйлыгы белән тулы интеграция
* **Namecheap** — Теркәү, озайту, пакетлы буш булуын тикшерү һәм бушлай WhoisGuard белән тулы интеграция
* **HostAfrica** — Теркәү, озайту, күчерүләр, DNS идарәсе, EPP кодын алу, registrar lock һәм ID саклау
* **Openprovider** — Теркәү, озайту, күчерүләр, DNS идарәсе, EPP кодын алу, registrar lock, WHOIS хосусыйлыгы һәм реселлер TLD синхронлау
* **Hostinger** — Теркәү, домен эзләү, nameserver яңартулары, registrar lock һәм WHOIS хосусыйлыгы; озайтулар Hostinger auto-renew тарафыннан башкарыла
* **GoDaddy** — Теркәү һәм озайту
* **ResellerClub** — Теркәү, озайту һәм DNS идарәсе
* **NameSilo** — Теркәү һәм озайту
* **Enom** — Теркәү, озайту һәм DNS идарәсе

### Техник нечкәлекләр {#technical-details}

* **Таләп итә:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **Архитектура:** Domain_Selling_Capability интерфейсы белән Integration Registry үрнәге
* **Autoloading:** PSR-4 класс картасы белән Jetpack autoloader
* **Стандартлар:** WordPress кодлау стандартлары, катгый типлаштыру, әзерләнгән statementлар

## Урнаштыру {#installation}

1. **Multisite Ultimate v2.4.12+** урнаштырылган һәм челтәрдә активлаштырылган булуын тәэмин итегез
2. Плагинны `/wp-content/plugins/ultimate-multisite-domain-seller/` адресына йөкләгез
3. WordPress Network Admin эчендәге **Плагиннар** бүлегеннән плагинны челтәрдә активлаштырыгыз
4. **Көйләү остасы** беренче активлаштыруда автоматик рәвештә эшли башлый — түбәндәге адымнарны үтәгез:
   * Домен теркәүчегезне тоташтырыгыз
   * Тоташуны сынагыз
   * Мөмкин булган TLDларны һәм күпләп сату бәяләрен импортлагыз
   * Бәя өстәмәсе белән беренче домен продуктыгызны булдырыгыз
5. **Ultimate Multisite > Рәсмиләштерү формалары** эчендәге рәсмиләштерү формагызга **Домен сайлау** кырын өстәгез

## Еш бирелә торган сораулар {#frequently-asked-questions}

### Multisite Ultimateның нинди версиясе кирәк? {#what-version-of-multisite-ultimate-is-required}

2.4.12 яки югарырак версия. Бу өстәмә WP Ultimo v1.x яки Ultimate Multisiteның 2.4.12дән түбән версияләре белән ярашмый.

### Кайсы домен тәэмин итүчеләре хуплана? {#which-domain-providers-are-supported}

Тугыз теркәүче хуплана: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo** һәм **Enom**. Мөмкинлекләр тәэмин итүчегә карап аерыла, чөнки теркәүче APIлары төрле операцияләрне ача; тәэмин итүче буенча күзәтү өчен көйләү кулланмасын карагыз. Өстәмәнең Integration Registry архитектурасы махсус тәэмин итүчеләр өстәүне җиңел итә — [эшләүче кулланмасын](/developer/integration-guide/custom-registrar) карагыз.

### Мин WHOIS хосусыйлык саклавын тәкъдим итә аламмы? {#can-i-offer-whois-privacy-protection}

Әйе. Һәр домен продуктының өч режимлы WHOIS хосусыйлык көйләнеше бар: Сүндерелгән, Һәрвакыт кертелгән яки Клиент сайлавы. Клиент сайлавы итеп куелганда, клиентлар рәсмиләштерү вакытында бәясе күрсәтелгән тамгалау шакмагын күрә (яки Namecheap WhoisGuard өчен «Бушлай»). Хосусыйлык билгесе теркәү вакытында автоматик рәвештә теркәүчегә тапшырыла.

### Мин төрле TLDлар өчен төрле бәяләр куя аламмы? {#can-i-set-different-pricing-for-different-tlds}

Әйе. Төрле TLD төркемнәре өчен аерым домен продуктлары булдырыгыз. Һәр продуктның үз өстәмә төре (процент, билгеләнгән өстәмә яки билгеләнгән бәя), хупланган TLDлары һәм кереш бәяләре була ала.

### Клиентлар нинди рәсмиләштерү вариантларын күрә? {#what-checkout-options-do-customers-see}

Домен сайлау кыры клиентларга өч кыстыргыч бирә: **Субдомен** (челтәрегездә бушлай субдомен куллану), **Яңа домен теркәү** (яңа домен эзләү һәм теркәү) һәм **Булган домен** (алар инде ия булган доменны бәйләү). Сез кабызган режимнар гына күренәчәк.

### Ул автоматик озайтуларны хуплыймы? {#does-it-support-automatic-renewals}

Әйе. Доменнарны автоматик озайтуга көйләп була, һәм алар клиентның әгъзалык статусы белән бәйләнгән. Озайту түләүләре клиентның гамәлдәге түләү шлюзы аша эшкәртелә.

### Клиентлар үз DNS язмаларын идарә итә аламы? {#can-customers-manage-their-dns-records}

Әйе. Клиентлар A, AAAA, CNAME, MX һәм TXT язмаларын өстәү, үзгәртү һәм бетерү өчен үз Dashboardларында DNS идарә интерфейсын ала. DNS идарәсе OpenSRS, ResellerClub, Enom, HostAfrica һәм Openprovider белән кулланыла ала. Хостингтагы доменнар өчен Hostinger DNS төп Hostinger домен-бәйләү интеграциясе тарафыннан эшкәртелә.

### Көйләү остасы бармы? {#is-there-a-setup-wizard}

Әйе. Көйләү остасы беренче активлаштыруда автоматик рәвештә эшли һәм сезне теркәүчене тоташтыру, TLDларны импортлау һәм беренче домен продуктыгызны булдыру аша алып бара. Сез шулай ук остазны өстәмә көйләүләреннән яңадан эшләтә аласыз.
