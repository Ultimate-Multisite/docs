# VAT

Multisite Ultimate: Европа VAT

Multisite Ultimate'ның килешү буенча салым эшкәртүен Европа VAT өчен яхшырак тәэмин итү өчен өстәмә мөмкинлекләр һәм кораллар өсти.
EU VAT ставкаларын Салым ставкалары битенә автоматик рәвештә өсти

## Урнаштыру

1. 'ultimate-multisite-vat' ны '/wp-content/plugins/' директориясенә йөкләгез
2. WordPress эчендәге 'Плагиннар' менюсы аша плагинны активлаштырыгыз
3. Ultimate Multisite > Көйләүләр > Салымнар бүлегенә керегез һәм EU VAT ны кабызыгыз.
4. Салым ставкаларына басыгыз, аста EU VAT ставкаларын яңартыгыз һәм салым ставкаларын саклагыз.
5. Әзер!
