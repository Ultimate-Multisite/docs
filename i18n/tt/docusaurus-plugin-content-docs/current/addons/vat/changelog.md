# VAT үзгәрешләр журналы

Версия 1.0.7 - 2026-02-03 көнне чыгарылды

* Төзәтү: German (DE) VAT номерлары VIES SOAP хезмәтендәге MS_MAX_CONCURRENT_REQ хаталары аркасында тикшерүне үтә алмый иде. Ышанычлырак тикшерү өчен иске SOAP API урынына рәсми EU VIES REST API кулланыла башлады.
* Төзәтү: VIES хезмәте вакытлыча эшләмәгәндә, VAT номерлары хәзер ялгыш кире кагылу урынына кабул ителә. Соңрак карап чыгу өчен журнал язмасы булдырыла.
* Төзәтү: VAT/Tax ID кыры хәзер EU әгъзалары өчен генә түгел, барлык илләр өчен дә күренә. EU булмаган салым ID-лары (мәсәлән, Швейцария CHE номерлары) VIES тикшерүеннән башка инвойста күрсәтү өчен саклана.
* Төзәтү: Greece (GR), Monaco (MC) һәм Isle of Man (IM) өчен VAT префиксын эзләү дөрес булмаган массив эзләү логикасы аркасында бозылган иде.
* Яхшырту: Site Exporter өстәмәсе белән бергә кулланылганда фаталь хаталар китереп чыгара алган кирәксез wp-cli-bundle dev бәйлелеге бетерелде.
* Яхшырту: Тулы unit test җыелмасы өстәлде (53 тест).

Версия 1.0.6 - 2026-01-25 көнне чыгарылды

* Төзәтү: Эчке B2B транзакцияләре хәзер VATны дөрес исәпли. EU VAT кагыйдәләре буенча, кире исәпләү клиент иле компания иле белән туры килгәндә түгел, ә бары тик чикара B2B транзакцияләренә генә кагыла.

* Үзгәреш: VAT ставка мәгълүмат чыганагы euvatrates.com сайтыннан ышанычлырак һәм актив алып барыла торган ibericode/vat-rates репозиториена күчерелде.
* Төзәтү: super_reduced_rates көйләмә кыйммәтендәге хәреф хатасы төзәтелде.
* Яхшырту: Хаталарны булдырмау өчен ил коды белән эшләүгә null тикшерүләре өстәлде.
