# Material WP

WordPress идарәче интерфейсы күп еллар буе үзгәрешсез калды. Әмма идарәче интерфейсының Material Design версиясе ничек күренер иде дип уйлаганыгыз бармы? 

Material WP — сезгә шул күренешне һәм хисне бирү өчен эшләнгән идарәче темасы; ул клиентларыгызга күбрәк көйләнгән Dashboard тәкъдим итәргә ярдәм итә.

## Мөмкинлекләр {#features}

- Үз CSS ярдәме
- Җиңел стиль көйләүләре
- Меню тәртибен үзгәртү
- Җиңел импорт/экспорт

## Документация {#documentation}

**Material WP белән идарәче панелегезне көйләү**

## Урнаштыру {#installation}

1. 'material-wp' ны '/wp-content/plugins/' директориясенә йөкләгез
2. Плагинны WordPress эчендәге 'Плагиннар' менюсы аша активлаштырыгыз
