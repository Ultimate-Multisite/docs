# Дәрес 10: Эшләтеп җибәрү көне {#lesson-10-launch-day}

Барысы да төзелде. Ишекләрне ачканчы, бу дәрес эшләтеп җибәрү алдыннан тикшерү исемлеге аша үткәрә, бернәрсә дә ватылмаган, югалмаган яки уңайсыз хәлгә калдырмый торган булуын тәэмин итү өчен.

## Кайда тукталып калган идек {#where-we-left-off}

FitSite-та калыплар, планнар, түләүне рәсмиләштерү, брендлау, беренчел көйләү һәм бәяләү барысы да конфигурацияләнгән. Хәзер без барысының да эшләвен тикшерәбез һәм туры эфирга чыгабыз.

## Эшләтеп җибәрү алдыннан тикшерү исемлеге {#pre-launch-checklist}

Һәр пунктны үтәгез. Берсен дә калдырмагыз.

### Платформа инфраструктурасы {#platform-infrastructure}

- [ ] Хостинг тотрыклы һәм йөкләнеш астында яхшы эшли
- [ ] Wildcard SSL актив һәм барлык субдоменнар HTTPS аша хезмәт күрсәтә
- [ ] Домен бәйләү эшли -- сайт ясап һәм махсус домен бәйләп сынагыз
- [ ] Резерв күчермәләр конфигурацияләнгән һәм сыналган (тикшерү өчен кимендә берсен торгызып карагыз)
- [ ] Мониторинг урнаштырылган -- платформа эштән туктаса, сез моны беләчәксез

### Калыплар {#templates}

- [ ] Өч калыпның барысы да яңа сайтларда дөрес йөкләнә
- [ ] Урын тутыргыч эчтәлек файдалы һәм хатасыз
- [ ] Барлык рәсемнәр тиешенчә лицензияләнгән (сток фото су билгеләре юк)
- [ ] Һәр калып битендә мобиль җайлашу эшли
- [ ] Бит йөкләнү тизлеге кабул итәрлек (GTmetrix яки PageSpeed Insights кебек корал белән сынагыз)
- [ ] Бер калыпта да ватылган сылтамалар яки югалган ресурслар юк

### Планнар һәм продуктлар {#plans-and-products}

- [ ] План тасвирламалары төгәл һәм нишага хас
- [ ] Бәяләр дөрес (айлык һәм еллык)
- [ ] Көйләү түләүләре тиешле планнарда конфигурацияләнгән
- [ ] Сынау чоры Starter планында эшли
- [ ] Өстәмә тәкъдимнәр түләүне рәсмиләштерү вакытында дөрес күренә
- [ ] Плагин һәм тема чикләүләре һәр план буенча дөрес мәҗбүр ителә

### Түләүне рәсмиләштерү агымы {#checkout-flow}

- [ ] Һәр планда тулы сынау теркәлүен башкарыгыз (сынау түләү режимын кулланыгыз)
- [ ] Калып сайлау һәр план өчен дөрес калыпларны күрсәтә
- [ ] Түләү уңышлы үтә
- [ ] Клиент теркәлгәннән соң сәламләү хатын ала
- [ ] Яңа сайт дөрес калып белән ясала
- [ ] Клиент шунда ук үзенең яңа сайтына керә ала
- [ ] Ташлама кодлары дөрес эшли

### Брендлау {#branding}

- [ ] Админ Dashboard FitSite брендлавын күрсәтә

- [ ] Счет-фактуралар сезнең бизнес мәгълүматлары белән дөрес күрсәтелә
- [ ] Маркетинг сайты эшли һәм түләүне рәсмиләштерү формасына сылтамалар бирә

### Беренчел көйләү {#onboarding}

- [ ] Барлык Quick Start сылтамалары дөрес битләргә күрсәтә
- [ ] Сәламләү хатлары сериясе конфигурацияләнгән һәм сыналган
- [ ] Белем базасы мәкаләләре бастырылган һәм кереп була
- [ ] Account бите дөрес план мәгълүматын һәм яңарту вариантларын күрсәтә

### Юридик һәм бизнес {#legal-and-business}

- [ ] Хезмәт күрсәтү шартлары бастырылган һәм түләүне рәсмиләштерүдән сылтама бирелгән
- [ ] Хосусыйлык сәясәте бастырылган һәм кереп була
- [ ] Акча кайтару сәясәте билгеләнгән һәм документлаштырылган
- [ ] Түләүләр алу өчен бизнес-оешма көйләнгән
- [ ] Түләү шлюзы тере режимда (сынау режимында түгел)
- [ ] Салым конфигурациясе сезнең юрисдикция өчен дөрес

## Йомшак эшләтеп җибәрү һәм каты эшләтеп җибәрү {#soft-launch-vs-hard-launch}

Ике этаплы эшләтеп җибәрүне карагыз:

### Этап 1: Йомшак эшләтеп җибәрү {#phase-1-soft-launch}

Җәмәгатьчелеккә чыгарганчы, 5-10 фитнес студия хуҗасын теркәлергә чакырыгыз. Болар сезнең бета клиентларыгыз. Аларга түбәндәгеләр өчен сизелерлек ташлама тәкъдим итегез (гомерлек 50% ташлама, яки 3 ай бушлай):

- Теркәлү һәм беренчел көйләү тәҗрибәсе турында намуслы фикер
- Аларның сайтын күрсәтмә мисал буларак кулланырга рөхсәт
- Очраган теләсә нинди хаталар яки проблемалар турында хәбәр итү

Бу сезгә җәмәгатьчелеккә ачылганчы чын клиент фикерен һәм күрсәтү өчен тере сайтлар бирә.

### Этап 2: Җәмәгатьчелеккә чыгару {#phase-2-public-launch}

Йомшак эшләтеп җибәрүдән килгән фикерләр кертелгәч:

- Түләү шлюзын тере режимга күчерегез
- Маркетинг сайтыгызны бастырыгыз
- Клиентлар җәлеп итүне башлагыз (Дәрес 11)
- Тиешле фитнес тармагы каналларында игълан итегез

## Эшләтеп җибәрү көне гамәлләре {#launch-day-actions}

Җәмәгатьчелеккә чыккан көнне:

1. **Тере түләүләргә күчегез** -- түләү шлюзында сынау режимын сүндерегез
2. **Тагын бер тапкыр тикшерегез** -- чын түләү белән тулы сынау теркәлүен башкарыгыз (соңыннан үзегезгә кире кайтарыгыз)
3. **Игътибар белән күзәтегез** -- хаталарны, уңышсыз теркәлүләрне яки түләү проблемаларын карап торыгыз
4. **Әзер булыгыз** -- беренче чын клиентларыгызга ярдәм кирәк булырга мөмкин, ә тиз җавап ышаныч тудыра
5. **Кыскача бәйрәм итегез** -- аннары эшкә кире кайтыгыз

## Нәрсә дөрес бармаска мөмкин {#what-can-go-wrong}

Түбәндәгеләргә әзер булыгыз:

- **Түләү шлюзы проблемалары**: Шлюз провайдерыгызның ярдәм контактын әзер тотыгыз
- **SSL сертификаты проблемалары**: Сертификатларны ничек тикшерергә һәм яңартырга икәнен белегез
- **Email җибәрү уңышсызлыклары**: Хатларның чыннан да килеп җитүен сынагыз (спам папкаларын тикшерегез)
- **Йөкләнеш астында җитештерүчәнлек**: Трафик кискен артса, хостингны ничек масштабларга икәнен белегез
- **Клиент буталуы**: Белем базаңны һәм ярдәм каналларын әзер тотыгыз

## Бу дәрестә без нәрсә төзедек {#what-we-built-this-lesson}

- **Комплекслы эшләтеп җибәрү алдыннан тикшерү исемлеге** инфраструктураны, эчтәлекне, түләүләрне һәм юридик мәсьәләләрне колачлый
- **Йомшак эшләтеп җибәрү стратегиясе** җәмәгатьчелеккә чыкканчы чын фикер алу өчен
- **Эшләтеп җибәрү көне процедуралары** ышаныч белән тере режимга чыгу өчен
- **Гадәттән тыш хәлләр планы** эшләтеп җибәрү көнендә еш очрый торган проблемалар өчен
