# Дәрес 7: Үзегезнеке итү

Клиентларыгыз беркайчан да үзләрен "ниндидер WordPress plugin" кулланган кебек хис итмәскә тиеш. Алар үзләрен FitSite -- үз тармаклары өчен төзелгән платформа кулланган кебек хис итәргә тиеш. Бу дәрес брендлау, white-labeling һәм платформаны продукт кебек тоелдыру турында.

## Кайда туктап калган идек

FitSite фитнес-студия хуҗаларын план сайлаудан тере сайтка кадәр алып бара торган эшли торган checkout агымына ия. Хәзер без бөтен тәҗрибәне бердәм, брендланган продукт кебек тоелдырабыз.

## Платформа доменыгыз

Брендыгызның нигезе — доменыгыз. FitSite өчен:

- **Төп домен**: `fitsite.com` (маркетинг сайтыгыз һәм челтәр тамыры)
- **Клиент сайтлары**: `studioname.fitsite.com` (субдоменнар)
- **Үз доменнары**: Growth һәм Pro планнарындагы клиентлар үз доменнарын бәйли ала

### Доменыгызны көйләү

1. Платформа доменыгызны теркәгез
2. Аны хостинг тәэмин итүчегезгә юнәлтегез
3. Клиент субдоменнары өчен wildcard DNS (`*.fitsite.com`) көйләгез
4. wildcard SSL актив булуын тикшерегез

Төгәл күрсәтмәләр өчен [Домен бәйләүне ничек көйләргә](/user-guide/domain-mapping/how-to-configure-domain-mapping) карагыз.

## Admin тәҗрибәсен White-Labeling

Фитнес-студия хуҗасы үз сайт Dashboard-ына кергәч, WordPress яки Ultimate Multisite брендын түгел, сезнең брендны күрергә тиеш.

### Үз Login бите

WordPress login битен түбәндәгеләрне күрсәтерлек итеп көйләгез:

- Сезнең FitSite логотибы
- Фитнеска туры килгән фон сурәтләре
- Сезнең бренд төсләре

### Dashboard брендлау

[Admin Page Creator](/addons/admin-page-creator) өстәмәсен яки махсус CSS кулланып:

- WordPress логотибын FitSite логотибыгыз белән алыштырыгыз
- Admin төсләр схемасын брендыгызга туры китереп көйләгез
- Фитнеска хас тиз сылтамалар һәм ярдәм ресурслары белән махсус dashboard виджеты өстәгез

### Махсус Admin битләре

Фитнес-студия хуҗалары өчен иң мөһим гамәлләрне алга чыгара торган махсус admin битләре булдыруны карагыз:

- "Дәресләр җәдвалын үзгәртү"
- "Тренер профильләрен яңарту"
- "Сайтыгызны карау"

Бу ниша өчен мөһим гамәлләрне стандарт WordPress менюсында яшерү урынына алгы планга чыгарып, өйрәнү катлаулылыгын киметә.

## Коммуникацияләрегезне брендлау

Һәр email, инвойс һәм хәбәр брендыгызны ныгытырга тиеш.

### Система email-лары

**Ultimate Multisite > Settings > Emails** бүлегенә күчегез һәм барлык система email-ларын көйләгез:

- **From name**: FitSite
- **From email**: hello@fitsite.com
- **Email шаблоннары**: Бренд төсләрегезне һәм логотибыгызны кулланыгыз
- **Тел**: Бөтен җирдә фитнеска хас булсын

Көйләнергә тиешле төп email-лар:

| Email | Гомуми версия | FitSite версиясе |
|-------|----------------|-----------------|
| Рәхим итү | "Яңа сайтыгыз әзер" | "Фитнес-студия сайтыгыз эшли башлады" |
| Түләү квитанциясе | "Түләү кабул ителде" | "FitSite язылу түләве расланды" |
| Сынау вакыты тәмамлану | "Сынау вакытыгыз тиздән тәмамлана" | "FitSite сынау вакытыгыз 3 көннән тәмамлана -- студия сайтыгызны эшләтеп калдырыгыз" |

### Инвойслар

Инвойс шаблоннарын түбәндәгеләр белән көйләгез:

- FitSite логотибыгыз һәм бренд төсләрегез
- Бизнес мәгълүматларыгыз
- Фитнеска хас продукт исемнәре (гомуми план ID-лары түгел)

## Клиентларга күренә торган сайт

Төп доменыгызга (`fitsite.com`) платформаны сата торган маркетинг сайты кирәк. Бу Ultimate Multisite челтәр admin-ыннан аерым -- ул бизнесыгызның җәмәгатькә күренә торган йөзе.

Төп битләр:

- **Баш бит**: Фитнес бизнеслары өчен ачык кыйммәт тәкъдиме
- **Мөмкинлекләр**: FitSite нәрсә эшли, фитнес терминнары белән
- **Бәяләр**: Нишага хас мөмкинлек чагыштырулары белән өч планыгыз
- **Мисаллар**: Платформада төзелгән сайтларны күрсәтү
- **Теркәлү**: Checkout формагызга сылтамалар

:::tip Маркетинг сайтыгыз челтәр сайты була ала
Маркетинг сайтыгызны үз челтәрегез эчендәге сайт итеп төзегез. Бу аны шул ук Dashboard-тан идарә итәргә мөмкинлек бирә һәм үз платформагызның мөмкинлекләрен күрсәтә.
:::

## Клиентлар өчен үз домены

Үз доменнарын үз эченә алган планнардагы клиентлар өчен процессны ачык итеп документлаштырыгыз:

1. Клиент доменын терки яки регистраторга күчерә
2. Клиент DNS-ны сезнең платформага юнәлтерлек итеп яңарта (төгәл язмаларны бирегез)
3. Ultimate Multisite домен бәйләүне һәм SSL-ны башкара

Бу процесс өчен махсус ярдәм мәкаләсе яки белем базасы язмасы булдырыгыз, аны техник булмаган фитнес-студия хуҗалары өчен языгыз.

## Әлегә FitSite челтәре

```
FitSite Network
├── WordPress Multisite (subdomain mode)
├── Ultimate Multisite (configured + branded)
├── Platform Domain (fitsite.com + wildcard SSL)
├── Site Templates (Studio Essential, Gym Pro, Fitness Chain)
├── Products (Starter, Growth, Pro + Order Bumps)
├── Checkout Flow (niche-specific, tested)
├── Branding
│   ├── Custom login page
│   ├── Branded admin dashboard
│   ├── Niche-specific system emails
│   ├── Branded invoices
│   └── Marketing site on fitsite.com
└── Ready for onboarding flow (next lesson)
```

## Бу дәрестә без нәрсә төзедек

- **Платформа домены һәм DNS** брендланган тәҗрибә өчен көйләнде
- **White-labeled admin** FitSite бренды белән бөтен җирдә
- **Көйләнгән коммуникацияләр** -- email-лар, инвойслар һәм хәбәрләр барысы да брендка туры килә
- **Маркетинг сайты** FitSite-ны фитнес-студия хуҗаларына сата
- **Үз домены документациясе** үз доменын теләгән клиентлар өчен
