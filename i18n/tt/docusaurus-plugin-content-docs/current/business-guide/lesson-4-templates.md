# Дәрес 4: Ниша шаблоннарын төзү

Шаблоннар — сезнең ниша кыйммәт тәкъдименең үзәге. Теркәлеп, инде фитнес сайтына охшаган сайтны -- дәресләр расписаниесе, тренер профильләре һәм туры килгән рәсемнәре белән -- күргән фитнес студиясе хуҗасының калу ихтималы буш бит күргән кешегә караганда күпкә югарырак.

## Кайда тукталган идек {#where-we-left-off}

Бездә Ultimate Multisite урнаштырылган һәм көйләнгән эшләүче FitSite челтәре бар. Хәзер без FitSite-ны фитнес бизнеслары өчен махсус төзелгән кебек тоелдыра торган шаблоннарны төзибез.

## Ни өчен ниша шаблоннары мөһим {#why-niche-templates-matter}

Гомуми шаблоннар клиентларыгызны авыр эшне үзләре башкарырга мәҗбүр итә: нинди битләр кирәклеген, нинди эчтәлек структурасы эшләвен һәм аны үз тармакларына туры килерлек итеп ничек күрсәтергә кирәклеген ачыклау. Ниша шаблоннары бу каршылыкны бетерә.

FitSite-ка теркәлгән фитнес студиясе хуҗасы түбәндәгеләрне күрергә тиеш:

- Фитнес студиясе сайтына охшаган баш бит
- Дәресләр, расписаниеләр, тренерлар һәм бәяләр өчен алдан булдырылган битләр
- Аларның тармагына туры килгән рәсемнәр һәм вакытлыча эчтәлек
- Фитнес өчен һөнәри һәм брендка туры килгән булып тоелган дизайн

Алар үз мәгълүматларын тутыра. Алар нульдән башламый.

## Шаблоннарыгызны планлаштыру {#planning-your-templates}

Төзүгә керешкәнче, нинди шаблоннар тәкъдим итәчәгегезне хәл итегез. FitSite өчен без өч шаблон булдырачакбыз:

### Шаблон 1: Studio Essential {#template-1-studio-essential}

Кече студияләр һәм шәхси тренерлар өчен.

- **Баш бит** hero-рәсем, дәресләр аерып күрсәтү һәм гамәлгә чакыру белән
- **Турында** бите студия тарихы һәм миссиясе белән
- **Дәресләр** бите расписание урнашуы белән
- **Тренерлар** бите профиль карточкалары белән
- **Элемтә** бите урнашу картасы һәм форма белән
- **Чиста, заманча дизайн** фитнеска туры килгән төсләр схемасы белән

### Шаблон 2: Gym Pro {#template-2-gym-pro}

Күбрәк хезмәтләре булган нык урнашкан спорт заллары өчен.

- Studio Essential эчендәге бар нәрсә, өстәвенә:
- **Әгъзалык** бите бәяләр таблицасы белән
- **Галерея** бите зал фотолары өчен
- **Блог** бүлеге фитнес киңәшләре һәм яңалыклар өчен
- Баш биттә **фикерләр** бүлеге
- **Күренеклерәк брендинг** мөмкинлекләре

### Шаблон 3: Fitness Chain {#template-3-fitness-chain}

Күп урында эшләүче оешмалар өчен.

- Gym Pro эчендәге бар нәрсә, өстәвенә:
- **Урыннар** бите берничә зал исемлеге белән
- **Франшиза/урын** аерым бите шаблоны
- Урынга хас детальләр белән **үзәкләштерелгән брендинг**
- Урыннар буенча **хезмәткәрләр каталогы**

## Шаблон сайтын төзү {#building-a-template-site}

Ultimate Multisite эчендә шаблон — яңа клиент сайтлары ничек күренергә тиеш булса, шулай көйләнгән WordPress сайты гына. Менә аны ничек булдыру:

### Адым 1: Шаблон сайтын булдыру {#step-1-create-the-template-site}

1. Челтәр администраторыгызда **Сайтлар > Яңа өстәү** бүлегенә күчегез
2. `template-studio-essential` исемле сайт булдырыгыз
3. Бу сайт сезнең эш мәйданыгызга әйләнә

### Адым 2: Теманы урнаштыру һәм көйләү {#step-2-install-and-configure-the-theme}

Шаблон сайты Dashboard-ына күчегез һәм:

1. Фитнес бизнеслары өчен яраклы theme активлаштырыгыз
2. Theme көйләүләрен, төсләрен һәм типографиясен көйләгез
3. Header һәм footer-ны фитнеска туры килгән навигация белән көйләгез

:::tip Theme сайлау
Фитнес өчен яхшы күренерлек дәрәҗәдә сыгылмалы, ләкин клиентларыгыз идарә итә алмаслык итеп артык катлаулы булмаган theme сайлагыз. Astra, GeneratePress яки Kadence кебек themes яхшы эшли, чөнки алар җиңел, көйләнә торган һәм яхшы ярдәм ителә.
:::

### Адым 3: Битләрне булдыру {#step-3-create-the-pages}

Һәр битне түбәндәгеләр белән төзегез:

- Фитнес өчен табигый яңгыраган **вакытлыча эчтәлек** ("[Studio Name]-ка рәхим итегез", "Lorem ipsum" түгел)
- Фитнес активлыкларын күрсәткән бушлай сток фото сайтларыннан **вакытлыча рәсемнәр**
- Бит төзүче яки блок мөхәррире ярдәмендә **функциональ урнашулар**

Мөмкин булган урыннарда вакытлыча эчтәлекне өйрәтүче итеп ясагыз. Гомуми тутыргыч текст урынына мондый эчтәлек языгыз: "Моны студиягезнең кыскача тасвирламасы һәм аны үзенчәлекле иткән нәрсәләр белән алыштырыгыз. Тренировка фәлсәфәгезне, тәҗрибә елларыгызны яки клиентлар нәрсә көтә алуын искә алыгыз."

### Адым 4: Plugins көйләү {#step-4-configure-plugins}

Фитнес студияләренә кирәк булган plugins урнаштырыгыз һәм активлаштырыгыз:

- Броньлау яки расписание plugin-ы (план дәрәҗәгезгә туры килсә)
- Элемтә формасы plugin-ы
- SEO plugin-ы (фитнеска кагылышлы килешү буенча көйләүләр белән алдан көйләнгән)

### Адым 5: Шаблон итеп билгеләү {#step-5-mark-as-template}

1. **Ultimate Multisite > Сайтлар** бүлегенә күчегез
2. Шаблон сайтын төзәтегез
3. **Сайт шаблоны** күчергечен кабызыгыз

Тәкъдим итәргә теләгән һәр шаблон өчен бу процессны кабатлагыз.

## Шаблон сыйфаты тикшерү исемлеге {#template-quality-checklist}

Шаблонны куллануга биргәнче, тикшерегез:

- [ ] Барлык битләр дөрес йөкләнә һәм һөнәри күренә
- [ ] Вакытлыча эчтәлек файдалы һәм нишага хас
- [ ] Рәсемнәр туры килә һәм тиешенчә лицензияләнгән
- [ ] Мобиль җайлашучанлык барлык битләрдә эшли
- [ ] Навигация логик һәм тулы
- [ ] Элемтә формалары эшли
- [ ] Ватылган сылтамалар яки югалган ресурслар юк
- [ ] Бит йөкләнү тизлеге кабул итәрлек

## Бу дәрестә без нәрсә төзедек {#what-we-built-this-lesson}

- Төрле зурлыктагы фитнес бизнеслары өчен эшләнгән **өч нишага хас шаблон**
- Платформаны махсус төзелгән кебек тоелдыра торган **фитнеска туры килгән эчтәлек һәм рәсемнәр**
- Клиентларны көйләү аша алып бара торган **өйрәтүче вакытлыча эчтәлек**
- Шаблоннарның эшкә әзер булуын тәэмин итү өчен **сыйфат тикшерү исемлеге**
