# 5 нче дәрес: Планнарыгызны проектлау

Сезнең план дәрәҗәләре бәя дәрәҗәләре генә түгел -- алар ниша клиентларыгызның чынлыкта ничек эшләвен чагылдыра. Бу дәрестә сез төрле этаплардагы фитнес бизнесларының реаль ихтыяҗларына туры килгән продукт дәрәҗәләрен проектлаячаксыз.

## Без кайда туктадык

FitSite өчен өч шаблон әзер (Studio Essential, Gym Pro, Fitness Chain). Хәзер клиентлар нәрсәгә керү алачагын һәм нәрсә өчен түләячәген билгели торган планнарны төзибез.

## Клиент сегментлары турында уйлау

Күпчелек кеше ясый торган хата -- планнарны техник функцияләр (саклагыч, үткәрү киңлеге, битләр саны) тирәсендә проектлау. Ниша клиентларыгыз болай уйламый. Зал хуҗасы үз бизнесын алып бару өчен нәрсә кирәклеге турында уйлый.

Фитнес студияләре өчен өч табигый сегмент бар:

| Сегмент | Алар кем | Аларга нәрсә кирәк |
|---------|-------------|----------------|
| **Ялгыз тренерлар / кечкенә студияләр** | 1-3 хезмәткәр, бер локация, яңа башлыйлар | Профессиональ сайт, дәресләр турында мәгълүмат, элемтә формасы |
| **Формалашкан спорт заллары** | 5-20 хезмәткәр, бер локация, үсештә | Югарыдагының барысы, өстәвенә броньлау, блог, галерея, махсус домен |
| **Фитнес челтәрләре** | Берничә локация, формалашкан бренд | Югарыдагының барысы, өстәвенә күп сайтлы структура, локация битләре, хезмәткәрләр белешмәсе |

Планнарыгыз очраклы функция җыелмаларына түгел, ә шушы сегментларга туры килергә тиеш.

## FitSite планнарын төзү

Һәр план өчен **Ultimate Multisite > Products > Add Product** бүлегенә күчегез.

- Тасвирлама: "Шәхси тренерга яки кечкенә студиягә онлайнда профессиональ күренү өчен кирәк булганның барысы."

**Чикләүләр**:

- Сез сайлаган тема: Мәҗбүри активлаштыру

- Продукт тәртибе: 3

## План төркемен көйләү

План төркеме клиентларның бары тик FitSite план гаиләсе эчендә генә югарырак яки түбәнрәк планга күчә алуын тәэмин итә. Һәр планның **Югарырак һәм түбәнрәк планга күчүләр** салынмасында:

1. Өч планның барысы өчен **План төркеме** итеп "FitSite Plans" куегыз
2. **Продукт тәртибе** итеп 1 (Starter), 2 (Growth), 3 (Pro) куегыз

Бу ачык үсү юлын булдыра: Starter → Growth → Pro.

## Заказ өстәмәләре өстәү

Заказ өстәмәләре -- заказны рәсмиләштерү вакытында тәкъдим ителә торган өстәмә продуктлар. FitSite өчен боларны карагыз:

- **Өстәмә саклагыч пакеты** ($19/ай) -- өстәмә 5 GB диск урыны
- **Өстенлекле ярдәм** ($29/ай) -- тизрәк җавап бирү вакытлары
- **Өстәмә сайт** ($39/ай) -- планы рөхсәт иткәннән күбрәк сайт кирәк булган клиентлар өчен

Боларны Ultimate Multisite эчендә **Пакет** төрендәге продуктлар итеп төзегез һәм аларны тиешле планнар белән бәйләгез.

## Ни өчен бу структура эшли

- **Starter** керү киртәләрен бетерә -- түбән бәя, гади тәкъдим, тренерларны тиз онлайнга чыгара
- **Growth** спорт заллары чынлап сораган функцияләрне өсти -- броньлау, галереялар, махсус доменнар
- **Pro** берничә локацияле ярдәм кирәк булган югары кыйммәтле сегментка хезмәт итә
- **Заказ өстәмәләре** клиентларга төп планнарны катлауландырмыйча көйләү мөмкинлеге бирә
- **Ачык югарырак планга күчү юлы** клиентлар китү урынына сезнең белән бергә үсә дигәнне аңлата

## Хәзергә кадәр FitSite челтәре

```
FitSite Network
├── WordPress Multisite (subdomain mode)
├── Ultimate Multisite (configured)
├── Hosting with wildcard SSL + domain mapping
├── Site Templates (Studio Essential, Gym Pro, Fitness Chain)
├── Products
│   ├── FitSite Starter ($49/mo) → Studio Essential template
│   ├── FitSite Growth ($99/mo) → Studio Essential + Gym Pro templates
│   ├── FitSite Pro ($199/mo) → All templates
│   └── Order Bumps (Extra Storage, Priority Support, Additional Site)
└── Ready for checkout configuration (next lesson)
```

## Бу дәрестә без нәрсә төзедек

- Реаль фитнес бизнесы сегментларына туры китерелгән **өч план дәрәҗәсе**
- Ultimate Multisite плагин һәм шаблон идарәләре ярдәмендә **функцияләрне чикләү**
- Ачык югарырак планга күчү юлы булган **план төркеме**
- Өстәмә керем өчен **заказ өстәмәсе продуктлары**
- Техник спецификацияләргә түгел, клиент кыйммәтенә нигезләнгән **бәяләү структурасы**
