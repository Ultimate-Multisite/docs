# Filter: wu_admin_notices

Võimaldab arendajatel filtreerida Ultimate Multisite lisatud adminiteavitusi.

## Parameetrid {#parameters}

| Nimi | Tüüp | Kirjeldus |
|------|------|-------------|
| $notices | `array` | Selle konkreetse paneeli teavituste loend. |
| $all_notices | `array` | Lisatud teavituste loend, paneelide kaupa eraldatud. |
| $panel | `string` | Paneel, mille teavitused tuua. |
| $filter | `string` | Kas suletavad teavitused on välja filtreeritud. |
| $dismissed_messages | `array` | Suletud teavituste võtmete loend. |
