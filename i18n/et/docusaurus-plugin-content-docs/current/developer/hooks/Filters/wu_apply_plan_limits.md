# Filter: wu_apply_plan_limits

Luba plugin arendajatel piirangud lühistada.

Seda filtrit saad kasutada suvalise koodi käivitamiseks enne mis tahes piirangute algatamist. Kui sinu filter tagastab mis tahes truthy väärtuse, liigub protsess edasi; kui see tagastab mis tahes falsy väärtuse, siis kood lõpetab töö ja ükski allolevatest hookidest ei käivitu.
