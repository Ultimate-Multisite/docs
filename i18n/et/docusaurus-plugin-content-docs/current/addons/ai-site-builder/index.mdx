# AI saidiehitaja

ultimate-multisite-ai-site-builder

## Nõuded

- WordPress 5.3 või uuem
- PHP 7.4 või uuem
- Multisite Ultimate plugin (aktiivne)

## Paigaldamine

1. Laadi lisa failid üles oma `/wp-content/plugins/` kataloogi
2. Aktiveeri plugin WordPressi menüü „Pluginad” kaudu
3. Seadista lisa sätted võrguhalduses

## Arendus

### Seadistamine

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Projekti struktuur

```
ultimate-multisite-ai-site-builder/
├── inc/                          # PHP klassid ja lisatavad failid
│   ├── class-multisite-ultimate-updater.php  # Uuenduste käsitleja
│   └── class-ai_site_builder-main.php             # Põhifunktsionaalsus
├── tests/                        # Ühiktestid
│   ├── bootstrap.php            # Testide käivitusfail
│   ├── class-*-base.php         # Testi baasklass
│   └── class-*-test.php         # Testijuhtumid
├── views/                       # Mallifailid
├── assets/                      # Staatilised varad
│   ├── css/                     # Stiililehed
│   ├── js/                      # JavaScripti failid
│   └── images/                  # Pildid
├── lang/                        # Keelefailid
├── composer.json                # PHP sõltuvused
├── package.json                 # Node.js sõltuvused
├── phpunit.xml.dist             # PHPUnit konfiguratsioon
├── .phpcs.xml.dist              # Koodistandardite konfiguratsioon
└── rector.php                   # Koodi moderniseerimise reeglid
```

### Testimine

Lisa sisaldab põhjalikku testimisraamistikku:

- **Ühiktestid**: PHPUnit-põhised testid koos WordPressi testikomplektiga
- **Testi baasklass**: Pakub abimeetodeid tavaliste testistsenaariumide jaoks
- **Testikatvus**: Seadistatud koodikatvuse jälgimiseks
- **Multisite testimine**: Testid käivitatakse multisite-keskkonnas

### Koodistandardid

- **WordPressi koodistandardid**: Jõustatakse PHPCS-i kaudu
- **PHP 7.4+ ühilduvus**: Toetatud on kaasaegsed PHP funktsioonid
- **Staatiline analüüs**: PHPStan integratsioon tüübikontrolliks
- **Koodi moderniseerimine**: Rectori reeglid PHP uuenduste jaoks

### Koostesüsteem

- **Varade töötlemine**: CSS/JS failide minimeerimine
- **Tõlkimine**: POT-faili genereerimine
- **Pakendamine**: Arhiivi loomine levitamiseks
- **Arendusrežiim**: Minimeerimata varad silumiseks

## Konfiguratsioon

Lisa toetab mitmesuguseid konfiguratsioonivalikuid, millele pääseb ligi võrguhalduse liidese kaudu.

## Hookid ja filtrid

### Toimingud

- `ultimate-multisite-ai-site-builder_init` - Käivitatakse pärast lisa initsialiseerimist
- `ultimate-multisite-ai-site-builder_loaded` - Käivitatakse pärast kõigi lisa failide laadimist

### Filtrid

- `ultimate-multisite-ai-site-builder_settings` - Muuda lisa sätteid
- `ultimate-multisite-ai-site-builder_enabled` - Tühista lisa lubamise/keelamise olek
