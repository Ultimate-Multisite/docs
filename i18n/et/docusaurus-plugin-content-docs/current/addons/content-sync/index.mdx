# Sisu sünkroonimine

Multisite Ultimate: sisu sünkroonimine

See Multisite Ultimate’i lisandmoodul võimaldab võrguadministraatoritel sünkroonida sisu mis tahes saidilt teistele võrgus olevatele saitidele. See toetab kõiki postituse tüüpe, sealhulgas postitusi, lehti, kohandatud postituse tüüpe ja Elementor malle. Lisandmoodul sisaldab mitmeastmelist viisardiliidest sisu ja sihtsaitide valimiseks, koos paketttöötluse ja reaalajas edenemise tagasisidega.

## Funktsioonid {#features}

* Sisu valimine: visuaalne kaartide ruudustik pisipiltide ja postituse tüübi filtreerimisega
* Kõik postituse tüübid: sünkrooni postitusi, lehti, tooteid, Elementor malle ja mis tahes kohandatud postituse tüüpi
* Saidi valimine: vali, millistele saitidele sisu sünkroonida, koos lähtesaidi välistamise ja alamsaitide eelvalikuga
* Paketttöötlus: sünkroonib korraga 10 saidile koos edenemise uuendustega
* Meedia sünkroonimine: kopeerib ja seob automaatselt ümber meediafailid (pildid, videod) postituse sisust sihtsaitidele
* Taksonoomia sünkroonimine: säilitab kategooriad, sildid ja kohandatud taksonoomiad
* Elementor tugi: täielik tugi Elementor mallidele koos CSS-i uuesti genereerimise ja malli ID-de ümberseostamisega
* Konfliktide lahendamine: võimalus olemasolev sisu asendada või see vahele jätta
* Vahemälu haldus: tühjendab pärast sünkroonimist sihtsaitidel automaatselt Elementor vahemälu

## Nõuded {#requirements}

* WordPress 5.3 või uuem
* PHP 7.4 või uuem
* Multisite Ultimate plugin (aktiivne)
* Elementor plugin (valikuline, vajalik Elementor malli sünkroonimiseks)

## Paigaldamine {#installation}

1. Laadi 'ultimate-multisite-content-sync' üles kataloogi '/wp-content/plugins/'
2. Aktiveeri plugin oma WordPress Network Admini menüü 'Pluginad' kaudu
3. Seadista sätted jaotises Ultimate Multisite → Seaded → Sisu sünkroonimine
4. Liigu jaotisesse Ultimate Multisite → Sisu sünkroonimine, et sünkroonimist alustada
