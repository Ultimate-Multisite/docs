# PayFast

Multisite Ultimate: Payfast Gateway

Lisab Payfasti – kiire ja turvalise veebimaksete lüüsi Lõuna-Aafrikas – Ultimate Multisite maksevalikuna.

## Paigaldamine

1. Laadi 'ultimate-multisite-payfast' üles kataloogi '/wp-content/plugins/'
2. Aktiveeri pistikprogramm WordPressi menüü 'Pistikprogrammid' kaudu
3. Järgi samm-sammult Wizardit, et pistikprogramm seadistada

## Kasutusjuhend

PayFast on Lõuna-Aafrika ettevõtete juhtiv veebimaksete töötleja. See on pühendunud veebikaupmeeste kasvu toetamisele, muutes tehingud sujuvaks ja turvaliseks. Ultimate Multisite kasutaja soovi põhjal otsustasime luua lisamooduli, mis teeb PayFastiga integreerimise võimalikuks. See artikkel näitab, kuidas seadistada PayFast oma makselüüsiks, et võtta makseid vastu oma Ultimate Multisite võrgustikus.

Esimene samm on lisamooduli paigaldamine.

# Kuidas paigaldada Payfasti lisamoodul:

Mine lisamoodulite lehele ja vali Ultimate Multisite: Payfast Gateway lisamoodul.

![Lisamoodulite leht](/img/admin/addons-page.png)

Lisamooduli aken avaneb. Klõpsa nupul Install Now.

![Lisamoodulite leht paigaldamise valikuga](/img/admin/addons-page.png)

Pärast paigaldamise lõpetamist suunatakse sind pistikprogrammide lehele. Siin klõpsa lihtsalt Network Activate ja _Payfast_ lisamoodul aktiveeritakse sinu võrgustikus.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# PayFasti seadistamine

Pärast lisamooduli paigaldamist saad nüüd minna Ultimate Multisite seadetesse ja alustada PayFasti konfigureerimist.

Lehel **Ultimate Multisite seaded** leiad vahekaardi "Maksed". Selle vahekaardi alt peaksid nägema "**Payfast** " koos lülitusnupuga selle makselüüsi lubamiseks või keelamiseks.

![Makselüüsi seaded](/img/config/settings-payment-gateways.png)Klõpsa nüüd lülitusnuppu, et see lubada. Seejärel laiendab see kasti ja kuvab eri väljad, mis tuleb täita, näiteks **Mechant ID, Merchant Key ja PassPhrase.** Selle teabe saad oma **Payfast** kontolt

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Järgmine samm on oma PayFast kontole sisse logida, et saada vajalik teave.

Pärast sisselogimist näed kohe oma **Payfast** konto dashboardil vajalikke juurdepääsuandmeid.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->Saad juurdepääsuandmed lihtsalt kopeerida ja kleepida ning täita üksikasjad oma Ultimate Multisite seadetes. Seejärel klõpsa muudatuste salvestamiseks nuppu **"Save settings** ".

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Ongi kõik, sinu PayFast maksete integratsioon on valmis!
