## Parameetrid {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $site_id | `int` | Teisaldatud saidi ID. |
| $old_network_id | `int` | Saidi algse võrgu ID. |
| $new_network_id | `int` | Võrgu ID, kuhu sait on teisaldatud. |
