# Fluent Forms

See lisamoodul võimaldab lisada Fluent Formsile piiranguid Multisite Ultimate’is toodetes, liikmesustes või saitidel.
Fluent Forms kasutab kohandatud postitusetüüpide asemel kohandatud tabeleid, seega tavapärased piiramisvalikud Fluent Formsiga ei tööta.

## Funktsioonid {#features}

- Lisa piirangud Fluent Formsile või Conversational Fluent Formsile
- Kliendid saavad limiidini jõudes sõbraliku sõnumi, mis palub neil oma plaani täiendada
