# Support Agents

Terve võrgu haldamine üksinda ei ole lihtne ülesanne. Kui sinu võrk hakkab kasvama, näed üha suuremat nõudlust toe järele. Hea mõte on kaasata inimesi, kes aitavad sul töökoormust vähendada. Täieliku juurdepääsu/õiguste andmine on aga riskantne ja võib sinu võrgu ohtu seada. Siin tulebki mängu see Support Agent lisamoodul.

See lisamoodul võimaldab sul luua uut tüüpi WordPress kasutaja nimega Support Agent ning annab sulle täieliku kontrolli selle üle, millised õigused sellel kasutajal on. Õiguse andmine on sama lihtne kui lülitusnupu sisselülitamine, ja kui otsustad selle eemaldada, siis selle väljalülitamine.

## Funktsioonid {#features}

- Üksikasjalik loend WordPress & Ultimate Multisite õigustest
- Luba ja keela juurdepääsu hõlpsalt
- Loo piiramatu arv Agent kontosid

## Paigaldamine {#installation}

1. Laadi 'ultimate-multisite-support-agents' üles kataloogi '/wp-content/plugins/'
2. Aktiveeri plugin WordPress menüü 'Plugins' kaudu
3. Järgi samm-sammulist viisardit plugini seadistamiseks
