# Shareable Linkide juhtimise plaanidega {#managing-shareable-links-for-plans}

Ultimate Multisite võimaldab teil plaanide linke otse klientidega jagada. Seda teha jaoks minud oma plaanide redigeerimislehtele ja klõpsake _Click to copy the Shareable Link_ (Jagatud linki kopienda) nupulile.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_Plaani redigeerimislehel leiate jagatud linkide luomise nupugi_

Märge, et see meheisme kasutab poolt määratud lahenduse (billing frequency) linki loomiseks. See tähendab, et kui teie poolt määratud lahendus on näiteks kuudkujuline, siis lõpuks teil oleksid sellise link: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Kui soovite jagatud linki oletada kvartaliseks, peate lisama lõppu 3:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Plaani _premium_ valimiseks aastaajaline järelevaljunduse korral lisate 1 ja sõna:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Pärast linki jagamist klientidega suunatakse nad teie registreerimise protsessi esimese sammule.
