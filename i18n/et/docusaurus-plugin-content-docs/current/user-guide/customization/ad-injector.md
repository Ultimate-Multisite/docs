# Reklaad reklaame oma võrku Ultimate Multisite'iga: Ad Injector {#placing-ads-on-your-network-with-ultimate-multisite-ad-injector}

Ultimate Multisite: Ad Injector on **tasuta lisamoodul**, mis annab teil tööriistad reklaamide paigutamiseks juba nende redigeerimisest.

Ultimate Multisite: Ad Injector installimise pärast minud minuga _Plugins_ lehelge minuga aktiveerida selle.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Klõpsake "Network Activate" (Väärkuulutamine) nupul._

Kuna lisamoodul on aktiivne, ilmub plaanide redigeerimisekränal _Advanced Options_ meta laukus uus valik nimetusega _Ad Settings_.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_Nüüd saate muuta reklaamiparametreid iga oma plaani jaoks._

Selle tabist saate lisada postide ja lehtede reklaamikoode esile (contenti enne), contenti kõrvale ja pärast seda.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Reklaamikoode sisestamise pärast ilmub see selle konkreetse plaani postidele ja lehtedele._

Võite ka reklaame sisaldada administraatorile (Admin Panel) lehekülgede, kasutajatele, kes on selle plaani kasutaja.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Minudenda "Admin Panel Ads" (Administraatorile reklaamikoode) valikku, kui soovite reklaame näidata oma kasutajate back-endil._

### Reklaadide üleliituse üle {#beyond-advertising}

Võite ka Ultimate Multisite: Ad Injectorist kasutada kõrge taseme plaane jaoks tärinike.

Minuga minuga "Admin Panel Ads" valikku ja lisage HTML-kood oma tuetustoolile.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

Nega saab teiega suhelda otseselt oma Admin Panelist!
