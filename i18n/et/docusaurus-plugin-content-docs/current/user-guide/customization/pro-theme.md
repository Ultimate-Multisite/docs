# Administrats paneli personalisatsioon PRO Theme'iga {#customizing-your-admin-panel-with-pro-theme}

Kliendite töö ajal meil on sageli vaja pakkuda klientidele WordPressi administrati paneelile spetsiaalne kogemus. Kui otsite professionaalsemalt välja näha ja tunnet, võib PRO Theme teil aidata.

PRO Theme on administrati teema, mille arendasid selleks, et aidata teil klientide dashboardi personaliseerimisel. See on ka Ultimate Multisite premium plugin core'i osa.

Installeerides seda, minudte _Plugins_ lehel ja klõpsake aktiivamiseks oma Networkil.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Aktiivige seda kogu networkile Plugin-lehelest_

_**MÄÄRKUS:**_ _Kui installite seda, ilmub see teie klientidele valikuna, mida saab märgida, et teema lisada ainult selle kasutajale._

Nüüd on teie administrati liutumine täiesti ümber disainitud! Kui teil on vasikasutujaks uue menüü nimega PRO Theme.
