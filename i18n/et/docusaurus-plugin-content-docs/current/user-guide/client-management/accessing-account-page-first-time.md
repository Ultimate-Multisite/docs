# Kuidas ma pääse oma kontoile esimest kord? {#how-can-i-access-my-account-page-for-the-first-time}

Vali tervitsemist Ultimate Multisite!

Pääset oma kontolehele linkiga, mida saadid e-posti pärast meie plugin osta:

<!-- Screenshot unavailable: Account page email with access link -->

Kasuta e-posti, mida sa esitatud osta ajal, ja parooli, mida sa sai e-postis.

<!-- Screenshot unavailable: Login page with email and password fields -->

Sest see on esimene kord, kui logiine tead oma kontolehele, peaksid oma paroolit muutma. Muuta see ja klõpsa _Save_ (Salvesta) nupulile.

<!-- Screenshot unavailable: Change password form with Save button -->

Oma kontolehel saadad hallata kõik oma kontdetail, nagu laskumised, uuesti võtmised, tühistused ja palju muud.
