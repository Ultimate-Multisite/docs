# Paneel ülevaade {#dashboard-overview}

Ultimate Multisite on oma menüü all eraldatud **Dashboard** ja lisab ka widget'e Network Dashboardile. See artiklit käsitleb, mida teid mõlemast paigast näha.

## Network Dashboard {#the-network-dashboard}

Ultimate Multisite asub kasulikud widget'id network-tasa dashboardi sekkoni, mida saab leida **My Sites → Network Admin → Dashboard** all.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

Siin on, mida iga widget pakub:

**Esimlikud sammud (First Steps):** Näidet pärast uut installimist ja mäletab teid seadistuste täitmise eest, nagu maksukättesaadavuse konfiguratsioon ja testi konto loomine.

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Kokkuvõte (Summary):** Annab ülevaate registreerimistulevikust ja müüritulost päevas. MRR (Monthly Recurring Revenue) annab projektsioonilise kokkuvõtte klientide hulgas, kes on müüritulutavaid jälgendusi.

![Summary widget](/img/admin/network-dashboard-summary.png)

**Aktiivsus (Activity Stream):** Annab ülevaate Ultimate Multisite'i sündmustest – registreerimised, tühistused, veebilehtide loomine, eemaldamine ja muutuvad teised tegevused.

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**Nüüd (Right Now):** Näitab lühikku ülevaate kasutajadest ja veebilehtidest networkis koos kiiralt linkidega uuside loomiseks või kasutajate loomiseks.

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**Uutised ja diskussioonid (News & Discussions):** Kõrvestab ja näitab viimasi Ultimate Multisite uudiseid, sealhulgas uuendusi, kriitilisi vigu ja turvalisuse parandusi.

![News widget](/img/admin/network-dashboard-news.png)

## Ultimate Multisite Dashboard {#the-ultimate-multisite-dashboard}

Samas kui network dashboard annab ülevaate kogu networkist,Ultimate Multisite dashboard (Ultimate Multisite menüü kõrge tasemel) annab teid infot teenustest.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Kuunulikuva (MRR) kasvu:** Näyttää kuukausittaisen erittelyn tuloksista 12 kuukauden kalenteriaikana. Uusi tulo näkyy sinisellä, kun taas peruutus näkyy punaisella.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**Päivaraamendustuki:** Suodata dashboardin tietoja ajanjakson mukaan keskittumaan siihen tietoon, jota tarvitset. Päivaraami valitsee, mitä widgeteita alla näytetään.

![Date range and dashboard stats](/img/admin/um-dashboard-date-range.png)

Dashboardin widgetit sisältävät:

- **Tulot** — Kokonaistulot ajanjaksolle MRR- ja tuotteittaiset erittelyt
- **Uudet jäsenyysmaksut** — Kuinka monta uutta jäsenyyttä luotiin valitulla ajanjakson aikana
- **Rekisteröitymissuhteet maittain** — Maantieteellinen jakautuminen rekisteröitymisiin
- **Rekisteröitymissuhteet lomakkeiden mukaan** — Kuinka monta asiakasta on rekisteröity kunkin kassalohkon kautta verrattuna hallinnikkojen lisäämiin
- **Viestiyleisimmät sivustot** — Useimmin vieraillut sivustot verkostossasi (vaatii vierailumäärän laskennan aktivoimisen Ultimate Multisite → Asetukset → Sivustot → Aktivoi vierailun rajoituksen ja laskennan)

![Dashboard statistics](/img/admin/um-dashboard-stats.png)
