# Menyesuaikan Panel Admin Anda dengan PRO Theme

Apabila membuat kerja untuk klien, kita sering perlu menyediakan pengalaman yang disesuaikan dalam dashboard admin WordPress untuk mereka. Jika anda mencari penampilan yang lebih profesional, PRO Theme mungkin dapat membantu anda.

PRO Theme ialah tema admin yang dibangunkan untuk membantu anda menyesuaikan Dashboard klien anda. Ia juga merupakan sebahagian daripada plugin premium teras Ultimate Multisite.

Selepas memasangnya, pergi ke halaman _Plugins_ anda dan klik untuk mengaktifkannya pada Network anda.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Aktifkan secara network-wide di halaman Plugins anda_

_**NOTA:**_ _Jika anda hanya memasangnya tanpa mengaktifkan secara network-wide, ia akan muncul di akaun klien anda sebagai pilihan yang boleh ditanda untuk menambah tema tersebut kepada pengguna itu sahaja._

Kini, antara muka admin anda telah direka semula sepenuhnya! Anda juga mempunyai menu baharu di lajur kiri yang dipanggil PRO Theme.
