# Meletakkan Iklan di Rangkaian Anda dengan Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector ialah **add-on percuma** yang memberi anda alat untuk meletakkan iklan pada pelan anda terus dari halaman edit pelan tersebut.

Selepas memasang Ultimate Multisite: Ad Injector, pergi ke halaman _Plugins_ anda untuk mengaktifkannya.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Klik butang Network Activate_

Setelah add-on diaktifkan, kotak meta _Advanced Options_ pada skrin edit pelan akan mempunyai pilihan baharu yang dipanggil _Ad Settings_.

![Ad Settings pada skrin edit pelan](/img/config/product-edit-full.png)

_Kini anda boleh mengedit tetapan iklan untuk setiap pelan anda_

Pada tab tersebut, anda boleh menambah kod iklan untuk catatan dan halaman di bahagian hadapan (front-end), sebelum dan selepas kandungan.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Selepas memasukkan kod iklan anda, ia akan dipaparkan pada catatan dan halaman pelan tersebut._

Anda juga boleh memaparkan iklan pada header Halaman Admin untuk pengguna pelan tersebut.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Pergi ke pilihan Admin Panel Ad Code jika anda mahu memaparkan iklan di bahagian belakang (back-end) pengguna anda_

### Lebih Daripada Sekadar Pengiklanan

Anda juga boleh menggunakan Ultimate Multisite: Ad Injector untuk menawarkan sokongan kepada pelan peringkat tinggi.

Pergi sahaja ke pilihan _Admin Panel Ads_ dan tambah kod HTML untuk alat sokongan anda.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Tambah sokongan pelanggan kepada pelan tertentu_

Dengan cara ini, pengguna boleh menghubungi anda terus dari Admin Panel mereka!
