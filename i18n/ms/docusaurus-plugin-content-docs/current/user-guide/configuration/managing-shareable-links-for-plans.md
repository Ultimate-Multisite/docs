# Menguruskan Pautan Boleh Kongsi untuk Pelan

Ultimate Multisite menawarkan pilihan untuk berkongsi pautan pelan terus dengan pelanggan anda. Untuk melakukan ini, pergi ke mana-mana halaman edit pelan anda dan klik butang _Click to copy the Shareable Link_.

![Halaman edit produk dengan butang pautan boleh kongsi](/img/config/product-edit-full.png)

_Di Halaman Edit Pelan, anda boleh menemui butang untuk mencipta pautan boleh kongsi_

Perlu diingat bahawa mekanisme ini menggunakan kekerapan bil lalai untuk membina pautan. Ini bermakna jika bil lalai anda adalah bulanan, contohnya, anda akan mendapat pautan seperti ini: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Jika anda mahu pautan boleh kongsi itu lalai kepada suku tahunan, anda perlu menambah 3 di hujungnya

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Untuk memilih pelan _premium_ dengan jadual tahunan, anda perlu menambah 1 kemudian perkataan berikut:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Perlu diingat bahawa anda tidak boleh mempunyai hanya satu pautan untuk kekerapan bil yang berbeza. Anda perlu berkongsi satu pautan khusus untuk setiap kekerapan bil anda.

Setelah anda berkongsi pautan dengan pelanggan anda, ia akan mengalihkan mereka ke langkah pertama aliran pendaftaran anda.
