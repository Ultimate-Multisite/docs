# Borang Checkout

Borang Checkout adalah cara yang mudah dan fleksibel untuk mencuba pendekatan berbeza dalam usaha menukar pelawat menjadi pelanggan baharu.

Ultimate Multisite 2.0 menawarkan editor Borang Checkout yang membolehkan anda mencipta seberapa banyak borang yang anda mahu, dengan medan yang berbeza, produk yang ditawarkan, dan sebagainya.

Untuk mengakses ciri ini, pergi ke menu Checkout Forms di bar sisi sebelah kiri.

![Senarai Borang Checkout](/img/config/checkout-forms-list.png)

Di halaman ini, anda boleh melihat semua borang checkout yang anda miliki.

Jika anda ingin mencipta borang baharu, klik sahaja Add Checkout Form di bahagian atas halaman.

Anda boleh memilih salah satu daripada tiga pilihan ini sebagai titik permulaan: single step, multi-step atau blank. Kemudian, klik Go to the Editor.

![Editor Borang Checkout](/img/config/checkout-form-editor.png)

Sebagai alternatif, anda boleh mengedit atau menduplikasi borang sedia ada dengan mengklik pilihan di bawah namanya. Di situ juga, anda akan menemui pilihan untuk menyalin shortcode borang atau memadamkan borang tersebut.

![Tindakan hover borang checkout](/img/config/checkout-form-hover-actions.png)

### Mengedit Borang Checkout

Anda boleh mencipta borang checkout untuk pelbagai tujuan. Dalam contoh ini, kita akan bekerja pada borang pendaftaran.

Selepas menavigasi ke editor borang checkout, berikan nama untuk borang anda (yang akan digunakan untuk rujukan dalaman sahaja) dan slug (digunakan untuk mencipta shortcode, contohnya).

![Editor Borang Checkout](/img/config/checkout-form-editor.png)

Borang terdiri daripada langkah dan medan. Anda boleh menambah langkah baharu dengan mengklik Add New Checkout Step.

![Tambah Langkah Checkout Baharu](/img/config/checkout-form-add-step.png)

Pada tab pertama tetingkap modal, isikan kandungan untuk langkah borang anda. Berikan ID, nama dan penerangan. Item ini kebanyakannya digunakan secara dalaman.

![Langkah borang checkout](/img/config/checkout-form-step.png)

Seterusnya, tetapkan keterlihatan langkah tersebut. Anda boleh memilih antara "Always show", "Only show for logged in users" atau "Only show for guests".

![Langkah borang checkout](/img/config/checkout-form-step.png)

Akhir sekali, konfigurasikan gaya langkah. Ini adalah medan pilihan.

![Langkah borang checkout](/img/config/checkout-form-step.png)

Sekarang, masa untuk menambah medan ke langkah pertama kita. Klik sahaja Add New Field dan pilih jenis bahagian yang anda mahu.

![Langkah borang checkout dengan medan](/img/config/checkout-form-step.png)

Setiap medan mempunyai parameter berbeza yang perlu diisi. Untuk entri pertama ini, kita akan memilih medan "Username".

![Langkah borang checkout](/img/config/checkout-form-step.png)

![Langkah borang checkout](/img/config/checkout-form-step.png)

![Langkah borang checkout](/img/config/checkout-form-step.png)

Anda boleh menambah seberapa banyak langkah dan medan yang anda perlukan. Untuk memaparkan produk anda supaya pelanggan boleh memilih, gunakan medan Pricing Table. Jika anda mahu membenarkan pelanggan memilih templat, tambahkan medan Template Selection. Dan seterusnya.

_**Nota:** username, email, password, site title, site URL, order summary, payment, dan submit button adalah medan wajib untuk mencipta borang checkout._

Semasa anda sedang bekerja pada borang checkout, anda sentiasa boleh menggunakan butang Preview untuk melihat bagaimana pelanggan anda akan melihat borang tersebut. Anda juga boleh bertukar antara paparan sebagai pengguna sedia ada atau pelawat.

![Simpan Borang Checkout](/img/config/checkout-form-save.png)

![Editor Borang Checkout](/img/config/checkout-form-editor.png)

Akhir sekali, di Advanced Options anda boleh mengkonfigurasi mesej untuk halaman "Thank You", menambah snippet untuk menjejak penukaran, menambah CSS tersuai ke borang checkout anda atau mengehadkannya kepada negara tertentu.

![Pilihan Lanjutan](/img/config/checkout-form-advanced.png)

Anda juga boleh mengaktifkan atau menyahaktifkan borang checkout secara manual dengan menogol pilihan ini di lajur kanan, atau memadamkan borang secara kekal.

![Togol aktif](/img/config/checkout-form-active.png)

Jangan lupa simpan borang checkout anda!

![Butang simpan](/img/config/checkout-form-save.png)

Untuk mendapatkan shortcode borang anda, klik Generate Shortcode dan salin hasil yang ditunjukkan pada tetingkap modal.

![Butang simpan dengan shortcode](/img/config/checkout-form-save.png)
