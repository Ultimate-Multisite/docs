# Hook dan Penapis Ultimate Multisite

Ultimate Multisite terdiri daripada banyak fungsi dan pertanyaan pangkalan data. Ia mempunyai hook dan penapis yang membolehkan anda mengubah suai atau menambah kod baharu kepada fungsi sedia ada.

Kami telah menyediakan senarai hook dan penapis ini sebagai rujukan untuk membantu pengguna mahir menyesuaikan rangkaian mereka dengan lebih lanjut. Anda boleh mengakses senarai tersebut dari Github dengan mengklik pautan di bawah.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
