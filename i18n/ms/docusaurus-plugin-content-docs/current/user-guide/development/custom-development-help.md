# Bagaimana Mendapatkan Bantuan Pembangunan Custom daripada Codeable?

Kadang-kadang, kita memerlukan penyesuaian yang hebat untuk menjadikan aliran rangkaian kita tepat seperti yang dikehendaki. Jika itulah yang anda cari, kami mengesyorkan [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), tempat di mana anda boleh mencari pembangun berpengalaman yang mahir dalam Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) ialah platform yang menghubungkan pelanggan dengan pembangun WordPress freelance yang berkelayakan, mampu mengendalikan projek dalam apa jua saiz. Mereka mempunyai proses permohonan yang teliti untuk memastikan anda dipadankan dengan pakar yang benar-benar berkelayakan untuk tugasan yang anda inginkan.

### Bagaimana untuk mencipta projek di Codeable?

Pertama, pergi ke [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) dan klik butang **Start a Project**.

Sekarang, untuk menghantar projek, klik di sebelah kiri pada **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

Dalam "I need help with", pilih **Customizing** :

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

Dalam "My", pilih **Plugin** :

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Kemudian klik butang **Continue**.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Sekarang, tiba masanya untuk menerangkan projek dan keperluan anda dengan sebaik mungkin. Cuba berikan maklumat sebanyak yang boleh!

<!-- Screenshot unavailable: Codeable project description form -->

Jawab soalan-soalan berikut untuk membantu dengan penetapan harga.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Selepas itu, jika anda belum mempunyai akaun [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), cipta satu dan klik butang **Publish My Project**.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

Sebaik sahaja anda menerbitkan projek [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) anda, platform ini akan menghubungkan anda dengan pembangun yang sepadan dengan kemahiran yang diperlukan untuk projek anda.

Proses ini akan membantu mencari orang yang tepat untuk kerja anda, yang mempunyai pengalaman dengan WordPress dan Ultimate Multisite!
