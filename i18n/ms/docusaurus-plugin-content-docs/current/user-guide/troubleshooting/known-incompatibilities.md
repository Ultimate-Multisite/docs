# Ketidakserasian yang diketahui dengan Plugin lain

Selepas memasang dan mengaktifkan Ultimate Multisite, anda mungkin menghadapi masalah disebabkan ketidakserasian yang belum diselesaikan dengan plugin lain.

Kami sedang berusaha untuk mengurangkan senarai ini dengan setiap keluaran baharu, tetapi jika sesuatu plugin tersenarai di sini, anda mungkin perlu menyahaktifkannya untuk memastikan Rangkaian Ultimate Multisite anda berfungsi sepenuhnya.

Berikut adalah senarai plugin tersebut dan apa yang perlu anda lakukan dalam situasi ini:

**Beberapa plugin WPMU DEV** Sebahagian plugin yang _tidak lagi_ diselenggara oleh WPMU DEV diketahui menyebabkan aliran pendaftaran Ultimate Multisite terhenti pada langkah terakhir (pengguna tidak dapat mencipta akaun). Berikut adalah senarai plugin yang menyebabkan masalah tersebut:

  * Signup Language: _Ultimate Multisite mempunyai add-on yang menawarkan fungsi yang sama, dipanggil Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Ini sudah terbina dalam Ultimate Multisite, jadi tidak perlu plugin tambahan._

  * New Blog Templates: _Ini sudah terbina dalam Ultimate Multisite, jadi tidak perlu plugin tambahan._

  * Set Password on Multisite Blog Creation: _Ini sudah terbina dalam Ultimate Multisite, jadi tidak perlu plugin tambahan._

  * Signup TOS: _Ini sudah terbina dalam Ultimate Multisite, jadi tidak perlu plugin tambahan._

  * Anti-Splog: Walaupun kurang berkuasa, ini juga _sudah terbina dalam Ultimate Multisite._

**Nota** : Semua plugin WPMU Dev yang lain boleh digunakan seperti biasa bersama Ultimate Multisite. Contohnya termasuk _Smush_ , _Forminator_ , _Defender,_ dan sebagainya.
