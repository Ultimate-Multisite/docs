# Bagaimana saya boleh mengakses halaman akaun untuk kali pertama?

Selamat datang ke Ultimate Multisite!

Akses halaman akaun anda melalui pautan yang anda terima dalam e-mel selepas membeli plugin kami:

<!-- Screenshot unavailable: Account page email with access link -->

Gunakan e-mel yang anda berikan semasa pembelian dan kata laluan yang anda terima dalam e-mel anda.

<!-- Screenshot unavailable: Login page with email and password fields -->

Oleh kerana ini adalah kali pertama anda log masuk ke akaun ini, anda akan diminta untuk menukar kata laluan anda. Tukarkan kata laluan tersebut dan klik butang _Save_.

<!-- Screenshot unavailable: Change password form with Save button -->

Di halaman akaun anda, anda boleh menguruskan semua butiran akaun seperti bil, pembaharuan, pembatalan, dan pelbagai perkara lain.
