# Bagaimana cara membatalkan langganan saya?

Jika anda mempunyai lesen Ultimate Multisite, anda boleh membatalkan pembaharuannya pada bila-bila masa. Ikuti sahaja langkah-langkah di bawah:


<!-- Screenshot unavailable: Account page email with access link -->

Gunakan e-mel yang anda berikan semasa pembelian dan kata laluan anda.

<!-- Screenshot unavailable: Login page with email and password fields -->

Ini adalah skrin awal Halaman Akaun anda:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

Untuk membatalkan langganan anda, pada menu di sebelah kiri, pergi ke _Renewing & Billing._

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

Klik anak panah di sebelah kanan untuk membuka tetingkap sisi. Kemudian, pilih pilihan yang akan membatalkan **pembaharuan automatik**.

Sistem akan menunjukkan mesej pengesahan kepada anda.

Jika anda pasti mahu membatalkan langganan anda, klik sahaja butang _**Cancel Renewals**_. Selepas tindakan ini, anda akan diminta untuk menjawab tinjauan ringkas.

Selesai! Langganan anda tidak akan diperbaharui secara automatik.

**Anda akan mempunyai kunci yang sah sehingga langganan anda tamat tempoh**. Sekiranya anda ingin mengaktifkan semula langganan anda, anda perlu melakukannya secara manual.
