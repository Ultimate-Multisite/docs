# Laman Web Pelanggan Anda (v2)

Selepas mendaftar di rangkaian anda, pelanggan anda boleh mengakses dashboard mereka.

Selain halaman **Account**, di mana mereka boleh melihat maklumat penting berkaitan pembayaran, keahlian, domain, had pelan dan sebagainya, mereka juga boleh mengakses halaman **My Sites**, di mana mereka boleh melihat dan menguruskan semua laman web mereka. Apa yang perlu mereka lakukan hanyalah klik pada My Sites di dashboard mereka.

![Pautan My Sites di dashboard pelanggan](/img/admin/customers-list.png)

Mereka akan dapat melihat semua laman web mereka, dengan butang **Manage** tepat di bawah setiap satu. Jika mereka klik padanya, mereka akan dialihkan ke **dashboard laman web tersebut**.

Mereka juga boleh melihat laman web mana yang merupakan laman **utama** mereka untuk pelan yang membenarkan mereka memiliki lebih daripada satu laman web.

![Halaman My Sites menunjukkan laman web pelanggan dengan butang Manage](/img/admin/customers-list.png)

Jika mereka ingin menukar laman web utama mereka, mereka perlu pergi ke halaman Account dengan mengklik menu **Account**.

![Pautan menu Account di dashboard pelanggan](/img/admin/customers-list.png)

Kemudian, mereka perlu klik pada **Change Default Site**.

![Butang Change Default Site di halaman account](/img/admin/customers-list.png)

Satu tetingkap pop-up akan muncul. Klik pada senarai drop-down dan pilih laman web yang anda mahu jadikan sebagai laman web utama.

![Pemilihan drop-down Change Default Site](/img/admin/customers-list.png)

Kemudian, klik sahaja pada **Change Default Site** dan selesai!

![Pengesahan pertukaran laman web lalai](/img/admin/customers-list.png)
