# Laman Web Pelanggan Anda (v2) {#your-clients-websites-v2}

Selepas mendaftar pada rangkaian anda, pelanggan anda boleh mengakses Dashboard mereka.

Selain halaman **Account**, tempat mereka boleh melihat maklumat penting berkaitan pembayaran, keahlian, domain, had pelan dan banyak lagi, mereka juga mempunyai akses kepada halaman **Laman Saya**, tempat mereka boleh melihat dan mengurus semua laman web mereka. Apa yang perlu mereka lakukan hanyalah mengklik Laman Saya pada Dashboard mereka.

![Pautan Laman Saya dalam Dashboard pelanggan](/img/account-page/my-sites-link.png)

Mereka akan ditunjukkan semua laman web mereka, dengan butang **Urus** betul-betul di bawah setiap satu. Jika mereka mengkliknya, mereka akan dialihkan ke **Dashboard laman web khusus ini**.

Mereka juga boleh melihat laman web mereka yang mana satu merupakan yang **utama** untuk pelan yang membenarkan mereka mempunyai lebih daripada satu laman web.

![Halaman Laman Saya yang menunjukkan laman web pelanggan dengan butang Urus](/img/account-page/my-sites-page.png)

Berikut ialah paparan penuh antara muka sunting laman:

![Antara muka penuh sunting laman](/img/admin/site-edit-full.png)

Jika mereka mahu menukar laman web utama mereka, mereka perlu pergi ke halaman Account dengan mengklik menu **Account**.

![Pautan menu Account dalam Dashboard pelanggan](/img/account-page/account-menu.png)

Anda juga boleh melihat dan menyunting butiran pelanggan:

![Antara muka sunting pelanggan](/img/admin/customer-edit.png)

Kemudian, mereka perlu mengklik **Tukar Laman Lalai**.

![Butang Tukar Laman Lalai pada halaman akaun](/img/account-page/change-default-site.png)

Tetingkap pop timbul akan muncul. Klik pada senarai juntai bawah dan pilih laman web yang ingin anda jadikan laman web utama.

![Pemilihan juntai bawah Tukar Laman Lalai](/img/account-page/change-default-site-dropdown.png)

Kemudian, klik sahaja **Tukar Laman Lalai** dan selesai!

![Mengesahkan penukaran laman lalai](/img/account-page/change-default-site-confirm.png)
