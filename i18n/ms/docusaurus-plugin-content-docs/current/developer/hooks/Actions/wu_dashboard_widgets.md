# Aksi: wu_dashboard_widgets {#action-wudashboardwidgets}

Membenarkan pembangun plugin untuk menambah widget ke Panel Dashboard Network.

## Parameter {#parameters}

| Nama | Jenis | Penerangan |
|------|------|-------------|
| $tab | `string` | Tab semasa. |
| $screen | `\WP_Screen` | Objek skrin. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Contoh halaman admin Ultimate Multisite. |

### Sejak {#since}

- 2.0.0
