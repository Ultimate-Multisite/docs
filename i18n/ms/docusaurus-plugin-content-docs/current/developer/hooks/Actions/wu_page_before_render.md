# Action: wu_page_before_render

Membenarkan pembangun plugin untuk menambah kandungan tambahan sebelum halaman dipaparkan.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $page_id | `string` | ID halaman ini. |
| $page | `object` | Objek halaman. |
