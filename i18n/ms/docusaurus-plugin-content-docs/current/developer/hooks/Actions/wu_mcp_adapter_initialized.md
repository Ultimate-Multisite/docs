# Action: wu_mcp_adapter_initialized

Dijalankan selepas adapter MCP diinisialisasi.

Membenarkan plugin dan tema lain mendaftarkan keupayaan mereka sendiri.

## Parameter

| Nama | Jenis | Penerangan |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | Instans adapter MCP. |

- 2.5.0
