# Aksi: wp_ultimo_load

Dipicu apabila semua kebergantungan (dependencies) telah dimuatkan.

Membenarkan pembangun plugin untuk menambah fungsi baharu. Contohnya, sokongan untuk penyedia Hosting baharu, dan lain-lain.
