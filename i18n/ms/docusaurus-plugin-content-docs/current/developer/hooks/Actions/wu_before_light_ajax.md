# Aksi: wu_before_light_ajax

Dalam sesetengah kes, kita mungkin perlu memuatkan kebergantungan tambahan untuk mengendalikan aksi.

Aksi ini boleh digunakan untuk memuatkan kebergantungan tambahan apabila diperlukan.
