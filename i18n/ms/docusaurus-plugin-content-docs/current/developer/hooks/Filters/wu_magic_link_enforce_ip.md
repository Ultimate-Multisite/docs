# Filter: wu_magic_link_enforce_ip

Menapis sama ada untuk menguatkuasakan pengesahan alamat IP.

Tetapkan kepada false untuk membenarkan token berfungsi dari rangkaian yang berbeza. Ini mengurangkan keselamatan tetapi meningkatkan kebolehgunaan (contohnya, untuk pengguna mudah alih yang menukar rangkaian).

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $enforce | `bool` | Sama ada untuk menguatkuasakan padanan alamat IP. |
