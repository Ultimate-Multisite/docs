# Filter: wu_plan_product_types

Menapis jenis produk yang perlu dianggap sebagai pelan.

Filter ini membenarkan *addon* untuk mendaftarkan jenis produk tambahan yang perlu dikenali sebagai pelan semasa pengesahan dan pengasingan.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | Array jenis produk yang perlu dianggap sebagai pelan. |
