# Filter: wu_magic_link_enforce_user_agent

Menapis sama ada untuk menguatkuasakan pengesahan user agent.

Tetapkan kepada `false` untuk membenarkan token berfungsi merentasi pelayar/peranti yang berbeza. Ini mengurangkan keselamatan tetapi meningkatkan kebolehgunaan.

## Parameter {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $enforce | `bool` | Sama ada untuk menguatkuasakan padanan user agent. |

### Sejak {#since}

- 2.0.0
