# Filter: wu_apply_plan_limits

Membenarkan pemaju plugin untuk memintas had-had tersebut.

Anda boleh menggunakan filter ini untuk menjalankan kod sesuka hati sebelum sebarang had dimulakan. Jika filter mengembalikan sebarang nilai yang dianggap benar (*truthy value*), proses akan diteruskan. Jika ia mengembalikan sebarang nilai yang dianggap salah (*falsy value*), kod akan berhenti dan tiada hook di bawah akan dijalankan.

### Sejak

- 1.7.0
