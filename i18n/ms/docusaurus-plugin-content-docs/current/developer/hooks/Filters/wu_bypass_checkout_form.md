## Parameter

| Nama | Jenis | Penerangan |
|------|------|-------------|
| $bypass | `string\|bool` | Sama ada kita perlu memintas borang pembayaran (checkout form) atau sebaliknya, memberikan rentetan (string) untuk dipaparkan menggantikan borang tersebut. |
| $atts | `array` | Parameter untuk blok/shortcode pembayaran. |
