# Filter: wu_mcp_include_ability

Filter ini digunakan untuk menimpa keputusan sama ada kemampuan (ability) MCP perlu dimasukkan.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $include | `bool` | Sama ada kemampuan itu perlu dimasukkan (True/False). |
| $ability_id | `string` | ID kemampuan yang dimaksudkan. |

- 2.4.8
