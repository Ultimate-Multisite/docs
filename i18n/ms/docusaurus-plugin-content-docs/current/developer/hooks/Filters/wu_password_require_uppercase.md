# Filter: wu_password_require_uppercase

Filter sama ada memerlukan huruf besar dalam kata laluan.

## Parameter {#parameters}

| Nama | Jenis | Penerangan |
|------|------|-------------|
| $require | `bool` | Sama ada memerlukan huruf besar. Lalatnya (default) adalah `true` apabila peraturan dikuatkuasakan. |
| $defender_active | `bool` | Sama ada Defender Pro Strong Password aktif. |
