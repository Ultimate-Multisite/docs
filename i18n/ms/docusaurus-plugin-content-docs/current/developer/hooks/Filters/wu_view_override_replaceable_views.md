# Filter: wu_view_override_replaceable_views

Membenarkan pembangun untuk menambah folder tambahan ke dalam senarai yang boleh diganti.

Berhati-hati, kerana membenarkan folder tambahan mungkin menyebabkan salinan yang lapuk dimuatkan, bukannya versi Ultimate Multisite.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $replaceable_views | `array` | Senarai folder yang dibenarkan. |
