# Filter: wu_documentation_get_link

Membenarkan pembangun plugin untuk menapis pautan.

Tidak pasti bagaimana ini boleh berguna, tetapi tidak mengapa untuk menyediakannya.

## Parameter {#parameters}

| Nama | Jenis | Penerangan |
|------|------|-------------|
| $link | `string` | Pautan yang didaftarkan |
| $slug | `string` | Slug yang digunakan untuk mendapatkan pautan |
| $default_link | `string` | Pautan lalai yang didaftarkan |

### Sejak {#since}

- 1.7.0
