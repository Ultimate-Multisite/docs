# Filter: wu_independent_billing_cycle_product_types {#filter-wuindependentbillingcycleproducttypes}

Menapis jenis produk yang mempunyai kitaran pengebilan bebas.

Produk dengan kitaran pengebilan bebas tidak dipaksa untuk sepadan dengan tempoh pengebilan pelan dalam troli. Ini berguna untuk produk seperti pendaftaran domain yang sentiasa dibayar secara tahunan tanpa mengira sama ada pelan tersebut bulanan atau tahunan.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | Senarai jenis produk dengan kitaran pengebilan bebas. |
