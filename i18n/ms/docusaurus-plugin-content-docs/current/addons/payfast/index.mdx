# PayFast

Ultimate Multisite: Gateway Payfast

Menambah Payfast – sebuah gateway pembayaran dalam talian yang pantas dan selamat di Afrika Selatan – sebagai pilihan pembayaran dalam Ultimate Multisite.

## Pemasangan {#installation}

1. Muat naik 'ultimate-multisite-payfast' ke direktori '/wp-content/plugins/'
2. Aktifkan plugin melalui menu 'Plugins' dalam WordPress
3. Ikuti Wizard langkah demi langkah untuk menyediakan plugin

## Panduan Penggunaan {#usage-guide}

PayFast ialah pemproses pembayaran dalam talian terkemuka untuk perniagaan di Afrika Selatan. Ia berdedikasi untuk membantu peniaga dalam talian berkembang dengan menjadikan transaksi lancar dan selamat. Mengikut permintaan pengguna Ultimate Multisite, kami memutuskan untuk mencipta add-on yang akan membolehkan integrasi dengan PayFast. Artikel ini akan menunjukkan cara menyediakan PayFast sebagai gateway pembayaran anda untuk menerima bayaran dalam rangkaian Ultimate Multisite anda.

Langkah pertama ialah memasang add-on ini.

# Cara memasang add-on Payfast:

Pergi ke halaman add-ons dan pilih add-on Ultimate Multisite: Payfast Gateway.

![Add-ons page](/img/admin/addons-page.png)

Tingkap add-on akan muncul. Klik pada Install Now.

![Add-ons page with install option](/img/admin/addons-page.png)

Selepas pemasangan selesai, anda akan dialihkan ke halaman plugins. Di sini, hanya klik pada Network Activate dan add-on _Payfast_ akan diaktifkan pada rangkaian anda.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# Menyediakan PayFast

Selepas memasang add-on, anda kini boleh pergi ke tetapan Ultimate Multisite dan mula mengkonfigurasi PayFast.

Di bawah halaman **Ultimate Multisite settings**, anda boleh mencari tab "Payments". Di bawah tab ini, anda akan menemui "**Payfast** " bersama butang toggle untuk mengaktifkan atau menyahaktifkan gateway pembayaran ini.

![Payment gateway settings](/img/config/settings-payment-gateways.png)Teruskan dan klik butang toggle untuk mengaktifkannya. Ia kemudian akan mengembangkan kotak itu untuk menunjukkan medan berbeza yang perlu anda isi seperti **Mechant ID, Merchant Key dan PassPhrase.** Anda boleh mendapatkan maklumat ini dari akaun **Payfast** anda

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Langkah seterusnya ialah log masuk ke akaun PayFast anda untuk mendapatkan maklumat yang anda perlukan.

Upon logging in, you can immediately see the credentials you need from your **Payfast** account dashboard.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->Anda hanya perlu salin dan tampal kredensial tersebut serta isikan butiran pada tetapan Ultimate Multisite anda. Kemudian klik butang **"Save settings** " untuk menyimpan perubahan.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Itu sahaja, integrasi pembayaran PayFast anda telah selesai!
