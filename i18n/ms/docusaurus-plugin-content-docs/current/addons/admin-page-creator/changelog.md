# Admin Page Creator Changelog {#admin-page-creator-changelog}

### Versi 1.9.0 - Dikeluarkan pada 2026-01-18 {#version-190---released-on-2026-01-18}

* Memperbaiki: Ralat fatal semasa mengaktifkan.
* Menyusun semula: Kelas sumber kandungan untuk menggunakan ciri Singleton dengan inisialisasi pembina induk yang betul.
* Memperbaiki: Ralat kompilasi template Vue.js yang menghalang pemilih jenis kandungan daripada dipaparkan.
* Memperbaiki: Properti data Vue yang hilang (template_id, external_link_url_embedable, external_link_url_checking).
* Memperbaiki: Editor TinyMCE tidak diinisialisasi dengan betul semasa memuatkan halaman.
* Memperbaiki: Fail sokongan Divi tidak dimuat secara automatik melalui Composer.
* Memperbaiki: Sintaks PHP yang rosak dalam kelas sokongan Divi.
* Penambahbaikan: Organisasi dan minifikasi aset JavaScript.
* Penambahbaikan: Keserasian dengan versi WordPress dan page builder terkini.
