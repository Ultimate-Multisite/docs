# Filter: wu_apc_should_display_admin_menu {#filter-wuapcshoulddisplayadminmenu}

## Parameter {#parameters}

| Nama | Jenis | Penerangan |
|------|------|-------------|
| $should_display_admin_menu | `bool` | Jika kita perlu memaparkan halaman tersebut. |
| $network_active | `bool` | Semak sama ada plugin ini aktif untuk admin rangkaian (network admin). |
