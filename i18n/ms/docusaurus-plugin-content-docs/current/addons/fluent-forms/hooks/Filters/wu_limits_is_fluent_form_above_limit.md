# Filter: wu_limits_is_fluent_form_above_limit {#filter-wulimitsisfluentformabovelimit}

Memeriksa sama ada jenis borang tertentu dibenarkan pada pelan ini. Ini membenarkan pembangun plugin untuk menapis nilai yang dikembalikan.

## Parameter {#parameters}

| Nama | Jenis | Penerangan |
|------|------|-------------|
| $value | `bool` | Sama ada jenis borang melebihi had atau tidak |
| $form_type | `string` | Jenis borang yang sedang diperiksa |
| $form_count | `int` | Kiraan borang semasa |
| $quota | `int` | Kuota yang dibenarkan |
