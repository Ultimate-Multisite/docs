# Fluent Forms

Add-on ini membolehkan anda menetapkan had (limits) untuk Fluent Forms pada Ultimate Multisite, sama ada pada produk, keahlian (memberships), atau situs. Fluent Forms menggunakan jadual tersuai (custom tables) berbanding jenis pos tersuai (custom post types), jadi pilihan had biasa tidak berfungsi dengan Fluent Forms.

## Ciri-ciri

*   Tambah had untuk Fluent Forms atau Conversational Fluent Forms
*   Pelanggan menerima mesej mesra yang mengingatkan mereka untuk menaik taraf pelan mereka apabila mereka mencapai had tersebut
