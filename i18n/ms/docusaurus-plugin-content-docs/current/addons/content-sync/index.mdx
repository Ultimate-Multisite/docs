# Sinkronisasi Kandungan

Multisite Ultimate: Sinkronisasi Kandungan

Addon ini untuk Multisite Ultimate membolehkan pentadbir rangkaian menyegerakkan kandungan dari mana-mana tapak ke tapak lain dalam rangkaian. Ia menyokong semua jenis pos termasuk pos, halaman, jenis pos tersuai, dan templat Elementor. Addon ini merangkumi antara muka wizard berbilang langkah untuk memilih kandungan dan tapak sasaran, dengan pemprosesan kelompok (batch processing) dan maklum balas kemajuan masa nyata.

## Ciri-ciri

* Pemilihan Kandungan: Grid kad visual dengan pratonton (thumbnails) dan penapis jenis pos
* Semua Jenis Pos: Sinkronisasi pos, halaman, produk, templat Elementor, dan sebarang jenis pos tersuai
* Pemilihan Tapak: Pilih tapak mana yang kandungan perlu disegerakkan, dengan pengecualian tapak sumber dan pra-pemilihan tapak anak
* Pemprosesan Kelompok (Batch Processing): Menyegerakkan ke 10 tapak pada satu masa dengan kemas kini kemajuan
* Sinkronisasi Media: Menyalin dan memetakan semula fail media (imej, video) secara automatik dari kandungan pos ke tapak sasaran
* Sinkronisasi Taksonomi: Mengekalkan kategori, tag, dan taksonomi tersuai
* Sokongan Elementor: Sokongan penuh untuk templat Elementor dengan penjanaan semula CSS dan pemetaan semula ID templat
* Penyelesaian Konflik: Pilihan untuk menggantikan kandungan sedia ada atau melangkauinya
* Pengurusan Cache: Membersihkan cache Elementor secara automatik pada tapak sasaran selepas sinkronisasi

## Keperluan

* WordPress 5.3 atau lebih tinggi
* PHP 7.4 atau lebih tinggi
* Plugin Multisite Ultimate (aktif)
* Plugin Elementor (pilihan, diperlukan untuk sinkronisasi templat Elementor)

## Pemasangan

1. Muat naik 'ultimate-multisite-content-sync' ke direktori '/wp-content/plugins/'
2. Aktifkan plugin melalui menu 'Plugins' dalam Pentadbir Rangkaian WordPress anda
3. Konfigurasikan tetapan dalam Ultimate Multisite → Settings → Content Sync
4. Navigasi ke Ultimate Multisite → Content Sync untuk memulakan sinkronisasi
