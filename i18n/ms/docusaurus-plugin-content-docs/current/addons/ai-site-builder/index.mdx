# AI Site Builder {#ai-site-builder}

ultimate-multisite-ai-site-builder

## Keperluan {#requirements}

- WordPress 5.3 atau lebih tinggi
- PHP 7.4 atau lebih tinggi
- Plugin Ultimate Multisite (aktif)

## Pemasangan {#installation}

1. Muat naik fail addon ke direktori `/wp-content/plugins/` anda
2. Aktifkan plugin melalui menu 'Plugins' dalam WordPress
3. Konfigurasikan tetapan addon dalam pentadbiran rangkaian (network admin)

## Pembangunan {#development}

### Persediaan {#setup}

```bash
# Pasang kebergantungan
composer install
npm install

# Jalankan ujian
npm run test

# Jalankan semakan piawaian kod
vendor/bin/phpcs
vendor/bin/phpstan

# Betulkan isu gaya kod
vendor/bin/phpcbf

# Bina untuk pengeluaran
npm run build
```

```
ultimate-multisite-ai-site-builder/
```

### Pengujian {#testing}

Addon ini merangkumi rangka kerja pengujian yang komprehensif:

- **Ujian Unit (Unit Tests)**: Ujian berasaskan PHPUnit dengan set ujian WordPress
- **Kelas Ujian Asas (Base Test Class)**: Menyediakan kaedah bantuan untuk senario ujian biasa
- **Liputan Ujian (Test Coverage)**: Dikonfigurasikan untuk menjejaki liputan kod
- **Pengujian Multisite (Multisite Testing)**: Ujian dijalankan dalam persekitaran multisite

### Piawaian Kod {#code-standards}

- **Piawaian Kod WordPress**: Dikuatkuasakan melalui PHPCS
- **Keserasian PHP 7.4+**: Menyokong ciri PHP moden
- **Analisis Statik**: Integrasi PHPStan untuk pemeriksaan jenis (type checking)
- **Pemodenan Kod**: Peraturan Rector untuk peningkatan PHP

### Sistem Bina (Build System) {#build-system}

- **Pemprosesan Aset**: Peminifan untuk fail CSS/JS
- **Terjemahan**: Penghasilan fail POT
- **Pembungkusan**: Penciptaan arkib untuk pengedaran
- **Mod Pembangunan**: Aset tanpa peminifan untuk penyahpepijatan (debugging)

## Konfigurasi {#configuration}

Addon ini menyokong pelbagai pilihan konfigurasi yang boleh diakses melalui antara muka pentadbiran rangkaian (network admin).

## Hooks dan Filters {#hooks-and-filters}

### Actions (Tindakan) {#actions}

- `ultimate-multisite-ai-site-builder_init` - Dijalankan selepas inisialisasi addon
- `ultimate-multisite-ai-site-builder_loaded` - Dijalankan selepas semua fail addon dimuatkan

### Filters (Penapis) {#filters}

- `ultimate-multisite-ai-site-builder_settings` - Ubah tetapan addon
- `ultimate-multisite-ai-site-builder_enabled` - Menimpa status aktif/tidak aktif addon
