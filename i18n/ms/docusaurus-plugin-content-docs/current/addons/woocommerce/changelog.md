Versi 2.0.2 - Dilancar pada 2025-07-05

* Pembaikan: Mengekalkan status keahlian semasa proses penurunan taraf.

* Pembaikan: Menambah tarikh langganan mula untuk digunakan dalam langganan Woocommerce.
