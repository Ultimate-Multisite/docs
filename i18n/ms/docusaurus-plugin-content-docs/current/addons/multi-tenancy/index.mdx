# Multi-Tenancy

Mengagihkan subsites merentasi pelayan dan pangkalan data untuk prestasi dan keselamatan

## Keperluan

- WordPress 5.3 atau lebih tinggi
- PHP 7.4 atau lebih tinggi
- Plugin Multisite Ultimate (aktif)

## Pemasangan

1. Muat naik fail addon ke direktori `/wp-content/plugins/` anda
2. Aktifkan plugin melalui menu 'Plugins' dalam WordPress
3. Konfigurasi tetapan addon dalam pentadbir rangkaian (network admin)

## Pembangunan

### Persediaan

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Struktur Projek


### Pengujian

Addon ini merangkumi rangka kerja pengujian yang komprehensif:

- **Unit Tests**: Ujian berasaskan PHPUnit dengan suite ujian WordPress
- **Base Test Class**: Menyediakan kaedah bantuan untuk senario ujian biasa
- **Test Coverage**: Dikonfigurasi untuk menjejaki liputan kod
- **Multisite Testing**: Ujian dijalankan dalam persekitaran multisite

### Standard Kod

- **WordPress Coding Standards**: Dikuatkuasakan melalui PHPCS
- **PHP 7.4+ Compatibility**: Menyokong ciri-ciri PHP moden
- **Static Analysis**: Integrasi PHPStan untuk pemeriksaan jenis (type checking)
- **Code Modernization**: Peraturan Rector untuk peningkatan PHP

### Sistem Bina (Build System)

- **Asset Processing**: Peminification untuk fail CSS/JS
- **Translation**: Penghasilan fail POT
- **Packaging**: Penciptaan arkib untuk pengedaran
- **Development Mode**: Aset yang tidak dipinifikasi untuk tujuan penyahpepijatan (debugging)

## Konfigurasi

Addon ini menyokong pelbagai pilihan konfigurasi yang boleh diakses melalui antara muka pentadbir rangkaian (network admin).

## Hooks dan Filters

### Actions

- `ultimate-multisite-addon-template_init` - Dipicu selepas inisialisasi addon
- `ultimate-multisite-addon-template_loaded` - Dipicu selepas semua fail addon dimuatkan

### Filters

- `ultimate-multisite-addon-template_settings` - Ubah tetapan addon
- `ultimate-multisite-addon-template_enabled` - Menimpa status aktif/tidak aktif addon
