# Action: wu_mt_providers_registered

Dijalankan selepas penyedia lalai (default providers) didaftarkan.

Gunakan hook ini untuk mendaftarkan penyedia tersuai (custom providers).

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $manager | `\Provider_Manager` | Contoh (instance) pengurus penyedia. |
