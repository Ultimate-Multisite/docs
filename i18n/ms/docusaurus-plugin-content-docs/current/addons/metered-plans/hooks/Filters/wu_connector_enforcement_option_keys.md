# Filter: wu_connector_enforcement_option_keys {#filter-wuconnectorenforcementoptionkeys}

Menapis senarai kunci pilihan penyedia AI yang dikuatkuasakan dari tapak utama.

Tambahkan kunci pilihan untuk plugin penyedia AI pihak ketiga atau tersuai supaya tetapan mereka juga diwarisi dari tapak utama pada subsites.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $keys | `string[]` | Nama kunci pilihan yang perlu dikuatkuasakan (sudah termasuk kunci konektor yang ditemui secara dinamik dan EXTRA_PROVIDER_OPTIONS). |
