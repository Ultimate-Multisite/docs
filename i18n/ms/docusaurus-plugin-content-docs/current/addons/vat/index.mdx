# VAT

Multisite Ultimate: VAT Eropah

Menambah ciri dan alat tambahan kepada pengurusan cukai lalai Multisite Ultimate untuk menyokong VAT Eropah dengan lebih baik.
Secara automatik menambah kadar VAT EU ke halaman Kadar Cukai dari

## Pemasangan

1. Muat naik 'ultimate-multisite-vat' ke direktori '/wp-content/plugins/'
2. Aktifkan plugin melalui menu 'Plugins' dalam WordPress
3. Pergi ke Ultimate Multisite > Settings > Taxes dan Aktifkan EU Vat.
4. Klik pada Tax Rates dan Kemas Kini Kadar VAT EU di bahagian bawah, kemudian Simpan Kadar Cukai.
5. Selesai!
