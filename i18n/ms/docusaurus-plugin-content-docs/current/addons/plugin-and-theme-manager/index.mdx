# Plugin & Theme Manager {#plugin--theme-manager}

Ultimate Multisite: Plugin and Theme Manager

Edit atau sembunyikan maklumat meta (tajuk, pengarang, gambar mini, dan penerangan) bagi plugin dan tema yang telah dipasang. Cipta kategori untuk plugin dan tema bagi membolehkan pengguna anda menapisnya dan menambah halaman Plugins tersuai yang cantik ke panel pengguna anda.

Untuk menggunakan plugin ini, anda hanya perlu pergi ke halaman Plugins atau Themes pada Network Admin anda dan klik pada "Edit Plugin/Theme info".

## Installation {#installation}

1. Muat naik 'ultimate-multisite-plugin-and-theme-manager' ke direktori '/wp-content/plugins/'
2. Aktifkan plugin melalui menu 'Plugins' dalam Network Admin WordPress anda

## Usage Guide {#usage-guide}

Ultimate Multisite: Plugin and Theme Manager ialah add-on Ultimate Multisite **premium** yang berkuasa yang memberikan anda kawalan penuh ke atas cara pengguna rangkaian anda melihat dan berinteraksi dengan pilihan plugin dan tema pada rangkaian anda.

Add-on ini berfungsi untuk tiga tujuan:

  1. Ia membolehkan anda mengubah metadata sepenuhnya bagi plugin dan tema. Ini termasuk nama, penerangan, nama pengarang, dan malah gambar mini!

  2. Ia membolehkan anda mengkategorikan plugin dan tema, untuk membolehkan pengguna anda menyusunnya dengan mudah.

  3. Ia memberikan anda pilihan untuk menggantikan halaman plugins lama yang membosankan dengan halaman baharu seperti tema, dengan pengisihan mengikut kategori.

Selepas memuat turun dan memasangnya, tab baharu — _Add-on Settings_ — akan dipaparkan pada halaman Ultimate Multisite Settings.
