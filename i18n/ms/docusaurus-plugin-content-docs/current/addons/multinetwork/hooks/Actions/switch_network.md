# Action: switch_network {#action-switchnetwork}

Dipicu apabila konteks rangkaian semasa ditukar.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $new_network_id | `int` | ID rangkaian yang ditukar kepadanya. |
| $old_network_id | `int` | ID rangkaian yang sebelum ini aktif. |
