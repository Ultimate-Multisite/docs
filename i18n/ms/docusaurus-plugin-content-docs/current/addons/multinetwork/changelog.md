# Log Perubahan Multi-Rangkaian

### 1.3.0 {#130}
* Baharu: Pratinjau templat rangkaian — semak dan pratonton templat rangkaian dengan panel langsung sebelum membeli.
* Baharu: Pemilih templat rangkaian semasa pembayaran — pelanggan boleh memilih templat rangkaian semasa aliran pembayaran pendaftaran.
* Baharu: SSO pautan ajaib merentasi rangkaian — pelanggan boleh mengakses sub-rangkaian dengan lancar melalui pengesahan pautan ajaib.
* Baharu: Pengklonan media rangkaian — media tapak web disalin dengan betul apabila mengklon rangkaian daripada templat.
* Pembaikan: Pengklonan rangkaian kini berfungsi tanpa tambahan multi-tenancy; keistimewaan super admin diberikan semula selepas klon.
* Pembaikan: Penciptaan rangkaian kini serasi dengan WooCommerce dan semua gerbang pembayaran.
* Pembaikan: Nombor port kini dimasukkan dalam penjanaan domain sandaran.
* Pembaikan: CSS pemilih templat dipindahkan ke helaian gaya yang diatur (enqueued stylesheet) untuk paparan yang betul pada semua tema.
* Pembaikan: Perlindungan terhadap kunci `network_id` yang tidak ditakrifkan dalam skop pertanyaan tapak web.
* Pembaikan: Semakan kebenaran rangkaian sebelum menukar konteks menghalang ralat sifat jadual `wpdb`.
