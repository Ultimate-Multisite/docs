# الوكلاء المدمجون {#built-in-agents}

يأتي Gratis AI Agent v1.9.0 مزودًا بخمسة وكلاء مدمجين، كلٌ منهم مهيأ مسبقًا بمجموعة مركزة من الأدوات، ومطالبة نظام مخصصة، واقتراحات بداية مناسبة للمهام الشائعة في ذلك المجال. يؤدي التبديل بين الوكلاء إلى تغيير ما يستطيع المساعد فعله وكيفية استجابته — دون أي إعداد من جانبك. يمكن لـ Superdav AI Agent v1.18.0 إضافة أدوات تراعي الجدولة، وسجلات تذكير، وبوابات موافقة، وإشعارات SMS إلى مسارات العمل هذه عند إعداد عمليات التكامل ذات الصلة.

## ما هو الوكيل؟ {#what-is-an-agent}

كل وكيل هو ملف تعريف إعدادات مسمى يجمع بين:

- **الأدوات** — القدرات التي يُسمح للوكيل باستدعائها (مثلًا، لدى كاتب المحتوى وصول إلى قدرات إنشاء المقالات؛ ولدى استوديو التصميم وصول إلى قدرات CSS و theme.json)
- **مطالبة النظام** — تعليمات تحدد نبرة الوكيل وأولوياته وقيوده
- **الاقتراحات** — مطالبات مكتوبة مسبقًا تظهر في واجهة الدردشة لمساعدتك على البدء بسرعة

## الوصول إلى منتقي الوكلاء {#accessing-the-agent-picker}

1. افتح لوحة **Gratis AI Agent** في الشريط الجانبي لإدارة WordPress.
2. انقر على **أيقونة الوكيل** في أعلى يسار ترويسة الدردشة (تتغير الأيقونة لتعكس الوكيل النشط).
3. يفتح **منتقي الوكلاء** كتراكب جدول نموذج. يُدرج كل وكيل مع أيقونته واسمه ووصف من سطر واحد.
4. انقر على صف وكيل لتفعيله. يتم تحديث ترويسة الدردشة فورًا.

يمكنك أيضًا تبديل الوكلاء أثناء المحادثة — تسري مطالبة النظام الخاصة بالوكيل الجديد ابتداءً من الرسالة التالية.

## الوكلاء الخمسة المدمجون {#the-five-built-in-agents}

### كاتب المحتوى {#content-writer}

**التركيز:** إنشاء وتحرير المقالات والصفحات ونماذج الاتصال.

**الأدوات المتاحة:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`. عند تفعيل تكاملات Superdav AI Agent v1.18.0، قد تتوفر أيضًا سياقات التقويم المهيأة، وبوابات الموافقة، والتذكيرات، وأدوات إشعارات SMS لمسارات العمل المعتمدة.

**ما يجيده:**
- صياغة ونشر مقالات مدونة من موجز أو مخطط
- إنشاء دفعات من صفحات الهبوط لموقع جديد
- بناء نماذج اتصال واستفسار
- تعيين الصور البارزة للمقالات من URL أو بحث
- صياغة رسائل متابعة للأحداث من سياق Google Calendar المهيأ، ثم التوقف مؤقتًا للحصول على الموافقة قبل إرسال الإشعارات

**اقتراحات البداية:**
- *اكتب مقال مدونة من 500 كلمة حول فوائد WordPress متعدد المواقع.*
- *أنشئ صفحات نبذة عنا، والخدمات، والاتصال وانشرها.*
- *أضف نموذج استفسار حجز إلى صفحة الاتصال.*
- *صغ تذكيرًا للحاضرين في حدث التقويم المهيأ ليوم غد وانتظر الموافقة قبل إرساله.*

---

### منشئ الموقع {#site-builder}

**التركيز:** إنشاء موقع ويب شامل من مطالبة واحدة.

**الأدوات المتاحة:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`. مع Superdav AI Agent v1.18.0، قد تتوفر أدوات الخدمة المُدارة، والموافقة، والتذكير، والتقويم، و SMS المهيأة حيث يفعّلها المسؤولون.

**ما يجيده:**
- إنشاء خطة بناء موقع متعددة المراحل لنوع عمل موصوف
- تنفيذ كل مرحلة ذاتيًا — البنية، والمحتوى، والتنقل، والتصميم
- التعافي من الأخطاء أثناء الخطة دون الحاجة إلى تدخل يدوي
- تثبيت الإضافات الموصى بها كجزء من البناء
- إنشاء نماذج اتصال مباشرة من واجهة الدردشة (Superdav AI Agent v1.10.0+)
- تنسيق تذكيرات الإطلاق أو متابعة الحاضرين دون إشعارات مكررة عند تفعيل بوابات الموافقة وسجلات التذكير

**اقتراحات البداية:**
- *ابنِ موقع معرض أعمال للتصوير الفوتوغرافي مع نوع مقال للمعرض، وصفحة حجز، ونموذج اتصال.*
- *أنشئ موقع مطعم مع قائمة طعام على الإنترنت، وساعات العمل، ونموذج استفسار لحجز طاولة.*
- *أعدّ موقعًا للاستشارات المستقلة مع صفحات خدمات، وقسم معرض أعمال، ومدونة.*
- *أضف نموذج اتصال إلى صفحة الاتصال باستخدام منشئ الموقع.*
- *بعد الموافقة على قائمة تحقق إطلاق الموقع، أرسل تذكير SMS إلى جهة اتصال صاحب المصلحة المهيأة.*

---

### استوديو التصميم {#design-studio}

**التركيز:** التخصيص البصري — الألوان، والطباعة، و CSS، وأنماط الكتل.

**الأدوات المتاحة:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**ما يجيده:**
- تطبيق إعدادات مسبقة مسماة للقالب (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- ضبط الطباعة العامة ولوحات الألوان بدقة عبر theme.json
- حقن CSS مخصص لتجاوزات خاصة بالعلامة التجارية
- التقاط لقطة شاشة لصفحة ومراجعتها بحثًا عن مشكلات في التصميم

**اقتراحات البداية:**
- *طبّق الإعداد المسبق warm-editorial ثم عيّن اللون الأساسي إلى #2d6a4f.*
- *التقط لقطة شاشة للصفحة الرئيسية وأخبرني بما ستُحسّنه.*
- *أنشئ نمط كتلة بطل قابلًا لإعادة الاستخدام مع صورة خلفية بعرض كامل وعنوان في الوسط.*

### مدير الإضافات {#plugin-manager}

**التركيز:** اكتشاف إضافات WordPress وتثبيتها وإدارتها.

**الأدوات المتاحة:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**ما يجيده:**
- التوصية بأفضل إضافة لحالة استخدام موصوفة
- تثبيت حزم القدرات من السجل
- تصفح كتالوج القدرات المتاحة حسب الفئة

**اقتراحات البداية:**
- *ما أفضل إضافة لدليل عضوية؟*
- *ثبّت حزمة قدرات WooCommerce.*
- *اعرض لي جميع حزم قدرات التجارة الإلكترونية المتاحة.*

---

### مساعد الدعم {#support-assistant}

**التركيز:** الإجابة عن الأسئلة حول محتوى الموقع وإعداداته وتكوين WordPress.

**الأدوات المتاحة:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**ما يبرع في فعله:**
- البحث في إعدادات وخيارات الموقع الحالية
- شرح أنواع المنشورات والتصنيفات والقوائم المكوّنة على الموقع
- الإجابة عن أسئلة "ماذا يفعل هذا الإعداد؟" عبر قراءة القيم الحية
- العمل كطبقة تشخيصية للقراءة فقط قبل إجراء التغييرات

**اقتراحات للبدء:**
- *ما الإضافات والإعدادات النشطة حاليًا على هذا الموقع؟*
- *اعرض كل أنواع المنشورات المخصصة المسجلة على هذا الموقع.*
- *ما قوائم التنقل الموجودة وأين تم تعيينها؟*

## تكاملات أتمتة Superdav {#superdav-automation-integrations}

عند تكوين تكاملات Superdav AI Agent v1.18.0، يمكن للوكلاء المدمجين المشاركة في سير عمل أتمتة أكثر أمانًا ووعيًا بالجدولة:

- **أدوات قراءة Google Calendar** تتيح للوكلاء فحص التقويمات والأحداث المكوّنة قبل صياغة أعمال المتابعة.
- **ربط جهات الاتصال والحضور** يساعد على مطابقة المشاركين في الأحداث مع مستخدمي WordPress أو جهات الاتصال المعروفة.
- **بوابات الموافقة البشرية** توقف الإجراءات الحساسة مؤقتًا حتى يراجعها مستخدم مخوّل ويؤكدها.
- **سجلات التذكير** تمنع الإشعارات المكررة عند إعادة محاولة المهام المجدولة أو تكرارها.
- **إشعارات TextBee SMS** ترسل الرسائل النصية المكوّنة فقط عند تمكين بيانات اعتماد SMS وأذونات سير العمل.

سير العمل الموصى به: اطلب من الوكيل إعداد الرسالة أو الإجراء، وراجع مطالبة الموافقة، ثم اسمح باستئناف الإجراء المعتمد. بالنسبة إلى التذكيرات المتكررة، أبقِ إزالة تكرار التذكيرات مفعّلة حتى لا يتم إشعار الحدث أو جهة الاتصال نفسها مرارًا.

## تخصيص الوكلاء {#customising-agents}

يمكن توسيع كل وكيل مدمج أو استبداله عبر مرشح `gratis_ai_agent_agents`.

### إضافة مطالبة نظام مخصصة إلى وكيل موجود {#adding-a-custom-system-prompt-to-an-existing-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### تسجيل وكيل جديد {#registering-a-new-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

يظهر الوكيل الجديد في منتقي الوكلاء فورًا بعد تشغيل المرشح.

### إزالة وكيل مدمج {#removing-a-built-in-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
