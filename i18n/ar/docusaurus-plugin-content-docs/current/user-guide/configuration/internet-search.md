# البحث عبر الإنترنت {#internet-search}

يضيف Gratis AI Agent v1.5.0 خاصية **البحث عبر الإنترنت** التي تسمح للمساعد الذكي باسترداد معلومات محدثة من الويب أثناء المحادثة. يتم تشغيل هذه الخاصية بواسطة [Brave Search API](https://brave.com/search/api/) أو [Tavily API](https://tavily.com/).

## كيف يعمل {#how-it-works}

عند تفعيل البحث عبر الإنترنت، يمكن للمساعد أن يستعلم تلقائيًا من مزود البحث الذي قمت بتكوينه عندما يحدد أن السؤال يتطلب معلومات حالية أو خارجية — على سبيل المثال، أخبار حديثة، أو أسعار مباشرة، أو وثائق ربما تكون قد تغيرت منذ تاريخ قطع تدريب النموذج.

يتم استرداد النتائج في الوقت الفعلي ويتم إدخالها في سياق المساعد قبل أن يقوم بتوليد الرد. ويشير المساعد إلى أنه استخدم نتائج البحث للإجابة على السؤال.

## تفعيل البحث عبر الإنترنت {#enabling-internet-search}

يتطلب البحث عبر الإنترنت مفتاح واجهة برمجة التطبيقات (API key) من مزود البحث الذي تختاره. لتكوينه:

1. اذهب إلى **Gratis AI Agent → Settings → Advanced**.
2. حدد قائمة **Internet Search Provider** واختر إما **Brave Search** أو **Tavily**.
3. أدخل مفتاح واجهة برمجة التطبيقات (API key) الخاص بك في الحقل المقابل. تظهر روابط التسجيل بجوار كل حقل.
4. انقر على **Save Settings**.

بمجرد حفظ المفتاح، تصبح خاصية البحث عبر الإنترنت متاحة تلقائيًا للمساعد.

## Brave Search {#brave-search}

### الحصول على مفتاح Brave Search API {#obtaining-a-brave-search-api-key}

1. قم بزيارة [Brave Search API page](https://brave.com/search/api/).
2. سجل للحصول على خطة. تتوفر طبقة مجانية بحد طلب شهري.
3. انسخ مفتاح واجهة برمجة التطبيقات (API key) الخاص بك من لوحة تحكم مطوري Brave Search.
4. الصقه في حقل **Brave Search API Key** في إعدادات Gratis AI Agent.

### حدود الاستخدام {#usage-limits}

يتم إصدار فواتير الاستخدام من Brave Search بناءً على عدد الاستعلامات التي تتم. كل رد من المساعد يطلق عملية بحث ويُحتسب كاستعلام واحد. راقب استخدامك في [Brave Search developer dashboard](https://brave.com/search/api/) لتجنب الرسوم غير المتوقعة.

## Tavily {#tavily}

يضيف Superdav AI Agent v1.10.0 **Tavily** كمزود بديل للبحث عبر الإنترنت، حيث يقدم نتائج بحث أكثر ثراءً وقدرات بحث متقدمة.

### الحصول على مفتاح Tavily API {#obtaining-a-tavily-api-key}

1. قم بزيارة [Tavily API page](https://tavily.com/).
2. سجل للحصول على حساب. تتوفر طبقة مجانية بحد طلب شهري.
3. انسخ مفتاح واجهة برمجة التطبيقات (API key) الخاص بك من لوحة تحكم Tavily.
4. الصقه في حقل **Tavily API Key** في إعدادات Gratis AI Agent.

### حدود الاستخدام {#usage-limits-1}

يتم إصدار فواتير الاستخدام من Tavily بناءً على عدد استدعاءات واجهة برمجة التطبيقات (API calls) التي تتم. كل رد من المساعد يطلق عملية بحث ويُحتسب كاستدعاء واحد. راقب استخدامك في [Tavily dashboard](https://tavily.com/) لتجنب الرسوم غير المتوقعة.

## تعطيل البحث عبر الإنترنت {#disabling-internet-search}

قم بإزالة مفتاح واجهة برمجة التطبيقات (API key) من حقل مزود البحث النشط واحفظ التغييرات. لن يتم تقديم خاصية البحث عبر الإنترنت للمساعد بعد ذلك.

:::note
يضيف البحث عبر الإنترنت تأخيرًا (latency) إلى الردود لأن المساعد يجب أن ينتظر نتائج البحث قبل إنشاء الإجابة. بالنسبة لحالات الاستخدام الحساسة للوقت، فكر فيما إذا كان البحث في الوقت الفعلي ضروريًا، أو ما إذا كانت المعرفة المضمنة في المساعد كافية.
:::
