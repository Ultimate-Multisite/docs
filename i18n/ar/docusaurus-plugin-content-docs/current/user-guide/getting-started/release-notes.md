# ملاحظات الإصدار

## الإصدار 2.13.0 — صدر في 2026-06-05 {#version-2130--released-on-2026-06-05}

- جديد: تمت إضافة دعم المستأجر السيادي لتدفقات حساب العميل، والدفع، والفوترة، والموقع، والفاتورة، وتبديل القالب، وربط النطاقات، بحيث يمكن لشبكات المستأجرين توجيه العملاء مرة أخرى إلى الموقع الرئيسي للإجراءات المُدارة.
- جديد: تمت إضافة فحوصات بيانات اعتماد التجديد للعضويات المتكررة بحيث يمكن للبوابات تعطيل التجديد التلقائي عند غياب اتفاقية فوترة محفوظة، أو اشتراك، أو رمز خزنة.
- جديد: تمت إضافة نشر loopback مُتحقق منه عبر HMAC لإنشاء المواقع المعلّقة لجعل تهيئة الموقع من الدفع أكثر موثوقية على الاستضافات التي تتأخر فيها المهام الخلفية.
- جديد: تمت إضافة نقاط توسيع للمطورين لعناوين URL الخاصة بـ SSO، ونطاقات أساس نموذج الدفع، وإنشاء سجلات النطاق تلقائيًا.
- إصلاح: أصبح SSO أكثر موثوقية عبر النطاقات المربوطة، وزيارات الوسيط المجهولة، وتسجيل الخروج، وتعارضات الاعتماديات بين الإضافات.
- إصلاح: يتعافى إنشاء الموقع المعلّق الآن من علامات النشر القديمة ويتجنب ترك العملاء عالقين على شاشة إنشاء الموقع.
- إصلاح: لم تعد سجلات النطاق تُنشأ لنطاقات أساس نموذج الدفع المشتركة، ويتم تخطي مهام الخلفية غير المستخدمة لمزود الاستضافة عندما لا يكون هناك تكامل نشط.
- إصلاح: لم تعد حالات الحافة في الدفع، وعنوان الفوترة، وإعادة تعيين كلمة المرور، والتحقق من البريد الإلكتروني، وتبديل القالب، والجولات، ولوحة تحكم العميل تمنع تدفقات العملاء العادية.
- إصلاح: تحافظ رسائل البث الإلكترونية الآن على خصوصية المستلمين مع تجنب أخطاء SMTP/الإضافة الفادحة عند فشل قوائم المستلمين أو وسائل نقل البريد.
- إصلاح: تتجنب الآن حالات الحافة في تجديدات العضوية، وعرض الانتهاء، وتحصيل المدفوعات الانتهاء الفوري، أو الأعطال، أو تفويت المدفوعات المطلوبة.
- تحسين: تم اختبار توافق WordPress حتى 7.0، وأُعيد بناء أصول Vue الإنتاجية من مصادر npm، وأصبحت تغطية Cypress من الطرف إلى الطرف تختبر المزيد من تدفقات الدفع، والإعداد، وSSO، والبوابات.

## الإصدار 2.12.0 — صدر في 2026-05-15 {#version-2120--released-on-2026-05-15}

- جديد: تمت إضافة Hostinger (hPanel) كمزود استضافة مدعوم مع تكامل ربط النطاقات
- جديد: تتعامل أداة تصدير الموقع الآن مع حزم استيراد الشبكة لاستعادة المواقع على مستوى الشبكة بسلاسة
- إصلاح: تستخدم رسائل البث الإلكترونية بنسخة BCC الآن ترويسة مستلمين غير معلنين لمنع كشف عناوين المستلمين
- إصلاح: لم يعد تاريخ انتهاء العضوية يتلف عند الحفظ بقيمة غير تاريخية
- إصلاح: تقوم تحديثات عضوية Stripe الآن بمسح الخصومات بشكل صحيح دون استدعاء deleteDiscount API المتقادمة
- إصلاح: أصبحت عمليات إعادة توجيه SSO على المواقع ذات النطاقات المربوطة محدودة الآن لمنع حلقات إعادة التوجيه اللانهائية
- إصلاح: يقوم اختيار منتقي صور معالج الإعداد الآن بتحديث نموذج البيانات الأساسي بشكل صحيح
- إصلاح: يحافظ CLI الخاص بأداة تصدير الموقع الآن على اختيار موقع الشبكة الافتراضي الصحيح
- تحسين: تمت إزالة wp-cli المضمّن من حزمة الإضافة، مما يقلل حجم الإضافة

## الإصدار 2.11.0 — صدر في 2026-05-11 {#version-2110--released-on-2026-05-11}
