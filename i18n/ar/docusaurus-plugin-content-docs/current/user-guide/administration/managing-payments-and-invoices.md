# إدارة المدفوعات والفواتير {#managing-payments-and-invoices}

## إعدادات الدفع {#payment-settings}

قبل أن تبدأ بتلقي المدفوعات، ستحتاج إلى تكوين الإعدادات المتعلقة بالدفع. انتقل إلى **Ultimate Multisite → الإعدادات** وانقر على تبويب **الدفع**.

![تبويب إعدادات الدفع](/img/admin/settings-payments-top.png)

إليك عرضًا كاملًا لصفحة إعدادات الدفع:

![صفحة إعدادات الدفع كاملة](/img/admin/settings-payments-full.png)

### خيارات الدفع العامة {#general-payment-options}

في الإعدادات العامة يمكنك تكوين:

- **العملة** — العملة الافتراضية المستخدمة للمعاملات
- **موضع العملة** — أين يظهر رمز العملة (قبل/بعد المبلغ)

![إعدادات بوابة الدفع](/img/admin/settings-payments-options.png)

### بوابات الدفع {#payment-gateways}

يدعم Ultimate Multisite عدة بوابات دفع. يمكنك تفعيل كل واحدة وتكوينها من تبويب إعدادات الدفع.

![تكوين بوابة الدفع](/img/admin/settings-payments-gateways.png)

تشمل البوابات المتاحة:

- **Stripe** — مدفوعات بطاقات الائتمان عبر Stripe
- **PayPal** — مدفوعات PayPal
- **يدوي** — للمعالجة غير المتصلة بالإنترنت أو المعالجة المخصصة للمدفوعات

لكل بوابة قسم تكوين خاص بها حيث تُدخل مفاتيح API وإعدادات أخرى.

![إعدادات بوابة إضافية](/img/admin/settings-payments-gateways-2.png)

### وضع Sandbox {#sandbox-mode}

يمكنك تفعيل **وضع Sandbox** لاختبار تكامل الدفع لديك قبل الإطلاق الفعلي. عندما يكون وضع Sandbox نشطًا، لن يتم إجراء أي رسوم حقيقية.

## عرض المدفوعات {#viewing-payments}

انتقل إلى صفحة **المدفوعات** ضمن Ultimate Multisite لرؤية جميع المعاملات عبر شبكتك.

![قائمة المدفوعات](/img/admin/payments-list.png)

يمكنك تصفية المدفوعات حسب الحالة (مكتملة، معلقة، فاشلة، مستردة) والبحث عن معاملات محددة.

انقر على دفعة لرؤية تفاصيلها الكاملة بما في ذلك عناصر البنود، والعضوية المرتبطة، ومعلومات العميل، وبيانات بوابة الدفع.

## الفواتير {#invoices}

يمكن لـ Ultimate Multisite إنشاء فواتير تلقائيًا للمدفوعات. يمكنك تخصيص قالب الفاتورة وتنسيق الترقيم من إعدادات الدفع.

تشمل خيارات تخصيص الفاتورة:

- **اسم الشركة وعنوانها** المعروضان على الفواتير
- **ترقيم الفواتير** التنسيق والتسلسل
- **الشعار** المعروض في رأس الفاتورة
- **نص تذييل مخصص** للشروط أو الملاحظات أو المعلومات القانونية

لتخصيص قالب الفاتورة، انتقل إلى **Ultimate Multisite → الإعدادات → الدفع** وابحث عن الإعدادات المتعلقة بالفواتير.
