# إدارة رسائل البريد الإلكتروني للنظام

يُرسل Ultimate Multisite رسائل بريد إلكتروني نظامية مختلفة لأحداث مثل تأكيد التسجيل، وإيصالات الدفع، وتغييرات العضوية. يمكنك تخصيص هذه القوالب لتتناسب مع علامتك التجارية.

## عرض رسائل البريد الإلكتروني للنظام

انتقل إلى **Ultimate Multisite → الإعدادات** وانقر على علامة التبويب **البريد الإلكتروني** لرؤية الإعدادات المتعلقة بالبريد الإلكتروني.

![إعدادات - علامة تبويب البريد الإلكتروني](/img/admin/settings-emails.png)

من هناك، يمكنك الوصول إلى صفحة **رسائل البريد الإلكتروني للنظام** لرؤية جميع القوالب.

![قائمة رسائل البريد الإلكتروني للنظام](/img/admin/system-emails-list.png)

## استيراد قوالب بريد إلكتروني جديدة

قد تقوم إصدارات Ultimate Multisite الجديدة والإضافات بتسجيل قوالب بريد إلكتروني جديدة. لمنع التعارضات، لا تُضاف القوالب الجديدة تلقائيًا — تحتاج إلى استيرادها يدويًا.

لـ استيراد قوالب بريد إلكتروني جديدة:

1. انتقل إلى صفحة **رسائل البريد الإلكتروني للنظام**
2. انقر على زر **إعادة التعيين والاستيراد** في أعلى الصفحة
3. فعّل خيار **استيراد رسائل البريد الإلكتروني** لرؤية القوالب المتاحة
4. اختر القوالب التي تريد استيرادها وأكد

## إعادة تعيين قوالب البريد الإلكتروني

إذا قمت بتخصيص قالب بريد إلكتروني وتريد العودة إلى المحتوى الافتراضي، لديك خياران:

1. **حذف وإعادة الاستيراد** — احذف البريد الإلكتروني للنظام واستورده مرة أخرى (هذا يمحو مقاييس الإرسال)
2. **إعادة التعيين عبر الأداة** — استخدم أداة **إعادة التعيين والاستيراد**، فعّل خيار **إعادة التعيين**، واختر رسائل البريد الإلكتروني التي تريد إعادة تعيينها

يفضل خيار إعادة التعيين لأنه يحافظ على مقاييس الإرسال والبيانات الوصفية الأخرى أثناء استعادة المحتوى الافتراضي.
