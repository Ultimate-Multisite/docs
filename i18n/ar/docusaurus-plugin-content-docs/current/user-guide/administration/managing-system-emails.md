# إدارة رسائل البريد الإلكتروني للنظام

يرسل Ultimate Multisite رسائل بريد إلكتروني متنوعة للنظام لأحداث مثل تأكيدات التسجيل، وإيصالات الدفع، وتغييرات العضوية. يمكنك تخصيص قوالب البريد الإلكتروني هذه لتطابق علامتك التجارية.

## عرض رسائل البريد الإلكتروني للنظام

انتقل إلى **Ultimate Multisite → Settings** وانقر على تبويب **Emails** لرؤية الإعدادات المتعلقة بالبريد الإلكتروني.

![تبويب Settings - Emails](/img/admin/settings-emails.png)

من هناك، يمكنك الوصول إلى صفحة **رسائل البريد الإلكتروني للنظام** لرؤية جميع قوالب البريد الإلكتروني.

![قائمة رسائل البريد الإلكتروني للنظام](/img/admin/system-emails-list.png)

يمكنك أيضًا تحرير قوالب البريد الإلكتروني الفردية باستخدام محرر البريد الإلكتروني للنظام:

![محرر البريد الإلكتروني للنظام](/img/admin/system-email-editor.png)

## استيراد قوالب بريد إلكتروني جديدة

قد تُسجّل إصدارات Ultimate Multisite الجديدة والإضافات قوالب بريد إلكتروني جديدة. لمنع التعارضات، لا تتم إضافة القوالب الجديدة تلقائيًا — تحتاج إلى استيرادها يدويًا.

لاستيراد قوالب بريد إلكتروني جديدة:

1. انتقل إلى صفحة **رسائل البريد الإلكتروني للنظام**
2. انقر على زر **Reset & Import** في أعلى الصفحة
3. بدّل خيار **Import Emails** لرؤية القوالب المتاحة
4. حدّد القوالب التي تريد استيرادها وأكّد

## إعادة تعيين قوالب البريد الإلكتروني

إذا كنت قد خصصت قالب بريد إلكتروني وتريد إعادته إلى المحتوى الافتراضي، فلديك خياران:

1. **الحذف وإعادة الاستيراد** — احذف البريد الإلكتروني للنظام واستورده مرة أخرى (يمحو هذا مقاييس الإرسال)
2. **إعادة التعيين عبر الأداة** — استخدم أداة **Reset & Import**، وبدّل خيار **Reset**، وحدد رسائل البريد الإلكتروني التي تريد إعادة تعيينها

يُفضّل خيار Reset لأنه يحافظ على مقاييس الإرسال والبيانات الوصفية الأخرى أثناء استعادة المحتوى الافتراضي.
