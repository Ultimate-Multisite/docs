# إدارة المواقع

المواقع (أو المواقع الفرعية) هي جوهر عملك في WaaS. Ultimate Multisite لديها 3 أنواع من المواقع:

- **مملوك للعميل** — المواقع التي تنتمي إلى عملاء محددين
- **قوالب المواقع** — مواقع محددة مسبقًا يمكن للعملاء اختيارها كنقطة انطلاق
- **الموقع الرئيسي** — موقع الشبكة الأساسي الخاص بك

## عرض المواقع

انتقل إلى **Ultimate Multisite → Sites** لرؤية جميع المواقع الفرعية في شبكتك. يتم تصنيف كل موقع حسب النوع ويمكنك التصفية حسب جميع المواقع، المملوك للعميل، القوالب، أو قيد الانتظار.

![صفحة قائمة المواقع](/img/admin/sites-list.png)

## إضافة موقع جديد

انقر على زر **إضافة موقع** لإنشاء موقع جديد. ستحتاج إلى توفير:

- **عنوان الموقع** — اسم الموقع الجديد
- **عنوان URL للموقع** — النطاق/المسار للموقع
- **نوع الموقع** — ما إذا كان موقعًا للعميل، قالبًا، أو موقعًا عاديًا

تتيح لك خيار **نسخ الموقع** إنشاء موقع جديد استنادًا إلى قالب موقع موجود. عند التفعيل، يمكنك اختيار القالب الذي تريد استخدامه كنقطة انطلاق. تأكد من تفعيل **نسخ الوسائط عند التكرار** لتضمين ملفات الوسائط.

## إدارة موقع موجود

انقر على **إدارة** في أي موقع لفتح صفحة **Edit Site**. هنا ستجد:

### المعلومات الأساسية

اسم الموقع، النوع، معرف الموقع، والوصف. سترى أيضًا النطاق المربوط، العضوية المرتبطة، وحساب العميل الذي يملك الموقع.

### خيارات الموقع

قم بتكوين قدرات الموقع وحدوده:

- **حدود الزيارات** — الحد الأقصى لعدد زيارات الموقع
- **حدود حساب المستخدم** — حدود لكل دور مستخدم
- **مساحة القرص** — حصة التخزين للموقع
- **نطاق مخصص** — تمكين ربط النطاق لهذا الموقع
- **رؤية الإضافات والقوالب** — التحكم في الإضافات والقوالب التي تظهر، المخفية، أو مفعلة مسبقًا

بشكل افتراضي، تتبع المواقع الحدود المحددة على مستوى العضوية. تعيين حدود على مستوى الموقع سيغطي إعدادات العضوية.

### الترابطات

أدناه خيارات الموقع ستجد معلومات حول:

- **النطاقات المربوطة** المرتبطة بالموقع
- **العضوية** التي يندرج تحتها الموقع
- **حساب العميل** المرتبط بالموقع

### الشريط الجانبي الأيمن

على الجانب الأيمن يمكنك:

- **تمكين/تعطيل الموقع** باستخدام زر تبديل
- **تغيير نوع الموقع** أو إعادة تعيين الملكية
- **تعيين صورة/صورة مصغرة للموقع** (مُعرضة على الواجهة الأمامية لقوالب المواقع)
- **حذف الموقع** نهائيًا

:::warning
حذف الموقع غير قابل للعودة. سيتم إزالة الموقع وجميع محتوياته نهائيًا.
:::
