# كيف يمكنني الوصول إلى صفحة حسابي للمرة الأولى؟

مرحبًا بكم في Ultimate Multisite!

ادخل إلى صفحة حسابك عبر الرابط الذي استلمته في بريدك الإلكتروني بعد شراء الإضافة الخاصة بنا:

<!-- Screenshot unavailable: Account page email with access link -->

استخدم البريد الإلكتروني الذي قدمته أثناء الشراء وكلمة المرور التي استلمتها في بريدك الإلكتروني.

<!-- Screenshot unavailable: Login page with email and password fields -->

نظرًا لأن هذه هي المرة الأولى التي تقوم فيها بتسجيل الدخول إلى هذا الحساب، سيُطلب منك تغيير كلمة المرور. غيّرها وانقر على زر _Save_.

<!-- Screenshot unavailable: Change password form with Save button -->

في صفحة حسابك، يمكنك إدارة جميع تفاصيل حسابك مثل الفواتير، التجديدات، الإلغاءات، والعديد من الأشياء الأخرى.
