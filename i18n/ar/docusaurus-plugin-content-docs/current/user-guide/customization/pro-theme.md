# تخصيص لوحة الإدارة الخاصة بك مع قالب PRO

غالبًا ما نحتاج عند العمل مع العملاء إلى توفير تجربة مخصصة لهم في لوحة تحكم WordPress. إذا كنت تبحث عن مظهر أكثر احترافية، قد يساعدك قالب PRO.

قالب PRO هو قالب إدارة تم تطويره لمساعدتك عند تخصيص لوحة تحكم عملائك. كما أنه جزء من مجموعة الإضافات المميزة لـ Ultimate Multisite.

بعد تثبيته، انتقل إلى صفحة _الإضافات_ وانقر لتفعيله على الشبكة.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_فعّله على مستوى الشبكة في صفحة الإضافات_

_**ملاحظة:**_ _إذا قمت بتثبيته فقط، سيظهر في حساب العميل كخيار يمكن تحديده لإضافة القالب فقط لهذا المستخدم._

الآن، تم إعادة تصميم واجهة الإدارة الخاصة بك بالكامل! كما لديك قائمة جديدة في العمود الأيسر تُسمى قالب PRO.
