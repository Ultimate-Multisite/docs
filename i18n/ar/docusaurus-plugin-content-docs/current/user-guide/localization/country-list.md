# إعادة ترتيب قائمة دول Ultimate Multisite

يتم ترتيب قائمة السحب المنسدلة لاختيار الدولة أبجدياً استناداً إلى النسخ الإنجليزية لأسماء الدول. قد يعني ذلك، أحياناً، أن بلد الشركة أو أغلب العملاء مخفي في وسط قائمة طويلة من أسماء الدول.

لمساعدتك على إصلاح ذلك وإضافة بلدك المفضل إلى أعلى قائمة الاختيار، يمكنك استخدام المقتطف أدناه (يمكنك إضافته إلى mu-plugin):

```php
add_filter('wu_get_countries', function($countries) {
    // غيّر DE إلى الرمز المكوّن من حرفين لبلدك
    // المثال أدناه لألمانيا.
    unset($countries['DE']);
    // غيّر DE إلى الرمز المكوّن من حرفين لبلدك
    // واسم بلدك
    // المثال أدناه لألمانيا.
    $countries = array_merge(array(
        'DE' => __('Germany', 'ultimate-multisite'),
    ), $countries);
    return $countries;
});
```
