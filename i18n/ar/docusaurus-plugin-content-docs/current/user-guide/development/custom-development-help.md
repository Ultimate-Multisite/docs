# كيف تحصل على مساعدة تطوير مخصص من Codeable؟

أحيانًا، نحتاج حقًا إلى تخصيص رائع لجعل تدفقات شبكتنا بالضبط كما نريد. إذا كان هذا ما تبحث عنه، نوصي بـ [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE)، وهو مكان يمكنك فيه العثور على مطورين ذوي خبرة في Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) هو منصة تربط العملاء بمطورين مستقلين مؤهلين في WordPress، قادرين على العمل على مشاريع بأي حجم. لديهم عملية طلب دقيقة لضمان مطابقتك مع خبراء مؤهلين خصيصًا للمهمة التي تفكر فيها.

### كيف تنشئ مشروعًا في Codeable؟

أولاً، اذهب إلى [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) وانقر على زر **Start a Project**.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

الآن، لتقديم مشروع، انقر على اليسار على **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

في 'I need help with'، اختر **Customizing** :

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

في 'My'، اختر **Plugin** :

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

ثم انقر على زر **Continue**.

<!-- Screenshot unavailable: Codeable project form Continue button -->

الآن حان الوقت لوصف مشروعك واحتياجاتك بأفضل طريقة ممكنة. حاول أن تقدم أكبر قدر ممكن من المعلومات!

<!-- Screenshot unavailable: Codeable project description form -->

أجب على الأسئلة التالية للمساعدة في التسعير.

<!-- Screenshot unavailable: Codeable project pricing questions -->

بعد ذلك، إذا لم يكن لديك حساب على [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) بعد، أنشئ واحدًا وانقر على زر **Publish My Project**.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

بمجرد نشر مشروعك على [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE)، ستقوم المنصة بربطك بمطورين يتطابقون مع مجموعة المهارات المطلوبة لمشروعك.

سيساعدك هذا العملية في العثور على الشخص المناسب لوظيفتك، الذي لديه خبرة في WordPress وUltimate Multisite!

أقل معدل لـ Codeable هو 70–120 دولار أمريكي في الساعة. ستتغير التقديرات وفقًا لنطاق العمل، التعقيد، وعجلة المشروع.

بمجرد حصولك على تقديرك، ستتمكن من توظيف المطور الذي يناسب احتياجاتك بشكل أفضل وبدء مشروعك معًا!
