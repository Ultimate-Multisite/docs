# Ultimate Multisite الخطافات والمرشحات

Ultimate Multisite يتألف من مجموعة كبيرة من الدوال واستعلامات قاعدة البيانات. يحتوي على خطافات ومرشحات تسمح لك بتعديل أو إضافة كود جديد إلى الدوال الموجودة.

أنشأنا قائمة بهذه الخطافات والمرشحات كمرجع لمساعدة المستخدمين المتقدمين على تخصيص شبكتهم بشكل أكبر. يمكنك الوصول إلى القائمة من GitHub بالنقر على الرابط أدناه.
