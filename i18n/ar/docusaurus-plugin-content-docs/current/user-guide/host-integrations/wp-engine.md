# تكامل WP Engine

## نظرة عامة
WP Engine هو منصة استضافة WordPress مدارة متميزة توفر أداءً محسنًا، أمانًا، وقابلية توسيع للمدونات على WordPress. يتيح هذا التكامل مزامنة النطاقات تلقائيًا بين Ultimate Multisite و WP Engine.

## الميزات
- المزامنة التلقائية للنطاقات
- دعم النطاقات الفرعية لتثبيتات المتعدد المواقع
- تكامل سلس مع أنظمة WP Engine الحالية

## المتطلبات
يكتشف التكامل تلقائيًا ما إذا كنت تستضيف على WP Engine ويستخدم واجهة برمجة التطبيقات المدمجة لـ WP Engine. لا حاجة لتكوين إضافي إذا كان إضافة WP Engine نشطة ومُكوَّنة بشكل صحيح.

ومع ذلك، إذا احتجت إلى تكوين التكامل يدويًا، يمكنك تعريف أحد هذه الثوابت في ملف `wp-config.php` الخاص بك:

```php
define('WPE_APIKEY', 'your_api_key'); // Preferred method
// OR
define('WPE_API', 'your_api_key'); // Alternative method
```

## تعليمات الإعداد

### 1. التحقق من إضافة WP Engine
إذا كنت تستضيف على WP Engine، يجب أن تكون إضافة WP Engine مثبتة ومفعلة بالفعل. تحقق من:
1. إضافة WP Engine نشطة
2. الملف `wp-content/mu-plugins/wpengine-common/class-wpeapi.php` موجود

### 2. تمكين التكامل
1. في لوحة تحكم WordPress الخاصة بك، انتقل إلى Ultimate Multisite > Settings
2. انتقل إلى علامة التبويب "Domain Mapping"
3. مرر لأسفل إلى "Host Integrations"
4. فعّل تكامل WP Engine
5. انقر على "Save Changes"

## كيف يعمل

### المزامنة النطاقية
عند تعيين نطاق في Ultimate Multisite:
1. يستخدم التكامل واجهة برمجة تطبيقات WP Engine لإضافة النطاق إلى تثبيت WP Engine الخاص بك
2. يتعامل WP Engine مع تكوين النطاق وإصدار شهادة SSL
3. عند إزالة تعيين النطاق، سيزيل التكامل النطاق من WP Engine

### دعم النطاقات الفرعية
للتثبيتات المتعددة المواقع باستخدام النطاقات الفرعية:
1. يضيف التكامل كل نطاق فرعي إلى WP Engine عند إنشاء موقع جديد
2. يتعامل WP Engine مع تكوين النطاق الفرعي
3. عند حذف موقع، سيزيل التكامل النطاق الفرعي من WP Engine

## ملاحظات مهمة

### نطاقات البدل (Wildcard)
للتثبيتات المتعددة المواقع باستخدام النطاقات الفرعية، يُنصح بالاتصال بدعم WP Engine لطلب تكوين نطاق بدل (Wildcard). يتيح ذلك لجميع النطاقات الفرعية العمل تلقائيًا دون الحاجة إلى إضافة كل واحد على حدة.

### شهادات SSL
يتعامل WP Engine تلقائيًا مع إصدار وتجديد شهادات SSL لجميع النطاقات المضافة عبر هذا التكامل. لا حاجة لتكوين إضافي.

## استكشاف الأخطاء وإصلاحها

### مشكلات اتصال واجهة برمجة التطبيقات
- تحقق من أن إضافة WP Engine نشطة ومُكوَّنة بشكل صحيح
- إذا قمت بتعريف مفتاح API يدويًا، تحقق من صحته
- اتصل بدعم WP Engine إذا كنت تواجه مشكلة مع واجهة برمجة التطبيقات

### النطاق غير مضاف
- تحقق من سجلات Ultimate Multisite لأي رسائل خطأ
- تحقق من أن النطاق غير مضاف بالفعل إلى WP Engine
- تأكد من أن خطة WP Engine الخاصة بك تدعم عدد النطاقات التي تقوم بإضافتها

### مشكلات النطاق الفرعي
- إذا لم تعمل النطاقات الفرعية، اتصل بدعم WP Engine لطلب تكوين نطاق بدل (Wildcard)
- تحقق من أن إعدادات DNS الخاصة بك مُكوَّنة بشكل صحيح للنطاق الرئيسي والنطاقات الفرعية
