# تكامل WPMU DEV

## نظرة عامة
WPMU DEV هو منصة WordPress شاملة تقدم استضافة، ملحقات، وخدمات لمواقع WordPress. يتيح هذا التكامل مزامنة النطاقات تلقائيًا وإدارة شهادات SSL بين Ultimate Multisite واستضافة WPMU DEV.

## الميزات
- المزامنة التلقائية للنطاقات
- إدارة شهادات SSL
- زيادة عدد محاولات التحقق من شهادة SSL

## المتطلبات
يكتشف التكامل تلقائيًا ما إذا كنت تستضيف على WPMU DEV ويستخدم API المدمج. لا حاجة لتكوين إضافي إذا كنت تستضيف على WPMU DEV.

يتحقق التكامل من وجود الثابت `WPMUDEV_HOSTING_SITE_ID`، الذي يُعرّف تلقائيًا عند الاستضافة على WPMU DEV.

## تعليمات الإعداد

### 1. التحقق من استضافة WPMU DEV
إذا كنت تستضيف على WPMU DEV، يجب أن تكون الثوابت اللازمة معرفة بالفعل. تحقق من أن:
1. الثابت `WPMUDEV_HOSTING_SITE_ID` معرف في بيئتك
2. لديك عضوية نشطة في WPMU DEV مع وصول إلى API

### 2. تمكين التكامل
1. في لوحة تحكم WordPress الخاصة بك، اذهب إلى Ultimate Multisite > Settings
2. انتقل إلى علامة التبويب "Domain Mapping"
3. مرر لأسفل إلى "Host Integrations"
4. فعّل تكامل WPMU DEV
5. انقر على "Save Changes"

## كيف يعمل

### المزامنة النطاق
عند تعيين نطاق في Ultimate Multisite:
1. يستخدم التكامل API WPMU DEV لإضافة النطاق إلى حساب الاستضافة الخاص بك
2. كما يضيف إصدار www للنطاق تلقائيًا
3. يتعامل WPMU DEV مع تكوين النطاق وإصدار شهادة SSL

### إدارة شهادة SSL
تم تكوين التكامل لزيادة عدد محاولات التحقق من شهادة SSL لاستضافة WPMU DEV، حيث قد يستغرق إصدار وتثبيت شهادات SSL بعض الوقت. افتراضيًا، سيحاول حتى 10 مرات للتحقق من شهادة SSL، مقارنةً بـ 5 محاولات قياسية.

## ملاحظات مهمة

### إزالة النطاق
حاليًا، لا يوفر API WPMU DEV طريقة لإزالة النطاقات. عند إزالة تعيين نطاق في Ultimate Multisite، سيظل النطاق موجودًا في حساب استضافة WPMU DEV الخاص بك. ستحتاج إلى إزالته يدويًا من لوحة تحكم استضافة WPMU DEV إذا لزم الأمر.

### مصادقة API
يستخدم التكامل مفتاح API WPMU DEV المخزن في قاعدة بيانات WordPress الخاصة بك كخيار `wpmudev_apikey`. يتم إعداد ذلك تلقائيًا عند توصيل موقعك بـ WPMU DEV.

## استكشاف الأخطاء وإصلاحها

### مشاكل اتصال API
- تحقق من أن موقعك متصل بشكل صحيح بـ WPMU DEV
- تحقق من أن خيار `wpmudev_apikey` مضبوط في قاعدة بيانات WordPress الخاصة بك
- تأكد من أن عضويتك في WPMU DEV نشطة

### مشاكل شهادة SSL
- قد يستغرق WPMU DEV بعض الوقت لإصدار شهادات SSL (عادةً 5-15 دقيقة)
- تم تكوين التكامل للتحقق حتى 10 مرات من شهادات SSL
- إذا لم تُصدر شهادات SSL بعد عدة محاولات، اتصل بدعم WPMU DEV

### النطاق غير مضاف
- تحقق من سجلات Ultimate Multisite لأي رسائل خطأ
- تحقق من أن النطاق غير مضاف بالفعل إلى WPMU DEV
- تأكد من أن خطة استضافة WPMU DEV الخاصة بك تدعم عدد النطاقات التي تضيفها
