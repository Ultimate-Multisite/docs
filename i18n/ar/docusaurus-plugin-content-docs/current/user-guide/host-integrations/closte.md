# تكامل Closte

## نظرة عامة
Closte هو منصة استضافة WordPress مُدارة مبنية على بنية تحتية من Google Cloud. يتيح هذا التكامل مزامنة النطاقات تلقائيًا وإدارة شهادات SSL بين Ultimate Multisite وCloste.

## الميزات
- المزامنة التلقائية للنطاقات
- إدارة شهادات SSL
- دعم نطاقات البدل (Wildcard)
- لا حاجة لتكوين إذا كنت تعمل على Closte

## المتطلبات
يجب تعريف الثابت التالي في ملف `wp-config.php` الخاص بك إذا كنت تستخدم Closte:

```php
define('CLOSTE_CLIENT_API_KEY', 'your_api_key');
```

عادةً ما يكون هذا الثابت معرفًا بالفعل إذا كنت تستضيف على Closte.

## تعليمات الإعداد

### 1. التحقق من مفتاح API الخاص بـ Closte

إذا كنت تستضيف على Closte، يجب أن يكون الثابت `CLOSTE_CLIENT_API_KEY` معرفًا بالفعل في ملف `wp-config.php` الخاص بك. يمكنك التحقق من ذلك عن طريق فحص ملف `wp-config.php`.

### 2. تمكين التكامل

1. في لوحة تحكم WordPress الخاصة بك، انتقل إلى Ultimate Multisite > Settings
2. انتقل إلى علامة التبويب "تعيين النطاق"
3. مرر لأسفل إلى "التكاملات المضيفة"
4. فعّل تكامل Closte
5. انقر على "حفظ التغييرات"

## كيف يعمل

عند تعيين نطاق في Ultimate Multisite:

1. يرسل التكامل طلبًا إلى API الخاص بـ Closte لإضافة النطاق إلى تطبيقك
2. يتعامل Closte تلقائيًا مع توفير شهادة SSL
3. عند إزالة تعيين النطاق، سيزيل التكامل النطاق من Closte

يعمل التكامل أيضًا مع إعداد فاصل التحقق من DNS في Ultimate Multisite، مما يتيح لك تكوين مدى تكرار فحص النظام لتعميم DNS وإصدار شهادة SSL.

## إنشاء سجل النطاق

يضمن هذا التكامل أنه عند إنشاء موقع أو تكراره، يتم إنشاء سجل نطاق تلقائيًا. هذا مهم بشكل خاص لتكامل Closte، حيث يثير إنشاء سجل النطاق طلبًا إلى API الخاص بـ Closte لإنشاء النطاق وشهادة SSL.

## استكشاف الأخطاء وإصلاحها

### مشكلات الاتصال بالـ API

- تحقق من أن مفتاح API الخاص بـ Closte صحيح
- تأكد من أن حسابك في Closte لديه الأذونات اللازمة

### مشكلات شهادة SSL

- قد يستغرق Closte بعض الوقت لإصدار شهادات SSL (عادةً 5-10 دقائق)
- تحقق من أن نطاقاتك تشير بشكل صحيح إلى عنوان IP الخاص بخادم Closte
- افحص سجلات DNS لنطاقك للتأكد من أنها مُكوَّنة بشكل صحيح

### النطاق غير مضاف

- تحقق من سجلات Ultimate Multisite لأي رسائل خطأ
- تأكد من أن النطاق غير مضاف بالفعل إلى Closte
- تأكد من أن سجلات DNS الخاصة بنطاقك مُكوَّنة بشكل صحيح

### فاصل التحقق من DNS

- إذا استغرقت شهادات SSL وقتًا طويلاً للإصدار، يمكنك ضبط فاصل التحقق من DNS في إعدادات تعيين النطاق
- الفاصل الافتراضي هو 300 ثانية (5 دقائق)، لكن يمكنك ضبطه حتى 10 ثوانٍ لتسريع الفحص أثناء الاختبار
