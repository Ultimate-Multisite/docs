# التعارضات المعروفة مع الإضافات الأخرى

بعد تثبيت وتفعيل Ultimate Multisite، قد تواجه مشاكل بسبب التعارضات غير المعالجة مع إضافات أخرى.

نحن نعمل بنشاط على تقليل هذه القائمة مع كل إصدار جديد، ولكن إذا كانت إضافة مدرجة هنا، قد تحتاج إلى تعطيلها من أجل الحصول على شبكة Ultimate Multisite تعمل بكامل وظائفها.

إليك قائمة بها وما يجب عليك فعله في هذه الحالات:

**بعض إضافات WPMU DEV** بعض الإضافات التي _لم تعد_ تحتفظ بها WPMU DEV معروفة بأنها تتسبب في توقف عملية تسجيل Ultimate Multisite عند الخطوة الأخيرة (لا يستطيع المستخدم إنشاء حساب). إليك قائمة بالإضافات التي تسبب هذه المشكلات:

  * Signup Language: _Ultimate Multisite لديها إضافة تقدم نفس الوظيفة، تُسمى Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _هذا مدمج في Ultimate Multisite، لذلك لا حاجة لإضافة إضافية._

  * New Blog Templates: _هذا مدمج في Ultimate Multisite، لذلك لا حاجة لإضافة إضافية._

  * Set Password on Multisite Blog Creation: _هذا مدمج في Ultimate Multisite، لذلك لا حاجة لإضافة إضافية._

  * Signup TOS: _هذا مدمج في Ultimate Multisite، لذلك لا حاجة لإضافة إضافية._

  * Anti-Splog: على الرغم من أنه أقل قوة، إلا أنه أيضًا _مدمج في Ultimate Multisite._

**ملاحظة** : يمكن استخدام جميع إضافات WPMU Dev الأخرى عادةً مع Ultimate Multisite. أمثلة على ذلك _Smush_، _Forminator_، _Defender_، إلخ.
