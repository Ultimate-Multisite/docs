# Superdav AI Agent

Superdav AI Agent هو مساعد ذكاء اصطناعي متقدم لمنصة WordPress، يوسع قدرات Gratis AI Agent بإضافة مهارات واعية بالسمة (theme-aware skills)، وقدرات بحث متقدمة عبر الإنترنت، وميزات محسّنة لمنشئ المواقع (site builder). في الإصدار 1.16.0، تمت إضافة إنشاء الشعارات المخصصة، والتحقق من تباين لوحة الألوان، وتحميل الصور في مقابلة الاكتشاف، وتوليد قوائم الضيافة المنظمة، وعرض المعاينة على سطح المكتب/الهاتف المحمول في Theme Builder.

## الميزات الرئيسية

- **مهارات تتكيف مع السمة (Theme-aware skills)** — يتكيف تلقائيًا مع الإرشادات والقدرات بناءً على سمة WordPress النشطة والإضافات المثبتة لديك.
- **مزودو بحث متعددون عبر الإنترنت** — اختر بين Brave Search و Tavily للحصول على نتائج بحث أغنى وقدرات بحث متقدمة.
- **نماذج الاتصال في منشئ المواقع** — أنشئ نماذج اتصال وظيفية بالكامل مباشرة من واجهة الدردشة باستخدام إضافة النماذج المثبتة لديك.
- **منشئ مواقع مُحسّن** — خطط بناء مواقع متعددة المراحل ومحسّنة مع اكتشاف أفضل للإضافات واستعادة أفضل للأخطاء.
- **مهارات السمة المدمجة** — إرشادات متخصصة لسمات الكتل (Block Themes)، والسمات الكلاسيكية (Classic Themes)، وKadence Blocks، وKadence Theme.
- **تكامل WooCommerce مُحسّن** — أُعيدت هيكلته لاستخدام واجهات برمجة تطبيقات WooCommerce الأصلية لتحسين الموثوقية والتوافق.
- **الكشف التلقائي عن المزود** — يكتشف ويحدّث تلقائيًا الإضافات والسمات المتاحة عند تفعيلها أو إلغاء تفعيلها.
- **إنشاء الشعار المخصص** — أنشئ وأضمّن شعارات SVG مخصصة مع تنقية تلقائية آمنة للمساحات الاسمية (namespace-safe sanitisation).
- **التحقق من تباين لوحة الألوان** — تحقق من أزواج الألوان لضمان التوافق مع معايير WCAG قبل تطبيقها على سمّتك.
- **اكتشاف مُعزز بالصور** — قم بتحميل الصور أثناء مقابلة الاكتشاف في Theme Builder للتأثير على اتجاه التصميم.
- **قوائم الضيافة** — قم بتوليد صفحات قوائم طعام ومشروبات منظمة للمطاعم والمقاهي وأعمال الضيافة الأخرى.
- **معاينات سطح المكتب والهاتف المحمول** — معاينة تصميمك على أجهزة سطح المكتب والهاتف المحمول أثناء اختيار اتجاه التصميم.
- **كشط المواقع من المستوى الأول (Tier 1 site scraping)** — أصبح sd-ai-agent/site-scrape متاحًا افتراضيًا لتحليل مواقع المنافسين ومصادر الإلهام.

## المتطلبات

- WordPress 6.0 أو أعلى
- PHP 8.1 أو أعلى
- WordPress Multisite (مُفعّل على مستوى الشبكة) أو تثبيت موقع واحد
- إضافة موصل مزود ذكاء اصطناعي نشطة (Anthropic، OpenAI، أو متوافقة)
- Gratis AI Agent v1.9.0 أو أعلى (Superdav AI Agent يوسع Gratis AI Agent)

## التثبيت

1. قم بتحميل ملفات الإضافة إلى `/wp-content/plugins/superdav-ai-agent`
2. قم بتفعيل الإضافة عبر شاشة **الإضافات (Plugins)** في WordPress
3. تأكد من تفعيل Gratis AI Agent أيضًا
4. انتقل إلى **Gratis AI Agent → الإعدادات (Settings)** وقم بتكوين مزود البحث عبر الإنترنت الخاص بك (اختياري)

## البدء السريع

بمجرد التفعيل، يقوم Superdav AI Agent تلقائيًا بتحسين تثبيت Gratis AI Agent الخاص بك بمهارات واعية بالسمة وقدرات إضافية. افتح لوحة **Gratis AI Agent** في الشريط الجانبي لإدارة WordPress وابدأ باستخدام الميزات الجديدة.

**أمثلة التعليمات:**

```
Add a contact form to the Contact page.
```

```
Search the web for the latest WordPress security best practices.
```

```
Create a hero section using the active theme's block patterns.
```

## المهارات الواعية بالسمة (Theme-Aware Skills)

يكتشف Superdav AI Agent تلقائيًا السمة النشطة والإضافات المثبتة لديك، ويحمّل المهارة المناسبة الواعية بالسمة في سياق المساعد. تشمل المهارات المتاحة ما يلي:

| المهارة | ينطبق على |
|---|---|
| **Block Themes** | السمات التي تستخدم بنية الكتل (Full Site Editing) الخاصة بـ WordPress |
| **Classic Themes** | سمات WordPress التقليدية التي تستخدم قوالب PHP والمحرر الكلاسيكي |
| **Kadence Blocks** | المواقع التي تحتوي على إضافة Kadence Blocks مُفعّلة |
| **Kadence Theme** | المواقع التي تستخدم سمة Kadence كسمة نشطة |

راجع [Theme-Aware Skills](../../user-guide/configuration/theme-aware-skills) للحصول على إرشادات مفصلة حول كل مهارة.

## مزودو البحث عبر الإنترنت (Internet Search Providers)

أضاف Superdav AI Agent v1.10.0 مزود **Tavily** كبديل لـ Brave Search لاستعلامات البحث عبر الإنترنت. يقدم كلا المزودين نتائج بحث في الوقت الفعلي ويمكن تكوينهما في **Gratis AI Agent → الإعدادات (Settings) → المتقدم (Advanced)**.

| المزود | الميزات | الإعداد |
|---|---|---|
| **Brave Search** | نتائج بحث سريعة ومركزة على الخصوصية | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | نتائج بحث أغنى، وقدرات بحث متقدمة | [Tavily API](https://tavily.com/) |

راجع [Internet Search](../../user-guide/configuration/internet-search) للحصول على تعليمات التكوين.

## نماذج الاتصال في منشئ المواقع (Site Builder Contact Forms)

يمكن لوكيل منشئ المواقع الآن إنشاء نماذج اتصال مباشرة من واجهة الدردشة. ما عليك سوى مطالبة منشئ المواقع بإنشاء نموذج اتصال، وسيكتشف تلقائيًا إضافة النماذج المثبتة لديك (Contact Form 7، WPForms، Fluent Forms، أو Gravity Forms) ويقوم بتوليد نموذج جاهز للاستخدام.

راجع [Site Builder Contact Forms](../../user-guide/administration/site-builder-contact-forms) للحصول على إرشادات مفصلة.

## ميزات منشئ المواقع (Theme Builder Features)

يتضمن Theme Builder في Superdav AI Agent v1.16.0 قدرات جديدة وقوية:

### مقابلة الاكتشاف مع تحميل الصور (Discovery Interview with Photo Upload)

تتضمن الآن مقابلة الاكتشاف في Theme Builder خطوة لتحميل الصور حيث يمكنك تقديم إلهام بصري. قم بتحميل صور لعلامتك التجارية، أو مواقع المنافسين، أو مصادر الإلهام لمساعدة Theme Builder على فهم اتجاه تصميمك.

راجع [Discovery Interview](./theme-builder/discovery-interview.md) للحصول على إرشادات مفصلة.

### اتجاه التصميم مع معاينات سطح المكتب والهاتف المحمول (Design Direction with Desktop and Mobile Previews)

راجع تصميمك على كل من أجهزة سطح المكتب والهاتف المحمول قبل أن يقوم Theme Builder بتوليد سمّتك الكاملة. شاهد المعاينات في الوقت الفعلي عند نقاط توقف استجابة متعددة.

راجع [Design Direction](./theme-builder/design-direction.md) للحصول على إرشادات مفصلة.

### قوائم الضيافة (Hospitality Menus)

يمكن لـ Theme Builder الآن إنشاء صفحات قوائم طعام ومشروبات منظمة للمطاعم والمقاهي والحانات وأعمال الضيافة الأخرى. تتضمن القوائم الأسعار، والأوصاف، والمعلومات الغذائية، وتحذيرات مسببات الحساسية.

راجع [Hospitality Menus](./theme-builder/hospitality-menus.md) للحصول على إرشادات مفصلة.

### قدرات جديدة

- **[Generate Logo SVG](./abilities/generate-logo-svg.md)** — إنشاء شعارات SVG مخصصة مع تنقية تلقائية
- **[Validate Palette Contrast](./abilities/validate-palette-contrast.md)** — التحقق من أزواج الألوان لضمان التوافق مع WCAG
- **[Create Menu](./abilities/create-menu.md)** — إنشاء قوائم تنقل بعلامات تنقل مميزة

راجع [Tools and Configuration](./theme-builder/tools.md) للحصول على معلومات حول الأدوات المتاحة وأداة كشط المواقع من المستوى الأول.

## التكوين (Configuration)

جميع الإعدادات موجودة ضمن **Gratis AI Agent → الإعدادات (Settings)** في لوحة تحكم WordPress. يضيف Superdav AI Agent خيارات التكوين التالية:

| الإعداد | الوصف |
|---|---|
| **Internet Search Provider** | اختر بين Brave Search و Tavily (اختياري) |
| **Tavily API Key** | مفتاح واجهة برمجة التطبيقات (API key) للبحث عبر الإنترنت في Tavily (إذا تم اختيار Tavily) |

## التحسينات في الإصدار v1.16.0

- **ذاكرة التخزين المؤقت لعميل الذكاء الاصطناعي (AI Client cache)** — مدعومة الآن بالبيانات المؤقتة (transients) للحفاظ على استمرارية البيانات عبر الطلبات، مما يمنع فقدان البيانات في مهام الوكيل طويلة الأمد.
- **أداة كشط المواقع من المستوى الأول (Tier 1 site-scrape tool)** — أصبح sd-ai-agent/site-scrape متاحًا افتراضيًا في Theme Builder لتحليل مواقع المنافسين ومصادر الإلهام.
- **روابط إجراءات صف الإضافة (Plugin row action links)** — تم تصحيحها وإعادة تسميتها للوضوح.

## التحسينات في الإصدار v1.10.0

- **إعادة هيكلة تكامل WooCommerce** — يستخدم الآن واجهات برمجة تطبيقات WooCommerce الأصلية لتحسين الموثوقية والتوافق.
- **التحديث التلقائي لقائمة المزودين** — يكتشف تلقائيًا متى يتم تفعيل الإضافات أو إلغاء تفعيلها ويحدّث قائمة المزودين المتاحة.
- **منشئ مواقع مُحسّن** — خطط بناء مواقع متعددة المراحل ومحسّنة مع استعادة أفضل للأخطاء.
- **مهارات واعية بالسمة** — أربع مهارات مدمجة جديدة تتكيف تلقائيًا مع سمّتك النشطة.

## إصلاح الأخطاء في الإصدار v1.10.0

- تم إصلاح قدرة `navigate-to` التي كانت تسبب حلقات إعادة تحميل لا نهائية في بعض صفحات الإدارة.
- تم إصلاح قدرة `list-posts` التي لم تكن تحل أسماء الفئات والعلامات بشكل صحيح إلى مسارات (slugs).
- تم إصلاح أوامر WP-CLI المفقودة لأسماء المساحات (namespace aliases) بعد إعادة هيكلة سابقة.
- تم إصلاح معالجة الأحداث التي تتعامل بأناقة مع المواقع التي لم يتم إنشاء جداول الأتمتة فيها بعد.
- تم إصلاح قدرة `memory-save` باستخدام البادئة الصحيحة لمساحة الاسم في مُنشئ التعليمات النظامية.
- تم إصلاح نتائج الأدوات القياسية (scalar tool results) ليتم تغليفها بشكل صحيح قبل إعادتها إلى الذكاء الاصطناعي.
- تم إصلاح إحصائيات الاستخدام للتعامل بشكل صحيح مع تنسيق مفتاح القدرة القديم عند الترقية من الإصدارات الأقدم.
