# Superdav AI Agent

Superdav AI Agent هو مساعد WordPress AI متقدم يوسّع Gratis AI Agent بمهارات واعية بالقالب، وقدرات بحث محسّنة على الإنترنت، وميزات بناء المواقع، وتكاملات أتمتة اختيارية. يضيف الإصدار 1.18.0 أدوات قراءة Google Calendar، وربط جهات الاتصال والحضور، وبوابات الموافقة البشرية، وإزالة تكرار التذكيرات، وإشعارات TextBee SMS، وتوفير خدمة Superdav مُدارة، وحزمة Advanced مرافقة موزعة بشكل منفصل لسير عمل المطورين الموثوقين.

## الميزات الرئيسية

- **مهارات واعية بالقالب** — تكيّف الإرشادات والقدرات تلقائيًا بناءً على قالب WordPress النشط لديك والإضافات المثبّتة
- **عدة مزودي بحث عبر الإنترنت** — اختر بين Brave Search وTavily للحصول على نتائج بحث أغنى وقدرات بحث متقدمة
- **نماذج اتصال لباني المواقع** — أنشئ نماذج اتصال كاملة الوظائف مباشرةً من واجهة الدردشة باستخدام إضافة النماذج المثبّتة لديك
- **باني مواقع محسّن** — خطط محسّنة لبناء المواقع متعددة المراحل مع اكتشاف أفضل للإضافات واستعادة أفضل من الأخطاء
- **مهارات قوالب مدمجة** — إرشادات متخصصة لقوالب Block Themes وClassic Themes وKadence Blocks وKadence Theme
- **تكامل WooCommerce محسّن** — أُعيدت هيكلته لاستخدام واجهات WooCommerce الأصلية من أجل موثوقية وتوافق أفضل
- **اكتشاف تلقائي للمزوّدين** — يكتشف الإضافات والقوالب المتاحة ويحدّثها تلقائيًا عند تفعيلها أو تعطيلها
- **إنشاء شعار مخصص** — أنشئ شعارات SVG مخصصة وضمّنها مع تعقيم تلقائي آمن لمساحات الأسماء
- **التحقق من تباين اللوحات** — افحص أزواج الألوان للتوافق مع إرشادات إمكانية الوصول WCAG قبل تطبيقها على قالبك
- **اكتشاف محسّن بالصور** — ارفع صورًا أثناء مقابلة اكتشاف Theme Builder للتأثير في اتجاه التصميم
- **قوائم الضيافة** — أنشئ صفحات قوائم طعام ومشروبات منظمة للمطاعم والمقاهي وأعمال الضيافة
- **معاينات سطح المكتب والجوال** — عاين تصميمك على أجهزة سطح المكتب والجوال أثناء اختيار اتجاه التصميم
- **استخلاص مواقع من المستوى 1** — أصبح sd-ai-agent/site-scrape متاحًا الآن افتراضيًا لتحليل مواقع المنافسين ومواقع الإلهام
- **أدوات قراءة Google Calendar** — اقرأ التقويمات والأحداث المهيأة حتى تتمكن الأتمتات من الاستدلال حول سياق الجدول القادم
- **ربط جهات الاتصال والحضور** — طابق مشاركي التقويم مع مستخدمي الموقع وجهات الاتصال المعروفة قبل إنشاء أعمال المتابعة
- **بوابات الموافقة البشرية** — أوقف الأتمتات الحساسة مؤقتًا للمراجعة قبل إرسال الرسائل أو استئناف العمل أو إجراء تغييرات
- **سجلات التذكيرات وإزالة التكرار** — تتبّع التذكيرات التي أُرسلت بالفعل حتى لا تُخطر عمليات الأتمتة المتكررة المستلم نفسه مرتين
- **إشعارات TextBee SMS** — أرسل إشعارات الرسائل النصية المهيأة عبر TextBee عند تمكين مزوّد SMS
- **توفير خدمة Superdav مُدارة** — اربط المواقع المدعومة بنقاط نهاية Superdav مستضافة مع إعداد تلقائي بدلًا من تهيئة نقطة النهاية يدويًا
- **حزمة Advanced المرافقة** — أدوات مطورين موثوقين موزعة بشكل منفصل لسير عمل نظام الملفات، وقاعدة البيانات، وWP-CLI، وموزّع REST، وباني الإضافات، ولقطة git، وإدارة المستخدمين، والقياس المعياري

## المتطلبات

- WordPress 6.0 أو أحدث
- PHP 8.1 أو أحدث
- WordPress Multisite (مفعّل على الشبكة) أو تثبيت موقع واحد
- إضافة موصل مزوّد AI نشطة (Anthropic أو OpenAI أو متوافقة)
- Gratis AI Agent v1.9.0 أو أحدث (يمتد Superdav AI Agent من Gratis AI Agent)

## التثبيت

1. ارفع ملفات الإضافة إلى `/wp-content/plugins/superdav-ai-agent`
2. فعّل الإضافة من خلال شاشة **الإضافات** في WordPress
3. تأكد من تفعيل Gratis AI Agent أيضًا
4. انتقل إلى **Gratis AI Agent → الإعدادات** وهيّئ مزوّد البحث عبر الإنترنت لديك، أو اتصال خدمة Superdav المُدارة، أو Google Calendar، أو بيانات اعتماد TextBee حسب الحاجة

تحتوي حزمة WordPress.org على ميزات Superdav AI Agent الأساسية. تُبنى حزمة Advanced المرافقة وتوزّع بشكل منفصل للبيئات التي تُثبَّت فيها أدوات المطورين الموثوقين وتُصرَّح بها صراحةً.

## البدء السريع

بمجرد التفعيل، يعزز Superdav AI Agent تلقائيًا تثبيت Gratis AI Agent لديك بمهارات واعية بالقالب وقدرات إضافية. افتح لوحة **Gratis AI Agent** في الشريط الجانبي لإدارة WordPress وابدأ باستخدام الميزات الجديدة.

**تعليمات مثال:**

```
أضف نموذج اتصال إلى صفحة الاتصال.
```

```
ابحث في الويب عن أحدث أفضل ممارسات أمان WordPress.
```

```
أنشئ قسمًا رئيسيًا باستخدام أنماط الكتل الخاصة بالقالب النشط.
```

```
راجع أحداث Google Calendar المهيأة ليوم غد وصغ مسودات تذكيرات للحضور.
```

```
أرسل تذكير SMS بعد أن أوافق على الرسالة.
```

## مهارات واعية بالقالب

يكتشف Superdav AI Agent تلقائيًا قالبك النشط والإضافات المثبّتة، ويحمّل المهارة المناسبة الواعية بالقالب إلى سياق المساعد. تشمل المهارات المتاحة:

| المهارة | ينطبق على |
|---|---|
| **Block Themes** | القوالب التي تستخدم بنية WordPress القائمة على الكتل (تحرير الموقع الكامل) |
| **Classic Themes** | قوالب WordPress التقليدية التي تستخدم قوالب PHP والمحرر الكلاسيكي |
| **Kadence Blocks** | المواقع التي تكون فيها إضافة Kadence Blocks نشطة |
| **Kadence Theme** | المواقع التي يكون فيها قالب Kadence هو القالب النشط |

راجع [المهارات الواعية بالقالب](../../user-guide/configuration/theme-aware-skills) للحصول على إرشادات مفصلة حول كل مهارة.

## مزودو البحث عبر الإنترنت

يضيف Superdav AI Agent v1.10.0 **Tavily** كبديل لـ Brave Search لاستعلامات البحث عبر الإنترنت. يقدم كلا المزودين نتائج بحث في الوقت الحقيقي ويمكن تهيئتهما في **Gratis AI Agent → الإعدادات → متقدم**.

| المزوّد | الميزات | الإعداد |
|---|---|---|
| **Brave Search** | نتائج بحث سريعة ومركّزة على الخصوصية | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | نتائج بحث أغنى، وقدرات بحث متقدمة | [Tavily API](https://tavily.com/) |

راجع [البحث عبر الإنترنت](../../user-guide/configuration/internet-search) للحصول على تعليمات التهيئة.

## تقاويم وموافقات وسير عمل الإشعارات

يوسّع Superdav AI Agent v1.18.0 سير عمل الأتمتة إلى ما هو أبعد من مهام المحتوى وبناء المواقع:

- **أدوات Google Calendar** تتيح للوكيل قراءة التقويمات المهيأة وتفاصيل الأحداث عند توفّر بيانات اعتماد التقويم.
- **مساعدات جهات الاتصال والحاضرين** تربط المشاركين في الحدث بمستخدمي WordPress أو بجهات الاتصال المعروفة قبل إعداد التذكيرات أو مهام المتابعة.
- **بوابات الموافقة البشرية** تسمح للأتمتة بالتوقف مؤقتًا حتى يراجع مستخدم مخوّل الإجراء المقترح.
- **سجلات التذكيرات** تمنع تكرار تذكيرات التقويم أو الإشعارات عند إعادة محاولة الأتمتة أو جدولتها بشكل متكرر.
- **دعم TextBee SMS** يتيح إشعارات الرسائل النصية عند تهيئة مزوّد TextBee.

استخدم هذه الأدوات للمساعدة الواعية بالجدول الزمني، ومتابعة الحاضرين، وسير عمل التذكيرات التي لا تزال تتطلب مراجعة بشرية قبل إرسال الرسائل الحساسة.

## نماذج الاتصال في باني الموقع

يمكن لوكيل باني الموقع الآن إنشاء نماذج اتصال مباشرة من واجهة الدردشة. ما عليك سوى أن تطلب من باني الموقع إنشاء نموذج اتصال، وسيكتشف تلقائيًا plugin النماذج المثبّت لديك (Contact Form 7 أو WPForms أو Fluent Forms أو Gravity Forms) وينشئ نموذجًا جاهزًا للاستخدام.

راجع [نماذج الاتصال في باني الموقع](../../user-guide/administration/site-builder-contact-forms) للحصول على إرشادات مفصلة.

## التهيئة

توجد جميع الإعدادات ضمن **Gratis AI Agent → الإعدادات** في إدارة WordPress. يضيف Superdav AI Agent خيارات التهيئة التالية:

| الإعداد | الوصف |
|---|---|
| **مزوّد البحث عبر الإنترنت** | اختر بين Brave Search وTavily (اختياري) |
| **مفتاح API لـ Tavily** | مفتاح API للبحث عبر الإنترنت باستخدام Tavily (إذا تم اختيار Tavily) |
| **خدمة Superdav المُدارة** | اربط المواقع المدعومة بخدمة Superdav المستضافة مع توفير تلقائي لنقاط النهاية |
| **بيانات اعتماد Google Calendar** | هيّئ وصول التقويم حتى يتمكن الوكيل من قراءة التقويمات والأحداث |
| **مزوّد TextBee SMS** | هيّئ بيانات اعتماد TextBee لسير عمل إشعارات SMS المعتمدة |

## حزمة المرافق المتقدمة

يضيف Superdav AI Agent v1.18.0 حزمة مرافق متقدمة موزعة بشكل منفصل لسير عمل المطورين الموثوقين. لا تُضمّن هذه الإمكانات في حزمة WordPress.org الأساسية. ثبّت حزمة المرافق فقط على المواقع التي يثق فيها المسؤولون صراحةً بالوكيل للوصول إلى أدوات المطورين.

تشمل الأدوات المتقدمة عمليات نظام الملفات، والوصول إلى قاعدة البيانات، وWP-CLI، وسير عمل موجّه REST، وأدوات إنشاء plugin، ولقطات git، ومساعدات إدارة المستخدمين، وأدوات قياس الأداء. أبقِ هذه الأدوات مقتصرة على المسؤولين الموثوقين وراجع تهيئة بوابة الموافقة قبل تمكينها على مواقع الإنتاج.

## التحسينات في v1.18.0

- **دعم Google Calendar** — اقرأ التقويمات والأحداث المهيأة للأتمتة الواعية بالجدول الزمني
- **ربط جهات الاتصال والحاضرين** — طابق المشاركين في التقويم مع مستخدمي الموقع وجهات الاتصال
- **بوابات الموافقة البشرية** — أوقف الأتمتة الحساسة مؤقتًا للمراجعة قبل المتابعة
- **إزالة تكرار التذكيرات** — سجّل التذكيرات لتجنب الإشعارات المكررة عند إعادة المحاولة أو التشغيل المتكرر
- **مزوّد TextBee SMS** — أرسل إشعارات الرسائل النصية المهيأة بعد الموافقات المطلوبة
- **إعداد خدمة Superdav المُدارة** — وفّر نقاط نهاية الخدمة المستضافة تلقائيًا للمواقع المدعومة
- **توزيع حزمة المرافق المتقدمة** — انشر أدوات المطورين الموثوقين بشكل منفصل عن بناء WordPress.org الأساسي
- **إصلاحات الموثوقية** — حسّن اختيار النموذج، ومهل الطلبات، ومعالجة نموذج الإعداد الافتراضي، ومعالجة نص الاستدلال، وإرشادات إعادة المحاولة، وتقوية رمز التقويم، وتقوية التذكيرات، وتشغيلات التهيئة الأولية، واستئناف القدرات المؤكدة، ومشكلات مراجعة تغليف WordPress.org

## التحسينات في v1.16.0

- **ذاكرة التخزين المؤقت لعميل AI** — أصبحت الآن مدعومة بـ transients للاستمرارية عبر الطلبات، مما يمنع فقدان البيانات في مهام الوكيل طويلة التشغيل
- **أداة كشط المواقع من المستوى 1** — أصبح sd-ai-agent/site-scrape متاحًا الآن افتراضيًا في Theme Builder لتحليل مواقع المنافسين ومواقع الإلهام
- **روابط إجراءات صف الإضافة** — تم تصحيحها وإعادة تسميتها لمزيد من الوضوح

## التحسينات في v1.10.0

- **إعادة هيكلة تكامل WooCommerce** — يستخدم الآن واجهات API الأصلية لـ WooCommerce لتحسين الموثوقية والتوافق
- **تحديث تلقائي لقائمة المزوّدين** — يكتشف تلقائيًا عند تفعيل الإضافات أو إلغاء تفعيلها ويحدّث قائمة المزوّدين المتاحة
- **مُنشئ مواقع محسّن** — تحسين خطط بناء المواقع متعددة المراحل مع استرداد أفضل من الأخطاء
- **مهارات واعية بالقالب** — أربع مهارات مدمجة جديدة تتكيّف تلقائيًا مع القالب النشط لديك

## إصلاحات الأخطاء في v1.10.0

- تم إصلاح قدرة `navigate-to` التي كانت تتسبب في حلقات إعادة تحميل لا نهائية في بعض صفحات الإدارة
- تم إصلاح قدرة `list-posts` التي لم تكن تحلّ أسماء التصنيفات والوسوم بشكل صحيح إلى أسماء لطيفة
- تم إصلاح أوامر WP-CLI التي كانت تفتقد الأسماء المستعارة لمساحة الأسماء بعد إعادة هيكلة سابقة
- تم إصلاح أتمتة الأحداث للتعامل بسلاسة مع المواقع التي لم تُنشأ فيها جداول الأتمتة بعد
- تم إصلاح قدرة `memory-save` لاستخدام بادئة مساحة الأسماء الصحيحة في مُنشئ تعليمات النظام
- تم إصلاح نتائج الأدوات العددية لتُغلّف بشكل صحيح قبل إعادتها إلى AI
- تم إصلاح إحصاءات الاستخدام للتعامل بشكل صحيح مع تنسيق مفتاح القدرة القديم عند الترقية من إصدارات أقدم
