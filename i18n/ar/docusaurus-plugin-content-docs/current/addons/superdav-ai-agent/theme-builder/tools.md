# الأدوات والتكوين {#tools-and-configuration}

يستخدم Theme Builder مجموعة من الأدوات لتحليل موقع WordPress الخاص بك وتصميمه وبنائه. في الإصدار 1.16.0، أصبحت **sd-ai-agent/site-scrape الآن أداة من المستوى 1**، ما يجعلها متاحة افتراضيًا في Theme Builder. يقدّم Superdav AI Agent v1.18.0 أيضًا حزمة مرافقة متقدمة موزّعة بشكل منفصل لتدفقات عمل المطوّرين الموثوقين؛ هذه الأدوات ليست جزءًا من حزمة WordPress.org الأساسية ويجب تثبيتها والتصريح بها بشكل منفصل.

## الأدوات المتاحة {#available-tools}

### أدوات المستوى 1 (متاحة دائمًا) {#tier-1-tools-always-available}

تتوفر أدوات المستوى 1 افتراضيًا في Theme Builder دون أي تكوين إضافي.

#### sd-ai-agent/site-scrape {#sd-ai-agentsite-scrape}

**الغرض:** تحليل المواقع الإلكترونية الحالية لاستخلاص إلهام التصميم وبنية المحتوى والوظائف.

**الإمكانات:**

- **تحليل المواقع الإلكترونية** — استخلاص وتحليل مواقع المنافسين أو مواقع الإلهام
- **استخلاص التصميم** — تحديد الألوان والخطوط وأنماط التخطيط
- **بنية المحتوى** — فهم تنظيم الصفحات والتسلسل الهرمي
- **اكتشاف الميزات** — تحديد plugins والوظائف
- **تحليل الأداء** — فحص سرعة الصفحة والتحسين
- **تحليل SEO** — مراجعة وسوم التعريف والبيانات المنظمة

**الاستخدام:**

```
حلّل تصميم example.com واستخدمه مصدر إلهام لموقعي.
```

**ما الذي يستخرجه:**

- لوحة الألوان واستخدام الألوان
- الطباعة (الخطوط والأحجام)
- بنية التخطيط والمسافات
- أنماط التنقل
- تنظيم المحتوى
- استخدام الصور والوسائط
- العناصر التفاعلية
- الاستجابة للأجهزة المحمولة

**القيود:**

- لا يمكنه استخلاص المواقع المحمية بكلمة مرور
- يلتزم بملف robots.txt وسياسات الموقع
- قد لا يلتقط المحتوى الديناميكي
- قد يكون تحليل المواقع المعتمدة بكثافة على JavaScript محدودًا
- قد تستغرق المواقع الكبيرة وقتًا أطول للتحليل

### أدوات المستوى 2 (اختيارية) {#tier-2-tools-optional}

تتوفر أدوات المستوى 2 عند تمكينها صراحةً في إعدادات Theme Builder.

#### التحليلات المتقدمة {#advanced-analytics}

توفر مقاييس أداء مفصلة:

- أوقات تحميل الصفحة
- Core Web Vitals
- درجة SEO
- درجة إمكانية الوصول
- درجة أفضل الممارسات

#### محسّن المحتوى {#content-optimizer}

يحلل ويقترح تحسينات لـ:

- سهولة القراءة
- تحسين SEO
- استخدام الكلمات المفتاحية
- بنية المحتوى
- موضع دعوة الإجراء

### أدوات المستوى 3 (Premium) {#tier-3-tools-premium}

تتطلب أدوات المستوى 3 تكوينًا إضافيًا أو ميزات premium.

#### مولّد المحتوى بالذكاء الاصطناعي {#ai-content-generator}

ينشئ محتوى لـ:

- أوصاف المنتجات
- صفحات الخدمات

- أوصاف التعريف
- نص دعوة الإجراء

#### أدوات التصميم المتقدمة {#advanced-design-tools}

توفر إمكانات تصميم متقدمة:

- إنشاء CSS مخصص
- إنشاء الحركات
- تصميم العناصر التفاعلية
- نظرية الألوان المتقدمة
- تحسين الطباعة

### أدوات الحزمة المرافقة المتقدمة (حزمة منفصلة) {#advanced-companion-tools-separate-package}

تُوزّع الحزمة المرافقة المتقدمة بشكل منفصل عن إصدار Superdav AI Agent الأساسي. وهي مخصصة لبيئات المطوّرين ومالكي المواقع الموثوقين حيث يسمح المسؤولون صراحةً للوكيل باستخدام أدوات عالية التأثير.

يمكن أن تتضمن إمكانات الحزمة المرافقة المتقدمة ما يلي:

- **أدوات نظام الملفات** — فحص الملفات المعتمدة وتعديلها في البيئات الموثوقة
- **أدوات قاعدة البيانات** — الاستعلام عن بيانات الموقع أو تحديثها عند التصريح صراحةً
- **أدوات WP-CLI** — تشغيل أوامر صيانة WordPress وفحصه
- **أدوات موزّع REST** — استدعاء REST endpoints المسجلة عبر تدفقات عمل إرسال مضبوطة
- **أدوات بناء plugin** — إنشاء هيكل أولي والتكرار على كود plugin
- **أدوات لقطة Git** — إنشاء لقطات قبل عمليات المطوّرين المحفوفة بالمخاطر
- **أدوات إدارة المستخدمين** — المساعدة في مراجعة سجلات المستخدمين أو تعديلها حيثما يُسمح بذلك
- **أدوات الاختبار المعياري** — تشغيل اختبارات معيارية للأداء أو الإمكانات لمراجعة التطوير

لا توثّق هذه الأدوات على أنها أدوات Theme Builder متاحة عمومًا لكل تثبيت. فهي متاحة فقط عند تثبيت الحزمة المرافقة المتقدمة وتمكينها وتقييدها بالمسؤولين الموثوقين.

## تكوين الأدوات {#tool-configuration}

### تمكين الأدوات {#enabling-tools}

لتمكين أدوات إضافية في Theme Builder:

1. انتقل إلى **Gratis AI Agent → Settings**
2. انتقل إلى **Theme Builder → Tools**
3. فعّل الأدوات أو عطّلها حسب الحاجة
4. احفظ الإعدادات

### أذونات الأدوات {#tool-permissions}

تتطلب بعض الأدوات أذونات:

- **استخلاص الموقع** — يتطلب الوصول إلى الإنترنت
- **التحليلات** — تتطلب تكامل Google Analytics
- **إنشاء المحتوى** — يتطلب مفاتيح API
- **الميزات المتقدمة** — قد تتطلب اشتراكًا premium
- **أدوات الحزمة المرافقة المتقدمة** — تتطلب الحزمة المتقدمة الموزّعة بشكل منفصل وثقة صريحة من المسؤول

### مفاتيح API وبيانات الاعتماد {#api-keys-and-credentials}

كوّن مفاتيح API للأدوات التي تتطلبها:

1. انتقل إلى **Gratis AI Agent → Settings → API Keys**
2. أدخل بيانات الاعتماد لكل أداة
3. اختبر الاتصال
4. احفظ بأمان

### تثبيت الحزمة المرافقة المتقدمة {#installing-the-advanced-companion-package}

تُصدر الحزمة المرافقة المتقدمة بشكل منفصل عن ملف ZIP الأساسي الخاص بـ WordPress.org. ثبّتها فقط من قناة التوزيع الرسمية للمشروع الخاصة بالإصدار، ثم قيّد الوصول إلى المسؤولين الموثوقين. راجع بوابات الموافقة البشرية وسلوك لقطة git قبل تمكين أدوات نظام الملفات أو قاعدة البيانات أو WP-CLI أو موزّع REST أو بناء plugin أو إدارة المستخدمين أو الاختبار المعياري على موقع إنتاج.

## استخدام sd-ai-agent/site-scrape {#using-sd-ai-agentsite-scrape}

### الاستخدام الأساسي {#basic-usage}

اطلب من Theme Builder تحليل موقع إلكتروني:

```
حلّل تصميم موقع منافسي على competitor.com
واقترح تحسينات تصميم لموقعي.
```

### تحليل محدد {#specific-analysis}

اطلب أنواعًا محددة من التحليل:

```
استخرج لوحة الألوان من example.com واستخدمها مصدر إلهام.
```

```
حلّل بنية التنقل في example.com وطبّق تنظيمًا مشابهًا
على موقعي.
```

```
افحص استجابة example.com للأجهزة المحمولة وتأكد من أن موقعي
مستجيب بالقدر نفسه.
```

### تحليل المقارنة {#comparison-analysis}

قارن بين عدة مواقع:

```
قارن تصميمي site1.com وsite2.com وأنشئ تصميمًا
يجمع أفضل العناصر من كليهما.
```

## قيود الأدوات والاعتبارات {#tool-limitations-and-considerations}

### تحديد المعدل {#rate-limiting}

- يخضع الاستخراج لتحديد المعدّل لمنع التحميل الزائد على الخادم
- قد يتم إبطاء الطلبات المتعددة إلى الموقع نفسه
- قد تستغرق المواقع الكبيرة وقتًا أطول للتحليل

### دقة المحتوى {#content-accuracy}

- قد لا يتم التقاط المحتوى الديناميكي بالكامل
- قد يكون المحتوى المعروض بواسطة JavaScript غير مكتمل
- قد لا تنعكس البيانات في الوقت الفعلي
- قد يكون بعض المحتوى خلف جدران دفع

### الاعتبارات القانونية والأخلاقية {#legal-and-ethical-considerations}

- احترم robots.txt وسياسات الموقع
- لا تستخرج محتوى محميًا بحقوق الطبع والنشر لإعادة إنتاجه
- استخدم التحليل للإلهام، وليس للنسخ
- تحقق من أن لديك حقوق استخدام المحتوى المستخرج
- اتبع شروط الخدمة للمواقع التي تم تحليلها

### تأثير الأداء {#performance-impact}

- قد يستغرق استخراج المواقع الكبيرة وقتًا
- قد تكون عمليات الاستخراج المتزامنة المتعددة أبطأ
- يؤثر اتصال الشبكة في السرعة
- قد تتطلب مجموعات البيانات الكبيرة مزيدًا من المعالجة

## أفضل الممارسات {#best-practices}

### استخدام تحليل الموقع {#using-site-analysis}

1. **حلّل مواقع متعددة** — اجمع الإلهام من عدة مصادر
2. **ركّز على البنية** — تعلّم أنماط التخطيط والتنظيم
3. **استخرج الألوان** — استخدم لوحات الألوان كنقاط بداية
4. **ادرس الطباعة** — حدّد مجموعات الخطوط التي تعجبك
5. **راجع التنقل** — افهم تنظيم القائمة

### الاستخراج الأخلاقي {#ethical-scraping}

1. **احترم robots.txt** — اتبع إرشادات الموقع
2. **لا تنسخ المحتوى** — استخدم التحليل للإلهام فقط
3. **تحقق من الحقوق** — تأكد من أنه يمكنك استخدام المحتوى المستخرج
4. **انسب المصادر** — اذكر مصادر الإلهام
5. **اتبع الشروط** — امتثل لشروط الخدمة الخاصة بالموقع

### تعظيم فعالية الأداة {#maximizing-tool-effectiveness}

1. **كن محددًا** — اطلب أنواعًا محددة من التحليل
2. **قدّم السياق** — اشرح غرض موقعك
3. **حدّد التوقعات** — صف أهداف التصميم لديك
4. **كرّر التحسين** — حسّن بناءً على النتائج
5. **اجمع بين الأدوات** — استخدم أدوات متعددة لتحليل شامل

## استكشاف الأخطاء وإصلاحها {#troubleshooting}

### الموقع لا يقبل الاستخراج {#site-wont-scrape}

- تحقق مما إذا كان الموقع متاحًا للعامة
- تأكد من أن robots.txt يسمح بالاستخراج
- جرّب موقعًا مختلفًا
- تحقق من اتصال الإنترنت
- اتصل بالدعم إذا استمرت المشكلة

### تحليل غير مكتمل {#incomplete-analysis}

- قد يحتوي الموقع على محتوى ديناميكي
- قد لا يتم عرض JavaScript بالكامل
- قد تنتهي مهلة المواقع الكبيرة
- جرّب تحليل صفحات محددة بدلًا من ذلك
- اطلب أنواع تحليل محددة

### أداء بطيء {#slow-performance}

- تستغرق المواقع الكبيرة وقتًا أطول للتحليل
- تكون عمليات الاستخراج المتزامنة المتعددة أبطأ
- يؤثر اتصال الشبكة في السرعة
- جرّب خلال ساعات خارج الذروة
- حلّل الأقسام الأصغر أولًا
