# الأدوات والإعدادات

يستخدم Theme Builder مجموعة من الأدوات لتحليل وتصميم وبناء موقعك على WordPress. في الإصدار 1.16.0، أصبحت أداة **sd-ai-agent/site-scrape أداة من المستوى 1 (Tier 1)**، مما يجعلها متاحة افتراضيًا في Theme Builder.

## الأدوات المتاحة

### أدوات المستوى 1 (متاحة دائمًا)

تتوفر أدوات المستوى 1 افتراضيًا في Theme Builder دون الحاجة إلى أي إعدادات إضافية.

#### sd-ai-agent/site-scrape

**الغرض:** تحليل المواقع الإلكترونية الحالية لاستخلاص الإلهام التصميمي، وهيكل المحتوى، والوظائف.

**القدرات:**

- **تحليل الموقع:** استخلاص وتحليل مواقع المنافسين أو المواقع الملهمة.
- **استخلاص التصميم:** تحديد الألوان والخطوط وأنماط التخطيط.
- **هيكل المحتوى:** فهم تنظيم الصفحة والتسلسل الهرمي.
- **اكتشاف الميزات:** تحديد الإضافات (plugins) والوظائف.
- **تحليل الأداء:** فحص سرعة الصفحة وتحسينها.
- **تحليل تحسين محركات البحث (SEO):** مراجعة العلامات الوصفية (meta tags) والبيانات المنظمة.

**طريقة الاستخدام:**

```
Analyze the design of example.com and use it as inspiration for my site.
```

**ما الذي يستخلصه:**

- لوحة الألوان واستخدام الألوان.
- الطباعة (الخطوط والأحجام).
- هيكل التخطيط والتباعد.
- أنماط التنقل.
- تنظيم المحتوى.
- استخدام الصور والوسائط.
- العناصر التفاعلية.
- الاستجابة على الأجهزة المحمولة.

**القيود:**

- لا يمكنه استخلاص البيانات من المواقع المحمية بكلمة مرور.
- يحترم ملفات robots.txt وسياسات الموقع.
- قد لا يلتقط المحتوى الديناميكي.
- قد يكون للتحليل محدودية في المواقع التي تعتمد بشكل كبير على JavaScript.
- قد تستغرق المواقع الكبيرة وقتًا أطول للتحليل.

### أدوات المستوى 2 (اختياري)

تتوفر أدوات المستوى 2 عند تمكينها بشكل صريح في إعدادات Theme Builder.

#### Advanced Analytics (التحليلات المتقدمة)

يوفر مقاييس أداء مفصلة:

- أوقات تحميل الصفحة.
- Core Web Vitals.
- درجة تحسين محركات البحث (SEO score).
- درجة إمكانية الوصول (Accessibility score).
- درجة أفضل الممارسات (Best practices score).

#### Content Optimizer (مُحسّن المحتوى)

يحلل ويقترح تحسينات لـ:

- سهولة القراءة.
- تحسين محركات البحث (SEO optimization).
- استخدام الكلمات المفتاحية.
- هيكل المحتوى.
- وضع عبارات الحث على اتخاذ إجراء (Call-to-action placement).

### أدوات المستوى 3 (متميزة/مدفوعة)

تتطلب أدوات المستوى 3 إعدادات إضافية أو ميزات متميزة.

#### AI Content Generator (مولد محتوى الذكاء الاصطناعي)

يولد المحتوى لـ:

- أوصاف المنتجات.
- صفحات الخدمات.
- منشورات المدونة.
- الأوصاف التعريفية (Meta descriptions).
- نصوص عبارات الحث على اتخاذ إجراء.

#### Advanced Design Tools (أدوات التصميم المتقدمة)

يوفر قدرات تصميم متقدمة:

- إنشاء CSS مخصص.
- إنشاء الرسوم المتحركة (Animation).
- تصميم العناصر التفاعلية.
- نظرية الألوان المتقدمة.
- تحسين الطباعة (Typography optimization).

## إعدادات الأدوات

### تمكين الأدوات

لتمكين أدوات إضافية في Theme Builder:

1. انتقل إلى **Gratis AI Agent → Settings**
2. اذهب إلى **Theme Builder → Tools**
3. قم بتبديل الأدوات (تشغيل/إيقاف) حسب الحاجة.
4. احفظ الإعدادات.

### أذونات الأدوات

تتطلب بعض الأدوات أذونات معينة:

- **استخلاص الموقع (Site scraping):** يتطلب الوصول إلى الإنترنت.
- **التحليلات (Analytics):** يتطلب دمج Google Analytics.
- **توليد المحتوى (Content generation):** يتطلب مفاتيح واجهة برمجة التطبيقات (API keys).
- **الميزات المتقدمة (Advanced features):** قد يتطلب اشتراكًا متميزًا.

### مفاتيح واجهة برمجة التطبيقات وبيانات الاعتماد (API Keys and Credentials)

قم بإعداد مفاتيح واجهة برمجة التطبيقات للأدوات التي تتطلبها:

1. اذهب إلى **Gratis AI Agent → Settings → API Keys**
2. أدخل بيانات الاعتماد لكل أداة.
3. اختبر الاتصال.
4. احفظ البيانات بأمان.

## استخدام sd-ai-agent/site-scrape

### الاستخدام الأساسي

اطلب من Theme Builder تحليل موقع إلكتروني:

```
Analyze the design of my competitor's site at competitor.com
and suggest design improvements for my site.
```

### التحليل المحدد

اطلب أنواعًا محددة من التحليل:

```
Extract the color palette from example.com and use it as inspiration.
```

```
Analyze the navigation structure of example.com and apply similar
organization to my site.
```

```
Check the mobile responsiveness of example.com and ensure my site
is equally responsive.
```

### تحليل المقارنة

قارن بين مواقع متعددة:

```
Compare the designs of site1.com and site2.com and create a design
that combines the best elements of both.
```

## قيود واعتبارات الأدوات

### تحديد معدل الطلبات (Rate Limiting)

- يتم تحديد معدل الاستخلاص لمنع إجهاد الخادم.
- قد يتم تقييد الطلبات المتعددة لنفس الموقع.
- قد تستغرق المواقع الكبيرة وقتًا أطول للتحليل.

### دقة المحتوى

- قد لا يتم التقاط المحتوى الديناميكي بالكامل.
- قد يكون المحتوى المُنشأ بواسطة JavaScript غير مكتمل.
- قد لا تنعكس البيانات في الوقت الفعلي.
- قد يكون بعض المحتوى خلف جدران الدفع (paywalls).

### الاعتبارات القانونية والأخلاقية

- احترام ملفات robots.txt وسياسات الموقع.
- لا تقم باستخلاص المحتوى المحمي بحقوق الطبع والنشر لإعادة إنتاجه.
- استخدم التحليل للإلهام، وليس للنسخ.
- تأكد من أن لديك الحق في استخدام المحتوى المستخلص.
- اتبع شروط خدمة المواقع التي يتم تحليلها.

### التأثير على الأداء

- قد يستغرق استخلاص المواقع الكبيرة وقتًا.
- قد تكون عمليات الاستخلاص المتزامنة المتعددة أبطأ.
- يؤثر اتصال الشبكة على السرعة.
- قد تتطلب مجموعات البيانات الكبيرة المزيد من المعالجة.

## أفضل الممارسات

### استخدام تحليل الموقع

1. **تحليل مواقع متعددة:** اجمع الإلهام من مصادر مختلفة.
2. **التركيز على الهيكل:** تعلم أنماط التخطيط والتنظيم.
3. **استخلاص الألوان:** استخدم لوحات الألوان كنقاط انطلاق.
4. **دراسة الطباعة:** حدد مجموعات الخطوط التي تعجبك.
5. **مراجعة التنقل:** افهم تنظيم القوائم.

### الاستخلاص الأخلاقي

1. **احترام robots.txt:** اتبع إرشادات الموقع.
2. **عدم نسخ المحتوى:** استخدم التحليل للإلهام فقط.
3. **التحقق من الحقوق:** تأكد من أنه يمكنك استخدام المحتوى المستخلص.
4. **نسب المصادر:** اذكر مصادر الإلهام.
5. **اتباع الشروط:** التزام بشروط خدمة الموقع.

### تعظيم فعالية الأدوات

1. **كن محددًا:** اطلب أنواعًا محددة من التحليل.
2. **قدم السياق:** اشرح الغرض من موقعك.
3. **تحديد التوقعات:** صف أهدافك التصميمية.
4. **التكرار:** قم بالتحسين بناءً على النتائج.
5. **دمج الأدوات:** استخدم أدوات متعددة لإجراء تحليل شامل.

## استكشاف الأخطاء وإصلاحها

### الموقع لا يتم استخلاصه

- تحقق مما إذا كان الموقع متاحًا للعامة.
- تحقق مما إذا كان ملف robots.txt يسمح بالاستخلاص.
- جرب موقعًا مختلفًا.
- تحقق من اتصال الإنترنت.
- اتصل بالدعم إذا استمرت المشكلة.

### التحليل غير مكتمل

- قد يحتوي الموقع على محتوى ديناميكي.
- قد لا يتم عرض JavaScript بالكامل.
- قد تنتهي صلاحية المواقع الكبيرة (timeout).
- حاول تحليل صفحات محددة بدلاً من ذلك.
- اطلب أنواع تحليل محددة.

### الأداء البطيء

- تستغرق المواقع الكبيرة وقتًا أطول للتحليل.
- عمليات الاستخلاص المتزامنة المتعددة أبطأ.
- يؤثر اتصال الشبكة على السرعة.
- حاول خلال ساعات غير الذروة.
- قم بتحليل أقسام أصغر أولاً.
