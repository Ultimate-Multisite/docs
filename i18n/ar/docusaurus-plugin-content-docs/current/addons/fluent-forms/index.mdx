# Fluent Forms

يتيح هذا المكوّن الإضافي إمكانية إضافة حدود إلى نماذج Fluent Forms على Multisite Ultimate في المنتجات، العضويات أو المواقع.
يستخدم Fluent Forms جداول مخصصة بدلاً من أنواع المنشورات المخصصة، لذا لا تعمل خيارات التحديد العادية مع Fluent Forms.

## الميزات

- إضافة حدود لنماذج Fluent Forms أو نماذج Fluent Forms الحوارية
- يحصل العملاء على رسالة ودية تشجعهم على ترقية خطتهم عند بلوغ الحد
