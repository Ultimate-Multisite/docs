# الشبكة المتعددة

يُوسّع إضافة Ultimate Multisite: Multinetwork منصة Multisite Ultimate الخاصة بك بميزات شبكة متعددة قوية. تتيح هذه الإضافة لعملائك شراء وإدارة شبكات WordPress المتعددة بالكامل، وليس فقط المواقع الفردية.

## الميزات الرئيسية

* **توفير الشبكة**: إنشاء وتكوين شبكات WordPress المتعددة بالكامل تلقائيًا لعملائك
* **إدارة الشبكة**: قدرات إدارة كاملة لبيئات الشبكة المتعددة
* **شبكات معزولة**: تعمل كل شبكة عميل بشكل مستقل مع جداول قاعدة بياناتها الخاصة وتكوينها
* **هندسة قابلة للتوسع**: التعامل مع شبكات متعددة بكفاءة مع تخصيص موارد محسّن
* **التكامل**: يتكامل بسلاسة مع أنظمة الاشتراك والفوترة في Multisite Ultimate

## حالات الاستخدام

* **استضافة العلامة البيضاء**: توفير شبكات WordPress المتعددة المخصصة للعلامة التجارية الخاصة بالعملاء
* **حلول الوكالات**: تمكين الوكالات من إدارة شبكات عملاء متعددة من منصة واحدة
* **نشر المؤسسات**: دعم المنظمات الكبيرة التي تتطلب شبكات متعددة معقدة
* **منصات التعليم**: إنشاء شبكات منفصلة للمدارس أو الأقسام المختلفة

## المتطلبات

* Multisite Ultimate 2.0.0 أو أعلى
* تثبيت WordPress Multisite
* PHP 7.4 أو أعلى
* MySQL 5.6 أو أعلى

## التثبيت

1. قم بتحميل ملفات الإضافة إلى دليل `/wp-content/plugins/ultimate-multisite-multinetwork`
2. فعّل الإضافة عبر شاشة 'الإضافات' في WordPress
3. قم بتكوين إعدادات الشبكة المتعددة عبر Multisite Ultimate → الإعدادات

## الأسئلة المتداولة

## ما هو الشبكة المتعددة؟

الشبكة المتعددة هي ميزة في WordPress تسمح لك بتشغيل شبكات WordPress متعددة مستقلة على تثبيت WordPress واحد.

## كيف يختلف هذا عن WordPress المتعدد العادي؟

يتيح WordPress المتعدد العادي وجود مواقع متعددة داخل شبكة واحدة. أما الشبكة المتعددة فتمكن من وجود شبكات متعددة، كل منها يحتوي على مواقع خاصة به، جميعها ضمن تثبيت WordPress واحد.

## هل يتطلب هذا تكوين خادم خاص؟

نعم، تتطلب الشبكة المتعددة إعدادًا وتكوينًا خاصًا للخادم. يرجى مراجعة الوثائق للحصول على تعليمات التثبيت المفصلة.
