# سجل تغييرات الشبكات المتعددة

### 1.3.0
* جديد: معاين قوالب الشبكة — تصفّح قوالب الشبكة وعاينها عبر لوحة مباشرة قبل الشراء.
* جديد: منتقي قوالب الشبكة عند الدفع — يمكن للعملاء اختيار قالب شبكة أثناء مسار دفع التسجيل.
* جديد: تسجيل دخول موحد عبر رابط سحري بين الشبكات — يمكن للعملاء الوصول إلى الشبكات الفرعية بسلاسة عبر مصادقة الرابط السحري.
* جديد: استنساخ وسائط الشبكة — يتم تكرار وسائط الموقع بشكل صحيح عند استنساخ الشبكات من قالب.
* إصلاح: يعمل استنساخ الشبكات الآن من دون الإضافة متعددة المستأجرين؛ تتم إعادة منح صلاحيات المدير الفائق بعد الاستنساخ.
* إصلاح: أصبح إنشاء الشبكات الآن متوافقًا مع WooCommerce وجميع بوابات الدفع.
* إصلاح: يتم الآن تضمين رقم المنفذ في توليد النطاق الاحتياطي.
* إصلاح: تم نقل CSS منتقي القوالب إلى ورقة أنماط مُدرجة في قائمة التحميل للعرض الصحيح على جميع القوالب.
* إصلاح: الحماية من مفتاح `network_id` غير المعرّف في نطاق استعلام الموقع.
* إصلاح: فحص إذن الشبكة قبل تبديل السياق يمنع أخطاء خاصية جدول `wpdb`.
