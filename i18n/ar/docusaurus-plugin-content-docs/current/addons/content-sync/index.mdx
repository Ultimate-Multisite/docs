# مزامنة المحتوى

Multisite Ultimate: مزامنة المحتوى

يتيح هذا المكوّن الإضافي لـ Multisite Ultimate لمسؤولي الشبكة مزامنة المحتوى من أي موقع إلى مواقع أخرى في الشبكة. يدعم جميع أنواع المنشورات بما في ذلك المقالات، الصفحات، أنواع المنشورات المخصصة، وقوالب Elementor. يتضمن المكوّن واجهة معالج متعددة الخطوات لاختيار المحتوى والمواقع المستهدفة، مع معالجة دفعات وتحديثات تقدم في الوقت الحقيقي.

## الميزات

* اختيار المحتوى: شبكة بطاقات مرئية مع صور مصغرة وتصفية حسب نوع المنشور
* جميع أنواع المنشورات: مزامنة المقالات، الصفحات، المنتجات، قوالب Elementor، وأي نوع منشور مخصص
* اختيار الموقع: حدد المواقع التي تريد مزامنتها، مع استبعاد موقع المصدر وتحديد المواقع الفرعية مسبقاً
* معالجة الدفعات: مزامنة حتى 10 مواقع في المرة الواحدة مع تحديثات التقدم
* مزامنة الوسائط: نسخ تلقائي وإعادة تعيين ملفات الوسائط (الصور، الفيديوهات) من محتوى المنشور إلى المواقع المستهدفة
* مزامنة التصنيف: الحفاظ على الفئات، الوسوم، والتصنيفات المخصصة
* دعم Elementor: دعم كامل لقوالب Elementor مع إعادة توليد CSS وإعادة تعيين معرف القالب
* حل النزاعات: خيار استبدال المحتوى الموجود أو تخطيه
* إدارة التخزين المؤقت: مسح تلقائي لتخزين Elementor المؤقت على المواقع المستهدفة بعد المزامنة

## المتطلبات

* WordPress 5.3 أو أحدث
* PHP 7.4 أو أحدث
* إضافة Multisite Ultimate (مفعلة)
* إضافة Elementor (اختيارية، مطلوبة لمزامنة قوالب Elementor)

## التثبيت

1. رفع مجلد 'ultimate-multisite-content-sync' إلى دليل '/wp-content/plugins/'
2. تفعيل الإضافة عبر قائمة 'الإضافات' في مدير شبكة WordPress
3. ضبط الإعدادات في Ultimate Multisite → Settings → Content Sync
4. الانتقال إلى Ultimate Multisite → Content Sync لبدء المزامنة
