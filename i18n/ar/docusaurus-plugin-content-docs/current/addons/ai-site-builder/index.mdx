# AI Site Builder

ultimate-multisite-ai-site-builder

## المتطلبات

- WordPress 5.3 أو أعلى
- PHP 7.4 أو أعلى
- ملحق Multisite Ultimate (نشط)

## التثبيت

1. قم بتحميل ملفات الملحق إلى دليل `/wp-content/plugins/` الخاص بك
2. فعّل الملحق من خلال قائمة 'الإضافات' في WordPress
3. قم بتكوين إعدادات الملحق في مدير الشبكة

## التطوير

### الإعداد

```bash
# تثبيت الاعتمادات
composer install
npm install

# تشغيل الاختبارات
npm run test

# تشغيل فحوصات معايير الكود
vendor/bin/phpcs
vendor/bin/phpstan

# إصلاح مشكلات نمط الكود
vendor/bin/phpcbf

# بناء للإنتاج
npm run build
```

### الاختبار

يحتوي الملحق على إطار اختبار شامل:

- **اختبارات الوحدة**: اختبارات مبنية على PHPUnit مع مجموعة اختبارات WordPress
- **فئة الاختبار الأساسية**: توفر طرق مساعدة لسيناريوهات الاختبار الشائعة
- **تغطية الاختبار**: مُكوَّنة لتتبع تغطية الكود
- **اختبار الشبكة المتعددة**: تُشغَّل الاختبارات في بيئة الشبكة المتعددة

### معايير الكود

- **معايير ترميز WordPress**: تُطبَّق عبر PHPCS
- **التوافق مع PHP 7.4+**: ميزات PHP الحديثة مدعومة
- **التحليل الثابت**: تكامل PHPStan للتحقق من الأنواع
- **تحديث الكود**: قواعد Rector لتحديث PHP

### نظام البناء

- **معالجة الأصول**: تصغير ملفات CSS/JS
- **الترجمة**: إنشاء ملفات POT
- **التغليف**: إنشاء أرشيف للتوزيع
- **وضع التطوير**: أصول غير مصغرة للخطأ

## التكوين

يدعم الملحق خيارات تكوين مختلفة يمكن الوصول إليها عبر واجهة مدير الشبكة.

## الخطافات والمرشحات

### الإجراءات

- `ultimate-multisite-ai-site-builder_init` - يُطلق بعد تهيئة الملحق
- `ultimate-multisite-ai-site-builder_loaded` - يُطلق بعد تحميل جميع ملفات الملحق

### المرشحات

- `ultimate-multisite-ai-site-builder_settings` - تعديل إعدادات الملحق
- `ultimate-multisite-ai-site-builder_enabled` - تجاوز حالة تمكين/تعطيل الملحق
