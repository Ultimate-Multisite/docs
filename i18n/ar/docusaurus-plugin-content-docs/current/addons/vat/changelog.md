# سجل تغييرات VAT

الإصدار 1.0.7 - تم إصداره في 2026-02-03

* إصلاح: كانت أرقام VAT الألمانية (DE) تفشل في التحقق بسبب أخطاء VIES SOAP service MS_MAX_CONCURRENT_REQ. تم الانتقال من SOAP API القديم إلى EU VIES REST API الرسمي لتحقق أكثر موثوقية.
* إصلاح: يتم الآن قبول أرقام VAT عندما تكون خدمة VIES غير متاحة مؤقتًا، بدلًا من رفضها بشكل غير صحيح. يتم إنشاء إدخال سجل للمراجعة لاحقًا.
* إصلاح: أصبح حقل VAT/Tax ID مرئيًا الآن لجميع البلدان، وليس لأعضاء الاتحاد الأوروبي فقط. يتم تخزين معرّفات الضرائب لغير الاتحاد الأوروبي (مثل أرقام Swiss CHE) لعرضها في الفاتورة دون تحقق VIES.
* إصلاح: كان البحث عن بادئة VAT لليونان (GR)، وموناكو (MC)، وجزيرة مان (IM) معطلًا بسبب منطق بحث غير صحيح في المصفوفة.
* تحسين: تمت إزالة تبعية التطوير غير الضرورية wp-cli-bundle التي قد تسبب أخطاء فادحة عند استخدامها إلى جانب الملحق Site Exporter.
* تحسين: تمت إضافة مجموعة اختبارات وحدة شاملة (53 اختبارًا).

الإصدار 1.0.6 - تم إصداره في 2026-01-25

* إصلاح: أصبحت معاملات B2B المحلية تفرض VAT بشكل صحيح الآن. وفقًا لقواعد VAT في الاتحاد الأوروبي، لا ينطبق الاحتساب العكسي إلا على معاملات B2B عبر الحدود، وليس عندما يطابق بلد العميل بلد الشركة.

الإصدار 1.0.5 - تم إصداره في 2026-01-22

* إصلاح: لم يكن الملحق يتم تحميله أو يعمل بشكل صحيح.
* تغيير: تم تبديل مصدر بيانات معدلات VAT من euvatrates.com إلى مستودع ibericode/vat-rates للحصول على بيانات أكثر موثوقية وتتم صيانتها بنشاط.
* إصلاح: تم تصحيح خطأ مطبعي في قيمة خيار super_reduced_rates.
* تحسين: تمت إضافة فحوصات null للتعامل مع رمز البلد لمنع الأخطاء.
* تغيير: تمت إزالة ملفات الترجمة المضمّنة لصالح التحديثات التلقائية عبر Traduttore.

الإصدار: 1.0.3 - تم إصداره في 2025-09-28
