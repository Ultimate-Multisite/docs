# إضافات Ultimate Multisite

تمتد الإضافات Ultimate Multisite بميزات ودمج إضافية. معظم الإضافات مدفوعة ويمكن شراؤها من [ultimatemultisite.com](https://ultimatemultisite.com).

## التثبيت

يمكن تثبيت الإضافات بطريقتين:

1. **من خلال صفحة الإضافة** -- انتقل إلى صفحة إدارة الإضافات في لوحة تحكم WordPress
2. **التحميل اليدوي** -- قم بتحميل من [حسابك](https://ultimatemultisite.com/my-account/downloads/) وقم بتحميله عبر الإضافات > إضافة جديد
