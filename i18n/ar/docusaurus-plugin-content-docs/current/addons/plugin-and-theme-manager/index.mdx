# Plugin & Theme Manager

Multisite Ultimate: Plugin and Theme Manager

تحرير أو إخفاء معلومات التعريف (العنوان، المؤلف، الصورة المصغرة والوصف) للإضافات والقوالب المثبتة. إنشاء فئات للإضافات والقوالب للسماح لمستخدميك بتصفيةها وإضافة صفحة إضافات مخصصة جميلة إلى لوحة مستخدميك.

لاستخدام الإضافة، يمكنك ببساطة الانتقال إلى صفحات الإضافات أو القوالب في مدير الشبكة الخاص بك والنقر على "تحرير معلومات الإضافة/القالب".

## التثبيت

1. رفع 'ultimate-multisite-plugin-and-theme-manager' إلى دليل '/wp-content/plugins/'
2. تفعيل الإضافة من خلال قائمة 'الإضافات' في مدير شبكة WordPress

## دليل الاستخدام

Ultimate Multisite: Plugin and Theme Manager هو إضافة قوية **مميزة** لـ Ultimate Multisite تمنحك السيطرة الكاملة على طريقة رؤية وتفاعل مستخدمي شبكتك مع خيارات الإضافة والقالب على شبكتك.

يؤدي هذا المكوّن ثلاث أغراض:

1. يتيح لك تغيير بيانات التعريف بالكامل لكل من الإضافات والقوالب. يشمل ذلك الاسم، الوصف، اسم المؤلف، وحتى الصور المصغرة!
2. يتيح لك تصنيف الإضافات والقوالب، للسماح لمستخدميك بفرزها بسهولة.
3. يمنحك خيار استبدال صفحة الإضافات القديمة المملة بصفحة شبيهة بالقوالب جديدة مع فرز حسب الفئات.

بعد تنزيله وتثبيته، ستظهر علامة تبويب جديدة - _إعدادات الإضافة_ \- في صفحة إعدادات Ultimate Multisite.
