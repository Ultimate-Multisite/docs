# Site Exporter سجل التغييرات

الإصدار: 1.0.4 - أصدر في 2025-11-25

* تصحيح التوافق مع Ultimate Multisite 2.4.

الإصدار: 1.0.2 - أصدر في 2025-09-28

* إعادة تسمية البادئة إلى ultimate-multisite؛ تحديث نطاق النص؛ رفع الإصدار.

### الإصدار 1.0.1 - أصدر في 2023-08-09

* تحسين: الآن يضمن الحصول على مسار الملف الصحيح لملف المستورد؛
* تصحيح: تعطيل ذاكرة التخزين المؤقت للكائنات لمنع الأخطاء أثناء الاستيراد؛
* تصحيح: تم إصلاح المشكلة التي تجعل رابط تنزيل إضافة المستورد لا يعمل؛

### الإصدار 1.0.0 - أصدر في 2022-12-23

* تصحيح: نافذة منبثقة نموذج التصدير لا تُحمَّل؛
* تصحيح: ملفات JavaScript لا تُحمَّل في صفحة التصدير؛
* داخلي: عملية بناء جديدة؛

### الإصدار 1.0.0-beta.4 - أصدر في 2022-01-13

* داخلي: أضيف مولد الخطافات والمرشحات؛
* داخلي: أضيف مخططات Ultimate Multisite لتحسين جودة حياة المطور؛
