# سجل التغييرات لـ Ultimate AI Connector لنقاط النهاية المتوافقة {#ultimate-ai-connector-for-compatible-endpoints-changelog}

## الإصدار 2.0.0 — صدر في 2026-04-24 {#version-200--released-on-2026-04-24}

* جديد: دعم متعدد المزودين — يمكنك الآن تهيئة نقاط نهاية الذكاء الاصطناعي (AI) المتعددة وتوجيه الطلبات مع التراجع التلقائي عبر المزودين.
* إصلاح: دمج حزمة تطوير البرامج (SDK) متعددة المزودين مع معرفات المزود الصحيحة، وعناوين التسجيل، وتحديد نطاق مرشح HTTP لكل مزود.
* إصلاح: بطاقة المزود الجديدة تتوسع تلقائيًا عند الإضافة؛ وتحديث كسر التخزين المؤقت (cache busting) عند تحديث الـ plugin.
* إصلاح: مساحة اسم فئة المزود الديناميكية لـ eval() والسلوك التلقائي للتوسع للمزودين المضافين حديثًا.
* إصلاح: يستخدم مكونات Card/CardBody/CardHeader/CardDivider المستقرة (لم تعد تجريبية) لضمان التوافق مع WordPress 6.9+.
* إصلاح: استبدال مقبض السحب (DragHandle) غير المحدد بأيقونة القبضة (grip icon) اليونيكود لإعادة ترتيب المزود بالسحب.
* إصلاح: إزالة طلب HTTP المعطّل الذي كان يتم إطلاقه عند تحميل كل صفحة.
* تحسين: ترقية سير عمل GitHub Actions إلى Node.js 24.
