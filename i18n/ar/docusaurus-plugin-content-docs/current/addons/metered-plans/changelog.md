# سجل تغييرات الخطط المقاسة

الإصدار 1.1.0 - صدر في 2026-05-05
- جديد: فوترة رموز AI للمواقع الفرعية متعددة المواقع — تتبّع وفوترة استخدام رموز AI عبر مواقع العملاء مع أسعار قابلة للتهيئة لكل رمز
- جديد: أُعيد بناء فرض الموصّل مع اكتشاف حدود ديناميكي وكتابة مباشرة، مما يضمن دقة فورية عبر جميع الموصّلات
- إصلاح: أصبحت ترقيات جداول قاعدة البيانات الآن متوافقة بشكل صحيح مع تعريفات مخطط BerlinDB، مما يمنع فشل الترقية في التثبيتات الجديدة
- إصلاح: تم تحويل استدعاءات ترقية قاعدة البيانات إلى التنسيق الصحيح، مما يحل حالات فشل الترقية الصامتة
- إصلاح: أصبحت القيم الكسرية مقبولة الآن في حقل إدخال ترميز تجاوز استخدام AI
- إصلاح: تم حل الأخطاء الفادحة ومشكلات التهيئة المزدوجة عند تحميل الإضافة
- محسّن: تمت إضافة سكربت npm باسم check-env حتى تقوم بيئات المطورين بتهيئة نفسها عند التشغيل الأول
