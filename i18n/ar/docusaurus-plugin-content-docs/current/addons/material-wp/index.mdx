# Material WP

واجهة إدارة WordPress كانت كما هي منذ زمن طويل. لكن هل تساءلت يومًا كيف سيبدو إصدار Material Design لواجهة الإدارة؟

Material WP هو قالب إداري تم تطويره لمنحك ذلك المظهر والشعور، مما يساعدك على تزويد عملائك بلوحة معلومات أكثر تخصيصًا.

## Features

- دعم CSS مخصص
- إعدادات تصميم سهلة
- إعادة ترتيب القوائم
- استيراد/تصدير سهل

## Installation

1. رفع 'material-wp' إلى دليل '/wp-content/plugins/'
2. تفعيل الإضافة من خلال قائمة 'الإضافات' في ووردبريس

## Usage Guide
