# Mailchimp

Multisite Ultimate: تكامل مايلتشيمب

دمج شبكة Multisite Ultimate الخاصة بك مع Mailchimp. يمكنك ربط بعض الخطط بقوائم Mailchimp وإضافة المشتركين تلقائيًا إلى تلك القوائم عند التسجيل.

## التثبيت

1. قم بتحميل 'ultimate-multisite-mailchimp' إلى دليل '/wp-content/plugins/'
2. فعّل الإضافة عبر قائمة 'الإضافات' في مدير شبكة WordPress الخاصة بك

## دليل الاستخدام

مزامنة رسائل البريد الإلكتروني للمشتركين مع قوائم Mailchimp سهلة جدًا باستخدام إضافة Ultimate Multisite: تكامل مايلتشيمب. يهدف هذا الدليل إلى مساعدتك في إعداد كل شيء!

## الحصول على مفتاح API

قبل أن نبدأ في التعمق في الإعدادات المختلفة المتاحة، نحتاج إلى مفتاح API الخاص بـ Mailchimp. سيسمح هذا لـ Ultimate Multisite بالتواصل مع واجهة Mailchimp API لاسترجاع القوائم والمجموعات، وإضافة عناوين البريد الإلكتروني إلى تلك القوائم والمجموعات عند تسجيل المستخدمين.

[حول مفاتيح API | Mailchimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

## وضع التكامل

توفر إضافة تكامل Mailchimp وضعين “تكامل”، مما يعني وجود طريقتين مختلفتين لاستخدام الإضافة لمزامنة عناوين البريد الإلكتروني.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_وضعان تكامل لمرونة أكبر!_

### الوضع 1: قوائم متعددة

الوضع الافتراضي يستخدم **قوائم متعددة**. سيسمح لك هذا الوضع باختيار قوائم Mailchimp متعددة لكل خطة. هذه هي الخيار الأفضل إذا قمت بتقسيم مستخدميك إلى قوائم مختلفة.

على سبيل المثال، لنفترض أن لديك 3 قوائم مختلفة على حسابك في Mailchimp: **النشرة الإخبارية**، **الخطة A**، و**الخطة B**. تريد أن يُدرج جميع المستخدمين، بغض النظر عن خطتهم، أيضًا في قائمة النشرة الإخبارية، حتى لا يفوتوا تحديثاتك.

لتحقيق ذلك، يمكنك استخدام وضع القوائم المتعددة لإضافة القوائم **النشرة الإخبارية** و**الخطة A** إلى إعدادات Mailchimp الخاصة بالخطة A، و**النشرة الإخبارية** و**الخطة B** إلى إعدادات Mailchimp الخاصة بالخطة B.

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_اختر القوائم الافتراضية للحسابات الجديدة والحسابات الملغاة_

### الوضع 2: قائمة واحدة + مجموعات متعددة

يسمح لك Mailchimp بتقسيم قائمتك إلى مجموعات أصغر من المستخدمين تُسمى **مجموعات**. يتيح لك ذلك مزيدًا من التحكم، حيث يمكنك إرسال حملة إلى مجموعة محددة من المشتركين دون الحاجة إلى إنشاء نسخ متعددة من نفس الحملة على Mailchimp (الذي يجعل من الصعب تحليل مقاييس تلك الحملة لاحقًا).

على سبيل المثال، يمكنك أن يكون لديك قائمة واحدة تُسمى **مشتركو شبكتي**، داخلها مجموعات لـ **الخطة A** و**الخطة B**.

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_اختر القوائم الافتراضية والمجموعات، إذا كنت تستخدم وضع التكامل 'قائمة واحدة + مجموعات متعددة'_

## القوائم والمجموعات الافتراضية

بعد اختيار وضع التكامل الذي يناسب حالتك الخاصة، حان الوقت لاختيار القوائم الافتراضية (والمجموعات، حسب الوضع). ستُستخدم القائمة الافتراضية عندما لا يختار أي خطة قائمة محددة.

يمكنك أيضًا اختيار القائمة الافتراضية للحسابات الملغاة. في كل مرة ينهي فيها المستخدم اشتراكه ويزيل حسابه من شبكتك، سيتم نقل عنوان بريده الإلكتروني إلى القائمة المخصصة للحسابات الملغاة.

## تمكين الاشتراك المزدوج

على نفس صفحة الإعدادات، ستتمكن من تمكين **الاشتراك المزدوج**. ستضيف هذه الميزة المستخدم إلى كل من القائمة الافتراضية المختارة و**القائمة المختارة لخطة المستخدم**. إذا لم يتم تمكين هذه الميزة، فسيتم إضافة المستخدم فقط إلى القائمة المختارة لخطة المستخدم. إذا لم يتم تعيين أي قائمة لتلك الخطة، فسيتم إضافة المستخدم إلى القائمة الافتراضية المختارة.

<!-- Screenshot unavailable: MailChimp double subscription toggle setting -->

_يمكنك أن يكون لديك مشتركين في القوائم الافتراضية والقائمة الخاصة بالخطط عند التسجيل_

## اختيار القوائم والمجموعات لكل خطة

اذهب إلى **الخطط** الخاصة بك واختر الخطة التي تريد تعديلها. لاحظ أنه في صندوق الميتا **خيارات متقدمة**، سيكون متاحًا علامة تبويب جديدة تُسمى Mailchimp.

في تلك العلامة، ستتمكن من اختيار قوائم متعددة أو قائمة واحدة ومجموعات متعددة (حسب وضع التكامل) لتلك الخطة.

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_اختر القوائم والمجموعات التي تريد استخدامها لكل خطتك_

عند تسجيل مستخدم جديد لتلك الخطة، سيتم إضافة عنوان بريده الإلكتروني إلى القوائم والمجموعات المختارة!
