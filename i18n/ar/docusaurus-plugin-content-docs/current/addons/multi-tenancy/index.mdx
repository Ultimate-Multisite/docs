# التعددية

توزيع المواقع الفرعية عبر الخوادم وقواعد البيانات لتحسين الأداء والأمان

## المتطلبات

- WordPress 5.3 أو أحدث
- PHP 7.4 أو أحدث
- إضافة Multisite Ultimate (نشطة)

## التثبيت

1. قم بتحميل ملفات الإضافة إلى دليل `/wp-content/plugins/` الخاص بك
2. فعّل الإضافة من خلال قائمة 'الإضافات' في ووردبريس
3. قم بتكوين إعدادات الإضافة في مدير الشبكة

## التطوير

### الإعداد

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### الاختبار

الإضافة تتضمن إطار اختبار شامل:

- **الاختبارات الوحدوية**: اختبارات مبنية على PHPUnit مع مجموعة اختبارات ووردبريس
- **فئة الاختبار الأساسية**: توفر طرق مساعدة لسيناريوهات الاختبار الشائعة
- **تغطية الاختبار**: مُكوَّنة لتتبع تغطية الكود
- **اختبار الشبكة المتعددة**: تُشغَّل الاختبارات في بيئة الشبكة المتعددة

### معايير الكود

- **معايير ترميز ووردبريس**: تُطبَّق عبر PHPCS
- **التوافق مع PHP 7.4+**: تدعم ميزات PHP الحديثة
- **التحليل الثابت**: تكامل PHPStan للتحقق من الأنواع
- **تحديث الكود**: قواعد Rector لترقية PHP

### نظام البناء

- **معالجة الأصول**: تصغير ملفات CSS/JS
- **الترجمة**: إنشاء ملفات POT
- **التغليف**: إنشاء أرشيف للتوزيع
- **وضع التطوير**: أصول غير مصغرة للخطأ

## التكوين

يدعم الإضافة خيارات تكوين مختلفة يمكن الوصول إليها عبر واجهة مدير الشبكة.

## الخطافات والمرشحات

### الإجراءات

- `ultimate-multisite-addon-template_init` - يُطلق بعد تهيئة الإضافة
- `ultimate-multisite-addon-template_loaded` - يُطلق بعد تحميل جميع ملفات الإضافة

### المرشحات

- `ultimate-multisite-addon-template_settings` - تعديل إعدادات الإضافة
- `ultimate-multisite-addon-template_enabled` - تجاوز حالة تمكين/تعطيل الإضافة
