# العملات المتعددة (Multi-Currency)

تتيح إضافة Ultimate Multisite: Multi-Currency لشبكتك قبول المدفوعات بعملات متعددة. تقوم الإضافة تلقائيًا بتحويل الأسعار باستخدام أسعار الصرف الحية أو اليدوية، وتسمح لك بتحديد أسعار دقيقة لكل منتج بأي عملة، كما تكتشف العملة المفضلة لكل زائر عبر تحديد الموقع الجغرافي — وكل ذلك يتم إدارته من لوحة تحكم إدارية واحدة.

## الميزات الرئيسية

- **التحويل التلقائي لأسعار الصرف** — يتم تحويل الأسعار من عملتك الأساسية باستخدام أسعار يتم جلبها من المزود الذي تختاره.
- **تجاوز الأسعار على مستوى المنتج** — يمكنك تعيين أسعار دقيقة خاصة بعملة معينة على أي منتج لتجاوز التحويل التلقائي.
- **حقل اختيار العملة عند التسجيل** — يتيح للعملاء اختيار عملتهم المفضلة أثناء عملية الدفع/التسجيل.
- **مزودو أسعار صرف متعددون** — ECB، Frankfurter، Stripe، أو أسعار يدوية بالكامل.
- **اكتشاف العملة بناءً على الموقع الجغرافي** — يحدد تلقائيًا العملة التي تتطابق مع بلد الزائر مسبقًا.
- **لوحة تحكم إدارية** — إدارة العملات النشطة وأسعار الصرف وإعدادات المزود من مكان واحد.

## المتطلبات

- Ultimate Multisite 2.0.0 أو أعلى
- تثبيت WordPress Multisite
- PHP 7.4 أو أعلى

## التثبيت

1. قم بتحميل ملفات الإضافة إلى المسار التالي: `/wp-content/plugins/ultimate-multisite-multi-currency`
2. قم بتفعيل الإضافة عبر شاشة **الإضافات (Plugins)** في WordPress (التفعيل على مستوى الشبكة).
3. انتقل إلى **Ultimate Multisite → Multi-Currency** لفتح لوحة التحكم الإدارية.
4. أضف العملات التي تريد قبولها وقم بتهيئة مزود أسعار الصرف الخاص بك.

## الإعدادات (Configuration)

### لوحة التحكم الإدارية (Admin Dashboard)

تُعد لوحة تحكم العملات المتعددة (**Ultimate Multisite → Multi-Currency**) هي المكان المركزي للقيام بما يلي:

- إضافة العملات أو تفعيلها أو تعطيلها.
- اختيار مزود أسعار الصرف وتحديث الأسعار عند الحاجة.
- مراجعة الأسعار الحالية لجميع العملات النشطة.
- تعيين أسعار يدوية عندما لا تكون المزودات التلقائية مناسبة.

### مزودو أسعار الصرف (Exchange Rate Providers)

| المزود | الوصف |
|---|---|
| **ECB** | البنك المركزي الأوروبي — أسعار يومية مجانية للشبكات القائمة على اليورو (EUR). |
| **Frankfurter** | واجهة برمجة تطبيقات (API) مرآة مفتوحة المصدر للبنك المركزي الأوروبي — مجانية، ولا تتطلب مفتاح API. |
| **Stripe** | يستخدم أسعار صرف Stripe الحية — يتطلب تكاملاً نشطاً مع Stripe. |
| **يدوي (Manual)** | تقوم بإدخال الأسعار مباشرة؛ لا يتطلب استدعاء واجهة برمجة تطبيقات خارجية. |

اختر مزودك تحت **Ultimate Multisite → Multi-Currency → Settings → Exchange Rate Provider**. يمكن تحديث الأسعار يدوياً في أي وقت أو جدولة تحديثها تلقائياً.

### تجاوز الأسعار على مستوى المنتج (Per-Product Pricing Overrides)

عندما لا يكون التحويل التلقائي دقيقاً بما فيه الكفاية، يمكنك تعيين سعر دقيق لعملة معينة على أي منتج:

1. افتح محرر المنتج (**Ultimate Multisite → Products → Edit**).
2. مرر إلى قسم **تسعير العملة (Currency Pricing)**.
3. أدخل السعر الدقيق لكل عملة تريد تجاوزها.
4. احفظ المنتج.

تستخدم المنتجات التي لا يوجد بها تجاوز السعر السعر المحول من عملتك الأساسية.

### حقل اختيار العملة عند التسجيل (Currency Selector Signup Field)

يمكن إضافة حقل اختيار العملة إلى مسار التسجيل/الدفع الخاص بك، بحيث يختار العملاء عملتهم المفضلة عند التسجيل:

1. اذهب إلى **Ultimate Multisite → Multi-Currency → Settings**.
2. قم بتفعيل **إظهار مُحدد العملة عند التسجيل (Show currency selector on signup)**.
3. يمكنك اختيار تعيين عملة افتراضية أو ترك تحديد الموقع الجغرافي لتحديد القيمة المحددة مسبقاً.

يتم تخزين العملة المختارة مع الاشتراك وتُستخدم لجميع الفواتير اللاحقة.

### اكتشاف العملة بناءً على الموقع الجغرافي (Geolocation-Based Currency Detection)

عند تفعيل اكتشاف الموقع الجغرافي، تقوم الإضافة بربط بلد الزائر بعملة وتحددها مسبقاً في حقل اختيار العملة. لتفعيل الميزة:

1. اذهب إلى **Ultimate Multisite → Multi-Currency → Settings**.
2. قم بتفعيل **اكتشاف العملة من موقع الزائر (Detect currency from visitor location)**.
3. قم بربط البلدان بالعملات في جدول **تخطيط الموقع الجغرافي (Geolocation Mapping)**.

إذا لم يكن هناك ربط لبلد الزائر، يتم استخدام العملة الأساسية للشبكة كخيار احتياطي.

## الأسئلة الشائعة

### ماذا يحدث إذا كان مزود أسعار الصرف غير متاح؟

تعود الإضافة إلى آخر الأسعار التي تم جلبها بنجاح. إذا لم يتم جلب أي أسعار على الإطلاق، فإنها تعود إلى الأسعار اليدوية التي قمت بتكوينها. يوصى بتكوين أسعار يدوية كشبكة أمان.

### هل يمكنني استخدام مزودين مختلفين لعملات مختلفة؟

لا — يتم تفعيل مزود واحد في كل مرة لجميع العملات. استخدم **يدوي (Manual)** إذا كنت بحاجة إلى تحكم كامل في الأسعار الفردية.

### هل يتطلب تحديد الموقع الجغرافي إضافة منفصلة؟

لا يتطلب أي إضافة خارجية لتحديد الموقع الجغرافي. تستخدم الإضافة خاصية الكشف المدمجة في WordPress من IP إلى بلد. للحصول على دقة أعلى، يمكن دمج خدمة تحديد موقع جغرافي مخصصة عبر خطافات (hooks) التصفية المقدمة.

### هل يتم تطبيق تجاوزات المنتجات على مستوى المنتج قبل أم بعد الضرائب؟

تحدد تجاوزات المنتجات السعر قبل الضرائب بالعملة المستهدفة. يتم حساب الضريبة فوق سعر التجاوز باستخدام إعدادات الضرائب الحالية لديك.
