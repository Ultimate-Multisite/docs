# وثائق Ultimate Multisite

مرحبًا بكم في وثائق Ultimate Multisite. يحول Ultimate Multisite WordPress Multisite إلى منصة كاملة كخدمة موقع (WaaS).

## أقسام الوثائق

### [دليل المستخدم](/user-guide/getting-started/ultimate-multisite-101)

كل ما تحتاجه لإعداد وتكوين وإدارة منصة WaaS الخاصة بك. يغطي التثبيت، بوابات الدفع، ربط النطاقات، إدارة العملاء، والمزيد.

### [وثائق المطور](/developer/hooks)

مرجع تقني للمطورين الذين يبنون على Ultimate Multisite. يتضمن مرجع الخطافات، وثائق REST API، وأدلة تطوير الإضافات.

### [الإضافات](/addons)

وثائق للإضافات المدفوعة التي توسع Ultimate Multisite بميزات إضافية مثل تكامل WooCommerce، التحليلات، بيع النطاقات، والمزيد.
