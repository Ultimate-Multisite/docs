# مرشح: wu_apply_plan_limits

السماح لمطوري الإضافات بقطع القيود.

يمكنك استخدام هذا المرشح لتشغيل كود عشوائي قبل أن يتم تفعيل أي من القيود. إذا أعاد المرشح أي قيمة صحيحة، سيستمر العملية، وإذا أعاد أي قيمة غير صحيحة، سيعود الكود ولن يتم تشغيل أي من الخطافات أدناه.
