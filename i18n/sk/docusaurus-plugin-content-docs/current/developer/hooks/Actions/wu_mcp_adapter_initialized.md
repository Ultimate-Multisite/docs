# Action: wu_mcp_adapter_initialized

Spustí sa po inicializácii MCP adaptera.

Umožňuje iným pluginom a témam zaregistrovať ich vlastné schopnosti.

## Parametre {#parameters}

| Názov | Typ | Popis |
|------|------|-------------|
| $mcp_adapter | `\MCP_Adapter` | Inštancia MCP adaptera. |

### Od verzie {#since}

- 2.5.0
