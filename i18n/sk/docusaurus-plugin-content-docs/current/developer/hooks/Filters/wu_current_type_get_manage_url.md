# Filter: wu_current_\{$type\}_get_manage_url

Umožňuje vývojárom upraviť parametre URL na správu stránky.

## Parametre

| Názov | Typ | Popis |
|------|------|-------------|
| $manage_site_url | `string` | URL na správu stránky. |
| $id | `int` | ID stránky. |
| $site_hash | `string` | Hash stránky. |

### Od verzie

- 2.0.9
### Zdroj

Definované v [`inc/class-current.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-current.php#L214) na riadku 214


## Návratová hodnota
Upravená URL na správu.
