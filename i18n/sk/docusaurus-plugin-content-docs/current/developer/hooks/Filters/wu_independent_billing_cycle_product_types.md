# Filter: wu_independent_billing_cycle_product_types

Filtrujte typy produktov, ktoré majú nezávislé fakturačné cykly.

Produkty s nezávislými fakturačnými cyklami nie sú v košíku nútené zodpovedať fakturačnému obdobiu plánu. Je to užitočné pre produkty, ako sú registrácie domén, ktoré sa vždy fakturujú ročne bez ohľadu na to, či je plán mesačný alebo ročný.

## Parametre

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | Pole typov produktov s nezávislými fakturačnými cyklami. |

### Od verzie

- 2.5.0
