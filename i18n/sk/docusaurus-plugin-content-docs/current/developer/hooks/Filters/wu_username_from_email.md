# Filter: wu_username_from_email {#filter-wuusernamefromemail}

Filtruje používateľské meno nového zákazníka.

## Parametre {#parameters}

| Názov | Typ | Popis |
|------|------|-------------|
| $username | `string` | Používateľské meno zákazníka. |
| $email | `string` | E-mailová adresa nového zákazníka. |
| $new_user_args | `array` | Pole argumentov nového používateľa, prípadne vrátane mena a priezviska. |
| $suffix | `string` | Reťazec pripojený k používateľskému menu, aby bolo jedinečné. |
