# Filter: wu_checkout_validation_rules {#filter-wucheckoutvalidationrules}

Umožňuje vývojárom pluginov filtrovať pravidlá validácie.

## Parametre {#parameters}

| Názov | Typ | Popis |
|------|------|-------------|
| $validation_rules | `array` | Pravidlá validácie, ktoré sa majú použiť. |
| $checkout | `\Checkout` | Trieda checkout. |
