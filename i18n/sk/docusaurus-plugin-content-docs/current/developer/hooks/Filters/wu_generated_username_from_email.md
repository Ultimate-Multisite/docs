# Filter: wu_generated_username_from_email {#filter-wugeneratedusernamefromemail}

Filter vygenerovaného používateľského mena zákazníka.

## Parametre {#parameters}

| Názov | Typ | Popis |
|------|------|-------------|
| $username | `string` | Vygenerované používateľské meno. |
| $email | `string` | E-mailová adresa nového zákazníka. |
| $new_user_args | `array` | Pole argumentov nového používateľa, prípadne vrátane mena a priezviska. |
| $suffix | `string` | Reťazec, ktorý sa pridá k používateľskému menu, aby bolo jedinečné. |
