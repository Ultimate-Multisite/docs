# Agenti podpory

Správa celej siete svojpomocne nie je jednoduchá úloha. Ako vaša sieť začne rásť, začnete vidieť rastúci dopyt po podpore. Bude dobrý nápad prizvať ľudí, ktorí vám pomôžu znížiť pracovné zaťaženie. Udelenie úplného prístupu/oprávnenia je však riskantné a môže ohroziť vašu sieť. Práve na to slúži tento doplnok Agenti podpory.

Tento doplnok vám umožňuje vytvoriť nový typ používateľa WordPress s názvom Agent podpory a dáva vám úplnú kontrolu nad tým, aké oprávnenia tento používateľ má. Udelenie oprávnenia je také jednoduché ako zapnutie prepínača a jeho vypnutie, ak sa rozhodnete oprávnenie odstrániť.

## Funkcie

- Podrobný zoznam oprávnení WordPress a Ultimate Multisite
- Jednoducho povoľte a zakážte prístup
- Vytvorte neobmedzený počet účtov agentov

## Inštalácia

1. Nahrajte 'ultimate-multisite-support-agents' do adresára '/wp-content/plugins/'
2. Aktivujte plugin cez menu 'Pluginy' vo WordPress
3. Postupujte podľa sprievodcu krok za krokom a nastavte plugin
