# Filter: networks_user_is_network_admin

Filtruje siete, ktorých je používateľ administrátorom.

## Parametre

| Názov | Typ | Opis |
|------|------|-------------|
| $ | `array\|bool` | Zoznam ID sietí alebo false, ak používateľ nemá žiadne siete. |
| $ | `int` | ID používateľa, pre ktorého sa majú vrátiť siete. |
