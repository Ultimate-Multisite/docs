# Superdav AI Agent

Superdav AI Agent je pokročilý WordPress AI asistent, ktorý rozširuje Gratis AI Agent o zručnosti prispôsobené téme, rozšírené možnosti internetového vyhľadávania, funkcie tvorby webu a voliteľné automatizačné integrácie. Verzia 1.18.0 pridáva nástroje na čítanie Google Calendar, mapovanie kontaktov a účastníkov, schvaľovacie brány človekom, deduplikáciu pripomienok, SMS upozornenia TextBee, poskytovanie spravovanej služby Superdav a samostatne distribuovaný sprievodný balík Advanced pre dôveryhodné vývojárske pracovné postupy.

## Kľúčové funkcie {#key-features}

- **Zručnosti prispôsobené téme** — automaticky prispôsobuje odporúčania a možnosti na základe vašej aktívnej WordPress témy a nainštalovaných pluginov
- **Viacerí poskytovatelia internetového vyhľadávania** — vyberte si medzi Brave Search a Tavily pre bohatšie výsledky vyhľadávania a pokročilé výskumné možnosti
- **Kontaktné formuláre nástroja na tvorbu webu** — vytvárajte plne funkčné kontaktné formuláre priamo z rozhrania chatu pomocou vášho nainštalovaného pluginu formulárov
- **Vylepšený nástroj na tvorbu webu** — vylepšené viacfázové plány tvorby webu s lepším zisťovaním pluginov a obnovou po chybách
- **Vstavané zručnosti pre témy** — špecializované odporúčania pre blokové témy, klasické témy, Kadence Blocks a Kadence Theme
- **Vylepšená integrácia WooCommerce** — prepracovaná tak, aby používala natívne WooCommerce API pre lepšiu spoľahlivosť a kompatibilitu
- **Automatická detekcia poskytovateľov** — automaticky zisťuje a obnovuje dostupné pluginy a témy pri ich aktivácii alebo deaktivácii
- **Generovanie vlastného loga** — generujte a vkladajte vlastné SVG logá s automatickou sanitizáciou bezpečnou pre menné priestory
- **Validácia kontrastu palety** — skontrolujte farebné páry z hľadiska súladu s prístupnosťou WCAG pred ich použitím vo vašej téme
- **Objavovanie vylepšené fotografiami** — nahrajte fotografie počas objavovacieho rozhovoru Theme Builder, aby ovplyvnili smer dizajnu
- **Menu pre pohostinstvo** — generujte štruktúrované stránky menu jedál a nápojov pre reštaurácie, kaviarne a podniky v pohostinstve
- **Náhľady pre počítač a mobil** — zobrazte náhľad svojho dizajnu na počítačoch a mobilných zariadeniach počas výberu dizajnového smeru
- **Scraping webu úrovne 1** — sd-ai-agent/site-scrape je teraz predvolene dostupný na analýzu konkurenčných a inšpiračných webov
- **Nástroje na čítanie Google Calendar** — čítajú nakonfigurované kalendáre a udalosti, aby automatizácie mohli uvažovať o kontexte nadchádzajúceho harmonogramu
- **Mapovanie kontaktov a účastníkov** — priraďte účastníkov kalendára k známym používateľom webu a kontaktom pred generovaním následnej práce
- **Schvaľovacie brány človekom** — pozastavte citlivé automatizácie na kontrolu pred odoslaním správ, obnovením práce alebo vykonaním zmien
- **Záznamy pripomienok a deduplikácia** — sledujte pripomienky, ktoré už boli odoslané, aby opakované spustenia automatizácie neupozornili toho istého príjemcu dvakrát
- **SMS upozornenia TextBee** — odosielajte nakonfigurované textové upozornenia cez TextBee, keď je povolený poskytovateľ SMS
- **Poskytovanie spravovanej služby Superdav** — pripojte podporované weby k hosťovaným Superdav endpointom s automatickým nastavením namiesto manuálnej konfigurácie endpointu
- **Sprievodný balík Advanced** — samostatne distribuované nástroje pre dôveryhodných vývojárov na pracovné postupy so súborovým systémom, databázou, WP-CLI, REST dispečerom, tvorbou pluginov, git snapshotom, správou používateľov a benchmarkmi

## Požiadavky {#requirements}

- WordPress 6.0 alebo vyšší
- PHP 8.1 alebo vyšší
- WordPress Multisite (aktivovaný v sieti) alebo inštalácia jedného webu
- Aktívny plugin konektora poskytovateľa AI (Anthropic, OpenAI alebo kompatibilný)
- Gratis AI Agent v1.9.0 alebo vyšší (Superdav AI Agent rozširuje Gratis AI Agent)

## Inštalácia {#installation}

1. Nahrajte súbory pluginu do `/wp-content/plugins/superdav-ai-agent`
2. Aktivujte plugin cez obrazovku **Pluginy** vo WordPress
3. Uistite sa, že je aktivovaný aj Gratis AI Agent
4. Prejdite na **Gratis AI Agent → Nastavenia** a podľa potreby nakonfigurujte svojho poskytovateľa internetového vyhľadávania, pripojenie spravovanej služby Superdav, Google Calendar alebo prihlasovacie údaje TextBee

Balík WordPress.org obsahuje základné funkcie Superdav AI Agent. Sprievodný balík Advanced sa zostavuje a distribuuje samostatne pre prostredia, kde sú nástroje dôveryhodných vývojárov výslovne nainštalované a autorizované.

## Rýchly štart {#quick-start}

Po aktivácii Superdav AI Agent automaticky vylepší vašu inštaláciu Gratis AI Agent o zručnosti prispôsobené téme a ďalšie možnosti. Otvorte panel **Gratis AI Agent** v bočnom paneli administrácie WordPress a začnite používať nové funkcie.

**Príklady pokynov:**

```
Pridaj kontaktný formulár na stránku Kontakt.
```

```
Vyhľadaj na webe najnovšie osvedčené postupy zabezpečenia WordPress.
```

```
Vytvor hero sekciu pomocou blokových vzorov aktívnej témy.
```

```
Skontroluj zajtrajšie nakonfigurované udalosti Google Calendar a priprav návrhy pripomienok pre účastníkov.
```

```
Odošli SMS pripomienku po tom, ako správu schválim.
```

## Zručnosti prispôsobené téme {#theme-aware-skills}

Superdav AI Agent automaticky zisťuje vašu aktívnu tému a nainštalované pluginy a načítava do kontextu asistenta príslušnú zručnosť prispôsobenú téme. Dostupné zručnosti zahŕňajú:

| Zručnosť | Platí pre |
|---|---|
| **Blokové témy** | Témy používajúce WordPress architektúru založenú na blokoch (Full Site Editing) |
| **Klasické témy** | Tradičné WordPress témy používajúce PHP šablóny a klasický editor |
| **Kadence Blocks** | Weby s aktívnym pluginom Kadence Blocks |
| **Kadence Theme** | Weby s témou Kadence ako aktívnou témou |

Pozrite si [Zručnosti prispôsobené téme](../../user-guide/configuration/theme-aware-skills) pre podrobné pokyny ku každej zručnosti.

## Poskytovatelia internetového vyhľadávania {#internet-search-providers}

Superdav AI Agent v1.10.0 pridáva **Tavily** ako alternatívu k Brave Search pre dopyty internetového vyhľadávania. Obaja poskytovatelia ponúkajú výsledky vyhľadávania v reálnom čase a možno ich nakonfigurovať v **Gratis AI Agent → Nastavenia → Pokročilé**.

| Poskytovateľ | Funkcie | Nastavenie |
|---|---|---|
| **Brave Search** | Rýchle výsledky vyhľadávania zamerané na súkromie | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Bohatšie výsledky vyhľadávania, pokročilé možnosti výskumu | [Tavily API](https://tavily.com/) |

Pokyny na konfiguráciu nájdete v časti [Internetové vyhľadávanie](../../user-guide/configuration/internet-search).

## Pracovné postupy kalendára, schvaľovania a oznámení {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 rozširuje automatizačné pracovné postupy nad rámec úloh súvisiacich s obsahom a tvorbou webov:

- **Nástroje Google Calendar** umožňujú agentovi čítať nakonfigurované kalendáre a podrobnosti udalostí, keď sú dostupné prihlasovacie údaje ku kalendáru.
- **Pomocníci pre kontakty a účastníkov** mapujú účastníkov udalostí na používateľov WordPress alebo známe kontakty pred prípravou pripomienok alebo nadväzných úloh.
- **Brány ľudského schvaľovania** umožňujú automatizácii pozastaviť sa, kým oprávnený používateľ neskontroluje navrhovanú akciu.
- **Záznamy pripomienok** zabraňujú duplicitným kalendárovým pripomienkam alebo pripomienkam oznámení, keď sa automatizácia opakuje alebo sa opakovane plánuje.
- **Podpora TextBee SMS** umožňuje oznámenia textovými správami, keď je nakonfigurovaný poskytovateľ TextBee.

Použite tieto nástroje na pomoc zohľadňujúcu rozvrh, následnú komunikáciu s účastníkmi a pracovné postupy pripomienok, ktoré si pred odoslaním citlivých správ stále vyžadujú ľudskú kontrolu.

## Kontaktné formuláre v nástroji na tvorbu webov {#site-builder-contact-forms}

Podrobné pokyny nájdete v časti [Kontaktné formuláre Site Builder](../../user-guide/administration/site-builder-contact-forms).

## Konfigurácia {#configuration}

Všetky nastavenia sa nachádzajú v časti **Gratis AI Agent → Settings** v administrácii WordPress. Superdav AI Agent pridáva nasledujúce možnosti konfigurácie:

| Nastavenie | Opis |
|---|---|
| **Poskytovateľ internetového vyhľadávania** | Vyberte si medzi Brave Search a Tavily (voliteľné) |
| **Tavily API Key** | API kľúč pre internetové vyhľadávanie Tavily (ak je vybraté Tavily) |
| **Spravovaná služba Superdav** | Pripojte podporované weby k hostovanej službe Superdav s automatickým poskytovaním endpointov |
| **Prihlasovacie údaje Google Calendar** | Nakonfigurujte prístup ku kalendáru, aby agent mohol čítať kalendáre a udalosti |
| **Poskytovateľ TextBee SMS** | Nakonfigurujte prihlasovacie údaje TextBee pre schválené pracovné postupy oznámení SMS |

## Pokročilý sprievodný balík {#advanced-companion-package}

Superdav AI Agent v1.18.0 pridáva samostatne distribuovaný pokročilý sprievodný balík pre dôveryhodné pracovné postupy vývojárov. Tieto možnosti nie sú zahrnuté v základnom balíku WordPress.org. Sprievodný balík nainštalujte iba na weboch, kde administrátori výslovne dôverujú agentovi, že môže pristupovať k vývojárskym nástrojom.

Pokročilé nástroje zahŕňajú operácie so súborovým systémom, prístup k databáze, WP-CLI, pracovné postupy REST dispečera, utility na tvorbu pluginov, git snímky, pomocníkov pre správu používateľov a nástroje benchmarkovania. Obmedzte tieto nástroje na dôveryhodných administrátorov a pred ich povolením na produkčných weboch skontrolujte konfiguráciu schvaľovacích brán.

## Vylepšenia vo v1.16.0 {#improvements-in-v1160}

- **Cache AI klienta** — teraz je podporovaná transientmi pre pretrvávanie medzi požiadavkami, čím sa predchádza strate dát pri dlhotrvajúcich úlohách agenta
- **Nástroj na získavanie dát zo stránky úrovne 1** — sd-ai-agent/site-scrape je teraz predvolene dostupný v Theme Builder na analýzu konkurenčných a inšpiračných webových stránok
- **Odkazy na akcie v riadku pluginu** — opravené a premenované pre väčšiu zrozumiteľnosť

## Vylepšenia vo v1.10.0 {#improvements-in-v1100}

- **Integrácia WooCommerce refaktorovaná** — teraz používa natívne WooCommerce API pre lepšiu spoľahlivosť a kompatibilitu
- **Automatické obnovenie zoznamu poskytovateľov** — automaticky zisťuje, keď sú pluginy aktivované alebo deaktivované, a aktualizuje zoznam dostupných poskytovateľov
- **Vylepšený tvorca stránok** — vylepšené viacfázové plány tvorby stránok s lepšou obnovou po chybách
- **Zručnosti zohľadňujúce tému** — štyri nové vstavané zručnosti, ktoré sa automaticky prispôsobia vašej aktívnej téme

## Opravy chýb vo v1.10.0 {#bug-fixes-in-v1100}

- Opravená schopnosť `navigate-to`, ktorá spôsobovala nekonečné slučky opätovného načítania na niektorých administračných stránkach
- Opravená schopnosť `list-posts`, ktorá správne neprevádzala názvy kategórií a značiek na slugy
- Opravené WP-CLI príkazy, ktorým po predchádzajúcom refaktorovaní chýbali aliasy menných priestorov
- Opravené elegantné spracovanie automatizácie udalostí na stránkach, kde ešte nie sú vytvorené tabuľky automatizácie
- Opravená schopnosť `memory-save`, ktorá používa správnu predponu menného priestoru v tvorcovi systémových inštrukcií
- Opravené správne zabalenie skalárnych výsledkov nástrojov pred ich vrátením AI
- Opravené štatistiky používania, aby správne spracúvali starší formát kľúča schopnosti pri aktualizácii zo starších verzií
