# Fluent Forms

Tento add-on umožňuje pridávať limity pre Fluent Forms na Multisite Ultimate v produktoch, členstvách alebo stránkach.
Fluent Forms používa vlastné tabuľky namiesto vlastných typov príspevkov, takže bežné možnosti obmedzovania s Fluent Forms nefungujú.

## Funkcie

- Pridajte limity pre Fluent Forms alebo Conversational Fluent Forms
- Zákazníci dostanú priateľskú správu, ktorá ich vyzve na upgrade ich plánu, keď dosiahnu limit
