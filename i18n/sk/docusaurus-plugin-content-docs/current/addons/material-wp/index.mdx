# Material WP

Administrátorské rozhranie WordPress je už celé roky rovnaké. Premýšľali ste však niekedy, ako by vyzerala verzia administrátorského rozhrania v štýle Material Design? 

Material WP je administrátorská téma vyvinutá tak, aby vám poskytla tento vzhľad a dojem a pomohla vám ponúknuť vašim klientom viac prispôsobený Dashboard.

## Funkcie

- Podpora vlastného CSS
- Jednoduché nastavenia štýlovania
- Zmena poradia menu
- Jednoduchý import/export

## Dokumentácia

**Prispôsobenie vášho administrátorského panela pomocou Material WP**

## Inštalácia

1. Nahrajte 'material-wp' do adresára '/wp-content/plugins/'
2. Aktivujte plugin cez menu 'Pluginy' vo WordPress

## Sprievodca používaním

Administrátorské rozhranie WordPress je už celé roky rovnaké. Premýšľali ste však niekedy, ako by vyzerala verzia administrátorského rozhrania v štýle Material Design?

Po **nainštalovaní** prejdite na svoju stránku _Pluginy_ a kliknutím ho aktivujte vo svojej sieti.
