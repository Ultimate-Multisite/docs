# Ako si môcť získať prispôsobenú pomoc od Codeable?

Inékrát potrebujeme skutočne super prispôsobenia, aby sme našich pracovných tokov dosiahli presne tak, ako chceme. Ak hľadáte akékoľvek riešenie, odporúčame [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), miesto, kde nájdete skúsených vývojárov s expertou v Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) je platforma, ktorá spája zákazníkov s kvalifikovanými freelancovými vývojárskymi WordPressom, ktorí sú schopní pracovať na projekto akékoľvek veľkosti. Majú veľmi podrobné aplikácie, aby ste boli eşotovaní s odborníkmi špecializovanými na úlohu, ktorú máte v mysli.

### Ako vytvoriť projekt v Codeable?

Prv prvej chvíli prejdite na [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) a kliknite na tlačidlo **Start a Project** (Začať projekt).

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Teraz, aby ste zaslali projekt, kliknite vľavo na **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

V sekcii "I need help with" (Potrebujem pomoc s) si vyberte **Customizing** (Prispôsobenie):

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

V sekcii "My" (Moje) si vyberte **Plugin** (Prípona).

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Potom kliknite na tlačidlo **Continue** (Pokračovať).

<!-- Screenshot unavailable: Codeable project form Continue button -->

Teraz je čas opísať váš projekt a potreby najlepším spôsobom, aký si môžete. Pokúšajte sa poskytnúť čo najväčšiu informáciu!

<!-- Screenshot unavailable: Codeable project description form -->

Odpovedzte na nasledujúce otázky, aby sme vám mohli pomôcť s cenou.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Po tom, ak ešte nem máte účet [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), vytvorte si jeden a kliknite na tlačidlo **Publish My Project** (Publikovať môj projekt).

<!-- Zobrazenie snímky obrazovky: vytvorenie účtu Codeable a tlačidlo Publish My Project -->

Keď budete mať svoj [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) projekt published, platforma vás spojí s vývojármi, ktorí odpovedajú na potreby vášho projektu.

Tento proces vám pomôže nájsť správnu osobu pre vašu prácu, ktorá má skúsenosti s WordPressom a Ultimate Multisite!

Minimálna mzdová cena Codeable je 70–120 USD za hodinu. Odhadky sa zmenia v závislosti od rozsahu práce, zložitosti a dopytu projektu.

Keď dostanete svoj odhad, budete môcť nájsť vývojára, ktorý najlepšie vyhovuje vašim potrebám a spúšťa si s ním váš projekt!
