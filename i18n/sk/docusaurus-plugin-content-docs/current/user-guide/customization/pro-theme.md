# Prispôsobenie vášho Admin Panelu s PRO témou

Často pri práci pre klientov potrebujeme poskytnúť našim klientom prispôsobený zážitok v admin panelu WordPress. Ak hľadáte profesionšnejší vzhľad a pocit, môže vám pomôc PRO Theme.

PRO Theme je admin téma vytvorená na pomoc pri prispôsobení dashboardov vašich klientov. Je tiež súčasťou základných pluginov premium Ultimate Multisite.

Po stiahnutí sa prejdite na stránku _Plugins_ a kliknite na aktiváciu v sieti.

<!-- Zobrazenie obrázka nedostupné: Stránka WP Plugins zobrazujúca tlačidlo Aktivácia Sieť pre PRO Theme -->

_Aktivujte ho v celej sieti na stránke Plugins_

_**POZNÁMKA:**_ _Ak ho len stiahnete, zobrazi sa klientovi ako možnosť, ktorú môžete označiť, aby ste tému pridali len pre daného používateľa._

Teraz máte úplne prenovený admin rozhranie! Na ľavej kolone máte aj nové menu s názvom PRO Theme.

<!-- Zobrazenie obrázka nedostupné: Prenovené rozhranie admin panelu PRO Theme -->

_Brand nový vzhľad vášho Admin Panelu_

Na tomto menu budete mať prístup k piesť tabám:

<!-- Zobrazenie obrázka nedostupné: Piesť tab Styling Settings pre PRO Theme -->

_Styling Settings_ (Nastavenia štýlu)

<!-- Zobrazenie obrázka nedostupné: Piesť tab Functionality pre PRO Theme -->

_Functionality_ (Funkcionalita)

<!-- Zobrazenie obrázka nedostupné: Piesť tab Custom CSS pre PRO Theme -->

_Custom CSS_ (Pranepísaný CSS)

<!-- Zobrazenie obrázka nedostupné: Piesť tab Import/Export pre PRO Theme -->

_Import/Export_ (Import/Export)

<!-- Zobrazenie obrázka nedostupné: Piesť tab Activate pre PRO Theme -->

_Aktivujte PRO Theme_

V každej z nich môžete zapnúť a vypnúť prvky štýlu, povoliť alebo blokovať funkcie, a poskytnúť svojim klientom prispôsobenie, ktoré hľadajú v sieti vašich stránok.
