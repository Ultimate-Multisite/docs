# Dokumentácia Ultimate Multisite {#ultimate-multisite-documentation}

Vitajte v dokumentácii Ultimate Multisite. Ultimate Multisite premieňa WordPress Multisite na plnohodnotnú platformu Webová stránka ako služba (WaaS).

## Sekcie dokumentácie {#documentation-sections}

### [Používateľská príručka](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

Všetko, čo potrebujete na nastavenie, konfiguráciu a správu svojej platformy WaaS. Zahŕňa inštaláciu, platobné brány, mapovanie domén, správu klientov a ďalšie.

### [Dokumentácia pre vývojárov](/developer/hooks) {#developer-documentation}

Technická referencia pre vývojárov stavajúcich na Ultimate Multisite. Obsahuje referenciu hooks, dokumentáciu REST API a príručky k vývoju doplnkov.

### [Doplnky](/addons) {#addons}

Dokumentácia k plateným doplnkovým zásuvným modulom, ktoré rozširujú Ultimate Multisite o ďalšie funkcie, ako je integrácia WooCommerce, analytika, predaj domén a ďalšie.
