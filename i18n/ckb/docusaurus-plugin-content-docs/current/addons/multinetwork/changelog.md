# گۆڕانکارییەکانی Multi-Network

### 1.3.0
* نوێ: پیشاندەری پێشوەختی داڕێژەی نێتۆرک — پێداچوونەوە و پێشبینینی داڕێژەکانی نێتۆرک بکە بە پانێڵێکی زیندوو پێش کڕین.
* نوێ: هەڵبژێری داڕێژەی نێتۆرک لە checkout — کڕیاران دەتوانن لە کاتی ڕەوتی signup checkout دا داڕێژەیەکی نێتۆرک هەڵبژێرن.
* نوێ: Cross-network magic link SSO — کڕیاران دەتوانن بەبێ کێشە لە ڕێگەی magic-link authentication ـەوە دەستیان بگات بە sub-networks.
* نوێ: لەبەرگرتنەوەی میدیای نێتۆرک — میدیای سایت بە دروستی دووبارە دەکرێتەوە کاتێک نێتۆرکەکان لە داڕێژەیەکەوە clone دەکرێن.
* چاککردن: ئێستا clone کردنی نێتۆرک بەبێ multi-tenancy addon کار دەکات؛ دەسەڵاتەکانی super admin دوای clone کردن دووبارە پێدەدرێنەوە.
* چاککردن: ئێستا دروستکردنی نێتۆرک لەگەڵ WooCommerce و هەموو دەروازەکانی پارەدان گونجاوە.
* چاککردن: ئێستا ژمارەی پۆرت لە دروستکردنی دۆمەینی fallback دا دەخرێتە ناوەوە.
* چاککردن: CSS ـی هەڵبژێری داڕێژە گوازرایەوە بۆ stylesheet ـێکی enqueued بۆ پیشاندانی دروست لەسەر هەموو theme ـەکان.
* چاککردن: پاراستن لە کلیلێکی پێناسەنەکراوی `network_id` لە scope ـی پرسیاری سایت.
* چاککردن: پشکنینی مۆڵەتی نێتۆرک پێش گۆڕینی context ڕێگری دەکات لە هەڵەکانی تایبەتمەندی خشتەی `wpdb`.
