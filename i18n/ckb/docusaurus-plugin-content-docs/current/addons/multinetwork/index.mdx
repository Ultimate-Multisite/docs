# فرە-تۆڕ {#multi-network}

پێوەکراوی Ultimate Multisite: فرەتۆڕ پلاتفۆرمی Ultimate Multisite ـەکەت بە کارایی بەهێزی فرەتۆڕ فراوان دەکات. ئەم پێوەکراوە ڕێگە بە کڕیارەکانت دەدات تەواوی تۆڕە فرەماڵپەڕەکانی WordPress بکڕن و بەڕێوەیان ببەن، نەک تەنها ماڵپەڕی تاکەکەسی.

## تایبەتمەندییە سەرەکییەکان {#key-features}

* **دابینکردنی تۆڕ**: بە شێوەی خۆکارانە تۆڕە فرەماڵپەڕە تەواوەکانی WordPress بۆ کڕیارەکانت دروست و ڕێکبخە
* **قاڵبەکانی تۆڕ**: با کڕیاران پێش تۆمارکردن قاڵبی تۆڕی پێشوەختە دروستکراو بگەڕێن، پێشبینی بکەن، و هەڵیبژێرن
* **بەڕێوەبردنی تۆڕ**: تواناکانی بەڕێوەبردنی تەواو بۆ ژینگەکانی فرەتۆڕ
* **تۆڕە دابڕاوەکان**: هەر تۆڕێکی کڕیار بە سەربەخۆیی بە خشتەکانی بنکەدراوە و ڕێکخستنی خۆی کار دەکات
* **SSO ـی نێوان تۆڕەکان**: بە پشتڕاستکردنەوەی magic-link دەستگەیشتنی بێکێشە بە ژێر-تۆڕەکانیان بدە بە کڕیاران
* **بیناسازییەکی فراوانبوونپەذیر**: چەندین تۆڕ بە کارایی و بە دابەشکردنی سەرچاوەی باشکراو بەڕێوە ببە
* **یەکخستن**: بە بێکێشە لەگەڵ سیستەمەکانی بەشداریکردن و پسوڵەکردنی Ultimate Multisite یەکدەگیرێتەوە

## حاڵەتەکانی بەکارهێنان {#use-cases}

* **خانەخوێتی White Label**: تۆڕە فرەماڵپەڕە تایبەت بە براندی خۆیان بۆ کڕیاران دابین بکە
* **چارەسەرەکانی ئاژانس**: ڕێگە بە ئاژانسەکان بدە چەندین تۆڕی کڕیار لە یەک پلاتفۆرمەوە بەڕێوە ببەن
* **دامەزراندنەکانی کۆمپانیای گەورە**: پشتگیری ڕێکخراوە گەورەکان بکە کە پێداویستیی فرە-تۆڕی ئاڵۆزیان هەیە
* **پلاتفۆرمەکانی پەروەردە**: تۆڕی جیاواز بۆ قوتابخانە یان بەشە جیاوازەکان دروست بکە

## پێداویستییەکان {#requirements}

* Ultimate Multisite 2.0.0 یان بەرزتر
* دامەزراندنی فرەماڵپەڕی WordPress
* PHP 7.4 یان بەرزتر
* MySQL 5.6 یان بەرزتر

## دامەزراندن {#installation}

1. فایلەکانی plugin باربکە بۆ بوخچەی `/wp-content/plugins/ultimate-multisite-multinetwork`
2. plugin لە ڕێگەی پەردەی 'Plugins' لە WordPress چالاک بکە
3. ڕێکخستنەکانی فرەتۆڕ لە ڕێگەی Ultimate Multisite → Settings ڕێکبخە

## قاڵبەکانی تۆڕ {#network-templates}

وەشانی 1.3.0 ڕەوتی دۆزینەوە و هەڵبژاردنی قاڵب زیاد دەکات بۆ کڕیاران کە تۆڕی نوێ دەکڕن. قاڵبەکانی تۆڕ بەکاربهێنە کاتێک دەتەوێت هەر تۆڕێکی نوێی کڕیار لە پێکهاتەیەکی ئامادەکراو، دیزاین، کتێبخانەی ناوەڕۆک، یان ڕێکخستنێکی تایبەت بە بوارێکەوە دەست پێ بکات.

### پێشبینیکەری قاڵب {#template-previewer}

پێشبینیکەری قاڵبی تۆڕ ڕێگە بە کڕیاران دەدات پێش checkout قاڵبە بەردەستەکان بگەڕێن. هەر قاڵبێک دەتوانرێت لە پانێڵی پێشبینیی زیندوودا بکرێتەوە، بۆ ئەوەی کڕیاران بتوانن پێش کڕینی تۆڕێک نەخشە و ناوەڕۆک بەراورد بکەن.

پێشبینیکەر بەکاربهێنە بۆ یارمەتیدانی کڕیاران لە:

* پێداچوونەوە بە دەرکەوتن و پێکهاتەی هەر قاڵبێکی تۆڕی بەردەست
* بەراوردکردنی چەندین قاڵب بەبێ جێهێشتنی ڕەوتی کڕین
* هەڵبژاردنی ئەو قاڵبەی پێش signup باشترین گونجانی لەگەڵ حاڵەتی بەکارهێنانیان هەیە

ناو و وەسفەکانی قاڵب بە شێوەیەک بهێڵەوە کە بۆ کڕیاران ئاسان و دۆستانە بێت. پێشبینیکەر زۆرترین سوودی هەیە کاتێک کڕیاران دەتوانن حاڵەتی بەکارهێنانی مەبەستدار تێبگەن، وەک تۆڕەکانی ئاژانس، تۆڕەکانی قوتابخانە، تۆڕەکانی فرانچایز، یان کۆمەڵە خانەخوێتییەکانی white-label.

### هەڵبژێری قاڵب لە checkout {#template-picker-at-checkout}

هەڵبژێری قاڵب دەتوانرێت وەک خانەیەک زیاد بکرێت بۆ ڕەوتی signup checkout. کاتێک ڕێکخرابێت، کڕیاران لە کاتی checkout ئەو قاڵبی تۆڕە هەڵدەبژێرن کە دەیانەوێت، و قاڵبە هەڵبژێردراوەکە کاتێک تۆڕە نوێیەکەیان دابین دەکرێت بەکاردهێنرێت.

بۆ بەکارهێنانی هەڵبژێر لە ڕەوتی checkout:

1. ئەو تۆڕانە ئامادە بکە کە دەتەوێت وەک قاڵب پێشکەشیان بکەیت.
2. خانەی هەڵبژاردنی قاڵبی Multinetwork لە فۆڕمی signup یان checkout ـی پەیوەندیدار چالاک بکە.
3. هەڵبژێرە کام قاڵبەکان دەبێت بۆ ئەو ڕەوتی checkout بەردەست بن.
4. ڕەوتی checkout بڵاو بکەوە و signup ـێک تاقی بکەوە بۆ دڵنیابوون لەوەی قاڵبە هەڵبژێردراوەکە بۆ تۆڕە نوێیەکە جێبەجێ دەکرێت.

ئەگەر ڕەوتێکی checkout تەنها یەک بەرهەمی تۆڕ پێشکەش بکات، هەڵبژێرەکە هێشتا دەتوانێت وەک هەنگاوێکی ڕێنوێنی onboarding سوودمەند بێت. بۆ ڕەوتەکانی فرە-پلان، تەنها ئەو قاڵبانە نیشان بدە کە لەگەڵ بەرهەم یان پلانی هەڵبژێردراو گونجاون.

## SSO ـی Magic Link ـی نێوان تۆڕەکان {#cross-network-magic-link-sso}

SSO ـی magic link ـی نێوان تۆڕەکان ڕێگە بە کڕیاران دەدات بەبێ ئەوەی دووبارە بە دەستی بچنەژوورەوە، لە نێوان ناوچەی Account ـی سەرەکی خۆیان و ژێر-تۆڕەکاندا بجوڵێن. کاتێک کڕیارێک دوای بەستەری دەستگەیشتنی نێوان تۆڕەکان دەکەوێت، Ultimate Multisite بەستەری magic link ـێکی کاتی سنووردار دروست دەکات و کڕیارەکە دەخاتە ژوورەوە بۆ تۆڕی مەبەست.

ئەمە ئەزموونەکە باشتر دەکات بۆ ئەو کڕیارانەی زیاتر لە یەک تۆڕ بەڕێوە دەبەن یان پێویستیانە دەستبەجێ دوای کڕین دەستگەیشتن بە تۆڕە قاڵبەکانی کۆپی کراو هەبێت.

لیستی پشکنینی بەڕێوەبەر:

* دڵنیا ببەوە پێوەکراوی Multinetwork لە سەرانسەری دامەزراندنی فرەماڵپەڕی WordPress چالاکە.
* دڵنیا ببەوە کڕیاران پێش ئەوەی بەستەرەکانی نێوان تۆڕەکان پێشکەش بکرێن دەستگەیشتنی دروستیان بە تۆڕ هەیە.
* دۆمەینەکانی تۆڕ و دۆمەینە نەخشەکراوەکان بە دروستی ڕێکخراو بهێڵەوە بۆ ئەوەی magic links بگەن بە تۆڕی مەبەستی خوازراو.
* Dashboard ـی کڕیار و هەر دوگمەیەکی دەستگەیشتنی دوای checkout پاش گۆڕینی ڕێکخستنەکانی تۆڕ، دۆمەین، یان checkout تاقی بکەوە.

magic links دەبێت وەک بەستەرەکانی چوونەژوورەوە مامەڵەیان لەگەڵ بکرێت. ئەوان بۆ دەستگەیشتنی بێکێشەی کڕیاران دیزاین کراون و نابێت بۆ دەرەوەی دانیشتنی مەبەستدار بنێردرێن یان دووبارە بەکاربهێنرێن.

## پرسیارە باوەکان {#frequently-asked-questions}

### فرەتۆڕ چییە؟ {#what-is-multinetwork}

فرەتۆڕ تایبەتمەندییەکی WordPress ـە کە ڕێگەت پێدەدات چەندین تۆڕی فرەماڵپەڕی سەربەخۆ لەسەر یەک دامەزراندنی WordPress بەکاربخەیت.

### ئەمە چۆن لە فرەماڵپەڕی ئاسایی جیاوازە؟ {#how-is-this-different-from-regular-multisite}

فرەماڵپەڕی ئاسایی ڕێگە بە چەندین ماڵپەڕ دەدات لەناو یەک تۆڕدا. فرەتۆڕ ڕێگە بە چەندین تۆڕ دەدات، هەر یەکەیان بە ماڵپەڕەکانی خۆیانەوە، هەموویان لەناو یەک دامەزراندنی WordPress دا.

### ئایا ئەمە پێویستی بە ڕێکخستنی تایبەتی ڕاژەکار هەیە؟ {#does-this-require-special-server-configuration}

بەڵێ، فرەتۆڕ پێویستی بە دامەزراندن و ڕێکخستنی دیاریکراوی ڕاژەکار هەیە. تکایە بۆ ڕێنماییە وردەکانی دامەزراندن سەیری بەڵگەنامەکان بکە.
