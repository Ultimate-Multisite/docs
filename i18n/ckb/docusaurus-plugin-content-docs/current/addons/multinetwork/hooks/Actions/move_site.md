# کردار: move_site

دوای ئەوە کاردەکات کە ماڵپەڕێک گوازرابێتەوە بۆ تۆڕێکی نوێ.

## پارامەتەرەکان

| ناو | جۆر | وەسف |
|------|------|-------------|
| $site_id | `int` | IDی ئەو ماڵپەڕەی گوازراوەتەوە. |
| $old_network_id | `int` | IDی تۆڕی بنەڕەتی بۆ ماڵپەڕەکە. |
| $new_network_id | `int` | IDی ئەو تۆڕەی ماڵپەڕەکە بۆی گوازراوەتەوە. |
