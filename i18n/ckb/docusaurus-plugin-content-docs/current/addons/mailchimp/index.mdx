# Mailchimp

Multisite Ultimate: یەکخستنی Mailchimp

تۆڕی Multisite Ultimate ـەکەت بە Mailchimp ـەکەتەوە یەکبخە. دەتوانیت هەندێک پلانی دیاریکراو بە لیستەکانی Mailchimp ـەوە ببەستیتەوە و لە کاتی تۆماربووندا بەشداربووان بە شێوەیەکی ئۆتۆماتیکی زیاد بکەیت بۆ ئەو لیستانە.

## دامەزراندن

1. 'ultimate-multisite-mailchimp' باربکە بۆ بوخچەی '/wp-content/plugins/'
2. plugin ـەکە لە ڕێگەی مێنیوی 'Plugins' لە WordPress Network Admin ـەکەتدا چالاک بکە

## ڕێبەری بەکارهێنان

هاوکاتکردنی ئیمەیڵی بەشداربووان لەگەڵ لیستەکانی MailChimp ـەکەت بە Add-on ـی Ultimate Multisite: MailChimp Integration ـی ئێمە زۆر ئاسانە. ئەم فێرکارییە ئامانجی ئەوەیە یارمەتیت بدات هەموو شتێک ڕێکبخەیت!

## وەرگرتنی API key

پێش ئەوەی دەست بکەین بە چوونە ناو ڕێکخستنە جیاوازە بەردەستەکان، پێویستمان بە API key ـی MailChimp هەیە. ئەمە ڕێگە بە Ultimate Multisite دەدات لەگەڵ MailChimp API قسە بکات بۆ وەرگرتنی لیستەکان، گرووپەکان، و زیادکردنی ناونیشانی ئیمەیڵ بۆ ئەو لیست و گرووپانە کاتێک بەکارهێنەران تۆمار دەبن.

ئەم فێرکارییە پەیڕەو بکە بۆ وەرگرتنی API key ـی MailChimp ـی خۆت: [دەربارەی API Keys | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

<!-- Screenshot unavailable: MailChimp API key settings in Ultimate Multisite Add-on Settings -->

_API key ـی MailChimp ـەکەت زیاد بکە لە_ _**Ultimate Multisite Settings - > Add-on Settings -> MailChimp**_

## دۆخی یەکخستن

Add-on ـی MailChimp Integration دوو “دۆخی یەکخستن” پێشکەش دەکات، واتە دوو ڕێگەی جیاواز هەیە کە دەتوانیت Add-on ـەکە بەکاربهێنیت بۆ هاوکاتکردنی ناونیشانەکانی ئیمەیڵ.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_دوو دۆخی یەکخستن بۆ نەرمونیانی زیاتر!_

### دۆخی ١: لیستی فرە

دۆخی بنەڕەتی سوود لە **لیستی فرە** وەردەگرێت. ئەم دۆخە ڕێگەت پێدەدات بۆ هەر یەکێک لە پلانەکان چەند لیستێکی MailChimp هەڵبژێریت. ئەمە باشترین هەڵبژاردەیە ئەگەر بەکارهێنەرەکانت بۆ لیستە جیاوازەکان دابەش دەکەیت.

بۆ نموونە، با وابزانین ٣ لیستی جیاوازت لە Account ـی MailChimp ـەکەتدا هەیە: **Newsletter** ، **Plan A** ، و **Plan B**. دەتەوێت هەموو بەکارهێنەران، بەبێ گوێدانە پلانیان، لە لیستی Newsletter ـیشدا بن، بۆ ئەوەی نوێکارییەکانت لەدەست نەدەن.

بۆ گەیشتن بەوە، دەتوانیت دۆخی Multiple Lists بەکاربهێنیت بۆ زیادکردنی هەردوو لیستی **Newsletter** و **Plan A** بۆ ڕێکخستنەکانی MailChimp ـی Plan A، و **Newsletter** و **Plan B** بۆ ڕێکخستنەکانی MailChimp ـی Plan B.

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_لیستە بنەڕەتییەکان بۆ Account ـە نوێکان و Account ـە هەڵوەشاوەکان هەڵبژێرە_

### دۆخی ٢: لیستی تاک + گرووپی فرە

MailChimp ڕێگەت پێدەدات لیستەکەت دابەش بکەیت بۆ گرووپی بچووکتر لە بەکارهێنەران کە پێیان دەوترێت **گرووپەکان**. ئەمە ڕێگەت پێدەدات کۆنترۆڵی زیاتر هەبێت چونکە دەتوانیت کەمپەینێک بۆ ژێرکۆمەڵەیەکی دیاریکراوی بەشداربووانت بنێریت بەبێ ئەوەی پێویست بکات چەند کۆپییەک لە هەمان کەمپەین لەسەر MailChimp دروست بکەیت (کە دواتر شیکردنەوەی پێوەرەکانی ئەو کەمپەینە قورس دەکات).

بۆ نموونە، دەتوانیت لیستێکی تاکت هەبێت بە ناوی **My Network Subscribers,** و لە ناویدا گرووپت هەبێت بۆ **Plan A** و **Plan B**.

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_لیست و گرووپە بنەڕەتییەکان هەڵبژێرە، ئەگەر دۆخی یەکخستنی “لیستی تاک + گرووپی فرە” بەکاردەهێنیت_

## لیست و گرووپە بنەڕەتییەکان

دوای ئەوەی ئەو دۆخی یەکخستنە هەڵدەبژێریت کە زیاتر بۆ کەیسی بەکارهێنانی تایبەتی تۆ مانادارە، کاتی هەڵبژاردنی لیستە بنەڕەتییەکانە (و گرووپەکان، بەپێی دۆخەکە). لیستی بنەڕەتی بەکاردەهێنرێت کاتێک پلانێکی دیاریکراو هیچ لیستێکی تایبەتی هەڵنەبژاردبێت.

هەروەها دەتوانیت لیستی بنەڕەتی بۆ Account ـە هەڵوەشاوەکان هەڵبژێریت. هەر کاتێک بەکارهێنەرێک subscription ـەکەی کۆتایی پێبهێنێت و Account ـەکەی لە تۆڕەکەت لاببات، ناونیشانی ئیمەیڵەکەی دەگوازرێتەوە بۆ ئەو لیستەی بۆ Account ـە هەڵوەشاوەکان دانراوە.

## چالاککردنی Double Subscription

لە هەمان لاپەڕەی ڕێکخستنەکاندا، دەتوانیت **Double Subscription** چالاک بکەیت. ئەم هەڵبژاردەیە بەکارهێنەرەکە زیاد دەکات بۆ هەردووکی لیستی بنەڕەتی هەڵبژێردراو و ئەو لیستەی بۆ پلانی ئەو بەکارهێنەرە هەڵبژێردراوە. ئەگەر ئەم هەڵبژاردەیە چالاک نەکرابێت، بەکارهێنەرەکە تەنها زیاد دەکرێت بۆ ئەو لیستەی بۆ پلانی بەکارهێنەرەکە هەڵبژێردراوە. ئەگەر هیچ لیستێک بۆ ئەو پلانە تایبەتییە دانەنرابێت، ئەوا بەکارهێنەرەکە زیاد دەکرێت بۆ لیستی بنەڕەتی هەڵبژێردراو.

<!-- Screenshot unavailable: MailChimp double subscription toggle setting -->

_دەتوانیت بەکارهێنەرەکانت لە کاتی تۆماربووندا لە هەردووکی لیستی بنەڕەتی و لیستی پلانەکەدا بەشدار بکەیت_

## هەڵبژاردنی لیست و گرووپ بۆ هەر یەکێک لە پلانەکان

بڕۆ بۆ **پلانەکان** ـت و ئەو پلانە هەڵبژێرە کە دەتەوێت دەستکاری بکەیت. تێبینی بکە کە لەسەر meta-box ـی **هەڵبژاردە پێشکەوتووەکان**، تابێکی نوێ بە ناوی MailChimp بەردەست دەبێت.

لەسەر ئەو تابە، دەتوانیت چەند لیستێک یان لیستێکی تاک و چەند گرووپێک هەڵبژێریت (بەپێی دۆخی یەکخستنەکە) بۆ ئەو پلانە.

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_ئەو لیست و گرووپانە هەڵبژێرە کە دەتەوێت بۆ هەر یەکێک لە پلانەکانت بەکاریان بهێنیت_

کاتێک بەکارهێنەرێکی نوێ بۆ ئەو پلانە تۆمار دەبێت، ناونیشانی ئیمەیڵەکەی زیاد دەکرێت بۆ لیست و گرووپە هەڵبژێردراوەکان!
