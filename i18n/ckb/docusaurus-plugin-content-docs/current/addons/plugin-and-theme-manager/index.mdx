# بەڕێوەبەری پێوەکراو و ڕووکار

Ultimate Multisite: بەڕێوەبەری پێوەکراو و ڕووکار

زانیارییە metaـی پێوەکراو و ڕووکارە دامەزراوەکان (ناونیشان، نووسەر، وێنۆچکە و وەسف) دەستکاری بکە یان بیشارەوە. پۆل بۆ پێوەکراو و ڕووکارەکان دروست بکە بۆ ئەوەی ڕێگە بە بەکارهێنەرەکانت بدەیت پاڵێویان بکەن و پەڕەیەکی جوانی تایبەتی پێوەکراوەکان زیاد بکەیت بۆ پانێلی بەکارهێنەرەکانت.

بۆ بەکارهێنانی پێوەکراوەکە، دەتوانیت تەنها بچیتە پەڕەکانی پێوەکراوەکان یان ڕووکارەکان لە بەڕێوەبەری تۆڕەکەت و کرتە لەسەر "دەستکاریکردنی زانیاری پێوەکراو/ڕووکار" بکەیت.

## دامەزراندن {#installation}

1. 'ultimate-multisite-plugin-and-theme-manager' باربکە بۆ بوخچەی '/wp-content/plugins/'
2. پێوەکراوەکە لە ڕێگەی مێنیوی 'پێوەکراوەکان' لە WordPress Network Adminـەکەت چالاک بکە

## ڕێنمایی بەکارهێنان {#usage-guide}

Ultimate Multisite: بەڕێوەبەری پێوەکراو و ڕووکار زیادکراوێکی Ultimate Multisiteـی **premium** و بەهێزە کە کۆنترۆڵی تەواوت پێدەدات لەسەر ئەوەی بەکارهێنەرانی تۆڕەکەت چۆن هەڵبژاردەکانی پێوەکراو و ڕووکار دەبینن و مامەڵەیان لەگەڵ دەکەن لەسەر تۆڕەکەت.

ئەم زیادکراوە سێ مەبەست جێبەجێ دەکات:

  1. ڕێگەت پێدەدات metadataـی پێوەکراو و ڕووکارەکان بە تەواوی بگۆڕیت. ئەمە ناو، وەسف، ناوی نووسەر، و تەنانەت وێنۆچکەکانیش دەگرێتەوە!

  2. ڕێگەت پێدەدات پێوەکراو و ڕووکارەکان پۆلێن بکەیت، بۆ ئەوەی ڕێگە بە بەکارهێنەرەکانت بدەیت بە ئاسانی ڕێکیان بخەن.

  3. هەڵبژاردەت پێدەدات پەڕەی کۆن و بێزارکەری پێوەکراوەکان بگۆڕیت بە پەڕەیەکی نوێی هاوشێوەی ڕووکارەکان کە بە پۆلەکان ڕێکدەخرێت.

دوای داگرتن و دامەزراندنی، بازدەرێکی نوێ - _ڕێکخستنەکانی زیادکراو_ \- لەسەر پەڕەی ڕێکخستنەکانی Ultimate Multisite دەردەکەوێت.

_لێرەدا هەڵبژاردە گشتییە زیادکراوەکان دەدۆزیتەوە کە لەلایەن Ultimate Multisite: بەڕێوەبەری پێوەکراو و ڕووکار زیاد کراون_

لە پەڕەی _پێوەکراوەکانی_ تۆڕەکەت ئێستا دەتوانیت هەموو زانیارییە metaـی پێوەکراوەکان دەستکاری بکەیت، وەک ناو، نووسەر، و تەنانەت وێنۆچکەی پێوەکراوەکەش.
