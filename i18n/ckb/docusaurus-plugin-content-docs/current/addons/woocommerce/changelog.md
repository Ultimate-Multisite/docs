# تۆماری گۆڕانکاریی یەکخستنی WooCommerce {#woocommerce-integration-changelog}

وەشانی 2.2.0 - بڵاوکرایەوە لە 2026-07-01
* نوێ: بڕەکانی باجی Ultimate Multisite ئێستا وەک هێڵی تێچووی جیاوازی WooCommerce لە کاتی تەواوکردنی کڕین دەردەکەون، کە کۆی گشتی باجەکان پێش پارەدان ڕوونتر دەکاتەوە.
* نوێ: ڕێکخستنی هەڵبژاردنی ئیرادی "هەڵپەساردنی دەستبەجێی ئەندامێتییەکان لە نوێکردنەوەی سەرنەکەوتوو" و فلتەری `wu_woo_suspend_on_payment_failure` زیادکرا بۆ ئەو ماڵپەڕانەی کە دەیانەوێت هەڵپەساردنی نوێکردنەوەی سەرنەکەوتوو لە ماوەی هەوڵدانەوەی WooCommerce Subscriptionsدا ڕووبدات.
* چاککراوە: دۆخەکانی WooCommerce Subscription هاوسەنگکران کە دەکرا دوای نوێکردنەوەی سەرنەکەوتوو یان چاکبووەوە لەگەڵ ئەندامێتییەکانی Ultimate Multisite ناهاوکات بمێننەوە.
* چاککراوە: دراوی فرۆشگای WooCommerce زیادکرا بۆ لیستی دراوەکانی Ultimate Multisite کاتێک نەبوو.
* چاککراوە: وردەکارییەکانی پسووڵەی کڕیار پارێزرا کاتێک ئابوونەداران دەنێردرانەوە بۆ تەواوکردنی کڕینی WooCommerce.
* باشکراوە: گونجانی لەگەڵ Jetpack Autoloader 5 زیادکرا.
* باشکراوە: دروستکردنی پاکێجی بڵاوکردنەوە پاککرایەوە بۆ ئەوەی zipەکانی GitHub و بازاڕ بوخچەکانی ئامادەکاریی توێکەڵ و فایلەکانی گەشەپێدان نەگرنەخۆ.

وەشانی 2.0.6 - بڵاوکرایەوە لە 2026-01-16
* باشکردن: ئابوونە سەرەکییەکان لە زیادکراوەکەدا بخەرەخۆ. چیتر پێویستی بە پاشگری Woocommerce Subscriptinos نییە.

وەشانی 2.0.5 - بڵاوکرایەوە لە 2026-01-09

وەشانی 2.0.1 - بڵاوکرایەوە لە 2023-08-09

* زیادکراوە: گونجان لەگەڵ Woocommerce 7.9.0.
* زیادکراوە: گونجان لەگەڵ Woocommerce Subscriptions 5.3.0.
* زیادکراوە: پشتگیری بۆ نوێکردنەوەی ئەندامێتی.
* زیادکراوە: ئاگادارکردنەوەکان دەربارەی تاقیکردنەوە و کرێی دامەزراندن لە Woocommerce.
* زیادکراوە: ناساندنی بەرهەمەکانی Ultimate Multisite Woocommerce بە نرخێکی meta.
* زیادکراوە: چاککردنەوەی یەکجارە زیادکرا بۆ نیشانکردنی هەموو بەرهەمە پەیوەندیدارەکانی Ultimate Multisite لە Woocommerce.
* زیادکراوە: بەرهەمە دروستکراوەکانی Ultimate Multisite لە لیستی Woocommerce لابران.
* باشکردن: داشکاندنی Woocommerceی نادووبارە دروستکرا بۆ جێبەجێکردن لەسەر سەبەتە.
* باشکردن: داشکاندنی دووبارەبووەوە گەڕێنرایەوە بۆ بەرهەمی Woocommerce.
* باشکردن: پێناسی داشکاندنی دووبارەبووەوە زیادکرا بۆ بەرهەمی Woocommerce.
* باشکردن: جۆری بەرهەم لە کاتی تەواوکردنی کڕین دڵنیـا کرا.
* چاککراوە: دۆخی ئەندامێتی لە ماوەی پرۆسەی دابەزاندندا پارێزرا.
* چاککراوە: پشکنرا کە ئایا ئابوونەیەک هەیە بۆ خۆپاراستن لە هەڵە لە ماوەی پرۆسەی هەڵوەشاندنەوەدا.
* چاککراوە: بەرواری دەستپێکردنی ئابوونە زیادکرا بۆ بەکارهێنان لە ئابوونەکانی Woocommerce.
* ناوخۆیی: پرۆسەی بنیاتنانی نوێی PHP 8.1 جێبەجێکرا.

وەشانی 2.0.0 - دووبارەنووسینەوەی تەواو.

* زیادکراوە: میتۆدی پرۆسەی هەڵوەشاندنەوە بۆ لابردنی ئابوونەی woo کاتێک دەروازەکە دەگۆڕدرێت یان ئەندامێتییەکە هەڵدەوەشێنرێتەوە؛
* زیادکراوە: بەڕێوەبەر بۆ دابەزاندن و بەرزکردنەوەی ئەندامێتییەکان؛
* باشکردن: پێویستییەکانی woocommerce لە فۆڕمی نوێکردنەوەی کڕیار لە ماڵپەڕە لاوەکییەکاندا باربکە بۆ ڕێگەدان بە نوێکردنەوەی Account؛
* باشکردن: سەبەتەی Woocommerce بە دروستی باربکە ئەگەر بوونی نەبێت؛
* باشکردن: دڵنیابوون لەوەی کە لەسەر خشتەکانی ماڵپەڕی سەرەکین کاتێک تەواوکردنی کڕین پرۆسە دەکرێت؛
* باشکردن: داواکاریی نوێکردنەوەی Ultimo بکە بە پشتبەستن بە نرخی داواکاریی ئابوونەی Woocommerce و نەک لە دوا پارەدان؛
* چاککراوە: بەستەری دوگمەی چوون بۆ ئەندامێتیی WU؛
* چاککراوە: داواکاریی Ultimo وەک پارەدراو دابنێ کاتێک نوێکردنەوەی ئابوونەکانی Woocommerce پارەدراوە؛
* بنیاتنان: MPB زیادبکە وەک بنیاتنەر؛
