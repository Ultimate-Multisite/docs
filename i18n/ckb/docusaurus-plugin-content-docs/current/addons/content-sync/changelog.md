# گۆڕانکارییەکانی Content Sync

### Version 1.1.0 - بڵاوکراوەتەوە لە 2025-12-31 {#version-110---released-on-2025-12-31}
* نوێ: پشتگیری بۆ هاوکاتکردنی هەموو جۆرەکانی پۆست (پۆستەکان، پەڕەکان، جۆرە پۆستی تایبەتەکان)
* نوێ: پاڵاوتنی جۆری پۆست لە هەنگاوی هەڵبژاردنی ناوەڕۆک
* نوێ: هەنگاوی هەڵبژاردنی سایتی سەرچاوە - هاوکاتکردن لە هەر سایتێکەوە، نەک تەنها سایتی سەرەکی
* نوێ: پێش-هەڵبژاردنی سایتی منداڵ بۆ سایتە قاڵبەکان
* نوێ: دەرهێنانی میدیا لە post_content بۆ هەموو جۆرەکانی پۆست
* نوێ: هاوکاتکردنی Taxonomy (پۆلەکان، تاگەکان، taxonomy تایبەتەکان)
* گۆڕاوە: ناوەکە لە "Elementor Template Sync" گۆڕا بۆ "Content Sync"
* گۆڕاوە: هەنگاوی "Select Templates" ناوی گۆڕا بۆ "Select Data"
* باشترکراوە: کۆدی تایبەت بە Elementor ئێستا تەنها بۆ جۆری پۆستی elementor_library کار دەکات

### Version 1.0.0 - بڵاوکراوەتەوە لە 2025-12-19 {#version-100---released-on-2025-12-19}
* بڵاوکردنەوەی سەرەتایی
* ویزاردی چەند-هەنگاوی بۆ هەڵبژاردنی قاڵب و سایت
* پرۆسەکردنی کۆمەڵەیی لەگەڵ فیدباکی پێشکەوتنی ڕاستەوخۆ
* هاوکاتکردنی تەواوی میدیا لەگەڵ دووبارە-نەخشەکردنی ID بە شێوەیەکی خۆکار
* یەکخستنی ڕێکخستنەکان لەگەڵ Ultimate Multisite
* پشتگیری دەرخستنی سایت لە ڕێگای ڕێکخستنەکانی تۆڕەوە
* پاککردنەوەی کاشی Elementor دوای هاوکاتکردن
* پاراستنی taxonomy ـی قاڵب
