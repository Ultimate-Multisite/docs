# هاوکاتکردنی ناوەڕۆک

Multisite Ultimate: هاوکاتکردنی ناوەڕۆک

ئەم پێوەکراوە بۆ Multisite Ultimate ڕێگە بە بەڕێوەبەرانی تۆڕ دەدات ناوەڕۆک لە هەر ماڵپەڕێکەوە بۆ ماڵپەڕەکانی تری ناو تۆڕ هاوکات بکەن. پشتگیری هەموو جۆرەکانی بابەت دەکات، لەوانە بابەتەکان، پەڕەکان، جۆرە بابەتی تایبەت، و قاڵبەکانی Elementor. ئەم پێوەکراوە ڕووکاری ویزاردی چەند-قۆناغی تێدایە بۆ هەڵبژاردنی ناوەڕۆک و ماڵپەڕە ئامانجەکان، لەگەڵ پرۆسەکردنی بە کۆمەڵ و فیدباکی پێشکەوتنی ڕاستەوخۆ.

## تایبەتمەندییەکان {#features}

* هەڵبژاردنی ناوەڕۆک: تۆڕی کارتی بینراو لەگەڵ وێنۆچکە و فلتەرکردنی جۆری بابەت
* هەموو جۆرەکانی بابەت: هاوکاتکردنی بابەتەکان، پەڕەکان، بەرهەمەکان، قاڵبەکانی Elementor، و هەر جۆرە بابەتی تایبەتێک
* هەڵبژاردنی ماڵپەڕ: هەڵبژاردنی ئەو ماڵپەڕانەی ناوەڕۆکیان بۆ هاوکات دەکرێت، لەگەڵ وەدەرنانی ماڵپەڕی سەرچاوە و پێش-هەڵبژاردنی ماڵپەڕی منداڵ
* پرۆسەکردنی بە کۆمەڵ: هەر جارەی بۆ ١٠ ماڵپەڕ هاوکات دەکات لەگەڵ نوێکردنەوەی پێشکەوتن
* هاوکاتکردنی میدیا: پەڕگە میدیاییەکان (وێنە، ڤیدیۆ) لە ناوەڕۆکی بابەتەوە بۆ ماڵپەڕە ئامانجەکان بە شێوەی خۆکار کۆپی و دووبارە نەخشەسازی دەکات
* هاوکاتکردنی پۆلێنکاری: هاوپۆلەکان، تاگەکان، و پۆلێنکارییە تایبەتەکان دەپارێزێت
* پشتگیری Elementor: پشتگیری تەواو بۆ قاڵبەکانی Elementor لەگەڵ دووبارە دروستکردنەوەی CSS و دووبارە نەخشەسازی ناسنامەی قاڵب
* چارەسەری ناکۆکی: هەڵبژاردە بۆ جێگرتنەوەی ناوەڕۆکی هەبوو یان بازدان لێی
* بەڕێوەبردنی کاش: دوای هاوکاتکردن، کاشی Elementor لەسەر ماڵپەڕە ئامانجەکان بە شێوەی خۆکار پاک دەکاتەوە

## پێداویستییەکان {#requirements}

* WordPress 5.3 یان بەرزتر
* PHP 7.4 یان بەرزتر
* پێوەکراوی Multisite Ultimate (چالاک)
* پێوەکراوی Elementor (ئارەزوومەندانە، پێویستە بۆ هاوکاتکردنی قاڵبی Elementor)

## دامەزراندن {#installation}

1. 'ultimate-multisite-content-sync' بار بکە بۆ بوخچەی '/wp-content/plugins/'
2. پێوەکراوەکە لە ڕێگەی مێنیوی 'پێوەکراوەکان' لە WordPress Network Admin ـی خۆت چالاک بکە
3. ڕێکخستنەکان لە Ultimate Multisite → ڕێکخستنەکان → هاوکاتکردنی ناوەڕۆک ڕێکبخە
4. بڕۆ بۆ Ultimate Multisite → هاوکاتکردنی ناوەڕۆک بۆ دەستپێکردنی هاوکاتکردن
