# فرە-دراو {#multi-currency}

Ultimate Multisite: فرە-دراو تۆڕەکەت توانادار دەکات پارەدان بە چەندین دراو وەربگرێت. نرخەکان بە شێوەیەکی خۆکار بە بەکارهێنانی نرخی ئاڵوگۆڕی زیندوو یان دەستی دەگۆڕێت، ڕێگەت پێدەدات نرخە وردەکان بۆ هەر دراوێک بۆ هەر بەرهەمێک دابنێیت، و دراوی پێویستی هەر سەردانکەرێک لە ڕێگەی شوێن-دۆزینەوە دەستنیشان دەکات — هەمووی لە Dashboardێکی بەڕێوەبەریی تاکەوە بەڕێوەدەبرێت.

## تایبەتمەندییە سەرەکییەکان {#key-features}

- **گۆڕینی خۆکاری نرخی ئاڵوگۆڕ** — نرخەکان لە دراوی بنەڕەتییەکەتەوە دەگۆڕدرێن بە بەکارهێنانی ئەو نرخانەی لە دابینکەری هەڵبژێردراوتەوە وەردەگیرێن
- **جێگرەوەکانی نرخدانان بۆ هەر بەرهەمێک** — نرخە وردە تایبەت بە دراو بۆ هەر بەرهەمێک دابنێ بۆ تێپەڕاندنی گۆڕینی خۆکار
- **خانەی هەڵبژاردنی دراو لە تۆماربوون** — ڕێگە بە کڕیاران دەدات لە کاتی checkout/تۆماربووندا دراوی پێویستیان هەڵبژێرن
- **چەند دابینکەری نرخی ئاڵوگۆڕ** — ECB، Frankfurter، Stripe، یان نرخە دەستییە تەواوەکان
- **دۆزینەوەی دراو لەسەر بنەمای شوێن-دۆزینەوە** — بە شێوەیەکی خۆکار ئەو دراوە پێش-هەڵدەبژێرێت کە لەگەڵ وڵاتی سەردانکەرەکە دەگونجێت
- **Dashboardی بەڕێوەبەری** — دراوە چالاکەکان، نرخەکانی ئاڵوگۆڕ، و ڕێکخستنەکانی دابینکەر لە یەک شوێنەوە بەڕێوەبەرە

## پێداویستییەکان {#requirements}

- Ultimate Multisite 2.0.0 یان بەرزتر
- دامەزراندنی WordPress Multisite
- PHP 7.4 یان بەرزتر

## دامەزراندن {#installation}

1. فایلەکانی pluginەکە باربکە بۆ `/wp-content/plugins/ultimate-multisite-multi-currency`
2. pluginەکە لە ڕێگەی پەردەی **Plugins** لە WordPress چالاک بکە (Network Activate)
3. بچۆ بۆ **Ultimate Multisite → فرە-دراو** بۆ کردنەوەی Dashboardی بەڕێوەبەری
4. ئەو دراوانە زیاد بکە کە دەتەوێت وەریان بگریت و دابینکەری نرخی ئاڵوگۆڕەکەت ڕێکبخە

## ڕێکخستن {#configuration}

### Dashboardی بەڕێوەبەری {#admin-dashboard}

Dashboardی بەڕێوەبەری فرە-دراو (**Ultimate Multisite → فرە-دراو**) شوێنی ناوەندییە بۆ:

- زیادکردن، چالاککردن، یان ناچالاککردنی دراوەکان
- هەڵبژاردنی دابینکەری نرخی ئاڵوگۆڕ و نوێکردنەوەی نرخەکان بەپێی داواکاری
- پێداچوونەوە بە نرخە ئێستاکان بۆ هەموو دراوە چالاکەکان
- دانانی نرخە دەستییەکان کاتێک دابینکەرە خۆکارەکان گونجاو نین

### دابینکەرانی نرخی ئاڵوگۆڕ {#exchange-rate-providers}

| دابینکەر | وەسف |
|---|---|
| **ECB** | بانکی ناوەندیی ئەوروپا — نرخە ڕۆژانەی خۆڕایی بۆ تۆڕەکانی پشتبەستوو بە EUR |
| **Frankfurter** | APIی ئاوێنەی ECBی سەرچاوە-کراوە — خۆڕایی، پێویستی بە کلیلی API نییە |
| **Stripe** | نرخە زیندووەکانی FXی Stripe بەکاردەهێنێت — پێویستی بە یەکخستنێکی چالاکی Stripe هەیە |
| **دەستی** | تۆ نرخەکان ڕاستەوخۆ داخل دەکەیت؛ هیچ بانگەوازێکی APIی دەرەکی نییە |

دابینکەرەکەت لە ژێر **Ultimate Multisite → فرە-دراو → ڕێکخستنەکان → دابینکەری نرخی ئاڵوگۆڕ** هەڵبژێرە. نرخەکان دەتوانرێت هەر کاتێک بەدەستی نوێ بکرێنەوە یان بۆ نوێبوونەوەی خۆکار خشتە بکرێن.

### جێگرەوەکانی نرخدانان بۆ هەر بەرهەمێک {#per-product-pricing-overrides}

کاتێک گۆڕینی خۆکار بە قەدەر پێویست ورد نییە، دەتوانیت نرخێکی ورد بۆ دراوێکی دیاریکراو لەسەر هەر بەرهەمێک دابنێیت:

1. دەستکاریکەری بەرهەم بکەرەوە (**Ultimate Multisite → بەرهەمەکان → دەستکاری**)
2. بڕۆ خوارەوە بۆ بەشی **نرخدانانی دراو**
3. نرخی ورد بۆ هەر دراوێک بنووسە کە دەتەوێت جێگرەوەی بکەیت
4. بەرهەمەکە پاشەکەوت بکە

بەرهەمەکان بەبێ جێگرەوە نرخە گۆڕدراوەکە لە دراوی بنەڕەتییەکەتەوە بەکاردەهێنن.

### خانەی هەڵبژاردنی دراو لە تۆماربوون {#currency-selector-signup-field}

خانەی هەڵبژاردنی دراو دەتوانرێت زیاد بکرێت بۆ ڕەوتی تۆماربوون/checkoutـەکەت بۆ ئەوەی کڕیاران لە کاتی تۆمارکردندا دراوی پێویستیان هەڵبژێرن:

1. بچۆ بۆ **Ultimate Multisite → فرە-دراو → ڕێکخستنەکان**
2. **پیشاندانی هەڵبژێری دراو لە تۆماربوون** چالاک بکە
3. بە ئارەزوو دراوێکی بنەڕەتی دابنێ یان بهێڵە شوێن-دۆزینەوە نرخە پێش-هەڵبژێردراوەکە دیاری بکات

دراوی هەڵبژێردراو لەگەڵ subscriptionەکە هەڵدەگیرێت و بۆ هەموو پارەدانەکانی دواتر بەکاردەهێنرێت.

### دۆزینەوەی دراو لەسەر بنەمای شوێن-دۆزینەوە {#geolocation-based-currency-detection}

کاتێک دۆزینەوەی شوێن-دۆزینەوە چالاکە، addonەکە وڵاتی سەردانکەرەکە بە دراوێکەوە دەبەستێتەوە و لە خانەی هەڵبژاردنی دراوەکەدا پێش-هەڵیدەبژێرێت. بۆ چالاککردن:

1. بچۆ بۆ **Ultimate Multisite → فرە-دراو → ڕێکخستنەکان**
2. **دۆزینەوەی دراو لە شوێنی سەردانکەرەوە** چالاک بکە
3. وڵاتان بە دراوەکانەوە لە خشتەی **نەخشەکردنی شوێن-دۆزینەوە** ببەستەوە

ئەگەر هیچ نەخشەکردنێک بۆ وڵاتی سەردانکەرێک نەبێت، دراوی بنەڕەتیی تۆڕەکە وەک جێگرەوە بەکاردەهێنرێت.

## پرسیارە دووبارەکان {#frequently-asked-questions}

### چی ڕوودەدات ئەگەر دابینکەری نرخی ئاڵوگۆڕ بەردەست نەبێت؟ {#what-happens-if-an-exchange-rate-provider-is-unavailable}

addonەکە دەگەڕێتەوە بۆ دوا نرخەکان کە بە سەرکەوتوویی وەرگیراون. ئەگەر هیچ نرخێک هەرگیز وەرنەگیرابێت، دەگەڕێتەوە بۆ ئەو نرخە دەستییانەی ڕێکتخستووە. پێشنیار دەکرێت نرخە دەستییەکان وەک تۆڕی پاراستن ڕێکبخرێن.

### دەتوانم دابینکەری جیاواز بۆ دراوە جیاوازەکان بەکاربهێنم؟ {#can-i-use-different-providers-for-different-currencies}

نەخێر — یەک دابینکەر لە یەک کاتدا بۆ هەموو دراوەکان چالاکە. ئەگەر پێویستت بە کۆنترۆڵی تەواو لەسەر نرخە تاکەکەسییەکان هەیە، **دەستی** بەکاربهێنە.

### ئایا شوێن-دۆزینەوە پێویستی بە pluginێکی جیاواز هەیە؟ {#does-geolocation-require-a-separate-plugin}

هیچ pluginێکی شوێن-دۆزینەوەی دەرەکی پێویست نییە. addonەکە دۆزینەوەی IP-بۆ-وڵاتی ناوخۆی WordPress بەکاردەهێنێت. بۆ وردیی زیاتر، خزمەتگوزارییەکی تایبەتی شوێن-دۆزینەوە دەتوانرێت لە ڕێگەی filter hookە دابینکراوەکانەوە یەکبخرێت.

### ئایا جێگرەوەکانی هەر بەرهەمێک پێش باج جێبەجێ دەکرێن یان دوای باج؟ {#are-per-product-overrides-applied-before-or-after-tax}

جێگرەوەکانی هەر بەرهەمێک نرخی پێش-باج لە دراوی ئامانجدا دادەنێن. باج لەسەر نرخی جێگرەوەکە بە بەکارهێنانی ڕێکخستنی باجی ئێستات حیساب دەکرێت.
