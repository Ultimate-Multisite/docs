# Tutor Multisite Compatibility

پێوەکراوێکی WordPress کە چاکسازی گونجاندن بۆ Tutor LMS لە ژینگە فرە-ماڵپەڕەکاندا زیاد دەکات.

## وەسف

ئەم پێوەکراوە چاکسازییە گرنگەکانی گونجاندن دابین دەکات بۆ کارپێکردنی Tutor LMS لەسەر تۆڕە فرە-ماڵپەڕەکانی WordPress. بەڕێوەبردنی تۆمارکردنی بەکارهێنەر، بەڕێوەبردنی مامۆستا، سنووردارکردنی کۆرس، و تایبەتمەندییەکانی تری تایبەت بە فرە-ماڵپەڕ بەڕێوە دەبات.

## تایبەتمەندییەکان

- **بەڕێوەبردنی خشتەی بنکەدراوە**: بە شێوەیەکی خۆکار خشتەکانی Tutor LMS کە بەهۆی سنووردارکردنەکانی کلیلی دەرەکی فرە-ماڵپەڕ دروست نابن، دەدۆزێتەوە و دروستیان دەکات
- **تۆمارکردنی بەکارهێنەری فرە-ماڵپەڕ**: مامەڵەکەرانی AJAXی تایبەت بۆ بەڕێوەبردنی تۆمارکردنی بەکارهێنەر لەنێوان ماڵپەڕەکانی تۆڕدا
- **بەڕێوەبردنی ڕۆڵی مامۆستا**: دیاریکردن و بەڕێوەبردنی خۆکاری ڕۆڵی مامۆستا لە ژینگە فرە-ماڵپەڕەکاندا
- **سنووردارکردنی کۆرس**: تێکەڵکردن لەگەڵ Ultimate Multisite بۆ بەڕێوەبردنی سنووری دروستکردنی کۆرس و فانێڵ بەپێی پلانی ماڵپەڕ
- **بینینەکانی بەڕێوەبەری تایبەت**: لیستی مامۆستا و ڕووکاری بەڕێوەبردن کە ئاگاداری فرە-ماڵپەڕە
- **ئامرازەکانی بەڕێوەبەر**: ڕووکاری بەڕێوەبردنی بنکەدراوە بۆ بەڕێوەبەرانی تۆڕ و ماڵپەڕ

## پێداویستییەکان

- WordPress 5.0+
- PHP 7.4+
- پێوەکراوی Tutor LMS (بە شێوەیەکی خۆکار لە ڕێگەی سەردێڕی `Requires Plugins` جێبەجێ دەکرێت)
- دامەزراندنی فرە-ماڵپەڕی WordPress
- پێوەکراوی Ultimate Multisite (بۆ کارایی سنووردارکردنی کۆرسەکان)

## دامەزراندن

1. فایلەکانی پێوەکراوەکە باربکە بۆ `/wp-content/plugins/tutor-multisite-compatibility/`
2. دڵنیابە Tutor LMS دامەزراوە و چالاک کراوە
3. پێوەکراوەکە لە ڕێگەی مێنیوی 'پێوەکراوەکان' لە WordPress چالاک بکە
4. پێوەکراوەکە بە شێوەیەکی خۆکار لەگەڵ Tutor LMS تێکەڵ دەبێت

## کارایی گوازراوە لە MU plugin

ئەم پێوەکراوە ئەو کاراییانە لەخۆ دەگرێت کە پێشتر لە `wp-content/mu-plugins/daves.php` بوون:

- سنووردارکردنی گۆڕینی دۆخی کۆرس
- سنووردارکردنی دروستکردنی کۆرس
- تێکەڵکردن و سنوورەکانی CartFlows
- بەڕێوەبردنی ڕۆڵی مامۆستای Tutor
- مامەڵەکردنی تۆمارکردنی بەکارهێنەری فرە-ماڵپەڕ
- پیشاندانی ناسنامەی بەرهەمی کۆرس

## بەڕێوەبردنی بنکەدراوە

### کێشەکە

لە ژینگە فرە-ماڵپەڕەکانی WordPressدا، دروستکردنی خشتەی Tutor LMS بۆ ئەو خشتانەی سنووردارکردنی کلیلی دەرەکیان بۆ خشتەی بەکارهێنەران هەیە سەرناکەوێت. ئەمە ڕوو دەدات چونکە:

- خشتە تایبەتەکانی ماڵپەڕ پێشگری وەک `wp_2_tutor_carts` بەکاردەهێنن
- کلیلە دەرەکییەکان ئاماژە بە `wp_2_users` دەکەن (کە بوونی نییە)
- خشتەی ڕاستەقینەی بەکارهێنەران `wp_users` ـە (لە سەرتاسەری تۆڕدا هاوبەشە)

### چارەسەرەکە

ئەم پێوەکراوە بە شێوەیەکی خۆکار:

1. **خشتە ونبووەکان دەدۆزێتەوە**: ئەو خشتانە دەپشکنێت کە بەهۆی سنووردارکردنەکانی کلیلی دەرەکی دروست نەکراون
2. **خشتەکان بە ئاماژەی دروست دروست دەکات**: ناوی گونجاوی خشتەی بەکارهێنەر بەکاردەهێنێت (`wp_users` لە فرە-ماڵپەڕدا)
3. **چەندین دۆخ بەڕێوە دەبات**:
   - چالاککردن/دووبارە چالاککردنی پێوەکراو
   - دروستکردنی ماڵپەڕی نوێ لە تۆڕدا
   - پشکنینی دەستیی خشتە لە ڕێگەی ڕووکاری بەڕێوەبەرەوە

### خشتە کاریگەرکراوەکان

- `tutor_coupon_usages` - کلیلی دەرەکی بۆ خشتەی بەکارهێنەران لەخۆ دەگرێت
- `tutor_carts` - کلیلی دەرەکی بۆ خشتەی بەکارهێنەران لەخۆ دەگرێت
- `tutor_notification_preferences` - خشتەی TutorPro کە کلیلی دەرەکی بۆ خشتەی بەکارهێنەران لەخۆ دەگرێت

### ڕووکاری بەڕێوەبەر

دەستت بە ئامرازەکانی بەڕێوەبردنی بنکەدراوە دەگات لە:

- **بەڕێوەبەری تۆڕ**: ڕێکخستنەکان → Tutor Multisite DB
- **بەڕێوەبەری ماڵپەڕ**: ئامرازەکان → Tutor Multisite DB

ڕووکارەکە ڕێگەت پێدەدات:
- بینینی خشتە ونبووەکان لە سەرجەم ماڵپەڕەکاندا
- دەستپێکردنی دەستیی دروستکردنی خشتە
- پشکنینی ڕێکخستنی بنکەدراوە
- بینینی زانیاری دۆخی ورد
