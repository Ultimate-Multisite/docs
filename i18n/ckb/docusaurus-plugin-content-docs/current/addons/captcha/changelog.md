# گۆڕانکارییەکانی Captcha

وەشان: 1.5.0 - بڵاوکرایەوە لە 2026-05-22
* نوێ: سنووردانەری ڕێژەی وەستاندنی تەواو — هەموو GET و POST ـێک لەسەر ڕووکارە پارێزراوەکانی Captcha دەژمێرێت (wp-login / register / lost-password / comments، WooCommerce my-account / checkout / pay-for-order، Ultimate Multisite checkout / inline-login) و بە HTTP 429، سەردێڕێکی `Retry-After`، و خەوی دواکەوتنی هەڕەمەکی (1–5 چرکە، بە سنووری توندی 15 چرکە) وەڵام دەداتەوە.
* نوێ: ڕێکخستنەکانی `cap_rate_limit_tarpit_min` / `cap_rate_limit_tarpit_max` بۆ ڕێکخستنی ماوەی دواکەوتن.
* نوێ: فلتەری `wu_cap_rate_limit_whitelist_ip` بۆ دەرکردنی مەوداکانی IP ـی متمانەپێکراو لە سنووردان.
* نوێ: کرداری `wu_cap_rate_limit_will_block` کە یەکسەر پێش ناردنی وەڵامی وەستاندنی تەواو کاردەکات.
* نوێ: دۆزینەوەی IP ـی کڕیار بە شێوەی بەرگری‌دار لە ساختەکاری. `Captcha_Core::get_client_ip()` (سەرچاوەی ڕاستی بۆ کلیلەکانی سەتڵی سنووری ڕێژە، captcha siteverify `remoteip`، و هاشەکانی IP ـی ئامار) ئێستا مۆدێلێکی متمانەی توند جێبەجێ دەکات: REMOTE_ADDR بنەماکەیە، `CF-Connecting-IP` تەنها کاتێک ڕێزی لێدەگیرێت کە هاوتای دەستبەجێ لەناو مەودای IP ـی ئێستای Cloudflare دابێت، و `X-Forwarded-For` تەنها کاتێک ڕێزی لێدەگیرێت کە هاوتای دەستبەجێ لە لیستی پراکسییە متمانەپێکراوەکانی ڕێکخراوی بەڕێوەبەر دابێت، لەگەڵ گەڕانێکی ڕاست‌بۆچەپ کە هەنگاوەکانی متمانەپێکراو/CF دەبازێنێت پێش ئەوەی لەسەر IP ـی سەردانکەر بجێگیر بێت.
* نوێ: ڕێکخستنی `cap_trust_cloudflare_headers` (بنەڕەت کوژاوەیە) — کاتێک لە پشت Cloudflare ـەوەن متمانە بە `CF-Connecting-IP` هەڵبژێرن. ئەم پێوەکراوە وێنەیەکی یەکخراوی Cloudflare CIDR لەخۆ دەگرێت و هەفتانە لە ڕێگەی wp-cron نوێی دەکاتەوە، لەگەڵ جێگرەوەی یەکخراو ئەگەر نوێکردنەوەکە سەرنەکەوت.
* نوێ: ڕێکخستنی `cap_trusted_proxies` — ناوچەی دەق بۆ CIDR ـەکان یان IP ـی سادە (هەر یەکەیان لە هێڵێک، کۆمێنتی `#` ڕێگەپێدراوە) کە پراکسی / هاوسەنگکەرەکانی بارکردنی پێشەوەی خۆت تێدا لیست دەکرێن. بەبێ ئەمە، `X-Forwarded-For` پشتگوێ دەخرێت تەنانەت کاتێک سنووردانەری ڕێژە چالاکیش بێت.
* نوێ: خۆدۆزینەوەی یەکەم‌چالاککردن بۆ دۆخی ئەگەری Cloudflare / پراکسی لەگەڵ ئاگادارییەکی بەڕێوەبەری "ڕێکخستنە دۆزراوەکان جێبەجێ بکە" بە یەک کرتە. پێوەکراوەکە هەرگیز نرخە پاشەکەوتکراوەکانت لەسەر نانووسێتەوە؛ ئەگەر هاتوچۆی دواتر پێشنیار بکات کە ڕێکخستنەکانت چیتر لەگەڵ ڕاستی ناگونجێن (بۆ نموونە Cloudflare مەوداکانی CIDR گۆڕیوە و CIDR ـی پراکسییەکەت ئێستا کۆنە)، ئاگادارییەکی ناگونجانی ناسڕدرێوە نوێکردنەوەی پێشنیارکراو پیشان دەدات.
* چاککراوە: دۆخی نەبینراو چیتر بە بێدەنگی `cap_security_level` بۆ FAST دانابەزێنێت — ئاستە ڕێکخراوەکەی بەڕێوەبەر ڕێزی لێدەگیرێت. فلتەرێکی نوێی `wu_cap_server_security_level` بەردەستە بۆ ئەو ماڵپەڕانەی کە لۆژیکی تایبەت دەیانەوێت.
* چاککراوە: ژمێرەری ئاماری `rate_limits_triggered` ئێستا لەسەر هەر بلۆکێک زیاد دەکات، نەک تەنها لەسەر ڕێڕەوی پاشگرەوەی دەگمەنی دوای سەرکەوتن.
* چاککراوە: `Captcha_Core::get_client_ip()` ئێستا تاکە سەرچاوەی ڕاستییە بۆ دیاریکردنی IP ـی سەردانکەر لە سنووردانەری ڕێژە، دابینکەرانی captcha (reCAPTCHA + hCaptcha `siteverify`)، و ئامار — کەلێنێکی ساختەکاری داخرا کە داواکارییە ڕاستەوخۆکانی ڕاژەکاری بنەڕەت کە سەردێڕی ساختەی `CF-Connecting-IP` ـیان هەڵگرتبوو، بە IP ـی ساختەکە دەخرانە سەتڵەوە نەک هاوتای ڕاستەقینە.
* چاککراوە: دەروازەی سنووری ڕێژەی فۆرمی کلاسیکی پارەدانی WooCommerce ئێستا لەسەر `template_redirect` (لە پێشینەیی 1) کاردەکات لەبری `woocommerce_before_checkout_form`. قولاپی ئاستی فۆرم کاتێک سەبەتەکە بەتاڵ بێت هەرگیز کارناکات، بۆیە هاتوچۆی لافاو کە هەرگیز بەرهەمێک زیاد ناکات بە تەواوی سنووردانەرەکەی دەرباز دەکرد.
* چاککراوە: دەروازەی سنووری ڕێژەی pay-for-order ـی WooCommerce ئێستا لەسەر `template_redirect` کاردەکات لەبری `woocommerce_before_pay_action`. ئەوەی دوایی تەنها دوای سەرکەوتنی `wp_verify_nonce('woocommerce-pay')` کاردەکات، کە واتە هێرشبەرانی بێ‌ناسنامە (مۆدێلی هەڕەشەی ڕاستەقینە) هەرگیز سنووردانەرەکەیان کارپێنەدەکرد.
* چاککراوە: دەروازەی سنووری ڕێژەی پارەدانی WooCommerce Store API (blocks) ئێستا لەسەر `rest_pre_dispatch` کاردەکات لەبری `woocommerce_store_api_checkout_update_order_from_request`. ئەوەی دوایی تەنها دوای ئەوە کاردەکات کە Store API سەبەتە و خانەکانی پسووڵە پشتڕاست دەکاتەوە، بۆیە بۆتە بێ‌ناسنامەکان 400 ـیان لە پشتڕاستکەرەوە وەردەگرت و هەرگیز سنووردانەرەکەیان دەستپێنەدەکرد.
* چاککراوە: دەروازەی سنووری ڕێژەی inline-login ـی Ultimate Multisite ئێستا لەسەر `wu_ajax_nopriv_wu_inline_login` بە پێشینەیی 1 کاردەکات (و هاوتای چوونەژوورەوەکراو) لەبری `wu_before_inline_login`. ئەوەی دوایی تەنها دوای سەرکەوتنی `check_ajax_referer('wu_checkout')` کاردەکات، بۆیە بۆتە بێ‌ناسنامەکان بەبێ nonce ـی دروستی wu_checkout، 403 ـیان وەردەگرت و هەرگیز سنووردانەرەکەیان دەستپێنەدەکرد.
* چاککراوە: `Rate_Limiter::enforce()` ئێستا پاسەوانێکی جارێک-بۆ-هەر-داواکاری جێبەجێ دەکات کە بە `surface|ip` کلیلکراوە، بۆیە قولاپە سەرەوەکان کە دووجار لە هەر ڕەندەرێکدا کاردەکەن (بەتایبەتی `wu_setup_checkout` لە Ultimate Multisite) چیتر ئاستی کاریگەری سنووری ڕێژە نیوە ناکەن.
* چاککراوە: دەروازەکانی ڕووکاری سنووری ڕێژە چیتر ڕاوێژ بە `Captcha_Core::is_whitelisted()` ناکەن (فلتەری `wu_captcha_whitelisted`). ئەو فلتەرە نیشان دەدات "captcha پێشتر لەلایەن ڕووکاری ترەوە چارەسەرکراوە" و لە پاراستنی لافاو جیاوازە — یەکخستنی WooCommerce قولاپی بۆ تێپەڕاندنی captcha ـی چوونەژوورەوەی WordPress بەکاردەهێنا کاتێک Woo nonce هەبوو، کە بۆ ژماردنی ڕێژە دەرچوو و وای کرد POST ـەکانی Woo سنووردانەرەکە دەرباز بکەن. فلتەری تایبەتی سنووری ڕێژە `wu_cap_rate_limit_whitelist_ip` تاکە دەربازکردنە کە ئێستا جێبەجێ دەبێت.
