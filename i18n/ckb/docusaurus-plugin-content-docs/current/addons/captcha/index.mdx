# کاپچا

**بۆتەکان ڕابگرە. داهات بپارێزە. بێ سەردەردی ڕێکخستنی سفر.**

تۆڕێکی فرەماڵپەڕ بە دۆمەینی تایبەت بەڕێوە دەبەیت؟ ئەو ئازارە دەزانیت: Google reCAPTCHA و hCaptcha پێویستیان بەوەیە *هەموو تاکە دۆمەینێک* لە کۆنسۆڵی بەڕێوەبردنیاندا تۆمار بکەیت. ماڵپەڕی کڕیارێکی نوێ زیاد دەکەیت؟ ڕێکخستنی کاپچا نوێ بکەرەوە. دۆمەینێکی تایبەت map دەکەیت؟ دووبارە نوێ بکەرەوە. هەرگیز کۆتایی نایەت.

**Ultimate Multisite: کاپچا هەموو شتێک دەگۆڕێت.**

## Cap Captcha: پاراستنی Self-Hosted کە هەر کار دەکات

دابینکەری نوێی **Cap Captcha** ـمان گۆڕانکارییەکی گەورەیە بۆ تۆڕە فرەماڵپەڕەکان:

* **ڕێکخستنی سفر** — addon ـەکە چالاک بکە و پارێزراوی. هیچ API key ـێک، هیچ هەژماری Google ـێک، هیچ dashboard ـی hCaptcha ـیەک، هیچ domain whitelisting ـێک نییە.
* **بە شێوەیەکی خۆکار لەسەر هەموو دۆمەینێک کار دەکات** — ژێردۆمەینەکان، دۆمەینە map کراوەکان، دۆمەینەکانی کڕیار—هەموویان دەستبەجێ پارێزراون.
* **Self-Hosted و تایبەت** — داتاکەت هەرگیز سێرڤەرەکەت بەجێناهێڵێت. بە تەواوی لەگەڵ GDPR گونجاوە و هیچ شوێنکەوتنی لایەنی سێیەم نییە.
* **Proof-of-Work ـی نادیار** — بەکارهێنەران لە پاشبنەماوە کێشەیەکی ژمێریاری چارەسەر دەکەن. هیچ کرتەکردنێک لەسەر هایدرانتی ئاگرکوژێنەوە نییە، هیچ تۆڕی وێنەی بێزارکەر نییە.
* **سووک** — تەنها 20KB. checkout ـەکەت خاو ناکاتەوە.

## پاراستن لە تاقیکردنەوەی کارتی WooCommerce

هێرشەکانی تاقیکردنەوەی کارت ساڵانە ملیارەها زیان بە فرۆشیاران دەگەیەنن. ساختەکاران بۆت بەکاردەهێنن بۆ پشتڕاستکردنەوەی کارتە بانکییە دزراوەکان لە دژی checkout ـی WooCommerce ـەکەت—تۆ کرێی پرۆسێسکردن دەدەیت، ئەوان ژمارە کارتی دروست وەردەگرن.

**ئێمە لەو شوێنەی گرنگترە پاراستن دەکەین:**

* **پاراستنی Store API** — هێرشەکانی بۆت لەسەر `/wp-json/wc/store/v1/checkout` بلۆک دەکات کە کاپچاکانی فۆڕمی نەریتی تێدەپەڕێنن
* **گونجاو لەگەڵ PayPal Payments** — بەبێ کێشە لەگەڵ WooCommerce PayPal Payments کار دەکات
* **ئاماری کاتی ڕاستەقینە** — بە وردی ببینە چەند هێرشێک بلۆک دەکەیت

## هێشتا پێویستت بە Google reCAPTCHA یان hCaptcha هەیە؟

ئێمە پشتیوانیت دەکەین. پشتیوانی تەواو بۆ:

* Google reCAPTCHA v2 (Checkbox)
* Google reCAPTCHA v2 (Invisible)
* Google reCAPTCHA v3 (Score-based)
* hCaptcha (Standard)
* hCaptcha (Invisible)

لە ڕێکخستنەکانی تۆڕەکەتەوە هەر کاتێک بتەوێت لە نێوان دابینکەراندا بگۆڕە.

## بۆ پێوانەی گەورە دروستکراوە

* ڕێکخستنەکانی سەرتاسەری تۆڕ لەگەڵ هەڵوەشاندنەوەی تایبەت بە هەر ماڵپەڕێک
* dashboard ـی ئامار بۆ چاودێریکردنی پاراستن
* ئاستەکانی ئاسایشی ڕێکخراو (خێرا، مامناوەند، زۆرترین)
* گونجاو بۆ پەرەپێدەران بە پێکهاتەی دابینکەری فراوانکراو

## دامەزراندن

1. `ultimate-multisite-captcha` بار بکە بۆ بوخچەی `/wp-content/plugins/` ـەکەت
2. plugin ـەکە لە ئاستی تۆڕدا چالاک بکە
3. **هەر ئەوەندە. تۆ پارێزراوی.**

Cap Captcha بە شێوەیەکی خۆکار چالاک دەبێت—هیچ ڕێکخستنێک نییە بۆ ڕێکخستن، هیچ API key ـێک نییە بۆ داخڵکردن. هەموو endpoint ـێکی پشتگیری کراو لە سەرتاسەری تۆڕەکەتدا لەو چرکەیەوە پارێزراوە کە چالاکی دەکەیت.

**ئارەزوومەندانە:** بۆ بەکارهێنانی Google reCAPTCHA یان hCaptcha لە جیاتی ئەوە، بڕۆ بۆ **Ultimate Multisite → Settings → Captcha** و دابینکەری پەسەندکراوت هەڵبژێرە.

## پرسیارە دووبارەکان

## پێویستم بە ڕێکخستنی هیچ شتێک هەیە؟

نەخێر! Cap Captcha کاتێک addon ـەکە چالاک دەکەیت بە شێوەیەکی خۆکار چالاک دەکرێت. هەموو تۆڕەکەت دەستبەجێ پارێزراوە—هیچ ڕێکخستنێک نییە بۆ ڕێکخستن، هیچ API key ـێک نییە بۆ داخڵکردن.

## پێویستم بە API key بۆ Cap Captcha هەیە؟

نەخێر. Cap Captcha بە تەواوی self-hosted ـە. هیچ هەژماری دەرەکی، هیچ تۆمارکردنی دۆمەین، هیچ سەردەردی ڕێکخستن نییە.

## ئەمە checkout ـەکەم خاو دەکاتەوە؟

Cap Captcha تەنها 20KB ـە و ئاستەنگەکانی proof-of-work لە پاشبنەماوە بەڕێوەدەبات. کڕیارەکانت هیچ دواخستنێک هەست پێناکەن.

## ئایا ئەمە لە هێرشەکانی تاقیکردنەوەی کارت دەپارێزێت؟

بەڵێ! بە پێچەوانەی کاپچاکانی نەریتی کە تەنها فۆڕمەکانی HTML دەپارێزن، یەکخستنی WooCommerce ـەکەمان ڕاستەوخۆ دەبەسترێتەوە بە Store API بۆ بلۆککردنی هێرشەکانی بۆت لە ئاستی API—لەو شوێنەی زۆربەی هێرشەکانی تاقیکردنەوەی کارت ڕوودەدەن.

## دەتوانم دابینکەرانی کاپچای جیاواز لەسەر ماڵپەڕە جیاوازەکان بەکاربهێنم؟

دابینکەری کاپچا لە ئاستی سەرتاسەری تۆڕدا دانراوە، بەڵام دەتوانیت ڕێکخستنەکانی قورسی بەپێی پێویستییەکانت بگۆڕیت.

## ئایا ئەمە لەگەڵ GDPR گونجاوە؟

Cap Captcha 100% self-hosted ـە و هیچ داتایەک بۆ خزمەتگوزارییە دەرەکییەکان نانێردرێت. بۆ Google reCAPTCHA و hCaptcha، تکایە سیاسەتە تایبەتمەندییە پەیوەندیدارەکانیان پێداچوونەوە بکە.

## سنووردانانی ڕێژە

Ultimate Multisite: Captcha v1.5.0 سنووردانەرێکی ڕێژەی hard-stop دەناسێنێت کە تۆڕەکەت لە هێرشەکانی brute-force و card-testing دەپارێزێت.

### چۆن کار دەکات

سنووردانەری ڕێژە هەموو داواکارییەکی GET و POST لەسەر ڕووکارە پارێزراوەکانی کاپچا دەژمێرێت:

* **endpoint ـەکانی چوونەژوورەوەی WordPress** — wp-login، register، lost-password، comments
* **checkout ـی WooCommerce** — my-account، checkout، pay-for-order
* **Ultimate Multisite** — checkout، inline-login

کاتێک سەردانکەرێک سنووری ڕێژە تێدەپەڕێنێت، plugin ـەکە وەڵام دەداتەوە بە:

* کۆدی دۆخی **HTTP 429 (Too Many Requests)**
* header ـی **Retry-After** کە دەستنیشان دەکات کەی دووبارە هەوڵ بدرێتەوە
* **خەوتنی tarpit ـی هەڕەمەکی** (بە بنەڕەت 1–5 چرکە، بە سنووری توند تا 15 چرکە) بۆ خاوکردنەوەی هێرشبەران

### ڕێکخستن

#### کاتی Tarpit

کۆنترۆڵی ئەو دواخستنە هەڕەمەکییە بکە کە بۆ داواکارییە سنووردارکراوەکانی ڕێژە جێبەجێ دەکرێت:

* **`cap_rate_limit_tarpit_min`** — کەمترین ماوەی خەوتن بە چرکە (بنەڕەت: 1)
* **`cap_rate_limit_tarpit_max`** — زۆرترین ماوەی خەوتن بە چرکە (بنەڕەت: 5، بە سنووری توند تا 15)

ئەم ڕێکخستنانە لە **Ultimate Multisite → Settings → Captcha** بەردەستن.

### توانای فراوانکردن

#### فلتەری IP Whitelist

مەوداکانی IP ـی جێی متمانە لە سنووردانانی ڕێژە بەدەر بکە:

```php
add_filter( 'wu_cap_rate_limit_whitelist_ip', function( $is_whitelisted, $ip ) {
    // Whitelist your monitoring service
    if ( $ip === '203.0.113.42' ) {
        return true;
    }
    return $is_whitelisted;
}, 10, 2 );
```

#### کرداری پێش-بلۆک

لۆژیکی تایبەت جێبەجێ بکە دەستبەجێ پێش ئەوەی بلۆکێکی سنووری ڕێژە بنێردرێت:

```php
add_action( 'wu_cap_rate_limit_will_block', function( $ip, $surface ) {
    // Log the block, notify admins, or update custom metrics
    error_log( "Rate limit triggered for $ip on $surface" );
}, 10, 2 );
```

## دۆزینەوەی IP ـی کڕیار و پرۆکسییە متمانەپێکراوەکان

Ultimate Multisite: Captcha v1.5.0 دۆزینەوەی client-IP ـی بەرگری لە spoofing دەناسێنێت. میتۆدی `Captcha_Core::get_client_ip()` ئێستا تاکە سەرچاوەی ڕاستییە بۆ دانانی IP ـی سەردانکەر لە سەرتاسەری:

* کلیلی bucket ـەکانی سنووردانی ڕێژە
* دابینکەری Captcha `remoteip` (reCAPTCHA, hCaptcha `siteverify`)
* hash ـەکانی IP ـی ئامار

### مۆدێلی متمانە

پێوەکراوەکە پلەبەندییەکی توندی متمانەی IP جێبەجێ دەکات:

1. **REMOTE_ADDR** — هەمیشە متمانەپێکراوە (IP ـی هاوتای دەستبەجێ)
2. **CF-Connecting-IP** — تەنها کاتێک ڕێزی لێ دەگیرێت کە:
   - هاوتای دەستبەجێ لە ناو مەودای IP ـی ئێستای Cloudflare دابێت
   - `cap_trust_cloudflare_headers` چالاک کرابێت (بنەڕەت: OFF)
3. **X-Forwarded-For** — تەنها کاتێک ڕێزی لێ دەگیرێت کە:
   - هاوتای دەستبەجێ لە لیستی proxy ـی متمانەپێکراوی ڕێکخراوی بەڕێوەبەریی تۆ دابێت
   - header ـەکە لە ڕاستەوە بۆ چەپ شیکردنەوەی بۆ دەکرێت، بە تێپەڕاندنی hop ـە متمانەپێکراوەکان/Cloudflare

### ڕێکخستن

#### متمانەی Header ـی Cloudflare

متمانە بە header ـی `CF-Connecting-IP` ـی Cloudflare چالاک بکە:

* **`cap_trust_cloudflare_headers`** — دایبنێ بۆ ON بۆ بەشداری کردن لە متمانەی `CF-Connecting-IP`
  - بنەڕەت: OFF (ناچالاک)
  - پێوەکراوەکە لەگەڵ snapshot ـێکی Cloudflare CIDR ـی هاوپێچدا دێت
  - مەوداکانی CIDR هەفتانە لە ڕێگەی wp-cron نوێ دەکرێنەوە
  - ئەگەر نوێکردنەوەکە سەرنەکەوێت، fallback ـی هاوپێچ بەکاردەهێنرێت

#### ڕێکخستنی Proxy ـی متمانەپێکراو

proxy و load-balancer ـەکانی هێڵی پێشەوەت دیاری بکە:

* **`cap_trusted_proxies`** — Textarea ـی CIDR یان IP ـی ڕووت (یەک دانە لە هەر هێڵێکدا)
  - کۆمێنت ڕێگەپێدراوە (ئەو هێڵانەی بە `#` دەست پێ دەکەن)
  - بەبێ ئەم ڕێکخستنە، `X-Forwarded-For` پشتگوێ دەخرێت تەنانەت کاتێک سنووردانەری ڕێژە چالاک بێت
  - نموونە:
    ```
    # Our load balancer
    192.0.2.0/24

    # Backup proxy
    198.51.100.50
    ```

#### دۆزینەوەی خۆکار لە یەکەم چالاککردن

کاتێک بۆ یەکەم جار سنووردانەری ڕێژە چالاک دەکەیت، پێوەکراوەکە دۆخی گونجاوی Cloudflare و proxy ـی تۆ دەدۆزێتەوە و ئاگادارییەکی بەڕێوەبەریی یەک-کرتەیی "ڕێکخستنە دۆزراوەکان جێبەجێ بکە" پیشان دەدات.

* پێوەکراوەکە **هەرگیز لەسەر** بەها پاشەکەوتکراوەکانی تۆ نانووسێتەوە
* ئەگەر هاتوچۆی دواتر پێشنیار بکات کە ڕێکخستنی تۆ چیتر لەگەڵ ڕاستی ناگونجێت (بۆ نموونە، Cloudflare مەوداکانی CIDR ـی گۆڕیوە)، ئاگادارییەکی ناگونجانی نەسڕاوە نوێکردنەوەی پێشنیارکراو دەردەخات

### بۆچی ئەمە گرنگە

مۆدێلی توندی متمانەی IP دەروازەیەکی مەترسیداری ساختەکاری دادەخات: داواکارییە ڕاستەوخۆکانی origin-server کە header ـی `CF-Connecting-IP` ـی ساختە هەڵدەگرن، پێشتر بە IP ـی ساختە bucket دەکران لەبری هاوتای ڕاستەقینە. ئەمە بە تایبەتی گرنگە بۆ:

* **سنووردانی ڕێژە** — هێرشکەران ناتوانن بە ساختەکردنی IP سنوورەکان تێپەڕێنن
* **ئامار** — پێوەرەکانی هێرشی تۆ IP ـی سەردانکەرانی ڕاستەقینە نیشان دەدەن، نە ئەوانەی ساختەکراون
* **پشتڕاستکردنەوەی Captcha** — دابینکەران IP ـی دروستی سەردانکەر بۆ هەڵسەنگاندنی مەترسی وەردەگرن
