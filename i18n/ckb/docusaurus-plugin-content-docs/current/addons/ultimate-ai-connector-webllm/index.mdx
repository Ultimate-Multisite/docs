# Ultimate AI Connector for WebLLM

Ultimate AI Connector for WebLLM توانای دەرئەنجامدانی AIـی ڕاستەوخۆ لە وێبگەڕەوە دەهێنێتە تۆڕی فرەماڵپەڕی WordPress ـەکەت. مۆدێلە گەورەکانی زمان بە تەواوی لەناو وێبگەڕدا بە بەکارهێنانی WebLLM و بزوێنەری MLC کارپێدەکات — بێ API key، بێ بانگەوازی دەرەکی، بێ ئەوەی هیچ داتایەک ئامێری بەکارهێنەر بەجێبهێڵێت.

## تایبەتمەندییە سەرەکییەکان

- **دەرئەنجامدان لە لای وێبگەڕ**: LLM بە خۆجێیی لە وێبگەڕی سەردانکەرەکەدا لە ڕێگەی WebLLM/MLC کاردەکات — پێویستی بە GPU ـی سێرڤەر نییە
- **ویدجێتی چاتی سەرهەڵدەر**: بەکارهێنەرانی چووەژوورەوە دەتوانن ڕاستەوخۆ لە فرۆنت ئێندەوە داوایان لە LLM ـی لای وێبگەڕ بکەن
- **نیشاندەری دۆخی admin-bar**: دۆخی کاتی ڕاستەقینەی بزوێنەری WebLLM لە شریتی بەڕێوەبەری WordPress دا دیارە
- **کاتی جێبەجێکردنی SharedWorker**: چەند تابی وێبگەڕێک یەک دانیشتنی GPU بەهاوبەشی بەکاردەهێنن لەبری ئەوەی لەسەر سەرچاوەکانی GPU ململا‌نێ بکەن
- **apiFetch middleware**: داواکارییەکانی WordPress REST کە لەگەڵ شێوازی AI Client SDK دەگونجێن، بە شێوەیەکی ڕوون و ئاشکرا ئاراستە دەکرێن بۆ WebLLM broker ـی خۆجێیی — بێ گەڕانەوەی HTTP ـی loopback
- **ڕووکاری ڕێکخستنەکانی ویدجێت**: ڕێکخستنەکانی پانێڵی پەیوەستکەر بۆ کاراکردن یان ناچالاککردنی ویدجێتی چات و ڕێکخستنی هەڵسوکەوتی auto-prompt
- **کاشی IndexedDB**: داگرتنی کێشی مۆدێل لە دووبارە ئاراستەکردنەوەکانی CDN ڕزگاری دەبێت کە ڕێڕەوی بنەڕەتی Cache API تێکدەدەن
- **یەکخستنی پاڵێوەری wpai**: خۆی دەبەستێتەوە بە پاڵێوەری `wpai_preferred_text_models` بۆ ئەوەی تایبەتمەندی AI Experiments کاتێک ڕێکخراوە ئاراستەی بزوێنەری وێبگەڕ بکات

## پێداویستییەکان

- WordPress 5.3 یان بەرزتر
- PHP 7.4 یان بەرزتر
- Ultimate Multisite plugin (چالاک)
- وێبگەڕێک بە پشتگیری WebGPU (Chrome 113+، Edge 113+، یان Firefox Nightly کە WebGPU چالاک کرابێت)

## دامەزراندن

1. فایلەکانی addon باربکە بۆ بوخچەی `/wp-content/plugins/` ـەکەت
2. plugin لە ڕێگەی مێنیوی 'Plugins' لە WordPress چالاک بکە
3. بچۆ بۆ **Ultimate Multisite → AI Connector** بۆ ڕێکخستنی addon

## ویدجێتی چاتی سەرهەڵدەر

ویدجێتی چاتی سەرهەڵدەر ڕێگە بە هەر بەکارهێنەرێکی چووەژوورەوە دەدات ڕاستەوخۆ لە فرۆنت ئێندەکەتەوە لەگەڵ LLM ـی لای وێبگەڕ کارلێک بکات، بێ ئەوەی ئەو پەڕەیە بەجێبهێڵێت کە لەسەریەتی.

### چی دەکات

کاتێک چالاک کرا، ئایکۆنی چاتێک لە گۆشەی هەموو پەڕەیەکی فرۆنت ئێند بۆ بەکارهێنەرانی چووەژوورەوە دەردەکەوێت. کرتەکردن لەسەر ئایکۆنەکە پانێڵێکی چات دەکاتەوە کە بەکارهێنەر دەتوانێت prompt بنووسێت و وەڵام لە مۆدێلی WebLLM ـی خۆجێیی کارپێکراو وەربگرێت. چونکە مۆدێلەکە بە تەواوی لەناو وێبگەڕدا کاردەکات، وەڵامەکان تایبەتن و هیچ پرۆسێسکردنێکی لای سێرڤەر تێدا نییە.

### نیشاندەری دۆخی Admin-Bar

شریتی بەڕێوەبەری WordPress نیشاندەرێکی دۆخی تێدایە کە دۆخی ئێستای بزوێنەری WebLLM پیشان دەدات:

| دۆخ | مانا |
|--------|---------|
| **بارکردن** | بزوێنەری MLC دەستپێدەکات یان کێشی مۆدێل دادەگرێت |
| **ئامادە** | مۆدێلەکە بارکراوە و بۆ دەرئەنجامدان بەردەستە |
| **بێکار** | بزوێنەرەکە بارکراوە بەڵام تابی SharedWorker چالاک نییە |
| **هەڵە** | بزوێنەرەکە نەیتوانیوە دەستپێبکات — بۆ وردەکارییەکان سەیری console ـی وێبگەڕ بکە |

نیشاندەرەکە بە کاتی ڕاستەقینە نوێ دەبێتەوە بێ پێویستی بە دووبارە بارکردنەوەی پەڕە.

### چۆن ویدجێتەکە چالاک یان ناچالاک بکەیت

1. لە network admin بچۆ بۆ **Ultimate Multisite → AI Connector**
2. پانێڵی **پەیوەستکەر** بدۆزەوە
3. **چالاککردنی ویدجێتی چاتی سەرهەڵدەر** چالاک یان ناچالاک بکە
4. ڕێکخستنەکان پاشەکەوت بکە

ویدجێتەکە هەروەها دەتوانرێت بۆ هەر ماڵپەڕێک لە بەڕێوەبەری تایبەتی ئەو ماڵپەڕەوە چالاک یان ناچالاک بکرێت، ئەگەر بەڕێوەبەری تۆڕ ئەو توانایەی پێدابێت.

## ڕێکخستنەکانی ویدجێت

پانێڵی پەیوەستکەر لە **Ultimate Multisite → AI Connector** ئەم ڕێکخستنانەی خوارەوەی بۆ ویدجێتی چاتی سەرهەڵدەر تێدایە:

### چالاککردنی ویدجێتی چاتی سەرهەڵدەر

ویدجێتی چات بۆ تەواوی تۆڕ چالاک یان ناچالاک دەکات. کاتێک ناچالاک کرا، ویدجێتەکە لەسەر هیچ پەڕەیەکی فرۆنت ئێنددا دەرناکەوێت، بەبێ گوێدانە ڕۆڵی بەکارهێنەر.

**بنەڕەت:** ناچالاک

### هەڵسوکەوتی Auto-Prompt

کۆنترۆڵ دەکات ئایا ویدجێتی چات کاتێک بەکارهێنەر دەیکاتەوە بە شێوەی خۆکار prompt دەنێرێت یان نا.

| هەڵبژاردە | هەڵسوکەوت |
|--------|-----------|
| **ناچالاک** | ویدجێتەکە بە چاتێکی بەتاڵ دەکرێتەوە — بەکارهێنەر prompt ـی خۆی دەنووسێت |
| **چوارچێوەی پەڕە** | ویدجێتەکە بە promptێکی پێشوەختە پڕکراوە دەکرێتەوە کە پشت بە ناونیشان و ناوەڕۆکی پەڕەی ئێستا دەبەستێت |
| **prompt ـی تایبەت** | ویدجێتەکە بە promptێکی تایبەت دەکرێتەوە کە تۆ لە خانەی خوارەوە دیاری دەکەیت |

کاتێک لەسەر **prompt ـی تایبەت** دانرا، خانەیەکی دەقی زیادە دەردەکەوێت کە دەتوانیت دەقی prompt ـی بنەڕەتی تێدا بنووسیت. پشتگیری گۆڕاوە بنەڕەتییەکانی قاڵب دەکات:

- `{site_name}` — ناوی ماڵپەڕی ئێستا
- `{page_title}` — ناونیشانی پەڕەی ئێستا
- `{user_display_name}` — ناوی پیشاندانی بەکارهێنەری چووەژوورەوە

**بنەڕەت:** ناچالاک

## کاتی جێبەجێکردنی SharedWorker

وەشانی 1.1.0 کاتی جێبەجێکردنی SharedWorker بۆ بزوێنەری MLC دەناسێنێت. پێشتر، هەر تابی وێبگەڕێک کە WebLLM ـی بەکاردەهێنا نموونەی تایبەتی خۆی لە مۆدێلەکە بار دەکرد، کە لەسەر یادگەی GPU ململا‌نێی دەکرد و لەسەر ئامێرەکانی خاوەن VRAM ـی سنووردار کێشەی کارایی دروست دەکرد.

لەگەڵ کاتی جێبەجێکردنی SharedWorker، یەک تاب وەک خانەخوێی بزوێنەر کاردەکات. هەموو تابەکانی دیکە لە ڕێگەی کەناڵی پەیامی worker ـەکەوە لەگەڵ ئەو نموونە تاکەیە پەیوەندی دەکەن. ئەنجامەکە:

- یەک دانیشتنی GPU کە لە نێوان هەموو تابە کراوەکاندا بەهاوبەشە
- وەڵامی خێراتر دوای ئەوەی مۆدێلەکە بار دەکرێت (بێ دەستپێکردنی دووبارە)
- بەکارهێنانی کەمترینی یادگە لە لووتکەدا لەسەر ئامێرەکە

SharedWorker بۆ بەکارهێنەران ڕوون و بێئاگادارکردنەوەیە. نیشاندەری دۆخی admin-bar هەمیشە دۆخی بزوێنەری هاوبەش نیشان دەدات، نە تابی تاکەکەس.

## apiFetch Middleware

addon ـەکە `apiFetch` middlewareێک دادەمەزرێنێت کە داواکارییەکانی WordPress REST API دەگرێتەوە کە لەگەڵ شێوازی AI Client SDK دەگونجێن. لەبری دروستکردنی داواکاری HTTP ـی loopback بۆ سێرڤەر، ئەم داواکارییانە ڕاستەوخۆ ئاراستە دەکرێن بۆ WebLLM broker ـی خۆجێیی کە لە SharedWorker دا کاردەکات.

ئەمە واتە plugin و theme ـەکان کە API ـی ستانداردی WordPress `apiFetch` بەکاردەهێنن بۆ بانگکردنی endpoint ـەکانی AI، کاتێک مۆدێلی لای وێبگەڕ بەردەستە بە شێوەی خۆکار سوودمەند دەبن، بێ پێویستی بە هیچ گۆڕانکارییەک لە کۆد.

## Hook و Filter ـەکان

### Filter ـەکان

- `wpai_preferred_text_models` — تۆماری بکە بزوێنەری وێبگەڕی WebLLM وەک مۆدێلێکی دەقی پێشنیارکراو. ئەم addonە خۆکارانە دەبەسترێتەوە بە ئەم filterە کاتێک بزوێنەرەکە ڕێکخراو و بەردەست بێت.
- `ultimate_webllm_widget_enabled` — دۆخی چالاکبوونی widget بۆ بەکارهێنەرێکی دیاریکراو یان کۆنتێکستێک جێگرەوە بکە. `true` یان `false` بگەڕێنەوە.
- `ultimate_webllm_auto_prompt` — دەقی auto-prompt بگۆڕە پێش ئەوەی بۆ widget بنێردرێت. زنجیرەی prompt و ئۆبجێکتی ئێستای `WP_Post` وەردەگرێت.

## چارەسەرکردنی کێشەکان

### widgetـی چات دەرناکەوێت

- دڵنیا ببەوە بەکارهێنەر چووەتە ژوورەوە — widget تەنها بۆ بەکارهێنەرانی پشتڕاستکراوە پیشان دەدرێت
- بپشکنە کە **Enable floating chat widget** لە پانێڵی Connectorدا کراوەتە سەر
- دڵنیا ببەوە وێبگەڕی بەکارهێنەر پشتگیری WebGPU دەکات (سەیری پێداویستییەکانی سەرەوە بکە)

### نیشاندەری admin-bar "Error" پیشان دەدات

کۆنسۆڵی پەرەپێدەری وێبگەڕ بکەرەوە (F12) و بەدوای هەڵە پەیوەندیدارەکانی WebLLMدا بگەڕێ. هۆکارە باوەکان:

- وێبگەڕ پشتگیری WebGPU ناکات
- داگرتنی کێشی مۆدێل شکستی هێنا — پەیوەندی تۆڕ بپشکنە و هەوڵ بدە کاشی IndexedDB لە ئامرازەکانی پەرەپێدەری وێبگەڕدا پاک بکەیتەوە (Application → IndexedDB)
- زیادکراوێکی وێبگەڕ SharedWorker بلۆک دەکات

### کێشی مۆدێل هەموو جارێک دادەگیرێت

ئەم addonە IndexedDB وەک پاشکەوتی کاش بەکاردەهێنێت بۆ دڵنیابوون لەوەی کێشی مۆدێل دوای CDN redirectەکانیش دەمێنێتەوە. ئەگەر کێشەکان لە هەر سەردانێکدا دووبارە دادەگیرێنەوە، بپشکنە کە IndexedDB لەلایەن ڕێکخستنێکی تایبەتمەندی وێبگەڕ یان زیادکراوێکەوە پاک ناکرێتەوە.
