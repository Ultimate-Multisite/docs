# چاوپێکەوتنی دۆزینەوە

**چاوپێکەوتنی دۆزینەوە** یەکەم هەنگاوە لە پرۆسەی دیزاینی Theme Builder. زانیاری دەربارەی ماڵپەڕەکەت، بازرگانیت، و پەسەندەکانی دیزاین کۆدەکاتەوە بۆ دروستکردنی themeێکی تایبەتمەندکراو.

## پوختە

چاوپێکەوتنی دۆزینەوە لە ڕێگەی زنجیرەیەک پرسیارەوە ڕێنماییت دەکات بۆ تێگەیشتن لە نیشانەی بازرگانیت، بینەری ئامانج، و ئامانجەکانی دیزاین. لە وەشانی 1.16.0، چاوپێکەوتنەکە ئێستا **هەنگاوی بارکردنی وێنە** لەخۆدەگرێت کە ڕێگەت پێدەدات ئیلهامی بینراو بۆ دیزاینەکەت دابین بکەیت.

## هەنگاوەکانی چاوپێکەوتن

### 1. زانیاری بازرگانی

- ناوی ماڵپەڕ و دروشمی کورت
- جۆری بازرگانی یان پیشەسازی
- ئامانجە سەرەکییەکانی بازرگانی
- وەسفی بینەری ئامانج

### 2. پەسەندەکانی دیزاین

- سیستەمی ڕەنگی پەسەندکراو
- شێوازی دیزاین (مۆدێرن، کلاسیک، کەمینەگەر، یاریخوازانە، هتد.)
- ماڵپەڕە ئیلهامبەخشەکان یان نیشانە بازرگانییەکان
- توخمە پێویستەکانی دیزاین

### 3. بارکردنی وێنە (نوێ لە v1.16.0)

یەک وێنە یان زیاتر باربکە بۆ کاریگەری لەسەر ئاراستەی دیزاین:

#### فۆرماتە پەسەندکراوەکان

| فۆرمات | پاشگر | وردەکاری |
|--------|-----------|---------|
| **JPEG** | `.jpg`, `.jpeg` | پێشنیارکراوە بۆ وێنەکان |
| **PNG** | `.png` | پاڵپشتی شەفافیەت دەکات |
| **WebP** | `.webp` | فۆرماتی مۆدێرن، قەبارەی پەڕگەی بچووکتر |
| **GIF** | `.gif` | تەنها وێنەی جێگیر (بێ جووڵانەوە) |

#### پێداویستییەکانی پەڕگە

- **زۆرترین قەبارەی پەڕگە:** 10 MB
- **کەمترین ڕەهەندەکان:** 400 × 300 پیکسڵ
- **ڕەهەندی پێشنیارکراو:** 1200 × 800 پیکسڵ یان گەورەتر
- **ڕێژەی ڕەهەند:** هەر ڕێژەیەک پەسەندە

#### چۆن وێنەکان کاریگەری لەسەر ئاراستەی دیزاین دەکەن

ئەو وێنانەی باریدەکەیت یارمەتی Theme Builder دەدەن بۆ تێگەیشتن لە:

- **پالێتی ڕەنگ** — ڕەنگە زاڵەکان دەرهێنراون و لە themeدا بەکاردەهێنرێن
- **شێوازی بینراو** — مۆدێرن، گوندنشینانە، کەمینەگەر، زیندوو، هتد.
- **کەش و هەست** — پیشەیی، داهێنەرانە، دۆستانە، لوکس، هتد.
- **توخمەکانی دیزاین** — نەخشەکان، تێکستچرەکان، و شێوازەکانی پێکهاتن
- **ئیلهامی تایپۆگرافی** — شێوازەکانی فۆنت کە لەگەڵ themeی بینراو دەگونجێن

### 4. پێکهاتەی ناوەڕۆک

- پەڕە سەرەکییە پێویستەکان (سەرەتا، دەربارە، خزمەتگوزارییەکان، پەیوەندی، هتد.)
- بەشی بلۆگ یان هەواڵ پێویستە
- کارایی بازرگانی ئەلیکترۆنی پێویستە
- تایبەتمەندییە تایبەتەکان یان یەکخستنەکان

### 5. پێداچوونەوە و پشتڕاستکردنەوە

- پوختەی هەموو زانیارییە دابینکراوەکان
- پشتڕاستکردنەوە پێش ئەوەی دروستکردنی دیزاین دەستپێبکات

## باشترین کرداری بارکردنی وێنە

### چی باربکەیت

- **ئیلهامی نیشانە** — لۆگۆکان، نموونە ڕەنگەکان، یان ڕێنماییەکانی نیشانە
- **ماڵپەڕەکانی ڕکابەر** — وێنەی شاشەی ماڵپەڕەکان کە پێت خۆشە
- **مۆدبۆردەکان** — کۆمەڵە وێنەیەک کە جوانیناسی خوازراوت نیشان دەدەن
- **وێنەی بەرهەمەکان** — ئەگەر بەرهەم هەبێت بۆ پیشاندان
- **وێنەی تیم یان ئۆفیس** — بۆ نیشاندانی کەسایەتی یان پیشەیی
- **وێنەی شێوازی ژیان** — نوێنەرایەتی بینەری ئامانجەکەت دەکات

### چی دووربخەیتەوە

- **وێنەی کوالیتی نزم** — وێنەی لێڵ یان پیکسڵکراو
- **وێنەی بێپەیوەند** — وێنەکان کە نوێنەرایەتی نیشانەکەت ناکەن
- **وێنەی پڕ لە دەق** — وێنەی شاشە کە دەقی زۆری تێدایە
- **کەرەستەی مافی چاپکراو** — دڵنیا ببەوە مافی بەکارهێنانی وێنەکانت هەیە
- **GIFی جووڵاو** — تەنها وێنە جێگیرەکان پرۆسێس دەکرێن

### ئامۆژگاری بۆ باشترین ئەنجام

1. **2-3 وێنە باربکە** — چەندین وێنە پێوەندێکی باشتر دابین دەکەن
2. **وێنەی بەرزکوالیتی بەکاربهێنە** — وێنەی ڕوون و باش ڕووناککراو باشترین کار دەکەن
3. **جۆراوجۆری تێبکە** — تێکەڵەی ڕەنگ، شێواز، و بابەت
4. **پێوەند دابین بکە** — بە کورتی وەسف بکە بۆچی هەر وێنەیەکت هەڵبژاردووە
5. **ورد ببە** — ئەگەر توخمێکی دیاریکراوت پێخۆشە، ئاماژەی پێ بکە

## دەرچوونی ئاراستەی دیزاین

دوای چاوپێکەوتنی دۆزینەوە، Theme Builder دروستی دەکات:

- **پالێتی ڕەنگ** — ڕەنگە سەرەکی، لاوەکی، و جەختکراوەکان
- **تایپۆگرافی** — خێزانەکانی فۆنت بۆ سەردێڕەکان و دەقی جەستە
- **شێوازی ڕێکخستن** — بۆشایی، هاوتاکردن، و ڕێکخستنی پێکهاتەکان
- **توخمە بینراوەکان** — ئایکۆنەکان، نەخشەکان، و توخمە جوانکارییەکان
- **مۆدبۆرد** — نوێنەرایەتی بینراوی ئاراستەی دیزاین

## هەنگاوەکانی داهاتوو

دوای تەواوکردنی چاوپێکەوتنی دۆزینەوە:

1. پێداچوونەوە بە ئاراستەی دیزاینی دروستکراو بکە
2. Feedback دابین بکە یان داوای ڕێکخستن بکە
3. بچۆ بۆ هەنگاوی [ئاراستەی دیزاین](./design-direction.md)
4. Theme Builder themeی تایبەتی تۆ دروست دەکات
