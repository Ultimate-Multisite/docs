# ئاراستەی دیزاین

هەنگاوی **ئاراستەی دیزاین** ڕێگەت پێدەدات پێش ئەوەی Theme Builder تەواوی theme دروست بکات، دیزاینی بینراوی themeـەکەت پێداچوونەوە، ورد بکەیتەوە، و کۆتایی پێبهێنیت.

## تێڕوانینی گشتی

دوای تەواوکردنی Discovery Interview، Theme Builder ئاراستەی دیزاینەکەت لەگەڵ **پیشاندانی پێشبینینی desktop و mobile** پیشان دەدات. ئەمە ڕێگەت پێدەدات پێش پابەندبوون بە theme، بە وردی ببینیت دیزاینەکەت لەسەر ئامێرە جیاوازەکان چۆن دەردەکەوێت.

## پێکهاتەکانی ئاراستەی دیزاین

### 1. پالێتی ڕەنگ

سیستەمی ڕەنگی themeـەکەت، بەشێوەیەک کە تێیدا هەیە:

- **ڕەنگی سەرەکی** — ڕەنگی سەرەکی براند کە بۆ دوگمەکان، لینکەکان، و جەختکردنەوەکان بەکاردەهێنرێت
- **ڕەنگی دووەم** — ڕەنگی تەواوکەر بۆ جۆراوجۆری
- **ڕەنگی جەختکردنەوە** — ڕەنگی دیارکردن بۆ توخمە گرنگەکان
- **ڕەنگە بێلایەنەکان** — خۆڵەمێشییەکان و سپییەکان بۆ پاشبنەماکان و دەق
- **ڕەنگی دەق** — ڕەنگی سەرەکی دەق بۆ خوێندنەوەی ئاسان

### 2. تایپۆگرافی

هەڵبژاردنی فۆنت بۆ:

- **فۆنتی سەردێڕ** — بۆ ناونیشانی پەڕەکان و سەردێڕی بەشەکان بەکاردەهێنرێت
- **فۆنتی جەستە** — بۆ دەقی پەرەگراف و ناوەڕۆک بەکاردەهێنرێت
- **قەبارەکانی فۆنت** — قەبارەدانانی وەڵامدەرەوە بۆ قەبارە جیاوازەکانی شاشە
- **بەرزی هێڵ** — بۆشایی نێوان هێڵەکان بۆ خوێندنەوەی ئاسان
- **کێشەکانی فۆنت** — جۆراوجۆرییەکانی ئەستوور، ئاسایی، و سووک

### 3. چیدمان و بۆشایی

- **پانی container** — زۆرترین پانی بۆ ناوەڕۆک
- **Padding و margins** — بۆشایی دەوروبەری توخمەکان
- **سیستەمی تۆڕ** — چیدمانی ستوون بۆ دیزاینی وەڵامدەرەوە
- **بۆشایی پێکهاتەکان** — دووری نێوان توخمەکانی UI

### 4. توخمە بینراوەکان

- **ستایلەکانی دوگمە** — دیزاینەکانی دوگمەی سەرەکی، دووەم، و سێیەم
- **دیزاینەکانی کارت** — چیدمان بۆ بلۆکەکانی ناوەڕۆک
- **ئایکۆنەکان** — ستایلی ئایکۆن و قەبارەدانان
- **سنوورەکان و سێبەرەکان** — قووڵایی بینراوی ناسک
- **کاریگەرییەکانی hover** — ستایلکردنی دۆخی کارلێککار

## پیشاندانی پێشبینی

### پێشبینی desktop

پێشبینی desktop دیزاینەکەت بە پانی تەواو پیشان دەدات (زۆرجار 1200px یان فراوانتر):

- مێنیوی ڕێنیشاندانی تەواو
- چیدمانی تەواو لەگەڵ هەموو ستوونەکان
- تایپۆگرافی و بۆشایی گەورە
- هەموو توخمە بینراوەکان بە قەبارەی تەواو

### پێشبینی mobile

پێشبینی mobile دیزاینەکەت بە پانی mobile پیشان دەدات (زۆرجار 375px):

- ڕێنیشاندانی وەڵامدەرەوە (مێنیوی hamburger)
- چیدمانی یەک-ستوون
- تایپۆگرافی و بۆشایی ڕێکخراو
- قەبارەی دوگمەی گونجاو بۆ دەستدان

### خاڵەکانی گۆڕانی وەڵامدەرەوە

Theme Builder پێشبینییەکان لەم خاڵانەی گۆڕاندا ڕێندەر دەکات:

| ئامێر | پانی | پێشبینی |
|--------|-------|---------|
| **Mobile** | 375px | تەلەفۆنی ستوونی |
| **Tablet** | 768px | tabletـی ئاسۆیی |
| **Desktop** | 1200px | desktopـی پانایی تەواو |
| **Large Desktop** | 1920px | پیشاندانی زۆر-پان |

## وردکردنەوەی دیزاینەکەت

### ئەنجامدانی ڕێکخستنەکان

دەتوانیت ئاراستەی دیزاینەکەت ورد بکەیتەوە بە:

1. **ڕێکخستنی ڕەنگەکان** — هەر ڕەنگێک لە پالێتەکە بگۆڕە
2. **گۆڕینی فۆنتەکان** — شێوەنووسی جیاواز هەڵبژێرە
3. **دەستکاریکردنی بۆشایی** — padding و margins ڕێکبخە
4. **نوێکردنەوەی چیدمان** — پانی container و ستوونەکانی تۆڕ بگۆڕە
5. **تایبەتکردنی توخمەکان** — ستایلەکانی دوگمە، دیزاینەکانی کارت، هتد. دەستکاری بکە

### نوێبوونەوەکانی پێشبینی

گۆڕانکارییەکان بە کاتی ڕاستەقینە ڕەنگ دەدەنەوە:

- پێشبینی desktop دەستبەجێ نوێ دەبێتەوە
- پێشبینی mobile دەستبەجێ نوێ دەبێتەوە
- هەموو خاڵەکانی گۆڕانی وەڵامدەرەوە نوێ دەکرێنەوە
- دەتوانیت لەنێوان پێشبینییەکاندا بگۆڕیت بۆ دڵنیابوون لە گۆڕانکارییەکان

### دیمەنی بەراورد

ئاراستەی دیزاینەکەت بەراورد بکە لەگەڵ:

- **دیزاینی ڕەسەن** — ببینە چی گۆڕاوە
- **دیزاینەکانی ڕکابەر** — لەگەڵ ماڵپەڕەکانی ئیلهام بەراورد بکە
- **وەشانە پێشووترەکان** — بگەڕێوە بۆ ئاراستەکانی دیزاینی پێشووتر

## پەسەندکردنی ئاراستەی دیزاین

کاتێک دڵخۆش بوویت بە دیزاینەکەت:

1. **هەردوو پێشبینییەکە پێداچوونەوە بکە** — دڵنیا ببە desktop و mobile باش دەردەکەون
2. **هەموو ڕەنگەکان بپشکنە** — دڵنیا ببە کۆنترەست و دەستپێگەیشتن هەیە
3. **تایپۆگرافی تاقی بکەوە** — خوێندنەوەی ئاسان لە هەموو قەبارەکاندا بسەلمێنە
4. **چیدمان پشتڕاست بکەوە** — بۆشایی و هاوتەرازی بپشکنە
5. **دیزاین پەسەند بکە** — بڕۆ بۆ دروستکردنی theme

## هەنگاوەکانی دواتر

دوای پەسەندکردنی ئاراستەی دیزاینەکەت:

1. Theme Builder تەواوی themeـەکەت دروست دەکات
2. themeـەکە لەسەر ماڵپەڕی WordPressـەکەت دامەزرێنرێت
3. دەتوانیت بە بەکارهێنانی تایبەتکاریکەری WordPress زیاتر تایبەتی بکەیت
4. بڕۆ بۆ [مێنیوەکانی میوانداری](./hospitality-menus.md) یان تایبەتمەندییەکانی تر

## باشترین شێوازەکان

- **لەسەر ئامێرە ڕاستەقینەکان تاقی بکەوە** — ئەگەر دەکرێت تەلەفۆن و tabletـی ڕاستەقینە بەکاربهێنە
- **خوێندنەوەی ئاسان بپشکنە** — دڵنیا ببە دەق لە هەموو قەبارەکاندا دەخوێندرێتەوە
- **کۆنترەست پشتڕاست بکەوە** — بۆ دەستپێگەیشتن [Validate Palette Contrast](../abilities/validate-palette-contrast.md) بەکاربهێنە
- **توانای کارکردن لەبەرچاو بگرە** — وێنە و فۆنتەکان بۆ خێرایی باش بکە
- **بۆ ناوەڕۆک پلان دابنێ** — دڵنیا ببە چیدمانەکە لەگەڵ ناوەڕۆکی ڕاستەقینەی تۆ کاردەکات

## چارەسەرکردنی کێشەکان

### پێشبینی نوێ نابێتەوە

- پەڕەکە نوێ بکەوە
- کاشی وێبگەڕ پاک بکەوە
- وێبگەڕێکی جیاواز تاقی بکەوە
- پەیوەندی ئینتەرنێت بپشکنە

### ڕەنگەکان جیاواز دەردەکەون

- ڕێکخستنەکانی ڕەنگی مۆنیتەر بپشکنە
- لەسەر ئامێرە جیاوازەکان تاقی بکەوە
- کۆنترەستی ڕەنگ بە ئامرازەکانی دەستپێگەیشتن پشتڕاست بکەوە
- شبیه‌سازییەکانی کوێری ڕەنگ لەبەرچاو بگرە

### کێشەکانی تایپۆگرافی

- دڵنیا ببە پەڕگەکانی فۆنت بار دەبن
- قەبارەی فۆنت لە خاڵە جیاوازەکانی گۆڕان بپشکنە
- بە ناوەڕۆکی ڕاستەقینە تاقی بکەوە
- درێژی هێڵ بۆ خوێندنەوەی ئاسان لەبەرچاو بگرە
