# ئامرازەکان و ڕێکخستن {#tools-and-configuration}

Theme Builder کۆمەڵێک ئامراز بەکاردەهێنێت بۆ شیکردنەوە، دیزاینکردن، و بنیاتنانی ماڵپەڕی WordPress ـەکەت. لە وەشانی 1.16.0 دا، **sd-ai-agent/site-scrape ئێستا ئامرازی Tier 1 ـە**، کە بە شێوەی بنەڕەتی لە Theme Builder دا بەردەستە. Superdav AI Agent v1.18.0 هەروەها پاکێجێکی هاوڕێی Advanced ـی جیاواز دابەشکراو دەناسێنێت بۆ ڕێکاری کاری گەشەپێدەری متمانەپێکراو؛ ئەو ئامرازانە بەشێک نین لە پاکێجی ناوەکی WordPress.org و پێویستە بە جیا دابمەزرێنرێن و ڕێپێدراو بن.

## ئامرازە بەردەستەکان {#available-tools}

### ئامرازەکانی Tier 1 (هەمیشە بەردەست) {#tier-1-tools-always-available}

ئامرازەکانی Tier 1 بە شێوەی بنەڕەتی لە Theme Builder دا بەردەستن بەبێ هیچ ڕێکخستنی زیادە.

#### sd-ai-agent/site-scrape {#sd-ai-agentsite-scrape}

**مەبەست:** شیکردنەوەی ماڵپەڕە هەبووەکان بۆ دەرهێنانی ئیلهامی دیزاین، پێکهاتەی ناوەڕۆک، و کارایی.

**تواناکان:**

- **شیکردنەوەی ماڵپەڕ** — کۆکردنەوە و شیکردنەوەی ماڵپەڕەکانی ڕکابەر یان ئیلهام
- **دەرهێنانی دیزاین** — ناسینەوەی ڕەنگەکان، فۆنتەکان، و شێوازەکانی ڕیزبەندی
- **پێکهاتەی ناوەڕۆک** — تێگەیشتن لە ڕێکخستنی پەڕە و پلەبەندی
- **دۆزینەوەی تایبەتمەندی** — ناسینەوەی plugin ـەکان و کارایی
- **شیکردنەوەی ئەدا** — پشکنینی خێرایی پەڕە و باشکردن
- **شیکردنەوەی SEO** — پێداچوونەوە بە meta tag ـەکان و داتای پێکهاتەدار

**بەکارهێنان:**

```
شیکردنەوەی دیزاینی example.com بکە و وەک ئیلهام بۆ ماڵپەڕەکەم بەکاریبهێنە.
```

**چی دەردەهێنێت:**

- پالتەی ڕەنگ و بەکارهێنانی ڕەنگ
- تایپۆگرافی (فۆنتەکان و قەبارەکان)
- پێکهاتەی ڕیزبەندی و بۆشایی
- شێوازەکانی ڕێنیشاندان
- ڕێکخستنی ناوەڕۆک
- بەکارهێنانی وێنە و میدیا
- توخمە کارلێککارەکان
- وەڵامدانەوەی مۆبایل

**سنوردارییەکان:**

- ناتوانێت ماڵپەڕە پارێزراوەکان بە وشەی نهێنی کۆبکاتەوە
- ڕێز لە robots.txt و سیاسەتەکانی ماڵپەڕ دەگرێت
- لەوانەیە ناوەڕۆکی داینامیک تۆمار نەکات
- ماڵپەڕە زۆر پشتبەستووەکان بە JavaScript لەوانەیە شیکردنەوەی سنورداریان هەبێت
- ماڵپەڕە گەورەکان لەوانەیە کاتی زیاتر بگرن بۆ شیکردنەوە

### ئامرازەکانی Tier 2 (ئارەزوومەندانە) {#tier-2-tools-optional}

ئامرازەکانی Tier 2 کاتێک بەردەستن کە بە ڕوونی لە ڕێکخستنەکانی Theme Builder دا چالاک کرابن.

#### شیکردنەوەی پێشکەوتوو {#advanced-analytics}

پێوانەکانی ئەدای ورد دابین دەکات:

- کاتەکانی بارکردنی پەڕە
- Core Web Vitals
- نمرەی SEO
- نمرەی دەستپێگەیشتن
- نمرەی باشترین ڕاهێنانەکان

#### باشکەری ناوەڕۆک {#content-optimizer}

شیدەکاتەوە و پێشنیاری باشکردن دەکات بۆ:

- خوێندنەوەئاسانی
- باشکردنی SEO
- بەکارهێنانی وشەی سەرەکی
- پێکهاتەی ناوەڕۆک
- شوێنی دانانی بانگەواز بۆ کردار

### ئامرازەکانی Tier 3 (Premium) {#tier-3-tools-premium}

ئامرازەکانی Tier 3 پێویستیان بە ڕێکخستنی زیادە یان تایبەتمەندییەکانی premium هەیە.

#### دروستکەری ناوەڕۆکی AI {#ai-content-generator}

ناوەڕۆک دروست دەکات بۆ:

- وەسفی بەرهەمەکان
- پەڕەکانی خزمەتگوزاری
- بابەتەکانی بلۆگ
- وەسفی meta
- دەقی بانگەواز بۆ کردار

#### ئامرازە پێشکەوتووەکانی دیزاین {#advanced-design-tools}

تواناکانی دیزاینی پێشکەوتوو دابین دەکات:

- دروستکردنی CSS ـی تایبەت
- دروستکردنی ئەنیمەیشن
- دیزاینی توخمی کارلێککار
- تیۆری ڕەنگی پێشکەوتوو
- باشکردنی تایپۆگرافی

### ئامرازە هاوڕێ پێشکەوتووەکان (پاکێجی جیاواز) {#advanced-companion-tools-separate-package}

پاکێجی هاوڕێی Advanced بە جیا لە بڵاوکراوەی ناوەکی Superdav AI Agent دابەش دەکرێت. ئەمە بۆ ژینگەکانی گەشەپێدەر و خاوەنی ماڵپەڕی متمانەپێکراو مەبەستدارە کە بەڕێوەبەران بە ڕوونی ڕێگە بە agent دەدەن ئامرازی کاریگەری بەرز بەکاربهێنێت.

تواناکانی هاوڕێی Advanced دەتوانێت ئەمانە لەخۆبگرێت:

- **ئامرازەکانی سیستەمی پەڕگە** — پشکنین و دەستکاریکردنی پەڕگە ڕێپێدراوەکان لە ژینگە متمانەپێکراوەکان
- **ئامرازەکانی بنکەدراوە** — پرسیارکردن یان نوێکردنەوەی داتای ماڵپەڕ کاتێک بە ڕوونی ڕێپێدراوە
- **ئامرازەکانی WP-CLI** — جێبەجێکردنی فەرمانەکانی چاکسازی و پشکنینی WordPress
- **ئامرازەکانی REST dispatcher** — بانگکردنی endpoint ـە تۆمارکراوەکانی REST لە ڕێگای ڕێکاری کاری ناردنی کۆنترۆڵکراو
- **ئامرازەکانی دروستکەری plugin** — دروستکردنی چوارچێوە و دووبارەکارکردن لەسەر کۆدی plugin
- **ئامرازەکانی وێنەی Git** — دروستکردنی وێنە پێش کردارە مەترسیدارەکانی گەشەپێدەر
- **ئامرازەکانی بەڕێوەبردنی بەکارهێنەر** — یارمەتیدان بۆ پێداچوونەوە یان گۆڕینی تۆمارەکانی بەکارهێنەر لەو شوێنانەی ڕێگەپێدراوە
- **ئامرازەکانی تاقیکردنەوەی پێوانەیی** — جێبەجێکردنی تاقیکردنەوەی ئەدا یان توانا بۆ پێداچوونەوەی گەشەپێدان

ئەمە وەک ئامرازەکانی Theme Builder ـی بەردەست بە گشتی بۆ هەموو دامەزراندنێک بەڵگەنامە مەکە. تەنها کاتێک بەردەستن کە پاکێجی هاوڕێی Advanced دامەزرابێت، چالاک کرابێت، و سنوردار کرابێت بۆ بەڕێوەبەرانی متمانەپێکراو.

## ڕێکخستنی ئامراز {#tool-configuration}

### چالاککردنی ئامرازەکان {#enabling-tools}

بۆ چالاککردنی ئامرازە زیادەکان لە Theme Builder:

1. بچۆ بۆ **Gratis AI Agent → Settings**
2. بچۆ بۆ **Theme Builder → Tools**
3. بە پێویستی ئامرازەکان on/off بکە
4. ڕێکخستنەکان پاشەکەوت بکە

### مۆڵەتەکانی ئامراز {#tool-permissions}

هەندێک ئامراز پێویستیان بە مۆڵەت هەیە:

- **کۆکردنەوەی ماڵپەڕ** — پێویستی بە دەستگەیشتن بە ئینتەرنێت هەیە
- **شیکردنەوە** — پێویستی بە یەکخستنی Google Analytics هەیە
- **دروستکردنی ناوەڕۆک** — پێویستی بە API key ـەکان هەیە
- **تایبەتمەندییە پێشکەوتووەکان** — لەوانەیە پێویستی بە بەشداریکردنی premium هەبێت
- **ئامرازە هاوڕێ پێشکەوتووەکان** — پێویستیان بە پاکێجی Advanced ـی بە جیا دابەشکراو و متمانەی ڕوونی بەڕێوەبەر هەیە

### API Key ـەکان و بڕوانامەکان {#api-keys-and-credentials}

API key ـەکان بۆ ئەو ئامرازانە ڕێکبخە کە پێویستیان پێیانە:

1. بچۆ بۆ **Gratis AI Agent → Settings → API Keys**
2. بڕوانامەکان بۆ هەر ئامرازێک بنووسە
3. پەیوەندی تاقی بکەوە
4. بە پارێزراوی پاشەکەوت بکە

### دامەزراندنی پاکێجی هاوڕێی Advanced {#installing-the-advanced-companion-package}

پاکێجی هاوڕێی Advanced بە جیا لە ZIP ـی ناوەکی WordPress.org بڵاودەکرێتەوە. تەنها لە کەناڵی فەرمی دابەشکردنی پڕۆژە بۆ بڵاوکراوەکە دایبمەزرێنە، پاشان دەستگەیشتن سنوردار بکە بۆ بەڕێوەبەرانی متمانەپێکراو. دەروازەکانی پەسەندکردنی مرۆیی و ڕەفتاری وێنەی git پێداچوونەوە بکە پێش چالاککردنی ئامرازەکانی سیستەمی پەڕگە، بنکەدراوە، WP-CLI، REST dispatcher، دروستکەری plugin، بەڕێوەبردنی بەکارهێنەر، یان تاقیکردنەوەی پێوانەیی لەسەر ماڵپەڕی بەرهەمهێنان.

## بەکارهێنانی sd-ai-agent/site-scrape {#using-sd-ai-agentsite-scrape}

### بەکارهێنانی بنەڕەتی {#basic-usage}

داوا لە Theme Builder بکە ماڵپەڕێک شیکاتەوە:

```
دیزاینی ماڵپەڕی ڕکابەرەکەم لە competitor.com شیکەرەوە
و پێشنیاری باشکردنی دیزاین بۆ ماڵپەڕەکەم بکە.
```

### شیکردنەوەی دیاریکراو {#specific-analysis}

داوای جۆرە دیاریکراوەکانی شیکردنەوە بکە:

```
پالتەی ڕەنگ لە example.com دەربهێنە و وەک ئیلهام بەکاریبهێنە.
```

```
پێکهاتەی ڕێنیشاندانی example.com شیکەرەوە و ڕێکخستنی هاوشێوە
بۆ ماڵپەڕەکەم جێبەجێ بکە.
```

```
وەڵامدانەوەی مۆبایلی example.com بپشکنە و دڵنیابە ماڵپەڕەکەم
بە هەمان شێوە وەڵامدەرەوەیە.
```

### شیکردنەوەی بەراوردکاری {#comparison-analysis}

چەند ماڵپەڕێک بەراورد بکە:

```
دیزاینەکانی site1.com و site2.com بەراورد بکە و دیزاینێک دروست بکە
کە باشترین توخمەکانی هەردووکیان تێکەڵ بکات.
```

## سنوردارییەکانی ئامراز و تێبینییەکان {#tool-limitations-and-considerations}

### سنوردارکردنی ڕێژە {#rate-limiting}

- سکرەیپکردن سنووردارکراوە بە ڕێژە بۆ ڕێگریکردن لە بارگرانیی سێرڤەر
- داواکارییە زۆرەکان بۆ هەمان ماڵپەڕ لەوانەیە خاو بکرێنەوە
- ماڵپەڕە گەورەکان لەوانەیە کاتی زیاتر بخایەنن بۆ شیکردنەوە

### وردیی ناوەڕۆک {#content-accuracy}

- ناوەڕۆکی داینامیک لەوانەیە بە تەواوی نەگیرێت
- ناوەڕۆکی پیشاندراو بە JavaScript لەوانەیە ناتەواو بێت
- داتای کاتی ڕاستەقینە لەوانەیە ڕەنگدانەوەی نەبێت
- هەندێک ناوەڕۆک لەوانەیە لە پشت دیواری پارەداندا بێت

### لەبەرچاوگرتنی یاسایی و ئەخلاقی {#legal-and-ethical-considerations}

- ڕێز لە robots.txt و سیاسەتەکانی ماڵپەڕ بگرە
- ناوەڕۆکی مافی چاپپارێزراو بۆ دووبارە بڵاوکردنەوە سکرەیپ مەکە
- شیکردنەوە بۆ وەرگرتنی ئیلهام بەکاربهێنە، نەک کۆپیکردن
- دڵنیا ببە مافت هەیە ناوەڕۆکی دەرهێنراو بەکاربهێنیت
- پابەند بە مەرجەکانی خزمەتگوزاریی ماڵپەڕە شیکراوەکان ببە

### کاریگەریی ئەدا {#performance-impact}

- سکرەیپکردنی ماڵپەڕە گەورەکان لەوانەیە کات بخایەنێت
- چەند سکرەیپکردنێکی هاوکات لەوانەیە خاوتر بن
- پەیوەندیی تۆڕ کاریگەری لەسەر خێرایی هەیە
- کۆمەڵەداتای گەورە لەوانەیە پڕۆسێسکردنی زیاتر پێویست بکات

## باشترین کردارەکان {#best-practices}

### بەکارهێنانی شیکردنەوەی ماڵپەڕ {#using-site-analysis}

1. **چەندین ماڵپەڕ شیکەوە** — ئیلهام لە چەند سەرچاوەیەک کۆبکەوە
2. **سەرنج بخەرە سەر پێکهاتە** — شێوازەکانی ڕێکخستن و ڕێککەوتن فێربە
3. **ڕەنگەکان دەربهێنە** — پالێتی ڕەنگ وەک خاڵی دەستپێک بەکاربهێنە
4. **تایپۆگرافی بخوێنەوە** — تێکەڵەکانی فۆنت کە پێت خۆشە بناسە
5. **گەڕان پێداچوونەوە بکە** — ڕێکخستنی لیستەکان تێبگە

### سکرەیپکردنی ئەخلاقی {#ethical-scraping}

1. **ڕێز لە robots.txt بگرە** — ڕێنماییەکانی ماڵپەڕ پەیڕەو بکە
2. **ناوەڕۆک کۆپی مەکە** — شیکردنەوە تەنها بۆ ئیلهام بەکاربهێنە
3. **مافەکان بسەلمێنە** — دڵنیا ببە دەتوانیت ناوەڕۆکی دەرهێنراو بەکاربهێنیت
4. **سەرچاوەکان ئاماژە پێبکە** — سەرچاوەکانی ئیلهام ستایش بکە
5. **مەرجەکان پەیڕەو بکە** — پابەند بە مەرجەکانی خزمەتگوزاریی ماڵپەڕ ببە

### زۆرترین کاریگەریی ئامراز {#maximizing-tool-effectiveness}

1. **دیاریکراو بە** — داوای جۆرە دیاریکراوەکانی شیکردنەوە بکە
2. **دەوروبەر دابین بکە** — مەبەستی ماڵپەڕەکەت ڕوون بکەرەوە
3. **چاوەڕوانی دابنێ** — ئامانجەکانی دیزاینت باس بکە
4. **دووبارەکاری بکە** — لەسەر بنەمای ئەنجامەکان وردتر بکەرەوە
5. **ئامرازەکان تێکەڵ بکە** — چەند ئامرازێک بۆ شیکردنەوەی گشتگیر بەکاربهێنە

## چارەسەرکردنی کێشەکان {#troubleshooting}

### ماڵپەڕەکە سکرەیپ نابێت {#site-wont-scrape}

- بپشکنە ئایا ماڵپەڕەکە بە گشتی دەستپێگەیشتووە
- بسەلمێنە robots.txt ڕێگە بە سکرەیپکردن دەدات
- ماڵپەڕێکی جیاواز تاقیبکەرەوە
- پەیوەندیی ئینتەرنێت بپشکنە
- ئەگەر کێشەکە بەردەوام بوو پەیوەندی بە پشتگیرییەوە بکە

### شیکردنەوەی ناتەواو {#incomplete-analysis}

- لەوانەیە ماڵپەڕەکە ناوەڕۆکی داینامیکی هەبێت
- JavaScript لەوانەیە بە تەواوی پیشان نەدرابێت
- ماڵپەڕە گەورەکان لەوانەیە کاتەکەیان تەواو بێت
- هەوڵبدە لەبری ئەوە لاپەڕە دیاریکراوەکان شیکەوە
- داوای جۆرە دیاریکراوەکانی شیکردنەوە بکە

### ئەدای خاو {#slow-performance}

- ماڵپەڕە گەورەکان کاتی زیاتر دەخایەنن بۆ شیکردنەوە
- چەند سکرەیپکردنێکی هاوکات خاوترن
- پەیوەندیی تۆڕ کاریگەری لەسەر خێرایی هەیە
- لە کاتەکانی دەرەوەی لووتکەدا تاقیبکەرەوە
- سەرەتا بەشە بچووکترەکان شیکەوە
