# مێنیوەکانی میوانداری

تایبەتمەندی **مێنیوەکانی میوانداری** ڕێگە بە Theme Builder دەدات پەڕەکانی مێنیوی خواردن و خواردنەوە بە شێوەیەکی ڕێکخراو دروست بکات و ڕاستەوخۆ لە ناو ماڵپەڕی WordPress ـەکەتدا بیخاتەوە.

## تێڕوانینێکی گشتی {#overview}

Theme Builder ئێستا دەتوانێت پەڕەی مێنیوی پیشەیی و ڕێکخراو بۆ بازرگانییەکانی میوانداری دروست بکات، لەوانە چێشتخانە، کافێ، بار، و خزمەتگوزارییەکانی ئامادەکردنی خواردن بۆ بۆنەکان. ئەم مێنیوانە بە تەواوی لەگەڵ دیزاینی ماڵپەڕەکەت یەکدەخرێن و بە ئاسانی دەتوانرێت نوێ بکرێنەوە و بەڕێوەببرێن.

## جۆرە پشتیوانیکراوەکانی میوانداری {#supported-hospitality-types}

- **چێشتخانەکان** — مێنیوی خواردنی خزمەتگوزاری تەواو
- **کافێکان** — مێنیوی قاوە و خواردنی سووک
- **بار و شوێنەکانی دانیشتن** — مێنیوی خواردنەوە و خواردنی سووکی پێش خواردن
- **نانەواییەکان** — مێنیوی شیرینی و نان
- **خزمەتگوزارییەکانی ئامادەکردنی خواردن بۆ بۆنەکان** — هەڵبژاردەکانی مێنیوی بۆنەکان
- **ترکی خواردن** — مێنیوی خزمەتگوزاری خواردنی گەڕۆک
- **کارگەکانی بیرە و شەراب** — مێنیوی خواردنەوە لەگەڵ وەسفەکان

## پێکهاتەی مێنیو {#menu-structure}

### هاوپۆلەکانی مێنیو {#menu-categories}

مێنیوەکان بۆ هاوپۆلەکان ڕێکدەخرێن:

- **خواردنە سووکەکانی پێش خواردن** — دەستپێکەر و پلیتە بچووکەکان
- **خواردنە سەرەکییەکان** — خواردنە سەرەکییەکان
- **خواردنە لاوەکییەکان** — هاوخواردن و سەوزەکان
- **شیرینییەکان** — خواردنە شیرینەکان
- **خواردنەوەکان** — خواردنەوەکان (کحولی و ناکحولی)
- **تایبەتەکان** — تایبەتە ڕۆژانە یان وەرزییەکان

### فۆرماتی بڕگەی مێنیو {#menu-item-format}

هەر بڕگەیەکی مێنیو ئەمانە لەخۆدەگرێت:

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### خانەکانی بڕگە {#item-fields}

| خانە | جۆر | وەسف |
|-------|------|-------------|
| `name` | string | ناوی خواردن یان خواردنەوە |
| `description` | string | وەسفی وردی بڕگەکە |
| `price` | string | نرخ (بە دراو فۆرماتکراوە) |
| `dietary_info` | array | تایبەتمەندییەکانی خۆراک (Vegan، Gluten-Free، هتد.) |
| `allergens` | array | هەستیارییە باوەکان (گوێز، Shellfish، هتد.) |
| `availability` | string | کەی بەردەستە (ڕۆژانە، وەرزی، هتد.) |

## دروستکردنی مێنیوەکانی میوانداری {#creating-hospitality-menus}

### هەنگاوی ١: زانیاری مێنیو دابین بکە {#step-1-provide-menu-information}

بە Theme Builder بڵێ دەربارەی مێنیوەکەت:

```
مێنیوی چێشتخانەیەک بۆ چێشتخانە ئیتاڵییەکەم دروست بکە بە خواردنە سووکەکانی پێش خواردن،
خواردنەکانی پاستا، خواردنە سەرەکییەکان، و شیرینییەکان. نرخ و وەسفەکانیش بخەرە ناوەوە.
```

### هەنگاوی ٢: Theme Builder مێنیو دروست دەکات {#step-2-theme-builder-generates-menu}

Theme Builder:

1. پەڕەیەکی مێنیوی ڕێکخراو دروست دەکات
2. دیزاینی دەکات بۆ ئەوەی لەگەڵ theme ـی ماڵپەڕەکەت بگونجێت
3. بڕگەکان بۆ هاوپۆلەکان ڕێکدەخات
4. نرخ و وەسفەکان فۆرمات دەکات
5. زانیاری خۆراک و هەستیاریکەرەکان زیاد دەکات

### هەنگاوی ٣: پێداچوونەوە و خاسپێکردن {#step-3-review-and-customize}

دەتوانیت:

1. بڕگەکانی مێنیو و نرخەکان دەستکاری بکەیت
2. هاوپۆل زیاد بکەیت یان لاببەیت
3. بڕگەکان لە ناو هاوپۆلەکاندا دووبارە ڕێکبخەیت
4. وەسفەکان و زانیاری خۆراک نوێ بکەیتەوە
5. ستایل و ڕێکخستنی لاپەڕە ڕێکبخەیت

## هەڵبژاردەکانی پیشاندانی مێنیو {#menu-display-options}

### پەڕەی مێنیوی تەواو {#full-menu-page}

پەڕەیەکی تایبەت کە مێنیوی تەواوی تۆ پیشان دەدات:

- بە پێی هاوپۆل ڕێکخراوە
- دەتوانرێت بگەڕێنرێت و پاڵێو بکرێت
- ڕێکخستنی گونجاو بۆ چاپ
- دیزاینی وەڵامدەرەوە بۆ مۆبایل

### ویجێتی مێنیو {#menu-widget}

بەشەکانی مێنیو لە پەڕەکانی تردا بخەرەوە:

- بڕگە دیارەکان لە پەڕەی سەرەکی
- تایبەتە ڕۆژانەکان لە لاتەنیشت
- مێنیوی خواردنەوە لە پەڕەی بار
- پیشاندانی شیرینی لە پێپەڕە

### PDF ـی مێنیو {#menu-pdf}

مێنیوی PDF ـی داگرتن‌پێکراو دروست بکە:

- فۆرماتکردنی پیشەیی
- کوالێتی ئامادە بۆ چاپ
- وێنە و وەسفەکان لەخۆدەگرێت
- ئاسانە بۆ هاوبەشکردن و ئیمەیڵکردن

## زانیاری خۆراک و هەستیاریکەرەکان {#dietary-and-allergen-information}

### تایبەتمەندییەکانی خۆراک {#dietary-attributes}

بڕگەکان بە زانیاری خۆراک نیشانە بکە:

- **Vegan** — هیچ بەرهەمێکی ئاژەڵی نییە
- **Vegetarian** — هیچ گۆشتێکی نییە
- **Gluten-Free** — سەلامەتە بۆ نەخۆشی سیلیاک
- **Dairy-Free** — هیچ بەرهەمی شیرەوە نییە
- **Nut-Free** — هیچ گوێزی دار یان بادەمەکانی نییە
- **Low-Carb** — کاربۆهایدراتی کەمکراوە
- **High-Protein** — پڕ لە پرۆتینە

### ئاگادارکردنەوەکانی هەستیاریکەر {#allergen-warnings}

هەستیاریکەرە باوەکان بخەرە ناوەوە:

- **گوێزەکان** — گوێزی دار و بادەمەکانی
- **Shellfish** — قەستەڵەدەرە دەریاییەکان و نەرمتەنەکان
- **ماسی** — هەموو جۆرەکانی ماسی
- **بەرهەمەکانی شیر** — شیر و بەرهەمەکانی شیر
- **هێلکە** — هێلکەی مریشک
- **سۆیا** — بەرهەمەکانی سۆیا
- **گلوتین** — گەنم و دانەوێڵە گلوتین‌دارەکان
- **کونجی** — تۆوی کونجی و ڕۆنی کونجی

## بەڕێوەبردنی مێنیو {#menu-management}

### نوێکردنەوەی نرخەکان {#updating-prices}

بە ئاسانی نرخەکانی مێنیو نوێ بکەوە:

1. بچۆ بۆ پەڕەی مێنیو
2. کرتە لەسەر "دەستکاریکردنی مێنیو" بکە
3. نرخەکانی بڕگەکان نوێ بکەوە
4. گۆڕانکارییەکان پاشەکەوت بکە
5. گۆڕانکارییەکان دەستبەجێ لەسەر ماڵپەڕەکەت دەردەکەون

### زیادکردنی بڕگە وەرزییەکان {#adding-seasonal-items}

جیاوازییە وەرزییەکانی مێنیو دروست بکە:

1. وەشانی نوێی مێنیو دروست بکە
2. بڕگە وەرزییەکان زیاد بکە
3. بڕگەکان وەک "وەرزی" نیشانە بکە
4. بەروارەکانی بەردەستبوون خشتەبەندی بکە
5. بە شێوەیەکی خۆکار لە کاتی وەرزدا پیشانی بدە

### بەڕێوەبردنی تایبەتەکان {#managing-specials}

تایبەتە ڕۆژانە یان هەفتانەکان پیشان بدە:

1. هاوپۆلی "تایبەتەکان" دروست بکە
2. بڕگەکان لەگەڵ بەروارەکانی بەردەستبوون زیاد بکە
3. تایبەتەکان لە پەڕەی سەرەکی گەش بکەوە
4. ڕۆژانە یان هەفتانە نوێی بکەوە
5. تایبەتە کۆنەکان ئەرشیف بکە

## یەکخستن لەگەڵ Theme Builder {#integration-with-theme-builder}

کاتێک Theme Builder بۆ ماڵپەڕەکانی میوانداری بەکاردەهێنیت:

1. **دۆزینەوەی خۆکاری مێنیو** — دەستنیشان دەکات ئەگەر تۆ بازرگانییەکی خواردن/خواردنەوە بیت
2. **دروستکردنی پەڕەی مێنیو** — پەڕەکانی مێنیوی پیشەیی دروست دەکات
3. **گونجاندنی دیزاین** — مێنیوەکان لەگەڵ theme ـی ماڵپەڕەکەت دەگونجێن
4. **باشکردنی مۆبایل** — مێنیوەکان بە جوانی لەسەر تەلەفۆنەکان پیشان دەدرێن
5. **باشکردنی SEO** — مێنیوەکان دۆستانەی بزوێنەری گەڕانن

## باشترین ڕێکارەکان {#best-practices}

### دیزاینی مێنیو {#menu-design}

- **ڕێکخستنی ڕوون** — پێکهاتەی هاوپۆلی لۆژیکی
- **وەسفە خوێندنەوەئاسانەکان** — تام‌هێنەر و زانیاری‌دار
- **نرخدانانی یەکگرتوو** — دراو و فۆرماتکردنی ڕوون
- **وێنە پیشەییەکان** — وێنەی خواردنی کوالێتی بەرز
- **بۆشایی سپی** — پەڕەکە زۆر قەرەباڵغ مەکە

### ناوەڕۆک {#content}

- **وەسفی ورد و دروست** — بڕگەکان بە وردی و ڕاستی وەسف بکە
- **گەشکردنەوەی تایبەتەکان** — تایبەتەکان وا بکە دیار بن
- **خستنە ناوەوەی هەستیاریکەرەکان** — هەمیشە هەستیاریکەرەکان لیست بکە
- **نوێکردنەوەی بەردەوام** — نرخ و بڕگەکان نوێ بهێڵەوە
- **بەکارهێنانی زمانی تام‌هێنەر** — وا بکە بڕگەکان بە تام دیار بن

### دەستپێگەیشتن {#accessibility}

- **فۆنتە خوێندنەوەئاسانەکان** — جۆرەنووسی ڕوون و خوێندنەوەئاسان بەکاربهێنە
- **کۆنتراستی پێویست** — دڵنیابە دەق خوێندنەوەئاسانە
- **ناونیشانەکانی خۆراک** — هەڵبژاردە خۆراکییەکان بە ڕوونی نیشانە بکە
- **ئاگادارکردنەوەکانی هەستیاریکەر** — هەستیاریکەرەکان بە شێوەیەکی بەرچاو پیشان بدە
- **دۆستانەی مۆبایل** — لەسەر هەموو ئامێرەکان تاقی بکەوە
