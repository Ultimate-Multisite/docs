# Superdav AI Agent

Superdav AI Agent یاریدەدەری پێشکەوتووی AI ـی WordPress ـە کە Gratis AI Agent فراوان دەکات بە تواناکانی ئاگادار لە theme، تواناکانی گەڕانی ئینتەرنێتی باشترکراو، تایبەتمەندییەکانی دروستکەری ماڵپەڕ، و یەکخستنەکانی خۆکارکردنی هەڵبژاردەیی. وەشانی 1.18.0 ئامرازەکانی خوێندنەوەی Google Calendar، هاوبەستکردنی پەیوەندی و بەشداربوو، دەروازەکانی پەسەندکردنی مرۆیی، لابردنی دووبارەبوونەوەی بیرخەرەوە، ئاگادارکردنەوەکانی TextBee SMS، دابینکردنی خزمەتگوزاری بەڕێوەبراوی Superdav، و پاکێجی هاوبەشی Advanced کە بەجیا بڵاو دەکرێتەوە بۆ ڕێکاری کارەکانی گەشەپێدەری متمانەپێکراو زیاد دەکات.

## تایبەتمەندییە سەرەکییەکان {#key-features}

- **تواناکانی ئاگادار لە theme** — بە شێوەیەکی خۆکار ڕێنمایی و تواناکان لەسەر بنەمای theme ـی چالاکی WordPress و plugin ـە دامەزراوەکانت دەگونجێنێت
- **چەند دابینکەری گەڕانی ئینتەرنێت** — لەنێوان Brave Search و Tavily هەڵبژێرە بۆ ئەنجامەکانی گەڕانی دەوڵەمەندتر و تواناکانی توێژینەوەی پێشکەوتوو
- **فۆڕمەکانی پەیوەندی دروستکەری ماڵپەڕ** — فۆڕمەکانی پەیوەندی بە تەواوی کارا ڕاستەوخۆ لە ڕووکاری چاتەوە بە بەکارهێنانی plugin ـی فۆڕمی دامەزراوت دروست بکە
- **دروستکەری ماڵپەڕی باشترکراو** — پلانی دروستکردنی ماڵپەڕی چەند قۆناغی باشترکراو لەگەڵ دۆزینەوەی plugin ـی باشتر و چاککردنەوەی هەڵە
- **تواناکانی theme ـی ناوخۆیی** — ڕێنمایی تایبەتمەند بۆ Block Themes، Classic Themes، Kadence Blocks، و Kadence Theme
- **یەکخستنی WooCommerce ـی باشترکراو** — دووبارە داڕێژراوەتەوە بۆ بەکارهێنانی API ـە ڕەسەنەکانی WooCommerce بۆ پشتپێبەستنی باشتر و گونجانی زیاتر
- **دۆزینەوەی خۆکاری دابینکەر** — بە شێوەیەکی خۆکار plugin و theme ـە بەردەستەکان دەدۆزێتەوە و نوێیان دەکاتەوە کاتێک چالاک یان ناچالاک دەکرێن
- **دروستکردنی لۆگۆی دەستکرد** — لۆگۆکانی SVG ـی دەستکرد دروست بکە و جێگیری بکە لەگەڵ پاککردنەوەی خۆکاری پارێزراو لە namespace
- **پشتڕاستکردنەوەی کۆنتراستی palette** — جووتە ڕەنگەکان بۆ گونجانی دەستگەیشتنی WCAG پێش جێبەجێکردن لە theme ـەکەت بپشکنە
- **دۆزینەوەی باشترکراو بە وێنە** — لە کاتی چاوپێکەوتنی دۆزینەوەی Theme Builder وێنە باربکە بۆ کاریگەریکردن لە ئاراستەی دیزاین
- **مێنیوی میوانداری** — پەڕەکانی مێنیوی خواردن و خواردنەوەی پێکهاتەدار بۆ چێشتخانە، کافێ، و بازرگانییەکانی میوانداری دروست بکە
- **پێشبینینی desktop و mobile** — لە کاتی هەڵبژاردنی ئاراستەی دیزاین، دیزاینەکەت لەسەر ئامێرەکانی desktop و mobile پێشبینین بکە
- **ڕووپێوکردنی ماڵپەڕی Tier 1** — sd-ai-agent/site-scrape ئێستا بە شێوەی بنەڕەتی بەردەستە بۆ شیکردنەوەی ماڵپەڕەکانی ڕکابەر و ئیلهام
- **ئامرازەکانی خوێندنەوەی Google Calendar** — ساڵنامە و ڕووداوە ڕێکخراوەکان بخوێنەوە بۆ ئەوەی خۆکارکردنەکان بتوانن دەربارەی دەقی خشتەی داهاتوو بیر بکەنەوە
- **هاوبەستکردنی پەیوەندی و بەشداربوو** — بەشداربووانی ساڵنامە لەگەڵ بەکارهێنەر و پەیوەندییە ناسراوەکانی ماڵپەڕ هاوتا بکە پێش دروستکردنی کاری بەدواداچوون
- **دەروازەکانی پەسەندکردنی مرۆیی** — خۆکارکردنە هەستیارەکان بۆ پێداچوونەوە ڕابگرە پێش ناردنی پەیام، دەستپێکردنەوەی کار، یان ئەنجامدانی گۆڕانکاری
- **تۆمارەکانی بیرخەرەوە و لابردنی دووبارەبوونەوە** — ئەو بیرخەرەوانە بدۆزەوە کە پێشتر نێردراون بۆ ئەوەی جێبەجێکردنی دووبارەی خۆکارکردن هەمان وەرگر دوو جار ئاگادار نەکاتەوە
- **ئاگادارکردنەوەکانی TextBee SMS** — کاتێک دابینکەری SMS چالاک کراوە، ئاگادارکردنەوەکانی پەیامی دەقی ڕێکخراو لە ڕێگەی TextBee بنێرە
- **دابینکردنی خزمەتگوزاری بەڕێوەبراوی Superdav** — ماڵپەڕە پشتگیریکراوەکان بە خاڵەکانی کۆتایی میوانداری‌کراوی Superdav ببەستەوە بە ڕێکخستنی خۆکار لەبری ڕێکخستنی دەستیی خاڵی کۆتایی
- **پاکێجی هاوبەشی Advanced** — ئامرازەکانی گەشەپێدەری متمانەپێکراو کە بە جیا بڵاو دەکرێنەوە بۆ ڕێکاری کارەکانی سیستەمی فایل، بنکەدراوە، WP-CLI، دابەشکەری REST، دروستکەری plugin، وێنەگرتنی git، بەڕێوەبردنی بەکارهێنەر، و پێوانەکردنی کارایی

## پێداویستییەکان {#requirements}

- WordPress 6.0 یان بەرزتر
- PHP 8.1 یان بەرزتر
- WordPress Multisite (network-activated) یان دامەزراندنی تاکە-ماڵپەڕ
- plugin ـی پەیوەستکەری دابینکەری AI ـی چالاک (Anthropic، OpenAI، یان گونجاو)
- Gratis AI Agent v1.9.0 یان بەرزتر (Superdav AI Agent، Gratis AI Agent فراوان دەکات)

## دامەزراندن {#installation}

1. فایلەکانی plugin باربکە بۆ `/wp-content/plugins/superdav-ai-agent`
2. plugin لە ڕێگەی پەردەی **Plugins** لە WordPress چالاک بکە
3. دڵنیابە Gratis AI Agent ـیش چالاک کراوە
4. بچۆ بۆ **Gratis AI Agent → Settings** و دابینکەری گەڕانی ئینتەرنێت، پەیوەندی خزمەتگوزاری بەڕێوەبراوی Superdav، Google Calendar، یان زانیارییەکانی TextBee ـت بە پێویستی ڕێکبخە

پاکێجی WordPress.org تایبەتمەندییە سەرەکییەکانی Superdav AI Agent لەخۆدەگرێت. پاکێجی هاوبەشی Advanced بەجیا دروست و بڵاو دەکرێتەوە بۆ ئەو ژینگانەی کە ئامرازەکانی گەشەپێدەری متمانەپێکراو بە ڕوونی دامەزراون و ڕێگەپێدراون.

## دەستپێکی خێرا {#quick-start}

کاتێک چالاک کرا، Superdav AI Agent بە شێوەیەکی خۆکار دامەزراندنی Gratis AI Agent ـەکەت بە تواناکانی ئاگادار لە theme و توانای زیاتر باشتر دەکات. پانێڵی **Gratis AI Agent** لە شریتی لایەنی بەڕێوەبەری WordPress بکەرەوە و دەست بکە بە بەکارهێنانی تایبەتمەندییە نوێیەکان.

**ڕێنماییە نموونەییەکان:**

```
فۆڕمی پەیوەندی زیاد بکە بۆ پەڕەی پەیوەندی.
```

```
لە وێبدا بگەڕێ بۆ نوێترین باشترین کردەوەکانی ئاسایشی WordPress.
```

```
بەشی hero دروست بکە بە بەکارهێنانی نەخشەکانی بلۆکی theme ـی چالاک.
```

```
پێداچوونەوە بکە بە ڕووداوە ڕێکخراوەکانی Google Calendar ـی سبەی و ڕەشنووسی بیرخەرەوەکان بۆ بەشداربووان ئامادە بکە.
```

```
دوای پەسەندکردنی پەیامەکە لەلایەن منەوە، بیرخەرەوەی SMS بنێرە.
```

## تواناکانی ئاگادار لە Theme {#theme-aware-skills}

Superdav AI Agent بە شێوەیەکی خۆکار theme ـی چالاکت و plugin ـە دامەزراوەکان دەدۆزێتەوە، توانای گونجاوی ئاگادار لە theme دادەگرێت بۆ دەقی یاریدەدەرەکە. تواناکانی بەردەست بریتین لە:

| توانا | جێبەجێ دەبێت بۆ |
|---|---|
| **Block Themes** | Theme ـەکانی بەکارهێنەری پێکهاتەی WordPress ـی بنیاتنراو لەسەر بلۆک (دەستکاریکردنی تەواوی ماڵپەڕ) |
| **Classic Themes** | Theme ـە نەریتییەکانی WordPress کە قالبەکانی PHP و دەستکاریکەری کلاسیکی بەکاردەهێنن |
| **Kadence Blocks** | ماڵپەڕەکان کە plugin ـی Kadence Blocks ـیان چالاکە |
| **Kadence Theme** | ماڵپەڕەکان کە theme ـی Kadence وەک theme ـی چالاکیان هەیە |

بڕوانە [تواناکانی ئاگادار لە Theme](../../user-guide/configuration/theme-aware-skills) بۆ ڕێنمایی ورد دەربارەی هەر توانایەک.

## دابینکەرانی گەڕانی ئینتەرنێت {#internet-search-providers}

Superdav AI Agent v1.10.0، **Tavily** وەک جێگرەوەیەک بۆ Brave Search بۆ پرسیارەکانی گەڕانی ئینتەرنێت زیاد دەکات. هەردوو دابینکەرەکە ئەنجامی گەڕانی کاتی ڕاستەقینە پێشکەش دەکەن و دەتوانرێن لە **Gratis AI Agent → Settings → Advanced** ڕێکبخرێن.

| دابینکەر | تایبەتمەندییەکان | ڕێکخستن |
|---|---|---|
| **Brave Search** | ئەنجامەکانی گەڕانی خێرا و سەرنج‌لەسەر-پاراستنی نهێنی | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | ئەنجامەکانی گەڕانی دەوڵەمەندتر، تواناکانی توێژینەوەی پێشکەوتوو | [Tavily API](https://tavily.com/) |

بۆ ڕێنماییەکانی ڕێکخستن سەیری [گەڕانی ئینتەرنێت](../../user-guide/configuration/internet-search) بکە.

## کارڕێژەکانی ڕۆژمێر، پەسەندکردن، و ئاگادارکردنەوە {#calendar-approval-and-notification-workflows}

Superdav AI Agent v1.18.0 کارڕێژەکانی ئۆتۆماتیککردن فراوان دەکات بۆ دەرەوەی ئەرکەکانی ناوەڕۆک و دروستکردنی ماڵپەڕ:

- **ئامرازەکانی Google Calendar** ڕێگا بە agent دەدەن ڕۆژمێرە ڕێکخراوەکان و وردەکارییەکانی ڕووداو بخوێنێتەوە کاتێک ناسنامەکانی ڕۆژمێر بەردەستن.
- **یارمەتیدەرەکانی پەیوەندی و ئامادەبوو** بەشداربووانی ڕووداو دەخەنەسەر بەکارهێنەرانی WordPress یان پەیوەندییە ناسراوەکان پێش ئامادەکردنی بیرخستنەوەکان یان ئەرکەکانی بەدواداچوون.
- **دەروازەکانی پەسەندکردنی مرۆیی** ڕێگا بە ئۆتۆماتیککردنێک دەدەن بوەستێت تاوەکو بەکارهێنەرێکی ڕێپێدراو کرداری پێشنیارکراو پێداچوونەوەی بۆ بکات.
- **تۆمارەکانی بیرخستنەوە** ڕێ لە دووبارەبوونەوەی بیرخستنەوەکانی ڕۆژمێر یان ئاگادارکردنەوە دەگرن کاتێک ئۆتۆماتیککردنەکە دووبارە هەوڵی بۆ دەدرێت یان بە شێوەیەکی دووبارە خشتە دەکرێت.
- **پشتیوانی TextBee SMS** ئاگادارکردنەوەی پەیامی دەقی چالاک دەکات کاتێک دابینکەری TextBee ڕێکخراوە.

ئەم ئامرازانە بەکاربهێنە بۆ یارمەتیی خشتە-ئاگا، بەدواداچوونی ئامادەبووان، و کارڕێژەکانی بیرخستنەوە کە هێشتا پێداچوونەوەی مرۆییان پێویستە پێش ئەوەی پەیامە هەستیارەکان بنێردرێن.

## فۆڕمەکانی پەیوەندی Site Builder {#site-builder-contact-forms}

agentی Site Builder ئێستا دەتوانێت فۆڕمەکانی پەیوەندی ڕاستەوخۆ لە ڕووکاری چاتەوە دروست بکات. بە سادەیی داوا لە Site Builder بکە فۆڕمی پەیوەندییەک دروست بکات، و ئەو بە شێوەیەکی ئۆتۆماتیکی pluginی فۆڕمی دامەزراوی تۆ دەدۆزێتەوە (Contact Form 7, WPForms, Fluent Forms, یان Gravity Forms) و فۆڕمێکی ئامادە-بۆ-بەکارهێنان دروست دەکات.

بۆ ڕێنمایی ورد سەیری [فۆڕمەکانی پەیوەندی Site Builder](../../user-guide/administration/site-builder-contact-forms) بکە.

بۆ ڕێنمایی ورد سەیری [مێنیوەکانی میوانداری](./theme-builder/hospitality-menus.md) بکە.

## ڕێکخستن {#configuration}

هەموو ڕێکخستنەکان لە ژێر **Gratis AI Agent → Settings** لە بەڕێوەبەری WordPress دان. Superdav AI Agent ئەم هەڵبژاردنەکانی ڕێکخستن زیاد دەکات:

| ڕێکخستن | وەسف |
|---|---|
| **دابینکەری گەڕانی ئینتەرنێت** | لە نێوان Brave Search و Tavily هەڵبژێرە (ئارەزوومەندانە) |
| **Tavily API Key** | کلیلی API بۆ گەڕانی ئینتەرنێتی Tavily (ئەگەر Tavily هەڵبژێردراوە) |
| **خزمەتگوزاری بەڕێوەبراوی Superdav** | ماڵپەڕە پشتیوانیکراوەکان بە خزمەتگوزاری میوانداری‌کراوی Superdavەوە پەیوەست بکە لەگەڵ دابینکردنی endpointی ئۆتۆماتیکی |
| **ناسنامەکانی Google Calendar** | دەستگەیشتنی ڕۆژمێر ڕێکبخە بۆ ئەوەی agent بتوانێت ڕۆژمێر و ڕووداوەکان بخوێنێتەوە |
| **دابینکەری TextBee SMS** | ناسنامەکانی TextBee ڕێکبخە بۆ کارڕێژە پەسەندکراوەکانی ئاگادارکردنەوەی SMS |

## پاکێجی هاوڕێی پێشکەوتوو {#advanced-companion-package}

Superdav AI Agent v1.18.0 پاکێجێکی هاوڕێی Advanced بە شێوەی جیا دابەشکراو زیاد دەکات بۆ کارڕێژەکانی گەشەپێدەری متمانەپێکراو. ئەم توانایانە لە پاکێجی سەرەکی WordPress.orgدا نەخرێنەوە. پاکێجی هاوڕێ تەنها لەو ماڵپەڕانە دابمەزرێنە کە بەڕێوەبەران بە ڕوونی متمانە بە agent دەکەن بۆ دەستگەیشتن بە ئامرازەکانی گەشەپێدەر.

ئامرازە پێشکەوتووەکان کرداری سیستەمی فایل، دەستگەیشتن بە بنکەدراوە، WP-CLI، کارڕێژەکانی REST dispatcher، سوودمەندییەکانی دروستکەری plugin، وێنەگرتنی git، یارمەتیدەرەکانی بەڕێوەبردنی بەکارهێنەر، و ئامرازی پێوانەکردنی خێرایی لەخۆدەگرن. ئەم ئامرازانە سنووردار بکە بە بەڕێوەبەرە متمانەپێکراوەکان و پێداچوونەوە بە ڕێکخستنی دەروازەی پەسەندکردن بکە پێش چالاککردنیان لەسەر ماڵپەڕەکانی بەرهەمهێنان.
