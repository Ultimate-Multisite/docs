# دروستکردنی مێنیو {#create-menu}

توانای **دروستکردنی مێنیو** مێنیوەکانی ڕێنیشاندانی WordPress دروست دەکات بە پشتگیری بۆ ناوی ڕێنیشاندانی جیاواز لە ناونیشانی پەڕەکان.

## تێڕوانینێکی گشتی {#overview}

ئەم توانایە کارایی ستانداردی دروستکردنی مێنیو فراوان دەکات بە توانای دیاریکردنی پارامیتەری `navigation_label`. ئەمە ڕێگەت پێدەدات مێنیو دروست بکەیت کە ناوی پیشاندراو لە ڕێنیشانداندا جیاواز بێت لە ناونیشانی پەڕە، و نەرمونیانی زیاتر لە پێکهاتەی ماڵپەڕ و ئەزموونی بەکارهێنەر دابین دەکات.

## پارامیتەرەکان {#parameters}

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | string | بەڵێ | ناوی مێنیو، وەک `Primary Navigation` |
| `location` | string | نەخێر | شوێنی ڕووکار بۆ تەرخانکردنی ئەم مێنیوە بۆی، وەک `primary` |
| `navigation_label` | string | نەخێر | ئەو ناوەی لە ڕێنیشانداندا پیشان بدرێت (جیاواز لە ناونیشانی پەڕە) |

## بەهای گەڕاوە {#return-value}

```json
{
  "success": true,
  "menu_id": 7,
  "name": "Primary Navigation",
  "navigation_label": "Main Menu"
}
```

## ناوی ڕێنیشاندان بەرامبەر ناونیشانی پەڕە {#navigation-label-vs-page-title}

پارامیتەری `navigation_label` ڕێگەت پێدەدات ناوی ناوخۆیی مێنیو لەو ناوەی بەکارهێنەران دەیبینن جیا بکەیتەوە:

- **`name`** — ناسێنەری ناوخۆیی مێنیو کە لەلایەن WordPress بەکاردەهێنرێت (وەک، "Primary Navigation")
- **`navigation_label`** — ئەو ناوەی بۆ سەردانیکەرانی ماڵپەڕ لە ڕێنیشانداندا پیشان دەدرێت (وەک، "Main Menu")

ئەمە بەسوودە کاتێک:
- شێوازی ناونانی ناوخۆییت جیاوازە لە ناوەکانی ڕووبەڕووی بەکارهێنەر
- دەتەوێت ناوەکان لە ڕێنیشانداندا کورتتر بن لە پانێڵی بەڕێوەبردن
- پێویستت بە پشتگیری چەند زمانێکە بە درێژی ناوی جیاواز
- مێنیو بۆ ناوچە دیاریکراوەکان یان گرووپی بەکارهێنەر دروست دەکەیت

## نموونەکانی بەکارهێنان {#usage-examples}

### نموونەی ١: مێنیوی سادە بە ناوی ڕێنیشاندان {#example-1-simple-menu-with-navigation-label}

**داواکاری:**
```
مێنیوی ڕێنیشاندانی سەرەکییەک دروست بکە بە ناوی "Main Navigation" و بە ناوی ڕێنیشاندانی "Menu".
```

**ئەنجام:**
```json
{
  "success": true,
  "menu_id": 7,
  "name": "Main Navigation",
  "navigation_label": "Menu"
}
```

### نموونەی ٢: مێنیو بۆ شوێنی ڕووکارێکی دیاریکراو {#example-2-menu-for-specific-theme-location}

**داواکاری:**
```
مێنیوی پێپەڕەیەک دروست بکە بە ناوی "Footer Links" بە ناوی ڕێنیشاندانی "Quick Links" و تەرخانی بکە بۆ شوێنی پێپەڕە.
```

**ئەنجام:**
```json
{
  "success": true,
  "menu_id": 8,
  "name": "Footer Links",
  "navigation_label": "Quick Links",
  "location": "footer"
}
```

## یەکگرتن لەگەڵ Theme Builder {#integration-with-theme-builder}

کاتێک Theme Builder بەکاردەهێنیت، توانای دروستکردنی مێنیو:

1. شوێنە بەردەستەکانی مێنیوی ڕووکار بە شێوەیەکی خۆکار دەدۆزێتەوە
2. مێنیو بە ناوی ڕێنیشاندانی گونجاو بۆ دیزاینەکەت دروست دەکات
3. مێنیوەکان تەرخان دەکات بۆ شوێنە دروستەکانی ڕووکار
4. پشتگیری زیادکردنی بڕگەی مێنیو دەکات دوای دروستکردن

## تواناکانی پەیوەندیدار {#related-abilities}

- **`add_menu_item`** — بڕگەکان زیاد بکە بۆ مێنیویەکی هەبوو
- **`update_menu`** — ناوی مێنیو بگۆڕە یان دووبارە تەرخانی بکە بۆ شوێنی ڕووکار
- **`delete_menu`** — مێنیوێک لە ماڵپەڕەکەت بسڕەوە

## باشترین کردەوەکان {#best-practices}

- **ناوی ڕێنیشاندانی ڕوون بەکاربهێنە** — ناوەکان بۆ بەکارهێنەران کورت و وەسفکار بهێڵەوە
- **لەگەڵ شوێنەکانی ڕووکار بگونجێنە** — مێنیوەکان تەرخان بکە بۆ شوێنی دروستی ڕووکار بۆ پیشاندانی گونجاو
- **پێکهاتەی مێنیو پلان بکە** — پێش دروستکردنی بڕگەکان، پلەبەندی مێنیوەکەت دیاری بکە
- **وەڵامدانەوە تاقی بکەرەوە** — دڵنیا ببە مێنیوەکان لەسەر ئامێرە مۆبایلەکان بە دروستی پیشان دەدرێن
- **ناوەکان خۆماڵی بکە** — ناوی ڕێنیشاندانی جیاواز بۆ وەشانی زمانی جیاواز بەکاربهێنە

## سنووردارێتییەکان {#limitations}

- ناوەکانی ڕێنیشاندان تەنها بۆ پیشاندانن؛ ناوی ناوخۆیی `name` هێشتا بۆ ناساندنی WordPress بەکاردەهێنرێت
- پشتگیری ڕووکار جیاوازە؛ هەموو ڕووکارەکان پشتگیری هەموو شوێنەکانی مێنیو ناکەن
- بڕگەکانی مێنیو دەبێت بە جیا دوای دروستکردنی مێنیو زیاد بکرێن
- گۆڕینی ناوی ڕێنیشاندان پێویستی بە نوێکردنەوەی مێنیوەکە هەیە
