# دروستکردنی لۆگۆی SVG

توانای **دروستکردنی لۆگۆی SVG** ڕێگە بە Theme Builder دەدات لۆگۆی SVGی تایبەتمەند دروست بکات و ڕاستەوخۆ لە ماڵپەڕی WordPress ـەکەتدا جێی بکاتەوە، لەگەڵ پاکسازیی ئۆتۆماتیکی سەلامەتی namespace.

## تێڕوانینێکی گشتی

ئەم توانایە لۆگۆی گرافیکی ڤێکتەری پێوانەکراو (SVG) دروست دەکات بە پشتبەستن بە ئاراستەی براندینگی ماڵپەڕەکەت و پەسەندکراوەکانی دیزاین. ئەو SVG ـانەی دروست دەکرێن بە شێوەیەکی ئۆتۆماتیکی پاکسازی دەکرێن بۆ دڵنیابوون لەوەی کە بۆ بەکارهێنان لە WordPress ـدا سەلامەتن، لە کاتێکدا یەکپارچەیی بینراو دەپارێزرێت.

## پارامێتەرەکان

| پارامێتەر | جۆر | پێویستە | وەسف |
|-----------|------|----------|-------------|
| `site_name` | string | بەڵێ | ناوی ئەو ماڵپەڕەی کە لۆگۆکەی بۆ دروست دەکرێت |
| `style` | string | بەڵێ | شێوازی دیزاین (بۆ نموونە، "modern"، "classic"، "minimalist"، "playful") |
| `colors` | array | نەخێر | ڕیزەیەک لە کۆدە ڕەنگەکانی hex بۆ بەکارهێنان لە لۆگۆکەدا (بۆ نموونە، `["#678233", "#ffffff"]`) |
| `width` | number | نەخێر | پانی SVG بە پیکسڵ (بنەڕەت: 200) |
| `height` | number | نەخێر | بەرزی SVG بە پیکسڵ (بنەڕەت: 200) |
| `include_text` | boolean | نەخێر | ئایا ناوی ماڵپەڕەکە وەک دەق لە لۆگۆکەدا بخاتەوە یان نا (بنەڕەت: true) |

## فۆرماتی دەرچوون

ئەم توانایە زنجیرەنووسی SVG دەگەڕێنێتەوە بە پێکهاتەی خوارەوە:

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## ڕەفتاری پاکسازیی SVG

ئەو SVG ـانەی دروست دەکرێن، پاکسازیی ئۆتۆماتیکی سەلامەتی namespaceیان بەسەردا دەچێت بۆ:

- **لابردنی تایبەتمەندییە ناسەلامەتەکان** — event handler ـەکان، scripts، و تایبەتمەندییە مەترسیدارە ئەگەرییەکان دەسڕێتەوە
- **پاراستنی namespace ـەکان** — namespace ـەکانی SVG (xmlns, xlink) دەپارێزێت بۆ پیشاندانی دروست
- **پشتڕاستکردنەوەی پێکهاتە** — دڵنیا دەبێتەوە کە SVG ـەکە لەگەڵ ستانداردەکانی W3C دەگونجێت
- **کۆدکردنی یەکەکان** — کارەکتەرە تایبەتەکان لە ناوەڕۆکی دەقەدا بە شێوەی دروست escape دەکات
- **لابردنی ئاماژە دەرەکییەکان** — stylesheets و ئاماژەکانی وێنەی دەرەکی دەسڕێتەوە

ئەمە دڵنیا دەکاتەوە کە SVG ـەکە سەلامەتە بۆ جێکردنەوەی ڕاستەوخۆ لە WordPress ـدا، بەبێ پێویستی بە پاکسازیی زیاتر.

## نموونەی بەکارهێنان

**داواکاری:**
```
لۆگۆیەکی نوێ بۆ ستارتاپی تەکنەلۆژییەکەم بە ناوی "CloudSync" دروست بکە، بە بەکارهێنانی ڕەنگەکانی شین و سپی.
```

**ئەنجام:**
ئەم توانایە لۆگۆی SVG دروست دەکات کە:
- ناوی ماڵپەڕەکە "CloudSync" تێدایە
- ئەو پلانی ڕەنگە شین و سپییە دیاریکراوە بەکاردەهێنێت
- شوێن ڕێساکانی دیزاینی نوێ دەکەوێت
- بە شێوەی ئۆتۆماتیکی پاکسازی کراوە و ئامادەی بەکارهێنانە

## یەکخستن لەگەڵ Theme Builder

کاتێک هەڵبژاردنی ئاراستەی دیزاینی Theme Builder بەکاردەهێنیت، توانای دروستکردنی لۆگۆی SVG:

1. ئاراستەی دیزاین و پالێتی ڕەنگەکانت شیدەکاتەوە
2. لۆگۆی SVG ـیەکی تایبەتمەند دروست دەکات کە لەگەڵ پەسەندکراوەکانت دەگونجێت
3. بە شێوەی ئۆتۆماتیکی لۆگۆکە لە ناوچەی header/branding ـی ماڵپەڕەکەتدا جێدەکاتەوە
4. SVG ـەکە وەک لۆگۆی تایبەت لە میدیای WordPress ـدا هەڵدەگرێت

## باشترین شێوازەکان

- **پەسەندکراوەکانی شێواز بە ڕوونی دابین بکە** — شێوازی دیزاینەکەی کە دەتەوێت وەسف بکە (نوێ، کلاسیک، یارییانە، هتد.)
- **ڕەنگەکان دیاری بکە** — ڕەنگەکانی براندەکەت بۆ یەکدەستی بخەرەوە
- **پیشاندانی تاقی بکەرەوە** — دڵنیا ببەوە کە لۆگۆکە بە دروستی لەسەر قەبارە جیاوازەکانی شاشە پیشان دەدرێت
- **زیاتر تایبەتمەندی پێبدە** — ئامرازەکانی تایبەتمەندکردنی لۆگۆی WordPress بەکاربهێنە بۆ ڕێکخستنی قەبارە و شوێندانان

## سنووردارییەکان

- لۆگۆکانی SVG وەک گرافیکی جێگیر دروست دەکرێن (جوڵاو نین)
- لۆگۆی ئاڵۆز بە توخمە زۆرەکانەوە لەوانەیە پێویستی بە جوانکاری دەستی هەبێت
- فۆنتی تایبەت پشتگیری ناکرێت؛ دەق فۆنتەکانی سیستەم بەکاردەهێنێت
- ڕەهەندی زۆر گەورە یان زۆر بچووک لەوانەیە کاریگەری لەسەر کوالیتی هەبێت
