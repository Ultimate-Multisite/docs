# تۆماری گۆڕانکارییەکان {#changelog}

## 1.18.0 — بڵاوکراوەتەوە لە 2026-06-29 {#1180--released-on-2026-06-29}

### نوێ {#new}

- **ئامرازەکانی Google Calendar** — خوێندنەوەی ڕۆژمێر و ڕووداوە ڕێکخراوەکان بۆ ئۆتۆماسیۆنە ئاگادار لە خشتەی کات
- **نەخشەکردنی پەیوەندییەکان و یارمەتیدەرەکانی بەشداربوو** — هاوتاکردنی بەشداربووانی ڕۆژمێر لەگەڵ بەکارهێنەرانی ماڵپەڕ و پەیوەندییەکان
- **دەروازەکانی پەسەندکردنی مرۆیی و تۆمارەکانی بیرخستنەوە** — ڕاگرتنی ئۆتۆماسیۆنەکان بۆ پێداچوونەوە و خۆپاراستن لە ئاگادارکردنەوەی دووبارە
- **دابینکەری SMS ـی TextBee** — ناردنی ئاگادارکردنەوەکانی پەیامی دەقی ڕێکخراو لە ڕێگەی TextBee
- **پاکێجی هاوەڵی پێشکەوتوو** — زیادکردنی ئامرازەکانی سیستەمی فایل، داتابەیس، WP-CLI، دابەشکەری REST، دروستکەری plugin، وێنەگرتنی git، بەڕێوەبردنی بەکارهێنەر، و تاقیکردنەوەی ئەدا کە بە جیا لە بنیاتنراوی WordPress.org دابەش دەکرێن و لەلایەن گەشەپێدەری متمانەپێکراوەوە دانراون

### باشترکراو {#improved}

- **ڕێکخستنی خزمەتگوزاری بەڕێوەبراوی Superdav** — زیادکردنی endpoints ـی خزمەتگوزاری میوانداری کراو و دابینکردنی پەیوەندیی خۆکار بۆ ماڵپەڕە پشتگیریکراوەکان
- **پاکەتکردنی بڵاوکراوە** — دروستکردنی ZIP ـی جیاواز بۆ ناوەڕۆک و Advanced، بڵاوکردنەوەی هەردووکیان لە GitHub، و ناردنی تەنها پاکێجی ناوەڕۆک بۆ WordPress.org

### چاککراو {#fixed}

- **متمانەپێکردنی داواکاریی AI** — باشترکردنی هەڵبژاردنی مۆدێل، کاتی چاوەڕوانیی داواکاری، مۆدێلی ڕێکخستنی بنەڕەتی، مامەڵەکردن لەگەڵ دەقی بیرکردنەوە، و ڕێنمایی هەوڵدانەوە بۆ بانگەوازی ئامرازی نادروست
- **بەهێزکردنی ڕۆژمێر و بیرخستنەوە** — بەهێزکردنی token ـەکانی Google Calendar و لابردنی دووبارەیی لە بیرخستنەوەکان
- **دەستپێکردنی بەکارهێنەر و دەستپێکردنەوەی پەسەندکردن** — چاککردنی دەستپێکردنی پێشەوە و دەستپێکردنەوەی توانای پشتڕاستکراو
- **کێشەکانی پێداچوونەوەی پاکەتکردنی WordPress.org** — چارەسەرکردنی فیدبەکی پێداچوونەوەی پاکەتکردن بۆ بڵاوکراوەی ناوەڕۆک

## 1.16.0 — بڵاوکراوەتەوە لە 2026-05-20 {#1160--released-on-2026-05-20}

### نوێ {#new-1}

- **توانای دروستکردنی Logo SVG** — Theme Builder ئێستا دەتوانێت Logo SVG ـی تایبەت دروست بکات و بیخاتە ناوەوە بە پاککردنەوەیەکی پارێزراو لە namespace
- **بارکردنی وێنە لە چاوپێکەوتنی دۆزینەوە** — چاوپێکەوتنی دۆزینەوەی Theme Builder ئێستا هەنگاوێکی بارکردنی وێنە لەخۆدەگرێت بۆ کۆنتێکستی دیزاینی دەوڵەمەندتر
- **توانای پشتڕاستکردنەوەی کۆنتراستی پالێت** — پشکنینی جووتە ڕەنگەکان بۆ گونجان لەگەڵ WCAG پێش ئەوەی بۆ theme ـێک جێبەجێ بکرێن
- **مێنیوەکانی میوانداری** — Theme Builder ئێستا دەتوانێت پەڕەی مێنیوی خۆراک و خواردنەوەی پێکهاتەدار بۆ بازرگانییەکانی میوانداری دروست بکات
- **پیشاندانی پێشبینینی سەر مێز و مۆبایل** — لە کاتی هەڵبژاردنی ئاراستەی دیزاینەکەدا دیزاینەکەت لەسەر ئامێرەکانی مێز و مۆبایل پێشبینین بکە
- **پارامیتەری ناوی ڕێدۆزی** — توانای Create Menu ئێستا پشتگیری `navigation_label` ـێکی جیاواز دەکات کە لە ناونیشانی پەڕە جیاوازە
- **بەردەستبوونی ئامرازی Tier 1** — `sd-ai-agent/site-scrape` ئێستا ئامرازێکی Tier 1 ـە کە بە شێوەی بنەڕەتی لە Theme Builder بەردەستە

### چاککراو {#fixed-1}

- **کاشی AI Client** — ئێستا بە transients پشتگیری دەکرێت بۆ مانەوەی نێوان داواکارییەکان، کە ڕێگری لە لەدەستچوونی داتا دەکات لە ئەرکە درێژخایەنەکانی agent
- **بەستەرەکانی کرداری ڕیزی plugin** — بۆ ڕوونی ڕاستکرانەوە و ناویان گۆڕدرا

## 1.10.0 — بڵاوکراوەتەوە لە 2026-05-05 {#1100--released-on-2026-05-05}

### نوێ {#new-2}

- **گەڕانی ئینتەرنێتی Tavily** — زیادکردنی Tavily وەک دابینکەری گەڕان بۆ ئەنجامی گەڕانی ئینتەرنێتی دەوڵەمەندتر لەگەڵ Brave Search
- **لێهاتووییە ناوخۆییەکان کە ئاگادارن لە theme** — ڕێنماییەکانی لێهاتوویی Block Themes، Classic Themes، Kadence Blocks، و Kadence Theme ئێستا لەگەڵ plugin دابین دەکرێن
- **توانای فۆرمی پەیوەندیی دروستکەری ماڵپەڕ** — زیادکردنی فۆرمی پەیوەندی بۆ هەر پەڕەیەک ڕاستەوخۆ لە ڕووکاری چاتەوە

### باشترکراو {#improved-1}

- **یەکخستنی WooCommerce دووبارە ڕێکخرا** — ئێستا API ـە ڕەسەنەکانی WooCommerce بەکاردەهێنێت بۆ متمانەپێکردن و گونجانی باشتر
- **لیستی دابینکەرەکان خۆکار نوێ دەبێتەوە** — کاتێک هەر plugin ـێک چالاک یان ناچالاک دەکرێت

### چاککراو {#fixed-2}

- **توانای navigate-to** — لوپی بارکردنەوەی بێکۆتایی لەسەر هەندێک پەڕەی بەڕێوەبەر چاککرا
- **توانای list-posts** — ئێستا ناوەکانی پۆل و تاگ بە دروستی دەکاتە slugs
- **فەرمانەکانی WP-CLI** — ناسناوەکانی namespace ـی ونبوو دوای ڕێکخستنەوەی پێشووتر گەڕێنرانەوە
- **ئۆتۆماسیۆنی ڕووداو** — بە نەرمی مامەڵە لەگەڵ ئەو ماڵپەڕانە دەکات کە خشتەکانی ئۆتۆماسیۆنیان هێشتا دروست نەکراون
- **توانای memory-save** — ئێستا پێشگری دروستی namespace لە دروستکەری ڕێنمایی سیستەمدا بەکاردەهێنێت
- **ئەنجامەکانی ئامرازی Scalar** — ئێستا پێش گەڕاندنەوەیان بۆ AI بە دروستی دەخرێنە قاڵبەوە
- **ئامارەکانی بەکارهێنان** — ئێستا فۆرماتی کۆنی کلیلی تواناکان لە کاتی نوێکردنەوە لە وەشانی کۆنترەوە بە دروستی مامەڵە دەکات
