# Filter: signup_get_available_languages

لیستی زمانە بەردەستەکان بۆ خۆتۆمارکردنی ماڵپەڕی پێشەوە فلتەر دەکات.

ناردنی ڕیزبەندییەکی بەتاڵ بۆ ئەم hook ـە دەرچووی ڕێکخستنەکە لە فۆڕمی خۆتۆمارکردن ناچالاک دەکات، و کاتێک ماڵپەڕەکە دروست دەکرێت زمانی بنەڕەتی بەکاردەهێنرێت. ئەو زمانانەی پێشتر دانەنراون دەسڕدرێنەوە.

## پارامێتەرەکان

| ناو | جۆر | وەسف |
|------|------|-------------|
| $available_languages | `array` | زمانە بەردەستەکان. |
