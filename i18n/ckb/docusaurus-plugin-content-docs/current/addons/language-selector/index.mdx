# هەڵبژێری زمان {#language-selector}

Ultimate Multisite: هەڵبژێری زمان لە کاتی خۆتۆمارکردن

خانەیەکی هەڵبژاردنی زمان زیاد بکە بۆ ڕەوتی خۆتۆمارکردن.

## دامەزراندن {#installation}

1. 'ultimate-multisite-language-selector' باربکە بۆ بوخچەی '/wp-content/plugins/'
2. pluginـەکە لە ڕێگەی مێنیوی 'Plugins' لە WordPress Network Adminـەکەت چالاک بکە

## ڕێنمایی بەکارهێنان {#usage-guide}

بەم add-onـە خۆڕاییە دەتوانیت ڕێگە بە بەکارهێنەرەکانت بدەیت ئەو زمانە هەڵبژێرن کە دەتەوێت لە ڕەوتی خۆتۆمارکردن و لە پەڕەی چوونەژوورەوەدا ببینرێت.

دوای [چالاککردنی مۆڵەتەکەت](../../user-guide/getting-started/installing-ultimate-multisite.md)، دەتوانیت add-onـەکانی Ultimate Multisite دابمەزرێنیت کە مۆڵەتەکەت دەستگەیشتنی پێیان پێدەدات.

بڕۆ بۆ پەڕەی ڕێکخستنەکانی Ultimate Multisiteـت، پاشان لە مێنیوەکەی لای چەپ، کرتە لەسەر نیشانەی _Add-ons_ بکە.

![پەڕەی Add-ons](/img/admin/addons-page.png)

_بڕۆ بۆ پەڕەی Add-ons بۆ دەستپێکردنی دامەزراندنی هەڵبژێری زمان_

لەوێدا، هەموو add-onـەکان دەبینیت کە پێشکەشیان دەکەین.

add-onـە خۆڕاییەکان بۆ هەموو کڕیارەکانمان بەردەستن، بەبێ گوێدانە پلانی بەشدارییەکەی کە تۆماریان کردووە.

کرتە لە دوگمەی _Install it Now!_ بکە لە خوار وێنۆچکەی add-onـەکە. پاشان لە پەڕەی Plugins چالاکی بکە.

![پەڕەی Add-ons لەگەڵ دوگمەی دامەزراندن](/img/admin/addons-page.png)

_کرتە لە دوگمەکە بکە بۆ دامەزراندنی add-onـەکە_

<!-- وێنەی شاشە بەردەست نییە: پەڕەی WP Plugins کە دوگمەی Network Activate بۆ add-onـی Language Selector پیشان دەدات -->

_لە پەڕەی Pluginsـەکەت چالاکی بکە_

ئێستا، بڕۆ بۆ مێنیوی ڕێکخستنەکانی Ultimate Multisite و بچۆ ناو تابی _Network Settings_. لەوێدا، لە خوارەوەی پەڕەکەدا، بەشێکی نوێ دەبینیت بە ناوی _Languages_.

![پەڕەی ڕێکخستنەکان](/img/config/settings-general.png)

_بەشێکی تەواو نوێ دەرکەوت!_

دەتوانیت بە بەتاڵی بەجێی بهێڵیت بۆ پیشاندانی هەموو زمانەکان وەک هەڵبژاردە، یان ئەوانە هەڵبژێریت کە دەتەوێت بەردەستیان بکەیت. زمانەکان ئەوانەن کە لە دامەزراندنی WordPressـەکەتدا داتمەزراندوون.

<!-- وێنەی شاشە بەردەست نییە: لیستی دابەزینی هەڵبژاردنی زمان کە زمانە بەردەستەکان لە دامەزراندنی WP پیشان دەدات -->

_لیستی زمانە بەردەستەکان لەم دامەزراندنی WPـەدا_

دوای پاشەکەوتکردنی ئەو گۆڕانکارییانەی کردووتە، مێنیویەکی هەڵبژاردن لەگەڵ ئەو زمانانەی هەڵتبژاردووە لە پەڕەی چوونەژوورەوە و هەروەها لە ڕەوتی خۆتۆمارکردن پیشان دەدرێت.

![ڕێکخستنەکانی چوونەژوورەوە و تۆمارکردن](/img/config/settings-login-registration.png)

_هەڵبژێری زمان لە پەڕەی چوونەژوورەوە_

![پەڕەی تۆمارکردن](/img/config/registration-page.png)

_هەڵبژێری زمان لە ڕەوتی خۆتۆمارکردن_

## دانانی زمان لە ڕێگەی URL {#setting-the-language-via-the-url}

هەروەها دەتوانیت زمانی پەڕەیەک بگۆڕیت بە تێپەڕاندنی پارامێتەری locale بۆ URL. ئەگەر دەتەوێت بۆ پەڕەی تۆمارکردن بە زمانی ئیتاڵی ئاراستەبکرێیت، بۆ نموونە، ?locale=it_IT زیاد بکە بۆ URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

پێویستە زمانەکە لە تۆڕی WordPressـەکەتدا دامەزرابێت بۆ ئەوەی ئەمە کار بکات.
