# نوێنەرانی پشتگیری

بەڕێوەبردنی تۆڕێکی تەواو بە تەنیا ئەرکێکی ئاسان نییە. کاتێک تۆڕەکەت دەست دەکات بە گەشەکردن، دەست دەکەیت بە بینینی داواکاریی زیاتر بۆ پشتگیری. بیرۆکەیەکی باش دەبێت کە کەسانێک بهێنیت کە یارمەتیت بدەن بارگرانیی کارەکەت کەم بکەیتەوە. بەڵام، دانانی دەستگەیشتن/مۆڵەتی تەواو مەترسیدارە و لەوانەیە تۆڕەکەت بخاتە مەترسییەوە. لێرەدایە ئەم پێوەکراوەی نوێنەری پشتگیری دێتە ئاراوە.

ئەم پێوەکراوە ڕێگەت پێدەدات جۆرێکی نوێی بەکارهێنەری WordPress دروست بکەیت بە ناوی نوێنەری پشتگیری، و کۆنترۆڵی تەواوت پێدەدات لەسەر ئەوەی ئەو بەکارهێنەرە چ مۆڵەتێکی هەیە. دانانی مۆڵەت هێندە ئاسانە وەک کاراکردنی دوگمەی toggle و ناچالاککردنی ئەگەر بڕیارت دا بیسڕیتەوە.

## تایبەتمەندییەکان {#features}

- لیستی بەخشکراوی مۆڵەتەکانی WordPress و Ultimate Multisite
- کاراکردن و ناچالاککردنی دەستگەیشتن بە ئاسانی
- دروستکردنی ژمارەیەکی بێسنوور لە Account ـەکانی نوێنەر

## دامەزراندن {#installation}

1. 'ultimate-multisite-support-agents' باربکە بۆ بوخچەی '/wp-content/plugins/'
2. plugin ـەکە لە ڕێگەی پێڕستی 'پێوەکراوەکان' لە WordPress کارا بکە
3. Wizard ـی هەنگاو بە هەنگاو شوێن بکەوە بۆ ڕێکخستنی plugin ـەکە
