# Loco Translate

ئادۆنی Ultimate Multisite: Loco Translate Integration تواناکانی وەرگێڕانی تایبەت بە ماڵپەڕ بۆ تۆڕی WordPress multisite ـەکەت دەهێنێت. ئەگەر هەرگیز پێویستت بووبێت ماڵپەڕە جیاوازەکانی ناو تۆڕەکەت وەرگێڕانی جیاوازیان بۆ هەمان plugin یان theme هەبێت، ئەم ئادۆنە چارەسەری تەواوە.

**ئەم ئادۆنە چی دەکات**

ئەم ئادۆنە بە شێوەیەکی بێکێشە Ultimate Multisite لەگەڵ plugin ـی بەناوبانگی Loco Translate یەکدەخات، کە ڕێگە بە هەر subsite ـێک لە تۆڕەکەت دەدات وەرگێڕانی تایبەتی خۆی هەبێت. جا پلاتفۆرمێکی فرە-ناوچەیی بەڕێوە دەبەیت، خزمەتی کۆمەڵگە زمانییە جیاوازەکان دەکەیت، یان تەنها پێویستت بە نەرمی وەرگێڕان لە سەرانسەری تۆڕەکەت هەیە، ئەم ئادۆنە کارەکە ئاسان دەکات.

**گونجاوە بۆ:**

* بازرگانییە فرە-ناوچەییەکان کە پێویستییە جیاوازەکانی localization ـیان هەیە
* دامەزراوە پەروەردەییەکان کە خزمەتی کۆمەڵگە جۆراوجۆرەکان دەکەن
* پلاتفۆرمەکانی SaaS کە پشتگیری فرەزمانە پێشکەش دەکەن
* ئاژانسەکانی وێب کە چەندین ماڵپەڕی client بەڕێوە دەبەن
* هەر تۆڕێکی multisite کە پێویستی بە نەرمی وەرگێڕان هەبێت

## دامەزراندن {#installation}

1. دڵنیابە Ultimate Multisite دامەزرابێت و لەسەر ئاستی تۆڕ چالاک کرابێت
2. plugin ـی Loco Translate دابمەزرێنە و چالاکی بکە
3. ئەم ئادۆنە باربکە بۆ بوخچەی plugins ـەکەت
4. ئادۆنی Loco Translate Integration لەسەر ئاستی تۆڕ چالاک بکە
5. دەست بکە بە دروستکردنی وەرگێڕانی تایبەت بە ماڵپەڕ لە ڕێگەی Loco Translate لەسەر هەر subsite ـێک

هەر ئەوەندە! یەکخستنەکە بە شێوەیەکی خۆکار کار دەکات بەبێ هیچ ڕێکخستنی زیادە.

## پرسیارە دووبارەکان {#frequently-asked-questions}

## ئایا پاش دامەزراندن پێویستە هیچ شتێک ڕێکبخەم؟ {#do-i-need-to-configure-anything-after-installation}

نەخێر! یەکخستنەکە خۆکارە. کاتێک هەردوو ئادۆنەکە و Loco Translate چالاکن، subsite ـەکان دەتوانن دەستبەجێ دەست بکەن بە دروستکردنی وەرگێڕانی تایبەتی خۆیان.

## چی ڕوودەدات ئەگەر subsite ـێک وەرگێڕانی تایبەت دروست نەکات؟ {#what-happens-if-a-subsite-doesnt-create-custom-translations}

ماڵپەڕەکە بە شێوەیەکی خۆکار وەرگێڕانەکانی سەرانسەری تۆڕ وەک جێگرەوە بەکاردەهێنێت. هیچ شتێک تێکناچێت، و هەموو شتێک بەبێ کێشە کار دەکات.

## ئایا subsite ـەکان دەتوانن دەستیان بە وەرگێڕانەکانی سەرانسەری تۆڕ بگات؟ {#can-subsites-access-network-wide-translations}

بەڕێوەبەرانی ئاسایی ماڵپەڕ تەنها وەرگێڕانەکانی ماڵپەڕی خۆیان دەبینن و دەستکاریان دەکەن. Super admin ـەکان دەتوانن دەستیان بە هەردوو وەرگێڕانی سەرانسەری تۆڕ و تایبەت بە ماڵپەڕ بگات، بەپێی ئەو ماڵپەڕەی دەیبینن.

## ئایا ئەمە لەگەڵ هەردوو plugins و themes کار دەکات؟ {#does-this-work-with-both-plugins-and-themes}

بەڵێ! وەرگێڕانی تایبەت بە ماڵپەڕ بۆ هەردوو plugins و themes ـی WordPress کار دەکات.

## ئایا ئەمە تۆڕی multisite ـەکەم خاو دەکاتەوە؟ {#will-this-slow-down-my-multisite-network}

نەخێر. ئەم ئادۆنە بارکردنی پەڕگەی کارا بەکاردەهێنێت کە تەنها کاتێک پێویست بێت پشکنین بۆ وەرگێڕانی تایبەت بە ماڵپەڕ دەکات. هیچ کاریگەرییەکی بەرچاوی لەسەر کارایی نییە.

## ئایا دەتوانم وەرگێڕانەکانی تۆڕ کۆپی بکەم بۆ ماڵپەڕێکی دیاریکراو وەک خاڵی دەستپێک؟ {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

بەڵێ! لە کاتی دەستکاریکردنی وەرگێڕان لە Loco Translate، بە ئاسانی دەتوانیت بە وەرگێڕانەکانی تۆڕ دەست پێ بکەیت و بۆ هەر ماڵپەڕێک تایبەتمەندیان پێ بدەیت.

## ئایا ئەمە لەگەڵ WooCommerce و plugins ـەکانی تر کار دەکات؟ {#does-this-work-with-woocommerce-and-other-plugins}

بێگومان! هەر plugin یان theme ـێک کە فەنکشنی وەرگێڕانی ستانداردی WordPress بەکاردەهێنێت، لەگەڵ وەرگێڕانی تایبەت بە ماڵپەڕ کار دەکات.
