# PayFast

Ultimate Multisite: دەروازەی Payfast

Payfast - دەروازەیەکی خێرا و پارێزراوی پارەدانی ئۆنلاین لە ئەفریقای باشوور - وەک هەڵبژاردنێکی پارەدان بۆ Ultimate Multisite زیاد دەکات.

## دامەزراندن {#installation}

1. 'ultimate-multisite-payfast' باربکە بۆ بوخچەی '/wp-content/plugins/'
2. pluginەکە لە ڕێگەی مینیوی 'Plugins' لە WordPress چالاک بکە
3. Wizardە هەنگاو بە هەنگاوەکە پەیڕەو بکە بۆ ڕێکخستنی pluginەکە

## ڕێبەری بەکارهێنان {#usage-guide}

PayFast یەکێکە لە پێشەنگترین پرۆسێسکەرەکانی پارەدانی ئۆنلاین بۆ بازرگانییەکان لە ئەفریقای باشوور. ئەو خۆی تەرخان کردووە بۆ یارمەتیدانی فرۆشیارانی ئۆنلاین تا گەشە بکەن بەوەی مامەڵەکان بێکێشە و پارێزراو دەکات. بەپێی داواکاری بەکارهێنەری Ultimate Multisite، ئێمە بڕیارمان دا add-onێک دروست بکەین کە یەکگرتن لەگەڵ PayFast مومکین بکات. ئەم وتارە پیشان دەدات چۆن PayFast وەک دەروازەی پارەدانت ڕێکبخەیت بۆ وەرگرتنی پارەدان لە تۆڕی Ultimate Multisite ـەکەتدا.

یەکەم هەنگاو دامەزراندنی add on ـەکەیە.

# چۆن add-on ـی Payfast دابمەزرێنیت:

بڕۆ بۆ پەڕەی add-ons و add-on ـی Ultimate Multisite: دەروازەی Payfast هەڵبژێرە.

![پەڕەی add-ons](/img/admin/addons-page.png)

پەنجەرەی add-on ـەکە دەردەکەوێت. کرتە لەسەر Install Now بکە.

![پەڕەی add-ons لەگەڵ هەڵبژاردەی دامەزراندن](/img/admin/addons-page.png)

دوای تەواوبوونی دامەزراندن، دەگوازرێیتەوە بۆ پەڕەی pluginەکان. لێرەدا، تەنها کرتە لەسەر Network Activate بکە و add-on ـی _Payfast_ لەسەر تۆڕەکەت چالاک دەبێت.

# ڕێکخستنی PayFast

دوای دامەزراندنی add-on ـەکە، ئێستا دەتوانیت بچیتە ڕێکخستنەکانی Ultimate Multisite و دەست بکەیت بە شێوەپێدانی PayFast.

لەژێر پەڕەی **ڕێکخستنەکانی Ultimate Multisite**، دەتوانیت تابی "Payments" بدۆزیتەوە. لەژێر ئەم تابەدا، دەبێت "**Payfast** " لەگەڵ دوگمەی toggle ببینیت بۆ چالاککردن یان ناچالاککردنی ئەم دەروازەی پارەدانە.

![ڕێکخستنەکانی دەروازەی پارەدان](/img/config/settings-payment-gateways.png)بڕۆ پێشەوە و کرتە لەسەر دوگمەی toggle بکە بۆ چالاککردنی. پاشان سندووقەکە فراوان دەبێت بۆ پیشاندانی خانە جیاوازەکان کە پێویستە پڕیان بکەیتەوە وەک **Mechant ID, Merchant Key and PassPhrase.** دەتوانیت ئەم زانیاریانە لە Account ـی **Payfast** ـەکەت بەدەست بهێنیت

<!-- وێنەی شاشە بەردەست نییە: ڕێکخستنە فراوانکراوەکانی PayFast کە خانەکانی Merchant ID, Merchant Key, و PassPhrase پیشان دەدات -->

هەنگاوی دواتر ئەوەیە بچیتە ژوورەوە بۆ Account ـی PayFast ـەکەت بۆ بەدەستهێنانی ئەو زانیاریانەی پێویستتن.

لە کاتی چوونەژوورەوەدا، دەتوانیت دەستبەجێ ناسنامەکان ببینیت کە پێویستتن لە Dashboard ـی Account ـی **Payfast** ـەکەتەوە.

<!-- وێنەی شاشە بەردەست نییە: Dashboard ـی Account ـی PayFast کە ناسنامەکانی Merchant ID و Merchant Key پیشان دەدات -->دەتوانیت بە ئاسانی ناسنامەکان کۆپی و پەیست بکەیت و وردەکارییەکان لە ڕێکخستنەکانی Ultimate Multisite ـەکەت پڕبکەیتەوە. پاشان کرتە لەسەر دوگمەی **"Save settings** " بکە بۆ پاشەکەوتکردنی گۆڕانکارییەکان.

<!-- وێنەی شاشە بەردەست نییە: ڕێکخستنەکانی Ultimate Multisite PayFast کە ناسنامەکان پڕکراونەتەوە و دوگمەی Save پیشان دەدات -->

ئەوەیە، یەکگرتنی پارەدانی PayFast ـەکەت تەواو بوو!
