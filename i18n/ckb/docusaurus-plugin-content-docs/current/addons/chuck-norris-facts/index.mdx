# ڕاستییەکانی Chuck Norris

کەمێک نوکتە و خۆشی زیاد بکە بۆ ئەرکە ڕۆژانەکانی بەڕێوەبردنی WordPress ـت بە پێوەکراوی **Ultimate Multisite - ڕاستییەکانی Chuck Norris**. کاتێک چالاک دەکرێت، ڕاستییەکی هەڕەمەکیی Chuck Norris لە گۆشەی سەرەوەی لای ڕاستی هەموو پەڕەیەکی بەڕێوەبەر نیشان دەدرێت.

گونجاوە بۆ ئەوەی پێکەنین بهێنێتە ناو تیمەکەت، کەشوهەوا سووک بکاتەوە لە ماوەی دانیشتنە درێژەکانی بەڕێوەبردن، یان تەنها ئاهەنگی ئەو ناوازەییە ئەفسانەییە بگێڕیت کە Chuck Norris ـە.

## بۆچی ڕاستییەکانی Chuck Norris؟ {#why-chuck-norris-facts}

* **بەرزکردنەوەی ورەی تیم**: توخمێکی خۆش زیاد بکە بۆ بەڕێوەبردنی تۆڕەکەت
* **شکاندنی سەهۆڵ**: دەستپێکەرێکی ناوازەی گفتوگۆیە بۆ ئەندامانی نوێی تیم
* **بەرزکردنەوەی بەرهەمهێنان**: پێکەنینێکی خێرا دەتوانێت سەرنج و بەرهەمهێنان باشتر بکات
* **نوکتەی بێکۆتایی**: ڕاستییەکانی Chuck Norris هەرگیز کۆن نابن

## تایبەتمەندییە سەرەکییەکان {#key-features}

**١١٠+ ڕاستیی ئەفسانەیی**
کۆمەڵەیەکی گشتگیر لە نوکتە و ڕاستییە کلاسیکییەکانی Chuck Norris کە ئاهەنگی خودی ئەفسانەکە دەگێڕن.

**نیشاندانی نەخۆشکەر**
ڕاستییەکان لە شریتی بەڕێوەبەر بە شێوەیەکی نەخۆشکەر دەرکەون کە دەست لە ڕەوتی کارەکەت نادات.

**ئامادەی وەرگێڕان**
هەموو ڕاستییەکان لە فەنکشنەکانی وەرگێڕاندا پێچراونەتەوە، ئەمەش ئاسان دەکات بۆ ناوخۆییکردن بۆ زمانەکەت یان دروستکردنی ڕاستیی تایبەت بۆ تیمەکەت.

**بە تەواوی وەڵامدەرەوە**
ڕاستییەکان بە دروستی لەسەر شاشەی دێسکتۆپ، تابلێت و مۆبایل نیشان دەدرێن - Chuck Norris لەسەر هەموو ئامێرەکان ناوازەیە.

**پشتیوانی RTL**
پشتیوانی تەواو بۆ زمانە ڕاست‌بۆچەپەکان، چونکە Chuck Norris سنوورەکانی زمان تێدەپەڕێنێت.

**گونجاو لەگەڵ دەستکاریکەری بلۆک**
ڕاستییەکان بە زیرەکی لە دەستکاریکەری بلۆکدا دەشاردرێنەوە بۆ دوورکەوتنەوە لە ناکۆکییەکانی ڕووکار.

**سەرنجی دەستپێگەیشتن**
دەقی گونجاوی خوێنەری شاشە لەخۆدەگرێت بۆ پابەندبوونی تەواوی دەستپێگەیشتن.

## گونجاوە بۆ {#perfect-for}

* بەڕێوەبەرانی تۆڕی multisite کە دەیانەوێت کەسایەتی زیاد بکەن
* تیمەکانی پەرەپێدان کە نوکتەی باش بە نرخ دەزانن
* هەر کەسێک کە پێویستی بە دۆزێکی ڕۆژانە لە ناوازەیی Chuck Norris هەیە
* ماڵپەڕەکان کە دەیانەوێت ژینگەی کارێکی خۆش و ئەرێنی دروست بکەن

## دامەزراندن {#installation}

1. دڵنیا ببەوە پێوەکراوی Ultimate Multisite لەسەر تۆڕی WordPress ـەکەت دامەزراوە و چالاک کراوە
2. فایلەکانی پێوەکراوەکە باربکە بۆ بوخچەی `/wp-content/plugins/` ـەکەت
3. پێوەکراوەکە لە ڕێگەی مێنیوی 'Plugins' لە WordPress دا لەسەر ئاستی تۆڕ چالاک بکە
4. لە هەموو پەڕەیەکی بەڕێوەبەردا چێژ لە ڕاستییە هەڕەمەکییەکانی Chuck Norris وەربگرە!

هیچ ڕێکخستنێک پێویست نییە - تەنها چالاکی بکە و پێبکەنە!

## پرسیارە زۆر دووبارەکان {#frequently-asked-questions}

## ئایا ئەمە پێویستی بە Ultimate Multisite هەیە؟ {#does-this-require-ultimate-multisite}

هەرچەندە وەک پێوەکراوێکی Ultimate Multisite دیزاین کراوە، پێوەکراوەکە بە سەربەخۆیی کار دەکات و دەتوانێت لەسەر هەر دامەزراندنێکی WordPress کار بکات.

## دەتوانم ڕاستیی خۆم زیاد بکەم؟ {#can-i-add-my-own-facts}

بەڵێ! ڕاستییەکان ئامادەی وەرگێڕانن. دەتوانیت فایلی `.po` ـی خۆت بە ڕاستیی تایبەت دروست بکەیت و بیخەیتە بوخچەی `/lang`. یان دەتوانیت ڕیزبەندی ڕاستییەکان بە شێوەی بەرنامەیی فلتەر بکەیت.

## ئایا ئەمە بەڕێوەبەرەکەم خاو دەکاتەوە؟ {#does-this-slow-down-my-admin}

هەرگیز! پێوەکراوەکە زۆر سووکە و هیچ کاریگەرییەکی لەسەر کارایی نییە. تەنها دەق نیشان دەدات - هیچ بانگکردنێکی API ـی دەرەکی یان پرۆسەی قورس نییە.

## دەتوانم لەسەر پەڕە دیاریکراوەکان ناچالاکی بکەم؟ {#can-i-disable-it-on-certain-pages}

ڕاستییەکان بە شێوەی خۆکار لەو پەڕانەدا دەشاردرێنەوە کە لەوانەیە دەستێوەردان بکەن (وەک دەستکاریکەری بلۆک). هەروەها دەتوانیت hook ـەکانی WordPress بەکاربهێنیت بۆ تایبەتکردنی ئەو شوێنانەی ڕاستییەکان تێیدا دەرکەون.

## ئایا ڕاستییەکان بۆ هەموو بینەران گونجاون؟ {#are-the-facts-appropriate-for-all-audiences}

بەڵێ! هەموو ڕاستییەکان نوکتەی پاک و خێزاندۆستن کە پێگەی ئەفسانەیی Chuck Norris دەستەجەمع دەکەن.

## دەتوانم ڕاستییەکان بۆ زمانێکی تر وەربگێڕم؟ {#can-i-translate-the-facts-to-another-language}

بە تەواوی! هەموو ڕاستییەکان فەنکشنەکانی وەرگێڕانی WordPress بەکاردەهێنن. فایلێکی وەرگێڕان بۆ زمانەکەت دروست بکە و بیخەیتە بوخچەی `/lang`. دۆمەینی دەق: `ultimate-multisite-chuck-norris-facts`

## ئایا لەگەڵ multisite کار دەکات؟ {#does-it-work-with-multisite}

بەڵێ! بە تایبەتی بۆ تۆڕەکانی multisite دیزاین کراوە و دەتوانرێت لەسەر ئاستی تۆڕ چالاک بکرێت.
