# AffiliateWP

Multisite Ultimate: یەکخستنی AffiliateWP

AffiliateWP بەهێز بەکاربهێنە بۆ گەشەپێدانی بنکەی کڕیارانی تۆڕی Ultimo ـەکەت!

## دامەزراندن {#installation}

1. 'ultimate-multisite-git ' باربکە بۆ بوخچەی '/wp-content/plugins/'
2. پێوەکراوەکە لە ڕێگەی مێنیوی 'پێوەکراوەکان' لە WordPress چالاک بکە
3. ڕێبەری هەنگاو بە هەنگاو شوێن بکە بۆ ڕێکخستنی پێوەکراوەکە

## ڕێنمایی بەکارهێنان {#usage-guide}

[AffiliateWP](https://affiliatewp.com/) بووەتە چارەسەری سەرەکی کاتێک پرۆگرامی هاوبەشی بۆ بازرگانیی ئۆنلاینەکەت دروست دەکەیت. Ultimate Multisite: AffiliateWP ئەو پردەیە کە توانای ئەوەت پێدەدات هەمان شت بۆ تۆڕە پریمیمەکەت دروست بکەیت.

## دەستپێکردن {#getting-started}

لە هەموو شتێک پێشتر، تۆ **پێویستە** [**AffiliateWP**](https://affiliatewp.com/) **چالاکت هەبێت**! ناتوانیت پاشکۆکەمان بەکاربهێنیت ئەگەر **ئەو پێوەکراوەت نەبێت کە لەگەڵیدا یەکدەخرێت**. ئەگەر پلان نییە ڕێگە بە کڕیارەکانت بدەیت ئەویش لەسەر ماڵپەڕەکانیان بەکاربهێنن، AffiliateWP پێویستە تەنها لەسەر ماڵپەڕە سەرەکییەکەت چالاک بێت.

هەروەها، پێویستت بە هەردوو **Ultimate Multisite** و **Ultimate Multisite: AffiliateWP** هەیە کە **بەسەرتاسەری تۆڕ** **چالاک کرابن**.

## زیادکردنی Ultimate Multisite وەک یەکخستنێکی AffiliateWP {#adding-ultimate-multisite-as-an-affiliatewp-integration}

لەسەر ماڵپەڕە سەرەکییەکەت، بڕۆ بۆ ڕێکخستنەکانی هاوبەشەکان، و لە تابی یەکخستنەکان، Ultimate Multisite نیشان بکە. ئەمە بە [AffiliateWP](https://affiliatewp.com/) دەڵێت دەست بکات بە شوێنکەوتنی سەردانەکان و گۆڕانەکان لە ڕەوتی تۆمارکردنی Ultimate Multisite.

<!-- وێنەی شاشە بەردەست نییە: تابی ڕێکخستنەکانی یەکخستنەکانی AffiliateWP لەگەڵ خانەی هەڵبژاردنی Ultimate Multisite -->

_خانەی Ultimate Multisite لە خوارەوە نیشان بکە و پاشەکەوتی بکە_

ئەمە نزیکەی هەموو ئەو شتەیە کە پێویستە. سەردانەکان کە لە بەستەرەکانی ڕەوانەکردنەوە دێن شوێنکەوت دەکرێن و کاتێک ئەو بەکارهێنەرە دواتر یەکەم پارەدانی خۆی دەکات، گۆڕانەکە دروست دەکرێت و کۆمیشنێک بۆ ئەو هاوبەشە دانراوە کە بەرپرسی گۆڕانەکەیە.

بەڵام تەنها یەکەم پارەدان کە لە بەکارهێنەری تازە دروستکراو وەردەگیرێت شوێنکەوت دەکرێت. بۆ چالاککردنی پارەدانە دووبارەبووەوەکانی ڕەوانەکردن، پێویستت بە [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) دەبێت

## چالاککردنی ڕەوانەکردنە دووبارەبووەوەکان {#enabling-recurring-referrals}

دوای دامەزراندنی [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)، بڕۆ بۆ تابی Recurring Referrals لە ڕێکخستنەکانی هاوبەشەکان، و بەکارهێنانی ڕەوانەکردنە دووبارەبووەوەکان چالاک بکە.

_هەندە سادەیە: Enable Recurring Referrals چالاک بکە و هەموو شتێک ئامادەیە!_
