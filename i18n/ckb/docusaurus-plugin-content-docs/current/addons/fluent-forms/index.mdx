# Fluent Forms {#fluent-forms}

ئەم add-on ـە وای دەکات بتوانرێت سنوور بۆ Fluent Forms لەسەر Multisite Ultimate لە بەرهەمەکان، ئەندامێتییەکان یان ماڵپەڕەکان زیاد بکرێت.
Fluent Forms خشتەی تایبەت بەکاردەهێنێت لەبری جۆرەکانی پۆستی تایبەت، بۆیە هەڵبژاردە ئاساییەکانی سنووردارکردن لەگەڵ Fluent Forms کار ناکەن.

## تایبەتمەندییەکان {#features}

- سنوور بۆ Fluent Forms یان Conversational Fluent Forms زیاد بکە
- کڕیاران پەیامێکی دۆستانە وەردەگرن کە داوایان لێدەکات پلانی خۆیان بەرز بکەنەوە کاتێک دەگەنە سنوورەکە
