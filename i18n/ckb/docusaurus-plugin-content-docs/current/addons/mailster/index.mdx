# Mailster {#mailster}

تۆڕی multisite ـەکەت بگۆڕە بۆ ئامێرێکی بەهێزی دروستکردنی لیستی ئیمەیڵ بە addon ـی Ultimate Multisite Mailster Integration. بە شێوەیەکی خۆکار کڕیارە نوێکان زیاد بکە بۆ لیستەکانی ئیمەیڵی Mailster کاتێک خۆیان تۆمار دەکەن یان کڕین ئەنجام دەدەن، پەیوەندییەکی بێکێشە لە نێوان پلاتفۆرمی ئەندامێتییەکەت و ستراتیژی بازاڕکردنی ئیمەیڵ دروست دەکات.

## تایبەتمەندییە سەرەکییەکان {#key-features}

**هاوکاتکردنی خۆکاری بەشداربوو**
* بە شێوەیەکی خۆکار کڕیاران زیاد بکە بۆ لیستەکانی Mailster کاتێک خۆیان تۆمار دەکەن
* هەڵبژێرە کە لە کاتی خۆتۆمارکردندا دەستبەجێ هاوکات بکرێت یان چاوەڕێی پشتڕاستکردنەوەی پارەدان بکات
* لیستەکانی ئیمەیڵەکەت بەبێ دەستتێوەردانی دەستی تازە و نوێکراوە بهێڵەوە

**بەڕێوەبردنی نەرمونیانی لیست**
* لیستی بنەڕەتی گشتی بۆ هەموو کڕیارە نوێکان دابنێ
* بنەڕەتەکان بە لیستە تایبەتەکانی بەرهەم بۆ کەمپەینی ئامانجدار جێگرەوە بکە
* بە ئاسانترین شێوە تەرخانکردنی لیستەکان بە ڕووکاری ئاشنای Mailster بەڕێوەببە

**هەڵبژاردنەکانی Opt-In ی گونجاو لەگەڵ GDPR**
* لە نێوان بەشداربوونی خۆکار یان ڕەزامەندیی لەسەر بنەمای checkbox هەڵبژێرە
* double opt-in چالاک بکە بۆ بەشداربوونی پشتڕاستکراوە
* پیشاندان/شاردنەوەی checkbox ـی opt-in لەسەر بنەمای پێداویستییەکانی پابەندبوونەکەت

**نەخشەکردنی زیرەکی داتا**
* بە شێوەیەکی خۆکار ناو و زانیاریی پەیوەندیی کڕیاران هاوکات بکە
* ناونیشانی billing بۆ خانەکانی بەشداربوو نەخشە بکە
* بەرواری خۆتۆمارکردن بۆ کەمپەینی ئامانجدار بەدوادا بچۆ

**کۆنترۆڵی ئاستی بەرهەم**
* لیستی جیاواز بۆ بەرهەمی جیاواز تەرخان بکە
* کەمپەینی ئامانجدار دروست بکە لەسەر بنەمای ئەوەی کڕیاران چی دەکڕن
* کاتێک پێویست بوو Mailster بۆ بەرهەمی دیاریکراو ناچالاک بکە

**ئەزموونی پیشەیی**
* یەکخستنی ڕەسەنی UI ـی Mailster بۆ هەڵبژاردنی لیست
* تۆمارکردنی ورد بۆ چارەسەرکردنی کێشە و پشتڕاستکردنەوە
* نوێکردنەوەی بێکێشەی بەشداربووانی هەبوو

## گونجاوە بۆ {#perfect-for}

* خاوەنەکانی ماڵپەڕی ئەندامێتی کە لیستی ئیمەیڵ دروست دەکەن
* پلاتفۆرمەکانی SaaS کە دەیانەوێت کڕیارە نوێکان پەروەردە بکەن
* دروستکەرانی کۆرس کە قوتابیان بەگوێرەی تۆماربوون دابەش دەکەن
* هەرکەسێک کە Ultimate Multisite لەگەڵ Mailster بەکاردەهێنێت

## چۆن کار دەکات {#how-it-works}

1. addon ـەکە لەسەر تۆڕی Ultimate Multisite ـەکەت دابمەزرێنە و چالاکی بکە
2. لیستە بنەڕەتییە گشتییەکان لە ڕێکخستنەکانی Ultimate Multisite ڕێکبخە
3. بە ئارەزوو، بە لیستە تایبەتەکانی بەرهەم جێگرەوەی بکە
4. دۆخی opt-in و کاتی بەشداربوون هەڵبژێرە
5. سەیری گەشەکردنی خۆکاری لیستەکانی Mailster ـەکەت بکە!

## دامەزراندن {#installation}

1. فایلەکانی plugin باربکە بۆ بوخچەی addons ـی Ultimate Multisite ـەکەت
2. دڵنیابە Mailster لەسەر ماڵپەڕی سەرەکییەکەت دامەزراوە و چالاکە
3. addon ـەکە لە ڕێگەی پەڕەی addons ـی Ultimate Multisite چالاک بکە
4. بچۆ بۆ Settings > Mailster Integration بۆ ڕێکخستن
5. لیستە بنەڕەتییەکانت هەڵبژێرە و پەسەندکارییەکانی opt-in ڕێکبخە

## پرسیارە دووبارەکان {#frequently-asked-questions}

## ئایا ئەم addon ـە بەبێ Ultimate Multisite کار دەکات؟ {#does-this-addon-work-without-ultimate-multisite}

نەخێر، ئەم addon ـە پێویستی بە Ultimate Multisite هەیە بۆ کارکردن. بە تایبەتی داڕێژراوە بۆ یەکخستنی Mailster لەگەڵ سیستەمی کڕیار و بەرهەمی Ultimate Multisite.

## ئایا پێویستە Mailster لەسەر هەموو ماڵپەڕەکان دامەزرابێت؟ {#do-i-need-mailster-installed-on-all-sites}

نەخێر، تەنها پێویستە Mailster لەسەر ماڵپەڕی سەرەکیی تۆڕەکەت دامەزرابێت و چالاک کرابێت. addon ـەکە بە شێوەیەکی خۆکار لە هەموو ژێرماڵپەڕەکانەوە پەیوەندی پێوە دەکات.

## دەتوانم لیستی جیاواز بۆ بەرهەمی جیاواز تەرخان بکەم؟ {#can-i-assign-different-lists-to-different-products}

بەڵێ! دەتوانیت لیستە بنەڕەتییە گشتییەکان دابنێیت، پاشان لەسەر بنەمای هەر بەرهەمێک جێگرەوەیان بکەیت. ئەمە ڕێگەت پێدەدات کەمپەینی ئیمەیڵی زۆر ئامانجدار دروست بکەیت لەسەر بنەمای ئەوەی کڕیاران چی دەکڕن.

## چی ڕوودەدات ئەگەر کڕیارێک پێشتر لە Mailster هەبێت؟ {#what-happens-if-a-customer-already-exists-in-mailster}

بە شێوەی بنەڕەتی، addon ـەکە بەشداربووانی هەبوو نوێ دەکاتەوە و زیادیشیان دەکات بۆ لیستە نوێکان. ئەگەر پێت باشە بەشداربووانی هەبوو تێپەڕێنیت، دەتوانیت ئەمە لە ڕێکخستنەکان ناچالاک بکەیت.

## ئایا ئەمە لەگەڵ GDPR گونجاوە؟ {#is-this-gdpr-compliant}

بەڵێ! addon ـەکە دۆخی opt-in لەسەر بنەمای checkbox لەخۆدەگرێت کە تێیدا کڕیاران دەبێت بە ئاشکرا ڕەزامەندی بدەن بۆ وەرگرتنی ئیمەیڵ. ئەمە پێداویستییەکانی GDPR جێبەجێ دەکات کاتێک لەگەڵ سیاسەتی تایبەتمەندیی گونجاو تێکەڵ دەکرێت.

## دەتوانم double opt-in بەکاربهێنم؟ {#can-i-use-double-opt-in}

بە تەواوی! ڕێکخستنی double opt-in چالاک بکە و Mailster پێش چالاککردنی بەشداربوونەکان ئیمەیڵی پشتڕاستکردنەوە دەنێرێت.
