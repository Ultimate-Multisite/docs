# Gratis AI Agent {#gratis-ai-agent}

Gratis AI Agent یارمەتیدەری AIـیەکی سەربەخۆی WordPressـە کە دەتوانێت بە ڕێنماییەکانی زمانی سروشتی، ماڵپەڕە تەواوەکانی WordPress پلان بکات، دروست بکات و بەڕێوەیان ببات. کۆمەڵێکی ڕێکخراو لە **تواناکان** پیشان دەدات — کردارە جیاکراوە و تاقیکراوەکان کە بریکارەکە دەتوانێت بانگیان بکات — کە هەموو شتێک لە بەڕێوەبردنی ناوەڕۆکەوە تا دەستکاریکردنی سیستەمی دیزاین دەگرێتەوە. وەشانی 1.9.0 تواناکانی دروستکردنی ناوەڕۆک، پێداچوونەوەی بینراوی پەڕە لە ڕێگەی وێنەگرتنی شاشە لەلایەنی client، پێنج پرۆفایلی بریکاری ناوخۆیی، و feature flag بۆ کۆنترۆڵی دەستگەیشتن و براندینگ زیاد دەکات.

## تایبەتمەندییە سەرەکییەکان {#key-features}

- **سیستەمی تواناکان** — کاتەلۆگێکی مۆدیولار و فراوانبووی کردارەکان کە بریکارەکە دەتوانێت لەسەر دامەزراندنی WordPressـەکەت بانگیان بکات
- **بەڕێوەبردنی ناوەڕۆک** — دروستکردنی بابەت و پەڕەکان (لەوانەش دیاریکردنی داڕێژەکانی پەڕە)، دروستکردنی کۆمەڵێک بابەت بە یەک بانگ، دانانی وێنەی دیاریکراو، و دروستکردنی فۆڕمەکانی پەیوەندی
- **پێداچوونەوەی بینراو** — گرتنی وێنەی شاشەی پەڕە زیندووەکان، جیاوازی نێوان دۆخی پێش/دوای، و وەرگرتنی پێداچوونەوەی دیزاینی دروستکراو بە AI کە ڕێکخستن، تایپۆگرافی، ڕەنگ و دەستگەیشتن دەگرێتەوە
- **پێنج بریکاری ناوخۆیی** — نووسەری ناوەڕۆک، دروستکەری ماڵپەڕ، ستۆدیۆی دیزاین، بەڕێوەبەری پێوەکراو، و یارمەتیدەری پشتگیری؛ لە ڕێگەی هەڵبژێری بریکار لە سەرپەڕەی چاتەکەدا دەتوانرێت بیانگۆڕیت
- **Feature flag** — دوگمەکانی کۆنترۆڵی دەستگەیشتن و براندینگ لە Settings → Feature Flags بۆ سنووردارکردنی ڕۆڵ، ناونانی سپی-لەیبڵ، و دەستکاریکردنی ویجێت
- **بەڕێوەبردنی جۆری بابەتی تایبەت** — تۆمارکردن، لیستکردن و سڕینەوەی جۆرە بابەتە تایبەتەکان لەگەڵ مانەوەیان دوای دەستپێکردنەوەکان
- **بەڕێوەبردنی پۆلێنی تایبەت** — تۆمارکردن، لیستکردن و سڕینەوەی پۆلێنە تایبەتەکان لەگەڵ مانەوەیان
- **تواناکانی سیستەمی دیزاین** — تێخستنی CSSـی تایبەت، بەڕێوەبردنی نەخشەکانی بلۆکی هەڵبژێردراو، دانانی لۆگۆی ماڵپەڕ، و جێبەجێکردنی پێشڕێکخستنەکانی theme.json
- **بەڕێوەبردنی ستایلە گشتییەکان** — خوێندنەوە و نووسینی بەهاکانی theme.json بۆ پالیەتی ڕەنگ، تایپۆگرافی، بۆشایی و زیاتر
- **بەڕێوەبردنی مێنیوی ڕێدۆزی** — دروستکردن و نوێکردنەوەی مێنیوەکانی ڕێدۆزی و بڕگەکانیان بە شێوەی بەرنامەیی
- **بەڕێوەبردنی هەڵبژاردنەکان** — خوێندنەوە و نووسینی هەڵبژاردنەکانی WordPress لەگەڵ لیستی ڕێگری ئاسایشی ناوخۆیی بۆ پاراستنی ڕێکخستنە گرنگەکان
- **تۆماری تواناکانی دامەزرێنراو** — دۆزینەوە و چالاککردنی پاکێجە توانای زیادەکان بەبێ دەستکاریکردنی پەڕگە سەرەکییەکانی پێوەکراو
- **هاوئاهەنگکردنی دروستکەری ماڵپەڕ v2** — پلانە فرە-هەنگاوەکانی دروستکردنی ماڵپەڕ لەگەڵ دۆزینەوەی پێوەکراو، شوێنکەوتنی پێشکەوتن، و چاککردنەوەی هەڵە
- **فەرمانی پێوانەی WP-CLI** — لە هێڵی فەرمانەوە یەکەی پێوانەی توانای بریکارەکە بۆ CI و هەڵسەنگاندنی مۆدێل جێبەجێ بکە

## پێداویستییەکان {#requirements}

- WordPress 6.0 یان بەرزتر
- PHP 8.1 یان بەرزتر
- WordPress فرە-ماڵپەڕ (چالاککراو لەسەر تۆڕ) یان دامەزراندنی تاک-ماڵپەڕ
- پێوەکراوی پەیوەستکەری دابینکەری AIـی چالاک (Anthropic، OpenAI، یان هاوتا)

## دامەزراندن {#installation}

1. پەڕگەکانی پێوەکراو باربکە بۆ `/wp-content/plugins/gratis-ai-agent`
2. پێوەکراوەکە لە ڕێگەی شاشەی **پێوەکراوەکان** لە WordPress چالاک بکە
3. بڕۆ بۆ **Gratis AI Agent → Settings** و دابینکەری AIـەکەت پەیوەست بکە
4. (ئارەزوومەندانە) **تۆماری تواناکان** بگەڕێ بۆ دامەزراندنی پاکێجە توانای زیادەکان

## دەستپێکی خێرا {#quick-start}

کاتێک چالاک کرا و بە دابینکەری AI پەیوەست بوو، پانێڵی **Gratis AI Agent** لە لایەنە لاوەکی بەڕێوەبەرایەتی WordPress بکەرەوە. ڕێنماییەکی زمانی سروشتی بنووسە و بریکارەکە هەنگاوە پێویستەکان بە بەکارهێنانی تواناکانی بەردەستی پلان و جێبەجێ دەکات.

**نموونەی ڕێنماییەکان:**

```
جۆرێکی بابەتی تایبەت بە ناوی "Portfolio" لەگەڵ پۆلێنێکی "Project Category" دروست بکە.
```

```
پالیەتێکی ڕەنگی دۆخی تاریک بخەرە ناو theme و ڕەنگی سەرەکی بکە بە #1a1a2e.
```

```
ماڵپەڕێکی چێشتخانە دروست بکە لەگەڵ مێنیوی ئۆنلاین، فۆڕمی حجزکردن، و پەڕەی پەیوەندی.
```

## تێڕوانینی گشتی تواناکان {#abilities-overview}

تواناکان کردارە ئەتۆمییەکانن کە بریکارەکە دەتوانێت ئەنجامیان بدات. هەر توانایەک پۆلێکی تۆمارکراوی PHPـە کە سکیمایەک (پارامێتەرەکان و جۆرەکانی گەڕاندنەوە) پیشان دەدات کە بریکارەکە لە کاتی جێبەجێکردندا دەیخوێنێتەوە. تواناکان بە پێی بوارە کاراییەکان کۆدەکرێنەوە:

| بوار | تواناکان |
|---|---|
| **بەڕێوەبردنی ناوەڕۆک** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **پێداچوونەوەی بینراو** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **جۆرە بابەتە تایبەتەکان** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **پۆلێنە تایبەتەکان** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **سیستەمی دیزاین** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **ستایلە گشتییەکان** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **مێنیوەکانی ڕێدۆزی** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **بەڕێوەبردنی هەڵبژاردنەکان** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **تواناکانی دامەزرێنراو** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **دروستکەری ماڵپەڕ** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

بۆ بەڵگەنامەی تەواوی پارامێتەر و نموونەکانی بەکارهێنان، سەیری [سەرچاوەی تواناکان](./abilities) بکە.

## WP-CLI {#wp-cli}

Gratis AI Agent بنەماڵەی فەرمانی `wp gratis-ai-agent` لەگەڵ خۆیدا دێنێت بۆ جێبەجێکردنی پێوانەکان و بەڕێوەبردنی تواناکان لە هێڵی فەرمانەوە. بۆ لیستی تەواوی فەرمانەکان سەیری [سەرچاوەی WP-CLI](./wp-cli) بکە.

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## ڕێکخستن {#configuration}

هەموو ڕێکخستنەکان لە ژێر **Gratis AI Agent → Settings** لە بەڕێوەبەرایەتی WordPressـدان:

| ڕێکخستن | وەسف |
|---|---|
| **دابینکەری AI** | پەیوەستکەری دابینکەری AIـی چالاک هەڵبژێرە و ڕێکی بخە |
| **مۆدێل** | مۆدێلی زمان هەڵبژێرە کە بۆ دانیشتنەکانی بریکار بەکاربهێنرێت |
| **زۆرترین Token** | بودجەی token بۆ هەر داواکارییەک (بنەڕەت 4096) |
| **کاتی بەسەرچوونی توانا** | زۆرترین چرکە کە توانایەکی تاک دەتوانێت کار بکات (بنەڕەت 30) |
| **تۆمارکردنی Debug** | بانگەکانی توانا و بیرکردنەوەی بریکار تۆمار بکە بۆ تۆماری debugـی WordPress |

## لیستی ڕێگری ئاسایشی بەڕێوەبردنی هەڵبژاردنەکان {#options-management-safety-blocklist}

توانای بەڕێوەبردنی هەڵبژاردنەکان لیستی ڕێگریکردنی ناوخۆیی تێدایە کە ڕێ لە ئەژەنتەکە دەگرێت هەڵبژاردنە هەستیارەکانی WordPress بخوێنێتەوە یان بینووسێت. لیستی ڕێگریکردنی بنەڕەتی ئەمانە دەگرێتەوە:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (هەڵبژاردنەکانی URL — بۆ ئەمانە ئامرازی گواستنەوەی تایبەت بەکاربهێنە)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (کلیلە نهێنییەکان)
- `active_plugins`, `template` (چالاککردنی plugin/theme — تواناکانی تایبەت بەکاربهێنە)
- ناسنامەکانی دەروازەی پارەدان و کلیلەکانی API کە لەلایەن pluginە دامەزراوەکانەوە تۆمارکراون

دەتوانرێت بەشە زیادەکان لە ڕێگەی فلتەری `gratis_ai_agent_options_blocklist` زیاد بکرێن.

## Hook و فلتەرەکان {#hooks-and-filters}

### کردارەکان {#actions}

- `gratis_ai_agent_ability_registered` — دوای تۆمارکردنی توانایەک کاردەکات، ناوی پۆلی تواناکە وەردەگرێت
- `gratis_ai_agent_plan_started` — کاتێک پلانی دروستکەری ماڵپەڕ دەست بە جێبەجێکردن دەکات کاردەکات
- `gratis_ai_agent_plan_completed` — کاتێک پلانێک بە سەرکەوتوویی تەواو دەبێت کاردەکات
- `gratis_ai_agent_plan_failed` — کاتێک پلانێک هەڵەیەکی چارەهەڵنەگر دەبینێت کاردەکات

### فلتەرەکان {#filters}

- `gratis_ai_agent_abilities` — پێش ئەوەی ئەژەنتەکە لیستی توانای تۆمارکراو بار بکات، دەستکاری بکە
- `gratis_ai_agent_options_blocklist` — لیستی ڕێگریکردنی سەلامەتی هەڵبژاردنەکان فراوان بکە یان کەم بکەرەوە
- `gratis_ai_agent_benchmark_questions` — پرسیارە بنچمارکە تایبەتەکان زیاد بکە بۆ کۆمەڵەی هەڵسەنگاندن
- `gratis_ai_agent_plan_plugins` — لە کاتی دروستکردنی پلانی ماڵپەڕ، لیستی دۆزینەوەی plugin دەستکاری بکە
