# تۆماری گۆڕانکارییەکان

## 1.9.0 — بڵاوکرایەوە لە 2026-04-28 {#190--released-on-2026-04-28}

### نوێ {#new}

- **توانای `create_contact_form`** — فۆڕمێکی پەیوەندی دروست دەکات بە بەکارهێنانی پێوەکراوی فۆڕمی چالاک (Contact Form 7, WPForms, Fluent Forms, یان Gravity Forms) و کورتەکۆدێکی ئامادە بۆ تێخستن لە هەر بابەتێک یان پەڕەیەکدا دەگەڕێنێتەوە.
- **توانای `set_featured_image`** — وێنەیەکی تایبەت بە بابەتێک یان پەڕەیەک دەدات لە ناسنامەی هاوپێچییەکی هەبووی کتێبخانەی میدیا یان URLێکی دوورەوە؛ کاتێک URL دابین بکرێت وێنەکە بە شێوەیەکی خۆکار هاوردە دەکات.
- **توانای `batch_create_posts`** — چەندین بابەت لە بانگکردنێکی تاکی توانادا دروست دەکات. هەمان پارامێتەرەکانی `create_post` پشتگیری دەکات، سەرکەوتن/شکستی هەر بابەتێک بە جیا ڕاپۆرت دەکات، و دۆخی ئارەزوومەندانەی `stop_on_error`ی هەیە.
- **پارامێتەری `page_template`** — بۆ `create_post` و `update_post` زیاد کرا. فایلی قاڵبی پەڕەی PHP (وەک `page-full-width.php`) لە کاتی دروستکردن یان نوێکردنەوەدا دیاری دەکات. بۆ گەڕانەوە بۆ بنەڕەتی ڕووکارەکە، دەقێکی بەتاڵ بۆ `update_post` بنێرە.
- **تواناکانی وێنەگرتنی شاشە لە لایەنی client** — `capture_screenshot`، `compare_screenshots`، و `review_page_design`. وێنەی تەواو یان viewportـی پەڕە زیندووەکان لە ڕێگەی وێبگەڕێکی headless لە لایەنی سێرڤەرەوە بگرە، دوو وێنەی شاشە بەراورد بکە، و پێداچوونەوەیەکی دیزاینی دروستکراو بە AI وەربگرە کە نەخشە، تایپۆگرافی، ڕەنگ، و دەستگەیشتن دەگرێتەوە.
- **پێنج بریکاری ناوخۆیی** — نووسەری ناوەڕۆک، دروستکەری ماڵپەڕ، ستۆدیۆی دیزاین، بەڕێوەبەری پێوەکراو، و یارمەتیدەری پشتگیری. هەر بریکارێک کۆمەڵە ئامرازێکی تایبەت، داواکارییەکی سیستەمی گونجاوکراو، و پێشنیارە سەرەتاییەکانی هەیە. لە ڕێگەی **هەڵبژێری بریکار**ی نوێ لە سەردێڕی چاتەکەدا دەتوانرێت بگۆڕدرێت. ببینە [بریکارە ناوخۆییەکان](../../user-guide/configuration/built-in-agents).
- **ئاڵاکانی تایبەتمەندی** — تابی نوێی **ڕێکخستنەکان → ئاڵاکانی تایبەتمەندی** لەگەڵ دوگمەکانی کۆنترۆڵی دەستگەیشتن (سنووردارکردن بە بەڕێوەبەران، سنووردارکردن بە بەڕێوەبەرانی تۆڕ، دەستگەیشتنی بەشداربوو، ناچالاککردن بۆ ئەندام نەبووان) و هەڵبژاردنەکانی براندینگ (شاردنەوەی ئاماژەی پێپەڕە، ناوی بریکاری تایبەت، شاردنەوەی هەڵبژێری بریکار، بەکارهێنانی ئایکۆنی ماڵپەڕ وەک وێنەی بریکاری چات). ببینە [ڕێکخستنەکانی Gratis AI Agent](../../user-guide/administration/gratis-ai-agent-settings).
- **گەڕاندنەوەی دانیشتنی کۆتایی** — پانێڵی چات ئێستا نوێترین گفتوگۆ بە شێوەیەکی خۆکار لە کاتی بارکردنی پەڕە و کردنەوەی ویجەتدا دووبارە بار دەکاتەوە، بۆیە دەقوبەست لە نێوان گەڕانەکانی پەڕەدا هەرگیز لەدەست ناچێت.
- **لینکەکانی کرداری پێوەکراو** — لینکی خێرا بۆ ڕێکخستنەکان و تۆماری تواناها ئێستا لە شاشەی **پێوەکراوەکان → پێوەکراوە دامەزراوەکان**ی WordPress لە خوار وەسفی پێوەکراوەکە دەردەکەون.

### باشترکراو {#improved}

- **دووبارە هەوڵدانی سەرچاوەی وێنە** — بریکارەکە ئێستا پێش ئەوەی بگەڕێتەوە بۆ وێنەیەکی دروستکراو بە AI لە کاتی شکستی داگرتندا، دووبارە هەوڵی هەموو سەرچاوە وێنە خۆڕاییە ڕێکخراوەکان دەداتەوە.
- **پانێڵی زانیاری مۆدێل** — هەمیشە لە سەردێڕی چاتدا دیارە؛ چیتر دوای یەکەم پەیام ناشاردرێتەوە.
- **ڕەفتاری خۆ-سکرۆڵ** — کاتێک بەکارهێنەر بۆ خوێندنەوە سکرۆڵ بەرەو سەرەوە دەکات، خۆ-سکرۆڵ ڕادەوەستێت؛ دوگمەیەکی هەڵواسراوی **سکرۆڵ بۆ خوارەوە** دەردەکەوێت و کاتێک بەکارهێنەر دەگاتە نوێترین پەیام بە شێوەیەکی خۆکار لادەچێت.
- **ڕووکاری هەڵبژێری بریکار** — وەک overlayـێکی خشتی فۆڕم لەگەڵ ئایکۆنی هەر بریکارێک دووبارە دیزاین کراوەتەوە، کە ناسینەوە و گۆڕینی بریکارەکان بە یەک نیگا ئاسانتر دەکات.
- **بەشەکانی JSی lazy-loaded** — کۆمەڵەی سەرەتایی JavaScriptـی ویجەتی چات ئێستا بە بەشەکانی lazy-loaded دابەش کراوە، قەبارەی کۆمەڵەی سەرەتایی بە 75–90% کەم دەکاتەوە.
- **دووبارە دیزاینکردنی ویجەتی چات** — ئایکۆنی یەکگرتووی AI جێی وێنەی تایبەتی پێشووی گرتووەتەوە؛ لەگەڵ سیستەمی بریکاری ناوخۆیی یەکدەگرێتەوە.
- **کردنە لینکی URL** — URLـەکان کە لە پەیامە سیستەمییەکان و قاپەکانی پەیامی هەڵەدا دەردەکەون، ئێستا وەک لینکی کرتەکراو پیشان دەدرێن.

### چاککراوە {#fixed}

- **دۆزینەوەی توانا** — وەسفەکان، ئاماژەکانی داواکاری سیستەم، و هاوتاکردنی namespace ڕاست کرانەوە بۆ ئەوەی هەموو تواناها بە شێوەیەکی پشتپێبەستراو لە لیستی ئامرازەکانی بریکاردا دەربکەون.
- **کاشی دابینکەران** — دابینکەران ئێستا لە سەرانسەری ماڵپەڕدا لە ڕێگەی ژمێرەری وەشانێکەوە کاش دەکرێن، کێشەکانی دابینکەری کۆن لە تۆڕەکانی multisiteدا ڕێگری دەکات.
- **`ability_invalid_output`** — لە 12 دەستکاریکەری توانا چارەسەر کرا؛ هەموویان وەڵامی JSONـی بە دروستی پێکهێنراو دەگەڕێننەوە.
- **ڕێچکەی `pending_client_tool_calls`** — لە سەرەتا تا کۆتایی پەیوەست کرا بۆ ئەوەی بانگکردنی ئامرازەکانی لایەنی client بە دروستی تەواو بن و ئەنجامەکان بۆ مۆدێل بگەڕێننەوە.
- **درەوەری مێژوو** — گۆڕانکارییەکانی نەگەڕێنراوە لە لیستی گەڕاندنەوە دەردەکرێن؛ لینکی **بینینی مێژووی تەواو** ئێستا بە دروستی کار دەکات.
- **سیستەمی گۆڕانکاری/گەڕاندنەوە** — پێنج هەڵەی جیاواز چاک کرانەوە و لە ژێر ڕووکاری نوێی بەڕێوەبەردا یەکخران.
- **ئاگادارکردنەوەی پاشەکەوتکردنی ڕێکخستنەکان** — ئاگادارکردنەوەی snackbar ئێستا دوای کرتەکردن لە **پاشەکەوتکردنی ڕێکخستنەکان** بە شێوەیەکی پشتپێبەستراو دەردەکەوێت.
- **مێنیوی دەقوبەستی زبڵدان** — هەڵبژاردەی **سڕینەوەی هەمیشەیی** زیاد کرا بۆ ئەوەی ئایتمەکان بەبێ مانەوە لە دیمەنی زبڵداندا بتوانرێت بە تەواوی بسڕدرێنەوە.
- **دەستکاریکردن و دووبارە ناردن** — کرتەکردن لە ئایکۆنی دەستکاریکردن ئێستا تەنها پەیامی کرتەکراو دەباتە دۆخی دەستکاریکردن، نە هەموو پەیامەکانی ناو ڕیزەکە.
- **بەرزی نەخشەی چات** — پانێڵی چات بەرزییەکەی دەگونجێنێت کاتێک ناوەڕۆکی تێخراو لەلایەن پێوەکراوەوە (ئاگاداری بەڕێوەبەر، بانەرەکان) لە سەرووی پەڕەکە دەردەکەوێت، ڕێگری دەکات لەوەی ناوچەی تێکردن لە شاشە بدەرپەڕێنرێت.

## 1.4.0 — بڵاوکرایەوە لە 2026-04-09 {#140--released-on-2026-04-09}

### نوێ {#new-1}

- **WP-CLI benchmark command** (`wp gratis-ai-agent benchmark`) — کۆمەڵەی تاقیکردنەوەی Agent Capabilities v1 لە command lineەوە بۆ CI pipelines و ڕێڕەوەکانی هەڵسەنگاندنی مۆدێل جێبەجێ دەکات. پاڵپشتی جێبەجێکردنی بە پێی هەر پرسیارێک، گۆڕینی provider/model، و دەرچوونی JSON/CSV دەکات.
- **کۆمەڵەی تاقیکردنەوەی Agent Capabilities v1** — کۆمەڵەیەکی ڕێکخراو لە داواکارییە ئاڵۆز و چەند-هەنگاویەکانە کە تەواوی ڕووبەری تواناکان تاقی دەکاتەوە، لەگەڵ نمرەدان و ڕاپۆرتی token/ماوە.
- **تواناکانی Custom Post Type** — `register_post_type`, `list_post_types`, `delete_post_type`. تۆمارکردنەکان لە ڕێگەی خشتەی optionsەوە لە دوای دەستپێکردنەوەکان بەردەوام دەمێننەوە.
- **تواناکانی Custom Taxonomy** — `register_taxonomy`, `list_taxonomies`, `delete_taxonomy`. پاڵپشتی taxonomiesی پلەبەندی و تەخت دەکات لەگەڵ rewrite slugی هەڵبژاردەیی.
- **تواناکانی سیستەمی دیزاین** — `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset`. پێنج presetی دەستنیشانکراو لەخۆدەگرێت: `minimal-dark`, `warm-editorial`, `corporate-blue`, `vibrant-startup`, `classic-blog`.
- **تواناکانی Global Styles** — `get_global_styles`, `set_global_styles`, `reset_global_styles`. بەهاکانی theme.json لە ڕێگەی WordPress Global Styles APIەوە دەخوێنێتەوە و دەنووسێت.
- **توانای بەڕێوەبردنی مێنیوی ڕێنیشاندەر** — `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus`. پاڵپشتی مێنیوی ناوخۆیی و دیاریکردنی شوێنی theme دەکات.
- **توانای بەڕێوەبردنی Options** — `get_option`, `set_option`, `delete_option`, `list_options`. blocklistێکی ناوخۆیی پاراستن لەخۆدەگرێت کە optionsی گرنگی WordPress لە دەستکاریکردن دەپارێزێت.
- **Installable Abilities Registry** — `list_available_abilities`, `install_ability`, `recommend_plugin`. پاکێجی تواناکان بدۆزەرەوە و چالاکیان بکە کە وەک WordPress plugin دابەشکراون.
- **Site Builder Orchestration v2** — `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`. پلانی دروستکردنی سایت بە چەند قۆناغ، لەگەڵ دۆزینەوەی plugin، ئاماژەکانی دەرچوونی هەنگاو، بەدواداچوونی پێشکەوتن، و چاککردنەوەی هەڵە بە شێوەی خۆکار.
- **پرسیاری تاقیکردنەوەی ماڵپەڕی چێشتخانە** (`q-restaurant-website`) — تاقیکردنەوەی سەرەتادا-کۆتایی کە تۆمارکردنی CPT، سیستەمی دیزاین، ڕێنیشاندان، و دۆزینەوەی plugin دادەپۆشێت.
- **AI provider connector plugins** زیادکران بۆ WordPress Playground blueprints بۆ ئامادەکردنی خێراتری گەشەپێدانی ناوخۆیی.

### باشترکراوە {#improved-1}

- README نوێکرایەوە بە بەڵگەنامەی AI provider connector و ڕێنماییەکانی ئامادەکردن.

### چاککراوە {#fixed-1}

- 25 شکستی تاقیکردنەوەی PHPUnit لە لقەی `main` چارەسەرکران.
- فۆرماتی URLی GitHub releases لە `blueprint.json` ڕاستکرایەوە.
- ژمارەدانەوەی Task ID بۆ خۆپاراستن لە تێکەڵبوون لەگەڵ IDی کۆن.
