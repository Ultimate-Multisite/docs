# VAT

Multisite Ultimate: VATی ئەورووپی

تایبەتمەندی و ئامرازی زیاتر زیاد دەکات بۆ بەڕێوەبردنی باجی بنەڕەتی Multisite Ultimate بۆ پشتگیریی باشتر لە VATی ئەورووپی.
بە شێوەیەکی خۆکار ڕێژەکانی VATی EU زیاد دەکات بۆ پەڕەی ڕێژەکانی باج لە

## دامەزراندن {#installation}

1. 'ultimate-multisite-vat' باربکە بۆ بوخچەی '/wp-content/plugins/'
2. pluginەکە چالاک بکە لە ڕێگەی مێنیوی 'Plugins' لە WordPress
3. بچۆ بۆ Ultimate Multisite > Settings > Taxes و EU Vat چالاک بکە.
4. کلیک بکە لەسەر ڕێژەکانی باج و لە خوارەوە ڕێژەکانی EU VAT نوێ بکەوە و ڕێژەکانی باج پاشەکەوت بکە.
5. تەواو!
