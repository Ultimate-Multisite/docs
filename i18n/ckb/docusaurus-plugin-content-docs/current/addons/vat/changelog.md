# گۆڕانکارییەکانی VAT {#vat-changelog}

وەشان 1.0.7 - بڵاوکراوەتەوە لە 2026-02-03

* چاککردنەوە: ژمارەکانی VAT ـی ئەڵمانیا (DE) بەهۆی هەڵەکانی VIES SOAP service MS_MAX_CONCURRENT_REQ لە پەسەندکردندا شکستیان دەهێنا. لە SOAP API ـی کۆنەوە گوازرایەوە بۆ EU VIES REST API ـی فەرمی بۆ پەسەندکردنی باوەڕپێکراوتر.
* چاککردنەوە: ئێستا ژمارەکانی VAT پەسەند دەکرێن کاتێک VIES service بە شێوەی کاتی بەردەست نییە، لە جیاتی ئەوەی بە هەڵە ڕەت بکرێنەوە. تۆمارێک دروست دەکرێت بۆ پێداچوونەوەی دواتر.
* چاککردنەوە: خانەی VAT/Tax ID ئێستا بۆ هەموو وڵاتان دیارە، نەک تەنها ئەندامانی EU. Tax ID ـەکانی دەرەوەی EU (وەک ژمارەکانی Swiss CHE) بۆ پیشاندانی فاکتورە هەڵدەگیرێن بەبێ پەسەندکردنی VIES.
* چاککردنەوە: گەڕانی پێشگری VAT بۆ یۆنان (GR)، مۆناکۆ (MC)، و Isle of Man (IM) بەهۆی لۆژیکی هەڵەی گەڕانی array شکابوو.
* باشترکردن: پێویستیی گەشەپێدانی ناپێویستی wp-cli-bundle لابرا کە دەیتوانی کاتێک لەگەڵ پاشکۆی Site Exporter بەکاردەهێنرا هەڵەی کوشندە دروست بکات.
* باشترکردن: کۆمەڵە تاقیکردنەوەی یەکەیی گشتگیر زیادکرا (53 تاقیکردنەوە).

وەشان 1.0.6 - بڵاوکراوەتەوە لە 2026-01-25

* چاککردنەوە: مامەڵە ناوخۆییەکانی B2B ئێستا بە دروستی VAT وەردەگرن. بەپێی یاساکانی VAT ـی EU، reverse charge تەنها بۆ مامەڵە سنووردارە نێودەوڵەتییەکانی B2B دەگونجێت، نەک کاتێک وڵاتی کڕیار لەگەڵ وڵاتی کۆمپانیا یەکسانە.

وەشان 1.0.5 - بڵاوکراوەتەوە لە 2026-01-22

* چاککردنەوە: Addon بار نەدەبوو یان بە دروستی کار نەدەکرد.
* گۆڕان: سەرچاوەی داتای ڕێژەی VAT لە euvatrates.com گۆڕا بۆ کۆگای ibericode/vat-rates بۆ داتای باوەڕپێکراوتر و بە چالاکی پارێزراو.
* چاککردنەوە: هەڵەی نووسینی بەهای هەڵبژاردەی super_reduced_rates ڕاستکرایەوە.
* باشترکردن: پشکنینی null بۆ مامەڵەکردن لەگەڵ کۆدی وڵات زیادکرا بۆ ڕێگریکردن لە هەڵەکان.
* گۆڕان: پەڕگەکانی وەرگێڕانی هاوپێچکراو لابران لە پێناو نوێکردنەوەی خۆکار لە ڕێگەی Traduttore.
