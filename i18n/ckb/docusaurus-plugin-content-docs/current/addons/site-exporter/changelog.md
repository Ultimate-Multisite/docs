# گۆڕانکارییەکانی Site Exporter {#site-exporter-changelog}

وەشان: 1.0.6 - بڵاوکراوەتەوە لە 2026-05-11
* نوێ: هەناردەکانی ماڵپەڕ ئێستا index.php ـێکی خۆ-دەستپێکەر لەخۆدەگرن، بۆ ئەوەی ZIP بتوانرێت لەسەر خانەخوێیەکی تازە دابمەزرێت بەبێ دامەزراندنی plugin ـێکی جیاواز
* نوێ: هەناردەی تۆڕ ڕێگە بە بەڕێوەبەران دەدات هەموو ماڵپەڕە لاوەکییەکان لە ئەرشیفێکی تاکدا لە پەڕەی بەڕێوەبردنی هەناردەی ماڵپەڕەوە هەناردە بکەن

وەشان: 1.0.5 - بڵاوکراوەتەوە لە 2026-05-05
* چاککردنەوە: __callStatic زیادکرا بۆ WP_CLI polyfill stub بۆ مامەڵەکردن لەگەڵ ڕێبازە mock نەکراوەکان وەک get_config، و ڕێگری لە شکستی test bootstrap دەکات

وەشان: 1.0.4 - بڵاوکراوەتەوە لە 2025-11-25

* چاککردنەوەی گونجان لەگەڵ Ultimate Multisite 2.4.

* prefix بگۆڕە بۆ ultimate-multisite؛ text domain نوێ بکەرەوە؛ version bump.
