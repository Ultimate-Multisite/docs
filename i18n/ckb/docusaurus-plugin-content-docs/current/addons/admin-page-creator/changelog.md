### وەشانی 1.9.0 - بڵاوکرایەوە لە 2026-01-18 {#version-190---released-on-2026-01-18}

* چاککرا: هەڵە کوشندەکان لە کاتی چالاککردندا.
* ڕێکخرایەوە: پۆلەکانی سەرچاوەی ناوەڕۆک بۆ بەکارهێنانی Singleton trait لەگەڵ دەستپێکردنی گونجاوی parent constructor
* چاککرا: هەڵەکانی کۆمپایلکردنی ڕووکارەی Vue.js کە ڕێگرییان دەکرد هەڵبژێری جۆری ناوەڕۆک پیشان بدرێت
* چاککرا: تایبەتمەندییەکانی داتای Vue کە ونبوون (template_id, external_link_url_embedable, external_link_url_checking)
* چاککرا: دەستکاریکەری TinyMCE بە شێوەیەکی دروست لە کاتی بارکردنی پەڕەدا دەستپێنەدەکرد
* چاککرا: فایلی پشتگیری Divi بەهۆی Composer ـەوە خۆکار بارنەدەکرا
* چاککرا: ڕستەسازی PHP تێکچوو لە پۆلی پشتگیری Divi دا
* باشترکرا: ڕێکخستنی سەرمایەکانی JavaScript و بچووککردنەوەیان
* باشترکرا: گونجاندن لەگەڵ نوێترین وەشانەکانی WordPress و دروستکەری پەڕە
