# دروستکەری پەڕەی بەڕێوەبەرایەتی {#admin-page-creator}

## ناوەڕۆک {#content}
دروستکەری پەڕەی بەڕێوەبەرایەتی ڕێگەت پێدەدات بە ئاسانی پەڕەی بەڕێوەبەرایەتی و ژێرپەڕەی نوێ لەناو Dashboard ـی بەڕێوەبەرایەتی WordPress زیاد بکەیت. دەتوانرێت بۆ دروستکردنی پەڕەی بەخێرهاتن یان پەڕەی فێرکاری بەکاربهێنرێت کە وەک پەڕەی نیشتنەوە بۆ بەکارهێنەرانی نوێ کار بکات. دەتوانیت دروستکەری پەڕە وەک Beaver Builder، Elementor، Brizy، و Oxygen بەکاربهێنیت بۆ دیزاینکردنی پەڕەکان بەبێ کێشە.

هەروەها پشتگیری HTML، CSS و JavaScript ـی تایبەت پێشکەش دەکات، هەروەها کۆنترۆڵی دەستپێگەیشتن لەسەر بنەمای پلە و ڕۆڵی بەکارهێنەر!

## تایبەتمەندییەکان {#features}
* پشتگیری HTML و CSS ـی تایبەت
* پشتگیری JavaScript
* دۆخی ناوەڕۆکی PHP
* مۆڵەت لەسەر بنەمای ڕۆڵی بەکارهێنەر
* پشتگیری بۆ دروستکەرەکانی پەڕە

## بەڵگەنامە {#documentation}
[دروستکردنی پەڕەی بەڕێوەبەرایەتی و ژێرپەڕەی تایبەت بە بەکارهێنانی Multisite Ultimate: دروستکەری پەڕەی بەڕێوەبەرایەتی](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## دامەزراندن {#installation}

1. **ultimate-multisite-admin-page-creator** باربکە بۆ بوخچەی **/wp-content/plugins/**
2. plugin ـەکە لە ڕێگەی مێنیوی **پێوەکراوەکان** لە WordPress Network Admin ـەکەت چالاک بکە

## ڕێبەری بەکارهێنان {#usage-guide}

لە کاتێکدا WordPress زیادکردنی ناوەڕۆک بۆ بەشی پێشەوەی ماڵپەڕەکان زۆر ئاسان دەکات، دروستکردنی پەڕە و ژێرپەڕە لە لای بەڕێوەبەرایەتی پێویستی بە هەندێک شارەزایی بنەڕەتی کۆدنووسین هەیە. Ultimate Multisite: دروستکەری پەڕەی بەڕێوەبەرایەتی ئامانجی ئەوەیە هەمان ئاسانیی بەکارهێنانی بەشی پێشەوە بهێنێتە بەشی پشتەوەی ماڵپەڕەکانی تۆڕەکەت.

## بۆچی دروستکەری پەڕەی بەڕێوەبەرایەتی بەکاربهێنین؟ {#why-use-admin-page-creator}

لەوانەیە بتەوێت لە پانێڵی بەڕێوەبەرایەتی کڕیارەکانتدا پەڕەیەکی تایبەت پێشکەش بکەیت کە فێرکارییەکانی تێدا بێت، یان لەوانەیە بتەوێت پەڕەیەکی تایبەت زیاد بکەیت کە پرسیارە دووبارەکان و ویجێتی پەیوەندی تێدا بێت. ئەگەرەکان لەگەڵ دروستکەری پەڕەی بەڕێوەبەرایەتی بێکۆتاییە، و ئەمە ئەو شتەیە کە لە فێرکاری ئەمڕۆدا دەیگەڕێینەوە.

![لیستی پەڕەکانی بەڕێوەبەرایەتی](/img/admin/admin-pages-list.png)

_دروستکردنی پەڕەی فێرکارییەکان لەگەڵ لیستی پەخشی YouTube ـی چێندراو ئێستا گونجاوە!_

_ئەمانە ئەنجامی کۆتاییە لەسەر پانێڵی Dashboard ـی بەکارهێنەر_

## دروستکردنی یەکەم پەڕەی بەڕێوەبەرایەتی تایبەتی خۆت {#creating-your-first-custom-admin-page}

دوای دامەزراندن و چالاککردنی Ultimate Multisite: دروستکەری پەڕەی بەڕێوەبەرایەتی، بڕۆ بۆ مێنیوی بەڕێوەبەرایەتی تۆڕەکەت و بڕۆ بۆ پەڕەکانی بەڕێوەبەرایەتی.

![لیستی پەڕەکانی بەڕێوەبەرایەتی تایبەت لەسەر بەڕێوەبەرایەتی تۆڕ](/img/admin/admin-pages-list.png)

_لیستی پەڕەکانی بەڕێوەبەرایەتی تایبەت لەسەر بەڕێوەبەرایەتی تۆڕ_

لیستی هەموو ئەو پەڕە بەڕێوەبەرایەتییانە دەبینیت کە تا ئێستا دروستت کردوون (لەبەر ئەوەی ئەمە یەکەم سەردانتە، لیستەکە بەتاڵ دەبێت). بەردەوام بە و کرتە لە **زیادکردنی پەڕەی بەڕێوەبەرایەتی نوێ** بکە. ڕەوانەی دەستکاریکەری پەڕەی بەڕێوەبەرایەتی دەکرێیت.

## توخمەکانی پەڕەی بەڕێوەبەرایەتی تایبەت {#the-elements-of-a-custom-admin-page}

لەسەر دەستکاریکەری پەڕەی بەڕێوەبەرایەتی، ژمارەیەک هەڵبژاردەی جیاوازت دەبێت بۆ دەستکاریکردنی پەڕەکەت.

### ناونیشانی پەڕە {#page-title}

ئەمە ئەو ناونیشانەیە کە لە سەرەوەی پەڕەکە پیشان دەدرێت، نە پێناسەی مێنیوی بەڕێوەبەرایەتی.

<!-- وێنەی شاشە بەردەست نییە: دەستکاریکەری پەڕەی بەڕێوەبەرایەتی کە خانەی ناونیشانی پەڕە پیشان دەدات -->

_ناونیشانی پەڕە داخڵ بکە_

### هەڵبژێری جۆری ناوەڕۆک {#content-type-selector}

Ultimate Multisite: دروستکەری پەڕەی بەڕێوەبەرایەتی دوو هەڵبژاردەت پێدەدات کاتێک دەگاتە ناوەڕۆکی پەڕەکە: یان دەتوانیت دەستکاریکەری بنەڕەتی WYSIWYG ـی WordPress بەکاربهێنیت یان بگۆڕیت بۆ دەستکاریکەری HTML.

ئەگەر شارەزای HTML نیت، لەسەر دەستکاریکەری بنەڕەتی WordPress بمێنەوە. دەتوانیت ناوەڕۆکی پەڕەکە وەک ئەوەی لەگەڵ بابەتێکی ئاسایی یان پەڕەیەکی ئاسایی دەیکەیت دەستکاری بکەیت.

_بەکارهێنانی دەستکاریکەری بنەڕەتی WYSIWYG ـی WordPress_

<!-- وێنەی شاشە بەردەست نییە: دروستکەری پەڕەی بەڕێوەبەرایەتی کە دەستکاریکەری HTML بە نیشاندانی ڕستەسازی بەکاردەهێنێت -->

_بەکارهێنانی هەڵبژاردەی دەستکاریکەری HTML، لەگەڵ نیشاندانی ڕستەسازی و ئاگادارکردنەوەکانی هەڵە_

## هەڵبژاردەکانی مێنیو {#menu-options}

لە لای تەنیشت، لەسەر meta-box ـی هەڵبژاردە گشتییەکان، دەتوانیت جۆری مێنیو، پێناسە، و زۆر شتی تر ڕێکبخەیت.

### جۆرەکانی مێنیو {#menu-types}

پەڕەی بەڕێوەبەرایەتی تایبەتی نوێت دەتوانێت دوو جۆری مێنیوی جیاوازی هەبێت: مێنیوی ئاستی سەرەوە و ژێرمێنیو.

پەڕەی مێنیوی ئاستی سەرەوە، وەک ناوەکە پێشنیاری دەکات، پەڕە نوێ دروستکراوەکەت زیاد دەکات بۆ مێنیوی بەڕێوەبەرایەتی لەگەڵ پەڕە بنەڕەتییەکانی WordPress.

ئەگەر هەڵبژاردەی ئاستی سەرەوە هەڵبژێریت، هەروەها دەتوانیت ڕیزی مێنیو و ئایکۆنی مێنیو هەڵبژێریت.

ئەگەر بتەوێت ئەم پەڕەیە بکەیتە پەڕەی ژێرمێنیو، دەتوانیت پەڕەی باوک هەڵبژێریت بۆ پەیوەستکردنی ئەم پەڕەیە پێی. دەتوانیت پەڕەی تایبەتی خۆت زیاد بکەیت بۆ پەڕە بنەڕەتییەکانی WordPress یان بۆ پەڕە بەڕێوەبەرایەتییە تایبەتەکانی ئاستی سەرەوەی خۆت.

<!-- وێنەی شاشە بەردەست نییە: هەڵبژاردەکانی جۆری مێنیوی دروستکەری پەڕەی بەڕێوەبەرایەتی لەگەڵ ڕێکخستنەکانی پێناسە، ڕیز، و ئایکۆن -->

_ئایتمی مێنیو بۆ ئەم پەڕەیەی تایبەت بە تەواوی دەستکاری بکە_

<!-- وێنەی شاشە بەردەست نییە: هەڵبژاردنی پەڕەی باوکی ژێرمێنیوی دروستکەری پەڕەی بەڕێوەبەرایەتی -->

_هەروەها دەتوانیت ئەم پەڕەیە وەک ژێرمێنیوی پەڕە مێنیویەکانی تر زیاد بکەیت، بەوانەی ئاستی سەرەوەی تایبەت بە خۆشتەوە._

## هەڵبژاردە پێشکەوتووەکان {#advanced-options}

دەبێت بتوانیت هەموو جۆرە ناوەڕۆکێک زیاد بکەیت بۆ پەڕە بەڕێوەبەرایەتییە تایبەتەکانت. ئەمە یاساکانی CSS و فایلەکان، JavaScript، و کتێبخانە دەرەکییەکانیش دەگرێتەوە. دەتوانیت ئەمە بە بەکارهێنانی تابی هەڵبژاردە پێشکەوتووەکان بکەیت.

### CSS ـی تایبەت {#custom-css}

لەسەر تابی CSS، دەتوانیت کۆدی CSS ـی تایبەتی خۆت زیاد بکەیت، هەروەها فایلەکانی CSS لە ماڵپەڕە دوورەکانەوە هاوردە بکەیت (وەک CDN ـەکان، Google Fonts، هتد).

<!-- وێنەی شاشە بەردەست نییە: دەستکاریکەری CSS ـی تایبەتی دروستکەری پەڕەی بەڕێوەبەرایەتی لەگەڵ هاوردەکردنی فایلی شێوازنامەی دەرەکی -->

_CSS ـی تایبەتیش پشتگیری دەکرێت. هەروەها دەتوانیت فایلەکانی شێوازی دەرەکی بخەیتە ناوەوە!_

### JavaScript ـی تایبەت {#custom-javascript}

هەمان شت بۆ JavaScript دروستە. دەتوانیت کتێبخانە دەرەکییەکان زیاد بکەیت بۆ بەکارهێنان لە کۆدی JavaScript ـی تایبەتی خۆت.

<!-- وێنەی شاشە بەردەست نییە: دەستکاریکەری JavaScript ـی تایبەتی دروستکەری پەڕەی بەڕێوەبەرایەتی لەگەڵ هاوردەکردنی کتێبخانەی دەرەکی -->

_کۆدی JavaScript ـی تایبەتی خۆت زیاد بکە!_

## مۆڵەتەکان {#permissions}

هەروەها لە هەڵبژاردە پێشکەوتووەکاندا، دەتوانیت هەڵبژێریت کام ژێرماڵپەڕەکان دەستپێگەیشتنیان بە ئەم پەڕەیە دەبێت. ئەمە بەسوودە ئەگەر پلان دابنێیت ناوەڕۆکی تایبەت و جیاواز پێشکەش بە پلەکانی پلانی جیاواز بکەیت، بۆ نموونە.

<!-- وێنەی شاشە بەردەست نییە: ڕێکخستنەکانی مۆڵەتی دروستکەری پەڕەی بەڕێوەبەرایەتی بۆ کۆنترۆڵی دەستپێگەیشتن لەسەر بنەمای پلان و ڕۆڵ -->

_کۆنترۆڵی تەواوت هەیە لەسەر ئەوەی ئەم پەڕەیە لە کوێ دەردەکەوێت_

**گرنگ** : هەردوو مەرجەکە دەبێت جێبەجێ بکرێن بۆ ئەوەی پەڕەکە پیشان بدرێت. ئەمە واتە ئەگەر بەکارهێنەرێک کڕیاری پلانێکی دیاریکراو بێت، بەڵام یەکێک لە ڕۆڵە هەڵبژێردراوەکانی نەبێت (یان پێچەوانە)، پەڕەکە لە پەنێڵەکەیاندا دەرناکەوێت.

## چالاک {#active}

ئەگەر دەتەوێت پەڕەیەکی custom admin بەبێ سڕینەوەی یان دەستکاریکردنی مۆڵەتەکان ناچالاک بکەیت، تەنها هەڵبژاردەی Active لە نیشانەکردن لاببە.

ناچالاککردنی پەڕەیەکی ئاستی سەرەوە بە شێوەیەکی ئۆتۆماتیکی هەموو لاپەڕە لاوەکییە پەیوەستەکان بەو پەڕەی باوانە ناچالاک دەکات.
