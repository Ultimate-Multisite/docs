# پلانی پێوانەکراو

ئەم addon ـەی Ultimate Multisite: Metered Plans شێوازی پارەوەرگرتنت لە کڕیارەکان دەگۆڕێت بە ناساندنی تواناکانی پارەوەرگرتنی پێوانەکراوی نەرم و گونجاو. لەبری سنوورە توند و نەگۆڕەکان کە کاتێک بەکارهێنەر تێیان دەپەڕێن ڕێگرییان لێ دەکەن، کرێی زیادە بەکاربهێنە کە بە شێوەی ئۆتۆماتیکی پارەی بەکارهێنانی زیادە لە کڕیارەکان وەردەگرێت.

## تایبەتمەندییە سەرەکییەکان {#key-features}

* **پارەوەرگرتنی پێوانەکراو**: پارە لە کڕیارەکان وەربگرە لەسەر بنەمای بەکارهێنانی ڕاستەقینە نەک ئاستە جێگیرەکان
* **بەڕێوەبردنی زیادەبەکارهێنان**: بە شێوەی ئۆتۆماتیکی پارەی ئەو بەکارهێنانە وەربگرە کە لە سنوورەکانی پلان زیاترە
* **شوێنکەوتنی بەکارهێنان**: خەرجکردنی سەرچاوەی کڕیار لە کاتی ڕاستەقینەدا چاودێری بکە
* **نرخدانانی نەرم**: نرخە تایبەتەکان بۆ سەرچاوە جیاوازەکان دابنێ (هەڵگرتن، باندویدث، ماڵپەڕەکان، هتد.)
* **پارەوەرگرتنی AI Token**: token ـەکانی input و output بۆ بەکارهێنانی AI لە subsite ـەکانی کڕیاردا بشوێنەوە و پارەیان لێ وەربگرە
* **دروستکردنی پسووڵەی ئۆتۆماتیکی**: کاتێک سنوورەکانی بەکارهێنان تێدەپەڕدرێن، پسووڵە بە شێوەی ئۆتۆماتیکی دروست بکە
* **ڕاپۆرتەکانی بەکارهێنان**: شیکاری و ڕاپۆرتی وردی بەکارهێنان بۆ کڕیاران و بەڕێوەبەران

## گونجاوە بۆ {#perfect-for}

* **دابینکەرانی خانەخوێکردن**: پلانە گەشەسەندووەکان پێشکەش بکە کە لەگەڵ پێویستییەکانی کڕیار گەشە دەکەن
* **سەکۆکانی SaaS**: پارەوەرگرتن لەسەر بنەمای خەرجکردنی ڕاستەقینەی سەرچاوە
* **تۆڕەکانی ئاژانس**: مۆدێلە نەرمەکانی نرخدانان بۆ پڕۆژەکانی client
* **خانەخوێکردنی فرۆشیاری دووبارە**: هاوبەشان توانادار بکە نرخدانانی پشتبەستوو بە بەکارهێنان پێشکەش بکەن

## چۆن کار دەکات {#how-it-works}

1. سەرچاوە پێوانەکراوەکان ڕێکبخە (بۆشایی دیسک، باندویدث، ژمارەی ماڵپەڕەکان، هتد.)
2. سنوورەکانی پلانی بنەڕەتی و نرخەکانی زیادەبەکارهێنان دابنێ
3. سیستەمەکە بە شێوەی ئۆتۆماتیکی بەکارهێنان لە سەرانسەری تۆڕەکەتدا شوێن دەکەوێت
4. کاتێک کڕیاران لە سنوورەکان تێدەپەڕن، کرێی زیادەبەکارهێنان هەژمار دەکرێت
5. پسووڵەکان بە شێوەی ئۆتۆماتیکی لەسەر بنەمای سووڕی پارەوەرگرتن دروست دەکرێن
6. کڕیاران دەتوانن بەکارهێنان و کرێی داهاتوویان لە کاتی ڕاستەقینەدا ببینن

## پارەوەرگرتنی AI Token {#ai-token-billing}

وەشانی 1.1.0 پارەوەرگرتنی AI token زیاد دەکات بۆ ئەو تۆڕانەی تایبەتمەندییەکانی AI لە ڕێگەی WordPress AI Client SDK پێشکەش دەکەن. Metered Plans گوێ لە داواکارییە تەواوبووەکانی دروستکردنی AI دەگرێت، ژمارەی token ـەکانی input و output بۆ subsite ـی ئێستا تۆمار دەکات، و ئەو token ـانە دەگۆڕێت بۆ خەرجی AI ـی پارەوەرگیراو بۆ ماوەی پارەوەرگرتنی ئێستای کڕیار.

پارەوەرگرتنی AI token بەکاربهێنە کاتێک دەتەوێت ئامرازەکانی AI لەسەر ماڵپەڕەکانی کڕیار پێشکەش بکەیت بەبێ ئەوەی تێچووی مۆدێلی پێشبینی‌نەکراو هەڵبگریت. هەر داواکارییەک دەخرێتە سەر subsite، کڕیار، ئەندامێتی، دابینکەر، و مۆدێل بۆ ئەوەی بەڕێوەبەران بتوانن بەکارهێنان وردبینی بکەن، لە کاتێکدا کڕیاران دەتوانن ببینن خەرجکردنی AI ـیان چۆن کاریگەری لەسەر account ـەکەیان دەکات.

### ڕێکخستنی نرخەکان بۆ هەر Token {#configure-per-token-rates}

نرخەکانی AI بۆ هەر مۆدێل وەک نرخە جیاوازەکانی input و output ڕێکدەخرێن. نرخەکان بۆ هەر ١ ملیۆن token هەژمار دەکرێن، کە لەگەڵ پەڕەکانی نرخدانانی دابینکەرانی باوی AI دەگونجێت.

1. لە بەڕێوەبەری تۆڕدا، **Multisite Ultimate → Settings → Metered Plans** بکەرەوە.
2. پێوانەکردنی خەرجی AI بۆ ئەو بەرهەم یان ئەندامێتییە چالاک بکە کە دەبێت پارەی بەکارهێنانی AI وەربگرێت.
3. تۆمارەکانی نرخی مۆدێل زیاد بکە بە بەکارهێنانی ID ـی مۆدێلەکان کە لەلایەن connector ـەکەت دەگەڕێندرێنەوە، وەک `gpt-4o`، `gpt-4o-mini`، یان ناسێنەری مۆدێلێکی دیکەی هاوگونجاو لەگەڵ OpenAI.
4. بەها جیاوازەکانی **نرخی Input** و **نرخی Output** بە دراوی ماڵپەڕەکەت بۆ هەر ١ ملیۆن token بنووسە.
5. ڕێگەپێدانی خەرجی AI ـی لەخۆگرتووی کڕیار و هەڵسوکەوتی زیادەبەکارهێنان بۆ پلانەکە دابنێ.

کاتێک ID ـی وردی مۆدێل ڕێکنەخراوە، Metered Plans دەتوانێت بە پێشگری مۆدێل هاوتاکردن بکات. بۆ نموونە، نرخێک کە بۆ `claude-sonnet-4` ڕێکخراوە دەتوانێت بۆ ID ـی مۆدێلە کات‌مۆرکراوەکان کاربکات کە بەو پێشگرە دەست پێ دەکەن. پەرەپێدەران هەروەها دەتوانن خشتەی کۆتایی نرخەکان بە فلتەری [`wu_ai_token_rates`](./hooks/Filters/wu_ai_token_rates) ڕێکبخەن.

### AI Connector ـە پشتگیریکراوەکان {#supported-ai-connectors}

پارەوەرگرتنی AI token لەگەڵ connector ـەکان کار دەکات کە ڕووداوەکانی دروستکردنی WordPress AI Client SDK دەنێرن و بەکارهێنانی token لە metadata ـی وەڵامدا لەخۆدەگرن. ئەمە خێزانی Ultimate Multisite AI connector لەخۆدەگرێت کە لێرەدا بەڵگەنامەکراون:

* [Ultimate AI Connector بۆ Endpoint ـە هاوگونجاوەکان](../ultimate-ai-connector-compatible-endpoints/) بۆ API ـە هاوگونجاوەکانی OpenAI، لەوانە OpenAI، Mistral، دەروازە هاوگونجاوەکانی Ollama، و دابینکەرانی دیکە کە metadata ـی بەکارهێنانی token ـی هاوگونجاو پیشان دەدەن
* [Ultimate AI Connector بۆ WebLLM](../ultimate-ai-connector-webllm/) بۆ بەکارهێنانی WebLLM ـی ڕەسەنی وێبگەڕ کاتێک داواکارییەکان لە ڕێچکەی ڕووداوی AI Client SDK ـەوە ئاراستە دەکرێن
* connector ـەکانی دابینکەری AI کە لەلایەن [Gratis AI Agent](../gratis-ai-agent/) بەکاردەهێنرێن کاتێک دابینکەری هەڵبژێردراو ژمارەی token ـەکانی prompt و completion لە ڕێگەی SDK ڕاپۆرت دەکات

connector ـە دەستکردەکان دەتوانن بە بەردانەوەی ڕووداوە ستانداردەکانی `wp_ai_client_before_generate_result` و `wp_ai_client_after_generate_result` بە metadata ـی مۆدێل، دابینکەر، و بەکارهێنانی token بەشدار بن. ڕێکخستنەکانی connector لە ماڵپەڕی سەرەکییەوە بۆ subsite ـەکان جێبەجێ دەکرێن بۆ ئەوەی ماڵپەڕەکانی کڕیار دابینکەر و سنوورە پەسەندکراوەکانی تۆڕ بەکاربهێنن.

### بەکارهێنانی Account ـی کڕیار {#customer-account-usage}

کڕیاران بەکارهێنانی AI لەگەڵ سەرچاوە پێوانەکراوەکانی دیکەیان لە ناوچەی account ـەکەیاندا دەبینن. ڕیزی خەرجی AI کۆی گشتی ماوەی پارەوەرگرتنی ئێستا پیشان دەدات کە لە token ـە تۆمارکراوەکانی prompt و completion، نرخە ڕێکخراوەکانی مۆدێل، و هەر ڕێگەپێدانێکی لەخۆگرتوو لەسەر پلانەکەوە هاتووە.

ئەگەر کڕیارێک شێوازی پارەدانێکی لە فایلدا هەبێت، بەکارهێنانی سەرووی ڕێگەپێدانی لەخۆگرتوو وەک زیادەبەکارهێنانی پارەوەرگیراو مامەڵەی لەگەڵ دەکرێت. ئەگەر پلانەکە بۆ ڕاگرتنێکی توند ڕێکخرابێت و کڕیار هیچ شێوازی پارەدانێکی بەردەست نەبێت، داواکارییەکانی AI دەتوانرێت قەدەغە بکرێن کاتێک ڕێگەپێدانەکە دەگاتە سنوور، و لە کڕیار داوا دەکرێت پێش بەردەوامبوون وردەکارییەکانی پارەدان زیاد بکات.

## سەرچاوە پشتگیریکراوەکان {#supported-resources}

* هەڵگرتنی دیسک
* باندویدث/گواستنەوەی داتا
* ژمارەی ماڵپەڕەکان
* ژمارەی بەکارهێنەران
* بەکارهێنانی AI token و خەرجی AI
* جۆرە دەستکردەکانی سەرچاوە

## پێداویستییەکان {#requirements}

* Multisite Ultimate 2.0.0 یان بەرزتر
* دامەزراندنی WordPress Multisite
* PHP 7.4 یان بەرزتر

## دامەزراندن {#installation}

1. فایلەکانی plugin بەرز بکەرەوە بۆ `/wp-content/plugins/ultimate-multisite-metered-plans`
2. plugin ـەکە لە ڕێگەی شاشەی 'Plugins' لە WordPress چالاک بکە
3. بڕۆ بۆ Multisite Ultimate → Settings → Metered Plans
4. سەرچاوە پێوانەکراوەکان و نرخدانانەکەت ڕێکبخە
5. پارەوەرگرتنی پێوانەکراو بۆ بەرهەمەکان لە دەستکاریکەری بەرهەمدا چالاک بکە

## پرسیارە زۆر دووبارەکان {#frequently-asked-questions}

## چی ڕوودەدات کاتێک کڕیارێک لە سنوورەکەی تێدەپەڕێت؟ {#what-happens-when-a-customer-exceeds-their-limit}

لەبری بلۆککردنی دەستگەیشتن، سیستەمەکە زیادەبەکارهێنانەکە تۆمار دەکات و بەپێی ئەو نرخانەی ڕێکت خستوون، دەیخاتە سەر وەسڵی داهاتوویان.

## دەتوانم نرخی زیادەبەکارهێنانی جیاواز بۆ بەرهەمە جیاوازەکان دابنێم؟ {#can-i-set-different-overage-rates-for-different-products}

بەڵێ، هەر بەرهەمێک دەتوانێت سەرچاوەی پێوانەکراوی تایبەت و نرخدانانی زیادەبەکارهێنانی تایبەتی هەبێت.

## کڕیاران چۆن ئاگادار دەکرێنەوە دەربارەی بەکارهێنان؟ {#how-are-customers-notified-about-usage}

کڕیاران دەتوانن بەکارهێنانی ئێستای خۆیان لە Dashboardی خۆیاندا ببینن و ئاگادارکردنەوەی ئارەزوومەندانە وەربگرن کاتێک نزیک دەبنەوە لە سنوورەکان یان لێیان تێدەپەڕن.

## بەکارهێنانی AI token چۆن نرخ دادەنرێت؟ {#how-is-ai-token-usage-priced}

بەکارهێنانی AI token لەسەر بنەمای نرخە ڕێکخراوەکانی هاتنەژوورەوە و دەرچوونی مۆدێلەکە بۆ هەر ١ ملیۆن token نرخ دادەنرێت. پلانی پێوانەکراو ژمارەی tokenەکانی prompt و completionی هەر داواکارییەک بەو نرخانە زەرب دەکات، ئەنجامەکە دەخاتە سەر خەرجی AIی ماوەی billingی ئێستای کڕیارەکە، و ڕێساکانی ڕێگەپێدان یان زیادەبەکارهێنانی پلانەکە جێبەجێ دەکات.

## هێشتا دەتوانم سنووری توند بەکاربهێنم؟ {#can-i-still-use-hard-limits}

بەڵێ، billingی پێوانەکراو بۆ هەر بەرهەمێک ئارەزوومەندانەیە. دەتوانیت بەرهەمەکانی سنووری توندی نەریتی لەگەڵ بەرهەمە پێوانەکراوەکان تێکەڵ بکەیت.
