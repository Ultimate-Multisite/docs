# تۆماری گۆڕانکاریی پلانە پێوانەکراوەکان

وەشان 1.1.0 - بڵاوکراوەتەوە لە 2026-05-05
- نوێ: پسووڵەکردنی AI token بۆ ژێر-سایتەکانی multisite — بەکارهێنانی AI token لە سەرانسەری سایتەکانی کڕیاردا بەدوادا بچۆ و پسووڵەی بکە، لەگەڵ نرخەکانی بۆ هەر tokenێک کە دەتوانرێت ڕێکبخرێن
- نوێ: جێبەجێکردنی گرێدەرەکان لە نوێوە دروستکرایەوە بە دۆزینەوەی سنووری دینامیکی و نووسینی ڕاستەوخۆ، کە وردیی کاتی ڕاستەقینە لە سەرجەم گرێدەرەکاندا دڵنیا دەکات
- چاککراوە: نوێکردنەوەکانی خشتەی داتابەیس ئێستا بە دروستی لەگەڵ پێناسەکانی schemaی BerlinDB هاوتا کراون، کە ڕێگری لە شکستی نوێکردنەوە لە دامەزراندنە نوێیەکاندا دەکات
- چاککراوە: callbackەکانی نوێکردنەوەی داتابەیس گۆڕان بۆ فۆرماتە دروستەکە کراون، کە شکستی بێدەنگی نوێکردنەوە چارەسەر دەکات
- چاککراوە: نرخە پارچەییەکان ئێستا لە خانەی داخڵکردنی AI Usage Overage Markup وەردەگیرێن
- چاککراوە: هەڵە کوشندەکان و کێشەکانی دەستپێکردنەوەی دووبارە لە کاتی بارکردنی plugin چارەسەر کران
- باشترکراوە: npm scriptی check-env زیادکرا بۆ ئەوەی ژینگەکانی گەشەپێدەر لە یەکەم جار کارپێکردندا خۆکارانە ڕێکبخرێن
