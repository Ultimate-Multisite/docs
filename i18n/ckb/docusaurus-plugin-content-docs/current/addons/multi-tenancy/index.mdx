# فرە-کرێداری

ژێرماڵپەڕەکان لەسەر ڕاژەکار و بنکەدراوەکان دابەش بکە بۆ کارایی و ئاسایش

## دیارترین خاڵەکانی بڵاوکردنەوەی وەشانی 1.2.0 {#version-120-release-highlights}

فرە-کرێداری 1.2.0 بنەمای کرێداری سەربەخۆ زیاد دەکات بۆ ڕەوتەکانی جیاکردنەوەی بەرهەمهێنان:

- چوونەژوورەوەی خۆکاری کرێداری بێ‌دۆخ لەگەڵ سەردانەکانی SSO کورت‌ماوە و پارێزراو لە دووبارە-بەکارهێنانەوە.
- جیاکردنەوەی بنکەدراوە و سیستەمی فایل بۆ هەر ژێرماڵپەڕێک بۆ کرێدارە سەربەخۆکان.
- توانای فرە-کرێداری Hostinger بۆ دابینکردنی کرێدار بە ئاگاداریی خانەخوێ.
- دەروازەکانی پشتڕاستکردنەوەی کۆچکردنی سەربەخۆ بۆ پشکنینی داتای کۆن و پرۆسێسی پاڵنانی async.
- بەستنی خانەخوێی بنکەدراوەی هەمان-ئامێر کە دەتوانرێت ڕێکبخرێت بۆ دامەزراندنەکانی Bedrock، FrankenPHP، و localhost.
- باشترکردنەکانی دەستپێکردنی کرێدار، نووسەری بنکەدراوە، دەستپێکردنی schema، و دابینکردنی بەکارهێنەر.
- لابردنی سەربەخۆی سەلامەتتر کە باوەڕپێکراویەکانی کرێدار پاک دەکاتەوە.

سەیری ڕێبەرەکانی بەکارهێنەر بکە بۆ [ڕەوتی کۆچکردنی سەربەخۆ](/user-guide/getting-started/sovereign-migration-workflow)، [جیاکردنەوەی فرە-کرێداری](/user-guide/configuration/multi-tenancy-isolation)، و [کردارەکانی کرێداری سەربەخۆ](/user-guide/administration/sovereign-tenant-operations).

## پێداویستییەکان {#requirements}

- WordPress 5.3 یان بەرزتر
- PHP 7.4 یان بەرزتر
- پێوەکراوی Multisite Ultimate (چالاک)

## دامەزراندن {#installation}

1. فایلەکانی پاشکۆکە باربکە بۆ بوخچەی `/wp-content/plugins/` ـەکەت
2. پێوەکراوەکە لە ڕێگەی مێنیوی 'پێوەکراوەکان' لە WordPress چالاک بکە
3. ڕێکخستنەکانی پاشکۆکە لە بەڕێوەبەری تۆڕ ڕێکبخە

## گەشەپێدان {#development}

### ئامادەکردن {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### تاقیکردنەوە {#testing}

پاشکۆکە چوارچێوەیەکی تاقیکردنەوەی گشتگیر لەخۆدەگرێت:

- **تاقیکردنەوەی یەکە**: تاقیکردنەوە بنیاتنراوەکان لەسەر PHPUnit لەگەڵ کۆمەڵەی تاقیکردنەوەی WordPress
- **پۆلی بنەڕەتی تاقیکردنەوە**: میتۆدە یارمەتیدەرەکان بۆ سیناریۆ باوەکانی تاقیکردنەوە دابین دەکات
- **داپۆشینی تاقیکردنەوە**: ڕێکخراوە بۆ بەدواداچوونی داپۆشینی کۆد
- **تاقیکردنەوەی فرەماڵپەڕ**: تاقیکردنەوەکان لە ژینگەی فرەماڵپەڕدا بەڕێوەدەچن

### ستانداردەکانی کۆد {#code-standards}

- **ستانداردەکانی کۆدنووسی WordPress**: لە ڕێگەی PHPCS جێبەجێ دەکرێت
- **گونجانی PHP 7.4+**: تایبەتمەندییە نوێکانی PHP پشتگیری دەکرێن
- **شیکاریی جێگیر**: تێکەڵکردنی PHPStan بۆ پشکنینی جۆر
- **نوێکردنەوەی کۆد**: یاساکانی Rector بۆ بەرزکردنەوەکانی PHP

### سیستەمی دروستکردن {#build-system}

- **پرۆسێسکردنی سامان**: بچووککردنەوە بۆ فایلەکانی CSS/JS
- **وەرگێڕان**: دروستکردنی فایلی POT
- **پاکێجکردن**: دروستکردنی ئەرشیف بۆ دابەشکردن
- **دۆخی گەشەپێدان**: سامانە بچووک‌نەکراوەکان بۆ هەڵەدۆزینەوە

## ڕێکخستن {#configuration}

پاشکۆکە پشتگیری هەڵبژاردە ڕێکخستنە جیاوازەکان دەکات کە لە ڕووکارەکەی بەڕێوەبەری تۆڕەوە دەستپێگەیشتوون.

### ڕێکخستنی کرێداری سەربەخۆ {#sovereign-tenant-configuration}

بۆ کرێدارە سەربەخۆکان، بنکەدراوەی کرێدار، ڕەگی سیستەمی فایل، بەستنی خانەخوێی بنکەدراوە، و توانای خانەخوێ ڕێکبخە پێش ئەوەی هاتوچۆ ئاراستەی کرێدار بکەیت. دوای هەر گۆڕانێک کە کاریگەری لەسەر باوەڕپێکراویەکانی بنکەدراوە، بەستنی خانەخوێ، ئاراستەکردن، یان دابینکردنی بەکارهێنەر هەیە، پشتڕاستکردنەوەی کۆچکردن بەڕێوەبەرە.

تەنها کاتێک `localhost` بەکاربهێنە کە کاتی جێبەجێکردنی کرێدار و مۆڵەتەکانی بنکەدراوە چاوەڕێی دەستگەیشتنی socket ـی ناوخۆیی یان هەمان-ئامێر دەکەن. `127.0.0.1` یان ناوی خانەخوێی خزمەتگوزاری بەکاربهێنە کاتێک ئەوە خانەخوێی مۆڵەتە کە لەلایەن دابینکەرەکەتەوە ڕێکخراوە.

### توانای Hostinger {#hostinger-capability}

توانای فرە-کرێداری Hostinger لەگەڵ token ـی تێکەڵکردنی هاوبەشی Hostinger کار دەکات. دڵنیا ببەوە کە دۆمەینەکە لە hPanel ئامادەیە، DNS ئاماژە بە Account ـی Hostinger دەکات، و ڕێکخستنەکانی بنکەدراوەی کرێدار پێش پشتڕاستکردنەوە لەگەڵ hPanel یەکدەگرنەوە.

## Hook و Filter ـەکان {#hooks-and-filters}

### Action ـەکان {#actions}

- `ultimate-multisite-addon-template_init` - دوای دەستپێکردنی پاشکۆکە ئاگر دەکرێت
- `ultimate-multisite-addon-template_loaded` - دوای بارکردنی هەموو فایلەکانی پاشکۆکە ئاگر دەکرێت

### Filter ـەکان {#filters}

- `ultimate-multisite-addon-template_settings` - ڕێکخستنەکانی پاشکۆکە دەستکاری بکە
- `ultimate-multisite-addon-template_enabled` - دۆخی چالاک/ناچالاکی پاشکۆکە بگۆڕە
