# Domain Seller

**Multisite Ultimate: Domain Seller** تۆڕی WordPress Multisite ـەکەت دەگۆڕێت بۆ پلاتفۆرمێکی بەهێزی فرۆشتنی دۆمەین. ڕێگە بدە کڕیارەکان دۆمەینی تایبەتی خۆیان ڕاستەوخۆ لە کاتی خۆتۆمارکردن یان لە Dashboard ـەکەیانەوە تۆمار بکەن، لەگەڵ پشکنینی بەردەستبوونی کاتی ڕاستەقینە و نرخدانانی ڕوون.

### بۆچی Domain Seller؟ {#why-domain-seller}

تۆمارکردنی دۆمەین یەکێکە لە داواکارییە زۆرترین upsell ـەکان بۆ هەر پلاتفۆرمێکی WaaS (Website as a Service). لەبری ئەوەی کڕیارەکانت بنێریت بۆ تۆمارکەرێکی لایەنی سێیەم، دەتوانیت دۆمەینەکان ڕاستەوخۆ پێشکەش بکەیت — داهاتی زیاتر لە تۆڕەکەتدا بهێڵیتەوە لە کاتێکدا ئەزموونێکی بێ‌کێشە بە کڕیارەکان دەدەیت.

### تایبەتمەندییە سەرەکییەکان {#key-features}

#### پشتگیری چەند تۆمارکەری دۆمەین {#multiple-domain-registrar-support}
* **OpenSRS** — تۆمارکردن، نوێکردنەوە، و بەڕێوەبردنی تەواو
* **Namecheap** — یەکخستنی تەواو لەگەڵ WhoisGuard و تاقیکردنەوەی sandbox
* **HostAfrica** — گەڕانی دۆمەین، نرخدانان، تۆمارکردن، گواستنەوە، DNS، و پاراستنی ID لە ڕێگەی Domains Reseller API
* **Openprovider** — نرخی reseller، تۆمارکردن، نوێکردنەوە، گواستنەوەکان، DNS، تایبەتمەندی WHOIS، و هاوکاتکردنی TLD
* **Hostinger** — گەڕان و تۆمارکردنی دۆمەین بە بەکارهێنانی token ـی یەکخراوی یەکخستنی Hostinger
* **ئەندازیاریی فراوانکراو** — لەسەر شێوازی Integration Registry دروستکراوە، کە زیادکردنی دابینکەری زیاتر ئاسان دەکات
* **تاقیکردنەوەی پەیوەندی ناوخۆیی** — پێش چوونە ژینگەی ڕاستەقینە زانیارییەکانی دابینکەر و پەیوەندیدان بسەلمێنە

#### ویزاردی ڕێکخستنی ڕێنوێنیکراو {#guided-setup-wizard}
* **دەستپێکردنی بە یەک کلیک** — لە ڕێکخستنی دابینکەر، هاوردەکردنی TLD، و دروستکردنی بەرهەمی دۆمەین ڕێنماییت دەکات
* **هاوکاتکردنی ئۆتۆماتیکی TLD** — هەموو TLD ـە بەردەستەکان و نرخدانانی کۆفرۆشی لە دابینکەرە پەیوەندیدارەکانت هاوردە دەکات
* **دروستکردنی بەرهەمی بنەڕەتی** — بەرهەمێکی دۆمەینی ئامادە بۆ بەکارهێنان دروست دەکات لەگەڵ بنەماکانی markup ـی گونجاو

#### نرخدانانی نەرمونیانی دۆمەین {#flexible-domain-pricing}
* **سێ دۆخی markup** — markup ـی ڕێژەیی، markup ـی جێگیر، یان نرخی جێگیر بۆ هەر TLD
* **بەرهەمە تایبەتەکانی TLD** — بەرهەمی دۆمەینی جیاواز بۆ گرووپە جیاوازەکانی TLD دروست بکە (.com/.net، .io، کۆدەکانی وڵات، هتد.)
* **نرخدانانی ناساندن** — نرخی ساڵی یەکەمی داشکاو پێشکەش بکە بۆ ڕاکێشانی تۆمارکردنی نوێ
* **نرخدانانی کاتی ڕاستەقینە** — نرخدانانی کۆفرۆشی زیندوو لە دابینکەرەکانەوە لەگەڵ کاشکردنی زیرەک
* **پشتگیری کوپۆن** — گونجانی تەواو لەگەڵ کۆدەکانی داشکاندنی Multisite Ultimate

#### پاراستنی تایبەتمەندی WHOIS {#whois-privacy-protection}
* **ڕێکخستنی بەپێی بەرهەم** — تایبەتمەندی WHOIS دابنێ بە ناچالاککراو، هەمیشە لەخۆگرتوو، یان هەڵبژاردەی کڕیار
* **بەشداریکردنی کڕیار** — کاتێک دابنرێت بە هەڵبژاردەی کڕیار، checkbox ـێکی checkout ڕێگە بە کڕیارەکان دەدات پاراستنی تایبەتمەندی زیاد بکەن
* **نرخدانانی ڕێکخراو** — نرخێکی ساڵانە بۆ زیادکراوی تایبەتمەندی دابنێ، یان بەخۆڕایی پێشکەشی بکە
* **یەکخستنی تۆمارکەر** — لە کاتی تۆمارکردندا بە شێوەی ئۆتۆماتیکی WhoisGuard (Namecheap) یان تایبەتمەندی WHOIS (OpenSRS) چالاک دەکات

#### ئەزموونی checkout ـی یەکگرتوو {#unified-checkout-experience}
* **سێ دۆخی دۆمەین** — کڕیارەکان لە نێوان subdomain (بەخۆڕایی)، تۆمارکردنی دۆمەینی نوێ، یان نەخشەکردنی دۆمەینێکی هەبوو کە پێشتر خاوەنی بوون هەڵدەبژێرن
* **گەڕانی دۆمەینی کاتی ڕاستەقینە** — پشکنینی بەردەستبوون بەهێزکراو بە AJAX بەبێ بارکردنەوەی پەڕە
* **پێشنیارە زیرەکەکان** — پێشنیارەکانی TLD ـی جێگرەوە کاتێک دۆمەینی خوازراو گیراوە
* **نرخدانانی ڕوون** — پیشاندانی ڕوونی نرخی تۆمارکردن، نرخی نوێکردنەوە، و کرێی ئارەزوومەندانەی تایبەتمەندی WHOIS
* **خانەکانی پەیوەندی تۆمارکەر** — زانیاری پێویستی تۆمارکەر (ناو، ناونیشان، تەلەفۆن) لەناوخۆدا لە کاتی checkout کۆدەکاتەوە
* **URL ـی ماڵپەڕی دروستکراوی ئۆتۆماتیکی** — ناونیشانی ماڵپەڕ بە شێوەی ئۆتۆماتیکی لە دۆمەینی هەڵبژێردراوەوە وەردەگیرێت

#### سیستەمی نوێکردنەوەی ئۆتۆماتیکی {#auto-renewal-system}
* **نوێکردنەوەکانی بەستراو بە ئەندامێتی** — دۆمەینەکان بە شێوەی ئۆتۆماتیکی نوێ دەکرێنەوە کاتێک ئەندامێتی کڕیار چالاکە
* **خشتەدانانی زیرەک** — کارەکانی نوێکردنەوە بە پێی بەرواری بەسەرچوونی دۆمەین دەخرێنە ڕیزەوە
* **لۆژیکی دووبارە هەوڵدان** — دووبارە هەوڵدانە زیرەکەکان بۆ هەوڵە شکستخواردووەکانی نوێکردنەوە
* **ئاگادارکردنەوەی ئیمەیڵ** — ئاگادارکردنەوەی ڕێکخراو بۆ نوێکردنەوەی داهاتوو، نوێکردنەوەی سەرکەوتوو، شکستەکان، و بەسەرچوونەکان

#### بەڕێوەبردنی DNS ـی کڕیار {#customer-dns-management}
* **DNS ـی خۆخزمەت** — کڕیارەکان دەتوانن تۆمارەکانی DNS (A, AAAA, CNAME, MX, TXT) لە Dashboard ـەکەیانەوە زیاد بکەن، دەستکاری بکەن، و بسڕنەوە
* **دۆخی دۆمەین** — دۆخی تۆمارکردنی کاتی ڕاستەقینە، بەروارەکانی بەسەرچوون، و زانیاری nameserver
* **نەخشەکردنی ئۆتۆماتیکی دۆمەین** — دۆمەینە تۆمارکراو یان هەبووەکان بە شێوەی ئۆتۆماتیکی بۆ ماڵپەڕی کڕیار نەخشە دەکرێن

#### ئامرازەکانی بەڕێوەبەر {#admin-tools}
* **تۆمارکردنی دەستی دۆمەین** — دۆمەینەکان لەبری کڕیارەکان لە پانێڵی بەڕێوەبەرییەوە تۆمار بکە
* **چاودێریکردنی دۆمەین** — هەموو دۆمەینەکانی کڕیار، دۆخەکان، و بەروارەکانی بەسەرچوون لە یەک پەڕەدا ببینە
* **تۆمارکردنی تایبەت بە دابینکەر** — کەناڵەکانی تۆماری جیاواز بۆ هەر تۆمارکەرێک (وەک domain-seller-namecheap.log)
* **وردەکارییەکانی تۆمارکردن** — زانیاری تۆمارکەر و داتاکانی تۆمارکردن لە پەڕەی دەستکاریکردنی دۆمەینی سەرەکی ببینە

### دابینکەرە پشتگیریکراوەکان {#supported-providers}

* **OpenSRS** — یەکخستنی تەواو لەگەڵ تۆمارکردن، نوێکردنەوە، بەڕێوەبردنی DNS، و تایبەتمەندی WHOIS
* **Namecheap** — یەکخستنی تەواو لەگەڵ تۆمارکردن، نوێکردنەوە، پشکنینی بەردەستبوونی بەکۆمەڵ، و WhoisGuard ـی بەخۆڕایی
* **HostAfrica** — تۆمارکردن، نوێکردنەوە، گواستنەوەکان، بەڕێوەبردنی DNS، وەرگرتنی کۆدی EPP، قوفڵی تۆمارکەر، و پاراستنی ID
* **Openprovider** — تۆمارکردن، نوێکردنەوە، گواستنەوەکان، بەڕێوەبردنی DNS، وەرگرتنی کۆدی EPP، قوفڵی تۆمارکەر، تایبەتمەندی WHOIS، و هاوکاتکردنی TLD ـی reseller
* **Hostinger** — تۆمارکردن، گەڕانی دۆمەین، نوێکردنەوەی nameserver، قوفڵی تۆمارکەر، و تایبەتمەندی WHOIS؛ نوێکردنەوەکان لەلایەن نوێکردنەوەی ئۆتۆماتیکی Hostinger ـەوە بەڕێوە دەبرێن
* **GoDaddy** — تۆمارکردن و نوێکردنەوە
* **ResellerClub** — تۆمارکردن، نوێکردنەوە، و بەڕێوەبردنی DNS
* **NameSilo** — تۆمارکردن و نوێکردنەوە
* **Enom** — تۆمارکردن، نوێکردنەوە، و بەڕێوەبردنی DNS

### وردەکارییە تەکنیکییەکان {#technical-details}

* **پێویستی بە:** PHP 7.4+، Multisite Ultimate 2.4.12+
* **ئەندازیاری:** شێوازی Integration Registry لەگەڵ ڕووکاری Domain_Selling_Capability
* **Autoloading:** autoloader ـی Jetpack لەگەڵ نەخشەکردنی پۆلی PSR-4
* **ستانداردەکان:** ستانداردەکانی کۆدنوسی WordPress، جۆرنووسینی توند، دەستەواژە ئامادەکراوەکان

## دامەزراندن {#installation}

1. دڵنیابە **Multisite Ultimate v2.4.12+** دامەزراوە و لەسەر تۆڕ چالاک کراوە
2. pluginەکە باربکە بۆ `/wp-content/plugins/ultimate-multisite-domain-seller/`
3. pluginەکە لە **Plugins** لە WordPress Network Admin لەسەر تۆڕ چالاک بکە
4. **Setup Wizard** بە شێوەیەکی خۆکار لە یەکەم چالاککردندا دەستپێدەکات — هەنگاوەکان پەیڕەو بکە بۆ ئەوەی:
   * پەیوەندی بە domain registrarـەکەت بکەیت
   * پەیوەندییەکە تاقی بکەیتەوە
   * TLDە بەردەستەکان و نرخی کۆفرۆشی هاوردە بکەیت
   * یەکەم بەرهەمی دۆمەینت بە زیادکردنی نرخ دروست بکەیت
5. خانەی **Domain Selection** زیاد بکە بۆ فۆرمی checkoutـەکەت لە **Ultimate Multisite > Checkout Forms**

## پرسیارە باوەکان {#frequently-asked-questions}

### چ وەشانێک لە Multisite Ultimate پێویستە؟ {#what-version-of-multisite-ultimate-is-required}

وەشانی 2.4.12 یان بەرزتر. addonـەکە لەگەڵ WP Ultimo v1.x یان وەشانەکانی Ultimate Multisite لە خوار 2.4.12 گونجاو نییە.

### کام دابینکەرەکانی دۆمەین پشتگیری دەکرێن؟ {#which-domain-providers-are-supported}

نۆ registrar پشتگیری دەکرێن: **OpenSRS**، **Namecheap**، **HostAfrica**، **Openprovider**، **Hostinger**، **GoDaddy**، **ResellerClub**، **NameSilo**، و **Enom**. تواناکان بە پێی دابینکەر جیاوازن چونکە APIـیەکانی registrar کردارە جیاوازەکان ئاشکرا دەکەن؛ بۆ پوختەیەکی دابینکەر بە دابینکەر سەیری ڕێنمایی ڕێکخستن بکە. پەیکەری Integration Registryـی addonـەکە زیادکردنی دابینکەرە تایبەتەکان ئاسان دەکات — سەیری [ڕێنمایی گەشەپێدەر](/developer/integration-guide/custom-registrar) بکە.

### دەتوانم پاراستنی تایبەتمەندی WHOIS پێشکەش بکەم؟ {#can-i-offer-whois-privacy-protection}

بەڵێ. هەر بەرهەمێکی دۆمەین ڕێکخستنی WHOIS Privacy هەیە بە سێ دۆخ: ناچالاک، هەمیشە لەخۆگرتوو، یان هەڵبژاردەی کڕیار. کاتێک لەسەر هەڵبژاردەی کڕیار دانرا، کڕیاران لە کاتی checkout خانەیەکی نیشانکردن دەبینن لەگەڵ نرخەکە (یان "Free" بۆ WhoisGuardـی Namecheap). ئاڵای تایبەتمەندی بە شێوەیەکی خۆکار لە کاتی تۆمارکردندا بۆ registrar دەنێردرێت.

### دەتوانم نرخگذاری جیاواز بۆ TLDی جیاواز دابنێم؟ {#can-i-set-different-pricing-for-different-tlds}

بەڵێ. بەرهەمی دۆمەینی جیاواز بۆ گرووپە جیاوازەکانی TLD دروست بکە. هەر بەرهەمێک دەتوانێت جۆری زیادکردنی نرخی تایبەتی خۆی هەبێت (ڕێژەی سەدی، زیادکردنی نرخی جێگیر، یان نرخی جێگیر)، TLDە پشتگیری‌کراوەکان، و نرخگذاری ناساندن.

### کڕیاران چ هەڵبژاردەیەکی checkout دەبینن؟ {#what-checkout-options-do-customers-see}

خانەی Domain Selection سێ تاب بە کڕیاران دەدات: **Subdomain** (بەکارهێنانی subdomainێکی خۆڕایی لەسەر تۆڕەکەت)، **Register New Domain** (گەڕان و تۆمارکردنی دۆمەینێکی نوێ)، و **Existing Domain** (نەخشەکردنی دۆمەینێک کە پێشتر خاوەنی). تەنها ئەو دۆخانەی چالاکیان دەکەیت دەرکەون.

### پشتگیری نوێکردنەوەی خۆکار دەکات؟ {#does-it-support-automatic-renewals}

بەڵێ. دۆمەینەکان دەتوانرێت بخرێنەسەر نوێکردنەوەی خۆکار و بە دۆخی ئەندامێتیی کڕیارەکەوە دەبەسترێنەوە. پارەدانی نوێکردنەوە لە ڕێگەی دەروازەی پارەدانی ئێستای کڕیارەکەوە پرۆسێس دەکرێت.

### دەتوانن کڕیاران تۆمارەکانی DNSـی خۆیان بەڕێوە ببەن؟ {#can-customers-manage-their-dns-records}

بەڵێ. کڕیاران ڕووکارێکی بەڕێوەبردنی DNS لە dashboardـی خۆیان وەردەگرن بۆ زیادکردن، دەستکاریکردن، و سڕینەوەی تۆمارەکانی A، AAAA، CNAME، MX، و TXT. بەڕێوەبردنی DNS لەگەڵ OpenSRS، ResellerClub، Enom، HostAfrica، و Openprovider بەردەستە. DNSـی Hostinger بۆ دۆمەینە میوانداریکراوەکان لەلایەن یەکخستنی سەرەکی نەخشەکردنی دۆمەینی Hostinger چارەسەر دەکرێت.

### هەیە setup wizard؟ {#is-there-a-setup-wizard}

بەڵێ. setup wizard لە یەکەم چالاککردندا بە شێوەیەکی خۆکار کار دەکات و ڕێنماییت دەکات بۆ پەیوەستکردنی registrarێک، هاوردەکردنی TLDەکان، و دروستکردنی یەکەم بەرهەمی دۆمەینت. هەروەها دەتوانیت wizardەکە لە ڕێکخستنەکانی addon دوبارە بخەیتەوە کار.
