# تۆماری گۆڕانکارییەکانی Ultimate AI Connector بۆ خاڵەکۆتاییە هاگونجاوەکان {#ultimate-ai-connector-for-compatible-endpoints-changelog}

## وەشانی 2.0.0 — بڵاوکراوەتەوە لە 2026-04-24 {#version-200--released-on-2026-04-24}

* نوێ: پشتگیریی فرە-دابینکەر — چەندین خاڵی کۆتایی AI ڕێکبخە و داواکارییەکان بە fallbackـی خۆکار لە نێوان دابینکەرەکاندا ڕێڕەو بکە.
* چاککراوە: یەکخستنی SDKـی فرە-دابینکەر لەگەڵ ناسنامەی دروستی دابینکەرەکان، URLەکانی تۆمارکردن، و سنووردارکردنی فلتەری HTTP بۆ هەر دابینکەرێک.
* چاککراوە: کارتی دابینکەری نوێ ئێستا کاتێک زیاد دەکرێت خۆکارانە فراوان دەبێت؛ هەڵوەشاندنەوەی cacheـی script لە کاتی نوێکردنەوەی پێوەکراودا.
* چاککراوە: namespaceـی پۆلی داینامیکی دابینکەر بۆ eval() و ڕەفتاری فراوانبوونی خۆکار بۆ دابینکەرە تازە زیادکراوەکان.
* چاککراوە: بۆ گونجانی WordPress 6.9+ پێکهاتە جێگیرەکانی Card/CardBody/CardHeader/CardDivider بەکاردەهێنێت (چیتر ئەزموونی نین).
* چاککراوە: DragHandleـی پێناسەنەکراو بە ئایکۆنی گیرکردنی unicode گۆڕدرایەوە بۆ ڕاکێشانی دابینکەر بۆ دووبارە ڕێکخستن.
* چاککراوە: داواکاریی HTTPـی ڕێگر لابرا کە لە هەر بارکردنی لاپەڕەیەکدا دەستپێدەکرد.
* باشترکراوە: کاریگەرییەکانی GitHub Actions بەرزکرانەوە بۆ Node.js 24.
