# Ultimate AI Connector بۆ خاڵە کۆتاییە گونجاوەکان

Ultimate AI Connector بۆ خاڵە کۆتاییە گونجاوەکان APIی دابینکەرانی AIی هاوگونج لەگەڵ OpenAI لەگەڵ تۆڕی فرەماڵپەڕی WordPressەکەت یەکدەخات. چەندین دابینکەری AI ڕێکبخە — هەر یەکەیان بە URLی خاڵی کۆتایی، کلیلی API، و مۆدێلی بنەڕەتیی خۆی — و داواکارییەکان بە گەڕانەوەی خۆکار ئاراستە بکە کاتێک دابینکەرێک بەردەست نییە.

## تایبەتمەندییە سەرەکییەکان {#key-features}

- **پشتگیری چەند-دابینکەر**: چەند کارتی دابینکەری AI زیاد بکە، هەر یەکەیان بە URLی خاڵی کۆتایی، کلیلی API، و مۆدێلی بنەڕەتیی خۆی
- **ڕاکێشان-بۆ-ڕیزکردنەوە**: دابینکەرەکان بەپێی پێشینەیی ڕێکبخە بە بەکارهێنانی دەستگرەکەی سەر هەر کارتێک؛ داواکارییەکان دابینکەرەکان بە ڕیز بەکاردەهێنن
- **ئاراستەکردنی خۆکاری fallback**: ئەگەر دابینکەری سەرەکی بەردەست نەبوو، داواکارییەکان بە شێوەی خۆکار دەگەڕێنەوە بۆ دابینکەری ڕێکخراوی دواتر
- **ڕێکخستنی تایبەت بە هەر دابینکەر**: هەر دابینکەرێک بە سەربەخۆیی بە ناسنامەکان و هەڵبژاردنی مۆدێلی خۆی ڕێکدەخرێت
- **ڕووکاری ڕێکخستنە کارتییەکان**: لەسەر پێکهاتە جێگیرەکانی Cardی WordPress 6.9+ دروستکراوە (CardBody, CardHeader, CardDivider)
- **کردنەوەی خۆکار لە کاتی زیادکردن**: کارتە نوێیەکانی دابینکەر بە شێوەی خۆکار دەکرێنەوە تاکو بتوانیت وردەکارییەکان دەستبەجێ پڕ بکەیتەوە
- **شکاندنی cacheی script**: scriptەکانی دابینکەر لە کاتی نوێکردنەوەی pluginدا cache-bust دەکرێن بۆ دڵنیابوون لە بارکردنی دوایین وەشان

## پێداویستییەکان {#requirements}

- WordPress 5.3 یان بەرزتر
- PHP 7.4 یان بەرزتر
- pluginی Ultimate Multisite (چالاک)
- خاڵی کۆتایی APIی هاوگونج لەگەڵ OpenAI (خۆخانەخوێکراو یان لەلایەن لایەنی سێیەم)

## دامەزراندن {#installation}

1. فایلەکانی addon باربکە بۆ بوخچەی `/wp-content/plugins/`ەکەت
2. pluginەکە لە ڕێگەی menuی 'Plugins' لە WordPress چالاک بکە
3. بچۆ بۆ **Ultimate Multisite → AI Connector** بۆ ڕێکخستنی addonەکە

## ڕێکخستن {#configuration}

### دامەزراندنی چەند-دابینکەر {#multi-provider-setup}

پەڕەی ڕێکخستن دیزاینێکی بنیاتنراو لەسەر کارت بەکاردەهێنێت کە تێیدا هەر کارتێک نوێنەرایەتی یەک دابینکەری AI دەکات. دەتوانیت هێندەی پێویستت دابینکەر زیاد بکەیت.

#### زیادکردنی دابینکەر {#adding-a-provider}

1. لە بەڕێوەبەرایەتی تۆڕدا بچۆ بۆ **Ultimate Multisite → AI Connector**
2. کرتە لە **زیادکردنی دابینکەر** بکە
3. کارتێکی نوێی دابینکەر دەردەکەوێت و بە شێوەی خۆکار دەکرێتەوە
4. وردەکارییەکانی دابینکەر پڕ بکەوە:
   - **ناوی دابینکەر** — پێناسەیەک بۆ ناسینەوەی ئەم دابینکەرە (بۆ نموونە "OpenAI"، "Mistral"، "Local Ollama")
   - **URLی خاڵی کۆتایی** — URLی بنەڕەتی APIی هاوگونج لەگەڵ OpenAI (بۆ نموونە `https://api.openai.com/v1`)
   - **کلیلی API** — کلیلی APIی تۆ بۆ ئەم دابینکەرە
   - **مۆدێلی بنەڕەتی** — ناسێنەری مۆدێل کە بە شێوەی بنەڕەت بۆ ئەم دابینکەرە بەکاردێت (بۆ نموونە `gpt-4o`، `mistral-small`)
5. کرتە لە **پاشەکەوتکردنی ڕێکخستنەکان** بکە

#### سڕینەوەی دابینکەر {#removing-a-provider}

کرتە لە دوگمەی **سڕینەوە** لەسەر هەر کارتێکی دابینکەر بکە بۆ سڕینەوەی. ڕێکخستنەکان پاشەکەوت بکە بۆ جێبەجێکردنی گۆڕانکارییەکە.

#### ڕیزکردنەوەی دابینکەرەکان {#reordering-providers}

ئایکۆنی دەستگرە (⣿) لە لای چەپی هەر کارتێکی دابینکەر ڕابکێشە بۆ ڕیزکردنەوەی. ڕیزەکە پێشینەیی fallback دیاری دەکات: داواکارییەکان بۆ دابینکەری یەکەم دەنێردرێن و ئەگەر داواکارییەک سەرنەکەوت، دەگەڕێنەوە بۆ دابینکەرەکانی دواتر.

### ئاراستەکردنی fallback {#fallback-routing}

کاتێک داواکارییەک بۆ دابینکەری سەرەکی سەرناکەوێت — بەهۆی timeout، سنووری ڕێژە، یان وەستانی خزمەتگوزاری — connectorەکە بە شێوەی خۆکار دووبارە بە دابینکەری دواتر لە لیستەکەدا هەوڵ دەداتەوە. ئەمە بە درێژایی لیستەکە بەردەوام دەبێت تا دابینکەرێک سەرکەوتوو دەبێت یان هەموو دابینکەرەکان تاقی کرابنەوە.

| دۆخ | ڕەفتار |
|----------|-----------|
| دابینکەری سەرەکی وەڵام دەداتەوە | داواکارییەکە بە شێوەی ئاسایی خزمەت دەکرێت |
| دابینکەری سەرەکی سەرناکەوێت | داواکارییەکە بە دابینکەری دواتر دووبارە هەوڵ دەدرێتەوە |
| هەموو دابینکەرەکان سەرناکەون | هەڵە دەگەڕێندرێتەوە بۆ بانگکەرەکە |

fallback بۆ plugin یان themeی بانگکار شەفافە — ئەوان یان وەڵامێکی سەرکەوتوو وەردەگرن یان هەڵەیەکی کۆتایی، بێ گوێدانە ئەوەی چەند دابینکەرێک تاقی کرابنەوە.

### سەرچاوەی ڕێکخستنی دابینکەر {#provider-configuration-reference}

| خانە | پێویستە | وەسف |
|-------|----------|-------------|
| ناوی دابینکەر | بەڵێ | پێناسەی پیشاندان بۆ ئەم دابینکەرە |
| URLی خاڵی کۆتایی | بەڵێ | URLی بنەڕەتی APIی هاوگونج لەگەڵ OpenAI |
| کلیلی API | بەڵێ | کلیلی پشتڕاستکردنەوە بۆ ئەم دابینکەرە |
| مۆدێلی بنەڕەتی | بەڵێ | ناسێنەری مۆدێل کە بەکاردێت کاتێک هیچ مۆدێلێک لە داواکارییەکەدا دیاری نەکراوە |

## چارەسەرکردنی کێشەکان {#troubleshooting}

### هیچ دابینکەرێک لیست نەکراوە {#no-providers-are-listed}

ئەگەر لیستی دابینکەرەکان دوای پاشەکەوتکردن بەتاڵە، دڵنیا ببە کە:

- دوای زیادکردنی دابینکەرەکان کرتە لە **پاشەکەوتکردنی ڕێکخستنەکان** کردووە
- وێبگەڕەکەت پەڕەی ڕێکخستنەکان لە پاشەکەوتکردن بلۆک نەکردووە (consoleی وێبگەڕ بۆ هەڵەکانی JavaScript بپشکنە)
- pluginەکە چالاکە و تۆ لە پەڕەی بەڕێوەبەرایەتی تۆڕدای

### داواکارییەکان بۆ هەموو دابینکەرەکان سەرناکەون {#requests-are-failing-for-all-providers}

- دڵنیا ببە کە URLی خاڵی کۆتایی هەر دابینکەرێک لە serverەکەتەوە دەگات (نەک تەنها لە وێبگەڕەکەتەوە)
- بپشکنە کە هەر کلیلی APIیەک دروستە و بەسەرنەچووە
- دۆخی سنووری ڕێژە و quotaی دابینکەرەکە لە dashboardی ئەواندا پێداچوونەوە بکە
- تۆمارکردنی debugی WordPress چالاک بکە (`WP_DEBUG_LOG`) بۆ گرتنی پەیامە وردەکانی هەڵە لە connectorەکەوە

### دابینکەرێک بۆ fallback بەکارناهێنرێت {#a-provider-is-not-being-used-for-fallback}

ڕاکێشان-بۆ-ڕیزکردنەوە ڕیزی fallback دەگۆڕێت. دڵنیا ببە دابینکەرەکەی چاوەڕێ دەکەیت داواکارییەکانی fallback وەربگرێت بە دروستی لە لیستەکەدا دانراوە و URLی خاڵی کۆتایی و کلیلی APIی ئەو دروستن.

### کارتی دابینکەری نوێ بە شێوەی خۆکار ناکرێتەوە {#new-provider-card-does-not-expand-automatically}

کردنەوەی خۆکار پێویستی بەوەیە JavaScript لە وێبگەڕەکەتدا چالاک بێت. ئەگەر کارتەکە لە کاتی کرتەکردن ناکرێتەوە، consoleی وێبگەڕ بۆ هەڵەکان بپشکنە. ناوەڕۆکی کارتەکە هەمیشە بە کرتەکردن لەسەر سەردێڕی کارتەکە دەستپێدەگات، تەنانەت ئەگەر کردنەوەی خۆکار کارا نەبێت.
