# ناتوانکارییە زانین لەگەڵ پلاتینی دیکەکان {#known-incompatibilities-with-other-plugins}

پاش جێهێشتنی و چالاککردنەوەی Ultimate Multisite، دەتوانیت ڕووبिडिओێنان بەهۆی ناساندنی ناتوانکارییەکان لەگەڵ پلاتینی دیکەکان ڕووبدات.

ئێمە بەردەوام کار دەکەین بۆ کەمکردنەوەی ئەم لیستە لەگەڵ هەر هەڵبژاردەیەکی نوێ، بەڵام ئەگەر پلاتینیەک لەم جایشیدا لێشێت، پێویستە بیکاتە ئاراستەی (deactivate) بۆ ئەوەی تۆڕی Ultimate Multisite بە تەواوی کار بکات.

لێرەدا لیستێکیان و ئەو شتانەی کە دەبێت لەو حاڵەتان بکەیت:

**شێوازەکان لە WPMU DEV:** هەندێک لە پلاتینی ئەم ڕیزبەندییە زیاتر نیشتەجێ نەکراون لەلایەن WPMU DEV، بەڵام ناسراوە کە دەبن پرۆسەی تۆمارکردنی Ultimate Multisite لە هەنگاو کۆتاییدا (کە بەکارهێنەر بتوانێت خاڵێکی دروست بکات) وەستاو بێت. لێرە لیستێکە لە پلاتینی ئەوان کە ئەم کێشەکانی دروست دەکەن:

  * Signup Language: _Ultimate Multisite وەک پێکهاتەیەکی زیادە (add-on) ی پێشکەش دەکات ئەو کارانەی هەمان شت دەکرێت، بە ناوی Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _ئەمە لە Ultimate Multisite دا دروستکراوە، بۆیە پێویستی بە پلاتینی زیاتر نییە._

  * New Blog Templates: _ئەمە لە Ultimate Multisite دا دروستکراوە، بۆیە پێویستی بە پلاتینی زیاتر نییە._

  * Set Password on Multisite Blog Creation: _ئەمە لە Ultimate Multisite دا دروستکراوە، بۆیە پێویستی بە پلاتینی زیاتر نییە._

  * Signup TOS: _ئەمە لە Ultimate Multisite دا دروستکراوە، بۆیە پێویستی بە پلاتینی زیاتر نییە._

  * Anti-Splog: سەرەڕای ئەوەی کاریگەرتر نییە، ئەمەش _لە Ultimate Multisite دا دروستکراوە._

**تێبینی:** هەموو پلاتینی دیکەی WPMU Dev دەتوانن بە شێوەیەکی ئاسایی لەگەڵ Ultimate Multisite بەکاربهێنرێت. نموونەکان: _Smush_، _Forminator_، _Defender_ و هتد.
