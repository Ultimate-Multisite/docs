# هەڵەی دامەزراندنی فایلەکە

فایلەکەی `sunrise.php` فایلی تایبەتە کە وۆرسپریډ (WordPress) لە کاتی دەستکردەوەی خۆی لێی دەگەڕێتەوە. بۆ ئەوەی وۆرسپریډ بتوانێت ئەم فایلە بەدەست بهێنێت، دەبێت لەناو دۆلێری **wp-content**دا دیار بێت.

کاتێک Ultimate Multisite چالاک دەکەیت و ڕێنماییەکانی دامەزراندن (setup wizard) وەک ئەو یەکەی کە لە شاتسکرۆپەکەدا هەیە، Ultimate Multisite هەوڵ دەدات فایلەکەی `sunrise.php` بۆ دۆلێری wp-content کۆپی بکات.

<!-- شاتسکرۆپ نەدێت: لاپەڕەی ڕێنمایی دامەزراندن کە هەنگاوی دامەزراندنی sunrise.php دەردەخات -->

زۆربەی کات، ئێمە دەتوانین بە سەرکەوتوویی فایلەکە کۆپی بکەین و هەموو شتێک کار دەکات. بەڵام ئەگەر شتێکی ورد نەکراوە (بۆ نموونە ڕێزماندنی دۆلێری)، دەتوانیت لەو ئەو حاڵەتەدا ببیت کە Ultimate Multisite ناتوانێت فایلەکە کۆپی بکات.

ئەگەر پەیامی هەڵەی Ultimo وەربگریت، دەبینیت کە بەم شێوەیە ڕوودەدات: **Sunrise copy failed** (کۆپکردنەوەی Sunrise شکستخۆدا).

<!-- شاتسکرۆپ نەدێت: پەیامی هەڵە کە دەریدەخات وای کردنی کۆپی Sunrise شکستخۆدا -->

بۆ چارەسەرکردنی ئەمە، دەتوانیت سادەکان فایلەکەی `sunrise.php` لەناو دۆلێری پلاگینی wp-ultimo کۆپی بکەیت و بۆ دۆلێری wp-content یەکێک لە ناوچەکانی خۆت چسپێنیتەوە (paste). دوای ئەوە، لاپەڕەی ڕێنماییەکە دەگەڕێریتەوە و تاقیکردنەوەکان دەبێت سەرکەوتوو بن.

<!-- شاتسکرۆپ نەدێت: مێنافەری فایل کە sunrise.php لەناو دۆلێری پلاگینی wp-ultimoدا دەردەخات --> بە هەر شێوەیەک، ئەمە دەتوانرێت ببێتە هۆی تاقیکردنەوەیەکی گشتی بۆ ڕێزماندنی دۆلێری خۆت بۆ ئەوەی لە داهاتوودا کێشەیەک نییە (نەک تەنها لەگەڵ Ultimate Multisite بەڵکو لەگەڵ پلاگین و قالبەکانی دیکەش).

**Health Check tool** کە بەشێکە لە وۆرسپریډ (دەتوانیت دەربکەیت لە ڕێگەی **admin panel > Tools > Health Check**)، توانای ئەوەی هەیە بڵێیت ئایا ڕێزماندنی دۆلێری تۆ بە ئەو بەهایانە دیاریکراون کە دەتوانن کێشەیان بۆ وۆرسپریډ دروست بکەن یان نا.

<!-- وێنەی دروست نییە: ئامێرەی Ultimate Multisite Health Check کە دۆخی ڕێگەدانەکانی فۆلเดەر دەردەخات -->
