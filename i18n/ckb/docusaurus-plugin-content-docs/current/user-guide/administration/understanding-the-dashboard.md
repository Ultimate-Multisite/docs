# تێگەیشتن لە داش보اردی Ultimate Multisite

پێش ئەوەی ئێمە پڕ بە ڕێکخستنە پێشکەوتوو و لایەنە کارکردنی Ultimate Multisite ببینین، بیستەوێت چاوەڕێ بکەیندا داشبۆردەکە.

وەک زۆرێک لە pluginەکانی WordPress، Ultimate Multisite یەکบันدرێکی مینیوی ئۆپەراسیۆنی بە ناوی **Ultimate Multisite** دروست دەکات کە چەندین پێکهاتەی مینیو لە خوارەوەی خۆیدا هەیە.

## ویجێتاڵەکانی داش보اردی تۆڕ (Network Dashboard Widgets) {#network-dashboard-widgets}

Ultimate Multisite چەند ویجێتاڵێکی بەسوود دەداتە ناو داشبۆردی ئۆپەراسیۆنی تۆڕەکە. ئەم داشبۆردە دەتوانیت بە ڕێگەی **My Sites → Network Admin → Dashboard** بگەیت. ئەم ویجێتاڵانە دیزاین کراون بۆ ئەوەی دەستگەیشتنێکی ئاسان بۆ زانیاری و کارکردنە باوەکان بدەن بە بەڕێوەبەری تۆڕەکە.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

### هەنگاوە یەکەمەکان (First Steps) {#first-steps}

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

ئەم ویجێتاڵە لەسەر جێی سەرەتایی دامەزراندنی Ultimate Multisite دەردەکەوێت و بەڕێوەبەری تۆڕەکە یاد دل دەکات بۆ ئەوەی کارەکانی دیکە تەواو بکات، وەک ڕێکخستنی شێوازی پامێرکردن (payment method) و دروستکردنی یاری تاقیکردنەوە.

### پوختەکان (Summary) {#summary}

![Summary widget](/img/admin/network-dashboard-summary.png)

ویجێتاڵی Summary ڕاپۆرت دەکات لەسەر ژمارەی ئەندامانی نوێکراوە و دراوەیان بۆ ئەو ڕۆژە. MRR (Monthly Recurring Revenue) ئاماژەیە بۆ کۆی گشتی پێشبینیکراو لەسەر کڕیاران بە یەکەم کە لە پڕۆژەکانی دراوەی پارەدا بەشدارن.

### ڕێکارکردن ئێستا (Activity Stream) {#activity-stream}

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

ئەم ویجێتاڵە ڕاپۆرت دەکات لەسەر ڕوودۆحەکانی سەر قۆناغەکانی Ultimate Multisite. ئەو ڕوودۆحانە دەشت دەبن: نوێکراوەکان، لێبوردن (cancellations)، دروستکردنی سایت، دیاریکردنی سایتان، و چالاکییەکانی دیکە.

تکایە بۆ وەسفی وردتر، بەشی Events لە dokumantacija ببینە.

### ئێستا چی دەبێت (Right Now) {#right-now}

![Right Now widget](/img/admin/network-dashboard-right-now.png)

ئەم ویجێتكە پێکەوەی کورتەیەک لەسەر بەکارهێنەران و شوێنەکان لە تۆڕەکە دەردەخات. پێشترووکی ئاسان لە سەرەوە ڕێگە دەدات بۆ دروستکردنی شوێن یان بەکارهێنەرە نوێکان بە تاکێک.

### هەواڵ و گفتوگۆکان (News and Discussions) {#news-and-discussions}

![News and Discussions widget](/img/admin/network-dashboard-news.png)

ئەم ویجێتكە نوێترین هەواڵەکانی Ultimate Multisite دەداتەوە و نیشان دەدات. لەسەر ئەم ویجێتان چاودێری بکە بۆ زانیارییەکانی نوێکارییەکان، باگەکانی گرنگ، و چارەسەرکردنی ئاسایشی.

## داشبۆرد Ultimate Multisite (Ultimate Multisite Dashboard) {#ultimate-multisite-dashboard}

لەداخڵدا، داشبۆردی نۆڕمەڵەی تۆڕەکە زانیاریی گشتی دەدات، بەڵام داشبۆردی Ultimate Multisite (کە لە سەرەوەی مێنیو Ultimate Multisite دادەنرێت) زانیاریی تایبەت بە ئەو خزمەتگوزینەیە دەدات.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

### گەشەکردنی دراوەی بازاڕی مانگانە (Monthly Recurring Revenue Growth) {#monthly-recurring-revenue-growth}

چارتەکە بۆ MRR (دراوەی بازاڕی مانگانە) پاشکۆی مانگانی لەسەر ماوەی ١٢ مانگی زمانی دەدات.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

دراوەی نوێ بە ڕەنگی زەرد (blue) نیشان دەدرێت، لەکاتێکدا لغارییەکان بە ڕەنگی قەڵەوی (red) دیار دەکرێن.

### پشتگیریکردنی خاڵەکانی کات (Date-range Support) {#date-range-support}

لەبەر ژمارەی کڕیاران و چالاکیەکە، ئامارەکان دەکرێت زۆر زیاتر بن. پشتگیریکردنی ماوەی کات کە لە داشبۆردەکەدا دروستکراوە، ڕێگە بە بەڕێوەبەری تۆڕەکە دەدات تەنها سەرنجی بدات لەسەر ئەو زانیارییە و ئەو ماوەیەەی پێویستە.

![Date range selector and stats](/img/admin/um-dashboard-date-range.png)

ماوەی کات، ئەو پۆلێسییە دەبێت کە ماوەی زانیاریی نیشان دەدات لە ویجێتا خوارەوە، لەوانەش دراوەکان، ئەندامنووسی نوێ، تۆمارکردن بەپێی وڵات، تۆمارکردن بەپێی فۆرم، و زیاترین شوێنەکانی سەردانی کراو.

![Dashboard statistics widgets](/img/admin/um-dashboard-stats.png)

:::tip
بۆ ئەوەی ویجێتكە **Most Visited Sites** کار بکات، لە ڕێگەی **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** دا، کۆنترۆڵی باڵا بۆ هەڵسەنگاندنی گۆڕینی سەردانەکان دەبێت.
:::
