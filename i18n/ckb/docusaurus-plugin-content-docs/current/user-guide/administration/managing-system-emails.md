# Manage System Emails

Ultimate Multisite دۆخی جیاواز لە ئیمێلەکانی سیستەم بۆ واقەتەکان وەک ڕاگەیاندنی تۆمارکردن، ڕاپۆرتەکانی پارەدان، و گۆڕینی ئەندامان دەدات. دەتوانیت ئەم قالبە ئیمێڵانە بگۆڕیت بۆ ئەوەی لەگەڵ براندەکەت یەک بێت.

## دابอ่านکردنی ئیمێلەکانی سیستەم {#viewing-system-emails}

بڕۆ بۆ **Ultimate Multisite → Settings** و پاشان سەر پەڕەی **Emails** بکە بۆ بینینی هە settings-ی پەیوەندیدار بە ئیمێل.

![Settings - Emails tab](/img/admin/settings-emails.png)

لەوێدا دەتوانیت دەربکەویتە پەڕەی **System Emails** بۆ بینینی هەموو قالبەکانی ئیمێڵ.

![System Emails list](/img/admin/system-emails-list.png)

دەتوانیت بەکاربهێنیت بۆ چاککردنی قالبەکانی ئیمێل تاکەکەسی لە ڕێگەی ویرتاژە ئیمێلەکانی سیستەمەوە:

![System email editor](/img/admin/system-email-editor.png)

## گێڕانەوەی قالبەکانی ئیمێل نوێیەکان {#importing-new-email-templates}

وەرگرتنی وەرگرتنی نۆچی Ultimate Multisite و add-ons دەتوانن قالبە ئیمێڵە نوێکان تۆمار بکەن. بۆ ئەوەی لەناوبرێن، قالبە نوێکان بە شێوەیەکی ئۆتۆماتیک زیاد ناکرێن — پێویستە بە دەست دانانەوەیان بێت.

بۆ وەرگرتنی قالبەکانی ئیمێل نوێ:

1. بچۆ بۆ پەڕەی **System Emails**
2. बटانی **Reset & Import** لە سەرەوەی ئەو پەڕەیە بکە
3. ڕەتکردنەوەی کارکردنی **Import Emails** بکە بۆ بینینی قالبەکان کە بەکارخستوونن
4. ئەو قالبانەی کە دەتەوێت وەربگریت هەڵبژێرە و پشتڕاست بکەرەوە

## گێڕانەوەی قالبەکانی ئیمێل {#resetting-email-templates}

ئەگەر یەکێک لە قالبەکانی ئیمێل بەکارخستوویت و دەتەوێت بگۆڕیت بۆ ناوەڕۆکی سەرەکی، دوو ڕێگە هەیە:

1. **دابڕین و دووبارە وەگرتن** — سیستەم ئیمێلەکە دەدابڕە و دووبارە وەربگریت (ئەم کارە پێوانەکانی فرستادن دەپاکرێت).
2. **گێڕانەوە بە شێوەیەکی ڕاستەوخۆ لەسەر ئامێرەکە** — بیکە بەکارکردنی ئامشتەکەی **Reset & Import**، ڕەتکردنەوەی کارکردنی **Reset** بکە، و ئەو ئیمێلانەی کە دەتەوێت بگۆڕیت.

بۆیە هەڵبژاردنی ڕێگای Reset باشترە چونکە پێوانەکانی فرستادن و زانیارییە دیکەکان پارێزراوەتەوە لەکاتێکدا ناوەڕۆکی سەرەکی دەگەڕێتەوە بۆ ئەوەی بەکاربهێنرێت.
