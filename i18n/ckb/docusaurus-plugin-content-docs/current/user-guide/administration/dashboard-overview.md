# نظۆکی سەرەکی (Dashboard Overview)

Ultimate Multisite هەیە **Dashboard** تایبەت بە دەوروبەری منوی Ultimate Multisite، هەروەها وێنەکان بۆ Network Dashboard زیاد دەکات. ئەم بابەتە باس لەوە دەکات کە چی لە هەردوو شوێنەکەدا دەبینیت.

## Network Dashboard (نۆتھوێك داشبۆرد)

Ultimate Multisite وێجێتە بەسوود بۆ داش보ردی ئاستی تۆڕی (network-level dashboard) دادەنێت کە لە **My Sites → Network Admin → Dashboard** دابوویت.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

ئەمە چی پێشکەش دەکات:

**First Steps:** لەسەر جێدانی سەرەتایی (fresh installation) دەردەکەوێت و یاد دل دەکات کە کارەکانی سیستەم وەک ڕێکخستنی شێوازی پامێرکردن و دروستکردنی یاری تاقیکردنەوەی کارلێکردن ئەنجام بدەیت.

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Summary:** ژمارەی تۆمارەکان و Einkا بۆ ڕۆژێک ڕاپۆرت دەکات. MRR (Monthly Recurring Revenue) ئاماژەیە بۆ کۆیاریکردنی پڕۆژەی لەسەر کڕیاران بەو ئەندامییانەی کە Einkا دروست دەکەن.

![Summary widget](/img/admin/network-dashboard-summary.png)

**Activity Stream:** دەربارەی ڕوودۆخەکان لەسەر Ultimate Multisite - تۆمارکردن، لێبوردن، دروستکردنی سایت، دیاریکردن و چالاکییەکانی دیکە راپۆرت دەکات.

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** پوختەیەکی کورت لەسەر بەکارهێنەران و سایتەکان لە تۆڕەکە دەردەخات، هەروەها ڕێگەیەک بۆ دروستکردنی سایت یان بەکارهێنەرێکی نوێ.

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** نوێترین هەواڵەکان و گفتوگۆکان لەسەر Ultimate Multisite دەگەڕێتەوە و نیشاندەدات، لەوانەش نوێکارییەکان، ئەو ڕەفتارە گرنگانەی کە کێشەیان هەیە، و چارەسەرکردنی ئاسایشی.

![News widget](/img/admin/network-dashboard-news.png)

## Ultimate Multisite Dashboard (داشبۆردی Ultimate Multisite)

لەگەڵ ئەوەی Network Dashboard زانیارییە گشتی لەسەر تۆڕەکە دەدات، Ultimate Multisite dashboard (لە سەرەوەی منوی Ultimate Multisite) زانیارییەکانی تایبەت بە ئەو خزمەتگوزارییە دەدات.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**نمو درآمد تکرارشونده ماهانه (Monthly Recurring Revenue Growth):** این بخش یک تفکیک ماهانه از درآمد در طول یک دوره ۱۲ ماه تقویمی را نمایش می‌دهد. درآمد جدید با رنگ آبی نشان داده می‌شود و لغو اشتراک‌ها با رنگ قرمز ظاهر می‌شوند.

![نمودار رشد MRR](/img/admin/um-dashboard-mrr.png)

**پشتیبانی از بازه زمانی (Date-range Support):** می‌توانید داده‌های داشبورد را بر اساس دوره زمانی فیلتر کنید تا روی اطلاعاتی که نیاز دارید تمرکز کنید. محدوده تاریخ تعیین می‌کند چه چیزی در ویجت‌های زیر نمایش داده شود.

![محدوده تاریخ و آمار داشبورد](/img/admin/um-dashboard-date-range.png)

ویجت‌های داشبورد شامل موارد زیر هستند:

- **درآمد (Revenue)** — کل درآمد برای دوره مورد نظر به همراه تفکیک MRR و درآمد به ازای هر محصول
- **اشتراک‌های جدید (New Memberships)** — تعداد اشتراک‌های جدیدی که در دوره انتخاب شده ایجاد شده‌اند
- **ثبت‌نام‌ها بر اساس کشورها (Signups by Countries)** — تفکیک جغرافیایی ثبت‌نام‌ها
- **ثبت‌نام‌ها بر اساس فرم‌ها (Signups by Forms)** — تعداد مشتریانی که از طریق هر فرم پرداخت ثبت‌نام کرده‌اند در مقابل کسانی که توسط مدیران اضافه شده‌اند
- **محبوب‌ترین سایت‌ها (Most Visited Sites)** — پربازدیدترین سایت‌های شبکه شما (نیاز به فعال کردن شمارش بازدید در **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** دارد)

![آمار داشبورد](/img/admin/um-dashboard-stats.png)
