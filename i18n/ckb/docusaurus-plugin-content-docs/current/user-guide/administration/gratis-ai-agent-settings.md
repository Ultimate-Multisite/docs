# ڕێکخستنەکانی Gratis AI Agent

شاشەی **Settings → Advanced** لە Gratis AI Agent ڕێکخستنی ئاستی بەڕێوەبەر بۆ یەکخستنەکانی backend دابین دەکات. ئەم پەڕەیە گواستنەوەی feedback، کلیلی دابینکەری گەڕان، دانانی خزمەتگوزاری بەڕێوەبردراوی Superdav، کۆنترۆڵەکانی Google Calendar، ڕێکخستنەکانی TextBee SMS، و feature flag ـەکانی سەرانسەری تۆڕ بەڵگەنامە دەکات.

## چوونەناو ڕێکخستنەکان

1. لە بەڕێوەبەرایەتی WordPress، بچۆ بۆ **Gratis AI Agent → Settings**.
2. کلیک لە تابی **Advanced** بکە.

## ڕێکخستنی endpoint ـی Feedback

endpoint ـی feedback داواکارییەکانی POST لە AI agent وەردەگرێت هەر کاتێک بەکارهێنەرێک feedback بنێرێت لە ڕێگەی دوگمەی پەنجەبۆخوارەوە، بانەری auto-prompt، یان فەرمانی `/report-issue`.

| خانە | وەسف |
|---|---|
| **Feedback Endpoint URL** | ئەو URL ـەی feedback ـە نێردراوەکان وەک داواکاری HTTP POST بە جەستەی JSON وەردەگرێت. |
| **Feedback API Key** | bearer token ـێکە کە لە `Authorization` header ـی هەر داواکارییەکی feedback دەنێردرێت. ئەگەر endpoint ـەکەت پێویستی بە پشتڕاستکردنەوە نییە، بەتاڵی بهێڵەوە. |

### JSON Payload ـی چاوەڕوانکراو

endpoint ـی feedback ـەکەت دەبێت جەستەی JSON ـێک وەربگرێت کە لانیکەم ئەم خانانەی خوارەوەی هەبێت:

```json
{
  "message_id": "msg_abc123",
  "conversation_id": "conv_xyz789",
  "feedback_text": "The answer was incorrect about pricing.",
  "triage_category": "factual_error"
}
```

لەوانەیە خانەی زیادە لە payload ـدا هەبن، بە پێی کۆنتێکستی گفتوگۆکە.

### بەهاکانی `triage_category`

چینی triage ـی AI یەکێک لەم بەهایانەی خوارەوە بۆ `triage_category` دیاری دەکات پێش ناردنی payload:

| بەها | واتا |
|---|---|
| `factual_error` | یارمەتیدەرەکە زانیاری ڕاستی هەڵەی دابین کرد. |
| `unhelpful_answer` | وەڵامەکە لە ڕووی تەکنیکییەوە ڕاست بوو، بەڵام سوودبەخش نەبوو. |
| `inappropriate_content` | وەڵامەکە ناوەڕۆکێکی تێدا بوو کە نابێت پیشانی بەکارهێنەران بدرێت. |
| `other` | feedback ـەکە لەگەڵ هیچ پۆلێکی ناسراو نەگونجا. |

### پشتڕاستکردنەوە

ئەگەر endpoint ـەکەت پێویستی بە پشتڕاستکردنەوە هەیە، خانەی **Feedback API Key** بخە سەر bearer token ـەکەت. agent ـەکە دەنێرێت:

```
Authorization: Bearer <your-api-key>
```

ئەگەر خانەی **Feedback API Key** بەتاڵ بێت، هیچ `Authorization` header ـێک نانێردرێت.

### ناچالاککردنی کۆکردنەوەی Feedback

هەردوو خانەی **Feedback Endpoint URL** و **Feedback API Key** بەتاڵ بهێڵەوە. دوگمەی پەنجەبۆخوارەوە و UI ـی feedback بە دیاریماوی بۆ بەکارهێنەران دەمێننەوە، بەڵام ناردنەکان بۆ هیچ خزمەتگوزارییەکی دەرەکی ناگوازرێنەوە.

## Brave Search API Key

هەروەها لە تابی **Advanced**، خانەی **Brave Search API Key** توانای [گەڕانی ئینتەرنێت](../configuration/internet-search) چالاک دەکات.

| خانە | وەسف |
|---|---|
| **Brave Search API Key** | API key ـەکەت لە dashboard ـی گەشەپێدەری Brave Search. بۆ چالاککردنی گەڕانی ئینتەرنێت لە یارمەتیدەری AI پێویستە. |

ناوی خانەکە بەستەرێکی کرتەکراوی بۆ پەڕەی تۆماربوونی Brave Search API لەخۆ دەگرێت. بۆ ناچالاککردنی گەڕانی ئینتەرنێت بەتاڵی بهێڵەوە.

بۆ بەڵگەنامەی کۆتایی-بەکارهێنەر دەربارەی ئەم تایبەتمەندییە، سەیری [گەڕانی ئینتەرنێت](../configuration/internet-search) بکە.

## خزمەتگوزاری بەڕێوەبردراوی Superdav

Superdav AI Agent v1.18.0 endpoint ـەکانی خزمەتگوزاری بەڕێوەبردراوی Superdav و دابینکردنی خۆکاری پەیوەندی بۆ ماڵپەڕە پشتگیریکراوەکان زیاد دەکات. ئەم کۆنترۆڵانە بەکاربهێنە کاتێک ماڵپەڕەکەت دەبێت ببەسترێتەوە بە دابینکەری میواندارکراو لەبری endpoint ـی خزمەتگوزارییەکی بە دەستی ڕێکخراو.

| خانە | وەسف |
|---|---|
| **Managed Superdav Service** | پەیوەندی خزمەتگوزاری میواندارکراوی Superdav بۆ ماڵپەڕە پشتگیریکراوەکان چالاک دەکات. |
| **Provision Connection** | دابینکردنی خۆکاری endpoint و ناسنامەکان دەست پێ دەکات. ئەمە بەکاربهێنە دوای پشتڕاستکردنەوەی ئەوەی ماڵپەڕەکە دەبێت دابینکەری بەڕێوەبردراو بەکاربهێنێت. |
| **Service Endpoint / Connection Status** | endpoint ـی ئێستا یان دۆخی پەیوەندی دوای دابینکردن پیشان دەدات. |

دوای دابینکردن، ڕێکخستنەکان پاشەکەوت بکە و دۆخی پەیوەندی پشتڕاست بکەرەوە پێش پشتبەستن بە workflow ـەکانی خزمەتگوزاری-بەڕێوەبردراو. ئەگەر دابینکردن شکستی هێنا، ڕێنمایی دووبارەهەوڵدانەوەی پیشاندراو پێداچوونەوە بکە و پشتڕاستی بکەرەوە کە ماڵپەڕەکە مۆڵەتی بەکارهێنانی دابینکەری میواندارکراوی هەیە.

## ڕێکخستنی Google Calendar

کاتێک تایبەتمەندییەکانی ساڵنامەی Superdav AI Agent v1.18.0 چالاکن، agent ـەکە دەتوانێت ساڵنامە ڕێکخراوەکان و وردەکارییەکانی ڕووداو بخوێنێتەوە. ئامرازەکانی ساڵنامە ئاراستەی خوێندنەوەن و بۆ بیرخستنەوەی ئاگادار بە خشتە، بەدواداچوونی بەشداربوو، و هاوتاکردنی پەیوەندی بەسوودن.

| خانە | وەسف |
|---|---|
| **Google Calendar Credentials** | ئەو credentials یان پەیوەندی token ـە پاشەکەوت دەکات کە بۆ خوێندنەوەی داتای ساڵنامە پێویستە. |
| **Calendar Selection** | سنوور دادەنێت کە agent ـەکە دەتوانێت چ ساڵنامە ڕێکخراوانە بپشکنێت. |
| **Calendar Connection Status** | پشتڕاست دەکاتەوە ئایا credentials ـەکانی ئێستا دەتوانن ساڵنامە و ڕووداوەکان بخوێننەوە. |

credentials ـی ساڵنامە تەنها بۆ ئەو ساڵنامانە سنووردار بکە کە agent ـەکە پێویستی پێیانە. ئەگەر دۆخەکە token ـێکی بەسەرچوو پیشان بدات، دووبارە ببەستەرەوە یان credentials بگۆڕە.

## ئاگادارکردنەوەکانی TextBee SMS

Superdav AI Agent v1.18.0 TextBee وەک دابینکەری SMS بۆ workflow ـە ڕێکخراوەکانی ئاگادارکردنەوە زیاد دەکات. ئاگادارکردنەوەکانی SMS دەبێت لەگەڵ دەروازەکانی پەسەندکردنی مرۆیی جوت بکرێن بۆ پەیامە هەستیارەکان یان ئەوانەی ڕوویان لە بەکارهێنەرە.

| خانە | وەسف |
|---|---|
| **TextBee API Key** | داواکارییەکان بۆ دابینکەری TextBee SMS پشتڕاست دەکاتەوە. |
| **TextBee Device / Sender** | ئەو نێرەر یان ئامێری TextBee هەڵدەبژێرێت کە بۆ پەیامە دەرچووەکان بەکاردێت، کاتێک دابینکەر پێویستی پێیەتی. |
| **SMS Notifications Enabled** | ڕێگە بە workflow ـە پەسەندکراوەکان دەدات ئاگادارکردنەوەی پەیامی دەقی بنێرن. بۆ ڕێگریکردن لە ناردنی SMS ناچالاکی بهێڵەوە. |

پەیامێکی تاقیکردنەوە تەنها بۆ ژمارەیەکی خاوەندارێتی بەڕێوەبەر بنێرە، پاشان ڕەفتاری دەروازەی پەسەندکردن پشتڕاست بکەرەوە پێش چالاککردنی بیرخستنەوەی خشتەکراو یان ڕووی لە بەشداربوو.

## Feature Flags

هەروەها لە v1.9.0 ناسێنراوە، تابی **Settings → Feature Flags** دوگمەی گۆڕینی چالاک/ناچالاک بۆ کارایییە ئارەزوومەندانەکان دابین دەکات. هەر flag ـێک یان چالاکە یان ناچالاکە بە شێوەی سەرانسەری تۆڕ؛ لەم کاتەدا هیچ جێگرەوەیەکی تایبەت بە ماڵپەڕ نییە.

### چوونەناو Feature Flags

1. لە بەڕێوەبەرایەتی WordPress، بچۆ بۆ **Gratis AI Agent → Settings**.
2. کلیک لە تابی **Feature Flags** بکە.

### Flag ـەکانی کۆنترۆڵی دەستپێگەیشتن

| ئاڵا | بنەڕەتی | وەسف |
|---|---|---|
| **سنووردارکردن بۆ بەڕێوەبەران** | ناچالاک | کاتێک چالاک بکرێت، تەنها ئەو بەکارهێنەرانەی ڕۆڵی `administrator`یان هەیە دەتوانن پانێڵی چاتی AI Agent بکەنەوە. هەموو ڕۆڵەکانی دیکە لە جیاتی ئەوە پەیامی "پەیوەندی بە بەڕێوەبەرەکەت بکە" دەبینن. |
| **سنووردارکردن بۆ Network Admins** | ناچالاک | کاتێک لەسەر تۆڕێکی multisite چالاک بکرێت، تەنها Super Admins دەتوانن agent بەکاربهێنن. بەڕێوەبەرانی ماڵپەڕە تاکەکان بلۆک دەکرێن. ئەگەر هەردووکیان چالاک بن، پێشینەیی لەسەر "سنووردارکردن بۆ بەڕێوەبەران" دەگرێت. |
| **ڕێگەدان بە دەستگەیشتنی Subscriber** | ناچالاک | کاتێک چالاک بکرێت، بەکارهێنەرانی خاوەن ڕۆڵی `subscriber` دەتوانن ڕووکاری چات بەکاربهێنن، بەڵام سنووردار دەبن بە تواناکانی تەنها-خوێندنەوە (دروستکردنی بابەت یان گۆڕینی ڕێکخستنەکان نییە). |
| **ناچالاککردن بۆ ئەندام نەبووان** | ناچالاک | لەگەڵ دۆخی ئەندامێتی Ultimate Multisite یەکدەخرێت. کاتێک چالاک بکرێت، چات بۆ ماڵپەڕەکانی کە ئەندامێتییەکی چالاکیان نییە دەشاردرێتەوە. |

### ئاڵاکانی براندینگ

| ئاڵا | بنەڕەتی | وەسف |
|---|---|---|
| **شاردنەوەی ژێرپەڕەی "Powered by Gratis AI Agent"** | ناچالاک | دێڕی دیاریکردنی براندینگ کە لە خوارەوەی ویدجێتی چات پیشان دەدرێت لادەبات. بۆ جێبەجێکردنی white-label پێشنیار دەکرێت. |
| **ناوی تایبەتی Agent** | *(بەتاڵ)* | پێناسەی بنەڕەتی "Gratis AI Agent" لە سەردێڕی چات و مێنیوی بەڕێوەبەرایەتی بە ناوی بەرهەمی خۆت دەگۆڕێت. بۆ بەکارهێنانی بنەڕەتی بەتاڵی بهێڵەرەوە. |
| **شاردنەوەی هەڵبژێری Agent** | ناچالاک | کاتێک چالاک بکرێت، بەکارهێنەران ناتوانن لەنێوان پێنج agentی ناوخۆیی بگۆڕن. agentی ئێستا جێگیر دەبێت بەوەی وەک بنەڕەتی لە ڕێکخستنەکان → گشتی ڕێکخراوە. |
| **بەکارهێنانی ئایکۆنی ماڵپەڕ وەک ئەڤاتاری چات** | ناچالاک | ئایکۆنی بنەڕەتی AI لە سەردێڕی ویدجێتی چات بە ئایکۆنی ماڵپەڕی WordPress دەگۆڕێت (لە ژێر دەرکەوتن → خۆکاریکردن → ناسنامەی ماڵپەڕ دیاری دەکرێت). |

### ئاڵاکانی سەلامەتی ئۆتۆماتکردن

Superdav AI Agent v1.18.0 دەروازەکانی پەسەندکردنی مرۆیی و تۆمارەکانی بیرخستنەوە بۆ جێبەجێکردنە ئۆتۆماتیکییە سەلامەتترەکان دەناسێنێت. ئەم کۆنترۆڵانە دەتوانن لە ئاڵاکانی تایبەتمەندی یان ڕێکخستنە پێشکەوتووەکانی ئۆتۆماتکردن دەربکەون، بە پێی پاکێجی دامەزراو.

| ئاڵا | بنەڕەتی | وەسف |
|---|---|---|
| **پێویستکردنی پەسەندکردنی مرۆیی** | پێشنیار دەکرێت چالاک بێت | ئۆتۆماتکردنە هەستیارەکان ڕادەگرێت تا بەکارهێنەرێکی ڕێپێدراو کرداری پێشنیارکراو بپشکنێت و پشتڕاستی بکاتەوە. |
| **لابردنی دووبارەیی بیرخستنەوە** | چالاک | بیرخستنەوە نێردراوەکان تۆمار دەکات، بۆ ئەوەی هەوڵدانەوەکان یان جێبەجێکردنە خشتەکراوەکان ئاگادارکردنەوەی دووبارە نەنێرن. |
| **چالاککردنی ئامرازەکانی ساڵنامە** | ناچالاک تا ڕێکدەخرێت | ڕێگە بە agent دەدات ساڵنامە و ڕووداوە ڕێکخراوەکانی Google بخوێنێتەوە. |
| **چالاککردنی ئاگادارکردنەوەکانی SMS** | ناچالاک تا ڕێکدەخرێت | ڕێگە بە کارڕەوە پەسەندکراوەکان دەدات دوای پاشەکەوتکردنی زانیارییەکانی چوونەژوورەوە، ئاگادارکردنەوەی TextBee SMS بنێرن. |

### جێبەجێکردنی گۆڕانکارییەکان

کرتە لەسەر **پاشەکەوتکردنی ڕێکخستنەکان** بکە دوای گۆڕینی هەر ئاڵایەک. گۆڕانکارییەکان دەستبەجێ کاریگەر دەبن — هیچ پاککردنەوەی cache یان چالاککردنەوەی plugin پێویست نییە.
