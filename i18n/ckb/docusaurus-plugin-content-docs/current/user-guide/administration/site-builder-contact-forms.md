# Forms Kontaktir (Contact Forms) {#site-builder-contact-forms}

Superdav AI Agent v1.10.0 تێدروستی دەبێت بۆ دروستکردنی فۆرمەکانی پەیوەندی ڕاستەوخۆ لە ناو پڕۆسەی چات بە بەکارهێنانی Site Builder agent. ئەمەش وەک ئەوەیە کە بتوانید فۆرمەکان بە تەواوی کارکەر بن بۆ هەر پەڕەیەکی یەکێک لە چاتەکەدا بدون ئەوەی لە چاتەکە بگەڕێیتەوە.

## گشتی (Overview) {#overview}

قابلی دروستکردنی فۆرمەکانی پەیوەندی Site Builder (`create_contact_form`) بە شێوەیەکی ئۆتۆماتیک پلاتینی فۆرمەکەت کە دامەزراوە دەدات دیاری دەکات و فۆرمێکی پەیوەندیدار دروست دەکات بەکاری بەکاردان لە تواناکانی تایبەتی ئەو پلاتینەی هەیە. پلاتینی فۆرمە پشتگیریکراو ئەمانە:

- Contact Form 7
- WPForms
- Fluent Forms
- Gravity Forms

یاری دەستوەردانی ئاسانترین پلاتینی فۆرم لەسەر سایتەکەت هەڵدەبژێرێت و فۆرمێکی پەیوەندیدار بۆ ئەو پلاتینە دروست دەکات.

## دروستکردنی فۆرم پەیوەندیدار (Creating a Contact Form) {#creating-a-contact-form}

بۆ دروستکردنی فۆرم پەیوەندیدار بە بەکارهێنانی Site Builder:

1. **Gratis AI Agent** چات پانێلەکە لە ئادمن پڕۆسەی WordPress بکەرەوە.
2. بۆ ئەوەی بگۆڕیت بۆ Site Builder agent، دەتوانیت ئایکۆنی agent-ەکە لە سەری چاتەکەدا بێتە سەر.
3. ئەو فۆرم پەیوەندیدارە باس بکە کە دەتەوێت دروست بکەیت. بۆ نموونە:

   > "فۆرمێکی پەیوەندیدار بۆ پەڕەی Contact دروست بکە بە فیلدەکانی ناو، ئیمەیڵ، پەیام و ژمارەی تەلەفۆن."

   یان سادە بڵێ:

   > "فۆرم پەیوەندیداری بۆ پەڕەی Contact دروست بکە."

4. Site Builder فۆرم پەیوەندیدارەکە دروست دەکات و کۆدێکی کورت (shortcode) دەدات کە ئامادەیە بۆ داڕشتن (embed).

## بەکارهێنانی کۆدێکی کورت دروستکراو (Using the Generated Shortcode) {#using-the-generated-shortcode}

پاش ئەوەی Site Builder فۆرم پەیوەندیدارەکە دروست کرد، ئەو کۆدێکی کورت دەداتەوە (بۆ نموونە، `[contact-form-7 id="123"]`). دەتوانیت:

1. **لە یەکێک لە پەڕان یان پۆستێکدا داڕشتن (Embed it in a page or post)** — کۆدێکی کورتەکە بکۆ و لە هەر پەڕەیەکی یان پۆستێکدا بەکاربهێنە بە بەکارهێنانی block editor یان classic editor.
2. **بە بەکارهێنانی Site Builder زیادکردن (Add it via the Site Builder)** — داوا بکە لە Site Builder کە فۆرمەکە ئۆتۆماتیک بۆ پەڕەیەکی دیاریکراو زیاد بکات:

   > "فۆرم پەیوەندیدارەکە بۆ پەڕەی Contact زیاد بکە."

٣. **بکارئینیدە لە شابلۆ (template)** — ئەگەر ئارامی PHP دەکەیت، دەتوانیت ئەو shortcode یە بۆ شێوەیەکی تر لە فایلی شابلۆی تێدا زیاد بکەیت.

## گۆڕینی فۆرم کنتاکت (Contact Form) {#customizing-the-contact-form}

پاش ئەوەی Site Builder فۆرم کنتاکتەکەی دروست بکات، دەتوانیت زیاتر بگۆڕیت:

### بە ڕێگەی ئۆتۆماتیک چات (Chat Interface) {#via-the-chat-interface}

بپرسە لە Site Builder بۆ گۆڕینی فۆرمەکە:

> "فۆرم کنتاکتەکە به‌روزرسیک بکە بۆ زیادکردنی فیلدی بابەت (subject field) و دروستکردنی فیلدی پەیام (message field) بە پێویستی."

Site Builder فۆرمەکە دەگۆڕێتەوە و shortcode ی گۆڕاوەکە دەداتەوە.

### بە ڕێگەی ئۆتۆماتیک ئەدمنی فۆرم پلەگینی (Form Plugin's Admin Interface) {#via-the-form-plugins-admin-interface}

دەتوانیت فۆرمەکە ڕاستەوخۆ لە تنظیماتەکانی خودی form plugin-ەکەت بگۆڕیت:

١. بچۆ بۆ **Contact Form 7** (یان form plugin-ی تۆ کە دامەزراوە) لە admin ی WordPress.
٢. فۆرمەکە بدۆزەرەوە کە لەلایەن Site Builder دروستکراوە.
٣. فیلدەکانی فۆرم، ئەو ڕێزموونانەی بۆ بەڕێوەبردنی (validation rules)، و ئیمەیڵ نوێکردن دەستکاری بکە وەک پێویستە.

## پلەگینەکانی فۆرم و یەکگرتوویی (Compatibility) {#form-plugins-and-compatibility}

Site Builder بە شێوەیەکی ئۆتۆماتیک دەدۆزێتەوە کە کام form plugin لەسەر سایتەکەت دامەزراوە و بۆ دروستکردنی فۆرم کنتاکتەکە لەو پلەگینە بەکاری دێت. ئەگەر چەندین form plugin دامەزراو بێت، Site Builder پێشەمستی دەدات بەم ڕێژەیە:

١. Contact Form 7
٢. WPForms
٣. Fluent Forms
٤. Gravity Forms

ئەگەر هیچ یەک لەم پلەگینانە دامەزراو نەبوون، Site Builder پێشنیار دەکات کە یەکێکیان دامەزێنیت پێش دروستکردنی فۆرم کنتاکتەکە.

## ئیمەیڵ نوێکردن (Email Notifications) {#email-notifications}

فۆرمەکانی کنتاکت کە لەلایەن Site Builder دروست دەکرێن، بە شێوەیەکی پیش‌تەوا، ڕێکخراون بۆ ئەوەی ئیمەیڵی نوێکردن بۆ ئیدارەی سایتەکە بڵاوبکەن. دەتوانیت ئیمەیڵی گیروانەکەی و پەیامی نوێکردنەکە بگۆڕیت:

١. بچۆ بۆ admin ی form plugin-ەکەت.
٢. فۆرمەکە بدۆزەرەوە کە لەلایەن Site Builder دروستکراوە.
٣. تنظیماتەکانی ئیمەیڵ نوێکردنەکە دەستکاری بکە.

بۆ ڕێنمایی زیاتر، سەردانی dokumantasi form plugin-ەکەت بکە:

- [ئەمرۆیی نوتifikەیشنەکانی Contact Form 7](https://contactform7.com/docs/)
- [ئەمرۆیی نوتifikەیشنەکانی WPForms](https://wpforms.com/docs/)
- [ئەمرۆیی نوتifikەیشنەکانی Fluent Forms](https://fluentforms.com/docs/)
- [ئەمرۆیی نوتifikەیشنەکانی Gravity Forms](https://docs.gravityforms.com/)

### فۆرمێک لەگەڵ لۆجیکی بەراوردکراو (Conditional Logic) {#form-with-conditional-logic}
