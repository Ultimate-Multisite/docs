# مهارات آگاه به قالب

Superdav AI Agent v1.10.0 چهار مهارت جدید داخلی و آگاه به قالب را عرضه می‌کند که به طور خودکار با قالب وردپرس فعال شما سازگار می‌شوند. این مهارت‌ها راهنمایی‌ها و قابلیت‌های تخصصی را ارائه می‌دهند که متناسب با معماری و ویژگی‌های قالب شما تنظیم شده‌اند.

## مهارت‌های آگاه به قالب چیست؟ {#what-are-theme-aware-skills}

مهارت‌های آگاه به قالب، پایگاه‌های دانش و مجموعه‌های ابزاری از پیش پیکربندی شده‌ای هستند که دستیار هوش مصنوعی بر اساس قالب وردپرس فعال در سایت شما به طور خودکار انتخاب می‌کند. وقتی شما بین قالب‌ها جابه‌جا می‌کنید، مهارت‌های موجود دستیار به صورت خودکار به‌روز می‌شوند — نیازی به تنظیم دستی نیست.

هر مهارت شامل موارد زیر است:

- **مستندات خاص قالب** — راهنمایی‌هایی برای استفاده و سفارشی‌سازی قالب
- **ارجاعات بلوک و الگو (pattern)** — بلوک‌ها، الگوها و گزینه‌های طراحی موجود
- **نمونه‌های سفارشی‌سازی** — قطعات کد و الگوهای پیکربندی برای کارهای رایج
- **بهترین شیوه‌ها (Best practices)** — توصیه‌هایی برای معماری و گردش کار قالب

## مهارت‌های آگاه به قالب موجود {#available-theme-aware-skills}

### قالب‌های بلوکی (Block Themes) {#block-themes}

**مناسب برای:** قالب‌هایی که از معماری مبتنی بر بلوک وردپرس (Full Site Editing) استفاده می‌کنند.

مهارت قالب‌های بلوکی راهنمایی‌هایی در مورد موارد زیر ارائه می‌دهد:

- ایجاد و ویرایش قالب‌ها با استفاده از ویرایشگر بلوک
- کار با الگوها و بلوک‌های قابل استفاده مجدد
- سفارشی‌سازی استایل‌های سراسری و تنظیمات theme.json
- استفاده از بلوک‌ها و تغییرات (variations) قالب
- ساخت الگوهای بلوکی سفارشی برای سایت خود

**به طور خودکار فعال می‌شود وقتی:** قالب فعال شما یک قالب بلوکی باشد (پشتیبانی از ویژگی `block-templates`).

### قالب‌های کلاسیک (Classic Themes) {#classic-themes}

**مناسب برای:** قالب‌های سنتی وردپرس که از قالب‌های PHP و ویرایشگر کلاسیک استفاده می‌کنند.

مهارت قالب‌های کلاسیک راهنمایی‌هایی در مورد موارد زیر ارائه می‌دهد:

کارکردن لەگەڵ فایلە template PHP و hooks
پێناساندنی ڕووخسارێ تێمەکە بە شێوەیەکی تایبەت (Customizer)
فراوانکردنی dartboard هاوکیدا (widget areas) و sidebar'ەکان
تغییردەدانی CSS و پەرەپێدانی child theme
فهمی هێزی تێم (theme hierarchy) و template tags

**ئەمە بە شێوەیەکی ئۆتۆماتیک دەست پێ دەکات کاتێک:** تێمەکە کە چالاکەیت کلاسیکی (ناوەکە) بێت (ناتوانرێت block بێت).

### Kadence Blocks {#kadence-blocks}
**پێناساندنی:** بۆ ئەو سایتانەی بەکارھێنانی plugin-ی Kadence Blocks بۆ دیزاینی پێشکەوتوو لەسەر بنەمای block.

فێری Kadence Blocks ڕێنمایی دەدات لەسەر:

- بەکارهێنانی کۆمەڵەی پیشەنگاوەکانی Kadence (وەک Hero، Testimonials، Pricing و هتد)
- ڕێکخستنی تنظیمەکانی kadence block و پيشنیاسەتەکان (responsive options)
- دروستکردنی لێدوانە تایبەت بە بەکارهێنانی grid و container blocks ی Kadence
- یەکگرتووکردنەوەی kadence blocks لەگەڵ تێمەکەت
- بەکارهێنانی سیستەمی دیزاینی (design system) و پيشنیاسەتەکانی kadence

**ئەمە بە شێوەیەکی ئۆتۆماتیک دەست پێ دەکات کاتێک:** plugin-ی Kadence Blocks لەسەر سایتەکەت چالاکە.

### Kadence Theme {#kadence-theme}
**پێناساندنی:** بۆ ئەو سایتانەی بەکارھێنانی تێمەکە (theme) ی Kadence بۆ دیزاینی بنەمای block و تایبەتمەندکردنەوە.

فێری Kadence Theme ڕێنمایی دەدات لەسەر:

- تغییردەدانی تێمەکەی kadence لە ڕێگەی global styles و theme.json
- بەکارهێنانی پترۆڵە (patterns) و template ی ناوخۆییەکانی kadence
- ڕێکخستنی تنظیمەکانی تێمەکە و پيشنیاسەتەکان
- دروستکردنی دیزاینە تایبەت بە سیستەمی دیزاینی kadence
- یەکگرتووکردنەوەی kadence لەگەڵ plugin و ئامرازە بەناوبانکراوەکان

**ئەمە بە شێوەیەکی ئۆتۆماتیک دەست پێ دەکات کاتێک:** تێمەکە (theme) ی Kadence چالاکە.

## چۆن Skills هەڵدەبژێردرێن؟ {#how-skills-are-selected}

Assistant بە شێوەیەکی ئۆتۆماتیک تێمەکەت و plugin-ەکان کە لەسەر هەر پەیامێکدا دەدات، دیاری دەکات. ئەگەر فێری یەکگرتووی پەیوەندیدار بە تێمەکە بێت موجود بێت، بە شێوەیەکی ئۆتۆماتیک بۆ contextی assistant دابنێریت. پێویست نییە بە دەستی هەڵدەبژێریت یان بگۆڕیت Skills.

### چەندین Skill (Multiple Skills) {#multiple-skills}

ئەگەر چەند مهارەتێک بۆ سایتەکەت دەگونجێن (بۆ نموونە، ئەگەر هەردوو Kadence Blocks و Kadence Theme چالاک بێت)، ئاسستێنت دەتوانێت دەسیل سەر هەردوو مهارەتی بەکاربهێنانی بکات و لە هەر یەکێکیان ڕێنمایی وەربگرێت.

### گۆڕینی Theme (سایتی) {#switching-themes}

کاتێک تۆ theme-ی چالاکت دەگۆڕیت، مهارەتەکانی ئاسستێنت بە شێوەیەکی خۆدۆز لە پەیامە دوایەکەدا به‌روزرسト دەبن. بۆ نموونە:

1. تۆ لە ئێستادا لە کار使用کردنی theme-ی بلاک (block theme) و مهارەتی **Block Themes** چالاکە.
2. تۆ گۆڕان بە theme-ێکی کلاسیکی (classic theme).
3. لە پەیامەکەت دوای ئەوە، مهارەتی **Classic Themes** بە شێوەیەکی خۆدۆز باردەبێت، و مهارەتی **Block Themes** زیاتر نییە.

## بەکارهێنانی مهارەتەکانی ئاوەزدار Theme (Theme-Aware Skills) {#using-theme-aware-skills}

بۆ بەکارهێنانی مهارەتی ئاوەزدار theme، تەنها ئەو شتەی دەتەوێت لە ناو ڕێکارەکەدا باس بکەیت. ئاسستێنت بە شێوەیەکی خۆدۆز سەرنج دەگرێت و ڕێنمایی مهارەتی گونجاو وەردەگرێت.

### نموونەکانی پرسیارکردن (Example Prompts) {#example-prompts}

**بۆ Block Themes:**
> "سەرچاوەیەکی سەرنجڕاکێش (hero section) بە وێنەی پسراوە و دەقێکی ناوەندکراو دروست بکە بە شێوازەکانی بلاک پترن (block patterns)."

**بۆ Classic Themes:**
> "خاڵێکی ویجێت (widget area) تایبەت بۆ لایەنی کاتژمێرەکە (sidebar) بە بەشی child theme زیاد بکە."

**بۆ Kadence Blocks:**
> "سەرچاوەیەکی گواستنەوەی شایەنی خەڵک (testimonials section) دروست بکە بە بەکارهێنانی بلاک Kadence Testimonials."

**بۆ Kadence Theme:**
> "دابەشکردنی پێکهاتەی ساردی (header layout) و ڕێنمایی ستایلینگ مێنیووی ناوبانێیەکە (navigation menu styling) بگۆڕە."

ئاسستێنت ڕێنمایی-ی تایبەت بە theme و نموونەکانی کۆد دەدات کە بۆ theme و plugin-ی چالاکت پێکهاتووە.

:::note
مهارەتی ئاوەزدار theme بە شێوەیەکی خۆدۆز لە Superdav AI Agent v1.10.0 و دوای ئەوەکاندا بە ئۆتۆماتیک دەبێتەوە. پێویستی بە ڕێکخستن یان پیکیری زیاتر نییە.
:::
