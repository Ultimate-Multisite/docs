# جستجوی اینترنت {#internet-search}

Gratis AI Agent v1.5.0 یک قابلیت **جستجوی اینترنت (Internet Search)** اضافه کرده که به دستیار هوش مصنوعی اجازه می‌دهد در طول گفتگو اطلاعات به‌روز از وب بازیابی کند. این قابلیت توسط [Brave Search API](https://brave.com/search/api/) یا [Tavily API](https://tavily.com/) پشتیبانی می‌شود.

## نحوه کارکرد {#how-it-works}

وقتی جستجوی اینترنت فعال باشد، دستیار به طور خودکار زمانی که تشخیص دهد یک سؤال نیاز به اطلاعات جاری یا خارجی دارد — مثلاً اخبار اخیر، قیمت‌های زنده، یا مستنداتی که ممکن است از زمان قطع آموزش مدل تغییر کرده باشند — می‌تواند از ارائه‌دهنده جستجوی پیکربندی شده شما پرس‌وجو کند.

نتایج در لحظه بازیابی شده و قبل از اینکه دستیار پاسخی تولید کند، در زمینه (context) دستیار قرار می‌گیرند. دستیار نشان می‌دهد که چه زمانی از نتایج جستجو برای پاسخ دادن به یک سؤال استفاده کرده است.

## فعال کردن جستجوی اینترنت {#enabling-internet-search}

جستجوی اینترنت نیاز به کلید API از ارائه‌دهنده جستجوی انتخابی شما دارد. برای پیکربندی آن:

۱. به **Gratis AI Agent → Settings → Advanced** بروید.
۲. منوی کشویی **Internet Search Provider** را پیدا کرده و یکی از **Brave Search** یا **Tavily** را انتخاب کنید.
۳. کلید API خود را در فیلد مربوطه وارد کنید. لینک‌های ثبت‌نام در کنار هر فیلد نمایش داده می‌شوند.
۴. روی **Save Settings** کلیک کنید.

پس از ذخیره کلید، قابلیت جستجوی اینترنت به طور خودکار برای دستیار در دسترس خواهد بود.

## Brave Search {#brave-search}

### دریافت کلید API Brave Search {#obtaining-a-brave-search-api-key}

۱. به [صفحه Brave Search API](https://brave.com/search/api/) بروید.
۲. برای یک طرح (plan) ثبت‌نام کنید. یک سطح رایگان با محدودیت درخواست ماهانه در دسترس است.
۳. کلید API خود را از داشبورد توسعه‌دهنده Brave Search کپی کنید.
۴. آن را در فیلد **Brave Search API Key** در تنظیمات Gratis AI Agent قرار دهید.

### محدودیت‌های استفاده {#usage-limits}

استفاده لە خزمەتگوزاریەکە بەپێی ژمارەی پرسیارەکان کە دەکەن، لەلایەن Brave Searchەوە بڕ دەدات. هەر وەڵامێکی AI کە لێی دەگەڕێتەوە بۆ گەڕان، وەک یەک پرسیار دادەنرێت. بۆ ئەوەی پارە بە شێوەیەکی ناخۆش نەبێت، بەکاری دەهێنە لە [داشبۆرد دۆزنەکانی پەرەپێدانی Brave Search](https://brave.com/search/api/).

## Tavily {#tavily}

Superdav AI Agent v1.10.0 **Tavily** وەک سەرچاوەیەکی جیاواز بۆ گەڕان لە ئینتەرنێت زیاد دەکات، کە ئەنجامی گەڕانێکی بەرفراوانتر و تواناکانی تێگەیشتنێکی پێشکەوتوو دەبەخشێت.

### بەدەست هێنانی کلیدی APIی Tavily {#obtaining-a-tavily-api-key}

1. سەردانی [لاپەڕەی APIی Tavily](https://tavily.com/) بکە.
2. بۆ هەژمار تۆمار بکە. پلەیەکی بێ بەرامبەر هەیە کە سنوورێکی مانگانەی داواکاری دەدات.
3. کلیدی APIی خۆت لە داشبۆردی Tavily کۆ بکەرەوە.
4. لە فیلدی **Tavily API Key** لە ڕێکخراوەکانی Gratis AI Agentدا چێوکرێت.

### سنوورەکانی بەکارهێنان {#usage-limits-1}

استفاده بەپێی ژمارەی پێداویستی APIی Tavily بڕ دەدات. هەر وەڵامێکی AI کە لێی گەڕان دەکات، وەک یەک داواکاری دادەنرێت. بۆ ئەوەی پارە بە شێوەیەکی ناخۆش نەبێت، لە [داشبۆردی Tavily](https://tavily.com/) بەکاری دەهێنە.

## ناتواندنەوەی گەڕان لە ئینتەرنێت {#disabling-internet-search}

کلیدی APIیەکە لە فیلدی سەرچاوەی گەڕانی چالاک و هەڵبژاردنی (active search provider field) پاک بکە و سیوکرێت (save). توانای گەڕان لە ئینتەرنێت دووبارە پێشکەش ناکرێت بۆ یارمەتیدەرەکە.

:::note
گەڕان لە ئینتەرنێت کێشەی خێرایی (latency) بە وەڵامەکان دەدات چونکە یارمەتیدەرەکە دەبێت بانتظامی ئەنجامی گەڕان بکات پێش ئەوەی وەڵامێک بدات. بۆ ئەو حاڵەتانەی کە کاتیان گرنگن، بیر لەوە بکەرەوە کە ئایا گەڕانی ڕاستەوخۆ (real-time search) پێویستە یان ئایا زانیارییە ناوخۆییەکانی یارمەتیدەرەکە بەسودهی دەبێت.
:::
