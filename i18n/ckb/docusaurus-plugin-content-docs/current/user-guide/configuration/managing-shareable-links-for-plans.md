# Manage Shareable Links for Plans {#managing-shareable-links-for-plans}

Ultimate Multisite به شما این امکان را می‌دهد که لینک‌های طرح‌ها را مستقیماً با مشتریانتان به اشتراک بگذارید. برای انجام این کار، به هر یک از صفحات ویرایش طرح‌های خود بروید و روی دکمه _Click to copy the Shareable Link_ کلیک کنید.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_در صفحه ویرایش طرح، می‌توانید دکمه ایجاد لینک‌های قابل اشتراک را پیدا کنید._

توجه داشته باشید که این مکانیزم از فرکانس صورت‌حساب پیش‌فرض برای ساخت لینک استفاده می‌کند. این بدان معناست که اگر صورت‌حساب پیش‌فرض شما ماهانه باشد، به عنوان مثال، در نهایت لینکی شبیه این خواهید داشت: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

اگر می‌خواهید لینک قابل اشتراک به طور پیش‌فرض سه ماهه باشد، باید ۳ را در انتها اضافه کنید:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

برای انتخاب طرح _premium_ بر اساس برنامه سالانه، باید ۱ و سپس کلمه "year" (سال) را اضافه کنید:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

به یاد داشته باشید که شما نمی‌توانید فقط یک لینک برای فرکانس‌های صورت‌حساب مختلف داشته باشید. باید برای هر فرکانس صورت‌حساب خود یک لینک خاص به اشتراک بگذارید.

هنگامی که لینک را با مشتریان خود به اشتراک می‌گذارید، آنها به اولین مرحله از جریان ثبت‌نام شما هدایت خواهند شد.
