# فرستادن ایمیل‌ها و پخش (نسخه ۲)

_**توجه مهم: این مقاله به Ultimate Multisite نسخه ۲.x اشاره دارد.**_

Ultimate Multisite یک قابلیت دارد که به شما اجازه می‌دهد با ارسال ایمیل به کاربر هدف یا گروهی از کاربران، و همچنین ارسال اطلاعیه‌ها در داشبورد مدیریت آن‌ها، با مشتریان خود ارتباط برقرار کنید.

## افزودن اطلاعیه‌های مدیریتی به داشبورد مشتریان با پخش‌ها (Broadcasts)

با استفاده از قابلیت پخش Ultimate Multisite، می‌توانید **اطلاعیه‌های مدیریتی** را به داشبورد مدیریت زیرساخت هر کاربر اضافه کنید.

این کار برای زمانی که نیاز دارید اطلاعیه‌ای مانند نگهداری سیستم یا ارائه محصولات و خدمات جدید به کاربران فعلی خود داشته باشید، بسیار مفید است. اینطور در داشبورد کاربر شما خواهد بود.

<!-- Screenshot unavailable: نمایش اطلاعیه مدیریتی پخش شده روی داشبورد زیرساخت مشتری -->

برای شروع یک اطلاعیه مدیریتی، به داشبورد مدیریت شبکه خود بروید و در منوی **Ultimate Multisite**، گزینه **Broadcasts** را پیدا خواهید کرد.

![صفحه لیست پخش‌ها در مدیریت Ultimate Multisite](/img/admin/broadcasts-list.png)

همچنین می‌توانید پخش‌های موجود را ویرایش کنید:

![رابط ویرایش پخش](/img/admin/broadcast-edit.png)

از این صفحه، روی دکمه **Add Broadcast** (افزودن پخش) در بالا کلیک کنید.

این کار پنجره مودال Add broadcast را باز می‌کند که می‌توانید نوع پخشی را که می‌خواهید ارسال کنید انتخاب نمایید.

ادامه دهید و **Message** (پیام) را انتخاب کرده سپس روی دکمه **Next Step** (گام بعدی) کلیک کنید.

![پنجره مودال افزودن پخش با انتخاب گزینه Message](/img/admin/broadcast-add-message.png)

پنجره بعدی از شما می‌پرسد که یا **Target customer** (مشتری هدف) را یا **Target product** (محصول هدف) را انتخاب کنید. توجه داشته باشید که می‌توانید بیش از یک کاربر یا بیش از یک محصول را انتخاب نمایید.

باشە بۆ گەڕان بەدوای یەکۆری کاربری یان پڕۆدۆکت، دەبێت دەستپێکردنی تایپکردنی ئەو وشەیە لەناو فیلدەکەدا بکەیت.

لە خاڵی **Message type** (جۆری پەیام)، دەتوانیت ڕەنگی ئاگادارییەکە هەڵبژێریت. ئەمە گرنگی پەیامەکەت بەهێز دەکاتەوە.

ئەو کات دەتوانیت **Next Step** (پوختەی دواتر) بێت.

![Target customers, target product and message type fields for a Message broadcast](/img/admin/broadcast-message-targets.png)

پەڕەی دوایەکە ئەو شوێنەیە کە دەتوانیت دەستپێبکەیت بە نووسینی پەیامەکەت بە بنچینەوە و ناوەڕۆکی ئەو پەیامەی دەتەوێت بۆ بەکارهێنەران بڵێیت.

![Broadcast message subject and content editor on the compose step](/img/admin/broadcast-edit.png)

پاش دروستکردنی پەیامەکەت، دەتوانیت **Send** (نێردران) دابەزێنیت.

و ئەمە هەموو کارەکە. ئاگاداری ئەدمن دەبێت بە خێرا لەسەر داشبۆردی بەکارهێنەرەکەت نیشان بدرێت.

## بۆ بەڵاوت بۆ کڕیارەکانت

بەکارھێنانی تایبەتمەندی بڵاوکردنەوەی Ultimate Multisite، دەتوانیت ئیمەیلی بۆ بەکارهێنەرەکانت بنێریت. هەڵبژاردەیەکی هەیە کە یەکەمین کار ئەوەیە ئیمەیڵەکە تەنها بۆ بەکارهێنەرانی دیاریکراو بنێریت یان گرووپێکی تایبەت لە بەکارهێنەران گرنگ بکەیت پێکەوە، بەپێی ئەو پڕۆدۆکت یان پلانیانەی کە سەر بەوە دەبن.

بۆ دەستپێکردنی بڵاوکردنەوەی ئیمەیڵ، بگواشبە داشبۆردی ئەدمن تۆڕەکەت و لە خوارەوە لە مینیوی Ultimate Multisite، دەتوانیت کارتی Broadcast (بڵاوکردنەوە) ببینیت.

![Broadcasts list page used as the starting point for an email broadcast](/img/admin/broadcasts-list.png)

لە ئەم پەڕە، دابەزێنە دایەنێکی **Add broadcast** (بەکارھێنانی بڵاوکردنەوە).

ئەم کارە دەبێت مۆدال ویندۆکەی Add broadcast بەرز بکات کە لەوێدا دەتوانیت هەڵبژێریت چ جۆرە بڵاوکردنەوەیەک دەتەوێت بنێریت. پێشوەختان **Email** (ئیمەیڵ) هەڵبژێرە و دوای ئەوە دابەزێنە دایەنێکی **Next Step**.

![Add broadcast modal with the Email option selected](/img/admin/broadcast-add-email.png)

پەڕەی دوایەکە پێشکەش دەکات یان **Target customer** (کڕیار دیاریکراو) یان **Target product** (پڕۆدۆکت دیاریکراو). ئاگاداری ئەوە بێت کە دەتوانیت زیاتر لە یەک بەکارهێنەر یان زیاتر لە یەک پڕۆدۆکت هەڵبژێریت.

بۆ گەڕان بۆ یەکپۆشێکی کاربری یان بەرهەم، دەبێت دەست بە تایپکردنی ئەو وشەیە بکە کە دەتەوێت لەناو فیلدەکەدا بنووسیت.

کاتێک گرێی سەرەکی خۆت هەڵدەبژێریت، دەتوانیت **Next Step** (خوونی پاشەکەوت) بگەڕێیتەوە.

![Target customers and target product selection for an Email broadcast](/img/admin/broadcast-email-targets.png)

پەڕەی دووەم ئەو شوێنە کە لەوێ دەتوانیت دەست بە نووسینی ئیمێلکەکەت بکەیت بە پڕکردنەوەی بابەت و ناوەڕۆک/پیغامێک کە دەتەوێت بۆ یوانە بڵێیت.

<!-- Screenshot unavailable: Email broadcast subject and content editor on the compose step -->

پاش دروستکردنی پیغامەکەت، دەتوانیت دابەزێن بەورە (Send).

وە ئەمە چەند ئاسانە بۆ دانانی ئیمێل بۆ یوانە بە بەکارهێنانی تایبەتمەندی broadcast.

## ئیمێلەکانی سیستەم (System emails)

ئیمێلەکانی سیستەم لە Ultimate Multisite، ئەو **ئاخفتنە ئۆتۆماتیکییە** کە سیستەم دوای کارکردنەکان وەک تۆمارکردن، پارەدان، مۆڵپەندی ناوچەکان و هتد، دەدات. ئەم ئیمێلانە دەتوانیت لە ڕێگەی تنظیماتەکانی Ultimate Multisite بگۆڕیت یان دەستکارییان بکەیت. هەروەها تایبەتمەندییەکی هەیە کە دەبێت بگەڕێیتەوە و تنظیماتە ئێستەکان لە جێهێشتنێکی Ultimate Multisite دیکەนำوارد بکەیت.

### گەڕاندەوە وนำواردکردن (Resetting & Importing)

وەرگرتنی وەرگرتنی نوێیەکانی Ultimate Multisite، هەروەها add-ons، دەتوانن لە کاتێکدا ئیمێل نوێکان تۆمار بکەن.

بۆ ڕێگرتن لە تێکچوون و کێشەی دیکەکان، **ئێمە نێویەری شابلۆکانی ئیمێل نوێیە وەک ئیمێلی سیستەم لەسەر دامەزراوەکەت بە شێوەیەکی ئۆتۆماتیک زیاد ناکەین**، ئەگەر بۆ کارکردنی وردی تایبەتمەیەکی دیاریکراو گرنگ بێت.

بەڵام، سوپەر ئادمین و ئەژمارەکان دەتوانن ئەم ئیمێلە نوێیە بە شێوەیەکی ناو ئامێرێکیนำواردکردن (importer tool)นำوارد بکەن. ئەو پڕۆسەیە ئیمێلێکی سیستەمێکی نوێ لەگەڵ ناوەڕۆک و ڕێکخستنی شابلۆی ئیمێلەکە دروست دەکات، کە دەبێت سوپەر ئادمین بتوانێت هەر گۆڕانکارییەک بکات یان بەم شێوەیە بپارێزێت.

#### چۆن ئیمێلەکانی سیستەمนำوارد بکەیت (How to import system emails)

بڕۆ بۆ صفحە ڕێکخستنی Ultimate Multisite Settings و بچوویت بۆ تابلۆی **Emails**.

![تابلۆی Emails لە ڕێکخستنی Ultimate Multisite کە بەشی System Emails دەردەکەوێت](/img/config/settings-emails-tab.png)

پاشان، لەسەر لایەنی دووەم (sidebar)، بژێرە بۆ دڵنیابوون لە کارکردن، دugu **Customize System Emails** دابنێ.

<!-- Screenshot unavailable: دۆخی Customize System Emails لە پۆستەی System Emails لە لایەنی دووەم -->

لە لاپەڕەی System Emails، دەتوانیت دەکەیت بەشێوەیەکی سەرەوە کارکردن بۆ **Reset & Import** وەڵامەکە بکەیت. پاشان، ئەگەر ئەو دۆخییە هەڵبژێریت، دەبێت پەنجەرەی import و reset بکەیت بگشاڵیت.

![دۆخی Reset or Import لە لاپەڕەی admin System Emails](/img/admin/system-emails-reset-import.png)

پاشان، دەتوانیت کارکردن بە ئارامی آپشنەکانی Import Emails بۆ بینینی ئەو سێتەیم ئەلیکراوانەی کە دەکرێت گرتەوە یان دابنێیت.

<!-- Screenshot unavailable: دۆخی Reset and Import لە پۆستەی System Emails کە آپشنەکانی Import Emails بازنەکراون -->

#### ریستکردنی System Emails

کاتێکی تر، دەتوانیت هەست بکەیت کە ئەو گۆڕانکارییانەی کردووە بۆ قالبێکی ئیمەیڵ دیاریکراو زیاتر کار ناکەن و دەتەوێت بیکەیت بە **دۆخی سەرەتایی**ی خۆی.

لە چەند حاڵەتی ئەم مەبەستدا، دوو هەڵبژاردەیە دەبینیت: دەتوانیت سادە بکەیت System Emailەکە لەناو ببیتەوە و دواتر گرتەوە (بەکارھێنانی ڕێنماییەکانی سەرەوە) - کە ئەم کارە بەڵگەنامەی فرستکردن و شتەکان دەبڕێت، بۆیە ئەم ڕێگایە کەمترین پێداویستیی هەیە.

یان دەتوانیت لە **Reset & Import tool** بەکاربهێنیت بۆ ریستکردنی ئەو قالبەی ئیمەیڵەکە.

بۆ ریستکردنی قالبێکی ئیمەیڵ، دەتوانیت هەنگاوەکان پەیڕەو بکەیت تا بگەیتە Reset & Import tool، پاشان، آپشن **Reset** وەڵامەکەی دابنێ و ئەو ئیمەیڵانە هەڵبژێرە کە دەتەوێت بە ناوەڕۆکی سەرەتاییی خۆیان دەگەڕێنەوە.
