# مهارەتی پێداویستیی سایت (Site Specification Skill) {#site-specification-skill}

مهارەتی پێداویستیی سایت (Site Specification skill) ڕێگایێکی ڕێکخراو بۆ تۆمارکردنی ئامانجەکان، خەڵکی یەکەم و ناسنامەی براندەکەت. ئەم زانیارییە لە یادەوەری **site_brief**-ەکەتدا هەڵدەگیرێت، کە کارگەکان لە هەموو جلساتەکانی دواتر داواکاریی پێدەکەن بۆ ئەوەی یارمەتییەکی پێکەنین و بەپێی دۆخ بدەن.

## Site Specification چییە؟ {#what-is-site-specification}

Site Specification بریتییە لە پڕۆسەی تۆمارکردنی:

- **بابەت/ئامانجی سایتەکە (Site purpose)**: سایتەکەت چی دەکات و بۆچی هەیە
- **خەڵکی ئامانجدار (Target audience)**: کی سەردانی سایتەکەت دەکات و چیت پێویستییانە
- **ناسنامەی براند (Brand identity)**: ڕەنگەکانت، توندی دەنگیی تۆ، و شێوازی بینراویاکەت
- **ئامانجەکانی کارگێڕیی (Business goals)**: سەرکەوتن بۆ سایتەکەت چۆن دەربکێت
- **پێکهاتەی ناوەڕۆک (Content structure)**: چۆن سایتەکەت ڕێکخراوە

ئەم پێداویستییە دەبێتە **site_brief** ی تۆ، یادەوەری پێکەنین کە کارگەکان لەلایەن ئەوەیەوە بە دۆخی سایتەکەمان تێدەگەن.

## بۆچی Site Specification بەکاربهێنیت؟ {#why-use-site-specification}

### پێکەنین لە هەموو جلساتەکاندا (Consistency Across Sessions) {#consistency-across-sessions}

بێ site_brief، پێویست دەبێت هەر جارێک کە دەستپێدەکەیتەوەی جلسەیەکی نوێ، ئامانجی سایتەکەت بەردەوام ڕوون بکەیتەوە. لەگەڵ ئەمە، کارگەکان دەتوانن بە خێرایی تێبگەن:

- ئامانج و خەڵکی ئامانجدارانی سایتەکەت
- ڕەنگ و توندی دەنگیی براندەکەت
- پێکهاتەی ناوەڕۆکەکەت
- ئامانجە کارگێڕییەکانت

### پێشنیارە باشتر (Better Recommendations) {#better-recommendations}

کارگەکان لە site_brief ی تۆ بەکاری دەهێنن بۆ:

- پێشنیاری تایبەتمەندییەکانی وەک ئەو شتانەی کە ڕێکدەخەن لەگەڵ ئامانجی سایتەکەتدا
- پێشنیارکردنی پێکهاتەی ناوەڕۆکێک کە لەگەڵ ئامانجەکانتدا یەک دەبێت
- پێشنیاری دیزاینێک کە لەگەڵ ناسنامەی براندەکەتدا ڕێکدەخات
- گەیشتن بەوەی کە تایبەتمەندییە نادەستکارییەکان پێشنیار بکرێن

### دەستپێکردنی خێراتر (Faster Onboarding) {#faster-onboarding}

کارگەکان نوێ (یان کارگەکان لە جلساتە نوێدا) دەتوانن بە خێرایی پڕۆسەکە بدەنەوە بە بخوێندنەوەی site_brief ی تۆ، لەبری پرسیارکردنی پرسیاری ڕوونكەر.

## دەستپێکردنی Site Specification {#initiating-site-specification}

### لە کاتی پێشوەختنی Theme Builder (During Theme Builder Onboarding) {#during-theme-builder-onboarding}

مهارەتی Site Specification بە شێوەیەکی ئۆتۆماتیک لە کاتی **Theme Builder onboarding flow** دا دەستپێدێت. کارگەکان یارمەتیدەرە ڕاستەوخۆ پرسیار دەکەن و site_brief ی تۆ دروست دەکەنەوە.

### دەستپێکردنی دەستکرد (Manual Initiation) {#manual-initiation}

تามารถ دەتوانی هەر کاتێک سەرەتاکردنی پێداویستی سایت بکەیت:

```
"بۆ من دیاریی تایبەتمەندی سایتەکە بکەین"
```

یان

```
"یارمەتیم بدە یەکەم بیرۆکەی سایتێک دروست بکەم"
```

## پڕۆسەی دیاریکردنی سایت (Site Specification Process) {#the-site-specification-process}

### هەنگاوی ١: مەبەستی سایتەکە (Site Purpose) {#step-1-site-purpose}

ئەژمارەکە پرسیار دەکات:

```
چی مەبەستی سەرەکی سایتەکەتە؟
- دوکان بۆ فرۆشتنی شتەکان (E-commerce store)
- ڕۆژنامەی کارەکان یان سایتێکی ناوەڕۆک (Blog or content site)
- پۆرتفۆلیۆ یان نیشاندانی کارەکان (Portfolio or showcase)
- بەرنامەیەکی SaaS (SaaS application)
- کۆمەڵگە یان فۆ럼 (Community or forum)
- دیکە: [بشاردە]
```

دەتوانی یەکێک لەو پۆلێیە هەڵبژێریت یان مەبەستت بە وشە بنووسیت.

### هەنگاوی ٢: خەڵکی ئامانجدار (Target Audience) {#step-2-target-audience}

```
خەڵکی سەرەکی تۆ چییە؟
- بەکارهێنەران / خەڵکی گشتی
- پیشەوانانی بازرگانی
- پڕۆگرامەران / بەکارهێنەرانی تەکنیکی
- خوێندکاران / پەروەردەکاران
- دیکە: [بشاردە]

ئامانج و پێداویستی سەرەکیان چییە؟
```

### هەنگاوی ٣: ناسنامەی براند (Brand Identity) {#step-3-brand-identity}

```
ڕەنگەکانی براندت چیین؟
- ڕەنگی سەرەکی: [یەکگرتوو ڕەنگ یان کۆدی هێکس]
- ڕەنگی دووەم: [یەکگرتوو ڕەنگ یان کۆدی هێکس]
- ڕەنگی پاشەکەوت (Accent color): [ئەویش ئەگەرییە]

چۆن دەتوانیت توندوتیژیی براندت وەسف بکەیت؟
- پیشەیی / کارگێڕیی
- ئایینی / هونەری
- وشکە / ئاسایی
- سادە / مۆدێرن
- گەرم / دوستانە
```

### هەنگاوی ٤: ئامانجەکانی بازرگانی (Business Goals) {#step-4-business-goals}

```
سەرکەوتن بۆ سایتەکەت چۆن دەردەکەوێت؟
- بەدەستهێنانی سەرچاوەکان (Leads generate)
- فرۆشتنی بەرهەمەکان
- دروستکردنی کۆمەڵگە
- بەشی زانین بەشداریکردن
- دامەزراندنی ڕێگای ناسراو (Authority establish)
- دیکە: [بشاردە]

نشانگر سەرەکی تۆ چییە؟
- دراوە (Revenue)
- بەکارهێنانی بەکارهێنەران (User engagement)
- گۆڕینی ناوەڕۆک (Content reach)
- گۆڕینەوەکان (Conversions)
- دیکە
```

### هەنگاوی ٥: پێکهاتەی ناوەڕۆک (Content Structure) {#step-5-content-structure}

```
ناوەرۆکت چۆن ڕێکخراوە؟
- ڕاستەوخۆ (هەموو ناوەڕۆک لە یەک ئاستدا بێت)
- پاشەکەوت (پۆل و زیرەکەکانی پاشەکەوت)
- زمانیی (بە شێوازی ڕۆژنامەی کارەکان)
- پێکەوە بە باشمارکردنی بەرهەم (Catalog)
- دیکە: [بشاردە]

چاوە ناوەڕۆکەکانت چیین؟
- بابەتەکانی بلاگ (Blog posts)
- لاپەڕەکانی بەرهەم (Product pages)
- لێکۆڵینەوەی حاڵەتەکان (Case studies)
- ڕێنمایی و پڕۆسەکان (Documentation)
- ڤیدیۆکان
- دیکە
```

## بیرگە (Memory) ی `site_brief` ی تۆ {#your-sitebrief-memory}

پاش تەواوکردنی دیاریکردنی سایت، زانیارییەکانی تۆ بە **site\_brief** لە یادەوە دەبێت لە یادەوەکانی ئەژمارەکەدا. ئەمە تۆماری ڕێکخراوەیە کە پێکهاتەی زیرەکی هەیە و لەخۆدەگرێت:

## دۆخی بینین و نوێکردنەوەی site_brief یەکەت {#viewing-and-updating-your-sitebrief}

### بینینی site_brief یەکەت {#view-your-sitebrief}

پرس بە ئەژمارەکە بکە:

```
"Show me my site brief"
```

یان

```
"What do you know about my site?"
```

ئەژمارەکە تایبەتمەندیی تۆی 저장کراوەکە دەردەخات.

### نوێکردنەوەی site_brief یەکەت {#update-your-sitebrief}

ئەگەر سایتت بگۆڕێت، دەتوانی نوێکاری بکەیت:

```
"Update my site brief: we're now targeting B2B customers"
```

یان

```
"Refresh my site specification"
```

ئەم کارە مهارەتی Site Specification بە وردی دەکاتەوە لەگەڵ زانیارییە ئێستای تۆ وەک بنەمای دەستپێکدا.

## چۆن ئەژمارەکان site_brief بەکاردێن {#how-agents-use-sitebrief}

### پێشنیارەکانی دیزاین {#design-recommendations}

کاتێک داواکاری گۆڕانکاری دیزاینی دەکەیت، ئەژمارەکان ئاماژە بە site_brief یەکەت دەدەن:

```
تۆ: "Redesign my homepage"
ئەژمار: "بنیات Site brief ی تۆ، تۆ نێوان ژنانی ٢٥-٤٥ ساڵدا مەبەستیت و توندی گەرم و کارگرییە. دیزاینێک دروست دەکەم کە سەرنجی بە زیوخۆ و بەهایە پاشاوانە دەدات."
```

### پێشنیاری تایبەتمەندییەکان (Feature Suggestions) {#feature-suggestions}

ئەژمارەکان تایبەتمەندییەکان پێشنیار دەکەن کە لەگەڵ ئامانجەکانتدا یەک دەبن:

```
تۆ: "What features should I add?"
ئەژمار: "بۆ ماجی زیوخۆیی و پاشاوانەی کڕیار، من پێشنیار دەکەم: بەرنامەی فەرمی (loyalty program)، ڕەوتەکانی کڕیار، نوێچەوارەوەی ئیمیل، و پێشنیاری بەرهەم."
```

### پێکهاتەی ناوەڕۆک (Content Structure) {#content-structure}

ئەژمارەکان ڕێکخستنی ناوەڕۆک لەسەر بنەمای پێکهاتەکەت پێشنیار دەکەن:

شما: «چگونه باید وبلاگم را سازماندهی کنم؟»
Agent: «از آنجایی که شما از یک کاتالوگ محصول با وبلاگ استفاده می‌کنید، پیشنهاد می‌کنم پست‌ها را بر اساس دسته‌بندی محصولات سازماندهی کنید و 'راهنمای استایل'هایی بسازید که چندین محصول را در کنار هم به نمایش بگذارند.»

## بهترین روش‌ها (Best Practices) {#best-practices}

### مشخص باشید {#be-specific}

به جای گفتن «مخاطب عمومی»، مخاطبان واقعی خود را توصیف کنید:

- ✓ «زنان ۲۵ تا ۴۵ ساله، علاقه‌مند به مد پایدار»
- ✗ «همه»

### به‌روزرسانی منظم انجام دهید {#update-regularly}

همانطور که سایت شما تکامل پیدا می‌کند، `site_brief` خود را به‌روز نگه دارید:

- وقتی مخاطبان جدیدی را هدف قرار می‌دهید
- وقتی خطوط تولید جدید محصول اضافه می‌کنید
- وقتی هویت برندتان تغییر می‌کند
- وقتی اهداف کسب‌وکار شما عوض می‌شود

### از اصطلاحات یکسان استفاده کنید {#use-consistent-terminology}

همان کلمات را در تمام جلسات به کار ببرید:

- ✓ همیشه بگویید «زیورآلات پایدار» (نه «زیورآلات سازگار با محیط زیست» و «محصولات سبز»)
- ✓ مخاطبان خود را به طور مداوم به یک شکل ارجاع دهید

### زمینه را در نظر بگیرید {#include-context}

زمینه را ارائه دهید که به عوامل کمک می‌کند عامل‌ها تصمیمات شما را درک کنند:

- «ما متخصصانی را هدف قرار می‌دهیم که کیفیت را بر قیمت اولویت می‌دهند»
- «مخاطبان ما با تکنولوژی آشنا هستند و انتظار طراحی مدرن دارند»
- «ما یک استارتاپ با بودجه محدود هستیم، بنابراین به راه‌حل‌های مقرون‌به‌صرفه نیاز داریم»

## رابطه با شروع کار Theme Builder (Theme Builder Onboarding) {#relationship-to-theme-builder-onboarding}

مهارت Site Specification در **جریان شروع کار Theme Builder** ادغام شده است. وقتی فرآیند شروع کار را کامل می‌کنید، `site_brief` شما به طور خودکار با اطلاعاتی که ارائه داده‌اید ایجاد می‌شود.

شما همچنین می‌توانید Site Specification را به صورت مستقل اجرا کنید اگر بخواهید:

- مشخصات خود را پس از تنظیم اولیه اصلاح کنید
- خلاصه سایت خود را با تکامل سایت به‌روزرسانی کنید
- یک مشخصات دقیق قبل از شروع Theme Builder بسازید

## عیب‌یابی (Troubleshooting) {#troubleshooting}

**`site_brief` من استفاده نمی‌شود**
- تأیید کنید که عامل به حافظه دسترسی دارد
- از عامل بخواهید «خلاصه سایت من را بازیابی کند» (recall my site brief)
- بررسی کنید که آیا حافظه در تنظیمات شما فعال است یا خیر

**من د پشتر دەستپێکردنەوە لەگەڵ یەکەم site_brief نوێیەکم دەکەم**
- بە ئاماژە بە ئەژمارەکە بپرسە: "Site brief-ەکەم پاک بکە و نوێی دەستپێبکە"
- پاشان Site Specification دووبارە چارەسەر بکە

**ئەژمارەکە پێشنیاری ئەوە دەکات کە لەگەڵ site_brief ی مندا یەک ناکات**
- بە ئەژمارەکە بپرسە: "Site brief-ەکەم بیرەوە ببینە"
- Site brief-ەکەت به‌روزرساکەر بکە ئەگەر پێویست بێت
- زانیاری زیاتر لە داواکارییەکانت بدە

## هەنگاوەکان بۆ پاشان {#next-steps}

پاش دیاریکردنی تایبەتمەندیی سایتەکەت:

1. **بۆ بەکارهێنانی Theme Builder**: یەکەم قالبێکی تایبەت بەسەر site_brief ی خۆت دروست بکە
2. **دیزاینی زیاتر بکە**: لە توانای Design System Aesthetics بۆ کارکردن لەسەر دیزاینی ورد و جزیل بێت
3. **گرنگی ناوەڕۆکەکە ڕێک بخەرەوە**: داواکارییەکانی ئەژمارکان بۆ پێشنیاری ساختارے ناوەڕۆک بدات
4. **تایبەتمەندییەکان دروست بکە**: داواکاریی تایبەتمەندییەکان بکە کە لەگەڵ ئامانجەکانی کارگێشتەکەت یەک دەبێت
