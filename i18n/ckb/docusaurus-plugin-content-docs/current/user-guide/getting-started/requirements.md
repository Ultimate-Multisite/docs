# پێداویستییەکان بۆ Ultimate Multisite

پێش ئەوەی کە Ultimate Multisite بخوێنیت، یەکەم شتێک کە دەتوانیت لەسەر بیر بکەیت، ئەو پێداویستنەی کە بۆ چارەسەرکردنی ئەم پڵۆگەکە پێویستن. بەبێ کێشە، تەنها چەند پێداویستیی بنەڕەتی هەیە بۆ چارەسەرکردنی Ultimate Multisite: **ناوەکانی دامێنی** (domain name)، **پایبەرووێکی هاستکردنی باش** (hosting provider) و **ناردنی WordPress Multisite**. ئەمە هەموو ئەو شتانەن کە پێویستت دەبێت!

## ناوی دامێنی (Domain Name)

پێشتر، باسی ناوی دامێنی بکەین.

کاتێک دەبینیت ناوی دامێنیکردن هەڵدەبگریت، بیر لەوە بکەرەوە کە تخصص تایبەتت چییە و چی دەبگەڕێت بە Ultimate Multisite: ئایا دەتەوێت ایجنسی دروست بکەیت؟ ئایا ئەو شوێنە جۆری دوکانە بفرۆیت؟ یان ڕەنگە هاستکردنی وێب گەردوون بۆ WordPress بکەیت؟ هەموو شتێک لەگەڵ Ultimate Multisite دەکرێت.

پڵۆگەکە خزمەتگوزاریی سابسکریپشن، کۆکردنەوەی پارە، فۆرمەکانی چوونە ناو (checkout forms)، وۆچەرەکانی کەمکردنەوە (discount vouchers) و پەیوەندیگەلانی بەکارهێنەران پێشکەش دەکات. کارێکی زۆر لە دامەزراندن، ڕێکخستن و پاراستنی WordPress Multisite دەکرێت بە یارمەتی Ultimate Multisite.

بۆ هەر تخصصێک کە هەڵدەبگریت، دەتوانیت بەم پڵۆگەکە بەڕێوەبەر بکەیت تا ئەو کاتەی کە شوێنەکان وەک خزمەتگوزاریی کارپێکردن (as a service) بفرۆیت! دڵنیابە ناوی دامێنێک هەڵبگرێت لەگەڵ پلانی کارەکەت.

## پێشکەشکارێکی هاستکردن (Hosting Provider)

ئەنجام لەوە دەبینیت کە ناوی دامێنێکی سەرنجڕاکێش هەیە کە تخصص و کارپێچوونی تۆ دیاری دەکات، پێویستە کۆمپانیای باشی هاستکردنی وێب هەڵبگریت بۆ پاراستنی تۆڕەکەت لەگەڵ Ultimate Multisite. چەند بژاردنێکی باش لە بازاڕدا هەن. هەندێک لە پێشکەشکارانی هاستکردن بە باشی لەگەڵ Ultimate Multisite کار دەکات، کە ئەمە دەکرێت هۆکاری دیاریکەر بێت بەپێی ئەوەی پلان دەکەیت دروست بکەیت.

Ultimate Multisite دەبێت لە هەر پڕۆفایدرێکی هاستکرد کار بکات. تەنها جیاوازی ئەوەیە کە هەندێک کۆمپانیا وەک _WPMU DEV Hosting_، _WP Engine_، _Closte_، _Cloudways_، _Gridpane_ و _Runcloud_ یان تایبەتمەندی زیاتر لە مێژووی پۆلێنکردنی دۆمن (domain mapping) دەبن. بۆ دیکەی هاستکرد، پێویستە دۆمێنە تایبەت بە دەستی ڕێک بخوازیت. هەروەها باشە یەکێک لە پڕۆفایدرەکانی هاستکرد بدۆزیت کە پشتگیریکردنی AutoSSL بکات. _Closte_، _Runcloud_ و _cPanel_ پشتگیریکردن دەکەن.

لە کاتی ئەوەی بچینە سەر تایبەتمەندیی سروری (server specifications)، هیچ فۆرمولای تایبەت نییە چونکە بەپێی تایبەتمەندی و کارکردن کە دەوێت و گەشەی تۆڕەکەت دەگۆڕێت. هاستکردنی بەشایی دەتوانرێت سەرەتای باش بێت، بەڵام لە کاتی ئەوەی تۆڕەکەت دەست بە گەشەکردن بکات، پێویستە VPS تایبەت بدۆزیتەوە.

## WordPress Multisite

ئێستا، بیستەین چەند شتێک لەسەر دامەزراندنی WordPress Multisite قسە بکەین.

WordPress Multisite ڕێنوێنی دەکات کە دەتوانیت تۆڕێکی وێبسایەت بە یەک جۆر دامەزراندن هەبێت. ئەمە تایبەتمەندییەکی ناوخۆییە، بەڵام بە شێوەی پیش‌دەکەوتنی سەرەتایی چالاک نییە. پێویستە ئەم تایبەتمەندییە لەناو دامەزراندنی WordPress و پۆلێنکردنی وێبسایتی خۆتدا دەستکاری بکەیت (enable) بکەیت.

هاتوو ئەوەی WordPress بە Multisite چالاک بکەیت، کاتی دامەزراندنی Ultimate Multisiteیە. بۆ دامەزراندنی دروست ی Ultimate Multisite، پێویستە هەبێت:

  * WordPress: v5.3+ (پێشنیار: کۆنترین و پایدارترین بڵاوکردەوە)

  * PHP: 7.4.x (پشتیوانی بۆ > 8.0 لە نزیک دەبێت)

  * MySQL: v5+ (پێشنیار: 5.6، لە حاڵەتی ئەوەی کە 8.0 لە پڕۆفایدرەکەت بەدەست نییە)

ئەمە تەنها پێداویستییەکانی نەفزار Ultimate Multisiteن.

بۆ ئەوەی لە یاد هەبێت، Ultimate Multisite دەتوانرێت لەگەڵ PHP 8.0 کار بکات، بەڵام ئێمە پێشترمان دەکەین بە PHP 7.4.x بکاربهێنیت.

هەروەها، ئێمە پێشنیار دەکەین لەسەر **دۆمنە سەرەکییەکان** دامەزرێنن، نەک لەسەر subdomain یان subdirectory. ڕەنگە لەسەر subdomain/subdirectory کار بکات، بەڵام دەکرێت هەندێک هەڵەی دروست بکات.

بۆ ڕێنمایی وردی لەسەر چۆنیەتی دامەزراندنی Ultimate Multisite، **لێرە clique**.
