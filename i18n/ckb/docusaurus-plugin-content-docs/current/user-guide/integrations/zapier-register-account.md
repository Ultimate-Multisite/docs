# رویداد: ثبت نام کردن از طریق Zapier

لە بابەتەکەدا کە [Integrating Ultimate Multisite with Zapier](zapier.md) دا، باس لەوە گێڕاوە چۆن دەتوانیت بەکاری بهێنیت Zapier بۆ ئەنجامدانی کارە جیاوازەکانی ناو Ultimate Multisite پێکەوە بەپێی چالاکییەکان (triggers) و ڕووداوەکان (events). لەم بابەتدا، دەبینین چۆن دەتوانیت ئەپڵیکەیشنی سێیەم (3rd party applications) یەکدەگرین. ئێمە Google Sheets وەک سەرچاوەی داتایەکەمان بەکاری دەهێنین بۆ ئەوەی ئەو زانیاریانە بۆ Ultimate Multisite بفرستێت بۆ تۆمارکردنی یەکجار.

پێش ئەوەی Zapier لەگەڵ ببینیت، بگە به **Ultimate Multisite > Settings > API & Webhooks** و دڵنیا بەوەی کە API چالاککراوە (enabled). کاتێک Zapier داواکەت کرد بۆ زانیارییەکانی یەکجار، کلیکی API Key و API Secret لەو شیلانەی خستنە دەستی ئەنجام بدە.

![API and Webhooks settings with API Key, API Secret, and Enable API options](/img/admin/settings-api-webhooks.png)

پێشتر، پێویستە یەک **Google Sheet** لەناو Google Driveی خۆت دروست بکەیت. دڵنیابە کە هەموو ستوونەکان بە شێوەیەکی دروست دیاری بکەیت بۆ ئەوەی دواتر بە ئاسانی بتوانین داتاکان وەک یەک دەستکاری بکەین (map).

پاش دروستکردنی Google Sheet، دەتوانیت بچیتە ژوورەوەی کاربری Zapier و دەستپێبکەیت دروستکردنی zap.

لە خاڵی گەڕان بۆ **"App event"** هەڵبژێرە **"Google Sheets"**.

پاشان لە خاڵی "**Event** " هەڵبژێرە **"New spreadsheet row"** و **"Continue"** بکە.

پێشتر، هەنگاوەکە پێشکەش دەکات کە دەوێت یەک **Google Account** هەڵبژێریت کە **Google Sheet** لەسەردا هەڵگرتووە. بۆیە تەنها دڵنیابە ئەو یەکجارەی دروستکراوە بە شێوەیەکی دروست دیاری کراوە.

لە خاڵی **"Set up trigger** "، پێویستە Google Spreadsheet و worksheet (شێوازی کار) کە دەتوانیت لەوێ بەکاری بهێنیت هەڵبژێریت و دیاری بکەیت کە داتاکان لە کوێ دەچێت. تەنها بەردەوام بیت و ئەوانە پڕ بکەرەوە و **"Continue"** بکە.

پاشان، بۆ ئەوەی دڵنیابیت کە Google Sheet-ەکەت بە شێوەیەکی دروست پەیوەندیدارە، پێویستە **"test your trigger** " (تستکردنی چالاکییەکە) ئەنجام بدە.

ئەگەر تستی سەرکەوتوو بێت، دەتوانیت ئەنجامییەک ببینیت کە هەندێک بەها لە Google Sheets یەکەکانی نیشان دەکات. بۆ بەردەوامبوون کلیکی **"Continue"** بکە.

پایین مرحله‌ای که باید انجام بدید، تنظیم اکشن دومی است که حساب کاربری را در Ultimate Multisite ایجاد یا ثبت می‌کند. در فیلد جستجو، روی "**Ultimate Multisite(2.0.2)** " کلیک کنید.

در قسمت "**Event** "، گزینه "**Register an Account in Ultimate Multisite** " را انتخاب کرده و سپس دکمه "**Continue** " را بزنید.

زیر بخش "**Set up an action**"، فیلدهای مختلفی برای داده‌های مشتری، عضویت‌ها، محصولات و غیره مشاهده خواهید کرد. می‌توانید مقادیر مورد نظر خود را در گوگل شیت نگاشت (map) کرده و آن‌ها را به فیلد مناسب اختصاص دهید تا همانطور که در تصویر زیر نشان داده شده است پر شوند.

پس از انجام نگاشت مقادیر، می‌توانید اکشن را تست کنید.
