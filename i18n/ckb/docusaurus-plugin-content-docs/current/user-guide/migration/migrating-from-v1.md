# گواستن لە V1ەوە بۆ 2.x {#migrating-from-v1}

## Ultimate Multisite لە خێزانی سەرەکی 1.xی خۆی گۆڕاو بۆ خێزانی 2.x. {#ultimate-multisite-has-switched-from-its-original-1x-family-of-releases-to-the-2x-family-of-releases}

Ultimate Multisite وەرگرتنی 2.0 و بەرزتر، نووسینی تەواوی کۆدەکە دەگۆڕێتەوە، واتە زۆر باش نییە شتێک هاوبەش بێت لە نێوان ئەو وەرگرتنە کۆنە و ئەو نوێیە. بۆیە، کاتێک لە 1.x بەرەو 2.x دەکەویت، پێویستە داتاکان بە فۆڕماتی بگۆڕیت کە وەرگرتنی نوێکان بتوانن تێبگەین.

بە خۆشییەوە، Ultimate Multisite 2.0+ **هەڵگرێکی (migrator) لەگەڵ دەهێنێت** کە لە ناوەندی کارکردنەکەدا دروستکراوە و توانای دۆزین داتاکانی ئەو وەرگرتنەی کۆنە و گۆڕینیان بۆ فۆرماتی نوێی ئەنجام دەدات. ئەم گواستنە ڕوودەدات لە ماوەی **Setup Wizard**ی 2.0+دا.

ئەم بەڵاوتە باس دەکات کە هەڵگرەکە چۆن کار دەکات، چی بکەیت لە حاڵەتی شکست، و چۆن کێشەکانی دیکە کە دەکرێت لەم پڕۆسەیەدا ڕووبدات چارەسەر بکەیت.

_**گرنگ: پێش ئەوەی دەست بە گواستن لە 1.x بەرەو 2.0 بکەیت، تکایە دڵنیابە کە بیکەیت Backup (کۆپی)ی داتابێسەکەی سایتت.**_

## هەنگاوە یەکەمەکان {#first-steps}

هەنگاوی یەکەم بریتییە لە داواکردنی فایلەکە .zip و دامەزراندنی وەرگرتنی 2.0 لە داشبۆردی بەڕێوەبەری تۆڕەکەت (network admin dashboard).

پاش ئەوەی [وەرگرتنی 2.0 دابنێیت و چالاک بکەیت](../getting-started/installing-ultimate-multisite.md)، سیستەم بە شێوەیەکی ئۆتۆماتیک دەدۆزێتەوە کە Multisite ی تۆ لەسەر وەرگرتنی کۆنەکە کار دەکات، و ئەم پەیامە لە سەرەوەی لاپەڕەی وەرگرتنەکەدا دەبینیت.

_**تێبینی: ئەگەر Ultimate Multisite 1.x لەسەر Multisite ی خۆت دامەزراوە، دەتوانیت هەڵبژاردن بکەیت کە وەرگرتنی نوێیەکەی بدۆزیتەوە بە جای ئەو وەرگرتنەی تازە دابنێویت. تکایە، پێش بگەڕێ بۆ **Replace current with uploaded**._

<!-- Screenshot unavailable: Plugin page showing option to replace v1 with uploaded v2 version -->

پەڕەی دووەم به شما یاد دەدات چ додаگرە کۆنەکەت لەگەڵ وەرگرتنی (version) 1.x چی جێگیرن. ئەو پەڕەیە ڕێنمایی دەدات کە ئایا وەرگرتنی ئێستات لەگەڵ وەرگرتنی 2.0 یارە، یان ئەگەر پێویستە دوای گۆڕینی (migration) وەرگرتنی نوساینەت بەرز بکەیتەوە بۆ ئەو додаگرە.

<!-- Screenshot unavailable: Message on the top of the plugins page about updating to v2.0, with list of add-ons that need updating -->

کاتێک ئامادە بوویت بۆ پێشوەختن، دەتوانن ئەو دۆمێنتییە بێت کە دەڵێت **بیزبیت بۆ دامەزراندنی وەرگرتنی (Installer) بۆ تەواوکردنی بەرزکردنەوەکان**.

<!-- Screenshot unavailable: Button saying Visit the Installer to finish the upgrade -->

پاش ئەوە، دەچێتە ئەو پەڕەی دامەزراندنی (installation wizard) کە چەند پەیامێکی بەخێوات دەدات. تەنها پێویستە **دەبگرە دەست** بۆ گواستنەوە بۆ پەڕەی دوایین.

<!-- Screenshot unavailable: Setup Wizard welcoming page with Get Started button -->

پاش ئەوەی **دەبگرە دەست**، دەچێتە ئەو پەڕەیە کە ڕێنمایی پێدات بە Pre-install Checks_._ ئەمە زانیاریی سیستەم و دامەزراندنی WordPress یەکەت دەردەخات و پێتان دەڵێت ئایا لەگەڵ **پێویستی Ultimate Multisite** یارە.

<!-- Screenshot unavailable: Pre-install Checks page showing system requirements confirmation -->

پاش ئەمە، هەنگاوەکە ئەوەیە کە کلێپێکی لایسنسی (license key) ی Ultimate Multisite بنووسیت و پلاتینی (plugin) چالاک بکەنەوە. ئەمە دڵنیایی دەدات کە هەموو تایبەتمەندییەکان، لەوانەش додаگرەکان، لەسەر سایتەکەت بەکاربهێنرێت.

<!-- Screenshot unavailable: License activation page with license key input and Agree and Activate button -->

پاش نووسینی کلێپی خۆت، **دۆست و چالاک بکە** (Agree & Activate) بڵێ.

پاش چالاککردنی لایسنسی، دەتوانیت دامەزراندنی ڕاستەقینەی دەست پێبکەیت بە کلیککردن لەسەر **Install** لە پەڕەی دووەم. ئەمە بە شێوەیەکی ئۆتۆماتیکی فایل و داتابەیسە پێویستەکانی دروست دەکات بۆ ئەوەی وەرگرتنی 2.0 کار بکات.

## ئێستا، میگریشن (Migration) {#now-the-migration}

میکرۆپریتور (migrator) تایبەتمەندی ئاسایشی هەیە کە دڵنیایی دەدات لە هەموو سیستەمی مۆڵت سایتییەکەت بۆ Ultimate Multisite، کە هەموو داتاکانی ئەوە بتوانن میگڕێنەوە بەبێ هیچ کێشەیەک. بۆ دەستپێکردنی پڕۆسەکە، बटانی **Run Check** (دۆخەکان چاک بکە) دەربکە.

لە دوای ئەنجامدانی چاڵاسان، دوو ڕێگەت هەیە: ئەنجامەکە دەتوانێت یان بە هەڵەیەک بێت یان بەبێ هەڵە بێت.

### لەگەڵ هەڵە (With Error) {#with-error}

ئەگەر پەیامی هەڵەیەک وەربگریت، پێویستە پەیوەندی ببەیتە تیمی پشتیوانی ئێمە بۆ ئەوەی بتوانن یارمەتیت بدەن لە چارەسەرکردنی ئەو هەڵەیە. دڵنیا بە کە **لۆگی هەڵەکە (error log) دەدات** کاتێک تیکێتی دروست دەکەیت. دەتوانی لۆگەکە داونبی بکەیت یان دەتوانی لینکەکەی بگەڕێیتەوە کە دەڵێت "پەیوەندی بە تیمی پشتیوانی ئێمە". ئەم کارە وێبچەکەی یارمەتیت دەدات لە ڕاستیدا، بە شێوەیەکی ئۆتۆماتیک فیلدەکانی پێویست بۆت پڕ دەکاتەوە کە لۆگەکانی هەڵەکە لە خوارەوەدا دەبینیت.

_**لەبەر ئەوەی سیستەم هەڵەیەکی دۆزیوەتەوە، ناتوانیت بە ئاسانەوە بۆ وەرگرتنی وەرگرتنی 2.0 پێش بەرەو ببیت. دوای ئەوە، دەتوانیت بگەڕویتەوە بۆ 1.x بۆ بەردەوامبوون لە کارکردنی تۆڕەکەت تا ئەو هەڵەیە چارەسەر بکرێت.**_

### بەبێ هەڵە (Without Error) {#without-error}

ئەگەر سیستەم هیچ هەڵەیەک نەدۆزێت، پەیامی سەرکەوتوو دەبینیت و **Migrate** دایە لە خوارەوە کە ڕێگە بدات بە پێشبردنی میگریشنەکە. لەم لاپەڕەیەدا، یادمان دەکرێت بیکەیت کۆنترۆڵێکی (backup) داتابیسەکەت دروست بکەیت پێش ئەوەی بەرەو پێشبردنی ببیت، کە ئێمە بەهێزترین ڕێنمایی دەکەین. ئەگەر لەبەر ئەوەی یەکێک لە کۆنترۆڵەکانت هەیە، **Migrate** دەست بکە.

_**ئەمە هەموو ئەو کارانە دەوێت!**_

دەتوانی بەردەوام بیت لە دانانی ویزارد بۆ نوێکردنەوەی لوگوتۆ و شتکارەکانی دیکە لەسەر تۆڕەکەت یان دەستپێبکە بە گەڕان لە منو的一 Ultimate Multisite 2.0 و پێکهاتەی نوێیەکەی. بە ئاسان، خۆت لێبکەوێت و خۆشی ببینەوە.
