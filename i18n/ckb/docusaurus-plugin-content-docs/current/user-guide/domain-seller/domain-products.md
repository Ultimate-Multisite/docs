# ئیست و پرایسنگی دامینەکان

دۆخی پڕۆدۆکەکانی دامین، شێوازێکە کە بەڕێوەبردنی فایەنس، TLDs (نواوەکانی ناو دۆمەین)، تایبەتمەندییە تایبەتەکانی WHOIS و هەڵبژاردنی پڕۆڤایدر دەکرێت. هەر پڕۆدۆکێکی دامین، پڕۆدۆکی ستاندارد Multisite Ultimate بووە کە تابلۆی زیاتری **Domain Settings** (سەرکردایەتیی دامین) لەخۆدەگرێت.

## دروستکردنی پڕۆدۆکێکی دامین

1. بگواشبە **Network Admin › Ultimate Multisite › Products**.
2. **Add New** (زیادکردنی نوێ) بگرە.
3. جۆری پڕۆدۆکەکە بۆ **Domain** دیاری بکە.
4. تابلۆی **Domain Settings** (لێرەدا دەبینیت) ڕێکبخە.
5. سیوەرەوە (Save).

پڕۆدۆکەکانی دامین لە لیستەکەی پڕۆدۆکە بە نشانێکی گەورە و پڕی **Domain** (دامین) دەردەکەون و دەتوانیت بە تابلۆی **Domain Products** فیلتر بکەیت.

## تابلۆی Domain settings

### Provider (پڕۆڤایدر)

ببینەوە کە چ پڕۆڤایدرێک کار دەکات بۆ تۆمارکردنی ئەم پڕۆدۆکە. بە شێوەیەکی پیش‌تراو، ئەوەی لە ڕێگەی تنظیمەکانی Domain Sellerدا دیاری کراوە (Default provider) دەبێت.

### Supported TLDs (TLDسە پشتگیریکراوەکان)

ئەم بەشە خالی لێ بێخە بۆ دروستکردنی **catch-all product** (پڕۆدۆکێکی گشتی) کە بۆ هەموو TLDیەکانی دیکە کار دەکات کە لە پڕۆدۆکی دیکەدا یەکگرتووە.

ئەگەر لیستێک لە TLDs بە دوو کاما (مثلاً `.com, .net, .org`) بنووسیت، دەتوانیت **TLD-specific product** (پڕۆدۆکێکی تایبەت بە TLD) دروست بکەیت کە تەنها بۆ ئەو کۆنێشنان کار دەکات.

**چۆنیەتی یەکگرتنی پڕۆدۆک:** کاتێک کڕیارێک دۆمایێک دەگەڕێت، ئەپڵۆگ (addon) پڕۆدۆکی وردترین یەکگرتووە هەڵدەبژێرێت. پڕۆدۆکێک کە `.com` لە لیستەکەی TLDدا هەیە پێشترە لە پڕۆدۆکی catch-all. ئەگەر هیچ پڕۆدۆکی تایبەت بە TLD نەبوو، پڕۆدۆکی گشتی (catch-all) بەکاردێت. ئەگەر هیچ پڕۆدۆکێک نەبێت، لێکردن دۆمایەکە نیشان دەدرێت.

### Markup type (جۆری مارکاپ)

سێ شێواز هەیە کە دیاری دەکەن چۆن نرخی خریداریکراو لەدا گەردونەوەی خەرجی بەپێی پاشخەرجی کارگێڕی دەکرێت:

| Mode (شێواز) | How it works (چۆنیەتی کارکردن) |
|---|---|
| **Percentage** (پەربڵاوجۆ) | پێکەنینیشانێکی ڕێژەیی لە خەرجی بەپێی کارگێڕی دەبێت. مارکاپێکی 20% لەسەر دۆمایەک کە خەرجی بەپێی کارگێڕی $10یە، $12 دەبێت. |
| **Fixed markup** (مارکاپێکی دیاریکراو) | نیشانەیەکی پارەی دیاریکراو زیاد دەکات. مارکاپێکی $5 لەسەر دۆمایەک کە خەرجی بەپێی کارگێڕی $10یە، $15 دەبێت. |
| **Fixed price** (نرخی دیاریکراو) | خەرجی بەپێی کارگێڕی بە تەواوی لەبەرچاو دەگرێت. هەمیشە ئەو بڕەی کە تۆ دەنووسیتدا پارە دەخات. |

### Introductory pricing (نرخی دەستپێکی پڕۆدۆک)

تفعيل عرض سعر تخفيض لأول سنة. قم بتعيين **سعر تعريفي** منفصل (سعر السنة الأولى) بجانب **سعر التجديد** العادي (السنة الثانية وما بعدها). يرى العميل كلا السعرين معًا أثناء الدفع ليعرف ما يمكن توقعه عند التجديد.

### خصوصية WHOIS

يتحكم هذا الإعداد في ما إذا كان يتم تقديم حماية خصوصية WHOIS للمجالات المسجلة من خلال هذا المنتج أم لا.

| الإعداد | السلوك |
|---|---|
| **معطل (Disabled)** | لا يتم تقديم أو تفعيل خصوصية WHOIS أبدًا. |
| **مُضمن دائمًا (Always Included)** | يتم تفعيل خصوصية WHOIS تلقائيًا عند التسجيل دون أي رسوم. |
| **اختيار العميل (Customer Choice)** | يظهر مربع اختيار أثناء الدفع. قم بتعيين **سعر الخصوصية** ليتم محاسبة المستخدم عليه لكل سنة، أو اتركه عند 0 لتقديمه مجانًا. |

بالنسبة لـ Namecheap، تستخدم خصوصية WHOIS WhoisGuard (مجانية دائمًا). بالنسبة لـ OpenSRS، تستخدم خدمة خصوصية OpenSRS (قد تكون لها تكلفة بالجملة).

## استيراد ومزامنة النطاقات الفرعية (TLD import and sync)

تعرض منتجات النطاقات الفرعية أسعار الجملة في الوقت الفعلي التي يتم جلبها من المزود المتصل. لكي يعمل هذا، يجب استيراد النطاقات الفرعية (TLDs).

- **المزامنة اليدوية:** الإعدادات › بائع النطاق الفرعي (Domain Seller) › مزامنة النطاقات الفرعية (حسب المزود)
- **المزامنة التلقائية:** تعمل يوميًا عبر مهمة cron مجدولة على جميع المزودين المُعدين.

بعد المزامنة، اذهب إلى علامة تبويب إعدادات النطاق الفرعي لأي منتج نطاق فرعي واستخدم مُحدد النطاقات الفرعية (TLD picker) لرؤية النطاقات الفرعية المتاحة مع أسعار الجملة الحالية الخاصة بها.

## التجديد التلقائي (Auto-renewal)

تعتمد تجديدات النطاقات الفرعية على حالة عضوية العميل:

- عندما تتجدد العضوية ويرتبط نطاق فرعي، يتم وضع تجديد النطاق الفرعي تلقائيًا في قائمة الانتظار.
- تستخدم محاولات التجديد بوابة الدفع النشطة للعميل.
- تتم إعادة محاولة التجديدات الفاشلة تلقائيًا مع تراجع أسي (exponential backoff).
- يتم إرسال إشعارات عبر البريد الإلكتروني للتجديدات الناجحة، والإخفاقات، وتواريخ الانتهاء القادمة.

معرفات قوالب البريد الإلكتروني لأحداث دورة حياة النطاق الفرعي:

| Event | Template ID |
|---|---|
| Domain registered | `domain_registered` |
| Domain renewed | `domain_renewed` |
| Renewal failed | `domain_renewal_failed` |
| Domain expiring soon | `domain_expiring_soon` |

## Admin: Registration manual a domain

باشە بۆ تۆمارکردنی دۆमेन بە شێوەیەکی دەستکرد لە بەرامبەر کڕیارێک، بەبێ ئەوەی ڕێگەی پڕۆسەی خریدارەکە بێت:

1. بگوا به **Network Admin › Ultimate Multisite › Register Domain**
2. کڕیارەکەت هەڵبژێرە و ناوی دۆمنەکە بنووسە
3. وردەکاری پەیوەندیدارەکانی تۆمارکار (ناوی، پیت، ژمارەی تەلەفۆن) پڕ بکەرەوە
4. بۆ **Register** بگەڕێ

رکوردەکەی دۆمنەکە دروست دەبێت و بە شتکۆی کڕیارەکە پەیوەست دەبێت.
