# وەرگرتنی پارە (v2)

_**تێبینی گرنگ: ئەم وتارە ئاماژە بە وەشانی 2.x ـی Ultimate Multisite دەکات.**_

Ultimate Multisite سیستەمێکی ناوخۆی ئەندامێتی و ژمێریاری هەیە. بۆ ئەوەی سیستەمی ژمێریاریمان کار بکات، باوترینی دەروازەکانی پارەدانمان تێکەڵ کردووە کە لە بازرگانی ئەلیکترۆنی بەکاردێن. دەروازەکانی پارەدانی بنەڕەتی لە Ultimate Multisite بریتین لە _Stripe_ ، _PayPal_ ، و پارەدانی دەستی. هەروەها دەتوانیت _WooCommerce_ ، _GoCardless_ و _Payfast_ بەکاربهێنیت بۆ وەرگرتنی پارە بە دامەزراندنی زیادکراوە تایبەتەکانیان.

## ڕێکخستنە بنەڕەتییەکان {#basic-settings}

دەتوانیت هەر یەکێک لەم دەروازەکانی پارەدانە لە ژێر ڕێکخستنەکانی پارەدانی Ultimate Multisite ڕێکبخەیت. دەتوانیت بیدۆزیتەوە بە چوون بۆ **مێنیوی Ultimate Multisite > ڕێکخستنەکان > پارەدانەکان.**

![پەڕەی ڕێکخستنەکانی پارەدان لە Ultimate Multisite کە پانێڵی پارەدانەکان پیشان دەدات](/img/config/payments-settings-page.png)

پێش ئەوەی دەروازەی پارەدانەکەت دابمەزرێنیت، تکایە سەیرێکی ڕێکخستنە بنەڕەتییەکانی پارەدان بکە کە دەتوانیت ڕێکیان بخەیت:

**ناچارکردنی نوێکردنەوەی خۆکار** **:** ئەمە دڵنیایی دەدات کە پارەدانەکە بە شێوەیەکی خۆکار لە کۆتایی هەر خولێکی ژمێریاری دووبارە دەبێتەوە، بە پێی ئەو دووبارەبوونەوەی ژمێریارییەی بەکارهێنەر هەڵیبژاردووە.

<!-- وێنەی شاشە بەردەست نییە: ڕێکخستنی دوگمەی Force Auto-Renew لە پەڕەی ڕێکخستنەکانی پارەدان -->

Ultimate Multisite v2.13.0 پێش پاشەکەوتکردنی ئەندامێتییەکی دووبارەبووەوە لەگەڵ چالاککردنی نوێکردنەوەی خۆکار، پشکنین دەکات ئایا دەروازەی چالاک باوەڕنامەیەکی نوێکردنەوەی دووبارە بەکارهێنراوی هەیە یان نا. باوەڕنامەی نوێکردنەوە دەتوانێت بەشدارییەکی دەروازە، ڕێککەوتنی ژمێریاری، vault token ـی پاشەکەوتکراو، یان شێوازی پارەدانی دووبارە بەکارهێنراوی هاوشێوە بێت. ئەگەر دەروازەکە ڕاپۆرت بکات کە هیچ باوەڕنامەیەکی بەکارهێنراوی گونجاو بوونی نییە، Ultimate Multisite ئەندامێتییەکە پاشەکەوت دەکات بەڵام نوێکردنەوەی خۆکار ناچالاک دەکات و دۆخی نەبوونی باوەڕنامەکە تۆمار دەکات، بۆ ئەوەی بەڕێوەبەرێک یان ڕەوتی پشتگیری بتوانێت داوا لە کڕیار بکات پێش بەرواری نوێکردنەوە دووبارە ڕێگە بە پارەدان بدات.

ئەمە ڕێگری دەکات لەوەی ئەندامێتییەک وا دەرکەوێت کە خۆکارانە نوێ دەبێتەوە کاتێک دەروازەکە تەنها دەتوانێت پارەدانەکانی جارێکی کۆبکاتەوە. زیادکراوەکانی دەروازە دەبێت دڵنیا بکەنەوە کە checkout ـە دووبارەبووەوەکان باوەڕنامەیەکی دووبارە بەکارهێنراو پاشەکەوت دەکەن، بە تایبەتی کاتێک دەروازەکە پشتگیری هەردوو دۆخی گرتنی پارەی جارێک و پارەدانی vaulted/subscription دەکات.

**ڕێگەدان بە تاقیکردنەوەکان بەبێ** **شێوازی پارەدان:** کاتێک ئەم هەڵبژاردنە چالاک بێت، کڕیارەکەت پێویست ناکات هیچ زانیارییەکی دارایی لە کاتی پرۆسەی تۆمارکردن زیاد بکات. ئەمە تەنها کاتێک پێویست دەبێت کە ماوەی تاقیکردنەوەکە کۆتایی دێت.

<!-- وێنەی شاشە بەردەست نییە: دوگمەی Allow Trials Without Payment Method لە پەڕەی ڕێکخستنەکانی پارەدان -->

**ناردنی فاکتور لەسەر پشتڕاستکردنەوەی پارەدان:** ئەمە هەڵبژاردنێکت پێدەدات کە دوای پارەدان فاکتور بنێریت یان نا. ئاگاداربە کە بەکارهێنەران دەستیان دەگات بە مێژووی پارەدانەکانیان لە ژێر Dashboard ـی ماڵپەڕە لاوەکییەکەیان. ئەم هەڵبژاردنە بۆ دەروازەی دەستی جێبەجێ نابێت.

<!-- وێنەی شاشە بەردەست نییە: دوگمەی Send Invoice on Payment Confirmation لە پەڕەی ڕێکخستنەکانی پارەدان -->

**شێوازی ژمارەدانانی فاکتور:** لێرەدا، دەتوانیت یان کۆدی ئاماژەی پارەدان یان شێوازی ژمارەی زنجیرەیی هەڵبژێریت. ئەگەر هەڵبژێریت کۆدی ئاماژەی پارەدان بۆ فاکتورەکانت بەکاربهێنیت، پێویست ناکات هیچ شتێک ڕێکبخەیت. ئەگەر هەڵبژێریت شێوازی ژمارەی زنجیرەیی بەکاربهێنیت، پێویستە **ژمارەی فاکتوری داهاتوو** ڕێکبخەیت (ئەم ژمارەیە وەک ژمارەی فاکتور بۆ فاکتوری داهاتووی دروستکراو لەسەر سیستەم بەکاردەهێنرێت. هەر جارێک فاکتورێکی نوێ دروست دەکرێت، بە یەک زیاد دەکرێت. دەتوانیت بیگۆڕیت و پاشەکەوتی بکەیت بۆ دووبارە ڕێکخستنەوەی ژمارەی زنجیرەیی فاکتور بۆ بەهایەکی دیاریکراو) و **پێشگری ژمارەی فاکتور.**

<!-- وێنەی شاشە بەردەست نییە: لیستی هەڵبژاردنی شێوازی ژمارەدانانی فاکتور لەگەڵ هەڵبژاردنەکانی Payment Reference Code و Sequential Number -->

<!-- وێنەی شاشە بەردەست نییە: خانەکانی ژمارەی فاکتوری داهاتوو و پێشگری ژمارەی فاکتور کە کاتێک Sequential Number هەڵبژێردراوە پیشان دەدرێن -->

## دەروازەکان لە کوێ بدۆزیتەوە: {#where-to-find-the-gateways}

دەتوانیت دەروازەکانی پارەدان لە هەمان پەڕە دابمەزرێنیت ( **Ultimate Multisite > ڕێکخستنەکان > پارەدانەکان**). ڕاست لە ژێر **دەروازە چالاکەکانی پارەدان** ، دەتوانیت ببینیت: _Stripe_ ، _Stripe_ _Checkout_ ، _PayPal_ و _دەستی_.

![بەشی دەروازە چالاکەکانی پارەدان کە Stripe، Stripe Checkout، PayPal و دەستی پیشان دەدات](/img/config/payments-active-gateways.png)

ئێمە وتارێکی تایبەتمان بۆ هەر دەروازەیەکی پارەدان هەیە کە ڕێنماییت دەکات لە ڕێگەی هەنگاوەکانی دامەزراندنی، کە دەتوانیت لە بەستەرەکانی خوارەوە بیاندۆزیتەوە.

دەتوانیت وردەکارییەکانی پارەدان ببینیت و دەستکاری بکەیت:

![ڕووکاری دەستکاریکردنی پارەدان](/img/admin/payment-edit.png)

ئەمە بینینێکی تەواوی پەڕەی دەستکاریکردنی پارەدانە:

![ڕووکاری تەواوی دەستکاریکردنی پارەدان](/img/admin/payment-edit-full.png)

هەروەها ئەمەش بینینێکی تەواوی ڕێکخستنەکانی دەروازەکانی پارەدانە:

![پەڕەی تەواوی ڕێکخستنەکانی دەروازەکانی پارەدان](/img/config/settings-payments-gateways-full.png)

ئێستا، ئەگەر دەتەوێت _WooCommerce_ ، _GoCardless_ یان _Payfast_ وەک دەروازەی پارەدانت بەکاربهێنیت، پێویستە **زیادکراوەکانیان دابمەزرێنیت و ڕێکیان بخەیت**.

### چۆنیەتی دامەزراندنی زیادکراوەی WooCommerce: {#how-to-install-the-woocommerce-add-on}

ئێمە تێدەگەین کە _Stripe_ و _PayPal_ لە هەندێک وڵاتدا بەردەست نین، ئەمەش بەکارهێنەرانی Ultimate Multisite سنووردار دەکات یان ڕێگرییان لێدەکات لە بەکارهێنانی کاریگەری plugin ـەکەمان. بۆیە زیادکراوەیەکمان دروست کرد بۆ تێکەڵکردنی _WooCommerce,_ کە pluginێکی زۆر باوی بازرگانی ئەلیکترۆنییە. گەشەپێدەرانی سەرانسەری جیهان زیادکراوەیان دروست کردووە بۆ تێکەڵکردنی دەروازە جیاوازەکانی پارەدان لەگەڵی. ئێمە سوودمان لەمە وەرگرت بۆ فراوانکردنی دەروازەکانی پارەدان کە دەتوانیت لەگەڵ سیستەمی ژمێریاری Ultimate Multisite بەکاریان بهێنیت.

_**گرنگ:** Ultimate Multisite: WooCommerce Integration پێویستی بەوەیە WooCommerce لانیکەم لەسەر ماڵپەڕە سەرەکییەکەت چالاککرابێت._

سەرەتا، تکایە بچۆ بۆ پەڕەی زیادکراوەکان. دەتوانیت بیدۆزیتەوە بە چوون بۆ **Ultimate Multisite > ڕێکخستنەکان**. دەبێت خشتەی **زیادکراوەکان** ببینیت. کرتە لەسەر **زیادکراوەکانمان بپشکنە** بکە.

<!-- وێنەی شاشە بەردەست نییە: خشتەی زیادکراوەکان لە لاتەنیشتی ڕێکخستنەکانی Ultimate Multisite لەگەڵ بەستەری Check our Add-ons -->

دوای کرتەکردن لەسەر **زیادکراوەکانمان بپشکنە** ، دەنێردرێیتەوە بۆ پەڕەی زیادکراوەکان. لێرە دەتوانیت هەموو زیادکراوەکانی Ultimate Multisite بدۆزیتەوە. کرتە لەسەر زیادکراوەی **Ultimate Multisite: WooCommerce Integration** بکە.

![پەڕەی زیادکراوەکان کە زیادکراوەکانی Ultimate Multisite پیشان دەدات، لەوانەش WooCommerce Integration](/img/addons/addons-page.png)

پەنجەرەیەک بە وردەکارییەکانی add-on ـەکە دەردەکەوێت. تەنها کرتە لەسەر **Install Now** بکە.

<!-- Screenshot unavailable: Ultimate Multisite WooCommerce Integration add-on details dialog with Install Now button -->

دوای تەواوبوونی دامەزراندن، دەنێردرێیتەوە بۆ پەڕەی plugin ـەکان. لێرەدا، تەنها کرتە لەسەر **Network Activate** بکە و add-on ـی WooCommerce لەسەر تۆڕەکەت چالاک دەکرێت.

<!-- Screenshot unavailable: Plugins page with the Network Activate link for the WooCommerce Integration add-on -->

دوای چالاککردنی، ئەگەر هێشتا plugin ـی WooCommerce ـت لەسەر ماڵپەڕەکەت دانەمەزراندووە و چالاکت نەکردووە، بیرخەرەوەیەکت پێدەگات.

<!-- Screenshot unavailable: Admin notice reminding the administrator to install and activate the WooCommerce plugin -->

بۆ خوێندنەوەی زیاتر دەربارەی add-on ـی WooCommerce Integration، **لێرە کرتە بکە**.
