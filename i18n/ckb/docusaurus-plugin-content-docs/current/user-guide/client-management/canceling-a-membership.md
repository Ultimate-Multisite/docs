# لغای ئەندازیاری یەکەم (v2)

_**تێبینی گرنگ: ئەم بابەتە ئاماژەیە بۆ Ultimate Multisite وەرگرتنی 2.x.**_

لغای کارپێربوون لەلایەن کڕیارەکانتان بە ئاسانی دەکرێت.

ئەگەر یوزەرەکەت دەیەوێت ئەندازیاری خۆی لغێنێت، پێویستە سەرەتا بگاتە لاپەڕەی **Account** (کاربر).

لە هەمان شوێوە، دەبێت سەری **Danger Zone** (ناوچەی مەترسیدار) بکەن و بۆ **Delete Account** (لغۆکردنی کارپێربوون) بگرن.

![Danger Zone لە لاپەڕەی کارپێربوون لەگەڵ دۆمبیلی لغۆکردنی کارپێربوون](/img/account-page/danger-zone.png)

یەکModuleName دەکرێتەوە کە داواکاریی پایداری (confirmation) دەکات. دەبێت **Confirm Account Deletion** (پایدارکردنی لغۆکردنی کارپێربوون) یەک بکەن و بۆ **Delete Account** بگرن.

![دۆمبیلی پشتڕاستکردنەوەی لغۆکردنی کارپێربوون](/img/account-page/delete-account-confirm.png)

ئەم کارە یوزەرەکە لە شوێنەکەدا دەچێت و ئەندازیارییەکەی لە تۆڕەکەت لغێنێت، هەروەها سایتەکەی لە تۆڕەکەت دەدبڕێتەوە.

![لە دوای لغۆکردنی کارپێربوون دچێت](/img/account-page/logged-out-after-delete.png)

![پشتڕاستکردنەوەی لغۆکردنی ئەندازیاری کارپێربوون](/img/account-page/membership-canceled.png)

**گرنگە:** ئاگادار باشید کە ئەم کارە کڕیارەکە لە تۆڕەکەت دەچێتە دەرەوە، چونکە یوزەرێک دەتوانرێت زیاتر لە یەک ئەندازیاری کارپێربوون یان هیچ ئەندازیاری کارپێربوونی پەیوەندیدار بەوە هەبێت.

![کڕیارەکە هێشتا لە تۆڕدا هەیە دوای لغاوەکردن](/img/admin/customer-after-cancel.png)

![کڕیارێک کە هیچ ئەندازیاری چالاک نییە](/img/admin/memberships-customer-empty.png)
