# چۆن دەتوانم سێبسنی خۆمم لغای بکەم؟ {#how-can-i-cancel-my-subscription}

ئەگەر بەڵگەنامەی Ultimate Multisite هەبێت، دەتوانیت هەر کاتێک کە دەتەوێت نوێکردنەوەکەی لغای بکەیت. تەنها چەند هەنگوعە خستنە پێش:

<!-- Screenshot unavailable: Account page email with access link -->

فەرموونی ئەو ئیمێلەی کە لە کاتی کڕیندا پێشکەش کردووە و پۆسڵەکەت بەکاربهێنە.

<!-- Screenshot unavailable: Login page with email and password fields -->

ئەمە سەرنجی یەکەمە بۆ لاگینی تۆ:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

بۆ لغایکردنی سێبسنی خۆت، لە منووی لایەنی چەپ، بگواشبە بە _Renewing & Billing._ (نوێکردنەوە و بڕین)

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

سەرنجی ئارامی ڕاست بۆ دابگرە بۆ گۆڕینی پەنجەرەیەکی لایەن. پاشان، دەبێت ئەو هەڵبژاردەی کە نوێکردنەوەی خۆت لغای دەکات هەڵبژێریت.

سیستەم یەکێک لەو ڕێگاکان پێشکەش دەکات بۆ بە confirماڵییەکە.

ئەگەر دڵنیایی هەیە دەتوانیت سێبسنتەکەت لغای بکەیت، تەنها پەنجرەی _**Cancel Renewals**_ (لغایکردنی نوێکردنەوەکان) بگرە. دوای ئەم کارە، پێشت دەپرسێت کە پرسیارێکی کورت بەجێبکەیت.

کرا! سێبسنتەکەت بە شێوەیەکی ئۆتۆماتیک نوێ نەکراوە.

**تا ئەو کاتەی سێبسنتەکەت کۆتایی دێت، کلای بەکارھێنانی وەرگرتوو دەبێت**. ئەگەر دەتەوێت سێبسنتەکەت دووبارە چالاک بکەیتەوە، پێویستە بە شێوەیەکی دەستی بۆ ئەنجام بدەیت.
