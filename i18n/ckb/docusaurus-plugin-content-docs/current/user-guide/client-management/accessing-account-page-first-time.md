# چۆن بۆ یەکەم جار دەتوانی بەشی تایبەتی خۆت ببینیت؟

بەخێربێن بۆ Ultimate Multisite!

لە ڕێگەی ئەو لینکەوە بچۆکە کە لە ئیمێلەکەدا دوای کڕینی پلاتین (plugin) ئێمە وەردەگرێت:

<!-- Screenshot unavailable: Account page email with access link -->

بۆ بەکارهێنانی ئیمێلی کە لە کاتی کڕیندا پێشکەش کردووە و ئەو وشەی پاسوورەی کە لە ئیمێلەکەدا دەستپێکردووە.

<!-- Screenshot unavailable: Login page with email and password fields -->

لە چونکە ئەمە یەکەم جارە کە دەتەوێت بچیتە ئەم بەشە، پێشت دەدرێت کە پاسوورەی خۆت بگۆڕیت. ئەو پاسوورەیە بگۆڕە و बटانی _Save_ (سەلامەتی) دەستکاری بکە.

<!-- Screenshot unavailable: Change password form with Save button -->

لە بەشی تایبەتی خۆت، دەتوانیت هەموو وردەکارییەکانی تایبەت بە ئەکاونتەکەت بەڕێوەبەر بکەیت، وەک بارکردنەوەکان (billings)، نوێکردنەوەکان (renewals)، لغاربوون (cancellations) و زۆر شتێکی دیکە.
