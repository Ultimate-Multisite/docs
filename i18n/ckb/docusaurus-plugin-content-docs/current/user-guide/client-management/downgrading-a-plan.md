# گۆڕینی پلاندان (v2)

_**تێبینی گرنگ: ئەم بابەتە ئاماژەیە بۆ Ultimate Multisite وەرگرتنی 2.x.**_

گۆڕینی پلاندان یان ئەبۆنمنتی شتێکی باوە لە کارلێنان کە مامەڵەی کێمیان هەیە یان بڕیار داوە کە پێویستیان بە سەرچاوە زۆرێک نییە بۆ بەڕێوەبردنی subsite.

## چۆنیەتی گۆڕینی پلاندان

کارلێن دەتوانن هەر کاتێک Login بکەن بۆ داشبۆردی admin ی subsite و لەسەر لاپەڕەی account (حساب) هەڵبژاردنی **Change** (گۆڕین).

![صفحة کارلێن حساب لەگەڵ کارت گواستنەوە و دغدغه گۆڕان](/img/account-page/membership-change-button.png)

کاتێک دەبێتە پشتبەری د assuming **Change**، بەکارهێنەر یان کارلێنان دەچێتە لاپەڕەی checkout (داواکاری پارەدان) کە لەوێ دەتوانن پلاندانێک هەڵبژێرن بۆ گۆڕینی ئەبۆنمنتی خۆیان.

![صفحة هەڵبژاردنی پلاند بۆ گۆڕین لەلایەن کارلێنەوە](/img/account-page/downgrade-picker.png)

لە ئەم نموونەدا، ئێمە پلاندانەکە دەگۆڕین لە **Premium** بۆ **Free**.

بۆ بەردەوامبوون، باری کاتێک دەبێت **Complete Checkout** (پایەندی پارەدان تەواو بکە) دابنێیت. دواتر دەگەڕێنەوە بۆ لاپەڕەی account کە پەیامێکی لەسەر گۆڕانەکە بۆ ئەبۆنمنتی نوێ نیشان دەدات. گۆڕانەکان لە **دوو ماوەی پارەدانەکەت** (next billing cycle) دەست پێ دەکەن.

![صفحة حساب کە بانر گۆڕینی ئەبۆنمتنی پاشەکەوتراوی هەیە نیشان دەدات](/img/account-page/pending-change.png)

### چی ڕوودەدات کاتێک کارلێنان پلاندانەکەی گۆڕێتەوە؟

گرنگە بڵێین کە گۆڕینی پلاندانەکە هیچ کاریگەریی لەسەر پێکهاتەی ئێستای subsite ی کارلێن ناکات.

ئەو بە شێوەیەکی ئۆتۆماتیک قالب (template) سایتەکە زیاد دەکات، چونکە گۆڕینی قالب سایتەکە هەموو زانیاری و ڕێکخستنی subsite یەکە دەپاکرێت و دەگەڕێتەوە بۆ سەرەتای کار. ئەمە بۆ ئەوەیە کە لەناوێی زیانێکی زۆر لە داتاکان نەبێت. بۆیە فەراشتنی دیسک، themes (قالبەکان)، plugins (پلانگینەکان) و هتد، هەموویان دەمێننەوە جگە لە posts (پستەکان).

ئەو چونکە ئێمە تێدەگۆڕین کە گرنگترین شت بۆ ئێوە ئەو سنوورە و کوتانە دەبینن کە لە ژێر هەر پلانی دابەشکردنی کار (plan) دیاری کردووە، بەڵام دەبێت بیر لە ئەوەش بکەنەوە کە چی زیانێک دەبێت بۆ زیرەکی ناوچەکە (subsite) ئەگەر یەکێک لە ڕێکخستنەکانی لەناویشیدا دیاریکراوە دیلیت بکرێت یان بگۆڕدرێت.

بۆ ئەو پۆستەکانەی سنوورێ پلانی دیاری کراو زیاتر دەبن، ئێوە 3 ڕێگای جیاوازت دەبینن: **پۆستەکان وەک خۆیان پاراستن** *,* **پۆستەکان بۆ باخچەی پاشەکەوت (trash) گواستنەوە** *,* یان **پۆستەکان بۆ نووسینی پڕۆژە (draft) گواستنەوە** *. دەتوانرێت ئەمە لە ژێر ڕێکخستنەکانی Ultimate Multisite دابەشکردنی کاردا دیاری بکەن.

![Network Admin Settings Sites page showing post-limit-exceeded behaviour options](/img/account-page/settings-sites-post-limit.png)

### چی دەبێت بە پامێرەکە (payment)؟

لە وەرگرتنی 2.0، دوو پێویستییەکیش نییە کە لە ڕێگەی گۆڕینی پاشکردنەوە (proration) بێت.

ئەمە چونکە سیستەمەکە چاوەڕێ دەکات تا ئەندازیی ئەندازیی یەکێک لە ئەندامانی ئێستا **دۆخی بارکردنەوەکەی تەواو بکات پێش ئەوەی** پلان/ئەندازی نوێ دەستپێبکێت. ڕەوانینی پاشکردنی نوێ بۆ ئەو ئەندازییە بە شێوەیەکی ئۆتۆماتیک داrang و لە چوارچێوەی بارکردنەوەی دوایەکەدا دادەنرێت و پارەکان وەردەگرێت.
