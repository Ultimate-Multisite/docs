# เว็บไซต์เดอะไคลเอนต์ของคุณ (v2)

หลังจากลงทะเบียนในเครือข่ายของคุณ ลูกค้าสามารถเข้าถึงแดชบอร์ดของตนได้

นอกจากหน้า **Account** ซึ่งพวกเขาจะเห็นข้อมูลสำคัญเกี่ยวกับวิธีการชำระเงิน สมาชิก โดเมน ข้อจำกัดของแผน และอื่นๆ แล้ว พวกเขายังสามารถเข้าถึงหน้า **My Sites** ซึ่งเป็นที่ที่พวกเขาสามารถดูและจัดการเว็บไซต์ทั้งหมดของตนได้ สิ่งเดียวที่ต้องทำคือคลิกที่ My Sites บนแดชบอร์ดของพวกเขา

![ลิงก์ My Sites ในแดชบอร์ดลูกค้า](/img/account-page/my-sites-link.png)

จะแสดงเว็บไซต์ทั้งหมดของพวกเขา พร้อมปุ่ม **Manage** อยู่ใต้แต่ละเว็บไซต์ หากพวกเขาคลิกที่ปุ่มนี้ พวกเขาจะถูกนำไปยัง **dashboard ของเว็บไซต์นั้นโดยเฉพาะ**

พวกเขายังสามารถดูได้ว่าเว็บไซต์ใดเป็นเว็บไซต์ **หลัก (primary)** สำหรับแผนที่อนุญาตให้มีเว็บไซต์มากกว่าหนึ่งเว็บไซต์

![หน้า My Sites แสดงเว็บไซต์ของลูกค้าพร้อมปุ่ม Manage](/img/account-page/my-sites-page.png)

นี่คือมุมมองเต็มของส่วนแก้ไขเว็บไซต์:

![อินเทอร์เฟซการแก้ไขเว็บไซต์แบบเต็ม](/img/admin/site-edit-full.png)

หากพวกเขาต้องการเปลี่ยนเว็บไซต์หลัก พวกเขาต้องไปที่หน้า Account โดยคลิกที่เมนู **Account**

![ลิงก์เมนู Account ในแดชบอร์ดลูกค้า](/img/account-page/account-menu.png)

พวกเขายังสามารถดูและแก้ไขรายละเอียดของลูกค้าได้:

![อินเทอร์เฟซการแก้ไขลูกค้า](/img/admin/customer-edit.png)

จากนั้น พวกเขาควรคลิกที่ **Change Default Site** (เปลี่ยนเว็บไซต์เริ่มต้น)

![ปุ่ม Change Default Site บนหน้า Account](/img/account-page/change-default-site.png)

จะมีหน้าต่างป๊อปอัปปรากฏขึ้น คลิกที่รายการแบบดรอปดาวน์และเลือกเว็บไซต์ที่คุณต้องการให้เป็นเว็บไซต์หลัก

![การเลือกในเมนู Change Default Site dropdown](/img/account-page/change-default-site-dropdown.png)

จากนั้น เพียงคลิกที่ **Change Default Site** และเสร็จเรียบร้อย!

![تاییدی بۆ تغییر سایت پیش‌فرض](/img/account-page/change-default-site-confirm.png)
