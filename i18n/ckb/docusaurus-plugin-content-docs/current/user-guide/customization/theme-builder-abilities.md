# تواناکتەکانی Theme Builder: Scaffold و Activate Block Themes

Superdav AI Agent v1.12.0 دوو تواناکتێکی بەهێز پێشکەش دەکات کە دەتوانیت لە ڕێگەی چاتەوە، theme-ی بلوک تایبەت دروست بکەیت و دامەزرێنیت (deploy) بکەیت.

## گشتیاری {#overview}

تواناکتەکانی **scaffold-block-theme** و **activate-theme** ئامراز دەدەن بە ئەژمارەکان بۆ:
- دروستکردنی theme-ی بلوکی تەواو و ئامادەکراو بۆ بەرهەمھێنان (production) لەسەر بنەمای داواکارییەکانت
- دامەزراندنی theme-ەکان بە شێوەیەکی خۆکار لەسەر سایتەکەت، بەبێ پێویستی دەستکاریی دەستکرد
- دروستکردنی ناسنامەی بینراوێکی یەکگرتوو لە ڕێگەی بڕوانامەکانی دیزاین

## Scaffold Block Theme (دامەزراندنی Theme بلوکی) {#scaffold-block-theme}

تواناکت **scaffold-block-theme** theme-ی نوێی WordPress block دروست دەکات کە پێکهاتەیەکی تەواوی theme ی پێشکەش دەکات، لەوانەش:

- `theme.json` configuration لەگەڵ design tokens (تۆکنە دیزاینەکە)
- فایلەکانی template بلوک بۆ لایەنە باوەکان (common layouts)
- سٹایل و جۆرەکانی بلوکی تایبەت
- Metadata ی theme و ڕاگەیاندنی پشتگیریکراوان

### چۆن بەکاربهێنیت {#how-to-invoke}

لە چاتەکەت لەگەڵ Superdav AI Agent، دەتوانی داواکاریی دروستکردنی theme بکەیت:

```
"Create a block theme called 'Modern Agency' with a blue and white color scheme,
sans-serif typography, and a professional layout"
```

Agent ئەنجام دەدات:
1. پێداویستی دیزاینەکەت لەسەر بنەمای گفتوگۆ کۆد بکات
2.โครงสร้าง تەواوی theme ی دروست بکات
3. هەموو فایلە پێویستەکانی theme دروست بکات
4. theme ی ئامادەکات بۆ دامەزراندن (activation)

### ئەنجامی پێداویست {#expected-output}

کاتێک تواناکتەکە بە سەرکەوتوو دەبێت، تۆ ئەمە دەبینیت:

- پایداریی ئەوەی theme ی دروستکراوە
- ناوی theme و شوێنی لەسەر سایتەکە
- پوختەی design tokens کە بەکارخراون (ڕەنگەکان، تایپۆگرافی، فاصلەکان)
- ئاستی ئامادەکراو بۆ دامەزراندن

نموونەی ئەنجام:
```
✓ Theme "Modern Agency" بە سەرکەوتوو scaffold کرا
  شوێن: /wp-content/themes/modern-agency/
  ڕەنگەکان: Primary #0066CC, Secondary #FFFFFF
  تایپۆگرافی: Inter (sans-serif)
  ئاست: ئامادە بۆ دامەزراندن
```

## Activate Theme (دامەزراندنی Theme) {#activate-theme}

قابلیتی **activate-theme** سایت شما را به یک قالب بلوکی جدید یا موجود تغییر می‌دهد.

### نحوه فراخوانی {#how-to-invoke-1}

بعد از ساختن یک قالب، می‌توانید آن را بلافاصله فعال کنید:

```
"فعال‌سازی قالب Modern Agency"
```

یا هر قالبی که از قبل وجود دارد را فعال کنید:

```
"تغییر به قالب Twentytwentyfour"
```

### خروجی مورد انتظار {#expected-output-1}

هنگامی که فعال‌سازی با موفقیت انجام می‌شود:

- تأیید قالب فعال شده
- نام قالب قبلی (برای مرجع)
- آدرس سایت جایی که قالب اکنون فعال است
- هر نکته تنظیمات خاص قالب

خروجی نمونه:
```
✓ قالب با موفقیت فعال شد
  قالب فعال: Modern Agency
  قالب قبلی: Twentytwentyfour
  فعال در: https://yoursite.com
  نکته: برای اطمینان از طرح جدید، صفحه اصلی خود را بررسی کنید
```

## گردش کار: ساخت و فعال‌سازی (Scaffold and Activate) {#workflow-scaffold-and-activate}

یک گردش کار معمول شامل هر دو قابلیت است:

۱. **درخواست تولید قالب**: "ایجاد یک قالب بلوکی برای صفحه فرود SaaS من"
۲. **Agent قالب را می‌سازد**: فایل‌ها و توکن‌های طراحی را تولید می‌کند
۳. **بررسی و اصلاح**: در صورت نیاز، تغییرات طراحی را بحث کنید
۴. **فعال‌سازی**: "قالب را اکنون فعال کن"
۵. **تأیید**: به سایت خود مراجعه کنید تا تأیید نمایید که طرح جدید فعال است

## توکن‌های طراحی و سفارشی‌سازی (Design Tokens and Customization) {#design-tokens-and-customization}

قالب‌های ساخته شده از توکن‌های طراحی وردپرس (از طریق `theme.json`) برای موارد زیر استفاده می‌کنند:

- **رنگ‌ها**: پالت اصلی، ثانویه، تأکیدی، خنثی
- **تایپوگرافی**: خانواده فونت، اندازه‌ها، وزن‌ها، ارتفاع خطوط
- **فاصله ها**: حاشیه (Padding)، مارجین (Margin)، مقیاس فاصله (Gap scales)
- **حاشیه‌ها**: توکن‌های شعاع و عرض
- **سایه‌ها**: سطوح ارتقاء (Elevation levels)

این توکن‌ها در `theme.json` متمرکز هستند، به طوری که تنظیم کل سیستم طراحی خود از یک فایل آسان است.

## محدودیت‌ها و نکات {#limitations-and-notes}

- Themes لە `/wp-content/themes/` دابەزراون و دەبێت پابەند بن بە ڕێزمانی ناونیشانەکانی WordPress
- چالاککردنەوەی (Activation) پێویستی بە ڕێگەپێدانی دروست لەسەر سایتەکەت هەیە

کۆدی PHP ی تایبەت لە ناو Theme'ەکاندا کەمە؛ بۆ کارکردنی ئاڵۆز، لە جیاتی ئەوە بەکاری دەهێنە **plugin** (پلانگین).
Theme-ی پۆلێمبۆک (Block themes) باشترین کارکردن دەکات لەگەڵ WordPress 5.9 و بەرزتر.

## چارەسەرکردنی کێشەکان (Troubleshooting) {#troubleshooting}

**Theme دوای دابەزراون دەردەکەوێت**
- چاک بکە ئایا دۆ렉ۆری Theme وہ هەیە و ڕێگەپێدانی دروست هەیە
- چاک بکە ئایا `theme.json` JSONی دروستە
- دڵنیابە ناوی Theme ی تۆ لەگەڵ Theme ی ئێستادا تێکەڵ نەکات

**چالاککردنەوە شکست دەخات**
- دڵنیا بە ئەوەی ڕێگەی Administrator (Administrator permissions) هەیە
- چاک بکە ئایا دۆ렉ۆری Theme بۆ WordPress بخوێنراوەتەوە
- تۆڕەکانی هەڵگرتنی Error ی WordPress بۆ وردەکارییەکان دەبینیت

**Design tokens جێبەجێ ناکرێن**
- چاک بکە ئایا سۆنتەکس (syntax) ی `theme.json` دروستە
- هەر pluginی کێشەی Cache یی پاک بکەرەوە
- دڵنیابە وەرگرتنی WordPress تۆن پشتیوانی سۆنتەکان کە بەکاری دەهێنیت

## هەنگاوەکانی دواتر (Next Steps) {#next-steps}

پاش چالاککردنەوەی Theme، دەتوانیت:
- لە **Design System Aesthetics skill** بەکاربهێنیت بۆ ڕێکخستنی تایبەتمەندییە تایبەتەکان وەک تایپۆگرافی، ڕەنگەکان و فاصلەکان (spacing)
- ستاای تایبەتی بلوکەکان بگۆڕیت لە ڕێگەی WordPress block editor
- CSS تایبەتی زیاتر لە فایل `style.css` ی Themeەکەدا زیاد بکە
- Templateی بلوکی تایبەت بۆ جۆرەکانی پەڕە (page types) دیاریکراو دروست بکە
