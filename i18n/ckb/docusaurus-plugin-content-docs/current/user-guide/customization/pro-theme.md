# ڕێکخستنی پۆینتی ئیدارە (Admin Panel) بە مۆدێل PRO Theme

زۆرجار کاتێک کار بۆ کڕی یان کلاینت دەکەین، پێویستی دەبینین کە ئەو کڕیان لە داشبۆردی WordPress-ەکەی ئەزموونی تایبەت و نایاب بێت. ئەگەر دەتەوێت شێوازێکی پڕۆفیشناڵتر بۆ دیزاینی هەست بەدەست بهێنیت، PRO Theme دەتوانێت یارمەتیت بدات.

PRO Theme مۆدێلێکە کە بۆ یارمەتیدانی تۆ لە زمانی گۆڕانکاریی داشبۆردی کلاینتە دروست کراوە. و هەروەها بەشێکە لە ناوەرۆکی سەرەکی (core) پلاتینی Ultimate Multisite.

پاش جێهێشتنی، بچۆ بۆ لاپەڕەی _Plugins_ و clique بکە بۆ چالاککردنەوەی (activate) لەسەر تۆڕەکەت (Network).

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_چالاککردنەوەی بە هەموو تۆڕەکە لە لاپەڕەی Plugins_

_**تێبینی:**_ _ئەگەر تەنها جێهێشتنی بکەیت، دەردەکەوێت بۆ کلاینتەکەت وەک یەکێک لە کارەکان کە دەتوانیت بەم شێوەیە دیاری بکەیت بۆ ئەوەی مۆدێلەکە تەنها بۆ ئەو بەکارهێنەرە زیاد بکرێت._

ئێستا، تۆ ئامۆژگاریی ئیدارەت بە تەواوی گۆڕاوە! هەروەها منوێکی نوێ لە لایەنی چەپ هەیە بە ناوی PRO Theme.
