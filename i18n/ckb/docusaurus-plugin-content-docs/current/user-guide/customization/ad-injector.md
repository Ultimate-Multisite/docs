# دانان بۆ ڕێکخستنی ئۆفڵایەکان لە تۆڕەکەت بە Ultimate Multisite: Ad Injector {#placing-ads-on-your-network-with-ultimate-multisite-ad-injector}

Ultimate Multisite: Ad Injector یەکێکە لە **پلانە بلاگینەکان** کە ئامرازەکانی پێشکەش دەکات بۆ ئەوەی دەتوانیت ڕیکلام لەسەر پلاندانەکانت لەو لایەنەی ویرایش دەکەیت.

پاش جێبەجێکردنی Ultimate Multisite: Ad Injector، بگواشبە بە لاپەڕەی _Plugins_ بۆ چالاککردنەوەی.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_کلیک بکە لە دغڕاندنی تۆڕ (Network Activate)_

کاتێک بلاگینی چالاک بوو، دەبێت لە ناو کۆتای _Advanced Options_ لە شاشەی ویرایش پلاندانەکەدا کارکردنێکی نوێ بە ناوی _Ad Settings_ هەبێت.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_ئێستا دەتوانیت ڕێکخستنی ڕیکلام بۆ هەر یەک لە پلاندانەکانت ویرایش بکەیت._

لەو تابلۆدا، دەتوانیت کۆدی ڕیکلامیەکە بۆ پۆستەکان و لاپەڕەکان لە پێش و دوای ناوەڕۆکەکەدا زیاد بکەیت.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_پاش داواکردنی کۆدی ڕیکلامیەکەت، لەسەر پۆستەکان و لاپەڕەکان بەو پلاندانەی دیاریکراوە دەردەکەوێت._

شایەنی ئەوەیە کە دەتوانیت ڕیکلام لە سەری (header) ی لاپەڕەکانی ئۆدمن پاشاڵ بۆ بەکارهێنەرانی ئەو پلاندانەیە زیاد بکەیت.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_بگەڕێ بۆ آپشنی کۆدی ڕیکلام لە پۆستەکانی ئۆدمن ئەگەر دەتەوێت ڕیکلام لە ناو بەشی پشتەوەی بەکارهێنەرەکەت نیشان بدات._

### زیاتر لە ڕیکلامکردن {#beyond-advertising}

دەتوانیت Ultimate Multisite: Ad Injector بەکاربهێنیت بۆ پشکنینی پلاندانەکانی ئۆفڵای بەرزتر.

تەنها بگواشبە بە آپشنی _Admin Panel Ads_ و کۆدی HTML-ی ئامراز پشتیوانیەکەت زیاد بکە.
