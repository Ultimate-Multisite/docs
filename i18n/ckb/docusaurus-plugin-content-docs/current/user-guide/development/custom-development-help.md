# چۆن یارمەتی تایبەت بۆ پەرەپێدانی لە کۆدەبل (Codeable) وەربگرین؟

هەندێک جار، ئێمە بە ڕاستی پێویست دەکەین وردکارییەکی زۆر باش هەبێت بۆ ئەوەی ڕێڕەوێژەکانی تۆڕمان بەو شێوازەی کە دەتانەوێت بێت. ئەگەر ئەمە ئەو شتەیە کە دەجووڵیت، ئێمە [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) پێشنیار دەکەین، شوێنێک کە لەوێدا دەتوانیت پەرەپێدەرانی ئەزموونی هەبێت و زانینی زۆر لە Ultimate Multisite بەدەستنەهێنن.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) پلاتفۆرمێکە کە کڕیارەکان دەبەستێتەوە بە پەرەپێدەرانی فری لانس کارکردنی WordPress کە مۆڵەتی پێویستیان هەیە و دەتوانن لەسەر پڕۆژەکانی هەر قۆناغێکی کاری گەشە بکەن. ئەوان ڕێکارێکی وردی داواکاریی هەیە بۆ دابینکردنی ئەوەی تۆ لەگەڵ کارەزایان کە تایبەت بە ئەو کارەی کە مەبەستتە، یەک دەبێتەوە.

### چۆن پڕۆژەیەکی لە Codeable دروست بکەین؟

پێشتر، بچۆ بۆ [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) و बटانی **Start a Project** (سەرچاوەتەوەی پڕۆژە دەستپێکردن) هەڵبژێرە.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

ئێستا، بۆ پێشکەشکردنی پڕۆژەیەکی کاری، لە لایەنی چەپ بۆ **Start a Project** (سەرچاوەتەوەی پڕۆژە دەستپێکردن) هەڵبژێرە.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

لە بەشی "I need help with" (من پێویستم بە یارمەتی هەیە لە)، **Customizing** (وردکاری) هەڵبژێرە :

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

لە بەشی "My" (من)، **Plugin** (پلانگین) هەڵبژێرە :

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

ئەو کاتدا، बटانی **Continue** (دایە بەردەوام بیت) دەست پێدەکەیت.

<!-- Screenshot unavailable: Codeable project form Continue button -->

ئێستا، کاتێکە بۆ وەسفکردنی پڕۆژەکەت و پێویستییەکانت بە باشترین شێواز کە دەتوانیت بکەیت. هەوڵ بدە زۆرترین فایدهی دەربڕین!

<!-- Screenshot unavailable: Codeable project description form -->

پاسان بەم پرسیارە بۆ یارمەتی دابینکردنی نرخ.

<!-- Screenshot unavailable: Codeable project pricing questions -->

پاش ئەوەی کە یەکەم کارە، ئەگەر هێشتا شتێکی [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) نییە، یەکێک لەو دروست بکە و د assuming **Publish My Project** बटانی هەڵبژێرە.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

کاتێک پڕۆژەی [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) یەکەم دابەش کرد، پلاتفۆرم دەست بە پەیوەندی تۆ لەگەڵ دزنوێنەرانی دەکات کە تواناکان لەگەڵ پێداویستی پڕۆژەکەت یێدات.

ئەم کارە یارمەتیت دەدات بۆ دۆزین ئەو کەسەی کە بۆ کارەکەت گونجاوە و ئەزموونی هەیە لە WordPress و Ultimate Multisite!

کەمترین نرخەکانی Codeable $70–$120 دراوە بە کاتژمێر. پێداچوونەوەکان دەگۆڕێن بەپێی قۆناغی کار، ئاستی پیچوەر و خێرایی پڕۆژەکە.

کاتێک پێداچوونەوەکەت گرت، دەتوانیت دزنوێنەرەکەت بگرە کە باشترین شێوەیە بۆ پێداویستییەکانی تۆ و یەکخۆ بەپشتى پڕۆژەکەت دەست بکە!
