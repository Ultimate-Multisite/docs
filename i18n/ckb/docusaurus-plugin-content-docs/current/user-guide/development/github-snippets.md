# چۆنی بەکارهێنانی قطعات (snippets) ی Ultimate Multisite لە گێڕی GitHub ی ئێمە

لە گێڕی GitHub-دا، قطعات کۆدی هەن کە زۆر داواکراون لەلایەن بەکارهێنەرانی Ultimate Multisite کە د tiếp دەوێت کارایییە بچووک وەک زیادکردنی سکرێپتی Google Analytics لە صفحات تۆمار (sign-up pages) یان پۆشاندنی کاتی meta box لە داشبۆردی ئیدارەدا ئەنجام بدەن.

ئەم بابەتە دەربڕینێت چۆن بەکاریان دەهێنیت یان بە تایبەت کوێ دەتوانیان ئەم کۆدان دابنێیت.

دەتوانید ئەم قطعاتە لە لۆکن خوارەوە بدۆزیتەوە:

https://github.com/next-press/wp-ultimo-snippets/

دوو ڕێگای هەیە بۆ زیادکردنی کۆدەکە:

  1. لە فایلەکەی functions.php ی تێمەکەتدا.

  2. Must-Use Plugins (mu-plugins)

# چۆن قطعتەکە لە فایلەکەی theme'ەکەتدا functions.php دابنێیت.

  1. بچۆ بۆ داشبۆردی ئیدارەی WordPress Network و بگواشبە Themes > Theme Editor (وێنەی خوارەوە ببینە).

  2. لەو لاپەڕەیە کە دەڵێت "Edit Themes"، دڵنیابە کە تێمەکەت کە چالاکەیت لەسەر دروستکراوە، لەسەر فیلدی dropdownی کە لە لای ڕاستی سەرەوەی شاشەکەتدا هەیە (تێبینی: وێنەی خوارەوە).

  3. سەر بە بەشی "Theme Files" کلیک بکە لەسەر فایلەکەی functions.php بۆ بارکردن. لە خاڵەکانی خوارەوە سکرول بکە و قطعت Ultimate Multisite کە لە گێڕی GitHub وەربگرتووین، دابنێیت.

<!-- وێنەکە بەردەست نییە: WordPress Theme Editor کە دەبینێت لە کارکردنی فایلەکەی functions.php -->

# چۆنی Must-Use Plugins (mu-plugins) دروست بکەیت.

WordPress تایبەتمەندیێک هەیە کە ڕێگە دەدات کۆمەڵێک کارایی تایبەت بەکاربهێنیت و پێی دەوترێت "Must-Use Plugins"، یان بە کورت MU-plugins.

ئەم mu-plugins تایبەتە پێش هەموو پلاتینی وردیەکانی دیکە باردەکرێن، و ناتوانن لەکاتی کارکردندا بگرنەوە (deactivate). لە تۆڕێکی multisite، کۆدەکە لەم mu-plugins یاندا لەسەر هەموو سایتەکانت بار دەکرێت.

1. بە FTP یان SSH دەتوانیت دەستکاری فایلی سیستەمی فایلەکانی WordPress بکەیتەوە.

٢. لە دایرێбак `wp-content`ی دامەزرێنانی WordPress یەکەت، دایرێبێکی نوێ بە ناوی: `mu-plugins` دروست بکە.

<!-- وێنەیەکی شێوەی نەدۆزراوەتەوە: فایلی مێنافەری فایلەکە دەبینێت کە دایرێب `wp-content` لەگەڵ فۆلเดری `mu-plugins` دەردەخات -->

٣. لەسەر کۆمپیوتەرەکەت، فایلی PHP ی نوێیەکی بە ناوی `wu-snippet.php` دروست بکە بە وردی (Notepad) یان هەر ویرتۆری کدێکی دیکە.

٤. کۆدنی Ultimate Multisite کە لە گیتھب (GitHub) وەربگرتوویت، دابنێ لەناو ئەو فایلییە و سیوڕانەوەش بکە. دەتوانیت ئەم کۆدانە لە سەرەوەی کۆدەکە زیاد بکەیت بۆ ئەوەی ناوی پڵگینی `mu` یەکەت دیاری بکەیت.
