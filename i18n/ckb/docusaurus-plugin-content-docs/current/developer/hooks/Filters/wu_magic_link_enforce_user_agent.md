# فلتەر: wu_magic_link_enforce_user_agent

فلتەر بکە کە ئایا پشتڕاستکردنەوەی user agent جێبەجێ بکرێت یان نا.

دابنێ بۆ false بۆ ئەوەی tokenەکان لە نێوان وێبگەڕ/ئامێری جیاوازدا کار بکەن. ئەمە ئاسایش کەم دەکاتەوە بەڵام بەکارهێنان ئاسانتر دەکات.

## پارامێتەرەکان {#parameters}

| ناو | جۆر | باسکردن |
|------|------|-------------|
| $enforce | `bool` | ئایا هاوتابوونی user agent جێبەجێ بکرێت یان نا. |

- 2.0.0
