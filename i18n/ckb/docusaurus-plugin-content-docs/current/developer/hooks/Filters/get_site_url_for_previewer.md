# Filter: get_site_url_for_previewer

ڕێگە بدە بە گەشەپێدەرانی plugin کە URLـی بەکارهاتوو لە previewerدا فلتەر بکەن

## پارامێتەرەکان

| ناو | جۆر | وەسف |
|------|------|-------------|
| $domain | `string` | دۆمەینی بنەڕەتی کە لە ئێستادا بەکاردێت، سوودبەخشە بۆ دەستکارییەکان |
| $domain_options | `array` | لیستی هەموو هەڵبژاردەکانی دۆمەین کە لە Ultimate Multisite Settings -&gt; Network Settings -&gt; Domain Options داخڵ کراون |

### لە وەشانی

- 1.7.2
