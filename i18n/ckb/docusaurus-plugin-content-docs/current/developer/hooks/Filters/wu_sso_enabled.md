# Filter: wu_sso_enabled

توانای cross-domain single-sign-on چالاک/ناچالاک بکە.

ئەم بەهایە filter بکە بۆ ئەوەی single-sign-on بە تەواوی ناچالاک بکەیت، یان لەبری ئەوە بە مەرج چالاکی بکەیت.

## پارامێتەرەکان

| ناو | جۆر | وەسف |
|------|------|-------------|
| $enabled | `bool` | ئایا SSO دەبێت چالاک بێت؟ True بۆ چالاک، false-ish بۆ ناچالاک. |

### لە وەشانی

- 2.0.11
### سەرچاوە

پێناسەکراوە لە [`inc/sso/class-sso.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/sso/class-sso.php#L110) لە هێڵی 110


## گەڕانەوەکان
ئەوەی SSO چالاکە یان نا.
