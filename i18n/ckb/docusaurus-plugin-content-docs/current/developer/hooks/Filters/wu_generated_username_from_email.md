# Filter: wu_generated_username_from_email

Filterی ناوی بەکارهێنەری دروستکراو لە ئیمەیڵ.

## پارامێتەرەکان

| ناو | جۆر | وەسف |
|------|------|-------------|
| $username | `string` | ناوی بەکارهێنەری دروستکراو. |
| $email | `string` | ناونیشانی ئیمەیڵی کڕیاری نوێ. |
| $new_user_args | `array` | Arrayی ئارگۆمێنتەکانی بەکارهێنەری نوێ، لەوانەیە ناوی یەکەم و کۆتایی تێدابێت. |
| $suffix | `string` | دەقێک زیاد بکە بۆ ناوی بەکارهێنەر بۆ ئەوەی تاکانە بێت. |
