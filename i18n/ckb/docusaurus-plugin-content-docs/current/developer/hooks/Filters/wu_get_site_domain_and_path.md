# Filter: wu_get_site_domain_and_path

ڕێگە بە گەشەپێدەران دەدات جووتەکانی domain/path دەستکاری بکەن.

ئەمە دەتوانێت بۆ چەندین شت بەسوود بێت، وەک جێبەجێکردنی جۆرێک لە چارەسەری staging، سێرڤەری جیاواز، هتد.

## پارامێتەرەکان

| ناو | جۆر | وەسف |
|------|------|-------------|
| $d | `object` | ئۆبجێکتی ئێستا کە کلیلی domain و pathی تێدایە. |
| $path_or_subdomain | `string` | path/subdomainی ڕەسەن کە بۆ فەنکشنەکە نێردراوە. |

### لە وەشانی

- 2.0.0
### سەرچاوە

پێناسە کراوە لە [`inc/functions/site.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/site.php#L235) لە هێڵی 235


## گەڕاندنەوە
ئۆبجێکتێک کە کلیلی domain و pathی تێدایە.
