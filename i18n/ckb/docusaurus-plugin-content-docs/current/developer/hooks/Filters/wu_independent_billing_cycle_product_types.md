# فلتەر: wu_independent_billing_cycle_product_types {#filter-wuindependentbillingcycleproducttypes}

ئەو جۆرە بەرهەمانە فلتەر بکە کە سووڕی پارەدانێکی سەربەخۆیان هەیە.

بەرهەمەکانی خاوەن سووڕی پارەدانی سەربەخۆ ناچار ناکرێن لە cartدا لەگەڵ ماوەی پارەدانی پلانەکەدا یەک بگرنەوە. ئەمە سوودمەندە بۆ بەرهەمگەلێک وەک تۆمارکردنی دۆمەین کە هەمیشە ساڵانە پارەیان دەدرێت، بەبێ گوێدانەوە بەوەی پلانەکە مانگانە بێت یان ساڵانە.
