# Filter: wu_apply_plan_limits {#filter-wuapplyplanlimits}

ڕێگە بە پەرەپێدەرانی plugin بدە سنووردارکردنەکان کورت‌بکەنەوە.

دەتوانیت ئەم filterە بەکاربهێنیت بۆ جێبەجێکردنی هەر codeێکی خوازراو پێش ئەوەی هەر یەک لە سنووردارکردنەکان دەست پێبکات. ئەگەر filterەکەت هەر بەهایەکی truthy بگەڕێنێتەوە، پڕۆسەکە بەردەوام دەبێت؛ ئەگەر هەر بەهایەکی falsy بگەڕێنێتەوە، codeەکە دەگەڕێتەوە و هیچ کام لە hookەکانی خوارەوە جێبەجێ نابن.
