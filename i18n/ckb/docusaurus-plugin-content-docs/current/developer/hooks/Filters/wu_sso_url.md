# فلتەر: wu_sso_url

URLـەکانی SSO کە دروستکراون فلتەر دەکات پێش ئەوەی بۆ کردارەکانی کڕیار لە نێوان دۆمەینەکاندا بگەڕێندرێنەوە.

ئەم فلتەرە بەکاربهێنە کاتێک یەکخستنێک پێویستی بە زیادکردنی کۆنتێکستی متمانەپێکراو هەیە بۆ بەستەری SSOـی کرێگری خاوەن سەروەری، یان گۆڕینی URLـی ناوبژیوان لە کاتێکدا کە پشتڕاستکردنەوەی تۆکەنی Ultimate Multisite دەپارێزرێت.

## پارامێتەرەکان

| ناو | جۆر | وەسف |
|------|------|-------------|
| $sso_url | `string` | URLـی SSOـی دروستکراو. |
| $user | `WP_User` | ئەو بەکارهێنەرەی بە سەردانی SSO ڕەسەنایەتی بۆ دەکرێت. |
| $site_id | `int` | IDـی ماڵپەڕی ئامانج بۆ سەردانەکە. |
| $redirect_to | `string` | URLـی مەبەست دوای پشتڕاستکردنەوەی سەرکەوتووی SSO. |

### لە وەشانی

- 2.13.0

### سەرچاوە

لە `inc/sso/class-sso.php` پێناسە کراوە.


## دەگەڕێنێتەوە

URLـی SSOـی فلتەرکراو.
