# فلتەر: wu_pre_save_settings

ڕێگە بە پەرەپێدەران بدە ڕێکخستنەکان پێش پاشەکەوتکردن لەلایەن Ultimate Multisite فلتەر بکەن.

## پارامێتەرەکان {#parameters}

| ناو | جۆر | وەسف |
|------|------|-------------|
| $settings | `array` | ئەو ڕێکخستنانەی کە پاشەکەوت دەکرێن. |
| $settings_to_save | `array` | ڕێکخستنە نوێیەکان بۆ زیادکردن. |
| $saved_settings | `array` | ڕێکخستنە پاشەکەوتکراوە هەنووکەییەکان. |
