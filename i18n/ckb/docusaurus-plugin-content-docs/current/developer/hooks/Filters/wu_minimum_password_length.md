# Filter: wu_minimum_password_length

کەمترین درێژیی وشەی نهێنی فلتەر بکە.

تەنها کاتێک جێبەجێ دەکرێت کە wu_enforce_password_rules ڕاست بێت.

## پارامێتەرەکان

| ناو | جۆر | وەسف |
|------|------|-------------|
| $min_length | `int` | کەمترین درێژیی وشەی نهێنی. بنەڕەت 12 (هاوتای Defender Pro). |
| $defender_active | `bool` | ئایا Defender Pro Strong Password چالاکە. |

### لە وەختی

- 2.4.0
### سەرچاوە

لە [`inc/class-scripts.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-scripts.php#L543) لە هێڵی 543 پێناسە کراوە.
