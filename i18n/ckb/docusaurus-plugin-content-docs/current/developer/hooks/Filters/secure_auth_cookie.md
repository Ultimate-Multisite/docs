# فلتەر: secure_auth_cookie

فلتەر دەکات کە ئایا auth cookie تەنها لەسەر HTTPS بنێردرێت یان نا.

## پارامیتەرەکان {#parameters}

| ناو | جۆر | وەسف |
|------|------|-------------|
| $secure | `bool` | ئایا cookie تەنها لەسەر HTTPS بنێردرێت یان نا. |
| $user_id | `int` | ناسنامەی بەکارهێنەر. |

- 3.1.0
