# Filter: wu_documentation_get_link

ڕێگە بدە بە گەشەپێدەرانی plugin کە لینکەکان فلتەر بکەن.

دڵنیا نیم چۆن ئەمە دەتوانێت سوودبەخش بێت، بەڵام بوونی زیان ناکات

## پارامیتەرەکان

| ناو | جۆر | وەسف |
|------|------|-------------|
| $link | `string` | ئەو لینکەی تۆمارکراوە |
| $slug | `string` | ئەو slug ـەی بۆ وەرگرتنەوەی لینکەکە بەکارهاتووە |
| $default_link | `string` | ئەو لینکی بنەڕەتییەی تۆمارکراوە |

### لە وەشانەوە

- 1.7.0
