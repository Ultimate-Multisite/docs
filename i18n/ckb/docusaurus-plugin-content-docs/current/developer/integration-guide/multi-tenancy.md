# یەکخستنی Multi-Tenancy

Ultimate Multisite: Multi-Tenancy 1.2.0 چەند خاڵی پەیوەندی یەکخستن دەگۆڕێت بۆ کرێدارە سەربەخۆکان، پشتڕاستکردنەوەی کۆچکردن، و ئۆتۆماتیککردنی ژیاننامەی کرێدار.

## ڕەوتی دەستپێکردنی کرێدار {#tenant-bootstrap-flow}

ئەو یەکخستنانەی کرێدار دروست دەکەن یان دەستکارییان دەکەن، دەبێت ئەم ڕیزە پەیڕەو بکەن:

1. تۆماری ڕیجستریی کرێدار و مۆدێلی جیاکردنەوە دیاری بکە.
2. نووسەری بنکەدراوەی کرێدار دروست بکە یان پشتڕاستی بکەرەوە.
3. شێماکەی کرێدار دەستپێبکە.
4. بەکارهێنەرانی کرێدار ئامادە بکە.
5. ڕێڕەوی کرێدار و ڕێڕەوەکانی سیستەمی پەڕگە تۆمار بکە.
6. پێش ئەوەی کرێدار ئاشکرا بکرێت، پشتڕاستکردنەوەی کۆچکردن جێبەجێ بکە.

وا مەزانە کرێداری سەربەخۆ دەتوانێت پەیوەندیی بنکەدراوەی تۆڕ دووبارە بەکاربهێنێت. ڕیجستریی کرێدار و ئەبستراکشنی نووسەر بەکاربهێنە کە لەلایەن زیادکراوەکەوە دابینکراون.

## قولاپەکانی SSO و REST {#sso-and-rest-hooks}

چوونەژوورەوەی ئۆتۆماتیکی کرێداری بێ‌دۆخ توکنی کورت‌ماوە بەکاردەهێنێت کە claimـی مەبەست، پاراستن لە دووبارەبەکارهێنانەوەی JTI، سنووری بەسەرچوون، و چەسپاندنی سەرچاوەی هەیە. ئەو یەکخستنانەی دوگمەی چوونەژوورەوە یان بەستەری بەڕێوەبردنی دوور زیاد دەکەن، دەبێت سەردانی کرێدار لە ڕێگەی ڕەوتی SSO پشتگیریکراوەوە دروست بکەن، نەک URL ـەکانی چوونەژوورەوەی کرێدار بە شێوەی ڕاستەوخۆ دابنێن.

ڕووداوەکانی پشکنینی API لەلای تۆڕ و پوختەی ڕۆژانە بۆ دەروازەکانی کرێداری سەربەخۆ بەردەستن. ئەو تۆمارانە بەکاربهێنە کاتێک سیستەمە دەرەکییەکان کە بانگی endpoint ـەکانی ژیاننامەی کرێدار دەکەن چارەسەر دەکەیت.

## URL ـەکانی کرداری کڕیاری سەربەخۆ {#sovereign-customer-action-urls}

Ultimate Multisite v2.13.0 کردارەکانی کڕیاری کرێداری سەربەخۆ دەگەڕێنێتەوە بۆ ماڵپەڕی سەرەکی بۆ ڕەوتەکانی account، checkout، billing، پسووڵە، ماڵپەڕ، گۆڕینی قاڵب، و domain-mapping. ئەو یەکخستنانەی بەستەرەکانی بەڕێوەبردنی لای کرێدار پیشان دەدەن، دەبێت ئەو کردارانە ئاراستەی پانێڵی کڕیاری ماڵپەڕی سەرەکی بکەن و ئامانجێکی گەڕانەوەی پشتڕاستکراوە لەخۆبگرن کاتێک بەکارهێنەر دەبێت بتوانێت دوای تەواوکردنی کردارەکە بگەڕێتەوە بۆ کرێدار.

wrapper ـی SSO ـی ناوەکی بەکاربهێنە بۆ بەستەرەکانی بەڕێوەبردنی نێوان-دۆمەین:

```php
$url = wu_with_sso($main_site_customer_url);
```

URL ـی دروستکراو هێشتا لە ڕێگەی `wu_sso_url` ـەوە filterable دەمێنێتەوە، کە URL ـی SSO، بەکارهێنەری ئێستا، ID ـی ماڵپەڕی ئامانج، و دەقوباری redirect وەردەگرێت. زیادکراوەکان دەتوانن ئەو فلتەرە بەکاربهێنن بۆ زیادکردنی دەقوباری تایبەت بە دابینکەر یان بۆ گۆڕینی URL ـی broker لە کاتێکدا پشتڕاستکردنەوەی توکنی Ultimate Multisite دەپارێزن.

دۆخی ئەندامێتی، پسووڵە، ناونیشانی-billing، قاڵب، یان بەڕێوەبردنی دۆمەین لەناو کرێداری سەربەخۆدا دووبارە مەکەرەوە. dashboard ـی کرێدار وەک دەستپێکەر و پانێڵی کڕیاری ماڵپەڕی سەرەکی وەک سیستەمی تۆماری فەرمی بۆ کردارە بەڕێوەبراوەکان مامەڵە پێوە بکە.

## پشتڕاستکردنەوەی کۆچکردن {#migration-verification}

دوای ئەوەی کۆچکردنێک یان یەکخستنێکی ژیاننامە، دراوەکانی کرێدار دەگۆڕێت، دەروازەکانی پشتڕاستکردنەوە جێبەجێ بکە:

- `wp tenant verify-no-legacy --site=<site-id>` پشتڕاستی دەکاتەوە کە کرێدار چیتر پشت بە ڕێڕەوە کۆنەکانی لای تۆڕ نابەستێت.
- `wp tenant verify-sovereign-push --site=<site-id>` پشتڕاستی دەکاتەوە کە کارەکانی sovereign push دەتوانن پرۆسە بکەن و بەتاڵ ببنەوە.

یەکخستنان دەبێت پشتڕاستکردنەوەی سەرنەکەوتوو وەک ڕێگرێکی بڵاوکردنەوە هەژمار بکەن و خۆیان بپارێزن لە نیشانکردنی کرێدارێک وەک live تا ئەو کێشەیە چارەسەر دەکرێت.

## سڕینەوەی کرێدار {#tenant-deletion}

ڕەوتەکانی سڕینەوە دەبێت بانگی ڕێڕەوی teardown ـی زیادکراوەکە بکەن بۆ ئەوەی بڕوانامەکانی بنکەدراوەی کرێدار پاک بکرێنەوە. یەکخستنە دەرەکییەکان دەتوانن دوای سەرکەوتنی teardown سەرچاوەکانی دابینکەر لاببەن، بەڵام نابێت بنکەدراوە یان بوخچەکانی host بسڕنەوە کاتێک پشتڕاستکردنەوە یان کارەکانی async push هێشتا کار دەکەن.

## ڕێڕەوکەری بنکەدراوەی بەسەرچوو {#deprecated-database-router}

`Database_Router` ـی کۆن بە stub ـێکی deprecation جێگۆڕکێی پێکراوە. یەکخستنە نوێیەکان دەبێت کرێدارەکان لە ڕێگەی ڕێڕەوکەری ئێستای ماڵپەڕ و API ـەکانی ڕیجستریی کرێدارەوە دیاری بکەن، نەک پشت بە پۆلی ڕێڕەوکەری کۆن ببەستن.
