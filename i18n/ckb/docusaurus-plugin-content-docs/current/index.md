# Dokumentasyona Ultimate Multisite

بسیار خوش آمدید به مستندات Ultimate Multisite. Ultimate Multisite تبدیل وردپرس 멀티사이트 را به یک پلتفرم وب‌سایت به عنوان سرویس (WaaS) کامل می‌کند.

## بخش‌های مستندات {#documentation-sections}

### [راهنمای کاربر](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

همه چیزهایی که برای راه‌اندازی، پیکربندی و مدیریت پلتفرم WaaS خود نیاز دارید. شامل نصب، درگاه‌های پرداخت، نگاشت دامنه، مدیریت مشتریان و موارد دیگر است.

### [مستندات توسعه‌دهندگان](/developer/hooks) {#developer-documentation}

مرجع فنی برای توسعه‌دهندگانی که بر روی Ultimate Multisite کار می‌کنند. شامل مرجع هوک‌ها (hooks reference)، مستندات REST API و راهنماهای توسعه افزونه (addon development guides).

### [افزونه‌ها](/addons) {#addons}

مستندات مربوط به افزونه‌های پولی که Ultimate Multisite را با ویژگی‌های اضافی مانند یکپارچه‌سازی WooCommerce، تحلیل داده (analytics)، فروش دامنه و موارد دیگر گسترش می‌دهند.

- [GitHub Issues](https://github.com/Ultimate-Multisite/ultimate-multisite/issues) -- گزارش باگ‌ها (bugs)
