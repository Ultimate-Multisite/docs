# មេរៀនទី 8: ការណែនាំអតិថិជនចាប់ផ្តើមប្រើប្រាស់ {#lesson-8-customer-onboarding}

ការធ្វើឱ្យអតិថិជនចុះឈ្មោះគឺបានត្រឹមពាក់កណ្តាលការងារប៉ុណ្ណោះ។ ប្រសិនបើពួកគេចូលប្រើ រួចមានអារម្មណ៍ថាស្មុគស្មាញ ហើយមិនត្រឡប់មកវិញទេ អ្នកបានបាត់បង់ពួកគេហើយ។ មេរៀននេះរៀបចំបទពិសោធន៍ដែលបម្លែងអ្នកចុះឈ្មោះថ្មីឱ្យក្លាយជាអតិថិជនសកម្ម និងមានការចូលរួម។

## កន្លែងដែលយើងបានបញ្ចប់ {#where-we-left-off}

FitSite ត្រូវបានដាក់ម៉ាកយីហោពេញលេញ និងមានលំហូរទូទាត់ប្រាក់ដែលដំណើរការ។ ឥឡូវនេះយើងផ្តោតលើអ្វីដែលកើតឡើងបន្ទាប់ពីម្ចាស់ស្ទូឌីយោហាត់ប្រាណបញ្ចប់ការចុះឈ្មោះ ហើយចូលដល់គេហទំព័រថ្មីរបស់ពួកគេជាលើកដំបូង។

## ហេតុអ្វីការណែនាំចាប់ផ្តើមប្រើប្រាស់មានសារៈសំខាន់ {#why-onboarding-matters}

30 នាទីដំបូងបន្ទាប់ពីចុះឈ្មោះ កំណត់ថាតើអតិថិជននឹងនៅបន្ត ឬបោះបង់។ ម្ចាស់ស្ទូឌីយោហាត់ប្រាណដែល៖

- ចូលប្រើ ហើយឃើញគេហទំព័រដែលមើលទៅដូចគេហទំព័រហាត់ប្រាណរួចហើយ → នៅបន្ត
- ដឹងច្បាស់ថាត្រូវធ្វើអ្វីបន្ទាប់ → នៅបន្ត
- មានអារម្មណ៍វង្វេងនៅក្នុង WordPress dashboard ទូទៅ → ចាកចេញ

គំរូសម្រាប់ទីផ្សារឯកទេសរបស់អ្នក (មេរៀនទី 4) ដោះស្រាយចំណុចទីមួយ។ មេរៀននេះដោះស្រាយចំណុចទីពីរ។

## បទពិសោធន៍ចូលប្រើលើកដំបូង {#the-first-login-experience}

### វីជេត Dashboard ស្វាគមន៍ {#welcome-dashboard-widget}

បង្កើតវីជេត dashboard ផ្ទាល់ខ្លួន ដែលស្វាគមន៍អតិថិជនថ្មី និងណែនាំពួកគេឱ្យរៀបចំ។ វាគួរបង្ហាញយ៉ាងច្បាស់នៅពេលពួកគេចូលប្រើជាលើកដំបូង។

**ការចាប់ផ្តើមរហ័ស FitSite:**

1. **បន្ថែមឈ្មោះ និងស្លាកសញ្ញាស្ទូឌីយោរបស់អ្នក** -- ភ្ជាប់ទៅកាន់ការកំណត់ Customizer ឬ Site Identity
2. **ធ្វើបច្ចុប្បន្នភាពកាលវិភាគថ្នាក់របស់អ្នក** -- ភ្ជាប់ផ្ទាល់ទៅកាន់កម្មវិធីកែទំព័រ Classes
3. **បន្ថែមគ្រូបង្វឹករបស់អ្នក** -- ភ្ជាប់ទៅកាន់កម្មវិធីកែទំព័រ Trainers
4. **កំណត់ព័ត៌មានទំនាក់ទំនងរបស់អ្នក** -- ភ្ជាប់ទៅកាន់កម្មវិធីកែទំព័រ Contact
5. **មើលគេហទំព័ររបស់អ្នកជាមុន** -- ភ្ជាប់ទៅកាន់ផ្នែកខាងមុខ

ជំហាននីមួយៗភ្ជាប់ផ្ទាល់ទៅកាន់ទំព័រ ឬការកំណត់ដែលពាក់ព័ន្ធ។ មិនចាំបាច់ស្វែងរកតាមម៉ឺនុយទេ។

### ធ្វើឱ្យ Dashboard សាមញ្ញ {#simplify-the-dashboard}

អតិថិជនថ្មីមិនចាំបាច់ឃើញធាតុម៉ឺនុយ WordPress ទាំងអស់ទេ។ សូមពិចារណា៖

- លាក់ធាតុម៉ឺនុយដែលមិនពាក់ព័ន្ធនឹងការគ្រប់គ្រងគេហទំព័រហាត់ប្រាណ (ឧទាហរណ៍ Comments ប្រសិនបើមិនប្រើ)
- រៀបលំដាប់ម៉ឺនុយឡើងវិញ ដើម្បីដាក់ធាតុដែលប្រើញឹកញាប់ជាងគេនៅខាងលើ
- បន្ថែមស្លាកម៉ឺនុយផ្ទាល់ខ្លួនដែលសមស្របនឹងទីផ្សារឯកទេស ("ស្ទូឌីយោរបស់អ្នក" ជំនួស "រូបរាង")

Addon [កម្មវិធីគ្រប់គ្រង plugin និង theme](/addons/plugin-and-theme-manager) អាចជួយគ្រប់គ្រងអ្វីដែលអតិថិជនមើលឃើញ។

## លំដាប់អ៊ីមែលស្វាគមន៍ {#welcome-email-sequence}

អ៊ីមែលស្វាគមន៍តែមួយមិនគ្រប់គ្រាន់ទេ។ រៀបចំលំដាប់ដែលណែនាំអតិថិជនក្នុងសប្តាហ៍ដំបូងរបស់ពួកគេ៖

### អ៊ីមែល 1: ស្វាគមន៍ (ភ្លាមៗបន្ទាប់ពីចុះឈ្មោះ) {#email-1-welcome-immediately-after-signup}

- ប្រធានបទ៖ "ស្វាគមន៍មកកាន់ FitSite -- គេហទំព័រស្ទូឌីយោរបស់អ្នកបានដំណើរការហើយ"
- ខ្លឹមសារ៖ តំណចូលប្រើ ជំហានចាប់ផ្តើមរហ័ស តំណទៅកាន់ធនធានជំនួយ
- សំឡេងនៃការសរសេរ៖ រំភើប លើកទឹកចិត្ត និងពាក់ព័ន្ធនឹងហាត់ប្រាណ

### អ៊ីមែល 2: ជោគជ័យរហ័ស (ថ្ងៃទី 1) {#email-2-quick-wins-day-1}

- ប្រធានបទ៖ "អ្វី 3 យ៉ាងដែលត្រូវធ្វើមុនគេលើ FitSite របស់អ្នក"
- ខ្លឹមសារ៖ បន្ថែមស្លាកសញ្ញារបស់អ្នក ធ្វើបច្ចុប្បន្នភាពរូបភាព hero លើទំព័រដើម បន្ថែមកាលវិភាគថ្នាក់របស់អ្នក
- រួមបញ្ចូលរូបថតអេក្រង់ដែលបង្ហាញច្បាស់ថាត្រូវចុចកន្លែងណា

### អ៊ីមែល 3: ធ្វើឱ្យវាជារបស់អ្នក (ថ្ងៃទី 3) {#email-3-make-it-yours-day-3}

- ប្រធានបទ៖ "ធ្វើឱ្យគេហទំព័រហាត់ប្រាណរបស់អ្នកលេចធ្លោ"
- ខ្លឹមសារ៖ កែសម្រួលពណ៌ បន្ថែមរូបថតគ្រូបង្វឹក សរសេររឿងរ៉ាវស្ទូឌីយោរបស់អ្នក
- ភ្ជាប់ទៅកាន់ឧទាហរណ៍គេហទំព័រហាត់ប្រាណល្អៗនៅលើវេទិកា

### អ៊ីមែល 4: ដំណើរការជាសាធារណៈ (ថ្ងៃទី 7) {#email-4-go-live-day-7}

- ប្រធានបទ៖ "ត្រៀមចែករំលែក FitSite របស់អ្នកទៅកាន់ពិភពលោកហើយឬនៅ?"
- ខ្លឹមសារ៖ បញ្ជីត្រួតពិនិត្យអ្វីដែលត្រូវផ្ទៀងផ្ទាត់មុនចែករំលែក របៀបភ្ជាប់ domain ផ្ទាល់ខ្លួន (ប្រសិនបើនៅលើ Growth/Pro) គន្លឹះចែករំលែកតាមបណ្តាញសង្គម

:::tip ស្វ័យប្រវត្តិកម្មអ៊ីមែល
ប្រើ [Webhooks](/user-guide/integrations/webhooks) ឬ [Zapier](/user-guide/integrations/zapier) ដើម្បីបង្កើតសកម្មភាពផ្ញើអ៊ីមែលទាំងនេះតាមរយៈវេទិកាទីផ្សារអ៊ីមែលរបស់អ្នក។ វាផ្តល់ឱ្យអ្នកនូវការគ្រប់គ្រងកាន់តែច្រើនលើពេលវេលា និងអនុញ្ញាតឱ្យអ្នកតាមដានការចូលរួម។

## ធនធានជំនួយ {#help-resources}

បង្កើតមាតិកាជំនួយសម្រាប់ទីផ្សារឯកទេស ដែលឆ្លើយសំណួរដែលម្ចាស់ស្ទូឌីយោហាត់ប្រាណពិតជាសួរ៖

### អត្ថបទមូលដ្ឋានចំណេះដឹង {#knowledge-base-articles}

- "របៀបធ្វើបច្ចុប្បន្នភាពកាលវិភាគថ្នាក់របស់អ្នក"
- "ការបន្ថែម និងកែសម្រួលប្រវត្តិរូបគ្រូបង្វឹក"
- "ការផ្លាស់ប្តូរស្លាកសញ្ញា និងពណ៌ស្ទូឌីយោរបស់អ្នក"
- "ការភ្ជាប់ឈ្មោះ domain ផ្ទាល់ខ្លួនរបស់អ្នក" (សម្រាប់អតិថិជន Growth/Pro)
- "ការបន្ថែមវីជេតកក់ទៅក្នុងគេហទំព័ររបស់អ្នក"

សរសេរអត្ថបទទាំងនេះសម្រាប់អ្នកប្រើដែលមិនមានជំនាញបច្ចេកទេស។ ប្រើរូបថតអេក្រង់។ ជៀសវាងពាក្យបច្ចេកទេស WordPress។

### វីដេអូណែនាំជំហានៗ {#video-walkthroughs}

ការថតអេក្រង់ខ្លីៗ (2-3 នាទី) ដែលបង្ហាញ៖

- ការចូលប្រើលើកដំបូង និងការណែនាំទិសដៅ
- ការកែសម្រួលទំព័រដើម
- ការធ្វើបច្ចុប្បន្នភាពកាលវិភាគថ្នាក់
- ការបន្ថែមគ្រូបង្វឹកថ្មី

វីដេអូទាំងនេះមិនចាំបាច់ជាការផលិតដែលស្អាតឥតខ្ចោះទេ។ អ្វីដែលសំខាន់គឺច្បាស់ មានប្រយោជន៍ និងពាក់ព័ន្ធនឹងទីផ្សារឯកទេស។

- ផ្តល់ប្រវត្តិវិក្កយបត្រ និងការទាញយកវិក្កយបត្រ
- ភ្ជាប់ទៅកាន់ធនធានជំនួយ

## ការវាស់វែងជោគជ័យនៃការណែនាំចាប់ផ្តើមប្រើប្រាស់ {#measuring-onboarding-success}

តាមដានសូចនាករទាំងនេះ ដើម្បីដឹងថាការណែនាំចាប់ផ្តើមប្រើប្រាស់របស់អ្នកកំពុងដំណើរការឬអត់៖

- **អត្រាសកម្មភាពដំបូង**៖ តើភាគរយប៉ុន្មាននៃអ្នកចុះឈ្មោះពិតជាកែសម្រួលគេហទំព័ររបស់ពួកគេក្នុងសប្តាហ៍ដំបូង?
- **ការចូលប្រើក្នុងសប្តាហ៍ដំបូង**៖ តើអតិថិជនថ្មីចូលប្រើប៉ុន្មានដងក្នុងសប្តាហ៍ដំបូងរបស់ពួកគេ?
- **សំបុត្រគាំទ្រពីអតិថិជនថ្មី**៖ បរិមាណខ្ពស់មានន័យថាការណែនាំចាប់ផ្តើមប្រើប្រាស់របស់អ្នកមានចន្លោះខ្វះខាត
- **ការបម្លែងពីសាកល្បងទៅបង់ប្រាក់**៖ ប្រសិនបើអ្នកផ្តល់ការសាកល្បង តើមានភាគរយប៉ុន្មានដែលបម្លែង?
