# មេរៀនទី 7: ធ្វើឱ្យវាជារបស់អ្នក

អតិថិជនរបស់អ្នកមិនគួរមានអារម្មណ៍ថាពួកគេកំពុងប្រើ "កម្មវិធីបន្ថែម WordPress មួយណាមួយ" ឡើយ។ ពួកគេគួរមានអារម្មណ៍ថាកំពុងប្រើ FitSite -- វេទិកាដែលបានបង្កើតសម្រាប់ឧស្សាហកម្មរបស់ពួកគេ។ មេរៀននេះគ្របដណ្តប់លើការបង្កើតម៉ាក ការធ្វើ white-label និងការធ្វើឱ្យវេទិកាមានអារម្មណ៍ដូចជាផលិតផលមួយ។

## កន្លែងដែលយើងបានបញ្ចប់

FitSite មានលំហូរ checkout ដែលដំណើរការបាន ដែលនាំម្ចាស់ស្ទូឌីយោហាត់ប្រាណពីការជ្រើសរើសគម្រោងទៅកាន់គេហទំព័រដែលកំពុងដំណើរការ។ ឥឡូវនេះ យើងនឹងធ្វើឱ្យបទពិសោធន៍ទាំងមូលមានអារម្មណ៍ស្របគ្នា និងមានម៉ាកច្បាស់លាស់ដូចជាផលិតផលមួយ។

## ដែនវេទិការបស់អ្នក

គ្រឹះនៃម៉ាករបស់អ្នកគឺដែនរបស់អ្នក។ សម្រាប់ FitSite៖

- **ដែនចម្បង**: `fitsite.com` (គេហទំព័រទីផ្សាររបស់អ្នក និងឫសបណ្តាញ)
- **គេហទំព័រអតិថិជន**: `studioname.fitsite.com` (ដែនរង)
- **ដែនផ្ទាល់ខ្លួន**: អតិថិជននៅលើគម្រោង Growth និង Pro អាចភ្ជាប់ដែនរបស់ខ្លួនបាន

### ការដំឡើងដែនរបស់អ្នក

1. ចុះឈ្មោះដែនវេទិការបស់អ្នក
2. ចង្អុលវាទៅកាន់អ្នកផ្តល់សេវាបង្ហោះរបស់អ្នក
3. កំណត់រចនាសម្ព័ន្ធ wildcard DNS (`*.fitsite.com`) សម្រាប់ដែនរងរបស់អតិថិជន
4. ធានាថា wildcard SSL សកម្ម

មើល [របៀបកំណត់រចនាសម្ព័ន្ធ Domain Mapping](/user-guide/domain-mapping/how-to-configure-domain-mapping) សម្រាប់សេចក្តីណែនាំលម្អិត។

## ការធ្វើ White-Label បទពិសោធន៍ Admin

នៅពេលម្ចាស់ស្ទូឌីយោហាត់ប្រាណចូលទៅក្នុង Dashboard របស់គេហទំព័ររបស់ពួកគេ ពួកគេគួរឃើញម៉ាករបស់អ្នក មិនមែន WordPress ឬម៉ាក Ultimate Multisite ទេ។

### ទំព័រចូលផ្ទាល់ខ្លួន

កែសម្រួលទំព័រចូល WordPress ដើម្បីបង្ហាញ៖

- រូបសញ្ញា FitSite របស់អ្នក
- រូបភាពផ្ទៃក្រោយសមស្របនឹងការហាត់ប្រាណ
- ពណ៌ម៉ាករបស់អ្នក

### ម៉ាកលើ Dashboard

ប្រើ addon [Admin Page Creator](/addons/admin-page-creator) ឬ CSS ផ្ទាល់ខ្លួន ដើម្បី៖

- ជំនួសរូបសញ្ញា WordPress ដោយរូបសញ្ញា FitSite របស់អ្នក
- កែសម្រួលពណ៌ admin ឱ្យត្រូវនឹងម៉ាករបស់អ្នក
- បន្ថែម widget ផ្ទាល់ខ្លួនលើ Dashboard ដែលមានតំណរហ័ស និងធនធានជំនួយជាក់លាក់សម្រាប់ការហាត់ប្រាណ

### ទំព័រ Admin ផ្ទាល់ខ្លួន

ពិចារណាបង្កើតទំព័រ admin ផ្ទាល់ខ្លួន ដែលបង្ហាញសកម្មភាពសំខាន់បំផុតសម្រាប់ម្ចាស់ស្ទូឌីយោហាត់ប្រាណ៖

- "កែសម្រួលកាលវិភាគថ្នាក់របស់អ្នក"
- "ធ្វើបច្ចុប្បន្នភាពប្រវត្តិគ្រូបង្វឹក"
- "មើលគេហទំព័ររបស់អ្នក"

វាជួយកាត់បន្ថយការរៀនសូត្រដោយដាក់សកម្មភាពដែលពាក់ព័ន្ធនឹងវិស័យនេះនៅខាងមុខ និងចំកណ្តាល ជំនួសឱ្យការលាក់វានៅក្នុងម៉ឺនុយ WordPress ស្តង់ដារ។

## ការដាក់ម៉ាកលើការទំនាក់ទំនងរបស់អ្នក

រាល់អ៊ីមែល វិក្កយបត្រ និងការជូនដំណឹងគួរតែពង្រឹងម៉ាករបស់អ្នក។

### អ៊ីមែលប្រព័ន្ធ

ចូលទៅកាន់ **Ultimate Multisite > Settings > Emails** ហើយកែសម្រួលអ៊ីមែលប្រព័ន្ធទាំងអស់៖

- **ឈ្មោះអ្នកផ្ញើ**: FitSite
- **អ៊ីមែលអ្នកផ្ញើ**: hello@fitsite.com
- **គំរូអ៊ីមែល**: ប្រើពណ៌ម៉ាក និងរូបសញ្ញារបស់អ្នក
- **ភាសា**: ប្រើពាក្យពាក់ព័ន្ធនឹងការហាត់ប្រាណពេញមួយអ៊ីមែល

អ៊ីមែលសំខាន់ៗដែលត្រូវកែសម្រួល៖

| អ៊ីមែល | កំណែទូទៅ | កំណែ FitSite |
|-------|----------------|-----------------|
| ស្វាគមន៍ | "គេហទំព័រថ្មីរបស់អ្នករួចរាល់ហើយ" | "គេហទំព័រស្ទូឌីយោហាត់ប្រាណរបស់អ្នកកំពុងដំណើរការហើយ" |
| បង្កាន់ដៃបង់ប្រាក់ | "បានទទួលការបង់ប្រាក់" | "បានបញ្ជាក់ការបង់ប្រាក់សម្រាប់ការជាវ FitSite" |
| ការសាកល្បងជិតបញ្ចប់ | "ការសាកល្បងរបស់អ្នកជិតបញ្ចប់ហើយ" | "ការសាកល្បង FitSite របស់អ្នកនឹងបញ្ចប់ក្នុង 3 ថ្ងៃ -- រក្សាគេហទំព័រស្ទូឌីយោរបស់អ្នកឱ្យដំណើរការ" |

### វិក្កយបត្រ

កែសម្រួលគំរូវិក្កយបត្រជាមួយ៖

- រូបសញ្ញា FitSite និងពណ៌ម៉ាករបស់អ្នក
- ព័ត៌មានអាជីវកម្មរបស់អ្នក
- ឈ្មោះផលិតផលជាក់លាក់សម្រាប់ការហាត់ប្រាណ (មិនមែនលេខសម្គាល់គម្រោងទូទៅទេ)

## គេហទំព័រដែលអតិថិជនមើលឃើញ

ដែនចម្បងរបស់អ្នក (`fitsite.com`) ត្រូវការគេហទំព័រទីផ្សារដែលលក់វេទិកានេះ។ វាដាច់ដោយឡែកពី admin បណ្តាញ Ultimate Multisite -- វាជាមុខមាត់សាធារណៈនៃអាជីវកម្មរបស់អ្នក។

ទំព័រសំខាន់ៗ៖

- **ទំព័រដើម**: សំណើតម្លៃច្បាស់លាស់សម្រាប់អាជីវកម្មហាត់ប្រាណ
- **លក្ខណៈពិសេស**: អ្វីដែល FitSite ធ្វើ ដោយប្រើពាក្យសម្រាប់ការហាត់ប្រាណ
- **តម្លៃ**: គម្រោងបីរបស់អ្នក ជាមួយការប្រៀបធៀបលក្ខណៈពិសេសជាក់លាក់តាមវិស័យ
- **ឧទាហរណ៍**: បង្ហាញគេហទំព័រដែលបានបង្កើតលើវេទិកានេះ
- **ចុះឈ្មោះ**: តំណទៅកាន់ទម្រង់ checkout របស់អ្នក

:::tip គេហទំព័រទីផ្សាររបស់អ្នកអាចជាគេហទំព័របណ្តាញមួយ
បង្កើតគេហទំព័រទីផ្សាររបស់អ្នកជាគេហទំព័រមួយនៅក្នុងបណ្តាញផ្ទាល់ខ្លួនរបស់អ្នក។ វាអនុញ្ញាតឱ្យអ្នកគ្រប់គ្រងវាពី Dashboard ដូចគ្នា ហើយបង្ហាញសមត្ថភាពរបស់វេទិការបស់អ្នកផ្ទាល់។
:::

## ដែនផ្ទាល់ខ្លួនសម្រាប់អតិថិជន

សម្រាប់អតិថិជននៅលើគម្រោងដែលរួមបញ្ចូលដែនផ្ទាល់ខ្លួន សូមរៀបរាប់ដំណើរការឱ្យច្បាស់៖

1. អតិថិជនចុះឈ្មោះ ឬផ្ទេរដែនរបស់ពួកគេទៅកាន់អ្នកចុះបញ្ជីដែន
2. អតិថិជនធ្វើបច្ចុប្បន្នភាព DNS ដើម្បីចង្អុលទៅកាន់វេទិការបស់អ្នក (ផ្តល់កំណត់ត្រាច្បាស់លាស់)
3. Ultimate Multisite គ្រប់គ្រង domain mapping និង SSL

បង្កើតអត្ថបទជំនួយ ឬធាតុមូលដ្ឋានចំណេះដឹងជាក់លាក់សម្រាប់ដំណើរការនេះ ដែលសរសេរសម្រាប់ម្ចាស់ស្ទូឌីយោហាត់ប្រាណដែលមិនមែនជាអ្នកបច្ចេកទេស។

## បណ្តាញ FitSite រហូតមកដល់ពេលនេះ

```
បណ្តាញ FitSite
├── WordPress Multisite (របៀបដែនរង)
├── Ultimate Multisite (បានកំណត់រចនាសម្ព័ន្ធ + មានម៉ាក)
├── ដែនវេទិកា (fitsite.com + wildcard SSL)
├── គំរូគេហទំព័រ (Studio Essential, Gym Pro, Fitness Chain)

├── ការដាក់ម៉ាក
│   ├── ទំព័រចូលផ្ទាល់ខ្លួន
│   ├── Dashboard admin ដែលមានម៉ាក
│   ├── អ៊ីមែលប្រព័ន្ធជាក់លាក់តាមវិស័យ
│   ├── វិក្កយបត្រដែលមានម៉ាក
│   └── គេហទំព័រទីផ្សារលើ fitsite.com
└── រួចរាល់សម្រាប់លំហូរ onboarding (មេរៀនបន្ទាប់)
```

## អ្វីដែលយើងបានបង្កើតក្នុងមេរៀននេះ

- **ដែនវេទិកា និង DNS** បានកំណត់រចនាសម្ព័ន្ធសម្រាប់បទពិសោធន៍ដែលមានម៉ាក
- **admin ដែលបានធ្វើ white-label** ជាមួយម៉ាក FitSite ពេញមួយបទពិសោធន៍
- **ការទំនាក់ទំនងដែលបានកែសម្រួល** -- អ៊ីមែល វិក្កយបត្រ និងការជូនដំណឹងទាំងអស់ស្របនឹងម៉ាក
- **គេហទំព័រទីផ្សារមួយ** ដែលលក់ FitSite ទៅកាន់ម្ចាស់ស្ទូឌីយោហាត់ប្រាណ
- **ឯកសារដែនផ្ទាល់ខ្លួន** សម្រាប់អតិថិជនដែលចង់ប្រើដែនផ្ទាល់ខ្លួន
