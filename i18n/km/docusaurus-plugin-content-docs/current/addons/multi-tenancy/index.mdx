# Multi-Tenancy

бһ…бҹӮбһҖбһ…бһ¶бһҷбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҡбһ„бһ‘бҹ…бһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ бһ“бһ·бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ•бҹ’бһҹбҹҒбһ„бҹ— бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһ“бһӣбҹ’бһ”бһҝбһ“ бһ“бһ·бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–

## бһ…бҹҶбһҺбһ»бһ…бһҹбҹҶбһҒбһ¶бһ“бҹӢбһ“бҹғбһҖбһ¶бһҡбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһҖбҹҶбһҺбҹӮ 1.2.0 {#version-120-release-highlights}

Multi-Tenancy 1.2.0 бһ”бһ“бҹ’бһҗбҹӮбһҳбһӮбҹ’бһҡбһ№бҹҮ tenant бһҜбһҖбһҡбһ¶бһҮбҹ’бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӣбҹҶбһ бһјбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбҹ–

- бһҖбһ¶бһҡбһ…бһјбһӣбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· tenant бһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ…бһјбһӣбһҳбһҫбһӣ SSO бһҡбһҷбҹҲбһ–бҹҒбһӣбһҒбҹ’бһӣбһё бһ“бһ·бһ„бһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһ¶бһҡбһ…бһ¶бһҖбҹӢбһЎбһҫбһ„бһңбһ·бһүбҹ”
- бһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҜбһҖбһҹбһ¶бһҡбһҸбһ¶бһҳбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҡбһ„ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ tenant бһҜбһҖбһҡбһ¶бһҮбҹ’бһҷбҹ”
- бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– multi-tenancy бһҡбһ”бһҹбҹӢ Hostinger бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶ tenant бһҠбҹӮбһӣбһҠбһ№бһ„бһ–бһё hostбҹ”
- бһ…бҹ’бһҡбһҖбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһёбһҜбһҖбһҡбһ¶бһҮбҹ’бһҷ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бһ¶бһҹбҹӢ бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ push бһўбһҹбһҳбһҖбһ¶бһӣбҹ”
- бһҖбһ¶бһҡбһ…бһ„бһ—бҹ’бһҮбһ¶бһ”бҹӢ host бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҸбҹӮбһҳбһҪбһҷбһҠбҹӮбһӣбһўбһ¶бһ…бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ”бһ¶бһ“ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ Bedrock, FrankenPHP бһ“бһ·бһ„ localhostбҹ”
- бһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһў bootstrap tenant, бһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ, bootstrap schema бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбҹ”
- бһҖбһ¶бһҡбһҡбһ»бҹҮбһҡбһҫ tenant бһҜбһҖбһҡбһ¶бһҮбҹ’бһҷбһұбҹ’бһҷбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҮбһ¶бһ„бһҳбһ»бһ“ бһҠбҹӮбһӣбһҹбһҳбҹ’бһўбһ¶бһҸбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҡбһ”бһҹбҹӢ tenantбҹ”

бһҳбһҫбһӣбһҹбҹҖбһңбһ—бҹ…бһҺбҹӮбһ“бһ¶бҹҶбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ [бһӣбҹҶбһ бһјбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһёбһҜбһҖбһҡбһ¶бһҮбҹ’бһҷ](/user-guide/getting-started/sovereign-migration-workflow), [бһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖ multi-tenancy](/user-guide/configuration/multi-tenancy-isolation) бһ“бһ·бһ„ [бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ tenant бһҜбһҖбһҡбһ¶бһҮбҹ’бһҷ](/user-guide/administration/sovereign-tenant-operations)бҹ”

## бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡ {#requirements}

- WordPress 5.3 бһ¬бһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„
- PHP 7.4 бһ¬бһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„
- plugin Multisite Ultimate (бһҹбһҖбһҳбҹ’бһҳ)

## бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ {#installation}

1. бһ•бҹ’бһ‘бһ»бһҖбһЎбһҫбһ„бһҜбһҖбһҹбһ¶бһҡ addon бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҗбһҸ `/wp-content/plugins/` бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
2. бһ’бҹ’бһңбһҫбһұбҹ’бһҷ plugin бһҹбһҖбһҳбҹ’бһҳбһҸбһ¶бһҳбһҳбҹүбһәбһ“бһ»бһҷ 'Plugins' бһҖбҹ’бһ“бһ»бһ„ WordPress
3. бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ addon бһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ“бҹӮбһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһҺбҹ’бһҸбһ¶бһү

### бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸ {#testing}

addon бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҖбҹ’бһҡбһ”бһҒбһҺбҹ’бһҢбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһ–бҹҒбһүбһӣбҹҒбһүбҹ–

- **бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһҜбһҖбһҸбһ¶**бҹ– бһҸбҹҒбһҹбҹ’бһҸбһ•бҹ’бһўбҹӮбһҖбһӣбһҫ PHPUnit бһҮбһ¶бһҳбһҪбһҷбһҲбһ»бһҸбһҸбҹҒбһҹбҹ’бһҸ WordPress
- **бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҸбҹҒбһҹбҹ’бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“**бҹ– бһ•бҹ’бһҸбһӣбҹӢбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҮбҹҶбһ“бһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҒбһҺбһ¶бһҡбһёбһҷбҹүбһјбһҸбҹҒбһҹбҹ’бһҸбһ‘бһјбһ‘бҹ…
- **бһңбһ·бһҹбһ¶бһӣбһ—бһ¶бһ–бһҸбҹҒбһҹбҹ’бһҸ**бҹ– бһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҠбһҫбһҳбҹ’бһ”бһёбһҸбһ¶бһҳбһҠбһ¶бһ“бһңбһ·бһҹбһ¶бһӣбһ—бһ¶бһ–бһҖбһјбһҠ
- **бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸ Multisite**бҹ– бһҸбҹҒбһҹбҹ’бһҸбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“ multisite

### бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҖбһјбһҠ {#code-standards}

- **бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠ WordPress**бҹ– бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳбһҡбһҷбҹҲ PHPCS
- **бһ—бһ¶бһ–бһҶбһ”бһӮбҹ’бһ“бһ¶бһҮбһ¶бһҳбһҪбһҷ PHP 7.4+**бҹ– бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһӣбһҖбҹ’бһҒбһҺбҹҲ PHP бһ‘бҹҶбһ“бһҫбһ”
- **бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһҹбҹ’бһҗбһ·бһҸбһ·**бҹ– бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ PHPStan бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бҹ’бһҡбһ—бҹҒбһ‘
- **бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҖбһјбһҠбһ‘бҹҶбһ“бһҫбһ”**бҹ– бһ…бҹ’бһ”бһ¶бһ”бҹӢ Rector бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ¶бһ”бҹӢбһӮбҹ’бһҡбҹҒбһҠ PHP

### бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бһ„бҹ’бһҖбһҫбһҸ {#build-system}

- **бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ’бһ“бһ’бһ¶бһ“**бҹ– бһ”бһ„бҹ’бһҡбһҪбһҳбһҜбһҖбһҹбһ¶бһҡ CSS/JS
- **бһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮ**бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡ POT
- **бһҖбһ¶бһҡбһңбҹҒбһ…бһҒбҹ’бһ…бһ”бҹӢ**бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡбһ”бһҺбҹ’бһҺбһҹбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бҹӮбһҖбһ…бһ¶бһҷ
- **бһҡбһ”бҹҖбһ”бһўбһ—бһ·бһңбһҢбҹ’бһҚ**бҹ– бһ’бһ“бһ’бһ¶бһ“бһҳбһ·бһ“бһ”бһ¶бһ“бһ”бһ„бҹ’бһҡбһҪбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҖбҹҶбһ бһ»бһҹ

## бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ {#configuration}

addon бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҮбһҳбҹ’бһҡбһҫбһҹбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҠбҹӮбһӣбһўбһ¶бһ…бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһҸбһ¶бһҳбһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһҺбҹ’бһҸбһ¶бһүбҹ”

### бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ tenant бһҜбһҖбһҡбһ¶бһҮбҹ’бһҷ {#sovereign-tenant-configuration}

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ tenant бһҜбһҖбһҡбһ¶бһҮбҹ’бһҷ бһҹбһјбһҳбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ tenant, root бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҜбһҖбһҹбһ¶бһҡ, бһҖбһ¶бһҡбһ…бһ„бһ—бҹ’бһҮбһ¶бһ”бҹӢ host бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– host бһҳбһ»бһ“бһ–бҹҒбһӣбһ”бһүбҹ’бһҮбһјбһ“бһ…бһҡбһ¶бһ…бһҡбһ‘бҹ…бһҖбһ¶бһ“бҹӢ tenantбҹ” бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһёбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҡбһ¶бһӣбҹӢбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҠбҹӮбһӣбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһҠбһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ, бһҖбһ¶бһҡбһ…бһ„бһ—бҹ’бһҮбһ¶бһ”бҹӢ host, бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһјбһ“бһ•бҹ’бһӣбһјбһң бһ¬бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбҹ”

бһ”бҹ’бһҡбһҫ `localhost` бһҸбҹӮбһ“бҹ…бһ–бҹҒбһӣ runtime бһҡбһ”бһҹбҹӢ tenant бһ“бһ·бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбҹҶбһ–бһ№бһ„бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһҸбһ¶бһҳ socket бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ¬бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҸбҹӮбһҳбһҪбһҷбҹ” бһ”бҹ’бһҡбһҫ `127.0.0.1` бһ¬бһҲбҹ’бһҳбҹ„бҹҮ host бһҹбҹҒбһңбһ¶ бһ“бҹ…бһ–бҹҒбһӣбһ“бҹ„бҹҮбһҮбһ¶бһўбҹ’бһ“бһҖ host бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бҹ”

### бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– Hostinger {#hostinger-capability}

бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– multi-tenancy бһҡбһ”бһҹбҹӢ Hostinger бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ token бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Hostinger бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҡбһҪбһҳбҹ” бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶ domain бһҳбһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ hPanel, DNS бһ…бһ„бҹ’бһўбһ»бһӣбһ‘бҹ… Account Hostinger бһ бһҫбһҷбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ tenant бһҸбҹ’бһҡбһјбһңбһӮбҹ’бһ“бһ¶бһ“бһ№бһ„ hPanel бһҳбһ»бһ“бһ–бҹҒбһӣбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбҹ”
