# PayFast

Multisite Ultimate: Payfast Gateway

បន្ថែម Payfast - gateway ទូទាត់ប្រាក់តាមអ៊ីនធឺណិតដែលលឿន និងមានសុវត្ថិភាពនៅអាហ្វ្រិកខាងត្បូង - ជាជម្រើសទូទាត់ប្រាក់របស់ Multisite Ultimate។

## ការដំឡើង

1. Upload 'ultimate-multisite-payfast' ទៅថត '/wp-content/plugins/'
2. បើកដំណើរការ plugin តាមរយៈម៉ឺនុយ 'Plugins' ក្នុង WordPress
3. អនុវត្តតាម Wizard មួយជំហានម្តងៗ ដើម្បីរៀបចំ plugin

## មគ្គុទ្ទេសក៍ប្រើប្រាស់

PayFast គឺជាអ្នកដំណើរការទូទាត់ប្រាក់តាមអ៊ីនធឺណិតឈានមុខ សម្រាប់អាជីវកម្មនៅអាហ្វ្រិកខាងត្បូង។ វាផ្តោតលើការជួយពាណិជ្ជករអនឡាញឱ្យរីកចម្រើន ដោយធ្វើឱ្យប្រតិបត្តិការរលូន និងមានសុវត្ថិភាព។ តាមសំណើរបស់អ្នកប្រើ Ultimate Multisite យើងបានសម្រេចបង្កើត addon ដែលអាចធ្វើឱ្យការរួមបញ្ចូលជាមួយ PayFast ប្រព្រឹត្តទៅបាន។ អត្ថបទនេះនឹងបង្ហាញពីរបៀបរៀបចំ PayFast ជា payment gateway របស់អ្នក ដើម្បីទទួលការទូទាត់ប្រាក់ក្នុងបណ្តាញ Ultimate Multisite របស់អ្នក។

ជំហានដំបូងគឺដំឡើង add on។

# ការរៀបចំ PayFast

បន្ទាប់ពីដំឡើង add-on រួច អ្នកអាចចូលទៅកាន់ការកំណត់ Ultimate Multisite ហើយចាប់ផ្តើមកំណត់រចនាសម្ព័ន្ធ PayFast។

ក្រោមទំព័រ **ការកំណត់ Ultimate Multisite** អ្នកអាចរកឃើញផ្ទាំង "Payments"។ នៅក្រោមផ្ទាំងនេះ អ្នកនឹងឃើញ "**Payfast** " ជាមួយប៊ូតុងបិទបើក ដើម្បីបើក ឬបិទ payment gateway នេះ។

![ការកំណត់ payment gateway](/img/config/settings-payment-gateways.png)បន្តទៅមុខ ហើយចុចប៊ូតុងបិទបើក ដើម្បីបើកវា។ បន្ទាប់មកវានឹងពង្រីកប្រអប់ ដើម្បីបង្ហាញវាលផ្សេងៗដែលអ្នកត្រូវបំពេញ ដូចជា **Mechant ID, Merchant Key និង PassPhrase.** អ្នកអាចយកព័ត៌មាននេះពី Account **Payfast** របស់អ្នក

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

ជំហានបន្ទាប់គឺចូលទៅ Account PayFast របស់អ្នក ដើម្បីយកព័ត៌មានដែលអ្នកត្រូវការ។

ពេលចូលបានហើយ អ្នកអាចឃើញ credentials ដែលអ្នកត្រូវការភ្លាមៗពី dashboard Account **Payfast** របស់អ្នក។

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->អ្នកអាចចម្លង និងបិទភ្ជាប់ credential ហើយបំពេញព័ត៌មានលម្អិតក្នុងការកំណត់ Ultimate Multisite របស់អ្នក។ បន្ទាប់មកចុចប៊ូតុង **"រក្សាទុកការកំណត់** " ដើម្បីរក្សាទុកការផ្លាស់ប្តូរ។
