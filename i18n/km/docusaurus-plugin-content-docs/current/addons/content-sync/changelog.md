# កំណត់ត្រាការផ្លាស់ប្តូរ Content Sync {#content-sync-changelog}

### កំណែ 1.1.0 - ចេញផ្សាយនៅ 2025-12-31 {#version-110---released-on-2025-12-31}
* ថ្មី៖ គាំទ្រការធ្វើសមកាលកម្មប្រភេទប្រកាសទាំងអស់ (ប្រកាស, ទំព័រ, ប្រភេទប្រកាសផ្ទាល់ខ្លួន)
* ថ្មី៖ ការចម្រោះប្រភេទប្រកាសនៅជំហានជ្រើសរើសមាតិកា
* ថ្មី៖ ជំហានជ្រើសរើសគេហទំព័រប្រភព - ធ្វើសមកាលកម្មពីគេហទំព័រណាមួយ មិនមែនតែគេហទំព័រចម្បងទេ
* ថ្មី៖ ការជ្រើសគេហទំព័រកូនជាមុនសម្រាប់គេហទំព័រគំរូ
* ថ្មី៖ ការស្រង់មេឌៀពី post_content សម្រាប់ប្រភេទប្រកាសទាំងអស់
* ថ្មី៖ ការធ្វើសមកាលកម្មចំណាត់ថ្នាក់ (ប្រភេទ, ស្លាក, ចំណាត់ថ្នាក់ផ្ទាល់ខ្លួន)
* បានផ្លាស់ប្តូរ៖ ប្តូរឈ្មោះពី "Elementor Template Sync" ទៅ "Content Sync"
* បានផ្លាស់ប្តូរ៖ ជំហាន "ជ្រើសរើសគំរូ" ត្រូវបានប្តូរឈ្មោះទៅជា "ជ្រើសរើសទិន្នន័យ"
* បានកែលម្អ៖ កូដជាក់លាក់សម្រាប់ Elementor ឥឡូវនេះដំណើរការតែសម្រាប់ប្រភេទប្រកាស elementor_library ប៉ុណ្ណោះ

### កំណែ 1.0.0 - ចេញផ្សាយនៅ 2025-12-19 {#version-100---released-on-2025-12-19}
* ការចេញផ្សាយដំបូង
* អ្នកណែនាំច្រើនជំហានសម្រាប់ជ្រើសរើសគំរូ និងគេហទំព័រ
* ដំណើរការជាកញ្ចប់ជាមួយមតិកែលម្អវឌ្ឍនភាពពេលវេលាពិត
* ធ្វើសមកាលកម្មមេឌៀពេញលេញជាមួយការផ្គូផ្គង ID ឡើងវិញដោយស្វ័យប្រវត្តិ
* ការរួមបញ្ចូលការកំណត់ជាមួយ Ultimate Multisite
* គាំទ្រការដកគេហទំព័រចេញតាមការកំណត់បណ្តាញ
* សម្អាតឃ្លាំងសម្ងាត់ Elementor បន្ទាប់ពីធ្វើសមកាលកម្ម
* រក្សាទុកចំណាត់ថ្នាក់គំរូ
