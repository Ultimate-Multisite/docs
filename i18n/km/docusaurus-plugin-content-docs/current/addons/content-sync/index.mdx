# ការធ្វើសមកាលកម្មមាតិកា

Multisite Ultimate: ការធ្វើសមកាលកម្មមាតិកា

addon នេះសម្រាប់ Multisite Ultimate អនុញ្ញាតឱ្យអ្នកគ្រប់គ្រងបណ្តាញធ្វើសមកាលកម្មមាតិកាពី site ណាមួយទៅកាន់ site ផ្សេងៗក្នុងបណ្តាញ។ វាគាំទ្រប្រភេទ post ទាំងអស់ រួមទាំង post, page, custom post type និង Elementor template។ addon នេះរួមបញ្ចូលផ្ទាំង wizard ជាច្រើនជំហានសម្រាប់ជ្រើសមាតិកា និង site គោលដៅ ដោយមានការដំណើរការជាក្រុម និងមតិត្រឡប់វឌ្ឍនភាពពេលវេលាពិត។

## លក្ខណៈពិសេស {#features}

* ការជ្រើសមាតិកា៖ ក្រឡាកាតដែលមើលឃើញជាមួយរូបភាពតូច និងការច្រោះតាមប្រភេទ post
* ប្រភេទ Post ទាំងអស់៖ ធ្វើសមកាលកម្ម post, page, product, Elementor template និង custom post type ណាមួយ
* ការជ្រើស Site៖ ជ្រើស site ដែលត្រូវធ្វើសមកាលកម្មមាតិកាទៅ ដោយដក site ប្រភពចេញ និងជ្រើស site កូនជាមុន
* ការដំណើរការជាក្រុម៖ ធ្វើសមកាលកម្មទៅ site ចំនួន ១០ ក្នុងមួយលើក ជាមួយបច្ចុប្បន្នភាពវឌ្ឍនភាព
* ការធ្វើសមកាលកម្មមេឌៀ៖ ចម្លង និងកំណត់ផែនទីឯកសារមេឌៀឡើងវិញដោយស្វ័យប្រវត្តិ (រូបភាព វីដេអូ) ពីមាតិកា post ទៅកាន់ site គោលដៅ
* ការធ្វើសមកាលកម្ម Taxonomy៖ រក្សា category, tag និង custom taxonomy
* ការគាំទ្រ Elementor៖ គាំទ្រពេញលេញសម្រាប់ Elementor template ជាមួយការបង្កើត CSS ឡើងវិញ និងការកំណត់ផែនទី template ID ឡើងវិញ
* ការដោះស្រាយជម្លោះ៖ ជម្រើសដើម្បីជំនួសមាតិកាដែលមានស្រាប់ ឬរំលងវា
* ការគ្រប់គ្រង Cache៖ សម្អាត Elementor cache ដោយស្វ័យប្រវត្តិនៅលើ site គោលដៅបន្ទាប់ពីធ្វើសមកាលកម្ម

## តម្រូវការ {#requirements}

* WordPress 5.3 ឬខ្ពស់ជាងនេះ
* PHP 7.4 ឬខ្ពស់ជាងនេះ
* Ultimate Multisite plugin (សកម្ម)
* Elementor plugin (ជាជម្រើស ត្រូវការសម្រាប់ការធ្វើសមកាលកម្ម Elementor template)

## ការដំឡើង {#installation}

1. ផ្ទុកឡើង 'ultimate-multisite-content-sync' ទៅកាន់ថត '/wp-content/plugins/'
2. ធ្វើឱ្យ plugin សកម្មតាមរយៈម៉ឺនុយ 'Plugins' ក្នុង WordPress Network Admin របស់អ្នក
3. កំណត់រចនាសម្ព័ន្ធការកំណត់ក្នុង Ultimate Multisite → Settings → Content Sync
4. ចូលទៅកាន់ Ultimate Multisite → Content Sync ដើម្បីចាប់ផ្តើមធ្វើសមកាលកម្ម
