# កំណត់ត្រា​ផ្លាស់ប្តូរ Loco Translate

### 1.0.1
* កែលម្អ​ភាព​ឆបគ្នា​ជាមួយ​កំណែ Loco Translate ចុងក្រោយ
* បង្កើន​ការគ្រប់គ្រង​ការចូលប្រើ ដើម្បី​សុវត្ថិភាព​កាន់តែប្រសើរ
* បង្កើន​ប្រសិទ្ធភាព​ដំណើរការ

* ថត​បកប្រែ​សម្រាប់​តំបន់បណ្ដាញ​នីមួយៗ
* ប្រព័ន្ធ​អាទិភាព​បកប្រែ
* ការអនុវត្ត​ការគ្រប់គ្រង​ការចូលប្រើ
* ការរួមបញ្ចូល UI យ៉ាងរលូន​ជាមួយ Loco Translate
