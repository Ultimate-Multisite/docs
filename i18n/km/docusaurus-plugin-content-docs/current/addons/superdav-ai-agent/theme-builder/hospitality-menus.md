# ម៉ឺនុយសម្រាប់អាជីវកម្មបដិសណ្ឋារកិច្ច {#hospitality-menus}

មុខងារ **ម៉ឺនុយសម្រាប់អាជីវកម្មបដិសណ្ឋារកិច្ច** អនុញ្ញាតឱ្យ Theme Builder បង្កើត និងបង្កប់ទំព័រម៉ឺនុយអាហារ និងភេសជ្ជៈដែលមានរចនាសម្ព័ន្ធ ដោយផ្ទាល់ទៅក្នុង WordPress របស់អ្នក។

## ទិដ្ឋភាពទូទៅ {#overview}

Theme Builder ឥឡូវអាចបង្កើតទំព័រម៉ឺនុយមានរចនាសម្ព័ន្ធ និងមានលក្ខណៈវិជ្ជាជីវៈ សម្រាប់អាជីវកម្មបដិសណ្ឋារកិច្ច រួមមានភោជនីយដ្ឋាន ហាងកាហ្វេ បារ និងសេវាកម្មផ្គត់ផ្គង់ម្ហូបអាហារសម្រាប់ពិធី។ ម៉ឺនុយទាំងនេះត្រូវបានបញ្ចូលជាមួយរចនាគេហទំព័ររបស់អ្នកយ៉ាងពេញលេញ ហើយអាចធ្វើបច្ចុប្បន្នភាព និងគ្រប់គ្រងបានយ៉ាងងាយស្រួល។

## ប្រភេទអាជីវកម្មបដិសណ្ឋារកិច្ចដែលគាំទ្រ {#supported-hospitality-types}

- **ភោជនីយដ្ឋាន** — ម៉ឺនុយអាហារសេវាកម្មពេញលេញ
- **ហាងកាហ្វេ** — ម៉ឺនុយកាហ្វេ និងអាហារស្រាលៗ
- **បារ និងកន្លែងអង្គុយកម្សាន្ត** — ម៉ឺនុយភេសជ្ជៈ និងអាហារសម្រន់
- **ហាងនំដុត** — ម៉ឺនុយនំប៉ាស្ត្រី និងនំប៉័ង
- **សេវាកម្មផ្គត់ផ្គង់ម្ហូបអាហារសម្រាប់ពិធី** — ជម្រើសម៉ឺនុយសម្រាប់ព្រឹត្តិការណ៍
- **រថយន្តលក់អាហារ** — ម៉ឺនុយសេវាអាហារចល័ត
- **រោងចក្រផលិតស្រាបៀរ និងស្រាទំពាំងបាយជូរ** — ម៉ឺនុយភេសជ្ជៈជាមួយការពិពណ៌នា

## រចនាសម្ព័ន្ធម៉ឺនុយ {#menu-structure}

### ប្រភេទម៉ឺនុយ {#menu-categories}

ម៉ឺនុយត្រូវបានរៀបចំជាប្រភេទ៖

- **អាហារសម្រន់មុនម្ហូបចម្បង** — អាហារចាប់ផ្តើម និងចានតូចៗ
- **ម្ហូបចម្បង** — មុខម្ហូបសំខាន់ៗ
- **ម្ហូបបន្ថែម** — ម្ហូបអម និងបន្លែ
- **បង្អែម** — អាហារផ្អែម
- **ភេសជ្ជៈ** — ភេសជ្ជៈ (មានអាល់កុល និងគ្មានអាល់កុល)
- **ម្ហូបពិសេស** — ម្ហូបពិសេសប្រចាំថ្ងៃ ឬតាមរដូវកាល

### ទម្រង់មុខម្ហូប {#menu-item-format}

មុខម្ហូបនីមួយៗរួមមាន៖

```json
{
  "name": "Grilled Salmon",
  "description": "Fresh Atlantic salmon with lemon butter sauce",
  "price": "$24.95",
  "dietary_info": ["Gluten-Free", "High-Protein"],
  "allergens": ["Fish"],
  "availability": "Daily"
}
```

#### វាលមុខម្ហូប {#item-fields}

| វាល | ប្រភេទ | ការពិពណ៌នា |
|-------|------|-------------|
| `name` | string | ឈ្មោះមុខម្ហូប ឬភេសជ្ជៈ |
| `description` | string | ការពិពណ៌នាលម្អិតអំពីមុខនេះ |
| `price` | string | តម្លៃ (បានធ្វើទ្រង់ទ្រាយជាមួយរូបិយប័ណ្ណ) |
| `dietary_info` | array | លក្ខណៈសម្បត្តិអាហារូបត្ថម្ភ (Vegan, Gluten-Free ជាដើម) |
| `allergens` | array | សារធាតុបង្កអាលែហ្ស៊ីទូទៅ (Nuts, Shellfish ជាដើម) |
| `availability` | string | ពេលដែលមានលក់ (ប្រចាំថ្ងៃ តាមរដូវកាល ជាដើម) |

## ការបង្កើតម៉ឺនុយសម្រាប់អាជីវកម្មបដិសណ្ឋារកិច្ច {#creating-hospitality-menus}

### ជំហានទី 1៖ ផ្តល់ព័ត៌មានម៉ឺនុយ {#step-1-provide-menu-information}

ប្រាប់ Theme Builder អំពីម៉ឺនុយរបស់អ្នក៖

```
បង្កើតម៉ឺនុយភោជនីយដ្ឋានសម្រាប់ភោជនីយដ្ឋានអ៊ីតាលីរបស់ខ្ញុំ ដែលមានអាហារសម្រន់មុនម្ហូបចម្បង
ម្ហូបប៉ាស្តា ម្ហូបចម្បង និងបង្អែម។ សូមបញ្ចូលតម្លៃ និងការពិពណ៌នា។
```

### ជំហានទី 2៖ Theme Builder បង្កើតម៉ឺនុយ {#step-2-theme-builder-generates-menu}

Theme Builder៖

1. បង្កើតទំព័រម៉ឺនុយមានរចនាសម្ព័ន្ធ
2. រចនាឱ្យស៊ីគ្នាជាមួយ theme របស់គេហទំព័រអ្នក
3. រៀបចំមុខម្ហូបជាប្រភេទ
4. ធ្វើទ្រង់ទ្រាយតម្លៃ និងការពិពណ៌នា
5. បន្ថែមព័ត៌មានអាហារូបត្ថម្ភ និងសារធាតុបង្កអាលែហ្ស៊ី

### ជំហានទី 3៖ ពិនិត្យ និងកែតម្រូវ {#step-3-review-and-customize}

អ្នកអាច៖

1. កែសម្រួលមុខម្ហូប និងតម្លៃ
2. បន្ថែម ឬដកប្រភេទចេញ
3. រៀបចំលំដាប់មុខម្ហូបឡើងវិញក្នុងប្រភេទ
4. ធ្វើបច្ចុប្បន្នភាពការពិពណ៌នា និងព័ត៌មានអាហារូបត្ថម្ភ
5. កែតម្រូវរចនាប័ទ្ម និងប្លង់

## ជម្រើសបង្ហាញម៉ឺនុយ {#menu-display-options}

### ទំព័រម៉ឺនុយពេញលេញ {#full-menu-page}

ទំព័រដាច់ដោយឡែកសម្រាប់បង្ហាញម៉ឺនុយពេញលេញរបស់អ្នក៖

- រៀបចំតាមប្រភេទ
- អាចស្វែងរក និងតម្រងបាន
- ប្លង់សមស្របសម្រាប់បោះពុម្ព
- រចនាឆ្លើយតបល្អលើឧបករណ៍ចល័ត

### វីជិតម៉ឺនុយ {#menu-widget}

បង្កប់ផ្នែកម៉ឺនុយនៅលើទំព័រផ្សេងទៀត៖

- មុខម្ហូបណែនាំនៅលើទំព័រដើម
- ម្ហូបពិសេសប្រចាំថ្ងៃនៅលើរបារចំហៀង
- ម៉ឺនុយភេសជ្ជៈនៅលើទំព័របារ
- ការបង្ហាញបង្អែមនៅបាតទំព័រ

### ម៉ឺនុយ PDF {#menu-pdf}

បង្កើតម៉ឺនុយ PDF ដែលអាចទាញយកបាន៖

- ទ្រង់ទ្រាយមានលក្ខណៈវិជ្ជាជីវៈ
- គុណភាពរួចរាល់សម្រាប់បោះពុម្ព
- រួមបញ្ចូលរូបភាព និងការពិពណ៌នា
- ងាយស្រួលចែករំលែក និងផ្ញើតាមអ៊ីមែល

## ព័ត៌មានអាហារូបត្ថម្ភ និងសារធាតុបង្កអាលែហ្ស៊ី {#dietary-and-allergen-information}

### លក្ខណៈសម្បត្តិអាហារូបត្ថម្ភ {#dietary-attributes}

សម្គាល់មុខម្ហូបដោយព័ត៌មានអាហារូបត្ថម្ភ៖

- **Vegan** — គ្មានផលិតផលពីសត្វ
- **Vegetarian** — គ្មានសាច់
- **Gluten-Free** — សុវត្ថិភាពសម្រាប់ជំងឺ celiac
- **Dairy-Free** — គ្មានផលិតផលទឹកដោះគោ
- **Nut-Free** — គ្មានគ្រាប់ធញ្ញជាតិពីដើមឈើ ឬសណ្តែកដី
- **Low-Carb** — កាត់បន្ថយកាបូអ៊ីដ្រាត
- **High-Protein** — សម្បូរប្រូតេអ៊ីន

### ការព្រមានអំពីសារធាតុបង្កអាលែហ្ស៊ី {#allergen-warnings}

រួមបញ្ចូលសារធាតុបង្កអាលែហ្ស៊ីទូទៅ៖

- **Nuts** — គ្រាប់ធញ្ញជាតិពីដើមឈើ និងសណ្តែកដី
- **Shellfish** — សត្វសំបករឹង និងសត្វសំបកទន់
- **Fish** — ប្រភេទត្រីទាំងអស់
- **Dairy** — ទឹកដោះគោ និងផលិតផលទឹកដោះគោ
- **Eggs** — ស៊ុតមាន់
- **Soy** — ផលិតផលសណ្តែកសៀង
- **Gluten** — ស្រូវសាលី និងគ្រាប់ធញ្ញជាតិដែលមាន gluten
- **Sesame** — គ្រាប់ល្ង និងប្រេងល្ង

## ការគ្រប់គ្រងម៉ឺនុយ {#menu-management}

### ការធ្វើបច្ចុប្បន្នភាពតម្លៃ {#updating-prices}

ធ្វើបច្ចុប្បន្នភាពតម្លៃម៉ឺនុយបានយ៉ាងងាយស្រួល៖

1. ចូលទៅកាន់ទំព័រម៉ឺនុយ
2. ចុច "កែសម្រួលម៉ឺនុយ"
3. ធ្វើបច្ចុប្បន្នភាពតម្លៃសម្រាប់មុខម្ហូប
4. រក្សាទុកការផ្លាស់ប្តូរ
5. ការផ្លាស់ប្តូរនឹងបង្ហាញភ្លាមៗនៅលើគេហទំព័ររបស់អ្នក

### ការបន្ថែមមុខម្ហូបតាមរដូវកាល {#adding-seasonal-items}

បង្កើតកំណែម៉ឺនុយតាមរដូវកាល៖

1. បង្កើតកំណែម៉ឺនុយថ្មី
2. បន្ថែមមុខម្ហូបតាមរដូវកាល
3. សម្គាល់មុខម្ហូបជា "តាមរដូវកាល"
4. កំណត់កាលវិភាគកាលបរិច្ឆេទដែលមានលក់
5. បង្ហាញដោយស្វ័យប្រវត្តិក្នុងរដូវកាល

### ការគ្រប់គ្រងម្ហូបពិសេស {#managing-specials}

បង្ហាញម្ហូបពិសេសប្រចាំថ្ងៃ ឬប្រចាំសប្តាហ៍៖

1. បង្កើតប្រភេទ "ម្ហូបពិសេស"
2. បន្ថែមមុខម្ហូបជាមួយកាលបរិច្ឆេទដែលមានលក់
3. រំលេចម្ហូបពិសេសនៅលើទំព័រដើម
4. ធ្វើបច្ចុប្បន្នភាពប្រចាំថ្ងៃ ឬប្រចាំសប្តាហ៍
5. ទុកបណ្ណសារម្ហូបពិសេសចាស់

## ការរួមបញ្ចូលជាមួយ Theme Builder {#integration-with-theme-builder}

នៅពេលប្រើ Theme Builder សម្រាប់គេហទំព័រអាជីវកម្មបដិសណ្ឋារកិច្ច៖

1. **ការរកឃើញម៉ឺនុយដោយស្វ័យប្រវត្តិ** — កំណត់ថាអ្នកជាអាជីវកម្មអាហារ ឬភេសជ្ជៈឬអត់
2. **ការបង្កើតទំព័រម៉ឺនុយ** — បង្កើតទំព័រម៉ឺនុយមានលក្ខណៈវិជ្ជាជីវៈ
3. **ការផ្គូផ្គងរចនា** — ម៉ឺនុយស៊ីគ្នាជាមួយ theme របស់គេហទំព័រអ្នក
4. **ការបង្កើនប្រសិទ្ធភាពសម្រាប់ឧបករណ៍ចល័ត** — ម៉ឺនុយបង្ហាញបានស្អាតលើទូរសព្ទ
5. **ការបង្កើនប្រសិទ្ធភាព SEO** — ម៉ឺនុយងាយស្រួលសម្រាប់ម៉ាស៊ីនស្វែងរក

## អនុសាសន៍ល្អបំផុត {#best-practices}

### រចនាម៉ឺនុយ {#menu-design}

- **ការរៀបចំច្បាស់លាស់** — រចនាសម្ព័ន្ធប្រភេទមានតក្កវិជ្ជា
- **ការពិពណ៌នាងាយអាន** — គួរឱ្យចង់ញ៉ាំ និងផ្តល់ព័ត៌មាន
- **តម្លៃស៊ីសង្វាក់គ្នា** — រូបិយប័ណ្ណ និងទ្រង់ទ្រាយច្បាស់លាស់
- **រូបថតវិជ្ជាជីវៈ** — រូបភាពអាហារគុណភាពខ្ពស់
- **ចន្លោះទំនេរ** — កុំដាក់ទំព័រឱ្យចង្អៀតពេក

### មាតិកា {#content}

- **ការពិពណ៌នាត្រឹមត្រូវ** — ពិពណ៌នាមុខម្ហូបឱ្យត្រឹមត្រូវ
- **រំលេចម្ហូបពិសេស** — ធ្វើឱ្យម្ហូបពិសេសលេចធ្លោ
- **រួមបញ្ចូលសារធាតុបង្កអាលែហ្ស៊ី** — តែងតែរាយបញ្ជីសារធាតុបង្កអាលែហ្ស៊ី
- **ធ្វើបច្ចុប្បន្នភាពជាប្រចាំ** — រក្សាតម្លៃ និងមុខម្ហូបឱ្យទាន់សម័យ
- **ប្រើភាសាគួរឱ្យចង់ញ៉ាំ** — ធ្វើឱ្យមុខម្ហូបស្តាប់ទៅឆ្ងាញ់

### ភាពងាយស្រួលប្រើប្រាស់ {#accessibility}

- **ពុម្ពអក្សរងាយអាន** — ប្រើពុម្ពអក្សរច្បាស់ និងអាចអានបាន
- **កម្រិតផ្ទុយគ្រប់គ្រាន់** — ធានាថាអត្ថបទអាចអានបាន
- **ស្លាកអាហារូបត្ថម្ភ** — សម្គាល់ជម្រើសអាហារូបត្ថម្ភឱ្យច្បាស់
- **ការព្រមានអំពីសារធាតុបង្កអាលែហ្ស៊ី** — បង្ហាញសារធាតុបង្កអាលែហ្ស៊ីឱ្យលេចធ្លោ
- **សមស្របសម្រាប់ឧបករណ៍ចល័ត** — សាកល្បងលើឧបករណ៍ទាំងអស់
