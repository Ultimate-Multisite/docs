# សម្ភាសន៍ស្វែងយល់ {#discovery-interview}

**សម្ភាសន៍ស្វែងយល់** គឺជាជំហានដំបូងក្នុងដំណើរការរចនារបស់ Theme Builder។ វាប្រមូលព័ត៌មានអំពីគេហទំព័រ អាជីវកម្ម និងចំណូលចិត្តផ្នែករចនារបស់អ្នក ដើម្បីបង្កើត theme ផ្ទាល់ខ្លួនមួយ។

## ទិដ្ឋភាពទូទៅ {#overview}

សម្ភាសន៍ស្វែងយល់នឹងណែនាំអ្នកតាមរយៈសំណួរជាបន្តបន្ទាប់ ដើម្បីយល់ពីម៉ាក អ្នកទស្សនាគោលដៅ និងគោលដៅរចនារបស់អ្នក។ ក្នុងកំណែ 1.16.0 សម្ភាសន៍នេះឥឡូវរួមបញ្ចូល **ជំហានផ្ទុករូបថតឡើង** ដែលអនុញ្ញាតឱ្យអ្នកផ្តល់ការបំផុសគំនិតផ្នែករូបភាពសម្រាប់ការរចនារបស់អ្នក។

## ជំហានសម្ភាសន៍ {#interview-steps}

### 1. ព័ត៌មានអាជីវកម្ម {#1-business-information}

- ឈ្មោះគេហទំព័រ និងពាក្យស្លោក
- ប្រភេទអាជីវកម្ម ឬឧស្សាហកម្ម
- គោលដៅអាជីវកម្មចម្បង
- ការពិពណ៌នាអំពីអ្នកទស្សនាគោលដៅ

### 2. ចំណូលចិត្តផ្នែករចនា {#2-design-preferences}

- ពណ៌ចម្រុះដែលពេញចិត្ត
- រចនាប័ទ្មរចនា (ទំនើប បុរាណ សាមញ្ញ លេងសើច ជាដើម)
- គេហទំព័រ ឬម៉ាកដែលបំផុសគំនិត
- ធាតុរចនាដែលត្រូវតែមាន

### 3. ផ្ទុករូបថតឡើង (ថ្មីក្នុង v1.16.0) {#3-photo-upload-new-in-v1160}

ផ្ទុករូបថតមួយ ឬច្រើនឡើង ដើម្បីជះឥទ្ធិពលលើទិសដៅរចនា៖

#### ទម្រង់ដែលទទួលយក {#accepted-formats}

| ទម្រង់ | កន្ទុយឯកសារ | ព័ត៌មានលម្អិត |
|--------|-----------|---------|
| **JPEG** | `.jpg`, `.jpeg` | បានណែនាំសម្រាប់រូបថត |
| **PNG** | `.png` | គាំទ្រភាពថ្លា |
| **WebP** | `.webp` | ទម្រង់ទំនើប ទំហំឯកសារតូចជាង |
| **GIF** | `.gif` | រូបភាពឋិតិវន្តប៉ុណ្ណោះ (គ្មានចលនា) |

#### តម្រូវការឯកសារ {#file-requirements}

- **ទំហំឯកសារអតិបរមា៖** 10 MB
- **វិមាត្រអប្បបរមា៖** 400 × 300 ភីកសែល
- **វិមាត្រដែលបានណែនាំ៖** 1200 × 800 ភីកសែល ឬធំជាង
- **សមាមាត្ររូបភាព៖** សមាមាត្រណាមួយក៏ទទួលយកបាន

#### របៀបដែលរូបថតជះឥទ្ធិពលលើទិសដៅរចនា {#how-photos-influence-design-direction}

រូបថតដែលអ្នកផ្ទុកឡើងជួយឱ្យ Theme Builder យល់ពី៖

- **បន្ទះពណ៌** — ពណ៌លេចធ្លោត្រូវបានទាញយក ហើយប្រើក្នុង theme
- **រចនាប័ទ្មរូបភាព** — ទំនើប បែបជនបទ សាមញ្ញ ស្រស់ឆើត ជាដើម
- **អារម្មណ៍ និងបរិយាកាស** — វិជ្ជាជីវៈ ច្នៃប្រឌិត រួសរាយ ប្រណីត ជាដើម
- **ធាតុរចនា** — លំនាំ វាយនភាព និងរចនាប័ទ្មសមាសភាព
- **ការបំផុសគំនិតផ្នែកអក្សរ** — រចនាប័ទ្មពុម្ពអក្សរដែលស៊ីគ្នាជាមួយ theme ផ្នែករូបភាព

### 4. រចនាសម្ព័ន្ធមាតិកា {#4-content-structure}

- ទំព័រចម្បងដែលត្រូវការ (ទំព័រដើម អំពីយើង សេវាកម្ម ទំនាក់ទំនង ជាដើម)
- ត្រូវការផ្នែកប្លុក ឬព័ត៌មាន
- ត្រូវការមុខងារពាណិជ្ជកម្មអេឡិចត្រូនិក
- លក្ខណៈពិសេស ឬការរួមបញ្ចូលពិសេស

### 5. ពិនិត្យ និងបញ្ជាក់ {#5-review-and-confirmation}

- សេចក្តីសង្ខេបនៃព័ត៌មានទាំងអស់ដែលបានផ្តល់
- ការបញ្ជាក់មុនពេលការបង្កើតរចនាចាប់ផ្តើម

## ការអនុវត្តល្អបំផុតសម្រាប់ការផ្ទុករូបថតឡើង {#photo-upload-best-practices}

### អ្វីដែលគួរផ្ទុកឡើង {#what-to-upload}

- **ការបំផុសគំនិតម៉ាក** — ឡូហ្គោ គំរូពណ៌ ឬគោលការណ៍ណែនាំម៉ាក
- **គេហទំព័រគូប្រកួត** — រូបថតអេក្រង់នៃគេហទំព័រដែលអ្នកសរសើរ
- **ផ្ទាំងអារម្មណ៍** — បណ្ដុំរូបភាពដែលតំណាងឱ្យសោភ័ណភាពដែលអ្នកចង់បាន
- **រូបថតផលិតផល** — ប្រសិនបើអ្នកមានផលិតផលដើម្បីបង្ហាញ
- **រូបថតក្រុមការងារ ឬការិយាល័យ** — សម្រាប់ការបង្កើតម៉ាកផ្ទាល់ខ្លួន ឬវិជ្ជាជីវៈ
- **រូបភាពរបៀបរស់នៅ** — តំណាងឱ្យអ្នកទស្សនាគោលដៅរបស់អ្នក

### អ្វីដែលគួរជៀសវាង {#what-to-avoid}

- **រូបភាពគុណភាពទាប** — រូបថតព្រិល ឬបែកភីកសែល
- **រូបភាពមិនពាក់ព័ន្ធ** — រូបថតដែលមិនតំណាងឱ្យម៉ាករបស់អ្នក
- **រូបភាពមានអត្ថបទច្រើន** — រូបថតអេក្រង់ដែលមានអត្ថបទច្រើន
- **សម្ភារៈមានរក្សាសិទ្ធិ** — ត្រូវប្រាកដថាអ្នកមានសិទ្ធិប្រើរូបភាពទាំងនោះ
- **GIF មានចលនា** — ដំណើរការតែរូបភាពឋិតិវន្តប៉ុណ្ណោះ

### គន្លឹះដើម្បីទទួលបានលទ្ធផលល្អបំផុត {#tips-for-best-results}

1. **ផ្ទុករូបភាព 2-3 ឡើង** — រូបភាពច្រើនផ្តល់បរិបទបានល្អជាង
2. **ប្រើរូបថតគុណភាពខ្ពស់** — រូបភាពច្បាស់ និងមានពន្លឺល្អ ដំណើរការល្អបំផុត
3. **បញ្ចូលភាពចម្រុះ** — លាយបញ្ចូលពណ៌ រចនាប័ទ្ម និងប្រធានបទផ្សេងៗ
4. **ផ្តល់បរិបទ** — ពិពណ៌នាខ្លីៗថាហេតុអ្វីអ្នកជ្រើសរូបភាពនីមួយៗ
5. **បញ្ជាក់ឱ្យច្បាស់** — ប្រសិនបើអ្នកចូលចិត្តធាតុជាក់លាក់មួយ សូមចង្អុលបង្ហាញវា

## លទ្ធផលទិសដៅរចនា {#design-direction-output}

បន្ទាប់ពីសម្ភាសន៍ស្វែងយល់ Theme Builder បង្កើត៖

- **បន្ទះពណ៌** — ពណ៌ចម្បង ពណ៌បន្ទាប់បន្សំ និងពណ៌សង្កត់ធ្ងន់
- **អក្សរសិល្ប៍** — គ្រួសារពុម្ពអក្សរសម្រាប់ចំណងជើង និងអត្ថបទតួ
- **រចនាប័ទ្មប្លង់** — គម្លាត ការតម្រឹម និងការរៀបចំសមាសភាគ
- **ធាតុរូបភាព** — រូបតំណាង លំនាំ និងធាតុតុបតែង
- **ផ្ទាំងអារម្មណ៍** — ការបង្ហាញជារូបភាពនៃទិសដៅរចនា

## ជំហានបន្ទាប់ {#next-steps}

បន្ទាប់ពីបញ្ចប់សម្ភាសន៍ស្វែងយល់៖

1. ពិនិត្យទិសដៅរចនាដែលបានបង្កើត
2. ផ្តល់មតិកែលម្អ ឬស្នើសុំការកែតម្រូវ
3. បន្តទៅជំហាន [ទិសដៅរចនា](./design-direction.md)
4. Theme Builder បង្កើត theme ផ្ទាល់ខ្លួនរបស់អ្នក
