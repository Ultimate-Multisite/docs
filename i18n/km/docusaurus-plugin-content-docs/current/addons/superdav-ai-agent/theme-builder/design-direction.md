# ទិសដៅរចនា

ជំហាន **ទិសដៅរចនា** អនុញ្ញាតឱ្យអ្នកពិនិត្យ កែលម្អ និងបញ្ចប់ការរចនាផ្នែកមើលឃើញរបស់ theme របស់អ្នក មុនពេល Theme Builder បង្កើត theme ពេញលេញ។

## ទិដ្ឋភាពទូទៅ

បន្ទាប់ពីបញ្ចប់ការសម្ភាសស្វែងយល់ Theme Builder បង្ហាញទិសដៅរចនារបស់អ្នកជាមួយ **ការបង្ហាញមើលជាមុនសម្រាប់កុំព្យូទ័រ និងទូរស័ព្ទ**។ វាអនុញ្ញាតឱ្យអ្នកមើលឃើញច្បាស់ថា ការរចនារបស់អ្នកនឹងមើលទៅដូចម្តេចលើឧបករណ៍ផ្សេងៗ មុនពេលសម្រេចប្រើ theme។

## សមាសធាតុនៃទិសដៅរចនា

### 1. ក្តារពណ៌

ពណ៌សម្រាប់ theme របស់អ្នក រួមមាន៖

- **ពណ៌ចម្បង** — ពណ៌ម៉ាកសំខាន់ដែលប្រើសម្រាប់ប៊ូតុង តំណ និងចំណុចសង្កត់
- **ពណ៌បន្ទាប់បន្សំ** — ពណ៌បន្ថែមដែលសមស្របសម្រាប់ភាពចម្រុះ
- **ពណ៌សង្កត់** — ពណ៌បន្លិចសម្រាប់ធាតុសំខាន់ៗ
- **ពណ៌អព្យាក្រឹត** — ពណ៌ប្រផេះ និងស សម្រាប់ផ្ទៃខាងក្រោយ និងអត្ថបទ
- **ពណ៌អត្ថបទ** — ពណ៌អត្ថបទចម្បងដើម្បីងាយអាន

### 2. អក្សរសាស្ត្រ

ជម្រើសពុម្ពអក្សរសម្រាប់៖

- **ពុម្ពអក្សរចំណងជើង** — ប្រើសម្រាប់ចំណងជើងទំព័រ និងក្បាលផ្នែក
- **ពុម្ពអក្សរខ្លឹមសារ** — ប្រើសម្រាប់អត្ថបទកថាខណ្ឌ និងមាតិកា
- **ទំហំអក្សរ** — ទំហំដែលឆ្លើយតបតាមទំហំអេក្រង់ផ្សេងៗ
- **កម្ពស់បន្ទាត់** — ចន្លោះរវាងបន្ទាត់ដើម្បីងាយអាន
- **ទម្ងន់អក្សរ** — បំរែបំរួលដូចជា ដិត ធម្មតា និងស្រាល

### 3. ប្លង់ និងចន្លោះ

- **ទទឹងកុងតេន័រ** — ទទឹងអតិបរមាសម្រាប់មាតិកា
- **ចន្លោះខាងក្នុង និងគម្លាតខាងក្រៅ** — ចន្លោះជុំវិញធាតុ
- **ប្រព័ន្ធក្រឡាចត្រង្គ** — ប្លង់ជួរឈរសម្រាប់ការរចនាដែលឆ្លើយតប
- **ចន្លោះរវាងសមាសធាតុ** — ចម្ងាយរវាងធាតុ UI

### 4. ធាតុមើលឃើញ

- **រចនាប័ទ្មប៊ូតុង** — ការរចនាប៊ូតុងចម្បង បន្ទាប់បន្សំ និងទីបី
- **ការរចនាកាត** — ប្លង់សម្រាប់ប្លុកមាតិកា
- **រូបតំណាង** — រចនាប័ទ្ម និងទំហំរូបតំណាង
- **ស៊ុម និងស្រមោល** — ជម្រៅមើលឃើញស្រាលៗ
- **បែបផែនពេលដាក់ពីលើ** — រចនាប័ទ្មស្ថានភាពអន្តរកម្ម

## ការបង្ហាញមើលជាមុន

### ការបង្ហាញមើលជាមុនលើកុំព្យូទ័រ

ការបង្ហាញមើលជាមុនលើកុំព្យូទ័របង្ហាញការរចនារបស់អ្នកនៅទទឹងពេញលេញ (ជាទូទៅ 1200px ឬធំជាង)៖

- ម៉ឺនុយរុករកពេញលេញ
- ប្លង់ពេញលេញជាមួយជួរឈរទាំងអស់
- អក្សរសាស្ត្រ និងចន្លោះធំ
- ធាតុមើលឃើញទាំងអស់នៅទំហំពេញ

### ការបង្ហាញមើលជាមុនលើទូរស័ព្ទ

ការបង្ហាញមើលជាមុនលើទូរស័ព្ទបង្ហាញការរចនារបស់អ្នកនៅទទឹងទូរស័ព្ទ (ជាទូទៅ 375px)៖

- ការរុករកដែលឆ្លើយតប (ម៉ឺនុយ hamburger)
- ប្លង់ជួរឈរតែមួយ
- អក្សរសាស្ត្រ និងចន្លោះដែលបានកែសម្រួល
- ទំហំប៊ូតុងងាយប៉ះ

### ចំណុចបំបែកដែលឆ្លើយតប

Theme Builder បង្ហាញការមើលជាមុននៅចំណុចបំបែកទាំងនេះ៖

| ឧបករណ៍ | ទទឹង | ការបង្ហាញមើលជាមុន |
|--------|-------|---------|
| **ទូរស័ព្ទ** | 375px | ទូរស័ព្ទបញ្ឈរ |
| **ថេប្លេត** | 768px | ថេប្លេតផ្តេក |
| **កុំព្យូទ័រ** | 1200px | កុំព្យូទ័រទទឹងពេញ |
| **កុំព្យូទ័រធំ** | 1920px | អេក្រង់ទូលាយខ្លាំង |

## កែលម្អការរចនារបស់អ្នក

### ការកែសម្រួល

អ្នកអាចកែលម្អទិសដៅរចនារបស់អ្នកដោយ៖

1. **កែពណ៌** — ផ្លាស់ប្តូរពណ៌ណាមួយក្នុងក្តារពណ៌
2. **ប្តូរពុម្ពអក្សរ** — ជ្រើសរើសរូបរាងអក្សរផ្សេងៗ
3. **កែចន្លោះ** — កែចន្លោះខាងក្នុង និងគម្លាតខាងក្រៅ
4. **ធ្វើបច្ចុប្បន្នភាពប្លង់** — ប្តូរទទឹងកុងតេន័រ និងជួរឈរក្រឡាចត្រង្គ
5. **កែតម្រូវធាតុតាមតម្រូវការ** — កែរចនាប័ទ្មប៊ូតុង ការរចនាកាត ជាដើម

### ការធ្វើបច្ចុប្បន្នភាពការបង្ហាញមើលជាមុន

ការផ្លាស់ប្តូរត្រូវបានបង្ហាញភ្លាមៗ៖

- ការបង្ហាញមើលជាមុនលើកុំព្យូទ័រធ្វើបច្ចុប្បន្នភាពភ្លាមៗ
- ការបង្ហាញមើលជាមុនលើទូរស័ព្ទធ្វើបច្ចុប្បន្នភាពភ្លាមៗ
- ចំណុចបំបែកដែលឆ្លើយតបទាំងអស់ត្រូវបានធ្វើបច្ចុប្បន្នភាព
- អ្នកអាចប្តូររវាងការបង្ហាញមើលជាមុន ដើម្បីផ្ទៀងផ្ទាត់ការផ្លាស់ប្តូរ

### ទិដ្ឋភាពប្រៀបធៀប

ប្រៀបធៀបទិសដៅរចនារបស់អ្នកជាមួយ៖

- **ការរចនាដើម** — មើលអ្វីដែលបានផ្លាស់ប្តូរ
- **ការរចនារបស់គូប្រកួត** — ប្រៀបធៀបជាមួយគេហទំព័របំផុសគំនិត
- **កំណែមុនៗ** — ត្រឡប់ទៅទិសដៅរចនាមុន

## ការអនុម័តទិសដៅរចនា

នៅពេលអ្នកពេញចិត្តនឹងការរចនារបស់អ្នក៖

1. **ពិនិត្យការបង្ហាញមើលជាមុនទាំងពីរ** — ផ្ទៀងផ្ទាត់ថាលើកុំព្យូទ័រ និងទូរស័ព្ទមើលទៅល្អ
2. **ពិនិត្យពណ៌ទាំងអស់** — ធានាថាមានកម្រិតផ្ទុយ និងភាពងាយចូលប្រើ
3. **សាកល្បងអក្សរសាស្ត្រ** — ផ្ទៀងផ្ទាត់ភាពងាយអាននៅគ្រប់ទំហំ
4. **បញ្ជាក់ប្លង់** — ពិនិត្យចន្លោះ និងការតម្រឹម
5. **អនុម័តការរចនា** — បន្តទៅការបង្កើត theme

## ជំហានបន្ទាប់

បន្ទាប់ពីអនុម័តទិសដៅរចនារបស់អ្នក៖

1. Theme Builder បង្កើត theme ពេញលេញរបស់អ្នក
2. theme ត្រូវបានដំឡើងលើគេហទំព័រ WordPress របស់អ្នក
3. អ្នកអាចកែតម្រូវបន្ថែមដោយប្រើឧបករណ៍កែតម្រូវរបស់ WordPress
4. បន្តទៅ [ម៉ឺនុយសេវាកម្មទទួលភ្ញៀវ](./hospitality-menus.md) ឬមុខងារផ្សេងៗ

## អនុវត្តន៍ល្អបំផុត

- **សាកល្បងលើឧបករណ៍ពិត** — ប្រើទូរស័ព្ទ និងថេប្លេតពិត ប្រសិនបើអាច
- **ពិនិត្យភាពងាយអាន** — ធានាថាអត្ថបទអាចអានបាននៅគ្រប់ទំហំ
- **ផ្ទៀងផ្ទាត់កម្រិតផ្ទុយ** — ប្រើ [ផ្ទៀងផ្ទាត់កម្រិតផ្ទុយនៃក្តារពណ៌](../abilities/validate-palette-contrast.md) សម្រាប់ភាពងាយចូលប្រើ
- **គិតពីប្រសិទ្ធភាព** — បង្កើនប្រសិទ្ធភាពរូបភាព និងពុម្ពអក្សរ ដើម្បីឱ្យលឿន
- **រៀបចំផែនការសម្រាប់មាតិកា** — ធានាថាប្លង់ដំណើរការល្អជាមួយមាតិកាពិតរបស់អ្នក

## ដោះស្រាយបញ្ហា

### ការបង្ហាញមើលជាមុនមិនធ្វើបច្ចុប្បន្នភាព

- ផ្ទុកទំព័រឡើងវិញ
- សម្អាតឃ្លាំងសម្ងាត់កម្មវិធីរុករក
- សាកល្បងកម្មវិធីរុករកផ្សេង
- ពិនិត្យការតភ្ជាប់អ៊ីនធឺណិត

### ពណ៌មើលទៅខុសគ្នា

- ពិនិត្យការកំណត់ពណ៌ម៉ូនីទ័រ
- សាកល្បងលើឧបករណ៍ផ្សេងៗ
- ផ្ទៀងផ្ទាត់កម្រិតផ្ទុយពណ៌ជាមួយឧបករណ៍ភាពងាយចូលប្រើ
- ពិចារណាឧបករណ៍ត្រាប់តាមភាពមិនអាចសម្គាល់ពណ៌

### បញ្ហាអក្សរសាស្ត្រ
