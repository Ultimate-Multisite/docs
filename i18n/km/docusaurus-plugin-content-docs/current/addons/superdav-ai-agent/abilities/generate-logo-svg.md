# បង្កើត​ឡូហ្គោ SVG {#generate-logo-svg}

សមត្ថភាព **បង្កើត​ឡូហ្គោ SVG** អនុញ្ញាតឱ្យ Theme Builder បង្កើត និងបញ្ចូល​ឡូហ្គោ SVG ផ្ទាល់ខ្លួនដោយផ្ទាល់ទៅក្នុងគេហទំព័រ WordPress របស់អ្នក ជាមួយការសម្អាតដែលមានសុវត្ថិភាពចំពោះ namespace ដោយស្វ័យប្រវត្តិ។

## ទិដ្ឋភាពទូទៅ {#overview}

សមត្ថភាពនេះបង្កើត​ឡូហ្គោ​ជា​ក្រាហ្វិក​វ៉ិចទ័រ​ដែលអាចពង្រីកបាន (SVG) ផ្អែកលើទិសដៅម៉ាក និងចំណូលចិត្តរចនារបស់គេហទំព័ររបស់អ្នក។ SVG ដែលបានបង្កើតនឹងត្រូវបានសម្អាតដោយស្វ័យប្រវត្តិ ដើម្បីធានាថាវាមានសុវត្ថិភាពសម្រាប់ប្រើក្នុង WordPress ខណៈនៅតែរក្សារូបរាងមើលឃើញឱ្យបានត្រឹមត្រូវ។

## ប៉ារ៉ាម៉ែត្រ {#parameters}

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `site_name` | string | បាទ/ចាស | ឈ្មោះគេហទំព័រដែលត្រូវបង្កើត​ឡូហ្គោឱ្យ |
| `style` | string | បាទ/ចាស | រចនាបថ (ឧ. "ទំនើប", "បុរាណ", "សាមញ្ញ", "លេងសើច") |
| `colors` | array | ទេ | អារេ​នៃ​កូដពណ៌ hex ដែលត្រូវប្រើក្នុង​ឡូហ្គោ (ឧ. `["#678233", "#ffffff"]`) |
| `width` | number | ទេ | ទទឹង SVG ជា​ភីកសែល (លំនាំដើម៖ 200) |
| `height` | number | ទេ | កម្ពស់ SVG ជា​ភីកសែល (លំនាំដើម៖ 200) |
| `include_text` | boolean | ទេ | ថាតើត្រូវបញ្ចូលឈ្មោះគេហទំព័រ​ជា​អត្ថបទក្នុង​ឡូហ្គោឬអត់ (លំនាំដើម៖ true) |

## ទ្រង់ទ្រាយលទ្ធផល {#output-format}

សមត្ថភាពនេះត្រឡប់ខ្សែអក្សរ SVG ដែលមានរចនាសម្ព័ន្ធដូចខាងក្រោម៖

```json
{
  "svg": "<svg xmlns=\"http://www.w3.org/2000/svg\" viewBox=\"0 0 200 200\">...</svg>",
  "width": 200,
  "height": 200,
  "colors_used": ["#678233", "#ffffff"],
  "sanitised": true
}
```

## ឥរិយាបថ​សម្អាត SVG {#svg-sanitisation-behaviour}

SVG ដែលបានបង្កើតនឹងឆ្លងកាត់ការសម្អាតដោយស្វ័យប្រវត្តិ ដែលមានសុវត្ថិភាពចំពោះ namespace ដើម្បី៖

- **យក attribute ដែលមិនមានសុវត្ថិភាពចេញ** — ដក event handler, script និង attribute ដែលអាចមានគ្រោះថ្នាក់
- **រក្សា namespace** — រក្សា SVG namespace (xmlns, xlink) ដើម្បីបង្ហាញបានត្រឹមត្រូវ
- **ផ្ទៀងផ្ទាត់រចនាសម្ព័ន្ធ** — ធានាថា SVG អនុលោមតាមស្តង់ដារ W3C
- **អ៊ិនកូដ entity** — គេចតួអក្សរពិសេសក្នុងមាតិកាអត្ថបទឱ្យបានត្រឹមត្រូវ
- **យកការយោងខាងក្រៅចេញ** — ដក stylesheet ខាងក្រៅ និងការយោងរូបភាពខាងក្រៅ

នេះធានាថា SVG មានសុវត្ថិភាពសម្រាប់បញ្ចូលដោយផ្ទាល់ក្នុង WordPress ដោយមិនចាំបាច់សម្អាតបន្ថែម។

## ឧទាហរណ៍ការប្រើប្រាស់ {#usage-example}

**Prompt:**
```
បង្កើត​ឡូហ្គោទំនើបសម្រាប់ startup បច្ចេកវិទ្យារបស់ខ្ញុំឈ្មោះ "CloudSync" ដោយប្រើពណ៌ខៀវ និងស។
```

**លទ្ធផល៖**
សមត្ថភាពនេះបង្កើត​ឡូហ្គោ SVG ដែល៖
- បញ្ចូលឈ្មោះគេហទំព័រ "CloudSync"
- ប្រើសំណុំពណ៌ខៀវ និងសដែលបានកំណត់
- អនុវត្តតាមគោលការណ៍រចនាទំនើប
- ត្រូវបានសម្អាតដោយស្វ័យប្រវត្តិ និងរួចរាល់សម្រាប់ប្រើ

## ការរួមបញ្ចូលជាមួយ Theme Builder {#integration-with-theme-builder}

នៅពេលប្រើការជ្រើសរើសទិសដៅរចនារបស់ Theme Builder សមត្ថភាពបង្កើត​ឡូហ្គោ SVG នឹង៖

1. វិភាគទិសដៅរចនា និងក្ដារពណ៌របស់អ្នក
2. បង្កើត​ឡូហ្គោ SVG ផ្ទាល់ខ្លួនដែលស្របនឹងចំណូលចិត្តរបស់អ្នក
3. បញ្ចូល​ឡូហ្គោដោយស្វ័យប្រវត្តិទៅក្នុងតំបន់ header/ម៉ាក​របស់គេហទំព័ររបស់អ្នក
4. រក្សាទុក SVG ជា​ឡូហ្គោផ្ទាល់ខ្លួនក្នុងមេឌៀ WordPress

## ការអនុវត្តល្អបំផុត {#best-practices}

- **ផ្តល់ចំណូលចិត្តរចនាបថឱ្យច្បាស់** — ពិពណ៌នារចនាបថដែលអ្នកចង់បាន (ទំនើប, បុរាណ, លេងសើច ជាដើម)
- **កំណត់ពណ៌** — បញ្ចូលពណ៌ម៉ាករបស់អ្នក ដើម្បីរក្សាភាពស៊ីសង្វាក់
- **សាកល្បងការបង្ហាញ** — ផ្ទៀងផ្ទាត់ថា​ឡូហ្គោបង្ហាញបានត្រឹមត្រូវលើទំហំអេក្រង់ផ្សេងៗ
- **កែតម្រូវបន្ថែម** — ប្រើឧបករណ៍កែតម្រូវ​ឡូហ្គោរបស់ WordPress ដើម្បីកែទំហំ និងទីតាំង

## ដែនកំណត់ {#limitations}

- ឡូហ្គោ SVG ត្រូវបានបង្កើតជា​ក្រាហ្វិកថេរ (មិនមានចលនា)
- ឡូហ្គោស្មុគស្មាញដែលមានធាតុច្រើន អាចត្រូវការការកែលម្អដោយដៃ
- មិនគាំទ្រ font ផ្ទាល់ខ្លួនទេ; អត្ថបទប្រើ font របស់ប្រព័ន្ធ
- វិមាត្រធំពេក ឬតូចពេក អាចប៉ះពាល់ដល់គុណភាព
