## តម្រូវការ

- WordPress 6.0 ឬខ្ពស់ជាងនេះ
- PHP 8.1 ឬខ្ពស់ជាងនេះ
- ការដំឡើង WordPress Multisite (បានបើកនៅកម្រិតបណ្តាញ) ឬវេបសាយតែមួយ
- plugin ភ្ជាប់អ្នកផ្តល់សេវា AI ដែលសកម្មមួយ (Anthropic, OpenAI ឬដែលឆបគ្នា)
- Gratis AI Agent v1.9.0 ឬខ្ពស់ជាងនេះ (Superdav AI Agent ពង្រីក Gratis AI Agent)

## ការដំឡើង

1. ផ្ទុកឯកសារ plugin ឡើងទៅ `/wp-content/plugins/superdav-ai-agent`
2. បើក plugin តាមអេក្រង់ **Plugins** ក្នុង WordPress
3. ធានាថា Gratis AI Agent ក៏បានបើកដែរ
4. ចូលទៅ **Gratis AI Agent → Settings** ហើយកំណត់អ្នកផ្តល់សេវាស្វែងរកអ៊ីនធឺណិត ការភ្ជាប់សេវា Superdav ដែលគ្រប់គ្រង Google Calendar ឬព័ត៌មានសម្គាល់ TextBee តាមតម្រូវការ

កញ្ចប់ WordPress.org មានមុខងារស្នូលរបស់ Superdav AI Agent។ កញ្ចប់ Advanced companion ត្រូវបានបង្កើត និងចែកចាយដោយឡែក សម្រាប់បរិស្ថានដែលឧបករណ៍អ្នកអភិវឌ្ឍន៍ដែលអាចទុកចិត្តបានត្រូវបានដំឡើង និងអនុញ្ញាតយ៉ាងច្បាស់។
