# бһ–бһ бһ»бһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺ

Ultimate Multisite: бһ–бһ бһ»бһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһ”бһҺбҹ’бһҸбһ¶бһүбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бһ‘бһҪбһӣбһҖбһ¶бһҡбһ‘бһјбһ‘бһ¶бһҸбҹӢбһҮбһ¶бһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһ…бҹ’бһҡбһҫбһ“бҹ” бһңбһ¶бһ”бһҳбҹ’бһӣбҹӮбһ„бһҸбһҳбҹ’бһӣбҹғбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһўбһҸбҹ’бһҡбһ¶бһ”бҹ’бһҸбһјбһҡбһ”бҹ’бһҡбһ¶бһҖбҹӢбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ бһ¬бһҠбҹ„бһҷбһҠбҹғ бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҸбһ¶бһҳбһ•бһӣбһ·бһҸбһ•бһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһҺбһ¶бһҳбһҪбһҷ бһ бһҫбһҷбһҡбһҖбһғбһҫбһүбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһјбһӣбһҳбһҫбһӣбһ“бһёбһҳбһҪбһҷбҹ—бһ–бҹҒбһүбһ…бһ·бһҸбҹ’бһҸбһҸбһ¶бһҳбһҡбһҷбҹҲбһ‘бһёбһҸбһ¶бҹҶбһ„бһ—бһјбһҳбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ вҖ” бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ–бһёбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҸбҹӮбһҳбһҪбһҷбҹ”

## бһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ— {#key-features}

- **бһҖбһ¶бһҡбһ”бһҳбҹ’бһӣбҹӮбһ„бһўбһҸбҹ’бһҡбһ¶бһ”бҹ’бһҸбһјбһҡбһ”бҹ’бһҡбһ¶бһҖбҹӢбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·** вҖ” бһҸбһҳбҹ’бһӣбҹғбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҳбҹ’бһӣбҹӮбһ„бһ–бһёбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһўбһҸбҹ’бһҡбһ¶бһҠбҹӮбһӣбһ‘бһ¶бһүбһҷбһҖбһ–бһёбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ
- **бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғбһҮбҹҶбһ“бһҪбһҹбһҸбһ¶бһҳбһ•бһӣбһ·бһҸбһ•бһӣ** вҖ” бһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҸбһ¶бһҳбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһӣбһҫбһ•бһӣбһ·бһҸбһ•бһӣбһҺбһ¶бһҳбһҪбһҷ бһҠбһҫбһҳбҹ’бһ”бһёбһҡбҹҶбһӣбһ„бһҖбһ¶бһҡбһ”бһҳбҹ’бһӣбҹӮбһ„бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·
- **бһңбһ¶бһӣбһҮбҹ’бһҡбһҫбһҹбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһ–бҹҒбһӣбһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮ** вҖ” бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбһҸбһ·бһҗбһ·бһҮбһ“бһҮбҹ’бһҡбһҫбһҹбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһҠбҹӮбһӣбһ–бһҪбһҖбһӮбҹҒбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһ“бҹ…бһ–бҹҒбһӣбһ‘бһјбһ‘бһ¶бһҸбҹӢ/бһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮ
- **бһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһўбһҸбҹ’бһҡбһ¶бһ”бҹ’бһҸбһјбһҡбһ”бҹ’бһҡбһ¶бһҖбҹӢбһ…бҹ’бһҡбһҫбһ“** вҖ” ECB, Frankfurter, Stripe, бһ¬бһўбһҸбҹ’бһҡбһ¶бһҠбҹ„бһҷбһҠбҹғбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„
- **бһҖбһ¶бһҡбһҡбһҖбһғбһҫбһүбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ‘бһёбһҸбһ¶бҹҶбһ„бһ—бһјбһҳбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ** вҖ” бһҮбҹ’бһҡбһҫбһҹбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ“бһ№бһ„бһ”бҹ’бһҡбһ‘бҹҒбһҹбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ…бһјбһӣбһҳбһҫбһӣбһҮбһ¶бһҳбһ»бһ“бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·
- **бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„** вҖ” бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһҹбһҖбһҳбҹ’бһҳ бһўбһҸбҹ’бһҡбһ¶бһ”бҹ’бһҸбһјбһҡбһ”бҹ’бһҡбһ¶бһҖбҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶бһ–бһёбһҖбһ“бҹ’бһӣбҹӮбһ„бһҸбҹӮбһҳбһҪбһҷ

## бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡ {#requirements}

- Ultimate Multisite 2.0.0 бһ¬бһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„бһ“бҹҒбҹҮ
- бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ WordPress Multisite
- PHP 7.4 бһ¬бһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„бһ“бҹҒбҹҮ

## бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ {#installation}

1. бһ•бҹ’бһ‘бһ»бһҖбһЎбһҫбһ„бһҜбһҖбһҹбһ¶бһҡ plugin бһ‘бҹ… `/wp-content/plugins/ultimate-multisite-multi-currency`
2. бһ”бһҫбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ plugin бһҸбһ¶бһҳбһҡбһҷбҹҲбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ **бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳ** бһҖбҹ’бһ“бһ»бһ„ WordPress (бһ”бһҫбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫбһ”бһҺбҹ’бһҸбһ¶бһү)
3. бһ‘бҹ…бһҖбһ¶бһ“бҹӢ **Ultimate Multisite вҶ’ бһ–бһ бһ»бһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺ** бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҫбһҖбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„
4. бһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ‘бһ‘бһҪбһӣ бһ бһҫбһҷбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһўбһҸбҹ’бһҡбһ¶бһ”бҹ’бһҸбһјбһҡбһ”бҹ’бһҡбһ¶бһҖбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

## бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ {#configuration}

### бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ {#admin-dashboard}

бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бһ бһ»бһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺ (**Ultimate Multisite вҶ’ бһ–бһ бһ»бһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺ**) бһӮбһәбһҮбһ¶бһҖбһ“бҹ’бһӣбҹӮбһ„бһҖбһҺбҹ’бһҸбһ¶бһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбҹ–

- бһ”бһ“бҹ’бһҗбҹӮбһҳ бһ”бһҫбһҖ бһ¬бһ”бһ·бһ‘бһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺ
- бһҮбҹ’бһҡбһҫбһҹбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһўбһҸбҹ’бһҡбһ¶бһ”бҹ’бһҸбһјбһҡбһ”бҹ’бһҡбһ¶бһҖбҹӢ бһ“бһ·бһ„бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһўбһҸбҹ’бһҡбһ¶бһҗбҹ’бһҳбһёбһҸбһ¶бһҳбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡ
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһўбһҸбҹ’бһҡбһ¶бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһҹбһҖбһҳбҹ’бһҳбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ
- бһҖбҹҶбһҺбһҸбҹӢбһўбһҸбҹ’бһҡбһ¶бһҠбҹ„бһҷбһҠбҹғбһ“бҹ…бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҳбһ·бһ“бһҹбһҳбһҹбҹ’бһҡбһ”

### бһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһўбһҸбҹ’бһҡбһ¶бһ”бҹ’бһҸбһјбһҡбһ”бҹ’бһҡбһ¶бһҖбҹӢ {#exchange-rate-providers}

|---|---|

| **Frankfurter** | API бһҶбҹ’бһӣбһ»бҹҮбһ”бһүбҹ’бһ…бһ¶бҹҶбһ„ ECB бһ”бҹ’бһҡбһ—бһ–бһ”бһҫбһҖбһ…бҹҶбһ  вҖ” бһҘбһҸбһӮбһ·бһҸбһҗбҹ’бһӣбҹғ бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ API key |
| **Stripe** | бһ”бҹ’бһҡбһҫбһўбһҸбҹ’бһҡбһ¶ FX бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһҡбһ”бһҹбҹӢ Stripe вҖ” бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Stripe бһҠбҹӮбһӣбһҹбһҖбһҳбҹ’бһҳ |
| **бһҠбҹ„бһҷбһҠбҹғ** | бһўбҹ’бһ“бһҖбһ”бһүбҹ’бһ…бһјбһӣбһўбһҸбҹ’бһҡбһ¶бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢ; бһҳбһ·бһ“бһҳбһ¶бһ“бһҖбһ¶бһҡбһ бҹ… API бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бһ‘бҹҒ |

бһҮбҹ’бһҡбһҫбһҹбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бҹ…бһҖбҹ’бһҡбҹ„бһҳ **Ultimate Multisite вҶ’ бһ–бһ бһ»бһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺ вҶ’ бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ вҶ’ бһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһўбһҸбҹ’бһҡбһ¶бһ”бҹ’бһҸбһјбһҡбһ”бҹ’бһҡбһ¶бһҖбҹӢ**бҹ” бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһўбһҸбҹ’бһҡбһ¶бһҗбҹ’бһҳбһёбһҠбҹ„бһҷбһҠбҹғбһ”бһ¶бһ“бһӮбҹ’бһҡбһ”бҹӢбһ–бҹҒбһӣ бһ¬бһҖбҹҶбһҺбһҸбҹӢбһ–бҹҒбһӣбһұбҹ’бһҷбһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ”

### бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғбһҮбҹҶбһ“бһҪбһҹбһҸбһ¶бһҳбһ•бһӣбһ·бһҸбһ•бһӣ {#per-product-pricing-overrides}

бһ“бҹ…бһ–бҹҒбһӣбһҖбһ¶бһҡбһ”бһҳбҹ’бһӣбҹӮбһ„бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһӣбһҫбһ•бһӣбһ·бһҸбһ•бһӣбһҺбһ¶бһҳбһҪбһҷбҹ–

1. бһ”бһҫбһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһ•бһӣбһ·бһҸбһ•бһӣ (**Ultimate Multisite вҶ’ бһ•бһӣбһ·бһҸбһ•бһӣ вҶ’ бһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣ**)
2. бһҡбҹҶбһҖбһ·бһӣбһ‘бҹ…бһ•бҹ’бһ“бҹӮбһҖ **бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺ**
3. бһ”бһүбҹ’бһ…бһјбһӣбһҸбһҳбҹ’бһӣбҹғбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһ“бһёбһҳбһҪбһҷбҹ—бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһҖбҹҶбһҺбһҸбҹӢбһҮбҹҶбһ“бһҪбһҹ
4. бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ•бһӣбһ·бһҸбһ•бһӣ

бһ•бһӣбһ·бһҸбһ•бһӣбһҠбҹӮбһӣбһҳбһ·бһ“бһҳбһ¶бһ“бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҮбҹҶбһ“бһҪбһҹбһ”бҹ’бһҡбһҫбһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһҳбҹ’бһӣбҹӮбһ„бһ–бһёбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

### бһңбһ¶бһӣбһҮбҹ’бһҡбһҫбһҹбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһ–бҹҒбһӣбһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮ {#currency-selector-signup-field}

бһўбһ¶бһ…бһ”бһ“бҹ’бһҗбҹӮбһҳбһңбһ¶бһӣбһҮбҹ’бһҡбһҫбһҹбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһ‘бҹ…бһӣбҹҶбһ бһјбһҡбһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮ/бһ‘бһјбһ‘бһ¶бһҸбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷбһўбһҸбһ·бһҗбһ·бһҮбһ“бһҮбҹ’бһҡбһҫбһҹбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһҠбҹӮбһӣбһ–бһҪбһҖбһӮбҹҒбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһ“бҹ…бһ–бҹҒбһӣбһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮбҹ–

1. бһ‘бҹ…бһҖбһ¶бһ“бҹӢ **Ultimate Multisite вҶ’ бһ–бһ бһ»бһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺ вҶ’ бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ**
2. бһ”бһҫбһҖ **бһ”бһ„бҹ’бһ бһ¶бһүбһ§бһ”бһҖбһҡбһҺбҹҚбһҮбҹ’бһҡбһҫбһҹбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһ“бҹ…бһ–бҹҒбһӣбһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮ**
3. бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹ бһҖбҹҶбһҺбһҸбҹӢбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳ бһ¬бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһ‘бһёбһҸбһ¶бҹҶбһ„бһ—бһјбһҳбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣбһҮбҹ’бһҡбһҫбһҹбһҮбһ¶бһҳбһ»бһ“

бһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһҠбҹӮбһӣбһ”бһ¶бһ“бһҮбҹ’бһҡбһҫбһҹбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҮбһ¶бһң бһ бһҫбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ…бҹҒбһүбһңбһ·бһҖбҹ’бһҖбһҷбһ”бһҸбҹ’бһҡбһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ”

### бһҖбһ¶бһҡбһҡбһҖбһғбһҫбһүбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ‘бһёбһҸбһ¶бҹҶбһ„бһ—бһјбһҳбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ {#geolocation-based-currency-detection}

бһ“бҹ…бһ–бҹҒбһӣбһ”бһҫбһҖбһҖбһ¶бһҡбһҡбһҖбһғбһҫбһүбһ‘бһёбһҸбһ¶бҹҶбһ„бһ—бһјбһҳбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ addon бһ“бһ№бһ„бһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһ”бҹ’бһҡбһ‘бҹҒбһҹбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ…бһјбһӣбһҳбһҫбһӣбһ‘бҹ…бһ“бһ№бһ„бһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺ бһ бһҫбһҷбһҮбҹ’бһҡбһҫбһҹбһңбһ¶бһҮбһ¶бһҳбһ»бһ“бһҖбҹ’бһ“бһ»бһ„бһңбһ¶бһӣбһҮбҹ’бһҡбһҫбһҹбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҫбһҖбҹ–

1. бһ‘бҹ…бһҖбһ¶бһ“бҹӢ **Ultimate Multisite вҶ’ бһ–бһ бһ»бһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺ вҶ’ бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ**
2. бһ”бһҫбһҖ **бһҡбһҖбһғбһҫбһүбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһ–бһёбһ‘бһёбһҸбһ¶бҹҶбһ„бһўбҹ’бһ“бһҖбһ…бһјбһӣбһҳбһҫбһӣ**
3. бһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһ”бҹ’бһҡбһ‘бҹҒбһҹбһ‘бҹ…бһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһҖбҹ’бһ“бһ»бһ„бһҸбһ¶бһҡбһ¶бһ„ **бһҖбһ¶бһҡбһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһ‘бһёбһҸбһ¶бҹҶбһ„бһ—бһјбһҳбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ**

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ·бһ“бһҳбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ‘бҹҒбһҹбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ…бһјбһӣбһҳбһҫбһӣбһ‘бҹҒ бһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҡбһ”бһҹбҹӢбһ”бһҺбҹ’бһҸбһ¶бһүбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹбһ”бһҳбҹ’бһҡбһ»бһ„бҹ”

## бһҹбҹҶбһҺбһҪбһҡбһҠбҹӮбһӣбһҹбһҪбһҡбһүбһ№бһҖбһүбһ¶бһ”бҹӢ {#frequently-asked-questions}

### бһҸбһҫбһҳбһ¶бһ“бһўбҹ’бһңбһёбһҖбһҫбһҸбһЎбһҫбһ„ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһўбһҸбҹ’бһҡбһ¶бһ”бҹ’бһҸбһјбһҡбһ”бҹ’бһҡбһ¶бһҖбҹӢбһҳбһ·бһ“бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“? {#what-happens-if-an-exchange-rate-provider-is-unavailable}

addon бһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһўбһҸбҹ’бһҡбһ¶бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһ‘бһ¶бһүбһҷбһҖбһҠбҹ„бһҷбһҮбҹ„бһӮбһҮбҹҗбһҷбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ·бһ“бһ’бҹ’бһӣбһ¶бһ”бҹӢбһ‘бһ¶бһүбһҷбһҖбһўбһҸбҹ’бһҡбһ¶бһҺбһ¶бһҳбһҪбһҷбһ‘бҹҒ бһңбһ¶бһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһўбһҸбҹ’бһҡбһ¶бһҠбҹ„бһҷбһҠбҹғбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бҹ” бһҹбһјбһҳбһҺбҹӮбһ“бһ¶бҹҶбһұбҹ’бһҷбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһўбһҸбҹ’бһҡбһ¶бһҠбҹ„бһҷбһҠбҹғбһҮбһ¶бһҹбҹҶбһҺбһ¶бһүбҹӢбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бҹ”

### бһҸбһҫбһҒбҹ’бһүбһ»бҹҶбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶бһ•бҹ’бһҹбҹҒбһ„бһӮбҹ’бһ“бһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһ•бҹ’бһҹбҹҒбһ„бһӮбҹ’бһ“бһ¶бһ”бһ¶бһ“бһ‘бҹҒ? {#can-i-use-different-providers-for-different-currencies}

бһҳбһ·бһ“бһ”бһ¶бһ“бһ‘бҹҒ вҖ” бһҳбһ¶бһ“бһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶бһҸбҹӮбһҳбһҪбһҷбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһҠбҹӮбһӣбһҹбһҖбһҳбҹ’бһҳбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ” бһ”бҹ’бһҡбһҫ **бһҠбҹ„бһҷбһҠбҹғ** бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ–бҹҒбһүбһӣбҹҒбһүбһӣбһҫбһўбһҸбҹ’бһҡбһ¶бһ“бһёбһҳбһҪбһҷбҹ—бҹ”

### бһҸбһҫбһ‘бһёбһҸбһ¶бҹҶбһ„бһ—бһјбһҳбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ plugin бһҠбһ¶бһ…бҹӢбһҠбҹ„бһҷбһЎбҹӮбһҖбһ‘бҹҒ? {#does-geolocation-require-a-separate-plugin}

бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ plugin бһ‘бһёбһҸбһ¶бҹҶбһ„бһ—бһјбһҳбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бһ‘бҹҒбҹ” addon бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһҡбһҖбһғбһҫбһү IP бһ‘бҹ…бһ”бҹ’бһҡбһ‘бҹҒбһҹбһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбһҖбһҹбҹ’бһҡбһ¶бһ”бҹӢбһҡбһ”бһҹбҹӢ WordPressбҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„бһ“бҹҒбҹҮ бһўбһ¶бһ…бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһңбһ¶бһ‘бһёбһҸбһ¶бҹҶбһ„бһ—бһјбһҳбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбһ¶бһ…бҹӢбһҠбҹ„бһҷбһЎбҹӮбһҖбһҸбһ¶бһҳбһҡбһҷбҹҲ filter hooks бһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһҸбһӣбҹӢбҹ”

### бһҸбһҫбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҮбҹҶбһ“бһҪбһҹбһҸбһ¶бһҳбһ•бһӣбһ·бһҸбһ•бһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҳбһ»бһ“ бһ¬бһҖбҹ’бһҡбҹ„бһҷбһ–бһ“бҹ’бһ’? {#are-per-product-overrides-applied-before-or-after-tax}

бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҮбҹҶбһ“бһҪбһҹбһҸбһ¶бһҳбһ•бһӣбһ·бһҸбһ•бһӣбһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғбһҳбһ»бһ“бһ–бһ“бҹ’бһ’бһҖбҹ’бһ“бһ»бһ„бһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһӮбҹ„бһӣбһҠбҹ…бҹ” бһ–бһ“бҹ’бһ’бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӮбһҺбһ“бһ¶бһ”бһ“бҹ’бһҗбҹӮбһҳбһӣбһҫбһҸбһҳбҹ’бһӣбҹғбһҖбҹҶбһҺбһҸбҹӢбһҮбҹҶбһ“бһҪбһҹ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ–бһ“бҹ’бһ’бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

## бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ {#changelog}

### бһҖбҹҶбһҺбҹӮ 1.0.0 вҖ” бһ”бһ¶бһ“бһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһ“бҹ…бһҗбҹ’бһ„бҹғбһ‘бһё 2026-04-02 {#version-100--released-on-2026-04-02}

- бһҗбҹ’бһҳбһёбҹ– бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ–бһ бһ»бһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бһҳбҹ’бһӣбҹӮбһ„бһўбһҸбҹ’бһҡбһ¶бһ”бҹ’бһҸбһјбһҡбһ”бҹ’бһҡбһ¶бһҖбҹӢбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·
- бһҗбҹ’бһҳбһёбҹ– бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғбһҮбҹҶбһ“бһҪбһҹбһҸбһ¶бһҳбһ•бһӣбһ·бһҸбһ•бһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбһҳбҹ’бһӣбҹғбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҸбһ¶бһҳбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺ
- бһҗбҹ’бһҳбһёбҹ– бһңбһ¶бһӣбһҮбҹ’бһҡбһҫбһҹбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһ–бҹҒбһӣбһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӣбҹҶбһ бһјбһҡбһ‘бһјбһ‘бһ¶бһҸбҹӢ
- бһҗбҹ’бһҳбһёбҹ– бһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһўбһҸбҹ’бһҡбһ¶бһ”бҹ’бһҸбһјбһҡбһ”бҹ’бһҡбһ¶бһҖбҹӢбҹ– ECB, Frankfurter, Stripe, бһ“бһ·бһ„бһҠбҹ„бһҷбһҠбҹғ
- бһҗбҹ’бһҳбһёбҹ– бһҖбһ¶бһҡбһҡбһҖбһғбһҫбһүбһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ‘бһёбһҸбһ¶бҹҶбһ„бһ—бһјбһҳбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ
- бһҗбҹ’бһҳбһёбҹ– бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺ бһ“бһ·бһ„бһўбһҸбҹ’бһҡбһ¶бһ”бҹ’бһҸбһјбһҡбһ”бҹ’бһҡбһ¶бһҖбҹӢ
