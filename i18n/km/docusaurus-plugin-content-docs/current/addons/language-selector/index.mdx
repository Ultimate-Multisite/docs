# ឧបករណ៍ជ្រើសរើសភាសា {#language-selector}

Ultimate Multisite: ឧបករណ៍ជ្រើសរើសភាសានៅពេលចុះឈ្មោះ

បន្ថែមវាលជ្រើសរើសភាសាទៅក្នុងលំហូរចុះឈ្មោះ។

## ការដំឡើង {#installation}

1. ផ្ទុកឡើង 'ultimate-multisite-language-selector' ទៅកាន់ថត '/wp-content/plugins/'
2. ធ្វើឱ្យ plugin សកម្មតាមរយៈម៉ឺនុយ 'Plugins' ក្នុង WordPress Network Admin របស់អ្នក

## មគ្គុទ្ទេសក៍ការប្រើប្រាស់ {#usage-guide}

ជាមួយ add-on ឥតគិតថ្លៃនេះ អ្នកអាចឱ្យអ្នកប្រើរបស់អ្នកជ្រើសរើសភាសាដែលពួកគេចង់ឃើញនៅលើលំហូរចុះឈ្មោះ និងនៅលើទំព័រចូល។

បន្ទាប់ពី [ធ្វើឱ្យអាជ្ញាបណ្ណរបស់អ្នកសកម្ម](../../user-guide/getting-started/installing-ultimate-multisite.md) អ្នកនឹងអាចដំឡើង add-ons របស់ Ultimate Multisite ដែលអាជ្ញាបណ្ណរបស់អ្នកផ្ដល់សិទ្ធិចូលប្រើ។

ចូលទៅកាន់ទំព័រការកំណត់ Ultimate Multisite របស់អ្នក ហើយបន្ទាប់មកនៅលើម៉ឺនុយខាងឆ្វេង ចុចស្លាក _Add-ons_។

![ទំព័រ Add-ons](/img/admin/addons-page.png)

_ចូលទៅកាន់ទំព័រ Add-ons ដើម្បីចាប់ផ្តើមដំឡើងឧបករណ៍ជ្រើសរើសភាសា_

នៅទីនោះ អ្នកនឹងឃើញ add-ons ទាំងអស់ដែលយើងផ្តល់ជូន។

add-ons ឥតគិតថ្លៃមានសម្រាប់អតិថិជនទាំងអស់របស់យើង មិនថាពួកគេចុះឈ្មោះជាវគម្រោងណាក៏ដោយ។

ចុចប៊ូតុង _Install it Now!_ នៅខាងក្រោមរូបភាពតូចរបស់ add-on។ បន្ទាប់មកធ្វើឱ្យវាសកម្មនៅលើទំព័រ Plugins។

![ទំព័រ Add-ons ដែលមានប៊ូតុងដំឡើង](/img/admin/addons-page.png)

_ចុចប៊ូតុងដើម្បីដំឡើង add-on_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_ធ្វើឱ្យវាសកម្មនៅលើទំព័រ Plugins របស់អ្នក_

ឥឡូវនេះ ចូលទៅកាន់ម៉ឺនុយការកំណត់របស់ Ultimate Multisite ហើយបើកផ្ទាំង _Network Settings_។ នៅទីនោះ នៅផ្នែកខាងក្រោមនៃទំព័រ អ្នកនឹងឃើញផ្នែកថ្មីមួយឈ្មោះ _Languages_។

![ទំព័រការកំណត់](/img/config/settings-general.png)

_ផ្នែកថ្មីស្រឡាងមួយលេចឡើង!_

អ្នកអាចទុកវាទទេដើម្បីបង្ហាញភាសាទាំងអស់ជាជម្រើស ឬជ្រើសរើសភាសាដែលអ្នកចង់ធ្វើឱ្យមាន។ ភាសាទាំងនោះគឺជាភាសាដែលអ្នកបានដំឡើងក្នុងការរៀបចំ WordPress របស់អ្នក។

_បញ្ជីភាសាដែលមាននៅលើការដំឡើង WP នេះ_

បន្ទាប់ពីរក្សាទុកការផ្លាស់ប្តូរដែលអ្នកបានធ្វើ ម៉ឺនុយជ្រើសរើសមួយដែលមានភាសាដែលអ្នកបានជ្រើស នឹងត្រូវបានបង្ហាញនៅលើទំព័រចូល និងនៅលើលំហូរចុះឈ្មោះ។

![ការកំណត់ការចូល និងការចុះឈ្មោះ](/img/config/settings-login-registration.png)

## កំណត់ភាសាតាមរយៈ URL {#setting-the-language-via-the-url}

អ្នកក៏អាចប្ដូរភាសារបស់ទំព័រមួយបាន ដោយឆ្លងប៉ារ៉ាម៉ែត្រ locale ទៅ URL។ ប្រសិនបើអ្នកចង់បញ្ជូនបន្តទៅទំព័រចុះឈ្មោះជាភាសាអ៊ីតាលី ឧទាហរណ៍ សូមបន្ថែម ?locale=it_IT ទៅ URL។

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

ភាសានោះត្រូវតែបានដំឡើងនៅលើបណ្តាញ WordPress របស់អ្នក ដើម្បីឱ្យវាដំណើរការ។
