# តម្រង៖ wu_cap_captcha_config

តម្រងការកំណត់​រចនាសម្ព័ន្ធ Cap Captcha។

អនុញ្ញាតឱ្យអ្នកអភិវឌ្ឍន៍កែសម្រួលការកំណត់ Cap Captcha។

## ប៉ារ៉ាម៉ែត្រ

| ឈ្មោះ | ប្រភេទ | ការពិពណ៌នា |
|------|------|-------------|
| $config | `array` | អារេការកំណត់រចនាសម្ព័ន្ធ Cap។ |
| $level | `string` | កម្រិតសុវត្ថិភាព (លឿន មធ្យម អតិបរមា)។ |
