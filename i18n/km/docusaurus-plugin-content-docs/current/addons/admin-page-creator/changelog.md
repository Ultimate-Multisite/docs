* បានកែលម្អ៖ ការរៀបចំ និង​បង្រួម​ទំហំ JavaScript asset
* បានកែលម្អ៖ ភាព​ឆបគ្នា​ជាមួយ WordPress ចុងក្រោយ និង​កំណែ page builder
