នៅលើផ្ទាំង CSS អ្នកអាចបន្ថែមកូដ CSS ផ្ទាល់ខ្លួនរបស់អ្នក ព្រមទាំងនាំចូលឯកសារ CSS ពីគេហទំព័រពីចម្ងាយ (ដូចជា CDNs, Google Fonts ជាដើម)។

## សិទ្ធិ

នៅក្នុងជម្រើសកម្រិតខ្ពស់ផងដែរ អ្នកនឹងអាចជ្រើសថាតើគេហទំព័ររងណាខ្លះនឹងមានសិទ្ធិចូលប្រើទំព័រនេះ។ វាមានប្រយោជន៍ ប្រសិនបើអ្នកមានគម្រោងផ្តល់មាតិកាផ្តាច់មុខផ្សេងៗគ្នាទៅកម្រិតគម្រោងផ្សេងៗគ្នា ឧទាហរណ៍។

_អ្នកមានការគ្រប់គ្រងពេញលេញលើកន្លែងដែលទំព័រនេះនឹងបង្ហាញ_

**សំខាន់** : លក្ខខណ្ឌទាំងពីរត្រូវតែបំពេញ ដើម្បីឲ្យទំព័រត្រូវបានបង្ហាញ។ នោះមានន័យថា ប្រសិនបើអ្នកប្រើជា client របស់ plan ជាក់លាក់មួយ ប៉ុន្តែមិនមាន roles មួយក្នុងចំណោម roles ដែលបានជ្រើស (ឬផ្ទុយទៅវិញ) ទំព័រនឹងមិនបង្ហាញនៅលើ panel របស់ពួកគេទេ។
