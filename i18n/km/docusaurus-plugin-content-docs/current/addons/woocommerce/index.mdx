# ការរួមបញ្ចូល WooCommerce

ការរួមបញ្ចូល WooCommerce ដំណើរការដូចគ្នាច្រើននឹងវិធីដែលច្រកទូទាត់ដោយដៃរបស់យើងដំណើរការ។ វាផ្តល់ភាពបត់បែនច្រើនដល់អ្នកប្រើរបស់អ្នក ដោយអនុញ្ញាតឱ្យពួកគេជ្រើសរើសទម្រង់ទូទាត់ណាមួយដែលមាននៅលើការដំឡើង WooCommerce របស់អ្នក។ ហើយវាងាយស្រួល៖ នៅពេលរយៈពេលចេញវិក្កយបត្ររបស់អ្នកប្រើបញ្ចប់ ការជាវរបស់អ្នកប្រើនឹងត្រូវដាក់ឱ្យផ្អាក។ ក្នុងពេលជាមួយគ្នា ការបញ្ជាទិញមួយនឹងត្រូវបានបង្កើតនៅលើការដំឡើង WooCommerce របស់គេហទំព័រចម្បងក្នុងបណ្តាញ។ បន្ទាប់មកអ្នកប្រើនឹងទទួលបានវិក្កយបត្រក្នុងអ៊ីមែលរបស់ពួកគេ ជាមួយតំណសម្រាប់ធ្វើការទូទាត់។ នៅពេលទទួលបានការទូទាត់ ការជាវនឹងត្រូវបានបន្ត។

## លក្ខណៈពិសេស {#features}

- ការរួមបញ្ចូលងាយស្រួលរវាង Ultimate Multisite និង WooCommerce
- មានវិធីទូទាត់ជាច្រើន
- ការចេញវិក្កយបត្រកើតឡើងវិញដោយស្វ័យប្រវត្តិ ដោយប្រើស្នូលការជាវដែលរួមបញ្ចូលមកជាមួយ និងអាចបន្ថែមការកែលម្អពី plugin WooCommerce Subscriptions តាមជម្រើស
- សរុបពេលទូទាត់រួមបញ្ចូលពន្ធ Ultimate Multisite ជាបន្ទាត់ថ្លៃសេវា WooCommerce ដាច់ដោយឡែក និងមិនជាប់ពន្ធ ដើម្បីឱ្យអតិថិជនអាចពិនិត្យចំនួនពន្ធមុនពេលទូទាត់

## ឯកសារ {#documentation}

**ការរៀបចំការរួមបញ្ចូល WooCommerce**

**ពង្រីកជម្រើសទូទាត់របស់អ្នកជាមួយ Ultimate Multisite៖ ការរួមបញ្ចូល WooCommerce**

## ការដំឡើង {#installation}

1. អាប់ឡូត 'ultimate-multisite-woocommerce' ទៅថត '/wp-content/plugins/'
2. ធ្វើឱ្យ plugin សកម្មតាមម៉ឺនុយ 'Plugins' ក្នុង WordPress
3. អនុវត្តតាម Wizard ជាជំហានៗ ដើម្បីរៀបចំ plugin

## មគ្គុទ្ទេសក៍ប្រើប្រាស់ {#usage-guide}

_**ចំណាំសំខាន់៖ អត្ថបទនេះសំដៅលើ Ultimate Multisite កំណែ 2.x។**_

_**ប្រយ័ត្ន៖** Ultimate Multisite: WooCommerce Integration ត្រូវការ WooCommerce ឱ្យបានសកម្មយ៉ាងហោចណាស់នៅលើគេហទំព័រចម្បងរបស់អ្នក។_

យើងយល់ថា _Stripe_ និង _PayPal_ មិនមាននៅក្នុងប្រទេសមួយចំនួន ដែលកំណត់ ឬរារាំងអ្នកប្រើ Ultimate Multisite មិនឱ្យប្រើ plugin របស់យើងបានប្រកបដោយប្រសិទ្ធភាព។ ដូច្នេះយើងបានបង្កើត **add-on ដើម្បីរួមបញ្ចូល _WooCommerce_**។

_WooCommerce_ គឺជា plugin ពាណិជ្ជកម្មអេឡិចត្រូនិកដែលពេញនិយមខ្លាំង។ អ្នកអភិវឌ្ឍន៍ជុំវិញពិភពលោកបានបង្កើត add-on ដើម្បីរួមបញ្ចូលច្រកទូទាត់ផ្សេងៗទៅក្នុងវា។ យើងបានប្រើអត្ថប្រយោជន៍នេះ ដើម្បីពង្រីកច្រកទូទាត់ដែលអ្នកអាចប្រើក្នុងប្រព័ន្ធចេញវិក្កយបត្រ Ultimate Multisite។

## ការរៀបចំវា {#setting-it-up}

ដំបូង អ្នកត្រូវដំឡើង add-on ការរួមបញ្ចូល WooCommerce ដែល **ឥតគិតថ្លៃសម្រាប់អ្នកប្រើដែលមានអាជ្ញាបណ្ណ**។ ដើម្បីដំឡើងវា ចុចលើ Ultimate Multisite នៅលើ dashboard របស់ super admin របស់អ្នក ហើយជ្រើស **Add-ons**។

![ទំព័រ Add-ons](/img/admin/addons-page.png)

នៅទីនេះ អ្នកអាចរកឃើញ add-on ទាំងអស់របស់ Ultimate Multisite។ ចុចលើ add-on **Ultimate Multisite: WooCommerce Integration**។

![ទំព័រ Add-ons ដែលបង្ហាញការរួមបញ្ចូល WooCommerce](/img/admin/addons-page.png)

បង្អួចមួយនឹងលេចឡើងជាមួយព័ត៌មានលម្អិតរបស់ add-on។ គ្រាន់តែចុចលើ **ដំឡើងឥឡូវនេះ** *។*

បន្ទាប់ពីដំឡើងរួច អ្នកនឹងត្រូវបានបញ្ជូនបន្តទៅទំព័រ plugin។ នៅទីនេះ គ្រាន់តែចុចលើ **ធ្វើឱ្យសកម្មលើបណ្តាញ** ហើយ add-on _WooCommerce_ នឹងត្រូវបានធ្វើឱ្យសកម្មនៅលើបណ្តាញរបស់អ្នក។

បន្ទាប់ពីធ្វើឱ្យវាសកម្ម ប្រសិនបើអ្នកនៅមិនទាន់បានដំឡើង និងធ្វើឱ្យ plugin WooCommerce សកម្មនៅលើគេហទំព័ររបស់អ្នកទេ **អ្នកនឹងទទួលបានការរំលឹក**។

<!-- រូបថតអេក្រង់មិនមាន៖ សេចក្តីជូនដំណឹង admin រំលឹកឱ្យដំឡើង និងធ្វើឱ្យ plugin WooCommerce សកម្ម -->

នៅពេលដំឡើងរួច អ្នកនឹងឃើញជម្រើសបន្ថែមមួយក្រោមការកំណត់ការទូទាត់ Ultimate Multisite។ បន្ទាប់មកអ្នកអាច **បើកប្រើ** ជម្រើស _WooCommerce_ ពីទីនោះ។

![ការកំណត់ច្រកទូទាត់ Ultimate Multisite ដែលបានបើក WooCommerce](/img/addons/woocommerce-payment-gateway.png)

អ្នកអាចកំណត់ការកំណត់មូលដ្ឋានមួយចំនួន ដូចជាឈ្មោះបង្ហាញដែលនឹងបង្ហាញនៅផ្នែកខាងមុខ និងសេចក្តីណែនាំដើម្បីណែនាំអ្នកប្រើរបស់អ្នកក្នុងលំហូរទូទាត់របស់ពួកគេ។ បន្ទាប់មកអ្នកអាចទៅកាន់ **ការកំណត់ _WooCommerce_ របស់គេហទំព័រចម្បងរបស់អ្នក** ដើម្បីបើក និងកំណត់ច្រកទូទាត់ដែលអ្នកចង់ប្រើ។

### បន្ទាត់ថ្លៃសេវាពន្ធពេលទូទាត់ {#checkout-tax-fee-lines}

នៅពេលការទូទាត់ Ultimate Multisite រួមបញ្ចូលពន្ធ ការរួមបញ្ចូលនេះនឹងបន្ថែមចំនួនពន្ធ Ultimate Multisite នីមួយៗទៅក្នុងការទូទាត់ WooCommerce ជាបន្ទាត់ថ្លៃសេវាដាច់ដោយឡែក និងមិនជាប់ពន្ធ។ វាធ្វើឱ្យសរុបពន្ធអាចមើលឃើញមុនពេលទូទាត់ ខណៈពេលការពារ WooCommerce មិនឱ្យគិតពន្ធលើថ្លៃសេវាពន្ធម្តងទៀត។

## របៀបគ្រប់គ្រងការទូទាត់ {#how-to-manage-the-payment}

ការរួមបញ្ចូល WooCommerce ដំណើរការដូចគ្នាច្រើននឹងវិធីដែល **ច្រកទូទាត់ដោយដៃ** របស់យើងដំណើរការ។ គុណវិបត្តិគឺថា ការទូទាត់នីមួយៗត្រូវតែបង់ដោយដៃនៅចុងរយៈពេលចេញវិក្កយបត្រ។ នេះជារបៀបដែលលំហូរដំណើរការ៖

  * នៅពេលវដ្តចេញវិក្កយបត្ររបស់អ្នកប្រើបញ្ចប់ សមាជិកភាពនឹងត្រូវដាក់ **ឱ្យផ្អាក**។ ក្នុងពេលជាមួយគ្នា ការបញ្ជាទិញមួយនឹងត្រូវបានបង្កើតនៅលើការដំឡើង _WooCommerce_ នៅលើគេហទំព័រចម្បងរបស់បណ្តាញអ្នក។

  * បន្ទាប់មកអ្នកប្រើនឹងទទួលបានវិក្កយបត្រក្នុងអ៊ីមែលរបស់ពួកគេ ជាមួយតំណសម្រាប់ធ្វើការទូទាត់ (តំណនេះក៏បង្ហាញក្នុងប្រវត្តិចេញវិក្កយបត្ររបស់ពួកគេពី dashboard គេហទំព័ររងផងដែរ)។ តំណនោះបញ្ជូនអ្នកប្រើទៅកាន់ **ការទូទាត់** របស់ _**WooCommerce**_ ដែលអ្នកប្រើអាចជ្រើសរើស **ច្រកទូទាត់ _WooCommerce_ ដែលមាន** មួយ ដើម្បីដំណើរការការទូទាត់របស់ពួកគេ។

  * នៅពេលទទួលបានការទូទាត់ សមាជិកភាពនឹងត្រូវបាន **បន្ត** ការទូទាត់ត្រូវបានកត់ត្រានៅលើ Ultimate Multisite ហើយការបញ្ជាទិញត្រូវបានកំណត់ថាបានបញ្ចប់។

### ចំណាំសំខាន់មួយចំនួន៖ {#some-important-notes}

  * មុខងារលំនាំដើមរបស់ Ultimate Multisite នៅតែដំណើរការជាមួយទម្រង់រួមបញ្ចូលនេះ។ ឧទាហរណ៍ **ការចេញប្រាក់សងវិញ** ពីអេក្រង់ **ការគ្រប់គ្រងការជាវ** នឹងទាក់ទងជាមួយ WooCommerce ដើម្បីចេញប្រាក់សងវិញដោយប្រើទម្រង់ទូទាត់ដែលបានជ្រើស។
