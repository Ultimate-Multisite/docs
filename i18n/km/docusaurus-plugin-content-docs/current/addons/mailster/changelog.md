# កំណត់ហេតុផ្លាស់ប្តូរ Mailster

### 1.0.0
* ការចេញផ្សាយដំបូង
* ធ្វើសមកាលកម្មអ្នកជាវដោយស្វ័យប្រវត្តិ ពេលចុះឈ្មោះ ឬបង់ប្រាក់
* ការកំណត់បញ្ជីតាមផលិតផលជាក់លាក់
* ការជ្រើសចូលដោយប្រើប្រអប់ធីក សម្រាប់ការអនុលោមតាម GDPR
* គាំទ្រការជ្រើសចូលពីរដង
* ការផ្គូផ្គងវាលយ៉ាងទូលំទូលាយ
* ការកត់ត្រាលម្អិត
