# ភ្នាក់ងារគាំទ្រ

ការគ្រប់គ្រងបណ្តាញទាំងមូលដោយខ្លួនឯងមិនមែនជាកិច្ចការងាយស្រួលទេ។ ពេលបណ្តាញរបស់អ្នកចាប់ផ្តើមរីកចម្រើន អ្នកនឹងចាប់ផ្តើមឃើញតម្រូវការគាំទ្រកាន់តែកើនឡើង។ វាជាគំនិតល្អក្នុងការនាំមនុស្សមកជួយបន្ថយបន្ទុកការងាររបស់អ្នក។ ទោះយ៉ាងណា ការផ្តល់សិទ្ធិចូលប្រើប្រាស់/ការអនុញ្ញាតពេញលេញមានហានិភ័យ ហើយអាចប៉ះពាល់ដល់បណ្តាញរបស់អ្នក។ នេះហើយជាកន្លែងដែលកម្មវិធីបន្ថែម​ភ្នាក់ងារគាំទ្រនេះចូលមកជួយ។

កម្មវិធីបន្ថែមនេះអនុញ្ញាតឱ្យអ្នកបង្កើតប្រភេទអ្នកប្រើ WordPress ថ្មីមួយ ដែលហៅថា ភ្នាក់ងារគាំទ្រ ហើយផ្តល់ឱ្យអ្នកនូវការគ្រប់គ្រងពេញលេញលើសិទ្ធិដែលអ្នកប្រើនោះមាន។ ការផ្តល់សិទ្ធិងាយស្រួលដូចការបើកប៊ូតុងបិទបើក ហើយបិទវាវិញ នៅពេលអ្នកសម្រេចចិត្តដកសិទ្ធិនោះចេញ។

## លក្ខណៈពិសេស {#features}

- បញ្ជីជាធាតុៗនៃសិទ្ធិ WordPress និង Ultimate Multisite
- បើក និងបិទការចូលប្រើប្រាស់បានយ៉ាងងាយស្រួល
- បង្កើត Account ភ្នាក់ងារចំនួនគ្មានកំណត់

## ការដំឡើង {#installation}

1. ផ្ទុកឡើង 'ultimate-multisite-support-agents' ទៅកាន់ថត '/wp-content/plugins/'
2. ធ្វើឱ្យកម្មវិធីបន្ថែមសកម្មតាមម៉ឺនុយ 'កម្មវិធីបន្ថែម' ក្នុង WordPress
3. អនុវត្តតាមជំនួយការណែនាំជាជំហានៗ ដើម្បីរៀបចំកម្មវិធីបន្ថែមនេះ
