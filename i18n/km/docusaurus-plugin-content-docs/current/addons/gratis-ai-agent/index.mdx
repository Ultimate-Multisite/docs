# Gratis AI Agent {#gratis-ai-agent}

Gratis AI Agent គឺជា​ជំនួយការ AI ស្វ័យប្រវត្តិ​សម្រាប់ WordPress ដែល​អាច​រៀបចំផែនការ បង្កើត និង​គ្រប់គ្រង​គេហទំព័រ WordPress ពេញលេញ តាម​រយៈ​ការណែនាំ​ជា​ភាសា​ធម្មជាតិ។ វា​បង្ហាញ​សំណុំ **សមត្ថភាព** ដែល​មាន​រចនាសម្ព័ន្ធ — សកម្មភាព​ដាច់ដោយឡែក និង​អាច​សាកល្បង​បាន ដែល agent អាច​ហៅ​ប្រើ — គ្របដណ្តប់​ចាប់ពី​ការគ្រប់គ្រង​មាតិកា រហូតដល់​ការប្ដូរ​ប្រព័ន្ធ​រចនា​តាម​តម្រូវការ។ កំណែ 1.9.0 បន្ថែម​សមត្ថភាព​បង្កើត​មាតិកា ការពិនិត្យ​ទំព័រ​ដោយ​មើលឃើញ​តាម​រយៈ​រូបថត​អេក្រង់​ខាង client ទម្រង់ agent ភ្ជាប់មក​ស្រាប់​ចំនួន​ប្រាំ និង​សញ្ញាបិទបើក​មុខងារ​សម្រាប់​ការគ្រប់គ្រង​សិទ្ធិ​ចូលប្រើ និង​ម៉ាកសញ្ញា។

## លក្ខណៈពិសេស​សំខាន់ៗ {#key-features}

- **ប្រព័ន្ធ​សមត្ថភាព** — កាតាឡុក​សកម្មភាព​បែប​ម៉ូឌុល និង​អាច​ពង្រីក​បាន ដែល agent អាច​ហៅ​ប្រើ​លើ​ការដំឡើង WordPress របស់​អ្នក
- **ការគ្រប់គ្រង​មាតិកា** — បង្កើត​ប្រកាស និង​ទំព័រ (រួមទាំង​ការកំណត់​គំរូ​ទំព័រ) បង្កើត​ប្រកាស​ច្រើន​ជា​ក្រុម​ក្នុង​ការហៅ​មួយ កំណត់​រូបភាព​លេចធ្លោ និង​បង្កើត​ទម្រង់​ទំនាក់ទំនង
- **ការពិនិត្យ​ដោយ​មើលឃើញ** — ចាប់យក​រូបថត​អេក្រង់​នៃ​ទំព័រ​ផ្ទាល់ ប្រៀបធៀប​ស្ថានភាព​មុន/ក្រោយ និង​ទទួល​ការពិនិត្យ​រចនា​ដែល​បង្កើត​ដោយ AI គ្របដណ្តប់​លើ​ប្លង់ អក្សរសិល្ប៍ ពណ៌ និង​ភាពងាយស្រួល​ប្រើប្រាស់
- **agent ភ្ជាប់មក​ស្រាប់​ចំនួន​ប្រាំ** — អ្នកសរសេរ​មាតិកា អ្នកបង្កើត​គេហទំព័រ ស្ទូឌីយោ​រចនា អ្នកគ្រប់គ្រង plugin និង​ជំនួយការ​គាំទ្រ; អាច​ប្ដូរ​បាន​តាម​រយៈ​ឧបករណ៍​ជ្រើស agent នៅ​ក្បាល​ជជែក
- **សញ្ញាបិទបើក​មុខងារ** — ប៊ូតុង​បិទបើក​ការគ្រប់គ្រង​សិទ្ធិ​ចូលប្រើ និង​ម៉ាកសញ្ញា​ក្នុង ការកំណត់ → សញ្ញាបិទបើក​មុខងារ សម្រាប់​ការរឹតបន្តឹង​តួនាទី ការដាក់ឈ្មោះ​បែប white-label និង​ការប្ដូរ widget តាម​តម្រូវការ
- **ការគ្រប់គ្រង​ប្រភេទ​ប្រកាស​ផ្ទាល់ខ្លួន** — ចុះឈ្មោះ រាយបញ្ជី និង​លុប​ប្រភេទ​ប្រកាស​ផ្ទាល់ខ្លួន ដោយ​រក្សាទុក​ជាប់លាប់​ក្រោយ​ចាប់ផ្ដើម​ឡើងវិញ
- **ការគ្រប់គ្រង​ចំណាត់ថ្នាក់​ផ្ទាល់ខ្លួន** — ចុះឈ្មោះ រាយបញ្ជី និង​លុប​ចំណាត់ថ្នាក់​ផ្ទាល់ខ្លួន ដោយ​រក្សាទុក​ជាប់លាប់
- **សមត្ថភាព​ប្រព័ន្ធ​រចនា** — បញ្ចូល CSS ផ្ទាល់ខ្លួន គ្រប់គ្រង​លំនាំ​ប្លុក​ដែល​បាន​ជ្រើសរើស កំណត់​ឡូហ្គោ​គេហទំព័រ និង​អនុវត្ត preset របស់ theme.json
- **ការគ្រប់គ្រង​រចនាប័ទ្ម​សកល** — អាន និង​សរសេរ​តម្លៃ theme.json សម្រាប់​បណ្ដុំ​ពណ៌ អក្សរសិល្ប៍ ចន្លោះ និង​ផ្សេងៗ​ទៀត
- **ការគ្រប់គ្រង​ម៉ឺនុយ​នាវីហ្គេសិន** — បង្កើត និង​ធ្វើបច្ចុប្បន្នភាព​ម៉ឺនុយ​នាវីហ្គេសិន និង​ធាតុ​របស់វា​តាម​កម្មវិធី
- **ការគ្រប់គ្រង​ជម្រើស** — អាន និង​សរសេរ​ជម្រើស WordPress ជាមួយ​បញ្ជី​ទប់ស្កាត់​សុវត្ថិភាព​ភ្ជាប់មក​ស្រាប់ ដើម្បី​ការពារ​ការកំណត់​សំខាន់ៗ
- **បញ្ជីចុះឈ្មោះ​សមត្ថភាព​ដែល​អាច​ដំឡើង​បាន** — រកឃើញ និង​ធ្វើឱ្យ​សកម្ម​កញ្ចប់​សមត្ថភាព​បន្ថែម ដោយ​មិន​ប្ដូរ​ឯកសារ plugin ស្នូល
- **ការសម្របសម្រួល​អ្នកបង្កើត​គេហទំព័រ v2** — ផែនការ​បង្កើត​គេហទំព័រ​ច្រើន​ជំហាន ជាមួយ​ការស្វែងរក plugin ការតាមដាន​វឌ្ឍនភាព និង​ការស្ដារ​កំហុស
- **ពាក្យបញ្ជា benchmark របស់ WP-CLI** — ដំណើរការ​សំណុំ benchmark សមត្ថភាព agent ពី​បន្ទាត់​ពាក្យបញ្ជា សម្រាប់ CI និង​ការវាយតម្លៃ​ម៉ូដែល

## តម្រូវការ {#requirements}

- WordPress 6.0 ឬ​ខ្ពស់ជាងនេះ
- PHP 8.1 ឬ​ខ្ពស់ជាងនេះ
- WordPress Multisite (បាន​ធ្វើឱ្យ​សកម្ម​លើ​បណ្ដាញ) ឬ​ការដំឡើង​គេហទំព័រ​តែមួយ
- plugin តភ្ជាប់​អ្នកផ្តល់​សេវា AI សកម្ម​មួយ (Anthropic, OpenAI, ឬ​ដែល​ឆបគ្នា)

## ការដំឡើង {#installation}

1. ផ្ទុក​ឯកសារ plugin ឡើង​ទៅ `/wp-content/plugins/gratis-ai-agent`
2. ធ្វើឱ្យ plugin សកម្ម​តាម​អេក្រង់ **Plugins** ក្នុង WordPress
3. ចូលទៅ **Gratis AI Agent → ការកំណត់** ហើយ​តភ្ជាប់​អ្នកផ្តល់​សេវា AI របស់​អ្នក
4. (ជម្រើស) រុករក **បញ្ជីចុះឈ្មោះ​សមត្ថភាព** ដើម្បី​ដំឡើង​កញ្ចប់​សមត្ថភាព​បន្ថែម

## ចាប់ផ្ដើម​រហ័ស {#quick-start}

បន្ទាប់ពី​បាន​ធ្វើឱ្យ​សកម្ម និង​តភ្ជាប់​ទៅ​អ្នកផ្តល់​សេវា AI សូម​បើក​ផ្ទាំង **Gratis AI Agent** ក្នុង​របារ​ចំហៀង​គ្រប់គ្រង WordPress។ វាយ​ការណែនាំ​ជា​ភាសា​ធម្មជាតិ ហើយ agent នឹង​រៀបចំផែនការ និង​អនុវត្ត​ជំហាន​ដែល​ត្រូវការ ដោយ​ប្រើ​សមត្ថភាព​ដែល​មាន។


## ទិដ្ឋភាពទូទៅ​អំពី​សមត្ថភាព {#abilities-overview}

| **ចំណាត់ថ្នាក់​ផ្ទាល់ខ្លួន** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **ប្រព័ន្ធ​រចនា** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |

| **អ្នកបង្កើត​គេហទំព័រ** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

## WP-CLI {#wp-cli}

Gratis AI Agent ភ្ជាប់មក​ជាមួយ​គ្រួសារ​ពាក្យបញ្ជា `wp gratis-ai-agent` សម្រាប់​ដំណើរការ benchmarks និង​គ្រប់គ្រង​សមត្ថភាព​ពី​បន្ទាត់​ពាក្យបញ្ជា។ សូមមើល [ឯកសារ​យោង WP-CLI](./wp-cli) សម្រាប់​បញ្ជី​ពាក្យបញ្ជា​ពេញលេញ។

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## បញ្ជី​ទប់ស្កាត់​សុវត្ថិភាព​សម្រាប់​ការគ្រប់គ្រង​ជម្រើស {#options-management-safety-blocklist}

សមត្ថភាពគ្រប់គ្រងជម្រើសរួមបញ្ចូលបញ្ជីទប់ស្កាត់ក្នុងខ្លួន ដែលការពារ agent មិនឱ្យអាន ឬសរសេរជម្រើស WordPress ដែលរសើប។ បញ្ជីទប់ស្កាត់លំនាំដើមគ្របដណ្តប់លើ៖

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (ជម្រើស URL — ប្រើឧបករណ៍ផ្លាស់ទីទិន្នន័យជាក់លាក់សម្រាប់ចំណុចទាំងនេះ)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (កូនសោសម្ងាត់)
- `active_plugins`, `template` (ការធ្វើឱ្យ plugin/theme សកម្ម — ប្រើសមត្ថភាពជាក់លាក់)
- ព័ត៌មានសម្ងាត់របស់ច្រកទូទាត់ និង API keys ដែលបានចុះបញ្ជីដោយ plugins ដែលបានដំឡើង

ធាតុបន្ថែមអាចត្រូវបានបន្ថែមតាមរយៈ filter `gratis_ai_agent_options_blocklist`។

## Hooks និង Filters {#hooks-and-filters}

### Actions {#actions}

- `gratis_ai_agent_ability_registered` — ត្រូវបានដំណើរការបន្ទាប់ពីសមត្ថភាពមួយត្រូវបានចុះបញ្ជី ហើយទទួលឈ្មោះ class របស់សមត្ថភាព
- `gratis_ai_agent_plan_started` — ត្រូវបានដំណើរការនៅពេលផែនការបង្កើតគេហទំព័រចាប់ផ្តើមអនុវត្ត
- `gratis_ai_agent_plan_completed` — ត្រូវបានដំណើរការនៅពេលផែនការបញ្ចប់ដោយជោគជ័យ
- `gratis_ai_agent_plan_failed` — ត្រូវបានដំណើរការនៅពេលផែនការជួបកំហុសដែលមិនអាចស្តារឡើងវិញបាន

### Filters {#filters}

- `gratis_ai_agent_abilities` — កែប្រែបញ្ជីសមត្ថភាពដែលបានចុះបញ្ជី មុនពេល agent ផ្ទុកវា
- `gratis_ai_agent_options_blocklist` — ពង្រីក ឬកាត់បន្ថយបញ្ជីទប់ស្កាត់សុវត្ថិភាពរបស់ជម្រើស
- `gratis_ai_agent_benchmark_questions` — បន្ថែមសំណួរវាស់ស្ទង់ផ្ទាល់ខ្លួនទៅក្នុងសំណុំវាយតម្លៃ
- `gratis_ai_agent_plan_plugins` — កែប្រែបញ្ជីស្វែងរក plugin កំឡុងពេលបង្កើតផែនការគេហទំព័រ
