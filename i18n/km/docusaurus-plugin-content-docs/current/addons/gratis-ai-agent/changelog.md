- **សមត្ថភាព `create_contact_form`** — បង្កើតទម្រង់ទំនាក់ទំនងដោយប្រើ plugin ទម្រង់ដែលកំពុងដំណើរការ (Contact Form 7, WPForms, Fluent Forms ឬ Gravity Forms) ហើយត្រឡប់ shortcode ដែលរួចរាល់សម្រាប់បង្កប់ក្នុងអត្ថបទ ឬទំព័រណាមួយ។
- **សមត្ថភាព `set_featured_image`** — កំណត់រូបភាពលេចធ្លោឱ្យអត្ថបទ ឬទំព័រ ពីលេខសម្គាល់ឯកសារភ្ជាប់ក្នុងបណ្ណាល័យមេឌៀដែលមានស្រាប់ ឬពី URL ពីចម្ងាយ; នាំចូលរូបភាពដោយស្វ័យប្រវត្តិនៅពេលផ្តល់ URL។
- **សមត្ថភាព `batch_create_posts`** — បង្កើតអត្ថបទច្រើនក្នុងការហៅសមត្ថភាពតែមួយ។ គាំទ្រប៉ារ៉ាម៉ែត្រដូចគ្នានឹង `create_post` រាយការណ៍ជោគជ័យ/បរាជ័យតាមអត្ថបទនីមួយៗ ហើយមានរបៀប `stop_on_error` ជាជម្រើស។
- **ប៉ារ៉ាម៉ែត្រ `page_template`** — បានបន្ថែមទៅ `create_post` និង `update_post`។ កំណត់ឯកសារគំរូទំព័រ PHP (ឧ. `page-full-width.php`) នៅពេលបង្កើត ឬកែប្រែ។ ផ្ញើខ្សែអក្សរទទេទៅ `update_post` ដើម្បីត្រឡប់ទៅលំនាំដើមរបស់ theme វិញ។
- **សមត្ថភាពថតអេក្រង់ផ្នែក client** — `capture_screenshot`, `compare_screenshots`, និង `review_page_design`។ ថតអេក្រង់ពេញ ឬតាម viewport នៃទំព័រដែលកំពុងផ្សាយផ្ទាល់តាមរយៈកម្មវិធីរុករកគ្មានផ្ទាំងបង្ហាញនៅផ្នែក server, ប្រៀបធៀបរូបថតអេក្រង់ពីរ ហើយទទួលបានការពិនិត្យរចនាដែលបង្កើតដោយ AI គ្របដណ្តប់លើប្លង់ អក្សររចនា ពណ៌ និងភាពងាយចូលប្រើ។
- **agent ភ្ជាប់មកស្រាប់ចំនួនប្រាំ** — អ្នកសរសេរមាតិកា, អ្នកបង្កើត site, ស្ទូឌីយោរចនា, អ្នកគ្រប់គ្រង plugin, និងជំនួយការគាំទ្រ។ agent នីមួយៗមានសំណុំឧបករណ៍ជាក់លាក់, system prompt ដែលកែតម្រូវតាមតួនាទី, និងសំណើចាប់ផ្តើម។ អាចប្តូរបានតាម **ឧបករណ៍ជ្រើស agent** ថ្មីនៅក្បាល chat។ មើល [agent ភ្ជាប់មកស្រាប់](../../user-guide/configuration/built-in-agents)។
- **សញ្ញាបើកមុខងារ** — ផ្ទាំង **ការកំណត់ → សញ្ញាបើកមុខងារ** ថ្មី មានប៊ូតុងបិទ/បើកការគ្រប់គ្រងសិទ្ធិចូលប្រើ (កម្រិតត្រឹមអ្នកគ្រប់គ្រង, កម្រិតត្រឹមអ្នកគ្រប់គ្រងបណ្តាញ, ការចូលប្រើរបស់អ្នកជាវ, បិទសម្រាប់អ្នកមិនមែនជាសមាជិក) និងជម្រើសម៉ាកសញ្ញា (លាក់ការបញ្ជាក់ប្រភពនៅ footer, ឈ្មោះ agent ផ្ទាល់ខ្លួន, លាក់ឧបករណ៍ជ្រើស agent, ប្រើ icon របស់ site ជា avatar ក្នុង chat)។ មើល [ការកំណត់ Gratis AI Agent](../../user-guide/administration/gratis-ai-agent-settings)។
- **ស្តារវគ្គចុងក្រោយ** — ផ្ទាំង chat ឥឡូវផ្ទុកឡើងវិញនូវការសន្ទនាថ្មីបំផុតដោយស្វ័យប្រវត្តិ នៅពេលផ្ទុកទំព័រ និងនៅពេលបើក widget ដូច្នេះបរិបទមិនបាត់បង់នៅពេលផ្លាស់ទីរវាងទំព័រ។
- **តំណសកម្មភាព plugin** — តំណរហ័សទៅកាន់ការកំណត់ និងបញ្ជីឈ្មោះសមត្ថភាព ឥឡូវបង្ហាញលើអេក្រង់ WordPress **Plugin → Plugin ដែលបានដំឡើង** ខាងក្រោមសេចក្តីពណ៌នា plugin។

### បានកែលម្អ {#improved}

- **ការព្យាយាមឡើងវិញប្រភពរូបភាព** — agent ឥឡូវព្យាយាមប្រភពរូបភាពឥតគិតថ្លៃដែលបានកំណត់ទាំងអស់ មុននឹងប្តូរទៅរូបភាពដែលបង្កើតដោយ AI នៅពេលទាញយកបរាជ័យ។
- **ផ្ទាំងព័ត៌មាន model** — បង្ហាញជានិច្ចនៅក្បាល chat; មិនលាក់ទៀតបន្ទាប់ពីសារដំបូង។
- **ឥរិយាបថរមូរស្វ័យប្រវត្តិ** — ការរមូរស្វ័យប្រវត្តិផ្អាកនៅពេលអ្នកប្រើរមូរឡើងដើម្បីអាន; ប៊ូតុងអណ្ដែត **រមូរទៅបាត** បង្ហាញឡើង ហើយបិទដោយស្វ័យប្រវត្តិនៅពេលអ្នកប្រើទៅដល់សារចុងក្រោយ។
- **UI ឧបករណ៍ជ្រើស agent** — បានរចនាឡើងវិញជា overlay តារាងទម្រង់ ដែលមាន icon សម្រាប់ agent នីមួយៗ ធ្វើឱ្យងាយស្រួលសម្គាល់ និងប្តូរ agent ភ្លាមៗ។
- **បំណែក JS ផ្ទុកយឺត** — កញ្ចប់ JavaScript ដំបូងរបស់ chat widget ឥឡូវត្រូវបានបំបែកជាបំណែកដែលផ្ទុកយឺត កាត់បន្ថយទំហំកញ្ចប់ដំបូង 75–90%។
- **ការរចនាថ្មីរបស់ chat widget** — icon AI តែមួយជំនួស avatar ផ្ទាល់ខ្លួនមុននេះ; ស្របគ្នាជាមួយប្រព័ន្ធ agent ភ្ជាប់មកស្រាប់។
- **ការបំលែង URL ជាតំណ** — URL ដែលបង្ហាញក្នុងសារប្រព័ន្ធ និងពពុះសារកំហុស ឥឡូវបង្ហាញជាតំណដែលអាចចុចបាន។

### បានជួសជុល {#fixed}

- **ភាពងាយរកឃើញសមត្ថភាព** — បានកែសម្រួលសេចក្តីពណ៌នា, ឯកសារយោង system prompt, និងការតម្រឹម namespace ដើម្បីឱ្យសមត្ថភាពទាំងអស់បង្ហាញក្នុងបញ្ជីឧបករណ៍របស់ agent ដោយទុកចិត្តបាន។
- **ឃ្លាំងសម្ងាត់ providers** — providers ឥឡូវត្រូវបានរក្សាទុកក្នុងឃ្លាំងសម្ងាត់ទូទាំង site តាមរយៈកម្មវិធីរាប់កំណែ ដើម្បីការពារបញ្ហា provider ចាស់នៅលើបណ្តាញ multisite។
- **`ability_invalid_output`** — បានដោះស្រាយនៅទូទាំងកម្មវិធីគ្រប់គ្រងសមត្ថភាព 12; ទាំងអស់ត្រឡប់ចម្លើយ JSON ដែលមានរចនាសម្ព័ន្ធត្រឹមត្រូវ។
- **ខ្សែដំណើរការ `pending_client_tool_calls`** — បានភ្ជាប់ពីដើមដល់ចប់ ដើម្បីឱ្យការហៅឧបករណ៍ផ្នែក client បញ្ចប់បានត្រឹមត្រូវ ហើយត្រឡប់លទ្ធផលទៅ model។
- **ថតប្រវត្តិ** — ការផ្លាស់ប្តូរដែលមិនអាចត្រឡប់វិញបាន ត្រូវបានដកចេញពីបញ្ជីត្រឡប់វិញ; តំណ **មើលប្រវត្តិពេញលេញ** ឥឡូវដំណើរការបានត្រឹមត្រូវ។
- **ប្រព័ន្ធផ្លាស់ប្តូរ/ត្រឡប់វិញ** — បានជួសជុលកំហុសដាច់ដោយឡែកចំនួនប្រាំ ហើយបង្រួបបង្រួមក្រោមចំណុចប្រទាក់អ្នកគ្រប់គ្រងថ្មី។
- **ការជូនដំណឹង Save Settings** — ការជូនដំណឹង snackbar ឥឡូវបង្ហាញដោយទុកចិត្តបាន បន្ទាប់ពីចុច **Save Settings**។
- **ម៉ឺនុយបរិបទធុងសំរាម** — បានបន្ថែមជម្រើស **លុបជាអចិន្ត្រៃយ៍** ដើម្បីឱ្យធាតុអាចត្រូវបានលុបចេញទាំងស្រុង ដោយមិនចាំបាច់ចាកចេញពីទិដ្ឋភាពធុងសំរាម។
- **កែសម្រួល និងផ្ញើឡើងវិញ** — ការចុច icon កែសម្រួល ឥឡូវចូលរបៀបកែសម្រួលសម្រាប់តែសារដែលបានចុចប៉ុណ្ណោះ មិនមែនសារទាំងអស់ក្នុងខ្សែសន្ទនា។
- **កម្ពស់ប្លង់ chat** — ផ្ទាំង chat សម្របកម្ពស់របស់វា នៅពេលមាតិកាដែល plugin បានបញ្ចូល (ការជូនដំណឹងអ្នកគ្រប់គ្រង, បដា) បង្ហាញខាងលើទំព័រ ដើម្បីការពារកុំឱ្យតំបន់បញ្ចូលត្រូវបានរុញចេញក្រៅអេក្រង់។

- **សមត្ថភាពគ្រប់គ្រង Options** — `get_option`, `set_option`, `delete_option`, `list_options`។ រួមមាន blocklist សុវត្ថិភាពដែលមានស្រាប់ ដើម្បីការពារ options សំខាន់ៗរបស់ WordPress ពីការកែប្រែ។

- **Site Builder Orchestration v2** — `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`។ ផែនការបង្កើត site ជាច្រើនដំណាក់កាល ជាមួយការស្វែងរក plugin ឯកសារយោងលទ្ធផលជំហាន ការតាមដានវឌ្ឍនភាព និងការស្តារកំហុសដោយស្វ័យប្រវត្តិ។
