# Mailchimp {#mailchimp}

Multisite Ultimate: ការរួមបញ្ចូល Mailchimp

ភ្ជាប់បណ្តាញ Multisite Ultimate របស់អ្នកទៅ Mailchimp របស់អ្នក។ អ្នកអាចភ្ជាប់គម្រោងជាក់លាក់ទៅនឹងបញ្ជី Mailchimp ហើយបន្ថែមអ្នកជាវទៅបញ្ជីទាំងនោះដោយស្វ័យប្រវត្តិនៅពេលចុះឈ្មោះ។

## ការដំឡើង {#installation}

1. ផ្ទុកឡើង 'ultimate-multisite-mailchimp' ទៅកាន់ថត '/wp-content/plugins/'
2. បើកដំណើរការកម្មវិធីបន្ថែមតាមរយៈម៉ឺនុយ 'កម្មវិធីបន្ថែម' ក្នុង WordPress Network Admin របស់អ្នក

## មគ្គុទ្ទេសក៍ប្រើប្រាស់ {#usage-guide}

ការធ្វើសមកាលកម្មអ៊ីមែលអ្នកជាវជាមួយបញ្ជី MailChimp របស់អ្នកគឺងាយស្រួលណាស់ជាមួយកម្មវិធីបន្ថែម Ultimate Multisite: MailChimp Integration របស់យើង។ មេរៀននេះមានគោលបំណងជួយអ្នករៀបចំអ្វីៗទាំងអស់!

## ការទទួល API key {#getting-an-api-key}

មុនពេលយើងចាប់ផ្តើមមើលការកំណត់ផ្សេងៗដែលមាន យើងត្រូវការ MailChimp API key មួយ។ វានឹងអនុញ្ញាតឱ្យ Ultimate Multisite ទំនាក់ទំនងជាមួយ MailChimp API ដើម្បីទាញយកបញ្ជី ក្រុម និងបន្ថែមអាសយដ្ឋានអ៊ីមែលទៅក្នុងបញ្ជី និងក្រុមទាំងនោះ នៅពេលអ្នកប្រើចុះឈ្មោះ។

ធ្វើតាមមេរៀននេះ ដើម្បីទទួលបាន MailChimp API key ផ្ទាល់ខ្លួនរបស់អ្នក៖ [អំពី API Keys | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

_បន្ថែម MailChimp API key របស់អ្នកនៅក្នុង_ _**Ultimate Multisite Settings - > Add-on Settings -> MailChimp**_

## របៀបរួមបញ្ចូល {#integration-mode}

កម្មវិធីបន្ថែម MailChimp Integration ផ្តល់ “របៀបរួមបញ្ចូល” ពីរ ដែលមានន័យថាមានវិធីពីរផ្សេងគ្នាដែលអ្នកអាចប្រើកម្មវិធីបន្ថែមនេះ ដើម្បីធ្វើសមកាលកម្មអាសយដ្ឋានអ៊ីមែល។

<!-- មិនមានរូបថតអេក្រង់៖ ការជ្រើសរើសរបៀបរួមបញ្ចូល MailChimp ជាមួយជម្រើសបញ្ជីច្រើន និងបញ្ជីតែមួយ -->

_របៀបរួមបញ្ចូលពីរ ដើម្បីភាពបត់បែនកាន់តែខ្លាំង!_

### របៀប 1: បញ្ជីច្រើន {#mode-1-multiple-lists}

របៀបលំនាំដើមប្រើ **បញ្ជីច្រើន**។ របៀបនេះនឹងអនុញ្ញាតឱ្យអ្នកជ្រើសរើសបញ្ជី MailChimp ច្រើនសម្រាប់គម្រោងនីមួយៗ។ នេះជាជម្រើសល្អបំផុត ប្រសិនបើអ្នកបែងចែកអ្នកប្រើរបស់អ្នកទៅជាបញ្ជីផ្សេងៗ។

ឧទាហរណ៍ សន្មតថាអ្នកមានបញ្ជី 3 ផ្សេងគ្នានៅលើ Account MailChimp របស់អ្នក៖ **ព្រឹត្តិបត្រព័ត៌មាន** , **គម្រោង A** , និង **គម្រោង B**។ អ្នកចង់ឱ្យអ្នកប្រើទាំងអស់ មិនថាពួកគេស្ថិតក្នុងគម្រោងណាក៏ដោយ ត្រូវបានបញ្ចូលក្នុងបញ្ជីព្រឹត្តិបត្រព័ត៌មានផងដែរ ដូច្នេះពួកគេនឹងមិនខកខានព័ត៌មានថ្មីៗរបស់អ្នកទេ។

ដើម្បីសម្រេចរឿងនេះ អ្នកអាចប្រើរបៀបបញ្ជីច្រើន ដើម្បីបន្ថែមទាំងបញ្ជី **ព្រឹត្តិបត្រព័ត៌មាន** និង **គម្រោង A** ទៅក្នុងការកំណត់ MailChimp របស់គម្រោង A ហើយបន្ថែម **ព្រឹត្តិបត្រព័ត៌មាន** និង **គម្រោង B** ទៅក្នុងការកំណត់ MailChimp របស់គម្រោង B។

<!-- មិនមានរូបថតអេក្រង់៖ ការជ្រើសរើសបញ្ជីលំនាំដើម MailChimp សម្រាប់គណនីថ្មី និងគណនីដែលបានបោះបង់ -->

_ជ្រើសរើសបញ្ជីលំនាំដើមសម្រាប់គណនីថ្មី និងសម្រាប់គណនីដែលបានបោះបង់_

### របៀប 2: បញ្ជីតែមួយ + ក្រុមច្រើន {#mode-2-single-list--multiple-groups}

MailChimp អនុញ្ញាតឱ្យអ្នកបែងចែកបញ្ជីរបស់អ្នកទៅជាក្រុមអ្នកប្រើតូចៗដែលហៅថា **ក្រុម**។ វាអនុញ្ញាតឱ្យអ្នកមានការគ្រប់គ្រងច្រើនជាងមុន ព្រោះអ្នកអាចផ្ញើយុទ្ធនាការទៅក្រុមរងជាក់លាក់មួយនៃអ្នកជាវរបស់អ្នក ដោយមិនចាំបាច់បង្កើតច្បាប់ចម្លងច្រើននៃយុទ្ធនាការដូចគ្នានៅលើ MailChimp ឡើយ (ដែលធ្វើឱ្យពិបាកវិភាគរង្វាស់នៃយុទ្ធនាការនោះនៅពេលក្រោយ)។

ឧទាហរណ៍ អ្នកអាចមានបញ្ជីតែមួយឈ្មោះ **អ្នកជាវបណ្តាញរបស់ខ្ញុំ,** ហើយនៅខាងក្នុងវាមានក្រុមសម្រាប់ **គម្រោង A** និង **គម្រោង B**។

<!-- មិនមានរូបថតអេក្រង់៖ ការជ្រើសរើសបញ្ជីតែមួយ និងក្រុម MailChimp សម្រាប់បញ្ជីលំនាំដើម -->

_ជ្រើសរើសបញ្ជី និងក្រុមលំនាំដើម ប្រសិនបើប្រើរបៀបរួមបញ្ចូល “បញ្ជីតែមួយ + ក្រុមច្រើន”_

## បញ្ជី និងក្រុមលំនាំដើម {#default-lists-and-groups}

បន្ទាប់ពីអ្នកជ្រើសរើសរបៀបរួមបញ្ចូលដែលសមស្របជាងសម្រាប់ករណីប្រើប្រាស់ជាក់លាក់របស់អ្នក នោះដល់ពេលជ្រើសរើសបញ្ជីលំនាំដើម (និងក្រុម អាស្រ័យលើរបៀប)។ បញ្ជីលំនាំដើមនឹងត្រូវបានប្រើ នៅពេលគម្រោងណាមួយមិនមានបញ្ជីជាក់លាក់ដែលបានជ្រើសរើស។

អ្នកក៏អាចជ្រើសរើសបញ្ជីលំនាំដើមសម្រាប់គណនីដែលបានបោះបង់ផងដែរ។ រាល់ពេលអ្នកប្រើបញ្ចប់ការជាវរបស់ខ្លួន ហើយលុបគណនីរបស់ខ្លួនចេញពីបណ្តាញរបស់អ្នក អាសយដ្ឋានអ៊ីមែលរបស់គាត់ ឬរបស់នាង នឹងត្រូវបានផ្លាស់ទីទៅបញ្ជីដែលបានកំណត់សម្រាប់គណនីដែលបានបោះបង់។

## ការបើកការជាវទ្វេ {#enabling-double-subscription}

នៅលើទំព័រការកំណត់ដដែល អ្នកនឹងអាចបើក **ការជាវទ្វេ**។ ជម្រើសនេះនឹងបន្ថែមអ្នកប្រើទៅទាំងបញ្ជីលំនាំដើមដែលបានជ្រើសរើស និងបញ្ជីដែលបានជ្រើសរើសសម្រាប់គម្រោងរបស់អ្នកប្រើនោះ។ ប្រសិនបើមិនបានបើកជម្រើសនេះ អ្នកប្រើនឹងត្រូវបានបន្ថែមតែទៅបញ្ជីដែលបានជ្រើសរើសសម្រាប់គម្រោងរបស់អ្នកប្រើប៉ុណ្ណោះ។ ប្រសិនបើមិនមានអ្វីត្រូវបានកំណត់សម្រាប់គម្រោងជាក់លាក់នោះទេ នោះអ្នកប្រើនឹងត្រូវបានបន្ថែមទៅបញ្ជីលំនាំដើមដែលបានជ្រើសរើស។

_អ្នកអាចឱ្យអ្នកប្រើរបស់អ្នកជាវទាំងបញ្ជីលំនាំដើម និងបញ្ជីរបស់គម្រោងនៅពេលចុះឈ្មោះ_

## ការជ្រើសរើសបញ្ជី និងក្រុមសម្រាប់គម្រោងនីមួយៗ {#selecting-lists-and-groups-for-each-of-the-plans}

ចូលទៅកាន់ **គម្រោង** របស់អ្នក ហើយជ្រើសរើសគម្រោងដែលអ្នកចង់កែសម្រួល។ សូមចំណាំថា នៅលើប្រអប់មេតា **ជម្រើសកម្រិតខ្ពស់** នឹងមានផ្ទាំងថ្មីមួយឈ្មោះ MailChimp។

នៅលើផ្ទាំងនោះ អ្នកនឹងអាចជ្រើសរើសបញ្ជីច្រើន ឬបញ្ជីតែមួយ និងក្រុមច្រើន (អាស្រ័យលើរបៀបរួមបញ្ចូល) សម្រាប់គម្រោងនោះ។

_ជ្រើសរើសបញ្ជី និងក្រុមដែលអ្នកចង់ប្រើសម្រាប់គម្រោងនីមួយៗរបស់អ្នក_

នៅពេលអ្នកប្រើថ្មីចុះឈ្មោះសម្រាប់គម្រោងនោះ អាសយដ្ឋានអ៊ីមែលរបស់គាត់នឹងត្រូវបានបន្ថែមទៅបញ្ជី និងក្រុមដែលបានជ្រើសរើស!
