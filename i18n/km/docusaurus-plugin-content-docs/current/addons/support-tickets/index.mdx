# សំបុត្រគាំទ្រ

**សំបុត្រគាំទ្រ** គឺជា addon ដ៏មានអានុភាពសម្រាប់ Ultimate Multisite ដែលបន្ថែមប្រព័ន្ធសំបុត្រគាំទ្រពេញលេញទៅកាន់បណ្តាញ WordPress របស់អ្នក។ វាស័ក្តិសមសម្រាប់វេទិកា SaaS អ្នកផ្តល់សេវា WaaS (គេហទំព័រជាសេវាកម្ម) និងបណ្តាញ multisite ដែលត្រូវការផ្តល់ការគាំទ្រអតិថិជនទាំងកម្រិតបណ្តាញ និងកម្រិតគេហទំព័រដាច់ដោយឡែក។

## ហេតុអ្វីត្រូវជ្រើសរើសសំបុត្រគាំទ្រ?

* **ការគាំទ្រច្រើនកម្រិត**: អ្នកគ្រប់គ្រងបណ្តាញដោះស្រាយសំបុត្រទូទាំងបណ្តាញ ខណៈម្ចាស់គេហទំព័រគ្រប់គ្រងការគាំទ្រអតិថិជនរបស់ខ្លួន
* **លំហូរការងារបែបវិជ្ជាជីវៈ**: ការគ្រប់គ្រងវដ្តជីវិតសំបុត្រពេញលេញ ដោយមានអាទិភាព ស្ថានភាព និងការចាត់តាំង
* **បទពិសោធន៍អតិថិជន**: ផ្ទាំងប្រទាក់ស្អាត និងឆ្លើយតបល្អ ដែលធ្វើឱ្យអតិថិជនងាយស្រួលស្នើសុំជំនួយ
* **មានសុវត្ថិភាព និងគួរឱ្យទុកចិត្ត**: បង្កើតឡើងតាមអនុវត្តល្អបំផុតផ្នែកសុវត្ថិភាពរបស់ WordPress និងបង្កើនប្រសិទ្ធភាពការដំណើរការ database
* **ការរួមបញ្ចូលរលូន**: ដំណើរការល្អឥតខ្ចោះជាមួយការគ្រប់គ្រងអតិថិជន និងគេហទំព័ររបស់ Ultimate Multisite

## មុខងារសំខាន់ៗ

**ការគ្រប់គ្រងសំបុត្រពេញលេញ**

* ប្រព័ន្ធលេខសំបុត្រតែមួយគត់ សម្រាប់យោងបានងាយ
* កម្រិតអាទិភាពដែលអាចកំណត់រចនាសម្ព័ន្ធបាន (ទាប, ធម្មតា, ខ្ពស់, សំខាន់បំផុត)
* លំហូរការងារស្ថានភាពពេញលេញ (បើក, កំពុងដំណើរការ, កំពុងរង់ចាំ, បានដោះស្រាយ, បានបិទ)
* ការបែងចែកប្រភេទសំបុត្រ (ទូទៅ, បច្ចេកទេស, ការចេញវិក្កយបត្រ, សំណើមុខងារ, របាយការណ៍កំហុស)
* ចាត់តាំងសំបុត្រទៅភ្នាក់ងារគាំទ្រ ដោយមានជម្រើសចាត់តាំងស្វ័យប្រវត្តិ

**ការសន្ទនាជាខ្សែស្រឡាយ**

* ការរៀបចំការសន្ទនាជាខ្សែស្រឡាយពេញលេញសម្រាប់ការឆ្លើយតបសំបុត្រ
* រចនាប័ទ្មខុសគ្នាសម្រាប់ការឆ្លើយតបរបស់បុគ្គលិក និងអតិថិជន
* កំណត់ចំណាំឯកជនរបស់បុគ្គលិក ដែលអតិថិជនមើលមិនឃើញ
* ផ្ទាំងប្រទាក់ដំណើរការដោយ AJAX ពេលវេលាជាក់ស្តែង
* កម្មវិធីកែសម្រួលអត្ថបទទំនើបដែលមានជម្រើសធ្វើទ្រង់ទ្រាយ

**ឯកសារភ្ជាប់**

* ការផ្ទុកឯកសារឡើងប្រកបដោយសុវត្ថិភាព ជាមួយការគាំទ្រអូសនិងទម្លាក់
* ការកំណត់ប្រភេទ និងទំហំឯកសារដែលអាចកំណត់រចនាសម្ព័ន្ធបាន
* សុវត្ថិភាពកម្រិតខ្ពស់ជាមួយការរកឃើញឯកសារដែលមានគ្រោះថ្នាក់
* ការទាញយកដែលបានការពារ ជាមួយការគ្រប់គ្រងសិទ្ធិចូលប្រើ

**ការជូនដំណឹងឆ្លាតវៃ**

* ការជូនដំណឹងតាមអ៊ីមែលដោយស្វ័យប្រវត្តិសម្រាប់សំបុត្រថ្មី ការឆ្លើយតប និងការផ្លាស់ប្តូរស្ថានភាព
* គំរូអ៊ីមែលដែលអាចកែសម្រួលបាន
* ការជូនដំណឹងផ្អែកលើតួនាទីសម្រាប់បុគ្គលិក និងអតិថិជន
* អថេរគំរូថាមវន្តសម្រាប់សារផ្ទាល់ខ្លួន

**ផ្ទាំងប្រទាក់អ្នកគ្រប់គ្រងដ៏មានអានុភាព**

* បញ្ជីសំបុត្រកម្រិតខ្ពស់ដែលមានការតម្រៀប និងតម្រង
* ការកែសម្រួលរហ័សសម្រាប់ស្ថានភាព អាទិភាព និងការចាត់តាំង
* សកម្មភាពជាបាច់សម្រាប់ការគ្រប់គ្រងឱ្យមានប្រសិទ្ធភាព
* Dashboard ស្ថិតិដែលមានរង្វាស់ប្រសិទ្ធភាព
* ផ្ទាំងការកំណត់ពេញលេញ

**វិបផតថលអតិថិជនស្អាត**

* ផ្ទាំងប្រទាក់ស្អាត និងឆ្លើយតបល្អលើឧបករណ៍ទាំងអស់
* ការរចនាផ្តោតលើទូរសព្ទមុនគេ
* ការស្វែងរក និងតម្រងកម្រិតខ្ពស់
* កម្មវិធីកែសម្រួលទំនើបជាមួយរបារឧបករណ៍ធ្វើទ្រង់ទ្រាយ
* ទម្រង់ដាក់ស្នើសំបុត្រងាយប្រើ

**ការរួមបញ្ចូលបត់បែន**

* ប្រើ shortcodes ឬធាតុ UI របស់ Ultimate Multisite
* បង្ហាញទម្រង់ និងបញ្ជីសំបុត្រនៅគ្រប់ទីកន្លែងលើគេហទំព័ររបស់អ្នក
* ដំណើរការរលូនជាមួយរចនាសម្ព័ន្ធបណ្តាញ multisite
* អាចពង្រីកបានជាមួយ hooks និង filters

## ស័ក្តិសមសម្រាប់

* វេទិកា SaaS ដែលត្រូវការការគាំទ្រអតិថិជន
* អ្នកផ្តល់សេវា WaaS ដែលគ្រប់គ្រងគេហទំព័រអតិថិជនច្រើន
* គេហទំព័រសមាជិកភាពដែលមានការគាំទ្រតាមកម្រិត
* អ្នកគ្រប់គ្រងបណ្តាញដែលគ្រប់គ្រងម៉ាកច្រើន
* ម្ចាស់គេហទំព័រដែលផ្តល់សេវាអតិថិជន

## ការដំឡើង

1. ត្រូវប្រាកដថា Ultimate Multisite plugin ត្រូវបានដំឡើង និងធ្វើឱ្យសកម្មលើបណ្តាញ WordPress របស់អ្នក
2. ផ្ទុកឯកសារ addon ឡើងទៅថត `/wp-content/plugins/` របស់អ្នក
3. ធ្វើឱ្យ plugin សកម្មនៅកម្រិតបណ្តាញ តាមរយៈម៉ឺនុយ 'Plugins' ក្នុង WordPress
4. ចូលទៅកាន់ Network Admin > Support Tickets ដើម្បីកំណត់រចនាសម្ព័ន្ធការកំណត់របស់អ្នក

## សំណួរដែលសួរញឹកញាប់

## តើវាត្រូវការ Ultimate Multisite ដែរឬទេ?

បាទ/ចាស addon នេះត្រូវការ Ultimate Multisite ដែលបានដំឡើង និងកំពុងសកម្ម។ វារួមបញ្ចូលយ៉ាងជ្រៅជាមួយមុខងារគ្រប់គ្រងអតិថិជន និងគេហទំព័ររបស់ Ultimate Multisite។

## តើម្ចាស់គេហទំព័រអាចដោះស្រាយការគាំទ្រអតិថិជនរបស់ខ្លួនបានទេ?

បានច្បាស់! ម្ចាស់គេហទំព័រអាចប្រើប្រព័ន្ធសំបុត្រ ដើម្បីផ្តល់ការគាំទ្រដល់អតិថិជនរបស់ខ្លួន ខណៈអ្នកគ្រប់គ្រងបណ្តាញអាចដោះស្រាយសំបុត្រគាំទ្រទូទាំងបណ្តាញ។

## តើអតិថិជនដាក់សំបុត្រដោយរបៀបណា?

អ្នកអាចបន្ថែមទម្រង់ដាក់ស្នើសំបុត្រនៅគ្រប់ទីកន្លែងលើគេហទំព័ររបស់អ្នក ដោយប្រើ shortcodes ឬធាតុ UI របស់ Ultimate Multisite។ អតិថិជនគ្រាន់តែបំពេញទម្រង់ ហើយដាក់ស្នើសំណើរបស់ពួកគេ។

## តើខ្ញុំអាចកែសម្រួលស្ថានភាព និងអាទិភាពសំបុត្របានទេ?

បាន! អ្នកអាចកែសម្រួលកម្រិតអាទិភាព ស្ថានភាព និងប្រភេទសំបុត្រទាំងអស់នៅក្នុងការកំណត់អ្នកគ្រប់គ្រងបណ្តាញ ដើម្បីឱ្យត្រូវនឹងលំហូរការងារគាំទ្ររបស់អ្នក។

## តើវាគាំទ្រឯកសារភ្ជាប់ទេ?

បាទ/ចាស ទាំងអតិថិជន និងបុគ្គលិកអាចភ្ជាប់ឯកសារទៅសំបុត្រ និងការឆ្លើយតបបាន។ អ្នកមានការគ្រប់គ្រងពេញលេញលើប្រភេទឯកសារដែលអនុញ្ញាត និងដែនកំណត់ទំហំ ដើម្បីសុវត្ថិភាព។

## តើការជូនដំណឹងត្រូវបានដោះស្រាយយ៉ាងដូចម្តេច?

ប្រព័ន្ធផ្ញើការជូនដំណឹងតាមអ៊ីមែលដោយស្វ័យប្រវត្តិសម្រាប់សំបុត្រថ្មី ការឆ្លើយតប និងការផ្លាស់ប្តូរស្ថានភាព។ អ្នកអាចកែសម្រួលគំរូអ៊ីមែលទាំងអស់នៅក្នុងការកំណត់។

## តើខ្ញុំអាចចាត់តាំងសំបុត្រទៅភ្នាក់ងារគាំទ្រជាក់លាក់បានទេ?

បាន អ្នកអាចចាត់តាំងសំបុត្រទៅភ្នាក់ងារដោយដៃ ឬបើកការចាត់តាំងស្វ័យប្រវត្តិ ដើម្បីចែកចាយសំបុត្រដោយស្វ័យប្រវត្តិ។

## តើមានកំណត់ចំណាំឯកជនសម្រាប់បុគ្គលិកទេ?

បាទ/ចាស សមាជិកបុគ្គលិកអាចបន្ថែមការឆ្លើយតបឯកជន ដែលមើលឃើញតែសមាជិកបុគ្គលិកផ្សេងទៀតប៉ុណ្ណោះ មិនមែនអតិថិជនទេ។
