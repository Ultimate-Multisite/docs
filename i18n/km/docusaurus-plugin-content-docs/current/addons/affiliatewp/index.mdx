# AffiliateWP {#affiliatewp}

Ultimate Multisite: ការរួមបញ្ចូល AffiliateWP

ប្រើ AffiliateWP ដ៏មានអានុភាព ដើម្បីពង្រីកមូលដ្ឋានអតិថិជននៃបណ្តាញ Ultimo របស់អ្នក!

## ការដំឡើង {#installation}

1. បង្ហោះ 'ultimate-multisite-git ' ទៅកាន់ថត '/wp-content/plugins/'
2. បើកដំណើរការកម្មវិធីបន្ថែមតាមរយៈម៉ឺនុយ 'កម្មវិធីបន្ថែម' ក្នុង WordPress
3. អនុវត្តតាមអ្នកជំនួយការជាជំហានៗ ដើម្បីរៀបចំកម្មវិធីបន្ថែម

## មគ្គុទ្ទេសក៍ប្រើប្រាស់ {#usage-guide}

[AffiliateWP](https://affiliatewp.com/) បានក្លាយជាដំណោះស្រាយពេញនិយម នៅពេលបង្កើតកម្មវិធីដៃគូផ្សព្វផ្សាយសម្រាប់អាជីវកម្មអនឡាញរបស់អ្នក។ Ultimate Multisite: AffiliateWP គឺជាស្ពានដែលអនុញ្ញាតឱ្យអ្នកបង្កើតអ្វីដូចគ្នាសម្រាប់បណ្តាញបង់ប្រាក់របស់អ្នក។

## ចាប់ផ្តើម {#getting-started}

ជាដំបូង អ្នក **ត្រូវតែមាន** [**AffiliateWP**](https://affiliatewp.com/) **ដែលបានបើកដំណើរការ**! អ្នកនឹងមិនអាចប្រើ add-on របស់យើងបានទេ ប្រសិនបើអ្នកមិនមាន **កម្មវិធីបន្ថែមដែលរួមបញ្ចូលជាមួយវា**។ ប្រសិនបើអ្នកមិនមានគម្រោងឱ្យអតិថិជនរបស់អ្នកប្រើវាលើ site របស់ពួកគេផងដែរ នោះ AffiliateWP គួរត្រូវបានបើកដំណើរការតែនៅលើ site មេរបស់អ្នកប៉ុណ្ណោះ។

លើសពីនេះ អ្នកត្រូវមានទាំង **Ultimate Multisite** និង **Ultimate Multisite: AffiliateWP** ដែលបាន **បើកដំណើរការ** **ទូទាំងបណ្តាញ**។

## បន្ថែម Ultimate Multisite ជាការរួមបញ្ចូល AffiliateWP {#adding-ultimate-multisite-as-an-affiliatewp-integration}

នៅលើ site មេរបស់អ្នក សូមចូលទៅកាន់ការកំណត់ដៃគូផ្សព្វផ្សាយ ហើយនៅផ្ទាំងការរួមបញ្ចូល សូមធីក Ultimate Multisite។ វានឹងប្រាប់ [AffiliateWP](https://affiliatewp.com/) ឱ្យចាប់ផ្តើមតាមដានការចូលមើល និងការបំប្លែងពីលំហូរចុះឈ្មោះរបស់ Ultimate Multisite។

<!-- រូបថតអេក្រង់មិនមាន៖ ផ្ទាំងការកំណត់ការរួមបញ្ចូល AffiliateWP ដែលមានប្រអប់ធីក Ultimate Multisite -->

_ធីកប្រអប់ Ultimate Multisite នៅខាងក្រោម ហើយរក្សាទុក_

នោះជាអ្វីទាំងអស់ដែលត្រូវធ្វើ។ ការចូលមើលដែលមកពីតំណណែនាំនឹងត្រូវបានតាមដាន ហើយនៅពេលអ្នកប្រើនោះធ្វើការទូទាត់លើកដំបូងនៅពេលក្រោយ ការបំប្លែងនឹងត្រូវបានបង្កើត ហើយកម្រៃជើងសារនឹងត្រូវបានទុកដាច់ដោយឡែកសម្រាប់ដៃគូផ្សព្វផ្សាយដែលទទួលខុសត្រូវចំពោះការបំប្លែងនោះ។

ទោះយ៉ាងណា មានតែការទូទាត់ដំបូងដែលទទួលបានពីអ្នកប្រើដែលទើបបង្កើតថ្មីប៉ុណ្ណោះនឹងត្រូវបានតាមដាន។ ដើម្បីបើកការទូទាត់ណែនាំដែលកើតឡើងជាបន្តបន្ទាប់ អ្នកត្រូវការ [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## បើកការណែនាំដែលកើតឡើងជាបន្តបន្ទាប់ {#enabling-recurring-referrals}

បន្ទាប់ពីដំឡើង [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/) សូមចូលទៅកាន់ផ្ទាំងការណែនាំដែលកើតឡើងជាបន្តបន្ទាប់ក្នុងការកំណត់ដៃគូផ្សព្វផ្សាយ ហើយបើកការប្រើការណែនាំដែលកើតឡើងជាបន្តបន្ទាប់។
