# Material WP

ចំណុចប្រទាក់គ្រប់គ្រងរបស់ WordPress នៅដដែលមកយូរហើយ។ ប៉ុន្តែអ្នកធ្លាប់គិតទេថា កំណែ Material Design នៃចំណុចប្រទាក់គ្រប់គ្រងនឹងមើលទៅដូចម្តេច? 

Material WP គឺជា theme សម្រាប់ផ្នែកគ្រប់គ្រង ដែលត្រូវបានបង្កើតដើម្បីផ្តល់រូបរាង និងអារម្មណ៍បែបនោះ ហើយជួយអ្នកផ្តល់ Dashboard ដែលបានប្ដូរតាមតម្រូវការច្រើនជាងមុនដល់ client របស់អ្នក។

## មុខងារ

- គាំទ្រ CSS ផ្ទាល់ខ្លួន
- ការកំណត់រចនាប័ទ្មងាយស្រួល
- រៀបលំដាប់ menu ឡើងវិញ
- នាំចូល/នាំចេញងាយស្រួល

## ឯកសារ

**ប្ដូរ Admin Panel របស់អ្នកតាមតម្រូវការជាមួយ Material WP**

## ការដំឡើង

1. បង្ហោះ 'material-wp' ទៅកាន់ directory '/wp-content/plugins/'
2. បើកដំណើរការ plugin តាមរយៈ menu 'Plugins' នៅក្នុង WordPress

## មគ្គុទ្ទេសក៍ប្រើប្រាស់

ចំណុចប្រទាក់គ្រប់គ្រងរបស់ WordPress នៅដដែលមកយូរហើយ។ ប៉ុន្តែអ្នកធ្លាប់គិតទេថា កំណែ Material Design នៃចំណុចប្រទាក់គ្រប់គ្រងនឹងមើលទៅដូចម្តេច?

Material WP គឺជា theme សម្រាប់ផ្នែកគ្រប់គ្រង ដែលត្រូវបានបង្កើតដើម្បីផ្តល់រូបរាង និងអារម្មណ៍បែបនោះ ហើយជួយអ្នកផ្តល់ Dashboard ដែលបានប្ដូរតាមតម្រូវការច្រើនជាងមុនដល់ client របស់អ្នក។ Material WP ក៏ជាផ្នែកមួយនៃ core របស់ plugin ពិសេសៗរបស់ Ultimate Multisite ផងដែរ។

បន្ទាប់ពី **ដំឡើងវា** សូមចូលទៅកាន់ទំព័រ _Plugins_ របស់អ្នក ហើយចុចដើម្បីបើកដំណើរការវានៅលើ Network របស់អ្នក។

_បើកដំណើរការវាទូទាំង network នៅលើទំព័រ Plugins របស់អ្នក_

_**ចំណាំ៖**_ _ប្រសិនបើអ្នកគ្រាន់តែបើកដំណើរការវា វានឹងបង្ហាញនៅខាង client របស់អ្នកជាជម្រើសមួយ ដែលអាចធីកដើម្បីបន្ថែម theme សម្រាប់តែ user នោះប៉ុណ្ណោះ។_

ឥឡូវនេះ ចំណុចប្រទាក់គ្រប់គ្រងរបស់អ្នកត្រូវបានរចនាឡើងវិញទាំងស្រុង! អ្នកក៏មាន menu ថ្មីមួយនៅជួរឈរខាងឆ្វេងដែលមានឈ្មោះថា Material WP
