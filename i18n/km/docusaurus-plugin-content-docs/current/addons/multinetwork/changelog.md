# កំណត់ត្រាបម្លាស់ប្តូរ Multi-Network {#multi-network-changelog}

### 1.3.0 {#130}
* ថ្មី៖ កម្មវិធីមើលជាមុនសម្រាប់ពុម្ពបណ្តាញ — រកមើល និងមើលពុម្ពបណ្តាញជាមុនតាមផ្ទាំងផ្ទាល់ មុនពេលទិញ។
* ថ្មី៖ កម្មវិធីជ្រើសពុម្ពបណ្តាញនៅពេល checkout — អតិថិជនអាចជ្រើសពុម្ពបណ្តាញ ក្នុងលំហូរ signup checkout។
* ថ្មី៖ SSO តាមតំណ magic link ឆ្លងបណ្តាញ — អតិថិជនអាចចូលប្រើបណ្តាញរងបានយ៉ាងរលូន តាមការផ្ទៀងផ្ទាត់តាម magic-link។
* ថ្មី៖ ការចម្លងមេឌៀបណ្តាញ — មេឌៀរបស់គេហទំព័រត្រូវបានចម្លងយ៉ាងត្រឹមត្រូវ ពេលចម្លងបណ្តាញពីពុម្ព។
* ជួសជុល៖ ការចម្លងបណ្តាញឥឡូវដំណើរការដោយមិនចាំបាច់មាន addon multi-tenancy; សិទ្ធិ super admin ត្រូវបានផ្តល់ឡើងវិញ បន្ទាប់ពីចម្លង។
* ជួសជុល៖ ការបង្កើតបណ្តាញឥឡូវត្រូវគ្នាជាមួយ WooCommerce និង payment gateways ទាំងអស់។
* ជួសជុល៖ ឥឡូវបានបញ្ចូលលេខច្រកក្នុងការបង្កើតដូមេន fallback។
* ជួសជុល៖ CSS របស់កម្មវិធីជ្រើសពុម្ព ត្រូវបានផ្លាស់ទីទៅ stylesheet ដែលបាន enqueued ដើម្បីបង្ហាញបានត្រឹមត្រូវលើ theme ទាំងអស់។
* ជួសជុល៖ ការពារកុំឲ្យមាន key `network_id` ដែលមិនបានកំណត់ ក្នុងវិសាលភាពសំណួរគេហទំព័រ។
* ជួសជុល៖ ការត្រួតពិនិត្យសិទ្ធិបណ្តាញមុនពេលប្ដូរបរិបទ ជួយការពារកំហុសលក្ខណសម្បត្តិតារាង `wpdb`។
