# бһӮбҹҶбһҡбһјбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳ

бһӮбҹҶбһҡбһјбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳ вҖ” Ultimate Multisite.

## бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡ {#requirements}

- WordPress 5.3 бһ¬бһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„
- PHP 7.4 бһ¬бһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„
- бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳ Multisite Ultimate (бһ”бһ¶бһ“бһ”бһҫбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ)

## бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ {#installation}

1. бһ•бҹ’бһ‘бһ»бһҖбһҜбһҖбһҹбһ¶бһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһЎбһҫбһ„бһ‘бҹ…бһҗбһҸ `/wp-content/plugins/` бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
2. бһ”бһҫбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһҸбһ¶бһҳбһҳбҹүбһәбһ“бһ»бһҷ 'бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳ' бһҖбҹ’бһ“бһ»бһ„ WordPress
3. бһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ”бһҹбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ“бҹӮбһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһҺбҹ’бһҸбһ¶бһү

### бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸ {#testing}

бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бҹҒбҹҮбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҖбҹ’бһҡбһ”бһҒбҹҗбһҺбҹ’бһҢбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһ–бҹҒбһүбһӣбҹҒбһүбҹ–

- **бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһҜбһҖбһҸбһ¶**бҹ– бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһ•бҹ’бһўбҹӮбһҖбһӣбһҫ PHPUnit бһҮбһ¶бһҳбһҪбһҷаёҠаёёаё”бһҸбҹҒбһҹбҹ’бһҸ WordPress
- **бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҸбҹҒбһҹбҹ’бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“**бҹ– бһ•бҹ’бһҸбһӣбҹӢбһҳбҹҒбһҸбһјбһҠбһҮбҹҶбһ“бһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҒбһҺбһ¶бһҡбһёбһҷбҹүбһјбһҸбҹҒбһҹбҹ’бһҸбһ‘бһјбһ‘бҹ…
- **бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһҸбҹҒбһҹбҹ’бһҸ**бҹ– бһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһұбҹ’бһҷбһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһҖбһјбһҠ
- **бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸ Multisite**бҹ– бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“ multisite

### бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҖбһјбһҠ {#code-standards}

- **бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠ WordPress**бҹ– бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳ PHPCS
- **бһ—бһ¶бһ–бһҸбҹ’бһҡбһјбһңбһӮбҹ’бһ“бһ¶бһҮбһ¶бһҳбһҪбһҷ PHP 7.4+**бҹ– бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹ PHP бһ‘бҹҶбһ“бһҫбһ”
- **бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһӢбһ·бһҸбһ·бһңбһ“бҹ’бһҸ**бҹ– бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ PHPStan бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бҹ’бһҡбһ—бҹҒбһ‘
- **бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ‘бҹҶбһ“бһҫбһ”бһҖбһҳбҹ’бһҳбһҖбһјбһҠ**бҹ– бһ…бҹ’бһ”бһ¶бһ”бҹӢ Rector бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ¶бһ”бҹӢбһҠбҹҒбһҸ PHP

### бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһ¶бһ„бһҹбһ„бҹӢ {#build-system}

- **бһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бҹ’бһҡбһ–бҹ’бһҷбһҹбһҖбһҳбҹ’бһҳ**бҹ– бһҖбһ¶бһҡбһ”бһ„бҹ’бһҡбһҪбһҳбһҜбһҖбһҹбһ¶бһҡ CSS/JS
- **бһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮ**бҹ– бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡ POT
- **бһҖбһ¶бһҡбһңбҹҒбһ…бһҒбҹ’бһ…бһ”бҹӢ**бҹ– бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹҗбһҺбҹ’бһҺбһҹбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бҹӮбһҖбһ…бһ¶бһҷ
- **бһҡбһ”бҹҖбһ”бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ**бҹ– бһ‘бҹ’бһҡбһ–бҹ’бһҷбһҹбһҖбһҳбҹ’бһҳбһҳбһ·бһ“бһ”бһ¶бһ“бһ”бһ„бҹ’бһҡбһҪбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹҶбһ”бһ¶бһҸбҹӢбһҖбҹҶбһ бһ»бһҹ

## бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ {#configuration}

бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бҹҒбҹҮбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҮбһҳбҹ’бһҡбһҫбһҹбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҠбҹӮбһӣбһўбһ¶бһ…бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҸбһ¶бһҳбһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһҺбҹ’бһҸбһ¶бһүбҹ”
