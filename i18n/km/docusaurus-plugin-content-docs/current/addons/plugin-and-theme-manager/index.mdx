# ឧបករណ៍គ្រប់គ្រងកម្មវិធីបន្ថែម និងរូបរាង {#plugin--theme-manager}

Multisite Ultimate: ឧបករណ៍គ្រប់គ្រងកម្មវិធីបន្ថែម និងរូបរាង

កែសម្រួល ឬលាក់ព័ត៌មានមេតា (ចំណងជើង អ្នកនិពន្ធ រូបភាពតូច និងការពិពណ៌នា) របស់កម្មវិធីបន្ថែម និងរូបរាងដែលបានដំឡើង។ បង្កើតប្រភេទសម្រាប់កម្មវិធីបន្ថែម និងរូបរាង ដើម្បីអនុញ្ញាតឱ្យអ្នកប្រើរបស់អ្នកត្រងវា ហើយបន្ថែមទំព័រកម្មវិធីបន្ថែមផ្ទាល់ខ្លួនស្អាតមួយទៅក្នុងផ្ទាំងរបស់អ្នកប្រើរបស់អ្នក។

ដើម្បីប្រើកម្មវិធីបន្ថែមនេះ អ្នកគ្រាន់តែចូលទៅកាន់ទំព័រកម្មវិធីបន្ថែម ឬរូបរាង នៅក្នុងផ្នែកគ្រប់គ្រងបណ្ដាញរបស់អ្នក ហើយចុចលើ "កែសម្រួលព័ត៌មានកម្មវិធីបន្ថែម/រូបរាង"។

## ការដំឡើង {#installation}

1. ផ្ទុកឡើង 'ultimate-multisite-plugin-and-theme-manager' ទៅកាន់ថត '/wp-content/plugins/'
2. បើកដំណើរការកម្មវិធីបន្ថែមតាមរយៈម៉ឺនុយ 'កម្មវិធីបន្ថែម' ក្នុង WordPress ផ្នែកគ្រប់គ្រងបណ្ដាញរបស់អ្នក

## មគ្គុទ្ទេសក៍ប្រើប្រាស់ {#usage-guide}

Ultimate Multisite: ឧបករណ៍គ្រប់គ្រងកម្មវិធីបន្ថែម និងរូបរាង គឺជា add-on **premium** របស់ Ultimate Multisite ដែលមានអានុភាព ហើយផ្តល់ឱ្យអ្នកនូវការគ្រប់គ្រងពេញលេញលើរបៀបដែលអ្នកប្រើបណ្ដាញរបស់អ្នកមើលឃើញ និងធ្វើអន្តរកម្មជាមួយជម្រើសកម្មវិធីបន្ថែម និងរូបរាងនៅលើបណ្ដាញរបស់អ្នក។

add-on នេះមានគោលបំណងបី៖

  1. វាអនុញ្ញាតឱ្យអ្នកផ្លាស់ប្តូរទិន្នន័យមេតារបស់ទាំងកម្មវិធីបន្ថែម និងរូបរាងបានទាំងស្រុង។ វារួមមានឈ្មោះ ការពិពណ៌នា ឈ្មោះអ្នកនិពន្ធ និងសូម្បីតែរូបភាពតូច!

  2. វាអនុញ្ញាតឱ្យអ្នកចាត់ប្រភេទកម្មវិធីបន្ថែម និងរូបរាង ដើម្បីឱ្យអ្នកប្រើរបស់អ្នកអាចតម្រៀបវាបានងាយស្រួល។

  3. វាផ្តល់ជម្រើសឱ្យអ្នកជំនួសទំព័រកម្មវិធីបន្ថែមចាស់ និងគួរឱ្យធុញ ដោយទំព័រថ្មីដែលមានរូបរាងដូចទំព័ររូបរាង និងអាចតម្រៀបតាមប្រភេទបាន។

បន្ទាប់ពីទាញយក និងដំឡើងវា ផ្ទាំងថ្មីមួយ - _ការកំណត់ add-on_ \- នឹងបង្ហាញនៅលើទំព័រការកំណត់ Ultimate Multisite។

_នៅទីនេះ អ្នកនឹងឃើញជម្រើសសកលដែលបានបន្ថែមដោយ Ultimate Multisite: ឧបករណ៍គ្រប់គ្រងកម្មវិធីបន្ថែម និងរូបរាង_

នៅលើទំព័រ _កម្មវិធីបន្ថែម_ នៃបណ្ដាញរបស់អ្នក ឥឡូវនេះអ្នកនឹងអាចកែសម្រួលព័ត៌មានមេតារបស់កម្មវិធីបន្ថែមទាំងអស់ ដូចជា ឈ្មោះ អ្នកនិពន្ធ និងសូម្បីតែរូបភាពតូចរបស់កម្មវិធីបន្ថែម។

ឧបករណ៍គ្រប់គ្រងកម្មវិធីបន្ថែម និងរូបរាង ផ្តល់ប្រភេទបង្ហាញពីរ៖ រចនាប័ទ្មកម្មវិធីបន្ថែម និងរចនាប័ទ្មរូបរាង។ ដូច្នេះចុងក្រោយ បន្ទាប់ពីធ្វើការកែតម្រូវដែលអ្នកត្រូវការ នេះគឺជារបៀបដែលអតិថិជនរបស់អ្នកនឹងឃើញទំព័រ _កម្មវិធីបន្ថែម_ របស់ពួកគេ៖
