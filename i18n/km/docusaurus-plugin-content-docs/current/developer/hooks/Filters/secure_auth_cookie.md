# តម្រង: secure_auth_cookie {#filter-secureauthcookie}

តម្រងថាតើ auth cookie គួរត្រូវបានផ្ញើតាម HTTPS ប៉ុណ្ណោះឬអត់។

## ប៉ារ៉ាម៉ែត្រ {#parameters}

| ឈ្មោះ | ប្រភេទ | ការពិពណ៌នា |
|------|------|-------------|
| $secure | `bool` | ថាតើ cookie គួរត្រូវបានផ្ញើតាម HTTPS ប៉ុណ្ណោះឬអត់។ |
| $user_id | `int` | លេខសម្គាល់អ្នកប្រើប្រាស់។ |
