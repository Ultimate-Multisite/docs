# бһҸбһҳбҹ’бһҡбһ„: wu_sso_url

бһҸбһҳбҹ’бһҡбһ„ URL SSO бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸ бһҳбһ»бһ“бһ–бҹҒбһӣбһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҸбҹ’бһҡбһЎбһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһўбһҸбһ·бһҗбһ·бһҮбһ“бһҶбҹ’бһӣбһ„бһҠбҹӮбһ“бҹ”

бһ”бҹ’бһҡбһҫбһҸбһҳбҹ’бһҡбһ„бһ“бҹҒбҹҮ бһ“бҹ…бһ–бҹҒбһӣбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бһҡбһ·бһ”бһ‘бһҠбҹӮбһӣбһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“бһ‘бҹ…бһҸбҹҶбһҺ SSO бһҡбһ”бһҹбҹӢ tenant бһҜбһҖбһҡбһ¶бһҮбҹ’бһҷ бһ¬бһҮбҹҶбһ“бһҪбһҹ URL бһҳбҹҒбһҖбһҺбҹ’бһҠбһ¶бһӣ бһҠбҹ„бһҷбһҡбһҖбҹ’бһҹбһ¶бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҹбһүбҹ’бһүбһ¶бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҡбһ”бһҹбҹӢ Ultimate Multisiteбҹ”

## бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ {#parameters}

| бһҲбҹ’бһҳбҹ„бҹҮ | бһ”бҹ’бһҡбһ—бҹҒбһ‘ | бһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶ |
|------|------|-------------|
| $sso_url | `string` | URL SSO бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбҹ” |
| $user | `WP_User` | бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҠбҹӮбһӣбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҠбҹ„бһҷбһҖбһ¶бһҡбһ…бһјбһӣбһҳбһҫбһӣ SSOбҹ” |
| $site_id | `int` | бһӣбҹҒбһҒбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһӮбҹ„бһӣбһҠбҹ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ…бһјбһӣбһҳбһҫбһӣбҹ” |
| $redirect_to | `string` | URL бһӮбҹ„бһӣбһҠбҹ…бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ SSO бһҮбҹ„бһӮбһҮбҹҗбһҷбҹ” |
