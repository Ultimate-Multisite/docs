# តម្រង៖ wu_get_checkout_variables {#filter-wugetcheckoutvariables}

អនុញ្ញាតឱ្យអ្នកអភិវឌ្ឍន៍ plugin ត្រងការកំណត់ជាមុននៃទំព័រ checkout។

សូមប្រុងប្រយ័ត្ន ព្រោះ key ដែលបាត់អាចធ្វើឱ្យ checkout នៅផ្នែកខាងមុខខូចទាំងស្រុង។

## ប៉ារ៉ាម៉ែត្រ {#parameters}

| ឈ្មោះ | ប្រភេទ | ការពិពណ៌នា |
|------|------|-------------|
| $variables | `array` | អថេរដែលបានបកប្រែតាមមូលដ្ឋាន។ |
