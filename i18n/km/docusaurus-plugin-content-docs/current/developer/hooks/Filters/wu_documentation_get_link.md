# តម្រង: wu_documentation_get_link {#filter-wudocumentationgetlink}

អនុញ្ញាតឱ្យអ្នកអភិវឌ្ឍន៍​កម្មវិធីបន្ថែម​ត្រង​តំណ។

| $slug | `string` | slug ដែលបានប្រើដើម្បីយកតំណ |
