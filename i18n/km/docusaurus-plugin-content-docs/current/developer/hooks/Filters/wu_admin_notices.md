# តម្រង: wu_admin_notices {#filter-wuadminnotices}

អនុញ្ញាតឱ្យអ្នកអភិវឌ្ឍន៍តម្រងការជូនដំណឹងអ្នកគ្រប់គ្រងដែលបានបន្ថែមដោយ Ultimate Multisite។

## ប៉ារ៉ាម៉ែត្រ {#parameters}

| ឈ្មោះ | ប្រភេទ | ការពិពណ៌នា |
|------|------|-------------|
| $notices | `array` | បញ្ជីការជូនដំណឹងសម្រាប់ផ្ទាំងជាក់លាក់នោះ។ |
| $all_notices | `array` | បញ្ជីការជូនដំណឹងដែលបានបន្ថែម ដោយបែងចែកតាមផ្ទាំង។ |
| $panel | `string` | ផ្ទាំងសម្រាប់ទាញយកការជូនដំណឹង។ |
| $filter | `string` | បង្ហាញថាការជូនដំណឹងដែលអាចបិទបាន ត្រូវបានតម្រងចេញឬអត់។ |
| $dismissed_messages | `array` | បញ្ជីកូនសោការជូនដំណឹងដែលបានបិទ។ |
