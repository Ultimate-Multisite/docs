# តម្រង៖ wu_magic_link_enforce_ip {#filter-wumagiclinkenforceip}

តម្រងថាតើត្រូវអនុវត្តការផ្ទៀងផ្ទាត់អាសយដ្ឋាន IP ឬអត់។

កំណត់ជា false ដើម្បីអនុញ្ញាតឱ្យ tokens ដំណើរការពីបណ្តាញផ្សេងៗ។ វាបន្ថយសុវត្ថិភាព ប៉ុន្តែបង្កើនភាពងាយស្រួលប្រើ (ឧទាហរណ៍ សម្រាប់អ្នកប្រើទូរស័ព្ទដែលប្តូរបណ្តាញ)។

## ប៉ារ៉ាម៉ែត្រ {#parameters}

| ឈ្មោះ | ប្រភេទ | ការពិពណ៌នា |
|------|------|-------------|
| $enforce | `bool` | ថាតើត្រូវអនុវត្តការផ្គូផ្គងអាសយដ្ឋាន IP ឬអត់។ |
