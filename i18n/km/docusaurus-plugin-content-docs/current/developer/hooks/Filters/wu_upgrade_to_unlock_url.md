# តម្រង: wu_upgrade_to_unlock_url

អនុញ្ញាតឱ្យអ្នកអភិវឌ្ឍន៍ផ្លាស់ប្តូរ URL សម្រាប់ upgrade ដើម្បីដោះសោ

## ប៉ារ៉ាម៉ែត្រ

| ឈ្មោះ | ប្រភេទ | ការពិពណ៌នា |
|------|------|-------------|
| $url | `string` | URL សម្រាប់ upgrade។ |
| $args | `array` | ម៉ូឌុល និងប្រភេទនៃកម្រិតដែលត្រូវការ upgrade។ |

### ប្រភព

បានកំណត់ក្នុង [`inc/functions/limitations.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/limitations.php#L177) នៅបន្ទាត់ 177
