# Filter: wu_magic_link_enforce_user_agent

ចម្រាញ់ថាតើត្រូវបង្ខំការផ្ទៀងផ្ទាត់ user agent ឬអត់។

កំណត់ជា false ដើម្បីអនុញ្ញាតឱ្យ tokens ដំណើរការឆ្លងកាត់ browsers/devices ផ្សេងៗ។ វាបន្ថយសុវត្ថិភាព ប៉ុន្តែបង្កើនភាពងាយស្រួលប្រើ។

## ប៉ារ៉ាម៉ែត្រ

| ឈ្មោះ | ប្រភេទ | ការពិពណ៌នា |
|------|------|-------------|
| $enforce | `bool` | ថាតើត្រូវបង្ខំឱ្យ user agent ត្រូវគ្នាឬអត់។ |
