# តម្រង: wu_use_domain_mapping

កំណត់ថាតើគួរប្រើការផ្គូផ្គងឬអត់

ជាទូទៅ អ្នកនឹងចង់អនុញ្ញាតឱ្យប្រើតែការផ្គូផ្គងដែលសកម្មប៉ុណ្ណោះ។ ទោះយ៉ាងណា ប្រសិនបើអ្នកចង់ប្រើតក្កវិជ្ជាកម្រិតខ្ពស់ជាងនេះ ឬអនុញ្ញាតឱ្យដែនដែលមិនសកម្មក៏អាចផ្គូផ្គងបានដែរ គ្រាន់តែចម្រោះនៅទីនេះ។
