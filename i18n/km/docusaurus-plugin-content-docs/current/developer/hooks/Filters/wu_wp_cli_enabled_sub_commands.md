# бһҸбһҳбҹ’бһҡбһ„: wu_wp_cli_enabled_sub_commands

| бһҲбҹ’бһҳбҹ„бҹҮ | бһ”бҹ’бһҡбһ—бҹҒбһ‘ | бһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶ |
|------|------|-------------|
| $sub_commands | `array` | sub_commands бһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳбҹ” |
| $command_base | `string` | бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“вҖӢбһҠбҹӮбһӣвҖӢбһ”бҹ’бһҡбһҫвҖӢбһҖбҹ’бһ“бһ»бһ„вҖӢбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ бһ—бҹ’бһӣбһ¶бһҳбҹ—вҖӢбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢвҖӢбһ–бһёвҖӢбһ«бһҹбҹ” |
| $this | `\Base_Manager` | бһўбһ„бҹ’бһӮвҖӢбһ’бһ¶бһҸбһ»вҖӢбһ“бҹғвҖӢбһңбһҸбҹ’бһҗбһ»бҹ” |
