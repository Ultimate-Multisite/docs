# សកម្មភាព៖ wu_dashboard_widgets {#action-wudashboardwidgets}

## ប៉ារ៉ាម៉ែត្រ {#parameters}

| ឈ្មោះ | ប្រភេទ | ការពិពណ៌នា |
|------|------|-------------|
| $tab | `string` | ផ្ទាំងបច្ចុប្បន្ន។ |
| $screen | `\WP_Screen` | វត្ថុ screen។ |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | instance ទំព័រគ្រប់គ្រង Ultimate Multisite។ |
