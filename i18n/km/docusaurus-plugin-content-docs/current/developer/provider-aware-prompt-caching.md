# бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ prompt бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҠбҹ„бһҷбһҠбһ№бһ„бһ–бһёбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶ {#provider-aware-prompt-caching}

Superdav AI Agent v1.12.0 бһҺбҹӮбһ“бһ¶бҹҶ **бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ prompt бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҠбҹ„бһҷбһҠбһ№бһ„бһ–бһёбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶** бһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҗбҹ’бһӣбҹғ API бһ“бһ·бһ„бһ—бһ¶бһ–бһҷбһәбһҸбһҶбҹ’бһӣбһҫбһҷбһҸбһ” бһҠбҹ„бһҷбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ prompt бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶ LLM бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” бһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶бһ“бһёбһҳбһҪбһҷбҹ—бһҳбһ¶бһ“бһҷбһ“бҹ’бһҸбһҖбһ¶бһҡ бһ“бһ·бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бҹ”

## бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ… {#overview}

бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ prompt бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбҹ–

- бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ prompt бһ’бҹҶбҹ— бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһүбһ№бһҖбһүбһ¶бһ”бҹӢбһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ
- бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҗбҹ’бһӣбҹғ API бһҠбҹ„бһҷбһҮбҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбһҠбҹӮбһӣбҹ—
- бһ”бһ„бҹ’бһҖбһҫбһ“бһӣбҹ’бһ”бһҝбһ“бһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҶбһҺбһҫбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ
- бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһңбһҠбҹ’бһҸбһҮбһёбһңбһ·бһҸбһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ

бһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҒбһ»бһҹбҹ—бһӮбҹ’бһ“бһ¶бҹ–

- **Google Gemini**: `cachedContents` API
- **Azure OpenAI**: бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ prompt бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҮбһ¶бһҳбһҪбһҷ TTL
- **OpenRouter**: бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҸбһ¶бһҳбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶
- **Vertex Anthropic**: бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ prompt бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ

## Google Gemini: `cachedContents` API {#google-gemini-cachedcontents-api}

Google Gemini бһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһҸбһ¶бһҳбһҡбһҷбҹҲ `cachedContents` APIбҹ”

### бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ {#configuration}

```php
$config = [
    'provider' => 'google-gemini',
    'model' => 'gemini-2.0-flash',
    'caching' => [
        'enabled' => true,
        'ttl' => 3600, // 1 hour in seconds
        'max_tokens' => 1000000, // Max tokens to cache
    ],
];
```

### бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ prompt бһҠбҹӮбһӣбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ {#creating-a-cached-prompt}

```php
use Superdav\AI\Providers\GoogleGemini;

$gemini = new GoogleGemini( $config );

$cached_content = $gemini->create_cached_content(
    [
        'system_prompt' => 'You are a helpful assistant...',
        'context' => 'Large context document...',
        'ttl' => 3600,
    ]
);

// Returns: ['cache_id' => 'abc123', 'expires_at' => timestamp]
```

### бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ prompt бһҠбҹӮбһӣбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ {#using-a-cached-prompt}

```php
$response = $gemini->generate(
    [
        'cache_id' => 'abc123',
        'prompt' => 'User question here',
    ]
);
```

### бһңбһҠбҹ’бһҸбһҮбһёбһңбһ·бһҸбһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ {#cache-lifecycle}

```php
// List cached contents
$caches = $gemini->list_cached_contents();

// Get cache details
$cache = $gemini->get_cached_content( 'abc123' );

// Extend cache TTL
$gemini->update_cached_content(
    'abc123',
    ['ttl' => 7200] // Extend to 2 hours
);

// Delete cache
$gemini->delete_cached_content( 'abc123' );
```

### бһңбһ·бһ’бһёбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Gemini {#best-practices-for-gemini}

- **бһҖбҹҶбһҺбһҸбҹӢ TTL бһұбҹ’бһҷбһҹбһҳбһҹбҹ’бһҡбһ”**: бһҗбҹ’бһӣбһ№бһ„бһҗбҹ’бһӣбҹӮбһ„бһҡбһңбһ¶бһ„бһҖбһ¶бһҡбһҹбһ“бҹ’бһҹбҹҶбһҗбҹ’бһӣбҹғ бһ“бһ·бһ„бһ—бһ¶бһ–бһ…бһ¶бһҹбҹӢбһ бһҪбһҹбһҹбһҳбҹҗбһҷбһҡбһ”бһҹбҹӢбһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ
- **бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ system prompt бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ**: бһ”бҹ’бһҡбһҫ system prompt бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһЎбһҫбһ„бһңбһ·бһүбһ“бҹ…бһӮбҹ’бһҡбһ”бҹӢбһҹбҹҶбһҺбһҫ
- **бһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ**: бһҸбһ¶бһҳбһҠбһ¶бһ“бһҗбһ¶бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҺбһ¶бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ…бҹ’бһҡбһҫбһ“бһ”бҹҶбһ•бһ»бһҸ
- **бһҹбһҳбҹ’бһўбһ¶бһҸбһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҠбҹӮбһӣбһ•бһ»бһҸбһҖбҹҶбһҺбһҸбҹӢ**: бһӣбһ»бһ”бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҠбҹӮбһӣбһҳбһ·бһ“бһ”бҹ’бһҡбһҫбһҮбһ¶бһ”бҹ’бһҡбһ…бһ¶бҹҶ

## Azure OpenAI: бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ prompt бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ {#azure-openai-prompt-caching}

Azure OpenAI бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ prompt бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ TTL бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ”

### бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ {#configuration-1}

```php
$config = [
    'provider' => 'azure-openai',
    'model' => 'gpt-4-turbo',
    'api_version' => '2024-08-01-preview',
    'caching' => [
        'enabled' => true,
        'cache_control' => 'max_age=3600',
    ],
];
```

### бһҖбһ¶бһҡбһ”бһҫбһҖбһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ {#enabling-caching}

```php
use Superdav\AI\Providers\AzureOpenAI;

$azure = new AzureOpenAI( $config );

$response = $azure->generate(
    [
        'system_prompt' => 'You are a helpful assistant...',
        'context' => 'Large context document...',
        'prompt' => 'User question here',
        'cache_control' => 'max_age=3600',
    ]
);

// Response includes cache usage:
// [
//     'content' => '...',
//     'cache_creation_input_tokens' => 1000,
//     'cache_read_input_tokens' => 500,
// ]
```

### Header бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ {#cache-headers}

Azure OpenAI бһ”бҹ’бһҡбһҫ HTTP header бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбҹ–

```
Cache-Control: max_age=3600
```

бһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡбҹ–

- `max_age=<seconds>`: бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһҷбҹҲбһ–бҹҒбһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢ
- `no_cache`: бһҖбһ»бҹҶбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҹбҹҶбһҺбһҫбһ“бҹҒбҹҮбһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ
- `no_store`: бһҖбһ»бҹҶбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ бһ“бһ·бһ„бһҖбһ»бҹҶбһ”бҹ’бһҡбһҫбһЎбһҫбһ„бһңбһ·бһү

### бһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ {#monitoring-cache-usage}

- **бһ”бҹ’бһҡбһҫбһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ ephemeral**: бһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһӮбҹ’бһӮбһҸбҹӮбһҳбһҪбһҷ
- **бһҖбҹҶбһҺбһҸбҹӢ max_tokens бһұбҹ’бһҷбһҹбһҳбһҹбҹ’бһҡбһ”**: бһҗбҹ’бһӣбһ№бһ„бһҗбҹ’бһӣбҹӮбһ„бһҡбһңбһ¶бһ„бһ‘бҹҶбһ бҹҶбһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ бһ“бһ·бһ„бһҗбҹ’бһӣбҹғбһ…бҹҶбһҺбһ¶бһҷ
- **бһҸбһ¶бһҳбһҠбһ¶бһ“бһҳбҹүбҹӮбһҸбҹ’бһҡбһ·бһҖбһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ**: бһҸбһ¶бһҳбһҠбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ
- **бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҮбһ¶бһҳбһҪбһҷбһ”бһ“бҹ’бһ‘бһ»бһҖбһҖбһ¶бһҡбһ„бһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ**: бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҗбһ¶бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһ•бҹ’бһҸбһӣбҹӢбһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҠбһӣбҹӢбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

## бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҶбҹ’бһӣбһ„бһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶ {#cross-provider-caching-strategy}

### бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҡбһҪбһҳ {#unified-configuration}

```php
$config = [
    'caching' => [
        'enabled' => true,
        'default_ttl' => 3600,
        'providers' => [
            'google-gemini' => [
                'ttl' => 3600,
                'max_tokens' => 1000000,
            ],
            'azure-openai' => [
                'cache_control' => 'max_age=3600',
            ],
            'vertex-anthropic' => [
                'cache_control' => [
                    'type' => 'ephemeral',
                    'max_tokens' => 1000000,
                ],
            ],
        ],
    ],
];
```

### бһҖбһ¶бһҡбһҡбһҖбһғбһҫбһүбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶ {#provider-detection}

```php
$provider = $config['provider'];

$cache_config = $config['caching']['providers'][ $provider ]
    ?? $config['caching'];

// Use provider-specific caching configuration
```

### бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ”бһҳбҹ’бһҡбһ»бһ„ {#fallback-strategy}

```php
try {
    // Try caching with primary provider
    $response = $primary_provider->generate( $request );
} catch ( CacheException $e ) {
    // Fall back to non-cached request
    $response = $primary_provider->generate(
        array_merge( $request, ['cache_control' => 'no_cache'] )
    );
}
```

## бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҗбҹ’бһӣбҹғбһ…бҹҶбһҺбһ¶бһҷ {#cost-optimization}

### бһӮбһҺбһ“бһ¶бһҖбһ¶бһҡбһҹбһ“бҹ’бһҹбҹҶ {#calculate-savings}

```php
$cache_created_tokens = $response['cache_creation_input_tokens'] ?? 0;
$cache_read_tokens = $response['cache_read_input_tokens'] ?? 0;
$regular_tokens = $response['input_tokens'] ?? 0;

// Typical pricing (varies by provider):
$cache_creation_cost = $cache_created_tokens * 0.00001; // 10x cheaper
$cache_read_cost = $cache_read_tokens * 0.000001; // 100x cheaper
$regular_cost = $regular_tokens * 0.00001;

$total_cost = $cache_creation_cost + $cache_read_cost + $regular_cost;
$savings = ($regular_tokens * 0.00001) - $total_cost;

echo "Estimated savings: \$$savings\n";
```

### бһӮбһ“бҹ’бһӣбһ№бҹҮбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– {#optimization-tips}

- **бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ system prompts бһ’бҹҶбҹ—бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ**: бһҹбһ“бҹ’бһҹбҹҶбһҗбҹ’бһӣбҹғбһ…бҹҶбһҺбһ¶бһҷбһ”бһ¶бһ“бһ…бҹ’бһҡбһҫбһ“бһ”бҹҶбһ•бһ»бһҸ
- **бһ”бҹ’бһҡбһҫбһ”бһҡбһ·бһ”бһ‘бһЎбһҫбһ„бһңбһ·бһү**: бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҜбһҖбһҹбһ¶бһҡбһ”бһҡбһ·бһ”бһ‘бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһүбһ№бһҖбһүбһ¶бһ”бҹӢбһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ
- **бһ”бһ¶бһ…бҹӢбһҹбҹҶбһҺбһҫ**: бһҠбһ¶бһҖбҹӢбһҹбҹҶбһҺбһҫбһҹбҹ’бһҡбһҠбҹҖбһ„бһӮбҹ’бһ“бһ¶бһҮбһ¶бһҖбҹ’бһҡбһ»бһҳ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһ“ cache hits бһұбҹ’бһҷбһ”бһ¶бһ“бһўбһҸбһ·бһ”бһҡбһҳбһ¶
- **бһҸбһ¶бһҳбһҠбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ**: бһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбһ¶бһҡбһҹбһ“бҹ’бһҹбҹҶбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ
- **бһҖбҹӮбһҸбһҳбҹ’бһҡбһјбһң TTL**: бһҗбҹ’бһӣбһ№бһ„бһҗбҹ’бһӣбҹӮбһ„бһҡбһңбһ¶бһ„бһҗбҹ’бһӣбҹғбһ…бҹҶбһҺбһ¶бһҷ бһ“бһ·бһ„бһ—бһ¶бһ–бһҗбҹ’бһҳбһё

## бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶ {#troubleshooting}

### бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫ {#cache-not-being-used}

- бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҗбһ¶бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҫбһҖбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҗбһ¶ prompts бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„ (бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбһ¶бһҡбһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ)
- бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҗбһ¶бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһ•бһ»бһҸбһҖбҹҶбһҺбһҸбҹӢ
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶

### бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһ”бһҡбһ¶бһҮбҹҗбһҷ {#cache-creation-failing}

- бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҗбһ¶бһ‘бҹҶбһ бҹҶбһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҹбҹ’бһҗбһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҗбһ¶бһңбһ¶бһҖбҹ’бһҷбһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
- бһ’бһ¶бһ“бһ¶бһҗбһ¶бһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҜбһҖбһҹбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢ

### бһҗбҹ’бһӣбҹғбһ…бҹҶбһҺбһ¶бһҷбһҳбһ·бһ“бһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖ {#unexpected-costs}

## бһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ {#next-steps}

1. **бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ**: бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
2. **бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ**: бһҡбҹҖбһ”бһ…бҹҶбһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҸбһ¶бһҳбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶
3. **бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ**: бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҗбһ¶бһңбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ prompts бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
4. **бһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ**: бһҸбһ¶бһҳбһҠбһ¶бһ“ cache hits бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһ“бҹ’бһҹбҹҶбһҗбҹ’бһӣбҹғбһ…бҹҶбһҺбһ¶бһҷ
5. **бһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–**: бһҖбҹӮбһҸбһҳбҹ’бһҡбһјбһң TTL бһ“бһ·бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһғбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһӣбһ‘бҹ’бһ’бһ•бһӣ
