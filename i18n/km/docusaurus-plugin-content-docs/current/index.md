# ឯកសារ Ultimate Multisite

សូមស្វាគមន៍មកកាន់ឯកសារ Ultimate Multisite។ Ultimate Multisite បម្លែង WordPress Multisite ទៅជាវេទិកា Website as a Service (WaaS) ពេញលេញ។

## ផ្នែកឯកសារ {#documentation-sections}

### [មគ្គុទ្ទេសក៍អ្នកប្រើ](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

អ្វីៗទាំងអស់ដែលអ្នកត្រូវការ ដើម្បីដំឡើង កំណត់រចនាសម្ព័ន្ធ និងគ្រប់គ្រងវេទិកា WaaS របស់អ្នក។ គ្របដណ្តប់លើការដំឡើង ច្រកទូទាត់ប្រាក់ ការភ្ជាប់ដែន ការគ្រប់គ្រងអតិថិជន និងច្រើនទៀត។

### [ឯកសារអ្នកអភិវឌ្ឍន៍](/developer/hooks) {#developer-documentation}

ឯកសារយោងបច្ចេកទេសសម្រាប់អ្នកអភិវឌ្ឍន៍ដែលបង្កើតលើ Ultimate Multisite។ រួមមានឯកសារយោង hooks ឯកសារ REST API និងមគ្គុទ្ទេសក៍អភិវឌ្ឍន៍ addon។

### [ផ្នែកបន្ថែម](/addons) {#addons}

ឯកសារសម្រាប់ plugin addon បង់ប្រាក់ ដែលពង្រីក Ultimate Multisite ជាមួយមុខងារបន្ថែម ដូចជាការរួមបញ្ចូល WooCommerce ការវិភាគ ការលក់ដែន និងច្រើនទៀត។

## ការទទួលជំនួយ {#getting-help}

- [ការពិភាក្សា GitHub](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) -- សួរសំណួរ និងចែករំលែកគំនិត
- [បញ្ហា GitHub](https://github.com/Ultimate-Multisite/ultimate-multisite/issues) -- រាយការណ៍កំហុស
- [WordPress.org](https://wordpress.org/plugins/ultimate-multisite/) -- ទាញយក plugin ស្នូល
