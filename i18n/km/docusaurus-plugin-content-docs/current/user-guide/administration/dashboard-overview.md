# ទិដ្ឋភាពទូទៅនៃแดชบอร์ด (Dashboard Overview)

Ultimate Multisite មាន **Dashboard** ផ្ទាល់ខ្លួនមួយនៅក្រោមเมนู Ultimate Multisite ហើយក៏បន្ថែម widgets ទៅក្នុង Network Dashboard ផងដែរ។ អត្ថបទនេះនឹងពន្យល់ពីអ្វីដែលអ្នកនឹងឃើញនៅក្នុងទីតាំងទាំងពីរនេះ។

## Network Dashboard

Ultimate Multisite បញ្ចូល widgets ដែលមានប្រយោជន៍ទៅក្នុង network-level dashboard ដែលអ្នកអាចរកបាននៅ **My Sites → Network Admin → Dashboard**។

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

នេះគឺជាអ្វីដែល widget នីមួយៗផ្តល់ជូន៖

**First Steps:** បង្ហាញនៅពេលដំឡើងថ្មី ហើយរំលឹកអ្នកឱ្យបំពេញកិច្ចការរៀបចំដូចជា ការកំណត់វិធីសាស្រ្តបង់ប្រាក់ និងការបង្កើតគណនីសាកល្បង។

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Summary:** រាយការណ៍ពីចំនួនអ្នកចុះឈ្មោះ និងចំណូលសម្រាប់ថ្ងៃនោះ។ MRR (Monthly Recurring Revenue) បង្ហាញពីប្រាក់ចំណូលសរុបដែលបានព្យាករណ៍លើអតិថិជនដែលមានអាជីវកម្មបង្កើតចំណូល។

![Summary widget](/img/admin/network-dashboard-summary.png)

**Activity Stream:** រាយការណ៍អំពីព្រឹត្តិការណ៍នានាដែលកើតឡើងនៅក្នុង Ultimate Multisite — ការចុះឈ្មោះ ការលុបចោល ការបង្កើតគេហទំព័រ ការលុប និងសកម្មភាពផ្សេងៗទៀត។

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** បង្ហាញសេចក្តីសង្ខេបខ្លីៗអំពីអ្នកប្រើប្រាស់ និងគេហទំព័រនៅក្នុង network រួមជាមួយ shortcut សម្រាប់បង្កើតគេហទំព័រថ្មី ឬអ្នកប្រើប្រាស់ថ្មី។

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** ទាញយក និងបង្ហាញព័ត៌មានលំដាប់ចុងក្រោយបំផុតរបស់ Ultimate Multisite រួមទាំងការធ្វើបច្ចុប្បន្នភាព កំហុសធ្ងន់ធ្ងរ និងការជួសជុលសុវត្ថិភាព។

![News widget](/img/admin/network-dashboard-news.png)

## Ultimate Multisite Dashboard

ខណៈពេលដែល network dashboard បង្ហាញព័ត៌មានទូទៅអំពី network នោះ ultimate multisite dashboard (នៅផ្នែកខាងលើនៃเมนู Ultimate Multisite) បង្ហាញព័ត៌មានអំពីសេវាកម្ម។

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**កំណើនចំណូលប្រចាំខែ (Monthly Recurring Revenue Growth):** បង្ហាញការបែងចែកចំណូលរៀងរាល់ខែក្នុងរយៈពេល ១២ ខែ។ ចំណូលថ្មីនឹងបង្ហាញជាពណ៌ខៀវ ចំណែកការលុបចោលនឹងបង្ហាញជាពណ៌ក្រហម។

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**ការគាំទ្រជួរថ្ងៃ (Date-range Support):** អាចច្រោះទិន្នន័យលើ Dashboard ដោយជ្រើសរើសរយៈពេលជាក់លាក់ ដើម្បីផ្តោតលើព័ត៌មានដែលអ្នកត្រូវការ។ ចន្លោះនៃកាលបរិច្ឆេទនឹងកំណត់ថាអ្វីខ្លះដែលបង្ហាញនៅក្នុង widgets ខាងក្រោម។

![Date range and dashboard stats](/img/admin/um-dashboard-date-range.png)

Widgets របស់ Dashboard រួមមាន៖

- **Revenue (ចំណូល)** — ចំនួនទឹកប្រាក់សរុបសម្រាប់រយៈពេលនោះ រួមជាមួយការបែងចែក MRR និងតាមផលិតផលនីមួយៗ។
- **New Memberships (ការចុះឈ្មោះថ្មី)** — មានចំនួនការចុះឈ្មោះជាថ្មីប៉ុន្មានដែលបានបង្កើតក្នុងរយៈពេលដែលបានជ្រើសរើស។
- **Signups by Countries (ការចុះឈ្មោះតាមប្រទេស)** — ការបែងចែកភូមិសាស្ត្រនៃការចុះឈ្មោះ។
- **Signups by Forms (ការចុះឈ្មោះតាមទម្រង់)** — ចំនួនអតិថិជនដែលបានចុះឈ្មោះតាមទម្រង់ Checkout នីមួយៗ បើប្រៀបធៀបនឹងចំនួនដែលអ្នកគ្រប់គ្រងបន្ថែមដោយផ្ទាល់។
- **Most Visited Sites (គេហទំព័រដែលចូលមើលច្រើនជាងគេ)** — គេហទំព័រដែលត្រូវបានចូលមើលច្រើនបំផុតនៅក្នុងបណ្តាញរបស់អ្នក (តម្រូវឱ្យបើកការរាប់ការចូលមើលនៅ **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**)
