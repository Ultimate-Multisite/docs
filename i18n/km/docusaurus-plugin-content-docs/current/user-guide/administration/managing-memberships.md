# ការគ្រប់គ្រងសមាជិកភាព (Managing Memberships) {#managing-memberships}

នៅក្នុង Ultimate Multisite អ្នកអាចគ្រប់គ្រងការជាសមាជិកភាព និងចូលប្រើការជាសេវាទាំងអស់នៅលើបណ្តាញរបស់អ្នក។ សូមពិនិត្យមើល និងកែសម្រួលព័ត៌មានលម្អិតនៃការជាសេវាដូចជា ផែនការ (plans) និងផលិតផល (products) ចំនួនគេហទំព័រ (sites) ចំនួនទឹកប្រាក់សម្រាប់ការគិតថ្លៃ និងភាពញឹកញាប់នៃការបង់ប្រាក់ ប្រវត្តិការបង់ប្រាក់ និងពេលវេលាដែលបានកំណត់។

ដើម្បីចូលទៅកាន់หน้านี้ សូមទៅកាន់ផ្ទាំង **Memberships** នៅក្នុងម៉ឺនុយ Ultimate Multisite។

![Memberships list page](/img/admin/memberships-list.png)

នៅលើទំព័រ Memberships អ្នកអាចមើលបញ្ជីអតិថិជនរបស់អ្នក និងផលិតផលដែលភ្ជាប់ជាមួយគណនីរបស់ពួកគេ។ អ្នកអាចច្រោះការជាសមាជិកភាពតាមស្ថានភាព៖ សកម្ម (active), រង់ចាំ (pending), ផ្អាក (on hold), កន្លងផុតកំណត់ (expired) ឬលុបចោល (cancelled)។

ការជាសមាជិកភាពឥតគិតថ្លៃត្រូវបានចាត់ទុកថាជាការជាសមាជិកភាព **ពេញមួយជីវិត**។ វាគ្មានកាលបរិច្ឆេទផុតកំណត់ទេ ហើយនឹងមិនផុតកំណត់ដោយស្វ័យប្រវត្តិឡើយ។ ប្រសិនបើផែនការឥតគិតថ្លៃគួរតែបញ្ចប់បន្ទាប់ពីរយៈពេលសាកល្បង ឬរយៈពេលជាក់លាក់ សូមប្រើផលិតផលដែលបង់ប្រាក់ដែលមានការសាកល្បង (trial) បិទការជាសមាជិកភាពដោយដៃ ឬផ្លាស់ប្តូរអតិថិជនទៅផែនការផ្សេងនៅពេលដែលការចូលប្រើរបស់ពួកគេត្រូវផ្លាស់ប្តូរ។

ចុចលើការជាសមាជិកភាពមួយដើម្បីចូលទៅកាន់ទំព័រដែលអ្នកអាចកែសម្រួលព័ត៌មានលម្អិតនៃការជាសមាជិកភាពបាន — ព័ត៌មានគិតថ្លៃ រមណីយដ្ឋានគេហទំព័រ ឬដែនចូល (visit limits) និងច្រើនទៀត។

![Membership edit interface](/img/admin/membership-edit.png)

នេះគឺជាទិដ្ឋភាពពេញលេញនៃទំព័រកែសម្រួលការជាសមាជិកភាព៖

![Membership edit full interface](/img/admin/membership-edit-full.png)

អ្នកក៏អាចបន្ថែមការជាសមាជិកភាពថ្មីដែលភ្ជាប់ជាមួយអ្នកប្រើប្រាស់ដែលបានចុះឈ្មោះរួចហើយនៅក្នុងបណ្តាញរបស់អ្នកផងដែរ។ ចុចលើ **Add Membership** ហើយកំណត់ព័ត៌មានលម្អិត។

![Add Membership modal](/img/admin/membership-add-modal.png)

បន្ទាប់មក ចុចលើ **Create Membership**។ ការជាសមាជិកភាពថ្មីនឹងបង្ហាញនៅលើបញ្ជី Memberships។
