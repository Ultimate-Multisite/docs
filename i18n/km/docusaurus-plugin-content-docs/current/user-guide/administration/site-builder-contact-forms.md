# бһ‘бһҳбҹ’бһҡбһ„бҹӢбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡ (Site Builder) {#site-builder-contact-forms}

Superdav AI Agent v1.10.0 бһ”бһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ–бһёбһ…бҹҶбһҺбһ»бһ…бһҮбһҮбҹӮбһҖ (chat interface) бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Site Builder agentбҹ” бһ“бҹҒбҹҮбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бҹҒбһүбһӣбҹҒбһүбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһӮбҹ’бһҡбһ”бҹӢбһ‘бҹҶбһ–бҹҗбһҡбһҺбһ¶бһҳбһҪбһҷбһҠбҹ„бһҷбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ…бһ¶бһҖбһ…бҹҒбһүбһ–бһёбһҖбһ¶бһҡбһҮбһҮбҹӮбһҖбһ“бҹ„бҹҮбһ‘бҹҒбҹ”

## бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ… (Overview) {#overview}

бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҡбһ”бһҹбҹӢ Site Builder (`create_contact_form`) бһ“бһ№бһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖ plugin бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· бһ бһҫбһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҠбһҫбһҳбһҡбһ”бһҹбҹӢ plugin бһ“бҹ„бҹҮбҹ” plugin бһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҡбһҪбһҳбһҳбһ¶бһ“бҹ–

- Contact Form 7
- WPForms
- Fluent Forms
- Gravity Forms

Assistant (бһҮбҹҶбһ“бһҪбһҷбһҖбһ¶бһҡ) бһ“бһ№бһ„бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ plugin бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҠбҹӮбһӣбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҠбҹӮбһӣбһҳбһ¶бһ“бһ“бҹ…бһӣбһҫбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· бһ бһҫбһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶбһЎбһҫбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ plugin бһ“бҹ„бҹҮбҹ”

## бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ‘бһ¶бһҖбҹӢбһ‘бһ„ (Creating a Contact Form) {#creating-a-contact-form}

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Site Builderбҹ–

1. бһ”бһҫбһҖ panel chat **Gratis AI Agent** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ WordPress adminбҹ”
2. бһ”бҹ’бһҸбһјбһҡбһ‘бҹ… agent **Site Builder** бһҠбҹ„бһҷбһ…бһ»бһ…бһӣбһҫ icon agent бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ header chatбҹ”
3. бһҡбҹҖбһ”бһҡбһ¶бһ”бҹӢбһўбҹҶбһ–бһёбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ–

   > "бһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ‘бҹ…бһҖбһ¶бһ“бҹӢ Contact page бһҠбҹ„бһҷбһҳбһ¶бһ“бһңбһ¶бһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҲбҹ’бһҳбҹ„бҹҮ (name), бһўбҹҠбһёбһҳбҹүбҹӮбһӣ (email), бһҹбһ¶бһҡ (message) бһ“бһ·бһ„бһӣбҹҒбһҒбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘ (phone number)бҹ”"

   бһ¬бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ“бһ·бһҷбһ¶бһҷбһҗбһ¶бҹ–

   > "бһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Contact pageбҹ”"

4. Site Builder бһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ‘бһ¶бһҖбҹӢбһ‘бһ„ бһ бһҫбһҷбһҸбҹ’бһҡбһЎбһ”бҹӢбһҳбһҖбһңбһ·бһүбһ“бһјбһң shortcode бһҠбҹӮбһӣбһҸбҹ’бһҡбҹҖбһҳбһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣ (embed)бҹ”

## бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Shortcode бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸ (Using the Generated Shortcode) {#using-the-generated-shortcode}

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһё Site Builder бһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҡбһҪбһ… бһңбһ¶бһҸбҹ’бһҡбһЎбһ”бҹӢбһҳбһҖбһңбһ·бһүбһ“бһјбһң shortcode (бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ– `[contact-form-7 id="123"]`)бҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бҹ–

1. **бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„ page бһ¬ post** вҖ” бһ…бһҳбҹ’бһӣбһ„ shortcode бһ бһҫбһҷбһ”бһ·бһ‘бһ—бҹ’бһҮбһ¶бһ”бҹӢбһңбһ¶бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ page бһ¬ post бһҺбһ¶бһҳбһҪбһҷбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ block editor бһ¬ classic editorбҹ”
2. **бһ”бһ“бҹ’бһҗбҹӮбһҳбһңбһ¶бһҸбһ¶бһҳбһҡбһҷбҹҲ Site Builder** вҖ” бһҹбһҪбһҡ Site Builder бһұбҹ’бһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ‘бҹ…бһҖбһ¶бһ“бҹӢ page бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҳбһҪбһҷбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ–

   > "бһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ‘бҹ…бһҖбһ¶бһ“бҹӢ Contact pageбҹ”"

бҹЈ. **бһ”бҹ’бһҡбһҫбһңбһ¶а№ғаёҷ template** вҖ” бһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһ—бһ¶бһ–бһҹбҹ’бһ‘бһ¶бһҸбҹӢбһҮбҹҶбһ“бһ¶бһүбһҮбһ¶бһҳбһҪбһҷ PHP бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ“бҹ’бһҗбҹӮбһҳ shortcode бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ file template бһҡбһ”бһҹбҹӢ theme бһ”бһ¶бһ“бҹ”

## бһҖбһ¶бһҡбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣ Contact Form {#customizing-the-contact-form}

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһё Site Builder бһ”бһ„бҹ’бһҖбһҫбһҸ contact form бһҡбһҪбһ…бһ бһҫбһҷ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһңбһ¶бһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бҹҖбһҸбһ”бһ¶бһ“бҹ–

### бһҸбһ¶бһҳбһҡбһҷбҹҲ Chat Interface {#via-the-chat-interface}

бһҹбһ»бҹҶбһұбҹ’бһҷ Site Builder бһҖбҹӮбһ”бҹ’бһҡбҹӮ form бһҠбҹ„бһҷбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶бҹ–

> "бһҹбһјбһҳбһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ– contact form бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһңбһ¶бһӣ subject бһ“бһ·бһ„бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһңбһ¶бһӣ message бһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһңбһ¶бһӣбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бҹҶбһ–бҹҒбһү (required)бҹ”"

Site Builder бһ“бһ№бһ„бһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ– form бһ бһҫбһҷбһҸбҹ’бһҡбһЎбһ”бҹӢбһҳбһҖбһңбһ·бһүбһ“бһјбһң shortcode бһҠбҹӮбһӣбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһ бһҫбһҷбҹ”

### бһҸбһ¶бһҳбһҡбһҷбҹҲ Admin Interface бһҡбһ”бһҹбҹӢ Form Plugin {#via-the-form-plugins-admin-interface}

бһўбҹ’бһ“бһҖбһҖбҹҸбһўбһ¶бһ…бһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣ form бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ (settings) бһҡбһ”бһҹбҹӢ form plugin бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҠбҹӮбһҡбҹ–

1. бһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢ **Contact Form 7** (бһ¬ form plugin бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„) бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ WordPress adminбҹ”
2. бһҡбһҖбһҳбһҫбһӣ form бһҠбҹӮбһӣ Site Builder бһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбҹ”
3. бһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһңбһ¶бһӣ form, бһ…бҹ’бһ”бһ¶бһ”бҹӢбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ (validation rules), бһ“бһ·бһ„бһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһҸбһ¶бһҳ email (email notifications) бһҸбһ¶бһҳбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбҹ”

## Form Plugins бһ“бһ·бһ„бһ—бһ¶бһ–бһҶбһ”бһӮбҹ’бһ“бһ¶ (Compatibility) {#form-plugins-and-compatibility}

Site Builder бһ“бһ№бһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҗбһ¶ form plugin бһҳбһҪбһҷбһҺбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„бһ“бҹ…бһӣбһҫбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһ”бҹ’бһҡбһҫбһңбһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸ contact formбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ¶бһ“ form plugin бһ…бҹ’бһҡбһҫбһ“бһҠбҹӮбһӣбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„ Site Builder бһ“бһ№бһ„бһ•бҹ’бһҸбһӣбҹӢбһўбһ¶бһ‘бһ·бһ—бһ¶бһ–бһҸбһ¶бһҳбһӣбҹҶбһҠбһ¶бһ”бҹӢбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

1. Contact Form 7
2. WPForms
3. Fluent Forms
4. Gravity Forms

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһӮбҹ’бһҳбһ¶бһ“ plugin бһҺбһ¶бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбҹ„бһҳгҒ“гӮҢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„бһ‘бҹҒ Site Builder бһ“бһ№бһ„бһҺбҹӮбһ“бһ¶бҹҶбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҠбҹҶбһЎбһҫбһ„бһҳбһҪбһҷбһҳбһ»бһ“бһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸ contact formбҹ”

## бһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһҸбһ¶бһҳ Email (Email Notifications) {#email-notifications}

Contact forms бһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷ Site Builder бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҮбһ¶бһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳбһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһүбһҫбһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһҸбһ¶бһҳ email бһ‘бҹ…бһҖбһ¶бһ“бҹӢ site administratorбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһўбһ¶бһҹбһҷбһҠбҹ’бһӢбһ¶бһ“ email бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бһ‘бһҪбһӣ бһ“бһ·бһ„бһҹбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһ”бһ¶бһ“бҹ–

1. бһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢ admin interface бһҡбһ”бһҹбҹӢ form plugin бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
2. бһҡбһҖбһҳбһҫбһӣ form бһҠбҹӮбһӣ Site Builder бһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбҹ”
3. бһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ“бҹғбһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһҸбһ¶бһҳ email (email notification settings)бҹ”

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸ бһҹбһјбһҳбһҳбһҫбһӣ documentation бһҡбһ”бһҹбҹӢ form plugin бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–

- [бһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһўбҹҠбһёбһҳбҹӮбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Contact Form 7](https://contactform7.com/docs/)
- [бһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһўбҹҠбһёбһҳбҹӮбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ WPForms](https://wpforms.com/docs/)
- [бһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһўбҹҠбһёбһҳбҹӮбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Fluent Forms](https://fluentforms.com/docs/)
- [бһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһўбҹҠбһёбһҳбҹӮбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Gravity Forms](https://docs.gravityforms.com/)

## бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ {#example-use-cases}

### Contact Form бһҹбһ¶бһҳбһүбҹ’бһү {#simple-contact-form}

> "бһ”бһ„бҹ’бһҖбһҫбһҸ contact form бһҹбһ¶бһҳбһүбҹ’бһүбһҠбҹӮбһӣбһҳбһ¶бһ“бһңбһ¶бһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҲбҹ’бһҳбҹ„бҹҮ бһўбҹҠбһёбһҳбҹүбҹӮбһӣ бһ“бһ·бһ„бһҹбһ¶бһҡбҹ”"

### Form бһ–бһ бһ»бһҮбҹҶбһ бһ¶бһ“ (Multi-Step Form) {#multi-step-form}

> "бһ”бһ„бҹ’бһҖбһҫбһҸ contact form бһҠбҹӮбһӣбһҳбһ¶бһ“бһҮбҹҶбһ бһ¶бһ“бһ‘бһёбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„ бһ“бһ·бһ„бһҮбҹҶбһ бһ¶бһ“бһ‘бһёбһ–бһёбһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ¶бһҡ бһ“бһ·бһ„бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҠбҹӮбһӣбһ–бҹҒбһүбһ…бһ·бһҸбҹ’бһҸбҹ”"

### Form бһҮбһ¶бһҳбһҪбһҷ Logic бһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢ (Conditional Logic) {#form-with-conditional-logic}

> "бһ”бһ„бҹ’бһҖбһҫбһҸ contact form бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһңбһ¶бһӣбһ•бҹ’бһҹбҹҒбһ„бҹ—бһӮбҹ’бһ“бһ¶бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ dropdown menuбҹ”"

### Form бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бҹ„бҹҮбһҜбһҖбһҹбһ¶бһҡ (File Upload) {#form-with-file-upload}

> "бһ”бһ„бҹ’бһҖбһҫбһҸ contact form бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһ„бҹ’бһ бҹ„бҹҮбһҜбһҖбһҹбһ¶бһҡ бһ¬ file бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбһҖбһҮбһ¶бһҳбһҪбһҷбҹ”"

:::note
бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ contact form бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Superdav AI Agent v1.10.0 бһ“бһ·бһ„бһҗбҹ’бһҳбһёбһҮбһ¶бһ„бһ“бҹҒбҹҮбҹ” Agent Site Builder бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ“бҹҒбҹҮбҹ”
