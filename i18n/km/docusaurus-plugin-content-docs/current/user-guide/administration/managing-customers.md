# ការគ្រប់គ្រងអតិថិជន {#managing-customers}

ទំព័រ **Customers** នឹងបង្ហាញបញ្ជីសមាជិកដែលបានចុះឈ្មោះនៅក្នុងបណ្តាញរបស់អ្នក។ ទំព័រនេះបង្ហាញព័ត៌មានមូលដ្ឋានរបស់អ្នកប្រើប្រាស់ រួមមាន ឈ្មោះ, ប្រវត្តិរូបអ៊ីម៉ែល, ការចូលប្រើប្រាស់ចុងក្រោយ, និងតំណ **switch to** ដែលអនុញ្ញាតឱ្យអ្នកចូលទៅកាន់ subsites របស់អតិថិជនក្នុងនាមជា site administrator។

![Customers list page](/img/admin/customers-list.png)

ចុចលើ **Manage** នៅខាងក្រោមប័ណ្ណអតិថិជន ដើម្បីមើលព័ត៌មានលម្អិតអំពីអតិថិជននោះ។

នៅលើទំព័រព័ត៌មានលម្អិតរបស់អតិថិជន ប្លុក **At a Glance** នឹងបង្ហាញ Customer ID, រយៈពេលចូលប្រើប្រាស់ចុងក្រោយ, និងចំនួនទឹកប្រាក់សរុបដែលអតិថិជននេះបានចំណាយ។

ប្លុក **Customer Options** អនុញ្ញាតឱ្យអ្នកកំណត់អតិថិជននេះជា **VIP** (ស្លាកមួយដើម្បីធ្វើឱ្យពួកគេងាយស្រួលរក)។ தாவែល **Billing Info** បង្ហាញព័ត៌មានការគិតថ្លៃដូចជា អ៊ីម៉ែល, អាសយដ្ឋាន, និងប្រទេស។

ប្លុក **Fields** បង្ហាញឈ្មោះអ្នកប្រើប្រាស់អតិថិជន, អ៊ីម៉ែល, ហើយអនុញ្ញាតឱ្យអ្នកផ្លាស់ប្តូរស្ថានភាព **Email Verification**។

ខាងក្រោមប្លុក Customer Options អ្នកនឹងឃើញប្លុកបន្ថែមចំនួនបួន៖

- **Memberships** — ការជាសមាជិកបច្ចុប្បន្នដែលអតិថិជនបានចុះឈ្មោះ
- **Payments** — ប្រវត្តិការទូទាត់របស់អតិថិជន
- **Sites** — គេហទំព័រដែលអតិថិជននេះកាន់កាប់
- **Events** — សកម្មភាពដែលទាក់ទងនឹងអតិថិជននេះ (Email verification, ការផ្លាស់ប្តូរការគិតថ្លៃ ជាដើម)

ចុចលើរូបតំណាងព្រួញ (→) នៅជាប់នឹងធាតុណាមួយដើម្បីចូលទៅកាន់ទំព័រព័ត៌មានលម្អិតរបស់វា។ ឧទាហរណ៍ ចុចលើព្រួញនៅជាប់នឹងគេហទំព័រមួយ នឹងនាំអ្នកទៅកាន់ទំព័រแก้ไขគេហទំព័រ ដែលអ្នកអាចកែប្រែការភ្ជាប់សមាជិកភាព, ប្រភេទគេហទំព័រ, និងស្ថានភាពសកម្ម។

ដូចគ្នានេះដែរ ចំពោះ memberships និង payments — នីមួយៗតំណភ្ជាប់ទៅទំព័រព័ត៌មានលម្អិតដែលអ្នកអាចកែសម្រួលស្ថានភាពសមាជិកភាព, អតិថិជនដែលពាក់ព័ន្ធ, ផលិតផល, ការកំណត់ payment gateway, និងសូម្បីតែដំណើរការការបង្វិលប្រាក់វិញ (refunds)។

នៅខាងស្តាំនៃទំព័រអតិថិជន អ្នកនឹងឃើញ៖

- **ការចូលប្រើចុងក្រោយ និង IP Address** — ពេលវេលាចូលប្រើចុងក្រោយ, អាសយដ្ឋាន IP, និងប្រទេស
- **Timestamps** — កាលបរិច្ឆេទបង្កើតគណនីអតិថិជន និងពេលដែលបានកែប្រែចុងក្រោយ
- **Delete** — ជម្រើសដើម្បីលុបអតិថិជនចេញជាអចិន្ត្រៃយ៍

:::
