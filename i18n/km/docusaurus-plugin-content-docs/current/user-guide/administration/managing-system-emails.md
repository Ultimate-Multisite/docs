# ការគ្រប់គ្រងអ៊ីមែលប្រព័ន្ធ {#managing-system-emails}

Ultimate Multisite ផ្ញើអ៊ីមែលប្រព័ន្ធផ្សេងៗសម្រាប់ព្រឹត្តិការណ៍នានាដូចជា ការបញ្ជាក់ការចុះឈ្មោះ, លិខិតទទួលលុយ, និងការផ្លាស់ប្តូរភាពជាសមាជិក។ អ្នកអាចកែសម្រួល قالب (template) អ៊ីមែលទាំងនេះឱ្យត្រូវនឹងម៉ាករបស់អ្នកបាន។

## ការមើលអ៊ីមែលប្រព័ន្ធ {#viewing-system-emails}

ចូលទៅកាន់ **Ultimate Multisite → Settings** ហើយចុចលើแท็บ **Emails** ដើម្បីមើលការកំណត់ដែលទាក់ទងនឹងអ៊ីមែល។

![Settings - Emails tab](/img/admin/settings-emails.png)

ពីទីនោះ អ្នកអាចចូលទៅកាន់ទំព័រ **System Emails** ដើម្បីមើល قالبអ៊ីមែលទាំងអស់បាន។

![System Emails list](/img/admin/system-emails-list.png)

អ្នកក៏អាចកែសម្រួល قالبអ៊ីមែលនីមួយៗដោយប្រើកម្មវិធីនិពន្ធ (editor) អ៊ីមែលប្រព័ន្ធ៖

![System email editor](/img/admin/system-email-editor.png)

## ការនាំចូល قالبអ៊ីមែលថ្មី {#importing-new-email-templates}

កំណែថ្មីនៃ Ultimate Multisite និង add-ons អាចចុះបញ្ជី قالبអ៊ីមែលថ្មីៗ។ ដើម្បីការពារការប៉ះទង្គិចគ្នា (conflicts) ក قالبថ្មីនឹងមិនត្រូវបានបន្ថែមដោយស្វ័យប្រវត្តិទេ — អ្នកត្រូវនាំចូលវាដោយដៃ។

ដើម្បីនាំចូល قالبអ៊ីមែលថ្មី៖

1. ទៅកាន់ទំព័រ **System Emails**
2. ចុចលើប៊ូតុង **Reset & Import** នៅខាងលើទំព័រ
3. បើកជម្រើស **Import Emails** ដើម្បីមើល قالبដែលមានសម្រាប់ប្រើប្រាស់
4. ជ្រើសរើស قالبដែលអ្នកចង់នាំចូល ហើយបញ្ជាក់ការជ្រើសរើស

## ការកំណត់ឡើងវិញនូវ قالبអ៊ីមែល {#resetting-email-templates}

ប្រសិនបើអ្នកបានកែសម្រួល قالبអ៊ីមែលមួយហើយ ហើយចង់ត្រឡប់ទៅខ្លឹមសារដើមវិញ អ្នកមានជម្រើសពីរ៖

1. **លុបហើយនាំចូលឡើងវិញ** — លុបអ៊ីមែលប្រព័ន្ធនោះចេញ ហើយនាំចូលវាម្ដងទៀត (វានឹងលុបទិន្នន័យនៃការផ្ញើ)
2. **កំណត់ឡើងវិញតាមរយៈឧបករណ៍** — ប្រើឧបករណ៍ **Reset & Import**, បើកជម្រើស **Reset**, ហើយជ្រើសរើសអ៊ីមែលដែលអ្នកចង់កំណត់ឡើងវិញ។

ជម្រើស Reset ត្រូវបានណែនាំជាង ព្រោះវារក្សាទុកទិន្នន័យនៃការផ្ញើ និងព័ត៌មានផ្សេងៗទៀត ខណៈពេលដែលបានស្ដារខ្លឹមសារដើមឡើងវិញ។
