# бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһўбҹҠбһёбһҳбҹӮбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’

Ultimate Multisite бһ•бҹ’бһүбһҫбһўбҹҠбһёбһҳбҹӮбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһҹбҹҒбһ„бҹ—бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһ“бһ¶бһ“бһ¶бһҠбһјбһ…бһҮбһ¶ бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҖбһ¶бһҡбһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮ, бһӣбһ·бһҒбһ·бһҸбһ‘бһ‘бһҪбһӣбһӣбһ»бһҷ, бһ“бһ·бһ„бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ—бһ¶бһ–бһҮбһ¶бһҹбһҳбһ¶бһҮбһ·бһҖбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣ ЩӮШ§Щ„ШЁ (template) бһўбҹҠбһёбһҳбҹӮбһӣбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһұбҹ’бһҷбһҸбҹ’бһҡбһјбһңбһ“бһ№бһ„бһҳбҹүбһ¶бһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бҹ”

## бһҖбһ¶бһҡбһҳбһҫбһӣбһўбҹҠбһёбһҳбҹӮбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ {#viewing-system-emails}

бһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢ **Ultimate Multisite вҶ’ Settings** бһ бһҫбһҷбһ…бһ»бһ…бһӣбһҫа№Ғаё—а№Үаёҡ **Emails** бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҠбҹӮбһӣбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„бһўбҹҠбһёбһҳбҹӮбһӣбҹ”

![Settings - Emails tab](/img/admin/settings-emails.png)

бһ–бһёбһ‘бһёбһ“бҹ„бҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ‘бҹҶбһ–бҹҗбһҡ **System Emails** бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣ ЩӮШ§Щ„ШЁбһўбҹҠбһёбһҳбҹӮбһӣбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ”бһ¶бһ“бҹ”

![System Emails list](/img/admin/system-emails-list.png)

бһўбҹ’бһ“бһҖбһҖбҹҸбһўбһ¶бһ…бһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣ ЩӮШ§Щ„ШЁбһўбҹҠбһёбһҳбҹӮбһӣбһ“бһёбһҳбһҪбһҷбҹ—бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бһ·бһ–бһ“бҹ’бһ’ (editor) бһўбҹҠбһёбһҳбҹӮбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бҹ–

![System email editor](/img/admin/system-email-editor.png)

## бһҖбһ¶бһҡбһ“бһ¶бҹҶбһ…бһјбһӣ ЩӮШ§Щ„ШЁбһўбҹҠбһёбһҳбҹӮбһӣбһҗбҹ’бһҳбһё {#importing-new-email-templates}

бһҖбҹҶбһҺбҹӮбһҗбҹ’бһҳбһёбһ“бҹғ Ultimate Multisite бһ“бһ·бһ„ add-ons бһўбһ¶бһ…бһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһё ЩӮШ§Щ„ШЁбһўбҹҠбһёбһҳбҹӮбһӣбһҗбҹ’бһҳбһёбҹ—бҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһ¶бһҡбһ”бҹүбҹҮбһ‘бһ„бҹ’бһӮбһ·бһ…бһӮбҹ’бһ“бһ¶ (conflicts) бһҖ ЩӮШ§Щ„ШЁбһҗбҹ’бһҳбһёбһ“бһ№бһ„бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ‘бҹҒ вҖ” бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ“бһ¶бҹҶбһ…бһјбһӣбһңбһ¶бһҠбҹ„бһҷбһҠбҹғбҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһ“бһ¶бҹҶбһ…бһјбһӣ ЩӮШ§Щ„ШЁбһўбҹҠбһёбһҳбҹӮбһӣбһҗбҹ’бһҳбһёбҹ–

1. бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ‘бҹҶбһ–бҹҗбһҡ **System Emails**
2. бһ…бһ»бһ…бһӣбһҫбһ”бҹҠбһјбһҸбһ»бһ„ **Reset & Import** бһ“бҹ…бһҒбһ¶бһ„бһӣбһҫбһ‘бҹҶбһ–бҹҗбһҡ
3. бһ”бһҫбһҖбһҮбһҳбҹ’бһҡбһҫбһҹ **Import Emails** бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣ ЩӮШ§Щ„ШЁбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ
4. бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ ЩӮШ§Щ„ШЁбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ“бһ¶бҹҶбһ…бһјбһӣ бһ бһҫбһҷбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ

## бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһЎбһҫбһ„бһңбһ·бһүбһ“бһјбһң ЩӮШ§Щ„ШЁбһўбҹҠбһёбһҳбҹӮбһӣ {#resetting-email-templates}

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣ ЩӮШ§Щ„ШЁбһўбҹҠбһёбһҳбҹӮбһӣбһҳбһҪбһҷбһ бһҫбһҷ бһ бһҫбһҷбһ…бһ„бҹӢбһҸбҹ’бһҡбһЎбһ”бҹӢбһ‘бҹ…бһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбһҠбһҫбһҳбһңбһ·бһү бһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҮбһҳбҹ’бһҡбһҫбһҹбһ–бһёбһҡбҹ–

1. **бһӣбһ»бһ”бһ бһҫбһҷбһ“бһ¶бҹҶбһ…бһјбһӣбһЎбһҫбһ„бһңбһ·бһү** вҖ” бһӣбһ»бһ”бһўбҹҠбһёбһҳбҹӮбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ“бҹ„бҹҮбһ…бҹҒбһү бһ бһҫбһҷбһ“бһ¶бҹҶбһ…бһјбһӣбһңбһ¶бһҳбҹ’бһҠбһ„бһ‘бҹҖбһҸ (бһңбһ¶бһ“бһ№бһ„бһӣбһ»бһ”бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹғбһҖбһ¶бһҡбһ•бҹ’бһүбһҫ)
2. **бһҖбҹҶбһҺбһҸбҹӢбһЎбһҫбһ„бһңбһ·бһүбһҸбһ¶бһҳбһҡбһҷбҹҲбһ§бһ”бһҖбһҡбһҺбҹҚ** вҖ” бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ **Reset & Import**, бһ”бһҫбһҖбһҮбһҳбҹ’бһҡбһҫбһҹ **Reset**, бһ бһҫбһҷбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһўбҹҠбһёбһҳбҹӮбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһҖбҹҶбһҺбһҸбҹӢбһЎбһҫбһ„бһңбһ·бһүбҹ”

бһҮбһҳбҹ’бһҡбһҫбһҹ Reset бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҺбҹӮбһ“бһ¶бҹҶбһҮбһ¶бһ„ бһ–бҹ’бһҡбҹ„бҹҮбһңбһ¶бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹғбһҖбһ¶бһҡбһ•бҹ’бһүбһҫ бһ“бһ·бһ„бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бҹҖбһҸ бһҒбһҺбҹҲбһ–бҹҒбһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһҹбҹ’бһҠбһ¶бһҡбһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбһҠбһҫбһҳбһЎбһҫбһ„бһңбһ·бһүбҹ”
