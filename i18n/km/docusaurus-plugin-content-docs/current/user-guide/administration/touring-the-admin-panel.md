# ការដើរលេងក្នុង Admin Panel

Ultimate Multisite គឺមានគោលបំណងឱ្យវាសាមញ្ញ និងងាយយល់បំផុត ប៉ុន្តែដោយសារតែឧបករណ៍ដ៏មានឥទ្ធិពលនេះ វាអាចជួយឱ្យអ្នកមានការណែនាំទស្សនកិច្ច។ សូមយើងដើរមើលទំព័រ admin ដើម្បីជួយអ្នកឱ្យសម្រួលខ្លួនបាន។

## Dashboard (ផ្ទាំងគ្រប់គ្រង) {#dashboard}

**Dashboard** របស់ Ultimate Multisite បង្ហាញរបាយការណ៍ និងស្ថិតិជាមូលដ្ឋាន រួមមានចំណូល សកម្មភាពគេហទំព័រ ការលូតលាស់សមាជិក ចំនួនអ្នកចូលមើល និងទិន្នន័យភូមិសាស្ត្រ។

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

នេះគឺជាទិដ្ឋភាពពេញលេញនៃ dashboard៖

![Ultimate Multisite Dashboard full page](/img/admin/um-dashboard-full.png)

ហើយផ្នែកខាងក្រោមនៃ dashboard៖

![Ultimate Multisite Dashboard bottom section](/img/admin/um-dashboard-bottom.png)

នេះក៏ជាទិដ្ឋភាពពេញលេញនៃ network dashboard ផងដែរ៖

![Network Dashboard full page](/img/admin/network-dashboard-full-page.png)

## Checkout Forms (ទម្រង់បង់ប្រាក់) {#checkout-forms}

**Checkout Forms** ផ្តល់ឱ្យអ្នកនូវភាពបត់បែនក្នុងការបង្កើតទំព័រចុះឈ្មោះផ្ទាល់ខ្លួន។ អ្នកអាចកែសម្រួលវាលបញ្ចូលទិន្នន័យ បញ្ចូល class និង script ផ្ទាល់ខ្លួន ហើយកំណត់ការចូលប្រើដោយផ្អែកលើប្រទេស ឬទីតាំងរបស់អ្នកចូលមើល។

![Checkout Forms list](/img/admin/checkout-forms-list.png)

## Products (ផលិតផល) {#products}

ផ្នែក **Products** គឺជាកន្លែងដែលអ្នកបង្កើតផែនការផ្សេងៗ កញ្ចប់ និងសេវាកម្មសម្រាប់ network របស់អ្នក។ កំណត់តម្លៃ ការបង់ប្រាក់ជាប្រចាំ និងដែនកំណត់/ quotas នៅកម្រិតផលិតផល។

![Products list](/img/admin/products-list.png)

## Memberships (សមាជិកភាព) {#memberships}

ទំព័រ **Memberships** បង្ហាញការជាវទាំងអស់នៅក្នុង network របស់អ្នក។ សូមពិនិត្យមើល និងកែសម្រួលព័ត៌មាននៃការជាវ រួមទាំងផែនការ ផលិតផល ចំនួនទឹកប្រាក់បង់ប្រាក់ និងភាពញឹកញាប់ ការប្រវត្តិការទូទាត់ និងពេលវេលាដែលបានធ្វើបច្ចុប្បន្នភាព។

![Memberships list](/img/admin/memberships-list.png)

## Payments (ការបង់ប្រាក់) {#payments}

ទំព័រ **Payments** ផ្តល់នូវការមើលលឿននៃប្រវត្តិការបង់ប្រាក់ទូទាំងបណ្តាញរបស់អ្នក រួមជាមួយព័ត៌មានប្រតិបត្តិការលម្អិត រួមទាំងផលិតផល និងចំនួនទឹកប្រាក់ជាក់លាក់។

![Payments list](/img/admin/payments-list.png)

## Customers {#customers}

ទំព័រ **Customers** បង្ហាញសមាជិកទាំងអស់ដែលបានចុះឈ្មោះនៅក្នុងបណ្តាញរបស់អ្នក រួមជាមួយព័ត៌មានអ្នកប្រើប្រាស់មូលដ្ឋាន រួមមាន ឈ្មោះ អ៊ីម៉ែលចុងក្រោយពេលចូល និងតំណ "switch to" ដើម្បីចូលទៅកាន់ subsites របស់ពួកគេ។

![Customers list](/img/admin/customers-list.png)

## Sites {#sites}

ទំព័រ **Sites** រាយបញ្ជី subsites ទាំងអស់នៅក្នុងបណ្តាញរបស់អ្នក រួមទាំង site templates និង subsites ដែលជាកម្មសិទ្ធិរបស់អតិថិជន។ អ្នកអាចគ្រប់គ្រង mapped domains, limits & quotas, plugins និង themes ក៏ដូចជា thumbnails នៃ site បានយ៉ាងងាយស្រួល។

![Sites list](/img/admin/sites-list.png)

## Domains {#domains}

ទំព័រ **Domains** ត្រូវបានបម្រុងទុកសម្រាប់ custom domains ដែលបានភ្ជាប់ទៅនឹង subsites ។ ក្នុងនាមជា super admin អ្នកអាចបន្ថែម ឬ map custom domain names ដោយដៃ និងមើល DNS records និង logs បាន។

![Domains list](/img/admin/domains-list.png)

## Discount Codes {#discount-codes}

**Discount Codes** អនុញ្ញាតឱ្យអ្នកបង្កើត coupon codes ដើម្បីផ្តល់ការបញ្ចុះតម្លៃដល់អ្នកប្រើប្រាស់ចុងក្រោយរបស់អ្នក។ កំណត់តម្លៃនៃការបញ្ចុះតម្លៃ និងកំណត់គោលដៅទៅកាន់អ្នកប្រើប្រាស់ ឬក្រុមជាក់លាក់ណាមួយ។

![Discount Codes list](/img/admin/discount-codes-list.png)

## Broadcasts {#broadcasts}

**Broadcasts** គឺជាឧបករណ៍សម្រាប់ផ្សព្វផ្សាយការជូនដំណឹង ឬផ្ញើសារឯកជនទៅកាន់អ្នកប្រើប្រាស់ subsites របស់អ្នក។ កំណត់ក្រុមអ្នកប្រើប្រាស់ជាក់លាក់ដោយផ្អែកលើផែនការ (plan) ឬផលិតផលដែលពួកគេបានចុះឈ្មោះ។

![Broadcasts list](/img/admin/broadcasts-list.png)

## Settings {#settings}

ទំព័រ **Settings** គឺជាកន្លែងដែលអ្នកកំណត់រចនាសម្ព័ន្ធ Ultimate Multisite — ការកំណត់នៃការចុះឈ្មោះ ការបង់ប្រាក់ API និង webhooks ការ map domain និងការរួមបញ្ចូលផ្សេងៗទៀត។

ការកំណត់ AI connector បញ្ជីអ្នកផ្តល់ OAuth ដែលគាំទ្របច្ចុប្បន្ន៖ Anthropic Max, OpenAI ChatGPT/Codex, និង Google AI Pro។ កាតអ្នកផ្តល់នីមួយៗអនុញ្ញាតឱ្យ super admins ភ្ជាប់គណនី (connect accounts) បទភ្ជាប់ឡើងវិញនូវគណនីដែលបានរក្សាទុក (refresh saved accounts) លុបគណនីតាមអ៊ីមែល (remove accounts by email) និងប្រើ fallback OAuth ដោយដៃ (manual OAuth fallback) នៅពេលដែលបរិយាកាស sandboxed រារាំងការបញ្ជូនប៊ូតុងកម្មវិធីរុករក (browser redirect)។ ជម្រើសការកំណត់សម្រាប់ Cursor Pro ត្រូវបានលុបចេញពី admin panel ហើយ។

គណនី ChatGPT/Codex គាំទ្រការប្រើប្រាស់ឧបករណ៍ដែលគាំទ្រដោយ connector ដែលអនុញ្ញាតឱ្យប្រតិបត្តិការប្រើឧបករណ៍ (tool use)។ ដូច្នេះ Workflow របស់ admin ដែលពឹងផ្អែកលើប្រតិបត្តិការដែលគាំទ្រដោយ connector អាចប្រើឥរិយាបថឧបករណ៍ Codex បាន បន្ទាប់ពីគណនី OpenAI ត្រូវបានភ្ជាប់រួចហើយ។

## Events {#events}

ទំព័រ **Events** រក្សាទុកកំណត់ត្រានៃព្រឹត្តិការណ៍ និង log ទាំងអស់នៅក្នុង network របស់អ្នក។ វាតាមដានសកម្មភាពដូចជា ការផ្លាស់ប្តូរផែនការ (plan changes), ការចុះឈ្មោះចូលប្រើប្រាស់ (signups) និងចលនាក្រោមផ្សេងទៀត — មានប្រយោជន៍សម្រាប់ការត្រួតពិនិត្យ multisite របស់អ្នក។

![Events list](/img/admin/events-list.png)

## Webhooks {#webhooks}

**Webhooks** អនុញ្ញាតឱ្យអ្នកបញ្ជូនទិន្នន័យទៅកម្មវិធីផ្សេងទៀត។ មានប្រយោជន៍សម្រាប់ការផ្ញើទិន្នន័យពី Ultimate Multisite ទៅកាន់ platform ដូចជា Zapier។

![Webhooks list](/img/admin/webhooks-list.png)
