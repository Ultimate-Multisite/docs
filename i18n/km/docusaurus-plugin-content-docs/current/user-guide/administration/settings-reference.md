# ការយោងការកំណត់រចនាសម្ព័ន្ធ (Settings Reference)

ទំព័រនេះតាមដានការកំណត់រចនាសម្ព័ន្ធដែលប៉ះពាល់ដល់ការគ្រប់គ្រងប្រចាំថ្ងៃ និងការផ្លាស់ប្តូរអាកប្បកិរិយាថ្មីៗនៅក្នុង Ultimate Multisite។

## ជម្រើសផ្សេងទៀត (Other Options)

តំបន់ **Other Options** នឹងបង្ហាញនៅក្រោម **Ultimate Multisite > Settings > Login & Registration**។

| ការកំណត់រចនាសម្ព័ន្ធ | ការពិពណ៌នា |
|---|---|
| **Enable Jumper** | បង្ហាញឧបករណ៍ Quick navigation ឈ្មោះ Jumper នៅក្នុងតំបន់ admin។ ប្រើវាដើម្បីលោតត្រង់ទៅកាន់អេក្រង់ Ultimate Multisite, network objects និងគោលដៅ admin ដែលគាំទ្រ។ បិទវាប្រសិនបើអ្នកមិនចង់ឱ្យផ្លូវកាត់នេះបង្ហាញទេ។ |

## ការរាយការណ៍អំពីកំហុស និង telemetry (Telemetry)

ការកំណត់ដែលអនុញ្ញាតឱ្យចូលរួមក្នុងការរាយការណ៍អំពីកំហុសពីមុនត្រូវបានលុបចេញពីទំព័រ settings។ Telemetry អាណត្តិសម្ងាត់ (Anonymous telemetry) ត្រូវបានបិទ ហើយមិនមាន UI toggle ដើម្បីបើកវាទេ។

ប្រសិនបើអ្នករក្សាទុក runbooks ឬរូបថតអេក្រង់សម្រាប់ទំព័រ settings សូមយកការយោងទៅកាន់វាល error-reporting opt-in ចាស់ចេញ ដើម្បីឱ្យអ្នកគ្រប់គ្រងមិនស្វែងរកការកំណត់រចនាសម្ព័ន្ធដែលគ្មានវត្តមានទៀតឡើយ។

## ការកំណត់ Import/Export (Import/Export settings)

ផ្ទាំង **Import/Export** ពិពណ៌នាអំពីការកំណត់រចនាសម្ព័ន្ធដែលវាគ្រប់គ្រង និងភ្ជាប់ដោយផ្ទាល់ទៅនឹង **Ultimate Multisite > Site Export** សម្រាប់ការបណ្ណាល័យ site និង network។ ប្រើផ្ទាំង settings សម្រាប់ការកំណត់រចនាសម្ព័ន្ធ import/export, ប្រើ **Tools > Export & Import** សម្រាប់លំហូរការងារ export/import សម្រាប់ single-site, ហើយប្រើឧបករណ៍ Site Export នៅពេលអ្នកត្រូវការ archive បញ្ចេញពេញលេញនៃ Network។

## ការកំណត់រចនាសម្ព័ន្ធ AI provider connector (AI provider connector settings)

ការកំណត់រចនាសម្ព័ន្ធ AI provider connector ឥឡូវនេះបង្ហាញតែ OAuth account pools ដែលគាំទ្រប៉ុណ្ណោះ៖

| ผู้ให้บริการ | ขั้นตอนการตั้งค่า |
|---|---|
| **Anthropic Max** | เชื่อมต่อบัญชี Anthropic Max หนึ่งบัญชีหรือมากกว่าด้วยปุ่ม OAuth ใช้ตัวเลือก OAuth แบบแมนนวลเป็นทางเลือกสำรอง หากเบราว์เซอร์แบบ sandboxed ไม่สามารถทำ redirect ได้โดยอัตโนมัติ |
| **OpenAI ChatGPT/Codex** | เชื่อมต่อบัญชี ChatGPT ผ่านเวิร์กโฟลว์ OAuth เดียวกัน การดำเนินการที่รองรับด้วย Connector สามารถใช้เครื่องมือเรียกของ ChatGPT Codex หลังจากเชื่อมต่อบัญชีแล้ว |
| **Google AI Pro** | เชื่อมต่อบัญชี Google AI Pro ผ่าน OAuth จากนั้นรีเฟรช connector หากรายการบัญชีไม่ได้รับการอัปเดตทันที |

Cursor Pro ไม่ใช่ผู้ให้บริการที่รองรับอีกต่อไป โปรดลบภาพหน้าจอภายใน, runbooks หรือขั้นตอนการเริ่มต้นใช้งานเก่าๆ ที่กล่าวถึงฟิลด์การตั้งค่าหรือเส้นทางของ connector สำหรับการตั้งค่า Cursor Pro ออกไป

เมื่อเพิ่มหรือลบบัญชีผู้ให้บริการ ให้กรอกที่อยู่อีเมลที่ถูกต้องสำหรับบัญชีที่จะรีเฟรชหรือลบทิ้ง และบันทึกการตั้งค่าของผู้ให้บริการก่อนทดสอบการทำงานที่มี Connector รองรับ
