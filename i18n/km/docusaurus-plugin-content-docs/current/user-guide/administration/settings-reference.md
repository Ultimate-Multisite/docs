# ឯកសារយោងការកំណត់

ទំព័រនេះតាមដានការកំណត់ដែលប៉ះពាល់ដល់ការគ្រប់គ្រងប្រចាំថ្ងៃ និងការផ្លាស់ប្តូរឥរិយាបថថ្មីៗក្នុង Ultimate Multisite។

## ជម្រើសផ្សេងទៀត {#other-options}

តំបន់ **ជម្រើសផ្សេងទៀត** បង្ហាញនៅក្រោម **Ultimate Multisite > ការកំណត់ > ចូល និងចុះឈ្មោះ**។

| ការកំណត់ | ការពិពណ៌នា |
|---|---|
| **បើក Jumper** | បង្ហាញឧបករណ៍រុករករហ័ស Jumper នៅក្នុងតំបន់គ្រប់គ្រង។ ប្រើវាដើម្បីលោតទៅកាន់អេក្រង់ Ultimate Multisite វត្ថុបណ្តាញ និងគោលដៅគ្រប់គ្រងដែលបានគាំទ្រ ដោយផ្ទាល់។ បិទវា ប្រសិនបើអ្នកមិនចង់ឱ្យឃើញផ្លូវកាត់នេះ។ |

## ការរាយការណ៍កំហុស និង telemetry {#error-reporting-and-telemetry}

ការកំណត់ចូលរួមសម្រាប់ការរាយការណ៍កំហុសពីមុន ត្រូវបានដកចេញពីទំព័រការកំណត់ហើយ។ telemetry អនាមិកត្រូវបានបិទ ហើយមិនមានប៊ូតុង UI ដើម្បីបើកវាទេ។

ប្រសិនបើអ្នកថែរក្សាសៀវភៅណែនាំប្រតិបត្តិការផ្ទៃក្នុង ឬរូបថតអេក្រង់សម្រាប់ទំព័រការកំណត់ សូមដកការយោងទៅកាន់វាលចូលរួមការរាយការណ៍កំហុសចាស់ចេញ ដើម្បីកុំឱ្យអ្នកគ្រប់គ្រងស្វែងរកការកំណត់ដែលលែងមានទៀត។

## ការកំណត់នាំចូល/នាំចេញ {#importexport-settings}

ផ្ទាំងការកំណត់ **នាំចូល/នាំចេញ** ពិពណ៌នាអំពីការកំណត់ណាដែលវាគ្រប់គ្រង ហើយភ្ជាប់ដោយផ្ទាល់ទៅ **Ultimate Multisite > ការនាំចេញសាយត៍** សម្រាប់បណ្ណសារសាយត៍ និងបណ្តាញ។ ប្រើផ្ទាំងការកំណត់សម្រាប់ការកំណត់រចនាសម្ព័ន្ធនាំចូល/នាំចេញ ប្រើ **ឧបករណ៍ > នាំចេញ និងនាំចូល** សម្រាប់ដំណើរការនាំចេញ/នាំចូលសាយត៍តែមួយ ហើយប្រើឧបករណ៍នាំចេញសាយត៍ នៅពេលអ្នកត្រូវការបណ្ណសារ Network Export ពេញលេញ។

## ការព្រមានអំពីសមតុល្យ Domain Seller HostAfrica {#domain-seller-hostafrica-balance-warning}

នៅពេល addon Domain Seller ត្រូវបានភ្ជាប់ទៅ HostAfrica អ្នកគ្រប់គ្រងបណ្តាញនឹងឃើញការព្រមានសមតុល្យទាបដែលអាចបិទបាន នៅពេលសមតុល្យគណនីអ្នកលក់បន្តទាបពេកសម្រាប់ការចុះឈ្មោះដូមែន ឬដំណើរការបន្តសុពលភាពដែលអាចទុកចិត្តបាន។

ចាត់ទុកសេចក្តីជូនដំណឹងនេះជាការព្រមានប្រតិបត្តិការ៖ បន្ថែមសមតុល្យអ្នកលក់បន្ត HostAfrica មុនពេលទទួលការចុះឈ្មោះដូមែនដែលបានបង់ប្រាក់បន្ថែម បន្ទាប់មកត្រឡប់ទៅការកំណត់ Domain Seller ឬអេក្រង់តាមដានដូមែន ដើម្បីបញ្ជាក់ថាការចុះឈ្មោះ និងការបន្តសុពលភាពអាចបន្តដំណើរការបានធម្មតា។
