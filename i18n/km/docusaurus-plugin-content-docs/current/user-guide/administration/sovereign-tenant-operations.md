# бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһўбҹ’бһ“бһҖбһҮбһҪбһӣбһўбһ’бһ·бһ”бһҸбҹҒбһҷбҹ’бһҷ (Sovereign Tenant Operations)

Ultimate Multisite: Multi-Tenancy 1.2.0 бһ”бһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһҪбһӣбһўбһ’бһ·бһ”бһҸбҹҒбһҷбҹ’бһҷбҹ– subsites бһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (database) бһ”бҹ’бһӣбҹӮбһҖбҹ—бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“, бһ”бҹ’бһҡбһ—бһ–бһҜбһҖбһҹбһ¶бһҡ (filesystem root) бһ“бһ·бһ„бһ”бһҡбһ·бһ”бһ‘бһ“бҹғбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ•бҹ’бһӣбһјбһң (routing context) бһҡбһ”бһҹбҹӢбһңбһ¶ бһҒбһҺбҹҲбһҠбҹӮбһӣбһ“бҹ…бһҸбҹӮбһўбһ¶бһ…бһҳбһҫбһӣбһғбһҫбһүбһ–бһёбһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһҺбҹ’бһҸбһ¶бһүбҹ”

бһҹбһјбһҳбһ”бҹ’бһҡбһҫаё«аёҷа№үаёІаёҷГ yбһ“бҹ…бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбһҸбһ·бһҗбһ·бһҮбһ“бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹҶбһ”бҹӮбһҖбһ…бҹҒбһү, бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҒбһҡбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһ–бһёбһ…бһҳбҹ’бһ„бһ¶бһҷ (remote-site handoffs) бһ¬бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ (migrations) бһҠбҹӮбһӣбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһё subsites бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһўбһ’бһ·бһ”бһҸбҹҒбһҷбҹ’бһҷбҹ”

## бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ (What changes for administrators) {#what-changes-for-administrators}

- **бһҖбһ¶бһҡбһ…бһјбһӣбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– (Stateless tenant autologin)** вҖ” бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһҺбҹ’бһҸбһ¶бһүбһўбһ¶бһ…бһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢ tenant бһўбһ’бһ·бһ”бһҸбҹҒбһҷбҹ’бһҷбһ”бһ¶бһ“бһҠбҹ„бһҷбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– session бһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһҷбҹҲбһ–бҹҒбһӣбһҷбһјбһҡбҹ” SSO token бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһңбһ·бһҹбһ¶бһӣбһ—бһ¶бһ–бһҸбһ¶бһҳбһӮбҹ„бһӣбһ”бҹҶбһҺбһ„, бһҖбҹҶбһҺбһҸбҹӢбһ”бҹ’бһҡбһ—бһ–бһҠбһҫбһҳ (origin-pinned), бһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһ¶бһҡбһӣбҹҒбһ„бһЎбһҫбһ„бһңбһ·бһү (replay-protected) бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ•бһ»бһҸбһҖбҹҶбһҺбһҸбҹӢбһҒбҹ’бһӣбһёбҹ—бҹ”
- **бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ•бҹ’бһӣбһјбһңбһҠбҹӮбһӣбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһўбһ’бһ·бһ”бһҸбҹҒбһҷбҹ’бһҷ (Sovereign-aware routing)** вҖ” бһ”бһҺбҹ’бһҸбһ¶бһүбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹҶбһ”бҹӮбһҖбһ…бҹҒбһүбһ–бһёбһҳбһ»бһ“ бһ“бһ·бһ„ tenant бһўбһ’бһ·бһ”бһҸбҹҒбһҷбҹ’бһҷ бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҸбһ¶бһҳбһҡбһҷбҹҲбһ•бҹ’бһӣбһјбһң router бһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһҠбҹӮбһӣбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ“бҹғбһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһЎбһҫбһ„бһңбһ·бһү (bootstrap differences) бһҡбһңбһ¶бһ„бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„бһ…бһ¶бһҹбҹӢ бһ“бһ·бһ„бһҗбҹ’бһҳбһёбҹ”
- **бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ (Verified migration state)** вҖ” бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ“бһ№бһ„бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһҹбһ·бһ‘бҹ’бһ’бһ·бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ, бһҹбһ·бһ‘бҹ’бһ’бһ·бһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (database writer permissions), бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһјбһҡбһҮбһҪбһҡ (queue drain status) бһ“бһ·бһ„бһҖбһ¶бһҡбһўбһңбһҸбҹ’бһҸбһҳбһ¶бһ“бһ“бҹғбһҸбһ¶бһҡбһ¶бһ„бһ…бһ¶бһҹбҹӢбҹ— бһҳбһ»бһ“бһ–бҹҒбһӣбһӮбҹҒбһ…бһ¶бһҸбҹӢбһ‘бһ»бһҖ tenant бһҗбһ¶бһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢбҹ”
- **бһҖбһ¶бһҡбһ”бҹ„бҹҮбһ”бһ„бҹӢбһҖбһ¶бһ“бҹӢбһҸбҹӮбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– (Safer teardown)** вҖ” бһҖбһ¶бһҡбһ”бҹ„бҹҮбһ”бһ„бҹӢ tenant бһўбһ’бһ·бһ”бһҸбҹҒбһҷбҹ’бһҷбһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһ“бһ№бһ„бһӣбһ»бһ”бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҡбһ”бһҹбҹӢ tenant бһ…бҹҒбһүбһҠбҹ„бһҷбһҹбҹ’бһўбһ¶бһҸ бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһ»бҹҶбһұбҹ’бһҷ tenant бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӣбһ»бһ”бһ“бҹ…бһҸбҹӮбһ‘бһ»бһҖбһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһ”бҹ’бһҡбһҫбһ‘бҹҖбһҸбҹ”

## бһҖбһ¶бһҡбһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢ tenant бһўбһ’бһ·бһ”бһҸбҹҒбһҷбҹ’бһҷ (Visiting a sovereign tenant) {#visiting-a-sovereign-tenant}

1. бһ”бһҫбһҖ **Network Admin > Ultimate Multisite > Sites**бҹ”
2. бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ tenant бһўбһ’бһ·бһ”бһҸбҹҒбһҷбҹ’бһҷбһ“бҹ„бҹҮбҹ”
3. бһ”бҹ’бһҡбһҫ **Visit (SSO)** бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһҮбҹҶбһ“бһҪбһҹбһҖбһ¶бһҡбһ…бһҳбҹ’бһӣбһ„бһ–бһ¶бһҖбҹ’бһҷбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ бһ¬бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһҺбһ“бһёбһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһҺбҹ’бһҸбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“бҹ”

бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ…бһјбһӣбһ“бҹҒбҹҮбһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸ token бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бһјбһӣбһҒбҹ’бһӣбһёбҹ—бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ tenant бһ“бҹ„бҹҮ бһ бһҫбһҷбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ SSO бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ audit trail бһҡбһ”бһҹбҹӢ tenantбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ”бҹҠбһјбһҸбһ»бһ„бһ”бһҡбһ¶бһҮбҹҗбһҷ бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҗбһ¶бһҸбһҫ domain бһҡбһ”бһҹбҹӢ tenant бһ“бҹ„бҹҮбһўбһ¶бһ…бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„бһҠбҹӮбһӣбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбһҠбҹӮбһҡбһ¬бһ‘бҹҒ бһ бһҫбһҷбһҗбһ¶бһҸбһҫ tenant бһ“бҹ„бҹҮбһўбһ¶бһ…бһ‘бҹ…бһҠбһӣбҹӢ endpoint SSO бһ“бҹ…бһҒбһ¶бһ„бһ”бһҺбҹ’бһҸбһ¶бһүбһ”бһ¶бһ“бһҠбҹӮбһҡбһ¬бһ‘бҹҒбҹ”

## бһ”бһүбҹ’бһҮбһёбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ site бһ–бһёбһ…бһҳбҹ’бһ„бһ¶бһҷ (Remote-site operations checklist) {#remote-site-operations-checklist}

бһҳбһ»бһ“бһ“бһ№бһ„бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ tenant бһўбһ’бһ·бһ”бһҸбҹҒбһҷбҹ’бһҷ бһ¬ remote tenant бһҹбһјбһҳбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбҹ–

- бһҠбҹӮбһ“бһҠбһёбһўбҹ’бһ“бһҖбһҮбһҪбһӣ (tenant domain) бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ‘бҹ…бһҖбһ¶бһ“бҹӢ Host бһҠбҹӮбһӣбһҮбһ¶бһҳбҹ’бһ…бһ¶бһҹбҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҜбһҖбһҹбһ¶бһҡбһҡбһ”бһҹбҹӢ tenantбҹ”
- Host бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ tenant бһҸбҹ’бһҡбһјбһңбһ“бһ№бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ host binding бһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„бһ“бҹ„бҹҮбҹ”
- бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ (migration verification commands) бһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ tenantбҹ”
- бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбһҪбһҡбһҳбҹҒбһўбҹҠбһ·бһҖ (Async migration queues) бһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһ бһјбһҡбһ…бҹҒбһүбһҳбһ»бһ“бһ–бҹҒбһӣбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ DNS бһ¬бһҖбһҳбҹ’бһҳбһҹбһ·бһ‘бҹ’бһ’бһ· (ownership changes)бҹ”
- бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ admin tenant бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҖбҹ’бһ“бһ»бһ„бһўбҹҶбһЎбһ»бһ„бһ–бҹҒбһӣбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ бһ бһҫбһҷбһўбһ¶бһ…бһ…бһјбһӣбһ”бһ¶бһ“бһҸбһ¶бһҳбһҡбһҷбҹҲ SSOбҹ”

## бһҖбһ¶бһҡбһӣбһ»бһ” tenant бһҠбҹӮбһӣбһҳбһ¶бһ“бһўбҹҶбһҺбһ¶бһ… (Deleting sovereign tenants) {#deleting-sovereign-tenants}

бһҖбһ¶бһҡбһӣбһ»бһ” tenant бһҠбҹӮбһӣбһҳбһ¶бһ“бһўбҹҶбһҺбһ¶бһ…бһӮбһәбһҮбһ¶бһҡбһҝбһ„бһҠбҹӮбһӣбһҳбһ·бһ“бһўбһ¶бһ…бһҷбһҖбһҳбһҖбһңбһ·бһүбһ”бһ¶бһ“ (destructive)бҹ” бһҹбһјбһҳбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– backup бһ“бһ·бһ„ export бһҮбһ¶бһҳбһ»бһ“бһҹбһ·бһ“ бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһӣбһ»бһ”бһ…бҹҒбһүбһ–бһёбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡ (site management screen)бҹ” бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ teardown 1.2.0 бһ“бһ№бһ„бһӣбһ»бһ” credentials бһ“бҹғ tenant database бһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһ“бҹғбһҖбһ¶бһҡбһҹбһҳбҹ’бһўбһ¶бһҸ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ“бҹ…бһҸбҹӮбһӮбһҪбһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҗбһ¶ user бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„ folder бһ“бҹ…бһӣбһҫ host бһ”бһ¶бһ“бһӣбһ»бһ”бһўбһҹбҹӢбһ бһҫбһҷбһ“бҹ…бһ–бҹҒбһӣбһ”бҹ’бһҡбһҫ external hosting panelsбҹ”

:::warning
бһҖбһ»бҹҶбһӣбһ»бһ” tenant бһҠбҹӮбһӣбһҳбһ¶бһ“бһўбҹҶбһҺбһ¶бһ… бһҒбһҺбҹҲбһ–бҹҒбһӣбһҠбҹӮбһӣбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ (migration verification) бһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ¬бһҒбһҺбҹҲбһ–бҹҒбһӣбһҠбҹӮбһӣбһҳбһ¶бһ“ job push бһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„бһҮбһҪбһҡбһҳбҹҒбһўбҹҠбһ·бһҖ (async queues)бҹ” бһҹбһјбһҳбһҡбһ„бҹӢбһ…бһ¶бҹҶбһұбҹ’бһҷбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ”бһүбҹ’бһ…бһ”бҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ»бҹҶбһұбҹ’бһҷ teardown бһӣбһ»бһ” credentials бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ job бһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһҡбһ„бҹӢбһ…бһ¶бҹҶбҹ”
:::
