# ការគ្រប់គ្រងការទូទាត់ និងវិក្កយបត្រ

## ការកំណត់នៃការទូទាត់ (Payment Settings)

មុនពេលអ្នកចាប់ផ្តើមទទួលការទូទាត់ អ្នកត្រូវកំណត់រចនាសម្ព័ន្ធការកំណត់ដែលពាក់ព័ន្ធនឹងការទូទាត់។ សូមចូលទៅកាន់ **Ultimate Multisite → Settings** ហើយចុចលើแท็บ **Payment**។

![Payment settings tab](/img/admin/settings-payments-top.png)

នេះគឺជាទិដ្ឋភាពពេញលេញនៃទំព័រការកំណត់នៃការទូទាត់៖

![Payment settings full page](/img/admin/settings-payments-full.png)

### ជម្រើសការទូទាត់ទូទៅ (General Payment Options)

នៅក្នុងការកំណត់ទូទៅ អ្នកអាចកំណត់រចនាសម្ព័ន្ធដូចខាងក្រោម៖

- **Currency** — ប្រភេទរូបិយប័ណ្ណដែលប្រើជាលំនាំដើមសម្រាប់ការប្រតិបត្តិការ
- **Currency Position** — ទីតាំងដែលនិមិត្តសញ្ញារូបិយប័ណ្ណបង្ហាញ (មុន/ក្រោយចំនួនទឹកប្រាក់)

![Payment gateway settings](/img/admin/settings-payments-options.png)

### ប្រព័ន្ធទូទាត់ (Payment Gateways)

Ultimate Multisite គាំទ្រប្រព័ន្ធទូទាត់ច្រើន។ អ្នកអាចបើកដំណើរការ និងកំណត់រចនាសម្ព័ន្ធនីមួយៗពីแท็บ Payment settings។

![Payment gateway configuration](/img/admin/settings-payments-gateways.png)

ប្រព័ន្ធទូទាត់ដែលមានគឺ៖

- **Stripe** — ការទូទាត់តាមកាតឥណទានតាមរយៈ Stripe
- **PayPal** — ការទូទាត់តាម PayPal
- **Manual** — សម្រាប់ការដំណើរការទូទាត់ក្រៅបណ្តាញ ឬតាមតម្រូវការផ្ទាល់ខ្លួន

ប្រព័ន្ធទូទាត់នីមួយៗមានផ្នែកកំណត់រចនាសម្ព័ន្ធរបស់វា ដែលអ្នកត្រូវបញ្ចូល API keys និងការកំណត់ផ្សេងទៀត។

![Additional gateway settings](/img/admin/settings-payments-gateways-2.png)

### របៀប Sandbox Mode (Sandbox Mode)

អ្នកអាចបើកដំណើរការ **Sandbox Mode** ដើម្បីសាកល្បងការរួមបញ្ចូលការទូទាត់របស់អ្នកមុនពេលដាក់ឱ្យដំណើរការពិត។ នៅពេលដែល sandbox mode ដំណើរការទេ នឹងមិនមានការគិតថ្លៃជាក់ស្តែងណាមួយត្រូវបានធ្វើឡើងទេ។

## ការមើលការទូទាត់ (Viewing Payments)

ចូលទៅកាន់ទំព័រ **Payments** ក្រោម Ultimate Multisite ដើម្បីមើលប្រតិបត្តិការទាំងអស់នៅទូទាំងបណ្តាញរបស់អ្នក។

![Payments list](/img/admin/payments-list.png)

អ្នកអាចច្រោះការទូទាត់តាមស្ថានភាព (បានបញ្ចប់, រង់ចាំ, របរាជ័យ, បានសងវិញ) និងស្វែងរកប្រតិបត្តិការជាក់លាក់ណាមួយបាន។

ចុចលើការបង់ប្រាក់ដើម្បីមើលព័ត៌មានលម្អិតពេញលេញ រួមទាំងรายการទំនិញ (line items) ការជាសមាជិកដែលពាក់ព័ន្ធ ព័ត៌មានអតិថិជន និងទិន្នន័យនៃ Payment gateway។

## Invoices (វិក្កយបត្រ)

Ultimate Multisite អាចបង្កើតវិក្កយបត្រដោយស្វ័យប្រវត្តិសម្រាប់ការបង់ប្រាក់។ អ្នកអាចកំណត់រចនាប័ទ្ម (customize) template វិក្កយបត្រ និងรูปแบบលេខ (numbering format) ពីការตั้งค่า Payment settings។

ជម្រើសនៃការកំណត់រចនាប័ទ្មវិក្កយបត្ររួមមាន៖

- **ឈ្មោះក្រុមហ៊ុន និងអាសយដ្ឋាន** ដែលបង្ហាញនៅលើវិក្កយបត្រ
- **รูปแบบលេខវិក្កយបត្រ (Invoice numbering)** និងលំដាប់លំដោយ
- **Logo** ដែលបង្ហាញនៅលើផ្នែកខាងលើនៃវិក្កយបត្រ
- **អត្ថបទសម្រាប់ផ្នែកខាងក្រោម (Custom footer text)** សម្រាប់លក្ខខណ្ឌ សំណូត្រ ឬព័ត៌មានផ្លូវច្បាប់

ដើម្បីកំណត់រចនាប័ទ្ម template វិក្កយបត្រ សូមចូលទៅកាន់ **Ultimate Multisite → Settings → Payment** ហើយស្វែងរកការตั้งค่าដែលទាក់ទងនឹងវិក្កយបត្រ។
