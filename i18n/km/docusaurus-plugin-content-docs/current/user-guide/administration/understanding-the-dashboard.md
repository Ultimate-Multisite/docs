# ការយល់ដឹងអំពី Ultimate Multisite Dashboard

មុននឹងយើងចូលទៅក្នុងជម្រើសការកំណត់កម្រិតខ្ពស់ និងផ្នែកប្រតិបត្តិការនៃ Ultimate Multisite សូមមកមើលផ្ទាំងគ្រប់គ្រង (dashboard) នេះជាមុនសិន។

ដូចប្រភពប្លុក WordPress ជាច្រើនដែរ Ultimate Multisite បង្កើតធាតុម៉ឺនុយនៅកម្រិតបណ្តាញដែលមានចំណងជើងថា **Ultimate Multisite** រួមជាមួយធាតុម៉ឺនុយរងមួយចំនួននៅខាងក្រោម។

## Widgets សម្រាប់ Network Dashboard

Ultimate Multisite បញ្ចូល widgets ដែលមានប្រយោជន៍មួយចំនួនទៅក្នុង network-level dashboard។ អ្នកអាចរកឃើញ dashboard នេះដោយចូលទៅកាន់ **My Sites → Network Admin → Dashboard**។ Widgets ទាំងនេះត្រូវបានរចនាឡើងដើម្បីផ្តល់នូវការចូលប្រើព័ត៌មាន និងប្រតិបត្តិការទូទៅបានយ៉ាងងាយស្រួលសម្រាប់អ្នកគ្រប់គ្រងបណ្តាញ (network administrator)។

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

### ជំហានដំបូង (First Steps)

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

Widget នេះនឹងបង្ហាញនៅពេលអ្នកដំឡើង Ultimate Multisite ជាលើកដំបូង ហើយវាជួយរំលឹកអ្នកគ្រប់គ្រងបណ្តាញឱ្យបំពេញភារកិច្ចបន្ថែមដូចជា ការកំណត់វិធីសាស្រ្តបង់ប្រាក់ (payment method) និងការបង្កើតគណនីសាកល្បង។

### សេចក្តីសង្ខេប (Summary)

![Summary widget](/img/admin/network-dashboard-summary.png)

Widget Summary នឹងរាយការណ៍អំពីចំនួនអ្នកចុះឈ្មោះចូលប្រើប្រាស់ (signups) និងចំណូលរបស់ពួកគេសម្រាប់ថ្ងៃនោះ។ MRR (Monthly Recurring Revenue) បង្ហាញពីប្រាក់ចំណូលសរុបដែលបានព្យាករណ៍សម្រាប់អតិថិជនដែលមានអាជីវកម្មបង្កើតចំណូល។

### លំហូរព្រឹត្តិការណ៍ (Activity Stream)

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

Widget នេះនឹងរាយការណ៍អំពីព្រឹត្តិការណ៍នានាដែលបានកើតឡើងនៅក្នុង Ultimate Multisite។ ព្រឹត្តិការណ៍ទាំងនោះរួមមាន ការចុះឈ្មោះចូលប្រើប្រាស់ (signups), ការលុបចោល (cancellations), ការបង្កើតគេហទំព័រថ្មី (site creations), ការលុប (deletions) និងសកម្មភាពផ្សេងៗទៀត។

សូមមើលផ្នែក Events នៅក្នុងឯកសារសម្រាប់ការពន្យល់លម្អិតបន្ថែម។

### ឥឡូវនេះ (Right Now)

![Right Now widget](/img/admin/network-dashboard-right-now.png)

Widget នេះបង្ហាញសេចក្តីសង្ខេបខ្លីៗអំពីអ្នកប្រើប្រាស់ និងគេហទំព័រនៅក្នុងបណ្តាញរបស់អ្នក។ ប៊ូតុងល្បឿននៅផ្នែកខាងលើអនុញ្ញាតឱ្យអ្នកបង្កើតគេហទំព័រថ្មី ឬអ្នកប្រើប្រាស់បានយ៉ាងងាយស្រួលតែមួយចុច។

### ព័ត៌មានថ្មីៗ និងការពិភាក្សា (News and Discussions)

![News and Discussions widget](/img/admin/network-dashboard-news.png)

Widget នេះទាញយក និងបង្ហាញព័ត៌មានថ្មីបំផុតរបស់ Ultimate Multisite។ សូមតាមដាន Widget នេះដើម្បីទទួលបានព័ត៌មានអំពីការធ្វើបច្ចុប្បន្នភាព បញ្ហាធ្ងន់ធ្ងរ និងការជួសជុលសុវត្ថិភាព។

## Ultimate Multisite Dashboard (ផ្ទាំងគ្រប់គ្រង Ultimate Multisite)

នៅពេលដែលផ្ទាំងគ្រប់គ្រងកម្រិតបណ្តាញបង្ហាញព័ត៌មានទូលំទូលាយអំពីបណ្តាញនោះ ផ្ទាំង Ultimate Multisite (ដែលមានទីតាំងនៅផ្នែកខាងលើនៃម៉ឺនុយ Ultimate Multisite) បង្ហាញព័ត៌មានអំពីសេវាកម្ម។

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

### ការលូតលាស់ចំណូលប្រចាំខែ (Monthly Recurring Revenue Growth)

តារាង MRR (Monthly Recurring Revenue) បង្ហាញការបែងចែកចំណូលប្រចាំខែក្នុងរយៈពេលប្រតិទិន ១២ ខែ។

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

ចំណូលថ្មីត្រូវបានតាមដានដោយពណ៌ខៀវ ខណៈពេលដែលការលុបចោលត្រូវបានបង្ហាញជាពណ៌ក្រហម។

### ការគាំទ្រជួរថ្ងៃ-ខែ (Date-range Support)

អាស្រ័យលើចំនួនអតិថិជន និងសកម្មភាព ស្ថិតិអាចក្លាយជាច្រើនពេក។ ការគាំទ្រជួរថ្ងៃ-ខែដែលបានដាក់បញ្ចូលក្នុងផ្ទាំងគ្រប់គ្រងអនុញ្ញាតឱ្យអ្នកគ្រប់គ្រងបណ្តាញផ្តោតតែលើព័ត៌មាន និងរយៈពេលពេលវេលាដែលត្រូវការប៉ុណ្ណោះ។

![Date range selector and stats](/img/admin/um-dashboard-date-range.png)

ជួរថ្ងៃ-ខែនេះគ្រប់គ្រងរយៈពេលនៃព័ត៌មានដែលបង្ហាញដោយ Widget ដែលនៅខាងក្រោមដោយផ្ទាល់ រួមទាំងចំណូល ការជាសមាជិកថ្មី ការចុះឈ្មោះតាមប្រទេស ការចុះឈ្មោះតាមទម្រង់ និងគេហទំព័រដែលបានចូលមើលច្រើនជាងគេ។

![Dashboard statistics widgets](/img/admin/um-dashboard-stats.png)

:::tip
ដើម្បីធ្វើឱ្យ Widget **Most Visited Sites** ដំណើរការ សូមបើកការរាប់ចំនួនអ្នកចូលមើលនៅ **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**។
:::
