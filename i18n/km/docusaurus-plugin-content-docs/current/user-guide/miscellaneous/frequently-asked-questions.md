# សំណួរដែលសួរញឹកញាប់ (Frequently Asked Questions)

ខាងក្រោមនេះគឺជាសំណួរដែលគេសួរញឹកញាប់ពេលមានការស្នើសុំជំនួយ។

  * **Ultimate Multisite ជាអ្វី?**

Ultimate Multisite គឺជា plugin សម្រាប់ WordPress ដែលបង្កើតឡើងសម្រាប់ដំឡើង WordPress multisite ដើម្បីបំប្លែងការដំឡើង WordPress របស់អ្នកទៅជាបណ្តាញគេហទំព័រពិសេសៗ – ដូច [WordPress.com](https://WordPress.com) ដែរ – ដែលអនុញ្ញាតឱ្យអតិថិជនបង្កើតគេហទំព័រតាមរយៈតម្លៃប្រចាំខែ ប្រចាំត្រីមាស ឬប្រចាំឆ្នាំ (អ្នកក៏អាចបង្កើតផែនការឥតគិតថ្លៃបានដែរ)។

  * **Ultimate Multisite នឹងដំណើរការដោយគ្មាន Multisite បានទេឬ?**

ទេ។ Ultimate Multisite ទាមទារឱ្យមាន WordPress multisite, subdirectory ឬ subdomain។

  * **តម្រូវការសម្រាប់ Ultimate Multisite មានអ្វីខ្លះ?**

Ultimate Multisite តម្រូវឱ្យមាន៖

**WordPress Version** : v5.3+ (ណែនាំ៖ កំណែចុងក្រោយដែលស្ថិតស្ថេរ)
**PHP Version** : 7.4.x (គាំទ្រដល់ > 8.0 នឹងមកដល់ឆាប់ៗនេះ)
**MySQL Version** : v5+ (ណែនាំ៖ កំណែ 5.6 ប្រសិនបើអ្នកមិនមាន 8.0 សម្រាប់ផ្តល់ឱ្យនៅលើ hosting provider របស់អ្នកទេ)

_Multisite (subdomain ឬ subdirectory) ក៏ត្រូវតែត្រូវបានបើកដំណើរការផងដែរ។_ [របៀបដំឡើង WordPress Multisite](/user-guide/getting-started/how-to-install-wordpress-multisite)។

  * **Ultimate Multisite អាចដំណើរការជាមួយ Multisite ដែលប្រើ Subdirectories បានទេ?**

បាន។ Ultimate Multisite គាំទ្រទាំងការដំឡើង subdirectory និងការដំឡើង subdirectory។

  * **Ultimate Multisite អាចដំណើរការជាមួយ Multisite ដែលប្រើ Subdomains បានទេ?**

បាន។ Ultimate Multisite គាំទ្រទាំងការដំឡើង subdirectory និងការដំឡើង subdomain។

  * **ខ្ញុំអាចប្រើ Ultimate Multisite ដើម្បីកំណត់ថាអ្នកប្រើប្រាស់អាចប្រើ Plugin និង Themes ណាខ្លះបានទេ?**

បាន។ ជាមួយ Ultimate Multisite អ្នកអាចជ្រើសរើសយក plugin និង themes ដែលមានសម្រាប់ផែនការនីមួយៗនៅលើបណ្តាញរបស់អ្នក។

  * **ខ្ញុំអាចប្រើ Ultimate Multisite ដើម្បីកំណត់ចំនួន posts ដែលអ្នកប្រើប្រាស់អាចបង្កើតនៅលើគេហទំព័ររបស់គាត់/នាងបានទេ?**

បាទ។ ជាមួយ Ultimate Multisite អ្នកអាចបង្កើតកំណត់ចំនួនទំព័រ (post quotas) សម្រាប់ **ប្រភេទទំព័រទាំងអស់ដែលមាន** នៅលើបណ្តាញរបស់អ្នក។ អ្នកក៏អាចកំណត់ចំនួនអ្នកប្រើប្រាស់ដែលអ្នកគ្រប់គ្រងអាចអញ្ជើញបាន, **បរិមាណទំហំዲስក**, និងច្រើនទៀតផងដែរ។

  * **Ultimate Multisite គាំទ្រ Site Templates ដែរឬទេ?**

បាទ។ ជាមួយ Ultimate Multisite អ្នកអាចបង្កើត Site Templates បានច្រើនតាមដែលអ្នកចង់បាន។ Site Templates មិនមែនជា themes ទេ វាគឺជាគេហទំព័រធម្មតាលើបណ្តាញរបស់អ្នក។ អ្នកអាចបើកដំណើរការ plugins, themes, និងធ្វើការកំណត់រចនាសម្ព័ន្ធ (customize) Site Template នោះតាមរបៀបដែលអ្នកចង់បាន។ ប្រសិនបើអ្នកប្រើប្រាស់ជ្រើសរើស template នោះ គេហទំព័ររបស់ពួកគេនឹងត្រូវបានបង្កើតឡើងជាការចម្លងនៃ template នោះ ដោយមានការកំណត់រចនាសម្ព័ន្ធទាំងអស់រួចហើយ។

អ្នកក៏អាចកំណត់ template ជាក់លាក់មួយសម្រាប់ plan ណាមួយបានដែរ។ ដោយធ្វើដូច្នេះ អ្នកប្រើប្រាស់គ្រប់រូបដែលចុះឈ្មោះលើ plan នោះនឹងទទួលបានគេហទំព័ររបស់ពួកគេជាការចម្លងនៃ Site Template នោះ។

  * **Ultimate Multisite ភ្ជាប់ជាមួយ WooCommerce ដែរឬទេ?**

បាទ!

ប្រសិនបើអ្នកចង់ឱ្យអ្នកប្រើប្រាស់បណ្តាញរបស់អ្នកមាន WooCommerce សម្រាប់ពួកគេ ដើម្បីអនុញ្ញាតឱ្យពួកគេបង្កើតហាង អ្នកអាចធ្វើបានដោយប្រើ Ultimate Multisite។ អ្នកថែមទាំងអាចកំណត់ចំនួនផលិតផលដែលអតិថិជនរបស់អ្នកនឹងអាចបង្កើតនៅលើ plan នីមួយៗទៀតផង។

ទាក់ទងនឹងការបង់ប្រាក់ជា subscription សម្រាប់ Ultimate Multisite អ្នកអាចប្រើ [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) ដើម្បីអនុញ្ញាតឱ្យអតិថិជន Ultimate Multisite របស់អ្នកបង់ប្រាក់ដោយប្រើ WooCommerce gateways នៅលើ site មេរបស់អ្នក។ តាមរបៀបនោះ អ្នកអាចទាញយកនូវបណ្ណាល័យដ៏ធំនៃ payment gateways ដែលបានបង្កើតឡើងសម្រាប់ WooCommerce ហើយប្រើវាដើម្បីគិតប្រាក់ (bill) អតិថិជន Ultimate Multisite របស់អ្នក។

  * **មានវិធីផ្លាស់ប្តូរ subscription ពី WPMUDEV Pro Sites ដែរឬទេ?**

បាទ/ចាស Ultimate Multisite ឥឡូវនេះផ្តល់ឱ្យអ្នកនូវ [ឧបករណ៍ដ៏មានប្រយោជន៍ដើម្បីជួយអ្នកក្នុងការផ្ទេរ (migrate)](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) បណ្តាញ Pro Sites របស់អ្នក។

* **ហេតុអ្វីបានជាแท็บ “Allowed Plugins” នៅលើหน้า Edit Plan របស់ខ្ញុំទទេ?**

Ultimate Multisite អនុញ្ញាតឱ្យអ្នកអនុញ្ញាត ឬបដិសេធការចូលប្រើប្រាស់ plugin ដែលមិនត្រូវបានបើកដំណើរការនៅលើ network របស់អ្នកប៉ុណ្ណោះ។ នៅពេលដែល plugin ត្រូវបានបើកដំណើរការជា network-activated វាមានលក្ខណៈធម្មជាតិថាវានឹងត្រូវបានផ្ទុកដោយស្វ័យប្រវត្តិនៅលើគ្រប់ site ទាំងអស់នៅក្នុង network របស់អ្នក។

ប្រសិនបើអ្នកមិនឃើញ plugin ណាមួយបង្ហាញនៅលើแท็บ “Allowed Plugins” សូមពិនិត្យមើលថាតើ plugin ដែលអ្នកចង់ប្រើបានបើកដំណើរការជា network ឬអត់។

* **ខ្ញុំអាចប្រើការរចនា Pricing Table របស់ខ្ញុំផ្ទាល់បានទេ?**

បាន។ ចាប់តាំងពីកំណែ 1.4.0 មក Ultimate Multisite គាំទ្រការជំនួស template (template overrides)។

* **Ultimate Multisite ដំណើរការជាមួយ WP Engine បានទេ?**

បាន។ Ultimate Multisite ត្រូវបានគាំទ្រដោយ WP Engine ជានិច្ច ប៉ុន្តែការធ្វើឱ្យ domain mapping ដំណើរការទាមទារជំហានបន្ថែមដោយដៃពីអ្នកគ្រប់គ្រង (admin) (ការបន្ថែម domain ទៅក្នុង panel របស់ WP Engine)។ **វាមិនមែនជាករណីបែបនេះទៀតទេ ចាប់តាំងពីកំណែ 1.6.0** Ultimate Multisite ភ្ជាប់ដោយផ្ទាល់ជាមួយ WP Engine API ហើយdomain នឹងត្រូវបានបន្ថែមទៅក្នុង panel ដោយស្វ័យប្រវត្តិបន្ទាប់ពីការ mapping។

* **អ្នកណែនាំ hosting provider ណា?**

Ultimate Multisite អាចដំណើរការជាមួយ hosting provider ណាមួយបាន។ មានតែភាពខុសគ្នានោះគឺថាក្រុមហ៊ុន hosting ខ្លះដូចជា Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane និង [Runcloud.io](http://Runcloud.io) មានការរួមបញ្ចូលគ្នា (integration) ច្រើនជាងមុនសម្រាប់ការ mapping domain ដូច្នេះពួកវាដំណើរការបានភ្លាមៗ។ ចំពោះ hosting ផ្សេងទៀត អ្នកប្រហែលជាត្រូវកំណត់ការ mapping domain ដោយដៃ។
