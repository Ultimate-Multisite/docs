# ការកែសម្រួល Admin Panel របស់អ្នកជាមួយ PRO Theme {#customizing-your-admin-panel-with-pro-theme}

ពេលខ្លះនៅពេលធ្វើការងារសម្រាប់អតិថិជន យើងត្រូវការផ្តល់បទពិសោធន៍ផ្ទាល់ខ្លួន (custom experience) នៅក្នុងផ្ទាំងគ្រប់គ្រង (admin dashboard) របស់ WordPress។ ប្រសិនបើអ្នកកំពុងស្វែងរករូបរាង និងអារម្មណ៍កាន់តែប្រកបវិជ្ជាជីវៈ PRO Theme អាចជួយអ្នកបាន។

PRO Theme គឺជា admin theme ដែលត្រូវបានបង្កើតឡើងដើម្បីជួយអ្នកក្នុងការកែសម្រួល Dashboard របស់អតិថិជនរបស់អ្នក។ ហើយវាក៏ជាផ្នែកមួយនៃ core plugin ពិសេស (premium plugins core) របស់ Ultimate Multisite ផងដែរ។

បន្ទាប់ពីដំឡើងរួច សូមចូលទៅកាន់ទំព័រ _Plugins_ របស់អ្នក ហើយចុចដើម្បីធ្វើឱ្យវាដំណើរការ (activate) នៅលើ Network របស់អ្នក។

_ធ្វើឱ្យវាដំណើរការពេញ Network នៅលើទំព័រ Plugins របស់អ្នក_

_**ចំណាំ:**_ _ប្រសិនបើអ្នកគ្រាន់តែដំឡើងវាទេ វាបង្ហាញនៅលើអតិថិជនរបស់អ្នកជាជម្រើសដែលអាចសម្គាល់ដើម្បីបន្ថែម theme នេះទៅកាន់អ្នកប្រើប្រាស់នោះតែប៉ុណ្ណោះ។_

ឥឡូវនេះ អ្នកមាន admin interface ដែលត្រូវបានរចនាឡើងវិញទាំងស្រុងហើយ! អ្នកក៏មានเมนูថ្មីនៅបណ្តោយបង្អួចខាងឆ្វេងដែលមានឈ្មោះថា PRO Theme។

នៅក្នុងផ្ទាំងនីមួយៗ អ្នកអាចបើក ឬបិទធាតុនៃការរចនា (styling items) បអនុញ្ញាត ឬបិទមុខងារ (functionalities) និងផ្តល់នូវការកែសម្រួលដែលអតិថិជនរបស់អ្នកប្រហែលជាកំពុងស្វែងរកនៅលើ network គេ។

នេះគឺជាឧទាហរណ៍នៃអ្វីដែលអ្នកអាចទទួលបានបន្ទាប់ពីការកែសម្រួលខ្លះ៖
