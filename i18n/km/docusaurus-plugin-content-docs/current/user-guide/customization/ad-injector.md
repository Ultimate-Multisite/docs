# ការដាក់ពាណិជ្ជកម្មនៅលើបណ្តាញរបស់អ្នកជាមួយ Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector គឺជា **add-on ឥតគិតថ្លៃ** ដែលផ្តល់ឱ្យអ្នកនូវឧបករណ៍ដើម្បីដាក់ពាណិជ្ជកម្មនៅលើផែនការរបស់អ្នកដោយផ្ទាល់ពីទំព័ររៀងៗខ្លួន។

បន្ទាប់ពីដំឡើង Ultimate Multisite: Ad Injector រួច សូមចូលទៅកាន់ទំព័រ _Plugins_ របស់អ្នកដើម្បីបើកដំណើរការវា។

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_ចុចប៊ូតុង Network Activate_

នៅពេលដែល add-on ត្រូវបានបើកដំណើរការហើយ ប្លុក _Advanced Options_ នៅលើអេក្រង់កែសម្រួលផែនការនឹងមានជម្រើសថ្មីមួយឈ្មោះថា _Ad Settings_។

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_ឥឡូវនេះអ្នកអាចកែសម្រួលការកំណត់ពាណិជ្ជកម្មសម្រាប់ផែនការនីមួយៗរបស់អ្នកបានហើយ_

នៅលើផ្ទាំងនោះ អ្នកនឹងអាចបន្ថែមលេខកូដពាណិជ្ជកម្មសម្រាប់ posts និង pages នៅផ្នែកខាងមុខ (front-end) មុន និងក្រោយខ្លឹមសារ។

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_បន្ទាប់ពីបញ្ចូលលេខកូដពាណិជ្ជកម្មរបស់អ្នករួច វានឹងបង្ហាញនៅលើ posts និង pages នៃផែនការជាក់លាក់នោះ។_

អ្នកក៏អាចដាក់ពាណិជ្ជកម្មនៅក្នុង header នៃ Admin Pages សម្រាប់អ្នកប្រើប្រាស់នៃផែនការនោះផងដែរ។

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_ចូលទៅកាន់ជម្រើស Admin Panel Ad Code ប្រសិនបើអ្នកចង់បង្ហាញពាណិជ្ជកម្មនៅលើ back-end របស់អ្នកប្រើប្រាស់_

### លើសពីការផ្សាយពាណិជ្ជកម្ម

អ្នកក៏អាចប្រើ Ultimate Multisite: Ad Injector ដើម្បីផ្តល់ការគាំទ្រដល់ផែនការកម្រិតខ្ពស់ផងដែរ។

គ្រាន់តែចូលទៅកាន់ជម្រើស _Admin Panel Ads_ ហើយបន្ថែម HTML code សម្រាប់ support tool របស់អ្នក។

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

_បន្ថែមប៊ូតុងគាំពលើផែនការជាក់លាក់_

តាមរបៀបនេះ អ្នកអាចទៅដល់អ្នកបានភ្លាមៗពី Admin Panel របស់ពួកគេ!
