# бһҖбһ¶бһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢ Ultimate Multisite бһҮбһ¶бһҳбһҪбһҷ Zapier

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһўбһҸбҹ’бһҗбһ”бһ‘бһҳбһҪбһҷ бһҷбһҫбһ„бһ”бһ¶бһ“бһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһўбҹҶбһ–бһё [Webhooks](webhooks.md) бһ“бһ·бһ„бһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһңбһ¶бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ—бһ¶бһӮбһёбһ‘бһёбһ”бһёбҹ”

бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ webhooks бһӮбһәбһҳбһ¶бһ“бһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһ”бһ“бҹ’бһҸбһ·бһ…бһ–бҹ’бһҡбҹ„бҹҮбһңбһ¶бһ‘бһ¶бһҳбһ‘бһ¶бһҡбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠ бһ“бһ·бһ„бһ…бһ¶бһ”бҹӢбһҷбһҖ payload (бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ)бҹ” бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ **Zapier** бһӮбһәбһҮбһ¶бһңбһ·бһ’бһёбһҳбһҪбһҷбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҮбҹҖбһҹбһңбһ¶бһ„бһ”бһүбҹ’бһ бһ¶бһ“бҹҒбҹҮбһ”бһ¶бһ“бҹ”

Zapier бһҳбһ¶бһ“бһҖбһ¶бһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбһ¶бһ„ 5000+ бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҖбһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҡбһңбһ¶бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҖбһ¶бһ“бҹӢбһҸбҹӮбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбҹ”

бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸ **Triggers** (бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһ„бҹ’бһҖ) бһҠбҹӮбһӣбһ“бһ№бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһ–бҹҒбһӣбһҳбһ¶бһ“бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҖбһҫбһҸбһЎбһҫбһ„бһ“бҹ…бһӣбһҫбһ”бһҺбҹ’бһҸбһ¶бһүбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ (бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһҺбһ“бһё бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһұбҹ’бһҷбһҳбһ¶бһ“бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ account_create) бһ¬бһ”бһ„бҹ’бһҖбһҫбһҸ **Actions** (бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–) бһ“бҹ…бһӣбһҫбһ”бһҺбҹ’бһҸбһ¶бһүбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбһҫбһҳбҹ’бһ”бһёбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ‘бҹ…бһ“бһ№бһ„бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ… (бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһҳбһ¶бһҮбһ·бһҖбһҗбҹ’бһҳбһёбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҺбҹ’бһҸбһ¶бһү Ultimate Multisite бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ)бҹ”

бһ“бҹҒбҹҮбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ‘бҹ…бһ”бһ¶бһ“бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮ **Ultimate Multisite Zapier's triggers** бһ“бһ·бһ„ actions бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹ„бһҷ [REST API](https://developer.ultimatemultisite.com/api/docs/)бҹ”

## бһҡбһ”бҹҖбһ”бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ

бһҠбҹҶбһ”бһјбһ„ бһҹбһјбһҳбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ Ultimate Multisite бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһүбҹ’бһҮбһёбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢ Zapierбҹ” бһ¬бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бһ»бһ…бһӣбһҫ [бһҸбҹҶбһҺбһ“бҹҒбҹҮ](https://zapier.com/apps/wp-ultimo/integrations)бҹ”

бһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢ dashboard бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһ…бһ»бһ…бһ”бҹҠбһјбһҸбһ»бһ„ **+** **Create Zap** бһ“бҹ…бһҡбһ”бһ¶бһҡбһ…бҹҶбһ бҹҖбһ„бһҒбһ¶бһ„бһҶбҹ’бһңбҹҒбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһҡбҹҖбһ”бһ…бҹҶ Zap бһҗбҹ’бһҳбһёбҹ”

![Zapier dashboard with Create Zap button](/img/admin/webhooks-list.png)

бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһјбһ“бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ‘бҹҶбһ–бҹҗбһҡбһ”бһ„бҹ’бһҖбһҫбһҸ Zapбҹ”

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһўбһ”бҹӢбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ бһҹбһјбһҳбһңбһ¶бһҷ "wp ultimo"бҹ” бһ…бһ»бһ…бһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҮбһҳбҹ’бһҡбһҫбһҹ **Beta**бҹ”

![Searching for WP Ultimo in Zapier app list](/img/admin/webhooks-list.png)

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһ бһҫбһҷ бһҹбһјбһҳбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҠбҹӮбһӣбһҳбһ¶бһ“бҹ– **New Ultimate Multisite Event**бҹ”

![Selecting New Ultimate Multisite Event trigger](/img/admin/webhooks-list.png)

бһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһ•бҹ’бһҸбһӣбҹӢбһұбҹ’бһҷ Zapier бһ“бһјбһңбһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ‘бҹ…бһҖбһ¶бһ“бҹӢ **бһ”бһҺбҹ’бһҸбһ¶бһүбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ**бҹ” бһҖбһ¶бһҡбһ…бһ»бһ…бһӣбһҫ **Sign in** (бһ…бһјбһӣ) бһ“бһ№бһ„бһ”бһҫбһҖбһ”бһ„бҹ’бһўбһҪбһ…бһҗбҹ’бһҳбһёбһҠбҹӮбһӣбһ‘бһ¶бһҳбһ‘бһ¶бһҡ **API credentials** (бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ API)бҹ”

![Prompt бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бһјбһӣ Zapier бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ API credentials](/img/admin/webhooks-list.png)

бһ‘бҹ…бһҖбһ¶бһ“бҹӢ network admin panel (бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһҺбҹ’бһҸбһ¶бһү) бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢ **Ultimate Multisite > Settings** > **API & Webhooks** бһҡбһҪбһ…бһҡбһҖбһҳбһҫбһӣбһ•бҹ’бһ“бҹӮбһҖ API Settingsбҹ”

бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҮбһҳбҹ’бһҡбһҫбһҹ **Enable API** (бһ”бһҫбһҖ API) бһ–бҹ’бһҡбҹ„бҹҮбһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҸбһҳбҹ’бһҡбһјбһңбһұбҹ’бһҷбһҳбһ¶бһ“бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ“бҹҒбҹҮбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бҹ”

![бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ API бһ“бһ·бһ„ Webhooks бһҡбһҪбһҳбһҮбһ¶бһҳбһҪбһҷбһҮбһҳбҹ’бһҡбһҫбһҹ Enable API](/img/admin/settings-api-webhooks.png)

бһ”бҹ’бһҡбһҫбһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„ **Copy to Clipboard** (бһ…бһҳбҹ’бһӣбһ„бһ‘бҹ… Clipboard) бһ“бҹ…бһӣбһҫбһңбһ¶бһӣ API Key бһ“бһ·бһ„ API Secret бһ бһҫбһҷбһ”бһ·бһ‘бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҸбһҳбҹ’бһӣбҹғбһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮбһ“бҹ…бһӣбһҫбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ integrationбҹ”

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһ¶бһӣ URL бһҹбһјбһҳбһҠбһ¶бһҖбҹӢ full URL бһҡбһ”бһҹбҹӢ network бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„ protocol (HTTP бһ¬ HTTPS)бҹ”

![бһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ integration бһҡбһ”бһҹбҹӢ Zapier бһҠбҹӮбһӣбһҳбһ¶бһ“бһңбһ¶бһӣ API Key, Secret бһ“бһ·бһ„ URL](/img/admin/webhooks-list.png)

бһ…бһ»бһ…бһ”бҹҠбһјбһҸбһ»бһ„ **Yes, Continue** (бһ”бһ¶бһ‘ бһҹбһјбһҳбһ”бһ“бҹ’бһҸ) бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бҹ…бһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһңбһёбҹ—бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһӣбҹ’бһў бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбҹ’бһңбһ¶бһӮбһҳбһ“бҹҚбһҠбҹ„бһҷбһӮбһҺбһ“бһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҗбҹ’бһҳбһё! бһ…бһ»бһ…бһӣбһҫ **Continue** (бһ”бһ“бҹ’бһҸ) бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸ trigger бһҗбҹ’бһҳбһёбҹ”

## бһҡбһ”бҹҖбһ”бһ”бһ„бҹ’бһҖбһҫбһҸ Trigger бһҗбҹ’бһҳбһё

бһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһӮбһҺбһ“бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ бһҫбһҷ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһҫбһӣбһғбһҫбһү events бһҠбҹӮбһӣбһҳбһ¶бһ“бҹ” бһҹбһјбһҳбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ event **payment_received** (бһҖбһ¶бһҡбһ‘бһ‘бһҪбһӣбһ”бҹ’бһҡбһ¶бһҖбҹӢ) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ tutorial бһ“бҹҒбҹҮбҹ”

![бһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ event payment_received бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Zapier trigger](/img/admin/webhooks-list.png)

бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ event бһ бһҫбһҷбһўбҹ’бһ“бһҖбһ…бһ»бһ… **continue** (бһ”бһ“бҹ’бһҸ) бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ **test step** (бһҮбҹҶбһ бһ¶бһ“бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„) бһҳбһҪбһҷбһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбҹ”

![Zapier test step бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ trigger](/img/admin/webhooks-list.png)

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһ“бҹҒбҹҮ Zapier бһ“бһ№бһ„бһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһҗбһ¶бһҸбһҫ Zap бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбһ¶бһ… **fetch payload бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһ‘бҹ…бһҖбһ¶бһ“бҹӢ event бһ“бҹ„бҹҮбһ”бһ¶бһ“**бҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ events бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ“бҹ…бһ–бҹҒбһӣбһўбһ“бһ¶бһӮбһҸ бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһјбһ“бһҳбһҖбҹ”

![Zapier trigger test бһ”бһүбҹ’бһ…бһ”бҹӢбһҠбҹ„бһҷбһҮбҹ„бһӮбһҮбҹҗбһҷбһҮбһ¶бһҳбһҪбһҷ payload](/img/admin/webhooks-list.png)

бһҖбҹ’бһ“бһ»бһ„бһңбһӮбҹ’бһӮбһ”бһ„бҹ’бһҡбҹҖбһ“бһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһ”бһ¶бһ“ **бһ”бһүбҹ’бһ…бһ”бҹӢбһҠбҹ„бһҷбһҮбҹ„бһӮбһҮбҹҗбһҷ** бһ бһҫбһҷбһ”бһ¶бһ“бһҸбҹ’бһҡбһЎбһ”бҹӢбһҳбһҖбһңбһ·бһүбһ“бһјбһңбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғ payloadбҹ” бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮбһ“бһ№бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҺбһ¶бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҷбһҫбһ„бһ–бҹҒбһӣбһ”бһ„бҹ’бһҖбһҫбһҸ actions (бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–)бҹ” Trigger бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһҪбһ…бһ бһҫбһҷ бһ“бһ·бһ„бһҸбҹ’бһҡбҹҖбһҳбһҒбҹ’бһӣбһҪбһ“бһҠбһҫбһҳбҹ’бһ”бһёбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бҹҖбһҸбҹ”

## бһҡбһ”бҹҖбһ”бһ”бһ„бҹ’бһҖбһҫбһҸ Actions

Actions бһ”бҹ’бһҡбһҫбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ–бһё triggers бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ’бһ¶бһҸбһ»бһҗбҹ’бһҳбһёбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ network бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡ **бһ”бһ„бҹ’бһҖбһҫбһҸ action step** бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ Ultimate Multisite **Beta** бһ“бһ·бһ„бһҮбһҳбҹ’бһҡбһҫбһҹ **Create Items on Ultimate Multisite** (бһ”бһ„бҹ’бһҖбһҫбһҸ Item бһ“бҹ…бһӣбһҫ Ultimate Multisite)бҹ”

![Creating an action with Create Items on Ultimate Multisite](/img/admin/webhooks-list.png)

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺ (authentication) бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбһјбһ…бһҠбҹӮбһӣбһҷбһҫбһ„бһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһҖбҹ’бһ“бһ»бһ„ **How to start** (бһҡбһ”бҹҖбһ”бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ) бһ¬бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ authentication бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһҪбһ…бҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһӮбҹ’бһӮбһ”бһ„бҹ’бһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ authentication бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһҪбһ…бһ–бһёбһҳбһ»бһ“бҹ”

![Selecting authentication for the Zapier action](/img/admin/webhooks-list.png)

### бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ Action (Setting up the Action)

бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶ **бһҮбҹҶбһ бһ¶бһ“бһҹбҹҶбһҒбһ¶бһ“бҹӢбһ“бҹғ action** бһ бһҫбһҷбһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбһўбҹ’бһңбһёбҹ—бһӮбһәбһҳбһ¶бһ“бһ—бһ¶бһ–бһҒбһ»бһҹбһ”бҹ’бһӣбҹӮбһҖбһ”бһ“бҹ’бһҸбһ·бһ…бҹ” бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҠбҹҶбһ”бһјбһ„бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһӮбһә **Item**бҹ” Item бһӮбһәбһҮбһ¶ **information model** (бһӮбҹҶбһҡбһјбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“) бһ“бҹғ network бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбһјбһ…бһҮбһ¶ **Customers, Payments, Sites, Emails** бһ“бһ·бһ„бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бҹҖбһҸбҹ”

![Choosing Item type for the Zapier action](/img/admin/webhooks-list.png)

бһ“бҹ…бһ–бҹҒбһӣбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ item бһ‘бһҳбҹ’бһҡбһ„бҹӢбһ“бһ№бһ„ **бһҡбҹҖбһ”бһ…бҹҶбһЎбһҫбһ„бһңбһ·бһүбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһңбһ¶бһӣбһҠбҹӮбһӣбһҸбһҳбҹ’бһҡбһјбһң бһ“бһ·бһ„бһҮбһҳбҹ’бһҡбһҫбһҹ (required and optional fields)** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ item бһҠбҹӮбһӣбһ”бһ¶бһ“бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһ“бҹ…бһ–бҹҒбһӣбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ item **Customer** (бһўбһҸбһ·бһҗбһ·бһҮбһ“) бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ“бҹғбһңбһ¶бһӣбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ‘бһҳбҹ’бһҡбһ„бҹӢбһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһўбҹ’бһңбһёбһӮбҹ’бһҡбһ”бҹӢбһҷбҹүбһ¶бһ„бһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹҶбһ–бҹҒбһүбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸ Customer бһҗбҹ’бһҳбһёбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ networkбҹ”

![Customer item fields in Zapier action setup](/img/admin/webhooks-list.png)

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бҹҶбһ–бҹҒбһүбһңбһ¶бһӣбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҗбһ¶ **required** (бһҸбһҳбҹ’бһҡбһјбһң) бһ бһҫбһҷбһ…бһ»бһ… continue (бһ”бһ“бҹ’бһҸ) бһңбһ¶бһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһҳбһҪбһҷбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһңбһ¶бһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹҶбһ–бҹҒбһү бһ“бһ·бһ„бһңбһ¶бһӣбһҠбҹӮбһӣбһ“бҹ…бһҹбһӣбҹӢбһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһ”бһ¶бһ“бһ”бҹҶбһ–бҹҒбһүбҹ”

![бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Zapier бһ”бһ„бҹ’бһ бһ¶бһүбһңбһ¶бһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹҶбһ–бҹҒбһү бһ“бһ·бһ„бһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһ”бһ¶бһ“бһ”бҹҶбһ–бҹҒбһү](/img/admin/webhooks-list.png)

бһ—бҹ’бһӣбһ¶бһҳбҹ—бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бһүбҹ’бһ…бһ”бҹӢбһҠбҹ„бһҷбһҮбҹ„бһӮбһҮбҹҗбһҷ бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– (action) бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бҹ” бһңбһ¶бһ–бһ·бһҸбһҮбһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢбһҺбһ¶бһҹбҹӢбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһӣбһҫбһ”бһҺбҹ’бһҸбһ¶бһүбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҗбһ¶бһҸбһҫбһ’бһ¶бһҸбһ»бһ“бҹ„бҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ¬бһҷбҹүбһ¶бһ„бһҺбһ¶бҹ”
