# бһҸбһҫбһҒбҹ’бһүбһ»бҹҶбһўбһ¶бһ…бһ…бһјбһӣбһҳбһҫбһӣбһ‘бҹҶбһ–бҹҗбһҡбһӮбһҺбһ“бһёбһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶбһ”бһ¶бһ“бһӣбһҫбһҖбһҠбҹҶбһ”бһјбһ„бһҠбҹ„бһҷбһҡбһ”бҹҖбһ”бһҺбһ¶? {#how-can-i-access-my-account-page-for-the-first-time}

бһҹбһјбһҳбһҹбҹ’бһңбһ¶бһӮбһҳбһ“бҹҚбһҳбһҖбһҖбһ¶бһ“бҹӢ Ultimate Multisite!

бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ‘бҹҶбһ–бҹҗбһҡбһӮбһҺбһ“бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбһ¶бһҳбһҡбһҷбҹҲбһҸбҹҶбһҺбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһўбҹҠбһёбһҳбҹӮбһӣбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһ¶бһ“бһ‘бһ·бһү plugin бһҡбһ”бһҹбҹӢбһҷбһҫбһ„бҹ–

<!-- бһҡбһјбһ”бһ—бһ¶бһ–бһҳбһ·бһ“бһҳбһ¶бһ“бһ‘бҹҒ: бһўбҹҠбһёбһҳбҹӮбһӣбһ‘бҹҶбһ–бҹҗбһҡбһӮбһҺбһ“бһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһҸбҹҶбһҺбһ…бһјбһӣ -->

бһҹбһјбһҳбһ”бҹ’бһҡбһҫбһўбҹҠбһёбһҳбҹӮбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ•бҹ’бһҸбһӣбҹӢбһ–бҹҒбһӣбһ‘бһ·бһү бһ“бһ·бһ„бһ–бһ¶бһҖбҹ’бһҷбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһўбҹҠбһёбһҳбҹӮбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

<!-- бһҡбһјбһ”бһ—бһ¶бһ–бһҳбһ·бһ“бһҳбһ¶бһ“бһ‘бҹҒ: бһ‘бҹҶбһ–бҹҗбһҡбһ…бһјбһӣбһҠбҹ„бһҷбһҳбһ¶бһ“бһ”бҹ’бһҡбһўбһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹҠбһёбһҳбҹӮбһӣ бһ“бһ·бһ„бһ–бһ¶бһҖбҹ’бһҷбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ -->

бһҠбҹ„бһҷбһҹбһ¶бһҡбһ“бҹҒбҹҮбһҮбһ¶бһӣбһҫбһҖбһҠбҹҶбһ”бһјбһ„бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһӮбһҺбһ“бһёбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһұбҹ’бһҷбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ–бһ¶бһҖбҹ’бһҷбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҹбһјбһҳбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһңбһ¶ бһ бһҫбһҷбһ…бһ»бһ…бһ”бҹҠбһјбһҸбһ»бһ„ _Save_ (бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ)бҹ”

<!-- бһҡбһјбһ”бһ—бһ¶бһ–бһҳбһ·бһ“бһҳбһ¶бһ“бһ‘бҹҒ: бһ‘бһҳбҹ’бһҡбһ„бҹӢбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ–бһ¶бһҖбҹ’бһҷбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹҠбһјбһҸбһ»бһ„ Save -->

бһ“бҹ…бһӣбһҫбһ‘бҹҶбһ–бҹҗбһҡбһӮбһҺбһ“бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӮбһҺбһ“бһёбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбһјбһ…бһҮбһ¶ бһҖбһ¶бһҡбһ…бҹҒбһүбһңбһ·бһҖбҹ’бһҖбһҷбһ”бһҸбҹ’бһҡ (billings) бһҖбһ¶бһҡбһ”бһ“бҹ’бһҸбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ (renewals) бһҖбһ¶бһҡбһӣбһ»бһ”бһ…бҹ„бһӣ (cancellations) бһ“бһ·бһ„бһўбҹ’бһңбһёбҹ—бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ‘бҹҖбһҸбҹ”
