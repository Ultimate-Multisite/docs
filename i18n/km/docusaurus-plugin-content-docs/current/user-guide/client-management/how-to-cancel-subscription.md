# តើខ្ញុំអាចបោះបង់ការជាវបានដោយរបៀបណា? {#how-can-i-cancel-my-subscription}

ប្រសិនបើអ្នកមានអាជ្ញាប័ណ្ណ Ultimate Multisite អ្នកអាចលុបចោលការបន្ត (renewal) បានគ្រប់ពេលដែលអ្នកចង់។ សូមធ្វើតាមជំហានខាងក្រោម៖

<!-- Screenshot unavailable: Account page email with access link -->

ប្រើអ៊ីមែលដែលអ្នកបានផ្តល់ក្នុងពេលទិញ និងពាក្យសម្ងាត់របស់អ្នក។

<!-- Screenshot unavailable: Login page with email and password fields -->

នេះគឺជាអេក្រង់ចាប់ផ្តើមនៃหน้ารបស់អ្នក (Account Page)៖

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

ដើម្បីលុបចោលការជាវរបស់អ្នក សូមទៅកាន់ម៉ឺនុយនៅខាងឆ្វេង ជ្រើសរើស _Renewing & Billing._។

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

ចុចលើរូបព្រួញនៅខាងស្តាំដើម្បីបើកបង្អួចមួយ។ បន្ទាប់មក អ្នកគួរតែជ្រើសរើសជម្រើសដែលនឹងលុបចោលការបន្តដោយស្វ័យប្រវត្តិ (**auto-renew**)。

ប្រព័ន្ធនឹងបង្ហាញសារបញ្ជាក់ដល់អ្នក។

ប្រសិនបើអ្នកប្រាកដថាចង់លុបចោលការជាវរបស់អ្នក សូមចុចលើប៊ូតុង _**Cancel Renewals**_។ បន្ទាប់ពីធ្វើសកម្មភាពនេះ អ្នកនឹងត្រូវបានសួរឆ្លើយសំណួរខ្លីៗមួយ។

រួចរាល់ហើយ! ការជាវរបស់អ្នកនឹងមិនត្រូវបានបន្តដោយស្វ័យប្រវត្តិទៀតទេ។

**អ្នកនឹងមានកូនសោ (key) ដែលត្រឹមត្រូវ រហូតដល់ការជាវរបស់អ្នកផុតកំណត់**។ ប្រសិនបើអ្នកចង់បើកដំណើរការការជាវឡើងវិញ អ្នកនឹងត្រូវធ្វើវាដោយដៃផ្ទាល់។
