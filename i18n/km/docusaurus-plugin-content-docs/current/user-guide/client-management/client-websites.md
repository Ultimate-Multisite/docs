# គេហទំព័ររបស់អតិថិជនអ្នក (v2) {#your-clients-websites-v2}

បន្ទាប់ពីចុះឈ្មោះចូលក្នុងបណ្តាញរបស់អ្នក អតិថិជនរបស់អ្នកអាចចូលទៅកាន់ dashboard របស់ពួកគេបាន។

ក្រៅពីទំព័រ **Account** ដែលពួកគេអាចមើលព័ត៌មានសំខាន់ៗទាក់ទងនឹងការបង់ប្រាក់ សមាជិកភាព ដែន (domains) ការកំណត់រចនាសម្ព័ន្ធផែនការ និងច្រើនទៀត ពួកគេក៏អាចចូលទៅកាន់ទំព័រ **My Sites** បានដែរ ដែលពួកគេអាចមើល និងគ្រប់គ្រងគេហទំព័រទាំងអស់របស់ពួកគេ។ អ្វីដែលពួកគេត្រូវធ្វើគឺគ្រាន់តែចុចលើ My Sites នៅលើ dashboard របស់ពួកគេប៉ុណ្ណោះ។

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

គេនឹងឃើញគេហទំព័រទាំងអស់របស់ពួកគេ ដោយមានប៊ូតុង **Manage** នៅខាងក្រោមនីមួយៗ។ ប្រសិនបើពួកគេចុចលើវា គេនឹងត្រូវបានបញ្ជូនទៅកាន់ **dashboard នៃគេហទំព័រជាក់លាក់នោះ**។

ពួកគេក៏អាចមើលបានថាគេហទំព័រមួយណាដែលជាគេហទំព័រ **ចម្បង (primary)** សម្រាប់ផែនការដែលអនុញ្ញាតឱ្យពួកគេមានគេហទំព័រច្រើនជាងមួយ។

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

នេះគឺជាទិដ្ឋភាពពេញលេញនៃចំណុចប្រទាក់កែសម្រួលគេហទំព័រ៖

![Site edit full interface](/img/admin/site-edit-full.png)

ប្រសិនបើពួកគេចង់ផ្លាស់ប្តូរគេហទំព័រចម្បងរបស់ពួកគេ ពួកគេត្រូវចូលទៅកាន់ទំព័រ Account ដោយចុចលើម៉ឺនុយ **Account**។

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

ពួកគេក៏អាចមើល និងកែសម្រួលព័ត៌មានលម្អិតរបស់អតិថិជនបានដែរ៖

![Customer edit interface](/img/admin/customer-edit.png)

បន្ទាប់មក ពួកគេគួរតែចុចលើ **Change Default Site**។

![Change Default Site button on account page](/img/account-page/change-default-site.png)

បង្អួចបណ្តោះអាសន្នមួយនឹងលេចឡើង។ ចុចលើបញ្ជីទម្លាក់ចុះ (drop-down list) ហើយជ្រើសរើសគេហទំព័រដែលអ្នកចង់ធ្វើជាគេហទំព័រចម្បង។

![Change Default Site dropdown selection](/img/account-page/change-default-site-dropdown.png)

បន្ទាប់មក គ្រាន់តែចុចលើ **Change Default Site** ហើយអ្នកបានរួចរាល់ហើយ!

![ការបញ្ជាក់ការផ្លាស់ប្តូរគេហទំព័រលំនាំដើម](/img/account-page/change-default-site-confirm.png)
