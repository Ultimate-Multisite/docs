# бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ Multisite {#multisite-setup-wizard}

Ultimate Multisite бһҳбһ¶бһ“бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбҹҶбһ“бһҪбһҷ (wizard) бһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбһҖбһҮбһ¶бһҳбһҪбһҷ бһҠбҹӮбһӣбһ“бһ№бһ„бһ”бҹҶбһ”бҹ’бһӣбҹӮбһ„бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ WordPress бһ”бҹӮбһ”бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ бһ‘бҹ…бһҮбһ¶бһ”бһҺбҹ’бһҸбһ¶бһү WordPress Multisite бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ” бһңбһ¶бһҮбһҪбһҷбһӣбһ»бһ”бһ”бҹҶбһ”бһ¶бһҸбҹӢбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһҜбһҖбһҹбһ¶бһҡ `wp-config.php` бһ¬бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹ„бһҷбһҠбҹғбҹ”

:::tip
бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ WordPress бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһ”бһҺбҹ’бһҸбһ¶бһү Multisite бһҡбһҪбһ…бһ бһҫбһҷ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһӣбһ»бһ”бһҮбҹҶбһ бһ¶бһ“бһ“бҹҒбҹҮбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„бһ”бһ¶бһ“бҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбҹҶбһ“бһҪбһҷбһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһҸбҹӮбһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣ Multisite бһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҫбһҖбһ‘бҹҒбҹ”
:::

## бһҸбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбҹҶбһ“бһҪбһҷбһ”бһ„бҹ’бһ бһ¶бһүбһ“бҹ…бһ–бҹҒбһӣбһҺбһ¶? {#when-does-the-wizard-appear}

бһ“бҹ…бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһ”бһҫбһҖ Ultimate Multisite бһ“бҹ…бһӣбһҫбһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ WordPress бһ”бҹӮбһ”бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ (бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶ Multisite) бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бҹҒбҹҮбһ“бһ№бһ„бһҡбһҖбһғбһҫбһүбһҗбһ¶ Multisite бһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҫбһҖ бһ бһҫбһҷбһңбһ¶бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбҹҶбһ“бһҪбһҷбһҠбҹҶбһЎбһҫбһ„ Multisite бһҮбҹҶбһ“бһҪбһҹбһұбҹ’бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбҹҶбһ“бһҪбһҷбһҠбҹҶбһЎбһҫбһ„бһ’бһҳбҹ’бһҳбһҸбһ¶бҹ”

бһўбҹ’бһ“бһҖбһҖбҹҸбһўбһ¶бһ…бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһңбһ¶бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ“бҹ… **WP Admin > Ultimate Multisite > Multisite Setup** бһҖбҹҸбһ”бһ¶бһ“бһҠбҹӮбһҡбҹ”

## бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҸбһҳбҹ’бһҡбһјбһң (Prerequisites) {#prerequisites}

бһҳбһ»бһ“бһ“бһ№бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбҹҶбһ“бһҪбһҷ бһҹбһјбһҳбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бҹ–

- бһўбҹ’бһ“бһҖбһҳбһ¶бһ“ **бһҹбһ·бһ‘бҹ’бһ’бһ·бһҮбһ¶бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ (administrator access)** бһӣбһҫбһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ WordPress бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
- бһҜбһҖбһҹбһ¶бһҡ `wp-config.php` бһ“бҹ…бһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ (server) бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбһ¶бһ… **бһҹбһҡбһҹбҹҒбһҡбһ”бһ¶бһ“ (writable)** бһҠбҹ„бһҷ web server
- бһўбҹ’бһ“бһҖбһҳбһ¶бһ“ **бһҖбһ¶бһҡаёӘаёіаёЈаёӯаёҮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҗбҹ’бһҳбһёбҹ— (recent backup)** бһ“бҹғбһҜбһҖбһҹбһ¶бһҡ бһ“бһ·бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

:::warning
бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбҹҶбһ“бһҪбһҷбһ“бҹҒбҹҮбһ“бһ№бһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҖбҹӮбһ”бҹ’бһҡбҹӮбһҜбһҖбһҹбһ¶бһҡ `wp-config.php` бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҸбһ¶бһҡбһ¶бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҗбҹ’бһҳбһёбҹ” бһҹбһјбһҳбһ’бҹ’бһңбһҫбһҖбһ¶бһҡаёӘаёіаёЈаёӯаёҮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (backup) бһҮбһ¶бһ“бһ·бһ…бҹ’бһ…бһҳбһ»бһ“бһ–бҹҒбһӣбһ”бһ“бҹ’бһҸбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ”
:::

## бһҮбҹҶбһ бһ¶бһ“бһ‘бһё бҹЎбҹ– бһҖбһ¶бһҡбһҹбҹ’бһңбһ¶бһӮбһҳбһ“бҹҚ (Welcome) {#step-1-welcome}

бһўбҹҒбһҖбҹ’бһҡбһ„бҹӢбһҠбҹҶбһ”бһјбһ„бһ“бһ№бһ„бһ–бһ“бҹ’бһҷбһӣбҹӢбһҗбһ¶ WordPress Multisite бһҮбһ¶бһўбҹ’бһңбһё бһ“бһ·бһ„бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбҹҶбһ“бһҪбһҷбһ“бһ№бһ„бһ’бҹ’бһңбһҫбҹ–

- бһ”бһҫбһҖбһҳбһ»бһҒбһ„бһ¶бһҡ Multisite бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ WordPress бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
- бһ”бһ„бҹ’бһҖбһҫбһҸбһҸбһ¶бһҡбһ¶бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһҺбҹ’бһҸбһ¶бһү (network database tables) бһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢ
- бһ”бһ“бҹ’бһҗбҹӮбһҳ constant multisite бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ `wp-config.php`
- бһ”бһҫбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ Ultimate Multisite бһҮбһ¶бһ”бһҺбҹ’бһҸбһ¶бһүбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҺбҹ’бһҸбһ¶бһүбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣ

![Multisite Setup Wizard - Welcome step](/img/installation/multisite-wizard/wizard-welcome.png)

бһ…бһ»бһ… **Continue** бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ“бҹ’бһҸбҹ”

## бһҮбҹҶбһ бһ¶бһ“бһ‘бһё бҹўбҹ– бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ”бһҺбҹ’бһҸбһ¶бһү (Network Configuration) {#step-2-network-configuration}

бһҮбҹҶбһ бһ¶бһ“бһ“бҹҒбҹҮбһ“бһ№бһ„бһҹбһҪбһҡбһўбҹ’бһ“бһҖбһұбҹ’бһҷбһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ”бһҺбҹ’бһҸбһ¶бһүбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

### бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡ (Site Structure) {#site-structure}

бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҡбһ”бҹҖбһ”бһҡбҹҖбһ”бһ…бҹҶбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҖбҹ’бһ“бһ»бһ„бһ”бһҺбҹ’бһҸбһ¶бһүбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–

- **Sub-domains** (бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҺбҹӮбһ“бһ¶бҹҶ) вҖ” бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһ“бһёбһҳбһҪбһҷбҹ—бһ“бһ№бһ„бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“ subdomain бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһҠбһјбһ…бһҮбһ¶ `site1.yourdomain.com`
- **Sub-directories** вҖ” бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҮбһ¶бһ•бҹ’бһӣбһјбһң (path) бһҠбһјбһ…бһҮбһ¶ `yourdomain.com/site1`

:::note
бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ sub-domains бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ **wildcard DNS** бһ“бһ·бһ„ **wildcard SSL certificate** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ domain бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” Hosting бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ WordPress бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ“бҹҒбҹҮбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ” бһҹбһјбһҳбһҳбһҫбһӣ [Ultimate Multisite 101](./ultimate-multisite-101) бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһӣбһҳбҹ’бһўбһ·бһҸбҹ”
:::

### Network Title (бһ…бҹҶбһҺбһ„бһҮбһҫбһ„бһ”бһҺбҹ’бһҸбһ¶бһү) {#network-title}

бһ”бһүбҹ’бһ…бһјбһӣбһҲбҹ’бһҳбҹ„бҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһҺбҹ’бһҸбһ¶бһүбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ‘бҹ…бһҮбһ¶бһ…бҹҶбһҺбһ„бһҮбһҫбһ„бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ”бһјбһҖбһ“бһ№бһ„бһ–бһ¶бһҖбҹ’бһҷ "Network"бҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһңбһ¶бһ”бһ¶бһ“бһ“бҹ…бһ–бҹҒбһӣбһҖбҹ’бһҡбҹ„бһҷбһ–бһёбһҖбһ¶бһҡаё•аёұа№үаёҮаё„а№ҲаёІ networkбҹ”

### Network Admin Email (бһўбҹҠбһёбһҳбҹӮбһӣбһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһҺбҹ’бһҸбһ¶бһү) {#network-admin-email}

бһўбһ¶бһҹбһҷбһҠбҹ’бһӢбһ¶бһ“бһўбҹҠбһёбһҳбҹӮбһӣбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһ“бҹғбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһҺбҹ’бһҸбһ¶бһүбҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ‘бҹ…бһҮбһ¶бһўбһ¶бһҹбһҷбһҠбҹ’бһӢбһ¶бһ“бһўбҹҠбһёбһҳбҹӮбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бҹ”

![Multisite Setup Wizard - Network Configuration](/img/installation/multisite-wizard/wizard-network-configuration.png)

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бҹҶбһ–бҹҒбһүбһңбһ¶бһ бһҫбһҷ бһҹбһјбһҳбһ…бһ»бһ… **Continue** бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ“бҹ’бһҸбҹ”

## бһҮбҹҶбһ бһ¶бһ“бһ‘бһё бҹЈбҹ– бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ (Step 3: Installation) {#step-3-installation}

бһ…бһ»бһ…бһӣбһҫбһ”бҹҠбһјбһҸбһ»бһ„ **Install** бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбҹ” Wizard бһ“бһ№бһ„бһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹҘ бһҮбҹҶбһ бһ¶бһ“бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҮбһ¶бһ”бҹӢбҹ—бһӮбҹ’бһ“бһ¶ бһҠбҹ„бһҷбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһңбһҢбҹ’бһҚбһ“бһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһңбһ¶бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бҹ–

![Multisite Setup Wizard - Installation pending](/img/installation/multisite-wizard/wizard-installation-pending.png)

| аёӮаёұа№үаёҷаё•аёӯаёҷ | аё„аёіаёӯаёҳаёҙаёҡаёІаёў |
|------|-------------|
| **а№Җаёӣаёҙаё”а№ғаёҠа№үаёҮаёІаёҷ Multisite** | а№Җаёһаёҙа№ҲаёЎаё„а№ҲаёІаё„аёҮаё—аёөа№Ҳ `WP_ALLOW_MULTISITE` аёҘаёҮа№ғаёҷа№„аёҹаёҘа№Ң `wp-config.php` |
| **аёӘаёЈа№үаёІаёҮа№Җаё„аёЈаё·аёӯаёӮа№ҲаёІаёў (Network)** | аёӘаёЈа№үаёІаёҮаё•аёІаёЈаёІаёҮаёҗаёІаёҷаёӮа№үаёӯаёЎаё№аёҘаёӘаёіаё«аёЈаёұаёҡ multisite (`wp_site`, `wp_sitemeta`, `wp_blogs`, аёҜаёҘаёҜ) а№ҒаёҘаё°а№ғаёӘа№ҲаёҒаёІаёЈаё•аёұа№үаёҮаё„а№ҲаёІа№Җаё„аёЈаё·аёӯаёӮа№ҲаёІаёўаёӮаёӯаёҮаё„аёёаё“аёҘаёҮа№„аёӣ |
| **аёӯаёұаёӣа№Җаё”аё•аёҒаёІаёЈаё•аёұа№үаёҮаё„а№ҲаёІ** | а№Җаёһаёҙа№ҲаёЎаё„а№ҲаёІаё„аёҮаё—аёөа№Ҳ multisite аёӘаёёаё”аё—а№үаёІаёўаёҘаёҮа№ғаёҷ `wp-config.php` (а№ҖаёҠа№Ҳаёҷ `MULTISITE`, `SUBDOMAIN_INSTALL`, `DOMAIN_CURRENT_SITE`, аёҜаёҘаёҜ) |
| **а№ҒаёҒа№үа№„аёӮаё„аёёаёҒаёҒаёөа№ү** | аё•аёЈаё§аёҲаёӘаёӯаёҡа№ғаё«а№үа№Ғаёҷа№Ҳа№ғаёҲаё§а№ҲаёІ URL аёӮаёӯаёҮа№Җаё§а№Үаёҡа№„аёӢаё•а№Ңаё–аё№аёҒаё•а№үаёӯаёҮа№ғаёҷа№ҖаёЎаё—аёІаё”аёІаё•а№үаёІаёӮаёӯаёҮа№Җаё„аёЈаё·аёӯаёӮа№ҲаёІаёў а№Җаёһаё·а№Ҳаёӯаёӣа№үаёӯаёҮаёҒаёұаёҷаёӣаёұаёҚаё«аёІаёҒаёІаёЈаёўаё·аёҷаёўаёұаёҷаё•аёұаё§аё•аёҷаё«аёҘаёұаёҮаёҲаёІаёҒа№Җаёӣаёҙаё”а№ғаёҠа№үаёҮаёІаёҷ |
| **а№Җаёӣаёҙаё”а№ғаёҠа№үаёҮаёІаёҷаёӣаёҘаёұа№ҠаёҒаёӯаёҙаёҷ Network Activate Plugin** | а№Җаёӣаёҙаё”а№ғаёҠа№үаёҮаёІаёҷ Ultimate Multisite а№ғаё«а№үаё—аёіаёҮаёІаёҷаё„аёЈаёӯаёҡаё„аёҘаёёаёЎаё—аёұа№үаёҮа№Җаё„аёЈаё·аёӯаёӮа№ҲаёІаёў |

а№Ғаё•а№ҲаёҘаё°аёӮаёұа№үаёҷаё•аёӯаёҷаёҲаё°а№ҒаёӘаё”аёҮаёӘаё–аёІаёҷаё°аёӯаёўа№ҲаёІаёҮа№ғаё”аёӯаёўа№ҲаёІаёҮаё«аёҷаё¶а№ҲаёҮаё”аёұаёҮаёҷаёөа№ү:

- **Pending (аёЈаёӯаё”аёіа№ҖаёҷаёҙаёҷаёҒаёІаёЈ)** вҖ” аёҒаёіаёҘаёұаёҮаёЈаёӯаёӣаёЈаё°аёЎаё§аёҘаёңаёҘаёӯаёўаё№а№Ҳ
- **Installing... (аёҒаёіаёҘаёұаёҮаё•аёҙаё”аё•аёұа№үаёҮ...)** вҖ” аёҒаёіаёҘаёұаёҮаё”аёіа№ҖаёҷаёҙаёҷаёҒаёІаёЈаёӯаёўаё№а№Ҳ
- **Success! (аёӘаёіа№ҖаёЈа№ҮаёҲ!)** вҖ” а№ҖаёӘаёЈа№ҮаёҲаёӘаёЎаёҡаё№аёЈаё“а№Ңа№ҖаёЈаёөаёўаёҡаёЈа№үаёӯаёўа№ҒаёҘа№үаё§
- **Error message (аёӮа№үаёӯаё„аё§аёІаёЎа№ҒаёӘаё”аёҮаёӮа№үаёӯаёңаёҙаё”аёһаёҘаёІаё”)** вҖ” а№ҖаёҒаёҙаё”аёӮа№үаёӯаёңаёҙаё”аёһаёҘаёІаё”аёӮаё¶а№үаёҷ (аёӮа№үаёӯаё„аё§аёІаёЎаёҲаё°аёӯаёҳаёҙаёҡаёІаёўаёӣаёұаёҚаё«аёІ)

а№ҖаёЎаё·а№Ҳаёӯаё—аёёаёҒаёӮаёұа№үаёҷаё•аёӯаёҷа№ҖаёӘаёЈа№ҮаёҲаёӘаёЎаёҡаё№аёЈаё“а№Ңаё”а№үаё§аёўаё„аё§аёІаёЎаёӘаёіа№ҖаёЈа№ҮаёҲ аё„аёёаё“аёҲаё°а№Җаё«а№ҮаёҷаёӘаё–аёІаёҷаё°аёӘаёөа№ҖаёӮаёөаёўаё§ "Success!" аёӘаёіаё«аёЈаёұаёҡа№Ғаё•а№ҲаёҘаё°аёЈаёІаёўаёҒаёІаёЈ:

![Multisite Setup Wizard - Installation complete](/img/installation/multisite-wizard/wizard-installation-complete.png)

аёҲаёІаёҒаёҷаёұа№үаёҷаё•аёұаё§аёҠа№Ҳаё§аёўаё•аёұа№үаёҮаё„а№ҲаёІ (wizard) аёҲаё°аё”аёіа№ҖаёҷаёҙаёҷаёҒаёІаёЈа№„аёӣаёўаёұаёҮаё«аёҷа№үаёІаёӘаёЈаёёаёӣаёҒаёІаёЈаё•аёҙаё”аё•аёұа№үаёҮа№Ӯаё”аёўаёӯаёұаё•а№ӮаёҷаёЎаёұаё•аёҙ

## аёӮаёұа№үаёҷаё•аёӯаёҷаё—аёөа№Ҳ 4: а№ҖаёӘаёЈа№ҮаёҲаёӘаёҙа№үаёҷ {#step-4-complete}

а№ҖаёЎаё·а№ҲаёӯаёҒаёІаёЈаё•аёҙаё”аё•аёұа№үаёҮа№ҖаёӘаёЈа№ҮаёҲаёӘаёЎаёҡаё№аёЈаё“а№Ң аё„аёёаё“аёҲаё°а№Җаё«а№ҮаёҷаёӮа№үаёӯаё„аё§аёІаёЎа№ҒаёҲа№үаёҮа№Җаё•аё·аёӯаёҷаё„аё§аёІаёЎаёӘаёіа№ҖаёЈа№ҮаёҲаё—аёөа№Ҳаёўаё·аёҷаёўаёұаёҷаё§а№ҲаёІ WordPress Multisite а№„аё”а№үаё–аё№аёҒа№Җаёӣаёҙаё”а№ғаёҠа№үаёҮаёІаёҷа№ҒаёҘа№үаё§

![Multisite Setup Wizard - Setup Complete](/img/installation/multisite-wizard/wizard-complete.png)

аё•аёӯаёҷаёҷаёөа№үаё„аёёаё“аёӘаёІаёЎаёІаёЈаё–аё”аёіа№ҖаёҷаёҙаёҷаёҒаёІаёЈаё•а№Ҳаёӯаё”а№үаё§аёўаё•аёұаё§аёҠа№Ҳаё§аёўаё•аёұа№үаёҮаё„а№ҲаёІ Ultimate Multisite а№Җаёһаё·а№ҲаёӯаёҒаёіаё«аёҷаё”аё„а№ҲаёІа№ҒаёһаёҘаё•аёҹаёӯаёЈа№ҢаёЎ WaaS аёӮаёӯаёҮаё„аёёаё“ (аёЈаёІаёўаёҘаё°а№Җаёӯаёөаёўаё”аёҡаёЈаёҙаё©аёұаё—, а№Җаёҷаё·а№үаёӯаё«аёІа№ҖаёЈаёҙа№ҲаёЎаё•а№үаёҷ, аёЈаё°аёҡаёҡаёҠаёіаёЈаё°а№ҖаёҮаёҙаёҷ аёҜаёҘаёҜ) а№„аё”а№үа№ҖаёҘаёў

:::note
бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ multisite бһ”бһүбҹ’бһ…бһ”бҹӢбһ бһҫбһҷ бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ (browser) бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһјбһ“бһ‘бҹ…бһҖбһ¶бһ“бҹӢ network admin бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһҫбһҖбһҗбҹ’бһҳбһёбҹ” бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһҸбҹ’бһҡбһјбһңбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸ бһ–бҹ’бһҡбҹ„бҹҮ cookie бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ (authentication cookies) бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ multisiteбҹ”
:::

## бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҮбҹҶбһ“бһҪбһҹбһҠбҹ„бһҷбһҠбҹғ (Manual Setup Fallback) {#manual-setup-fallback}

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бһ¶бҹҶбһҳбһ»бһҒ (wizard) бһҳбһ·бһ“бһўбһ¶бһ…бһҹбһҡбһҹбҹҒбһҡбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ `wp-config.php` бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ‘бҹҒ (бһҠбҹ„бһҷбһҹбһ¶бһҡбһ”бһүбҹ’бһ бһ¶бһҖбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһҜбһҖбһҹбһ¶бһҡ бһ¬бһҖбһ¶бһҡбһҡбһ№бһҸбһ”бһ“бҹ’бһҸбһ№бһ„бһҡбһ”бһҹбҹӢ server) бһңбһ¶бһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһҖбһјбһҠбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ“бҹ’бһҗбҹӮбһҳбһҠбҹ„бһҷбһҠбҹғбҹ–

1. **wp-config.php constants** бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бҹ…бһ–бһёбһӣбһҫбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢ `/* That's all, stop editing! */`
2. **.htaccess rewrite rules** бһҠбҹӮбһӣбһҹбһҳбһҹбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ (subdomain бһ¬ subdirectory)

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҠбҹ„бһҷбһҠбҹғбһҡбһҪбһ… бһҹбһјбһҳбһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһ‘бҹҶбһ–бҹҗбһҡ (refresh the page) бһ бһҫбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бһ¶бҹҶбһҳбһ»бһҒбһ“бһ№бһ„бһҡбһҖбһғбһҫбһүбһҗбһ¶ multisite бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ бһҫбһҷбҹ”

## бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶ (Troubleshooting) {#troubleshooting}

### бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бһ¶бҹҶбһҳбһ»бһҒбһ“бһ·бһҷбһ¶бһҷбһҗбһ¶ wp-config.php бһҳбһ·бһ“бһўбһ¶бһ…бһҹбһҡбһҹбҹҒбһҡбһ”бһ¶бһ“бһ‘бҹҒ (is not writable) {#the-wizard-says-wp-configphp-is-not-writable}

бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ web server бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһҹбһҡбһҹбҹҒбһҡбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ `wp-config.php`бҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бҹ–

- бһ”бҹ’бһҸбһјбһҡбһҖбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһҜбһҖбһҹбһ¶бһҡбһҮбһ¶бһ”бһҺбҹ’бһҸбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“бһ‘бҹ…бһҮбһ¶ `644` бһ¬ `666`
- бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһ“бҹғбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҠбҹ„бһҷбһҠбҹғбһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһҠбҹ„бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бһ¶бҹҶбһҳбһ»бһҒ
- бһҹбһҪбһҡбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢ hosting бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҮбҹҶбһ“бһҪбһҷ

### бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҳбһ·бһ“бһўбһ¶бһ…бһ…бһјбһӣбһ”бһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҠбҹҶбһЎбһҫбһ„ (subdomains) {#sites-are-not-accessible-after-setup-subdomains}

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ subdomain бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбҹҶбһҺбһҸбҹӢ **wildcard DNS** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹӮбһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бһүбҹ’бһҮбһё DNSбҹ–

```
Type: A (бһ¬ CNAME)
Host: *
Value: [your server IP]
```

бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢ hosting бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ“бҹҒбҹҮбҹ”

### бһ”бһүбҹ’бһ бһ¶бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҠбҹҶбһЎбһҫбһ„ (Authentication issues after setup) {#authentication-issues-after-setup}

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ…бҹҒбһүбһ–бһёбһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫ бһ¬бһҮбһҪбһ”бһ”бһүбҹ’бһ бһ¶ cookie бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҠбҹҶбһЎбһҫбһ„ multisiteбҹ–

1. бһӣбһ»бһ” cookies бһҡбһ”бһҹбҹӢ browser бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһ“бҹ„бҹҮ
2. бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸбһ“бҹ… `yourdomain.com/wp-login.php`
3. бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ”бһүбҹ’бһ бһ¶бһ“бҹ…бһҸбҹӮбһҳбһ¶бһ“ бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҗбһ¶ `wp-config.php` бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢ `COOKIE_DOMAIN` бһҮбһ¶ `false` вҖ” бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһ”бһүбҹ’бһ бһ¶бһҠбҹӮбһӣбһӮбҹҒбһҠбһ№бһ„бһҡбһҪбһ…бһ бһҫбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ multisite бһ“бҹ…бһӣбһҫ subdomainбҹ”

### бһҖбһ¶бһҡбһ”бҹ„бҹҮбһҮбҹҶбһ бһ¶бһ“бһҳбһҪбһҷбһ”бһҡбһ¶бһҮбҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһўбҹҶбһЎбһ»бһ„бһ–бҹҒбһӣбһҠбҹҶбһЎбһҫбһ„ (A step failed during installation) {#a-step-failed-during-installation}

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҮбҹҶбһ бһ¶бһ“бһҺбһ¶бһҳбһҪбһҷбһ“бҹғбһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„бһ”бһ„бҹ’бһ бһ¶бһүбһҖбҹҶбһ бһ»бһҹбҹ–

бҹЎ. бһҹбһјбһҳбһҖбһҸбҹӢбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҹбһ¶бһҡбһҖбҹҶбһ бһ»бһҹбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһүбҹ”
бҹў. бһҸбҹ’бһҡбһЎбһ”бҹӢбһ‘бҹ…бһҮбҹҶбһ бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһңбһ·бһү бһ бһҫбһҷбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸбҹ”
бҹЈ. бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҖбҹҶбһ бһ»бһҹбһ“бҹ…бһҸбҹӮбһ”бһ“бҹ’бһҸбһҖбһҫбһҸбһҳбһ¶бһ“ бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣ PHP error log бһҡбһ”бһҹбҹӢ server бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ”
бҹӨ. бһҮбҹҶбһ бһ¶бһ“бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢбһҡбһҪбһ…бһ бһҫбһҷбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбҹҶбһӣбһ„бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫбһҖбһҖбҹ’бһҡбҹ„бһҷ (installer бһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһҠбһјбһ…бһӮбҹ’бһ“бһ¶)
