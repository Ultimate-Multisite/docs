- **ជួសជុល:** កាលបរិច្ឆេទផុតកំណត់នៃសមាជិក (Membership expiration date) មិនទៀតទេដែលខូចខាតនៅពេលរក្សាទុកដោយប្រើតម្លៃដែលមិនមែនជាកាលបរិច្ឆេទ
- **ជួសជុល:** ការធ្វើបច្ចុប្បន្នភាពសមាជិក Stripe ឥឡូវនេះអាចលុបការបញ្ចុះតម្លៃបានត្រឹមត្រូវ ដោយមិនចាំបាច់ហៅ API deleteDiscount ដែលត្រូវបានលុបចោលទៀតទេ

- **ជួសជុល:** ការជ្រើសរើសរូបភាពនៅក្នុង Setup wizard ឥឡូវនេះធ្វើបច្ចុប្បន្នឱ្យម៉ូដែលទិន្នន័យខាងក្រោមបានត្រឹមត្រូវ
- **ជួសជុល:** Site Exporter CLI ឥឡូវនេះរក្សាទុកការជ្រើសរើសគេហទំព័រអណ្តូង (network site selection) ជាលំនាំដើមដែលត្រឹមត្រូវ
