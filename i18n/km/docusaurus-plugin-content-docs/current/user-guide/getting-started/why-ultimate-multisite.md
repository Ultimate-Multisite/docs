# ហេតុអ្វីបានជា Ultimate Multisite

បើអ្នកបានអានអត្ថបទនេះ អ្នកប្រហែលជាបានឮថា plugin របស់យើងអាចជាដំណោះស្រាយដើម្បីធ្វើឱ្យគំនិតអាជីវកម្មរបស់អ្នកក្លាយជាការពិត។ យើងសង្ឃឹមថាអ្នកនឹងទទួលបានចម្លើយដែលអ្នកត្រូវការនៅចុងបញ្ចប់នៃអត្ថបទនេះ។

## Ultimate Multisite ជាអ្វី? {#what-is-ultimate-multisite}

Ultimate Multisite គឺជា WordPress Multisite plugin ដែលអនុញ្ញាតឱ្យអ្នកផ្តល់ជូនអតិថិជននូវ WaaS (Website as a Service)។ ជាមួយនឹង plugin របស់យើង អ្នកអាចមានកម្រិតការជាវផ្សេងៗគ្នា ហើយឱ្យអតិថិជនបង់ថ្លៃប្រចាំខែដើម្បីឱ្យគេមានគេហទំព័រដែលបង្ហោះនៅក្នុងបណ្តាញ Multisite របស់អ្នក។

Ultimate Multisite អនុញ្ញាតឱ្យអ្នកទទួលការទូទាត់តាមរយៈ gateway ផ្សេងៗ (ដូចជា WooCommerce ឬ Stripe) បង្កើត site templates ដើម្បីធ្វើឱ្យជីវិតរបស់អ្នកកាន់តែងាយស្រួលនៅពេលអ្នកប្រើប្រាស់ថ្មីចុះឈ្មោះនៅលើបណ្តាញរបស់អ្នក, គូសផែនទី domain និងធ្វើឱ្យពាក្យស្នើសុំចុះឈ្មោះរបស់អ្នកមានលក្ខណៈផ្ទាល់ខ្លួនដើម្បីបង្កើតបទពិសោធន៍ដែលត្រូវបានកែច្នៃសម្រាប់អតិថិជនរបស់អ្នក។

ហើយនេះគ្រាន់តែជាការចាប់ផ្តើមប៉ុណ្ណោះ ដោយសារអ្នកអាចពង្រីកមុខងាររបស់វាជាមួយនឹងadd-ons និង webhooks បាន។

## លក្ខណៈពិសេសផ្សេងទៀត {#other-features}

Ultimate Multisite ក៏ផ្តល់ជូននូវ៖

  * ការរួមបញ្ចូល Domain Mapping ជាមួយអ្នកផ្តល់សេវាកម្មបង្ហោះ WordPress ដែលគ្រប់គ្រង (Managed WordPress hosting providers)៖ Ultimate Multisite នឹងបន្ថែម domain ដែលបានគូសផែនទីដោយស្វ័យប្រវត្តិទៅក្នុង admin panels របស់ WP Engine, Cloudways, cPanel, RunCloud, Closte និងច្រើនទៀត នៅពេលដែល domain ថ្មីត្រូវបានគូសផែនទី ដើម្បីកុំឱ្យអ្នកធ្វើវាដោយដៃ។ ក្នុងករណីខ្លះ យើងថែមទាំងអាចដំណើរការ SSL certificate fetching ជូនអ្នកទៀតផង។

  * Webhooks៖ Ultimate Multisite មានប្រព័ន្ធ Webhook ដែលមានឥទ្ធិពល ដែលអនុញ្ញាតឱ្យអ្នករួមបញ្ចូលបណ្តាញរបស់អ្នកជាមួយសេវាកម្មផ្សេងៗគ្នាជាច្រើន។ យើងថែមទាំងផ្តល់នូវការរួមបញ្ចូលដោយផ្ទាល់ជាមួយ Zapier ដែលបើកបណ្តាញរបស់អ្នកទៅកាន់កម្មវិធី និងសេវាកម្មជាង ១០០០+ ដែលមាននៅលើវេទិការបស់ពួកគេ។

* หน้า Admin ที่ทรงพลังเพื่อช่วยให้ลูกค้าของคุณใช้งานด้วย Page Builder ที่คุณชื่นชอบ: Ultimate Multisite มี add-on ที่มีประโยชน์ชื่อว่า Admin Pages Creator ซึ่งโดยพื้นฐานแล้วอนุญาตให้คุณเพิ่มหน้า Admin ที่ปรับแต่งได้อย่างสมบูรณ์ให้กับแดชบอร์ดของลูกค้าของคุณ โดยใช้ template จาก Elementor, Beaver Builder และ Brizy

* ธีม Admin สองแบบที่แตกต่างกันเพื่อปรับแต่งประสบการณ์แดชบอร์ดของลูกค้า: Check Material WP และ PRO

และฟีเจอร์ขั้นสูงอื่นๆ:

* รองรับการ Override Template: ด้วย Ultimate Multisite คุณสามารถแทนที่ HTML template เริ่มต้นสำหรับขั้นตอนการสมัครสมาชิกด้วยเวอร์ชันของคุณเองได้ โดยไม่ต้องกังวลว่าจะต้องแก้ไขไฟล์หลักของ plugin

* Search and Replace API: คุณยังสามารถใส่ placeholder ลงใน site templates เช่น {{site_name}} และใช้ Search and Replace API เพื่อเติมข้อมูลเหล่านั้นโดยอัตโนมัติด้วยข้อมูลที่ลูกค้าให้มาหลังจากสมัครสมาชิกเสร็จสิ้น

## กรณีการใช้งาน (Use cases) {#use-cases}

ตอนนี้ถึงเวลาที่จะดูตัวอย่างกรณีการใช้งานที่เป็นไปได้ของ Ultimate Multisite แล้วครับ

### กรณีที่ 1: เอเจนซี่ (An Agency) {#case-1-an-agency}

โดยปกติแล้ว ทักษะหลักของเอเจนซี่จะอยู่ที่การออกแบบเว็บไซต์ที่มีส่วนประกอบต่างๆ เช่น การโฮสต์หรือการตลาดถูกระบุเป็นบริการเพิ่มเติม

สำหรับเอเจนซี่ Ultimate Multisite นำเสนอข้อเสนอที่มีมูลค่าอย่างเหลือเชื่อในความสามารถในการโฮสต์และจัดการเว็บไซต์หลายแห่งบนแพลตฟอร์มเดียว ยิ่งไปกว่านั้น เอเจนซี่ที่กำหนดมาตรฐานการออกแบบของตนด้วยธีมเฉพาะ เช่น GeneratePress, Astra, OceanWP หรืออื่นๆ สามารถใช้ประโยชน์จากความสามารถของ Ultimate Multisite เพื่อเปิดใช้งานธีมเหล่านี้โดยอัตโนมัติสำหรับแต่ละไซต์ใหม่

คุณจะสามารถควบคุม plugin และ theme ได้อย่างใกล้ชิดในระดับผลิตภัณฑ์ (per-product basis) ผ่านการตั้งค่าของ Ultimate Multisite ซึ่งช่วยให้สามารถทำให้ plugin และ theme พร้อมใช้งานหรือซ่อนได้ รวมถึงสถานะการเปิดใช้งานเมื่อมีการสร้าง (instantiated) สำหรับไซต์ใหม่

เอเจนซี่จะรู้สึกสบายใจยิ่งขึ้นด้วย Ultimate Multisite ที่ช่วยให้พวกเขาสามารถทำในสิ่งที่พวกเขาถนัดที่สุด นั่นคือการออกแบบเว็บไซต์ที่ยอดเยี่ยม

### กรณีศึกษาที่ 2: ผู้ให้บริการเฉพาะกลุ่ม (Niche Provider) {#case-2-niche-provider}

มีคำกล่าวเก่าๆ ว่า "ทำสิ่งใดสิ่งหนึ่งให้ดี" สำหรับผู้เชี่ยวชาญหลายคน นี่หมายถึงการสร้างผลิตภัณฑ์หรือบริการโดยมีแนวคิดหลักเพียงอย่างเดียว

หากคุณเน้นไปที่การพัฒนาเครือข่ายเว็บไซต์เฉพาะกลุ่ม คุณอาจต้องการให้บริการโดยใช้โครงสร้างและแพลตฟอร์มที่เป็นมาตรฐาน นั่นคือจุดที่เทมเพลตเว็บไซต์สามารถช่วยได้!

หนึ่งในฟีเจอร์ที่เป็นนวัตกรรมของ Ultimate Multisite คือ เว็บไซต์แบบเทมเพลต (template site) ซึ่งหมายถึงเว็บไซต์ที่มีการติดตั้งและเปิดใช้งานธีม (theme), ติดตั้งและเปิดใช้งานปลั๊กอินที่จำเป็น, และมีการสร้างโพสต์หรือหน้าตัวอย่าง เมื่อลูกค้าสร้างเว็บไซต์ใหม่โดยใช้เทมเพลต เนื้อหาและการตั้งค่าของเทมเพลตจะถูกคัดลอกไปยังเว็บไซต์ที่สร้างขึ้นใหม่นั้น

สำหรับผู้ให้บริการเว็บไซต์และบริการเฉพาะกลุ่ม สิ่งนี้ให้ความได้เปรียบที่ไม่มีใครเทียบได้ในด้านความสามารถในการสร้างเว็บไซต์ที่พร้อมใช้งานทันที พร้อมปลั๊กอินและการออกแบบที่กำหนดเอง ลูกค้าเพียงแค่ต้องให้ข้อมูลขั้นต่ำเพื่อทำให้บริการเสร็จสมบูรณ์เท่านั้น

### กรณีศึกษาที่ 3: การโฮสต์ WordPress (WordPress Web Hosting) {#case-3-wordpress-web-hosting}

มีหลายวิธีในการโฮสต์เว็บไซต์ WordPress แต่แทบไม่เคยง่ายเท่ากับการให้บริการพื้นที่เว็บแก่ลูกค้าพร้อมกับติดตั้ง WordPress เวอร์ชันที่เตรียมไว้ให้แล้ว นี่เป็นเพราะการตัดสินใจและข้อควรพิจารณาหลายอย่างต้องมารวมกันเพื่อให้ได้บริการที่มีความหมาย

Ultimate Multisite โดดเด่นในด้านนี้ด้วยการนำเสนอโซลูชันแบบครบวงจร (turnkey solution) สำหรับการโฮสต์เว็บไซต์ WordPress ในส่วนของโซลูชันประกอบด้วยกลไกหลักเพื่อให้บริการแบบสมัครสมาชิก (subscription services), การเก็บเงิน, ฟอร์มชำระเงิน (checkout forms), คูปองส่วนลด และการสื่อสารกับลูกค้า

ការងារសំខាន់ៗជាច្រើនដែលត្រូវធ្វើដើម្បីដំឡើង បញ្ configurar និងថែរក្សា WordPress Multisite ឱ្យត្រឹមត្រូវ គឺត្រូវបានសម្រួលដោយ Ultimate Multisite ដែលអនុញ្ញាតឱ្យអ្នកគ្រប់គ្រងបណ្តាញ (network administrators) ផ្តោតតែលើចំណុចដែលទាក់ទងនឹងសេវាកម្ម ឬផ្នែកឯកទេសរបស់ពួកគេប៉ុណ្ណោះ ដូចជា កម្រិតផលិតផល តម្លៃ និងការផ្តល់ជូនសេវាកម្ម។

សម្រាប់អ្នកអភិវឌ្ឍន៍ដែលចង់រួមបញ្ចូលជាមួយ Ultimate Multisite ក៏មានដំណោះស្រាយដែលមាន RESTful API និង Webhooks ដ៏ទូលំទូលាយសម្រាប់ការជូនដំណឹងអំពីព្រឹត្តិការណ៍នានាផងដែរ។

ដោយមិនចាំបាច់ពឹងផ្អែកលើ plugin និង license ច្រើនផ្សេងៗទៀតទេ Ultimate Multisite ផ្តល់នូវដំណោះស្រាយសម្បូរបែបដែលមានមុខងារច្រើន និងអាចប្រៀបធៀបបាននឹង Wix, Squarespace, [WordPress.com](https://WordPress.com) និងផ្សេងៗទៀត។
