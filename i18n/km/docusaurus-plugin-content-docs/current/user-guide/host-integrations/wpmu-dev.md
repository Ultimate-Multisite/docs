# ការរួមបញ្ចូលជាមួយ WPMU DEV {#wpmu-dev-integration}

## ទិដ្ឋភាពទូទៅ {#overview}
WPMU DEV គឺជាវេទិកា WordPress ដ៏ពេញលេញ ដែលផ្តល់នូវការបង្ហោះ (hosting) ប្លុកបន្ថែម (plugins) និងសេវាកម្មសម្រាប់គេហទំព័រ WordPress។ ការរួមបញ្ចូលនេះអនុញ្ញាតឱ្យមានការធ្វើសមកាលកម្មដែន (domain syncing) ដោយស្វ័យប្រវត្តិ និងការគ្រប់គ្រងវិញ្ញាបនបត្រ SSL រវាង Ultimate Multisite និង hosting របស់ WPMU DEV។

## លក្ខណៈពិសេស {#features}
- ការធ្វើសមកាលកម្មដែនដោយស្វ័យប្រវត្តិ
- ការគ្រប់គ្រងវិញ្ញាបនបត្រ SSL
- ការព្យាយាមផ្ទៀងផ្ទាត់វិញ្ញាបនបត្រ SSL ដែលបានពន្យារពេលបន្ថែមទៀត

## តម្រូវការ {#requirements}
ការរួមបញ្ចូលនេះនឹងរកឃើញដោយស្វ័យប្រវត្តិថាតើអ្នកកំពុងបង្ហោះនៅលើ WPMU DEV ហើយប្រើ API ដែលមានស្រាប់។ មិនចាំបាច់កំណត់រចនាសម្ព័ន្ធបន្ថែមទេ ប្រសិនបើអ្នកកំពុងបង្ហោះនៅលើ WPMU DEV។

ការរួមបញ្ចូលនេះនឹងត្រួតពិនិត្យមើលវត្តមាននៃ constant `WPMUDEV_HOSTING_SITE_ID` ដែលត្រូវបានកំណត់ដោយស្វ័យប្រវត្តិនៅពេលបង្ហោះលើ WPMU DEV។

## របៀបដំឡើង {#setup-instructions}

### ១. ផ្ទៀងផ្ទាត់ការបង្ហោះរបស់ WPMU DEV {#1-verify-wpmu-dev-hosting}
ប្រសិនបើអ្នកកំពុងបង្ហោះនៅលើ WPMU DEV constant ដែលចាំបាច់គួរតែត្រូវបានកំណត់រួចហើយ សូមពិនិត្យមើល៖

1. constant `WPMUDEV_HOSTING_SITE_ID` ត្រូវបានកំណត់នៅក្នុង environment របស់អ្នក។
2. អ្នកមានសមាជិក WPMU DEV សកម្មដែលមានការចូលប្រើ API។

### ២. បើកដំណើរការការរួមបញ្ចូល (Enable the Integration) {#2-enable-the-integration}
1. នៅក្នុង WordPress admin របស់អ្នក សូមចូលទៅកាន់ Ultimate Multisite > Settings
2. ចូលទៅកាន់แท็บ "Domain Mapping"
3. រំលងចុះក្រោមទៅផ្នែក "Host Integrations"
4. បើកដំណើរការការរួមបញ្ចូល WPMU DEV
5. ចុចលើ "Save Changes"

## វាដំណើរការដូចម្តេច {#how-it-works}

### ការធ្វើសមកាលកម្មដែន (Domain Syncing) {#domain-syncing}
នៅពេលដែលដែនត្រូវបានកំណត់នៅក្នុង Ultimate Multisite៖

1. ការរួមបញ្ចូលនេះប្រើ API របស់ WPMU DEV ដើម្បីបន្ថែមដែនទៅក្នុងគណនី hosting របស់អ្នក។
2. វាថែមទាំងបន្ថែមកំណែ www នៃដែនដោយស្វ័យប្រវត្តិផងដែរ។
3. WPMU DEV នឹងគ្រប់គ្រងការកំណត់រចនាសម្ព័ន្ធដែន និងការចេញវិញ្ញាបនបត្រ SSL។

### ការគ្រប់គ្រងវិញ្ញាបនបត្រ SSL {#ssl-certificate-management}

ការតភ្ជាប់នេះត្រូវបានកំណត់ដើម្បីបង្កើនចំនួននៃការសាកល្បងផ្ទៀងផ្ទាត់វិញ្ញាបនបត្រ SSL សម្រាប់ hosting របស់ WPMU DEV ដោយសារវាអាចចំណាយពេលខ្លះសម្រាប់ការចេញ និងដំឡើងវិញ្ញាបនបត្រ SSL។ ជាទូទៅ វាព្យាយាមរហូតដល់ ១០ ដងសម្រាប់ការផ្ទៀងផ្ទាត់វិញ្ញាបនបត្រ SSL បើប្រៀបធៀបនឹងការសាកល្បងស្តង់ដារ ៥ ដង។

## ចំណាំសំខាន់ៗ {#important-notes}

### ការលុបឈ្មោះដែន (Domain Removal) {#domain-removal}

បច្ចុប្បន្ន API របស់ WPMU DEV មិនមានវិធីដើម្បីលុបឈ្មោះដែនទេ។ នៅពេលដែលការផ្គូផ្គងឈ្មោះដែនត្រូវបានលុបនៅក្នុង Ultimate Multisite នោះ ឈ្មោះដែននឹងនៅតែស្ថិតនៅក្នុងគណនី hosting របស់ WPMU DEV របស់អ្នក។ អ្នកនឹងត្រូវលុបវាដោយដៃចេញពី dashboard hosting របស់ WPMU DEV ប្រសិនបើចាំបាច់។

### ការផ្ទៀងផ្ទាត់ API (API Authentication) {#api-authentication}

ការតភ្ជាប់នេះប្រើប្រាស់ API key របស់ WPMU DEV ដែលត្រូវបានរក្សាទុកនៅក្នុង database របស់ WordPress ជាជម្រើស `wpmudev_apikey`។ វាត្រូវបានកំណត់ដោយស្វ័យប្រវត្តិនៅពេលអ្នកភ្ជាប់គេហទំព័ររបស់អ្នកទៅ WPMU DEV។

## ការដោះស្រាយបញ្ហា (Troubleshooting) {#troubleshooting}

### បញ្ហាការតភ្ជាប់ API (API Connection Issues) {#api-connection-issues}
- សូមពិនិត្យមើលថាតើគេហទំព័ររបស់អ្នកត្រូវបានភ្ជាប់ទៅ WPMU DEV យ៉ាងត្រឹមត្រូវឬអត់
- សូមពិនិត្យមើលថាតើជម្រើស `wpmudev_apikey` ត្រូវបានកំណត់នៅក្នុង database របស់ WordPress របស់អ្នកឬអត់
- សូមប្រាកដថាការជាសមាជិក WPMU DEV របស់អ្នកគឺកំពុងដំណើរការ (active)

### បញ្ហាវិញ្ញាបនបត្រ SSL (SSL Certificate Issues) {#ssl-certificate-issues}
- WPMU DEV អាចចំណាយពេលខ្លះក្នុងការចេញវិញ្ញាបនបត្រ SSL (ជាធម្មតា ៥-១៥ នាទី)។
- ការតភ្ជាប់នេះត្រូវបានកំណត់ដើម្បីពិនិត្យរហូតដល់ ១០ ដងសម្រាប់វិញ្ញាបនបត្រ SSL។
- ប្រសិនបើវិញ្ញាបនបត្រ SSL នៅតែមិនទាន់ចេញបន្ទាប់ពីព្យាយាមច្រើនដង សូមទាក់ទងក្រុម support របស់ WPMU DEV។

### ឈ្មោះដែនមិនត្រូវបានបន្ថែម (Domain Not Added) {#domain-not-added}
- សូមពិនិត្យមើល logs របស់ Ultimate Multisite សម្រាប់សារកំហុសណាមួយ។
- សូមផ្ទៀងផ្ទាត់ថាតើឈ្មោះដែននោះមិនទាន់ត្រូវបានបន្ថែមទៅក្នុង WPMU DEV រួចហើយឬនៅ។
- សូមប្រាកដថាផែនការ hosting របស់ WPMU DEV របស់អ្នកគាំទ្រចំនួនឈ្មោះដែនដែលអ្នកកំពុងបន្ថែម។
