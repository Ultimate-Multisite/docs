# ការគ្រប់គ្រងពន្ធ

Ultimate Multisite មាន​ម៉ូឌុល​ប្រមូល​ពន្ធ​ដែល​បង្កប់​ក្នុង plugin ស្នូល​របស់​យើង។ ដូច្នេះ បើ​អ្នក​ត្រូវ​ប្រមូល​ពន្ធ​លក់​លើ​ផែនការ កញ្ចប់ និង​សេវាកម្ម​របស់​អ្នក អ្នក​អាច​ធ្វើ​បាន​យ៉ាង​ងាយ ដោយ​មិន​ចាំបាច់​ដំឡើង add-ons បន្ថែម​ឡើយ។

សម្រាប់​ក្រុមហ៊ុន​ដែល​ស្ថិត​នៅ​អឺរ៉ុប យើង​ផ្តល់ **add-on** មួយ ដែល​បន្ថែម​ឧបករណ៍ និង​មុខងារ​ដើម្បី **គាំទ្រ VAT** ឱ្យ​អនុលោម​បាន​កាន់តែ​ល្អ។

Ultimate Multisite មិន​ដាក់​ឯកសារ ឬ​បង់​ពន្ធ​ជំនួស​អ្នក​ទៅ​រដ្ឋាភិបាល​ទេ។ យើង​គ្រាន់តែ​ជួយ​អ្នក​ប្រមូល​ពន្ធ​ដែល​សមស្រប​នៅ​ពេល​ធ្វើ​ប្រតិបត្តិការ​ប៉ុណ្ណោះ។ **អ្នក​នៅតែ​ត្រូវ​បង់​ពន្ធ​ដោយ​ខ្លួនឯង។**

## ការបើក​ការប្រមូល​ពន្ធ

ការប្រមូល​ពន្ធ​មិន​ត្រូវ​បាន​បើក​ជា​លំនាំដើម​ទេ។ ដើម្បី​បើក​វា អ្នក​ត្រូវ​ទៅកាន់ **Ultimate Multisite > ការកំណត់ > ពន្ធ** ហើយ​បិទបើក​ដើម្បី​បើក​ការកំណត់ បើក​ពន្ធ។

![បិទបើក​បើក​ពន្ធ​នៅ​ផ្នែក​ខាងលើ​នៃ​ទំព័រ​ការកំណត់​ពន្ធ](/img/config/settings-taxes-enable.png)

នេះ​ជា​ទិដ្ឋភាព​ពេញលេញ​នៃ​ទំព័រ​ការកំណត់​ពន្ធ៖

![ទំព័រ​ពេញ​នៃ​ការកំណត់​ពន្ធ](/img/config/settings-taxes-full.png)

អ្នក​ក៏​អាច​មើល​ការកំណត់​ពន្ធ​សម្រាប់​ផលិតផល​នីមួយៗ​បាន​ដែរ៖

![ការកំណត់​ពន្ធ​សម្រាប់​ផលិតផល](/img/config/settings-taxes.png)

### មិន​រួម​ពន្ធ និង​រួម​ពន្ធ

តាម​លំនាំដើម តម្លៃ​ផលិតផល​ទាំងអស់​របស់​អ្នក​មិន​រួម​ពន្ធ​ទេ មាន​ន័យ​ថា ពន្ធ **មិន​ត្រូវ​បាន​រួម​បញ្ចូល** ក្នុង​តម្លៃ​ផលិតផល​ទេ។ បើ​យើង​កំណត់​ថា​អតិថិជន​គួរ​បង់​ពន្ធ​លើ​ការទិញ​មួយ យើង​នឹង​បន្ថែម​ពន្ធ **លើស​ពី** សរុប​រង។

បើ​អ្នក​ចង់​ឱ្យ​តម្លៃ​ផលិតផល​របស់​អ្នក​រួម​ពន្ធ អ្នក​អាច​ធ្វើ​បាន​ដោយ​បើក​ការកំណត់ **ពន្ធ​រួម​បញ្ចូល**។

![ជួរ​បិទបើក​ពន្ធ​រួម​បញ្ចូល​នៅ​ខាងក្រោម​ការកំណត់​បើក​ពន្ធ](/img/config/settings-taxes-inclusive.png)

កុំ​ភ្លេច **រក្សាទុក** ការផ្លាស់ប្តូរ​ដែល​អ្នក​បាន​ធ្វើ។

## ការបង្កើត​អត្រា​ពន្ធ

បន្ទាប់ពី​អ្នក​បើក​ការប្រមូល​ពន្ធ អ្នក​ត្រូវ​បង្កើត​អត្រា​ពន្ធ​សម្រាប់​ទីតាំង​ជាក់លាក់ ដោយ​ប្រើ​កម្មវិធី​កែសម្រួល​អត្រា​ពន្ធ​របស់​យើង។

អ្នក​អាច​ចូល​ប្រើ​កម្មវិធី​កែសម្រួល​បាន ដោយ​ចុច​ប៊ូតុង **គ្រប់គ្រង​អត្រា​ពន្ធ** នៅ​របារ​ចំហៀង​នៃ​ទំព័រ​ការកំណត់​ពន្ធ។

![តំណ​គ្រប់គ្រង​អត្រា​ពន្ធ​ក្នុង​ផ្ទាំង​អត្រា​ពន្ធ​នៅ​លើ​ទំព័រ​ការកំណត់](/img/config/settings-taxes-manage-rates.png)

នៅ​ទំព័រ​កម្មវិធី​កែសម្រួល​អត្រា​ពន្ធ អ្នក​អាច​បន្ថែម​អត្រា​ពន្ធ​ថ្មី​ដោយ​ចុច​ប៊ូតុង **បន្ថែម​ជួរ​ថ្មី**។

![តារាង​កម្មវិធី​កែសម្រួល​អត្រា​ពន្ធ​ដែល​មាន​ប៊ូតុង​បន្ថែម​ជួរ​ថ្មី​នៅ​ខាងលើ](/img/config/tax-rates-editor.png)

អ្នក​ត្រូវ​ផ្តល់ **ចំណងជើង** ដល់​អត្រា​ពន្ធ​នីមួយៗ (ប្រើ​នៅ​លើ​វិក្កយបត្រ)។ បន្ទាប់មក អ្នក​អាច​ជ្រើស **ប្រទេស** (តម្រូវ​ឱ្យ​មាន), **រដ្ឋ,** និង **ទីក្រុង** (ទាំងពីរ​ជា​ជម្រើស) ដែល​ពន្ធ​នេះ​នឹង​ត្រូវ​បាន​គិត។ ចុងក្រោយ បន្ថែម **អត្រា​ពន្ធ​ជា​ភាគរយ**។

### ប្រភេទ​ពន្ធ

អ្នក​ក៏​អាច​បង្កើត​ប្រភេទ​ពន្ធ​ច្រើន ដើម្បី​បន្ថែម​អត្រា​ពន្ធ​ផ្សេងគ្នា​សម្រាប់​ប្រភេទ​ផលិតផល​ផ្សេងៗ។

ចុច **បន្ថែម​ប្រភេទ​ពន្ធ​ថ្មី** បន្ទាប់មក​សរសេរ​ឈ្មោះ​ប្រភេទ​របស់​អ្នក ហើយ​ចុច **បង្កើត**។

![ប៊ូតុង​បន្ថែម​ប្រភេទ​ពន្ធ​ថ្មី​នៅ​ខាងលើ​កម្មវិធី​កែសម្រួល​អត្រា​ពន្ធ](/img/config/tax-categories-add.png)

![វាល​បញ្ចូល​ឈ្មោះ​ប្រភេទ​ពន្ធ​ក្នុង modal បង្កើត​ប្រភេទ](/img/config/tax-categories-create-modal.png)

ដើម្បី​រកមើល​ប្រភេទ ចុច **ប្តូរ** ហើយ​ជ្រើស​ប្រភេទ​ដែល​អ្នក​ចង់​បន្ថែម​ពន្ធ​ថ្មី​ទៅ។

![ប៊ូតុង dropdown ប្តូរ ដើម្បី​ប្តូរ​រវាង​ប្រភេទ​ពន្ធ](/img/config/tax-categories-switch.png)

![dropdown ជ្រើស​ប្រភេទ​ពន្ធ​ដែល​បង្ហាញ​ប្រភេទ​ដែល​មាន](/img/config/tax-categories-select.png)

អ្នក​អាច​កំណត់​ប្រភេទ​ពន្ធ​សម្រាប់​ផលិតផល​ជាក់លាក់​មួយ ដោយ​ទៅកាន់ **ទំព័រ​កែសម្រួល​ផលិតផល** ហើយ​បន្ទាប់មក​ទៅ​ផ្ទាំង​ពន្ធ។

![ផ្ទាំង​ពន្ធ​របស់​ផលិតផល​ដែល​មាន​ប្រភេទ​ពន្ធ និង​បិទបើក​ត្រូវ​គិត​ពន្ធ](/img/config/product-taxes.png)

នៅ​លើ​អេក្រង់​ដដែល អ្នក​អាច​បិទ​បិទបើក **ត្រូវ​គិត​ពន្ធ​ឬ?** ដើម្បី​ឱ្យ Ultimate Multisite ដឹង​ថា វា​មិន​គួរ​ប្រមូល​ពន្ធ​លើ​ផលិតផល​នោះ​ទេ។

## ការគាំទ្រ VAT អឺរ៉ុប

ដូច​បាន​លើកឡើង​មុន​នេះ យើង​មាន add-on សម្រាប់​អតិថិជន​នៅ​សហភាព​អឺរ៉ុប ដែល​មាន​តម្រូវការ​បន្ថែម​ដោយសារ​បទប្បញ្ញត្តិ VAT អឺរ៉ុប។

ឧបករណ៍ VAT របស់​យើង​ជួយ​លើ​ចំណុច​សំខាន់​មួយចំនួន៖

  * ការផ្ទុក​អត្រា VAT របស់​សហភាព​អឺរ៉ុប​បាន​ងាយស្រួល;

  * ការប្រមូល និង​ផ្ទៀងផ្ទាត់​លេខ VAT - និង​ការគិត​បញ្ច្រាស​សម្រាប់​អង្គភាព​ដែល​លើកលែង VAT (ដូចជា​ក្រុមហ៊ុន​ដែល​មាន​លេខ VAT ត្រឹមត្រូវ);

ដើម្បី​ដំឡើង add-on នោះ សូម​ទៅកាន់ **Ultimate Multisite > ការកំណត់** ហើយ​ចុច​តំណ​របារ​ចំហៀង **ពិនិត្យ Add-ons របស់​យើង**។

![របារ​ចំហៀង​ទំព័រ​ការកំណត់​ដែល​មាន​តំណ​ពិនិត្យ Add-ons របស់​យើង](/img/config/settings-taxes-addons-link.png)

អ្នក​នឹង​ត្រូវ​បាន​បញ្ជូន​បន្ត​ទៅ​ទំព័រ add-ons របស់​យើង។ នៅទីនោះ អ្នក​អាច​ស្វែងរក **Ultimate Multisite VAT add-on** ហើយ​ដំឡើង​វា។

បើ​អ្នក​ត្រឡប់​ទៅ **ផ្ទាំង​ការកំណត់​ពន្ធ** អ្នក​នឹង​ឃើញ​ជម្រើស​ថ្មីៗ​មាន​ឱ្យ​ប្រើ។ បិទបើក​ជម្រើស **បើក​ការគាំទ្រ VAT** ដើម្បី​បើក​ឧបករណ៍ VAT ថ្មី។ កុំ​ភ្លេច **រក្សាទុក** ការកំណត់​របស់​អ្នក!

### ការទាញ​អត្រា​ពន្ធ VAT

ឧបករណ៍​មួយ​ដែល integration របស់​យើង​បន្ថែម គឺ​សមត្ថភាព​ផ្ទុក​អត្រា​ពន្ធ​សម្រាប់​រដ្ឋ​សមាជិក​សហភាព​អឺរ៉ុប។ វា​អាច​ធ្វើ​បាន​ដោយ​ចូល​ទៅ​ទំព័រ​កម្មវិធី​កែសម្រួល​អត្រា​ពន្ធ បន្ទាប់ពី​បើក​ការគាំទ្រ EU VAT។
