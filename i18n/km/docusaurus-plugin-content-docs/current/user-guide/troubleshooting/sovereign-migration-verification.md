# бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһўбһ’бһ·бһ”бһҸбҹҒбһҷбҹ’бһҷбһ—бһ¶бһ– (Sovereign Migration Verification) {#sovereign-migration-verification}

Ultimate Multisite: Multi-Tenancy 1.2.0 бһҳбһ¶бһ“бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ WP-CLI бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ tenant бһҠбҹӮбһӣбһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһўбһ’бһ·бһ”бһҸбҹҒбһҷбҹ’бһҷбҹ” бһҹбһјбһҳбһ”бҹ’бһҡбһҫбһңбһ¶ бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ tenant, бһҖбһ¶бһҡбһ…бһјбһӣбһҳбһҫбһӣ SSO бһ¬бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„бһҠбҹ„бһҷбһЎбҹӮбһҖ бһҳбһ·бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбһјбһ…бһҖбһ¶бһҡбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбһ‘бҹҒбҹ”

## бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ (Commands to run) {#commands-to-run}

бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ–бһёбһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ WordPress бһӣбһҫбһ”бһҺбҹ’бһҸбһ¶бһүбҹ–

```bash
wp tenant verify-no-legacy --site=<site-id>
wp tenant verify-sovereign-push --site=<site-id>
```

бһ”бҹ’бһҡбһҫ site ID бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ tenant бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбҹ” бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһ‘бһёбһҳбһҪбһҷбһ“бһ№бһ„бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҗбһ¶бһҸбһҫбһҸбҹҒбһ“бһ„бҹӢбһ“бҹ„бҹҮбһҳбһ·бһ“бһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бһ¶бһҹбҹӢбҹ—бһ“бҹ…бһҒбһ¶бһ„бһ”бһҺбҹ’бһҸбһ¶бһүбһ‘бҹҖбһҸбһ‘бҹҒбҹ” бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһ‘бһёбһ–бһёбһҡбһ“бһ№бһ„бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҗбһ¶ job бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ sovereign push бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ“бһ·бһ„бһ”бһ„бҹ’бһ бһјбһҡбһ…бҹҒбһүбһ”бһ¶бһ“бҹ”

## бһҖбһ¶бһҡбһ”бһҡбһ¶бһҮбҹҗбһҷбһ‘бһјбһ‘бҹ… (Common failures) {#common-failures}

### бһҖбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһӮбҹ’бһ“бһ¶бһҮбһ¶бһҳбһҪбһҷ host {#database-grants-do-not-match-the-host}

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҡбһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһ–бһёбһ”бһүбҹ’бһ бһ¶ grant бһ¬ writer-user бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣ database host бһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбҹ” `localhost`, `127.0.0.1` бһ“бһ·бһ„бһҲбҹ’бһҳбҹ„бҹҮ service бһҖбҹ’бһ“бһ»бһ„ container бһӮбһәбһҮбһ¶ database host MySQL бһ•бҹ’бһҹбҹҒбһ„бҹ—бһӮбҹ’бһ“бһ¶бҹ” бһҹбһјбһҳбһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ– binding host tenant бһ¬ database grants бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһЎбһҫбһ„бһңбһ·бһүбҹ”

### бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ Bedrock бһ¬ local socket бһҳбһ·бһ“бһўбһ¶бһ…бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ”бһ¶бһ“бһ‘бҹҒ {#bedrock-or-local-installs-cannot-connect}

Bedrock бһ“бһ·бһ„ local socket installs бһўбһ¶бһ…бһҡбһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһҗбһ¶ database бһҮбһ¶ `localhost` бһҒбһҺбҹҲбһ–бҹҒбһӣбһҠбҹӮбһӣ runtime бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҸбһ¶бһҳбһҡбһҷбҹҲбһўбһ¶бһҹбһҷбһҠбҹ’бһӢбһ¶бһ“бһ’бһҳбҹ’бһҳбһҸбһ¶ (normalized address)бҹ” бһҖбҹҶбһҺбҹӮ 1.2.0 бһ’бҹ’бһңбһҫбһұбҹ’бһҷ host string бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҸбҹӮбһҳбһҪбһҷбһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮ override host бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҖбҹҶбһҺбһҸбҹӢбһҠбҹ„бһҷбһҒбҹ’бһӣбһҪбһ“бһҜбһ„бһ“бҹ…бһҸбҹӮбһўбһ¶бһ…бһ”бҹүбҹҮбһ‘бһ„бҹ’бһӮбһ·бһ…бһҮбһ¶бһҳбһҪбһҷ database grants бһ”бһ¶бһ“бҹ”

### Async push queue бһҳбһ·бһ“бһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһјбһҡбһ…бҹҒбһүбһ‘бҹҒ (does not drain) {#async-push-queue-does-not-drain}

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫ `verify-sovereign-push` бһҳбһ·бһ“бһ”бһүбҹ’бһ…бһ”бҹӢ бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣ Action Scheduler бһ¬ async runner бһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбҹ” бһҹбһјбһҳбһӣбһ»бһ” job бһҠбҹӮбһӣбһ”бһҡбһ¶бһҮбҹҗбһҷ бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶бһңбһ¶бһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҠбһҫбһҳбҹ’бһ”бһёбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸ бһ¬бһ”бҹ„бҹҮбһ”бһ„бҹӢбһ…бҹ„бһӣбҹ”

### бһ…бҹҶбһ“бһҪбһ“бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ tenant бһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ‘бҹҒ (Tenant user count is wrong) {#tenant-user-count-is-wrong}

бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһӮбһҪбһҡбһҸбҹӮбһ”бһ„бҹ’бһҖбһҫбһҸбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ tenant бһҠбҹӮбһӣбһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһўбһ’бһ·бһ”бһҸбҹҒбһҷбҹ’бһҷбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҠбҹҶбһЎбһҫбһ„бһҠбҹӮбһӣбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„ missing бһҹбһјбһҳбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбҹҶбһ бһ¶бһ“бһ“бҹғбһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһЎбһҫбһ„бһңбһ·бһү бһҳбһ»бһ“бһ“бһ№бһ„бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳ SSO бһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸбҹ”

### бһҖбһ¶бһҡбһ…бһјбһӣбһҳбһҫбһӣ SSO бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҠбһ·бһҹбҹҒбһ’ (SSO visit is rejected) {#sso-visit-is-rejected}

бһҖбһ¶бһҡбһ…бһјбһӣбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· (autologin) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһҮбһ·бҹҮ (tenant) бһҠбҹӮбһӣбһӮбҹ’бһҳбһ¶бһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– (stateless) бһҸбһҳбҹ’бһҡбһјбһңбһұбҹ’бһҷбһҲбҹ’бһҳбҹ„бҹҮбһҠбҹӮбһ“ (domain), бһҖбһјбһ“бһҹбҹ„бһҠбһҫбһҳ (origin pin), бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ“бҹғ аҰҹа§ӢаҰ•а§ҮаҰЁ (token purpose), nonce, бһ“бһ·бһ„бһҗбҹ’бһ„бҹғбһ•бһ»бһҸбһҖбҹҶбһҺбһҸбҹӢ (expiry) бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһӮбҹ’бһ“бһ¶бҹ” бһҹбһјбһҳбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶ URL бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһ·бҹҮбһӮбһәбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ бһҫбһҷбһҖбһ¶бһҡбһ…бһјбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһ—бҹ’бһӣбһ¶бһҳбҹ—бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһ„бҹ’бһҖбһҫбһҸ SSO visitбҹ”

## бһ“бҹ…бһ–бҹҒбһӣбһҺбһ¶бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸ {#when-to-retry}

бһҹбһјбһҳбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһЎбһҫбһ„бһңбһ·бһүбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҳбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ“бһёбһҳбһҪбһҷбҹ—бҹ” бһҖбһ»бҹҶбһ”бҹ’бһҸбһјбһҡбһ…бһҡбһ¶бһ…бһҡбһҺбҹҚбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ (production traffic), бһҖбһ»бҹҶбһӣбһ»бһ”бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбһҫбһҳ (source data), бһ¬бһҖбһ»бҹҶбһҠбһҖбһҷбһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ’бҹ’бһңбһҫ migrration бһҡбһ бһјбһҸбһҠбһӣбҹӢбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ”бһ¶бһ“бһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢбҹ”
