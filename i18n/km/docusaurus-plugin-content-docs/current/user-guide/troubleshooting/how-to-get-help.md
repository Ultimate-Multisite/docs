# бһҡбһ”бҹҖбһ”бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҮбҹҶбһ“бһҪбһҷ {#how-to-get-help}

бһўбҹ’бһ“бһҖбһҳбһ¶бһ“ License key бһҠбҹӮбһӣбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ бһҫбһҷбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбһҖбһҮбһ¶бһҳбһҪбһҷбһ“бҹ„бҹҮбһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбҹ’бһҡбһ»бһҳбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҮбһҪбһ”бһ”бҹ’бһҡбһ‘бҹҮбһҖбҹ’бһ“бһ»бһ„бһўбҹҶбһЎбһ»бһ„бһ–бҹҒбһӣбһ“бҹғбһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶ Ultimate Multisiteбҹ”

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһӣбһҫаёҠа№ҲаёӯаёҮаё—аёІаёҮаёҒаёІаёЈбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһҖбһ¶бһ“бҹӢ License бһ“бһ·бһ„бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҺбһ¶бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“бһ“бҹ…бһҖбҹ’бһ”бҹӮбһҡбһҒбҹ’бһӣбһҪбһ“бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҖбҹ’бһҡбһ»бһҳбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бҹ”

## Help Site (бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҮбҹҶбһ“бһҪбһҷ) {#help-site}

бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бһјбһӣбһҳбһҫбһӣбһўбһҸбҹ’бһҗбһ”бһ‘ бһ“бһ·бһ„бһҳбҹҒбһҡбҹҖбһ“бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ–бһё help site бһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһҖбһ”бһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһҺбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮбҹ”

[)

бһҷбһҫбһ„бһӣбһҫбһҖбһ‘бһ№бһҖбһ…бһ·бһҸбҹ’бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҹбҹ’бһӮбһ¶бһӣбҹӢбһ–бһёбһҳбһ»бһҒбһ„бһ¶бһҡ бһ“бһ·бһ„бһҳбһ»бһҒбһңбһ·бһҮбҹ’бһҮбһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҡбһ”бһҹбҹӢ Ultimate Multisiteбҹ” бһҳбһ¶бһ“бһўбһҸбҹ’бһҗбһ”бһ‘ "Getting Started" (бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбһ¶бһҖбҹӢбһҮбһјбһ“бһ“бҹ…бһӣбһҫ help site бһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһ бһҫбһҷбһ…бһҳбҹ’бһӣбһҫбһҷбһ…бҹҶбһ–бҹ„бҹҮбһҹбҹҶбһҺбһҪбһҡбһ‘бһјбһ‘бҹ…бһҖбҹҸбһўбһ¶бһ…бһҡбһҖбһ”бһ¶бһ“бһ“бҹ…бһ‘бһёбһ“бҹ„бҹҮбһҠбҹӮбһҡ бһҠбһјбһ…бһҮбһ¶бһңбһ·бһ’бһёбһҠбҹҶбһЎбһҫбһ„ add-ons, бһҡбһ”бҹҖбһ”бһ”бһүбҹ’бһ…бһјбһӣ payment gateways бһҮбһ¶бһҠбһҫбһҳбҹ”

## Contact Support (бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҳбһҖбһҖбһ¶бһ“бҹӢбһҖбҹ’бһҡбһ»бһҳбһӮбһ¶бҹҶбһ‘бҹ’бһҡ) {#contact-support}

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҮбһ¶бһ”бҹӢбһӮбһ¶бҹҶбһ„ бһ“бһ·бһ„бһҳбһ·бһ“бһўбһ¶бһ…бһҡбһҖбһ…бһҳбҹ’бһӣбһҫбһҷбһ–бһё help site бһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһ”бһ¶бһ“ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҖбҹ’бһҡбһ»бһҳбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһ”бһ¶бһ“бһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһ…бһ»бһ…бһӣбһҫбһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„ speech bubble бһҠбҹӮбһӣбһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһҒбһ¶бһ„бһҹбҹ’бһҸбһ¶бҹҶбһҠбҹғбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹғбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһ¬бһҠбҹ„бһҷбһ•бҹ’бһүбһҫбһўбҹҠбһёбһҳбҹӮбһӣбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ‘бҹ… support@ultimatemultisite.comбҹ”

бһ“бҹ…бһ–бҹҒбһӣбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҖбҹ’бһҡбһ»бһҳбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ бһ¶бһ•бҹ’бһ“бҹӮбһҖбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ бһҹбһјбһҳбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһӣбһҳбҹ’бһўбһ·бһҸбһўбҹҶбһ–бһёбһ”бһүбҹ’бһ бһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҡбһҪбһҳбһҮбһ¶бһҳбһҪбһҷбһҡбһјбһ”бһҗбһҸбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ (screenshot) бһҹбһ¶бһҡбһҖбҹҶбһ бһ»бһҹ бһ¬бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҺбһ¶бһҳбһҪбһҷбһҠбҹӮбһӣбһўбһ¶бһ…бһҮбһҪбһҷбһҠбһӣбҹӢбһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖбһҮбҹҶбһ“бһҪбһҷбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһұбҹ’бһҷбһҷбһӣбҹӢ бһ“бһ·бһ„бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ”бһүбҹ’бһ бһ¶бһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбҹ” бһўбҹ’бһ“бһҖбһҖбҹҸбһҸбҹ’бһҡбһјбһңбһ•бҹ’бһҸбһӣбҹӢ license key бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ•бһ„бһҠбҹӮбһҡбҹ” бһҳбһ¶бһ“бһҸбҹӮбһўбҹ’бһ“бһҖбһҖбһ¶бһ“бҹӢ License бһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ·бһ‘бҹ’бһ’бһ·бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбҹ”

## Community Site (бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҹбһ бһӮбһҳбһ“бҹҚ) {#community-site}

[community site](https://community.ultimatemultisite.com/) бһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһӮбһәбһҮбһ¶бһҖбһ“бҹ’бһӣбҹӮбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Ultimate Multisite бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶бҹ” бһңбһ¶бһҠбһҫбһҡбһҸбһҪбһҮбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҠбҹӮбһӣбһҹбһҳбһ¶бһҮбһ·бһҖбһўбһ¶бһ…бһ”бһ„бҹ’бһ бҹ„бҹҮбһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһӣбһҫбһҖбһ¶бһҡбһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶ бһ•бҹ’бһҸбһӣбҹӢбһҠбҹҶбһ”бһјбһ“бҹ’бһҳбһ¶бһ“ бһҹбһ бһҖбһ¶бһҡ бһ¬бһ”бҹ’бһҡбһҳбһјбһӣбһ’бһ“бһ’бһ¶бһ“бһ–бһёбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ•бҹ’бһҸбһӣбҹӢ Ultimate Multisite license key бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ„бһҡбһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢ spaces/pages бһҜбһҖбһҮбһ“бҹ”
