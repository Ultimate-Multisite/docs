# របៀបទទួលបានជំនួយ

អ្នកមាន License key ដែលត្រឹមត្រូវ ហើយភ្ជាប់មកជាមួយនោះគឺជាការចូលប្រើប្រាស់ក្រុមរបស់យើងដើម្បីជួយអ្នកដោះស្រាយបញ្ហាដែលអ្នកអាចជួបប្រទះក្នុងអំឡុងពេលនៃការរៀបចំ Ultimate Multisite។

មេរៀននេះគ្របដណ្តប់លើช่องทางการគាំទ្រដែលមានសម្រាប់អ្នកកាន់ License និងព័ត៌មានប្រភេទណាដែលអ្នកត្រូវមាននៅក្បែរខ្លួនដើម្បីទាក់ទងក្រុមគាំទ្ររបស់យើង។

## Help Site (គេហទំព័រជំនួយ) {#help-site}

អ្នកអាចចូលមើលអត្ថបទ និងមេរៀនផ្សេងៗពី help site របស់យើង ដែលអ្នកអាចរកបាននៅក្នុងតំណភ្ជាប់ខាងក្រោមនេះ។

យើងលើកទឹកចិត្តឱ្យអ្នកប្រើប្រាស់ស្គាល់ពីមុខងារ និងមុខវិជ្ជាមូលដ្ឋានរបស់ Ultimate Multisite។ មានអត្ថបទ "Getting Started" (ចាប់ផ្តើម) សម្រាប់ដាក់ជូននៅលើ help site របស់យើង ហើយចម្លើយចំពោះសំណួរទូទៅក៏អាចរកបាននៅទីនោះដែរ ដូចជាវិធីដំឡើង add-ons, របៀបបញ្ចូល payment gateways ជាដើម។

## Contact Support (ទំនាក់ទំនងមកកាន់ក្រុមគាំទ្រ) {#contact-support}

ប្រសិនបើអ្នកជាប់គាំង និងមិនអាចរកចម្លើយពី help site របស់យើងបាន អ្នកអាចទាក់ទងក្រុមគាំទ្ររបស់យើងបានតាមរយៈការចុចលើរូបតំណាង speech bubble ដែលស្ថិតនៅខាងស្តាំដៃខាងក្រោមនៃគេហទំព័ររបស់យើង ឬដោយផ្ញើអ៊ីមែលផ្ទាល់ទៅ support@ultimatemultisite.com។

នៅពេលទាក់ទងក្រុមគាំទ្រសម្រាប់បញ្ហាផ្នែកបច្ចេកទេស សូមប្រាកដថាអ្នកផ្តល់ការពិពណ៌នាលម្អិតអំពីបញ្ហារបស់អ្នក រួមជាមួយរូបថតអេក្រង់ (screenshot) សារកំហុស ឬព័ត៌មានណាមួយដែលអាចជួយដល់បុគ្គលិកជំនួយរបស់យើងឱ្យយល់ និងវាយតម្លៃបញ្ហាបានយ៉ាងងាយស្រួល។ អ្នកក៏ត្រូវផ្តល់ license key របស់អ្នកផងដែរ។ មានតែអ្នកកាន់ License ដែលកំពុងដំណើរការប៉ុណ្ណោះដែលមានសិទ្ធិទទួលបានការគាំទ្រ។
