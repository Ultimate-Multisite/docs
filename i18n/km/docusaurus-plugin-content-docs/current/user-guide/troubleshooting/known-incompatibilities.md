# ភាពមិនត្រូវគ្នាដែលគេស្គាល់ជាមួយ Plugin ផ្សេងទៀត {#known-incompatibilities-with-other-plugins}

បន្ទាប់ពីដំឡើង និងបើកដំណើរការ Ultimate Multisite រួច អ្នកប្រហែលជាជួបបញ្ហាដោយសារតែមានភាពមិនត្រូវគ្នាមួយចំនួនជាមួយ plugin ផ្សេងៗ។

ពួកយើងកំពុងខិតខំធ្វើឱ្យបញ្ជីនេះតូចទៅវិញជាមួយនឹងការចេញផ្សាយថ្មីរៀងរាល់ពេល ប៉ុន្តែប្រសិនបើ plugin ណាមួយត្រូវបានចុះបញ្ជីនៅទីនេះ អ្នកប្រហែលជាចាំបាច់ត្រូវបិទវាដើម្បីឱ្យ Ultimate Multisite Network ដំណើរការពេញលេញ។

ខាងក្រោមនេះគឺជាបញ្ជីនៃ plugin ទាំងនោះ និងអ្វីដែលអ្នកគួរធ្វើក្នុងករណីទាំងនេះ៖

**Plugin WPMU DEV ខ្លះៗ** Plugin មួយចំនួនដែល _មិនត្រូវបានថែរក្សាទៀត_ ដោយ WPMU DEV ត្រូវបានគេស្គាល់ថាបង្កឱ្យមានការរារាំងដំណើរការចុះឈ្មោះ Ultimate Multisite នៅជំហានចុងក្រោយ (អ្នកប្រើប្រាស់មិនអាចបង្កើតគណនីបាន)។ នេះគឺជាបញ្ជីនៃ plugin ដែលបង្កបញ្ហាទាំងនោះ៖

  * Signup Language: _Ultimate Multisite មាន add-on ដែលផ្តល់មុខងារដូចគ្នា គឺ Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _នេះត្រូវបានបង្កើតបញ្ចូលក្នុង Ultimate Multisite រួចហើយ ដូច្នេះមិនចាំបាច់មាន plugin បន្ថែមទេ។_

  * New Blog Templates: _នេះត្រូវបានបង្កើតបញ្ចូលក្នុង Ultimate Multisite រួចហើយ ដូច្នេះមិនចាំបាច់មាន plugin បន្ថែមទេ។_

  * Set Password on Multisite Blog Creation: _នេះត្រូវបានបង្កើតបញ្ចូលក្នុង Ultimate Multisite រួចហើយ ដូច្នេះមិនចាំបាច់មាន plugin បន្ថែមទេ។_

  * Signup TOS: _នេះត្រូវបានបង្កើតបញ្ចូលក្នុង Ultimate Multisite រួចហើយ ដូច្នេះមិនចាំបាច់មាន plugin បន្ថែមទេ។_

  * Anti-Splog: ទោះបីជាវាមានអំណាចតិចជាងក៏ដោយ ក៏វាក៏ _ត្រូវបានបង្កើតបញ្ចូលក្នុង Ultimate Multisite ដែរ_។
