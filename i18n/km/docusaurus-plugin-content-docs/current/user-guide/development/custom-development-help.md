# របៀបទទួលបានជំនួយអភិវឌ្ឍន៍ផ្ទាល់ខ្លួនពី Codeable?

ជួនកាល យើងពិតជាត្រូវការការប្ដូរតាមបំណងដ៏អស្ចារ្យខ្លះ ដើម្បីធ្វើឱ្យលំហូរការងាររបស់យើងដំណើរការបានត្រឹមត្រូវតាមដែលយើងចង់បាន។ ប្រសិនបើអ្នកកំពុងស្វែងរកអ្វីនោះទេ យើងសូមណែនាំ [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) ដែលជាកន្លែងដែលអ្នកអាចស្វែងរកអ្នកអភិវឌ្ឍន៍ដែលមានបទពិសោធន៍ជំនាញលើ Ultimate Multisite។

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) គឺជាវេទិកាមួយដែលភ្ជាប់អតិថិជនជាមួយអ្នកអភិវឌ្ឍន៍ WordPress Freelance ដែលមានលក្ខណៈសម្បត្តិគ្រប់គ្រាន់ ដែលអាចធ្វើការលើគម្រោងគ្រប់ទំហំ។ ពួកគេមានដំណើរការដាក់ពាក្យដ៏យកចិត្តទុកដាក់ ដើម្បីធានាថាអ្នកត្រូវបានផ្គូផ្គងជាមួយអ្នកជំនាញដែលមានគុណភាពសម្រាប់ភារកិច្ចដែលអ្នកមាននៅក្នុងចិត្តជាក់លាក់។

### របៀបបង្កើត project មួយនៅក្នុង Codeable?

ដំបូង សូមចូលទៅកាន់ [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) ហើយចុចលើប៊ូតុង **Start a Project** (ចាប់ផ្តើមគម្រោង)។

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

ឥឡូវនេះ ដើម្បីដាក់ស្នើ project សូមចុចទៅខាងឆ្វេងដើម្បី **Start a Project** (ចាប់ផ្តើមគម្រោង)។

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

នៅក្នុងផ្នែក "I need help with" (ខ្ញុំត្រូវការជំនួយជាមួយ) សូមជ្រើសរើស **Customizing** (ការប្ដូរតាមបំណង)៖

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

នៅក្នុងផ្នែក "My" (របស់ខ្ញុំ) សូមជ្រើសរើស **Plugin** (Addon)៖

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

បន្ទាប់មក ចុចលើប៊ូតុង **Continue** (បន្ត)។

<!-- Screenshot unavailable: Codeable project form Continue button -->

ឥឡូវនេះ ដល់ពេលដែលត្រូវពិពណ៌នាអំពី project និងតម្រូវការរបស់អ្នកតាមរបៀបល្អបំផុតដែលអាចធ្វើទៅបានហើយ។ សូមព្យាយាមផ្តល់ព័ត៌មានឱ្យបានច្រើនតាមដែលអាចធ្វើទៅបាន!

<!-- Screenshot unavailable: Codeable project description form -->

ឆ្លើយសំណួរខាងក្រោមដើម្បីជួយក្នុងការកំណត់តម្លៃ (pricing)។

<!-- Screenshot unavailable: Codeable project pricing questions -->

បន្ទាប់មក បើអ្នកមិនទាន់មានគណនី [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) ទេ សូមបង្កើតមួយ ហើយចុចលើប៊ូតុង **Publish My Project**។

<!-- រូបភាពមិនអាចបង្ហាញបាន៖ ការបង្កើតគណនី Codeable និងប៊ូតុង Publish My Project -->

នៅពេលអ្នកបានបង្ហោះ project [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) របស់អ្នកហើយ វេទិកានឹងភ្ជាប់អ្នកជាមួយអ្នកអភិវឌ្ឍន៍ដែលត្រូវនឹងជំនាញដែលត្រូវការសម្រាប់ project របស់អ្នក។

ដំណើរការនេះនឹងជួយក្នុងការស្វែងរកមនុស្សត្រឹមត្រូវសម្រាប់ வேலរបស់អ្នក ដែលមានបទពិសោធន៍ជាមួយ WordPress និង Ultimate Multisite!

អត្រាមínimum របស់ Codeableគឺ $70–$120 USD ក្នុងមួយម៉ោង។ ការប៉ាន់ស្មាននឹងផ្លាស់ប្តូរទៅតាមវិសាលភាពការងារ ភាពស្មុគស្មាញ និងភាពបន្ទាន់នៃ project។

នៅពេលអ្នកទទួលបានការប៉ាន់ស្មាន អ្នកនឹងអាចជួលអ្នកអភិវឌ្ឍន៍ដែលសមស្របបំផុតនឹងតម្រូវការរបស់អ្នក ហើយចាប់ផ្តើម project ជាមួយគ្នាបាន!
