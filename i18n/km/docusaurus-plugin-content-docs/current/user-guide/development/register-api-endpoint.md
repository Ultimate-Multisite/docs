# бһ…бҹҶбһҺбһ»бһ…бһ”бһүбҹ’бһ…бһ”бҹӢ API бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Ultimate Multisite

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһӮбҹ’бһӮбһ»бһ‘бҹ’бһ‘бҹҒбһҹбһҖбҹҚбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бһ–бһёбһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ endpoint API бһҡбһ”бһҹбҹӢ Ultimate Multisite /register бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ (onboarding process) бһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһҸбһ·бһҗбһ·бһҮбһ“бһҗбҹ’бһҳбһёбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҺбҹ’бһҸбһ¶бһүбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ“бһ·бһ„бһҡбһ”бҹҖбһ”бһ’бҹ’бһңбһҫбһңбһ¶бһҮбһ¶бһҳбһҪбһҷ Zapierбҹ”

Endpoint бһ“бҹҒбҹҮбһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ POST бһ бһҫбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ…бһҠбҹ„бһҷ URL _**https://yoursite.com/wp-json/wu/v2/register**_бҹ” бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ бҹ…бһ“бҹҒбҹҮ бһҳбһ¶бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ…бҹҶбһ“бһҪбһ“ бҹӨ бһҷбҹүбһ¶бһ„бһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҺбҹ’бһҸбһ¶бһүбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–

  * бһ”бһ„бҹ’бһҖбһҫбһҸбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ WordPress бһҗбҹ’бһҳбһё бһ¬бһҖбҹҶбһҺбһҸбҹӢбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺбһҸбһ¶бһҳбһҡбһҷбҹҲ user IDбҹ”
  * бһ”бһ„бҹ’бһҖбһҫбһҸ Customer бһҗбҹ’бһҳбһёбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Ultimate Multisite бһ¬бһҖбҹҶбһҺбһҸбҹӢбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺбһҸбһ¶бһҳбһҡбһҷбҹҲ customer IDбҹ”
  * бһ”бһ„бҹ’бһҖбһҫбһҸбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҗбҹ’бһҳбһёбһ“бҹ…бһӣбһҫбһ”бһҺбҹ’бһҸбһ¶бһү WordPressбҹ”
  * бһ“бҹ…бһ‘бһёбһ”бҹҶбһ•бһ»бһҸ бһ”бһ„бҹ’бһҖбһҫбһҸ Membership бһҗбҹ’бһҳбһёбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Ultimate Multisiteбҹ”

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ API credentials бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһңбһ¶ бһҹбһјбһҳбһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢ network admin panel бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ‘бҹ…бһҖбһ¶бһ“бҹӢ **Ultimate Multisite > Settings** > **API & Webhooks,** бһ бһҫбһҷбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ•бҹ’бһ“бҹӮбһҖ API Settingsбҹ”

![API Settings section in Ultimate Multisite](/img/config/settings-api.png)

бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ–бҹҒбһүбһӣбҹҒбһүбһ“бҹғбһ‘бҹҶбһ–бҹҗбһҡбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ APIбҹ–

![API settings full page](/img/config/settings-api-full.png)

бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ **Enable API** бһ бһҫбһҷбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“ API credentials бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһҹбһјбһҳбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ–бһё endpoint бһҡбһҪбһ…бһ”бһ„бҹ’бһҖбһҫбһҸ action бһҖбһ¶бһҡбһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Zapierбҹ”

## бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ body бһ“бҹғ Endpoint {#endpoint-body-parameters}

бһҹбһјбһҳбһҳбһҫбһӣбһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһҡбһҪбһҳбһ“бҹғбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбһ”бҹ’бһ”бһ”бһҡбһҳбһ¶бһҠбҹӮбһӣбһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһ”бһүбҹ’бһҮбһјбһ“бһ‘бҹ…бһҖбһ¶бһ“бҹӢ endpointбҹ” бһ“бҹ…бһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһ“бҹғбһўбһҸбҹ’бһҗбһ”бһ‘бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһғбһҫбһүбһҖбһ¶бһҡбһ бҹ… (call) бһ–бҹҒбһүбһӣбҹҒбһүбҹ”

бһўбҹ’бһ“бһҖбһҠбһ№бһ„бһ–бһёбһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһ“бһ·бһ„бһҡбһёбһҖбһҡбһ¶бһҷбһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһңбһёбһӮбҹ’бһҡбһ”бҹӢбһҷбҹүбһ¶бһ„бһҠбҹӮбһӣбһҖбҹҶбһҺбҹӮбһҗбҹ’бһҳбһёбһ“бҹғ Zapier бһ•бҹ’бһҸбһӣбҹӢбһҮбһјбһ“бһ‘бҹҒ? бһҡбҹҖбһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбҹ” (link?)

### бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ action {#creating-an-action}

бһҠбһҫбһҳбҹ’бһ”бһёбһ–бһ“бҹ’бһҷбһӣбҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫ registration endpoint бһҮбһ¶бһҳбһҪбһҷ Zapier бһ”бһ¶бһ“бһҖбһ¶бһ“бҹӢбһҸбҹӮбһ…бҹ’бһ”бһ¶бһҹбҹӢ бһҷбһҫбһ„бһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷ Google Formsбҹ” бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһҠбҹӮбһӣбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹҶбһ–бҹҒбһү бһ бһҫбһҷбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһҹбһ“бҹ’бһӣбһ№бһҖбһ…бһҳбҹ’бһӣбһҫбһҷбһ“бҹғбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ“бҹ„бҹҮ бһҖбһ¶бһҡбһҮбһ¶бһҹбһҳбһ¶бһҮбһ·бһҖбһҗбҹ’бһҳбһёбһҳбһҪбһҷбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҺбҹ’бһҸбһ¶бһү Ultimate Multisiteбҹ”

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Google Forms бһҹбһјбһҳбһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бһҳбҹ’бһҡбһ„бҹӢбһҠбҹӮбһӣбһҳбһ¶бһ“бһңбһ¶бһӣбһҸбһ·бһ…бһ”бҹҶбһ•бһ»бһҸбһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһҳбһ¶бһҮбһ·бһҖбһҗбҹ’бһҳбһёбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҺбҹ’бһҸбһ¶бһүбҹ”

<!-- бһҡбһјбһ”бһ—бһ¶бһ–бһҳбһ·бһ“бһўбһ¶бһ…бһ”бһ„бҹ’бһ бһ¶бһүбһ”бһ¶бһ“бһ‘бҹҒбҹ– бһ‘бһҳбҹ’бһҡбһ„бҹӢ Google Forms бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһңбһ¶бһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһҳбһ¶бһҮбһ·бһҖбһҗбҹ’бһҳбһё -->

бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Zapier бһҹбһјбһҳбһ”бһ„бҹ’бһҖбһҫбһҸ Zap бһҗбҹ’бһҳбһё бһ бһҫбһҷбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бһҳбҹ’бһҡбһ„бҹӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбҹ’бһ“бһ»бһ„ Google бһҸбһ¶бһҳбһҡбһҷбҹҲ spreadsheet бһҠбҹӮбһӣбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ”

<!-- бһҡбһјбһ”бһ—бһ¶бһ–бһҳбһ·бһ“бһўбһ¶бһ…бһ”бһ„бҹ’бһ бһ¶бһүбһ”бһ¶бһ“бһ‘бҹҒбҹ– бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ (trigger configuration) бһҡбһ”бһҹбҹӢ Zapier бһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҖбһ¶бһ“бҹӢ spreadsheet бһ“бҹғ Google Forms -->

бһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһ бһҫбһҷ! бһ‘бһҳбҹ’бһҡбһ„бҹӢ Google Forms бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷ Zapier бһ бһҫбһҷбһҸбҹ’бһҡбҹҖбһҳбһҒбҹ’бһӣбһҪбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷбһ”бһҺбҹ’бһҸбһ¶бһүбҹ” бһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһҹбһјбһҳбһ”бһ“бҹ’бһҸбһ‘бҹ…бһ•бҹ’бһ“бҹӮбһҖ Action бһҠбҹӮбһӣбһ“бһ№бһ„бһҖбһҫбһҸбһЎбһҫбһ„бһ–бһё Trigger бһҠбҹӮбһӣ Google Forms бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһҠбҹӮбһӣбһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹҶбһ–бҹҒбһүбҹ”

бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Ultimate Multisite бһҗбҹ’бһҳбһё бһ бһҫбһҷбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһңбһ¶бҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘ Zap бһ“бҹҒбҹҮ бһҹбһјбһҳбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҮбһҳбҹ’бһҡбһҫбһҹ Register (бһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮ)бҹ”

<!-- бһҡбһјбһ”бһ—бһ¶бһ–бһҳбһ·бһ“бһўбһ¶бһ…бһ”бһ„бҹ’бһ бһ¶бһүбһ”бһ¶бһ“бһ‘бҹҒбҹ– бһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ Action бһҡбһ”бһҹбҹӢ Zapier бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Ultimate Multisite бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҮбһҳбҹ’бһҡбһҫбһҹ Register -->

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҮбҹҶбһ бһ¶бһ“бһҠбҹҶбһ”бһјбһ„бһ“бҹҒбҹҮ бһҹбһјбһҳбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһӮбһҺбһ“бһёбһҠбҹӮбһӣбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷ Zap аёҷаёөа№үбҹ”<!-- бһҡбһјбһ”бһ—бһ¶бһ–бһҳбһ·бһ“бһўбһ¶бһ…бһ”бһ„бҹ’бһ бһ¶бһүбһ”бһ¶бһ“бһ‘бҹҒбҹ– бһҮбҹҶбһ бһ¶бһ“бһҖбһ¶бһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢбһӮбһҺбһ“бһёбһҡбһ”бһҹбҹӢ Zapier бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Ultimate Multisite -->

бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҠбҹӮбһӣбһҡбһҹбһҫбһ”бһ”бҹҶбһ•бһ»бһҸбһ“бҹғбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбҹ” бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһңбһ¶бһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһҳбһҖбһ–бһё Google Forms бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһңбһ¶бһӣбһўбһ”бҹ’бһ”бһ”бһҡбһҳбһ¶бһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡ endpoint register бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ“бҹӮбһҖбһҳбһ»бһ“бһ“бҹғбһўбһҸбҹ’бһҗбһ”бһ‘бһ“бҹҒбҹҮбҹ”

бһҖбҹ’бһ“бһ»бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮ бһҷбһҫбһ„бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ username, email, password, name бһ“бһ·бһ„ URL бһ“бҹғбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ” бһ…бҹҶбһҺбҹӮбһҖбһҜбһ…бҹҶбһҺбһ»бһ…бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһҪбһ…бһ бһҫбһҷ бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷбһҹбһҳбһ¶бһҮбһ·бһҖбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһ“бҹ…бһӣбһҫ Google Forms бһ“бҹҒбҹҮбһҳбһ¶бһ“аёЈаё№аёӣа№Ғаёҡаёҡбһ•бһӣбһ·бһҸбһ•бһӣ бһ“бһ·бһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ”
