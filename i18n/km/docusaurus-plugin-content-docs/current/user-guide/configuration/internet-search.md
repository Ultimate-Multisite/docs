# ការស្វែងរកតាមអ៊ីនធឺណិត

Gratis AI Agent v1.5.0 បានបន្ថែមសមត្ថភាព **Internet Search** ដែលអនុញ្ញាតឱ្យជំនួយការ AI អាចទាញយកព័ត៌មានថ្មីៗពីបណ្តាញអ៊ីនធឺណិតក្នុងអំឡុងពេលនៃការសន្ទនា។ វាដំណើរការដោយប្រើ [Brave Search API](https://brave.com/search/api/) ឬ [Tavily API](https://tavily.com/)។

## របៀបដែលវាដំណើរការ {#how-it-works}

នៅពេលបើក Internet search អ្នកជំនួយការអាចធ្វើការសាកសួរទៅកាន់អ្នកផ្តល់ការស្វែងរកដែលបានកំណត់របស់អ្នកដោយស្វ័យប្រវត្តិ នៅពេលដែលវាសម្រេចថាសំណួរនោះទាមទារព័ត៌មានថ្មីៗ ឬព័ត៌មានពីខាងក្រៅ — ឧទាហរណ៍ ព័ត៌មានថ្មីៗ តម្លៃផ្ទាល់ ឬឯកសារដែលប្រហែលជាផ្លាស់ប្តូរចាប់តាំងពីពេលវេលាដែលម៉ូដែលត្រូវបានបណ្តុះបណ្តាល។

លទ្ធផលនឹងត្រូវបានទាញយកក្នុងពេលវេលាជាក់ស្តែង ហើយបញ្ចូលទៅក្នុងបរិបទរបស់ជំនួយការ មុនពេលវាបង្កើតចម្លើយ។ ជំនួយការនឹងបង្ហាញនៅពេលណាដែលវាបានប្រើលទ្ធផលនៃការស្វែងរកដើម្បីឆ្លើយសំណួរមួយ។

## ការបើកដំណើរការ Internet Search {#enabling-internet-search}

Internet search ទាមទារ API key ពីអ្នកផ្តល់ការស្វែងរកដែលអ្នកជ្រើសរើស។ ដើម្បីកំណត់រចនាសម្ព័ន្ធ៖

1. ចូលទៅកាន់ **Gratis AI Agent → Settings → Advanced**។
2. រកមើលបញ្ជីស្រទាប់ចុះឈ្មោះ **Internet Search Provider** ហើយជ្រើសរើស **Brave Search** ឬ **Tavily**។
3. បញ្ចូល API key របស់អ្នកក្នុងវាលដែលត្រូវគ្នា។ តំណភ្ជាប់ចុះឈ្មោះនឹងបង្ហាញនៅជាប់នឹងវាលនីមួយៗ។
4. ចុចលើ **Save Settings**។

នៅពេលរក្សាទុក key រួច សមត្ថភាព Internet Search នឹងមានសម្រាប់ជំនួយការដោយស្វ័យប្រវត្តិ។

## Brave Search {#brave-search}

### ការទទួលបាន Brave Search API Key {#obtaining-a-brave-search-api-key}

1. ចូលទៅកាន់ [Brave Search API page](https://brave.com/search/api/)។
2. ចុះឈ្មោះសម្រាប់ផែនការមួយ។ មានកម្រិតឥតគិតថ្លៃដែលមានដែនកំណត់សំណើប្រចាំខែ។
3. កូពី API key របស់អ្នកពី developer dashboard របស់ Brave Search។
4. បិទភ្ជាប់វាទៅក្នុងវាល **Brave Search API Key** នៅក្នុងការកំណត់រចនាសម្ព័ន្ធ Gratis AI Agent។

### កម្រិតនៃការប្រើប្រាស់ {#usage-limits}

ការប្រើប្រាស់ត្រូវបានគិតថ្លៃដោយ Brave Search ដោយផ្អែកលើចំនួនសំណួរដែលបានធ្វើ។ ការឆ្លើយតប AI នីមួយៗដែលបង្កឱ្យមានការស្វែងរក នឹងត្រូវបានរាប់ជាសំណួរមួយ។ សូមតាមដានការប្រើប្រាស់របស់អ្នកនៅក្នុង [Brave Search developer dashboard](https://brave.com/search/api/) ដើម្បីជៀសវាងការគិតថ្លៃមិនបានរំពឹងទុក។

## Tavily {#tavily}

Superdav AI Agent v1.10.0 បានបន្ថែម **Tavily** ជាអ្នកផ្តល់សេវាស្វែងរកអ៊ីនធឺណិតជំនួសមួយ ដែលផ្តល់នូវលទ្ធផលស្វែងរកកាន់តែសម្បូរបែប និងសមត្ថភាពស្រាវជ្រាវขั้นสูง។

### ការទទួលបាន Tavily API Key {#obtaining-a-tavily-api-key}

1. ចូលទៅកាន់ [Tavily API page](https://tavily.com/)។
2. ចុះឈ្មោះសម្រាប់គណនីមួយ។ មានកម្រិតឥតគិតថ្លៃដែលមានដែនកំណត់សំណួរប្រចាំខែ។
3. កូពី API key របស់អ្នកពី Tavily dashboard។
4. បិទភ្ជាប់វាទៅក្នុងវាល **Tavily API Key** នៅក្នុងការកំណត់នៃ Gratis AI Agent។

### កម្រិតនៃការប្រើប្រាស់ {#usage-limits-1}

ការប្រើប្រាស់ត្រូវបានគិតថ្លៃដោយ Tavily ដោយផ្អែកលើចំនួន API calls ដែលបានធ្វើ។ ការឆ្លើយតប AI នីមួយៗដែលបង្កឱ្យមានការស្វែងរក នឹងត្រូវបានរាប់ជា one call។ សូមតាមដានការប្រើប្រាស់របស់អ្នកនៅក្នុង [Tavily dashboard](https://tavily.com/) ដើម្បីជៀសវាងការគិតថ្លៃមិនបានរំពឹងទុក។

## ការបិទការស្វែងរកអ៊ីនធឺណិត {#disabling-internet-search}

លុប API key ចេញពីវាលអ្នកផ្តល់សេវាស្វែងរកដែលកំពុងដំណើរការ ហើយរក្សាទុក (save)។ សមត្ថភាពស្វែងរកអ៊ីនធឺណិតនឹងមិនត្រូវបានផ្តល់ជូនដល់ assistant ទៀតទេ។

:::note
ការស្វែងរកអ៊ីនធឺណិតបន្ថែមការពន្យារពេលដល់ការឆ្លើយតប ព្រោះ assistant ត្រូវរង់ចាំលទ្ធផលស្វែងរកមុននឹងបង្កើតចម្លើយ។ សម្រាប់ករណីដែលត្រូវការពេលវេលាជាក់លាក់ សូមពិចារណាថាតើការស្វែងរកពេលវេលាជាក់ស្តែង (real-time search) មានចាំបាច់ឬទេ ឬតើចំណេះដឹងដែលមានស្រាប់របស់ assistant គឺគ្រប់គ្រាន់ហើយឬនៅ។
:::
