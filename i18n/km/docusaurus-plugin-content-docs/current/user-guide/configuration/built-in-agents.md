# ភ្នាក់ងារដែលភ្ជាប់មកស្រាប់

Gratis AI Agent v1.9.0 មានភ្នាក់ងារភ្ជាប់មកស្រាប់ចំនួនប្រាំ។ ភ្នាក់ងារនីមួយៗត្រូវបានកំណត់រចនាសម្ព័ន្ធជាមុនជាមួយសំណុំឧបករណ៍ដែលមានគោលដៅច្បាស់លាស់ system prompt ដែលបានសម្របសម្រួល និងសំណើចាប់ផ្ដើមដែលសមនឹងភារកិច្ចទូទៅក្នុងផ្នែកនោះ។ ការប្ដូររវាងភ្នាក់ងារនឹងផ្លាស់ប្ដូរអ្វីដែលជំនួយការអាចធ្វើបាន និងរបៀបដែលវាឆ្លើយតប — ដោយអ្នកមិនចាំបាច់កំណត់រចនាសម្ព័ន្ធអ្វីឡើយ។ Superdav AI Agent v1.18.0 អាចបន្ថែមឧបករណ៍ដែលដឹងពីកាលវិភាគ កំណត់ត្រារំលឹក ច្រកអនុម័ត និងការជូនដំណឹងតាម SMS ទៅក្នុងលំហូរការងារទាំងនេះ នៅពេលការភ្ជាប់ពាក់ព័ន្ធត្រូវបានកំណត់រចនាសម្ព័ន្ធ។

## ភ្នាក់ងារគឺជាអ្វី?

ភ្នាក់ងារនីមួយៗគឺជាប្រវត្តិរូបកំណត់រចនាសម្ព័ន្ធដែលមានឈ្មោះ ដែលរួមបញ្ចូល៖

- **ឧបករណ៍** — សមត្ថភាពដែលភ្នាក់ងារត្រូវបានអនុញ្ញាតឱ្យហៅប្រើ (ឧ. អ្នកសរសេរមាតិកាមានសិទ្ធិចូលប្រើសមត្ថភាពបង្កើតប្រកាស; ស្ទូឌីយោរចនាមានសិទ្ធិចូលប្រើសមត្ថភាព CSS និង theme.json)
- **System prompt** — សេចក្ដីណែនាំដែលកំណត់សំឡេង អាទិភាព និងដែនកំណត់របស់ភ្នាក់ងារ
- **សំណើ** — prompt ដែលបានសរសេរជាមុន បង្ហាញក្នុងផ្ទាំងជជែក ដើម្បីជួយឱ្យអ្នកចាប់ផ្ដើមបានលឿន

## ការចូលប្រើឧបករណ៍ជ្រើសរើសភ្នាក់ងារ

1. បើកផ្ទាំង **Gratis AI Agent** នៅក្នុងរបារចំហៀងគ្រប់គ្រង WordPress។
2. ចុច **រូបតំណាងភ្នាក់ងារ** នៅផ្នែកខាងឆ្វេងលើនៃក្បាលជជែក (រូបតំណាងផ្លាស់ប្ដូរដើម្បីបង្ហាញភ្នាក់ងារដែលកំពុងសកម្ម)។
3. **ឧបករណ៍ជ្រើសរើសភ្នាក់ងារ** បើកជាស្រទាប់លើប្រភេទតារាងទម្រង់។ ភ្នាក់ងារនីមួយៗត្រូវបានរាយជាមួយរូបតំណាង ឈ្មោះ និងសេចក្ដីពិពណ៌នាមួយបន្ទាត់។
4. ចុចជួរភ្នាក់ងារ ដើម្បីធ្វើឱ្យវាសកម្ម។ ក្បាលជជែកធ្វើបច្ចុប្បន្នភាពភ្លាមៗ។

អ្នកក៏អាចប្ដូរភ្នាក់ងារនៅកណ្ដាលការសន្ទនាបានដែរ — system prompt របស់ភ្នាក់ងារថ្មីនឹងមានប្រសិទ្ធភាពចាប់ពីសារបន្ទាប់។

## ការកែសម្រួល Agent

agent ដែលមានស្រាប់នីមួយៗអាចត្រូវបានពង្រីក ឬជំនួសតាមរយៈ filter `gratis_ai_agent_agents`។

### បន្ថែម system prompt ផ្ទាល់ខ្លួនទៅ agent ដែលមានស្រាប់

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### ចុះបញ្ជី agent ថ្មី

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

agent ថ្មីបង្ហាញនៅក្នុង Agent Picker ភ្លាមៗបន្ទាប់ពី filter ដំណើរការ។

add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
