# ការគ្រប់គ្រងតំណភ្ជាប់ដែលអាចចែករំលែកបានសម្រាប់ផែនការ (Plans) {#managing-shareable-links-for-plans}

Ultimate Multisite ផ្តល់ជូនអ្នកនូវជម្រើសមួយដើម្បីចែករំលែកតំណភ្ជាប់ផែនការដោយផ្ទាល់ទៅកាន់អតិថិជនរបស់អ្នក។ ដើម្បីធ្វើដូចនេះ សូមចូលទៅកាន់ទំព័រแก้ไข (edit pages) នៃផែនការណាមួយរបស់អ្នក ហើយចុចលើប៊ូតុង _Click to copy the Shareable Link_។

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_នៅលើទំព័រแก้ไขផែនការ អ្នកអាចរកឃើញប៊ូតុងសម្រាប់បង្កើតតំណភ្ជាប់ដែលអាចចែករំលែកបាន_

សូមចំណាំថា យន្តការនេះប្រើប្រាស់ភាពញឹកញាប់នៃការគិតថ្លៃ (billing frequency) តាមលំនាំដើមដើម្បីបង្កើតតំណភ្ជាប់។ មានន័យថា ប្រសិនបើការគិតថ្លៃលំនាំដើមរបស់អ្នកគឺប្រចាំខែ ឧទាហរណ៍ នៅទីបំផុតអ្នកនឹងទទួលបានតំណភ្ជាប់ដូចនេះ៖ [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

ប្រសិនបើអ្នកចង់ឱ្យតំណភ្ជាប់ដែលអាចចែករំលែកបានកំណត់ជាប្រចាំត្រីមាស (quarterly) អ្នកនឹងត្រូវបន្ថែមលេខ 3 នៅខាងចុង៖

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

ដើម្បីជ្រើសរើសផែនការ _premium_ តាមកាលវិភាគប្រចាំឆ្នាំ អ្នកនឹងបន្ថែមលេខ 1 រួចបន្តដោយពាក្យ៖

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

សូមចងចាំថា អ្នកមិនអាចមានតំណភ្ជាប់តែមួយសម្រាប់ភាពញឹកញាប់នៃការគិតថ្លៃផ្សេងៗគ្នាបានទេ។ អ្នកគួរតែចែករំលែកតំណភ្ជាប់ជាក់លាក់មួយសម្រាប់ភាពញឹកញាប់នៃការគិតថ្លៃនីមួយៗរបស់អ្នក។

នៅពេលអ្នកចែករំលែកតំណភ្ជាប់ជាមួយអតិថិជនរបស់អ្នក វានឹងបញ្ជូនពួកគេទៅជំហានដំបូងនៃដំណើរការចុះឈ្មោះ (signup flow) របស់អ្នក។
