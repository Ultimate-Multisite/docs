# Tài Liệu Ultimate Multisite

Chào mừng đến với tài liệu Ultimate Multisite. Ultimate Multisite biến WordPress Multisite thành một nền tảng Website as a Service (WaaS) hoàn chỉnh.

## Các Phần Tài Liệu {#documentation-sections}

### [Hướng dẫn sử dụng](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

Mọi thứ bạn cần để thiết lập, cấu hình và quản lý nền tảng WaaS của mình. Bao gồm cài đặt, cổng thanh toán (payment gateways), ánh xạ tên miền (domain mapping), quản lý khách hàng (client management), và nhiều hơn thế nữa.

### [Tài liệu dành cho Nhà phát triển](/developer/hooks) {#developer-documentation}

Tài liệu tham khảo kỹ thuật dành cho các nhà phát triển xây dựng trên Ultimate Multisite. Bao gồm tài liệu tham khảo về hooks, tài liệu REST API, và hướng dẫn phát triển addon.

### [Tiện ích mở rộng](/addons) {#addons}

Tài liệu cho các plugin addon trả phí giúp mở rộng Ultimate Multisite với các tính năng bổ sung như tích hợp WooCommerce, phân tích (analytics), bán tên miền (domain selling), và nhiều hơn thế nữa.
