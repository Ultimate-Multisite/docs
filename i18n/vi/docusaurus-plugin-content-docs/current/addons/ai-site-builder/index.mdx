# AI Site Builder

ultimate-multisite-ai-site-builder

## Yêu cầu {#requirements}

- WordPress 5.3 trở lên
- PHP 7.4 trở lên
- Plugin Ultimate Multisite (đang kích hoạt)

## Cài đặt {#installation}

1. Tải các tệp addon lên thư mục `/wp-content/plugins/` của bạn
2. Kích hoạt plugin qua menu 'Plugins' trong WordPress
3. Cấu hình các cài đặt addon trong khu vực quản trị mạng (network admin)

## Phát triển {#development}

### Thiết lập {#setup}

```bash
# Cài đặt các dependencies
composer install
npm install

# Chạy các bài kiểm thử
npm run test

# Chạy kiểm tra tiêu chuẩn mã nguồn
vendor/bin/phpcs
vendor/bin/phpstan

# Sửa các vấn đề về phong cách mã
vendor/bin/phpcbf

# Xây dựng cho môi trường sản xuất
npm run build
```

### Kiểm thử {#testing}

Addon bao gồm một bộ khung kiểm thử toàn diện:

- **Kiểm thử đơn vị (Unit Tests)**: Các bài kiểm thử dựa trên PHPUnit cùng bộ kiểm thử của WordPress.
- **Lớp kiểm thử cơ sở (Base Test Class)**: Cung cấp các phương thức hỗ trợ cho các kịch bản kiểm thử chung.
- **Độ bao phủ kiểm thử (Test Coverage)**: Được cấu hình để theo dõi độ bao phủ mã nguồn.
- **Kiểm thử Multisite**: Các bài kiểm thử chạy trong môi trường multisite.

### Tiêu chuẩn mã nguồn {#code-standards}

- **Tiêu chuẩn mã nguồn WordPress**: Được thực thi bằng PHPCS.
- **Tương thích PHP 7.4+**: Hỗ trợ các tính năng PHP hiện đại.
- **Phân tích tĩnh (Static Analysis)**: Tích hợp PHPStan để kiểm tra kiểu dữ liệu.
- **Hiện đại hóa mã nguồn (Code Modernization)**: Các quy tắc Rector để nâng cấp PHP.

### Hệ thống xây dựng (Build System) {#build-system}

- **Xử lý tài sản (Asset Processing)**: Thu gọn (Minification) cho các tệp CSS/JS.
- **Dịch thuật (Translation)**: Tạo tệp POT.
- **Đóng gói (Packaging)**: Tạo kho lưu trữ (archive) để phân phối.
- **Chế độ phát triển (Development Mode)**: Tài sản chưa thu gọn để gỡ lỗi.

## Cấu hình {#configuration}

Addon hỗ trợ nhiều tùy chọn cấu hình khác nhau, có thể truy cập qua giao diện quản trị mạng (network admin).

## Hooks và Filters {#hooks-and-filters}

### Actions {#actions}

- `ultimate-multisite-ai-site-builder_init` - Được kích hoạt sau khi khởi tạo addon.
- `ultimate-multisite-ai-site-builder_loaded` - Được kích hoạt sau khi tất cả các tệp addon được tải.

### Filters {#filters}

- `ultimate-multisite-ai-site-builder_settings` - Sửa đổi cài đặt addon.
- `ultimate-multisite-ai-site-builder_enabled` - Ghi đè trạng thái bật/tắt của addon.
