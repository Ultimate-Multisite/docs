# Bộ chọn Ngôn ngữ

Ultimate Multisite: Bộ chọn Ngôn ngữ trên trang Đăng ký

Thêm trường chọn ngôn ngữ vào quy trình đăng ký.

## Cài đặt {#installation}

1. Tải 'ultimate-multisite-language-selector' lên thư mục '/wp-content/plugins/'
2. Kích hoạt plugin này thông qua menu 'Plugins' trong WordPress Network Admin của bạn.

## Hướng dẫn sử dụng {#usage-guide}

Với add-on miễn phí này, bạn có thể cho phép người dùng của mình chọn ngôn ngữ mà họ muốn xem trên quy trình đăng ký và trên trang đăng nhập.

Sau khi [kích hoạt giấy phép của bạn](../../user-guide/getting-started/installing-ultimate-multisite.md), bạn sẽ có thể cài đặt các add-on Ultimate Multisite mà giấy phép của bạn cho phép truy cập.

Truy cập trang cài đặt Ultimate Multisite của bạn, sau đó, trên menu bên trái, nhấp vào nhãn _Add-ons_.

![Add-ons page](/img/admin/addons-page.png)

_Truy cập trang Add-ons để bắt đầu cài đặt Bộ chọn Ngôn ngữ_

Tại đó, bạn sẽ tìm thấy tất cả các add-on mà chúng tôi cung cấp.

Các add-on miễn phí có sẵn cho tất cả khách hàng của chúng tôi, bất kể họ đăng ký gói dịch vụ nào.

Nhấp vào nút _Install it Now!_ bên dưới ảnh thu nhỏ của add-on. Sau đó, kích hoạt nó trên trang Plugins.

![Add-ons page with install button](/img/admin/addons-page.png)

_Nhấp vào nút này để cài đặt add-on_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_Kích hoạt nó trên trang Plugins của bạn_

Bây giờ, hãy vào menu Cài đặt (Settings) của Ultimate Multisite và chọn tab _Network Settings_. Ở đó, ở cuối trang, bạn sẽ thấy một mục mới tên là _Languages_.

![Settings page](/img/config/settings-general.png)

_Một mục hoàn toàn mới đã xuất hiện!_

Bạn có thể để trống để hiển thị tất cả các ngôn ngữ dưới dạng tùy chọn, hoặc chọn những ngôn ngữ bạn muốn kích hoạt. Các ngôn ngữ này là những ngôn ngữ bạn đã cài đặt trong hệ thống WordPress của mình.

## Thiết lập Ngôn ngữ qua URL {#setting-the-language-via-the-url}

Bạn cũng có thể thay đổi ngôn ngữ của một trang bằng cách bỏ qua tham số locale trong URL. Ví dụ, nếu bạn muốn chuyển hướng đến trang đăng ký bằng tiếng Ý, hãy thêm ?locale=it_IT vào URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Ngôn ngữ đó phải được cài đặt trên mạng WordPress của bạn thì tính năng này mới hoạt động.
