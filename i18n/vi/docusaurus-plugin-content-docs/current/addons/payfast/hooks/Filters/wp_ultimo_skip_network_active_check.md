# Bộ lọc: wp_ultimo_skip_network_active_check {#filter-wpultimoskipnetworkactivecheck}

Cho phép các nhà phát triển bỏ qua (short-circuit) việc kiểm tra này.

Điều này rất hữu ích khi bạn sử dụng các thiết lập tùy chỉnh dựa trên composer và các loại thiết lập khác, ví dụ như Bedrock, nơi việc sử dụng plugin dưới dạng mu-plugins là tiêu chuẩn.

- 2.0.0
