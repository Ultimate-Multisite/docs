# Filter: wp_ultimo_skip_network_active_check

Cho phép các nhà phát triển bỏ qua (short-circuit) việc kiểm tra này.

Điều này rất hữu ích khi bạn đang sử dụng các thiết lập tùy chỉnh khác, chẳng hạn như các thiết lập dựa trên composer, ví dụ như Bedrock, nơi việc sử dụng các plugin dưới dạng mu-plugins là tiêu chuẩn.
