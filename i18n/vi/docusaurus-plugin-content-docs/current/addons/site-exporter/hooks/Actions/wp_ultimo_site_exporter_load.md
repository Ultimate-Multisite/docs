# Action: wp_ultimo_site_exporter_load

Kích hoạt khi tất cả các thành phần phụ thuộc đã được tải xong.

Cho phép các nhà phát triển plugin thêm các chức năng mới. Ví dụ: hỗ trợ các nhà cung cấp Hosting mới, v.v.
