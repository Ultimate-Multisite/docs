# Nhật ký thay đổi của Plugin & Theme Manager

Version: 2.0.8 - Phát hành ngày 2026-05-05
* Cải tiến: Danh sách plugin hiện sử dụng bộ nhớ đệm (cache) của trang web để giảm các truy vấn cơ sở dữ liệu và cải thiện hiệu suất.
* Cải tiến: Bộ tải tự động (autoloader) của plugin sẽ bỏ qua khi bộ tải tự động gốc của Bedrock đã tải các dependencies (phụ thuộc) rồi.

Version: 2.0.7 - Phát hành ngày 2026-01-01
* Cải tiến: Các bản cập nhật về khả năng tương thích.
