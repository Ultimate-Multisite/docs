# Filter: wp_ultimo_skip_network_active_check {#filter-wpultimoskipnetworkactivecheck}

Cho phép các nhà phát triển bỏ qua (short-circuit) việc kiểm tra này.

Điều này rất hữu ích khi bạn sử dụng các thiết lập tùy chỉnh dựa trên Composer và các cấu hình khác, chẳng hạn như Bedrock, nơi việc sử dụng các plugin dưới dạng mu-plugins là quy chuẩn.
