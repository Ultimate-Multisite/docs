# Plugin & Theme Manager

Ultimate Multisite: Plugin and Theme Manager

Cho phép bạn chỉnh sửa hoặc ẩn các thông tin chi tiết (tiêu đề, tác giả, ảnh đại diện và mô tả) của các plugin và theme đã cài đặt. Bạn có thể tạo danh mục cho các plugin và theme để người dùng lọc và thêm một trang Plugins tùy chỉnh đẹp mắt vào bảng điều khiển người dùng của họ.

Để sử dụng plugin này, bạn chỉ cần truy cập trang Plugins hoặc Themes trên khu vực quản trị mạng (Network Admin) và nhấp vào "Edit Plugin/Theme info" (Chỉnh sửa thông tin Plugin/Theme).

## Installation

1. Tải 'ultimate-multisite-plugin-and-theme-manager' lên thư mục '/wp-content/plugins/'
2. Kích hoạt plugin thông qua menu 'Plugins' trong khu vực quản trị mạng WordPress của bạn

## Usage Guide

Ultimate Multisite: Plugin and Theme Manager là một add-on **premium** mạnh mẽ của Ultimate Multisite, giúp bạn kiểm soát hoàn toàn cách người dùng mạng của bạn xem và tương tác với các tùy chọn plugin và theme trên mạng.

Add-on này phục vụ ba mục đích:

  1. Nó cho phép bạn thay đổi hoàn toàn siêu dữ liệu (metadata) của cả plugin và theme. Điều này bao gồm tên, mô tả, tên tác giả, và thậm chí cả ảnh đại diện!

  2. Nó cho phép bạn phân loại các plugin và theme, giúp người dùng của bạn dễ dàng sắp xếp chúng.

  3. Nó cung cấp tùy chọn thay thế trang plugins cũ, nhàm chán bằng một trang mới giống như themes, có khả năng sắp xếp theo danh mục.

Sau khi tải xuống và cài đặt, một tab mới - _Add-on Settings_ sẽ xuất hiện trên trang Cài đặt Ultimate Multisite.
