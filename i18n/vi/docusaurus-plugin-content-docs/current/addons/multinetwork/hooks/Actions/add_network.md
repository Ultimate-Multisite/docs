# Action: add_network

Được kích hoạt sau khi một mạng lưới (network) mới được thêm vào.

## Tham số {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $new_network_id | `int` | ID của mạng lưới vừa được thêm. |
| $r | `array` | Mảng liên kết đầy đủ các đối số của mạng lưới. |

### Từ phiên bản {#since}

- 1.3.0
