# Action: update_network

Được kích hoạt sau khi một mạng lưới hiện có được cập nhật.

## Tham số

| Name | Type | Description |
|------|------|-------------|
| $network_id | `int` | ID của mạng lưới được thêm vào. |
| $args | `array` | Mảng liên kết chứa các đối số của mạng lưới. |

### Từ phiên bản

- 1.3.0
### Nguồn

Được định nghĩa tại [`inc/functions/network.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-multinetwork/blob/main/inc/functions/network.php#L1422) ở dòng 1422
