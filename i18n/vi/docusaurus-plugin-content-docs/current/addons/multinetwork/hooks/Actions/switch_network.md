# Action: switch_network

Được kích hoạt khi ngữ cảnh mạng hiện tại được chuyển đổi.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $new_network_id | `int` | ID của mạng mà đang được chuyển đến. |
| $old_network_id | `int` | ID của mạng trước đó. |

- 1.3.0
