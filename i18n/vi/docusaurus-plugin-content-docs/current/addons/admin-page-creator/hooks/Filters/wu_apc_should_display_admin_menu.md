# Bộ lọc: wu_apc_should_display_admin_menu

Cho phép quản trị viên ẩn các trang menu quản trị nếu họ đã hoàn thành quá trình tùy chỉnh.

## Tham số {#parameters}

| Tên | Kiểu dữ liệu | Mô tả |
|------|------|-------------|
| $should_display_admin_menu | `bool` | Nếu chúng ta cần hiển thị các trang. |
| $network_active | `bool` | Kiểm tra xem plugin có đang hoạt động ở cấp quản trị mạng hay không. |

- 1.4.0
