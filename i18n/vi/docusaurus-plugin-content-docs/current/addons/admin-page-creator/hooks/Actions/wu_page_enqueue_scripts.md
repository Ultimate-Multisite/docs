# Action: wu_page_enqueue_scripts

Thêm một action tùy chỉnh ở đây để chúng ta có nơi để enqueue (tải) các script và style khi cần thiết và đảm bảo chúng chỉ được tải trên các trang phù hợp.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $ | `\WU_Page` | Trang quản trị hiện tại |

### Since {#since}

- 1.3.0
