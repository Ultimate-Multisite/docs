# Action: wu_mt_provider_log

Hành động này được kích hoạt khi một nhà cung cấp (provider) ghi lại một thông báo.

## Tham số {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $message | `string` | Thông báo nhật ký. |
| $level | `string` | Mức độ nhật ký. |
| $context | `array` | Ngữ cảnh bổ sung. |
