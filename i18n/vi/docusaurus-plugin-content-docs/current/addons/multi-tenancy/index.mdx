# Đa thuê bao {#multi-tenancy}

Phân phối các trang con trên nhiều máy chủ và cơ sở dữ liệu để tăng hiệu năng và bảo mật

## Điểm nổi bật của bản phát hành phiên bản 1.2.0 {#version-120-release-highlights}

Multi-Tenancy 1.2.0 bổ sung nền tảng bên thuê độc lập cho các quy trình cô lập trong môi trường sản xuất:

- Tự động đăng nhập bên thuê không trạng thái với các lượt truy cập SSO ngắn hạn, được bảo vệ chống phát lại.
- Cô lập cơ sở dữ liệu và hệ thống tệp theo từng trang con cho các bên thuê độc lập.
- Năng lực đa thuê bao Hostinger để cấp phát bên thuê có nhận biết máy chủ.
- Các cổng xác minh di chuyển độc lập để kiểm tra dữ liệu cũ và xử lý đẩy bất đồng bộ.
- Liên kết máy chủ cơ sở dữ liệu cùng máy có thể cấu hình cho các bản cài đặt Bedrock, FrankenPHP và localhost.
- Cải thiện khởi tạo bên thuê, trình ghi cơ sở dữ liệu, khởi tạo lược đồ và cấp phát người dùng.
- Gỡ bỏ độc lập an toàn hơn, có dọn dẹp thông tin xác thực của bên thuê.

Xem các hướng dẫn người dùng về [quy trình di chuyển độc lập](/user-guide/getting-started/sovereign-migration-workflow), [cô lập đa thuê bao](/user-guide/configuration/multi-tenancy-isolation), và [vận hành bên thuê độc lập](/user-guide/administration/sovereign-tenant-operations).

## Yêu cầu {#requirements}

- WordPress 5.3 trở lên
- PHP 7.4 trở lên
- plugin Multisite Ultimate (đang hoạt động)

## Cài đặt {#installation}

1. Tải các tệp addon lên thư mục `/wp-content/plugins/` của bạn
2. Kích hoạt plugin thông qua menu 'Plugins' trong WordPress
3. Cấu hình cài đặt addon trong khu vực quản trị mạng

## Phát triển {#development}

### Thiết lập {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Kiểm thử {#testing}

Addon bao gồm một khung kiểm thử toàn diện:

- **Kiểm thử đơn vị**: Kiểm thử dựa trên PHPUnit với bộ kiểm thử WordPress
- **Lớp kiểm thử cơ sở**: Cung cấp các phương thức hỗ trợ cho những kịch bản kiểm thử phổ biến
- **Phạm vi kiểm thử**: Được cấu hình để theo dõi độ bao phủ mã
- **Kiểm thử multisite**: Các kiểm thử chạy trong môi trường multisite

### Tiêu chuẩn mã {#code-standards}

- **Tiêu chuẩn mã hóa WordPress**: Được thực thi thông qua PHPCS
- **Tương thích PHP 7.4+**: Hỗ trợ các tính năng PHP hiện đại
- **Phân tích tĩnh**: Tích hợp PHPStan để kiểm tra kiểu
- **Hiện đại hóa mã**: Quy tắc Rector cho các nâng cấp PHP

### Hệ thống build {#build-system}

- **Xử lý tài nguyên**: Thu nhỏ các tệp CSS/JS
- **Dịch thuật**: Tạo tệp POT
- **Đóng gói**: Tạo tệp lưu trữ để phân phối
- **Chế độ phát triển**: Tài nguyên chưa thu nhỏ để gỡ lỗi

## Cấu hình {#configuration}

Addon hỗ trợ nhiều tùy chọn cấu hình có thể truy cập thông qua giao diện quản trị mạng.

### Cấu hình bên thuê độc lập {#sovereign-tenant-configuration}

Đối với các bên thuê độc lập, hãy cấu hình cơ sở dữ liệu bên thuê, thư mục gốc hệ thống tệp, liên kết máy chủ cơ sở dữ liệu và năng lực máy chủ trước khi định tuyến lưu lượng đến bên thuê. Chạy xác minh di chuyển sau mỗi thay đổi ảnh hưởng đến thông tin xác thực cơ sở dữ liệu, liên kết máy chủ, định tuyến hoặc cấp phát người dùng.

Chỉ sử dụng `localhost` khi runtime của bên thuê và quyền cấp cơ sở dữ liệu yêu cầu socket cục bộ hoặc quyền truy cập cùng máy. Sử dụng `127.0.0.1` hoặc tên máy chủ dịch vụ khi đó là máy chủ cấp quyền được nhà cung cấp của bạn cấu hình.

### Năng lực Hostinger {#hostinger-capability}

Năng lực đa thuê bao Hostinger hoạt động với token tích hợp Hostinger dùng chung. Xác nhận tên miền có trong hPanel, DNS trỏ đến Account Hostinger, và cài đặt cơ sở dữ liệu bên thuê khớp với hPanel trước khi xác minh.

## Hook và bộ lọc {#hooks-and-filters}

### Action {#actions}

- `ultimate-multisite-addon-template_init` - Được kích hoạt sau khi khởi tạo addon
- `ultimate-multisite-addon-template_loaded` - Được kích hoạt sau khi tất cả tệp addon đã được tải

### Bộ lọc {#filters}

- `ultimate-multisite-addon-template_settings` - Sửa đổi cài đặt addon
- `ultimate-multisite-addon-template_enabled` - Ghi đè trạng thái bật/tắt addon
