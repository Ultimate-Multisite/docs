# Đồng bộ hóa Nội dung (Content Sync) {#content-sync}

Multisite Ultimate: Content Sync

Addon này dành cho Multisite Ultimate cho phép quản trị viên mạng đồng bộ hóa nội dung từ bất kỳ site nào sang các site khác trong mạng. Nó hỗ trợ tất cả các loại bài viết (post types), bao gồm bài viết (posts), trang (pages), loại bài viết tùy chỉnh (custom post types), và các template của Elementor. Addon bao gồm một giao diện hướng dẫn nhiều bước để chọn nội dung và các site đích, cùng với tính năng xử lý hàng loạt và phản hồi tiến độ theo thời gian thực.

## Tính năng {#features}

* **Chọn Nội dung (Content Selection):** Lưới thẻ trực quan với ảnh thu nhỏ và bộ lọc loại bài viết.
* **Tất cả các Loại Bài viết (All Post Types):** Đồng bộ hóa bài viết, trang, sản phẩm, template Elementor và bất kỳ loại bài viết tùy chỉnh nào.
* **Chọn Site (Site Selection):** Chọn các site nhận nội dung đồng bộ, có khả năng loại trừ site nguồn và chọn trước các site con.
* **Xử lý Hàng loạt (Batch Processing):** Đồng bộ hóa lên 10 site cùng lúc với cập nhật tiến độ.
* **Đồng bộ hóa Media (Media Sync):** Tự động sao chép và ánh xạ lại các tệp media (hình ảnh, video) từ nội dung bài viết sang các site đích.
* **Đồng bộ hóa Taxonomy (Taxonomy Sync):** Bảo toàn các danh mục (categories), thẻ (tags) và các taxonomy tùy chỉnh.
* **Hỗ trợ Elementor:** Hỗ trợ đầy đủ cho các template Elementor với khả năng tái tạo CSS và ánh xạ lại ID template.
* **Giải quyết Xung đột (Conflict Resolution):** Tùy chọn thay thế nội dung hiện có hoặc bỏ qua.
* **Quản lý Cache (Cache Management):** Tự động xóa cache của Elementor trên các site đích sau khi đồng bộ hóa.

## Yêu cầu {#requirements}

* WordPress 5.3 trở lên
* PHP 7.4 trở lên
* Plugin Ultimate Multisite (đang kích hoạt)
* Plugin Elementor (tùy chọn, bắt buộc cho việc đồng bộ hóa template Elementor)

## Cài đặt {#installation}

1. Tải 'ultimate-multisite-content-sync' lên thư mục '/wp-content/plugins/'
2. Kích hoạt plugin thông qua menu 'Plugins' trong WordPress Network Admin của bạn.
3. Cấu hình cài đặt tại Ultimate Multisite → Settings → Content Sync
4. Điều hướng đến Ultimate Multisite → Content Sync để bắt đầu đồng bộ hóa.
