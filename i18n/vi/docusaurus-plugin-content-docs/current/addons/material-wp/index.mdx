# Material WP {#material-wp}

Giao diện quản trị của WordPress đã khá quen thuộc rồi. Nhưng bạn đã bao giờ tự hỏi giao diện admin theo phong cách Material Design sẽ trông ra sao chưa?

Material WP là một theme admin được phát triển để mang lại cho bạn vẻ ngoài và cảm giác đó, giúp bạn cung cấp một Dashboard tùy chỉnh hơn cho khách hàng của mình.

## Features {#features}

- Hỗ trợ CSS tùy chỉnh
- Cài đặt kiểu dáng dễ dàng
- Sắp xếp lại menu
- Nhập/xuất dễ dàng

## Documentation {#documentation}

**Tùy chỉnh Bảng điều khiển Admin của bạn với Material WP**

## Installation {#installation}

1. Tải 'material-wp' lên thư mục '/wp-content/plugins/'
2. Kích hoạt plugin thông qua menu 'Plugins' trong WordPress

## Usage Guide {#usage-guide}

Giao diện quản trị của WordPress đã khá quen thuộc rồi. Nhưng bạn đã bao giờ tự hỏi giao diện admin theo phong cách Material Design sẽ trông ra sao chưa?

Material WP là một theme admin được phát triển để mang lại cho bạn vẻ ngoài và cảm giác đó, giúp bạn cung cấp một Dashboard tùy chỉnh hơn cho khách hàng của mình. Material WP cũng là một phần của bộ plugin cao cấp của Ultimate Multisite.

Sau khi **cài đặt**, hãy truy cập trang _Plugins_ và nhấp để kích hoạt nó trên Network của bạn.
