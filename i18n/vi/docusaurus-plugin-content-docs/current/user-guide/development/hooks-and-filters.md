# Hook và filter của Ultimate Multisite

Ultimate Multisite được xây dựng từ rất nhiều hàm và truy vấn cơ sở dữ liệu. Plugin cung cấp các hook và filter cho phép bạn chỉnh sửa hoặc thêm code mới vào các hàm có sẵn.

Chúng tôi đã tạo một danh sách các hook và filter này để làm tài liệu tham khảo, giúp những người dùng nâng cao có thể tùy biến mạng lưới của mình sâu hơn. Bạn có thể truy cập danh sách này trên Github bằng cách nhấp vào liên kết bên dưới.
