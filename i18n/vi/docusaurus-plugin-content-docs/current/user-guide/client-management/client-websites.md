# Trang web của khách hàng (v2)

Sau khi đăng ký trên mạng lưới của bạn, khách hàng có thể truy cập vào bảng điều khiển của họ.

Ngoài trang **Tài khoản**, nơi họ có thể xem các thông tin quan trọng về thanh toán, gói thành viên, tên miền, giới hạn gói dịch vụ và nhiều thông tin khác, họ còn có thể truy cập trang **Trang web của tôi** để xem và quản lý tất cả các trang web của mình. Họ chỉ cần nhấp vào Trang web của tôi trên bảng điều khiển.

![Liên kết Trang web của tôi trong bảng điều khiển khách hàng](/img/admin/customers-list.png)

Họ sẽ thấy tất cả các trang web của mình, với nút **Quản lý** ngay bên dưới mỗi trang web. Khi nhấp vào nút này, họ sẽ được chuyển đến **bảng điều khiển của trang web đó**.

Họ cũng có thể xem trang web nào là trang web **chính** đối với các gói dịch vụ cho phép họ sở hữu nhiều hơn một trang web.

![Trang Trang web của tôi hiển thị các trang web của khách hàng với các nút Quản lý](/img/admin/customers-list.png)

Nếu muốn thay đổi trang web chính, họ cần vào trang Tài khoản bằng cách nhấp vào menu **Tài khoản**.

![Liên kết menu Tài khoản trong bảng điều khiển khách hàng](/img/admin/customers-list.png)

Sau đó, họ nhấp vào **Thay đổi trang web mặc định**.

![Nút Thay đổi trang web mặc định trên trang tài khoản](/img/admin/customers-list.png)

Một cửa sổ bật lên sẽ xuất hiện. Nhấp vào danh sách thả xuống và chọn trang web bạn muốn đặt làm trang web chính.

![Lựa chọn thả xuống Thay đổi trang web mặc định](/img/admin/customers-list.png)

Sau đó, chỉ cần nhấp vào **Thay đổi trang web mặc định** là xong!

![Xác nhận thay đổi trang web mặc định](/img/admin/customers-list.png)
