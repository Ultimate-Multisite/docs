# Đặt quảng cáo trên mạng lưới của bạn với Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector là một **add-on miễn phí** cung cấp cho bạn các công cụ để đặt quảng cáo trên các gói dịch vụ ngay từ trang chỉnh sửa của chúng.

Sau khi cài đặt Ultimate Multisite: Ad Injector, hãy vào trang _Plugins_ để kích hoạt nó.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Nhấp vào nút Network Activate_

Khi add-on đã được kích hoạt, hộp meta _Advanced Options_ trên màn hình chỉnh sửa gói dịch vụ sẽ có một tùy chọn mới gọi là _Ad Settings_.

![Cài đặt quảng cáo trên màn hình chỉnh sửa gói dịch vụ](/img/config/product-edit-full.png)

_Giờ đây bạn có thể chỉnh sửa cài đặt quảng cáo cho từng gói dịch vụ của mình_

Trong tab đó, bạn có thể thêm mã quảng cáo cho bài viết và trang ở giao diện front-end, trước và sau nội dung.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Sau khi chèn mã quảng cáo, nó sẽ hiển thị trên các bài viết và trang của gói dịch vụ cụ thể đó._

Bạn cũng có thể đặt quảng cáo ở phần header của các trang Admin dành cho người dùng của gói dịch vụ đó.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Vào tùy chọn Admin Panel Ad Code nếu bạn muốn hiển thị quảng cáo ở back-end của người dùng_

### Không chỉ là quảng cáo

Bạn cũng có thể sử dụng Ultimate Multisite: Ad Injector để cung cấp hỗ trợ cho các gói dịch vụ cao cấp.

Chỉ cần vào tùy chọn _Admin Panel Ads_ và thêm mã HTML cho công cụ hỗ trợ của bạn.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Thêm hỗ trợ khách hàng cho các gói dịch vụ cụ thể_

Bằng cách này, mọi người có thể liên hệ với bạn ngay từ Admin Panel của họ!
