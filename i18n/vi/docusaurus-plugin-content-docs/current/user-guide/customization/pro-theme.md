# Tùy chỉnh bảng quản trị với PRO Theme

Khi làm việc với khách hàng, chúng ta thường cần mang đến cho họ một trải nghiệm riêng biệt trong bảng quản trị WordPress. Nếu bạn đang tìm kiếm một giao diện chuyên nghiệp và tinh tế hơn, PRO Theme có thể giúp bạn.

PRO Theme là một theme quản trị được phát triển để hỗ trợ bạn tùy chỉnh Dashboard cho khách hàng. Đây cũng là một phần trong bộ plugin cao cấp của Ultimate Multisite.

Sau khi cài đặt, hãy vào trang _Plugins_ và nhấp kích hoạt trên toàn bộ Network.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Kích hoạt trên toàn Network tại trang Plugins_

_**LƯU Ý:**_ _Nếu bạn chỉ cài đặt mà không kích hoạt trên Network, theme sẽ hiển thị như một tùy chọn mà khách hàng có thể đánh dấu để áp dụng riêng cho tài khoản của họ._

Giờ đây, giao diện quản trị của bạn đã được thiết kế lại hoàn toàn! Bạn cũng sẽ thấy một menu mới ở cột bên trái có tên PRO Theme.

<!-- Screenshot unavailable: PRO Theme redesigned admin panel interface -->

_Diện mạo hoàn toàn mới cho bảng quản trị của bạn_
