# Filter: wu_is_mcp_enabled

Cho phép các nhà phát triển plugin thiết lập (buộc) một trạng thái cụ thể cho adapter MCP.

## Tham số

| Name | Type | Description |
|------|------|-------------|
| $enabled | `bool` | Trạng thái cho biết bộ adapter MCP có được bật hay không. |

### Từ phiên bản

- 2.5.0
### Nguồn

Được định nghĩa trong [`inc/class-mcp-adapter.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-mcp-adapter.php#L250) tại dòng 250
