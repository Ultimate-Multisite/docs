# Filter: wu_stripe_existing_product_id {#filter-wustripeexistingproductid}

Lọc ID của sản phẩm cần kiểm tra. Nếu ID này tồn tại, gói đăng ký mới sẽ sử dụng sản phẩm này.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $product_id | `string` | ID của sản phẩm Stripe cần kiểm tra. |
| $name | `string` | Tên sản phẩm. |
