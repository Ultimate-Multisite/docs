# Filter: wu_plan_product_types {#filter-wuplanproducttypes}

Lọc các loại sản phẩm mà hệ thống nên xem là các gói dịch vụ (plans).

Bộ lọc này cho phép các add-on đăng ký thêm các loại sản phẩm cần được nhận diện là các gói dịch vụ (plans) trong quá trình xác thực và phân loại.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | Mảng các loại sản phẩm cần được coi là các gói dịch vụ. |

### Since {#since}

- 2.3.0
