# Bộ lọc: wu_checkout_validation_aliases

Cho phép các nhà phát triển plugin thêm các bí danh (aliases) tùy chỉnh vào trình xác thực form.

## Tham số {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $validation_aliases | `array` | Mảng với định dạng id => bí danh. |
| $checkout | `\Checkout` | Đối tượng lớp checkout. |

### Từ phiên bản {#since}

- 2.1
