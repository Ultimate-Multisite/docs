# Bộ lọc: wu_checkout_field_templates

Đây là nơi các API của chúng tôi để thêm mẫu trường (field templates) mới được kết nối.

Bạn không nên sử dụng bộ lọc này trực tiếp. Thay vào đó, hãy sử dụng hàm `wu_register_field_template()`.
