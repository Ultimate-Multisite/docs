# Filter: wu_add_new_domain_modal_fields

Bộ lọc này dùng để lọc các trường dữ liệu (fields) trong cửa sổ modal thêm tên miền mới.

Nó cho phép các addon (ví dụ: Domain Seller) tùy chỉnh hoặc thay thế các trường đăng ký tên miền.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $fields | `array` | Các trường dữ liệu của form. |

### Since {#since}

- 2.1.0
