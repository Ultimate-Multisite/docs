# Action: wu_register_integrations

Hành động này được kích hoạt để cho phép các tích hợp (integrations) tự đăng ký.

Core đăng ký các tích hợp của nó tại đây.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $registry | `\Integration_Registry` | Đối tượng (instance) registry. |

### Since

- 2.5.0
