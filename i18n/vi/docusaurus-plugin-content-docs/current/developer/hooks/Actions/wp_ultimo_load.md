# Action: wp_ultimo_load

Được kích hoạt khi tất cả các thành phần phụ thuộc (dependencies) đã được tải xong.

Hành động này cho phép các nhà phát triển plugin thêm các chức năng mới. Ví dụ: hỗ trợ các nhà cung cấp Hosting mới, v.v.

### Since

- 2.0.0
