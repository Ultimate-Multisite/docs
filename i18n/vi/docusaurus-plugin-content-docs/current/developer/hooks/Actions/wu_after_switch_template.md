# Action: wu_after_switch_template {#action-wuafterswitchtemplate}

Cho phép các nhà phát triển plugin móc (hook) các hàm của họ sau khi người dùng hoặc quản trị viên siêu cấp thay đổi mẫu (template) của trang web.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $id | `int` | ID Trang web |

- 1.9.8
