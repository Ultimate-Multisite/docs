# Dashboard Baxışı

Ultimate Multisite, Ultimate Multisite menyusunda xüsusi bir **Dashboard** (Nəzarət Paneli) təqdim edir və həm də Network Dashboard-a widgetlər əlavə edir. Bu məqalədə hər iki yerdə nə ilə qarşılaşacağınızı izah edirik.

## Network Dashboard {#the-network-dashboard}

Ultimate Multisite, **My Sites → Network Admin → Dashboard** yerində tapılan şəbəkə səviyyəli dashboard-a faydalı widgetlər əlavə edir.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

Hər bir widgetin nə təqdim etdiyini aşağıda göstəririk:

**First Steps (İlk Addımlar):** Yeni quraşdırmada görünür və sizə ödəniş metodunu konfiqurasiya etmək və test hesabı yaratmaq kimi qurulma tapşırıqlarını tamamlamanızı xatırladır.

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Summary (Xülasə):** Gün ərzində qeydiyyat olunan istifadəçi sayını və gəliri hesabat edir. MRR (Aylıq Təkrar Gəlir) isə gəlir yaradan üzvlüyü olan müştərilər arasında proqnozlaşdırılmış ümumi məbləği göstərir.

![Summary widget](/img/admin/network-dashboard-summary.png)

**Activity Stream (Aktivlik Axını):** Ultimate Multisite üzrə hadisələri — qeydiyyatlar, ləğvatlar, saytlar yaratmaq, silmək və digər fəaliyyətləri — hesabat edir.

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**Right Now (Hazırda):** Şəbəkədəki istifadəçilər və saytlar haqqında qısa bir xülasə göstərir və yeni saytlar və ya istifadəçilər yaratmaq üçün qısa yollar təqdim edir.

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**News & Discussions (Xəbər və Müzakirələr):** Ən son Ultimate Multisite xəbərlərini, o cümlədən yeniləmələr, kritik buglar və təhlükəsizlik düzəlişlərini çəkir və göstərir.

![News widget](/img/admin/network-dashboard-news.png)

## Ultimate Multisite Dashboard {#the-ultimate-multisite-dashboard}

Şəbəkə dashboard-u şəbəkə haqqında ümumi məlumat təqdim edərkən, Ultimate Multisite dashboard-u (Ultimate Multisite menyusunun yuxarı səviyyəsində) xidmət haqqında məlumat təqdim edir.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Monthly Recurring Revenue Growth (Aylıq Təkrar Gəlir Artımı):** 12 aylıq bir kalendar dövrü ərzində gəlirin aylıq bölünməsini göstərir. Yeni gəlir mavi rəngdə, ləğvatlar isə qırmızı rəngdə görünür.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**Date-range Support (Tarix Aralığı Dəstəyi):** Dashboard məlumatlarını müəyyən bir dövrə görə filtrləməklə, ehtiyac duyduğunuz məlumatlara fokuslana bilərsiniz. Tarix aralığı, aşağıdakı widgetlərdə nə göstəriləcəyini müəyyən edir.

![Date range and dashboard stats](/img/admin/um-dashboard-date-range.png)

Dashboard widgetlərinə aşağıdakılar daxildir:

- **Revenue (Gəlir)** — MRR və məhsulə görə bölünmələrlə dövrün ümumi gəliri
- **New Memberships (Yeni Üzlüyü)** — Seçilmiş dövrdə neçə yeni üzvlük yaradıldığı
- **Signups by Countries (Ölkələr üzrə Qeydiyyat)** — Qeydiyyatların coğrafi bölünməsi
- **Signups by Forms (Formalar üzrə Qeydiyyat)** — Müştərilərin hər bir checkout forması vasitəsilə qeydiyyatdan keçmə sayı və ya administratorlar tərəfindən əlavə edilmə sayı
- **Most Visited Sites (Ən Çox Ziyar Edilən Saytlar)** — Şəbəkənizdə ən çox ziyarət edilən saytlar (Sayt ziyarət sayını **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** yerində aktivləşdirmək tələb olunur)

![Dashboard statistics](/img/admin/um-dashboard-stats.png)
