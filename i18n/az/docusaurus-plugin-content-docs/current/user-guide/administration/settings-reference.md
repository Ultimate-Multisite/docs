# Ayarlar Referansı

Bu səhifə Ultimate Multisite-də gündəlik idarəetməyə və son davranış dəyişikliklərinə təsir edən ayarları izləyir.

## Digər Seçimlər

**Digər Seçimlər** sahəsi **Ultimate Multisite > Ayarlar > Giriş və Qeydiyyat** altında görünür.

| Ayar | Təsvir |
|---|---|
| **Jumper-ı Aktivləşdir** | Admin sahəsində Jumper sürətli naviqasiya alətini göstərir. Bu aləti Ultimate Multisite ekranlarına, şəbəkə obyektlərinə və dəstəklənən admin yerlərinə birbaşa keçmək üçün istifadə edin. Bu qısayolun görünməməsini istəyirsinizsə, onu deaktiv edin. |

## Xəta hesabatı və telemetriya

Əvvəlki xəta hesabatı üçün razılaşma (opt-in) ayarı ayarlar səhifəsindən çıxarılıb. Anonim telemetriya deaktivləşdirilib və onu aktiv etmək üçün UI düyməsi yoxdur.

Əgər ayarlar səhifəsi üçün daxili iş kitabları və ya skrinşotlar saxlayırsınızsa, administratorların artıq mövcud olmayan bir ayarı axtarmaması üçün köhnə xəta hesabatı razılaşması sahələrinə istinadları çıxarın.

## İdxal/İxrac ayarları

**İdxal/İxrac** ayarları tabı hansı ayarları idarə etdiyini təsvir edir və sayt və şəbəkə arxivləri üçün birbaşa **Ultimate Multisite > Sayt İxracı**-ya keçir. İdxal/ixrac konfiqurasiyası üçün ayarlar tabından, tək sayt ixracı/ixracı iş axını üçün isə **Alətlər > İxrac və İdxal**-dan istifadə edin və tam Şəbəkə İxracı arxivi lazım olduqda Sayt İxracı alətindən istifadə edin.
