# PRO Theme ilə Admin Panelinizi Özəlləşdirmək

Müştərilərlə işləyərkən tez-tez müştərilərimizə WordPress admin dashboard-unda özəlləşdirilmiş bir təcrübə təqdim etmək ehtiyacı yaranır. Əgər daha peşəkar bir görünüş və hissiyat axtarırsınızsa, PRO Theme sizə kömək edə bilər.

PRO Theme, müştərilərinizin Dashboard-unu özəlləşdirməyiniz zamanı sizə kömək etmək üçün hazırlanmış bir admin temasıdır. Həmçinin, Ultimate Multisite-ın premium plugin-lərinin əsas hissəsidir.

Quraşdırdıqdan sonra, _Plugins_ səhifənizə keçin və Şəbəkənizdə (Network) aktivləşdirmək üçün klikləyin.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Plugins səhifənizdə şəbəkə genişliyində aktivləşdirin_

_**QEYD:**_ _Əgər sadəcə quraşdırırsınızsa, bu, müştərilərinizin üçün yalnız həmin istifadəçiyə temanı əlavə etmək üçün işarələnmə bilən bir seçim kimi görünəcək._

Beləliklə, admin interfeysiniz tamamilə yeniləndi! Həmçinin, soldakı sütunda PRO Theme adlı yeni bir menyu əlavə olunur.
