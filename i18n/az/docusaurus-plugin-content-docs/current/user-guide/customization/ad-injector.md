# Ultimate Multisite: Ad Injector ilə şəbəkənizdə reklam yerləşdirmək

Ultimate Multisite: Ad Injector, planlarınızın redaktə səhifəsindən birbaşa reklam yerləşdirməyiniz üçün vasitələr təqdim edən **pulsuz bir add-on**dur.

Ultimate Multisite: Ad Injector-u quraşdırdıqdan sonra, onu aktivləşdirmək üçün _Plugins_ səhifənizə keçin.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Network Activate düymasına klikləyin_

Add-on aktiv olduqdan sonra, plan redaktə ekranındakı _Advanced Options_ meta box-unda _Ad Settings_ adlı yeni bir seçim olacaq.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_İndi hər bir plan üçün reklam parametrlərini redaktə edə bilərsiniz_

Bu tabda, ön-ekran (front-end) üçün məqalələr və səhifələr üçün, məzmunun əvvəlində və sonra reklam kodlarını əlavə edə biləcəksiniz.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Reklam kodunuzu yerləşdirdikdən sonra, bu, həmin planın məqalələrində və səhifələrində görünəcək._

Həmçinin, həmin planın istifadəçiləri üçün Admin Pages-in başlıq hissəsinə (header) reklamlar daxil etdirə bilərsiniz.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Əgər istifadəçinizin arxa tərəfində (back-end) reklamlar göstərmək istəyirsinizsə, Admin Panel Ad Code seçiminə keçin_

### Reklamçılıqdan Kənarda

Ultimate Multisite: Ad Injector-dan yüksək səviyyəli planlara dəstək təklif etmək üçün də istifadə edə bilərsiniz.

Sadəcə _Admin Panel Ads_ seçiminə keçin və dəstək alətinizi üçün HTML kodunu əlavə edin.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Müştəri dəstəyini müəyyən planlara əlavə edin_

Bu yolla insanlar sizə birbaşa Admin Panel-ləri vasitəsilə çatacaqlar!
