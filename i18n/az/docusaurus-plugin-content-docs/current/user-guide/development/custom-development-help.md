# Codeable-dan Xüsusi İnkişaf Dəstəyi Necə Alınır?

Bəzən şəbəkənin (network) axışını tam istədiyimiz kimi etmək üçün həqiqətən də güclü özəlləşdirməyə ehtiyac duyuruq. Əgər axtardığınız şey budursa, sizə [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE)-ni tövsiyə edirik. Bu platformada Ultimate Multisite təcrübəsi olan peşəkar developerlər tapa bilərsiniz.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) müştəriləri hər ölçüdə layihələr üzərində işləyə bilən ixtisaslı WordPress freelance developerləri ilə birləşdirən bir platformadır. Onlar, sizin nəzərdə tutduğunuz iş üçün xüsusi ixtisaslaşmış mütəxəssislərlə eşləşdirilməyinizə əmin olmaq üçün diqqətli bir müraciət prosesindən keçirirlər.

### Codeable-da Layihə Necə Yaradılır?

Əvvəlcə [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE)-a daxil olun və **Start a Project** düymasına klikləyin.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

İndi isə layihə təqdim etmək üçün soldakı **Start a Project** düymasına klikləyin.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

"I need help with" (Nə barədə köməyə ehtiyacım var) hissəsində **Customizing** (Özəlləşdirmə) seçin:

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

"My" (Mənim) hissəsində **Plugin** seçin:

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Sonra **Continue** düymasına klikləyin.

<!-- Screenshot unavailable: Codeable project form Continue button -->

İndi isə layihənizi və ehtiyaclarınızı mümkün olan ən yaxşı şəkildə təsvir etmək vaxtıdır. Mümkün qədər çox məlumat verməyə çalışın!

<!-- Screenshot unavailable: Codeable project description form -->

Qiymətləndirməyə kömək etmək üçün aşağıdakı sualları cavablandırın.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Bundan sonra, əgər hələ [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) hesabınız yoxdursa, bir hesabı yaradın və **Publish My Project** düymasına klikləyin.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) layihənizi dərc etdikdən sonra, platforma sizin layihəniz üçün tələb olunan bacarıq dəsti ilə uyğunlaşan developerlərlə sizi birləşdirəcək.

Bu proses, WordPress və Ultimate Multisite təcrübəsi olan, işiniz üçün doğru şəxsi tapmağa kömək edəcək!

Codeable-ın minimum qiyməti saatda 70–120 USD-dir. Təxminlər işin miqyasına, mürəkkəbliyinə və layihənin təciliyinə görə dəyişəcək.

Təxmini qiyməti aldıqdan sonra, ehtiyaclarınıza ən uyğun developeri işə götürə və layihənizə birlikdə başlaya biləcəksiniz!
