# Ultimate Multisite Hooks və filters

Ultimate Multisite bir çox funksiyalardan və verilənlər bazası sorğularından ibarətdir. Mövcud funksiyalara yeni kod əlavə etməyə və ya onları dəyişdirməyə imkan verən hooks və filters-lərə malikdir.

Təcrübəli istifadəçilərə şəbəkələrini daha da fərdiləşdirmələri üçün bu hooks və filters-lərin bir siyahısını istinad məqsədi ilə hazırladıq. Siyahıya aşağıdakı linkə klikləyərək Github-dan daxil ola bilərsiniz.
