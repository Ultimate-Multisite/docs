# Tez-tez Verilən Suallar

Dəstək sorğuları zamanı tez-tez rast gəlinən bəzi **Tez-tez Verilən Suallar** bunlardır.

  * **Ultimate Multisite nədir?**

Ultimate Multisite, WordPress multisite quraşdırmaları üçün hazırlanmış bir WordPress plugin-idir. Bu plugin sizin WordPress quraşdırmanızı [WordPress.com](https://WordPress.com) kimi premium bir sayt şəbəkəsinə çevirir və müştərilərin aylıq, rüblük və ya illik ödənişlər yolu ilə (pulsuz planlar yaratmaq da mümkündür) saytlar yaratmasına imkan verir.

  * **Ultimate Multisite Multisite olmadan işləyə bilərmi?**

Xeyr. Ultimate Multisite WordPress multisite, subdirectory və ya subdomain tələb edir.

  * **Ultimate Multisite üçün tələblər nələrdir?**

Ultimate Multisite aşağıdakıları tələb edir:

**WordPress Version** : v5.3+ (Tövsiyə olunur: Son Stabil Versiya)
**PHP Version** : 7.4.x (8.0-dan yuxarı dəstəyi yaxın zamanda gəlir)
**MySQL Version** : v5+ (Tövsiyə olunur: Versiya 5.6, əgər hosting provayderinizdə 8.0 mövcud deyilsə)

_Multisite (subdomain və ya subdirectory) də aktivləşdirilməlidir._ [How to Install WordPress Multisite](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **Ultimate Multisite Subdirectories ilə Multisite ilə işləyə bilərmi?**

Bəli. Ultimate Multisite həm subdirectory, həm də subdirectory quraşdırmalarını dəstəkləyir.

  * **Ultimate Multisite Subdomains ilə Multisite ilə işləyə bilərmi?**

Bəli. Ultimate Multisite həm subdirectory, həm də subdomain quraşdırmalarını dəstəkləyir.

  * **İstifadəçilərin hansı Plugin və Mövzuları (Themes) istifadə edə biləcəyini məhdudlaşdırmaq üçün Ultimate Multisite-dən istifadə edə bilərəmmi?**

Bəli. Ultimate Multisite ilə şəbəkənizdə hər plan üçün hansı plugin və mövzuların (themes) mövcud olacağını seçə bilərsiniz.

  * **İstifadəçilərin öz saytlarında yaratdıra biləcəyi məqalə sayını məhdudlaşdırmaq üçün Ultimate Multisite-dən istifadə edə bilərəmmi?**

Bəli. Ultimate Multisite ilə şəbəkənizdə **hər mövcud post növü** üçün məqalə kvotaları təyin edə bilərsiniz. Həmçinin adminin dəvət edə biləcəyi istifadəçi sayını, **disk sahəsini** və daha çox şeylə məhdudlaşdıra bilərsiniz.

  * **Ultimate Multisite Sayt Şablonlarını (Site Templates) dəstəkləyirmi?**

Bəli. Ultimate Multisite ilə istədiyiniz qədər Sayt Şablonları (Site Templates) yarada bilərsiniz. Sayt Şablonları mövzular (themes) deyil, şəbəkənizdə normal saytlardır. Pluginləri, mövzuları aktiv edə və Sayt Şablonunu istədiyiniz kimi fərdiləşdirə bilərsiniz. Əgər bir istifadəçi bu şablonu seçərsə, onun saytı bu şablonun bir nüsxəsi kimi yaradılacaq və bütün fərdiləşdirmələr yerində olacaq.

Bu zamanda bir plan üçün müəyyən bir şablon təyin edə bilərsiniz. Bunu etməklə, həmin planda qeydiyyatdan keçən hər istifadəçinin saytı həmin Sayt Şablonunun bir nüsxəsi kimi yaradılacaq.

  * **Ultimate Multisite WooCommerce ilə inteqrasiya edirmi?**

Bəli!

Şəbəkə istifadəçilərinizin WooCommerce-dən istifadə etməsini, onlara mağazalar qurmağa imkan verməsini istəyirsinizsə, bu, Ultimate Multisite ilə mümkündür. Hətta müştərilərinizin hər plan səviyyəsində neçə məhsul yarada biləcəyini məhdudlaşdıra bilərsiniz.

Ultimate Multisite üçün abunə ödənişləri baxımından, [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) istifadə edərək Ultimate Multisite müştərilərinizin əsas saytınızda WooCommerce gateway-ləri ilə ödəniş etməsinə imkan verə bilərsiniz. Bu yolla, WooCommerce üçün hazırlanmış böyük ödəniş gateway-ləri kitabxanasından istifadə edərək Ultimate Multisite müştərilərinizi hesablaşdıra bilərsiniz.

  * **WPMUDEV Pro Sites-dan abunəlikləri köçürmək yolu varmı?**

Bəli, Ultimate Multisite indi sizə Pro Sites şəbəkənizi [köçürməyiniz üçün köməkçi bir alət təklif edir](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/).

  * **Plan Redaktə səhifəmdə “Allowed Plugins” (İcazə Verilmiş Pluginlər) tabı niyə boşdur?**

Ultimate Multisite yalnız şəbəkənizdə aktivləşdirilməmiş pluginlərə girişə icazə verməyə və ya bu icazəni ləğv etməyə imkan verir. Bir plugin şəbəkə səviyyəsində aktivləşdirildikdə, təbiətindən, şəbəkənizin hər saytında avtomatik yüklənəcək.

Əgər “Allowed Plugins” tabında heç bir plugin görmürsünüzsə, istədiyiniz pluginin şəbəkə səviyyəsində aktiv olub-olmadığını yoxlayın.

  * **Öz qiymət cədvəli dizaynı istifadə edə bilərəmmi?**

Bəli. 1.4.0 versiyasından etibarən, Ultimate Multisite şablon üstələnmələrini (template overrides) dəstəkləyir.

  * **Ultimate Multisite WP Engine ilə işləyirmi?**

Bəli. Ultimate Multisite həmişə WP Engine tərəfindən dəstəklənmişdir, lakin domen xəritələnməsinin işləməsi üçün admin tərəfindən əl ilə əlavə addımlar tələb olunurdu (domenun WP Engine panelinə əlavə edilməsi). **1.6.0 versiyasından etibarən bu artıq belə deyil.** Ultimate Multisite birbaşa WP Engine API ilə inteqrasiya edir və xəritələnmədən sonra domenlər avtomatik olaraq panellə əlavə olunur.

  * **Hansı hosting provayderini tövsiyə edirsiniz?**

Ultimate Multisite istənilən hosting provayderi ilə işləməlidir. Yeganə fərq ondadır ki, Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane və [Runcloud.io](http://Runcloud.io) kimi bəzi hosting şirkətləri domen xəritələnməsi imkanları üçün daha sıx inteqrasiyaya malikdir, buna görə də qutu-dan-qutu işləyirlər. Digər hostinqlər üçün isə domen xəritələnməsini əl ilə qurmanız lazım gələ bilər.
