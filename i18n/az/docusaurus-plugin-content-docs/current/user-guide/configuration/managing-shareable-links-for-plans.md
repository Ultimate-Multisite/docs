# Planlar üçün Paylaşılan Linkləri İdarə Etmək {#managing-shareable-links-for-plans}

Ultimate Multisite sizə plan linklərini birbaşa müştərilərinizlə paylaşmaq imkanı verir. Bunu etmək üçün istənilən planın redaktə səhifəsinə keçin və "Paylaşılan Linki Kopyala" düyməsinə klikləyin.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_Planın Redaktə Səhifəsində, paylaşılan linklər yaratmaq üçün düyməni tapa bilərsiniz._

Diqqət edin ki, bu mexanizm linki yaratmaq üçün standart (default) ödəniş tezliyindən istifadə edir. Bu o deməkdir ki, məsələn, standart ödənişiniz aylıqdırsa, sonda belə bir linkiniz olacaq: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Əgər paylaşılan linkin rüblük (quarterly) olmasını istəyirsinizsə, sona 3 əlavə etməlisiniz:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

_premium_ planını illik (yearly) cədvəllə seçmək üçün isə 1 və sonra sözü əlavə edəcəksiniz:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Yadda saxlayın ki, fərqli ödəniş tezlikləri üçün yalnız bir linkiniz ola bilməz. Hər bir ödəniş tezliyi üçün ayrı-ayrı, spesifik bir link paylaşmalısınız.

Linkinizi müştərilərinizlə paylaşdıqdan sonra, onları qeydiyyat axınınızın (signup flow) ilk addımına yönləndirəcək.
