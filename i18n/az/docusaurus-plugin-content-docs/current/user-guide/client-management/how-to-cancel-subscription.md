# Abunəliyimi necə ləğv edə bilərəm?

Əgər Ultimate Multisite lisenziyanız varsa, onun yenilənməsini istədiyiniz vaxt ləğv edə bilərsiniz. Sadəcə aşağıdakı addımları izləyin:


<!-- Screenshot unavailable: Account page email with access link -->

Satın alma zamanı verdiyiniz e-poçt ünvanından və şifrənizdən istifadə edin.

<!-- Screenshot unavailable: Login page with email and password fields -->

Bu, sizin Account Page-inizin ilkin ekranıdır:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

Abunəliyinizi ləğv etmək üçün, sol menyudan `Renewing & Billing` bölməsinə keçin.

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

Sağdakı ox işarəsinə klikləyərək yan pəncərəni açın. Sonra, **avtomatik yenilənməni** ləğv edəcək seçimi seçməlisiniz.

Sistem sizə bir təsdiqləmə mesajı göstərəcək.

Abunəliyinizi ləğv etmək istədiyinizə əminsinizsə, sadəcə `**Cancel Renewals**` düyməsinə klikləyin. Bu əməliyyatdan sonra, sizdən qısa bir sorğu anketini doldurmanız xahiş ediləcək.

Hazırdır! Abunəliyiniz avtomatik yenilənməyəcək.

**Abunəliyiniz bitənə qədər etibarlı açarınız olacaq**. Əgər abunəliyinizi yenidən aktivləşdirmək istəyirsinizsə, bunu əl ilə etməlisiniz.
