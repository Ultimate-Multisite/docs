# Müştərinizin Saytları (v2) {#your-clients-websites-v2}

Şəbəkənizdə qeydiyyatdan keçdikdən sonra müştəriləriniz öz dashboardlarına daxil ola biləcəklər.

**Hesab** səhifəsindən, burada ödənişləri, üzvlük hüquqlarını, domenləri, plan məhdudiyyətlərini və daha çoxu ilə bağlı vacib məlumatları görə biləcəkləri yerinə, onlar həm də bütün saytlarını görə və idarə edə biləcəkləri **Saytlarım** səhifəsinə daxil olurlar. Onların etməli olduğu yeganə şey dashboardlarında "Saytlarım" düyməsinə klikləməkdir.

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

Onlara bütün saytları göstəriləcək və hər birinin altında bir **İdarə et** düyməsi olacaq. Əgər bu düyməyə klikləsərlər, **bu müəyyən saytın dashboarduna** yönləndiriləcəklər.

Onlar həmçinin, bir neçə sayt sahibi olmağa imkan verən planlar üçün hansı saytlarının **əsas** olduğunu da görə biləcəklər.

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

Sayt redaktə interfeysinin tam görünüşü budur:

![Site edit full interface](/img/admin/site-edit-full.png)

Əgər əsas saytlarını dəyişmək istəyirlərsə, **Hesab** menyusuna klikləyərək Hesab səhifəsinə getmələri lazımdır.

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

Müştəri detallarını da görə və redaktə edə bilərsiniz:

![Customer edit interface](/img/admin/customer-edit.png)

Sonra, **Default Site-ı Dəyişdir** düyməsinə klikləməlidirlər.

![Change Default Site button on account page](/img/account-page/change-default-site.png)

Bir pop-up pəncərə açılacaq. Aşağı açılan siyahıya klikləyin və əsas sayt olmasını istədiyiniz saytı seçin.

![Change Default Site dropdown selection](/img/account-page/change-default-site-dropdown.png)

Sonra, sadəcə **Default Site-ı Dəyişdir** düyməsinə klikləyin və hazırdır!

![Confirming the default site change](/img/account-page/change-default-site-confirm.png)
