# İlk dəfə hesab səhifəminə necə daxil ola bilərəm? {#how-can-i-access-my-account-page-for-the-first-time}

Ultimate Multisite-ə xoş gəlmisiniz!

Pluginimizi aldıqdan sonra e-poçtunuza gələn link vasitəsilə hesab səhifənizə daxil olun:

<!-- Screenshot unavailable: Account page email with access link -->

Alış-veriş zamanı verdiyiniz e-poçt ünvanından və e-poçtunuzda aldığınız şifrədən istifadə edin.

<!-- Screenshot unavailable: Login page with email and password fields -->

Bu hesabda ilk dəfə daxil olduğunuz üçün, sizdən şifrənizi dəyişməyiniz xahiş ediləcək. Şifrənizi dəyişin və _Save_ düymasına basın.

<!-- Screenshot unavailable: Change password form with Save button -->

Hesab səhifənizdə fakturalandırma, yeniləmələr, ləğv etmələr və digər bir çox məlumat kimi bütün hesab detallarınızı idarə edə bilərsiniz.
