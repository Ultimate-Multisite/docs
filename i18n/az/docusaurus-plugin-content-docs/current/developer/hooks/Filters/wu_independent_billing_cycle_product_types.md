# Filter: wu_independent_billing_cycle_product_types

Müstəqil ödəniş dövrünə malik məhsul növlərini filtrləyir.

Müstəqil ödəniş dövrünə malik məhsullar, səbətdəki planın ödəniş dövrünə uyğunlaşmaq məcburiyyətində deyil. Bu, plan aylıq və ya illik olmasından asılı olmayaraq, həmişə illik ödəniş tələb edən domen qeydiyyatları kimi məhsullar üçün faydalıdır.

## Parametrlər {#parameters}

| Ad | Növ | Təsvir |
|------|------|-------------|
| $types | `array` | Müstəqil ödəniş dövrünə malik məhsul növlərinin massivi. |
