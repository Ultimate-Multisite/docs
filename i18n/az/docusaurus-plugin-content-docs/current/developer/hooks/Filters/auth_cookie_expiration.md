# Filter: auth_cookie_expiration

Autentifikasiya cookie-sinin bitmə müddətini tənzimləyir.

## Parametrlər

| Name | Type | Description |
|------|------|-------------|
| $length | `int` | Bitmə müddəti (saniyə ilə). |
| $user_id | `int` | İstifadəçi ID-si. |
| $remember | `bool` | İstifadəçinin girişini yadda saxlamaq. Varsayılan olaraq yox. |

### Nə vaxtdan

- 2.8.0
