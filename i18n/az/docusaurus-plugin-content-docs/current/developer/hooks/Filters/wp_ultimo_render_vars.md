# Filter: wp_ultimo_render_vars

## Parametrlər {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $args | `array` | Render çağırışı ilə ötürülən dəyişənləri ehtiva edən massiv. |
| $view | `string` | Render ediləcək görünüşün adı. |
| $default_view | `string` | Alternativ görünüşün adı. |
