# Filter: wu_magic_link_enforce_user_agent

İstifadəçi agenti yoxlanılmasının tətbiq edilmədiyini filtrləyir.

Tokenlərin müxtəlif brauzerlərdə və ya cihazlarda işləməsinə imkan vermək üçün bunu `false` olaraq təyin edin. Bu, təhlükəsizliyi azaldır, lakin istifadə rahatlığını artırır.

## Parametrlər {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $enforce | `bool` | İstifadəçi agenti uyğunluğunun tətbiq edilməsi. |
