# Filter: wu_username_from_email {#filter-wuusernamefromemail}

Yeni müştəri istifadəçi adını təyin edir.

## Parametrlər {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $username | `string` | Müştəri istifadəçi adı. |
| $email | `string` | Yeni müştəri e-poçt ünvanı. |
| $new_user_args | `array` | Yeni istifadəçi arqumentlərinin massivi, bəlkə ki, ad və soyadları da daxil edir. |
| $suffix | `string` | İstifadəçi adını unikal etmək üçün əlavə ediləcək sətir. |
