# Filter: wu_admin_notices {#filter-wuadminnotices}

Developerlərə Ultimate Multisite tərəfindən əlavə edilmiş admin bildirişlərini (notices) filtrləməyə imkan verir.

## Parametrlər {#parameters}

| Ad | Növ | Təsvir |
|------|------|-------------|
| $notices | `array` | O müəyyən panel üçün bildirişlər siyahısı. |
| $all_notices | `array` | Əlavə edilmiş, panelə görə ayrılmış bildirişlər siyahısı. |
| $panel | `string` | Bildirişləri almaq üçün panel. |
| $filter | `string` | Əgər silinə bilən bildirişlər filtrlənmişdirsə. |
| $dismissed_messages | `array` | Silinmiş bildiriş açarları siyahısı. |
