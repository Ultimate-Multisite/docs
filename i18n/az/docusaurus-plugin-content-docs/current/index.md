# Ultimate Multisite Dokumentasiyası {#ultimate-multisite-documentation}

Ultimate Multisite dokumentasiyasına xoş gəlmisiniz. Ultimate Multisite, WordPress Multisite-i tam bir Website as a Service (WaaS) platformuna çevirir.

## Dokumentasiya Bölmələri {#documentation-sections}

### [User Guide](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

WaaS platformunuzu qurmaq, konfiqurasiya etmək və idarə etmək üçün lazım olan hər şey. Quraşdırmadan, ödəniş qapılarından, domen xəritələnməsindən, müştəri idarəetməsindən və daha çox şeylə əhatələnir.

### [Developer Documentation](/developer/hooks) {#developer-documentation}

Ultimate Multisite üzərində işləyən developerlər üçün texniki istinad. Hooks istinadı, REST API dokumentasiyası və addon hazırlama təlimatları daxildir.

### [Addons](/addons) {#addons}

Ultimate Multisite-i WooCommerce inteqrasiyası, analitika, domen satışı və daha çox funksionallıqla genişləndirən ödənişli addon pluginlər üçün dokumentasiya.
