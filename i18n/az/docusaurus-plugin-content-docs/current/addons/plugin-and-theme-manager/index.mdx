# Plugin və Tema Meneceri

Multisite Ultimate: Plugin və Tema Meneceri

Qurulmuş plugin və temaların meta məlumatlarını (başlıq, müəllif, miniatür və təsvir) redaktə edin və ya gizlədin. İstifadəçilərinizin onları filtrləməsinə imkan vermək və istifadəçilərin panelinə gözəl, fərdiləşdirilmiş bir Plugins səhifəsi əlavə etmək üçün plugin və temalar üçün kateqoriyalar yaradın.

Plugindən istifadə etmək üçün sadəcə şəbəkə administratorunuzdakı Plugins və ya Themes səhifələrinə keçin və "Edit Plugin/Theme info" düyməsinə klikləyin.

## Quraşdırma {#installation}

1. Upload 'ultimate-multisite-plugin-and-theme-manager' to the '/wp-content/plugins/' kataloquna
2. Plugin-i WordPress Network Admin-dəki 'Plugins' menyusu vasitəsilə aktivləşdirin

## İstifadə Təlimatı {#usage-guide}

Ultimate Multisite: Plugin and Theme Manager, şəbəkə istifadəçilərinizin şəbəkənizdəki plugin və tema seçimləri ilə necə qarşılaşdığı və necə qarşılıqlı əlaqə qurduğu üzərində tam nəzarət edən güclü bir **premium** Ultimate Multisite əlavəsidir.

Bu əlavə üç məqsədə xidmət edir:

  1. Həm plugin, həm də temaların meta məlumatlarını tamamilə dəyişməyə imkan verir. Buna ad, təsvir, müəllif adı və hətta miniatürlər də daxildir!

  2. İstifadəçilərinizin onları asanlıqla sıralamasına imkan vermək üçün plugin və temaları kateqoriyalaşdırmağa icazə verir.

  3. Köhnə, darıxdırıcı plugins səhifəsini kateqoriyalara görə sıralanmış yeni, temalara bənzər bir səhifə ilə əvəz etmək seçimi verir.

Yükləyib qurduqdan sonra, Ultimate Multisite Settings səhifəsində yeni bir tab — _Add-on Settings_ — görünəcək.
