# Dəstək Tiketləri (Support Tickets)

**Dəstək Tiketləri** (Support Tickets), Ultimate Multisite üçün güclü bir əlavədir və WordPress şəbəkənizə hərtərəfli bir dəstək tiketi sistemi əlavə edir. SaaS platformaları, WaaS (Website as a Service) təminatçıları və həm şəbəkə, həm də fərdi sayt səviyyəsində müştəri dəstəyi verməli olan multisayt şəbəkələri üçün mükəmməldir.

## Niyə Dəstək Tiketlərini Seçməlisiniz? {#why-choose-support-tickets}

* **Çox Səviyyəli Dəstək**: Şəbəkə administratorları şəbəkəyə yayılmış tiketlər ilə məşğul olur, sayt sahibləri isə öz müştəri dəstəklərini idarə edirlər.
* **Peşəkar İş Axını**: Prioritetlər, statuslar və təyin etmələr ilə tam tiket dövrü idarəetməsi.
* **Müştəri Təcrübəsi**: Müştərilərin kömək almağını asanlaşdıran gözəl, responsiv interfeys.
* **Təhlükəsiz və Etibarlı**: WordPress təhlükəsizlik ən yaxşı təcrübələri ilə qurulub və optimallaşdırılmış verilənlər bazası performansı təmin edir.
* **Problemizasiya Edilməz İntegrasiya**: Ultimate Multisite-ın müştəri və sayt idarəetməsi ilə mükəmməl işləyir.

## Əsas Xüsusiyyətlər {#key-features}

**Tam Tiket İdarəetməsi**

* Asan istinad üçün unikal tiket nömrələndirmə sistemi
* Konfiqurasiya edilə bilən prioritet səviyyələri (Aşağı, Normal, Yüksək, Kritik)
* Hərtərəfli status iş axını (Açılışda, İrəliləyir, Gözləyir, Həll Edilib, Bağlanıb)
* Tiket kateqoriyası (Ümumi, Texniki, Fakturasiya, Xüsusiyyət Tələbi, Bug Hesabatı)
* Tiketləri istehsalçılara (support agents) seçimlə avtomatik təyin etmə

**İpək (Threaded) Konversasiyalar**

* Tiket cavabları üçün tam konversasiya ipək quruluşu
* Heyət və müştəri cavabları üçün fərqli stilizasiya
* Müştərilərin görə bilmədiyi məxfi heyət qeydləri
* Real vaxt AJAX gücləndirilmiş interfeys
* Formatlama seçimləri ilə müasir mətn redaktoru

**Fayl Əlavələri**

* Sürük-bırak (drag-and-drop) dəstəyi ilə təhlükəsiz fayl yükləməsi
* Konfiqurasiya edilə bilən fayl tipi və ölçü məhdudiyyətləri
* Zərərli fayl aşkarlanması ilə təkmil təhlükəsizlik
* Giriş nəzarəti ilə qorunan yükləmələr

**Ağıllı Bildirişlər**

* Yeni tiketlər, cavablar və status dəyişiklikləri üçün avtomatik e-poçt xəbərdarlığı
* Fərdiləşdirilə bilən e-poçt şablonları
* Heyət və müştərilər üçün rola əsaslanan bildirişlər
* Şəxsi mesajlar üçün dinamik şablon dəyişənləri

**Güclü Admin İnterfeysi**

* Sıralama və filtrasiya ilə təkmil tiket siyahısı
* Status, prioritet və təyin etmə üçün sürətli redaktə
* Effektiv idarəetmə üçün toplu əməliyyətlər
* Performans göstəriciləri ilə statistikası paneli (dashboard)
* Hərtərəfli parametrlər paneli

**Gözəl Müştəri Portalı**

* Bütün cihazlarda təmiz, responsiv interfeys
* Mobil-ə ilk baxış (Mobile-first) dizayn
* Təkmil axtarış və filtrasiya
* Formatlama panelinə malik müasir redaktor
* İstifadəsi asan tiket təqdim formaları

**Fərqli İntegrasiya**

* Shortcode-lar və ya Ultimate Multisite UI elementlərindən istifadə edin
* Tiket formalarını və siyahılarını saytınızın istənilən yerində göstərin
* Multisayt şəbəkə quruluşu ilə problemizasiya edilməz işləyir
* Hooks və filters ilə genişləndirilə bilər

## Kimlər üçün idealdır {#perfect-for}

* Müştəri dəstəyinə ehtiyacı olan SaaS platformaları
* Bir neçə müştəri saytını idarə edən WaaS təminatçıları
* Səviyyəli dəstəyə malik üzvlük saytları
* Bir neçə brendi idarə edən şəbəkə administratorları
* Müştəri xidməti göstərən sayt sahibləri

## Quraşdırma {#installation}

1. Ultimate Multisite plugin-inin WordPress şəbəkənizdə quraşdırılıb və aktiv olduğundan əmin olun.
2. Əlavə fayllarını `/wp-content/plugins/` kataloquna yükləyin.
3. Plugin-i WordPress-də 'Plugins' menyusu vasitəsilə şəbəkə səviyyəsində aktivləşdirin.
4. Parametrlərinizi konfiqurasiya etmək üçün Network Admin > Support Tickets yolunu izləyin.

## Tez-tez Verilən Suallar {#frequently-asked-questions}

## Bunun üçün Ultimate Multisite tələb olunurmu? {#does-this-require-ultimate-multisite}

Bəli, bu əlavənin quraşdırılıb aktiv olması üçün Ultimate Multisite tələb olunur. O, Ultimate Multisite-ın müştəri və sayt idarəetməsi xüsusiyyətləri ilə dərin şəkildə inteqrasiya olunur.

## Sayt sahibləri öz müştəri dəstəklərini idarə edə bilərmi? {#can-site-owners-handle-their-own-customer-support}

Əlbəttə ki! Sayt sahibləri öz müştərilərinə dəstək göstərmək üçün tiket sistemindən istifadə edə bilərlər, halbuki şəbəkə administratorları şəbəkəyə yayılmış dəstək tiketlərini idarə edə bilərlər.

## Müştərilər tiketləri necə təqdim edirlər? {#how-do-customers-submit-tickets}

Shortcode-lar və ya Ultimate Multisite UI elementləri istifadə edərək saytınızın istənilən yerinə tiket təqdim formaları əlavə edə bilərsiniz. Müştərilər sadəcə formanı doldurur və sorğularını təqdim edirlər.

## Tiket statuslarını və prioritetlərini fərdiləşdirə bilərəmmi? {#can-i-customize-the-ticket-statuses-and-priorities}

Bəli! Dəstək iş axınınızla uyğunlaşdırmaq üçün prioritet səviyyələrini, statusları və tiket növlərini şəbəkə administratoru parametrlərində tam şəkildə fərdiləşdirə bilərsiniz.

## Fayl əlavələri dəstəkləyirmi? {#does-it-support-file-attachments}

Bəli, həm müştərilər, həm də heyət üzvləri tiketlərə və cavablara fayl əlavə edə bilərlər. Təhlükəsizlik üçün icazə verilən fayl növləri və ölçü limitləri üzərində tam nəzarətiniz var.

## Bildirişlər necə idarə olunur? {#how-are-notifications-handled}

Sistem yeni tiketlər, cavablar və status dəyişiklikləri üçün avtomatik e-poçt bildirişləri göndərir. Bütün e-poçt şablonlarını parametrlərdə fərdiləşdirə bilərsiniz.

## Tiketləri müəyyən dəstək agentlərinə təyin edə bilərəmmi? {#can-i-assign-tickets-to-specific-support-agents}

Bəli, tiketlərini agentlərə əl ilə təyin edə və ya tiketlərini avtomatik paylamaq üçün avtomatik təyin etməni aktivləşdirə bilərsiniz.

## Heyət üçün məxfi qeydlər varmı? {#are-there-private-notes-for-staff}

Bəli, heyət üzvləri yalnız digər heyət üzvlərinə görünən, müştərilərə görünməyən məxfi cavablar əlavə edə bilərlər.

## Mobil cihazlarda istifadəsi asandır? {#is-it-mobile-friendly}

Əlbəttə ki! Bütün interfeys mobil-ə ilk baxış (mobile-first) yanaşması ilə qurulub və bütün cihazlarda gözəl işləyir.

## Shortcode-lar istifadə edərək tiket siyahılarını və formalarını göstərə bilərəmmi? {#can-i-display-ticket-lists-and-forms-using-shortcodes}

Bəli! Mövcud shortcode-lar:
* `[wu_support_tickets]` - Müştəri tiket siyahısını göstərir
* `[wu_support_ticket_form]` - Yeni tiket təqdim formasını göstərir
* `[wu_support_ticket_view ticket_id="123"]` - Müəyyən tiketi göstərir

## Hansı fayl növləri yüklənə bilər? {#what-file-types-can-be-uploaded}

İcazə verilən fayl növlərini parametrlərdə konfiqurasiya edə bilərsiniz. Sistem zərərli fayllara qarşı qorunmaq üçün təkmil təhlükəsizlik skanlayıcısı ehtiva edir.

## Tiket məlumatlarını ixport edə bilərəmmi? {#can-i-export-ticket-data}

Bəli, əlavə tiket məlumatları və statistikaları üçün ixport imkanları ehtiva edir.
