# Multi-Şəbəkə Değişiklik Qeydi

### 1.3.0 {#130}
* Yeni: Şəbəkə şablonu önizləyicisi — satın almadan əvvəl canlı panel ilə şəbəkə şablonlarını gəzin və önizləyin.
* Yeni: Ödənişdə şəbəkə şablonu seçici — müştərilər qeydiyyat ödəniş axını zamanı şəbəkə şablonu seçə biləcəklər.
* Yeni: Şəbəkələr arası magic link SSO — müştərilər magic-link autentifikasiyası vasitəsilə alt-şəbəkələrə problem olmadan daxil ola biləcəklər.
* Yeni: Şəbəkə media klonaşdırması — şəbəkələri şablondan klonaşdırarkən sayt media faylları düzgün şəkildə dublikasiya olunur.
* Düzəliş: Şəbəkə klonaşdırması indi multi-tenancy addon olmadan işləyir; super admin hüquqları klonlaşdırmadan sonra yenidən verilir.
* Düzəliş: Şəbəkə yaratmaq indi WooCommerce və bütün ödəniş qapıları ilə uyğundur.
* Düzəliş: Port nömrəsi indi fallback domen yaratmasında nəzərə alınır.
* Düzəliş: Şablon seçici CSS-i bütün mövzularda düzgün göstərilməsi üçün enqueued stylesheet-ə köçürüldü.
* Düzəliş: Site sorğusu (site query) sahəsində `network_id` açarının təyin edilməməsi (undefined) hallarına qarşı qorunma əlavə edildi.
* Düzəliş: Context switching-dən əvvəl şəbəkə icazəsi yoxlanılması `wpdb` cədvəl xüsusiyyətləri səhvlərinin qarşısını alır.
