# Mailster

Ultimate Multisite Mailster Integration addon ilə multisite şəbəkənizi güclü bir e-poçt siyahısı yaratma maşınlığına çevirin. Yeni müştərilər qeydiyyatdan keçdikdə və ya alış-veriş etdikdə onları avtomatik olaraq Mailster e-poçt siyahılarınıza əlavə edin. Bu, üzvlük platformanız ilə e-poçt marketinq strategiyanız arasında problemizə birbaşa bağlantı yaradır.

## Əsas Xüsusiyyətlər

**Avtomatik Abunəçi Sinxronizasiyası**
* Müştərilər qeydiyyatdan keçdikdə avtomatik olaraq Mailster siyahılarına əlavə olunur
* Qeydiyyat zamanı dərhal sinxronizasiya etmək və ya ödəniş təsdiqini gözləmək seçimi
* E-poçt siyahılarınızı əl ilə müdaxilə etmədən təzə və güncel saxlayın

**Fiks List İdarəetməsi**
* Bütün yeni müştərilər üçün global default siyahılar təyin edin
* Hədəflənmiş kampaniyalar üçün məhsula xas siyahılarla default ayarları dəyişin
* Mailster-in tanınmış interfeysi ilə siyahı təyin etməni asanlıqla idarə edin

**GDPR-Uyğun Opt-In Seçimləri**
* Avtomatik abunəlik və ya qutu (checkbox) əsaslı razılıq arasından seçim edin
* Təsdiq edilmiş abunəliklər üçün double opt-in aktivləşdirin
* Uyğunluq tələblərinizə əsasən opt-in qutusunu göstərmək/gizlətmək

**Ağıllı Data Mapping**
* Müştəri adlarını və əlaqə məlumatlarını avtomatik sinxronizasiya edin
* Faktura ünvanlarını abunəçi sahələrinə xəritələyin
* Hədəflənmiş kampaniyalar üçün qeydiyyat tarixlərini izləyin

**Məhsul Səviyyəsində Nəzarət**
* Müxtəlif məhsullara fərqli siyahılar təyin edin
* Müştərilərin nə aldıqlarına əsaslanaraq hədəflənmiş kampaniyalar yaradın
* Lazım olduqda müəyyən məhsullar üçün Mailster-i deaktiv edin

**Peşəkar Təcrübə**
* Siyahı seçimi üçün yerli Mailster UI inteqrasiyası
* Problemlərin aradan qaldırılması və yoxlama üçün ətraflı loglama
* Mövcud abunəçilərin problemizə inteqrasiyası

## Kimlər üçün idealdır

* E-poçt siyahıları qurmaq istəyən üzvlük saytı sahibləri
* Yeni müştəriləri böyütmək istəyən SaaS platformaları
* Tələbələri qeydiyyatına görə segmentasiya edən kurs yaradıcıları
* Ultimate Multisite ilə Mailster istifadə edən hər kəs

## Necə İşləyir

1. Addon-u Ultimate Multisite şəbəkənizdə quraşdırın və aktivləşdirin
2. Ultimate Multisite ayarlarında global default siyahıları konfiqurasiya edin
3. İsteğe bağlı olaraq məhsula xas siyahılarla dəyişdirin
4. Opt-in rejimini və abunəlik vaxtını seçin
5. Mailster siyahılarınızın avtomatik böyüdüyünü izləyin!

## Quraşdırma

1. Plugin fayllarını Ultimate Multisite addons kataloquna yükləyin
2. Mailster-in əsas saytınızda quraşdırılıb və aktiv olduğundan əmin olun
3. Addon-u Ultimate Multisite addons səhifəsi vasitəsilə aktivləşdirin
4. Konfiqurasiya etmək üçün Settings > Mailster Integration yolunu izləyin
5. Default siyahılarınızı seçin və opt-in üstünlüklərini konfiqurasiya edin

## Tez-tez Verilən Suallar

## Bu addon Ultimate Multisite olmadan işləyirmi?

Xeyr, bu addon-un işləməsi üçün Ultimate Multisite tələb olunur. O, xüsusi olaraq Mailster-i Ultimate Multisite-in müştəri və məhsul sistemi ilə inteqrasiya etmək üçün nəzərdə tutulub.

## Bütün saytlarda Mailster quraşdırmalıyam?

Xeyr, Mailster yalnız əsas şəbəkə saytınızda quraşdırılıb və aktiv olmalıdır. Addon avtomatik olaraq bütün subsitələrdən ona qoşulacaq.

## Müxtəlif məhsullara fərqli siyahılar təyin edə bilərəmmi?

Bəli! Global default siyahılar təyin edə, sonra onları məhsul-by-məhsul dəyişdirə bilərsiniz. Bu, müştərilərin nə aldıqlarına əsaslanaraq yüksək hədəflənmiş e-poçt kampaniyaları yaratmanıza imkan verir.

## Əgər müştəri artıq Mailster-də mövcuddursa nə baş verir?

Default olaraq, addon mövcud abunəçiləri yeniləyəcək və onları yeni siyahılara əlavə edəcək. Əgər mövcud abunəçiləri atlamağı üstün tutursunuzsa, bunu ayarlar hissəsində deaktiv edə bilərsiniz.

## Bu GDPR-uyğundur?

Bəli! Addon, müştərilərin e-poçt almağa açıq olmaq üçün açıq razılıq verməli olduğu qutu (checkbox) əsaslı opt-in rejimi ehtiva edir. Bu, düzgün məxfiliyyət siyasətləri ilə birləşdirildikdə GDPR tələblərinə cavab verir.

## Double opt-in istifadə edə bilərəmmi?

Əlbəttə! Double opt-in ayarını aktivləşdirin və Mailster abunəlikləri aktivləşdirməzdən əvvəl təsdiq e-poçtları göndərəcək.
