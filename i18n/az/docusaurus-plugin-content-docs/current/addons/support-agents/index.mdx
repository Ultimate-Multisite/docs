# Dəstək Agentləri

Bütün bir şəbəkəni tək idarə etmək asan bir iş deyil. Şəbəkəniz böyüməyə başladığı zaman, dəstək tələbatının artdığını görməyə başlayacaqsınız. İş yükünüzü yüngülləşdirməyə kömək edəcək insanları işə götürmək yaxşı bir fikirdir. Lakin, tam giriş/icazə vermək risklidir və şəbəkənizi zədələyə bilər. Məhz bu Support Agent add-on-u bu problemi həll edir.

Bu add-on sizə Support Agent adında yeni bir WordPress istifadəçi tipi yaratmağa imkan verir və bu istifadəçinin hansı icazələrə malik olacağı üzərində tam nəzarət edir. İcazə vermək bir toggle düymasını yandırmaq qədər asandır və əgər onu aradan qaldırmağa qərar verərsinizsə, sadəcə söndürmək kifayətdir.

## Xüsusiyyətlər {#features}

- WordPress və Ultimate Multisite icazələrinin detallı siyahısı
- Girişə asanlıqla icazə vermək və ya ləğv etmək
- Məhdudiyyətsiz sayda Agent hesabı yaratmaq

## Quraşdırma {#installation}

1. 'ultimate-multisite-support-agents' faylını '/wp-content/plugins/' kataloquna yükləyin
2. WordPress-də 'Plugins' menyusu vasitəsilə plugin-i aktivləşdirin
3. Plugin-i qurmaq üçün addım-addım Wizard-ı izləyin
