# Domain Seller {#domain-seller}

**Multisite Ultimate: Domain Seller** WordPress Multisite şəbəkənizi güclü domen satış platformasına çevirir. Müştərilərə qeydiyyat zamanı və ya öz dashboard-larından xüsusi domenləri birbaşa qeydiyyatdan keçirməyə imkan verin; real vaxtda əlçatanlıq yoxlaması və şəffaf qiymətləndirmə ilə.

### Niyə Domain Seller? {#why-domain-seller}

Domen qeydiyyatı istənilən WaaS (Website as a Service) platforması üçün ən çox tələb olunan əlavə satışlardan biridir. Müştərilərinizi üçüncü tərəf registratora göndərmək əvəzinə, domenləri birbaşa təklif edə bilərsiniz — müştərilərə maneəsiz təcrübə təqdim edərkən gəlirin daha böyük hissəsini şəbəkənizdə saxlayırsınız.

### Əsas Xüsusiyyətlər {#key-features}

#### Çoxsaylı Domen Registratoru Dəstəyi {#multiple-domain-registrar-support}
* **OpenSRS** — Tam qeydiyyat, yeniləmə və idarəetmə
* **Namecheap** — WhoisGuard və sandbox testləri ilə tam inteqrasiya
* **HostAfrica** — Domains Reseller API vasitəsilə domen axtarışı, qiymətləndirmə, qeydiyyat, köçürmə, DNS və ID qoruması
* **Openprovider** — Reseller qiymətləndirməsi, qeydiyyat, yeniləmə, köçürmələr, DNS, WHOIS məxfiliyi və TLD sinxronizasiyası
* **Hostinger** — Paylaşılan Hostinger inteqrasiya tokenindən istifadə edərək domen axtarışı və qeydiyyatı
* **Genişləndirilə bilən Arxitektura** — Integration Registry nümunəsi üzərində qurulub, daha çox provayder əlavə etməyi asanlaşdırır
* **Daxili Bağlantı Testi** — Canlı istifadəyə keçməzdən əvvəl provayder etimad məlumatlarını və bağlantını yoxlayın

#### İstiqamətləndirilmiş Quraşdırma Sihirbazı {#guided-setup-wizard}
* **Bir kliklə başlanğıc** — Provayder konfiqurasiyası, TLD idxalı və domen məhsulu yaradılması boyunca sizi yönləndirir
* **Avtomatik TLD sinxronizasiyası** — Qoşulmuş provayderlərinizdən bütün əlçatan TLD-ləri və topdansatış qiymətlərini idxal edir
* **Standart məhsul yaradılması** — Məntiqli marja standartları ilə istifadəyə hazır domen məhsulu yaradır

#### Çevik Domen Qiymətləndirməsi {#flexible-domain-pricing}
* **Üç marja rejimi** — Faiz marjası, sabit marja və ya hər TLD üçün sabit qiymət
* **TLD-yə xüsusi məhsullar** — Fərqli TLD qrupları üçün ayrıca domen məhsulları yaradın (.com/.net, .io, ölkə kodları və s.)
* **Giriş qiymətləndirməsi** — Yeni qeydiyyatları cəlb etmək üçün endirimli birinci il qiyməti təklif edin
* **Real vaxt qiymətləndirməsi** — Ağıllı keşləmə ilə provayderlərdən canlı topdansatış qiymətləri
* **Kupon dəstəyi** — Multisite Ultimate endirim kodları ilə tam uyğunluq

#### WHOIS Məxfilik Qoruması {#whois-privacy-protection}
* **Hər məhsul üzrə konfiqurasiya** — WHOIS məxfiliyini Disabled, Always Included və ya Customer Choice kimi təyin edin
* **Müştəri opt-in seçimi** — Customer Choice olaraq təyin edildikdə, checkout işarə qutusu müştərilərə məxfilik qoruması əlavə etməyə imkan verir
* **Konfiqurasiya edilə bilən qiymətləndirmə** — Məxfilik əlavəsi üçün illik qiymət təyin edin və ya onu pulsuz təklif edin
* **Registrator inteqrasiyası** — Qeydiyyat zamanı avtomatik olaraq WhoisGuard (Namecheap) və ya WHOIS məxfiliyini (OpenSRS) aktivləşdirir

#### Vahid Checkout Təcrübəsi {#unified-checkout-experience}
* **Üç domen rejimi** — Müştərilər subdomen (pulsuz), yeni domen qeydiyyatı və ya artıq sahib olduqları mövcud domeni xəritələndirmə arasında seçim edirlər
* **Real vaxt domen axtarışı** — Səhifə yenilənməsi olmadan AJAX əsaslı əlçatanlıq yoxlaması
* **Ağıllı təkliflər** — Üstünlük verilən domen tutulduqda alternativ TLD təklifləri
* **Şəffaf qiymətləndirmə** — Qeydiyyat qiymətinin, yeniləmə qiymətinin və istəyə bağlı WHOIS məxfilik haqqının aydın göstərilməsi
* **Qeydiyyatçı əlaqə sahələri** — Checkout zamanı tələb olunan qeydiyyatçı məlumatlarını (ad, ünvan, telefon) eyni yerdə toplayır
* **Avtomatik yaradılan sayt URL-i** — Sayt ünvanı seçilmiş domendən avtomatik olaraq çıxarılır

#### Avtomatik Yeniləmə Sistemi {#auto-renewal-system}
* **Membership-ə bağlı yeniləmələr** — Müştərinin membership-i aktiv olduğu müddətdə domenlər avtomatik yenilənir
* **Ağıllı planlaşdırma** — Yeniləmə işləri domenin bitmə tarixlərinə əsasən növbəyə qoyulur
* **Təkrar cəhd məntiqi** — Uğursuz yeniləmə cəhdləri üçün ağıllı təkrar cəhdlər
* **Email bildirişləri** — Yaxınlaşan yeniləmələr, uğurlu yeniləmələr, uğursuzluqlar və bitmələr üçün konfiqurasiya edilə bilən xəbərdarlıqlar

#### Müştəri DNS İdarəetməsi {#customer-dns-management}
* **Özünəxidmət DNS** — Müştərilər dashboard-larından DNS qeydlərini (A, AAAA, CNAME, MX, TXT) əlavə edə, redaktə edə və silə bilərlər
* **Domen statusu** — Real vaxt qeydiyyat statusu, bitmə tarixləri və nameserver məlumatı
* **Avtomatik domen xəritələndirməsi** — Qeydiyyatdan keçmiş və ya mövcud domenlər avtomatik olaraq müştərinin saytına xəritələndirilir

#### Admin Alətləri {#admin-tools}
* **Manual domen qeydiyyatı** — Admin panelindən müştərilərin adından domenləri qeydiyyatdan keçirin
* **Domen monitorinqi** — Bütün müştəri domenlərini, statusları və bitmə tarixlərini bir səhifədən görün
* **Provayderə xüsusi jurnallaşdırma** — Hər registrator üçün ayrıca jurnal kanalları (məsələn, domain-seller-namecheap.log)
* **Qeydiyyat detalları** — Əsas domen redaktə səhifəsində qeydiyyatçı məlumatını və qeydiyyat məlumatlarını görün

### Dəstəklənən Provayderlər {#supported-providers}

* **OpenSRS** — Qeydiyyat, yeniləmə, DNS idarəetməsi və WHOIS məxfiliyi ilə tam inteqrasiya
* **Namecheap** — Qeydiyyat, yeniləmə, toplu əlçatanlıq yoxlaması və pulsuz WhoisGuard ilə tam inteqrasiya
* **HostAfrica** — Qeydiyyat, yeniləmə, köçürmələr, DNS idarəetməsi, EPP kodunun əldə edilməsi, registrator kilidi və ID qoruması
* **Openprovider** — Qeydiyyat, yeniləmə, köçürmələr, DNS idarəetməsi, EPP kodunun əldə edilməsi, registrator kilidi, WHOIS məxfiliyi və reseller TLD sinxronizasiyası
* **Hostinger** — Qeydiyyat, domen axtarışı, nameserver yeniləmələri, registrator kilidi və WHOIS məxfiliyi; yeniləmələr Hostinger auto-renew tərəfindən idarə olunur
* **GoDaddy** — Qeydiyyat və yeniləmə
* **ResellerClub** — Qeydiyyat, yeniləmə və DNS idarəetməsi
* **NameSilo** — Qeydiyyat və yeniləmə
* **Enom** — Qeydiyyat, yeniləmə və DNS idarəetməsi

### Texniki Detallar {#technical-details}

* **Tələb olunur:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **Arxitektura:** Domain_Selling_Capability interfeysi ilə Integration Registry nümunəsi
* **Autoloading:** PSR-4 sinif xəritələndirməsi ilə Jetpack autoloader
* **Standartlar:** WordPress kodlaşdırma standartları, strict typing, prepared statements

## Quraşdırma {#installation}

1. **Multisite Ultimate v2.4.12+** quraşdırıldığından və şəbəkə səviyyəsində aktivləşdirildiyindən əmin olun
2. Plugin-i `/wp-content/plugins/ultimate-multisite-domain-seller/` qovluğuna yükləyin
3. WordPress Network Admin-də **Pluginlər** bölməsindən plugin-i şəbəkə səviyyəsində aktivləşdirin
4. **Quraşdırma sehrbazı** ilk aktivləşdirmə zamanı avtomatik işə düşür — bu addımları izləyin:
   * Domen qeydiyyatçınızı qoşun
   * Bağlantını test edin
   * Mövcud TLD-ləri və topdansatış qiymətlərini idxal edin
   * Qiymət əlavəsi ilə ilk domen məhsulunuzu yaradın
5. **Ultimate Multisite > Ödəniş formaları** bölməsində ödəniş formanıza **Domen seçimi** sahəsini əlavə edin

## Tez-tez verilən suallar {#frequently-asked-questions}

### Multisite Ultimate-in hansı versiyası tələb olunur? {#what-version-of-multisite-ultimate-is-required}

Versiya 2.4.12 və ya daha yuxarı. Addon WP Ultimo v1.x və ya 2.4.12-dən aşağı Ultimate Multisite versiyaları ilə uyğun deyil.

### Hansı domen provayderləri dəstəklənir? {#which-domain-providers-are-supported}

Doqquz qeydiyyatçı dəstəklənir: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo** və **Enom**. İmkanlar provayderdən asılı olaraq dəyişir, çünki qeydiyyatçı API-ləri fərqli əməliyyatları təqdim edir; provayderlər üzrə ümumi baxış üçün quraşdırma bələdçisinə baxın. Addon-un İnteqrasiya Reyestri arxitekturası fərdi provayderlər əlavə etməyi asanlaşdırır — [developer bələdçisinə](/developer/integration-guide/custom-registrar) baxın.

### WHOIS məxfilik qoruması təklif edə bilərəm? {#can-i-offer-whois-privacy-protection}

Bəli. Hər domen məhsulunda üç rejimli WHOIS Məxfiliyi ayarı var: Deaktiv, Həmişə daxil edilir və ya Müştəri seçimi. Müştəri seçimi təyin edildikdə, müştərilər ödəniş zamanı qiymətlə (və ya Namecheap-in WhoisGuard-ı üçün "Pulsuz") birlikdə seçim qutusu görürlər. Məxfilik bayrağı qeydiyyat zamanı avtomatik olaraq qeydiyyatçıya ötürülür.

### Fərqli TLD-lər üçün fərqli qiymətlər təyin edə bilərəm? {#can-i-set-different-pricing-for-different-tlds}

Bəli. Fərqli TLD qrupları üçün ayrıca domen məhsulları yaradın. Hər məhsulun öz əlavə qiymət növü (faiz, sabit əlavə qiymət və ya sabit qiymət), dəstəklənən TLD-ləri və giriş qiymətləri ola bilər.

### Müştərilər hansı ödəniş seçimlərini görür? {#what-checkout-options-do-customers-see}

Domen seçimi sahəsi müştərilərə üç tab təqdim edir: **Subdomen** (şəbəkənizdə pulsuz subdomendən istifadə edin), **Yeni domen qeydiyyatdan keçirin** (yeni domen axtarın və qeydiyyatdan keçirin) və **Mövcud domen** (artıq sahib olduqları domeni xəritələndirin). Yalnız aktiv etdiyiniz rejimlər görünəcək.

### Avtomatik yeniləmələri dəstəkləyirmi? {#does-it-support-automatic-renewals}

Bəli. Domenlər avtomatik yenilənməyə ayarlana bilər və müştərinin üzvlük statusuna bağlanır. Yeniləmə ödənişləri müştərinin mövcud ödəniş gateway-i vasitəsilə emal olunur.

### Müştərilər DNS qeydlərini idarə edə bilərlər? {#can-customers-manage-their-dns-records}

Bəli. Müştərilər dashboard-larında A, AAAA, CNAME, MX və TXT qeydlərini əlavə etmək, redaktə etmək və silmək üçün DNS idarəetmə interfeysi əldə edirlər. DNS idarəetməsi OpenSRS, ResellerClub, Enom, HostAfrica və Openprovider ilə mövcuddur. Host edilmiş domenlər üçün Hostinger DNS əsas Hostinger domen-xəritələndirmə inteqrasiyası tərəfindən idarə olunur.

### Quraşdırma sehrbazı varmı? {#is-there-a-setup-wizard}

Bəli. Quraşdırma sehrbazı ilk aktivləşdirmə zamanı avtomatik işə düşür və qeydiyyatçını qoşmaq, TLD-ləri idxal etmək və ilk domen məhsulunuzu yaratmaq üçün sizi addım-addım yönləndirir. Sehrbazı addon ayarlarından yenidən də işə sala bilərsiniz.
