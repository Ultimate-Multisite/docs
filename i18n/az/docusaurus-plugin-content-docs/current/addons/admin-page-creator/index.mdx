# Admin Page Creator

## Content {#content}
Admin Page Creator, WordPress admin dashboard daxilində yeni admin səhifələri və alt-səhifələr yaratmağınıza kömək edəcək. Bu, yeni istifadəçilər üçün başlanğıc səhifəsi kimi xidmət edə biləcək bir xoş gəlmisiniz səhifəsi və ya təlimat səhifəsi yaratmaq üçün istifadə edilə bilər. Səhifələri çətinlik yaşamadan dizayn etmək üçün Beaver Builder, Elementor, Brizy və Oxygen kimi page builder-lərdən istifadə edə bilərsiniz.

Həmçinin, plan və istifadəçi roluna əsaslanan giriş nəzarəti, xüsusi HTML, CSS və JavaScript dəstəyi təklif edir!

## Features {#features}
* Xüsusi HTML və CSS Dəstəyi
* JavaScript Dəstəyi
* PHP Məzmun Rejimi
* İstifadəçi Roluna Əsaslanan İcazə
* Page Builder-lərə Dəstək

## Installation {#installation}

1. **ultimate-multisite-admin-page-creator**-ı **/wp-content/plugins/** kataloquna yükləyin
2. Plugin-i WordPress Network Admin-inizdəki **Plugins** menyusundan aktivləşdirin

## Usage Guide {#usage-guide}

WordPress saytlərin ön hissəsinə (front end) məzmun əlavə etməyi çox asanlaşdırsa da, admin tərəfində səhifələr və alt-səhifələr yaratmaq üçün bəzi əsas kodlama bacarıqları tələb edir. Ultimate Multisite: Admin Page Creator, bu asan istifadə təcrübəsini ön hissədən (front end) şəbəkə saytlarınızın arxa hissəsinə (back end) gətirməyi hədəfləyir.

## Why use Admin Page Creator? {#why-use-admin-page-creator}

Bəlkə müştərilərinizin admin panelində təlimatlar olan xüsusi bir səhifə təklif etmək istəyirsiniz, bəlkə də FAQ və kontakt widget-i olan xüsusi bir səhifə əlavə etmək istəyirsiniz. Admin Page Creator ilə imkanlar sonsuzdur və bu günki təlimatımızda məhz bunu araşdıracağıq.

_Təlimat səhifəsi yaratmaq və buna gömülü YouTube playlist-i əlavə etmək indi mümkündür!_

## Creating your first Custom Admin Page {#creating-your-first-custom-admin-page}

Ultimate Multisite: Admin Page Creator-ı quraşdırıb aktivləşdirdiyinizdən sonra, network admin menyusuna keçin və Admin Pages bölməsinə daxil olun.

![Custom Admin Pages list on Network Admin](/img/admin/admin-pages-list.png)

_Network Admin-dəki Xüsusi Admin Səhifələri siyahısı_

Hazırda yaratdığınız bütün admin səhifələrinin siyahısını görəcəksiniz (bu ilk ziyarətiniz olduğu üçün siyahı boş olacaq). **Add new Admin Page** düymasına klikləyin. Siz Admin Page editor-ə yönləndiriləcəksiniz.

Əgər HTML-ə tanış deyilsinizsə, default WordPress editor-də qalın. Səhifənin məzmununu normal bir Post və ya Page ilə olduğu kimi redaktə edə biləcəksiniz.

## Menu Options {#menu-options}

Tərəfdə, General Options meta-box-unda, menyu növünü, etiketini və daha çox şeyləri konfiqurasiya edə biləcəksiniz.

### Menu Types {#menu-types}

Yeni xüsusi admin səhifəniz iki fərqli menyu növünə malik ola bilər: top-level menyu və alt-menyu.

Adından göründüyü kimi, top-level menyu səhifəsi yeni yaratdığınız səhifəni default WordPress səhifələri ilə yanaşı admin menyusuna əlavə edəcək.

Əgər top-level seçimini edirsinizsə, menyu sırasını və menyu ikonunu da seçə biləcəksiniz.

Əgər bu səhifəni alt-menyu səhifəsi etmək istəyirsinizsə, bu səhifəni əlavə etmək üçün bir ana səhifə (parent page) seçə bilərsiniz. Xüsusi səhifənizi default WordPress səhifələrinə və ya top-level xüsusi admin səhifələrinizə əlavə edə bilərsiniz.

<!-- Screenshot unavailable: Admin Page Creator menu type options with label, order, and icon settings -->

_Bu Xüsusi Səhifə üçün menyu elementini tam şəkildə fərdiləşdirin_

<!-- Screenshot unavailable: Admin Page Creator sub-menu parent page selection -->

_Bu səhifəni öz top-level xüsusi səhifələriniz də daxil olmaqla, digər menyu səhifələrinin alt-menyusu kimi əlavə edə bilərsiniz._

## Advanced Options {#advanced-options}

Xüsusi admin səhifələrinizə hər cür məzmun əlavə edə bilməlisiniz. Buna CSS qaydaları və faylları, JavaScript və xarici kitabxanalar da daxildir. Bunu Advanced Options Tab-dan istifadə edərək edə bilərsiniz.

### Custom CSS {#custom-css}

CSS tab-ında öz xüsusi CSS kodunuzu əlavə edə, həmçinin uzaq saytlardan (CDN-lər, Google Fonts və s.) CSS faylları idxal edə bilərsiniz.

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_Xüsusi CSS də də dəstəklənir. Xarici stil fayllarını da daxil edə bilərsiniz!_

### Custom JavaScript {#custom-javascript}

JavaScript üçün də eyni şey doğrudur. Xüsusi JavaScript kodunuzda istifadə etmək üçün xarici kitabxanalar əlavə edə bilərsiniz.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Öz JavaScript kodunuzu əlavə edin!_

## Permissions {#permissions}

Həmçinin Advanced Options-da hansı alt-saytların bu səhifəyə çıxış hüququ olacağını seçə biləcəksiniz. Bu, məsələn, fərqli plan səviyyələrinə fərqli eksklüziv məzmun təqdim etməyi planlaşdırırsınızsa faydalıdır.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Bu səhifənin harada görünəcəyinə tam nəzarətiniz var._

**Vacib** : Səhifənin görünməsi üçün hər iki şərt ödənmiş olmalıdır. Bu o deməkdir ki, əgər istifadəçi müəyyən bir plana sahib müştəridirsə, lakin seçilmiş rollardan birinə malik deyil (və ya əksinə), səhifə onların panelində görünməyəcək.

## Active {#active}

Xüsusi admin səhifəsini silmədən və ya icazələrlə oynamaqdan çəkinərək deaktiv etmək istəyirsinizsə, sadəcə Active seçimini işarələməyin.

Bir top-level səhifəni deaktiv etmək, ona bağlı olan bütün alt-səhifələri avtomatik olaraq deaktiv edəcək.
