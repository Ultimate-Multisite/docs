# Admin Page Creator Dəyişikliklər Siyahısı

### Versiya 1.9.0 - 2026-01-18-də buraxılıb

* Düzəldildi: Aktivasiya zamanı yaranan kritik səhvlər.
* Yeniləndi: Content source sinifləri, düzgün valideyn konstruktor ilkinləşdirilməsi ilə Singleton trait-dən istifadə etməsi üçün yeniləndi.
* Düzəldildi: Content type selector-un görünməməsinə səbəb olan Vue.js template kompilyasiya səhvləri.
* Düzəldildi: Missing Vue data properties (template_id, external_link_url_embedable, external_link_url_checking) çatışmazlığı.
* Düzəldildi: Sayfa yüklənərkən TinyMCE redaktorunun düzgün işə düşməməsi.
* Düzəldildi: Divi dəstəkləyici faylının Composer vasitəsilə avtomatik yüklənməməsi.
* Düzəldildi: Divi dəstəkləyici sinifində korrupsiya olmuş PHP sintaksisi.
* Təkmilləşdirildi: JavaScript asset təşkilatı və minifikasiyası.
* Təkmilləşdirildi: Ən son WordPress və page builder versiyaları ilə uyğunluq.
