# Filter: signup_get_available_languages

Ön tərəf saytına qeydiyyat üçün mövcud dillər siyahısını filtrləyir.

Bu hook-a boş bir massiv ötürmək, qeydiyyat formasında ayarın görünməsini deaktiv edir və sayt yaradılarkən standart dil istifadə olunacaq. Əvvəlcədən quraşdırılmamış dillər siyahıdan çıxarılacaq.

## Parametrlər

| Name | Type | Təsvir |
|------|------|-------------|
| $available_languages | `array` | Mövcud dillər. |

### Nə vaxtdan

- 4.4.0
