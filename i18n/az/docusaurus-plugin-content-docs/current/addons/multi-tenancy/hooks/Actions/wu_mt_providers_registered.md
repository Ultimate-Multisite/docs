# Action: wu_mt_providers_registered

Default providerlar qeydiyyata alındıqdan sonra işə düşür.

Bu hook-dan istifadə edərək öz fərdi (custom) providerları qeydiyyata ala bilərsiniz.

## Parametrlər {#parameters}

| Ad | Tip | Təsvir |
|------|------|-------------|
| $manager | `\Provider_Manager` | Provider meneceri nümunəsi (instance). |

- 1.0.0
