# Fluent Forms

Bu add-on, Multisite Ultimate-də məhsullar, üzvlüklər və ya saytlar üzrə Fluent Forms-a limitlər əlavə etməyə imkan verir.
Fluent Forms, custom post type-lar əvəzinə custom cədvəllərdən istifadə etdiyi üçün, normal limitləmə seçimləri Fluent Forms ilə işləmir.

## Xüsusiyyətlər

- Fluent Forms və ya Conversational Fluent Forms üçün limitlər əlavə edin
- Müştərilər limitə çatdıqda, planlarını yüksəltməyə təşviq edən dostça bir mesaj alırlar.
