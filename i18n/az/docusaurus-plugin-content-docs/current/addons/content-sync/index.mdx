# Məzmun Sinxronizasiyası {#content-sync}

Multisite Ultimate: Məzmun Sinxronizasiyası

Bu addon, Multisite Ultimate üçün nəzərdə tutulub və şəbəkə administratorlarına istənilən saytdan şəbəkədəki digər saytlara məzmun sinxronizasiya etməyə imkan verir. Postlar, səhifələr, fərdiləşdirilmiş post tipləri və Elementor şablonları daxil olmaqla bütün məzmun növlərini dəstəkləyir. Addon, məzmun və hədəf saytları seçmək üçün çoxaddımlı bir interfeys (wizard) təqdim edir; bu interfeys toplu emal (batch processing) və real vaxtda irəliləmə göstəricisi ilə yanaşı gəlir.

## Xüsusiyyətlər {#features}

* **Məzmun Seçimi:** Miniatürlər və post tipi filtrləməsi ilə vizual kart paneli.
* **Bütün Post Tipləri:** Postları, səhifələri, məhsulları, Elementor şablonlarını və istənilən fərdiləşdirilmiş post tipini sinxronizasiya edin.
* **Sayt Seçimi:** Məzmunun hansı saytlara sinxronizasiya ediləcəyini seçin; mənbə saytın istisna edilməsi və uşaq saytların əvvəlcədən seçilməsi funksionallığı var.
* **Toplu Emal:** İrəliləmə yeniləmələri ilə birlikdə bir dəfəyə 10 sayta sinxronizasiya edir.
* **Media Sinxronizasiyası:** Post məzmunundan hədəf saytlara media fayllarını (şəkillər, videolar) avtomatik köçürür və yenidən yerləşdirir (remaps).
* **Taksonomiya Sinxronizasiyası:** Kateqoriyaları, etiketləri və fərdiləşdirilmiş taksonomiyaları saxlayır.
* **Elementor Dəstəyi:** CSS yenidən generasiyası və şablon ID-sinin yenidən yerləşdirilməsi (remapping) ilə Elementor şablonları üçün tam dəstək.
* **Konfliktlərin Həlli:** Mövcud məzmunu əvəz etmək və ya onu atlamaq (skip) seçimi.
* **Keş İdarəetməsi:** Sinxronizasiyadan sonra hədəf saytlardakı Elementor keşini avtomatik təmizləyir.

## Tələblər {#requirements}

* WordPress 5.3 və ya daha yuxarı
* PHP 7.4 və ya daha yuxarı
* Multisite Ultimate plugin (aktiv)
* Elementor plugin (isteğe bağlı, Elementor şablonu sinxronizasiyası üçün tələb olunur)

## Quraşdırma {#installation}

1. 'ultimate-multisite-content-sync' faylını '/wp-content/plugins/' kataloquna yükləyin.
2. Plugin-i WordPress Network Admin-dəki 'Plugins' menyusu vasitəsilə aktivləşdirin.
3. Ultimate Multisite → Settings → Content Sync yerində parametrləri təyin edin.
4. Sinxronizasiyaya başlamaq üçün Ultimate Multisite → Content Sync səhifəsinə keçin.
