# Content Sync Değişiklik Qeydi

### Version 1.1.0 - Yayınlanma tarixi: 2025-12-31 {#version-110---released-on-2025-12-31}
* Yeni: Bütün post tiplərinin (posts, pages, xüsusi göstri növləri) senkronizasiya dəstəyi
* Yeni: Content seçimi addımında post tipi filtrləməsi
* Yeni: Mənbə saytı seçimi addımı - yalnız əsas saytdan deyil, istənilən saytdan senkronizasiya
* Yeni: Şablon saytları üçün uşaq saytın ilkin seçilməsi
* Yeni: Bütün post tipləri üçün post_content-dən media çıxarılması
* Yeni: Taksonomiya senkronizasiyası (kateqoriyalar, taglər, xüsusi taksonomiyalar)
* Değişdirildi: "Elementor Template Sync" adından "Content Sync" adına dəyişdirildi
* Değişdirildi: "Select Templates" addımı "Select Data" adına dəyişdirildi
* Təkmilləşdirildi: Elementor-ə xas kod indi yalnız elementor_library post tipində işləyir

### Version 1.0.0 - Yayınlanma tarixi: 2025-12-19 {#version-100---released-on-2025-12-19}
* İlkin buraxılış
* Şablon və sayt seçimi üçün bir neçə addımlı siqil (wizard)
* Real vaxtda irəliləmə geri bildirimi ilə yığın (batch) emalı
* Avtomatik ID yenidən xəritələnməsi ilə tam media senkronizasiyası
* Ultimate Multisite ilə ayarlar inteqrasiyası
* Network ayarları vasitəsilə sayt istisna etmə dəstəyi
* Senkronizasiyadan sonra Elementor keşinin təmizlənməsi
* Şablon taksonomiyasının qorunması
