# Ölçülən Planlar Değişiklik Qeydi {#metered-plans-changelog}

Versiya 1.1.0 - Yayınlanma tarixi 2026-05-05
- Yeni: Multisite alt saytları üçün AI token əsaslı ödəniş — konfiqurasiya edilə bilən hər token üçün qiymətlərlə müştəri saytları boyunca AI token istifadəsinin izlənilməsi və ödənilməsi.
- Yeni: Connector tətbiqi, dinamik limit aşkarlanması və write-through ilə yeniləndi, bu da bütün connectorlarda real vaxt dəqiqliyini təmin edir.
- Düzəliş: Verilənlər bazası cədvəli yeniləmələri indi BerlinDB şema tərifləri ilə düzgün uyğunlaşır, bu da yeni quraşdırılmalarda yeniləmə uğursuzluqlarını aradan qaldırır.
- Düzəliş: Verilənlər bazası yeniləmə callback funksiyaları düzgün formatda çevrildi, bu da sessiz yeniləmə uğursuzluqlarını həll edir.
- Düzəliş: AI Usage Overage Markup giriş sahəsində kəsr dəyərlər qəbul edilir.
- Düzəliş: Plugin yüklənməsi zamanı kritik səhvlər və iki dəfə ilkinləşdirmə problemləri aradan qaldırıldı.
- İyileşdirildi: Developer mühitlərinin ilk işlənmədə özünü konfiqurasiya etməsini təmin edən check-env npm script əlavə edildi.
