# Material WP {#material-wp}

WordPress admin interfeysi çoxdan eyni qalır. Amma Material Design versiyası ilə admin interfeysinin necə görünə biləcəyini düşündünüz mü?

Material WP, sizə bu görünüşü və hissi vermək üçün hazırlanmış bir admin temasıdır. Bu, müştərilərinizə daha fərdiləşdirilmiş bir Dashboard təqdim etməyinizə kömək edir.

## Xüsusiyyətlər {#features}

- Xüsusi CSS dəstəyi
- Sadə stil ayarları
- Menyunu yenidən sıralamaq
- Asan import/export

## Dokumentasiya {#documentation}

**Material WP ilə Admin Panelinizi Fərdiləşdirmək**

## Quraşdırma {#installation}

1. 'material-wp'-ni '/wp-content/plugins/' kataloquna yükləyin
2. WordPress-də 'Plugins' menyusu vasitəsilə plugin-i aktivləşdirin

## İstifadə Təlimatı {#usage-guide}

WordPress admin interfeysi çoxdan eyni qalır. Amma Material Design versiyası ilə admin interfeysinin necə görünə biləcəyini düşündünüz mü?

Material WP, sizə bu görünüşü və hissi vermək üçün hazırlanmış bir admin temasıdır. Bu, müştərilərinizə daha fərdiləşdirilmiş bir Dashboard təqdim etməyinizə kömək edir. Material WP həmçinin Ultimate Multisite-ın premium plugins core hissəsidir.
