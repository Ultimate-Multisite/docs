# WooCommerce Integration Değişiklik Qeydi

Version 2.0.6 - Yayınlanma tarixi 2026-01-16
* İyileşdirmə: Əsas abunəlikləri addon-a daxil etdi. Artıq WooCommerce Subscriptions extension-ına ehtiyac yoxdur.

Version 2.0.5 - Yayınlanma tarixi 2026-01-09
* İyileşdirmə: Tərcümələri glotpress API-dən yükləyir.
* Düzəliş: Bəzi sayfa qurucularında kritik xəta.
* Düzəliş: Müştəri əsas saytın üzvü olduqda sonsuz yönləndirmə.

Version 2.0.4 - Yayınlanma tarixi 2025-11-14
* Əlavə edildi: Daha çox dillər üçün tərcümələr.
* Dəyişdirildi: Adı Ultimate Multisite: Woocommerce Integration olaraq dəyişdirildi.
* Əlavə edildi: Woocommerce 10.2.1 ilə uyğunluq.
* Əlavə edildi: Woocommerce Subscriptions 7.7.0 ilə uyğunluq.
* Düzəliş: PHP 8.4 ilə uyğunluq
* Düzəliş: WC hesab səhifəsi mövcud olmadıqda yönləndirmə.

Version 2.0.3 - Yayınlanma tarixi 2025-08-13
* Dəyişdirildi: Yeni marketplace ilə avtomatik yeniləmələr aktivləşdirildi.

Version 2.0.2 - Yayınlanma tarixi 2025-07-05
* Dəyişdirildi: Adı Multisite Ultimate: Woocommerce Integration olaraq dəyişdirildi.
* Əlavə edildi: Woocommerce 9.8.1 ilə uyğunluq.
* Əlavə edildi: Woocommerce Subscriptions 7.3.0 ilə uyğunluq.
* Düzəliş: Müştəri tərəfindən abunəliyin ləğvi.
* Düzəliş: Checkout bloku istifadə edərkən kritik xəta.
* İyileşdirmə: İndi WooCommerce yüksək performanslı fərdiləşdirilmiş sifariş cədvəlləri ilə uyğundur.
* Düzəliş: WooCommerce checkout-da ləğv etmək hələ də üzvlüyü yüksəltə bilər.

Version 2.0.1 - Yayınlanma tarixi 2023-08-09

* Əlavə edildi: Woocommerce 7.9.0 ilə uyğunluq.
* Əlavə edildi: Woocommerce Subscriptions 5.3.0 ilə uyğunluq.
* Əlavə edildi: Üzvlük yeniləmələri üçün dəstəklənmə.
* Əlavə edildi: Woocommerce-da sınaq və qurulma haqları haqqında bildirişlər.
* Əlavə edildi: Ultimate Multisite Woocommerce məhsullarının meta dəyəri ilə müəyyən edilməsi.
* Əlavə edildi: Bütün Ultimate Multisite ilə bağlı Woocommerce məhsullarını işarələmək üçün bir dəfəlik düzəliş yerləşdirildi.
* Əlavə edildi: Ultimate Multisite tərəfindən yaradılan məhsullar Woocommerce siyahısından çıxarıldı.
* İyileşdirmə: Səbətə tətbiq ediləcək təkrarlanmayan bir Woocommerce endirimi yaradıldı.
* İyileşdirmə: Təkrarlanan endirimi Woocommerce məhsuluna bərpa edildi.
* İyileşdirmə: Woocommerce məhsuluna təkrarlanan endirim etiketi əlavə edildi.
* İyileşdirmə: Checkout-da məhsul növünün təmin edilməsi.
* Düzəliş: Aşağı düşmə prosesi zamanı üzvlük statusu qorundu.
* Düzəliş: Ləğv etmə prosesi zamanı xətaları aradan qaldırmaq üçün abunəlik mövcud olub-olmadığı yoxlanıldı.
* Düzəliş: Woocommerce abunəliklərində istifadə üçün başlanğıc abunəlik tarixi əlavə edildi.
* Daxili: Yeni PHP 8.1 quruluş prosesi tətbiq edildi.

Version 2.0.0 - Tam yenidən yazıldı.

* Əlavə edildi: Gateway dəyişdirildikdə və ya üzvlük ləğv edildikdə woo abunəliyini çıxaran proses ləğv etmə metodu;
* Əlavə edildi: Üzvlükləri aşağı və yuxarı düşürmək üçün handler;
* İyileşdirmə: Hesab yeniləməyə imkan vermək üçün subsitlərdə müştəri yeniləmə forması üzrə woocommerce asılılıqları yüklənir;
* İyileşdirmə: Woocommerce səbəti mövcud deyilsə düzgün yüklənməsi;
* İyileşdirmə: Checkout prosesi zamanı əsas sayt cədvəllərində olduğumuzdan əmin edilməsi;
* İyileşdirmə: Ultimo yeniləmə sifarişini Woocommerce abunəlik sifariş dəyərinə əsaslandırmaq, son ödənişdən deyil;
* Düzəliş: WU Membership düyməsinə keçid;
* Düzəliş: Woocommerce abunəliklərinin yenilənməsi ödənişli olduqda Ultimo sifarişini ödənişli olaraq təyin etmək;
* Quruluş: MPB-ni qurucu kimi əlavə etmək;
