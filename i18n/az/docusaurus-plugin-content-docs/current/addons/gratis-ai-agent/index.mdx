# Gratis AI Agent

Gratis AI Agent, təbii dil təlimatları vasitəsilə tam WordPress saytları planlaşdıra, qurara və idarə edə bilən avtonom bir WordPress AI köməkçisidir. O, məzmun idarəetməsindən dizayn sisteminin fərdiləşdirilməsinə qədər hər şeyi əhatə edən, agentin çağıra biləcəyi ayrılmış, test edilə bilən hərəkətlər dəsti olan **qabiliyyətlər** (abilities) təqdim edir. 1.9.0 versiyası məzmun yaratma qabiliyyətlərini, müştəri tərəfindən alınan skrinşotlar vasitəsilə vizual səhifə nəzərdən keçirməni, beş daxili agent profilini və giriş nəzarəti və brendləmə üçün xüsusiyyət bayraqlarını əlavə edir.

## Əsas Xüsusiyyətlər {#key-features}

- **Qabiliyyətlər sistemi** — agentin WordPress quraşdırmanızda çağıra biləcəyi modulyar, genişləndirilə bilən bir hərəkətlər kataloqu
- **Məzmun İdarəetməsi** — məqalələr və səhifələr yaratmaq (səhifə şablonlarının təyin edilməsi daxil olmaqla), bir çağrıda bir neçə məqalə yaratmaq, xüsusi şəkillər təyin etmək və əlaqə formaları qurmaq
- **Vizual Nəzərdən Keçirmə** — canlı səhifələrin skrinşotlarını çəkmək, əvvəl/sonra vəziyyətlərini müqayisə etmək və yerləşmə, tipoqrafiya, rəng və əlçatanlıq sahələrini əhatə edən AI tərəfindən yaradılmış dizayn nəzərdən keçirməsi almaq
- **Beş daxili agent** — Məzmun Yazarı, Sayt Qurucusu, Dizayn Studiyası, Plugin Meneceri və Dəstək Asistanı; chat başlığındakı Agent Seçici vasitəsilə dəyişdirilə bilər
- **Xüsusiyyət bayraqları** — Rol məhdudiyyətləri, white-label adlandırma və widget fərdiləşdirilməsi üçün Settings → Feature Flags-də giriş nəzarəti və brendləmə düymələri
- **Custom Post Type idarəetməsi** — Yenidən başladılmasında da davamlılığı qoruyan fərdiləşdirilmiş məqalə növləri yaratmaq, siyahılaşdırmaq və silmək
- **Custom Taxonomy idarəetməsi** — Yenidən başladılmasında da davamlılığı qoruyan fərdiləşdirilmiş taksonomiyalar yaratmaq, siyahılaşdırmaq və silmək
- **Dizayn Sistemi qabiliyyətləri** — fərdiləşdirilmiş CSS yerləşdirmək, küratörlənmiş blok nümunələrini idarə etmək, sayt loqosunu təyin etmək və theme.json presetlərini tətbiq etmək
- **Global Styles idarəetməsi** — rəng palitraları, tipoqrafiya, boşluq və s. üçün theme.json dəyərlərini oxumaq və yazmaq
- **Naviqasiya Menyusu idarəetməsi** — naviqasiya menyularını və onların elementlərini proqramlaşdırılmış şəkildə yaratmaq və yeniləmək
- **Options İdarəetməsi** — Kritik parametrləri qorumaq üçün daxili təhlükəsizlik siyahısı ilə WordPress parametrlərini oxumaq və yazmaq
- **Quraşdırıla Bilən Qabiliyyətlər Reyestri** — əsas plugin fayllarını dəyişdirmədən əlavə qabiliyyət paketlərini kəşf edib aktivləşdirmək
- **Site Builder Orchestration v2** — plugin kəşfi, inkişaf izlənməsi və səhv bərpası ilə bir neçə addımlı sayt qurma planları
- **WP-CLI benchmark komandası** — CI və model qiymətləndirilməsi üçün agentin bacarıq qiymətləndirmə dəstini komanda xəttindən işə salmaq

## Tələblər {#requirements}

- WordPress 6.0 və ya daha yuxarı
- PHP 8.1 və ya daha yuxarı
- WordPress Multisite (şəbəkə səviyyəsində aktivləşdirilmiş) və ya tək sayt quraşdırması
- Aktiv bir AI provider connector plugin (Anthropic, OpenAI və ya uyğun)

## Quraşdırma {#installation}

1. Plugin fayllarını `/wp-content/plugins/gratis-ai-agent` yoluna yükləyin
2. WordPress-də **Plugins** ekranı vasitəsilə plugin-i aktivləşdirin
3. **Gratis AI Agent → Settings** səhifəsinə keçin və AI provider-ınızı qoşun
4. (İxtiyari) Əlavə qabiliyyət paketlərini quraşdırmaq üçün **Abilities Registry**-də gəzin

## Tez Başlanğıc {#quick-start}

Aktivləşdirildikdən və bir AI provider-a qoşulduqdan sonra, WordPress admin yan çubuğunda **Gratis AI Agent** panelinə daxil olun. Təbii dil təlimatı yazın və agent mövcud qabiliyyətlərindən istifadə edərək tələb olunan addımları planlaşdıracaq və yerinə yetirəcək.

**Nümunə təlimatlar:**

```
"Portfolio" adlı fərdiləşdirilmiş məqalə növü və "Project Category" taksonomiyası yaradın.
```

```
Tema üçün qaranlıq rejim rəng palitrası yerləşdirin və əsas rəngi #1a1a2e təyin edin.
```

```
Online menyu, rezervasiya forması və əlaqə səhifəsi olan bir restoran veb saytı qur.
```

## Qabiliyyətlər Baxışı {#abilities-overview}

Qabiliyyətlər agentin yerinə yetirə biləcəyi atomar hərəkətlərdir. Hər bir qabiliyyət, agentin işləmə zamanı oxuduğu bir sxem (parametrlər və qaytarma növləri) təqdim edən qeyd edilmiş bir PHP sinifidir. Qabiliyyətlər funksional sahələrə ayrılır:

| Sahə | Qabiliyyətlər |
|---|---|
| **Məzmun İdarəetməsi** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **Vizual Nəzərdən Keçirmə** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **Custom Post Types** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **Custom Taxonomies** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **Dizayn Sistemi** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **Global Styles** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **Naviqasiya Menyuları** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **Options İdarəetməsi** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **Quraşdırıla Bilən Qabiliyyətlər** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **Site Builder** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

Tam parametrlər sənədləşməsi və istifadə nümunələri üçün [Abilities Reference](./abilities) səhifəsinə baxın.

## WP-CLI {#wp-cli}

Gratis AI Agent, benchmarklar işlətmək və qabiliyyətləri komanda xəttindən idarə etmək üçün `wp gratis-ai-agent` komanda ailəsi ilə gəlir. Tam komanda siyahısı üçün [WP-CLI Reference](./wp-cli) səhifəsinə baxın.

```bash
# Bütün benchmark dəstini işə sal
wp gratis-ai-agent benchmark

# Quraşdırılmış qabiliyyətləri siyahıla
wp gratis-ai-agent abilities list

# Bir qabiliyyət paketini quraşdır
wp gratis-ai-agent abilities install <slug>
```

## Konfiqurasiya {#configuration}

Bütün parametrlər WordPress admin panelində **Gratis AI Agent → Settings** altında yerləşir:

| Parametr | Təsvir |
|---|---|
| **AI Provider** | Aktiv AI provider connector-ı seçin və konfiqurasiya edin |
| **Model** | Agent sessiyaları üçün istifadə ediləcək dil modelini seçin |
| **Max Tokens** | Hər sorğu üçün token büdcəsi (default 4096) |
| **Ability Timeout** | Bir qabiliyyətin işləmə biləcəyi maksimum saniyə (default 30) |
| **Debug Logging** | Qabiliyyət çağırışlarını və agent məntiqini WordPress debug log-una yazmaq |

## Options İdarəetməsi Təhlükəsizlik Siyahısı {#options-management-safety-blocklist}

Options Management qabiliyyəti, agentin həssas WordPress parametrlərini oxumasını və ya yazmasını əngəlləyən daxili bir təhlükəsizlik siyahısı ehtiva edir. Default siyahı aşağıdakıları əhatə edir:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (URL parametrləri — bunlar üçün xüsusi miqrasiya aləti istifadə edin)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (gizli açarlar)
- `active_plugins`, `template` (plugin/tema aktivləşdirilməsi — xüsusi qabiliyyətlərdən istifadə edin)
- Quraşdırılmış pluginlər tərəfindən qeyd edilmiş ödəniş brauzeri və API açarları

Əlavə girişlər `gratis_ai_agent_options_blocklist` filtri vasitəsilə əlavə edilə bilər.

## Hooks və Filters {#hooks-and-filters}

### Actions (Hərəkətlər) {#actions}

- `gratis_ai_agent_ability_registered` — bir qabiliyyət qeyd edildikdən sonra işə düşür, qabiliyyət sinif adını alır
- `gratis_ai_agent_plan_started` — sayt qurucu planı icra etməyə başladığı zaman işə düşür
- `gratis_ai_agent_plan_completed` — plan uğurla başa çatdıqda işə düşür
- `gratis_ai_agent_plan_failed` — plan bərpa edilə bilməyən bir səhvə rastlaşdıqda işə düşür

### Filters (Filtrlər) {#filters}

- `gratis_ai_agent_abilities` — agent yükləməzdən əvvəl qeyd edilmiş qabiliyyətlər siyahısını dəyişdirir
- `gratis_ai_agent_options_blocklist` — parametrlər təhlükəsizlik siyahısını genişləndirir və ya azaldır
- `gratis_ai_agent_benchmark_questions` — qiymətləndirmə dəstisinə xüsusi benchmark sualları əlavə edir
- `gratis_ai_agent_plan_plugins` — sayt planı yaradılarkən plugin kəşfi siyahısını dəyişdirir
