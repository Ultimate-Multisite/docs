# PayFast

Multisite Ultimate: Payfast Gateway

PayFast - Cənubi Afrikadakı sürətli və təhlükəsiz onlayn ödəniş qapısıdır - bu qapını Multisite Ultimate ödəniş seçimi kimi əlavə edir.

## Quraşdırma (Installation)

1. 'ultimate-multisite-payfast' faylını '/wp-content/plugins/' kataloquna yükləyin.
2. WordPress-də 'Plugins' menyusu vasitəsilə plugin-i aktivləşdirin.
3. Plugin-i qurmaq üçün addım-addım Wizard-dan keçin.

## İstifadə Təlimatı (Usage Guide)

PayFast Cənubi Afrikadakı bizneslər üçün əsas onlayn ödəniş emalçısıdır. O, əməliyyatları problemizənsiz və təhlükəsiz edərək onlayn ticarətçilərin böyüməsinə kömək etməyə yönəlmişdir. Ultimate Multisite istifadəçisinin tələbi ilə, biz PayFast ilə inteqrasiyanı mümkün edəcək bir əlavə (addon) yaratmağa qərar verdik. Bu məqalədə, PayFast-ı Ultimate Multisite şəbəkənizdə ödəniş almaq üçün ödəniş qapınız kimi necə quracağınızı göstəririk.

İlk addım əlavəni (add-on) quraşdırmaqdır.

# Payfast əlavəsinin necə quraşdırılması:

Add-ons səhifəsinə keçin və Ultimate Multisite: Payfast Gateway əlavəsini seçin.

![Add-ons page](/img/admin/addons-page.png)

Add-on pəncərəsi açılacaq. "Install Now" düyməsinə klikləyin.

![Add-ons page with install option](/img/admin/addons-page.png)

Quraşdırma tamamlandıqdan sonra, plugins səhifəsinə yönləndiriləcəksiniz. Burada sadəcə "Network Activate" düyməsinə klikləyin və _Payfast_ əlavəsi şəbəkənizdə aktivləşdiriləcək.

# PayFast-ı Qurmaq (Setting Up PayFast)

Add-onu qurduqdan sonra, indi Ultimate Multisite parametrlərinə keçə və PayFast-ı konfiqurasiya etməyə başlaya bilərsiniz.

**Ultimate Multisite settings** səhifəsi altında "Payments" (Ödənişlər) tabını tapa bilərsiniz. Bu tab altında, ödəniş qapısını aktiv və ya deaktiv etmək üçün bir düymə ilə birlikdə "**Payfast**" hissəsini görməlisiniz.

![Payment gateway settings](/img/config/settings-payment-gateways.png)İndi isə onu aktivləşdirmək üçün düyməyə klikləyin. Bu, daha sonra **Merchant ID, Merchant Key və PassPhrase** kimi doldurmanız lazım olan müxtəlif sahələri göstərən bir qutu açacaq. Bu məlumatları **Payfast** hesabınızdan əldə edə bilərsiniz.

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Növbəti addım lazım olan məlumatları almaq üçün PayFast hesabınıza daxil olmaqdır (login).

Daxil olduqdan sonra, **Payfast** hesabınızın dashboard-undan lazım olan giriş məlumatlarını (credentials) dərhal görə bilərsiniz.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->Sadəcə bu məlumatları kopyalayıb yapışdırın və Ultimate Multisite parametrlərinizdə detalları doldurun. Sonra dəyişiklikləri saxlamaq üçün **"Save settings** " düyməsinə klikləyin.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Bu qədər, PayFast ödəniş inteqrasiyanız tamamlandı!
