# Superdav AI Agent

Superdav AI Agent, Gratis AI Agent-i tema yönümlü bacarıqlar, təkmilləşdirilmiş internet axtarışı imkanları, sayt qurucu xüsusiyyətləri və istəyə bağlı avtomatlaşdırma inteqrasiyaları ilə genişləndirən qabaqcıl WordPress AI köməkçisidir. 1.18.0 versiyası Google Calendar oxuma alətləri, kontakt və iştirakçı uyğunlaşdırması, insan təsdiqi mərhələləri, xatırlatma deduplikasiyası, TextBee SMS bildirişləri, idarə olunan Superdav xidmətinin təmin edilməsi və etibarlı tərtibatçı iş axınları üçün ayrıca paylanan Advanced tamamlayıcı paketini əlavə edir.

## Əsas Xüsusiyyətlər

- **Tema yönümlü bacarıqlar** — rəhbərliyi və imkanları aktiv WordPress temanıza və quraşdırılmış plaginlərə əsasən avtomatik uyğunlaşdırır
- **Çoxsaylı internet axtarış təminatçıları** — daha zəngin axtarış nəticələri və qabaqcıl araşdırma imkanları üçün Brave Search və Tavily arasında seçim edin
- **Sayt qurucu kontakt formaları** — quraşdırılmış forma plagininizdən istifadə edərək birbaşa çat interfeysindən tam işlək kontakt formaları yaradın
- **Təkmilləşdirilmiş sayt qurucu** — daha yaxşı plagin aşkarlanması və xəta bərpası ilə yaxşılaşdırılmış çoxmərhələli sayt qurma planları
- **Daxili tema bacarıqları** — Block Themes, Classic Themes, Kadence Blocks və Kadence Theme üçün ixtisaslaşmış rəhbərlik
- **Təkmilləşdirilmiş WooCommerce inteqrasiyası** — daha yaxşı etibarlılıq və uyğunluq üçün yerli WooCommerce API-lərindən istifadə edəcək şəkildə yenidən işlənib
- **Avtomatik təminatçı aşkarlanması** — plaginlər və temalar aktivləşdirildikdə və ya deaktiv edildikdə mövcud plaginləri və temaları avtomatik aşkar edir və yeniləyir
- **Fərdi loqo yaradılması** — avtomatik namespace üçün təhlükəsiz sanitizasiya ilə fərdi SVG loqoları yaradın və yerləşdirin
- **Palitra kontrastının yoxlanması** — temanıza tətbiq etməzdən əvvəl rəng cütlərini WCAG əlçatanlıq uyğunluğu üçün yoxlayın
- **Foto ilə gücləndirilmiş kəşf** — dizayn istiqamətinə təsir etmək üçün Theme Builder kəşf müsahibəsi zamanı fotolar yükləyin
- **Qonaqpərvərlik menyuları** — restoranlar, kafelər və qonaqpərvərlik biznesləri üçün strukturlaşdırılmış yemək və içki menyusu səhifələri yaradın
- **Masaüstü və mobil önizləmələr** — dizayn istiqaməti seçimi zamanı dizaynınızı masaüstü və mobil cihazlarda önizləyin
- **Tier 1 sayt scraping** — rəqib və ilham saytlarını təhlil etmək üçün sd-ai-agent/site-scrape indi standart olaraq əlçatandır
- **Google Calendar oxuma alətləri** — avtomatlaşdırmaların yaxınlaşan cədvəl konteksti haqqında nəticə çıxara bilməsi üçün konfiqurasiya edilmiş təqvimləri və tədbirləri oxuyun
- **Kontakt və iştirakçı uyğunlaşdırması** — izləmə işi yaratmazdan əvvəl təqvim iştirakçılarını məlum sayt istifadəçiləri və kontaktlarla uyğunlaşdırın
- **İnsan təsdiqi mərhələləri** — mesajlar göndərməzdən, işi davam etdirməzdən və ya dəyişikliklər etməzdən əvvəl həssas avtomatlaşdırmaları nəzərdən keçirmə üçün dayandırın
- **Xatırlatma qeydləri və deduplikasiya** — artıq göndərilmiş xatırlatmaları izləyin ki, təkrar avtomatlaşdırma icraları eyni alıcıya iki dəfə bildiriş göndərməsin
- **TextBee SMS bildirişləri** — SMS təminatçısı aktiv olduqda TextBee vasitəsilə konfiqurasiya edilmiş mətn mesajı bildirişləri göndərin
- **İdarə olunan Superdav xidmət təminatı** — dəstəklənən saytları əl ilə endpoint konfiqurasiyası əvəzinə avtomatik quraşdırma ilə host edilən Superdav endpoint-lərinə qoşun
- **Advanced tamamlayıcı paketi** — fayl sistemi, verilənlər bazası, WP-CLI, REST dispatcher, plagin qurucusu, git snapshot, istifadəçi idarəetməsi və benchmark iş axınları üçün ayrıca paylanan etibarlı tərtibatçı alətləri

## Tələblər

- WordPress 6.0 və ya daha yuxarı
- PHP 8.1 və ya daha yuxarı
- WordPress Multisite (şəbəkə üzrə aktivləşdirilmiş) və ya tək-sayt quraşdırması
- Aktiv AI təminatçı konnektor plagini (Anthropic, OpenAI və ya uyğun)
- Gratis AI Agent v1.9.0 və ya daha yuxarı (Superdav AI Agent, Gratis AI Agent-i genişləndirir)

## Quraşdırma

1. Plagin fayllarını `/wp-content/plugins/superdav-ai-agent` ünvanına yükləyin
2. Plagini WordPress-də **Plaginlər** ekranı vasitəsilə aktivləşdirin
3. Gratis AI Agent-in də aktivləşdirildiyinə əmin olun
4. **Gratis AI Agent → Settings** bölməsinə keçin və ehtiyac olduqda internet axtarış təminatçınızı, idarə olunan Superdav xidmət bağlantısını, Google Calendar və ya TextBee məlumatlarını konfiqurasiya edin

WordPress.org paketi əsas Superdav AI Agent xüsusiyyətlərini ehtiva edir. Advanced tamamlayıcı paketi etibarlı tərtibatçı alətlərinin açıq şəkildə quraşdırıldığı və səlahiyyətləndirildiyi mühitlər üçün ayrıca qurulur və paylanır.

## Tez Başlanğıc

Aktivləşdirildikdən sonra Superdav AI Agent, Gratis AI Agent quraşdırmanızı tema yönümlü bacarıqlar və əlavə imkanlarla avtomatik təkmilləşdirir. WordPress admin yan panelində **Gratis AI Agent** panelini açın və yeni xüsusiyyətlərdən istifadə etməyə başlayın.

**Nümunə təlimatlar:**

```
Kontakt səhifəsinə kontakt forması əlavə edin.
```


```
Aktiv temanın blok nümunələrindən istifadə edərək hero bölməsi yaradın.
```

```
Sabahkı konfiqurasiya edilmiş Google Calendar tədbirlərini nəzərdən keçirin və iştirakçılar üçün xatırlatmaların qaralamasını hazırlayın.
```

## Tema Yönümlü Bacarıqlar

Superdav AI Agent aktiv temanızı və quraşdırılmış plaginlərinizi avtomatik aşkar edir, uyğun tema yönümlü bacarığı köməkçinin kontekstinə yükləyir. Mövcud bacarıqlara daxildir:

| Bacarıq | Tətbiq Edilir |
|---|---|
| **Block Themes** | WordPress blok əsaslı (Full Site Editing) arxitekturasından istifadə edən temalar |
| **Classic Themes** | PHP şablonlarından və klassik redaktordan istifadə edən ənənəvi WordPress temaları |
| **Kadence Blocks** | Kadence Blocks plagini aktiv olan saytlar |
| **Kadence Theme** | Aktiv tema kimi Kadence theme olan saytlar |

Hər bacarıq üzrə ətraflı rəhbərlik üçün [Tema Yönümlü Bacarıqlar](../../user-guide/configuration/theme-aware-skills) bölməsinə baxın.

## İnternet Axtarış Təminatçıları

Superdav AI Agent v1.10.0 internet axtarış sorğuları üçün Brave Search-ə alternativ kimi **Tavily** əlavə edir. Hər iki təminatçı real vaxt axtarış nəticələri təklif edir və **Gratis AI Agent → Settings → Advanced** bölməsində konfiqurasiya edilə bilər.

| Provayder | Xüsusiyyətlər | Quraşdırma |
|---|---|---|
| **Brave Search** | Sürətli, məxfiliyə yönəlmiş axtarış nəticələri | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Daha zəngin axtarış nəticələri, qabaqcıl araşdırma imkanları | [Tavily API](https://tavily.com/) |

Konfiqurasiya təlimatları üçün [İnternet Axtarışı](../../user-guide/configuration/internet-search) bölməsinə baxın.

## Təqvim, Təsdiq və Bildiriş İş Axınları

Superdav AI Agent v1.18.0 avtomatlaşdırma iş axınlarını məzmun və sayt qurma tapşırıqlarından kənara genişləndirir:

- **Google Calendar alətləri** təqvim giriş məlumatları mövcud olduqda agentə konfiqurasiya edilmiş təqvimləri və tədbir təfərrüatlarını oxumağa imkan verir.
- **Kontakt və iştirakçı köməkçiləri** xatırlatmalar və ya izləmə tapşırıqları hazırlamazdan əvvəl tədbir iştirakçılarını WordPress istifadəçiləri və ya məlum kontaktlarla uyğunlaşdırır.
- **İnsan təsdiqi mərhələləri** avtomatlaşdırmaya təklif olunan əməl səlahiyyətli istifadəçi tərəfindən nəzərdən keçirilənə qədər dayanmağa imkan verir.
- **Xatırlatma qeydləri** avtomatlaşdırma təkrar sınandıqda və ya dəfələrlə planlaşdırıldıqda təkrarlanan təqvim və ya bildiriş xatırlatmalarının qarşısını alır.
- **TextBee SMS dəstəyi** TextBee provayderi konfiqurasiya edildikdə mətn mesajı bildirişlərini aktivləşdirir.

Bu alətlərdən cədvələ həssas yardım, iştirakçı izləməsi və həssas mesajlar göndərilməzdən əvvəl hələ də insan yoxlaması tələb edən xatırlatma iş axınları üçün istifadə edin.

## Sayt Qurucusu Kontakt Formaları

Sayt Qurucusu agenti indi kontakt formalarını birbaşa çat interfeysindən yarada bilər. Sadəcə Sayt Qurucusundan kontakt forması yaratmağı istəyin; o, quraşdırılmış forma plugin-inizi (Contact Form 7, WPForms, Fluent Forms və ya Gravity Forms) avtomatik aşkarlayacaq və istifadəyə hazır forma yaradacaq.

Ətraflı təlimat üçün [Sayt Qurucusu Kontakt Formaları](../../user-guide/administration/site-builder-contact-forms) bölməsinə baxın.

## Konfiqurasiya

Bütün parametrlər WordPress idarəetməsində **Gratis AI Agent → Settings** altında yerləşir. Superdav AI Agent aşağıdakı konfiqurasiya seçimlərini əlavə edir:

| Parametr | Təsvir |
|---|---|
| **Internet Search Provider** | Brave Search və Tavily arasında seçim edin (istəyə bağlı) |
| **Tavily API Key** | Tavily internet axtarışı üçün API açarı (Tavily seçilibsə) |
| **Managed Superdav Service** | Dəstəklənən saytları avtomatik son nöqtə təminatı ilə host edilən Superdav xidmətinə qoşun |
| **Google Calendar Credentials** | Agentin təqvimləri və tədbirləri oxuya bilməsi üçün təqvim girişini konfiqurasiya edin |
| **TextBee SMS Provider** | Təsdiqlənmiş SMS bildirişi iş axınları üçün TextBee giriş məlumatlarını konfiqurasiya edin |

## Qabaqcıl Müşayiətçi Paket

Superdav AI Agent v1.18.0 etibarlı developer iş axınları üçün ayrıca paylanan Qabaqcıl müşayiətçi paket əlavə edir. Bu imkanlar WordPress.org əsas paketinə daxil edilmir. Müşayiətçi paketi yalnız administratorların agentin developer alətlərinə çıxışına açıq şəkildə etibar etdiyi saytlarda quraşdırın.

Qabaqcıl alətlərə fayl sistemi əməliyyatları, verilənlər bazasına giriş, WP-CLI, REST dispatcher iş axınları, plugin qurucusu utilitləri, git anlıq görüntüləri, istifadəçi idarəetməsi köməkçiləri və benchmark alətləri daxildir. Bu alətləri etibarlı administratorlarla məhdudlaşdırın və onları production saytlarda aktivləşdirməzdən əvvəl təsdiq mərhələsi konfiqurasiyasını nəzərdən keçirin.
