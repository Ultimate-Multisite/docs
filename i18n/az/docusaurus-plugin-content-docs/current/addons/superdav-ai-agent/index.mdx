# Superdav AI Agent

Superdav AI Agent, Gratis AI Agent-i tema-əsaslı bacarıqlarla, internet axtarış imkanlarının gücləndirilməsi və sayt qurucu funksionallığının yaxşılaşdırılması ilə genişləndirən, inkişaf etmiş bir WordPress Süni İntellekt (AI) köməkçisidir. 1.16.0 versiyası xüsusi loqo yaratma, palitra kontrastı yoxlaması, kəşfiyyat müsahibəsində foto yükləmə, strukturlaşdırılmış qonaqpəmvəliyi menyusu yaratma və Theme Builder-də masaüstü/mobil önizləmə render etmə funksionallıqlarını əlavə edir.

## Əsas Xüsusiyyətlər

- **Tema-əsaslı bacarıqlar** — aktiv WordPress temanıza və quraşdırılmış pluginlərinizə əsasən yol göstərməni və imkanları avtomatik olaraq uyğunlaşdırır.
- **Bir neçə internet axtarış provayuderi** — daha zəngin axtarış nəticələri və inkişaf etmiş tədqiqat imkanları üçün Brave Search və Tavily arasında seçim edə bilərsiniz.
- **Sayt qurucu kontakt formaları** — quraşdırılmış form plugininizdən istifadə edərək söhbət interfeysindən birbaşa tam funksional kontakt formaları yaradın.
- **Gücləndirilmiş sayt qurucu** — daha yaxşı plugin kəşfi və xəta bərpası ilə təkmilləşdirilmiş çoxmərhəlli sayt qurma planları.
- **Daxili tema bacarıqları** — Block Themes, Classic Themes, Kadence Blocks və Kadence Theme üçün xüsusi yol göstərmə.
- **Təkmilləşdirilmiş WooCommerce inteqrasiyası** — daha yaxşı etibarlılıq və uyğunluq üçün yerli WooCommerce API-lərindən istifadə etməsi üçün yenidən qurulub.
- **Avtomatik provayuder aşkarlanması** — pluginlər aktivləşdirildikdə və ya deaktive ediləndə mövcud pluginləri və temaları avtomatik aşkarlayır və yeniləyir.
- **Xüsusi loqo yaratma** — avtomatik namespace-safe sanitasiya ilə xüsusi SVG loqoları yaradın və yerləşdirin.
- **Palitra kontrastı yoxlaması** — temanıza tətbiq etməzdən əvvəl rəng cütlüklərini WCAG əlçatanlığı uyğunluğuna görə yoxlayır.
- **Foto ilə zəngin kəşfiyyat** — Dizayn istiqamətini təsir etmək üçün Theme Builder kəşfiyyat müsahibəsi zamanı foto yükləyin.
- **Qonaqpəmvəliyi menyuları** — restoranlar, kafeler və qonaqpəmvəliyi biznesləri üçün strukturlaşdırılmış yemək və içki menyu səhifələri yaradır.
- **Masaüstü və mobil önizləmələri** — Dizayn istiqamətini seçərkən dizaynınızı masaüstü və mobil cihazlarda önizləyin.
- **Tier 1 sayt skreypinqi** — sd-ai-agent/site-scrape artıq rəqib və ilham verici saytları analiz etmək üçün standart olaraq mövcuddur.

## Tələblər

- WordPress 6.0 və ya daha yuxarı
- PHP 8.1 və ya daha yuxarı
- WordPress Multisite (şəbəkə səviyyəsində aktivləşdirilmiş) və ya tək sayt quraşdırması
- Aktiv bir AI provayuder konnektor plugin-i (Anthropic, OpenAI və ya uyğun)
- Gratis AI Agent v1.9.0 və ya daha yuxarı (Superdav AI Agent, Gratis AI Agent-i genişləndirir)

## Quraşdırma

1. Plugin fayllarını `/wp-content/plugins/superdav-ai-agent` yoluna yükləyin.
2. WordPress-də **Pluginlər** ekranı vasitəsilə plugin-i aktivləşdirin.
3. Gratis AI Agent-in də aktivləşdirildiyinə əmin olun.
4. **Gratis AI Agent → Settings** səhifəsinə keçin və internet axtarış provayuderinizi konfiqurasiya edin (isteğe bağlı).

## Tez Başlanğıc

Aktivləşdirildikdən sonra Superdav AI Agent, Gratis AI Agent quraşdırmanızı tema-əsaslı bacarıqlar və əlavə imkanlarla avtomatik olaraq gücləndirir. WordPress admin yan çubuğunda **Gratis AI Agent** panelinə daxil olun və yeni funksionallıqdan istifadə etməyə başlayın.


## Tema-əsaslı Bacarıqlar

Superdav AI Agent aktiv temanı və quraşdırılmış pluginləri avtomatik aşkarlayır və müvafiq tema-əsaslı bacarığı köməkçinin kontekstinə yükləyir. Mövcud bacarıqlar bunlardır:

| Bacarıq | Hansı yerlərə tətbiq olunur |
|---|---|
| **Block Themes** | WordPress blok əsaslı (Full Site Editing) arxitekturasından istifadə edən temalar |
| **Classic Themes** | PHP şablonlarından və klassik redaktorundan istifadə edən ənənəvi WordPress temaları |
| **Kadence Blocks** | Kadence Blocks plugin-i aktiv olan saytlar |
| **Kadence Theme** | Aktiv tema kimi Kadence Theme istifadə edən saytlar |

Hər bir bacarıq haqqında ətraflı təlimatlar üçün [Theme-Aware Skills](../../user-guide/configuration/theme-aware-skills) səhifəsinə baxın.

## İnternet Axtarış Provayuderləri

Superdav AI Agent v1.10.0, internet axtarış sorğuları üçün Brave Search-ə alternativ olaraq **Tavily** əlavə edir. Hər iki provayuder real vaxt axtarış nəticələri təklif edir və **Gratis AI Agent → Settings** bölməsində konfiqurasiya edilə bilər.

| Provayuder | Xüsusiyyətlər | Qurulması |
|---|---|---|
| **Brave Search** | Tez, məxfiliyətə fokuslanmış axtarış nəticələri | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Daha zəngin axtarış nəticələri, inkişaf etmiş tədqiqat imkanları | [Tavily API](https://tavily.com/) |

Konfiqurasiya təlimatları üçün [Internet Search](../../user-guide/configuration/internet-search) səhifəsinə baxın.

## Sayt Qurucu Kontakt Formaları

Sayt Qurucu agenti artıq kontakt formalarını birbaşa söhbət interfeysindən yarada bilər. Sadəcə Sayt Qurucudan kontakt forması yaratmasını istəyin və o, quraşdırılmış form plugininizi (Contact Form 7, WPForms, Fluent Forms və ya Gravity Forms) avtomatik aşkarlayaraq istifadəyə hazır bir forma yaradacaq.

Ətraflı təlimatlar üçün [Site Builder Contact Forms](../../user-guide/administration/site-builder-contact-forms) səhifəsinə baxın.

## Theme Builder Xüsusiyyətləri

Superdav AI Agent v1.16.0-dəki Theme Builder güclü yeni imkanları ehtiva edir:

### Foto Yükləmə ilə Kəşfiyyat Müsahibəsi

Theme Builder kəşfiyyat müsahibəsi artıq vizual ilham verə biləcəyiniz bir foto yükləmə addımını ehtiva edir. Theme Builder-in dizayn istiqamətini anlamağa kömək etmək üçün brendinizin, rəqib saytlarınızın və ya dizayn ilhamlarının fotoşəkillərini yükləyin.

Ətraflı təlimatlar üçün [Discovery Interview](./theme-builder/discovery-interview.md) səhifəsinə baxın.

### Masaüstü və Mobil Önizləmə ilə Dizayn İstiqaməti

Theme Builder tam temanızı yaratmadan əvvəl dizaynınızı həm masaüstü, həm də mobil cihazlarda nəzərdən keçirin. Bir neçə responsive breakpoint-də real vaxt önizləmələrinə baxın.

Ətraflı təlimatlar üçün [Design Direction](./theme-builder/design-direction.md) səhifəsinə baxın.

### Qonaqpəmvəliyi Menyuları

Theme Builder artıq restoranlar, kafeler, barlar və digər qonaqpəmvəliyi biznesləri üçün strukturlaşdırılmış yemək və içki menyu səhifələri yarada bilər. Menyular qiymətləri, təsvirləri, diyet məlumatları və allergen xəbərdאַר etmələri ehtiva edir.

Ətraflı təlimatlar üçün [Hospitality Menus](./theme-builder/hospitality-menus.md) səhifəsinə baxın.

### Yeni Bacarıqlar

- **[Generate Logo SVG](./abilities/generate-logo-svg.md)** — avtomatik sanitasiya ilə xüsusi SVG loqoları yaradın.
- **[Validate Palette Contrast](./abilities/validate-palette-contrast.md)** — rəng cütlüklərini WCAG uyğunluğuna görə yoxlayın.
- **[Create Menu](./abilities/create-menu.md)** — müxtəlif naviqasiya etiketləri ilə naviqasiya menyuları yaradın.

Mövcud alətlər və Tier 1 sayt-skrape aləti haqqında məlumat üçün [Tools and Configuration](./theme-builder/tools.md) səhifəsinə baxın.

## Konfiqurasiya

Bütün parametrlər WordPress admin panelində **Gratis AI Agent → Settings** altında yerləşir. Superdav AI Agent aşağıdakı konfiqurasiya seçimlərini əlavə edir:

| Parametr | Təsvir |
|---|---|
| **Internet Search Provider** | Brave Search və Tavily arasında seçim edin (isteğe bağlı) |
| **Tavily API Key** | Tavily internet axtarışı üçün API açarı (Tavily seçildikdə) |

## v1.16.0-də Təkmilləşdirmələr

- **AI Client cache** — indi transiyentlər vasitəsilə dəstəklənir, bu da uzun müddət davam edən agent tapşırıqlarında məlumat itkisinin qarşısını alır.
- **Tier 1 site-scrape aləti** — sd-ai-agent/site-scrape artıq Theme Builder-də rəqib və ilham verici saytları analiz etmək üçün standart olaraq mövcuddur.
- **Plugin sətiri hərəkət linkləri** — aydınlıq üçün düzəldilib və yenidən adlandırılıb.

## v1.10.0-də Təkmilləşdirmələr

- **WooCommerce inteqrasiyası yenidən quruldu** — daha yaxşı etibarlılıq və uyğunluq üçün indi yerli WooCommerce API-lərindən istifadə edir.
- **Provayuder siyahısının avtomatik yenilənməsi** — pluginlər aktivləşdirildikdə və ya deaktive ediləndə avtomatik aşkarlayır və mövcud provayuder siyahısını yeniləyir.
- **Sayt qurucu gücləndirilməsi** — daha yaxşı xəta bərpası ilə təkmilləşdirilmiş çoxmərhəlli sayt qurma planları.
- **Tema-əsaslı bacarıqlar** — aktiv temanıza avtomatik uyğunlaşan dörd yeni daxili bacarıq.

## v1.10.0-də Bug Düzəlişləri

- Bəzi admin səhifələrində sonsuz yenilənmə dövrünə səbəb olan `navigate-to` bacarığı düzəldildi.
- `list-posts` bacarığının kateqoriya və tag adlarını slug-lara düzgün şəkildə həll etmədiyi düzəldildi.
- Əvvəlki yenidən qurulmadan sonra WP-CLI əmrlərinin namespace aliaslarının çatışmaması düzəldildi.
- Təşkilatlanma cədvəlləri hələ yaradılmamış saytlarda avtomatlaşdırmanın zərif şəkildə idarə edilməsi düzəldildi.
- `memory-save` bacarığının sistem təlimat qurucusunda düzgün namespace prefiksindən istifadə etməsi düzəldildi.
- Skalyar alət nəticələrinin AI-yə qaytarılmadan əvvəl düzgün şəkildə bükülməsi düzəldildi.
- İstifadə statistikalarının köhnə versiyalardan yüksəldikdə köhnə bacarıq açar formatını düzgün idarə etməsi düzəldildi.
