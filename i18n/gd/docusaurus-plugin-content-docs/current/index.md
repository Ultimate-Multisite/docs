# Sgrìobhainnean Ultimate Multisite

Fàilte gu sgrìobhainnean Ultimate Multisite. Bidh Ultimate Multisite ag atharrachadh WordPress Multisite gu àrd-ùrlar làn Làrach-lìn mar Sheirbheis (WaaS).

## Earrannan nan Sgrìobhainnean

### [Stiùireadh Cleachdaiche](/user-guide/getting-started/ultimate-multisite-101)

A h-uile rud a dh’fheumas tu gus an t-àrd-ùrlar WaaS agad a stèidheachadh, a rèiteachadh, agus a stiùireadh. Tha e a’ còmhdach stàladh, geataichean pàighidh, mapadh raointean-lìn, stiùireadh luchd-dèiligidh, agus barrachd.

### [Sgrìobhainnean Luchd-leasachaidh](/developer/hooks)

Iomradh teicnigeach do luchd-leasachaidh a tha a’ togail air Ultimate Multisite. A’ gabhail a-steach iomradh hooks, sgrìobhainnean REST API, agus stiùiridhean leasachaidh addon.

### [Addons](/addons)

Sgrìobhainnean airson plugins addon pàighte a leudaicheas Ultimate Multisite le feartan a bharrachd mar amalachadh WooCommerce, anailitigs, reic raointean-lìn, agus barrachd.
