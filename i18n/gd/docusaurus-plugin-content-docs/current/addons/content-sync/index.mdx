# Content Sync

Multisite Ultimate: Content Sync

Leigidh an tuilleadan seo airson Multisite Ultimate le rianairean lìonra susbaint a shioncronachadh bho làrach sam bith gu làraichean eile san lìonra. Tha e a’ toirt taic do gach seòrsa puist, a’ gabhail a-steach puist, duilleagan, seòrsaichean puist gnàthaichte, agus teamplaidean Elementor. Tha an tuilleadan a’ gabhail a-steach eadar-aghaidh draoidh ioma-cheumach airson susbaint agus làraichean targaid a thaghadh, le giollachd ann am baidsean agus fios-air-ais adhartais fìor-ùine.

## Feartan {#features}

* Taghadh Susbaint: Griod chairtean lèirsinneach le dealbhagan agus sìoladh a rèir seòrsa puist
* Gach Seòrsa Puist: Sioncronaich puist, duilleagan, batharan, teamplaidean Elementor, agus seòrsa puist gnàthaichte sam bith
* Taghadh Làraich: Tagh dè na làraichean ris an tèid susbaint a shioncronachadh, le làrach tùsail air a dùnadh a-mach agus làraichean cloinne ro-thaghte
* Giollachd ann am Baidsean: Sioncronaich gu 10 làraichean aig an aon àm le ùrachaidhean adhartais
* Sioncronachadh Meadhanan: Dèan lethbhreac agus ath-mhapaich faidhlichean mheadhanan (dealbhan, bhidiothan) gu fèin-obrachail bho shusbaint puist gu làraichean targaid
* Sioncronachadh Tacsonamaidh: Glèidh roinnean-seòrsa, tagaichean, agus tacsonamaidhean gnàthaichte
* Taic Elementor: Taic iomlan do theamplaidean Elementor le ath-ghineadh CSS agus ath-mhapadh ID teamplaid
* Fuasgladh Chòmhstri: Roghainn gus susbaint a tha ann mu thràth a chur na h-àite no a leum seachad air
* Riaghladh Cache: Glanaidh e cache Elementor gu fèin-obrachail air làraichean targaid às dèidh sioncronachaidh

## Riatanasan {#requirements}

* WordPress 5.3 no nas àirde
* PHP 7.4 no nas àirde
* plugin Multisite Ultimate (gnìomhach)
* plugin Elementor (roghainneil, riatanach airson sioncronachadh teamplaidean Elementor)

## Stàladh {#installation}

1. Luchdaich suas 'ultimate-multisite-content-sync' dhan eòlaire '/wp-content/plugins/'
2. Cuir an gnìomh am plugin tron chlàr-taice 'Plugins' anns an WordPress Network Admin agad
3. Rèitich roghainnean ann an Ultimate Multisite → Settings → Content Sync
4. Seòl gu Ultimate Multisite → Content Sync gus tòiseachadh air sioncronachadh
