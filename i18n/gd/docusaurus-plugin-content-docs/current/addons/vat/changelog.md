# Log nan atharrachaidhean VAT {#vat-changelog}

Tionndadh 1.0.7 - Air fhoillseachadh air 2026-02-03

* Ceartachadh: Bha àireamhan VAT Gearmailteach (DE) a’ fàiligeadh dearbhaidh ri linn mhearachdan MS_MAX_CONCURRENT_REQ ann an seirbheis VIES SOAP. Chaidh atharrachadh bhon SOAP API dìleabach gu API oifigeil EU VIES REST airson dearbhadh nas earbsaiche.
* Ceartachadh: Tha àireamhan VAT a-nis gan gabhail nuair a tha seirbheis VIES sealach neo-ri fhaighinn, an àite a bhith gan diùltadh gu ceàrr. Thèid inntrigeadh loga a chruthachadh airson lèirmheas nas fhaide air adhart.
* Ceartachadh: Tha raon VAT/Tax ID a-nis ri fhaicinn airson a h-uile dùthaich, chan ann dìreach buill an EU. Thèid IDan cìse neo-EU (m.e. àireamhan CHE na h-Eilbheise) a stòradh airson taisbeanadh air fàirdeal gun dearbhadh VIES.
* Ceartachadh: Bha lorg ro-leasachan VAT airson na Grèige (GR), Monaco (MC), agus Eilean Mhanainn (IM) briste ri linn loidsig lorg array mhì-cheart.
* Leasachadh: Chaidh eisimeileachd leasachaidh wp-cli-bundle neo-riatanach a thoirt air falbh a dh’fhaodadh mearachdan marbhtach adhbharachadh nuair a chleachdar i còmhla ris an tuilleadan Site Exporter.
* Leasachadh: Chaidh sreath dheuchainnean unit fharsaing a chur ris (53 deuchainn).

Tionndadh 1.0.6 - Air fhoillseachadh air 2026-01-25

* Ceartachadh: Tha gnothaichean dachaigheil B2B a-nis a’ cur VAT an sàs gu ceart. A rèir riaghailtean VAT an EU, chan eil reverse charge a’ buntainn ach ri gnothaichean B2B thar-chrìche, chan ann nuair a tha dùthaich a’ chustamaire co-ionann ri dùthaich a’ chompanaidh.
