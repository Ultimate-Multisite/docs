# Fluent Forms {#fluent-forms}

Tha an tuilleadan seo ga dhèanamh comasach crìochan a chur ri Fluent Forms air Multisite Ultimate ann am batharan, ballrachdan no làraichean.
Bidh Fluent Forms a’ cleachdadh chlàran gnàthaichte an àite seòrsaichean puist gnàthaichte, agus mar sin chan obraich na roghainnean cuibhreachaidh àbhaisteach le Fluent Forms.
