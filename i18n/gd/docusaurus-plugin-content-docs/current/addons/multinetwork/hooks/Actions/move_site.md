# Gnìomh: move_site

Thèid a chur an gnìomh às dèidh do làrach a bhith air a gluasad gu lìonra ùr.

## Paramadairean

| Ainm | Seòrsa | Tuairisgeul |
|------|------|-------------|
| $site_id | `int` | ID na làraich a chaidh a ghluasad. |
| $old_network_id | `int` | ID an lìonra thùsail airson na làraich. |
| $new_network_id | `int` | ID an lìonra dhan deach an làrach a ghluasad. |
