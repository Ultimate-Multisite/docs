# Loga Atharrachaidhean Ioma-Lìonra

### 1.3.0 {#130}
* Ùr: Ro-shealladair teamplaidean lìonra — brabhsaich agus ro-sheall teamplaidean lìonra le pannal beò mus ceannaich thu.
* Ùr: Taghadair teamplaid lìonra aig checkout — faodaidh luchd-ceannach teamplaid lìonra a thaghadh rè sruth checkout an t-soidhnidh.
* Ùr: SSO ceangail dhraoidheil thar-lìonraidhean — faodaidh luchd-ceannach faighinn gu fo-lìonraidhean gu rèidh tro dhearbhadh ceangail dhraoidheil.
* Ùr: Clònadh mheadhanan lìonra — tha meadhanan làraich air an dùblachadh gu ceart nuair a thathar a’ clònadh lìonraidhean bho theamplaid.
* Càradh: Tha clònadh lìonra a-nis ag obair às aonais an addon ioma-mhàladaireachd; tha sochairean super admin air an ath-bhuileachadh às dèidh clònadh.
* Càradh: Tha cruthachadh lìonra a-nis co-chòrdail ri WooCommerce agus a h-uile geata pàighidh.
* Càradh: Tha àireamh puirt a-nis air a gabhail a-steach ann an gineadh àrainn fallback.
* Càradh: Chaidh CSS an taghadair teamplaid a ghluasad gu duilleag-stoidhle enqueued airson taisbeanadh ceart air gach theme.
* Càradh: Dìon an aghaidh iuchair `network_id` neo-mhìnichte ann an raon ceist làraich.
* Càradh: Bidh sgrùdadh ceadan lìonra mus dèanar atharrachadh co-theacsa a’ cur casg air mearachdan seilbh clàir `wpdb`.
