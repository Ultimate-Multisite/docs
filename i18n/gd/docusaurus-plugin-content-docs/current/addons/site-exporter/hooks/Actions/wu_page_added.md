# Gnìomh: wu_page_added

Leig le luchd-leasachaidh plugin rudan a bharrachd a ruith nuair a thèid duilleagan a chlàradh.

Eu-coltach ri wu_page_load, nach ruith ach nuair a thathar a’ coimhead air duilleag shònraichte, ruithidh an hook seo aig clàradh airson gach duilleag rianachd a thèid a chur ris a’ cleachdadh còd WP Ultimo.

## Paramadairean

| Ainm | Seòrsa | Tuairisgeul |
|------|------|-------------|
| $page_id | `string\|\WP_Ultimo_Site_Exporter\Admin_Pages\Base_Admin_Page` | ID na duilleige seo. |
| $page_hook | `string\|\WP_Ultimo_Site_Exporter\Admin_Pages\Base_Admin_Page` | An hook a chaidh a ghineadh don duilleag seo. |
