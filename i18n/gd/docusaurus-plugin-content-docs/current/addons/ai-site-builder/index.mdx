# Togail Làraich AI

ultimate-multisite-ai-site-builder

## Riatanasan {#requirements}

- WordPress 5.3 no nas àirde
- PHP 7.4 no nas àirde
- plugin Multisite Ultimate (gnìomhach)

## Stàladh {#installation}

1. Luchdaich suas faidhlichean an addon dhan eòlaire `/wp-content/plugins/` agad
2. Cuir an plugin an gnìomh tron chlàr-taice 'Plugins' ann an WordPress
3. Rèitich roghainnean an addon ann an rianachd an lìonraidh

## Leasachadh {#development}

### Suidheachadh {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Deuchainnean {#testing}

Tha an addon a’ gabhail a-steach frèam-obrach deuchainn coileanta:

- **Deuchainnean Aonad**: Deuchainnean stèidhichte air PHPUnit le sreath dheuchainnean WordPress
- **Clas Deuchainn Bunaiteach**: A’ toirt seachad dòighean-cuideachaidh airson suidheachaidhean deuchainn cumanta
- **Còmhdach Deuchainn**: Air a rèiteachadh gus còmhdach còd a leantainn
- **Deuchainnean Multisite**: Ruithidh deuchainnean ann an àrainneachd multisite

### Inbhean Còd {#code-standards}

- **Inbhean Còdaidh WordPress**: Air an cur an gnìomh tro PHPCS
- **Co-chòrdalachd PHP 7.4+**: Taic do fheartan PHP ùr-nodha
- **Mion-sgrùdadh Statach**: Amalachadh PHPStan airson sgrùdadh sheòrsaichean
- **Ùrachadh Còd**: Riaghailtean Rector airson ùrachadh PHP

### Siostam Togail {#build-system}

- **Giullachd Maoin**: Mionachadh airson faidhlichean CSS/JS
- **Eadar-theangachadh**: Gineadh faidhle POT
- **Pacadh**: Cruthachadh tasglainn airson sgaoileadh
- **Modh Leasachaidh**: Maoin gun mhionachadh airson dì-bhugachadh

## Rèiteachadh {#configuration}

Tha an addon a’ toirt taic do dhiofar roghainnean rèiteachaidh a tha ruigsinneach tro eadar-aghaidh rianachd an lìonraidh.

## Hooks agus Criathragan {#hooks-and-filters}

### Gnìomhan {#actions}

- `ultimate-multisite-ai-site-builder_init` - Air a chur gu dol an dèidh tòiseachadh an addon
- `ultimate-multisite-ai-site-builder_loaded` - Air a chur gu dol an dèidh dha faidhlichean uile an addon a bhith air an luchdadh

### Criathragan {#filters}

- `ultimate-multisite-ai-site-builder_settings` - Atharraich roghainnean an addon
- `ultimate-multisite-ai-site-builder_enabled` - Sgrìobh thairis air inbhe comasachaidh/dì-chomasachaidh an addon
