# Ioma-mhàladaireachd

Sgaoil fo-làraichean thar fhrithealaichean is stòran-dàta airson dèanadas agus tèarainteachd

## Prìomh nithean ann an sgaoileadh tionndaidh 1.2.0

Tha Multi-Tenancy 1.2.0 a’ cur bun-stèidh màladair uachdaranach ris airson sruthan-obrach aonrachaidh riochdachaidh:

- Fèin-logadh a-steach màladair gun staid le tursan SSO geàrr-bheò, dìonta an aghaidh ath-chluiche.
- Aonrachadh stòr-dàta is siostam-fhaidhlichean gach fo-làraich airson màladairean uachdaranach.
- Comas ioma-mhàladaireachd Hostinger airson ullachadh mhàladairean mothachail air òsta.
- Geataichean dearbhaidh imrich uachdaranaich airson sgrùdaidhean dàta dìleabach agus làimhseachadh putadh async.
- Ceangal òsta stòr-dàta air an aon inneal a ghabhas rèiteachadh airson stàlaidhean Bedrock, FrankenPHP, agus localhost.
- Leasachaidhean air bootstrap màladair, sgrìobhadair stòr-dàta, bootstrap sgeama, agus ullachadh chleachdaichean.
- Teardown uachdaranach nas sàbhailte a ghlanas teisteanasan màladair.

Faic na stiùiridhean cleachdaiche airson [sruth-obrach imrich uachdaranaich](/user-guide/getting-started/sovereign-migration-workflow), [aonrachadh ioma-mhàladaireachd](/user-guide/configuration/multi-tenancy-isolation), agus [obrachaidhean màladair uachdaranaich](/user-guide/administration/sovereign-tenant-operations).

## Riatanasan

- WordPress 5.3 no nas àirde
- PHP 7.4 no nas àirde
- Plugan Multisite Ultimate (gnìomhach)

## Stàladh

1. Luchdaich suas faidhlichean an tuilleadain don eòlaire `/wp-content/plugins/` agad
2. Cuir an gnìomh am plugan tron chlàr-taice 'Plugins' ann an WordPress
3. Rèitich roghainnean an tuilleadain ann an rianaire an lìonraidh

## Leasachadh

### Suidheachadh

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Structar a’ phròiseict

```
ultimate-multisite-addon-template/
├── inc/                          # Clasaichean PHP agus in-ghabhail

│   └── class-multi_tenancy-main.php             # Prìomh-ghnìomhachd
├── tests/                        # Deuchainnean aonad
│   ├── bootstrap.php            # Bootstrap deuchainn
│   ├── class-*-base.php         # Clas deuchainn bunaiteach
│   └── class-*-test.php         # Cùisean deuchainn
├── views/                       # Faidhlichean teamplaid
├── assets/                      # Maoin statach
│   ├── css/                     # Duilleagan-stoidhle
│   ├── js/                      # Faidhlichean JavaScript
│   └── images/                  # Dealbhan
├── lang/                        # Faidhlichean cànain
```

### Deuchainn

Tha frèam-obrach deuchainn coileanta san tuilleadan:

- **Deuchainnean Aonad**: Deuchainnean stèidhichte air PHPUnit le sreath dheuchainnean WordPress
- **Clas Deuchainn Bunaiteach**: A’ toirt seachad dòighean-cuideachaidh airson suidheachaidhean deuchainn cumanta
- **Còmhdach Deuchainn**: Air a rèiteachadh gus còmhdach còd a leantainn
- **Deuchainn Multisite**: Ruithidh deuchainnean ann an àrainneachd multisite

### Inbhean Còd

- **Inbhean Còdaidh WordPress**: Air an cur an sàs tro PHPCS
- **Co-chòrdalachd PHP 7.4+**: Taic do fheartan PHP nuadh
- **Mion-sgrùdadh Statach**: Amalachadh PHPStan airson sgrùdadh sheòrsaichean
- **Ùrachadh Còd**: Riaghailtean Rector airson ùrachaidhean PHP

### Siostam Togail

- **Làimhseachadh Maoin**: Mionachadh airson faidhlichean CSS/JS
- **Eadar-theangachadh**: Gineadh faidhle POT
- **Pacaigeadh**: Cruthachadh tasglainn airson sgaoileadh
- **Modh Leasachaidh**: Maoin gun mhionachadh airson dì-bhugachadh

## Rèiteachadh

Tha an tuilleadan a’ cur taic ri diofar roghainnean rèiteachaidh a gheibhear tro eadar-aghaidh rianaire an lìonraidh.

### Rèiteachadh màladair uachdaranaich

Airson màladairean uachdaranach, rèitich stòr-dàta a’ mhàladair, freumh an t-siostaim-fhaidhlichean, ceangal òsta an stòr-dàta, agus comas an òsta mus stiùir thu trafaig chun a’ mhàladair. Ruith dearbhadh imrich às dèidh gach atharrachaidh a bheir buaidh air teisteanasan stòr-dàta, ceangal òsta, stiùireadh, no ullachadh chleachdaichean.

Cleachd `localhost` a-mhàin nuair a tha runtime a’ mhàladair agus tabhartasan an stòr-dàta an dùil ri socaid ionadail no ruigsinneachd air an aon inneal. Cleachd `127.0.0.1` no ainm-òsta seirbheise nuair is e sin òsta an tabhartais a tha air a rèiteachadh leis an t-solaraiche agad.

### Comas Hostinger

Bidh comas ioma-mhàladaireachd Hostinger ag obair leis an tòcan amalachaidh Hostinger co-roinnte. Dearbhaich gu bheil an àrainn an làthair ann an hPanel, gu bheil DNS a’ comharrachadh gu Account Hostinger, agus gu bheil roghainnean stòr-dàta a’ mhàladair a’ freagairt ri hPanel mus tèid dearbhadh a dhèanamh.

## Hooks agus Filters

### Gnìomhan

- `ultimate-multisite-addon-template_init` - Air a losgadh às dèidh tòiseachadh an tuilleadain
- `ultimate-multisite-addon-template_loaded` - Air a losgadh às dèidh do gach faidhle tuilleadain a bhith air a luchdadh

### Filters

- `ultimate-multisite-addon-template_settings` - Atharraich roghainnean an tuilleadain
- `ultimate-multisite-addon-template_enabled` - Tar-àithnich inbhe comasachaidh/dì-chomasachaidh an tuilleadain
