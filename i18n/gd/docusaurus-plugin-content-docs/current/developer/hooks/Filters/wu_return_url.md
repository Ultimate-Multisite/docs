# Criathrag: wu_return_url

Leig le luchd-leasachaidh URL tilleadh a’ gheata-phàighidh atharrachadh a thèid a chleachdadh às dèidh phròiseasan crìochnachaidh ceannach.

## Paramadairean {#parameters}

| Ainm | Seòrsa | Tuairisgeul |
|------|------|-------------|
| $return_url | `string` | an URL airson ath-stiùireadh às dèidh a’ phròiseis. |
| $gateway | `self` | eisimpleir a’ gheata-phàighidh. |
| $payment | `\WP_Ultimo\Models\Payment` | eisimpleir pàighidh Ultimate Multisite. |
| $cart | `\WP_Ultimo\Checkout\Cart` | òrdugh cairt Ultimate Multisite làithreach. |
