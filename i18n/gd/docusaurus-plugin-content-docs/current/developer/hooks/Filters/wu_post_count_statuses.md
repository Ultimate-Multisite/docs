# Criathrag: wu_post_count_statuses {#filter-wupostcountstatuses}

| Ainm | Seòrsa | Tuairisgeul |
|------|------|-------------|
| $post_status | `array` | Liosta nan inbhean puist |
| $post_type | `string` | Slug seòrsa a’ phuist |
