# Criathrag: wu_get_checkout_variables

Leig le luchd-leasachaidh plugin na ro-shuidhichidhean aig duilleag checkout a chriathradh.

Bi faiceallach, faodaidh iuchraichean a tha a dhìth an checkout a bhriseadh gu tur air an front-end.

## Paramadairean

| Ainm | Seòrsa | Tuairisgeul |
|------|------|-------------|
| $variables | `array` | Caochladairean ionadaichte. |
| $checkout | `\Checkout` | An class checkout. |

### Bho

- 2.0.0
### Tùs

Air a mhìneachadh ann an [`inc/checkout/class-checkout.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout.php#L1970) aig loidhne 1970


## Tilleadh
An array ùr de chaochladairean.
