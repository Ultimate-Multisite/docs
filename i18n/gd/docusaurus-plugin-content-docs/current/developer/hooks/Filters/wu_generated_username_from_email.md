# Criathrag: wu_generated_username_from_email

Criathraich ainm-cleachdaiche teachdaiche a chaidh a ghineadh.

## Paramadairean {#parameters}

| Ainm | Seòrsa | Tuairisgeul |
|------|------|-------------|
| $username | `string` | Ainm-cleachdaiche a chaidh a ghineadh. |
| $email | `string` | Seòladh puist-d ùr an teachdaiche. |
| $new_user_args | `array` | Array de argamaidean cleachdaiche ùir, ’s dòcha a’ gabhail a-steach ciad ainm agus sloinneadh. |
| $suffix | `string` | Cuir string ri ainm-cleachdaiche gus a dhèanamh gun samhail. |
