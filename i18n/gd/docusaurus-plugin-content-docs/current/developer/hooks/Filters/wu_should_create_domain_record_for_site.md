# Criathrag: wu_should_create_domain_record_for_site {#filter-wushouldcreatedomainrecordforsite}

Criathraich a bheil Ultimate Multisite bu chòir dha clàr àrainn a chruthachadh airson làrach ùr-chruthaichte.

Cleachd a’ chriathrag seo gus cruthachadh fèin-obrachail clàr-àrainn a chumail fodha no a chur dheth airson làraichean a chleachdas àrainn bhunait foirm-checkout cho-roinnte, òstair a-staigh, no àrainn a stiùireas amalachadh eile air leth.

## Paramadairean {#parameters}

| Ainm | Seòrsa | Tuairisgeul |
|------|------|-------------|
| $create | `bool` | A bheil an clàr àrainn ri chruthachadh. |
| $site | `WP_Site` | An nì làraich ùr-chruthaichte. |

### Bho {#since}

- 2.13.0

### Tùs {#source}

Air a mhìneachadh ann an `inc/functions/domain.php`.


## Tilleadh {#returns}

Boolean a’ sealltainn a bheil an clàr àrainn ri chruthachadh.
