# Criathrag: wu_domain_has_correct_dns

Leig le luchd-leasachaidh plugin sgrùdaidhean ùra a chur ris gus na toraidhean a mhìneachadh.

## Paramadairean

| Ainm | Seòrsa | Tuairisgeul |
|------|------|-------------|
| $result | `bool` | an toradh làithreach. |
| $domain | `self` | An instance domain làithreach. |
| $domains_and_ips | `array` | Liosta nan domain agus nan IP a chaidh a lorg air an sgrùdadh DNS. |

### Bho

- 2.0.4
### Tùs

Air a mhìneachadh ann an [`inc/models/class-domain.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/models/class-domain.php#L455) aig loidhne 455


## Tillidhean
