# Filter: wp_ultimo_render_vars

Leig le luchd-leasachaidh plugin caochladair a bharrachd a chur ris a’ cho-theacsa render gu cruinneil.

## Paramadairean

| Ainm | Seòrsa | Tuairisgeul |
|------|------|-------------|
| $args | `array` | Array anns a bheil caochladairean a chaidh a thoirt seachad leis a’ ghairm render. |
| $view | `string` | Ainm an view a thèid a render. |
| $default_view | `string` | Ainm an fallback_view |

### Bho

- 2.0.0
