# Criathrag: wu_minimum_password_length

Criathraich an fhad as lugha airson facal-faire.

Cha tèid a chur an gnìomh ach nuair a tha wu_enforce_password_rules fìor.

## Paramadairean {#parameters}

| Ainm | Seòrsa | Tuairisgeul |
|------|------|-------------|
| $min_length | `int` | An fhad as lugha airson facal-faire. Bunaiteach 12 (a’ freagairt ri Defender Pro). |
| $defender_active | `bool` | A bheil Defender Pro Strong Password gnìomhach. |
