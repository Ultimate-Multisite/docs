## Paramadairean {#parameters}

| Ainm | Seòrsa | Tuairisgeul |
|------|------|-------------|
| $id | `string` | ID na duilleige seo. |
| $page_hook | `string` | Dubhan duilleige na duilleige seo. |
| $admin_page | `self` | Instans na duilleige. |
