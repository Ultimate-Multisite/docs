# Gnìomh: wu_page_added

Leig le luchd-leasachaidh plugin rudan a bharrachd a ruith nuair a thèid duilleagan a chlàradh.

Eu-coltach ri wu_page_load, nach ruith ach nuair a thathar a’ faicinn duilleag shònraichte, ruithidh an hook seo aig clàradh airson gach duilleag rianachd a thèid a chur ris a’ cleachdadh còd Ultimate Multisite.

## Paramadairean

| Ainm | Seòrsa | Tuairisgeul |
|------|------|-------------|
| $page_id | `string` | ID na duilleige seo. |
| $page_hook | `string` | Ainm hook na duilleige seo. |

- 2.0.0
