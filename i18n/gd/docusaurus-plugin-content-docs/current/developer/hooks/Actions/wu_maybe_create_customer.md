# Gnìomh: wu_maybe_create_customer

Leig le luchd-leasachaidh phlugan rudan a bharrachd a dhèanamh nuair a thèid an neach-ceannach a chur ris.

Seo far am bi sinn a’ cur nan dubhan ris airson an customer-&gt;user a chur ris a’ phrìomh làrach cuideachd, mar eisimpleir.

## Paramadairean {#parameters}

| Ainm | Seòrsa | Tuairisgeul |
|------|------|-------------|
| $customer | `\Customer` | An neach-ceannach a chaidh a chruthachadh, ’s dòcha. |
| $checkout | `\Checkout` | An clas pàighidh làithreach. |
