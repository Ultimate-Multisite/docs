#### A' chluasachadh a' chluasach (Restoring from a Network Import Bundle)

 airson a' chluasachadh sitean a' chluasach (restore sites from a network import bundle):

1. Install Ultimate Multisite ar an host a tha agad
2. Finich an seòrsa setup multisite
3. Ràdh ar **Ultimate Multisite > Site Export** ar an ngrath new
4. Cluich ar **Import Network Bundle**
5. Thoir an ZIP file bundle import air a' chluasach
6. Finich an t-uisge a tha ar an sealladh

A thàirgseachais (network import bundles) a tha a' chluasach:
- Containt sitean (posts, pages, custom post types)
- Seòranna site agus optiona
- Rôl is eadar-dheargais uisgeir (user roles and permissions)
- Pluginhean is theamhan (air a tha a' chluasach anns an bundle)
- Uisgeadh media (air a tha a' chluasach anns an bundle)
- Tabeada database agus datha a' chluasach
