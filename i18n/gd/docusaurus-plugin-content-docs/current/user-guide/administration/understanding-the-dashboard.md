# Aims a bhith a' chunnadh an Dashboard Ultimate Multisite {#understanding-the-ultimate-multisite-dashboard}

Sgiathach faic an dashboard.

Mar a tha daoine eile de WordPress plugin, a tha Ultimate Multisite a thrasach ùr-luachda (network-level menu entry) a' chiallachadh **Ultimate Multisite** le muckle earrachdan sub-menu a' chùl.

## Widgets Dashboard na Ngrathdhaidh {#network-dashboard-widgets}

A tha Ultimate Multisite a thrasach beag widget-e fìor air an dashboard na ngrathdhaidh (network-level). Tha an dashboard seo a fhadhlan gu leithid **My Sites → Network Admin → Dashboard**. Tha na widgets a' chùl airson fhaighinn earrachdan agad aca agus coimheadan a tha deiseil airson an admin na ngrathdhaidh.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

### Prìomh Chluas {#first-steps}

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

Tha an widget seo a' chùl air aithreachais de dhârdail Ultimate Multisite agus a tha a' chùl airson an admin na ngrathdhaidh a' chluas earrachdan eile mar a' chùl fìor (configuring a payment method) agus a' creachadh accant test.
