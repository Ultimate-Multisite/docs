# Form Comharrachadh an Site Builder {#site-builder-contact-forms}

Rinn Superdav AI Agent v1.10.0 a' chlàradh achanach airson comharrachdan comharrach (contact forms) gu sònraich a' leat a' chlàradh chat a' leat a' chlàradh Site Builder. Tha seo a' gineadh eadar-dhealaichean comharrachdan fhuilich air aon seòl gu mòr gu mòr, gu h-uile a tha a' leat a' chlàradh chat a' leat a' chlàradh.

## Aims {#overview}

Tha eadar-dhealaichean comharrachdan Site Builder (`create_contact_form`) a' faicinn gu sònraich an plugin comharrach (form plugin) a tha a' leat a' chlàradh agus a gineadh comharrach air a chàrachd a' plugin. Tha eadar-dhealaichean comharrachdan a' tòisichdairidh:

- Contact Form 7
- WPForms
- Fluent Forms
- Gravity Forms

Tha an àiteach a' leat a' gineadh an plugin comharrach is beart air a chàrachd agus a gineadh comharrach comharrach a tha a' leat a' chlàradh.

## Comharrachadh Comharrach {#creating-a-contact-form}

2. Ath a' chlàradh dhan eadar-dhealaichean **Site Builder** gu leithid an icon eadar-dhealaichean anns an seòl chat.

## A' chàrrachadh an fòram de chàrrtaidh (Contact Form) {#using-the-generated-shortcode}

An dàrna tar, an uile a tha a' chàrrachadh an fòram air a bhith a' chùlchòid (shortcode):

### Guile an sealladh chat (Chat Interface) {#customizing-the-contact-form}

Glaidh an Site Builder gu sgrùdadh an fòram:

> "Sgrùdadh an fòram de chàrrtaidh airson a bhith a' chùlchòid (subject field) agus a bhith a' chùlchòid (message field) a' chùlchòid (required)."

Bhaidh an Site Builder a' chàrrachadh an fòram agus a' tòiseachadh an chùlchòid a' chàrrachail.

### Guile an sealladh admin an plugin fòram (Form Plugin's Admin Interface) {#via-the-chat-interface}

Is eadar thu a tha a' chàrrachadh an fòram gu dìreach anns a' chàrrtaidh an plugin fòram:

1. Comas gu **Contact Form 7** (amar a bhith agat ann an admin WordPress).
2. Thoir an fòram a thabhair an Site Builder.
3. Chàrrachail na fòrraichean, riaghailtean chàrrtaidh (validation rules), agus na coimhreachdan email gu sònraichte mar tha thu a' chùlchòid.
