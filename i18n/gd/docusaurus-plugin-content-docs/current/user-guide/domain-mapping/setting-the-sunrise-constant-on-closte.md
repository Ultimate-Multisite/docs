## Ar Closte dashboard {#on-the-closte-dashboard}

Prìomhach, [leasadh an t-acaid Closte] (https://app.closte.com/), thaobh an meall a 'Sites' (Sites menu item), a losgadh an lìn Dashboard ar an site a tha thu a' obair air:

<!-- Screenshot unavailable: Closte dashboard showing Sites menu and Dashboard link -->

Tha e ag tòiseachadh gu mheas a' chuid nua de sinnean ar an lìn a' chùl a' chluais. Taisg an riaghladh gu pàirt a 'Settings' (Settings page) a' le sinnean:

<!-- Screenshot unavailable: Closte left sidebar menu showing Settings option -->

An uair a tha thu a' deabhausda ar **Settings**, dh'fhaigh an tab WP-Config, agus an fhaigh an fhaighinn "Additional wp-config.php content" ar sinnean:

<!-- Screenshot unavailable: Closte Settings page with WP-Config tab showing Additional wp-config.php content field -->

Mar a tha thu a' deabhausda air install Ultimate Multisite, rinn thu a constanta sunrise ar sinnean. Taisg a' chuid lìn ùr agus a' leasadh an lìn a' a tha a' leathair a' leathair a' leathair:

define('SUNRISE', true);

Tha e a' chuid, tha thu a' deabhausda. Ràdh an t-aistriú Ultimate Multisite agus ràth an lìn a' leathair a' leathair a' leathair airson a' dèanamh cinnteach gu sgaidh.
