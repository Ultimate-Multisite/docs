# Event: Register an account via Zapier {#event-register-an-account-via-zapier}

An article a' chleachdadh [Integrating Ultimate Multisite with Zapier](zapier.md) a' cheannachadh am fhaighinn a' chùineachadh le Zapier airson ealaichean cumhachdach ann an Ultimate Multisite gu fìor. San article seo, rinn am fhaighinn a' chùineachadh le app-ealaichean a tha sealltainn. Bidh sinn a' faighinn Google Sheets a bhith mar sinneadh datha agus a sgaidh an fhaighinn sin dhan Ultimate Multisite airson a fhaighinn a' chùineachadh.

Sgàil eadar Zapier a' cheannachadh, rinn dhan **Ultimate Multisite > Settings > API & Webhooks** agus dìreach a' faighinn an API Key agus an API Secret san sealladh seo gu bheil an API a' fhaighinn.

![API and Webhooks settings with API Key, API Secret, and Enable API options](/img/admin/settings-api-webhooks.png)

Tha eadar, tha thu a' faighinn **Google Sheet** ann an Google Drive agad. Bidh thu a' faighinn a' chùineachadh gu fìor air chùlann eachdraidh (column) gu bheil thu a' faighinn datha gu fìor a' chleachdadh a dhèanamh a tha thu a' faighinn sin.

Tha eadar Google Sheet a' cheannachadh, is eadar a' faighinn a' chùineachadh dhan account Zapier agad agus a' faighinn a' chùineachadh le zap a' fhaighinn.

An tairm airson an sealladh **"App event"** (Event an app) a' cheannachadh, rinn **"Google Sheets"**.

An uair a tha thu a' cheannachadh dhan "**Event**", rinn **"New spreadsheet row"** (Nàrr sinneadh sinneadh) agus thonadh **"Continue"** (Tòisich).

Tha an tairm eile ag fhaighinn a' chùineachadh airson **Google Account** (Account Google) a màir gu leathanas a tha an **Google Sheet** a' cur. Mar sin, dìreach a' faighinn a' chùineachadh gu fìor air an account Google sin a' bheachd.

An tairm dhan **"Set up trigger"** (Sealladh sealladh), thu faighinn a' cheannachadh agus a' cur an Google Sheet agus an worksheet (chluain) a tha thu a' faighinn datha gu fìor. Dìreach a' faighinn sin agus thonadh **"Continue"**.

An tairm eile is **"test your trigger"** (Teastaidh a' chùineachadh) airson a' faighinn a' chùineachadh gu fìor air an Google Sheet agad.

Tha eadar a tha thu a' teastaidh, rinn thu sgaidh a' chùineachadh a' faighinn datha gu fìor a' chluain agad.

Mar sin, mar a tha an teastaidh agad, rinn thu sgaidh a' chùineachadh a' faighinn daoine a' chluain agad.

An t-sealladh latha is a bhith a' chlàradh an t-aithris a tha a' creachadh no a' sgrìobhadh earrachda (account) anns an Ultimate Multisite. Air a' chlàradh an fhadhraidh, thànaidh "**Ultimate Multisite(2.0.2)**".

An t-aithris "**Event**", tha thu a' chlàradh "**Register an Account in Ultimate Multisite**" agus tha thu a' chlàradh an t-aithris "**Continue**".
