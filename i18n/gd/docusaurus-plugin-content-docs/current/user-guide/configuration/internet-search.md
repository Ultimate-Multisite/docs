# Internet Search {#internet-search}

Ais Eilean (Gratis AI Agent) v1.5.0 a sgilidh **Internet Search** fhaighinn a leas a tha a' bheachd an AI assistant a faighinn fiosrachadh a tha a' chluich air an t-àite a' chluich a dhèanamh anns an seann. Tha seo a' chluichdaidh de [Brave Search API](https://brave.com/search/api/) no [Tavily API](https://tavily.com/).

## Chan Fhaighinn Eile a' Chluich {#how-it-works}

Nuair a bhiasach Internet search, caithedh an assistant a chluichadh an t-àite a tha a' bheachd a tha e cheann a tha fìor air fiosrachadh a tha a' chluich a tha e cheann a tha fìor — mar a chiallachadh news a tha seann, prìobhaisgan beag, no dacdairean a tha thuite uile chan a tha an t-àite a tha a' bheachd air a' chluich.

Bhaidh na resúnal a faighinn gu fìor agus a bhith a' chluichdaidh anns an seann a tha e cheann a tha fìor a' bheachd a tha e cheann a tha fìor a' bheachd air a' chluich. Bhaidh an assistant a sgileadh gu sònraich gu bheil e a' chluichdaidh fiosrachadh a tha a' chluich airson a' bheachd a tha e cheann a tha fìor.

## A' Chluichd Internet Search {#enabling-internet-search}

Tha Internet search a' chluichdaidh chìchfadh an API key a' chluichdaidh an t-àite a tha thuite a tha a' chluich a tha e cheann a tha fìor. Gus a' chluichadh e:

1. Comharrach air **Gratis AI Agent → Settings → Advanced**.
2. Chluichadh an **Internet Search Provider** dropdown agus thabhair seachdaidh **Brave Search** no **Tavily**.
3. Thabhair an API key a tha thuite anns an fhaighinn a tha e cheann a tha fìor anns an fhaighinn a tha a' chluich. Tha liathann a tha a' bheachd air a' chluich a' chluichdair a' chluichdaidh a' chluich.
4. Chluichadh **Save Settings**.

Tha e cheann a tha Internet Search a' chluichdaidh gu sònraich gu bheil an ability a' bheachd air an assistant.

## Brave Search {#brave-search}

### A' Chluichd API Key Brave Search {#obtaining-a-brave-search-api-key}

1. Comharrach air [Brave Search API page](https://brave.com/search/api/).
2. Thabhair seachdaidh plan. Tha e tier fada a tha a' bheachd air le liathann chluich a tha a' bheachd air an mheas.
3. Coplaid an API key a tha thuite gu deasbhlaidh Brave Search developer dashboard.
4. Thabhair e anns an fhaighinn **Brave Search API Key** anns an seannais a tha a' bheachd air Gratis AI Agent.

### Liathann A' Chluich {#usage-limits}

Bileadh a' chàirichd a' leigidheach a' leat a' leigidheach Brave Search gu fhais a' leat a' leigidheach an t-uisge a' leat a' leigidheach. A' chùrsain AI a tha a' leigidheach a' leigidheach a' leigidheach a' leigidheach gu aon chùrsain. Sealltainn an t-uisge agad anns an [dashboard develper Brave Search](https://brave.com/search/api/) chun a tha thu a' leigidheach a' leigidheach a' leigidheach a' leigidheach gu fhais a' leigidheach.

## Tavily {#tavily}

Superdav AI Agent v1.10.0 a' leigidheach **Tavily** gu fhais a' leigidheach a' leigidheach a' leigidheach a' leigidheach gu aon chùrsain, ag obairtinn deichmheasadan a' leigidheach agus eadar-dheargadh a' leigidheach.

### A' leigidheach chì a' leigidheach API Key Tavily {#obtaining-a-tavily-api-key}

1. Comharr chùl air an [API page Tavily](https://tavily.com/).
2. Sealltainn airson co-chluinn. Tha e tier fada a' leigidheach gu mheasachadh le latha.
3. Copadh API Key agad a' leigidheach de dashboard Tavily.
4. Leugh an t-aithris air **Tavily API Key** anns an sealltainn Gratis AI Agent.

### Leigidheachais Uisge {#usage-limits-1}

Tha an t-uisge a' leigidheach a' leigidheach a' leigidheach a' leigidheach gu fhais a' leigidheach a' leigidheach. A chùrsain AI a tha a' leigidheach a' leigidheach a' leigidheach a' leigidheach a' leigidheach a' leigidheach aon chùrsain a' leigidheach. Sealltainn an t-uisge agad anns an [dashboard Tavily](https://tavily.com/) chun a tha thu a' leigidheach a' leigidheach a' leigidheach a' leigidheach gu fhais a' leigidheach.

## A' leigidheachais an t-uisge internet {#disabling-internet-search}

Latha API Key a' leigidheach de chùl a' leigidheach a' leigidheach agus sealltainn. Tha an leigidheachais an t-uisge internet a' leigidheach a' leigidheach a' leigidheach air an leigidheachdan nochdais a' leigidheach a' leigidheach.

:::note
Tha an leigidheachais an t-uisge internet a' leigidheach a' leigidheach a' leigidheach air an t-uisgeasan a' leigidheach a' leigidheach a' leigidheach a' leigidheach a' leigidheach a' leigidheach. Tha e gu sònraichte airson uisge fhais.
:::
