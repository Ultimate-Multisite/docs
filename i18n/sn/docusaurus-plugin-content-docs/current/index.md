# Zvinyorwa zveUltimate Multisite

Tikugashirei kuzvinyorwa zveUltimate Multisite. Ultimate Multisite inoshandura WordPress Multisite kuita chikuva chakazara cheWebsite as a Service (WaaS).

## Zvikamu zveZvinyorwa {#documentation-sections}

### [Gwaro reMushandisi](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

Zvese zvaunoda kuti umise, ugadzirise, uye utarise chikuva chako cheWaaS. Zvinosanganisira kuisa, nzira dzekubhadhara, kubatanidza domain, kutarisira vatengi, nezvimwe.

### [Zvinyorwa zveVagadziri](/developer/hooks) {#developer-documentation}

Referensi yehunyanzvi yevagadziri vari kuvaka paUltimate Multisite. Inosanganisira referensi ye hooks, zvinyorwa zveREST API, uye magwaro ekugadzira zviwedzerwa.

### [Zviwedzerwa](/addons) {#addons}

Zvinyorwa zveplugin dzezviwedzerwa zvinobhadharwa zvinowedzera Ultimate Multisite nezvimwe zvinhu zvakaita sekubatanidzwa kweWooCommerce, analytics, kutengesa domain, nezvimwe.
