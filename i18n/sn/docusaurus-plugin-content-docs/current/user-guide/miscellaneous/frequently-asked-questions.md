# Maswali Anayohulikizwa

Ndi zvinhu izvi zvinoreva **Maswali Anayohulikizwa** anozuva anonyanya kuita support requests.

  * **Chii ndicho Ultimate Multisite?**

Ultimate Multisite ndiyo plugin ya WordPress, inotora install yako ya WordPress kuti iwe uone network yakakwana ye sites – zvichiri se [WordPress.com](https://WordPress.com) – inozova kuita kuti clients vave vana sites kubva pamweya, kwapa mwezi, kana mwaka (unogona kutaurawo Free plans).

  * **Zvinotenderera Ultimate Multisite kunge kuenda pane Multisite?**

Haina. Ultimate Multisite inoda WordPress multisite, subdirectory, kana subdomain.

  * **Zvinoda kuti Ultimate Multisite zvinhu zvinodiwa?**

Ultimate Multisite inoda:

**WordPress Version** : v5.3+ (Inotenderera: Last Stable Version)
**PHP Version** : 7.4.x (Support ku > 8.0 kunoita zvakanaka)
**MySQL Version** : v5+ (Inotenderera: Version 5.6, kana iwe haona 8.0 kwenye hosting provider yako)

_Multisite (subdomain kana subdirectory) inofanira kuitwa kuti iwe uone._ [How to Install WordPress Multisite](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **Chii chii chinogona kuenda neMultisite neSubdirectories?**

Ehe. Ultimate Multisite inozviona subdirectories uye subdirectory installs.

  * **Chii chii chinogona kuenda neMultisite neSubdomains?**

Ehe. Ultimate Multisite inozviona subdirectory installs uye subdomain installs.

  * **Ndinogona kuita kuti Ultimate Multisite itore mhinduro pamusoro pePlugins uye Themes zvinogone kuwedzera users vavo?**

Ehe. NeUltimate Multisite unogona kutora zvinhu zvinogone kuwedzera plugins uye themes dzine kuwedzera kuiri pairi plan yako kwenye network yako.

  * **Ndinogona kuita kuti Ultimate Multisite itore mhinduro pamusoro pekuita kuti user anogone kuwedzera posts nguti?**

Ndi. Nezvino nemunhu wosemweya (Ultimate Multisite) unogona kuita post quotas (kuvaka quota dzemabhuku) kune **zvinovaka zvose zvinobva munzira yenyamba yako**. Unogonawo kunge kuita kuti umunhu anogone kuita kuita admin anogone kuita imba dzinogona, **kubata rudo (amount of disk space),** uye zvinhu zvakawanda.

  * **Ndi Ultimate Multisite inoshandisa Site Templates?**

Eyi. NeUltimate Multisite unogona kuita Site Templates kune kunge zvakawanda zvako. Site Templates hazi ndiyo themes; ndiyo sites dzine kuratidza munzira yako yenyamba. Unogona kuita plugins, themes, uye kuva neSite Template iye sei unoziva. Kana mutsva anobvumira iye template, site yake itai kuti iwe yakaita copy ya iye template, nekuti zvose zvinogone kuva necustomization dzakasiya.

Unogonawo kunge kuita template yakagadzirwa kune plan yese. Nekuti unoti, munhu wese anobvumira kuita plan iye anogona kuita site yake yakaita duplicate ya Site Template iye.

  * **Ndi Ultimate Multisite inoshandisa WooCommerce?**

Eyi!

Kana uchida mutsva wenyamba wako anogone kuita WooCommerce, kuti anogone kuita stores, izvi zvinoita neUltimate Multisite. Unogonawo kunge kubata rudo (limit) kwemabhuku anogone kuita vose vachange vachange kuita munzira dzinobva dzeplan.

Pamashure nemabhuku ekubhadhara (subscription payments) kune Ultimate Multisite, unogona kushandisa [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) kuti mutsva wenyamba wako anogone kuita kubhadhara nekubata gateways dzine WooCommerce munzira yako yenyamba. Nekuti izvi, unogona kuta zvinhu zvinobva dzepayment gateways dzakagadzirwa neWooCommerce uye kuzviona kuti uita billing kwemutsva wenyamba wako weUltimate Multisite.

  * **Ndi kune nzira yekubata subscriptions kubva kuWPMUDEV Pro Sites?**

Ndiye, Ultimate Multisite ndinenge kuenda kuona [zvikonzero zvakakubatsira kuti uende migiroro (migrate)](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) kwenye mweya wako we Pro Sites network.

  * **Ndiwo chii chii chinokwanisa tab ya “Allowed Plugins” yakabva yakanyanya kuona munzira yekubeditanisa (Plan Edit Page)?**

Ultimate Multisite inozivisa kunenge uingine kana ukavishandisa plugins dzakachifamba kwenye network yako. Kana plugin yakachifamba kwenye network, nekuti iweya, itaiendwa nekuita zvinhu zvinotanga (default) munzira yese ye sites dzako.

Kana usine kuona plugins nekuona tab ya “Allowed Plugins”, ndichida uone kana plugin unozvikudzidzera inoshandiswa kwenye network yako kana haisi.

  * **Ndinokwanisa kushandisa nzira yangu yekupfuma (Pricing Table design)?**

Ehe. Kuteerera v1.4.0, Ultimate Multisite inoshandisa template overrides.

  * **Ultimate Multisite inogona kushanda nemweya we WP Engine?**

Ehe. Ultimate Multisite inozivisa zvakawanda neWP Engine, asi kuti domain mapping itaienda kuenda kunoda zvinotanga (manual extra steps) kubva pamusoro yemadmini (kuita domain yese kwenye WP Engine panel). **Iyiweya haina chii mune v1.6.0.** Ultimate Multisite inoshandisa neWP Engine API nekuti domains dzaiyenda kuenda pane automatically panguva mapping.

  * **Ndinokumbira munhu anoda kuti ndizivisa (recommend) we hosting provider?**

Ultimate Multisite inogona kushanda nemweya wese we hosting provider. Chii chinotanga kunenge chii chimwe, nzvimbo dzinhu dzakaita Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane uye [Runcloud.io](http://Runcloud.io) inobva neintegration yakakwana kune domain mapping pane zvinhu zvinotanga (out of the box). Kune hosting dzine kunzwisisa kwakasiyana, unogona kuita domain mapping manually.
