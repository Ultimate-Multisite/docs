# Kutiya Panel yako nePRO Theme

Kati kutaura na wainene vachikuru, kungekuva kuti tichida kuva nemashoko ekutenderedze kwakawanda munzira yakare ya admin dashboard ya WordPress. Kana uchitera kuona zvinobva nezinovaka chinobva, PRO Theme inogona kukubatsira.

PRO Theme ndiyo theme ya admin inogadzirwa kuti ibatsire panguva unoremekedza Dashboard ya wainene vako. Inoonekawo kunoita chiiwo chekuti inodiwa munzira yako yekutenderedze. Inoonekawo kunoita chiiwo chekuti inodiwa munzira yako yekutenderedze.

Panguva unigona kuichinjisa, enda pakupfupisa kwako _Plugins_ uye chibvumira kuti uichinjise pane Network yako.

<!-- Screenshot isina: WP Plugins page inoshow Network Activate button for PRO Theme -->

_Chinjisa pane Network yako munzira ya Plugins_

_**CHINTO:**_ _Kana uri kuichinjisa chete, ichishowira panguva wainene vako se chiiwo chinogona kuva kune kunge kuita theme chete kwemushandisana wese.

Nekuti zvakaita, unenge uine interface ya admin yako yakare! Unenge uine menu yakare munzira yekutanga inosanganiswe nePRO Theme.
