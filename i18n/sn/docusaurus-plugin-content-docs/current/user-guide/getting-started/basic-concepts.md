# Dzidziso Zvinhu {#basic-concepts}

Kune mutsva wenyu we WordPress Multisite uye munhu anowanikwa kunge achatanga kuenda nekuita zvinhu zvakawanda. Kuti unzwisise zvinhu izvi, ndiyo zvinotora nguva yakawanda nekuti uchida kuona kuti uri kuona platformi iyo inogara sei kumusoro wose.

Muriwezi weye, tinoda kuita zvekuona uye kupepa zvinhu zvakakosha mumweya we WordPress. Zuva refu ndizvo zvinotora kunzwisisa kwemumweya weye, zuva refu zvinotora kunzwisisa kune vatsviki, uye zuva refu zvinotora kunzwisisa mabvumwe.

## WordPress Multisite {#wordpress-multisite}

**Multisite** ye WordPress ndiyo uri weWordPress inoziva kuti unogona kuita nekuita nekugadzira uye kuita zvinhu zvakawanda (manage) nemapuro dzakawanda ekubereka zvinobva mupfumbiro weye. Unogona kuita zvepuro dzese, kuita zvemafundo, themes, uye user roles. Unogona kuita nekutanga kunge mhamweya nekuita zvinhu zvakawanda (hundreds and thousands) zvinobva mupfumbiro weye.

## Mupenyu wekudzi (Hosting provider) {#database}

Mupenyu wekudzi ndicho chinhu chinotenderera kuti vanhu nevanhu vachikuru vave vanyane vana zvigone kuita zvinangwa zavo kubva munzvimbo yezuva yakawanda (World Wide Web). Zvinhu zvinhu zvinotendererwa nempenyu wekudzi zvichakanganisa, asi chinogona kuita kuti zvinhu zvakaita se kupfeka website, kunge kuratidza space dzemakore pamusoro pehost, uye kuonekera nekutaurirana neInternet.

## Domain (Mabasa) {#hosting-provider}

Domain name ndicho chiri chiri chinotarisirwa vanhu kuti vachifare site yako. Chinoreva kune browser kuti rive kuti riveitei site yako. Sei zvinhu zvinogona kuita, domain ndiyo nzvimbo inoshandiswa nevanhu kuti vanyire website yako online. uye, sekuva nechirevo munzvimbo yakako. Kana uchida kuenda kwenye website yedu, unofanira kunyora address yawe mu browser's address inozivikanwa [_www.ultimatemultisite.com_ _, _where [**ultimatemultisite.com**](http://ultimatemultisite.com) is domain name._

## Subdomain (Subdomain) {#domain}

Subdomain ndiyo aina ya hierarchy ye website inosvika pamusoro pedomain main, asi pansi nekuita content organization munzvimbo yakawanda, inogona kuva website yayo yenyu. Inoonekwa se [**https://site1.domain.com/**](https://site1.domain.com/) where _site1_ is the subdomain name and [_domain.com_](http://domain.com) is the main domain.

## Subdirectory (Subdirectory) {#subdomain}

Subdirectory ndiyo aina ya hierarchy ye website inosvika pamusoro peroot domain inoshandisa folders kuti kuita content organization munzvimbo yakawanda. Subdirectory ndiyo subfolder, uye mabasa anoita zvinhu anogona kushandiswa kubva mupenyu wese. Inoonekwa se [**https://domain.com/site1**](https://domain.com/site1) where _site1_ is the subdirectory name and [_domain.com_](http://domain.com) is the main domain.

## Subsite {#subdirectory}

Subsite ndiyo chine subsite unogona uri wako unogona kuita munzira yakachengetedza (Multisite network). Ino hingarama **subdomain** kana **subdirectory**, kune zvinhu zvakasiyana-nyanya kubva pamwe nepamwe sei WordPress Multisite yako inenge imapfumbwa.

## Super Admin {#subsite}

WordPress Super Admin ndiye user role inenge achiita zvose kuita subsites pane Multisite network. Kune vanhu vachiti munzira ya Multisite, iye ndiyo **kuvaka chokwadi chekuchengetedza (highest level of access)** chausi unogona kupa kuWordPress yako.

## Plugin {#super-admin}

Mune chinangwa, plugin inenge setero ye code inenge ichiita zvinhu zvakawanda kuWordPress site yako. Izvi zvinogona kunge zvinhu zvakaipa nekuurudza logo ya login kana zvinhu zvakaoma nekuita e-commerce functionality. _Woocommerce_ uye _Contact Form_ ndine zvakanaka kubva pamwe ye plugin.

Munzira ya WordPress Multisite, plugins inenge kuinstalla nemaoko kuno dashboard yechikuru (network admin dashboard) nekuti Super Admin. Subsite Admins vanogona kuactivate uye ku-deactivate plugins pane subsite yavo chete.

## Themes {#plugin}

WordPress theme ndiyo group ye files (_graphics, style sheets, uye code_) inenge ichiita zvinhu zvinoreva kunzwisisa (overall appearance) ye site. Inenge ichiita zvose zvefront-end styling sefont styling, layout ya page, colors, kana zvinhu zvakasiyana.

Maviri anogona kuinstalla munzira ya WordPress Multisite neSuper Admin uye vanogona kuactivate pane subsite level neSubsite admins.

## Site Template {#themes}

**Site Template** ndiyo site inenge yakachengetedza (boilerplate) inenge ichiita zvakasiyana kana iwe unenge uchitika kuita subsites mumapfumbwa yako.

Iyi inogona kuva kuti unenge uita base site, kuactivate plugins dzakasiyana, kuita theme inenge yakachengetedza uye kunge kune kuratidza (customize) sei unenge uchitika. Ndiyo, kana mutenda wako anenge achiita account yenyu, pane kunge kuita WordPress site inotanga nekuva nenzira isingazvinogona kuita zvinhu yakasiyana, anenge achiita copy ye base site yako inenge ine customizations uye contents dzakachengetedza zvinenge dzinowanikwa.

## Domain Mapping {#site-template}

**Domain mapping** neWordPress ni nzira yekuti user vave kuenda ku host wavo wose, kubva munzvimbo dze website. Mu WordPress Multisite, subsites zvakwanisa kuwedzwa nekuti zvinogadzirwa mune subdirectory kana subdomain. Domain mapping inotora kuti users vesubsite vanogone kushandisa top-level domain zvakaita sei [**joesbikeshop.com**](http://joesbikeshop.com) kuti address ye site yavo ine kunzwisisa kwazvo.

## SSL {#domain-mapping}

SSL inoratidza **Secure Sockets Layer**. Iyi certificate ya digital inozvinononwa (authenticates) chiri chete chete chewebsite uye inopa connection inotevera (encrypted). Nazi nezuva izvi, inogona kuwedzwa setechnology yakakodzera yekuti connection ye internet iwe ine kura uye inozvaka data dzine kunzwisisa kuti zvinhu zviri kuteveredza pakati dze systems, inokurudzira vanhu vakavandudzwa kuona uye kupfura nhaka dzose dzakatarwa, kusanganisira zvinogone kubva munzvimbo dze kunzwisisa. Browsers dzakakwana dzinoda SSL, izvi zvinoita kuti iwe uone kuta kana ukapfura website.

## Media {#ssl}

Media inoratidza images, audio, video, uye files dzine kunzwisisa zvinogadzirwa kuwebsite.

Network sites dzinopa database imwe chete mune WordPress Multisite, dzinogona kuwedzera paths dzakasiyana-siyana pamusoro pefilesystem kuti media files zvinhu zviri wose.

Location ya standard ye WordPress (wp-content/uploads) inoratidzorwa; asi path yake inobva kutaura kuti itanatsanangurira ID yakagadzirwa y network site. Nekuti izvi, media files yemweya ye network site inogona kuone sei wp-contents/uploads/site/[id].

## Permalinks {#media}

Permalinks ndiyo URLs dzakare dze post kana page yako individual mune site yako. Permalinks dzine kunzwisisa kuti **pretty links**wo. Nechokwadi, WordPress URLs dzinogona kuita query string format inoratidzorwa sei:

[**http://www.example.com/registration**](http://www.example.com/registration)

## Ultimate Multisite {#permalinks}

Ultimate Multisite ndiyo plugin ya WordPress, inotora install yako ya WordPress iwe nzira ya mtandao wa tovuti za kiwango cha juu – zvichineka [WordPress.com](https://WordPress.com) – kunoita wateja kuunda tovuti kupitia malipo ya mwezi, mwezi mmoja au mwaka (unaiweza pia kuunda mipango ya bure).

## Checkout Form {#ultimate-multisite}

Checkout Form ndiyo fomu ya agoti inayooneka moja au inayooneka kwa hatua kadhaa. Inahusisha kuunda subsite, usajili wa wanachama, na akaunti za watumiaji kupitia usajili wa Ultimate Multisite. Inahusu sehemu mbalimbali na fomu za malipo ambazo mtumiaji anapaswa kuzituma wakati wa kusajili.

## Webhook {#checkout-form}

Webhook (inaita pia web callback au HTTP push API) ndiyo njia ambayo app inatoa habari kwa programu zingine kwa wakati halisi. Webhook inawasilisha data kwa programu zingine mara tu inapotokea, kunyanyiwa kuti uwe na data haraka.

**Ultimate Multisite webhooks** zinafungua fursa zisizo na mwisho, zikuruhusu watawala wa mtandao kufanya aina zote za mchanganyiko unaowapendeza lakini wenye manufaa, hasi paya inatumika pamoja na huduma kama _Zapier_ na IFTTT_.

## Events {#webhook}

Event ndiyo kitendo kinachotokea matokari ya kitendo cha mtumiaji au chanzo kingine, kama vile kukumbuka (mouse click). Ultimate Multisite inahifadhi rekodi ya vitendo vyote na log zinazoendelea ndani ya mtandao wako wote. Inafuatilia shughuli mbalimbali zinazotokea katika multisite yako, kama vile kubadilisha mipango.
