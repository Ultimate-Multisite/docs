# Kufamba plani (v2)

_**CHINARI CHINARI: Iriweyi iri rinorefererwa ku Ultimate Multisite version 2.x.**_

Kufamba kana plani kana subscription ndiri chirevo chinotevera kuti vachine client vachitenda kana vane rudo raunenge rine budget yakapfuura kana vakati vachiti vanovaka zvakawanda kuti vave vachida kuita subsite yavo.

## Kuita sei kufamba plani

Vachine client vanogona kufamba plani yavo mazuva ose nekuita login ku dashboard ya admin ya subsite yavo uye kupfeka **Change** pano pachikamu chepaka pacho (account page).

![Customer account page with Your Membership card and Change button](/img/account-page/membership-change-button.png)

Pachipfeka button ye **Change**, user/client anogona kuenda ku checkout page unozivaka plani yakaiita kuti vave vashandane subscription yavo.

![Downgrade plan options page on the customer side](/img/account-page/downgrade-picker.png)

Mune iwe, tinofamba plani kubva **Premium** ku **Free**.

Kuti utande, user anofanira kupfeka button ye **Complete Checkout**. Izvi zvichatanga kuti vave vashandane paka account page inotarisirwa (pending change) inotarisirira chirevo chemembership. Mutaura dziri dzinogona kuita effect panguva yakare yechine client.

![Account page showing pending membership change banner](/img/account-page/pending-change.png)

### Chii chinotevera kana user kufamba plani yavo

Zvinotarisirwa kuti kufamba plani hachishandane configuration yakaiita panguva yauser subsite.

Hachishandane template ye site nekuti kushandana template kunogona kuenda kupfuura uye kudzera (erase and reset) subsite yese. Izvi zvichitendwa kuti vave vanokwanisa kupera kusawana kwakawanda kwe data. Saka disk space, themes, plugins kana zvinhasi zvinhasi zvinotarisirwa zvichinja asi posts.

Tiri zvinhu, tinoziva kuti chero chinhu chinotiri kwako chiri kuita kunzwisisa pamusoro pe limits uye quotas dzakasiyana dzinogadzirwa panguva dzese, asi tiri kuona kuti kune zvinhu zviri kuita kunzwisisa kwemushandawane we subsite kana tikazvisanisa kana kusvika sei zvinotevera.

Pansi pe posts dzakasiyana dzinotevera limit raunogadzirwa panguva dzese, unenge uchida kuti uone zvinhu zvinotevera: **Ramba posts zvako** *,* **Simba posts kune trash** *,* kana **Simba posts kune draft** *.* Unogona kuita iye pano mumakore anogadzirwa panguva dzese.

![Network Admin Settings Sites page showing post-limit-exceeded behaviour options](/img/account-page/settings-sites-post-limit.png)

### Chii chiri kuitika pamusoro pe payment

Muvero we 2.0, haunoda kunge kuendesa zvinhu pamusoro pe proration (kuti zvinobva nekuita).

Iyi ndiyo nekuti system inotarisira kuti membership yakanaka yakaita **kuramba kuratidza cycle yayo yekubhadhara panguva dzese pane** kuti plan/membership yakare itevere. Rakaonwa zvakarero rwe mubhadharo we new membership riri kuita uye rikushandiswa nekukurumidzo kwemushandawane wekuramba kuratidza cycle yayo.
