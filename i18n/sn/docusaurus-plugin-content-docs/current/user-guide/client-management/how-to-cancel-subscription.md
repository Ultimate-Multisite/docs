# Unguva sei chibvumiro? (How can I cancel my subscription?) {#how-can-i-cancel-my-subscription}

Kana uri Ultimate Multisite license, unogona kuva kufamba nekuvuma kuti uunyangwe (renew) mupfungwa yako. Ramba unenge utsvaka zvakanaka:

<!-- Screenshot unavailable: Account page email with access link -->

Gadzira email yakasiita nguva yakagadzirwa uye password yako.

<!-- Screenshot unavailable: Login page with email and password fields -->

Iyi ndiyo chiri chiri chepfungwa yako ya Account Page:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

Kuti ununge (cancel) unyangwe wako, munzira (menu) inosvika pamwe mupfupi, endo **Renewing & Billing**.

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

Chibvumira (arrow) chiri pamwe, kuti ufungire window yechipfupi. Ndiyo, unofanira kutaura chii chii chinoreva kuva nekuva **auto-renew**.

System ichoichivaka kunge kuita confirmation message.

Kana uri kuona kuti unoda kufamba nekuva nekuva, kutaura button inosvika **Cancel Renewals**. Mushaka iyi, uchizokwanisa kuenda kuita survey yakafanana.

Kugadzirwa! Unyangwe wako hauzovuma kuunyangwe nekuti uunyangwe wako unozivikanwa (valid key) kunge kuva. Kana uri kuona kuti unoda kuenda nekuva, unofanira kuratidza zvakanaka.
