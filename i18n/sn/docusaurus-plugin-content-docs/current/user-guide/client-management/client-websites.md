# Websites Vachikwa Vechikondiwe (v2) {#your-clients-websites-v2}

Panda kuingira mweya wenyu munzvimbo yenyu, vachikondiwa vana vachangopara kuona dashboard yavo.

Kati pe **Account** page, inotiri kuti vanzwire zvinhu zvinotarisirwa nezvekuita nekuita (payments), membership, domains, plan limitations uye zvinhu zvakawanda, pane panowo kunenge kune **My Sites** page, inotiri kuti vone uye vashandise zvinhu zvavo zvose zvewebsites. Zvose zvinoita ndiri ku-click "My Sites" pa dashboard yavo.

![My Sites link in the customer dashboard](/img/account-page/my-sites-link.png)

Vachivonwa zvose zvese websites vavo, ne button ya **Manage** imiiri pamusoro peiri. Kana vano-click, vachibva ku dashboard yese website iye.

Maviri vanozviona kuti upi wewebsites wavo uri we **primary** (mupfungwa) kana plan dzine zvinhu zvinoita kuti vanowe websites yakawanda.

![My Sites page showing customer websites with Manage buttons](/img/account-page/my-sites-page.png)

Iyi ndiyo nzira yose yekuti vone interface yeitenziwe (site edit):

![Site edit full interface](/img/admin/site-edit-full.png)

Kana vanoita kuti vave website primary, vanofanira kuenda pa Account page neku-click "Account" menu.

![Account menu link in customer dashboard](/img/account-page/account-menu.png)

Vanogonawo kuona uye kutaura zvinotarisirwa nezvemutauro wavo (customer details):

![Customer edit interface](/img/admin/customer-edit.png)

Zvakanaka, vanofanira ku-click "Change Default Site".

![Change Default Site button on account page](/img/account-page/change-default-site.png)

Chivimbo chinotarisirwa chachirochiri (pop-up window) chinotarisirwa. Iyiya drop-down list, uye vanoreva upi wewebsite anoda kuti uve primary website.

![Change Default Site dropdown selection](/img/account-page/change-default-site-dropdown.png)

Zvakanaka, vano-click "Change Default Site" uye ndakubatsira!
