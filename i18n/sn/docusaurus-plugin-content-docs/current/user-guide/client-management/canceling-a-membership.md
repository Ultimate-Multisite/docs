# Kuitira Membershipi (v2) {#canceling-a-membership-v2}

_**CHINDIRA CHINDIRA: Iriweyi iri rinorefererwa ku Ultimate Multisite v2.x.**_

Kuitira membership kubva paunogona zvakanyanya kuitika.

Kana mutenda wako anoda kuitira membership yake, anofanira kuenda pane **Account** page yake pane kuteerera.

Pangu, anofanira kuva kuita head ku **Danger Zone**, uye kuita kuti **Delete Account**.

![Account page Danger Zone with Delete Account button](/img/account-page/danger-zone.png)

Modyuli imbeita inoda kukumbira kuitirwa kwake. Anofanira kuva anogona kuonetsa (toggle) option ya **Confirm Account Deletion** uye kuita kuti **Delete Account**.

![Confirm Account Deletion dialog](/img/account-page/delete-account-confirm.png)

Izvi zvichadzisa kuti avimbiswe (log out) pane site yako uye kuitire membership yake pa network yako, uye kunyera site yake kubva pa network yako.

![Logged out after account deletion](/img/account-page/logged-out-after-delete.png)

![Membership canceled confirmation](/img/account-page/membership-canceled.png)

**Zvinoreva:** Zvinhu izvi hazvachizovalira mutenda wako pa network yako, sezvo mutenda anogona kuva nemembership yakawanda kana haana membership yakanaka kubva kwake.

![Customer still exists in the network after cancellation](/img/admin/customer-after-cancel.png)

![Customer with no active memberships](/img/admin/memberships-customer-empty.png)
