# Ungava kuenda pa ukunguva kwawo pa account yako? {#how-can-i-access-my-account-page-for-the-first-time}

Muliye kuamukana mu Ultimate Multisite!

Ungava kuenda pa ukunguva kwawo pa account yako kupfungwa ne link unengewe m'email nyowani apo uyo muaunza plugin yedu:

<!-- Screenshot unavailable: Account page email with access link -->

Gara ne email uyo wako uyo wakaunza mu kunzi ne password uyo wakaunza mu email.

<!-- Screenshot unavailable: Login page with email and password fields -->

Chiri chiri, iwe nguva yakakare yakanzi kuingira mu account iyi, vachida kuti utange kugara ne password yako. Gara neya uye click button ya _Save_.

<!-- Screenshot unavailable: Change password form with Save button -->

Pa account yako, unogona kuramba kutaura zvose zvinotipa ngozi dzako pachikamu chete, zvichiri kuita zvinhu zvakawanda.
