# Kurudzera Kufamba neMembenship (Managing Memberships)

Mu Ultimate Multisite, unogona ukubata nekuvaka nekuvaka izinhlelo zokubhalisa (memberships) nezokubhalisa zonke izinkosi (subscriptions) ezinekhaya yakho. Ungava kuva nekuva nekuva zvitendelero zezi membership, zvichiri kutaura zvinhu zvakaita plans uye products, kunge kuita nani dzimwe sites, kunge kuita nani kwemashure ekubhalisa (billing amount) uye nguva dzinogona kutamba (frequency), nzira yakazara ye-payment history, uye timestamps.

Kuti utange ipeji iyi, ndichida kuti uende ku **Memberships** tab mu menu ya Ultimate Multisite.

![Memberships list page](/img/admin/memberships-list.png)

Pa Memberships page unogona kuona list ye vafungamiri (customers) vako uye products dzinekuita nazo accounts. Ungava kuita filtering kwe memberships nestatus yavo: active, pending, on hold, expired, kana cancelled.

Memberships dzinobva dzinovaka (Free memberships) dzichingizwa kuti dzinogona kutamba kune **lifetime** membership. Dzinofambanira kunge kuita expiration date uye hazvazovaka nekuita zvakanaka. Kana plan yakafamba kufamba panguva yakaita trials kana nguva yakaita fixed, unogona kuita product inotamba (paid product) inine trial, kuita cancellation manually yembenship, kana kuita customer iwe kune plan inotraura kana access yawo inoda kutamba.

Chiramba ne membership kuti uende kupeji inotamba zvinhu — billing info, site or visit limits, uye zvinhu zvakawanda.

![Membership edit interface](/img/admin/membership-edit.png)

Ndiye full view ye peji ya kuramba (membership edit page):

![Membership edit full interface](/img/admin/membership-edit-full.png)

Ungava kuita nekuvaka memberships dzinotsanana neusers vakare vakaenda kwenye mu network yako. Chira **Add Membership** uye utange zvinhu zvinotevera (configure the details).

![Add Membership modal](/img/admin/membership-add-modal.png)

Zvakanaka, ndichida kuti uita **Create Membership**. New membership inozvaka kuone pangu list ye memberships.
