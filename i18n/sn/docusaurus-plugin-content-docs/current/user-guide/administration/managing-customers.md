# Kurudzera Vamwe (Managing Customers) {#managing-customers}

Ukupfupi we **Customers** kunokwona list ye vana vakareka vachiri munzwe wenyu. Ipeji iyi inokwona maheniso arapfumo anonyanya, zvichisinga zvinobva neitaura, nguva yakakwana yekutanga kuitanga (last login), uye link ya **kuuta ku** inokupa kuti utange kutanga mune subsite yeumunhu se administrator we site.

![Customers list page](/img/admin/customers-list.png)

Chirareka **Manage** pamusoro pe card ye munhu kuti uve nekuona maheniso ekutanga pane munhu iye.

Papeji yekutanga yemunhu, chiri kuona block we **At a Glance** chinokwera ID ya munhu, nguva yakakwana yekutanga kuitanga, uye kuita zvinhu zvakawanda anogona kutanga munhu iye.

Block we **Customer Options** inokupa kuti uita iye munhu se **VIP** (tag inokupa kuratidza zvakanaka). Tab ye **Billing Info** inokwera maheniso ekutanga zvichiri kuitanga, zvakaitwa, uye dinda.

Block we **Fields** inokwera username ya munhu iye, email, uye inokupa kuti uunetse status ye **Email Verification**.

Pamabasa akare pamusoro pe Customer Options, uchapota blocks chete chete:

- **Memberships** — Zvinokwanisa zvakawanda anonyanya munhu iye.
- **Payments** — Kuratidza kwekuti munhu iye akanga anopai.
- **Sites** — Sites munhu iye anowaka.
- **Events** — Zvinhu zviri kuita nemunhu iye (kufamba kwememail verification, kusimbisa maheniso, etc.).

Chirareka icon ye arrow (→) pamusoro pe chii chii kuti uende pane pepa yake. Sei, chirareka arrow pamusoro pe site chinokupa kuenda papeji yekutanga yedit site unokwanisa kutaura membership association, site type, uye status yakati.

Chii chinogona kutanga pamusoro pe memberships uye payments — zvimwe zvinokwera papeji dzakutanga dzinokupa kuti uedit status ye membership, munhu anonyanya, products, settings ye payment gateway, uye kuda kuita refunds.

Pamapfumo pane customer page, uchapota:

- **Kupeduro uye Zvinovaka (Last Login & IPs)** — Ndatenda nguva yaunoda kuita, chishoma chekuita (IP address), uye dzimba dzawo (country)

- **Zvinoreva Kuti (Timestamps)** — Ndinonaka kunzwisisa kana akauntina rauno racho rakuva nekuti wakare wakanaka.

- **Kuteka/Kusvika (Delete)** — Ndiri kuona chokwadi chekuti unogona kudera akauntina rako kubva.

:::warning
Kupedzisira, kudera akauntina kunoita zvinotora nguva yakawanda uye hakuna kurudzirwa.
:::
