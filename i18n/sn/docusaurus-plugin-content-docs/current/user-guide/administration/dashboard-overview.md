# Zvinoro Rino (Dashboard Overview) {#dashboard-overview}

Ultimate Multisite inenge **Dashboard** wosei wechipe chiri mune yechinyorwa (menu), uye inenge inopa widgets kune Network Dashboard. Iri rurimi rudo rine kuona mumweya wese.

## Network Dashboard {#the-network-dashboard}

Ultimate Multisite inenge inopa widgets dzinogona kutanga mumweya we-network, unogona kunzwisisa pamwe chete **My Sites → Network Admin → Dashboard**.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

Iyi ndiyo zvinhu zvinoreva kuti widget dziri kuopa:

**First Steps:** Inenge inowanikwa mumweya wekufamba kwakare uye inokupa chirevo chekuita zvinhu zvekuita, zvichiri kuita sekuvaka nzira dzehupenyu (configuring a payment method) uye kuita account yekutsimbisa.

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Summary:** Inoreva kuita nani nevanhu vachifamba uye mari yakagadzirwa muno. MRR (Monthly Recurring Revenue) inoreva kuti kune vanhu vose anogona kuita mari kubva pamwechete wekugadzira mari.

![Summary widget](/img/admin/network-dashboard-summary.png)

**Activity Stream:** Inoreva zvinhu zvakasiita mumweya wese wa Ultimate Multisite — kuva kwevanhu, kupera kwemhando (cancellations), kuva kwezvinangwa dzine nzira dzehupenyu (site creations), kupera kwezvinangwa, uye zvinhu zvose zvakasiita.

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** Inoreva mhinduro yakafanana yezvinhu zvevanhu nezinangwa mumweya wese, inopa shortcuts dzeKutanga nzira dzeva zinangwa (create new sites) kana vanhu.

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** Inenge inopara uye inoreva zvinhu zvinotanga mumweya wese wa Ultimate Multisite, zvichiri updates, bugs dzakakosha, uye mhinduro dzehutanga (security fixes).

![News widget](/img/admin/network-dashboard-news.png)

## Ultimate Multisite Dashboard {#the-ultimate-multisite-dashboard}

Kune chiri chiri chaizvo, network dashboard inopa maererano akakosha paneweye pane network yese, kunyanya ultimate multisite dashboard (panguva rechipe chete chemenu), inopa maererano anogona kuita service yese.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Kushuru Kwemiti Kuremu KuneMwe:** Inooneka uchikuru hwemiti hwakare over mwe wese wekare. Hwemiti hachekare (New revenue) hoshwa mune chiviri, uye kupera kwezvimbo (cancellations) hoshwa mune chiviri.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**Kugadzirisa Mazuva:** Onge kuenda kuteerera data ye dashboard nekuvaka nguva kuti uone zvinhu zvako zvinodzidzwa. Kuti mazuva ekare (date range) inogadzirisa zvinhu zvichionekwa mu widgets dzakare.

![Date range and dashboard stats](/img/admin/um-dashboard-date-range.png)

- **Revenue** — Hwemiti hwose hweperiod neMRR uye kugadzirisa kwezvinhu zvakare (per-product breakdowns).
- **New Memberships** — Kuratidza kuita kunge nani kwemembership dzakare mumwe mwe wese wekare.
- **Signups by Countries** — Kugadzirisa kune dambudziko rekuita signups nechina (geographic breakdown of signups).
