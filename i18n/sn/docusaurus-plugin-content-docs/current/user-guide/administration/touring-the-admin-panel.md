# Kuenda Kuti Admin Panel (Touring the Admin Panel) {#touring-the-admin-panel}

Ultimate Multisite inotora kuti iwe uone kuti ndiri nyore uye kunzwisisa, asi nekuti iweine chine zvinokosha zvakawanda, kana kuti iweine tour yakagadzirwa. Tiri kuenda pamapuro dzako dzinopuro kuti ndikubatsire utange uri munzvimbo.

## Dashboard (Dashboard) {#dashboard}

**Dashboard** ya Ultimate Multisite inonyora zvinokosha zvepuro uye analytics, zvichisinga nekushara, kunge kuratidza kuita zvinhu pamusoro pehitori, kudzera kwemabhiri, vanhu vakawanda vachiti, uye data dzehuni.

![Ultimate Multisite Dashboard bottom section](/img/admin/um-dashboard-bottom.png)

![Network Dashboard full page](/img/admin/network-dashboard-full-page.png)

## Checkout Forms (Checkout Forms) {#checkout-forms}

**Checkout Forms** inopa chokwadi kuti uve nekuita registration pages dzako dzinogadzirwa. Unogona kuva nekuwedzera fields, kuingira classes uye scripts dzako dzinogadzirwa, uye kupera zvinhu nekuti unoreva country kana location ya munhu anopinga.

![Checkout Forms list](/img/admin/checkout-forms-list.png)

## Products (Products) {#products}

Section ye **Products** ndiyo inotanga kuita plans dzakasiyana, packages, uye services dzinogadzirwa kwemabasa yako. Unogona kuita zvinhu zvepuro, kuratidza nzira dzehupenyu hwemabhiri, uye kuita limits & quotas pamusoro peproduct.

![Products list](/img/admin/products-list.png)

## Memberships (Memberships) {#memberships}

Page ye **Memberships** inonyora subscriptions dzese dzine munzvimbo yako. Unogona kuone uye kutaura zvinotevera zvesubscription zvichisinga nemabhiri, products, nzira dzehupenyu hwemabhiri, uye timestamps (timestamps).

![Memberships list](/img/admin/memberships-list.png)

## Payments (Payments) {#payments}

Upeji wepupi wese **Payments** unonyera kungoona nzira yakapfuma yepayments panyera pamusoro pezvinangwa zvaunenge vose mu network yako, nekuona maerer anokwanisa uye kuona zvinhu zvakakosha nezvematumiki nekuona matumiki.

![Payments list](/img/admin/payments-list.png)

## Customers {#customers}

Upeji we**Customers** unoshowira vese vanhu vakareka vose vachiri mu network yako nekuona maerer anokwanisa kubva, email address, nguva yakakwana yekutamba, uye link ya "switch to" yekuita kuti uingire ku subsite yavo.

![Customers list](/img/admin/customers-list.png)

## Sites {#sites}

Upeji we**Sites** unoshowira vese subsites vakareka vose mu network yako zvichiri kuona site templates uye subsites dzakareka nevanhu. Unogona kuita zvinhu zvakasiyana semanzo (mapped domains), limits & quotas, plugins uye themes, uye thumbnails dzeiteita.

![Sites list](/img/admin/sites-list.png)

## Domains {#domains}

Upeji we**Domains** unotsanangurwa kwemafanana anogadzirwa (custom domains) anogone kuita subsites. Se super admin, unogona kuita zvinhu zvekuita kana kuita mapping kwecustom domain names uye kuona DNS records nelogs.

![Domains list](/img/admin/domains-list.png)

## Discount Codes {#discount-codes}

**Discount Codes** kunoita kuti uingine coupon codes kuti utange discounts kune vanhu vose vakareka. Unogona kuita zvinhu nezvevunhu kana groups dzakasiyana dzinogone kuva target.

![Discount Codes list](/img/admin/discount-codes-list.png)

## Broadcasts {#broadcasts}

**Broadcasts** ndiyo tool yakagadzirwa kuti utange announcements kana kutaura messages dzakareka kune vanhu vose vakareka mu subsite yako. Unogona kuita target specific user groups nekuti plan kana product yakati inozivikanwa neva.

![Broadcasts list](/img/admin/broadcasts-list.png)

## Settings {#settings}

Upeji we**Settings** ndiyo pano unogona kuitira configuration kweUltimate Multisite — settings dze registration, payments, API uye webhooks, mapping dze domains, uye zvinhu zvakasiyana dziri kuita integration.

Kusaratu zvinore dziri mweya wose wechinhu chineonekwa: Anthropic Max, OpenAI ChatGPT/Codex, uye Google AI Pro. Mnetsa inonyanya inozuva super admins kuita konekiti (connect accounts), kurefresha accounts dzakarekurwa, kuenda nemaoko accounts neemail, uye kuita manual OAuth fallback kana ienvironment inoshandiswa sandboxed inosanganisa browser redirect. Zvinhu zveCursor Pro setup options zvinowanikwa pane admin panel zvakave kuva.

Accounts dzeChatGPT/Codex dzine support dze tool use inoshandiswa connector, izviro rinopasa kuti operation ine tools, zvichiri kuita workflows dzakarekurwa dzinopara kushandisa Codex tool behavior panguva OpenAI account inokonekwa.

## Events {#events}

Nzira ye **Events** inopasa rekare rinozvibvunza zvinhu zvose zvinotiri (logs) mumweya wako. Inoita tracking kwezvinhu zvakaita kubva mberi, signups, uye kunge zvinhu zvine kuita — inokwanisa kutaura multisite yako.

![Events list](/img/admin/events-list.png)

## Webhooks {#webhooks}

**Webhooks** dzinoita kuti uwe data kuita application inotra. Zvinopasa kuti uita sending data kubva Ultimate Multisite kuplatform dzakaita Zapier.

![Webhooks list](/img/admin/webhooks-list.png)
