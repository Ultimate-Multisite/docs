# Settings Reference

Iye peina ya dzongedzo dzinoratidza zvinhu zvinogona kuva kune kuratidza mazuva-mazuva uye kugadzirisa zvinhu zvakachengetedza mu Ultimate Multisite.

## Other Options

Chirongo **Other Options** chiripo panguva **Ultimate Multisite > Settings > Login & Registration**.

| Setting | Kuratidzo |
|---|---|
| **Enable Jumper** | Inoratidza zvinoreva Jumper quick navigation tool mu admin area. Shandisa iye kuti uende nehurongwa (jump) dzakachengetedza kune Ultimate Multisite screens, network objects, uye maoko anogona kuenda mune admin destinations. Zvisanisa iye kana haunoda iye hurongwa iweone. |

## Error reporting and telemetry

Kuratidzo yakare ya kukubatsira matambudziko (error-reporting opt-in setting) yakadikirwa panguva dzongedzo dzinoratidza. Telemetry inosanganisira anonyanya (anonymous telemetry) inozvisanisa, uye hauna toggle mu UI kuti uone iye.

Kana uri kuita runbooks kana screenshots dzinogona kune settings page, shandisa kuratidza zvinoreva ya error-reporting opt-in yakanaka yakare kuti administrators hazvireku zvinhu zvakare pane setting inosanganisira.

## Import/Export settings

Tabo **Import/Export** inoratidza kuti inogadzirisa zvinhu izvi zvinoreva uye inokutanga (links) dzose kune **Ultimate Multisite > Site Export** dzinogadzirisa archives dze site uye network. Shandisa tabo ya settings kuti uregure configuration ye import/export, shandisa **Tools > Export & Import** kuti uende workflow we export/import we single-site, uye shandisa Site Export tool kana unoda archive yakare yese ye Network Export.

## AI provider connector settings

Settings dzemakumbiro (connector) ekubatsira nemakumbiro (AI provider connector settings) zvinoone anoratidza anonyanya (supported OAuth account pools) chete:

| Provider | Setup flow |
|---|---|
| **Anthropic Max** | Shandisa akaunti (accounts) dzimwe kana zvinhu dzinobva Anthropic Max neOAuth button. Shandisa manual OAuth fallback kana browser insandiso (sandboxed browser) haunogone kuita redirect nekuti chine kuteerwa nekuda ye zvinotora. |
| **OpenAI ChatGPT/Codex** | Shandisa akaunti dzakachengetedwa (ChatGPT accounts) kuburikidzo weflow weOAuth. Operations dziri dzinobva neConnector inogona kuita tool calls dzine Codex ya ChatGPT panguva akaunti yakashandiswa. |
| **Google AI Pro** | Shandisa OAuth kuti ushandise akaunti dzakachengetedwa (Google AI Pro), ndipo shandisa refresh kana kudzera connector kana zvinhu hazvirei zvichingopfuura nekuti list yaakaunti haizvinobva mufaro. |

Cursor Pro haunobva provider anogona kuwedzera. Onedza kudera screenshots dzakare, runbooks, kana steps dzekubatsira (onboarding) dzine matanho ekubvira Cursor Pro kana maunhu anogona kutora connector paths.

Kana uchishandisa kana kudzera akaunti dzemprovider, ndichida kuti ushure email address yakakodzera yakaunti inobva nekudzera kana kudera uye ushure settings dzinobva provider panguva unotsvaga kuita operations dzine connector.
