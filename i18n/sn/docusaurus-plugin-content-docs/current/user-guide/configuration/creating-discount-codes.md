# Kuratira Codes (v2)

_**CHINDIRA CHINDIRA: Iriro rudo: Iriro rudo: Ultimate Multisite version 2.x.**_

Ndinenge nemunenge kuti ndinenge kuita kuratira codes kuitira vana vako zvinhu zvakawanda (discounts) pamusoro pe subscriptions yavo. uye kuratira kunenge kune kune!

## Kuratira neKutambura Codes

Kuti utatire kana kutambura discount code, ndichida kuti uenda ku **Ultimate Multisite > Discount Codes**.

![Discount codes list — empty state before any codes exist](/img/config/discount-codes-empty.png)

Panguva iyu, uchaona list yese ya codes dzakaita zvako zvinotamburwa.

Unogona kuita **Add Discount Code** kuti utatire coupon chinobva chine kune, kana unogona kutambura zvino raunenge rauriro nekuita **Edit**.

![Discount codes list with hover actions showing Edit and Delete links](/img/config/discount-codes-list-hover.png)

![Add Discount Code button in the page header](/img/config/discount-codes-add-button.png)

Uchibva ku page inotamburwa kuti utatire kana kutambura coupon code yako. Mune iye, tiri kuratira uno chinobva chine kune.

![Discount code edit page with all sections visible](/img/config/discount-code-edit.png)

Tiri kuona zvinhu zvinotamburwa pano:

**Enter Discount Code:** Iyi kunoita saka ya discount code yako. Iyi haina code inosimbisa vana vako pamwe chete munyoyo we checkout form.

**Description:** Panguva iyu, unogona kuivisa zvakanyanya kuti coupon iyi inosimbisa kuti iwe ndiri kunei.

![Discount code name and description fields at the top of the edit page](/img/config/discount-code-description.png)

Unogonawo kuona discount code yakaita yauriro kana yakasiya:

![Discount code active status](/img/config/discount-code-active.png)

**Coupon code:** Iyi ndiyo kunoita kuti ufanane code inosimbisa vana vako pamwe chete munyoyo we checkout.

![Coupon code field where customers enter the code at checkout](/img/config/discount-code-coupon-field.png)

**Kupedzera (Discount):** Panguva iunyangana, unogona kuita **percentage** kana **amount yakapfuma** yekupedzera kw coupon yako.

![Discount amount setting with percentage or fixed-amount dropdown](/img/config/discount-code-amount.png)

**Kuti zvinhu zvinorehwa (Apply to renewals):** Kana uongona iye, coupon code iyi itaiita **kune kuita pfuma yakare**. Kana iingara, coupon code iyi itaiita **kusvika pfungwa yekutanga chete**. Zvinotaiita kuti zvinhu zvose zvinorehwa zvakare zvazvo.

**Kupedzera kupfuma (Setup fee discount):** Kana uongona iye, coupon code iyi haitaiita **kufamba nepfuma yekutanga** yepfuma. Kana iingara, unogona kuita pfuma (percentage kana amount yakapfuma) inotaiita kuti coupon code iyi itaiita nepfuma yekupedzera kw yako.

![Apply to renewals and setup fee discount toggle options](/img/config/discount-code-renewals.png)

**Kuti iwe uliita (Active):** Utaita kana kutaiita kana kuita coupon code iyi nekuti unenge uri mutaiita kana ukatiye.

![Active toggle to manually enable or disable the discount code](/img/config/discount-code-active.png)

Panguva **Advanced Options**, tinenge nemazano aya:

![Discount code advanced options](/img/config/discount-code-advanced.png)

**Kupedzera kuita (Limit uses):**

  * **Uses:** Pane iye, unogona kuona kuti coupon code iyi yaiita pfuma nani.

  * **Max uses:** Izvi zvinotaiita kuti zvinhu zvakawanda kuti vanhu vanyise kuita pfuma necoupon code iyi. Zvinhu zvakare, kana uingara 10 pano, coupon code iyi itaiita pfuma chete 10 nguva. Panguva iye inenge yakaita izvi, coupon code iyi haitaiita pfuma anokuti.

![Limit uses setting with current uses count and max uses field](/img/config/discount-code-limit-uses.png)

**Tarekuro uye kunge kuita:** Mutauro wese unogona kuita kuti uongeze tareche yechakudaro (start date) ne/ne tareche yekutapera (expiration date) kune coupon yako.

![Start and expiration date fields for scheduling the discount code](/img/config/discount-code-dates.png)

**Kufamba nehu (Limit products):** Kana uva **Select products** (Chinekuita zvinoreva "Select products") onye zvako, zvose zvako dzakaita zvichazowezwa kuone. Unogona kuita kuti utangeze (ne/ne) kuti upi product inozvazvidzivirira coupon code iye. Products anozvisarirwa (toggled off) pano haizvo izvi kana mutenda wako anoda kuita zvinhu izvi necoupon code iye.

![Limit products section with per-product toggle switches](/img/config/discount-code-limit-products.png)

Panguva unenge uyuwo onozvazvidzivirira zvinoseka, ndichida kuti utangeze **Save Discount Code** (Save Coupon Code) kuti utapere coupon yako uye zvakakosha!

![Save Discount Code button at the bottom of the edit page](/img/config/discount-code-save.png)

Coupon iye ndeyekuti yakanzira kuenda munzvimbo yako uye, kubva pano, unogona kuita kuti **edit or delete** (kugadzirisa kana kudera) iye.

![Discount code row in the list with Edit and Delete hover actions](/img/config/discount-codes-list-hover.png)
