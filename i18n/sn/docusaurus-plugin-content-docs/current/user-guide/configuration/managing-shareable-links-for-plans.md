# Kuramba Kuti Linki Rino Kutiwe {#managing-shareable-links-for-plans}

Ultimate Multisite inakupa chokwadi kuti unoshereva link dzako rine share (shareable links) zvakaita kuti uone kuita nekuvaka nekuda kwezvakazi vako. Iyi nzira, enda kuiri panyaya dzekutanga dzeplan yako uye ndigadzirwa "Click to copy the Shareable Link" button.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_Panguya panyaya dzekutanga dzeplan, unoziva kuti uri nebutton inosimbisa link dzine share._

Ziva kuti nzira iyi inoratidza kuti iwe utangei kuita link nekuda kwezvakazi zvakare (default billing frequency). Izviro rinoreva kuti kana zvakare dzekutanga dzeplan yako ndiri mwezi, onoziva kuti pamusoro, uchatanga nelink inogona kuita izvi: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Kana uchida link rine share rine zvakare dzekutanga dzeplan ndiri mwezi, unofanira kuita kuti uone 3 pamusoro:

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Kana uchida utangei neplan _premium_ nekuda kwezvakare dzekutanga dzeplan ndiri mwaka, unofanira kuita kuti uone 1 uye kune chakanaka:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Ziva kuti hauna link imwe chete yakaita kuti zvakare dzekutanga dzeplan dziri nekuita zvakasiyana-siyana. Unofanira kushereva link imwe yakasiyana kune vakare vako vekutanga dzeplan.

Ngepa kana uchishereva link iye kuvakazi vako, izo dzavo dzichizvinyira kuti vave pamba pechinhu chekutyi (signup flow) chakanaka.
