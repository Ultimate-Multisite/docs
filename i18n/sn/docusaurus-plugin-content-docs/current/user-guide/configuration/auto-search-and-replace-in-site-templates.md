# Auto Search and Replace mu Site Templates (v2)

_**Iriyi tutorial iyi inenge WP UItimo version 2.x.**_

Chimweya chekare chinechiri cha Ultimate Multisite ndiko kunenge kuita zvinhu, mupfungwa, uye fields dzinogadzirwa kune registration form. Ndapana kana tiri kuita iwe data iye, tinogona kushandisa iye kuti tikwanise kuita pre-populate content mu matambudziko ekuti site template yakasiitwa. Ndipo, kana site yenyu yakapushwa, Ultimate Multisite ichagadzirisa placeholders nekuita zvinhu zvakakosha zvinogadzirwa nekuti zvakaitwa nguva yekugadzirwa.

Kune iwe, unogona kuita template sites dzako neplaceholders. Placeholders dzinogadzirwa nemazwi anogadzirwa - {{placeholder_name}}.

Ndapana, unogona kutarisa field inosimbisa kuti utore iye data.

Mudzidzi wako anogona kuita iye field nguva yekugadzirwa.

Ultimate Multisite ichagadzirisa placeholders nedata inogadzirwa neudzidzi wako nekuti zvakaitwa nekuita.
