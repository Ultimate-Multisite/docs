# Sei kuupedza Matendo Ekuviri Kune Codeable? {#how-to-get-custom-development-help-from-codeable}

Nge kune nguva dzakawanda, tinoda kuti tive nemazuva anokosha ekugadzirisa zvinhu zvinogona kutamba zvakakosha kuti mweya wenyu (network flows) uonekane seunoda. Kana iye ndiyo inotipa, tinopa Codeable [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), unyore wakasi kungava uri kuona vafungwarai vanogara vakaona (experienced developers) nemakore ekugadzirisa Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) ndiyo platform inosimbisa vanhu nekugadzirisa WordPress, uye vanogona kutenderera kuita mari dzinogara vakaona (qualified freelance developers), vanogona kushanda pamapurojekiti panguva yese. Vane nekuita application yakakwana kuti zvinhu zvakakosha zvinogone kunge uri munhu anogara nekugadzirisa zvinhu zvine kuremekedzo (experts) vachiri kuita zvako.

### Sei kuita project mu Codeable? {#how-to-create-a-project-in-codeable}

Mazuva akapfiri, ndonga kuenda ku [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) uye chibvumira **Start a Project** (Simba Project).

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Nekuti unenge uri kutamba project, ndonga kuita kuteerera munzvimbo rekutanga.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

Mu "I need help with" (Ndinoda uswo), chine **Customizing** (Kugadzirisa) :

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

Mu "My" (Moyo wangu), chine **Plugin** (Addon).

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Zvakanaka, ndonga kuita kuteerera **Continue** (Kutanga).

<!-- Screenshot unavailable: Codeable project form Continue button -->

Nekuti zvakaita, ndiri nguva yekupfiri zvinhu dzako uye unoda kuti uone nzira yakakwana. Ziva zvakawanda zvose!

<!-- Screenshot unavailable: Codeable project description form -->

Zivisa maonero aya kuti zvibatsire nemarhi (pricing).

<!-- Screenshot unavailable: Codeable project pricing questions -->

Pasiya iwe hauna [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) account? Ndinoda uyu, unogona kuita uye kupfuma button inotiri **Publish My Project**.

<!-- Screenshot is not available: Codeable account creation and Publish My Project button -->

Ngepa iwe project yako ya [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), platformi iyi itanirisa nevadzidzisi (developers) vanoita zvinhu zviri pamwe nemahiran anogona kuita project yako.

Iyi nzira inogona kuva yakakubatsira kuuta munhu wakanaka kuti aita ijobo yako, anowana kunzwisisa WordPress uye Ultimate Multisite!

Riti rechinangwa (minimum rate) racho rauri $70–$120 USD pane chidzidzo chekunzuva. Riti rinogona kuitika kubatana nemashoko ekuti project yako inenge yakapfura, inenge yakanyanya kunge yakakodzera, uye inenge yakatarwa zvishoma.

Ngepa iwe riti rauri, uchaona kuti unogona kuuta developer anowakanaka uye kutanga project yako pamwe!
