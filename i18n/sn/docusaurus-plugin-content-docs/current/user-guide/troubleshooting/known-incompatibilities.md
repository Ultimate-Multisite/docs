# Zvinovhu zvakasiitika nePlugin chinotevera

Panguva unenge uchivhara uye utaichirudzirwa (install) nekutevera Ultimate Multisite, unogona kuona matambudziko kubva paunenge uine zvinotevera (incompatibilities) nezvinhasi (plugins) zvakasiitika.

Tiri kutora nguva dzinotevera izvi nekuita mhinduro dzese dzinotevera, asi kana plugin inotevera pano, unogona kuona kuti unenge uchida kukuteka (deactivate) iye kuti Ultimate Multisite Network yako itarisirirwe zvakanaka.

Iyi ruzivo rine zvinotevera uye zvinoita:

**Zvinotevera zvinotevera dzine WPMU DEV** Zvinhasi zvakawanda zviri kuva kunenge zvakasiitika nekuti WPMU DEV haanenge anogadzirisa (maintain) zvinhasi zvakawanda, uye zvinoita kuti kuratidzorera Ultimate Multisite kuteerere (registration flow) kutarisirirwa mumapfumo ekutanga (last step) (unenge akaita account). Iyi ruzivo rinotevera zvinotevera izvi:

  * Signup Language: _Ultimate Multisite inenge neadd-on inoshandisa zvinhasi izvi, inosanganiswa kuti Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Iyi inenge yakaita mune Ultimate Multisite, asi hauna kupfungwa kune plugin inotevera.

  * New Blog Templates: _Iyi inenge yakaita mune Ultimate Multisite, asi hauna kupfungwa kune plugin inotevera.

  * Set Password on Multisite Blog Creation: _Iyi inenge yakaita mune Ultimate Multisite, asi hauna kupfungwa kune plugin inotevera.

  * Signup TOS: _Iyi inenge yakaita mune Ultimate Multisite, asi hauna kupfungwa kune plugin inotevera.

  * Anti-Splog: Kunyangwe inosimba pane zvakawanda, iyi inenge yakaita mune Ultimate Multisitewo.

**Chii chine kukwanisa kuenda nePlugin chinotevera chinotevera?** Zvinotevera zvinotevera dzose dzinotevera dzine WPMU Dev dzinogona kushandiswa zvakanaka nemunhu kuti Ultimate Multisitewo. Mupfungwa, izvi ndizvo: _Smush_, _Forminator_, _Defender_, uye zvinhasi zvakawanda.
