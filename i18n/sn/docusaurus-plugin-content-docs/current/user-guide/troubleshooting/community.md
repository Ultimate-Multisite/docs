# Kuita sei kuita Kujira Ultimate Multisite Community {#how-to-join-ultimate-multisite-community}

Ultimate Multisite Community ndiyo space inotenderi kuti uone nemunhu wose we Ultimate Multisite uye kuti munenge uchishandisa zvinhu, zvinhu zvinogona kubatanidza, uye zvinhu zvakawanda. Kuti utange kuenda kune Ultimate Multisite Community, enda ku https://community.ultimatemultisite.com/home.

Pa welcome page, enda ku link inoti “**creating your account here** ”.

<!-- Screenshot is not available: Community welcome page with creating your account link -->

Utaingwa kuenda kwenye page yechinjira (register). Ndizvo unofanira kuita username, email yako uye password. Ita Captcha uye chibvumirano **Register**.

<!-- Screenshot is not available: Community register page with username, email, password fields and Captcha -->

Panguva uingirowe, ndiri nguva yekuti utore profile yako. Pa page iyi, unogona kuita upload photo yako, kuita dzina rako racho rine chinhu chiri (full name), kuita bio yakakura, uye zvinhu zvakasiyana-nyanya. Socha link dzako dzinomweya (social media links) uye shandisa permissions zvinogona kubva pamusoro pe account yako.

<!-- Screenshot is not available: Community profile creation page with photo upload and bio fields -->

Ndiri pano pa channel ya “Introductions”, unogona kuita self-introduction uye kuona zvakawanda nezve vanhu vose vakare. Unogona kuita post yako nekuita click ku button inoti **New Post**.

<!-- Screenshot is not available: Community Introductions channel with New Post button -->

Channels dzakachena (Lounge section) uye channel ya “Dev Logs” ndizvo dzinogona kuva kune vanhu vose. Asi unofanira kuita permission yakasiyana kuti uone channels dzine mupfungwa (other channels) mu community.

<!-- Screenshot is not available: Community channels showing Lounge section and Dev Logs channel -->

Utawanirwa access iyi kana uingira license key yako ya Ultimate Multisite inotenderwa kwangu. Kuti uramba, shandisa link yakanaka (private channel link) uye ndipo shandisa button inoti **Unlock**.

<!-- Screenshot is not available: Channel ya komiti inotanga button ye Unlock -->

Utawanzwa ku ukwepe wenyu we kuremekedza (unlock access page). Ingira chiti chelisensi chenyu cheUltimate Multisite uye ndikira button ye Unlock Access:

<!-- Screenshot is not available: Ukwepe we komiti nekuingira chiti chelisensi -->

Lisensi chenyu chinokurimwa (validated) uye uchaona kuti utaonana nemakomiti ekupfuura. Izvi zvichakanaka kuita kusvika nguva 24, zvinhu zviri kuitika, ndipo unenge ukira link yakaita kuti urudzwe ku komiti uye utanenge uri kuenda mazuva anonyanya nekuti tinoremekedza chivimbo chenyu.

<!-- Screenshot is not available: Ukwepe we lisensi wenyu chinokurimwa -->
