# Sefa: wu_should_create_domain_record_for_site

Inosefa kana Ultimate Multisite ichifanira kugadzira rekodhi yedomeini yesaiti ichangobva kugadzirwa.

Shandisa sefa iyi kudzivirira kana kunonotsa kugadzirwa kweotomatiki kwerekodhi yedomeini kumasaiti anoshandisa domeini yepasi yefomu rekubhadhara yakagovaniswa, mugamuchiri wemukati, kana domeini ichatarisirwa zvakasiyana neimwe kubatanidzwa.

## Maparamita {#parameters}

| Zita | Rudzi | Tsananguro |
|------|------|-------------|
| $create | `bool` | Kana rekodhi yedomeini ichifanira kugadzirwa. |
| $site | `WP_Site` | Chinhu chesaiti ichangobva kugadzirwa. |

### Kubva {#since}

- 2.13.0

### Kwakabva {#source}

Yakatsanangurwa mu `inc/functions/domain.php`.


## Zvinodzosa {#returns}

Boolean inoratidza kana kugadzira rekodhi yedomeini.
