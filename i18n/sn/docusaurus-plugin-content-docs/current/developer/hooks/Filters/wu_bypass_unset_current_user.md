# Sefa: wu_bypass_unset_current_user {#filter-wubypassunsetcurrentuser}

Bvumira vagadziri kuti vapfuure kodhi yekubvisa mushandisi aripo.

Kudzosa chero chinhu chisiri null kuchapfuura kubviswa kwemushandisi aripo akapinda. Izvi zvinogona kubatsira mune dzimwe mamiriro, semuenzaniso, kana uchibata nemasaiti madiki ari kushandiswa semapaneru evatungamiriri.

## Maparamita {#parameters}

| Zita | Rudzi | Tsananguro |
|------|------|-------------|
| $null_or_bypass | `mixed` | Null kuti uenderere mberi, chero chimwe chinhu kuti upfuure pazviri. |
| $current_user | `false\|\WP_User` | Chinhu chemushandisi aripo. |
