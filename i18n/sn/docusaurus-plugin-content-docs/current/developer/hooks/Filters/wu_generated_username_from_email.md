# Sefa: wu_generated_username_from_email

Sefa zita remushandisi remutengi rakagadzirwa.

## Maparamita {#parameters}

| Zita | Rudzi | Tsananguro |
|------|------|-------------|
| $username | `string` | Zita remushandisi rakagadzirwa. |
| $email | `string` | Kero yeemail yemutengi mutsva. |
| $new_user_args | `array` | Array yemaparamita emushandisi mutsva, pamwe ichisanganisira mazita ekutanga neekupedzisira. |
| $suffix | `string` | Wedzera string kuzita remushandisi kuti riite rakasarudzika. |
