## Maparamita {#parameters}

| Zita | Rudzi | Tsananguro |
|------|------|-------------|
| $username | `string` | Zita remushandisi remutengi. |
| $email | `string` | Kero yeemail yemutengi mutsva. |
| $new_user_args | `array` | Array yemaargs emushandisi mutsva, pamwe inosanganisira mazita ekutanga neekupedzisira. |
| $suffix | `string` | Wedzera string kuzita remushandisi kuti rive rakasiyana. |
