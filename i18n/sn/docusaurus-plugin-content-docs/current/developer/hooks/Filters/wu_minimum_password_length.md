# Sefa: wu_minimum_password_length

Sefa hurefu hushoma hwepassword.

Inongomanikidzwa kana wu_enforce_password_rules iri true.

## Maparamita

| Zita | Rudzi | Tsananguro |
|------|------|-------------|
| $min_length | `int` | Hurefu hushoma hwepassword. Default 12 (inoenderana neDefender Pro). |
| $defender_active | `bool` | Kana Defender Pro Strong Password iri kushanda. |
