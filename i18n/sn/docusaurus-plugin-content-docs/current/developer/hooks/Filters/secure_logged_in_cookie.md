# Sefa: secure_logged_in_cookie {#filter-secureloggedincookie}

Inosefa kuti cookie yekupinda inofanira kutumirwa chete kuburikidza neHTTPS here.

## Maparamita {#parameters}

| Zita | Rudzi | Tsananguro |
|------|------|-------------|
| $secure_logged_in_cookie | `bool` | Kuti cookie yekupinda inofanira kutumirwa chete kuburikidza neHTTPS here. |
| $user_id | `int` | ID yemushandisi. |
| $secure | `bool` | Kuti cookie yeauth inofanira kutumirwa chete kuburikidza neHTTPS here. |

### Kubva {#since}

- 3.1.0
