# Sefa: wu_connector_enforcement_option_keys

Sefa rondedzero yemakiyi esarudzo emupi weAI anosimbiswa kubva kusaiti huru.

Wedzera makiyi esarudzo emaAI provider plugins akagadzirwa pachako kana echitatu-bato kuitira kuti marongero awo agarewo nhaka kubva kusaiti huru pamasaiti madiki.

## Maparamita {#parameters}

| Zita | Rudzi | Tsananguro |
|------|------|-------------|
| $keys | `string[]` | Mazita emakiyi esarudzo ekusimbisa (atova nemakiyi econnector akawanikwa otomatiki uye EXTRA_PROVIDER_OPTIONS). |
