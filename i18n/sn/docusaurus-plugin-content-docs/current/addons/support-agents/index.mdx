# Vamiriri veRutsigiro {#support-agents}

Kutarisira network yose uri wega harisi basa riri nyore. Sezvo network yako painotanga kukura, uchatanga kuona kuwedzera kwekudiwa kwerutsigiro. Zvichava zano rakanaka kuunza vanhu vachakubatsira kuderedza mutoro webasa rako. Zvisinei, kupa mukana/mvumo yakazara kune ngozi uye kunogona kukanganisa network yako. Ndipo panopinda wedzero iyi yeMumiriri weRutsigiro.

Wedzero iyi inokubvumira kugadzira rudzi rutsva rwemushandisi weWordPress anonzi Mumiriri weRutsigiro, uye inokupa kutonga kwakazara pamusoro pemvumo dzine mushandisi iyeye. Kupa mvumo kuri nyore sekubatidza bhatani rekubatidza/kudzima uye kuridzima kana wafunga kuibvisa.

## Zvimiro {#features}

- Rondedzero yakapatsanurwa yemvumo dzeWordPress & Ultimate Multisite
- Gonesa & dzima mukana zviri nyore
- Gadzira huwandu husina muganhu hweAgent Account

## Kuiswa {#installation}

1. Isa 'ultimate-multisite-support-agents' mudhairekitori re'/wp-content/plugins/'
2. Batidza wedzero kuburikidza nemenyu ye'Wedzero' muWordPress
3. Tevera Wizard yenhanho nenhanho kugadzira marongero ewedzero
