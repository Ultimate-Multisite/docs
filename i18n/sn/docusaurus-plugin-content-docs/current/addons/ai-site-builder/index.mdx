# AI Site Builder

ultimate-multisite-ai-site-builder

## Zvinodiwa

- WordPress 5.3 kana yepamusoro
- PHP 7.4 kana yepamusoro
- Multisite Ultimate plugin (iri kushanda)

## Kuiswa

1. Isa mafaera eaddon mudhairekitori rako re`/wp-content/plugins/`
2. Shandisa plugin kuburikidza nemenyu ye'Plugins' muWordPress
3. Gadzirisa marongero eaddon mu network admin

## Kuvandudza

### Kugadzirisa

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Kuyedza

Addon iyi inosanganisira hurongwa hwakazara hwekuyedza:

- **Miedzo yeUnit**: Miedzo yakavakirwa paPHPUnit ine WordPress test suite
- **Kirasi yeBvunzo yePasi**: Inopa nzira dzinobatsira dzemamiriro ebvunzo akajairika
- **Kufukidzwa kweBvunzo**: Yakagadzirirwa kutevera kufukidzwa kwekodhi
- **Kuyedza Multisite**: Miedzo inomhanya munzvimbo ye multisite

### Mitemo yeKodhi

- **WordPress Coding Standards**: Inomanikidzwa kuburikidza nePHPCS
- **Kuenderana nePHP 7.4+**: Zvimiro zvemazuva ano zvePHP zvinotsigirwa
- **Ongororo Isingachinji**: Kubatanidzwa kwePHPStan kwekuongorora mhando
- **Kuvandudza Kodhi**: Mitemo yeRector yekusimudzira PHP

### Hurongwa hweKuvaka

- **Kugadziriswa kweZviwanikwa**: Kuderedza mafaera eCSS/JS
- **Dudziro**: Kugadzirwa kwefaira rePOT
- **Kupakira**: Kugadzirwa kwearchive yekugovera
- **Modhi yeKuvandudza**: Zviwanikwa zvisina kuderedzwa zvekutsvaga zvikanganiso

## Kugadziriswa

Addon inotsigira sarudzo dzakasiyana dzekugadzirisa dzinowanikwa kuburikidza ne network admin interface.

## Hooks neFilters

### Actions

- `ultimate-multisite-ai-site-builder_init` - Inotangwa mushure mekutanga kweaddon
- `ultimate-multisite-ai-site-builder_loaded` - Inotangwa mushure mekunge mafaera ese eaddon atakurwa

### Filters

- `ultimate-multisite-ai-site-builder_settings` - Chinja marongero eaddon
- `ultimate-multisite-ai-site-builder_enabled` - Pfuurira pamusoro pemamiriro ekugonesa/kudzima addon
