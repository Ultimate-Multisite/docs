# Material WP {#material-wp}

Chitarisiko chekutonga cheWordPress chave chakangofanana kwemakore mazhinji. Asi wakambofunga here kuti vhezheni yeMaterial Design yechitarisiko chekutonga yaizotaridzika sei? 

Material WP i theme yekutonga yakagadzirirwa kukupa kutaridzika uye manzwiro iwayo, ichikubatsira kupa vatengi vako Dashboard yakagadziridzwa zvakanyanya.

## Zvimiro {#features}

- Rutsigiro rweCustom CSS
- Zvirongwa zviri nyore zvekushongedza
- Kurongazve menyu
- Kupinza/kuburitsa zviri nyore

## Zvinyorwa {#documentation}

**Kugadzirisa Admin Panel yako neMaterial WP**

## Kuiswa {#installation}

1. Isa 'material-wp' mudhairekitori re'/wp-content/plugins/'
2. Shandisa plugin kuburikidza nemenyu ye'Plugins' muWordPress
