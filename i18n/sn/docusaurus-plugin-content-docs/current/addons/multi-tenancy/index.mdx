# Multi-Tenancy {#multi-tenancy}

Govera masaiti madiki kumaseva nemadhatabhesi kuitira mashandiro akanaka nekuchengeteka

## Zvakakosha mukuburitswa kweVersion 1.2.0 {#version-120-release-highlights}

Multi-Tenancy 1.2.0 inowedzera hwaro hwemuroja akazvimirira hwemafambisirwo ekupatsanura munzvimbo yekugadzira:

- Kupinda otomatiki kwemuroja kusina state nekushanya kweSSO kupfupi-nguva uye kwakadzivirirwa kubva pakudzokororwa.
- Kupatsanurwa kwedhatabhesi nehurongwa hwemafaira pasaiti diki yega yega yevanoja vakazvimirira.
- Kugona kweHostinger multi-tenancy kwekugovera vanochengeta nzvimbo kunoziva host.
- Mageti ekusimbisa kutamiswa kwakazvimirira ekuongorora data rekare uye kugadzirisa async push.
- Kusungwa kwehost yedhatabhesi pamuchina mumwe kunogadzirisika kweBedrock, FrankenPHP, uye kuiswa kwelocalhost.
- Kuvandudzwa kwekutanga kwemuroja, munyori wedhatabhesi, kutanga schema, uye kugovera mushandisi.
- Kubvisa kwakachengeteka kwemuroja akazvimirira kunochenesa zvitupa zvemuroja.

Ona magwaro emushandisi e[maitiro ekutamisa kwakazvimirira](/user-guide/getting-started/sovereign-migration-workflow), [kupatsanurwa kwe multi-tenancy](/user-guide/configuration/multi-tenancy-isolation), uye [mashandiro emuroja akazvimirira](/user-guide/administration/sovereign-tenant-operations).

- WordPress 5.3 kana yepamusoro

## Kugadziriswa {#configuration}

Addon inotsigira sarudzo dzakasiyana dzekugadzirisa dzinowanikwa kuburikidza neinterface yenetwork admin.

### Kugadziriswa kwemuroja akazvimirira {#sovereign-tenant-configuration}

Kune vanochengeta nzvimbo vakazvimirira, gadzirisa dhatabhesi remuroja, mudzi wehurongwa hwemafaira, kusungwa kwehost yedhatabhesi, uye kugona kwehost usati waendesa traffic kumuroja. Mhanyisa kusimbisa kutamiswa mushure mechinja chese chinobata zvitupa zvedhatabhesi, kusungwa kwehost, routing, kana kugoverwa kwemushandisi.

Shandisa `localhost` chete kana runtime yemuroja nezvibvumirano zvedhatabhesi zvichitarisira local socket kana kuwana pamuchina mumwe. Shandisa `127.0.0.1` kana zita rehost rebasa kana iri iyo grant host yakagadziriswa nemupi wako.

### Kugona kweHostinger {#hostinger-capability}

Kugona kweHostinger multi-tenancy kunoshanda ne shared Hostinger integration token. Simbisa kuti domain iripo mu hPanel, DNS inonongedzera kuHostinger account, uye marongero edhatabhesi remuroja anoenderana ne hPanel usati wasimbisa.
