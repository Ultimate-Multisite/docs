# Kuwiriranisa Zvemukati

Multisite Ultimate: Kuwiriranisa Zvemukati

Wedzero iyi yeMultisite Ultimate inobvumira vatungamiri venetiweki kuwiriranisa zvemukati kubva kune chero saiti kuenda kune mamwe masaiti munetiweki. Inotsigira marudzi ese ezvinyorwa kusanganisira zvinyorwa, mapeji, zvigadzirwa, marudzi ezvinyorwa akagadzirirwa, uye matemplate eElementor. Wedzero iyi inosanganisira interface yewizard ine nhanho dzakawanda yekusarudza zvemukati nemasaiti anonangwa, iine kugadzirisa mumapoka uye mhinduro yekufambira mberi panguva chaiyo.

## Zvimiro

* Kusarudza Zvemukati: Gridi yemakadhi inooneka ine mifananidzo midiki uye kusefa nerudzi rwechinyorwa
* Marudzi Ese eZvinyorwa: Wiriranisa zvinyorwa, mapeji, zvigadzirwa, matemplate eElementor, uye chero rudzi rwechinyorwa rwakagadzirirwa
* Kusarudza Saiti: Sarudza masaiti ekuti zvemukati zviwiriraniswe kwairi, nekubvisa saiti yekwakabva uye kusarudzwa kare kwemasaiti evana
* Kugadzirisa Mumapoka: Inowiriranisa kumasaiti 10 panguva imwe chete iine zvigadziriso zvekufambira mberi
* Kuwiriranisa Midhiya: Inokopa uye inoronga patsva mafaera emidhiya (mifananidzo, mavhidhiyo) kubva muzvemukati zvechinyorwa kuenda kumasaiti anonangwa otomatiki
* Kuwiriranisa Taxonomy: Inochengetedza zvikamu, ma tag, uye taxonomy dzakagadzirirwa
* Rutsigiro rweElementor: Rutsigiro rwakazara rwematemplate eElementor nekugadzirwa patsva kweCSS uye kurongwazve kweID yetemplate
* Kugadzirisa Kusawirirana: Sarudzo yekutsiva zvemukati zviripo kana kuzvisvetuka
* Kutarisira Cache: Inobvisa cache yeElementor pamasaiti anonangwa mushure mekuwiriranisa otomatiki

## Zvinodiwa

* WordPress 5.3 kana yepamusoro
* PHP 7.4 kana yepamusoro
* plugin yeMultisite Ultimate (yakabatidzwa)
* plugin yeElementor (sarudzo, inodiwa pakuwiriranisa template yeElementor)

## Kuiswa

1. Isa 'ultimate-multisite-content-sync' mudhairekitori re'/wp-content/plugins/'
2. Batidza plugin kuburikidza nemenu ye'Plugins' muWordPress Network Admin yako
3. Gadzirisa marongero muUltimate Multisite → Marongero → Kuwiriranisa Zvemukati
4. Enda kuUltimate Multisite → Kuwiriranisa Zvemukati kuti utange kuwiriranisa
