# AffiliateWP

Multisite Ultimate: Kubatanidzwa kweAffiliateWP

Shandisa AffiliateWP ine simba kukudza hwaro hwevatengi veUltimo Network yako!

## Kuiswa

1. Isa 'ultimate-multisite-git ' mudhairekitori re'/wp-content/plugins/'
2. Batidza plugin kuburikidza nemenyu ye'Plugins' muWordPress
3. Tevera Wizard nhanho nenhanho kuti ugadzirise plugin

## Gwaro Rekushandisa

[AffiliateWP](https://affiliatewp.com/) yava mhinduro inonyanya kushandiswa pakuvaka chirongwa chevadyidzani chebhizinesi rako repamhepo. Ultimate Multisite: AffiliateWP ibhiriji richakugonesa kuvaka chinhu chimwe chete kunetiweki yako yepamusoro.

## Kutanga

Chekutanga, **unofanira kuva ne** [**AffiliateWP**](https://affiliatewp.com/) **yakabatidzwa**! Hauzokwanisi kushandisa wedzero yedu kana usina **plugin inobatana nayo**. Kana usingarongi kurega vatengi vako vachiishandisa pamasaiti avo zvakare, AffiliateWP inofanira kubatidzwa chete pasaiti yako huru.

Zvakare, uchada kuti zvose **Ultimate Multisite** ne **Ultimate Multisite: AffiliateWP** **zvibatidzwe** **panetiweki yose**.

## Kuwedzera Ultimate Multisite seKubatanidzwa kweAffiliateWP

Pasaiti yako huru, enda kuZvirongwa zveVadyidzani, uye patebhu yeKubatanidzwa, maka Ultimate Multisite. Izvi zvichaudza [AffiliateWP](https://affiliatewp.com/) kuti itange kutevera kushanyira nekushandurwa kunobva mukuyerera kwekunyoresa kweUltimate Multisite.

<!-- Mufananidzo hausipo: tebhu yezvirongwa zveKubatanidzwa kweAffiliateWP ine bhokisi rekumaka reUltimate Multisite -->

_Maka bhokisi reUltimate Multisite pazasi wochengeta_

Ndizvo zvinenge zvese zvinodiwa. Kushanyira kunobva kuzvinongedzo zvekutumirwa kuchateverwa uye kana mushandisi iyeye azoita muripo wake wekutanga, kushandurwa kuchagadzirwa uye komisheni ichaiswa parutivi kumudyidzani ane basa rekushandurwa kwacho.

Muripo wekutanga chete unogamuchirwa kubva kumushandisi achangogadzirwa ndiwo uchateverwa, zvakadaro. Kuti ugonese miripo yekutumirwa inodzokororwa, uchada [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/)

## Kugonesa Kutumirwa Kunodzokororwa

Mushure mekuisa [AffiliateWP Recurring Referrals Add-on](https://affiliatewp.com/add-ons/pro/recurring-referrals/), enda kuZvirongwa zveVadyidzani patebhu yeKutumirwa Kunodzokororwa, uye gonesa kushandiswa kwekutumirwa kunodzokororwa.
