# Site Exporter

## Zviri mukati {#content}

Tinonzwisisa kudiwa kwekukwanisa kutamisa masaiti madiki mukati menetiweki yako uchienda kuisirwa kwesaiti imwe chete uye zvinopesana nazvo kana mutengi wako akasarudza kutamisa webhusaiti yake. Chinowedzerwa cheSite exporter neimporter chinoita kuti basa iri rive nyore. Chishandiso ichi chinoburitsa uye chinopinza masaiti chichishandisa mafaera e zip ayo anogonawo kushanda se backup.

## Zvimiro {#features}

* Kuburitsa uye Kupinza Masaiti

* Interface iri nyore kushandisa

* Gadzira backup yesaiti

## Zvinyorwa {#documentation}
**Mabudisiro uye Mapinzirwo eSaiti**

## Kuiswa {#installation}

1. Isa **ultimate-multisite-site-exporter** kudhairekitori re **/wp-content/plugins/**
2. Batidza chinowedzerwa kuburikidza nemenu ye **Zvinowedzerwa** mu WordPress
3. Tevera Wizard nhanho nenhanho kuti umise chinowedzerwa

## Gwaro Rekushandisa {#usage-guide}

Ne Ultimate Multisite unogona kuburitsa saiti kubva munetiweki yako kana kupinza saiti kuisirwa kwako kwe multisite kana single-site.

Kutanga maitiro, enda kupeji redu rezvinowedzerwa uye uise chinowedzerwa cheUltimate Multisite: Site Exporter & Importer.![Peji rezvinowedzerwa](/img/admin/addons-page.png)

![Peji rezvinowedzerwa rine sarudzo yekuisa](/img/admin/addons-page.png)

Enda kupeji rako reZvinowedzerwa wodzvanya kuti **Network Activate** chinowedzerwa chawabva kuisa.<!-- Mufananidzo hauwanikwi: Peji reWP Zvinowedzerwa richiratidza bhatani reNetwork Activate reSite Exporter addon -->

Sarudzo ye **Export & Import** inogona kuwanikwa pamenu ye Ultimate Multisite, kuruboshwe.

<!-- Mufananidzo hauwanikwi: Menu yeUltimate Multisite ichiratidza sarudzo yeExport and Import -->

## Mabudisiro esaiti {#how-to-export-a-site}

Kuti uburitse saiti kubva munetiweki yako, enda ku **Export & Import** pamenu ye Ultimate Multisite uye, patebhu ye **Export Sites**, dzvanya kuti **Generate new Site Export**.

<!-- Mufananidzo hauwanikwi: Tebhu yeExport Sites ine bhatani reGenerate new Site Export -->

Tsvaga saiti yaunoda kuburitsa uye sarudza zvinhu zvaunoda kusanganisa mufaera rakaburitswa. Unogona kusanganisa matheme, zvinowedzerwa, uye uploads. Kana saiti yako yakakura zvakanyanya, unogona kusarudza kumhanyisa maitiro ekuburitsa kumashure.<!-- Mufananidzo hauwanikwi: Sarudzo dzekuburitsa saiti dzine kutsvaga saiti, mabhokisi ekutarisa ekusanganisa matheme/zvinowedzerwa/uploads -->

Dzvanya bhatani re **Export Site**. Saiti yako yakaburitswa ichawanikwa parondedzero ye **Existing Exports**. Dzvanya bhatani re **Download ZIP** kuti udhawunirodhe faira. MaExport ZIP anosanganisira `index.php` inozvitangisa yega uye `readme.txt`, saka anogona kuiswa pahost itsva pasina kutanga waisa chinowedzerwa cheimporter chakasiyana.<!-- Mufananidzo hauwanikwi: Rondedzero yeExisting Exports ine bhatani reDownload ZIP -->

## Mabudisiro enetiweki yese {#how-to-export-the-whole-network}

Kuti uburitse masaiti madiki ese pamwe chete, enda ku **Export & Import** pamenu ye Ultimate Multisite uye shandisa sarudzo ye **Network Export** pachidzitiro chekuburitsa. Archive inogadzirwa ine masaiti madiki ese muZIP imwe chete yekutamisa mahost, kuvakazve staging, kana backup dzepadanho renetiweki.

Maexport enetiweki anogona kuva makuru. Mhanyisa kana traffic iri shoma uye simbisa kuti server ine nzvimbo yekuchengetera yakakwana ye archive uye chero uploads, zvinowedzerwa, uye matheme akasanganisirwa.

## Mapinzirwo esaiti {#how-to-import-a-site}

Kuti upinze saiti, enda kutebhu ye **Import a Site** uye upload faira re .zip resaiti yaunoda kupinza. Unogona kuchinja URL kuita itsva kana kusiya URL yekare nekusiya nzvimbo isina chinhu. Uyewo, unogona kutarisa bhokisi kuti udzime faira re .zip kubva pakuisirwa kwako mushure mekuunza. Wobva wadzvanya **Begin Import**.

<!-- Mufananidzo hauwanikwi: Tebhu yeImport a Site ine nzvimbo yeupload yezip uye bhatani reBegin Import -->

Maitiro anogona kutora nguva shoma. Uchaona import yakamirira murondedzero iri pasi peimporter.

<!-- Mufananidzo hauwanikwi: Import yakamirira murondedzero iri pasi peimporter -->

Unogona kumisa maitiro nekudzvanya **Cancel Import**.

<!-- Mufananidzo hauwanikwi: Bhatani reCancel Import rekumisa maitiro ekuimport -->

Kana maitiro apera, saiti yako yatovepo munetiweki yako.

## Mapinzirwo esaiti pakuisirwa kumwe chete {#how-to-import-a-site-on-a-single-install}

Kuti upinze saiti kukuisirwa kwako kumwe chete, tanga waisa uye wabatidza importer pasaiti yako. Unogona kuwana link yekuidhawunirodha patebhu yeexporter.

<!-- Mufananidzo hauwanikwi: Tebhu yeExporter ine link yekudhawunirodha single-site importer -->

Unogona kuwana importer pamenu ye **Zvishandiso**, kuruboshwe, pachinhu chidiki che **Export & Import**.

<!-- Mufananidzo hauwanikwi: Menu yeZvishandiso ye single-site ichiratidza submenu yeExport and Import -->

Upload faira re .zip resaiti yaunoda kupinza. Unogona kuchinja URL kuita itsva kana kusiya URL yekare nekusiya nzvimbo isina chinhu. Uyewo, unogona kutarisa bhokisi kuti udzime faira re .zip kubva pakuisirwa kwako mushure mekuunza. Wobva wadzvanya **Begin Import**.

<!-- Mufananidzo hauwanikwi: Fomu reimport re single-site rine upload yezip uye bhatani reBegin Import -->

Maitiro anogona kutora nguva shoma. Uchaona import yakamirira murondedzero iri pasi peimporter.

<!-- Mufananidzo hauwanikwi: Import yakamirira ye single-site murondedzero -->

Unogona kumisa maitiro nekudzvanya **Cancel Import**.

<!-- Mufananidzo hauwanikwi: Bhatani reCancel Import rekumisa maitiro ekuimport -->

Kana maitiro apera, saiti yako ichatsiviwa nesaiti yawabva kupinza.
