# Plugin & Theme Manager

Multisite Ultimate: Plugin and Theme Manager

Gadzirisa kana kuviga ruzivo rwe meta (zita, munyori, thumbnail netsananguro) rwezviwedzerwa nemadingindira akaiswa. Gadzira zvikamu zvezviwedzerwa nemadingindira kuti ubvumire vashandisi vako kuzvisefa uye wedzera peji reZviwedzerwa rakagadziridzwa zvakanaka kupanera yevashandisi vako.

Kuti ushandise plugin, unogona kungoenda kumapeji ako eZviwedzerwa kana Madingindira pa network admin yako wobaya pa "Gadzirisa ruzivo rwePlugin/Theme".

## Kuiswa

1. Isa 'ultimate-multisite-plugin-and-theme-manager' mudhairekitori re'/wp-content/plugins/'
2. Shandisa plugin kuburikidza nemenyu ye'Zviwedzerwa' muWordPress Network Admin yako

## Gwaro Rekushandisa

Ultimate Multisite: Plugin and Theme Manager iwedzero ine simba **inobhadharwa** yeUltimate Multisite inokupa kutonga kuzere pamusoro pemaitiro anoita vashandisi venetwork yako pakuona nekudyidzana nesarudzo dzeplugin nedzedingindira panetwork yako.

Wedzero iyi ine zvinangwa zvitatu:

  1. Inokutendera kuchinja zvachose metadata yezviwedzerwa nemadingindira zvese. Izvi zvinosanganisira zita, tsananguro, zita remunyori, uye kunyange ma thumbnail!

  2. Inokutendera kuronga zviwedzerwa nemadingindira muzvikamu, kuti ubvumire vashandisi vako kuzvisarudza zviri nyore.

  3. Inokupa sarudzo yekutsiva peji rekare, rinofinha rezviwedzerwa nepeji idzva rakafanana nemadingindira rine kuronga nezvikamu.

Mushure mekuidhawunirodha nekuiisa, tebhu itsva - _Marongero eWedzero_ \- ichaonekwa papeji reUltimate Multisite Settings.
