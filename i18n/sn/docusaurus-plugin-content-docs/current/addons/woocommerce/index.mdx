# Kubatanidzwa kweWooCommerce

Kubatanidzwa kweWooCommerce kunoshanda nenzira yakafanana zvikuru nemashandiro anoita Gedhi reManyorerwo redu. Kunopa kuchinjika kukuru kuvashandisi vako, kuchivabvumira kusarudza chero mafomu ekubhadhara aripo paWooCommerce yakaiswa yako. Uye zviri nyore: kana nguva yekubhadhara yemushandisi yapera, kunyoreswa kwemushandisi kunoiswa pakumbomiswa. Panguva imwe chete, odha ichagadzirwa paWooCommerce yakaiswa pasaiti huru yenetwork. Mushandisi anobva agamuchira invoice muemail yake, ine link yekuita kubhadhara. Kana kubhadhara kwagamuchirwa, kunyoreswa kunovandudzwa.

## Zvimiro

- Kubatanidzwa kuri nyore pakati peUltimate Multisite neWooCommerce
- Nzira dzakawanda dzekubhadhara dziripo
- Kubhadhara kunodzokororwa otomatiki uchishandisa core yezvinyoreso yakasanganisirwa, pamwe nekuvandudzwa kweWooCommerce Subscriptions plugin kana uchida
- Mari dzese paCheckout dzinosanganisira mitero yeUltimate Multisite semitsara yemari yeWooCommerce yakaparadzana, isingatereswi kuitira kuti vatengi vaone huwandu hwemutero vasati vabhadhara

## Zvinyorwa

**Kugadzira Kubatanidzwa kweWooCommerce**

**Kuwedzera Sarudzo dzako dzeKubhadhara neUltimate Multisite: Kubatanidzwa kweWooCommerce**

## Kuiswa

1. Isa 'ultimate-multisite-woocommerce' mudhairekitori re'/wp-content/plugins/'
2. Batidza pulagini kuburikidza nemenyu ye'Plugins' muWordPress
3. Tevera Wizard nhanho nenhanho kuti ugadzirise pulagini

## Gwaro Rekushandisa

_**CHERECHEDZO YAKAKOSHA: Chinyorwa ichi chinoreva Ultimate Multisite vhezheni 2.x.**_

_**CHENJERERA:** Ultimate Multisite: Kubatanidzwa kweWooCommerce kunoda kuti WooCommerce ibatidzwe kanenge pasaiti yako huru._

Tinonzwisisa kuti _Stripe_ ne_PayPal_ hazviwanikwi mune dzimwe nyika, izvo zvinoganhurira kana kukanganisa vashandisi veUltimate Multisite kushandisa pulagini yedu zvinobudirira. Saka takagadzira **wedzero yekubatanidza _WooCommerce_**.

_WooCommerce_ ipulagini yeCommerce yakakurumbira zvikuru. Vagadziri pasi rose vakagadzira wedzero dzekubatanidza magedhi akasiyana ekubhadhara kwairi. Takashandisa mukana uyu kuwedzera magedhi ekubhadhara aunogona kushandisa muhurongwa hwekubhadharisa hweUltimate Multisite.

### Mitsara yemari yemutero paCheckout

Kana checkout yeUltimate Multisite ichisanganisira mitero, kubatanidzwa kunowedzera huwandu hwemutero wega wega weUltimate Multisite kucheckout yeWooCommerce semutsara wemari wakaparadzana, usingatereswi. Izvi zvinoita kuti huwandu hwemutero hurambe huchioneka usati wabhadhara uku zvichidzivirira WooCommerce kuti isaterese mari yemutero zvakare.

## Maitiro ekugadzirisa kubhadhara

Kubatanidzwa kweWooCommerce kunoshanda nenzira yakafanana zvikuru nemashandiro anoita **Gedhi reManyorerwo** redu. Chakaipira ndechekuti kubhadhara kwega kwega kunofanira kubhadharwa nemaoko pakupera kwenguva yekubhadharisa. Heano mashandiro anoita kuyerera kwacho:

  * Kana kutenderera kwekubhadharisa kwemushandisi kwapera, unhu hwenhengo hunoiswa **pakumbomiswa**. Panguva imwe chete, odha ichagadzirwa pa_WooCommerce_ yakaiswa pasaiti huru yenetwork yako.

  * Vashandisi vanobva vagamuchira invoice muemail yavo, ine link yekuita kubhadhara (link yacho inoonekwawo munhoroondo yavo yekubhadharisa kubva paDashboard yesubsite yavo). Link iyoyo inoendesa mushandisi ku _**WooCommerce**_ **checkout** , uko vashandisi vachakwanisa kusarudza rimwe re **magedhi e_WooCommerce_ aripo** kuitira kugadzirisa kubhadhara kwavo.

  * Kana kubhadhara kwagamuchirwa, unhu hwenhengo **hunovandudzwa** , kubhadhara kunonyorwa paUltimate Multisite uye odha inoiswa kuti yapera.

### Zvimwe Zvinyorwa Zvakakosha:

  * Kushanda kwekutanga kweUltimate Multisite kuchiri kushanda nefomu rekubatanidzwa iri. Semuenzaniso, **kuburitsa madzosero emari** kubva pachidzitiro che **Subscription Management** kuchataurirana neWooCommerce kuti kuburitse kudzoserwa kwemari uchishandisa fomu rekubhadhara rakasarudzwa.

  * Ultimate Multisite: WooCommerce Integration inovimba neWP-Cron kuti ishande. Izvi zvinoshanda zvakanaka zvikuru nguva zhinji, asi kana saiti iine traffic shoma ipapo cron haikurudzirwi kuti itarise zviitiko zvakarongwa uye chiitiko chinopotswa, izvo zvinogona kukonzera kunonoka kwekuitwa.

## WooCommerce Subscriptions Plugin

Kutangisa muripo nemaoko mwedzi wega wega hakusi kwakanakira vashandisi vekupedzisira nekuti subscriptions zvinotarisirwa kudzokorora zvoga billing cycle yega yega. WooCommerce Integration add-on yava kusanganisira core subscriptions code, saka recurring products dzinogara dzichishandisa subscriptions pakubhadharisa kunodzokororwa.

Haudi **kutenga** kana kuisa [WooCommerce Subscriptions plugin](https://woocommerce.com/products/woocommerce-subscriptions/) yakaparadzana kuti utengese recurring products kuburikidza neintegration iyi. Kana plugin iyoyo ikatengwa, ikaiswa, uye ichiwanikwa pa main site yako, Ultimate Multisite ichaishandisa uye WooCommerce icharatidza mamwe maficha ekutarisira subscription. Kana isiri kuwanikwa, bundled subscriptions core ichiri kubata recurring products.

![WooCommerce subscription integration marongero](/img/addons/woocommerce-subscription-integration.png)

Uye ndizvozvo. Vashandisi vako vekupedzisira havachazodi kubhadhara nemaoko pakupera kwe billing cycle yavo ye recurring products.

Kunze kwekuti order inogadzirwa pa main site yako pasi pe _WooCommerce_, ichagadzirawo subscription yeAccount yega yega.

### Maitiro ekumisa pakakundikana kuvandudza

Nekumisikidza, memberships dzinoramba dziri active apo WooCommerce Subscriptions ichigadzirisa failed-payment retry schedule yayo. Izvi zvinochengetedza normal WooCommerce Subscriptions grace window kuitira kuti renewal yakadzoreredzwa igone kudzosa subscription uye membership kuti zvienderane pasina kukurumidza kukanganisa access.

Administrators vanoda access control yakasimba vanogona kugonesa **Misa Memberships Pakarepo paFailed Renewals** muWooCommerce Integration settings. Kana opt-in setting iyi yagoneswa, Ultimate Multisite inomisa membership inoenderana pakarepo kana renewal payment ikakundikana panzvimbo pekumirira kuti WooCommerce Subscriptions retry handling ipere.

Developers vanogona kutsiva decision imwechete ne `wu_woo_suspend_on_payment_failure` filter. Hook reference pages anogadzirwa kubva ku add-on source, saka overview iyi inongotaura zita re filter chete kuitira custom suspension policies.
