# Sefa: wp_ultimo_skip_network_active_check

Inobvumira vagadziri kuti vapfuure kuongororwa kwekushandiswa panetwork.

Izvi zvinobatsira pakushandisa composer-based uye mamwe marongero akagadzirirwa, akadai seBedrock, semuenzaniso, apo kushandisa maplugin se mu-plugins kuri kwakajairika.

## Maparamita

| Zita | Rudzi | Tsananguro |
|------|------|-------------|
| $skip_network_activation_check | `bool` | Kana tichifanira kusvetuka kuongorora kana kwete, zvinotangira pa false. |

### Kubva

- 2.0.0
### Kwakabva

Yakatsanangurwa mu[`inc/class-requirements.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-woocommerce/blob/main/inc/class-requirements.php#L272) pamutsetse 272


## Zvinodzoserwa
true kana uchida kusvetuka kuongorora, false kana zvisiri izvo.
