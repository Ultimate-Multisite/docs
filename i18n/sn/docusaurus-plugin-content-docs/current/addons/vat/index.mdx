# VAT

Multisite Ultimate: VAT yeEurope

Inowedzera maficha nematurusi ekuwedzera pakubata mitero kwekutanga kweMultisite Ultimate kuti itsigire zvirinani VAT yeEurope.
Inowedzera otomatiki mareti eEU VAT kupeji reTax Rates kubva

## Kuiswa

1. Isa 'ultimate-multisite-vat' mudhairekitori re'/wp-content/plugins/'
2. Gonesa plugin kuburikidza nemenyu ye'Plugins' muWordPress
3. Enda kuUltimate Multisite > Settings > Taxes wobva wagonesa EU VAT.
4. Enda wodzvanya paTax Rates uye Update EU VAT Rates pazasi wobva wachengeta Tax Rates.
5. Zvapera!
