# PayFast

Multisite Ultimate: Payfast Gateway

Inowedzera Payfast - gedhi rekubhadhara repamhepo rinokurumidza uye rakachengeteka muSouth Africa - sesarudzo yekubhadhara yeMultisite Ultimate.

## Kuisa

1. Isa 'ultimate-multisite-payfast' mudhairekitori re'/wp-content/plugins/'
2. Batidza wedzero kuburikidza nemenyu ye'Mawedzero' muWordPress
3. Tevera Wizard nhanho nenhanho kuti ugadzirise wedzero

## Gwaro Rekushandisa

PayFast ipurosesa inotungamira yekubhadhara pamhepo yemabhizinesi muSouth Africa. Yakazvipira kubatsira vatengesi vepamhepo kukura nekuita kuti kutengeserana kuve nyore uye kwakachengeteka. Maererano nechikumbiro chemushandisi weUltimate Multisite, takasarudza kugadzira wedzero ichaita kuti kubatanidza nePayFast kuve kunogoneka. Chinyorwa ichi chicharatidza magadzirirwo ePayFast segedhi rako rekubhadhara kuti ugamuchire mubhadharo munetiweki yako yeUltimate Multisite.

Nhanho yekutanga kuisa wedzero yacho.

# Maitiro ekuisa wedzero yePayfast:

Enda kupeji remawedzero uye sarudza wedzero yeUltimate Multisite: Payfast Gateway.

![Peji remawedzero](/img/admin/addons-page.png)

Hwindo rewedzero richabuda. Dzvanya paInstall Now.

![Peji remawedzero rine sarudzo yekuisa](/img/admin/addons-page.png)

Kana kuisa kwapera, uchaendeswa kupeji remawedzero. Pano, ingodzvanya paNetwork Activate uye wedzero ye_Payfast_ ichabatidzwa panetiweki yako.

# Kugadzirisa PayFast

Mushure mekuisa wedzero, unogona zvino kuenda kumarongero eUltimate Multisite wotanga kugadzirisa PayFast.

Pasi pepeji re**marongero eUltimate Multisite**, unogona kuwana tebhu ye"Kubhadhara". Pasi petebhu iyi, unofanira kuona "**Payfast** " pamwe nebhatani rekubatidza kana kudzima gedhi iri rekubhadhara.

![Marongero egedhi rekubhadhara](/img/config/settings-payment-gateways.png)Enderera mberi wodzvanya bhatani rekubatidza kuti urigone. Rinobva rawedzera bhokisi kuratidza minda yakasiyana yaunofanira kuzadza yakadai se **Mechant ID, Merchant Key uye PassPhrase.** Unogona kuwana ruzivo urwu kubva kuAccount yako ye**Payfast**

<!-- Mufananidzo hausipo: marongero ePayFast akawedzerwa achiratidza minda yeMerchant ID, Merchant Key, uye PassPhrase -->

Nhanho inotevera kupinda muAccount yako yePayFast kuti uwane ruzivo rwaunoda.

Paunopinda, unogona pakarepo kuona zvitupa zvaunoda kubva paDashboard yeAccount yako ye**Payfast**.

<!-- Mufananidzo hausipo: Dashboard yeAccount yePayFast ichiratidza zvitupa zveMerchant ID neMerchant Key -->Unogona kungokopa wonamira chitupa uye kuzadza mashoko pamangongero ako eUltimate Multisite. Wobva wadzvanya bhatani re**"Save settings** " kuti uchengetedze shanduko.

<!-- Mufananidzo hausipo: marongero eUltimate Multisite PayFast ane zvitupa zvakazadzwa uye bhatani reSave -->

Ndizvozvo, kubatanidzwa kwako kwekubhadhara kwePayFast kwapera!
