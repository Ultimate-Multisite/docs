# Chisarudziri cheMutauro {#language-selector}

Multisite Ultimate: Chisarudziri cheMutauro pakunyoresa

Wedzera nzvimbo yekusarudza mutauro kumaitiro ekunyoresa.

## Kuiswa {#installation}

1. Isa 'ultimate-multisite-language-selector' mudhairekitori re'/wp-content/plugins/'
2. Shandisa chinowedzerwa kuburikidza nemenu ye'Zvinowedzerwa' muWordPress Network Admin yako

## Gwaro Rekushandisa {#usage-guide}

Neiyi wedzero yemahara unogona kurega vashandisi vako vachisarudza mutauro wavanoda kuona pamaitiro ekunyoresa uye papeji rekupinda.

Mushure me[kuvhura rezinesi rako](../../user-guide/getting-started/installing-ultimate-multisite.md) uchakwanisa kuisa wedzero dzeUltimate Multisite dzinopihwa mukana nerezinesi rako.

Enda kupeji rezvirongwa zveUltimate Multisite yako uye ipapo, pamenu iri kuruboshwe, tinya chinyorwa che_Add-ons_.

![Peji rewedzero](/img/admin/addons-page.png)

_Enda kupeji reAdd-ons kuti utange kuisa Chisarudziri cheMutauro_

Ipapo, uchawana wedzero dzese dzatinopa.

Wedzero dzemahara dziripo kune vatengi vedu vese, zvisinei nehurongwa hwekunyorera hwavakasaina.

Tinya bhatani re_Install it Now!_ riri pazasi pemufananidzo mudiki wewedzero. Wobva waishandisa papeji reZvinowedzerwa.

![Peji rewedzero rine bhatani rekuisa](/img/admin/addons-page.png)

_Tinya bhatani kuti uise wedzero_

<!-- Mufananidzo hauwanikwi: peji reWP Plugins rinoratidza bhatani reNetwork Activate reLanguage Selector addon -->

_Ishandise papeji rako reZvinowedzerwa_

Iye zvino, enda kumenu yeSettings yeUltimate Multisite uye pinda mutabhu ye_Network Settings_. Ipapo, pazasi pepeji, uchawana chikamu chitsva chinonzi _Languages_.

![Peji rezvirongwa](/img/config/settings-general.png)

_Chikamu chitsva chose chinoonekwa!_

Unogona kuchisiya chisina chinhu kuti uratidze mitauro yese sesarudzo kana kusarudza yaungada kuti iwanikwe. Mitauro yacho ndiyo yawakaisa mugadziriro yako yeWordPress.

<!-- Mufananidzo hauwanikwi: dropdown yekusarudza mutauro inoratidza mitauro iripo pakuiswa kweWP -->

_Rondedzero yemitauro iripo pakuiswa kweWP uku_

Mushure mekuchengetedza shanduko dzawakaita, menu yekusarudza ine mitauro yawakasarudza icharatidzwa papeji reLogin pamwe chete nepaMaitiro eKunyoresa.

![Zvirongwa zveLogin neRegistration](/img/config/settings-login-registration.png)

_Chisarudziri cheMutauro papeji reLogin_

![Peji reRegistration](/img/config/registration-page.png)

_Chisarudziri cheMutauro paMaitiro eKunyoresa_

## Kugadzirisa Mutauro kuburikidza neURL {#setting-the-language-via-the-url}

Unogonawo kuchinja mutauro wepeji nekupfuura parameter ye locale kuURL. Kana uchida kuendesa kupeji rekunyoresa muchiItalian, semuenzaniso, wedzera ?locale=it_IT kuURL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Mutauro unofanira kunge wakaiswa paWordPress network yako kuti izvi zvishande.
