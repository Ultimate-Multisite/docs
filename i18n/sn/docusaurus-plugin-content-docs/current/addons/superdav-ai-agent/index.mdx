# Superdav AI Agent

Superdav AI Agent mubatsiri wepamberi weWordPress AI unowedzera Gratis AI Agent nehunyanzvi hunoziva dingindira, kugona kutsvaga paindaneti kwakavandudzwa, maficha ekuvaka saiti, uye kubatanidzwa kweotomatiki kunosarudzwa. Shanduro 1.18.0 inowedzera maturusi ekuverenga Google Calendar, kuenzanisa macontact nevapindi, masuwo ekubvumidzwa nemunhu, kubvisa kudzokororwa kwezviyeuchidzo, zviziviso zveTextBee SMS, kupihwa kwesevhisi yeSuperdav inotarisirwa, uye package yeAdvanced companion inoparadzirwa yakaparadzana yemafambiro ebasa evagadziri vanovimbwa.

## Maficha Akakosha

- **Hunyanzvi hunoziva dingindira** — hunogadzirisa otomatiki nhungamiro nekugona zvichienderana nedingindira rako reWordPress riri kushanda uye mapulagini akaiswa
- **Vapeji vakawanda vekutsvaga paindaneti** — sarudza pakati peBrave Search neTavily kuti uwane mibairo yekutsvaga yakapfuma uye kugona kutsvakurudza kwepamberi
- **Mafomu ekutaurirana emuvaki wesaiti** — gadzira mafomu ekutaurirana anoshanda zvizere zvakananga kubva muinterface yekutaura uchishandisa pulagini refomu rawakaisa
- **Muvaki wesaiti akavandudzwa** — zvirongwa zvekuvaka saiti zvematanho akawanda zvakavandudzwa nekuwanikwa kwemapulagini kuri nani uye kupora kubva pazvikanganiso
- **Hunyanzvi hwedingindira hwakavakirwa mukati** — nhungamiro yakasarudzika yeBlock Themes, Classic Themes, Kadence Blocks, neKadence Theme
- **Kubatanidzwa kweWooCommerce kwakavandudzwa** — kwakagadzirwazve kushandisa maAPI eWooCommerce emuno kuti kuvimbika nekuenderana zvive nani
- **Kuonekwa kweprovider otomatiki** — hunoona uye hunovandudza otomatiki mapulagini nemadingindira aripo kana avhurwa kana kuvharwa
- **Kugadzirwa kwelogo yakasarudzika** — gadzira uye nyudza malogo eSVG akasarudzika nekucheneswa kunochengetedza namespace otomatiki
- **Kuongororwa kwekusiyana kwemavara epalette** — tarisa mapeya emavara kuti aenderane neWCAG accessibility usati waashandisa kudingindira rako
- **Kuwanikwa kwakavandudzwa nemifananidzo** — upload mifananidzo panguva yekubvunzurudza kwekuwanikwa kweTheme Builder kuti ikanganise nzira yedhizaini
- **Mamenyu ehospitality** — gadzira mapeji emenyu ezvekudya nezvinwiwa ane chimiro cherestaurant, cafe, nemabhizinesi ehospitality
- **Zviratidzo zvepadesktop nepamobile** — tarisa dhizaini yako padesktop nepamidziyo yemobile panguva yekusarudza nzira yedhizaini
- **Kukwenya saiti kweTier 1** — sd-ai-agent/site-scrape yava kuwanikwa nekusarudzika pakuongorora mawebhusaiti evakwikwidzi neanopa kufemerwa
- **Maturusi ekuverenga Google Calendar** — verenga makarenda nezviitiko zvakagadziriswa kuitira kuti maotomatiki afunge nezvemamiriro ehurongwa huri kuuya
- **Kuenzanisa macontact nevapindi** — fananidza vatori vechikamu vekarenda nevashandisi vesaiti nemacontact anozivikanwa usati wagadzira basa rekutevera
- **Masuwo ekubvumidzwa nemunhu** — misa maotomatiki ane njodzi kuti aongororwe usati watumira mameseji, kutangazve basa, kana kuita shanduko
- **Marekodhi ezviyeuchidzo nekubvisa kudzokororwa** — tevera zviyeuchidzo zvakatotumirwa kuitira kuti kumhanya kweotomatiki kunodzokororwa kusazivise mugamuchiri mumwe chete kaviri
- **Zviziviso zveTextBee SMS** — tumira zviziviso zvemameseji akagadziriswa kuburikidza neTextBee kana provider weSMS akagoneswa
- **Kupihwa kwesevhisi yeSuperdav inotarisirwa** — batanidza masaiti anotsigirwa kumigumo yeSuperdav yakagamuchirwa nekugadzwa otomatiki panzvimbo pekugadzirisa mugumo nemaoko
- **Package yeAdvanced companion** — maturusi evagadziri vanovimbwa anoparadzirwa akaparadzana emafambiro ebasa efilesystem, database, WP-CLI, REST dispatcher, muvaki wepulagini, git snapshot, manejimendi evashandisi, uye benchmark

## Zvinodiwa

- WordPress 6.0 kana yepamusoro
- PHP 8.1 kana yepamusoro
- WordPress Multisite (yakavhurwa panetwork) kana kuiswa kwesaiti imwe
- Pulagini reAI provider connector riri kushanda (Anthropic, OpenAI, kana rinoenderana)
- Gratis AI Agent v1.9.0 kana yepamusoro (Superdav AI Agent inowedzera Gratis AI Agent)

## Kuisa

1. Upload mafaera epulagini ku`/wp-content/plugins/superdav-ai-agent`
2. Vhura pulagini kuburikidza nechidzitiro che**Mapulagini** muWordPress
3. Ita shuwa kuti Gratis AI Agent yakavhurwawo
4. Enda ku**Gratis AI Agent → Settings** uye gadzirisa provider wako wekutsvaga paindaneti, kubatana kwesevhisi yeSuperdav inotarisirwa, Google Calendar, kana magwaro eTextBee sezvinodiwa

Package yeWordPress.org ine maficha makuru eSuperdav AI Agent. Package yeAdvanced companion inovakwa uye inoparadzirwa yakaparadzana yenzvimbo umo maturusi evagadziri vanovimbwa akaiswa uye akapihwa mvumo zviri pachena.

## Kutanga Nekukurumidza

Kana yavhurwa, Superdav AI Agent inovandudza otomatiki kuiswa kwako kweGratis AI Agent nehunyanzvi hunoziva dingindira uye kugona kwekuwedzera. Vhura panel ye**Gratis AI Agent** mubara repadivi readmin yeWordPress wotanga kushandisa maficha matsva.

**Mirayiridzo yemuenzaniso:**

```
Wedzera fomu rekutaurirana kupeji reContact.
```

```
Tsvaga pawebhu maitiro akanakisa ekuchengetedzwa kweWordPress achangoburwa.
```

```
Gadzira chikamu chehero uchishandisa mapatani emabhuroki edingindira riri kushanda.
```

```
Ongorora zviitiko zveGoogle Calendar zvakagadziriswa zvamangwana uye nyora zvinyorwa zvezviyeuchidzo zvevapindi.
```

```
Tumira chiyeuchidzo cheSMS mushure mekunge ndabvumidza meseji.
```

## Hunyanzvi Hunoziva Dingindira

Superdav AI Agent inoona otomatiki dingindira rako riri kushanda nemapulagini akaiswa, ichirodha hunyanzvi hwakakodzera hunoziva dingindira mumamiriro emubatsiri. Hunyanzvi huripo hunosanganisira:

| Hunyanzvi | Hunoshanda Ku |
|---|---|
| **Block Themes** | Madingindira anoshandisa architecture yeWordPress yakavakirwa pamabhuroki (Full Site Editing) |
| **Classic Themes** | Madingindira echinyakare eWordPress anoshandisa matemplate ePHP needhita yekare |
| **Kadence Blocks** | Masaiti ane pulagini reKadence Blocks riri kushanda |
| **Kadence Theme** | Masaiti ane dingindira reKadence sedingindira riri kushanda |

Ona [Hunyanzvi Hunoziva Dingindira](../../user-guide/configuration/theme-aware-skills) kuti uwane nhungamiro yakadzama pahunyanzvi hwega hwega.

## Vapeji Vekutsvaga paIndaneti

Superdav AI Agent v1.10.0 inowedzera **Tavily** seimwe nzira yeBrave Search yemibvunzo yekutsvaga paindaneti. Vapeji vaviri ava vanopa mibairo yekutsvaga yenguva chaiyo uye vanogona kugadziriswa mu**Gratis AI Agent → Settings → Advanced**.

| Mupi | Zvimiro | Kugadzirisa |
|---|---|---|
| **Brave Search** | Mibairo yekutsvaga inokurumidza, inotarisira kuvanzika | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Mibairo yekutsvaga yakapfuma, kugona kwekutsvakurudza kwepamusoro | [Tavily API](https://tavily.com/) |

Ona [Kutsvaga paInternet](../../user-guide/configuration/internet-search) kuti uwane mirayiridzo yekugadzirisa.

## Mafambiro eKarenda, Mvumo, uye Zviziviso

Superdav AI Agent v1.18.0 inowedzera mafambiro eautomation kupfuura mabasa ezvemukati uye ekuvaka saiti:

- **Zvishandiso zveGoogle Calendar** zvinoita kuti agent averenge makarenda akagadziriswa uye ruzivo rwezviitiko kana magwaro ekupinda mukarenda aripo.
- **Vabatsiri vevanhu vekubata nevapindi** vanobatanidza vapindi vezviitiko nevashandisi veWordPress kana vanhu vekubata vanozivikanwa vasati vagadzira zviyeuchidzo kana mabasa ekutevera.
- **Magedhi emvumo yemunhu** anobvumira automation kumbomira kusvikira mushandisi ane mvumo aongorora chiito chakurudzirwa.
- **Marekodhi ezviyeuchidzo** anodzivirira kudzokororwa kwezviyeuchidzo zvekarenda kana zviziviso kana automation ikaedzwazve kana kurongwa kakawanda.
- **Rutsigiro rweTextBee SMS** runogonesa zviziviso zvemameseji kana mupi weTextBee agadziriswa.

Shandisa zvishandiso izvi parubatsiro runoziva hurongwa, kutevera vapindi, uye mafambiro ezviyeuchidzo achiri kuda kuongororwa nemunhu mameseji ane ngozi asati atumirwa.

## Kugadzirisa

Zvirongwa zvese zviri pasi pe **Gratis AI Agent → Settings** muWordPress admin. Superdav AI Agent inowedzera sarudzo dzekugadzirisa dzinotevera:

| Chirongwa | Tsananguro |
|---|---|
| **Internet Search Provider** | Sarudza pakati peBrave Search neTavily (zvekusarudza) |
| **Tavily API Key** | API key yeTavily internet search (kana Tavily yasarudzwa) |
| **Managed Superdav Service** | Batanidza masaiti anotsigirwa kusevhisi yeSuperdav yakagamuchirwa ine kugoverwa kwe endpoint otomatiki |
| **Google Calendar Credentials** | Gadzirisa kuwana karenda kuitira kuti agent averenge makarenda nezviitiko |
| **TextBee SMS Provider** | Gadzirisa magwaro eTextBee emafambiro ezviziviso zveSMS akatenderwa |

## Package Yepamusoro Inoperekedza

Superdav AI Agent v1.18.0 inowedzera package yeAdvanced inoperekedza inogoverwa yakaparadzana yemafambiro evagadziri vanovimbwa. Kugona uku hakusanganisirwi mu package yepakati yeWordPress.org. Isa package inoperekedza chete pamasaiti apo maadministrator vanovimba pachena kuti agent iwane zvishandiso zvevagadziri.

Zvishandiso zveAdvanced zvinosanganisira mabasa efilesystem, kuwana database, WP-CLI, mafambiro eREST dispatcher, zvishandiso zvekuvaka plugin, git snapshots, vabatsiri vekutarisira vashandisi, uye zvishandiso zvebenchmark. Chengeta zvishandiso izvi zvakaganhurirwa kumaadministrator anovimbwa uye ongorora kugadziriswa kweapproval-gate usati wazvigonesa pamasaiti ari kushanda.
