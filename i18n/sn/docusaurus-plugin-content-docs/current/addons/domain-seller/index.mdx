# Domain Seller {#domain-seller}

**Multisite Ultimate: Domain Seller** inoshandura network yako yeWordPress Multisite kuva platform ine simba yekutengesa madomain. Rega vatengi vanyorese madomain avo akasarudzika zvakananga panguva yekunyoresa kana kubva muDashboard yavo, nekutarisa kuwanikwa panguva chaiyo uye mitengo iri pachena.

### Nei Domain Seller? {#why-domain-seller}

Kunyoresa domain ndechimwe chezvinhu zvinonyanya kukumbirwa sekuwedzera kutengeswa kune chero platform yeWaaS (Webhusaiti seSevhisi). Panzvimbo pekutumira vatengi vako kune munyoresi wechitatu-bato, unogona kupa madomain zvakananga — uchichengeta mari yakawanda mu network yako uku uchipa vatengi chiitiko chisina zvipingamupinyi.

### Zvinhu Zvikuru {#key-features}

#### Rutsigiro rweVanyoresi veDomain Vakawanda {#multiple-domain-registrar-support}
* **OpenSRS** — Kunyoresa kuzere, kuvandudza, uye manejimendi
* **Namecheap** — Kubatanidzwa kwakakwana neWhoisGuard uye kuyedza kwe sandbox
* **HostAfrica** — Kutsvaga domain, mitengo, kunyoresa, kutamisa, DNS, uye kuchengetedzwa kweID kuburikidza neDomains Reseller API
* **Openprovider** — Mitengo yevatengesi zvakare, kunyoresa, kuvandudza, kutamisa, DNS, kuvanzika kweWHOIS, uye kuwiriranisa TLD
* **Hostinger** — Kutsvaga domain uye kunyoresa uchishandisa token yekubatanidza yeHostinger yakagovaniswa
* **Chimiro Chinowedzerwa** — Yakavakwa paIntegration Registry pattern, zvichiita kuti zvive nyore kuwedzera vamwe vanopa
* **Kuyedza Kubatana Kwakavakirwa-mukati** — Simbisa magwaro evanopa uye kubatana usati waenda live

#### Wizard yeSetup Inotungamirirwa {#guided-setup-wizard}
* **Kutanga neklik imwe** — Inokutungamira pakugadzirisa mupi, kupinza TLD, uye kugadzira product yedomain
* **Kuwiriranisa TLD otomatiki** — Inopinza maTLD ese aripo uye mitengo yeholesale kubva kuvanopa vako vakabatana
* **Kugadzira product yekutanga** — Inogadzira product yedomain yakagadzirira kushandiswa ine defaults dzemarkup dzine musoro

#### Mitengo yeDomain Inochinjika {#flexible-domain-pricing}
* **Nzira nhatu dzemarkup** — Percentage markup, fixed markup, kana mutengo wakatarwa paTLD
* **Products dzakanangana neTLD** — Gadzira products dzedomain dzakasiyana dzemapoka eTLD akasiyana (.com/.net, .io, macode enyika, nezvimwe)
* **Mitengo yekusuma** — Ipa mutengo wakaderedzwa wegore rekutanga kukwezva kunyoresa kutsva
* **Mitengo yenguva chaiyo** — Mitengo yeholesale live kubva kuvanopa ine caching yakangwara
* **Rutsigiro rweCoupon** — Kuenderana kuzere nemacode ekuderedza mutengo eMultisite Ultimate

#### Kudzivirirwa kweKuvanzika kweWHOIS {#whois-privacy-protection}
* **Kugadzirisa pa-product** — Seta kuvanzika kweWHOIS kuDisabled, Always Included, kana Customer Choice
* **Mutengi kusarudza kupinda** — Kana yaiswa kuCustomer Choice, checkbox yecheckout inobvumira vatengi kuwedzera kudzivirirwa kwekuvanzika
* **Mitengo inogadzirisika** — Seta mutengo pagore weprivacy add-on, kana kuipa mahara
* **Kubatanidzwa nemunyoresi** — Inogonesa otomatiki WhoisGuard (Namecheap) kana kuvanzika kweWHOIS (OpenSRS) panguva yekunyoresa

#### Chiitiko cheCheckout Chakabatana {#unified-checkout-experience}
* **Nzira nhatu dzedomain** — Vatengi vanosarudza pakati pe subdomain (yemahara), kunyoresa domain itsva, kana kumapa domain iripo yavanotova nayo
* **Kutsvaga domain panguva chaiyo** — Kutariswa kwekuwanikwa kunofambiswa neAJAX pasina kurodhazve peji
* **Mazano akangwara** — Mazano emamwe maTLD kana domain inodiwa yatotorwa
* **Mitengo iri pachena** — Kuratidzwa kwakajeka kwemutengo wekunyoresa, mutengo wekuvandudza, uye muripo weWHOIS privacy ungasarudzwa
* **Mafield ekubatana nemunyoresi** — Inounganidza ruzivo runodiwa rwemunyoresi (zita, kero, foni) inline panguva yecheckout
* **URL yesite inogadzirwa otomatiki** — Kero yesite inotorwa otomatiki kubva kudomain yakasarudzwa

#### System yeAuto-Renewal {#auto-renewal-system}
* **Kuvandudza kwakabatana nemembership** — Madomain anovandudzwa otomatiki apo membership yemutengi ichiri kushanda
* **Kuronga kwakangwara** — Mabasa ekuvandudza anoiswa mumutsetse zvichienderana nemazuva ekupera kwedomain
* **Logic yekuyedzazve** — Kuyedzazve kwakangwara pakuedza kuvandudza kwakundikana
* **Zviziviso zveemail** — Zviyeuchidzo zvinogadzirisika zvekuvandudza kuri kuuya, kuvandudza kwakabudirira, kukundikana, uye kupera

#### Manejimendi yeDNS yeMutengi {#customer-dns-management}
* **DNS yekuzvishandira** — Vatengi vanogona kuwedzera, kugadzirisa, uye kudzima marecord eDNS (A, AAAA, CNAME, MX, TXT) kubva muDashboard yavo
* **Mamiriro edomain** — Mamiriro ekunyoresa panguva chaiyo, mazuva ekupera, uye ruzivo rwenameserver
* **Kumapa domain otomatiki** — Madomain akanyoreswa kana aripo anomapiwa otomatiki kusite yemutengi

#### Zvishandiso zveAdmin {#admin-tools}
* **Kunyoresa domain nemaoko** — Nyoresa madomain pachinzvimbo chevatengi kubva pa admin panel
* **Kutarisa domain** — Ona madomain ese evatengi, mamiriro, uye mazuva ekupera kubva papeji rimwe
* **Logging yakatarisana nemupi** — Machannel elog akaparadzana emunyoresi wega wega (semuenzaniso, domain-seller-namecheap.log)
* **Zvinhu zvekarejista** — Ona ruzivo rwemunyoresi uye data rekunyoresa papeji huru yekugadzirisa domain

### Vanopa Vanotsigirwa {#supported-providers}

* **OpenSRS** — Kubatanidzwa kuzere nekarejista, kuvandudza, manejimendi yeDNS, uye kuvanzika kweWHOIS
* **Namecheap** — Kubatanidzwa kuzere nekarejista, kuvandudza, kutarisa kuwanikwa kwebatch, uye WhoisGuard yemahara
* **HostAfrica** — Kunyoresa, kuvandudza, kutamisa, manejimendi yeDNS, kutora EPP code, registrar lock, uye kuchengetedzwa kweID
* **Openprovider** — Kunyoresa, kuvandudza, kutamisa, manejimendi yeDNS, kutora EPP code, registrar lock, kuvanzika kweWHOIS, uye kuwiriranisa TLD yevatengesi zvakare
* **Hostinger** — Kunyoresa, kutsvaga domain, kuvandudzwa kwenameserver, registrar lock, uye kuvanzika kweWHOIS; kuvandudza kunobatwa neHostinger auto-renew
* **GoDaddy** — Kunyoresa uye kuvandudza
* **ResellerClub** — Kunyoresa, kuvandudza, uye manejimendi yeDNS
* **NameSilo** — Kunyoresa uye kuvandudza
* **Enom** — Kunyoresa, kuvandudza, uye manejimendi yeDNS

### Zvinhu zveTekinoroji {#technical-details}

* **Inoda:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **Chimiro:** Integration Registry pattern ine Domain_Selling_Capability interface
* **Autoloading:** Jetpack autoloader ine PSR-4 class mapping
* **Miyero:** WordPress coding standards, strict typing, prepared statements

## Kuisa {#installation}

1. Iva nechokwadi chekuti **Multisite Ultimate v2.4.12+** yakaiswa uye yashandisirwa pa network
2. Isa chowedzerwa ku `/wp-content/plugins/ultimate-multisite-domain-seller/`
3. Shandisai chowedzerwa pa network kubva ku **Zviwedzerwa** muWordPress Network Admin
4. **Mubatsiri Wekumisikidza** anotanga otomatiki pakushandisirwa kwekutanga — tevera matanho kuti:
   * Ubatanidze murejisitiri wako we domain
   * Uyedze kubatana
   * Upinze TLDs dziripo nemitengo ye wholesale
   * Ugadzire chigadzirwa chako chekutanga che domain chine markup yemitengo
5. Wedzera nzvimbo ye **Sarudzo yeDomain** kufomu rako rekupedzisa kutenga mu **Ultimate Multisite > Mafomu Ekupedzisa Kutenga**

## Mibvunzo Inowanzo Bvunzwa {#frequently-asked-questions}

### Ndeipi vhezheni yeMultisite Ultimate inodiwa? {#what-version-of-multisite-ultimate-is-required}

Vhezheni 2.4.12 kana yepamusoro. Chowedzerwa hachienderani neWP Ultimo v1.x kana mavhezheni eUltimate Multisite ari pasi pe2.4.12.

### Ndevapi vanopa domain vanotsigirwa? {#which-domain-providers-are-supported}

Marejisitiri mapfumbamwe anotsigirwa: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo**, uye **Enom**. Kugona kunosiyana zvichienderana nemupi nekuti maAPI emarejisitiri anoratidza mashandiro akasiyana; ona gwaro rekumisikidza kuti uwane pfupiso yemupi mumwe nemumwe. Dhizaini yeIntegration Registry yechowedzerwa inoita kuti zvive nyore kuwedzera vanopa vakagadziridzwa — ona [gwaro remugadziri](/developer/integration-guide/custom-registrar).

### Ndingapa dziviriro yeWHOIS privacy here? {#can-i-offer-whois-privacy-protection}

Hongu. Chigadzirwa chega chega che domain chine marongero eWHOIS Privacy ane nzira nhatu: Yakadzimwa, Inogara Yakabatanidzwa, kana Sarudzo yeMutengi. Kana yaiswa kuSarudzo yeMutengi, vatengi vanoona bhokisi rekusarudza panguva yekupedzisa kutenga rine mutengo (kana "Yemahara" yeNamecheap's WhoisGuard). Mureza we privacy unotumirwa otomatiki kumurejisitiri panguva yekunyoresa.

### Ndingaisa mitengo yakasiyana yeTLDs dzakasiyana here? {#can-i-set-different-pricing-for-different-tlds}

Hongu. Gadzira zvigadzirwa zve domain zvakasiyana zvemapoka eTLD akasiyana. Chigadzirwa chega chega chinogona kuva nerudzi rwacho rwe markup (muzana, markup yakatarwa, kana mutengo wakatarwa), TLDs dzinotsigirwa, uye mitengo yekutanga.

### Ndedzipi sarudzo dzekupedzisa kutenga dzinoonekwa nevatengi? {#what-checkout-options-do-customers-see}

Nzvimbo yeSarudzo yeDomain inopa vatengi matabhu matatu: **Subdomain** (shandisa subdomain yemahara pa network yako), **Nyoresa Domain Itsva** (tsvaga uye nyoresa domain itsva), uye **Domain Iripo** (batanidza domain yavanotova nayo). Nzira dzaunogonesa chete ndidzo dzichaonekwa.

### Inotsigira kuvandudzwa otomatiki here? {#does-it-support-automatic-renewals}

Hongu. Madomain anogona kuiswa kuti avandudzwe otomatiki uye anosungirirwa kumamiriro enhengo emutengi. Kubhadhara kwekuvandudza kunogadziriswa kuburikidza negedhi rekubhadhara riripo remutengi.

### Vatengi vanogona kutonga DNS records dzavo here? {#can-customers-manage-their-dns-records}

Hongu. Vatengi vanowana nzvimbo yekutonga DNS mudashboard yavo kuti vawedzere, vagadzirise, uye vadzime marekodhi eA, AAAA, CNAME, MX, uye TXT. Kutonga DNS kunowanikwa neOpenSRS, ResellerClub, Enom, HostAfrica, uye Openprovider. Hostinger DNS yemadomain akahostwa inobatwa necore Hostinger domain-mapping integration.

### Pane mubatsiri wekumisikidza here? {#is-there-a-setup-wizard}

Hongu. Mubatsiri wekumisikidza anomhanya otomatiki pakushandisirwa kwekutanga uye anokutungamirira pakubatanidza murejisitiri, kupinza TLDs, uye kugadzira chigadzirwa chako chekutanga che domain. Unogonawo kumhanyisa zvakare mubatsiri kubva kumarongero echowedzerwa.
