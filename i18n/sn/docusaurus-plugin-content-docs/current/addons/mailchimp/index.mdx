# Mailchimp {#mailchimp}

Multisite Ultimate: Kubatanidza neMailchimp

Batanidza network yako yeMultisite Ultimate neMailchimp yako. Unogona kubatanidza mamwe maplan neMailchimp Lists uye kuwedzera otomatiki vanyoresi kuma list iwayo pavanonyoresa.

## Kuisa {#installation}

1. Isa 'ultimate-multisite-mailchimp' mudhairekitori re'/wp-content/plugins/'
2. Activate plugin kuburikidza nemenyu ye'Plugins' muWordPress Network Admin yako

## Gwaro reKushandisa {#usage-guide}

Kuwiriranisa maemail evanyoresi nema list ako eMailChimp kuri nyore chaizvo neUltimate Multisite: MailChimp Integration Add-on yedu. Dzidziso iyi ine chinangwa chekukubatsira kugadzirisa zvese!

## Kuwana API key {#getting-an-api-key}

Tisati tatanga kupinda mumasetting akasiyana aripo, tinoda MailChimp API key. Izvi zvichabvumira Ultimate Multisite kutaura neMailChimp API kuti itore ma list, mapoka, uye kuwedzera maemail address kuma list nemapoka iwayo kana vashandisi vanyoresa.

Tevedzera dzidziso iyi kuti uwane MailChimp API key yako: [Nezve API Keys | MailChimp](https://kb.mailchimp.com/integrations/api-integrations/about-api-keys)

<!-- Screenshot unavailable: MailChimp API key settings in Ultimate Multisite Add-on Settings -->

_Wedzera MailChimp API key yako pa_ _**Ultimate Multisite Settings - > Add-on Settings -> MailChimp**_

## Maitiro eKubatanidza {#integration-mode}

MailChimp Integration add-on inopa “maitiro ekubatanidza” maviri, zvinoreva kuti pane nzira mbiri dzakasiyana dzaunogona kushandisa add-on kuwiriranisa maemail address.

<!-- Screenshot unavailable: MailChimp integration mode selection with Multiple Lists and Single List options -->

_Maitiro maviri ekubatanidza kuti pave nekuchinjika kukuru!_

### Maitiro 1: Multiple Lists {#mode-1-multiple-lists}

Maitiro ekutanga anoshandisa **Multiple Lists**. Maitiro aya achakubvumira kusarudza maMailChimp lists akawanda kune rimwe nerimwe rema plan. Iyi ndiyo sarudzo yakanakisisa kana uchipatsanura vashandisi vako muma list akasiyana.

Semuenzaniso, ngatitii une ma list matatu akasiyana paMailChimp account yako: **Newsletter** , **Plan A** , uye **Plan B**. Unoda kuti vashandisi vese, zvisinei neplan yavo, vavewo muNewsletter list, kuitira kuti vasapotsa zvigadziriso zvako.

Kuti uite izvozvo, unogona kushandisa maitiro eMultiple Lists kuwedzera ma list ese ari maviri **Newsletter** ne**Plan A** kuPlan A MailChimp Settings, uye **Newsletter** ne**Plan B** kuPlan B MailChimp Settings.

<!-- Screenshot unavailable: MailChimp default lists selection for new accounts and canceled accounts -->

_Sarudza ma list ekutanga emaaccount matsva uye emaaccount akanzurwa_

### Maitiro 2: Single List + Multiple Groups {#mode-2-single-list--multiple-groups}

MailChimp inokubvumira kupatsanura list yako kuita mapoka madiki evashandisi anonzi **Groups**. Izvi zvinokubvumira kuva nekutonga kwakawanda sezvo uchigona kutumira campaign kuchikamu chakati chevanyoresi vako pasina kugadzira makopi akawanda e campaign imwe chete paMailChimp (izvo zvinoita kuti zviome kuongorora metrics e campaign iyoyo gare gare).

Semuenzaniso, unogona kuva ne list imwe inonzi **My Network Subscribers,** uye mukati mayo muve nemapoka e**Plan A** ne**Plan B**.

<!-- Screenshot unavailable: MailChimp single list and groups selection for default lists -->

_Sarudza ma list nemapoka ekutanga, kana uchishandisa maitiro ekubatanidza e“single list + multiple groups”_

## Ma List neMapoka Ekutanga {#default-lists-and-groups}

Mushure mekunge wasarudza kuti ndeapi maitiro ekubatanidza anoenderana zvakanyanya nenyaya yako yekushandisa chaiyo, yave nguva yekusarudza ma list ekutanga (nemapoka, zvichienderana nemaitiro). List yekutanga ichashandiswa kana plan yakapihwa isina List chaiyo yakasarudzwa.

Unogonawo kusarudza list yekutanga yemaaccount akanzurwa. Pose apo mushandisi anomisa subscription yake uye obvisa account yake kubva munetwork yako, email address yake ichaendeswa ku list yakagadzirirwa maaccount akanzurwa.

## Kugonesa Double Subscription {#enabling-double-subscription}

Papeji rimwe chete remasetting, uchakwanisa kugonesa **Double Subscription**. Sarudzo iyi ichawedzera mushandisi ku list yekutanga yakasarudzwa UYE ku list yakasarudzwa yeplan yemushandisi iyeye. Kana sarudzo iyi isina kugoneswa, mushandisi achangowedzerwa chete ku list yakasarudzwa yeplan yemushandisi. Kana pasina yakagadzirirwa plan iyoyo chaiyo, mushandisi achawedzerwa ku list yekutanga yakasarudzwa.

<!-- Screenshot unavailable: MailChimp double subscription toggle setting -->

_Unogona kuita kuti vashandisi vako vanyoreswe ku list yekutanga pamwe ne list yeplan panguva yekunyoresa_

## Kusarudza Ma List neMapoka kune rimwe nerimwe rema Plan {#selecting-lists-and-groups-for-each-of-the-plans}

Enda ku **Plans** dzako uye sarudza plan yaunoda kugadzirisa. Cherechedza kuti pa **Advanced Options** meta-box, tab itsva inonzi MailChimp ichawanikwa.

Pa tab iyoyo, uchakwanisa kusarudza ma list akawanda kana list imwe chete nemapoka akawanda (zvichienderana nemaitiro ekubatanidza) eplan iyoyo.

<!-- Screenshot unavailable: MailChimp tab on plan edit page showing lists and groups selection -->

_Sarudza ma Lists neGroups aunoda kushandisa kune rimwe nerimwe rema plan ako_

Kana mushandisi mutsva akanyoresa ku plan iyoyo, email address yake ichawedzerwa kuma list nemapoka akasarudzwa!
