# Admin Page Creator

## Zviri mukati
Admin page creator ichakubvumira kuwedzera zviri nyore mapeji matsva eadmin nemapeji madiki mukati meWordPress admin dashboard. Inogona kushandiswa kugadzira peji rekugamuchira kana peji redzidziso rinogona kushanda sepeji rekutanga revashandisi vatsva. Unogona kushandisa page builder yakaita seBeaver Builder, Elementor, Brizy, neOxygen kugadzira mapeji pasina kunetseka.

Inopawo rutsigiro rwecustom HTML, CSS neJavaScript, pamwe nekudzora kuwana kunoenderana neplan nebasa remushandisi!

## Zvimiro
* Inotsigira Custom HTML neCSS
* Rutsigiro rweJavaScript
* PHP Content Mode
* Mvumo inoenderana nebasa reMushandisi
* Rutsigiro kuma page builders

## Zvinyorwa
[Kugadzira Custom Admin Pages nemapeji madiki uchishandisa Multisite Ultimate: Admin Page Creator](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## Kuisa

1. Isa **ultimate-multisite-admin-page-creator** kudhairekitori re **/wp-content/plugins/**
2. Activate plugin kuburikidza nemenu ye **Plugins** muWordPress Network Admin yako

## Gwaro rekushandisa

Kunyange zvazvo WordPress ichiita kuti zvive nyore chaizvo kuwedzera zvemukati kumberi kwemasaiti, kugadzira mapeji nemapeji madiki kudivi readmin kunoda humwe hunyanzvi hwekutanga hwekukodha. Ultimate Multisite: Admin Page Creator inovavarira kuunza nyore kushandisa kwakafanana kwekumberi kudivi rekumashure remasaiti enetwork yako.

## Sei uchishandisa Admin Page Creator?

Pamwe unoda kupa peji rakagadziridzwa papaneli yeadmin yevatengi vako rine tutorials, pamwe unoda kuwedzera peji rakagadziridzwa rine FAQs newidget yekubata. Mikana haina magumo neAdmin Page Creator, uye ndizvo zvatichaongorora mudzidziso yanhasi.

## Kugadzira Custom Admin Page yako yekutanga

Mushure mekuisa uye kuita activate Ultimate Multisite: Admin Page Creator, enda kumenu yeadmin yenetwork yako uye uende kuAdmin Pages.

![Rondedzero yeCustom Admin Pages paNetwork Admin](/img/admin/admin-pages-list.png)

_Rondedzero yeCustom Admin Pages paNetwork Admin_

Uchaona rondedzero yemapeji ese eadmin awakagadzira kusvika pari zvino (sezvo uku kuri kushanya kwako kwekutanga, rondedzero ichange isina chinhu). Enderera mberi wobaya **Wedzera Admin Page itsva**. Uchaendeswa kueditor yeAdmin Page.

## Sarudzo dzeMenu

Kudivi, paGeneral Options meta-box, uchakwanisa kugadzirisa rudzi rwemenu, label, nezvimwe zvakawanda.

### Mhando dzeMenu

Peji rako idzva recustom admin rinogona kuva nemhando mbiri dzemenu: top-level menu uye submenu.

Peji re top-level menu, sezvinorehwa nezita, richawedzera peji rako rawangogadzira kumenu yeadmin pamwe chete nemapeji eWordPress aripo.

Kana ukasarudza sarudzo ye top-level, uchakwanisawo kusarudza kurongeka kwemenu neicon yemenu.

Kana uchida kuita kuti peji iri rive peji re sub-menu, uchakwanisa kusarudza peji remubereki rekunamatira peji iri pariri. Unogona kuwedzera peji rako rakagadziridzwa kumapeji eWordPress aripo kana kumapeji ako ecustom admin e top-level.

## Sarudzo dzepamusoro

Unofanira kukwanisa kuwedzera mhando dzese dzezvemukati kumapeji ako ecustom admin. Izvi zvinosanganisira mitemo nemafaira eCSS, JavaScript, uye maraibhurari ekunze zvakare. Unogona kuita izvozvo uchishandisa Advanced Options Tab.

### Custom CSS

PaCSS tab, unogona kuwedzera code yako yeCSS yakagadziridzwa, pamwe nekuimport mafaira eCSS kubva kumasaiti ari kure (akaita seCDNs, Google Fonts, nezvimwe).

<!-- Screenshot haipo: Editor yecustom CSS yeAdmin Page Creator ine import yeexternal stylesheet -->

_Custom CSS inotsigirwawo. Unogonawo kusanganisira mafaira estyle ekunze!_

### Custom JavaScript

Zvakafanana zvinoshandawo kuJavaScript. Unogona kuwedzera maraibhurari ekunze ekushandisa paJavaScript code yako yakagadziridzwa.

<!-- Screenshot haipo: Editor yecustom JavaScript yeAdmin Page Creator ine import yeraibhurari yekunze -->

## Mvumo

Zvakare muAdvanced Options, uchakwanisa kusarudza kuti ndeapi ma sub-sites achava nekuwana peji iri. Izvi zvinobatsira kana uri kuronga kushandira zvemukati zvakasiyana uye zvakavharirwa kumapoka akasiyana eplan, semuenzaniso.

<!-- Screenshot haipo: Marongero emvumo eAdmin Page Creator ekudzora kuwana kunoenderana neplan nebasa -->

_Une kutonga kuzere pamusoro pekuti peji iri richaratidzika kupi_

**Zvakakosha** : Mamiriro ose ari maviri anofanira kuzadzikiswa kuti peji iratidzwe. Izvi zvinoreva kuti kana mushandisi ari client yeplan yakati, asi asina rimwe remabasa akasarudzwa (kana zvinopesana nazvo) peji harizooneki papaneru yake.

## Active

Kana uchida kudzima custom admin page pasina kuidzima zvachose kana kukanganisa permissions, ingobvisa kusarudzwa kweActive option.

Kudzima top-level page kuchadzima otomatiki ma sub-pages ese akabatanidzwa nepeji remubereki iroro.
