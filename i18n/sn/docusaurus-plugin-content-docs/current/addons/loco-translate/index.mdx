# Loco Translate {#loco-translate}

Wedzero reUltimate Multisite: Loco Translate Integration rinounza kugona kwekushandura kwakanangana nesaiti kune network yako yeWordPress multisite. Kana wakamboda kuti masaiti akasiyana ari munetwork yako ave neshanduro dzakasiyana dzechinowedzerwa chimwe chete kana dingindira rimwe chete, wedzero iri ndiro mhinduro yakakwana.

**Zvinoitwa Newedzero Iri**

Wedzero iri rinobatanidza Ultimate Multisite zvisina kunetsa nechinowedzerwa chakakurumbira cheLoco Translate, zvichiita kuti saiti imwe neimwe yepasi munetwork yako ichengete shanduro dzayo dzakagadzirirwa. Ungave uchimhanyisa chikuva chematunhu mazhinji, kushandira nharaunda dzemitauro yakasiyana, kana kungoda kuchinjika kwekushandura munetwork yako yose, wedzero iri rinoita kuti zvive nyore.

**Rakakodzera Kune:**

* Mabhizimisi ematunhu mazhinji ane zvinodiwa zvakasiyana zvekugadzirisa zvemuno
* Masangano edzidzo anoshandira nharaunda dzakasiyana
* Mapuratifomu eSaaS anopa rutsigiro rwemitauro mizhinji
* Maweb agencies anotarisira masaiti evatengi vakawanda
* Chero network ye multisite inoda kuchinjika kwekushandura

## Kuiswa {#installation}

1. Ita shuwa kuti Ultimate Multisite yakaiswa uye yakabatidzwa panetwork
2. Isa uye batidza chinowedzerwa cheLoco Translate
3. Isa wedzero iri mudhairekitori rako rezvinowedzerwa
4. Batidza panetwork wedzero reLoco Translate Integration
5. Tanga kugadzira shanduro dzakanangana nesaiti kuburikidza neLoco Translate pane chero saiti yepasi

Ndizvozvo! Kubatanidzwa kunoshanda otomatiki pasina chero imwe gadziriso yekuwedzera.

## Mibvunzo Inowanzo Bvunzwa {#frequently-asked-questions}

## Ndinofanira kugadzirisa chimwe chinhu mushure mekuiswa here? {#do-i-need-to-configure-anything-after-installation}

Kwete! Kubatanidzwa kunoitika otomatiki. Kana wedzero neLoco Translate zvese zvabatidzwa, masaiti epasi anogona kutanga pakarepo kugadzira shanduro dzawo dzakagadzirirwa.

## Chii chinoitika kana saiti yepasi ikasagadzira shanduro dzakagadzirirwa? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

Saiti ichashandisa otomatiki shanduro dzepanetiweki yose sechitsivi chekudzokera kwachiri. Hapana chinotyoka, uye zvese zvinoshanda zvisina kunetsa.

## Masaiti epasi anogona kuwana shanduro dzepanetiweki yose here? {#can-subsites-access-network-wide-translations}

Maadmin esaiti akajairika anoona uye kugadzirisa chete shanduro dzesaiti yavo. Ma super admins anogona kuwana shanduro dzepanetiweki yose pamwe nedzakanangana nesaiti zvichienderana nesaiti yavari kuona.

## Izvi zvinoshanda nezvinowedzerwa nemadingindira zvese here? {#does-this-work-with-both-plugins-and-themes}

Ehe! Shanduro dzakanangana nesaiti dzinoshanda kuzvinowedzerwa zveWordPress nemadingindira zvese.

## Izvi zvichanonotsa network yangu ye multisite here? {#will-this-slow-down-my-multisite-network}

Kwete. Wedzero rinoshandisa kurodha mafaera kunoshanda zvakanaka kunongotarisa shanduro dzakanangana nesaiti pazvinenge zvichidiwa chete. Hapana kukanganisa kunooneka pakuita.

## Ndinogona kukopa shanduro dzepanetiweki kune saiti yakati kuti sechinhu chekutangira here? {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

Ehe! Paunenge uchigadzirisa shanduro muLoco Translate, unogona kutanga zviri nyore neshanduro dzepanetiweki wozodzigadzirisa kune saiti imwe neimwe.

## Izvi zvinoshanda neWooCommerce nezvimwe zvinowedzerwa here? {#does-this-work-with-woocommerce-and-other-plugins}

Zvirokwazvo! Chero chinowedzerwa kana dingindira rinoshandisa mabasa akajairika eWordPress ekushandura richashanda neshhanduro dzakanangana nesaiti.
