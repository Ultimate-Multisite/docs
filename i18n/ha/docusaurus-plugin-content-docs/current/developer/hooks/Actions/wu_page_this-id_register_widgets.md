# Action: wu_page_\{$this->id\}_register_widgets

Yana aiki bayan an yi rajistar widgets don wannan shafi.

The dynamic portion of the hook name, `$this-&gt;id`, refers to the page id.

## Sigogi

| Suna | Nau'i | Bayani |
|------|------|-------------|
| $id | `string` | ID na shafin. |
| $page_hook | `string` | Hook na shafin. |
| $page | `object` | Object na shafin. |
