# Tace: wu_sso_enabled {#filter-wussoenabled}

Kunna/kashe ikon single-sign-on tsakanin domain-daban.

Tace wannan ƙima don kashe single-sign-on gaba ɗaya, ko kuma kunna shi bisa sharadi maimakon haka.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $enabled | `bool` | Ya kamata a kunna SSO? True don kunnawa, false-ish don kashewa. |

### Tun daga {#since}

- 2.0.11
### Tushe {#source}

An bayyana shi a [`inc/sso/class-sso.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/sso/class-sso.php#L110) a layi na 110
