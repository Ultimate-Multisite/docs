# Filter: wu_get_site_domain_and_path {#filter-wugetsitedomainandpath}

Bai wa masu haɓakawa damar sarrafa ma’auratan domain/path.

Wannan na iya zama da amfani ga abubuwa da dama, kamar aiwatar da wani irin tsarin staging, sabobin daban-daban, da sauransu.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $d | `object` | Object na yanzu mai ɗauke da maɓallan domain da path. |
| $path_or_subdomain | `string` | Asalin path/subdomain da aka aika zuwa function. |

### Tun daga {#since}

- 2.0.0
