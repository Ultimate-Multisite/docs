# Aiki: move_site {#action-movesite}

Yana faruwa bayan an matsar da rukunin yanar gizo zuwa sabuwar cibiyar sadarwa.

## Sigogi {#parameters}

| Suna | Nau'i | Bayani |
|------|------|-------------|
| $site_id | `int` | ID na rukunin yanar gizon da aka matsar. |
| $old_network_id | `int` | ID na asalin cibiyar sadarwa don rukunin yanar gizon. |
| $new_network_id | `int` | ID na cibiyar sadarwar da aka matsar da rukunin yanar gizon zuwa gare ta. |

### Tun daga {#since}

- 1.3.0
