# Mai Gina Shafi ta AI {#ai-site-builder}

ultimate-multisite-ai-site-builder

## Bukatu {#requirements}

- WordPress 5.3 ko mafi girma
- PHP 7.4 ko mafi girma
- Multisite Ultimate plugin (mai aiki)

## Shigarwa {#installation}

1. Loda fayilolin addon zuwa kundin `/wp-content/plugins/` naka
2. Kunna plugin ta menu na 'Plugins' a cikin WordPress
3. Daidaita saitunan addon a cikin mai gudanarwar hanyar sadarwa

## Ci gaba {#development}

### Saiti {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Tsarin Aiki {#project-structure}

```
ultimate-multisite-ai-site-builder/
```

### Gwaji {#testing}

Addon ɗin yana ƙunshe da cikakken tsarin gwaji:

- **Gwaje-gwajen Unit**: Gwaje-gwaje bisa PHPUnit tare da WordPress test suite
- **Base Test Class**: Yana samar da hanyoyin taimako don al’amuran gwaji na gama-gari
- **Rufin Gwaji**: An daidaita shi don bin diddigin code coverage
- **Gwajin Multisite**: Gwaje-gwaje suna gudana a cikin muhallin multisite

### Ƙa’idojin Code {#code-standards}

- **WordPress Coding Standards**: Ana tilasta su ta hanyar PHPCS
- **Dacewa da PHP 7.4+**: Ana goyon bayan fasalolin PHP na zamani
- **Bincike na Static**: Haɗin PHPStan don duba nau’i
- **Sabunta Code**: Dokokin Rector don haɓaka PHP

### Tsarin Gina {#build-system}

- **Sarrafa Assets**: Rage girman fayilolin CSS/JS
- **Fassara**: Samar da fayil ɗin POT
- **Kunshewa**: Ƙirƙirar archive don rabawa
- **Yanayin Ci gaba**: Assets marasa ragewa don debugging

## Daidaitawa {#configuration}

Addon ɗin yana goyon bayan zaɓuɓɓukan daidaitawa daban-daban da ake iya samu ta hanyar hanyar sadarwar mai gudanarwa.

## Hooks da Filters {#hooks-and-filters}

### Actions {#actions}

- `ultimate-multisite-ai-site-builder_init` - Ana kunna shi bayan fara addon
- `ultimate-multisite-ai-site-builder_loaded` - Ana kunna shi bayan an loda duk fayilolin addon

### Filters {#filters}

- `ultimate-multisite-ai-site-builder_settings` - Gyara saitunan addon
- `ultimate-multisite-ai-site-builder_enabled` - Sauya matsayin kunnawa/kashewa na addon
