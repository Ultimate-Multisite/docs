# گڑولکردن لە فایلەکە {#error-installing-the-sunrise-file}

فایلەکەی `sunrise.php` فایلی تایبەتە کە وۆردپريس (WordPress) لە کاتی سەرەتایی کارکردنی خۆی دەگەڕێتەوە بۆی. بۆ ئەوەی وۆردپريس بتوانێت ئەم فایلییە بەدوای بگرێت، دەبێت لەناو دۆلێری **wp-content**دا جێگیر بێت.

کاتێک Ultimate Multisite چالاک دەکەیت و ڕێنماییەکانی سیستەم (setup wizard) وەک ئەو یەکەی کە لە شاتسکرۆپەدا هەیە، Ultimate Multisite هەوڵ دەدات فایلی `sunrise.php` ئێمە بۆ دۆلێری wp-content کۆپی بکات.

زۆربەی کات، دەتوانین بە سەرکەوتوویی فایلەکە کۆپی بکەین و هەموو شتێک کار دەکات. بەڵام ئەگەر شتێک بە دروست نەبێت (وەک ڕێزموونی دۆلێری فایلی)، دەتوانیت لەو کاتی بگەیت کە Ultimate Multisite ناتوانێت فایلەکە کۆپی بکات.

ئەگەر پەیامی هەڵەی Ultimo وەربگریت، دەبینیت کە بەم شێوەیە ڕوودەدات: **Sunrise copy failed** (کۆپکردنەوەی Sunrise شکستخۆدا).

بۆ چارەسەرکردنی ئەمە، دەتوانیت سادەکان فایل `sunrise.php` لەناو دۆلێری پلاتینی wp-ultimo کۆپی بکەیت و لە ناو دۆلێری wp-content یەکەت چسپێنیتەوە. دوای ئەوەی ئەوە بکەیت، پەڕەی ڕێنماییەکە بەردەوام بکە و تاقیکردنەوەکان دەبێت سەرکەوتوو بن.

لە هەر حاڵەتێکدا، ئەمە دەتوانرێت بگەڕێیت بۆ چاککردنی ڕێزموونی دۆلێری فایلی خۆت بۆ ئەوەی لە داهاتوودا کێشەیەک بەسەر نێزیک نەبێت (نەک تەنها لەگەڵ Ultimate Multisite، بەڵکو لەگەڵ پلاتینەکان و قالبەکانیش).

**Health Check tool** کە بەشێکە لە وۆردپريس (دەتوانیت بە ڕێگەی **admin panel > Tools > Health Check** بۆی دەربکەیت)، توانای ئەوەی هەیە بڵێیت ئایا ڕێزموونی دۆلێری فایلی تۆ بە ئەو بەهایانە کراوە کە دەتوانرێت ببێتە هۆی کێشەیەک لەگەڵ وۆردپريسدا.
