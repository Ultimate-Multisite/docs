# Zabi da amfani da snippets na Ultimate Multisite a GitHub repository ɗinmuwar mu

Akwai ƙananan kodin da ke cikin GitHub repository wanda masu amfani na Ultimate Multisite suke bukata sosai, waɗanda suke so su ƙara ƙarin ayyuka kamar ƙara script na Google Analytics a hanyoyin shiga (sign-up pages) ko ninka wani meta box daga dashboard na admin.

Wannan rubutun zai nuna maka yadda za ka yi amfani da su ko kuma inda za ka saka waɗannan kodin.

Za ka samu snippets a link a nan:

https://github.com/next-press/wp-ultimo-snippets/

Akwai hanyoyi biyu don ƙara kudin:

  1. A fayil ɗin functions.php na theme ɗinka.

  2. Must-Use Plugins (mu-plugins)

# Yadda za ka saka snippet a fayil ɗin functions.php na theme ɗinka.

  1. Shiga dashboard na admin na WordPress Network ɗinka kuma ka je Themes > Theme Editor (Ka gani screenshot a ƙasa).

  2. A shafin "Edit Themes", tabbatar cewa ka zaɓi theme ɗinka da ke aiki a cikin dropdown field da ke kan gefen yau da sauri na wajen haguwa (screenshot #3 a ƙasa).

  3. Ka danna fayil ɗin functions.php a ƙarƙashin sashi "Theme Files" don shiga shi. Ka sauka wajen ƙasa kuma ka dabba Ultimate Multisite snippet da ka samu daga GitHub repository.

# Yadda za ka ƙirƙiro Must-Use Plugins (mu-plugins)

WordPress yana da wata hanya tana ba shi damar shiga ayyuka na musamman wanda ake kira "Must-Use Plugins", ko a taƙa cewa "mu-plugins".

Wannan mu-plugins na musamman suna shiga kafin duk wani plugin na yau da kullum, kuma ba za a iya sanya su a waje (deactivate) ba. A cikin multisite network, kodin da ke cikin waɗannan mu-plugins zai shiga a dukkan sites a cikin install ɗinka.

1. Yi amfani da FTP ko SSH don samun damar shiga fayil ɗin WordPress ɗinka.

2. Lo' cikin direktorya wp-content na WordPress ɗinka, ƙirƙiro direktorya mai suna: mu-plugins.

<!-- Screenshot ba shi ba: File manager wanda ke nuna direktorya wp-content da folder mu-plugins -->

3. Ka ƙirƙira fayil PHP na sabon a kan kwamfutarka mai suna wu-snippet.php ta amfani da Notepad ko wata code editor.

4. Ka saka code snippet na Ultimate Multisite da ka samu daga GitHub repository a cikin wannan fayil kuma ka danna save. Za ka iya ƙara wannan code a sama na code snippet don samar da sunan mu plugin ɗinka.
