# Action: wu_dashboard_{$this->tab}_widgets {#action-wudashboardthis-tabwidgets}

Inaruhusu watengenezaji wa plugin kuongeza widgets kwenye Panel ya Dashboard ya Mtandao.

## Viparametrika {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $tab | `string` | Tab ya sasa. |
| $screen | `\WP_Screen` | Kifaa cha skrini (screen object). |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Toleo la ukurasa wa admin wa Ultimate Multisite. |
