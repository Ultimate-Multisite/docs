# Action: wu_maybe_create_customer

Inaruhusu watengenezaji wa plugin kufanya vitu vingine zaidi wakati mteja anapoongezwa.

Hapa ndipo tunapoongeza 'hooks' za kuunganisha mteja (customer) na mtumiaji (user) kwenye tovuti kuu, kwa mfano.

## Viparametrika {#parameters}

| Jina | Aina | Maelezo |
|------|------|-------------|
| $customer | `\Customer` | Mteja ambaye angeweza kuundwa. |
| $checkout | `\Checkout` | Darasa la ukaguzi (checkout) lililopo. |
