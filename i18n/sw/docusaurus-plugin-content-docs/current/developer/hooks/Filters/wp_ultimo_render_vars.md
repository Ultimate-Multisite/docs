# Filter: wp_ultimo_render_vars {#filter-wpultimorendervars}

Inaruhusu watengenezaji wa plugin kuongeza vigezo (variables) vya ziada katika muktadha wa kuonyesha (render context) kwa kiwango cha jumla.

## Vigezo (Parameters) {#parameters}

| Jina | Aina | Maelezo |
|------|------|-------------|
| $args | `array` | Array inayohifadhi vigezo vilivyopitishwa na wito wa kuonyesha (render call). |
| $view | `string` | Jina la view inayotakiwa kuonyeshwa. |
| $default_view | `string` | Jina la view ya dharura (fallback_view). |

### Tangu {#since}

- 2.0.0
### Chanzo {#source}

Imefafanuliwa katika [`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L31) kwenye mstari wa 31.
