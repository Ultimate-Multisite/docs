# Filter: wu_get_checkout_variables

Inaruhusu watengenezaji wa plugin kufiltera mipangilio ya awali ya ukosefu wa malipo wa ukurasa.

Tahadhari: Kukosa vifungo (keys) kunaweza kuvunja kabisa ukosefu wa malipo kwenye sehemu ya mbele (front-end).

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $variables | `array` | Vigezo vilivyotambulishwa. |
| $checkout | `\Checkout` | Darasa la ukosefu wa malipo. |
