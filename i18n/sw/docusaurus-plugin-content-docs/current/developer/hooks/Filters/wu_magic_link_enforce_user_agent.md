# Filter: wu_magic_link_enforce_user_agent {#filter-wumagiclinkenforceuseragent}

Filter ambayo huamua kama utahitaji kulazimisha uthibitisho wa user agent.

Weka kuwa `false` ili kuruhusu tokens kufanya kazi kwenye browsers/devices tofauti. Hii inapunguza usalama lakini huongeza uwezo wa kutumia (usability).

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $enforce | `bool` | Je, ni lazima kutambua user agent. |
