# Filter: wu_wp_config_reference_hook_line_patterns

Tunachanganya mifumo mitatu (patterns) tunapojaribu kujua tunapoweza kuingiza *constants* zetu:

1. Tunatafuta ufafanuzi wa variable ya `$table_prefix`; 2. Tunatafuta ufafanuzi tata zaidi wa `$table_prefix` — zile zinazotumia *env variables*, kwa mfano; 3. Ikiwa hilo halipo, tunatafuta maoni ya 'Happy Publishing'; 4. Ikiwa hata hilo halipo, tunatafuta mwanzo wa faili. *Key* inawakilisha *pattern*, na *value* inawakilisha idadi ya mistari ya kuongeza. Inawezekana kupitisha idadi hasi ya mistari ili kuandika kabla ya mstari uliopatikana, badala ya kuandika baada yake.
