# Filter: wu_magic_link_enforce_ip

Inawezekana kuamua kama unataka kuthibitisha anwani ya IP.

Ukiweka thamani kuwa `false`, utaruhusu tokens kufanya kazi kutoka kwenye mitandao tofauti. Hii inapunguza usalama lakini huongeza uwezo wa kutumia (kwa mfano, kwa watumiaji wa simu wanaobadilisha mitandao).

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $enforce | `bool` | Kama unataka kuthibitisha anwani ya IP. |

- 2.0.0
