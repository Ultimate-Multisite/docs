# Filter: wu_independent_billing_cycle_product_types

Hii inafiltera aina za bidhaa (product types) ambazo zina mizunguko ya malipo huru (independent billing cycles).

Bidhaa zenye mizunguko ya malipo huru hazilazimishwi kufanana na kipindi cha malipo cha mpango (plan) katika sehemu ya kukariri (cart). Hii ni muhimu kwa bidhaa kama vile usajili wa nyanja (domain registrations) ambazo huwezi kuweka malipo ya kila mwaka bila kujali kama mpango ni wa kila mwezi au wa kila mwaka.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $types | `array` | Array ya aina za bidhaa zenye mizunguko ya malipo huru. |

### Since

- 2.5.0
