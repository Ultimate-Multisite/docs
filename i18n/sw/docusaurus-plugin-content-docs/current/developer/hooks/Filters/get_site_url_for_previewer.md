# Filter: get_site_url_for_previewer

Inaruhusu watengenezaji wa plugin kufilta (filter) URL inayotumika katika muonyeshaji (previewer).

## Vigezo (Parameters)

| Jina (Name) | Aina (Type) | Maelezo (Description) |
|------|------|-------------|
| $domain | `string` | Nyanja (domain) ya msingi inayotumika kwa sasa, ni muhimu kwa kufanya mabadiliko. |
| $domain_options | `array` | Orodha ya chaguo zote za nyanja zilizowekwa katika Ultimate Multisite Settings -> Network Settings -> Domain Options. |

### Tangu (Since)

- 1.7.2
