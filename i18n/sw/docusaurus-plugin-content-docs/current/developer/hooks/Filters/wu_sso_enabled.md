# Filter: wu_sso_enabled

Hii inaruhusu kuwasha au kuzima uwezo wa *single-sign-on* (SSO) wa kuvuka nyanja (cross-domain).

Tumia thamani hii kufuta kabisa *single-sign-on*, au kuuwezesha kwa masharti badala yake.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $enabled | `bool` | Je, SSO inapaswa kuwashwa? `True` kwa kuwasha, na `false` kwa kuzima. |

### Since {#since}

- 2.0.11
### Source {#source}

Imefafanuliwa katika [`inc/sso/class-sso.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/sso/class-sso.php#L110) kwenye mstari wa 110


## Returns {#returns}
Inaweza kurudisha kama SSO imewashwa au la.
