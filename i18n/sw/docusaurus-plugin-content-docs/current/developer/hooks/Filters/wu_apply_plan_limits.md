# Filter: wu_apply_plan_limits

Inaruhusu watengenezaji wa plugins kupita au kubatilisha mipaka (limitations).

Unaweza kutumia filter hii kuendesha code yoyote unayotaka kabla ya mipaka yoyote kuanza kutumika. Ikiwa filter inarudisha thamani yoyote inayotambulika kuwa kweli (truthy value), mchakato utaendelea. Ikiwa inarudisha thamani inayotambulika kuwa si kweli (falsy value), code itasimama na hakuna kutoka kwenye hooks zote zilizopo hapa chini zitazotekelezwa.
