# Filter: wu_admin_notices {#filter-wuadminnotices}

Inaruhusu watengenezaji kuchuja taarifa za admin ambazo zimeongezwa na Ultimate Multisite.

## Viparametrika {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $notices | `array` | Orodha ya taarifa kwa sehemu (panel) hiyo maalum. |
| $all_notices | `array` | Orodha ya taarifa zilizoongezwa, zimegawanywa kulingana na sehemu (panel). |
| $panel | `string` | Sehemu (panel) inayohitajika taarifa hizo. |
| $filter | `string` | Ikiwa taarifa zinazoweza kufutwa zimechujuliwa. |
| $dismissed_messages | `array` | Orodha ya funguo (keys) za taarifa zilizofutwa. |
