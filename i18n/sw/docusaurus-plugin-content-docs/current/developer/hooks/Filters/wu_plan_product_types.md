# Filter: wu_plan_product_types

Hii filter inafunga aina za bidhaa ambazo zinapaswa kutibiwa kama mipango (plans).

Filter hii inaruhusu *addons* (zana za ziada) kujisajili na kuongeza aina za bidhaa ambazo zinapaswa kutambuliwa kama mipango katika uthibitisho na kutenganisha data.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | Array ya aina za bidhaa ambazo zinapaswa kutibiwa kama mipango. |

### Since

- 2.3.0
