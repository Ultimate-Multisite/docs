# Filter: wu_post_count

Inaruhusu watengenezaji wa plugin kubadilisha jumla ya idadi ya chapisho.

## Vigezo (Parameters) {#parameters}

| Jina | Aina | Maelezo |
|------|------|-------------|
| $count | `int` | Jumla ya idadi ya chapisho. |
| $post_counts | `object` | Kichwa cha (object) cha WordPress kinachorejeshwa na kazi ya `wp_count_posts`. |
| $post_type | `string` | *Slug* ya aina ya chapisho. |
