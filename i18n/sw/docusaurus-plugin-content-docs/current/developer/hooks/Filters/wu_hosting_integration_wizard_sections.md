# Filter: wu_hosting_integration_wizard_sections

## Vigezo (Parameters) {#parameters}

| Name | Type | Maelezo |
|------|------|-------------|
| $sections | `array` | Sehemu za mwongozo. |
| $integration | `\WP_Ultimo\Integrations\Host_Providers\Base_Host_Provider\|\WP_Ultimo\Integrations\Integration` | Upatanisho unaokuwa unarekebishwa. |
| $page | `\Hosting_Integration_Wizard_Admin_Page` | Tohari ya sehemu ya mwongozo. |
