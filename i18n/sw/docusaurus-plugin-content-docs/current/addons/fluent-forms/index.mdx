# Fluent Forms {#fluent-forms}

Toleo hili la ziada (add-on) linakuwezesha kuweka mipaka (limits) kwa Fluent Forms kwenye Multisite Ultimate, iwe ni kwenye bidhaa, ushuru, au tovuti.
Fluent Forms hutumia meza maalum (custom tables) badala ya aina za chapisho maalum (custom post types), kwa hivyo chaguo za kawaida za kuweka mipaka hazifanyi kazi na Fluent Forms.

## Vipengele {#features}

- Weka mipaka kwa Fluent Forms au Conversational Fluent Forms
- Wateja wanapofika mipaka, wanapokea ujumbe wa kirafiki uliowahamasisha kuboresha mpango wao (plan).
