| Jina | Aina | Maelezo |

| $wu_line_item | `\WP_Ultimo\Checkout\Line_Item` | Instance ya line item ya Ultimate Multisite. |
| $wc_product | `\WC_Product` | Bidhaa (product) ya WooCommerce. |
