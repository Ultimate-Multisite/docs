# Filter: signup_get_available_languages {#filter-signupgetavailablelanguages}

Huchuja orodha ya lugha zinazopatikana kwa ajili ya usajili wa tovuti kutoka kwa mtumiaji (front-end).

Kupitisha *array* tupu kwenye *hook* hii kutafuta kuondoa kuonyeshwa kwa mipangilio kwenye fomu ya usajili, na badala yake, lugha ya kiwango (default language) itatumika wakati wa kuunda tovuti. Lugha ambazo hazipo tayari zitatolewa.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $available_languages | `array` | Lugha zinazopatikana. |

### Since {#since}

- 4.4.0
