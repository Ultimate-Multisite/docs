# Admin Page Creator {#admin-page-creator}

## Maudhui {#content}
Admin page creator itakuruhusu kuongeza kurasa mpya za admin na sub-pages ndani ya WordPress admin dashboard kwa urahisi. Inaweza kutumika kuunda ukurasa wa karibisho au ukurasa wa mafunzo ambao unaweza kutumika kama ukurasa wa kuingia kwa watumiaji wapya. Unaweza kutumia page builder kama Beaver Builder, Elementor, Brizy, na Oxygen kutengeneza kurasa bila shida.

Pia inatoa uwezo wa kuongeza HTML, CSS, na JavaScript maalum, pamoja na udhibiti wa ufikiaji kulingana na mpango (plan) na jukumu la mtumiaji (user role)!

## Vipengele {#features}
* Inasaidia HTML na CSS Maalum
* Usaidizi wa JavaScript
* Hali ya Maudhui ya PHP
* Ruhusa kulingana na Jukumu la Mtumiaji
* Inasaidia page builders

## Usakinishaji {#installation}

1. Upload **ultimate-multisite-admin-page-creator** kwenye saraka ya **/wp-content/plugins/**
2. Washa plugin kupitia menyu ya **Plugins** katika WordPress Network Admin yako

## Mwongozo wa Matumizi {#usage-guide}

Ingawa WordPress inafanya iwe rahisi sana kuongeza maudhui kwenye sehemu ya mbele (front end) ya tovuti, kuunda kurasa na sub-pages upande wa admin kunahitaji ujuzi wa msingi wa kuandika code. Ultimate Multisite: Admin Page Creator inalenga kuleta urahisi huo wa matumizi wa sehemu ya mbele hadi upande wa nyuma (back end) wa tovuti zako za network.

## Kwa nini kutumia Admin Page Creator? {#why-use-admin-page-creator}

Labda unataka kutoa ukurasa maalum kwenye panel ya admin ya wateja wako wenye mafunzo, labda unataka kuongeza ukurasa maalum wenye maswali ya mara kwa mara (FAQs) na widget ya mawasiliano. Uwezekano ni mwingi na Admin Page Creator, na ndio tutachopitia katika mafunzo ya leo.

## Kuunda Admin Page yako ya Kwanza Maalum {#creating-your-first-custom-admin-page}

Baada ya kusakinisha na kuwasha Ultimate Multisite: Admin Page Creator, nenda kwenye menyu yako ya network admin na ubonyeze Admin Pages.

![Custom Admin Pages list on Network Admin](/img/admin/admin-pages-list.png)

_Orodha ya Custom Admin Pages kwenye Network Admin_

Utaona orodha ya admin pages zote ulizozitengeneza hadi sasa (kwa kuwa hii ni ziara yako ya kwanza, orodha itakuwa tupu). Endelea na kubonyeza **Add new Admin Page**. Utahukariwa kwenye editor ya Admin Page.

## Chaguo za Menyu {#menu-options}

Kando, kwenye meta-box ya General Options, utaweza kuweka mipangilio ya aina ya menyu, jina, na mengi zaidi.

### Menu Types (Aina za Menyu) {#menu-types}

Admin page yako mpya ya custom inaweza kuwa na aina mbili tofauti za menyu: menyu kuu (top-level menu) na sub-menu.

Ukurasa wa menyu kuu, kama jina linavyopendekeza, utaongeza ukurasa wako mpya kwenye menyu ya admin pamoja na kurasa za kawaida za WordPress.

Ikiwa utachagua chaguo la kiwango cha juu (top-level), utaweza kuchagua mpangilio wa menyu na ikoni ya menyu pia.

Ikiwa unataka kufanya kurasa hii kuwa sub-menu page, utaweza kuchagua ukurasa mkuu (parent page) kuunganisha kurasa hii nayo. Unaweza kuongeza ukurasa wako maalum kwenye kurasa za kawaida za WordPress au kwenye admin pages zako za custom za kiwango cha juu.

## Chaguo za Advanced (Za Juu) {#advanced-options}

Unapaswa kuwa na uwezo wa kuongeza aina zote za maudhui kwenye custom admin pages zako. Hii inajumuisha CSS rules na files, JavaScript, na vilevile libraries za nje. Unaweza kufanya hivyo kwa kutumia Advanced Options Tab.

### Custom CSS {#custom-css}

Kwenye tab ya CSS, unaweza kuongeza code yako ya CSS maalum, pamoja na kuingiza (import) CSS files kutoka kwa sites za mbali (kama CDNs, Google Fonts, n.k.).

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_CSS maalum pia inasaudiwa. Unaweza pia kuingiza faili za mtindo (style files) za nje!_

### Custom JavaScript {#custom-javascript}

Kile kile kinatumika kwa JavaScript. Unaweza kuongeza libraries za nje kutumia kwenye code yako ya JavaScript maalum.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Ongeza code yako ya JavaScript!_

## Ruhusa (Permissions) {#permissions}

Pia kwenye Advanced Options, utaweza kuchagua sub-sites zipi zitakuwa na ufikiaji kwenye kurasa hii. Hii ni muhimu ikiwa unalenga kutoa maudhui tofauti ya kipekee kwa viwango tofauti vya mpango, kwa mfano.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Una udhibiti kamili wa mahali pa kurasa hii kuonekana_

**Muhimu** : Masharti yote mawili lazima yatimizwe ili kurasa ionekane. Hii inamaanisha kwamba ikiwa mtumiaji ni mteja wa mpango fulani, lakini hana moja ya majukumu yaliyochaguliwa (au kinyume chake), kurasa haitawapo kwenye panel yake.

## Active (Imehudhurika) {#active}

Ikiwa unataka kuzima custom admin page bila kuifuta au kuingilia mipangilio ya ruhusa, toa tu tiki kwenye chaguo la Active.

Kuzima kurasa ya kiwango cha juu (top-level page) kutazima kiotomatiki sub-pages zote zilizounganishwa na kurasa kuu hiyo.
