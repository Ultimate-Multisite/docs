# Filter: wu_connector_enforcement_option_keys

Badilisha orodha ya funguo za chaguo za AI provider zinazotawaliwa kutoka kwa tovuti kuu.

Ongeza funguo za chaguo kwa plugins za AI provider za kubuni au za watu wengine ili mipangilio yao ipelekwe (au kurithiwa) kutoka tovuti kuu kwenye subsites.

## Vigezo {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $keys | `string[]` | Majina ya funguo za chaguo zinazotawaliwa (yanaojumuisha funguo za kuunganisha zilizogunduliwa kiotomatiki na EXTRA_PROVIDER_OPTIONS). |
