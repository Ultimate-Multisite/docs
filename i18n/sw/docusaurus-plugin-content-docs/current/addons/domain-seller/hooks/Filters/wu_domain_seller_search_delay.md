# Filter: wu_domain_seller_search_delay {#filter-wudomainsellersearchdelay}

Hii inafiltera muda wa kuchelewesha utafutaji wa nyanja (domain search debounce delay) kwa milisekunde.

Ongeza thamani hii ili kupunguza wito wa API (API calls) kwenye muunganisho polepole.

## Vigezo {#parameters}

| Jina | Aina | Maelezo |
|------|------|-------------|
| $delay | `int` | Muda wa kuchelewesha utafutaji kwa milisekunde. Kiwango cha awali ni 500. |
