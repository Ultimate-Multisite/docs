# AI Site Builder

ultimate-multisite-ai-site-builder

## Mahitaji (Requirements) {#requirements}

- WordPress 5.3 au zaidi
- PHP 7.4 au zaidi
- Plugin ya Ultimate Multisite (iwe imefunguliwa/active)

## Usakinishaji (Installation) {#installation}

1. Pakia faili za addon kwenye saraka yako ya `/wp-content/plugins/`
2. Fungulia plugin kupitia menyu ya 'Plugins' ndani ya WordPress
3. Sanifisha mipangilio ya addon katika sehemu ya kiadmini ya network (network admin)

## Uendelezaji (Development) {#development}

### Kuandaa (Setup) {#setup}

```bash
# Install dependencies
composer install
npm install

# Run tests
npm run test

# Run code standards checks
vendor/bin/phpcs
vendor/bin/phpstan

# Fix code style issues
vendor/bin/phpcbf

# Build for production
npm run build
```

### Muundo wa Mradi (Project Structure) {#project-structure}

```
ultimate-multisite-ai-site-builder/
├── inc/                          # PHP classes na sehemu za kuunganisha (includes)

│   ├── bootstrap.php            # Bootstrap ya vipimo

│   └── class-*-test.php         # Mifano ya vipimo (Test cases)

├── assets/                      # Rasilimali zisizobadilika (Static assets)
│   ├── css/                     # Stylesheets

├── lang/                        # Faili za lugha
```

### Vipimo (Testing) {#testing}

Addon inajumuisha mfumo kamili wa kufanya vipimo:

- **Unit Tests**: Vipimo vinavyotegemea PHPUnit na seti ya vipimo ya WordPress
- **Base Test Class**: Inatoa njia za msaada kwa ajili ya matukio ya kawaida ya vipimo
- **Test Coverage**: Imewekwa kufuatilia ufuatiliaji wa msimbo
- **Multisite Testing**: Vipimo vinatendeka katika mazingira ya multisite

### Viwango vya Msimbo (Code Standards) {#code-standards}

- **WordPress Coding Standards**: Vinatunzwa kupitia PHPCS
- **PHP 7.4+ Compatibility**: Vipengele vya kisasa vya PHP vinasaudiwa
- **Static Analysis**: Uunganishaji wa PHPStan kwa ajili ya kuangalia aina za data (type checking)
- **Code Modernization**: Kanuni za Rector kwa ajili ya kuboresha PHP

### Mfumo wa Kuunda (Build System) {#build-system}

- **Asset Processing**: Kupunguza ukubwa (Minification) kwa faili za CSS/JS
- **Translation**: Kuunda faili za POT
- **Packaging**: Kuunda archive kwa ajili ya usambazaji
- **Development Mode**: Rasilimali zisizopunguzwa ukubwa kwa ajili ya kutatua matatizo (debugging)

## Kuweka Mipangilio (Configuration) {#configuration}

Addon inasaidia chaguo mbalimbali za kuweka mipangilio ambazo zinaweza kufikiwa kupitia kiadmini cha network.

## Hooks na Filters {#hooks-and-filters}

### Actions (Vitendo) {#actions}

- `ultimate-multisite-ai-site-builder_init` - Inatokea baada ya kuanzishwa kwa addon
- `ultimate-multisite-ai-site-builder_loaded` - Inatokea baada ya faili zote za addon kupakiwa

### Filters (Vichujio) {#filters}

- `ultimate-multisite-ai-site-builder_settings` - Kubadilisha mipangilio ya addon
- `ultimate-multisite-ai-site-builder_enabled` - Kubatilisha hali ya kuwasha/kizima kwa addon
