# Action: switch_network

Inatokea wakati mazingira ya mtandao (network context) yanabadilishwa.

## Viparametrika {#parameters}

| Jina | Aina | Maelezo |
|------|------|-------------|
| $new_network_id | `int` | Kitambulisho (ID) cha mtandao unaoenda kuunganishwa nalo. |
| $old_network_id | `int` | Kitambulisho (ID) cha mtandao uliokuwa uliopo awali. |
