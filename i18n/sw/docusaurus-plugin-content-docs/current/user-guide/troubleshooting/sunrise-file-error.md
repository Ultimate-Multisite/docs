# Hitilafu ya Kusakinisha Faili ya Sunrise

Faili ya sunrise.php ni faili maalum ambayo WordPress inaitafuta wakati inapojiandaa kuanza kufanya kazi. Ili WordPress iweze kugundua faili ya sunrise.php, inahitaji kuwekwa ndani ya **folda ya wp-content**.

Unapoamilisha Ultimate Multisite na kupitia mchawi wa usanidi kama unavyoonekana kwenye picha ya skrini, Ultimate Multisite inajaribu kunakili faili yetu ya sunrise.php kwenye folda ya wp-content.

<!-- Screenshot unavailable: Setup wizard page showing sunrise.php installation step -->

Mara nyingi, tunaweza kunakili faili kwa mafanikio na kila kitu kinafanya kazi vizuri. Hata hivyo, ikiwa kuna jambo ambalo halijasanidiwa vizuri (ruhusa za folda, kwa mfano), unaweza kukutana na hali ambapo Ultimate Multisite haiwezi kunakili faili.

Ukisoma ujumbe wa hitilafu ambao Ultimo inakupa, utaona ndiyo hasa kilichotokea hapa: **Kunakili kwa Sunrise kumeshindikana**.

<!-- Screenshot unavailable: Error message showing Sunrise copy failed -->

Ili kurekebisha hilo, unaweza kunakili faili ya sunrise.php iliyoko ndani ya folda ya plugin ya wp-ultimo na kuibandika kwenye folda yako ya wp-content. Baada ya kufanya hivyo, pakia upya ukurasa wa mchawi na ukaguzi unapaswa kupita.

<!-- Screenshot unavailable: File manager showing sunrise.php inside wp-ultimo plugin folder --> Kwa vyovyote vile, hii inaweza kuhitaji ukaguzi wa jumla wa ruhusa za folda zako ili kuepuka matatizo siku zijazo (si tu na Ultimate Multisite bali pia na plugins na themes nyingine).

**Zana ya Health Check** ambayo ni sehemu ya WordPress (unaweza kuifikia kupitia **paneli ya msimamizi ya tovuti yako kuu > Tools > Health Check**) inaweza kukujulisha ikiwa ruhusa za folda zako zimewekwa kwa thamani zinazoweza kusababisha matatizo na WordPress.
