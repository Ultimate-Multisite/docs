# Matatizo yanayojulikana na Plugins nyingine

Baada ya kusakinisha na kuwezesha Ultimate Multisite, unaweza kukutana na matatizo kutokana na kutopatana na plugins nyingine ambavyo havijasuluishiwa.

Tunafanya kazi kwa bidii kupunguza orodha hii kwa kila toleo jipya, lakini ikiwa plugin imeorodheshwa hapa, unaweza kuhitaji kuizima ili kuwa na Mtandao wa Ultimate Multisite unaofanya kazi kikamilifu.

Hapa kuna orodha yao na unachopaswa kufanya katika hali hizi:

**Baadhi ya plugins za WPMU DEV** Baadhi ya plugins ambazo _haziendelezwi tena_ na WPMU DEV zinajulikana kusababisha mchakato wa usajili wa Ultimate Multisite kusimama hatua ya mwisho (mtumiaji hawezi kuunda akaunti). Hapa kuna orodha ya plugins zinazosababisha matatizo hayo:

  * Signup Language: _Ultimate Multisite ina add-on inayotoa huduma sawa, inayoitwa Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Hii imejengwa ndani ya Ultimate Multisite, kwa hivyo hakuna haja ya plugin ya ziada._

  * New Blog Templates: _Hii imejengwa ndani ya Ultimate Multisite, kwa hivyo hakuna haja ya plugin ya ziada._

  * Set Password on Multisite Blog Creation: _Hii imejengwa ndani ya Ultimate Multisite, kwa hivyo hakuna haja ya plugin ya ziada._

  * Signup TOS: _Hii imejengwa ndani ya Ultimate Multisite, kwa hivyo hakuna haja ya plugin ya ziada._

  * Anti-Splog: Ingawa si yenye nguvu sana, hii pia _imejengwa ndani ya Ultimate Multisite._

**Kumbuka** : Plugins nyingine zote za WPMU Dev zinaweza kutumika kawaida pamoja na Ultimate Multisite. Mifano ni pamoja na _Smush_ , _Forminator_ , _Defender,_ n.k.
