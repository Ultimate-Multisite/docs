# Utafutaji wa Mtandao

Gratis AI Agent v1.5.0 ina uwezo wa **Utafutaji wa Mtandao** ambao unaruhusu msaidizi wa AI kupata habari za kisasa kutoka kwenye mtandao wakati wa mazungumzo. Uwezo huu unatokana na [Brave Search API](https://brave.com/search/api/) au [Tavily API](https://tavily.com/).

## Inafanya Kazi Vipi

Wakati utafutaji wa mtandao unapowekwa, msaidizi anaweza kuuliza kiotomatiki chanzo chako cha utafutaji kilichowekwa (configured search provider) anapogundua kwamba swali linahitaji habari za sasa au za nje — kwa mfano, habari za hivi karibuni, bei halisi, au hati za kazi ambazo zingeweza kubadilika tangu muda wa kufundishwa kwa mfumo huo.

Matokeo hupatikana kwa wakati halisi na kuingizwa kwenye muktadha wa msaidizi kabla ya kutoa jibu. Msaidizi anataja wakati amefanya matumizi ya matokeo ya utafutaji kujibu swali.

## Kuwezesha Utafutaji wa Mtandao

Utafutaji wa mtandao unahitaji API key kutoka kwa chanzo chako cha utafutaji ulichochagua. Ili kuweka mipangilio:

1. Nenda kwa **Gratis AI Agent → Settings → Advanced**.
2. Tafuta orodha ya **Internet Search Provider** na uchague kati ya **Brave Search** au **Tavily**.
3. Ingiza API key yako kwenye sehemu inayolingana. Viungo vya kujisajili (Sign-up links) vinaonyeshwa kando ya kila sehemu.
4. Bonyeza **Save Settings**.

Mara tu key imepigwa hifadhi, uwezo wa Utafutaji wa Mtandao unakuwa upatao kwa msaidizi kiotomatiki.

## Brave Search

### Kupata Brave Search API Key

1. Tembelea [Brave Search API page](https://brave.com/search/api/).
2. Jisajili kwa mpango. Kuna kiwango cha bure kinachopatikana na kikomo cha ombi cha kila mwezi.
3. Nakili API key yako kutoka kwa dashibodi ya watengenezaji wa Brave Search.
4. Bandika kwenye sehemu ya **Brave Search API Key** katika mipangilio ya Gratis AI Agent.

### Vikomo vya Matumizi

Matumizi yanapangwa na Brave Search kulingana na idadi ya maswali yaliyofanywa. Kila jibu la AI linalochochea utafutaji linahesabiwa kama swali moja. Fuatilia matumizi yako kwenye [Brave Search developer dashboard](https://brave.com/search/api/) ili kuepuka malipo yasiyotarajiwa.

## Tavily

Superdav AI Agent v1.10.0 ina **Tavily** kama chanzo kingine cha utafutaji wa mtandao, ikitoa matokeo ya utafutaji yanayoboreshwa na uwezo wa utafiti wa hali ya juu.

### Kupata Tavily API Key

1. Tembelea [Tavily API page](https://tavily.com/).
2. Jisajili kwa akaunti. Kuna kiwango cha bure kinachopatikana na kikomo cha ombi cha kila mwezi.
3. Nakili API key yako kutoka kwa dashibodi ya Tavily.
4. Bandika kwenye sehemu ya **Tavily API Key** katika mipangilio ya Gratis AI Agent.

### Vikomo vya Matumizi

Matumizi yanapangwa na Tavily kulingana na idadi ya API calls zilizofanywa. Kila jibu la AI linalochochea utafutaji linahesabiwa kama wito mmoja (one call). Fuatilia matumizi yako kwenye [Tavily dashboard](https://tavily.com/) ili kuepuka malipo yasiyotarajiwa.

## Kuzima Utafutaji wa Mtandao

Ondoa API key kutoka kwa sehemu ya chanzo cha utafutaji kilicho hai na hifadhi. Uwezo wa Utafutaji wa Mtandao hautawasilishwa kwa msaidizi tena.

:::note
Utafutaji wa mtandao unaongeza muda wa majibu kwa sababu msaidizi lazima asubiri matokeo ya utafutaji kabla ya kutoa jibu. Kwa matumizi yanayohitaji haraka, zingatia kama utafutaji wa wakati halisi ni muhimu au kama maarifa yaliyojengwa ndani ya msaidizi yanatosha.
:::
