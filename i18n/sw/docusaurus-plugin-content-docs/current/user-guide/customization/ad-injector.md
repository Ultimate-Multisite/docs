# Kuweka Matangazo kwenye Mtandao wako kwa kutumia Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector ni **nyongeza ya bure** inayokupa zana za kuweka matangazo kwenye mipango yako moja kwa moja kutoka kwenye ukurasa wake wa kuhariri.

Baada ya kusakinisha Ultimate Multisite: Ad Injector, nenda kwenye ukurasa wako wa _Plugins_ ili kuiwezesha.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Bofya kitufe cha Network Activate_

Mara nyongeza inapokuwa hai, kisanduku cha _Advanced Options_ kwenye skrini ya kuhariri mpango kitakuwa na chaguo jipya linaloitwa _Ad Settings_.

![Ad Settings kwenye skrini ya kuhariri mpango](/img/config/product-edit-full.png)

_Sasa unaweza kuhariri mipangilio ya matangazo kwa kila mpango wako_

Kwenye kichupo hicho, utaweza kuongeza misimbo ya matangazo kwa machapisho na kurasa upande wa mbele, kabla na baada ya maudhui.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_Baada ya kuingiza msimbo wa tangazo lako, litaonekana kwenye machapisho na kurasa za mpango huo mahususi._

Unaweza pia kuweka matangazo kwenye sehemu ya juu ya Kurasa za Admin kwa watumiaji wa mpango huo.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Nenda kwenye chaguo la Admin Panel Ad Code ikiwa unataka kuonyesha matangazo kwenye sehemu ya nyuma ya mtumiaji wako_

### Zaidi ya Matangazo

Unaweza pia kutumia Ultimate Multisite: Ad Injector kutoa msaada kwa mipango ya ngazi ya juu.

Nenda tu kwenye chaguo la _Admin Panel Ads_ na uongeze msimbo wa HTML kwa zana yako ya msaada.

_Ongeza msaada wa wateja kwa mipango mahususi_

Kwa njia hii watu wanaweza kuwasiliana nawe moja kwa moja kutoka kwenye Admin Panel yao!
