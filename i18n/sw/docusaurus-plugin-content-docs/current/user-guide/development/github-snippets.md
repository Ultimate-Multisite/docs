# Jinsi ya kutumia vipande vya msimbo vya Ultimate Multisite kutoka kwenye hazina yetu ya GitHub

Kuna vipande vya msimbo vilivyopo kwenye hazina ya GitHub ambavyo mara nyingi vinaombwa na watumiaji wa Ultimate Multisite wanaotaka kuongeza utendaji mdogo kama kuweka script ya Google Analytics kwenye kurasa za usajili au kuficha kisanduku cha meta kutoka kwenye dashibodi ya msimamizi.

Makala hii itakuonyesha jinsi ya kutumia au hasa mahali pa kuweka msimbo huu.

Unaweza kupata vipande vya msimbo kwenye kiungo hapa chini.

https://github.com/next-press/wp-ultimo-snippets/

Kuna njia 2 za kuongeza msimbo

  1. Kwenye faili ya functions.php ya mandhari yako.

  2. Must-Use Plugins (mu-plugins)

# Jinsi ya kuongeza kipande cha msimbo kwenye faili ya functions.php ya mandhari yako.

  1. Ingia kwenye dashibodi ya msimamizi wa WordPress Network na nenda Themes > Theme Editor (Angalia picha ya skrini hapa chini).

  2. Kwenye ukurasa wa "Edit Themes", hakikisha umechagua mandhari yako inayotumika kwenye sehemu ya kuchagua iliyoko upande wa juu kulia wa skrini yako (#3 kwenye picha ya skrini hapa chini).

  3. Bofya faili ya functions.php chini ya sehemu ya "Theme Files" ili kupakia faili hiyo. Sogea chini hadi mwisho na ubandike kipande cha msimbo cha Ultimate Multisite ulichokipata kutoka kwenye hazina ya GitHub.

# Jinsi ya kuunda Must-Use Plugins (mu-plugins)

WordPress ina kipengele kinachokuruhusu kupakia utendaji maalum unaoitwa "Must-Use Plugins", au "mu-plugins" kwa kifupi.

Hizi mu-plugins maalum zinapakiwa kabla ya plugins nyingine zote za kawaida, na haziwezi kuzimwa. Katika mtandao wa multisite, msimbo ulio ndani ya mu-plugins hizi utapakiwa kwenye tovuti zote zilizo kwenye usakinishaji wako.

1\. Tumia FTP au SSH kufikia mfumo wa faili wa usakinishaji wako wa WordPress.

2\. Ndani ya saraka ya wp-content ya usakinishaji wako wa WordPress, unda saraka mpya iitwayo: mu-plugins.

<!-- Screenshot unavailable: File manager showing wp-content directory with mu-plugins folder -->

3\. Unda faili mpya ya PHP kwenye kompyuta yako iitwayo wu-snippet.php ukitumia Notepad au programu yoyote ya kuhariri msimbo.

4\. Weka kipande cha msimbo cha Ultimate Multisite ulichokipata kutoka kwenye hazina ya GitHub ndani ya faili na uihifadhi. Unaweza pia kuongeza msimbo huu juu ya kipande cha msimbo ili kuweka lebo kwenye mu plugin yako.
