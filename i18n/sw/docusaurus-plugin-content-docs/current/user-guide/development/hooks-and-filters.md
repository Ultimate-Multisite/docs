# Huku na Vichungi vya Ultimate Multisite

Ultimate Multisite imeundwa na kazi nyingi na hoja za hifadhidata. Ina huku na vichungi ambavyo vitakuruhusu kurekebisha au kuongeza msimbo mpya kwenye kazi zilizopo.

Tumeunda orodha ya huku na vichungi hizi kama rejea kusaidia watumiaji wa hali ya juu kubinafsisha zaidi mtandao wao. Unaweza kufikia orodha kutoka Github kwa kubofya kiungo kilicho hapa chini.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
