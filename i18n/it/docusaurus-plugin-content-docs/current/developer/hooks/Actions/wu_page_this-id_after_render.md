# Azione: wu_page_{$this->id}_after_render

Consenti agli sviluppatori di plugin di aggiungere contenuti aggiuntivi dopo aver stampato la pagina

## Parameters

| Nome | Tipo | Descrizione |
|------|------|-------------|
| $page_id | `string` | L'id di questa pagina |
| $page | `object` | L'oggetto pagina. |

### Da

- 1.8.2
