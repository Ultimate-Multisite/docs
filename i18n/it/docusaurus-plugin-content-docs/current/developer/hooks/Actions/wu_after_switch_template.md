# Azione: wu_after_switch_template

Consenti agli sviluppatori di plugin di agganciare funzioni dopo che un utente o un super amministratore cambia il modello del sito

## Parametri

| Nome | Tipo | Descrizione |
|------|------|-------------|
| $id | `int` | ID del sito |

### Da

- 1.9.8
