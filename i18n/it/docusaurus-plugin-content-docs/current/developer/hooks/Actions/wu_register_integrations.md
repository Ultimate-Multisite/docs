# Action: wu_register_integrations

Scatena l'abilitazione delle integrazioni a registrarsi.

Il core registra le sue integrazioni qui.

## Parametri

| Name | Type | Description |
|------|------|-------------|
| $registry | `\Integration_Registry` | L'istanza del registro. |

- 2.5.0
