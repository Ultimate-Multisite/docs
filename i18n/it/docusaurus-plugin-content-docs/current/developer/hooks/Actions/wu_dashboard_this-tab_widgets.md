# Azione: wu_dashboard_${this->tab}_widgets

Consenti agli sviluppatori di plugin di aggiungere widget al pannello di Dashboard di Rete.

## Parametri

| Nome | Tipo | Descrizione |
|------|------|-------------|
| $tab | `string` | La scheda corrente. |
| $screen | `\WP_Screen` | L'oggetto dello schermo. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Istanza della pagina di amministrazione Ultimate Multisite. |

- 2.0.0
