# Action: wu_{$this->model}_post_delete

Si attiva dopo che un oggetto è stato salvato nel database.

## Parametri

| Nome | Tipo | Descrizione |
|------|------|-------------|
| $result | `bool` | Vero se l'oggetto è stato eliminato con successo. |
| $this | `\Base_Model` | L'istanza dell'oggetto. |
