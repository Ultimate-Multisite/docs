# Azione: wu_page_after_render

Consenti agli sviluppatori di plugin di aggiungere contenuto aggiuntivo dopo che abbiamo stampato la pagina

## Parametri

| Nome | Tipo | Descrizione |
|------|------|-------------|
| $page_id | `string` | L'id di questa pagina |
| $page | `object` | L'oggetto pagina. |
