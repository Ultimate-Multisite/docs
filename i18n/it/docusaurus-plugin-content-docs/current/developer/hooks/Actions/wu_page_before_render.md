# Azione: wu_page_before_render

Consente agli sviluppatori di plugin di aggiungere contenuti aggiuntivi prima di stampare la pagina.

## Parametri

| Nome | Tipo | Descrizione |
|------|------|-------------|
| $page_id | `string` | L'id di questa pagina. |
| $page | `object` | L'oggetto pagina. |
