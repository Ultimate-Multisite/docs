# Filtro: wu_magic_link_enforce_user_agent

Filtra se è necessario applicare la verifica dell'user agent.

Impostare su false per consentire ai token di funzionare su diversi browser/dispositivi. Ciò riduce la sicurezza ma aumenta l'usabilità.

## Parameters

| Name | Type | Descrizione |
|------|------|-------------|
| $enforce | `bool` | Se è necessario applicare la corrispondenza dell'user agent. |

- 2.0.0
