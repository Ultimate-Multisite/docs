# Filtro: wu_admin_notices

Consenti agli sviluppatori di filtrare le notifiche di amministrazione aggiunte da Ultimate Multisite.

## Parametri

| Nome | Tipo | Descrizione |
|------|------|-------------|
| $notices | `array` | Elenco delle notifiche per quel pannello particolare. |
| $all_notices | `array` | Elenco delle notifiche aggiunte, separate per pannello. |
| $panel | `string` | Pannello da cui recuperare le notifiche. |
| $filter | `string` | Se le notifiche annullabili sono state filtrate. |
| $dismissed_messages | `array` | Elenco delle chiavi delle notifiche annullate. |

### Da

- 2.0.0

### Fonte

Definito in [`inc/class-admin-notices.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-admin-notices.php#L121) alla riga 121
