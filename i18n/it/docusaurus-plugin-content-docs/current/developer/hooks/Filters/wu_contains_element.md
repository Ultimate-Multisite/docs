# Filter: wu_contains_element

Consente agli sviluppatori di modificare i risultati della ricerca iniziale.

Questo è utile per i costruttori di terze parti e simili.

## Parameters

| Nome | Tipo | Descrizione |
|------|------|-------------|
| $contains_elements | `bool` | Se l'elemento è contenuto nel contenuto. |
| $content | `string` | Il contenuto in esame. |
| $element | `self` | L'elemento corrente. |
| $post | `null|\WP_Post` | post da verificare. |
