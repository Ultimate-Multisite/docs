# Filtro: wu_plan_product_types

Filtra i tipi di prodotto che dovrebbero essere trattati come piani.

Questo filtro consente agli addon di registrare tipi di prodotto aggiuntivi che dovrebbero essere riconosciuti come piani nella validazione e nella segregazione.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | Array di tipi di prodotto da trattare come piani. |
