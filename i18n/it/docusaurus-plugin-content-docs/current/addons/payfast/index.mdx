# PayFast

Multisite Ultimate: Payfast Gateway

Aggiunge Payfast - un gateway di pagamenti online veloce e sicuro in Sudafrica - come opzione di pagamento per Multisite Ultimate.

## Installation

1. Carica 'ultimate-multisite-payfast' nella directory '/wp-content/plugins/'
2. Attiva il plugin tramite il menu 'Plugins' in WordPress
3. Segui la procedura guidata passo passo per configurare il plugin

## Usage Guide

PayFast è un principale processore di pagamenti online per le aziende in Sudafrica. È dedicato ad aiutare i commercianti online a crescere rendendo le transazioni fluide e sicure. Su richiesta degli utenti di Ultimate Multisite, abbiamo deciso di creare un addon che renda possibile l'integrazione con PayFast. In questo articolo mostriamo come configurare PayFast come gateway di pagamento per ricevere pagamenti nella tua rete Ultimate Multisite.

Il primo passo è installare l'addon.

# Come installare l'addon Payfast:

Vai alla pagina degli addon e seleziona l'addon Ultimate Multisite: Payfast Gateway.

![Add-ons page](/img/admin/addons-page.png)

Si aprirà la finestra dell'addon. Fai clic su Installa ora.

![Add-ons page with install option](/img/admin/addons-page.png)

Dopo l'installazione, verrai reindirizzato alla pagina dei plugin. Qui, fai semplicemente clic su Network Activate e l'addon _Payfast_ verrà attivato sulla tua rete.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PayFast addon -->

# Configurare PayFast

Dopo aver installato l'addon, puoi ora andare nelle impostazioni di Ultimate Multisite e iniziare a configurare PayFast.

Sulla pagina **Ultimate Multisite settings**, puoi trovare la scheda 'Payments'. In questa scheda, dovresti trovare '**Payfast**' insieme a un pulsante di interruttore per abilitare o disabilitare questo gateway di pagamento.

![Payment gateway settings](/img/config/settings-payment-gateways.png)Go ahead and click the toggle button to enable it. It will then expand the box to show different fields that you need to populate such as **Mechant ID, Merchant Key and PassPhrase.** You can get this information from your **Payfast** account

<!-- Screenshot unavailable: PayFast expanded settings showing Merchant ID, Merchant Key, and PassPhrase fields -->

Il passo successivo è accedere al tuo account PayFast per ottenere le informazioni necessarie.

Una volta effettuato l'accesso, puoi vedere immediatamente le credenziali di cui hai bisogno dal cruscotto del tuo account **Payfast**.

<!-- Screenshot unavailable: PayFast account dashboard showing Merchant ID and Merchant Key credentials -->You can simply copy and paste the credential and populate the details on your Ultimate Multisite settings. Then click the **"Save settings** " button to save the changes.

<!-- Screenshot unavailable: Ultimate Multisite PayFast settings with credentials populated and Save button -->

Ecco fatto, la tua integrazione di pagamento PayFast è completa!
