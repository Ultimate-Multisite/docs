# Fluent Forms

Questo add-on consente di aggiungere limiti a Fluent Forms su Multisite Ultimate in prodotti, abbonamenti o siti.  
Fluent Forms utilizza tabelle personalizzate invece di tipi di post personalizzati, quindi le opzioni di limitazione normali non funzionano con Fluent Forms.
