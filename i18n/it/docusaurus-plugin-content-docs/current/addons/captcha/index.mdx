# Captcha

**Stop Bots. Protect Revenue. Zero Configuration Hassle.**

Running a multisite network with custom domains? You know the pain: Google reCAPTCHA and hCaptcha require you to register *every single domain* in their admin console. Add a new customer site? Update the captcha config. Map a custom domain? Update again. It never ends.

**Ultimate Multisite: Captcha changes everything.**

## Cap Captcha: Self-Hosted Protection That Just Works

Our new **Cap Captcha** provider is a game-changer for multisite networks:

* **Zero Configuration** — Attiva l'addon e sei protetto. Nessuna chiave API, nessun account Google, nessun cruscotto hCaptcha, nessuna whitelist di domini.
* **Works on Every Domain Automatically** — Sottodomini, domini mappati, domini dei clienti—tutti protetti in modo immediato.
* **Self-Hosted & Private** — I tuoi dati non lasciano mai il tuo server. Completamente conforme al GDPR senza tracciamento di terze parti.
* **Invisible Proof-of-Work** — Gli utenti risolvono un puzzle computazionale in background. Nessun clic sui rubinetti antincendio, nessuna frustrazione con griglie di immagini.
* **Lightweight** — Solo 20KB. Non rallenterà il tuo checkout.

## WooCommerce Card Testing Protection

Gli attacchi di test delle carte costano ai commercianti miliardi ogni anno. I truffatori usano bot per validare carte di credito rubate contro il tuo checkout WooCommerce—tu paghi le commissioni di elaborazione, loro ottengono numeri di carte validi.

**Proteggiamo dove conta di più:**

* **Store API Protection** — Blocca gli attacchi bot su `/wp-json/wc/store/v1/checkout` che bypassano i tradizionali captcha dei moduli
* **PayPal Payments Compatible** — Funziona senza soluzione di continuità con WooCommerce PayPal Payments
* **Real-Time Statistics** — Vedi esattamente quanti attacchi stai bloccando

## Installation

1. Carica `ultimate-multisite-captcha` nella tua directory `/wp-content/plugins/`
2. Attiva il plugin a livello di rete
3. **Ecco fatto. Sei protetto.**

Cap Captcha si attiva automaticamente—nessuna impostazione da configurare, nessuna chiave API da inserire. Ogni endpoint supportato in tutta la tua rete è protetto al momento dell'attivazione.

**Opzionale:** Per usare Google reCAPTCHA o hCaptcha invece, vai su **Ultimate Multisite → Settings → Captcha** e seleziona il provider preferito.

## Frequently Asked Questions

## Devo configurare qualcosa?

No! Cap Captcha è abilitato automaticamente quando attivi l'addon. L'intera rete è protetta immediatamente—nessuna impostazione da configurare, nessuna chiave API da inserire.

## Ho bisogno di chiavi API per Cap Captcha?

No. Cap Captcha è completamente self-hosted. Nessun account esterno, nessuna registrazione di dominio, nessuna fatica di configurazione.

## Questo rallenterà il mio checkout?

Cap Captcha è solo 20KB e esegue sfide proof-of-work in background. I tuoi clienti non noteranno alcun ritardo.

## Questo protegge contro gli attacchi di test delle carte?

Sì! A differenza dei tradizionali captcha che proteggono solo i moduli HTML, la nostra integrazione WooCommerce si collega direttamente alla Store API per bloccare gli attacchi bot a livello API—dove avvengono la maggior parte degli attacchi di test delle carte.

## Posso usare provider captcha diversi su siti diversi?

Il provider captcha è impostato a livello di rete, ma puoi regolare le impostazioni di difficoltà in base alle tue esigenze.

## È conforme al GDPR?

Cap Captcha è 100% self-hosted senza inviare dati a servizi esterni. Per Google reCAPTCHA e hCaptcha, consulta le rispettive politiche sulla privacy.
