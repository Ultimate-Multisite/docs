# Site Exporter

## Contenuto

Comprendiamo la necessità di poter spostare i sottositi all'interno della tua rete verso un'installazione a singolo sito e viceversa, qualora il tuo cliente decidesse di migrare il proprio sito web. Il Site exporter and importer addon rende questo compito più semplice. Questo strumento esporta e importa siti utilizzando file zip che possono anche servire come backup.

## Funzionalità

* Esporta e importa siti

* Interfaccia facile da usare

* Crea backup del sito

## Documentazione

**Come esportare e importare un sito**

## Installazione

1. Carica **ultimate-multisite-site-exporter** nella directory **/wp-content/plugins/**

2. Attiva il plugin tramite il menu **Plugins** in WordPress

3. Segui la procedura guidata passo passo per configurare il plugin

## Guida all'uso

Con Ultimate Multisite puoi esportare un sito dalla tua rete o importare un sito nella tua installazione multisite o single-site.

Per avviare il processo, vai alla nostra pagina add-on e installa l'add-on Ultimate Multisite: Site Exporter & Importer.!

To start the process, go to our add-ons page and install the Ultimate Multisite: Site Exporter & Importer add-on.![Add-ons page](/img/admin/addons-page.png)

Vai alla tua pagina Plugins e fai clic su **Network Activate** l'add-on che hai appena installato.!

L'opzione **Export & Import** si trova nel menu Ultimate Multisite, sul lato sinistro.

## Come esportare un sito

Per esportare un sito dalla tua rete, vai su **Export & Import** nel menu Ultimate Multisite e, nella scheda **Export Sites**, fai clic su **Generate new Site Export**.

<!-- Screenshot unavailable: Ultimate Multisite menu showing Export and Import option -->

Cerca il sito che desideri esportare e seleziona gli elementi da includere nel file esportato. Puoi includere temi, plugin e caricamenti. Se il tuo sito è troppo grande, puoi scegliere di eseguire il processo di esportazione in background.

<!-- Screenshot unavailable: Export Sites tab with Generate new Site Export button -->

Fai clic sul pulsante **Export Site**. Il tuo sito esportato sarà disponibile nella lista **Existing Exports**. Fai clic sul pulsante **Download ZIP** per scaricare il file.

Search for the site you want to export and select the items you want to include in the exported file. You can include themes, plugins, and uploads. If your site is too big, you can choose to run the export process in the background.<!-- Screenshot unavailable: Site export options with site search, include themes/plugins/uploads checkboxes -->

## Come importare un sito

Per importare un sito, vai alla scheda **Import a Site** e carica il file .zip del sito che desideri importare. Puoi modificare l'URL in uno nuovo o mantenere l'URL vecchio lasciando lo spazio vuoto. Inoltre, puoi selezionare la casella per eliminare il file .zip dalla tua installazione dopo l'importazione. Quindi fai clic su **Begin Import**.

Click the **Export Site** button. Your exported site will be available on the **Existing Exports** list. Click the **Download ZIP** button to download the file.<!-- Screenshot unavailable: Existing Exports list with Download ZIP button -->

Il processo può richiedere un po' di tempo. Vedrai un'importazione in sospeso nella lista sotto l'importatore.

<!-- Screenshot unavailable: Import a Site tab with zip upload field and Begin Import button -->

Puoi interrompere il processo facendo clic su **Cancel Import**.

<!-- Screenshot unavailable: Pending import in the list below the importer -->

Dopo che il processo è completo, il tuo sito è già disponibile sulla tua rete.

## Come importare un sito su un'installazione singola

Per importare un sito nella tua installazione singola, prima installa e attiva l'importatore sul tuo sito. Puoi trovare il link per scaricarlo nella scheda exporter.

<!-- Screenshot unavailable: Cancel Import button to stop the import process -->

Puoi trovare l'importatore nel menu **Tools**, sul lato sinistro, nella voce secondaria **Export & Import**.

<!-- Screenshot unavailable: Exporter tab with link to download single-site importer -->

Carica il file .zip del sito che desideri importare. Puoi modificare l'URL in uno nuovo o mantenere l'URL vecchio lasciando lo spazio vuoto. Inoltre, puoi selezionare la casella per eliminare il file .zip dalla tua installazione dopo l'importazione. Quindi fai clic su **Begin Import**.

<!-- Screenshot unavailable: Single-site Tools menu showing Export and Import submenu -->

Il processo può richiedere un po' di tempo. Vedrai un'importazione in sospeso nella lista sotto l'importatore.

<!-- Screenshot unavailable: Single-site import form with zip upload and Begin Import button -->

Puoi interrompere il processo facendo clic su **Cancel Import**.

<!-- Screenshot unavailable: Single-site pending import in the list -->

Dopo che il processo è terminato, il tuo sito sarà sostituito dal sito che hai appena importato.
