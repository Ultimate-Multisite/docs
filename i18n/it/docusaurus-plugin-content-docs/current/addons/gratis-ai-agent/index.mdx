# Gratis AI Agent

Gratis AI Agent è un assistente AI autonomo per WordPress che può pianificare, costruire e gestire interi siti WordPress tramite istruzioni in linguaggio naturale. Espone un set strutturato di **abilità** (abilities) — azioni discrete e testabili che l'agente può invocare — che coprono tutto, dalla gestione dei contenuti alla personalizzazione del sistema di design. La versione 1.9.0 aggiunge abilità di creazione di contenuti, revisione visiva delle pagine tramite screenshot lato client, cinque profili di agente integrati e feature flag per il controllo degli accessi e il branding.

## Funzionalità Principali {#key-features}

- **Sistema di abilità (Abilities system)** — un catalogo modulare ed estensibile di azioni che l'agente può eseguire sulla tua installazione WordPress
- **Gestione dei Contenuti (Content Management)** — crea post e pagine (inclusa l'assegnazione di template di pagina), crea in blocco più post con una singola chiamata, imposta immagini in evidenza e costruisce moduli di contatto
- **Revisione Visiva (Visual Review)** — cattura screenshot di pagine live, confronta stati prima/dopo e fornisce una revisione di design generata dall'AI che copre layout, tipografia, colore e accessibilità
- **Cinque agenti integrati** — Content Writer, Site Builder, Design Studio, Plugin Manager e Support Assistant; selezionabili tramite l'Agent Picker nell'intestazione della chat
- **Feature flags** — interruttori per il controllo degli accessi e il branding in Impostazioni → Feature Flags per restrizioni di ruolo, nomi white-label e personalizzazione dei widget
- **Gestione dei Custom Post Type** — registra, elenca e elimina custom post type con persistenza anche dopo riavvii
- **Gestione dei Custom Taxonomy** — registra, elenca ed elimina custom taxonomy con persistenza
- **Abilità del Design System** — iniettare CSS personalizzato, gestire pattern di blocchi curati, impostare il logo del sito e applicare preset theme.json
- **Gestione degli Stili Globali (Global Styles)** — leggere e scrivere valori theme.json per palette di colori, tipografia, spaziatura e altro
- **Gestione dei Menu di Navigazione (Navigation Menu management)** — creare e aggiornare menu di navigazione e i loro elementi programmaticamente
- **Gestione delle Opzioni (Options Management)** — leggere e scrivere le opzioni di WordPress con un blocco di sicurezza integrato per proteggere le impostazioni critiche
- **Registro Abilità Installabili (Installable Abilities Registry)** — scoprire e attivare pacchetti di abilità aggiuntivi senza modificare i file core del plugin
- **Site Builder Orchestration v2** — piani di costruzione del sito multi-step con scoperta di plugin, tracciamento dei progressi e recupero errori
- **Comando benchmark WP-CLI** — eseguire la suite di benchmark delle capacità dell'agente dalla riga di comando per CI e valutazione del modello

## Requisiti {#requirements}

- WordPress 6.0 o superiore
- PHP 8.1 o superiore
- WordPress Multisite (attivato a livello di network) o installazione single-site
- Un plugin connettore AI attivo (Anthropic, OpenAI o compatibile)

## Installazione {#installation}

1. Carica i file del plugin in `/wp-content/plugins/gratis-ai-agent`
2. Attiva il plugin dalla schermata **Plugin** di WordPress
3. Vai su **Gratis AI Agent → Impostazioni** e connetti il tuo provider AI
4. (Opzionale) Sfoglia il **Registro Abilità (Abilities Registry)** per installare pacchetti di abilità aggiuntivi

## Avvio Rapido {#quick-start}

Una volta attivato e connesso a un provider AI, apri il pannello **Gratis AI Agent** nella barra laterale di amministrazione di WordPress. Digita un'istruzione in linguaggio naturale e l'agente pianificherà ed eseguirà i passaggi necessari utilizzando le abilità disponibili.

**Esempi di istruzioni:**

```
Create a "Portfolio" custom post type with a "Project Category" taxonomy.
```

```
Inject a dark-mode colour palette into the theme and set the primary colour to #1a1a2e.
```

```
Build a restaurant website with an online menu, booking form, and contact page.
```

## Panoramica delle Abilità (Abilities Overview) {#abilities-overview}

Le abilità sono le azioni atomiche che l'agente può eseguire. Ogni abilità è una classe PHP registrata che espone uno schema (parametri e tipi di ritorno) che l'agente legge in tempo di esecuzione. Le abilità sono raggruppate in aree funzionali:

| Area | Abilità |
|---|---|
| **Gestione dei Contenuti** | `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form` |
| **Revisione Visiva** | `capture_screenshot`, `compare_screenshots`, `review_page_design` |
| **Custom Post Types** | `register_post_type`, `list_post_types`, `delete_post_type` |
| **Custom Taxonomies** | `register_taxonomy`, `list_taxonomies`, `delete_taxonomy` |
| **Design System** | `inject_custom_css`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `apply_theme_json_preset` |
| **Stili Globali** | `get_global_styles`, `set_global_styles`, `reset_global_styles` |
| **Menu di Navigazione** | `create_menu`, `update_menu`, `add_menu_item`, `remove_menu_item`, `list_menus` |
| **Gestione delle Opzioni** | `get_option`, `set_option`, `delete_option`, `list_options` |
| **Abilità Installabili** | `list_available_abilities`, `install_ability`, `recommend_plugin` |
| **Site Builder** | `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error` |

Consulta la [Riferimento Abilità (Abilities Reference)](./abilities) per la documentazione completa dei parametri e degli esempi di utilizzo.

## WP-CLI {#wp-cli}

Gratis AI Agent include una famiglia di comandi `wp gratis-ai-agent` per eseguire benchmark e gestire le abilità dalla riga di comando. Consulta la [Riferimento WP-CLI (WP-CLI Reference)](./wp-cli) per l'elenco completo dei comandi.

```bash
# Run the full benchmark suite
wp gratis-ai-agent benchmark

# List installed abilities
wp gratis-ai-agent abilities list

# Install an ability pack
wp gratis-ai-agent abilities install <slug>
```

## Configurazione {#configuration}

Tutte le impostazioni si trovano in **Gratis AI Agent → Impostazioni** nell'amministrazione di WordPress:

| Impostazione | Descrizione |
|---|---|
| **AI Provider** | Seleziona e configura il connettore provider AI attivo |
| **Model** | Scegli il modello linguistico da utilizzare per le sessioni dell'agente |
| **Max Tokens** | Budget di token per richiesta (predefinito 4096) |
| **Ability Timeout** | Tempo massimo in secondi che un'abilità singola può eseguire (predefinito 30) |
| **Debug Logging** | Registra le chiamate alle abilità e il ragionamento dell'agente nel log di debug di WordPress |

## Blocco di Sicurezza per la Gestione delle Opzioni {#options-management-safety-blocklist}

L'abilità di Gestione delle Opzioni include un blocco di sicurezza integrato che impedisce all'agente di leggere o scrivere opzioni sensibili di WordPress. Il blocco di sicurezza predefinito copre:

- `admin_email`, `blogadmin_email`
- `siteurl`, `home` (opzioni URL — usa uno strumento di migrazione dedicato per queste)
- `auth_key`, `secure_auth_key`, `logged_in_key`, `nonce_key` (chiavi segrete)
- `active_plugins`, `template` (attivazione di plugin/tema — usa abilità dedicate)
- Credenziali e chiavi API dei gateway di pagamento registrati dai plugin installati

Vengono aggiunte voci aggiuntive tramite il filtro `gratis_ai_agent_options_blocklist`.

## Hook e Filtri {#hooks-and-filters}

### Actions {#actions}

- `gratis_ai_agent_ability_registered` — attivato dopo che un'abilità è stata registrata, riceve il nome della classe abilità
- `gratis_ai_agent_plan_started` — attivato quando inizia l'esecuzione di un piano Site Builder
- `gratis_ai_agent_plan_completed` — attivato quando un piano termina con successo
- `gratis_ai_agent_plan_failed` — attivato quando un piano incontra un errore irrecuperabile

### Filters {#filters}

- `gratis_ai_agent_abilities` — modifica l'elenco delle abilità registrate prima che l'agente lo carichi
- `gratis_ai_agent_options_blocklist` — estendi o riduci il blocco di sicurezza delle opzioni
- `gratis_ai_agent_benchmark_questions` — aggiungi domande di benchmark personalizzate alla suite di valutazione
- `gratis_ai_agent_plan_plugins` — modifica l'elenco della scoperta di plugin durante la generazione del piano del sito
