# Material WP {#material-wp}

L’interfaccia di amministrazione di WordPress è rimasta la stessa per anni. Ma ti sei mai chiesto come sarebbe una versione Material Design dell’interfaccia di amministrazione? 

Material WP è un theme di amministrazione sviluppato per darti quell’aspetto e quella sensazione, aiutandoti a offrire ai tuoi clienti un Dashboard più personalizzato.

## Funzionalità {#features}

- Supporto CSS personalizzato
- Impostazioni di stile facili
- Riordinamento del menu
- Importazione/esportazione facile

## Documentazione {#documentation}

**Personalizzare il tuo pannello di amministrazione con Material WP**

## Installazione {#installation}

1. Carica 'material-wp' nella directory '/wp-content/plugins/'
2. Attiva il plugin tramite il menu 'Plugin' in WordPress

## Guida all’uso {#usage-guide}

L’interfaccia di amministrazione di WordPress è rimasta la stessa per anni. Ma ti sei mai chiesto come sarebbe una versione Material Design dell’interfaccia di amministrazione?

Material WP è un theme di amministrazione sviluppato per darti quell’aspetto e quella sensazione, aiutandoti a offrire ai tuoi clienti un Dashboard più personalizzato. Material WP fa anche parte del core dei plugin premium di Ultimate Multisite.

Dopo averlo **installato**, vai alla tua pagina _Plugin_ e fai clic per attivarlo sulla tua Network.
