# Material WP

L’interfaccia di amministrazione di WordPress è rimasta la stessa per anni. Ma ti sei mai chiesto come sarebbe una versione Material Design dell’interfaccia di amministrazione?

Material WP è un tema di amministrazione sviluppato per darti quell’aspetto e quella sensazione, aiutandoti a fornire ai tuoi clienti un Dashboard più personalizzato.

## Features

- Custom CSS support
- Easy styling settings
- Menu reordering
- Easy import/export

## Documentation

**Customizing your Admin Panel with Material WP**

## Installation

1. Upload 'material-wp' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

## Usage Guide

L’interfaccia di amministrazione di WordPress è rimasta la stessa per anni. Ma ti sei mai chiesto come sarebbe una versione Material Design dell’interfaccia di amministrazione?

After **installing it**, go to your _Plugins_ page and click to activate it on your Network.
