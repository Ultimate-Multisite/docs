# Registro delle modifiche degli Agenti di supporto

Versione: 1.0.13 - Rilasciata il 2025-09-28

* Rinomina prefisso in ultimate-multisite; aggiorna dominio di testo; incremento versione.

Versione 1.0.12 - Rilasciata il 2025-09-17

* Corretto: compatibilità con Ultimate Multisite 2.4.4

Versione 1.0.11 - Rilasciata il 2023-11-21

* Corretto: compatibilità con Ultimate Multisite 2.3.0

Versione 1.0.10 - Rilasciata il 2023-08-30

* Corretto: rimozione di Bulk Support Agents

Versione 1.0.9 - Rilasciata il 2023-08-09

* Aggiunto: aggiunta della capacità di leggere i moduli di checkout.
* Corretto: layout del modulo di autorizzazione degli Agenti di supporto;
* Corretto: messaggio di avviso sul modulo di cancellazione dell'agente di supporto;
* Corretto: widget della dashboard di rete nascosti per gli utenti non autorizzati.
* Corretto: corrette capacità di modifica e cancellazione degli Agenti di supporto.
* Corretto: processo di creazione degli Agenti di supporto corretto.
* Corretto: impedito l'aggiunta di un cliente come agente di supporto.
* Corretto: reindirizzati gli utenti non autorizzati alla pagina di amministrazione di modifica dell'agente di supporto.
* Corretto: migliorato il layout del modulo di autorizzazione degli Agenti di supporto.
* Interno: implementato un nuovo processo di build.

Versione 1.0.8 - Rilasciata il 2022-01-21

* Interno: aggiunto generatore di hook e filtri;
* Interno: aggiunti stubs di Ultimate Multisite per la qualità della vita dello sviluppatore;
* Aggiunto: capacità del modulo di checkout;
* Corretto: disallineamento del nome della capacità tra i moduli di Ultimate Multisite e le opzioni di questo add-on;

Versione 1.0.7 - Rilasciata il 2021-12-02

* Corretto: eliminazione degli Agenti di supporto non funziona;
* Corretto: eliminazione a blocchi degli Agenti di supporto non funziona;

Versione 1.0.6 - Rilasciata il 2021-11-22

* Corretto: aggiunta di nuovo modal non funziona dopo le ottimizzazioni delle prestazioni sul core;
* Corretto: vista grid-item presente sul core e non sul plugin;

Versione 1.0.5 - Rilasciata il 2021-10-31

* Corretto: piccole incompatibilità con PHP 8;
* Migliorato: wp_ultimo_skip_network_active_check ora utilizza wu_is_must_use();

Versione 1.0.4 - 2021-09-24

* Aggiunto: filtro wp_ultimo_skip_network_active_check per configurazioni basate su mu-plugins;

Versione 1.0.3 - Rilasciata il 2021-08-29

* Aggiunto: supporto per REST API - gli Agenti di supporto possono essere creati tramite REST;

Versione 1.0.2 - Rilasciata il 2021-07-26

* Corretto: aggiornamento delle autorizzazioni degli agenti non funziona;

Versione 1.0.1 - Rilasciata il 2021-05-28

* Aggiunto: avviso quando l'utente è già un super amministratore;
* Aggiunto: autorizzazione per consentire il cambio utente;
* Migliorato: campi del modulo migliori;

Versione 1.0.0 - Rilascio Iniziale
