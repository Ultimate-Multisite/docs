# Support Agents

Gestire un'intera rete da soli non è un compito facile. Man mano che la tua rete cresce, noterai una crescente richiesta di supporto. Sarà una buona idea coinvolgere persone che ti aiutino a alleggerire il carico di lavoro. Tuttavia, concedere pieno accesso/permessi è rischioso e può compromettere la tua rete. È qui che entra in gioco questo add-on Support Agent.

Questo add-on ti consente di creare un nuovo tipo di utente WordPress chiamato Support Agent, e ti dà il controllo totale sui permessi che quell'utente ha. Concedere permessi è semplice come attivare un pulsante di interruzione e disattivarlo se decidi di rimuoverlo.

## Funzionalità

- Elenco puntato delle autorizzazioni WordPress & Ultimate Multisite
- Abilita e disabilita l'accesso con facilità
- Crea un numero illimitato di account Agent

## Installazione

1. Carica `ultimate-multisite-support-agents` nella directory `/wp-content/plugins/`
2. Attiva il plugin tramite il menu `Plugins` in WordPress
3. Segui la procedura guidata passo passo per configurare il plugin
