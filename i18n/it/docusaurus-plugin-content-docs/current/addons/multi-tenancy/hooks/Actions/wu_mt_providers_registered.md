# Action: wu_mt_providers_registered

Si attiva dopo che i provider predefiniti sono stati registrati.

Usa questo hook per registrare provider personalizzati.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $manager | `\Provider_Manager` | L'istanza del provider manager. |
