# Admin Page Creator

## Contenuto {#content}
Admin Page Creator ti permetterà di aggiungere facilmente nuove pagine di amministrazione e sottopagine all'interno della Dashboard di amministrazione WordPress. Può essere usato per creare una pagina di benvenuto o una pagina tutorial che può fungere da pagina di destinazione per i nuovi utenti. Puoi usare builder di pagine come Beaver Builder, Elementor, Brizy e Oxygen per progettare le pagine senza difficoltà.

Offre anche supporto per HTML, CSS e JavaScript personalizzati, oltre al controllo degli accessi basato su piano e ruolo utente!

## Funzionalità {#features}
* Supporto per HTML e CSS personalizzati
* Supporto JavaScript
* Modalità contenuto PHP
* Autorizzazione basata sul ruolo utente
* Supporto per builder di pagine

## Documentazione {#documentation}
[Creare pagine di amministrazione e sottopagine personalizzate usando Multisite Ultimate: Admin Page Creator](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## Installazione {#installation}

1. Carica **ultimate-multisite-admin-page-creator** nella directory **/wp-content/plugins/**
2. Attiva il plugin tramite il menu **Plugin** nel tuo WordPress Network Admin

## Guida all'uso {#usage-guide}

Mentre WordPress rende davvero facile aggiungere contenuti al front end dei siti, creare pagine e sottopagine dal lato amministrazione richiede alcune competenze di codifica di base. Ultimate Multisite: Admin Page Creator mira a portare la stessa facilità d'uso del front end al back end dei siti della tua rete.

## Perché usare Admin Page Creator? {#why-use-admin-page-creator}

Forse vuoi offrire una pagina personalizzata nel pannello di amministrazione dei tuoi clienti con tutorial, forse vuoi aggiungere una pagina personalizzata con FAQ e un widget di contatto. Le possibilità sono infinite con Admin Page Creator, ed è questo che esploreremo nel tutorial di oggi.

## Creare la tua prima pagina di amministrazione personalizzata {#creating-your-first-custom-admin-page}

Dopo aver installato e attivato Ultimate Multisite: Admin Page Creator, vai al menu di amministrazione della tua rete e vai su Admin Pages.

![Elenco delle pagine di amministrazione personalizzate nel Network Admin](/img/admin/admin-pages-list.png)

_Elenco delle pagine di amministrazione personalizzate nel Network Admin_

Vedrai un elenco di tutte le pagine di amministrazione che hai creato finora (dato che questa è la tua prima visita, l'elenco sarà vuoto). Procedi e fai clic su **Add new Admin Page**. Verrai reindirizzato all'editor della pagina di amministrazione.

Nell'editor della pagina di amministrazione, avrai diverse opzioni per personalizzare la tua pagina.

## Opzioni del menu {#menu-options}

Sul lato, nel meta-box Opzioni generali, potrai configurare il tipo di menu, l'etichetta e molto altro.

### Tipi di menu {#menu-types}

La tua nuova pagina di amministrazione personalizzata può avere due diversi tipi di menu: menu di primo livello e sottomenu.

Una pagina di menu di primo livello, come suggerisce il nome, aggiungerà la pagina appena creata al menu di amministrazione insieme alle pagine predefinite di WordPress.

Se selezioni l'opzione di primo livello, potrai selezionare anche l'ordine del menu e un'icona del menu.

Se vuoi rendere questa pagina una pagina di sottomenu, potrai selezionare una pagina genitore a cui collegare questa pagina. Puoi aggiungere la tua pagina personalizzata alle pagine predefinite di WordPress o alle tue pagine di amministrazione personalizzate di primo livello.

## Opzioni avanzate {#advanced-options}

Dovresti essere in grado di aggiungere ogni tipo di contenuto alle tue pagine di amministrazione personalizzate. Ciò include regole e file CSS, JavaScript e anche librerie esterne. Puoi farlo usando la scheda Opzioni avanzate.

### CSS personalizzato {#custom-css}

Nella scheda CSS, puoi aggiungere il tuo codice CSS personalizzato, oltre a importare file CSS da siti remoti (come CDN, Google Fonts, ecc.).

## Autorizzazioni {#permissions}

Sempre nelle Opzioni avanzate, potrai selezionare quali sottositi avranno accesso a questa pagina. Questo è utile se stai pianificando di offrire contenuti esclusivi diversi a diversi livelli di piano, ad esempio.

<!-- Screenshot non disponibile: impostazioni delle autorizzazioni di Admin Page Creator per il controllo degli accessi basato su piano e ruolo -->

_Hai il controllo totale su dove verrà visualizzata questa pagina_

**Importante** : entrambe le condizioni devono essere soddisfatte affinché la pagina venga visualizzata. Ciò significa che se l'utente è un cliente di un determinato piano, ma non ha uno dei ruoli selezionati (o viceversa), la pagina non apparirà nel suo pannello.

## Attiva {#active}

Se vuoi disabilitare una pagina admin personalizzata senza eliminarla o modificare i permessi, deseleziona semplicemente l'opzione Attiva.

La disattivazione di una pagina di primo livello disattiverà automaticamente tutte le sottopagine collegate a quella pagina padre.
