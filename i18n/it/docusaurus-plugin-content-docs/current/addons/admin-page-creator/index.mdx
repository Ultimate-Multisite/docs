# Admin Page Creator

## Contenuto
Admin page creator ti permetterà di aggiungere facilmente nuove pagine amministrative e sottopagine all'interno del pannello di amministrazione di WordPress. Può essere utilizzato per creare una pagina di benvenuto o una pagina tutorial che può fungere da pagina di atterraggio per i nuovi utenti. Puoi utilizzare page builder come Beaver Builder, Elementor, Brizy e Oxygen per progettare le pagine senza alcuna difficoltà.

Offre inoltre supporto per HTML, CSS e JavaScript personalizzati, oltre al controllo degli accessi basato sul piano e sul ruolo utente!

## Funzionalità
* Supporto per HTML e CSS personalizzati
* Supporto JavaScript
* Modalità contenuto PHP
* Permessi basati sul ruolo utente
* Supporto per page builder

## Documentazione
[Creare pagine amministrative personalizzate e sottopagine usando Multisite Ultimate: Admin Page Creator](https://help.multisiteultimate.com/article/305-creating-custom-admin-pages-and-sub-pages-using-ultimate-multisite-admin-page-creator)

## Installazione

1. Carica **ultimate-multisite-admin-page-creator** nella directory **/wp-content/plugins/**
2. Attiva il plugin tramite il menu **Plugins** nell'admin di rete di WordPress

## Guida all'uso

Mentre WordPress rende davvero semplice aggiungere contenuti alla parte front-end dei siti, creare pagine e sottopagine sul lato amministrativo richiede alcune competenze di base di programmazione. Ultimate Multisite: Admin Page Creator mira a portare la stessa facilità d'uso del front-end al back-end dei tuoi siti di rete.

## Perché usare Admin Page Creator?

Forse vuoi offrire una pagina personalizzata sul pannello amministrativo dei tuoi clienti con tutorial, oppure vuoi aggiungere una pagina personalizzata con FAQ e un widget di contatto. Le possibilità sono infinite con Admin Page Creator, e questo è ciò che esploreremo nel tutorial di oggi.

## Creare la tua prima pagina amministrativa personalizzata

Dopo aver installato e attivato Ultimate Multisite: Admin Page Creator, vai al menu di amministrazione di rete e accedi a Admin Pages.

![Custom Admin Pages list on Network Admin](/img/admin/admin-pages-list.png)

_Elenco delle pagine amministrative personalizzate nell'admin di rete_

Vedrai un elenco di tutte le pagine amministrative che hai creato finora (poiché è la tua prima visita, l'elenco sarà vuoto). Vai avanti e fai clic su **Add new Admin Page**. Sarai reindirizzato all'editor della pagina amministrativa.

## Gli elementi di una pagina amministrativa personalizzata

All'editor della pagina amministrativa avrai diverse opzioni per personalizzare la tua pagina.

Questo è il titolo che verrà visualizzato nella parte superiore della pagina, non l'etichetta del menu amministrativo.

## Opzioni del menu

Sul lato, nella meta-box Opzioni generali, potrai configurare il tipo di menu, l'etichetta e molto altro.

### Tipi di menu

La tua nuova pagina amministrativa personalizzata può avere due tipi di menu diversi: menu di primo livello e sottomenù.

Una pagina di menu di primo livello, come suggerisce il nome, aggiungerà la tua pagina appena creata al menu amministrativo accanto alle pagine predefinite di WordPress.

Se selezioni l'opzione di primo livello, potrai anche selezionare l'ordine del menu e un'icona del menu.

Se vuoi trasformare questa pagina in un sottomenù, potrai selezionare una pagina genitore a cui attaccare questa pagina. Puoi aggiungere la tua pagina personalizzata alle pagine predefinite di WordPress o alle tue pagine amministrative personalizzate di primo livello.

## Opzioni avanzate

Dovresti essere in grado di aggiungere tutti i tipi di contenuto alle tue pagine amministrative personalizzate. Ciò include regole e file CSS, JavaScript e librerie esterne. Puoi farlo utilizzando la scheda Opzioni avanzate.

### CSS personalizzato

Nella scheda CSS, puoi aggiungere il tuo codice CSS personalizzato, oltre a importare file CSS da siti remoti (come CDN, Google Fonts, ecc.).

<!-- Screenshot unavailable: Admin Page Creator custom CSS editor with external stylesheet import -->

_Il CSS personalizzato è supportato. Puoi anche includere file di stile esterni!_

### JavaScript personalizzato

Lo stesso vale per JavaScript. Puoi aggiungere librerie esterne da utilizzare nel tuo codice JavaScript personalizzato.

<!-- Screenshot unavailable: Admin Page Creator custom JavaScript editor with external library import -->

_Aggiungi il tuo codice JavaScript!_

## Permessi

Anche nelle Opzioni avanzate, potrai selezionare quali sottositi avranno accesso a questa pagina. È utile se prevedi di fornire contenuti esclusivi diversi a diversi livelli di piano, ad esempio.

<!-- Screenshot unavailable: Admin Page Creator permissions settings for plan and role-based access control -->

_Hai il pieno controllo su dove questa pagina verrà visualizzata_

**Importante** : Entrambe le condizioni devono essere soddisfatte affinché la pagina venga visualizzata. Ciò significa che se l'utente è cliente di un determinato piano, ma non possiede uno dei ruoli selezionati (o viceversa), la pagina non apparirà sul suo pannello.

## Attivo

Se vuoi disabilitare una pagina amministrativa personalizzata senza eliminarla o modificare le autorizzazioni, deseleziona semplicemente l'opzione Attivo.

Disattivare una pagina di primo livello disattiverà automaticamente tutte le sottopagine collegate a quella pagina genitore.
