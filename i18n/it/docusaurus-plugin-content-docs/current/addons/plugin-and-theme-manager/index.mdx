# Plugin & Theme Manager

Multisite Ultimate: Plugin and Theme Manager

Modifica o nascondi le informazioni meta (titolo, autore, miniatura e descrizione) dei plugin e temi installati. Crea categorie per plugin e temi per consentire ai tuoi utenti di filtrare e aggiungi una bella pagina Plugins personalizzata al pannello dei tuoi utenti.

Per usare il plugin, puoi semplicemente navigare alle pagine Plugin o Themes sul tuo amministratore di rete e fare clic su "Modifica informazioni Plugin/Theme".

## Installation

1. Carica 'ultimate-multisite-plugin-and-theme-manager' nella directory '/wp-content/plugins/'
2. Attiva il plugin tramite il menu 'Plugins' nell'Amministratore di rete WordPress

## Usage Guide

Ultimate Multisite: Plugin and Theme Manager è un potente add-on **premium** Ultimate Multisite che ti dà il pieno controllo sul modo in cui i tuoi utenti di rete vedono e interagiscono con le opzioni plugin e tema sulla tua rete.

Questo add-on serve tre scopi:

  1. Ti consente di modificare completamente i metadati di plugin e temi. Ciò include nome, descrizione, nome autore e persino miniature!

  2. Ti consente di categorizzare plugin e temi, per permettere ai tuoi utenti di ordinarli facilmente.

  3. Ti offre l'opzione di sostituire la vecchia, noiosa pagina plugin con una nuova pagina simile a temi con ordinamento per categorie.

Dopo averlo scaricato e installato, una nuova scheda - _Impostazioni Add-on_ \- verrà mostrata nella pagina Impostazioni Ultimate Multisite.
