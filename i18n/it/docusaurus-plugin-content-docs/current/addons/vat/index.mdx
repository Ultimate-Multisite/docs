## Installazione

1. Carica 'ultimate-multisite-vat' nella directory '/wp-content/plugins/'  
2. Attiva il plugin tramite il menu 'Plugins' in WordPress  
3. Vai su Ultimate Multisite > Impostazioni > Tasse e abilita l'IVA UE.  
4. Vai, fai clic su Aliquote fiscali e aggiorna le aliquote IVA UE in fondo e salva le aliquote fiscali.  
5. Fatto!
