# Action: switch_network

Si attiva quando il contesto di rete corrente viene cambiato.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $new_network_id | `int` | ID della rete a cui si sta passando. |
| $old_network_id | `int` | ID della rete precedentemente attiva. |
