# Chuck Norris Facts

Aggiungi un po' di umorismo alle tue attività di amministrazione quotidiana di WordPress con l'addon **Ultimate Multisite - Chuck Norris Facts**. Quando attivato, vedrai un fatto casuale di Chuck Norris visualizzato nell'angolo in alto a destra di ogni pagina di amministrazione. Perfetto per portare sorrisi al tuo team, alleggerire l'atmosfera durante lunghe sessioni di amministrazione o semplicemente celebrare la leggendaria grandiosità di Chuck Norris.

* **Rompi il ghiaccio**: Ottimo spunto di conversazione per i nuovi membri del team
* **Aumento della produttività**: Una risata veloce può migliorare la concentrazione e la produttività

## Installation

1. Assicurati che il plugin Ultimate Multisite sia installato e attivato sulla tua rete WordPress
2. Carica i file dell'addon nella directory `/wp-content/plugins/`
3. Attiva il plugin a livello di rete tramite il menu 'Plugins' in WordPress
4. Goditi fatti casuali di Chuck Norris su ogni pagina di amministrazione!

Nessuna configurazione necessaria - basta attivare e sorridere!

## Frequently Asked Questions

## Does this require Ultimate Multisite?

Sebbene sia stato progettato come addon di Ultimate Multisite, il plugin funziona in modo indipendente e può essere eseguito su qualsiasi installazione di WordPress.

## Can I add my own facts?

Sì! I fatti sono pronti per la traduzione. Puoi creare il tuo file `.po` con fatti personalizzati e posizionarlo nella directory `/lang`. Oppure puoi filtrare l'array dei fatti in modo programmatico.

## Does this slow down my admin?

Per niente! Il plugin è estremamente leggero e non influisce sulle prestazioni. Visualizza solo testo - nessuna chiamata API esterna o elaborazione pesante.

## Can I disable it on certain pages?

I fatti sono nascosti automaticamente sulle pagine dove potrebbero interferire (come l'editor a blocchi). Puoi anche utilizzare gli hook di WordPress per personalizzare dove appaiono i fatti.

## Are the facts appropriate for all audiences?

Sì! Tutti i fatti sono umorismo pulito e adatto alla famiglia che celebra lo status leggendario di Chuck Norris.

## Can I translate the facts to another language?

Assolutamente! Tutti i fatti utilizzano le funzioni di traduzione di WordPress. Crea un file di traduzione per la tua lingua e posizionalo nella directory `/lang`. Dominio di testo: `ultimate-multisite-chuck-norris-facts`

## Does it work with multisite?

Sì! È stato progettato specificamente per reti multisito e può essere attivato a livello di rete.
