# Registro delle modifiche del Domain Seller

Versione: 2.0.1 - Rilasciata il 2025-09-28

* Rinomina il prefisso in ultimate-multisite; aggiorna il dominio di testo; incremento di versione.



### Versione 2.0.0 - 2024-XX-XX

**Rivoluzione principale per Multisite Ultimate v2**

* **Nuovo:** Revisione completa con architettura moderna PHP 7.4+
* **Nuovo:** Integrazione senza soluzione di continuità con il sistema di checkout Multisite Ultimate v2
* **Nuovo:** Gestione dei prodotti di dominio con opzioni di prezzo flessibili
* **Nuovo:** Architettura di supporto per più fornitori di domini
* **Nuovo:** Integrazione di rinnovo automatico e abbonamento
* **Nuovo:** Interfaccia di gestione dei domini per i clienti
* **Nuovo:** Monitoraggio dei domini e registri per l'amministratore
* **Nuovo:** Supporto coupon per i prodotti di dominio
* **Nuovo:** Gestione completa delle impostazioni
* **Nuovo:** Base di codice estendibile e amichevole per gli sviluppatori
* **Migliorato:** Fornitore OpenSRS aggiornato con pieno supporto delle funzionalità
* **Migliorato:** UI moderna coerente con Multisite Ultimate v2
* **Corretto:** Tutto il codice v1 deprecato aggiornato agli standard v2
* **Rimosso:** Compatibilità legacy v1 (cambiamento che rompe)
