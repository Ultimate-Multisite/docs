# Conflitti noti con altri plugin

Dopo aver installato e attivato Ultimate Multisite, potresti riscontrare problemi a causa di incompatibilità non risolte con altri plugin.

Stiamo lavorando attivamente per ridurre questa lista con ogni nuova versione, ma se un plugin è elencato qui, potresti doverlo disattivare per avere una rete Ultimate Multisite pienamente funzionale.

Ecco un elenco di questi plugin e cosa dovresti fare in questi casi:

**Alcuni plugin WPMU DEV** Alcuni dei plugin _non più_ mantenuti da WPMU DEV sono noti per causare l'arresto del flusso di registrazione di Ultimate Multisite all'ultimo passaggio (l'utente non è in grado di creare un account). Ecco un elenco dei plugin che causano questi problemi:

  * Lingua di registrazione: _Ultimate Multisite ha un add-on che offre la stessa funzionalità, chiamato Ultimate Multisite: Language Switcher._
  * Imposta la descrizione del blog alla creazione del blog: _È già integrato in Ultimate Multisite, quindi non è necessario un plugin aggiuntivo._
  * Nuovi modelli di blog: _È già integrato in Ultimate Multisite, quindi non è necessario un plugin aggiuntivo._
  * Imposta la password alla creazione del blog multisito: _È già integrato in Ultimate Multisite, quindi non è necessario un plugin aggiuntivo._
  * Termini di servizio di registrazione: _È già integrato in Ultimate Multisite, quindi non è necessario un plugin aggiuntivo._
  * Anti-Splog: Anche se meno potente, è anch'esso _integrato in Ultimate Multisite._

**Nota** : Tutti gli altri plugin WPMU Dev possono essere usati normalmente insieme a Ultimate Multisite. Esempi includono _Smush_, _Forminator_, _Defender_, ecc.
