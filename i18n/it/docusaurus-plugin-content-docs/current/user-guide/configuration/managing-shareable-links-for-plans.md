# Gestire i link condivisibili per i piani

Ultimate Multisite ti offre la possibilità di condividere i link dei piani direttamente con i tuoi clienti. Per farlo, vai a una qualsiasi delle pagine di modifica dei tuoi piani e fai clic sul pulsante _Fai clic per copiare il link condivisibile_.

![Pagina di modifica del prodotto con pulsante per link condivisibile](/img/config/product-edit-full.png)

_Nella pagina di modifica del piano, puoi trovare il pulsante per creare link condivisibili_

Nota che questo meccanismo utilizza la frequenza di fatturazione predefinita per creare il link. Questo significa che se la tua fatturazione predefinita è mensile, ad esempio, alla fine avrai un link come questo: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Se desideri che il link condivisibile sia impostato di default su trimestrale, dovrai aggiungere 3 alla fine

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Per selezionare il piano _premium_ con una frequenza annuale, aggiungeresti 1 e poi la parola:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Tieni presente che non puoi avere un solo link per diverse frequenze di fatturazione. Dovresti condividere un link specifico per ciascuna delle tue frequenze di fatturazione.

Una volta condiviso il link con i tuoi clienti, li reindirizzerà al primo passaggio del tuo flusso di registrazione.
