# Personalizzare il tuo pannello di amministrazione con PRO Theme

Spesso, quando si lavora per clienti, si ha la necessità di offrire ai propri clienti un'esperienza personalizzata nella dashboard di amministrazione di WordPress. Se stai cercando un aspetto e una sensazione più professionali, PRO Theme potrebbe aiutarti.

PRO Theme è un tema amministrativo sviluppato per aiutarti quando personalizzi la Dashboard dei tuoi clienti. E fa anche parte del core dei plugin premium di Ultimate Multisite.

Dopo averlo installato, vai alla pagina _Plugins_ e fai clic per attivarlo sulla tua rete.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Attivalo a livello di rete nella tua pagina Plugins_

**NOTA:** Se lo installi solo, verrà mostrato ai tuoi clienti come un'opzione che può essere selezionata per aggiungere il tema solo a quell'utente.

Ora hai la tua interfaccia di amministrazione completamente ridisegnata! Hai anche un nuovo menu nella colonna sinistra chiamato PRO Theme.

<!-- Screenshot unavailable: PRO Theme redesigned admin panel interface -->

_Un aspetto completamente nuovo per il tuo pannello di amministrazione_
