# Personalizzare il tuo pannello Admin con PRO Theme

Spesso, quando lavoriamo per i clienti, abbiamo la necessità di offrire loro un’esperienza personalizzata nella Dashboard di WordPress. Se stai cercando un aspetto e una sensazione più professionali, PRO Theme potrebbe aiutarti.

PRO Theme è un tema admin sviluppato per aiutarti nella personalizzazione della Dashboard dei tuoi clienti. Ed è anche parte del core dei plugin premium di Ultimate Multisite.

Dopo averlo installato, vai alla tua pagina _Plugin_ e fai clic per attivarlo sulla tua rete.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Attivalo a livello di rete dalla tua pagina Plugin_

_**NOTA:**_ _Se lo installi soltanto, verrà mostrato ai tuoi clienti come un’opzione che può essere selezionata per aggiungere il tema solo a quell’utente._

Ora hai la tua interfaccia admin completamente ridisegnata! Hai anche un nuovo menu nella colonna sinistra chiamato PRO Theme.
