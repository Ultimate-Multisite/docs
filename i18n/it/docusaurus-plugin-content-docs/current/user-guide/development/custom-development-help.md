# Come ottenere aiuto per lo sviluppo personalizzato da Codeable?

A volte, abbiamo davvero bisogno di una personalizzazione fantastica per rendere i nostri flussi di rete esattamente come vogliamo. Se è questo ciò che stai cercando, ti consigliamo [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), un luogo dove puoi trovare sviluppatori esperti con competenza in Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) è una piattaforma che collega i clienti con sviluppatori freelance qualificati WordPress, in grado di lavorare su progetti di qualsiasi dimensione. Hanno un processo di candidatura meticoloso per garantire che tu sia abbinato a esperti specificamente qualificati per il compito che hai in mente.

### Come creare un progetto su Codeable?

Per prima cosa, vai su [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) e fai clic sul pulsante **Start a Project**.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Ora, per inviare un progetto, fai clic a sinistra su **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

In 'Ho bisogno di aiuto con', scegli **Customizing** :

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

In 'My', scegli **Plugin** :

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

Quindi fai clic sul pulsante **Continue**.

<!-- Screenshot unavailable: Codeable project form Continue button -->

Ora è il momento di descrivere il tuo progetto e le tue esigenze nel miglior modo possibile. Cerca di fornire quante più informazioni possibile!

<!-- Screenshot unavailable: Codeable project description form -->

Rispondi alle seguenti domande per aiutare nella determinazione del prezzo.

<!-- Screenshot unavailable: Codeable project pricing questions -->

Dopo di ciò, se non hai ancora un account [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), creane uno e fai clic sul pulsante **Publish My Project**.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

Una volta pubblicato il tuo progetto su [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE), la piattaforma ti collegherà con sviluppatori che corrispondono al set di competenze richiesto per il tuo progetto.

Questo processo aiuterà a trovare la persona giusta per il tuo lavoro, che abbia esperienza con WordPress e Ultimate Multisite!

Il tasso minimo di Codeable è di $70–$120 USD all'ora. Le stime cambieranno in base all'ambito del lavoro, alla complessità e all'urgenza del progetto.

Una volta ottenuta la tua stima, potrai assumere lo sviluppatore che meglio si adatta alle tue esigenze e avviare insieme il tuo progetto!
