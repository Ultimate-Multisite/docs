# Филтър: wu_mt_syncable_user_meta_keys {#filter-wumtsyncableusermetakeys}

Филтрира списъка от ключове на метаданни на потребителите, които трябва да бъдат синхронизирани с отдалечени възли.

## Параметри {#parameters}

| Име | Тип | Описание |
|------|------|-------------|
| $syncable_keys | `array` | По подразбиране ключовете на метаданни, които да бъдат синхронизирани. |
