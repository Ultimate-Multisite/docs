# Changelog на Създателя на Административни Страници

### Версия 1.9.0 - Публикувана на 2026-01-18

* Поправих: Критични грешки при активиране.
* Преструктурирах: Класовете за източници на съдържание, за да използват Singleton trait с правилна инициализация на родителски конструктор
* Поправих: Грешки при компилиране на Vue.js шаблони, които пречиха на избора на тип съдържание да се покаже
* Поправих: Липсващи Vue данни (template_id, external_link_url_embedable, external_link_url_checking)
* Поправих: Editor TinyMCE не се инициализира правилно при зареждане на страницата
* Поправих: Файлът за поддръжка на Divi не се зарежда автоматично чрез Composer
* Поправих: Повредена PHP синтактична конструкция в класът за поддръжка на Divi
* Подобрих: Организация и минифициране на JavaScript активите
* Подобрих: Съвместимостта с най-новите версии на WordPress и page builder-ите
