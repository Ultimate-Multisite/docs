# Filter: wu_apc_should_display_admin_menu

Позволява на администраторите да скрият страниците в менюто на админ панела, след като са приключили процеса на персонализиране.

## Параметри

| Име | Тип | Описание |
|------|------|-------------|
| $should_display_admin_menu | `bool` | Ако трябва да се показват страниците. |
| $network_active | `bool` | Проверява дали плагинът е активен за администратора на мрежата. |
