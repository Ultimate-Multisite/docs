# Filter: wu_apc_get_meta

Позволява на разработчиците да филтрират крайните стойности и да добавят нови случаи на обработка.

## Параметри

| Name | Type | Описание |
|------|------|-------------|
| $default_value | `string` | Стойността по подразбиране. |
| $meta_type | `string` | Тип на мета данните, които трябва да бъдат извлечени. |
| $meta_key | `string` | Име на мета полето. |

### От версия

- 1.4.0
### Източник

Дефиниран в [`inc/wu-apc-functions.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-admin-page-creator/blob/main/inc/wu-apc-functions.php#L93) на линия 93


## Връща
Нова стойност по подразбиране.
