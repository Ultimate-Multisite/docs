# Admin Page Creator

## Съдържание {#content}
Admin Page Creator Ви позволява лесно да добавяте нови админ страници и подстраници директно в админ панела на WordPress. Може да се използва за създаване на страница за добре дошли или учебна страница, която може да служи като целева страница (landing page) за новите потребители. Можете да използвате page builder като Beaver Builder, Elementor, Brizy и Oxygen, за да разработите страниците без никакви затруднения.

Също така предлага поддръжка за персонализиран HTML, CSS и JavaScript, както и контрол на достъпа, базиран на плана и ролята на потребителя!

## Функции {#features}
* Поддръжка на персонализиран HTML и CSS
* Поддръжка на JavaScript
* PHP Content Mode
* Разрешения, базирани на ролята на потребителя
* Поддръжка на page builders

## Инсталация {#installation}

1. Качете **ultimate-multisite-admin-page-creator** в директорията **/wp-content/plugins/**
2. Активирайте plugin чрез менюто **Plugins** в админ панела на вашата WordPress Network

## Наръчник за употреба {#usage-guide}

Докато WordPress прави добавянето на съдържание към фронт енда на сайтовете изключително лесно, създаването на страници и подстраници от админ страна изисква някои основни умения по кодиране. Ultimate Multisite: Admin Page Creator има за цел да пренесе тази лекота на употреба, характерна за фронт енда, и в бекенда на сайтовете на вашата мрежа.

## Защо да използвате Admin Page Creator? {#why-use-admin-page-creator}

Може би искате да предложите персонализирана страница в админ панела на вашите клиенти с учебни материали, може би искате да добавите персонализирана страница с често задавани въпроси (FAQs) и контактен виджет. Възможностите с Admin Page Creator са безкрайни, и това е това, което ще изследваме в днешния ни урок.

## Създаване на първата Ви персонализирана админ страница {#creating-your-first-custom-admin-page}

След инсталиране и активиране на Ultimate Multisite: Admin Page Creator, отидете в менюто на админа на мрежата си и изберете Admin Pages.

![Custom Admin Pages list on Network Admin](/img/admin/admin-pages-list.png)

_Списък с персонализирани админ страници в админ панела на мрежата_

Ще видите списък на всички админ страници, които сте създали досега (тъй като това е първото Ви посещение, списъкът ще е празен). Просто кликнете върху **Add new Admin Page**. Ще бъдете пренасочени към редактора на админ страницата.

## Меню опции {#menu-options}

Отстрани, в мета-бокса General Options, ще можете да конфигурирате типа на менюто, етикета и много други неща.

### Menu Types {#menu-types}

Вашата нова персонализирана админ страница може да има два различни типа меню: главно меню (top-level menu) и подменю (submenu).

Главно меню, както подсказва името, ще добави новосъздадената Ви страница към админ менюто, заедно с дефинитивните страници на WordPress.

Ако изберете опцията за главно меню, ще можете да изберете и реда на менюто, и икона за менюто.

Ако искате тази страница да бъде подменю, ще можете да изберете родителска страница, към която да прикрепите тази страница. Можете да добавите вашата персонализирана страница към стандартните страници на WordPress или към Вашите персонализирани главни админ страници.

## Advanced Options {#advanced-options}

Трябва да можете да добавяте всякакви видове съдържание към Вашите персонализирани админ страници. Това включва CSS правила и файлове, JavaScript и външни библиотеки. Това можете да направите с помощта на Advanced Options Tab.

### Custom CSS {#custom-css}

В таба за CSS можете да добавите собствен персонализиран CSS код, както и да импортирате CSS файлове от отдалечени сайтове (като CDN, Google Fonts и др.).

## Active {#active}

Ако искате да деактивирате персонализирана админ страница, без да я изтривате или да променяте разрешенията, просто отмаркирайте опцията Active.

Деактивирането на главно меню автоматично ще деактивира всички подстраници, прикрепени към този родителски страници.
