# Журнал с промени на Mailster

### 1.0.0
* Първоначално пускане
* Автоматично синхронизиране на абонати при регистрация или плащане
* Назначаване на списъци, специфични за продукта
* Избор (opt-in) чрез поле с квадратче за съответствие с GDPR
* Поддръжка на двойно съгласие (double opt-in)
* Изчерпателно мапиране на полета
* Подробно логиране
