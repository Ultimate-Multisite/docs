# Журнал с промени за Ultimate AI Connector за съвместими ендпойнти

## Версия 2.0.0 — Публикувана на 2026-04-24 {#version-200--released-on-2026-04-24}

* Добавена: Поддръжка на множество доставчици — можете да конфигурирате множество AI ендпойнти и да маршрутизирате заявките с автоматично превключване (fallback) между доставчиците.
* Поправка: Интеграция на Multi-provider SDK с правилните ID на доставчиците, URL за регистрация и обхват (scoping) на HTTP филтрите за всеки доставчик.
* Поправка: Новата карта за доставчик сега се разширява автоматично при добавяне; добавена защита срещу кеширане на скриптове при актуализация на плагина.
* Поправка: Динамично пространство имена (namespace) за класа на доставчика за eval() и автоматично разширяване на поведението за новодобавени доставчици.
* Поправка: Използва стабилните компоненти Card/CardBody/CardHeader/CardDivider (вече не са експериментални) за съвместимост с WordPress 6.9+.
* Поправка: Заместена е невалидната DragHandle с unicode икона за хват (grip icon) за пренареждане на доставчиците чрез плъзгане.
* Поправка: Елиминиран е блокиращият HTTP заявка, която се изпраща при всяко зареждане на страница.
* Подобрено: Работflows на GitHub Actions са актуализирани до Node.js 24.
