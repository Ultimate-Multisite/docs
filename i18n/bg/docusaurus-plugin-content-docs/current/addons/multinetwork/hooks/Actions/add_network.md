# Action: add_network {#action-addnetwork}

Изпълнява се след като е добавена нова мрежа.

## Параметри {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $new_network_id | `int` | ID на добавената мрежа. |
| $r | `array` | Пълен асоциативен масив с аргументи на мрежата. |
