# Действие: update_network {#action-updatenetwork}

Активира се след, че съществуваща мрежа е била актуализирана.

## Параметри {#parameters}

| Име | Тип | Описание |
|------|------|-------------|
| $network_id | `int` | ID на добавената мрежа. |
| $args | `array` | Асоциативен масив от аргументи на мрежата. |

### От версия {#since}

- 1.3.0
