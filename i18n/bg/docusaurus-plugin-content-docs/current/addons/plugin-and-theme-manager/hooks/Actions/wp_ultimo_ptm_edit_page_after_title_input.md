# Action: wp_ultimo_ptm_edit_page_after_title_input

Позволява на разработчиците на плагини да добавят допълнителна информация под полето за въвеждане на текст.

## Параметри

| Name | Type | Description |
|------|------|-------------|
| $ | `object` | Обект, съдържащ информацията |
| $ | `\WU_Page` | Инстанция на страница Ultimate Multisite |

### От версия

- 1.8.2
### Източник

Дефинирано в [`views/base/edit.php`](https://github.com/Ultimate-Multisite/ultimate-multisite-plugin-and-theme-manager/blob/main/views/base/edit.php#L95) на линия 95
