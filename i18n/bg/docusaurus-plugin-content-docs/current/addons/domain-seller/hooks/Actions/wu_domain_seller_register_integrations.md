# Действие: wu_domain_seller_register_integrations

Позволява на други плагини да регистрират допълнителни интеграции за продавачи на домейни.

## Параметри {#parameters}

| Име | Тип | Описание |
|------|------|-------------|
| $registry | `\WP_Ultimo\Integrations\Integration_Registry` | Регистраторът на интеграциите. |
