# Филтър: wp_ultimo_skip_network_active_check

Позволява на разработчиците да заобиколят тази проверка.

Това е полезно, когато използвате настройки, базирани на Composer или други персонализирани среди, като например Bedrock, където нормално е да се използват плагини като mu-plugins.
