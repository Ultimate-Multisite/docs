# Fluent Forms

Този ад-он позволява да добавите лимити към Fluent Forms в Multisite Ultimate за продукти, членства или сайтове.
Fluent Forms използва персонализирани таблици (custom tables) вместо персонализирани типове съдържание (custom post types), така че нормалните опции за лимитиране да не работят с него.

## Характеристики

- Добавяне на лимити за Fluent Forms или Conversational Fluent Forms
- Клиентите получават приятелски съобщение, което ги насърчава да надградят плана си, когато достигнат лимита.
