# Filter: wu_password_require_special {#filter-wupasswordrequirespecial}

Филтър, който определя дали да се изискват специални символи в паролите.

## Параметри {#parameters}

| Име | Тип | Описание |
|------|------|-------------|
| $require | `bool` | Дали да се изискват специални символи. По подразбиране е true, когато правилата са наложени. |
| $defender_active | `bool` | Дали Defender Pro Strong Password е активен. |
