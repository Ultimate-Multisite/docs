# Filter: wu_plan_product_types

Филтрира продуктовите типове, които трябва да бъдат третирани като планове.

Този филтър позволява на аддоновете да регистрират допълнителни продуктови типове, които трябва да бъдат признати като планове при валидиране и групиране.

## Параметри

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | Масив от продуктови типове, които трябва да бъдат третирани като планове. |

### Начинове за употреба

- 2.3.0
### Източник

Дефиниран в [`inc/functions/product.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/product.php#L196) на ред 196
