# Филтър: wu_documentation_get_link

Позволява на разработчиците на плагини да филтрират връзките.

Не съм сигурен как точно това може да бъде полезно, но няма как да навреди да го имаме.

## Параметри

| Име | Тип | Описание |
|------|------|-------------|
| $link | `string` | Регистрираната връзка |
| $slug | `string` | Слогът, използван за извличане на връзката |
| $default_link | `string` | Регистрираната линк по подразбиране |

- 1.7.0
