# Filter: wu_current_\{$type\}_get_manage_url

Позволява на разработчиците да променят параметрите на URL адрес за управление на сайта.

## Параметри

| Име | Тип | Описание |
|------|------|-------------|
| $manage_site_url | `string` | URL адрес за управление на сайта. |
| $id | `int` | ID на сайта. |
| $site_hash | `string` | Хеш на сайта. |

### От

- 2.0.9
### Източник

Дефиниран в [`inc/class-current.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-current.php#L214) на ред 214


## Връща
Променен URL адрес за управление.
