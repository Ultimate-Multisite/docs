# Филтър: wu_cart_get_subtotal

Филтрира общата сума на "началната стойност".

## Параметри {#parameters}

| Име | Тип | Описание |
|------|------|-------------|
| $subtotal | `float` | Обща сума, която трябва да бъде платена днес. |
| $this | `\WP_Ultimo\Checkout\Cart` | Обект на количката. |
