# Филтър: wu_billing_address_fields

Позволява на разработчиците на плагини да филтрират полетата за адрес за фактуриране.

## Параметри

| Име | Тип | Описание |
|------|------|-------------|
| $fields | `array` | Масив с данни за адрес за фактуриране. |
| $zip_only | `bool` | Ако ни е необходим само данните за пощенски код и държава. |
