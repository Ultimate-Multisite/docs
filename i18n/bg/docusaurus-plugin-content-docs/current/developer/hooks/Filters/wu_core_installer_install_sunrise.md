# Филтър: wu_core_installer_install_sunrise

Позволява на доставчиците на хостинг да инсталират константата по различен начин.

Връщането на `true` ще предотврати Ultimate Multisite от опит да запише във файла `wp-config`.

## Параметри

| Name | Type | Description |
|------|------|-------------|
| $short_circuit | `bool` | |

- 2.0.0
