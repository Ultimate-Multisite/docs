# Филтър: wu_sso_url

Филтрира генерираните SSO URL адреси, преди да бъдат върнати за действия на клиенти между домейни.

Използвайте този филтър, когато интеграция трябва да добави доверен контекст към SSO връзка на суверенен tenant или да замени URL адреса на посредника, като запази валидацията на token на Ultimate Multisite.

## Параметри {#parameters}

| Име | Тип | Описание |
|------|------|-------------|
| $sso_url | `string` | Генериран SSO URL адрес. |
| $user | `WP_User` | Потребител, който ще бъде удостоверен чрез SSO посещението. |
| $site_id | `int` | ID на целевия сайт за посещението. |
| $redirect_to | `string` | URL адрес на дестинацията след успешна SSO валидация. |

### От версия {#since}

- 2.13.0

### Източник {#source}

Дефинирано в `inc/sso/class-sso.php`.


## Връща {#returns}

Филтриран SSO URL адрес.
