# Филтър: wu_password_require_number

Филтрира дали в паролите трябва да има числа.

## Параметри {#parameters}

| Име | Тип | Описание |
|------|------|-------------|
| $require | `bool` | Дали е необходимо да има числа. По подразбиране е `true`, когато правилата са активни. |
| $defender_active | `bool` | Дали е активен Defender Pro Strong Password. |
