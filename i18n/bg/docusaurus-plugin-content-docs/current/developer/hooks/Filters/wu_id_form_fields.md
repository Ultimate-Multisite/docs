# Filter: wu_\{$id\}_form_fields

Филтрира полетата на една форма. Формата се идентифицира по ID-то, което се съдържа в името на филтъра.

## Параметри

| Name | Type | Описание |
|------|------|-------------|
| $fields | `array` | Списък с полетата на формата. |

### От версия

- 2.0.0
