Филтрира дали да се изисква използването на малки букви в паролите.

## Параметри {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $require | `bool` | Дали да се изисква използването на малки букви. По подразбиране е true, когато правилата са активни. |
| $defender_active | `bool` | Дали е активен Defender Pro Strong Password. |
