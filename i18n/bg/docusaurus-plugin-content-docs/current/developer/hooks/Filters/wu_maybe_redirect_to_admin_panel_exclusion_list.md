# Филтър: wu_maybe_redirect_to_admin_panel_exclusion_list

Създава списък с изключения от параметри, които предотвратяват автоматичното пренасочване.

Това е необходимо, защото в против случай конструкторът на страници няма да може да редактира страницата за вход, след като тя бъде дефинирана.

- 2.0.4
