# Filter: wu_add_new_domain_modal_fields {#filter-wuaddnewdomainmodalfields}

Филтрира полетата за модала за добавяне на нов домейн.

Това позволява на аддонове (например Domain Seller) да модифицират или да заменят полетата за регистрация на домейн.

## Параметри {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $fields | `array` | Полетата на формата. |
