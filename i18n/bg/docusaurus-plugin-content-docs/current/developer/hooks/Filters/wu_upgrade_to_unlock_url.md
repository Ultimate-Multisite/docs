# Филтър: wu_upgrade_to_unlock_url

Позволява на разработчиците да променят URL адреса за отключване на надграждането

## Параметри

| Име | Тип | Описание |
|------|------|-------------|
| $url | `string` | URL адресът за надграждане. |
| $args | `array` | Модулът и типът на ограничението, което трябва да бъде надградено. |

### Източник

Дефиниран в [`inc/functions/limitations.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/limitations.php#L177) на ред 177
