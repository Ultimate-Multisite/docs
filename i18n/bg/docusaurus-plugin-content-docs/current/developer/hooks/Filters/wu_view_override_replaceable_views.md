# Filter: wu_view_override_replaceable_views

Позволява на разработчиците да добавят допълнителни папки към списъка с заместваеми.

Бъдете внимателни, тъй като позволяването на допълнителни папки може да доведе до зареждане на остарели копия, вместо версиите на Ultimate Multisite.

## Параметри

| Name | Type | Description |
|------|------|-------------|
| $replaceable_views | `array` | Списък с разрешени папки. |

### Since

- 2.0.0
