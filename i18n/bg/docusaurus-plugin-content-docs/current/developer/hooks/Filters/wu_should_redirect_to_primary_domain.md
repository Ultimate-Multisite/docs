# Филтър: wu_should_redirect_to_primary_domain

Позволява на разработчиците да прекъснат пренасочването, предотвратявайки му да се случи.

## Параметри

| Name | Type | Description |
|------|------|-------------|
| $should_redirect | `bool` | Дали трябва да се извърши пренасочване или не. |

- 2.0.0
