# Филтър: wu_rest_update_item

Определя дали заявката може да продължи или трябва да бъде прекъсната.

## Параметри

| Name | Type | Описание |
|------|------|-------------|
| $allowed | `bool` | Първоначална стойност за връщане. |
| $rest_base | `array` | Слог (slug) на обекта/сущността. |
| $this | `\Base_Manager` | Инстанцията на обекта. |
