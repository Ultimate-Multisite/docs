# Филтър: wu_checkout_validation_rules

Позволява на разработчиците на плагини да филтрират правилата за валидация.

## Параметри {#parameters}

| Име | Тип | Описание |
|------|------|-------------|
| $validation_rules | `array` | Правилата за валидация, които трябва да бъдат използвани. |
| $checkout | `\Checkout` | Класът за чекаута. |

- 2.0.20
