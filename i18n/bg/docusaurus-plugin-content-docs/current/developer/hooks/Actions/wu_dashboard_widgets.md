# Action: wu_dashboard_widgets

Позволява на разработчиците на плагини да добавят виджета (widgets) към Панел на мрежата (Network Dashboard Panel).

## Параметри {#parameters}

| Име | Тип | Описание |
|------|------|-------------|
| $tab | `string` | Текущият таб. |
| $screen | `\WP_Screen` | Обектът на екрана. |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Инстанция на административната страница на Ultimate Multisite. |

### От версия {#since}

- 2.0.0
