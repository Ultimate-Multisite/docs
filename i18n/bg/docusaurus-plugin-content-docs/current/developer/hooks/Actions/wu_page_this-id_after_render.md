# Action: wu_page_\{$this->id\}_after_render

Позволява на разработчиците на плагини да добавят допълнително съдържание след като изведем страницата.

## Параметри

| Name | Type | Описание |
|------|------|-------------|
| $page_id | `string` | Идентификаторът (ID) на тази страница |
| $page | `object` | Обектът на страницата. |
