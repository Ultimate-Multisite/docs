# Действие: wu_site_created {#action-wusitecreated}

Изстрелва, когато сайтът бъде създаден за първи път.

Не изстрелва, ако е дублиран от шаблон.

## Параметри {#parameters}

| Име | Тип | Описание |
|------|------|-------------|
| $data | `array` | Обектите данни, които ще бъдат запазени. |
| $site | `\Site` | Инстанцията на обекта. |
