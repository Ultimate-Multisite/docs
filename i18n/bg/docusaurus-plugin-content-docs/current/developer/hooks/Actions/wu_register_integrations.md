# Action: wu_register_integrations {#action-wuregisterintegrations}

Извиква се, за да позволи на интеграциите да се регистрират сами.

Основният код регистрира своите интеграции тук.

## Параметри {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $registry | `\Integration_Registry` | Инстанцията на регистъра. |
