# Пренареждане на списъка с държави на Ultimate Multisite

Падащото меню за избор на държава е подредено азбукално въз основа на английските варианти на имената на държавите. Това може да означава, че понякога държавата на вашата компания и/или на повечето от вашите клиенти е погребана в средата на огромен списък с имена на държави.

За да помогнете да поправите това и да добавите държавата по избор в началото на списъка, можете да използвате кода по-долу (можете да го добавите в mu-plugin):

```php
add_filter('wu_get_countries', function($countries) {

// Променете DE на двубуквения код на вашата държава

// Пример по-долу за Германия.

unset($countries['DE']);

// Променете DE на двубуквения код на вашата държава

// и името на държавата ви.

// Пример по-долу за Германия.

$countries = array_merge(array( 'DE' => __('Germany', 'ultimate-multisite'), ), $countries);

return $countries;

});
```
