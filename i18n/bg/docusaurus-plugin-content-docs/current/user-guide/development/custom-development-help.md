# Как да получите специализирана помощ за разработка от Codeable?

Понякога наистина ни е нужна някаква страхотна персонализация, за да накараме мрежата ни да работи точно както искаме. Ако търсите това, препоръчваме [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) – място, където можете да намерите опитни разработчици, които имат опит с Ultimate Multisite.

[Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) е платформа, която свързва клиенти с квалифицирани фрийланс разработчици за WordPress, които могат да работят по проекти във всеки размер. Те имат изключително внимателен процес на кандидатстване, за да гарантират, че ще бъдете свързани с експерти, които са специално квалифицирани за задачата, която имате предвид.

### Как да създадете проект в Codeable? {#how-to-create-a-project-in-codeable}

Първо, отидете на [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) и кликнете върху бутона **Start a Project**.

<!-- Screenshot unavailable: Codeable homepage with Start a Project button -->

Сега, за да подадете проект, кликнете отляво върху **Start a Project**.

<!-- Screenshot unavailable: Codeable project submission page with Start a Project option -->

В полето "I need help with" (Имам нужда от помощ с), изберете **Customizing** (Персонализиране):

<!-- Screenshot unavailable: Codeable project form with Customizing selected -->

В полето "My" (Моето), изберете **Plugin** (Плъгин):

<!-- Screenshot unavailable: Codeable project form with Plugin selected -->

След това кликнете върху бутона **Continue** (Продължи).

<!-- Screenshot unavailable: Codeable project form Continue button -->

Сега е време да опишете проекта и нуждите си по най-добрия възможен начин. Опитайте се да предоставите възможно най-много информация!

<!-- Screenshot unavailable: Codeable project description form -->

Отговорете на следните въпроси, за да помогнете с ценообразуването.

<!-- Screenshot unavailable: Codeable project pricing questions -->

След това, ако все още нямате [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) акаунт, създайте го и кликнете върху бутона **Publish My Project**.

<!-- Screenshot unavailable: Codeable account creation and Publish My Project button -->

След като публикувате [Codeable](https://codeable.io/partners/wpultimo/?ref=7wHwE) проекта си, платформата ще ви свърже с разработчици, чиито умения съответстват на изискванията за вашия проект.

Този процес ще ви помогне да намерите правилния човек за работата си, който има опит с WordPress и Ultimate Multisite!

Минималната ставка на Codeable е $70–$120 USD на час. Оценките ще варират в зависимост от обхвата на работата, сложността и спешността на проекта.

След като получите оценка, ще можете да наемете разработчика, който най-добре отговаря на вашите нужди и да стартирате проекта си заедно!
