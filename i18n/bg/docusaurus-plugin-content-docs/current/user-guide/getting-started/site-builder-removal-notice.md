# Уведомление за премахване на режима Site Builder {#site-builder-mode-removal-notice}

**Режим Site Builder е премахнат в Superdav AI Agent v1.12.0.** Ако сте използвали режима Site Builder, трябва да преминете към **агента Setup Assistant** за създаване на теми и настройка на сайта.

## Какво се случи? {#what-happened}

### Site Builder Mode (Legacy) {#site-builder-mode-legacy}

Режим Site Builder беше интерфейс, базиран на поредица от стъпки (wizard), който позволяваше:

- Създаване на сайтове от шаблони
- Конфигуриране на основни настройки
- Избор на тема
- Настройване на първоначално съдържание

### Какво го замени? {#what-replaced-it}

**Агентът Setup Assistant** сега управлява цялата функционалност на Site Builder, като предлага:

- По-гъвкава настройка, ръководена от агент
- Подобри опции за персонализиране на теми
- Интеграция с онбординга на Theme Builder
- Запаметяване на паметта (site_brief) за бъдещи сесии

## Ако сте използвали Site Builder Mode {#if-you-were-using-site-builder-mode}

### Вашите сайтове са в безопасност {#your-sites-are-safe}

- Съществуващите сайтове, създадени с Site Builder mode, продължават да работят
- Няма загуба на данни или прекъсване на работата на сайта
- Можете да продължите да управлявате сайтовете си нормално

### Мигрирайте към агента Setup Assistant {#migrate-to-setup-assistant-agent}

За ново настройване на сайт или промяна на тема, използвайте агента Setup Assistant:

```
"Help me set up a new site"
```

или

```
"Start the Theme Builder onboarding"
```

Агентът Setup Assistant предоставя същата функционалност, но с повече гъвкавост.

## Сравнение: Site Builder срещу Setup Assistant {#comparison-site-builder-vs-setup-assistant}

| Функция | Site Builder (Премахнат) | Setup Assistant (Нов) |
|---------|------------------------|----------------------|
| Метод на настройка | Форма с стъпки (Wizard form) | Разговор на агент (Agent conversation) |
| Избор на тема | Предварително зададени шаблони | Кастомерно генериране |
| Персонализация | Ограничени опции | Пълен дизайн системен набор |
| Site brief | Не се запазва | Запаметяваща памет (Persistent memory) |
| Бъдещи сесии | Повторно настройване | Използване на запазения site_brief |
| Гъвкавост | Фиксиран работен процес | Адаптивен разговор |

## Мигриране към агента Setup Assistant {#migrating-to-setup-assistant-agent}

### За нови сайтове {#for-new-sites}

Вместо да използвате Site Builder mode:

1. Поискайте: "Help me set up a new site"
2. Агентът Setup Assistant ще ви ръководи през:
   - Целта и целите на сайта
   - Целевата аудитория
   - Идентичността на марката
   - Генериране на тема
   - Първоначална конфигурация

### За съществуващи сайтове {#for-existing-sites}

Ако имате съществуващ сайт, създаден с Site Builder mode:

1. Можете да продължите да го използвате, както е
2. За актуализиране на темата, поискайте: "Redesign my site"
3. Агентът Setup Assistant ще ви помогне да създадете нова тема
4. Данните на вашия сайт остават непроменени

### За промяна на теми {#for-theme-changes}

Вместо избора на тема в Site Builder mode:

1. Поискайте: "Change my theme"
2. Агентът Setup Assistant ще:
   - Ви попита за предпочитанията ви в дизайна
   - Генерира кастомизирана тема
   - Я активира на вашия сайт

## Основни разлики {#key-differences}

### Site Builder Mode {#site-builder-mode}

```
1. Изберете шаблон
2. Изберете тема
3. Конфигурирайте основни настройки
4. Готово
```

### Setup Assistant Agent {#setup-assistant-agent}

```
1. Опишете целта на сайта си
2. Дефинирайте целевата аудитория
3. Изберете предпочитания за дизайна
4. Агентът генерира кастомизирана тема
5. Агентът активира темата
6. Site brief се запазва за бъдещи сесии
```

## Предимства на агента Setup Assistant {#benefits-of-setup-assistant-agent}

### По-гъвкав {#more-flexible}

- Опишете сайта си с естествен език
- Получавате персонализирани препоръки
- Адаптира се към вашите специфични нужди

### По-добра персонализация {#better-customization}

- Генериране на кастомизирана тема
- Решения в рамките на дизайн системата
- Запаметяване на дизайн токени (Persistent design tokens)

### Запаметяваща памет {#persistent-memory}

- Вашият site_brief се запазва
- Бъдещите агенти разбират сайта ви
- Няма нужда да повтаряте информацията за настройката

### Интегриран работен процес {#integrated-workflow}

- Онбординг на Theme Builder
- Умение Design System Aesthetics
- Контрол на видимостта на възможностите (Ability Visibility controls)
- Всичко работи заедно безпроблемно

## Отстраняване на проблеми {#troubleshooting}

### Не намирам Site Builder mode {#i-cant-find-site-builder-mode}

Режимът Site Builder е премахнат. Използвайте вместо това агента Setup Assistant:

```
"Help me set up a new site"
```

### Искам да възпроизвеждам сайт от Site Builder {#i-want-to-recreate-a-site-from-site-builder}

Можете да го възпроизведете с агента Setup Assistant:

1. Поискайте: "Help me set up a new site"
2. Опишете целта и дизайна на оригиналния си сайт
3. Агентът ще генерира подобна тема
4. Вашият site_brief ще бъде запазен за бъдеща справка

### Моят съществуващ сайт от Site Builder не работи {#my-existing-site-builder-site-isnt-working}

Съществуващите сайтове, създадени с Site Builder mode, продължават да работят. Ако изпитвате проблеми:

1. Проверете дали темата ви все още е активна
2. Проверете дали вашите плагини са активирани
3. Проверете логовете за грешки на WordPress
4. Свържете се с поддръжката, ако проблемите продължат

### Мога ли все още да използвам старите си шаблони на Site Builder? {#can-i-still-use-my-old-site-builder-templates}

Шаблоните на Site Builder вече не са налични. Въпреки това:

- Съществуващите ви сайтове продължават да работят
- Можете да възпроизведете подобни сайтове с агента Setup Assistant
- Агентът Setup Assistant предоставя повече опции за персонализация

## Следващи стъпки {#next-steps}

1. **За нови сайтове**: Използвайте агента Setup Assistant
2. **За съществуващи сайтове**: Продължете да ги използвате, както са
3. **За промяна на теми**: Поискайте помощ от агента Setup Assistant
4. **За фини настройка на дизайна**: Използвайте умението Design System Aesthetics
