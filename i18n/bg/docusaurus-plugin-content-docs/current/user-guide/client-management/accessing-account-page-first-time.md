# Как мога да достъпя страницата си за профил за първи път?

Добре дошли в Ultimate Multisite!

Достъпете страницата си за профил чрез линка, който сте получили в имейла си след покупката на нашия плагин:

<!-- Screenshot unavailable: Account page email with access link -->

Използвайте имейла, който сте предоставили по време на покупката, и паролата, която сте получили в имейла си.

<!-- Screenshot unavailable: Login page with email and password fields -->

Тъй като влизате в този профил за първи път, ще бъдете помолени да смените паролата си. Сменете я и натиснете бутона _Save_.

<!-- Screenshot unavailable: Change password form with Save button -->

На страницата си за профил можете да управлявате всичките си детайли, като например фактуриране, подновяване, отказ и много други неща.
