# Вградени агенти {#built-in-agents}

Gratis AI Agent v1.9.0 се доставя с пет вградени агента, всеки предварително конфигуриран с фокусиран набор от инструменти, персонализиран системен prompt и начални предложения, съобразени с обичайни задачи в тази област. Превключването между агенти променя какво може да прави асистентът и как отговаря — без никаква конфигурация от ваша страна. Superdav AI Agent v1.18.0 може да добавя инструменти, съобразени с график, записи за напомняния, approval gates и SMS известия към тези работни процеси, когато свързаните интеграции са конфигурирани.

## Какво е агент? {#what-is-an-agent}

Всеки агент е именуван конфигурационен профил, който комбинира:

- **Инструменти** — възможностите, които агентът има право да извиква (напр. Content Writer има достъп до възможности за създаване на публикации; Design Studio има достъп до CSS и theme.json възможности)
- **Системен prompt** — инструкции, които задават тона, приоритетите и ограниченията на агента
- **Предложения** — предварително написани prompt-и, показвани в чат интерфейса, за да ви помогнат да започнете бързо

## Достъп до Agent Picker {#accessing-the-agent-picker}

1. Отворете панела **Gratis AI Agent** в страничната лента на администрацията на WordPress.
2. Щракнете върху **иконата на агент** в горния ляв ъгъл на заглавката на чата (иконата се променя, за да отрази активния агент).
3. **Agent Picker** се отваря като наслагване с формуляр-таблица. Всеки агент е изброен със своята икона, име и едноредово описание.
4. Щракнете върху ред на агент, за да го активирате. Заглавката на чата се обновява незабавно.

Можете също да превключвате агенти по средата на разговор — системният prompt на новия агент влиза в сила от следващото съобщение.

## Петте вградени агента {#the-five-built-in-agents}

### Content Writer {#content-writer}

**Фокус:** Създаване и редактиране на публикации, страници и контактни формуляри.

**Налични инструменти:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`. При активирани интеграции на Superdav AI Agent v1.18.0, конфигуриран контекст от календар, approval gates, напомняния и инструменти за SMS известия може също да са достъпни за одобрени работни процеси.

**Какво прави добре:**
- Чернови и публикуване на блог публикации по кратко описание или план
- Създаване на партиди от целеви страници за нов сайт
- Изграждане на контактни формуляри и формуляри за запитвания
- Задаване на featured images на публикации от URL или търсене
- Изготвяне на последващи съобщения за събитие от конфигуриран контекст на Google Calendar, след което поставяне на пауза за одобрение преди изпращане на известия

**Начални предложения:**
- *Напишете блог публикация от 500 думи за ползите от WordPress multisite.*
- *Създайте страница „За нас“, „Услуги“ и „Контакт“ и ги публикувайте.*
- *Добавете формуляр за запитване за резервация към страницата „Контакт“.*
- *Изгответе напомняне за участниците в утрешното конфигурирано календарно събитие и изчакайте одобрение, преди да го изпратите.*

---

### Site Builder {#site-builder}

**Фокус:** Създаване на уебсайт от край до край от един prompt.

**Налични инструменти:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`. Със Superdav AI Agent v1.18.0 конфигурирани инструменти за управлявана услуга, одобрение, напомняне, календар и SMS може да са налични там, където администраторите ги активират.

**Какво прави добре:**
- Генериране на многофазен план за изграждане на сайт за описан тип бизнес
- Автономно изпълнение на всяка фаза — структура, съдържание, навигация, дизайн
- Възстановяване от грешки по средата на плана без необходимост от ръчна намеса
- Инсталиране на препоръчани plugins като част от изграждането
- Създаване на контактни формуляри директно от чат интерфейса (Superdav AI Agent v1.10.0+)
- Координиране на напомняния за стартиране или последващи съобщения към участници без дублиращи се известия, когато approval gates и записи за напомняния са активирани

**Начални предложения:**
- *Изградете сайт за фотографско портфолио с тип публикации за галерия, страница за резервации и контактна форма.*
- *Създайте уебсайт за ресторант с онлайн меню, работно време и формуляр за запитване за резервация на маса.*
- *Настройте сайт за консултант на свободна практика със страници за услуги, секция портфолио и блог.*
- *Добавете контактна форма към страницата „Контакт“ чрез site builder.*
- *След като контролният списък за стартиране на сайта бъде одобрен, изпратете SMS напомняне до конфигурирания контакт на заинтересованата страна.*

### Design Studio {#design-studio}

**Фокус:** Визуална персонализация — цветове, типография, CSS и block patterns.

**Налични инструменти:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Какво прави добре:**
- Прилагане на именувани theme presets (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- Фина настройка на глобална типография и цветови палитри чрез theme.json
- Инжектиране на custom CSS за специфични за бранда overrides
- Правене на screenshot на страница и преглеждането ѝ за проблеми с дизайна

**Начални предложения:**
- *Приложете preset-а warm-editorial и след това задайте основния цвят на #2d6a4f.*
- *Направете screenshot на началната страница и ми кажете какво бихте подобрили.*
- *Създайте reusable hero block pattern с фоново изображение на цялата ширина и центрирано заглавие.*

### Plugin Manager {#plugin-manager}

**Фокус:** Откриване, инсталиране и управление на WordPress plugins.

**Налични инструменти:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Какво прави добре:**
- Препоръчване на най-добрия plugin за описан случай на употреба
- Инсталиране на пакети с възможности от регистъра
- Преглеждане на наличния каталог с възможности по категория

**Начални предложения:**
- *Кой е най-добрият plugin за membership directory?*
- *Инсталирайте пакета с WooCommerce възможности.*
- *Покажете ми всички налични пакети с ecommerce възможности.*

---

### Support Assistant {#support-assistant}

**Фокус:** Отговаряне на въпроси относно съдържанието на сайта, настройките и WordPress конфигурацията.

**Налични инструменти:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Какво прави добре:**
- Преглежда текущите настройки и опции на сайта
- Обяснява какви типове публикации, таксономии и менюта са конфигурирани на сайта
- Отговаря на въпроси „какво прави тази настройка?“, като чете актуални стойности
- Служи като диагностичен слой само за четене преди извършване на промени

**Начални предложения:**
- *Кои plugins и настройки са активни в момента на този сайт?*
- *Изброй всички персонализирани типове публикации, регистрирани на този сайт.*
- *Какви навигационни менюта съществуват и къде са назначени?*

## Superdav Automation интеграции {#superdav-automation-integrations}

Когато интеграциите на Superdav AI Agent v1.18.0 са конфигурирани, вградените агенти могат да участват в по-безопасни автоматизирани работни процеси, съобразени с графици:

- **Инструментите за четене на Google Calendar** позволяват на агентите да преглеждат конфигурираните календари и събития, преди да изготвят последваща работа.
- **Съпоставянето на контакти и участници** помага да се свържат участниците в събитието с WordPress потребители или познати контакти.
- **Контролите за човешко одобрение** спират чувствителни действия, докато упълномощен потребител не ги прегледа и потвърди.
- **Записите за напомняния** предотвратяват дублирани известия, когато планирани задачи се опитват отново или се повтарят.
- **TextBee SMS известията** изпращат конфигурирани текстови съобщения само когато SMS идентификационните данни и разрешенията за работния процес са активирани.

Препоръчителен работен процес: помолете агента да подготви съобщението или действието, прегледайте подканата за одобрение, след което позволете на одобреното действие да продължи. За повтарящи се напомняния дръжте дедупликацията на напомнянията активирана, така че едно и също събитие или контакт да не бъде уведомяван многократно.

## Персонализиране на агенти {#customising-agents}

Всеки вграден агент може да бъде разширен или заменен чрез филтъра `gratis_ai_agent_agents`.

### Добавяне на персонализиран системен prompt към съществуващ агент {#adding-a-custom-system-prompt-to-an-existing-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### Регистриране на нов агент {#registering-a-new-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

Новият агент се появява в Agent Picker веднага след като филтърът се изпълни.

### Премахване на вграден агент {#removing-a-built-in-agent}

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
