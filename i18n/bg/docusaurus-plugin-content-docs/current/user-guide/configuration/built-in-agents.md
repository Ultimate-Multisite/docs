# Вградени Агенти

Gratis AI Agent v1.9.0 включва пет вградени агента, всеки от които е предварително настроен с фокусиран набор от инструменти, специален системн промпт и начални предложения, съобразени с често срещани задачи в тази област. Превключването между агентите променя какво може да прави асистентът и как да отговаря — без да е необходима никаква конфигурация от ваша страна.

## Какво е Агент?

Всеки агент е профил на конфигурация с име, който комбинира:

- **Инструменти (Tools)** — способностите, които агентът може да използва (например, Content Writer има достъп до възможности за създаване на публикации; Design Studio има достъп до възможности за CSS и theme.json)
- **Системн промпт (System prompt)** — инструкции, които задават тона, приоритетите и ограниченията на агента
- **Предложения (Suggestions)** — предварително написани промптове, които се показват в чат интерфейса, за да ви помогнат да започнете бързо

## Достъп до Избирателя на Агенти (Agent Picker)

1. Отворете панела **Gratis AI Agent** в страничната лента на административния панел на WordPress.
2. Кликнете върху **иконата на агента** в горния ляв ъгъл на заглавието на чата (иконата се променя, за да покаже активния агент).
3. **Agent Picker** се отваря като форма, която се появява върху екрана. Всеки агент е изброен с иконата, името и едноредови описание.
4. Кликнете върху реда на агента, за да го активирате. Заглавието на чата се актуализира незабавно.

Можете също така да сменяте агентите по време на разговор — системният промпт на новия агент започва да действа от следващото съобщение.

## Петте Вградени Агента

### Content Writer (Писач на Съдържание)

**Фокус:** Създаване и редактиране на публикации, страници и контактни форми.

**Налични инструменти:** `create_post`, `update_post`, `batch_create_posts`, `set_featured_image`, `create_contact_form`, `get_option`, `list_post_types`

**Какво прави добре:**
- Създаване и публикуване на блог постове въз основа на кратко описание или план
- Създаване на групи от страници за качване на нов сайт
- Изграждане на контактни и заявки форми
- Задаване на главни изображения на публикации от URL или търсене

**Начални предложения:**
- *Напиши блог пост от 500 думи за ползите от WordPress multisite.*
- *Създай страници „За нас“, „Услуги“ и „Контакти“ и ги публикувай.*
- *Добави форма за заявки за резервация към страницата „Контакти“.*

---

### Site Builder (Конструктор на Сайтове)

**Фокус:** Създаване на уебсайт от начало до край само с един промпт.

**Налични инструменти:** `create_site_plan`, `execute_site_plan`, `get_plan_progress`, `handle_plan_error`, `create_post`, `batch_create_posts`, `register_post_type`, `create_menu`, `add_menu_item`, `inject_custom_css`, `apply_theme_json_preset`, `install_ability`, `recommend_plugin`, `create_contact_form`

**Какво прави добре:**
- Генерира многофазен план за изграждане на сайт, базиран на описания бизнес тип
- Изпълнява всяка фаза автономно — структура, съдържание, навигация, дизайн
- Възстановява от грешки по време на плана, без да изисква ръчна намеса
- Инсталира препоръчителни плагини като част от изграждането
- Създава контактни форми директно от чат интерфейса (Superdav AI Agent v1.10.0+)

**Начални предложения:**
- *Изгради портфолио сайт за фотографи с тип публикация „галерия“, страница за резервации и контактна форма.*
- *Създай уебсайт за ресторант с онлайн меню, часове на отваряне и форма за заявка за резервация на маса.*
- *Настрой сайт за фрийланс консултации с страници за услуги, секция портфолио и блог.*
- *Добави контактна форма към страницата „Контакти“ с помощта на конструктора на сайтове.*

---

### Design Studio (Дизайн Студио)

**Фокус:** Визуална персонализация — цветове, типография, CSS и блок шаблони.

**Налични инструменти:** `inject_custom_css`, `apply_theme_json_preset`, `get_global_styles`, `set_global_styles`, `reset_global_styles`, `add_block_pattern`, `list_block_patterns`, `set_site_logo`, `capture_screenshot`, `review_page_design`

**Какво прави добре:**
- Прилага предварително зададени теми (minimal-dark, warm-editorial, corporate-blue, vibrant-startup, classic-blog)
- Прецизно настройване на глобалната типография и палитрата от цветове чрез theme.json
- Вмъкване на персонализиран CSS за премахване на стилове специфични за марката
- Изпращане на снимка на страница и преглед на дизайна за проблеми

**Начални предложения:**
- *Приложи предварителната настройка warm-editorial и след това задай основния цвят на #2d6a4f.*
- *Изпрати снимка на началната страница и кажи ми какво би подобрил.*
- *Създай повторно използваем блок шаблон за херo с фоново изображение пълна ширина и центриран заглавен ред.*

---

### Plugin Manager (Мениджър на Плагини)

**Фокус:** Откриване, инсталиране и управление на плагини за WordPress.

**Налични инструменти:** `list_available_abilities`, `install_ability`, `recommend_plugin`, `get_option`, `set_option`

**Какво прави добре:**
- Препоръчва най-добрия плагин за описан сценарий на употреба
- Инсталира пакети способности от регистъра
- Преглежда наличния каталог на способности по категории

**Начални предложения:**
- *Какъв е най-добрият плагин за директория за членство?*
- *Инсталирай пакета способности WooCommerce.*
- *Покажи ми всички налични пакети способности за електронна търговия.*

---

### Support Assistant (Асистент за Поддръжка)

**Фокус:** Отговаряне на въпроси за съдържанието, настройките и конфигурацията на WordPress на сайта.

**Налични инструменти:** `get_option`, `list_options`, `list_post_types`, `list_taxonomies`, `list_menus`, `list_available_abilities`

**Какво прави добре:**
- Проучване на текущите настройки и опции на сайта
- Обясняване какви типове публикации, таксономии и менюта са конфигурирани на сайта
- Отговаряне на въпроси като „какво прави тази настройка?“ чрез четене на активни стойности в реално време
- Служи като диагностичен слой, който само чете, преди да бъдат направени промени

**Начални предложения:**
- *Какви плагини и настройки са активни в момента на този сайт?*
- *Изброи всички регистрирани персонализирани типове публикации на този сайт.*
- *Какви навигационни менюта съществуват и къде са назначени?*

## Персонализиране на Агентите

Всеки вграден агент може да бъде разширен или заменен чрез филтъра `gratis_ai_agent_agents`.

### Добавяне на персонализиран системн промпт към съществуващ агент

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    if ( isset( $agents['content-writer'] ) ) {
        $agents['content-writer']['system_prompt'] .= "\n\nAlways write in British English and use the Oxford comma.";
    }
    return $agents;
} );
```

### Регистриране на нов агент

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    $agents['seo-specialist'] = [
        'name'          => 'SEO Specialist',
        'description'   => 'Optimises posts and pages for search engines.',
        'icon'          => 'dashicons-search',
        'tools'         => [ 'get_option', 'set_option', 'create_post', 'update_post', 'list_post_types' ],
        'system_prompt' => 'You are an SEO specialist. Focus on keyword optimisation, meta descriptions, and structured data.',
        'suggestions'   => [
            'Review the homepage title and meta description.',
            'Suggest title tag improvements for the five most recent posts.',
        ],
    ];
    return $agents;
} );
```

Новият агент се появява в Agent Picker веднага след изпълнение на филтъра.

### Премахване на вграден агент

```php
add_filter( 'gratis_ai_agent_agents', function ( array $agents ): array {
    unset( $agents['plugin-manager'] );
    return $agents;
} );
```
