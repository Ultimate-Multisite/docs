# Поставяне на реклами в мрежата си с Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector е **безплатно допълнение**, което ви предоставя инструменти за поставяне на реклами във вашите планове, директно от страницата им за редактиране.

След инсталиране на Ultimate Multisite: Ad Injector, отидете на страницата си _Plugins_, за да го активирате.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Кликнете върху бутона Network Activate_

След като допълнението бъде активирано, мета полето _Advanced Options_ на екрана за редактиране на плана ще има нова опция, наречена _Ad Settings_.

![Ad Settings on the plan edit screen](/img/config/product-edit-full.png)

_Сега можете да редактирате настройките на рекламите за всеки от вашите планове_

В тази таб ще можете да добавите рекламните кодове за публикации и страници в фронтенда, преди и след съдържанието.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_След като вмъкнете кода на рекламата си, тя ще се покаже в публикациите и страниците на този конкретен план._

Можете също така да включите реклами в заглавието (хедера) на административните страници за потребителите на този план.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_Отидете на опцията Admin Panel Ad Code, ако искате да покажете реклами в бекенда на потребителя си_

### Повече от рекламирането {#beyond-advertising}

Можете също така да използвате Ultimate Multisite: Ad Injector, за да предложите поддръжка на планове с висок клас.

Просто отидете на опцията _Admin Panel Ads_ и добавете HTML кода за инструмента за поддръжка.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_Добавете клиентска поддръжка към конкретни планове_

Така хората могат да се свържат с вас директно от своя Admin Panel!
