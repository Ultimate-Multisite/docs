# Кастомизиране на панела за управление с PRO Theme {#customizing-your-admin-panel-with-pro-theme}

Често когато работим с клиенти, имаме нужда да им предоставим персонализирано преживяване в панела за управление (dashboard) на WordPress. Ако търсите по-професионален вид и усещане, PRO Theme може да ви помогне.

PRO Theme е административна тема, разработена, за да ви помогне при кастомизирането на панела за управление на вашите клиенти. Освен това, тя е част от премиум ядро на плагините на Ultimate Multisite.

След инсталиране, отидете на страницата си _Plugins_ и натиснете, за да го активирате в мрежата (Network).

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Активирайте го за цялата мрежа на страницата Plugins_

_**ЗАБЕЛЕЖКА:**_ _Ако просто го инсталирате, той ще се покаже на клиентите ви като опция, която може да бъде маркирана, за да добави темата само за този потребител._

Сега имате административен интерфейс, който е напълно преработен! Имате и ново меню в лявата колона, наречено PRO Theme.
