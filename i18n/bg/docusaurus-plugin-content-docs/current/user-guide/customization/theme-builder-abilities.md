# Способности за изграждане на теми: Създаване и активиране на блок теми {#theme-builder-abilities-scaffold-and-activate-block-themes}

Superdav AI Agent v1.12.0 въвежда две мощни способности, които ви позволяват да генерирате и внедрявате персонализирани блок теми директно от чат интерфейса.

## Преглед {#overview}

Способностите **scaffold-block-theme** и **activate-theme** позволяват на агентите да:
- Генерират пълни, готови за работа блок теми въз основа на вашите спецификации
- Автоматично активират теми върху сайта ви, без нужда от ръчна намеса
- Създават свързани визуални идентичности чрез насочени дизайнерски решения

## Създаване на Блок Тема (Scaffold Block Theme) {#scaffold-block-theme}

Способността **scaffold-block-theme** генерира нова WordPress блок тема с пълна структура, включваща:

- Конфигурация `theme.json` с дизайн токени
- Файлове за блок шаблони за често срещани разположения
- Персонализирани стилове и вариации на блокове
- Метаданни и декларации за поддръжка на темата

### Как се извиква {#how-to-invoke}

В чата си с Superdav AI Agent можете да поискате генериране на тема:

```
"Create a block theme called 'Modern Agency' with a blue and white color scheme,
sans-serif typography, and a professional layout"
```

Агентът ще:
1. Събере вашите дизайнерски предпочитания чрез разговор
2. Генерира пълната структура на темата
3. Създава всички необходими файлове на темата
4. Подготвя темата за активиране

### Очакван резултат {#expected-output}

Когато способността работи успешно, ще видите:

- Потвърждение, че темата е изградена (scaffolded)
- Име и местоположение на темата
- Обобщение на приложените дизайн токени (цветове, типография, интервали)
- Статус "Готов за активиране"

Примерен изход:
```
✓ Theme "Modern Agency" scaffolded successfully
  Location: /wp-content/themes/modern-agency/
  Colors: Primary #0066CC, Secondary #FFFFFF
  Typography: Inter (sans-serif)
  Status: Ready to activate
```

## Активиране на Тема (Activate Theme) {#activate-theme}

Способността **activate-theme** превключва сайта ви към новоизградена или съществуваща блок тема.

### Как се извиква {#how-to-invoke-1}

След изграждане на тема, можете да я активирате веднага:

```
"Activate the Modern Agency theme"
```

Или активирайте която и да е съществуваща тема:

```
"Switch to the Twentytwentyfour theme"
```

### Очакван резултат {#expected-output-1}

Когато активирането е успешно:

- Потвърждение за активната тема
- Име на предишната тема (за справка)
- URL на сайта, където темата е активна
- Всякакви бележки за настройка, специфични за темата

Примерен изход:
```
✓ Theme activated successfully
  Active theme: Modern Agency
  Previous theme: Twentytwentyfour
  Live at: https://yoursite.com
  Note: Check your homepage to verify the layout
```

## Работен процес: Изграждане и Активиране {#workflow-scaffold-and-activate}

Типичен работен процес комбинира двете способности:

1. **Поискане за генериране на тема**: "Create a block theme for my SaaS landing page"
2. **Агентът изгражда темата**: Генерира файлове и дизайн токени
3. **Преглед и доработка**: Обсъждате промени в дизайна, ако е необходимо
4. **Активиране**: "Activate the theme now"
5. **Проверка**: Посетете сайта си, за да потвърдите, че новият дизайн е активен

## Дизайн Токени и Кастомизация {#design-tokens-and-customization}

Изградените теми използват WordPress дизайн токени (чрез `theme.json`) за:

- **Цветове**: Основен (Primary), вторичен (Secondary), акцентен (Accent), неутрален (Neutral) палета
- **Типография**: Семейства на шрифтове, размери, тегла, височина на редовете
- **Интервал**: Падинг (Padding), марджин (Margin), мащаби на интервали (Gap scales)
- **Граници**: Токени за радиус и ширина
- **Сенки**: Нива на повдигане (Elevation levels)

Тези токени са централизирани в `theme.json`, което улеснява настройването на цялата ви дизайн система от един файл.

## Ограничения и Бележки {#limitations-and-notes}

- Темите се изграждат в `/wp-content/themes/` и трябва да следват конвенциите за именопис на WordPress
- Активирането изисква подходящи права на вашия WordPress сайт
- Кастомният PHP код в темите е минимален; използвайте плагини за сложна функционалност
- Блок темите работят най-добре с WordPress 5.9 и по-късно

## Отстраняване на неизправности {#troubleshooting}

**Темата не се появява след изграждане**
- Проверете дали директорията на темата съществува и има правилни права
- Проверете дали `theme.json` е валиден JSON
- Уверете се, че името на темата не влиза в конфликт с вече съществуващи теми

**Активирането се проваля**
- Потвърдете, че имате администраторски права
- Проверете дали директорията на темата е четима от WordPress
- Прегледайте логовете за грешки на WordPress за подробности

**Дизайн токените не се прилагат**
- Проверете дали синтаксисът на `theme.json` е коректен
- Изчистете всички кеширащи плагини (caching plugins)
- Проверете дали версията на WordPress поддържа токените, които използвате

## Следващи стъпки {#next-steps}

След активиране на темата можете да:
- Използвате **Design System Aesthetics skill**, за да подобрите типографията, цветовете и интервалите
- Кастомизирате отделни стилове на блокове чрез WordPress block editor
- Добавите кастомен CSS във файла `style.css` на темата
- Създадете кастомни блок шаблони за специфични типове страници
