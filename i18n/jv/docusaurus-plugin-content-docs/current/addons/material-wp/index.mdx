# Material WP

Antarmuka admin WordPress wis padha wae nganti suwe banget. Nanging apa panjenengan tau mikir kepiye rupane yen antarmuka admin duwe versi Material Design? 

Material WP yaiku tema admin sing dikembangake kanggo menehi tampilan lan rasa kuwi, mbantu panjenengan nyedhiyakake Dashboard sing luwih dikustomisasi kanggo klien-klien panjenengan.

## Fitur

- Dhukungan CSS kustom
- Setelan gaya sing gampang
- Ngatur maneh menu
- Impor/ekspor sing gampang

## Dokumentasi

**Ngustomisasi Panel Admin panjenengan nganggo Material WP**

## Instalasi

1. Unggah 'material-wp' menyang direktori '/wp-content/plugins/'
2. Aktifake piranti tambahan liwat menu 'Piranti Tambahan' ing WordPress

## Pandhuan Panggunaan

Antarmuka admin WordPress wis padha wae nganti suwe banget. Nanging apa panjenengan tau mikir kepiye rupane yen antarmuka admin duwe versi Material Design?

Sawise **nginstal**, pindhah menyang kaca _Piranti Tambahan_ panjenengan lan klik kanggo ngaktifake ing Network panjenengan.
