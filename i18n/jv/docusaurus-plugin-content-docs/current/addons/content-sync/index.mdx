# Content Sync {#content-sync}

Multisite Ultimate: Content Sync

Addon iki kanggo Multisite Ultimate ngidini administrator jaringan nyinkronake konten saka situs apa wae menyang situs liyane ing jaringan. Iki ndhukung kabeh jinis kiriman kalebu kiriman, kaca, jinis kiriman kustom, lan template Elementor. Addon iki kalebu antarmuka tuntunan pirang-pirang langkah kanggo milih konten lan situs target, kanthi pamrosesan batch lan umpan balik kemajuan wektu nyata.

## Fitur {#features}

* Pamilihan Konten: Kisi kertu visual kanthi thumbnail lan panyaringan jinis kiriman
* Kabeh Jinis Kiriman: Nyinkronake kiriman, kaca, produk, template Elementor, lan jinis kiriman kustom apa wae
* Pamilihan Situs: Pilih situs endi sing arep disinkronake kontené, kanthi pangecualian situs sumber lan pra-pamilihan situs anak
* Pamrosesan Batch: Nyinkronake menyang 10 situs saben wektu kanthi nganyari kemajuan
* Sinkronisasi Media: Kanthi otomatis nyalin lan ngatur maneh pemetaan berkas media (gambar, video) saka konten kiriman menyang situs target
* Sinkronisasi Taksonomi: Njaga kategori, tag, lan taksonomi kustom
* Dhukungan Elementor: Dhukungan lengkap kanggo template Elementor kanthi regenerasi CSS lan pemetaan maneh ID template
* Resolusi Konflik: Opsi kanggo ngganti konten sing wis ana utawa ngliwati
* Manajemen Cache: Kanthi otomatis mbusak cache Elementor ing situs target sawise sinkronisasi

## Syarat {#requirements}

* WordPress 5.3 utawa luwih dhuwur
* PHP 7.4 utawa luwih dhuwur
* Plugin Multisite Ultimate (aktif)
* Plugin Elementor (opsional, dibutuhake kanggo sinkronisasi template Elementor)

## Instalasi {#installation}

1. Unggah 'ultimate-multisite-content-sync' menyang direktori '/wp-content/plugins/'
2. Aktifake plugin liwat menu 'Plugins' ing WordPress Network Admin sampeyan
3. Atur setelan ing Ultimate Multisite → Settings → Content Sync
4. Navigasi menyang Ultimate Multisite → Content Sync kanggo miwiti nyinkronake
