# Content Sync Changelog {#content-sync-changelog}

### Versi 1.1.0 - Dirilis ing 2025-12-31 {#version-110---released-on-2025-12-31}
* Anyar: Dhukungan kanggo nyelarasake kabeh jinis kiriman (kiriman, kaca, jinis kiriman kustom)
* Anyar: Panyaring jinis kiriman ing langkah pilihan konten
* Anyar: Langkah pilihan situs sumber - selarasake saka situs apa wae, ora mung situs utama
* Anyar: Pra-pilihan situs anak kanggo situs template
* Anyar: Ekstraksi media saka post_content kanggo kabeh jinis kiriman
* Anyar: Panyelarasan taksonomi (kategori, tag, taksonomi kustom)
* Diowahi: Ganti jeneng saka "Elementor Template Sync" dadi "Content Sync"
* Diowahi: Langkah "Pilih Template" diganti jeneng dadi "Pilih Data"
* Ditambah apik: Kode khusus Elementor saiki mung mlaku kanggo jinis kiriman elementor_library

### Versi 1.0.0 - Dirilis ing 2025-12-19 {#version-100---released-on-2025-12-19}
* Rilis wiwitan
* Wisaya multi-langkah kanggo pilihan template lan situs
* Pangolahan batch kanthi umpan balik kemajuan wektu nyata
* Panyelarasan media lengkap kanthi pemetaan ulang ID otomatis
* Integrasi setelan karo Ultimate Multisite
* Dhukungan pangecualian situs liwat setelan jaringan
* Mbusak cache Elementor sawise sinkronisasi
* Njaga taksonomi template
