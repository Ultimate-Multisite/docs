# Pangatur Plugin & Tema

Ultimate Multisite: Pangatur Plugin lan Tema

Sunting utawa dhelikake informasi meta (judhul, pangripta, gambar cilik lan katrangan) saka plugin lan tema sing wis dipasang. Gawe kategori kanggo plugin lan tema supaya pangguna sampeyan bisa nyaring lan nambah kaca Plugin kustom sing apik menyang panel pangguna sampeyan.

Kanggo nggunakake plugin iki, sampeyan cukup navigasi menyang kaca Plugin utawa Tema ing admin jaringan sampeyan lan klik "Sunting info Plugin/Tema".

## Instalasi {#installation}

1. Unggah 'ultimate-multisite-plugin-and-theme-manager' menyang direktori '/wp-content/plugins/'
2. Aktifake plugin liwat menu 'Plugin' ing WordPress Network Admin sampeyan

## Pandhuan Panggunaan {#usage-guide}

Ultimate Multisite: Pangatur Plugin lan Tema iku add-on Ultimate Multisite **premium** sing kuat lan menehi sampeyan kontrol lengkap marang cara pangguna jaringan sampeyan ndeleng lan sesambungan karo opsi plugin lan tema ing jaringan sampeyan.

Add-on iki nduweni telung tujuan:

  1. Iki ngidini sampeyan ngganti metadata saka plugin lan tema kanthi lengkap. Iki kalebu jeneng, katrangan, jeneng pangripta, lan malah gambar cilik!

  2. Iki ngidini sampeyan nggolongake plugin lan tema, supaya pangguna sampeyan gampang ngurutake.

  3. Iki menehi sampeyan opsi kanggo ngganti kaca plugin lawas sing mboseni nganggo kaca anyar sing mirip tema kanthi pangurutan miturut kategori.

Sawise diundhuh lan dipasang, tab anyar - _Setelan Add-on_ \- bakal katon ing kaca Setelan Ultimate Multisite.

Ing kaca _Plugin_ jaringan sampeyan, saiki sampeyan bakal bisa nyunting kabeh informasi meta plugin, kayata jeneng, pangripta, lan malah gambar cilik plugin.
