# PayFast {#payfast}

Multisite Ultimate: Payfast Gateway

Nambahake Payfast - gerbang pambayaran online sing cepet lan aman ing Afrika Kidul - minangka pilihan pambayaran Multisite Ultimate.

## Instalasi {#installation}

1. Unggah 'ultimate-multisite-payfast' menyang direktori '/wp-content/plugins/'
2. Aktifake plugin liwat menu 'Plugins' ing WordPress
3. Tindakake Wizard langkah demi langkah kanggo nyetel plugin

## Pandhuan Panggunaan {#usage-guide}

PayFast yaiku pangolah pambayaran online utama kanggo bisnis ing Afrika Kidul. Iki darmabakti kanggo mbantu pedagang online tuwuh kanthi nggawe transaksi dadi lancar lan aman. Miturut panjaluk pangguna Ultimate Multisite, kita mutusake nggawe add-on sing bakal ndadekake integrasi karo PayFast bisa ditindakake. Artikel iki bakal nuduhake carane nyetel PayFast minangka gerbang pambayaran kanggo nampa pambayaran ing jaringan Ultimate Multisite sampeyan.

Langkah kapisan yaiku nginstal add on.

# Carane nginstal add-on Payfast: {#how-to-install-the-payfast-add-on}

Menyang kaca add-ons banjur pilih add-on Ultimate Multisite: Payfast Gateway.

![Kaca add-ons](/img/admin/addons-page.png)

Jendhela add-on bakal muncul. Klik Install Now.

![Kaca add-ons nganggo pilihan instalasi](/img/admin/addons-page.png)

Sawise instalasi rampung, sampeyan bakal dialihake menyang kaca plugins. Ing kene, cukup klik Network Activate lan add-on _Payfast_ bakal diaktifake ing jaringan sampeyan.

<!-- Gambar layar ora kasedhiya: kaca WP Plugins nuduhake tombol Network Activate kanggo addon PayFast -->

# Nyetel PayFast {#setting-up-payfast}

Sawise nginstal add-on, sampeyan saiki bisa menyang setelan Ultimate Multisite lan miwiti ngonfigurasi PayFast.

Ing kaca **setelan Ultimate Multisite**, sampeyan bisa nemokake tab "Payments". Ing tab iki, sampeyan kudu ndeleng "**Payfast** " bebarengan karo tombol toggle kanggo ngaktifake utawa mateni gerbang pambayaran iki.

![Setelan gerbang pambayaran](/img/config/settings-payment-gateways.png)Terusake lan klik tombol toggle kanggo ngaktifake. Banjur kothak bakal ngembang kanggo nuduhake macem-macem field sing kudu sampeyan isi kayata **Mechant ID, Merchant Key lan PassPhrase.** Sampeyan bisa entuk informasi iki saka Account **Payfast** sampeyan

<!-- Gambar layar ora kasedhiya: setelan PayFast sing digedhekake nuduhake field Merchant ID, Merchant Key, lan PassPhrase -->

Langkah sabanjure yaiku mlebu menyang Account PayFast sampeyan kanggo njupuk informasi sing dibutuhake.

Sawise mlebu, sampeyan bisa langsung ndeleng kredensial sing dibutuhake saka dashboard Account **Payfast** sampeyan.

<!-- Gambar layar ora kasedhiya: dashboard Account PayFast nuduhake kredensial Merchant ID lan Merchant Key -->Sampeyan mung bisa nyalin lan nempel kredensial banjur ngisi rincian ing setelan Ultimate Multisite sampeyan. Banjur klik tombol **"Save settings** " kanggo nyimpen owah-owahan.

<!-- Gambar layar ora kasedhiya: setelan PayFast Ultimate Multisite nganggo kredensial sing wis diisi lan tombol Save -->

Wis rampung, integrasi pambayaran PayFast sampeyan wis rampung!
