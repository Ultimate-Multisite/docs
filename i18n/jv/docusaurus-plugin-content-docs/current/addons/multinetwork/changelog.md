# Cathetan Owah-owahan Multi-Network {#multi-network-changelog}

### 1.3.0 {#130}
* Anyar: Pratinjau template jaringan — telusuri lan pratinjau template jaringan nganggo panel langsung sadurunge tuku.
* Anyar: Pamilih template jaringan ing checkout — pelanggan bisa milih template jaringan nalika alur checkout signup.
* Anyar: SSO tautan ajaib antarjaringan — pelanggan bisa ngakses subjaringan kanthi mulus liwat autentikasi magic-link.
* Anyar: Kloning media jaringan — media situs diduplikasi kanthi bener nalika ngloning jaringan saka template.
* Ndandani: Kloning jaringan saiki bisa mlaku tanpa addon multi-tenancy; hak istimewa super admin diwenehake maneh sawisé kloning.
* Ndandani: Pembuatan jaringan saiki kompatibel karo WooCommerce lan kabeh gateway pambayaran.
* Ndandani: Nomer port saiki kalebu ing pembuatan domain fallback.
* Ndandani: CSS pamilih template dipindhah menyang stylesheet sing dienqueue kanggo rendering sing bener ing kabeh theme.
* Ndandani: Pangayoman saka kunci `network_id` sing ora ditemtokake ing cakupan kueri situs.
* Ndandani: Priksa idin jaringan sadurunge ngalih konteks nyegah kasalahan properti tabel `wpdb`.
