# VAT

Multisite Ultimate: VAT Éropah

Nambah fitur lan piranti tambahan menyang pangaturan pajeg gawan Multisite Ultimate kanggo luwih ndhukung VAT Éropah.
Kanthi otomatis nambah tarif EU VAT menyang kaca Tarif Pajeg saka

## Pamasangan

1. Unggah 'ultimate-multisite-vat' menyang direktori '/wp-content/plugins/'
2. Aktifake plugin liwat menu 'Plugins' ing WordPress
3. Pindhah menyang Ultimate Multisite > Setelan > Pajeg lan aktifake EU VAT.
4. Klik ing Tarif Pajeg lan Nganyari Tarif EU VAT ing sisih ngisor banjur Simpen Tarif Pajeg.
5. Rampung!
