# Superdav AI Agent

Superdav AI Agent yaiku asisten AI WordPress tingkat lanjut sing nggedhekake Gratis AI Agent nganggo katrampilan sing ngerti tema, kemampuan nggoleki internet sing ditambah, fitur pambangun situs, lan integrasi otomatisasi opsional. Versi 1.18.0 nambahake piranti maca Google Calendar, pemetaan kontak lan peserta, gerbang persetujuan manungsa, deduplikasi pangeling, kabar SMS TextBee, panyedhiyan layanan Superdav sing dikelola, lan paket pendamping Advanced sing didistribusikake kapisah kanggo alur kerja pangembang sing dipercaya.

## Fitur Utama

- **Katrampilan sing ngerti tema** — kanthi otomatis nyetel pandhuan lan kemampuan adhedhasar tema WordPress aktif lan panyambung sing wis dipasang
- **Pirang-pirang panyedhiya telusur internet** — pilih antarane Brave Search lan Tavily kanggo asil telusur sing luwih sugih lan kemampuan riset tingkat lanjut
- **Formulir kontak pambangun situs** — gawe formulir kontak sing fungsional kanthi lengkap langsung saka antarmuka chat nganggo panyambung formulir sing wis sampeyan pasang
- **Pambangun situs sing ditambah** — rencana pambangunan situs multi-tahap sing luwih apik kanthi panemuan panyambung lan pemulihan kasalahan sing luwih apik
- **Katrampilan tema bawaan** — pandhuan khusus kanggo Tema Blok, Tema Klasik, Kadence Blocks, lan Kadence Theme
- **Integrasi WooCommerce sing luwih apik** — direfaktor supaya nganggo API WooCommerce asli kanggo keandalan lan kompatibilitas sing luwih apik
- **Deteksi panyedhiya otomatis** — kanthi otomatis ndeteksi lan nyegerake panyambung lan tema sing kasedhiya nalika diaktifake utawa dipateni
- **Nggawe logo kustom** — gawe lan sematake logo SVG kustom kanthi sanitasi otomatis sing aman namespace
- **Validasi kontras palet** — priksa pasangan warna kanggo kepatuhan aksesibilitas WCAG sadurunge ditrapake menyang tema sampeyan
- **Panemuan sing ditambah foto** — unggah foto sajrone wawancara panemuan Theme Builder kanggo mengaruhi arah desain
- **Menu hospitality** — gawe kaca menu panganan lan omben-omben terstruktur kanggo restoran, kafe, lan bisnis hospitality
- **Pratinjau desktop lan mobile** — pratinjau desain sampeyan ing piranti desktop lan mobile sajrone pilihan arah desain
- **Scraping situs Tier 1** — sd-ai-agent/site-scrape saiki kasedhiya kanthi gawan kanggo nganalisis situs web pesaing lan inspirasi
- **Piranti maca Google Calendar** — maca kalender lan acara sing wis dikonfigurasi supaya otomatisasi bisa nalar babagan konteks jadwal sing bakal teka
- **Pemetaan kontak lan peserta** — cocogake peserta kalender karo pangguna situs lan kontak sing wis dikenal sadurunge nggawe tugas tindak lanjut
- **Gerbang persetujuan manungsa** — ngaso otomatisasi sensitif kanggo ditinjau sadurunge ngirim pesen, nerusake tugas, utawa nggawe owah-owahan
- **Cathetan pangeling lan deduplikasi** — lacak pangeling sing wis dikirim supaya eksekusi otomatisasi bola-bali ora menehi kabar marang panampa sing padha kaping pindho
- **Kabar SMS TextBee** — kirim kabar pesen teks sing wis dikonfigurasi liwat TextBee nalika panyedhiya SMS diaktifake
- **Panyedhiyan layanan Superdav sing dikelola** — sambungake situs sing didhukung menyang titik pungkasan Superdav sing di-host kanthi persiyapan otomatis tinimbang konfigurasi titik pungkasan manual
- **Paket pendamping Advanced** — piranti pangembang dipercaya sing didistribusikake kapisah kanggo alur kerja filesystem, database, WP-CLI, dispatcher REST, pambangun panyambung, snapshot git, manajemen pangguna, lan benchmark

## Syarat

- WordPress 6.0 utawa luwih dhuwur
- PHP 8.1 utawa luwih dhuwur
- WordPress Multisite (diaktifake jaringan) utawa instalasi situs tunggal
- Panyambung panyedhiya AI aktif (Anthropic, OpenAI, utawa sing kompatibel)
- Gratis AI Agent v1.9.0 utawa luwih dhuwur (Superdav AI Agent nggedhekake Gratis AI Agent)

## Instalasi

1. Unggah berkas panyambung menyang `/wp-content/plugins/superdav-ai-agent`
2. Aktifake panyambung liwat layar **Plugins** ing WordPress
3. Pesthekake Gratis AI Agent uga diaktifake
4. Navigasi menyang **Gratis AI Agent → Settings** lan konfigurasi panyedhiya telusur internet, sambungan layanan Superdav sing dikelola, Google Calendar, utawa kredensial TextBee miturut kabutuhan

Paket WordPress.org ngemot fitur inti Superdav AI Agent. Paket pendamping Advanced dibangun lan didistribusikake kapisah kanggo lingkungan ing ngendi piranti pangembang dipercaya dipasang lan diwenehi wewenang kanthi eksplisit.

## Katrampilan Sing Ngerti Tema

Superdav AI Agent kanthi otomatis ndeteksi tema aktif lan panyambung sing wis dipasang, banjur ngemot katrampilan sing ngerti tema sing cocog menyang konteks asisten. Katrampilan sing kasedhiya kalebu:

| Katrampilan | Ditrapake Kanggo |
|---|---|
| **Tema Blok** | Tema sing nggunakake arsitektur WordPress adhedhasar blok (Full Site Editing) |
| **Tema Klasik** | Tema WordPress tradisional sing nggunakake cithakan PHP lan editor klasik |
| **Kadence Blocks** | Situs kanthi panyambung Kadence Blocks aktif |
| **Kadence Theme** | Situs kanthi Kadence theme minangka tema aktif |

Delengen [Katrampilan Sing Ngerti Tema](../../user-guide/configuration/theme-aware-skills) kanggo pandhuan rinci babagan saben katrampilan.

## Panyedhiya Telusur Internet

Superdav AI Agent v1.10.0 nambahake **Tavily** minangka alternatif kanggo Brave Search kanggo pitakon telusur internet. Kaloro panyedhiya nawakake asil telusur wektu nyata lan bisa dikonfigurasi ing **Gratis AI Agent → Settings → Advanced**.

| Panyedhiya | Fitur | Setup |
|---|---|---|
| **Brave Search** | Asil panggolèkan cepet, fokus marang privasi | [Brave Search API](https://brave.com/search/api/) |
| **Tavily** | Asil panggolèkan luwih sugih, kapabilitas riset majeng | [Tavily API](https://tavily.com/) |

Delengen [Panggolèkan Internet](../../user-guide/configuration/internet-search) kanggo pandhuan konfigurasi.

## Alur Kerja Kalender, Persetujuan, lan Notifikasi

Superdav AI Agent v1.18.0 ngembangaké alur kerja otomatisasi ngluwihi tugas kontèn lan pambangunan situs:

- **Piranti Google Calendar** ngidini agent maca kalender sing wis dikonfigurasi lan rincian acara nalika kredensial kalender kasedhiya.
- **Pambantu kontak lan peserta** memetakan peserta acara menyang pangguna WordPress utawa kontak sing dikenal sadurungé nyiapaké pangeling utawa tugas tindak lanjut.
- **Gerbang persetujuan manungsa** ngidini otomatisasi mandheg nganti pangguna sing nduwé wewenang mriksa tumindak sing diusulaké.
- **Cathetan pangeling** nyegah pangeling kalender utawa notifikasi dobel nalika otomatisasi dicoba manèh utawa dijadwalaké bola-bali.
- **Dhukungan SMS TextBee** ngaktifaké notifikasi pesen teks nalika panyedhiya TextBee dikonfigurasi.

Gunakna piranti iki kanggo pambantu sing ngerti jadwal, tindak lanjut peserta, lan alur kerja pangeling sing isih mbutuhaké tinjauan manungsa sadurungé pesen sensitif dikirim.

## Konfigurasi

Kabèh setelan ana ing **Gratis AI Agent → Settings** ing admin WordPress. Superdav AI Agent nambahaké pilihan konfigurasi ing ngisor iki:

| Setelan | Katrangan |
|---|---|
| **Panyedhiya Panggolèkan Internet** | Pilih antarane Brave Search lan Tavily (opsional) |
| **Tavily API Key** | API key kanggo panggolèkan internet Tavily (yen Tavily dipilih) |
| **Layanan Superdav sing Dikelola** | Sambungaké situs sing didhukung menyang layanan Superdav sing di-host kanthi panyedhiyan endpoint otomatis |
| **Kredensial Google Calendar** | Konfigurasi akses kalender supaya agent bisa maca kalender lan acara |
| **Panyedhiya SMS TextBee** | Konfigurasi kredensial TextBee kanggo alur kerja notifikasi SMS sing disetujoni |

## Paket Pendamping Advanced

Superdav AI Agent v1.18.0 nambahaké paket pendamping Advanced sing didistribusèkaké kapisah kanggo alur kerja pangembang sing dipercaya. Kapabilitas iki ora dibundel ing paket inti WordPress.org. Pasang paket pendamping mung ing situs sing administratoré kanthi cetha percaya marang agent kanggo ngakses piranti pangembang.

Piranti Advanced kalebu operasi sistem berkas, akses basis data, WP-CLI, alur kerja dispatcher REST, utilitas pambangun plugin, snapshot git, pambantu manajemen pangguna, lan piranti benchmark. Watesi piranti iki mung kanggo administrator sing dipercaya lan tinjau konfigurasi gerbang persetujuan sadurungé ngaktifaké ing situs produksi.
