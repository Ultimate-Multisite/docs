# Pamilih Basa {#language-selector}

Ultimate Multisite: Pamilih Basa nalika Ndhaftar

Tambahake kolom pamilih basa menyang alur ndhaptar.

## Instalasi {#installation}

1. Unggah 'ultimate-multisite-language-selector' menyang direktori '/wp-content/plugins/'
2. Aktifake plugin liwat menu 'Plugin' ing WordPress Network Admin sampeyan

## Pandhuan Panggunaan {#usage-guide}

Kanthi tambahan gratis iki, sampeyan bisa ngidini pangguna milih basa sing dikarepake kanggo dideleng ing alur ndhaptar lan ing kaca mlebu.

Sawise [ngaktifake lisensi sampeyan](../../user-guide/getting-started/installing-ultimate-multisite.md) sampeyan bakal bisa nginstal tambahan Ultimate Multisite sing diwenehake akses dening lisensi sampeyan.

Menyang kaca setelan Ultimate Multisite sampeyan banjur, ing menu ing sisih kiwa, klik label _Tambahan_.

![Kaca tambahan](/img/admin/addons-page.png)

_Menyang kaca Tambahan kanggo miwiti nginstal Pamilih Basa_

Ing kana, sampeyan bakal nemokake kabeh tambahan sing ditawakake.

Tambahan gratis kasedhiya kanggo kabeh pelanggan kita, tanpa gumantung marang rencana langganan sing dipilih nalika ndhaptar.

Klik tombol _Instal Saiki!_ ing ngisor gambar cilik tambahan. Banjur aktifake ing kaca Plugin.

![Kaca tambahan nganggo tombol instal](/img/admin/addons-page.png)

_Klik tombol kanggo nginstal tambahan_

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Language Selector addon -->

_Aktifake ing kaca Plugin sampeyan_

Saiki, menyang menu Setelan Ultimate Multisite banjur mlebu tab _Setelan Network_. Ing kana, ing sisih ngisor kaca, sampeyan bakal nemokake bagean anyar sing diarani _Basa_.

![Kaca setelan](/img/config/settings-general.png)

_Bagean anyar katon!_

Sampeyan bisa ninggalake kosong kanggo nampilake kabeh basa minangka pilihan utawa milih endi wae sing pengin digawe kasedhiya. Basa-basa kasebut yaiku sing wis sampeyan instal ing pangaturan WordPress sampeyan.

<!-- Screenshot unavailable: Language selection dropdown showing available languages on WP install -->

_Dhaptar basa sing kasedhiya ing instalasi WP iki_

Sawise nyimpen owah-owahan sing sampeyan gawe, menu pilihan nganggo basa sing sampeyan pilih bakal ditampilake ing kaca Mlebu uga ing Alur Ndhaftar.

![Setelan mlebu lan registrasi](/img/config/settings-login-registration.png)

_Pamilih Basa ing Kaca Mlebu_

![Kaca registrasi](/img/config/registration-page.png)

_Pamilih Basa ing Alur Ndhaftar_

## Nyetel Basa liwat URL {#setting-the-language-via-the-url}

Sampeyan uga bisa ngganti basa sawijining kaca kanthi ngirim parameter locale menyang URL. Yen sampeyan pengin ngalihake menyang kaca ndhaptar nganggo basa Italia, contone, tambahake ?locale=it_IT menyang URL.

[https://yournetwork.com/wp-signup.php?locale=it_IT](https://yournetwork.com/wp-signup.php?locale=it%5FIT)

Basa kasebut kudu wis diinstal ing network WordPress sampeyan supaya iki bisa mlaku.
