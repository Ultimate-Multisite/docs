# Saringan: signup_get_available_languages

Nyaring dhaptar basa sing kasedhiya kanggo pendaftaran situs ing front-end.

Nerusaké array kosong menyang hook iki bakal mateni output setelan ing formulir pendaftaran, lan basa gawan bakal digunakaké nalika nggawe situs. Basa sing durung diinstal bakal dibusak.

## Paramèter {#parameters}

| Jeneng | Jinis | Katrangan |
|------|------|-------------|
| $available_languages | `array` | Basa sing kasedhiya. |
