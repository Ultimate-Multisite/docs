# Fluent Forms {#fluent-forms}

Add-on iki ndadekake bisa nambah watesan kanggo Fluent Forms ing Multisite Ultimate ing produk, keanggotaan utawa situs.
Fluent Forms nggunakake tabel kustom tinimbang jinis kiriman kustom mula opsi watesan biasa ora bisa digunakake karo Fluent Forms.

## Fitur {#features}

- Tambah watesan kanggo Fluent Forms utawa Conversational Fluent Forms
- Pelanggan entuk pesen sing ramah sing njaluk supaya padha nganyarke plan nalika tekan watesan
