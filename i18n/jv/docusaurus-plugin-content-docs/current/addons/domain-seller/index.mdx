# Domain Seller

**Multisite Ultimate: Domain Seller** ngowahi jejaring WordPress Multisite panjenengan dadi platform adol domain sing kuwat. Ayo pelanggan ndhaptar domain kustom langsung nalika sign-up utawa saka Dashboard, kanthi pamriksaan kasedhiyan wektu-nyata lan rega sing cetha.

### Napa Domain Seller?

Pendaftaran domain minangka salah siji upsell sing paling kerep dijaluk kanggo platform WaaS (Website as a Service) apa wae. Tinimbang ngirim pelanggan panjenengan menyang registrar pihak katelu, panjenengan bisa nawakake domain langsung — njaga luwih akeh pangasilan ing jejaring panjenengan sekaligus menehi pengalaman sing tanpa gesekan marang pelanggan.

### Fitur Utama

#### Dhukungan Pirang-pirang Registrar Domain
* **OpenSRS** — Pendaftaran, nganyari, lan pangelolaan lengkap
* **Namecheap** — Integrasi lengkap karo WhoisGuard lan pengujian sandbox
* **HostAfrica** — Panelusuran domain, rega, pendaftaran, transfer, DNS, lan perlindhungan ID liwat Domains Reseller API
* **Openprovider** — Rega reseller, pendaftaran, nganyari, transfer, DNS, privasi WHOIS, lan sinkronisasi TLD
* **Hostinger** — Panelusuran lan pendaftaran domain nganggo token integrasi Hostinger sing dienggo bareng
* **Arsitektur sing Bisa Dikembangake** — Dibangun nganggo pola Integration Registry, supaya gampang nambah panyedhiya liyane
* **Pengujian Sambungan Gawan** — Validasi kredensial panyedhiya lan konektivitas sadurunge diluncurake

#### Wisaya Setup sing Nuntun
* **Onboarding sak-klik** — Nuntun panjenengan liwat konfigurasi panyedhiya, impor TLD, lan nggawe produk domain
* **Sinkronisasi TLD otomatis** — Ngimpor kabeh TLD sing kasedhiya lan rega grosir saka panyedhiya sing wis disambungake
* **Pembuatan produk baku** — Nggawe produk domain siap digunakake kanthi standar markup sing wajar

#### Rega Domain sing Fleksibel
* **Telung mode markup** — Markup persentase, markup tetep, utawa rega tetep saben TLD
* **Produk khusus TLD** — Gawe produk domain kapisah kanggo klompok TLD sing beda (.com/.net, .io, kode negara, lsp.)
* **Rega pambuka** — Nawakake rega taun kapisan sing didiskon kanggo narik pendaftaran anyar
* **Rega wektu-nyata** — Rega grosir langsung saka panyedhiya kanthi caching cerdas
* **Dhukungan kupon** — Kompatibilitas lengkap karo kode diskon Multisite Ultimate

#### Perlindhungan Privasi WHOIS
* **Konfigurasi saben produk** — Setel privasi WHOIS dadi Dipateni, Tansah Klebu, utawa Pilihan Pelanggan
* **Opt-in pelanggan** — Nalika disetel dadi Pilihan Pelanggan, kothak centhang nalika proses tuku ngidini pelanggan nambah perlindhungan privasi
* **Rega sing bisa dikonfigurasi** — Setel rega saben taun kanggo add-on privasi, utawa tawakake gratis
* **Integrasi registrar** — Kanthi otomatis ngaktifake WhoisGuard (Namecheap) utawa privasi WHOIS (OpenSRS) nalika pendaftaran

#### Pengalaman Tuku sing Manunggal
* **Telung mode domain** — Pelanggan milih antarane subdomain (gratis), ndhaptar domain anyar, utawa mapping domain sing wis diduweni
* **Panelusuran domain wektu-nyata** — Pamriksaan kasedhiyan nganggo AJAX tanpa muat ulang kaca
* **Saran cerdas** — Saran TLD alternatif nalika domain sing dikarepake wis dijupuk
* **Rega sing cetha** — Tampilan cetha kanggo rega pendaftaran, rega nganyari, lan biaya privasi WHOIS opsional
* **Field kontak registrant** — Ngumpulake info registrant sing dibutuhake (jeneng, alamat, telpon) langsung nalika proses tuku
* **URL situs sing digawe otomatis** — Alamat situs kanthi otomatis dijupuk saka domain sing dipilih

#### Sistem Nganyari Otomatis
* **Nganyari sing kagandheng karo membership** — Domain nganyari otomatis nalika membership pelanggan isih aktif
* **Penjadwalan cerdas** — Tugas nganyari dilebokake ing antrian adhedhasar tanggal kadaluwarsa domain
* **Logika nyoba maneh** — Nyoba maneh kanthi cerdas kanggo upaya nganyari sing gagal
* **Notifikasi email** — Tandha sing bisa dikonfigurasi kanggo nganyari sing bakal teka, nganyari sukses, kegagalan, lan kadaluwarsa

#### Pangolahan DNS Pelanggan
* **DNS mandiri** — Pelanggan bisa nambah, nyunting, lan mbusak rekaman DNS (A, AAAA, CNAME, MX, TXT) saka Dashboard
* **Status domain** — Status pendaftaran wektu-nyata, tanggal kadaluwarsa, lan informasi nameserver
* **Mapping domain otomatis** — Domain sing didaftar utawa sing wis ana kanthi otomatis dimapping menyang situs pelanggan

#### Piranti Admin
* **Pendaftaran domain manual** — Ndhaptar domain atas jeneng pelanggan saka panel admin
* **Pemantauan domain** — Deleng kabeh domain pelanggan, status, lan tanggal kadaluwarsa saka siji kaca
* **Logging khusus panyedhiya** — Kanal log kapisah kanggo saben registrar (umpamane, domain-seller-namecheap.log)
* **Rincian pendaftaran** — Deleng info registrant lan data pendaftaran ing kaca nyunting domain inti

### Panyedhiya sing Didukung

* **OpenSRS** — Integrasi lengkap karo pendaftaran, nganyari, pangolahan DNS, lan privasi WHOIS
* **Namecheap** — Integrasi lengkap karo pendaftaran, nganyari, pamriksaan kasedhiyan batch, lan WhoisGuard gratis
* **HostAfrica** — Pendaftaran, nganyari, transfer, pangolahan DNS, njupuk kode EPP, registrar lock, lan perlindhungan ID
* **Openprovider** — Pendaftaran, nganyari, transfer, pangolahan DNS, njupuk kode EPP, registrar lock, privasi WHOIS, lan sinkronisasi TLD reseller
* **Hostinger** — Pendaftaran, lookup domain, nganyari nameserver, registrar lock, lan privasi WHOIS; nganyari ditangani dening auto-renew Hostinger
* **GoDaddy** — Pendaftaran lan nganyari
* **ResellerClub** — Pendaftaran, nganyari, lan pangolahan DNS
* **NameSilo** — Pendaftaran lan nganyari
* **Enom** — Pendaftaran, nganyari, lan pangolahan DNS

### Rincian Teknis

* **Mbutuhake:** PHP 7.4+, Multisite Ultimate 2.4.12+
* **Arsitektur:** Pola Integration Registry kanthi antarmuka Domain_Selling_Capability
* **Autoloading:** Autoloader Jetpack kanthi pemetaan kelas PSR-4
* **Standar:** Standar coding WordPress, strict typing, prepared statements

## Instalasi

1. Pesthekake **Multisite Ultimate v2.4.12+** wis dipasang lan diaktifake ing jaringan
2. Unggah panyambung menyang `/wp-content/plugins/ultimate-multisite-domain-seller/`
3. Aktifake panyambung ing jaringan saka **Panyambung** ing Admin Jaringan WordPress
4. **Wisaya Persiyapan** mbukak kanthi otomatis nalika aktivasi pisanan — tindakake langkah-langkah kanggo:
   * Nyambungake registrar domain sampeyan
   * Nguji sambungan
   * Ngimpor TLD sing kasedhiya lan rega grosir
   * Nggawe produk domain pisanan sampeyan nganggo markup rega
5. Tambah kolom **Pamilihan Domain** menyang formulir pambayaran sampeyan ing **Ultimate Multisite > Formulir Pambayaran**

## Pitakonan sing Asring Ditakokake

### Versi Multisite Ultimate apa sing dibutuhake?

Versi 2.4.12 utawa luwih dhuwur. Tambahan iki ora kompatibel karo WP Ultimo v1.x utawa versi Ultimate Multisite ing ngisor 2.4.12.

### Panyedhiya domain endi wae sing didhukung?

Sangang registrar didhukung: **OpenSRS**, **Namecheap**, **HostAfrica**, **Openprovider**, **Hostinger**, **GoDaddy**, **ResellerClub**, **NameSilo**, lan **Enom**. Kapabilitas beda-beda miturut panyedhiya amarga API registrar nyedhiyakake operasi sing beda; delengen pandhuan persiyapan kanggo ringkesan saben panyedhiya. Arsitektur Registry Integrasi tambahan iki nggawe gampang kanggo nambah panyedhiya khusus — delengen [pandhuan pangembang](/developer/integration-guide/custom-registrar).

### Apa aku bisa nawakake pangayoman privasi WHOIS?

Ya. Saben produk domain duwe setelan Privasi WHOIS nganggo telung mode: Dipateni, Tansah Kalebu, utawa Pilihan Pelanggan. Nalika disetel dadi Pilihan Pelanggan, pelanggan ndeleng kothak centhang nalika pambayaran nganggo regane (utawa "Gratis" kanggo WhoisGuard saka Namecheap). Tandha privasi kanthi otomatis diterusake menyang registrar nalika registrasi.

### Apa aku bisa nyetel rega sing beda kanggo TLD sing beda?

Ya. Gawe produk domain kapisah kanggo klompok TLD sing beda. Saben produk bisa duwe jinis markup dhewe (persentase, markup tetep, utawa rega tetep), TLD sing didhukung, lan rega pambuka.

### Opsi pambayaran apa sing dideleng pelanggan?

Kolom Pamilihan Domain menehi pelanggan telung tab: **Subdomain** (nggunakake subdomain gratis ing jaringan sampeyan), **Ndaftar Domain Anyar** (nggoleki lan ndaftar domain anyar), lan **Domain sing Wis Ana** (memetakake domain sing wis diduweni). Mung mode sing sampeyan aktifake sing bakal katon.

### Apa ndhukung nganyari otomatis?

Ya. Domain bisa disetel supaya nganyari otomatis lan digandhengake karo status keanggotaan pelanggan. Pambayaran nganyari diproses liwat gateway pambayaran pelanggan sing wis ana.

### Apa pelanggan bisa ngatur cathetan DNS?

Ya. Pelanggan entuk antarmuka pangelolaan DNS ing dashboard kanggo nambah, nyunting, lan mbusak cathetan A, AAAA, CNAME, MX, lan TXT. Pangolahan DNS kasedhiya karo OpenSRS, ResellerClub, Enom, HostAfrica, lan Openprovider. DNS Hostinger kanggo domain sing di-host ditangani dening integrasi inti pemetaan domain Hostinger.

### Apa ana wisaya persiyapan?

Ya. Wisaya persiyapan mlaku kanthi otomatis nalika aktivasi pisanan lan nuntun sampeyan nyambungake registrar, ngimpor TLD, lan nggawe produk domain pisanan sampeyan. Sampeyan uga bisa mbukak maneh wisaya saka setelan tambahan.
