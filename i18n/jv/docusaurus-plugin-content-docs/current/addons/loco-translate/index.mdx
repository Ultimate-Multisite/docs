# Loco Translate

Tambahan Integrasi Ultimate Multisite: Loco Translate nggawa kapabilitas terjemahan khusus situs menyang jaringan multisite WordPress panjenengan. Yen panjenengan tau mbutuhake situs beda ing jaringan panjenengan nduweni terjemahan beda kanggo plugin utawa tema sing padha, tambahan iki dadi solusi sing sampurna.

**Apa Sing Ditindakake Tambahan Iki**

Tambahan iki nggabungake Ultimate Multisite kanthi mulus karo plugin Loco Translate sing populer, supaya saben subsite ing jaringan panjenengan bisa njaga terjemahan kustom dhewe. Apa panjenengan nglakokake platform multi-regional, nglayani komunitas basa sing beda-beda, utawa mung mbutuhake keluwesan terjemahan ing saindhenging jaringan panjenengan, tambahan iki nggawe kabeh dadi gampang.

**Cocog Kanggo:**

* Bisnis multi-regional kanthi kabutuhan lokalisasi sing beda-beda
* Lembaga pendhidhikan sing nglayani komunitas maneka warna
* Platform SaaS sing nawakake dhukungan multibasa
* Agensi web sing ngatur pirang-pirang situs klien
* Jaringan multisite apa wae sing mbutuhake keluwesan terjemahan

## Instalasi {#installation}

1. Pesthekake Ultimate Multisite wis diinstal lan diaktifake ing jaringan
2. Instal lan aktifake plugin Loco Translate
3. Unggah tambahan iki menyang direktori plugins panjenengan
4. Aktifake tambahan Integrasi Loco Translate ing jaringan
5. Miwiti nggawe terjemahan khusus situs liwat Loco Translate ing subsite apa wae

Wis ngono wae! Integrasi iki bisa mlaku otomatis tanpa konfigurasi tambahan apa wae.

## Pitakonan Sing Asring Ditakokake {#frequently-asked-questions}

## Apa aku kudu ngonfigurasi apa wae sawise instalasi? {#do-i-need-to-configure-anything-after-installation}

Ora! Integrasi iki otomatis. Sawise tambahan lan Loco Translate loro-lorone aktif, subsite bisa langsung miwiti nggawe terjemahan kustom dhewe.

## Apa sing kedadeyan yen subsite ora nggawe terjemahan kustom? {#what-happens-if-a-subsite-doesnt-create-custom-translations}

Situs kasebut bakal otomatis nggunakake terjemahan saindhenging jaringan minangka cadangan. Ora ana sing rusak, lan kabeh bisa mlaku kanthi mulus.

## Apa subsite bisa ngakses terjemahan saindhenging jaringan? {#can-subsites-access-network-wide-translations}

Admin situs biasa mung ndeleng lan nyunting terjemahan situs dhewe. Super admin bisa ngakses terjemahan saindhenging jaringan lan khusus situs gumantung situs endi sing lagi dideleng.

## Apa iki bisa digunakake karo plugin lan tema? {#does-this-work-with-both-plugins-and-themes}

Ya! Terjemahan khusus situs bisa digunakake kanggo plugin lan tema WordPress.

## Apa iki bakal nggawe jaringan multisite-ku dadi alon? {#will-this-slow-down-my-multisite-network}

Ora. Tambahan iki nggunakake pemuatan berkas sing efisien sing mung mriksa terjemahan khusus situs nalika dibutuhake. Ora ana pengaruh kinerja sing katon.

## Apa aku bisa nyalin terjemahan jaringan menyang situs tartamtu minangka titik wiwitan? {#can-i-copy-network-translations-to-a-specific-site-as-a-starting-point}

Ya! Nalika nyunting terjemahan ing Loco Translate, panjenengan bisa kanthi gampang miwiti saka terjemahan jaringan lan ngatur khusus kanggo saben situs.

## Apa iki bisa digunakake karo WooCommerce lan plugin liyane? {#does-this-work-with-woocommerce-and-other-plugins}

Mesthi! Plugin utawa tema apa wae sing nggunakake fungsi terjemahan standar WordPress bakal bisa digunakake karo terjemahan khusus situs.
