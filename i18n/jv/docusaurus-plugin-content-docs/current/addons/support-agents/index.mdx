# Agen Dhukungan {#support-agents}

Ngatur sakabehing jaringan dhewe dudu tugas sing gampang. Nalika jaringan sampeyan wiwit tuwuh, sampeyan bakal wiwit weruh panjaluk dhukungan sing saya mundhak. Becike sampeyan ngajak wong sing bisa mbantu nyuda beban kerja sampeyan. Nanging, menehi akses/idin lengkap iku beboyo lan bisa mbebayani jaringan sampeyan. Ing kene tambahan Agen Dhukungan iki migunani.

Tambahan iki ngidini sampeyan nggawe jinis pangguna WordPress anyar sing diarani Support Agent, lan menehi sampeyan kontrol lengkap marang idin apa wae sing diduweni pangguna kuwi. Menehi idin gampang kaya nguripake tombol toggle lan mateni maneh yen sampeyan mutusake arep nyopot idin kasebut.

## Fitur {#features}

- Dhaptar rinci idin WordPress & Ultimate Multisite
- Aktifake & nonaktifake akses kanthi gampang
- Gawe akun Agent tanpa wates jumlahé

## Instalasi {#installation}

1. Unggah 'ultimate-multisite-support-agents' menyang direktori '/wp-content/plugins/'
2. Aktifake tambahan liwat menu 'Plugins' ing WordPress
3. Tindakake Wizard langkah demi langkah kanggo nyetel tambahan kasebut
