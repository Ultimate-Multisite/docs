# Ngelola Pelanggan {#managing-customers}

Halaman **Customers** nun bakal nampikake daftar anggota sing wis terdaftar ing jaringan sampeyan. Halaman iki nuduhake informasi dasar pengguna kaya jeneng, alamat email, kapan terakhir mlebu (last login), lan link **switch to** sing ngidini sampeyan mlebu menyang subsite pengguna minangka administrator situs.

![Customers list page](/img/admin/customers-list.png)

Klik **Manage** ing ngisor kartu pelanggan kanggo ndelok informasi rinci babagan pelanggan kasebut.

Ing halaman detail pelanggan, blok **At a Glance** nuduhake ID pelanggan, wektu mlebu terakhir, lan total jumlah sing wis dienggo dening pelanggan iki.

Blok **Customer Options** ngidini sampeyan nggawe pelanggan iki dadi **VIP** (tag kanggo nggawe dhewe luwih gampang ditemokake). Tab **Billing Info** nuduhake informasi penagihan kaya email, alamat, lan negara.

Blok **Fields** nuduhake username pelanggan, email, lan ngidini sampeyan ngowahi status **Email Verification**.

Ing ngisor blok Customer Options bakal ana papat blok tambahan:

- **Memberships** — Langganan saiki sing dianggur dening pelanggan
- **Payments** — Riwayat pembayaran pelanggan
- **Sites** — Situs-situs sing duwe pelanggan iki
- **Events** — Acara-acara sing terkait karo pelanggan iki (verifikasi email, perubahan penagihan, lsp.)

Klik ikon panah (→) ing sisih saben item kanggo navigasi menyang halaman detailé. Contone, klik panah ing sisih situs bakal nggawa sampeyan menyang halaman edit situs yaiku papan panggonan sing bisa sampeyan modifikasi asosiasi keanggotaan, tipe situs, lan status aktif.

Iki uga berlaku kanggo memberships lan payments — saben link ngarah menyang halaman detail sing bisa sampeyan edit status keanggotaan, pelanggan sing terkait, produk, pengaturan payment gateway, lan bahkan proses pengembalian dana (refund).

Ing sisih tengene halaman pelanggan, sampeyan bakal nemokake:

- **Login Terakhir & IP** — Wektu login terakhir, alamat IP, lan negara
- **Stempel Waktu (Timestamps)** — Kapan akun pelanggan dibuat lan diubah terakhir
- **Hapus (Delete)** — Pilihan kanggo ngapus pelanggan permanen
