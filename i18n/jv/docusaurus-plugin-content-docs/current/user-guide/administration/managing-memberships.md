# Ngelola Keanggotaan {#managing-memberships}

Ing Ultimate Multisite, sampeyan bisa ngatur keanggotaan lan akses kabeh langganan ing jaringan sampeyan. Tinjau lan edit detail langganan kaya rencana (plans) lan produk, jumlah situs, jumlah tagihan lan frekuensinya, riwayat pembayaran, lan stempel waktu (timestamps).

Kanggo nggo ngakses halaman iki, tindakake tab **Memberships** ing menu Ultimate Multisite.

![Memberships list page](/img/admin/memberships-list.png)

Ing halaman Memberships, sampeyan bisa ndelok daftar pelanggan lan produk sing ana gandheng cenenge karo akun mereka. Sampeyan bisa filter keanggotaan miturut status: aktif (active), pending, on hold, kedaluwarsa (expired), utawa dibatalkan (cancelled).

Keanggotaan gratis diperlakukan minangka keanggotaan **seumur hidup** (lifetime). Keanggotaan iki ora bakal duwe tanggal kedaluwarsa lan ora bakal kedaluwarsa otomatis. Yen rencana gratis kudu rampung sawise uji coba (trial) utawa periode tartamtu, gunakake produk berbayar kanthi masa percobaan (trial), batalkan keanggotaan manual, utawa pindhahake pelanggan menyang rencana liyane nalika aksesé perlu owah.

Klik ing keanggotaan kanggo nggo ngakses halaman sing bisa sampeyan edit detail keanggotaan — informasi penagihan (billing info), batasan situs utawa kunjungan (site or visit limits), lan liyane.

![Membership edit interface](/img/admin/membership-edit.png)

Iki tampilan lengkap saka halaman edit keanggotaan:

![Membership edit full interface](/img/admin/membership-edit-full.png)

Sampeyan uga bisa nambah keanggotaan anyar sing ana gandheng cenenge karo pengguna sing wis terdaftar ing jaringan sampeyan. Klik **Add Membership** lan atur detailé.

![Add Membership modal](/img/admin/membership-add-modal.png)

Terus klik **Create Membership**. Keanggotaan anyar bakal muncul ing daftar keanggotaan.
