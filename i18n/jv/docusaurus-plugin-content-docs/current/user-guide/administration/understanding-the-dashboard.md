# Ngerti Dashboard Ultimate Multisite {#understanding-the-ultimate-multisite-dashboard}

Saben dhisik kita mlebu ing pengaturan sing luwih canggih lan aspek operasional Ultimate Multisite, mangga ndelok dhisik dashboard-e.

Kaya akeh plugin WordPress liyane, Ultimate Multisite nggawe menu tingkat jaringan kanthi jeneng **Ultimate Multisite** karo sawetara sub-menu ing ngisoré.

## Widget Dashboard Jaringan (Network Dashboard Widgets) {#network-dashboard-widgets}

Ultimate Multisite nyuntikake sawetara widget sing migunani menyang dashboard tingkat jaringan. Dashboard iki bisa ditemokake kanthi menehi wejangan menyang **My Sites → Network Admin → Dashboard**. Widget-widget kuwi digawe kanggo menehi akses gampang marang informasi lan operasi umum kanggo administrator jaringan.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

### Langkah-langkah Kapisan (First Steps) {#first-steps}

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

Widget iki bakal muncul nalika instalasi anyar Ultimate Multisite lan ngelingake administrator jaringan kanggo ngrampungake tugas tambahan kaya ngatur metode pembayaran lan nggawe akun uji coba.

### Ringkesan (Summary) {#summary}

![Summary widget](/img/admin/network-dashboard-summary.png)

Widget Summary lapor babagan jumlah pendaftar lan pendapatan sing wis mlebu kanggo dina kuwi. MRR (Monthly Recurring Revenue) nuduhake total perkiraan kanggo pelanggan kanthi keanggotaan sing nggawe pendapatan.

### Aliran Aktivitas (Activity Stream) {#activity-stream}

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

Widget iki lapor babagan kedadeyan ing sak kabeh Ultimate Multisite. Kedadeyan kalebu pendaftaran, pembatalan, pembuatan situs, penghapusan, lan aktivitas liyane.

Mangga deleng bagian Events ing dokumentasi kanggo deskripsi sing luwih rinci.

### Saiki (Right Now) {#right-now}

![Right Now widget](/img/admin/network-dashboard-right-now.png)

Widget utilitas iki nuduh ringkesan singkat babagan user lan situs ing jaringan kasebut. Shortcut sing gampang nang dhuwur ngidini akses siji klik kanggo nggawe situs utawa user anyar.

### Berita lan Diskusi {#news-and-discussions}

![News and Discussions widget](/img/admin/network-dashboard-news.png)

Widget iki njupuk lan nuduh berita Ultimate Multisite paling anyar. Wenehi perhatian marang widget iki kanggo informasi babagan pembaruan, bug penting, lan perbaikan keamanan.

## Ultimate Multisite Dashboard {#ultimate-multisite-dashboard}

Ing kene dashboard tingkat jaringan nuduh informasi utuh babagan jaringan, dashboard Ultimate Multisite (sing ana ing dhuwur menu Ultimate Multisite) nuduh informasi babagan layanan kasebut.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

### Pertumbuhan Pendapatan Berulang Bulanan (Monthly Recurring Revenue) {#monthly-recurring-revenue-growth}

Grafik MRR (Pendapatan Berulang Bulanan) nuduh rincian bulanan pendapatan sajrone periode kalender 12 sasi.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

Pendapatan anyar dilacak nganggo warna biru, dene pembatalan katon nganggo warna abang (merah).

### Dukungan Rentang Tanggal {#date-range-support}

Kanthi jumlah pelanggan lan aktivitas, statistik bisa dadi rame banget. Dukungan rentang tanggal sing wis digawe ing dashboard ngidini administrator jaringan fokus mung marang informasi lan periode wektu sing dibutuhake.

![Date range selector and stats](/img/admin/um-dashboard-date-range.png)

Rentang tanggal iki ngatur periode informasi sing ditampilake dening widget ing ngisor, kalebu pendapatan, anggota anyar, pendaftaran per negara, pendaftaran per formulir, lan situs paling akeh dikunjungi.

![Dashboard statistics widgets](/img/admin/um-dashboard-stats.png)

:::tip
Kanggo nggawe widget **Most Visited Sites** bisa makarya, aktifake penghitungan kunjungan ing **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**.
:::
