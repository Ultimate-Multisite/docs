# Tinjauan Dashboard {#dashboard-overview}

Ultimate Multisite duwe **Dashboard** khusus di bawah menu Ultimate Multisite, sekaligus menambahkan widget ke Network Dashboard. Artikel iki bakal njelasake apa sing bakal sampeyan deleng ing kedua panggonan kuwi.

## Network Dashboard {#the-network-dashboard}

Ultimate Multisite ngetokake widget-widget sing migunake ing dashboard level jaringan sing ditemokake ing **My Sites → Network Admin → Dashboard**.

![Network Dashboard with Ultimate Multisite widgets](/img/admin/network-dashboard-full.png)

Iki isine saben widget:

**First Steps:** Muncul nalika instalasi anyar lan ngelingake sampeyan kanggo ngrampungake tugas persiapan kaya konfigurasi metode pembayaran lan nggawe akun uji coba.

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Summary:** Lapor babagan jumlah pendaftar lan pendapatan saben dina. MRR (Monthly Recurring Revenue) nuduh total perkiraan kanggo pelanggan kanthi keanggotaan sing nggawa pendapatan.

![Summary widget](/img/admin/network-dashboard-summary.png)

**Activity Stream:** Melaporkan kedadeyan ing sakabehing Ultimate Multisite — pendaftaran, pembatalan, pembuatan situs, penghapusan, lan aktivitas liyane.

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** Nampilake ringkesan singkat babagan pengguna lan situs ing jaringan kanthi pintasan kanggo nggawe situs utawa pengguna anyar.

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** Ngambil lan nampilake berita Ultimate Multisite sing paling anyar kalebu pembaruan, bug kritis, lan perbaikan keamanan.

![News widget](/img/admin/network-dashboard-news.png)

## Dashboard Ultimate Multisite {#the-ultimate-multisite-dashboard}

Sanajan network dashboard nuduh informasi utuh babagan jaringan, dashboard Ultimate Multisite (ing tingkat paling dhuwur menu Ultimate Multisite) nuduh informasi babagan layanan kasebut.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Pertumbuhan Pendapatan Berulang Bulanan (Monthly Recurring Revenue Growth):** Menampilkan rincian pendapatan bulanan selama periode kalender 12 bulan. Pendapatan baru akan ditampilkan dalam warna biru sementara pembatalan muncul dalam warna merah.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**Dukungan Rentang Tanggal (Date-range Support):** Saring data dashboard berdasarkan periode waktu untuk fokus pada informasi yang Anda butuhkan. Rentang tanggal mengontrol apa yang ditampilkan di widget di bawah ini.

![Date range and dashboard stats](/img/admin/um-dashboard-date-range.png)

Widget dashboard meliputi:

- **Pendapatan (Revenue)** — Total pendapatan untuk periode tersebut beserta rincian MRR dan per produk
- **Keanggotaan Baru (New Memberships)** — Berapa banyak keanggotaan baru yang dibuat dalam periode yang dipilih
- **Pendaftaran Berdasarkan Negara (Signups by Countries)** — Rincian geografis dari pendaftaran
- **Pendaftaran Berdasarkan Formulir (Signups by Forms)** — Berapa banyak pelanggan yang mendaftar melalui setiap formulir checkout dibandingkan dengan yang ditambahkan oleh admin
- **Situs Paling Dikunjungi (Most Visited Sites)** — Situs yang paling sering dikunjungi di jaringan Anda (membutuhkan pengaktifan penghitungan kunjungan di **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting**)

![Dashboard statistics](/img/admin/um-dashboard-stats.png)
