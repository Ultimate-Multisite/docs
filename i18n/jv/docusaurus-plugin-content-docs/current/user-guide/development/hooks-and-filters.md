# Ultimate Multisite Hooks and filters {#ultimate-multisite-hooks-and-filters}

Ultimate Multisite kuwi is digawe saka akeh banget fungsi lan query database. Iki duwe hook lan filter sing bakal ngidini sampeyan ngowahi utawa nambah kode anyar kanggo fungsi sing wis ana.

Kita gawe daftar hook lan filter iki minangka referensi supaya pengguna sing luwih canggih bisa ngowahi jaringan (network) dewe luwih jero maneh. Sampeyan bisa akses daftar kuwi saka Github kanthi klik link ing ngisor iki.
