# Ngatur Panel Adminmu nganggo PRO Theme {#customizing-your-admin-panel-with-pro-theme}

Sering nalika nggawe kerja kanggo klien, kita butuh menehi pengalaman khusus marang klien ing dashboard admin WordPress. Yen sampeyan lagi nyari tampilan lan rasa sing luwih profesional, PRO Theme bisa mbantu sampeyan.

PRO Theme iku tema admin sing dikembangake kanggo mbantu sampeyan nalika ngatur Dashboard klienmu. Lan iki uga bagian saka inti plugin premium Ultimate Multisite.

Sawise diinstal, tindakna menyang halaman _Plugins_ lan klik kanggo aktifake ing Network-mu.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_Aktifke ing sak jaringan sampeyan ing halaman Plugins_

_**CATATAN:**_ _Yen sampeyan mung nginstal, iku bakal muncul ing klienmu minangka pilihan sing bisa ditandai kanggo nambah tema mung kanggo pengguna kuwi._

Saiki, antarmuka adminmu wis didesain ulang total! Sampeyan uga duwe menu anyar ing kolom kiwa sing jenenge PRO Theme.
