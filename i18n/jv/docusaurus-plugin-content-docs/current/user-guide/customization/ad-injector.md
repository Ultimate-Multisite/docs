# Nempat Iklan ing jaringanmu nganggo Ultimate Multisite: Ad Injector

Ultimate Multisite: Ad Injector iku **add-on gratis** sing nganti sampeyan duwe piranti kanggo nempat iklan ing rencana (plan) langsung saka halaman edité.

Sawise instalasi Ultimate Multisite: Ad Injector, tindakake menyang halaman _Plugins_ sampeyan kanggo ngaktifake nganggo add-on kasebut.

<!-- Screenshot ora tersedia: Halaman WP Plugins nuduh tombol Network Activate kanggo add-on Ad Injector -->

_Klik tombol Network Activate_

Sawise add-on wis aktif, kotak meta _Advanced Options_ ing layar edit rencana bakal duwe pilihan anyar sing jenenge _Ad Settings_.

![Ad Settings ing layar edit rencana](/img/config/product-edit-full.png)

_Saiki sampeyan bisa ngedit pengaturan iklan kanggo saben rencana._

Ing tab kasebut, sampeyan bakal bisa nambah kode iklan kanggo postingan lan halaman ing front-end, sadurunge lan sawise konten.

<!-- Screenshot ora tersedia: Lapangan kode iklan kanggo postingan lan halaman sadurunge konten -->

<!-- Screenshot ora tersedia: Conto iklan sing ditampilkan ing halaman postingan kanggo rencana tartamtu -->

_Sawise ngetokake kode iklanmu, iku bakal muncul ing postingan lan halaman rencana tartamtu._

Sampeyan uga bisa nggawe iklan kalebu ing header Halaman Admin kanggo pengguna saka rencana kasebut.

<!-- Screenshot ora tersedia: Pilihan Kode Iklan Panel Admin ing pengaturan rencana -->

<!-- Screenshot ora tersedia: Conto iklan sing ditampilkan ing panel admin pengguna -->

_Tindakake pilihan Kode Iklan Panel Admin yen sampeyan arep nampilake iklan ing back-end pengguna._

### Luwih saka Iklan

Sampeyan uga bisa nggunakake Ultimate Multisite: Ad Injector kanggo menehi dukungan marang rencana tingkat dhuwur (high-tier plans).

Cukup tindakake pilihan _Admin Panel Ads_ lan tambahi kode HTML kanggo alat dukunganmu.

<!-- Screenshot ora tersedia: Pilihan Admin Panel Ads kanthi kode HTML alat dukungan -->

<!-- Screenshot tidak tersedia: Widget dukungan pelanggan ditampilkan di panel admin untuk paket tertentu -->

_Tambah dukungan pelanggan ke paket tertentu_

Dengan cara ini, orang-orang bisa langsung menghubungi Anda dari Admin Panel mereka!
