# Website Klien Sampeyan (v2) {#your-clients-websites-v2}

Sawise daftar ing jaringan sampeyan, pelanggan bisa ngakses dashboard-e.

Salajengipun saka halaman **Account**, sing ana informasi penting babagan pembayaran, keanggotaan, domain, batasan paket lan liyane, dheweke uga bisa akses halaman **My Sites**, ing ngendi dheweke bisa ndelok lan ngatur kabeh website-e. Sing kudu ditindakake mung klik My Sites ing dashboard-e.

![Link My Sites ing dashboard pelanggan](/img/account-page/my-sites-link.png)

Kabeh website-e bakal ditunjuka, karo tombol **Manage** tepat ing ngisor saben website. Yen diklik, dheweke bakal diarahkan menyang **dashboard website spesifik kasebut**.

Dheweke uga bisa ndelok website endi sing dadi **primary** kanggo paket sing ngidini duwe luwih saka siji website.

![Halaman My Sites nuduhake website pelanggan karo tombol Manage](/img/account-page/my-sites-page.png)

Iki tampilan lengkap antarmuka edit website:

![Antarmuka edit website lengkap](/img/admin/site-edit-full.png)

Yen arep ngowahi website utama, dheweke kudu menyang halaman Account kanthi klik menu **Account**.

![Link menu Account ing dashboard pelanggan](/img/account-page/account-menu.png)

Dheweke uga bisa ndelok lan ngedit detail pelanggan:

![Antarmuka edit pelanggan](/img/admin/customer-edit.png)

Terus, dheweke kudu klik **Change Default Site**.

![Tombol Change Default Site ing halaman account](/img/account-page/change-default-site.png)

Sawijining jendela pop-up bakal muncul. Klik ing daftar drop-down lan pilih website endi sing arep sampeyan dadi website utama.

![Pemilihan dropdown Change Default Site](/img/account-page/change-default-site-dropdown.png)

Terus, cukup klik **Change Default Site** lan wis rampung!

![Konfirmasi perubahan situs default](https://example.com/images/account-page/change-default-site-confirm.png)
