# Kepiye carane aku bisa mlebu halaman akun kanggo pisanan? {#how-can-i-access-my-account-page-for-the-first-time}

Sugeng rawuh ing Ultimate Multisite!

Bisa mlebu halaman akunmu liwat link sing wis dikirim nang email sawise tuku plugin kita:

<!-- Screenshot unavailable: Email halaman akun karo link akses -->

Gunakake email sing wis sampeyan weneh nalika tuku lan password sing wis kok terima nang email.

<!-- Screenshot unavailable: Halaman login nganggo kolom email lan password -->

Amarga iki pisanan sampeyan mlebu akun iki, sampeyan bakal diminta kanggo ngowahi password. Ganti dadi liyane banjur klik tombol _Save_.

<!-- Screenshot unavailable: Formulir ganti password karo tombol Save -->

Ing halaman akunmu, sampeyan bisa ngatur kabeh detail akunmu kaya pembayaran (billings), perpanjangan (renewals), pembatalan (cancellations), lan akeh bab liyane.
