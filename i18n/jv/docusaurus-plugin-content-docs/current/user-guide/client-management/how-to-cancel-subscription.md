# Kepiye caraku mbatalake langganan?

Yen sampeyan duwe lisensi Ultimate Multisite, sampeyan bisa mbatalake perpanjangan kapan wae sampeyan karep. Cukup tindakake langkah-langkah ing ngisor iki:

<!-- Screenshot unavailable: Account page email with access link -->

Gunakake email sing wis sampeyan paring nalika tuku lan password sampeyan.

<!-- Screenshot unavailable: Login page with email and password fields -->

Iki layar awal Account Page panjenengan:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

Kanggo mbatalake langganan, ing menu ing kiwa, tindakna menyang _Renewing & Billing._

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

Klik panah ing tengen kanggo mbukak jendela samping. Banjur, sampeyan kudu milih pilihan sing bakal mbatalake **auto-renew**.

Sistem bakal nuduhake pesan konfirmasi marang sampeyan.

Yen sampeyan yakin arep mbatalake langganan, cukup klik tombol _**Cancel Renewals**_. Sawise tindakan iki, sampeyan bakal ditanya kanggo ngisi survei cekak.

Wis rampung! Langganan panjenengan ora bakal diperpanjang otomatis.

**Panjenengan bakal duwe kunci sing sah nganti langganan panjenengan kedaluwarsa**. Yen sampeyan arep ngaktifake maneh langganan, sampeyan kudu nindakake manual.
