# Ngatur Link yang Bisa Dibagikan kanggo Paket

Ultimate Multisite nawakake pilihan supaya sampeyan bisa bagi link paket langsung marang pelanggan. Supaya nglakoni iki, tindakna menyang salah siji halaman edit paketmu lan klik tombol _Click to copy the Shareable Link_.

![Product edit page with shareable link button](/img/config/product-edit-full.png)

_Ing Halaman Edit Paket, sampeyan bisa nemokake tombol kanggo nggawe link sing bisa dibagikan._

Cathet yen mekanisme iki nganggo frekuensi penagihan standar kanggo mbangun link. Iki tegese yen penagihan standar sampeyan sebulan, contone, ing akhir, sampeyan bakal duwe link kaya ngene: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

Yen panjenengan kepengin link sing bisa dibagikan dadi frekuensi kuartalan (3 bulanan), sampeyan kudu nambah 3 ing akhir

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

Kanggo milih paket _premium_ kan jadwal taun, sampeyan bakal nambah 1 banjur tembung:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

Ingat yen sampeyan ora bisa duwe mung siji link kanggo frekuensi penagihan sing beda-beda. Sampeyan kudu bagi siji link spesifik kanggo saben frekuensi penagihanmu.

Sawise sampeyan bagi link kuwi marang pelanggan, link kasebut bakal ngarahake marang langkah kapisan alur pendaftaran (signup flow) panjenengan.
