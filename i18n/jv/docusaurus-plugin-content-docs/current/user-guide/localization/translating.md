# Menerjemahkan Ultimate Multisite {#translating-ultimate-multisite}

Ultimate Multisite dikembangkan dalam bahasa Inggris, tapi plugin ini bisa diterjemahkan sepenuhnya dan kami sangat terbuka untuk kontribusi terjemahan.

## Terjemahan Plugin Inti {#core-plugin-translations}

Plugin inti Ultimate Multisite diterjemahkan melalui platform resmi **WordPress.org translation**:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

Ini adalah sistem terjemahan komunitas WordPress yang standar. Ketika terjemahannya sudah mencapai batas persetujuan yang cukup, mereka akan otomatis didistribusikan ke semua pengguna Ultimate Multisite melalui pembaruan plugin WordPress.

**Kami butuh bantuanmu!** Kalau kamu bisa bahasa selain Inggris, silakan pertimbangkan untuk ikut menerjemahkan. Bahkan menerjemahkan beberapa teks pun sangat membantu. Untuk mulai:

1. Kunjungi [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)
2. Pilih bahasamu
3. Klik "Contribute Translation" (Kontribusi Terjemahan)
4. Kamu perlu akun WordPress.org gratis

Kalau kamu mau jadi editor terjemahan untuk bahasamu (bisa menyetujui terjemahan), hubungi kami melalui [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions).

Yen sampeyan milih kanggo nggawe terjemahan dhewe kanggo proyekmu dewe tinimbang kontribusi marang sumber asline, ana rong pilihan:

### Nggunakake Poedit {#addon-translations}

Sampeyan bisa nggunakake aplikasi desktop **[Poedit](https://poedit.net)** kanggo nerjemah file `.pot`:

1. Unduh Poedit saka [poedit.net](https://poedit.net)
2. Bukak file `.pot` saka `wp-content/plugins/ultimate-multisite/lang/`
3. Terjemah string ing antarmuka Poedit
4. Simpan -- Poedit bakal ngasilake file `.po` lan `.mo`
5. Letakake file hasil output ing folder `lang`

### Nggunakake Loco Translate {#local-translations}

Plugin WordPress **[Loco Translate](https://wordpress.org/plugins/loco-translate/)** nggampangake sampeyan nerjemah langsung saka admin panel tanpa perlu alat desktop. Plugin iki bakal ngurusane nggawe lan kompilasi file terjemahan kanggo sampeyan.
