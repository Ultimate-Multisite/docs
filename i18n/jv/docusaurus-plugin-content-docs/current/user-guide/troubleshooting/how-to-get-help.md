# Cara Nyuwun Tulung {#how-to-get-help}

Sampeyan wis duwe License key sing sah, lan ana akses kanggo tim kita kanggo mbantu ngrampungake masalah sing mungkin ketemu nalika sampeyan lagi nggatekake Ultimate Multisite.

Pelajaran iki ngandharake saluran dukungan sing ana kanggo pemegang lisensi lan informasi jinis apa sing kudu sampeyan siapake kanggo nyuwun tulung marang tim dukungan kita.

## Help Site (Situs Bantuan) {#help-site}

Sampeyan bisa mlebu artikel lan tutorial beda saka situs bantuan kita sing bisa sampeyan temok ing link ing ngisor iki.

[)

Kita ngajak para penggunak supaya kenal karo fitur-fitur lan fungsi dasar Ultimate Multisite. Ana artikel "Getting Started" (Mulai) sing ana ing situs bantuan kita lan jawaban kanggo pitakonan umum uga bisa ditemokake ing kono, kaya cara instal add-ons, cara nggabungake payment gateway, lsp.

## Contact Support (Hubungi Dukungan) {#contact-support}

Yen sampeyan kebingan lan ora bisa nemokake jawabane saka situs bantuan kita, sampeyan bisa ngontak tim dukungan kanthi klik ikon gelem (speech bubble icon) sing ana ing sisih tengene ngisor situs web kita ATAU kanthi ngirim email langsung menyang support@ultimatemultisite.com.

Nalika ngontak dukungan kanggo masalah teknis, monggo pastikan sampeyan menehi deskripsi rinci babagan masalahmu kasebut karo screenshot, pesan error, utawa informasi apa wae sing bakal mbantu wong-wong dukungan kita supaya gampang mangerteni lan nimbang masalahmu. Sampeyan uga kudu menehi license key sampeyan. Mung pemegang lisensi aktif sing duwe hak kanggo dukungan.

## Community Site (Situs Komunitas) {#community-site}

[community site](https://community.ultimatemultisite.com/) kita minangka papan kanggo para pengguna Ultimate Multisite supaya bisa ngobrol karo saben liyan. Iki dadi sistem informasi ing ngendi anggota bisa posting, komen ing diskusi, menehi saran, kolaborasi, utawa nggoleki bantuan bareng (crowdsource). Sampeyan bakal dipersaroake menehi license key Ultimate Multisite sampeyan kanggo mbukak ruang/halaman pribadi.
