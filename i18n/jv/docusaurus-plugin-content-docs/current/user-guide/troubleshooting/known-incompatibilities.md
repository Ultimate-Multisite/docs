# Ketidakcocokan yang Diketahui dengan Plugin Lain

Setelah menginstal dan mengaktifkan Ultimate Multisite, kamu mungkin akan mengalami masalah karena ada ketidakcocokan yang belum terselesaikan dengan plugin lain.

Kami sedang giat bekerja untuk mengurangi daftar ini di setiap rilis baru, tapi kalau sebuah plugin tercantum di sini, kamu mungkin perlu menonaktifkannya supaya jaringan Ultimate Multisite bisa berfungsi sepenuhnya.

Ini daftarnya dan apa yang harus kamu lakukan dalam kasus seperti itu:

**Beberapa plugin WPMU DEV** Beberapa plugin yang _tidak lagi_ didukung oleh WPMU DEV diketahui menyebabkan alur registrasi Ultimate Multisite berhenti di langkah terakhir (pengguna tidak bisa membuat akun). Ini dia daftar plugin yang menyebabkan masalah tersebut:

  * Signup Language: _Ultimate Multisite punya add-on yang menawarkan fungsi yang sama, namanya Ultimate Multisite: Language Switcher._

  * Set Blog Description on Blog Creation: _Ini sudah ada di dalam Ultimate Multisite, jadi tidak perlu plugin tambahan._

  * New Blog Templates: _Ini juga sudah ada di dalam Ultimate Multisite, jadi tidak perlu plugin tambahan._

  * Set Password on Multisite Blog Creation: _Ini sudah ada di dalam Ultimate Multisite, jadi tidak perlu plugin tambahan._

  * Signup TOS: _Ini sudah ada di dalam Ultimate Multisite, jadi tidak perlu plugin tambahan._

  * Anti-Splog: Meskipun kurang kuat, ini juga _sudah ada di dalam Ultimate Multisite._

**Catatan**: Semua plugin WPMU Dev lainnya bisa digunakan secara normal bersama Ultimate Multisite. Contohnya termasuk _Smush_, _Forminator_, _Defender_, dll.
