# Pertanyaan yang Sering Diajukan

Iki beberapa **Pertanyaan yang Sering Diajukan** yang sering muncul saat ada permintaan dukungan.

  * **Apa itu Ultimate Multisite?**

Ultimate Multisite iku plugin WordPress, digawe khusus kanggo instalasi WordPress multisite, sing ngubah instalasi WordPress sampeyan dadi jaringan situs premium – kaya [WordPress.com](https://WordPress.com) – ngidini klien nggawe situs kanthi bayaran bulanan, kuartalan, utawa taunan (sampeyan uga bisa nggawe paket Gratis).

  * **Apa Ultimate Multisite bakal kerja tanpa Multisite?**

Ora. Ultimate Multisite mbutuh WordPress multisite, subdirectory, utawa subdomain.

  * **Apa syarat-syarat kanggo Ultimate Multisite?**

Ultimate Multisite mbutuh:

**Versi WordPress** : v5.3+ (Direkomendasikan: Versi Stabil Terakhir)
**Versi PHP** : 7.4.x (Dukungan menyang > 8.0 segera hadir)
**Versi MySQL** : v5+ (Direkomendasikan: Versi 5.6, yen sampeyan ora duwe 8.0 ing penyedia hostingmu)

_Multisite (subdomain utawa subdirectory) uga kudu diaktifake._ [Cara Instal WordPress Multisite](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **Apa Ultimate Multisite bakal kerja karo Multisite nganggo Subdirectory?**

Ya. Ultimate Multisite ndukung instalasi subdirectory lan instalasi subdirectory.

  * **Apa Ultimate Multisite bakal kerja karo Multisite nganggo Subdomain?**

Ya. Ultimate Multisite ndukung instalasi subdirectory lan instalasi subdomain.

  * **Apa aku bisa nggunakake Ultimate Multisite kanggo ngbatasi plugin lan tema sing bisa digunakake pengguna?**

Ya. Kanthi Ultimate Multisite sampeyan bisa milih plugin lan tema apa wae sing tersedia kanggo saben paket ing jaringanmu.

  * **Apa aku bisa nggunakake Ultimate Multisite kanggo ngbatasi jumlah postingan sing bisa digawe sawijining pengguna ing situsé?**

Ya. Dengan Ultimate Multisite, sampeyan bisa bikin kuota postingan kanggo **setiap tipe postingan** sing ana ing jaringan sampeyan. Sampeyan uga bisa ngetokake jumlah pengguna sing bisa diundang admin, **jumlah ruang disk**, lan akeh maneh.

  * **Apa Ultimate Multisite duwe dukungan Site Templates?**

Ya. Dengan Ultimate Multisite, sampeyan bisa nggawe sebanyak mungkin Site Template sing dikarepake. Site Template kuwi dudu tema, nanging situs biasa ing jaringan sampeyan. Sampeyan bisa ngaktifake plugin, tema, lan ngowahi Site Template kasebut kanthi cara apa wae sing dikarepake. Yen sawijining pengguna milih template kuwi, situsé bakal digawe kaya salinan saka template kuwi, karo kabeh penyesuaian wis ana.

Sampeyan uga bisa nentokake template tartamtu kanggo rencana (plan). Kanthi nggawe ngono, saben pengguna sing daftar ing rencana kuwi bakal duwe situsé digawe duplikat saka Site Template kuwi.

  * **Apa Ultimate Multisite nyambung karo WooCommerce?**

Ya!

Yen sampeyan pengin pengguna jaringan sampeyan bisa nganggo WooCommerce kanggo nggawe toko, iku bisa ditindakake nganggo Ultimate Multisite. Sampeyan uga bisa ngetokake jumlah produk sing bakal bisa digawe dening klien sampeyan ing saben tingkatan rencana (plan tier).

Kanggo pembayaran langganan kanggo Ultimate Multisite, sampeyan bisa nganggone [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) kanggo menehi ijin marang klien Ultimate Multisite sampeyan kanggo mbayar nganggo gateway WooCommerce ing situs utama sampeyan. Cara kuwi, sampeyan bisa nggoleki perpustakaan gedhe payment gateway sing dikembangake kanggo WooCommerce lan nganggone kasebut kanggo mbayar utang marang klien Ultimate Multisite sampeyan.

  * **Apa ana cara kanggo migrasi langganan saka WPMUDEV Pro Sites?**

Iya, Ultimate Multisite saiki nawakake sampeyan [alat bantu sing migrasi] (https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) kanggo jaringan Pro Sites panjenengan.

  * **Kenapa tab “Allowed Plugins” kosong pas saya edit Plan?**

Ultimate Multisite mung ngidini sampeyan menehi izin utawa nolak akses plugin sing ora diaktifake ing jaringan panjenengan. Nalika plugin wis aktif ing jaringan, saka sifaté dhewe, plugin kuwi bakal otomatis dimuat ing saben situs ing jaringan panjenengan.

Yen sampeyan ora nemokake plugin apa wae ing tab “Allowed Plugins”, coba cek apakah plugin sing dikarepake iku wis aktif ing jaringan utawa durung.

  * **Apa saya bisa nggunakake desain Tabel Harga (Pricing Table) saya dhewe?**

Bisa. Mulai versi 1.4.0, Ultimate Multisite ndukung *template overrides*.

  * **Apa Ultimate Multisite cocok karo WP Engine?**

Bisa. Ultimate Multisite wis tansah didukung dening WP Engine, nanging kanggo ngajak pemetaan domain (domain mapping) bisa kerja, butuh langkah tambahan manual saka admin (nambah domain ing panel WP Engine). **Iki ora masalah maneh wiwit versi 1.6.0.** Ultimate Multisite saiki langsung nyambung karo API WP Engine lan domain bakal otomatis ditambahkan menyang panel sawise pemetaan.

  * **Penyedia hosting sing sampeyan rekomendasi?**

Ultimate Multisite bisa kerja karo sapa wae penyedia hosting. Bedane mung, sawetara perusahaan hosting kaya Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane lan [Runcloud.io](http://Runcloud.io) duwe integrasi sing luwih ketat kanggo kemampuan pemetaan domain, mula dhewe wis bisa langsung kerja tanpa perlu setup tambahan. Kanggo hosting liyane, sampeyan mungkin perlu ngatur pemetaan domain manual.
