# Saringan: wu_minimum_password_length

Nyaring dawa minimal tembung sandhi.

Mung dileksanakake nalika wu_enforce_password_rules iku true.

## Paramèter

| Name | Type | Description |
|------|------|-------------|
| $min_length | `int` | Dawa minimal tembung sandhi. Gawan 12 (cocog karo Defender Pro). |
| $defender_active | `bool` | Apa Defender Pro Strong Password aktif. |
