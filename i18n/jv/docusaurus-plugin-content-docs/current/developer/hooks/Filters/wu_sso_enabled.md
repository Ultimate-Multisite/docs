# Saringan: wu_sso_enabled

Aktifake/pateni kapabilitas single-sign-on lintas-domain.

Saring aji iki kanggo mateni single-sign-on kanthi sakabehe, utawa ngaktifake kanthi kondisional minangka gantine.

## Paramèter

| Jeneng | Tipe | Katrangan |
|------|------|-------------|
| $enabled | `bool` | Apa SSO kudu diaktifake? True kanggo aktif, kaya false kanggo mati. |

### Wiwit

- 2.0.11
### Sumber

Ditetepake ing [`inc/sso/class-sso.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/sso/class-sso.php#L110) ing baris 110


## Wangsulan
Apa SSO diaktifake utawa ora.
