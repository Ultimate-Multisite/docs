# Saringan: wu_get_site_domain_and_path

Ngidini pangembang ngowahi pasangan domain/path.

Iki bisa migunani kanggo sawetara perkara, kayata ngetrapake sawernaning solusi staging, server sing beda-beda, lsp.

## Paramèter

| Jeneng | Jinis | Katrangan |
|------|------|-------------|
| $d | `object` | Objek saiki sing ngemot kunci domain lan path. |
| $path_or_subdomain | `string` | Path/subdomain asli sing diterusake menyang fungsi. |
