# Piwulang 4: Mbangun Cithakan Ceruk

Cithakan iku inti saka tawaran nilai ceruk panjenengan. Sing nduwé studio kabugaran sing ndaftar lan ndeleng situs sing wis katon kaya situs web kabugaran -- kanthi jadwal kelas, profil pelatih, lan gambar sing trep -- luwih gedhé kamungkinané bakal tetep tinimbang sing ndeleng kanvas kosong.

## Ngendi Kita Pungkasané Mandheg {#where-we-left-off}

Kita wis nduwé jaringan FitSite sing mlaku kanthi Ultimate Multisite wis dipasang lan dikonfigurasi. Saiki kita mbangun cithakan sing nggawe FitSite krasa dirancang khusus kanggo bisnis kabugaran.

## Napa Cithakan Ceruk Penting {#why-niche-templates-matter}

Cithakan umum meksa pelanggan panjenengan nindakake pagawean sing abot: nemtokake kaca apa sing dibutuhake, struktur konten apa sing cocog, lan kepiye nggawe katon pas kanggo industrié. Cithakan ceruk ngilangaké alangan kuwi.

Sing nduwé studio kabugaran sing ndaftar menyang FitSite kuduné ndeleng:

- Kaca ngarep sing katon kaya situs web studio kabugaran
- Kaca kanggo kelas, jadwal, pelatih, lan rega wis digawe
- Gambar lan konten placeholder sing cocog karo industrié
- Desain sing krasa profesional lan selaras karo merek kabugaran

Dheweke ngisi rinciané. Dheweke ora miwiti saka nol.

## Ngrancang Cithakan Panjenengan {#planning-your-templates}

Sadurungé mbangun, putusaké cithakan apa sing arep ditawaraké. Kanggo FitSite, kita bakal nggawe telu:

### Cithakan 1: Studio Essential {#template-1-studio-essential}

Kanggo studio cilik lan pelatih pribadi.

- **Kaca ngarep** kanthi gambar hero, sorotan kelas, lan ajakan tumindak
- Kaca **Babagan** kanthi crita lan misi studio
- Kaca **Kelas** kanthi tata letak jadwal
- Kaca **Pelatih** kanthi kertu profil
- Kaca **Kontak** kanthi peta lokasi lan formulir
- **Desain resik lan modern** kanthi skema warna sing cocog kanggo kabugaran

### Cithakan 2: Gym Pro {#template-2-gym-pro}

Kanggo gym sing wis mapan kanthi layanan luwih akèh.

- Kabèh sing ana ing Studio Essential, ditambah:
- Kaca **Keanggotaan** kanthi tabel rega
- Kaca **Galeri** kanggo foto fasilitas
- Bagéan **Blog** kanggo tips kabugaran lan pawarta
- Bagéan **Testimoni** ing kaca ngarep
- Opsi **branding sing luwih katon**

### Cithakan 3: Fitness Chain {#template-3-fitness-chain}

Kanggo operasi multi-lokasi.

- Kabèh sing ana ing Gym Pro, ditambah:
- Kaca **Lokasi** kanthi pirang-pirang dhaptar fasilitas
- Cithakan subkaca **waralaba/lokasi**
- **Branding terpusat** kanthi rincian khusus lokasi
- **Direktori staf** ing sakabèhé lokasi

## Mbangun Situs Cithakan {#building-a-template-site}

Ing Ultimate Multisite, cithakan iku mung situs WordPress sing dikonfigurasi kaya sing panjenengan karepaké supaya situs pelanggan anyar katon. Mangkéné carané nggawe:

### Langkah 1: Gawe Situs Cithakan {#step-1-create-the-template-site}

1. Bukak **Situs > Tambah Anyar** ing admin jaringan panjenengan
2. Gawe situs kanthi jeneng `template-studio-essential`
3. Situs iki dadi kanvas kerja panjenengan

### Langkah 2: Pasang lan Konfigurasi Tema {#step-2-install-and-configure-the-theme}

Pindhah menyang Dashboard situs cithakan banjur:

1. Aktifaké tema sing cocog kanggo bisnis kabugaran
2. Konfigurasi setelan tema, warna, lan tipografi
3. Siapaké header lan footer kanthi navigasi sing cocog kanggo kabugaran

:::tip Pamilihan Tema
Pilih tema sing cukup fleksibel supaya katon apik kanggo kabugaran nanging ora nganti rumit banget nganti pelanggan panjenengan ora bisa ngatur. Tema kaya Astra, GeneratePress, utawa Kadence cocog amarga entheng, bisa dikustomisasi, lan didhukung kanthi apik.
:::

### Langkah 3: Gawe Kaca-kaca {#step-3-create-the-pages}

Bangun saben kaca kanthi:

- **Konten placeholder** sing diwaca alami kanggo kabugaran ("Sugeng rawuh ing [Jeneng Studio]" dudu "Lorem ipsum")
- **Gambar placeholder** saka situs foto stok gratis sing nuduhaké kegiatan kabugaran
- **Tata letak fungsional** nggunakake page builder utawa editor blok

Gawe konten placeholder dadi instruktif yen bisa. Tinimbang teks pangisi umum, tulisen konten kaya: "Ganti iki nganggo katrangan cekak babagan studio panjenengan lan apa sing nggawe istimewa. Sebutaké filosofi latihan panjenengan, taun pengalaman, utawa apa sing bisa diarepaké klien."

### Langkah 4: Konfigurasi Tambahan {#step-4-configure-plugins}

Pasang lan aktifaké tambahan sing dibutuhaké studio kabugaran:

- Tambahan booking utawa penjadwalan (yen cocog karo tingkatan plan panjenengan)
- Tambahan formulir kontak
- Tambahan SEO (wis dikonfigurasi nganggo standar gawan sing relevan karo kabugaran)

### Langkah 5: Tandhani minangka Cithakan {#step-5-mark-as-template}

1. Bukak **Ultimate Multisite > Situs**
2. Owahi situs cithakan
3. Aktifaké toggle **Cithakan Situs**

Balèni proses iki kanggo saben cithakan sing arep panjenengan tawaraké.

## Dhaptar Priksa Kualitas Cithakan {#template-quality-checklist}

Sadurungé nggawe cithakan kasedhiya, priksa:

- [ ] Kabèh kaca mbukak kanthi bener lan katon profesional
- [ ] Konten placeholder migunani lan khusus ceruk
- [ ] Gambar cocog lan lisensiné bener
- [ ] Responsivitas mobile mlaku ing kabèh kaca
- [ ] Navigasi logis lan lengkap
- [ ] Formulir kontak mlaku
- [ ] Ora ana tautan rusak utawa aset sing ilang
- [ ] Kacepetan muat kaca bisa ditampa
