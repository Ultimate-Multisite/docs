# PRO Theme वापरून तुमचे Admin Panel सानुकूलित करा

बऱ्याचदा क्लायंट कामात आपल्याला WordPress च्या admin dashboard मध्ये क्लायंटला एक सानुकूलित अनुभव देण्याची गरज असते. जर तुम्ही अधिक व्यावसायिक रूप आणि अनुभव शोधत असाल, तर PRO Theme तुम्हाला मदत करू शकते.

PRO Theme हे एक admin theme आहे जे तुमच्या क्लायंटचे Dashboard सानुकूलित करताना तुम्हाला मदत करण्यासाठी विकसित केले आहे. आणि हे Ultimate Multisite च्या premium plugins core चा देखील भाग आहे.

ते install केल्यानंतर, तुमच्या _Plugins_ पेजवर जा आणि तुमच्या Network वर ते activate करण्यासाठी क्लिक करा.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for PRO Theme -->

_तुमच्या Plugins पेजवर ते network-wide activate करा_

_**टीप:**_ _जर तुम्ही फक्त ते install केले, तर ते तुमच्या क्लायंटला एक पर्याय म्हणून दिसेल जो फक्त त्या युजरसाठी theme जोडण्यासाठी निवडता येतो._

आता, तुमचे admin interface पूर्णपणे नव्याने डिझाइन केले आहे! तुम्हाला डाव्या स्तंभात PRO Theme नावाचा एक नवीन मेनू देखील दिसेल.
