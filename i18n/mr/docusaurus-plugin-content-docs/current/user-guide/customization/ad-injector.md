# Ultimate Multisite: Ad Injector वापरून तुमच्या नेटवर्कवर जाहिराती लावणे

Ultimate Multisite: Ad Injector हे एक **मोफत add-on** आहे जे तुम्हाला तुमच्या plans वर जाहिराती लावण्याची सुविधा त्यांच्या edit page वरूनच देते.

Ultimate Multisite: Ad Injector install केल्यानंतर, ते activate करण्यासाठी तुमच्या _Plugins_ page वर जा.

<!-- Screenshot unavailable: WP Plugins page showing Network Activate button for Ad Injector addon -->

_Network Activate बटणावर क्लिक करा_

Add-on active झाल्यावर, plan edit screen वरील _Advanced Options_ meta box मध्ये _Ad Settings_ नावाचा एक नवीन पर्याय दिसेल.

![Plan edit screen वरील Ad Settings](/img/config/product-edit-full.png)

_आता तुम्ही तुमच्या प्रत्येक plan साठी जाहिरातींची settings बदलू शकता_

त्या tab मध्ये, तुम्ही front-end वरील posts आणि pages साठी, content च्या आधी आणि नंतर जाहिरातींचे codes जोडू शकता.

<!-- Screenshot unavailable: Ad code fields for posts and pages before content -->

<!-- Screenshot unavailable: Example of ad displayed on a post page for a specific plan -->

_तुमच्या जाहिरातीचा code टाकल्यानंतर, ती त्या विशिष्ट plan च्या posts आणि pages वर दिसेल._

तुम्ही त्या plan च्या वापरकर्त्यांसाठी Admin Pages च्या header मध्येही जाहिराती दाखवू शकता.

<!-- Screenshot unavailable: Admin Panel Ad Code option in plan settings -->

<!-- Screenshot unavailable: Example of ad displayed on the user admin panel -->

_तुमच्या वापरकर्त्यांच्या back-end वर जाहिराती दाखवायच्या असतील तर Admin Panel Ad Code पर्यायावर जा_

### जाहिरातींच्या पलीकडे

तुम्ही Ultimate Multisite: Ad Injector वापरून उच्च-स्तरीय plans ना support देखील देऊ शकता.

फक्त _Admin Panel Ads_ पर्यायावर जा आणि तुमच्या support tool साठी HTML code जोडा.

<!-- Screenshot unavailable: Admin Panel Ads option with support tool HTML code -->

<!-- Screenshot unavailable: Customer support widget displayed on the admin panel for a specific plan -->

_विशिष्ट plans ना customer support जोडा_

अशा प्रकारे लोक थेट त्यांच्या Admin Panel वरून तुमच्याशी संपर्क साधू शकतात!
