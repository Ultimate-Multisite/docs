# रिलीझ नोट्स

## व्हर्जन २.१२.० — रिलीझ तारीख: २०२६-०५-१५

- नवीन: डोमेन मॅपिंग इंटिग्रेशनसह होस्टिंगर (hPanel) ला समर्थित होस्ट प्रदाता म्हणून जोडले.
- नवीन: नेटवर्क-व्यापी साइट रिस्टोरेशनसाठी साइट एक्सपोर्टर आता नेटवर्क इम्पोर्ट बंडल्स हाताळतो.
- सुधारणा: BCC ब्रॉडकास्ट ईमेल आता प्राप्तकर्त्यांचे पत्ते उघड होऊ नयेत यासाठी 'undisclosed-recipients' हेडर वापरतात.
- सुधारणा: गैर-तारीख (non-date) मूल्य वापरून सेव्ह करताना मेंबरशिपची मुदत तारीख आता खराब होत नाही.
- सुधारणा: Stripe मेंबरशिप अपडेट्स आता deprecated deleteDiscount API ला कॉल न करता सवलती (discounts) योग्यरित्या काढून टाकतात.
- सुधारणा: डोमेन-मॅप केलेल्या साइट्सवरील SSO रीडायरेक्ट आता अनंत रीडायरेक्ट लूप्स (infinite redirect loops) टाळण्यासाठी मर्यादित केले आहेत.
- सुधारणा: सेटअप विजार्ड इमेज पिकरची निवड आता मूलभूत डेटा मॉडेल (underlying data model) योग्यरित्या अपडेट करते.
- सुधारणा: साइट एक्सपोर्टर CLI आता योग्य डीफॉल्ट नेटवर्क साइट निवड जपून ठेवतो.
- सुधारित: प्लगइन पॅकेजमधून बंडल केलेला wp-cli काढून टाकण्यात आला आहे, ज्यामुळे प्लगइनचा आकार कमी झाला आहे.

## व्हर्जन २.११.० — रिलीझ तारीख: २०२६-०५-११

- नवीन: साइट एक्सपोर्ट आता एक सेल्फ-बूटिंग `index.php` बंडल करतात, जेणेकरून ZIP फाईल वेगळ्या प्लगइन इन्स्टॉलेशनशिवाय नवीन होस्टवर इन्स्टॉल करता येईल.
- नवीन: नेटवर्क एक्सपोर्ट प्रशासकांना साइट एक्सपोर्ट ॲडमिन पृष्ठावरून एकाच आर्काइव्हमध्ये सर्व सबसाइट्स एक्सपोर्ट करण्याची सुविधा देते.
- नवीन: 'Allow Site Templates' प्लॅन टॉगल आता फॉलबॅक चेन (fallback chain) द्वारे लागू केला जातो, ज्यामुळे प्लॅन मर्यादांसाठी टेम्पलेटची उपलब्धता योग्यरित्या प्रतिबंधित होते.
- नवीन: चेकआउट फॉर्म एडिटरला इशारा देतो की आवश्यक फील्ड कॉन्फिगर न करता उत्पादन जोडले आहे.
- नवीन: इम्पोर्ट/एक्सपोर्ट सेटिंग्ज टॅब आता त्याचा व्याप्ती (scope) स्पष्टपणे वर्णन करतो आणि थेट साइट एक्सपोर्ट टूलशी लिंक करतो.

## व्हर्जन २.१०.० — रिलीझ तारीख: २०२६-०५-०५

- नवीन: OAuth फ्लॅग गेटसह मॅन्युअल क्रेडेंशियल एंट्रीसाठी PayPal गाइडेड सेटअप विजार्ड. यामुळे सहज गेटवे कॉन्फिगरेशन होते.
- नवीन: टेम्पलेट स्विच ग्राहक पॅनेलमध्ये 'current-template' कार्ड, कायमस्वरूपी ग्रिड आणि **Reset current template** बटण डिझाइन केले आहे.
- सुधारणा: टेम्पलेट स्विचिंग आता AJAX अयशस्वी झाल्यास UI ला थांबवत नाही.
- सुधारणा: टेम्पलेट स्विचिंग परमिशन स्टेट्स अनधिकृत ॲक्सेसपासून सुरक्षित केल्या आहेत.
- सुधारणा: साइट ओव्हरराइड इनपुट्स सेव्ह करण्यापूर्वी व्हॅलिडेट केले जातात.
- सुधारणा: ॲड्रेस रिकामी असल्यास बिलिंग ॲड्रेस प्रॉम्प्ट आता दिसतो.
- सुधारणा: PHP 8.1 null-to-string deprecation notices सोडवले आहेत.
- सुधारणा: 'Currents' हुक आता init पूर्वी lazy-load होतो, ज्यामुळे वेळेच्या समस्या (timing issues) टळतात.
- सुधारणा: सर्व लॉगिन फ्लोमध्ये फिल्टर केलेला SSO पाथ (path) पाळला जातो.
- सुधारणा: ब्लँक साइट आयडेंटिटी पर्याय सेव्ह करताना जपले जातात.

## व्हर्जन २.९.० — रिलीझ तारीख: २०२६-०४-३०

- नवीन: **Tools > Export & Import** अंतर्गत सिंगल-साइट एक्सपोर्ट आणि इम्पोर्ट जोडले.
- सुधारणा: एक्सपोर्ट ZIP फाईल्स आता प्रमाणित डाउनलोड एंडपॉईंट (authenticated download endpoint) द्वारे सर्व्ह केल्या जातात.
- सुधारणा: पेंडिंग एक्सपोर्ट/इम्पोर्ट क्वेरीमध्ये SQL इंजेक्शन धोका आणि क्वेरी समस्या दुरुस्त केल्या.
- सुधारणा: प्रशासकाने ग्राहक ईमेल मॅन्युअली सत्यापित केल्यावर पेंडिंग साइट प्रकाशित होत नाही.
- सुधारणा: मेंबरशिप गहाळ असल्यास 'pending_site' रेकॉर्ड्स स्वच्छ केले जातात.
- सुधारणा: सेटिंग्ज नेव्हिगेशन पॅडिंग आणि सर्च अँकर नेव्हिगेशन दुरुस्त केले.
- सुधारणा: पेंडिंग साइट्स आता 'All Sites' व्ह्यूमध्ये पहिल्या क्रमांकावर दिसतात.
- सुधारणा: स्क्रीनशॉट प्रदाता (mShots) साठी User-Agent हेडर जोडले, ज्यामुळे 403 त्रुटी (errors) टळतात.
- सुधारणा: इम्पोर्ट क्रॉन शेड्यूल सर्क्युलर डिपेंडन्सी (circular dependency) सोडवली.
- सुधारणा: वापरकर्ता सेटिंग्ज कीजमध्ये टूर आयडी (Tour IDs) अंडरस्कोरमध्ये सामान्य केले.
- सुधारित: चांगल्या सुसंगततेसाठी Alchemy/Zippy ऐवजी ZipArchive वापरले आहे.

## व्हर्जन २.८.० — रिलीझ तारीख: २०२६-०४-२९

- नवीन: 'Other Options' सेटिंग्ज UI मध्ये 'Enable Jumper' टॉगल जोडला.
- नवीन: चेकआउट फॉर्म्स लिस्ट टेबलमध्ये स्टेटस कॉलम जोडला.
- नवीन: कस्टम MU-plugin sunrise एक्सटेंशन्ससाठी ॲडॉन सनराइज फाइल लोडर.
- सुधारित: सेटिंग्ज पृष्ठावरून एरर-रिपोर्टिंग ऑप्ट-इन सेटिंग काढून टाकण्यात आले.
- सुधारणा: धन्यवाद पृष्ठ साइट कार्ड - इमेज आता मर्यादित (constrained) आणि लिंक्स योग्यरित्या स्टाईल केले आहेत.
- सुधारणा: स्क्रीनशॉट प्रदाता thum.io वरून WordPress.com mShots वर बदलला.
- सुधारणा: 'Enable Registration' आणि 'Default Role' आता नवीन इन्स्टॉलेशनवर योग्य डीफॉल्ट सेट केले जातात.
- सुधारणा: `get_site_url()` आता डोमेनमध्ये पोर्ट असल्यास रिकामी व्हॅल्यू परत करत नाही.
- सुधारणा: `copy_media` सेटिंग रिकामी असताना मीडिया फाईल्स आता योग्यरित्या कॉपी होतात.
- सुधारणा: नेटवर्क-ॲक्टिव्हेट साइटमेटा (sitemeta) लिहिल्यानंतर ऑब्जेक्ट कॅशे (Object cache) योग्यरित्या अवैध (invalidated) होतो.
- सुधारणा: ३-भागांच्या डोमेनसाठी DNS पडताळणीवर कस्टम डोमेन प्राथमिक म्हणून स्वतःहून प्रमोट होतो.
- सुधारणा: पेमेंटची मुदत संपल्यास पेंडिंग मेंबरशिप रद्द होते.
- सुधारणा: इनलाइन लॉगिन प्रॉम्प्ट (inline login prompt) बंद झाल्यावर पासवर्ड स्ट्रेंथ चेकर पुन्हा जोडला जातो.
- सुधारणा: साइट आधीच तयार असल्यास धन्यवाद पृष्ठावर अनंत पेज रीलोड थांबला.
- सुधारणा: प्लगइन सक्रिय झाल्यावर आणि सेटिंग्ज सेव्ह झाल्यावर WP core रजिस्ट्रेशन पर्याय सिंक केला जातो.
- सुधारणा: PHP 8.4 सुसंगततेसाठी `calculate_expiration` मध्ये Null expiration guard जोडला.
- सुधारणा: ग्राहककडे आधीच सक्रिय मेंबरशिप असल्यास डुप्लिकेट साइनअप ब्लॉक केले जातात.
- सुधारणा: चेकआउटमध्ये `date_expiration` साठी Null चेक जोडला.
- सुधारणा: साइट प्रोव्हिजनिंग अधिक मजबूत केले — मर्यादा, मेंबरशिप अनुमान (membership inference), डोमेन प्रमोशन.
- सुधारणा: तपासणी अयशस्वी झाल्यास प्री-इन्स्टॉल चेक स्टेटस लेबल 'NOT Activated' असे दुरुस्त केले.
- सुधारणा: ईमेल पडताळणी URL साठी चेकआउट डोमेन वापरले जाते.
- सुधारणा: पासवर्ड फील्ड नसताना चेकआउटनंतर ऑटो-लॉगिन होते.
- सुधारणा: मोफत मेंबरशिप आता मुदत संपत नाहीत — त्यांना आयुष्यभराचे (lifetime) मानले जाते.
- सुधारणा: ईमेल पडताळणी गेट साइट प्रकाशित होईपर्यंत थांबवतो जोपर्यंत ग्राहक ईमेल सत्यापित करत नाही.
- सुधारणा: SES v2 API एंडपॉईंट बेस पाथ आणि आयडेंटिटी रूट दुरुस्त केले.
- सुधारणा: `wu_inline_login_error` हुक प्री-सबमिट कॅच ब्लॉकमध्ये उत्सर्जित केला जातो.
