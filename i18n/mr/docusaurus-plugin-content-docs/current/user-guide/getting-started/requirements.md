# Ultimate Multisite आवश्यकता

Ultimate Multisite खरेदी करण्यापूर्वी तुम्ही स्वतःला पहिला प्रश्न विचाराल तो म्हणजे आमचे plugin चालवण्यासाठी काय आवश्यकता आहेत. सुदैवाने, Ultimate Multisite चालवण्यासाठी फक्त काही मूलभूत आवश्यकता आहेत: एक **domain name**, एक चांगला **hosting provider** आणि WordPress **Multisite** installation. बस एवढेच तुम्हाला हवे आहे!

## Domain Name

आधी, domain name बद्दल बोलूया.

Domain name निवडताना, तुमची विशिष्ट niche काय आहे आणि Ultimate Multisite सोबत तुम्ही काय तयार कराल हे लक्षात ठेवा: तुम्हाला एजन्सी तयार करायची आहे का? दुकानासारख्या वेबसाइट्स विकायच्या आहेत का? किंवा कदाचित WordPress वेब होस्टिंग? Ultimate Multisite सोबत सर्व काही शक्य आहे.

आमचे plugin subscription सेवा, payment collection, checkout forms, discount vouchers आणि ग्राहक संवाद प्रदान करते. WordPress Multisite install, configure आणि maintain करण्यासाठी आवश्यक असलेले बरेचसे काम Ultimate Multisite द्वारे सोपे केले जाते.

जोपर्यंत तुम्ही वेबसाइट्स सेवा म्हणून विकता, तोपर्यंत तुम्ही निवडलेली कोणतीही niche आमच्या plugin वापरून व्यवस्थापित केली जाऊ शकते! तुमच्या व्यवसाय योजनेशी जुळणारे domain name निवडण्याची खात्री करा.

## Hosting Provider

आता तुमच्याकडे तुमची niche आणि व्यवसाय दर्शवणारे आकर्षक domain name आहे, तुम्हाला Ultimate Multisite सोबत तुमचे network host करण्यासाठी एक चांगली वेब होस्टिंग कंपनी निवडावी लागेल. बाजारात काही चांगले पर्याय आहेत. काही hosting providers Ultimate Multisite सोबत चांगले integrate होतात, जे तुम्ही काय बांधण्याची योजना करत आहात त्यावर अवलंबून निर्णायक घटक असू शकते.

Ultimate Multisite कोणत्याही hosting provider मध्ये काम करायला हवे. फक्त फरक एवढाच आहे की _WPMU DEV Hosting_, _WP Engine_, _Closte_, _Cloudways_, _Gridpane_ आणि _Runcloud_ सारख्या काही होस्टिंग कंपन्यांकडे domain mapping साठी अधिक घट्ट integration आहे. इतर होस्टिंगसाठी, तुम्हाला custom domains मॅन्युअली map करावे लागतील. AutoSSL ला support करणारा hosting provider घेणे देखील चांगली कल्पना आहे. _Closte_, _Runcloud_ आणि _cPanel_ याला support करतात.

सर्व्हर specifications बद्दल बोलायचे झाले तर, कोणताही विशिष्ट formula नाही कारण ते तुम्ही कोणती features आणि functions ऑफर कराल आणि तुमच्या network चा आकार यावर अवलंबून असेल. Shared hosting ही चांगली सुरुवात असू शकते, पण तुमचे network वाढू लागताच, तुम्हाला कदाचित dedicated VPS लागेल.

## WordPress Multisite

आता, WordPress Multisite installation बद्दल थोडे बोलूया.

WordPress Multisite तुम्हाला एकाच installation वर साइट्सचे network ठेवण्याची परवानगी देते. ही एक built-in feature आहे, पण ती default म्हणून active नाही. तुम्हाला ही feature तुमच्या WordPress installation आणि वेब होस्टिंग control panel मध्ये enable करणे आवश्यक आहे.

Multisite enabled असलेले WordPress हे **Ultimate Multisite चा पाया आहे**. Ultimate Multisite चा मुख्य वापर युजर्सना वेबसाइट्स सेवा म्हणून व्यवस्थापित आणि विक्री करण्यात मदत करणे असल्याने, हे Multisite enabled असलेल्या WordPress वर करणे आवश्यक आहे.

आमच्याकडे **एक सविस्तर लेख** आहे जो तुमच्या WordPress installation वर Multisite feature enable करण्यासाठी काय करणे आवश्यक आहे ते step by step शिकवतो.

तुमच्या WordPress वर Multisite enable झाल्यानंतर, Ultimate Multisite install करण्याची वेळ आली आहे. Ultimate Multisite योग्यरित्या install करण्यासाठी, तुमच्याकडे असणे आवश्यक आहे:

  * WordPress: v5.3+ (शिफारस: शेवटची stable version)

  * PHP: 7.4.x (> 8.0 साठी support लवकरच येत आहे)

  * MySQL: v5+ (शिफारस: 5.6, जर तुमच्या hosting provider वर 8.0 उपलब्ध नसेल तर)

Ultimate Multisite साठी या एकमेव software आवश्यकता आहेत.

लक्षात ठेवा की Ultimate Multisite PHP 8.0 सोबत काम करू शकते पण आम्ही ते PHP 7.4.x सोबत चालवण्याची शिफारस करतो.

तसेच, आम्ही ते **main domains** वर install करण्याची शिफारस करतो, subdomains किंवा subdirectories वर नाही. ते subdomains/subdirectories वर काम करू शकते, पण काही errors येऊ शकतात.

Ultimate Multisite कसे install करायचे याच्या सविस्तर मार्गदर्शिकेसाठी, **येथे क्लिक करा**.
