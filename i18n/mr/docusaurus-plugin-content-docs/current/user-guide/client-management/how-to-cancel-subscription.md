# माझी सदस्यता कशी रद्द करावी?

जर तुमच्याकडे Ultimate Multisite लायसन्स असेल, तर तुम्ही कधीही त्याचे नूतनीकरण रद्द करू शकता. खालील पायऱ्या फॉलो करा:


<!-- Screenshot unavailable: Account page email with access link -->

खरेदी करताना दिलेला ईमेल आणि तुमचा पासवर्ड वापरा.

<!-- Screenshot unavailable: Login page with email and password fields -->

हे तुमच्या Account Page चे सुरुवातीचे स्क्रीन आहे:

<!-- Screenshot unavailable: Account page initial screen showing subscription details -->

तुमची सदस्यता रद्द करण्यासाठी, डाव्या बाजूच्या मेनूमध्ये _Renewing & Billing_ वर जा.

<!-- Screenshot unavailable: Renewing & Billing page with cancel option -->

साइड विंडो उघडण्यासाठी उजवीकडील बाणावर क्लिक करा. त्यानंतर, **auto-renew** रद्द करणारा पर्याय निवडा.

सिस्टम तुम्हाला एक पुष्टीकरण संदेश दाखवेल.

जर तुम्हाला खात्री असेल की तुम्हाला तुमची सदस्यता रद्द करायची आहे, तर फक्त _**Cancel Renewals**_ बटणावर क्लिक करा. या क्रियेनंतर, तुम्हाला एक छोटी सर्वेक्षण भरण्यास सांगितले जाईल.

झालं! तुमची सदस्यता आपोआप नूतनीकृत होणार नाही.

**तुमची सदस्यता संपेपर्यंत तुमच्याकडे वैध की असेल**. जर तुम्हाला तुमची सदस्यता पुन्हा सक्रिय करायची असेल, तर ते तुम्हाला स्वतः करावे लागेल.
