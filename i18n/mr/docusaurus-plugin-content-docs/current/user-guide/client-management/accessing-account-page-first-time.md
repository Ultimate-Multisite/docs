# माझ्या खाते पृष्ठावर पहिल्यांदा कसे प्रवेश करायचे?

Ultimate Multisite मध्ये आपले स्वागत आहे!

आमचे plugin खरेदी केल्यानंतर तुम्हाला ई-मेलमध्ये मिळालेल्या लिंकद्वारे तुमच्या खाते पृष्ठावर प्रवेश करा:

<!-- Screenshot unavailable: Account page email with access link -->

खरेदी करताना दिलेला ई-मेल आणि तुम्हाला ई-मेलमध्ये मिळालेला पासवर्ड वापरा.

<!-- Screenshot unavailable: Login page with email and password fields -->

तुम्ही या खात्यात पहिल्यांदा लॉग इन करत असल्याने, तुम्हाला पासवर्ड बदलण्यास सांगितले जाईल. तो बदला आणि _Save_ बटणावर क्लिक करा.

<!-- Screenshot unavailable: Change password form with Save button -->

तुमच्या खाते पृष्ठावर, तुम्ही बिलिंग, नूतनीकरण, रद्दीकरण आणि इतर अनेक गोष्टींसारखे सर्व खाते तपशील व्यवस्थापित करू शकता.
