# तुमच्या ग्राहकांच्या वेबसाइट्स (v2)

तुमच्या नेटवर्कवर नोंदणी केल्यानंतर, तुमचे ग्राहक त्यांचे dashboard वापरू शकतात.

**Account** पेज व्यतिरिक्त, जिथे ते त्यांच्या पेमेंट्स, मेंबरशिप्स, डोमेन्स, प्लॅन मर्यादा आणि इतर महत्त्वाची माहिती पाहू शकतात, त्यांना **My Sites** पेजवर देखील प्रवेश आहे, जिथे ते त्यांच्या सर्व वेबसाइट्स पाहू आणि व्यवस्थापित करू शकतात. त्यांना फक्त त्यांच्या dashboard वरील My Sites वर क्लिक करायचे आहे.

![ग्राहक dashboard मधील My Sites लिंक](/img/admin/customers-list.png)

त्यांना त्यांच्या सर्व वेबसाइट्स दिसतील, प्रत्येकाच्या खाली **Manage** बटण असेल. त्यावर क्लिक केल्यास, ते त्या **विशिष्ट वेबसाइटच्या dashboard वर** पोहोचतील.

ज्या प्लॅन्समध्ये एकापेक्षा जास्त वेबसाइट्स ठेवता येतात, त्यामध्ये कोणती वेबसाइट **primary** आहे हे देखील ते पाहू शकतात.

![Manage बटणांसह ग्राहक वेबसाइट्स दाखवणारे My Sites पेज](/img/admin/customers-list.png)

त्यांना primary वेबसाइट बदलायची असल्यास, त्यांना **Account** मेनूवर क्लिक करून Account पेजवर जावे लागेल.

![ग्राहक dashboard मधील Account मेनू लिंक](/img/admin/customers-list.png)

त्यानंतर, त्यांनी **Change Default Site** वर क्लिक करावे.

![Account पेजवरील Change Default Site बटण](/img/admin/customers-list.png)

एक pop-up विंडो दिसेल. drop-down list वर क्लिक करा आणि तुम्हाला कोणती वेबसाइट primary करायची आहे ते निवडा.

![Change Default Site dropdown निवड](/img/admin/customers-list.png)

मग, फक्त **Change Default Site** वर क्लिक करा आणि झाले!

![Default site बदलाची पुष्टी](/img/admin/customers-list.png)
