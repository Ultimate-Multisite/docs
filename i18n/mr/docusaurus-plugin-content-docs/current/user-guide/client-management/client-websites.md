# तुमच्या ग्राहकांची संकेतस्थळे (v2) {#your-clients-websites-v2}

तुमच्या नेटवर्कवर नोंदणी केल्यानंतर, तुमचे ग्राहक त्यांच्या Dashboard मध्ये प्रवेश करू शकतात.

**Account** पृष्ठाशिवाय, जिथे ते त्यांच्या पेमेंट्स, सदस्यत्वे, डोमेन्स, प्लॅन मर्यादा आणि इतर महत्त्वाची माहिती पाहू शकतात, त्यांना **माझ्या साइट्स** पृष्ठावरही प्रवेश असतो, जिथे ते त्यांची सर्व संकेतस्थळे पाहू आणि व्यवस्थापित करू शकतात. त्यांना फक्त त्यांच्या Dashboard वर माझ्या साइट्सवर क्लिक करायचे आहे.

![ग्राहक Dashboard मधील माझ्या साइट्स लिंक](/img/account-page/my-sites-link.png)

त्यांना त्यांची सर्व संकेतस्थळे दाखवली जातील, प्रत्येकाच्या अगदी खाली **व्यवस्थापित करा** बटण असेल. त्यांनी त्यावर क्लिक केल्यास, त्यांना **या विशिष्ट संकेतस्थळाच्या Dashboard** कडे पुनर्निर्देशित केले जाईल.

एकापेक्षा जास्त संकेतस्थळे ठेवण्याची परवानगी देणाऱ्या प्लॅन्ससाठी त्यांच्या संकेतस्थळांपैकी कोणते **प्राथमिक** आहे तेही ते पाहू शकतात.

![व्यवस्थापित करा बटणांसह ग्राहकांची संकेतस्थळे दाखवणारे माझ्या साइट्स पृष्ठ](/img/account-page/my-sites-page.png)

साइट संपादन इंटरफेसचे संपूर्ण दृश्य येथे आहे:

![साइट संपादन पूर्ण इंटरफेस](/img/admin/site-edit-full.png)

त्यांना त्यांचे प्राथमिक संकेतस्थळ बदलायचे असल्यास, त्यांनी **Account** मेनूवर क्लिक करून Account पृष्ठावर जायला हवे.

![ग्राहक Dashboard मधील Account मेनू लिंक](/img/account-page/account-menu.png)

तुम्ही ग्राहक तपशीलही पाहू आणि संपादित करू शकता:

![ग्राहक संपादन इंटरफेस](/img/admin/customer-edit.png)

त्यानंतर, त्यांनी **डीफॉल्ट साइट बदला** वर क्लिक करावे.

![Account पृष्ठावरील डीफॉल्ट साइट बदला बटण](/img/account-page/change-default-site.png)

एक पॉप-अप विंडो दिसेल. ड्रॉप-डाउन सूचीवर क्लिक करा आणि कोणते संकेतस्थळ प्राथमिक संकेतस्थळ बनवायचे आहे ते निवडा.

![डीफॉल्ट साइट बदला ड्रॉपडाउन निवड](/img/account-page/change-default-site-dropdown.png)

त्यानंतर, फक्त **डीफॉल्ट साइट बदला** वर क्लिक करा आणि झाले!

![डीफॉल्ट साइट बदलाची पुष्टी करणे](/img/account-page/change-default-site-confirm.png)
