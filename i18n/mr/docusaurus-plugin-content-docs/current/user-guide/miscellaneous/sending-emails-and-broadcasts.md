# ईमेल आणि ब्रॉडकास्ट पाठवणे (v2)

_**महत्त्वाची सूचना: हा लेख Ultimate Multisite आवृत्ती 2.x संबंधित आहे.**_

Ultimate Multisite मध्ये एक वैशिष्ट्य आहे जे तुम्हाला तुमच्या ग्राहकांशी संवाद साधण्यास मदत करते - तुम्ही एखाद्या विशिष्ट वापरकर्त्याला किंवा वापरकर्त्यांच्या गटाला ईमेल पाठवू शकता, तसेच त्यांच्या admin dashboard वर घोषणा प्रसारित करण्यासाठी सूचना दाखवू शकता.

## Broadcasts वापरून तुमच्या ग्राहकांच्या dashboard वर admin सूचना जोडा

Ultimate Multisite च्या broadcast वैशिष्ट्याचा वापर करून, तुम्ही तुमच्या वापरकर्त्यांच्या subsite admin dashboard वर **admin notices** जोडू शकता.

जेव्हा तुम्हाला सिस्टम मेंटेनन्सची माहिती द्यायची असेल किंवा तुमच्या विद्यमान वापरकर्त्यांना नवीन उत्पादने किंवा सेवांची ऑफर द्यायची असेल, तेव्हा हे अत्यंत उपयुक्त आहे. तुमच्या वापरकर्त्याच्या dashboard वर admin notice असे दिसेल.

![ग्राहकाच्या dashboard वर दाखवलेली Admin notice broadcast](/img/admin/broadcasts-list.png)

Admin notice सुरू करण्यासाठी, तुमच्या network admin dashboard वर जा आणि **Ultimate Multisite** मेनू अंतर्गत, तुम्हाला **Broadcasts** हा पर्याय दिसेल.

![Ultimate Multisite admin मधील Broadcasts मेनू](/img/admin/broadcasts-list.png)

या पेजवरून, वरच्या बाजूला असलेल्या **Add Broadcast** बटणावर क्लिक करा.

यामुळे Add broadcast modal विंडो उघडेल जिथे तुम्ही कोणत्या प्रकारचा broadcast पाठवायचा आहे ते निवडू शकता.

**Message** निवडा आणि नंतर **Next Step** बटणावर क्लिक करा.

![Message प्रकार निवडलेली Add broadcast modal](/img/admin/broadcasts-list.png)

पुढील विंडोमध्ये तुम्हाला **Target customer** किंवा **Target product** विचारले जाईल. लक्षात घ्या की तुम्ही एकापेक्षा जास्त वापरकर्ते किंवा एकापेक्षा जास्त उत्पादने निवडू शकता.

वापरकर्ता खाते किंवा उत्पादन शोधण्यासाठी, फील्डमध्ये keyword टाइप करायला सुरुवात करा.

**Message type** फील्ड अंतर्गत, तुम्ही सूचनेचा रंग निवडू शकता. यामुळे तुमच्या संदेशाची तातडी अधोरेखित होईल.

त्यानंतर तुम्ही **Next Step** वर क्लिक करू शकता.

![Broadcast साठी Target customer आणि product निवड](/img/admin/broadcasts-list.png)

पुढील विंडोमध्ये तुम्ही विषय आणि वापरकर्त्यांना प्रसारित करायचा मजकूर/संदेश टाकून तुमचा संदेश लिहू शकता.

![Broadcast संदेशाचा विषय आणि मजकूर संपादक](/img/admin/broadcasts-list.png)

तुमचा संदेश तयार केल्यावर, तुम्ही **Send** बटण दाबू शकता.

बस्स, इतकेच. Admin notice तुमच्या वापरकर्त्याच्या dashboard वर लगेच दिसायला लागेल.

## तुमच्या ग्राहकांना ईमेल पाठवा

Ultimate Multisite च्या broadcast वैशिष्ट्याचा वापर करून, तुम्ही तुमच्या वापरकर्त्यांना ईमेल पाठवू शकता. तुमच्याकडे फक्त विशिष्ट वापरकर्त्यांना ईमेल पाठवण्याचा पर्याय आहे किंवा त्यांनी subscribe केलेल्या उत्पादन किंवा plan च्या आधारावर विशिष्ट वापरकर्ता गटाला लक्ष्य करू शकता.

ईमेल broadcast सुरू करण्यासाठी, तुमच्या network admin dashboard वर जा आणि Ultimate Multisite मेनू अंतर्गत, तुम्हाला Broadcast हा पर्याय दिसेल.

![Ultimate Multisite admin मधील Broadcasts पेज](/img/admin/broadcasts-list.png)

या पेजवरून, वरच्या बाजूला असलेल्या **Add broadcast** बटणावर क्लिक करा.

यामुळे Add broadcast modal विंडो उघडेल जिथे तुम्ही कोणत्या प्रकारचा broadcast पाठवायचा आहे ते निवडू शकता. **Email** निवडा आणि नंतर **Next Step** बटणावर क्लिक करा.

![Email प्रकार निवडलेली Add broadcast modal](/img/admin/broadcasts-list.png)

पुढील विंडोमध्ये तुम्हाला **Target customer** किंवा **Target product** विचारले जाईल. लक्षात घ्या की तुम्ही एकापेक्षा जास्त वापरकर्ते किंवा एकापेक्षा जास्त उत्पादने निवडू शकता.

तुमचे लक्ष्य प्रेक्षक निवडल्यावर, तुम्ही **Next Step** वर क्लिक करू शकता.

## System emails

Ultimate Multisite मधील System emails म्हणजे नोंदणी, पेमेंट, domain mapping इत्यादी काही क्रियांनंतर सिस्टमद्वारे पाठवल्या जाणाऱ्या **स्वयंचलित सूचना**. हे ईमेल Ultimate Multisite settings मधून संपादित किंवा बदलता येतात. यात एक वैशिष्ट्य देखील आहे जे तुम्हाला दुसऱ्या Ultimate Multisite installation मधून विद्यमान settings रीसेट आणि import करण्याची परवानगी देते.

### रीसेट करणे आणि Import करणे

नवीन Ultimate Multisite आवृत्त्या, तसेच add-ons, वेळोवेळी नवीन emails नोंदणीकृत करू शकतात आणि करतील.

समस्या आणि इतर अडचणी टाळण्यासाठी, **आम्ही नवीन email templates तुमच्या install वर System Emails म्हणून आपोआप जोडणार नाही**, जोपर्यंत ते एखाद्या विशिष्ट वैशिष्ट्याच्या योग्य कार्यासाठी अत्यंत आवश्यक नसतील.

तथापि, super admins आणि agents importer टूल वापरून या नव्याने नोंदणीकृत emails import करू शकतात. ही प्रक्रिया नवीन email template च्या मजकूर आणि कॉन्फिगरेशनसह एक नवीन system email तयार करेल, ज्यामुळे super admin ला हवे ते बदल करता येतील किंवा ते जसे आहेत तसे ठेवता येतील.

#### System emails कसे import करावे

तुमच्या Ultimate Multisite Settings पेजवर जा आणि **Emails** टॅबवर जा.

![Ultimate Multisite settings मधील Emails टॅब](/img/config/settings-emails.png)

त्यानंतर, sidebar वर **Customize System Emails** बटणावर क्लिक करा.

![Sidebar वरील Customize System Emails बटण](/img/config/settings-emails.png)

System Emails पेजवर, तुम्हाला वरच्या बाजूला **Reset & Import** action बटण दिसेल. त्या बटणावर क्लिक केल्यावर import आणि reset modal विंडो उघडेल.

![System Emails पेजवरील Reset and Import action बटण](/img/config/settings-emails.png)

त्यानंतर, तुम्ही Import Emails पर्याय toggle करून कोणते system emails import साठी उपलब्ध आहेत ते पाहू शकता.

![उपलब्ध system emails दाखवणारे Import Emails पर्याय](/img/config/settings-emails.png)

#### System Emails रीसेट करणे

कधीकधी, तुम्हाला जाणवेल की तुम्ही एखाद्या email template मध्ये केलेले बदल आता तुमच्यासाठी काम करत नाहीत आणि तुम्हाला ते **मूळ स्थितीत** रीसेट करायचे आहेत.

अशा परिस्थितीत, तुमच्याकडे दोन पर्याय आहेत: तुम्ही फक्त system email हटवू शकता आणि ते परत import करू शकता (वरील सूचना वापरून) - यामुळे send metrics आणि इतर गोष्टी मिटतील, त्यामुळे ही पद्धत कमी पसंतीची आहे.

किंवा तुम्ही तो email template रीसेट करण्यासाठी **Reset & Import tool** वापरू शकता.

Email template रीसेट करण्यासाठी, तुम्ही Reset & Import tool पर्यंत पोहोचण्यासाठी वरील पायऱ्या फॉलो करू शकता, आणि नंतर, **Reset** पर्याय toggle करा आणि तुम्हाला मूळ मजकुरावर परत आणायचे असलेले emails निवडा.

![Email templates मूळ स्थितीत परत आणण्यासाठी Reset पर्याय](/img/config/settings-emails.png)
