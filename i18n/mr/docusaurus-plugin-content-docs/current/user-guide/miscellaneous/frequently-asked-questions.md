# वारंवार विचारले जाणारे प्रश्न

येथे काही **वारंवार विचारले जाणारे प्रश्न** दिले आहेत जे सहाय्य विनंत्यांमध्ये अनेकदा समोर येतात.

  * **Ultimate Multisite म्हणजे काय?**

Ultimate Multisite हे WordPress multisite installs साठी तयार केलेले एक WordPress plugin आहे, जे तुमच्या WordPress install ला [WordPress.com](https://WordPress.com) सारख्या प्रीमियम साइट्सच्या नेटवर्कमध्ये रूपांतरित करते – ज्यामुळे ग्राहकांना मासिक, त्रैमासिक किंवा वार्षिक शुल्कावर साइट्स तयार करता येतात (तुम्ही मोफत योजनाही तयार करू शकता).

  * **Ultimate Multisite हे Multisite शिवाय काम करेल का?**

नाही. Ultimate Multisite ला WordPress multisite, subdirectory किंवा subdomain आवश्यक आहे.

  * **Ultimate Multisite साठी काय आवश्यकता आहेत?**

Ultimate Multisite साठी आवश्यक:

**WordPress Version** : v5.3+ (शिफारस: शेवटची स्थिर आवृत्ती)  
**PHP Version** : 7.4.x (> 8.0 साठी सपोर्ट लवकरच येत आहे)  
**MySQL Version** : v5+ (शिफारस: आवृत्ती 5.6, जर तुमच्या hosting provider वर 8.0 उपलब्ध नसेल तर)

_Multisite (subdomain किंवा subdirectory) देखील सक्रिय करणे आवश्यक आहे._ [WordPress Multisite कसे Install करावे](/user-guide/getting-started/how-to-install-wordpress-multisite).

  * **Ultimate Multisite हे Subdirectories सह Multisite वर काम करेल का?**

हो. Ultimate Multisite subdirectory installs आणि subdirectory installs दोन्हींना सपोर्ट करते.

  * **Ultimate Multisite हे Subdomains सह Multisite वर काम करेल का?**

हो. Ultimate Multisite subdirectory installs आणि subdomain installs दोन्हींना सपोर्ट करते.

  * **वापरकर्ते कोणते Plugins आणि Themes वापरू शकतात हे मर्यादित करण्यासाठी मी Ultimate Multisite वापरू शकतो का?**

हो. Ultimate Multisite सह तुम्ही तुमच्या नेटवर्कवरील प्रत्येक योजनेसाठी कोणते plugins आणि themes उपलब्ध आहेत ते निवडू शकता.

  * **वापरकर्ता त्याच्या/तिच्या साइटवर किती पोस्ट्स तयार करू शकतो हे मर्यादित करण्यासाठी मी Ultimate Multisite वापरू शकतो का?**

हो. Ultimate Multisite सह तुम्ही तुमच्या नेटवर्कवरील **प्रत्येक उपलब्ध post type** साठी पोस्ट कोटा सेट करू शकता. तुम्ही admin किती वापरकर्त्यांना आमंत्रित करू शकतो, **डिस्क स्पेसचे प्रमाण** आणि बरेच काही मर्यादित करू शकता.

  * **Ultimate Multisite Site Templates ला सपोर्ट करते का?**

हो. Ultimate Multisite सह तुम्ही हव्या तितक्या Site Templates तयार करू शकता. Site Templates म्हणजे themes नाहीत, ते तुमच्या नेटवर्कवरील सामान्य साइट्स आहेत. तुम्ही plugins, themes सक्रिय करू शकता आणि त्या Site Template ला तुम्हाला हवे तसे customize करू शकता. जर वापरकर्त्याने तो template निवडला, तर त्याची किंवा तिची साइट त्या template ची प्रत म्हणून तयार होईल, सर्व customizations सह.

तुम्ही एखाद्या योजनेसाठी विशिष्ट template देखील सेट करू शकता. असे केल्याने, त्या योजनेवर साइन अप करणाऱ्या प्रत्येक वापरकर्त्याची साइट त्या Site Template ची प्रत म्हणून तयार होईल.

  * **Ultimate Multisite WooCommerce सोबत integrate होते का?**

हो!

जर तुम्हाला तुमच्या नेटवर्क वापरकर्त्यांना WooCommerce उपलब्ध करून द्यायचे असेल, त्यांना स्टोअर्स सेट अप करण्याची परवानगी द्यायची असेल, तर ते Ultimate Multisite वापरून शक्य आहे. तुम्ही प्रत्येक योजना स्तरावर तुमचे ग्राहक किती products तयार करू शकतील हेही मर्यादित करू शकता.

Ultimate Multisite साठी subscription payments च्या बाबतीत, तुम्ही [Ultimate Multisite: WooCommerce Integration Add-on](https://ultimatemultisite.com/addons/wp-ultimo-woocommerce/) वापरून तुमच्या Ultimate Multisite ग्राहकांना तुमच्या मुख्य साइटवर WooCommerce gateways वापरून पैसे भरण्याची परवानगी देऊ शकता. अशा प्रकारे, तुम्ही WooCommerce साठी विकसित केलेल्या payment gateways च्या मोठ्या संग्रहाचा वापर करून तुमच्या Ultimate Multisite ग्राहकांना बिल करू शकता.

  * **WPMUDEV Pro Sites वरून subscriptions migrate करण्याचा काही मार्ग आहे का?**

हो, Ultimate Multisite आता तुम्हाला [तुमच्या Pro Sites नेटवर्कला migrate करण्यासाठी एक उपयुक्त साधन](https://ultimatemultisite.com/addons/wp-ultimo-pro-sites-migrator/) देते.

  * **माझ्या Plan Edit Page वर "Allowed Plugins" tab रिकामा का आहे?**

Ultimate Multisite फक्त तुमच्या नेटवर्कवर network-activated नसलेल्या plugins ना परवानगी देणे किंवा नाकारणे तुम्हाला करू देते. जेव्हा एखादा plugin network-activated असतो, तेव्हा स्वाभाविकपणे तो तुमच्या नेटवर्कच्या प्रत्येक साइटवर डीफॉल्ट म्हणून लोड होतो.

जर तुम्हाला "Allowed Plugins" tab वर कोणतेही plugins दिसत नसतील, तर इच्छित plugin network-active आहे की नाही ते तपासा.

  * **मी माझ्या स्वतःच्या Pricing Table डिझाइनचा वापर करू शकतो का?**

हो. आवृत्ती 1.4.0 पासून, Ultimate Multisite template overrides ला सपोर्ट करते.

  * **Ultimate Multisite WP Engine सोबत काम करते का?**

हो. Ultimate Multisite ला WP Engine ने नेहमीच सपोर्ट केले आहे, परंतु domain mapping काम करण्यासाठी admin कडून मॅन्युअल अतिरिक्त पावले आवश्यक होती (WP Engine panel मध्ये domain जोडणे). **आवृत्ती 1.6.0 पासून आता हे आवश्यक नाही.** Ultimate Multisite थेट WP Engine API सोबत integrate होते आणि mapping नंतर domains आपोआप panel मध्ये जोडले जातात.

  * **तुम्ही कोणता hosting provider शिफारस करता?**

Ultimate Multisite कोणत्याही hosting provider सोबत काम करायला हवे. फक्त फरक हा आहे की Closte, Serverpilot, WPMUDEV, WP Engine, Cloudways, Gridpane आणि [Runcloud.io](http://Runcloud.io) सारख्या काही hosting कंपन्यांकडे domain mapping क्षमतांसाठी घट्ट integration आहे त्यामुळे ते लगेच काम करतात. इतर hostings साठी, तुम्हाला domain mapping मॅन्युअली सेट अप करावे लागू शकते.
