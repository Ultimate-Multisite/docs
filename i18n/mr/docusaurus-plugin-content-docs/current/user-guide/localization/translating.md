# Ultimate Multisite चे भाषांतर

Ultimate Multisite इंग्रजीत विकसित केले आहे, पण हे plugin पूर्णपणे भाषांतर करण्यायोग्य आहे आणि आम्ही भाषांतर योगदानांचे स्वागत करतो.

## मुख्य Plugin भाषांतरे

मुख्य Ultimate Multisite plugin चे भाषांतर अधिकृत **WordPress.org भाषांतर platform** द्वारे केले जाते:

**[translate.wordpress.org/projects/wp-plugins/ultimate-multisite/](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/)**

ही WordPress समुदायाची मानक भाषांतर प्रणाली आहे. जेव्हा भाषांतरे पुरेशा मंजुरी मर्यादेपर्यंत पोहोचतात, तेव्हा ती WordPress plugin updates द्वारे सर्व Ultimate Multisite वापरकर्त्यांना आपोआप वितरित केली जातात.

**आम्हाला तुमच्या मदतीची गरज आहे!** जर तुम्ही इंग्रजी व्यतिरिक्त दुसरी भाषा बोलत असाल, तर कृपया भाषांतरात योगदान देण्याचा विचार करा. काही strings चे भाषांतर केले तरी मदत होते. सुरू करण्यासाठी:

1. [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ultimate-multisite/) ला भेट द्या
2. तुमची भाषा निवडा
3. "Contribute Translation" वर क्लिक करा
4. तुम्हाला मोफत WordPress.org खाते लागेल

जर तुम्हाला तुमच्या भाषेसाठी भाषांतर संपादक बनायचे असेल (भाषांतरे मंजूर करण्याची क्षमता), तर [GitHub Discussions](https://github.com/Ultimate-Multisite/ultimate-multisite/discussions) द्वारे संपर्क साधा.

## Addon भाषांतरे

Addon plugins चे भाषांतर वेगळ्या ठिकाणी केले जाते:

**[translate.ultimatemultisite.com](https://translate.ultimatemultisite.com/)**

Addon भाषांतरे सध्या सुरुवातीचा आधार म्हणून AI-निर्मित आहेत. ती सर्व समर्थित भाषा कव्हर करतात पण त्यात अस्ताव्यस्त वाक्यरचना किंवा अचूकतेच्या त्रुटी असू शकतात. आम्ही मानवी भाषांतरकारांचे स्वागत करतो जे त्यांचे पुनरावलोकन करून सुधारणा सुचवतील आणि ती अधिक नैसर्गिक व अचूक बनवतील.

## स्थानिक भाषांतरे

जर तुम्ही upstream योगदान देण्याऐवजी तुमच्या स्वतःच्या प्रकल्पासाठी स्थानिक भाषांतरे तयार करू इच्छित असाल, तर दोन पर्याय आहेत:

### Poedit वापरून

तुम्ही `.pot` फाइलचे भाषांतर करण्यासाठी **[Poedit](https://poedit.net)** desktop application वापरू शकता:

1. [poedit.net](https://poedit.net) वरून Poedit डाउनलोड करा
2. `wp-content/plugins/ultimate-multisite/lang/` मधून `.pot` फाइल उघडा
3. Poedit interface मध्ये strings चे भाषांतर करा
4. Save करा -- Poedit `.po` आणि `.mo` फाइल्स तयार करते
5. Output फाइल्स `lang` फोल्डरमध्ये ठेवा

### Loco Translate वापरून

**[Loco Translate](https://wordpress.org/plugins/loco-translate/)** WordPress plugin तुम्हाला कोणत्याही desktop tools शिवाय थेट admin panel मधून भाषांतर करू देते. ते तुमच्यासाठी भाषांतर फाइल्स तयार करणे आणि compile करणे हाताळते.
