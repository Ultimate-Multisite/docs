# Ultimate Multisite Hooks आणि filters

Ultimate Multisite हे अनेक functions आणि database queries चा वापर करून बनवलेले आहे. यात hooks आणि filters आहेत जे तुम्हाला विद्यमान functions मध्ये बदल करण्यास किंवा नवीन code जोडण्यास मदत करतात.

प्रगत वापरकर्त्यांना त्यांचे network अधिक customize करण्यासाठी आम्ही या hooks आणि filters ची यादी संदर्भ म्हणून तयार केली आहे. खालील लिंकवर क्लिक करून तुम्ही ही यादी Github वरून पाहू शकता.

https://github.com/next-press/wp-ultimo-hooks-and-filters/blob/main/wp-ultimo.md
