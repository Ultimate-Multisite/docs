# सिस्टम ईमेल्स व्यवस्थापित करणे

Ultimate Multisite विविध सिस्टम ईमेल्स पाठवते जसे की साइनअप पुष्टीकरण, पेमेंट पावत्या आणि मेंबरशिप बदलांसाठी. तुम्ही हे ईमेल टेम्प्लेट्स तुमच्या ब्रँडनुसार सानुकूलित करू शकता.

## सिस्टम ईमेल्स पाहणे

**Ultimate Multisite → Settings** वर जा आणि ईमेल-संबंधित सेटिंग्ज पाहण्यासाठी **Emails** टॅबवर क्लिक करा.

![Settings - Emails tab](/img/admin/settings-emails.png)

तिथून, तुम्ही सर्व ईमेल टेम्प्लेट्स पाहण्यासाठी **System Emails** पेजवर जाऊ शकता.

![System Emails list](/img/admin/system-emails-list.png)

## नवीन ईमेल टेम्प्लेट्स इंपोर्ट करणे

नवीन Ultimate Multisite आवृत्त्या आणि add-ons नवीन ईमेल टेम्प्लेट्स नोंदणी करू शकतात. conflicts टाळण्यासाठी, नवीन टेम्प्लेट्स आपोआप जोडले जात नाहीत — तुम्हाला ते स्वतः इंपोर्ट करावे लागतात.

नवीन ईमेल टेम्प्लेट्स इंपोर्ट करण्यासाठी:

1. **System Emails** पेजवर जा
2. पेजच्या वरच्या बाजूला असलेल्या **Reset & Import** बटणावर क्लिक करा
3. उपलब्ध टेम्प्लेट्स पाहण्यासाठी **Import Emails** पर्याय चालू करा
4. तुम्हाला इंपोर्ट करायचे असलेले टेम्प्लेट्स निवडा आणि पुष्टी करा

## ईमेल टेम्प्लेट्स रीसेट करणे

तुम्ही एखादे ईमेल टेम्प्लेट सानुकूलित केले असेल आणि ते डीफॉल्ट कंटेंटवर परत आणायचे असेल, तर तुमच्याकडे दोन पर्याय आहेत:

1. **डिलीट करा आणि पुन्हा इंपोर्ट करा** — सिस्टम ईमेल डिलीट करा आणि पुन्हा इंपोर्ट करा (यामुळे send metrics मिटतात)
2. **टूलद्वारे रीसेट करा** — **Reset & Import** टूल वापरा, **Reset** पर्याय चालू करा आणि तुम्हाला रीसेट करायचे असलेले ईमेल्स निवडा

Reset पर्याय अधिक चांगला आहे कारण यामुळे send metrics आणि इतर metadata जतन राहतात आणि डीफॉल्ट कंटेंट पुनर्संचयित होतो.
