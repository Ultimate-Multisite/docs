# Dashboard चा आढावा

Ultimate Multisite चे एक समर्पित **Dashboard** Ultimate Multisite मेनूमध्ये आहे, आणि ते Network Dashboard मध्ये widgets देखील जोडते. या लेखात तुम्हाला दोन्ही ठिकाणी काय दिसेल ते समाविष्ट आहे.

## Network Dashboard

Ultimate Multisite नेटवर्क-स्तरीय dashboard मध्ये उपयुक्त widgets जोडते जे **My Sites → Network Admin → Dashboard** येथे आढळते.

![Ultimate Multisite widgets सह Network Dashboard](/img/admin/network-dashboard-full.png)

प्रत्येक widget काय प्रदान करते ते येथे आहे:

**First Steps:** नवीन installation वर दिसते आणि तुम्हाला payment method कॉन्फिगर करणे आणि test account तयार करणे यासारखी setup कार्ये पूर्ण करण्याची आठवण करून देते.

![First Steps widget](/img/admin/network-dashboard-first-steps.png)

**Summary:** दिवसासाठी signups आणि revenue ची संख्या दर्शवते. MRR (Monthly Recurring Revenue) revenue-generating memberships असलेल्या ग्राहकांमधील अंदाजित एकूण रक्कम दर्शवते.

![Summary widget](/img/admin/network-dashboard-summary.png)

**Activity Stream:** Ultimate Multisite मधील घडामोडींचा अहवाल देते — signups, cancellations, site creations, deletions आणि इतर activities.

![Activity Stream widget](/img/admin/network-dashboard-activity-stream.png)

**Right Now:** नेटवर्कमधील users आणि sites चा संक्षिप्त सारांश नवीन sites किंवा users तयार करण्यासाठी shortcuts सह दर्शवते.

![Right Now widget](/img/admin/network-dashboard-right-now.png)

**News & Discussions:** updates, critical bugs आणि security fixes यासह नवीनतम Ultimate Multisite बातम्या आणते आणि दर्शवते.

![News widget](/img/admin/network-dashboard-news.png)

## Ultimate Multisite Dashboard

Network dashboard नेटवर्कबद्दल सर्वसमावेशक माहिती सादर करत असताना, Ultimate Multisite dashboard (Ultimate Multisite मेनूच्या सर्वात वरच्या स्तरावर) सेवेबद्दल माहिती सादर करते.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

**Monthly Recurring Revenue Growth:** 12 महिन्यांच्या calendar कालावधीत महिन्याच्या हिशेबाने revenue चे विभाजन दर्शवते. नवीन revenue निळ्या रंगात दर्शवला जातो तर cancellations लाल रंगात दिसतात.

![MRR Growth chart](/img/admin/um-dashboard-mrr.png)

**Date-range Support:** तुम्हाला आवश्यक असलेल्या माहितीवर लक्ष केंद्रित करण्यासाठी कालावधीनुसार dashboard डेटा फिल्टर करा. Date range खालील widgets मध्ये काय दर्शवले जाते ते नियंत्रित करते.

![Date range आणि dashboard stats](/img/admin/um-dashboard-date-range.png)

Dashboard widgets मध्ये समाविष्ट आहे:

- **Revenue** — कालावधीसाठी एकूण revenue MRR आणि प्रति-product विभाजनासह
- **New Memberships** — निवडलेल्या कालावधीत किती नवीन memberships तयार झाल्या
- **Signups by Countries** — signups चे भौगोलिक विभाजन
- **Signups by Forms** — किती ग्राहकांनी प्रत्येक checkout form द्वारे signup केले विरुद्ध admins ने जोडले
- **Most Visited Sites** — तुमच्या नेटवर्कवरील सर्वाधिक भेट दिलेल्या sites (यासाठी **Ultimate Multisite → Settings → Sites → Enable Visits Limitation and Counting** येथे visit counting सक्षम करणे आवश्यक आहे)

![Dashboard statistics](/img/admin/um-dashboard-stats.png)
