# Admin Panel ची सफर

Ultimate Multisite शक्य तितके सोपे आणि स्पष्ट असण्याचा प्रयत्न करते, पण इतक्या शक्तिशाली साधनासाठी एक मार्गदर्शित सफर उपयुक्त ठरू शकते. चला admin pages मधून फिरूया आणि तुम्हाला सगळं नीट समजून घेऊया.

## Dashboard

Ultimate Multisite **Dashboard** मध्ये मूलभूत अहवाल आणि analytics दिसतात, ज्यात महसूल, साइट क्रियाकलाप, सदस्यत्व वाढ, अभ्यागत संख्या, आणि भौगोलिक माहिती यांचा समावेश आहे.

![Ultimate Multisite Dashboard](/img/admin/um-dashboard-top.png)

## Checkout Forms

**Checkout Forms** तुम्हाला सानुकूल नोंदणी पृष्ठे तयार करण्याची लवचिकता देतात. तुम्ही फील्ड्स सानुकूलित करू शकता, custom classes आणि scripts जोडू शकता, आणि अभ्यागतांच्या देश किंवा स्थानावर आधारित प्रवेश प्रतिबंधित करू शकता.

![Checkout Forms list](/img/admin/checkout-forms-list.png)

## Products

**Products** विभागात तुम्ही तुमच्या network साठी विविध योजना, पॅकेजेस, आणि सेवा तयार करता. किंमती, बिलिंग वारंवारता, आणि मर्यादा व कोटा product स्तरावर निश्चित करा.

![Products list](/img/admin/products-list.png)

## Memberships

**Memberships** पृष्ठ तुमच्या network मधील सर्व subscriptions दाखवते. योजना, products, बिलिंग रक्कम आणि वारंवारता, पेमेंट इतिहास, आणि timestamps यासह subscription तपशील पहा आणि संपादित करा.

![Memberships list](/img/admin/memberships-list.png)

## Payments

**Payments** पृष्ठ तुमच्या संपूर्ण network मधील पेमेंट इतिहासाचे त्वरित दृश्य देते, ज्यात विशिष्ट products आणि रकमांसह तपशीलवार व्यवहार माहिती असते.

![Payments list](/img/admin/payments-list.png)

## Customers

**Customers** पृष्ठ तुमच्या network मध्ये नोंदणीकृत सर्व सदस्य दाखवते, ज्यात नाव, ईमेल पत्ता, शेवटचे लॉगिन, आणि त्यांच्या subsite मध्ये लॉगिन करण्यासाठी "switch to" लिंक यासह मूलभूत वापरकर्ता माहिती असते.

![Customers list](/img/admin/customers-list.png)

## Sites

**Sites** पृष्ठ तुमच्या network मधील सर्व subsites सूचीबद्ध करते, ज्यात site templates आणि ग्राहक-मालकीच्या साइट्स यांचा समावेश आहे. mapped domains, मर्यादा व कोटा, plugins आणि themes, आणि site thumbnails सहज व्यवस्थापित करा.

![Sites list](/img/admin/sites-list.png)

## Domains

**Domains** पृष्ठ subsites ला map केलेल्या custom domains साठी समर्पित आहे. super admin म्हणून, तुम्ही manually custom domain names जोडू किंवा map करू शकता आणि DNS records व logs पाहू शकता.

![Domains list](/img/admin/domains-list.png)

## Discount Codes

**Discount Codes** तुम्हाला तुमच्या end-users ना सवलती देण्यासाठी coupon codes तयार करू देतात. सवलत मूल्य निर्दिष्ट करा आणि विशिष्ट वापरकर्ते किंवा गटांना लक्ष्य करा.

![Discount Codes list](/img/admin/discount-codes-list.png)

## Broadcasts

**Broadcasts** हे तुमच्या subsite वापरकर्त्यांना घोषणा पाठवण्यासाठी किंवा खाजगी संदेश पाठवण्यासाठी एक साधन आहे. ते subscribe केलेल्या योजना किंवा product वर आधारित विशिष्ट वापरकर्ता गटांना लक्ष्य करा.

![Broadcasts list](/img/admin/broadcasts-list.png)

## Settings

**Settings** पृष्ठ म्हणजे जिथे तुम्ही Ultimate Multisite कॉन्फिगर करता — नोंदणी सेटिंग्ज, पेमेंट्स, API आणि webhooks, domain mapping, आणि इतर integrations.

![Settings page](/img/admin/settings-general.png)

## Events

**Events** पृष्ठ तुमच्या network मधील सर्व events आणि logs ची नोंद ठेवते. हे योजना बदल, साइनअप्स, आणि इतर हालचाली यासारख्या क्रियाकलापांचा मागोवा घेते — तुमच्या multisite चे निरीक्षण करण्यासाठी उपयुक्त.

![Events list](/img/admin/events-list.png)

## Webhooks

**Webhooks** तुम्हाला दुसऱ्या application ला डेटा पाठवण्याची परवानगी देतात. Ultimate Multisite वरून Zapier सारख्या platforms ला डेटा पाठवण्यासाठी उपयुक्त.

![Webhooks list](/img/admin/webhooks-list.png)
