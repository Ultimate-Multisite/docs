# पेमेंट आणि इनव्हॉइस व्यवस्थापन

## पेमेंट सेटिंग्ज

पेमेंट स्वीकारणे सुरू करण्यापूर्वी, तुम्हाला पेमेंट-संबंधित सेटिंग्ज कॉन्फिगर कराव्या लागतील. **Ultimate Multisite → Settings** वर जा आणि **Payment** टॅबवर क्लिक करा.

![Payment settings tab](/img/admin/settings-payments-top.png)

### सामान्य पेमेंट पर्याय

सामान्य सेटिंग्जमध्ये तुम्ही हे कॉन्फिगर करू शकता:

- **Currency** — व्यवहारांसाठी वापरली जाणारी डीफॉल्ट चलन
- **Currency Position** — चलन चिन्ह कुठे दिसेल (रकमेच्या आधी/नंतर)

![Payment gateway settings](/img/admin/settings-payments-options.png)

### पेमेंट गेटवे

Ultimate Multisite अनेक पेमेंट गेटवेंना सपोर्ट करते. तुम्ही Payment सेटिंग्ज टॅबमधून प्रत्येक गेटवे सक्षम आणि कॉन्फिगर करू शकता.

![Payment gateway configuration](/img/admin/settings-payments-gateways.png)

उपलब्ध गेटवेमध्ये हे समाविष्ट आहेत:

- **Stripe** — Stripe द्वारे क्रेडिट कार्ड पेमेंट
- **PayPal** — PayPal पेमेंट
- **Manual** — ऑफलाइन किंवा कस्टम पेमेंट प्रोसेसिंगसाठी

प्रत्येक गेटवेचा स्वतःचा कॉन्फिगरेशन विभाग असतो जिथे तुम्ही API keys आणि इतर सेटिंग्ज एंटर करता.

![Additional gateway settings](/img/admin/settings-payments-gateways-2.png)

### Sandbox Mode

तुमचे पेमेंट इंटिग्रेशन लाइव्ह करण्यापूर्वी टेस्ट करण्यासाठी तुम्ही **Sandbox Mode** सक्षम करू शकता. Sandbox mode सक्रिय असताना, कोणतेही खरे पैसे कापले जाणार नाहीत.

## पेमेंट पाहणे

तुमच्या नेटवर्कवरील सर्व व्यवहार पाहण्यासाठी Ultimate Multisite अंतर्गत **Payments** पेजवर जा.

![Payments list](/img/admin/payments-list.png)

तुम्ही स्थितीनुसार (completed, pending, failed, refunded) पेमेंट फिल्टर करू शकता आणि विशिष्ट व्यवहार शोधू शकता.

पूर्ण तपशील पाहण्यासाठी पेमेंटवर क्लिक करा, ज्यामध्ये line items, संबंधित मेंबरशिप, ग्राहक माहिती आणि पेमेंट गेटवे डेटा समाविष्ट असतो.

## इनव्हॉइस

Ultimate Multisite पेमेंटसाठी आपोआप इनव्हॉइस तयार करू शकते. तुम्ही Payment सेटिंग्जमधून इनव्हॉइस टेम्पलेट आणि नंबरिंग फॉरमॅट कस्टमाइझ करू शकता.

इनव्हॉइस कस्टमायझेशन पर्यायांमध्ये हे समाविष्ट आहे:

- इनव्हॉइसवर दिसणारे **कंपनीचे नाव आणि पत्ता**
- **इनव्हॉइस नंबरिंग** फॉरमॅट आणि क्रम
- इनव्हॉइस हेडरवर दिसणारा **लोगो**
- अटी, नोट्स किंवा कायदेशीर माहितीसाठी **कस्टम फूटर टेक्स्ट**

इनव्हॉइस टेम्पलेट कस्टमाइझ करण्यासाठी, **Ultimate Multisite → Settings → Payment** वर जा आणि इनव्हॉइस-संबंधित सेटिंग्ज शोधा.
