# पेमेंट्स आणि इनव्हॉइसेस व्यवस्थापित करणे

## पेमेंट सेटिंग्ज {#payment-settings}

पेमेंट्स स्वीकारण्यास सुरुवात करण्यापूर्वी, तुम्हाला पेमेंट-संबंधित सेटिंग्ज कॉन्फिगर कराव्या लागतील. **Ultimate Multisite → Settings** येथे जा आणि **Payment** टॅबवर क्लिक करा.

![पेमेंट सेटिंग्ज टॅब](/img/admin/settings-payments-top.png)

पेमेंट सेटिंग्ज पृष्ठाचे संपूर्ण दृश्य येथे आहे:

![पेमेंट सेटिंग्ज पूर्ण पृष्ठ](/img/admin/settings-payments-full.png)

### सामान्य पेमेंट पर्याय {#general-payment-options}

सामान्य सेटिंग्जमध्ये तुम्ही कॉन्फिगर करू शकता:

- **चलन** — व्यवहारांसाठी वापरले जाणारे डीफॉल्ट चलन
- **चलनाची स्थिती** — चलन चिन्ह कुठे दिसते (रकमेच्या आधी/नंतर)

![पेमेंट गेटवे सेटिंग्ज](/img/admin/settings-payments-options.png)

### पेमेंट गेटवे {#payment-gateways}

Ultimate Multisite अनेक पेमेंट गेटवेला समर्थन देते. तुम्ही Payment सेटिंग्ज टॅबमधून प्रत्येक एक सक्षम आणि कॉन्फिगर करू शकता.

![पेमेंट गेटवे कॉन्फिगरेशन](/img/admin/settings-payments-gateways.png)

उपलब्ध गेटवे यामध्ये समाविष्ट आहेत:

- **Stripe** — Stripe द्वारे क्रेडिट कार्ड पेमेंट्स
- **PayPal** — PayPal पेमेंट्स
- **मॅन्युअल** — ऑफलाइन किंवा सानुकूल पेमेंट प्रक्रियेसाठी

प्रत्येक गेटवेसाठी स्वतःचा कॉन्फिगरेशन विभाग आहे, जिथे तुम्ही API keys आणि इतर सेटिंग्ज प्रविष्ट करता.

![अतिरिक्त गेटवे सेटिंग्ज](/img/admin/settings-payments-gateways-2.png)

### Sandbox Mode {#sandbox-mode}

लाइव्ह जाण्यापूर्वी तुमचे पेमेंट इंटिग्रेशन तपासण्यासाठी तुम्ही **Sandbox Mode** सक्षम करू शकता. सँडबॉक्स मोड सक्रिय असताना, कोणतेही वास्तविक शुल्क आकारले जाणार नाही.

## पेमेंट्स पाहणे {#viewing-payments}

तुमच्या नेटवर्कमधील सर्व व्यवहार पाहण्यासाठी Ultimate Multisite अंतर्गत **Payments** पृष्ठावर जा.

![पेमेंट्स यादी](/img/admin/payments-list.png)

तुम्ही पेमेंट्स स्थितीनुसार (पूर्ण, प्रलंबित, अयशस्वी, परतावा दिलेले) फिल्टर करू शकता आणि विशिष्ट व्यवहार शोधू शकता.

लाइन आयटम, संबंधित membership, ग्राहक माहिती, आणि पेमेंट गेटवे डेटा यासह संपूर्ण तपशील पाहण्यासाठी पेमेंटवर क्लिक करा.

## इनव्हॉइसेस {#invoices}

Ultimate Multisite पेमेंट्ससाठी आपोआप इनव्हॉइसेस तयार करू शकते. तुम्ही Payment सेटिंग्जमधून इनव्हॉइस टेम्पलेट आणि क्रमांकन स्वरूप सानुकूलित करू शकता.

इनव्हॉइस सानुकूलन पर्यायांमध्ये समाविष्ट आहे:

- इनव्हॉइसेसवर प्रदर्शित केलेले **कंपनीचे नाव आणि पत्ता**
- **इनव्हॉइस क्रमांकन** स्वरूप आणि क्रम
- इनव्हॉइस हेडरवर प्रदर्शित केलेला **लोगो**
- अटी, नोंदी किंवा कायदेशीर माहितीसाठी **सानुकूल फूटर मजकूर**

इनव्हॉइस टेम्पलेट सानुकूलित करण्यासाठी, **Ultimate Multisite → Settings → Payment** येथे जा आणि इनव्हॉइस-संबंधित सेटिंग्ज शोधा.
