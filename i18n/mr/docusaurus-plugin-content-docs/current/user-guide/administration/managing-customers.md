# ग्राहक व्यवस्थापन

**Customers** पेज तुमच्या नेटवर्कमध्ये नोंदणीकृत सदस्यांची यादी दाखवतो. या पेजवर नाव, ईमेल पत्ता, शेवटचे लॉगिन यासारखी मूलभूत वापरकर्ता माहिती दिसते, तसेच **switch to** लिंक असते जी तुम्हाला वापरकर्त्याच्या subsite मध्ये site administrator म्हणून लॉगिन करू देते.

![Customers list page](/img/admin/customers-list.png)

त्या ग्राहकाबद्दल सविस्तर माहिती पाहण्यासाठी ग्राहकाच्या कार्डखाली **Manage** वर क्लिक करा.

ग्राहक तपशील पेजवर, **At a Glance** ब्लॉक ग्राहक ID, शेवटच्या लॉगिनची वेळ आणि या ग्राहकाने खर्च केलेली एकूण रक्कम दाखवतो.

**Customer Options** ब्लॉक तुम्हाला या ग्राहकाला **VIP** म्हणून सेट करू देतो (त्यांना सहज शोधता यावे यासाठी एक टॅग). **Billing Info** टॅब ईमेल, पत्ता आणि देश यासारखी बिलिंग माहिती दाखवतो.

**Fields** ब्लॉक ग्राहकाचे username, ईमेल दाखवतो आणि तुम्हाला **Email Verification** स्थिती बदलण्याची परवानगी देतो.

Customer Options ब्लॉकच्या खाली तुम्हाला चार अतिरिक्त ब्लॉक्स मिळतील:

- **Memberships** — ग्राहकाने सदस्यत्व घेतलेली सध्याची subscriptions
- **Payments** — ग्राहकाचा पेमेंट इतिहास
- **Sites** — या ग्राहकाच्या मालकीच्या साइट्स
- **Events** — या ग्राहकाशी संबंधित घटना (ईमेल पडताळणी, बिलिंग बदल, इ.)

कोणत्याही आयटमच्या तपशील पेजवर जाण्यासाठी त्याच्या शेजारी असलेल्या बाण चिन्हावर (→) क्लिक करा. उदाहरणार्थ, साइटच्या शेजारी बाणावर क्लिक केल्यास तुम्ही साइट संपादन पेजवर जाता जिथे तुम्ही membership association, साइट प्रकार आणि सक्रिय स्थिती बदलू शकता.

हेच memberships आणि payments साठी लागू होते — प्रत्येक तपशील पेजशी जोडलेले आहे जिथे तुम्ही membership स्थिती, संबंधित ग्राहक, उत्पादने, payment gateway सेटिंग्ज संपादित करू शकता आणि refunds देखील प्रक्रिया करू शकता.

ग्राहक पेजच्या उजव्या बाजूला, तुम्हाला आढळेल:

- **Last Login & IPs** — शेवटच्या लॉगिनची वेळ, IP पत्ता आणि देश
- **Timestamps** — ग्राहक खाते कधी तयार केले आणि शेवटचे बदलले
- **Delete** — ग्राहक कायमचा हटवण्याचा पर्याय

:::warning
ग्राहक हटवणे कायमस्वरूपी आहे आणि ते पूर्ववत करता येत नाही.
:::
