# साइट टेम्पलेट्समध्ये Auto Search and Replace (v2)

_**या ट्यूटोरियलसाठी WP UItimo आवृत्ती 2.x आवश्यक आहे.**_

Ultimate Multisite च्या सर्वात शक्तिशाली वैशिष्ट्यांपैकी एक म्हणजे नोंदणी फॉर्ममध्ये मनमानी मजकूर, रंग आणि निवड फील्ड्स जोडण्याची क्षमता. एकदा तो डेटा कॅप्चर झाला की, निवडलेल्या साइट टेम्पलेटच्या काही भागांमधील सामग्री पूर्व-भरित करण्यासाठी आपण त्याचा वापर करू शकतो. त्यानंतर, नवीन साइट प्रकाशित झाल्यावर, Ultimate Multisite नोंदणीदरम्यान प्रविष्ट केलेल्या प्रत्यक्ष माहितीसह स्थानधारक बदलून टाकेल.

उदाहरणार्थ, तुम्ही तुमच्या टेम्पलेट साइट्स स्थानधारकांसह तयार करू शकता. स्थानधारक दुहेरी कर्ली ब्रेसेसमध्ये घालावेत - {{placeholder_name}}.

त्यानंतर, तो डेटा कॅप्चर करण्यासाठी तुम्ही फक्त जुळणारे नोंदणी फील्ड जोडू शकता

तुमचा ग्राहक नोंदणीदरम्यान ते फील्ड भरू शकेल.

Ultimate Multisite नंतर ग्राहकाने दिलेल्या डेटासह स्थानधारक आपोआप बदलून टाकेल.

## **"स्थानधारकांनी भरलेला टेम्पलेट" समस्या सोडवणे** {#solving-the-template-full-of-placeholders-problem}

हे सर्व उत्तम आहे, पण आपल्याला एक कुरूप समस्या येते: आता आमचे साइट टेम्पलेट्स - जे आमचे ग्राहक पाहू शकतात - कुरूप स्थानधारकांनी भरलेले असतात, जे फारसे काही सांगत नाहीत.

ते सोडवण्यासाठी, आम्ही स्थानधारकांसाठी बनावट मूल्ये सेट करण्याचा पर्याय देतो, आणि तुमचे ग्राहक भेट देत असताना टेम्पलेट साइट्सवरील त्यांची सामग्री शोधून बदलण्यासाठी आम्ही ती मूल्ये वापरतो.

तुम्ही **Ultimate Multisite > Settings > Sites** येथे जाऊन, Site Template Options क्षेत्रापर्यंत स्क्रोल करून, आणि नंतर **Edit Placeholders** लिंकवर क्लिक करून टेम्पलेट स्थानधारक संपादकात प्रवेश करू शकता.

![Sites सेटिंग्ज पृष्ठातील Site Template Options क्षेत्र](/img/config/settings-sites-templates-section.png)

हे तुम्हाला स्थानधारकांच्या सामग्री संपादकाकडे घेऊन जाईल, जिथे तुम्ही स्थानधारक आणि त्यांची संबंधित सामग्री जोडू शकता.

![टेम्पलेट स्थानधारक संपादक प्रवेश बिंदू](/img/config/settings-sites-templates-section.png)
