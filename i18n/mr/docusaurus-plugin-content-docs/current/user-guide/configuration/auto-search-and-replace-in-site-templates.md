# साइट टेम्प्लेट्समध्ये ऑटो शोधा आणि बदला (v2)

_**या ट्युटोरियलसाठी WP UItimo आवृत्ती 2.x आवश्यक आहे.**_

Ultimate Multisite च्या सर्वात शक्तिशाली वैशिष्ट्यांपैकी एक म्हणजे नोंदणी फॉर्ममध्ये कोणतेही टेक्स्ट, रंग आणि सिलेक्ट फील्ड्स जोडण्याची क्षमता. एकदा ही माहिती मिळाल्यावर, आपण ती निवडलेल्या साइट टेम्प्लेटच्या विशिष्ट भागांमध्ये आधीच भरण्यासाठी वापरू शकतो. त्यानंतर, जेव्हा नवीन साइट प्रकाशित होते, तेव्हा Ultimate Multisite नोंदणी दरम्यान भरलेल्या वास्तविक माहितीने प्लेसहोल्डर्स बदलते.

उदाहरणार्थ, तुम्ही तुमच्या टेम्प्लेट साइट्स प्लेसहोल्डर्ससह तयार करू शकता. प्लेसहोल्डर्स दुहेरी कुरळे कंसांनी वेढलेले असावेत - {{placeholder_name}}.

त्यानंतर, ती माहिती मिळवण्यासाठी तुम्ही फक्त जुळणारे नोंदणी फील्ड जोडू शकता

तुमचा ग्राहक नोंदणी दरम्यान ते फील्ड भरू शकेल.

Ultimate Multisite नंतर ग्राहकाने दिलेल्या माहितीने प्लेसहोल्डर्स आपोआप बदलेल.

## **"प्लेसहोल्डर्सने भरलेले टेम्प्लेट" समस्येचे निराकरण**

हे सर्व छान आहे, पण आपल्यासमोर एक कुरूप समस्या येते: आता आपल्या साइट टेम्प्लेट्स - ज्या आपले ग्राहक भेट देऊ शकतात - कुरूप प्लेसहोल्डर्सने भरलेल्या आहेत जे फारसे काही सांगत नाहीत.

हे सोडवण्यासाठी, आम्ही प्लेसहोल्डर्ससाठी तात्पुरती मूल्ये सेट करण्याचा पर्याय देतो, आणि तुमचे ग्राहक टेम्प्लेट साइट्सला भेट देत असताना त्या मूल्यांचा वापर करून शोधा आणि बदला क्रिया करतो.

**Ultimate Multisite > Settings > Sites** वर जाऊन तुम्ही टेम्प्लेट प्लेसहोल्डर्स एडिटरमध्ये प्रवेश करू शकता, आणि नंतर साइडबारवर Edit Placeholders लिंकवर क्लिक करा.

![Sites सेटिंग्ज साइडबारमधील Edit Placeholders लिंक](/img/config/settings-sites.png)

यामुळे तुम्ही प्लेसहोल्डर्स कंटेंट एडिटरवर जाल, जिथे तुम्ही प्लेसहोल्डर्स आणि त्यांचे संबंधित कंटेंट जोडू शकता.

![प्लेसहोल्डर नावे आणि मूल्यांसह प्लेसहोल्डर्स कंटेंट एडिटर](/img/config/settings-sites.png)
