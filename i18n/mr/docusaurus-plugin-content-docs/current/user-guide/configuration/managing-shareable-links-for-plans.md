# प्लॅनसाठी शेअर करण्यायोग्य लिंक्स व्यवस्थापित करणे

Ultimate Multisite तुम्हाला तुमच्या क्लायंट्ससोबत प्लॅन लिंक्स थेट शेअर करण्याचा पर्याय देते. हे करण्यासाठी, तुमच्या कोणत्याही प्लॅनच्या edit page वर जा आणि _Click to copy the Shareable Link_ बटणावर क्लिक करा.

![शेअर करण्यायोग्य लिंक बटण असलेले Product edit page](/img/config/product-edit-full.png)

_प्लॅनच्या Edit Page वर, शेअर करण्यायोग्य लिंक्स तयार करण्यासाठी बटण मिळेल_

लक्षात घ्या की ही यंत्रणा लिंक तयार करण्यासाठी डिफॉल्ट बिलिंग फ्रिक्वेन्सी वापरते. म्हणजे जर तुमचे डिफॉल्ट बिलिंग मासिक असेल, उदाहरणार्थ, शेवटी तुम्हाला अशी लिंक मिळेल: [_**https://network.com/register/premium/**_](https://network.com/register/premium/)

जर तुम्हाला शेअर करण्यायोग्य लिंक तिमाही साठी डिफॉल्ट असावी असे वाटत असेल, तर तुम्हाला शेवटी 3 जोडावे लागेल

[_**https://network.com/register/premium/3**_](https://network.com/register/premium/3)

_premium_ प्लॅन वार्षिक शेड्यूलवर निवडण्यासाठी, तुम्ही 1 आणि नंतर हा शब्द जोडाल:

[_**https://network.com/register/premium/1/year**_](https://network.com/register/premium/1/year)

लक्षात ठेवा की वेगवेगळ्या बिलिंग फ्रिक्वेन्सीसाठी तुम्ही एकच लिंक ठेवू शकत नाही. तुम्ही तुमच्या प्रत्येक बिलिंग फ्रिक्वेन्सीसाठी एक विशिष्ट लिंक शेअर करावी.

एकदा तुम्ही तुमच्या ग्राहकांसोबत लिंक शेअर केली की, ती त्यांना तुमच्या साइनअप फ्लोच्या पहिल्या टप्प्यावर रिडायरेक्ट करेल.
