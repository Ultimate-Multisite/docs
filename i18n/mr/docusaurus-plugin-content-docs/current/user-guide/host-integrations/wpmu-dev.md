# WPMU DEV इंटिग्रेशन

## विहंगावलोकन
WPMU DEV हे एक सर्वसमावेशक WordPress प्लॅटफॉर्म आहे जे WordPress साइट्ससाठी होस्टिंग, plugins आणि सेवा प्रदान करते. हे इंटिग्रेशन Ultimate Multisite आणि WPMU DEV होस्टिंग दरम्यान स्वयंचलित डोमेन सिंकिंग आणि SSL सर्टिफिकेट व्यवस्थापन सक्षम करते.

## वैशिष्ट्ये
- स्वयंचलित डोमेन सिंकिंग
- SSL सर्टिफिकेट व्यवस्थापन
- वाढीव SSL सर्टिफिकेट पडताळणी प्रयत्न

## आवश्यकता
तुम्ही WPMU DEV वर होस्ट करत असल्यास हे इंटिग्रेशन आपोआप ते ओळखते आणि built-in API वापरते. तुम्ही WPMU DEV वर होस्ट करत असल्यास कोणत्याही अतिरिक्त कॉन्फिगरेशनची आवश्यकता नाही.

हे इंटिग्रेशन `WPMUDEV_HOSTING_SITE_ID` constant ची उपस्थिती तपासते, जी WPMU DEV वर होस्ट करताना आपोआप परिभाषित केली जाते.

## सेटअप सूचना

### 1. WPMU DEV होस्टिंगची पडताळणी करा

तुम्ही WPMU DEV वर होस्ट करत असल्यास, आवश्यक constants आधीच परिभाषित असायला हवेत. याची खात्री करा:

1. तुमच्या environment मध्ये `WPMUDEV_HOSTING_SITE_ID` constant परिभाषित आहे
2. तुमच्याकडे API अ‍ॅक्सेससह सक्रिय WPMU DEV सदस्यत्व आहे

### 2. इंटिग्रेशन सक्षम करा

1. तुमच्या WordPress admin मध्ये, Ultimate Multisite > Settings वर जा
2. "Domain Mapping" टॅबवर जा
3. "Host Integrations" पर्यंत खाली स्क्रोल करा
4. WPMU DEV इंटिग्रेशन सक्षम करा
5. "Save Changes" वर क्लिक करा

## हे कसे कार्य करते

### डोमेन सिंकिंग

Ultimate Multisite मध्ये डोमेन मॅप केल्यावर:

1. इंटिग्रेशन तुमच्या होस्टिंग अकाउंटमध्ये डोमेन जोडण्यासाठी WPMU DEV API वापरते
2. ते आपोआप डोमेनची www आवृत्ती देखील जोडते
3. WPMU DEV डोमेन कॉन्फिगरेशन आणि SSL सर्टिफिकेट जारी करण्याचे काम हाताळते

### SSL सर्टिफिकेट व्यवस्थापन

WPMU DEV होस्टिंगसाठी SSL सर्टिफिकेट पडताळणी प्रयत्नांची संख्या वाढवण्यासाठी इंटिग्रेशन कॉन्फिगर केले आहे, कारण SSL सर्टिफिकेट जारी आणि इंस्टॉल होण्यास काही वेळ लागू शकतो. डीफॉल्टनुसार, ते SSL सर्टिफिकेट पडताळणीसाठी 10 वेळा प्रयत्न करेल, तर मानक 5 प्रयत्न आहेत.

## महत्त्वाच्या नोट्स

### डोमेन काढणे

सध्या, WPMU DEV API डोमेन काढण्याचा मार्ग प्रदान करत नाही. Ultimate Multisite मध्ये डोमेन मॅपिंग काढल्यावर, डोमेन तुमच्या WPMU DEV होस्टिंग अकाउंटमध्ये राहील. आवश्यक असल्यास तुम्हाला ते WPMU DEV होस्टिंग dashboard वरून स्वतः काढावे लागेल.

### API ऑथेंटिकेशन

इंटिग्रेशन तुमच्या WordPress डेटाबेसमध्ये `wpmudev_apikey` option म्हणून संग्रहित WPMU DEV API key वापरते. तुम्ही तुमची साइट WPMU DEV शी कनेक्ट करता तेव्हा हे आपोआप सेट होते.

## समस्यानिवारण

### API कनेक्शन समस्या
- तुमची साइट WPMU DEV शी योग्यरित्या कनेक्ट आहे याची पडताळणी करा
- तुमच्या WordPress डेटाबेसमध्ये `wpmudev_apikey` option सेट आहे का ते तपासा
- तुमचे WPMU DEV सदस्यत्व सक्रिय असल्याची खात्री करा

### SSL सर्टिफिकेट समस्या
- WPMU DEV ला SSL सर्टिफिकेट जारी करण्यास काही वेळ लागू शकतो (सामान्यतः 5-15 मिनिटे)
- इंटिग्रेशन SSL सर्टिफिकेटसाठी 10 वेळा तपासण्यासाठी कॉन्फिगर केले आहे
- अनेक प्रयत्नांनंतरही SSL सर्टिफिकेट जारी होत नसल्यास, WPMU DEV support शी संपर्क साधा

### डोमेन जोडला गेला नाही
- कोणत्याही एरर मेसेजेससाठी Ultimate Multisite logs तपासा
- डोमेन आधीच WPMU DEV मध्ये जोडला नाही याची पडताळणी करा
- तुम्ही जोडत असलेल्या डोमेन्सची संख्या तुमच्या WPMU DEV होस्टिंग प्लॅनद्वारे समर्थित आहे याची खात्री करा
