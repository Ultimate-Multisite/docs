# WP Engine इंटिग्रेशन

## आढावा
WP Engine हे एक प्रीमियम मॅनेज्ड WordPress होस्टिंग प्लॅटफॉर्म आहे जे WordPress साइट्ससाठी ऑप्टिमाइझ्ड परफॉर्मन्स, सुरक्षा आणि स्केलेबिलिटी प्रदान करते. हे इंटिग्रेशन Ultimate Multisite आणि WP Engine दरम्यान स्वयंचलित डोमेन सिंकिंग सक्षम करते.

## वैशिष्ट्ये
- स्वयंचलित डोमेन सिंकिंग
- मल्टीसाइट इन्स्टॉलेशनसाठी सबडोमेन सपोर्ट
- WP Engine च्या विद्यमान सिस्टमशी अखंड इंटिग्रेशन

## आवश्यकता
तुम्ही WP Engine वर होस्ट करत असाल तर इंटिग्रेशन स्वयंचलितपणे ते ओळखते आणि बिल्ट-इन WP Engine API वापरते. जर WP Engine plugin सक्रिय आणि योग्यरित्या कॉन्फिगर केलेले असेल तर कोणत्याही अतिरिक्त कॉन्फिगरेशनची आवश्यकता नाही.

तथापि, जर तुम्हाला इंटिग्रेशन मॅन्युअली कॉन्फिगर करायचे असेल, तर तुम्ही तुमच्या `wp-config.php` फाइलमध्ये यापैकी एक constant डिफाइन करू शकता:

```php
define('WPE_APIKEY', 'your_api_key'); // प्राधान्य पद्धत
// OR
define('WPE_API', 'your_api_key'); // पर्यायी पद्धत
```

## सेटअप सूचना

### 1. WP Engine Plugin तपासा

तुम्ही WP Engine वर होस्ट करत असाल तर WP Engine plugin आधीच इन्स्टॉल आणि सक्रिय असावे. खात्री करा की:

1. WP Engine plugin सक्रिय आहे
2. `wp-content/mu-plugins/wpengine-common/class-wpeapi.php` ही फाइल अस्तित्वात आहे

### 2. इंटिग्रेशन सक्षम करा

1. तुमच्या WordPress admin मध्ये, Ultimate Multisite > Settings वर जा
2. "Domain Mapping" टॅबवर नेव्हिगेट करा
3. "Host Integrations" पर्यंत खाली स्क्रोल करा
4. WP Engine इंटिग्रेशन सक्षम करा
5. "Save Changes" वर क्लिक करा

## हे कसे काम करते

### डोमेन सिंकिंग

जेव्हा Ultimate Multisite मध्ये डोमेन मॅप केला जातो:

1. इंटिग्रेशन तुमच्या WP Engine इन्स्टॉलेशनमध्ये डोमेन जोडण्यासाठी WP Engine API वापरते
2. WP Engine डोमेन कॉन्फिगरेशन आणि SSL सर्टिफिकेट जारी करणे हाताळते
3. जेव्हा डोमेन मॅपिंग काढले जाते, तेव्हा इंटिग्रेशन WP Engine मधून डोमेन काढून टाकते

### सबडोमेन सपोर्ट

सबडोमेन मल्टीसाइट इन्स्टॉलेशनसाठी:

1. जेव्हा नवीन साइट तयार होते तेव्हा इंटिग्रेशन प्रत्येक सबडोमेन WP Engine मध्ये जोडते
2. WP Engine सबडोमेन कॉन्फिगरेशन हाताळते
3. जेव्हा साइट डिलीट केली जाते, तेव्हा इंटिग्रेशन WP Engine मधून सबडोमेन काढून टाकते

## महत्त्वाच्या नोट्स

### वाइल्डकार्ड डोमेन

सबडोमेन मल्टीसाइट इन्स्टॉलेशनसाठी, वाइल्डकार्ड डोमेन कॉन्फिगरेशनसाठी विनंती करण्यासाठी WP Engine सपोर्टशी संपर्क साधण्याची शिफारस केली जाते. यामुळे प्रत्येक सबडोमेन वैयक्तिकरित्या जोडण्याची गरज न पडता सर्व सबडोमेन स्वयंचलितपणे काम करतात.

### SSL सर्टिफिकेट्स

या इंटिग्रेशनद्वारे जोडलेल्या सर्व डोमेनसाठी WP Engine स्वयंचलितपणे SSL सर्टिफिकेट जारी करणे आणि रिन्यूअल हाताळते. कोणत्याही अतिरिक्त कॉन्फिगरेशनची आवश्यकता नाही.

## समस्यानिवारण

### API कनेक्शन समस्या
- WP Engine plugin सक्रिय आणि योग्यरित्या कॉन्फिगर केलेले आहे याची खात्री करा
- जर तुम्ही API key मॅन्युअली डिफाइन केली असेल, तर ती बरोबर आहे का ते तपासा
- API मध्ये समस्या असल्यास WP Engine सपोर्टशी संपर्क साधा

### डोमेन जोडला गेला नाही
- कोणत्याही एरर मेसेजसाठी Ultimate Multisite लॉग्स तपासा
- डोमेन आधीच WP Engine मध्ये जोडलेला नाही याची खात्री करा
- तुम्ही जोडत असलेल्या डोमेनच्या संख्येला तुमचा WP Engine प्लॅन सपोर्ट करतो याची खात्री करा

### सबडोमेन समस्या
- सबडोमेन काम करत नसल्यास, वाइल्डकार्ड डोमेन कॉन्फिगरेशनसाठी विनंती करण्यासाठी WP Engine सपोर्टशी संपर्क साधा
- मुख्य डोमेन आणि सबडोमेनसाठी तुमची DNS सेटिंग्ज योग्यरित्या कॉन्फिगर केलेली आहेत याची खात्री करा
