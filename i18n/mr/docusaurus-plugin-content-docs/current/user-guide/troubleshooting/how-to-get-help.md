# मदत कशी मिळवायची

तुम्हाला तुमची वैध License key मिळाली आहे, आणि त्यासोबतच Ultimate Multisite सेट अप करताना येणाऱ्या समस्या सोडवण्यासाठी आमच्या टीमची मदत घेण्याची सुविधा तुम्हाला मिळते.

या धड्यात license धारकांसाठी उपलब्ध असलेल्या support माध्यमांची आणि आमच्या support टीमशी संपर्क साधताना तुमच्याकडे कोणती माहिती तयार असावी याची माहिती दिली आहे.

## Help Site

खालील लिंकवर तुम्हाला आमच्या help site वरील विविध लेख आणि tutorials मिळू शकतात.

[)

आम्ही आमच्या वापरकर्त्यांना Ultimate Multisite च्या मूलभूत वैशिष्ट्यांशी आणि कार्यप्रणालीशी परिचित होण्यासाठी प्रोत्साहित करतो. आमच्या help site वर "Getting Started" लेख उपलब्ध आहेत आणि सामान्य प्रश्नांची उत्तरेही तिथे मिळू शकतात, जसे की add-ons कसे install करायचे, payment gateways कसे integrate करायचे, इत्यादी.

## Support शी संपर्क साधा

जर तुम्ही अडकलात आणि आमच्या help site वर उत्तर मिळत नसेल, तर तुम्ही आमच्या support टीमशी संपर्क साधू शकता - आमच्या website च्या उजव्या खालच्या कोपऱ्यातील speech bubble icon वर क्लिक करून किंवा थेट support@ultimatemultisite.com वर email पाठवून.

तांत्रिक समस्यांसाठी support शी संपर्क साधताना, कृपया तुमच्या समस्येचे सविस्तर वर्णन, screenshot, error messages, किंवा आमच्या support टीमला तुमची समस्या सहज समजण्यासाठी आणि तपासण्यासाठी उपयुक्त ठरेल अशी कोणतीही माहिती द्या. तुम्हाला तुमची license key देखील द्यावी लागेल. फक्त active license धारकांनाच support मिळण्याचा हक्क आहे.

## Community Site

आमची [community site](https://community.ultimatemultisite.com/) ही Ultimate Multisite वापरकर्त्यांना एकमेकांशी संवाद साधण्यासाठी एक व्यासपीठ आहे. हे एक माहिती प्रणाली म्हणून काम करते जिथे सदस्य पोस्ट करू शकतात, चर्चांवर टिप्पणी करू शकतात, सल्ला देऊ शकतात, सहकार्य करू शकतात किंवा crowdsource करू शकतात. खाजगी spaces/pages अनलॉक करण्यासाठी तुम्हाला तुमची Ultimate Multisite license key द्यावी लागेल.
