# इतर प्लगइन्ससोबत ज्ञात असंगतता

Ultimate Multisite इन्स्टॉल आणि सक्रिय केल्यानंतर, इतर प्लगइन्ससोबत न सोडवलेल्या असंगततेमुळे तुम्हाला समस्या येऊ शकतात.

आम्ही प्रत्येक नवीन रिलीझसोबत ही यादी कमी करण्यासाठी सक्रियपणे काम करत आहोत, परंतु जर एखादा प्लगइन येथे सूचीबद्ध असेल, तर पूर्णपणे कार्यशील Ultimate Multisite Network मिळवण्यासाठी तुम्हाला तो निष्क्रिय करावा लागेल.

येथे त्यांची यादी आणि अशा परिस्थितीत तुम्ही काय करावे ते दिले आहे:

**काही WPMU DEV प्लगइन्स** WPMU DEV द्वारे _आता देखभाल न केलेले_ काही प्लगइन्स Ultimate Multisite नोंदणी प्रक्रिया शेवटच्या टप्प्यावर थांबवतात (वापरकर्ता खाते तयार करू शकत नाही). या समस्या निर्माण करणाऱ्या प्लगइन्सची यादी येथे आहे:

  * Signup Language: _Ultimate Multisite मध्ये समान कार्यक्षमता देणारा एक अॅड-ऑन आहे, ज्याला Ultimate Multisite: Language Switcher म्हणतात._

  * Set Blog Description on Blog Creation: _हे Ultimate Multisite मध्ये अंगभूत आहे, त्यामुळे अतिरिक्त प्लगइनची गरज नाही._

  * New Blog Templates: _हे Ultimate Multisite मध्ये अंगभूत आहे, त्यामुळे अतिरिक्त प्लगइनची गरज नाही._

  * Set Password on Multisite Blog Creation: _हे Ultimate Multisite मध्ये अंगभूत आहे, त्यामुळे अतिरिक्त प्लगइनची गरज नाही._

  * Signup TOS: _हे Ultimate Multisite मध्ये अंगभूत आहे, त्यामुळे अतिरिक्त प्लगइनची गरज नाही._

  * Anti-Splog: कमी शक्तिशाली असले तरी, हे देखील Ultimate Multisite मध्ये _अंगभूत आहे._

**टीप** : इतर सर्व WPMU Dev प्लगइन्स Ultimate Multisite सोबत सामान्यपणे वापरता येतात. उदाहरणार्थ _Smush_ , _Forminator_ , _Defender,_ इत्यादी.
