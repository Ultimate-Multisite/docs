# Ultimate Multisite दस्तऐवजीकरण

Ultimate Multisite च्या दस्तऐवजीकरणावर आपले स्वागत आहे. Ultimate Multisite हे WordPress Multisite ला पूर्ण Website as a Service (WaaS) प्लॅटफॉर्ममध्ये रूपांतरित करते.

## दस्तऐवजीकरण विभाग {#documentation-sections}

### [वापरकर्ता मार्गदर्शक](/user-guide/getting-started/ultimate-multisite-101) {#user-guide}

तुम्ही तुमचे WaaS प्लॅटफॉर्म सेट करण्यासाठी, कॉन्फिगर करण्यासाठी आणि व्यवस्थापित करण्यासाठी आवश्यक असलेली सर्व माहिती येथे मिळेल. यात इन्स्टॉलेशन, पेमेंट गेटवे, डोमेन मॅपिंग, क्लायंट व्यवस्थापन आणि बरेच काही कव्हर केले आहे.

### [डेव्हलपर](/developer/hooks) {#developer-documentation}

Ultimate Multisite वर काम करणाऱ्या डेव्हलपर्ससाठी तांत्रिक संदर्भ. यात हुक्सचा संदर्भ, REST API दस्तऐवजीकरण आणि addon डेव्हलपमेंट मार्गदर्शक तत्त्वे समाविष्ट आहेत.

### [पूरक घटक](/addons) {#addons}

ही माहिती अशा सशुल्क addon plugins साठी आहे, जे Ultimate Multisite ला WooCommerce इंटिग्रेशन, ॲनालिटिक्स, डोमेन विक्री आणि इतर अतिरिक्त वैशिष्ट्यांसह विस्तारित करतात.
