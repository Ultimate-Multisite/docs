# Filter: wu_view_override_replaceable_views {#filter-wuviewoverridereplaceableviews}

डेव्हलपर्सना प्रतिस्थापित करण्यायोग्य (replaceable) यादीत अतिरिक्त फोल्डर्स जोडण्याची ही परवानगी देते.

काळजीपूर्वक वापरा, कारण अतिरिक्त फोल्डर्सची परवानगी दिल्यास, Ultimate Multisite च्या आवृत्त्यांऐवजी जुन्या प्रती लोड होण्याची शक्यता आहे.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $replaceable_views | `array` | अनुमत फोल्डर्सची यादी. |

### Since {#since}

- 2.0.0
### Source {#source}

[`inc/functions/template.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/template.php#L50) मध्ये लाइन ५० वर परिभाषित केले आहे.
