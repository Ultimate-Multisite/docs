# Filter: wu_add_new_domain_modal_fields

नवीन डोमेन जोडण्याच्या (add new domain modal) विंडोमधील फील्ड्सला फिल्टर करतो.

यामुळे अॅडऑनला (उदा. Domain Seller) डोमेन नोंदणीचे फील्ड्स बदलण्याची किंवा त्यांची जागा घेण्याची (replace) परवानगी मिळते.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $fields | `array` | फॉर्मचे फील्ड्स. |
