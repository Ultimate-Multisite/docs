# Filter: wu_documentation_get_link

प्लगइन डेव्हलपर्सना लिंक्स फिल्टर करण्याची परवानगी देते.

हे कशासाठी उपयोगी ठरेल हे स्पष्ट नाही, पण ते ठेवण्यात काहीच वाईट नाही.

## पॅरामीटर्स {#parameters}

| नाव | प्रकार | वर्णन |
|------|------|-------------|
| $link | `string` | नोंदवलेली लिंक |
| $slug | `string` | लिंक मिळवण्यासाठी वापरलेला स्लग |
| $default_link | `string` | नोंदवलेली डीफॉल्ट लिंक |
