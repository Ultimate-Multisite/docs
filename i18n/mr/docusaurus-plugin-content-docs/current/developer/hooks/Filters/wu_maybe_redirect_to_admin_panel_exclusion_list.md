# फिल्टर: wu_maybe_redirect_to_admin_panel_exclusion_list

आपोआप रीडायरेक्ट (auto-redirect) होण्यापासून रोखण्यासाठी पॅरामीटर्सची एक वगळण्याची यादी (exclusion list) तयार करतो.

हे आवश्यक आहे, कारण अन्यथा, एकदा लॉगिन पेज (login page) परिभाषित केल्यावर पेज बिल्डरला ते एडिट करता येणार नाही.

### कधीपासून (Since)

- 2.0.4
### स्रोत (Source)

[`inc/checkout/class-checkout-pages.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-checkout-pages.php#L357) मध्ये लाइन 357 वर परिभाषित केले आहे.
