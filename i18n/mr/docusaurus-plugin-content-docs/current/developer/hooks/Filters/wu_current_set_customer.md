# Filter: wu_current_set_customer

विकसकांना (developers) डीफॉल्ट वर्तनात बदल करण्याची आणि सध्याच्या ग्राहकाला (current customer) वेगळ्या पद्धतीने सेट करण्याची परवानगी देतो.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| $customer | `\WP_Ultimo\Models\Customer` | सेट करण्यासाठी सध्याचा ग्राहक (current customer). |
| $ | `self` | सध्याच्या क्लासची इन्स्टन्स (Current class instance). |

- 2.0.9
