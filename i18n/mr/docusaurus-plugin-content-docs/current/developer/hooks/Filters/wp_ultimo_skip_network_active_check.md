# फिल्टर: wp_ultimo_skip_network_active_check

विकसकांना (developers) या तपासणीला बायपास (short-circuit) करण्याची परवानगी देतो.

जेव्हा तुम्ही composer-आधारित आणि इतर कस्टमाइज्ड सेटअप्स (custom setups) वापरत असता, जसे की Bedrock, जिथे mu-plugins म्हणून प्लगइन्स वापरणे ही सामान्य पद्धत आहे, तेव्हा हे उपयुक्त ठरते.

- 2.0.0
