# Filter: wu_cart_get_subtotal

"प्रारंभिक रक्कम" (initial amount) या एकूण रकमेला फिल्टर करण्यासाठी याचा वापर होतो.

## पॅरामीटर्स {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $subtotal | `float` | आजची एकूण देय रक्कम. |
| $this | `\WP_Ultimo\Checkout\Cart` | कार्ट ऑब्जेक्ट. |

### Source {#source}

[`inc/checkout/class-cart.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/checkout/class-cart.php#L2112) या फाईलमधील २११२ क्रमांकाच्या ओळीत हे परिभाषित केले आहे.
