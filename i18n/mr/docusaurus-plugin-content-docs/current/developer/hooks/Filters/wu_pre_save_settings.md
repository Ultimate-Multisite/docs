# Filter: wu_pre_save_settings

Ultimate Multisite द्वारे सेव्ह करण्यापूर्वी सेटिंग्स फिल्टर करण्याची सोय डेव्हलपर्सना मिळते.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $settings | `array` | ज्या सेटिंग्स सेव्ह करायच्या आहेत. |
| $settings_to_save | `array` | जोडायच्या नवीन सेटिंग्स. |
| $saved_settings | `array` | सध्या सेव्ह केलेल्या सेटिंग्स. |

### Since {#since}

- 2.0.18
### Source {#source}

[`inc/class-settings.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-settings.php#L296) या फाईलमध्ये लाइन २९६ वर परिभाषित केले आहे.
