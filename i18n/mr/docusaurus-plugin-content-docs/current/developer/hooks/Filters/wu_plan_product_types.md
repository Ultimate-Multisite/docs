# Filter: wu_plan_product_types {#filter-wuplanproducttypes}

ज्या प्रोडक्ट प्रकारांना 'प्लॅन' (plans) म्हणून मानावे, ते फिल्टर करतो.

हा फिल्टर add-ons ला अतिरिक्त प्रोडक्ट प्रकार नोंदवण्याची (register) परवानगी देतो, जे व्हॅलिडेशन (validation) आणि वर्गीकरण (segregation) मध्ये प्लॅन म्हणून ओळखले जावेत.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $plan_types | `array` | ज्या प्रोडक्ट प्रकारांना प्लॅन म्हणून मानावे, अशा प्रोडक्ट प्रकारांची ॲरे (array). |

### Since {#since}

- 2.3.0
