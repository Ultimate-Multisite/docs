# Filter: wu_current_\{$type\}_get_manage_url

## पॅरामीटर्स {#parameters}

| नाव | प्रकार | वर्णन |
|------|------|-------------|
| $manage_site_url | `string` | व्यवस्थापित साइट URL. |
| $id | `int` | साइट आयडी. |
| $site_hash | `string` | साइट हॅश. |

### कधीपासून {#since}
