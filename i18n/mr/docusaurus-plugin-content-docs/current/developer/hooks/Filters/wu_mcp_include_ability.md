# Filter: wu_mcp_include_ability

MCP (Multi-Capability Plugin) ची ability समाविष्ट करण्याची (include) निर्णयावर परिणाम करण्यासाठी वापरला जाणारा फिल्टर.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $include | `bool` | ability समाविष्ट करायची की नाही (True असल्यास समाविष्ट होईल). |
| $ability_id | `string` | ज्या ability चा ID आहे. |

### Since {#since}

- 2.4.8

[`inc/class-mcp-adapter.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/class-mcp-adapter.php#L286) मध्ये लाइन 286 वर परिभाषित केले आहे.
