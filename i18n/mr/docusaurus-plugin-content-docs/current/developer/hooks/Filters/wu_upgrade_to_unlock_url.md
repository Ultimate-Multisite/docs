# फिल्टर: wu_upgrade_to_unlock_url

डेव्हलपर्सना अपग्रेड टू अनलॉक URL बदलण्याची परवानगी देतो.

## पॅरामीटर्स

| नाव | प्रकार | वर्णन |
|------|------|-------------|
| $url | `string` | अपग्रेड URL. |
| $args | `array` | ज्या मर्यादेला (limit) अपग्रेड करण्याची गरज आहे, त्याचा मॉड्यूल आणि प्रकार. |

### स्रोत

[`inc/functions/limitations.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/functions/limitations.php#L177) मध्ये लाइन १७७ वर परिभाषित केले आहे.
