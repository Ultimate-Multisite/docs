# Action: wu_sso_loaded {#action-wussoloaded}

प्लगइन डेव्हलपर्सना गरज असल्यास अतिरिक्त हुक्स (hooks) जोडण्याची परवानगी देण्यासाठी.

SSO हे 'sunrise' वर चालणारे काहीतरी असल्यामुळे, हे 'init' पर्यंत पुढे ढकलावे लागेल.

## पॅरामीटर्स {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $sso | `self` | SSO क्लास. |
