# Action: wu_before_light_ajax

काही वेळा, ॲक्शन (actions) हाताळण्यासाठी आपल्याला अतिरिक्त कार्यक्षमता (extra juice) लोड करण्याची गरज पडेल.

ही ॲक्शन कधी आवश्यक आहे, तेव्हा अतिरिक्त अवलंबित्व (dependencies) लोड करण्यासाठी वापरली जाऊ शकते.
