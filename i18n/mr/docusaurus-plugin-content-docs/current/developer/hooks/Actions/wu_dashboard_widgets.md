# Action: wu_dashboard_widgets {#action-wudashboardwidgets}

प्लगइन डेव्हलपर्सना नेटवर्क डॅशबोर्ड पॅनेलमध्ये (Network Dashboard Panel) विजेट्स जोडण्याची परवानगी देते.

## Parameters {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $tab | `string` | सध्याचा टॅब (current tab). |
| $screen | `\WP_Screen` | स्क्रीन ऑब्जेक्ट (screen object). |
| $page | `\WP_Ultimo\Admin_Pages\Dashboard_Admin_Page` | Ultimate Multisite admin page ची इन्स्टन्स (instance). |
