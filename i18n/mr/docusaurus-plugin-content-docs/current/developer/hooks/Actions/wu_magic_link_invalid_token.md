# Action: wu_magic_link_invalid_token {#action-wumagiclinkinvalidtoken}

जेव्हा एखादे अवैध मॅजिक लिंक टोकन (invalid magic link token) आढळते, तेव्हा हे ॲक्शन ट्रिगर होते.

## पॅरामीटर्स {#parameters}

| Name | Type | Description |
|------|------|-------------|
| $reason | `string` | अवैध टोकनचे कारण. |

### Since {#since}

- 2.0.0

[`inc/sso/class-magic-link.php`](https://github.com/Ultimate-Multisite/ultimate-multisite/blob/main/inc/sso/class-magic-link.php#L326) या फाईलमध्ये, लाइन ३२६ वर परिभाषित केले आहे.
