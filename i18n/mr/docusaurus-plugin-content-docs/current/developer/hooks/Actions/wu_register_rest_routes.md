# Action: wu_register_rest_routes

अतिरिक्त रूट्स (routes) नोंदणीकृत (register) करण्यासाठी हे फंक्शन वापरले जाते.

हे फंक्शन आमच्या `/register` एंडपॉईंटद्वारे (endpoint) वापरले जाते.

## पॅरामीटर्स (Parameters) {#parameters}

| नाव (Name) | प्रकार (Type) | वर्णन (Description) |
|------|------|-------------|
| $api_instance | `self` | सध्याचे API इन्स्टन्स (API instance). |
