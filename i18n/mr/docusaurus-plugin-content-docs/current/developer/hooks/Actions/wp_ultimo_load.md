# Action: wp_ultimo_load

जेव्हा सर्व dependencies लोड होतात, तेव्हा हे ट्रिगर होते.

यामुळे plugin डेव्हलपर्सना नवीन कार्यक्षमता (functionality) जोडता येते. उदाहरणार्थ, नवीन Hosting providers साठी सपोर्ट जोडणे, इत्यादी.
