# VAT बदलांचा इतिहास {#vat-changelog}

Version 1.0.7 - रिलीझ झाले 2026-02-03

* दुरुस्ती (Fix): VIES SOAP service MS_MAX_CONCURRENT_REQ त्रुटीमुळे जर्मन (DE) VAT नंबर्सचे प्रमाणीकरण (validation) अयशस्वी होत होते. अधिक विश्वसनीय प्रमाणीकरणासाठी जुन्या SOAP API ऐवजी अधिकृत EU VIES REST API वापरण्यात आले आहे.
* दुरुस्ती (Fix): VIES सेवा तात्पुरती अनुपलब्ध असतानाही आता VAT नंबर्स स्वीकारले जातात, आणि ते चुकीच्या पद्धतीने नाकारले जात नाहीत. नंतर पुनरावलोकनासाठी एक लॉग एंट्री तयार केली जाते.
* दुरुस्ती (Fix): VAT/Tax ID फील्ड आता सर्व देशांसाठी दृश्यमान (visible) आहे, केवळ EU सदस्यांसाठी नाही. नॉन-EU कर आयडी (उदा. स्विस CHE नंबर्स) VIES प्रमाणीकरण न करता इनव्हॉइस डिस्प्लेसाठी साठवले जातात.
* दुरुस्ती (Fix): चुकीच्या ॲरे लुकअप लॉजिकमुळे ग्रीस (GR), मॉनॅको (MC) आणि आयल ऑफ मॅन (IM) साठी VAT प्रीफिक्स लुकअप तुटला होता.
* सुधारणा (Improvement): Site Exporter addon सोबत वापरल्यास घातक त्रुटी (fatal errors) निर्माण करू शकणारे अनावश्यक wp-cli-bundle dev dependency काढून टाकण्यात आले.
* सुधारणा (Improvement): सर्वसमावेशक युनिट टेस्ट सूट (53 चाचणी) जोडण्यात आले.

Version 1.0.6 - रिलीझ झाले 2026-01-25

* दुरुस्ती (Fix): आता देशांतर्गत B2B व्यवहारांवर (transactions) योग्यरित्या VAT आकारला जातो. EU VAT नियमांनुसार, रिव्हर्स चार्ज (reverse charge) केवळ सीमापार B2B व्यवहारांना लागू होतो, जेव्हा ग्राहकाचे देश कंपनीच्या देशाशी जुळत नाहीत.

Version 1.0.5 - रिलीझ झाले 2026-01-22

* दुरुस्ती (Fix): Addon लोड होत नव्हता किंवा व्यवस्थित काम करत नव्हता.
* बदल (Change): अधिक विश्वसनीय आणि सक्रियपणे देखरेख केलेल्या डेटासाठी VAT दर डेटा स्रोत euvatrates.com वरून ibericode/vat-rates रिपॉझिटरीमध्ये बदलण्यात आला.
* दुरुस्ती (Fix): super_reduced_rates ऑप्शन व्हॅल्यूमधील टायपो (typo) दुरुस्त करण्यात आला.
* सुधारणा (Improvement): त्रुटी टाळण्यासाठी देश कोड हाताळणीसाठी (country code handling) null checks जोडण्यात आले.
* बदल (Change): Traduttore द्वारे स्वयंचलित अपडेट्सच्या (automatic updates) बदल्यात बंडल केलेले ट्रान्सलेशन फाइल्स काढून टाकण्यात आले.
