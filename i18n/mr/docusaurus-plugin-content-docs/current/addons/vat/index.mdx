# VAT

Multisite Ultimate: युरोपीय VAT

हे Multisite Ultimate च्या डिफॉल्ट टॅक्स हँडलिंगमध्ये अतिरिक्त फीचर्स आणि टूल्स जोडते, ज्यामुळे युरोपीय VAT ला अधिक चांगल्या प्रकारे सपोर्ट मिळतो.
स्वयंचलितपणे EU VAT दर 'Tax Rates' पेजवर जोडले जातात.

## Installation {#installation}

1. 'ultimate-multisite-vat' ही फाईल '/wp-content/plugins/' या डिरेक्टरीमध्ये अपलोड करा.
2. WordPress मधील 'Plugins' मेनूद्वारे प्लगइन ॲक्टिव्हेट करा.
3. Ultimate Multisite > Settings > Taxes येथे जा आणि EU Vat ऑन करा.
4. Tax Rates वर क्लिक करा आणि खाली दिलेले EU VAT दर अपडेट करा आणि Tax Rates सेव्ह करा.
5. झालं!
